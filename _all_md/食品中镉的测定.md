2015-01-28 发布

2015-07-28 实施

# 前言

本标准代替GB/T5009.15—2003《食品中镉的测定》。

本标准与GB/T5009.15—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中镉的测定”；

——删除第二法原子吸收光谱法、第三法比色法、第四法原子荧光法。

# 食品安全国家标准

# 食品中镉的测定

# 1 范围

本标准规定了各类食品中镉的石墨炉原子吸收光谱测定方法。

本标准适用于各类食品中镉的测定。

# 2 原理

试样经灰化或酸消解后，注入一定量样品消化液于原子吸收分光光度计石墨炉中，电热原子化后吸收 $228.8\mathrm{nm}$ 共振线，在一定浓度范围内，其吸光度值与镉含量成正比，采用标准曲线法定量。

# 3 试剂和材料

注1：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。  
注2：所用玻璃仪器均需以硝酸溶液 $(1 + 4)$ 浸泡 $24\mathrm{h}$ 以上，用水反复冲洗，最后用去离子水冲洗干净。

# 3.1 试剂

3.1.1 硝酸 $(\mathrm{HNO}_3)$ ：优级纯。  
3.1.2 盐酸（HCl）：优级纯。  
3.1.3 高氯酸 $\mathrm{(HClO_4)}$ ：优级纯。  
3.1.4 过氧化氢 $(\mathrm{H}_2\mathrm{O}_2, 30\%)$ 。  
3.1.5 磷酸二氢铵 $(\mathrm{NH_4H_2PO_4})$ 。

# 3.2 试剂配制

3.2.1 硝酸溶液 $(1\%)$ ：取 $10.0~\mathrm{mL}$ 硝酸加入 $100~\mathrm{mL}$ 水中，稀释至 $1000~\mathrm{mL}$ 。  
3.2.2 盐酸溶液 $(1 + 1)$ ：取 $50~\mathrm{mL}$ 盐酸慢慢加入 $50~\mathrm{mL}$ 水中。  
3.2.3 硝酸-高氯酸混合溶液 $(9 + 1)$ ，取9份硝酸与1份高氯酸混合。  
3.2.4 磷酸二氢铵溶液 $(10\mathrm{g / L})$ ：称取 $10.0\mathrm{g}$ 磷酸二氢铵，用 $100~\mathrm{mL}$ 硝酸溶液 $(1\%)$ 溶解后定量移入 $1000~\mathrm{mL}$ 容量瓶，用硝酸溶液 $(1\%)$ 定容至刻度。

# 3.3 标准品

金属镉(Cd)标准品，纯度为 $99.99\%$ 或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 镉标准储备液 $(1000\mathrm{mg / L})$ ：准确称取 $1\mathrm{g}$ 金属镉标准品（精确至 $0.0001\mathrm{g}$ )于小烧杯中，分次加 $20~\mathrm{mL}$ 盐酸溶液 $(1 + 1)$ 溶解，加2滴硝酸，移入 $1000~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀；或购买经国家认证并授予标准物质证书的标准物质。

3.4.2 镉标准使用液 $(100\mathrm{ng / mL})$ ：吸取镉标准储备液 $10.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用硝酸溶液（ $1\%$ ）定容至刻度，如此经多次稀释成每毫升含 $100.0\mathrm{ng}$ 镉的标准使用液。  
3.4.3 锯标准曲线工作液：准确吸取镉标准使用液 $0\mathrm{mL}, 0.50\mathrm{mL}, 1.0\mathrm{mL}, 1.5\mathrm{mL}, 2.0\mathrm{mL}, 3.0\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中，用硝酸溶液(1%)定容至刻度，即得到含镉量分别为 $0\mathrm{ng / mL}, 0.50\mathrm{ng / mL}, 1.0\mathrm{ng / mL}, 1.5\mathrm{ng / mL}, 2.0\mathrm{ng / mL}, 3.0\mathrm{ng / mL}$ 的标准系列溶液。

# 4 仪器和设备

4.1 原子吸收分光光度计，附石墨炉。  
4.2 锅空心阴极灯。  
4.3 电子天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.4 可调温式电热板、可调温式电炉。  
4.5 马弗炉。  
4.6 恒温干燥箱。  
4.7 压力消解器、压力消解罐。  
4.8 微波消解系统：配聚四氟乙烯或其他合适的压力罐。

# 5 分析步骤

# 5.1 试样制备

5.1.1 干试样：粮食、豆类，去除杂质；坚果类去杂质、去壳；磨碎成均匀的样品，颗粒度不大于 $0.425 \mathrm{~mm}$ 。储于洁净的塑料瓶中，并标明标记，于室温下或按样品保存条件下保存备用。  
5.1.2 鲜(湿)试样：蔬菜、水果、肉类、鱼类及蛋类等，用食品加工机打成匀浆或碾磨成匀浆，储于洁净的塑料瓶中，并标明标记，于 $-16\,^{\circ}\mathrm{C} \sim -18\,^{\circ}\mathrm{C}$ 冰箱中保存备用。  
5.1.3 液态试样：按样品保存条件保存备用。含气样品使用前应除气。

# 5.2 试样消解

可根据实验室条件选用以下任何一种方法消解，称量时应保证样品的均匀性：

a）压力消解罐消解法：称取干试样 $0.3\mathrm{g}\sim 0.5\mathrm{g}$ （精确至 $0.0001\mathrm{g})$ 、鲜(湿)试样 $1\mathrm{g}\sim 2\mathrm{g}$ （精确到 $0.001\mathrm{g})$ 于聚四氟乙烯内罐，加硝酸 $5\mathrm{mL}$ 浸泡过夜。再加过氧化氢溶液 $(30\%)$ $2\mathrm{mL}\sim 3\mathrm{mL}$ （总量不能超过罐容积的1/3)。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱， $120^{\circ}C\sim 160^{\circ}C$ 保持 $4\mathrm{h}\sim 6\mathrm{h}$ ，在箱内自然冷却至室温，打开后加热赶酸至近干，将消化液洗入 $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 容量瓶中，用少量硝酸溶液 $(1\%)$ 洗涤内罐和内盖3次，洗液合并于容量瓶中并用硝酸溶液 $(1\%)$ 定容至刻度，混匀备用；同时做试剂空白试验。  
b）微波消解：称取干试样 $0.3\mathrm{g}\sim 0.5\mathrm{g}$ （精确至 $0.0001\mathrm{g})$ 、鲜(湿)试样 $1\mathrm{g}\sim 2\mathrm{g}$ （精确到 $0.001\mathrm{g})$ 置于微波消解罐中，加 $5\mathrm{mL}$ 硝酸和 $2\mathrm{mL}$ 过氧化氢。微波消化程序可以根据仪器型号调至最佳条件。消解完毕，待消解罐冷却后打开，消化液呈无色或淡黄色，加热赶酸至近干，用少量硝酸溶液 $(1\%)$ 冲洗消解罐3次，将溶液转移至 $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 容量瓶中，并用硝酸溶液 $(1\%)$ 定容至刻度，混匀备用；同时做试剂空白试验。  
c）湿式消解法：称取干试样 $0.3\mathrm{g}\sim 0.5\mathrm{g}$ （精确至 $0.0001\mathrm{g})$ 、鲜(湿)试样 $1\mathrm{g}\sim 2\mathrm{g}$ （精确到 $0.001\mathrm{g})$ 于锥形瓶中，放数粒玻璃珠，加 $10~\mathrm{mL}$ 硝酸-高氯酸混合溶液 $(9 + 1)$ ，加盖浸泡过夜，加一小漏斗在电热板上消化，若变棕黑色，再加硝酸，直至冒白烟，消化液呈无色透明或略带微黄色，放冷后将消化液洗入 $10~\mathrm{mL}\sim 25~\mathrm{mL}$ 容量瓶中，用少量硝酸溶液 $(1\%)$ 洗涤锥形瓶3次，洗液合并

于容量瓶中并用硝酸溶液 $(1\%)$ 定容至刻度，混匀备用；同时做试剂空白试验。

d）干法灰化：称取 $0.3\mathrm{g}\sim 0.5\mathrm{g}$ 干试样(精确至 $0.0001\mathrm{g})$ 、鲜(湿)试样 $1\mathrm{g}\sim 2\mathrm{g}$ （精确到 $0.001\mathrm{g})$ 、液态试样 $1\mathrm{g}\sim 2\mathrm{g}$ （精确到 $0.001\mathrm{g})$ 于瓷坩埚中，先小火在可调式电炉上炭化至无烟，移入马弗炉 $500^{\circ}C$ 灰化 $6\mathrm{h}\sim 8\mathrm{h}$ ，冷却。若个别试样灰化不彻底，加 $1\mathrm{mL}$ 混合酸在可调式电炉上小火加热，将混合酸蒸干后，再转入马弗炉中 $500^{\circ}C$ 继续灰化 $1\mathrm{h}\sim 2\mathrm{h}$ ，直至试样消化完全，呈灰白色或浅灰色。放冷，用硝酸溶液 $(1\%)$ 将灰分溶解，将试样消化液移入 $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 容量瓶中，用少量硝酸溶液 $(1\%)$ 洗涤瓷坩埚3次，洗液合并于容量瓶中并用硝酸溶液 $(1\%)$ 定容至刻度，混匀备用；同时做试剂空白试验。

注：实验要在通风良好的通风橱内进行。对含油脂的样品，尽量避免用湿式消解法消化，最好采用干法消化，如果必须采用湿式消解法消化，样品的取样量最大不能超过 $1\mathrm{g}$ 。

# 5.3 仪器参考条件

根据所用仪器型号将仪器调至最佳状态。原子吸收分光光度计(附石墨炉及镉空心阴极灯)测定参考条件如下：

波长 $228.8 \mathrm{~nm}$ ，狭缝 $0.2 \mathrm{~nm} \sim 1.0 \mathrm{~nm}$ ，灯电流 $2 \mathrm{~mA} \sim 10 \mathrm{~mA}$ ，干燥温度 $105^{\circ} \mathrm{C}$ ，干燥时间 $20 \mathrm{~s}$ ；  
——灰化温度 $400^{\circ}\mathrm{C} \sim 700^{\circ}\mathrm{C}$ ，灰化时间 $20\mathrm{s} \sim 40\mathrm{s}$ ；  
——原子化温度 $1300^{\circ}\mathrm{C} \sim 2300^{\circ}\mathrm{C}$ ，原子化时间 $3\mathrm{s} \sim 5\mathrm{s}$ ；  
——背景校正为氘灯或塞曼效应。

# 5.4 标准曲线的制作

将标准曲线工作液按浓度由低到高的顺序各取 $20~\mu \mathrm{L}$ 注入石墨炉，测其吸光度值，以标准曲线工作液的浓度为横坐标，相应的吸光度值为纵坐标，绘制标准曲线并求出吸光度值与浓度关系的一元线性回归方程。

标准系列溶液应不少于5个点的不同浓度的镉标准溶液，相关系数不应小于0.995。如果有自动进样装置，也可用程序稀释来配制标准系列。

# 5.5 试样溶液的测定

于测定标准曲线工作液相同的实验条件下，吸取样品消化液 $20\mu \mathrm{L}$ （可根据使用仪器选择最佳进样量），注入石墨炉，测其吸光度值。代入标准系列的一元线性回归方程中求样品消化液中镉的含量，平行测定次数不少于两次。若测定结果超出标准曲线范围，用硝酸溶液 $(1\%)$ 稀释后再行测定。

# 5.6 基体改进剂的使用

对有干扰的试样，和样品消化液一起注入石墨炉 $5\mu \mathrm{L}$ 基体改进剂磷酸二氢铵溶液（ $10~\mathrm{g / L}$ ），绘制标准曲线时也要加入与试样测定时等量的基体改进剂。

# 6 分析结果的表述

试样中镉含量按式(1)进行计算：

$$
X = \frac {\left(c _ {1} - c _ {0}\right) \times V}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中镉含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$c_{1}$ ——试样消化液中镉含量，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{0}$ ——空白液中镉含量，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样消化液定容总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量或体积，单位为克或毫升（g 或 mL）；

1000 ——换算系数。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

方法检出限为 $0.001\mathrm{mg / kg}$ ，定量限为 $0.003\mathrm{mg / kg}$

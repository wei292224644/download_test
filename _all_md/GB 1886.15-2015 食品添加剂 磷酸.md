# 检验方法

# A.1 安全提示

本试验方法中使用的部分试剂具有毒性、腐蚀性及易燃，操作者应小心谨慎。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。使用易燃品时，严禁使用明火加热。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 氢氧化钠溶液： $40~\mathrm{g / L}$   
A.3.1.2 硝酸银溶液： $10\mathrm{g / L}$   
A.3.1.3 酚酞指示液： $10\mathrm{g / L}$

# A.3.2 鉴别方法

称取约 $1\mathrm{g}$ 试样，置于 $100~\mathrm{mL}$ 烧杯中，加 $10~\mathrm{mL}$ 水、滴酚酞指示液，用氢氧化钠溶液调至中性，滴加硝酸银溶液，有黄色沉淀生成，该沉淀能溶于稀硝酸 $(5\%)$ 或氨水。

# A.4 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 含量的测定

# A.4.1 重量法（仲裁法）

# A.4.1.1 方法提要

在盐酸介质中试样与加入的喹钼柠酮沉淀剂生成磷钼酸喹啉沉淀，经过滤、洗涤、烘干及称重后，确定磷酸含量。

# A.4.1.2 试剂和材料

# A.4.1.2.1 盐酸。

# A.4.1.2.2 喹钼柠酮溶液的配制：

a）称取 $70\mathrm{g}$ 钼酸钠溶解于 $150~\mathrm{mL}$ 水中，此溶液为溶液A；  
b）称取 $60\mathrm{g}$ 柠檬酸溶解于 $150~\mathrm{mL}$ 水和 $85~\mathrm{mL}$ 硝酸的混合溶液中，此溶液为溶液B；  
c）在搅拌下将溶液A倒入溶液B中，此溶液为溶液C；  
d) 在 $100 \mathrm{~mL}$ 水中加入 $35 \mathrm{~mL}$ 硝酸, 再加入 $5 \mathrm{~mL}$ 喹啉, 此溶液为溶液 D;

e) 将溶液 D 倒入溶液 C 中, 混匀。放置 $12 \mathrm{~h}$ 后, 用玻璃砂坩埚过滤, 再加入 $280 \mathrm{~mL}$ 丙酮, 用水稀释至 $1000 \mathrm{~mL}$ 混匀, 贮存于聚乙烯瓶中。

# A.4.1.3 仪器和设备

A.4.1.3.1 玻璃砂坩埚：滤板孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$   
A.4.1.3.2 电烘箱：温度能控制在 $180^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 或 $250^{\circ}\mathrm{C} \pm 10^{\circ}\mathrm{C}$ 。

# A.4.1.4 分析步骤

# A.4.1.4.1 试验溶液的制备

称取约 $1\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $100\mathrm{mL}$ 烧杯中，加 $5\mathrm{mL}$ 盐酸及适量的水，盖上表面皿，煮沸 $10\mathrm{min}$ ，冷却后移入 $500\mathrm{mL}$ 容量瓶中，加 $10\mathrm{mL}$ 盐酸，用水稀释至刻度，摇匀。

# A.4.1.4.2 空白试验溶液的制备

在制备试验溶液的同时，除不加试样外，其他操作和加入的试剂量同A.4.1.4.1处理。

# A.4.1.4.3 测定

用移液管移取 $10\mathrm{mL}$ 试验溶液、空白试验溶液分别置于 $250~\mathrm{mL}$ 烧杯中，加水至总体积约 $100~\mathrm{mL}$ ，加 $50~\mathrm{mL}$ 唑钼柠酮溶液，盖上表面皿，在水浴中加热至烧杯内的物质达 $75^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ ，保温 $30~\mathrm{s}$ （在加试剂和加热过程中不应使用明火，不应搅拌，以免凝结成块）。取出并冷却至室温，冷却过程中搅拌3次～4次，用预先在 $180^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 或 $250^{\circ}\mathrm{C} \pm 10^{\circ}\mathrm{C}$ 下恒重过的玻璃砂坩埚过滤，先将上层清液过滤，以倾泻法用洗瓶冲洗沉淀6次，每次用水约 $30~\mathrm{mL}$ ，最后将沉淀移入玻璃砂坩埚内过滤。再用水洗涤4次，将玻璃砂坩埚连同沉淀置于电烘箱中，从温度稳定时开始计时，温度控制在 $180^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ ，放置 $45~\mathrm{min}$ ，或 $250^{\circ}\mathrm{C} \pm 10^{\circ}\mathrm{C}$ 放置 $15~\mathrm{min}$ ，取出，置于干燥器中冷却至室温，称重。

# A.4.1.5 结果计算

磷酸 $\mathrm{(H_3PO_4)}$ 含量的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w_{1} = \frac{0.04428\times\left(m_{1} - m_{2}\right)}{m\times\frac{10}{500}}\times 100\% \quad \dots \tag{A.1}
$$

式中：

0.04428——将磷钼酸喹啉换算成磷酸的系数；

$m_{1}$ ——测定试验溶液的沉淀质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——测定空白试验溶液的沉淀质量，单位为克 $(\mathrm{g})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

10 ——用移液管移取的试验溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

500 ——制备的试验溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

取平行测定结果的算术平均值为测定结果；平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.4.2 容量法

# A.4.2.1 方法提要

根据磷酸性质，以百里香酚酞为指示液，用氢氧化钠标准滴定溶液直接滴定，以确定磷酸含量。

# A.4.2.2 试剂和材料

A.4.2.2.1 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 1\mathrm{mol / L}$

A.4.2.2.2 百里香酚酞指示液： $1\mathrm{g / L}$

# A.4.2.3 分析步骤

称取约 $1.5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，移入 $250~\mathrm{mL}$ 锥形瓶中，加 $120~\mathrm{mL}$ 水和5滴百里香酚酞指示液，用氢氧化钠标准滴定溶液滴定至浅蓝色即为终点。

# A.4.2.4 结果计算

磷酸 $\mathrm{(H_3PO_4)}$ 含量的质量分数 $\varpi_{1}$ ，按式(A.2)计算：

$$
w _ {1} = \frac {\frac {V}{1000} \times c \times M}{m} \times 100 \% \tag{A.2}
$$

式中：

V ——滴定所消耗氢氧化钠标准滴定溶液(A.4.2.2.1)的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——毫升和升的单位转换系数；

$c$ ——氢氧化钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——磷酸的摩尔质量的 $\frac{1}{2}$ ，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(\frac{1}{2}\mathrm{H}_3\mathrm{PO}_4) = 49.00]$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

取平行测定结果的算术平均值为测定结果；平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 氟化物（以F计）的测定

# A.5.1 方法提要

在中性条件下，以饱和甘汞电极为参比电极，用氟离子选择性电极直接测量溶液的电极电位，采用标准曲线法测定氟含量。

# A.5.2 试剂和材料

A.5.2.1 盐酸溶液： $1 + 1$

A.5.2.2 硝酸溶液： $1 + 15$

A.5.2.3 氢氧化钠溶液： $200\mathrm{g / L}$

A.5.2.4 柠檬酸-柠檬酸三钠缓冲溶液： $\mathrm{pH} = 5.5 \sim 6$ 。称取 $270 \mathrm{~g}$ 柠檬酸三钠 $(\mathrm{Na}_{3} \mathrm{C}_{6} \mathrm{H}_{5} \mathrm{O}_{7} \cdot 2 \mathrm{H}_{2} \mathrm{O})$ 和 $24 \mathrm{~g}$ 柠檬酸 $(\mathrm{C}_{6} \mathrm{H}_{6} \mathrm{O}_{7} \cdot \mathrm{H}_{2} \mathrm{O})$ 用水溶解，稀释至 $1000 \mathrm{~mL}$ ，混匀。

A.5.2.5 溴甲酚绿指示液： $1\mathrm{g / L}$

A.5.2.6 氟化物标准溶液: $1 \mathrm{~mL}$ 含氟(F) $2 \mu \mathrm{g}$ 。临用时配制。用移液管移取 $2 \mathrm{~mL}$ 按 GB/T 602 配制的氟化物标准溶液, 置于 $100 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度, 摇匀。

# A.5.3 仪器和设备

A.5.3.1 氟离子选择性电极。

A.5.3.2 饱和甘汞电极。

A.5.3.3 电位计：精度为 $2\mathrm{mV}$ /格。

A.5.3.4 电磁搅拌器。

# A.5.4 分析步骤

# A.5.4.1 工作曲线的绘制

用移液管移取 $0.00\mathrm{mL},1.00\mathrm{mL},3.00\mathrm{mL},5.00\mathrm{mL},7.00\mathrm{mL},10.00\mathrm{mL}$ 氟化物标准溶液，分别置于 $50~\mathrm{mL}$ 容量瓶中，加入 $1\mathrm{mL}$ 盐酸溶液、5滴柠檬酸-柠檬酸三钠缓冲溶液和2滴溴甲酚绿指示液，用氢氧化钠溶液调至溶液为蓝色，再用硝酸溶液调至溶液恰成黄色，加 $20~\mathrm{mL}$ 柠檬酸-柠檬酸三钠缓冲溶液，用水稀释至刻度，混匀。将溶液倒入 $50~\mathrm{mL}$ 干燥的烧杯中，置于电磁搅拌器上，插入氟离子选择性电极和饱和甘汞电极，连接电位计，搅拌片刻，调整电位计零点后，进行测量，记录平衡时的电位值。以氟化物(以F计)质量的数值的对数值为横坐标，相应的电位值为纵坐标，绘制标准曲线。

# A.5.4.2 试验溶液的制备

称取约 $10\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $100\mathrm{mL}$ 烧杯中，用少量水稀释，用氢氧化钠溶液调至中性，移入 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。

# A.5.4.3 测定

用移液管移取 $10\mathrm{mL}$ 试验溶液，置于 $50~\mathrm{mL}$ 容量瓶中，加入 $1\mathrm{mL}$ 盐酸溶液、5滴柠檬酸-柠檬酸三钠缓冲溶液和2滴溴甲酚绿指示液，用氢氧化钠溶液调至溶液为蓝色，再用硝酸溶液调至溶液恰呈黄色，加 $20~\mathrm{mL}$ 柠檬酸-柠檬酸三钠缓冲溶液，用水稀释至刻度，混匀。将溶液倒入 $50~\mathrm{mL}$ 干燥的烧杯中，置于电磁搅拌器上，插入氟离子选择性电极和饱和甘汞电极，连接电位计，搅拌片刻，调整电位计零点后，进行测量，记录平衡时的电位值。从标准曲线上查出相应的氟化物(以F计)质量的数值的对数，查反对数得到氟化物(以F计)质量的数值。

# A.5.5 结果计算

氟化物（以F计)的质量分数 $\omega_{2}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.3)计算：

$$
w _ {2} = \frac {\frac {m _ {1}}{1 0 ^ {6}}}{m \times \frac {1 0}{1 0 0}} \times 1 0 0 \tag {A.3}
$$

式中：

$m_{1}$ ——试验溶液中氟化物(以F计)的质量，单位为微克 $(\mu \mathrm{g})$ ；

$10^{6}$ ——微克和克的单位转换系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

10 ——用移液管移取的试验溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

100 ——制备的试验溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

取平行测定结果的算术平均值为测定结果；平行测定结果的绝对差值不大于 $0.0005\%$ 。

# A.6 易氧化物（以 $\mathbf{H}_3\mathbf{PO}_3$ 计）的测定

# A.6.1 方法提要

在过量酸性硫酸铈标准滴定溶液中,与试样中的易氧化物反应,过量硫酸铈标准滴定溶液用硫酸亚

铁铵标准滴定溶液滴定。

# A.6.2 试剂和材料

A.6.2.1 硫酸溶液： $1 + 2$   
A.6.2.2 硫酸银溶液 $(10\mathrm{g / L})$ ：称取 $1\mathrm{g}$ 硫酸银，加 $50~\mathrm{mL}$ 硫酸溶液 $(1 + 2)$ 溶解，用水稀释至 $100~\mathrm{mL}$ 摇匀。  
A.6.2.3 硫酸铈标准滴定溶液： $c[\mathrm{Ce(SO_4)_2}] = 0.1\mathrm{mol / L}$   
A.6.2.4 硫酸亚铁铵标准滴定溶液： $c\big[(\mathrm{NH}_4)_2\mathrm{Fe}(\mathrm{SO}_4)_2\big] = 0.1\mathrm{mol / L}$   
A.6.2.5 1,10-菲啰啉-亚铁指示液。

# A.6.3 分析步骤

在 $250~\mathrm{mL}$ 锥形瓶中，加入 $40~\mathrm{mL}$ 硫酸溶液，用移液管移取 $10.00~\mathrm{mL}$ 硫酸铈标准滴定溶液，再加入 $10~\mathrm{mL}$ 硫酸银溶液，摇匀。滴加 $10\mathrm{g}$ （约 $6\mathrm{mL}$ 试样，加 $40~\mathrm{mL}$ 水，煮沸 $15\mathrm{min}$ ，冷却。用水稀释至原体积，加2滴1,10-菲啰啉-亚铁指示液，用硫酸亚铁铵标准滴定溶液滴定至溶液呈红色。同时做空白试验。

# A.6.4 结果计算

易氧化物(以 $\mathrm{H}_3\mathrm{PO}_3$ 计)的质量分数 $\varpi_{3}$ ，按式(A.4)计算：

$$
w _ {3} = \frac {\frac {V _ {1} - V _ {2}}{1000} \times c \times M}{m} \times 100 \% \tag{A.4}
$$

式中：

$V_{1}$ ——空白试验时消耗硫酸亚铁铵标准滴定溶液(A.6.2.4)的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴定时消耗硫酸亚铁铵标准滴定溶液(A.6.2.4)的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——毫升和升的单位转换系数；

$c$ ——硫酸亚铁铵标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——亚磷酸的摩尔质量的 $\frac{1}{2}$ , 单位为克每摩尔 $(\mathrm{g/mol})$ , $\left[M\left(\frac{1}{2} \mathrm{H}_{3} \mathrm{PO}_{3}\right)=41.00\right]$ ;

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

取平行测定结果的算术平均值为测定结果；平行测定结果的绝对差值不大于 $0.0005\%$ 。

# A.7 砷（As)的测定

# A.7.1 二乙基二硫代氨基甲酸银法（仲裁法）

# A.7.1.1 方法提要

在硫酸介质中，用金属锌将砷还原为砷化氢气体，以二乙基二硫代氨基甲酸银-吡啶溶液吸收，对生成的紫红色胶体银溶液作吸光度测定。

# A.7.1.2 试剂和材料

A.7.1.2.1 无砷锌粒。

A.7.1.2.2 硫酸溶液： $1 + 4$

A.7.1.2.3 碘化钾溶液： $150~\mathrm{g / L}$

A.7.1.2.4 氯化亚锡溶液： $400\mathrm{g / L}$

A.7.1.2.5 二乙基二硫代氨基甲酸银-吡啶溶液：称取 $0.5\mathrm{g}$ 二乙基二硫代氨基甲酸银，溶于 $100~\mathrm{mL}$ 吡啶，置于棕色瓶避光。

A.7.1.2.6 砷标准溶液： $1\mathrm{mL}$ 含砷(As) $1\mu \mathrm{g}$ ，临用时配制。用移液管移取 $1\mathrm{mL}$ 按GB/T602中规定的砷标准溶液，置于 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。

A.7.1.2.7 乙酸铅棉花。

A.7.1.3.2 分光光度计：带有 $1\mathrm{cm}$ 比色池。

# A.7.1.4 分析步骤

# A.7.1.4.1 工作曲线的绘制

用移液管移取 $0.00\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},4.00\mathrm{mL},6.00\mathrm{mL},8.00\mathrm{mL}$ 砷标准溶液，分别置于6个砷发生瓶中，加水至 $40~\mathrm{mL}$ ，加 $20~\mathrm{mL}$ 硫酸溶液，混匀。加入 $2\mathrm{mL}$ 碘化钾溶液、 $2\mathrm{mL}$ 氯化亚锡溶液，摇匀。放置 $15\mathrm{min}$ 。将装有乙酸铅棉花的吸收管中加入 $3\mathrm{mL}$ 二乙基二硫代氨基甲酸银-吡啶溶液，在砷发生瓶中加入 $3\mathrm{g}$ 无砷金属锌粒，立即将吸收管与砷发生瓶联接。待反应完全后（常温下约 $45\mathrm{min}$ ），将吸收管内的吸收液在 $540~\mathrm{nm}$ 处，用 $1\mathrm{cm}$ 比色池测定吸光度。由每个砷标准溶液的吸光度减去空白溶液的吸光度，以标准溶液中所含的砷(As)的微克数为横坐标，以对应的吸光度为纵坐标，绘制工作曲线。

# A.7.1.4.2 测定

称取约 $10\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于砷发生瓶中，加水至 $40~\mathrm{mL}$ ，混匀。加 $20~\mathrm{mL}$ 硫酸溶液，混匀。加入 $2\mathrm{mL}$ 碘化钾溶液、 $2\mathrm{mL}$ 氯化亚锡溶液，摇匀。放置 $15\mathrm{min}$ 。将装有乙酸铅棉花的吸收管中加入 $3\mathrm{mL}$ 二乙基二硫代氨基甲酸银-吡啶溶液，在砷发生瓶中加入 $3\mathrm{g}$ 无砷金属锌粒，立即将吸收管与砷发生瓶联接。待反应完全后（常温下约 $45\mathrm{min}$ ），将吸收管内的吸收液在 $540~\mathrm{nm}$ 处，用 $1\mathrm{cm}$ 比色池测定吸光度。从测定试验溶液的吸光度中减去空白溶液的吸光度，由工作曲线上查出相应的砷（As）含量。

# A.7.1.5 结果计算

砷(As)的质量分数 $w_{4}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.5)计算：

$$
w _ {4} = \frac {m _ {1}}{m} \times 100 \% \tag{A.5}
$$

式中：

$m_{1}$ ——由工作曲线上查出的砷的质量分数，单位为微克 $(\mu \mathrm{g})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

取平行测定结果的算术平均值为测定结果；平行测定结果的绝对差值不大于 $0.2\mathrm{mg / kg}$ 。

# A.7.2.1 方法提要

在硫酸介质中，用金属锌将砷还原为砷化氢，砷化氢与溴化汞反应生成红棕色或浅黄砷斑，与标准色斑比较。

A.7.2.2.4 氯化亚锡溶液： $400\mathrm{g / L}$

A.7.2.2.5 砷标准溶液： $1\mathrm{mL}$ 含砷（As） $1\mu \mathrm{g}$ ，临用时配制。用移液管移取 $1\mathrm{mL}$ 按GB/T602中规定的砷标准溶液，置于 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。

A.7.2.2.6 溴化汞试纸。

A.7.2.2.7 乙酸铅棉花。

A.7.2.2.8 标准色斑的制备：用移液管移取 $5.0\mathrm{mL}$ 砷标准溶液置于砷发生瓶中。加水至 $40~\mathrm{mL}$ ，加 $20~\mathrm{mL}$ 硫酸溶液，混匀。加入 $2\mathrm{mL}$ 碘化钾溶液、 $2\mathrm{mL}$ 氯化亚锡溶液，摇匀。

# A.8 重金属（以 $\mathbf{Pb}$ 计）的测定

# A.8.1 方法提要

在 $\mathrm{pH}3\sim 4$ 条件下，试样中的重金属离子与硫化氢作用，生成棕色悬浮物，与同法处理的铅标准溶液比较。

# A.8.2 试剂和材料

A.8.2.1 冰乙酸。  
A.8.2.2 氨水溶液： $1 + 1$   
A.8.2.3 硫化钠-丙三醇溶液配制：称取 $5\mathrm{g}$ 硫化钠，溶于 $10~\mathrm{mL}$ 水和 $30~\mathrm{mL}$ 丙三醇混合溶液中。此溶液应避光、密封保存于棕色瓶中，三个月内有效。  
A.8.2.4 铅标准溶液: $1 \mathrm{~mL}$ 含铅(Pb) $10 \mu \mathrm{g}$ , 临用时配制。用移液管移取 $10 \mathrm{~mL}$ 按 GB/T 602 配制的铅标准溶液, 置于 $100 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度, 摇匀。  
A.8.2.5 铅标准比色液的配制：用移液管移取 $5\mathrm{mL}$ 试验溶液和适量 $(1\mathrm{mL}\sim 3\mathrm{mL})$ 铅标准溶液，置于 $50~\mathrm{mL}$ 比色管中，与试验溶液同时同样处理。为求得重金属的含量，需配制一系列的铅标准比色液。

# A.8.3 分析步骤

# A.8.3.1 试验溶液的制备

称取约 $10\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ 。置于 $50~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。

# A.8.3.2 测定

用移液管移取 $25~\mathrm{mL}$ 试验溶液，置于 $50~\mathrm{mL}$ 比色管中，用氨水溶液调至 $\mathrm{pH}3\sim 4$ （用精密试纸检验)，用水稀释至约 $40~\mathrm{mL}$ ，加入 $10~\mathrm{mL}$ 新制备的饱和硫化氢水，摇匀，放置 $10\mathrm{min}$ ，与铅标准比色液比较。

取与试样管浊度相当的标准管中的硫酸盐的量进行计算。当试样管浊度介于两支标准管浊度之间时，按两标准管中硫酸盐量的平均值进行计算。

# A.8.4 结果计算

重金属(以Pb计)的质量分数 $\varpi_{5}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.6)计算：

$$
w_{5} = \frac{V\times 10}{m\times\frac{25 - 5}{50}}\times 100\% \tag{A.6}
$$

式中：

$V$ ——所取铅标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

10 ——铅标准溶液的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；  
$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；  
25 ——用移液管移取的试验溶液的体积，单位为毫升 $(\mathrm{mL})$ ；  
5 ——铅标准比色液中试验溶液的体积，单位为毫升 $(\mathrm{mL})$ ；  
50 ——制备的试验溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

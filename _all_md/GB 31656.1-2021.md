# 高效液相色谱法

# 1 范围

本文件规定了水产品中甲苯咪唑及其代谢物氨基甲苯咪唑、羟基甲苯咪唑残留检测的制样和液相色谱测定方法。

本文件适用于鱼、虾和蟹可食组织中甲苯咪唑及其代谢物氨基甲苯咪唑和羟基甲苯咪唑残留量的检测。

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的甲苯咪唑及其代谢物，用乙酸乙酯提取，正己烷除脂，阳离子固相萃取柱净化，高效液相色谱紫外检测器测定，外标法定量。

# 5 试剂和材料

以下所用试剂，除另有注明外均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.3 正己烷 $(\mathrm{C_6H_{14}})$ ：色谱纯。  
5.1.4 乙酸乙酯 $\mathrm{(CH_3COOCH_2CH_3)}$ ：色谱纯。  
5.1.5 N，N-二甲基甲酰胺 $\mathrm{(C_3H_7NO)}$ ：色谱纯。  
5.1.6 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。  
5.1.7 磷酸二氢铵 $(\mathrm{NH_4H_2PO_4})$ 。  
5.1.8 三乙胺 $\left[(\mathrm{C}_2\mathrm{H}_5)_3\mathrm{N}\right]$   
5.1.9 甲酸（HCOOH）。  
5.1.10 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。

# 5.2 标准品

甲苯咪唑(mebendazole, $\mathrm{C_{16}H_{13}N_3O_3}$ , CAS号: 31431-39-7)、羟基甲苯咪唑(5-hydroxymebendazole, $\mathrm{C_{16}H_{15}N_3O_3}$ , CAS号: 60254-95-7)和氨基甲苯咪唑(amino-mebendazole, $\mathrm{C_{14}H_{11}N_3O}$ , CAS号: 52329-60-9): 含量 $\geqslant 98\%$ 。

# 5.3 溶液配制

5.3.1 $80\%$ 甲醇溶液：取甲醇 $80~\mathrm{mL}$ ，加水至 $100~\mathrm{mL}$ ，混匀。  
5.3.2 $1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，加水至 $100~\mathrm{mL}$ ，混匀。  
5.3.3 $0.05\mathrm{mol} / \mathrm{L}$ 磷酸二氢铵-三乙胺溶液：取磷酸二氢铵 $5.8\mathrm{g}$ ，加水溶解并稀释至 $1000\mathrm{mL}$ ，加三乙胺 $1\mathrm{mL}$ ，混匀。  
5.3.4 $0.05\mathrm{mol} / \mathrm{L}$ 磷酸二氢铵缓冲液：取磷酸二氢铵 $2.9\mathrm{g}$ ，加水 $450~\mathrm{mL}$ 使溶解，用磷酸调 $\mathrm{pH}$ 至2.0，用水稀释至 $500~\mathrm{mL}$ 。  
5.3.5 $5\%$ 氨化甲醇溶液：取氨水 $5\mathrm{mL}$ 、甲醇 $95~\mathrm{mL}$ ，混匀；临用前配制。  
5.3.6溶解液：取N，N-二甲基甲酰胺 $150~\mathrm{mL}$ ，用 $0.05\mathrm{mol / L}$ 磷酸二氢铵缓冲液稀释至 $500~\mathrm{mL}$

# 5.4 标准溶液制备

5.4.1 标准储备液 $(100\mu \mathrm{g / mL})$ ：分别取甲苯咪唑、氨基甲苯咪唑和羟基甲苯咪唑各 $10\mathrm{mg}$ ，精密称定，加甲酸 $5\mathrm{mL}$ 使其溶解，并用甲醇稀释定容至 $100~\mathrm{mL}$ 容量瓶，摇匀，配制成浓度各为 $100~\mu \mathrm{g / mL}$ 的标准储备液。一 $18^{\circ}C$ 以下避光保存，有效期6个月。  
5.4.2 混合标准中间液：分别准确量取甲苯咪唑、氨基甲苯咪唑和羟基甲苯咪唑标准储备液适量，用甲醇稀释配制成含氨基甲苯咪唑 $5\mu \mathrm{g / mL}$ 、羟基甲苯咪唑 $2.5\mu \mathrm{g / mL}$ 和甲苯咪唑 $2.5\mu \mathrm{g / mL}$ 的混合标准中间液。 $-18^{\circ}C$ 以下避光保存，有效期3个月。

# 5.5 材料

5.5.1 混合型阳离子固相萃取柱： $60~\mathrm{mg / 3~mL}$ ，或相当者。  
5.5.2 水相聚砜醚针式滤器： $0.45\mu \mathrm{m}$

# 6 仪器和设备

6.1 高效液相色谱仪，配紫外检测器  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 离心机： $6000\mathrm{r / min}$   
6.4 旋转蒸发仪。  
6.5 固相萃取装置。  
6.6 超声波清洗仪。  
6.7 氮吹仪。  
6.8 鸡心瓶： $100~\mathrm{mL}$

# 7 试样的制备与保存

# 7.1 试样的制备

按GB/T30891—2014中附录B的要求制样。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的混合标准溶液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存，有效期3个月。

# 8 测定步骤

# 8.1 提取

取试样 $3\mathrm{g}$ （准确至 $\pm 0.03\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加水 $2\mathrm{mL}$ ，涡旋 $30~\mathrm{s}$ ，使其分散；加乙酸乙酯

$10~\mathrm{mL}$ ，振荡 $2\mathrm{min}$ ，超声 $5\mathrm{min},4500\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，乙酸乙酯转入 $100~\mathrm{mL}$ 鸡心瓶。残渣加乙酸乙酯 $10~\mathrm{mL}$ ，按上述方法重复提取2次。合并乙酸乙酯至 $100~\mathrm{mL}$ 鸡心瓶中， $40^{\circ}C$ 旋转蒸发至干。加 $80\%$ 甲醇溶液 $3\mathrm{mL}$ ，涡旋混合 $1\mathrm{min}$ ，加正己烷 $3\mathrm{mL}$ ，混合 $1\mathrm{min}$ ，将溶液转入 $10~\mathrm{mL}$ 离心管中， $6000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，去除正己烷层；下层再加正己烷 $3\mathrm{mL}$ ，重复去脂一次。向下层溶液中加 $1\%$ 甲酸水溶液 $3\mathrm{mL}$ ，混匀， $6000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液备用。

# 8.2 净化

固相萃取柱依次用甲醇、水各 $3\mathrm{mL}$ 活化，将上清液过柱；依次用水、甲醇各 $4\mathrm{mL}$ 淋洗，弃去流出液，抽干。用 $5\%$ 氨化甲醇溶液 $5\mathrm{mL}$ 洗脱，洗脱液接收至 $10\mathrm{mL}$ 离心管中， $40^{\circ}\mathrm{C}$ 氮气吹干。加溶解液 $1.0\mathrm{mL}$ 溶解残余物，涡旋混合 $1\mathrm{min}$ ，用水相针式滤器过滤至进样小瓶中，供高效液相色谱测定。

# 8.3 标准工作曲线的制备

精密量取混合标准中间液适量，用溶解液稀释配制成甲苯咪唑、羟基甲苯咪唑浓度分别为 $0.02\mu \mathrm{g / mL}$ 产 $0.05~\mu \mathrm{g / mL},0.1~\mu \mathrm{g / mL},0.25~\mu \mathrm{g / mL},0.5~\mu \mathrm{g / mL}$ 和 $1.0~\mu \mathrm{g / mL}$ ，氨基甲苯咪唑浓度为 $0.04~\mu \mathrm{g / mL}$ $0.1\mu \mathrm{g / mL},0.2\mu \mathrm{g / mL},0.5\mu \mathrm{g / mL},1.0\mu \mathrm{g / mL}$ 和 $2\mu \mathrm{g / mL}$ 的系列标准工作液，供高效液相色谱测定。分别以甲苯咪唑、氨基甲苯咪唑和羟基甲苯咪唑的峰面积为纵坐标，相应浓度为横坐标，绘制标准曲线，求回归方程和相关系数。此标准工作液现用现配。

# 8.4.2 测定法

取系列标准工作液和试样溶液，作单点或多点校准，按外标法以色谱峰面积定量。标准溶液和试样液中甲苯咪唑、氨基甲苯咪唑和羟基甲苯咪唑的响应值均应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液液相色谱图见附录A。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中甲苯咪唑、氨基甲苯咪唑和羟基甲苯咪唑的残留量按公式(1)计算。

$$
X _ {i} = \frac {C _ {i} \times V \times f}{m} \times 1 0 0 0 \tag {1}
$$

式中：

$X_{i}$ ——试样中甲苯咪唑或代谢物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_i$ ——从标准工作曲线计算得到的试样溶液中甲苯咪唑或代谢物浓度的数值，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——溶解残余物体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释倍数；

1000——换算系数；

$m$ 供试试样质量的数值，单位为克 $(\mathrm{g})$

# 10 方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法甲苯咪唑和羟基甲苯咪唑检测限为 $5\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$

本方法氨基甲苯咪唑检测限为 $10~\mu \mathrm{g / kg}$ ，定量限为 $20~\mu \mathrm{g / kg}$

# 10.2 准确度

本方法甲苯咪唑、羟基甲苯咪唑在 $10\mu \mathrm{g / kg}\sim 200\mu \mathrm{g / kg}$ ，氨基甲苯咪唑在 $20~\mu \mathrm{g / kg}\sim 400~\mu \mathrm{g / kg}$ 的添加浓度范围内，回收率为 $70\% \sim 100\%$ 。

# 10.3 精密度

本方法批内相对标准偏差 $= 15\%$ ，批间相对标准偏差 $= 15\%$ 。

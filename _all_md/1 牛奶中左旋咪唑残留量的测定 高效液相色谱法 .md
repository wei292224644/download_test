# 1 范围

本标准规定了牛奶中左旋咪唑残留量检测的制样和高效液相色谱测定方法。

本标准适用于牛奶中左旋咪唑残留量检测。

# 3原理

试料中残留的左旋咪唑，用碳酸盐缓冲液和乙酸乙酯溶液提取， $\mathrm{C}_{18}$ 柱净化，甲醇洗脱，高效液相色谱测定，外标法定量。

# 4 试剂与材料

以下所用的试剂，除特别注明外均为分析纯试剂；水为符合GB/T6682规定的一级水。

4.1盐酸左旋咪唑标准品，含量 $\geq 99.0\%$   
4.2 甲醇  
4.3 无水碳酸氢钠：优级纯。  
4.4碳酸钠   
4.5 磷酸二氢钠   
4.6 浓氨水  
4.7 二乙胺：优级纯。  
4.8 乙腈

# 4.9 磷酸

# 4.10 盐酸

4.11 $\mathrm{C_{18}}$ 固相萃取柱： $500\mathrm{mg / 3mL}$   
4.12 $10\%$ 磷酸溶液：取磷酸 $70~\mathrm{mL}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。  
4.13 3 mol/L盐酸溶液：取盐酸250 mL，用水溶解并稀释至1000 mL。  
4.14 碳酸氢钠饱和水溶液：取水 $1000 \mathrm{~mL}$ ，加无水碳酸氢钠至不溶解为止，现用现配。  
4.15 碳酸钠饱和水溶液：取水 $500 \mathrm{~mL}$ ，加碳酸钠至不溶解为止，现用现配。  
4.16 碳酸盐缓冲液（约 $\mathrm{pH} = 9$ ）：取碳酸氢钠饱和水溶液 $900 \mathrm{~mL}$ ，加碳酸钠饱和水溶液 $100 \mathrm{~mL}$ ，混匀。  
4.17 $0.02 \mathrm{~mol} / \mathrm{L}$ 磷酸二氢钠二乙胺缓冲溶液: 取磷酸二氢钠 $2.44 \mathrm{~g}$ , 加水 $850 \mathrm{~mL}$ 溶解, 加二乙胺 $3 \mathrm{~mL}$ 混合, 用 $10 \%$ 磷酸调 $\mathrm{pH}$ 至 7.5, 用水稀释至 $1000 \mathrm{~mL}$ 。  
4.18 $1 \mathrm{mg} / \mathrm{mL}$ 左旋咪唑标准贮备液：精密称取盐酸左旋咪唑标准品 $10 \mathrm{mg}$ ，于 $10 \mathrm{~mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的左旋咪唑标准贮备液。 $2 \sim 4^{\circ} \mathrm{C}$ 保存，有效期3个月。  
4.19 $10 \mu \mathrm{g} / \mathrm{mL}$ 左旋咪唑标准工作液: 精密量取 $1 \mathrm{mg} / \mathrm{mL}$ 左旋咪唑标准贮备液 $1.0 \mathrm{~mL}$ , 于 $100 \mathrm{~mL}$ 量瓶中,用甲醇稀释至刻度, 配制成浓度为 $10 \mu \mathrm{g} / \mathrm{L}$ 的左旋咪唑标准工作液。 $2 \sim 4^{\circ} \mathrm{C}$ 保存, 有效期3个月。

# 5 仪器和设备

5.1 高效液相色谱仪：配紫外检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 电子天平：感量 $0.01\mathrm{g}$ 。  
5.4 均质机。  
5.5 冷冻高速离心机。  
5.6 电热恒温水浴锅。  
5.7 旋涡混合器。  
5.8茄形瓶： $50~\mathrm{mL}$   
5.9 离心管  
5.10 滤膜： $0.45\mu \mathrm{m}$ 。

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或冷藏的空白或供试牛奶，混合均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-20℃以下保存。

# 7 测定步骤

# 7.1 提取

称取试料（ $5 \pm 0.05$ ）g，于离心管中，加碳酸盐缓冲液 $5 \mathrm{~mL}$ ，加乙酸乙酯 $10 \mathrm{~mL}$ ，混匀， $6000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取上清液于茄形瓶中，再加乙酸乙酯 $10 \mathrm{~mL}$ 萃取一次，合并两次上清液，于 $50^{\circ} \mathrm{C}$ 水浴旋转蒸发至干，加碳酸盐缓冲液 $5 \mathrm{~mL}$ 溶解残余物，备用。

# 7.2 净化

$\mathrm{C}_{18}$ 柱依次用水 $3 \mathrm{~mL}$ 、甲醇 $3 \mathrm{~mL}$ 和碳酸盐缓冲液 $3 \mathrm{~mL}$ 活化，取备用液过柱，用水 $3 \mathrm{~mL}$ 淋洗，用甲醇 $5 \mathrm{~mL}$ 洗脱，收集洗脱液，于 $50^{\circ} \mathrm{C}$ 水浴氮气吹干，用流动相 $1.0 \mathrm{~mL}$ 溶解残余物，滤膜过滤，供高效液相色谱测定。

# 7.3 标准曲线的制备

准确量取 $10\mu \mathrm{g} / \mathrm{mL}$ 左旋咪唑标准工作液适量，用流动相稀释，配制成浓度为10、20、50、100、200、400和 $800~\mu \mathrm{g} / \mathrm{L}$ 的系列标准工作液，供高效液相色谱测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.4 测定

# 7.4.1 液相色谱条件

色谱柱： $\mathrm{C}_{18}$ （ $150 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ ，粒径 $5 \mu \mathrm{m}$ ），或相当者；

流动相： $0.02\mathrm{mol} / \mathrm{L}$ 磷酸二氢钠二乙胺缓冲溶液+甲醇(70+30，v/v);

流速： $1\mathrm{mL / min}$

检测波长： $220~\mathrm{nm}$

进样量： $50~\mu \mathrm{L}$

柱温： $30^{\circ}\mathrm{C}$ 。

# 7.4.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试样溶液中左旋咪唑响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和空白添加试样溶液的高效液相色谱图见附录A。

# 7.5 空白试验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算与表述

试料中左旋咪唑类的残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算

$$
X = \frac {A \times C s \times V}{A s \times m}
$$

式中：

$X$ ——供试试料中左旋咪唑残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

A—试样溶液中左旋咪唑的峰面积；

$A_{S}$ ——标准工作液中左旋咪唑的峰面积；

$\mathrm{C}_{\mathrm{S}}$ ——标准工作液中左旋咪唑的浓度, $\mu \mathrm{g} / \mathrm{L}$ ;

$V$ ——溶解残余物所用流动相体积， $\mathrm{mL}$ ；

$m$ —供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度、精密度

# 9.1灵敏度

本方法的检测限为 $2.5\mu \mathrm{g / kg}$ ，定量限为 $5\mu \mathrm{g / kg}$

# 9.2准确度

本方法在 $5\sim 20~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

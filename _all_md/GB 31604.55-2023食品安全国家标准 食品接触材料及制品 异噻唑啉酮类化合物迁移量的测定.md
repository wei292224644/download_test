# 异噻唑啉酮类化合物迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中1,2-苯并异噻唑基-3(2H)-酮、2-甲基-3(2H)-异噻唑啉酮和5-氯-2-甲基-2H-异噻唑-3-酮迁移量的测定方法。

本标准适用于食品接触用塑料、涂料和涂层、橡胶、纸和纸板，以及使用油墨或黏合剂的食品接触材料及制品中1,2-苯并异噻唑基-3(2H)-酮、2-甲基-3(2H)-异噻唑啉酮和5-氯-2-甲基-2H-异噻唑-3-酮迁移量的测定。

# 第一法 高效液相色谱法

# 2 原理

食品接触材料及制品根据GB31604.1及GB5009.156进行迁移试验后，采用高效液相色谱法进行检测。其中水、 $4\%$ （体积分数）乙酸、 $10\%$ （体积分数）乙醇、 $20\%$ （体积分数）乙醇、 $50\%$ （体积分数）乙醇食品模拟物及化学替代溶剂 $95\%$ （体积分数）乙醇浸泡液过滤后直接进样；含油脂食品模拟物和化学替代溶剂异辛烷浸泡液通过甲醇-水溶液萃取后取下层过滤进样。采用紫外检测器或二极管阵列检测器检测，峰面积外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为色谱纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 酸性、含乙醇、含油脂食品模拟物及化学替代溶剂：所用试剂需满足GB5009.156的要求。  
3.1.2 甲醇 $\mathrm{(CH_4O)}$ 。  
3.1.3 四氢呋喃 $\mathrm{(C_4H_8O)}$ 。  
3.1.4 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ 。

# 3.2 试剂配制

3.2.1 酸性、含乙醇食品模拟物及化学替代溶剂：按GB5009.156的规定配制。  
3.2.2 甲醇溶液：分别量取 $50~\mathrm{mL}$ 甲醇和 $50~\mathrm{mL}$ 水，混匀。

# 3.3 标准品

3.3.1 1,2-苯并异噻唑基-3(2H)-酮 $\mathrm{(C_7H_5NOS}$ ,CAS号：2634-33-5）：纯度 $\geqslant 95\%$ ，或经国家认证并授

予标准物质证书的标准品。

3.3.2 2-甲基-3(2H)-异噻唑啉酮 $\mathrm{(C_4H_5NOS, CAS}$ 号：2682-20-4）：纯度 $\geqslant 95\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.3 5-氯-2-甲基-2H-异噻唑-3-酮 $\mathrm{(C_4H_4ClNOS, CAS}$ 号：26172-55-4）：纯度 $\geqslant 95\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 异噻唑啉酮标准储备溶液 $(1000\mathrm{mg / L}$ ，溶剂为甲醇)：分别准确称取3种异噻唑啉酮标准品各 $25\mathrm{mg}$ （精确到 $0.1\mathrm{mg})$ ，用甲醇溶解后转移至同一 $25~\mathrm{mL}$ 棕色容量瓶中，并用甲醇定容至刻度后摇匀。将溶液转移至棕色玻璃瓶中，于 $4^{\circ}C$ 冰箱中避光密闭保存，保存期为6个月。  
3.4.2 异噻唑啉酮标准中间溶液 $\mathrm{A}(100\mathrm{mg / L}$ ，溶剂为甲醇)：分别准确移取异噻唑啉酮标准储备溶液 $1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 棕色容量瓶中，用甲醇定容至刻度后摇匀。将溶液转移至棕色玻璃容器中，于在 $4^{\circ}C$ 冰箱中避光密闭保存，保存期为3个月。  
3.4.3 异噻唑啉酮标准中间溶液 $\mathrm{B}(10\mathrm{mg / L}$ ，溶剂为甲醇)：准确移取 $1\mathrm{mL}$ 异噻唑啉酮标准中间溶液A于 $10~\mathrm{mL}$ 棕色容量瓶中，用甲醇定容至刻度后摇匀。将溶液转移至棕色玻璃容器中，于 $4^{\circ}C$ 冰箱中避光密闭保存，保存期为3个月。  
3.4.4 异噻唑啉酮标准中间溶液 $\mathrm{C}(100\mathrm{mg / L}$ ，溶剂为四氢呋喃)：分别准确移取3种异噻唑啉酮标准储备溶液 $1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 棕色容量瓶中，用四氢呋喃定容至刻度后摇匀。将溶液转移至棕色玻璃容器中，于 $4^{\circ}C$ 冰箱中避光密闭保存，保存期为3个月。  
3.4.5 异噻唑啉酮标准中间溶液 $\mathrm{D}(10\mathrm{mg / L}$ ，溶剂为四氢呋喃)：准确移取 $1\mathrm{mL}$ 异噻唑啉酮标准中间溶液C于 $10~\mathrm{mL}$ 棕色容量瓶中，用四氢呋喃定容至刻度后摇匀。将溶液转移至棕色玻璃容器中，于 $4^{\circ}C$ 冰箱中避光密闭保存，保存期为3个月。

# 3.4.6 标准工作溶液配制如下：

a) 水、酸性、含乙醇食品模拟物的标准工作溶液: 分别准确移取 $30 \mu \mathrm{L}, 50 \mu \mathrm{L}, 100 \mu \mathrm{L}, 300 \mu \mathrm{L}, 500 \mu \mathrm{L}$ 标准中间溶液 A 于 5 个 $10 \mathrm{~mL}$ 容量瓶中, 用水定容, 得到异噻唑啉酮质量浓度分别为 $0.30 \mathrm{mg} / \mathrm{L}, 0.50 \mathrm{mg} / \mathrm{L}, 1.0 \mathrm{mg} / \mathrm{L}, 3.0 \mathrm{mg} / \mathrm{L}, 5.0 \mathrm{mg} / \mathrm{L}$ 的标准工作溶液。采用同样方式, 分别用 $4 \%$ (体积分数)乙酸、 $10 \%$ (体积分数)乙醇、 $20 \%$ (体积分数)乙醇、 $50 \%$ (体积分数)乙醇食品模拟物及化学替代溶剂 $95 \%$ (体积分数)乙醇配制同样浓度系列的异噻唑啉酮标准工作溶液, 用针式尼龙过滤器过滤, 待上机测定。  
b）异辛烷标准工作溶液：分别准确移取 $30~\mu \mathrm{L},50~\mu \mathrm{L},100~\mu \mathrm{L},300~\mu \mathrm{L},500~\mu \mathrm{L}$ 标准中间溶液C于5个 $10~\mathrm{mL}$ 容量瓶中，用异辛烷定容，得到异噻唑啉酮质量浓度分别为 $0.30~\mathrm{mg / L}$ 、 $0.50~\mathrm{mg / L},1.0~\mathrm{mg / L},3.0~\mathrm{mg / L},5.0~\mathrm{mg / L}$ 的标准工作溶液。分别移取 $2\mathrm{mL}$ 该系列标准工作溶液至5支具塞玻璃离心管中，各加入 $2\mathrm{mL}$ 甲醇溶液(3.2.2)，振荡 $1\mathrm{min}$ ，静置分层后，取下层的甲醇溶液，用针式尼龙过滤器过滤，待上机测定。  
c）含油脂食品模拟物标准工作溶液：分别准确称取 $5.00\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）含油脂食品模拟物橄榄油至5个具塞玻璃离心管中，再分别加入 $15\mu \mathrm{L},25\mu \mathrm{L},50\mu \mathrm{L},150\mu \mathrm{L},250\mu \mathrm{L}$ 标准中间溶液C，涡旋振荡混匀，得到异噻唑啉酮质量浓度分别为 $0.30~\mathrm{mg / kg},0.50~\mathrm{mg / kg},1.0~\mathrm{mg / kg},$ $3.0~\mathrm{mg / kg},5.0~\mathrm{mg / kg}$ 的标准工作溶液。在每支玻璃离心管中各加入 $3\mathrm{mL}$ 正己烷，再各加入 $5\mathrm{mL}$ 甲醇溶液(3.2.2)，涡旋振荡 $1\mathrm{min}$ ，静置分层后，取下层的甲醇溶液，用针式尼龙过滤器过滤，待上机测定。

# 4 仪器和设备

4.1 高效液相色谱仪：配紫外检测器或二极管阵列检测器。  
4.2 分析天平：感量分别为 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$   
4.3 微量移液器： $10~\mu \mathrm{L}, 100~\mu \mathrm{L}$ 和 $1000~\mu \mathrm{L}$   
4.4 恒温干燥箱或恒温水浴装置。  
4.5 涡旋振荡器。  
4.6 针式尼龙过滤器：滤膜孔径 $0.22\mu \mathrm{m}$

# 5 分析步骤

# 5.1 迁移试验

食品接触材料及制品按照GB31604.1和GB5009.156的规定进行迁移试验。迁移试验所得浸泡液若不能立即测试，应置于 $4^{\circ}\mathrm{C}$ 下避光保存不超过 $72\mathrm{h}$ 。若进行下一步试验，应将浸泡液恢复至室温后再使用。

# 5.2 试样溶液和空白试液的制备

# 5.2.1 水性、酸性、含乙醇食品模拟物试样溶液

取迁移试验所得浸泡液 $1\mathrm{mL}\sim 2\mathrm{mL}$ ，用针式尼龙过滤器过滤后，待上机测定。

# 5.2.2 化学替代溶剂异辛烷试样溶液

移取 $2\mathrm{mL}$ 迁移试验后得到的异辛烷浸泡液于具塞玻璃离心管中，加入 $2\mathrm{mL}$ 甲醇溶液(3.2.2)，涡旋振荡 $1\mathrm{min}$ ，静置分层后，取下层的甲醇溶液，用针式尼龙过滤器过滤后，待上机测定。

# 5.2.3 含油脂食品模拟物试样溶液

称取 $5.00\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )迁移试验后得到的橄榄油浸泡液至具塞玻璃离心管中，加入 $3\mathrm{mL}$ 正己烷，再加入 $5\mathrm{mL}$ 甲醇溶液(3.2.2)，涡旋振荡 $1\mathrm{min}$ ，静置分层后，取下层的甲醇溶液，用针式尼龙过滤器过滤后，待上机测定。

# 5.2.4 空白试液的制备

取未与食品接触材料及制品接触的食品模拟物和化学替代溶剂，分别按5.1及 $5.2.1\sim 5.2.3$ 所述步骤处理，制得空白试液，待上机测定。

# 5.3 仪器参考条件

5.3.1 色谱柱：可耐受高水相的 $\mathrm{C_{18}}$ 柱，柱长 $150\mathrm{mm}\times$ 柱内径 $3.0\mathrm{mm}\times$ 固定相粒径 $3.5\mu \mathrm{m}$ ，或同等性能色谱柱。  
5.3.2 流动相：A为水，B为甲醇， $\mathrm{A + B = 8 + 2}$   
5.3.3 流速： $1\mathrm{mL} / \mathrm{min}$   
5.3.4 柱温： $40^{\circ}C$   
5.3.5 进样量： $5\mu \mathrm{L}$   
5.3.6 检测波长：2-甲基-3(2H)-异噻唑啉酮和5-氯-2-甲基-2H-异噻唑-3-酮为 $275~\mathrm{nm},1,2$ -苯并异噻

唑基-3(2H)-酮为 $224~\mathrm{nm}$

# 5.4 标准工作曲线绘制

按照5.3所列仪器参考条件，将标准系列工作液按浓度从低到高的顺序分别注入高效液相色谱仪中，测定各待测目标物相应的峰面积，以异噻唑啉酮的浓度为横坐标、对应的峰面积为纵坐标，绘制标准工作曲线，得到线性回归方程。标准工作溶液的色谱图参见附录A。

# 5.5 试样溶液的测定

# 5.5.1 定性测定

按照5.3所列仪器参考条件，分别测定试样溶液和标准工作溶液，如果试样溶液与标准溶液中相对应的色谱峰保留时间偏差在 $\pm 2.5\%$ 范围以内，则可判断样品中存在相应的待测物。

# 5.5.2 定量测定

按照5.3所列仪器参考条件，分别将试样溶液和空白试液分别注入高效液相色谱仪中，得到各待测目标物的色谱峰峰面积，根据标准工作曲线计算得到试样溶液和空白试液中异噻唑啉酮的含量。

# 6 分析结果的表述

# 6.1 非密封制品类食品接触材料及制品中异噻唑啉酮特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

对于除了盖子、密封圈、连接件等密封制品（以下简称密封制品）以外的食品接触材料及制品，异噻唑啉酮特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(1)进行计算。

$$
X _ {1} = \frac {\left(c - c _ {0}\right) \times V}{S} \times \frac {S _ {0}}{m _ {1}} \tag {1}
$$

式中：

$X_{1}$ ——异噻唑啉酮的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样溶液中异噻唑啉酮的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白试液中异噻唑啉酮的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

S——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$S_{0}$ ——样品实际使用中与食品接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$m_{1}$ ——样品实际接触固态食品的质量，或实际接触液态食品的体积所对应的食品质量，单位为千克 $(\mathrm{kg})$ ；各种液态食品按密度为 $1\mathrm{kg} / \mathrm{L}$ 将其体积换算为相应的质量。

结果至少保留2位有效数字。

# 6.2 密封制品类食品接触材料及制品中异噻唑啉酮特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

当预期用途已知时，对于密封制品类食品接触材料及制品，异噻唑啉酮特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(2)进行计算。

$$
X _ {2} = \frac {\left(c - c _ {0}\right) \times V}{S} \times \frac {S _ {0}}{m _ {2}} \tag {2}
$$

式中：

$X_{2}$ ——异噻唑啉酮的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样溶液中异噻唑啉酮的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白试液中异噻唑啉酮的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

$S$ ——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$S_{0}$ ——密封制品实际使用中与食品接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$m_{2}$ ——密封制品实际使用容器盛装固态食品的质量，或实际接触液态食品的体积所对应的食品质量，单位为千克 $(\mathrm{kg})$ ；各种液态食品按密度为 $1\mathrm{kg} / \mathrm{L}$ 将其体积换算为相应的质量。

结果至少保留2位有效数字。

# 6.3 密封制品类食品接触材料及制品中异噻唑啉酮特定迁移量的计算（以 $\mathrm{mg}$ /件表示）

当预期用途未知时，密封制品类食品接触材料及制品异噻唑啉酮特定迁移量以 $\mathrm{mg}/$ 件表示时，按式(3)进行计算，此时还需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {3} = \frac {\left(c - c _ {0}\right) \times V}{n} \tag {3}
$$

式中：

$X_{3}$ ——异噻唑啉酮特定迁移量，单位为毫克每件 $(\mathrm{mg}/$ 件）；

$c$ ——试样溶液中异噻唑啉酮的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

$c_{0}$ ——空白试液中异噻唑啉酮的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$n$ ——浸泡时所用密封制品的数量，单位为件。

结果至少保留2位有效数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

当迁移试验中的 $\mathrm{S} / \mathrm{V}$ 与样品在实际使用情形下的 $\mathrm{S} / \mathrm{V}$ 相同时，本方法对食品模拟物、化学替代溶剂中单种异噻唑啉酮迁移量的检出限为 $0.10\mathrm{mg / kg}$ ，定量限为 $0.30\mathrm{mg / kg}$ 。当迁移试验中 $\mathrm{S} / \mathrm{V}$ 与样品在实际使用情形下的 $\mathrm{S} / \mathrm{V}$ 不同时，本方法对食品模拟物、化学替代溶剂中单种异噻唑啉酮迁移量的检出限和定量限需按第6章中的计算公式进行换算。

# 第二法 液相色谱-串联质谱法

# 9 原理

食品接触材料及制品根据GB31604.1及GB5009.156进行迁移试验后，采用液相色谱-串联质谱法进行检测。其中水、 $4\%$ （体积分数)乙酸、 $10\%$ （体积分数)乙醇、 $20\%$ （体积分数)乙醇、 $50\%$ （体积分数)乙醇食品模拟物及化学替代溶剂 $95\%$ （体积分数)乙醇浸泡液过滤后直接进样；含油脂食品模拟物和化学替代溶剂异辛烷浸泡液通过甲醇-水溶液萃取后取下层过滤进样。选择反应监测模式进行检测，峰面积外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为色谱纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 10.1 试剂

10.1.1 酸性、含乙醇、含油脂食品模拟物及化学替代溶剂：所用试剂需满足GB5009.156的要求。  
10.1.2 甲醇 $(\mathrm{CH}_4\mathrm{O})$ 。

# 前言

本标准代替GB4789.35—2010《食品安全国家标准 食品微生物学检验 乳酸菌检验》、SN/T1941.1—2007《进出口食品中乳酸菌检验方法 第1部分：分离与计数方法》。

本标准与GB4789.35—2010相比，主要变化如下：

——增加了乳酸菌总数计数培养条件的选择及结果说明；  
——修改了改良MRS培养基成分；  
——修改了平板计数的接种方法和接种量。

# 食品安全国家标准

# 食品微生物学检验 乳酸菌检验

# 1 范围

本标准规定了含乳酸菌食品中乳酸菌(lactic acid bacteria)的检验方法。

本标准适用于含活性乳酸菌的食品中乳酸菌的检验。

# 2 术语和定义

# 2.1

# 乳酸菌 lactic acid bacteria

一类可发酵糖主要产生大量乳酸的细菌的通称。本标准中乳酸菌主要为乳杆菌属(Lactobacillus)、双歧杆菌属(Bifidobacterium)和嗜热链球菌属(Streptococcus)。

# 3 设备和材料

除微生物实验室常规灭菌及培养设备外，其他设备和材料如下：

3.1 恒温培养箱： $36\mathrm{^\circ C}\pm 1\mathrm{^\circ C}$   
3.2 冰箱： $2^{\circ}C\sim 5^{\circ}C$   
3.3 均质器及无菌均质袋、均质杯或灭菌乳钵。  
3.4 天平：感量 $0.01\mathrm{g}$ 。  
3.5 无菌试管： $18\mathrm{mm} \times 180\mathrm{mm}, 15\mathrm{mm} \times 100\mathrm{mm}$ 。  
3.6 无菌吸管： $1\mathrm{mL}$ （具 $0.01\mathrm{mL}$ 刻度）、 $10\mathrm{mL}$ （具 $0.1\mathrm{mL}$ 刻度）或微量移液器及吸头。  
3.7 无菌锥形瓶： $500~\mathrm{mL},250~\mathrm{mL}$ 。

4.1 生理盐水：见A.1。

4.3 MC培养基(ModifiedChalmers培养基)：见A.4。  
4.4 $0.5\%$ 蔗糖发酵管：见A.5。  
4.4 $0.5\%$ 纤维二糖发酵管：见A.5。  
4.6 $0.5\%$ 麦芽糖发酵管：见A.5。  
4.7 $0.5\%$ 甘露醇发酵管：见A.5。  
4.8 $0.5\%$ 水杨苷发酵管：见A.5。  
4.9 $0.5\%$ 山梨醇发酵管：见A.5。  
4.10 $0.5\%$ 乳糖发酵管：见A.5。

4.11 七叶苷发酵管：见A.6。

# 6 操作步骤

# 6.1 样品制备

6.1.1 样品的全部制备过程均应遵循无菌操作程序。  
6.1.2 冷冻样品可先使其在 $2^{\circ}\mathrm{C} \sim 5^{\circ}\mathrm{C}$ 条件下解冻，时间不超过 $18\mathrm{h}$ ，也可在温度不超过 $45^{\circ}\mathrm{C}$ 的条件解冻，时间不超过 $15\mathrm{min}$ 。  
6.1.3 固体和半固体食品：以无菌操作称取 $25\mathrm{g}$ 样品，置于装有 $225~\mathrm{mL}$ 生理盐水的无菌均质杯内，于 $8000\mathrm{r / min}\sim 10000\mathrm{r / min}$ 均质 $1\mathrm{min}\sim 2\mathrm{min}$ ，制成 $1:10$ 样品匀液；或置于 $225~\mathrm{mL}$ 生理盐水的无菌均质袋中，用拍击式均质器拍打 $1\mathrm{min}\sim 2\mathrm{min}$ 制成 $1:10$ 的样品匀液。  
6.1.4 液体样品：液体样品应先将其充分摇匀后以无菌吸管吸取样品 $25\mathrm{mL}$ 放入装有 $225\mathrm{mL}$ 生理盐水的无菌锥形瓶(瓶内预置适当数量的无菌玻璃珠)中，充分振摇，制成 $1:10$ 的样品匀液。

# 6.2 步骤

6.2.1 用 $1 \mathrm{~mL}$ 无菌吸管或微量移液器吸取 $1:10$ 样品匀液 $1 \mathrm{~mL}$ , 沿管壁缓慢注于装有 $9 \mathrm{~mL}$ 生理盐水的无菌试管中 (注意吸管尖端不要触及稀释液), 振摇试管或换用 1 支无菌吸管反复吹打使其混合均匀, 制成 $1:100$ 的样品匀液。

6.2.2 另取 $1\mathrm{mL}$ 无菌吸管或微量移液器吸头，按上述操作顺序，做10倍递增样品匀液，每递增稀释一次，即换用1次 $1\mathrm{mL}$ 灭菌吸管或吸头。

# 6.2.3.2 双歧杆菌计数

根据对待检样品双歧杆菌含量的估计，选择2个～3个连续的适宜稀释度，每个稀释度吸取 $1\mathrm{mL}$ 样品匀液于灭菌平皿内，每个稀释度做两个平皿。稀释液移入平皿后，将冷却至 $48^{\circ}\mathrm{C}$ 的莫匹罗星锂盐和半胱氨酸盐酸盐改良的MRS培养基倾注入平皿约 $15\mathrm{mL}$ ，转动平皿使混合均匀。 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 厌氧培养 $72\mathrm{h} \pm 2\mathrm{h}$ ，培养后计数平板上的所有菌落数。从样品稀释到平板倾注要求在 $15\mathrm{min}$ 内完成。

# 6.2.3.3 嗜热链球菌计数

根据待检样品嗜热链球菌活菌数的估计，选择2个～3个连续的适宜稀释度，每个稀释度吸取 $1\mathrm{mL}$ 样品匀液于灭菌平皿内，每个稀释度做两个平皿。稀释液移入平皿后，将冷却至 $48^{\circ}\mathrm{C}$ 的MC培养基倾注入平皿约 $15~\mathrm{mL}$ ，转动平皿使混合均匀。 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 需氧培养 $72\mathrm{h} \pm 2\mathrm{h}$ ，培养后计数。嗜热链球菌在MC琼脂平板上的菌落特征为：菌落中等偏小，边缘整齐光滑的红色菌落，直径 $2\mathrm{mm} \pm 1\mathrm{mm}$ ，菌落背面为粉红色。从样品稀释到平板倾注要求在 $15\mathrm{min}$ 内完成。

# 6.2.3.4 乳杆菌计数

根据待检样品活菌总数的估计，选择2个～3个连续的适宜稀释度，每个稀释度吸取 $1\mathrm{mL}$ 样品匀液于灭菌平皿内，每个稀释度做两个平皿。稀释液移入平皿后，将冷却至 $48^{\circ}\mathrm{C}$ 的MRS琼脂培养基倾注入平皿约 $15\mathrm{mL}$ ，转动平皿使混合均匀。 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 厌氧培养 $72\mathrm{h} \pm 2\mathrm{h}$ 。从样品稀释到平板倾注要求在 $15\mathrm{min}$ 内完成。

# 6.3 菌落计数

注：可用肉眼观察，必要时用放大镜或菌落计数器，记录稀释倍数和相应的菌落数量。菌落计数以菌落形成单位（colony-forming units, CFU)表示。

6.3.1 选取菌落数在 $30\mathrm{CFU} \sim 300\mathrm{CFU}$ 之间、无蔓延菌落生长的平板计数菌落总数。低于 $30\mathrm{CFU}$ 的平板记录具体菌落数，大于 $300\mathrm{CFU}$ 的可记录为多不可计。每个稀释度的菌落数应采用两个平板的平均数。  
6.3.2 其中一个平板有较大片状菌落生长时，则不宜采用，而应以无片状菌落生长的平板作为该稀释度的菌落数；若片状菌落不到平板的一半，而其余一半中菌落分布又很均匀，即可计算半个平板后乘以2，代表一个平板菌落数。  
6.3.3 当平板上出现菌落间无明显界线的链状生长时，则将每条单链作为一个菌落计数。

# 6.4 结果的表述

6.4.1 若只有一个稀释度平板上的菌落数在适宜计数范围内，计算两个平板菌落数的平均值，再将平均值乘以相应稀释倍数，作为每克或每毫升中菌落总数结果。  
6.4.2 若有两个连续稀释度的平板菌落数在适宜计数范围内时，按式(1)计算：

$$
N = \frac {\sum C}{\left(n _ {1} + 0 . 1 n _ {2}\right) d} \quad \dots \tag {1}
$$

式中：

$N$ ——样品中菌落数；

$\sum C$ ——平板（含适宜范围菌落数的平板）菌落数之和；

$n_1$ ——第一稀释度(低稀释倍数)平板个数；

$n_2$ ——第二稀释度(高稀释倍数)平板个数；

$d$ ——稀释因子(第一稀释度)。

6.4.3 若所有稀释度的平板上菌落数均大于300CFU，则对稀释度最高的平板进行计数，其他平板可记录为多不可计，结果按平均菌落数乘以最高稀释倍数计算。  
6.4.4 若所有稀释度的平板菌落数均小于30CFU，则应按稀释度最低的平均菌落数乘以稀释倍数计算。  
6.4.5 若所有稀释度(包括液体样品原液)平板均无菌落生长，则以小于1乘以最低稀释倍数计算。  
6.4.6 若所有稀释度的平板菌落数均不在 $30\mathrm{CFU}\sim 300$ CFU之间，其中一部分小于30CFU或大于 $300\mathrm{CFU}$ 时，则以最接近30CFU或300CFU的平均菌落数乘以稀释倍数计算。

# 6.5 菌落数的报告

6.5.1 菌落数小于100CFU时，按“四舍五入”原则修约，以整数报告。  
6.5.2 菌落数大于或等于100CFU时，第3位数字采用“四舍五入”原则修约后，取前2位数字，后面用0代替位数；也可用10的指数形式来表示，按“四舍五入”原则修约后，采用两位有效数字。  
6.5.3 称重取样以 $\mathrm{CFU / g}$ 为单位报告，体积取样以 $\mathrm{CFU / mL}$ 为单位报告。

# 7 结果与报告

根据菌落计数结果出具报告，报告单位以 $\mathrm{CFU / g(mL)}$ 表示。

# A.7 革兰氏染色液

# A.7.1 结晶紫染色液

# A.7.1.1 成分

结晶紫 $1.0\mathrm{g}$

$95\%$ 乙醇 $20~\mathrm{mL}$

$1\%$ 草酸铵水溶液 $80~\mathrm{mL}$

# A.7.1.2 制法

将结晶紫完全溶解于乙醇中，然后与草酸铵溶液混合。

# A.7.2 革兰氏碘液

# A.7.2.1 成分

碘 $1.0\mathrm{g}$

碘化钾 $2.0\mathrm{g}$

蒸馏水 $300~\mathrm{mL}$

# A.7.2.2 制法

将碘与碘化钾先进行混合，加入蒸馏水少许充分振摇，待完全溶解后，再加蒸馏水至 $300~\mathrm{mL}$ 。

# A.7.3 沙黄复染液

# A.7.3.1 成分

沙黄 $0.25\mathrm{g}$

$95\%$ 乙醇 $10~\mathrm{mL}$

蒸馏水 $90~\mathrm{mL}$

# A.7.3.2 制法

将沙黄溶解于乙醇中，然后用蒸馏水稀释。

# A.7.4 染色法

A.7.4.1 将涂片在酒精灯火焰上固定，滴加结晶紫染色液，染 $1\mathrm{min}$ ，水洗。  
A.7.4.2 滴加革兰氏碘液，作用 $1\mathrm{min}$ ，水洗。  
A.7.4.3 滴加 $95\%$ 乙醇脱色，约 $15 \mathrm{~s} \sim 30 \mathrm{~s}$ ，直至染色液被洗掉，不要过分脱色，水洗。  
A.7.4.4 滴加复染液，复染 $1\mathrm{min}$ 。水洗、待干、镜检。

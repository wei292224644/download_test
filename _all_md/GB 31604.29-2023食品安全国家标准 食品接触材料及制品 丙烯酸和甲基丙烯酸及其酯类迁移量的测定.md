2023-09-06 发布

2024-03-06 实施

# 前言

本标准代替GB31604.29—2016《食品安全国家标准 食品接触材料及制品 甲基丙烯酸甲酯迁移量的测定》。

本标准与GB31604.29—2016相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 丙烯酸和甲基丙烯酸及其酯类迁移量的测定”；  
——修改了范围；  
——修改了“气相色谱-质谱法”的部分技术参数；  
——增加了“第二部分 液相色谱法”。

# 甲基丙烯酸及其酯类迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中丙烯酸和甲基丙烯酸及其酯类迁移量的测定方法。

第一部分气相色谱-质谱法适用于食品接触材料及制品中18种丙烯酸酯和甲基丙烯酸酯类迁移量的测定，包括2-丙烯酸-2-甲基丙基酯、丙烯酸乙酯、丙烯酸正丁酯、丙烯酸异丙酯、丙烯酸丙酯、2-丙烯酸-1,1-二甲基乙基酯、丙烯酸苯甲酯、2-丙烯酸辛酯、丙烯酸仲丁酯、甲基丙烯酸乙酯、甲基丙烯酸异丁酯、甲基丙烯酸丁酯、2-甲基-2-丙烯酸-1,1-二甲基乙基酯、甲基丙烯酸苯酯、2-甲基-2-丙烯酸丙基酯、甲基丙烯酸苯甲酯、甲基丙烯酸仲丁酯、甲基丙烯酸异丙酯。

第二部分液相色谱法适用于食品接触材料及制品中6种丙烯酸和甲基丙烯酸及其酯类迁移量的测定，包括丙烯酸、甲基丙烯酸、丙烯酸-2-羟乙基酯、2-羟基乙基-2-甲基-2-丙烯酸酯、丙烯酸甲酯、甲基丙烯酸甲酯。

# 第一部分 气相色谱-质谱法

# 2 原理

迁移试验所得水基食品模拟物、油脂类食品模拟物采用顶空进样， $95\%$ （体积分数）乙醇、异辛烷模拟物经乙酸乙酯稀释后直接进样，目标物质在气相色谱毛细管柱中分离，用质谱法检测，采用外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.2 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。  
3.1.3 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.4 无水乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
3.1.5 无水乙醇 $\mathrm{(C_2H_6O)}$ 。  
3.1.6 $95\%$ 乙醇 $\mathrm{(C_2H_6O)}$ 。  
3.1.7 异辛烷 $\mathrm{(C_8H_{18})}$

# 3.2 试剂配制

3.2.1 食品模拟物：按照GB5009.156的规定配制。

3.2.2 异辛烷-乙酸乙酯溶液 $(1 + 4)$ ：将 $25~\mathrm{mL}$ 异辛烷和 $100~\mathrm{mL}$ 乙酸乙酯混合均匀。  
3.2.3 $95\%$ 乙醇-乙酸乙酯溶液 $(1 + 4)$ ：将 $25~\mathrm{mL}95\%$ 乙醇和 $100~\mathrm{mL}$ 乙酸乙酯混合均匀。

# 3.3 标准品

18种丙烯酸酯和甲基丙烯酸酯类标准品（见附录A)：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 标准储备溶液 $(1000\mathrm{mg / L})$ ：准确称取18种标准品(精确至 $0.1\mathrm{mg}$ )，称量质量计算方式为 $25\mathrm{mg}$ 除以折算系数，折算系数见附录A，分别用乙酸乙酯溶解后转移至 $25~\mathrm{mL}$ 容量瓶中，并用乙酸乙酯定容至刻度，混匀。将溶液转移至棕色玻璃容器中，于 $4^{\circ}C$ 下避光保存，保存期6个月。  
3.4.2 混合标准中间液 $\mathrm{A}(50\mathrm{mg / L})$ : 分别准确吸取 $0.5\mathrm{mL}$ 标准储备溶液 $(1000\mathrm{mg / L})$ 于 $10\mathrm{mL}$ 容量瓶中, 加甲醇定容至刻度, 混匀。将溶液转移至棕色玻璃容器中, 于 $4^{\circ}\mathrm{C}$ 下避光保存, 保存期6个月。  
3.4.3 混合标准中间液 $\mathrm{B}(50\mathrm{mg / L})$ ：分别准确吸取 $0.5\mathrm{mL}$ 标准储备溶液 $(1000\mathrm{mg / L})$ 于 $10\mathrm{mL}$ 容量瓶中，加乙酸乙酯定容至刻度，混匀。将溶液转移至棕色玻璃容器中，于 $4^{\circ}C$ 下避光保存，保存期6个月。  
3.4.4 水基食品模拟物系列标准工作溶液：分别准确吸取 $100\mu \mathrm{L}, 150\mu \mathrm{L}, 200\mu \mathrm{L}, 400\mu \mathrm{L}, 1000\mu \mathrm{L}$ 混合标准中间液 $\mathrm{A}(50\mathrm{mg / L})$ 于 $100~\mathrm{mL}$ 容量瓶中，用 $50\%$ （体积分数）乙醇溶液定容，所得标准工作溶液中各目标物质浓度分别为 $0.050\mathrm{mg / L}, 0.075\mathrm{mg / L}, 0.10\mathrm{mg / L}, 0.20\mathrm{mg / L}, 0.50\mathrm{mg / L}$ 。分别移取 $10.0~\mathrm{mL}$ 各浓度溶液于已加入 $3.0\mathrm{g}$ 氯化钠的 $20~\mathrm{mL}$ 玻璃顶空瓶中，立即用隔垫和铝盖密封，待测。采用相同方式，分别用水、 $4\%$ （体积分数）乙酸、 $10\%$ （体积分数）乙醇、 $20\%$ （体积分数）乙醇配制相同浓度系列的标准工作溶液。临用现配。  
3.4.5 油脂类食品模拟物系列标准工作溶液: 分别准确称取 $5.00 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ ) 橄榄油模拟物于 5 个 $20 \mathrm{~mL}$ 玻璃顶空瓶中, 用移液器分别移取 $30 \mu \mathrm{L}, 80 \mu \mathrm{L}, 100 \mu \mathrm{L}, 150 \mu \mathrm{L}, 300 \mu \mathrm{L}$ 混合标准中间液 $\mathrm{B}(50 \mathrm{mg} / \mathrm{L})$ 至 $20 \mathrm{~mL}$ 玻璃顶空瓶中, 立即用隔热和铝盖密封, 涡旋均匀后待测。所得标准工作溶液中各目标物质浓度分别为 $0.30 \mathrm{mg} / \mathrm{kg}, 0.80 \mathrm{mg} / \mathrm{kg}, 1.0 \mathrm{mg} / \mathrm{kg}, 1.5 \mathrm{mg} / \mathrm{kg}, 3.0 \mathrm{mg} / \mathrm{kg}$ 。临用现配。  
3.4.6 异辛烷系列标准工作溶液：分别移取 $50~\mu \mathrm{L},150~\mu \mathrm{L},200~\mu \mathrm{L},300~\mu \mathrm{L},600~\mu \mathrm{L}$ 混合标准中间液B(50mg/L)于 $10~\mathrm{mL}$ 容量瓶中，用异辛烷-乙酸乙酯溶液(1+4)定容至刻度，混匀。所得标准工作溶液中各目标物质浓度分别为 $0.25\mathrm{mg / L},0.75\mathrm{mg / L},1.0\mathrm{mg / L},1.5\mathrm{mg / L},3.0\mathrm{mg / L}$ 。临用现配。  
3.4.7 $95\%$ (体积分数)乙醇系列标准工作溶液：分别移取 $50~\mu \mathrm{L},150~\mu \mathrm{L},200~\mu \mathrm{L},300~\mu \mathrm{L},600~\mu \mathrm{L}$ 混合标准中间液 $\mathrm{B}(50\mathrm{mg / L})$ 于 $10~\mathrm{mL}$ 容量瓶中，用 $95\%$ 乙醇-乙酸乙酯溶液(1+4)定容至刻度。所得标准工作溶液中各目标物质浓度分别为 $0.25\mathrm{mg / L},0.75\mathrm{mg / L},1.0\mathrm{mg / L},1.5\mathrm{mg / L},3.0\mathrm{mg / L}$ 。临用现配。

# 4 仪器和设备

4.1 气相色谱-质谱联用仪，配有电子轰击源(EI)和顶空进样器。  
4.2 涡旋振荡器。  
4.3 电子天平：感量分别为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。

# 5 分析步骤

# 5.1 迁移试验

按照GB5009.156和GB31604.1的要求对食品接触材料及制品进行迁移试验。迁移试验所得浸泡液如不能立即测定，应置于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱中避光保存，保存期不超过一周。若进行下一步试验，应将浸泡液恢复至室温。

# 5.2 试液的制备

# 5.2.1 水基食品模拟物试液的制备

移取 $10.0\mathrm{mL}$ 迁移试验所得浸泡液于已加入 $3.0\mathrm{g}$ 氯化钠的 $20~\mathrm{mL}$ 顶空瓶中，立即用隔垫和铝盖密封，待测。

# 5.2.2 油脂类食品模拟物试液的制备

称取 $5.00\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )迁移试验所得浸泡液于 $20~\mathrm{mL}$ 顶空瓶中，立即用隔垫和铝盖密封，待测。

# 5.2.3 异辛烷试液和 $95\%$ （体积分数）乙醇试液的制备

移取 $2.00\mathrm{mL}$ 迁移试验所得浸泡液于 $10~\mathrm{mL}$ 容量瓶中，用乙酸乙酯定容，摇匀后，经微孔滤膜过滤后，待测。

# 5.3 空白试液的制备

按照5.1和5.2所述步骤处理未与食品接触材料接触的食品模拟物、异辛烷和 $95\%$ （体积分数）乙醇试液。

# 5.4 仪器参考条件

5.4.1 顶空进样器条件如下：

a）气液平衡时间： $20\mathrm{min}$   
b）顶空温度：水基食品模拟物试液设置为 $70^{\circ}\mathrm{C}$ ，油脂类食品模拟物设置为 $120^{\circ}\mathrm{C}$ 。  
c）进样针温度： $70^{\circ}\mathrm{C}$ 。  
d）振荡速度： $500\mathrm{r / min}$   
e) 进样量: $250 \mu \mathrm{L}$ 。

5.4.2 气相色谱质谱测定参考条件如下：

a）色谱柱：聚乙二醇毛细管柱：柱长 $30.0\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ，或性能等效的分析柱。  
b）程序升温：初始温度 $35^{\circ}\mathrm{C}$ ，保持 $8\mathrm{min}$ ； $5^{\circ}\mathrm{C} / \mathrm{min}$ 升至 $60^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ ； $20^{\circ}\mathrm{C} / \mathrm{min}$ 升至 $160^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$ ； $40^{\circ}\mathrm{C} / \mathrm{min}$ 升至 $220^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ 。  
c）进样口温度： $290~\mathrm{^\circ C}$ 。  
d）质谱接口温度： $280^{\circ}C$ 。  
e) 离子源温度: $230^{\circ} \mathrm{C}$ 。  
f) 载气： $\mathrm{He}$ （纯度 $>99.999\%$ ）， $1.0~\mathrm{mL / min}$ 恒流。  
g）进样量： $1\mu \mathrm{L}$   
h）进样方式：分流进样，水基食品模拟物试液分流比为 $20:1$ ，油脂类食品模拟物试液分流比为

$50:1$ ，异辛烷试液和 $95\%$ （体积分数)乙醇试液分流比为 $2:1$ 。

i） 电离方式：EI。  
j）溶剂延迟： $5.0\mathrm{min}$   
k）质谱扫描方式：选择离子模式(SIM)。  
1) SIM采集参数：见附录B。

# 5.5 标准曲线的制作

按照5.4.1和5.4.2所列的仪器参考条件，将系列标准工作溶液分别注入气相色谱-串联质谱仪中，测定相应的峰面积。以标准系列工作溶液中目标物质浓度为横坐标，以对应的定量离子峰面积为纵坐标，绘制标准工作曲线。标准工作液的典型总离子流色谱图见附录C。

# 5.7 定量测定

按照5.4.1和5.4.2所列的仪器参考条件，对试样试液(5.2)和空白试液(5.3)进行测定，得到目标物质峰定量离子峰面积，根据标准曲线计算试样试液、空白试液中目标物质的浓度 $c_{\setminus}c_{0}$ 。

# 6 分析结果的表述

# 6.1 非密封制品类食品接触材料及制品丙烯酸和甲基丙烯酸及其酯类特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

对于除了盖子、密封圈、连接件等密封制品(以下简称密封制品)以外的食品接触材料及制品，丙烯酸和甲基丙烯酸及其酯类特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(1)进行计算。

$$
X _ {1} = \frac {\left(c - c _ {0}\right) \times V}{S} \times F \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——目标物质(以丙烯酸/甲基丙烯酸计)的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样浸泡液中目标物质(以丙烯酸/甲基丙烯酸计)的含量,单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白浸泡液中目标物质(以丙烯酸/甲基丙烯酸计)的含量, 单位为毫克每升 $(\mathrm{mg} / \mathrm{L})$ 或毫克每千克 $(\mathrm{mg} / \mathrm{kg})$ ；

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

$S$ ——迁移试验中试样与浸泡液的接触面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$F$ ——在实际使用情形下，食品接触材料及制品的接触面积(S)与食品或食品模拟物体积(V)的

比 $(S / V)$ , 单位为平方分米每千克 $(\mathrm{dm}^2 / \mathrm{kg})$ 。在实际使用情形下, 当 $S / V$ 已知时, $F$ 即为实际 $S / V$ ; 当 $S / V$ 无法估算时, $F$ 采用 $6 \mathrm{~dm}^2 / \mathrm{kg}$ , 即 $6 \mathrm{~dm}^2$ 食品接触材料及制品接触 $1 \mathrm{~kg}$ 食品或食品模拟物, 各种液态食品密度通常以 $1 \mathrm{~kg} / \mathrm{L}$ 计。

以重复性条件下获得的两次独立测定结果的算术平均值表示，计算结果保留2位有效数字。

# 6.2 密封制品类食品接触材料及制品丙烯酸和甲基丙烯酸及其酯类特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

当预期用途已知时，对于密封制品类食品接触材料及制品，丙烯酸和甲基丙烯酸及其酯类特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(2)进行计算。

$$
X _ {2} = \frac {\left(c - c _ {0}\right) \times V}{S} \times \frac {S _ {0}}{m _ {2}} \tag {2}
$$

式中：

$X_{2}$ ——目标物质(以丙烯酸/甲基丙烯酸计)的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样浸泡液中目标物质(以丙烯酸/甲基丙烯酸计)的含量, 单位为毫克每升 $(\mathrm{mg} / \mathrm{L})$ 或毫克每千克 $(\mathrm{mg} / \mathrm{kg})$ ；

$c_{0}$ ——空白浸泡液中目标物质(以丙烯酸/甲基丙烯酸计)的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

S——迁移试验中试样与浸泡液的接触面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$S_{0}$ ——密封制品实际使用中与食品接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$m_{2}$ ——密封制品实际使用容器盛装固态食品的质量,或实际接触液态食品的体积所对应的食品质量,单位为千克 $(\mathrm{kg})$ ;各种液态食品按密度为 $1 \mathrm{~kg} / \mathrm{L}$ 将其体积换算为相应的质量。

以重复性条件下获得的两次独立测定结果的算术平均值表示，计算结果保留2位有效数字。

# 6.3 密封制品类食品接触材料及制品丙烯酸和甲基丙烯酸及其酯类特定迁移量的计算（以 $\mathrm{mg}$ /件表示）

当预期用途未知时，密封制品类食品接触材料及制品，丙烯酸和甲基丙烯酸及其酯类特定迁移量以 $\mathrm{mg}/$ 件表示时，按式(3)计算，需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {3} = \frac {\left(c - c _ {0}\right) \times V}{n} \tag {3}
$$

式中：

$X_{3}$ ——目标物质(以丙烯酸/甲基丙烯酸计)的特定迁移量，单位为毫克每件 $(\mathrm{mg}/$ 件）；

$c$ ——试样浸泡液中目标物质(以丙烯酸/甲基丙烯酸计)的含量, 单位为毫克每升 $(\mathrm{mg} / \mathrm{L})$ 或毫克每千克 $(\mathrm{mg} / \mathrm{kg})$ ；

$c_{0}$ ——空白浸泡液中目标物质(以丙烯酸/甲基丙烯酸计)的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

$n$ ——浸泡用密封制品的数量，单位为件。

以重复性条件下获得的两次独立测定结果的算术平均值表示，计算结果保留2位有效数字。

# 7 精密度

在重复性条件下获得两次独立测定结果的绝对差值不得超过其算术平均值的 $10\%$ 。

# 8 其他

# 第二部分 液相色谱法

# 9 原理

迁移试验所得水基食品模拟物、 $95\%$ （体积分数）乙醇模拟物过滤后直接进样，油脂类食品模拟物、异辛烷模拟物经甲醇萃取后进样，目标物质在液相色谱柱中分离，用紫外检测器或者二极管阵列检测器检测，采用外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
10.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
10.1.3 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ ：优级纯。  
10.1.4 无水乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
10.1.5 无水乙醇 $(\mathrm{C}_2\mathrm{H}_6\mathrm{O})$ 。  
10.1.6 $95\%$ 乙醇 $\mathrm{(C_2H_6O)}$ 。  
10.1.7 异辛烷 $(\mathrm{C_8H_{18}})$ 。

# 10.2 试剂配制

10.2.1 食品模拟物：按照GB5009.156的规定配制。   
10.2.2 $80\%$ 甲醇溶液：量取 $80~\mathrm{mL}$ 甲醇与 $20~\mathrm{mL}$ 水，混匀。  
10.2.3 $0.1\%$ 磷酸溶液：称取 $1.00\mathrm{g}$ 磷酸至 $1\mathrm{L}$ 容量瓶中，用水定容至刻度，混匀。

# 10.3 标准品

6种丙烯酸和甲基丙烯酸及其酯类标准品(见附录D)：纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准品。

# 10.4 标准溶液配制

10.4.1 标准储备液 $(1000\mathrm{mg / L})$ : 准确称取 6 种标准品 (精确至 $0.1\mathrm{mg}$ ), 称量质量计算方式为 $25\mathrm{mg}$ 除以折算系数, 折算系数见附录 D, 分别用甲醇溶解后转移至 $25\mathrm{mL}$ 容量瓶中, 并用甲醇定容至刻度, 混匀。将溶液转移至棕色玻璃容器中, 于 $4^{\circ}\mathrm{C}$ 下避光保存, 保存期 6 个月。  
10.4.2 混合标准中间液 $(50\mathrm{mg / L})$ : 分别准确吸取标准储备液 $(1000\mathrm{mg / L}) 0.5\mathrm{mL}$ 于 $10\mathrm{mL}$ 容量瓶中, 加甲醇定容至刻度, 混匀。将溶液转移至棕色棕色玻璃容器中, 于 $4^{\circ}\mathrm{C}$ 下避光保存, 保存期6个月。  
10.4.3 水基食品模拟物系列标准工作溶液：分别准确吸取 $25\mu \mathrm{L}, 50\mu \mathrm{L}, 100\mu \mathrm{L}, 200\mu \mathrm{L}, 500\mu \mathrm{L}$ 混合标准中间液 $(50\mathrm{mg / L})$ 于 $5\mathrm{mL}$ 容量瓶中，用 $50\%$ （体积分数）乙醇溶液定容。所得标准工作溶液中各目标物质浓度分别为 $0.25\mathrm{mg / L}, 0.50\mathrm{mg / L}, 1.0\mathrm{mg / L}, 2.0\mathrm{mg / L}, 5.0\mathrm{mg / L}$ 。采取相同方式，分别用水、 $4\%$ （体积分数）乙酸、 $10\%$ （体积分数）乙醇、 $20\%$ （体积分数）乙醇配制相同浓度系列的标准工作溶液。临用现配。  
10.4.4 油脂类食品模拟物系列标准工作溶液：分别称取 $5.00\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )橄榄油模拟物于6个离心管中，用移液器分别移取 $30~\mu \mathrm{L},60~\mu \mathrm{L},100~\mu \mathrm{L},150~\mu \mathrm{L},300~\mu \mathrm{L}$ 混合标准中间液( $50~\mathrm{mg / L})$ 涡旋混匀后，加入 $5\mathrm{mL}$ 甲醇，涡旋 $1\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上层溶液经微孔滤膜后，待测。所得标准工作溶液中各目标物质浓度分别为 $0.30~\mathrm{mg / kg},0.60~\mathrm{mg / kg},1.0~\mathrm{mg / kg},1.5~\mathrm{mg / kg},3.0~\mathrm{mg / kg}$ 。临用现配。  
10.4.5 异辛烷系列标准工作溶液：分别移取 $5.00 \mathrm{~mL}$ 异辛烷于 6 个离心管中，用移液器分别移取 $30 \mu \mathrm{L}, 60 \mu \mathrm{L}, 100 \mu \mathrm{L}, 120 \mu \mathrm{L}, 300 \mu \mathrm{L}$ 混合标准中间液 $(50 \mathrm{mg} / \mathrm{L})$ 涡旋混匀后，加入 $5.00 \mathrm{~mL} 80 \%$ 甲醇溶液，涡旋混匀 $1 \mathrm{~min}, 4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，取下层溶液过滤后，待测。所得标准工作溶液中各目标物质浓度分别为 $0.30 \mathrm{mg} / \mathrm{L}, 0.60 \mathrm{mg} / \mathrm{L}, 1.0 \mathrm{mg} / \mathrm{L}, 1.2 \mathrm{mg} / \mathrm{L}, 3.0 \mathrm{mg} / \mathrm{L}$ 。临用现配。  
10.4.6 $95\%$ (体积分数)乙醇标准工作溶液: 分别准确吸取 $25 \mu \mathrm{L}, 50 \mu \mathrm{L}, 100 \mu \mathrm{L}, 200 \mu \mathrm{L}, 500 \mu \mathrm{L}$ 混合标准中间液 $(50 \mathrm{mg} / \mathrm{L})$ 于 $5 \mathrm{~mL}$ 容量瓶中, 用 $95\%$ (体积分数)乙醇溶液定容。所得标准工作溶液中各目标物质浓度分别为 $0.25 \mathrm{mg} / \mathrm{L}, 0.50 \mathrm{mg} / \mathrm{L}, 1.0 \mathrm{mg} / \mathrm{L}, 2.0 \mathrm{mg} / \mathrm{L}, 5.0 \mathrm{mg} / \mathrm{L}$ 。临用现配。

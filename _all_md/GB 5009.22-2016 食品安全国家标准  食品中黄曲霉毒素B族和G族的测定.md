# 前言

本标准代替GB/T5009.22—2003《食品中黄曲霉毒素 $\mathbf{B}_1$ 的测定》、GB/T5009.23—2006《食品中黄曲霉毒素 $\mathrm{B_1}$ 、 $\mathrm{B}_2$ 、 $\mathrm{G}_1$ 、 $\mathrm{G}_2$ 的测定》、GB5009.24—2010《食品安全国家标准食品中黄曲霉毒素 $\mathrm{M}_{1}$ 和 $\mathrm{B}_1$ 的测定》、GB/T23212—2008《牛奶和奶粉中黄曲霉毒素 $\mathrm{B_1}$ 、 $\mathrm{B}_2$ 、 $\mathrm{G}_1$ 、 $\mathrm{G}_2$ 、 $\mathrm{M}_1$ 、 $\mathrm{M}_2$ 的测定液相色谱-荧光检测法》、GB/T18979—2003《食品中黄曲霉毒素的测定免疫亲和层析净化高效液相色谱法和荧光光度法》、SN0339—1995《出口茶叶中黄曲霉毒素 $\mathrm{B_1}$ 检验方法》、SN/T1664—2005《牛奶和奶粉中黄曲霉毒素 $\mathrm{M_1}$ 、 $\mathrm{B}_1$ 、 $\mathrm{B}_2$ 、 $\mathrm{G}_1$ 、 $\mathrm{G}_2$ 含量的测定》、SN/T1101—2002《进出口油籽及粮谷中黄曲霉毒素的检验方法》、SN0637—1997《出口油籽、坚果及坚果制品中黄曲霉毒素的检验方法液相色谱法》、SN/T1736—2006《进出口蜂蜜中黄曲霉毒素的检验方法高效液相色谱法》、NY/T1286—2007《花生黄曲霉毒素 $\mathbf{B}_1$ 的测定高效液相色谱法》。

本标准与GB/T5009.22—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中黄曲霉毒素B族和G族的测定”；  
根据GB2761—2011的要求，增加了方法的适用范围；  
——增加了同位素稀释液相色谱-串联质谱法为第一法；  
——增加了高效液相色谱-柱前衍生法为第二法；  
——增加了高效液相色谱-柱后衍生法为第三法；  
——修改了酶联免疫法，并将方法名称更改为酶联免疫吸附筛查法；  
——增加了免疫亲和柱以及酶联免疫试剂盒质量判定要求与方法；  
——修改了测定组分为黄曲霉毒素B族和G族化合物。

# 食品安全国家标准

# 食品中黄曲霉毒素B族和G族的测定

# 1 范围

本标准规定了食品中黄曲霉毒素 $\mathrm{B}_{1}$ 、黄曲霉毒素 $\mathrm{B}_{2}$ 、黄曲霉毒素 $\mathrm{G}_{1}$ 、黄曲霉毒素 $\mathrm{G}_{2}$ （以下简称 $\mathrm{AFT~B_1}$ 、 $\mathrm{AFT~B_2}$ 、 $\mathrm{AFT~G_1}$ 和 $\mathrm{AFT~G_2}$ )的测定方法。

本标准第一法为同位素稀释液相色谱-串联质谱法，适用于谷物及其制品、豆类及其制品、坚果及籽类、油脂及其制品、调味品、婴幼儿配方食品和婴幼儿辅助食品中 $\mathrm{AFTB_1}$ 、 $\mathrm{AFTB_2}$ 、 $\mathrm{AFTG_1}$ 和 $\mathrm{AFTG_2}$ 的测定。

本标准第二法为高效液相色谱-柱前衍生法，适用于谷物及其制品、豆类及其制品、坚果及籽类、油脂及其制品、调味品、婴幼儿配方食品和婴幼儿辅助食品中 $\mathrm{AFTB_1}$ 、 $\mathrm{AFTB_2}$ 、 $\mathrm{AFTG_1}$ 和 $\mathrm{AFTG_2}$ 的测定。

本标准第三法为高效液相色谱-柱后衍生法，适用于谷物及其制品、豆类及其制品、坚果及籽类、油脂及其制品、调味品、婴幼儿配方食品和婴幼儿辅助食品中 $\mathrm{AFTB_1}$ 、 $\mathrm{AFTB_2}$ 、 $\mathrm{AFTG_1}$ 和 $\mathrm{AFTG_2}$ 的测定。

本标准第四法为酶联免疫吸附筛查法，适用于谷物及其制品、豆类及其制品、坚果及籽类、油脂及其制品、调味品、婴幼儿配方食品和婴幼儿辅助食品中 $\mathrm{AFTB_1}$ 的测定。

本标准第五法为薄层色谱法，适用于谷物及其制品、豆类及其制品、坚果及籽类、油脂及其制品、调味品中 $\mathrm{AFTB_1}$ 的测定。

# 第一法 同位素稀释液相色谱-串联质谱法

# 2 原理

试样中的黄曲霉毒素 $\mathrm{B}_{1}$ 、黄曲霉毒素 $\mathrm{B}_{2}$ 、黄曲霉毒素 $\mathrm{G}_{1}$ 、黄曲霉毒素 $\mathrm{G}_{2}$ ，用乙腈-水溶液或甲醇-水溶液提取，提取液用含 $1\%$ Triton X-100(或吐温-20)的磷酸盐缓冲溶液稀释后（必要时经黄曲霉毒素固相净化柱初步净化），通过免疫亲和柱净化和富集，净化液浓缩、定容和过滤后经液相色谱分离，串联质谱检测，同位素内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.3 乙酸铵 $\mathrm{(CH_3COONH_4)}$ ：色谱纯。  
3.1.4 氯化钠(NaCl)。

3.1.5 磷酸氢二钠 $\mathrm{(Na_2HPO_4)}$ 。  
3.1.6 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
3.1.7 氯化钾（KCl）。  
3.1.8 盐酸（HCl）。  
3.1.9 Triton X-100 $\left[\mathrm{C}_{14} \mathrm{H}_{22} \mathrm{O}\left(\mathrm{C}_{2} \mathrm{H}_{4} \mathrm{O}\right)_{n}\right]$ （或吐温 $-20, \mathrm{C}_{58} \mathrm{H}_{114} \mathrm{O}_{26}$ ）。

# 3.2 试剂配制

3.2.1 乙酸铵溶液 $(5\mathrm{mmol} / \mathrm{L})$ ：称取 $0.39\mathrm{g}$ 乙酸铵，用水溶解后稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.2 乙腈-水溶液 $(84 + 16)$ ：取 $840~\mathrm{mL}$ 乙腈加入 $160~\mathrm{mL}$ 水，混匀。  
3.2.3 甲醇-水溶液 $(70 + 30)$ ：取 $700~\mathrm{mL}$ 甲醇加入 $300~\mathrm{mL}$ 水，混匀。  
3.2.4 乙腈-水溶液 $(50 + 50)$ ：取 $50~\mathrm{mL}$ 乙腈加入 $50~\mathrm{mL}$ 水，混匀。  
3.2.5 乙腈-甲醇溶液 $(50 + 50)$ ：取 $50~\mathrm{mL}$ 乙腈加入 $50~\mathrm{mL}$ 甲醇，混匀。  
3.2.6 $10\%$ 盐酸溶液：取 $1\mathrm{mL}$ 盐酸，用纯水稀释至 $10~\mathrm{mL}$ ，混匀。  
3.2.7 磷酸盐缓冲溶液（以下简称PBS)：称取 $8.00\mathrm{g}$ 氯化钠、 $1.20\mathrm{g}$ 磷酸氢二钠（或 $2.92\mathrm{g}$ 十二水磷酸氢二钠）、 $0.20\mathrm{g}$ 磷酸二氢钾、 $0.20\mathrm{g}$ 氯化钾，用 $900~\mathrm{mL}$ 水溶解，用盐酸调节 $\mathrm{pH}$ 至 $7.4\pm 0.1$ ，加水稀释至 $1000~\mathrm{mL}$ 。  
3.2.8 $1\%$ Triton X-100(或吐温-20)的PBS：取 $10~\mathrm{mL}$ Triton X-100(或吐温-20)，用PBS稀释至 $1000~\mathrm{mL}$ 。

# 3.4 标准溶液配制

3.4.1 标准储备溶液 $(10\mu \mathrm{g / mL})$ ：分别称取 $\mathrm{AFTB_1}$ 、 $\mathrm{AFTB_2}$ 、 $\mathrm{AFTG_1}$ 和 $\mathrm{AFTG_2}1\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ ），用乙腈溶解并定容至 $100~\mathrm{mL}$ 。此溶液浓度约为 $10~\mu \mathrm{g / mL}$ 。溶液转移至试剂瓶中后，在一 $20^{\circ}C$ 下避光保存，备用。临用前进行浓度校准(校准方法参见附录A)。  
3.4.2 混合标准工作液 $(100\mathrm{ng / mL})$ ：准确移取混合标准储备溶液 $(1.0\mu \mathrm{g / mL})1.00\mathrm{mL}$ 至 $100~\mathrm{mL}$ 容量瓶中，乙腈定容。此溶液密封后避光一 $20^{\circ}C$ 下保存，三个月有效。  
3.4.3 混合同位素内标工作液 $(100\mathrm{ng / mL})$ ：准确移取 $0.5\mu \mathrm{g} / \mathrm{mL}^{13}\mathrm{C}_{17}$ -AFT $\mathrm{B}_1$ 、 ${}^{13}\mathrm{C}_{17}$ -AFT $\mathrm{B}_2$ 、 ${}^{13}\mathrm{C}_{17}-$ AFT $\mathrm{G}_1$ 和 ${}^{13}\mathrm{C}_{17}$ -AFT $\mathrm{G}_2$ 各 $2.00~\mathrm{mL}$ ，用乙腈定容至 $10~\mathrm{mL}$ 。在 $-20^{\circ}\mathrm{C}$ 下避光保存，备用。  
3.4.4 标准系列工作溶液：准确移取混合标准工作液 $(100\mathrm{ng / mL})10\mu \mathrm{L},50\mu \mathrm{L},100\mu \mathrm{L},200\mu \mathrm{L}$ 、 $500\mu \mathrm{L},800\mu \mathrm{L},1000\mu \mathrm{L}$ 至 $10~\mathrm{mL}$ 容量瓶中，加入 $200\mu \mathrm{L}100\mathrm{ng / mL}$ 的同位素内标工作液，用初始流动相定容至刻度，配制浓度点为 $0.1\mathrm{ng / mL},0.5\mathrm{ng / mL},1.0\mathrm{ng / mL},2.0\mathrm{ng / mL},5.0\mathrm{ng / mL},8.0\mathrm{ng / mL},$

$10.0\mathrm{ng / mL}$ 的系列标准溶液。

# 5 分析步骤

使用不同厂商的免疫亲和柱，在样品上样、淋洗和洗脱的操作方面可能会略有不同，应该按照供应商所提供的操作说明书要求进行操作。

警示：整个分析操作过程应在指定区域内进行。该区域应避光（直射阳光）、具备相对独立的操作台和废弃物存放装置。在整个实验过程中，操作者应按照接触剧毒物的要求采取相应的保护措施。

# 5.1 样品制备

# 5.1.1 液体样品（植物油、酱油、醋等）

采样量需大于 $1\mathrm{L}$ ，对于袋装、瓶装等包装样品需至少采集3个包装（同一批次或号），将所有液体样品在一个容器中用匀浆机混匀后，其中任意的 $100\mathrm{g(mL)}$ 样品进行检测。

# 5.1.2 固体样品（谷物及其制品、坚果及籽类、婴幼儿谷类辅助食品等）

采样量需大于 $1\mathrm{kg}$ ，用高速粉碎机将其粉碎，过筛，使其粒径小于 $2\mathrm{mm}$ 孔径试验筛，混合均匀后缩分至 $100\mathrm{g}$ ，储存于样品瓶中，密封保存，供检测用。

# 5.1.3 半流体（腐乳、豆鼓等）

采样量需大于 $1\mathrm{kg(L)}$ ，对于袋装、瓶装等包装样品需至少采集3个包装(同一批次或号)，用组织

捣碎机捣碎混匀后，储存于样品瓶中，密封保存，供检测用。

# 5.2 样品提取

# 5.2.1 液体样品

# 5.2.1.1 植物油脂

# 5.3 样品净化

# 5.3.1 免疫亲和柱净化

# 5.3.1.1 上样液的准备

准确移取 $4\mathrm{mL}$ 上清液，加入 $46~\mathrm{mL}~1\%$ Trition X-100(或吐温-20)的PBS(使用甲醇-水溶液提取时可减半加入)，混匀。

# 5.3.1.2 免疫亲和柱的准备

# 5.3.1.3 试样的净化

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

式中：

$X$ ——试样中 $\mathrm{AFTB_1}$ 、 $\mathrm{AFTB_2}$ 、 $\mathrm{AFTG_1}$ 或 $\mathrm{AFTG_2}$ 的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$V_{1}$ ——试样提取液体积(植物油脂、固体、半固体按加入的提取液体积；酱油、醋按定容总体积)，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——净化液的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$V_{2}$ ——净化柱净化后的取样液体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

称取 $20.00\mathrm{g}$ 研磨均匀的试样，置于 $250~\mathrm{mL}$ 具塞锥形瓶中，加入 $20~\mathrm{mL}$ 正己烷或石油醚与 $50~\mathrm{mL}$ 甲醇水溶液。振荡 $30\mathrm{min}$ ，静置片刻，以叠成折叠式快速定性滤纸过滤，滤液静置分层后，取 $24~\mathrm{mL}$ 甲醇水层（相当 $8\mathrm{g}$ 试样，其中包括 $8\mathrm{g}$ 干酱类本身约含有 $4\mathrm{mL}$ 水的体积在内）置于分液漏斗中，加入 $20~\mathrm{mL}$ 三氯甲烷，以下按33.1.1.1自“振摇 $2\mathrm{min}$ ，静置分层……”起依法操作。最后加入 $2\mathrm{mL}$ 苯-乙腈混合液。此溶液每毫升相当于 $4\mathrm{g}$ 试样。

# 33.2 测定

# 33.2.1 单向展开法

# 33.2.1.1 薄层板的制备

称取约 $3\mathrm{g}$ 硅胶 $\mathrm{G}$ ，加相当于硅胶量2倍～3倍的水，用力研磨 $1\mathrm{min} \sim 2\mathrm{min}$ 至成糊状后立即倒于涂布器内，推成 $5\mathrm{cm} \times 20\mathrm{cm}$ ，厚度约 $0.25\mathrm{mm}$ 的薄层板三块。在空气中干燥约 $15\mathrm{min}$ 后，在 $100^{\circ}\mathrm{C}$ 活化 $2\mathrm{h}$ ，取出，放干燥器中保存。一般可保存 $2\mathrm{d} \sim 3\mathrm{d}$ ，若放置时间较长，可再活化后使用。

# 33.2.1.2 点样

将薄层板边缘附着的吸附剂刮净，在距薄层板下端 $3\mathrm{cm}$ 的基线上用微量注射器或血色素吸管滴加样液。一块板可滴加4个点，点距边缘和点间距约为 $1\mathrm{cm}$ ，点直径约 $3\mathrm{mm}$ 。在同一块板上滴加点的大小应一致，滴加时可用吹风机用冷风边吹边加。滴加样式如下：

第一点： $0\mu \mathrm{L}$ AFT $\mathrm{B}_{1}$ 标准工作液 $(0.04~\mu \mathrm{g / mL})$ 。

第二点： $20\mu \mathrm{L}$ 样液。

第三点： $20\mu \mathrm{L}$ 样液 $+10~\mu \mathrm{L}0.04~\mu \mathrm{g / mL}$ AFT $\mathbf{B}_1$ 标准工作液。

第四点： $20\mu \mathrm{L}$ 样液 $+10~\mu \mathrm{L}~0.2~\mu \mathrm{g / mL}$ AFT $\mathbf{B}_1$ 标准工作液。

# 33.2.1.3 展开与观察

在展开槽内加 $10\mathrm{mL}$ 无水乙醚，预展 $12\mathrm{cm}$ ，取出挥干。再于另一展开槽内加 $10\mathrm{mL}$ 丙酮-三氯甲烷 $(8 + 92)$ ，展开 $10\mathrm{cm} \sim 12\mathrm{cm}$ ，取出。在紫外光下观察结果，方法如下。

由于样液点上加滴 $\mathrm{AFTB_1}$ 标准工作液，可使 $\mathrm{AFTB_1}$ 标准点与样液中的 $\mathrm{AFTB_1}$ 荧光点重叠。如样液为阴性，薄层板上的第三点中 $\mathrm{AFTB_1}$ 为 $0.0004\mu \mathrm{g}$ ，可用作检查在样液内 $\mathrm{AFTB_1}$ 最低检出量是否正常出现：如为阳性，则起定性作用。薄层板上的第四点中 $\mathrm{AFTB_1}$ 为 $0.002\mu \mathrm{g}$ ，主要起定位作用。

若第二点在与 $\mathrm{AFTB_1}$ 标准点的相应位置上无蓝紫色荧光点，表示试样中 $\mathrm{AFTB_1}$ 含量在 $5\mu \mathrm{g / kg}$ 以下，如在相应位置上有蓝紫色荧光点，则需进行确证试验。

# 33.2.1.4 确证试验

为了证实薄层板上样液荧光系由 $\mathrm{AFTB_1}$ 产生的，加滴三氟乙酸，产生 $\mathrm{AFTB_1}$ 的衍生物，展开后此衍生物的比移值在0.1左右。于薄层板左边依次滴加两个点。

第一点： $0.04 \mu \mathrm{g} / \mathrm{mL}$ AFT $\mathrm{B}_{1}$ 标准工作液 $10 \mu \mathrm{L}$ 。

第二点： $20\mu \mathrm{L}$ 样液。于以上两点各加一小滴三氟乙酸盖于其上，反应 $5\mathrm{min}$ 后，用吹风机吹热风 $2\mathrm{min}$ 后，使热风吹到薄层板上的温度不高于 $40^{\circ}\mathrm{C}$ ，再于薄层板上滴加以下两个点。

第三点： $0.04 \mu \mathrm{g} / \mathrm{mL}$ AFT $\mathrm{B}_{1}$ 标准工作液 $10 \mu \mathrm{L}$ 。

第四点： $20\mu \mathrm{L}$ 样液。

再展开(同16.2.1.3)，在紫外光灯下观察样液是否产生与 $\mathrm{AFTB_1}$ 标准点相同的衍生物。未加三氟乙酸的三、四两点，可依次作为样液与标准的衍生物空白对照。

# 33.2.1.5 稀释定量

样液中的 $\mathrm{AFTB_1}$ 荧光点的荧光强度如与 $\mathrm{AFTB_1}$ 标准点的最低检出量 $(0.0004\mu \mathrm{g})$ 的荧光强度一致，则试样中 $\mathrm{AFTB_1}$ 含量即为 $5\mu \mathrm{g / kg}$ 。如样液中荧光强度比最低检出量强，则根据其强度估计减少滴加微升数或将样液稀释后再滴加不同微升数，直至样液点的荧光强度与最低检出量的荧光强度一致为止。滴加式样如下：

第一点： $10\mu \mathrm{L}$ AFT $\mathbf{B}_1$ 标准工作液 $(0.04~\mu \mathrm{g / mL})$

第二点：根据情况滴加 $10\mu \mathrm{L}$ 样液。

第三点：根据情况滴加 $15\mu \mathrm{L}$ 样液。

第四点：根据情况滴加 $20~\mu \mathrm{L}$ 样液。

# 33.2.1.6 结果计算

试样中 $\mathrm{AFTB_1}$ 的含量按式(5)计算：

$$
X = 0. 0 0 0 4 \times \frac {V _ {1} \times f}{V _ {2} \times m} \times 1 0 0 0 \quad \dots \tag {5}
$$

式中：

$X$ ——试样中 $\mathrm{AFTB_1}$ 的含量，单位为微克每千克 $(\mu \mathrm{g / kg})$ ；

0.0004——AFT $\mathrm{B}_{1}$ 的最低检出量，单位为微克 $(\mu \mathrm{g})$

$V_{1}$ ——加入苯-乙腈混合液的体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——样液的总稀释倍数；

$V_{2}$ ——出现最低荧光时滴加样液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——加入苯-乙腈混合液溶解时相当试样的质量，单位为克(g)；

1000 ——换算系数。

结果表示到测定值的整数位。

# 33.2.2 双向展开法

如用单向展开法展开后，薄层色谱由于杂质干扰掩盖了 $\mathrm{AFTB_1}$ 的荧光强度，需采用双向展开法。薄层板先用无水乙醚作横向展开，将干扰的杂质展至样液点的一边而 $\mathrm{AFTB_1}$ 不动，然后再用丙酮-三氯甲烷(8+92)作纵向展开，试样在 $\mathrm{AFTB_1}$ 相应处的杂质底色大量减少，因而提高了方法灵敏度。如用双向展开中滴加两点法展开仍有杂质干扰时，则可改用滴加一点法。

# 33.2.2.1 滴如两点法

# 33.2.2.1.1 点样

取薄层板三块，在距下端 $3\mathrm{cm}$ 基线上滴加 $\mathrm{AFTB_1}$ 标准使用液与样液。即在三块板的距左边缘 $0.8~\mathrm{cm}\sim 1~\mathrm{cm}$ 处各滴加 $10~\mu \mathrm{L}$ AFT $\mathrm{B_1}$ 标准使用液 $(0.04~\mu \mathrm{g / mL})$ ，在距左边缘 $2.8~\mathrm{cm}\sim 3~\mathrm{cm}$ 处各滴加 $20~\mu \mathrm{L}$ 样液，然后在第二块板的样液点上加滴 $10~\mu \mathrm{L}$ AFT $\mathrm{B_1}$ 标准使用液 $(0.04~\mu \mathrm{g / mL})$ ，在第三块板的样液点上加滴 $10~\mu \mathrm{L}0.2~\mu \mathrm{g / mL}$ AFT $\mathrm{B_1}$ 标准使用液。

# 33.2.2.1.2 展开

33.2.2.1.2.1 横向展开：在展开槽内的长边置一玻璃支架，加 $10~\mathrm{mL}$ 无水乙醇，将上述点好的薄层板靠标准点的长边置于展开槽内展开，展至板端后，取出挥干，或根据情况需要时可再重复展开1次～2次。

33.2.2.1.2.2 纵向展开：挥干的薄层板以丙酮-三氯甲烷 $(8 + 92)$ 展开至 $10\mathrm{cm}\sim 12\mathrm{cm}$ 为止。丙酮与三氯甲烷的比例根据不同条件自行调节。

# 33.2.2.1.3 观察及评定结果

在紫外光灯下观察第一、二板，若第二板的第二点在 $\mathrm{AFTB_1}$ 标准点的相应处出现最低检出量，而第一板在与第二板的相同位置上未出现荧光点，则试样中 $\mathrm{AFTB_1}$ 含量在 $5\mu \mathrm{g / kg}$ 以下。

若第一板在与第二板的相同位置上出现荧光点，则将第一板与第三板比较，看第三板上第二点与第一板上第二点的相同位置上的荧光点是否与 $\mathrm{AFTB_1}$ 标准点重叠，如果重叠，再进行确证试验。在具体测定中，第一、二、三板可以同时做，也可按照顺序做。如按顺序做，当在第一板出现阴性时，第三板可以省略，如第一板为阳性，则第二板可以省略，直接作第三板。

# 33.2.2.1.4 确证试验

另取薄层板两块，于第四、第五两板距左边缘 $0.8\mathrm{cm}\sim 1\mathrm{cm}$ 处各滴加 $10~\mu \mathrm{L}$ AFT $\mathrm{B}_{1}$ 标准使用液 $(0.04\mu \mathrm{g / mL})$ 及1小滴三氟乙酸；在距左边缘 $2.8\mathrm{cm}\sim 3\mathrm{cm}$ 处，于第四板滴加 $20~\mu \mathrm{L}$ 样液及1小滴三氟乙酸，于第五板滴加 $20~\mu \mathrm{L}$ 样液、 $10~\mu \mathrm{L}$ AFT $\mathrm{B}_{1}$ 标准使用液 $(0.04\mu \mathrm{g / mL})$ 及1小滴三氟乙酸。反应 $5\mathrm{min}$ 后，用吹风机吹热风 $2\mathrm{min}$ ，使热风吹到薄层极上的温度不高于 $40^{\circ}C$ 。再用双向展开法展开后，

观察样液是否产生与 $\mathrm{AFTB_1}$ 标准点重叠的衍生物。观察时，可将第一板作为样液的衍生物空白板。如样液 $\mathrm{AFTB_1}$ 含量高时，则将样液稀释后，按33.2.1.4做确证试验。

# 33.2.2.1.5 稀释定量

如样液 $\mathrm{AFTB_1}$ 含量高时，按16.3.1.5稀释定量操作。如 $\mathrm{AFTB_1}$ 含量低，稀释倍数小，在定量的纵向展开板上仍有杂质干扰，影响结果的判断，可将样液再做双向展开法测定，以确定含量。

# 33.2.2.1.6 结果计算

同33.2.1.6。

# 33.2.2.2 滴加一点法

# 33.2.2.2.1 点样

取薄层板三块，在距下端 $3\mathrm{cm}$ 基线上滴加 $\mathrm{AFTB_1}$ 标准使用液与样液。即在三块板臣左边缘 $0.8~\mathrm{cm}\sim 1~\mathrm{cm}$ 处各滴加 $20~\mu \mathrm{L}$ 样液，在第二板的点上加 $10~\mu \mathrm{L}$ $\mathrm{AFTB_1}$ 标准使用液 $(0.04~\mu \mathrm{g / mL})$ 。在第三板的点上加滴 $10~\mu \mathrm{L}$ $\mathrm{AFTB_1}$ 标准榕液 $(0.2~\mu \mathrm{g / mL})$ 。

# 免疫亲和柱验证方法

# B.1 柱容量验证

在 $30~\mathrm{mL}$ 的 $1\%$ Triton X-100(或吐温-20)-PBS中加入 $600~\mathrm{ng}~\mathrm{AFT}~\mathrm{B}_1$ 标准储备溶液，充分混匀。分别取同一批次3根免疫亲和柱，每根柱的上样量为 $10~\mathrm{mL}$ 。经上样、淋洗、洗脱，收集洗脱液，用氮气吹干至 $1~\mathrm{mL}$ ，用初始流动相定容至 $10~\mathrm{mL}$ ，用液相色谱仪分离测定 $\mathrm{AFT}~\mathrm{B}_1$ 的含量。

结果判定：结果 $\mathrm{AFTB_1\geqslant 160ng}$ ，为可使用商品。

# B.2 柱回收率验证

在 $30~\mathrm{mL}$ 的 $1\%$ Triton X-100(或吐温-20)-PBS中加入 $600~\mathrm{ng}~\mathrm{AFT}~\mathrm{B}_1$ 标准储备溶液，充分混匀。分别取同一批次3根免疫亲和柱，每根柱的上样量为 $10~\mathrm{mL}$ 。经上样、淋洗、洗脱，收集洗脱液，用氮气吹干至 $1~\mathrm{mL}$ ，用初始流动相定容至 $10~\mathrm{mL}$ ，用液相色谱仪分离测定 $\mathrm{AFT}~\mathrm{B}_1$ 的含量。

结果判定：结果 $\mathrm{AFTB_1\geqslant 160ng}$ ，即回收率 $\geq 80\%$ ，为可使用商品。

# B.3 交叉反应率验证

在 $30~\mathrm{mL}$ 的 $1\%$ Triton X-100(或吐温-20)-PBS中加入 $300\mathrm{ngAFTG_2}$ 标准储备溶液，充分混匀。分别取同一批次3根免疫亲和柱，每根柱的上样量为 $10~\mathrm{mL}$ 。经上样、淋洗、洗脱，收集洗脱液，用氮气吹干至 $1\mathrm{mL}$ ，用初始流动相定容至 $10~\mathrm{mL}$ ，用液相色谱仪分离测定 $\mathrm{AFTG_2}$ 的含量。

结果判定：结果 $\mathrm{AFT~G_2\geqslant 80~ng}$ ，为可同时测定 $\mathrm{AFT~B_1}$ 、 $\mathrm{AFT~B_2}$ 、 $\mathrm{AFT~G_1}$ 、 $\mathrm{AFT~G_2}$ 时使用的商品。

# 酶联免疫试剂盒的质量判定方法

选取小麦粉或其他阴性样品，根据所购酶联免疫试剂盒的检出限，在阴性基质中添加3个浓度水平的 $\mathrm{AFTB_1}$ 标准溶液（ $2\mu \mathrm{g / kg},5\mu \mathrm{g / kg},10\mu \mathrm{g / kg})$ 。按照说明书操作方法，用读数仪读数，做三次平行实验。针对每个加标浓度，回收率在 $50\% \sim 120\%$ 容许范围内的该批次产品方可使用。

注：当试剂盒用于特殊膳食用食品基质检测时，需根据其限量，考察添加浓度水平为 $0.2\mu \mathrm{g / kg}$ AFT $\mathbf{B}_1$ 标准溶液的回收率。

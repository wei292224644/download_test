# 1 范围

本标准适用于以维生素 $\mathrm{K}_{3}$ 、七烯萜醇为原料，或者以维生素 $\mathrm{K}_{3}$ 、法尼醇和香叶醇为原料，经化学合成制得的食品营养强化剂维生素 $\mathrm{K}_{2}$ 。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

(全-E)-2-(3,7,11,15,19,23,27-七甲基-2,6,10,14,18,22,26-二十八碳七烯基)-3-甲基-1,4-萘醌

$$
\mathrm {C} _ {4 6} \mathrm {H} _ {6 4} \mathrm {O} _ {2}
$$

# 附录A

# 检验方法

# A.1 安全提示

本标准试验方法中使用的部分试剂具有毒性或腐蚀性，使用时需小心谨慎并按照相关规定操作。若溅到皮肤上应立即用水冲洗，严重者应立即治疗。在使用挥发性酸时，要在通风橱中进行。

# A.2 一般规定

除非另有说明，在分析中仅使用确认为分析纯试剂和GB/T6682中规定的一级水。试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

维生素 $\mathrm{K}_2$ 对光敏感，试验过程应注意避光。

# A.3 鉴别试验

按维生素 $\mathrm{K}_2$ 含量的测定方法进行高效液相色谱分析，试样与维生素 $\mathrm{K}_2$ 标准品的保留时间进行对照。试样色谱图的主峰应与标准品主峰保留时间一致。

# A.4 维生素 $\mathbf{K}_2$ 含量的测定

# A.4.1 方法原理

试样用四氢呋喃溶解，按规定条件进行高效液相色谱分析，与维生素 $\mathrm{K}_2$ 标准品的保留时间进行对照，峰面积外标法定量，测定试样中维生素 $\mathrm{K}_2$ 的含量。

# A.4.2 试剂和材料

A.4.2.1 四氢呋喃：色谱纯。  
A.4.2.2 无水乙醇：色谱纯。  
A.4.2.3 维生素 $\mathrm{K}_2$ 标准品：七烯甲萘醌(MK-7，分子式： $\mathrm{C_{46}H_{64}O_2}$ ，CAS号：2124-57-4)，纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
A.4.2.4 维生素 $\mathrm{K}_2$ 标准储备溶液：称取 $25\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )维生素 $\mathrm{K}_2$ 标准品(A.4.2.3)于 $50~\mathrm{mL}$ 棕色容量瓶中，加入 $1\mathrm{mL}$ 四氢呋喃(A.4.2.1)，溶解后，用无水乙醇(A.4.2.2)定容至刻度，混匀，过滤膜，作为储备溶液，于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 避光保存，有效期1个月。  
A.4.2.5 维生素 $\mathrm{K}_2$ 标准溶液：吸取 $5\mathrm{mL}$ 维生素 $\mathrm{K}_2$ 标准储备溶液(A.4.2.4)至 $25~\mathrm{mL}$ 棕色容量瓶中，用无水乙醇(A.4.2.2)定容至刻度，混匀，过滤膜，现用现配。  
A.4.2.6 试样溶液：称取 $25\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )试样于 $50~\mathrm{mL}$ 棕色容量瓶中，加入 $1\mathrm{mL}$ 四氢呋喃(A.4.2.1)，用无水乙醇(A.4.2.2)溶解并定容至刻度。吸取 $5\mathrm{mL}$ 该溶液至 $25~\mathrm{mL}$ 棕色容量瓶中，用无水乙醇(A.4.2.2)定容至刻度，混匀，过滤膜。  
A.4.2.7 微孔滤膜： $0.45\mu \mathrm{m}$ ，有机系。

# A.4.3 仪器和设备

A.4.3.1 高效液相色谱仪：配有紫外检测器或二极管阵列检测器。  
A.4.3.2 分析天平：感量为 $0.1\mathrm{mg}$

# A.4.4 参考色谱条件

A.4.4.1 色谱柱： $\mathrm{C_{18}}$ 色谱柱， $4.6 \mathrm{~mm} \times 25 \mathrm{~cm}$ ，粒径 $2.6 \mu \mathrm{m}$ ，或等效色谱柱。

A.4.4.2 流动相：无水乙醇：水 $= 97:3$   
A.4.4.3 流速： $0.7\mathrm{mL / min}$   
A.4.4.4 柱温： $25^{\circ}C$   
A.4.4.5 进样量： $10~\mu \mathrm{L}$   
A.4.4.6 检测波长： $268~\mathrm{nm}$   
A.4.4.7 检测时间：至少1.5倍于维生素 $\mathrm{K}_2$ 峰的保留时间。

# A.4.5 分析步骤

# A.4.5.1 系统适应性

取维生素 $\mathrm{K}_2$ 标准溶液(A.4.2.5)注入液相色谱仪中，记录色谱图，连续进样6次，维生素 $\mathrm{K}_2$ 峰面积响应值的相对标准偏差(RSD)应不大于 $3.0\%$ ，维生素 $\mathrm{K}_2$ 主峰与它最近的杂质峰的分离度 $R$ 应不小于1.5。

# A.4.5.2 试样测定

系统适应性合格后，分别取标准溶液(A.4.2.5)和试样溶液(A.4.2.6) $10\mu \mathrm{L}$ 注入液相色谱仪中进行测定，以保留时间定性，峰面积外标法定量。参考色谱图见附录B中图B.1。

# A.4.6 结果计算

维生素 $\mathrm{K}_2$ 含量 $W_{1}$ 以 $\omega / \%$ 计，按公式(A.1)计算：

$$
W _ {1} = \frac {A _ {\mathrm {u}}}{A _ {\mathrm {s}}} \times \frac {C _ {\mathrm {s}}}{C _ {\mathrm {u}}} \times 1 0 0 \tag {A.1}
$$

式中：

$A_{\mathrm{u}}$ ——试样中维生素 $\mathrm{K}_2$ 对应峰面积；

$A_{s}$ ——标准品中维生素 $\mathrm{K}_2$ 对应峰面积；

$C_{\mathrm{s}}$ ——标准溶液中维生素 $\mathrm{K}_{2}$ 的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$C_{\mathrm{u}}$ ——试样溶液中维生素 $\mathrm{K}_{2}$ 的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

100 ——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的2次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# A.5 维生素 $\mathbf{K}_2$ 顺式异构体相对含量的测定

# A.5.1 方法原理

试样用四氢呋喃、乙醇溶解，使用高效液相色谱分离，紫外检测器检测，根据相对保留时间定性，按顺式峰面积占顺式、反式峰面积总和的百分比测定试样中维生素 $\mathrm{K}_2$ 顺式异构体的相对含量。

# A.5.2 试剂和溶液

A.5.2.1 无水乙醇：色谱纯。  
A.5.2.2 甲醇：色谱纯。  
A.5.2.3 四氢呋喃：色谱纯。

A.5.2.4 维生素 $\mathrm{K}_2$ 标准品：七烯甲萘醌（MK-7，分子式： $\mathrm{C_{46}H_{64}O_2}$ ，CAS号：2124-57-4），纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
A.5.2.5 维生素 $\mathrm{K}_2$ 标准储备溶液：称取 $40\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )维生素 $\mathrm{K}_2$ 标准品(A.5.2.4)于 $100~\mathrm{mL}$ 棕色容量瓶中，加入 $2\mathrm{mL}$ 四氢呋喃(A.5.2.3)摇动溶解，用无水乙醇(A.5.2.1)定容至刻度，混匀，过滤膜，作为储备溶液，于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 避光保存，建议保存期不超过1个月。  
A.5.2.6 维生素 $\mathrm{K}_2$ 标准溶液：吸取 $1.0\mathrm{mL}$ 维生素 $\mathrm{K}_2$ 储备溶液(A.5.2.5)至 $10\mathrm{mL}$ 棕色容量瓶中，用无水乙醇(A.5.2.1)定容至刻度，混匀，过滤膜(注意避光，在配好之后立即进样）。  
A.5.2.7 试样溶液：称取 $40\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )试样于 $100~\mathrm{mL}$ 棕色容量瓶中，加入 $2\mathrm{mL}$ 四氢呋喃(A.5.2.3)摇动溶解，用无水乙醇(A.5.2.1)定容至刻度。吸取 $1.0~\mathrm{mL}$ 该溶液至 $10~\mathrm{mL}$ 棕色容量瓶中，用无水乙醇(A.5.2.1)定容至刻度，混匀，过滤膜(注意避光，配好之后立即进样）。  
A.5.2.8 微孔滤膜： $0.45\mu \mathrm{m}$ ，有机系。

# A.5.3 仪器和设备

A.5.3.1 高效液相色谱仪：配有紫外检测器或二极管阵列检测器。  
A.5.3.2 分析天平：感量为 $0.1\mathrm{mg}$

# A.5.4 参考色谱条件

A.5.4.1 色谱柱： $\mathrm{C}_{30}$ 色谱柱， $4.6\mathrm{mm}\times 25\mathrm{cm}$ ，粒径 $5\mu \mathrm{m}$ ，或等效色谱柱。  
A.5.4.2 流动相：水：无水乙醇：甲醇：四氢呋喃 $= 1:15:80:10$ 。  
A.5.4.3 流速： $0.8\mathrm{mL / min}$   
A.5.4.4 柱温： $25^{\circ}C$   
A.5.4.5 进样量： $20~\mu \mathrm{L}$   
A.5.4.6 检测波长： $268~\mathrm{nm}$   
A.5.4.7 检测时间：至少1.5倍于维生素 $\mathrm{K}_2$ 反式异构体峰的保留时间。

# A.5.5 分析步骤

# A.5.5.1 系统适应性

取维生素 $\mathrm{K}_2$ 标准溶液(A.5.2.6)注入液相色谱仪中，记录色谱图，连续进样6次，维生素 $\mathrm{K}_2$ 峰面积响应值的相对标准偏差(RSD)应不大于 $3.0\%$ 。维生素 $\mathrm{K}_2$ 反式异构体和维生素 $\mathrm{K}_2$ 顺式异构体之间的分离度 $R$ 应不小于1.5。

# A.5.5.2 试样测定

系统适应性合格后，取试样溶液(A.5.2.7) $20~\mu \mathrm{L}$ 注入液相色谱仪中进行测定，以保留时间定性，按顺式峰面积占顺式、反式峰面积总和的百分比测定试样中维生素 $\mathbf{K}_2$ 顺式异构体的相对含量。参考色谱图见附录B中图B.2。

# A.5.6 结果计算

维生素 $\mathrm{K}_2$ 顺式异构体相对含量 $W_{2}$ 以相对面积 $/\%$ 计，按公式(A.2)计算：

$$
W _ {2} = \frac {A _ {\mathrm {C}}}{A _ {\mathrm {T}} + A _ {\mathrm {C}}} \times 1 0 0 \tag {A.2}
$$

式中：

$A_{\mathrm{C}}$ ——试样中维生素 $\mathrm{K}_{2}$ 顺式异构体对应的峰面积；

$A_{\mathrm{T}}$ ——试样中维生素 $\mathrm{K}_{2}$ 反式异构体对应的峰面积；

100 ——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的2次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

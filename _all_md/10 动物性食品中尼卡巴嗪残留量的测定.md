# 液相色谱-串联质谱法

# 1 范围

本标准规定了动物性食品中尼卡巴嗪残留标志物4,4-二硝基苯缩脲残留量检测的制样和液相色谱-串联质谱测定方法。

本标准适用于鸡的肌肉组织和鸡蛋中尼卡巴嗪残留标志物4, 4'-二硝基均二苯脲残留量的检测。

# 3原理

试料中残留的4,4-二硝基苯缩脲，用乙腈提取，正己烷除脂， $75\%$ 甲醇水溶液萃取，液相色谱-串联质谱法测定，内标法定量。

# 4 试剂和材料

以下所用的试剂，除特别注明者外均为分析纯试剂；水为符合GB/T6682规定的一级水。

4.1 4,4'-二硝基均二苯脲对照品：含量 $\geqslant 98.0\%$   
4.2 4,4'-二硝基均二苯脲- $\mathrm{D_8}$ 对照品：含量 $\geqslant 98.0\%$   
4.3 乙腈：色谱纯。  
4.4 甲醇：色谱纯。  
4.5 无水硫酸钠  
4.6 正己烷  
4.7 乙酸铵

# 4.8 二甲基甲酰胺

4.9 $0.1\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液：取乙酸铵 $1.93\mathrm{g}$ ，用水溶解并稀释至 $250\mathrm{mL}$ 。  
4.10 $75\%$ 甲醇水溶液：取甲醇 $75~\mathrm{mL}$ ，用水溶解并稀释至 $100~\mathrm{ml}$ 。

4.11 $75 \%$ 甲醇水溶液饱和的正己烷 取 $75 \%$ 甲醇水溶液 $100 \mathrm{~mL}$ , 加正己烷 $100 \mathrm{~mL}$ , 摇匀, 静置分层,取上层液。

4.12 $1 \mathrm{mg} / \mathrm{mL} 4, 4'$ -二硝基均二苯脲标准贮备液 精密称取 $4, 4'$ -二硝基均二苯脲对照品 $10 \mathrm{mg}$ , 于 $10 \mathrm{~mL}$ 量瓶内, 用二甲基甲酰胺溶解并稀释至刻度, 配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的 $4, 4'$ -二硝基均二苯脲标准贮备液。 $2 \sim 8^{\circ} \mathrm{C}$ 保存, 有效期3个月。  
4.13 $10 \mu \mathrm{g} / \mathrm{mL} 4, 4^{\prime}$ -二硝基均二苯脲标准工作液: 精密量取 $1 \mathrm{mg} / \mathrm{mL} 4, 4^{\prime}$ -二硝基均二苯脲标准贮备液 $1.0 \mathrm{~mL}$ , 于 $100 \mathrm{~mL}$ 量瓶中, 用甲醇溶解并稀释至刻度, 配制成浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 的 $4, 4^{\prime}$ -二硝基均二苯脲标准工作液。 $-18^{\circ} \mathrm{C}$ 保存, 有效期3个月。  
4.14 $1 \mathrm{mg} / \mathrm{mL} 4, 4^{\prime}$ -二硝基均二苯脲- $\mathrm{D}_{8}$ 标准贮备液: 精密称取 $4, 4^{\prime}$ -二硝基均二苯脲- $\mathrm{D}_{8}$ 对照品 $10 \mathrm{mg}$ ,于 $10 \mathrm{~mL}$ 量瓶内, 用二甲基甲酰胺溶解并稀释至刻度, 配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的 $4, 4^{\prime}$ -二硝基均二苯脲- $\mathrm{D}_{8}$ 标准贮备液。 $2 \sim 8^{\circ} \mathrm{C}$ 保存, 有效期3个月。  
4.15 $10 \mu \mathrm{g} / \mathrm{mL} 4, 4^{\prime}$ -二硝基均二苯脲- $\mathrm{D}_{8}$ 标准工作液: 精密量取 $1 \mathrm{mg} / \mathrm{mL} 4, 4^{\prime}$ -二硝基均二苯脲- $\mathrm{D}_{8}$ 标准贮备液 $1.0 \mathrm{~mL}$ , 于 $100 \mathrm{~mL}$ 量瓶中, 用甲醇溶解并稀释至刻度, 配制成浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 的 $4, 4^{\prime}$ -二硝基均二苯脲- $\mathrm{D}_{8}$ 标准工作液。- $18^{\circ} \mathrm{C}$ 保存, 有效期3个月。

# 5 仪器和设备

5.1 液相色谱-串联质谱仪：配电喷雾离子源。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 涡旋振荡器  
5.5 离心机  
5.6 氮吹仪   
5.7 离心管： $50~\mathrm{mL}$   
5.8 滤膜： $0.2\mu \mathrm{m}$ 。

# 6 试料的制备与保存

# 6.1 试料的制备

# 6.1.1 鸡蛋

取适量新鲜或冷藏的空白或供试鸡蛋，去壳，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.1.2 鸡肌肉

取适量新鲜或冷冻的空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-20°C以下保存。

# 7 测定步骤

# 7.1 提取与净化

称取试料（ $2 \pm 0.02$ ）g，于 $50 \mathrm{~mL}$ 离心管中，添加 $10 \mu \mathrm{g} / \mathrm{mL} 4,4^{\prime}$ -二硝基均二苯脲- $\mathrm{D}_{8}$ 标准工作液适量，加无水硫酸钠 $2 \mathrm{~g}$ ，乙腈 $8 \mathrm{~mL}$ ，涡旋 $0.5 \mathrm{~min}$ ，超声 $5 \mathrm{~min}$ ， $5000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取上清液，于 $40^{\circ} \mathrm{C}$ 氮气吹干，加 $75 \%$ 甲醇水溶液饱和的正己烷 $1 \mathrm{~mL}$ ，涡旋 $10 \mathrm{~s}$ ，再加 $75 \%$ 甲醇水溶液 $1.0 \mathrm{~mL}$ ，充分涡旋混合，于 $40^{\circ} \mathrm{C}$ 水浴中静置 $5 \mathrm{~min}$ ， $2000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，取下层清液，滤膜过滤，供液相色谱-串联质谱测定。

# 7.2 标准曲线的制备

精密量取 $10 \mu \mathrm{g} / \mathrm{mL} 4, 4^{\prime}$ -二硝基均二苯脲标准工作液和 $10 \mu \mathrm{g} / \mathrm{mL} 4, 4^{\prime}$ -二硝基均二苯脲- $\mathrm{D}_{8}$ 标准工作液适量，用 $75\%$ 甲醇水溶液稀释，配制成 $4, 4^{\prime}$ -二硝基均二苯脲- $\mathrm{D}_{8}$ 浓度均为 $100 \mathrm{ng} / \mathrm{mL}$ 以及 $4, 4^{\prime}$ -二硝基均二苯脲浓度为2、10、20、50、200和 $500 \mathrm{ng} / \mathrm{mL}$ 系列对照溶液，供液相色谱-串联质谱法测定。以特征离子质量色谱峰面积为纵坐标，标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.3 测定

# 7.3.1 液相色谱条件

色谱柱： $\mathrm{C_{18}}$ （ $150 \mathrm{~mm} \times 2.1 \mathrm{~mm}$ ，粒径 $5 \mu \mathrm{m}$ ），或相当者；

流动相：甲醇 $+0.1 \mathrm{~mol} / \mathrm{L}$ 乙酸铵溶液（ $75 + 25$ ， $\mathrm{v} / \mathrm{v}$ ）；

流速： $0.2\mathrm{mL / min}$

柱温： $30^{\circ}\mathrm{C}$ ；

进样量： $20~\mu \mathrm{L}$

# 7.3.2 质谱条件

离子源：电喷雾离子源；

扫描方式：负离子扫描；

检测方式：多反应监测；

电离电压： $3.0\mathrm{kV}$

源温： $110^{\circ}\mathrm{C}$

雾化温度： $350^{\circ}C$ 。

锥孔气流速： $50\mathrm{L / h}$

# 8 结果计算和表述

单点校准: $C = \frac{A A_{i S}^{\prime} C_{S} C_{i S}}{A_{i S} A_{S} C_{i S}^{\prime}}$

或标准曲线校准: 由 $\frac{A_{S}}{A_{i S}^{\prime}} = a \frac{C_{S}}{C_{i S}^{\prime}} + b$

求得 $a$ 和 $b$ ，则 $C = \frac{C_{iS}}{a} \left( \frac{A}{A_{iS}} - b \right)$

试料中DNC残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算：

$$
X = \frac {C V}{m}
$$

式中：

$X$ ——供试试料中 $4, 4^{\prime}$ -二硝基均二苯脲的残留量， $\mu \mathrm{g} / \mathrm{kg}$ ；

C——试样溶液中4, 4′-二硝基均二苯脲的浓度，ng/mL；

$C_{is}$ ——试样溶液中4, $4^{\prime}$ -二硝基均二苯脲- $\mathrm{D}_{8}$ 的浓度， $\mathrm{ng/mL}$ ；

$C_{s}$ ——对照溶液中 $4,4^{\prime}$ -二硝基均二苯脲的浓度， $\mathrm{ng/mL}$ ；

$C_{is}^{\prime}$ ——对照溶液中4, $4^{\prime}$ -二硝基均二苯脲- $\mathrm{D}_{8}$ 的浓度， $\mathrm{ng / mL}$

$A$ ——试样溶液中 $4,4^{\prime}$ -二硝基均二苯脲的峰面积；

$A_{is}$ ——试样溶液中4, 4'-二硝基均二苯脲-D8的峰面积；

$A_{s}$ ——对照溶液中 $4,4^{\prime}$ -二硝基均二苯脲的峰面积；

$A_{is}^{\prime}$ ——对照溶液中4,4'-二硝基均二苯脲- $\mathrm{D}_{8}$ 的峰面积；

$V$ ——溶解残余物所用 $75 \%$ 甲醇水溶液的体积, $\mathrm{mL}$ ;

$m$ ——供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度和精密度

# 9.1 灵敏度

本方法的检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1\mu \mathrm{g / kg}$

# 9.2 准确度

本方法鸡蛋试料在 $1\sim 10~\mu \mathrm{g / kg}$ 添加浓度、鸡肌肉试料在 $1\sim 300~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $80\% \sim 120\%$ 。

# 9.3 精密度

本方法的批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

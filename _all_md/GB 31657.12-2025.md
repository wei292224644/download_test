# 1 范围

本文件规定了蜂产品中氨基糖苷类药物残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于蜂蜜和蜂王浆中链霉素、双氢链霉素、庆大霉素、新霉素、卡那霉素、安普霉素、大观霉素、潮霉素、妥布霉素残留量的测定。

# 4 原理

蜂蜜和蜂王浆中残留的药物用磷酸盐缓冲液提取，提取液分成两等份后分别经混合型强阳离子和混合型弱阳离子交换固相萃取柱净化，复溶液用液相色谱-串联质谱测定、外标法定量。

# 5 试剂和材料

以下所用试剂,除特殊注明外均为分析纯试剂,水为符合 GB/T 6682 规定的一级水。

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯  
5.1.2 胰蛋白酶 $\mathrm{(C_6H_{12}O_{12}P_3)}$ ： $1000\mathrm{U}$ mg～2000U，mg  
5.1.3 七氟丁酸 $\mathrm{(C_3F_7CO)(OH)}$ ：HFBA色谱纯。  
5.1.4 乙酸 $\mathrm{(CH_3COOH)}$ ：色谱纯。  
5.1.5 乙酸铵 $\mathrm{(CH_3COONH_4)}$ ：色谱纯。  
5.1.6 甲酸（HCOOH）：色谱纯。  
5.1.7 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.8 乙二胺四乙酸二钠 $\mathrm{(Na_2EDTA)}$   
5.1.9 三氯乙酸 $(\mathrm{CCl}_3\mathrm{COOH})$   
5.1.10 氢氧化钠 $(\mathrm{NaOH})$   
5.1.11 氨水 $\mathrm{(NH_4OH)}$   
5.1.12 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$

# 5.2 溶液配制

5.2.1 $50\%$ 甲醇溶液：取甲醇 $10~\mathrm{mL}$ ，用水稀释至 $20~\mathrm{mL}$   
5.2.2 $0.3\%$ 乙酸水溶液：取乙酸 $300~\mu \mathrm{L}$ ，用水稀释至 $100~\mathrm{mL}$

5.2.3 $0.3\%$ 乙酸水-甲醇溶液：取 $0.3\%$ 乙酸水溶液 $5\mathrm{mL}$ ，用甲醇稀释至 $10~\mathrm{mL}$ 。  
5.2.4 磷酸盐缓冲液（含 $5\%$ 三氯乙酸和 $0.4 \mathrm{mmol} / \mathrm{L}$ 乙二胺四乙酸二钠）：取磷酸二氢钾 $1.36 \mathrm{~g}$ 、乙二胺四乙酸二钠 $0.15 \mathrm{~g}$ 和三氯乙酸 $50 \mathrm{~g}$ ，用水溶解并稀释至 $1000 \mathrm{~mL}$ 。  
5.2.5 氢氧化钠溶液 $(5\mathrm{mol} / \mathrm{L})$ ：取氢氧化钠 $20\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ ，混匀。  
5.2.6 乙酸甲醇溶液：取乙酸 $10\mathrm{mL}$ 和甲醇 $70\mathrm{mL}$ ，用水稀释至 $100\mathrm{mL}$ ，混匀。  
5.2.7 氨水甲醇溶液：取氨水 $20\mathrm{mL}$ 和甲醇 $60\mathrm{mL}$ ，用水稀释至 $100\mathrm{mL}$ ，混匀。  
5.2.8 乙酸铵溶液 $(0.5\mathrm{mol} / \mathrm{L})$ ：取乙酸铵 $3.85\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
5.2.9 0.1%甲酸溶液(含 $0.5\mathrm{mmol / L}$ 乙酸铵):取甲酸 $1\mathrm{mL}$ 和 $0.5\mathrm{mol / L}$ 乙酸铵溶液 $1\mathrm{mL}$ ,加水稀释至 $1000~\mathrm{mL}$ ,混匀。

# 5.3 标准品及其纯度

链霉素、双氢链霉素、庆大霉素、新霉素、卡那霉素、安普霉素、大观霉素、潮霉素和妥布霉素，具体见附录A。

# 5.4 标准溶液配制

5.4.1 标准储备液：分别准确称取11种氨基糖苷类药物标准品 $10\mathrm{mg}$ ，用 $0.3\%$ 乙酸水-甲醇溶液溶解，并定容至 $10\mathrm{mL}$ ，配制成 $1000\mu \mathrm{g} / \mathrm{mL}$ 的标准储备液，放置于塑料材质容器中。于 $-18^{\circ}\mathrm{C}$ 避光保存，有效期3个月。  
5.4.2 混合标准工作液: 精密量取 11 种氨基糖苷类药物标准储备液各 $125 \mu \mathrm{L}$ , 于 $25 \mathrm{~mL}$ 容量瓶中, 用 $0.3 \%$ 乙酸水溶液定容至刻度, 配制成浓度为 $5 \mu \mathrm{g} / \mathrm{mL}$ 的氨基糖苷类药物混合标准工作液, 放置于塑料材质容器中。于 $-18^{\circ} \mathrm{C}$ 避光保存, 有效期 1 个月。

5.5.1 混合型强阳离子交换固相萃取柱： $200\mathrm{mg / 6mL}$ ，或相当者。  
5.5.2 混合型弱阳离子交换固相萃取柱： $150\mathrm{mg / 3mL}$ ，或相当者。  
5.5.3 聚醚砜滤膜： $0.22\mu \mathrm{m}$

6.4 水浴振荡器。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试蜂产品,将其搅拌均匀。对有结晶的蜂蜜样品,在密闭情况下,置于不超过 $60^{\circ} \mathrm{C}$ 的水浴中解晶,待样品全部融化后搅匀,冷却至室温。制备好的试样置于样品瓶中,密封,并做上标记。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；

c) 取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

蜂蜜样品于室温下避光保存；蜂王浆样品于 $-18^{\circ}C$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取蜂蜜试料 $(5\pm 0.05)\mathrm{g}$ ，蜂王浆试料 $(2\pm 0.05)\mathrm{g}$ 于 $50~\mathrm{mL}$ 具塞塑料离心管中（蜂王浆试料中预先加入 $0.25\mathrm{g}$ 胰蛋白酶和 $5\mathrm{mL}$ 水，混匀， $37^{\circ}C$ 水浴反应 $2\textrm{h})$ ，加入 $10~\mathrm{mL}$ 磷酸盐缓冲液，涡旋振荡 $1\mathrm{min}$ ，超声 $5\mathrm{min}$ ，于 $4^{\circ}C$ 下 $8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液于另一塑料刻度离心管中，残渣加 $5\mathrm{mL}$ 磷酸盐缓冲液重复提取1次，合并上清液，用磷酸盐缓冲液定容至 $20~\mathrm{mL}$ ，备用。

# 8.2 净化

依次用甲醇、水各 $5\mathrm{mL}$ 分别活化混合型强阳离子交换固相萃取柱、混合型弱阳离子交换固相萃取柱。备用液分成两等份，分别为备用液A和备用液B。取备用液A过混合型强阳离子交换固相萃取柱，再依次用超纯水、甲醇各 $7.5\mathrm{mL}$ 淋洗抽干、 $5\mathrm{mL}$ 氨水甲醇溶液洗脱，用于分析新霉素、卡那霉素、安普霉素、大观霉素、潮霉素和妥布霉素。取备用液B用氢氧化钠溶液调节 $\mathsf{pH}$ 为7.5，过混合型弱阳离子交换固相萃取柱，用 $7.5\mathrm{mL}$ 水淋洗抽干；用 $5\mathrm{mL}$ 乙酸甲醇溶液洗脱，用于分析链霉素、双氢链霉素和庆大霉素。两部分洗脱液分别于 $40^{\circ}C$ 下氮吹至近干，用 $2\mathrm{mL}0.3\%$ 乙酸水HFBA溶液（99：1）复溶，复溶液经 $0.22\mu \mathrm{m}$ 滤膜过滤后放入塑料进样瓶中，供液相色谱串联质谱测定。

# 8.3 基质匹配标准曲线的制备

精密量取混合标准工作液适量，分别加入经提取和净化的蜂蜜或蜂王浆空白试料残渣中， $40^{\circ}\mathrm{C}$ 水浴氮气吹干，加入 $2\mathrm{mL}0.3\%$ 乙酸水HFBA溶液 $(99:1)$ 涡旋溶解残余物，配制成浓度为 $5.0~\mu \mathrm{g / L},10.0~\mu \mathrm{g / L},$ $50.0~\mu \mathrm{g / L},100.0~\mu \mathrm{g / L},200.0~\mu \mathrm{g / L}$ 和 $500.0~\mu \mathrm{g / L}$ 的基质匹配系列混合标准溶液，微孔滤膜过滤，供液相色谱-串联质谱测定。以待测物的峰面积为纵坐标、相应的标准溶液浓度为横坐标，绘制基质匹配标准曲线，计算回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在相同试验条件下，试样溶液中氨基糖苷类药物的保留时间与基质匹配标准溶液中对应的保留时间的偏差在 $\pm 0.1\mathrm{min}$ 以内；且样品中各组分定性离子的相对丰度与浓度接近的基质匹配标准溶液中对应的定性离子的相对丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.4.3.2 定量测定

分别取适量试样溶液和相应浓度的基质匹配标准工作液，做单点校准或多点校准，以色谱峰峰面积积分值定量。基质匹配标准工作液及试样液中药物的响应值均应在仪器检测的线性范围内，试样液进样过程中应穿插标准工作液，以便准确定量。对于试料中氨基糖苷类药物残留量超过仪器测定线性范围的，应进行适当倍数稀释后再进行分析。氨基糖苷类药物基质匹配标准溶液的特征离子质量色谱图见附录B。

# 8.5 空白试验

取空白试样，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中氨基糖苷类药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {s}} \times A \times V \times 1 0 0 0}{A _ {\mathrm {s}} \times m \times 1 0 0 0} \times f \tag {1}
$$

式中：

$X$ ——试样中相应的氨基糖苷类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{s}}$ ——基质匹配标准溶液中相应的氨基糖苷类药物浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

A ——试样中相应的氨基糖苷类药物峰面积；

$V$ ——试样最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$A_{s}$ ——基质匹配标准溶液中相应的氨基糖苷类药物峰面积；

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$ ；

$f$ ——稀释因子，公式中为2。

注：庆大霉素残留量以庆大霉素 ${\mathrm{C}}_{1}$ 、庆大霉素 ${\mathrm{C}}_{1}$ 、庆大霉素 ${\mathrm{C}}_{2} + {\mathrm{C}}_{4}$ 总和计

计算结果以平行测定结果的算术平均值表示，保留2位有效数字。

# 10 检测方法的灵敏度、正确度和精密度

# 10.1 灵敏度

本方法蜂蜜中链霉素、双氧链霉素、卡那霉素、庆大霉素 $\mathrm{C_1}$ 、庆大霉素 $\mathrm{C_2 + C_3}$ 、潮霉素检出限为 $2.5\mu \mathrm{g / kg}$ ，定量限为 $5\mu \mathrm{g / kg}$ ；新霉素、妥普霉素、妥布霉素、庆大霉素 $\mathrm{C_4}$ 、大观霉素检出限为 $10\mu \mathrm{g / kg}$ ，定量限为 $25\mu \mathrm{g / kg}$ 。

蜂王浆中链霉素、双氢链霉素、卡那霉素、庆大霉素 $C_1$ 、庆大霉素 $C_2 - C_3$ 、潮霉素检出限为 $5\mu \mathrm{g} / \mathrm{kg}$ 定量限为 $10~\mu \mathrm{g} / \mathrm{kg}$ ；安普霉素、新霉素、妥布霉素、庆大霉素 $C_4$ 、大观霉素检出限为 $12.5\mu \mathrm{g} / \mathrm{kg}$ ，定量限为 $25~\mu \mathrm{g} / \mathrm{kg}$ 。

# 10.2 正确度

本方法蜂蜜在 $5\mu \mathrm{g / kg}\sim 100\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$ ，蜂王浆在 $10~\mu \mathrm{g / kg}\sim$ $100~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $= 1.3\%$ ，批间相对标准偏差 $= 20\%$ 。

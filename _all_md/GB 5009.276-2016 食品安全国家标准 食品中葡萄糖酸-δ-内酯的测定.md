# 前言

本标准代替GB/T9695.17—2008《肉与肉制品 葡萄糖酸-δ-内酯含量的测定》。

本标准与GB/T9695.17—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中葡萄糖酸-δ-内酯的测定”；  
——增加了第一法的检出限、定量限；  
——修改了5.1中试样制备的方法；  
——增加了“第二法 高效液相色谱法”。

# 食品安全国家标准

# 食品中葡萄糖酸-δ-内酯的测定

# 1 范围

本标准规定了食品中葡萄糖酸-δ-内酯含量的测定方法。

本标准第一法适用于肉与肉制品、豆制品、饮料中葡萄糖酸-δ-内酯的测定，第二法适用于食品中葡萄糖酸-δ-内酯含量的测定。

# 第一法 分光光度计法

# 2 原理

用沸水提取试样中的葡萄糖酸-δ-内酯，过滤上清液，在氢氧化钾作用下葡萄糖酸-δ-内酯转化成葡萄糖酸盐。葡萄糖酸盐通过葡萄糖酸激酶和6-磷酸葡萄糖脱氢酶的作用，产生5-磷酸核酮糖并使烟酰胺腺嘌呤二核苷酸磷酸(NADP)还原。用分光光度计测量还原型烟酰胺腺嘌呤二核苷酸磷酸的吸光度，然后计算出葡萄糖酸-δ-内酯的含量。

$$
\text {葡 萄 糖 酸 盐} + \text {腺 嘌 吟 核 苷 三 磷 酸} \xrightarrow {\text {葡 萄 糖 酸 激 酶}} 6 - \text {磷 酸 葡 萄 糖 酸 盐} + \text {二 磷 酸 豚 苷}
$$

$$
6 - \text {磷 酸 葡 萄 糖 酸 盐} + \mathrm {N A D P} \xrightarrow {6 - \text {磷 酸 葡 萄 糖 脱 氢 酶}} 5 - \text {磷 酸 核 酮 糖} + \mathrm {N A D P H} + \mathrm {C O} _ {2} + \mathrm {H} ^ {+}
$$

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 无水乙醇 $\mathrm{(C_2H_6O)}$   
3.1.2 石油醚：沸程 $30^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$   
3.1.3 氢氧化钾(KOH)。  
3.1.4 双甘氨肽 $\mathrm{(C_4H_8N_2O_3)}$   
3.1.5 氯化镁 $(\mathrm{MgCl}_2)$ 。  
3.1.6 碳酸氢钠 $(\mathrm{NaHCO}_3)$ 。  
3.1.7 烟酰胺腺嘌呤二核苷酸磷酸钠盐 $\left(\mathrm{C}_{21} \mathrm{H}_{27} \mathrm{~N}_{7} \mathrm{NaO}_{17} \mathrm{P}_{3}\right)$ 。  
3.1.8 腺嘌呤核苷三磷酸钠盐 $\left(\mathrm{C}_{10} \mathrm{H}_{14} \mathrm{~N}_{5} \mathrm{Na}_{2} \mathrm{O}_{13} \mathrm{P}_{3}\right)$ 。  
3.1.9 6-磷酸葡萄糖脱氢酶(活力， $110\mathrm{U / mg})$ 。  
3.1.10 葡萄糖酸激酶（活力， $13\mathrm{U / mg})$ ）。

# 3.2 试剂配制

3.2.1 氢氧化钾溶液 $(2\mathrm{mol} / \mathrm{L})$ ：称取 $11.22\mathrm{g}$ 于 $50~\mathrm{mL}$ 小烧杯中，加水溶解后转移至 $100~\mathrm{mL}$ 容量瓶，

定容。

3.2.2 缓冲液：分别称取双甘氨肽 $2.64\mathrm{g}$ 和氯化镁 $0.284\mathrm{g}$ 溶于 $150~\mathrm{mL}$ 水中，用氢氧化钾溶液调 $\mathrm{pH}$ 至8.0，定容到 $200~\mathrm{mL}$ 容量瓶（可在 $4^{\circ}\mathrm{C}$ 下保存4周）。  
3.2.3 烟酰胺腺嘌呤二核苷酸磷酸盐(NADP)溶液 $(10\mathrm{mg / mL})$ ：称取烟酰胺腺嘌呤二核苷酸磷酸钠盐 $50~\mathrm{mg}$ 于带盖的小瓶中，用 $5.0~\mathrm{mL}$ 水溶解（可在 $4^{\circ}C$ 下保存4周）。  
3.2.4 腺嘌呤核苷三磷酸(ATP)溶液 $(50\mathrm{mg / mL})$ : 称取腺嘌呤核苷三磷酸钠盐 $250\mathrm{mg}$ 和碳酸氢钠 $250\mathrm{mg}$ 于带盖的小瓶中, 加入 $5.0\mathrm{mL}$ 水溶解 (可在 $4^{\circ}\mathrm{C}$ 下保存 4 周)。  
3.2.5 6-磷酸葡萄糖脱氢酶(6-PGDH)溶液 $(2.0\mathrm{mg / mL})$ ：每毫升6-磷酸葡萄糖脱氢酶(6-PGDH)悬液中含有6-PGDH $2.0~\mathrm{mg}$ ，葡萄糖酸激酶和还原型NADP氧化酶不超过 $0.05\%$ （可在 $4^{\circ}C$ 下保存6个月）。  
3.2.6 葡萄糖酸激酶(GK)溶液 $(1.0\mathrm{mg / mL})$ ：每毫升葡萄糖酸激酶(GK)悬液中含有 $\mathrm{GK}1.0\mathrm{mg}$ NADP氧化酶含量不超过 $0.05\%$ （可在 $4^{\circ}C$ 下保存6个月）。

# 4 仪器和设备

4.1 紫外可见分光光度计。  
4.2 分析天平：感量 $1\mathrm{mg}$ 和 $0.1\mathrm{mg}$   
4.3 匀质机。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 肉与肉制品、豆制品等固体样品

取有代表性的样品不少于 $200\mathrm{g}$ ，用匀质机制成匀浆。制成后的试样应尽快分析，若不立即分析，应密封冷冻贮存。贮存的试样在启用时应重新混匀。

# 5.1.2 饮料等液体样品

摇匀后直接取样。

# 5.2 提取

肉制品：称取 $2\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 试样于 $50~\mathrm{mL}$ 烧杯中，加入 $10~\mathrm{mL}$ 石油醚洗涤，用玻璃棒搅拌均匀，静置 $15\mathrm{min}$ 后，用干燥滤纸过滤，重复洗涤三次。再用 $10~\mathrm{mL}$ 无水乙醇洗涤三次。然后将滤纸上残渣转移至原小烧杯中，加 $30~\mathrm{mL}$ 水煮沸，待冷却后用氢氧化钾溶液 $(2\mathrm{mol} / \mathrm{L})$ 调至溶液 $\mathsf{pH}$ 为10，用水定容至 $100~\mathrm{mL}$ 容量瓶。过滤，滤液待上机用。

豆制品：称取 $2\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 试样于 $50~\mathrm{mL}$ 烧杯中，加 $30~\mathrm{mL}$ 水提取，煮沸，冷却后用氢氧化钾溶液 $(2\mathrm{mol} / \mathrm{L})$ 调至溶液 $\mathrm{pH}$ 为10.0，用水定容至 $100~\mathrm{mL}$ 容量瓶。过滤，滤液待上机用。

饮料：过滤，准确吸取 $30~\mathrm{mL}$ 滤液用氢氧化钾溶液 $(2\mathrm{mol} / \mathrm{L})$ 调至溶液 $\mathrm{pH}$ 为10，用水定容至 $100~\mathrm{mL}$ 容量瓶，待上机用。

# 5.3 测定

取两个 $1\mathrm{cm}$ 比色皿，一个为试验皿，一个为空白皿。分别按序加入缓冲液(3.2.2) $2.50~\mathrm{mL}$ 、NADP溶液 $(10\mathrm{mg / mL})0.1\mathrm{mL}$ 和ATP溶液 $(50\mathrm{mg / mL})0.1\mathrm{mL}$ ，加盖混匀；分别在试验皿中加入提取液 $0.2\mathrm{mL}$ ，空白皿中加入水 $0.2\mathrm{mL}$ ，加盖混匀。再分别加入6-PGDH溶液 $(2.0\mathrm{mg / mL})0.05\mathrm{mL}$ ，加盖混

匀后放置 $5\mathrm{min}$ ，以空白为参比在 $365~\mathrm{nm}$ 或 $340~\mathrm{nm}$ 波长处测吸光度。

$A_{1}$ ——试验溶液的吸光度值；

$A_{\mathrm{1B}}$ ——空白溶液的吸光度值。

取GK溶液 $(1.0\mathrm{mg / mL})0.01\mathrm{mL}$ 分别置于上述两个比色皿中，立即混匀。 $10~\mathrm{s}\sim 15~\mathrm{s}$ 内在 $365~\mathrm{nm}$ 或 $340~\mathrm{nm}$ 波长处测吸光度。以后每隔 $2\mathrm{min}$ 测一次，直到吸光度恒定为止。绘制出吸光度值与时间的关系曲线，并查出吸光度最大值。

$A_{2}$ ——试验溶液的吸光度最大值；

$A_{2\mathrm{B}}$ ——空白溶液的吸光度最大值。

# 6 分析结果的表述

试样中葡萄糖酸-δ-内酯含量按式(1)计算：

$$
\begin{array}{l} X = 0. 9 0 8 \Delta A \times \frac {2 . 9 6 \times 1 9 6 . 1}{K \times 0 . 2 \times 1 0 0 0} \times \frac {1 0 0}{1 0 0 0} \times \frac {1 0 0}{m} \\ = 5. 2 7 0 5 \times \frac {\Delta A}{K \times 0 . 2 \times m} \quad \dots \dots \dots \dots \dots \dots \tag {1} \\ \end{array}
$$

式中：

$X$ ——试样中葡萄糖酸-δ-内酯的含量， $\%$ ；

0.908——葡萄糖酸-δ-内酯与D-(+)-葡萄糖酸相对分子量的比值；

$$
\Delta A \quad - (A _ {2} - A _ {1}) - (A _ {2 B} - A _ {1 B});
$$

2.96 ——比色液总体积，单位为毫升 $(\mathrm{mL})$ ；

196.1—D- $(+)$ -葡萄糖酸的相对分子量；

$K$ ——摩尔吸光系数，在 $365 \mathrm{~nm}$ 波长时， $K = 3.50 \mathrm{~cm}^2 / \mu \mathrm{mol}$ ；在 $340 \mathrm{~nm}$ 波长时， $K = 6.23 \mathrm{~cm}^2 / \mu \mathrm{mol}$ ；

0.2 制备比色液所用提取液的体积，单位为毫升 $(\mathrm{mL})$

1000——单位换算系数；

100 提取液的定容体积，单位为毫升 $(\mathrm{mL})$

100 小数与百分数之间的换算系数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当称样量为 $10\mathrm{g}$ 时，方法检出限为 $0.0075\%$ ，定量限为 $0.025\%$ 。

# 第二法 高效液相色谱法

# 9 原理

葡萄糖酸-δ-内酯在水中缓慢水解生成葡萄糖酸和少量的葡萄糖酸-γ-内酯并达到水解平衡。试样经水煮沸、提取、定容、过滤，用高效液相色谱分离。通过待测液中葡萄糖酸的峰面积与葡萄糖酸-δ-内

酯标准水解为葡萄糖酸的峰面积比较，采用外标法定量计算试样中的葡萄糖酸-δ-内酯的含量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$

10.1.2 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。

# 10.2 试剂配制

磷酸二氢钾-磷酸缓冲液：称 $1.36\mathrm{g}$ 磷酸二氢钾加 $0.5\mathrm{mL}$ 磷酸用水溶解后定容至 $1\mathrm{L}$ 。

# 10.3 标准品

葡萄糖酸-δ-内酯 $\mathrm{(C_6H_{10}O_6}$ ，CAS号：90-80-2）：纯度 $>98\%$

# 10.4 标准溶液配制

# 10.4.1 葡萄糖酸-δ-内酯标准储备液（5.0 mg/mL）

准确称取 $0.250\mathrm{g}$ 葡萄糖酸- $\delta$ -内酯标准品（精确至 $0.1\mathrm{mg}$ ），用水加热煮沸提取，冷却后定容至 $50~\mathrm{mL}$ 容量瓶。

# 10.4.2 葡萄糖酸-δ-内酯标准系列工作液

依次吸取 $0.50\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},5.00\mathrm{mL},10.0\mathrm{mL}$ 葡萄糖酸-δ-内酯标准储备液 $(5.0\mathrm{mg}/$ mL)于 $100.0~\mathrm{mL}$ 容量瓶中，用磷酸二氢钾-磷酸缓冲液分别定容至刻度，得到浓度分别为 $0.025\mathrm{mg}/$ $\mathrm{mL},0.05\mathrm{mg / mL},0.10\mathrm{mg / mL},0.25\mathrm{mg / mL},0.50\mathrm{mg / mL}$ 的葡萄糖酸-δ-内酯标准系列工作液。

# 11 仪器和设备

11.1 高效液相色谱仪，配示差折光检测器。  
11.2 天平：感量 $1\mathrm{mg}$ 和 $0.1\mathrm{mg}$   
11.3 匀质机。

# 12 分析步骤

# 12.1 试样制备

固体或半固体样品使用匀质机粉碎，液体样品用匀浆机打成匀浆。制成后的试样应尽快分析，若不立即分析，应密封冷冻贮存。贮存的试样在启用时应重新混匀。

# 12.2 提取

称取 $2\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 试样于 $50~\mathrm{mL}$ 烧杯中，加 $30~\mathrm{mL}$ 磷酸二氢钾-磷酸缓冲液煮沸提取，冷却后用磷酸二氢钾-磷酸缓冲液定容至 $100~\mathrm{mL}$ 容量瓶，放置 $2\textrm{h}$ 后经 $0.45\mu \mathrm{m}$ 水相微孔滤膜过滤，滤液待上机用。

# 12.3 仪器参考条件

12.3.1 色谱柱：有机酸柱，柱长 $4.6 \mathrm{~mm}$ ，内径 $250 \mathrm{~mm}$ ，膜厚 $5 \mu \mathrm{m}$ ，或分离效果相当的其他硅胶键合柱。  
12.3.2 柱温： $30^{\circ}C$ 。  
12.3.3 示差检测器检测池温度： $35^{\circ}C$   
12.3.4 流动相：磷酸二氢钾-磷酸缓冲液。  
12.3.5 流速： $0.8\mathrm{mL / min}$   
12.3.6 进样量： $10~\mu \mathrm{L}$

# 12.4 标准曲线的制作

将标准系列工作液分别注入高效液相色谱仪中，测定相应的葡萄糖酸峰面积，以标准系列工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。葡萄糖酸-δ-内酯标准的色谱图见图A.1。

# 12.5 试样溶液的测定

将试样溶液注入高效液相色谱仪中，得到葡萄糖酸峰面积，与葡萄糖酸-δ-内酯标准水解为葡萄糖酸的峰面积比较，根据标准曲线得到待测液中的葡萄糖酸-δ-内酯的含量。

# 13 分析结果的表述

试样中葡萄糖酸-δ-内酯含量按式(2)计算：

$$
X = \frac {\rho \times V}{1 0 0 0 \times m} \times 1 0 0 \tag {2}
$$

式中：

$X$ ——试样中葡萄糖酸-δ-内酯的含量， $\%$ ；

$\rho$ ——试样溶液中葡萄糖酸-δ-内酯的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V$ ——提取液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

当称样量为 $10\mathrm{g}$ 时，方法检出限为 $0.0075\%$ ，定量限为 $0.025\%$ 。

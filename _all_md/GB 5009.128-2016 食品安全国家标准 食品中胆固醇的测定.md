2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替GB/T5009.128—2003《食品中胆固醇的测定》、GB/T22220—2008《食品中胆固醇的测定 高效液相色谱法》和GB/T9695.24—2008《肉与肉制品 胆固醇含量测定》。

本标准与GB/T5009.128—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中胆固醇的测定”；  
——增加了气相色谱法作为第一法，高效液相色谱法作为第二法；比色法改为第三法；  
——修改了GB/T9695.24—2008气相色谱法的前处理方法中提取溶剂、无水乙醇的添加量和定容体积。

# 食品安全国家标准

# 食品中胆固醇的测定

# 1 范围

本标准规定了食品中胆固醇的测定方法。

本标准适用于食品中胆固醇的测定，第一法气相色谱法适用于肉及肉制品、蛋及蛋制品、乳及乳制品等各类动物性食品以及植物油脂中胆固醇的测定；第二法高效液相色谱法适用于肉及肉制品、蛋及蛋制品、乳及乳制品等各类动物性食品中胆固醇的测定；第三法比色法适用于肉及肉制品、蛋及蛋制品等动物性食品中胆固醇的测定。

# 第一法 气相色谱法

# 2 原理

样品经无水乙醇-氢氧化钾溶液皂化，石油醚和无水乙醚混合提取，提取液浓缩至干，无水乙醇溶解定容后，采用气相色谱法检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 无水乙醇 $\mathrm{(C_2H_5OH)}$   
3.1.3 石油醚：沸程 $30^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$   
3.1.4 无水乙醚 $(\mathrm{C_4H_{10}O})$ 。  
3.1.5 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.6 氢氧化钾(KOH)。

# 3.2 试剂配制

3.2.1 $60\%$ 氢氧化钾溶液：称取 $60\mathrm{g}$ 氢氧化钾，缓慢加水溶解，并定容至 $100~\mathrm{mL}$   
3.2.2 石油醚-无水乙醚混合液 $(1 + 1$ ，体积比)：将石油醚和无水乙醚等体积混合均匀。

# 3.3 标准品

胆固醇标准品 $\mathrm{(C_{27}H_{46}O}$ ，CAS号：57-88-5）：纯度 $\geqslant 99\%$

# 3.4 标准溶液配制

# 3.4.1 胆固醇标准储备液（ $1.0\mathrm{mg / mL}$ ）

称取胆固醇标准品 $0.05\mathrm{g}$ （精确至 $0.1\mathrm{mg}$ ），用无水乙醇溶解并定容至 $50~\mathrm{mL}$ ，放置 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 密封

可贮藏半年。

# 3.4.2 胆固醇标准系列工作液

分别吸取标准储备液 $(1.0\mathrm{mg / mL})25\mu \mathrm{L},50\mu \mathrm{L},100\mu \mathrm{L},500\mu \mathrm{L},2000\mu \mathrm{L}$ ，用无水乙醇定容至 $10~\mathrm{mL}$ 该标准系列工作液的浓度分别为 $2.5\mu \mathrm{g} / \mathrm{mL},5\mu \mathrm{g} / \mathrm{mL},10\mu \mathrm{g} / \mathrm{mL},50\mu \mathrm{g} / \mathrm{mL},200\mu \mathrm{g} / \mathrm{mL}$ 。现用现配。

# 4 仪器和设备

4.1 气相色谱仪：配有氢火焰离子化检测器(FID)。  
4.2 电子天平：感量为 $1\mathrm{mg}$ 和 $0.1\mathrm{mg}$   
4.3 匀浆机。  
4.4 皂化装置。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 肉及肉制品等各类固体试样

取样品的可食部分 $200\mathrm{g}$ 进行均质。将试样装入密封的容器里，防止变质和成分变化。试样应在均质化 $24\mathrm{h}$ 内尽快分析。

# 5.1.2 植物油脂、乳品等液体试样

取混匀后的均匀液体试样装入密封容器里待测。

# 5.2 样品处理

# 5.2.1 皂化

称取制备后的样品 $0.25\mathrm{g}\sim 10\mathrm{g}$ （准确至 $0.001\mathrm{g}$ ，胆固醇含量约为 $0.5\mathrm{mg}\sim 5\mathrm{mg}$ ），于 $250~\mathrm{mL}$ 圆底烧瓶中，加入 $30~\mathrm{mL}$ 无水乙醇， $10~\mathrm{mL}60\%$ 氢氧化钾溶液，混匀。将试样在 $100^{\circ}\mathrm{C}$ 磁力搅拌加热电热套皂化回流 $1\textrm{h}$ ，不时振荡防止试样黏附在瓶壁上，皂化结束后，用 $5\mathrm{mL}$ 无水乙醇自冷凝管顶端冲洗其内部，取下圆底烧瓶，用流水冷却至室温。

# 5.2.2 提取

定量转移全部皂化液于 $250~\mathrm{mL}$ 分液漏斗中，用 $30~\mathrm{mL}$ 水分2次～3次冲洗圆底烧瓶，洗液并入分液漏斗，再用 $40~\mathrm{mL}$ 石油醚-无水乙醚混合液 $(1 + 1$ ，体积比)分2次～3次冲洗圆底烧瓶并入分液漏斗，振摇 $2\mathrm{min}$ ，静置，分层。转移水相，合并三次有机相，用水每次 $100~\mathrm{mL}$ 洗涤提取液至中性，初次水洗时轻轻旋摇，防止乳化，提取液通过约 $10\mathrm{g}$ 无水硫酸钠脱水转移到 $150~\mathrm{mL}$ 平底烧瓶中。

# 5.2.3 浓缩

将上述平底烧瓶中的提取液在真空条件下蒸发至近干，用无水乙醇溶解并定容至 $5\mathrm{mL}$ ，待气相色谱仪测定。

不同试样的前处理需要同时做空白试验。

# 5.3 测定

# 5.3.1 仪器参考条件

a）色谱柱：DB-5弹性石英毛细管柱，柱长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ ，粒径 $0.25\mu \mathrm{m}$ ，或同等性能的色谱柱；  
b）载气：高纯氮气，纯度 $\geq 99.999\%$ ；恒流 $2.4~\mathrm{mL / min}$   
c）柱温（程序升温)：初始温度为 $200^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $30^{\circ}\mathrm{C / min}$ 速率升至 $280^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ ；  
d）进样口温度 $280^{\circ}\mathrm{C}$   
e）检测器温度： $290~\mathrm{^\circ C}$   
f）进样量： $1\mu \mathrm{L}$   
g）进样方式：不分流进样，进样 $1\mathrm{min}$ 后开阀；  
h）空气流量： $350~\mathrm{mL / min}$   
i） 氢气流量： $30~\mathrm{mL / min}$

# 5.3.2 标准曲线的制作

分别取胆固醇标准系列工作液注入气相色谱仪，在上述色谱条件下测定标准溶液的响应值（峰面积），以浓度为横坐标、峰面积为纵坐标，制作标准曲线。

# 5.3.3 测定

试样溶液注入气相色谱仪，测定峰面积，由标准曲线得到试样溶液中胆固醇的浓度。根据保留时间定性，外标法定量。胆固醇标准溶液的色谱图见图A.1。

# 6 分析结果的表述

试样中胆固醇的含量按式(1)计算：

$$
X = \frac {\rho \times V}{m \times 1 0 0 0} \times 1 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——试样中胆固醇含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$\rho$ ——试样溶液中胆固醇的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样溶液最终定容的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克(g)；

1000、100——换算系数。

计算结果应扣除空白。结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当称样量为 $0.5\mathrm{g}$ ，定容体积为 $5.0\mathrm{mL}$ ，方法的检出限为 $0.3\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $1.0\mathrm{mg} / 100\mathrm{g}$ 。

# 第三法 比色法

# 16 原理

样品进行脂肪提取后的油脂，经无水乙醇-氢氧化钾溶液皂化，用石油醚提取，浓缩后加入冰乙酸，以硫酸铁铵试剂作为显色剂，采用分光光度计，在 $560~\mathrm{nm}\sim 575~\mathrm{nm}$ 波长下检测，外标法定量。

# 17 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 17.1 试剂

17.1.1 无水乙醇 $\mathrm{(C_2H_5OH)}$   
17.1.2 石油醚：沸程 $30^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$   
17.1.3 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。  
17.1.4 冰乙酸 $\mathrm{(C_2H_4O_2)}$ ：优级纯。  
17.1.5 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。

17.1.6 硫酸铁铵 $\mathrm{[FeNH_4(SO_4)_2\bullet H_2O]}$   
17.1.7 钢瓶氮气 $(\mathrm{N}_2)$ ：纯度 $99.99\%$ 。  
17.1.8 海砂。  
17.1.9 氢氧化钾(KOH)。  
17.1.10 氢氧化钠(NaOH)。  
17.1.11 盐酸（HCl）。  
17.1.12 乙醚 $(\mathrm{C}_2\mathrm{H}_5\mathrm{O})$ 。

# 17.2 试剂配制

17.2.1 铁矾储备液：称取 $4.463\mathrm{g}$ 硫酸铁铵 $\left[\mathrm{FeNH}_{4}(\mathrm{SO}_{4})_{2} \cdot \mathrm{H}_{2}\mathrm{O}\right]$ 于 $100\mathrm{mL}$ 磷酸中（如果不能充分溶解，超声后取上清液），贮藏于干燥器内，此液在室温中稳定。  
17.2.2 铁矾显色液：吸取铁矾储备液 $10\mathrm{mL}$ ，用硫酸定容至 $100\mathrm{mL}$ 。贮藏于干燥器内，以防吸水。  
17.2.3 $50\%$ 氢氧化钾溶液：称取 $50\mathrm{g}$ 氢氧化钾，用水溶解，并定容至 $100~\mathrm{mL}$   
17.2.4 $5\%$ 氯化钠溶液：称取 $5\mathrm{g}$ 氯化钠，用水溶解，并定容至 $100~\mathrm{mL}$   
17.2.5 盐酸溶液 $(1 + 1)$ ：将盐酸与水等体积混合均匀。  
17.2.6 氢氧化钠溶液 $(240\mathrm{g / L})$ ：称取 $24\mathrm{g}$ 氢氧化钠，用水溶解并定容至 $100~\mathrm{mL}$ 。  
17.2.7 海砂：取用水洗去泥土的海砂或河砂，先用盐酸溶液 $(1 + 1)$ 煮沸 $0.5\mathrm{h}$ ，用水洗至中性再用氢氧化钠溶液 $(240\mathrm{g / L})$ 煮沸 $0.5\mathrm{h}$ ，用水洗至中性，经 $100^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 干燥备用。

# 17.3 标准品

胆固醇标准品 $\mathrm{(C_{27}H_{46}O}$ ,CAS号：57-88-5）：纯度 $\geqslant 99\%$ 。

# 19 分析步骤

# 19.1 胆固醇标准曲线的制作

吸取胆固醇标准工作液 $0.0\mathrm{mL},0.5\mathrm{mL},1.0\mathrm{mL},1.5\mathrm{mL},2.0\mathrm{mL}$ 分别置于 $10~\mathrm{mL}$ 试管中，在各管内加入冰乙酸使总体积均达 $4\mathrm{mL}$ 。沿管壁加入 $2\mathrm{mL}$ 铁矾显色液，混匀，在 $15\mathrm{min}\sim 90\mathrm{min}$ 内，在 $560~\mathrm{nm}\sim 575~\mathrm{nm}$ 波长下比色。以胆固醇标准浓度为横坐标，吸光度为纵坐标制作标准曲线。

# 19.2 测定

# 19.2.1 食品中脂肪的提取与测定

根据食品种类分别用索氏脂肪提取法，研磨浸提法和罗高氏法提取脂肪。并计算出每 $100\mathrm{g}$ 食品

中的脂肪含量。

# 19.2.2 食品中胆固醇的测定

将提取的油脂3滴～4滴(约含胆固醇 $300\mu \mathrm{g}\sim 500\mu \mathrm{g}$ )，置于 $25~\mathrm{mL}$ 试管中，准确记录其质量。加入 $4\mathrm{mL}$ 无水乙醇， $0.5\mathrm{mL}50\%$ 氢氧化钾溶液，混匀，装上冷凝管，在 $65^{\circ}C$ 恒温水浴锅中皂化 $1\textrm{h}$ 。皂化时每隔 $20\mathrm{min}\sim 30\mathrm{min}$ 振摇一次使皂化完全。皂化完毕，取出试管，用流水冷却。加入 $3\mathrm{mL}5\%$ 氯化钠溶液， $10~\mathrm{mL}$ 石油醚，盖紧玻璃塞，在电动振荡器上振摇 $2\mathrm{min}$ ，静置分层（一般约需 $1\textrm{h}$ 以上）。

取上层石油醚液 $2\mathrm{mL}$ ，置于 $10~\mathrm{mL}$ 具塞玻璃试管内，在 $65^{\circ}C$ 水浴中用氮气吹干，加入 $4\mathrm{mL}$ 冰乙酸， $2\mathrm{mL}$ 铁矾显色液，混匀，放置 $15\mathrm{min}$ 后在 $560~\mathrm{nm}\sim 575~\mathrm{nm}$ 波长下比色，测得吸光度，在标准曲线上查出相应的胆固醇含量。

不同试样的前处理需要同时做空白试验。

# 20 分析结果的表述

试样中胆固醇的含量按式(3)计算：

$$
X = \frac {A \times C \times V _ {1}}{V _ {2} \times m \times 1 0 0 0} \tag {3}
$$

式中：

$X$ ——试样中胆固醇含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

A 测得的吸光度值在胆固醇标准曲线上的胆固醇含量，单位为微克 $(\mu \mathrm{g})$

$C$ ——试样中脂肪含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$V_{1}$ ——石油醚总体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——取出的石油醚体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ —称取食品油脂试样量，单位为克 $(\mathrm{g})$

1000 ——换算系数。

计算结果应扣除空白。结果保留三位有效数字。

# 21 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 22 其他

方法的检出限为 $2.4\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $7.2\mathrm{mg} / 100\mathrm{g}$ 。

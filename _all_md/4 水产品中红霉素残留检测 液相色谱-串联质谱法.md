# 水产品中红霉素残留量的测定

# 液相色谱-串联质谱法

# 1 范围

本标准规定了水产品中红霉素残留量检测的制样和液相色谱-串联质谱测定方法。

本标准适用于水产品可食性组织中红霉素残留量的检测。

# 3原理

试料中残留的红霉素，用乙腈提取，正己烷除脂，HLB 柱净化，液相色谱-串联质谱测定，内标法定量。

# 4 试剂和材料

以下所用试剂，除特别注明外均为分析纯试剂；水为符合GB/T6682规定的一级水。

4.1 红霉素标准品：含量 $\geq 97\%$   
4.2 红霉素同位素内标物标准品：红霉素同位素（erythromycin A，N，N-Dimethyl- ${}^{13}\mathrm{C}_2$ ）含量 $\geqslant 90\%$   
4.3 乙腈：色谱纯。  
4.4 正己烷：色谱纯。  
4.5 甲醇：色谱纯。  
4.6 乙酸  
4.7 氯化钠   
4.8 十二水磷酸氢二钠  
4.9 磷酸  
4.10 HLB固相萃取柱： $200\mathrm{mg / 6mL}$ ，或相当者。

4.11 0.1 mol/L 磷酸盐缓冲溶液：取十二水磷酸氢二钠 $17.92 \mathrm{~g}$ ，用水溶解并稀释至 $500 \mathrm{~mL}$ ，用磷酸调 $\mathrm{pH}$ 值至 8.0，现配现用。  
4.12 $40\%$ 甲醇溶液：取甲醇 $40~\mathrm{mL}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
4.13 $30\%$ 甲醇溶液：取甲醇 $30~\mathrm{mL}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
4.14 $0.01\%$ 乙酸水溶液：取乙酸 $10~\mu \mathrm{L}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。  
4.15 $100 \mu \mathrm{g} / \mathrm{mL}$ 红霉素标准贮备液：精密称取红霉素标准品 $10 \mathrm{mg}$ ，于 $100 \mathrm{~mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $100 \mu \mathrm{g} / \mathrm{mL}$ 的红霉素标准贮备液。-18 ℃以下避光保存，有效期6个月。  
4.16 $5 \mu \mathrm{g} / \mathrm{mL}$ 红霉素标准工作液: 精密量取 $100 \mu \mathrm{g} / \mathrm{m}$ 红霉素标准贮备液 $500 \mu \mathrm{L}$ , 于 $10 \mathrm{~mL}$ 量瓶中,用甲醇稀释至刻度, 配制成浓度为 $5 \mu \mathrm{g} / \mathrm{mL}$ 的红霉素标准工作液。-18 ℃以下避光保存, 有效期3个月。  
4.17 $100 \mu \mathrm{g} / \mathrm{mL}$ 红霉素同位素内标物标准贮备液: 精密称取红霉素同位素内标物标准品 $10 \mathrm{mg}$ , 于 $100 \mathrm{~mL}$ 量瓶中, 用甲醇溶解并稀释至刻度, 配制成浓度为 $100 \mu \mathrm{g} / \mathrm{mL}$ 红霉素同位素内标物标准贮备液。 -18 ℃以下避光保存, 有效期6个月。  
4.18 200 ng/mL 红霉素同位素内标物标准工作液: 精密量取 $100 \mu \mathrm{g} / \mathrm{mL}$ 红霉素同位素内标物标准贮备液 $200 \mu \mathrm{L}$ , 于 $100 \mathrm{~mL}$ 量瓶中, 用甲醇稀释至刻度, 配制成浓度为 $200 \mathrm{ng} / \mathrm{mL}$ 的红霉素同位素内标物标准工作液。-18 ℃以下避光保存, 有效期3个月。

# 5 仪器和设备

5.1 液相色谱-串联质谱仪：配电喷雾离子源。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 振荡器  
5.5 均质机  
5.6 离心机  
5.7 酸度计   
5.8 旋转蒸发仪   
5.9茄形瓶： $100~\mathrm{mL}$   
5.10 离心管： $50~\mathrm{mL}$   
5.11 滤膜： $0.22\mu \mathrm{m}$

# 6 试料的制备与保存

# 6.1 试料的制备

取水产品空白或供试可食性组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。

——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

- $20^{\circ} \mathrm{C}$ 以下保存，保存期3个月。

# 7 测定步骤

# 7.1 提取

称取试料（ $5 \pm 0.05$ ）g，于 $50 \mathrm{~mL}$ 离心管中，加 $200 \mathrm{ng} / \mathrm{mL}$ 红霉素同位素内标物标准工作液 $50 \mu \mathrm{L}$ ，加乙腈 $15 \mathrm{~mL}$ ，立即搅拌使其分散，超声 $5 \mathrm{~min}$ ，振荡 $5 \mathrm{~min}$ ， $4000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取上清液，于另一离心管中，残渣中加乙腈 $5 \mathrm{~mL}$ ，重复提取一次，合并两次上清液，加氯化钠 $2 \mathrm{~g}$ 和正己烷 $15 \mathrm{~mL}$ ，振荡 $5 \mathrm{~min}$ ， $4000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，弃上层正己烷液，取下层液，于 $100 \mathrm{~mL}$ 茄形瓶中，于 $40^{\circ} \mathrm{C}$ 旋转蒸发至干，用磷酸盐缓冲溶液 $5 \mathrm{~mL}$ ，溶解残余物，加正己烷 $5 \mathrm{~mL}$ ，混合 $1 \mathrm{~min}$ ，转至离心管中， $6000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，弃正己烷层，再加正己烷 $5 \mathrm{~mL}$ ，重复操作一次，取下层溶液，备用。

# 7.2 净化

HLB柱依次用甲醇 $5 \mathrm{~mL}$ 、水 $5 \mathrm{~mL}$ 和磷酸盐缓冲溶液 $5 \mathrm{~mL}$ 活化，取备用液过柱，用磷酸盐缓冲液 $5 \mathrm{~mL}$ 洗涤离心管，洗涤液过柱，流干，依次用水 $5 \mathrm{~mL}$ 和 $40 \%$ 甲醇溶液 $5 \mathrm{~mL}$ 淋洗，抽干，用甲醇 $8 \mathrm{~mL}$ 洗脱，收集洗脱液，于茄形瓶中，于 $40^{\circ} \mathrm{C}$ 旋转蒸发至干，用 $30 \%$ 甲醇溶液 $1.0 \mathrm{~mL}$ 溶解残余物，滤膜过滤，供液相色谱-串联质谱测定。

# 7.3 标准曲线的制备

精密量取 $5 \mu \mathrm{g} / \mathrm{mL}$ 红霉素标准工作液和 $200 \mathrm{ng} / \mathrm{mL}$ 红霉素同位素内标物标准工作液适量, 用 $30 \%$ 甲醇溶液溶解并稀释, 配制成红霉素同位素内标物浓度为 $10 \mathrm{ng} / \mathrm{mL}$ 以及红霉素浓度为 1、5、10、25、50、100 和 $500 \mathrm{ng} / \mathrm{mL}$ 的系列标准溶液, 供液相色谱-串联质谱测定。以特征离子质量色谱峰面积为纵坐标, 标准溶液浓度为横坐标, 绘制标准曲线。求回归方程和相关系数。

# 7.4 测定

# 7.4.1 液相色谱条件

色谱柱： $\mathrm{C}_{18}$ （ $150 \mathrm{~mm} \times 2.1 \mathrm{~mm}$ ，粒径 $5 \mu \mathrm{m}$ ），或相当者；

流动相：A：甲醇，B： $0.01 \text{‰}$ 乙酸水溶液；

梯度洗脱： $0.0\sim 1.0\mathrm{min}$ ， $30\% \mathrm{A}$ ； $1.01\sim 7\mathrm{min}$ ， $95\% \mathrm{A}$ ； $7.01\sim 17\mathrm{min}$ ； $30\% \mathrm{A}$

流速： $0.3\mathrm{mL / min}$

柱温： $25~\mathrm{^\circ C}$

进样量： $25~\mu \mathrm{L}$

# 7.4.2 质谱条件

离子源：电喷雾离子源；

扫描方式：正离子扫描；

检测方式：多反应监测；

喷雾电压： $4700\mathrm{V}$

离子传输毛细管温度： $300^{\circ}C$

源内碰撞诱导解离电压： $8\mathrm{V}$

雾化气流速： $12.3\mathrm{L / h}$

辅助气流速： $1.7\mathrm{L / h}$

定性、定量离子对和碰撞能量见表1。

# 7.4.3 测定法

# 7.4.3.1 定性测定

通过试样色谱图的保留时间与相应标准品的保留时间、色谱峰的特征离子与相应浓度标准溶液色谱峰的特征离子相对照定性。试样与标准品保留时间的相对偏差不大于 $5\%$ ；试样特征离子的相对丰度与浓度相当标准溶液的相对丰度一致，相对丰度偏差不超过 $\pm 20\%$ ，则可判断试样中存在相应的被测物。

# 7.4.3.2 定量测定

取试样溶液和标准溶液，按外标法，以峰面积定量，标准溶液及试样溶液中的红霉素响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，红霉素标准溶液和空白添加试样溶液中特征离子质量色谱图见附录A。

# 7.5 空白试验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算和表述

试料中对红霉素的残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算

$$
X = \frac {C \times V}{m}
$$

式中：

$X$ ——供试试料中红霉素的残留量， $\mu \mathrm{g} / \mathrm{kg}$ ；

$C$ ——试样溶液中红霉素的浓度, $\mathrm{ng} / \mathrm{mL}$ ;

$V$ ——溶解残余物所用体积, $\mathrm{mL}$ ;

$m$ ——供试试料质量, g。

注：计算结果需扣除空白值。测定结果用两次平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度、精密度

# 9.1 灵敏度

本方法的检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1\mu \mathrm{g / kg}$

# 9.2 准确度

本方法在 $1\sim 400~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

# 前言

本标准代替GB/T23749—2009《食品中叶绿素铜钠的测定 分光光度法》。

本标准与GB/T23749—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中叶绿素铜钠的测定”；  
——修改了原标准的测定波长；  
——修改了原标准的测定步骤。

# 食品安全国家标准

# 食品中叶绿素铜钠的测定

# 1 范围

本标准规定了果蔬汁(肉)饮料、碳酸饮料、风味饮料、配制酒、糖果、罐头中叶绿素铜钠的测定方法。本标准适用于果蔬汁(肉)饮料、碳酸饮料、风味饮料、配制酒、糖果、罐头中叶绿素铜钠的测定。

# 2 原理

试样中的叶绿素铜钠在酸性条件下经聚酰胺粉吸附，解吸液洗脱，分光光度计测定，标准曲线法定量。

# 3 试剂与材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂与材料

3.1.1 氢氧化钠。  
3.1.2 乙酸铵。  
3.1.3 甲醇。  
3.1.4 冰乙酸。  
3.1.5 聚酰胺粉：粒径 $0.150\mathrm{mm}\sim 0.180\mathrm{mm}$

# 3.2 试剂配制

3.2.1 氢氧化钠溶液 $(4\mathrm{mol} / \mathrm{L})$ ：称取 $16.0\mathrm{g}$ 氢氧化钠，用水溶解并定容至 $100~\mathrm{mL}$ 。  
3.2.2 氢氧化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $0.40\mathrm{g}$ 氢氧化钠，用水溶解并定容至 $100~\mathrm{mL}$ 。  
3.2.3 乙酸铵缓冲溶液 $(0.2\mathrm{mol} / \mathrm{L})$ ：称取 $7.708\mathrm{g}$ 乙酸铵，用水溶解并定容至 $500~\mathrm{mL}$ 。  
3.2.4 解吸液： $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液 $+$ 甲醇 $= 1 + 10$ （体积比）。

# 3.3 标准品

叶绿素铜钠，含量 $\geq 99.0\%$

# 3.4 标准溶液配制

# 3.4.1 标准贮备溶液

精确称取经 $105\,^{\circ}\mathrm{C} \pm 1\,^{\circ}\mathrm{C}$ 干燥至恒重并按其纯度折算为 $100\%$ 质量的叶绿素铜钠标准品 $0.0500\,^{\circ}\mathrm{g}$ ，用水溶解并定容至 $100\,^{\circ}\mathrm{mL}$ 棕色容量瓶中，此溶液浓度为 $500\, \mu\mathrm{g/mL}$ ，当天配制，避光保存。

# 3.4.2 标准工作溶液

准确移取 $500\mu \mathrm{g / mL}$ 标准溶液 $10~\mathrm{mL}$ 至 $100~\mathrm{mL}$ 烧杯中，加入 $0.2\mathrm{mol} / \mathrm{L}$ 的乙酸铵溶液 $30~\mathrm{mL}$ ，用

4 mol/L 氢氧化钠溶液和冰乙酸调 $\mathrm{pH} 5 \sim 6$ 。加入 $3.0 \mathrm{~g}$ 聚酰胺粉, 充分搅拌 $2 \mathrm{~min}$ , 避光静置 $5 \mathrm{~min}$ , 用约 $20 \mathrm{~mL}$ 蒸馏水转移至 G3 砂芯漏斗中抽滤, 弃去滤液。用 $75 \mathrm{~mL}$ 解吸液分 3 次解吸色素: 每次倒入约 $25 \mathrm{~mL}$ 解吸液, 浸泡 $2 \mathrm{~min}$ , 再振摇 $2 \mathrm{~min}$ , 抽滤并用 $20 \mathrm{~mL}$ 解吸液洗净抽滤瓶中残液。收集滤液, 用解吸液定容至 $100 \mathrm{~mL}$ , 配制成浓度为 $50 \mu \mathrm{g} / \mathrm{mL}$ 的标准溶液, 此溶液临用时配制。

# 4 仪器与设备

4.1 分光光度计。  
4.2 天平：感量 $0.0001\mathrm{g}$ 。  
4.3 G3砂芯漏斗。  
4.4 抽滤装置：真空泵、抽滤瓶。  
4.5 恒温干燥箱。  
4.6 小型样品粉碎机。  
4.7 瓷研钵。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 叶绿素铜钠测定试样预处理

# 5.1.1.1 饮料、酒样品的预处理

将样品摇匀，准确称取 $5\mathrm{mL}\sim 10\mathrm{mL}$ （精确至 $0.1\mathrm{mL}$ )样品至 $100~\mathrm{mL}$ 烧杯中，在 $55^{\circ}C\sim 60^{\circ}C$ 的水浴中加热 $3\mathrm{min}\sim 5\mathrm{min}$ ，去除酒精。

# 5.1.1.2 罐头样品的预处理

取有代表性的样品置于捣碎机中充分捣碎，准确称取 $1\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 混匀浆液至 $100~\mathrm{mL}$ 烧杯中。

# 5.1.1.3 糖果样品的预处理

将样品置于瓷研钵中研细、混匀，准确称取 $1\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 样品至 $100~\mathrm{mL}$ 烧杯中。

# 5.1.2 被测样品溶液后期处理

向含有被测样品粉末或样品浆液的 $100~\mathrm{mL}$ 烧杯中加入 $0.2\mathrm{mol} / \mathrm{L}$ 的乙酸铵溶液 $30~\mathrm{mL}$ ，溶解并混匀样液，用 $4\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液和冰乙酸调 $\mathrm{pH}5\sim 6$ 。加入 $3.0\mathrm{g}$ 聚酰胺粉，充分搅拌 $2\mathrm{min}$ 。将样品溶液用约 $20~\mathrm{mL}60^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 蒸馏水转移至G3砂芯漏斗中抽滤，弃去滤液。再用 $75~\mathrm{mL}$ 解吸液分3次解吸色素，抽滤并用 $20~\mathrm{mL}$ 解吸液洗净抽滤瓶中残液，收集滤液，用解吸液定容至 $100~\mathrm{mL}$ 。

# 5.2 仪器条件

5.2.1 测定波长： $405~\mathrm{nm}$   
5.2.2 比色Ⅲ：1cm。

# 5.3 标准曲线的制作

分别取标准工作液 $0\mathrm{mL},5.0\mathrm{mL},10\mathrm{mL},20\mathrm{mL},30\mathrm{mL},40\mathrm{mL},50\mathrm{mL}$ 至 $100~\mathrm{mL}$ 容量瓶中，用解

吸液稀释至刻度，配制成浓度为 $0\mu \mathrm{g / mL},5\mu \mathrm{g / mL},10\mu \mathrm{g / mL},20\mu \mathrm{g / mL},30\mu \mathrm{g / mL},40\mu \mathrm{g / mL},$ $50~\mu \mathrm{g / mL}$ 的标准系列。以 $0\mu \mathrm{g / mL}$ 溶液为空白，测定其吸光值。以浓度为横坐标，以吸光值为纵坐标绘制标准曲线。

# 5.4 试样溶液的测定

取经过前处理的样品的制备液，以标准曲线的 $0\mu \mathrm{g / mL}$ 为空白，测定其吸光值，根据标准曲线获得样品溶液中叶绿素铜钠的浓度。

# 6 分析结果表述

试样中叶绿素铜钠的含量按式(1)计算：

$$
X = \frac {c \times V}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中叶绿素铜钠的含量，单位为克每千克 $(\mathrm{g / kg})$ 或克每升 $(\mathrm{g / L})$ ；

$c$ ——从标准曲线上查得的叶绿素铜钠的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样品定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——称取样品量，单位为克或毫升（g 或 mL）。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留小数点后三位。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本标准检出限为 $0.001\mathrm{g / kg}$ ，定量限为 $0.005\mathrm{g / kg}$

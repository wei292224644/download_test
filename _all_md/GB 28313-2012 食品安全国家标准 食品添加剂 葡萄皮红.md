# 食品安全国家标准

# 食品添加剂 葡萄皮红

# 1 范围

本标准适用于以葡萄皮或葡萄榨汁后的皮渣为主要原料，经水或食用乙醇提取、精制而成的食品添加剂葡萄皮红。其主要着色成分为花色苷。葡萄皮红产品可含有糊精等辅料。

# 2 分子式、结构式

# 2.1 分子式

芍药花花青素: $\mathrm{C_{16}H_{13}O_6X}$

锦葵色素: $\mathrm{C}_{17} \mathrm{H}_{15} \mathrm{O}_{7} \mathrm{X}$

飞燕草花青素: $\mathrm{C}_{15} \mathrm{H}_{11} \mathrm{O}_{7} \mathrm{X}$

牵牛花花青素: $\mathrm{C}_{16} \mathrm{H}_{13} \mathrm{O}_{7} \mathrm{X}$

X:酸基团

# 2.2 结构式

![](images/ad4400ab0ae62a518635ab1e16309d281c268178f0b069cddcc221f92e259c83.jpg)

芍药花花青素: $\mathrm{R} = \mathrm{OCH}_{3}$ ; $\mathrm{R}^{\prime} = \mathrm{H}$

锦葵色素：R， $\mathrm{R}^{\prime} = \mathrm{OCH}_{3}$

飞燕草花青素: $R, R' = OH$

牵牛花花青素： $\mathrm{R = OCH_3}$ ； $\mathrm{R}^{\prime} = \mathrm{OH}$

# 附录A

# 检验方法

# A. 1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 最大吸收峰

取样品 $0.1\mathrm{g}$ ，用 $\mathrm{pH}3.0$ 柠檬酸-磷酸氢二钠缓冲溶液溶解并稀释至 $100\mathrm{mL}$ ，此试样液在 $515\mathrm{nm}\sim 535\mathrm{nm}$ 范围内有最大吸收峰。

# A. 2.2 颜色反应

取 $0.1 \mathrm{~g}$ 样品, 溶于 $50 \mathrm{~mL}$ 水中, 溶液呈红色~紫红色, 在溶液中加入氢氧化钠溶液 (4.3 g氢氧化钠溶于 $100 \mathrm{~mL}$ 水中), 溶液颜色转为蓝色或深绿色。

# A. 3 色价的测定

# A.3.1 试剂和材料

A. $3.1.1 0.2 \mathrm{~mol} / \mathrm{L}$ 磷酸氢二钠溶液: 精确称取磷酸氢二钠 $\left(\mathrm{Na}_{2} \mathrm{HPO}_{4} \cdot 12 \mathrm{H}_{2} \mathrm{O}\right) 71.64 \mathrm{~g}$ , 用水溶解并定容至 $1000 \mathrm{~mL}$ 。  
A. 3.1.2 $0.1 \mathrm{~mol} / \mathrm{L}$ 柠檬酸溶液: 精确称取柠檬酸 $\left(\mathrm{C}_{6} \mathrm{H}_{8} \mathrm{O}_{7} \cdot \mathrm{H}_{2} \mathrm{O}\right) 21.01 \mathrm{~g}$ , 用水溶解并定容至 $1000 \mathrm{~mL}$ 。

A. 3.1.3 pH 3.0 柠檬酸-磷酸氢二钠缓冲溶液：取 $0.2 \mathrm{~mol} / \mathrm{L}$ 磷酸氢二钠溶液 $4.11 \mathrm{~mL}$ 与 $0.1 \mathrm{~mol} / \mathrm{L}$ 柠檬酸溶液 $15.89 \mathrm{~mL}$ 混合而成。

# A.3.2 仪器和设备

分光光度计。

# A.3.3 分析步骤

称取 $0.1\mathrm{g}\sim 0.2\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，用 $\mathrm{pH}3.0$ 柠檬酸-磷酸氢二钠缓冲溶液溶解，转移至 $100~\mathrm{mL}$ 容量瓶中，加 $\mathrm{pH}3.0$ 柠檬酸-磷酸氢二钠缓冲溶液定容至刻度，摇匀。取此试样液置于 $1\mathrm{cm}$ 比色皿中，以 $\mathrm{pH}3.0$ 柠檬酸-磷酸氢二钠缓冲溶液做空白对照，用分光光度计在 $515~\mathrm{nm}\sim 535~\mathrm{nm}$ 范围内的最大吸收波长处测定吸光度（吸光度值应控制在 $0.3\sim 0.7$ 之间，否则应调整试样液浓度，再重新测定吸光度）。

# A.3.4结果计算

色价以被测试样浓度为 $1\%$ 、用 $1\mathrm{cm}$ 比色皿、在 $515\mathrm{nm} \sim 535\mathrm{nm}$ 范围内最大吸收波长处测得的吸光度 $E_{\mathrm{lcm}}^{1\%}(515 \sim 535)\mathrm{nm}$ 计，按公式（A.1）计算：

$$
E _ {1 \mathrm {cm}} ^ {1 \%} (515 \sim 535) \mathrm {nm} = \frac {A}{c} \times \frac {1}{100} \dots \tag{A.1}
$$

式中：

A——实际测定试样液的吸光度值；

$c$ ——被测试样液浓度的数值，单位为克每毫升（ $\mathrm{g/mL}$ ）。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $2.5\%$ 。

# A. 4 二氧化硫的测定

# A.4.1 测定方法

采用GB/T5009.34中第二法蒸馏法测得试样中的二氧化硫总含量（ $x$ ），然后按A.4.2中的计算公式换算成以一个色价计的二氧化硫含量（ $y$ ）。

# A.4.2 结果计算

二氧化硫含量以一个色价产品中的二氧化硫质量分数y计, 数值以 $\mathrm{mg} / \mathrm{kg}$ 表示, 按公式 (A.2) 计算:

$$
y = \frac{x\times 1000}{E_{\mathrm{cm}}^{1\%}(515\sim 535)\mathrm{nm}}\dots \tag{A.2}
$$

式中：

$x$ ——按照GB/T5009.34中第二法蒸馏法测得的试样中二氧化硫总含量，单位为克每千克（ $\mathrm{g/kg}$ ）；

1000——单位换算系数；

$E_{\mathrm{1cm}}^{1\%}(515 \sim 535) \mathrm{nm}$ ——被测试样的色价。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

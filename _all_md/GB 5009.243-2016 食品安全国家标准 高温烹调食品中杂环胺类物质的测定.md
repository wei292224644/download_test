# 食品安全国家标准

# 高温烹调食品中杂环胺类物质的测定

# 1 范围

本标准规定了高温烹调食品中2-氨基-3,4-二甲基咪唑并[4,5-f]喹啉(MeIQ),2-氨基-3,8-二甲基咪唑并[4,5-f]喹啉(MeIQx),2-氨基-3,4,8-三甲基咪唑并[4,5-f]喹啉(4,8-DiMeIQx),2-氨基-3,7,8-三甲基咪唑并[4,5-f]喹啉(7,8-DiMeIQx),2-氨基-1-甲基-6-苯基-咪唑并[4,5-b]吡啶(PhIP)等杂环胺的液相色谱-质谱/质谱测定方法。

本标准适用于烤鱼、烤肉及其制品中MeIQ、MeIQx、4,8-DiMeIQx、7,8-DiMeIQx、PhIP的测定。

# 2 原理

试样采用氢氧化钠/甲醇溶液提取，固相萃取柱净化，液相色谱-串联质谱检测，内标法定量。

# 3 试剂和材料

除非另有说明，所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 氢氧化钠。  
3.1.2 乙酸铵：纯度 $\geqslant 98\%$   
3.1.3 甲醇：色谱纯。  
3.1.4 乙醇：色谱纯。  
3.1.5 正己烷：色谱纯。  
3.1.6 二氯甲烷：色谱纯。  
3.1.7 乙腈：色谱纯。  
3.1.8 冰乙酸：色谱纯。

# 3.2 试剂配制

3.2.1 氢氧化钠溶液 $(40\mathrm{g / L})$ ：称取 $40.0\mathrm{g}$ 氢氧化钠，用水溶解并定容至 $1\mathrm{L}$ 。  
3.2.2 氢氧化钠溶液 $(4\mathrm{g} / \mathrm{L})$ ：量取 $40~\mathrm{g / L}$ 氢氧化钠溶液(3.2.1) $50~\mathrm{mL}$ ，加入 $450~\mathrm{mL}$ 水，混合均匀。  
3.2.3 $40\mathrm{g / L}$ 氢氧化钠-甲醇混合溶液 $(70 + 30$ ，体积分数）：量取 $40~\mathrm{g / L}$ 氢氧化钠溶液(3.2.1) $70~\mathrm{mL}$ 加入 $30~\mathrm{mL}$ 甲醇，混合均匀。  
3.2.4 $4\mathrm{g / L}$ 氢氧化钠-甲醇混合溶液 $(45 + 55$ ，体积分数）：量取 $4\mathrm{g / L}$ 氢氧化钠溶液(3.2.2) $45~\mathrm{mL}$ ，加入 $55~\mathrm{mL}$ 甲醇，混合均匀。  
3.2.5 乙醇-二氯甲烷混合溶液 $(10 + 90$ ，体积分数)：量取 $10~\mathrm{mL}$ 乙醇，加入 $90~\mathrm{mL}$ 二氯甲烷，混合均匀。  
3.2.6 乙腈-水溶液 $(5 + 95$ ，体积分数)：量取 $5\mathrm{mL}$ 乙腈，加 $95~\mathrm{mL}$ 水，混合均匀。

3.2.7 乙酸-乙酸铵缓冲液：称取 $1.155\mathrm{g}$ 乙酸铵，用 $450~\mathrm{mL}$ 水溶解，用乙酸调 $\mathrm{pH}$ 至 $5.0\pm 0.5$ ，加水定容至 $500~\mathrm{mL}$ 。  
3.2.8 乙酸缓冲液-乙腈混合溶液 $(50 + 50$ ，体积分数)：量取乙酸-乙酸铵缓冲液(3.2.7) $50~\mathrm{mL}$ ，加入 $50~\mathrm{mL}$ 乙腈，混合均匀。

# 3.3 标准品

3.3.1 杂环胺标准物质： $\mathrm{MeIQ(C_{12}H_{12}N_4,77094 - 11 - 2)}$ 、 $\mathrm{MeIQx(C_{11}H_{11}N_5,77500 - 04 - 0)}$ 、4，8-DiMeIQx $(\mathrm{C_{12}H_{13}N_5},95896 - 78 - 9)$ 、7，8-DiMeIQx $(\mathrm{C_{12}H_{13}N_5},92180 - 79 - 5)$ 、PhIP $(\mathrm{C_{13}H_{12}N_4},105650 - 23 - 5)$ ，纯度均大于 $99\%$ 。  
3.3.2 内标标准物质4，7，8-TriMeIQx $\mathrm{(C_{13}H_{15}N_5}$ ，132898-07-8)，纯度大于 $99\%$ 。

# 3.4 标准溶液配制

3.4.1 杂环胺标准储备液：将MeIQ、MeIQx、4，8-DiMeIQx、7，8-DiMeIQx、PhIP(3.3.1)分别用乙腈配制成浓度为 $10.0~\mu \mathrm{g / mL}$ 的标准储备液。  
3.4.2 内标储备液：将4，7，8-TriMeIQx(3.3.2)用乙腈配制成浓度为 $10.0\mu \mathrm{g / mL}$ 的标准储备液。  
3.4.3 混合标准工作液：吸取杂环胺标准储备液(3.4.1)及内标储备液(3.4.2)，乙腈-水溶液(3.2.6)稀释，得到杂环胺浓度分别为 $0.5\mu \mathrm{g} / \mathrm{L}, 1.0\mu \mathrm{g} / \mathrm{L}, 5.0\mu \mathrm{g} / \mathrm{L}, 20.0\mu \mathrm{g} / \mathrm{L}, 50.0\mu \mathrm{g} / \mathrm{L}, 100\mu \mathrm{g} / \mathrm{L}$ ，内标浓度为 $20.0\mu \mathrm{g} / \mathrm{L}$ 的混合标准溶液。  
3.4.4 内标工作液：吸取适量内标储备液(3.3.2)，用乙腈配制成浓度为 $200\mu \mathrm{g} / \mathrm{L}$ 的内标工作液。

# 3.5 材料

3.5.1 微孔滤膜： $0.2\mu \mathrm{m}$ ，有机系。  
3.5.2 苯乙烯二乙烯基苯共聚物固相萃取柱(Lichrolut EN或相当者， $3\mathrm{mL},200\mathrm{mg}$ ）。

# 4 仪器与设备

4.1 液相色谱仪-质谱/质谱仪：配有电喷雾离子源。  
4.2 电子天平：感量为 $0.01\mathrm{mg},1\mathrm{mg}$   
4.3 pH计：感量为0.01。  
4.4 高速离心机，转速不低于 $10000\mathrm{r / min}$   
4.5 氮气浓缩仪。  
4.6 固相萃取装置。  
4.7 均质器。  
4.8 旋涡振荡器。

# 5 分析步骤

# 5.1 试样制备

烤鱼、烤肉及其制品取可食部分，捣碎混匀，标识后 $-18^{\circ} \mathrm{C}$ 冷冻保存。

# 5.2 试样处理

5.2.1 提取：称取试样 $2\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $200\mu \mathrm{L}$ 内标工作液(3.4.4)，再加入

$9.8\mathrm{mL}40\mathrm{g / L}$ 氢氧化钠-甲醇混合溶液(3.2.3)，均质 $1\mathrm{min}$ 。均质器刀头分别用 $5.0\mathrm{mL}40\mathrm{g / L}$ 氢氧化钠-甲醇混合溶液(3.2.3)各洗涤两次，洗涤液合并至样品提取离心管中。试样在 $10000\mathrm{r / min}$ 条件下离心 $10\mathrm{min}$ ，待净化。  
5.2.2 净化：固相萃取柱(3.5.2)预先依次用 $2\mathrm{mL}$ 甲醇、 $3\mathrm{mL}4\mathrm{g / L}$ 氢氧化钠溶液(3.2.2)活化。量取 $10~\mathrm{mL}$ 提取液加入固相萃取柱中，弃去流出液后，依次用 $3\mathrm{mL}4\mathrm{g / L}$ 氢氧化钠-甲醇混合溶液(3.2.4）、 $2\mathrm{mL}$ 正己烷洗淋，每次淋洗完后都需将柱体内淋洗溶液抽干，最后用 $1.5\mathrm{mL}$ 乙醇-二氯甲烷溶液(3.2.5)洗脱，洗脱流速小于 $1\mathrm{mL / min}$ 。洗脱液于 $35^{\circ}C$ 水浴下氮气浓缩至近干后，加入 $1.0\mathrm{mL}$ 乙酸缓冲液-乙腈混合溶液(3.2.8)，涡旋混匀，微孔滤膜(3.5.1)过滤至进样小瓶，待上机分析测定。

# 5.3 测定

# 5.3.1 液相色谱条件

5.3.1.1 色谱柱： $\mathrm{C_{18}}$ 柱 $(2.5\mu \mathrm{m},100\mathrm{mm}\times 2.1\mathrm{mm})$ 或相当者。  
5.3.1.2 流动相：A为乙酸-乙酸铵缓冲液(3.2.7)，B为乙腈。梯度洗脱程序参见附录A中A.1。  
5.3.1.3 流速： $0.3\mathrm{mL / min}$   
5.3.1.4 柱温： $40^{\circ}C$ 。  
5.3.1.5 进样量： $5\mu \mathrm{L}$

# 5.3.2 质谱条件

5.3.2.1 电离方式：电喷雾电离正离子模式（ESI+）。  
5.3.2.2 扫描方式：多反应监测（MRM）。  
5.3.2.3 其他质谱条件见附录B。

# 5.3.3 液相色谱-质谱/质谱测定

通过液相色谱将试样溶液进行分离后注入串联四极杆质谱仪中，得到相应的保留时间及信号响应值，根据标准曲线得到相应目标化合物的浓度。

进行样品测定时，如果检出的质量色谱峰保留时间与标准样品一致，并且在扣除背景后的样品谱图中，各定性离子的相对丰度与浓度接近的同样条件下得到的标准溶液谱图相比，最大允许相对偏差不超过表1中规定的范围，则可判断样品中存在对应的待测物。在上述仪器条件下，标准溶液的液相色谱-质谱/质谱多反应监测(MRM)色谱图参见附录C中图C.1。

# 5.4 空白试验

除不加试样外，均按5.2操作步骤进行。

# 6 结果计算

试样中杂环胺含量按式(1)计算。计算结果需扣除空白值。

$$
X = \frac {c \times V}{m} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中杂环胺含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c$ ——根据标准曲线上计算得出的样液中杂环胺浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样液所代表的最终试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 7 回收率和精密度

本方法在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 50~\mu \mathrm{g} / \mathrm{kg}$ 添加浓度的回收率为 $63.3\% \sim 124\%$ ，精密度范围为 $6.26\% \sim$ $18.2\%$ 。

# 液相色谱-串联质谱法

# 1 范围

本文件规定了猪、鸡肌肉、肝脏和鱼、虾可食性组织中氯霉素残留检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、鸡肌肉、肝脏和鱼、虾可食组织中氯霉素残留量的测定。

# 2 规范性引用文件

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的氯霉素，采用间位氯霉素或氘代氯霉素作内标，依次用乙腈、 $4\%$ 氯化钠去蛋白，正己烷脱脂，乙酸乙酯提取，固相萃取柱净化，氮气吹干，液相色谱-串联质谱法测定，内标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $(\mathrm{CH}_3\mathrm{CN})_2$ 色谱纯

5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.3 正己烷 $(\mathrm{C_6H_{14}})$ 。

5.1.4 乙酸乙酯 $\mathrm{(CH_3COOC_2H_5)}$

5.1.5 氯化钠 $(\mathrm{NaCl})$ 。

# 5.2 标准品

5.2.1 氯霉素(CAP，Chloramphenicol， $\mathrm{C_{11}H_{12}Cl_2N_2O_5}$ ，CAS号：56-75-7)，含量 $\geq 98.0\%$

5.2.2 内标：间位氯霉素(m-CAP)或氘代氯霉素(CAP- $\mathrm{D}_5$ ，Chloramphenicol $\mathrm{D}_5$ ， $\mathrm{C_{11}H_7D_5Cl_2N_2O_5}$ ，CAS号：202480-68-0)，含量 $\geq 98.0\%$ 。

# 5.3 溶液配制

5.3.1 $4\%$ 氯化钠溶液：取氯化钠 $4\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ ，临用前配制。

5.3.2 水饱和乙酸乙酯溶液：取乙酸乙酯 $400~\mathrm{mL}$ ，于 $500~\mathrm{mL}$ 棕色试剂瓶中，加水 $50~\mathrm{mL}$ ，加盖、振摇、静置，使用时取上层溶液。

# 5.4 标准溶液制备

5.4.1 氯霉素标准储备液：取氯霉素约 $10\mathrm{mg}$ ，精密称定，用甲醇适量超声使溶解并稀释定容至 $100\mathrm{mL}$ 容量瓶，制成浓度为 $100\mu \mathrm{g} / \mathrm{mL}$ 的标准储备液。 $-18^{\circ}\mathrm{C}$ 保存，有效期1年。

5.4.2 内标储备液：取内标 $10\mathrm{mg}$ ，精密称定，用甲醇适量超声使溶解并稀释至 $100~\mathrm{mL}$ 容量瓶，制成浓度为 $100~\mu \mathrm{g / mL}$ 的内标储备液。 $-18^{\circ}C$ 保存，有效期1年。  
5.4.3 氯霉素标准工作液：精密量取氯霉素标准储备液适量，用流动相稀释成浓度为 $10\mathrm{ng / mL}$ 和 $50~\mathrm{ng / mL}$ 的标准工作液。 $4^{\circ}C$ 保存，有效期3个月。  
5.4.4 内标工作液：精密量取内标储备液适量，用流动相稀释成浓度为 $20\mathrm{ng / mL}$ 的内标工作液。 $4^{\circ}C$ 保存，有效期3个月。

# 5.5 材料

$\mathrm{C_{18}}$ 周相萃取柱： $500~\mathrm{mg / 3~mL}$ ，或相当者。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 涡旋混合器。  
6.4 组织匀浆器。  
6.5 低温离心机。  
6.6 样品浓缩仪。  
6.7 固相萃取装置。

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取试料 $5\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，加内标工作液 $250~\mu \mathrm{L}$ 、乙腈 $5\mathrm{mL},4\%$ 氯化钠溶液 $5\mathrm{mL}$ ，涡旋振荡 $2\mathrm{min},4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液，残渣重复提取1次，合并上清液。加正己烷 $5\mathrm{mL}$ ，涡旋振荡 $1\mathrm{min},2000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，弃去上层液，正己烷重复处理1次。加水饱和的乙酸乙酯溶液 $5\mathrm{mL}$ ，涡旋振荡 $1\mathrm{min},2000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，上层液转移到 $15\mathrm{mL}$ 离心管中，重复提取1次，合并提取液，氮气吹干，加水-乙腈 $(95:5)3\mathrm{mL}$ 使溶解，备用。

# 8.2 净化

取固相萃取柱，依次用甲醇、水各 $10\mathrm{mL}$ 预洗。取备用液过柱，加水洗柱2次，每次 $3\mathrm{mL}$ ，加流动相 $4\mathrm{mL}$ ，以 $1\mathrm{mL} / \mathrm{min}$ 速度洗脱，收集洗脱液。加水饱和乙酸乙酯溶液 $4\mathrm{mL}$ ，涡旋振荡 $1\mathrm{min}$ ， $2000\mathrm{r} / \mathrm{min}$ 离心 $5\mathrm{min}$ ，取上层液，重复处理1次，合并上层溶液，氮气吹干。加流动相 $1.0\mathrm{mL}$ 溶解，滤过，供液相色谱-串联质谱测定。

# 8.4 测定

# 8.4.1 液相色谱条件

a）色谱柱： $\mathrm{C_{18}(150~mm\times 3.2~mm,5~\mu m)}$   
b）流动相：甲醇-水（50：50）；  
c) 流速: $0.4 \mathrm{~mL} / \mathrm{min}$ ;   
d） 柱温： $30^{\circ}C$ ：  
e）进样量： $20~\mu \mathrm{L}$   
运行时间：10min。

# 8.4.2 质谱条件

a）离子源：电喷雾离子源；  
b）扫描方式：负离子扫描；  
c）检测方式：多反应监测；  
d）电离电压： $2.8\mathrm{kV}$   
e）源温： $120^{\circ}C$   
f）碰撞气：氩气 $3.0\times 10^{-3}$ mbar;

h）锥孔气速： $20\mathrm{L / h}$   
i）数据采集窗口：10min；  
i）驻留时间：0.6s；

# 8.4.3 测定法

取试料溶液和相应的标准溶液，作单点或多点校准，按内标法以色谱峰面积定量。氯霉素特征离子质谱图见图A.1，标准溶液中各特征离子的质量色谱图分别见图A.2。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算

试料中待测物残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times A _ {i S} ^ {\prime} \times C _ {S} \times C _ {i S} \times V}{A _ {i S} \times A _ {S} \times C _ {i S} ^ {\prime} \times m} \tag {1}
$$

式中：

$X$ ——试样中氯霉素残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A_{\mathrm{S}}$ ——对照溶液中氯霉素的峰面积；

$A_{iS}^{\prime}$ ——对照溶液中内标的峰面积；

$C_{S}$ ——对照溶液氯霉素标准品浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$C_{\mathrm{is}}^{\prime}$ ——对照溶液中氯霉素内标浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$C_{iS}$ ——供试溶液中氯霉素内标浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A$ 供试溶液中氯霉素的峰面积；

$A_{iS}$ ——试样溶液氯霉素内标的峰面积；

V ——溶解残余物所用溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

10.1 灵敏度

本方法氯霉素的检测限为 $0.1\mu \mathrm{g / kg}$ ，定量限为 $0.2\mu \mathrm{g / kg}$

10.2 准确度

本方法在 $0.1\mu \mathrm{g / kg}\sim 10~\mu \mathrm{g / kg}$ 添加浓度的回收率为 $70\% \sim 120\%$

10.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

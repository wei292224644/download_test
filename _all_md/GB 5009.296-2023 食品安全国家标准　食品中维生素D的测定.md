2023-09-06 发布

# 前言

本标准代替GB5009.82—2016《食品安全国家标准 食品中维生素A、D、E的测定》中第三法“食品中维生素D的测定 液相色谱-串联质谱法”和第四法“食品中维生素D的测定 高效液相色谱法”。

本标准与GB5009.82—2016相比，主要变化如下：

——标准名称修改为《食品安全国家标准 食品中维生素D的测定》；  
——增加了在线柱切换-反相液相色谱法；  
——增加了样品预制备方法；  
——修改了液相色谱-串联质谱法的线性范围和仪器参考条件；  
——修改了附录中标准校正溶液的配制方法。

# 食品安全国家标准

# 食品中维生素D的测定

# 1 范围

本标准规定了食品中维生素D的测定方法。

第一法正相色谱净化-反相液相色谱法适用于添加了麦角钙化醇或胆钙化醇的食品中维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 的测定。

第二法在线柱切换-反相液相色谱法适用于食品中维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 的测定。

第三法液相色谱-串联质谱法适用于食品中维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 的测定。

# 第一法 正相色谱净化-反相液相色谱法

# 2 原理

试样经氢氧化钾乙醇溶液皂化，液液萃取净化、浓缩后，用正相高效液相色谱仪通过硅胶柱将维生素D与其他杂质分离，将收集的馏分浓缩后，再经反相色谱柱分离维生素 $\mathrm{D}_2$ 与维生素 $\mathrm{D}_3$ ，紫外检测器检测，内标法(或外标法)定量。当试样中不含维生素 $\mathrm{D}_2$ 时，可用维生素 $\mathrm{D}_2$ 作内标测定维生素 $\mathrm{D}_3$ ；当试样中不含维生素 $\mathrm{D}_3$ 时，可用维生素 $\mathrm{D}_3$ 作内标测定维生素 $\mathrm{D}_2$ 。否则，用外标法测定。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 无水乙醇 $\mathrm{(C_2H_6O)}$ ：色谱纯。  
3.1.2 抗坏血酸 $(\mathrm{C_6H_8O_6})$ 。  
3.1.3 2,6-二叔丁基对甲酚 $\mathrm{(C_{15}H_{24}O)}$ ：简称BHT。  
3.1.4 $\alpha$ -淀粉酶(CAS号：9000-90-2，中温淀粉酶，来源于芽孢杆菌属)：酶活力 $\geqslant 1.5$ U/mg。  
3.1.5 氢氧化钾(KOH)。  
3.1.6 正己烷 $(\mathrm{C_6H_{14}})$ 。  
3.1.7 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.8 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.9 环己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{12}\right)$ 。  
3.1.10 异丙醇 $(\mathrm{C_3H_8O})$ 。  
3.1.11 石油醚：沸程为 $30^{\circ}\mathrm{C} \sim 60^{\circ}\mathrm{C}$ 。

# 3.2 试剂配制

3.2.1 氢氧化钾溶液（ $50\%$ ，质量分数）：称取 $50\mathrm{g}$ 氢氧化钾，溶于 $50\mathrm{g}$ 水中，冷却后储存于聚乙烯瓶中。  
3.2.2 BHT-乙醇溶液 $(0.2\mathrm{g} / 100\mathrm{mL})$ ：称取 $1.0\mathrm{g}$ BHT，溶于 $500~\mathrm{mL}$ 无水乙醇中，临用前配制。  
3.2.3 异丙醇-环己烷-正己烷溶液 $(2 + 125 + 125)$ ：将异丙醇、环己烷和正己烷按 $2:125:125$ 的体积比混合均匀，超声脱气。  
3.2.4 甲醇-水溶液 $(19 + 1)$ ：将甲醇和水按 $19:1$ 的体积比混合均匀，超声脱气。

# 3.3 标准品

3.3.1 维生素 $\mathrm{D}_2$ 标准品：麦角钙化醇 $(\mathrm{C}_{28}\mathrm{H}_{44}\mathrm{O},\mathrm{CAS}$ 号：50-14-6)，纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.2 维生素 $\mathrm{D}_3$ 标准品：胆钙化醇（ $\mathrm{C}_{27}\mathrm{H}_{44}\mathrm{O}$ ，CAS号：67-97-0），纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 维生素 $\mathrm{D}_{2}$ 标准储备溶液 $(1000\mathrm{mg / L})$ ：准确称取 $50~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )维生素 $\mathrm{D}_2$ 标准品于小烧杯中，用无水乙醇溶解并转移至 $50~\mathrm{mL}$ 容量瓶中，定容至刻度，混匀。按附录A进行储备液浓度校正，将该储备溶液装至棕色试剂瓶中，密封后在一 $18^{\circ}C$ 下避光保存，保存期6个月。  
3.4.2 维生素 $\mathrm{D}_{3}$ 标准储备溶液 $(1000\mathrm{mg / L})$ ：准确称取 $50~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )维生素 $\mathrm{D}_{3}$ 标准品于小烧杯中，用无水乙醇溶解并转移至 $50~\mathrm{mL}$ 容量瓶中，定容至刻度，混匀。按附录A进行储备液浓度校正，将该储备溶液装至棕色试剂瓶中，密封后在 $-18^{\circ}C$ 下避光保存，保存期6个月。  
3.4.3 维生素 $\mathrm{D}_2$ 标准使用液 $(10.0\mathrm{mg / L})$ ：准确吸取维生素 $\mathrm{D}_2$ 标准储备液 $(1000\mathrm{mg / L})1.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用无水乙醇稀释至刻度，混匀。在 $-18^{\circ}C$ 下避光保存，保存期3个月。  
3.4.4 维生素 $\mathrm{D}_3$ 标准使用液 $(10.0\mathrm{mg / L})$ ：准确吸取维生素 $\mathrm{D}_3$ 标准储备液 $(1000\mathrm{mg / L})1.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用无水乙醇稀释至刻度，混匀。在 $-18^{\circ}C$ 下避光保存，保存期3个月。

# 3.4.5 标准系列工作溶液

3.4.5.1 当用维生素 $\mathrm{D}_{3}$ 作内标测定维生素 $\mathrm{D}_{2}$ 时：分别吸取维生素 $\mathrm{D}_{2}$ 标准使用液（ $10.0\mathrm{mg / L}$ ） $0.50\mathrm{mL}, 1.00\mathrm{mL}, 2.00\mathrm{mL}, 4.00\mathrm{mL}, 6.00\mathrm{mL}$ 和 $10.00\mathrm{mL}$ 棕色容量瓶中，各加入维生素 $\mathrm{D}_{3}$ 内标使用液（ $10.0\mathrm{mg / L}$ ） $2.50\mathrm{mL}$ ，加甲醇稀释至刻度，混匀。此标准系列工作溶液质量浓度分别为 $50\mu \mathrm{g / L}, 100\mu \mathrm{g / L}, 200\mu \mathrm{g / L}, 400\mu \mathrm{g / L}, 600\mu \mathrm{g / L}, 1000\mu \mathrm{g / L}$ ，维生素 $\mathrm{D}_{3}$ 内标质量浓度为 $250\mu \mathrm{g / L}$ 。临用前配制。  
3.4.5.2 当用维生素 $\mathrm{D}_2$ 作内标测定维生素 $\mathrm{D}_3$ 时：分别吸取维生素 $\mathrm{D}_3$ 标准使用液（ $10.0\mathrm{mg / L}$ ） $0.50\mathrm{mL}, 1.00\mathrm{mL}, 2.00\mathrm{mL}, 4.00\mathrm{mL}, 6.00\mathrm{mL}$ 和 $10.00\mathrm{mL}$ 棕色容量瓶中，各加入维生素 $\mathrm{D}_2$ 内标使用液（ $10.0\mathrm{mg / L}$ ） $2.50\mathrm{mL}$ ，加甲醇稀释至刻度，混匀。此标准系列工作溶液质量浓度分别为 $50\mu \mathrm{g / L}, 100\mu \mathrm{g / L}, 200\mu \mathrm{g / L}, 400\mu \mathrm{g / L}, 600\mu \mathrm{g / L}, 1000\mu \mathrm{g / L}$ ，维生素 $\mathrm{D}_2$ 内标质量浓度为 $250\mu \mathrm{g / L}$ 。临用前配制。  
3.4.5.3 当用外标法同时测定维生素 $\mathrm{D}_2$ 和 $\mathrm{D}_3$ 时：分别吸取维生素 $\mathrm{D}_2$ 标准使用液（ $10.0\mathrm{mg / L}$ ）、维生素 $\mathrm{D}_3$ 标准使用液（ $10.0\mathrm{mg / L}$ ）各 $0.50\mathrm{mL}, 1.00\mathrm{mL}, 2.00\mathrm{mL}, 4.00\mathrm{mL}, 6.00\mathrm{mL}$ 和 $10.00\mathrm{mL}$ 于 $100\mathrm{mL}$ 棕色容量瓶中，加甲醇稀释至刻度，混匀。此标准系列工作溶液质量浓度分别为 $50~\mu \mathrm{g / L}$ 、 $100~\mu \mathrm{g / L}, 200~\mu \mathrm{g / L}, 400~\mu \mathrm{g / L}, 600~\mu \mathrm{g / L}, 1000~\mu \mathrm{g / L}$ 。临用前配制。

# 3.5 材料

3.5.1 广泛 $\mathrm{pH}$ 试纸： $\mathrm{pH}$ 范围 $1 \sim 14$   
3.5.2 玻璃层析柱：柱径 $2\mathrm{cm}\sim 3\mathrm{cm}$ ，柱长 $10~\mathrm{cm}\sim 15~\mathrm{cm}$   
3.5.3 微孔滤膜：有机系，孔径 $0.45\mu \mathrm{m}$

# 4 仪器和设备

4.1 正相高效液相色谱仪：带紫外检测器，进样器配 $500\mu \mathrm{L}$ 定量环。  
4.2 反相高效液相色谱仪：带紫外检测器，进样器配 $100\mu \mathrm{L}$ 定量环。  
4.3 天平：感量为 $0.1\mathrm{mg}, 0.01\mathrm{g}$ 。  
4.4 紫外分光光度计：带 $1\mathrm{cm}$ 石英比色皿。  
4.5 磁力搅拌器：带加热、控温功能。  
4.6 恒温振荡器。  
4.7 旋转蒸发仪。  
4.8 恒温水浴锅。  
4.9 氮吹浓缩仪。  
4.10 超声波清洗器。  
4.11 分液漏斗萃取振荡器。

# 5 分析步骤

# 5.1 样品前处理

处理过程应避免紫外光照射。

# 5.1.1 试样制备

将一定数量的样品按要求经过缩分、粉碎、均质后，储存于样品袋中，避光冷藏，尽快测定。

# 5.1.1.1 婴幼儿配方食品、特殊医学用途配方食品、乳及乳制品、冷冻饮品和饮料

固体试样：称取均质后的试样约 $25\mathrm{g}(m_{1}$ ，精确至 $0.01\mathrm{g})$ 至 $250~\mathrm{mL}$ 干燥玻璃瓶中，加入约 $200~\mathrm{mL}$ 温水 $(40^{\circ}\mathrm{C}\sim 45^{\circ}\mathrm{C})$ ，记录加水后的溶液质量 $(m_2$ ，精确至 $0.01\mathrm{g})$ 。充分混匀溶解，室温避光放置 $15\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振摇 $30~\mathrm{s}$ 。称取制备后的浆液 $20\mathrm{g}\sim 50\mathrm{g}(m_3$ ，精确至 $0.01\mathrm{g})$ 至 $150~\mathrm{mL}$ 平底烧瓶中。按式(1)计算样品质量 $(m)$ 。

$$
m = \frac {m _ {1} \times m _ {3}}{m _ {2}} \tag {1}
$$

液体试样：称取摇匀后的试样 $20\mathrm{g}\sim 50\mathrm{g}(m$ ，精确至 $0.01\mathrm{g})$ 至 $150~\mathrm{mL}$ 平底烧瓶中。

# 5.1.1.2 谷物及其制品、焙烤食品、婴幼儿谷类辅助食品及其他含淀粉试样

称取均质后的试样 $5\mathrm{g}\sim 10\mathrm{g}(m$ ，精确至 $0.01\mathrm{g})$ 于 $150~\mathrm{mL}$ 平底烧瓶中，加入约 $20~\mathrm{mL}$ 温水 $(40^{\circ}C\sim$ $45^{\circ}C)$ ，混匀。婴幼儿谷类辅助食品可参考5.1.1.1先制成浆液，再称取制备后的浆液 $20\mathrm{g}\sim 50\mathrm{g}(m_{3}$ ，精确至 $0.01\mathrm{g})$ 至 $150~\mathrm{mL}$ 平底烧瓶中。

# 5.1.1.3 其他食品

称取均质后的固体试样 $5\mathrm{g}\sim 20\mathrm{g}(m$ ，精确至 $0.01\mathrm{g})$ 或液体试样 $20\mathrm{g}\sim 50\mathrm{g}(m$ ，精确至 $0.01\mathrm{g})$ 于

$150~\mathrm{mL}$ 平底烧瓶中，固体试样需加入 $20~\mathrm{mL} \sim 30~\mathrm{mL}$ 温水（ $40^{\circ}\mathrm{C} \sim 45^{\circ}\mathrm{C}$ ），混匀。

# 5.1.2 试样皂化

# 5.1.2.1 婴幼儿配方食品、特殊医学用途配方食品、乳及乳制品、冷冻饮品、饮料和其他食品

于5.1.1.1和5.1.1.3所述平底烧瓶中，加入 $100\mu \mathrm{L}$ 内标使用溶液 $(10.0\mathrm{mg / L})$ ，再加入 $1.0\mathrm{g}$ 抗坏血酸和 $30~\mathrm{mL}\sim 50~\mathrm{mL}$ BHT-乙醇溶液 $(0.2\mathrm{g} / 100\mathrm{mL})$ ，混匀，再加入 $20~\mathrm{mL}\sim 30~\mathrm{mL}$ 氢氧化钾溶液（ $50\%$ ，质量分数)，边加边振摇，混匀后于磁力搅拌器或恒温振荡器中皂化，温度 $80^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ ，时间 $30\mathrm{min}$ (或温度 $25^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}$ ，时间 $16\mathrm{h}\pm 2\mathrm{h}$ )，皂化后立即用冷水冷却至室温。

# 5.1.2.2 谷物及其制品、焙烤食品、婴幼儿谷类辅助食品及其他含淀粉试样

于5.1.1.2所述平底烧瓶中，加入 $100\mu \mathrm{L}$ 内标使用溶液（ $10.0\mathrm{mg / L}$ ）和 $1.0\mathrm{g}\alpha$ -淀粉酶，放入 $60^{\circ}C$ 恒温水浴振荡 $30\mathrm{min}$ ，向酶解液中加入 $1.0\mathrm{g}$ 抗坏血酸和 $30~\mathrm{mL}$ BHT-乙醇溶液 $(0.2\mathrm{g} / 100\mathrm{mL})$ ，混匀，再加入 $10~\mathrm{mL}\sim 20~\mathrm{mL}$ 氢氧化钾溶液( $50\%$ ，质量分数)，边加边振摇，混匀后于磁力搅拌器或恒温振荡器中皂化，温度 $80^{\circ}C\pm 2^{\circ}C$ ，时间 $30\mathrm{min}$ (或温度 $25^{\circ}C\pm 5^{\circ}C$ ，时间 $16\mathrm{h}\pm 2\mathrm{h}$ )，皂化后立即用冷水冷却至室温。

注：如用外标法，则不加内标，其余操作同上，皂化条件选择温度 $25\mathrm{^{\circ}C}\pm 5\mathrm{^{\circ}C}$ ，皂化时间 $16\mathrm{h}\pm 2\mathrm{h}$

# 5.1.3 试样提取与浓缩

# 5.1.3.1 试样提取

将上述皂化液用 $30~\mathrm{mL}$ 水转入 $250~\mathrm{mL}$ 分液漏斗中，加入 $50~\mathrm{mL}$ 石油醚，振荡萃取 $5\mathrm{min}$ ，将下层溶液转移至另一 $250~\mathrm{mL}$ 分液漏斗中，加入 $50~\mathrm{mL}$ 石油醚重复萃取1次～2次，合并石油醚层。用约 $150~\mathrm{mL}$ 水洗涤石油醚层，去除下层水相，重复洗涤至少3次，直至将石油醚层洗至中性（可用广泛 $\mathrm{pH}$ 试纸检测下层溶液）。

# 5.1.3.2 试样浓缩

将洗涤后的石油醚层缓慢流过预先填充厚度 $3\mathrm{cm}\sim 5\mathrm{cm}$ 无水硫酸钠粉末的玻璃层析柱，滤入 $250~\mathrm{mL}$ 旋转蒸发瓶或氮吹浓缩管中，用 $15~\mathrm{mL}$ 石油醚冲洗分液漏斗及玻璃层析柱内无水硫酸钠2次，合并入蒸发瓶或氮吹浓缩管中，并将其接在旋转蒸发仪或氮吹浓缩仪上，于 $40^{\circ}C$ 水浴中旋蒸或氮吹浓缩，待石油醚剩余约 $1\mathrm{mL}$ 时，取下蒸发瓶或浓缩管，用石油醚将浓缩液转移至 $2\mathrm{mL}$ 容量瓶中，氮吹至近干，用正己烷定容至刻度，过 $0.45\mu \mathrm{m}$ 微孔滤膜，供正相高效液相色谱净化。

# 5.2 仪器参考条件

# 5.2.1 维生素D待测液的净化

# 5.2.1.1 正相色谱参考条件

正相色谱参考条件如下：

a）色谱柱：硅胶柱，柱长 $250~\mathrm{mm}$ ，柱内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或相当者；  
b）流动相：异丙醇-环己烷-正己烷溶液 $(2 + 125 + 125)$   
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d）波长： $264~\mathrm{nm}$   
e) 柱温： $35^{\circ}\mathrm{C}$ ；  
f) 进样量: $500 \mu \mathrm{L}$ 。

# 5.2.1.2 正相色谱系统适用性试验

各取维生素 $\mathrm{D}_2$ 标准使用液和维生素 $\mathrm{D}_3$ 标准使用液 $200\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 具塞试管中， $40^{\circ}\mathrm{C}$ 水浴中氮气吹至近干。残渣用 $10~\mathrm{mL}$ 正己烷振荡溶解。取该溶液 $500\mu \mathrm{L}$ 注入正相高效液相色谱仪中测定，确定维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 的保留时间。

# 5.2.1.3 试样待测液正相色谱净化

将 $500\mu \mathrm{L}$ 待测液注入正相液相色谱仪中，根据维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 标准溶液保留时间收集维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 馏分于试管中。将试管置于 $40^{\circ}\mathrm{C}$ 水浴中氮气吹至近干，准确加入 $1.0~\mathrm{mL}$ 甲醇，振荡溶解残渣，即为试样测定液。

注：全自动(在线)或半自动(离线)的馏分收集器可优化操作参数后使用。

# 5.2.2 反相液相色谱参考条件

反相液相色谱参考条件如下：

a）色谱柱：多环芳烃（PAH） $\mathrm{C_{18}}$ 柱，柱长 $150~\mathrm{mm}$ ，柱内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或相当者；  
b）流动相：甲醇-水溶液 $(19 + 1)$   
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d） 波长： $264~\mathrm{nm}$   
e) 柱温： $35^{\circ}\mathrm{C}$ ；  
f) 进样量: $100 \mu \mathrm{L}$ 。

# 5.3 标准曲线的制作

将标准系列工作溶液分别注入反相液相色谱仪中，内标法以标准系列工作溶液中维生素 $\mathrm{D}_2$ （或维生素 $\mathrm{D}_3$ )与其对应内标的质量比为横坐标，以维生素 $\mathrm{D}_2$ (或维生素 $\mathrm{D}_3$ )的峰面积与其对应内标的峰面积比为纵坐标，绘制标准曲线。外标法以标准系列工作溶液中维生素 $\mathrm{D}_2$ (或维生素 $\mathrm{D}_3$ )的浓度为横坐标，维生素 $\mathrm{D}_2$ (或维生素 $\mathrm{D}_3$ )的峰面积为纵坐标，绘制标准曲线。维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 标准溶液的色谱图参见附录B中图B.1。

# 5.4 试样溶液的测定

# 5.4.1 定性分析

将试样溶液注入液相色谱仪中进行测定，同样测试条件下，试样溶液中维生素 $\mathrm{D}_2$ 、维生素 $\mathrm{D}_3$ 的保留时间与标准工作溶液中相应的保留时间相比，偏差应在 $\pm 2.5\%$ 以内。

# 5.4.2 定量测定

将试样溶液注入反相液相色谱仪中，得到维生素 $\mathrm{D}_2$ （或维生素 $\mathrm{D}_3$ )的峰面积，根据相应的标准曲线得到试样溶液中维生素 $\mathrm{D}_2$ （或维生素 $\mathrm{D}_3$ )的浓度。试样溶液中的响应值应在标准曲线线性范围内，超过线性范围，可适当减少取样量重新测定。

# 5.5 空白试验

不称取试样，按试样的皂化、提取和浓缩分析步骤操作，应不含有干扰待测组分的物质。内标法需要做不加内标的样品试验，确认加入内标的可行性。

# 6 分析结果的表述

试样中维生素 $\mathrm{D}_2$ （或维生素 $\mathrm{D}_3$ )的含量，内标法按式(2)和式(3)计算，外标法按式(4)计算。

$$
X = \frac {N \times 1 0 0}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中维生素 $\mathrm{D}_2$ (或维生素 $\mathrm{D}_3$ ) 的含量, 单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$ ;

$N$ ——试样溶液中维生素 $\mathrm{D}_2$ (或维生素 $\mathrm{D}_3$ ) 与其对应内标的峰面积比值对应的质量, 单位为纳克 $(\mathrm{ng})$ , 按式(3)计算;

100 ——由每克换算为每百克的换算系数；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ ；

1000 由 ng 换算为 $\mu \mathrm{g}$ 的换算系数。

$$
N = K \times N _ {\mathrm {i s}} \quad \dots \tag {3}
$$

式中：

$K$ ——从标准曲线查得的质量比；

$N_{\mathrm{is}}$ ——试样中加入的内标质量，单位为纳克 $(\mathrm{ng})$ 。

$$
X = \frac {c \times V _ {1} \times V _ {3} \times 1 0 0}{m \times V _ {2} \times 1 0 0 0} \times f \quad \dots \tag {4}
$$

式中：

$X$ ——试样中维生素 $\mathrm{D}_2$ (或维生素 $\mathrm{D}_3$ ) 的含量, 单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$

$c$ ——根据标准曲线得到的进样溶液中维生素 $\mathrm{D}_2$ (或维生素 $\mathrm{D}_3$ ) 的质量浓度, 单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ;

$V_{1}$ ——正相色谱净化时试样溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——反相色谱测定时试样溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

100 ——由每克换算为每百克的换算系数；

$m$ ——试样的取样量，单位为克(g)；

$V_{2}$ ——正相色谱净化时试样溶液取用体积，单位为毫升 $(\mathrm{mL})$ ；

1000——由 $\mu \mathrm{g} / \mathrm{L}$ 换算为 $\mu \mathrm{g} / \mathrm{mL}$ 的换算系数；

$f$ ——稀释因子。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，结果保留3位有效数字。

如试样中同时含有维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ ，维生素D的测定结果以维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 的含量之和计算。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 第二法 在线柱切换-反相液相色谱法

# 9 原理

试样经氢氧化钾乙醇溶液皂化，液液萃取或固相萃取净化、浓缩后，一维液相色谱通过 $\mathrm{C_8}$ 柱将维生素D与其他杂质分离后，由柱切换阀转入二维液相色谱中，通过 $\mathrm{C_{18}}$ 柱分离维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ ，紫外检测器检测，内标法(或外标法)定量。当试样中不含维生素 $\mathrm{D}_2$ 时，可用维生素 $\mathrm{D}_2$ 作内标测定维生素 $\mathrm{D}_3$ ；当试样中不含维生素 $\mathrm{D}_3$ 时，可用维生素 $\mathrm{D}_3$ 作内标测定维生素 $\mathrm{D}_2$ 。否则，用外标法测定。

# 12.1.3.1 液液萃取法

于上述装有皂化液的离心管中加入 $5\mathrm{mL}$ 水，混匀，加入 $20~\mathrm{mL}$ 乙酸乙酯-正己烷混合溶液 $(3 + 2)$ ，振荡提取 $10\mathrm{min},8000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，将上层溶液转移至另一 $50~\mathrm{mL}$ 的离心管中，于原离心管中再加入 $10~\mathrm{mL}$ 乙酸乙酯-正己烷混合溶液 $(3 + 2)$ ，再次振荡提取 $10\mathrm{min}$ ，高速离心机 $8000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，合并上层有机相于同一离心管中，加水至 $45~\mathrm{mL}$ ，振荡 $30~\mathrm{s},8000~\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，将上层有机相转移至旋蒸瓶中，于 $40^{\circ}C$ 旋蒸至约 $1\mathrm{mL}$ ，用乙酸乙酯-正己烷混合溶液 $(3 + 2)$ 转移至 $10~\mathrm{mL}$ 试管中，置于氮吹仪中吹至近干，用约 $4\mathrm{mL}$ 乙腈-甲醇溶液 $(3 + 1)$ 分3次溶解并转移至 $5\mathrm{mL}$ 容量瓶中，加水定容至刻度，混匀，过 $0.45\mu \mathrm{m}$ 微孔滤膜，供液相色谱测定。

# 12.1.3.2 固相萃取法

固相萃取法适用于婴幼儿配方食品、特殊医学用途配方食品、调制乳(粉)、豆浆(粉)。

于上述皂化液中加入 $0\mathrm{mL}\sim 15\mathrm{mL}$ 去离子水(使用不同品牌的固相萃取柱，去离子水的加量不同，需经验证后确定)，涡旋混匀， $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。全部上清液转移至固相萃取柱中(临用前依次用 $6\mathrm{mL}$ 甲醇、 $6\mathrm{mL}$ 水活化平衡)上样，过柱速度控制在不超过60滴/min。用 $10~\mathrm{mL}$ 甲醇-水溶液 $(1+$ 19)洗涤离心管，涡旋 $30~\mathrm{s},8000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将上清液一并转移到固相萃取柱上样，重复操作2次。上样完毕，负压抽干固相萃取柱，用 $8\mathrm{mL}$ 乙腈-甲醇溶液 $(3 + 1)$ 分3次洗脱，洗脱液加水定容至 $10~\mathrm{mL}$ ，过 $0.45\mu \mathrm{m}$ 微孔滤膜，取滤液进样。

注：如仪器灵敏度或进样体积受限，可将洗脱液氮吹至一定体积，再加水定容至适当体积，保持最终进样液中的溶剂体系为初始流动相体系。全自动在线固相萃取法可优化操作参数后使用。

# 12.4.1 定性分析

# 13 分析结果的表述

试样中维生素 $\mathrm{D}_2$ （或维生素 $\mathrm{D}_3$ )的含量，内标法按式(2)和式(3)计算，外标法按式(5)计算。

$$
X = \frac {c \times V \times 1 0 0}{m \times 1 0 0 0} \times f \tag {5}
$$

式中：

$X$ ——试样中维生素 $\mathrm{D}_2$ （或维生素 $\mathrm{D}_3$ )的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$

$c$ ——根据标准曲线得到的进样溶液中维生素 $\mathrm{D}_2$ (或维生素 $\mathrm{D}_3$ ) 的质量浓度, 单位为微克每升

$(\mu \mathrm{g} / \mathrm{L})$

$V$ ——进样溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

100 ——由每克换算为每百克的换算系数；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ ；

1000——由 $\mu \mathrm{g} / \mathrm{L}$ 换算为 $\mu \mathrm{g} / \mathrm{mL}$ 的换算系数；

$f$ ——稀释因子。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，结果保留3位有效数字。

如试样中同时含有维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ ，维生素D的测定结果以维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 的含量之和计算。

# 14 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 16 原理

# 19.1.1.3 油脂及其制品、肉及肉制品、水产及其制品、蛋及蛋制品

# 19.1.2 试样皂化

# 19.1.2.1 婴幼儿配方食品、特殊医学用途配方食品、乳及乳制品、冷冻饮品和饮料等试样

按19.1.1.1、19.1.1.3、19.1.1.4描述的样品前处理方法称取适量试样于 $50~\mathrm{mL}$ 离心管中，加入 $100\mu \mathrm{L}$ 维生素 $\mathrm{D}_2 - \mathrm{d}_3$ 和维生素 $\mathrm{D}_3 - \mathrm{d}_3$ 混合内标使用液，再加入 $0.4\mathrm{g}$ 抗坏血酸，涡旋 $1\mathrm{min}$ ，加入 $10~\mathrm{mL}$ BHT-乙醇溶液 $(0.2\mathrm{g} / 100\mathrm{mL})$ ，涡旋 $30~\mathrm{s}$ ，再加入 $5\mathrm{mL}$ 氢氧化钾溶液（ $50\%$ ，质量分数），涡旋混匀后于恒温振荡器中皂化，温度 $80^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ ，时间 $30\mathrm{min}$ （或温度 $25^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}$ ，时间 $16\mathrm{h}\pm 2\mathrm{h}$ )，皂化后立即用冷水冷却至室温。

# 19.1.2.2 谷物及其制品、焙烤食品、婴幼儿谷类辅助食品及其他含淀粉试样

在19.1.1.2描述的 $150~\mathrm{mL}$ 平底烧瓶中，加入 $400\mu \mathrm{L}$ 维生素 $\mathrm{D}_2 - \mathrm{d}_3$ 和维生素 $\mathrm{D}_3 - \mathrm{d}_5$ 混合内标使用液以及 $1.0\mathrm{g}\alpha$ -淀粉酶，放入1颗磁力搅拌子，加上瓶塞，放入 $60^{\circ}\mathrm{C}$ 磁力搅拌器中酶解 $30\mathrm{min}$ ，立即冷却至室温，向酶解液中加入 $1.0\mathrm{g}$ 抗坏血酸和 $30~\mathrm{mL}$ BHT-乙醇溶液 $(0.2\mathrm{g} / 100\mathrm{mL})$ ，混匀。再加入 $10~\mathrm{mL}\sim$ $20~\mathrm{mL}$ 氢氧化钾溶液( $50\%$ ，质量分数)，边加边振摇，混匀后于磁力搅拌器或恒温振荡器中皂化，温度 $80^{\circ}\mathrm{C}\pm$ $2^{\circ}\mathrm{C}$ ，时间 $30\mathrm{min}$ (或温度 $25^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}$ ，时间 $16\mathrm{h}\pm 2\mathrm{h}$ )，皂化后立即用冷水冷却至室温。用乙醇-水溶液 $(2 + 3)$ 将皂化液转移至 $100~\mathrm{mL}$ 的容量瓶中，定容至刻度，摇匀，准确移取 $25~\mathrm{mL}$ 皂化液于 $50~\mathrm{mL}$ 的离心管中。

# 19.1.2.3 含蜂蜡等胶质样品和其他食品

在19.1.1.5描述的 $150~\mathrm{mL}$ 平底烧瓶中，加入 $400~\mu \mathrm{L}$ 维生素 $\mathrm{D}_2 - \mathrm{d}_3$ 和维生素 $\mathrm{D}_3 - \mathrm{d}_3$ 混合内标溶液，加入 $1.0\mathrm{g}$ 抗坏血酸和 $30~\mathrm{mL}$ BHT-乙醇溶液 $(0.2\mathrm{g} / 100\mathrm{mL})$ ，放入1颗磁力搅拌子，将平底烧瓶置于磁力搅拌器中搅拌 $10\mathrm{min}$ ，混匀，再加入约 $20~\mathrm{mL}$ 温水 $(40^{\circ}\mathrm{C}\sim 45^{\circ}\mathrm{C})$ ，混匀。再加入 $10~\mathrm{mL}\sim 20~\mathrm{mL}$ 氢氧化钾溶液 $(50\%)$ ，质量分数)，边加边振摇，混匀后于磁力搅拌器或恒温振荡器中皂化，温度 $80^{\circ}\mathrm{C}\pm$ $2^{\circ}\mathrm{C}$ ，时间 $30\mathrm{min}$ (或温度 $25^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}$ ，时间 $16\mathrm{h}\pm 2\mathrm{h}$ )，皂化后立即用冷水冷却至室温。用乙醇-水溶液 $(2 + 3)$ 将皂化液转移至 $100~\mathrm{mL}$ 容量瓶中，摇匀，准确移取 $25~\mathrm{mL}$ 皂化液于 $50~\mathrm{mL}$ 离心管中。

# 19.1.3 试样提取

向上述离心管中加入 $20~\mathrm{mL}$ 正己烷，振荡提取 $3\mathrm{min},8000\mathrm{r / min}$ 离心 $3\mathrm{min}$ 。转移上层清液到另一个 $50~\mathrm{mL}$ 离心管，加入 $25~\mathrm{mL}$ 水，振荡 $30~s$ ，在 $8000~\mathrm{r / min}$ 条件下离心 $3\mathrm{min}$ ，待净化。

将固相萃取柱依次用 $8\mathrm{mL}$ 乙酸乙酯、 $8\mathrm{mL}$ 正己烷活化平衡，将上层提取液转移至固相萃取柱净化。用 $6\mathrm{mL}$ 乙酸乙酯-正己烷混合溶液 $(1 + 19)$ 淋洗， $6\mathrm{mL}$ 乙酸乙酯-正己烷混合溶液 $(3 + 17)$ 洗脱。洗脱液置于 $40^{\circ}\mathrm{C}$ 下氮吹浓缩装置吹至近干，加入 $1.0\mathrm{mL}$ 甲醇，超声 $30~\mathrm{s}$ ，涡旋 $30~\mathrm{s}$ ，过 $0.22\mu \mathrm{m}$ 微孔滤膜供仪器测定。

# 19.6 分析结果的表述

试样中维生素 $\mathrm{D}_2$ （或维生素 $\mathrm{D}_3$ )的含量，按式(6)和式(7)计算。

$$
X = \frac {N \times 1 0 0}{m \times 1 0 0 0} \tag {6}
$$

式中：

$X$ ——试样中维生素 $\mathrm{D}_2$ (或维生素 $\mathrm{D}_3$ ) 的含量, 单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$ ;

$N$ ——试样溶液中维生素 $\mathrm{D}_2$ （或维生素 $\mathrm{D}_3$ )与其同位素内标的峰面积比值对应的质量，单位为纳克(ng)，按式(7)计算；

100 ——由每克换算为每百克的换算系数；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ ；

1000——由 $\mathrm{ng}$ 换算为 $\mu \mathrm{g}$ 的换算系数。

$$
N = K \times N _ {\mathrm {i s}} \quad \dots \tag {7}
$$

式中：

$K$ ——从标准曲线查得的质量比；

$N_{\mathrm{is}}$ ——试样中加入的内标的质量，单位为纳克 $(\mathrm{ng})$ 。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，结果保留3位有效数字。

如试样中同时含有维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ ，维生素D的测定结果以维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 的含量之和计算。

# 19.7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

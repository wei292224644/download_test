# 食品中伏马毒素的测定

2016-08-31 发布

2017-03-01 实施

# 前言

本标准代替GB/T25228—2010《粮油检验 玉米及其制品中伏马毒素含量测定 免疫亲和柱净化高效液相色谱法和荧光光度法》、SN/T1958—2007《进出口食品中伏马毒素 $\mathbf{B}_1$ 残留量检测方法 酶联免疫吸附法》、SN/T1572—2005《进出口粮谷中伏马毒素检验方法 高效液相色谱法》。

本标准将以上标准进行了整合，主要修订如下：

——标准名称修改为“食品安全国家标准 食品中伏马毒素的测定”；  
——伏马毒素种类增加为伏马毒素 $\mathrm{B}_{1} 、 \mathrm{~B}_{2} 、 \mathrm{~B}_{3}$ 三种；  
——增加了免疫亲和柱净化-柱后衍生高效液相色谱法作为第一法；  
——增加了高效液相色谱-串联质谱联用法作为第二法；  
——取消荧光光度法；  
——改变了样品提取溶液；  
——改变了免疫亲和柱净化-柱前衍生高效液相色谱法流动相的组成。

# 食品安全国家标准

# 食品中伏马毒素的测定

# 1 范围

本标准规定了玉米及其制品中伏马毒素 $\mathrm{B}_{1}$ 、伏马毒素 $\mathrm{B}_{2}$ 、伏马毒素 $\mathrm{B}_{3}$ （以下简写为 $\mathrm{FB}_{1} 、 \mathrm{FB}_{2} 、 \mathrm{FB}_{3}$ ）的测定方法。

本标准第一法为免疫亲和柱净化-柱后衍生高效液相色谱法，第二法为高效液相色谱-串联质谱联用法，第三法为免疫亲和柱净化-柱前衍生高效液相色谱法，适用于玉米及其制品中伏马毒素的测定。

# 第一法 免疫亲和柱净化-柱后衍生高效液相色谱法

# 2 原理

样品用乙腈-水溶液提取，经稀释后过免疫亲和柱净化，去除脂肪、蛋白质、色素及碳水化合物等干扰物质。经高效液相色谱分离后柱后邻苯二甲醛衍生，荧光检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法使用的试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.3 乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.4 氢氧化钠(NaOH)。  
3.1.5 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.6 磷酸氢二钠 $\mathrm{(Na_2HPO_4)}$   
3.1.7 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$ 。  
3.1.8 氯化钾(KCl)。  
3.1.9 硼砂 $\mathrm{(Na_2B_4O_7\bullet 10H_2O)}$ 。  
3.1.10 2-巯基乙醇 $(\mathrm{C}_2\mathrm{H}_6\mathrm{OS})$ 。  
3.1.11 邻苯二甲醛 $(\mathrm{OPA},\mathrm{C}_8\mathrm{H}_6\mathrm{O}_2)$ 。  
3.1.12 吐温-20 $(\mathrm{C}_{58}\mathrm{H}_{114}\mathrm{O}_{26})$ 。

# 3.2 溶液配制

3.2.1 甲酸水溶液 $(0.1\%)$ ：吸取 $1\mathrm{mL}$ 甲酸，加入到 $999~\mathrm{mL}$ 水中，混合均匀。  
3.2.2 乙腈-水溶液 $(50 + 50)$ ：分别量取 $500~\mathrm{mL}$ 乙腈和 $500~\mathrm{mL}$ 水，混合均匀。  
3.2.3 乙腈-水溶液 $(20 + 80)$ ：分别量取 $200~\mathrm{mL}$ 乙腈和 $800~\mathrm{mL}$ 水，混合均匀。

3.2.4 甲醇-乙酸溶液 $(98 + 2)$ ：吸取 $2\mathrm{mL}$ 乙酸，加入到 $98~\mathrm{mL}$ 甲醇中，混合均匀。  
3.2.5 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：准确称取氢氧化钠 $4.0\mathrm{g}$ ，溶于 $100~\mathrm{mL}$ 水，混合均匀。  
3.2.6 磷酸盐缓冲液(PBS)：称取 $8.0\mathrm{g}$ 氯化钠、 $1.2\mathrm{g}$ 磷酸氢二钠、 $0.2\mathrm{g}$ 磷酸二氢钾、 $0.2\mathrm{g}$ 氯化钾，用 $980~\mathrm{mL}$ 水溶解，用盐酸调整 $\mathrm{pH}$ 至7.4，用水稀释至 $1000~\mathrm{mL}$ ，混合均匀。  
3.2.7 吐温-20/PBS溶液 $(0.1\%)$ ：吸取 $1\mathrm{mL}$ 吐温-20，加入磷酸盐缓冲液(3.2.6)并稀释至 $1000~\mathrm{mL}$ 混合均匀。  
3.2.8 硼砂溶液 $(0.05\mathrm{mol} / \mathrm{L},\mathrm{pH}10.5)$ ：称取硼砂 $19.1\mathrm{g}$ ，溶于 $980~\mathrm{mL}$ 水中，用氢氧化钠溶液调 $\mathrm{pH}$ 至10.5，用水稀释至 $1000~\mathrm{mL}$ ，混合均匀。  
3.2.9 衍生溶液: 称取 $2.0 \mathrm{~g}$ 邻苯二甲醛, 溶于 $20 \mathrm{~mL}$ 甲醇中, 用硼砂溶液 $(0.05 \mathrm{~mol} / \mathrm{L}, \mathrm{pH} 10.5)$ (3.2.8) 稀释至 $500 \mathrm{~mL}$ , 加入 2-巯基乙醇 $500 \mu \mathrm{L}$ , 混匀, 装入棕色瓶中, 现用现配。

# 3.3 标准品

3.3.1 伏马毒素 $\mathrm{B_1(FB_1,C_{34}H_{59}NO_{15})}$ ，纯度 $\geqslant 95\%$ ，或有证标准溶液。  
3.3.2 伏马毒素 $\mathrm{B_2(FB_2,C_{34}H_{59}NO_{14})}$ ，纯度 $\geqslant 95\%$ ，或有证标准溶液。  
3.3.3 伏马毒素 $\mathrm{B_3(FB_3,C_{34}H_{59}NO_{14})}$ ，纯度 $\geqslant 95\%$ ，或有证标准溶液。

# 3.4 标准溶液配制

3.4.1 标准储备溶液 $(0.1\mathrm{mg / mL})$ ：分别准确称取 $\mathrm{FB}_1$ 、 $\mathrm{FB}_2$ 、 $\mathrm{FB}_3$ 各 $0.01\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ) 至小烧杯中，用乙腈-水溶液(3.2.2)溶解，并转移至 $100~\mathrm{mL}$ 容量瓶中，定容至刻度。此溶液密封后避光一 $20^{\circ}C$ 保存。有效期6个月。  
3.4.2 混合标准溶液：准确吸取 $\mathrm{FB_1}$ 标准储备液 $1\mathrm{mL},\mathrm{FB_2}$ 和 $\mathrm{FB_3}$ 标准储备液 $0.5\mathrm{mL}$ 至同一 $10~\mathrm{mL}$ 容量瓶中，加乙腈-水溶液(3.2.2)稀释至刻度，得到 $\mathrm{FB_1}$ 浓度为 $10~\mu \mathrm{g / mL},\mathrm{FB_2}$ 和 $\mathrm{FB_3}$ 浓度为 $5\mu \mathrm{g / mL}$ 的混合标准溶液。再稀释10倍，得到 $\mathrm{FB_1}$ 浓度为 $1\mu \mathrm{g / mL},\mathrm{FB_2}$ 和 $\mathrm{FB_3}$ 浓度为 $0.5\mu \mathrm{g / mL}$ 的混合标准溶液。此溶液密封后避光 $4^{\circ}C$ 保存，有效期6个月。  
3.4.3 混合标准工作溶液：准确吸取混合标准溶液，用乙腈-水溶液(3.2.3)稀释，配制成 $\mathrm{FB_1}$ 浓度依次为 $20\mathrm{ng / mL},80\mathrm{ng / mL},160\mathrm{ng / mL},240\mathrm{ng / mL},320\mathrm{ng / mL},400\mathrm{ng / mL},\mathrm{FB_2}$ 和 $\mathrm{FB_3}$ 浓度依次为 $10\mathrm{ng / mL},40\mathrm{ng / mL},80\mathrm{ng / mL},120\mathrm{ng / mL},160\mathrm{ng / mL},200\mathrm{ng / mL}$ 的系列混合标准工作溶液。

# 4 仪器和设备

4.1 高效液相色谱仪，带荧光检测器。  
4.2 柱后衍生系统。  
4.3 天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
4.4 均质器。  
4.5 振荡器。  
4.6 氮吹仪。  
4.7 离心机：转速 $\geqslant 4000\mathrm{r / min}$   
4.8 免疫亲和柱(柱容量 $\geqslant 5000\mathrm{ng}$ ， $\mathrm{FB}_{1}$ 柱回收率 $\geqslant 80\%$ )（柱容量及柱回收率验证方法参见附录B）。注：对于每个批次的亲和柱在使用前需进行质量检验。  
4.9 微孔滤膜： $0.45\mu \mathrm{m}$ ，有机型。

# 5 分析步骤

# 5.1 样品制备

将固体样品按四分法缩分至 $1\mathrm{kg}$ ，全部用谷物粉碎机磨碎并细至粒度小于 $1\mathrm{mm}$ ，混匀分成2份作

为试样，分别装入洁净的容器内，密封，标识后置于 $4^{\circ}\mathrm{C}$ 下避光保存。

玉米油样品直接取2份作为试样，分别装入洁净的容器内，密封，标识后置于 $4^{\circ}\mathrm{C}$ 下避光保存。

在制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 5.2 试样提取

准确称取固体样品 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 样品于 $50~\mathrm{mL}$ 离心管中，加入 $20~\mathrm{mL}$ 乙腈-水溶液(3.2.2)，涡旋或振荡提取 $20\mathrm{min}$ ，取出后，在 $4000\mathrm{r / min}$ 下离心 $5\mathrm{min}$ ，将上清液转移至另一离心管中。

玉米油样品操作同固体样品，提取液在下层。

# 5.3 试样净化

取 $2\mathrm{mL}$ 提取液，加入 $47~\mathrm{mL}$ 吐温-20/PBS溶液(3.2.7)，混合均匀后过免疫亲和柱，流速控制在 $1\mathrm{mL} / \mathrm{min}\sim 3\mathrm{mL} / \mathrm{min}$ ，用 $10~\mathrm{mL}$ PBS缓冲液淋洗免疫亲和柱，分别用 $1\mathrm{mL}$ 甲醇-乙酸溶液(3.2.4)洗脱免疫亲和柱三次，收集洗脱液， $55^{\circ}C$ 下氮吹至干，加入 $1\mathrm{mL}$ 乙腈-水溶液(3.2.3)溶解残渣。涡旋 $30~s$ ，过 $0.45\mu \mathrm{m}$ 微孔滤膜后，收集于进样瓶中，待测。

注：由于不同厂商提供的免疫亲和柱操作程序可能不同，实际操作时，请参照厂商提供的操作说明和程序使用。

# 5.5 试样溶液的测定

在5.4项色谱条件下，将 $50.0\mu \mathrm{L}$ 系列伏马毒素混合标准工作溶液(3.4.3)按浓度从低到高依次注入高效液相色谱仪；待仪器条件稳定后，以目标物质的浓度为横坐标（ $x$ 轴），目标物质的峰面积为纵坐标（ $y$ 轴），对各个数据点进行最小二乘线性拟合，标准工作曲线按式(1)计算：

$$
y = a x + b \quad \dots \tag {1}
$$

式中：

$y$ ——目标物质的峰面积比；

$a$ ——回归曲线的斜率；

$x$ ——目标物质的浓度；

$b$ ——回归曲线的截距。

标准工作溶液和样液中待测物的响应值均应在仪器线性响应范围内，如果样品含量超过标准曲线范围，需稀释后再测定。

# 5.6 空白试验

不称取试样，按5.2和5.3的步骤做空白实验。应确认不含有干扰待测组分的物质。

# 6 分析结果的表述

待测样品中 $\mathrm{FB}_1$ 、 $\mathrm{FB}_2$ 、 $\mathrm{FB}_3$ 的含量按式(2)计算：

$$
X = \frac {c _ {i} \times V \times f}{m} \tag {2}
$$

式中：

$X$ ——待测样品中 $\mathrm{FB}_1, \mathrm{FB}_2, \mathrm{FB}_3$ 的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c_{i}$ ——待测物进样液中 $\mathrm{FB}_{1},\mathrm{FB}_{2},\mathrm{FB}_{3}$ 的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$

$V$ ——定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试液稀释倍数；

$m$ ——样品的称样量，单位为克 $(\mathrm{g})$ 。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留两位有效数字。

# 7 精密度

样品中伏马毒素含量在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当称样量为 $5\mathrm{g}$ 时， $\mathrm{FB}_1$ 、 $\mathrm{FB}_2$ 、 $\mathrm{FB}_3$ 的检出限分别为 $17~\mu \mathrm{g / kg},8~\mu \mathrm{g / kg},8~\mu \mathrm{g / kg}$ ；定量限分别为 $50~\mu \mathrm{g / kg},25~\mu \mathrm{g / kg},25~\mu \mathrm{g / kg}$ 。

# 第二法 高效液相色谱-串联质谱联用法

# 9 原理

样品加入同位素内标，乙腈-水溶液提取，经稀释后过免疫亲和柱或强阴离子交换固相萃取柱净化，去除脂肪、蛋白质、色素及碳水化合物等干扰物质。净化液中的伏马毒素经过高效液相色谱分离，串联质谱检测，同位素内标法定量。

# 12.6 定量测定

在5.4高效液相色谱-串联质谱联用分析条件下，将 $10.0\mu \mathrm{L}$ 系列伏马毒素混合标准溶液(10.4.4)按浓度从低到高依次注入高效液相色谱-串联质谱联用仪；待仪器条件稳定后，以目标物质和内标的浓度比为横坐标 $(x$ 轴)，目标物质和内标的峰面积比为纵坐标 $(y$ 轴)，对各个数值点进行最小二乘线性拟合，标准工作曲线按式(3)计算：

$$
y = a x + b \quad \dots \tag {3}
$$

式中：

$y$ ——目标物质/内标的峰面积比；

$a$ ——回归曲线的斜率；

$x$ ——目标物质/内标的浓度比；

$b$ ——回归曲线的截距。

标准工作溶液和样液中待测物的响应值均应在仪器线性响应范围内，如果含量超过标准曲线范围，则重新取样，增加相应内标添加量，使内标浓度与待测液浓度相匹配，然后稀释到适当浓度后分析。

# 13 分析结果的表述

本方法采用内标法定量。

含量按式(4)计算：

$$
X = \frac {c \times c _ {i} \times A \times A _ {\mathrm {s i}} \times V}{c _ {\mathrm {s i}} \times A _ {i} \times A _ {\mathrm {s}} \times m} \tag {4}
$$

式中：

$X$ ——样品中待测组分的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c$ ——标准溶液中待测组分的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{i}$ ——测定液中待测组分的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A$ ——测定液中待测组分的峰面积；

$A_{\mathrm{si}}$ ——标准溶液中内标物质的峰面积；

$V$ ——定容体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{si}$ ——标准溶液中内标物质的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A_{i}$ ——测定液中内标物质的峰面积；

$A_{\mathrm{s}}$ ——标准溶液中待测组分的峰面积；

$m$ ——样品称样量，单位为克 $(\mathrm{g})$ 。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留两位有效数字。

# 19.1 样品制备

将固体样品按四分法缩分至 $1\mathrm{kg}$ ，全部用谷物粉碎机磨碎并细至粒度小于 $1\mathrm{mm}$ ，混匀分成2份作为试样，分别装入洁净的容器内，密封，标识后置于 $4^{\circ}\mathrm{C}$ 下避光保存。

玉米油样品直接取2份作为试样，分别装入洁净的容器内，密封，标识后置于 $4^{\circ}\mathrm{C}$ 下避光保存。

在制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 19.2 试样提取

准确称取固体样品 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )样品于 $50~\mathrm{mL}$ 离心管中，加入 $20~\mathrm{mL}$ 乙腈-水(17.2.2)，涡旋或震荡提取 $20\mathrm{min}$ ，取出后，在 $4000\mathrm{r / min}$ 下离心 $5\mathrm{min}$ ，将上清液转移至另一离心管中。

玉米油样品操作同固体样品，提取液在下层。

# 19.3 试样净化

取 $2\mathrm{mL}$ 提取液，加入 $47~\mathrm{mL}$ 吐温-20/PBS溶液(17.2.7)，混合均匀后过免疫亲和柱，流速控制在 $1\mathrm{mL / min}\sim 3\mathrm{mL / min}$ ，用 $10~\mathrm{mL}$ PBS缓冲液淋洗免疫亲和柱，分别用 $1\mathrm{mL}$ 甲醇-乙酸溶液(17.2.4)洗脱免疫亲和柱三次，收集洗脱液， $55^{\circ}C$ 下氮吹至干，加入 $500~\mu \mathrm{L}$ 乙腈-水溶液(17.2.2)溶解残渣。涡旋 $30~s$ ，过 $0.45\mu \mathrm{m}$ 微孔滤膜后，收集于进样瓶中，待测。

注：由于不同厂商提供的免疫亲和柱操作程序可能不同，实际操作时，请参照厂商提供的操作说明和程序使用。

# 19.4 衍生

取 $100\mu \mathrm{L}$ 标准溶液或样品溶液于进样瓶中，加入 $100~\mu \mathrm{L}$ 衍生溶液(17.2.9)，涡旋混合 $30~\mathrm{s}$ ，在 $2\mathrm{min}$ 内进样分析。

# 19.6 试样溶液的测定

在12.4色谱条件下，将 $50.0\mu \mathrm{L}$ 伏马毒素系列混合标准工作溶液(17.4.3)按浓度从低到高依次注入高效液相色谱仪；待仪器条件稳定后，以目标物质的浓度为横坐标 $(x$ 轴)，目标物质的峰面积为纵坐标 $(y$ 轴)，对各个数据点进行最小二乘线性拟合，标准工作曲线：

$$
y = a x + b \quad \dots \tag {5}
$$

式中：

$y$ ——目标物质的峰面积比；

$a$ ——回归曲线的斜率；

$x$ ——目标物质的浓度；

$b$ ——回归曲线的截距。

标准工作溶液和样液中待测物的响应值均应在仪器线性响应范围内，如果样品含量超过标准曲线范围，需稀释后再测定。

# 20 分析结果的表述

$$
X = \frac {c _ {i} \times V \times f}{m} \tag {6}
$$

式中：

$X$ ——待测样品中 $\mathrm{FB}_1, \mathrm{FB}_2, \mathrm{FB}_3$ 的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c_{i}$ ——待测物进样液中 $\mathrm{FB}_{1} 、 \mathrm{FB}_{2} 、 \mathrm{FB}_{3}$ 的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试液稀释倍数；

$m$ ——样品的称样量，单位为克 $(\mathrm{g})$ 。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留两位有效数字。

样品中伏马毒素含量在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 免疫亲和柱柱容量及柱回收率验证方法

# B.1 柱容量验证方法

在 $30~\mathrm{mL}$ 的吐温-20/PBS溶液 $(0.1\%)$ 中加入 $15\mu \mathrm{g} \mathrm{FB}_{1}$ 标准储备溶液，充分混匀。分别取同一批次3根免疫亲和柱，每根柱的上样量为 $10~\mathrm{mL}(5\mu \mathrm{g} \mathrm{FB}_{1})$ 。经上样、淋洗、洗脱，收集洗脱液，用氮气吹至约 $1~\mathrm{mL}$ ，用乙腈-水溶液 $(20 + 80)$ 定容至 $10~\mathrm{mL}$ ，用液相色谱仪分离测定 $\mathrm{FB}_{1}$ 的含量。

结果判定：结果 $\mathrm{FB}_{1} \geqslant 4.5 \mu \mathrm{g}$ ，为柱容量满意。

# B.2 柱回收率验证方法

取 $2\mathrm{mL}$ 空白样品提取液，加入 $47~\mathrm{mL}$ 吐温-20/PBS溶液 $(0.1\%)$ ，加入 $15\mu \mathrm{g}~\mathrm{FB}_1$ 标准储备溶液，用吐温-20/PBS溶液 $(0.1\%)$ 定容至 $50~\mathrm{mL}$ ，充分混匀。分别取同一批次3根免疫亲和柱，每根柱的上样量为 $10~\mathrm{mL}(3\mu \mathrm{g}~\mathrm{FB}_1)$ 。经上样、淋洗、洗脱，收集洗脱液，用氮气吹干至 $1\mathrm{mL}$ ，用乙腈-水溶液 $(20+$ 80)定容至 $10~\mathrm{mL}$ ，用液相色谱仪分离测定 $\mathrm{FB}_1$ 的含量。

结果判定：结果 $\mathrm{FB}_{1} \geqslant 2.4 \mu \mathrm{g}$ ，则回收率 $\geqslant 80\%$ ，为柱回收率满意。

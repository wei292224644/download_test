# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T0655—2012《出口食品中敌麦丙残留量的检测方法》。与SN/T0655—2012相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“出口食品”改为“食品”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN 0655—1997、SN/T 0655—2012。

# 食品安全国家标准

# 食品中噻节因残留量的检测方法

# 1 范围

本标准规定了大米、白菜、柑橘、马铃薯、茶叶、板栗、鸡肉、鱼肉、猪肉、牛肉、猪肝、蜂蜜、鸡蛋、牛奶中噻节因残留量的气相色谱测定方法。

本标准适用于大米、白菜、柑橘、马铃薯、茶叶、板栗、鸡肉、鱼肉、猪肉、牛肉、猪肝、蜂蜜、鸡蛋、牛奶中噻节因残留量的测定和确证。其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

试样用甲醇一水混合溶剂振荡提取，提取液用正己烷和三氯甲烷进行液一液分配后，经串联弗罗里硅土和中性氧化铝固相萃取柱净化，用配有电子捕获检测器的气相色谱仪进行测定，气相色谱一质谱仪确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 甲醇 $\mathrm{(CH_4O, CAS}$ 号：67-56-1）：色谱纯。

4.1.2 正己烷 $\mathrm{(C_6H_{14}}$ ，CAS号： $110 - 54 - 3$ ：优级纯。

4.1.3 三氯甲烷 $\mathrm{(CHCl_3}$ ,CAS号：67-66-3)：色谱纯。

4.1.4 丙酮 $\mathrm{(C_3H_6O}$ ,CAS号： $67 - 64 - 1$ )：色谱纯。

4.1.5 氯化钠(NaCl，CAS号：7647-14-5)。

4.1.6 无水硫酸钠 $\mathrm{(Na_2SO_4}$ ,CAS号：7757-82-6)：使用前在 $650^{\circ}C$ 灼烧 $4\mathrm{h}$ 后，储存于干燥器中，冷却后备用。

# 4.2 溶液配制

4.2.1 甲醇—水溶液 $(9 + 1$ ，体积比)：准确移取 $90~\mathrm{mL}$ 甲醇和 $10~\mathrm{mL}$ 水混合均匀。

4.2.2 丙酮一三氯甲烷溶液 $(8 + 100$ ，体积比)：准确移取 $8\mathrm{mL}$ 丙酮和 $100~\mathrm{mL}$ 三氯甲烷混合均匀。

4.2.3 $10\%$ 氯化钠溶液: 准确称取 $10 \mathrm{~g}$ 氯化钠于烧杯中加入适量水溶解, 转移至 $100 \mathrm{~mL}$ 容量瓶, 定容至刻度, 摇匀。

# 4.3 标准品

噻节因标准品 $\mathrm{(C_6H_{10}O_4S_2}$ ，CAS号：55290-64-7）：纯度 $\geqslant 99\%$

# 4.4 标准溶液配制

4.4.1 标准储备液：准确称取 $50\mathrm{mg}$ 噬节因农药标准品（精确至 $0.1\mathrm{mg}$ ）于 $50~\mathrm{mL}$ 棕色容量瓶，用丙酮溶解并定容至刻度，摇匀，配制浓度为 $1000\mu \mathrm{g / mL}$ 标准储备液，于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 避光保存。  
4.4.2 标准中间溶液：准确吸取 $1.0\mathrm{mL}$ 标准储备溶液于 $100\mathrm{mL}$ 棕色容量瓶中，用丙酮溶解并定容至刻度，配制浓度为 $1000\mu \mathrm{g / mL}$ 标准中间溶液，于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 避光保存。  
4.4.3 标准工作溶液：根据检测需要移取一定体积的标准中间溶液逐级稀释成适当浓度的标准工作溶液。标准工作溶液需现配现用。

# 4.5 材料

4.5.1 弗罗里硅土固相萃取柱: $3 \mathrm{~mL} / 500 \mathrm{mg}$ , 或相当者。  
4.5.2 中性氧化铝固相萃取柱: $3 \mathrm{~mL} / 500 \mathrm{mg}$ , 或相当者。

# 5 仪器和设备

5.1 气相色谱仪：配有电子捕获检测器(ECD)。  
5.2 气相色谱—质谱仪：配有负化学电离源(NCI)。  
5.3 粉碎机。  
5.4 组织捣碎机。  
5.5 固相萃取装置。  
5.6 离心机： $6000\mathrm{r / min}$   
5.7 涡旋振荡器。  
5.8 旋转蒸发仪。  
5.9 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.10 鸡心瓶： $200~\mathrm{mL}$

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按GB2763的规定执行。在取样和制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 水果、蔬菜及坚果

取有代表性样品 $500\mathrm{g}$ , 切碎后不可用水洗涤, 用组织捣碎机将样品加工成浆状。混匀, 装入洁净的盛样容器内, 密封并标明标记。

# 6.2.2 茶叶及粮谷

取有代表性样品 $500\mathrm{g}$ , 用粉碎机粉碎并通过40目筛。混匀, 装入洁净的盛样容器内, 密封并标明标记。

# 6.2.3 肉及肉制品

取有代表性样品 $500\mathrm{g}$ ，将其切碎后，用组织捣碎机将样品加工成浆状，混匀，装入洁净的盛样容器内，密封并标明标记。

# 6.2.4 蜂蜜及牛奶

取有代表性样品 $500\mathrm{g}$ , 倒入洁净的搪瓷混样桶中, 充分搅拌均匀, 装入洁净的盛样容器内, 密封并标明标记。

# 6.3 试样保存

茶叶、牛奶、蜂蜜、坚果及粮谷类等试样于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存；水果、蔬菜、肉及肉制品类等试样于

$-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 7 测定步骤

# 7.1 提取

称取 $2\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样于 $50\mathrm{mL}$ 具塞离心试管中，加 $4\mathrm{mL}$ 水混匀，静置 $15\mathrm{min}$ ，加入 $5\mathrm{mL}$ 甲醇一水溶液置于涡旋振荡器上振荡提取 $5\mathrm{min}$ ，离心 $4\mathrm{min}$ （离心速度为 $6000\mathrm{r / min}$ ），吸取上层清液于另一 $50\mathrm{mL}$ 具塞离心试管中，残渣用 $5\mathrm{mL}$ 甲醇一水溶液重复提取1次，合并提取液。往提取液中加入 $2\mathrm{mL}$ 氯化钠溶液和 $5\mathrm{mL}$ 正己烷振荡提取 $2\mathrm{min}$ ，离心 $1\mathrm{min}$ （离心速度为 $5000\mathrm{r / min}$ ），吸取并弃去正己烷层，用 $5\mathrm{mL}$ 正己烷再提取两次并弃去正己烷层。加入 $5\mathrm{mL}$ 三氯甲烷振荡提取 $3\mathrm{min}$ ，离心 $1\mathrm{min}(5000\mathrm{r / min})$ ，吸取下层三氯甲烷层于 $200\mathrm{mL}$ 鸡心瓶中，用 $5\mathrm{mL}$ 三氯甲烷再提取两次，合并提取液，将提取液于 $40^{\circ}\mathrm{C}$ 水浴下旋转浓缩至约 $1\mathrm{mL}$ ，待净化。

# 7.2 净化

在弗罗里硅土固相萃取柱中加入约 $1\mathrm{cm}$ 高的无水硫酸钠，并串接在中性氧化铝固相萃取柱顶部，将该串联柱安装于固相萃取装置。加样前先用 $10\mathrm{mL}$ 丙酮一三氯甲烷溶液预淋洗萃取柱，弃去淋洗液，当液面到达无水硫酸钠顶部时，迅速将7.1得到的试样提取浓缩液转入串联固相萃取柱中，用 $2\mathrm{mL}$ 三氯甲烷洗涤鸡心瓶，重复洗涤3次，将全部洗涤液转入固相萃取柱中，然后用 $15\mathrm{mL}$ 丙酮一三氯甲烷溶液洗脱，收集所有流出液于另一 $200\mathrm{mL}$ 鸡心瓶中，于 $40^{\circ}\mathrm{C}$ 水浴中旋转蒸发至近干，用丙酮溶解定容至 $1.0\mathrm{mL}$ ，供GC-ECD测定和GC-MS确证。

# 7.3 测定

# 7.3.1 气相色谱参考条件

a）色谱柱：HP-5石英毛细管柱， $30\mathrm{m}\times 0.32\mathrm{mm}$ （内径），膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）升温程序：初始温度 $50^{\circ}\mathrm{C}$ ，以 $20^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升至 $160^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，再以 $5^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升至 $230^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ ；  
c）进样口温度： $250^{\circ}\mathrm{C}$   
d) 检测器温度: $300^{\circ} \mathrm{C}$ ;   
e) 载气：氮气，纯度 $\geqslant 99.999\%$ ，流量 $1.0~\mathrm{mL / min}$   
f) 尾吹气: $25 \mathrm{~mL} / \mathrm{min}$ ;  
g）进样方式：无分流进样， $1.0\mathrm{min}$ 后开阀；  
h）进样量： $1\mu \mathrm{L}$

# 7.3.2 气相色谱一质谱参考条件

a）色谱柱：HP-5MS石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度： $100^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，以 $5^{\circ}\mathrm{C / min}$ 的速率升至 $160^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，再以 $20^{\circ}\mathrm{C / min}$ 的速率升至 $240^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ ；  
c）进样口温度： $250^{\circ}\mathrm{C}$   
d） 色谱一质谱接口温度： $280^{\circ}C$   
e) 载气：氦气，纯度 $\geqslant 99.999\%$ ，1.0 mL/min;  
f）进样量： $1\mu \mathrm{L}$   
g）进样方式：不分流进样， $1\mathrm{min}$ 后开阀；  
h）电离方式：负化学离子源(NCI)；  
i）反应气：甲烷 $(40\%)$ ，流量为 $2.0~\mathrm{mL / min}$   
j）测定方式：选择离子监测方式(SIM)；  
k）溶剂延迟时间： $8\mathrm{min}$

# 7.3.3 色谱测定与确证

7.3.3.1 根据样液中噻节因农药的含量情况，选定浓度相近的噻节因标准工作溶液。标准工作溶液和样液中噻节因农药的响应值均应在仪器检测的线性范围内。对标准工作液和样液等体积交替进样测定，以保留时间定性，测量样液与标准工作液的峰面积比较定量。在 $7.3.1 \sim 7.3.2$ 规定的条件下，噻节因标准品色谱图参见图 A.1。  
7.3.3.2 样品测定时，如果检出色谱峰的保留时间与标准样品相一致，则对其进行质谱确证。在扣除背景后的样品质谱图中，所选择的离子均出现，而且所选择的离子丰度比与标准样品的离子丰度比相一致（允许偏差范围见表1），则可判断样品中存在这种农药或相关化学品。如果不能确证，应重新进样，以扫描方式(有足够灵敏度)或采用增加其他确证离子的方式或用其他灵敏度更高的分析仪器来确证。在上述气相色谱一质谱条件下，噻节因的监测离子 $(m / z)$ 及离子丰度比为 $128:210:211:212 = 32:100:11:12$ ，噻节因标准品总离子色谱图和负化学离子质谱图(NCI)参见图B.1、图C.1。

# 7.4 空白试验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中噻节因农药的残留含量，计算结果需扣除空白值。

$$
X = \frac {A \times C _ {s} \times V}{A _ {s} \times m}. \tag {1}
$$

式中：

$X$ ——试样中噻节因农药的残留含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

A ——样液中噻节因农药的峰面积；  
$C_{s}$ ——标准工作液中噻节因农药的浓度, 单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ 。  
V——最终样液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准工作液中噻节因农药的峰面积；

$m$ ——最终样液所代表的试样质量，单位为克 $(\mathrm{g})$ 。

计算结果应扣除空白值，保留2位有效数字；当结果大于 $1\mathrm{mg / kg}$ 时，保留3位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 D 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 E 的要求。

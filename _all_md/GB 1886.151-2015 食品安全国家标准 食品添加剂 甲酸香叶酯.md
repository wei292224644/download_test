# 食品安全国家标准

# 食品添加剂 甲酸香叶酯

# 1 范围

本标准适用于食品添加剂甲酸香叶酯。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

反式-3，7-二甲基-2，6-辛二烯-1-醇甲酸酯

# 2.2 分子式

$$
\mathrm {C} _ {1 1} \mathrm {H} _ {1 8} \mathrm {O} _ {2}
$$

# 2.3 结构式

![](images/2b8fcb621cfaab36a2d79d3b129d20936e7409df9e14cfa69a4941c05f9d4a08.jpg)

# 2.4 相对分子质量

182.26（按2007年国际相对原子质量）

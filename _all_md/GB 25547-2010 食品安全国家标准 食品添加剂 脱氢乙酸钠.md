# 1 范围

本标准适用于以脱氢乙酸和氢氧化钠（或碳酸钠或碳酸氢钠）反应制得的食品添加剂脱氢乙酸钠。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

3-（1-羟基亚甲基）-6-甲基-1，2-吡喃-2，4（3H）-二酮钠

\mathrm {C} _ {8} \mathrm {H} _ {7} \mathrm {N a O} _ {4} \cdot \mathrm {H} _ {2} \mathrm {O}

# 检验方法

# A.1 警示

试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和防护措施。

# A. 2 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682—2008中规定的三级水。

试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603之规定制备。

# A.3 鉴别试验

取约 $1.5 \mathrm{~g}$ 实验室样品，溶于 $10 \mathrm{~mL}$ 水中，加入 $5 \mathrm{~mL}$ 盐酸溶液（1+3），抽滤收集晶体，用 $10 \mathrm{~mL}$ 水冲洗，在 $75^{\circ} \mathrm{C} \sim 80^{\circ} \mathrm{C}$ 下干燥 $4 \mathrm{~h}$ 。此晶体应在 $109^{\circ} \mathrm{C} \sim 111^{\circ} \mathrm{C}$ 熔化。

# A. 4 脱氢乙酸钠的测定

# A.4.1 方法提要

样品以冰乙酸为溶剂，以 $p$ -萘酚苯为指示剂，用高氯酸标准滴定溶液滴定，根据消耗高氯酸标准滴定溶液的体积计算以 $\mathrm{C_8H_7NaO_4}$ 计的脱氢乙酸钠含量。

# A.4.2 试剂和材料

A. 4.2.1 冰乙酸。

A. 4.2.2 高氯酸标准滴定溶液: $c\left(\mathrm{HClO}_{4}\right)=0.1 \mathrm{~mol} / \mathrm{L}$ 。

A.4.2.3 $p$ -萘酚苯指示液： $10\mathrm{g / L}$ ，以冰乙酸为溶剂。

# A.4.3 分析步骤

称取约 $0.5 \mathrm{~g}$ 实验室样品，精确至 $0.0002 \mathrm{~g}$ ，置于 $125 \mathrm{~mL}$ 锥形瓶中。量取 $25 \mathrm{~mL}$ 冰乙酸，加入1滴 $p$ -萘酚苯指示液，用高氯酸标准滴定溶液滴定至蓝色，将此已中和至蓝色的冰乙酸倒入称有样品的锥形瓶中，样品完全溶解后用高氯酸标准滴定溶液滴定至原来的蓝色即为终点。

# A.4.4 结果计算

脱氢乙酸钠（以 $\mathrm{C}_{8} \mathrm{H}_{7} \mathrm{NaO}_{4}$ 计）的质量分数 $w_{1}$ ，数值以%表示，按式(A.1)计算：

$$
w _ {1} = \frac {(V / 1000) c M}{m \left(1 - w _ {2}\right)} \times 100 \% \tag{A.1}
$$

式中：

$V$ ——试料消耗高氯酸标准滴定溶液(A.4.2.2)体积的数值,单位为毫升 $(\mathrm{mL})$

$c$ ——高氯酸标准滴定溶液浓度的准确数值,单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——试料质量的数值,单位为克(g);

$M$ ——脱氢乙酸钠 $\left(\mathrm{C}_{8} \mathrm{H}_{7} \mathrm{NaO}_{4}\right)$ 的摩尔质量的数值，单位为克每摩尔（ $\mathrm{g/mol}$ ）（ $M=190.1$ ）；

$w_{2}$ ——A.6测定的水分的质量分数， $\%$ 。

取两次平行测定结果的算术平均值为报告结果。两次平行测定结果的绝对差值不大于 $0.2\%$

# A.5 游离碱试验

# A.5.1 试剂和材料

A.5.1.1 硫酸标准滴定溶液： $c(1 / 2\mathrm{H}_2\mathrm{SO}_4) = 0.100\mathrm{mol} / \mathrm{L}$   
A.5.1.2 酚酞指示液： $10\mathrm{g / L}$

# A.5.2 分析步骤

称取 $1.0\mathrm{g}$ 实验室样品，精确至 $0.01\mathrm{g}$ ，溶于约 $20\mathrm{mL}$ 无二氧化碳的水，加2滴酚酞指示液，溶液显红色，再加 $0.4\mathrm{mL} \pm 0.02\mathrm{mL}$ 硫酸标准滴定溶液，摇匀，红色应消失。

# A.6 水分的测定

按GB/T6283直接滴定法进行。

取两次平行测定结果的算术平均值为报告结果。两次平行测定结果的绝对差值不大于 $0.2\%$

# A.7 氯化物的测定

# A.7.1 试剂和材料

A.7.1.1 硝酸溶液： $1 + 10$   
A.7.1.2 硝酸银溶液： $17\mathrm{g / L}$   
A.7.1.3 氯化物（Cl）标准溶液： $0.1\mathrm{mg / mL}$

# A.7.2 分析步骤

称取 $1\mathrm{g}$ 实验室样品,精确至 $0.01\mathrm{g}$ ,加水约 $30\mathrm{mL}$ 溶解,边搅拌边加 $9.5\mathrm{mL}$ 硝酸溶液,过滤,水洗,合并洗液和滤液置于 $100\mathrm{mL}$ 比色管中,加水至 $50~\mathrm{mL}$ 为试验溶液。另取一只比色管,加入 $1\mathrm{mL}\pm 0.02\mathrm{mL}$ 氯化物（Cl）标准溶液,加 $30~\mathrm{mL}$ 水,加 $6\mathrm{mL}$ 硝酸溶液,加水配至 $50\mathrm{mL}$ 为标准比浊溶液。分别于两比色管中加入 $1\mathrm{mL}$ 硝酸银溶液,充分混匀,避开日光直射,放置 $5\mathrm{min}$ 。在黑色背景下侧面或轴向进行观察,试验溶液的浊度不得大于标准比浊溶液的浊度。

# A.8 砷的测定

按GB/T5009.76砷斑法进行。按湿法消解处理样品。测定时量取 $10.0\mathrm{mL}$ 试样溶液（相当于 $1.0\mathrm{g}$ 实验室样品）。限量标准液的配制：用移液管移取 $3.0\mathrm{mL}$ 砷（As）标准溶液（相当于 $3.0\mu \mathrm{g}$ As），与试样同时同样处理。

# A.9 铅的测定

# A.9.1 比色法（仲裁法）

按GB/T5009.75进行。临用前，将 $1\mathrm{mg / mL}$ 的铅标准溶液稀释成 $5\mu \mathrm{g / mL}$ 的铅标准溶液。测定时吸取 $25\mathrm{mL}\pm 0.02\mathrm{mL}$ 试样溶液（相当于 $2.5\mathrm{g}$ 实验室样品）及 $1\mathrm{mL}\pm 0.02\mathrm{mL}$ 铅标准溶液（相当于 $5\mu \mathrm{g}$ Pb），分别置于 $125\mathrm{mL}$ 分液漏斗中，铅标准溶液中加 $1\%$ 硝酸溶液至 $25\mathrm{mL}$ 。

# A.9.2 原子吸收光谱法

按GB5009.12进行。试样处理按GB/T5009.75进行。采用石墨炉原子吸收光谱法时，可视样品情况将试样溶液进行适当的稀释。

# 检验方法

# A.1 警告

本标准试验方法中使用的部分试剂具有毒性或腐蚀性，使用时需小心谨慎并按相关规定操作。若溅到皮肤上应立即用水冲洗，严重者应立即治疗。在使用挥发性酸时，应在通风橱中进行。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水及以上试验用水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 冰乙酸。

A.3.1.2 吡啶。

A.3.1.3 醋酸酐。

A.3.1.4 硫化铵溶液：含量 $\geq 8\%$ 。本试液的灼烧残渣量不得大于 $0.05\%$ ，遇硫酸镁试液或氯化钙试液不得发生浑浊。如本试液出现明显的硫磺沉淀，则该溶液不能使用。本溶液应盛放于暗棕色小瓶中，装满，存放在阴冷、暗处。

A.3.1.5 盐酸溶液： $1\mathrm{mol} / \mathrm{L}$ 。量取 $8.4~\mathrm{mL}$ 盐酸 $(36.5\% \sim 38.0\%)$ ，溶于水并定容至 $100~\mathrm{mL}$ 。

# A.3.2 仪器和设备

分析天平：感量为 $0.1\mathrm{g}$ 和 $1\mathrm{mg}$

# A.3.3 鉴别方法

A.3.3.1 锰离子的鉴别。

A.3.3.1.1 方法提要

锰盐溶液遇硫化铵试液即产生浅橙色沉淀，该沉淀可溶于冰乙酸。

A.3.3.1.2 试样溶液

称取约 $1\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）试样，溶解于 $20~\mathrm{mL}~1~\mathrm{mol / L}$ 的盐酸溶液中。

A.3.3.1.3 操作步骤

在 $20~\mathrm{mL}$ 试样溶液中逐滴加入硫化铵溶液，生成浅橙色沉淀。静置 $10\mathrm{min}$ 后，再逐滴加入冰乙酸，该沉淀可溶解。

A.3.3.2 柠檬酸盐的鉴别

A.3.3.2.1 方法提要

柠檬酸盐在吡啶-醋酸酐中会产生胭脂红色。

# A.3.3.2.2 操作方法

在 $15\mathrm{mL}$ 吡啶中加入 $5\mathrm{mg}\sim 10\mathrm{mg}$ (精确至 $1\mathrm{mg}$ ) 试样, 再加入 $5\mathrm{mL}$ 醋酸酐, 轻轻摇动, 产生胭脂红色。

# A.4 柠檬酸锰 $\left[\mathbf{Mn}_{3}\left(\mathbf{C}_{6} \mathbf{H}_{5} \mathbf{O}_{7}\right)_{2}\right]$ 含量的测定

# A.4.1 方法提要

在 $\mathrm{pH}$ 为10的氨-氯化铵缓冲溶液中，以铬黑T为指示剂，用乙二胺四乙酸二钠标准溶液滴定试样溶液，根据乙二胺四乙酸二钠标准滴定溶液的用量，计算以 $\mathrm{Mn}_3(\mathrm{C}_6\mathrm{H}_5\mathrm{O}_7)_2$ 计的柠檬酸锰的含量。

# A.4.2 试剂和材料

# A.4.2.1 盐酸。

# A.4.2.2 氢氧化钠溶液： $1\mathrm{mol} / \mathrm{L}$

A.4.2.3 氨-氯化铵缓冲溶液: $\mathrm{pH} \approx 10$ 。称取氯化铵 $67.5 \mathrm{~g}$ , 溶于适量水, 加入 $28 \%$ 氨水 $570 \mathrm{~mL}$ , 用水稀释至 $1000 \mathrm{~mL}$ 。

A.4.2.4 乙二胺四乙酸二钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{EDTA}) = 0.05\mathrm{mol / L}$

A.4.2.5 铬黑T指示剂：将 $200\mathrm{mg}$ 铬黑T和 $2\mathrm{g}$ 盐酸羟胺 $\mathrm{(NH_2OH\cdot HCl)}$ 溶解在适量甲醇中，用甲醇定容至 $50~\mathrm{mL}$ ，然后过滤。将溶液保存在避光容器中，两周内使用。

# A.4.3 仪器和设备

天平：感量为 $1\mathrm{mg}$ 。

# A.4.4 分析步骤

称取约 $0.35\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )，置于 $250~\mathrm{mL}$ 烧杯中，加入 $100~\mathrm{mL}$ 水和 $1\mathrm{mL}$ 盐酸，同时不断搅拌，在加热板上 $75^{\circ}\mathrm{C}\sim 80^{\circ}\mathrm{C}$ 加热，使试样完全溶解。加入 $25~\mathrm{mL}0.05~\mathrm{mol / L}$ 乙二胺四乙酸二钠溶液，如果需要，用 $1\mathrm{mol / LNaOH}$ 调节 $\mathrm{pH}$ 为 $10.0\pm 0.2$ 。加 $10~\mathrm{mL}$ 氨-氯化铵缓冲溶液和约8滴铬黑T指示剂。用 $0.05\mathrm{mol / L}$ 乙二胺四乙酸二钠滴定液继续滴定直到纯蓝色即为终点，并且蓝色最少保持 $3\mathrm{min}$ 。同时进行空白试验。

# A.4.5 结果计算

柠檬酸锰 $\left[\mathrm{Mn}_{3}\left(\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{O}_{7}\right)_{2}\right]$ 含量（以干基计）的质量分数 $\omega_{1}$ 按式(A.1)计算。

$$
w _ {1} = \frac {\left(V - V _ {0}\right) \times c \times M}{m _ {1} \times \left(1 - w _ {2}\right) \times 3 \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V$ ——试样溶液消耗EDTA标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白溶液消耗EDTA标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$

c ——EDTA标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——柠檬酸锰的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(\mathrm{Mn}_3(\mathrm{C}_6\mathrm{H}_5\mathrm{O}_7)_2) = 543.02]$

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$w_{2}$ ——试样干燥失重的质量分数， $\%$ ；

3 ——换算因子；

1000 ——换算因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 干燥失重

# A.5.1 仪器和设备

A.5.1.1 天平：感量为 $0.1\mathrm{mg}$   
A.5.1.2 真空干燥箱。

# A.5.2 分析步骤

称取试样 $1.0\mathrm{g}\sim 2.0\mathrm{g}$ （精确至 $0.0001\mathrm{g})$ ，参照GB5009.3第二法减压干燥法进行操作。减压干燥真空干燥箱的压力和温度为：压力不超过 $20~\mathrm{mmHg}$ ，干燥温度为 $135^{\circ}C$ ，时间 $16\mathrm{h}$ 。

试样的干燥失重，按式(A.2)进行计算。

$$
w _ {2} = \frac {m _ {2} - m _ {3}}{m _ {2} - m _ {4}} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$m_{2}$ —称量瓶和试样的质量，单位为克 $(\mathrm{g})$

$m_{3}$ —称量瓶和试样干燥后的质量，单位为克 $(\mathrm{g})$

$m_{4}$ —称量瓶的质量，单位为克 $(\mathrm{g})$ 。

干燥失重 $\geqslant 1\mathrm{g} / 100\mathrm{g}$ 时，计算结果保留三位有效数字；干燥失重 $< 1\mathrm{g} / 100\mathrm{g}$ 时，计算结果保留两位有效数字。

# A.8 硫酸盐的测定

# A.8.1 方法提要

在酸性介质中，试样中的硫酸根离子与钡离子生成硫酸钡沉淀，与同法处理的硫酸根标准溶液比较，作限量试验。

# A.8.2 试剂和材料

A.8.2.1 硫酸钾。  
A.8.2.2 氯化钡。  
A.8.2.3 盐酸。  
A.8.2.4 过氧化氢。  
A.8.2.5 乙醇 $(95\%)$ 。  
A.8.2.6 盐酸溶液： $1 + 1$   
A.8.2.7 硫酸钾乙醇溶液 $(0.2\mathrm{g} / \mathrm{L})$ ：称取 $0.2\mathrm{g}$ 硫酸钾，溶于 $700~\mathrm{mL}$ 水中，用乙醇 $(95\%)$ 稀释至 $1000~\mathrm{mL}$ 。  
A.8.2.8 硫酸盐 $\left(\mathrm{SO}_{4}\right)$ 标准溶液: $0.1 \mathrm{mg} / \mathrm{mL}$ 。称取 $0.181 \mathrm{~g}$ 硫酸钾, 加水使之溶解, 再移入 $1000 \mathrm{~mL}$ 容量瓶中, 稀释至刻度。  
A.8.2.9 氯化钡溶液： $250~\mathrm{g / L}$

# A.8.3 分析步骤

A.8.3.1 试样溶液：称取 $2.0\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），置于 $100\mathrm{mL}$ 烧杯中，加 $10\mathrm{mL}$ 水，滴加盐酸至试样溶解，加1滴过氧化氢使暗色褪去，加热煮沸 $2\mathrm{min}$ ，冷却至室温，转移至 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。用移液管吸取试液 $25.0\mathrm{mL}$ 置于 $25\mathrm{mL}$ 比色管中，加 $0.5\mathrm{mL}$ 盐酸溶液。  
A.8.3.2 晶种液：按 $0.25\mathrm{mL}$ 硫酸钾乙醇溶液与 $1\mathrm{mL}$ 氯化钡溶液比例混合，准确放置 $1\mathrm{min}$ 。  
A.8.3.3 标准溶液：准确取 $1.0\mathrm{mL}$ 硫酸盐标准溶液置于 $25~\mathrm{mL}$ 比色管中，加水至刻度，再加 $0.5\mathrm{mL}$ 盐酸溶液。  
A.8.3.4 分别取 $1.25 \mathrm{~mL}$ 晶种液加入到试样溶液和标准溶液比色管中，摇匀，放置 $5 \mathrm{~min}$ ，在黑色背景下，轴向观察，所呈浊度与标准比浊溶液比较。

# A.8.4 结果判定

试样溶液所呈浊度不得深于标准比浊溶液，即试样中的硫酸盐不大于 $0.02\%$ 。

# 1 范围

本文件规定了动物性食品中127种药物残留筛查的制样和液相色谱-高分辨质谱检测方法。

本文件适用于畜禽等可食组织、禽蛋、羊奶和牛奶中127种药物残留的筛查。

# 3 术语与定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中目标物用 $0.1\mathrm{mol} / \mathrm{L}$ Mcllvaine- $\mathrm{Na_2EDTA}$ 缓冲溶液和乙腈提取, 氯化钠和无水硫酸钠盐析, 固相萃取柱净化后, 液相色谱-高分辨质谱检测分析。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水

# 5.1 试剂

5.1.1 甲醇 $(\mathrm{CH}_3\mathrm{OH})_2$ 色谱纯

5.1.2 乙腈 $\mathrm{(CH_3CN)}$ 色谱纯

5.1.3 甲酸（CHOOH）：色谱纯。

5.1.4 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ：色谱纯。

5.1.5 二甲基亚砜 $\mathrm{(C_2H_6OS)}$

5.1.6 丙酮 $(\mathrm{C_3H_6O})$ 。

5.1.7 磷酸氢二钠 $\mathrm{(Na_2HPO_4\cdot 12H_2O)}$

5.1.8 乙二胺四乙酸二钠 $(\mathrm{C_{10}H_{14}N_2O_8Na_2}\cdot 2\mathrm{H_2O})$

5.1.9 枸橼酸 $(\mathrm{C_6H_8O_7}\cdot \mathrm{H_2O})$ 。

5.1.10 无水硫酸钠 $\mathrm{(Na_2SO_4)}$

5.1.11 氯化钠 $(\mathrm{NaCl})$

# 5.2 溶液配制

5.2.1 $5\%$ 氨化甲醇：取氨水 $5\mathrm{mL}$ ，用甲醇稀释至 $100~\mathrm{mL}$ ，混匀，即得。

5.2.2 $20\%$ 甲醇溶液：取甲醇 $20~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀，即得。

5.2.3 $0.1\mathrm{mol} / \mathrm{L}$ Mcllvaine- $\mathrm{Na_2}$ EDTA缓冲溶液：取磷酸氢二钠 $44.08\mathrm{g}$ 、乙二胺四乙酸二钠 $37.2\mathrm{g}$ 、枸橼酸 $8.08\mathrm{g}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。

5.2.4 $0.1\%$ 甲酸溶液：取甲酸 $0.5 \mathrm{~mL}$ ，用水稀释至 $500 \mathrm{~mL}$ ，混匀，即得。  
5.2.5 $0.1\%$ 甲酸乙腈溶液：取甲酸 $0.5 \mathrm{~mL}$ ，用乙腈稀释至 $500 \mathrm{~mL}$ ，混匀，即得。  
5.2.6 $0.03\%$ 氨水溶液：取氨水 $0.15 \mathrm{~mL}$ ，用水稀释至 $500 \mathrm{~mL}$ ，混匀，即得。  
5.2.7 $0.03\%$ 氨化乙腈溶液：取氨水 $0.15 \mathrm{~mL}$ ，用乙腈稀释至 $500 \mathrm{~mL}$ ，混匀，即得。

# 5.3 标准品

16类127种药物，信息详见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液：取127种药物标准品各适量（相当于各有效成分 $10\mathrm{mg}$ ），精密称定，分别于 $10\mathrm{mL}$ 棕色容量瓶中，加入附录A中表A.1每种药物对应的溶剂适量使溶解并用甲醇稀释定容至刻度，分别配成浓度均为 $1\mathrm{mg/mL}$ 各药物的标准储备液， $-18^{\circ}\mathrm{C}$ 以下避光冷冻保存，保存期为6个月。其中，大环内酯类药物保存期为3个月，解热镇痛类药物、 $\beta$ -内酰胺类药物现用现配。  
5.4.2 混合标准中间液：参照表A.1中药物的类别，移取每个类别中全部药物的标准储备液各 $100\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 棕色容量瓶中，并用甲醇稀释至刻度，分别配成浓度为 $10~\mu \mathrm{g / mL}$ 的磺胺类、氟喹诺酮类、硝基咪唑类、抗病毒类、林可胺类、酰胺醇类、β-内酰胺类、喹噁啉类、解热镇痛类、精神类、抗组胺类、大环内酯类、驱虫类、农药类、截短侧耳类和其他类共16类混合标准中间液。现用现配。  
5.4.3 混合标准工作液：分别准确移取适量的混合标准中间液，用 $20\%$ 甲醇溶液稀释成系列标准工作溶液，现用现配。  
5.5 材料  
5.5.1 HyperSep Retain PEP固相萃取柱： $500\mathrm{mg / 6mL}$ ，填料为改性多孔聚苯乙烯DVB，或相当者。  
5.5.2 有机微孔尼龙滤膜: $0.22 \mu \mathrm{m}$ 。   
5.5.3 离心管： $50~\mathrm{mL}$

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

取适量新鲜或解冻的空白或供试奶样品，混合均匀。

取适量新鲜或冷藏的空白或供试鸡蛋，去壳后混合均匀。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；

c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

# 8.1.1 肌肉、禽蛋

称取试料 $(5\pm 0.05)\mathrm{g}$ 于 $50~\mathrm{mL}$ 离心管中，加入 $3.0\mathrm{mL}0.1\mathrm{mol / L}$ Mcllvaine- $\mathrm{Na_2EDTA}$ 缓冲溶液，涡旋 $10s$ ，加入 $10.0\mathrm{mL}$ 乙腈，涡旋混匀后，振荡 $10\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，转移上清液于另一 $50~\mathrm{mL}$ 离心管中，残渣用 $3.0\mathrm{mL}0.1\mathrm{mol / L}$ Mcllvaine- $\mathrm{Na_2EDTA}$ 缓冲溶液和 $10.0\mathrm{mL}$ 乙腈重复提取，合并2次上清液。上清液加入 $6\mathrm{g}$ 无水硫酸钠和 $1.5\mathrm{g}$ 氯化钠，涡旋 $30s$ ，静置 $10\mathrm{min}$ 后 $7000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，吸出上层液(乙腈层)备用。

# 8.1.2 牛奶、羊奶

称取试料 $(5\pm 0.05)\mathrm{g}$ 于 $50~\mathrm{mL}$ 离心管中，加入 $20.0~\mathrm{mL}$ 乙腈、涡旋混匀后，振荡 $10\mathrm{min}$ ， $7000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，转移上清液于另一 $50~\mathrm{mL}$ 离心管中。上清液加入 $4\mathrm{g}$ 无水硫酸钠和 $1\mathrm{g}$ 氯化钠，涡旋 $30~\mathrm{s}$ ，静置 $10\mathrm{min}$ 后 $7000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，吸出上层液（乙腈层）备用。

# 8.2 净化

HyperSe Retain PEP固相萃取柱依次用 $4\mathrm{mL}$ 甲醇和 $4\mathrm{mL}$ 水活化平衡后，取 $2\mathrm{mL}$ 备用液润洗小柱，弃去流出液。再移取适量备用液通过小柱，收集流出液并准确移取 $4\mathrm{mL}$ 于 $40^{\circ}\mathrm{C}$ 下氮吹干，加入 $200\mu \mathrm{L}$ 甲醇后涡旋混匀 $10\mathrm{s}$ ，再加入 $800\mu \mathrm{L}$ 纯水涡旋混匀 $30\mathrm{s}$ ，超声 $2\mathrm{mm}$ 后，过 $0.22\mu \mathrm{m}$ 尼龙微孔滤膜，上机测定。

# 8.3 测定

# 8.3.1 色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 样，柱长 $150\mathrm{mm}$ ，内径 $3.0\mathrm{mm}$ ，粒径 $1.8\mu \mathrm{m}$ ，或性能相当者；  
b) 柱温: $30^{\circ} \mathrm{C}$ ;   
c) 进样量: $10 \mu \mathrm{L}$ ;   
d) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ ;   
e）流动相：A为正离子模式 $0.1\%$ 甲酸水，负离子模式 $0.03\%$ 氨水；B为正离子模式 $0.1\%$ 甲酸乙腈，负离子模式 $0.03\%$ 氨水乙腈；正离子模式梯度洗脱程序见表1，负离子模式梯度洗脱程序见表2。

# 8.4 高分辨质谱数据库的构建

质谱库中包含各药物的保留时间、精确相对分子质量、二级碎片离子、同位素丰度比和二级质谱图，127种药物质谱库分析参数见附录A。

# 8.5 筛查

在 $5\mathrm{ppm}$ 质量窗口范围内进行提取：

a）目标物信号响应 $S / N > 3$ ；当 $S / N$ 不存在时，那么至少5个连续的扫描点才能确定为一个信号；  
b）目标物保留时间与质谱库中的保留时间参数偏差 $\leqslant 0.2\mathrm{min}$ 或者士 $2.5\%$ 以内(不超过 $0.5\mathrm{min}$ )；  
c）目标物与高分辨质谱数据库相比，可以匹配上一级母离子和一个二级碎片离子，且一级母离子质量精度偏差 $< 5 \mathrm{ppm}$ ，二级碎片离子质量精度偏差 $< 10 \mathrm{ppm}$ 。

同时满足以上3项条件，则判定样品为疑似阳性样品。

注：质谱库保留时间需使用标准品不定期进行校正。

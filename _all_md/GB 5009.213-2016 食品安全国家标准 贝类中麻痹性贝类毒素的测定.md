# 前言

本标准代替GB/T5009.213—2008《贝类中麻痹性贝类毒素的测定》、GB/T23215—2008《贝类中多种麻痹性贝类毒素含量的测定液相色谱-荧光检测法》、SC/T3023—2004《麻痹性贝类毒素的测定生物法》、SN0352—1995《出口贝类麻痹性贝类毒素检验方法》、SN/T1735—2006《进出口贝类产品中麻痹性贝类毒素检验方法高效液相色谱法》和SN/T1773—2006《进出口贝类中麻痹性贝类毒素检验方法酶联免疫吸附试验法》。

本标准与GB/T5009.213—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 贝类中麻痹性贝类毒素的测定”；  
——增加了酶联免疫吸附法；  
——增加了液相色谱-串联质谱法。

# 食品安全国家标准

# 贝类中麻痹性贝类毒素的测定

# 1 范围

本标准规定了贝类中麻痹性贝类毒素测定的小鼠生物法，酶联免疫吸附方法，液相色谱法和液相色谱-串联质谱法。

本标准适用于牡蛎、扇贝等贝类及其制品中麻痹性贝类毒素的检测。

# 小鼠生物法

# 2 原理

用盐酸提取贝类中麻痹性贝类毒素(PSP)。记录小鼠腹腔注射提取液后的死亡时间，根据麻痹性贝类毒素致小鼠死亡时间与鼠单位关系的对照表查出鼠单位(MU)，并按小鼠体重对鼠单位进行校正得到校正鼠单位(CMU)，计算得到每 $100\mathrm{g}$ 样品中PSP的鼠单位。以石房蛤毒素作为标准，将鼠单位换算成毒素的微克数，计算每 $100\mathrm{g}$ 贝肉中的PSP微克数。测定结果代表存在于贝肉内各种化学结构的PSP毒素总量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 氢氧化钠(NaOH)。  
3.1.2 盐酸（HCl）。  
3.1.3 无水乙醇 $\mathrm{(CH_3CH_2OH)}$

# 3.2 试剂配制

3.2.1 氢氧化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：将 $4.0\mathrm{g}$ 氢氧化钠溶于 $1\mathrm{L}$ 水中。  
3.2.2 盐酸溶液 $(0.18\mathrm{mol} / \mathrm{L})$ ：将 $15.5\mathrm{mL}$ 盐酸用蒸馏水稀释至 $1\mathrm{L}$ 。  
3.2.3 盐酸溶液 $(5\mathrm{mol} / \mathrm{L})$ ：将 $45~\mathrm{mL}$ 盐酸用水稀释至 $100~\mathrm{mL}$ 。  
3.2.4 酸性乙醇溶液：量取无水乙醇 $200~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀，用盐酸溶液 $(5\mathrm{mol} / \mathrm{L})$ 调节 $\mathrm{pH}$ 至 $2.0\sim 4.0$ 。

# 3.3 标准品

石房蛤毒素标准品 $(\mathrm{STX},\mathrm{C}_{10}\mathrm{H}_{17}\mathrm{N}_7\mathrm{O}_4\cdot 2\mathrm{HCl},\mathrm{CAS}$ 号35554-08-6）：纯度 $\geqslant 98.0\%$ 。

# 3.4 标准溶液的配制

3.4.1 石房蛤毒素标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：准确称取适量STX标准品，用酸性乙醇溶液溶解并定容，

配制成STX的质量浓度为 $100\mu \mathrm{g / mL}$ 的标准储备液。

3.4.2 石房蛤毒素标准工作液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取 $1\mathrm{mL}$ 石房蛤毒素标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ ，用水稀释，用盐酸溶液(5mol/L)调节 $\mathrm{pH}$ 至 $2.0\sim 4.0$ ，用水定容至 $100~\mathrm{mL}$ 。该标准工作液于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 下可保存 $30\mathrm{d}$ 。

# 3.5 材料

3.5.1 小鼠：体重为 $19\mathrm{g}\sim 21\mathrm{g}$ 的健康ICR系雄性小鼠。  
3.5.2 pH计或 $\mathrm{pH}$ 试纸。

# 4 仪器和设备

4.1 均质器。  
4.2 分析天平：感量为 $0.1\mathrm{g}$ 和 $0.0001\mathrm{g}$   
4.3 离心机：转速 $\geq 2000\mathrm{r / min}$

# 5 分析步骤

注：为避免毒素的危害，应戴手套进行操作。移液管及移液器吸头等用过的器材、废弃的提取液等应在次氯酸钠溶液 $(5\%)$ 中浸泡 $1\textrm{h}$ 以上以使毒素分解。对于动物实验过程中产生的污水、废弃物及动物尸体处理，应参照GB14925执行。

# 5.1 样品采集

从 $2\mathrm{kg}$ 以上样品中采取足够贝类个数，去壳使贝肉达 $200\mathrm{g}$ 以上。不能及时送检的新鲜贝类，开壳将贝肉分离，将沥水后的 $200\mathrm{g}$ 贝肉放入 $200\mathrm{mL}$ 盐酸溶液 $(0.18\mathrm{mol} / \mathrm{L})$ 中，置 $4^{\circ}\mathrm{C}$ 保存，备检。

# 5.2 试样制备

# 5.2.1 牡蛎、蛤及贻贝

用清水将贝类样品外表彻底洗净，切断闭壳肌，开壳，用蒸馏水淋洗内部去除泥沙及其他异物。将闭壳肌和连接在胶合部的组织分开，仔细取出贝肉，切勿割破贝体。严禁加热或用麻醉剂开壳。收集约 $200\mathrm{g}$ 贝肉分散置于筛子中沥水 $5\mathrm{min}$ (不要使肉堆积)，捡出碎壳等杂物，将贝肉均质备用。

# 5.2.2 扇贝

取可食部分用作检测，按5.2.1均质后备用。

# 5.2.3 冷冻贝类

在室温下，使冷冻的样品(带壳或脱壳的)自然融化，按5.2.1开壳、淋洗、取肉、均质、备用。

# 5.2.4 贝类罐头

将罐内所有内容物(肉及液体)充分均质。如果是大罐，将贝肉沥水并收集沥下的液体分别称重并存放固形物和汤汁，将固形物和汤汁按原罐装比例混合，均质后备用。

# 5.2.5 用酸保存的贝肉

沥去酸液，分别存放贝肉及酸液，备用。

# 5.2.6 贝肉干制品

干制品可于等体积盐酸溶液 $(0.18\mathrm{mol} / \mathrm{L})$ 中浸泡 $24\mathrm{h}\sim 48\mathrm{h}(4^{\circ}\mathrm{C}$ 冷藏)，沥去酸液，分别存放贝肉及酸液备用。

# 5.3 PSP标准品对照试验

# 5.3.1 PSP标准工作液的配制

用 $10\mathrm{mL}, 15\mathrm{mL}, 20\mathrm{mL}, 25\mathrm{mL}$ 和 $30\mathrm{mL}$ 水分别稀释 $10\mathrm{mL}$ 石房蛤毒素标准工作液，配制成系列浓度的标准稀释液。

# 5.3.2 中位数死亡时间的PSP标准工作液选择

取按5.3.1配制的系列浓度的标准稀释液各 $1\mathrm{mL}$ ，腹腔注射小鼠数只，选择中位数死亡时间为 $5\mathrm{min}\sim 7\mathrm{min}$ 的浓度剂量。如某浓度稀释液已达到要求，还需以 $1\mathrm{mL}$ 水的增减量进行补充稀释试验。

每只小鼠试验前称重，以10只小鼠为一组，用中位数死亡时间在 $5\mathrm{min}\sim 7\mathrm{min}$ 范围内的两个浓度的标准稀释液注射小鼠，测定并记录每只小鼠腹腔注射完毕至停止呼吸的所需死亡时间。

# 5.3.3 毒素转换系数(CF)的计算

# 5.3.3.1 小鼠中位数死亡时间的选择

计算5.3.2中所选择浓度的标准稀释液受试组的中位数死亡时间。弃去中位数死亡时间小于 $5\mathrm{min}$ 或大于 $7\mathrm{min}$ 的受试组；选择中位数死亡时间在 $5\mathrm{min}\sim 7\mathrm{min}$ 的受试组，该受试组中可允许有个别小鼠的死亡时间可小于 $5\mathrm{min}$ 或大于 $7\mathrm{min}$ 。

# 5.3.3.2 校正鼠单位（CMU）的计算

对于所选定的中位数死亡时间为 $5\mathrm{min}\sim 7\mathrm{min}$ 的受试组，根据附录A查得该组中每只小鼠死亡时间所对应的鼠单位(MU)，再根据附录B查得组中每只小鼠体重所对应的体重校正系数，同一只小鼠的体重校正系数与鼠单位相乘得到该只受试小鼠的校正鼠单位(CMU)。

# 5.3.3.3 毒素转换系数(CF)的计算

单只小鼠的毒素转换系数(CF)按式(1)计算：

$$
\mathrm {C F} = \frac {c}{\mathrm {C M U}} \tag {1}
$$

式中：

CF ——毒素转换系数，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$c$ ——每毫升 STX 标准稀释液中的毒素含量, 单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

CMU——校正鼠单位。

得到单只小鼠的毒素转换系数(CF)后，再计算每组10只小鼠的平均CF值，即为组内毒素转换系数 $(\mathrm{CF}_1)$ 。

# 5.3.3.4 组间毒素转换系数 $(\mathbf{CF}_2)$ 的计算

取不同受试组的组内毒素转换系数的平均值，即为组间毒素转换系数 $(\mathrm{CF}_2)$ 。以组间毒素转换系数进行试样毒力的计算。

# 5.3.3.5 CF值定期检查

如PSP检测间隔时间较长，每次测定时要用适当的标准稀释液注射5只小鼠，重新测定CF值。如果一周有几次检测，则用中位数死亡时间 $5\mathrm{min}\sim 7\mathrm{min}$ 的标准稀释液每周检查一次，测得的CF值应在原测定CF值的 $\pm 20\%$ 范围内。若结果不符，则用同样的标准稀释液另外注射5只小鼠，综合之前注射的5只小鼠的结果，计算出CF值。并用同样的标准稀释液注射第二组10只小鼠，将第二组求出的CF值和第一组的CF值进行平均，即为一个新的CF值。

重复检查的CF值通常在原结果的 $\pm 20\%$ 之内，若经常发现有较大偏差，在进行常规检测前应调查该方法中是否存在可变影响因素。

# 5.4 试样提取

取 $100\mathrm{g}$ 试样于烧杯中，加盐酸溶液 $(0.18\mathrm{mol} / \mathrm{L})100\mathrm{mL}$ 充分搅拌，均质，调节 $\mathrm{pH}$ 至 $2.0\sim 4.0$ ，必要时，可逐滴加入盐酸溶液 $(5\mathrm{mol} / \mathrm{L})$ 或氢氧化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ 调节 $\mathrm{pH}$ ，加碱时速度要慢，同时需不断搅拌，防止局部碱化破坏毒素。将混合物加热煮沸 $5\mathrm{min}$ ，冷却至室温，移至量筒中并稀释至 $200~\mathrm{mL}$ ，调节 $\mathrm{pH}$ 至 $2.0\sim 4.0$ 。

将混合物倒回烧杯，搅拌均匀，自然沉降至上清液呈半透明状，不堵塞注射针头即可，必要时将混合物或上清液以 $3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，或用滤纸过滤。收集上清液备用。

# 5.5 小鼠试验

取 $19.0\mathrm{g}\sim 21.0\mathrm{g}$ 健康ICR雄性小鼠6只，称重并记录重量。随机分为实验组和空白对照组两组，每组3只。

对每只实验组小鼠腹腔注射 $1\mathrm{mL}$ 试样提取液，对每只空白对照组腹腔注射 $1\mathrm{mL}$ 盐酸溶液 $(0.18\mathrm{mol} / \mathrm{L})$ 。注射过程中若有一滴以上提取液溢出，须将该只小鼠丢弃，并重新注射一只小鼠。记录注射完毕时间，仔细观察并记录小鼠停止呼吸时的死亡时间(到小鼠呼出最后一口气止)。

若注射试样提取液后，1只或2只小鼠的死亡时间大于 $7\mathrm{min}$ ，需再注射至少三只小鼠。

若小鼠的死亡时间小于 $5\mathrm{min}$ ，稀释试样提取液后，再注射3只小鼠，直至小鼠在 $5\mathrm{min} \sim 7\mathrm{min}$ 内死亡；稀释试样提取液时，需逐滴加入盐酸溶液 $(0.18\mathrm{mol / L})$ 调节 $\mathrm{pH}$ 至 $2.0 \sim 4.0$ 。

# 6 分析结果的表述

# 6.1 以质量分数计的PSP毒力的计算与结果表述

# 6.1.1 以质量分数计的PSP毒力的计算

每 $100\mathrm{g}$ 试样中PSP的含量按式(2)计算：

$$
X = \mathrm {C M U} _ {1} \times \mathrm {C F} _ {2} \times \mathrm {D F} \times 2 0 0 \quad \dots \tag {2}
$$

式中：

$X$ ——每 $100\mathrm{g}$ 样品中PSP的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$

$\mathrm{CMU}_{1}$ ——试样受试组小鼠的中位数校正鼠单位；

$\mathrm{CF_2}$ ——组间毒素转换系数，单位为微克每毫升 $(\mu \mathrm{g / mL})$

DF —稀释倍数；

200 ——试样提取液的体积，单位为毫升 $(\mathrm{mL})$ 。

注：根据检测样品受试组的小鼠死亡时间，查出附录A对应的鼠单位；根据附录B查出小鼠体重所对应的体重校正系数，两者相乘得到该只小鼠的 $\mathrm{CMU}$ 。选取受试组中3只小鼠 $\mathrm{CMU}$ 的中位数，即为 $\mathrm{CMU}_1$ 。

# 6.1.2 以质量分数计的PSP毒力的结果表述

若空白对照组小鼠正常，则报告待测样品中PSP毒素含量为： $\times \times \times \mu \mathrm{g} / 100\mathrm{g}$ 。

# 6.2 以MU计的PSP毒力的计算与结果表述

# 6.2.1 以MU计的PSP毒力的计算

每 $100\mathrm{g}$ 试样中以MU计的PSP毒力按式(3)计算：

$$
Y = \mathrm {C M U} _ {1} \times \mathrm {D F} \times 2 0 0 \quad \dots \tag {3}
$$

式中：

Y ——每 $100\mathrm{g}$ 样品的MU值，单位为鼠单位每百克 $(\mathrm{MU} / 100\mathrm{g})$

$\mathrm{CMU}_{1}$ ——实验组小鼠的中位数校正鼠毒力单位；

DF —稀释倍数；

200 ——试样提取液的体积，单位为毫升 $(\mathrm{mL})$ 。

注：对于取得PSP标准品有困难的实验室，可按式(3)计算以MU计的PSP毒力。

# 6.2.2 以MU计的PSP毒力的判断与结果表述

在空白对照组小鼠正常的情况下进行如下判断和表述：

若实验组中位数死亡时间小于 $5\mathrm{min}$ ，则应对样品提取液进行稀释，再选取3只小鼠进行试验，直至得到中位数死亡时间为 $5\mathrm{min} \sim 7\mathrm{min}$ 为止，根据最后的稀释液实验结果计算样品的鼠单位毒力，报告该样品的鼠单位毒力为： $\times \times \times \mathrm{MU} / 100\mathrm{g}$ 。

若实验组中位数死亡时间大于 $7\mathrm{min}$ ，则直接计算确定样品鼠单位毒力，报告该样品的鼠单位毒力为： $\times \times \times$ MU/100 g。

若实验组中所有小鼠在观察 $15\mathrm{min}$ 内均不死亡，可报告该样品的鼠单位毒力小于 $400\mathrm{MU} / 100\mathrm{g}$ 。

# 酶联免疫吸附法

# 7 原理

游离麻痹性贝类毒素与其酶标记物竞争麻痹性贝类毒素抗体，同时麻痹性贝类毒素抗体与捕捉抗体连接。没有被结合的酶标记物在洗涤步骤中被除去。结合的酶标记物将无色的发色剂转化为蓝色的产物。加入反应停止液后使颜色由蓝转变为黄色。用酶标仪在 $450~\mathrm{nm}$ 波长下测量微孔溶液的吸光度值，试样中麻痹性贝类毒素含量与吸光度值成反比，按绘制的标准曲线定量计算。

# 10.4 测定

将包被有麻痹性贝类毒素捕捉抗体的微孔条插入微孔架并做好标记，其中包括空白对照孔、标准液孔和样液孔，分别做平行孔。向空白对照孔加入 $50~\mu \mathrm{L}$ 磷酸盐缓冲液，标准液孔加入 $50~\mu \mathrm{L}$ 麻痹性贝类毒素标准系列工作液，样液孔加入 $50~\mu \mathrm{L}$ 样液。加入 $50~\mu \mathrm{L}$ 麻痹性贝类毒素酶标记物至每个微孔，轻轻混合；再加入 $50~\mu \mathrm{L}$ 麻痹性贝类毒素抗体至每个微孔，彻底混合，用黏胶纸封住微孔以防溶液挥发， $20^{\circ} \mathrm{C} \sim 25^{\circ} \mathrm{C}$ 黑暗避光处孵育 $15 \mathrm{~min}$ 。孵育结束后，倒去孔中液体，每个微孔注入 $250~\mu \mathrm{L}$ 洗脱液冲洗，翻转微孔板，倾去孔内液体，再重复以上洗板操作5次，在吸水纸上拍干。每孔加 $100~\mu \mathrm{L}$ 过氧化氢和TMB，充分混合， $20^{\circ} \mathrm{C} \sim 25^{\circ} \mathrm{C}$ 黑暗避光处孵育 $15 \mathrm{~min}$ 。每孔加入 $100~\mu \mathrm{L}$ 硫酸溶液（ $1 \mathrm{~mol} / \mathrm{L}$ ）迅速混匀，终止反应，在 $10 \mathrm{~min}$ 内测量并记录 $450 \mathrm{~nm}$ 波长下的吸光度值。若测定的样液的质量浓度超出标准曲线的线性范围，可扩大稀释倍数后重新进行测定。

# 10.5 标准曲线的制作

以麻痹性贝类毒素标准系列工作液质量浓度以10为底的对数值为横坐标，以式(4)计算的百分比吸光度值为纵坐标，绘制标准曲线。

麻痹性贝类毒素标准液和样液的百分比吸光度值按式(4)计算：

$$
A = \frac {S}{S _ {0}} \times 100 \% \tag{4}
$$

式中：

$A$ ——百分比吸光度值；

S —— 麻痹性贝类毒素标准液或样液的平均吸光度值；

$S_{0}$ —— $0\mu \mathrm{g} / \mathrm{kg}$ 的麻痹性贝类毒素标准液的平均吸光度值。

# 11 分析结果的表述

试样中麻痹性贝类毒素的含量按式(5)计算：

$$
X = \frac {\rho \times V \times f}{m} \tag {5}
$$

式中：

$X$ ——试样中麻痹性贝类毒素的含量，单位为微克每克 $(\mu \mathrm{g} / \mathrm{g})$ ；

$\rho$ ——由标准曲线得到的试样待测液中麻痹性贝类毒素的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样提取液的体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释倍数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

注：任何麻痹性贝类毒素含量值大于 $800\mu \mathrm{g / kg}$ 的样品即被认为是有害的，对人类食用不安全。

# 17.1 试样中各种麻痹性贝类毒素含量的计算

试样中各种麻痹性贝类毒素的含量按式(6)计算：

$$
X _ {i} = \frac {\rho_ {i} \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {6}
$$

式中：

$X_{i}$ ——试样中各种麻痹性贝类毒素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho_{i}$ ——从标准工作曲线得到的试样溶液中麻痹性贝类毒素的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

V ——用于超滤的试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算因子；

$m$ ——与超滤试样溶液相当的试样质量，单位为克 $(\mathrm{g})$ 。

计算结果应扣除空白值。计算结果保留三位有效数字。

# 17.2 总毒力计算

试样中麻痹性贝类毒素总毒力 $(\mathrm{STX}_{\mathrm{eq}})$ 按式(7)计算：

$$
\mathrm {S T X} _ {\mathrm {e q}} = \sum_ {i = 1} ^ {n} X _ {i} \cdot r _ {i} \tag {7}
$$

式中：

$\mathrm{STX_{eq}}$ ——试样中麻痹性贝类毒素总毒力，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$X_{i}$ ——各种麻痹性贝类毒素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$r_i$ ——麻痹性贝类毒素的毒性因子。

注：试样中各种麻痹性贝类毒素的含量需按照附录F中的毒性因子(TEFs)，统一转换为总毒力 $(\mathrm{STX}_{\mathrm{eq}})$ 。

# 24.1 试样中各麻痹性贝类毒素含量的计算

试样中各麻痹性贝类毒素的含量按式(8)计算：

$$
X _ {i} = \frac {\rho_ {i} \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {8}
$$

式中：

$X_{i}$ ——试样中各麻痹性贝类毒素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho_{i}$ ——由基质标准工作曲线得到的试样溶液中各麻痹性贝类毒素的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——用于超滤的试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算因子；

$m$ ——与超滤试样溶液相当的试样质量，单位为克 $(\mathrm{g})$ 。

计算结果应扣除空白值。计算结果保留三位有效数字。

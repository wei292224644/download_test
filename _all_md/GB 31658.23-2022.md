# 1 范围

本文件规定了动物性食品中甲硝唑、羟基甲硝唑、地美硝唑和羟基地美硝唑残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、牛、羊和鸡的肌肉、肝脏和肾脏组织中甲硝唑、羟基甲硝唑、地美硝唑和羟基地美硝唑残留量的测定。

# 2 规范性引用文件

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中残留的硝基味唑类药物经乙酸乙酯提取，正己烷液液萃取除脂，固相萃取柱净化，液相色谱-串联质谱检测，基质匹配内标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ ：色谱纯。  
5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.3 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯  
5.1.4 正己烷 $\mathrm{(C_6H_{14})}$ ：色谱纯。  
5.1.5 甲酸（HCOOH）：色谱纯。  
5.1.6 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。  
5.1.7 盐酸(HCl)。

# 5.2 溶液配制

# 5.3 标准品

5.2.1 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液：取浓盐酸 $8.3\mathrm{mL}$ ，加水稀释至 $1\mathrm{L}$ ，混匀。  
5.2.2 $2\%$ 氨水溶液：取氨水 $2\mathrm{mL}$ ，加水稀释至 $100~\mathrm{mL}$ ，混匀，现用现配。  
5.2.3 洗脱液：取甲醇 $80~\mathrm{mL}$ ，加水 $15~\mathrm{mL}$ 、氨水 $5~\mathrm{mL}$ ，混匀，现用现配。  
5.2.4 $0.1\%$ 甲酸水溶液：取水 $500~\mathrm{mL}$ ，加甲酸 $500~\mu \mathrm{L}$ ，混匀。  
5.2.5 $0.1\%$ 甲酸乙腈溶液：取乙腈 $500~\mathrm{mL}$ ，加甲酸 $500~\mu \mathrm{L}$ ，混匀。

甲硝唑、地美硝唑、羟基甲硝唑、羟基地美硝唑、甲硝唑- $\mathrm{D}_3$ 、地美硝唑- $\mathrm{D}_3$ 、羟基甲硝唑- $\mathrm{D}_2$ 、羟基地美硝唑- $\mathrm{D}_3$ ，含量均 $\geqslant 95\%$ 。具体见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液：取甲硝唑、地美硝唑、羟基甲硝唑、羟基地美硝唑标准品适量（相当于各有效成分 $10\mathrm{mg}$ ），精密称定，用甲醇溶解并稀释定容于 $10\mathrm{mL}$ 容量瓶，配制成浓度为 $1\mathrm{mg/mL}$ 的甲硝唑、地美硝唑、羟基甲硝唑和羟基地美硝唑标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期6个月。  
5.4.2 内标储备液：取甲硝唑- $\mathrm{D}_3$ 、地美硝唑- $\mathrm{D}_3$ 、羟基甲硝唑- $\mathrm{D}_2$ 和羟基地美硝唑- $\mathrm{D}_3$ 标准品适量（相当于各有效成分 $10\mathrm{mg}$ )，精密称定，用甲醇溶解并稀释定容于 $10~\mathrm{mL}$ 容量瓶，配制成浓度为 $1\mathrm{mg / mL}$ 的甲硝唑- $\mathrm{D}_3$ 、地美硝唑- $\mathrm{D}_3$ 、羟基甲硝唑- $\mathrm{D}_2$ 和羟基地美硝唑- $\mathrm{D}_3$ 内标储备液。 $-18^{\circ}C$ 以下保存，有效期6个月。  
5.4.3 $10\mu \mathrm{g / mL}$ 混合标准工作液：分别精密量取标准储备液各 $0.1\mathrm{mL}$ 于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的混合标准工作液。一 $18^{\circ}C$ 以下保存，有效期6个月。  
5.4.4 $1\mu \mathrm{g / mL}$ 混合标准工作液：精密量取 $10~\mu \mathrm{g / mL}$ 混合标准工作液 $1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $1\mu \mathrm{g / mL}$ 的混合标准工作液。一 $18^{\circ}C$ 以下保存，有效期3个月。  
5.4.5 $0.1\mu \mathrm{g / mL}$ 混合标准工作液：精密量取 $1\mu \mathrm{g / mL}$ 混合标准工作液 $1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $0.1\mu \mathrm{g / mL}$ 的混合标准工作液。一 $18^{\circ}C$ 以下保存，有效期1个月。  
5.4.6 $10\mu \mathrm{g / mL}$ 混合内标工作液：分别精密量取内标储备液各 $0.1\mathrm{mL}$ 于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的混合内标工作液。一 $18^{\circ}C$ 以下保存，有效期6个月。  
5.4.7 $1\mu \mathrm{g / mL}$ 混合内标工作液：精密量取 $10~\mu \mathrm{g / mL}$ 内标工作液 $1\mathrm{mL}$ 于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $1\mu \mathrm{g / mL}$ 的混合内标工作液。一 $18^{\circ}C$ 以下保存，有效期3个月。

# 5.5 材料

5.5.2 微孔滤膜： $0.2\mu \mathrm{m}$ ，水相。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试样 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 塑料离心管，加 $1\mu \mathrm{g / mL}$ 混合内标工作液 $10~\mu \mathrm{L}$ ，涡旋 $30~\mathrm{s}$ 混匀，静置 $10\mathrm{min}$ 。加乙酸乙酯 $15~\mathrm{mL}$ ，涡旋 $2\mathrm{min}$ ， $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。取上清液于另一 $50~\mathrm{mL}$ 离心管中，残渣中加乙酸乙酯 $15~\mathrm{mL}$ ，重复提取一次，合并2次提取液，于 $25^{\circ}C$ 水浴氮气吹干。

# 8.2 净化

残余物中加 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液 $5\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ 充分溶解，加正己烷 $5\mathrm{mL}$ ，振摇 $1\mathrm{min}$ ， $5000\mathrm{r} / \mathrm{min}$ 离心 $5\mathrm{min}$ ，弃正己烷层。下层再加正己烷 $5\mathrm{mL}$ 重复除脂一次，弃正己烷层，备用。

固相萃取柱依次用甲醇 $2\mathrm{mL}$ 和 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液 $2\mathrm{mL}$ 活化，取备用液过柱，依次用 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液 $2\mathrm{mL}$ 、甲醇 $1\mathrm{mL}$ 和 $2\%$ 氨水 $1\mathrm{mL}$ 淋洗，用洗脱液 $2\mathrm{mL}$ 洗脱。收集洗脱液， $35^{\circ}\mathrm{C}$ 水浴氮气吹干，残余物中加水 $0.5\mathrm{mL}$ 涡旋 $1\mathrm{min}$ ，滤膜过滤，供液相色谱-串联质谱测定。

# 8.3 基质匹配标准曲线的制备

取经提取和净化的空白试料溶液，加入适量的混合标准工作液和 $1\mu \mathrm{g / mL}$ 混合内标工作液 $10~\mu \mathrm{L}$ $35^{\circ}C$ 水浴氮气吹干，加水 $0.5\mathrm{mL}$ 涡旋 $1\mathrm{min}$ ，配制成浓度为 $2\mu \mathrm{g / L},4\mu \mathrm{g / L},20\mu \mathrm{g / L},40\mu \mathrm{g / L},200\mu \mathrm{g / L}$ 和 $400~\mu \mathrm{g / L}$ 的基质匹配标准溶液，过滤后供液相色谱-串联质谱测定。以测得的硝基咪唑类药物与相应内标的特征离子峰面积之比为纵坐标，标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在同样测试条件下，试料溶液中硝基咪唑类药物的保留时间与基质匹配标准工作液中硝基咪唑类药物的保留时间之比，偏差在 $\pm 2.5\%$ 以内，且检测到的相对离子丰度，应当与浓度相当的基质匹配标准溶液离子相对丰度一致。其允许偏差应符合表3要求。

# 8.4.3.2 定量测定

取试料溶液和基质匹配标准工作液，作单点或多点校准，按内标法定量。基质匹配标准工作液及试料溶液中目标物的响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，标准溶液特征离子质量色谱图见附录B。

# 8.5 空白试验

取空白试料，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中硝基咪唑类药物的残留量按标准曲线或公式(1)计算

$$
X = \frac {A \times A _ {i s} ^ {\prime} \times C _ {s} \times C _ {i s} \times V}{A _ {i s} \times A _ {s} \times C _ {i s} ^ {\prime} \times m}. \tag {1}
$$

式中：

$X$ ——试样中硝基咪唑类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

A ——供试试样溶液中硝基咪唑类药物的峰面积；

$A_{is}^{\prime}$ ——基质匹配标准溶液中内标的峰面积；

$C_{s}$ ——基质匹配标准溶液中硝基咪唑类药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{is}$ ——供试试样中内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$V$ ——供试试样最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{is}$ ——供试试样溶液中内标的峰面积；

$A_{s}$ ——基质匹配标准溶液中硝基咪唑类药物的峰面积；

$C_{is}^{\prime}$ ——基质匹配标准溶液中内标浓度的数值，单位为微克每升 $(\mu \mathrm{g / L})$

$m$ ——供试试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $1\mu \mathrm{g / kg}\sim 10\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

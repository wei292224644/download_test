# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T3035—2011《出口植物源性食品中环己烯酮类除草剂残留量的测定液相色谱一质谱/质谱法》。与SN/T3035—2011相比，主要变化如下：

——标准文本格式修改为食品安全国家标准文本格式；

——标准名称中“出口植物源性食品”改为“植物源性食品”。

——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T3035—2011。

# 液相色谱一质谱/质谱法

# 1 范围

本标准规定了大米、大豆、玉米、小白菜、马铃薯、大蒜、葡萄和橙中吡喃草酮（teproloxydim）、禾草灭（alloxydim）、噻草酮（cycloxydim）、苯草酮（methoxyphenone）、烯禾啶（sethoxydim）和烯草酮（clethodim)6种环己烯酮类除草剂的液相色谱一串联质谱法检测方法。

本标准适用于大米、大豆、玉米、小白菜、马铃薯、大蒜、葡萄和橙中吡喃草酮、禾草灭、噻草酮、苯草酮、烯禾啶和烯草酮6种环己烯酮类除草剂残留量的检测与确证。其他食品可参照执行。

# 3 原理

采用乙腈提取试样中残留的环己烯酮类除草剂，提取液经 $\mathrm{C_{18}}$ 和Envi-Carb固相萃取柱净化，液相色谱一质谱/质谱检测和确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
4.1.2 正己烷 $\mathrm{(C_6H_{14})}$ ：色谱纯。  
4.1.3 乙酸 $\mathrm{(CH_3COOH)}$ ：色谱纯。  
4.1.4 丙酮 $(\mathrm{C_3H_6O})$ ：优级纯。  
4.1.5 氯化钠(NaCl)。

# 4.2 溶液配制

4.2.1 正己烷一丙酮 $(6 + 4)$ ：量取 $60~\mathrm{mL}$ 正己烷，加入 $40~\mathrm{mL}$ 丙酮，混匀。  
4.2.2 乙腈一水溶液 $(1 + 1)$ ：量取 $500~\mathrm{mL}$ 乙腈，用水定容至 $1\mathrm{L}$ ，混匀。  
4.2.3 $0.1\%$ 乙酸水溶液：取 $1.0 \mathrm{~mL}$ 乙酸，用水稀释并定容至 $1000 \mathrm{~mL}$ 。

# 4.3 标准品

标准物质：吡喃草酮(CAS号：79-41-9)、禾草灭(CAS号：55635-13-7)、噻草酮(CAS号：101205-02-1)、苯草酮(CAS号：87820-88-0)、烯禾啶(CAS号：74051-80-2)和烯草酮(CAS号：99129-21-2)，纯度 $\geq 99\%$ （相关信息参见表A.1）。

# 4.4 标准溶液配制

4.4.1 环己烯酮类农药标准储备溶液：分别准确称取适量的环己烯酮类农药标准物质，用乙腈配制成浓度为 $1000\mathrm{mg / L}$ 的标准储备溶液，标准溶液避光于 $-18^{\circ}C$ 保存，保存期为12个月。  
4.4.2 环己烯酮类农药混合中间标准溶液：吸取适量的各标准储备溶液，用乙腈稀释成浓度为 $1.0\mathrm{mg / L}$ 的混合中间标准溶液， $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 避光保存，保存期为3个月。  
4.4.3 基质混合标准工作溶液：吸取适量的混合中间标准溶液，用空白样品提取液配成浓度为 $0\mu \mathrm{g} / \mathrm{L}, 10.0\mu \mathrm{g} / \mathrm{L}, 20.0\mu \mathrm{g} / \mathrm{L}, 50.0\mu \mathrm{g} / \mathrm{L}, 100\mu \mathrm{g} / \mathrm{L}$ 和 $200\mu \mathrm{g} / \mathrm{L}$ 的基质混合标准工作溶液。当天配制。

# 4.5 材料

4.5.1 石墨化炭黑固相萃取柱: $3 \mathrm{~mL} / 250 \mathrm{mg}$ , 或相当者。使用前用 $3 \mathrm{~mL}$ 正己烷一丙酮溶液活化, 保持柱体湿润。  
4.5.2 $\mathrm{C_{18}}$ 固相萃取柱：使用前用 $5\mathrm{mL}$ 乙腈活化，保持柱体湿润。  
4.5.3 微孔滤膜： $0.22\mu \mathrm{m}$ ，有机相。

# 5 仪器和设备

5.1 液相色谱一质谱/质谱仪：配有电喷雾离子源(ESI)。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 组织捣碎机。  
5.4 粉碎机。  
5.5 涡旋振荡器。  
5.6 固相萃取装置，带真空泵。  
5.7 氮吹浓缩仪。  
5.8 离心机：转速 $\geq 5000\mathrm{r / min}$   
5.9 离心管： $50~\mathrm{mL}$   
5.10 刻度试管： $15\mathrm{mL}$

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按照GB2763的规定执行。在取样和制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 玉米、大豆和大米

取样品约 $500 \mathrm{~g}$ , 用粉碎机充分粉碎, 混匀。试样均分为 2 份, 装入洁净容器, 密封, 并标明标记。

# 6.2.2 小白菜、马铃薯、大蒜、葡萄和橙

取样品约 $500\mathrm{g}$ ，将其(不可用水洗)切碎后，用捣碎机将样品加工成浆状，混匀。试样均分为2份，装入洁净容器，密封，并标明标记。于 $-18^{\circ}\mathrm{C}$ 以下冷冻存放。

# 6.3 试样保存

玉米、大豆和大米试样于常温下保存。小白菜、马铃薯、大蒜、葡萄和橙试样于 $-18^{\circ}C$ 以下冷冻存放。

# 7 分析步骤

# 7.1 提取

# 7.1.1 玉米、大豆和大米

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样于 $50\mathrm{mL}$ 具塞离心管中，加入 $10\mathrm{mL}$ 水，涡旋混匀后放置 $1\mathrm{h}$ 。然后加入 $4.0\mathrm{g}$ 氯化钠，再加入 $15\mathrm{mL}$ 乙腈高速均质提取 $3\mathrm{min}$ ，振荡提取 $15\mathrm{min}$ ，于 $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将乙腈层转移至 $25\mathrm{mL}$ 容量瓶中。残渣再用 $10\mathrm{mL}$ 乙腈重复提取一次，合并提取液，并用乙腈定容至 $25\mathrm{mL}$ 。

# 7.1.2 小白菜、马铃薯、大蒜、葡萄和橙

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样于 $50\mathrm{mL}$ 具塞离心管中，然后加入 $2.0\mathrm{g}$ 氯化钠，再加入 $15\mathrm{mL}$ 乙腈高速均质提取 $3\mathrm{min}$ ，振荡提取 $15\mathrm{min}$ ，于 $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将乙腈层转移至 $25\mathrm{mL}$ 容量瓶中。残渣再用 $10\mathrm{mL}$ 乙腈重复提取一次，合并提取液，并用乙腈定容至 $25\mathrm{mL}$ 。

# 7.2 净化

# 7.2.1 大豆和玉米

吸取 $5.0 \mathrm{~mL}$ 提取液于 $\mathrm{C}_{18}$ 固相萃取柱中，以约 $1.5 \mathrm{~mL} / \mathrm{min}$ 的流速使样液全部通过固相萃取柱，再用 $3 \mathrm{~mL}$ 乙腈淋洗并抽干固相萃取柱，收集全部流出液于 $15 \mathrm{~mL}$ 离心管中， $40^{\circ} \mathrm{C}$ 下用氮气吹至近干。残渣用 $3.0 \mathrm{~mL}$ 正己烷一丙酮溶解，将溶解液加入处理过的Envi-Carb固相萃取柱中，以约 $1.5 \mathrm{~mL} / \mathrm{min}$ 的流速使样液全部通过固相萃取柱，再用 $5 \mathrm{~mL}$ 正己烷一丙酮润洗样品管，并将润洗液一并加入Envi-Carb固相萃取柱，收集全部流出液于 $15 \mathrm{~mL}$ 离心管中，于 $40^{\circ} \mathrm{C}$ 水浴中氮吹至近干，残余物用 $1.0 \mathrm{~mL}$ 乙腈一水溶解，涡旋混匀后，过微孔滤膜，供液相色谱一质谱/质谱仪测定。

# 7.2.2 大米、小白菜、马铃薯、大蒜、葡萄和橙

取 $5.0\mathrm{mL}$ 提取液于 $15\mathrm{mL}$ 离心管中， $40^{\circ}\mathrm{C}$ 下用 $\mathrm{N}_2$ 吹至近干。残渣用 $2.0\mathrm{mL}$ 正己烷一丙酮溶解，将溶解液加入处理过的Envi-Carb固相萃取柱中，以约 $1.5\mathrm{mL} / \mathrm{min}$ 的流速使样液全部通过固相萃取柱，再用 $5\mathrm{mL}$ 正己烷一丙酮润洗样品管，并将润洗液一并加入Envi-Carb固相萃取柱，收集全部流出液于 $15\mathrm{mL}$ 离心管中，于 $40^{\circ}\mathrm{C}$ 水浴中氮吹至近干，残余物用 $1.0\mathrm{mL}$ 乙腈一水溶解，涡旋混匀后，过微孔滤膜，供液相色谱一质谱/质谱仪测定。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中各环已烯酮类农药的含量。

$$
X _ {i} = \frac {A _ {i} \times C _ {s} \times V}{A _ {s i} \times m}. \tag {1}
$$

式中：

$X_{i}$ ——试样中各环己烯酮类农药的残留含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A_{i}$ ——样液中各环己烯酮类农药的峰面积；

$C_{s}$ ——标准工作液中各环己烯酮类农药的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积, 单位为毫升 $(\mathrm{mL})$ ;

$A_{s}$ ——标准工作液中各环己烯酮类农药的峰面积；

$m$ ——最终样液所代表的试样量，单位为克(g)。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

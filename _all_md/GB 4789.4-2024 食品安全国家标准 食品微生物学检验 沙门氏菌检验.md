# 5 操作步骤

# 5.1 预增菌

无菌操作取 $25\mathrm{g(mL)}$ 样品，置于盛有 $225~\mathrm{mL}$ BPW的无菌均质杯中，以 $8000\mathrm{r / min}\sim 10000\mathrm{r / min}$

均质 $1 \mathrm{~min} \sim 2 \mathrm{~min}$ , 或置于盛有 $225 \mathrm{~mL}$ BPW 的无菌均质袋内, 用拍击式均质器拍打 $1 \mathrm{~min} \sim 2 \mathrm{~min}$ 。对于液态样品, 也可置于盛有 $225 \mathrm{~mL}$ BPW 的无菌锥形瓶或其他合适容器中振荡混匀。如需调节 $\mathrm{pH}$ 时, 用 $1 \mathrm{~mol} / \mathrm{L}$ NaOH 或 $\mathrm{HCl}$ 调 $\mathrm{pH}$ 至 $6.8 \pm 0.2$ 。无菌操作将样品转至 $500 \mathrm{~mL}$ 锥形瓶或其他合适容器内 (如均质杯本身具有无孔盖或使用均质袋时, 可不转移样品), 置于 $36^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 培养 $8 \mathrm{~h} \sim 18 \mathrm{~h}$ 。

对于乳粉，无菌操作称取 $25\mathrm{g}$ 样品，缓缓倾倒在广口瓶或均质袋内 $225~\mathrm{mL}$ BPW的液体表面，勿调节 $\mathrm{pH}$ ，也暂不混匀，室温静置 $60\mathrm{min}\pm 5\mathrm{min}$ 后再混匀，置于 $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 培养 $16\mathrm{h}\sim 18\mathrm{h}$ 。

冷冻样品如需解冻，取样前在 $40^{\circ}\mathrm{C} \sim 45^{\circ}\mathrm{C}$ 的水浴中解冻不超过 $15\mathrm{min}$ ，或在 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 冰箱缓慢化冻不超过 $18\mathrm{h}$ 。

# 5.2 选择性增菌

轻轻摇动预增菌的培养物，移取 $0.1\mathrm{mL}$ 转种于 $10~\mathrm{mL}$ RVS中，混匀后于 $42^{\circ}C\pm 1^{\circ}C$ 培养 $18\mathrm{h}\sim$ $24\mathrm{h}$ 。同时，另取 $1\mathrm{mL}$ 转种于 $10~\mathrm{mL}$ TTB中后混匀，低背景菌的样品(如深加工的预包装食品等)置于 $36^{\circ}C\pm 1^{\circ}C$ 培养 $18\mathrm{h}\sim 24\mathrm{h}$ ，高背景菌的样品(如生鲜禽肉等)置于 $42^{\circ}C\pm 1^{\circ}C$ 培养 $18\mathrm{h}\sim 24\mathrm{h}$ 。

如有需要，可将预增菌的培养物在 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 冰箱保存不超过 $72\mathrm{h}$ ，再进行选择性增菌。

# 5.3 分离

振荡混匀选择性增菌的培养物后，用直径 $3\mathrm{mm}$ 的接种环取每种选择性增菌的培养物各一环，分别划线接种于一个BS琼脂平板和一个XLD琼脂平板(也可使用HE琼脂平板、沙门氏菌显色培养基平板或其他合适的分离琼脂平板)，于 $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 分别培养 $40\mathrm{h}\sim 48\mathrm{h}$ (BS琼脂平板)或 $18\mathrm{h}\sim 24\mathrm{h}$ (XLD琼脂平板、HE琼脂平板、沙门氏菌显色培养基平板)，观察各个平板上生长的菌落，是否符合表1的菌落特征。

如有需要，可将选择性增菌的培养物在 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 冰箱保存不超过 $72\mathrm{h}$ ，再进行分离。

# 5.5.2 多价菌体抗原(O)鉴定

在玻片上划出两个约 $1\mathrm{cm}\times 2\mathrm{cm}$ 的区域，挑取待测菌培养物，各放约一环于玻片上的每一区域上部，在其中一个区域下部加一滴多价菌体(O)血清，在另一区域下部加入一滴生理盐水，作为对照。再用无菌的接种环或针将两个区域内的待测菌培养物，分别与血清和生理盐水研成乳状液。将玻片倾斜摇动混合 $1\mathrm{min}$ ，并对着黑暗背景进行观察，与对照相比，出现可见的菌体凝集者为阳性反应。O血清不凝集时，将菌株接种在琼脂含量较高(如 $2\% \sim 3\%$ )的培养基上培养后再鉴定，如果是由于Vi抗原的存在而阻止了O血清的凝集反应时，可挑取待测菌培养物在 $1\mathrm{mL}$ 生理盐水中制成浓菌液，在沸水中水浴 $20\mathrm{min}\sim 30\mathrm{min}$ ，冷却后再进行鉴定。

注：不同厂商沙门氏菌诊断血清的组成、鉴定操作及结果判断，可能存在差异。使用商品化的沙门氏菌诊断血清进行血清学鉴定时，应遵循其产品说明。

# 5.5.3 多价鞭毛抗原（H)鉴定

按5.5.2的操作，将多价菌体(O)血清换成多价鞭毛(H)血清，进行多价鞭毛抗原(H)鉴定。H抗原发育不良时，将菌株接种在半固体琼脂平板的中央，待菌落蔓延生长时，在其边缘部分取菌鉴定；或将菌株接种在装有半固体琼脂的小玻管培养1代～2代，自远端取菌再进行鉴定。

# 5.6 血清学分型（选做项目）

# 5.6.1 O抗原的鉴定

用 $\mathrm{A} \sim \mathrm{F}$ 多价O血清做玻片凝集试验，同时用生理盐水做对照。在生理盐水中自凝者为粗糙型菌株，不能分型。

# 5.6.2.1 简易平板法

将半固体琼脂平板烘干表面水分，挑取已知相的H因子血清1环，滴在半固体平板表面，正置平板片刻待血清吸收，在滴加血清部位的中央点种待测菌株，翻转平板置于 $36\,^{\circ}\mathrm{C} \pm 1\,^{\circ}\mathrm{C}$ 培养后，在形成蔓延生长的菌苔边缘取菌鉴定。

# 5.6.2.2 小玻管法

将 $1\mathrm{mL}\sim 2\mathrm{mL}$ 半固体琼脂熔化后冷却至 $48~^\circ \mathrm{C}$ 左右，加入已知相的H因子血清 $0.05\mathrm{mL}\sim$ $0.1\mathrm{mL}$ ，混匀后装入 $3\mathrm{mm}\times 50\mathrm{mm}$ 两端开口的小玻管内。待琼脂凝固后，用接种针挑取待测菌，接种于小玻管一端的琼脂内。将小玻管平放在平皿内，置于 $36~^\circ \mathrm{C}\pm 1~^\circ \mathrm{C}$ 培养，并采取保湿措施以防琼脂中水分蒸发而干缩。每天观察结果，待另一相细菌解离后，从小玻管另一端挑取细菌进行鉴定。培养基内血清的浓度应有适当的比例，过高时细菌不能生长，过低时同一相细菌的动力不能抑制。一般按原血清 $1:(200\sim 800)$ 的量加入。

# 5.6.2.3 小套管法

在装有大约 $10\mathrm{mL}$ 半固体琼脂培养基的试管中，插入 $3\mathrm{mm}\times 50\mathrm{mm}$ 两端开口的小玻管（下端开口要留一个缺口，不要平齐），小玻管的上端应高出于培养基的表面， $121^{\circ}\mathrm{C}$ 高压灭菌 $15\mathrm{min}$ 后备用。临用时加热熔化，并冷却至 $48^{\circ}\mathrm{C}$ 左右，挑取已知相的H因子血清1环，加入小玻管中的培养基内，略加搅动使其混匀。待琼脂凝固后，在小玻管中的半固体表层内接种待测菌，于 $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 培养，每天观察结果，待另一相细菌解离后，从小玻管外的半固体表面取菌鉴定，或将所取的菌转种 $1\%$ 琼脂斜面，于 $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 培养后再进行鉴定。

# 5.6.3 Vi抗原的鉴定

用Vi因子血清进行鉴定。已知具有Vi抗原的菌型有：伤寒沙门氏菌，丙型副伤寒沙门氏菌，都柏林沙门氏菌。

# 5.6.4 血清型的判定

根据血清学分型鉴定的结果，按照附录B或有关沙门氏菌属抗原表判定血清型。

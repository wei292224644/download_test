# 前言

本标准代替GB5009.137—2016《食品安全国家标准 食品中锑的测定》。

本标准与GB5009.137—2016相比，主要变化如下：

——增加了电感耦合等离子体质谱法为第二法；

——修改了第一法氢化物原子荧光光谱法的还原剂反应条件。

# 食品安全国家标准

# 食品中锑的测定

# 1 范围

本标准规定了食品中锑的氢化物原子荧光光谱和电感耦合等离子体质谱测定方法。

本标准适用于食品中锑的测定。

# 第一法 氢化物原子荧光光谱法

# 2 原理

试样经酸加热消解后，在酸性介质中，试样中的锑与硼氢化钠或硼氢化钾反应生成挥发性的锑氢化物，由载气带入原子化器中进行原子化，在锑空心阴极灯的激发下产生原子荧光，其荧光强度与锑含量成正比，并外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 硝酸 $(\mathrm{HNO}_3)$ 。  
3.1.2 盐酸（HCl）。  
3.1.3 高氯酸 $\mathrm{(HClO_4)}$ 。  
3.1.4 硫脲 $\left[\left(\mathrm{NH}_{2}\right)_{2} \mathrm{CS}\right]$ ：分析纯。  
3.1.5 碘化钾(KI)：分析纯。  
3.1.6 抗坏血酸 $\left(\mathrm{C}_{6} \mathrm{H}_{8} \mathrm{O}_{6}\right)$ ：分析纯。  
3.1.7 硼氢化钾 $\mathrm{(KBH_4)}$ 或硼氢化钠 $(\mathrm{NaBH}_4)$   
3.1.8 氢氧化钾(KOH)或氢氧化钠(NaOH)。  
3.1.9 酒石酸 $(\mathrm{C_4H_6O_6})$ ：纯度 $\geq 99.5\%$   
3.1.10 氩气（Ar）：纯度 $>99.99\%$

# 3.2 试剂配制

3.2.1 硝酸-高氯酸混合酸 $(10 + 1)$ ：量取 $10\mathrm{mL}$ 高氯酸缓慢加入 $100~\mathrm{mL}$ 硝酸中，混匀。  
3.2.2 盐酸溶液 $(1 + 1)$ ：量取 $50~\mathrm{mL}$ 盐酸，加入到 $50~\mathrm{mL}$ 水中，混匀。  
3.2.3 硫脲-抗坏血酸溶液：分别称取 $10.0\mathrm{g}$ 硫脲、 $10.0\mathrm{g}$ 抗坏血酸，溶于 $100\mathrm{mL}$ 水中，混匀，避光保存。临用现配。  
3.2.4 硫脲-碘化钾溶液：分别称取 $2.0\mathrm{g}$ 硫脲、 $10.0\mathrm{g}$ 碘化钾，溶于 $100\mathrm{mL}$ 水中，混匀，避光保存。临用现配。

3.2.5 氢氧化钾溶液 $(5.0\mathrm{g} / \mathrm{L})$ ：称取 $5.0\mathrm{g}$ 氢氧化钾，用水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。临用现配。该溶液中的氢氧化钾也可用氢氧化钠代替。  
3.2.6 硼氢化钾溶液 $(20\mathrm{g / L})$ ：称取 $10.0\mathrm{g}$ 硼氢化钾，用氢氧化钾溶液 $(5.0\mathrm{g / L})$ 溶解并稀释至 $500~\mathrm{mL}$ ，混匀，临用现配。该溶液中的硼氢化钾也可用 $7.0\mathrm{g}$ 硼氢化钠代替。  
3.2.7 硝酸溶液 $(1 + 5)$ ：量取 $100~\mathrm{mL}$ 硝酸，加入 $500~\mathrm{mL}$ 水中，混匀。  
3.2.8 盐酸溶液 $(5 + 95)$ ：量取 $50~\mathrm{mL}$ 盐酸，加入 $950~\mathrm{mL}$ 水中，混匀。  
3.2.9 硝酸溶液 $(1 + 1)$ ：量取 $50~\mathrm{mL}$ 硝酸，加入 $50~\mathrm{mL}$ 水中，混匀。  
3.2.10 硝酸溶液 $(2.5\mathrm{mol} / \mathrm{L})$ ：量取 $173~\mathrm{mL}$ 硝酸，用水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。

# 3.3 标准品

金属锑(Sb，CAS号：7440-36-0)：纯度 $>99.99\%$

# 3.4 标准溶液配制

3.4.1 锑标准储备液 $(1000\mathrm{mg / L})$ : 称取 $0.50\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ) 金属锑于烧杯中, 加 $5.0\mathrm{g}$ 酒石酸和 $10.0\mathrm{mL}$ 硝酸溶液 $(1 + 1)$ , 溶解后转入 $500\mathrm{mL}$ 容量瓶中, 用 $2.5\mathrm{mol / L}$ 硝酸溶液定容至刻度, 混匀, 置于聚乙烯塑料瓶中, 室温下避光保存, 有效期为 2 年。或使用经国家认证并授予标准物质证书的锑标准溶液。  
3.4.2 锑标准中间液 $(100\mathrm{mg / L})$ ：准确吸取 $1.00~\mathrm{mL}$ 锑标准储备液 $(1000\mathrm{mg / L})$ 于 $10~\mathrm{mL}$ 容量瓶中，加水至刻度，混匀，置于聚乙烯塑料瓶中， $0^{\circ}C\sim 5^{\circ}C$ 密封避光保存，有效期为6个月。  
3.4.3 锑标准使用液 $(1.00\mathrm{mg / L})$ ：准确吸取 $1.00~\mathrm{mL}$ 锑标准中间液 $(100\mathrm{mg / L})$ 于 $100~\mathrm{mL}$ 容量瓶中，加水至刻度，混匀，置于聚乙烯塑料瓶中， $0^{\circ}C\sim 5^{\circ}C$ 密封避光保存，有效期为1个月。  
3.4.4 锑标准系列溶液：分别准确吸取梯标准使用液 $(1.00\mathrm{mg / L})0\mathrm{mL},0.100\mathrm{mL},0.200\mathrm{mL}$ 、 $0.400~\mathrm{mL}$ 、 $0.600~\mathrm{mL}$ 、 $0.800~\mathrm{mL}$ 、 $1.00~\mathrm{mL}$ 和 $2.00~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加入少量水稀释后，加入 $10~\mathrm{mL}$ 盐酸溶液 $(1 + 1)$ 、 $10~\mathrm{mL}$ 硫脲-碘化钾溶液或硫脲-抗坏血酸溶液，加水定容至刻度，混匀。此锑标准系列溶液的质量浓度为 $0\mu \mathrm{g / L},1.00\mu \mathrm{g / L},2.00\mu \mathrm{g / L},4.00\mu \mathrm{g / L},6.00\mu \mathrm{g / L},8.00\mu \mathrm{g / L},10.0\mu \mathrm{g / L},$ $20.0\mu \mathrm{g / L}$ 。放置 $1\textrm{h}$ 后测定。临用现配。

注：可根据仪器的灵敏度、样品中锑的实际含量及不同仪器型号确定标准系列溶液中锑的质量浓度范围。

# 4 仪器和设备

注：所有玻璃器皿及聚四氟乙烯消解内罐和内盖均需用硝酸溶液 $(1 + 5)$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

4.1 原子荧光光谱仪：配锑空心阴极灯。  
4.2 天平：感量分别为 $1\mathrm{mg}$ 和 $0.1\mathrm{mg}$   
4.3 可调式电热板。  
4.4 可调式电炉。  
4.5 微波消解系统：配聚四氟乙烯消解罐。  
4.6 恒温干燥箱。  
4.7 样品粉碎设备：匀浆机、高速粉碎机等。  
4.8 压力消解器。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固态样品

# 5.1.1.1 干样

豆类、谷物、菌类、茶叶、干制水果、焙烤食品等低含水量样品，取可食部分，粉碎均匀；对于固体乳制品、蛋白粉、面粉等呈均匀状的粉状样品，摇匀。

# 5.1.1.2 鲜样

蔬菜、水果、水产品等高含水量样品，洗净晾干，取可食部分匀浆均匀；对于肉类、蛋类等样品取可食部分匀浆均匀。

# 5.1.1.3 速冻及罐头食品

经解冻的速冻食品及罐头样品，取可食部分(包括可食用液体部分)匀浆均匀。

# 5.1.2 液态样品

软饮料、液态调味品等样品摇匀。

# 5.1.3 半固态样品

搅拌、匀浆均匀。

# 5.2 样品前处理

# 5.2.1 湿法消解

称取固体试样 $0.25\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或移取液体试样 $1.00~\mathrm{mL}\sim 5.00~\mathrm{mL}$ 于消化容器中，含有乙醇或二氧化碳的样品先在电热板上低温加热除去乙醇或二氧化碳，加入硝酸-高氯酸混合酸 $(10 + 1)$ $5\mathrm{mL}\sim 10\mathrm{mL}$ 浸泡放置 $1\textrm{h}$ 或过夜，再置于电热板上加热消解，如消解过程溶液色泽较深，稍冷后补加少量硝酸，继续消解，消解至冒白烟，消化液呈无色透明或略带黄色，再继续加热赶酸至 $0.5\mathrm{mL}\sim 1\mathrm{mL}$ 止，冷却后将溶液转移至 $25~\mathrm{mL}$ 容器中，并用少量水多次冲洗容器，全部转移后加入 $2.5\mathrm{mL}$ 盐酸溶液 $(1 + 1)$ ，加入硫脲-碘化钾溶液或硫脲-抗坏血酸溶液 $2.5~\mathrm{mL}$ ，用水稀释定容至 $25~\mathrm{mL}$ ，摇匀，放置 $1\textrm{h}$ 后测定。同时做空白试验。

# 5.2.2 微波消解

称取固体试样 $0.2\mathrm{g}\sim 0.8\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，含水分较多的样品可适当增加取样量至 $1\mathrm{g}$ 或移取液体试样 $1.00\mathrm{mL}\sim 3.00\mathrm{mL}$ ，置于微波消解罐中，含有乙醇或二氧化碳的样品先在电热板上低温加热除去乙醇或二氧化碳，加硝酸 $5\mathrm{mL}\sim 7\mathrm{mL}$ ，按照微波消解的操作步骤消解试样（消解参考条件见附录A）。消解完毕，待消解罐冷却后打开，于 $140^{\circ}\mathrm{C}\sim 160^{\circ}\mathrm{C}$ 加热赶酸至 $0.5\mathrm{mL}\sim 1\mathrm{mL}$ 止，用少量水分多次冲洗消解罐，合并洗涤液于 $25\mathrm{mL}$ 容器中，加入 $2.5\mathrm{mL}$ 盐酸溶液 $(1 + 1)$ ，加入硫脲-碘化钾溶液或硫脲-抗坏血酸溶液 $2.5\mathrm{mL}$ ，用水稀释定容至 $25\mathrm{mL}$ ，摇匀，放置 $1\mathrm{h}$ 后测定。同时做空白试验。

# 5.2.3 压力罐消解

称取固体试样 $0.2\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，含水量较多的样品可适当增加取样量至 $2\mathrm{g}$ 或移取液体试样 $1.00\mathrm{mL}\sim 5.00\mathrm{mL}$ ，置于聚四氟乙烯内罐中，含有乙醇或二氧化碳的样品先在电热板上低温加热除去乙醇或二氧化碳，加硝酸 $4\mathrm{mL}\sim 8\mathrm{mL}$ 浸泡过夜。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱，于 $140^{\circ}\mathrm{C}\sim 160^{\circ}\mathrm{C}$ 下保持 $4\mathrm{h}\sim 5\mathrm{h}$ ，在箱内自然冷却至室温，开盖取出内罐，于 $140^{\circ}\mathrm{C}\sim 160^{\circ}\mathrm{C}$ 下加热赶酸至 $0.5\mathrm{mL}\sim 1\mathrm{mL}$ 止，用少量水分多次冲洗消解罐，合并洗涤液于 $25\mathrm{mL}$ 容器中，加入 $2.5\mathrm{mL}$ 盐酸溶液 $(1 + 1)$ ，加入硫脲-碘化钾溶液或硫脲-抗坏血酸溶液 $2.5\mathrm{mL}$ ，用水稀释定容至 $25\mathrm{mL}$ ，摇匀，放置 $1\mathrm{h}$ 后测定。同时做空白试验。

# 5.3 仪器参考条件

仪器参考条件：光电倍增管电压为 $270\mathrm{V}$ ；空心阴极灯电流为 $40\mathrm{mA}$ ；原子化器高度为 $8\mathrm{mm}$ ；载气为氩气；载气流速为 $400\mathrm{mL / min}$ ；屏蔽气流速为 $800\mathrm{mL / min}$ ；测量方式为标准曲线法；读数方式为峰面积；读数时间为 $10\mathrm{s}$ ；延迟时间为 $1\mathrm{s}$ ；进样体积为 $0.5\mathrm{mL}$ 。

# 5.4 标准曲线的制作

设定好仪器最佳条件，将炉温升至所需温度后，稳定 $20\mathrm{min}\sim 30\mathrm{min}$ 后测量。参考载流条件为：以盐酸 $(5 + 95)$ 为载流，硼氢化钾溶液 $(20~\mathrm{g / L})$ 为还原剂，连续用标准系列溶液的零管进样，待读数稳定之后，锑标准系列溶液按浓度由低到高的顺序分别导入仪器，测定荧光值。以锑标准系列溶液的质量浓度为横坐标，相应的荧光值为纵坐标，绘制标准曲线。

# 5.5 试样溶液的测定

将空白溶液和试样溶液分别注入原子荧光光谱仪中，测定荧光值，根据标准曲线得到溶液中锑元素的浓度。

# 6 分析结果的表述

试样中锑的含量按照公式(1)进行计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times f}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中锑的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$\rho$ ——由标准曲线得到的试样溶液中锑的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——由标准曲线得到的空白溶液中锑的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样消化液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样溶液稀释倍数；

$m$ ——试样称样量或体积，单位为克或毫升(g或 $\mathrm{mL}$ )；

1000——换算系数。

当锑含量 $\geqslant 1.00\mathrm{mg / kg}$ （或 $\mathrm{mg / L}$ )时，计算结果保留3位有效数字；当锑含量 $<  1.00~\mathrm{mg / kg}$ （或 $\mathrm{mg / L})$ 时，计算结果保留2位有效数字。

# 7 精密度

样品中锑含量大于 $1\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )时，在重复性条件下获得的两次独立测定结果的绝对差值

不得超过算术平均值的 $10\%$ ；小于或等于 $1\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )且大于 $0.1\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ ；小于或等于 $0.1\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当取样量为 $0.5\mathrm{g}$ （或 $2.0~\mathrm{mL}$ ），定容体积为 $25~\mathrm{mL}$ 时，该方法的检出限为 $0.01\mathrm{mg / kg}$ （或 $0.003\mathrm{mg / L})$ ，定量限为 $0.04\mathrm{mg / kg}$ (或 $0.01\mathrm{mg / L})$ 。

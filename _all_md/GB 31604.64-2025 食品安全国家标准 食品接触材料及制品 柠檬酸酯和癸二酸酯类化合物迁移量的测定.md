# 癸二酸酯类化合物迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中柠檬酸三乙酯、癸二酸二正丁酯、乙酰柠檬酸三丁酯、癸二酸二(2-乙基己基)酯、癸二酸二异辛酯和癸二酸二(2,2,6,6-四甲基-4-哌啶)酯迁移量的测定方法。

本标准适用于塑料、橡胶、黏合剂、涂料和涂层、纸等食品接触材料及制品中柠檬酸三乙酯、癸二酸二正丁酯、乙酰柠檬酸三丁酯、癸二酸二(2-乙基己基)酯、癸二酸二异辛酯和癸二酸二(2,2,6,6-四甲基4-哌啶)酯迁移量的测定。

# 2 原理

食品接触材料及制品采用食品模拟物或化学替代溶剂进行迁移试验，水性、酸性、含乙醇食品模拟物浸泡液经正己烷萃取后上机测定；含油脂食品模拟物浸泡液经乙腈提取，提取液经基质分散萃取净化、氮吹浓缩后上机测定；化学替代溶剂(95%乙醇和异辛烷)浸泡液直接上机测定。试液通过气相色谱柱分离，质谱检测器检测，峰面积外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。试验过程中宜避免使用任何可能对分析造成污染的塑料设备，玻璃器皿和设备在使用前宜清洗干净。

# 3.1 试剂

3.1.1 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
3.1.2 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。  
3.1.3 $25\%$ （质量分数）氨水。  
3.1.4 硫酸镁 $(\mathrm{MgSO_4})$ 。  
3.1.5 无水乙醇 $\mathrm{(C_2H_6O)}$ ：色谱纯。  
3.1.6 冰乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
3.1.7 异辛烷 $\left(\mathrm{C}_{8} \mathrm{H}_{18}\right)$ ：色谱纯。  
3.1.8 橄榄油：应符合GB5009.156中油脂类食品模拟物的规定。

# 3.2 材料

3.2.1 乙二胺-N-丙基硅烷化硅胶填料(PSA)： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}$   
3.2.2 十八烷基硅烷键合硅胶填料 $(\mathrm{C}_{18})$ : $40~\mu \mathrm{m}\sim 60~\mu \mathrm{m}$ 。

# 3.3 试剂配制

3.3.1 酸性、含乙醇和含油脂食品模拟物及化学替代溶剂：按照GB31604.1和GB5009.156的规定。

3.3.2 $10\%$ （质量分数)氨水：量取 $211~\mathrm{mL}~25\%$ 氨水和 $289~\mathrm{mL}$ 水，混匀。

# 3.4 标准品

3.4.1 柠檬酸三乙酯 $\left(\mathrm{C}_{12} \mathrm{H}_{2} \mathrm{O}_{7}\right)$ , CAS 号: 77-93-0): 纯度 $\geqslant 98\%$ , 或经国家认证并授予标准物质证书的标准品。  
3.4.2癸二酸二正丁酯 $\mathrm{(C_{18}H_{34}O_4}$ ，CAS号：109-43-3）：纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.4.3 乙酰柠檬酸三丁酯 $\mathrm{(C_{20}H_{34}O_8}$ ,CAS号：77-90-7)：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.4.4癸二酸二(2-乙基己基)酯 $\mathrm{(C_{26}H_{50}O_4}$ ,CAS号：122-62-3)；纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.4.5 癸二酸二异辛酯 $\mathrm{(C_{26}H_{50}O_4}$ ，CAS号：27214-90-0）：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.4.6 癸二酸二(2,2,6,6-四甲基-4-哌啶)酯 $\mathrm{(C_{28}H_{52}N_2O_4}$ , CAS号:52829-07-9):纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.5 标准溶液的配制

# 3.5.1 单一标准储备液 $(1000\mathrm{mg / L})$ 的配制

分别准确称取 $10\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )柠檬酸三乙酯、癸二酸二正丁酯、乙酰柠檬酸三丁酯、癸二酸二(2-乙基己基)酯、癸二酸二异辛酯和癸二酸二(2,2,6,6-四甲基-4-哌啶)酯标准品，用正己烷溶解后转移至 $10~\mathrm{mL}$ 棕色容量瓶中，用正己烷定容至刻度并摇匀，将溶液转移至棕色玻璃瓶中，于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中避光保存，有效期6个月。

# 3.5.2 混合标准中间溶液 $\mathbf{A}(100\mathrm{mg / L})$ 的配制

分别准确吸取以上6种单一标准储备液各 $1.00\mathrm{mL}$ 于同一个 $10~\mathrm{mL}$ 棕色容量瓶中，用无水乙醇准确定容至刻度后摇匀，将溶液转移至棕色玻璃瓶中，于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中避光保存，有效期3个月。

# 3.5.3 混合标准中间溶液 $\mathbf{B}(100\mathrm{mg / L})$ 的配制

分别准确吸取以上6种单一标准储备液各 $1.00\mathrm{mL}$ 于同一个 $10~\mathrm{mL}$ 棕色容量瓶中，用正己烷准确定容至刻度后摇匀，将溶液转移至棕色玻璃瓶中，于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中避光保存，有效期3个月。

# 3.5.4 酸性食品模拟物标准工作溶液

分别吸取 $5.00\mathrm{mL}4\%$ 乙酸于6支刻度玻璃管中，准确移取 $0\mathrm{mL},0.020\mathrm{mL},0.10\mathrm{mL},0.20\mathrm{mL}$ $0.40~\mathrm{mL}$ 和 $0.50~\mathrm{mL}$ 混合标准中间溶液A，加入 $4 \%$ 乙酸并定容至 $10~\mathrm{mL}$ ，混匀后准确移取 $2.4~\mathrm{mL}10\%$ （质量分数)氨水，再分别加入 $5.00~\mathrm{mL}$ 正己烷，涡旋振荡萃取 $5\mathrm{min},4000\mathrm{r / min}$ 转速下离心 $5\mathrm{min}$ ，取上层正己烷溶液转移至洁净容量瓶中，再分别加入 $5.00~\mathrm{mL}$ 正己烷重复上述步骤，合并两次溶液并用正己烷定容至 $10~\mathrm{mL}$ 。得到质量浓度分别为 $0\mathrm{mg / L},0.20\mathrm{mg / L},1.0\mathrm{mg / L},2.0\mathrm{mg / L},4.0\mathrm{mg / L}$ 和 $5.0~\mathrm{mg / L}$ 的混合标准工作溶液。临用现配。

# 3.5.5 水、含乙醇食品模拟物标准工作溶液

分别吸取 $5.00\mathrm{mL}$ 食品模拟物于6支刻度玻璃管中，用移液器准确移取 $0\mathrm{mL}, 0.020\mathrm{mL}, 0.10\mathrm{mL}, 0.20\mathrm{mL}, 0.40\mathrm{mL}$ 和 $0.50\mathrm{mL}$ 混合标准中间溶液A，加入对应的产品模拟物并定容至

$10~\mathrm{mL}$ ，混匀后准确移取 $0.25~\mathrm{mL}~10\%$ （质量分数)氨水，再加入 $5.00~\mathrm{mL}$ 正己烷，涡旋振荡萃取 $5\mathrm{min},4000\mathrm{r / min}$ 下离心 $5\mathrm{min}$ ，取上层正己烷溶液转移至洁净容量瓶中，再分别加入 $5.00~\mathrm{mL}$ 正己烷重复上述步骤，合并两次溶液并用正己烷定容至 $10~\mathrm{mL}$ 。得到质量浓度分别为 $0\mathrm{mg / L},0.20\mathrm{mg / L}$ 、 $1.0\mathrm{mg / L},2.0\mathrm{mg / L},4.0\mathrm{mg / L}$ 和 $5.0\mathrm{mg / L}$ 的混合标准工作溶液。临用现配。

# 3.5.6 含油脂食品模拟物标准工作溶液

分别称取 $3.0\mathrm{g}$ （精确至 $0.1\mathrm{g}$ )橄榄油于6支玻璃试管中，准确移取 $0\mathrm{mL},0.015\mathrm{mL},0.030\mathrm{mL}$ 产 $0.060~\mathrm{mL},0.12~\mathrm{mL}$ 和 $0.15\mathrm{mL}$ 的混合标准中间溶液B，混匀后加入 $0.25\mathrm{mL}10\%$ （质量分数)氨水和 $3.00~\mathrm{mL}$ 乙腈，涡旋振荡萃取 $5\mathrm{min}$ ，静置分层，取上层乙腈溶液转移至洁净容量瓶中。加入 $3.00~\mathrm{mL}$ 乙腈如上重复提取一次，合并两次提取液至内含 $900~\mathrm{mg}$ 硫酸镁、 $150~\mathrm{mg}$ PSA、 $150~\mathrm{mg}$ $\mathrm{C_{18}}$ 的 $15~\mathrm{mL}$ 离心管中， $4000~\mathrm{r / min}$ 下离心 $5\mathrm{min}$ 。取 $4.00~\mathrm{mL}$ 上层乙腈溶液氮吹至近干，再用乙腈定容至 $1.00~\mathrm{mL}$ ，得到质量浓度分别为 $0\mathrm{mg / kg},0.50\mathrm{mg / kg},1.0\mathrm{mg / kg},2.0\mathrm{mg / kg},4.0\mathrm{mg / kg}$ 和 $5.0~\mathrm{mg / kg}$ 的混合标准工作溶液。临用现配。

# 3.5.7 化学替代溶剂（ $95\%$ 乙醇和异辛烷）标准工作溶液

分别准确移取 $0\mathrm{mL}, 0.020\mathrm{mL}, 0.10\mathrm{mL}, 0.20\mathrm{mL}, 0.40\mathrm{mL}, 0.50\mathrm{mL}$ 混合标准中间溶液B于6个 $10\mathrm{mL}$ 容量瓶中，用 $95\%$ 乙醇或异辛烷定容至刻度并摇匀，得到质量浓度分别为 $0\mathrm{mg / L}, 0.20\mathrm{mg / L}, 1.0\mathrm{mg / L}, 2.0\mathrm{mg / L}, 4.0\mathrm{mg / L}$ 和 $5.0\mathrm{mg / L}$ 的混合标准工作溶液。临用现配。

# 6 分析结果的表述

# 6.1 食品接触材料及制品中目标分析物迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

食品接触材料及制品，单种目标分析物的特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(1)进行计算：

$$
X _ {1} = \frac {\left(\rho - \rho_ {0}\right) \times V}{S} \times \frac {S _ {1}}{V _ {1}} \tag {1}
$$

式中：

$X_{1}$ ——目标分析物的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——试样浸泡液中单种目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho_0$ ——空白浸泡液中单种目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$V$ ——试样浸泡液体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

S——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$S_{1}$ ——样品实际使用中与食品接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$V_{1}$ ——对于非密封制品，指食品接触材料及制品实际使用时接触固态食品质量或液体食品体积对应的质量，单位为千克 $(\mathrm{kg})$ ；对于密封制品，指食品接触材料及制品实际配套使用容器盛装固态食品质量或液体食品体积对应的质量，单位为千克 $(\mathrm{kg})$ ；各种液态食品按密度为 $1\mathrm{kg} / \mathrm{L}$ 将体积换算为对应的质量。

当实际使用情形下的 $\frac{S_1}{V_1}$ 未知或无法估算时， $\frac{S_1}{V_1}$ 按 $6\mathrm{dm}^2/\mathrm{kg}$ 计，即 $6\mathrm{dm}^2$ 食品接触材料及制品接触 $1\mathrm{kg}$ 食品或食品模拟物。

计算结果至少保留2位有效数字。

# 6.2 密封制品类食品接触材料及制品中目标分析物迁移量的计算（以 $\mathbf{mg}$ /件表示）

当预期用途未知时，密封制品类食品接触材料及制品中单种目标分析物的特定迁移量以 $\mathrm{mg}$ /件表示时，按式(2)计算，此时需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {2} = \frac {\left(\rho - \rho_ {0}\right) \times V}{n} \tag {2}
$$

式中：

$X_{2}$ ——目标分析物的特定迁移量，单位为毫克每件 $(\mathrm{mg}/$ 件）；

$\rho$ ——试样浸泡液中单种目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho_0$ ——空白浸泡液中单种目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$V$ ——试样浸泡液体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

$n$ ——浸泡用密封制品的数量，单位为件。

计算结果至少保留2位有效数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

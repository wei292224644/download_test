# 前言

本标准代替GB8318—2008《食品添加剂生姜（精）油（蒸馏)》。

本标准与GB8318—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 生姜油”。

# 食品安全国家标准

# 食品添加剂 生姜油

# 1 范围

本标准适用于用水蒸气蒸馏法从生姜(Zingiber officinale Rosc.)的根茎中提取的食品添加剂生姜油。

# A.2 操作条件

A.2.1 柱：毛细管柱，长 $60\mathrm{m}$ ，内径 $0.25\mathrm{mm}$   
A.2.2 固定相：聚二甲基硅氧烷。  
A.2.3 膜厚： $0.25\mu \mathrm{m}$   
A.2.4 色谱炉温度： $60^{\circ}\mathrm{C}$ 恒温 $2\mathrm{min}$ ；然后线性程序升温从 $60^{\circ}\mathrm{C} \sim 150^{\circ}\mathrm{C}$ ，速率 $3^{\circ}\mathrm{C} / \mathrm{min}$ ；再线性程序升温从 $150^{\circ}\mathrm{C} \sim 180^{\circ}\mathrm{C}$ ，速率 $1^{\circ}\mathrm{C} / \mathrm{min}$ ；接着线性程序升温从 $180^{\circ}\mathrm{C} \sim 220^{\circ}\mathrm{C}$ ，速率 $5^{\circ}\mathrm{C} / \mathrm{min}$ ；最后在 $220^{\circ}\mathrm{C}$ 恒温 $5\mathrm{min}$ 。  
A.2.5 进样口温度： $250^{\circ}C$   
A.2.6 检测器温度： $250^{\circ}C$ □  
A.2.7 检测器：氢火焰离子化检测器。

A.2.8 载气：氮气。  
A.2.9 载气流速： $1\mathrm{mL / min}$   
A.2.10 进样量： $0.2\mu \mathrm{L}$   
A.2.11 分流比： $100:1$

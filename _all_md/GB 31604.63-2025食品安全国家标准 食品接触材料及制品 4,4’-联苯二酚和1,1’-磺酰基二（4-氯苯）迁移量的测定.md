# 1,1'-磺酰基二(4-氯苯)迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中 $4,4^{\prime}$ -联苯二酚和1， $1^{\prime}$ -磺酰基二(4-氯苯)迁移量的测定方法。

本标准适用于塑料、涂层类食品接触材料及制品中 $4,4^{\prime}$ -联苯二酚和 $1,1^{\prime}$ -磺酰基二(4-氯苯)迁移量的测定。

# 2 原理

食品接触材料及制品进行迁移试验后，水、 $4\%$ （体积分数)乙酸、 $10\%$ （体积分数)乙醇、 $20\%$ （体积分数)乙醇、 $50\%$ （体积分数)乙醇食品模拟物经过滤后直接进样；化学替代溶剂 $95\%$ （体积分数)乙醇、异辛烷氮吹后，用乙腈-水溶液复溶，过滤后进样；玉米油食品模拟物由乙腈萃取后，经固相柱萃取。采用液相色谱法检测，外标法定量。

# 3 试剂和材料

3.1.2 无水乙醇 $\mathrm{(C_2H_6O)}$ 。  
3.1.3 $95\%$ 乙醇 $\mathrm{(C_2H_6O)}$ 。  
3.1.4 异辛烷 $\mathrm{(C_8H_{18})}$   
3.1.5 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。

3.2.2 乙腈-水溶液 $(1 + 1)$ ：将乙腈和水按 $1:1$ 的体积比混合均匀。

# 3.3 标准品

3.3.1 $4,4^{\prime}$ -联苯二酚 $\mathrm{(C_{12}H_{10}O_2}$ ，CAS号：92-88-6)：纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准物质/标准样品。  
3.3.2 $1,1^{\prime}$ -磺酰基二(4-氯苯)( $\mathrm{C_{12}H_8Cl_2O_2S}$ ,CAS号：80-07-9)：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质/标准样品。

# 3.4 标准溶液配制

# 3.4.1 标准储备液（ $1000\mathrm{mg / L}$ ）

准确称取 $4,4^{\prime}$ -联苯二酚和 $1,1^{\prime}$ -磺酰基二(4-氯苯)标准品各 $50~\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ), 分别用甲醇溶解后转移至2个 $50~\mathrm{mL}$ 容量瓶中, 用甲醇定容至刻度, 混匀。将溶液转移至棕色玻璃容器中, 于 $4^{\circ}\mathrm{C}$ 下避光密闭保存, 有效期为6个月。

# 3.4.2 $4,4^{\prime}$ -联苯二酚标准中间液A(50mg/L)

吸取 $5.0\mathrm{mL}4,4^{\prime}$ -联苯二酚的标准储备液 $(1000\mathrm{mg / L})$ 于 $100~\mathrm{mL}$ 容量瓶中，用丙酮定容至刻度，混匀，得到 $4,4^{\prime}$ -联苯二酚标准中间液A，质量浓度为 $50.0\mathrm{mg / L}$ 。将溶液转移至棕色玻璃容器中，于 $4^{\circ}\mathrm{C}$ 下避光密闭保存，有效期为3个月。

# 3.4.3 混合标准中间液 $\mathbf{A}(5\mathrm{mg} / \mathrm{L})$

分别吸取 $0.5\mathrm{mL}4,4^{\prime}$ -联苯二酚和 $1,1^{\prime}$ -磺酰基二(4-氯苯)的标准储备液 $(1000\mathrm{mg / L})$ 于 $100~\mathrm{mL}$ 容量瓶中，用丙酮定容至刻度，混匀，得到混合标准中间液A，两个物质的质量浓度均为 $5.0~\mathrm{mg / L}$ 。将溶液转移至棕色玻璃容器中，于 $4^{\circ}\mathrm{C}$ 下避光密闭保存，有效期为3个月。

# 3.4.4 标准系列工作液

# 3.4.4.1 标准系列工作液（适用于水、 $4\%$ 乙酸、 $10\%$ 乙醇、 $20\%$ 乙醇、 $50\%$ 乙醇食品模拟物）

分别吸取 $50~\mu \mathrm{L},100~\mu \mathrm{L},200~\mu \mathrm{L},400~\mu \mathrm{L},1000~\mu \mathrm{L}$ 混合标准中间液A( $5\mathrm{mg / L})$ 于5个 $10~\mathrm{mL}$ 容量瓶中，再分别吸取 $200~\mu \mathrm{L},400~\mu \mathrm{L},1000~\mu \mathrm{L},2000~\mu \mathrm{L}4,4^{\prime}$ -联苯二酚标准中间液 $\mathrm{A}(50\mathrm{mg / L})$ 于4个 $10~\mathrm{mL}$ 容量瓶中，均用乙腈-水溶液 $(1 + 1)$ 定容至刻度，混匀，得到 $4,4^{\prime}$ -联苯二酚标准系列工作液质量浓度为 $0.025\mathrm{mg / L},0.050\mathrm{mg / L},0.10\mathrm{mg / L},0.20\mathrm{mg / L},0.50\mathrm{mg / L},1.0\mathrm{mg / L},2.0\mathrm{mg / L},5.0\mathrm{mg / L},$ $10.0\mathrm{mg / L},1,1^{\prime}$ -磺酰基二(4-氯苯)标准系列工作液质量浓度为 $0.025\mathrm{mg / L},0.050\mathrm{mg / L},0.10\mathrm{mg / L},$ $0.20\mathrm{mg / L},0.50\mathrm{mg / L}$ 。临用现配。

# 3.4.4.2 标准系列工作液（适用于化学替代溶剂）

分别吸取 $50~\mu \mathrm{L},100~\mu \mathrm{L},200~\mu \mathrm{L},400~\mu \mathrm{L},1000~\mu \mathrm{L}$ 混合标准中间液A( $5\mathrm{mg / L})$ 于5个 $10~\mathrm{mL}$ 容量瓶中，再分别吸取 $200~\mu \mathrm{L},400~\mu \mathrm{L},1000~\mu \mathrm{L},2000~\mu \mathrm{L}4,4^{\prime}$ -联苯二酚标准中间液 $\mathrm{A}(50\mathrm{mg / L})$ 于4个 $10~\mathrm{mL}$ 容量瓶中，均用相应的化学替代溶剂( $95\%$ 乙醇或异辛烷)定容至刻度，混匀，得到 $4,4^{\prime}$ -联苯二酚标准系列工作液质量浓度为 $0.025\mathrm{mg / L},0.050\mathrm{mg / L},0.10\mathrm{mg / L},0.20\mathrm{mg / L},0.50\mathrm{mg / L},1.0\mathrm{mg / L},$ $2.0\mathrm{mg / L},5.0\mathrm{mg / L},10.0\mathrm{mg / L},1,1^{\prime}$ -磺酰基二(4-氯苯)标准系列工作液质量浓度为 $0.025\mathrm{mg / L}$ $0.050\mathrm{mg / L},0.10\mathrm{mg / L},0.20\mathrm{mg / L},0.50\mathrm{mg / L}$ 。临用现配。上机前需按照5.1.2.2与化学替代溶剂浸泡液同步处理。

# 5 分析步骤

# 5.1 试液制备

# 5.1.1 迁移试验

食品接触材料及制品按照GB31604.1和GB5009.156的要求进行迁移试验。迁移试验所得浸泡液如不能立即测试，应置于 $4^{\circ}\mathrm{C}$ 冰箱中避光保存不超过 $72\mathrm{h}$ 。若进行下一步试验，应将浸泡液恢复至室温后使用。

# 5.1.2 浸泡液的处理

# 5.1.2.1 水、4%乙酸、10%乙醇、20%乙醇、50%乙醇食品模拟物试液

# 5.1.2.2 化学替代溶剂试液

# 5.3 标准曲线的制作

按照5.2所列仪器参考条件，将标准工作液分别注入高效液相色谱仪中，测定相应的峰面积。根据试液中 $4,4^{\prime}$ -联苯二酚的浓度选取质量浓度范围 $0.025\mathrm{mg / L}\sim 0.5\mathrm{mg / L}$ 或者 $0.5\mathrm{mg / L}\sim 10.0\mathrm{mg / L}$ 的标准系列工作溶液，以 $4,4^{\prime}$ -联苯二酚的质量浓度为横坐标，以对应色谱峰的峰面积为纵坐标，绘制标准工作曲线，得到 $4,4^{\prime}$ -联苯二酚线性回归方程。以标准系列工作溶液中 $1,1^{\prime}$ -磺酰基二(4-氯苯)的质量浓度为横坐标，以对应色谱峰的峰面积为纵坐标，绘制标准工作曲线，得到 $1,1^{\prime}$ -磺酰基二(4-氯苯)的线性回归方程。

# 6 分析结果的表述

# 6.1 食品接触材料及制品4,4'-联苯二酚和1,1'-磺酰基二(4-氯苯)特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

食品接触材料及制品试样中 $4,4^{\prime}$ -联苯二酚和 $1,1^{\prime}$ -磺酰基二(4-氯苯)特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(1)进行计算：

$$
X _ {1} = \frac {\left(c - c _ {0}\right) \times V}{S} \times F \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——4, $4^{\prime}$ -联苯二酚或1, $1^{\prime}$ -磺酰基二(4-氯苯)的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$

$c$ ——浸泡液中 $4,4^{\prime}$ -联苯二酚或 $1,1^{\prime}$ -磺酰基二(4-氯苯)的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；  
$c_{0}$ ——空白浸泡液中 $4,4^{\prime}$ -联苯二酚或 $1,1^{\prime}$ -磺酰基二(4-氯苯)的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；  
$V$ ——迁移试验中试样浸泡液的体积或质量，单位为升(L)或千克(kg)；  
S——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；  
$F$ ——食品接触材料及制品可预见使用情形下实际的接触面积与食品体积(质量)比(以下简称实际 $S / V$ )，单位为平方分米每千克 $(\mathrm{dm}^2 /\mathrm{kg})$ 。各种液态食品通常密度以 $1\mathrm{kg / L}$ 计将其体积换算为质量来计算 $S / V$ 。当实际 $S / V$ 已知时， $F$ 即为可预见使用情形下的最大 $S / V$ ；当实际 $S / V$ 未知时， $F$ 采用 $6\mathrm{dm}^2 /\mathrm{kg}$ ，即 $6\mathrm{dm}^2$ 食品接触材料及制品与 $1\mathrm{kg}$ 食品接触。

以重复性条件下获得的两次独立测定结果的算术平均值表示，计算结果保留2位有效数字。

# 6.2 密封制品类食品接触材料及制品 $4,4^{\prime}$ -联苯二酚和 $1,1^{\prime}$ -磺酰基二（4-氯苯）特定迁移量的计算（以 $\mathrm{mg}$ /件表示）

密封制品类食品接触材料及制品试样中 $4,4^{\prime}$ -联苯二酚和 $1,1^{\prime}$ -磺酰基二(4-氯苯)特定迁移量以 $\mathrm{mg}/$ 件表示时，按式(2)计算，需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {2} = \frac {\left(c - c _ {0}\right) \times V}{n} \tag {2}
$$

式中：

$X_{2}$ ——4， $4^{\prime}$ -联苯二酚或1， $1^{\prime}$ -磺酰基二(4-氯苯)的特定迁移量，单位为毫克每件 $(\mathrm{mg / }$ 件）；

$c$ ——浸泡液中 $4,4^{\prime}$ -联苯二酚或 $1,1^{\prime}$ -磺酰基二(4-氯苯)的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白浸泡液中 $4,4^{\prime}$ -联苯二酚或 $1,1^{\prime}$ -磺酰基二(4-氯苯)的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$V$ ——迁移试验中试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

$n$ ——浸泡用密封制品的数量，单位为件。

以重复性条件下获得的两次独立测定结果的算术平均值表示，计算结果保留2位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不应超过其算术平均值的 $15\%$ 。

# 11 分析步骤

# 11.1 试液制备

# 11.1.1 迁移试验

食品接触材料及制品按照GB31604.1和GB5009.156的要求进行迁移试验。迁移试验所得浸泡

液如不能立即测试，应置于 $4^{\circ}\mathrm{C}$ 冰箱中避光保存不超过 $72\mathrm{h}$ 。若进行下一步试验，应将浸泡液恢复至室温后使用。

# 11.1.2 浸泡液的处理

# 11.1.2.1 浸泡液的提取

# 11.1.2.1.1 浸泡液的提取（4，4'-联苯二酚）

准确称取 $10\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 迁移试验后得到的玉米油浸泡液于 $50~\mathrm{mL}$ 具塞离心管中，加入 $250~\mu \mathrm{L}$ 丙酮，再加入 $10~\mathrm{mL}$ 乙腈，涡旋振荡 $30~\mathrm{s}, 5000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，立即移取全部上层清液至 $25~\mathrm{mL}$ 具塞刻度管中。再用 $10~\mathrm{mL}$ 乙腈重复提取一次，合并溶液，在溶液中加入 $50~\mu \mathrm{L}$ 二甲基亚砜，混匀，在 $45^{\circ}\mathrm{C}$ 水浴条件下氮吹至 $3\mathrm{mL}$ ，转移至 $10~\mathrm{mL}$ 容量瓶中，用水反复冲洗刻度管，洗液合并于 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度后混匀，备用。

# 11.1.2.1.2 浸泡液的提取[1,1'-磺酰基二(4-氯苯)]

准确称取 $10\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 迁移试验后得到的玉米油浸泡液于 $50~\mathrm{mL}$ 具塞离心管中，加入 $250~\mu \mathrm{L}$ 丙酮，再加入 $10~\mathrm{mL}$ 乙腈，涡旋振荡 $30~\mathrm{s}, 5000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，立即移取全部上层清液至 $25~\mathrm{mL}$ 具塞刻度管中。再用 $10~\mathrm{mL}$ 乙腈重复提取一次，合并溶液，在溶液中加入 $50~\mu \mathrm{L}$ 二甲基亚砜，混匀，在 $45^{\circ}C$ 水浴条件下氮吹至 $5~\mathrm{mL}$ ，转移至 $10~\mathrm{mL}$ 容量瓶中，用水反复冲洗刻度管，洗液合并于 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度后混匀，备用。

# 11.1.2.2 提取液的净化

# 11.1.2.2.1 提取液的净化（4， $4^{\prime}$ -联苯二酚）

将浸泡液的提取溶液 $(4,4^{\prime}$ -联苯二酚)全部转移至预先活化的聚苯乙烯二乙烯苯固相萃取柱，流速小于1滴/s，弃去流出液，抽干固相萃取柱，准确加入 $6\mathrm{mL}$ 乙腈-水溶液 $(3 + 2)$ 洗脱，流速小于1滴/s，收集洗脱液，取上层清液经微孔滤膜过滤后，待测定。

# 11.1.2.2.2 提取液的净化[1,1'-磺酰基二(4-氯苯)]

将浸泡液的提取溶液 $[1,1^{\prime}$ -磺酰基二(4-氯苯)]全部转移至预先活化的聚苯乙烯二乙烯苯固相萃取柱，流速小于1滴/s，弃去流出液，抽干固相萃取柱。用 $9\mathrm{mL}$ 乙腈-水溶液 $(3 + 2)$ 淋洗，流速小于1滴/s，弃去流出液，再用 $6\mathrm{mL}$ 乙腈-水溶液 $(9 + 1)$ 洗脱固相萃取柱，收集洗脱液于 $10~\mathrm{mL}$ 具塞刻度玻璃管中，并准确加入 $50~\mu \mathrm{L}$ 二甲基亚砜，混匀。在 $45^{\circ}C$ 条件下氮吹至近干后，用乙腈-水溶液 $(1 + 1)$ 定容至 $1\mathrm{mL}$ ，混匀，经微孔滤膜过滤后，待测定。

# 11.3 标准曲线的制作

按照11.2所列仪器参考条件，将标准工作液分别注入高效液相色谱仪中，测定相应的峰面积。根据试液中 $4,4^{\prime}$ -联苯二酚的质量分数选取范围 $0.025\mathrm{mg / kg}\sim 0.5\mathrm{mg / kg}$ 或者 $0.5\mathrm{mg / kg}\sim 10.0\mathrm{mg / kg}$ 的标准系列工作溶液，以 $4,4^{\prime}$ -联苯二酚的质量分数为横坐标，以对应色谱峰的峰面积为纵坐标，绘制标准工作曲线，得到 $4,4^{\prime}$ -联苯二酚线性回归方程。以标准系列工作溶液中 $1,1^{\prime}$ -磺酰基二(4-氯苯)的质量分数为横坐标，以对应色谱峰的峰面积为纵坐标，绘制标准工作曲线，得到 $1,1^{\prime}$ -磺酰基二(4-氯苯)的线性回归方程。

# 11.4 试液的测定

# 11.4.1 定性测定

按照11.2所列仪器参考条件，分别测定试液和标准工作溶液，如果试液与标准工作溶液色谱峰保留时间偏差在 $\pm 2.5\%$ 范围内，则可判断试样中存在相应的待测物。4, $4^{\prime}$ -联苯二酚和 $1,1^{\prime}$ -磺酰基二(4-氯苯)标准工作溶液的高效液相色谱图见附录A中图A.4～图A.5。

# 11.4.2 定量测定

将试液(11.1.2)和空白试液(11.1.3)注入高效液相色谱仪中，得到峰面积，根据标准工作曲线得到试液中 $4,4^{\prime}$ -联苯二酚和 $1,1^{\prime}$ -磺酰基二(4-氯苯)的浓度。

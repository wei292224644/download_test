# 前言

本标准代替GB5009.189—2016《食品安全国家标准 食品中米酵菌酸的测定》。

本标准与GB5009.189—2016相比，主要变化如下：

——增加了第二法液相色谱-质谱/质谱法；

——修改了适用范围；

——修改了液相色谱法的结果计算和表述。

# 食品安全国家标准

# 食品中米酵菌酸的测定

# 1 范围

本标准规定了食品中米酵菌酸的测定方法。

本标准第一法适用于银耳及其制品、酵米面及其制品中米酵菌酸的测定。

本标准第二法适用于银耳及其制品、木耳及其制品、谷物及其制品中米酵菌酸的测定。

# 第一法 液相色谱法

# 2 原理

试样中的米酵菌酸经溶剂提取，混合型强阴离子交换柱或液液萃取法净化、浓缩后，高效液相色谱仪分析，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。  
3.1.3 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ： $25\% \sim 28\%$   
3.1.4 甲酸 $\mathrm{(CH_2O_2)}$ 。  
3.1.5 盐酸（HCl）。  
3.1.6 磷酸 $\mathrm{(H_3PO_4)}$ 。  
3.1.7 碳酸氢钠 $\mathrm{(NaHCO_3)}$ 。  
3.1.8 石油醚 $\left(\mathrm{C}_{5} \mathrm{H}_{12} \mathrm{O}_{2}\right)$ ：沸程 $30^{\circ} \mathrm{C} \sim 60^{\circ} \mathrm{C}$ 。  
3.1.9 无水乙醚 $(\mathrm{C_4H_{10}O})$ 。  
3.1.10 三氯甲烷 $\mathrm{(CHCl_3)}$

# 3.2 试剂配制

3.2.1 磷酸溶液 $(45.4\%)$ ：量取 $45.4~\mathrm{mL}$ 磷酸，置于 $100~\mathrm{mL}$ 容量瓶中，用水稀释定容至刻度，摇匀。  
3.2.2 碳酸氢钠溶液 $(40\mathrm{g / L})$ ：称取 $40\mathrm{g}$ 碳酸氢钠加水溶解，转移至 $1000~\mathrm{mL}$ 容量瓶中定容至刻度，摇匀。  
3.2.3 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ：量取 $50~\mathrm{mL}$ 盐酸，置于 $100~\mathrm{mL}$ 容量瓶中，加水稀释定容至刻度，摇匀。  
3.2.4 氨水-甲醇溶液：量取 $80~\mathrm{mL}$ 甲醇，加入 $1.0~\mathrm{mL}$ 氨水，加水定容至 $100~\mathrm{mL}$ ，摇匀。  
3.2.5 甲酸-甲醇溶液 $(2\%)$ ：吸取 $2.0~\mathrm{mL}$ 甲酸，加甲醇至 $100~\mathrm{mL}$ ，摇匀。

3.2.6 甲酸水溶液 $(\mathrm{pH}2.5)$ ：水用甲酸调节 $\mathrm{pH}$ 至 $2.5 \pm 0.1$ ，摇匀。

# 3.3 标准品

米酵菌酸 $\mathrm{(C_{28}H_{38}O_7}$ ,CAS号：11076-19-0）：纯度 $\geqslant 95\%$ ，或经国家认证并授予标准物质证书的标准品。

注：标准物质可以使用满足溯源要求的商品化标准溶液。

# 3.4 标准溶液配制

3.4.1 米酵菌酸标准储备液 $(0.1\mathrm{mg / mL})$ ：准确称取米酵菌酸标准品 $10.0\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ ），用甲醇溶解，转移至 $100~\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度。置于 $-20^{\circ}C$ 冰箱中避光保存，有效期为6个月。  
3.4.2 米酵菌酸标准系列工作液：分别准确吸取米酵菌酸标准储备液用甲醇稀释定容，配制成米酵菌酸质量浓度分别为 $0.1\mu \mathrm{g / mL}, 0.3\mu \mathrm{g / mL}, 0.5\mu \mathrm{g / mL}, 1.0\mu \mathrm{g / mL}, 2.0\mu \mathrm{g / mL}, 5.0\mu \mathrm{g / mL}, 10.0\mu \mathrm{g / mL}$ 和 $20.0\mu \mathrm{g / mL}$ 的标准工作溶液。临用现配。

# 3.5 材料

固相萃取柱：混合型强阴离子交换柱 $(60\mathrm{mg} / 3\mathrm{mL})$ 或等效固相萃取柱，临用前依次加 $5\mathrm{mL}$ 甲醇和 $5\mathrm{mL}$ 水活化，保持柱体湿润。

# 4 仪器和设备

4.1 高效液相色谱仪，配二极管阵列检测器或紫外检测器。  
4.2 天平：感量分别为 $0.01\mathrm{g}$ 和 $0.01\mathrm{mg}$ 。  
4.3 固相萃取装置（带真空泵）。  
4.4 氮吹仪。  
4.5 旋转蒸发仪（带减压装置）。  
4.6 pH计：精度为0.01。  
4.7 粉碎机（带 $\phi 0.425\mathrm{mm}$ 筛）。  
4.8 超声波振荡器： $30\mathrm{kHz}\sim 50\mathrm{kHz}$

# 5 分析步骤

# 5.1 试样制备

酵米面及其制品不少于 $1\mathrm{kg}$ ，银耳及其制品不少于 $600\mathrm{g}$ ，干试样经粉碎过 $\phi 0.425\mathrm{mm}$ 筛；鲜（湿）试样剪碎或切碎，匀浆处理，混合均匀，供检测样本每份不少于 $200\mathrm{g}$ ，贮存于样品瓶或塑封袋中，密封 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 冷藏保存。

# 5.2 固相萃取法

# 5.2.1 试样提取

称取 $20\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样，置于锥形瓶中，加入约 $50~\mathrm{mL}$ 氨水-甲醇溶液，混匀，室温下避光浸泡 $1\mathrm{h}$ ，置于超声波振荡器中超声提取约 $30\mathrm{min}$ 后用中速滤纸过滤，将滤液转移至 $100~\mathrm{mL}$ 容量瓶中，于残渣中加入约 $40~\mathrm{mL}$ 氨水-甲醇溶液，涡旋混匀后置于超声波振荡器中超声提取约 $30\mathrm{min}$ 后用中速滤纸过滤，将滤液转移至同一 $100~\mathrm{mL}$ 容量瓶中，并用氨水-甲醇溶液定容至刻度，混匀。准确量取

$50.0\mathrm{mL}$ 提取液，置于 $80^{\circ}\mathrm{C}$ 水浴中旋转蒸发浓缩至约 $3\mathrm{mL}$ ，待净化。

# 5.2.2 试样净化

将浓缩后的试样全部转移到已活化的固相萃取柱中，依次用 $5\mathrm{mL}$ 水和 $5\mathrm{mL}$ 甲醇淋洗，弃去流出液，再用 $6\mathrm{mL}$ 甲酸-甲醇溶液 $(2\%)$ 洗脱，收集洗脱液，于 $40^{\circ}\mathrm{C}$ 水浴中氮吹至干，准确加入 $0.5\mathrm{mL}$ 甲醇，涡旋溶解，混匀，经 $0.45\mu \mathrm{m}$ 微孔有机滤膜过滤后备用。

# 5.3 液-液萃取法

# 5.3.1 试样提取

称取 $20\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）试样置于锥形瓶中，加入约 $20~\mathrm{mL}$ 甲醇，室温下避光浸泡 $1\textrm{h}$ ，再加入约 $70~\mathrm{mL}$ 三氯甲烷和 $0.2\mathrm{mL}$ 磷酸溶液 $(45.4\%)$ ，振荡 $30\mathrm{min}$ ，过滤，将滤液转移至 $100~\mathrm{mL}$ 容量瓶中，用三氯甲烷定容至刻度，混匀，准确量取滤液 $50.0~\mathrm{mL}$ ，待萃取。

# 5.3.2 试样萃取

将上述滤液移入 $150~\mathrm{mL}$ 分液漏斗中，加入与滤液等体积的碳酸氢钠溶液 $(40~\mathrm{g / L})$ ，振摇 $2\mathrm{min}$ ，静置待分层后，取出下层于另一分液漏斗中。再用碳酸氢钠溶液 $(40~\mathrm{g / L})10~\mathrm{mL}$ 重复提取两次，轻摇。将三次提取的碳酸氢钠溶液合并，加入 $25~\mathrm{mL}$ 三氯甲烷，振摇 $2\mathrm{min}$ ，静置分层后弃去三氯甲烷，于分液漏斗中慢慢加入盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ，调该溶液 $\mathrm{pH}$ 至 $2\sim 3$ ，加入石油醚 $50~\mathrm{mL}$ ，振摇 $3\mathrm{min}$ ，静置分层，取出石油醚层于旋蒸瓶中，再分别用石油醚 $30~\mathrm{mL}$ 和 $20~\mathrm{mL}$ 各提取一次，将石油醚层并入同一瓶中，于 $40^{\circ}C$ 水浴中旋蒸浓缩至干，用少量甲醇分次将旋蒸瓶中提取物溶解并转移至 $5.00~\mathrm{mL}$ 玻璃离心管或浓缩瓶中，于 $40^{\circ}C$ 下氮吹浓缩至干，准确加入 $0.5~\mathrm{mL}$ 甲醇，涡旋溶解，混匀，经 $0.45\mu \mathrm{m}$ 微孔有机滤膜过滤后备用。

# 5.4 仪器参考条件

5.4.1 色谱柱： $\mathrm{C_{18}}$ 色谱柱（柱长 $250~\mathrm{mm}$ ，柱内径 $4.6~\mathrm{mm}$ ，填料粒径 $5\mu \mathrm{m}$ ）或同等性能的色谱柱。  
5.4.2 流动相：甲醇 $+$ 甲酸水溶液 $(\mathrm{pH}2.5) = 75 + 25$   
5.4.3 流速： $1.0~\mathrm{mL / min}$   
5.4.4 柱温： $30^{\circ}C$   
5.4.5 检测波长： $267~\mathrm{nm}$   
5.4.6 进样量： $20~\mu \mathrm{L}$

# 5.5 标准曲线的制作

将 $20~\mu \mathrm{L}$ 标准系列工作液分别注入液相色谱仪中，测定相应的峰面积，以标准系列工作液中米酵菌酸的质量浓度为横坐标，以峰面积的响应值为纵坐标，绘制标准曲线。米酵菌酸标准溶液的色谱图参见附录A中图A.1。

# 5.6 试样溶液的测定

将试样溶液注入液相色谱仪中，得到峰面积，以保留时间定性，根据标准曲线得到待测液中米酵菌酸的质量浓度。

# 6 结果计算和表述

试样中米酵菌酸的含量按式(1)计算。

$$
X = \frac {\rho \times V _ {1} \times V _ {3} \times 1 0 0 0}{m \times V _ {2}} \tag {1}
$$

式中：

$X$ ——试样中米酵菌酸的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中米酵菌酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V_{1}$ ——试样溶液提取液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——样品经净化后的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——单位换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——用于净化量取的试样溶液体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法的检出限为 $5\mu \mathrm{g / kg}$ ，定量限为 $15~\mu \mathrm{g / kg}$

# 第二法 液相色谱-质谱/质谱法

# 9 原理

试样中的米酵菌酸用氨化甲醇提取后，混合型强阴离子交换柱净化，采用液相色谱-串联质谱仪检测，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 10.1 试剂

10.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
10.1.2 甲酸 $(\mathrm{CH}_2\mathrm{O}_2)$ ：色谱纯。  
10.1.3 甲醇 $\mathrm{(CH_3OH)}$ 。  
10.1.4 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ： $25\% \sim 28\%$   
10.1.5 硫酸铵 $\mathrm{(H_8N_2O_4S)}$ 。

# 10.2 试剂配制

10.2.1 氨水-甲醇溶液：量取 $80~\mathrm{mL}$ 甲醇，加入 $1.0~\mathrm{mL}$ 氨水，加水定容到 $100~\mathrm{mL}$ ，混匀。  
10.2.2 甲酸-甲醇溶液 $(2\%)$ ：吸取 $2.0~\mathrm{mL}$ 甲酸，加甲醇至 $100~\mathrm{mL}$ ，混匀。  
10.2.3 $0.1\%$ 甲酸水溶液：吸取甲酸 $1.0~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
10.2.4 乙腈-水溶液 $(1:1)$ ：取乙腈 $50~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。

# 10.3 米酵菌酸标准品

米酵菌酸 $\mathrm{(C_{28}H_{38}O_7}$ ，CAS号：11076-19-0）：纯度 $\geq 95\%$ ，或经国家认证并授予标准物质证书的标准品。

注：标准物质可以使用满足溯源要求的商品化标准溶液。

# 10.4 标准溶液配制

10.4.1 米酵菌酸标准储备液 $(0.1\mathrm{mg / mL})$ : 准确称取米酵菌酸标准品 $10.0\mathrm{mg}$ (精确至 $0.01\mathrm{mg}$ ), 用甲醇溶解, 转移至 $100\mathrm{mL}$ 容量瓶中, 用甲醇定容至刻度。置于 $-20^{\circ}\mathrm{C}$ 冰箱中避光保存, 有效期为 6 个月。  
10.4.2 米酵菌酸标准中间液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取米酵菌酸标准储备溶液 $100~\mu \mathrm{L}$ ，用甲醇稀释并定容至 $10.00~\mathrm{mL}$ 。临用现配。  
10.4.3 米酵菌酸标准系列工作溶液：分别准确吸取米酵菌酸标准中间液用乙腈-水溶液(1:1)稀释定容，配制成米酵菌酸质量浓度分别为 $1.0\mathrm{ng / mL},2.0\mathrm{ng / mL},5.0\mathrm{ng / mL},10.0\mathrm{ng / mL},25.0\mathrm{ng / mL}$ 和 $50.0\mathrm{ng / mL}$ 的标准工作溶液。临用现配。

# 10.5 材料

固相萃取柱：混合型强阴离子交换柱 $(60\mathrm{mg} / 3\mathrm{mL})$ 或等效固相萃取柱，临用前依次用 $5.0~\mathrm{mL}$ 甲醇和 $5.0~\mathrm{mL}$ 水活化，保持柱体湿润。

# 11 仪器和设备

11.1 液相色谱-串联质谱仪：配有电喷雾离子源。  
11.2 天平：感量分别为 $0.01\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。  
11.3 固相萃取装置（带真空泵）。  
11.4 氮吹仪。  
11.5 粉碎机（带 $\phi 0.425\mathrm{mm}$ 筛）。  
11.6 超声波振荡器： $30\mathrm{kHz}\sim 50\mathrm{kHz}$

# 12 分析步骤

# 12.1 试样制备

谷物及其制品不少于 $1\mathrm{kg}$ ，银耳及其制品、木耳及其制品不少于 $600\mathrm{g}$ ，干试样经粉碎过 $\phi 0.425\mathrm{mm}$ 筛；鲜(湿)试样剪碎或切碎，匀浆处理，混合均匀，供检测样本每份不少于 $200\mathrm{g}$ ，贮存于样品瓶或塑封袋中，密封 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 冷藏保存。

# 12.2 试样提取

# 12.2.1 谷物及其制品、银耳及其制品、干木耳

称取 $2.5\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $10~\mathrm{mL}$ 甲醇-氨水溶液（干木耳加入 $20~\mathrm{mL}$ 甲醇-氨水溶液），充分混匀，室温下置于暗处浸泡 $1\textrm{h}$ 后，超声提取 $30\mathrm{min}$ ，离心 $5\mathrm{min}$ ，取上清液至 $25.0~\mathrm{mL}$ 刻度离心管中（干木耳试样至 $50.0~\mathrm{mL}$ 刻度离心管中），残渣加入 $10~\mathrm{mL}$ 氨水-甲醇溶液（干木耳试样加入 $20~\mathrm{mL}$ 氨水-甲醇溶液）重复提取一次，合并提取液，用甲醇-氨水溶液定容至 $25.0~\mathrm{mL}$ （干木耳试样定容至 $50.0~\mathrm{mL}$ ），准确移取 $5.00~\mathrm{mL}$ 上清液（干木耳试样准确移取 $10.0~\mathrm{mL}$ 上清液），待净化。

# 12.2.2 鲜湿木耳

称取 $2.5\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ ) 于 $50~\mathrm{mL}$ 离心管中，加入 $10~\mathrm{mL}$ 甲醇-氨水溶液，充分混匀，室温下置于暗处浸泡 $1\mathrm{h}$ 后加入 $3\mathrm{g}$ 硫酸铵，超声提取 $30\mathrm{min}$ ，离心 $5\mathrm{min}$ ，取上清液至 $25.0~\mathrm{mL}$ 刻度离心管中，残渣加入 $10~\mathrm{mL}$ 甲醇-氨水溶液重复提取一次，合并提取液，用甲醇-氨水溶液定容至 $25.0~\mathrm{mL}$ ，准确移取 $5.0~\mathrm{mL}$ 上清液，待净化。

# 12.3 试样净化

将待净化液转移到经活化平衡的固相萃取柱中，依次用 $5\mathrm{mL}$ 水和 $5\mathrm{mL}$ 甲醇淋洗，弃去流出液，再用 $6\mathrm{mL}$ 甲酸-甲醇溶液 $(2\%)$ 洗脱，收集洗脱液，于 $40^{\circ}\mathrm{C}$ 水浴中氮吹至干，准确加入 $1.0\mathrm{mL}$ 乙腈-水溶液 $(1:1)$ ，涡旋溶解残渣，经 $0.22\mu \mathrm{m}$ 微孔滤膜过滤，供液相色谱-串联质谱分析。

# 12.4 仪器参考条件

# 12.4.1 液相色谱条件

液相色谱条件如下：

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱（柱长 $100\mathrm{mm}$ ，柱内径 $2.1\mathrm{mm}$ ，填料粒径 $1.7\mu \mathrm{m})$ 或同等性能的色谱柱。  
b) 流动相：A为 $0.1\%$ 甲酸水溶液，B为乙腈，梯度洗脱程序见表1。  
c) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ 。  
d） 柱温： $40^{\circ}C$ 。  
e）进样量： $5\mu \mathrm{L}$

# 12.6 定量测定

# 12.6.1 工作曲线的制作

将米酵菌酸标准系列工作溶液分别注入液相色谱-质谱/质谱仪中，测定相应的质量色谱峰面积，以标准系列工作溶液的质量浓度为横坐标，以质量色谱峰峰面积的响应值为纵坐标，绘制工作曲线。米酵菌酸标准溶液的质量色谱图参见附录B中图B.1。

# 12.6.2 试样溶液的测定

将试样溶液按仪器参考条件进行测定，得到相应的试样溶液的质量色谱峰面积。根据工作曲线得到试样溶液中米酵菌酸的质量浓度。

# 13 结果计算和表述

试样中米酵菌酸含量按式(2)计算。

$$
X = \frac {\rho \times V _ {1} \times V _ {3} \times 1 0 0 0}{m \times V _ {2} \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中米酵菌酸的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——由工作曲线得出的试样溶液中米酵菌酸的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V_{1}$ ——试样溶液提取液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——样品经净化洗脱后的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的称样量，单位为克(g)；

$V_{2}$ ——用于净化移取的试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——单位换算系数。

计算结果保留3位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 15 其他

本方法的检出限为 $1\mu \mathrm{g / kg}$ ，定量限为 $3\mu \mathrm{g / kg}$

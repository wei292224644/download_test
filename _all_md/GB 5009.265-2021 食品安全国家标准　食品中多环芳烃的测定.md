2021-09-07 发布

# 前言

本标准代替GB5009.265—2016《食品安全国家标准 食品中多环芳烃的测定》。

本标准与GB5009.265—2016相比，主要变化如下：

将原第二法气相色谱-质谱法修订为第一法，修改了测定项目、样品前处理、气相色谱质谱条件、计算方法等；

——将原第一法高效液相色谱法修订为第二法，修改了序号，简化了原理，统一了试样制备。

# 食品安全国家标准

# 食品中多环芳烃的测定

# 1 范围

本标准第一法规定了食品中16种多环芳烃的气相色谱-质谱测定方法；第二法规定了食品中15种多环芳烃的液相色谱测定方法。

本标准第一法适用于粮食及其制品、肉及其制品、水产及其制品、动植物油脂及其制品中16种多环芳烃(苯并 $[c]$ 芪、苯并 $[a]$ 葱、环戊并 $[c,d]$ 茴、茴、5-甲基茴、苯并 $[b]$ 荧葱、苯并 $[k]$ 荧葱、苯并 $[j]$ 荧葱、苯并 $[a]$ 芘、茚并 $[1,2,3-c,d]$ 芘、二苯并 $[a,h]$ 葱、苯并 $[g,h,i]$ 芘、二苯并 $[a,l]$ 芘、二苯并 $[a,e]$ 芘、二苯并 $[a,i]$ 芘、二苯并 $[a,h]$ 芘)的测定；适用于婴幼儿配方乳粉、婴幼儿辅助食品中4种多环芳烃（苯并 $[a]$ 葱、茴、苯并 $[b]$ 荧葱、苯并 $[a]$ 芘)的测定。

本标准第二法适用于粮食及其制品、肉及其制品、水产及其制品、蔬菜、动植物油脂中15种多环芳烃(萘、苊、芴、菲、蒽、荧蒽、芘、苯并[a]蒽、萜、苯并[b]荧蒽、苯并[k]荧蒽、苯并[a]芘、茚并[1,2,3-c,d]芘、二苯并[a,h]蒽和苯并[g,h,i]芘)的测定。

# 第一法 气相色谱-质谱法

# 2 原理

试样中多环芳烃经溶剂提取，氢氧化钾乙醇溶液皂化，固相萃取柱净化，浓缩后用气相色谱-质谱联用仪测定，同位素内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。  
3.1.2 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：色谱纯。  
3.1.3 丙酮 $\mathrm{(C_3H_6O)}$ ：色谱纯。  
3.1.4 环己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{12}\right)$ ：色谱纯。  
3.1.5 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
3.1.6 异辛烷 $\left(\mathrm{C}_{8} \mathrm{H}_{18}\right)$ ：色谱纯。  
3.1.7 无水乙醇 $\mathrm{(C_2H_6O)}$ ：色谱纯。  
3.1.8 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ：质量分数约 $25\%$   
3.1.9 无水乙醚 $(\mathrm{C_4H_{10}O})$ 。  
3.1.10 石油醚 $\left(\mathrm{C}_{5} \mathrm{H}_{12} \mathrm{O}_{2}\right)$ ：沸程为 $30^{\circ} \mathrm{C} \sim 60^{\circ} \mathrm{C}$ 。

3.1.11 氢氧化钾(KOH)。  
3.1.12 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：使用前 $400^{\circ}C$ 下烘烤 $2\mathrm{h}$ 后放干燥器内待用。

# 3.2 试剂配制

3.2.1 氢氧化钾乙醇溶液 $(0.3\mathrm{mol} / \mathrm{L})$ ：称取 $1.68\mathrm{g}$ 氢氧化钾，用 $100~\mathrm{mL}$ 无水乙醇超声溶解，临用现配。  
3.2.2 氢氧化钾乙醇溶液 $(1.5\mathrm{mol} / \mathrm{L})$ ：称取 $8.40\mathrm{g}$ 氢氧化钾，用 $100~\mathrm{mL}$ 无水乙醇超声溶解，临用现配。  
3.2.3 环己烷-乙酸乙酯溶液 $(1 + 1)$ ：将环己烷和乙酸乙酯按 $1:1$ （体积比）混合均匀。  
3.2.4 二氯甲烷-乙酸乙酯溶液 $(1 + 1)$ ：将二氯甲烷和乙酸乙酯按 $1:1$ （体积比）混合均匀。  
3.2.5 丙酮-异辛烷溶液 $(1 + 1)$ ：将丙酮和异辛烷按 $1:1$ （体积比）混合均匀。

# 3.3 标准溶液

# 3.3.1 多环芳烃标准溶液

多环芳烃标准溶液（含苯并 $[c]$ 芬、苯并 $[a]$ 葱、环戊并 $[c,d]$ 芘、茴、5-甲基茴、苯并 $[b]$ 荧蒽、苯并 $[k]$ 荧蒽、苯并 $[j]$ 荧蒽、苯并 $[a]$ 芘、茚并 $[1,2,3-c,d]$ 芘、二苯并 $[a,h]$ 葱、苯并 $[g,h,i]$ 芘、二苯并 $[a,l]$ 芘、二苯并 $[a,e]$ 芘、二苯并 $[a,i]$ 芘、二苯并 $[a,h]$ 芘， $10.0\mu\mathrm{g/mL}$ 或相应浓度）：经国家认证并授予标准物质证书的标准溶液， $-18^{\circ}\mathrm{C}$ 下避光保存。多环芳烃CAS号见附录A。

# 3.3.2 同位素多环芳烃内标溶液

同位素多环芳烃内标溶液（至少含 $\mathrm{D}_{12}$ -苯并 $[a]$ 葱、 $\mathrm{D}_{12}$ -萜、 $\mathrm{D}_{12}$ -苯并 $[b]$ 荠葱、 $\mathrm{D}_{12}$ -苯并 $[a]$ 芘、 $\mathrm{D}_{12}$ -茚并 $[1,2,3-c,d]$ 芘、 $\mathrm{D}_{14}$ -二苯并 $[a,h]$ 葱、 $\mathrm{D}_{12}$ -苯并 $[g,h,i]$ 芘， $100.0\mu \mathrm{g / mL}$ 或相应浓度），一 $18^{\circ}C$ 下避光保存。同位素多环芳烃内标CAS号见附录A。

警告——多环芳烃是已知的致癌、致畸、致突变的物质，并且致癌性随着苯环数的增加而增加。进行标准溶液配制等操作时应特别注意安全防护，应在通风柜中操作，尽量减少暴露。

# 3.4 标准使用液配制

3.4.1 多环芳烃标准使用液 $(500.0\mathrm{ng / mL})$ ：吸取多环芳烃标准溶液 $(10.0~\mu \mathrm{g / mL})0.50~\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用丙酮-异辛烷溶液 $(1 + 1)$ 定容至刻度，混匀，将溶液转移至棕色玻璃存储瓶中， $4^{\circ}C$ 下避光密封保存，保存期3个月。  
3.4.2 多环芳烃标准使用液 $(100.0\mathrm{ng / mL})$ ：吸取多环芳烃标准使用液 $(500.0\mathrm{ng / mL})2.0\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用丙酮-异辛烷溶液 $(1 + 1)$ 定容至刻度，混匀，将溶液转移至棕色玻璃存储瓶中， $4^{\circ}C$ 下避光密封保存，保存期3个月。  
3.4.3 同位素多环芳烃内标使用液 $(200.0\mathrm{ng / mL})$ ：吸取同位素多环芳烃内标溶液 $(100.0\mu \mathrm{g / mL})$ $0.10~\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，用丙酮-异辛烷溶液(1+1)定容至刻度，混匀，将溶液转移至棕色玻璃存储瓶中， $4^{\circ}C$ 下避光密封保存，保存期1年。  
3.4.4 多环芳烃标准系列工作液：取8个锥底进样瓶，分别吸取多环芳烃标准使用液 $(100.0\mathrm{ng / mL})$ $20.0~\mu \mathrm{L},40.0~\mu \mathrm{L},50.0~\mu \mathrm{L},80.0~\mu \mathrm{L}$ 和多环芳烃标准使用液 $(500.0\mathrm{ng / mL})20.0~\mu \mathrm{L},40.0~\mu \mathrm{L},50.0~\mu \mathrm{L}$ 、 $100.0~\mu \mathrm{L}$ ，再各加同位素多环芳烃内标使用液 $(200.0\mathrm{ng / mL})100.0~\mu \mathrm{L}$ ，用丙酮-异辛烷溶液 $(1 + 1)$ 补充体积至 $200~\mu \mathrm{L}$ ，多环芳烃标准系列工作液浓度为 $10.0\mathrm{ng / mL},20.0\mathrm{ng / mL},25.0\mathrm{ng / mL},40.0\mathrm{ng / mL}$ $50.0~\mathrm{ng / mL},100.0~\mathrm{ng / mL},125.0~\mathrm{ng / mL},250.0~\mathrm{ng / mL}$ ，内标为 $100.0~\mathrm{ng / mL}$ ，临用现配。

# 5.1 样品前处理

# 5.1.1 试样制备

# 5.1.1.1 粮食及其制品

取样品 $200\mathrm{g}\sim 500\mathrm{g}$ ，用高速粉碎机将其充分粉碎，混合均匀后分出 $100\mathrm{g}$ 左右，装于洁净样品袋或瓶中，密封标识后 $-18^{\circ}C$ 下冷冻保存，供检测用。

# 5.1.1.2 肉及其制品、水产及其制品、蔬菜

取样品 $200\mathrm{g}\sim 500\mathrm{g}$ ，将其可食部分先切碎，经均质器充分搅碎，取 $100\mathrm{g}$ 左右装于洁净样品袋或瓶中，密封标识后 $-18^{\circ}C$ 下冷冻保存，供检测用。

# 5.1.1.3 动植物油脂及制品

取散装动植物油脂及制品 $100\mathrm{g}\sim 200\mathrm{g}$ ，装于洁净样品瓶中，密封标识后放阴凉处保存，供检测用。定型包装动植物油脂贴标签放阴凉处保存，供检测用。

# 5.1.1.4 婴幼儿配方乳粉、婴幼儿辅助食品

取需粉碎样品 $200\mathrm{g}\sim 500\mathrm{g}$ ，用高速粉碎机将其充分粉碎，混合均匀后分出 $100\mathrm{g}$ 左右，装于洁净样品袋或瓶中，密封标识后 $4^{\circ}C$ 下冷藏保存，供检测用。

不需粉碎的婴幼儿配方乳粉、婴幼儿辅助食品贴标签放阴凉处保存，供检测用。

# 5.1.2 试样提取

# 5.1.2.1 粮食及其制品

称取试样 $5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 具塞离心管中，加入 $100\mu \mathrm{L}$ 同位素多环芳烃内标使用液 $(200.0\mathrm{ng / mL})$ 振荡混合后静置 $30\mathrm{min}$ ，加入 $5\mathrm{g}$ 无水硫酸钠、 $20~\mathrm{mL}$ 环己烷-乙酸乙酯混合溶液（1+1)，涡旋提取 $1\mathrm{min}$ ，超声提取 $15\mathrm{min}$ ，以 $10000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，吸取上清液至 $15~\mathrm{mL}$ 预称重的离心

管中(精确至 $0.01 \mathrm{~g}$ ), $45^{\circ} \mathrm{C}$ 水浴下氮气吹干溶剂得提取物, 称重, 计算提取物质量 (精确至 $0.01 \mathrm{~g}$ ), 提取物待皂化。

若提取物质量大于 $1\mathrm{g}$ ，则称出 $1\mathrm{g}$ 提取物（精确至 $0.01\mathrm{g}$ ），置于另一 $15\mathrm{mL}$ 具塞离心管中待皂化。

# 5.1.2.2 肉及其制品、水产及其制品

称取试样 $5\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ), 置于 $50~\mathrm{mL}$ 具塞离心管中, 加入 $100~\mu \mathrm{L}$ 同位素多环芳烃内标使用液 $(200.0\mathrm{ng / mL})$ 振荡混合后静置 $30\mathrm{min}$ , 加入 $20\mathrm{g}$ 无水硫酸钠, 用金属角匙搅拌分散, 加入 $20~\mathrm{mL}$ 环己烷-乙酸乙酯混合溶液 $(1 + 1)$ , 涡旋提取 $1\mathrm{min}$ , 进一步超声提取 $15\mathrm{min}, 10000\mathrm{r / min}$ 离心 $3\mathrm{min}$ , 吸取上清液至 $15~\mathrm{mL}$ 预称重的离心管中, $45^{\circ}\mathrm{C}$ 水浴下用氮气吹干溶剂得提取物, 称重, 计算提取物质量 (精确至 $0.01\mathrm{g}$ ), 提取物待皂化。

若提取物质量大于 $1\mathrm{g}$ ，则称出 $1\mathrm{g}$ 提取物（精确至 $0.01\mathrm{g}$ ），置于另一 $15\mathrm{mL}$ 具塞离心管中待皂化。

# 5.1.2.3 动植物油脂及制品

称取试样 $0.5\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，置于 $15~\mathrm{mL}$ 具塞离心管中，加入 $100~\mu \mathrm{L}$ 同位素多环芳烃内标使用液（ $200.0\mathrm{ng / mL})$ ，待皂化。

# 5.1.2.4 婴幼儿配方乳粉

称取试样 $2\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ), 置于 $50\mathrm{mL}$ 具塞离心管中, 加入 $100\mu \mathrm{L}$ 同位素多环芳烃内标使用液 $(200.0\mathrm{ng / mL})$ , 加 $10\mathrm{mL}$ 水, 涡旋溶解, 加 $2\mathrm{mL}$ 氨水充分混匀, 在 $65^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 水浴中放置 $10\mathrm{min}$ , 取出冷却至室温。加入 $10\mathrm{mL}$ 无水乙醇, 缓慢混匀, 加入 $8\mathrm{mL}$ 无水乙醚, 涡涡振荡 $5\mathrm{min}$ , 再加入 $8\mathrm{mL}$ 石油醚, 涡涡振荡 $5\mathrm{min}$ , $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ , 吸出上层有机相至 $15\mathrm{mL}$ 预称重的离心管中, $45^{\circ}\mathrm{C}$ 水浴下用氮气吹干溶剂得提取物, 称重, 计算提取物质量 (精确至 $0.01\mathrm{g}$ ), 提取物待皂化。

# 5.1.2.5 婴幼儿辅助食品

称取 $2\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 试样，置于 $50~\mathrm{mL}$ 具塞离心管中，加入 $100\mu \mathrm{L}$ 同位素多环芳烃内标使用液 $(200.0\mathrm{ng / mL})$ ，加 $10~\mathrm{mL}$ 水、 $10~\mathrm{mL}$ 无水乙醇，涡旋溶解，加入 $8\mathrm{mL}$ 无水乙醚，漩涡振荡 $5\mathrm{min}$ ，再加入 $8\mathrm{mL}$ 石油醚，漩涡振荡 $5\mathrm{min}$ ，以 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，吸出上层有机相至 $15~\mathrm{mL}$ 预称重的离心管中， $45^{\circ}\mathrm{C}$ 水浴中用氮气吹干溶剂得提取物，称重，计算提取物质量(精确至 $0.01\mathrm{g}$ )，提取物待皂化。

# 5.1.3 皂化

5.1.3.1 提取物质量小于或等于 $0.2\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ), 加 $5\mathrm{mL}0.3\mathrm{mol/L}$ 氢氧化钾乙醇溶液, 涡旋混匀, 室温放置 $5\mathrm{min}$ , 加 $4\mathrm{mL}$ 水、 $5\mathrm{mL}$ 正己烷, 涡旋提取 $2\mathrm{min}$ , $10000\mathrm{r/min}$ 离心 $2\mathrm{min}$ , 上层正己烷提取液待净化。  
5.1.3.2 提取物质量在 $0.2\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 之间，加入 $5\mathrm{mL}1.5\mathrm{mol / L}$ 氢氧化钾乙醇溶液，加盖涡旋混匀，放置 $70~^\circ \mathrm{C}\pm 2~^\circ \mathrm{C}$ 水浴中进行皂化 $3\mathrm{min}$ 。皂化完成取出用自来水冷却至室温，加 $4\mathrm{mL}$ 水、 $5\mathrm{mL}$ 正己烷，涡旋提取 $2\mathrm{min}$ ，以 $10000\mathrm{r / min}$ 离心 $2\mathrm{min}$ ，上层正己烷提取液待净化。  
5.1.3.3 动植物油试样中加入 $5 \mathrm{~mL} 1.5 \mathrm{~mol} / \mathrm{L}$ 氢氧化钾乙醇溶液（固体油脂试样先放置 $50^{\circ} \mathrm{C}$ 水浴融化后加入），加盖涡旋混匀，放置 $70^{\circ} \mathrm{C} \pm 2^{\circ} \mathrm{C}$ 水浴中进行皂化 $3 \mathrm{~min}$ 。皂化完成取出用自来水冷却至室温，加 $4 \mathrm{~mL}$ 水、 $5 \mathrm{~mL}$ 正己烷，涡旋提取 $2 \mathrm{~min}, 10000 \mathrm{r} / \mathrm{min}$ 离心 $2 \mathrm{~min}$ ，上层正己烷提取液待净化。  
注：苯并 $[c]$ 芬对碱不稳定，测定苯并 $[c]$ 芬时，皂化时间视油脂质量而定，以 $1\mathrm{g}$ 提取物皂化 $3\mathrm{min}$ 按比例折算。

# 5.1.4 净化

在固相萃取小柱上加 $1\mathrm{g}$ 左右无水硫酸钠，依次用 $3\mathrm{mL}$ 二氯甲烷、 $3\mathrm{mL}$ 正己烷淋洗活化柱子，活

化结束后吸取全部正己烷提取液转移到固相萃取柱上，流速控制在 $1.0~\mathrm{mL / min}$ 左右，待提取液全部通过填料层后，先用 $5\mathrm{mL}$ 正己烷淋洗除杂，再用 $5\mathrm{mL}$ 二氯甲烷-乙酸乙酯溶液 $(1 + 1)$ 洗脱，收集洗脱液于 $10~\mathrm{mL}$ 锥底玻璃试管内。洗脱液在 $40^{\circ}C$ 水浴中用氮气吹干，加 $0.1\mathrm{mL}$ 丙酮-异辛烷溶液 $(1 + 1)$ 涡旋溶解残留物，并转移至锥底进样瓶中，供GC-MS测定。

# 5.1.5 空白试验

除不加试样外，采用与试样完全相同的分析步骤进行。

# 5.1.6 平行试验

按上述步骤，对同一试样进行平行试验测定。

# 6 结果计算和表述

试样中多环芳烃含量 $X_{i}$ 按式(1)计算。

$$
X _ {i} = \frac {\left(N _ {i} - N _ {i 0}\right) \times 1 0 0 0}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X_{i}$ ——试样中多环芳烃 $i$ 的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$N_{i}$ ——试样溶液中多环芳烃 $i$ 的峰面积与对应内标色谱峰的峰面积比值对应的质量，单位为纳克 $(\mathrm{ng})$ ， $N_{i}$ 按式(2)计算：

$$
N _ {i} = K _ {i} \times N _ {i s} \quad \dots \tag {2}
$$

式中：

$K_{i}$ ——从标准曲线查得的质量比；

$N_{is}$ ——样品中加入的内标物的量，单位为纳克 $(\mathrm{ng})$ ；

$N_{i0}$ ——空白试验溶液中多环芳烃 $i$ 的峰面积与对应内标色谱峰的峰面积比值对应的质量，单位为纳克 $(\mathrm{ng})$ ， $N_{i0}$ 按式(3)计算：

$$
N _ {i 0} = K _ {i 0} \times N _ {0 i s} \quad \dots \tag {3}
$$

式中：

$K_{i0}$ ——从标准曲线查得的质量比；

$N_{0i\mathrm{s}}$ ——空白试验溶液中加入的内标物的量，单位为纳克 $(\mathrm{ng})$ ；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

以重复条件下获得的两次独立测定结果的算术平均值表示，计算结果 $\geqslant 10.0\mu \mathrm{g / kg}$ 时，保留三位有效数字； $<  10.0~\mu \mathrm{g / kg}$ 时，保留两位有效数字。

# 10 原理

试样中的多环芳烃用有机溶剂提取，用PSA(N-丙基乙二胺)和 $\mathrm{C_{18}}$ 固相萃取填料净化或用弗罗里硅土固相萃取柱净化，用高效液相色谱分离，测定各种多环芳烃在不同激发波长和发射波长处的荧光强度，外标法定量。

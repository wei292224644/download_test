# 1 范围

本文件规定了猪和家禽可食性组织中维吉尼亚霉素 $\mathrm{M}_{1}$ 残留检测的制样和液相色谱-串联质谱测定方法。

方法。本文件适用于猪和家禽（包括鸡、鸭和鹅）的肌肉、肝脏、肾脏及皮+脂中维吉尼亚霉素 $\mathbf{M}_{1}$ 残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T 6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的维吉尼亚霉素 $\mathrm{M}_{1}$ , 用乙腈提取, 正已烷除脂, 液相色谱串联质谱检测, 外标法定量。

# 5 试剂和材料

以下所用的试剂，除特别注明者外均为分析纯试剂；水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.2 甲酸(HCOOH)：色谱纯。

5.1.3 乙腈 $\mathrm{(CH_3CN)}$

5.1.4 正己烷 $(\mathrm{C_6H_{14}})$

# 5.2 溶液配制

5.2.1 乙腈溶液：取乙腈 $600~\mathrm{mL}$ 、水 $400~\mathrm{mL}$ ，混匀。

5.2.2 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀。

# 5.3 标准品

维吉尼亚霉素 $\mathrm{M}_{1}$ (Virginiamycin $\mathrm{M}_{1},\mathrm{C}_{28}\mathrm{H}_{35}\mathrm{N}_{3}\mathrm{O}_{7}$ ,CAS号：21411-53-0)含量 $\geq 90\%$ 。

# 5.4 标准溶液制备

5.4.1 标准储备液：取维吉尼亚霉素 $\mathrm{M}_{1}$ 标准品 $10\mathrm{mg}$ ，精密称定，用乙腈适量使溶解并稀释定容至 $200\mathrm{mL}$ 棕色容量瓶，配制成浓度为 $50~\mu \mathrm{g / mL}$ 的维吉尼亚霉素 $\mathrm{M}_{1}$ 标准储备液。 $4^{\circ}\mathrm{C}$ 以下保存，有效期 $7\mathrm{d}$ 。

5.4.2 标准中间液：准确量取标准储备液 $0.1\mathrm{mL}, 0.5\mathrm{mL}, 1\mathrm{mL}, 2\mathrm{mL}$ 和 $3\mathrm{mL}$ ，分别置 $50\mathrm{mL}$ 棕色容量瓶中，用乙腈稀释至刻度，配制成浓度为 $0.1\mu \mathrm{g} / \mathrm{mL}, 0.5\mu \mathrm{g} / \mathrm{mL}, 1\mu \mathrm{g} / \mathrm{mL}, 2\mu \mathrm{g} / \mathrm{mL}$ 和 $3\mu \mathrm{g} / \mathrm{mL}$ 的维吉尼亚霉素 $\mathbf{M}_{1}$ 标准中间液。 $4^{\circ}\mathrm{C}$ 以下保存，有效期 $7\mathrm{d}$ 。

5.5 材料  
5.5.1 刻度试管： $10~\mathrm{mL}$   
5.5.2 有机滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾电离源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 均质机。  
6.4 超声仪。  
6.5 离心机： $\geqslant 4000\mathrm{r / min}$   
6.6 涡旋振荡器。  
6.7 移液器。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试组织，绞碎并均质。

a）取均质的供试样品，作为供试试样；  
b）取均质的空白样品，作为空白试样；  
c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

$-40^{\circ} \mathrm{C}$ 以下保存，避免反复冻融。

# 8 测定步骤

# 8.1 提取

取试料 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，加乙腈 $4\mathrm{mL}$ （肌肉组织先加水 $2\mathrm{mL}$ ，涡旋 $2\mathrm{min}$ ，超声 $30\mathrm{min}$ $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液转移至 $15\mathrm{mL}$ 具塞离心管中。残渣加乙腈 $2\mathrm{mL}$ 重复提取1遍，合并上清液，备用，得到提取液。

# 8.2 净化

取上述提取液，加水至约 $9.5\mathrm{mL}$ ，涡旋混合 $30~\mathrm{s}$ 加正己烷 $3\mathrm{mL}$ ，涡旋 $30~\mathrm{s},4000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ 弃去上层正己烷，重复除脂1次。将下层溶液转移至 $10~\mathrm{mL}$ 刻度试管中，加水至 $10~\mathrm{mL}$ ，涡旋混合 $30~\mathrm{s}$ ，备用。取备用液（肝脏：取备用液 $2.0~\mathrm{mL}$ ，加乙腈溶液 $4.0~\mathrm{mL}$ ，混匀；肾脏、皮脂：取备用液 $2.0~\mathrm{mL}$ ，加乙腈溶液 $6.0~\mathrm{mL}$ ，混匀），过滤，供液相色谱-串联质谱法测定。

# 8.3 标准曲线的制备

# 8.3.1 家禽肌肉、肝脏基质匹配标准曲线的制备

准确量取浓度为 $0.1\mu \mathrm{g / mL},0.5\mu \mathrm{g / mL},1\mu \mathrm{g / mL},2\mu \mathrm{g / mL}$ 和 $3\mu \mathrm{g / mL}$ 的维吉尼亚霉素 $\mathbf{M}_1$ 标准中间液各 $200~\mu \mathrm{L}$ ，分别加于按7.1和7.2正己烷除脂处理后的空白组织提取液中，加水至 $10~\mathrm{mL}$ ，混匀，同法处理。制备成维吉尼亚霉素 $\mathbf{M}_1$ 浓度为 $2\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL},40\mathrm{ng / mL}$ 和 $60~\mathrm{ng / mL}$ 的系列基质匹配标准溶液，过滤，临用现配，供液相色谱联用质谱仪测定。以测得特征离子峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.3.2 其他组织标准曲线的制备

分别准确量取浓度为 $0.1\mu \mathrm{g / mL},0.5\mu \mathrm{g / mL},1\mu \mathrm{g / mL},2\mu \mathrm{g / mL}$ 和 $3\mu \mathrm{g / mL}$ 的维吉尼亚霉素 $\mathbf{M}_1$ 标准中间液各 $200~\mu \mathrm{L}$ ，加乙腈溶液至 $10.0~\mathrm{mL}$ ，配制成维吉尼亚霉素 $\mathbf{M}_1$ 浓度为 $2\mathrm{ng / mL},10\mathrm{ng / mL}$ 、 $20\mathrm{ng / mL},40\mathrm{ng / mL}$ 和 $60\mathrm{ng / mL}$ 系列标准工作液，临用现配，供液相色谱联用质谱仪测定。以测得特

征离子峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.5 测定法

# 8.5.1 定性测定

取试料溶液和相应的基质匹配标准工作液/标准工作液，作单点或多点校准，按外标法以峰面积定量，基质匹配标准工作液/标准工作液及试料溶液中的维吉尼亚霉素 $\mathrm{M}_{1}$ 响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，维吉尼亚霉素 $\mathrm{M}_{1}$ 基质匹配标准溶液的特征离子质量色谱图见附录A。

# 8.6 空白试验

取空白试料，除不加药物外，采用完全相同的步骤进行平行操作。

# 9 结果计算

试样中维吉尼亚霉素 $\mathrm{M}_{1}$ 残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times C _ {\mathrm {S}} \times V}{A _ {\mathrm {S}} \times m} \times f \tag {1}
$$

式中：

$X$ ——试样中维吉尼亚霉素 $\mathrm{M}_{5}$ 残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ ——标准工作液中维吉尼亚霉素 $\mathrm{M}_{1}$ 浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$A$ ——试样溶液中维吉尼亚霉素 $\mathrm{M}_{1}$ 峰面积；

$V$ ——定容后溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{\mathrm{S}}$ ——标准工作液中维吉尼亚毒系 $\mathrm{M}_{1}$ 峰面积；

$f$ ——稀释倍数

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法维吉尼亚霉素 $\mathrm{M}_{1}$ 在肌肉、肝脏、肾脏和皮脂的检测限分别为 $2\mu \mathrm{g / kg},6\mu \mathrm{g / kg},8\mu \mathrm{g / kg}$ 和 $8\mu \mathrm{g / kg}$ ，肌肉的定量限为 $10~\mu \mathrm{g / kg}$ ，其他组织定量限为 $50~\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法维吉尼亚霉素 $\mathrm{M}_{1}$ 在家禽和猪肌肉组织 $10\mu \mathrm{g / kg}\sim 200\mu \mathrm{g / kg}$ 、肝脏和肾脏组织 $50~\mu \mathrm{g / kg}\sim$ $600~\mu \mathrm{g / kg}$ 、皮脂组织 $50~\mu \mathrm{g / kg}\sim 800~\mu \mathrm{g / kg}$ 测定浓度水平上的回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $20\%$ ，批间相对标准偏差 $20\%$

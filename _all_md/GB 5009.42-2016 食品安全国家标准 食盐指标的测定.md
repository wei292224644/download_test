# 前言

本标准代替GB/T5009.42—2003《食盐卫生标准的分析方法》。

本标准与GB/T5009.42—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食盐指标的测定”；  
——删除了感官检查、水不溶物、氟、锌、铜、亚硝酸盐等的分析方法；  
——修改了氯化钠、氯离子、镁、硫酸根、铅、钡、碘等的分析方法；  
——增加了钙、氯化钾的分析方法；  
——明确了食盐中总砷、镉、总汞的分析方法。

# 食品安全国家标准

# 食盐指标的测定

# 1 范围

本标准规定了食盐中氯化钠、铅、总砷、镉、总汞、钡、氯化钾、亚铁氰化钾、碘的检测方法。  
本标准适用于食盐指标的测定。

# 2 氯化钠的测定

# 2.1 水分的测定

按GB5009.3中直接干燥法操作，将干燥温度设定为 $140~\mathrm{^\circ C}\pm 2~\mathrm{^\circ C}$ 。干燥至前后两次质量差不超过 $5\mathrm{mg}$ ，即为恒重。

# 2.2 氯离子的测定

# 2.2.1 原理

样品溶解后，用铬酸钾作指示剂，用硝酸银标准滴定溶液滴定，测定氯离子的含量。

# 2.2.2 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 2.2.2.1 试剂

2.2.2.1.1 硝酸银。

2.2.2.1.2 铬酸钾。

# 2.2.2.2 试剂的配制

2.2.2.2.1 硝酸银标准滴定溶液 $(0.1\mathrm{mol} / \mathrm{L})$ 。

2.2.2.2.2 铬酸钾指示剂：称取 $10\mathrm{g}$ 铬酸钾溶于 $100~\mathrm{mL}$ 水中，搅拌下滴加硝酸银溶液至出现红棕色沉淀，过滤。

# 2.2.3 仪器和设备

2.2.3.1 天平：感量 $0.1\mathrm{mg}$

2.2.3.2 水浴锅。

# 2.2.4 分析步骤

# 2.2.4.1 试样处理

称取 $25\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 粉碎的试样于 $400\mathrm{mL}$ 烧杯中，加约 $200\mathrm{mL}$ 的水，加热，用玻璃棒搅拌至全部溶解。冷却后转移至 $500\mathrm{mL}$ 容量瓶，加水定容，摇匀，必要时过滤。吸取 $25.00\mathrm{mL}$ 试样溶液于

$250~\mathrm{mL}$ 容量瓶中，用水定容，混匀。

# 2.2.4.2 测定

吸取 $25.00 \mathrm{~mL}$ 稀释的试样溶液(2.2.4.1)于 $150 \mathrm{~mL}$ 锥形瓶中，加水至 $50 \mathrm{~mL}$ ，加入4滴铬酸钾指示剂(2.2.2.2.2)，边搅拌边用硝酸银标准滴定溶液(2.2.2.2.1)滴定，直至悬浊液中出现稳定的桔红色为终点。同时做空白试验。

# 2.2.5 分析结果的表述

试样中氯离子含量按式(1)计算：

$$
X _ {1} = \frac {\left(V _ {1} - V _ {0}\right) \times c \times 3 5 . 4 5 3 \times f}{m \times 1 0 0 0} \times 1 0 0 \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——试样中氯离子含量， $\%$ ；

$V_{1}$ ——硝酸银标准滴定溶液的用量，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验硝酸银标准滴定溶液的用量，单位为毫升 $(\mathrm{mL})$ ；

$c$ —硝酸银标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$

35.453 ——氯离子的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ；

$f$ ——试样液稀释倍数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

100、1000——单位换算系数。

计算结果表示到小数点后两位。

# 2.2.6 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

按GB/T5009.92中原子吸收分光光度法操作。

# 2.4 镁的测定

按GB/T5009.90操作。

# 2.5 硫酸根的测定（EDTA络合滴定法）

# 2.5.1 原理

过量的氯化钡与试样中硫酸根生成难溶的硫酸钡沉淀，剩余的钡离子用乙二胺四乙酸二钠(EDTA)标准溶液滴定，间接法测定硫酸根。

# 2.5.2 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 2.5.2.1 试剂

2.5.2.1.1 盐酸。

2.5.2.1.2 氨水。

2.5.2.1.3 氯化铵。

2.5.2.1.4 无水乙醇。  
2.5.2.1.5 乙二胺四乙酸二钠镁 $\mathrm{(C_{10}H_{12}MgN_2Na_2O_8\cdot 4H_2O,Mg - EDTA)}$   
2.5.2.1.6 乙二胺四乙酸二钠(EDTA)。  
2.5.2.1.7 铬黑T。  
2.5.2.1.8 盐酸羟胺。  
2.5.2.1.9 氯化钡。

# 2.5.2.2 试剂的配制

2.5.2.2.1 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $12~\mathrm{mL}$ 浓盐酸，用水稀释至 $100~\mathrm{mL}$ 。  
2.5.2.2.2 氨-氯化铵缓冲溶液 $(\mathrm{pH} \approx 10)$ : 称取 $20 \mathrm{~g}$ 氯化铵, 用水溶解, 加入 $100 \mathrm{~mL}$ 氨水, 用水稀释至 $1 \mathrm{~L}$ 。  
2.5.2.2.3 乙二胺四乙酸二钠镁溶液：称取 $17.2\mathrm{g}$ 乙二胺四乙酸二钠镁，溶于 $1\mathrm{L}$ 水中。  
2.5.2.2.4 乙二胺四乙酸二钠标准滴定溶液 $(0.02\mathrm{mol} / \mathrm{L})$ 。  
2.5.2.2.5 铬黑T指示剂：称取 $0.2\mathrm{g}$ 铬黑T和 $2\mathrm{g}$ 盐酸羟胺，无水乙醇溶解并稀释至 $100~\mathrm{mL}$ ，贮于棕色瓶内。  
2.5.2.2.6 氯化钡溶液 $(0.02\mathrm{mol} / \mathrm{L})$ ：称取 $2.40\mathrm{g}$ 氯化钡，溶于 $500~\mathrm{mL}$ 水中，室温放置 $24\mathrm{h}$ ，过滤后使用。

吸取 $5.00\mathrm{mL}$ 氯化钡溶液置于 $150~\mathrm{mL}$ 锥形瓶中，加入 $5\mathrm{mL}$ 乙二胺四乙酸二钠镁溶液(2.5.2.2.3)、 $10\mathrm{mL}$ 无水乙醇、 $5\mathrm{mL}$ 氨-氯化铵缓冲溶液(2.5.2.2.2)、4滴铬黑T指示剂(2.5.2.2.5)，用EDTA标准滴定溶液(2.5.2.2.4)滴定至溶液由酒红色变为亮蓝色为止，记录消耗的EDTA标准滴定溶液体积 $V_{1}$ 。

# 2.5.3 仪器和设备

2.5.3.1 天平：感量 $0.1\mathrm{mg}$

2.5.3.2 水浴锅。

# 2.5.4 分析步骤

# 2.5.4.1 试样处理

称取 $25\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )粉碎的试样于 $400~\mathrm{mL}$ 烧杯中，加约 $200~\mathrm{mL}$ 的水，置沸水浴上加热，用玻璃棒搅拌至全部溶解。冷却后转移至 $500~\mathrm{mL}$ 容量瓶，加水定容，摇匀，必要时过滤。当试样中待测离子含量过高时可适当稀释后再测定。

# 2.5.4.2 测定

吸取一定体积(使溶液中硫酸根含量在 $8\mathrm{mg}$ 以下)的试样溶液(2.5.4.1)，置于 $150~\mathrm{mL}$ 锥形瓶中，加1滴盐酸溶液(2.5.2.2.1)，加入 $5.00~\mathrm{mL}$ 氯化钡溶液(2.5.2.2.6)，搅拌片刻，放置 $5\mathrm{min}$ ，加入 $5\mathrm{mL}$ 乙二胺四乙酸二钠镁溶液， $10~\mathrm{mL}$ 无水乙醇， $5\mathrm{mL}$ 氨-氯化铵缓冲溶液、4滴铬黑T指示剂，用EDTA标准滴定溶液滴定至溶液由酒红色变为亮蓝色为止，记录消耗的EDTA标准滴定溶液体积 $V_{3}$ 。

溶液中钙镁总量的滴定：吸取与测定硫酸根体积相同的试样溶液，置于 $150~\mathrm{mL}$ 锥形瓶中，加水至 $25~\mathrm{mL}$ ，加入 $5\mathrm{mL}$ 氨-氯化铵缓冲溶液、4滴铬黑T指示剂，用EDTA标准滴定溶液滴定至溶液由酒红色变为亮蓝色为止，记录消耗的EDTA标准溶液的体积 $V_{2}$ 。

# 2.5.5 分析结果的表述

试样中硫酸根的含量按式(2)计算：

$$
X _ {2} = \frac {\left(V _ {1} + V _ {2} - V _ {3}\right) \times c \times 9 6 . 0 6 \times F}{m \times 1 0 0 0} \times 1 0 0 \dots \tag {2}
$$

式中：

$X_{2}$ ——试样中硫酸根的含量， $\%$ ；

$V_{1}$ ——滴定氯化钡溶液时EDTA标准滴定溶液的用量，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴定钙镁总量时EDTA标准滴定溶液的用量，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——滴定硫酸根时EDTA标准滴定溶液的用量，单位为毫升 $(\mathrm{mL})$ ；

c ——EDTA标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

96.06 ——硫酸根的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ；

$F$ ——试样液稀释倍数；

$m$ ——试样的质量，单位为克(g)；

100、1000——单位换算系数。

计算结果表示到小数点后两位。

# 2.5.6 精密度

在重复性条件下，获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 3 铅的测定

# 3.1 原理

试样经处理后，铅离子在一定 $\mathrm{pH}$ 条件下与二乙基二硫代氨基甲酸钠（DDTC）形成络合物，经4-甲基-2-戊酮萃取分离，导入原子吸收光谱仪中，电热原子化后吸收 $283.3\mathrm{nm}$ 共振线，在一定浓度范围内，其吸收值与铅含量成正比，与标准系列比较定量。

# 3.2 试剂和材料

注：除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的二级水。所用试剂用时现配。

# 3.2.1 试剂

3.2.1.1 硝酸。  
3.2.1.2 高氯酸。  
3.2.1.3 硫酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{SO}_{4}\right]$ 。  
3.2.1.4 柠檬酸铵。  
3.2.1.5 溴百里酚蓝。  
3.2.1.6 二乙基二硫代氨基甲酸钠（DDTC）。  
3.2.1.7 氨水。  
3.2.1.8 4-甲基-2-戊酮(MIBK)。

# 3.2.2 试剂的配制

3.2.2.1 混合酸：硝酸-高氯酸 $(9 + 1)$ 。  
3.2.2.2 硫酸铵溶液 $(300\mathrm{g / L})$ ：称取 $30\mathrm{g}$ 硫酸铵，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
3.2.2.3 柠檬酸铵溶液 $(250\mathrm{g / L})$ ：称取 $25\mathrm{g}$ 柠檬酸铵，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
3.2.2.4 溴百里酚蓝溶液 $(1\mathrm{g / L})$ 。  
3.2.2.5 二乙基二硫代氨基甲酸钠(DDTC)溶液 $(50\mathrm{g / L})$ ：称取 $5\mathrm{g}$ 二乙基二硫代氨基甲酸钠，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
3.2.2.6 氨水溶液 $(1 + 1)$ 。  
3.2.2.7 硝酸溶液 $(5 + 95)$ ：量取 $50~\mathrm{mL}$ 硝酸缓慢倒入 $950~\mathrm{mL}$ 水中，混匀。

# 3.2.3 标准品

铅标准储备液 $(1.0\mathrm{mg / mL})$ 。

# 3.2.4 标准溶液配制

铅标准使用液：精确吸取铅标准储备液于 $100~\mathrm{mL}$ 容量瓶中，用硝酸溶液(3.2.2.7)定容至刻度。如此经多次稀释成每毫升分别含 $0.0\mathrm{ng},5.0\mathrm{ng},10.0\mathrm{ng},20.0\mathrm{ng},40.0\mathrm{ng}$ 铅的标准使用液。

# 3.3 仪器和设备

3.3.1 原子吸收光谱仪，附石墨炉及铅空心阴极灯。  
3.3.2 天平：感量 $0.001\mathrm{g}$   
3.3.3 可调式电热板、可调式电炉。

# 3.4 分析步骤

# 3.4.1 试样处理

准确称取 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ 试样于 $100~\mathrm{mL}$ 烧杯中，加少量水溶解，再加少量混合酸(3.2.2.1)，加热煮沸，放冷后全部转移至 $50~\mathrm{mL}$ 容量瓶中，稀释至刻度，混匀备用。

# 3.4.2 萃取分离

视试样情况，吸取 $25.0\mathrm{mL}\sim 50.0\mathrm{mL}$ 按3.4.1制备的样液及试剂空白液，分别置于 $125~\mathrm{mL}$ 分液漏斗中，补加水至 $60~\mathrm{mL}$ 。加 $2\mathrm{mL}$ 柠檬酸铵溶液(3.2.2.3)，溴百里酚蓝溶液(3.2.2.4)3滴～5滴，用氨水溶液(3.2.2.6)调 $\mathsf{pH}$ 至溶液由黄变蓝，加硫酸铵溶液(3.2.2.2) $10.0~\mathrm{mL}$ ，DDTC溶液(3.2.2.5) $10~\mathrm{mL}$ ，摇匀。放置 $5\mathrm{min}$ 左右，加入 $10.0~\mathrm{mL}$ 的MIBK(3.2.1.8)，剧烈振摇提取 $1\mathrm{min}$ ，静置分层后，弃去水层，将MIBK层放入 $10~\mathrm{mL}$ 带塞刻度管中，备用。分别吸取铅标准使用液 $10.0~\mathrm{mL}$ 于 $125~\mathrm{mL}$ 分液漏斗中，与试样相同方法萃取。同时做试剂空白。

# 7 钡的测定

# 7.1 原理

钡离子与硫酸根生成硫酸钡沉淀，利用比浊作限量测定。

# 7.2 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 7.2.1 试剂

7.2.1.1 硫酸。  
7.2.1.2 氯化钡。

# 7.2.2 试剂的配制

7.2.2.1 硫酸溶液：量取 $5.7\mathrm{mL}$ 硫酸，倒入 $50~\mathrm{mL}$ 水中，再加水稀释至 $100~\mathrm{mL}$   
7.2.2.2 钡标准溶液：准确称取 $2.6680\mathrm{g}$ 氯化钡 $\mathrm{(BaCl_2\cdot 2H_2O)}$ ，溶于水，移入 $100~\mathrm{mL}$ 容量瓶中，加水至刻度，摇匀， $1\mathrm{mL}$ 此溶液相当于 $15.0\mathrm{mg}$ 钡。

# 7.3 仪器和设备

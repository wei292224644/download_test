# 前言

本标准代替GB5009.12—2017《食品安全国家标准 食品中铅的测定》。

本标准与GB5009.12—2017相比，主要变化如下：

——增加了第一法石墨炉原子吸收光谱法中需除盐样品的前处理方法；

——删除了第四法二硫腙比色法；

——修改了第一法石墨炉原子吸收光谱法的检出限和定量限。

# 食品安全国家标准

# 食品中铅的测定

# 1 范围

本标准规定了食品中铅的石墨炉原子吸收光谱、电感耦合等离子体质谱和火焰原子吸收光谱测定方法。

本标准适用于食品中铅的测定。

# 第一法 石墨炉原子吸收光谱法

# 2 原理

试样消解处理后，经石墨炉原子化，在 $283.3\mathrm{nm}$ 处测定吸光度。在一定浓度范围内，铅的吸光度值与铅含量成正比，与标准系列比较定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为 $\mathrm{GB / T6682}$ 规定的二级水。

3.1.1 硝酸 $(\mathrm{HNO}_3)$ 。  
3.1.2 高氯酸 $\mathrm{(HClO_4)}$ 。  
3.1.3 磷酸二氢铵 $\mathrm{(NH_4H_2PO_4)}$ 。

3.1.5 乙酸铵 $\mathrm{(CH_3COONH_4)}$ 。  
3.1.6 乙酸钠 $\mathrm{(CH_3COONa)}$ 。

# 5.2 试样前处理

# 5.2.1 湿法消解

称取固体试样 $0.2\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.50~\mathrm{mL}\sim 5.00~\mathrm{mL}$ 于带刻度消化管中，含乙醇或二氧化碳的样品先在电热板上低温加热除去乙醇或二氧化碳，加入 $10~\mathrm{mL}$ 硝酸和 $0.5\mathrm{mL}$ 高氯酸，放数粒玻璃珠，在可调式电热炉上消解（参考条件： $120^{\circ}C / 0.5\mathrm{h}\sim 1\mathrm{h}$ ；升至 $180^{\circ}C / 2\mathrm{h}\sim$ $4\mathrm{h}$ 、升至 $200^{\circ}C\sim 220^{\circ}C)$ 。若消化液呈棕褐色，再加少量硝酸，消解至冒白烟，消化液呈无色透明或略带黄色，赶酸至近干，停止消解，冷却后用水定容至 $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ ，混匀备用。同时做试剂空白试验。亦可采用锥形瓶，于可调式电热板上，按上述操作方法进行湿法消解。

注：可根据实际情况调节加入的硝酸、高氯酸体积。

# 5.2.2 微波消解

称取固体试样 $0.2\mathrm{g}\sim 2\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.50~\mathrm{mL}\sim 3.00~\mathrm{mL}$ 于微波消解罐中，含乙醇或二氧化碳的样品先在电热板上低温加热除去乙醇或二氧化碳，加入 $5\mathrm{mL}\sim 10\mathrm{mL}$ 硝酸（可根据试样的称样量、性质调整硝酸使用量)，按照微波消解的操作步骤消解试样，消解条件参考附录A。冷却后取出消解罐，在电热板上于 $140^{\circ}C\sim 160^{\circ}C$ 赶酸至近干。消解罐放冷后，将消化液转移至 $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 容量瓶中，用少量水洗涤消解罐2次～3次，合并洗涤液于容量瓶中并用水定容至刻度，混匀备用。同时做试剂空白试验。

# 5.2.3 压力罐消解

称取固体试样 $0.2\mathrm{g}\sim 2\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.50~\mathrm{mL}\sim 5.00~\mathrm{mL}$ 于消解内罐中，含乙醇或二氧化碳的样品先在电热板上低温加热除去乙醇或二氧化碳，加入 $5\mathrm{mL}\sim 10\mathrm{mL}$ 硝酸（可根据试样的称样量、性质调整硝酸使用量）。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱，于 $140^{\circ}C$ $\sim$ $160^{\circ}C$ 下保持 $4\mathrm{h}\sim 5\mathrm{h}$ 。冷却后缓慢旋松外罐，取出消解内罐，放在可调式电热板上于 $140^{\circ}C\sim 160^{\circ}C$ 赶酸至近干。冷却后将消化液转移至 $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 容量瓶中，用少量水洗涤内罐和内盖2次～3次，合并洗涤液于容量瓶中并用水定容至刻度，混匀备用。同时做试剂空白试验。

注：食盐、酱油、腌渍食品、火锅底料和方便面盐包等高盐食品可采用除盐操作,具体参见附录 B。

# 6 分析结果的表述

试样中铅的含量按式(1)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中铅的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$\rho$ ——试样溶液中铅的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白溶液中铅的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样消化液的定容体积(如按需除盐试样前处理，则为洗脱液体积)，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样称样量或移取体积，单位为克或毫升（g或 $\mathrm{mL}$ ）；

1000——换算系数。

当铅含量 $\geqslant 1.00\mathrm{mg / kg}$ （或 $\mathrm{mg / L}$ )时，计算结果保留3位有效数字；当铅含量 $<  1.00~\mathrm{mg / kg}$ （或 $\mathrm{mg / L})$ 时，计算结果保留2位有效数字。

# 7 精密度

样品中铅含量大于 $1\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ ；小于或等于 $1\mathrm{mg / kg}$ 且大于 $0.1\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ ；小于或等于 $0.1\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当称样量为 $0.5\mathrm{g}$ （或 $0.5\mathrm{mL}$ ），定容体积为 $10~\mathrm{mL}$ 时，方法的检出限为 $0.02\mathrm{mg / kg}$ （或 $0.02\mathrm{mg / L})$ ，定量限为 $0.04\mathrm{mg / kg}$ 或 $0.04\mathrm{mg / L})$ 。

对于生乳、巴氏杀菌乳、灭菌乳、果蔬汁类及其饮料［含浆果及小粒水果的果蔬汁类及其饮料、浓缩果蔬汁(浆)除外]、液态婴幼儿配方食品等样品，当称样量为 $2\mathrm{g}$ （或 $2\mathrm{mL}$ ，定容体积为 $10~\mathrm{mL}$ 时，方法的检出限为 $0.005\mathrm{mg / kg}$ （或 $0.005\mathrm{mg / L}$ )，定量限为 $0.01\mathrm{mg / kg}$ （或 $0.01\mathrm{mg / L}$ ）。

# 第三法 火焰原子吸收光谱法

# 9 原理

试样经处理后，铅离子在一定 $\mathrm{pH}$ 条件下与二乙基二硫代氨基甲酸钠（DDTC)形成络合物，经4-甲基-2-戊酮(MIBK)萃取分离，导入原子吸收光谱仪中，经火焰原子化，在 $283.3\mathrm{nm}$ 处测定吸光度。在一定浓度范围内铅的吸光度值与铅含量成正比，与标准系列比较定量。

# 12.3.2 标准曲线的制作

分别吸取铅标准使用液 $0\mathrm{mL}, 0.25\mathrm{mL}, 0.50\mathrm{mL}, 1.00\mathrm{mL}, 1.50\mathrm{mL}$ 和 $2.00\mathrm{mL}$ （相当于 $0\mu \mathrm{g}$ 、 $2.50\mu \mathrm{g}, 5.00\mu \mathrm{g}, 10.0\mu \mathrm{g}, 15.0\mu \mathrm{g}$ 和 $20.0\mu \mathrm{g}$ 铅）于 $125\mathrm{mL}$ 分液漏斗中，补加水至 $60\mathrm{mL}$ 。加 $2\mathrm{mL}$ 柠檬酸铵溶液 $(250\mathrm{g} / \mathrm{L})$ ，溴百里酚蓝水溶液 $(1\mathrm{g} / \mathrm{L})3$ 滴～5滴，用氨水溶液 $(1 + 1)$ 调 $\mathrm{pH}$ 至溶液由黄变蓝，加硫酸铵溶液 $(300\mathrm{g} / \mathrm{L})10\mathrm{mL}$ ，DDTC溶液 $(50\mathrm{g} / \mathrm{L})10\mathrm{mL}$ ，摇匀。放置 $5\mathrm{min}$ 左右，加入 $10\mathrm{mL}$ MIBK，剧烈振摇提取 $1\mathrm{min}$ ，静置分层后，弃去水层，将MIBK层放入 $10\mathrm{mL}$ 带塞刻度管中，得到标准系列溶液。

将标准系列溶液按质量由低到高的顺序分别导入火焰原子化器，原子化后测其吸光度值，以铅的质量为横坐标，吸光度值为纵坐标，制作标准曲线。

# 12.3.3 试样溶液的测定

将试样消化液及试剂空白溶液分别置于 $125~\mathrm{mL}$ 分液漏斗中，补加水至 $60~\mathrm{mL}$ 。加 $2\mathrm{mL}$ 柠檬酸铵溶液 $(250~\mathrm{g / L})$ ，溴百里酚蓝水溶液 $(1\mathrm{g / L})3$ 滴～5滴，用氨水溶液 $(1 + 1)$ 调 $\mathrm{pH}$ 至溶液由黄变蓝，加硫酸铵溶液 $(300~\mathrm{g / L})10~\mathrm{mL}$ ，DDTC溶液 $(50~\mathrm{g / L})10~\mathrm{mL}$ ，摇匀。放置 $5\mathrm{min}$ 左右，加入 $10~\mathrm{mL}$ MIBK，剧烈振摇提取 $1\mathrm{min}$ ，静置分层后，弃去水层，将MIBK层放入 $10~\mathrm{mL}$ 带塞刻度管中，得到试样溶液和空白溶液。

将试样溶液和空白溶液分别导入火焰原子化器，原子化后测其吸光度值，与标准系列比较定量。

# 13 分析结果表述

试样中铅的含量按式(2)计算。

$$
X = \frac {m _ {1} - m _ {0}}{m _ {2}} \tag {2}
$$

式中：

$X$ ——试样中铅的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$ ；

$m_{1}$ ——试样溶液中铅的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_0$ ——空白溶液中铅的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_{2}$ ——试样称样量或移取体积，单位为克或毫升（g或 $\mathrm{mL}$ ）；

当铅含量 $\geqslant 10.0\mathrm{mg / kg}$ （或 $\mathrm{mg / L}$ )时，计算结果保留3位有效数字；当铅含量 $<  10.0~\mathrm{mg / kg}$ （或 $\mathrm{mg / L})$ 时，计算结果保留2位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

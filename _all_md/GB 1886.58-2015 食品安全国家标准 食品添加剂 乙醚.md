# 食品添加剂 乙醚

# 1 范围

本标准适用于以乙醇或乙烯为原料在硫酸作用下脱水后分馏制得的食品添加剂乙醚。

# 2 分子式、结构式和相对分子质量

# 2.1 分子式

$$
\mathrm {C} _ {4} \mathrm {H} _ {1 0} \mathrm {O}
$$

# 2.2 结构式

$$
\mathrm {H} _ {3} \mathrm {C} \underset {\vee} {\mathrm {O}} \underset {\vee} {\mathrm {C H} _ {3}}
$$

# 2.3 相对分子质量

74.12（按2007年国际相对原子质量）

# 附录A

# 检验方法

# A.1 警示

试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和防护措施。

# A.2 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682中规定的三级水。试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备；所用溶液除另有说明外，均为水溶液。

# A.3 鉴别试验

# A.3.1 溶解性

不溶于水，易溶于乙醇。

# A.3.2 密度

按GB/T5009.2的规定进行测定。 $20^{\circ}\mathrm{C}$ 时密度应为 $\mathrm{0.713~g / cm^3\sim 0.715~g / cm^3}$ □

# A.4 醛和酮的测定

# A.4.1 试剂和材料

纳氏试液：取碘化钾 $10\mathrm{g}$ 溶于 $10~\mathrm{mL}$ 水中，在搅拌下缓慢加入氯化汞饱和液，一直到剩余的细小红色沉淀不再溶解。另取氢氧化钾 $30\mathrm{g}$ 溶于 $60~\mathrm{mL}$ 水中，用冰冷却后加入上述混合液中。再加 $1\mathrm{mL}$ 氯化汞饱和溶液。用水稀释至 $200~\mathrm{mL}$ ，静置沉淀，取上清液。将 $2\mathrm{mL}$ 该试剂，加入到 $100~\mathrm{mL}$ 氯化铵的无氨水溶液 $(3.3\mu \mathrm{g} / \mathrm{mL})$ 中，应立即出现淡黄棕色。否则应重新配制。

# A.4.2 分析步骤

取纳氏试液 $2\mathrm{mL}$ ，放入一个容量约为 $12\mathrm{mL}$ ，直径为 $1.5\mathrm{cm}$ 的具塞试管中，用试样加满试管。盖好塞子，强烈振摇 $10\mathrm{s}$ ，然后在暗处静置 $5\mathrm{min}$ 。如果该过程没有颜色或沉淀产生，则样品通过试验。如果有颜色或沉淀产生，则取足量的试样，于分馏柱中进行分馏，并用该流出液重复上述试验，无颜色或沉淀产生，即通过试验。

# A.5 酸度的测定

# A.5.1 试剂和材料

A.5.1.1 95%乙醇。  
A.5.1.2 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.02\mathrm{mol / L}$

A.5.1.3 酚酞指示液： $10\mathrm{g / L}$

# A.5.2 分析步骤

在 $50~\mathrm{mL}$ 具塞量筒中加入 $20~\mathrm{mL}$ 乙醇，加 $0.5\mathrm{mL}$ 酚酞指示液，滴加氢氧化钠标准滴定溶液至粉色刚刚出现，并在轻摇下保持 $30~\mathrm{s}$ 。再加入 $0.3\mathrm{mL}$ 氢氧化钠标准滴定溶液和 $25~\mathrm{mL}$ 试样，盖上塞子，混合并轻摇保持 $30~\mathrm{s}$ ，粉色不消失即为通过试验。

# A.6 过氧化物试验

# A.6.1 试剂和材料

A.6.1.1 碘化钾溶液： $100\mathrm{g / L}$ （即用即配）。  
A.6.1.2 碘标准滴定溶液： $c\left(\frac{1}{2}\mathrm{I}_2\right) = 0.001\mathrm{mol / L}$   
A.6.1.3 标准比色溶液：用 $8\mathrm{mL}$ 碘化钾溶液加 $0.5\mathrm{mL}$ 的碘标准滴定溶液，混合备用。

# A.6.2 分析步骤

取碘化钾溶液 $8\mathrm{mL}$ ，放入具塞试管中，加入 $4\mathrm{mL}$ 试样，盖好塞子并且无气泡存留，强烈振摇，然后在暗处静置 $30\mathrm{min}$ 。与标准比色溶液比较，出现的黄色不深于标准比色溶液，即为通过试验。

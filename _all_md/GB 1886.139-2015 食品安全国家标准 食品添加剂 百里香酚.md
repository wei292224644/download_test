# 食品安全国家标准

# 食品添加剂 百里香酚

# 1 范围

本标准适用于以天然萜类含氧化合物或其他原料，经化学反应制得的食品添加剂百里香酚（又名麝香草酚）。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

5-甲基-2-异丙基苯酚

# 2.2 分子式

$$
\mathrm {C} _ {1 0} \mathrm {H} _ {1 4} \mathrm {O}
$$

# 2.3 结构式

![](images/a6e2ea2726f493971c5e64122b2b033ed52ebc0f21cc997f215521fe760071d9.jpg)

# 2.4 相对分子质量

150.22（按2007年国际相对原子质量）

# 附录A

# 百里香酚含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

试样制备：称取试样 $1\mathrm{g}$ 溶于 $10~\mathrm{mL}~95\%$ （体积分数）乙醇中，摇匀备用。面积归一化法：按GB/T11538—2006中10.4测定含量。

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行。

食品添加剂百里香酚气相色谱图(面积归一化法)参见附录B。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ 。  
B.2.2 固定相：硝基对苯二酸改性的聚乙二醇。  
B.2.3 膜厚: $0.5 \mu \mathrm{m}$ 。  
B.2.4 色谱炉温度：线性程序升温从 $80^{\circ}\mathrm{C} \sim 240^{\circ}\mathrm{C}$ ，速率 $4^{\circ}\mathrm{C / min}$ 。  
B.2.5 进样口温度： $250^{\circ}C$ 。  
B.2.6 检测器温度： $250^{\circ}C$ 。  
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 载气流速： $1\mathrm{mL / min}$   
B.2.10 进样量： $1\mu \mathrm{L}$   
B.2.11 分流比： $30:1$

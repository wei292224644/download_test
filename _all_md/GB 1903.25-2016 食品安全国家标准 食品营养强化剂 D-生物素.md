# 食品营养强化剂 D-生物素

# 1 范围

本标准适用于以溴化物、苯烯酸、钠盐等为原料，经化学合成制得的食品营养强化剂D-生物素。

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度应为分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 红外吸收光谱试验

采用溴化钾涂片法，按照GB/T6040测定红外吸收光谱，测得的红外光谱应与D-生物素对照图谱一致(对照图谱见附录B)。

# A.2.2 溴水褪色试验

配制试样饱和水溶液，逐滴加入到橙黄色溴水中，溴水褪色。

# A.3 D-生物素含量（以干基计）的测定

# A.3.1 试剂和材料

A.3.1.1 D-生物素标准品：纯度应为 $99\%$ 以上。

A.3.1.2 乙腈：色谱纯。

A.3.1.3 缓冲溶液：称取一水高氯酸钠 $1\mathrm{g}$ ，置于 $500~\mathrm{mL}$ 水中溶解，加入 $1\mathrm{mL}$ 磷酸，定容至 $1000~\mathrm{mL}$ 。

A.3.1.4 稀释剂：乙腈 $+$ 水 $= 1 + 4$

# A.3.2 仪器和设备

高效液相色谱仪：配紫外检测器（检测波长为 $200\mathrm{nm}$ ）。

# A.3.3 参考色谱条件

A.3.3.1 色谱柱： $15\mathrm{cm}\times 4.6\mathrm{mm}$ （内径)，填料粒径 $3\mu \mathrm{m}$ ，ODS柱或其他等效色谱柱。

A.3.3.2 流动相：乙腈 $+$ 缓冲溶液 $= 8.5 + 91.5$

A.3.3.3 流速： $1.2\mathrm{mL / min}$

A.3.3.4 进样量： $50~\mu \mathrm{L}$

注：系统适用性为拖尾因子不大于1.5，且重复注入标准溶液两次，所得响应面积的相对平均偏差小于 $2.0\%$ 。

# A.3.4 分析步骤

# A.3.4.1 标准溶液的制备

取适量D-生物素标准品溶于稀释剂，配置成 $0.1\mathrm{mg / mL}$ 的标准溶液，必要时超声处理使完全溶解。

# A.3.4.2 试样液的制备

取适量D-生物素试样(经 $105~\mathrm{^\circ C}$ ，4h干燥处理)溶于稀释剂，配置成 $0.1\mathrm{mg / mL}$ 的试样液，必要时超声处理使完全溶解。

# A.3.4.3 测定

在参考色谱条件下，分别对标准溶液和试样液进行测定，记录主峰面积，根据式(A.1)计算出试样中D-生物素的含量。

# A.3.5 结果计算

D-生物素含量(以干基计)的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {r _ {\mathrm {U}}}{r _ {\mathrm {S}}} \times \frac {c _ {\mathrm {S}}}{c _ {\mathrm {U}}} \times 100 \% \tag{A.1}
$$

式中：

$r_{\mathrm{U}}$ ——试样液色谱分析得到的主峰面积；

$r_{\mathrm{S}}$ ——标准溶液色谱分析得到的主峰面积；

$c_{\mathrm{S}}$ ——标准溶液中D-生物素的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$c_{\mathrm{U}}$ ——试样液中D-生物素的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ 。

试验结果以平行测定结果的算术平均值为准。

# A.4 澄清度的测定

取试样 $1.0\mathrm{g}$ 于 $0.5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液，溶解定容至 $10.0~\mathrm{mL}$ ，此时溶液应呈无色透明。

# 前言

本标准代替GB/T4789.38-2008《食品卫生微生物学检验大肠杆菌计数》。

本标准与 GB/T 4789.38-2008 相比，主要变化如下：

——修改了标准的中文名称；

——修改了培养基和试剂；

——将“第二法大肠杆菌VRB-MUG平板计数法”改为“大肠埃希氏菌平板计数法（第二法）”；

——删除了“第三法 大肠杆菌 Petrifilm™ 测试片计数法”；

——修改了附录A。

# 食品安全国家标准

# 食品微生物学检验 大肠埃希氏菌计数

# 1 范围

本标准规定了食品中大肠埃希氏菌（Escherichia coli）计数的方法。

本标准适用于食品中大肠埃希氏菌的计数，其中大肠埃希氏菌平板计数法（第二法）不适用于贝类产品。

# 2 术语和定义

# 2.1 大肠埃希氏菌 Escherichia coli

大肠杆菌

广泛存在于人和温血动物的肠道中，能够在 $44.5^{\circ} \mathrm{C}$ 发酵乳糖产酸产气，IMViC（靛基质、甲基红、VP试验、柠檬酸盐）生化试验为 $+ + - -$ 或 $- + - -$ 的革兰氏阴性杆菌。以此作为粪便污染指标来评价食品的卫生状况，推断食品中肠道致病菌污染的可能性。

# 2.2 最可能数

基于泊松分布的一种间接计数方法，简称为MPN。

# 3 设备和材料

除微生物实验室常规灭菌及培养设备外，其他设备和材料如下：

a） 恒温培养箱： $36^{\circ}C\pm 1^{\circ}C$   
b） 冰箱： $2^{\circ}C\sim 5^{\circ}C$   
c） 恒温水浴箱： $44.5\mathrm{^\circ C}\pm 0.2\mathrm{^\circ C}$   
d) 天平：感量为 $0.1\mathrm{g}$   
c） 均质器；  
d） 振荡器；  
e）无菌吸管： $1\mathrm{mL}$ （具 $0.01\mathrm{mL}$ 刻度）、 $10\mathrm{mL}$ （具 $0.1\mathrm{mL}$ 刻度）或微量移液器及吸头；  
f） 无菌锥形瓶：容量 $500~\mathrm{mL}$   
g） 无菌培养皿：直径 $90\mathrm{mm}$   
h) pH计或pH比色管或精密pH试纸；  
i） 菌落计数器；  
j）紫外灯：波长 $360\mathrm{nm}\sim 366\mathrm{nm}$ ，功率 $\leqslant 6\mathrm{W}$ 。

# 5.2 操作步骤

# 5.2.1 样品的稀释

5.2.1.1固体和半固体样品：称取 $25\mathrm{g}$ 样品，放入盛有 $225~\mathrm{mL}$ 磷酸盐缓冲液的无菌均质杯内，8000r/min～10000r/min均质 $1\mathrm{min}\sim 2\mathrm{min}$ ，制成1:10样品匀液，或放入盛有 $225~\mathrm{mL}$ 磷酸盐缓冲液的无菌均质袋中，用拍击式均质器拍打 $1\mathrm{min}\sim 2\mathrm{min}$ 制成1:10的样品匀液。  
5.2.1.2 液体样品：以无菌吸管吸取 $25 \mathrm{~mL}$ 样品置盛有 $225 \mathrm{~mL}$ 磷酸盐缓冲液的无菌锥形瓶（瓶内预置适当数量的无菌玻璃珠）中，充分混匀，制成 1:10 的样品匀液。  
5.2.1.3 样品匀液的 $\mathrm{pH}$ 值应在 $6.5\sim 7.5$ 之间，必要时分别用 $1\mathrm{mol} / \mathrm{LNaOH}$ 或 $1\mathrm{mol} / \mathrm{L}\mathrm{HCl}$ 调节。  
5.2.1.4 用 $1 \mathrm{~mL}$ 无菌吸管或微量移液器吸取 $1:10$ 样品匀液 $1 \mathrm{~mL}$ , 沿管壁缓缓注入 $9 \mathrm{~mL}$ 磷酸盐缓冲液的无菌试管中 (注意吸管或吸头尖端不要触及稀释液面), 振摇试管或换用 $1$ 支 $1 \mathrm{~mL}$ 无菌吸管或吸头反复吹打, 使其混合均匀, 制成 $1:100$ 的样品匀液。  
5.2.1.5 根据对样品污染状况的估计，按上述操作，依次制成10倍递增系列稀释样品匀液。每递增稀释1次，换用1支 $1\mathrm{mL}$ 无菌吸管或吸头。从制备样品匀液至样品接种完毕，全过程不得超过 $15\mathrm{min}$ 。

# 5.2.2 初发酵试验

每个样品，选择3个适宜的连续稀释度的样品匀液（液体样品可以选择原液），每个稀释度接种3管月桂基硫酸盐胰蛋白胨（LST）肉汤，每管接种 $1\mathrm{mL}$ （如接种量超过 $1\mathrm{mL}$ ，则用双料LST肉汤）， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h} \pm 2\mathrm{h}$ ，观察小倒管内是否有气泡产生， $24\mathrm{h} \pm 2\mathrm{h}$ 产气者进行复发酵试验，如未产气则继续培养 $48\mathrm{h} \pm 2\mathrm{h}$ 。产气者进行复发酵试验。如所有LST肉汤管均未产气，即可报告大肠埃希氏菌MPN结果。

# 5.2.3 复发酵试验

用接种环从产气的LST肉汤管中分别取培养物1环，移种于已提前预温至 $45\%$ 的EC肉汤管中，放入带盖的 $44.5\% \pm 0.2\%$ 水浴箱内。水浴的水面应高于肉汤培养基液面，培养 $24\mathrm{h} \pm 2\mathrm{h}$ ，检查小倒管内是否有气泡产生，如未有产气则继续培养至 $48\mathrm{h} \pm 2\mathrm{h}$ 。记录在 $24\mathrm{h}$ 和 $48\mathrm{h}$ 内产气的EC肉汤管数。如所有EC肉汤管均未产气，即可报告大肠埃希氏菌MPN结果；如有产气者，则进行EMB平板分离培养。

# 5.2.4 伊红美蓝平板分离培养

轻轻振摇各产气管，用接种环取培养物分别划线接种于EMB平板， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $18\mathrm{h} \sim 24\mathrm{h}$ 。观察平板上有无具黑色中心有光泽或无光泽的典型菌落。

# 5.2.5 营养琼脂斜面或平板培养

从每个平板上挑 5 个典型菌落, 如无典型菌落则挑取可疑菌落。用接种针接触菌落中心部位, 移种到营养琼脂斜面或平板上, $36^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ , 培养 $18 \mathrm{~h} \sim 24 \mathrm{~h}$ 。取培养物进行革兰氏染色和生化试验。

# 5.2.6 鉴定

取培养物进行靛基质试验、MR-VP试验和柠檬酸盐利用试验。大肠埃希氏菌与非大肠埃希氏菌的生化鉴别见表1。

# 5.3 大肠埃希氏菌MPN计数的报告

大肠埃希氏菌为革兰氏阴性无芽胞杆菌，发酵乳糖、产酸、产气，IMViC生化试验为 $+$ ——或一十一。只要有1个菌落鉴定为大肠埃希氏菌，其所代表的LST肉汤管即为大肠埃希氏菌阳性。依据LST肉汤阳性管数查MPN表（见附录B)，报告每 $\mathrm{g}$ （mL）样品中大肠埃希氏菌MPN值。

图2 大肠埃希氏菌平板计数法检验程序

# 6.2 操作步骤

# 6.2.1 样品的稀释

按5.2.1进行。

# 6.2.2 平板计数

6.2.2.1 选取 2 个～3 个适宜的连续稀释度的样品匀液，每个稀释度接种 2 个无菌平皿，每皿 $1 \mathrm{~mL}$ 。同时取 $1 \mathrm{~mL}$ 稀释液加入无菌平皿做空白对照。  
6.2.2.2 将 $10 \mathrm{~mL} \sim 15 \mathrm{~mL}$ 冷至 $45^{\circ} \mathrm{C} \pm 0.5^{\circ} \mathrm{C}$ 的结晶紫中性红胆盐琼脂（VRBA）倾注于每个平皿中。小心旋转平皿，将培养基与样品匀液充分混匀。待琼脂凝固后，再加 $3 \mathrm{~mL} \sim 4 \mathrm{~mL}$ VRBA-MUG覆盖平板表层。凝固后翻转平板， $36^{\circ} \mathrm{C} \pm 1$ ℃培养 $18 \mathrm{~h} \sim 24 \mathrm{~h}$ 。

# 6.3 平板菌落数的选择

选择菌落数在 $10\mathrm{CFU} \sim 100\mathrm{CFU}$ 之间的平板，暗室中 $360\mathrm{nm} \sim 366\mathrm{nm}$ 波长紫外灯照射下，计数平板上发浅蓝色荧光的菌落。

检验时用已知MUG阳性菌株（如大肠埃希氏菌ATCC25922）和产气肠杆菌（如ATCC13048）做阳性和阴性对照。

# 6.4 大肠埃希氏菌平板计数的报告

两个平板上发荧光菌落数的平均数乘以稀释倍数, 报告每 $\mathrm{g}(\mathrm{mL})$ 样品中大肠埃希氏菌数, 以 $\mathrm{CFU} / \mathrm{g}(\mathrm{mL})$ 表示。若所有稀释度（包括液体样品原液）平板均无菌落生长, 则以小于1乘以最低稀释倍数报告。

# 附录A

# 检验方法

# A.1 警示

试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和防护措施。

# A. 2 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682中规定的三级水。试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603之规定制备；所用溶液除另有说明外，均为水溶液。

# A.3 鉴别试验

# A.3.1 溶解性

易溶于水。移取 $50~\mathrm{mL}$ 试样，置于 $100~\mathrm{mL}$ 比色管中，用水稀释至 $100~\mathrm{mL}$ ，混匀，在约 $10^{\circ}\mathrm{C}$ 水浴中静置 $30\mathrm{min}$ ，应无薄雾或浑浊出现。

# A.3.2 折光率

按GB/T614的规定进行测定。折光率n（20，D）应为：1.364。

# A.3.3 沸点

按照GB/T7534中规定的方法进行测定。沸点应为 $78~\mathrm{^\circ C}$ 。

# A.3.4 红外光谱

试样的红外光谱图应与乙醇红外标准谱图基本一致。乙醇红外标准谱图见附录B图B.1。

# A. 4 乙醇含量的测定

# A.4.1 方法一（仲裁法）

乙醇含量的测定以乙醇的相对密度值判定，按GB/T5009.2中的规定进行测定。相对密度应为： $d_{25}^{25}$ $\leqslant 0.8096$ 或 $d_{15.56}^{15.56}\leqslant 0.8161$ 。相当于乙醇含量(体积分数)不小于 $94.9\%$

# A.4.2 方法二

按GB394.2-2008中第5章酒精度的规定进行测定。

# A. 5 酸度的测定

# A.5.1 试剂和材料

A.5.1.1 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.02\mathrm{mol / L}$

A.5.1.2 酚酞指示液： $10\mathrm{g / L}$

# A.5.2 分析步骤

向装有 $25 \mathrm{~mL}$ 水的带玻璃塞的烧瓶中加 $10 \mathrm{~mL}$ 试样，加 $0.5 \mathrm{~mL}$ 酚酞指示液，用氢氧化钠标准滴定溶液滴定至粉红色刚出现，并保持30s不变，然后加 $25 \mathrm{~mL}$ （约 $20 \mathrm{~g}$ ）试样，混匀。再用氢氧化钠标准滴定溶液滴定至粉红色，消耗氢氧化钠标准滴定溶液的体积不应大于 $1.0 \mathrm{~mL}$ 。相当于酸度不大于 $0.05 \mathrm{~g} / \mathrm{L}$ 。

# A. 6 碱度的测定

# A.6.1 试剂和材料

A.6.1.1 硫酸溶液： $c$ （ $\mathrm{H}_2\mathrm{SO}_4) = 0.02\mathrm{mol} / \mathrm{L}$   
A.6.1.2 甲基红指示液： $1\mathrm{g / L}$

# A.6.2 分析步骤

在 $25 \mathrm{~mL}$ 水中加入 2 滴甲基红指示液, 滴加 $0.02 \mathrm{~mol} / \mathrm{L}$ 的硫酸溶液, 直到刚出现红色, 然后加入 $25 \mathrm{~mL}$ 试样 (约 $20 \mathrm{~g}$ ), 混匀。使红色恢复, 加入 $0.02 \mathrm{~mol} / \mathrm{L}$ 的硫酸溶液的体积不大于 $2.0 \mathrm{~mL}$ 。相当于乙醇的碱度不大于 $0.003 \%$ 。

# A. 7 酮和其他醇的含量

# A.7.1 方法提要

采用气相色谱法，在选定的色谱条件下，使试样经色谱柱分离，用氢火焰离子化检测器检测，用面积归一化法定量。

# A.7.2 仪器和设备

A.7.2.1 气相色谱仪：配有氢火焰离子化检测器。整机灵敏度和稳定性符合GB/T9722中的有关规定。  
A.7.2.2 微量注射器： $10~\mu \mathrm{L}$

# A.7.3 色谱柱及操作条件

本标准推荐的色谱柱和色谱操作条件见表A.1。其他能达到同等分离程度的色谱柱及色谱操作条件也可使用。

# A.7.4 分析步骤

根据仪器说明书，调节仪器至表A.1所示的操作条件，待仪器稳定后即可开始进样分析。用色谱工作站处理计算结果。

# A.7.5 结果计算

某一被测组分的质量分数 $w_{\mathrm{i}}$ ，按式（A.1）计算：

$$
w _ {i} = \frac {A _ {i}}{\sum A i} \times 100 \% \dots \tag{A.1}
$$

式中：

$A_{\mathrm{i}}$ ——某一被测组分的峰面积；

$\sum A_{i}$ ——所有组分的峰面积之和。

# A.8 杂醇油试验

量取 $10 \mathrm{~mL}$ 试样，加入 $1 \mathrm{~mL}$ 甘油和 $1 \mathrm{~mL}$ 水，混匀，逐滴滴在清洁、无味的滤纸上，自然挥发至干，应始终无异臭发生。

# A.9 易碳化物试验

移取 $10\mathrm{mL}$ 硫酸置于锥形瓶中，冷却至约 $10^{\circ}\mathrm{C}$ ，逐滴加入 $10\mathrm{mL}$ 试样，并持续搅动。溶液的颜色为无色或不深于混合前试剂或试样的颜色，即为通过试验。

# A. 10 易氧化物试验

# A. 10.1 试剂和材料

高锰酸钾溶液： $0.1\mathrm{mol} / \mathrm{L}$

# A.10.2 分析步骤

移取 $20 \mathrm{~mL}$ 试样置于预先冷却至 $15^{\circ} \mathrm{C}$ 的比色管中，加 $0.1 \mathrm{~mL}$ 高锰酸钾溶液，混匀，静止 $5 \mathrm{~min}$ ，粉色没有完全消失，即为通过试验。

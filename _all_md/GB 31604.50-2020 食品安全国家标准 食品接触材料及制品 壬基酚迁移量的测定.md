# 食品安全国家标准

# 食品接触材料及制品

# 壬基酚迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品壬基酚迁移量的液相色谱-质谱/质谱测定方法。

本标准适用于食品接触材料及制品以水、 $4\%$ （体积分数)乙酸溶液、 $10\%$ （体积分数)乙醇溶液、 $20\%$ （体积分数)乙醇溶液、 $50\%$ （体积分数)乙醇溶液、橄榄油、 $95\%$ （体积分数)乙醇溶液为食品模拟物或采用 $95\%$ （体积分数)乙醇溶液、异辛烷化学替代溶剂经迁移试验所得浸泡液中壬基酚的测定。

# 2 原理

以 $4\%$ （体积分数)乙酸溶液为食品模拟物经迁移试验所得浸泡液加氨水中和后进样测定；以橄榄油为食品模拟物经迁移试验所得浸泡液经乙腈萃取和正己烷净化后进样测定；以异辛烷为化学替代溶剂经迁移试验所得浸泡液蒸干后用甲醇复溶，进样测定；其他迁移试验所得浸泡液、化学替代溶剂浸泡液直接进样测定。浸泡液中的壬基酚采用液相色谱-质谱/质谱法测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 冰乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
3.1.2 无水乙醇 $(\mathrm{C}_2\mathrm{H}_6\mathrm{O})$ 。  
3.1.3 $95\%$ 乙醇。  
3.1.4 异辛烷 $\left(\mathrm{C}_{8} \mathrm{H}_{18}\right)$ 。  
3.1.5 橄榄油：化学纯，符合GB5009.156的要求。  
3.1.6 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.7 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。  
3.1.8 正己烷 $(\mathrm{C_6H_{14}})$ 。  
3.1.9 氨水：色谱纯，体积分数 $\geq 25\%$

# 3.2 试剂配制

$4\%$ （体积分数)乙酸溶液、 $10\%$ （体积分数)乙醇溶液、 $20\%$ （体积分数)乙醇溶液、 $50\%$ （体积分数)乙醇溶液的配制按GB5009.156操作。

# 3.3 标准品

壬基酚(支链异构体的混合物， $\mathrm{C_{15}H_{24}O}$ ，CAS号：84852-15-3)标准溶液： $1000\mathrm{mg / L}$ 。

# 3.4 标准溶液配制

# 3.4.1 标准中间液（ $100\mathrm{mg / L}$ ）

准确移取壬基酚标准溶液 $(1000\mathrm{mg / L})1.00\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释并定容至刻度，混匀。此溶液在 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中保存，有效期为6个月。

# 3.4.2 标准中间液 $(5\mathrm{mg / L})$

准确移取壬基酚标准中间液 $(100\mathrm{mg / L})0.50\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释并定容至刻度，混匀。

# 3.4.3 标准工作溶液

# 3.4.3.1 $4\%$ （体积分数）乙酸标准工作溶液

分别准确移取标准中间液 $(5\mathrm{mg / L})0.02\mathrm{mL},0.04\mathrm{mL},0.06\mathrm{mL},0.10\mathrm{mL},0.20\mathrm{mL},0.40\mathrm{mL}$ 于6个 $10~\mathrm{mL}$ 容量瓶中，用 $4 \%$ （体积分数)乙酸溶液定容至刻度。得到质量浓度分别为 $0.01\mathrm{mg / L},0.02\mathrm{mg / L},$ $0.03\mathrm{mg / L},0.05\mathrm{mg / L},0.10\mathrm{mg / L},0.20\mathrm{mg / L}$ 标准工作溶液。上机前需按照5.1.2.1与食品模拟物浸泡液同步处理。

# 3.4.3.2 油基食品模拟物标准工作溶液

分别准确称取 $5.00\mathrm{g}$ （精确到 $0.1\mathrm{mg}$ )橄榄油于6个具塞玻璃离心管中，分别加入标准中间液 $(5\mathrm{mg / L})0.01\mathrm{mL},0.02\mathrm{mL},0.03\mathrm{mL},0.05\mathrm{mL},0.10\mathrm{mL},0.20\mathrm{mL}$ ，涡旋混匀，得到分别为 $0.01\mathrm{mg / kg}$ 、 $0.02\mathrm{mg / kg},0.03\mathrm{mg / kg},0.05\mathrm{mg / kg},0.10\mathrm{mg / kg},0.20\mathrm{mg / kg}$ 标准工作溶液。上机前需按照5.1.2.2与食品模拟物浸泡液同步处理。

# 3.4.3.3 异辛烷标准工作溶液

分别准确移取标准中间液 $(5\mathrm{mg / L})0.02\mathrm{mL},0.04\mathrm{mL},0.06\mathrm{mL},0.10\mathrm{mL},0.20\mathrm{mL},0.40\mathrm{mL}$ 于6个 $10~\mathrm{mL}$ 容量瓶中，用异辛烷定容至刻度，混匀，得到质量浓度分别为 $0.01\mathrm{mg / L},0.02\mathrm{mg / L},$ $0.03\mathrm{mg / L},0.05\mathrm{mg / L},0.10\mathrm{mg / L},0.20\mathrm{mg / L}$ 标准工作溶液。上机前需按照5.1.2.3与食品模拟物浸泡液同步处理。

# 3.4.3.4 其他食品模拟物、化学替代溶剂标准工作溶液

分别吸取标准中间液 $(5\mathrm{mg / L})0.02\mathrm{mL},0.04\mathrm{mL},0.06\mathrm{mL},0.10\mathrm{mL},0.20\mathrm{mL},0.40\mathrm{mL}$ 于6个 $10~\mathrm{mL}$ 容量瓶中，用相应的食品模拟物或化学替代溶剂定容至刻度，得到质量浓度分别为 $0.01\mathrm{mg / L}$ 、 $0.02\mathrm{mg / L},0.03\mathrm{mg / L},0.05\mathrm{mg / L},0.10\mathrm{mg / L},0.20\mathrm{mg / L}$ 的其他水基食品模拟物或化学替代溶剂标准工作溶液。

# 4 仪器和设备

4.1 液相色谱串联三重四极杆质谱仪：配电喷雾离子源(ESI)。  
4.2 涡旋振荡器。  
4.3 分析天平：感量 $0.0001\mathrm{g}, 0.01\mathrm{g}$ 。

4.4 高速离心机：最高转速 $>10000\mathrm{r / min}$

# 5 分析步骤

# 5.1 试液的制备

# 5.1.1 迁移试验

食品接触材料及制品按照GB31604.1和GB5009.156的要求进行迁移试验。迁移试验所得浸泡液如不能立即测试，应置于 $4\%$ 冰箱中避光保存。若进行下一步试验，应将浸泡液恢复至室温后使用。

# 5.1.2 浸泡液的处理

# 5.1.2.1 $4\%$ （体积分数）乙酸浸泡液的处理

移取迁移试验所得 $4\%$ （体积分数)乙酸浸泡液 $5.0~\mathrm{mL}$ ，加入 $0.5\mathrm{mL}$ 氨水混匀后，经 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 后，上清液待测。

# 5.1.2.2 油基食品模拟物的处理

称取 $5.00\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )迁移试验所得橄榄油浸泡液至具塞玻璃离心管中，加入 $2\mathrm{mL}$ 正己烷混匀，再加入 $5.0~\mathrm{mL}$ 乙腈，涡旋振荡 $0.5\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上层乙腈溶液经 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液待测。

# 5.1.2.3 异辛烷浸泡液的处理

移取迁移试验所得异辛烷浸泡液 $10.0\mathrm{mL}$ ，于 $40^{\circ}\mathrm{C}$ 旋蒸至近干，准确移取 $10.0\mathrm{mL}$ 甲醇溶解残渣，经 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液待测。

# 5.1.2.4 其他食品模拟物浸泡液、化学替代溶剂浸泡液的处理

移取适量迁移试验所得浸泡液，经 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液待测。

# 5.1.3 空白试液的制备

按5.1.1和5.1.2处理未与食品接触材料及制品接触的食品模拟物、化学替代溶剂。

# 5.2 仪器参考条件

# 5.2.1 液相色谱条件

液相色谱参考条件如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱， $2.6\mu \mathrm{m},2.1\mathrm{mm}\times 150\mathrm{mm}$ （或等效色谱柱）。  
b）流动相：甲醇-水（ $90 + 10$ ，体积比）。  
c) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ 。  
d) 柱温: $40^{\circ} \mathrm{C}$ 。  
e) 进样量: $2 \mu \mathrm{L}$ 。

# 5.2.2 质谱条件

质谱参考条件如下：

a）离子化模式：电喷雾电离负离子模式（ESI-）。

b）质谱扫描模式：多反应监测（MRM）。  
c）其他质谱参数参见附录A中表A.1。

# 5.3 标准曲线的绘制

按照5.2所列的仪器参考条件，对标准工作溶液进行测定，以标准工作溶液中壬基酚的浓度为横坐标，以对应的定量离子峰面积为纵坐标，绘制标准曲线，得到线性方程。壬基酚标准工作溶液的色谱图参见附录B。

# 5.4 试液的测定

# 5.4.1 定性测定

按照5.2所列的仪器参考条件，测定食品模拟物试液和标准工作溶液，若试液与标准溶液中待测物的质量色谱峰保留时间的偏差在 $\pm 2.5\%$ 范围内；所有定性离子的信噪比均超过3：1，且定性离子对的相对丰度与浓度相当的标准溶液的相对丰度一致，相对丰度偏差不超过表1的规定，则可判断样品中存在相应的待测物。

# 5.4.2 定量测定

按照5.2所列的仪器参考条件，分别对试样溶液(5.1.2)和空白溶液(5.1.3)进行测定，得到目标物峰面积，从标准曲线计算试样溶液、空白溶液中壬基酚的含量 $c_{\setminus}c_{0}$ ，空白溶液中壬基酚的含量 $c_{0}$ 不应高于方法检出限浓度。

# 6 分析结果的表述

# 6.1 非密封制品类食品接触材料及制品壬基酚特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

对于除了盖子、密封圈、连接件等密封制品（以下简称密封制品）以外的食品接触材料及制品，壬基酚特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(1)进行计算：

$$
X _ {1} = \frac {\left(c - c _ {0}\right) \times V}{S} \times F \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——壬基酚的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样浸泡液中壬基酚的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白浸泡液中壬基酚的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$V$ ——试样浸泡液体积或质量，单位为升(L)或千克(kg)；

S——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$F$ ——在实际使用情形下，食品接触材料及制品的接触面积 $(S)$ 与食品或食品模拟物体积 $(V)$ 的比 $(S / V)$ ，各种液态食品密度通常以 $1\mathrm{kg} / \mathrm{L}$ 计，单位为平方分米每千克 $(\mathrm{dm}^2 /\mathrm{kg})$ 。当实际使用情形下， $S / V$ 已知时， $F$ 即为实际 $S / V$ ；当 $S / V$ 无法估算时， $F$ 采用 $6\mathrm{dm}^2 /\mathrm{kg}$ ，即 $6\mathrm{dm}^2$ 食品接触材料及制品接触 $1\mathrm{kg}$ 食品或食品模拟物。

结果保留至少2位有效数字。

# 6.2 密封制品类食品接触材料及制品壬基酚特定迁移量的计算（以 $\mathrm{mg}$ /件表示）

当预期用途未知时，密封制品类食品接触材料及制品壬基酚特定迁移量以 $\mathrm{mg}/$ 件表示时，按式(2)计算，需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {2} = \frac {\left(c - c _ {0}\right) \times V}{n} \tag {2}
$$

式中：

$X_{2}$ ——壬基酚特定迁移量，单位为毫克每件 $(\mathrm{mg}/$ 件）；

$n$ ——浸泡用密封制品的数量，单位为件。

结果保留至少2位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不应超过其算术平均值的 $10\%$ 。

# 8 其他

当迁移试验中 $S / V$ 与实际使用情形下 $S / V$ 相同时，本方法对各食品模拟物、化学替代溶剂中壬基酚的检出限均为 $0.005\mathrm{mg / kg}$ ，定量限为 $0.01\mathrm{mg / kg}$ 。

# 前言

本标准代替GB5009.123—2014《食品安全国家标准 食品中铬的测定》。

本标准与GB5009.123—2014相比，主要变化如下：

——增加了第二法“电感耦合等离子体质谱法”；

——修改了试样制备；

——修改了湿式消解法、压力罐消解法、微波消解法和干式消解法。

# 食品安全国家标准

# 食品中铬的测定

# 1 范围

本标准规定了食品中铬的石墨炉原子吸收光谱和电感耦合等离子体质谱测定方法。

本标准适用于食品中铬的测定。

# 第一法 石墨炉原子吸收光谱法

# 2 原理

试样消解处理后，经石墨炉原子化，在 $357.9\mathrm{nm}$ 处测定吸光度。在一定浓度范围内铬的吸光度值与铬含量成正比，与标准系列比较定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 硝酸 $\mathrm{(HNO_3)}$ 。  
3.1.2 高氯酸 $\mathrm{(HClO_4)}$   
3.1.3 磷酸二氢铵 $(\mathrm{NH_4H_2PO_4})$ 。

# 3.2 试剂配制

3.2.1 硝酸溶液 $(5 + 95)$ ：量取 $50~\mathrm{mL}$ 硝酸，缓慢加入 $950~\mathrm{mL}$ 水中，混匀。  
3.2.2 硝酸溶液 $(1 + 1)$ ：量取 $50~\mathrm{mL}$ 硝酸，缓慢加入 $50~\mathrm{mL}$ 水中，混匀。  
3.2.3 磷酸二氢铵溶液 $(20\mathrm{g / L})$ ：称取 $2\mathrm{g}$ 磷酸二氢铵，加少量硝酸溶液 $(5 + 95)$ 溶解，然后用硝酸溶液 $(5 + 95)$ 定容至 $100~\mathrm{mL}$ ，混匀。

# 3.3 标准品

重铬酸钾 $\mathrm{(K_2Cr_2O_7}$ ，CAS号：7778-50-9)：纯度 $>99.99\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 铬标准储备溶液 $(1000\mathrm{mg / L})$ ：准确称取重铬酸钾 $(110^{\circ}\mathrm{C}$ ，烘 $2\textrm{h})0.2829\textrm{g}$ ，溶于水中，移入 $100~\mathrm{mL}$ 容量瓶中，用硝酸溶液 $(5 + 95)$ 稀释至刻度，混匀。此溶液铬的质量浓度为 $1000\mathrm{mg / L}$ 。  
3.4.2 铬标准中间液 $(1000\mu \mathrm{g} / \mathrm{L})$ : 准确吸取铬标准储备液 $(1000\mathrm{mg / L})1.00\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中, 加硝酸溶液 $(5 + 95)$ 至刻度, 混匀。再准确吸取上述溶液 $1.00~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中, 加硝酸溶液

$(5 + 95)$ 至刻度，混匀。此溶液铬的质量浓度为 $1000~\mu \mathrm{g / L}$ 。

3.4.3 铬标准系列溶液：分别准确吸取铬标准中间液 $0\mathrm{mL}, 0.150\mathrm{mL}, 0.400\mathrm{mL}, 0.800\mathrm{mL}, 1.20\mathrm{mL}$ 和 $1.60\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中，加硝酸溶液 $(5 + 95)$ 至刻度，混匀。此系列溶液铬的质量浓度分别为 $0\mu \mathrm{g} / \mathrm{L}, 1.50\mu \mathrm{g} / \mathrm{L}, 4.00\mu \mathrm{g} / \mathrm{L}, 8.00\mu \mathrm{g} / \mathrm{L}, 12.0\mu \mathrm{g} / \mathrm{L}$ 和 $16.0\mu \mathrm{g} / \mathrm{L}$ 。临用现配。

注：可根据仪器的灵敏度及样品中铬的实际含量确定标准系列溶液中铬的质量浓度。

# 4 仪器和设备

注：所有玻璃器皿及聚四氟乙烯消解内罐均需硝酸溶液 $(1 + 5)$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

4.1 原子吸收光谱仪：配石墨炉原子化器，附铬空心阴极灯。  
4.2 电子天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.3 可调式电热板或可调式电炉。  
4.4 微波消解系统：配聚四氟乙烯消解内罐。  
4.5 压力消解罐：配聚四氟乙烯消解内罐。  
4.6 恒温干燥箱。  
4.7 马弗炉。  
4.8 样品粉碎设备：匀浆机、高速粉碎机。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固态样品

# 5.1.1.1 干样

豆类、谷物、菌类、茶叶、干制水果、焙烤食品等低含水量样品，取可食部分，必要时经高速粉碎机粉碎均匀；固体乳制品、蛋白粉、面粉等呈均匀状的粉状样品，摇匀。

# 5.1.1.2 鲜样

蔬菜、水果、水产品等高含水量样品，必要时洗净，晾干，取可食部分匀浆均匀；肉类、蛋类等样品，取可食部分匀浆均匀。

# 5.1.1.3 速冻及罐头食品

经解冻的速冻食品及罐头样品，取可食部分匀浆均匀。

# 5.1.2 液态样品

软饮料、调味品等样品摇匀。

# 5.1.3 半固态样品

搅拌均匀。

# 5.2 方法的试验条件

# 5.2.1 试样前处理

# 5.2.1.1 湿式消解法

固体试样称取 $0.2\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，液体试样准确移取或称取 $0.500~\mathrm{mL(g)}\sim 5.00~\mathrm{mL(g)}$

(精确至 $0.001 \mathrm{~g}$ ) 于带刻度消化管中, 含乙醇或二氧化碳的样品先低温加热除去乙醇或二氧化碳, 加入 $10 \mathrm{~mL}$ 硝酸和 $0.5 \mathrm{~mL}$ 高氯酸, 在可调式电炉上消解 (参考条件: $120^{\circ} \mathrm{C}$ 保持 $0.5 \mathrm{~h} \sim 1 \mathrm{~h}$ , 升至 $180^{\circ} \mathrm{C}$ 保持 $2 \mathrm{~h} \sim 4 \mathrm{~h}$ , 升至 $200^{\circ} \mathrm{C} \sim 220^{\circ} \mathrm{C}$ )。若消化液呈棕褐色, 冷却后, 再加少量硝酸, 消解至冒白烟, 消化液呈无色透明或略带黄色, 赶酸至 $0.5 \mathrm{~mL}$ 左右后取出消化管, 冷却后用水定容至 $10 \mathrm{~mL}$ 或 $25 \mathrm{~mL}$ , 混匀备用。同时做空白试验。也可采用锥形瓶, 于可调式电热板上, 按上述操作方法进行湿法消解。

# 5.2.1.2 微波消解法

固体试样称取 $0.2\mathrm{g}\sim 0.5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，含水分较多的样品可适当增加取样量至 $1\mathrm{g}$ ），液体试样准确移取或称取 $0.500\mathrm{mL(g)}\sim 3.00\mathrm{mL(g)}$ （精确至 $0.001\mathrm{g}$ ）于微波消解罐中，含乙醇或二氧化碳的样品先低温加热除去乙醇或二氧化碳，加入 $5\mathrm{mL}\sim 10\mathrm{mL}$ 硝酸，按照微波消解的操作步骤消解试样，消解条件参考附录A中表A.1。必要时，在加酸后加盖放置 $1\mathrm{h}$ 或过夜后再按照微波消解的操作步骤消解试样。冷却后取出消解罐，于 $140~^\circ \mathrm{C}\sim 160~^\circ \mathrm{C}$ 赶酸至 $1\mathrm{mL}$ 左右。消解罐放冷后，将消化液转移至 $10\mathrm{mL}$ 或 $25\mathrm{mL}$ 容量瓶中，用少量水洗涤消解罐2次～3次，合并洗涤液于容量瓶中并用水定容至刻度，混匀备用。同时做空白试验。

# 5.2.1.3 压力罐消解法

固体试样称取 $0.2\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，含水分较多的样品可适当增加取样量至 $2\mathrm{g}$ ），液体试样准确移取或称取 $0.500\mathrm{mL(g)}\sim 5.00\mathrm{mL(g)}$ （精确至 $0.001\mathrm{g}$ ）于消解内罐中，含乙醇或二氧化碳的样品先低温加热除去乙醇或二氧化碳，加入 $5\mathrm{mL}\sim 10\mathrm{mL}$ 硝酸。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱，于 $140^{\circ}\mathrm{C}\sim 160^{\circ}\mathrm{C}$ 下保持 $4\mathrm{h}\sim 5\mathrm{h}$ 。必要时，在加酸后加盖放置 $1\mathrm{h}$ 或过夜后再旋紧不锈钢外套，放入恒温干燥箱消解试样。冷却后缓慢旋松不锈钢外套，取出消解内罐，于 $140^{\circ}\mathrm{C}\sim 160^{\circ}\mathrm{C}$ 赶酸至 $1\mathrm{mL}$ 左右。冷却后将消化液转移至 $10\mathrm{mL}$ 或 $25\mathrm{mL}$ 容量瓶中，用少量水洗涤内罐和内盖2次～3次，合并洗涤液于容量瓶中并用水定容至刻度，混匀备用。同时做空白试验。

# 5.2.1.4 干式消解法

固体试样称取 $0.5\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，液体试样准确移取或称取 $2.00~\mathrm{mL(g)}\sim 10.0~\mathrm{mL(g)}$ （精确至 $0.001\mathrm{g})$ 于坩埚中，小火加热，炭化至无烟，转移至马弗炉中，于 $550^{\circ}C$ 灰化 $3\mathrm{h}\sim 4\mathrm{h}$ 。冷却，取出，对于灰化不彻底的试样，加数滴硝酸，小火加热，小心蒸干，再转入 $550^{\circ}C$ 马弗炉中，继续灰化 $1\mathrm{h}\sim 2\mathrm{h}$ 至试样呈白灰状，冷却，取出，用适量硝酸溶液 $(1 + 1)$ 溶解后转移至 $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 容量瓶中，用少量水洗涤坩埚2次～3次，合并洗涤液于容量瓶中并用水定容至刻度，混匀备用。同时做空白试验。

仪器操作条件参见表A.2。

# 5.4 标准曲线的制作

按质量浓度由低到高的顺序分别取 $10\mu \mathrm{L}$ 标准系列溶液、 $5\mu \mathrm{L}$ 磷酸二氢铵溶液（可根据使用仪器选择最佳进样量），同时注入石墨管，原子化后测其吸光度值，以质量浓度为横坐标、吸光度值为纵坐标，绘制标准曲线。

注：磷酸二氢铵溶液作为基体改进剂，可根据使用仪器及样品基质选择添加。

# 5.5 试样溶液的测定

在测定标准曲线的相同实验条件下, 吸取 $10 \mu \mathrm{L}$ 空白溶液或试样消化液、 $5 \mu \mathrm{L}$ 磷酸二氢铵溶液 (可根据使用仪器选择最佳进样量), 同时注入石墨管, 原子化后测其吸光度值。根据标准曲线得到待测液

中铬的质量浓度。若测定结果超出标准曲线范围，用硝酸溶液 $(5 + 95)$ 稀释后测定。

# 6 分析结果的表述

试样中铬的含量按公式(1)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times f \times V}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中铬的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 或毫克每升 $(\mathrm{mg / L})$ ；

$\rho$ ——试样消化液中铬的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白溶液中铬的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$f$ ——稀释倍数；

$V$ ——试样消化液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量或体积，单位为克(g)或毫升(mL)；

1000——换算系数。

当铬含量 $\geqslant 1\mathrm{mg / kg(mg / L)}$ 时，计算结果保留3位有效数字，当铬含量 $<  1~\mathrm{mg / kg(mg / L)}$ 时，计算结果保留2位有效数字。

# 7 精密度

试样中铬含量 $>1\mathrm{mg / kg(mg / L)}$ 时，在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ ； $0.1\mathrm{mg / kg(mg / L)} <$ 试样中铬含量 $\leqslant 1\mathrm{mg / kg(mg / L)}$ 时，在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ ；试样中铬含量 $\leqslant 0.1\mathrm{mg / kg(mg / L)}$ 时，在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当取样量为 $0.5\mathrm{g}$ 或 $2\mathrm{mL}$ ，定容体积为 $10\mathrm{mL}$ 时，本方法的检出限为 $0.01\mathrm{mg/kg}$ 或 $0.003\mathrm{mg/L}$ ，定量限为 $0.03\mathrm{mg/kg}$ 或 $0.008\mathrm{mg/L}$ 。

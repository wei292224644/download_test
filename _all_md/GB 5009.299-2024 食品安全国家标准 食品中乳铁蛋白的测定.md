# 食品安全国家标准

# 食品中乳铁蛋白的测定

# 1 范围

本标准规定了食品中乳铁蛋白的高效液相色谱测定方法。

本标准适用于巴氏杀菌乳、调制乳、含乳饮料、调制乳粉及乳基婴幼儿配方食品中牛乳铁蛋白含量的测定。

本标准不适用于风味发酵乳、豆基婴幼儿配方食品中牛乳铁蛋白的测定。

# 2 原理

试样中的牛乳铁蛋白经磷酸盐缓冲液提取，肝素亲和柱富集净化后，反相高效液相色谱柱分离，紫外检测器于 $280~\mathrm{nm}$ 处检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 三氟乙酸 $\mathrm{(CF_3CO_2H)}$ ：色谱纯。  
3.1.3 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.4 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。  
3.1.5 磷酸氢二钠 $\mathrm{(Na_2HPO_4)}$

# 3.2 试剂配制

3.2.1 磷酸盐缓冲液I：称取 $7.1\mathrm{g}$ 磷酸氢二钠、 $58.4\mathrm{g}$ 氯化钠，加 $900~\mathrm{mL}$ 水溶解，用磷酸调 $\mathrm{pH}$ 至8.0 $\pm 0.2$ ，加水定容至 $1000~\mathrm{mL}$ ，临用现配。  
3.2.2 磷酸盐缓冲液Ⅱ：称取 $28.4\mathrm{g}$ 磷酸氢二钠，加 $900~\mathrm{mL}$ 水溶解，用磷酸调 $\mathrm{pH}$ 至 $7.5\pm 0.2$ ，加水定容至 $1000~\mathrm{mL}$ ，临用现配。  
3.2.3 $0.1\%$ 三氟乙酸溶液：量取 $1\mathrm{mL}$ 三氟乙酸注入 $900~\mathrm{mL}$ 水中，用水稀释至 $1000~\mathrm{mL}$ ，过 $0.45\mu \mathrm{m}$ 滤膜。

# 3.3 标准品

牛乳铁蛋白标准品(CAS号：146897-68-9)：纯度 $\geqslant 95\%$ ，铁含量 $\leqslant 35~\mathrm{mg / 100~g}$ ，或经国家认证并授予标准物质证书的标准品。

注：标准品可以使用满足溯源要求的商品化标准溶液。

# 3.4 标准溶液配制

3.4.1 牛乳铁蛋白标准储备溶液 $(5.0\mathrm{mg / mL})$ ：称取 $50~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )牛乳铁蛋白标准品用水溶解，转移至 $10~\mathrm{mL}$ 容量瓶并定容至刻度，混匀， $0^{\circ}C\sim 4^{\circ}C$ 条件下保存，可保存3个月。  
3.4.2 牛乳铁蛋白标准中间溶液 $(0.50\mathrm{mg / mL})$ ：吸取 $1.0~\mathrm{mL}$ 牛乳铁蛋白标准储备溶液，用磷酸盐缓

冲液I定容至 $10~\mathrm{mL}$ ，临用现配。

3.4.3 牛乳铁蛋白标准系列工作溶液：准确移取上述牛乳铁蛋白标准中间溶液，用磷酸盐缓冲液I稀释定容，配制成浓度分别为 $1.0\mu \mathrm{g / mL},5.0\mu \mathrm{g / mL},10.0\mu \mathrm{g / mL},50.0\mu \mathrm{g / mL},100.0\mu \mathrm{g / mL}$ 和 $200.0\mu \mathrm{g / mL}$ 的标准系列工作液，临用现配。

# 3.5 材料

3.5.1 肝素亲和柱： $1\mathrm{mL}$ ，柱容量 $\geq 2\mathrm{mg}$ ，或性能相当者(验证方法见附录B)。

注：使用不同厂商的肝素亲和柱，在样品提取、上样、淋洗和洗脱的操作方面可能会略有不同，应根据肝素亲和柱特点进行优化。

3.5.2 滤膜，水相， $0.45\mu \mathrm{m}$   
3.5.3 水相针头滤器：聚醚砜材质， $0.45\mu \mathrm{m}$ 。

# 4 仪器和设备

4.1 高效液相色谱仪：配有紫外检测器或相当者。  
4.2 天平：感量分别为 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$ 。  
4.3 冷冻离心机： $4^{\circ}\mathrm{C}$ ，转速 $\geqslant 10000\mathrm{r / min}$   
4.4 pH计：精度为0.01。  
4.5 涡旋振荡器。  
4.6 恒温水浴锅。  
4.7 固相萃取柱装置。

# 5 分析步骤

# 5.1 样品前处理

# 5.1.1 试样制备

取有代表性的样品至少 $200\mathrm{g}$ ，液态样品摇匀后储存于容器中作为试样，于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存，备用；固态样品混匀后储存于容器中作为试样，常温下干燥保存，备用。

# 5.1.2 试样提取

# 5.1.2.1 液态试样（巴氏杀菌乳、调制乳、含乳饮料）

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样于烧杯中，加入适量磷酸盐缓冲液Ⅱ提取，分次将样品转移至 $25~\mathrm{mL}$ 容量瓶，用磷酸盐缓冲液Ⅱ定容至刻度，充分混匀后转入离心管，在 $4^{\circ}\mathrm{C}$ 条件下 $10000\mathrm{r / min}$ 离心 $8\mathrm{min}$ ，取样液待净化。

# 5.1.2.2 固态试样（调制乳粉、乳基婴幼儿配方食品）

称取 $1\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 试样于烧杯中，加入温热(温度不超过 $50~^\circ \mathrm{C})$ 的磷酸盐缓冲液Ⅱ，搅拌使样品溶解，冷却至室温，分次将样品转移至 $50~\mathrm{mL}$ 容量瓶并用磷酸盐缓冲液Ⅱ定容至刻度，充分混匀后转入离心管中，取 $40.0~\mathrm{mL}$ 溶液在 $4^{\circ}C$ 条件下 $10000\mathrm{r / min}$ 离心 $8\mathrm{min}$ ，取样液待净化。

# 5.1.3 试样净化

肝素亲和柱使用前先加入 $5.0\mathrm{mL}$ 磷酸盐缓冲液Ⅱ活化。准确移取 $10.0\mathrm{mL}$ 上述样液过柱，控制流速在 $1\mathrm{mL} / \mathrm{min}\sim 3\mathrm{mL} / \mathrm{min}$ ，待样液滴完后，用 $10.0\mathrm{mL}$ 磷酸盐缓冲液Ⅱ淋洗，弃去流出液；再用

$5.0\mathrm{mL}$ 磷酸盐缓冲液I洗脱，收集洗脱液并用磷酸盐缓冲液I定容至 $5.00~\mathrm{mL}$ ，过 $0.45\mu \mathrm{m}$ 水相针头滤器，供上机检测分析用。

注：在样品过柱前，可根据实际样品中乳铁蛋白含量调整具体上样液体积。

# 5.2 仪器参考条件

5.2.1 色谱柱： $\mathrm{C}_4, 5 \mu \mathrm{m}, 300 \AA, 250 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ （内径）或相当者。  
5.2.2 检测波长： $280~\mathrm{nm}$   
5.2.3 柱温： $30^{\circ}C$ 。  
5.2.4 进样体积： $50~\mu \mathrm{L}$   
5.2.5 流速： $1.0~\mathrm{mL / min}$   
5.2.6 流动相： $0.1\%$ 三氟乙酸溶液和乙腈，梯度洗脱参数见表1。

# 5.3 标准曲线的制作

将牛乳铁蛋白标准系列工作溶液分别注入高效液相色谱仪中，测定相应的峰面积，以标准工作液中牛乳铁蛋白的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。牛乳铁蛋白标准溶液的色谱图见附录A中图A.1。

# 5.4 试样溶液的测定

将试样溶液注入高效液相色谱仪中，得到峰面积，根据标准曲线得到试样中牛乳铁蛋白的浓度。

# 6 分析结果的表述

试样中牛乳铁蛋白含量按式(1)计算：

$$
X = \frac {\rho \times V _ {1} \times V _ {3} \times 1 0 0}{m \times V _ {2} \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中牛乳铁蛋白的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$\rho$ ——根据标准工作曲线计算得到的试样溶液中牛乳铁蛋白的浓度, 单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ;

$V_{1}$ ——试样提取液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——洗脱液定容体积，单位为毫升 $(\mathrm{mL})$

100 ——换算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——样液过柱体积，单位为毫升 $(\mathrm{mL})$

1000 ——换算系数。

计算结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当液态试样的取样量为 $5.00\mathrm{g}$ ，定容体积为 $25~\mathrm{mL}$ 时，本方法中液态试样检出限为 $0.3\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $1\mathrm{mg} / 100\mathrm{g}$ ；当固态试样的取样量为 $5.00\mathrm{g}$ ，定容体积为 $50~\mathrm{mL}$ 时，检出限为 $2\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $6\mathrm{mg} / 100\mathrm{g}$ 。

# 肝素亲和柱验证方法

# B.1 柱容量验证

取 $5.0\mathrm{mg / mL}$ 牛乳铁蛋白标准储备溶液 $0.40~\mathrm{mL}$ 代替样液过肝素亲和柱，按5.1.3操作，用液相色谱仪测牛乳铁蛋白的含量，计算其回收率。同一批次至少测试2根肝素亲和柱。

# B.2 结果判定

牛乳铁蛋白回收率 $\geqslant 95\%$ ，为可使用的商品。

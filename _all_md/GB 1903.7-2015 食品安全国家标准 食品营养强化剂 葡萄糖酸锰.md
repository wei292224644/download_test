# 食品安全国家标准

# 食品营养强化剂 葡萄糖酸锰

# 1 范围

本标准适用于以葡萄糖酸或葡萄糖酸内酯、硫酸锰为主要原料，经加工制得的食品营养强化剂葡萄糖酸锰产品。

# 2 分子式、结构式和相对分子质量

# 2.1 分子式

$\mathrm{C_{12}H_{22}MnO_{14}}$ （无水化合物）， $\mathrm{C_{12}H_{22}MnO_{14}}\cdot 2\mathrm{H_2O}$ （二水化合物）

无水化合物：445.24（按2011年国际相对原子质量）

二水化合物：481.27（按2011年国际相对原子质量）

# 检验方法

# A.1 安全提示

本标准试验方法中使用的部分试剂具有毒性或腐蚀性，使用时需小心谨慎并按相关规定操作。若溅到皮肤上应立即用水冲洗，严重者应立即治疗。在使用挥发性酸时，要在通风橱中进行。

# A.2 一般规定

除非另有说明，在分析中仅使用确认为分析纯试剂和GB/T6682中规定的三级水。试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 冰乙酸。  
A.3.1.2 氨溶液：取浓度为 $28\%$ 的氨水 $400~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。本试液中氨含量为 $9.5\% \sim 10.5\%$ 。  
A.3.1.3 硫化氢气体：以硫化亚铁和硫酸为原料，用启普气体发生器制备。  
A.3.1.4 硫酸镁溶液：溶解 $12\mathrm{g}$ 硫酸镁 $(\mathrm{MgSO_4}\cdot 7\mathrm{H_2O})$ 于适量水中，用水稀释至 $100~\mathrm{mL}$   
A.3.1.5 氯化钙溶液：氯化钙饱和水溶液。   
A.3.1.6 $10\%$ 硫化铵溶液：用硫化氢气体饱和氨溶液(A.3.1.2)后，再加入其2/3体积的氨溶液。

本溶液的灼烧残渣量不得大于 $0.05\%$ ，遇硫酸镁溶液(A.3.1.4)或氯化钙溶液(A.3.1.5)不得发生浑浊，如本溶液出现明显的硫化物沉淀，则已不能用。本溶液应存放于棕色小瓶中，置于阴冷暗处。

A.3.1.7 试样溶液：配制浓度为 $5\%$ 的溶液，必要时 $60^{\circ}\mathrm{C}$ 水浴加热溶解。  
A.3.1.8 苯肼：临用时蒸馏。  
A.3.1.9 溴化钾：光谱纯。

# A.3.2 锰离子 $(\mathbf{Mn}^{2+})$ 的鉴别

# A.3.2.1 方法提要

锰盐溶液遇硫化铵溶液即产生橙红色沉淀，该沉淀可溶于冰乙酸。

# A.3.2.2 分析步骤

在烧杯中加入 $20~\mathrm{mL}$ 试样溶液(A.3.1.7)，再加入 $0.5\mathrm{mL}$ 硫化铵溶液(A.3.1.6)，静置 $10\mathrm{min}$ 后观察应产生橙红色沉淀。再逐滴加入冰乙酸，沉淀应溶解。

# A.3.3 葡萄糖酸根的鉴别

# A.3.3.1 化学鉴别法

# A.3.3.1.1 方法提要

试样在乙酸介质中，与苯肼共热，生成黄色葡萄糖酰苯肼结晶。

# A.3.3.1.2 分析步骤

称取 $0.5\mathrm{g}$ 试样，加 $10~\mathrm{mL}$ 水，溶解（必要时微热溶解），取该溶液 $5\mathrm{mL}$ 于 $10~\mathrm{mL}$ 试管中，加入 $0.7\mathrm{mL}$ 冰乙酸和 $1\mathrm{mL}$ 苯肼(A.3.1.8)，水浴加热 $30\mathrm{min}$ ，冷却至室温，用玻璃棒摩擦试管内壁，应析出黄色结晶。

# A.3.3.2 红外光谱法

采用溴化钾压片法，按照GB/T6040的规定进行试验，试样的红外光谱应与对照的谱图一致。对照谱图见附录B。

# A.4 葡萄糖酸锰 $(\mathbf{C}_{12}\mathbf{H}_{22}\mathbf{MnO}_{14})$ 含量的测定

# A.4.1 方法提要

以铬黑T为指示剂，用乙二胺四乙酸二钠标准滴定溶液滴定试样溶液，根据乙二胺四乙酸二钠标准滴定溶液的用量，计算以 $\mathrm{C_{12}H_{22}MnO_{14}}$ 计的葡萄糖酸锰的含量。

# A.4.2 试剂和材料

A.4.2.1 乙二胺四乙酸二钠(EDTA二钠)： $c(\mathrm{Na}_2\mathrm{EDTA}) = 0.05\mathrm{mol / L}$

A.4.2.2 抗坏血酸。

A.4.2.3 氨-氯化铵缓冲液：称取氯化铵 $67.5\mathrm{g}$ ，溶于适量水，加入 $28\%$ 氨水 $570\mathrm{mL}$ ，用水稀释至 $1000\mathrm{mL}$ 。

A.4.2.4 铬黑T溶液：取铬黑 $\mathrm{T}200\mathrm{mg}$ 和盐酸胺（ $\mathrm{(NH_2OH\cdot HCl)}$ ） $2\mathrm{g}$ ，用甲醇溶解并稀释至 $50~\mathrm{mL}$ 过滤。贮于遮光容器中，限两周内使用。

# A.4.3 分析步骤

精确称取约 $700\mathrm{mg}$ 试样于 $250~\mathrm{mL}$ 锥形瓶中，加水 $50~\mathrm{mL}$ 溶解（必要时加热），加抗坏血酸 $1\mathrm{g}$ ，加入 $10~\mathrm{mL}$ 氨-氯化铵缓冲液和5滴铬黑T试液，以 $0.05\mathrm{mol} / \mathrm{L}$ EDTA二钠溶液滴定至深蓝色。同时做空白试验。

# A.4.4 结果计算

样品中葡萄糖酸锰的质量分数 $(w_{1})$ （以干基计），按式(A.1)计算：

$$
w _ {1} = \frac {c \times \left(V - V _ {0}\right) \times 445.24}{m \times 1000 \times (1 - x)} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$\varpi_{1}$ ——试样中葡萄糖酸锰的质量分数， $\%$ ；

$c$ ——EDTA二钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

V ——滴定试样消耗EDTA二钠溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗EDTA二钠标准滴定液的体积，单位为毫升 $(\mathrm{mL})$

445.24 葡萄糖酸锰的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000 —— 换算系数；

$x$ ——试样中的水分含量， $\%$ 。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 水分的测定

# A.5.1 试剂

A.5.1.1 甲醇：色谱纯。

A.5.1.2 甲酰胺：如试剂中水的质量分数大于 $0.05\%$ ，应在 $500~\mathrm{mL}$ 甲酰胺中加入5A分子筛 $50~\mathrm{g}$ ，塞上瓶塞，放置过夜，吸取上层清液使用。

A.5.1.3 提取溶剂：将甲醇(A.5.1.1)和甲酰胺(A.5.1.2)按 $1:1$ （体积分数）混合，摇匀。

# A.5.2 分析步骤

称取试样约 $0.1\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )，加入到适量的提取溶剂(A.5.1.3)中，滴定前搅拌 $30\mathrm{min}$ 。按GB/T6283中“直接电量滴定法”测定。

# A.6 还原性物质的测定

# A.6.1 方法提要

在沸腾条件下，还原糖与碱性柠檬酸铜溶液反应生成相当量的氧化亚铜 $\mathrm{(Cu_2O)}$ 沉淀，冷却后加入盐酸使溶液呈酸性，并使 $\mathrm{Cu_2O}$ 沉淀溶解。然后准确加入过量碘溶液进行氧化，用硫代硫酸钠溶液滴定过量的碘。硫代硫酸钠标准溶液空白试验滴定量减去其试样滴定量得到一个差值，由此差值计算还原性物质的量。

# A.6.2 试剂和材料

A.6.2.1 碱性柠檬酸铜溶液：

溶液A：称取 $173\mathrm{g}$ 柠檬酸钠（ $\mathrm{C_6H_5O_7Na_3\cdot 2H_2O)}$ 和 $100\mathrm{g}$ 无水碳酸钠，加温水溶解并稀释至 $700~\mathrm{mL}$ （若溶液浑浊时过滤澄清）。

溶液B：称取 $17.3\mathrm{g}$ 硫酸铜结晶，加水溶解并稀释至 $100~\mathrm{mL}$

临用前取 $100\mathrm{mL}$ 溶液B，在不断振摇下，缓缓加入 $700~\mathrm{mL}$ 溶液A。冷却后加水稀释至 $1000~\mathrm{mL}$ 。

A.6.2.2 乙酸溶液： $1:10$ （体积分数）。

A.6.2.3 碘标准溶液： $c\left(\frac{1}{2}\mathrm{I}_2\right) = 0.05\mathrm{mol / L}$

A.6.2.4 盐酸溶液： $2.7\mathrm{mol} / \mathrm{L}$

A.6.2.5 淀粉指示剂： $0.5\%$ ，临用新配。

A.6.2.6 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$

# A.6.3 分析步骤

称取约 $1.0\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )，置于 $250~\mathrm{mL}$ 碘瓶中，加 $10~\mathrm{mL}$ 水溶解（如果不能完全溶解可

适当加热，冷却至室温)，加入 $25.0~\mathrm{mL}$ 碱性柠檬酸铜溶液，瓶口用小表面皿盖住，准确微沸 $5\mathrm{min}$ 后，迅速冷却至室温，加 $25.0~\mathrm{mL}$ 乙酸溶液，摇匀。准确加入 $10.0~\mathrm{mL}$ 碘标准溶液，加入 $10.0~\mathrm{mL}$ 盐酸溶液，密塞，摇匀。再加 $3.0~\mathrm{mL}$ 淀粉指示剂，立即用硫代硫酸钠标准溶液滴定至溶液深蓝色消失，并将滴定结果用空白试验校正。每毫升硫代硫酸钠标准滴定溶液 $(0.1\mathrm{mol} / \mathrm{L})$ 相当于 $2.7\mathrm{mg}$ 葡萄糖。

# A.6.4 结果计算

还原性物质（以 $\mathrm{C_6H_{12}O_6}$ 计）的质量分数 $\varpi_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {\left(V _ {0} - V _ {1}\right) \times c \times M}{m \times 1000} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$w_{2}$ ——样品中还原性物质（以 $\mathrm{C_6H_{12}O_6}$ 计）的质量分数， $\%$ ；

$V_{0}$ ——滴定空白溶液所消耗的硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——滴定试样溶液所消耗的硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫代硫酸钠标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——还原糖的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(3 / 20\mathrm{C_6H_{12}O_6}) = 27]$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

# A.7 氯化物（以Cl计）的测定

# A.7.1 试剂和材料

A.7.1.1 硝酸溶液：取 $25~\mathrm{mL}$ 浓硝酸，加水稀释至 $100~\mathrm{mL}$   
A.7.1.2 硝酸银溶液： $17\mathrm{g / L}$   
A.7.1.3 盐酸标准滴定溶液： $0.02\mathrm{mol / L}$

# A.7.2 分析步骤

称取 $1.0\mathrm{g}$ 试样（精确到 $0.01\mathrm{g}$ ），用水溶解并定容至 $50~\mathrm{mL}$ （如果不能完全溶解可适当加热），作为试样溶液（浓度为 $20~\mathrm{g / L}$ ）。取 $0.7\mathrm{mL}$ 盐酸标准滴定溶液(A.7.1.3)，用水定容至 $50~\mathrm{mL}$ ，作为标准试验溶液。分别取试样溶液和标准试验溶液 $5.0~\mathrm{mL}$ 于 $25~\mathrm{mL}$ 比色管中，其他按GB/T9729规定进行操作。

# A.8 硫酸盐（以 $\mathrm{SO}_4$ 计）的测定

# A.8.1 试剂和材料

A.8.1.1 盐酸溶液：取 $20~\mathrm{mL}$ 浓盐酸，加水稀释至 $100~\mathrm{mL}$   
A.8.1.2 硫酸钾乙醇溶液： $0.2\mathrm{g / L}$   
A.8.1.3 氯化钡溶液： $250~\mathrm{g / L}$

A.8.1.4 硫酸盐标准溶液：准确称取 $0.2000\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )已在 $105~^\circ \mathrm{C}\sim 110~^\circ \mathrm{C}$ 烘干 $2\textrm{h}$ 的硫酸钠（ $\mathrm{Na_2SO_4}$ ，优级纯)，置于 $300~\mathrm{mL}$ 烧杯中，加适量蒸馏水溶解，移于 $1000~\mathrm{mL}$ 容量瓶中，用蒸馏水稀释至刻度摇匀。此溶液每毫升含硫酸钠 $0.2\mathrm{mg}$ 。

# A.8.2 分析步骤

取 $5.0\mathrm{mL}$ A.7.2中的试样溶液 $(20~\mathrm{g / L})$ 于 $25~\mathrm{mL}$ 比色管中，同时另取 $1.0~\mathrm{mL}$ 硫酸盐 $\mathrm{(SO_4^{2 - })}$ 标

准溶液（相当于 $0.2\mathrm{mgSO}_4^{2 - }$ )按GB/T9728规定进行操作。

# A.9 砷（As）的测定

称取约 $1.0\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )，按GB5009.76“湿法消解”的规定进行样品处理，最终将消化液转移并用水定容于 $50~\mathrm{mL}$ 容量瓶，其他操作按GB5009.76“氢化物原子荧光光度法”测定。

# A.10 铅(Pb)的测定

称取约 $1.0\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )，按GB5009.12“湿法消解”的规定进行样品处理，最终将消化液转移并用水定容于 $50~\mathrm{mL}$ 容量瓶，其他按GB5009.12“石墨炉原子吸收光谱法”测定。

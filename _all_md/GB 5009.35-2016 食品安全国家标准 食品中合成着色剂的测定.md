# 食品中合成着色剂的测定

2016-08-31 发布

2017-03-01 实施

# 前言

本标准代替GB/T5009.35—2003《食品中合成着色剂的测定》。

本标准与GB/T5009.35—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中合成着色剂的测定”；  
——增加了试剂的级别和分子式；  
——增加了标准品；  
——修改了计算公式；  
——修改了色谱图；  
——删除第二法薄层色谱法；  
——删除第三法示波极谱法。

# 食品安全国家标准

# 食品中合成着色剂的测定

# 1 范围

本标准规定了饮料、配制酒、硬糖、蜜饯、淀粉软糖、巧克力豆及着色糖衣制品中合成着色剂（不含铝色锭)的测定方法。

本标准适用于饮料、配制酒、硬糖、蜜饯、淀粉软糖、巧克力豆及着色糖衣制品中合成着色剂（不含铝色锭)的测定。

# 2 原理

食品中人工合成着色剂用聚酰胺吸附法或液-液分配法提取，制成水溶液，注入高效液相色谱仪，经反相色谱分离，根据保留时间定性和与峰面积比较进行定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ 。  
3.1.3 盐酸（HCl）。  
3.1.4 冰醋酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.5 甲酸（HCOOH）。  
3.1.6 乙酸铵 $\mathrm{(CH_3COONH_4)}$ 。  
3.1.7 柠檬酸 $\mathrm{(C_6H_8O_7\bullet H_2O)}$ 。  
3.1.8 硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.9 正丁醇 $(\mathrm{C_4H_{10}O})$ 。  
3.1.10 三正辛胺 $(\mathrm{C}_{24}\mathrm{H}_{51}\mathrm{N})$ 。  
3.1.11 无水乙醇 $\mathrm{(CH_3CH_2OH)}$   
3.1.12 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ：含量 $20\% \sim 25\%$   
3.1.13 聚酰胺粉（尼龙6）：过 $200\mu \mathrm{m}$ （目）筛。

# 3.2 试剂配制

3.2.1 乙酸铵溶液 $(0.02\mathrm{mol} / \mathrm{L})$ ：称取 $1.54\mathrm{g}$ 乙酸铵，加水至 $1000~\mathrm{mL}$ ，溶解，经 $0.45\mu \mathrm{m}$ 微孔滤膜过滤。  
3.2.2 氨水溶液：量取氨水 $2\mathrm{mL}$ ，加水至 $100~\mathrm{mL}$ ，混匀。  
3.2.3 甲醇-甲酸溶液 $(6 + 4$ ，体积比)：量取甲醇 $60~\mathrm{mL}$ ，甲酸 $40~\mathrm{mL}$ ，混匀。

3.2.4 柠檬酸溶液：称取 $20\mathrm{g}$ 柠檬酸，加水至 $100~\mathrm{mL}$ ，溶解混匀。  
3.2.5 无水乙醇-氨水-水溶液 $(7 + 2 + 1$ ，体积比)：量取无水乙醇 $70~\mathrm{mL}$ 、氨水溶液(3.2.2) $20~\mathrm{mL}$ 、水 $10~\mathrm{mL}$ ，混匀。  
3.2.6 三正辛胺-正丁醇溶液 $(5\%)$ ：量取三正辛胺 $5\mathrm{mL}$ ，加正丁醇至 $100~\mathrm{mL}$ ，混匀。  
3.2.7 饱和硫酸钠溶液。  
3.2.8 pH6的水：水加柠檬酸溶液调 $\mathrm{pH}$ 到6。  
3.2.9 pH4的水：水加柠檬酸溶液调 $\mathrm{pH}$ 到4。

# 3.3 标准品

3.3.1 柠檬黄(CAS:1934-21-0)。  
3.3.2 新红(CAS:220658-76-4)。  
3.3.3 荠菜红(CAS:915-67-3)。  
3.3.4 胀脂红(CAS:2611-82-7)。  
3.3.5 日落黄(CAS:2783-94-0)。  
3.3.6 亮蓝(CAS:3844-45-9)。  
3.3.7 赤藓红(CAS:16423-68-0)。

# 3.4 标准溶液配制

3.4.1 合成着色剂标准贮备液 $(1\mathrm{mg / mL})$ ：准确称取按其纯度折算为 $100\%$ 质量的柠檬黄、日落黄、苋菜红、胭脂红、新红、赤藓红、亮蓝各 $0.1\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），置 $100~\mathrm{mL}$ 容量瓶中，加 $\mathrm{pH}6$ 的水到刻度。配成水溶液 $(1.00\mathrm{mg / mL})$ 。  
3.4.2 合成着色剂标准使用液 $(50\mu \mathrm{g / mL})$ : 临用时将标准贮备液加水稀释 20 倍, 经 $0.45\mu \mathrm{m}$ 微孔滤膜过滤。配成每毫升相当 $50.0\mu \mathrm{g}$ 的合成着色剂。

# 4 仪器和设备

4.1 高效液相色谱仪，带二极管阵列或紫外检测器。  
4.2 天平：感量为 $0.001\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
4.3 恒温水浴锅。  
4.4 G3垂融漏斗。

# 5 分析步骤

# 5.1 试样制备

5.1.1 果汁饮料及果汁、果味碳酸饮料等：称取 $20\mathrm{g}\sim 40\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，放入 $100~\mathrm{mL}$ 烧杯中。含二氧化碳样品加热或超声驱除二氧化碳。  
5.1.2 配制酒类：称取 $20\mathrm{g}\sim 40\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，放入 $100~\mathrm{mL}$ 烧杯中，加小碎瓷片数片，加热驱除乙醇。  
5.1.3 硬糖、蜜饯类、淀粉软糖等：称取 $5\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )粉碎样品，放入 $100~\mathrm{mL}$ 小烧杯中，加水 $30~\mathrm{mL}$ ，温热溶解，若样品溶液 $\mathrm{pH}$ 较高，用柠檬酸溶液调 $\mathrm{pH}$ 到6左右。  
5.1.4 巧克力豆及着色糖衣制品：称取 $5 \mathrm{~g} \sim 10 \mathrm{~g}$ （精确至 $0.001 \mathrm{~g}$ ），放入 $100 \mathrm{~mL}$ 小烧杯中，用水反复洗涤色素，到巧克力豆无色素为止，合并色素漂洗液为样品溶液。

# 5.2 色素提取

5.2.1 聚酰胺吸附法：样品溶液加柠檬酸溶液调 $\mathrm{pH}$ 到6，加热至 $60^{\circ} \mathrm{C}$ ，将 $1 \mathrm{~g}$ 聚酰胺粉加少许水调成粥状，倒入样品溶液中，搅拌片刻，以G3垂融漏斗抽滤，用 $60^{\circ} \mathrm{C} \mathrm{pH}$ 为4的水洗涤3次～5次，然后用甲醇-甲酸混合溶液洗涤3次～5次（含赤藓红的样品用5.2.2法处理），再用水洗至中性，用乙醇-氨水-水混合溶液解吸3次～5次，直至色素完全解吸，收集解吸液，加乙酸中和，蒸发至近干，加水溶解，定容至 $5 \mathrm{~mL}$ 。经 $0.45 \mu \mathrm{m}$ 微孔滤膜过滤，进高效液相色谱仪分析。  
5.2.2 液-液分配法（适用于含赤藓红的样品）：将制备好的样品溶液放入分液漏斗中，加 $2\mathrm{mL}$ 盐酸、三正辛胺-正丁醇溶液 $(5\%)$ $10~\mathrm{mL}\sim 20~\mathrm{mL}$ ，振摇提取，分取有机相，重复提取，直至有机相无色，合并有机相，用饱和硫酸钠溶液洗2次，每次 $10~\mathrm{mL}$ ，分取有机相，放蒸发皿中，水浴加热浓缩至 $10~\mathrm{mL}$ ，转移至分液漏斗中，加 $10~\mathrm{mL}$ 正己烷，混匀，加氨水溶液提取2次～3次，每次 $5~\mathrm{mL}$ ，合并氨水溶液层（含水溶性酸性色素），用正己烷洗2次，氨水层加乙酸调成中性，水浴加热蒸发至近干，加水定容至 $5\mathrm{mL}$ 。经 $0.45\mu \mathrm{m}$ 微孔滤膜过滤，进高效液相色谱仪分析。

# 5.3 仪器参考条件

5.3.1 色谱柱： $C_{18}$ 柱， $4.6\mathrm{mm}\times 250\mathrm{mm},5\mu \mathrm{m}$   
5.3.2 进样量： $10~\mu \mathrm{L}$   
5.3.3 柱温： $35^{\circ}C$   
5.3.4 二极管阵列检测器波长范围： $400\mathrm{nm}\sim 800\mathrm{nm}$ ，或紫外检测器检测波长： $254~\mathrm{nm}$

# 5.4 测定

将样品提取液和合成着色剂标准使用液分别注入高效液相色谱仪，根据保留时间定性，外标峰面积法定量。

# 6 分析结果的表述

试样中着色剂含量按式(1)计算：

$$
X = \frac {c \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中着色剂的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$c$ ——进样液中着色剂的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

V ——试样稀释总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克(g)；

1000 ——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

方法检出限：柠檬黄、新红、苋菜红、胭脂红、日落黄均为 $0.5\mathrm{mg / kg}$ ，亮蓝、赤藓红均为 $0.2\mathrm{mg / kg}$ （检测波长 $254~\mathrm{nm}$ 时亮蓝检出限为 $1.0\mathrm{mg / kg}$ ，赤藓红检出限为 $0.5\mathrm{mg / kg})$ 。

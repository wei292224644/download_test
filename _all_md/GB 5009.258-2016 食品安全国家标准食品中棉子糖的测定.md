# 食品安全国家标准

# 食品中棉子糖的测定

# 1 范围

本标准规定了食品中棉子糖含量测定的离子色谱方法。

本标准适用于婴幼儿配方食品、婴幼儿谷类辅助食品、婴幼儿罐装辅助食品、饮料及豆粉食品中棉子糖含量的测定。

# 2 原理

试样采用乙腈沉淀、高速离心的方式去除样品中的蛋白质和脂肪，经固相萃取柱净化后，注入离子色谱仪，氢氧化钠溶液为淋洗液，经阴离子交换柱分离，脉冲安培检测器检测。保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。  
3.1.2 $50\%$ 氢氧化钠溶液 $(\mathrm{NaOH})$ ：色谱纯。  
3.1.3 $\alpha$ -淀粉酶：酶活力 $\geqslant 1.5 \mathrm{U} / \mathrm{mg}$ 。  
3.1.4 氮气 $(\mathrm{N}_2)$ ：纯度 $\geq 99.99\%$

# 3.2 试剂配制

氢氧化钠溶液 $(250\mathrm{mmol / L})$ ：在 $2\textrm{L}$ 塑料试剂瓶中加入 $500~\mathrm{mL}$ 经过 $0.22\mu \mathrm{m}$ 尼龙滤膜过滤的蒸馏水，移取 $13.1\mathrm{mL}$ 氢氧化钠(3.1.2)溶液至水面以下，加蒸馏水至 $1000~\mathrm{mL}$ ，通入氮气保护后，摇匀备用。

# 3.3 标准品

棉子糖标准品 $(\mathrm{C_{18}H_{32}O_{16}})$ ：纯度 $\geq 99.0\%$

# 3.4 标准溶液配制

3.4.1 标准储备溶液 $(1000\mathrm{mg / L})$ ：准确称取棉子糖 $100\mathrm{mg}$ （精确到 $0.1\mathrm{mg}$ ）于 $50~\mathrm{mL}$ 烧杯中，用蒸馏水溶解，转移到 $100~\mathrm{mL}$ 容量瓶中，用水定容。摇匀， $4^{\circ}C$ 保存，有效期为1个月。  
3.4.2 标准使用溶液 $(50.0\mathrm{mg / L})$ ：准确吸取 $5\mathrm{mL}$ 标准储备溶液于 $100~\mathrm{mL}$ 容量瓶中，用蒸馏水定容至刻度。现用现配。  
3.4.3 标准系列工作溶液：分别准确吸取棉子糖标准使用溶液 $0\mathrm{mL},1\mathrm{mL},2\mathrm{mL},5\mathrm{mL},10\mathrm{mL}$

$20~\mathrm{mL},50~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用蒸馏水定容。得到标准曲线工作溶液浓度分别为 $0.000~\mathrm{mg / L}$ 产 $0.500~\mathrm{mg / L},1.00~\mathrm{mg / L},2.50~\mathrm{mg / L},5.00~\mathrm{mg / L},10.0~\mathrm{mg / L},25.0~\mathrm{mg / L}$

# 4 仪器和设备

4.1 离子色谱仪：配置脉冲安培检测器。  
4.2 天平：感量为 $0.1\mathrm{mg}$   
4.3 离心机：转速 $\geqslant 10000\mathrm{r / min}$   
4.4 涡旋混合器。  
4.5 超声波清洗器。  
4.6 固相萃取装置。  
4.7 培养箱： $60^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。  
4.8 固相萃取柱: OnGuard II RP 柱 (填料为反相聚二乙烯基苯聚合物) 和 OnGuard II H 柱 (填料为 H 型聚苯乙烯基强酸树脂), 或其他等效柱。规格为 $1.0 \mathrm{~mL}, \mathrm{H}$ 柱为 $650 \mathrm{mg}, \mathrm{RP}$ 柱为 $350 \mathrm{mg}$ 。  
4.9 $0.22\mu \mathrm{m}$ 水性滤膜针头滤器。  
4.10 注射器： $15~\mathrm{mL}$

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 试样预处理

5.1.1.1 乳粉、米粉、豆粉等试样：取试样约 $100\mathrm{g}$ （其他固体试样先进行粉碎、研磨），装入能够容纳2倍试样体积的带盖容器中，通过反复摇晃和颠倒容器使试样充分混匀直到使试样均一化。  
5.1.1.2 液体试样：取试样约 $100~\mathrm{mL}$ ，装入能够容纳2倍试样体积的带盖容器中，通过反复摇晃和颠倒容器使试样充分混匀直到使试样均一化。  
5.1.1.3 肉泥等其他半固体试样：称取试样 $50.0\mathrm{g}$ ，称取等量的水与之混合，涡旋混匀，超声 $20\mathrm{min}$ 。

# 5.2 试样处理

5.2.1 米粉等含淀粉的试样：称取混合均匀的试样 $1\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.1\mathrm{mg}$ ，含有棉子糖约为 $20\mathrm{mg}$ ）于离心管中，加入 $1\mathrm{g}\alpha$ -淀粉酶，固体试样需用约 $50\mathrm{mL}45^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 的水使其溶解，混合均匀后充氮，盖上管盖，置于 $60^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 培养箱内酶解 $30\mathrm{min}$ 。冷却至室温后，用水定容至 $100\mathrm{mL}$ 。  
5.2.2 乳粉、豆粉、饮料、肉泥等其他试样：称取混合均匀的试样 $1\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.1\mathrm{mg}$ ，含有棉子糖约为 $20\mathrm{mg}$ ）于 $100\mathrm{mL}$ 烧杯中，加水溶解，固体试样需用约 $50\mathrm{mL}45^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 的水使其溶解，转移到 $100\mathrm{mL}$ 容量瓶中，用水定容。

# 5.3 净化

5.3.1 吸取 $5 \mathrm{~mL}$ 经过处理的试样溶液于 $50 \mathrm{~mL}$ 具塞锥形瓶中，加入 $10 \mathrm{~mL}$ 水，摇匀，超声 $30 \mathrm{~min}$ ，转移到 $50 \mathrm{~mL}$ 容量瓶中，加入 $30 \mathrm{~mL}$ 乙腈，用水定容。转移到 $50 \mathrm{~mL}$ 离心管中，于 $10000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取上清液备用（可以根据样液中棉子糖含量调整稀释倍数）。  
5.3.2 固相萃取柱的活化：将OnGuard II RP柱使用前依次用 $10~\mathrm{mL}$ 甲醇、 $15~\mathrm{mL}$ 蒸馏水进行活化；OnGuard IIH柱用 $10~\mathrm{mL}$ 蒸馏水进行活化。必要时可以使用固相萃取装置，液体流速控制在 $1~\mathrm{mL} / \mathrm{min}$ 。  
5.3.3 将 $15 \mathrm{~mL}$ 注射器的柱筒部分、 $0.22 \mu \mathrm{m}$ 水性滤膜针头滤器、OnGuard II RP 柱、OnGuard II H

柱顺序连接,接在固相萃取装置上。取备用的上清液(5.3.1)约 ${15.0}\mathrm{\;{mL}}$ 加入注射器柱筒,调节固相萃取装置,液体流速控制在 $1\mathrm{\;{mL}}/\mathrm{{min}}$ 。弃去前面 $3\mathrm{\;{mL}}$ 流出液,收集后面的流出液待测。

# 5.4 仪器参考条件

5.4.1 色谱柱：CarboPac PA20(150 mm×3 mm)；保护柱：CarboPac PA20(30 mm×3 mm)，或性能相当的离子色谱柱。  
5.4.2 淋洗液：水 $+250 \mathrm{mmol} / \mathrm{L}$ 氢氧化钠溶液 $= 58 + 42$ ，等度洗脱。  
5.4.3 流速： $0.4\mathrm{mL / min}$   
5.4.4 检测器：脉冲安培检测器，金电极， $\mathrm{Ag / AgCl}$ 参比电极，检测池温度为 $30^{\circ}C$ 。  
5.4.5 柱温： $30^{\circ}C$ 。  
5.4.6 进样体积： $10~\mu \mathrm{L}$

# 5.5 测定

# 5.5.1 标准曲线的制作

将标准系列工作溶液按照从低浓度到高浓度的顺序依次进样，得到各浓度标准溶液的色谱图。以棉子糖的浓度 $(\mathrm{mg / L})$ 为横坐标，以峰面积为纵坐标，绘制标准曲线或计算线性回归方程（标准溶液离子色谱图见图A.1）。

# 5.5.2 测定

在与标准曲线测定相同的工作条件下，取 $10~\mu \mathrm{L}$ 试样溶液注入离子色谱仪，以保留时间定性，测得峰面积，根据标准曲线得到待测样液中棉子糖的浓度(试样溶液离子色谱图参见图A.2)。

# 6 分析结果的表述

试样中棉子糖含量按式(1)计算：

$$
X = \frac {C \times V \times K}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中棉子糖的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$C$ ——试样测定液中棉子糖质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样制备时定容体积，单位为毫升 $(\mathrm{mL})$ ；

$K$ ——试样溶液稀释倍数，包括试样制备和净化过程的两步稀释；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

检出限和定量限：取样量为 $1.0\mathrm{g}$ ，定容体积为 $100~\mathrm{mL}$ ，稀释倍数为10时，测定棉子糖的检出限为 $0.1\mathrm{g / kg}$ ；定量限为 $0.3\mathrm{g / kg}$ 。

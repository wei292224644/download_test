# 前言

本标准代替GB1895—2004《食品添加剂 硫酸铝钾》。

本标准与GB1895—2004相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 硫酸铝钾(又名钾明矾)”;  
——增加了硫酸铝钾干燥品的技术要求；  
——修改了铅指标要求；  
——删除了水不溶物、重金属、水分指标要求；  
——增加了硒指标要求。

本标准适用于天然明矾矿石法和氢氧化铝法、铝矾土法生产的食品添加剂硫酸铝钾。

# 2 分子式和相对分子质量

# 2.1 分子式

# 2.2 相对分子质量

十二水合硫酸铝钾：474.37（按2013年国际相对原子质量）

# 附录A

# 检验方法

# A.1 警示

本检验方法中使用的部分试剂具有腐蚀性，操作者须小心谨慎！如溅到皮肤或眼睛上应立即用水冲洗，严重者应立即治疗。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸。

A.3.1.2 盐酸溶液： $1 + 4$

A.3.1.3 氯化钡溶液： $50~\mathrm{g / L}$

A.3.1.4 乙酸铵溶液： $100\mathrm{g / L}$

A.3.1.5 氢氧化钠溶液： $40~\mathrm{g / L}$

A.3.1.6 氨水溶液 $(10\%)$ 。

A.3.1.7 乙酸铅溶液: $100 \mathrm{~g} / \mathrm{L}$ 。称取 $10.0 \mathrm{~g}$ 乙酸铅, 加入适量煮沸冷却的水溶解, 滴加冰乙酸, 使溶液澄清, 再加煮沸冷却的水至 $100 \mathrm{~mL}$ , 摇匀。

A.3.1.8 硫化铵溶液：取 $60~\mathrm{mL}$ 氨水溶液，通硫化氢气体饱和后，再加 $40~\mathrm{mL}$ 氨水溶液。

A.3.1.9 红色石蕊试纸。

# A.3.2 仪器和设备

A.3.2.1 真空干燥箱：温度能控制为 $35^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ ，真空度能控制为 $80\mathrm{kPa} \sim 90\mathrm{kPa}$ 。

A.3.2.2 试验筛： $\phi 200\mathrm{mm}\times 50\mathrm{mm} - 0.25 / 0.16\mathrm{GB / T}6003.1 - 2012$

# A.3.3 鉴别方法

# A.3.3.1 硫酸根的鉴别

称取约 $10\mathrm{g}$ 预先研磨并在 $35^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 及真空度 $80\mathrm{kPa} \sim 90\mathrm{kPa}$ 下的真空干燥箱中干燥至质量恒定的试样，精确至 $0.0002\mathrm{g}$ ，置于 $150\mathrm{mL}$ 烧杯中，加入 $80\mathrm{mL}$ 水，加热溶解。冷却后移入 $500\mathrm{mL}$ 容量瓶中，加 $1\mathrm{mL}$ 盐酸溶液，用水稀释至刻度，摇匀(浑浊时可过滤，弃去初始滤液)，此溶液为试样溶液A。

取试样溶液A，加氯化钡溶液，即发生白色沉淀，在盐酸溶液或硝酸溶液中均不溶。

取试样溶液A，加乙酸铅溶液，即发生白色沉淀，可在乙酸铵溶液或氢氧化钠溶液中溶解。

# A.3.3.2 铝离子的鉴别

取试样溶液A，加氨水溶液或硫化铵溶液，即发生白色沉淀，能在盐酸溶液或冰乙酸中溶解，不溶于氨水溶液或铵盐溶液。

# A.3.3.3 钾离子的鉴别

取铂丝，用盐酸润湿后，在无色火焰中燃烧至无色，蘸本品在无色火焰中燃烧，用蓝色玻璃透视，火焰即显紫色。

# A.3.3.4 铵离子的鉴别

称取约 $1\mathrm{g}$ 试样，置于 $50~\mathrm{mL}$ 烧杯中，加 $10~\mathrm{mL}$ 氢氧化钠溶液，汽浴加热 $1\mathrm{min}$ ，无氨气放出（用润湿红色石蕊试纸检验，不变蓝）。

# A.4 硫酸铝钾含量的测定

# A.4.1 试剂和材料

A.4.1.1 氨水溶液： $1 + 1$

A.4.1.2 盐酸溶液： $1 + 1$

A.4.1.3 乙酸-乙酸钠缓冲溶液 $(\mathrm{pH}\approx 6)$ 。

A.4.1.4 乙二胺四乙酸二钠(EDTA)溶液： $c(\mathrm{EDTA}) = 0.05\mathrm{mol / L}$

A.4.1.5 氯化锌标准滴定溶液: $c(\mathrm{ZnCl}_2) = 0.05 \mathrm{~mol} / \mathrm{L}$ 。用移液管移取 GB/T 601 中氯化锌标准滴定溶液 $100 \mathrm{~mL}$ , 置于 $200 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度。

A.4.1.6 二甲酚橙指示液： $2\mathrm{g / L}$

A.4.1.7 刚果红试纸。

# A.4.2 仪器和设备

A.4.2.1 电热恒温干燥箱：温度能控制为 $200^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。

A.4.2.2 试验筛： $\phi 200\mathrm{mm}\times 50\mathrm{mm} - 0.25 / 0.16\mathrm{GB / T}6003.1 - 2012$

# A.4.3 分析步骤

# A.4.3.1 硫酸铝钾 $\left[\mathrm{AlK}(\mathrm{SO}_4)_2\cdot 12\mathrm{H}_2\mathrm{O}\right]$ （以干基计）的测定

用移液管移取 $25\mathrm{mL}$ 试样溶液A，置于 $250~\mathrm{mL}$ 锥形瓶中，再用移液管移取 $50~\mathrm{mL}$ 乙二胺四乙酸二钠溶液，煮沸 $5\mathrm{min}$ ，冷却至室温，加入一小块刚果红试纸，然后用氨水溶液调至试纸呈紫红色 $(\mathrm{pH}5\sim 6)$ ，加 $15~\mathrm{mL}$ 乙酸-乙酸钠缓冲溶液后加入3滴～4滴二甲酚橙指示液，用氯化锌标准滴定溶液滴定至橙黄色即为终点。

同时同样做空白试验，空白试验溶液除不加试样外，其他加入试剂的种类和量(标准滴定溶液除外)与测定试验相同。

# A.4.3.2 硫酸铝钾 $\left[\mathrm{AlK}(\mathrm{SO}_4)_2\right]$ （以干基计）的测定

称取约 $5.0\mathrm{g}$ 预先研磨且通过试验筛并在 $200^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 电热恒温干燥箱中干燥 $4\mathrm{h}$ 的试样，精确至

0.0002g, 置于 $150 \mathrm{~mL}$ 烧杯中, 加入 $80 \mathrm{~mL}$ 水, $12 \mathrm{~mL}$ 盐酸溶液, 加热溶解。冷却后移入 $500 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度, 摇匀。

用移液管移取 $25~\mathrm{mL}$ 上述试样溶液，置于 $250~\mathrm{mL}$ 锥形瓶中，再用移液管移取 $50~\mathrm{mL}$ 乙二胺四乙酸二钠溶液，煮沸 $5\mathrm{min}$ ，冷却至室温，加入一小块刚果红试纸，然后用氨水溶液调至试纸呈紫红色（pH $5\sim 6$ ），加 $15~\mathrm{mL}$ 乙酸-乙酸钠缓冲溶液后加入3滴～4滴二甲酚橙指示液，用氯化锌标准滴定溶液滴定至橙黄色即为终点。

同时同样做空白试验，空白试验溶液除不加试样外，其他加入试剂的种类和量(标准滴定溶液除外)与测定试验相同。

# A.4.4 结果计算

硫酸铝钾含量的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {c \times \left(V _ {0} - V _ {1}\right) \times M \times 500}{m \times 1000 \times 25} \times 100 \% \dots \tag{A.1}
$$

式中：

$c$ ——氯化锌标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{0}$ ——空白试验溶液消耗的氯化锌标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——试样溶液消耗氯化锌标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

M ——十二水合硫酸铝钾的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})\{M[\mathrm{AlK}(\mathrm{SO}_4)_2\cdot 12\mathrm{H}_2\mathrm{O}] =$ 474.37}; 硫酸铝钾干燥品的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})\{M[\mathrm{AlK}(\mathrm{SO}_4)_2] =$ 258.19};

500 ——试样溶液的总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算因子；

25 ——移取试样溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A.5 干燥减量的测定

# A.5.1 仪器和设备

A.5.1.1 电热恒温干燥箱：温度能控制为 $200^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。

A.5.1.2 称量瓶： $\phi 50\mathrm{mm}\times 30\mathrm{mm}$

# A.5.2 分析步骤

称取约 $6.0\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于预先于 $200^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 的电热恒温干燥箱中干燥至质量恒定的称量瓶中，于 $200^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 下干燥 $4\mathrm{h}$ 。

# A.5.3 结果计算

干燥减量的质量分数 $\varpi_{2}$ ，按式（A.2）计算：

$$
w _ {2} = \frac {m _ {1} - m _ {2}}{m _ {3}} \times 100 \% \tag{A.2}
$$

式中：

$m_{1}$ —称量瓶和试样干燥前的质量，单位为克 $(\mathrm{g})$

$m_{2}$ ——称量瓶和试样干燥后的质量，单位为克 $(\mathrm{g})$ ；

$m_{3}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.5\%$ 。

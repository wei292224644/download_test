# 食品添加剂 半乳甘露聚糖

# 1 范围

本标准适用于以瓜尔豆[Cyamops tetragonolobus(L.) Taub]、田箐豆、香豆、决明子、刺槐豆的胚乳片或其胶为原料，经半乳糖苷酶、半纤维素酶、蛋白酶一种或两种以上的酶酶解、纯化等步骤加工制得的低聚合度的的食品添加剂半乳甘露聚糖。

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 方法提要

通过两次凝胶试验，确定样品属性；在此基础上样品再经酸水解，用高效液相色谱法测定半乳糖、甘露糖含量，通过含量比值确定样品组成。

# A.2.2 凝胶试验

# A.2.2.1 试剂和材料

A.2.2.1.1 异丙醇。  
A.2.2.1.2 四硼酸钠。  
A.2.2.1.3 $50\mathrm{g / L}$ 四硼酸钠溶液：取 $50\mathrm{g}$ 四硼酸钠加水定容至 $1\mathrm{L}$   
A.2.2.1.4 黄原胶

# A.2.2.2 仪器和设备

布氏黏度计：低速转子和No.2转子。

# A.2.2.3 分析步骤

# A.2.2.3.1 凝胶试验一

# A.2.2.3.1.1 黏度计测定条件

布氏黏度计，低速转子， $5^{\circ}\mathrm{C},30\mathrm{r / min}$ （系数为0.2），60s。

# A.2.2.3.1.2 测定步骤

称取 $20.0\mathrm{g}$ 试样于烧杯中，用 $4\mathrm{mL}$ 异丙醇湿润，边激烈搅拌边加水 $200\mathrm{mL}$ ，完全溶解后停止激烈搅拌，冷却至 $5^{\circ}\mathrm{C}$ ，用黏度计测定，取其水溶液 $10\mathrm{mL}$ 与 $10\mathrm{mL}$ 四硼酸钠溶液混合，再冷却至 $5^{\circ}\mathrm{C}$ ，放置 $10\mathrm{min}$ 后，再次测定黏度，黏度增加 $1\mathrm{mPa}\cdot \mathrm{s}$ 以上为通过试验。

# A.2.2.3.2 凝胶试验二

# A.2.2.3.2.1 黏度计测定条件

布氏黏度计，No.2转子， $5^{\circ}\mathrm{C},30\mathrm{r / min}$ （系数为10），60s。

# A.2.2.3.2.2 测定步骤

称取 $1.0\mathrm{g}$ 黄原胶粉体，边搅拌边加水 $200~\mathrm{mL}$ ，为对照组，冷却至 $5^{\circ}\mathrm{C}$ 时，测定黏度。

称取 $1.0\mathrm{g}$ 试样于烧杯中，与 $1.0\mathrm{g}$ 黄原胶粉体混合，然后加入 $4\mathrm{mL}$ 异丙醇均一搅拌，边搅拌边加水 $200\mathrm{mL}$ ，搅拌至完全溶解后。取该溶液 $100\mathrm{mL}$ 在水浴中加热 $10\mathrm{min}$ ，冷却至 $5^{\circ}\mathrm{C}$ 时，测定黏度与对照组相比，黏度增加 $50\mathrm{mPa}\cdot \mathrm{s}$ 以上为通过试验。

# A.2.3 高效液相色谱试验

# A.2.3.1 试剂和材料

A.2.3.1.1 水：GB/T6682规定的二级水及以上规格。  
A.2.3.1.2 $11.99\mathrm{mol} / \mathrm{L}$ 硫酸：取 $95\%$ 浓硫酸 $10~\mathrm{mL}$ 和水 $6.5\mathrm{mL}$ 后，往水中逐渐添加浓硫酸，将密度调整为1.6338。  
A.2.3.1.3 7.5 mol/L 氢氧化钠溶液：称取氢氧化钠 $30.0\mathrm{g}$ ，加水溶解并定容至 $100~\mathrm{mL}$ 。  
A.2.3.1.4 $0.5\mathrm{mol / L}$ $\mathrm{pH8.7}$ 硼酸缓冲液：取硼酸 $30.9\mathrm{g}$ ，用氢氧化钠调整 $\mathrm{pH}$ 至8.7，用水定容至 $1000~\mathrm{mL}$ 。  
A.2.3.1.5 甘露糖标准品。  
A.2.3.1.6 半乳糖标准品。  
A.2.3.1.7 $10~\mu \mathrm{g / mL}$ 半乳糖 $+10~\mu \mathrm{g / mL}$ 甘露糖混合标准溶液：取半乳糖和甘露糖的标准液各 $0.1\mathrm{g}$ 、加水定容至 $25~\mathrm{mL}$ 、稀释400倍。  
A.2.3.1.8 $25\mu \mathrm{g / mL}$ 半乳糖 $+25~\mu \mathrm{g / mL}$ 甘露糖混合标准溶液：取半乳糖和甘露糖的标准液各 $0.1\mathrm{g}$ 、加水定容至 $25~\mathrm{mL}$ 、稀释160倍。  
A.2.3.1.9 $50~\mu \mathrm{g / mL}$ 半乳糖 $+50~\mu \mathrm{g / mL}$ 甘露糖混合标准溶液：取半乳糖和甘露糖的标准液各 $0.1\mathrm{g}$ 、加水定容至 $25~\mathrm{mL}$ 、稀释80倍。  
A.2.3.1.10 $0.06\mathrm{mol / L}$ L-精氨酸溶液：取 $10\mathrm{gL}$ -精氨酸，加水定容至 $1000~\mathrm{mL}$

# A.2.3.2 仪器和设备

A.2.3.2.1 高效液相色谱分析仪：配有荧光检测器。  
A.2.3.2.2 针头式过滤器：孔径 $0.45\mu \mathrm{m}$   
A.2.3.2.3 天平：感量 $0.1\mathrm{mg}$   
A.2.3.2.4 高压釜。  
A.2.3.2.5 密度测定仪。

# A.2.3.3 色谱参考条件

A.2.3.3.1 色谱柱：聚苯乙烯基强阴离子交换柱（粒径 $8\mu \mathrm{m}$ ，柱长 $150\mathrm{mm}\times$ 直径 $4.6\mathrm{mm}$ ）或同等性能的色谱柱。

A.2.3.3.2 流动相： $0.5\mathrm{mol / L},\mathrm{pH}8.7$ 硼酸缓冲液。  
A.2.3.3.3 柱温： $60~\mathrm{^\circ C}$   
A.2.3.3.4 流速： $0.4\mathrm{mL / min}$   
A.2.3.3.5 进样量： $20~\mu \mathrm{L}$   
A.2.3.3.6 荧光激发波长： $320~\mathrm{nm}$   
A.2.3.3.7 荧光发射波长： $430~\mathrm{nm}$   
A.2.3.3.8 柱后反应：反应液 $0.06\mathrm{mol / L}$ L-精氨酸溶液。

A.2.3.3.9 反应液流量： $0.7\mathrm{mL / min}$   
A.2.3.3.10 反应温度： $150~\mathrm{^\circ C}$

# A.2.3.4 分析步骤

# A.2.3.4.1 试样溶液的制备

称取 $0.6\mathrm{g}\sim 1.2\mathrm{g}$ 半乳甘露聚糖试样，精确至 $0.001\mathrm{g}$ ，加入 $11.99\mathrm{mol / L}$ 硫酸 $4\mathrm{mL}$ ，室温搅拌 $1\mathrm{h}$ ，然后加水 $112\mathrm{mL}$ ，置于高压釜 $(121^{\circ}\mathrm{C})$ 中 $1\mathrm{h}$ 。然后冷却至室温，用 $7.5\mathrm{mol / L}$ 氢氧化钠溶液中和至 $\mathrm{pH} = 7$ ，用水定容至 $200~\mathrm{mL}$ ，滤纸滤过，然后将滤液用针头式过滤器(孔径 $0.45\mu \mathrm{m}$ ) 滤过后备用。

# A.2.3.4.2 标准曲线的绘制

分别吸取 $20~\mu \mathrm{L} 10~\mu \mathrm{g / mL}$ 半乳糖 $+10~\mu \mathrm{g / mL}$ 甘露糖混合标准溶液、 $25~\mu \mathrm{g / mL}$ 半乳糖 $+25~\mu \mathrm{g / mL}$ 甘露糖混合标准溶液和 $50~\mu \mathrm{g / mL}$ 半乳糖 $+50~\mu \mathrm{g / mL}$ 甘露糖混合标准溶液注入液相色谱仪中，记录峰值，绘制标准曲线。

# A.2.3.4.3 测定

吸取 $20~\mu \mathrm{L}$ 试样溶液注入液相色谱仪中，并记录相应的色谱图谱，记录峰值。

# A.2.3.5 结果计算

试样中甘露糖和半乳糖含量的比值 $\omega_{3}$ ，按式(A.1)计算：

$$
w _ {3} = \frac {w _ {1}}{w _ {2}} \quad \dots \tag {A.1}
$$

$$
w_{1} = \frac{c_{1}\times V}{m_{1}}\times 100\%
$$

$$
w_{2} = \frac{c_{2}\times V}{m_{1}}\times 100\%
$$

式中：

$\varpi_{1}$ ——试样中甘露糖含量（质量分数）， $\%$ ；

$\omega_{2}$ ——试样中半乳糖含量（质量分数）， $\%$ ；

$c_{1}$ ——试样中甘露糖的浓度，单位为克每百毫升 $(\mathrm{g} / 100~\mathrm{mL})$

$c_{2}$ ——试样中半乳糖的浓度，单位为克每百毫升 $(\mathrm{g} / 100~\mathrm{mL})$

$V$ ——试样溶液的容量（定容后容量），单位为毫升 $(\mathrm{mL})$ ；

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

# A.2.3.6 结果判定

试样中甘露糖和半乳糖含量的比值 $w_{3} = 1:1 \sim 2.5:1$ ，则试样为半乳甘露聚糖。

# A. 3 相对分子质量

# A.3.1 试剂和材料

A.3.1.1 水：GB/T6682规定的二级水及以上规格。  
A.3.1.2 普鲁兰多糖标准品： $200\mathrm{K}$   
A.3.1.3 普鲁兰多糖标准溶液：称取 $1.000\mathrm{g}$ 普鲁兰多糖标准品，加水定容至 $100~\mathrm{mL}$

# A.3.2 仪器和设备

高效液相色谱分析仪：配有示差折光检测器。

# A.3.3 色谱参考条件

A.3.3.1 色谱柱：尺寸排阻色谱柱（粒径 $6\mu \mathrm{m}$ ，孔径 $45\mathrm{nm}$ ，柱长 $150\mathrm{mm}\times$ 直径 $6\mathrm{mm}$ ）或同等性能的色谱柱。  
A.3.3.2 色谱柱：尺寸排阻色谱(粒径 $4\mu \mathrm{m}$ ，孔径 $2.5\mathrm{nm}$ ，柱长 $150\mathrm{mm}\times$ 直径 $6\mathrm{mm}$ )或同等性能的色谱柱。  
A.3.3.3 流动相：水。  
A.3.3.4 柱温： $80^{\circ}C$ 。  
A.3.3.5 流速： $0.3\mathrm{mL / min}$   
A.3.3.6 进样量： $20~\mu \mathrm{L}$

# A.3.4 分析步骤

# A.3.4.1 试样溶液的制备

称取 $1.000\mathrm{g}$ 半乳甘露聚糖试样，加水定容至 $100~\mathrm{mL}$

# A.3.4.2 测定

将试样溶液和普鲁兰多糖标准溶液分别注入液相色谱仪，记录峰面积。

# A.3.5 结果表述

色谱图中，普鲁兰多糖该峰最高峰右侧的峰面积占所有峰面积的 $80\%$ 以上，即可判定试样 $80\%$ 以上的相对分子质量分布 $\leqslant 200000$ 。

# A.4 黏度的测定

# A.4.1 仪器和设备

布氏黏度计。

# A.4.2 测定条件

A.4.2.1 转子型号：低速转子。  
A.4.2.2 转速： $30\mathrm{r / min}$ （系数0.2）。  
A.4.2.3 测定温度： $5\%$   
A.4.2.4 测定时间： $60~\mathrm{s}$ □

# A.4.3 分析步骤

准确称取 $5.000\mathrm{g}$ 试样，边搅拌边用水稀释到 $100\mathrm{g}$ 。完全溶解后，冷却至 $5^{\circ}\mathrm{C}$ ，用布氏黏度计测定黏度。

# A.5 酸不溶物的测定

# A.5.1 试剂和材料

A.5.1.1 硅藻土：经 $105\,^{\circ}\mathrm{C},4\,\mathrm{h}$ 干燥处理。  
A.5.1.2 硫酸。

# A.5.2 仪器和设备

A.5.2.1 电热恒温水浴。  
A.5.2.2 电热恒温干燥箱。  
A.5.2.3 古氏坩埚（经 $105^{\circ}\mathrm{C},4\mathrm{h}$ 干燥处理）。  
A.5.2.4 干燥器。

# A.5.3 分析步骤

称取试样 $2\mathrm{g}$ ，精确至 $0.001\mathrm{g}$ ，溶于一盛有 $150~\mathrm{mL}$ 水和 $1.5\mathrm{mL}$ 硫酸的 $250~\mathrm{mL}$ 烧杯中。用表面皿盖住烧杯，在沸水浴上加热 $6\mathrm{h}$ ，加热过程中用玻璃棒经常摩擦烧杯内壁。加热完后，称取硅藻土 $0.5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ），加入到试样中，使用已知质量的古氏坩埚进行过滤。用 $50^{\circ}\mathrm{C}$ 温水洗涤滤渣3次～5次，每次 $25~\mathrm{mL}$ ，将坩埚连同内容物于 $105^{\circ}\mathrm{C}$ 下干燥 $4\mathrm{h}$ ，在干燥器内冷却后称量。

# A.5.4 结果计算

酸不溶物的质量分数 $\varpi_{4}$ ，按式(A.2)计算：

$$
w _ {4} = \frac {m _ {2} - m _ {3} - m _ {4}}{m _ {5}} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$m_{2}$ ——最终称量的总质量，单位为克 $(\mathrm{g})$ ；

$m_{3}$ ——硅藻土的质量，单位为克 $(\mathrm{g})$ ；

$m_{4}$ ——空坩埚的质量，单位为克(g)；

$m_{5}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

实验结果以平行测定结果的算术平均值为准(保留一位小数)。在重复性条件下获得的2次独立测定结果的绝对差值不大于 $0.2\%$ 。

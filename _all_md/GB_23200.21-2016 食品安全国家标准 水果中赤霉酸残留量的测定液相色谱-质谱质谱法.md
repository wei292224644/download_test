# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T0350—2012《出口水果中赤霉素残留量的测定液相色谱—质谱/质谱法》。与SN/T0350—2012相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；

——标准名称中“进出口水果”改为“水果”。

——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN 0350—1995、SN/T 0350—2012。

# 食品安全国家标准

# 水果中赤霉酸残留量的测定 液相色谱一质谱/质谱法

# 1 范围

本标准规定了水果中赤霉酸残留量的制样和液相色谱一质谱/质谱测定方法。

本标准适用于苹果、橘子、桃子、梨和葡萄中赤霉酸残留量的检测，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682 分析实验室用水规格和试验方法。

# 3 原理

用乙腈提取试样中残留的赤霉酸，提取液经液液分配净化后，用液相色谱一质谱/质谱测定和确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ ：色谱纯。  
4.1.2 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
4.1.3 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。  
4.1.4 甲酸 $(\mathrm{CH}_2\mathrm{O}_2)$ ：色谱纯。  
4.1.5 磷酸二氢钾 $\mathrm{(K_2HPO_4)}$   
4.1.6 氢氧化钠(NaOH)。  
4.1.7 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。  
4.1.8 氯化钠 $(\mathrm{NaCl})$ 。

# 4.2 溶液配制

4.2.1 硫酸水溶液 $(\mathrm{pH}2.5)$ ：1滴硫酸加入 $100~\mathrm{mL}$ 水中，调节水 $\mathrm{pH}$ 为2.5。  
4.2.2 磷酸盐缓冲溶液 $(\mathrm{pH}7):6.7\mathrm{g}$ 磷酸二氢钾和 $1.2\mathrm{g}$ 氢氧化钠溶解于 $1\mathrm{L}$ 水中。  
4.2.3 $0.15\%$ 甲酸溶液：移取 $0.15\mathrm{mL}$ 甲酸，用水稀释至 $100~\mathrm{mL}$

# 4.3 标准品

赤霉酸标准品(gibberellicacid，CAS号：77-06-5， $\mathrm{C_{19}H_{22}O_6}$ ）：纯度 $\geqslant 98\%$

# 4.4 标准溶液配制

4.4.1 赤霉酸标准储备溶液：称取适量标准品，用甲醇溶解，溶液浓度为 $100\mu \mathrm{g / mL}$ 。 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冷藏避光保存。有效期3个月。

4.4.2 标准工作溶液：根据需要，用空白样品溶液将标准储备液稀释成 $4\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},$ $100~\mathrm{ng / mL}$ 和 $150~\mathrm{ng / mL}$ 的标准工作溶液，相当于样品中含有 $8\mu \mathrm{g} / \mathrm{kg},10\mu \mathrm{g} / \mathrm{kg},20\mu \mathrm{g} / \mathrm{kg},200\mu \mathrm{g} / \mathrm{kg}$ 和 $300~\mu \mathrm{g} / \mathrm{kg}$ 赤霉酸。临用前配制。

# 4.5 材料

有机相微孔滤膜： $0.45 \mu \mathrm{m}$ 。

# 5 仪器和设备

5.1 液相色谱—质谱/质谱仪：配有电喷雾离子源。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 pH计。  
5.4 旋转蒸发器。  
5.5 涡旋振荡器。  
5.6 离心机： $4000\mathrm{r / min}$

# 6 试样制备与保存

从所取全部样品中取出有代表性样品约 $500 \mathrm{~g}$ , 取样部位按 GB 2763 的规定执行。用粉碎机粉碎, 混合均匀, 均分成 2 份, 分别装入洁净容器作为试样, 密封, 并标明标记。将试样于 $-18^{\circ} \mathrm{C}$ 冷冻保存。

在抽样和制样的操作过程中，应防止样品污染或发生残留物含量的变化。

# 7 测定步骤

# 7.1 提取

称取 $5 \mathrm{~g}$ 试样(精确到 $0.01 \mathrm{~g}$ ) 置于 $50 \mathrm{~mL}$ 塑料离心管中, 加入 $25 \mathrm{~mL}$ 乙腈和 $2 \mathrm{~g}$ 氯化钠, 涡旋 $1 \mathrm{~min}$ , 以 $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 将上层乙腈提取液转移至浓缩瓶中。下层溶液再用 $20 \mathrm{~mL}$ 乙腈提取一次, 合并乙腈提取液, 在 $45^{\circ} \mathrm{C}$ 以下水浴减压浓缩至近干。用 $10 \mathrm{~mL}$ 硫酸水溶液将残渣转移至 $50 \mathrm{~mL}$ 塑料离心管中, 加入 $20 \mathrm{~mL}$ 乙酸乙酯, 涡旋 $1 \mathrm{~min}$ , 以 $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 乙酸乙酯转移至另一 $50 \mathrm{~mL}$ 塑料离心管中, 再加入 $20 \mathrm{~mL}$ 乙酸乙酯, 重复上述操作, 合并乙酸乙酯提取液。加入 $10 \mathrm{~mL}$ 磷酸盐缓冲溶液, 涡旋, 以 $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 分取磷酸盐缓冲盐溶液, 乙酸乙酯层中再加入 $10 \mathrm{~mL}$ 磷酸盐缓冲溶液提取一次, 合并磷酸盐缓冲溶液, 滴加 $50 \%$ 硫酸溶液调节溶液 $\mathrm{pH}$ 为 $2.5 \pm 0.2$ 。加入 $20 \mathrm{~mL}$ 乙酸乙酯, 涡旋 $1 \mathrm{~min}$ , 以 $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 将上层乙酸乙酯转移至浓缩瓶中, 磷酸盐缓冲盐溶液层中再加入 $20 \mathrm{~mL}$ 乙酸乙酯提取一次, 合并乙酸乙酯提取液在 $45^{\circ} \mathrm{C}$ 以下水浴减压浓缩至近干。加 $10.0 \mathrm{~mL}$ 甲醇一水 (1+1, 体积比) 溶解残渣, 混匀, 过 $0.45 \mu \mathrm{m}$ 滤膜, 供液相色谱一质谱/质谱仪测定。

# 7.2 测定

# 7.2.1 液相色谱一质谱/质谱

液相色谱—质谱/质谱参考条件如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱， $150\mathrm{mm}\times 4.6\mathrm{mm}$ （内径），膜厚 $5\mu \mathrm{m}$ ，或相当者；  
b) 流动相: 乙腈-0.15%甲酸水溶液(35+65,体积比);  
c) 流速: $0.4 \mathrm{~mL} / \mathrm{min}$ ;   
d）进样量： $30~\mu \mathrm{L}$   
e) 离子源: 电喷雾离子源;  
f) 扫描方式: 负离子扫描;  
g）检测方式：多反应监测；  
h）雾化气、气帘气、辅助气和碰撞气均为高纯氮气；使用前，应调节各气体流量以使质谱灵敏度达

到检测要求，参考条件参见表A.1。

# 7.2.2 液相色谱一质谱/质谱测定

根据样液中赤霉酸的含量情况，选定响应值适宜的标准工作液进行色谱分析，标准工作液应有5个浓度水平。待测样液中赤霉酸的响应值均应在仪器检测的工作曲线范围内。在上述色谱条件下，赤霉酸的参考保留时间约为 $4.9\mathrm{min}$ 。标准溶液的选择性离子流图参见图B.1。

# 7.2.3 液相色谱一质谱/质谱确证

按照上述条件测定样品和标准工作液，如果检测的质量色谱峰保留时间与标准工作液一致，允许偏差小于 $\pm 2.5\%$ ；定性离子对的相对丰度与浓度相当标准工作液的相对丰度一致，相对丰度允许偏差不超过表1规定，则可判断样品中存在相应的被测物。

# 7.3 空白试验

除不加试样外，均按 $7.1\sim 7.2$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中赤霉酸残留含量。

$$
X = \frac {C _ {i} \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中赤霉酸的残留量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_i$ ——从标准曲线上得到的赤霉酸浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——最终样液代表的试样质量，单位为克(g)；

1000——换算系数。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录C的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录 D 的要求。

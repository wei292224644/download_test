# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T1737.3—2010《除草剂残留量检验方法 第3部分：液相色谱一质谱/质谱法测定进出口食品中环己烯酮类除草剂残留量》。与SN/T1737.3—2010相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“进出口食品”改为“食品”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T1737.3—2010。

# 1 范围

本标准规定了食品中吡喃草酮、禾草灭、噻草酮、烯草酮、烯禾啶、丁苯草酮、三甲苯草酮、环苯草酮残留量的液相色谱一质谱/质谱测定方法。

本标准适用于大米、大豆、猪肉、牛肝、鸡肝、牛奶、橙、蓝莓、菠菜、洋葱、核桃仁、茶叶中吡喃草酮、禾草灭、噻草酮、烯草酮、烯禾啶、丁苯草酮、三甲苯草酮、环苯草酮残留量的测定和确证，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

试样中残留的环己烯酮类除草剂用酸性乙腈或乙腈提取，提取液经N-丙基乙二胺(PSA)、十八烷基硅烷(ODS)和石墨化炭黑净化，用液相色谱一质谱/质谱仪检测和确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

4.1.2 乙酸 $\mathrm{(CH_3COOH)}$ (99.5%)。

4.1.3 甲酸（HCOOH） $(88.0\%)$

4.1.4 乙酸钠 $\mathrm{(CH_3COONa)}$

4.1.5 氯化钠(NaCl)。

4.1.6 无水硫酸镁 $(\mathrm{MgSO_4})$ ：经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置于干燥器内备用。

# 4.2 溶液配制

4.2.1 $1\%$ 乙酸乙腈溶液：取 $10~\mathrm{mL}$ 乙酸，加入 $990~\mathrm{mL}$ 乙腈，混匀。  
4.2.2 $60\%$ 乙腈水溶液：取 $60~\mathrm{mL}$ 乙腈，加入 $40~\mathrm{mL}$ 水，混匀。  
4.2.3 甲酸—水溶液 $(0.2 + 999.8$ ，体积比)：取 $0.2\mathrm{mL}$ 甲酸，用水溶解并稀释至 $1000~\mathrm{mL}$ 。

# 4.3 标准品

环己烯酮类标准品：烯草酮(Clethodim)，CAS号：99129-21-2，纯度 $\geqslant 99\%$ ；吡喃草酮(Te-praloxydim)，CAS号：149979-41-9，纯度 $\geqslant 99\%$ ；噻草酮(Cycloxydim)，CAS号：101205-02-1，纯度 $\geqslant$ $93.5\%$ ；三甲苯草酮(Tralkoxydim)，CAS号：87820-88-0，纯度 $\geqslant 93.5\%$ ；禾草灭(Alloxym)，CAS号： $55634 - 91 - 8$ ，纯度 $\geqslant 98\%$ ；环苯草酮(Profoxydim)，CAS号：139001-49-3，纯度 $\geqslant 98.5\%$ ；丁苯草酮

(Butroxydim), CAS号:138164-12-2;纯度 $\geqslant 99\%$ ;烯禾啶(Sethoxydim), CAS号:74051-80-2;纯度 $\geqslant$ $99\%$ 。

# 4.4 标准溶液配制

4.4.1 环己烯酮类标准储备溶液：准确称取适量环己烯酮类的标准品，用乙腈溶解并定容至 $100\mathrm{mL}$ 棕色容量瓶中，得浓度为 $100\mu \mathrm{g / mL}$ 的标准储备溶液，此溶液避光储存。  
4.4.2 混合标准中间溶液：准确吸取环己烯酮类的标准储备溶液各 $2.5\mathrm{mL}$ 于 $50~\mathrm{mL}$ 棕色容量瓶中，用乙腈稀释成浓度为 $5.0\mu \mathrm{g / mL}$ 的混合标准中间溶液，此溶液避光储存。  
4.4.3 混合标准工作溶液：根据需要使用前用空白样品基质配制适当混合标准工作溶液。

# 4.5 材料

4.5.1 石墨化炭黑吸附剂： $120\mu \mathrm{m}\sim 400\mu \mathrm{m},$   
4.5.2 N-丙基乙二胺(PSA)吸附剂: $40 \mu \mathrm{m} \sim 100 \mu \mathrm{m}$ 。  
4.5.3 十八烷基硅烷(ODS)键合相： $40\mu \mathrm{m}\sim 100\mu \mathrm{m}$   
4.5.4 水相滤膜： $0.22\mu \mathrm{m}$

# 5 仪器和设备

5.1 液相色谱—质谱/质谱联用仪：配ESI电离源。  
5.2 电子天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 粉碎机。  
5.4 组织捣碎机。  
5.5 离心机： $4000\mathrm{r / min}$   
5.6 涡旋振荡器。  
5.7 氮吹仪。  
5.8 均质器： $10000\mathrm{r / min}$   
5.9 螺旋盖聚丙烯离心管： $50~\mathrm{mL},15~\mathrm{mL}$ 。  
5.10 玻璃具塞离心管： $15~\mathrm{mL}$

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按 GB 2763 的规定执行。在取样和制样的操作过程中,应防止样品污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 橙、蓝莓、菠菜、洋葱

取有代表性样品 $500\mathrm{g}$ (不可用水洗涤), 将其切碎后, 用组织捣碎机将样品加工成浆状, 装入洁净容器作为试样, 密封并标明标记。

# 6.2.2 大豆、大米、茶叶、核桃仁

取有代表性样品 $500\mathrm{g}$ ，用粉碎机将样品粉碎，混匀，装入洁净容器作为试样，密封并标明标记。

# 6.2.3 猪肉、牛肝、鸡肝

取有代表性样品 $500\mathrm{g}$ , 切碎后, 用组织捣碎机将样品充分捣碎均匀, 装入洁净容器作为试样, 密封并标明标记。

# 6.3 试样保存

大豆、大米、茶叶、核桃仁和牛奶等试样于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存；橙、蓝莓、菠菜、洋葱、猪肉、牛肝、鸡肝等试样于 $-18^{\circ}\mathrm{C}$ 以下保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 大豆、大米、茶叶和核桃仁

称取试样 $5\mathrm{g}$ （茶叶试样 $2\mathrm{g}$ ，精确到 $0.01\mathrm{g}$ ）于 $50\mathrm{mL}$ 螺旋盖聚丙烯离心管中，加入 $10\mathrm{mL}$ 水，于涡旋振荡器上混合 $30\mathrm{s}$ ，放置 $15\mathrm{min}$ 。加入 $3\mathrm{g}$ 无水硫酸镁、 $1\mathrm{g}$ 氯化钠、 $1\mathrm{g}$ 乙酸钠和 $15\mathrm{mL}1\%$ 乙酸乙腈溶液，用均质器以 $10000\mathrm{r / min}$ 均质 $2\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。将上清液转移至 $25\mathrm{mL}$ 容量瓶中。再用 $10\mathrm{mL}1\%$ 乙酸乙腈溶液重复提取一次，合并提取液于同一 $25\mathrm{mL}$ 容量瓶中，并用乙腈定容至刻度，待净化。

# 7.1.2 猪肉、牛肝、鸡肝、牛奶、橙、菠菜和洋葱

称取试样约 $5\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 于 $50\mathrm{mL}$ 螺旋盖聚丙烯离心管中，加入 $5\mathrm{mL}$ 水、 $3\mathrm{g}$ 无水硫酸镁、 $1\mathrm{g}$ 氯化钠、 $1\mathrm{g}$ 乙酸钠和 $15\mathrm{mL}1\%$ 乙酸乙腈溶液，用均质器以 $10000\mathrm{r / min}$ 均质 $2\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。将上清液转移至 $25\mathrm{mL}$ 容量瓶中。再用 $10\mathrm{mL}1\%$ 乙酸乙腈溶液重复提取一次，合并提取液于同一 $25\mathrm{mL}$ 容量瓶中，并用乙腈定容至刻度，待净化。

# 7.1.3 蓝莓

称取试样约 $5\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 螺旋盖聚丙烯离心管中，加入 $5\mathrm{mL}$ 水、 $3\mathrm{g}$ 无水硫酸镁、 $1\mathrm{g}$ 氯化钠、 $1\mathrm{g}$ 乙酸钠和 $15~\mathrm{mL}$ 乙腈，用均质器以 $10000\mathrm{r / min}$ 均质 $2\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。将上清液转移至 $25~\mathrm{mL}$ 容量瓶中。再用 $10~\mathrm{mL}$ 乙腈重复提取一次，合并提取液于同一 $25~\mathrm{mL}$ 容量瓶中，并用乙腈定容至刻度，待净化。

# 7.2 净化

# 7.2.1 大豆、大米、核桃仁、猪肉、牛肝、牛奶、洋葱

移取 $8\mathrm{mL}$ 7.1.1或7.1.2中的提取液于 $15\mathrm{mL}$ 螺旋盖聚丙烯离心管中，加入 $300\mathrm{mg}$ 无水硫酸镁、 $250\mathrm{mg}$ PSA和 $500\mathrm{mg}$ ODS，在涡旋振荡器上混合 $2\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。准确移取 $5\mathrm{mL}$ 净化液于 $15\mathrm{mL}$ 玻璃具塞离心管中，经 $60^{\circ}\mathrm{C}$ 氮吹仪吹干后，用 $60\%$ 乙腈水溶液溶解并定容 $1.0\mathrm{mL}$ ，过0.22 $\mu \mathrm{m}$ 滤膜供液相色谱一质谱/质谱仪测定。

# 7.2.2 鸡肝、橙、菠菜、蓝莓、茶叶

移取 $8\mathrm{mL}7.1.1\sim 7.1.3$ 中的提取液于 $15~\mathrm{mL}$ 螺旋盖聚丙烯离心管中，加入 $300\mathrm{mg}$ 无水硫酸镁、 $250\mathrm{mgPSA},500\mathrm{mgODS}$ 和 $10\mathrm{mg}$ 石墨化炭黑，在涡旋振荡器上混合 $2\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。准确移取 $5\mathrm{mL}$ 净化液于 $15\mathrm{mL}$ 玻璃具塞离心管中，经 $60^{\circ}\mathrm{C}$ 氮吹仪吹干后，用 $60\%$ 乙腈水溶液溶解并定容 $1.0\mathrm{mL}$ ，过 $0.22\mu \mathrm{m}$ 滤膜供液相色谱一质谱/质谱仪测定。

7.4 空白试验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

8 结果计算和表达

用 LC-MS/MS 色谱数据处理机或按式(1)计算试样中环己烯酮类残留含量。

$$
X _ {i} = \frac {A _ {i} \times C _ {s} \times V}{A _ {s} \times m} \dots \tag {1}
$$

式中：

$X_{i}$ ——试样中环己烯酮类残留量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A_{i}$ ——样液中环己烯酮类的峰面积；

$C_{s}$ ——标准工作液中环己烯酮类的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{si}$ ——标准工作液中环己烯酮类的峰面积；

$m$ ——最终样液所代表的试样质量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

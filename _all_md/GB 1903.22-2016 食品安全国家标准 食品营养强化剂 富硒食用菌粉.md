# 1 范围

本标准适用于以食用菌为载体，经发酵培养将培养基中含有的亚硒酸钠转化为有机态硒，再经粉碎、干燥制得的食品营养强化剂富硒食用菌粉。

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度应为分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 有机硒占总硒质量百分比的测定

# A.2.1 试剂和溶液

A.2.1.1 硝酸：优级纯。  
A.2.1.2 高氯酸：优级纯。  
A.2.1.3 盐酸：优级纯。  
A.2.1.4 氢氧化钠：优级纯。  
A.2.1.5 混合酸：硝酸 $+$ 高氯酸 $(4 + 1)$   
A.2.1.6 盐酸溶液： $6\mathrm{mol} / \mathrm{L}$   
A.2.1.7 硼氢化钠溶液: $8 \mathrm{~g} / \mathrm{L}$ 。称取 $8.0 \mathrm{~g}$ 硼氢化钠 $\left(\mathrm{NaBH}_{4}\right)$ , 溶于氢氧化钠溶液 $(5 \mathrm{~g} / \mathrm{L})$ 中, 定容至 $1000 \mathrm{~mL}$ 。  
A.2.1.8 铁氰化钾溶液: $100 \mathrm{~g} / \mathrm{L}$ 。称取 $10.0 \mathrm{~g}$ 铁氰化钾 $\left[\mathrm{K}_{3} \mathrm{Fe}(\mathrm{CN})_{6}\right]$ , 用适量蒸馏水溶解后定容成 $100 \mathrm{~mL}$ , 混匀。  
A.2.1.9 硒标准储备液：精确称取 $100.0\mathrm{mg}$ 硒(光谱纯)，溶于少量硝酸中，加 $2\mathrm{mL}$ 高氯酸，置沸水浴中加热 $3\mathrm{h} \sim 4\mathrm{h}$ ，冷却后再加 $8.4\mathrm{mL}$ 盐酸，再置沸水浴中煮 $2\mathrm{min}$ ，定容至 $1000\mathrm{mL}$ ，此储备液浓度为每毫升相当于 $100\mu \mathrm{g}$ 础。  
A.2.1.10 硒标准实验液：取 $100\mu \mathrm{g / mL}$ 础标准储备液 $1.0~\mathrm{mL}$ ，用水定容至 $100~\mathrm{mL}$ ，此实验液浓度为 $1\mu \mathrm{g / mL}$ 。

# A.2.2 仪器和设备

A.2.2.1 AFS-230双道原子荧光光度计或同类仪器。  
A.2.2.2 电热板。  
A.2.2.3 自动控温消化炉。

# A.2.3 无机硒测定的试样制备

称取约 $1\mathrm{g}$ 试样，准确至 $0.0001\mathrm{g}$ ，于 $50~\mathrm{mL}$ 烧杯中，加入 $25~\mathrm{mL}$ 水，在磁力搅拌器上搅拌 $30\mathrm{min}$ ，使试样充分溶解，转移到 $100~\mathrm{mL}$ 容量瓶，加水定容，摇匀，静置于冰箱 $(4^{\circ}\mathrm{C})1\mathrm{h}$ 。将定容后的溶液于 $5000\mathrm{r / min}$ 离心 $30\mathrm{min}$ ，同时做空白试验。取上清液 $10~\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，分别加入 $12.5~\mathrm{mL}$ 盐酸、 $2.5~\mathrm{mL}$ 铁氰化钾溶液，定容，混匀待测。

# A.2.4 分析步骤

按照GB5009.93氢化物原子荧光光谱法测定无机硒含量，再由之前得到的总硒含量减去无机硒含

量得到有机硒含量，从而计算出有机硒的百分含量。

# A.2.5 结果计算

A.2.5.1 无机硒含量 $X_{1}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.1)计算：

$$
X _ {1} = \frac {\left(c _ {2} - c _ {1}\right) \times 1 0 0 \times 1 0 0 0}{m _ {1} \times 1 0 0 0 \times 1 0 0 0} \quad \dots \tag {A.1}
$$

式中：

$c_{2}$ ——试样提取液测定浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{1}$ ——无机硒空白测定浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

100 ——试样的定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m_{1}$ ——无机硒测定时称取的试样质量，单位为克 $(\mathrm{g})$ 。

结果保留3位有效数字。

A.2.5.2 有机硒含量 $X_{2}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.2)计算：

$$
X _ {2} = X - X _ {1} \quad \dots \tag {A.2}
$$

式中：

$X$ ——试样中总硒的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$X_{1}$ ——试样中无机硒的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 。

A.2.5.3 有机硒占总硒含量的质量分数 $\omega_{1}$ ，按式(A.3)计算：

$$
w _ {1} = \frac {X _ {2}}{X} \times 100 \% \tag{A.3}
$$

式中：

$X_{2}$ ——试样中有机硒的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$X$ ——试样中总硒的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 。

试验结果以平行测定结果的算术平均值为准。结果保留至小数点后1位。

# 前言

本标准代替GB5009.240—2016《食品安全国家标准 食品中伏马毒素的测定》。

本标准与GB5009.240—2016相比，主要变化如下：

——标准名称修改为《食品安全国家标准 食品中伏马菌素的测定》；  
扩大了方法的适用范围；  
——修改了各方法的线性范围；  
——修改了各方法的检出限、定量限；  
——修改了柱容量及柱回收率验证方法。

# 食品安全国家标准

# 食品中伏马菌素的测定

# 1 范围

本标准规定了食品中伏马菌素的免疫亲和柱净化-柱后衍生高效液相色谱、高效液相色谱-串联质谱以及免疫亲和柱净化-柱前衍生高效液相色谱测定方法。

本标准适用于谷物及其制品、婴幼儿谷类辅助食品、植物油脂中伏马菌素 $\mathrm{B}_{1}$ 、伏马菌素 $\mathrm{B}_{2}$ 、伏马菌素 $\mathrm{B}_{3}$ （以下简称 $\mathrm{FB}_{1} 、 \mathrm{FB}_{2} 、 \mathrm{FB}_{3}$ ) 的测定。

# 第一法 免疫亲和柱净化-柱后衍生高效液相色谱法

# 2 原理

试样经提取，免疫亲和柱净化， $\mathrm{C}_{18}$ 反相色谱柱分离，邻苯二甲醛衍生，荧光检测器检测，外标法定量。

3.2.6 磷酸盐缓冲液(PBS)：称取 $8.0\mathrm{g}$ 氯化钠、 $1.2\mathrm{g}$ 磷酸氢二钠、 $0.2\mathrm{g}$ 磷酸二氢钾、 $0.2\mathrm{g}$ 氯化钾，用 $980~\mathrm{mL}$ 水溶解，用盐酸调整 $\mathrm{pH}$ 至7.4，用水稀释至 $1000~\mathrm{mL}$ ，混合均匀。

# 5 分析步骤

# 5.1 试样制备

谷物及其制品、婴幼儿谷类辅助食品采样量不低于 $1\mathrm{kg}$ ；样品量不足 $1\mathrm{kg}$ ，取全部试样。用高速粉碎机将其粉碎，粉碎粒度应小于 $1\mathrm{mm}$ 。混合均匀后储存于洁净的容器内，密封后置于 $4^{\circ}\mathrm{C}$ 下避光保存。

植物油脂采样量不低于 $1\mathrm{kg}$ ；样品量不到 $1\mathrm{kg}$ ，取全部试样。混合均匀后储存于洁净的容器内，密封后置于 $4^{\circ}\mathrm{C}$ 下避光保存。

在制样的操作过程中，应防止样品受到污染或发生伏马菌素含量的变化。

# 5.2 试样提取

# 5.2.1 谷物及其制品、婴幼儿谷类辅助食品

准确称取试样 $20\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）于 $250~\mathrm{mL}$ 三角烧瓶中，准确加入 $100~\mathrm{mL}$ 乙腈-水 $(50 + 50)$ 提取液，超声或振荡提取 $20\mathrm{min}$ ，转移 $20~\mathrm{mL}$ 提取液至 $50~\mathrm{mL}$ 离心管中，在 $4000~\mathrm{r / min}$ 下离心 $10\mathrm{min}$ ，待净化。

# 5.2.2 植物油脂

植物油脂提取过程按5.2.1步骤操作，提取液为下层。

# 5.3 试样净化

# 5.3.1 婴幼儿谷类辅助食品

准确移取 $5.0\mathrm{mL}$ 提取液，加入 $45.0\mathrm{mL}$ 吐温-20/PBS溶液进行稀释，混合均匀后在 $4000\mathrm{r / min}$ 下离心 $10\mathrm{min}$ ，取上清液全部过免疫亲和柱，流速控制 $1\mathrm{mL} / \mathrm{min}\sim 2\mathrm{mL} / \mathrm{min}$ ，然后用 $10\mathrm{mL}$ PBS缓冲液和 $10\mathrm{mL}$ 水依次淋洗免疫亲和柱，再用 $3\mathrm{mL}$ 甲醇-乙酸溶液分3次洗脱免疫亲和柱，收集合并洗脱液， $55^{\circ}\mathrm{C}$ 下氮气吹干，加入 $1\mathrm{mL}$ 乙腈-水溶液 $(20 + 80)$ 溶解残渣，涡旋 $30~s$ ，过微孔滤膜后，收集于进样瓶中，待测。

# 5.3.2 谷物及其制品、植物油脂

准确移取 $0.5\mathrm{mL}$ 提取液，加入 $4.5\mathrm{mL}$ 吐温-20/PBS溶液进行稀释，混合均匀后在 $4000\mathrm{r / min}$ 下离心 $10\mathrm{min}$ ，上清液全部过免疫亲和柱，流速控制 $1\mathrm{mL} / \mathrm{min}\sim 2\mathrm{mL} / \mathrm{min}$ ，然后依次用 $10\mathrm{mL}$ PBS缓冲液和 $10\mathrm{mL}$ 水淋洗免疫亲和柱，再用 $3\mathrm{mL}$ 甲醇-乙酸溶液分3次洗脱免疫亲和柱，收集合并洗脱液。

$55~\mathrm{^\circ C}$ 下氮气吹干，加入 $1\mathrm{mL}$ 乙腈-水溶液 $(20 + 80)$ 溶解残渣，涡旋 $30~\mathrm{s}$ ，过微孔滤膜后，收集于进样瓶中，待测。

注：由于不同厂商提供的免疫亲和柱操作程序可能不同，实际操作时，请参照厂商提供的操作说明和程序使用。

# 5.5 标准曲线的制作

将混合标准系列工作液从低浓度到高浓度分别注入高效液相色谱仪中，测定相应的峰面积。以标准溶液浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。伏马菌素标准溶液的柱后衍生-高效液相色谱图参见附录A。

# 5.6 试样溶液的测定

将待测溶液进样，得到对应的峰面积，根据标准曲线计算待测溶液中 $\mathrm{FB}_1$ 、 $\mathrm{FB}_2$ 、 $\mathrm{FB}_3$ 的浓度。

# 5.7 空白试验

不称取试样，按5.2和5.3的步骤做空白试验。

# 6 分析结果的表述

待测样品中 $\mathrm{FB}_1$ 、 $\mathrm{FB}_2$ 、 $\mathrm{FB}_3$ 的含量按式(1)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V _ {1} \times V _ {2} \times 1 0 0 0}{m \times V _ {3} \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——待测样品中 $\mathrm{FB}_1, \mathrm{FB}_2, \mathrm{FB}_3$ 的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——待测溶液中 $\mathrm{FB}_1, \mathrm{FB}_2, \mathrm{FB}_3$ 的质量浓度，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$\rho_0$ ——空白试验测得 $\mathrm{FB}_1, \mathrm{FB}_2, \mathrm{FB}_3$ 的质量浓度，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$V_{1}$ ——提取溶液的体积，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——进样溶液的定容体积，单位为毫升 $(\mathrm{mL})$

1000——换算系数；

$m$ ——样品的称样量，单位为克 $(\mathrm{g})$ ；

$V_{3}$ ——用于净化的提取溶液体积，单位为毫升 $(\mathrm{mL})$ 。

测定结果用平行测定的算术平均值表示，保留3位有效数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

谷物及其制品、植物油脂，当称样量为 $20.0\mathrm{g}$ 时， $\mathrm{FB}_1$ 、 $\mathrm{FB}_2$ 和 $\mathrm{FB}_3$ 的检出限分别为 $60.0~\mu \mathrm{g / kg}$ 、 $30.0~\mu \mathrm{g / kg}$ 和 $30.0~\mu \mathrm{g / kg}$ ，定量限分别为 $200~\mu \mathrm{g / kg}$ 、 $100~\mu \mathrm{g / kg}$ 和 $100~\mu \mathrm{g / kg}$ 。

婴幼儿谷类辅助食品，当称样量为 $20.0\mathrm{g}$ 时， $\mathrm{FB_1,FB_2}$ 和 $\mathrm{FB_3}$ 的检出限分别为 $6.00~\mu \mathrm{g / kg},3.00~\mu \mathrm{g / kg}$ 和 $3.00~\mu \mathrm{g / kg}$ ，定量限分别为 $20.0~\mu \mathrm{g / kg},10.0~\mu \mathrm{g / kg}$ 和 $10.0~\mu \mathrm{g / kg}$ 。

# 13 分析结果的表述

本方法采用内标法定量。待测样品中 $\mathrm{FB_1}$ 、 $\mathrm{FB_2}$ 、 $\mathrm{FB_3}$ 的含量按式(2)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V _ {1} \times V _ {2} \times 1 0 0 0}{m \times V _ {3} \times 1 0 0 0} \tag {2}
$$

式中：

$X$ —待测样品中 $\mathrm{FB}_1, \mathrm{FB}_2, \mathrm{FB}_3$ 的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$\rho$ ——按照内标法在标准曲线中对应的 $\mathrm{FB}_1, \mathrm{FB}_2, \mathrm{FB}_3$ 的质量浓度，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$\rho_0$ ——空白试验测得 $\mathrm{FB}_1, \mathrm{FB}_2, \mathrm{FB}_3$ 的质量浓度，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$V_{1}$ ——提取溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——进样溶液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——样品的称样量，单位为克 $(\mathrm{g})$ ；

$V_{3}$ ——用于净化的提取溶液体积，单位为毫升 $(\mathrm{mL})$ 。

测定结果用平行测定的算术平均值表示，保留3位有效数字。

# 20 分析结果的表述

待测样品中 $\mathrm{FB}_1$ 、 $\mathrm{FB}_2$ 、 $\mathrm{FB}_3$ 的含量按式(3)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V _ {1} \times V _ {2} \times 1 0 0 0}{m \times V _ {3} \times 1 0 0 0} \tag {3}
$$

式中：

$X$ —待测样品中 $\mathrm{FB}_1\text{、}\mathrm{FB}_2\text{、}\mathrm{FB}_3$ 的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$\rho$ ——待测溶液中 $\mathrm{FB}_1, \mathrm{FB}_2, \mathrm{FB}_3$ 的浓度，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$\rho_0$ ——空白试验测得 $\mathrm{FB}_1, \mathrm{FB}_2, \mathrm{FB}_3$ 的浓度，单位为纳克每毫升 $(\mathrm{ng/mL})$ ；

$V_{1}$ ——提取溶液的体积，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——进样溶液的定容体积，单位为毫升 $(\mathrm{mL})$

1000——换算系数；

$m$ ——样品的称样量，单位为克 $(\mathrm{g})$ ；

$V_{3}$ ——用于净化的提取溶液体积，单位为毫升 $(\mathrm{mL})$ 。

测定结果用平行测定的算术平均值表示，保留3位有效数字。

# 21 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 柱容量及柱回收率验证方法

# D.1 柱容量验证方法

准确移取 $\mathrm{FB}_{1}$ 标准储备溶液 $(100\mu \mathrm{g / mL})250\mu \mathrm{L}$ 至 $50~\mathrm{mL}$ 容量瓶中，用吐温-20/PBS溶液 $(0.1\%)$ 定容至刻度，充分混匀。分别取同一批次3根免疫亲和柱，每根柱的上样量为 $10~\mathrm{mL}(5\mu \mathrm{g}$ $\mathrm{FB}_{1})$ 。经上样、淋洗、洗脱，收集洗脱液，用氮气吹至约 $1\mathrm{mL}$ ，用乙腈-水溶液 $(20 + 80)$ 定容至 $10~\mathrm{mL}$ ，用液相色谱仪分离测定 $\mathrm{FB}_{1}$ 的含量。

结果判定： $\mathrm{FB}_1 \geqslant 4.5 \mu \mathrm{g}$ ，为柱容量满意。

# D.2 柱回收率验证方法

准确移取 $\mathrm{FB}_1$ 、 $\mathrm{FB}_2$ 及 $\mathrm{FB}_3$ 标准储备溶液 $(100\mu \mathrm{g} / \mathrm{mL})$ 各 $50~\mu \mathrm{L}$ 至 $50~\mathrm{mL}$ 容量瓶中，用吐温-20/ PBS溶液 $(0.1\%)$ 定容至刻度，充分混匀。分别取同一批次3根免疫亲和柱，每根柱的上样量为 $10~\mathrm{mL}$ $\mathrm{(FB_1,FB_2}$ 及 $\mathrm{FB}_3$ 各 $1\mu \mathrm{g})$ 。经上样、淋洗、洗脱，收集洗脱液，用氮气吹至约 $1\mathrm{mL}$ ，用乙腈-水溶液 $(20+$ 80)定容至 $10~\mathrm{mL}$ ，用液相色谱仪分离测定 $\mathrm{FB}_1$ 、 $\mathrm{FB}_2$ 及 $\mathrm{FB}_3$ 的含量。

结果判定： $\mathrm{FB}_1 \geqslant 0.9 \mu \mathrm{g}, \mathrm{FB}_2 \geqslant 0.9 \mu \mathrm{g}, \mathrm{FB}_3 \geqslant 0.9 \mu \mathrm{g}$ ，则 $\mathrm{FB}_1$ 、 $\mathrm{FB}_2$ 及 $\mathrm{FB}_3$ 回收率 $\geqslant 90\%$ ，为柱回收率满意。

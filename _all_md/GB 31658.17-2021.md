# 食品安全国家标准

# 动物性食品中四环素类、磺胺类和喹诺酮类药物残留量的测定液相色谱-串联质谱法

# 1 范围

本文件规定了动物性食品中四环素类、磺胺类和喹诺酮类药物残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于牛、羊、猪和鸡的肌肉、肝脏和肾脏组织中四环素类（四环素、金霉素、土霉素、多西环素）、磺胺类（乙酰磺胺、磺胺吡啶、磺胺嘧啶、磺胺甲噁唑、磺胺噻唑、磺胺甲嘧啶、磺胺二甲异噁唑、磺胺甲噻二唑、苯甲酰磺胺、磺胺二甲异嘧啶、磺胺二甲嘧啶、磺胺间甲氧嘧啶、磺胺甲氧哒嗪、磺胺对甲氧嘧啶、磺胺氯哒嗪、磺胺邻二甲氧嘧啶、磺胺间二甲氧嘧啶、磺胺苯吡唑、酞磺胺噻唑）和喹诺酮类（诺氟沙星、依诺沙星、环丙沙星、培氟沙星、洛美沙星、达氟沙星、恩诺沙星、氧氟沙星、麻保沙星、沙拉沙星、二氟沙星、噁唑酸、氟甲喹）药物残留量的测定。

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的四环素类、磺胺类和喹诺酮类药物，用Mellvaine-Na2EDFA缓冲液提取，亲水亲脂平衡型固相萃取柱净化，液相色谱串联质谱法测定，外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.3 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ ：色谱纯。

5.1.4 甲酸（HCOOH）：色谱纯。

5.1.5 乙二胺四乙酸二钠·二水 $(\mathrm{C_{10}H_{14}N_2Na_2O_8}\cdot 2\mathrm{H_2O})$ 。

5.1.6 浓氨水 $(\mathrm{NH}_3\mathrm{H}_2\mathrm{O})$

5.1.7 磷酸氢二钠·十二水 $\mathrm{(Na_2HPO_4\cdot 12H_2O)}$

5.1.8 磷酸二氢钠·二水 $\mathrm{(NaH_2PO_4\cdot 2H_2O)}$

5.1.9 柠檬酸·一水 $(\mathrm{C_6H_8O_7}\cdot \mathrm{H_2O})$ 。

5.1.10 氢氧化钠 $(\mathrm{NaOH})$

# 5.2 溶液配制

5.2.1 $0.05\mathrm{mol} / \mathrm{L}$ 磷酸二氢钠溶液：取磷酸二氢钠·二水 $7.8\mathrm{g}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。

5.2.2 $0.05\mathrm{mol} / \mathrm{L}$ 磷酸氢二钠溶液：取磷酸氢二钠·十二水 $17.9\mathrm{g}$ ，用水溶解并稀释至 $1000\mathrm{mL}$ 。

5.2.3 磷酸盐缓冲液：取 $0.05\mathrm{mol} / \mathrm{L}$ 磷酸二氢钠溶液 $190~\mathrm{mL}$ ，用 $0.05\mathrm{mol} / \mathrm{L}$ 磷酸氢二钠溶液稀释至 $1000~\mathrm{mL}$ 。

5.2.4 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：取氢氧化钠 $4\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。

5.2.5 $0.03\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：取 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液 $3\mathrm{mL}$ ，用水稀释至 $100\mathrm{mL}$ 。

5.2.6 Mcllvaine- $\mathrm{Na_2EDTA}$ 缓冲液：取柠檬酸·一水 $12.9\mathrm{g}$ 、磷酸氢二钠·十二水 $10.9\mathrm{g}$ 、乙二胺四乙酸二钠·二水 $39.2\mathrm{g}$ ，加水 $900~\mathrm{mL}$ ，用 $1\mathrm{mol} / \mathrm{L}$ 的氢氧化钠溶液调 $\mathrm{pH}$ 至 $5.0\pm 0.2$ ，用水稀释至 $1000~\mathrm{mL}$ 。

5.2.7 洗脱液：取甲醇 $150~\mathrm{mL}$ ，加乙酸乙酯 $150~\mathrm{mL}$ 、浓氨水 $6~\mathrm{mL}$ ，混匀。

5.2.8 复溶液：取水 $40~\mathrm{mL}$ ，加甲醇 $5\mathrm{mL}$ 、乙腈 $5\mathrm{mL}$ 、甲酸 $0.05\mathrm{mL}$ ，混匀。

# 5.3 标准品

5.3.1 四环素类：四环素、金霉素、土霉素、多西环素含量均 $\geq 95.0\%$ 。具体见附录A。

5.3.2 磺胺类：乙酰磺胺、磺胺吡啶、磺胺嘧啶、磺胺甲噁唑、磺胺噻唑、磺胺甲嘧啶、磺胺二甲异噁唑、磺胺甲噻二唑、苯甲酰磺胺、磺胺二甲异嘧啶、磺胺二甲嘧啶、磺胺间甲氧嘧啶、磺胺甲氧哒嗪、磺胺对甲氧嘧啶、磺胺氯哒嗪、磺胺邻二甲氧嘧啶、磺胺间二甲氧嘧啶、磺胺苯吡唑、酞磺胺噻唑含量均 $\geq 95.0\%$ 。具体见附录A。

5.3.3 喹诺酮类：诺氟沙星、依诺沙星、环丙沙星、培氟沙星、洛美沙星、达氟沙星、恩诺沙星、氧氟沙星、麻保沙星、沙拉沙星、二氟沙星、噻唑酸、氟甲喹含量均 $\geq 95.0\%$ 。具体见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液 $(1\mathrm{mg / mL})$ ：取标准品各适量(相当于各活性成分约 $10~\mathrm{mg}$ )，精密称定，于 $10~\mathrm{mL}$ 容量瓶中，四环素类、磺胺类药物加甲醇溶解并稀释至刻度；喹诺酮类药物加 $0.03\mathrm{mol / L}$ 氢氧化钠溶液溶解并稀释至刻度，配制成浓度均为 $1\mathrm{mg / mL}$ 的标准储备液。 $-18^{\circ}C$ 以下保存，有效期6个月。

5.4.2 混合标准工作液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：分别取 $1\mathrm{mg / mL}$ 的标准储备液各 $0.1\mathrm{mL}$ ，于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的混合标准工作液。一 $18^{\circ}C$ 以下保存，有效期1个月。

5.4.3 混合标准工作液 $(1\mu \mathrm{g / mL})$ ：精密量取 $10~\mu \mathrm{g / mL}$ 的混合标准工作液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $1\mu \mathrm{g / mL}$ 的混合标准工作液。 $-18^{\circ}C$ 以下保存，有效期1个月。

5.4.4 混合标准工作液 $(0.1\mu \mathrm{g / mL})$ ：精密量取 $10~\mu \mathrm{g / mL}$ 的混合标准工作液 $0.1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $0.1\mu \mathrm{g / mL}$ 的混合标准工作液。一 $18^{\circ}C$ 以下保存，有效期1个月。

# 5.5 材料

5.5.1 亲水亲脂平衡型固相萃取柱： $200\mathrm{mg / 6mL}$ ，或相当者。

5.5.2 微孔尼龙滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 高速冷冻离心机： $-2^{\circ}\mathrm{C},14000\mathrm{r / min}$   
6.4 组织匀浆机。  
6.5 涡旋混合器。  
6.6 固相萃取装置。  
6.7 氮吹仪。  
6.8 超声波清洗仪。

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试料 $1\mathrm{g}$ （准确至 $\pm 0.01\mathrm{g}$ ，加McllvaineNaEDTA缓冲液 $8\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，超声 $20\mathrm{min}$ $-2^{\circ}C$ 、 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液。残渣中加磷酸盐缓冲液 $8\mathrm{mL}$ ，重复提取1次，合并2次提取液，混匀，备用。

# 8.2 净化

固相萃取柱依次用甲醇 $5\mathrm{mL}$ 和水 $5\mathrm{mL}$ 活化，取备用液过柱，依次用水 $5\mathrm{mL}$ 和 $20\%$ 甲醇水溶液 $5\mathrm{mL}$ 淋洗，抽干，用洗脱液 $10\mathrm{mL}$ 洗脱。收集洗脱液， $45^{\circ}C$ 水浴氮气吹干。加入复溶液 $1.0\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ 溶解残余物， $14000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，微孔滤膜过滤，液相色谱串联质谱测定。

# 8.3 基质匹配标准曲线的制备

精密量取混合标准工作液适量，分别加入6份经提取和净化的空白试样残渣中， $45^{\circ}\mathrm{C}$ 水浴氮气吹干，加入复溶液 $1.0\mathrm{mL}$ 涡旋溶解残余物，配制成浓度为 $2\mu \mathrm{g} / \mathrm{L}, 10\mu \mathrm{g} / \mathrm{L}, 50\mu \mathrm{g} / \mathrm{L}, 100\mu \mathrm{g} / \mathrm{L}, 250\mu \mathrm{g} / \mathrm{L}$ 和 $500\mu \mathrm{g} / \mathrm{L}$ 的基质匹配系列混合标准溶液，微孔滤膜过滤，液相色谱串联质谱仪测定。以测得特征离子峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.4.4 定量测定

按8.4.1和8.4.2设定仪器条件，以基质匹配标准溶液浓度为横坐标，以峰面积为纵坐标，绘制标准

# 8.5 空白试验

取空白试料，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中四环素类、磺胺类、喹诺酮类药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times C _ {\mathrm {S}} \times V}{A _ {\mathrm {S}} \times m} \tag {1}
$$

式中：

$X$ ——试样中四环素类、磺胺类、喹诺酮类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ 基质匹配标准溶液中四环素类、磺胺类、喹诺酮类药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$A$ ——试样溶液中四环素类、磺胺类、喹诺酮类药物的色谱峰面积；

$A_{\mathrm{S}}$ 基质匹配标准溶液中四环素类、磺胺类、喹诺酮类药物的色谱峰面积；

$V$ ——试样最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ 供试试样质量的数值，单位为克 $(\mathrm{g})$

# 10 方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $2\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $10~\mu \mathrm{g / kg}\sim 500~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 110\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leq 15\%$ ，批间相对标准偏差 $\leq 20\%$ 。

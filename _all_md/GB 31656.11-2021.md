# 方法一 高效液相色谱法

# 1 范围

本文件规定了水产品中土霉素、四环素、金霉素和多西环素残留量测定的高效液相色谱法。

本文件适用于鱼、虾、蟹、鳖及海参等水产品可食组织中土霉素、四环素、金霉素和多西环素残留量的测定。

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中土霉素、四环素、金霉素和多西环素经柠檬酸缓冲溶液提取后，固相萃取柱净化，液相色谱荧光检测器检测，外标法定量。

# 5 试剂与材料

以下所用的试剂，除特别注明外均为分析纯；水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.2 乙酸镁 $(\mathrm{C_4H_6O_4Mg}\cdot 4\mathrm{H_2O})$   
5.1.3 咪唑 $(\mathrm{C_3H_4N_2})$   
5.1.4 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
5.1.5 柠檬酸 $(\mathrm{C_6H_8O_7}\cdot \mathrm{H_2O})$ ：优级纯。  
5.1.6 磷酸氢二钠 $\mathrm{(Na_2HPO_4)}$ ：优级纯。  
5.1.7 乙二胺四乙酸二钠 $(\mathrm{Na}_2\mathrm{EDTA}\cdot 2\mathrm{H}_2\mathrm{O})$ ：优级纯。  
5.1.8 冰乙酸 $\mathrm{(CH_3COOH)}$

# 5.2 溶液配制

5.2.1 磷酸二氢钾溶液：取磷酸二氢钾 $13.6\mathrm{g}$ ，加水适量使溶解并稀释至 $1000~\mathrm{mL}$ 。  
5.2.2 甲醇溶液：取甲醇 $5\mathrm{mL}$ 、水 $95~\mathrm{mL}$ ，混匀。  
5.2.3 柠檬酸缓冲液：取柠檬酸 $6.30\mathrm{g}$ 、磷酸氢二钠 $13.8\mathrm{g}$ 、乙二胺四乙酸二钠 $1.86\mathrm{g}$ ，加水 $400\mathrm{mL}$ 使溶解，用冰乙酸调 $\mathrm{pH}$ 至4.0，再用水稀释至 $500\mathrm{mL}$ 。  
5.2.4 咪唑缓冲溶液：取咪唑 $68.0\mathrm{g}$ 、乙二胺四乙酸二钠 $0.40\mathrm{g}$ 、乙酸镁 $10.0\mathrm{g}$ ，加水 $800\mathrm{mL}$ 使溶解，用

冰乙酸调 $\mathrm{pH}$ 至7.2，再用水稀释至 $1000~\mathrm{mL}$ ，经 $0.45\mu \mathrm{m}$ 水相微孔滤膜过滤后备用。

5.2.5 甲醇-咪唑缓冲溶液：取甲醇 $20~\mathrm{mL}$ 、咪唑缓冲溶液 $80~\mathrm{mL}$ ，混匀。

# 5.3 标准品

盐酸四环素、盐酸金霉素、盐酸土霉素、盐酸多西环素，含量均 $\geq 95\%$ ，具体见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液 $(100\mu \mathrm{g / mL})$ ：分别准确称取适量标准品，用甲醇溶解稀释定容配制成土霉素、四环素、金霉素和多西环素浓度均为 $100~\mu \mathrm{g / mL}$ 的标准储备液。一 $18^{\circ}C$ 以下避光储存，有效期1个月。  
5.4.2 混合标准中间液：分别准确移取土霉素、四环素、金霉素和多西环素标准储备液 $1.0\mathrm{mL}, 1.0\mathrm{mL}$ 、 $2.0\mathrm{mL}$ 和 $2.0\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶，用甲醇稀释并定容，配制成土霉素、四环素、金霉素和多西环素浓度分别为 $1.0\mu \mathrm{g} / \mathrm{mL}, 1.0\mu \mathrm{g} / \mathrm{mL}, 2.0\mu \mathrm{g} / \mathrm{mL}$ 和 $2.0\mu \mathrm{g} / \mathrm{mL}$ 的混合标准中间液。 $-18^{\circ}\mathrm{C}$ 以下避光储存，有效期为 $7\mathrm{d}$ 。  
5.5 材料  
5.5.1 固相萃取柱： $500\mathrm{mg / 6mL}$ ，OasisHLB或性能相当者。  
5.5.2 水相微孔滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 高效液相色谱仪：配荧光检测器  
6.2 分析天平：感量 $0.01\mathrm{g}$   
6.3 分析天平：感量 $0.00001\mathrm{g}$   
6.4 涡旋振荡器。  
6.5 离心机：转速 $6000\mathrm{r / min}$ 或以上。   
6.6 固相萃取装置（带负压抽滤器）。  
6.7 氮吹仪。  
6.8 组织匀浆机。

# 7 试样的制备与保存

# 7.1 试样的制备

按GB/T30891—2014附录B的要求制样。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取试样 $5\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，加柠檬酸缓冲液 $20~\mathrm{mL}$ ，涡旋震荡 $2\mathrm{min}$ ，超声提取 $10\mathrm{min}$ ，以 $6000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液于另一 $50~\mathrm{mL}$ 离心管中。向残渣加柠檬酸缓冲液 $10~\mathrm{mL}$ ，重复提取2次，合并3次上清液，待净化。

# 8.2 净化

将OasisHLB固相萃取柱依次用甲醇 $10\mathrm{mL}$ 、水 $10\mathrm{mL}$ 、柠檬酸缓冲液 $5\mathrm{mL}$ 活化，移入待净化液，用甲醇溶液 $10\mathrm{mL}$ ，水 $10\mathrm{mL}$ 淋洗，抽干，加甲醇 $10\mathrm{mL}$ 洗脱，收集洗脱液， $40^{\circ}\mathrm{C}$ 氮气吹干，加磷酸二氢钾溶液 $1.0\mathrm{mL}$ 溶解残渣，超声 $1\mathrm{min}$ ，过 $0.22\mu \mathrm{m}$ 水相微孔滤膜后，供高效液相色谱测定。

# 8.3 标准曲线的制备

准确移取土霉素、四环素、金霉素和多西环素混合标准中间溶液适量，用甲醇-咪唑缓冲溶液稀释成土霉素和四环素浓度分别为 $50\mathrm{ng / mL},100\mathrm{ng / mL},200\mathrm{ng / mL},500\mathrm{ng / mL},1000\mathrm{ng / mL}$ ，金霉素和多西环素浓度分别为 $100\mathrm{ng / mL},200\mathrm{ng / mL},400\mathrm{ng / mL},1000\mathrm{ng / mL},2000\mathrm{ng / mL}$ 的混合标准系列工作溶液供高效液相色谱测定。以待测物峰面积为纵坐标、相应的标准溶液浓度为横坐标，绘制标准曲线。计算回归方程和相关系数。

# 8.4 测定

# 8.4.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱（ $250\mathrm{mm}\times 4.6\mathrm{mm},5\mu \mathrm{m})$ ，或相当者；  
b） 柱温： $40^{\circ}\mathrm{C}$   
c）进样量： $30~\mu \mathrm{L}$   
d) 流速: $0.8 \mathrm{~mL} / \mathrm{min}$ ;   
e）激发波长： $380~\mathrm{nm}$   
f）发射波长： $520\mathrm{nm}$   
g）流动相：甲醇-咪唑缓冲溶液。

# 8.4.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，以色谱峰面积定量，外标法计算。标准工作液及试样溶液中土霉素、四环素、金霉素和多西环素响应值应在仪器检测的线性范围之内。标准溶液的高效液相色谱图见附录B中的图B.1。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测物残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V}{A _ {\mathrm {S}} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中土霉素、四环素、金霉素和多西环素含量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

A 试样中土霉素、四环素、金霉素和多西环素的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中土霉素、四环素、金霉素和多西环素的峰面积；

$C_{\mathrm{S}}$ ——标准溶液中土霉素、四环素、金霉素和多西环素浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

V 最终试样定容体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ 供试试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法在水产品可食组织中土霉素和四环素的检测限均为 $10.0\mu \mathrm{g / kg}$ ，多西环素和金霉素的检测限均为 $20.0\mu \mathrm{g / kg}$ ；土霉素和四环素的定量限均为 $20.0\mu \mathrm{g / kg}$ ，多西环素和金霉素定量限均为 $40.0\mu \mathrm{g / kg}$ 。

# 10.2 准确度

土霉素和四环素在 $20.0\mu \mathrm{g / kg}\sim 200\mu \mathrm{g / kg}$ 添加浓度的回收率为 $60\% \sim 110\%$ ；多西环素和金霉素在 $40.0\mu \mathrm{g / kg}\sim 400\mu \mathrm{g / kg}$ 添加浓度的回收率为 $60\% \sim 110\%$ 。

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

# 14 原理

试样中残留的土霉素、四环素、金霉素和多西环素，用 $\mathrm{Na_2EDTA - Mcllvaine}$ 缓冲溶液提取，醋酸铅沉淀，正己烷除脂，固相萃取柱净化，液相色谱-串联质谱测定，以基质校正标准曲线进行外标法定量。

# 18 测定步骤

# 18.1 提取

取试样 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，加入 $\mathrm{Na_2EDTA - Mcllvaine}$ 缓冲溶液 $6.0~\mathrm{mL}$ 、醋酸铅溶液 $2.0~\mathrm{mL}$ ，涡旋混合 $1\mathrm{min}$ ，超声 $10\mathrm{min}$ 后，在 $4^{\circ}\mathrm{C}$ 下以 $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，移取上清液；残渣分别用 $\mathrm{Na_2EDTA - }$

Mcllvaine缓冲溶液 $6.0~\mathrm{mL}$ 重复提取2次，合并3次提取液。

# 18.2 净化

向上述提取液中加入正己烷 $10\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ 后， $8000\mathrm{r / min}$ 的速率离心 $5\mathrm{min}$ ，去除上层液，下层溶液待进一步净化（对于鳗鲡、蟹和海参样品，需按照上述方法净化2次）。

固相萃取柱预先依次用甲醇 $5\mathrm{mL}$ 和水 $5\mathrm{mL}$ 活化，取 $10.0\mathrm{mL}$ 提取液上样，控制流速约 $1\mathrm{mL} / \mathrm{min}$ ，依次用水 $5\mathrm{mL}$ 和甲醇溶液 $5\mathrm{mL}$ 淋洗萃取柱，弃去全部流出液，减压抽干 $5\mathrm{min}$ 。最后用甲醇-乙酸乙酯 $5\mathrm{mL}$ 洗脱，洗脱液于 $40^{\circ}\mathrm{C}$ 下用氮气吹至近干，用甲酸溶液 $(0.1\%)$ -乙腈 $1.0\mathrm{mL}$ 溶解残留物，涡旋混匀，过 $0.22\mu \mathrm{m}$ 微孔滤膜，为防药物变性，需及时供液相色谱-质谱仪测定。

# 18.3 标准曲线的制备

精密量取 $1\mu \mathrm{g / mL}$ 混合标准工作液，依次加入6份经提取和净化处理的空白试料洗脱液中，氮气吹至近干后，用甲酸溶液 $(0.1\%)$ -乙腈 $1.0~\mathrm{mL}$ 溶解残留物，配制成 $5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL},$ $50\mathrm{ng / mL},100\mathrm{ng / mL}$ 和 $250\mathrm{ng / mL}$ 的基质混合标准系列工作溶液，现用现配。以基质校正标准工作液的浓度为横坐标，峰面积为纵坐标，绘制标准曲线，求回归方程和相关系数。

# 19 结果计算和表述

试样中待测物残留量按标准曲线或公式(2)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V _ {1} \times V _ {3}}{A _ {\mathrm {S}} \times V _ {2} \times m} \dots \tag {2}
$$

式中：

$X$ ——试样中土霉素、四环素、金霉素和多西环素含量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

A ——试样中土霉素、四环素、金霉素和多西环素的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中土霉素、四环素、金霉素和多西环素的峰面积；

$C_{\mathrm{S}}$ ——标准溶液中土霉素、四环素、金霉素和多西环素浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$V_{1}$ 提取溶液总体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——吸取出用于净化的提取溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——最终试样定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——供试试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 20 检测方法的灵敏度、准确度和精密度

# 20.1 灵敏度

本方法在水产品可食组织中土霉素、四环素、金霉素和多西环素的检测限均为 $5.00\mu \mathrm{g / kg}$ ，定量限均为 $10.0~\mu \mathrm{g / kg}$ 。

# 20.2 准确度

本方法在水产品可食组织中添加 $10.0\mu \mathrm{g / kg}\sim 200\mu \mathrm{g / kg}$ 浓度下的回收率为 $60\% \sim 120\%$

# 20.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

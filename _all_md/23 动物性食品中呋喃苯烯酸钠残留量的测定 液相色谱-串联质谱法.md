# 液相色谱-串联质谱法

# 1 范围

本标准规定了动物性食品中呋喃苯烯酸钠残留量检测的制样和液相色谱-串联质谱测定法。

本标准适用于猪和鸡的肌肉和肝脏组织中呋喃苯烯酸钠残留量的检测。

# 3 原理

试料中残留的呋喃苯烯酸钠，用 $0.5\%$ 甲酸-乙酸乙酯提取，固相萃取柱净化，液相色谱-串联质谱ESI正离子模式测定，外标法定量。

# 4 试剂和材料

以下所用的试剂，除特别注明外均为分析纯试剂，水为符合GB/T6682规定的一级水。

4.1 呋喃苯烯酸钠标准品：含量 $\geq 98.0\%$   
4.2 甲醇  
4.3 正己烷  
4.4 乙睛：色谱纯。  
4.5 乙酸乙酯  
4.6 甲酸  
4.7 HLB固相萃取柱： $60\mathrm{mg / 3mL}$ ，或相当者。  
4.8 $0.5\%$ 甲酸溶液：取甲酸 $0.5 \mathrm{~mL}$ ，用水溶解并稀释至 $100 \mathrm{~mL}$ 。  
4.9 $0.5\%$ 甲酸-乙酸乙酯溶液：取乙酸乙酯 $200~\mathrm{mL}$ ，加 $0.5\%$ 甲酸溶液 $10~\mathrm{mL}$ ，混匀，现配现用。  
4.10 $1 \mathrm{mg} / \mathrm{mL}$ 吠喃苯烯酸钠标准贮备液: 精密称取呋喃苯烯酸钠标准品 $10 \mathrm{mg}$ , 于 $10 \mathrm{~mL}$ 量瓶中,用二甲亚砜溶解并稀释至刻度, 配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的呋喃苯烯酸钠标准贮备液。-20°C以下保

存，有效期6个月。

4.11 $10 \mu \mathrm{g} / \mathrm{mL}$ 呋喃苯烯酸钠标准工作液: 精密量取 $1 \mathrm{mg} / \mathrm{mL}$ 呋喃苯烯酸钠标准贮备液 $1.0 \mathrm{~mL}$ ,于 $100 \mathrm{~mL}$ 容量瓶中, 用流动相溶解并稀释至刻度, 配制成浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 的呋喃苯烯酸钠标准工作液。 $-20^{\circ} \mathrm{C}$ 以下保存, 有效期 6 个月。

# 5 仪器和设备

5.1 液相色谱-串联质谱仪：配有电喷雾离子源（ESI）。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 冷冻高速离心机  
5.5 振荡器  
5.6 旋涡混合器  
5.7 均质机  
5.8 氮吹仪   
5.9 旋转蒸发仪   
5.10 离心管： $50~\mathrm{mL}$ 。  
5.11 茄形瓶   
5.12 滤膜： $0.22\mu \mathrm{m}$ 。

# 6样品的制备与保存

# 6.1 试料的制备

取新鲜或冷冻空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试材料。  
——取均质后的空白样品，作为空白材料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-20℃以下保存。

# 7 测定步骤

# 7.1 提取

取试料 $(2\pm 0.05)\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管中，加 $0.5\%$ 甲酸-乙酸乙酯溶液 $15~\mathrm{mL}$ ，漩涡混合，振

荡 $5 \mathrm{~min}, 8000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ , 取乙酸乙酯层于茄形瓶中, 残渣加 $0.5 \%$ 甲酸-乙酸乙酯溶液 $10 \mathrm{~mL}$ ,重复提取一次, 合并两次乙酸乙酯层, 于 $45^{\circ} \mathrm{C}$ 旋转蒸发至干, 用 $50 \%$ 乙腈水溶液 $3 \mathrm{~mL}$ 溶解残余物,转至离心管中, 用 $50 \%$ 乙腈水溶液 $3 \mathrm{~mL}$ 洗涤茄形瓶, 合并两次洗液, 加正己烷 $3 \mathrm{~mL}$ , 振荡混匀后, $5000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 取下层液, 备用。

# 7.2 净化

HLB 柱依次用甲醇 $3 \mathrm{~mL}$ 和水 $3 \mathrm{~mL}$ 活化，取备用液过柱。用水 $6 \mathrm{~mL}$ 淋洗，抽干，加乙腈 $3 \mathrm{~mL}$ 洗脱，收集洗脱液，于 $50^{\circ} \mathrm{C}$ 氮气吹干，用流动相 $1.0 \mathrm{~mL}$ 溶解残余物，转入 $1.5 \mathrm{~mL}$ 离心管中，10000 r/min 离心 $10 \mathrm{~min}$ ，取上清液，滤膜过滤，液相色谱-串联质谱测定。

# 7.3 标准曲线的制备

分别精密量取 $10 \mu \mathrm{g} / \mathrm{mL}$ 呋喃苯烯酸钠标准工作液适量，用流动相稀释，配制成浓度为 0.2、0.5、2、5 和 $8 \mu \mathrm{g} / \mathrm{L}$ 系列标准溶液，供液相色谱-串联质谱测定。以特征离子质量色谱峰面积为纵坐标，标准溶液浓度为横坐标，绘制标准曲线。

# 7.4 测定

# 7.4.1 液相色谱条件

色谱柱： $\mathrm{C_{18}}$ （ $150 \mathrm{~mm} \times 2.1 \mathrm{~mm}$ ，粒径 $5 \mu \mathrm{m}$ ），或相当者；

流动相：乙腈+水（40+60，v/v）；

柱温： $40^{\circ}\mathrm{C}$ ；

进样量： $10~\mu \mathrm{L}$

# 7.4.2 质谱条件

离子源：电喷雾离子源；

扫描方式：正离子扫描；

检测方式：选择反应监测；

喷雾电压： $4000\mathrm{V}$

离子传输管温度： $330^{\circ}\mathrm{C}$

鞘气压力：40 arb;

辅助气压力：6arb;

选择反应监测的优化参数见表1。

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算与表述

由标准曲线校准：由 $A s = a C s + b$ ，求得 $a$ 和 $b$ ，则 $C = (A - b) / a$

试料中呋喃苯烯酸钠的残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算

$$
X = \frac {A \times \mathcal {C} S \times V}{A _ {\mathrm {s}} \times m}
$$

式中：

$X$ ——试料中呋喃苯烯酸钠的残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

$C_{s}$ ——标准溶液中呋喃苯烯酸钠的浓度, $\mathrm{ng/mL}$ ;

$A$ ——试样溶液中呋喃苯烯酸钠的峰面积；

$A_{s}$ ——标准溶液中呋喃苯烯酸钠的峰面积；

$V$ ——溶解残余物所用流动相体积, $\mathrm{mL}$ ;

$m$ ——供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度、精密度

# 9.1灵敏度

本方法的检测限为 $0.2\mu \mathrm{g / kg}$ ，定量限为 $0.5\mu \mathrm{g / kg}$ 。

# 9.2 准确度

本方法在 $0.5\sim 2\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$

# 9.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 25\%$

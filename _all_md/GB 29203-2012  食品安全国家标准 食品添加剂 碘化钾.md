# 2.2 分子式

166.0（按2007年国际相对原子质量）

# 附录A

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008中规定的三级水。本标准试验中所需标准溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸。

A.3.1.2 过氧化氢。

A.3.1.3 淀粉指示液： $10\mathrm{g / L}$

# A.3.2 鉴别方法

# A.3.2.1 钾离子的鉴别

称量 $0.1\mathrm{g}$ 试样，加水溶解。用铂丝环蘸取盐酸，在火焰上燃烧至无色。再蘸取试验溶液在火焰上燃烧，在钴玻璃下观察，火焰应呈紫色。

# A.3.2.2 碘离子的鉴别

称量 $0.1\mathrm{g}$ 试样，加 $20~\mathrm{mL}$ 水、 $1\mathrm{mL}$ 盐酸、2滴淀粉指示液和2滴过氧化氢，溶液应呈现蓝色。

# A.4 碘化钾(KI)含量的测定

# A.4.1 方法提要

在盐酸介质溶液中，用碘酸钾标准滴定溶液进行滴定，通过三氯甲烷层颜色的消失判断终点。

# A.4.2 试剂和材料

A.4.2.1 盐酸。

A.4.2.2 三氯甲烷。

A.4.2.3 碘酸钾标准滴定溶液： $c(1 / 6\mathrm{KIO}_3) = 0.3\mathrm{mol} / \mathrm{L}$

# A.4.3 分析步骤

称取约 $0.5\mathrm{g}$ 按照A.5操作后的试样，精确至 $0.0002\mathrm{g}$ 。置于 $250~\mathrm{mL}$ 锥形瓶中，加入 $10~\mathrm{mL}$ 水

溶解，加入 $35~\mathrm{mL}$ 盐酸，摇匀。加 $5\mathrm{mL}$ 三氯甲烷，用碘酸钾标准滴定溶液进行滴定（近终点时，剧烈振摇），直至三氯甲烷层无色后静置 $5\mathrm{min}$ 不再呈紫色为止，即为终点。

# A.4.4 结果计算

碘化钾(KI)含量的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {V \times c \times M}{m \times 1000} \times 100 \% \tag{A.1}
$$

式中：

$V$ ——滴定试验溶液所消耗的碘酸钾标准滴定溶液的体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——碘酸钾标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

$M$ ——碘酸钾 $(1 / 6\mathrm{KIO}_3)$ 的摩尔质量的数值，单位为克每摩尔 $(\mathrm{g / mol})(M = 55.33)$ ；

1000——换算因子。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.02\%$ 。

# A.5 干燥减量的测定

# A.5.1 仪器和设备

A.5.1.1 称量瓶： $\phi 30\mathrm{mm}\times 25\mathrm{mm}$

A.5.1.2 电热恒温干燥箱：温度能控制在 $105\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}$ 。

# A.5.2 分析步骤

在预先于 $105\mathrm{^\circ C}\pm 2\mathrm{^\circ C}$ 下干燥至质量恒定的称量瓶中称取约 $2\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于电热恒温干燥箱内，在 $105\mathrm{^\circ C}\pm 2\mathrm{^\circ C}$ 下干燥 $4\mathrm{h}$ ，称量。

# A.5.3 结果计算

干燥减量的质量分数 $w_{2}$ ，按式（A.2）计算：

$$
w _ {2} = \frac {m _ {1} - m _ {2}}{m} \times 100 \% \tag{A.2}
$$

式中：

$m_{1}$ ——干燥前试样和称量瓶的质量的数值，单位为克(g)；

$m_{2}$ ——干燥后试样和称量瓶质量的数值，单位为克(g)；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.1\%$ 。

# A.6 碘酸盐测定

# A.6.1 试剂和材料

A.6.1.1 硫酸溶液： $3 + 97$

A.6.1.2 碘酸盐标准溶液： $1\mathrm{mL}$ 含碘酸钾 $\mathrm{(KIO_3)}0.010\mathrm{mg}$ 。称取 $0.010\mathrm{g}$ 碘酸钾，置于 $1000~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。贮存于棕色瓶中。

A.6.1.3 淀粉指示液： $10\mathrm{g / L}$

A.6.1.4 无氨和无二氧化碳的水：将无氨的水倒入烧瓶中，煮沸 $15\mathrm{min}$ ，立即用装有钠石灰管的胶塞塞紧，冷却。

# A.6.2 分析步骤

称取 $1.10\mathrm{g}\pm 0.01\mathrm{g}$ 试样，溶解在 $5\mathrm{mL}$ 无氨和无二氧化碳的水中，然后转移到 $10\mathrm{mL}$ 比色管中，加入2滴淀粉指示液，加2滴硫酸溶液，混匀，用无氨和无二氧化碳的水稀释至刻度。其溶液的颜色不应深于标准比色液。

标准比色液是用移液管移取 $0.4\mathrm{mL}$ 碘酸盐标准溶液置于 $10~\mathrm{mL}$ 比色管中，称取 $0.10\mathrm{g}\pm 0.01\mathrm{g}$ 试样，加无氨和无二氧化碳的水溶解，再加入2滴淀粉指示液，加2滴硫酸溶液，混匀，用无氨和无二氧化碳的水稀释至刻度。

# A.7 硝酸盐、亚硝酸盐和氨的测定

# A.7.1 试剂和材料

A.7.1.1 氢氧化钠溶液： $40~\mathrm{g / L}$

A.7.1.2 铝丝。

A.7.1.3 石蕊试纸。

# A.7.2 分析步骤

称取 $1.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $50~\mathrm{mL}$ 试管中，加 $5\mathrm{mL}$ 水溶解，然后加 $5\mathrm{mL}$ 氢氧化钠溶液，加 $0.2\mathrm{g}$ 铝丝。用脱脂棉塞住试管口，在试管口上面放一条湿润的红色石蕊试纸。将试管置于沸水浴上加热 $15\mathrm{min}$ ，试纸不应变蓝。

# A.8 硫代硫酸盐和钡的测定

# A.8.1 试剂和材料

硫酸溶液： $3 + 97$

# A.8.2 分析步骤

称取 $0.50\mathrm{g}\pm 0.01\mathrm{g}$ 试样，用 $10~\mathrm{mL}$ 无氨和无二氧化碳的水(同A.6.1.4)溶解，加2滴硫酸溶液， $1\mathrm{min}$ 内溶液不应混浊。

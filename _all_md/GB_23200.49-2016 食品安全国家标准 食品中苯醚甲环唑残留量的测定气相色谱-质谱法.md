本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T1975—2007《进出口食品中苯醚甲环唑残留量的检测方法 气相色谱一质谱法》。与SN/T1975—2007相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；

——标准范围中增加“其他食品可参照执行”。

# 食品中苯醚甲环唑残留量的测定 气相色谱一质谱法

# 1 范围

本标准规定了食品中苯醚甲环唑的制样、气相色谱一质谱联用检测和确证方法。

本标准适用于豌豆、紫苏、胡萝卜、菠菜、大米、黄豆、中药材、茶叶、杏仁、西柚、菠萝、草莓、酱油、醋、蜂蜜、王浆干粉、猪肉、牛肉、鸡肉、鳗和龙虾仁等食品中苯醚甲环唑残留量的测定和确证。其他食品可参照执行。

# 3 原理

试样中的苯醚甲环唑用乙酸乙酯提取, 经串联活性炭和中性氧化铝双柱法或弗罗里硅土单柱法固相萃取净化后由气相色谱一质谱联用仪测定与确证, 外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。

4.1.2 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。

4.1.3 丙酮 $\mathrm{(C_3H_6O)}$ ：色谱纯。

4.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置于密闭容器中备用。

# 4.2 溶液配制

正己烷 $+$ 丙酮 $(3 + 2$ ，体积比)混合溶剂：取 $300~\mathrm{mL}$ 正己烷，加入 $200~\mathrm{mL}$ 丙酮，摇匀备用。

# 4.3 标准品

苯醚甲环唑标准物质（Difenoconazole， $\mathrm{C_{19}H_{17}Cl_2N_3O_3}$ ，CAS号：119446-68-3）：纯度 $\geq 99.5\%$ 。

# 4.4 标准溶液配制

4.4.1 苯醚甲环唑标准储备溶液：准确称取适量的苯醚甲环唑标准品，用乙酸乙酯稀释配制成 $200\mu \mathrm{g} / \mathrm{mL}$ 的标准储备液， $4^{\circ}\mathrm{C}$ 下保存。  
4.4.2 苯醚甲环唑标准工作液：根据需要用正己烷稀释成适当浓度的标准工作溶液， $4^{\circ}\mathrm{C}$ 下保存。

# 4.5 材料

4.5.1 活性炭固相萃取柱： $3\mathrm{mL} / 250\mathrm{mg}$ ，或相当者。  
4.5.2 中性氧化铝固相萃取柱： $\mathrm{N - Al_2O_3}$ ， $3\mathrm{mL} / 250\mathrm{mg}$ ，或相当者。  
4.5.3 弗罗里硅土固相萃取柱：Florisil, $3\mathrm{mL} / 0.5\mathrm{g}$ ，或相当者。

# 5 仪器和设备

5.1 气相色谱一质谱仪：配有负化学离子源(NCI)。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 固相萃取装置，带真空泵。  
5.4 组织捣碎机。  
5.5 粉碎机。  
5.6 离心机。  
5.7 旋转蒸发器。

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按照 GB 2763 的规定执行。在取样和制样的操作过程中,应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 蔬菜或水果类

取代表性样品 $500\mathrm{g}$ , 将其切碎后, 依次用捣碎机将样品加工成浆状。混匀, 均分成2份作为试样, 分装入洁净的容器内, 密闭并标明标记。

# 6.2.2 茶叶、坚果及粮谷类

取代表性样品 $500 \mathrm{~g}$ , 用粉碎机粉碎并通过 $2.0 \mathrm{~mm}$ 圆孔筛。混匀, 分装入洁净的容器内, 密闭并标明标记。

# 6.2.3 肉及肉制品

取代表性样品 $500 \mathrm{~g}$ , 将其切碎后, 依次用捣碎机将样品加工成浆状, 混匀, 分装入洁净的盛样袋内, 密封并标明标记。

# 6.2.4 调味品

取代表性样品 $500\mathrm{g}$ ，混匀，分装入洁净的容器内，密闭并标明标记。

# 6.2.5 蜂产品

取有代表性样品 $500 \mathrm{~g}$ , 对无结晶的蜂蜜样品将其搅拌均匀; 对有结晶析出的蜂蜜样品, 在密闭情况下, 将样品瓶置于不超过 $60^{\circ} \mathrm{C}$ 的水浴中温热, 振荡, 待样品全部融化后搅匀, 迅速冷却至室温, 在融化时必须注意防止水分挥发。制备好的试样均分成两份, 分别装入样品瓶中, 密封, 并标明标记。

# 6.3 试样保存

茶叶、蜂产品、调味品及粮谷类等试样于 $4^{\circ} \mathrm{C}$ 保存; 水果蔬菜类和肉及肉制品类等试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 7 分析步骤

# 7.1 提取

对于含水量较低的或油脂含量较高的样品(茶叶、大米、豆类、肉及肉制品、蜂产品等),准确称取 $5\mathrm{g}$ 均匀试样(精确至 $0.01\mathrm{g}$ )。对于含水量较高的试样(蔬菜、水果、酱油及醋类调味品等),准确称取 $10\mathrm{g}$ 均匀试样(精确至 $0.01\mathrm{g}$ )。将称取的样品置于 $250\mathrm{mL}$ 的具塞锥形瓶中,加入 $50\mathrm{mL}$ 乙酸乙酯,加入 $15\mathrm{g}$ 无水硫酸钠,放置于振荡器中振荡 $40\mathrm{min}$ ,过滤于 $150\mathrm{mL}$ 浓缩瓶中。再加入 $20\mathrm{mL}$ 乙酸乙酯重复提取一次,合并提取液, $40^{\circ}\mathrm{C}$ 下减压浓缩至干。用 $3\mathrm{mL}$ 正己烷溶解,待净化。

# 7.2 净化

# 7.2.1 含色素较高的样品

将活性炭小柱与中性氧化铝小柱串联，依次用 $5\mathrm{mL}$ 丙酮、 $5\mathrm{mL}$ 正己烷活化，将正己烷提取溶液过柱，过完后再用 $3\mathrm{mL}$ 正己烷清洗瓶子并过柱，保持液滴流速约为 $2\mathrm{mL} / \mathrm{min}$ ，去掉滤液，抽干后，用 $5\mathrm{mL}$ 正己烷 $+$ 丙酮（ $3 + 2$ ，体积比）混合溶剂进行洗脱，收集洗脱液于 $10\mathrm{mL}$ 小试管中，于 $40^{\circ}\mathrm{C}$ 水浴中氮气吹干，用正己烷定容 $1.0\mathrm{mL}$ ，并过 $0.45\mu \mathrm{m}$ 有机相滤膜，供气相色谱一质谱测定和确证。

# 7.2.2 蜂产品或油脂含量较高的样品

将弗罗里硅土固相萃取小柱依次用 $5\mathrm{mL}$ 丙酮、 $5\mathrm{mL}$ 正己烷活化，然后将正己烷提取溶液过柱，以下步骤同7.2.1。

# 7.3 测定

# 7.3.1 气相色谱一质谱参考条件

a）色谱柱：石英弹性毛细管柱DB-17ms, $30\mathrm{m}\times 0.25\mathrm{mm}$ (内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度： $200^{\circ}\mathrm{C}\xrightarrow{10^{\circ}\mathrm{C / min}}300^{\circ}\mathrm{C}$ （保持 $10\mathrm{min}$ ）；  
c）进样口温度： $300^{\circ}\mathrm{C}$   
d) 色谱—质谱接口温度: $280^{\circ} \mathrm{C}$ ;   
e) 载气：氦气，纯度 $\geqslant 99.999\%$ ，流速 $1.0~\mathrm{mL / min}$   
f) 进样量: $1 \mu \mathrm{L}$ ;   
g）进样方式：不分流进样， $1.5\mathrm{min}$ 后开阀；  
h） 电离方式：NCI；  
i）电离能量： $216.5\mathrm{eV}$   
j）离子源温度： $150^{\circ}\mathrm{C}$   
k） 四极杆温度： $150^{\circ}\mathrm{C}$   
1) 反应气：甲烷，纯度 $\geqslant 99.99\%$ ，反应气流速： $2\mathrm{mL} / \mathrm{min}$   
m）检测方式：选择离子监测方式（SIM)；  
n）选择监测离子 $(m / z)$ ：定量离子348；定性离子310，350，405；  
o）溶剂延迟时间： $12\mathrm{min}$

# 7.3.2 色谱测定与确证

7.3.2.1 根据样液中苯醚甲环唑含量的情况，选定峰面积相近的标准工作溶液，对标准工作液和样液等体积参插进样。标准工作溶液和样液中苯醚甲环唑的相应值均应在仪器的线性范围内。  
7.3.2.2 如果样液与标准工作溶液的选择离子色谱图中，在相同保留时间处有色谱峰出现，并且在扣除背景后的样品质量色谱图中，所选离子均出现，所选择离子的丰度比与标准品对应离子的丰度比，其值在允许范围内（允许范围见表1）。在7.3.1条件下，苯醚甲环唑的保留时间是 $15.74\mathrm{min}$ ，其监测离子 $(m / z)$ 丰度比为 $310:348:350:405 = 45:100:67:13$ 对其进行确证；根据定量离子 $m / z348$ 对其进行外标法定量。在7.3.1条件下，苯醚甲环唑标准物的气相色谱—质谱总离子流色谱图和全扫描质谱图参见图A.1和图B.1。

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中苯醚甲环唑残留量。

$$
C _ {x} = \frac {A _ {x} \times C _ {s} \times V _ {x}}{A _ {s} \times m} \quad \dots \tag {1}
$$

式中：

$C_x$ ——试样中苯醚甲环唑残留量，单位为微克每克 $(\mu \mathrm{g} / \mathrm{g})$ ；

$A_{x}$ ——样液中苯醚甲环唑定量离子的峰面积；

$C_s$ ——标准工作液中苯醚甲环唑的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V_{x}$ ——样液最后定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准工作液中苯醚甲环唑定量离子的峰面积；

$m$ ——最终样液所代表的试样质量，单位为克(g)。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 C 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 D 的要求。

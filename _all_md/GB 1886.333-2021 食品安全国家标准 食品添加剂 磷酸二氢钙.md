# 前言

本标准代替GB25559—2010《食品安全国家标准 食品添加剂 磷酸二氢钙》。

本标准与GB25559—2010相比，主要变化如下：

——修改了范围；  
——将“氟化物（以F计）”修改为“氟(F)”;   
——将氟的检验方法修改为GB/T5009.18；  
——将铅的检验方法修改为GB5009.75—2014第一法或GB5009.12—2017第二法、第三法、第四法；  
——将砷的检验方法修改为GB5009.76或GB5009.11；  
——将重金属的检验方法修改为GB5009.74—2014。

# 1 范围

本标准适用于以磷酸氢钙(或磷酸三钙、氢氧化钙、碳酸钙)和食品添加剂磷酸(含湿法磷酸)为原料生产的食品添加剂磷酸二氢钙。

# 2 分子式和相对分子质量

无水磷酸二氢钙： $\mathrm{Ca(H_2PO_4)_2}$

一水磷酸二氢钙： $\mathrm{Ca(H_2PO_4)_2\cdot H_2O}$

# 2.2 相对分子质量

无水磷酸二氢钙：234.05（按2018年国际相对原子质量）

一水磷酸二氢钙：252.07（按2018年国际相对原子质量）

# 检验方法

警示：本检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应小心谨慎！必要时，应在通风橱中进行。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603之规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 硝酸溶液： $1 + 1$

A.2.1.2 盐酸溶液： $1 + 3$

A.2.1.3 草酸铵溶液： $35\mathrm{g / L}$

A.2.1.4钼酸铵溶液： $60~\mathrm{g / L}$ ，称取 $6\mathrm{g}$ 钼酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{6}\mathrm{MO}_{7}\mathrm{O}_{24}\cdot 4\mathrm{H}_{2}\mathrm{O}\right]$ 溶于 $50~\mathrm{mL}$ 水中，边搅拌边缓慢加入 $50~\mathrm{mL}$ 硝酸溶液，贮存于棕色试剂瓶中。

# A.2.2 鉴别方法

# A.2.2.1 钙离子的鉴别

称取约 $0.1\mathrm{g}$ 试样，加入 $2\mathrm{mL}$ 盐酸溶液， $8\mathrm{mL}$ 水，加热至 $40^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ ，加入 $5\mathrm{mL}$ 草酸铵溶液，产生白色沉淀。

# A.2.2.2 磷酸根的鉴别

称取约 $0.1\mathrm{g}$ 试样，滴加硝酸溶液使试样溶解，再过量 $1\mathrm{mL}$ 硝酸溶液，加热至 $40^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ ，加入 $10\mathrm{mL}$ 钼酸铵溶液，产生黄色沉淀。

# A.3 磷酸二氢钙（以Ca计）含量的测定

# A.3.1 氧化还原滴定法（方法一）

# A.3.1.1 方法提要

试样经盐酸溶解，加入草酸铵溶液后用氨水调节溶液的 $\mathrm{pH}$ ，在沸水浴中煮沸，生成草酸钙沉淀，用高锰酸钾标准滴定溶液滴定草酸钙沉淀，得出结果。

# A.3.1.2 试剂和材料

A.3.1.2.1 盐酸溶液： $1 + 3$

A.3.1.2.2 草酸铵溶液： $33\mathrm{g / L}$

A.3.1.2.3 洗涤液：取 $10\mathrm{mL}$ 草酸铵溶液，用水稀释至 $1000~\mathrm{mL}$   
A.3.1.2.4 氨水溶液： $1 + 1$   
A.3.1.2.5 硫酸溶液： $1 + 6$   
A.3.1.2.6 高锰酸钾标准滴定溶液： $c(1 / 5\mathrm{KMnO}_4) = 0.1\mathrm{mol} / \mathrm{L}$   
A.3.1.2.7 甲基橙指示液： $1\mathrm{g / L}$   
A.3.1.2.8 甲基红指示液： $1\mathrm{g / L}$

# A.3.1.3 仪器和设备

A.3.1.3.1 恒温水浴箱。  
A.3.1.3.2 玻璃砂坩埚：孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$

# A.3.1.4 分析步骤

称取约 $0.5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 烧杯中，加入 $10~\mathrm{mL}$ 盐酸溶液，使试样溶解，加3滴甲基橙指示液，煮沸 $5\mathrm{min}$ 。如有必要，在煮沸过程中加入盐酸溶液或水使试样溶液的 $\mathrm{pH}$ 和体积保持不变。加2滴甲基红指示液， $30~\mathrm{mL}$ 草酸铵溶液，接着边搅拌边滴加氨水溶液，直至溶液的粉红色刚刚消失。将烧杯置于恒温水浴箱中煮沸 $30\mathrm{min}$ ，冷却至室温。待沉淀沉降，用倾析法将上层置于清洁干燥的玻璃砂坩埚抽滤，用 $30~\mathrm{mL}$ 冷的洗涤液（ $20^{\circ}\mathrm{C}$ 以下）洗涤烧杯中的沉淀，洗涤液转移至玻璃砂坩埚抽滤，重复洗涤至沉淀全部转移至玻璃砂坩埚中，最后各用 $10~\mathrm{mL}$ 水洗涤两次。将玻璃砂坩埚放入烧杯中，加入 $100~\mathrm{mL}$ 水和 $50~\mathrm{mL}$ 硫酸溶液，用滴定管加入 $35~\mathrm{mL}$ 高锰酸钾标准滴定溶液，搅拌至颜色消失，加热至大约 $70^{\circ}\mathrm{C}$ ，用高锰酸钾标准滴定溶液滴定至溶液呈粉红色，并在 $30~\mathrm{s}$ 内不消失即为终点。同时进行空白试验。

空白试验除不加试样外，其他操作及加入试剂的种类和量(标准滴定溶液除外)与测定试验相同。

# A.3.1.5 结果计算

磷酸二氢钙(以 $\mathrm{Ca}$ 计)的质量分数 $\varpi_{1}$ 按式(A.1)计算。

$$
w _ {1} = \frac {\left[ \left(V _ {1} - V _ {0}\right) / 1000 \right] \times c _ {1} \times M _ {1}}{m _ {1}} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——滴定试样溶液所消耗的高锰酸钾标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验所消耗的高锰酸钾标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算因子；

$c_{1}$ ——高锰酸钾标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{1}$ ——钙 $(1 / 2\mathrm{Ca})$ 的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})(M_1 = 20.039)$ ；

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.3.2 硫酸锌返滴定法（方法二）

# A.3.2.1 方法提要

在试样溶液中，加入过量的乙二胺四乙酸二钠标准滴定溶液，与钙离子络合，以KB混合液为指示剂，用硫酸锌标准滴定溶液滴定过量的乙二胺四乙酸二钠标准滴定溶液。

# A.3.2.2 试剂和材料

A.3.2.2.1 盐酸溶液： $1 + 1$

A.3.2.2.2 氨-氯化铵缓冲溶液（甲）： $\mathrm{pH}\approx 10$

A.3.2.2.3 乙二胺四乙酸二钠(EDTA)标准滴定溶液： $c(\mathrm{EDTA}) = 0.05\mathrm{mol / L}$

A.3.2.2.4 酸性铬蓝K-萘酚绿B混合指示液（KB指示液）。

A.3.2.2.5 硫酸锌标准滴定溶液： $c(\mathrm{ZnSO}_4\cdot 7\mathrm{H}_2\mathrm{O}) = 0.05\mathrm{mol / L}$

配制：称取 $15\mathrm{g}$ 硫酸锌，加水溶解，用水稀释至 $1000~\mathrm{mL}$ ，摇匀。

标定：移取 $25.00\mathrm{mL}$ 配制的硫酸锌标准滴定溶液，置于锥形瓶中，加 $10~\mathrm{mL}$ 氨-氯化铵缓冲溶液（甲）及 $75~\mathrm{mL}$ 水，加约 $0.02\mathrm{g}$ 铬黑T指示剂，用 $0.05\mathrm{mol} / \mathrm{L}$ 的乙二胺四乙酸二钠(EDTA)标准滴定溶液滴定至溶液由紫色变为纯蓝色，并保持 $30~\mathrm{s}$ 不褪色，即为终点。同时进行空白试验。

空白试验除不加试样外，其他操作及加入试剂的种类和量(标准滴定溶液除外)与测定试验相同。

硫酸锌标准滴定溶液的浓度 $c_{2}$ ，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ，按式(A.2)计算。

$$
c _ {2} = \frac {\left(V _ {2} - V _ {3}\right) \times c _ {3}}{V _ {4}} \quad \dots \tag {A.2}
$$

式中：

$V_{2}$ ——滴定所消耗的乙二胺四乙酸二钠(EDTA)标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$

$V_{3}$ ——空白试验所消耗的乙二胺四乙酸二钠(EDTA)标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{3}$ ——乙二胺四乙酸二钠(EDTA)标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{4}$ ——移取硫酸锌标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

# A.3.2.3 分析步骤

称取约 $0.5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $100\mathrm{mL}$ 烧杯中，用少量水润湿，加 $5\mathrm{mL}$ 盐酸溶液使试样全部溶解。全部转移至 $250\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。必要时干过滤，弃去前 $20\mathrm{mL}$ 滤液。

移取 $25.00\mathrm{mL}$ 上述试样溶液，置于 $500~\mathrm{mL}$ 锥形瓶中，再用另一移液管移入 $25.00~\mathrm{mL}$ 乙二胺四乙酸二钠(EDTA)标准滴定溶液，加 $50~\mathrm{mL}$ 水及 $10~\mathrm{mL}$ 氨-氯化铵缓冲溶液（甲），摇匀，放置 $5\mathrm{min}$ 。加2滴 $\sim 3$ 滴KB指示液，用硫酸锌标准滴定溶液滴定至溶液由蓝色变为蓝紫色，并保持 $30~\mathrm{s}$ 内紫色不褪色，即为终点。同时进行空白试验。

空白试验除不加试样外，其他操作及加入试剂的种类和量(标准滴定溶液除外)与测定试验相同。

# A.3.2.4 结果计算

磷酸二氢钙(以 $\mathrm{Ca}$ 计)的质量分数 $\varpi_{1}$ 按式(A.3)计算。

$$
w _ {1} = \frac {\left[ \left(V _ {5} - V _ {6} / 1000\right) \right] \times c _ {4} \times M _ {2}}{m _ {2} \times 25 / 250} \times 100 \% \quad \dots \tag{A.3}
$$

式中：

$V_{5}$ ——空白试验所消耗的硫酸锌标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{6}$ ——滴定试样溶液所消耗的硫酸锌标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算因子；

$c_{4}$ ——硫酸锌标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{2}$ ——钙(Ca)的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})(M_2 = 40.078)$ ；

$m_{2}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

25 ——移取试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

250 ——试样溶液定容的体积，单位为毫升 $(\mathrm{mL})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.4 干燥减量的测定

# A.4.1 仪器和设备

A.4.1.1 称量瓶： $\phi 50\mathrm{mm}\times 30\mathrm{mm}$   
A.4.1.2 电热恒温干燥箱：控温范围为 $60^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。

# A.4.2 分析步骤

用已于 $60~\mathrm{^\circ C}\pm 2$ ℃条件下干燥3h的称量瓶，称取约 $1.5\mathrm{g}$ 试样（一水物），精确至 $0.0002\mathrm{g}$ ，置于电热恒温干燥箱内，在 $60~\mathrm{^\circ C}\pm 2$ ℃条件下干燥3h，取出冷却至室温，称量。

# A.4.3 结果计算

干燥减量的质量分数 $\omega_{2}$ 按式(A.4)计算。

$$
w _ {2} = \frac {m _ {3} - m _ {4}}{m _ {5}} \times 100 \% \tag{A.4}
$$

$m_{3}$ ——干燥前试样和称量瓶的质量，单位为克 $(\mathrm{g})$ ；

$m_{4}$ ——干燥后试样和称量瓶的质量，单位为克 $(\mathrm{g})$ ；

$m_{5}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.1\%$ 。

# A.5 灼烧减量的测定

# A.5.1 仪器和设备

A.5.1.1 瓷坩埚: $30 \mathrm{~mL}$ 。

A.5.1.2 高温炉：控温范围为 $800^{\circ}\mathrm{C} \pm 50^{\circ}\mathrm{C}$ 。

# A.5.2 分析步骤

用已于 $800^{\circ}\mathrm{C} \pm 50^{\circ}\mathrm{C}$ 条件下灼烧 $30\mathrm{min}$ 的瓷坩埚，称取约 $1.5\mathrm{g}$ 试样（无水物），精确至 $0.0002\mathrm{g}$ ，置于高温炉内，在 $800^{\circ}\mathrm{C} \pm 50^{\circ}\mathrm{C}$ 条件下灼烧 $30\mathrm{min}$ ，取出冷却至室温，称量。

# A.5.3 结果计算

灼烧减量的质量分数 $w_{3}$ 按式(A.5)计算。

$$
w _ {3} = \frac {m _ {6} - m _ {7}}{m _ {8}} \times 100 \% \tag{A.5}
$$

式中：

$m_{6}$ ——灼烧前试样和坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{7}$ ——灼烧后残余物和坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{8}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.1\%$ 。

# A.6 澄清度的测定

# A.6.1 试剂和材料

A.6.1.1 盐酸。

A.6.1.2 硝酸溶液： $1 + 2$

A.6.1.3 硝酸银溶液： $20\mathrm{g / L}$

A.6.1.4 氯化物标准溶液： $1\mathrm{mL}$ 溶液含氯(Cl) $0.010\mathrm{mg}$ 。移取 $10.00\mathrm{mL}$ 按GB/T602要求配制的氯化物标准溶液，置于 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。该溶液现用现配。

# A.6.2 仪器和设备

恒温水浴箱。

# A.6.3 分析步骤

标准比浊溶液的配制：移取 $1.2\mathrm{mL}$ 氯化物标准溶液，置于 $50~\mathrm{mL}$ 比色管中，加 $20~\mathrm{mL}$ 水、 $1\mathrm{mL}$ 硝酸溶液、 $1\mathrm{mL}$ 硝酸银溶液，用水稀释至刻度，摇匀。在暗处放置 $15\mathrm{min}$ 。

称取 $2.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $50~\mathrm{mL}$ 烧杯中，加 $18~\mathrm{mL}$ 水及 $2\mathrm{mL}$ 盐酸，在沸水浴中加热 $5\mathrm{min}$ 溶解。冷却后全部转移至 $50~\mathrm{mL}$ 比色管中，用水稀释至刻度，摇匀。试样溶液所呈浊度不大于标准比浊溶液为通过试验。

# A.7 碳酸盐的测定

# A.7.1 试剂和材料

盐酸。

# A.7.2 分析步骤

称取 $2.0\mathrm{g}$ 试样，精确至 $0.1\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 锥形瓶中，加 $6\mathrm{mL}$ 水煮沸，冷却后滴加 $2\mathrm{mL}$ 盐酸，试样溶液不产生气泡为通过试验。

# A.8 游离酸及其副盐的测定

# A.8.1 试剂和材料

A.8.1.1 氢氧化钠溶液： $1\mathrm{mol} / \mathrm{L}$

A.8.1.2 甲基橙指示液： $1\mathrm{g / L}$

# A.8.2 分析步骤

称取 $1.0\mathrm{g}$ 试样，精确至 $0.1\mathrm{g}$ ，置于研钵中，加 $3\mathrm{mL}$ 水研磨，全部转移至 $250\mathrm{mL}$ 烧杯中，加 $100\mathrm{mL}$ 水摇匀，滴加1滴甲基橙指示液，加入 $1\mathrm{mL}$ 氢氧化钠溶液，试样溶液呈黄色为通过试验。

# A.9 铅(Pb)的测定

按GB5009.75—2014中第一法或GB5009.12—2017中第二法、第三法、第四法规定的方法进行测定，试验中所用水为 $\mathrm{GB / T6682}$ 规定的二级水。

# A.10 砷（As)的测定

按GB5009.76或GB5009.11规定的方法进行测定，试验中所用水为GB/T6682规定的二级水。

# A.11 重金属(以Pb计)的测定

# A.11.1 试剂和材料

A.11.1.1 水：符合GB/T6682中二级水的规定。

A.11.1.2 其他试剂同GB5009.74—2014的第3章。

# A.11.2 仪器和设备

同GB5009.74—2014第4章。

# A.11.3 分析步骤

称取 $2.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $50~\mathrm{mL}$ 烧杯中，加 $5\mathrm{mL}$ 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ 及 $10~\mathrm{mL}$ 水，加热溶解，冷却后滴加氨水至白色沉淀出现，再加入少量盐酸溶液至沉淀消失，必要时干过滤(用水洗涤5次），全部转移至 $50~\mathrm{mL}$ 纳氏比色管中，加 $5\mathrm{mL}$ pH3.5的乙酸盐缓冲溶液，混匀。标准比色溶液是移取 $2.00~\mathrm{mL}$ 铅标准使用液置于 $50~\mathrm{mL}$ 纳氏比色管中，加 $25~\mathrm{mL}$ 水及 $5\mathrm{mL}$ pH3.5的乙酸盐缓冲溶液，混匀，然后按照GB5009.74—2014中6.4的规定进行测定。

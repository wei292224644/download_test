# 液相色谱-串联质谱法

# 1 范围

本文件规定了动物性食品中 $\alpha^{-}$ 群勃龙和 $\beta^{-}$ 群勃龙残留量测定的制样和液相色谱串联质谱检测方法。

本文件适用于猪、牛、羊、鸡的肌肉、脂肪、肝脏和肾脏组织及兔肉、鸡蛋、牛奶和羊奶中 $\alpha$ 群勃龙和 $\beta$ 群勃龙残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的 $\alpha$ 群勃龙和 $\beta$ 群勃龙在弱酸性条件下酶解，用叔丁基甲醚提取，正出烷除脂，HLB和氨基固相萃取柱净化，液相色谱串联质谱法测定，外标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.2 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ ：色谱纯。

5.1.3 叔丁基甲醚 $(\mathrm{C}_5\mathrm{H}_{12}\mathrm{O})$

5.1.4 正己烷 $(\mathrm{C_6H_{14}})$ 。

5.1.5 冰乙酸 $\mathrm{(CH_3COOH)}$

5.1.6 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ：含量 $(\mathrm{NH}_{3}$ ）约 $25\%$

5.1.7 乙酸钠 $(\mathrm{C_2H_3NaO_2})$

5.1.8 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ 。

5.1.9 $\beta$ 葡萄糖苷酸酶/芳基硫酸酯酶： $>100000\mathrm{U / mL}$

# 5.2 溶液配制

5.2.1 $0.04\mathrm{mol / L}$ 乙酸钠溶液：取乙酸钠 $3.28\mathrm{g}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。  
5.2.2 $80\%$ 甲醇水溶液：取甲醇 $80~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$   
5.2.3 $10\%$ 甲醇水溶液：取甲醇 $10~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$   
5.2.4 $2\%$ 氨水溶液：取氨水 $8\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$   
5.2.5 甲醇- $2\%$ 氨水溶液 $(5:95)$ ：取甲醇 $5\mathrm{mL}$ ，加 $2\%$ 氨水(5.2.4) $95~\mathrm{mL}$ ，混匀。

5.2.6 甲醇- $2\%$ 氨水溶液 $(40:60)$ ：取甲醇 $40~\mathrm{mL}$ ，加 $2\%$ 氨水(5.2.4) $60~\mathrm{mL}$ ，混匀。  
5.2.7 丙酮-甲醇溶液 $(80:20)$ ：取丙酮 $80~\mathrm{mL}$ ，加甲醇 $20~\mathrm{mL}$ ，混匀。

# 5.3 标准品

5.3.1 $\alpha$ 群勃龙（ $17\alpha$ -Trenbolone， $\mathrm{C_{18}H_{22}O_2}$ ，CAS号：80657-17-6）：含量 $\geq 98.5\%$   
5.3.2 $\beta$ 群勃龙（ $17\beta$ -Trenbolone， $\mathrm{C_{18}H_{22}O_2}$ ，CAS号：10161-33-8）：含量 $\geq 98.0\%$

# 5.4 标准溶液制备

5.4.1 标准储备液：取 $\alpha$ 群勃龙、 $\beta$ 群勃龙标准品各约 $10\mathrm{mg}$ ，精密称定，用甲醇适量使溶解并稀释定容至 $10\mathrm{mL}$ 容量瓶，配制成浓度均为 $1\mathrm{mg/mL}$ 的 $\alpha$ 群勃龙、 $\beta$ 群勃龙标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期6个月。  
5.4.2 混合标准中间液：准确量取 $\alpha$ 群勃龙、 $\beta$ 群勃龙标准储备液各 $1\mathrm{mL}$ ，于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $100~\mu \mathrm{g / mL}$ 的混合标准中间液。一 $18^{\circ}C$ 以下保存，有效期3个月。  
5.4.3 混合标准工作液：分别精密量取混合标准中间液适量，用甲醇稀释，配制成浓度分别为 $20\mathrm{ng / mL}$ 、 $50\mathrm{ng / mL},100\mathrm{ng / mL},200\mathrm{ng / mL},500\mathrm{ng / mL}$ 和 $1000\mathrm{ng / mL}$ 的混合标准工作液。现配现用。

# 5.5 材料

5.5.1 HLB固相萃取柱： $200\mathrm{mg / 3mL}$ ，或相当者。  
5.5.2 氨基固相萃取柱： $500\mathrm{mg / 6mL}$ ，或相当者。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和感量 $0.01\mathrm{g}$   
6.3 pH计。  
6.4 离心机：转速可达 $5000\mathrm{r / min}$   
6.5 涡旋混合器。  
6.6 恒温摇床。  
6.7 水平振荡器。  
6.8 固相萃取装置。  
6.9 氮吹仪。  
6.10 尼龙微孔滤膜: $0.22 \mu \mathrm{m}$ 。

# 7 试料的制备与保存

# 7.1 试料的制备

# 7.1.1 肌肉、脂肪、肝脏和肾脏组织

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.1.2 奶

取适量新鲜或解冻的空白或供试奶，混合均匀。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.1.3 鸡蛋

取适量新鲜或冷藏的空白或供试鸡蛋，去壳，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
e）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 酶解

取试料 $5\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，置 $50~\mathrm{mL}$ 离心管，加 $0.04\mathrm{mol / L}$ 乙酸钠溶液 $10~\mathrm{mL}$ ，涡旋混合，用冰乙酸调 $\mathrm{pH}$ 至 $4.3\sim 4.8$ 。加 $\beta$ 葡萄糖苷酸酶/芳基硫酸酯酶 $20~\mu \mathrm{L}$ ，涡旋混合，于恒温水浴摇床， $37^{\circ}C$ 振荡 $14\mathrm{h}$ 以上。

# 8.2 提取

酶解液冷却至室温，加叔丁基甲醚 $10\mathrm{mL}$ （牛奶、羊奶试料先加入乙腈 $5\mathrm{mL}$ ，涡旋混合，振荡 $10\mathrm{min}$ $5000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上层液于另一离心管中。下层液用叔丁基甲醚 $10\mathrm{mL}$ 重复提取一次，合并2次上层液，于 $50^{\circ}C$ 水浴氮气吹干。加入 $80\%$ 甲醇水溶液 $4\mathrm{mL}$ ，超声 $2\mathrm{min}$ ，溶解残余物。加正己烷 $5\mathrm{mL}$ （脂肪试料加 $10\mathrm{mL}$ )，振荡 $5\mathrm{min}$ ， $5000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，弃上层正己烷。再加入正己烷 $5\mathrm{mL}$ （脂肪试料加 $10\mathrm{mL}$ ）重复上述操作。取下层水相， $50^{\circ}C$ 水浴氮气吹至约 $0.5\mathrm{mL}$ ，加入 $10\%$ 甲醇水溶液 $3\mathrm{mL}$ ，超声 $5\mathrm{min}$ ，溶解残余物备用。

# 8.3 净化

HLB固相萃取柱依次用甲醇 $3\mathrm{mL}$ 、水 $3\mathrm{mL}$ 活化，取备用液过柱，依次用甲醇- $2 \%$ 氨水溶液 $(5:95)$ $3\mathrm{mL}$ 、甲醇- $2 \%$ 氨水溶液 $(40:60)3\mathrm{mL}$ 和水 $3\mathrm{mL}$ 淋洗，抽干。取用丙酮-甲醇溶液 $(80:20)5\mathrm{mL}$ 活化的氨基固相萃取柱串接在HLB固相萃取柱下方，用丙酮-甲醇 $(80:20)5\mathrm{mL}$ 洗脱，收集洗脱液，于 $50^{\circ}C$ 水浴氮气吹干。用 $80 \%$ 甲醇溶液 $1.0\mathrm{mL}$ 溶解残余物，超声 $5\mathrm{min}$ ，涡旋 $1\mathrm{min}$ ，过 $0.22\mu \mathrm{m}$ 尼龙微孔滤膜，供液相色谱-串联质谱测定。

# 8.4 基质添加标准曲线的制备

准确量取混合标准工作液适量，分别加入6份空白试料中，添加浓度分别为 $0.4\mu \mathrm{g / kg},1\mu \mathrm{g / kg}$ 产 $2\mu \mathrm{g} / \mathrm{kg},4\mu \mathrm{g} / \mathrm{kg},10\mu \mathrm{g} / \mathrm{kg}$ 和 $20~\mu \mathrm{g / kg}$ ，充分混匀。按酶解、提取和净化步骤操作，制成浓度分别为 $2\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL},50\mathrm{ng / mL}$ 和 $100\mathrm{ng / mL}$ 的系列基质添加标准溶液，供液相色谱-串联质谱仪测定。以定量离子对质量色谱峰面积为纵坐标、相对应的标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.6 测定法

# 8.6.1 定性测定

在同样测试条件下，试料溶液中 $\alpha$ 群勃龙和 $\beta$ 群勃龙色谱峰的保留时间与基质添加标准溶液中相应峰的保留时间相对偏差在 $\pm 2.5\%$ 以内，且检测到的相对离子丰度，应当与浓度相当的基质添加标准溶液相对离子丰度一致。其允许偏差应符合表3的要求。

# 8.6.2 定量测定

取试料溶液和相应的基质添加标准溶液，作单点或多点校准，按外标法以特征离子质量色谱峰面积定量，基质添加标准溶液及试料溶液中的 $\alpha$ 群勃龙和 $\beta$ 群勃龙响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，基质添加标准溶液中 $\alpha$ 群勃龙和 $\beta$ 群勃龙特征离子质量色谱图见附录A。

# 8.7 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算与表述

试料中 $\alpha$ 群勃龙或 $\beta$ 群勃龙残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V}{A _ {\mathrm {S}} \times m} \tag {1}
$$

式中：

$X$ ——供试试料中相应的 $\alpha$ 群勃龙或 $\beta$ 群勃龙残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ 基质匹配标准溶液中相应 $\alpha$ 群勃龙或 $\beta$ 群勃龙浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$A_{s}$ ——基质匹配标准溶液中相应 $\alpha$ 群勃龙或 $\beta$ 群勃龙峰面积；

A. 供试试料溶液中 $\alpha$ 群勃龙或 $\beta$ 群勃龙峰面积；

$V$ ——溶解残余物所用溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——供试试料质量的数值，单位为克(g)。

# 10 检测方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法在肝脏中的检测限为 $1.0\mu \mathrm{g / kg}$ ，定量限为 $2.0~\mu \mathrm{g / kg}$ ；在肌肉、肾脏、脂肪、鸡蛋和奶中的检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1.0\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法在肝脏 $2\mu \mathrm{g / kg}\sim 10~\mu \mathrm{g / kg}$ 添加浓度水平的回收率为 $60\% \sim 120\%$ ，在肌肉、肾脏、脂肪、鸡蛋和奶 $1\mu \mathrm{g / kg}\sim 10~\mu \mathrm{g / kg}$ 添加浓度水平的回收率为 $60\% \sim 120\%$ 。

# 10.3 精密度

本方法的批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

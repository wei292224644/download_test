# 动物性食品中五氯酚钠残留量的测定

# 气相色谱-质谱法

# 1 范围

本标准规定了动物性食品中五氯酚钠残留量检测的制样和气相色谱质谱法。

本标准适用于猪的肌肉、肝脏和肾脏及鸡的肌肉和肝脏组织中五氯酚钠残留量的检测。

# 3原理

试料中残留的五氯酚钠，用三氯乙酸沉淀蛋白，环己烷-乙酸乙酯溶液提取，乙酸酐-吡啶溶液衍生，气相色谱-质谱法测定，外标法定量。

# 4 试剂和材料

以下所用的试剂，除特别注明外均为分析纯试剂；水为符合GB/T6682规定的一级水。

4.1 五氯酚钠标准品：含量 $\geqslant 98.0\%$   
4.2 碳酸钾   
4.3 正己烷  
4.4 环己烷   
4.5 乙酸乙酯  
4.6 乙酸酐  
4.7 吡啶

# 4.8 三氯乙酸

# 4.9 浓盐酸

4.10 $5\%$ 三氯乙酸溶液：取氯乙酸溶液 $5\mathrm{g}$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。  
4.11 $2.4\mathrm{mol} / \mathrm{L}$ 盐酸溶液：取浓盐酸 $20~\mathrm{mL}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
4.12 环己烷-乙酸乙酯溶液：取乙酸乙酯 $10\mathrm{mL}$ ，加环己烷 $40\mathrm{mL}$ ，混匀。  
4.13 乙酸酐-吡啶溶液：取乙酸酐 $10\mathrm{mL}$ ，加吡啶 $10\mathrm{mL}$ ，混匀，现配现用。  
4.14 $0.2\mathrm{mol} / \mathrm{L}$ 碳酸钾溶液：取碳酸钾 $27.6\mathrm{g}$ ，用水溶解并稀释至 $1000\mathrm{mL}$ 。  
4.15 $1 \mathrm{mg} / \mathrm{mL}$ 五氯酚钠标准贮备液：精密称取五氯酚钠标准品 $10 \mathrm{mg}$ ，于 $10 \mathrm{~mL}$ 量瓶中，用水溶解并稀释至刻度，配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的五氯酚钠标准贮备液。-20°C以下保存，有效期为3个月。  
4.16 $2 \mu \mathrm{g} / \mathrm{mL}$ 五氯酚钠标准工作液: 精密量取 $1 \mathrm{mg} / \mathrm{mL}$ 五氯酚钠标准贮备液 $200 \mu \mathrm{L}$ , 于 $100 \mathrm{~mL}$ 量瓶中, 用水稀释至刻度, 配制成浓度为 $2 \mu \mathrm{g} / \mathrm{mL}$ 的五氯酚钠标准工作液, 现配现用。

# 5 仪器和设备

5.1 气相色谱质谱仪：配EI源。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 高速离心机  
5.5 振荡器  
5.6 旋涡混合器  
5.7 组织匀浆机  
5.8 氮吹仪   
5.9 旋转蒸发仪   
5.10 离心管： $50~\mathrm{mL}$

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

——取均质的供试样品，作为供试试料。

——取均质的空白样品，作为空白试料。  
——取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-20°C以下贮存备用。

# 7 测定步骤

# 7.1 基质匹配标准曲线的制备

准确量取 $2\mu \mathrm{g} / \mathrm{mL}$ 五氯酚钠标准工作液适量，用水配制成浓度为2、5、10、20、50、 $100~\mu \mathrm{g} / \mathrm{L}$ 的系列标准溶液，各取 $1.0\mathrm{mL}$ ，分别溶解经提取、蒸吹干后的6份空白试料残余物，混匀，用 $2.4\mathrm{mol} / \mathrm{L}$ 盐酸溶液调 $\mathrm{pH}$ 至小于2.0，加环己烷-乙酸乙酯溶液 $5\mathrm{mL}$ ，旋涡混合， $8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，移取有机层，再加环己烷-乙酸乙酯溶液 $5\mathrm{mL}$ ，旋涡混合， $8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，合并两次有机层，于 $45^{\circ}\mathrm{C}$ 旋转蒸干。按衍生化步骤操作，供气相色谱质谱测定。以特征离子质量色谱峰面积为纵坐标，对照溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.2 提取

称取试料（ $5 \pm 0.05$ ）g，于 $50 \mathrm{~mL}$ 离心管中，加水 $10 \mathrm{ml}$ ，旋涡混匀， $8000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取上清液，加 $5 \%$ 三氯乙酸 $5 \mathrm{ml}$ ，旋涡混匀， $8000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取上清液，用 $2.4 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液调 $\mathrm{pH}$ 至小于2.0，加环己烷-乙酸乙酯溶液 $5 \mathrm{~mL}$ ，旋涡混合， $8000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，移取有机层，再加环己烷-乙酸乙酯溶液 $5 \mathrm{~mL}$ ，旋涡混合， $8000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，合并两次有机层，于 $45^{\circ} \mathrm{C}$ 旋转蒸干。

# 7.3 衍生化

残余物中加环己烷-乙酸乙酯 $1.0\mathrm{mL}$ ，加无水硫酸钠少量，加乙酸酐-吡啶溶液 $100~\mu \mathrm{L}$ ，密封，于 $60^{\circ}\mathrm{C}$ 反应 $15\mathrm{min}$ ，冷却，加碳酸钾溶液 $0.5\mathrm{mL}$ ，充分混匀，静置分层，取有机相，供气相色谱质谱测定。

# 7.4 测定

# 7.4.1 色谱条件

色谱柱：DB-5MS $5\%$ 苯基甲基聚硅氧烷（ $30\mathrm{m}\times 0.25\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ），或相当者；

进样口温度： $250^{\circ}\mathrm{C}$

进样方式：不分流进样；

进样体积： $2\mu \mathrm{L}$

柱温：初温 $140^{\circ}\mathrm{C}$ ，保留 $2\mathrm{min}$ ，以 $10^{\circ}\mathrm{C / min}$ 升温至 $200^{\circ}\mathrm{C}$ ，以 $15^{\circ}\mathrm{C / min}$ 升温至 $280^{\circ}\mathrm{C}$ ，保持 $3.0\mathrm{min}$ 。

载气：氦气， $1.0 \mathrm{~mL} / \mathrm{min}$ （恒流）

# 7.4.2 质谱条件：

EI源：70ev;

溶剂延迟：3min；

GC/MS 传输线温度： $280^{\circ}\mathrm{C}$ ；

选择离子检测方式，检测离子 $\mathrm{Sim}(\mathrm{M} / \mathrm{Z})$ ：266，308，237，165，130;

定量离子：266。

# 7.4.3 测定法

通过试样色谱图的保留时间与标准溶液的保留时间、色谱峰的特征离子与相应浓度标准品各色谱峰的特征离子相对照定性。试样溶液中的离子相对丰度与基质匹配标准溶液中的离子相对丰度相比，符合表1的要求。基质匹配标准溶液和添加试液中特征离子的质量色谱图分别见附录A。

# 8 结果计算与表述

单点校准: $C = \frac{C s A}{A s}$

或基质匹配标准曲线校准：由 $As = aCs + b$

求得a和b，则 $C = \frac{A - b}{a}$

试料中五氯酚钠的残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算

$$
X = \frac {C V}{m}
$$

式中：

$X$ ——供试试料中五氯酚钠的残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

Cs——标准溶液中五氯酚钠浓度，ng/mL；

C——供试试料溶液中五氯酚钠浓度，ng/mL；

As——标准溶液中五氯酚钠的衍生物的峰面积；

$A$ ——供试试料溶液中五氯酚钠的衍生物的峰面积；

$V$ ——残余物定容体积, $\mathrm{mL}$ ;

$m$ ——供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9检测方法灵敏度、准确度、精密度

# 9.1灵敏度

本方法的检测限为 $0.25\mu \mathrm{g / kg}$

本方法的定量限, 肌肉组织中为 $0.5 \mu \mathrm{g} / \mathrm{kg}$ , 肝脏和肾脏组织中为 $1 \mu \mathrm{g} / \mathrm{kg}$ 。

# 9.2 准确度

本方法在肌肉组织 $0.5 \sim 2 \mu \mathrm{g} / \mathrm{kg}$ 、在肝脏和肾脏组织 $1 \sim 4 \mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率范围为 $60\% \sim 120\%$ 。

# 9.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

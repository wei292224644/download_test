# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2574—2010《进出口蜂王浆中双甲脒及其代谢产物残留量检测方法气相色谱一质谱法》。与SN/T2574—2010相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“进出口蜂王浆”改为“蜂王浆”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T2574—2010。

# 气相色谱一质谱法

# 1 范围

本标准规定了蜂王浆中双甲脒及其代谢产物残留量的气相色谱一质谱测定及确证方法。

本标准适用于蜂王浆中双甲脒及其代谢产物残留量的测定和确证，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

样品经酸水解，碱化后用正己烷一乙醚(2+1,体积比)混合溶剂提取，酸、碱液一液分配净化，气相色谱一质谱测定和确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
4.1.2 乙醚 $\mathrm{(C_4H_{10}O)}$ ：色谱纯。  
4.1.3 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ ：色谱纯。  
4.1.4 氯化钠 $(\mathrm{NaCl})$ ： $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，在干燥器内冷却至室温，储于密封瓶中备用。

# 4.2 溶液配制

4.2.1 盐酸溶液 $(2.0\mathrm{mol} / \mathrm{L})$

用移液管移取 $90~\mathrm{mL}$ 盐酸，加水稀释定容至 $500~\mathrm{mL}$

4.2.2 氢氧化钠溶液 $(10\mathrm{mol} / \mathrm{L})$

称取 $40\mathrm{g}$ 氢氧化钠，加水溶解定容至 $100~\mathrm{mL}$

# 4.3 标准品

4.3.1 双甲脒(amintraz, $\mathrm{C_{19}H_{23}N_3}$ , CAS号: 33089-61-1, 相对分子质量为293)标准物质: 纯度 $>99\%$ 。  
4.3.2 2,4-二甲基苯胺(2,4-Dimethyl aniline, $\mathrm{C_8H_{11}N}$ , CAS号:95-68-1,相对分子质量为121)标准物质:纯度 $>99\%$ 。

# 4.4 标准溶液配制

# 4.4.1 双甲脒标准溶液

准确称取适量的双甲脒标准物质, 用丙酮配成浓度为 $100 \mu \mathrm{g} / \mathrm{mL}$ 的标准储备液。根据需要, 用丙

酮稀释至适当浓度的标准工作液。保存于 $4^{\circ}\mathrm{C}$ 冰箱内。

# 4.4.2 2,4-二甲基苯胺标准溶液

准确称取适量的2,4-二甲基苯胺标准物质，用丙酮配成浓度为 $100\mu \mathrm{g / mL}$ 的标准储备液。根据需要，用丙酮稀释至适当浓度的标准工作液。保存于 $4^{\circ}\mathrm{C}$ 冰箱内。

# 5 仪器和设备

5.1 气相色谱—质谱仪：配有电子轰击源（EI）。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 涡旋振荡器。  
5.4 离心机：转速 $>5000\mathrm{r / min}$   
5.5 套式恒温器： $50^{\circ}\mathrm{C}\sim 200^{\circ}\mathrm{C}$   
5.6 全玻璃回流装置。

# 6 试样制备与保存

样品取样部位按GB2763的规定执行。取代表性样品 $500\mathrm{g}$ ，将其用力搅拌均匀，装入洁净容器内，密封，并标明标记。试样于 $-18^{\circ}\mathrm{C}$ 冷冻保存。

在抽样和制样的操作过程中，应防止样品污染或发生残留物含量的变化。

# 7 分析步骤

# 7.1 水解

称取 $4 \mathrm{~g}$ 试样(精确到 $0.01 \mathrm{~g}$ ) 于 $100 \mathrm{~mL}$ 烧瓶中, 加 $15 \mathrm{~mL}$ 盐酸溶液 (4.5), 于涡旋振荡器上混匀。装上回流冷凝装置, $120^{\circ} \mathrm{C}$ 加热回流 $2 \mathrm{~h}$ , 停止加热, 在室温下继续通冷却水 $15 \mathrm{~min}$ , 移至冰浴中冷却 $15 \mathrm{~min}$ 。

# 7.2 提取和净化

将水解液转移到 $50 \mathrm{~mL}$ 具塞离心管中, 先加 $10 \mathrm{~mL}$ 正己烷一乙醚 $(2+1$ , 体积比) 混合溶剂, 再加 $4 \mathrm{~mL}$ 氢氧化钠溶液, 涡旋混匀, 在冰浴中冷却 $5 \mathrm{~min}$ , 以 $4000 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ 。将上层有机相转移到另一 $50 \mathrm{~mL}$ 具塞离心管中。水相分 2 次, 每次用 $10 \mathrm{~mL}$ 正己烷一乙醚 $(2+1$ , 体积比) 混合溶剂萃取, 合并上层有机相。分 2 次, 每次用 $5 \mathrm{~mL}$ 盐酸溶液萃取, 合并盐酸提取液并定容至 $10 \mathrm{~mL}$ , 分取 $5 \mathrm{~mL}$ 盐酸提取液于 $10 \mathrm{~mL}$ 具塞离心管中。加 $2 \mathrm{~mL}$ 正己烷一丙酮 $(2+1$ , 体积比) 混合溶剂, $2 \mathrm{~mL}$ 氢氧化钠溶液萃取 1 次, 离心分层。上层有机相供气相色谱一质谱测定, 外标法定量。

# 7.3 测定

# 7.3.1 气相色谱一质谱参考条件

a）色谱柱：DB-5ms石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径），膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b) 色谱柱温度: $50^{\circ} \mathrm{C}$ 保持 $2 \mathrm{~min}$ ; 以 $10^{\circ} \mathrm{C} / \mathrm{min}$ 的升温速率升至 $160^{\circ} \mathrm{C}$ , 保持 $1 \mathrm{~min}$ ; 以 $30^{\circ} \mathrm{C} / \mathrm{min}$ 的升温速率升至 $320^{\circ} \mathrm{C}$ , 保持 $5 \mathrm{~min}$ ;  
c）载气：氦气，纯度 $\geq 99.999\%$ ，流速 $1.0~\mathrm{mL / min}$   
d) 进样量: $2 \mu \mathrm{L}$ ;   
e）进样方式：脉冲不分流进样，脉冲压力 $25\mathrm{psi},0.75\mathrm{min}$ 后开阀；  
f）溶剂延迟时间： $8\mathrm{min}$   
g）进样口温度： $250^{\circ}C$   
h) 接口温度: $280^{\circ} \mathrm{C}$ ;   
i）溶剂延迟时间： $8\mathrm{min}$

j）离子源：电子轰击源（EI）；  
k）电离能量： $70\mathrm{eV}$   
1) 测定方式：选择离子监测方式；  
m）选择离子及相对丰度：见表1。

# 7.3.2 定量测定

根据样液中2,4-二甲基苯胺含量的情况，选定与样液中2,4-二甲基苯胺浓度相近的标准工作溶液。标准工作溶液和样液中2,4-二甲基苯胺响应值应在仪器测定的线性范围内。标准工作溶液和样液等体积穿插进样测定。在上述色谱条件下，2,4-二甲基苯胺的保留时间约为 $10.0\mathrm{min}$ ，标准品的总离子流图参见附录A中图A.1。

# 7.3.3 定性测定

对标准溶液及样液均按7.3.1规定的条件进行测定。如果样液与标准溶液在相同的保留时间有峰出现，则对其进行质谱确证。在扣除背景后的样品谱图中，所选择离子全部出现，同时所选择的离子的离子丰度比与标准品相关离子的相对丰度一致，波动范围符合表2的最大容许偏差之内（见表2），可判定样品中存在2,4-二甲基苯胺。被确证的样品可判定为2,4-二甲基苯胺阳性检出。2,4-二甲基苯胺标准品的质谱图参见图A.2。

# 7.4 空白试验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理软件或按式(1)计算试样中双甲脒的残留含量。

$$
X = \frac {1 . 2 1 \times A \times c \times V}{A _ {S} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中双甲脒残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

1.21——由2,4-二甲基苯胺分子量换算为双甲脒分子量的转换系数；

A ——样液中2,4-二甲基苯胺的峰面积；

$c$ ——标准工作液中2,4-二甲基苯胺的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——最终样液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{S}$ ——标准工作液中2,4-二甲基苯胺峰面积；

$m$ ——最终样液所代表的试样量，单位为克(g)。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

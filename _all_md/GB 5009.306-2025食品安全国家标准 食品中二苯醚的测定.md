# 食品安全国家标准

# 食品中二苯醚的测定

# 1 范围

本标准规定了食品中二苯醚的气相色谱和气相色谱-质谱的测定方法。  
本标准适用于水果、水果制品、果蔬汁、软糖中二苯醚的测定。

# 第一法 气相色谱法

# 2 原理

试样中的二苯醚用正己烷提取，经盐析萃取，萃取液再经 $N$ -丙基乙二胺(PSA)固相吸附剂、石墨化碳黑(GCB)固相吸附剂净化，气相色谱仪配备氢火焰离子化检测器检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
3.1.2 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.3 无水硫酸镁 $(\mathrm{MgSO_4})$ 。  
3.1.4 柠檬酸钠 $(\mathrm{C_6H_5Na_3O_7})$ 。  
3.1.5 柠檬酸氢二钠 $(\mathrm{C_6H_6Na_2O_7})$ 。

# 3.2 标准品

二苯醚(Phenyl ether, $\mathrm{C_{12}H_{10}O}$ , CAS号: 101-84-8): 标准物质纯度 $\geqslant 99\%$ , 或经国家认证并授予标准物质证书的标准品。

# 3.3 标准溶液配制

3.3.1 二苯醚标准储备溶液 $(1000\mathrm{mg / L})$ : 准确称取 $10\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ) 二苯醚标准品, 用正己烷溶解并定容至 $10\mathrm{mL}$ , 于 $4^{\circ}\mathrm{C}$ 避光贮存, 保存期为 3 个月, 或直接购买溯源性符合要求的标准溶液。  
3.3.2 二苯醚标准中间溶液 $(10.0\mathrm{mg / L})$ ：准确吸取二苯醚标准储备溶液 $0.25~\mathrm{mL}$ 于 $25~\mathrm{mL}$ 容量瓶中，用正己烷定容。于 $4^{\circ}C$ 避光贮存，保存期为1周。  
3.3.3 标准工作溶液：准确吸取二苯醚标准中间液 $0.50\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},5.00\mathrm{mL}$ 和 $10.00~\mathrm{mL}$ ，用正己烷定容至 $10~\mathrm{mL}$ ，配制成质量浓度分别为 $0.5\mu \mathrm{g / mL},1.0\mu \mathrm{g / mL},2.0\mu \mathrm{g / mL},$ $5.0\mu \mathrm{g / mL}$ 和 $10.0\mu \mathrm{g / mL}$ 的标准工作溶液。临用现配。

# 3.4 材料

3.4.1 $N$ -丙基乙二胺(PSA)固相吸附剂：粒度 $40~\mu \mathrm{m}\sim 60~\mu \mathrm{m}$ 。  
3.4.2 石墨化碳黑(GCB)固相吸附剂：粒度 $120\mu \mathrm{m}\sim 400\mu \mathrm{m}$ 。

# 4 仪器和设备

4.1 气相色谱仪：配有氢火焰离子化检测器(FID)。  
4.2 分析天平：感量分别为 $0.001\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
4.3 离心机：转速 $\geqslant 5000\mathrm{r / min}$   
4.4 旋涡混合器。

# 5 分析步骤

# 5.1 试样制备

水果样品参照GB2763附录A食品类别及测定部位取样、水果制品取可食部分、软糖取全部，经捣碎机充分捣碎均匀；果蔬汁样品充分混匀后直接取样。样品的取样量应不少于 $500\mathrm{g}$ ，制备好的试样应装入洁净容器，密封，标注样品信息尽快检测。

# 5.2 提取

5.2.1 水果、果蔬汁：称取试样 $5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $10~\mathrm{mL}$ 正己烷，涡旋振荡提取 $1\mathrm{min}$ 。  
5.2.2 水果制品、软糖：称取试样 $5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $10~\mathrm{mL}$ 约 $50^{\circ}C$ 温水，超声溶解 $30\mathrm{min}$ 后，准确加入 $10~\mathrm{mL}$ 正己烷，涡旋振荡提取 $1\mathrm{min}$ 。

# 5.3 盐析

分别称取 $1\mathrm{g}$ 氯化钠、 $4\mathrm{g}$ 无水硫酸镁、 $1\mathrm{g}$ 柠檬酸钠、 $0.5\mathrm{g}$ 柠檬酸氢二钠于正己烷-水混合溶液中，涡旋混匀 $1\mathrm{min}$ ，静置冷却至室温，于 $5000\mathrm{r / min}$ 条件下离心 $5\mathrm{min}$ ，待净化。

注：也可直接使用吸附剂相同的商品化盐包。

# 5.4 净化

转移出 $5\mathrm{mL}$ 上层正己烷提取液于含有 $885~\mathrm{mg}$ 无水硫酸镁、 $150~\mathrm{mgN}$ -丙基乙二胺（PSA）固相吸附剂、 $15\mathrm{mg}$ 石墨化碳黑(GCB)固相吸附剂的离心管中，旋涡混匀 $1\mathrm{min}$ ，于 $5000~\mathrm{r / min}$ 条件下离心 $5\mathrm{min}$ ，取上层清液约 $1\mathrm{mL}$ 待测定。

注：也可直接使用吸附剂相同的商品化净化包。

# 5.5 仪器参考条件

# 5.5.1 气相色谱参考条件

气相色谱参考条件如下：

a) 色谱柱: $5 \%$ 苯基-甲基聚硅氧烷非极性柱 $\left[30 \mathrm{~m} \times 0.32 \mathrm{~mm}\right.$ (内径), $0.25 \mu \mathrm{m}$ (膜厚)], 或相当者;   
b）色谱柱温度： $80~^\circ \mathrm{C}$ 保持 $1\mathrm{min}$ ，以 $10^{\circ}C / \mathrm{min}$ 的升温速率升至 $150~\mathrm{^\circ C}$ ，保持 $10\mathrm{min}$ ，以

$30^{\circ}\mathrm{C / min}$ 的升温速率升至 $280^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$

c）进样口温度： $285~\mathrm{^\circ C}$   
d) 载气流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
e) 燃烧气：氢气，流量约 $30~\mathrm{mL / min}$   
f) 助燃气：空气，流量约 $300~\mathrm{mL / min}$   
g）进样量： $1\mu \mathrm{L}$   
h）进样方式：不分流进样， $1.0\mathrm{min}$ 后开阀。

# 5.5.2 标准曲线的制作

将标准系列工作液分别注入气相色谱仪中，测定相应的峰面积。以标准工作液中的二苯醚的浓度为横坐标、峰面积为纵坐标，绘制标准曲线。二苯醚标准溶液的色谱图见附录A中图A.1。

# 5.5.3 试样溶液的测定

将试样溶液注入气相色谱仪中，根据标准曲线得到待测液中二苯醚的浓度。若试样溶液中二苯醚的质量浓度超过标准曲线范围，则应进行适当稀释后测定。

# 5.6 空白试验

除不加试样外，均按上述测定步骤进行。

# 6 分析结果的表述

试样中二苯醚的含量按式(1)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中二苯醚的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由标准曲线得出的试样液中待测物的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$\rho_0$ ——由标准曲线得出的空白中待测物的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

V ——加入提取液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算因子。

注：计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不应超过算术平均值的 $10\%$ 。

# 8 其他

称样量为 $5.0\mathrm{g}$ ，提取液体积为 $10~\mathrm{mL}$ 时，检出限为 $0.5\mathrm{mg / kg}$ ，定量限为 $2.0\mathrm{mg / kg}$ 。

# 12.5.2 标准曲线的制作

将标准系列工作液分别注入气相色谱-质谱仪中，测定相应的峰面积，以标准系列工作液中二苯醚的浓度为横坐标、峰面积为纵坐标、绘制标准曲线。

# 12.6 定性测定

按照气相色谱-质谱条件测定试样溶液和标准工作溶液，如果被测化合物保留时间与标准品保留时间相差不超过 $\pm 0.5\%$ ，并且在扣除背景后的样品质谱图中，所选择的离子均出现且信噪比 $\geqslant 3$ ，定性离子对的相对丰度(是用相对于最强离子丰度的强度百分比表示)与浓度相当的标准工作溶液的相对丰度允许偏差不超过 $\pm 15\%$ 的范围，则可判断样品中存在对应的被测物。二苯醚标准溶液的选择离子流色谱图和质谱图见附录B中图B.1～图B.2。

# 12.7 定量测定

将试样溶液注入气相色谱-质谱仪中，得到二苯醚的峰面积根据标准曲线得到试样溶液中二苯醚的浓度。若试样溶液中二苯醚的质量浓度超过标准曲线范围，则应进行适当稀释后测定。

# 12.8 空白试验

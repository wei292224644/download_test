# 前言

本标准代替GB/T5413.21-1997《婴幼儿配方食品和乳粉钙、铁、锌、钠、钾、镁、铜、和锰的测定》。

本标准与GB/T5413.21-1997相比，主要变化如下：

——第一法中增加了可以直接购买有证标准溶液；  
——第一法中修改了标准储备液及标准工作液的配制浓度；  
——第一法中修改了试样处理稀释步骤；  
——增加了第二法电感耦合等离子体原子发射光谱测定方法。

本标准所代替标准的历次版本发布情况为：

GB5413-1985、GB/T5413.21-1997。

# 第一法 火焰原子吸收分光光度法

# 3 原理

试样经干法灰化，分解有机质后，加酸使灰分中的无机离子全部溶解，直接吸入空气—乙炔火焰中原子化，并在光路中分别测定钙、铁、锌、钠、钾、镁、铜和锰原子对特定波长谱线的吸收。测定钙、镁时，需用镧作释放剂，以消除磷酸干扰。

# 6 分析步骤

# 6.1 试样处理

称取混合均匀的固体试样约 $5\mathrm{g}$ 或液体试样约 $15\mathrm{g}$ （精确到 $0.0001\mathrm{g}$ ）于坩埚（5.4）中，在电炉上微火炭化至不再冒烟，再移入马弗炉（5.5）中， $490^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 灰化约 $5\mathrm{h}$ 。如果有黑色炭粒，冷却后，则滴加少许硝酸溶液（4.14）湿润。在电炉上小火蒸干后，再移入 $490^{\circ}\mathrm{C}$ 高温炉中继续灰化成白色灰烬。冷却至室温后取出，加入 $5\mathrm{mL}$ 盐酸B（4.13），在电炉上加热使灰烬充分溶解。冷却至室温后，移入 $50\mathrm{mL}$ 容量瓶中，用水定容，同时处理至少两个空白试样。

# 6.2 试样待测液的制备

# 6.2.1 钙、镁待测液

从 $50~\mathrm{mL}$ 的试液（6.1）中准确吸取 $1.0~\mathrm{mL}$ 到 $100~\mathrm{mL}$ 容量瓶中，加 $2.0~\mathrm{mL}$ 镧溶液（4.15），用水定容。同样方法处理空白试液。

# 6.2.2 钠待测液

从 $50~\mathrm{mL}$ 的试液（6.1）中准确吸取 $1.0~\mathrm{mL}$ 到 $100~\mathrm{mL}$ 容量瓶中，用盐酸A（4.12）定容。同样方法处理空白试液。

# 6.2.3 钾待测液

从 $50 \mathrm{~mL}$ 的试液（6.1）中准确吸取 $0.5 \mathrm{~mL}$ 到 $100 \mathrm{~mL}$ 容量瓶中，用盐酸 A（4.12）定容。同样方法处理空白试液。

# 6.2.4 铁、锌、锰、铜待测液

用 $50~\mathrm{mL}$ 的试液（6.1）直接上机测定。同时测定空白试液（6.1）。

6.2.5 为保证试样待测试液浓度在标准曲线线性范围内，可以适当调整试液定容体积和稀释倍数。

# 7 分析结果的表述

试样中钙、镁、钠、钾、铁、锌的含量按式（1）计算：

$$
X = \frac {\left(c _ {1} - c _ {2}\right) \times V \times f}{m \times 1 0 0 0} \times 1 0 0 \dots \tag {1}
$$

式中：

$X$ ——试样中各元素的含量, 单位为毫克每百克 $(\mathrm{mg} / 100 \mathrm{~g})$ ;

$c_{1}$ 测定液中元素的浓度, 单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ;

$c_{2}$ 测定空白液中元素的浓度，单位为微克每毫升（ $\mu \mathrm{g} / \mathrm{mL}$ ）；

$V$ ——样液体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——样液稀释倍数；

$m$ ——试样的质量，单位为克（g）。

试样中锰、铜的含量按式（2）计算：

$$
X = \frac {\left(c _ {1} - c _ {2}\right) \times V \times f}{m} \times 1 0 0 \dots \tag {2}
$$

式中：

$X$ ——试样中各元素的含量, 单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$ ;

$c_{1}$ ——测定液中元素的浓度, 单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ;

$c_{2}$ ——测定空白液中元素的浓度, 单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ;

$V$ ——样液体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——样液稀释倍数；

$m$ ——试样的质量，单位为克（g）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，钙、镁、钠、钾、锰、铜、铁、锌结果保留三位有效数字。

# 8 精密度

在重复性条件下获得两次独立测定结果的绝对差值，钙、镁、钠、钾、铁、锌不得超过算术平均值的 $10\%$ ；铜和锰不得超过算术平均值的 $15\%$ 。

# 第二法 电感耦合等离子体原子发射光谱测定方法

# 9 原理

试样经干法灰化消解，稀释至合适体积后用电感耦合等离子体原子发射光谱仪测定，外标法定量。

# 12 分析步骤

# 12.1 试样制备

试样为粉末状样品，分析前应将试样充分混匀；试样为较大颗粒样品，应分取适量样品粉碎后测定。

# 12.2 试样处理

称取 $5\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ ）于瓷坩埚中，在电炉上微火炭化至不冒烟，移入马弗炉中 $550^{\circ}\mathrm{C}$ 加热 $2\mathrm{h}$ ，如有黑色炭粒，冷却后加少许硝酸溶液（10.3）湿润，小火蒸干后再移入马弗炉中 $550^{\circ}\mathrm{C}$ 加热半小时，取出冷却，加盐酸B（10.5） $5\mathrm{mL}$ ，在电炉上小心加热使灰分充分溶解，冷却后转移至 $25\mathrm{mL}$ 容量瓶中，用水定容至刻度，若有沉淀需过滤。待测。

# 12.3 平行试验

按以上步骤，对同一试样进行平行试验测定。

# 12.4 空白试验

除不加试样外，其它均按试样处理和测定步骤进行。

# 13 分析结果的表述

按式（3）计算各元素的含量：

$$
X = \frac {1 0 0 \times \left(c _ {1} - c _ {2}\right) \times V \times f}{1 0 0 0 \times m} \dots \tag {3}
$$

式中：

$X$ ——被测元素含量，单位为毫克每百克（ $\mathrm{mg} / 100 \mathrm{~g}$ ）；

$c_{I}$ ——试样溶液中元素的浓度，单位为微克每毫升（ $\mu \mathrm{g} / \mathrm{mL}$ ）；

$c_{2}$ ——空白溶液中元素的浓度，单位为微克每毫升（ $\mu \mathrm{g} / \mathrm{mL}$ ）；

$V$ ——试样溶液体积, 单位为毫升 $(\mathrm{mL})$ ;

$f$ ——试样溶液稀释倍数；

$m$ ——试样的质量，单位为克（g）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 14 精密度

在重复性条件下获得两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 前言

本标准代替GB5009.247—2016《食品安全国家标准 食品中纽甜的测定》。

本标准与GB5009.247—2016相比，主要变化如下：

——修改了标准的适用范围；

——修改了样品前处理方法；

——增加了第二法 液相色谱-串联质谱法。

# 食品安全国家标准

# 食品中纽甜的测定

# 1 范围

本标准规定了食品中纽甜的液相色谱和液相色谱-串联质谱测定方法。

本标准适用于食品中纽甜的测定。

# 第一法 液相色谱法

# 2 原理

试样用混合提取液提取，含胶基试样经水浴溶解、高脂肪试样经正己烷脱脂，蛋白沉淀剂沉淀蛋白，固相萃取净化后液相色谱仪测定，保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 甲酸（HCOOH）：色谱纯。  
3.1.3 三乙胺 $(\mathrm{C_6H_{15}N})$ 。  
3.1.4 乙酸铵 $\mathrm{(CH_3COONH_4)}$ ：色谱纯。  
3.1.5 冰乙酸 $\mathrm{(CH_3COOH)}$ ：色谱纯。  
3.1.6 亚铁氰化钾 $\left[\mathrm{K}_{4} \mathrm{Fe}(\mathrm{CN})_{6} \cdot 3 \mathrm{H}_{2} \mathrm{O}\right]$ 。  
3.1.7 乙酸锌 $\left[\mathrm{Zn}(\mathrm{CH}_3\mathrm{COO})_2\cdot 2\mathrm{H}_2\mathrm{O}\right]$   
3.1.8 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。  
3.1.9 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ 。

# 3.2 试剂配制

3.2.1 甲酸-三乙胺缓冲液：分别吸取 $0.8\mathrm{mL}$ 甲酸和 $2.5\mathrm{mL}$ 三乙胺，加水定容至 $1\mathrm{L},\mathrm{pH}$ 约为4.5。  
3.2.2 混合提取液：量取甲醇 $400~\mathrm{mL}$ 至 $1\mathrm{L}$ 烧杯，加 $500~\mathrm{mL}$ 甲酸-三乙胺缓冲液，用甲酸调 $\mathrm{pH}$ 到4.5，转移至 $1\mathrm{L}$ 容量瓶，用甲酸-三乙胺缓冲液定容至刻度。  
3.2.3 混合洗脱液：量取甲醇 $700~\mathrm{mL}$ 至1L容量瓶，用甲酸-三乙胺缓冲液定容至刻度。  
3.2.4 乙酸铵溶液(20 mmol/L)：称取 $1.54 \mathrm{~g}$ 乙酸铵，用 $500 \mathrm{~mL}$ 水溶解，加入 $1.0 \mathrm{~mL}$ 冰乙酸，加水定容至 $1 \mathrm{~L}$ 。  
3.2.5 亚铁氰化钾溶液 $(92\mathrm{g} / \mathrm{L})$ ：称取 $106\mathrm{g}$ 亚铁氰化钾，加入适量水溶解，用水定容至 $1\mathrm{L}$ 。  
3.2.6 乙酸锌溶液 $(183\mathrm{g} / \mathrm{L})$ ：称取 $220\mathrm{g}$ 乙酸锌，加入适量水溶解，加入 $30~\mathrm{mL}$ 冰乙酸，用水定容至 $1\mathrm{L}$ 。

3.2.7 氨水溶液 $(20\%)$ ：量取 $200~\mathrm{mL}$ 氨水至1L容量瓶，用水定容至刻度。

# 3.3 标准品

纽甜标准品 $\mathrm{(C_{20}H_{30}N_2O_5}$ ,CAS号：165450-17-9)：纯度 $\geqslant 99.0\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 标准储备液 $(1.00\mathrm{mg / mL})$ ：准确称取 $100\mathrm{mg}$ （精确至 $0.0001\mathrm{g}$ )标准品，甲酸-三乙胺缓冲液溶解并定容至 $100~\mathrm{mL}$ 。 $4^{\circ}C$ 下避光保存，有效期3个月。  
3.4.2 标准工作液：分别吸取适量标准储备液 $(1.00\mathrm{mg / mL})$ ，用甲酸-三乙胺缓冲液配制成质量浓度为 $0.500~\mu \mathrm{g / mL},1.00~\mu \mathrm{g / mL},5.00~\mu \mathrm{g / mL},10.0~\mu \mathrm{g / mL},50.0~\mu \mathrm{g / mL}$ 和 $100~\mu \mathrm{g / mL}$ 系列标准工作液，临用现配。

# 3.5 材料

3.5.1 固相萃取柱： $500\mathrm{mg / 6mL}$ ，聚苯乙烯吡咯烷酮聚合物填料，或性能相当者。  
3.5.2 微孔滤膜：有机系， $0.45\mu \mathrm{m}$

# 4 仪器和设备

4.1 高效液相色谱仪：带紫外检测器或二极管阵列检测器。  
4.2 天平：感量分别为 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
4.3 pH计：精度为0.01。  
4.4 涡旋混合器。  
4.5 离心机：转速 $\geq 4000\mathrm{r / min}$   
4.6 氮吹仪。  
4.7 超声波清洗器。  
4.8 粉碎机。  
4.9 水浴锅。  
4.10 固相萃取装置。

# 5 分析步骤

# 5.1 试样前处理

# 5.1.1 试样制备

液态试样摇匀待提取；基质均匀的半固态试样和粉状试样待提取，其他样品需匀浆或粉碎均匀待提取，制备好的试样 $0^{\circ}\mathrm{C} \sim 5^{\circ}\mathrm{C}$ 保存待检测。

# 5.1.2 试样提取

# 5.1.2.1 含胶基的糖果、果冻及鸡精等试样

准确称取 $5\mathrm{g}$ 试样(精确到 $0.01\mathrm{g}$ ) 于 $50~\mathrm{mL}$ 离心管中，加入 $25~\mathrm{mL}$ 混合提取液，于 $60^{\circ}\mathrm{C}$ 水浴振荡 $30\mathrm{min}$ 。冷却至室温，用甲酸或氨水溶液 $(20\%)$ 调 $\mathrm{pH}$ 至4.5，超声 $30\mathrm{min}$ 。加入 $1\mathrm{mL}$ 亚铁氰化钾溶

液 $(92\mathrm{g / L})$ 和 $1\mathrm{mL}$ 乙酸锌溶液 $(183~\mathrm{g / L})$ ，涡旋混匀， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。上清液经滤纸过滤并转移至 $50~\mathrm{mL}$ 容量瓶。残渣加入 $20~\mathrm{mL}$ 混合提取液，振荡 $5\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液经滤纸过滤合并，用混合提取液定容至刻度，作为试样提取液待净化。

# 5.1.2.2 油脂类调味料、巧克力及奶油等试样

准确称取 $5\mathrm{g}$ 试样（精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $25~\mathrm{mL}$ 混合提取液，于 $60^{\circ}\mathrm{C}$ 水浴振荡 $30\mathrm{min}$ 。加 $5\mathrm{mL}$ 正己烷，振荡 $5\mathrm{min}$ ， $4000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去正己烷层。之后的操作同5.1.2.1“用甲酸或氨水溶液 $(20\%)$ 调 $\mathrm{pH}$ 至4.5”后续步骤。

# 5.1.2.3 其余试样

准确称取 $5 \mathrm{~g}$ 试样（精确到 $0.01 \mathrm{~g}$ ）于 $50 \mathrm{~mL}$ 离心管中，加入 $25 \mathrm{~mL}$ 混合提取液，涡旋 $5 \mathrm{~min}$ 。之后的操作同 5.1.2.1“用甲酸或氨水溶液 $(20\%)$ 调 $\mathrm{pH}$ 至 4.5”后续步骤。

# 5.1.3 试样净化

固相萃取柱使用前依次用 $6\mathrm{mL}$ 甲醇和 $6\mathrm{mL}$ 水活化，保持柱体湿润。取 $10.0\mathrm{mL}$ 上述试样提取液过固相萃取柱， $5\mathrm{mL}$ 甲酸-三乙胺缓冲液淋洗，弃去全部流出液。用 $8\mathrm{mL}$ 混合洗脱液洗脱，收集洗脱液于 $60^{\circ}\mathrm{C}$ 水浴氮吹浓缩至约 $1\mathrm{mL}$ ，用甲酸-三乙胺缓冲液定容至 $2.0\mathrm{mL}$ ，作为试样溶液待上机测定。

# 5.2 仪器参考条件

5.2.1 色谱柱： $\mathrm{C_{18}}$ ， $150\mathrm{mm}\times 4.6\mathrm{mm},5\mu \mathrm{m}$ ，或相当者。  
5.2.2 柱温： $30^{\circ}C$   
5.2.3 流动相：A相为甲醇；B相为乙酸铵溶液(20mmol/L)。梯度洗脱程序见表1。  
5.2.4 流速： $1.0~\mathrm{mL / min}$   
5.2.5 检测波长： $210~\mathrm{nm}$   
5.2.6 进样量： $50~\mu \mathrm{L}$

# 5.3 标准曲线的制作

将系列标准工作液分别注入液相色谱仪，测定相应的峰面积。以系列标准工作液中纽甜的浓度为横坐标，以峰面积的响应值为纵坐标，绘制标准曲线。纽甜标准溶液的色谱图见附录A中图A.1。

# 5.4 试样溶液的测定

将试样溶液经 $0.45\mu \mathrm{m}$ 滤膜过滤后注入液相色谱仪。保留时间定性，峰面积定量，根据标准曲线得到试样溶液中纽甜的浓度。

# 5.5 空白试验

除不加试样外，均按上述测定步骤进行。

# 6 分析结果的表述

试样中纽甜含量按公式(1)计算：

$$
X = \frac {\rho \times V \times V _ {1} \times 1 0 0 0}{m \times V _ {2} \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中纽甜含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中纽甜的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——试样提取液定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——用于净化的试样提取液体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留三位有效数字。

# 7 精密度

计算结果在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

当称样量为 $5\mathrm{g}$ 时，本方法的检出限为 $1\mathrm{mg / kg}$ 、定量限为 $5\mathrm{mg / kg}$ 。

# 13 分析结果的表述

试样中纽甜的含量按公式(2)计算：

$$
X = \frac {\rho \times V \times V _ {1} \times 1 0 0 0}{m \times V _ {2} \times 1 0 0 0 \times 1 0 0 0} \quad \dots \tag {2}
$$

式中：

$X$ ——试样中纽甜含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中纽甜的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样溶液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——试样提取液定容体积，单位为毫升 $(\mathrm{mL})$

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——用于净化的试样提取液体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留三位有效数字。

# 14 精密度

计算结果在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 15 其他

当称样量为 $5\mathrm{g}$ 时，本方法的检出限为 $0.03\mathrm{mg / kg}$ 、定量限为 $0.1\mathrm{mg / kg}$ 。

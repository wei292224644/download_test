# 食品安全国家标准

# 鸡可食组织中抗球虫药物残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了鸡可食组织中常山酮、氯苯胍、盐霉素、莫能菌素、甲基盐霉素、马度米星铵和拉沙洛西7种抗球虫药物残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鸡肌肉、肝脏和皮脂(皮+脂)中常山酮、氯苯胍、盐霉素、莫能菌素、甲基盐霉素、马度米星铵和拉沙洛西7种抗球虫药物残留量的检测。

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的抗球虫药物经胰蛋白酶酶解，乙酸乙酯提取，固相萃取柱净化，液相色谱-串联质谱法检测，基质匹配外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.3 乙酸 $\mathrm{(CH_3COOH)}$ ：色谱纯。  
5.1.4 甲酸(HCOOH)：色谱纯。  
5.1.5 乙酸乙酯 $\mathrm{(CH_3CH_2OOCCH_3)}$ ：色谱纯。  
5.1.6 胰蛋白酶：来源于牛胰腺， $\geqslant 10000\mathrm{U / mg}$   
5.1.7 碳酸钠 $\mathrm{(Na_2CO_3)}$

# 5.2 溶液配制

5.2.1 $10\%$ 碳酸钠溶液：取碳酸钠 $20\mathrm{g}$ ，加水适量使溶解并稀释至 $200~\mathrm{mL}$ ，混匀。  
5.2.2 $1\%$ 乙酸溶液：取乙酸 $1\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。  
5.2.3 $20\%$ 甲醇溶液：取甲醇 $20~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。  
5.2.4 洗脱液：取甲醇 $100~\mathrm{mL}$ ，加乙酸乙酯 $200~\mathrm{mL}$ ，混匀。  
5.2.5 复溶液：取甲醇 $50~\mathrm{mL}$ ，加水 $50~\mathrm{mL}$ 、甲酸 $0.1~\mathrm{mL}$ ，混匀。

# 5.3 标准品

常山酮、氯苯胍、盐霉素、莫能菌素、甲基盐霉素、马度米星铵、拉沙洛西含量 $\geq 95\%$ 。具体见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液 $(1\mathrm{mg / mL})$ ：取常山酮、氯苯胍、盐霉素、莫能菌素、甲基盐霉素、马度米星铵、拉沙洛西标准品适量(相当于各有效成分 $10~\mathrm{mg}$ )，精密称定，分别加甲醇适量使溶解并稀释定容于 $10~\mathrm{mL}$ 容量瓶，配制成浓度均为 $1\mathrm{mg / mL}$ 的标准储备液。 $-18^{\circ}C$ 以下避光保存，有效期6个月。  
5.4.2 混合标准工作液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：分别精密量取标准储备液各 $0.1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的混合标准工作液。一 $18^{\circ}C$ 以下避光保存，有效期6个月。  
5.4.3 混合标准工作液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：精密量取 $10~\mu \mathrm{g} / \mathrm{mL}$ 的混合标准工作液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 的混合标准工作液。一 $18^{\circ}\mathrm{C}$ 以下避光保存，有效期3个月。  
5.4.4 混合标准工作液 $(0.1\mu \mathrm{g} / \mathrm{mL})$ ：精密量取 $1\mu \mathrm{g} / \mathrm{mL}$ 的混合标准工作液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $0.1\mu \mathrm{g} / \mathrm{mL}$ 的混合标准工作液。 $-18^{\circ}\mathrm{C}$ 以下避光保存，有效期1个月。

# 5.5 材料

5.5.1 三键键合 $\mathrm{C_{18}}$ 固相萃取柱： $500\mathrm{mg / 6mL}$ ，或相当者。  
5.5.2 尼龙微孔滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$   
6.3 高速冷冻离心机： $10000\mathrm{r / min}$   
6.4 组织匀浆机。  
6.5 涡旋混合器。  
6.6 固相萃取装置。  
6.7 氮吹仪。  
6.8 水浴摇床。  
6.9 pH计。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试组织，绞碎并均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试料 $1\mathrm{g}$ （准确至 $\pm 0.01\mathrm{g}$ ，加胰蛋白酶 $25\mathrm{mg}$ 、水 $5\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，用 $10\%$ 碳酸钠溶液调 $\mathrm{pH}$ 至 $7.5,40^{\circ}C$ 水浴过夜酶解。取出放至室温，加 $10\%$ 碳酸钠溶液 $1\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ 。加乙酸乙酯 $9\mathrm{mL}$ ，涡旋 $5\mathrm{min}$ ，于 $4^{\circ}\mathrm{C},10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，转移上清液，残渣重复提取1次。合并2次提取液，用乙酸乙酯稀释至 $20.0~\mathrm{mL}$ ，混匀。取提取液 $2.0\mathrm{mL}$ 于 $35^{\circ}C$ 水浴氮气吹干，加乙腈 $1\mathrm{mL}$ 涡旋 $1\mathrm{min}$ ，加水 $10~\mathrm{mL}$ ，混匀，备用。

# 8.2 净化

固相萃取柱依次用甲醇 $5\mathrm{mL},1\%$ 乙酸溶液 $5\mathrm{mL}$ 活化，取备用液全部过柱，控制流速为每 $2s\sim 3s$

1滴。用 $1\%$ 乙酸溶液 $3\mathrm{mL}$ 和 $20\%$ 甲醇溶液 $3\mathrm{mL}$ 淋洗，用洗脱液 $10\mathrm{mL}$ 洗脱。收集洗脱液， $35^{\circ}C$ 水浴氮气吹干，加复溶液 $1.0\mathrm{mL}$ 涡旋 $1\mathrm{min}$ ，于 $-4^{\circ}C, 10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液，过滤膜后供液相色谱-串联质谱测定。

# 8.3 基质匹配标准曲线的制备

取6份空白样品经提取和净化后，加入适量的标准工作液， $35^{\circ}\mathrm{C}$ 水浴氮气吹干，加复溶液 $1.0\mathrm{mL}$ 涡旋 $1\mathrm{min}$ ，配制成浓度为 $0.5\mu \mathrm{g} / \mathrm{L}, 1\mu \mathrm{g} / \mathrm{L}, 5\mu \mathrm{g} / \mathrm{L}, 10\mu \mathrm{g} / \mathrm{L}, 25\mu \mathrm{g} / \mathrm{L}$ 和 $50\mu \mathrm{g} / \mathrm{L}$ 的基质匹配标准溶液，于 $-4^{\circ}\mathrm{C}, 10000\mathrm{r} / \mathrm{min}$ 离心 $10\mathrm{min}$ ，取上清液，过滤膜后供液相色谱-串联质谱测定。以测得特征离子峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在同样测试条件下, 试料溶液中抗球虫药物的保留时间与基质匹配标准工作液中抗球虫药物的保留时间之比, 偏差在 $\pm 2.5\%$ 以内, 且检测到的相对离子丰度, 应当与浓度相当的基质匹配标准溶液相对离子丰度一致。其允许偏差应符合表3的要求。

# 8.4.3.2 定量测定

取试料溶液和基质匹配标准工作液，作单点或多点校准，按外标法定量。基质匹配标准工作液及试料溶液中目标物的响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，标准溶液特征离子质量色谱图见附录B。

# 8.5 空白试验

取空白试料,除不加药物外,采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中抗球虫药物的残留量按标准曲线或公式（1）计算。

$$
X = \frac {A \times C _ {s} \times V _ {1} \times V _ {2}}{A _ {s} \times V _ {2} \times m}. \tag {1}
$$

式中：

$X$ ——试样中抗球虫药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——试样中抗球虫药物的峰面积；

$C_{\mathrm{s}}$ ——基质匹配标准溶液中抗球虫药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V_{1}$ ——提取液体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——分取提取液体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{3}$ ——复溶液体积的数值，单位为毫升 $(\mathrm{mL})$

$A_{s}$ ——基质匹配标准溶液中抗球虫药物的峰面积；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $5\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $10\mu \mathrm{g / kg}\sim 100\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 20\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

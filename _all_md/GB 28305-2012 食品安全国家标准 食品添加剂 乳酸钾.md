# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 盐酸溶液： $0.1\mathrm{mol} / \mathrm{L}$   
A.2.1.2 硫酸溶液： $1\mathrm{mL}$ 硫酸加 $19\mathrm{mL}$ 水配制成。  
A.2.1.3 高锰酸钾试液： $0.1\mathrm{mol} / \mathrm{L}$   
A.2.1.4 吗啡啉溶液： $1 + 4$   
A.2.1.5 亚硝基铁氰化钠试液： $50\mathrm{g / L}$

# A.2.2 分析步骤

A. 2.2.1 钾盐的鉴别: 取铂丝, 用盐酸溶液湿润后, 蘸取试样, 在无色火焰中燃烧, 火焰即显紫色; 但有少量的钠盐混存时, 须隔蓝色玻璃透视方能辨认。  
A. 2.2.2 乳酸盐的鉴别: 取 $5 \mathrm{~mL}$ 试样, 加入 $2 \mathrm{~mL}$ 硫酸溶液混匀, 再加 $2 \mathrm{~mL}$ 高锰酸钾试液, 加热, 应有乙醛气体产生。乙醛气体的识别采用等体积的吗啡啉溶液和亚硝基铁氰化钠试液的混合液湿润过的滤纸, 滤纸与气体相接触呈蓝色。

# A.3 乳酸钾含量的测定

# A.3.1 试剂和材料

A.3.1.1 乙酸酐。  
A.3.1.2 冰乙酸。  
A.3.1.3 高氯酸标准滴定溶液： $c(\mathrm{HClO}_4) = 0.1\mathrm{mol / L}$   
A.3.1.4 结晶紫指示液： $5\mathrm{g / L}$

# A.3.2 分析步骤

# A.3.2.1 电位滴定法

称取相当于含有约 $300 \mathrm{mg}$ 乳酸钾的液体试样, 精确至 $0.0001 \mathrm{~g}$ , 加入 $10 \mathrm{~mL}$ 乙酸酐和 $50 \mathrm{~mL}$ 冰乙酸, 混合均匀, 静置约 $20 \mathrm{~min}$ , 用电位滴定仪以高氯酸标准滴定溶液滴定, 同法做空白滴定。

# A.3.2.2 手工滴定法

称取相当于含有约 $300 \mathrm{mg}$ 乳酸钾的液体试样, 精确至 $0.0001 \mathrm{~g}$ , 加入 $10 \mathrm{~mL}$ 乙酸酐和 $50 \mathrm{~mL}$ 冰乙酸, 混合均匀, 静置约 $20 \mathrm{~min}$ , 加1滴结晶紫指示液, 用高氯酸标准滴定溶液滴定, 滴定至溶液显蓝绿色, $30 \mathrm{~s}$ 内不褪色, 即为滴定终点, 同时做空白试验对滴定结果进行校正。

# A.3.3 结果计算

乳酸钾含量以乳酸钾（ $\mathrm{C_3H_5KO_3}$ ）的质量分数 $w_{1}$ 计，数值以%表示，按公式（A.1）计算：

$$
w _ {1} = \frac {\left(V _ {1} - V _ {0}\right) \times c \times 0.1282}{m} \times 100 \% \dots \tag{A.1}
$$

式中：

$V_{I}$ ——试样滴定消耗高氯酸标准滴定溶液体积的数值，单位为毫升（ $\mathrm{mL}$ ）；

$V_{0}$ ——空白滴定消耗高氯酸标准滴定溶液体积的数值，单位为毫升（ $\mathrm{mL}$ ）；

$c$ ——高氯酸标准滴定溶液实际浓度的数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

0.1282——乳酸钾（ $\mathrm{C_3H_5KO_3}$ ）的摩尔质量的数值，单位为克每毫摩尔（ $\mathrm{g / mmol}$ ） $[M(\mathrm{C_3H_5KO_3}) = 128.2]$ ；

$m$ ——试样质量的数值，单位为克（g）。

实验结果以平行测定结果的算术平均值为准。平行测定结果的绝对差值不得超过 $0.5\%$

# A. 4 pH的测定

取1份试样，加入5份等体积的水，配制成待测溶液（1+5），摇匀后用pH计测定。

# A.5 氯化物的测定

# A.5.1 试剂和材料

A. 5.1.1 氯化物标准溶液: $0.1 \mathrm{mg} / \mathrm{mL}$ 。使用时, 将此标准溶液稀释 10 倍, 使氯化物 (Cl) 浓度为 $0.01 \mathrm{mg} / \mathrm{mL}$ 。

A.5.1.2 硝酸溶液： $1 + 2$

A.5.1.3 硝酸银溶液： $17\mathrm{g / L}$

# A.5.2 分析步骤

称取约 $1 \mathrm{~g}$ 试样，精确至 $0.01 \mathrm{~g}$ ，用水溶解后，转移至 $50 \mathrm{~mL}$ 容量瓶中，加水定容至刻度，摇匀。准确吸取 $5 \mathrm{~mL}$ 此试样液，置于 $50 \mathrm{~mL}$ 比色管中，加水使成约 $35 \mathrm{~mL}$ ，加 $10 \mathrm{~mL}$ 硝酸溶液，再加 $1 \mathrm{~mL}$ 硝酸银溶液，加水稀释至 $50 \mathrm{~mL}$ ，摇匀，在暗处放置 $5 \mathrm{~min}$ ，置于黑色背景上，与标准管比较，其浊度不得深于标准管，即为通过试验。

标准管的制备：准确吸取 $5\mathrm{mL}$ 稀释10倍后的氯化物标准溶液与试样管同时同样处理。

# A.6 硫酸盐的测定

# A.6.1 试剂和材料

A.6.1.1 盐酸溶液： $3\mathrm{mol} / \mathrm{L}$   
A.6.1.2 氯化钡溶液： $250\mathrm{g / L}$   
A.6.1.3 硫酸盐标准溶液： $0.1\mathrm{mg / mL}$

# A.6.2 分析步骤

称取 $2 \mathrm{~g}$ 试样，精确至 $0.01 \mathrm{~g}$ ，置于 $50 \mathrm{~mL}$ 比色管中，加适量水，加盐酸使成中性，继续加水稀释到约 $40 \mathrm{~mL}$ ，加 $2 \mathrm{~mL}$ 盐酸溶液，摇匀，加 $5 \mathrm{~mL}$ 氯化钡溶液，用水稀释至 $50 \mathrm{~mL}$ ，充分摇匀，放置 $10 \mathrm{~min}$ ，置于黑色背景上，与标准管比较，其浊度不得深于标准管，即为通过试验。

标准管的制备：准确吸取 $1\mathrm{mL}$ 硫酸盐标准溶液与试样管同时同样处理。

# A.7 氰化物的测定

注：氰化钾有剧毒，此实验需在通风橱中小心进行，防止皮肤接触、吸入溶液和蒸汽。

# A.7.1 试剂和材料

A.7.1.1 对苯二胺盐酸盐。  
A.7.1.2 吡啶。  
A.7.1.3 盐酸。  
A.7.1.4 氰化钾。  
A.7.1.5 正丁醇。

A.7.1.6 氢氧化钠溶液： $0.1\mathrm{mol} / \mathrm{L}$

A.7.1.7 氢氧化钠溶液： $20~\mathrm{g / L}$

A.7.1.8 磷酸溶液： $5\mathrm{g / L}$

A.7.1.9 亚砷酸钠溶液： $20~\mathrm{g / L}$

A. 7.1.10 溴试剂: $100 \mathrm{~g} / \mathrm{L}$ , 称取 $10 \mathrm{~g}$ 溴, 置用凡士林涂塞的玻璃瓶中, 加水振摇稀释定容至 $100 \mathrm{~mL}$ 。本液应置暗处保存。

# A.7.2 仪器和设备

分光光度计。

# A.7.3 分析步骤

# A.7.3.1 溶液A的制备

称取 $0.2 \mathrm{~g}$ 对苯二胺盐酸盐，加入 $100 \mathrm{~mL}$ 水，加热使溶解，冷却后有固体沉淀析出，保留上清液用于制备溶液B。

# A.7.3.2 溶液B的制备

量取 $128 \mathrm{~mL}$ 吡啶，加入 $365 \mathrm{~mL}$ 水，加入 $10 \mathrm{~mL}$ 盐酸，混匀。加入 $30 \mathrm{~mL}$ 溶液A制备过程中的上清液，混合均匀，放置 $24 \mathrm{~h}$ 后备用。此混合溶液贮存在棕色瓶中可保存3周。

# A.7.3.3 标准溶液的制备

称取 $0.25 \mathrm{~g}$ 氰化钾, 用 $0.1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液溶解后, 移入 $100 \mathrm{~mL}$ 容量瓶中, 加 $0.1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液稀释至刻度, 混匀。精确量取上述 $10 \mathrm{~mL}$ 液体, 移入 $1000 \mathrm{~mL}$ 容量瓶中, 用 $0.1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液稀释至刻度, 混匀。氰化物的浓度为 $0.01 \mathrm{mg} / \mathrm{mL}$ 。

# A.7.3.4 样品溶液的制备

称取相当于含有 $20.0 \mathrm{~g}$ 乳酸钾的液体试样, 置于 $100 \mathrm{~mL}$ 容量瓶中, 加水稀释至刻度, 混匀。

# A.7.3.5 测定

精确吸取 $10 \mathrm{~mL}$ 样品溶液, 置于 $50 \mathrm{~mL}$ 烧杯中。精确吸取 $0.10 \mathrm{~mL}$ 标准溶液, 置于另一个 $50 \mathrm{~mL}$ 烧杯中, 加入 $10 \mathrm{~mL}$ 水。将两个烧杯置于冰浴中, 慢慢加入 $20 \mathrm{~g} / \mathrm{L}$ 氢氧化钠溶液调节溶液 $\mathrm{pH}$ 在 $9 \sim 10$ , 缓慢搅动避免过热。静置 $3 \mathrm{~min}$ , 然后缓慢的加入磷酸溶液调节溶液 $\mathrm{pH}$ 在 $5 \sim 6$ , 调节过程中用 $\mathrm{pH}$ 计进行测量。

将上述溶液分别移入装有 $25 \mathrm{~mL}$ 冷水的 $100 \mathrm{~mL}$ 规格的分液漏斗中，用少量冷水冲洗 $\mathrm{pH}$ 计和烧杯，洗液合并到各自的分液漏斗中。分别加 $2 \mathrm{~mL}$ 溴试剂，塞紧，混匀。分别加 $2 \mathrm{~mL}$ 亚砷酸钠溶液，塞紧，混匀。分别加 $10 \mathrm{~mL}$ 正丁醇，塞紧，混匀。最后分别加 $5 \mathrm{~mL}$ 溶液B，混匀，放置 $15 \mathrm{~min}$ 。分离水相，得到醇相，过滤到 $1 \mathrm{~cm}$ 的比色皿中。用分光光度计在 $480 \mathrm{~nm}$ 处，分别测定两份溶液的吸光度。

# A.7.4 结果判定

样品溶液所对应的吸光度值不大于标准溶液所对应的吸光度值，即为通过试验。

# A. 8 甲醇和甲酯的测定

# A.8.1 试剂和材料

A.8.1.1 高锰酸钾。

A.8.1.2 磷酸。

A.8.1.3 硫酸。

A.8.1.4 草酸。

A.8.1.5 无水甲醇。

A.8.1.6 乙醇溶液： $1 + 10$

A.8.1.7 氢氧化钾溶液： $5\mathrm{mol} / \mathrm{L}$

A.8.1.8 碱性品红-亚硫酸试液。

# A.8.2 仪器和设备

分光光度计。

# A.8.3 分析步骤

# A.8.3.1 溶液A的制备

称取 $3 \mathrm{~g}$ 高锰酸钾, 溶于由 $15 \mathrm{~mL}$ 磷酸和 $70 \mathrm{~mL}$ 水组成的混合溶液中, 加水稀释至 $100 \mathrm{~mL}$ 。

# A.8.3.2 溶液B的制备

量取 $50 \mathrm{~mL}$ 硫酸，小心加入到 $50 \mathrm{~mL}$ 水中，混匀。冷却后加入 $5 \mathrm{~g}$ 草酸，待溶解后混匀。

# A.8.3.3 标准溶液的制备

称取 $1.0 \mathrm{~g}$ 无水甲醇，移入 $100 \mathrm{~mL}$ 容量瓶中，用水稀释至刻度，混匀。精确吸取 $1 \mathrm{~mL}$ 此溶液，移入 $100 \mathrm{~mL}$ 容量瓶中，用乙醇溶液稀释至刻度，混匀。

# A.8.3.4 样品溶液的制备

称取 $40.0 \mathrm{~g}$ 乳酸钾试样，置入圆底烧瓶中，加入 $10 \mathrm{~mL}$ 水。小心加入 $30 \mathrm{~mL}$ 氢氧化钾溶液，连接冷凝器，水蒸气加热进行蒸馏，收集馏分到已置入 $10 \mathrm{~mL}$ 乙醇的收集瓶中，继续收集馏分直至收集瓶中约 $80 \mathrm{~mL}$ ，停止。用水稀释馏分至 $100.0 \mathrm{~mL}$ ，得到样品溶液。

# A.8.3.5 测定

分别量取 $10.0\mathrm{mL}$ 标准溶液和样品溶液，分别置于 $25~\mathrm{mL}$ 容量瓶中，分别加入 $5.0~\mathrm{mL}$ 溶液A，混匀。 $15\mathrm{min}$ 后，分别加入 $2.0~\mathrm{mL}$ 溶液B，用玻棒搅拌至溶液无色后，加入 $5.0~\mathrm{mL}$ 碱性品红-亚硫酸试液，用水稀释至刻度，混匀。 $2\mathrm{h}$ 后，分别将上述两份溶液置于 $1\mathrm{cm}$ 比色皿中，以水做空白对照，用分光光度计在 $575~\mathrm{nm}$ 附近的最大吸收波长处，分别测定两份溶液的吸光度。

# A.8.4 结果判定

样品溶液所对应的吸光度值不大于标准溶液所对应的吸光度值，即为通过试验。

# A. 9 柠檬酸盐、草酸盐、磷酸盐或酒石酸盐的测定

# A.9.1 试剂和材料

A.9.1.1 氨水溶液： $1 + 1$

A.9.1.2 盐酸溶液： $1 + 5$

A. 9.1.3 氯化钙溶液: $75 \mathrm{~g} / \mathrm{L}$ , 称取 $7.5 \mathrm{~g}$ 氯化钙 $\left(\mathrm{CaCl}_{2} \cdot 2 \mathrm{H}_{2} \mathrm{O}\right)$ , 加水溶解, 稀释定容至 $100 \mathrm{~mL}$ 。

# A.9.2 分析步骤

吸取 $5 \mathrm{~mL}$ 试样，用刚煮沸并冷却后的水稀释至 $50 \mathrm{~mL}$ 。取 $4 \mathrm{~mL}$ 稀释后的样液，用氨水溶液或盐酸溶液调整样液 $\mathrm{pH}$ 为 $7.3 \sim 7.7$ ，加入 $1 \mathrm{~mL}$ 氯化钙溶液后在沸水浴锅中加热 $5 \mathrm{~min}$ ，溶液不得产生浑浊，即为通过试验。

# A.10 还原糖的测定

于 $10 \mathrm{~mL}$ 热的费林溶液中, 加入 5 滴试样, 应无红色沉淀产生, 即为通过试验。

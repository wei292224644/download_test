# 1 范围

本标准适用于由己酰间苯二酚还原并经重结晶制得的食品添加剂4-己基间苯二酚。

$$
\mathrm {C} _ {1 2} \mathrm {H} _ {1 8} \mathrm {O} _ {2}

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准滴定溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 溶解性试验

试样易溶于乙醚和丙酮，微溶于水。

# A.3.2 硝酸反应

# A.3.2.1 试剂和材料

硝酸。

# A.3.2.2 鉴别方法

取 $1.0\mathrm{mL}$ 试样的饱和溶液，加 $1\mathrm{mL}$ 硝酸出现亮红色。

# A.3.3 溴水反应

# A.3.3.1 试剂和材料

A.3.3.1.1 溴的饱和溶液：于 $100~\mathrm{mL}$ 水中边搅拌边加入 $2\mathrm{mL}\sim 3\mathrm{mL}$ 溴。溶液保存于具玻璃瓶塞的玻璃瓶中，玻璃塞磨口处应涂抹凡士林。

A.3.3.1.2 氨水溶液： $4 + 6$

# A.3.3.2 鉴别方法

移取 $1.0\mathrm{mL}$ 试样的饱和溶液，加入 $1\mathrm{mL}$ 溴的饱和溶液，应产生黄色絮状沉淀。再加入 $2\mathrm{mL}$ 氨水溶液，黄色沉淀应溶解，溶液应呈现黄色。

# A.3.4 红外光谱

用氯化钠(光谱纯)研磨分散试样，压片后利用红外光谱仪检测所得谱图应与附录B中图B.1一致。

# A.4 4-己基间苯二酚 $(\mathbf{C}_{12}\mathbf{H}_{18}\mathbf{O}_2)$ 含量（以干基计）的测定

# A.4.1 试剂和材料

A.4.1.1 甲醇。

A.4.1.2 盐酸。

A.4.1.3 三氯甲烷。

A.4.1.4 碘化钾溶液：称取 $16.5\mathrm{g}$ 碘化钾，溶于 $100~\mathrm{mL}$ 水中。保存于不透光的试剂瓶中，使用期为7d。

A.4.1.5 溴溶液: $c\left(\frac{1}{2} \mathrm{Br}_{2}\right)=0.1 \mathrm{~mol} / \mathrm{L}$ 。称取 $15.0 \mathrm{~g}$ 溴化钾、 $3 \mathrm{~g}$ 溴酸钾溶于水中, 稀释至 $1000 \mathrm{~mL}$ , 储存于塑料瓶中。

A.4.1.6 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_4) = 0.1\mathrm{mol / L}$

A.4.1.7 淀粉指示液。

# A.4.2 仪器和设备

玻璃干燥器：内置硅胶。

# A.4.3 分析步骤

称取 $0.1\mathrm{g}$ 已于玻璃干燥器中干燥 $4\mathrm{h}$ 的试样，精确至 $0.0002\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 碘量瓶，加入 $10~\mathrm{mL}$ 甲醇，使试样溶解，量取 $30.0~\mathrm{mL}$ 溴溶液移入碘量瓶中，快速加入 $5\mathrm{mL}$ 盐酸立即塞紧塞子，边振摇边用大量水冲洗碘量瓶的外壁，持续 $5\mathrm{min}$ ，冷却至室温后再放置 $5\mathrm{min}$ 。在碘量瓶的水封槽中加入 $6\mathrm{mL}$ 碘化钾溶液，小心松动玻璃塞，使碘化钾溶液流入瓶中立即塞紧玻璃塞，轻摇。加入 $1\mathrm{mL}$ 三氯甲烷，用硫代硫酸钠标准滴定溶液滴定，近终点时，加 $3\mathrm{mL}$ 淀粉指示液，继续滴定至溶液蓝色消失。

同时进行空白试验。空白试验除不加试样外其他操作及加入试剂的种类和量(标准滴定溶液除外)与测定试验相同。

# A.4.4 结果计算

4-己基间苯二酚 $\mathrm{(C_{12}H_{18}O_2)}$ 含量(以干基计)的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\left(V _ {0} - V _ {1}\right) \times c \times M}{m \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{0}$ ——空白试验所消耗的硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——滴定试样溶液所消耗的硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫代硫酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

M 4-己基间苯二酚的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M(\frac{1}{4}\mathrm{C}_{12}\mathrm{H}_{18}\mathrm{O}_2) = 48.57]$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A.5 酸度的测定

# A.5.1 试剂和材料

# A.5.1.1 无二氧化碳的水。

A.5.1.2 氢氧化钠标准滴定溶液: $c(\mathrm{NaOH}) = 0.02 \mathrm{~mol} / \mathrm{L}$ 。移取 $20 \mathrm{~mL}$ 氢氧化钠标准滴定溶液 $c(\mathrm{NaOH}) = 1 \mathrm{~mol} / \mathrm{L}$ , 置于聚乙烯容器中, 用无二氧化碳的水稀释至 $1000 \mathrm{~mL}$ 。

A.5.1.3 甲基红指示液。

# A.5.2 仪器和设备

微量滴定管：分度值为 $0.01\mathrm{mL}$ 或 $0.02\mathrm{mL}$

# A.5.3 分析步骤

称取 $0.250\mathrm{g}\pm 0.001\mathrm{g}$ 试样，溶于 $500~\mathrm{mL}$ 无二氧化碳的水中振摇 $5\mathrm{min}$ ，滴加3滴甲基红指示液，用氢氧化钠标准滴定溶液滴定至溶液呈红色为终点，氢氧化钠标准滴定溶液的消耗量不应大于1.0 mL。

# A.6 硫酸盐灰分的测定

# A.6.1 试剂和材料

硫酸溶液： $20\%$ 。

# A.6.2 仪器和设备

A.6.2.1 铂增蜗： $30~\mathrm{mL}$

A.6.2.2 高温炉：温度能控制为 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 。

# A.6.3 分析步骤

称取约 $2\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于已在 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 下质量恒定的铂坩埚中，加入适量的硫酸溶液使试样完全润湿，置于电热板上缓慢加热，直到试样干燥并完全炭化，冷却，加入 $0.5\mathrm{mL}$ 硫酸溶液润湿试样残渣，再置于电热板上加热至白烟冒尽，移入 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 的高温炉中灼烧 $15\mathrm{min}$ 。

# A.6.4 结果计算

硫酸盐灰分的质量分数 $\varpi_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m _ {1} - m _ {2}}{m} \times 100 \% \tag{A.2}
$$

式中：

$m_{1}$ ——硫酸盐灰分和铂坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——铂坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.05\%$ 。

# A.7 间苯二酚和其他酚类的测定

# A.7.1 试剂和材料

A.7.1.1 氯化铁溶液：称取 $9.0\mathrm{g}$ 氯化铁 $\left(\mathrm{FeCl}_3\cdot 6\mathrm{H}_2\mathrm{O}\right)$ ，溶于 $100~\mathrm{mL}$ 水中。

A.7.1.2 定性滤纸。

# A.7.2 分析步骤

称取约 $1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 烧杯中，加入 $50~\mathrm{mL}$ 水，振摇 $5\mathrm{min}$ 后，用定性滤纸过滤，滤液滴入3滴氯化铁溶液后不应呈现红色或绿色。

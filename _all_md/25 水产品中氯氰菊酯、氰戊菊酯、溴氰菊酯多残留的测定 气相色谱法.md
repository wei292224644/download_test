# 水产品中氯氰菊酯、氰戊菊酯、溴氰菊酯多残留的测定

# 气相色谱法

# 1 范围

本标准规定了水产品中氯氰菊酯、氰戊菊酯和溴氰菊酯残留量的制样和气相色谱测定方法。

本标准适用于鱼和虾可食性组织中氯氰菊酯、氰戊菊酯和溴氰菊酯残留量的检测。

# 3 原理

试料中残留的氯氰菊酯、氰戊菊酯和溴氰菊酯，用氯化钠脱水，乙腈提取，正己烷除脂， $\mathrm{C}_{18}$ 柱净化，气相色谱测定，外标法定量。

# 4 试剂和材料

以下所用的试剂，除特别注明外均为分析纯试剂；水为符合GB/T6682规定的一级水。

4.1 氯氰菊酯、氰戊菊酯、溴氰菊酯标准品：含量 $\geq 99.0\%$

4.2 乙腈：色谱纯。  
4.3 甲醇：色谱纯。  
4.4 正己烷：色谱纯。  
4.5 苯：色谱纯。  
4.6 乙酸乙酯：色谱纯。  
4.7 氯仿：色谱纯。  
4.8 氯化钠：优级纯， $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，冷却，干燥保存，备用。

4.9 甲醇水溶液：取甲醇 $100 \mathrm{~mL}$ ，用水溶解并稀释至 $300 \mathrm{~mL}$ 。  
4.10 乙腈正己烷溶液：取乙腈 $10 \mathrm{~mL}$ ，用正己烷溶解并稀释至 $110 \mathrm{~mL}$ 。  
4.11 $\mathrm{C_{18}}$ 固相萃取柱： $200\mathrm{mg / 3mL}$ ，或相当者。  
4.12 氧化铝固相萃取柱： $200\mathrm{mg} / 3\mathrm{mL}$ ，或相当者。  
4.13 $100 \mu \mathrm{g} / \mathrm{mL}$ 氯氰菊酯、氰戊菊酯和溴氰菊酯混合标准贮备液：精密称取氯氰菊酯、氰戊菊酯和溴氰菊酯标准品各 $10 \mathrm{mg}$ ，于 $100 \mathrm{~mL}$ 棕色量瓶中，用苯溶解并稀释至刻度，配制成浓度为 $100 \mu \mathrm{g} / \mathrm{mL}$ 的混合标准贮备液。 $2 \sim 8^{\circ} \mathrm{C}$ 保存，有效期3个月。  
4.14 $5 \mu \mathrm{g} / \mathrm{mL}$ 氯氰菊酯、氰戊菊酯和溴氰菊酯混合标准工作液: 精密量取 $100 \mu \mathrm{g} / \mathrm{mL}$ 氯氰菊酯、氰戊菊酯和溴氰菊酯混合标准贮备液 $500 \mu \mathrm{L}$ , 于 $10 \mathrm{~mL}$ 棕色量瓶中, 用苯稀释至刻度, 配制成浓度为 $5 \mu \mathrm{g} / \mathrm{mL}$ 的氯氰菊酯、氰戊菊酯和溴氰菊酯混合标准工作液。 $2 \sim 8^{\circ} \mathrm{C}$ 保存, 有效期3个月。

# 5 仪器和设备

5.1 气相色谱仪：配63Ni电子捕获检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 均质机  
5.5 旋转蒸发仪   
5.6 离心机  
5.7 调速混匀器  
5.8 氮吹仪   
5.9 固相萃取装置  
5.10 鸡心瓶： $125~\mathrm{mL}$   
5.11 具塞离心管： $10\mathrm{mL}$ 、 $50~\mathrm{mL}$

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或解冻的空白或供试鱼和虾可食性组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-20 ℃ 以下保存。

# 7 测定步骤

# 7.1 提取

称取试料（ $5 \pm 0.05$ ）g，于 $50 \mathrm{~mL}$ 具塞玻璃离心管中，加乙腈 $15 \mathrm{~mL}$ ，振荡 $5 \mathrm{~min}$ ，加氯化钠 $1.5 \mathrm{~g}$ 振荡 $2 \mathrm{~min}$ ， $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，取上清液于另一具塞玻璃离心管中，残渣中加乙腈 $15 \mathrm{~mL}$ ，重复提取一次，合并两次上清液，加乙腈正己烷溶液 $10 \mathrm{~mL}$ ，振荡混合 $2 \mathrm{~min}$ ， $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，除去上层正己烷液，乙腈液中加正己烷 $10 \mathrm{~mL}$ ，重复提取一次，弃上层正己烷液。取乙腈层液至鸡心瓶中，于 $45^{\circ} \mathrm{C}$ 水浴旋转蒸发至干，用甲醇 $2 \mathrm{~mL}$ 溶解残余物，加水 $4 \mathrm{~mL}$ ，混匀，备用。

# 7.2 净化和浓缩

$\mathrm{C}_{18}$ 柱依次用甲醇 $5 \mathrm{~mL}$ 、氯仿 $5 \mathrm{~mL}$ 、甲醇 $5 \mathrm{~mL}$ 和甲醇水溶液 $5 \mathrm{~mL}$ 活化，取备用液过柱，控制流速1滴/秒。用甲醇水溶液 $5 \mathrm{~mL}$ 淋洗，流干，加苯 $4 \mathrm{~mL}$ 洗脱，收集洗脱液，备用。

氧化铝柱用乙腈 $5 \mathrm{~mL}$ 活化，取备用洗脱液过柱。加苯 $1 \mathrm{~mL}$ 洗脱，重复3次。合并三次洗脱液于 $10 \mathrm{~mL}$ 刻度离心管中，用苯定容至 $5 \mathrm{~mL}$ ，供气相色谱测定。

# 7.3 标准曲线的制备

精密量取 $5 \mu \mathrm{g} / \mathrm{mL}$ 氯氰菊酯、氰戊菊酯和溴氰菊酯混合标准贮备液适量，用苯稀释，配制成浓度分别为0.5、1、5、25、50、100和 $500 \mathrm{ng} / \mathrm{mL}$ 的系列混合标准溶液，供气相色谱测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.4 测定

# 7.4.1 色谱条件

检测器： ${}^{63}\mathrm{Ni}$ 电子捕获检测器。

色谱柱：5%苯基和95%聚二甲基硅氧烷。

检测器温度： $310~\mathrm{^\circ C}$ 。

进样口温度： $230~^\circ \mathrm{C}$ 。

进样方式：不分流进样。

载气： $99.999\%$ 高纯氮气。

柱流速： $2.5\mathrm{mL / min}$ ，尾吹为： $25\mathrm{mL / min}$ 。

柱温程序：初始柱温 $160\mathrm{^\circ C}$ ，维持 $1\mathrm{min}$ ， $30~\mathrm{^\circ C / min}$ 升至 $250~\mathrm{^\circ C}$ ，维持 $2\mathrm{min}$ ， $5\mathrm{^\circ C / min}$ 升至 $280~\mathrm{^\circ C}$ ，维持 $10\mathrm{min}$ 。

进样量： $1\mu \mathrm{L}$

# 7.4.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试样溶液中氯氰菊酯、氰戊菊酯、溴氰菊酯响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和空白添加试样溶液的高效液相色谱图见附录A。

# 7.5 空白试验

除不加试料外，采用完全相同的测定步骤进行平行操作。

# 8 结果计算和表述

试料中氯氰菊酯、氰戊菊酯、溴氰菊酯的残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算

$$
X = \frac {A \times C s \times V}{A s \times m}
$$

式中：

$X$ ——供试试料中相应的氯氰菊酯、氰戊菊酯、溴氰菊酯的残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

$C s$ ——标准溶液相应的氯氰菊酯、氰戊菊酯、溴氰菊酯的浓度, $\mu g / \mathrm{mL}$ ;

$A$ ——试样中相应的氯氰菊酯、氰戊菊酯、溴氰菊酯的峰面积；

$A_{s}$ ——标准工作溶液中相应的氯氰菊酯、氰戊菊酯、溴氰菊酯的峰面积；

$V$ ——最终试样体积, $\mathrm{mL}$ ;

m——供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度、精密度

# 9.1 灵敏度

本方法的检测限为 $0.2\mu \mathrm{g / kg}$ ，定量限为 $1\mu \mathrm{g / kg}$ 。

# 9.2 准确度

本方法在 $1\sim 10~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率 $70\% \sim 100\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

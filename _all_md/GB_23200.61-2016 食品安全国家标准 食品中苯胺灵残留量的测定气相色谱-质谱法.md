# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2456—2010《进出口食品中苯胺灵残留量的测定 气相色谱一质谱法》。与SN/T2456—2010相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“进出口食品”改为“食品”；  
标准范围中增加“其他食品可参照执行”。

# 食品安全国家标准

# 食品中苯胺灵残留量的测定 气相色谱一质谱法

# 1 范围

本标准规定了食品中苯胺灵残留测定的气相色谱一质谱测定方法。

本标准适用于毛豆、芥末菜、芦柑、花生、茶叶、姜、香菇、鳗、猪肉、鸡肝、大米、大豆等食品中苯胺灵残留量的测定和确证。其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

试样中残留的苯胺灵用乙酸乙酯一正己烷(1+1，体积比)混合溶剂提取，植物源食品以石墨炭黑固相萃取小柱净化，动物源食品或含脂量高的食品以凝胶渗透色谱(GPC)净化，气相色谱一质谱仪测定及确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为农残级，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 正己烷 $(\mathrm{C_6H_{14}})$ 。  
4.1.2 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ 。  
4.1.3 氯化钠(NaCl)，分析纯。  
4.1.4 环己烷 $(\mathrm{C_6H_{12}})$ 。

# 4.2 溶液配制

4.2.1 乙酸乙酯—正己烷(1+1，体积比)混合溶剂：量取 $100\mathrm{mL}$ 乙酸乙酯和 $100\mathrm{mL}$ 正己烷，混匀。  
4.2.2 饱和氯化钠溶液：氯化钠加水配成饱和溶液。  
4.2.3 乙酸乙酯一环己烷(1+1，体积比)混合溶剂：量取 $100\mathrm{mL}$ 乙酸乙酯和 $100\mathrm{mL}$ 环己烷，混匀。

# 4.3 标准品

农药标准品：苯胺灵(Propham，CAS号：122-42-9， $\mathrm{C_{10}H_{13}NO_2}$ ）标准品纯度 $\geq 95\%$

# 4.4 标准溶液配制

4.4.1 标准储备液 $(100\mathrm{mg / L})$ ：准确称取标准品 $10.0\mathrm{mg}$ ，用正己烷溶解并定容至 $100~\mathrm{mL}$ ，该标准储备液置于 $4^{\circ}C$ 冰箱中。  
4.4.2 标准工作液：根据需要取适量标准储备液，以正己烷稀释成适当浓度的标准工作液。标准工作液要现配现用。  
4.5 材料

4.5.1 ENVI-Carb石墨炭黑固相萃取小柱： $3\mathrm{mL} / 250\mathrm{mg}$ ，或相当者。  
4.5.2 GPC柱填料： $20\mathrm{g}0.038\mathrm{mm}\sim 0.074\mathrm{mm}$ 的Bio-beadsS-X3。

# 5 仪器和设备

5.1 气相色谱仪：配质量检测器。  
5.2 凝胶渗透色谱仪。  
5.3 粉碎机。  
5.4 组织捣碎机。  
5.5 涡旋振荡器。  
5.6 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.7 具螺旋盖聚四氟乙烯塑料离心管， $50\mathrm{mL}$   
5.8 离心机： $2500\mathrm{r / min}$   
5.9 移液器： $1\mathrm{mL}\sim 5\mathrm{mL},100\mu \mathrm{L}\sim 1000\mu \mathrm{L}_{\circ}$

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按照 GB 2763 的规定执行。在取样和制样过程中,应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 水果蔬菜类

取芦柑、芥菜、毛豆、姜、花生等水果及蔬菜有代表性样品至少 $500 \mathrm{~g}$ (不可水洗), 切碎后, 用组织捣碎机将样品加工成浆状, 混匀, 均分为 2 份作为试样, 分装入洁净盛样袋内, 密闭并标识。

# 6.2.2 大米、大豆、茶叶、食用菌(干)类

取茶叶、食用菌(干)类等有代表性样品至少 ${300}\mathrm{\;g}$ ,用粉碎机粉碎并通过 ${2.0}\mathrm{\;{mm}}$ 圆孔筛,混匀,均分成 2 份作为试样,分装入洁净盛样袋内,密闭并标识。

# 6.2.3 肉制品及内脏类

取猪肉、鸡肝、鱼肉等肉及肉制品等有代表性样品至少 $500 \mathrm{~g}$ , 切碎后用组织捣碎机将样品加工成浆状, 混匀, 均分成 2 份作为试样, 分装入洁净盛样袋内, 密闭并标识。

# 6.3 试样保存

试样在 $-18^{\circ}\mathrm{C}$ 条件下保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 水果蔬菜类

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样，置于 $50\mathrm{mL}$ 具塞塑料离心管中，加氯化钠 $3\mathrm{g}$ ，再加入 $10\mathrm{mL}$ 乙酸乙酯一正己烷混合溶剂涡旋振荡器混匀 $30\mathrm{s}$ 后，超声提取 $20\mathrm{min}, 2500\mathrm{r/min}$ 离心 $3\mathrm{min}$ ，取上层有机相于另一试管中，再分别加入 $5\mathrm{mL}$ 乙酸乙酯一正己烷混合溶剂重复提取2次，合并提取液， $40^{\circ}\mathrm{C}$ 氮吹浓缩至约 $1\mathrm{mL}$ ，按7.2.1步骤净化。

# 7.1.2 茶叶、大米、食用菌(干)类

称取 $2.5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样置于 $50\mathrm{mL}$ 具塞塑料离心管中, 加入 $2.5\mathrm{mL}$ 饱和氯化钠溶液, 加入 $5\mathrm{mL}$ 乙酸乙酯一正己烷混合溶剂, 涡旋振荡器混匀 $30\mathrm{s}$ , 超声提取 $20\mathrm{min}$ , 以 $2500\mathrm{r/min}$ 离心

3 min, 取上层有机相于另一试管中。再分别加入乙酸乙酯一正己烷混合溶剂重复提取 2 次, 合并提取液, $40^{\circ} \mathrm{C}$ 氮吹浓缩至约 $1 \mathrm{~mL}$ , 按 7.2.2 步骤净化。

# 7.1.3 大豆、肉制品及内脏类

对于鸡肝、猪肉、鱼肉等样品称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样置于 $50\mathrm{mL}$ 具塞塑料离心管中，加氯化钠 $3\mathrm{g}$ ，再加入 $10\mathrm{mL}$ 混合溶剂涡旋振荡器混匀 $30\mathrm{s}$ ，后超声提取 $20\mathrm{min}$ ， $2500\mathrm{r/min}$ 离心 $3\mathrm{min}$ ，取上层有机相于另一试管中，再分别加入 $5\mathrm{mL}$ 乙酸乙酯一正己烷混合溶剂重复提取2次，合并提取液， $40^{\circ}\mathrm{C}$ 氮吹浓缩后定容至 $10.0\mathrm{mL}$ ，按7.2.3步骤净化。

# 7.2 净化

# 7.2.1 水果蔬菜类

按7.1.1步骤提取的样品，将ENVI-Carb小柱装在固相萃取的真空抽滤装置上，先用 $3\mathrm{mL}$ 乙酸乙酯一正己烷混合溶剂预淋洗小柱，保持流速约为 $1\mathrm{mL / min}$ 。将提取液过ENVI-Carb小柱，再用6 $\mathrm{mL}$ 乙酸乙酯一正己烷混合溶剂洗脱，收集全部洗脱液，置于 $40^{\circ}\mathrm{C}$ 下氮吹至近干，用乙酸乙酯定容 $2.0\mathrm{mL}$ ，供GC-MS分析。

# 7.2.2 茶叶、大米、食用菌(干)类

按7.1.2步骤提取的样品，将ENVI-Carb小柱装在固相萃取的真空抽滤装置上，先用 $3\mathrm{mL}$ 乙酸乙酯一正己烷混合溶剂预淋洗小柱，保持流速约为 $1\mathrm{mL} / \mathrm{min}$ 。将提取液过ENVI-Carb小柱，再用 $6\mathrm{mL}$ 乙酸乙酯一正己烷混合溶剂洗脱，收集全部洗脱液，置于 $40^{\circ}\mathrm{C}$ 下氮吹至近干，用乙酸乙酯定容 $1.0\mathrm{mL}$ ，供GC-MS分析。

# 7.2.3 大豆、肉制品及内脏类

按7.1.3步骤提取的样品，取 $5\mathrm{mL}$ 提取液过GPC柱，用乙酸乙酯一环己烷 $(1 + 1$ ，体积比)混合溶剂洗脱，保持流速 $4.7~\mathrm{mL / min}$ ，收集第 $30~\mathrm{mL}\sim$ 第 $43~\mathrm{mL}$ 的洗脱液，浓缩至近干，用乙酸乙酯定量至 $1.0\mathrm{mL}$ ，供GC-MS分析。

# 7.3 测定

# 7.3.1 气相色谱一质谱参考条件

a）色谱柱：DB-5MS石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度：初始温度 $80^{\circ}\mathrm{C}$ ，以 $10^{\circ}\mathrm{C / min}$ 上升至 $130^{\circ}\mathrm{C}$ ，以 $0.5^{\circ}\mathrm{C / min}$ 上升至 $136^{\circ}\mathrm{C}$ ，以 $40^{\circ}\mathrm{C / min}$ 上升至 $280^{\circ}\mathrm{C}$ 保持 $2\mathrm{min}$   
c）进样口温度： $260^{\circ}\mathrm{C}$   
d) 离子源: $230^{\circ} \mathrm{C}$ ;   
e）选择监测离子 $(m / z)$ ：定量离子120；定性离子93，137，179；  
f) 载气：氦气，纯度 $\geqslant 99.999\%$ ，流速 $1\mathrm{mL} / \mathrm{min}$   
g) 进样量: $1 \mu \mathrm{L}$ ;   
h）进样方式：不分流进样；  
i）四极杆温度： $250^{\circ}\mathrm{C}$   
j） 连接口温度： $280^{\circ}C$

# 7.3.2 色谱测定与确证

根据试样中被测物的含量情况，选取响应值适宜的标准工作液进行色谱分析，对标准工作液和样液等体积参插进样。标准工作液和待测样液中苯胺灵的响应值均应在仪器线性响应范围内。

如果样液与标准工作溶液的选择离子色谱图中，在相同保留时间处有色谱峰出现，并且在扣除背景后的样品质谱图中， $m / z 120, 93, 137, 179$ 均出现，所选离子的丰度比与标准品对应离子的丰度比，其值在允许范围内（允许范围见表1)则可以判定样品中存在苯胺灵残留。在上述的仪器条件下，苯胺灵的参考保留时间为 $14.2\mathrm{min}$ （参见图B.1)，离子丰度比 $(m / z)93:120:137:179 = 100:34:42:54$ 。

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中苯胺灵的含量。

$$
X = \frac {A \times C _ {s} \times V \times f}{A _ {s} \times m}. \tag {1}
$$

式中：

$X$ ——试样中农药残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A$ ——样液中农药的峰面积；  
$C_s$ ——标准工作液中农药的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；  
$V$ ——样品最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释或浓缩倍数；

$A_{s}$ ——标准工作液中农药的峰面积；

$m$ ——称样量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 D 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 E 的要求。

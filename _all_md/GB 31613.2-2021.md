# 液相色谱-串联质谱法

1 范围

本文件规定了猪、鸡可食性组织中泰万菌素和3-乙酰泰乐菌素残留检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、鸡皮脂、肌肉、肝脏、肾脏、鸡蛋中泰万菌素和3-乙酰泰乐菌素残留量的测定。

2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

3 术语和定义

本文件没有需要界定的术语和定义。

4 原理

试样中残留的泰万菌素和3-乙酰泰乐菌素，用 $50\%$ 乙腈提取，正己烷和酸性氧化铝净化，液相色谱串联质谱测定，用基质匹配标准溶液外标法定量。

5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 甲酸（HCOOH）：色谱纯。  
5.1.3 正已烷 $(\mathrm{C_6H_{14}})$   
5.1.4 无水硫酸镁 $(\mathrm{MgSO_4})$   
5.1.5 氯化钠 $(\mathrm{NaCl})$   
5.2 溶液配制  
5.2.1 $0.02\mathrm{mol} / \mathrm{L}$ 酒石酸水溶液：取酒石酸 $0.3\mathrm{g}$ ，加水 $100\mathrm{mL}$ 使溶解。  
5.2.2 $0.1\%$ 甲酸水溶液：取甲酸 $0.5 \mathrm{~mL}$ ，溶于 $500 \mathrm{~mL}$ 水中。  
5.2.3 $50\%$ 乙腈酒石酸溶液：取乙腈 $50~\mathrm{mL}$ ，用 $0.02\mathrm{mol} / \mathrm{L}$ 酒石酸水溶液稀释至 $100~\mathrm{mL}$ ，混匀，即得。  
5.3 标准品  
5.3.1 泰万菌素（Tylvalosin， $\mathrm{C_{53}H_{87}NO_{19}}$ ，CAS号：63409-12-1），含量 $\geq 93.0\%$   
5.3.2 3-乙酰泰乐菌素(3-acetyltylosin， $\mathrm{C_{48}H_{79}NO_{18}}$ ，CAS号：63409-10-9)，含量 $\geqslant 95.0\%$   
5.4 标准溶液制备  
5.4.1 泰万菌素标准储备液 $(1\mathrm{mg / mL})$ ：取泰万菌素对照品 $10\mathrm{mg}$ ，精密称定，用乙腈溶解并定容至 $10~\mathrm{mL}$ 棕色容量瓶，摇匀，即得。 $-18^{\circ}C$ 以下保存，有效期6个月。

5.4.23-乙酰泰乐菌素标准储备液 $(1\mathrm{mg / mL})$ ：取3-乙酰泰乐菌素对照品 $10\mathrm{mg}$ ，精密称定，置 $10~\mathrm{mL}$ 棕色容量瓶，用乙腈溶解并稀释至刻度，摇匀即得。 $-18^{\circ}C$ 以下保存，有效期6个月。  
5.4.3 混合标准工作液 $(10\mu \mathrm{g / mL})$ ：准确吸取泰万菌素、3-乙酰泰乐菌素标准储备液各 $1\mathrm{mL}$ 置 $100~\mathrm{mL}$ 棕色容量瓶，用乙腈稀释至刻度，摇匀，即得。 $-18^{\circ}C$ 以下保存，有效期3个月。  
5.5 材料  
5.5.1 酸性氧化铝：100日～200日。  
5.5.2 微孔滤膜：尼龙， $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和感量 $0.00001\mathrm{g}$   
6.3 离心管： $50~\mathrm{mL},10~\mathrm{mL}$   
6.4 涡旋混合器。  
6.5 水平振荡器。  
6.6 高速离心机：最大转速不低于 $8000\mathrm{r / min}$

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或冷藏的空白或供试样品，匀浆均质后备用。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 样品的保存

$-20^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管，加水 $5.0~\mathrm{mL}$ 、乙腈 $5.0~\mathrm{mL}$ ，涡旋混匀，加正己烷 $10~\mathrm{mL}$ ，振荡 $5\mathrm{min}$ ，再加入无水硫酸镁与氯化钠各 $2\mathrm{g}$ ，振荡 $5\mathrm{min},8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，移取中间乙腈层清液于另一离心管，备用。

# 8.2 净化

准确移取备用液 $0.5\mathrm{mL}, 0.02\mathrm{mol} / \mathrm{L}$ 酒石酸溶液 $0.5\mathrm{mL}$ ，共 $1.0\mathrm{mL}$ 。加酸性氧化铝 $0.2\mathrm{g}$ ，涡旋混匀 $30\mathrm{s}$ ， $10000\mathrm{r} / \mathrm{min}$ 离心 $5\mathrm{min}$ ，取上层，过 $0.22\mu \mathrm{m}$ 微孔滤膜，置于棕色进样瓶中，供液相色谱-串联质谱仪测定。

# 8.3 基质标准曲线的制备

取空白试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ )于 $50~\mathrm{mL}$ 离心管，按8.1与8.2中步骤操作后获得空白试样的基质溶液。精密量取标准工作液适量，用空白试样基质溶液配制成浓度为 $0.5\mathrm{ng / mL},1.0\mathrm{ng / mL},$ $2.0\mathrm{ng / mL},5.0\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL}$ 的系列标准工作液，上机测定。以特征离子质量色谱峰面积为纵坐标、标准溶液浓度为横坐标，绘制标准曲线。若需使用单点校准，则必须配制在此标准曲线进样浓度范围 $(0.5\mathrm{ng / mL}\sim 100\mathrm{ng / mL})$ 内的与样品溶液进样浓度最接近的基质匹配对照品溶液进行校准，标准曲线的线性相关系数应不低于0.99。

# 8.5 测定法

# 8.5.1 定性测定

在同样测试条件下，试样液中泰万菌素、3-乙酰泰乐菌素的保留时间与标准工作液中相应目标物的保留时间偏差在 $\pm 2.5\%$ 以内，且检测到的相对离子丰度，应当与浓度相当的校正标准溶液相对离子丰度一致。其允许偏差应符合表3的要求。

# 8.5.2 定量测定

取试样溶液和相应的标准工作液，作单点或多点校准，按外标法以色谱峰面积定量，标准工作液及试样溶液中的目标物响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，泰万菌素和3-乙酰泰乐菌素标准溶液特征离子质量色谱图见附录A。

# 8.6 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测药物的残留量按公式(1)计算。

$$
X = \frac {C _ {S} \times A \times V _ {1} \times V _ {2}}{A _ {S} \times V _ {3} \times m} \dots \tag {1}
$$

式中：

$X$ ——试料中泰万菌素或3乙酰泰乐菌素残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ 基质匹配标准溶液中泰万菌素或3乙酰泰乐菌素浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$A_{\mathrm{S}}$ 基质匹配标准溶液中泰万菌素或3乙酰泰乐菌素峰面积；

A ——试样溶液中泰万菌素或3乙酰泰乐菌素峰面积；

$V_{1}$ ——净化后进样溶液总体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——乙腈提取液总体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——移取备用液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $2.5\mu \mathrm{g / kg}$ ，定量限为 $5\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $5\mu \mathrm{g / kg}\sim 200\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 110\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leq 15\%$ ，批间相对标准偏差 $\leq 15\%$ 。

# 前言

本标准代替GB2513—2004《食品添加剂 高锰酸钾》。

本标准与GB2513—2004相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 高锰酸钾”。

# 检验方法

# A.1 安全提示

本试验方法中使用的部分试剂具有毒害性，操作者应小心谨慎。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。使用易燃品时，严禁使用明火加热。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 乙醇。

A.3.1.2 氢氧化钠溶液： $40~\mathrm{g / L}$

A.3.1.3 乙酸溶液： $1 + 7$

A.3.1.4 亚硝酸钴钠溶液： $100\mathrm{g / L}$ 。配制：称取 $0.1\mathrm{g}$ 亚硝酸钴钠，置于 $50~\mathrm{mL}$ 烧杯中，加 $10~\mathrm{mL}$ 水溶解。此溶液用前配制。

# A.3.2 鉴别方法

A.3.2.1 称 $0.05 \mathrm{~g}$ 试样, 加 $5 \mathrm{~mL}$ 水溶解, 加入 $1 \mathrm{~mL}$ 乙醇, $0.3 \mathrm{~mL}$ 氢氧化钠溶液。溶液呈绿色, 加热溶液至沸, 有棕黑色沉淀。  
A.3.2.2 将 A.3.2.1 中所得混合物过滤, 取 $1 \mathrm{~mL}$ 滤液, 然后加入 $1 \mathrm{~mL}$ 乙酸溶液和 $1 \mathrm{~mL}$ 亚硝酸钴钠溶液, 立即产生黄色沉淀。

# A.4 高锰酸钾 $\mathrm{(KMnO_4)}$ 含量的测定

# A.4.1 方法提要

采用碘量法，以淀粉为指示液，用硫代硫酸钠标准滴定溶液进行滴定，确定其含量。

# A.4.2 试剂和材料

A.4.2.1 碘化钾。  
A.4.2.2 盐酸溶液： $1 + 4$   
A.4.2.3 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$   
A.4.2.4 淀粉指示液： $5\mathrm{g / L}$

# A.4.3 分析步骤

称取约 $0.3\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 烧杯中，加水使试样溶解，将溶液转移至 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。用移液管移取 $20~\mathrm{mL}$ 该溶液，置于 $250~\mathrm{mL}$ 锥形瓶中，加 $20~\mathrm{mL}$ 水， $1\mathrm{g}$ 碘化钾和 $10~\mathrm{mL}$ 盐酸溶液用硫代硫酸钠标准滴定溶液滴定，近终点时加 $3\mathrm{mL}$ 淀粉指示液，继续滴定至蓝色消失。同时作空白试验。

# A.4.4 结果计算

高锰酸钾 $\mathrm{(KMnO_4)}$ 含量的质量分数 $w_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\frac {V - V _ {0}}{1 000} \times c \times M}{m \times \frac {20}{100}} \times 100 \% \tag{A.1}
$$

式中：

V ——滴定试验溶液时消耗的硫代钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验时消耗的硫代钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$c$ ——硫代硫酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——高锰酸钾的摩尔质量的 $\frac{1}{5}$ , 单位为克每摩尔 $(\mathrm{g/mol})$ , $[M(\frac{1}{5} \mathrm{KMnO}_4) = 31.60]$ ;

$m$ ——试样的质量，单位为克(g)；

20 从试样稀释液中移取的体积，单位为毫升 $(\mathrm{mL})$

100 ——稀释后试样溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

取平行测定结果的算术平均值为测定结果，平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 氯化物（以CI计）的测定

# A.5.1 方法提要

试样经处理后，在硝酸介质中，氯离子与银离子生成难溶的氯化银。当氯离子含量较低时，在一定时间内氯化银呈悬浮体，使溶液混浊，可用于氯化物的目视比浊法测定。

# A.5.2 试剂和材料

A.5.2.1 乙醇。

A.5.2.2 硝酸溶液： $1 + 6$

A.5.2.3 硝酸银溶液： $100\mathrm{g / L}$

A.5.2.4 氯化物标准溶液： $1\mathrm{mL}$ 含氯化物 $(\mathrm{Cl})5\mu \mathrm{g}$ ，使用前配制。用移液管移取 $5\mathrm{mL}$ 按GB/T602配制的氯化物标准溶液，置于 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。

# A.5.3 分析步骤

# A.5.3.1 试验溶液的制备

称取约 $1.25\mathrm{g}$ 试样，精确至 $0.001\mathrm{g}$ 。置于 $50~\mathrm{mL}$ 烧杯中，加 $40~\mathrm{mL}$ 水溶解，加 $3\mathrm{mL}$ 乙醇，加热至沸 $2\mathrm{min}\sim 3\mathrm{min}$ 。冷却，全部移入 $50~\mathrm{mL}$ 容量瓶中，用水稀释刻度，摇匀。用中速定量滤纸干过滤，收集滤液(弃去前 $10~\mathrm{mL}$ 滤液)，此溶液为试样溶液A(溶液应为无色)，用于氯化物、硫酸盐含量的测定。

# A.5.3.2 测定

用移液管移取 $20~\mathrm{mL}$ 试样溶液A，置于 $25~\mathrm{mL}$ 比色管中，加 $1\mathrm{mL}$ 硝酸溶液、 $1\mathrm{mL}$ 硝酸银溶液，用水稀释至刻度，摇匀。避光放置 $5\mathrm{min}$ ，所呈浊度不应深于标准比浊溶液。

标准比浊溶液的配制：用移液管移取 $10\mathrm{mL}$ 氯化物标准溶液，与试验溶液的测定同时同样处理。

# A.6 硫酸盐（以 $\mathrm{SO}_4$ 计）的测定

# A.6.1 方法提要

试样经处理后，在酸性介质中，硫酸根离子与钡离子生成难溶的硫酸钡，当硫酸根离子含量较低时，在一定时间内硫酸钡呈悬浮体，使溶液混浊，可用于硫酸盐的目视比浊法测定。

# A.6.2 试剂和材料

A.6.2.1 乙酸溶液： $2 + 3$   
A.6.2.2 氯化钡溶液： $250~\mathrm{g / L}$   
A.6.2.3 硫酸盐标准溶液： $1\mathrm{mL}$ 含硫酸盐 $(\mathrm{SO}_4)0.01\mathrm{mg}$ 。用移液管移取 $10\mathrm{mL}$ 按GB/T602配制的硫酸盐标准溶液，置于 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。  
A.6.2.4 标准比浊溶液的配制：用移液管移取 $15\mathrm{mL}$ 硫酸盐标准溶液，置于 $25\mathrm{mL}$ 比色管中，加水至约 $20\mathrm{mL}$ ，加 $0.5\mathrm{mL}$ 乙酸溶液、 $1\mathrm{mL}$ 氯化钡溶液，用水稀释至刻度，摇匀。

# A.6.3 分析步骤

用移液管移取 $12\mathrm{mL}$ 试样溶液A，用移液管移取 $15\mathrm{mL}$ 硫酸盐标准溶液，置于 $25\mathrm{mL}$ 比色管中，加水至约 $20\mathrm{mL}$ ，加 $0.5\mathrm{mL}$ 乙酸溶液、 $1\mathrm{mL}$ 氯化钡溶液，用水稀释至刻度，摇匀。放置 $5\mathrm{min}$ ，所呈浊度不应深于标准比浊溶液。

# A.7 水不溶物的测定

# A.7.1 方法提要

称取一定量的试样溶于水，经过滤后，在一定的温度条件下烘干至恒重，称量后，确定水不溶物含量。

# A.7.2 仪器和设备

A.7.2.1 玻璃砂坩埚：滤板孔径为 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$   
A.7.2.2 电烘箱：温度能控制在 $105^{\circ}\mathrm{C} \sim 110^{\circ}\mathrm{C}$ 。

# A.7.3 分析步骤

称取约 $2.5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 烧杯中， $150~\mathrm{mL}$ 水温热溶解。用已预先在 $105^{\circ}\mathrm{C} \sim 110^{\circ}\mathrm{C}$ 条件下恒重的玻璃砂坩埚过滤，用水洗涤至滤出液完全无色，置于电烘箱中，在 $105^{\circ}\mathrm{C} \sim$ $110^{\circ}\mathrm{C}$ 条件下干燥至恒重。

# A.7.4 结果计算

水不溶物的质量分数 $\varpi_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m _ {1}}{m} \times 100 \% \tag{A.2}
$$

式中：

$m_{1}$ ——干燥后不溶物的质量，单位为克 $(\mathrm{g})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

取平行测定结果的算术平均值为测定结果，平行测定的绝对差值不大于 $0.02\%$ 。

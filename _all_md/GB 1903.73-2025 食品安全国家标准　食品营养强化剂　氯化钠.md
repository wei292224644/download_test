# 附录A

# 检验方法

# A.1 安全提示

本标准的检验方法中使用的硫酸等部分试剂具有毒性或者腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

除另有规定外，本标准所用试剂的纯度应为分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品应分别按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸。

A.3.1.2 硝酸。

A.3.1.3 硝酸溶液： $c(\mathrm{HNO}_3) = 0.1\mathrm{mol / L}$ 。量取 $6.4~\mathrm{mL}$ 硝酸(A.3.1.2)，加水稀释至 $1000~\mathrm{mL}$

A.3.1.4 硝酸银溶液： $17\mathrm{g / L}$

# A.3.2 仪器和设备

电子天平：感量 $0.1\mathrm{g}$ 。

# A.3.3 钠离子鉴别

称取约 $1\mathrm{g}$ 试样，溶于 $10~\mathrm{mL}$ 水中，用铂丝蘸取盐酸(A.3.1.1)在无色火焰上燃烧至无色，再蘸取试验溶液少许，在无色火焰上燃烧，火焰应呈鲜黄色。

# A.3.4 氯离子鉴别

称取约 $1\mathrm{g}$ 试样，置于 $125~\mathrm{mL}$ 三角瓶中，加水 $50~\mathrm{mL}$ 溶解后，再加 $10~\mathrm{mL}$ 硝酸溶液(A.3.1.3)使成酸性，滴加几滴硝酸银溶液(A.3.1.4)，应生成白色凝乳状沉淀。

# A.4 酸碱度试验

# A.4.1 试剂和材料

A.4.1.1 氢氧化钠。  
A.4.1.2 盐酸。  
A.4.1.3 溴麝香草酚蓝。  
A.4.1.4 氢氧化钠溶液: $c(\mathrm{NaOH}) = 0.02 \, \mathrm{mol} / \mathrm{L}$ 。称取 $0.8 \, \mathrm{g}$ 氢氧化钠(A.4.1.1), 加水溶解并稀释至 $1000 \, \mathrm{mL}$ 。  
A.4.1.5 溴麝香草酚蓝指示液：取溴麝香草酚蓝(A.4.1.3) $0.1\mathrm{g}$ ，加氢氧化钠溶液(A.4.1.4) $8\mathrm{mL}$ 使之溶解，再加水稀释至 $200\mathrm{mL}$ 。  
A.4.1.6 盐酸溶液： $c(\mathrm{HCl}) = 0.02\mathrm{mol / L}$

# A.4.2 仪器和设备

电子天平：感量 $0.01\mathrm{g}$ 。

# A.4.3 分析步骤

称取试样 $5.0\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），加水 $50~\mathrm{mL}$ 溶解后，加溴麝香草酚蓝指示液（A.4.1.5） $0.10~\mathrm{mL}$ 。如显黄色，加氢氧化钠溶液(A.4.1.4) $0.10~\mathrm{mL}$ ，观察颜色变化；如显蓝色或绿色，加盐酸溶液(A.4.1.6) $0.20~\mathrm{mL}$ ，观察颜色变化。

# A.4.4 结果判定

试样溶液加溴麝香草酚蓝指示液(A.4.1.5)后，如显黄色，加氢氧化钠溶液(A.4.1.4)，变蓝色；如显蓝色或绿色，加盐酸溶液(A.4.1.6)，变黄色，即为试样酸碱度通过试验。

# A.5 溶液的澄清度

# A.5.1 方法提要

将试样溶液与规定的浊度标准液相比较，以检查溶液的澄清程度。

# A.5.2 试剂和材料

A.5.2.1 硫酸肼：使用前于 $105\,^{\circ}\mathrm{C}$ 下干燥至恒重。  
A.5.2.2 乌洛托品。  
A.5.2.3 乌洛托品溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 乌洛托品(A.5.2.2)，加水溶解并稀释至 $100~\mathrm{mL}$ 。  
A.5.2.4 浊度标准贮备液：称取 $1.00\mathrm{g}$ 硫酸肼(A.5.2.1)，加水溶解并稀释至 $100~\mathrm{mL}$ 。放置6h后，与 $100~\mathrm{mL}$ 乌洛托品溶液(A.5.2.3)混合摇匀，避光保存。  
A.5.2.5 浊度标准中间液：取浊度标准贮备液(A.5.2.4) $15.0~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。所得溶液在光程为 $1\mathrm{cm}$ 时，在 $550~\mathrm{nm}$ 的波长处测定，其吸光度应为 $0.12\sim 0.15$ 。溶液保存时间不超过 $48\mathrm{h}$ 。  
A.5.2.6 浊度标准液：取浊度标准中间液(A.5.2.5) $2.50~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ 。现配现用。使用前充分摇匀。

# A.5.3 仪器和设备

A.5.3.1 电子天平：感量 $0.01\mathrm{g}$   
A.5.3.2 恒温干燥箱。

# A.5.4 分析步骤

称取 $5.0\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )，加水 $25~\mathrm{mL}$ 溶解。比较试样溶液与标准液的浊度。

# A.5.5 结果判定

目视所得溶液浊度不大于浊度标准液，即为试样溶液的澄清度为澄清。

# A.6 溴化物的测定

# A.6.1 方法提要

在微酸性介质中，氯胺T将溴化物氧化为游离溴，游离溴再与酚红反应生成四溴酚红，其颜色随溴化物含量的增大呈黄绿色至紫色。吸光度与溴化物含量在一定浓度范围内成正比，将试样溶液与一定

量标准溴化物溶液用同法处理后所测得的吸光度进行比较。

# A.6.2 试剂和材料

A.6.2.1 氢氧化钠。

A.6.2.2 乙酸。

A.6.2.3 苯酚红。

A.6.2.4 硫酸胺。

A.6.2.5 氯胺 $\mathrm{T: C_7H_7ClNNaO_2S\cdot 3H_2O}$ 。

A.6.2.6 溴化钾：使用前于 $105\,^{\circ}\mathrm{C}$ 下干燥至恒重。

A.6.2.7 硫代硫酸钠。

A.6.2.8 氢氧化钠溶液: $c(\mathrm{NaOH}) = 2 \, \mathrm{mol} / \mathrm{L}$ 。称取 $80 \, \mathrm{g}$ 氢氧化钠(A.6.2.1), 溶于水, 移入 $1000 \, \mathrm{mL}$ 容量瓶中, 稀释至刻度。

A.6.2.9 乙酸溶液： $c(\mathrm{CH}_3\mathrm{CO}_2\mathrm{H}) = 2\mathrm{mol} / \mathrm{L}$ 。称取 $113.2~\mathrm{mL}$ 乙酸(A.6.2.2)，用水稀释至 $1000~\mathrm{mL}$

A.6.2.10 苯酚红溶液：称取苯酚红(A.6.2.3) $33\mathrm{mg}$ ，加氢氧化钠溶液(A.6.2.8) $1.5\mathrm{mL}$ ，加水溶解并稀释至 $100\mathrm{mL}$ 。

A.6.2.11 苯酚红混合液: 称取硫酸铵(A.6.2.4) $25 \mathrm{mg}$ , 加水 $235 \mathrm{~mL}$ , 加氢氧化钠溶液 (A.6.2.8) $105 \mathrm{~mL}$ , 加乙酸溶液 (A.6.2.9) $135 \mathrm{~mL}$ , 摇匀, 再加苯酚红溶液 (A.6.2.10) $25 \mathrm{~mL}$ , 摇匀, 必要时, 调节 $\mathrm{pH}$ 至 4.7。

A.6.2.12 氯胺T溶液 $(0.1\mathrm{g / L})$ ：称取 $0.01\mathrm{g}$ 氯胺T(A.6.2.5)，溶于水，移入 $100~\mathrm{mL}$ 容量瓶中，稀释至刻度。现配现用。

A.6.2.13 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$

A.6.2.14 溴化钾标准溶液：称取 $30\mathrm{mg}$ 溴化钾(A.6.2.6)，加水溶解并稀释至 $100~\mathrm{mL}$ 。取 $1\mathrm{mL}$ 所得溶液，稀释至 $100~\mathrm{mL}$ ，每 $1\mathrm{mL}$ 溴化钾标准溶液中含 $\mathrm{Br}^{-}2\mu \mathrm{g}$ 。

# A.6.3 仪器和设备

A.6.3.1 电子天平：感量 $0.001\mathrm{g}$

A.6.3.2 分光光度计。

# A.6.4 分析步骤

称取 $2.0\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )，置于 $50~\mathrm{mL}$ 烧杯中，加水溶解后定量转移至 $100~\mathrm{mL}$ 容量瓶中，稀释至刻度，摇匀。量取 $5\mathrm{mL}$ 所得溶液，置于 $10~\mathrm{mL}$ 比色管中，加苯酚红混合液(A.6.2.11) $2.0~\mathrm{mL}$ 和氯胺T溶液(A.6.2.12) $1.0~\mathrm{mL}$ ，立即混匀，放置 $2\mathrm{min}$ ，加硫代硫酸钠标准滴定溶液(A.6.2.13) $0.15~\mathrm{mL}$ ，用水稀释至刻度，摇匀，作为试样溶液。

另取溴化钾标准溶液(A.6.2.14) $5.0~\mathrm{mL}$ ，置于 $10~\mathrm{mL}$ 比色管中，加苯酚红混合液(A.6.2.11) $2.0~\mathrm{mL}$ 和氯胺T溶液(A.6.2.12) $1.0~\mathrm{mL}$ ，立即混匀，放置 $2\mathrm{min}$ ，加硫代硫酸钠标准滴定溶液(A.6.2.13) $0.15~\mathrm{mL}$ ，用水稀释至刻度，摇匀，作为对照溶液。分别取对照溶液与试样溶液，置于 $1\mathrm{cm}$ 比色皿中，以水为空白，用分光光度计于波长 $590~\mathrm{nm}$ 处测定吸光度。

# A.6.5 结果判定

试样溶液的吸光度不大于对照溶液的吸光度，即为试样中的溴化物小于 $0.01\%$ 。

# A.7 磷酸盐的测定

# A.7.1 方法提要

在强酸性条件下,试样溶液中的磷酸盐与钼酸铵反应,生成淡黄色的磷钼黄。磷钼黄子被氯化亚锡

还原成蓝色的磷钼蓝。蓝色深浅与磷酸盐含量成正比，用目测法比较试样溶液与标准溶液的颜色。

# A.7.2 试剂和材料

A.7.2.1 盐酸。  
A.7.2.2 硫酸。  
A.7.2.3 钼酸铵。  
A.7.2.4 氯化亚锡。  
A.7.2.5 磷酸二氢钾：使用前于 $105~\mathrm{^\circ C}$ 下干燥 $2\mathrm{h}$   
A.7.2.6 硫酸溶液：取硫酸(A.7.2.2)56 mL与水混匀并定容至100 mL。  
A.7.2.7钼酸铵硫酸溶液：称取钼酸铵(A.7.2.3) $2.5\mathrm{g}$ ，加水 $20~\mathrm{mL}$ 使溶解，加硫酸溶液（A.7.2.6） $50~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ 。  
A.7.2.8 盐酸溶液：取盐酸(A.7.2.1) $18~\mathrm{mL}$ 与水混匀并定容至 $100~\mathrm{mL}$   
A.7.2.9 酸性氯化亚锡溶液：称取氯化亚锡(A.7.2.4) $20\mathrm{g}$ ，加盐酸(A.7.2.1)使溶解成 $50~\mathrm{mL}$ ，过滤取滤液备用。滤液保存时间为3个月。  
A.7.2.10 氯化亚锡盐酸溶液：吸取酸性氯化亚锡溶液(A.7.2.9) $1\mathrm{mL}$ ，加盐酸溶液(A.7.2.8) $10\mathrm{mL}$ ，混匀，现配现用。  
A.7.2.11 磷酸盐标准溶液：准确称取 $0.716\mathrm{g}$ 磷酸二氢钾(A.7.2.5)，加水溶解并稀释至 $1000~\mathrm{mL}$ 。取 $1\mathrm{mL}$ 所得溶液，用水稀释至 $100~\mathrm{mL}$ ，每 $1\mathrm{mL}$ 磷酸盐标准溶液中含 $\mathrm{PO_4}^{3 - }5\mu \mathrm{g}$

# A.7.3 仪器和设备

电子天平：感量 $0.001\mathrm{g}$ 。

# A.7.4 分析步骤

称取 $0.4\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )，加水溶解并稀释至 $100~\mathrm{mL}$ ，置于 $100~\mathrm{mL}$ 比色管中，加钼酸铵硫酸溶液(A.7.2.7) $4\mathrm{mL}$ ，加氯化亚锡盐酸溶液(A.7.2.10) $0.1~\mathrm{mL}$ ，摇匀，放置 $10\mathrm{min}$ 。

取磷酸盐标准溶液(A.7.2.11) $2.0~\mathrm{mL}$ ，置于另一支 $100~\mathrm{mL}$ 比色管中，加钼酸铵硫酸溶液(A.7.2.7) $4~\mathrm{mL}$ ，加氯化亚锡盐酸溶液(A.7.2.10) $0.1~\mathrm{mL}$ ，摇匀，放置 $10~\mathrm{min}$ 。用目测法比较试样溶液与标准溶液的颜色。

# A.7.5 结果判定

试样溶液的颜色不深于标准溶液的颜色，即为试样中的磷酸盐小于 $0.0025\%$ 。

# A.8 硫酸盐的测定

# A.8.1 方法提要

在酸性条件下，氯化钠溶液中的硫酸根离子与氯化钡溶液生成硫酸钡沉淀，用目测法比较试样溶液与标准溶液的浊度。

# A.8.2 试剂和材料

A.8.2.1 盐酸。  
A.8.2.2 氯化钡。  
A.8.2.3 硫酸钾。  
A.8.2.4 盐酸溶液 $(V + V):1 + 2$

A.8.2.5 氯化钡溶液： $250\mathrm{g / L}$ ，称取 $25\mathrm{g}$ 氯化钡(A.8.2.2)，加水溶解并稀释至 $100~\mathrm{mL}$ 。  
A.8.2.6 硫酸钾标准溶液：称取 $0.181\mathrm{g}$ 硫酸钾(A.8.2.3)，加水溶解并稀释至 $1000~\mathrm{mL}$ ，每 $1\mathrm{mL}$ 硫酸钾标准溶液中含 $\mathrm{SO}_4^{2-} 100\mu \mathrm{g}$ 。

# A.8.3 仪器和设备

电子天平：感量 $0.001\mathrm{g}$

# A.8.4 分析步骤

称取 $5.0\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），置于烧杯中，加 $40~\mathrm{mL}$ 水溶解，转移至 $50~\mathrm{mL}$ 比色管中，加 $2\mathrm{mL}$ 盐酸溶液（A.8.2.4），摇匀，再加 $5\mathrm{mL}$ 氯化钡溶液（A.8.2.5），用水稀释至 $50~\mathrm{mL}$ ，充分摇匀，放置 $10\mathrm{min}$ 。

同时取 $1.0\mathrm{mL}$ 硫酸钾标准溶液(A.8.2.6)于另一支 $50~\mathrm{mL}$ 比色管中，加水 $40~\mathrm{mL}$ ，加 $2\mathrm{mL}$ 盐酸溶液(A.8.2.4)，摇匀，再加 $5\mathrm{mL}$ 氯化钡溶液(A.8.2.5)，用水稀释至 $50~\mathrm{mL}$ ，充分摇匀，放置 $10\mathrm{min}$ 。将两者同置黑色背景上，从比色管上方向下观察，用目测法比较试样溶液与标准溶液的浊度。

# A.8.5 结果判定

试样溶液的浑浊不深于标准溶液，即为试样中的硫酸盐小于 $0.002\%$ 。

# A.9 钾盐试验

# A.9.1 方法提要

试样中的钾盐与四苯硼钠溶液生成沉淀，用目测法比较试样溶液与标准溶液的浊度。

# A.9.2 试剂和材料

A.9.2.1 冰醋酸。  
A.9.2.2 四苯硼钠。  
A.9.2.3 硫酸钾。  
A.9.2.4 醋酸溶液：取冰醋酸(A.9.2.1) $60~\mathrm{mL}$ ，加水稀释至 $1000~\mathrm{mL}$ 。

# A.10 铁盐试验

# A.10.1 方法提要

在盐酸酸性溶液中，试样溶液中的铁盐与硫氰酸铵生成红色可溶性硫氰酸铁配位离子，用目测法比较试样溶液与标准溶液的颜色。

# A.10.2 试剂和材料

A.10.2.1 盐酸。  
A.10.2.2 硫酸。  
A.10.2.3 过硫酸铵。  
A.10.2.4 硫氰酸钠。  
A.10.2.5 十二水硫酸铁铵。  
A.10.2.6 盐酸溶液：取盐酸(A.10.2.1) $234~\mathrm{mL}$ ，加水稀释至 $1000~\mathrm{mL}$ 。  
A.10.2.7 硫氰酸钠溶液： $300\mathrm{g / L}$   
A.10.2.8 铁标准溶液：称取 $0.863\mathrm{g}$ 十二水硫酸铁铵(A.10.2.5)，加水溶解后，与 $2.5\mathrm{mL}$ 硫酸(A.10.2.2)混合，并加水稀释至 $1000~\mathrm{mL}$ 。临用前，取 $10~\mathrm{mL}$ 并用水稀释至 $100~\mathrm{mL}, 1~\mathrm{mL}$ 标准铁溶液中含 $\mathrm{Fe}^{3+}$ $10~\mu \mathrm{g}$ 。

# A.10.3 仪器和设备

电子天平：感量 $0.001\mathrm{g}$ 。

# A.10.4 分析步骤

称取试样 $5.0\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ), 溶解于 $25\mathrm{mL}$ 水中, 移置 $50\mathrm{mL}$ 纳氏比色管中, 加 $4\mathrm{mL}$ 盐酸溶液(A.10.2.6)和 $50\mathrm{mg}$ 过硫酸铵(A.10.2.3), 用水稀释至 $35\mathrm{mL}$ , 加入硫氰酸钠溶液(A.10.2.7) $3\mathrm{mL}$ , 用水稀释至 $50\mathrm{mL}$ 。

取 $1.5\mathrm{mL}$ 铁标准溶液(A.10.2.8)，与试样同样处理。用目测法比较试样溶液与标准溶液的颜色。

# A.10.5 结果判定

试样溶液颜色不深于标准溶液，即为试样中的铁盐小于 $0.0003\%$ 。

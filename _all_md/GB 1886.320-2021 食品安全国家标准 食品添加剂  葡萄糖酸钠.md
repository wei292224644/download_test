# 食品添加剂 葡萄糖酸钠

# 1 范围

本标准适用于以淀粉发酵产生的葡萄糖酸和氢氧化钠为主要原料，经化学反应浓缩、结晶干燥后制得的食品添加剂葡萄糖酸钠。

# 附录A

# 检验方法

警示：试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和防护措施。

# A.1 一般规定

本标准所用试剂和水在未注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品在未注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

A.2.1 试剂和材料

A.2.1.1 浓盐酸。

A.2.1.2 冰乙酸。

A.2.1.3 乙醇。

A.2.1.4 苯肼。

A.2.1.5 铂丝。

# A.2.2 仪器和设备

A.2.2.1 水浴锅。

A.2.2.2 电子天平：感量为 $0.01\mathrm{g}$

# A.2.3 鉴别方法

# A.2.3.1 钠离子的鉴别

# A.2.3.1.1 方法原理

根据钠离子在无色火焰上燃烧、火焰为亮黄色的现象，鉴别钠离子的存在。

# A.2.3.1.2 测定步骤

称取约 $1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $10\mathrm{mL}$ 水中，用铂丝蘸取盐酸在无色火焰上燃烧至无色，再蘸取试验溶液少许，在无色火焰上燃烧，火焰应呈亮黄色。

# A.2.3.2 葡萄糖酸的鉴别

# A.2.3.2.1 方法原理

试样在冰乙酸介质中，与苯肼共热，生成黄色葡萄糖酰苯肼结晶。

# A.2.3.2.2 测定步骤

取约 $0.5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $10\mathrm{mL}$ 试管中，加 $5\mathrm{mL}$ 水，溶解（必要时加热），加 $0.7\mathrm{mL}$ 冰乙酸和 $1\mathrm{mL}$ 苯肼，在水浴上加热 $30\mathrm{min}$ ，放至室温，用玻璃棒摩擦试管内壁，则析出黄色的结晶。

# A.3 葡萄糖酸钠含量的测定

# A.3.1 常规滴定法

# A.3.1.1 方法原理

试样以冰乙酸为溶剂，以结晶紫为指示剂，用高氯酸标准滴定溶液滴定，根据消耗高氯酸标准滴定溶液的体积计算葡萄糖酸钠的含量。

# A.3.1.2 试剂和材料

A.3.1.2.1 冰乙酸。  
A.3.1.2.2 结晶紫指示液： $2\mathrm{g / L}$   
A.3.1.2.3 高氯酸标准滴定溶液： $c(\mathrm{HClO}_4) = 0.1\mathrm{mol / L}$

# A.3.1.3 仪器和设备

电子天平：感量为 $0.0001\mathrm{g}$ 。

# A.3.1.4 分析步骤

称取测定干燥减量后的试样约 $0.4\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 干燥的锥形瓶中，加 $50~\mathrm{mL}$ 冰乙酸（必要时可用电热板稍微加热），加2滴～3滴结晶紫指示液，用高氯酸标准滴定溶液滴定至溶液由紫色经蓝色最后变为绿色即为终点。除不加试样外，使用相同数量的试剂溶液做空白试验。使用时，高氯酸标准滴定液的温度应与标定时的温度相同；若其温度差小于 $4^{\circ}\mathrm{C}$ 时，应将高氯酸标准滴定溶液的浓度修正到使用温度下的浓度；若其温度差大于 $4^{\circ}\mathrm{C}$ 时，应重新标定。

# A.3.1.5 结果计算

葡萄糖酸钠（以 $\mathrm{C_6H_{11}NaO_7}$ 计)的质量分数 $\varpi_{1}$ 按式(A.1)计算。

$$
w _ {1} = \frac {\left(V _ {1} - V _ {0}\right) \times c _ {1} \times M _ {1}}{m _ {1} \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——试样溶液消耗高氯酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白溶液消耗高氯酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——温度校正后高氯酸标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{1}$ ——葡萄糖酸钠的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})(M_{1} = 218.14)$ ；

$m_{1}$ —称取测定干燥减量后试样的质量，单位为克 $(\mathrm{g})$

1000——换算系数。

计算结果保留三位有效数字。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.3\%$ 。

# A.3.2 电位滴定法

# A.3.2.1 方法原理

试样以冰乙酸为溶剂，采用电位滴定仪用高氯酸标准滴定溶液滴定，在滴定过程中通过测量电位变化以确定滴定终点，并根据消耗高氯酸标准滴定溶液的体积计算葡萄糖酸钠的含量。

# A.3.2.2 试剂和材料

A.3.2.2.1 冰乙酸。

A.3.2.2.2 高氯酸标准滴定溶液： $c(\mathrm{HClO}_4) = 0.1\mathrm{mol / L}$

# A.3.2.3 仪器和设备

A.3.2.3.1 电子天平：感量为 $0.0001\mathrm{g}$ 。  
A.3.2.3.2 电位滴定仪。

# A.3.2.4 分析步骤

称取测定干燥减量后的试样约 $0.4\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 干燥的锥形瓶中，加 $50~\mathrm{mL}$ 冰乙酸（必要时可用电热板稍微加热），采用电位滴定仪用高氯酸标准滴定溶液滴定。除不加试样外，使用相同数量的试剂溶液做空白试验。使用时，高氯酸标准滴定液的温度应与标定时的温度相同；若其温度差小于 $4^{\circ}\mathrm{C}$ 时，应将高氯酸标准滴定溶液的浓度修正到使用温度下的浓度；若其温度差大于 $4^{\circ}\mathrm{C}$ 时，应重新标定。

# A.3.2.5 结果计算

葡萄糖酸钠（以 $\mathrm{C_6H_{11}NaO_7}$ 计)的质量分数 $\varpi_{2}$ 按式(A.2)计算。

$$
w _ {2} = \frac {\left(V _ {2} - V _ {0}\right) \times c _ {1} \times M _ {1}}{m _ {1} \times 1000} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$V_{2}$ ——试样溶液消耗高氯酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白溶液消耗高氯酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——温度校正后高氯酸标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{1}$ ——葡萄糖酸钠的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})(M_{1} = 218.14)$ ；

$m_{1}$ ——称取测定干燥减量后试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果保留三位有效数字。取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.3\%$ 。

# A.4 氯化物（以 $\mathrm{Cl^-}$ 计）的测定

# A.4.1 方法原理

在酸性条件下，葡萄糖酸钠溶液中的氯离子与硝酸银溶液生成白色氯化银沉淀，用目测法与标准溶液比较浊度。

# A.4.2 试剂和材料

A.4.2.1 硝酸溶液：量取硝酸 $105\mathrm{mL}$ ，用水定容至 $1000\mathrm{mL}$ 。  
A.4.2.2 硝酸银溶液： $17\mathrm{g / L}$ 。精确称取硝酸银 $17.0\mathrm{g}$ ，用水溶解并定容至 $1000~\mathrm{mL}$ 。  
A.4.2.3 氯化物标准溶液： $0.1\mathrm{mg / mL}$ 。按GB/T602配制。

# A.4.3 仪器和设备

电子天平：感量为 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。

# A.4.4 分析步骤

# A.4.4.1 试样溶液制备

称取 $0.4\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 纳氏比色管中，加 $20~\mathrm{mL}$ 水溶解，作为试样溶液。

# A.4.4.2 标准溶液制备

移取 $2.00\mathrm{mL}$ 氯化钠标准溶液，置于 $50~\mathrm{mL}$ 纳氏比色管中，用 $20~\mathrm{mL}$ 水稀释，作为标准溶液。

# A.4.4.3 测定

在试样溶液和标准溶液中分别加入 $10\mathrm{mL}$ 硝酸溶液和 $1\mathrm{mL}$ 硝酸银溶液，并分别用水定容至 $50~\mathrm{mL}$ 缓慢摇匀，避光放置 $5\mathrm{min}$ 。

将两者同置黑色背景上，从比色管上方向下观察，比较所产生的浊度。试样溶液的浊度不深于标准溶液的浊度，即试样中的氯化物不大于 $0.05\%$ 。

# A.5 硫酸盐（以 $\mathbf{SO}_4^{2 - }$ 计）的测定

# A.5.1 方法原理

在酸性条件下，葡萄糖酸钠溶液中的硫酸根离子与氯化钡溶液生成硫酸钡沉淀，用目测法与标准溶液比较浊度。

# A.5.2 试剂和材料

A.5.2.1 盐酸溶液：量取盐酸 $234~\mathrm{mL}$ ，用水定容至 $1000~\mathrm{mL}$

A.5.2.2 氯化钡溶液： $250\mathrm{g / L}$ 。精确称取 $25.0\mathrm{g}$ 氯化钡，用水溶解并定容至 $100~\mathrm{mL}$

A.5.2.3 硫酸盐标准溶液： $0.1\mathrm{mg / mL}$ 。按GB/T602配制。

# A.5.3 仪器和设备

电子天平：感量为 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。

# A.5.4 分析步骤

# A.5.4.1 试样溶液制备

称取 $0.4\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 纳氏比色管中，加 $20~\mathrm{mL}$ 水溶解，作为试样溶液。

# A.5.4.2 标准溶液制备

移取 $2.00\mathrm{mL}$ 硫酸盐标准溶液，置于 $50~\mathrm{mL}$ 纳氏比色管中，用 $20~\mathrm{mL}$ 水稀释，作为标准溶液。

# A.5.4.3 测定

在试样溶液和标准溶液中分别加入 $2\mathrm{mL}$ 盐酸溶液和 $5\mathrm{mL}$ 氯化钡溶液，并分别用水定容至 $50~\mathrm{mL}$ 缓慢摇匀，放置 $10\mathrm{min}$

将两者同置黑色背景上，从比色管上方向下观察，比较所产生的浊度。试样溶液的浊度不深于标准溶液的浊度，即试样中的硫酸盐不大于 $0.05\%$ 。

# A.6 还原物质（以D-葡萄糖计）的测定

# A.6.1 方法原理

还原糖将二价铜离子还原成氧化亚铜，剩余的二价铜离子在酸性条件下与碘离子反应生成定量的碘，以硫代硫酸钠标准溶液滴定生成的碘，从而计算出试样中还原糖的含量。

# A.6.2 试剂和材料

# A.6.2.1 碱性柠檬酸铜溶液的配制

溶液A：称取 $173\mathrm{g}$ 柠檬酸钠(枸橼酸钠)和 $100\mathrm{g}$ 无水碳酸钠，加 $40^{\circ}\mathrm{C}$ 左右温水使溶解成 $700~\mathrm{mL}$ （若溶液显浑浊过滤使澄清）。

溶液B：称取 $17.3\mathrm{g}$ 五水硫酸铜，加水使溶解成 $100~\mathrm{mL}$ 。

临用前取 $100~\mathrm{mL}$ 溶液B，不断振摇，缓缓加入 $700~\mathrm{mL}$ 溶液A，冷却后，加水定容至 $1000~\mathrm{mL}$ 。

A.6.2.2 碘标准滴定溶液： $c(1 / 2\mathrm{I}_2) = 0.1\mathrm{mol} / \mathrm{L}$   
A.6.2.3 硫代硫酸钠标准滴定溶液： $c = 0.1\mathrm{mol / L}$   
A.6.2.4 淀粉指示液： $10\mathrm{g / L}$   
A.6.2.5 乙酸溶液：量取 $96~\mathrm{mL}$ 乙酸，加水稀释至 $1000~\mathrm{mL}$   
A.6.2.6 盐酸溶液： $3\mathrm{mol} / \mathrm{L}$ 。量取 $270~\mathrm{mL}$ 盐酸，加水稀释至 $1000~\mathrm{mL}$

# A.6.3 仪器和设备

A.6.3.1 电子天平：感量为 $0.0001\mathrm{g}$   
A.6.3.2 电炉。

# A.6.4 分析步骤

称取约 $1.0\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置 $250~\mathrm{mL}$ 碘量瓶中，加 $20~\mathrm{mL}$ 水（必要时加热）使溶解，冷却至室温，准确加入 $25.0~\mathrm{mL}$ 碱性柠檬酸铜溶液，瓶口用小表面皿或小烧杯盖住，准确微沸 $5\mathrm{min}$ 后，迅速静置冷却至室温，加 $25.0~\mathrm{mL}$ 乙酸溶液，摇匀，精确加入 $10.0~\mathrm{mL}$ 碘标准滴定溶液、 $10.0~\mathrm{mL}$ 盐酸溶液，再加 $3.0~\mathrm{mL}$ 淀粉指示液，摇匀后，立即用硫代硫酸钠标准溶液滴定至溶液显亮蓝色。除不加试样外，使用相同数量的试剂溶液做空白试验。

# A.6.5 结果计算

还原物质(以D-葡萄糖计)的质量分数 $\varpi_{3}$ 按式(A.3)计算。

$$
w _ {3} = \frac {\left(V _ {3} - V _ {4}\right) \times c _ {2} \times M _ {2}}{m _ {2} \times 1000} \times 100 \% \quad \dots \tag{A.3}
$$

式中：

$V_{3}$ ——空白溶液消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{4}$ ——试样溶液消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{2}$ ——硫代硫酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{2}$ ——还原糖 $(3 / 20\mathrm{C}_{6}\mathrm{H}_{12}\mathrm{O}_{6})$ 的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})(M_2 = 27.00)$

$m_{2}$ ——称取试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果保留三位有效数字。取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.2\%$ 。

# 检验方法

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试剂中所用标准溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

A.2.1 称取 $0.03\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $100\mathrm{mL}$ 水中。取此液 $3\mathrm{mL}$ ，加入 $1\mathrm{mL}$ 盐酸和 $1\mathrm{mL}$ 溴试液，在水浴中加热 $30\mathrm{min}$ ，鼓入空气吹除溴后，加入 $0.2\mathrm{mL}3,5-$ 二羟基甲苯-乙醇溶液 $(1\rightarrow 10)$ ，再加入 $3\mathrm{mL}$ 硫酸铁铵-盐酸溶液 $(1\rightarrow 1000)$ ，在水浴中加热 $20\mathrm{min}$ ，应显绿色。  
A.2.2 称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $20~\mathrm{mL}$ 水中。取此液 $5\mathrm{mL}$ ，加入 $2\mathrm{mL}$ 氧化镁溶液，应不产生沉淀。再加入硝酸 $7\mathrm{mL}$ ，煮沸 $10\mathrm{min}$ 后，加入氢氧化钠溶液 $(40~\mathrm{g / L})$ 中和，再加入钼酸铵溶液，加热时，生成黄色沉淀，再加氢氧化钠溶液 $(40~\mathrm{g / L})$ 或氨水溶液 $(2 + 3)$ 时，沉淀溶解。  
A.2.3 称取 $0.02\mathrm{g}$ 试样，加 $1000~\mathrm{mL}$ 盐酸溶液 $(1\rightarrow 1000)$ 溶解制成的溶液，在波长 $260~\mathrm{nm}\sim 264~\mathrm{nm}$ 处有最大吸收带。  
A.2.4 试样呈现钠盐反应。

# A.3 $5^{\prime}$ -尿苷酸二钠 $(\mathbf{C}_9\mathbf{H}_{11}\mathbf{N}_2\mathbf{Na}_2\mathbf{O}_9\mathbf{P})$ 含量（以干基计）的测定

# A.3.1 分析步骤

称取 $0.5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，加盐酸溶液 $(1\rightarrow 1000)$ 溶解并准确配至 $1000~\mathrm{mL}$ ，准确量取此溶液 $10~\mathrm{mL}$ ，加盐酸溶液 $(1\rightarrow 1000)$ 并准确配至成 $250~\mathrm{mL}$ ，作为检测液。测定在波长 $260~\mathrm{nm}$ 处检测液的吸光度 $A$ 。

# A.3.2 结果计算

$5^{\prime}$ -尿苷酸二钠 $\mathrm{(C_9H_{11}N_2Na_2O_9P)}$ 含量(以干基计)的质量分数 $\varpi$ ，按式(A.1)计算：

$$
w = \frac {0 . 5 \times 1 . 8 5 9 \times A}{m} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

0.5 ——换算系数；

1.859——换算系数；

A ——检测液的吸光度；

$m$ ——换算成干基后试样的质量，单位为克 $(\mathrm{g})$ 。

# A.4 澄清度的测定

# A.4.1 试剂和材料

A.4.1.1 硝酸溶液： $1 + 2$

A.4.1.2 糊精溶液： $20\mathrm{g / L}$

A.4.1.3 硝酸银溶液： $20\mathrm{g / L}$

A.4.1.4 浊度标准溶液：含氯(Cl) $0.01\mathrm{mg / mL}$ 。量取 $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$ 盐酸标准滴定溶液 $14.1~\mathrm{mL}\pm$ $0.02\mathrm{mL}$ ，置于 $50~\mathrm{mL}$ 容量瓶中，稀释至刻度。量取该溶液 $10~\mathrm{mL}\pm 0.02~\mathrm{mL}$ 于 $1000~\mathrm{mL}$ 容量瓶中，加水稀释至刻度，摇匀。

# A.4.2 分析步骤

称取约 $1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于比色管中，加水溶解并稀释至 $25\mathrm{mL}$ ，作为试验溶液；取另一只比色管，准确加入 $0.50\mathrm{mL}$ 浊度标准溶液，加水至约 $20\mathrm{mL}$ ，加 $1\mathrm{mL}$ 硝酸溶液， $0.2\mathrm{mL}$ 糊精溶液及 $1\mathrm{mL}$ 硝酸银溶液，加水至 $25\mathrm{mL}$ ，摇匀，避光放置 $15\mathrm{min}$ ，作为标准比浊溶液。

在无阳光直射情况下，轴向及侧向观察，试验溶液的浊度不得大于标准比浊溶液的浊度。

# A.5 吸光度比的测定

称取 $0.020\mathrm{g}$ 试样，加盐酸 $(1\rightarrow 1000)$ 溶解并配至 $1000~\mathrm{mL}$ ，测定此溶液在波长 $250~\mathrm{nm},260~\mathrm{nm}$ 及 $280~\mathrm{nm}$ 处的吸光度 $A_{1},A_{2}$ 及 $A_{3}$ ，计算 $A_{1} / A_{2}$ 和 $A_{3} / A_{2}$ 。 $A_{1} / A_{2}$ 为 $0.70\sim 0.78$ ， $A_{3} / A_{2}$ 为 $0.34\sim$ 0.42。

# A.6 其他核酸分解物的测定

称取 $0.10\mathrm{g}$ 试样，加水溶解并配制成 $20~\mathrm{mL}$ ，作为检测液。量取检测液 $1\mu \mathrm{L}$ ，不用对照液，以乙醇-乙二醇一甲醚-盐酸 $(1\rightarrow 10)$ 的混合液 $(2:2:1)$ 作为展开溶剂，进行薄层色谱分析。采用预先在 $60^{\circ}\mathrm{C} \sim 80^{\circ}\mathrm{C}$ 干燥 $20\mathrm{min}$ 、以薄层色谱用硅胶(掺入荧光剂)作为载体的薄层板。当展开溶剂顶端由原线上升约 $10\mathrm{cm}$ 高时停止展开，风干后在暗处、紫外线(波长约 $250\mathrm{nm}$ ) 下观察，只应看出有一个斑点。

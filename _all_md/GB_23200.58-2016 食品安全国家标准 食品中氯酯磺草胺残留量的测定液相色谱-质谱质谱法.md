# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2386—2009《进出口食品中氯酯磺草胺残留量的测定液相色谱一质谱/质谱法》。与SN/T2386—2009相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
标准名称中“进出口食品”改为“食品”；  
标准范围中增加“其他食品可参照执行”。

# 食品安全国家标准食品中氯酯磺草胺残留量的测定液相色谱一质谱/质谱法

# 1 范围

本标准规定了食品中氯酯磺草胺残留量的液相色谱一质谱/质谱法。

本标准适用于玉米、大米、大豆、栗子、核桃、菜心、番茄、辣椒、菠菜、洋葱、橙、草莓、芒果、樱桃、香菇、茶叶等植物源性产品和鸡肝、猪肝、牛奶等动物源性产品中氯酯磺草胺残留量的检测和确证，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 方法原理

试样中残留氯酯磺草胺采用 $1\%$ 乙酸乙腈提取，分散固相萃取剂净化，液相色谱—串联质谱仪检测，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN)}$   
4.1.2 乙酸铵 $\mathrm{(CH_3COONH_4)}$   
4.1.3 乙酸 $\mathrm{(CH_3COOH)}$ 。  
4.1.4 甲酸（HCOOH）。  
4.1.5 无水硫酸镁 $(\mathrm{MgSO_4})$ 。  
4.1.6 无水乙酸钠 $\mathrm{(CH_3COONa)}$

# 4.2 溶液配制

4.2.1 乙酸—乙腈溶液(1+99，体积比)：取 $10\mathrm{mL}$ 乙酸，加入 $990~\mathrm{mL}$ 乙腈，混匀。  
4.2.2 $5\mathrm{mmol / L}$ 乙酸铵溶液(含 $0.1\%$ 甲酸): 准确称取 $77.0\mathrm{mg}$ 乙酸铵于 $200\mathrm{mL}$ 容量瓶中, 先用少量水溶解后加入 $200\mu \mathrm{L}$ 甲酸, 再用水定容至刻度, 混匀, 现用现配。

# 4.3 标准品

氯酯磺草胺标准品(cloransulam-methyl)：CAS号：147150-35-4，分子式 $\mathrm{C_{15}H_{13}FN_5O_5SCl}$ ，相对分子质量429.81，纯度 $\geq 99\%$ 。

# 4.4 标准溶液配制

4.4.1 氯酯磺草胺标准储备溶液：称取适量的氯酯磺草胺标准品(精确至 $0.1\mathrm{mg}$ )，用乙腈溶解，配制

成 $1000\mu \mathrm{g / mL}$ 的标准储备溶液， $-18^{\circ}C$ 下保存。

4.4.2 氯酯磺草胺标准工作溶液：根据需要用乙腈稀释成 $1\mu \mathrm{g / mL}$ 标准工作溶液，一 $18^{\circ}C$ 下保存。  
4.4.3 基质空白溶液：将已知不同基质的阴性样品分别按照“7 分析步骤”处理后得到的溶液。  
4.4.4 基质标准工作液：根据实验需要吸取适量氯酯磺草胺标准工作溶液，用基质空白溶液稀释成适当浓度的标准工作液，现用现配。

# 4.5 材料

4.5.1 石墨化炭黑吸附剂： $120\mu \mathrm{m}\sim 400\mu \mathrm{m}$   
4.5.2 N-丙基乙二胺(PSA)吸附剂： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}$   
4.5.3 十八烷基硅烷(ODS)键合相吸附剂 $\left(\mathrm{C}_{18}\right): 40 \mu \mathrm{m} \sim 60 \mu \mathrm{m}$ 。  
4.5.4 微孔滤膜： $0.2\mu \mathrm{m}$ ，有机相。

# 5 仪器和设备

5.1 高效液相色谱—串联质谱仪：配有电喷雾离子源(ESI)。  
5.2 粉碎机。  
5.3 组织捣碎机。  
5.4 天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.5 超声波振荡器。  
5.6 涡旋振荡器。  
5.7 离心机：转速 $\geq 4000\mathrm{r / min}$   
5.8 聚四氟乙烯离心管： $50~\mathrm{mL}$ ，具塞。

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按GB2763的规定执行。在取样和制样的操作过程中，应防止样品受到污染或使氯酯磺草胺残留含量发生变化。

# 6.2 玉米、大米、大豆、栗子、核桃、茶叶

取代表性样品 $500 \mathrm{~g}$ , 用粉碎机粉碎。混匀, 均分成两份作为试样, 分装入洁净的盛样袋内, 密闭, 于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 保存。

# 6.3 菜心、番茄、辣椒、菠菜、洋葱、橙、草莓、芒果、樱桃、香菇

取代表性样品 $1000 \mathrm{~g}$ (不可用水洗涤), 切碎后, 依次用捣碎机将样品加工成浆状。混匀, 均分成两份作为试样, 分装入洁净的盛样袋内, 密闭。将试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 6.4 鸡肝、猪肝

取代表性样品 $500 \mathrm{~g}$ , 将其切碎后, 依次用捣碎机将样品加工成浆状。混匀, 均分成两份作为试样, 分装入洁净的盛样袋内, 密闭。将试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 6.5 牛奶

取代表性样品 $500\mathrm{g}$ ，混匀，均分成两份作为试样，分装入洁净的盛样瓶内，密闭。将试样于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 条件下保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 玉米、大米、大豆、栗子、核桃、茶叶

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样于 $50\mathrm{mL}$ 聚四氟乙烯离心管中, 加适量水 $(5\mathrm{mL} \sim 8\mathrm{mL}$ , 视具体样品而定) 浸泡 $0.5\mathrm{h}$ 。加入 $1\%$ 乙酸乙腈使乙腈和水总体积为 $20\mathrm{mL}$ , 以 $10000\mathrm{r / min}$ 均质 $0.5\mathrm{min}$ , 摇匀, 在 $40^{\circ}\mathrm{C}$ 以下超声提取 $30\mathrm{min}$ , 于 $8500\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取上清液 $1\mathrm{mL}$ , 加 PSA 和 $\mathrm{C}_{18}$ 各 $50\mathrm{mg}$ , 色素干扰大的样品如茶叶等, 加入 $5\mathrm{mg}$ 石墨化炭黑, 剧烈振摇 $1\mathrm{min}, 8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ , 取上清液用 $0.2\mu \mathrm{m}$ 滤膜过滤后, 用于液相色谱一质谱/质谱仪测定。

# 7.1.2 菜心、番茄、辣椒、菠菜、洋葱、橙、草莓、芒果、樱桃、香菇、鸡肝、猪肝

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样于 $50\mathrm{mL}$ 聚四氟乙烯离心管中。加入 $1\%$ 乙酸乙腈定容至 $20\mathrm{mL}$ , 以下操作步骤同 7.1.1。

# 7.1.3 牛奶

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样于 $50\mathrm{mL}$ 聚四氟乙烯离心管中。加入 $1\%$ 乙酸乙腈定容至 $20\mathrm{mL}$ , 以下操作步骤同7.1.1。

# 7.2.3 标准曲线的绘制

进行样品分析前根据需要用空白样品提取溶液稀释混合标准中间溶液，配成适当浓度的基质混合标准工作液，用于做标准工作曲线，而且宜现用现配。

以 $5\mathrm{g}$ 空白样品用乙腈定容到 $20~\mathrm{mL}$ 的比例，通过前处理步骤(7.1)制备基质空白溶液，将标准中间溶液稀释至 $2.0\mathrm{ng / mL},5.0\mathrm{ng / mL},10.0\mathrm{ng / mL},12.0\mathrm{ng / mL},15.0\mathrm{ng / mL},20.0\mathrm{ng / mL}$ ，不过原点做标准工作曲线。

# 7.3 定量测定

本方法中液相色谱一串联质谱采用外标校准曲线法定量测定。为减少基质对定量测定的影响, 定量用标准曲线应采用基质混合标准工作溶液绘制的标准工作曲线, 并且保证所测样品中农药的响应值均在线性范围以内。根据样液中氯酯磺草胺含量的情况, 选定浓度相近的标准工作溶液。在上述仪器条件下, 氯酯磺草胺的保留时间约为 $1.17 \mathrm{~min}$ , 多反应离子监测图参见图 B.1。

# 7.4 定性测定

对标准溶液及样液均按上述规定的条件进行检查，如果样液与标准溶液在相同的保留时间有峰出现，则对其进行质谱确证，在扣除背景后的样品谱图中，所选择离子全部出现，同时所选择离子的离子丰度比与标准品相关离子的相对丰度一致，相对离子丰度偏差不超过表2规定的范围，则可判定样品中存在氯酯磺草胺。氯酯磺草胺标准品的总离子流图和质谱图参见附录B。

# 8 结果计算和表述

液相色谱—质谱/质谱测定采用标准曲线法定量，标准曲线法定量结果按式(1)计算。

$$
X = c \times \frac {V}{m} \tag {1}
$$

式中：

$X$ ——试样中氯酯磺草胺农药的残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——从标准曲线上得到的被测组分溶液浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样品提取液总体积，单位为毫升 $(\mathrm{mL})$ （本方法中为 $20.0 \mathrm{~mL}$ ）；

$m$ ——最终样品溶液所代表试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 C 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 D 的要求。

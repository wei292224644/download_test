$$
\begin{array}{c} 0 \\ \parallel \\ \mathrm {C H _ {3}} \mathrm {- C - C H _ {3}} \end{array}
$$

58.08（按2007年国际相对原子质量）

# 检验方法

# A.1 警示

试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和防护措施。

# A. 2 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682—2008中规定的三级水。试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603之规定制备；所用溶液在未指明溶剂时，均指水溶液。

# A.3 鉴别试验

# A.3.1 溶解性

取适量试样，加入水或乙醇，制成试样溶液（1+10)，样品在 $5\mathrm{min}$ 内应完全澄清透明。

# A.3.2 密度

按GB/T4472的规定进行测定。密度（ $\rho_{20}$ ）应为 $0.789\mathrm{g / cm^3}\sim 0.792\mathrm{g / cm^3}$

# A.3.3 折光率

按GB/T614的规定进行测定。折光率（20， $D$ ）应为 $1.358\sim 1.360$

# A. 4 酸（以乙酸计）的测定

# A.4.1 试剂和材料

A.4.1.1 氢氧化钠标准滴定溶液： $c$ （ $\mathrm{NaOH}) = 0.1\mathrm{mol / L}$

A.4.1.2 酚酞指示液： $\mathrm{I0~g / L}$

A.4.1.3 无二氧化碳的水。

# A.4.2 分析步骤

取 $100 \mathrm{~mL}$ 不含二氧化碳水，置于锥形瓶中，加 $4 \sim 5$ 滴酚酞指示液，用氢氧化钠标准滴定溶液滴定至溶液呈粉红色，用量筒加入 $100 \mathrm{~mL}$ 试样，再用氢氧化钠标准滴定溶液滴定至溶液呈粉红色，保持30s不褪色即为终点。

# A.4.3 结果计算

酸[以乙酸（ $\mathrm{CH}_3\mathrm{COOH}$ ）计]的质量分数 $w$ ，按式（A.1）计算：

$$
w = \frac {\left(V _ {1} / 1000\right) \times c M}{V \rho_ {t}} \times 100 \% \tag{A.1}
$$

式中：

$V_{1}$ ——氢氧化钠标准滴定溶液的体积的数值，单位为毫升（ $\mathrm{mL}$ ）；

$c$ ——氢氧化钠标准滴定溶液浓度的准确数值，单位为摩尔每升（mol/L）；

$M$ ——乙酸的摩尔质量的数值，单位为克每摩尔（ $\mathrm{g/mol}$ ）（ $M=60.1$ ）；

$\rho_{\mathrm{t}}$ ——测定温度下样品的密度的数值，单位为克每立方厘米 $(\mathrm{g} / \mathrm{cm}^{3})$ ；

1000——换算因子；

$V$ ——试样的体积的数值（ $V = 50$ ），单位为毫升（ $\mathrm{mL}$ ）。

取两次平行测定结果的算术平均值为报告结果。两次平行测定结果的绝对差值不大于这两个测定值的算术平均值的 $20\%$

# A.5 苯酚的测定

# A.5.1 试剂和材料

A. 5.1.1 亚硝酸钠硫酸溶液: $20 \mathrm{~g} / \mathrm{L}$ 。称取 $0.1 \mathrm{~g}$ 亚硝酸钠溶于 $5 \mathrm{~mL}$ 硫酸。

A.5.1.2 氢氧化钠溶液： $c$ （ $\mathrm{NaOH}) = 2\mathrm{mol} / \mathrm{L}$

# A.5.2 分析步骤

在坩埚中加入 $3 \mathrm{~mL}$ 试样，在 $60^{\circ} \mathrm{C}$ 下蒸发至干，加入3滴亚硝酸钠硫酸溶液，保持 $2 \mathrm{~min} \sim 3 \mathrm{~min}$ 。小心加入 $3 \mathrm{~mL}$ 氢氧化钠溶液。没有颜色产生即为符合指标要求。

# A. 6 易氧化物试验

# A.6.1 试剂和材料

配制高锰酸钾溶液用水：取一定量的水加入适量的高锰酸钾溶液（约 $0.3 \mathrm{~g} / \mathrm{L}$ ）使呈淡粉红色，煮沸 $30 \mathrm{~min}$ 。如淡粉红色消失，补加高锰酸钾溶液再呈淡粉红色。冷却至室温，备用。

高锰酸钾溶液（ $0.3\mathrm{~g/L}$ ）：称取 $0.3\mathrm{~g}$ 高锰酸钾，精确至 $0.001\mathrm{~g}$ ，用预先处理过的水溶解，置于 $1000\mathrm{~mL}$ 棕色容量瓶中，并稀释至刻度，摇匀。此溶液室温下避光保存，两周内有效。

# A.6.2 分析步骤

取 $30 \mathrm{~mL}$ 样品, 加入 $0.1 \mathrm{~mL}$ 高锰酸钾溶液, 振摇, 此溶液在 $20^{\circ} \mathrm{C}$ 下保持 $15 \mathrm{~min}$ 不褪色即为通过试验。

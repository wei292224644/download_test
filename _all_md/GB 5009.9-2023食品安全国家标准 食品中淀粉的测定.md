# 前言

本标准代替GB5009.9—2016《食品安全国家标准 食品中淀粉的测定》。

本标准与GB5009.9—2016相比，主要变化如下：

——修改了范围；

——增加了除去试样中可溶性糖的微糖检验方法；

——增加了含有麦芽糊精样品的洗涤步骤。

# 食品安全国家标准

# 食品中淀粉的测定

# 1 范围

本标准规定了食品中淀粉的测定方法。

本标准第一法和第二法适用于食品(肉制品除外)中淀粉的测定；第三法适用于肉制品中淀粉的测定。本标准不适用于添加经水解产生还原糖物质(麦芽糊精和可溶性糖除外)的食品中淀粉测定。

# 第一法 酶水解法

# 2 原理

试样经去除脂肪及可溶性糖后，淀粉依次经淀粉酶酶解和盐酸水解成葡萄糖，测定葡萄糖含量，并折算成样品中淀粉含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 碘 $(\mathrm{I}_2)$ 。  
3.1.2 碘化钾(KI)。  
3.1.3 $\alpha$ -淀粉酶：EC 3.2.1.1，酶活力 $\geqslant 1.6 \mathrm{U} / \mathrm{mg}$ 。  
3.1.4 无水乙醇 $(\mathrm{C}_2\mathrm{H}_5\mathrm{OH})$ 或 $95\%$ 乙醇。  
3.1.5 石油醚：沸程为 $60^{\circ}\mathrm{C} \sim 90^{\circ}\mathrm{C}$ 。  
3.1.6 乙醚 $(\mathrm{C_4H_{10}O})$ 。  
3.1.7 甲苯 $(\mathrm{C_7H_8})$ 。  
3.1.8 三氯甲烷 $\mathrm{(CHCl_3)}$   
3.1.9 盐酸（HCl）。  
3.1.10 氢氧化钠(NaOH)。  
3.1.11 硫酸铜 $(\mathrm{CuSO}_4\cdot 5\mathrm{H}_2\mathrm{O})$ 。  
3.1.12 酒石酸钾钠 $\mathrm{(C_4H_4O_6KNa\cdot 4H_2O)}$ 。  
3.1.13 亚铁氰化钾 $\left[\mathrm{K}_{4} \mathrm{Fe}(\mathrm{CN})_{6} \cdot 3 \mathrm{H}_{2} \mathrm{O}\right]$ 。  
3.1.14 亚甲蓝 $\mathrm{(C_{16}H_{18}ClN_3S\cdot 3H_2O)}$ ：指示剂。  
3.1.15 甲基红 $(\mathrm{C_{15}H_{15}N_3O_2})$ ：指示剂。  
3.1.16 $\alpha^{-}$ 萘酚（ $\mathrm{C_{10}H_8O)}$ ）。  
3.1.17 浓硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。

# 3.2 试剂配制

3.2.1 甲基红指示液 $(2\mathrm{g} / \mathrm{L})$ ：称取甲基红 $0.20\mathrm{g}$ ，用 $95\%$ 乙醇溶解并定容至 $100~\mathrm{mL}$ 。  
3.2.2 盐酸溶液 $(1 + 1)$ ：量取 $50~\mathrm{mL}$ 盐酸与 $50~\mathrm{mL}$ 水混匀。  
3.2.3 氢氧化钠溶液 $(200\mathrm{g / L})$ ：称取 $20\mathrm{g}$ 氢氧化钠，加水溶解并稀释至 $100~\mathrm{mL}$ 。  
3.2.4 碱性酒石酸铜甲液：称取 $15\mathrm{g}$ 硫酸铜及 $0.050\mathrm{g}$ 亚甲蓝，溶于水中并稀释至 $1000~\mathrm{mL}$ 。  
3.2.5 碱性酒石酸铜乙液：称取 $50\mathrm{g}$ 酒石酸钾钠、 $75\mathrm{g}$ 氢氧化钠，溶于水中，再加入 $4\mathrm{g}$ 亚铁氰化钾，完全溶解后，用水稀释至 $1000\mathrm{mL}$ ，贮存于橡胶塞玻璃瓶内。  
3.2.6 淀粉酶溶液 $(5\mathrm{g / L})$ ：称取 $\alpha$ -淀粉酶 $0.5\mathrm{g}$ ，加 $100~\mathrm{mL}$ 水溶解，临用现配；也可加入数滴甲苯或三氯甲烷防止长霉，置于 $4^{\circ}C$ 冰箱中。  
3.2.7 碘溶液：称取 $3.6\mathrm{g}$ 碘化钾溶于 $20~\mathrm{mL}$ 水中，加入 $1.3\mathrm{g}$ 碘，溶解后加水稀释至 $100~\mathrm{mL}$ 。  
3.2.8 乙醇溶液（ $85\%$ ，体积分数）：取 $85~\mathrm{mL}$ 无水乙醇，加水稀释至 $100~\mathrm{mL}$ 混匀。也可用 $95\%$ 乙醇配制。  
3.2.9 乙醇溶液（ $40\%$ ，体积分数）：取 $40~\mathrm{mL}$ 无水乙醇，加水稀释至 $100~\mathrm{mL}$ 混匀。也可用 $95\%$ 乙醇配制。  
3.2.10 $\alpha$ -萘酚乙醇溶液（ $10~\mathrm{g / L}$ ）：称取 $\alpha$ -萘酚 $1\mathrm{g}$ ，用 $95\%$ 乙醇溶解并稀释至 $100~\mathrm{mL}$ 。

# 3.3 标准品

D-无水葡萄糖 $\mathrm{(C_6H_{12}O_6}$ ,CAS号：50-99-7)：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

葡萄糖标准溶液 $(1.00\mathrm{mg / mL})$ ：准确称取 $1\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )经过 $98^{\circ}C\sim 100^{\circ}C$ 干燥 $2\textrm{h}$ 的D-无水葡萄糖，加水溶解后加入 $5\mathrm{mL}$ 盐酸，并以水定容至 $1000~\mathrm{mL}$ 。此溶液每毫升相当于 $1.00\mathrm{mg}$ 葡萄糖。临用现配。

# 4 仪器和设备

4.1 40目筛：孔径 $0.425\mathrm{mm}$   
4.2 天平：感量为 $1\mathrm{mg}$ 和 $0.1\mathrm{mg}$   
4.3 恒温水浴锅：可加热至 $100^{\circ}\mathrm{C}$ 。  
4.4 回流装置，并附 $250~\mathrm{mL}$ 锥形瓶。  
4.5 组织捣碎机。  
4.6 电炉。  
4.7 滴定管： $25~\mathrm{mL}$

# 5 分析步骤

# 5.1 试样制备

取试样可食部分磨碎，过40目筛，称取 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ；不易磨碎试样，可准确加入适量水并记录质量，匀浆后称取相当于原样质量 $2\mathrm{g}\sim 5\mathrm{g}$ 。置于放有折叠慢速滤纸的漏斗内，先用 $50~\mathrm{mL}$ 石油醚或乙醚分5次洗除脂肪(可用玻棒轻轻搅动分散样品)，再用乙醇溶液 $(85\%)$ ，体积分数)分次洗去可

溶性糖类至微糖检验结果为阴性。含有麦芽糊精的试样，先用 $100~\mathrm{mL}$ 乙醇溶液（ $85\%$ ，体积分数）洗涤，再用乙醇溶液（ $40\%$ ，体积分数）洗涤至微糖检验结果为阴性。滤干乙醇溶液，将残留物移入 $250~\mathrm{mL}$ 烧杯内，用 $50~\mathrm{mL}$ 水洗净滤纸，洗液并入烧杯内，将烧杯置于沸水浴加热至糊化完全，一般需 $15\mathrm{min}$ ，放冷至 $60^{\circ}\mathrm{C}$ 以下，加 $20~\mathrm{mL}$ 淀粉酶溶液，在 $55^{\circ}\mathrm{C} \sim 60^{\circ}\mathrm{C}$ 保温 $1\mathrm{h}$ ，并时时搅拌。然后取1滴此液加1滴碘溶液，应不显现蓝色。若显蓝色，再加热糊化并加 $20~\mathrm{mL}$ 淀粉酶溶液，继续保温，直至加碘溶液不显蓝色为止。加热至沸，冷后移入 $250~\mathrm{mL}$ 容量瓶中，用适量体积的水洗涤烧杯，并转入容量瓶中，加水至刻度，混匀，过滤，并弃去初滤液。

取 $50.00\mathrm{mL}$ 滤液，置于 $250~\mathrm{mL}$ 锥形瓶中，加 $5\mathrm{mL}$ 盐酸 $(1 + 1)$ ，装上回流冷凝器，在沸水浴中回流 $1\mathrm{h}$ ，冷后加2滴甲基红指示液，用氢氧化钠溶液 $(200~\mathrm{g / L})$ 中和至中性，溶液转入 $100~\mathrm{mL}$ 容量瓶中，洗涤锥形瓶，洗液并入 $100~\mathrm{mL}$ 容量瓶中，加水至刻度，混匀备用。

# 5.2 微糖检验方法

取洗涤液 $2\mathrm{mL}$ 在小试管中，加入 $\alpha^{-}$ 萘酚乙醇溶液 $(10\mathrm{g / L})4$ 滴，沿管壁缓缓加入浓硫酸 $1\mathrm{mL}$ 。在水与酸的界面出现紫色环，判定为阳性；在水与酸的界面出现黄绿色环，判定为阴性。

注：由于洗涤液中含有乙醇和水，加入浓硫酸时，需沿试管壁缓慢加入并保证试管口不对着人。

# 5.3 测定

# 5.3.1 标定碱性酒石酸铜溶液

吸取 $5.00\mathrm{mL}$ 碱性酒石酸铜甲液及 $5.00\mathrm{mL}$ 碱性酒石酸铜乙液，置于 $150~\mathrm{mL}$ 锥形瓶中，加水 $10~\mathrm{mL}$ ，加入玻璃珠2粒，从滴定管滴加约 $9\mathrm{mL}$ 葡萄糖标准溶液，控制在 $2\mathrm{min}$ 内加热至沸，保持溶液呈沸腾状态，以每2s1滴的速度继续滴加葡萄糖，直至溶液蓝色刚好褪去为终点，记录消耗葡萄糖标准溶液的总体积，同时做3份平行，取其平均值，计算标定碱性酒石酸铜溶液(甲、乙液各半)相当于葡萄糖的质量 $m_{1}(\mathrm{mg})$ ，结果按照式(1)计算。

注：也可以按上述方法标定 $4\mathrm{mL}\sim 20\mathrm{mL}$ 碱性酒石酸铜溶液(甲、乙液各半)来适应试样中还原糖的浓度变化。试样溶液测定使用碱性酒石酸铜溶液体积与标定碱性酒石酸铜溶液体积相同。

# 5.3.2 试样溶液预测

吸取 $5.00\mathrm{mL}$ 碱性酒石酸铜甲液及 $5.00\mathrm{mL}$ 碱性酒石酸铜乙液，置于 $150~\mathrm{mL}$ 锥形瓶中，加水 $10~\mathrm{mL}$ ，加入玻璃珠2粒，控制在 $2\mathrm{min}$ 内加热至沸，保持沸腾以先快后慢的速度，从滴定管中滴加试样溶液，并保持溶液沸腾状态，待溶液颜色变浅时，以每2s1滴的速度滴定，直至溶液蓝色刚好褪去为终点。记录试样溶液的消耗体积。当试样溶液中葡萄糖浓度过高时，应适当稀释后再进行正式测定，使每次滴定消耗试样溶液的体积控制在与标定碱性酒石酸铜溶液时所消耗的葡萄糖标准溶液的体积相近，约 $10~\mathrm{mL}$

# 5.3.3 试样溶液测定

吸取 $5.00\mathrm{mL}$ 碱性酒石酸铜甲液及 $5.00\mathrm{mL}$ 碱性酒石酸铜乙液，置于 $150~\mathrm{mL}$ 锥形瓶中，加水 $10\mathrm{mL}$ ，加入玻璃珠2粒，从滴定管滴加比预测体积少 $1\mathrm{mL}$ 的试样溶液至锥形瓶中，使在 $2\mathrm{min}$ 内加热至沸，保持沸腾状态继续以每2s1滴的速度滴定，直至蓝色刚好褪去为终点，记录试样溶液消耗体积。同法平行操作3份，得出平均消耗体积。结果按式(2)计算。

当试样溶液浓度过低， $25.00\mathrm{mL}$ 无法滴定至终点时，则采取直接加入 $10.00\mathrm{mL}$ 试样溶液，免去加水 $10\mathrm{mL}$ ，再用葡萄糖标准溶液滴定至终点，计算消耗的体积与标定时消耗的葡萄糖标准溶液体积之差相当于 $10\mathrm{mL}$ 试样溶液中所含葡萄糖的量。结果按式(3)、式(4)计算。

# 5.3.4 试剂空白测定

同时量取 $20.00 \mathrm{~mL}$ 水及与试样溶液处理时相同量的淀粉酶溶液, 按反滴定法做试剂空白试验。即用葡萄糖标准溶液滴定试剂空白溶液至终点, 记录消耗的体积与标定时消耗的葡萄糖标准溶液体积之差相当于 $10 \mathrm{~mL}$ 样液中所含葡萄糖的量。按式(5)、式(6)计算试剂空白中葡萄糖的含量。

# 6 分析结果的表述

6.1 标定碱性酒石酸铜溶液相当于葡萄糖质量。

$$
m _ {1} = \rho \times V _ {\mathrm {s}} \quad \dots \tag {1}
$$

式中：

$m_{1}$ ——标定用碱性酒石酸铜溶液(甲、乙液各半)相当于葡萄糖的质量，单位为毫克 $(\mathrm{mg})$ ；

$\rho$ ——葡萄糖标准溶液质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V_{\mathrm{s}}$ ——标定碱性酒石酸铜溶液(甲、乙液各半)时消耗的葡萄糖标准溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

6.2 试样中葡萄糖含量按式(2)计算。

$$
X _ {1} = \frac {m _ {1}}{\frac {5 0}{2 5 0} \times \frac {V _ {1}}{1 0 0}} \tag {2}
$$

式中：

$X_{1}$ ——所称试样中葡萄糖的质量，单位为毫克 $(\mathrm{mg})$ ；

$m_{1}$ ——标定用碱性酒石酸铜溶液(甲、乙液各半)相当于葡萄糖的质量，单位为毫克 $(\mathrm{mg})$ ；

50 测定用样品溶液体积，单位为毫升 $(\mathrm{mL})$

250 ——样品定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——测定时平均消耗试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

100——测定用样品的定容体积，单位为毫升 $(\mathrm{mL})$ 。

6.3 当试样中淀粉浓度过低时葡萄糖含量按式(3)、式(4)进行计算。

$$
X _ {2} = \frac {m _ {2}}{\frac {5 0}{2 5 0} \times \frac {1 0}{1 0 0}} \tag {3}
$$

$$
m _ {2} = m _ {1} \left(1 - \frac {V _ {2}}{V _ {\mathrm {s}}}\right) \tag {4}
$$

式中：

$X_{2}$ ——所称试样中葡萄糖的质量，单位为毫克 $(\mathrm{mg})$ ；

$m_{2}$ ——定碱性酒石酸铜溶液(甲、乙液各半)时消耗的葡萄糖标准溶液的体积与加入试样后消耗的葡萄糖标准溶液体积之差相当于葡萄糖的质量，单位为毫克 $(\mathrm{mg})$ ；

50 测定用样品溶液体积，单位为毫升 $(\mathrm{mL})$

250 ——样品定容体积，单位为毫升 $(\mathrm{mL})$ ；

10 直接加入的试样体积，单位为毫升 $(\mathrm{mL})$

100 测定用样品的定容体积，单位为毫升 $(\mathrm{mL})$

$m_{1}$ ——标定用碱性酒石酸铜溶液(甲、乙液各半)相当于葡萄糖的质量，单位为毫克 $(\mathrm{mg})$ ；

$V_{2}$ ——加入试样后消耗的葡萄糖标准溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{s}$ ——标定碱性酒石酸铜溶液(甲、乙液各半)时消耗的葡萄糖标准溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

6.4 试剂空白值按式(5)、式(6)计算。

$$
X _ {0} = \frac {m _ {0}}{\frac {5 0}{2 5 0} \times \frac {1 0}{1 0 0}} \tag {5}
$$

$$
m _ {0} = m _ {1} \left(1 - \frac {V _ {0}}{V _ {s}}\right) \tag {6}
$$

式中：

$X_{0}$ ——试剂空白值，单位为毫克 $(\mathrm{mg})$ ；

$m_{0}$ ——标定碱性酒石酸铜溶液(甲、乙液各半)时消耗的葡萄糖标准溶液的体积与加入空白后消耗的葡萄糖标准溶液体积之差相当于葡萄糖的质量,单位为毫克 $(\mathrm{mg})$ ；

50 测定用样品溶液体积，单位为毫升 $(\mathrm{mL})$

250 ——样品定容体积，单位为毫升 $(\mathrm{mL})$ ；

10 直接加入的试样体积，单位为毫升 $(\mathrm{mL})$

100——测定用样品的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m_{1}$ ——标定用碱性酒石酸铜溶液(甲、乙液各半)相当于葡萄糖的质量,单位为毫克 $(\mathrm{mg})$ ；

$V_{0}$ ——加入空白试样后消耗的葡萄糖标准溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{s}$ ——标定碱性酒石酸铜溶液(甲、乙液各半)时消耗的葡萄糖标准溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

6.5 试样中淀粉的含量按式(7)计算。

$$
X = \frac {\left(X _ {1} - X _ {0}\right) \times 0 . 9}{m \times 1 0 0 0} \times 1 0 0 \text {或} X = \frac {\left(X _ {2} - X _ {0}\right) \times 0 . 9}{m \times 1 0 0 0} \times 1 0 0 \quad \dots \dots \dots \dots \dots \dots \tag {7}
$$

式中：

$X$ ——试样中淀粉的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

0.9 ——葡萄糖折算成淀粉的换算系数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

结果 $< 1\mathrm{g} / 100\mathrm{g}$ ，保留2位有效数字。结果 $\geqslant 1\mathrm{g} / 100\mathrm{g}$ ，保留3位有效数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 酸水解法

# 8 原理

试样经去除脂肪及可溶性糖后，淀粉经盐酸水解成葡萄糖，测定葡萄糖含量，并折算成样品中淀粉含量。

# 9 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 9.1 试剂

9.1.1 盐酸（HCl）。  
9.1.2 氢氧化钠 $\mathrm{(NaOH)}$ 。

9.1.3 乙酸铅 $\mathrm{(PbC_4H_6O_4\cdot 3H_2O)}$

9.1.4 硫酸钠 $\mathrm{(Na_2SO_4)}$ 。

9.1.5 石油醚：沸程为 $60^{\circ}\mathrm{C} \sim 90^{\circ}\mathrm{C}$ 。

9.1.6 乙醚 $(\mathrm{C_4H_{10}O})$ 。

9.1.7 无水乙醇 $\mathrm{(C_2H_5OH)}$ 或 $95\%$ 乙醇。

9.1.8 甲基红 $(\mathrm{C_{15}H_{15}N_3O_2})$ ：指示剂。

9.1.9 精密 $\mathrm{pH}$ 试纸： $6.8\sim 7.2$

9.1.10 $\alpha$ -萘酚（ $\mathrm{C_{10}H_8O}$ ）。

# 第三法 皂化-酸水解法

# 14 原理

试样经氢氧化钾-乙醇皂化除去脂肪后，再去除可溶性糖，淀粉经盐酸水解成葡萄糖，测定葡萄糖含量，并折算成样品中淀粉含量。

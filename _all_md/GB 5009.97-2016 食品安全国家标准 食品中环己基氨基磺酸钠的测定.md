2016-08-31 发布

2017-03-01 实施

# 前言

本标准代替GB/T5009.97—2003《食品中环己基氨基磺酸钠的测定》。

本标准与GB/T5009.97—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中环己基氨基磺酸钠的测定”；  
——气相色谱法改为使用毛细管柱进行分析；  
——优化了样品处理、衍生化和测定的条件；  
——增加液相色谱法；  
——增加液相色谱-质谱/质谱法；  
——删除 GB/T 5009.97—2003《食品中环己基氨基磺酸钠的测定》中的第二法比色法和第三法薄层色谱法。

# 食品安全国家标准

# 食品中环己基氨基磺酸钠的测定

# 1 范围

本标准规定了食品中环己基氨基磺酸钠(甜蜜素)的三种测定方法——气相色谱法、液相色谱法和液相色谱-质谱/质谱法。

本标准气相色谱法适用于饮料类、蜜饯凉果、果丹类、话化类、带壳及脱壳熟制坚果与籽类、水果罐头、果酱、糕点、面包、饼干、冷冻饮品、果冻、复合调味料、腌渍的蔬菜、腐乳食品中环己基氨基磺酸钠的测定。

本标准气相色谱法不适用于白酒中该化合物的测定。

本标准液相色谱法适用于饮料类、蜜饯凉果、果丹类、话化类、带壳及脱壳熟制坚果与籽类、配制酒、水果罐头、果酱、糕点、面包、饼干、冷冻饮品、果冻、复合调味料、腌渍的蔬菜、腐乳食品中环己基氨基磺酸钠的测定。

本标准液相色谱-质谱/质谱法适用于白酒、葡萄酒、黄酒、料酒中环己基氨基磺酸钠的测定。

# 第一法 气相色谱法

# 2 原理

食品中的环己基氨基磺酸钠用水提取，在硫酸介质中环己基氨基磺酸钠与亚硝酸反应，生成环己醇亚硝酸酯，利用气相色谱氢火焰离子化检测器进行分离及分析，保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 正庚烷 $\left[\mathrm{CH}_3\left(\mathrm{CH}_2\right)_5\mathrm{CH}_3\right]$   
3.1.2 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.3 石油醚：沸程为 $30^{\circ}\mathrm{C} \sim 60^{\circ}\mathrm{C}$ 。  
3.1.4 氢氧化钠(NaOH)。  
3.1.5 硫酸 $\mathrm{(H_2SO_4)}$ 。  
3.1.6 亚铁氰化钾 $\{\mathrm{K}_4[\mathrm{Fe}(\mathrm{CN})_6]\cdot 3\mathrm{H}_2\mathrm{O}\}$   
3.1.7 硫酸锌 $(\mathrm{ZnSO}_4\cdot 7\mathrm{H}_2\mathrm{O})$ 。  
3.1.8 亚硝酸钠 $(\mathrm{NaNO}_2)$ 。

# 3.2 试剂配制

3.2.1 氢氧化钠溶液 $(40\mathrm{g} / \mathrm{L})$ ：称取 $20\mathrm{g}$ 氢氧化钠，溶于水并稀释至 $500~\mathrm{mL}$ ，混匀。

3.2.2 硫酸溶液 $(200\mathrm{g / L})$ ：量取 $54~\mathrm{mL}$ 硫酸小心缓缓加入 $400~\mathrm{mL}$ 水中，后加水至 $500~\mathrm{mL}$ ，混匀。  
3.2.3 亚铁氰化钾溶液 $(150\mathrm{g / L})$ ：称取折合 $15\mathrm{g}$ 亚铁氰化钾(3.1.6)，溶于水稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.4 硫酸锌溶液 $(300\mathrm{g / L})$ ：称取折合 $30\mathrm{g}$ 硫酸锌的试剂(3.1.7)，溶于水并稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.5 亚硝酸钠溶液 $(50\mathrm{g / L})$ ：称取 $25\mathrm{g}$ 亚硝酸钠，溶于水并稀释至 $500~\mathrm{mL}$ ，混匀。

# 3.3 标准品

环己基氨基磺酸钠标准品 $\mathrm{(C_6H_{12}NSO_3Na)}$ ：纯度 $\geq 99\%$

# 3.4 标准溶液的配制

3.4.1 环己基氨基磺酸标准储备液 $(5.00\mathrm{mg / mL})$ ：精确称取 $0.5612\mathrm{g}$ 环己基氨基磺酸钠标准品，用水溶解并定容至 $100~\mathrm{mL}$ ，混匀，此溶液 $1.00~\mathrm{mL}$ 相当于环己基氨基磺酸 $5.00\mathrm{mg}$ （环己基氨基磺酸钠与环己基氨基磺酸的换算系数为0.8909）。置于 $1^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱保存，可保存12个月。  
3.4.2 环己基氨基磺酸标准使用液 $(1.00\mathrm{mg / mL})$ ：准确移取 $20.0~\mathrm{mL}$ 环己基氨基磺酸标准储备液用水稀释并定容至 $100~\mathrm{mL}$ ，混匀。置于 $1^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱保存，可保存6个月。

# 4 仪器与设备

4.1 气相色谱仪：配有氢火焰离子化检测器(FID)。  
4.2 涡旋混合器。  
4.3 离心机：转速 $\geqslant 4000\mathrm{r / min}$   
4.4 超声波振荡器。  
4.5 样品粉碎机。  
4.6 $10~\mu \mathrm{L}$ 微量注射器。  
4.7 恒温水浴锅。  
4.8 天平：感量 $1\mathrm{mg},0.1\mathrm{mg}$

# 5 分析步骤

# 5.1 试样溶液的制备

# 5.1.1 液体试样处理

5.1.1.1 普通液体试样摇匀后称取 $25.0\mathrm{g}$ 试样（如需要可过滤），用水定容至 $50~\mathrm{mL}$ 备用。  
5.1.1.2 含二氧化碳的试样：称取 $25.0\mathrm{g}$ 试样于烧杯中， $60^{\circ}\mathrm{C}$ 水浴加热 $30\mathrm{min}$ 以除二氧化碳，放冷，用水定容至 $50~\mathrm{mL}$ 备用。  
5.1.1.3 含酒精的试样：称取 $25.0\mathrm{g}$ 试样于烧杯中，用氢氧化钠溶液(3.2.1)调至弱碱性 $\mathrm{pH}7\sim 8,60^{\circ}\mathrm{C}$ 水浴加热 $30\mathrm{min}$ 以除酒精，放冷，用水定容至 $50~\mathrm{mL}$ 备用。

# 5.1.2 固体、半固体试样处理

5.1.2.1 低脂、低蛋白样品（果酱、果冻、水果罐头、果丹类、蜜饯凉果、浓缩果汁、面包、糕点、饼干、复合调味料、带壳熟制坚果和籽类、腌渍的蔬菜等）：称取打碎、混匀的样品 $3.00\mathrm{g}\sim 5.00\mathrm{g}$ 于 $50~\mathrm{mL}$ 离心管中，加 $30~\mathrm{mL}$ 水，振摇，超声提取 $20\mathrm{min}$ ，混匀，离心 $(3000\mathrm{r / min})10\mathrm{min}$ ，过滤，用水分次洗涤残渣，收集滤液并定容至 $50~\mathrm{mL}$ ，混匀备用。  
5.1.2.2 高蛋白样品(酸乳、雪糕、冰淇淋等奶制品及豆制品、腐乳等)：冰棒、雪糕、冰淇淋等分别放置于

$250~\mathrm{mL}$ 烧杯中，待融化后搅匀称取；称取样品 $3.00\mathrm{g}\sim 5.00\mathrm{g}$ 于 $50~\mathrm{mL}$ 离心管中，加 $30~\mathrm{mL}$ 水，超声提取 $20\mathrm{min}$ ，加 $2\mathrm{mL}$ 亚铁氰化钾溶液，混匀，再加入 $2\mathrm{mL}$ 硫酸锌溶液，混匀，离心 $(3000\mathrm{r / min})10\mathrm{min}$ ，过滤，用水分次洗涤残渣，收集滤液并定容至 $50~\mathrm{mL}$ ，混匀备用。

5.1.2.3 高脂样品(奶油制品、海鱼罐头、熟肉制品等)：称取打碎、混匀的样品 $3.00\mathrm{g}\sim 5.00\mathrm{g}$ 于 $50~\mathrm{mL}$ 离心管中，加入 $25~\mathrm{mL}$ 石油醚，振摇，超声提取 $3\mathrm{min}$ ，再混匀，离心（ $1000\mathrm{r / min}$ 以上） $10\mathrm{min}$ ，弃石油醚，再用 $25~\mathrm{mL}$ 石油醚提取一次，弃石油醚， $60^{\circ}\mathrm{C}$ 水浴挥发去除石油醚，残渣加 $30~\mathrm{mL}$ 水，混匀，超声提取 $20\mathrm{min}$ ，加 $2\mathrm{mL}$ 亚铁氰化钾溶液，混匀，再加入 $2\mathrm{mL}$ 硫酸锌溶液，混匀，离心 $(3000\mathrm{r / min})10\mathrm{min}$ ，过滤，用水洗涤残渣，收集滤液并定容至 $50~\mathrm{mL}$ ，混匀备用。

# 5.1.3 衍生化

准确移取液体试样溶液(5.1.1)、固体、半固体试样溶液(5.1.2) $10.0~\mathrm{mL}$ 于 $50~\mathrm{mL}$ 带盖离心管中。离心管置试管架上冰浴中 $5\mathrm{min}$ 后，准确加入 $5.00~\mathrm{mL}$ 正庚烷，加入 $2.5~\mathrm{mL}$ 亚硝酸钠溶液， $2.5~\mathrm{mL}$ 硫酸溶液，盖紧离心管盖，摇匀，在冰浴中放置 $30\mathrm{min}$ ，其间振摇3次～5次；加入 $2.5\mathrm{g}$ 氯化钠，盖上盖后置旋涡混合器上振动 $1\mathrm{min}$ （或振摇60次～80次），低温离心（ $3000~\mathrm{r / min}$ ） $10\mathrm{min}$ 分层或低温静置 $20\mathrm{min}$ 至澄清分层后取上清液放置 $1^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱冷藏保存以备进样用。

# 5.2 标准溶液系列的制备及衍生化

准确移取 $1.00\mathrm{mg / mL}$ 环己基氨基磺酸标准溶液 $0.50~\mathrm{mL},1.00~\mathrm{mL},2.50~\mathrm{mL},5.00~\mathrm{mL},10.0~\mathrm{mL}$ 、 $25.0\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，加水定容。配成标准溶液系列浓度为： $0.01\mathrm{mg / mL},0.02\mathrm{mg / mL},0.05\mathrm{mg / mL}$ $0.10\mathrm{mg / mL},0.20\mathrm{mg / mL},0.50\mathrm{mg / mL}$ 。临用时配制以备衍生化用。

准确移取标准系列溶液 $10.0\mathrm{mL}$ 同(5.1.3)衍生化。

# 5.3 测定

# 5.3.1 色谱条件

5.3.1.1 色谱柱：弱极性石英毛细管柱（内涂 $5\%$ 苯基甲基聚硅氧烷， $30 \mathrm{~m} \times 0.53 \mathrm{~mm} \times 1.0 \mu \mathrm{m}$ ）或等效柱。  
5.3.1.2 柱温升温程序：初温 $55\%$ 保持 $3\mathrm{min}$ ， $10^{\circ}\mathrm{C / min}$ 升温至 $90^{\circ}\mathrm{C}$ 保持 $0.5\mathrm{min}$ ， $20^{\circ}\mathrm{C / min}$ 升温至 $200^{\circ}\mathrm{C}$ 保持 $3\mathrm{min}$ 。  
5.3.1.3 进样口：温度 $230^{\circ}\mathrm{C}$ ；进样量 $1\mu \mathrm{L}$ ，不分流/分流进样，分流比 $1:5$ （分流比及方式可根据色谱仪器条件调整）。  
5.3.1.4 检测器：氢火焰离子化检测器(FID)，温度 $260^{\circ}\mathrm{C}$ 。  
5.3.1.5 载气：高纯氮气，流量 $12.0\mathrm{mL / min}$ ，尾吹 $20~\mathrm{mL / min}$ 。  
5.3.1.6 氢气： $30~\mathrm{mL / min}$ ；空气 $330~\mathrm{mL / min}$ （载气、氢气、空气流量大小可根据仪器条件进行调整）。

# 5.3.2 色谱分析

分别吸取 $1\mu \mathrm{L}$ 经衍生化处理的标准系列各浓度溶液上清液(5.2)，注入气相色谱仪中，可测得不同浓度被测物的响应值峰面积，以浓度为横坐标，以环己醇亚硝酸酯和环己醇两峰面积之和为纵坐标，绘制标准曲线。

在完全相同的条件下进样 $1\mu \mathrm{L}$ 经衍生化处理的试样待测液上清液(5.1.3)，保留时间定性，测得峰面积，根据标准曲线得到样液中的组分浓度；试样上清液响应值若超出线性范围，应用正庚烷稀释后再进样分析。平行测定次数不少于两次。

# 6 分析结果的表述

试样中环己基氨基磺酸含量按式(1)计算：

$$
X _ {1} = \frac {c}{m} \times V \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——试样中环己基氨基磺酸的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$c$ ——由标准曲线计算出定容样液中环己基氨基磺酸的浓度, 单位为毫克每毫升 $(\mathrm{mg} / \mathrm{mL})$ ;

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$V$ ——试样的最后定容体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

取样量 $5\mathrm{g}$ 时，本方法检出限为 $0.010\mathrm{g / kg}$ ，定量限 $0.030\mathrm{g / kg}$ 。

# 第二法 高效液相色谱法

# 9 原理

食品中的环己基氨基磺酸钠用水提取后，在强酸性溶液中与次氯酸钠反应，生成 $N,N-$ 二氯环己胺，用正庚烷萃取后，利用高效液相色谱法检测，保留时间定性，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 正庚烷 $\left[\mathrm{CH}_{3}\left(\mathrm{CH}_{2}\right)_{5} \mathrm{CH}_{3}\right]$ ：色谱纯。  
10.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
10.1.3 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。  
10.1.4 次氯酸钠(NaClO)。  
10.1.5 碳酸氢钠 $\mathrm{(NaHCO_3)}$ 。  
10.1.6 硫酸锌 $(\mathrm{ZnSO}_4\cdot 7\mathrm{H}_2\mathrm{O})$ 。  
10.1.7 亚铁氰化钾 $\left\{\mathrm{K}_4\left[\mathrm{Fe}(\mathrm{CN})_6\right] \cdot 3\mathrm{H}_2\mathrm{O}\right\}$ 。  
10.1.8 石油醚：沸程为 $30^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$

# 10.2 试剂配制

10.2.1 硫酸溶液 $(1 + 1)$ ： $50~\mathrm{mL}$ 硫酸小心缓缓加入 $50~\mathrm{mL}$ 水中，混匀。

10.2.2 次氯酸钠溶液：用次氯酸钠(10.1.4)稀释，保存于棕色瓶中，保持有效氯含量 $50~\mathrm{g / L}$ 以上，混匀，市售产品需及时标定，临用时配制。  
10.2.3 碳酸氢钠溶液 $(50\mathrm{g / L})$ ：称取 $5\mathrm{g}$ 碳酸氢钠，用水溶解并稀释至 $100~\mathrm{mL}$ ，混匀。  
10.2.4 硫酸锌溶液 $(300\mathrm{g / L})$ ：称取折合 $30\mathrm{g}$ 硫酸锌(10.1.6)，溶于水并稀释至 $100~\mathrm{mL}$ ，混匀。  
10.2.5 亚铁氰化钾溶液 $(150\mathrm{g / L})$ ：称取折合 $15\mathrm{g}$ 亚铁氰化钾(10.1.7)，溶于水并稀释至 $100~\mathrm{mL}$ 混匀。

# 10.3 标准品

环己基氨基磺酸钠标准品 $\mathrm{(C_6H_{12}NSO_3Na)}$ ：纯度 $\geq 99\%$ 。

# 10.4 标准溶液配制

10.4.1 环己基氨基磺酸标准储备液 $(5.00\mathrm{mg / mL})$ ：按第一法3.4.1 配制。  
10.4.2 环己基氨基磺酸标准中间液 $(1.00\mathrm{mg / mL})$ ：按第一法3.4.2 配制。  
10.4.3 环己基氨基磺酸标准曲线系列工作液：分别吸取标准中间液(10.4.2) $0.50 \mathrm{~mL}, 1.0 \mathrm{~mL}, 2.5 \mathrm{~mL}, 5.0 \mathrm{~mL}, 10.0 \mathrm{~mL}$ 至 $50 \mathrm{~mL}$ 容量瓶中，用水定容。该标准系列浓度分别为 $10.0 \mu \mathrm{g} / \mathrm{mL}, 20.0 \mu \mathrm{g} / \mathrm{mL}, 50.0 \mu \mathrm{g} / \mathrm{mL}, 100 \mu \mathrm{g} / \mathrm{mL}, 200 \mu \mathrm{g} / \mathrm{mL}$ 。临用现配。

# 11 仪器和设备

11.1 液相色谱仪，配有紫外检测器或二极管阵列检测器。  
11.2 超声波振荡器。  
11.3 离心机：转速 $\geqslant 4000\mathrm{r / min}$   
11.4 样品粉碎机。  
11.5 恒温水浴锅。  
11.6 天平：感量 $1\mathrm{mg},0.1\mathrm{mg}$

# 12 分析步骤

# 12.1 试样溶液的制备

# 12.1.1 固体类和半固体类试样处理

称取均质后试样 $5.00\mathrm{g}$ 于 $50~\mathrm{mL}$ 离心管中，加入 $30~\mathrm{mL}$ 水，混匀，超声提取 $20\mathrm{min}$ ，离心 $(3000\mathrm{r / min})$ $20\mathrm{min}$ ，将上清液转出，用水洗涤残渣并定容至 $50~\mathrm{mL}$ 备用。含高蛋白类样品可在超声提取时加入 $2.0~\mathrm{mL}$ 硫酸锌溶液(10.2.4)和 $2.0~\mathrm{mL}$ 亚铁氰化钾溶液(10.2.5)。含高脂质类样品可在提取前先加入 $25~\mathrm{mL}$ 石油醚(10.1.8)振摇后弃去石油醚层除脂。

# 12.1.2 液体类试样处理

12.1.2.1 普通液体试样摇匀后可直接称取样品 $25.0\mathrm{g}$ ，用水定容至 $50~\mathrm{mL}$ 备用（如需要可过滤）。  
12.1.2.2 含二氧化碳的试样：称取 $25.0\mathrm{g}$ 试样于烧杯中， $60^{\circ}\mathrm{C}$ 水浴加热 $30\mathrm{min}$ 以除二氧化碳，放冷，用水定容至 $50~\mathrm{mL}$ 备用。  
12.1.2.3 含酒精的试样：称取 $25.0\mathrm{g}$ 试样于烧杯中，用氢氧化钠溶液调至弱碱性 $\mathrm{pH}7\sim 8,60^{\circ}\mathrm{C}$ 水浴加热 $30\mathrm{min}$ 以除酒精，放冷，用水定容至 $50~\mathrm{mL}$ 备用。  
12.1.2.4 含乳类饮料称取试样 $25.0\mathrm{g}$ 于 $50~\mathrm{mL}$ 离心管中，加入 $3.0~\mathrm{mL}$ 硫酸锌溶液(10.2.4)和 $3.0~\mathrm{mL}$

亚铁氰化钾溶液(10.2.5)，混匀，离心分层后，将上清液转出，用水洗涤残渣并定容至 $50~\mathrm{mL}$ 备用。

# 12.1.3 衍生化

准确移取 $10~\mathrm{mL}$ 已制备好的试样溶液(12.1.1或12.1.2)，加入 $2.0~\mathrm{mL}$ 硫酸溶液(10.2.1)， $5.0~\mathrm{mL}$ 正庚烷(10.1.1)，和 $1.0~\mathrm{mL}$ 次氯酸钠溶液(10.2.2)，剧烈振荡 $1\mathrm{min}$ ，静置分层，除去水层后在正庚烷层中加入 $25~\mathrm{mL}$ 碳酸氢钠溶液(10.2.3)，振荡 $1\mathrm{min}$ 。静置取上层有机相经 $0.45\mu \mathrm{m}$ 微孔有机相滤膜过滤，滤液备进样用。

# 12.2 仪器参考条件

12.2.1 色谱柱： $\mathrm{C_{18}}$ 柱， $5\mu \mathrm{m}, 150\mathrm{mm} \times 3.9\mathrm{mm}(\mathrm{i}, \mathrm{d})$ ，或同等性能的色谱柱。  
12.2.2 流动相：乙腈 $+$ 水 $(70 + 30)$ 。  
12.2.3 流速： $0.8\mathrm{mL / min}$   
12.2.4 进样量： $10~\mu \mathrm{L}$   
12.2.5 柱温： $40^{\circ}\mathrm{C}$ 。  
12.2.6 检测器：紫外检测器或二极管阵列检测器。  
12.2.7 检测波长： $314~\mathrm{nm}$

# 12.3 标准曲线的制作

移取 $10\mathrm{mL}$ 环己基氨基磺酸标准系列工作液(10.4.3)按12.1.3衍生化。取过 $0.45\mu \mathrm{m}$ 微孔有机相滤膜后的溶液 $10~\mu \mathrm{L}$ 分别注入液相色谱仪中，测定相应的峰面积，以标准工作溶液的浓度为横坐标，以环己基氨基磺酸钠衍生化产物 $N,N$ -二氯环己胺峰面积为纵坐标，绘制标准曲线。

# 12.4 样品的测定

将衍生后试样溶液(12.1.3) $10\mu \mathrm{L}$ 注入液相色谱仪中，保留时间定性，测得峰面积，根据标准曲线得到试样定容溶液中环己基氨基磺酸的浓度，平行测定次数不少于两次。

# 13 分析结果的表述

试样中环己基氨基磺酸含量按式(2)计算：

$$

式中：

$X_{2}$ ——试样中环己基氨基磺酸的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$V$ ——试样的最后定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000 ——由 $\mu \mathrm{g} / \mathrm{g}$ 换算成 $\mathrm{g} / \mathrm{kg}$ 的换算因子。

# 第三法 液相色谱-质谱/质谱法

# 16 原理

酒样经水浴加热除去乙醇后以水定容，用液相色谱-质谱/质谱仪测定其中的环己基氨基磺酸钠，外标法定量。

# 17 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 17.1 试剂

17.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
17.1.2 乙酸铵 $\mathrm{(CH_3COONH_4)}$ 。  
17.1.3 $10\mathrm{mmol / L}$ 乙酸铵溶液：称取 $0.78\mathrm{g}$ 乙酸铵，用水溶解并稀释至 $1000~\mathrm{mL}$ ，摇匀后经 $0.22\mu \mathrm{m}$ 水相滤膜过滤备用。  
17.2 标准品

环己基氨基磺酸钠标准品 $\mathrm{(C_6H_{12}NSO_3Na)}$ ：纯度 $\geqslant 99\%$ 。

# 17.3 标准溶液配制

17.3.1 环己基氨基磺酸标准储备液 $(5.00\mathrm{mg / mL})$ ：按第一法3.4.1 配制。  
17.3.2 环己基氨基磺酸标准中间液 $(1.00\mathrm{mg / mL})$ ：按第一法3.4.2 配制。  
17.3.3 环己基氨基磺酸标准工作液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：用水将 $1.00~\mathrm{mL}$ 标准中间液(17.3.2)定容至 $100~\mathrm{mL}$ 。放置于 $1^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱可保存一周。  
17.3.4 环己基氨基磺酸标准曲线系列工作液：分别吸取适量体积的标准工作液(17.3.3)，用水稀释，配成浓度分别为 $0.01\mu \mathrm{g / mL}, 0.05\mu \mathrm{g / mL}, 0.1\mu \mathrm{g / mL}, 0.5\mu \mathrm{g / mL}, 1.0\mu \mathrm{g / mL}, 2.0\mu \mathrm{g / mL}$ 的系列标准工作溶液。使用前配置。  
17.4 微孔滤膜： $0.22\mu \mathrm{m}$ ，水相。

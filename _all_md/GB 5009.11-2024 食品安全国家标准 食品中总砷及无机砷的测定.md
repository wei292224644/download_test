# 前言

本标准代替GB5009.11—2014《食品安全国家标准 食品中总砷及无机砷的测定》。

本标准与GB5009.11—2014相比，主要变化如下：

第一篇 食品中总碑的测定

——增加了石墨炉原子吸收光谱法为第三法；  
——删除了食品中总砷测定的银盐法；  
——修改了氢化物发生原子荧光光谱法为第一法，修改了试样消解方法；  
——修改了电感耦合等离子体质谱法为第二法。

第二篇 食品中无机砷的测定

——增加了第一法、第二法中稻米试样中无机砷提取的微波辅助提取法；  
——修改了第一法、第二法的适用范围、检出限、定量限、精密度和分析结果表述；  
——修改了第一法、第二法中试样的预处理方法、提取方法和分离测定条件。

# 食品中总碑及无机碑的测定

# 1 范围

本标准第一篇规定了食品中总砷的测定方法。

本标准第一篇第一法、第二法适用于食品中总碑的测定，第三法适用于食品（乳粉和调制乳粉、油脂及其制品、调味品、特殊膳食用食品除外）中总碑的测定。

本标准第二篇规定了食品中无机砷的测定方法。

本标准第二篇适用于谷物及其制品、水产动物及其制品、食用菌及其制品、油脂及其制品、调味品、婴幼儿辅助食品、藻类及其制品中无机砷的测定。

# 第一篇 食品中总碑的测定

# 第一法 氢化物发生原子荧光光谱法

# 2 原理

试样经消解处理后，加入硫脲使五价砷预还原为三价砷，再加入硼氢化钠或硼氢化钾使三价砷还原生成砷化氢，由氩气载入石英原子化器中分解为原子态砷，在砷空心阴极灯的发射光激发下产生原子荧光，其荧光强度在固定条件下与被测液中的砷浓度成正比，外标法定量。

3.2.4 氢氧化钠溶液 $(100\mathrm{g / L})$ ：称取 $10.0\mathrm{g}$ 氢氧化钠，溶于水并稀释至 $100~\mathrm{mL}$ ，混匀。

3.2.6 盐酸溶液 $(1 + 1)$ ：量取 $100~\mathrm{mL}$ 盐酸，缓缓倒入 $100~\mathrm{mL}$ 水中，混匀。  
3.2.7 硫酸溶液 $(1 + 9)$ ：量取 $100~\mathrm{mL}$ 硫酸，缓缓倒入 $900~\mathrm{mL}$ 水中，混匀。

# 3.4 标准溶液配制

3.4.2 砷标准使用液 $(1.00\mathrm{mg / L}$ ，以As计)：准确吸取 $1.00~\mathrm{mL}$ 砷标准储备液 $(100\mathrm{mg / L})$ 于 $100~\mathrm{mL}$ 容量瓶中，用硝酸溶液 $(2 + 98)$ 稀释定容至刻度。于 $2^{\circ}C\sim 8^{\circ}C$ 冰箱中避光保存，有效期3个月。  
3.4.3 砷标准系列溶液：取 $25~\mathrm{mL}$ 容量瓶或比色管7支，依次准确加入砷标准使用液 $(1.00\mathrm{mg / L})$ 、 $0.00~\mathrm{mL},0.05~\mathrm{mL},0.10~\mathrm{mL},0.25~\mathrm{mL},0.50~\mathrm{mL},1.50~\mathrm{mL}$ 和 $3.00~\mathrm{mL}$ （分别相当于砷浓度 $0.0~\mu \mathrm{g / L}$ 、 $2.0~\mu \mathrm{g / L},4.0~\mu \mathrm{g / L},10.0~\mu \mathrm{g / L},20.0~\mu \mathrm{g / L},60.0~\mu \mathrm{g / L}$ 和 $120.0~\mu \mathrm{g / L}$ )，各加硫酸溶液 $(1 + 9)12.5~\mathrm{mL}$ ，硫脲+抗坏血酸溶液 $2~\mathrm{mL}$ ，补加水至刻度，混匀，放置 $30\mathrm{min}$ 后测定。临用现配。

注：可根据仪器的灵敏度及样品中砷的实际含量微调标准系列溶液中砷的质量浓度范围。

# 5.1 试样制备

# 5.1.1 固体样品

# 5.1.1.1 干样

谷物、干食用菌、水产动物干制品等样品，取可食部分粉碎均匀；对于固体乳制品、面粉、婴幼儿谷类辅助食品等呈均匀状的粉末样品，摇匀。

# 5.1.3 半固体样品

# 5.2 试样消解

# 5.2.1 湿法消解

固体试样称取 $0.5\mathrm{g}\sim 2.5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，液体试样称取 $5.0\mathrm{g}\sim 10.0\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 于消解瓶或消解管中，加 $20~\mathrm{mL}$ 硝酸， $4\mathrm{mL}$ 高氯酸， $1.25\mathrm{mL}$ 硫酸，放置过夜。次日，于 $120^{\circ}C\sim 200^{\circ}C$ 逐级升温加热消解，若消解液处理至 $5\mathrm{mL}$ 左右时仍有未分解物质或色泽变深，补加硝酸 $5\mathrm{mL}\sim 10~\mathrm{mL}$ ，再消解至 $2\mathrm{mL}$ 左右，如此反复2次～3次，注意避免炭化，继续加热消解至消解液 $1\mathrm{mL}$ 左右，呈无色澄清，且消解瓶或消解管中充满白烟（对于有机砷较高的水产动物及其制品、食用菌及其制品、鱼油及其制品、磷虾油及其制品、藻类及其制品等样品，将消解温度升至 $280^{\circ}C\sim 300^{\circ}C$ 继续加热消解至消解液为 $0.5\mathrm{mL}$ 左右，呈无色澄清，且消解瓶或消解管中充满白烟)。冷却后，沿消解容器壁缓慢加水约 $10~\mathrm{mL}$ 再蒸发至消解瓶或消解管充满白烟。冷却，用水将消解液转入 $25~\mathrm{mL}$ 容量瓶或比色管中，加入 $2\mathrm{mL}$ 硫脲十抗坏血酸溶液，用水定容至刻度，混匀，放置 $30\mathrm{min}$ ，待测。同时做空白试验。

# 5.2.2 干灰化法

固体试样称取 $1.0\mathrm{g}\sim 2.5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，液体试样(油脂样品除外)称取 $4.0\mathrm{g}$ (精确至 $0.001\mathrm{g})$ ，置于 $50~\mathrm{mL}\sim 100~\mathrm{mL}$ 坩埚中。加 $10~\mathrm{mL}$ 硝酸镁溶液 $(150~\mathrm{g / L})$ 混匀，低热蒸干，将 $1\mathrm{g}$ 氧化镁覆盖在干渣上（对于油脂样品称取 $1.00\mathrm{g}$ 于 $50~\mathrm{mL}\sim 100~\mathrm{mL}$ 坩埚中，直接加入 $0.2\mathrm{g}$ 氧化镁覆盖在油脂上），于电炉上炭化至无黑烟，移入 $550^{\circ}C$ 马弗炉灰化 $4\mathrm{h}$ 。取出放冷，小心加入 $5\mathrm{mL}\sim 10\mathrm{mL}$ 盐酸溶液 $(1 + 1)$ 以中和氧化镁并溶解灰分，转入 $25~\mathrm{mL}$ 容量瓶或比色管，向容量瓶或比色管中加入 $2\mathrm{mL}$ 硫脲十抗坏血酸溶液，另用硫酸溶液 $(1 + 9)$ 分次洗涤坩埚后，合并洗涤液并定容至刻度，混匀，放置 $30\mathrm{min}$ ，待测。同时做空白试验。

# 5.2.3 微波消解法

固体试样和油脂及其制品试样称取 $0.2\mathrm{g}\sim 0.8\mathrm{g}$ (精确至 $0.001\mathrm{g}$ )，含水分较多的试样或液体试样称取 $1.0\mathrm{g}\sim 3.0\mathrm{g}$ (精确至 $0.001\mathrm{g}$ )于消解罐中，加入 $5\mathrm{mL}\sim 8\mathrm{mL}$ 硝酸，放置 $30\mathrm{min}$ 以上。对于肉类、油脂等难消解的样品再加入 $0.5\mathrm{mL}\sim 1\mathrm{mL}$ 过氧化氢，盖好安全阀，将消解罐放入微波消解系统中。根据不同类型的样品，设置适宜的微波消解程序（见附录A表A.1)，按相关步骤进行消解，消解结束后，于 $135~^\circ \mathrm{C}\sim 145~^\circ \mathrm{C}$ 赶酸至 $1\mathrm{mL}\sim 2\mathrm{mL}$ 。将消化液转移至 $25~\mathrm{mL}$ 容量瓶或比色管，用少量硫酸溶液 $(1 + 9)$ 洗涤消解罐3次，合并洗涤液于容量瓶或比色管中并加入 $2\mathrm{mL}$ 硫脲十抗坏血酸溶液，用硫酸溶液 $(1 + 9)$ 定容至刻度，混匀，放置 $30\mathrm{min}$ ，待测。同时做空白试验。

注：微波消解法不适用有机砷含量较高的水产动物及其制品、食用菌及其制品、鱼油、磷虾油及其制品、水产调味品、藻类及其制品等基质复杂的样品。

# 5.2.4 压力罐消解法

固体试样和油脂及其制品试样称取 $0.2\mathrm{g}\sim 1.0\mathrm{g}$ (精确至 $0.001\mathrm{g}$ )，鲜样或液体试样称取 $1.0\mathrm{g}\sim 5.0\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )于消解内罐中，加入 $5\mathrm{mL}$ 硝酸浸泡过夜。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱， $140^{\circ}\mathrm{C}\sim 160^{\circ}\mathrm{C}$ 保持 $3\mathrm{h}\sim 4\mathrm{h}$ ，自然冷却至室温，然后缓慢旋松不锈钢外套，将消解内罐取出，放在控温电热板上 $135^{\circ}\mathrm{C}\sim 145^{\circ}\mathrm{C}$ 赶酸至 $1\mathrm{mL}\sim 2\mathrm{mL}$ 。将消化液转移至 $25\mathrm{mL}$ 容量瓶或比色管，用少量硫酸溶液 $(1 + 9)$ 洗涤消解罐3次，合并洗涤液于容量瓶或比色管中并加入 $2\mathrm{mL}$ 硫脲十抗坏血酸溶液，用硫酸溶液 $(1 + 9)$ 定容至刻度，混匀，放置 $30\mathrm{min}$ ，待测。同时做空白试验。

注：压力罐消解法不适用有机砷含量较高的水产动物及其制品、食用菌及其制品、鱼油、磷虾油及其制品、水产调味品、藻类及其制品等基质复杂的样品。

# 5.3 仪器参考条件

负高压：260V；砷空心阴极灯电流： $50\mathrm{mA}\sim 80\mathrm{mA}$ ；载气：氩气；载气流速： $500~\mathrm{mL / min}$ ；屏蔽气流速： $800~\mathrm{mL / min}$ ；测量方式：荧光强度；读数方式：峰面积。

# 5.4 标准曲线的制作

仪器预热稳定后，将试剂空白、标准系列溶液依次引入仪器进行原子荧光强度的测定。以原子荧光强度为纵坐标，砷质量浓度为横坐标，制作标准曲线，得到回归方程。

# 5.5 试样溶液的测定

相同条件下，将空白溶液和样品溶液分别引入仪器进行测定。根据回归方程计算出样品中砷元素的质量浓度。

# 6 分析结果的表述

试样中砷的含量按式(1)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中砷的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——试样溶液中砷的含量，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白液中砷的含量，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样消化液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

当砷含量 $\geqslant 1.00\mathrm{mg / kg}$ 时，计算结果保留3位有效数字；当砷含量 $<  1.00~\mathrm{mg / kg}$ 时，计算结果保留2位有效数字。

# 7 精密度

样品中砷含量大于 $1.00\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ ；小于或等于 $1.00\mathrm{mg / kg}$ 且大于 $0.10\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ ；小于或等于 $0.10\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当固体样品称样量为 $1.0\mathrm{g}$ ，定容体积为 $25~\mathrm{mL}$ 时，方法检出限为 $0.01\mathrm{mg / kg}$ ，方法定量限为 $0.04\mathrm{mg / kg}$ 。当液体样品取样量为 $2\mathrm{g}$ ，定容体积为 $25~\mathrm{mL}$ 时，方法检出限为 $0.005\mathrm{mg / kg}$ ，方法定量限为 $0.02\mathrm{mg / kg}$ 。

# 9 原理

试样消解处理后，经石墨炉原子化，在 $193.7\mathrm{nm}$ 处测定吸光度。在一定浓度范围内砷的吸光度值与砷含量成正比，外标法定量。

# 13 分析结果的表述

试样中砷的含量按式(2)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中砷的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——试样溶液中砷的含量，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白液中砷的含量，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样消化液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

当砷含量 $\geqslant 1.00\mathrm{mg / kg}$ 时，计算结果保留3位有效数字；当砷含量 $<  1.00~\mathrm{mg / kg}$ 时，计算结果保留2位有效数字。

# 20 分析结果的表述

试样中 $\mathrm{As(III)}$ 或 $\mathrm{As(V)}$ 含量按式(3)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {3}
$$

式中：

$X$ ——试样中 $\mathrm{As(III)}$ 或 $\mathrm{As(V)}$ 的含量（以As计），单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——测定溶液中 $\mathrm{As(III)}$ 或 $\mathrm{As(V)}$ 的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白溶液中 $\mathrm{As(III)}$ 或 $\mathrm{As(V)}$ 的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

V ——试样提取液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

无机砷含量等于As（Ⅲ）含量与As（V）含量之和。

当无机砷含量 $\geqslant 1.00\mathrm{mg / kg}$ 时，计算结果保留3位有效数字，当无机砷含量 $<  1.00~\mathrm{mg / kg}$ 时，计算结果保留2位有效数字。

# 21 精密度

样品中无机砷含量大于 $1.00\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ ；小于或等于 $1.00\mathrm{mg / kg}$ 且大于 $0.10\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ ；小于或等于 $0.10\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 第二法 液相色谱-电感耦合等离子体质谱联用法

# 23 原理

试样中无机砷经稀硝酸等提取后，以液相色谱进行分离，分离后的目标化合物经过雾化由载气送入电感耦合等离子体-炬焰中，经过蒸发、解离、原子化、电离等过程，经离子采集系统进入质谱仪，质谱仪根据质荷比进行分离测定。以保留时间和质荷比定性，外标法定量。

# 27 分析结果的表述

试样中 $\mathrm{As(III)}$ 或 $\mathrm{As(V)}$ 的含量按式(4)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {4}
$$

式中：

$X$ ——试样中 $\mathrm{As(III)}$ 或 $\mathrm{As(V)}$ 的含量（以As计），单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——测定溶液中 $\mathrm{As(III)}$ 或 $\mathrm{As(V)}$ 的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白溶液中 $\mathrm{As(III)}$ 或 $\mathrm{As(V)}$ 的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样提取液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

无机砷含量等于 $\mathrm{As(III)}$ 含量与 $\mathrm{As(V)}$ 含量之和。

当无机砷含量 $\geqslant 1.00\mathrm{mg / kg}$ 时，计算结果保留3位有效数字，当无机砷含量 $<  1.00~\mathrm{mg / kg}$ 时，计算结果保留2位有效数字。

# 生乳冰点的测定

2016-08-31 发布

2017-03-01 实施

# 前言

本标准代替GB5413.38—2010《食品安全国家标准 生乳冰点的测定》。

本标准与GB5413.38—2010相比，主要变化如下：

——修改了原理；  
——修改了“试剂和材料”；  
——在“试剂和材料”的“氯化钠标准溶液”中增加“标准溶液C”；  
——在“分析步骤”的“仪器校准”中增加“C校准”和“质控校准”。

# 食品安全国家标准

# 生乳冰点的测定

# 1 范围

本标准规定了热敏电阻冰点仪测定生乳冰点的方法。

本标准适用于生乳冰点的测定。

# 2 原理

生乳样品过冷至适当温度，当被测乳样冷却到 $-3\,^{\circ}\mathrm{C}$ 时，通过瞬时释放热量使样品产生结晶，待样品温度达到平衡状态，并在 $20\,s$ 内温度回升不超过 $0.5\,m\,^{\circ}\mathrm{C}$ ，此时的温度即为样品的冰点。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯或以上等级，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 乙二醇 $(\mathrm{C}_2\mathrm{H}_6\mathrm{O}_2)$ 。  
3.1.2 氯化钠 $(\mathrm{NaCl})$ 。

# 3.2 试剂配制

3.2.1 氯化钠(NaCl)：氯化钠磨细后置于干燥箱中， $130^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 干燥 $24\mathrm{h}$ 以上，于干燥器中冷却至室温。  
3.2.2 冷却液：量取 $330~\mathrm{mL}$ 乙二醇(3.1.1)于 $1000~\mathrm{mL}$ 容量瓶中，用水定容至刻度并摇匀，其体积分数为 $33\%$ 。

# 3.3 氯化钠标准溶液

3.3.1 标准溶液 A: 称取 $6.763 \mathrm{~g}$ 氯化钠(3.2.1), 溶于 $1000 \mathrm{~g} \pm 0.1 \mathrm{~g}$ 水中。将标准溶液分装贮存于容量不超过 $250 \mathrm{~mL}$ 的聚乙烯塑料瓶中, 并置于 $5^{\circ} \mathrm{C}$ 左右冰箱冷藏, 保存期限为两个月。其冰点值为 $-400 \mathrm{~m}^{\circ} \mathrm{C}$ 。  
3.3.2 标准溶液B：称取 $9.475\mathrm{g}$ 氯化钠(3.2.1)，溶于 $1000\mathrm{g}\pm 0.1\mathrm{g}$ 水中。将标准溶液分装贮存于容量不超过 $250~\mathrm{mL}$ 的聚乙烯塑料瓶中，并置于 $5^{\circ}C$ 左右冰箱冷藏，保存期限为两个月。其冰点值为 $-557\mathrm{m}^{\circ}\mathrm{C}$ 。  
3.3.3 标准溶液C：称取 $10.220\mathrm{g}$ 氯化钠(3.2.1)，溶于 $1000\mathrm{g}\pm 0.1\mathrm{g}$ 水中。将标准溶液分装贮存于容量不超过 $250~\mathrm{mL}$ 的聚乙烯塑料瓶中，并置于 $5^{\circ}C$ 左右冰箱冷藏，保存期限为两个月。其冰点值为 $-600\mathrm{m}^{\circ}\mathrm{C}$ 。

# 4 仪器和设备

4.1 分析天平：感量 $0.0001\mathrm{g}$ 。

4.2 热敏电阻冰点仪：检测装置、冷却装置、搅拌金属棒、结晶装置(见图1)及温度显示仪。

# a）检测装置及冷却装置

温度传感器为直径为 $1.60 \mathrm{~mm} \pm 0.4 \mathrm{~mm}$ 的玻璃探头，在 $0^{\circ} \mathrm{C}$ 时的电阻在 $3 \Omega \sim 30 \mathrm{k} \Omega$ 之间。传感器转轴的材质和直径应保证向样品的热传递值控制在 $2.5 \times 10^{-3} \mathrm{~J} / \mathrm{s}$ 以内。当探头在测量位置时，热敏电阻的顶部应位于样品管的中轴线，且顶部离内壁与管底保持相等距离（见图1）。温度传感器和相应的电子线路在 $-600 \mathrm{m}^{\circ} \mathrm{C} \sim 400 \mathrm{m}^{\circ} \mathrm{C}$ 之间测量分辨率为 $1 \mathrm{~m}^{\circ} \mathrm{C}$ 。冷却装置应保持冷却液体的温度恒定在 $-7^{\circ} \mathrm{C} \pm 0.5^{\circ} \mathrm{C}$ 。

# b） 搅拌金属棒

耐腐蚀，在冷却过程中搅拌测试样品。

搅拌金属棒应根据相应仪器的安放位置来调整振幅。正常搅拌时金属棒不得碰撞玻璃传感器或样品管壁。

# c） 结晶装置

当测试样品达到 $-3.0^{\circ} \mathrm{C}$ 时，启动结晶的机械振动装置，在结晶时使搅拌金属棒在 $1 \mathrm{~s} \sim 2 \mathrm{~s}$ 内加大振幅，使其碰撞样品管壁。

4.3 干燥箱：温度可控制在 $130^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。  
4.4 样品管：硼硅玻璃，长度 $50.5\mathrm{mm}\pm 0.2\mathrm{mm}$ ，外部直径为 $16.0\mathrm{mm}\pm 0.2\mathrm{mm}$ ，内部直径为 $13.7\mathrm{mm}\pm 0.3\mathrm{mm}$ 。  
4.5 称量瓶。  
4.6 容量瓶： $1000~\mathrm{mL}$ ，符合GB/T12806—2011等级A的要求。  
4.7 干燥器：内有硅胶湿度计。  
4.8 移液器： $1\mathrm{mL}\sim 5\mathrm{mL}$   
4.9 聚乙烯瓶：容量不超过 $250~\mathrm{mL}$

# 5 分析步骤

# 5.1 试样制备

测试样品要保存在 $0\mathrm{^\circ C}\sim 6\mathrm{^\circ C}$ 的冰箱中并于 $48\mathrm{h}$ 内完成测定。测试前样品应放至室温，且测试样品和氯化钠标准溶液测试时的温度应保持一致。

# 5.2 仪器预冷

开启热敏电阻冰点仪(4.2)，等待热敏电阻冰点仪传感探头升起后，打开冷阱盖，按生产商规定加入相应体积冷却液(3.2.2)，盖上盖子，冰点仪进行预冷。预冷 $30\mathrm{min}$ 后，开始测量。

# 5.3 校准

# 5.3.1 原则

校准前应按表1配制不同冰点值的氯化钠标准溶液。可选择表1中两个不同冰点值的氯化钠标准溶液进行仪器校准，两个氯化钠标准溶液冰点差值不应少于 $100\mathrm{m}^{\circ}\mathrm{C}$ ，且覆盖到被测样品相近冰点值范围。

# 5.3.2 仪器校准

5.3.2.1 A 校准：分别取 $2.5 \mathrm{~mL}$ 标准溶液 A(3.3.1)，依次放入三个样品管中，在启动后的冷阱中插入

装有校准液A的样品管。当重复测量值在 $-400\mathrm{m}^{\circ}\mathrm{C}\pm 2\mathrm{m}^{\circ}\mathrm{C}$ 校准值时，完成校准。

5.3.2.2 B 校准：分别取 $2.5 \mathrm{~mL}$ 标准溶液 B(3.3.2)，依次放入三个样品管中，在启动后的冷阱中插入装有校准液 B 的样品管。当重复测量值在 $-557 \mathrm{~m}^{\circ} \mathrm{C} \pm 2 \mathrm{~m}^{\circ} \mathrm{C}$ 校准值时，完成校准。  
5.3.2.3 C 校准：测定生羊乳时，还应使用 C 校准。分别取 $2.5 \mathrm{~mL}$ 标准溶液 C(3.3.3)，依次放入三个样品管中，在启动后的冷阱中插入装有校准溶液 C 的样品管。当重复测量值在 $-600 \mathrm{m}^{\circ} \mathrm{C} \pm 2 \mathrm{m}^{\circ} \mathrm{C}$ 校准值时，完成校准。

# 5.3.3 质控校准

在每次开始测试前应使用质控校准。连续测定乳样时，冰点仪每小时至少进行一次质控校准。如两次测量的算术平均值与氯化钠标准溶液 $(-512\mathrm{m}^{\circ}\mathrm{C})$ 差值大于 $2\mathrm{m}^{\circ}\mathrm{C}$ 时，应重新开展仪器校准（5.3.2）。

# 5.4 样品测定

5.4.1 轻轻摇匀待测试样(5.1)，应避免混入空气产生气泡。移取 $2.5 \mathrm{~mL}$ 试样至一个干燥清洁的样品管中，将样品管放到已校准过的热敏电阻冰点仪(4.2)的测量孔中。开启冰点仪冷却试样，当温度达到 $-3.0^{\circ} \mathrm{C} \pm 0.1^{\circ} \mathrm{C}$ 时试样开始冻结，当温度达到平衡(在 $20 \mathrm{~s}$ 内温度回升不超过 $0.5 \mathrm{~m}^{\circ} \mathrm{C}$ )时，冰点仪停止测量，传感头升起，显示温度即为样品冰点值。测试结束后，应保证探头和搅拌金属棒清洁、干燥。  
5.4.2 如果试样在温度达到 $-3.0^{\circ}\mathrm{C} \pm 0.1^{\circ}\mathrm{C}$ 前已开始冻结，需重新取样测试(5.4.1)。如果第二次测试的冻结仍然太早发生，那么将剩余的样品于 $40^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 加热 $5\mathrm{min}$ ，以融化结晶脂肪，再重复样品测定步骤(5.4.1)。  
5.4.3 测定结束后，移走样品管，并用水冲洗温度传感器和搅拌金属棒并擦拭干净。  
5.4.4 记录试样的冰点测定值。

# 6 分析结果的表述

生乳样品的冰点测定值取两次测定结果的平均值，单位以 $\mathrm{m}^{\circ}\mathrm{C}$ 计，保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不超过 $4\mathrm{m}^{\circ}\mathrm{C}$ 。

# 8 其他

方法检出限为 $2\mathrm{m}^{\circ}\mathrm{C}$ 。

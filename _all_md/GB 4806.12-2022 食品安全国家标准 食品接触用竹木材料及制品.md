# 食品安全国家标准

# 食品接触用竹木材料及制品

# 1 范围

本标准适用于食品接触用竹木材料及制品。

# 2 术语和定义

# 2.1 食品接触用竹木材料及制品

在正常使用条件下，各种已经或预期可能与食品或食品添加剂(以下简称食品)接触，或其成分可能转移到食品中的，以竹、木或软木为原料制成的材料及制品，包括软木塞和植物纤维板制品等。

# 3 基本要求

食品接触用竹木材料及制品应符合GB4806.1的规定。

# 4 技术要求

# 4.1 原料要求

4.1.1 食品接触用竹木材料及制品使用的原料不应对人体健康产生危害。  
4.1.2 食品接触用竹木材料及制品中油、蜡及其他添加剂的使用应符合GB9685及相关公告的规定。

# 4.2 感官要求

# 4.3.2 其他理化指标

食品接触用竹木材料及制品应符合GB9685及相关公告对油、蜡及其他添加剂的特定迁移限量(SML)、特定迁移总量限量[SML(T)]和最大残留量(QM)等理化指标的规定。

# 4.4 微生物限量

预期与食品直接接触，且不经过消毒或清洗直接使用的竹木材料及制品的微生物应符合GB14934的规定，与食用、烹饪或者加工前需经去皮、去壳或清洗的食品接触的竹木材料及制品除外。

# 4.5 其他技术要求

使用了涂料、油墨和(或)黏合剂等材料的食品接触用竹木材料及制品，还应符合涂料、油墨和(或)黏合剂等相应食品安全国家标准的规定。

# 5 其他

# 5.1 迁移试验

迁移试验应按GB31604.1和GB5009.156的规定执行。本标准有特殊规定的除外。

# 5.2 标签标识

标签标识应符合GB4806.1的规定。

# 附录A

# 食品接触用竹木材料及制品中二氧化硫迁移量的测定

# A.1 原理

选择水作为食品模拟物，对试样进行迁移试验，试样中二氧化硫迁移至水中，在密闭装置中对试验溶液进行酸化、蒸馏，释放出其中的二氧化硫，释放物用乙酸铅溶液吸收。吸收后的溶液用浓盐酸酸化，碘标准溶液滴定，根据所消耗的碘标准溶液量计算得到试样中二氧化硫的迁移量。

# A.2 试剂和材料

按照GB31604.32—2016第一法(滴定法)中第3章进行。

# A.3 仪器和设备

按照GB31604.32—2016第一法(滴定法)中第4章进行，其中碘量瓶更改为梨形瓶，全玻璃蒸馏装置参见图A.1。

# A.4 分析步骤

# A.4.1 食品模拟物试液的制备

按照GB5009.156和GB31604.1的要求，选择水作为模拟物，对样品进行迁移试验，得到食品模拟物试验溶液。迁移试验过程中，应避免样品受到其他来源的二氧化硫污染，并密封以防止二氧化硫挥发损失。

# A.4.2 空白试液的制备

按照A.4.1的操作处理未与竹木材料及制品接触的食品模拟物。

# A.4.3 蒸馏

量取 $250~\mathrm{mL}$ 迁移试验所得试验溶液，置于 $500~\mathrm{mL}$ 圆底烧瓶b（见图A.1）中；在圆底烧瓶a（见图A.1)中加入 $250~\mathrm{mL}$ 水，为防止测试溶液暴沸，加入2颗 $\sim 3$ 颗玻璃珠，按照GB31604.32—2016第一法(滴定法)5.2进行蒸馏，从“装上冷凝装置”做起。

空白试验溶液同时按照上述操作进行蒸馏。

# A.4.4 滴定

按照GB31604.32—2016第一法(滴定法)5.3进行滴定。空白试验溶液同时进行滴定。

# A.5 结果与表述

试样中二氧化硫迁移量按式(A.1)计算：

$$
X = \frac {\left(V _ {1} - V _ {0}\right) \times c \times 3 2 \times V _ {2}}{S _ {1} \times 0 . 2 5} \times \frac {S _ {2}}{m} \quad \dots \tag {A.1}
$$

式中：

$X$ ——试样中二氧化硫迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$V_{1}$ ——滴定试验溶液所用碘标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白试验溶液所用碘标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——碘标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

32 ——1L碘标准溶液 $\left[\left(\frac{1}{2}\mathrm{I}_2\right) = 1.0\mathrm{mol} / \mathrm{L}\right]$ 相当的二氧化硫的质量，单位为克每摩尔 $(\mathrm{g / mol})$

$V_{2}$ ——试样浸泡溶液的体积，单位为升 $(\mathrm{L})$

$S_{1}$ ——试样测试时的接触面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

0.25——试样溶液的蒸馏体积，单位为升（L）；

$S_{2}$ ——试样实际接触食品的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$m$ ——试样实际接触固态食品的质量，或实际接触液态食品的体积所对应的食品质量，单位为千克 $(\mathrm{kg})$ ；各种液态食品通常按密度为 $1\mathrm{kg} / \mathrm{L}$ 将其体积换算为相应的质量。

注：试样实际接触食品的面积体积比不能确定时，式(A.1)中的“ $S_{2} / m$ ”取 $6\mathrm{dm}^2 /\mathrm{kg}$

计算结果保留三位有效数字。

# A.6 精密度

当二氧化硫迁移量不高于 $5\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测试结果的绝对差值不得超过算术平均值的 $20\%$ 。当二氧化硫迁移量高于 $5\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测试结果的绝对差值不得超过算术平均值的 $10\%$ 。

# A.7 其他

本方法食品模拟物中二氧化硫迁移量的定量限为 $1.0\mathrm{mg / kg}$

# 附录B

# 食品接触用竹木材料及制品中噻菌灵、邻苯基苯酚、抑霉唑和联苯的测定

# B.1 原理

取代表性样品，经破碎后用甲醇提取，旋转蒸发至干，用甲醇定容，过滤后，用配二极管阵列检测器或紫外检测器的高效液相色谱仪进行检测，外标法定量。

# B.2 试剂和材料

# B.2.1 试剂

除非另有说明，所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

B.2.1.1 乙腈：色谱纯。  
B.2.1.2 甲醇：色谱纯。  
B.2.1.3 磷酸。  
B.2.1.4 十二烷基磺酸钠。  
B.2.1.5 流动相溶液：准确称取十二烷基磺酸钠 $0.681\mathrm{g}$ ，加入 $350~\mathrm{mL}$ 甲醇、 $50~\mathrm{mL}$ 乙腈、 $100~\mathrm{mL}$ 水和 $1\mathrm{mL}$ 磷酸，混匀，过滤后使用。

# B.2.2 标准品

B.2.2.1 噬菌灵标准品 $\mathrm{(C_{10}H_7N_3S}$ ,CAS号：148-79-8)：纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准物质。  
B.2.2.2 邻苯基苯酚标准品 $\left(\mathrm{C}_{12} \mathrm{H}_{10} \mathrm{O}, \mathrm{CAS}\right.$ 号：90-43-7）：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质。  
B.2.2.3 抑霉唑标准品 $\mathrm{(C_{14}H_{14}Cl_2N_2O}$ ，CAS号：35554-44-0）：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质。  
B.2.2.4 联苯标准品 $\left(\mathrm{C}_{12} \mathrm{H}_{10}, \mathrm{CAS}\right.$ 号：92-52-4）：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质。

# B.2.3 标准溶液配制

B.2.3.1 标准储备液 $(1000\mathrm{mg / L})$ ：分别准确称量噻菌灵、邻苯基苯酚、抑霉唑和联苯标准品 $10\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )，甲醇溶解后各自转移至 $10~\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度，混匀，于 $4^{\circ}C$ 下保存，有效期为3个月。  
B.2.3.2 混合标准中间液 $(50\mathrm{mg / L})$ ：分别准确吸取 $0.5\mathrm{mL}$ 标准储备液(B.2.3.1)于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释并定容至刻度，混匀，于 $4^{\circ}C$ 下保存，有效期为1个月。  
B.2.3.3 标准工作溶液：分别准确吸取噻菌灵、邻苯基苯酚、抑霉唑和联苯的混合标准中间液(B.2.3.2) $0.12\mathrm{mL}, 0.2\mathrm{mL}, 0.6\mathrm{mL}, 2\mathrm{mL}, 4\mathrm{mL}$ 于5个 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释并定容至刻度，混匀。得到噻菌灵、邻苯基苯酚、抑霉唑和联苯浓度为 $0.6\mathrm{mg / L}, 1\mathrm{mg / L}, 3\mathrm{mg / L}, 10\mathrm{mg / L}, 20\mathrm{mg / L}$ 的标准工作溶液。现用现配。

# B.3 仪器和设备

B.3.1 高效液相色谱仪：配二极管阵列检测器或紫外检测器。  
B.3.2 分析天平：感量为 $0.1\mathrm{mg},0.01\mathrm{g}$   
B.3.3 超声波发生器。  
B.3.4 旋转蒸发仪。  
B.3.5 $0.45\mu \mathrm{m}$ 尼龙微孔滤膜。

# B.4 分析步骤

# B.4.1 试样处理

对于可用剪刀、斜口钳等工具直接剪碎的样品，如筷子、蒸笼等，直接使用剪刀、斜口钳等剪成粒径不超过 $2.5\mathrm{mm}$ 的颗粒，装入洁净容器内，混匀。

对于难以用剪刀剪碎的样品，如较厚的砧板等，可用锯条或电锯等工具将代表性部位切割成小块，再用剪刀等工具剪成粒径不超过 $2.5\mathrm{mm}$ 的颗粒，装入洁净容器内，混匀。对于由层间压合或粘合而成的样品，在进行采样时，取样部位应包含所有的复合层。

# B.4.2 前处理步骤

称取试样 $2\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $30~\mathrm{mL}$ 甲醇，超声提取 $40\mathrm{min}$ 后过滤，残渣使用 $20~\mathrm{mL}$ 甲醇进行淋洗，将淋洗液与滤液合并于梨形瓶中，不高于 $35^{\circ}C$ 水浴下旋转蒸发至干，准确加入 $1\mathrm{mL}$ 甲醇重新溶解，经 $0.45\mu \mathrm{m}$ 微孔滤膜过滤后高效液相色谱法进行分析。

# B.4.3 空白溶液的制备

除不加试样外，采用与B.4.2完全相同的分析步骤、试剂和用量。

# B.4.4 色谱参考条件

B.4.4.1 色谱柱： $\mathrm{C_{18}}$ 液相色谱柱， $250 \mathrm{~mm} \times 4.6 \mathrm{~mm}, 5 \mu \mathrm{m}$ ，或等效柱。  
B.4.4.2 流动相：流动相溶液（B.2.1.5）， $100\%$ 。  
B.4.4.3 流速： $0.5\mathrm{mL / min}$   
B.4.4.4 检测波长：噻菌灵、邻苯基苯酚、联苯检测波长为 $247\mathrm{nm}$ ，抑霉唑的检测波长为 $226\mathrm{nm}$ 。  
B.4.4.5 进样量： $20~\mu \mathrm{L}$   
B.4.4.6 柱温： $40^{\circ}C$ 。

# B.4.5 标准曲线的绘制

将标准工作溶液(B.2.3.3)分别注入高效液相色谱仪中，测定相应的物质，以各标准工作溶液的浓度为横坐标，以各相应的物质的峰面积为纵坐标，绘制标准曲线。标准溶液的色谱图参见图B.1。

# B.4.6 试样溶液的测定

将试样溶液(B.4.2)及空白溶液(B.4.3)依次注入液相色谱仪中，根据标准曲线得到待测液中各物质的浓度。

# B.5 分析结果的表述

试样中各物质的残留量按式(B.1)计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {B.1}
$$

式中：

$X$ ——试样中噻菌灵、邻苯基苯酚、抑霉唑或联苯的残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——由标准曲线得到的试样溶液中噻菌灵、邻苯基苯酚、抑霉唑或联苯的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$c_{0}$ ——由标准曲线得到的空白溶液中噻菌灵、邻苯基苯酚、抑霉唑或联苯的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样定容的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

# B.6 精密度

当待测物含量不高于 $1\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测试结果的绝对差值不得超过算术平均值的 $20\%$ 。当待测物含量高于 $1\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测试结果的绝对差值不得超过算术平均值的 $10\%$ 。

# B.7 其他

当取样量为 $2\mathrm{g}$ 时，噻菌灵、邻苯基苯酚、抑霉唑和联苯四种物质残留量的检出限均为 $0.15\mathrm{mg / kg}$ ，定量限均为 $0.30\mathrm{mg / kg}$ 。

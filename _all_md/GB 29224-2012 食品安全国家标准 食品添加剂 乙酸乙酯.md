# 检验方法

# A.1 警示

试验方法规定的一些试验过程可能导致危险情况，操作者应采取适当的安全和防护措施。

# A. 2 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682—2008中规定的三级水。试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603之规定制备；所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

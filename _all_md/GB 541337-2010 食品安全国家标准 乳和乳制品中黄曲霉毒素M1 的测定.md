# 食品安全国家标准

# 乳和乳制品中黄曲霉毒素 $\mathrm{M}_{1}$ 的测定

# 1 范围

本标准规定了乳和乳制品中黄曲霉毒素 $\mathbf{M}_1$ 的测定方法。

标准第一法适用于乳和乳制品中黄曲霉毒素 $\mathbf{M}_{1}$ 的测定；第二法适用于乳、乳粉，以及低脂乳、脱脂乳、低脂乳粉和脱脂乳粉中黄曲霉毒素 $\mathbf{M}_{1}$ 的测定；第三法适用于乳和乳粉中黄曲霉毒素 $\mathbf{M}_{1}$ 的测定；第四法适用于液态乳和乳粉中黄曲霉毒素 $\mathbf{M}_{1}$ 的测定。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

# 第一法 免疫亲和层析净化 液相色谱—串联质谱法

# 3 原理

试样液体或固体试样提取液经均质、超声提取、离心，取上清液经免疫亲和柱净化，洗脱液经氮气吹干，定容，微孔滤膜过滤，经液相色谱分离，电喷雾离子源离子化，多反应离子监测（MRM）方式检测。基质加标外标法定量。

# 4 试剂和材料

除非另有规定，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

4.1 甲酸（HCOOH）。  
4.2 乙腈 $(\mathrm{CH}_3\mathrm{CN})$ ：色谱纯。  
4.3 石油醚（ $\mathrm{C_nH_{2n + 2}}$ ）：沸程为 $30^{\circ}C\sim 60^{\circ}C$   
4.4 三氯甲烷（ $\mathrm{CHCl}_3$ ）。  
4.5 氮气：纯度 $\geq 99.9\%$   
4.6 黄曲霉毒素 $\mathrm{M}_{1}$ 标准样品：纯度 $\geq 98\%$   
4.7 乙腈-水溶液 $(1 + 4)$ ：在 $400~\mathrm{mL}$ 水中加入 $100~\mathrm{mL}$ 乙腈。  
4.8 乙腈-水溶液 $(1 + 9)$ ：在 $450~\mathrm{mL}$ 水中加入 $50~\mathrm{mL}$ 乙腈。  
4.9 $0.1\%$ 甲酸水溶液：吸取 $1\mathrm{mL}$ 甲酸（4.1），用水稀释至 $1000~\mathrm{mL}$ 。  
4.10 乙腈-甲醇溶液（50+50）：在 $500\mathrm{mL}$ 乙腈中加入 $500\mathrm{mL}$ 甲醇。  
4.11 氢氧化钠溶液（0.5 mol/L）：称取2g氢氧化钠溶解于100mL水中。

# 4.12 空白基质溶液

分别称取与待测样品基质相同的、不含所测黄曲霉毒素的阴性试样8份于 $100 \mathrm{~mL}$ 烧杯中。以下操作按6.1试液提取和6.2净化步骤进行。合并所得8份试样的纯化液，用 $0.22 \mu \mathrm{m}$ 微孔滤膜的一次性滤头（5.23）过滤。弃去前 $0.5 \mathrm{~mL}$ 滤液，接取少量滤液供液相色谱一质谱联用仪检测。

获得色谱一质谱图后，对照附录A中的图A.2，在相应的保留时间处，应不含黄曲霉毒素 $\mathbf{M}_1$ 。剩余滤液转移至棕色瓶中，在- $20^{\circ}\mathrm{C}$ 电冰箱内保存，供配制标准系列溶液使用。

4.13 黄曲霉毒素 $\mathrm{M}_{1}$ 标准储备溶液：分别称取标准品黄曲霉毒素 $\mathrm{M}_{1} 0.10 \mathrm{mg}$ （精确至 $0.01 \mathrm{mg}$ ），用三氯甲烷（4.4）溶解定容至 $10 \mathrm{~mL}$ 。此标准溶液浓度为 $0.01 \mathrm{mg/mL}$ 。溶液转移至棕色玻璃瓶中后，在 $-20^{\circ} \mathrm{C}$ 电冰箱内保存，备用。  
4.14 黄曲霉毒素 $\mathrm{M}_{1}$ 标准系列溶液：吸取黄曲霉毒素 $\mathrm{M}_{1}$ 标准储备溶液（4.13） $10~\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 容量瓶中，用氮气将三氯甲烷吹至近干，空白基质溶液（4.12）定容至刻度，所得浓度为 $10~\mathrm{ng / mL}$ 的 $\mathrm{M}_{1}$ 标准中间溶液。再用空白基质溶液（4.12）将黄曲霉毒素 $\mathrm{M}_{1}$ 标准中间溶液稀释为 $0.5~\mathrm{ng / mL}$ 、 $0.8~\mathrm{ng / mL}$ 、 $1.0~\mathrm{ng / mL}$ 、 $2.0~\mathrm{ng / mL}$ 、 $4.0~\mathrm{ng / mL}$ 、 $6.0~\mathrm{ng / mL}$ 、 $8.0~\mathrm{ng / mL}$ 的系列标准工作液。

# 6 分析步骤

# 6.1 试液提取

6.1.1 乳：称取 $50 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ ) 混匀的试样，置于 $50 \mathrm{~mL}$ 具塞离心管 (5.7) 中，在水浴 (5.8) 中加热到 $35^{\circ} \mathrm{C} \sim 37^{\circ} \mathrm{C}$ 。在 6000 转/分钟下离心 $15 \mathrm{~min}$ 。收集全部上清液，供净化用。  
6.1.2 发酵乳(包括固体状、半固体状和带果肉型): 称取 $50 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ ) 混匀的试样, 用 $0.5 \mathrm{~mol} / \mathrm{L}$ 的氢氧化钠溶液(4.11)在酸度计(5.18)指示下调 $\mathrm{pH}$ 至7.4, 在 9500 转/分钟下匀浆 $(5.4) 5 \mathrm{~min}$ , 以下按 6.1.1 进行操作。  
6.1.3 乳粉和粉状婴幼儿配方食品：称取 $10 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ ) 试样，置于 $250 \mathrm{~mL}$ 烧杯中。将 $50 \mathrm{~mL}$ 已预热到 $50^{\circ} \mathrm{C}$ 的水加入到乳粉中，用玻璃棒将其混合均匀。如果乳粉仍未完全溶解，将烧杯置于 $50^{\circ} \mathrm{C}$ 的水浴(5.8)中放置 $30 \mathrm{~min}$ 。溶解后冷却至 $20^{\circ} \mathrm{C}$ ，移入 $100 \mathrm{~mL}$ 容量瓶中，用少量的水分次洗涤烧杯，洗涤液一并移入容量瓶中，用水定容至刻度，摇匀后分别移至两个 $50 \mathrm{~mL}$ 离心管(5.7)中，在6000转/分钟下离心 $15 \mathrm{~min}$ ，混合上清液，用移液管移取 $50 \mathrm{~mL}$ 上清液供净化处理用。  
6.1.4 干酪：称取经切细、过10目圆孔筛混匀的试样 $5 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ )，置于 $50 \mathrm{~mL}$ 离心管(5.7)中，加 $2 \mathrm{~mL}$ 水和 $30 \mathrm{~mL}$ 甲醇，在9500 转/分钟下匀浆 $5 \mathrm{~min}$ ，超声提取 $30 \mathrm{~min}$ ，在6000 转/分钟下离心 $15 \mathrm{~min}$ 。收集上清液并移入 $250 \mathrm{~mL}$ 分液漏斗中。在分液漏斗中加入 $30 \mathrm{~mL}$ 石油醚（4.3），振摇 $2 \mathrm{~min}$ ，待分层后，将下层移于 $50 \mathrm{~mL}$ 烧杯中，弃去石油醚层。重复用石油醚提取2次。将下层溶液移到 $100 \mathrm{~mL}$ 圆底烧瓶中，减压浓缩至约 $2 \mathrm{~mL}$ ，浓缩液倒入离心管中，烧瓶用乙腈-水溶液 $(1+4)$ （4.7） $5 \mathrm{~mL}$ 分2次洗涤，洗涤液一并倒入 $50 \mathrm{~mL}$ 离心管中，加水稀释至约 $50 \mathrm{~mL}$ ，在6000 转/分钟下离心 $5 \mathrm{~min}$ ，上清液供净化处理。  
6.1.5奶油：称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样，置于 $50~\mathrm{mL}$ 烧杯中，用 $20~\mathrm{mL}$ 石油醚(4.3)将其溶解并移于 $250\mathrm{mL}$ 具塞锥形瓶中。加 $20~\mathrm{mL}$ 水和 $30~\mathrm{mL}$ 甲醇，振荡 $30\mathrm{min}$ 后，将全部液体移于分液漏斗中，待分层后，将下层溶液全部移到 $100~\mathrm{mL}$ 圆底烧瓶中，在旋转蒸发仪（5.17）中减压浓缩至约 $5\mathrm{mL}$ ，加水稀释至约 $50~\mathrm{mL}$ ，供净化处理。

# 6.2 净化

# 6.2.1 免疫亲和柱的准备

将一次性的 $50~\mathrm{mL}$ 注射器筒与亲和柱(5.20)上顶部相串联，再将亲和柱与固相萃取装置连接起来。

注：根据免疫亲和柱的使用说明书要求，控制试液的pH值。

# 6.2.2 试样的纯化

将以上6.1试液提取液移至 $50 \mathrm{~mL}$ 注射器筒(5.21)中，调节固相萃取装置的真空系统,控制试样以 $2 \mathrm{~mL}$ /分钟 $\sim 3 \mathrm{~mL}$ /分钟稳定的流速过柱。取下 $50 \mathrm{~mL}$ 的注射器筒，装上 $10 \mathrm{~mL}$ 注射器筒。注射器筒内加入水，以稳定的流速洗柱，然后，抽干亲和柱。脱开真空系统，在亲和柱下部放入 $10 \mathrm{~mL}$ 刻度试管，上部装上另一个 $10 \mathrm{~mL}$ 注射器筒，加入 $4 \mathrm{~mL}$ 乙腈(4.2)，洗脱黄曲霉毒素 $\mathbf{M}_{1}$ ，洗脱液收集在刻度试管中(5.11)中，洗脱时间不少于60秒。然后用氮气缓缓地在 $30^{\circ} \mathrm{C}$ 下将洗脱液蒸发至近干（如果蒸发至干，会损失黄曲霉毒素 $\mathbf{M}_{1}$ ），用乙腈-水溶液（ $1+9$ ）稀释至 $1 \mathrm{~mL}$ 。

# 6.5 定性

试样中黄曲霉毒素 $\mathbf{M}_1$ 色谱峰的保留时间与相应标准色谱峰的保留时间相比较，变化范围应在 $\pm 2.5\%$ 之内。

黄曲霉毒素 $\mathbf{M}_{1}$ 的定性离子的重构离子色谱峰的信噪比应大于等于 $3(\mathrm{S} / \mathrm{N} \geqslant 3)$ ，定量离子的重构离子色谱峰的信噪比应大于等于 $10(\mathrm{S} / \mathrm{N} \geqslant 10)$ 。

# 6.6 试样测定

按照6.3和6.4确立的条件，测定试液（6.2）和标准系列溶液（4.14）中黄曲霉毒素 $\mathbf{M}_1$ 的离子强度，外标法定量。色谱图见附录A的图A.2。

色谱参考保留时间：黄曲霉毒素 $\mathrm{M}_{1} 3.23 \mathrm{~min}$ 。

# 6.7 空白试验

不称取试样，按6.5的步骤做空白实验。应确认不含有干扰被测组分的物质。

# 6.8 标准曲线绘制

将标准系列溶液（4.14）由低到高浓度进样检测，以峰面积-浓度作图，得到标准曲线回归方程。

# 6.9 定量测定

待测样液中被测组分的响应值应在标准曲线线性范围内，超过线性范围时，则应将样液用空白基质溶液稀释后重新进样分析或减少取样量，重新按6.1进行处理后再进样分析。

# 7 分析结果的表述

外标法定量，按式（1）计算黄曲霉毒素 $\mathbf{M}_{1}$ 的残留量。

$$
X = \frac {A \times V \times f \times 1}{m} \dots \tag {1}
$$

式中：

$X$ ——试样中黄曲霉毒素 $\mathrm{M}_{1}$ 的含量, 单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ;

A——试样中黄曲霉毒素 $\mathrm{M}_{1}$ 的浓度，单位为纳克每毫升（ $\mathrm{ng/mL}$ ）；

$V$ ——样品定容体积，单位毫升（ $\mathrm{mL}$ ）；

$f$ ——样液稀释因子；

$m$ ——试样的称样量，单位克（g）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 8 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 免疫亲和层析净化高效液相色谱法

# 9 原理

亲和柱内含有的黄曲霉毒素 $\mathbf{M}_{1}$ 特异性单克隆抗体交联在固体支持物上，当试样通过亲和柱时，抗体选择性的与黄曲霉毒素 $\mathbf{M}_{1}$ (抗原)键合，形成抗体一抗原复合体。用水洗柱除去柱内杂质，然后用洗脱剂洗脱吸附在柱上的黄曲霉毒素 $\mathbf{M}_{1}$ ，收集洗脱液。用带有荧光检测器的高效液相色谱仪测定洗脱液中黄曲霉毒素 $\mathbf{M}_{1}$ 含量。

# 10 试剂和材料

除非另有规定，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

10.1 免疫亲和柱：亲和柱的最大容量不小于 $100 \mathrm{ng}$ 黄曲霉毒素 $\mathrm{M}_{1}$ (相当于 $50 \mathrm{~mL}$ 浓度为 $2 \mu \mathrm{g} / \mathrm{L}$ 的试样), 当标准溶液含有 $4 \mathrm{ng}$ 黄曲霉毒素 $\mathrm{M}_{1}$ (相当于 $50 \mathrm{~mL}$ 浓度为 $80 \mathrm{ng} / \mathrm{L}$ 的试样) 时回收率不低于 $80 \%$ 。应该定期检查亲和柱的柱效和回收率, 对于每个批次的亲和柱至少检查一次 (见10.1.1和10.1.2)。

# 10.1.1 柱效检查

用移液管(11.4)移取 $1.0 \mathrm{~mL}$ 的黄曲霉毒素 $\mathrm{M}_{1}$ 标准储备液(10.5.2)到 $20 \mathrm{~mL}$ 的锥形试管中(11.9)。用恒流的氮气(10.3)将液体慢慢吹干，然后用 $10 \mathrm{~mL} 10 \%$ 的乙腈水溶液(10.2.2)溶解残渣，用力摇荡。

将该溶液加入到 $40 \mathrm{~mL}$ 的水中，充分混匀，全部通过免疫亲和柱（10.1）。按说明书要求使用免疫亲和柱。淋洗免疫亲和柱后，洗脱黄曲霉毒素 $\mathbf{M}_{1}$ 。将洗脱液进行适当稀释后，用高效液相色谱仪测定免疫亲和柱键合的黄曲霉毒素 $\mathbf{M}_{1}$ 含量。

计算黄曲霉毒素 $\mathbf{M}_1$ 的回收率，将其结果与 10.1 中所要求的指标进行比较。

# 10.1.2 回收率检查

# 10.5.1 浓度的校正

黄曲霉毒素 $\mathrm{M}_{1}$ 三氯甲烷标准溶液浓度为 $10\mu \mathrm{g / mL}$ 。根据下面的方法，在最大吸收波段测定溶液的吸光度，以确定黄曲霉毒素 $\mathrm{M}_{1}$ 的实际浓度。

用分光光度计（11.11）在 $340 \mathrm{~nm} \sim 370 \mathrm{~nm}$ 处测定，扣除三氯甲烷的空白本底，读取标准溶液的吸光度值。在接近 $360 \mathrm{~nm}$ 最大吸收波段 $\lambda_{\max}$ 处，测得吸光度值为 A，根据式（2）计算浓度值：

$$
c _ {i} = \frac {A \times M \times 1 0 0}{\varepsilon} \dots \tag {2}
$$

式中：

$c_{i}$ ——黄曲霉毒素 $\mathrm{M}_{1}$ 实际浓度, 单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ;

$A$ ——在 $\lambda_{\max}$ 处测得的吸光度值；

$M - 328 \mathrm{~g} / \mathrm{mol}$ , 黄曲霉毒素 $\mathrm{M}_{1}$ 摩尔质量, 单位为克每摩尔 (g/mol);

$\varepsilon$ ——19950，溶于三氯甲烷中的黄曲霉毒素 $\mathrm{M}_{1}$ 的吸光系数，单位为平方米每摩尔（ $\mathrm{m}^{2} / \mathrm{mol}$ ）。

# 10.5.2 标准储备液

# 第三法 免疫层析净化荧光分光度法

# 15 原理

试样经过离心、脱脂、过滤，滤液经含有黄曲霉毒素M<sub>1</sub>特异性单克隆抗体的免疫亲和柱层析净化，黄曲霉毒素M<sub>1</sub>交联在层析介质中的抗体上。此抗体对黄曲霉毒素M<sub>1</sub>具有专一性，当样品通过亲和柱时，抗体选择性的与所有存在的黄曲霉毒素M<sub>1</sub>(抗原)键合。用甲醇-水(1+9)将免疫亲和柱上杂质除去，以甲醇-水(8+2)通过免疫亲和柱洗脱，将溴溶液衍生后的洗脱液置于荧光光度计中测定黄曲霉毒素M<sub>1</sub>含量。

# 第四法 双流向酶联免疫法

# 20 原理

利用酶联免疫竞争原理，样品中残留的黄曲霉毒素 $\mathbf{M}_{1}$ 与定量特异性酶标抗体反应，多余的游离酶标抗体则与酶标板内的包被抗原结合，通过流动洗涤，加入酶显色底物显色后，与标准点比较定性。

# 21 试剂和材料

除非另有规定，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

21.1 黄曲霉毒素 $\mathrm{M}_{1}$ 双流向酶联免疫试剂盒， $2^{\circ} \mathrm{C} \sim 7^{\circ} \mathrm{C}$ 保存。

21.1.1 黄曲霉毒素 $\mathbf{M}_{1}$ 系列标准溶液。

21.1.2 酶联免疫试剂颗粒（含特异性酶标抗体）。

21.1.2.1 抗黄曲霉毒素 $\mathbf{M}_{1}$ 抗体。

警告：不应破损，否则立即销毁。

21.1.2.2 酶结合物。

21.1.3 酶显色底物。

# 22 仪器和设备

22.1 样品试管，带有密封盖，内置酶联免疫试剂颗粒（21.1.2）。

22.2 移液器（管）， $450~\mu \mathrm{L}\pm 50~\mu \mathrm{L}$   
22.3 酶联免疫检测加热器， $40^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 。  
22.4 双流向酶联免疫检测读数仪。

# 23 分析步骤

23.1加热器（22.3）预热到 $45^{\circ}C\pm 5^{\circ}C$ ，并至少保持 $15\mathrm{min}$   
23.2 液体试样或乳粉试样复原后振摇混匀，移取 $450 \mu \mathrm{L}$ 至样品试管（22.1）中，充分振摇，使其中的酶联免疫试剂颗粒（21.1.2）完全溶解。  
23.3 将样品试管（22.1）和酶联免疫检测试剂盒（21.1）同时置于预热过的加热器内保温，保温时间 $5 \mathrm{~min} \sim 6 \mathrm{~min}$ 。使试样中的黄曲霉毒素 $\mathrm{M}_{1}$ 和酶联免疫试剂颗粒中的酶标记黄曲霉毒素 $\mathrm{M}_{1}$ 抗体结合。  
23.4 将样品试管内的全部内容物倒入试剂盒（21.1）的样品池中，样品将流经“结果显示窗口”向绿色的激活环流去。  
23.5 当激活环的绿色开始消失变为白色时，立即用力按下激活环按键。  
23.6 试剂盒（21.1）继续放置在加热器（22.3）中保温保持 $4\mathrm{min}$ ，使显色反应完成。  
23.7 将试剂盒从加热器中取出水平放置，立即进行检测结果判读，结果判读应在 $1\mathrm{min}$ 内完成。

# 24 分析结果的表述

24.1 目测判读结果

试样点的颜色深于质控点，或两者颜色相当，检测结果为阴性。

试样点的颜色浅于质控点，检测结果为阳性。

24.2 双流向酶联免疫检测读数仪判读结果

数值 $\leqslant 1.05$ ，显示Negative，检测结果为阴性。

数值 $>1.05$ ，显示Positive，检测结果为阳性。

注：阳性样品需用第一法定量检测方法进一步确认。

# 25 其他

本标准第一法的定量限为 $0.01\mu \mathrm{g / kg}$ (以乳计)；第二法乳粉中黄曲霉毒素 $\mathbf{M}_1$ 的最低检测限为0.08 $\mu \mathrm{g / kg}$ ，乳中黄曲霉毒素 $\mathbf{M}_1$ 的最低检测限为 $0.008\mu \mathrm{g / L}$ ；第三法乳中的黄曲霉毒素 $\mathbf{M}_1$ 的检出限为0.1 $\mu \mathrm{g / L}$ ，乳粉中黄曲霉毒素 $\mathbf{M}_1$ 的检出限为 $0.1\mu \mathrm{g / kg}$ ；第四法的检出限为 $0.5\mu \mathrm{g / kg}$ 。

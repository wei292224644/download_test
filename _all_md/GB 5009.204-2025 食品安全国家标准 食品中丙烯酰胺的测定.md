# 前言

本标准代替GB5009.204—2014《食品安全国家标准 食品中丙烯酰胺的测定》。

本标准与GB5009.204—2014相比，主要变化如下：

——修改了标准的适用范围；

——第一法中增加了“基质分散固相萃取法”试样净化方法；

——删除了第一法“离子阱串联质谱仪”章节。

# 食品安全国家标准

# 食品中丙烯酰胺的测定

# 1 范围

本标准规定了食品中丙烯酰胺的测定方法。

本标准第一法液相色谱-质谱/质谱法适用于食品中丙烯酰胺的测定。

本标准第二法气相色谱-质谱法适用于除咖啡、茶叶、食糖、婴幼儿配方食品和婴幼儿辅助食品外的食品中丙烯酰胺的测定。

# 第一法 液相色谱-质谱/质谱法

# 2 原理

试样中的丙烯酰胺用水提取，经过基质分散固相萃取法或固相萃取柱法或层析柱法净化后，以液相色谱-质谱/质谱仪检测，内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲酸（HCOOH）：色谱纯。  
3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.3 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.4 正己烷 $(n - C_6H_{14})$ 。  
3.1.5 乙酸乙酯 $\mathrm{(CH_3COOC_2H_5)}$   
3.1.6 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.7 硫酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{SO}_{4}\right]$ 。  
3.1.8 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.9 干冰 $(\mathrm{CO}_{2})$ 。

# 3.2 试剂配制

3.2.1 饱和硫酸铵溶液：称取 $80\mathrm{g}$ 硫酸铵晶体，加入水 $100\mathrm{mL}$ ，超声溶解，室温放置。临用现配。  
3.2.2 甲酸-水溶液 $(0.1\%)$ ：准确移取 $1\mathrm{mL}$ 甲酸，用水稀释至 $1000~\mathrm{mL}$ ，混合均匀。临用现配。

# 3.3 标准品

3.3.1丙烯酰胺 $\mathrm{(C_3H_5NO}$ ，CAS号：79-06-1)标准品：纯度 $\geqslant 99\%$ ，或经国家认证授予标准物质证书的标准品。

3.3.2 $^{13}\mathrm{C}_{3}$ -丙烯酰胺 $\left({}^{13}\mathrm{C}_{3}\mathrm{H}_{5}\mathrm{NO},\right.$ CAS号：287399-26-2）标准品：纯度 $\geq 98\%$ 。

# 3.4 标准溶液配制

# 3.4.1 丙烯酰胺标准溶液配制

3.4.1.1 丙烯酰胺标准储备液 $(1000\mathrm{mg / L})$ ：准确称取丙烯酰胺标准品 $10.0\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ )，用甲醇溶解并定容至 $10~\mathrm{mL}$ ，混匀，使丙烯酰胺质量浓度为 $1000\mathrm{mg / L}$ 。将溶液转移至棕色玻璃具塞密闭容器中，置于 $-18^{\circ}C$ 冰箱中避光保存。有效期为6个月。  
3.4.1.2 丙烯酰胺中间液 $(100\mathrm{mg / L})$ ：移取丙烯酰胺标准储备液 $(1000\mathrm{mg / L})1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，加甲醇稀释并定容至刻度线，混匀，使丙烯酰胺质量浓度为 $100\mathrm{mg / L}$ 。将溶液转移至棕色玻璃具塞密闭容器中，置于一 $18^{\circ}C$ 冰箱中避光保存。有效期为3个月。  
3.4.1.3 丙烯酰胺工作液 $\mathrm{I}(10\mathrm{mg / L})$ ：移取丙烯酰胺中间液 $(100\mathrm{mg / L})1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用水稀释并定容至刻度线，混匀，使丙烯酰胺质量浓度为 $10\mathrm{mg / L}$ 。临用现配。  
3.4.1.4 丙烯酰胺工作液Ⅱ（ $1\mathrm{mg / L}$ ）：移取丙烯酰胺工作液I（ $10\mathrm{mg / L}$ ） $1\mathrm{mL}$ 于 $10\mathrm{mL}$ 容量瓶中，用水稀释并定容至刻度线，混匀，使丙烯酰胺质量浓度为 $1\mathrm{mg / L}$ 。临用现配。

# 3.4.2 $^{13}\mathrm{C}_3$ -丙烯酰胺内标溶液配制

3.4.2.1 ${}^{13}\mathrm{C}_{3}$ -丙烯酰胺储备液( $1000\mathrm{mg / L})$ ：准确称取 ${}^{13}\mathrm{C}_{3}$ -丙烯酰胺标准品 $10.0\mathrm{mg}$ （精确至 $0.01\mathrm{mg})$ ，用甲醇溶解并定容至 $10~\mathrm{mL}$ ，混匀，使 ${}^{13}\mathrm{C}_{3}$ -丙烯酰胺质量浓度为 $1000\mathrm{mg / L}$ 。将溶液转移至棕色玻璃具塞密闭容器中，置于一 $18^{\circ}C$ 冰箱中避光保存。有效期为6个月。  
3.4.2.2 ${}^{13}\mathrm{C}_{3}$ -丙烯酰胺工作液 $(10\mathrm{mg / L})$ ：移取 ${}^{13}\mathrm{C}_{3}$ -丙烯酰胺内标储备液 $(1000\mathrm{mg / L})1\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用水稀释并定容至刻度线，混匀，使 ${}^{13}\mathrm{C}_{3}$ -丙烯酰胺内标质量浓度为 $10\mathrm{mg / L}$ 。临用现配。

# 3.4.3 标准系列工作液

分别移取丙烯酰胺工作液 $\mathrm{II}(1\mathrm{mg / L})0.10\mathrm{mL},0.50\mathrm{mL},1.00\mathrm{mL}$ 和丙烯酰胺工作液I（ $10\mathrm{mg / L})$ $0.20~\mathrm{mL},0.50~\mathrm{mL},1.00~\mathrm{mL}$ 至 $10  \mathrm{mL}$ 容量瓶中，加入 ${}^{13}\mathrm{C}_{3}$ -丙烯酰胺内标工作液 $(10\mathrm{mg / L})0.10\mathrm{mL}$ ，用水稀释至刻度线，混匀。标准系列溶液中丙烯酰胺的质量浓度分别为 $10~\mu \mathrm{g / L},50~\mu \mathrm{g / L},100~\mu \mathrm{g / L},$ $200~\mu \mathrm{g / L},500~\mu \mathrm{g / L}$ 和 $1000~\mu \mathrm{g / L},{}^{13}\mathrm{C}_3$ -丙烯酰胺内标浓度为 $100~\mu \mathrm{g / L}$ 。可根据实际样品溶液的浓度适当调整标准系列工作液的浓度。临用现配。

# 3.5 材料

3.5.1 颗粒硅藻土：ExtrelutTM20或相当产品。  
3.5.2 基质分散净化管：含 $20\mathrm{mg}$ 石墨化炭黑、 $150\mathrm{mg}$ 十八烷基吸附剂、 $150\mathrm{mg}$ 乙二胺- $N$ -丙基硅烷、 $200\mathrm{mg}$ 强阳离子交换材料和 $150\mathrm{mg}$ 硫酸镁，或等效产品。  
3.5.3 亲水亲油平衡固相萃取柱：填料为二乙烯苯和 $N$ -乙烯基吡咯烷酮共聚物，柱容量为 $6 \mathrm{~mL}$ ，填料量为 $200 \mathrm{mg}$ ，或等效产品。  
3.5.4 混合硅胶键合固相萃取柱：填料为强阳离子交换和强阴离子交换混合键合，柱容量为 $3\mathrm{mL}$ ，填料量为 $200\mathrm{mg}$ ，或等效产品。  
3.5.5 玻璃层析柱：柱长 $30~\mathrm{cm}$ ，柱内径 $1.8~\mathrm{cm}$   
3.5.6 水系微孔滤膜： $0.22\mu \mathrm{m}$ 和 $0.45\mu \mathrm{m}$ 。

# 4 仪器和设备

4.1 液相色谱-质谱/质谱仪(LC-MS/MS)，配电喷雾离子源。

4.2 组织粉碎机。  
4.3 旋转蒸发仪。  
4.4 氮吹仪。  
4.5 振荡器。  
4.6 涡旋混合器。  
4.7 电子天平：感量为 $0.01\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.8 离心机：转速 $\geqslant 10000\mathrm{r / min}$

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 焙烤食品、膨化食品、油炸谷物食品

取 $100\mathrm{g}$ 试样，加入 $100\mathrm{g}$ 干冰后，经组织粉碎机粉碎均匀，待干冰挥发完全后，于 $-18^{\circ}C$ 冷冻保存。

注：干冰温度极低，应使用厚棉手套或其他遮蔽物(如夹子)来取干冰；干冰升华时会释放出二氧化碳导致组织粉碎机内部压力上升，应注意其承压能力，防止腔体因压力过高爆破，并确保环境通风良好。

# 5.1.2 咖啡、茶叶、食糖、婴幼儿配方食品和婴幼儿辅助食品

取 $100\mathrm{g}$ 试样，经组织粉碎机粉碎均匀(速溶咖啡和婴幼儿乳粉直接取样)，于 $-18^{\circ}C$ 冷冻保存。

注：在粉碎过程中，防止样品被过度加热而影响测定结果。

# 5.2 试样提取

准确称取试样 $0.5\mathrm{g}\sim 2\mathrm{g}$ （精确到 $0.001\mathrm{g})$ ，加入 $10\mathrm{mg / L}^{13}\mathrm{C}_3$ -丙烯酰胺内标工作溶液 $10~\mu \mathrm{L}$ （层析柱法)或 $20~\mu \mathrm{L}$ (基质分散固相萃取法和固相萃取柱法)，再加入水 $10~\mathrm{mL}$ ，振摇 $30\mathrm{min}$ 后，以 $4000\mathrm{r}/$ min离心 $10\mathrm{min}$ ，取上清液待净化。如无法通过离心获得上清液，如速溶咖啡、婴幼儿乳粉等基质样品，则取混悬液待净化。

# 5.3 试样净化

# 5.3.1 基质分散固相萃取法（适用于所有食品）

在试样提取的上清液中，加入 $5\mathrm{mL}$ 正己烷，振荡萃取 $5\mathrm{min}$ ，于 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，除去有机相，再用 $5\mathrm{mL}$ 正己烷重复萃取一次，在提取液加入 $10\mathrm{mL}$ 乙腈，振荡提取 $10\mathrm{min}$ ，加入 $0.5\mathrm{g}$ 氯化钠和 $4\mathrm{g}$ 无水硫酸钠作为盐析剂，振荡 $5\mathrm{min}$ 。以 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取 $2.00\mathrm{mL}$ 上清液于基质分散净化管中，涡旋 $5\mathrm{min}$ ，以 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取 $1\mathrm{mL}$ 上清液于氮吹仪中浓缩至近干，用 $0.20\mathrm{mL}$ 水复溶，经 $0.22\mu \mathrm{m}$ 水系滤膜过滤，待测定。

# 5.3.2 其他净化法（适用于除咖啡、茶叶、食糖、婴幼儿配方食品和婴幼儿辅助食品外的食品）

# 5.3.2.1 固相萃取柱法

在试样提取的上清液中加入 $5\mathrm{mL}$ 正己烷，振荡萃取 $10\mathrm{min}$ ，于 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，除去有机相，再用 $5\mathrm{mL}$ 正己烷重复萃取一次，取提取液 $6\mathrm{mL}$ 经 $0.45\mu \mathrm{m}$ 水系滤膜过滤，收集滤液待进行亲水亲油平衡固相萃取柱净化处理。亲水亲油平衡固相萃取柱使用前依次用 $3\mathrm{mL}$ 甲醇、 $3\mathrm{mL}$ 水活化。取上述滤液 $5.00\mathrm{mL}$ ，加在亲水亲油平衡固相萃取柱上，收集流出液，并用 $4\mathrm{mL}80\%$ 的甲醇水溶液洗脱，收

集全部洗脱液, 并与流出液合并, 待进行混合键合固相萃取柱净化; 混合键合固相萃取柱依次用 $3 \mathrm{~mL}$ 甲醇、 $3 \mathrm{~mL}$ 水活化, 将亲水亲油平衡固相萃取柱净化的全部洗脱液上样, 在重力作用下流出, 收集全部流出液, 在氮气下将流出液浓缩至近干, 用水定容至 $1.00 \mathrm{~mL}$ , 经 $0.22 \mu \mathrm{m}$ 水系滤膜过滤, 待测定。

# 5.3.2.2 层析柱法

在试样提取的上清液中，加入硫酸铵 $15\mathrm{g}$ ，振荡 $10\mathrm{min}$ ，使其充分溶解，于 $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液 $10\mathrm{mL}$ ，备用。如上清液不足 $10\mathrm{mL}$ ，则用饱和硫酸铵溶液补足。取洁净玻璃层析柱，在底部填少许玻璃棉并压紧，依次填装 $10\mathrm{g}$ 无水硫酸钠、 $2\mathrm{g}$ 硅藻土。称取 $5\mathrm{g}$ 硅藻土与上述试样上清液搅拌均匀后，装入层析柱中。用 $70\mathrm{mL}$ 正己烷淋洗，控制流速 $2\mathrm{mL / min}$ ，弃去正己烷淋洗液。用 $70\mathrm{mL}$ 乙酸乙酯洗脱，控制流速为 $2\mathrm{mL / min}$ ，收集乙酸乙酯洗脱溶液，并在 $45^{\circ}\mathrm{C}$ 水浴中减压旋转蒸发至近干，用乙酸乙酯洗涤蒸发瓶残渣3次（每次 $1\mathrm{mL}$ ），并将其转移至已加入 $1.00\mathrm{mL}$ 水的试管中，涡旋振荡。在氮气流下吹去上层有机相后，加入 $1\mathrm{mL}$ 正己烷，涡旋振荡，于 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取下层水相，经 $0.22\mu \mathrm{m}$ 水系滤膜过滤，待测定。

# 5.5 标准曲线的制作

将标准系列工作溶液分别注入液相色谱-质谱/质谱系统，测定相应的丙烯酰胺及其内标的峰面积，以各标准系列工作溶液中丙烯酰胺质量浓度 $(\mu \mathrm{g} / \mathrm{L})$ 为横坐标，以丙烯酰胺及其 ${}^{13}\mathrm{C}_3$ -丙烯酰胺内标定量离子质量色谱图上测得的峰面积比为纵坐标，绘制标准曲线。丙烯酰胺标准溶液的质量色谱图参见附录A的图A.1。

# 5.6 定性测定

按照5.4仪器条件测定标准溶液和试样溶液，被测试样中目标化合物色谱峰的保留时间与相应标准溶液色谱峰的保留时间相比较，变化范围应在 $\pm 2.5\%$ 之内。目标化合物质谱定量和定性离子均出现，且同一检测批次，对同一化合物，样品中目标化合物的定性离子和定量离子的相对丰度比与质量浓度相当的标准溶液相比，其允许偏差不超过表3规定的范围，则可判断试样中存在对应的被测物。

# 5.7 定量测定

将试样溶液注入液相色谱-质谱/质谱系统中，测得丙烯酰胺 $(m / z72 > 55)$ 和 $^{13}\mathrm{C}_3$ -丙烯酰胺内标

$(m / z72 > 58)$ 的峰面积比，根据标准曲线得到待测液中丙烯酰胺质量浓度 $(\mu \mathrm{g} / \mathrm{L})$ 。待测样液中目标物的响应值应在标准曲线范围内，超过线性范围则应适当调整取样量重新测定或适当调整标准曲线的线性范围。

# 5.8 空白试验

除不称取试样外，均按上述测定步骤进行。如空白试验上机溶液中目标物质量浓度高于方法定量限对应浓度，应更换试剂或实验器皿，直至本底值低于定量限。

试样中丙烯酰胺含量按式(1)计算：

$$
X = \frac {\rho \times V _ {1} \times V _ {3} \times 1 0 0 0}{m \times V _ {2} \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中丙烯酰胺的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——试样中丙烯酰胺 $(m / z72 > 55)$ 色谱峰与 ${}^{13}\mathrm{C}_3$ -丙烯酰胺内标 $(m / z72 > 58)$ 色谱峰的峰面积比值对应的丙烯酰胺质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V_{1}$ ——试样提取液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——样品经净化后的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——单位换算系数；

$m$ ——取样量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——用于净化移取的试样溶液体积，单位为毫升 $(\mathrm{mL})$ 。

当结果 $\leqslant 100$ 时，保留3位有效数字；当结果 $>100$ 时，保留至小数点后1位。

注：基质分散固相萃取法试样提取液体积 $V_{1}$ 以乙腈加入量 $10\mathrm{mL}$ 计，用于净化移取的试样溶液体积 $V_{2}$ 以 $1\mathrm{mL}$ 计。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当取样量为 $1\mathrm{g}$ 时，本方法的检出限为 $3\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$ 。

# 第二法 气相色谱-质谱法

# 9 原理

试样中的丙烯酰胺用水提取，经填充柱法净化、溴试剂衍生后，以气相色谱-质谱仪检测，内标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 正己烷 $(n - C_6H_{14})$ 。  
10.1.2 乙酸乙酯 $\mathrm{(CH_3COOC_2H_5)}$   
10.1.3 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $400^{\circ}C$ ，烘烤 $4\mathrm{h}$   
10.1.4 硫酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{SO}_{4}\right]$ 。  
10.1.5 硫代硫酸钠 $\mathrm{(Na_2S_2O_3\cdot 5H_2O)}$   
10.1.6 溴 $(\mathrm{Br}_2)$ 。  
10.1.7 氢溴酸（HBr）：含量 $>48.0\%$   
10.1.8 溴化钾(KBr)。

# 10.2 试剂配制

10.2.1 饱和溴水：量取 $100 \mathrm{~mL}$ 水，置于 $200 \mathrm{~mL}$ 的棕色试剂瓶中，加入 $8 \mathrm{~mL}$ 溴， $4^{\circ} \mathrm{C}$ 避光放置 $8 \mathrm{~h}$ ，上层为饱和溴水溶液。临用现配。  
10.2.2 溴试剂：称取溴化钾 $20.0\mathrm{g}$ ，加水 $50\mathrm{mL}$ ，使完全溶解，再加入 $1.0\mathrm{mL}$ 氢溴酸和 $16.0\mathrm{mL}$ 饱和溴水，摇匀，用水稀释至 $100\mathrm{mL},4^{\circ}\mathrm{C}$ 避光保存。临用现配。  
10.2.3 硫代硫酸钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取硫代硫酸钠 $2.48\mathrm{g}$ ，加水 $50~\mathrm{mL}$ ，使完全溶解，用水稀释至 $100~\mathrm{mL},4^{\circ}\mathrm{C}$ 避光保存。临用现配。  
10.2.4 饱和硫酸铵溶液：称取 $80\mathrm{g}$ 硫酸铵晶体，加入水 $100\mathrm{mL}$ ，超声溶解，室温放置。临用现配。

# 10.3 标准品

10.3.1 丙烯酰胺 $\left(\mathrm{C}_{3} \mathrm{H}_{5} \mathrm{NO}, \mathrm{CAS}\right.$ 号：79-06-1)标准品：纯度 $\geqslant 99\%$ ，或经国家认证授予标准物质证书的标准品。  
10.3.2 $^{13}\mathrm{C}_{3}$ -丙烯酰胺 $(^{13}\mathrm{C}_3\mathrm{H}_5\mathrm{NO}$ ,CAS号：287399-26-2)标准品：纯度 $\geq 98\%$ 。

# 10.4 标准溶液配制

10.4.1 丙烯酰胺溶液：同3.4.1。  
10.4.2 丙烯酰胺内标溶液：同3.4.2。  
10.4.3 标准系列工作液: 分别准确移取丙烯酰胺工作液 $\mathrm{II}(1\mathrm{mg} / \mathrm{L}) 0.10\mathrm{mL}, 0.50\mathrm{mL}, 2.00\mathrm{mL}$ 和丙烯酰胺工作液 $\mathrm{I}(10\mathrm{mg} / \mathrm{L}) 0.50\mathrm{mL}, 1.00\mathrm{mL}$ 至 $10\mathrm{mL}$ 容量瓶中, 加入 ${ }^{13} \mathrm{C}_{3}$ -丙烯酰胺内标使用液 $(10\mathrm{mg} / \mathrm{L}) 0.10\mathrm{mL}$ , 用水稀释至刻度线, 混匀。标准系列溶液中丙烯酰胺质量浓度分别为 $10\mu \mathrm{g} / \mathrm{L}$ 、 $50\mu \mathrm{g} / \mathrm{L}, 200\mu \mathrm{g} / \mathrm{L}, 500\mu \mathrm{g} / \mathrm{L}$ 和 $1000\mu \mathrm{g} / \mathrm{L}, {}^{13} \mathrm{C}_{3}$ -丙烯酰胺内标浓度为 $100\mu \mathrm{g} / \mathrm{L}$ 。可根据实际样品溶液的浓度适当调整标准系列工作液的浓度。临用现配。

# 10.5 材料

10.5.1 颗粒硅藻土： $\mathrm{Extrelut}^{\mathrm{TM}}$ 20或相当产品。  
10.5.2 玻璃层析柱：柱长 $30~\mathrm{cm}$ ，柱内径 $1.8~\mathrm{cm}$ 。

11.1 气相色谱-四极杆质谱联用仪（GC-MS）。

11.3 旋转蒸发仪。

11.4 氮吹仪。

# 12 分析步骤

# 12.1 试样制备

同5.1。

# 12.2 试样提取

准确称取试样 $2\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ），加入 $10\mathrm{mg / L}^{13}\mathrm{C}_3$ -丙烯酰胺内标溶液 $10~\mu \mathrm{L}$ ，再加入水 $10~\mathrm{mL}$ ，振荡 $30\mathrm{min}$ 后，于 $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液备用。

# 12.3 试样净化

在试样提取的上清液中，加入硫酸铵 $15\mathrm{g}$ ，振荡 $10\mathrm{min}$ ，使其充分溶解，于 $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液 $10\mathrm{mL}$ ，备用。如上清液不足 $10\mathrm{mL}$ ，则用饱和硫酸铵溶液补足。取洁净玻璃层析柱，在底部填少许玻璃棉并压紧，依次填装 $10\mathrm{g}$ 无水硫酸钠、 $2\mathrm{g}$ 硅藻土。称取 $5\mathrm{g}$ 硅藻土与上述试样上清液搅拌均匀后，装入层析柱中。用 $70\mathrm{mL}$ 正己烷淋洗，控制流速 $2\mathrm{mL / min}$ ，弃去正己烷淋洗液。用 $70\mathrm{mL}$ 乙酸乙酯洗脱，控制流速为 $2\mathrm{mL / min}$ ，收集乙酸乙酯洗脱溶液，并在 $45^{\circ}\mathrm{C}$ 水浴中减压旋转蒸发至近干，用乙酸乙酯洗涤蒸发瓶残渣3次（每次 $1\mathrm{mL}$ ），并将其转移至已加入 $1.00\mathrm{mL}$ 水的试管中，涡旋振荡。在氮气流下吹去上层有机相后，加入 $1\mathrm{mL}$ 正己烷，涡旋振荡，于 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取下层水相备用衍生。

# 12.4 衍生

# 12.4.1 试样的衍生

试样的衍生：在试样提取液中加入溴试剂 $1\mathrm{mL}$ ，涡旋振荡， $4^{\circ}C$ 放置至少 $1\textrm{h}$ 后，加入 $0.1\mathrm{mol / L}$ 硫代硫酸钠溶液约 $100~\mu \mathrm{L}$ ，涡旋振荡除去剩余的衍生剂；加入 $2\mathrm{mL}$ 乙酸乙酯，涡旋振荡 $1\mathrm{min}$ ，于 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，吸取上层有机相转移至加有 $0.1\mathrm{g}$ 无水硫酸钠的试管中，加入乙酸乙酯 $2\mathrm{mL}$ 重复萃取，合并有机相；静置至少 $0.5\mathrm{h}$ ，转移至另一试管，在氮气流下吹至近干，加 $0.1\mathrm{mL}$ 乙酸乙酯溶解残渣，待测定。

# 12.4.2 标准工作溶液的衍生

量取标准系列溶液各 $1.0~\mathrm{mL}$ ，按照上述试样衍生方法同步操作。

# 12.5 仪器参考条件

# 12.5.1 色谱条件

a）色谱柱： $5\%$ 苯基-甲基聚硅氧烷的弱极性毛细管柱（柱长 $30~\mathrm{m}$ ，内径 $0.25  \mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m})$ 或等效柱。  
b）进样口温度： $120^{\circ}\mathrm{C}$ 保持 $2\mathrm{min}$ ，以 $40^{\circ}\mathrm{C / min}$ 速度升至 $240^{\circ}\mathrm{C}$ ，并保持 $5\mathrm{min}$ 。  
c）色谱柱程序温度： $65~\mathrm{^\circ C}$ 保持 $1\mathrm{min}$ ，以 $15^{\circ}C / \mathrm{min}$ 速度升至 $200^{\circ}C$ ，再以 $40^{\circ}C / \mathrm{min}$ 的速度升

至 $240^{\circ} \mathrm{C}$ , 并保持 $5 \mathrm{~min}$ 。

d) 载气：高纯氦气(纯度 $>99.999\%$ )，柱前压为 $69 \mathrm{~mPa}$ ，相当于 $10 \mathrm{psi}$ 。  
e）不分流进样，进样体积 $1\mu \mathrm{L}$

c）传输线温度： $250~\mathrm{^\circ C}$ □  
d）离子源温度： $200^{\circ}C$ 。  
e）溶剂延迟： $6\mathrm{min}$

# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T0529—2013《出口肉品中甲氧滴滴涕残留量检验方法 气相色谱/质谱法》。与SN/T0529—2013相比，主要变化如下：

标准文本格式修改为食品安全标准文本格式；  
标准名称和范围中“出口肉品”改为“肉品”；  
标准范围中增加“其他食品可参照执行”。

# 食品安全国家标准

# 肉品中甲氧滴滴涕残留量的测定 气相色谱一质谱法

# 1 范围

本标准规定了肉品中甲氧滴滴涕残留量的气相色谱—质谱检测方法。

本标准适用于鸡肉、鸭肉、猪肉中甲氧滴滴涕残留量的测定，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

样品中的甲氧滴滴涕经乙酸乙酯一环己烷(1+1,体积比)提取，提取液浓缩后经凝胶渗透色谱(GPC)系统净化，用乙酸乙酯一环己烷(1+1,体积比)洗脱，洗脱液浓缩至干，定容后，气相色谱一质谱仪选择离子检测，外标法定量。

# 4 试剂和材料

除非另有说明，所有试剂均为农残级，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ 。

4.1.2 环己烷 $(\mathrm{C_6H_{12}})$ 。

4.1.3 正己烷 $(\mathrm{C_6H_{14}})$ 。

4.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：分析纯，用前在 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，储存于干燥器中，冷却后备用。

# 4.2 溶液配制

乙酸乙酯一环己烷溶液 $(1 + 1$ ，体积比)：取 $200~\mathrm{mL}$ 乙酸乙酯，加入 $200~\mathrm{mL}$ 环己烷，摇匀备用。

# 4.3 标准品

甲氧滴滴涕标准品（Methoxychlor, $\mathrm{C_{16}H_{15}Cl_3O_2}$ ，相对分子质量：345.66，CAS号：72-43-5）：纯度 $\geqslant 99\%$ 。

# 4.4 标准溶液配制

# 4.4.1 甲氧滴滴涕标准储备溶液

准确称取适量的甲氧滴滴涕标准品，用正己烷配制成 $100\mu \mathrm{g / mL}$ 的标准储备溶液。避光于 $0^{\circ}\mathrm{C}\sim$ $4^{\circ}\mathrm{C}$ 保存，保存期为6个月。

# 4.4.2 甲氧滴滴滴涕标准工作溶液

根据检测要求，吸取适量上述标准储备溶液于容量瓶中，用正己烷稀释配成适当质量浓度的标准工作溶液。避光于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存，保存期为1个月。

# 5 仪器和设备

5.1 气相色谱—质谱仪：配备电子轰击源（EI）。  
5.2 组织捣碎机。  
5.3 均质器。  
5.4 旋转蒸发仪。  
5.5 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.6 离心机。  
5.7 凝胶渗透色谱系统(GPC)：内装BiobeadsS-X3填料的净化柱。  
5.8 涡旋振荡器。

# 6 样品制备与保存

# 6.1 试样制备

从原始样品中取出部分有代表性样品, 取样部位按 GB 2763 的规定执行, 放入高速组织捣碎机中捣碎均匀。充分混匀, 用四分法缩分出不少于 $500 \mathrm{~g}$ 作为试样。装入清洁的容器中, 加封后标明标记。

# 6.2 试样保存

将试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

在试样制备与保存的操作过程中,应防止样品受到污染或发生残留物含量的变化。

# 7 分析步骤

# 7.1 提取

称取 $10 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ ) 样品于 $100 \mathrm{~mL}$ 具塞离心管中, 加入 $20 \mathrm{~g}$ 无水硫酸钠, 加入乙酸乙酯一环己烷 $(1+1$ , 体积比) $40 \mathrm{~mL}$ , 均质 $2 \mathrm{~min}, 4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 上清液通过装有无水硫酸钠的漏斗, 收集于 $100 \mathrm{~mL}$ 旋蒸瓶中, 残渣中再加入 $20 \mathrm{~mL}$ 乙酸乙酯一环己烷 $(1+1$ , 体积比)。按上述步骤提取一次, 合并提取液于 $40^{\circ} \mathrm{C}$ 下旋转蒸发浓缩至近干。

# 7.2 净化

# 7.2.1 凝胶渗透色谱条件

凝胶渗透色谱条件如下：

a）净化柱： $200\mathrm{mm}\times 25\mathrm{mm}$ ，内装Bio-BeadsS-X3填料；  
b）流动相：乙酸乙酯一环己烷混合溶剂(1+1，体积比)；  
c) 流速: $4.7 \mathrm{~mL} / \mathrm{min}$ ;   
d) 进样量: $5 \mathrm{~mL}$ ;   
e) 开始收集时间：9.5min;  
f) 结束收集时间： $13\mathrm{min}$

# 7.2.2 凝胶渗透色谱净化

用 $10 \mathrm{~mL}$ 乙酸乙酯一环己烷 $(1+1$ , 体积比)溶解残渣并洗涤, 转移入凝胶渗透色谱进样瓶中, 充分混匀。如有颗粒状物质存在, 则离心或过滤。 $5 \mathrm{~mL}$ 样液注入经校准的GPC进样管, 通过凝胶渗透色谱系统进行净化。用乙酸乙酯一环己烷溶液 $(1+1$ , 体积比)洗脱, 洗脱液收集于 $100 \mathrm{~mL}$ 梨形烧瓶中。于 $40^{\circ} \mathrm{C}$ 以下旋转蒸发至近干。用正己烷(或其他合适的溶剂)将其溶解并定容至 $1.0 \mathrm{~mL}$ 。

# 7.3 测定

# 7.3.1 气相色谱一质谱参考条件

a）色谱柱：DB-35MS $(30\mathrm{m}\times 0.25\mathrm{mm}\times 0.25\mu \mathrm{m})$ 石英毛细管柱或相当者。  
b）色谱柱温度： $80^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，然后以 $20^{\circ}\mathrm{C} / \mathrm{min}$ 程序升温至 $180^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ ，再以 $20^{\circ}\mathrm{C} / \mathrm{min}$ 程序升温至 $230^{\circ}\mathrm{C}$ ，保持 $7\mathrm{min}$ ，以 $10^{\circ}\mathrm{C} / \mathrm{min}$ 程序升温至 $290^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ 。  
c）进样口温度： $260^{\circ}C$   
d）进样方式：不分流进样。  
e）进样量： $1\mu \mathrm{L}$   
f) 载气：氦气，纯度 $\geqslant 99.999\%$ ，流速 $1.0~\mathrm{mL / min}$   
g） 电子轰击源： $70\mathrm{eV}$   
h) 离子源温度: $230^{\circ} \mathrm{C}$ ;   
i）GC-MS接口温度： $280^{\circ}\mathrm{C}$   
j）选择离子监测(SIM)：定量离子：227；定性离子：344、274。

# 7.3.2 定性测定

分别等体积注入标准工作溶液及样品溶液于气相色谱一质谱仪中，按上述条件进行分析，甲氧滴滴涕出峰时间为 $23.5\mathrm{min}$ 。甲氧滴滴涕农药的CAS号、保留时间和监测离子丰度比参见附录A，甲氧滴滴涕农药标准物质的总离子流色谱图参见附录B，甲氧滴滴涕农药标准品质谱图和选择离子质谱图参见附录C。进行样品测定时，如果检出的色谱峰保留时间与标准品的保留时间相一致，并且所选择的离子的丰度与标准品对应离子的丰度相比，偏差在表1允许的范围内，则可判断样品中存在甲氧滴滴涕。

# 7.3.3 定量测定

定量采用单个离子定量，配制系列浓度的标准溶液，得到峰面积与浓度的关系，外标法定量测定。

# 7.4 空白实验

除不称取试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中甲氧滴滴涕残留含量。

$$
X = \frac {A \times c \times v}{A _ {s} \times m}. \tag {1}
$$

式中：

$X$ ——试样中甲氧滴滴涕的残留含量，单位为毫克每千克 $(\mathrm{mg / kg})$

$A$ ——样液中甲氧滴滴涕的色谱峰面积；

$A_{s}$ ——标准工作溶液中甲氧滴滴涕的色谱峰面积；

$c$ ——标准工作溶液中甲氧滴滴涕的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$v$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$

$m$ ——最终样液所代表的样品质量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录D的要求。

9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录 E 的要求。

10.1 定量限

本方法甲氧滴滴涕的定量限为 $0.005\mathrm{mg / kg}$

# 食品安全国家标准

# 食品中对羟基苯甲酸酯类的测定

2016-08-31 发布

2017-03-01 实施

# 前言

本标准代替GB/T5009.31—2003《食品中对羟基苯甲酸酯类的测定》。

本标准与GB/T5009.31—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中对羟基苯甲酸酯类的测定”；  
扩大了对羟基苯甲酸酯类的检测范围，增加了对羟基苯甲酸甲酯、对羟基苯甲酸丁酯的检测；  
——修改了分析步骤中酱油、醋、果汁液体试样称量方式的表述；  
——修改了对羟基苯甲酸酯类的标准曲线范围；  
——气相色谱法增加了毛细管柱分析测定条件，删除了填充柱分析测定条件；  
——增加了食品中对羟基苯甲酸酯类测定方法的定量限；  
——修改了计算公式，将计算结果以对羟基苯甲酸酯类计，改为计算结果以对羟基苯甲酸计。

# 食品安全国家标准

# 食品中对羟基苯甲酸酯类的测定

# 1 范围

本标准规定了食品中对羟基苯甲酸甲酯、对羟基苯甲酸乙酯、对羟基苯甲酸丙酯、对羟基苯甲酸丁酯的气相色谱方法。

本标准适用于酱油、醋、饮料及果酱中对羟基苯甲酸甲酯、对羟基苯甲酸乙酯、对羟基苯甲酸丙酯、对羟基苯甲酸丁酯的测定。

# 2 原理

试样酸化后，对羟基苯甲酸酯类用乙醚提取，浓缩近千用乙醇复溶，并利用氢火焰离子化检测器气相色谱法进行分离测定，保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 无水乙醚 $(\mathrm{C}_2\mathrm{H}_5\mathrm{OC}_2\mathrm{H}_5)$ ：重蒸。  
3.1.2 无水乙醇 $\mathrm{(C_2H_5OH)}$ ：优级纯。  
3.1.3 盐酸（HCl）。  
3.1.4 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.5 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.6 碳酸氢钠 $(\mathrm{NaHCO}_3)$ 。

# 3.2 试剂配制

3.2.1 饱和氯化钠溶液：称取 $40\mathrm{g}$ 氯化钠(3.1.4)加 $100~\mathrm{mL}$ 水充分搅拌溶解。  
3.2.2 碳酸氢钠溶液 $(10\mathrm{g / L})$ ：称取 $1\mathrm{g}$ 碳酸氢钠(3.1.6)，溶于水并稀释至 $100~\mathrm{mL}$ 。  
3.2.3 盐酸溶液 $(1:1)$ ：量取 $50~\mathrm{mL}$ 盐酸(3.1.3)，用水稀释至 $100~\mathrm{mL}$ 。

# 3.3 标准品

3.3.1 对羟基苯甲酸甲酯 $\mathrm{(C_8H_8O_3)}$ ，纯度 $\geq 99.8\%$   
3.3.2 对羟基苯甲酸乙酯 $\mathrm{(C_9H_{10}O_3)}$ ，纯度 $\geq 99.7\%$   
3.3.3 对羟基苯甲酸丙酯 $\left(\mathrm{C}_{10} \mathrm{H}_{12} \mathrm{O}_3\right)$ ，纯度 $\geq 99.3\%$ 。  
3.3.4 对羟基苯甲酸丁酯 $(\mathrm{C_{11}H_{14}O_3})$ ，纯度 $\geq 99.7\%$

# 3.4 标准溶液配制

3.4.1 单个对羟基苯甲酸酯类标准储备液 $(1.00\mathrm{mg / mL})$ ：准确称取对羟基苯甲酸甲酯、对羟基苯甲酸

乙酯、对羟基苯甲酸丁酯、对羟基苯甲酸丙酯标准物质各 $0.0500\mathrm{g}$ 于 $50.0\mathrm{mL}$ 容量瓶中，用无水乙醇溶解并定容至刻度，置 $4^{\circ}\mathrm{C}$ 左右冰箱保存，可保存1个月。

3.4.2 对羟基苯甲酸酯类标准中间液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：分别准确吸取单个对羟基苯甲酸酯类标准储备液(3.4.1) $1.0~\mathrm{mL}$ 于 $10.0~\mathrm{mL}$ 容量瓶中，用无水乙醇稀释至刻度，摇匀。临用时配制。  
3.4.3 对羟基苯甲酸酯类标准工作液 $1\sim 5$ ：分别吸取对羟基苯甲酸酯类标准中间液(3.4.2) $0.40~\mathrm{mL}$ 、 $1.0~\mathrm{mL},2.0~\mathrm{mL},5.0~\mathrm{mL},10.0~\mathrm{mL}$ 容量瓶中，用无水乙醇稀释并定容。此即为 $4.0~\mu \mathrm{g / mL}$ 、 $10.0~\mu \mathrm{g / mL},20.0~\mu \mathrm{g / mL},50.0~\mu \mathrm{g / mL},100~\mu \mathrm{g / mL}$ 的标准工作液 $1\sim 5$ 的浓度，临用时配制。  
3.4.4 对羟基苯甲酸酯类标准工作液6和标准工作液7(200 $\mu \mathrm{g} / \mathrm{mL}, 300 \mu \mathrm{g} / \mathrm{mL})$ ：分别吸取对羟基苯甲酸酯类标准储备液(3.4.1) $2.0 \mathrm{~mL}, 3.0 \mathrm{~mL}$ 于 $10.0 \mathrm{~mL}$ 容量瓶中，用无水乙醇稀释至刻度，摇匀。临用时配制。

# 4 仪器与设备

4.1 气相色谱仪：配有氢火焰离子化检测器(FID)。  
4.2 天平：感量 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.3 旋转蒸发仪。  
4.4 涡旋混匀器。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 试样处理

5.1.1.1 酱油、醋、饮料：一般液体试样摇匀后可直接取样。称取 $5\mathrm{~g}$ (精确至 $0.001\mathrm{~g}$ ) 试样于小烧杯中，并转移至 $125\mathrm{~mL}$ 分液漏斗中，用 $10\mathrm{~mL}$ 饱和氯化钠溶液分次洗涤小烧杯，合并洗涤液于 $125\mathrm{~mL}$ 分液漏斗，加入 $1\mathrm{~mL}1:1$ 盐酸酸化，摇匀，分别以 $75\mathrm{~mL}, 50\mathrm{~mL}, 50\mathrm{~mL}$ 无水乙醚提取三次，每次 $2\mathrm{~min}$ ，放置片刻，弃去水层，合并乙醚层于 $250\mathrm{~mL}$ 分液漏斗中，加入 $10\mathrm{~mL}$ 饱和氯化钠溶液洗涤一次，再分别以碳酸氢钠溶液 $30\mathrm{~mL}, 30\mathrm{~mL}, 30\mathrm{~mL}$ 洗涤三次，弃去水层。用滤纸吸去漏斗颈部水分，将有机层经过无水硫酸钠(约 $20\mathrm{~g}$ ) 滤入浓缩瓶中，在旋转蒸发仪上浓缩近干，用氮气除去残留溶剂，准确加入 $2.0\mathrm{~mL}$ 无水乙醇溶解残留物，供气相色谱用。  
5.1.1.2 果酱：称取 $5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）事先均匀化的果酱试样于 $100~\mathrm{mL}$ 具塞试管中，加入 $1\mathrm{mL}$ $1:1$ 盐酸酸化， $10~\mathrm{mL}$ 饱和氯化钠溶液，涡旋混匀 $1\mathrm{min} \sim 2\mathrm{min}$ ，使其为均匀溶液，再分别以 $50~\mathrm{mL}$ 、 $30~\mathrm{mL}, 30~\mathrm{mL}$ 无水乙醚提取三次，每次 $2\mathrm{min}$ ，用吸管转移至 $250~\mathrm{mL}$ 分液漏斗中，加入 $10~\mathrm{mL}$ 饱和氯化钠溶液洗涤一次，再分别以碳酸氢钠溶液 $30~\mathrm{mL}, 30~\mathrm{mL}, 30~\mathrm{mL}$ 洗涤三次，弃去水层。用滤纸吸去漏斗颈部水分，将有机层经过无水硫酸钠（约 $20\mathrm{g}$ ）滤入浓缩瓶中，在旋转蒸发仪上浓缩近千，用氮气除去残留溶剂，准确加入 $2.0~\mathrm{mL}$ 无水乙醇溶解残留物，供气相色谱用。

# 5.3 标准曲线的制作

将 $1.0\mu \mathrm{L}$ 的标准系列工作液(3.4.3、3.4.4)分别注入气相色谱仪中，测定相应的不同浓度标准的峰面积，以标准工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。

# 5.4 试样溶液的测定

将 $1.0\mu \mathrm{L}$ 的试样溶液(5.1.1.1、5.1.1.2)注入气相色谱仪中，以保留时间定性，得到相应的峰面积，根据标准曲线得到待测液中组分浓度；试样待测液响应值若超出标准曲线线性范围，应用乙醇稀释后再进样分析。

# 6 分析结果的表述

试样中对羟基苯甲酸含量按式(1)计算：

$$
X _ {i} = \frac {c \times V \times f}{m} \tag {1}
$$

式中：

$X_{i}$ ——试样中对羟基苯甲酸的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——由标准曲线计算出进样液中对羟基苯甲酸酯类的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——对羟基苯甲酸酯类转换为对羟基苯甲酸的换算系数；

$m$ ——试样质量，单位为克（g）。

说明：0.9078——对羟基苯甲酸甲酯转换为对羟基苯甲酸的换算系数；0.8312——对羟基苯甲酸乙酯转换为对羟基苯甲酸的换算系数；0.7665——对羟基苯甲酸丙酯转换为对羟基苯甲酸的换算系数；0.7111——对羟基苯甲酸丁酯转换为对羟基苯甲酸的换算系数。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不超过算术平均值的 $10\%$ 。

# 8 其他

当试样量为 $5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）、定容体积为 $2.0~\mathrm{mL}$ 时，对羟基苯甲酸甲酯、对羟基苯甲酸乙酯、对羟基苯甲酸丙酯、对羟基苯甲酸丁酯的方法定量限(LOQ)为 $2.0\mathrm{mg / kg}$ ；对羟基苯甲酸甲酯、对羟基苯甲酸乙酯、对羟基苯甲酸丙酯、对羟基苯甲酸丁酯的检出限(LOD)为 $0.6\mathrm{mg / kg}$ 。

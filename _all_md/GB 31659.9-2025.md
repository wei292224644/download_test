# 前言

# 食品安全国家标准

# 液相色谱-串联质谱法

# 1 范围

本文件规定了禽蛋中卡巴氧代谢物喹噁啉-2羧酸(QCA)和喹乙醇代谢物3-甲基喹噁啉-2-羧酸(MQCA)残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鸡蛋、鸭蛋、鹌鹑蛋中QCA和MQCA残留量的测定，其他禽蛋的检测可参照执行。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款，其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中残留的 QCA 和 MQCA 残留经偏磷酸溶液水解提取, 叔丁基甲醚萃取后, 用磷酸盐缓冲液反萃取, 混合型强阴离子交换性净化, 酸性甲醇洗脱, 液相色谱-串联质谱法测定, 内标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的二级水。

# 5.1 试剂

5.1.1 甲醇 $(\mathrm{CH}_3(\mathrm{OH})_2$ ：色谱纯  
5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.3 甲酸（HCOOH）：色谱纯。  
5.1.4 偏磷酸 $\mathrm{(HPO_3)}$ 。  
5.1.5 叔丁基甲醚 $\left[\left(\mathrm{CH}_{3}\right)_{3} \mathrm{COOH}_{2}\right]$   
5.1.6 十二水合磷酸氢二钠 $\mathrm{[Na_2HPO_4\cdot 12H_2O]}$   
5.1.7 氢氧化钠（NaOH）。

# 5.2 溶液配制

5.2.1 $10\%$ 偏磷酸 $-20\%$ 乙腈溶液：称取 $100\mathrm{g}$ 偏磷酸，放入 $1000~\mathrm{mL}$ 烧杯中，加入 $600~\mathrm{mL}$ 水，加热沸腾至溶解，冷却至室温后，加入 $200~\mathrm{mL}$ 乙腈，然后加水稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.2 磷酸盐缓冲液：称取十二水合磷酸氢二钠 $(\mathrm{Na_2HPO_4}\cdot 12\mathrm{H_2O})3.58\mathrm{g}$ ，加入 $900~\mathrm{mL}$ 水溶解，用磷酸调节 $\mathrm{pH}$ 至7.0，加水稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.3 $0.05\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：称取 $0.5\mathrm{g}$ 氢氧化钠，加水溶解，冷却至室温后，加水稀释至 $250~\mathrm{mL}$ 混匀。  
5.2.4 $2\%$ 甲酸甲醇：取 $2\mathrm{mL}$ 甲酸，加甲醇稀释至 $100~\mathrm{mL}$ ，混匀。  
5.2.5 $0.1\%$ 甲酸-甲醇溶液：取 $10~\mathrm{mL}$ 甲醇和 $100~\mu \mathrm{L}$ 甲酸，加水稀释至 $100~\mathrm{mL}$ ，混匀。

5.2.6 $0.1\%$ 甲酸水溶液：取 $1.0 \mathrm{~mL}$ 甲酸，加水稀释至 $1000 \mathrm{~mL}$ ，混匀。

# 5.3 标准品

5.3.1 喹噁啉-2-羧酸（quinoxaline-2-carboxylic acid, QCA, $\mathrm{C_9H_5N_2O_2}$ , CAS号: 879-65-2）含量 $\geqslant$ $97.0\%$ 。  
5.3.2 3-甲基喹噁啉-2-羧酸(methyl-3-quinoxaline-2-carboxylic acid, MQCA, $\mathrm{C_{10}H_8N_2O_2}$ , CAS号: 74003-63-7)含量 $\geqslant 97.0\%$ 。  
5.3.3 内标：氘代喹噁啉-2-羧酸（quinoxaline-2-carboxylic acid-D4，QCA-D4，C9HD4N2O2）含量≥98.0%  
5.3.4 内标：氘代3-甲基喹噁啉-2-羧酸（methyl-3-quinoxaline-2-carboxylic acid-D₄，MQCA-D₄，C₁₀H₁D₄N₂O₂）含量≥98.0%。

# 5.4 标准溶液的制备

5.4.1 标准储备液：取QCA和MQCA标准品适量(相当于各有效成分约 $10\mathrm{mg}$ )，精密称定，用甲醇溶解并稀释定容于 $100~\mathrm{mL}$ 棕色容量瓶，配制成浓度为 $100\mathrm{mg / L}$ 的QCA和MQCA标准储备液， $-18^{\circ}C$ 以下避光保存，有效期6个月。  
5.4.2 内标储备液：取 $\mathrm{QCA - D_4}$ 和 $\mathrm{MQCA - D_4}$ 标准品适量（相当于各有效成分约 $10\mathrm{mg}$ ），精密称定，用甲醇溶解并稀释定容于 $100~\mathrm{mL}$ 棕色容量瓶，配制成浓度为 $100\mathrm{mg / L}$ 的 $\mathrm{QCA - D_4}$ 和 $\mathrm{MQCA - D_4}$ 内标储备液， $-18^{\circ}\mathrm{C}$ 以下避光保存，有效期6个月。  
5.4.3 混合标准工作液：分别精密移取QCA和MQCA标准储备液各 $0.2\mathrm{mL}$ 于 $100~\mathrm{mL}$ 棕色容量瓶，用甲醇稀释至刻度，配制成浓度为 $200~\mu \mathrm{g / L}$ 的混合标准工作液， $4^{\circ}C$ 以下避光保存，有效期1个月。  
5.4.4 混合内标工作液：分别精密移取 $\mathrm{QCA - D_4}$ 和 $\mathrm{MQCA - D_4}$ 内标储备液各 $0.2\mathrm{mL}$ 于 $100~\mathrm{mL}$ 棕色容量瓶，用甲醇稀释至刻度，配制成浓度为 $200~\mu \mathrm{g / L}$ 的混合内标工作液， $4^{\circ}C$ 以下避光保存，有效期1个月。

# 5.5 材料

5.5.1 混合型强阴离子交换柱： $60\mathrm{mg} / 3\mathrm{mL}$ ，或相当者。  
5.5.2 微孔滤膜： $0.22\mu \mathrm{m}$ ，有机相。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配有电喷雾离子源(ESI)。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
6.3 氮吹仪。  
6.4 超声波清洗仪。  
6.5 酸度计。  
6.6 涡旋混合器。  
6.7 高速离心机：不低于 $8000\mathrm{r / min}$   
6.8 组织匀浆机。  
6.9 固相萃取装置。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或冷藏的空白或供试禽蛋，去壳并使均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试料 $(5\pm 0.05)\mathrm{g}$ ，于 $50~\mathrm{mL}$ 聚丙烯离心管，加 $50~\mu \mathrm{L}$ 混合内标工作液和 $6\mathrm{mL}10\%$ 偏磷酸- $20\%$ 乙腈溶液，涡旋 $1\mathrm{min}$ 混匀，超声 $10\mathrm{min},8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。取上清液于另一 $50~\mathrm{mL}$ 离心管中，残渣中加 $6\mathrm{mL}10\%$ 偏磷酸- $20\%$ 乙腈溶液，重复提取1次，合并2次提取液。在提取液中加 $10~\mathrm{mL}$ 叔丁基甲醚，涡旋 $1\mathrm{min}$ 混匀， $4500\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。转移上层有机相至另一 $50~\mathrm{mL}$ 离心管，用 $10~\mathrm{mL}$ 叔丁基甲醚重复提取1次，合并上层有机相。加入 $10~\mathrm{mL}$ 磷酸盐缓冲液，涡旋 $2\mathrm{min},4500\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去上层有机相，下层磷酸缓冲液备用。

# 8.2 净化

固相萃取柱依次用甲醇和水各 $3\mathrm{mL}$ 活化，取备用液过柱，依次用 $0.05\mathrm{mol} / \mathrm{L}$ NaOH溶液和甲醇溶液各 $3\mathrm{mL}$ 淋洗，用 $2\%$ 甲酸-甲醇溶液 $6\mathrm{mL}$ 洗脱。收集洗脱液： $40^{\circ}\mathrm{C}$ 水浴氮气吹干，残余物中加 $0.1\%$ 甲酸-甲醇溶液 $1.0\mathrm{mL}$ 池旋 $1\mathrm{min}$ ，过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱仪测定。

# 8.3 标准曲线的制备

精密量取适量混合标准工作液及混合内标标准工作液，用初始流动相稀释配制成浓度为 $1\mathrm{ng / mL}$ 、 $2\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL},40\mathrm{ng / mL}$ 的系列标准溶液（内标均为 $10\mathrm{ng / mL}$ ，供液相色谱-串联质谱仪测定。以测得的待测物与对应内标的特征离子峰面积之比为纵坐标，标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.4.3 定性测定

在相同测试条件下，试料溶液中QCA和MQCA与其内标 $\mathrm{QCA - D_4}$ 和 $\mathrm{MQCA - D_4}$ 的保留时间之比与标准溶液中QCA和MQCA与其内标 $\mathrm{QCA - D_4}$ 和 $\mathrm{MQCA - D_4}$ 的保留时间之比偏差在 $1\%$ 以内；且检测到的相对离子丰度应与浓度相当的校正标准溶液相对丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.4.4 定量测定

取试料溶液和相应的标准溶液，做单点或多点校准，按内标法定量，试料溶液及标准溶液中目标物的响应值均应在仪器检测的线性范围之内。在上述色谱-质谱条件下，标准溶液特征离子质量色谱图见附录A。

# 8.5 空白试验

取空白试样，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A _ {i} \times A _ {i s} ^ {\prime} \times C _ {s} \times C _ {i s} \times V}{A _ {i s} \times A _ {s} \times C _ {i s} ^ {\prime} \times m} \tag {1}
$$

式中：

$X$ ——供试试样中待测物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A_{i}$ ——供试试料溶液中待测物的峰面积；

$A_{is}^{\prime}$ ——标准溶液中同位素内标的峰面积；

$C_{s}$ ——标准溶液中待测物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{is}$ ——供试试料溶液中同位素内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——供试试样最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{is}$ ——供试试料溶液中同位内标的峰面积；

$A_{s}$ ——标准溶液中待测物的峰面积；

$C_{is}^{\prime}$ ——标准溶液中同位素内标浓度的数值，单位为微克每升 $(\mu \mathrm{g / L})$

$m$ ——供试试样质量的数值，单位为克 $(\mathbf{g})$ 。

# 10 检测方法的灵敏度、正确度和精密度

# 10.1 灵敏度

本方法中QCA和MQCA的检出限均为 $0.2\mu \mathrm{g / kg}$ ，定量限均为 $0.5\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法中QCA和MQCA在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 10.0\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 20\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

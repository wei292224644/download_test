# 前言

本标准代替GB/T5009.118—2008《谷物中T-2毒素的测定》、GB/T23501—2009《食品中T-2毒素的测定免疫亲和层析净化高效液相色谱法》、SN/T1771—2006《进出口粮谷中T-2毒素的测定免疫亲和柱-液相色谱法》、SN/T2676—2010《进出口粮谷中T-2毒素的检测方法酶联免疫吸附法》。

本标准与GB/T5009.118—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中T-2毒素的测定”；  
——增加了适用范围；  
——增加了直接ELISA法二。

# 食品安全国家标准

# 食品中T-2毒素的测定

# 1 范围

本标准规定了食品中T-2毒素的测定方法。

本标准第一法适用于粮食及粮食制品，酒类，酱油、醋、酱及酱制品中T-2毒素含量的测定，第二法、第三法适用于粮食及粮食制品中T-2毒素的测定。

# 第一法 免疫亲和层析净化液相色谱法

# 2 原理

用提取液提取试样中的T-2毒素，经免疫亲和柱净化、衍生后，用高效液相色谱荧光检测器测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.3 甲苯 $\mathrm{(C_6H_5CH_3)}$ ：色谱纯。  
3.1.4 4-二甲基氨基吡啶。  
3.1.5 1-蒽腈(1-anthroylnitrile,1-AN)。

# 3.2 试剂配制

3.2.1 提取液：甲醇-水（8：2，体积比）。  
3.2.2 4-二甲基氨基吡啶溶液: 准确称取 $0.0325 \mathrm{~g} 4-$ 二甲基氨基吡啶于 $100 \mathrm{~mL}$ 容量瓶中, 用甲苯定容至刻度。  
3.2.3 1-蒽腈溶液：准确称取 $0.030\mathrm{g}1$ -蒽腈于 $100~\mathrm{mL}$ 容量瓶中，用甲苯定容至刻度。

# 3.3 标准品

T-2毒素 $\mathrm{(C_{24}H_{34}O_9}$ ，CAS号：21259-20-1)：纯度 $\geqslant 98.0\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 标准储备液：准确称取适量的T-2毒素标准品（精确至 $0.0001\mathrm{g}$ ），用乙腈溶解，配制成浓度为

$100\mu \mathrm{g / mL}$ 的标准储备液，一 $18^{\circ}C$ 以下避光保存。

3.4.2 标准工作液：根据需要准确吸取适量的标准储备液，用乙腈稀释，配制成 $5\mathrm{ng / mL},10\mathrm{ng / mL}$ 、 $50~\mathrm{ng / mL},100~\mathrm{ng / mL},200~\mathrm{ng / mL}$ 的标准工作液， $4^{\circ}C$ 避光保存。

# 3.5 材料

3.5.1 T-2毒素免疫亲和柱：柱规格 $1\mathrm{mL}$ 或 $3\mathrm{mL}$ ，柱容量 $\geqslant 1500\mathrm{ng}$ ，或等效柱。  
3.5.2 玻璃纤维滤纸：直径 $11\mathrm{cm}$ ，孔径 $1.5\mu \mathrm{m}$ ，无荧光特性。

# 4 仪器和设备

4.1 高效液相色谱仪：配有荧光检测器。  
4.2 高速粉碎机：转速 $\geq 12000\mathrm{r / min}$   
4.3 均质器：转速 $\geqslant 12000\mathrm{r / min}$   
4.4 氮吹仪。  
4.5 离心机。  
4.6 涡旋混合仪。  
4.7 空气压力泵。  
4.8 试验筛：孔径 $1.0\mathrm{mm}$   
4.9 天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
4.10 超声波发生器：功率 $>180\mathrm{W}$ 。  
4.11 玻璃注射器： $10~\mathrm{mL}$

# 5 分析步骤

# 5.1 提取

# 5.1.1 粮食及粮食制品

将样品研磨，硬质的粮食等用高速粉碎机磨细并通过试验筛。称取 $25.0\mathrm{g}$ （精确到 $0.1\mathrm{g}$ ）试样于烧杯中，用提取液溶解，转移至容量瓶中，用提取液定容至 $100~\mathrm{mL}$ ，混匀，涡旋振荡提取 $10\mathrm{min}$ ，定量滤纸过滤。移取 $10.0~\mathrm{mL}$ 滤液加入 $40~\mathrm{mL}$ 水稀释混匀，经玻璃纤维滤纸过滤至滤液澄清，滤液备用。

# 5.1.2 酱油、醋、酱及酱制品

称取 $25.0\mathrm{g}$ （精确到 $0.1\mathrm{g}$ )混匀的试样，用甲醇定容至 $50.0~\mathrm{mL}$ ，超声提取 $10\mathrm{min}$ ，定量滤纸过滤。移取 $10.0~\mathrm{mL}$ 滤液加入 $40~\mathrm{mL}$ 水稀释混匀，经玻璃纤维滤纸过滤至滤液澄清，滤液备用。

# 5.1.3 酒类

取脱气酒类试样（含二氧化碳的酒类使用前先置于 $4^{\circ}\mathrm{C}$ 冰箱冷藏 $30\mathrm{min}$ ，过滤或超声脱气）或其他不含二氧化碳的酒类试样 $20.0\mathrm{g}$ （精确到 $0.1\mathrm{g}$ ）于 $50~\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度，摇匀，定量滤纸过滤。移取 $10.0~\mathrm{mL}$ 滤液加入 $40~\mathrm{mL}$ 水稀释，经玻璃纤维滤纸过滤至滤液澄清，滤液备用。

# 5.2 净化和洗脱

将免疫亲和柱连接于玻璃注射器下，准确移取 $10.0\mathrm{mL}5.1$ 中的提取滤液，注入玻璃注射器中。将空气压力泵与玻璃注射器连接，调节压力使溶液以约1滴/s的流速缓慢通过免疫亲和柱，直至空气进

入亲和柱中。用 $10~\mathrm{mL}$ 水淋洗亲和柱，流速为1滴/s～2滴/s，直至空气进入亲和柱中，弃去全部流出液，抽干小柱。准确加入 $1.0~\mathrm{mL}$ 甲醇洗脱，流速约为1滴/s，收集洗脱液。

# 5.3 衍生

5.3.1 标准工作液的衍生：取不同浓度的标准工作液各 $1\mathrm{mL}$ ，在 $50^{\circ}\mathrm{C}$ 下用氮气吹干，加入 $50~\mu \mathrm{L}4-$ 二甲基氨基吡啶溶液和 $50~\mu \mathrm{L}1-$ 葱腈溶液，在涡旋混合器上混匀 $1\mathrm{min}$ ， $50^{\circ}\mathrm{C}$ 反应 $15\mathrm{min}$ ，在冰水中冷却 $10\mathrm{min}$ 后取出， $50^{\circ}\mathrm{C}$ 下氮气吹干，用 $1.0\mathrm{mL}$ 流动相溶解，待HPLC测定。

5.3.2 样品的衍生：将5.2中洗脱液在 $50~\mathrm{^\circ C}$ 下用氮气吹干，按5.3.1步骤进行。

# 5.4 测定

# 5.4.1 高效液相色谱参考条件

高效液相色谱参考条件列出如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $150~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或等效柱；  
b）流动相：乙腈-水（75：25，体积比）；  
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d）检测波长：激发波长 $381\mathrm{nm}$ ，发射波长 $470~\mathrm{nm}$   
e) 进样量: $20 \mu \mathrm{L}$ ;   
f） 柱温： $35^{\circ}C$

# 5.4.2 色谱测定

在5.4.1色谱条件下，将系列T-2毒素标准工作液按浓度从低到高依次注入高效液相色谱仪。待仪器稳定后，以目标物质的浓度为横坐标 $(x$ 轴)，目标物质的峰面积为纵坐标 $(y$ 轴)，对各个数据点进行最小二乘线性拟合，标准工作曲线按式(1)计算：

$$
y = a x + b \quad \dots \tag {1}
$$

式中：

$y$ ——目标物质的峰面积比；

$a$ ——回归曲线的斜率；

$x$ ——目标物质的浓度；

$b$ ——回归曲线的截距。

标准工作液和样液中待测物的响应值均应在仪器线性响应范围内，如果样品含量超过标准曲线范围，需稀释后再测定。

# 5.5 空白试验

不称取试样，按5.1、5.2和5.3的步骤做空白试验。应确认不含有干扰待测组分的物质。

# 5.6 结果计算

试样中T-2毒素的含量按式(2)计算：

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0} \times f \tag {2}
$$

式中：

$X$ ——试样中T-2毒素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——试样测定液中T-2毒素的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——衍生化后的定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——单位换算常数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ ；

$f$ ——稀释倍数。

注：计算结果需扣除空白值，保留两位有效数字。

# 6 精密度

样品中T-2毒素的含量在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 7 其他

本方法对粮食及粮食制品中T-2毒素的检出限为 $10\mu \mathrm{g / kg}$ ，定量限为 $33~\mu \mathrm{g / kg}$ ；对酒类，酱油、醋、酱及酱制品中T-2毒素的检出限为 $5\mu \mathrm{g / kg}$ ，定量限为 $17\mu \mathrm{g / kg}$ 。

# 第二法 间接ELISA法

# 8 原理

将已知抗原吸附在固相载体表面，洗除未吸附抗原，加入一定量抗体与待测试样(含有抗原)提取液的混合液，竞争温育后，在固相载体表面形成抗原-抗体复合物。洗除多余抗体成分，然后加入酶标记的抗球蛋白的第二抗体结合物，与吸附在固体表面的抗原-抗体复合物相结合，再加入酶的底物。在酶的催化作用下，底物发生降解反应，产生有色产物，通过酶标仪，测出酶底物的降解量，从而推知被测试样中的抗原量。

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 9.1 试剂

9.1.1 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。  
9.1.2 石油醚 $\mathrm{(C_7H_7BrMg)}$   
9.1.3 三氯甲烷 $\mathrm{(CHCl_3)}$   
9.1.4 无水乙醇 $\mathrm{(C_2H_5OH)}$   
9.1.5 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ 。  
9.1.6 二甲基甲酰胺 $\mathrm{(C_3H_7NO)}$   
9.1.7 四甲基联苯胺(TMB)。  
9.1.8 吐温-20 $(\mathrm{C}_{58}\mathrm{H}_{114}\mathrm{O}_{26})$ 。  
9.1.9 $30\%$ 过氧化氢 $(30\% \mathrm{H}_2\mathrm{O}_2)$   
9.1.10 碳酸钠 $(\mathrm{Na}_2\mathrm{CO}_3)$ 。  
9.1.11 碳酸氢钠 $\mathrm{(NaHCO_3)}$ 。  
9.1.12 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$

9.1.13 磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4\cdot 12\mathrm{H}_2\mathrm{O})$ 。  
9.1.14 氯化钠 $(\mathrm{NaCl})$ 。  
9.1.15 氯化钾(KCl)。  
9.1.16 柠檬酸 $(\mathrm{C_6H_8O_7}\cdot \mathrm{H_2O})$ 。  
9.1.17 抗体：杂交瘤细胞系产生的抗T-2毒素的特异性单克隆抗体。  
9.1.18 抗原：T-2毒素与载体蛋白-牛血清白蛋白(BSA)的结合物。  
9.1.19 兔抗鼠免疫球蛋白与辣根过氧化酶的结合物（酶标二抗）。

# 9.2 试剂配制

9.2.1 ELISA缓冲液系统。  
9.2.1.1 包被缓冲液为 $\mathrm{pH} 9.6$ 的碳酸盐缓冲液, 称取 $1.59 \mathrm{~g}$ 碳酸钠、 $2.93 \mathrm{~g}$ 碳酸氢钠, 加水稀释至 $1000 \mathrm{~mL}$ 。  
9.2.1.2 洗液为含 $0.05\%$ 吐温-20 的 $\mathrm{pH} 7.4$ 的磷酸盐缓冲液 (简称 PBS-T)。配制方法为: 称取 $0.2 \mathrm{~g}$ 磷酸二氢钾、 $2.9 \mathrm{~g}$ 磷酸氢二钠、 $8.0 \mathrm{~g}$ 氯化钠、 $0.2 \mathrm{~g}$ 氯化钾、 $0.5 \mathrm{~mL}$ 吐温-20, 加水至 $1000 \mathrm{~mL}$ 。  
9.2.1.3 底物缓冲液为 $\mathrm{pH} 5.0$ 的磷酸-柠檬酸缓冲液, 配制方法为: $0.1 \mathrm{~mol} / \mathrm{L}$ 柠檬酸, 即称取柠檬酸 $19.2 \mathrm{~g}$ , 加水至 $1000 \mathrm{~mL}$ , 为甲液; $0.2 \mathrm{~mol} / \mathrm{L}$ 磷酸氢二钠, 即称取磷酸氢二钠 $71.7 \mathrm{~g}$ , 加水至 $1000 \mathrm{~mL}$ , 为乙液; 取甲液 $24.3 \mathrm{~mL}$ , 乙液 $25.7 \mathrm{~mL}$ , 加水至 $100 \mathrm{~mL}$ 即可。  
9.2.1.4 底物溶液：取 $50~\mu \mathrm{L}$ TMB 溶液（ $10\mathrm{mg}$ TMB 溶于 $1\mathrm{mL}$ 二甲基甲酰胺中），加 $10\mathrm{mL}$ 底物缓冲液及 $10~\mu \mathrm{L}30\%$ 过氧化氢，混匀。

# 9.3 标准品

T-2 毒素 $\left(\mathrm{C}_{24} \mathrm{H}_{34} \mathrm{O}_{9}\right)$ , CAS 号: 21259-20-1), 纯度 $\geqslant 98.0\%$ 。或经国家认证并授予标准物质证书的标准物质。

# 9.4 标准溶液配制

9.4.1 标准储备液：准确称取适量标准品（精确至 $0.0001\mathrm{g}$ ），用甲醇溶解，配制成浓度为 $1000\mu \mathrm{g / mL}$ 的标准储备液， $-18\%$ 以下避光保存。  
9.4.2 标准工作液：根据需要用 $20\%$ 甲醇的PBS(配制方法同PBS-T，不加吐温-20即可)将标准储备液稀释成适当浓度的标准工作液。

# 9.5 材料

9.5.1 酶标板（48孔或96孔）。  
9.5.2 具 $0.2\mathrm{mL}$ 尾管的 $10~\mathrm{mL}$ 小浓缩瓶。  
9.5.3 层析柱：在层析柱下端与小管相连结处塞约 $0.1\mathrm{g}$ 脱脂棉，尽量塞紧，先装入 $0.5\mathrm{g}$ 中性氧化铝，敲平表面，再加入 $0.4\mathrm{g}$ 活性炭，敲紧。

# 10 仪器和设备

注：所有玻璃器皿均用硫酸洗液浸泡，用水冲洗。

10.1 酶标仪。  
10.2 电动振荡器。  
10.3 电热恒温水浴锅。  
10.4 天平：感量为 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。

# 11 分析步骤

# 11.1 提取

称取 $20.0\mathrm{g}$ 粉碎并通过20目筛的试样，置 $200~\mathrm{mL}$ 具塞锥形烧瓶中，加 $8\mathrm{mL}$ 水和 $100~\mathrm{mL}$ 三氯甲烷-无水乙醇(4：1，体积比)，密塞，振荡 $1\textrm{h}$ ，滤纸过滤，取 $25~\mathrm{mL}$ 滤液于蒸发皿中，置 $90^{\circ}C$ 水浴上通风挥干。用 $50~\mathrm{mL}$ 石油醚分次溶解蒸发皿中残渣，洗入 $250~\mathrm{mL}$ 分液漏斗中，再用 $20~\mathrm{mL}$ 甲醇-水 $(4:1)$ 分次洗涤，转入同一分液漏斗中，振荡 $1.5\mathrm{min}$ ，静置约 $15\mathrm{min}$ ，收集下层甲醇-水提取液过层析柱净化。

将过柱后的洗脱液倒入蒸发皿中，并于水浴锅上浓缩至干，趁热加 $3\mathrm{mL}$ 乙酸乙酯，加热至沸，挥干，再重复一次，最后加 $3\mathrm{mL}$ 乙酸乙酯，冷至室温后转入浓缩瓶中。用适量乙酸乙酯洗涤蒸发皿，并入浓缩瓶中，将浓缩瓶置 $95~^\circ \mathrm{C}$ 水浴锅上，挥干冷却后，用含 $20\%$ 甲醇的PBS定容，供ELISA检测。

# 11.2 检测

11.2.1 用T-2-BSA(4 $\mu \mathrm{g / mL}$ )包被酶标板，每孔 $100~\mu \mathrm{L},4^{\circ}C$ 过夜。   
11.2.2 酶标板用PBS-T洗3次，每次 $3\mathrm{min}$ 后，加入不同浓度的标准工作液(制作标准曲线)或试样提取液(检测试样中的毒素含量)与抗体溶液的混合液 $(1:1$ ，体积比，每孔 $100~\mu \mathrm{L}$ ，该混合液应于使用前的前一天配好， $4^{\circ}C$ 过夜备用)，置 $37^{\circ}C1\mathrm{h}$ 。  
11.2.3 酶标板洗3次，每次 $3\mathrm{min}$ 后，加入酶标二抗，每孔 $100~\mu \mathrm{L}, 37^{\circ}\mathrm{C}1.5\mathrm{h}$ 。  
11.2.4 同上述洗涤后，加入底物溶液，每孔 $100\mu \mathrm{L},37^{\circ}\mathrm{C}30\mathrm{min}$   
11.2.5 用 $1\mathrm{mol} / \mathrm{L}$ 硫酸溶液终止反应，每孔 $50~\mu \mathrm{L}$ ，于 $450~\mathrm{nm}$ 处测定吸光度值。

# 12 分析结果的表述

试样中T-2毒素的含量，按式(3)计算：

$$
X = m _ {1} \times \frac {V _ {1}}{V _ {2}} \times f \times \frac {1}{m} \tag {3}
$$

式中：

$X$ ——试样中T-2毒素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$m_{1}$ ——酶标板上所测得的T-2毒素的量，根据标准曲线求得，单位为纳克(ng)；

$V_{1}$ ——试样提取液的体积，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——滴加样液的体积，单位为毫升 $(\mathrm{mL})$

$f$ ——样液的总稀释倍数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

注：测定结果保留小数点后一位有效数字。

# 13 精密度

样品中T-2毒素的含量在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 14 其他

方法检出限为 $1\mu \mathrm{g / kg}$ ，定量限为 $3\mu \mathrm{g / kg}$

# 第三法 直接ELISA法

# 15 原理及依据

15.1 直接 ELISA 法一：将已知抗原吸附在固相载体表面，洗除未吸附的抗原，加入一定量的酶标记抗体与试样（含有抗原）提取液的混合液，竞争温育后，在固相载体表面形成抗原-抗体-酶复合物。洗除多余部分，加入酶的底物。在酶的催化作用下，底物发生降解反应，产生有色物质，通过酶标仪，测出酶底物的降解量，从而推知被测试样中的抗原量。  
15.2 直接ELISA法二：样品中的T-2毒素与T-2毒素酶标记物竞争结合至包被在微孔板上的抗体。通过洗涤除去微孔上未结合的T-2毒素与T-2毒素酶标记物，然后加入反应底物，用酶标仪测定吸光度，根据吸光度值得出试样中的T-2毒素含量。

# 18 分析步骤

# 18.1 提取

# 18.1.1 直接ELISA法一

称取 $20\mathrm{g}$ 粉碎并通过20目筛的试样，置 $200~\mathrm{mL}$ 具塞锥形烧瓶中，加 $8\mathrm{mL}$ 水和 $100~\mathrm{mL}$ 三氯甲烷-无水乙醇(4：1，体积比)，密塞，振荡 $1\textrm{h}$ ，通过滤纸过滤，取 $25~\mathrm{mL}$ 滤液于蒸发皿中，置 $90^{\circ}C$ 水浴上通风挥干。用 $50~\mathrm{mL}$ 石油醚分次溶解蒸发皿中残渣，洗入 $250~\mathrm{mL}$ 分液漏斗中，再用 $20~\mathrm{mL}$ 甲醇-水（4：1，体积比)分次洗涤，转入同一分液漏斗中，振荡 $1.5\mathrm{min}$ ，静置约 $15\mathrm{min}$ ，收集下层甲醇-水提取液过层析柱净化。

将过柱后的洗脱液倒入蒸发皿中，并于水浴锅上浓缩至干，趁热加 $3\mathrm{mL}$ 乙酸乙酯，加热至沸，挥干，再重复一次，最后加 $3\mathrm{mL}$ 乙酸乙酯，冷至室温后转入浓缩瓶中。用适量乙酸乙酯洗涤蒸发皿，并入浓缩瓶中，将浓缩瓶置 $95^{\circ}C$ 水浴锅上，挥干冷却后，用含 $20\%$ 甲醇的PBS定容，供ELISA检测。

# 18.1.2 直接ELISA法二

将样品按四分法缩分至 $1\mathrm{kg}$ ，全部磨碎至颗粒可通过20目筛大小，混匀，均分成两份作为试样，分

别装入清洁的容器内，密封。称取 $25.0\mathrm{g}$ 试样(精确到 $0.1\mathrm{g}$ )，加入 $125\mathrm{mL}$ 甲醇-水（7：3，体积比），均质 $1\mathrm{min} \sim 2\mathrm{min}, 5000\mathrm{r} / \mathrm{min}$ 离心 $10\mathrm{min}$ 。离心后取 $50~\mu \mathrm{L}$ 滤液加入 $300~\mu \mathrm{L}$ 样品稀释液，混匀。取 $50~\mu \mathrm{L}$ 进行酶联免疫测定，最后稀释倍数为35。高浓度的样品，毒素含量如超出标准曲线范围，可进一步稀释，直至样品中毒素浓度在标准曲线范围以内。

# 18.2 检测

# 18.2.1 直接ELISA法一

18.2.1.1 用T-2-BSA $(4\mu \mathrm{g / mL})$ 包被酶标板，每孔 $100~\mu \mathrm{L}$ ， $4^{\circ}C$ 过夜。  
18.2.1.2 酶标板用PBS-T洗3次，每次 $3\mathrm{min}$ 后，加入不同浓度的标准工作液(制作标准曲线)或试样提取液(检测试样中的毒素含量)与抗体-酶结合物溶液 $(1:100$ ，体积比)的混合液 $(1:1$ ，体积比，每孔 $100~\mu \mathrm{L}$ ，该混合液应于使用前的前一天配好， $4^{\circ}C$ 过夜备用)，置 $37^{\circ}C$ 1.5h。  
18.2.1.3 酶标板洗3次，每次 $3\mathrm{min}$ 后，加入底物溶液。每孔 $100~\mu \mathrm{L}, 37^{\circ}\mathrm{C} 30\mathrm{min}$   
18.2.1.4 用 $1\mathrm{mol} / \mathrm{L}$ 硫酸溶液终止反应，每孔 $50~\mu \mathrm{L}$ ，于 $450~\mathrm{nm}$ 处测定吸光度值。

# 18.2.2 直接ELISA法二

18.2.2.1 将测定需用的微孔条插入微孔架，记录标准液和样液在微孔架上的位置。吸取 $50 \mu \mathrm{L}$ T-2 毒素标准液（浓度为 $0 \mathrm{ng} / \mathrm{mL}, 0.1 \mathrm{ng} / \mathrm{mL}, 0.2 \mathrm{ng} / \mathrm{mL}, 0.4 \mathrm{ng} / \mathrm{mL}, 0.8 \mathrm{ng} / \mathrm{mL}, 1.6 \mathrm{ng} / \mathrm{mL}$ ）和样液至各微孔。  
18.2.2.2 吸取 $50~\mu \mathrm{L}$ T-2 毒素酶标记物至各微孔, 均匀混合, 依次加入 $50~\mu \mathrm{L}$ T-2 毒素抗体, 混合, 以封口膜将微孔覆盖防止试液挥发, 于 $22^{\circ} \mathrm{C} \sim 25^{\circ} \mathrm{C}$ 避光孵育 $1 \mathrm{~h}$ 。  
18.2.2.3 倒出孔中液体，以 $250~\mu \mathrm{L}$ 蒸馏水反复清洗微孔，重复操作3次以上，将微孔倒置在吸水纸上拍打数次，以保证完全除去微孔中洗液。  
18.2.2.4 迅速加入 $100\mu \mathrm{L}$ 底物至各微孔，于 $22^{\circ}C\sim 25^{\circ}C$ 避光孵育 $30\mathrm{min}$   
18.2.2.5 孵育完毕后加入 $100 \mu \mathrm{L}$ 反应停止液至各微孔, 充分混匀, 上酶标仪测量并记录每个微孔试液 $450 \mathrm{~nm}$ 波长处的吸光度值 (加入反应终止液后应在 $30 \mathrm{~min}$ 内读取吸光度)。

# 19 分析结果的表述

# 19.1 直接ELISA法一

试样中T-2毒素的含量按式(4)计算：

$$
X = m _ {1} \times \frac {V _ {1}}{V _ {2}} \times f \times \frac {1}{m} \tag {4}
$$

式中：

$X$ ——试样中T-2毒素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$m_{1}$ ——酶标板上所测得的T-2毒素的量，根据标准曲线求得，单位为纳克(ng)；

$V_{1}$ ——试样提取液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴加样液的体积，单位为毫升 $(\mathrm{mL})$

$f$ ——样液的总稀释倍数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

注：测定结果保留小数点后一位有效数字。

# 19.2 直接ELISA法二

试样中T-2毒素的含量按式(5)计算：

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {5}
$$

式中：

$X$ ——试样中T-2毒素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——从标准工作曲线上得到的样品中T-2毒素含量，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——样品溶液的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——单位换算常数；

$m$ ——样品溶液所代表的最终试样质量，单位为克 $(\mathrm{g})$ 。

注：测定结果保留小数点后一位有效数字。

# 20 精密度

样品中T-2毒素的含量在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

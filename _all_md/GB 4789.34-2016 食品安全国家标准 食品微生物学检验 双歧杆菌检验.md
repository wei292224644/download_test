# 前言

本标准代替GB4789.34—2012《食品安全国家标准 食品微生物学检验 双歧杆菌的鉴定》。

本标准与GB4789.34—2012相比，主要变化如下：

——增加了双歧杆菌的计数方法；  
——增加了MRS培养基；  
——修改了标准的适用范围；  
——修改了附录B为可选项。

# 1 范围

本标准规定了双歧杆菌(Bifidobacterium)的鉴定及计数方法。

本标准适用于双歧杆菌纯菌菌种的鉴定及计数。本标准适用于食品中仅含有单一双歧杆菌的菌种鉴定。本标准适用于食品中仅含有双歧杆菌属的计数，即食品中可包含一个或多个不同的双歧杆菌菌种。

# 2 设备和材料

除微生物实验室常规灭菌及培养设备外，其他设备和材料如下：

2.1 恒温培养箱： $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 。  
2.2 冰箱： $2^{\circ}C\sim 5^{\circ}C$   
2.3 天平：感量 $0.01\mathrm{g}$ 。  
2.4 无菌试管： $18\mathrm{mm} \times 180\mathrm{mm}, 15\mathrm{mm} \times 100\mathrm{mm}$ 。  
2.5 无菌吸管： $1\mathrm{mL}$ （具 $0.01\mathrm{mL}$ 刻度）、 $10\mathrm{mL}$ （具 $0.1\mathrm{mL}$ 刻度）或微量移液器 $(200\mu\mathrm{L} \sim 1000\mu\mathrm{L})$ 及配套吸头。  
2.6 无菌培养皿：直径 $90~\mathrm{mm}$

# 5 操作步骤

# 5.1 无菌要求

全部操作过程均应遵循无菌操作程序。

# 5.2 双歧杆菌的鉴定

# 5.2.1 纯菌菌种

5.2.1.1 样品处理：半固体或液体菌种直接接种在双歧杆菌琼脂平板或MRS琼脂平板。固体菌种或真空冷冻干燥菌种，可先加适量灭菌生理盐水或其他适宜稀释液，溶解菌粉。  
5.2.1.2 接种：接种于双歧杆菌琼脂平板或MRS琼脂平板。 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 厌氧培养 $48\mathrm{h} \pm 2\mathrm{h}$ ，可延长至 $72\mathrm{h} \pm 2\mathrm{h}$ 。

# 5.2.2 食品样品

5.2.2.1 样品处理：取样 $25.0\mathrm{g(mL)}$ ，置于装有 $225.0~\mathrm{mL}$ 生理盐水的灭菌锥形瓶或均质袋内，于 $8000\mathrm{r / min}\sim 10000\mathrm{r / min}$ 均质 $1\mathrm{min}\sim 2\mathrm{min}$ ，或用拍击式均质器拍打 $1\mathrm{min}\sim 2\mathrm{min}$ ，制成 $1:10$ 的样品匀液。冷冻样品可先使其在 $2^{\circ}C\sim 5^{\circ}C$ 条件下解冻，时间不超过 $18\mathrm{h}$ ；也可在温度不超过 $45^{\circ}C$ 的条件解冻，时间不超过 $15\mathrm{min}$ 。  
5.2.2.2 接种或涂布：将上述样品匀液接种在双歧杆菌琼脂平板或MRS琼脂平板，或取 $0.1\mathrm{mL}$ 适当稀释度的样品匀液均匀涂布在双歧杆菌琼脂平板或MRS琼脂平板。 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 厌氧培养 $48\mathrm{h} \pm 2\mathrm{h}$ ，可延长至 $72\mathrm{h} \pm 2\mathrm{h}$ 。  
5.2.2.3 纯培养：挑取3个或以上的单个菌落接种于双歧杆菌琼脂平板或MRS琼脂平板。 $36\mathrm{^\circ C}\pm$ $1\mathrm{^\circ C}$ 厌氧培养 $48\mathrm{h}\pm 2\mathrm{h}$ ，可延长至 $72\mathrm{h}\pm 2\mathrm{h}$

# 5.2.3 菌种鉴定

5.2.3.1 涂片镜检：挑取双歧杆菌平板或MRS平板上生长的双歧杆菌单个菌落进行染色。双歧杆菌为革兰氏染色阳性，呈短杆状、纤细杆状或球形，可形成各种分支或分叉等多形态，不抗酸，无芽孢，无动力。  
5.2.3.2 生化鉴定：挑取双歧杆菌平板或MRS平板上生长的双歧杆菌单个菌落，进行生化反应检测。过氧化氢酶试验为阴性。双歧杆菌的主要生化反应见表1。可选择生化鉴定试剂盒或全自动微生物生化鉴定系统。

# 5.3 双歧杆菌的计数

# 5.3.1 纯菌菌种

5.3.1.1 固体和半固体样品的制备：以无菌操作称取 $2.0\mathrm{g}$ 样品，置于盛有 $198.0~\mathrm{mL}$ 稀释液的无菌均质杯内， $8000\mathrm{r / min} \sim 10000\mathrm{r / min}$ 均质 $1\mathrm{min} \sim 2\mathrm{min}$ ，或置于盛有 $198.0~\mathrm{mL}$ 稀释液的无菌均质袋中，用拍击式均质器拍打 $1\mathrm{min} \sim 2\mathrm{min}$ ，制成 $1:100$ 的样品匀液。

5.3.1.2 液体样品的制备：以无菌操作量取 $1.0\mathrm{mL}$ 样品，置于 $9.0\mathrm{mL}$ 稀释液中，混匀，制成 $1:10$ 的样品匀液。

# 5.3.2 食品样品

5.3.2.1 样品处理：取样 $25.0\mathrm{g(mL)}$ ，置于装有 $225.0~\mathrm{mL}$ 生理盐水的灭菌锥形瓶或均质袋内，于 $8000\mathrm{r / min}\sim 10000\mathrm{r / min}$ 均质 $1\mathrm{min}\sim 2\mathrm{min}$ ，或用拍击式均质器拍打 $1\mathrm{min}\sim 2\mathrm{min}$ ，制成 $1:10$ 的样品匀液。冷冻样品可先使其在 $2^{\circ}C\sim 5^{\circ}C$ 条件下解冻，时间不超过 $18\mathrm{h}$ ；也可在温度不超过 $45^{\circ}C$ 的条件解冻，时间不超过 $15\mathrm{min}$ 。

# 5.3.3 系列稀释及培养

用 $1\mathrm{mL}$ 无菌吸管或微量移液器，制备10倍系列稀释样品匀液，于 $8000\mathrm{r / min}\sim 10000\mathrm{r / min}$ 均质 $1\mathrm{min}\sim 2\mathrm{min}$ ，或用拍击式均质器拍打 $1\mathrm{min}\sim 2\mathrm{min}$ 。每递增稀释一次，即换用1次 $1\mathrm{mL}$ 灭菌吸管或吸头。根据对样品浓度的估计，选择2个～3个适宜稀释度的样品匀液，在进行10倍递增稀释时，吸取 $1.0\mathrm{mL}$ 样品匀液于无菌平皿内，每个稀释度做两个平皿。同时，分别吸取 $1.0\mathrm{mL}$ 空白稀释液加入两个无菌平皿内作空白对照。及时将 $15\mathrm{mL}\sim 20\mathrm{mL}$ 冷却至 $46^{\circ}C$ 的双歧杆菌琼脂培养基或MRS琼脂培养基(可放置于 $46^{\circ}C\pm 1^{\circ}C$ 恒温水浴箱中保温)倾注平皿，并转动平皿使其混合均匀。从样品稀释到平板倾注要求在 $15\mathrm{min}$ 内完成。待琼脂凝固后，将平板翻转， $36^{\circ}C\pm 1^{\circ}C$ 厌氧培养 $48\mathrm{h}\pm 2\mathrm{h}$ ，可延长至 $72\mathrm{h}\pm 2\mathrm{h}$ 。培养后计数平板上的所有菌落数。

# 5.3.4 菌落计数

5.3.4.1 可用肉眼观察，必要时用放大镜或菌落计数器，记录稀释倍数和相应的菌落数量。菌落计数以菌落形成单位(colony-forming units, CFU)表示。  
5.3.4.2 选取菌落数在 $30\mathrm{CFU} \sim 300\mathrm{CFU}$ 之间、无蔓延菌落生长的平板计数菌落总数。低于 $30\mathrm{CFU}$ 的平板记录具体菌落数，大于 $300\mathrm{CFU}$ 的可记录为多不可计。每个稀释度的菌落数应采用两个平板的平均数。  
5.3.4.3 其中一个平板有较大片状菌落生长时，则不宜采用，而应以无片状菌落生长的平板作为该稀释度的菌落数；若片状菌落不到平板的一半，而其余一半中菌落分布又很均匀，即可计算半个平板后乘以2，代表一个平板菌落数。  
5.3.4.4 当平板上出现菌落间无明显界线的链状生长时，则将每条单链作为一个菌落计数。

# 5.3.5 结果的表述

5.3.5.1 若只有一个稀释度平板上的菌落数在适宜计数范围内，计算两个平板菌落数的平均值，再将平均值乘以相应稀释倍数，作为每克或每毫升中菌落总数结果。  
5.3.5.2 若有两个连续稀释度的平板菌落数在适宜计数范围内时，按式(1)计算：

$$
N = \frac {\sum C}{\left(n _ {1} + 0 . 1 n _ {2}\right) d} \tag {1}
$$

式中：

$N$ ——样品中菌落数；

$\sum C$ ——平板（含适宜范围菌落数的平板）菌落数之和；

$n_1$ ——第一稀释度（低稀释倍数）平板个数；

$n_2$ ——第二稀释度（高稀释倍数）平板个数；

$d$ ——稀释因子（第一稀释度）。

5.3.5.3 若所有稀释度的平板上菌落数均大于 $300\mathrm{CFU}$ ，则对稀释度最高的平板进行计数，其他平板可

记录为多不可计，结果按平均菌落数乘以最高稀释倍数计算。

5.3.5.4 若所有稀释度的平板菌落数均小于30CFU，则应按稀释度最低的平均菌落数乘以稀释倍数计算。  
5.3.5.5 若所有稀释度(包括液体样品原液)平板均无菌落生长，则以小于1乘以最低稀释倍数计算。  
5.3.5.6 若所有稀释度的平板菌落数均不在 $30\mathrm{CFU} \sim 300\mathrm{CFU}$ 之间，其中一部分小于 $30\mathrm{CFU}$ 或大于 $300\mathrm{CFU}$ 时，则以最接近 $30\mathrm{CFU}$ 或 $300\mathrm{CFU}$ 的平均菌落数乘以稀释倍数计算。

# 5.3.6 菌落数的报告

5.3.6.1 菌落数小于100CFU时，按“四舍五入”原则修约，以整数报告。  
5.3.6.2 菌落数大于或等于100CFU时，第3位数字采用“四舍五入”原则修约后，取前2位数字，后面用0代替位数；也可用10的指数形式来表示，按“四舍五入”原则修约后，采用两位有效数字。  
5.3.6.3 称重取样以 $\mathrm{CFU / g}$ 为单位报告，体积取样以 $\mathrm{CFU / mL}$ 为单位报告。

# 5.4 结果与报告

根据5.2.3.1、5.2.3.2、5.2.3.3结果，报告双歧杆菌属的种名。根据5.3.6菌落计数结果出具报告，报告单位以 $\mathrm{CFU / g(mL)}$ 表示。

# A.1.2 制法

# 附录B

# 双歧杆菌的有机酸代谢产物检测方法

# B.1 双歧杆菌培养液制备

挑取双歧杆菌琼脂平板或MRS琼脂平板上纯培养的双歧杆菌接种于PYG液体培养基，同时用未接种菌的PYG液体培养基做空白对照，厌氧， $36^{\circ}\mathrm{C}\pm 1$ ℃培养 $48\mathrm{h}$

# B.2 标准液的配制

B.2.1 乙酸标准溶液：准确吸取分析纯冰乙酸 $5.7\mathrm{mL}$ ，加水稀释至 $100.0\mathrm{mL}$ ，摇匀，进行标定，配成约 $1.0\mathrm{mol} / \mathrm{L}$ 的乙酸标准溶液。标定方法为：准确称取乙酸 $3.0\mathrm{g}$ ，加水 $15.0\mathrm{mL}$ ，酚酞指示液2滴，用 $1.0\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液滴定，并将滴定结果用空白试验校正。 $1.0\mathrm{mL}1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液相当于 $60.05\mathrm{mg}$ 的乙酸。  
B.2.2 乙酸使用液：将经标定的乙酸标准溶液用水稀释至 $20.0\mathrm{mmol / L}$   
B.2.3 乳酸标准溶液：吸取分析纯乳酸 $8.4\mathrm{mL}$ ，加水稀释至 $100.0\mathrm{mL}$ ，摇匀，进行标定，配成约 $1.0\mathrm{mol} / \mathrm{L}$ 的乳酸标准溶液。标定方法为：准确称取乳酸 $1.0\mathrm{g}$ ，加水 $50.0\mathrm{mL}$ ，加入 $1\mathrm{mol} / \mathrm{mL}$ 氢氧化钠滴定液 $25.0\mathrm{mL}$ ，煮沸 $5\mathrm{min}$ ，加入酚酞指示液2滴，同时用 $0.5\mathrm{mol} / \mathrm{mL}$ 硫酸滴定液滴定，并将滴定结果用空白试验校正。 $1.0\mathrm{mL}1\mathrm{mol} / \mathrm{mL}$ 氢氧化钠溶液相当于 $90.08\mathrm{mg}$ 的乳酸。  
B.2.4 乳酸使用液：将乳酸标准溶液用水稀释至 $20.0\mathrm{mmol / L}$

# B.3 方法

# B.3.1 乙酸的处理

取双歧杆菌培养液 $2.0\mathrm{mL}\sim 3.0\mathrm{mL}$ 放入 $10~\mathrm{mL}$ 离心管中，加入 $0.2\mathrm{mL}50\%$ 硫酸溶液（体积分数），混匀，加入 $2.0\mathrm{mL}$ 丙酮，混匀后加过量氯化钠，剧烈振摇 $1\mathrm{min}$ ，再加入 $2.0\mathrm{mL}$ 乙醚，振摇 $1\mathrm{min}$ 后，于 $3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将上清液转入另一试管中，下层溶液用 $2.0\mathrm{mL}$ 丙酮和 $2.0\mathrm{mL}$ 乙醚重复提取2次，合并有机相，于 $40^{\circ}\mathrm{C}$ 水浴中用氮气吹至尽干，用 $20\mathrm{mmol / L}$ 的磷酸二氢钠溶液 $(\mathrm{pH}2.0)$ -乙腈 $(99 + 1)$ 溶解并定容至 $1.0~\mathrm{mL}$ ，混匀后备用。同样操作步骤处理乙酸标准和空白培养液。

# B.3.2 乳酸的处理

取双歧杆菌培养液 $2.0\mathrm{mL}\sim 3.0\mathrm{mL}$ 放入 $10~\mathrm{mL}$ 比色管中， $100^{\circ}C$ 水浴 $10\mathrm{min}$ ，加入 $0.2\mathrm{mL}50\%$ （体积分数）硫酸溶液，混匀，加入 $1.0\mathrm{mL}$ 甲醇，于 $58^{\circ}C$ 水浴 $30\mathrm{min}$ 后加水 $1.0~\mathrm{mL}$ ，加三氯甲烷 $1.0\mathrm{mL}$ ，振摇 $3\mathrm{min},3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取三氯甲烷层分析。同样操作步骤处理乳酸标准和空白培养液。

# B.3.3 液相色谱条件

色谱柱：ZorbaxSb-Aq液相色谱柱 $(4.6\times 150\mathrm{mm},5\mu \mathrm{m})$ 或其他等效色谱柱；流动相： $20~\mathrm{mmol / L}$ 的磷酸二氢钠溶液(pH2.0)-乙腈(99+1)，等度洗脱，流速 $1\mathrm{mL} / \mathrm{min}$ ；柱温箱： $35^{\circ}C$ ；紫外检测波长： $210~\mathrm{nm}$ ；外标定量。

# B.4 结果计算

$$
X = \frac {A _ {\text {样}} - A _ {\text {空}}}{A _ {\text {标}}} \times c \quad \dots \tag {B.1}
$$

式中：

$X$ ——样品培养液中乙酸或乳酸的含量，单位为微摩尔每毫升 $(\mu \mathrm{mol} / \mathrm{mL})$ ；

$A_{\text{样}}$ ——样品培养液中乙酸或乳酸的峰面积；

$A_{\text{空}}$ ——空白培养液中乙酸或乳酸的峰面积；

$A_{\text{标}}$ ——乙酸标准或乳酸标准的峰面积；

$c$ ——乙酸标准或乳酸标准的浓度，单位为微摩尔每毫升 $(\mu \mathrm{mol} / \mathrm{mL})$ 。

# B.5 允许差

相对相差 $\leqslant 15\%$

# B.6 结果判定

如果乙酸 $(\mu \mathrm{mol} / \mathrm{mL})$ 与乳酸 $(\mu \mathrm{mol} / \mathrm{mL})$ 比值大于1，可判定为是双歧杆菌的有机酸代谢产物。

# 食品安全国家标准

# 食品添加剂 肉桂酸

# 1 范围

本标准适用于由苯甲醛或肉桂醛为原料制得的食品添加剂肉桂酸。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

3-苯基丙烯酸

# 2.2 分子式

$\mathrm{C_9H_8O_2}$

# 附录A

# 肉桂酸含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T 11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A. 1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

试样制备：称取本品 $1\mathrm{g}$ 溶于 $5\mathrm{mL}$ 无水乙醇中，微热，摇匀备用。

# A. 3 重复性及结果表示

按GB/T11538—2006中11.4规定执行。

食品添加剂肉桂酸典型气相色谱图(面积归一化法)参见附录B。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$   
B.2.2 固定相：聚乙二醇。  
B.2.3 膜厚： $0.25\mu \mathrm{m}$   
B.2.4 色谱炉温度： $240^{\circ}C$ 恒温。  
B.2.5 进样口温度： $240^{\circ}C$   
B.2.6 检测器温度： $240^{\circ}C$   
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 载气流速： $1.5\mathrm{mL / min}$   
B.2.10 进样量： $0.2\mu \mathrm{L}$   
B.2.11 分流比： $1 / 40$

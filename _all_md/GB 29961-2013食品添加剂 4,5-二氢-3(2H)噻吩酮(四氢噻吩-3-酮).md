# 食品安全国家标准

# 食品添加剂 4,5-二氢-3(2H)噻吩酮（四氢噻吩-3-酮）

# 1 范围

本标准适用于由巯基乙酸和丙烯酸为原料制得的食品添加剂4,5-二氢-3(2H)噻吩酮(四氢噻吩-3-酮)。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

4,5-二氢-3(2H)噻吩酮

# 2.2 分子式

$\mathrm{C_4H_6OS}$

# 2.3 结构式

![](images/9722a19cae976c0883774b55175e6b6d1318282890c45cbc960e4d37767f72eb.jpg)

# 2.4 相对分子质量

102.15（按2007年国际相对原子质量）

# 附录A

# 4,5-二氢-3(2H)噻吩酮含量的测定

# A. 1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。

A.1.2 柱：毛细管柱。

A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

# A. 3 重复性及结果表示

按GB/T11538—2006中11.4规定进行，应符合要求。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ 。  
B.2.2 固定相：聚乙二醇。  
B.2.3 膜厚： $0.25\mu \mathrm{m}$   
B.2.4 色谱炉温度： $130\,^{\circ}\mathrm{C}$ 恒温 $10\, \mathrm{min}$ ；然后线性程序升温从 $130\,^{\circ}\mathrm{C} \sim 200\,^{\circ}\mathrm{C}$ ，速率 $10\,^{\circ}\mathrm{C} / \mathrm{min}$ ；最后在 $200\,^{\circ}\mathrm{C}$ 恒温 $20\, \mathrm{min}$ 。  
B.2.5 进样口温度： $240^{\circ}C$ 。  
B.2.6 检测器温度： $240^{\circ}C$ 。  
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 柱前压： $60\mathrm{kPa}$   
B.2.10 进样量：约 $0.2\mu \mathrm{L}$   
B.2.11 分流比： $100:1$

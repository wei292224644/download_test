# 食品安全国家标准

# 食品添加剂 肉桂醇

# 1 范围

本标准适用于以肉桂醛为原料经化学反应制得的食品添加剂肉桂醇。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

3-苯基烯丙醇

# 2.2 分子式

$\mathrm{C_9H_{10}O}$

# 2.3 结构式

![](images/188776e388e1dbda665f414cce107355a60a8eb9ad345761440bf9a31fc3bd58.jpg)

# 2.4 相对分子质量

134.18（按2007年国际相对原子质量）

# 肉桂醇含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ 。  
B.2.2 固定相： $5\%$ 苯基甲基聚硅氧烷。  
B.2.3 膜厚: $0.25 \mu \mathrm{m}$ 。  
B.2.4 色谱炉温度： $100^{\circ}\mathrm{C}$ 恒温 $1\mathrm{min}$ ；然后线性程序升温从 $100^{\circ}\mathrm{C} \sim 250^{\circ}\mathrm{C}$ ，速率 $10^{\circ}\mathrm{C} / \mathrm{min}$ ；最后在 $250^{\circ}\mathrm{C}$ 恒温 $5\mathrm{min}$ 。  
B.2.5 进样口温度： $250^{\circ}C$ 。  
B.2.6 检测器温度： $250^{\circ}C$ 。  
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 载气流速： $1.2\mathrm{mL / min}$   
B.2.10 进样量：约 $0.25\mu \mathrm{L}$   
B.2.11 分流比： $80:1$

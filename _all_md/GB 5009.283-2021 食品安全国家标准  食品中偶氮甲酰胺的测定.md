# 食品安全国家标准

# 食品中偶氮甲酰胺的测定

# 1 范围

本标准规定了食品中偶氮甲酰胺的液相色谱测定方法。

本标准适用于小麦粉中偶氮甲酰胺的测定。

# 2 原理

试样用 $N,N-$ 二甲基甲酰胺溶液提取，离心，用三苯基磷衍生，衍生液经液相色谱测定，外标法定量。

# 3 试剂和材料

警示——三苯基膦和 $N, N$ -二甲基甲酰胺操作时应避免接触皮肤和眼睛，在通风环境中操作。

除非另有说明，本方法所有试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.2 $N,N$ -二甲基甲酰胺 $\mathrm{(C_3H_7NO)}$   
3.1.3 无水硫酸镁（ $\mathrm{MgSO_4}$ ）。  
3.1.4 三苯基膦 $(\mathrm{C_{18}H_{15}P})$ 。

# 3.2 试剂配制

三苯基膦衍生溶液：准确称取三苯基膦 $100\mathrm{mg}$ ，用 $N,N-$ 二甲基甲酰胺溶解并定容至 $10~\mathrm{mL}$ ，配制成浓度为 $10.0\mathrm{mg / mL}$ 的衍生溶液，存放于避光处备用。临用前配制。

# 3.3 标准品

偶氮甲酰胺标准品 $\mathrm{(C_2H_4N_4O_2}$ ，CAS：123-77-3)，纯度 $>97\%$ 。或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 标准储备溶液：准确称取偶氮甲酰胺标准品 $50\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ），用 $N,N-$ 二甲基甲酰胺溶解并定容至 $500~\mathrm{mL}$ ，混匀，配制成浓度为 $100~\mu \mathrm{g / mL}$ 的标准储备溶液，将溶液转移至棕色玻璃容器中， $-20^{\circ}C$ 下避光保存备用，有效期 $20\mathrm{d}$ 。  
3.4.2 空白基质提取溶液：使用空白小麦粉按照5.2方法进行试样处理，得到空白基质提取溶液。空白小麦粉中偶氮甲酰胺的含量应低于本方法检出限。

3.4.3 标准系列工作溶液: 分别吸取 $100 \mu \mathrm{g} / \mathrm{mL}$ 的标准储备液 $50.0 \mu \mathrm{L}, 100 \mu \mathrm{L}, 200 \mu \mathrm{L}, 500 \mu \mathrm{L}, 1000 \mu \mathrm{L}$ 于一组 $10 \mathrm{~mL}$ 容量瓶中, 用空白基质提取溶液定容至刻度, 混匀, 配制成浓度分别为 $0.500 \mu \mathrm{g} / \mathrm{mL}, 1.00 \mu \mathrm{g} / \mathrm{mL}, 2.00 \mu \mathrm{g} / \mathrm{mL}, 5.00 \mu \mathrm{g} / \mathrm{mL}, 10.0 \mu \mathrm{g} / \mathrm{mL}$ 的标准系列工作液。临用前配制。

# 3.5 材料

有机微孔滤膜： $0.22\mu \mathrm{m}$ 。

# 4 仪器和设备

4.1 高效液相色谱仪：配有二极管阵列检测器或紫外检测器。  
4.2 分析天平：感量为 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$   
4.3 涡旋混合器。  
4.4 高速离心机：转速 $\geqslant 10000\mathrm{r / min}$   
4.5 可控温设备：可设定温度 $20^{\circ}\mathrm{C} \sim 30^{\circ}\mathrm{C}$ 。

# 5 分析步骤

# 5.1 试样制备

称取具代表性小麦粉样品 $200\mathrm{g}$ ，混合均匀，贮存于棕色广口瓶中，备用。

# 5.2 试样处理

称取 $2.50\mathrm{g}$ 试样，置于 $50~\mathrm{mL}$ 具塞离心管中，加入 $1\mathrm{g}$ 无水硫酸镁，混合均匀，准确加入 $25.0~\mathrm{mL}$ $N,N$ -二甲基甲酰胺，在涡旋振荡器上充分混合，振荡提取 $20\mathrm{min}$ ，静置， $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液待衍生。

# 5.3 衍生

# 5.3.1 试样溶液的衍生化

吸取 $1.00\mathrm{mL}$ 滤液，置于 $10~\mathrm{mL}$ 具塞离心管中，加入 $100\mu \mathrm{L}$ 三苯基麟衍生溶液，于涡旋混合器上充分混合， $20^{\circ}C\sim 30^{\circ}C$ 环境温度下，避光静置衍生 $12\mathrm{h}$ 以上（若环境温度达不到要求，可在 $20^{\circ}C\sim$ $30^{\circ}C$ 可控温设备中进行衍生反应）。衍生液过 $0.22\mu \mathrm{m}$ 有机微孔滤膜后，供液相色谱仪测定（待测液在 $20^{\circ}C\sim 30^{\circ}C$ 环境温度下 $60\mathrm{h}$ 之内测定）。

# 5.3.2 标准溶液的衍生化

吸取 $1.00\mathrm{mL}$ 不同浓度的标准系列工作溶液，按5.3.1操作步骤，与试样溶液同步进行衍生，制备标准溶液工作曲线。

# 5.4 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱 $(4.6\mathrm{mm}\times 250\mathrm{mm},5\mu \mathrm{m})$ 或性能相当者；  
b）流动相：A为水，B为甲醇，梯度洗脱程序见表1；  
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d) 柱温: $30^{\circ} \mathrm{C}$ ;

e) 检测波长: $230 \mathrm{~nm}$ ;   
f) 进样量: $10 \mu \mathrm{L}$ 。

# 5.5 标准曲线的制作

将衍生后的标准系列工作溶液分别注入高效液相色谱仪中，测定相应衍生物的峰面积，以标准系列工作溶液中偶氮甲酰胺的浓度为横坐标，以衍生物的峰面积的响应值为纵坐标，绘制标准曲线。偶氮甲酰胺标准溶液衍生物色谱图参见附录A中图A.1。

# 5.6 试样溶液的测定

将试样衍生液注入高效液相色谱仪中，得到偶氮甲酰胺衍生物的色谱峰面积，根据标准曲线得到试样溶液中偶氮甲酰胺的浓度。偶氮甲酰胺浓度应在标准曲线线性范围之内，超出线性范围则应使用 $N,N-$ 二甲基甲酰胺溶液稀释试样提取溶液，将空白基质提取溶液同步稀释后配制标准曲线，同步衍生，再进行分析。

# 6 分析结果的表述

试样中偶氮甲酰胺的含量按式(1)计算。

$$
X = \frac {c \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中偶氮甲酰胺的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——由标准曲线得出的样液中偶氮甲酰胺的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样提取溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克(g)；

1000——单位换算系数。

计算结果保留两位有效数字。

# 7 精密度

在重复条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当取样量为 $2.50\mathrm{g}$ 时，提取溶液体积为 $25.0~\mathrm{mL}$ 时，本方法检出限为 $2.0\mathrm{mg / kg}$ ，定量限为 $5.0\mathrm{mg / kg}$ 。

# 前言

本标准代替GB/T5009.34—2003《食品中亚硫酸盐的测定》。

本标准与GB/T5009.34—2003相比，主要修改如下：

——标准名称修改为“食品安全国家标准 食品中二氧化硫的测定”；  
——删除第一法和附录A；  
——原第二法蒸馏法改为滴定法。

# 食品安全国家标准

# 食品中二氧化硫的测定

# 1 范围

本标准规定了果脯、干菜、米粉类、粉条、砂糖、食用菌和葡萄酒等食品中总二氧化硫的测定方法。  
本标准适用于果脯、干菜、米粉类、粉条、砂糖、食用菌和葡萄酒等食品中总二氧化硫的测定。

# 2 原理

在密闭容器中对样品进行酸化、蒸馏，蒸馏物用乙酸铅溶液吸收。吸收后的溶液用盐酸酸化，碘标准溶液滴定，根据所消耗的碘标准溶液量计算出样品中的二氧化硫含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 盐酸（HCl）。  
3.1.2 硫酸 $\mathrm{(H_2SO_4)}$ 。  
3.1.3 可溶性淀粉 $\left[\left(\mathrm{C}_{6} \mathrm{H}_{10} \mathrm{O}_{5}\right)_{\mathrm{n}}\right]$ 。  
3.1.4 氢氧化钠(NaOH)。  
3.1.5 碳酸钠 $\mathrm{(Na_2CO_3)}$ 。  
3.1.6 乙酸铅 $\mathrm{(C_4H_6O_4Pb)}$ 。  
3.1.7 硫代硫酸钠 $\mathrm{(Na_2S_2O_3\cdot 5H_2O)}$ 或无水硫代硫酸钠 $\mathrm{(Na_2S_2O_3)}$ 。  
3.1.8 碘 $(\mathrm{I}_2)$ 。  
3.1.9 碘化钾(KI)。

# 3.2 试剂配制

3.2.1 盐酸溶液 $(1 + 1)$ ：量取 $50~\mathrm{mL}$ 盐酸，缓缓倾入 $50~\mathrm{mL}$ 水中，边加边搅拌。  
3.2.2 硫酸溶液 $(1 + 9)$ ：量取 $10\mathrm{mL}$ 硫酸，缓缓倾入 $90~\mathrm{mL}$ 水中，边加边搅拌。  
3.2.3 淀粉指示液 $(10\mathrm{g / L})$ ：称取 $1\mathrm{g}$ 可溶性淀粉，用少许水调成糊状，缓缓倾入 $100~\mathrm{mL}$ 沸水中，边加边搅拌，煮沸 $2\mathrm{min}$ ，放冷备用，临用现配。  
3.2.4 乙酸铅溶液 $(20\mathrm{g} / \mathrm{L})$ ：称取 $2\mathrm{g}$ 乙酸铅，溶于少量水中并稀释至 $100~\mathrm{mL}$ 。

# 3.3 标准品

重铬酸钾 $\mathrm{(K_2Cr_2O_7)}$ ，优级纯，纯度 $\geq 99\%$

# 3.4 标准溶液配制

3.4.1 硫代硫酸钠标准溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $25\mathrm{g}$ 含结晶水的硫代硫酸钠或 $16\mathrm{g}$ 无水硫代硫酸钠溶

于 $1000 \mathrm{~mL}$ 新煮沸放冷的水中，加入 $0.4 \mathrm{~g}$ 氢氧化钠或 $0.2 \mathrm{~g}$ 碳酸钠，摇匀，贮存于棕色瓶内，放置两周后过滤，用重铬酸钾标准溶液标定其准确浓度。或购买有证书的硫代硫酸钠标准溶液。

3.4.2 碘标准溶液 $\left[c(1 / 2\mathrm{I}_2) = 0.10\mathrm{mol} / \mathrm{L}\right]$ ：称取 $13\mathrm{g}$ 碘和 $35\mathrm{g}$ 碘化钾，加水约 $100~\mathrm{mL}$ ，溶解后加入3滴盐酸，用水稀释至 $1000~\mathrm{mL}$ ，过滤后转入棕色瓶。使用前用硫代硫酸钠标准溶液标定。  
3.4.3 重铬酸钾标准溶液 $\left[c\left(1 / 6\mathrm{K}_2\mathrm{Cr}_2\mathrm{O}_7\right) = 0.1000\mathrm{mol} / \mathrm{L}\right]$ ：准确称取 $4.9031\mathrm{g}$ 已于 $120^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 电烘箱中干燥至恒重的重铬酸钾，溶于水并转移至 $1000~\mathrm{mL}$ 量瓶中，定容至刻度。或购买有证书的重铬酸钾标准溶液。  
3.4.4 碘标准溶液 $[c(1/2\mathrm{I}_2) = 0.01000\mathrm{mol} / \mathrm{L}]$ ：将 $0.1000\mathrm{mol} / \mathrm{L}$ 碘标准溶液用水稀释10倍。

# 4 仪器和设备

4.1 全玻璃蒸馏器： $500~\mathrm{mL}$ ，或等效的蒸馏设备。  
4.2 酸式滴定管： $25~\mathrm{mL}$ 或 $50~\mathrm{mL}$   
4.3 剪切式粉碎机。  
4.4 碘量瓶： $500~\mathrm{mL}$

# 5 分析步骤

# 5.1 样品制备

果脯、干菜、米粉类、粉条和食用菌适当剪成小块，再用剪切式粉碎机剪碎，搅均匀，备用。

# 5.2 样品蒸馏

称取 $5\mathrm{g}$ 均匀样品（精确至 $0.001\mathrm{g}$ ，取样量可视含量高低而定），液体样品可直接吸取 $5.00\mathrm{mL} \sim 10.00\mathrm{mL}$ 样品，置于蒸馏烧瓶中。加入 $250\mathrm{mL}$ 水，装上冷凝装置，冷凝管下端插入预先备有 $25\mathrm{mL}$ 乙酸铅吸收液的碘量瓶的液面下，然后在蒸馏瓶中加入 $10\mathrm{mL}$ 盐酸溶液，立即盖塞，加热蒸馏。当蒸馏液约 $200\mathrm{mL}$ 时，使冷凝管下端离开液面，再蒸馏 $1\mathrm{min}$ 。用少量蒸馏水冲洗插入乙酸铅溶液的装置部分。同时做空白试验。

# 5.3 滴定

向取下的碘量瓶中依次加入 $10\mathrm{mL}$ 盐酸、 $1\mathrm{mL}$ 淀粉指示液，摇匀之后用碘标准溶液滴定至溶液颜色变蓝且 $30\mathrm{s}$ 内不褪色为止，记录消耗的碘标准滴定溶液体积。

# 6 分析结果的表述

试样中二氧化硫的含量按式(1)计算：

$$
X = \frac {\left(V - V _ {0}\right) \times 0 . 0 3 2 \times c \times 1 0 0 0}{\mathrm {m}} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中的二氧化硫总含量(以 $\mathrm{SO}_2$ 计)，单位为克每千克 $(\mathrm{g / kg})$ 或克每升 $(\mathrm{g / L})$ ；

$V$ ——滴定样品所用的碘标准溶液体积，单位为毫升（ $\mathrm{mL}$ ）；

$V_{0}$ ——空白试验所用的碘标准溶液体积，单位为毫升 $(\mathrm{mL})$ ；

0.032—1 mL 碘标准溶液 $[c(1/2\mathrm{I}_2) = 1.0\mathrm{mol} / \mathrm{L}]$ 相当于二氧化硫的质量，单位为克(g)；

$c$ ——碘标准溶液浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

m ——试样质量或体积，单位为克(g)或毫升(mL)。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，当二氧化硫含量 $\geqslant 1\mathrm{g / kg}$ (L)时，结果保留三位有效数字；当二氧化硫含量 $<  1\mathrm{g / kg(L)}$ 时，结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测试结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当取 $5\mathrm{g}$ 固体样品时，方法的检出限(LOD)为 $3.0~\mathrm{mg / kg}$ ，定量限为 $10.0~\mathrm{mg / kg}$ ；当取 $10~\mathrm{mL}$ 液体样品时，方法的检出限(LOD)为 $1.5~\mathrm{mg / L}$ ，定量限为 $5.0~\mathrm{mg / L}$ 。

# 1 范围

本标准适用于联合制碱法、直接中和法、复分解法或离子交换法生产的食品添加剂氯化铵。

# 2 分子式和相对分子质量

# 2.1 分子式

$\mathrm{NH_4Cl}$

# 2.2 相对分子质量

53.49（按2011年国际相对原子质量）

# 检验方法

# A.1 警示

本检验方法中使用的部分试剂具有腐蚀性，操作者应小心谨慎！如溅到皮肤上应立即用水冲洗，严重者应立即治疗。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 硝酸。

A.3.1.2 氨水溶液： $2 + 3$

A.3.1.3 硝酸银溶液： $42\mathrm{g / L}$

A.3.1.4 氢氧化钠溶液： $40~\mathrm{g / L}$

A.3.1.5 红色石蕊试纸。

# A.3.2 鉴别方法

# A.3.2.1 氯离子的鉴别

取约 $0.2\mathrm{g}$ 试样，加 $20~\mathrm{mL}$ 水溶解，加硝酸银溶液即产生白色沉淀，此沉淀不溶于硝酸，溶于过量的氨水溶液中。

# A.3.2.2 铵的鉴别

取约 $0.2\mathrm{g}$ 试样，加 $20~\mathrm{mL}$ 水溶解，加入氢氧化钠溶液，释放出有刺激味的气体，该气体可使湿润的红色石蕊试纸变蓝。

# A.4 氯化铵 $(\mathrm{NH}_4\mathrm{Cl})$ 含量（以干基计）的测定

# A.4.1 试剂和材料

A.4.1.1 氢氧化钠溶液： $400\mathrm{g / L}$

A.4.1.2 硫酸溶液： $c(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4) = 0.5\mathrm{mol / L}$

A.4.1.3 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.5\mathrm{mol / L}$

A.4.1.4 甲基红-亚甲基蓝混合指示液。  
A.4.1.5 广泛 $\mathrm{pH}$ 试纸。  
A.4.1.6 无氨的水。

# A.4.2 仪器和设备

A.4.2.1 蒸馏装置：按图 A.1 装备蒸馏装置，或使用其他具有相同蒸馏能力的定氮蒸馏仪，仪器的磨口连接处应涂硅脂密封。

单位为毫米

A.4.2.3 防爆沸颗粒：沸石或毛细玻璃管。

# A.4.3 分析步骤

称取约 $0.9\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于蒸馏瓶中。加 $300~\mathrm{mL}$ 无氨的水，摇动使试料溶解。加入几粒防爆沸颗粒，将蒸馏瓶连接于蒸馏装置上。在吸收瓶中用移液管移入 $50.0~\mathrm{mL}$ 硫酸溶液，加4滴～5滴甲基红-亚甲基蓝混合指示液，加适量水，将导流管出口位于吸收液液面下约 $1.5~\mathrm{cm}$ 。

从蒸馏瓶口加入 $20~\mathrm{mL}$ 氢氧化钠溶液，马上塞上防溅球管。开通冷却水，开启蒸馏加热装置，沸腾

时根据泡沫产生程度调节供热强度，避免泡沫溢出或液滴带出。蒸馏出至少 $150~\mathrm{mL}$ 馏出液后，用广泛 $\mathrm{pH}$ 试纸检查流出的液滴，如不呈碱性则结束蒸馏。用少量水冲洗导流管的下端，取下吸收瓶。

用氢氧化钠标准滴定溶液滴定吸收瓶中的吸收液，至溶液呈灰绿色为终点。

同时进行空白试验。空白试验除不加试样外其他操作及加入试剂的种类和量(标准滴定溶液除外)与测定试验相同。

# A.4.4 结果计算

氯化铵 $\mathrm{(NH_4Cl)}$ 含量（以干基计)的质量分数 $w_{1}$ 按式(A.1)计算：

$$
w _ {1} = \frac {\left(V _ {0} - V _ {1}\right) \times c \times M}{m \times \left(1 - w _ {2}\right) \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{0}$ ——空白试验所消耗的氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——滴定试验溶液所消耗的氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——氢氧化钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——氯化铵的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{NH_4Cl}) = 53.49]$

$m$ ——试样的质量，单位为克(g)；

$w_{2}$ ——按A.5测定的干燥减量的质量分数；

1000 ——换算因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A.5 干燥减量的测定

# A.5.1 仪器和设备

A.5.1.1 电热恒温干燥箱： $100^{\circ}\mathrm{C}\sim 105^{\circ}\mathrm{C}$

A.5.1.2 称量瓶： $\phi 50\mathrm{mm}\times 30\mathrm{mm}$

# A.5.2 分析步骤

称取约 $5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于预先于 $100^{\circ}\mathrm{C} \sim 105^{\circ}\mathrm{C}$ 的电热恒温干燥箱中干燥至质量恒定的称量瓶中，于 $100^{\circ}\mathrm{C} \sim 105^{\circ}\mathrm{C}$ 下干燥至质量恒定。

# A.5.3 结果计算

干燥减量的质量分数 $w_{2}$ 按式(A.2)计算：

$$
w _ {2} = \frac {m _ {1} - m _ {2}}{m} \times 100 \% \tag{A.2}
$$

式中：

$m_{1}$ —称量瓶和试样干燥前的质量，单位为克 $(\mathrm{g})$

$m_{2}$ —称量瓶和试样干燥后的质量，单位为克 $(\mathrm{g})$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.05\%$ 。

# A.6 灼烧残渣的测定

# A.6.1 仪器和设备

A.6.1.1 瓷蒸发皿：容积为 $50~\mathrm{mL}$   
A.6.1.2 高温炉：能控制温度为 $500^{\circ}\mathrm{C} \sim 600^{\circ}\mathrm{C}$ 。

# A.6.2 分析步骤

称取约 $10\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于预先于 $500^{\circ}\mathrm{C} \sim 600^{\circ}\mathrm{C}$ 的高温炉中灼烧至质量恒定的瓷蒸发皿中，置于电炉上加热升华，直至无白烟冒出为止，置于 $500^{\circ}\mathrm{C} \sim 600^{\circ}\mathrm{C}$ 的高温炉中灼烧至质量恒定。

# A.6.3 结果计算

灼烧残渣的质量分数 $w_{3}$ 按式(A.3)计算：

$$
w _ {3} = \frac {m _ {1} - m _ {2}}{m} \times 100 \% \tag{A.3}
$$

式中：

$m_{1}$ ——灼烧后瓷蒸发皿和残渣的质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——瓷蒸发皿的质量，单位为克 $(\mathrm{g})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.1\%$ 。

# A.7 澄清度试验

# A.7.1 试剂和材料

# A.7.1.1 浊度标准贮备液

称取 $1.00\mathrm{g}$ 已于 $105~^\circ \mathrm{C}$ 干燥至质量恒定的硫酸肼，加适量水溶解，必要时可在 $40~^\circ \mathrm{C}$ 的水浴中温热溶解，转移至 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀，放置 $4\mathrm{h}\sim 6\mathrm{h}$ 。取此溶液与等体积的六亚甲基四胺溶液 $(100\mathrm{g / L})$ 混合，摇匀，于 $25~^\circ \mathrm{C}\pm 1~^\circ \mathrm{C}$ 下避光静置 $24\mathrm{h}$ ，制得浊度标准储备液。该溶液放置于阴凉、避光处保存，使用期为2个月，使用前摇匀。

# A.7.1.2 浊度标准原液

用移液管移取 $15.0\mathrm{mL}$ 浊度标准贮备液，置 $1000~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。取适量溶液置于 $1\mathrm{cm}$ 比色皿中，使用可见分光光度计，以水调零，于 $550~\mathrm{nm}$ 波长处测定溶液的吸光度，其吸光度应在 $0.12\sim 0.15$ 之间。该溶液应在 $48\mathrm{h}$ 内使用，使用前摇匀。

# A.7.1.3 水

本试验所用的水均为新煮沸并放冷的GB/T6682规定的三级水。

# A.7.2 仪器和设备

比色管：容积 $100~\mathrm{mL}$ ，内径 $15  \mathrm{mm}\sim 16  \mathrm{mm}$ ，平底，无色，一套比色管的玻璃颜色和刻线高度应

相同。

# A.7.3 分析步骤

# A.7.3.1 浊度标准溶液的制备

用移液管移取 $5.0\mathrm{mL}$ 浊度标准原液，置于 $100~\mathrm{mL}$ 比色管中，用水稀释至刻度，摇匀，放置 $5\mathrm{min}$ 。该溶液现用现配，使用前充分摇匀。

# A.7.3.2 测定

称取 $10.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于比色管中，加适量水溶解，用水稀释至刻度，摇匀。在黑色背景上，于漫射光下，从水平方向观察比较，试样溶液所产生的浊度小于浊度标准溶液产生的浊度即为通过试验。

注：测定应在试样溶液配制后的 $5\mathrm{min}$ 内完成。

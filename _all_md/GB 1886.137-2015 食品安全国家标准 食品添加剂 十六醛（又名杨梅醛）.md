# 食品安全国家标准

# 食品添加剂 十六醛（又名杨梅醛）

# 1 范围

本标准适用于以苯乙酮、 $\alpha$ -氯乙酸乙酯为原料经化学反应制得的食品添加剂十六醛（又名杨梅醛或草莓醛）。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

2,3-环氧-3-甲基-3-苯基丙酸乙酯或3-甲基-3-苯基缩水甘油酸乙酯

# 2.2 分子式

$$
\mathrm {C} _ {1 2} \mathrm {H} _ {1 4} \mathrm {O} _ {3}
$$

# 2.3 结构式

![](images/27450bc63bf55b2b28265b4b62af957d78e9d8379f9b33c5269d83441575e53f.jpg)

# 2.4 相对分子质量

206.24（按2007年国际相对原子质量）

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。

面积归一化法：按GB/T11538—2006中10.4测定含量。

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.53\mathrm{mm}$   
B.2.2 固定相： $100\%$ 二甲基聚硅氧烷。  
B.2.3 膜厚： $1.5\mu \mathrm{m}$   
B.2.4 色谱炉温度： $160~\mathrm{^\circ C}$ 恒温 $10\mathrm{min}$ ，然后线性程序升温从 $160~\mathrm{^\circ C}\sim 250~\mathrm{^\circ C}$ ，速率 $5\mathrm{^\circ C / min}$ 。  
B.2.5 进样口温度： $250^{\circ}C$   
B.2.6 检测器温度： $250^{\circ}C$   
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 载气流速： $7\mathrm{mL / min}$   
B.2.10 进样量：约 $0.2\mu \mathrm{L}$   
B.2.11 分流比： $60:1$

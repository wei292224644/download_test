# 1 范围

本文件规定了禽蛋中青霉素V、青霉素G、氨苄西林、氯唑西林、阿莫西林、头孢氨苄、头孢喹肟残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于禽蛋中青霉素V、青霉素G、氨苄西林、氯唑西林、阿莫西林、头孢氨苄、头孢喹肟残留量的测定。

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中残留的青霉素 $X$ 、青霉素G、氨苄西林、氯唑西林、阿莫西林、头孢氨苄、头孢喹肟，经 $80\%$ 乙腈水溶液提取，固相萃取柱净化浓缩液相色谱-串联质谱测定，是质匹配标准溶液内标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 甲酸（HCOOH）：色谱纯。  
5.1.3 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯

# 5.2 溶液配制

5.2.1 $50\%$ 乙腈水溶液：取乙腈 $50~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$   
5.2.2 $80\%$ 乙腈水溶液：取乙腈 $80~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ 。  
5.2.3 $0.1\%$ 甲酸乙腈溶液：取甲酸 $1\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$   
5.2.4 $0.1\%$ 甲酸水溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$

# 5.3 标准品

青霉素V、青霉素G、氨苄西林、氯唑西林、阿莫西林、头孢氨苄、头孢喹肟、青霉素 $\mathrm{V - D_5}$ 、青霉素- $\mathrm{D}_7$ 、氨苄西林- $\mathrm{D}_5$ 、氯唑西林- $^{13}\mathrm{C}_4$ 、阿莫西林- $\mathrm{D}_4$ 、头孢氨苄- $\mathrm{D}_5$ 、头孢喹肟- $\mathrm{D}_7$ 标准品，含量均 $\geqslant 95\%$ ，具体见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液：取青霉素V、青霉素G、氨苄西林、氯唑西林、阿莫西林、头孢氨苄、头孢喹肟标准品各适量（相当于有效成分约 $10\mathrm{mg}$ ），精密称定，分别用 $50\%$ 乙腈水溶解并稀释定容于 $100~\mathrm{mL}$ 容量瓶中，配

制成浓度为 $100\mu \mathrm{g / mL}$ 的标准储备液， $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1个月。

5.4.2 内标储备液：取青霉素 $\mathrm{V - D_5}$ 、青霉素- $\mathrm{D}_7$ 、氨苄西林- $\mathrm{D}_5$ 、氯唑西林- ${}^{13}\mathrm{C}_4$ 、阿莫西林- $\mathrm{D}_4$ 、头孢氨苄 $\mathrm{D}_5$ 、头孢喹肟- $\mathrm{D}_7$ 标准品各适量(相当于有效成分约 $10~\mathrm{mg})$ ，精密称定，分别用 $50\%$ 乙腈水溶解并稀释定容于 $100~\mathrm{mL}$ 容量瓶中，配制成浓度为 $100~\mu \mathrm{g / mL}$ 的内标储备液， $2^{\circ}\mathrm{C}\sim 8^{\circ}\mathrm{C}$ 保存，有效期1个月。  
5.4.3 混合标准工作液：分别精密量取 $100\mu \mathrm{g / mL}$ 的青霉素V、青霉素G、氨苄西林、氯唑西林、阿莫西林、头孢氨苄、头孢喹肟标准储备液各 $0.1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 容量瓶中，用 $50\%$ 乙腈水稀释至刻度，配制成浓度为 $100\mu \mathrm{g} / \mathrm{L}$ 的混合标准工作液，临用现配。  
5.4.4 混合内标工作液：分别精密量取 $100\mu \mathrm{g / mL}$ 的青霉素V- $\mathbf{D}_5$ 、青霉素- $\mathbf{D}_7$ 、氨苄西林- $\mathbf{D}_5$ 、氯唑西林- ${}^{13}\mathrm{C}_4$ 、阿莫西林- $\mathbf{D}_4$ 、头孢氨苄- $\mathbf{D}_5$ 、头孢喹肟- $\mathbf{D}_7$ 内标储备液各 $0.1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 容量瓶中，用 $50\%$ 乙腈水稀释至刻度，配制成浓度为 $100\mu \mathrm{g / L}$ 的混合内标工作液，临用现配。

# 5.5 材料

5.5.1固相萃取柱：反相混合型亲水亲脂平衡共聚物固相萃取柱， $200\mathrm{mg / 6mL}$ ，或相当者。  
5.5.2 微孔尼龙滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量分别为 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
6.3 涡旋混合器。  
6.4 振荡器。  
6.5 冷冻离心机：转速 $10000\mathrm{r / min}$   
6.6 氮吹仪。  
6.7 固相萃取装置。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或冷藏的空白或供试禽蛋，去壳后混合均匀。

a）取均质的供试样品，作为供试试样；  
b）取均质的空白样品，作为空白试样；  
c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取禽蛋试料 $(2 \pm 0.05)$ g, 置于 $50 \mathrm{~mL}$ 塑料离心管, 添加混合内标工作液 $100 \mu \mathrm{L}$ , 加 $80 \%$ 乙腈水溶液 $6.0 \mathrm{~mL}$ , 涡旋混匀 $1 \mathrm{~min}$ , 中速振荡提取 $5 \mathrm{~min}, 4^{\circ} \mathrm{C} 10000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ , 取上清液备用。

# 8.2 净化

将固相萃取柱安装到预先清洁过的真空萃取装置上，直接取上清液 $3.0\mathrm{mL}$ 通过小柱，保持自然流速，收集全部流出液。 $40^{\circ}\mathrm{C}$ 水浴氮吹至近干，加入水 $1.0\mathrm{mL}$ 溶解残余物，涡旋混匀，过 $0.22\mu \mathrm{m}$ 微孔滤膜，供液相色谱-串联质谱测定。

# 8.3 基质匹配标准曲线的制备

精密量取混合标准工作液和混合内标工作液适量，用水稀释配制成青霉素V、青霉素G、氨苄西林、氯唑西林、阿莫西林、头孢氨苄、头孢喹肟浓度为 $1\mu \mathrm{g} / \mathrm{L},2\mu \mathrm{g} / \mathrm{L},5\mu \mathrm{g} / \mathrm{L},10\mu \mathrm{g} / \mathrm{L},20\mu \mathrm{g} / \mathrm{L},50\mu \mathrm{g} / \mathrm{L}$ ，内标

浓度为 $5\mu \mathrm{g} / \mathrm{L}$ 的系列标准工作溶液，从中各取 $1.0~\mathrm{mL}$ 分别加入6份经8.1提取和8.2净化的空白试料浓缩液中，充分溶解混匀为基质匹配标准工作溶液，临用现配，微孔滤膜过滤，供液相色谱-串联质谱测定。以测试药物和其对应内标的特征离子峰面积之比为纵坐标、标准溶液浓度为横坐标绘制标准曲线，求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在相同测试条件下，试料溶液中 $\beta$ -内酰胺类药物与其相应内标的保留时间之比与基质匹配标准溶液中 $\beta$ -内酰胺类药物与其相应内标的保留时间之比偏差在 $1\%$ 以内；且检测到的相对离子丰度应与浓度相当的基质匹配标准溶液相对离子丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.4.3.2 定量测定

取试料溶液和基质匹配标准溶液，做单点或多点校准，按内标法定量。基质匹配标准溶液及试料溶液中目标物的响应值均应在仪器检测的线性范围内。在上述液相色谱-串联质谱条件下， $\beta$ -内酰胺类药物标准溶液特征离子质量色谱图见附录B。

# 8.5 空白试验

取空白试样，除不加药物外，采用完全相同的测定步骤进行平行测定。

# 9 结果计算和表述

试样中 $\beta$ -内酰胺类药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A _ {i} \times A ^ {\prime} {} _ {i s} \times C _ {s} \times C _ {i s} \times V \times V _ {2}}{A _ {i s} \times A _ {s} \times C ^ {\prime} {} _ {i s} \times V _ {1} \times m} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中 $\beta$ -内酰胺类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A_{i}$ ——试料溶液中 $\beta$ -内酰胺类药物的峰面积；

$\Lambda^{\prime}_{is}$ ——基质匹配标准溶液中 $\beta$ 内酰胺类药物内标的峰面积；

$C_{s}$ ——基质匹配标准溶液中 $\beta$ -内酰胺类药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{is}$ ——试料溶液中 $\beta$ -内酰胺类药物内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——溶解残余物所用溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——提取液总体积的数值，单位为毫升 $(\mathrm{mL})$

$A_{is}$ ——试料溶液中 $\beta$ -内酰胺类药物内标的峰面积；

$A_{s}$ ——基质匹配标准溶液中 $\beta$ -内酰胺类药物的峰面积；

$C_{is}^{\prime}$ ——基质匹配标准溶液中 $\beta$ 内酰胺类药物内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V_{1}$ ——用于净化过柱的提取液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——供试试料质量的数值，单位为克 $(\mathrm{g})$ ；

# 10 方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法青霉素V、青霉素G、氨苄西林、氯唑西林、阿莫西林、头孢氨苄、头孢喹肟的检出限为 $1\mu \mathrm{g / kg}$ ，定量限为 $2\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法青霉素V、青霉素G、氨苄西林、氯唑西林、阿莫西林、头孢氨苄、头孢喹肟在 $2\mu \mathrm{g} / \mathrm{kg}\sim 20~\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

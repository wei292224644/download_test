# 水产品中青霉素类药物多残留的测定

# 高效液相色谱法

# 1 范围

本标准规定了水产品中4种青霉素类药物残留量检测的制样和高效液相色谱测定方法。

本标准适用于鱼可食性组织中青霉素G、苯唑西林、双氯青霉素和乙氧萘青霉素单个或多个药物残留量的检测。

# 3原理

试料中残留的青霉素类药物，用乙腈提取，HLB 柱净化，1,2,4-三氮唑和氯化汞溶液衍生，高效液相色谱-紫外测定，外标法定量。

# 4 试剂与材料

以下所用的试剂，除特别注明外均为分析纯试剂，水符合GB/T6682规定的一级水。

4.1 青霉素G、苯唑西林、双氯青霉素和乙氧萘青霉素标准品：含量 $\geq 98\%$   
4.2 甲醇  
4.3 乙腈  
4.4 五水硫代硫酸钠  
4.5 二水磷酸二氢钠

4.6 无水磷酸氢二钠  
4.7 硫酸铵   
4.8 氢氧化钠  
4.9 正己烷  
4.10 1,2,4-三氮唑   
4.11 氯化汞（II）  
4.12 HLB固相萃取柱： $60\mathrm{mg} / 3\mathrm{mL}$ ，或相当者。  
4.13 流动相 A: 取五水硫代硫酸钠 $3.9 \mathrm{~g}$ , 用水溶解, 再加入二水磷酸二氢钠 $10.2 \mathrm{~g}$ 、无水磷酸氢二钠 $4.9 \mathrm{~g}$ 、硫酸铵 $6.8 \mathrm{~g}$ , 用水溶解, 并稀释至 $1000 \mathrm{~mL}$ , 滤膜过滤。  
4.14 氯化汞（II）溶液：取氯化汞（II） $0.27\mathrm{g}$ ，用水溶解并稀释至 $10\mathrm{mL}$ ，现配现用。  
4.15 $5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：取氢氧化钠 $20\mathrm{g}$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。  
4.16 衍生化试剂: 取 1,2,4-三氮唑 $13.78 \mathrm{~g}$ , 用水 $60 \mathrm{~mL}$ 溶解, 加氯化汞溶液 $10 \mathrm{~mL}$ , 用 $5 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液调节 $\mathrm{pH}$ 至 9.0, 用水稀释到 $100 \mathrm{~mL} 。2 \sim 8^{\circ} \mathrm{C}$ 避光保存, 有效期 3 个月。  
4.17 $1 \mathrm{mg} / \mathrm{mL}$ 青霉素类药物标准贮备液：精密称取青霉素 G、苯唑西林、双氯青霉素和乙氧萘青霉素的标准品各 $10 \mathrm{mg}$ ，分别于 $10 \mathrm{~mL}$ 量瓶中，用水溶解并稀释至刻度，配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的青霉素 G、苯唑西林、双氯青霉素和乙氧萘青霉素标准贮备液，分装，立刻于 $-20^{\circ} \mathrm{C}$ 以下避光保存，有效期 1 个月。使用时不得反复冻融。  
4.18 $10 \mu \mathrm{g} / \mathrm{mL}$ 青霉素 G 标准工作液：精密量取 $1 \mathrm{mg} / \mathrm{mL}$ 青霉素 G 标准贮备液 $1.0 \mathrm{~mL}$ ，于 $100 \mathrm{~mL}$ 量瓶中，用水溶解并稀释至刻度，配制成浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 的青霉素 G 标准工作液，分装，立刻于 $-20^{\circ} \mathrm{C}$ 以下避光保存，有效期 1 个月。使用时不得反复冻融。  
4.19 $10 \mu \mathrm{g} / \mathrm{mL}$ 苯唑西林、双氯青霉素和乙氧萘青霉素混合标准工作液：精密量取 $1 \mathrm{mg} / \mathrm{mL}$ 苯唑西林、双氯青霉素和乙氧萘青霉素标准贮备液各 $1.0 \mathrm{~mL}$ ，于 $100 \mathrm{~mL}$ 量瓶中，用水溶解并稀释至刻度，配制成浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 的苯唑西林、双氯青霉素和乙氧萘青霉素混合标准工作液，分装，立刻于 $-20^{\circ} \mathrm{C}$ 以下避光保存，有效期1个月。使用时不得反复冻融。

5.1 高效液相色谱仪：配紫外检测器。

5.3 冷冻高速离心机

5.5 旋涡混合器  
5.6 固相萃取装置  
5.7 组织匀浆机  
5.8 氮气吹干装置  
5.9 离心管： $50~\mathrm{mL}$   
5.10 滤膜： $0.45\mu \mathrm{m}$

# 6 试料的制备与保存

# 6.1 试料的制备

鱼去鳞、去皮, 沿背部取肌肉组织, 或取鱼肌肉+皮组织, 绞碎, 并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-20°C以下保存。

# 7 测定步骤

# 7.1 标准曲线的制备

分别精密量取 $10\mu \mathrm{g / mL}$ 青霉素G标准工作液及 $10~\mu \mathrm{g / mL}$ 苯唑西林、双氯青霉素和乙氧萘青霉素混合标准工作液适量，用流动相稀释，配制成青霉素G浓度为20、50、100、200、400、800和1600 $\mu \mathrm{g} / \mathrm{L}$ ，苯唑西林、双氯青霉素和乙氧萘青霉素浓度为100、200、400、800、1600、3200和 $6400~\mu \mathrm{g / L}$ 的系列青霉素类药物混合标准溶液，各取 $500~\mu \mathrm{L}$ ，按衍生化步骤操作，供高效液相色谱测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.2 提取

称取试料（ $5 \pm 0.05$ ）g，于 $50 \mathrm{~mL}$ 离心管中，加乙腈 $15 \mathrm{~mL}$ ，混匀，加正己烷 $5 \mathrm{~mL}$ ，漩涡混匀， $6000 \mathrm{r} / \mathrm{min}$ 离心 $15 \mathrm{~min}$ ，弃正己烷层液，取下层液，于另一离心管中。残渣中再加乙腈 $10 \mathrm{~mL}$ 、正己烷 $5 \mathrm{~mL}$ ，漩涡混匀， $6000 \mathrm{r} / \mathrm{min}$ 离心 $15 \mathrm{~min}$ ，弃正己烷层液，合并两次下层液，于 $40^{\circ} \mathrm{C}$ 水浴旋转蒸干，用水 $5 \mathrm{~mL}$ 溶解残余物，备用。

# 7.3 净化

HLB柱依次用甲醇 $3\mathrm{mL}$ 和水 $3\mathrm{mL}$ 活化，取备用液过柱，用水 $1\mathrm{mL}$ 淋洗，乙腈 $3\mathrm{mL}$ 洗脱，收集洗脱液，于 $45\sim 50^{\circ}C$ 水浴氮气吹干。用流动相 $1.0\mathrm{mL}$ 溶解残余物，旋涡混匀，备用。

# 7.4衍生化

准确量取备用液 $500 \mu \mathrm{L}$ , 于 $1.5 \mathrm{~mL}$ 聚丙烯离心管中, 加衍生化试剂 $500 \mu \mathrm{L}$ , 旋涡混匀, 于 $65^{\circ} \mathrm{C}$ 水浴反应 $10 \mathrm{~min}$ , 快速冰浴冷却, 于 $4^{\circ} \mathrm{C} 10000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ , 取上清液, 供高效液相色谱测定。

# 7.5 测定

# 7.5.1 色谱条件

色谱柱： $\mathrm{C_{18}}$ （ $250\mathrm{mm}\times 4.6\mathrm{mm}$ ，粒径 $5\mu \mathrm{m})$ ，或相当者。

流动相：流动相 $\mathrm{A} +$ 乙腈（ $65 + 35$ ， $\mathrm{v / v})$ 。

流速： $1\mathrm{mL / min}$

紫外检测波长： $325\mathrm{nm}$

柱温： $30^{\circ}\mathrm{C}$ 。

进样量： $50~\mu \mathrm{L}$

# 7.5.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试样溶液中青霉素类药物响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和空白添加试样溶液的高效液相色谱图见附录A。

# 7.5 空白试验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算和表述

单点校准

试料中青霉素类药物的残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算

$$
X = \frac {A \times C S \times V}{A S \times m}
$$

式中：

$X$ ——供试试料中相应的青霉素类药物残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

A—试样溶液中相应的青霉素类药物衍生物峰面积；

$A_{S}$ ——标准工作液中相应的青霉素类药物衍生物峰面积；

$\mathrm{C}_{\mathrm{S}}$ ——标准工作液中相应的青霉素类药物浓度, $\mu \mathrm{g} / \mathrm{L}$ ;

$V$ ——衍生化后所得溶液总体积, $\mathrm{mL}$ ;

$m$ ——供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9检验方法灵敏度、准确度、精密度

# 9.1 灵敏度

本方法青霉素G的检测限为 $3\mu \mathrm{g} / \mathrm{kg}$ , 定量限为 $10\mu \mathrm{g} / \mathrm{kg}$ , 苯唑西林、双氯青霉素、乙氧萘青霉素的检测限为 $10\mu \mathrm{g} / \mathrm{kg}$ , 定量限为 $50\mu \mathrm{g} / \mathrm{kg}$ 。

# 9.2 准确度

本方法在 $10\sim 600~\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$

# 9.3 精密度

本方法批内相对标准偏差 $\leqslant 20\%$ ，批间相对标准偏差 $\leqslant 20\%$

# 动物性食品中赛拉嗪及代谢物2,6-二甲基苯胺残留量的测定液相色谱-串联质谱法

1 范围

本文件规定了动物性食品中赛拉嗪及代谢物2，6-二甲基苯胺残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、牛和羊肌肉、脂肪、肝脏和肾脏组织中赛拉嗪及2,6-二甲基苯胺残留量的检测。

2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

3 术语和定义

本文件没有需要界定的术语和定义。

4 原理

试料中残留的赛拉嗪和2,6二甲基苯胺，用乙腈或者氨水乙腈提取，乙腈饱和的正己烷萃取，净化，液相色谱-串联质谱正离子模式测定，外标法定量。

5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.3 正己烷 $(\mathrm{C_6H_{14}})$ 。

5.1.4 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。

5.1.5 甲酸（HCOOH）。

5.1.6 无水硫酸钠 $\mathrm{(Na_2SO_4)}$

5.2 溶液配制

5.2.1 $5\%$ 氨水乙腈：取氨水 $50~\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$ 。

5.2.2 $20\%$ 乙腈水溶液：取乙腈 $20~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$

5.2.3 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$

5.2.4 乙腈饱和正己烷：取正己烷 $30~\mathrm{mL}$ ，加入乙腈 $5\mathrm{mL}$ ，涡旋混匀，振荡 $10\mathrm{min}$ ，用分液漏斗分离，取上层液体。

5.3 标准品

5.3.1 盐酸赛拉嗪(Xylazine Hydrochloride, $\mathrm{C_{12}H_{16}N_2S}$ ·HCl，CAS号：7361-61-7)，含量 $\geq 98\%$ 见附录A。

5.3.2 2,6-二甲基苯胺[2,6-dimethylaniline， $(\mathrm{CH}_3)_2\mathrm{C}_6\mathrm{H}_3\mathrm{NH}_2$ ，CAS号：87-62-7]，含量 $\geqslant 99.0\%$ 见附录A。

5.4 标准溶液制备

5.4.1 标准储备液：取盐酸赛拉嗪、2,6-二甲基苯胺标准品各 $10\mathrm{mg}$ ，精密称定，分别加乙腈适量使溶解并稀释定容至 $10\mathrm{mL}$ 容量瓶，摇匀，配制成浓度均为 $1\mathrm{mg/mL}$ 的赛拉嗪、2,6-二甲基苯胺标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期3个月。

5.4.2 标准工作液：精密量取赛拉嗪、2,6-二甲基苯胺标准储备液各适量，分别于 $10~\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，摇匀，分别配制成赛拉嗪浓度为 $0.4\mu \mathrm{g / mL},2,6 - 2$ 甲基苯胺浓度为 $1\mu \mathrm{g / mL}$ 的标准工作液。 $2^{\circ}C\sim 8^{\circ}C$ 保存，现配现用。

5.5 材料

滤膜： $0.22\mu \mathrm{m}$ ，水相。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配有电喷雾离子源（ESI）。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 均质机。  
6.4 涡旋混合器。  
6.5 振荡器。  
6.6 冷冻离心机。  
6.7 氮吹仪。  
6.8 超声仪。

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试组织，并使均质。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-20^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取试料 $5\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，加无水硫酸钠 $2\mathrm{g}$ 、乙腈 $10\mathrm{mL}$ ，涡旋混匀，振荡 $10\mathrm{min}$ ， $4^{\circ}\mathrm{C}$ 产 $10000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ （脂肪试料加 $5\%$ 氨水乙腈 $10\mathrm{mL}$ ，于 $4^{\circ}\mathrm{C},8000\mathrm{r / min}$ 离心 $10\mathrm{min})$ ，取上清液，重复提取1次，合并2次上清液。 $40^{\circ}C$ 氮气吹至近干，加乙腈 $1.0~\mathrm{mL}$ 使溶解，涡旋混匀，备用。

# 8.2 净化

取备用液，加乙腈饱和的正己烷 $0.5\mathrm{mL}$ ，涡旋混匀， $4^{\circ}\mathrm{C},10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取下层溶液 $200~\mu \mathrm{L}$ ，于 $5\mathrm{mL}$ 离心管中，加水 $800~\mu \mathrm{L}$ ，涡旋混匀，于 $4^{\circ}\mathrm{C},10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取溶液，过滤，供液相色谱-串联质谱测定。

# 8.3 标准曲线的制备

精密量取赛拉嗪、2,6-二甲基苯胺的标准工作液适量，用 $20\%$ 乙腈水溶液稀释，配制成赛拉嗪浓度为 $0\mathrm{ng / mL},0.2\mathrm{ng / mL},1\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},50\mathrm{ng / mL},100\mathrm{ng / mL},2,6-$ 二甲基苯胺浓度为

$0\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL},50\mathrm{ng / mL},100\mathrm{ng / mL}$ 的系列混合标准溶液，供液相色谱-串联质谱测定，以特征离子质量色谱峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.5 测定法

# 8.5.1 定性测定

通过试样色谱图的保留时间与相应标准品的保留时间、各色谱峰的特征离子与相应浓度标准溶液各色谱峰的特征离子相对照定性，试样与标准品保留时间的相对偏差不大于 $\pm 2.5\%$ ；试样特征离子的相对丰度与浓度相当混合标准溶液的相对丰度一致，相对离子丰度偏差不超过表3的规定，则可判断试样中存在相应的被测物。

# 8.5.2 定量测定

取试料溶液和标准溶液，按外标法以色谱峰面积定量，标准溶液及试料溶液中的赛拉嗪和2,6-二甲基苯胺响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，赛拉嗪和2.6-二甲基苯胺标准溶液特征离子质量色谱图见附录B。

# 8.6 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试料中待测物的残留量按标准曲线或公式(1)计算

$$
X = \frac {A \times C _ {\mathrm {S}} \times V}{A _ {\mathrm {S}} \times m} \times f \tag {1}
$$

式中：

$X$ 供试试料中相应的赛拉嗪残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$C_{\mathrm{S}}$ ——对照溶液中相应的赛拉嗪或2，6二甲基苯胺浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

A 试料溶液中相应的赛拉嗪或2，6二甲基苯胺的峰面积；

$A_{\mathrm{S}}$ 对照溶液中相应的赛拉嗪或2，6二甲基苯胺的峰面积；

$V$ ——用于配制上机检测试液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ 供试试料质量的数值，单位为克 $(\mathrm{g})$

$f$ ——稀释因子，公式中为5。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法赛拉嗪检测限为 $0.06\mu \mathrm{g / kg}$ ，定量限为 $0.2\mu \mathrm{g / kg};2,6-$ 二甲基苯胺检测限为 $1.5\mu \mathrm{g / kg}$ ，定量限为 $5\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法赛拉嗪在 $0.2\mu \mathrm{g} / \mathrm{kg}\sim 100\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$ ；2，6-二甲基苯胺在 $5\mu \mathrm{g} / \mathrm{kg}\sim 100\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$ 。

# 10.3 精密度

本方法批内相对标准偏差 $\leq 15\%$ ，批间相对标准偏差 $\leq 20\%$ 。

# 前言

本标准代替GB/T23502—2009《食品中赭曲霉毒素A的测定免疫亲和层析净化高效液相色谱法》、GB/T25220—2010《粮油检验粮食中赭曲霉毒素A的测定高效液相色谱法和荧光光度法》、GB/T5009.96—2003《谷物和大豆中赭曲霉毒素A的测定》、SN/T1746—2006《进出口大豆、油菜籽和食用植物油中赭曲霉毒素A的检验方法》、SN/T1940—2007《进出口食品中赭曲霉毒素A的测定方法》和SN 0211—1993《出口粮谷中棕曲霉毒素A的检验方法》。

本标准与GB/T23502—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中赭曲霉毒素A的测定”；  
——增加了第三法免疫亲和层析净化液相色谱-串联质谱法和第四法酶联免疫吸附测定法；  
——增加了适用范围并优化了提取方法；  
——删除了免疫亲和柱层析净化荧光光度法。

# 食品安全国家标准

# 食品中赭曲霉毒素A的测定

# 1 范围

本标准规定了食品中赭曲霉毒素A的测定方法。

本标准第一法适用于谷物、油料及其制品、酒类、酱油、醋、酱及酱制品、葡萄干、胡椒粒/粉中赭曲霉毒素A的测定，第二法适用于玉米、稻谷(糙米)、小麦、小麦粉、大豆、咖啡、葡萄酒中赭曲霉毒素A的测定，第三法适用于玉米、小麦等粮食产品、辣椒及其制品等、啤酒等酒类、酱油等产品、生咖啡、熟咖啡中赭曲霉毒素A的测定，第四法适用于玉米、小麦、大麦、大米、大豆及其制品中赭曲霉毒素A的测定，第五法适用于小麦、玉米、大豆中赭曲霉毒素A的测定。

# 第一法 免疫亲和层析净化液相色谱法

# 2 原理

用提取液提取试样中的赭曲霉毒素A，经免疫亲和柱净化后，采用高效液相色谱结合荧光检测器测定赭曲霉毒素A的含量，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.3 冰乙酸 $\mathrm{(C_2H_4O_2)}$ ：色谱纯。  
3.1.4 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.5 聚乙二醇 $\left[\mathrm{HOCH}_{2}\left(\mathrm{CH}_{2} \mathrm{O} \cdot \mathrm{CH}_{2}\right)_{n} \mathrm{CH}_{2} \mathrm{OH}\right]$ 。  
3.1.6 吐温 $20(\mathrm{C}_{58}\mathrm{H}_{114}\mathrm{O}_{26})$ 。  
3.1.7 碳酸氢钠 $\mathrm{(NaHCO_3)}$ 。  
3.1.8 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
3.1.9 浓盐酸（HCl）。  
3.1.10 氮气 $(\mathrm{N}_2)$ ：纯度 $\geq 99.9\%$

# 3.2 试剂配制

3.2.1 提取液I：甲醇-水 $(80 + 20)$

3.2.2 提取液Ⅱ：称取 $150.0\mathrm{g}$ 氯化钠、 $20.0\mathrm{g}$ 碳酸氢钠溶于约 $950~\mathrm{mL}$ 水中，加水定容至 $1\mathrm{L}$ 。  
3.2.3 提取液Ⅲ：乙腈-水 $(60 + 40)$ 。  
3.2.4 冲洗液：称取 $25.0\mathrm{g}$ 氯化钠、 $5.0\mathrm{g}$ 碳酸氢钠溶于约 $950~\mathrm{mL}$ 水中，加水定容至 $1\mathrm{L}$ 。  
3.2.5 真菌毒素清洗缓冲液：称取 $25.0\mathrm{g}$ 氯化钠、 $5.0\mathrm{g}$ 碳酸氢钠溶于水中，加入 $0.1\mathrm{mL}$ 吐温20，用水稀释至 $1\mathrm{L}$ 。  
3.2.6 磷酸盐缓冲液：称取 $8.0\mathrm{g}$ 氯化钠、 $1.2\mathrm{g}$ 磷酸氢钠、 $0.2\mathrm{g}$ 磷酸二氢钾、 $0.2\mathrm{g}$ 氯化钾溶解于约 $990~\mathrm{mL}$ 水中，用浓盐酸调节 $\mathrm{pH}$ 至7.0，用水稀释至 $1\mathrm{L}$ 。  
3.2.7 碳酸氢钠溶液 $(10\mathrm{g / L})$ ：称取 $1.0\mathrm{g}$ 碳酸氢钠，用水溶解并稀释到 $100~\mathrm{mL}$ 。  
3.2.8 淋洗缓冲液：在 $1000\mathrm{mL}$ 磷酸盐缓冲液中加入 $1.0\mathrm{mL}$ 吐温20。

# 3.3 标准品

赭曲霉毒素 $\mathrm{A}(\mathrm{C}_{20}\mathrm{H}_{18}\mathrm{ClNO}_6$ ，CAS号：303-47-9)，纯度 $\geq 99\%$ 。或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 赪曲霉毒素A标准储备液：准确称取一定量的赭曲霉毒素A标准品，用甲醇-乙腈 $(50 + 50)$ 溶解，配成 $0.1\mathrm{mg / mL}$ 的标准储备液，在 $-20^{\circ}C$ 保存，可使用3个月。  
3.4.2 耪曲霉毒素A标准工作液：根据使用需要，准确移取一定量的赭曲霉毒素A标准储备液(3.4.1)，用流动相稀释，分别配成相当于 $1\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL},50\mathrm{ng / mL}$ 的标准工作液， $4^{\circ}C$ 保存，可使用7d。

# 3.5 材料

3.5.1 赫曲霉毒素A免疫亲和柱：柱规格 $1\mathrm{mL}$ 或 $3\mathrm{mL}$ ，柱容量 $\geqslant 100\mathrm{ng}$ ，或等效柱。  
3.5.2 定量滤纸。  
3.5.3 玻璃纤维滤纸：直径 $11\mathrm{cm}$ ，孔径 $1.5\mu \mathrm{m}$ ，无荧光特性。

# 4 仪器和设备

4.1 分析天平：感量 $0.001\mathrm{g}$   
4.2 高效液相色谱仪，配荧光检测器。  
4.3 高速均质器： $\geqslant 12000\mathrm{r / min}$   
4.4 玻璃注射器： $10~\mathrm{mL}$   
4.5 试验筛：孔径 $1\mathrm{mm}$   
4.6 空气压力泵。  
4.7 超声波发生器：功率 $>180\mathrm{W}$ 。  
4.8 氮吹仪。  
4.9 离心机： $\geqslant 10000\mathrm{r / min}$   
4.10 涡旋混合器。  
4.11 往复式摇床： $\geqslant 250\mathrm{r / min}$   
4.12 pH计：精度为0.01。

# 5 分析步骤

# 5.1 试样制备与提取

# 5.1.1 谷物、油料及其制品

# 5.1.1.1 粮食和粮食制品

颗粒状样品需全部粉碎通过试验筛(孔径 $1\mathrm{mm}$ )，混匀后备用。

提取方法1：称取试样 $25.0\mathrm{g}$ （精确到 $0.1\mathrm{g}$ ），加入 $100~\mathrm{mL}$ 提取液Ⅲ，高速均质 $3\mathrm{min}$ 或振荡 $30\mathrm{min}$ ，定量滤纸过滤，移取 $4\mathrm{mL}$ 滤液加入 $26~\mathrm{mL}$ 磷酸盐缓冲液混合均匀，混匀后于 $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清作为滤液A备用。

提取方法2：称取试样 $25.0\mathrm{g}$ （精确到 $0.1\mathrm{g}$ )，加入 $100~\mathrm{mL}$ 提取液I，高速均质 $3\mathrm{min}$ 或振荡 $30\mathrm{min}$ ，定量滤纸过滤，移取 $10~\mathrm{mL}$ 滤液加入 $40~\mathrm{mL}$ 磷酸盐缓冲液稀释至 $50~\mathrm{mL}$ ，混合均匀，经玻璃纤维滤纸过滤，滤液B收集于干净容器中，备用。

# 5.1.1.2 食用植物油

准确称取试样 $5.0\mathrm{g}$ （精确到 $0.1\mathrm{g}$ ），加入 $1\mathrm{g}$ 氯化钠及 $25\mathrm{mL}$ 提取液I，振荡 $30\mathrm{min}$ ，于 $6000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，移取 $15\mathrm{mL}$ 上层提取液，加入 $30\mathrm{mL}$ 磷酸盐缓冲液混合均匀，经玻璃纤维滤纸过滤，滤液C收集于干净容器中，备用。

# 5.1.1.3 大豆、油菜籽

准确称取试样 $50.0\mathrm{g}$ （精确到 $0.1\mathrm{g}$ ）（大豆需要磨细且粒度 $\leqslant 2\mathrm{mm}$ ）于均质器配置的搅拌杯中，加入 $5\mathrm{g}$ 氯化钠及 $100~\mathrm{mL}$ 甲醇(适用于油菜籽)或 $100~\mathrm{mL}$ 提取液I，以均质器高速均质提取 $1\mathrm{min}$ 。定量滤纸过滤，移取 $10~\mathrm{mL}$ 滤液并加入 $40~\mathrm{mL}$ 水稀释，经玻璃纤维滤纸过滤至滤液澄清，滤液D收集于干净容器中，备用。

# 5.1.2 酒类

取脱气酒类试样（含二氧化碳的酒类样品使用前先置于 $4^{\circ}\mathrm{C}$ 冰箱冷藏 $30\mathrm{min}$ ，过滤或超声脱气）或其他不含二氧化碳的酒类试样 $20.0\mathrm{g}$ （精确到 $0.1\mathrm{g}$ ），置于 $25\mathrm{mL}$ 容量瓶中，加提取液Ⅱ定容至刻度，混匀，经玻璃纤维滤纸过滤至滤液澄清，滤液E收集于干净容器中，备用。

# 5.1.3 酱油、醋、酱及酱制品

称取 $25.0\mathrm{g}$ （精确到 $0.1\mathrm{g}$ )混匀的试样，加提取液I定容至 $50~\mathrm{mL}$ ，超声提取 $5\mathrm{min}$ 。定量滤纸过滤，移取 $10~\mathrm{mL}$ 滤液于 $50~\mathrm{mL}$ 容量瓶中，加水定容至刻度，混匀，经玻璃纤维滤纸过滤至滤液澄清，滤液F收集于干净容器中，备用。

# 5.1.4 葡萄干

称取粉碎试样 $50.0\mathrm{g}$ （精确到 $0.1\mathrm{g}$ ）于均质器配置的搅拌杯中，加入 $100~\mathrm{mL}$ 碳酸氢钠溶液，将搅拌杯置于均质器上，以 $22000\mathrm{r / min}$ 高速均质提取 $1\mathrm{min}$ 。定量滤纸过滤，准确移取 $10~\mathrm{mL}$ 滤液并加入 $40~\mathrm{mL}$ 淋洗缓冲液稀释，经玻璃纤维滤纸过滤至滤液澄清，滤液G收集于干净容器中，备用。

# 5.1.5 胡椒粒/粉

# 5.4 试样测定

# 5.4.1 高效液相色谱参考条件

高效液相色谱参考条件列出如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $150~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或等效柱；  
b）流动相：乙腈-水-冰乙酸 $(96 + 102 + 2)$   
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d) 柱温: $35^{\circ} \mathrm{C}$ ;   
e) 进样量: $50 \mu \mathrm{L}$ ;   
f）检测波长：激发波长 $333\mathrm{nm}$ ，发射波长 $460\mathrm{nm}$ 。

# 5.4.2 色谱测定

在5.4.1色谱条件下，将赭曲霉毒素A标准工作溶液按浓度从低到高依次注入高效液相色谱仪，待仪器条件稳定后，以目标物质的浓度为横坐标 $(x$ 轴)，目标物质的峰面积积为纵坐标 $(y$ 轴)，对各个数据点进行最小二乘线性拟合，标准工作曲线按式(1)计算：

$$
y = a x + b \quad \dots \tag {1}
$$

式中：

$y$ ——目标物质的峰面积比；

$a$ ——回归曲线的斜率；

$x$ ——目标物质的浓度；

$b$ ——回归曲线的截距。

标准工作溶液和样液中待测物的响应值均应在仪器线性响应范围内，如果样品含量超过标准曲线范围，需稀释后再测定。

# 5.4.3 空白试验

不称取试样按5.1、5.2和5.3的步骤做空白试验。应确认不含有干扰待测组分的物质。

# 6 分析结果的表述

试样中赭曲霉毒素A的含量按式(2)计算：

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0} \times f \quad \dots \tag {2}
$$

式中：

$X$ ——试样中赭曲霉毒素A的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——试样测定液中赭曲霉毒素A的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

V ——试样测定液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——单位换算常数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$f$ ——稀释倍数。

计算结果时需扣除空白值，检测结果以两次测定值的算数平均值表示，计算结果保留两位有效数字。

# 7 精密度

样品中赭曲霉毒素A含量在重复性条件下获得的两次独立测试结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 33 分析步骤

# 33.1 试样制备

称取 $250.0\mathrm{g}$ 试样经粉碎并通过20目筛后，混匀后备用。

# 33.2 试样提取

# 33.2.1 甲法

称取试样 $20.0\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 于 $200\mathrm{mL}$ 具塞锥形瓶中, 加入 $100\mathrm{mL}$ 三氯甲烷和 $10\mathrm{mL}0.1\mathrm{mol/L}$ 磷酸, 在振荡器上振荡提取 $30\mathrm{min}$ , 将提取液通过快速定性滤纸过滤; 取 $20\mathrm{mL}$ 滤液置于 $250\mathrm{mL}$ 分液漏斗中, 加 $50\mathrm{mL}0.1\mathrm{mol/L}$ 碳酸氢钠溶液振摇 $2\mathrm{min}$ , 静置分层后, 将三氯甲烷层放入另一个 $100\mathrm{mL}$ 分液漏斗中, 如有少量乳化层, 或即使三氯甲烷层全部乳化均可放入分液漏斗中, 加入 $50\mathrm{mL}0.1\mathrm{mol/L}$ 碳酸氢钠溶液重复提取三氯甲烷层, 静置分层后弃去三氯甲烷层, 如三氯甲烷层仍乳

化, 弃去, 不影响结果。碳酸氢钠水层并入第一个分液漏斗中, 加约 $5.5 \mathrm{~mL} 2 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液调节 $\mathrm{pH} 2 \sim 3$ , 加入 $25 \mathrm{~mL}$ 三氯甲烷振摇 $2 \mathrm{~min}$ , 静置分层后, 放三氯甲烷层于另一个盛有 $100 \mathrm{~mL}$ 水的 $250 \mathrm{~mL}$ 分液漏斗中, 酸水层再用 $10 \mathrm{~mL}$ 三氯甲烷振摇、提取、静置, 将三氯甲烷层并入同一分液漏斗中。振摇、静置分层, 用脱脂棉擦干分液漏斗下端, 放三氯甲烷层于一个 $75 \mathrm{~mL}$ 蒸发皿中, 将蒸发皿置蒸汽浴上通风挥干。用约 $8 \mathrm{~mL}$ 三氯甲烷分次将蒸发皿中的残渣溶解, 转入具尾管的 $10 \mathrm{~mL}$ 浓缩瓶中, 置 $80^{\circ} \mathrm{C}$ 水浴锅上用蒸汽加热吹氮气 $(\mathrm{N}_{2})$ 浓缩至干, 加入 $0.2 \mathrm{~mL}$ 苯-乙腈 $(98 + 2)$ 溶解残渣, 摇匀, 供薄层色谱点样用。

# 33.2.2 乙法

称取试样 $20.0\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）于 $200~\mathrm{mL}$ 具塞锥形瓶中，加入 $30~\mathrm{mL}$ 石油醚和 $100~\mathrm{mL}$ 甲醇-水 $(55 + 45)$ ，在瓶塞上抹上一层水盖严防漏。在振荡器上振荡提取 $30\mathrm{min}$ 后，通过快速定性滤纸滤入分液漏斗中，待下层甲醇水层分清后，取出 $20~\mathrm{mL}$ 滤液置于 $100~\mathrm{mL}$ 分液漏斗中，调节 $\mathrm{pH}5\sim 6$ 。加入 $25~\mathrm{mL}$ 三氯甲烷振摇 $2\mathrm{min}$ ，静置分层后放出三氯甲烷层于另一分液漏斗中，再用 $10~\mathrm{mL}$ 三氯甲烷重复振摇提取甲醇水层，如发生乳化现象，可滴加甲醇促使其分层，将三氯甲烷层合并于同一分液漏斗中，加入 $50~\mathrm{mL}\sim 100~\mathrm{mL}$ 氯化钠溶液(加入量视品种不同而异，大豆加 $100~\mathrm{mL}$ ，小麦、玉米则加 $50~\mathrm{mL}$ 左右)，振摇放置(如为大豆试样提取液还须轻轻反复倒转分液漏斗，使乳化层逐渐上升。如乳化严重可加入少许甲醇)，待三氯甲烷层澄清后，用脱脂棉擦干分液漏斗下端，放三氯甲烷层于 $75~\mathrm{mL}$ 蒸发皿中（如为大豆试样须再加入 $10~\mathrm{mL}$ 三氯甲烷振摇，三氯甲烷层合并于同一蒸发皿中)，将蒸发皿置蒸汽浴上通风挥干。以下操作自“用约 $8~\mathrm{mL}$ 三氯甲烷分次将蒸发皿中的残渣溶解”起，按甲法操作。

# 33.3 试样测定

# 33.3.1 薄层板的制备

称取 $4\mathrm{g}$ 硅胶G，加约 $10~\mathrm{mL}$ 水于乳钵中研磨至糊状。立即倒入涂布器内制成 $5\mathrm{cm}\times 20\mathrm{cm}$ 、厚度 $0.3\mathrm{mm}$ 的薄层板三块，在空气中干燥后，在 $105^{\circ}C\sim 110^{\circ}C$ 活化 $1\textrm{h}$ ，取出放干燥器中保存。

# 33.3.2 点样

取两块薄层板，在距薄层板下端 $2.5\mathrm{cm}$ 的基线上用微量注射器滴加两个点：在距板左边缘 $1.7\mathrm{cm}$ 处滴加赭曲霉毒素A标准工作液 $8\mu \mathrm{L}$ ，在距板左边缘 $2.5\mathrm{cm}$ 处滴加样液 $25\mu \mathrm{L}$ ，然后在第二块板的样液点上滴加赭曲霉毒素A标准工作液 $8\mu \mathrm{L}$ ，点样时，需边滴加边用电吹风吹干，交替使用冷热风。

# 33.3.3 展开

# 33.3.3.1 展开剂

横展剂：乙醚或乙醚-甲醇-水 $(94 + 5 + 1)$ 。

纵展剂：

a）甲苯-乙酸乙酯-甲酸-水 $(6 + 3 + 1.2 + 0.06)$ 或甲苯-乙酸乙酯-甲酸 $(6 + 3 + 1.4)$   
b）苯-冰乙酸 $(9 + 1)$ 。

# 33.3.3.2 展开

横向展开：在展开槽内倒入 $10\mathrm{mL}$ 横展剂，先将薄层板纵展至离原点 $2\mathrm{cm}\sim 3\mathrm{cm}$ ，取出通风挥发溶剂 $1\mathrm{min}\sim 2\mathrm{min}$ 后，再将该薄层板靠标准点的长边置于同一展开槽内的溶剂中横展，如横展剂不够，可添加适量，展至板端过 $1\mathrm{min}$ ，取出通风挥发溶剂 $2\mathrm{min}\sim 3\mathrm{min}$ 。

纵向展开：在另一展开槽内倒入 $10\mathrm{mL}$ 纵展剂，将经横展后的薄层板纵展至前沿距原点 $13~\mathrm{cm}\sim$

$15~\mathrm{cm}$ 。取出通风挥干至板面无酸味 $(5\mathrm{min}\sim 10\mathrm{min})$ 。

# 33.3.3.3 观察与评定

将薄层色谱板置 $365\mathrm{nm}$ 波长紫外光灯下观察：

a）在紫外光灯下将两板相互比较，若第二块板的样液点在赭曲霉毒素A标准点的相应处出现最低检出量，而在第一板相同位置上未出现荧光点，则试样中的赭曲霉毒素A含量在本测定方法的最低检测量为 $10\mu \mathrm{g / kg}$ 以下。  
b）如果第一板样液点在与第二板样液点相同位置上出现荧光点，则看第二板样液的荧光点是否与滴加的标准荧光点重叠，再进行以下的定量与确证试验。

# 33.3.4 稀释定量

比较样液中赭曲霉毒素A与标准赭曲霉毒素A点的荧光强度，估计稀释倍数。

薄层板经双向展开后，当阳性样品中赭曲霉毒素A含量高时，赭曲霉毒素A的荧光点会被横向拉长、使点变扁，或分成两个黄绿色荧光点。这是因为在横展过程中原点上赭曲霉毒素A的量超过了硅胶的吸附能力，原点上的杂质和残留溶剂在横展中将赭曲霉毒素A点横向拉长了，这时可根据赭曲霉毒素A黄绿色荧光的总强度与标准荧光强度比较，估计需减少的滴加微升数或所需稀释倍数。经稀释后测定含量时可在样液点的左边基线上滴加二个标准点，赭曲霉毒素A的量可为 $4\mathrm{ng},8\mathrm{ng}$ 。比较样液与两个标准赭曲霉毒素A荧光点的荧光强度，概略定量。

# 33.3.5 确证试验

用碳酸氢钠-乙醇溶液喷洒色谱板，在室温下干燥，于长波紫外光灯下观察，这时赭曲霉毒素A荧光点应由黄绿色变为蓝色，而且荧光强度有所增加，可使方法检出限达 $5\mu \mathrm{g / kg}$ ，但概略定量仍按喷洒前所显黄绿色荧光计。

# 34 分析结果的表述

试样中赭曲霉毒素A含量按式(8)计算：

$$
X = A \times \frac {V _ {1}}{V _ {2}} \times f \times \frac {1 0 0 0}{m} \tag {8}
$$

式中：

$X$ ——试样中赭曲霉毒素A的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ —薄层板上测得样液点上赭曲霉毒素A赭曲霉毒素A的量，单位为微克 $(\mu \mathrm{g})$

$V_{1}$ ——苯-乙腈混合液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——出现最低荧光点时滴加样液的体积，单位为毫升 $(\mathrm{mL})$

$f$ ——样液的总稀释倍数；

$m$ ——苯-乙腈溶解时相当样品的质量，单位为克 $(\mathrm{g})$ 。

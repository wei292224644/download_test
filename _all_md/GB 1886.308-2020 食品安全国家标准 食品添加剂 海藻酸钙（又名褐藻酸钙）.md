# 食品安全国家标准

# 食品添加剂 海藻酸钙（又名褐藻酸钙）

# 1 范围

本标准适用于从海带(Laminaria)、巨藻(Macrocystis)、泡叶藻(Ascophyllum)等褐藻类植物中经提取加工制成的食品添加剂海藻酸钙(又名褐藻酸钙)。

# 2 分子式、结构式和相对分子质量

# 2.1 分子式

$$
\left[ \mathrm {C} _ {6} \mathrm {H} _ {7} \mathrm {C a} _ {1 / 2} \mathrm {O} _ {6} \right] _ {n}
$$

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度应在分析纯以上，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 乙醇。

A.2.1.2 乙醚。

A.2.1.3 盐酸。

A.2.1.4 异丙醚。

A.2.1.5 氢氧化钠溶液： $10\mathrm{g / L}$

A.2.1.6 氯化钙溶液： $25\mathrm{g / L}$

A.2.1.7 试样溶液 $(5\mathrm{g} / \mathrm{L})$ ：称取 $0.5\mathrm{g}$ 试样，在搅拌状态下慢慢加入 $100~\mathrm{mL}$ 氢氧化钠溶液(A.2.1.5)中，不断搅拌，缓慢溶解至呈均匀的溶液，备用。

A.2.1.8 硫酸铵饱和溶液: 称取 $78 \mathrm{~g}$ 硫酸铵, 加入 $100 \mathrm{~mL}$ 水, 在加热状态下使其溶解, 放置过夜, 上层清液即为硫酸铵饱和溶液。

A.2.1.9 1,3-二羟基萘乙醇溶液 $(10\mathrm{g / L})$ ：称取约 $1\mathrm{g}1,3-$ 二羟基萘溶于 $100~\mathrm{mL}$ 无水乙醇，混匀（现用现配）。

# A.2.2 鉴别

# A.2.2.1 可溶性试验

称取约 $0.5\mathrm{g}$ 试样4份，分别加入 $100~\mathrm{mL}$ 水、 $100~\mathrm{mL}$ 氢氧化钠溶液、 $100~\mathrm{mL}$ 乙醇及 $100~\mathrm{mL}$ 乙醚，试样在氢氧化钠溶液中缓慢溶解，不溶于水、乙醚、乙醇。

# A.2.2.2 氯化钙沉淀试验

取 $10\mathrm{mL}$ 试样溶液(A.2.1.7)，加入 $2\mathrm{mL}$ 的氯化钙溶液，形成凝胶状沉淀。该方法可区分海藻酸钙与阿拉伯树胶、卡拉胶、明胶、刺梧桐树胶、槐豆胶、甲基纤维素、黄蓍胶。

# A.2.2.3 硫酸铵沉淀试验

取 $10\mathrm{mL}$ 试样溶液(A.2.1.7)，加入 $5\mathrm{mL}$ 的硫酸铵饱和溶液，没有形成沉淀。该方法可区分海藻酸钙与琼脂、卡拉胶、果胶、明胶、槐豆胶、甲基纤维素、淀粉。

# A.2.2.4 海藻酸盐鉴定

取 $5\mathrm{mg}$ 试样放入试管中，加入 $5\mathrm{mL}$ 水，加入 $1\mathrm{mL}$ 新制的1,3-二羟基萘乙醇溶液和 $5\mathrm{mL}$ 盐酸摇匀。把上述混合物加热至沸，煮沸 $3\mathrm{min}$ ，冷却到 $15^{\circ}\mathrm{C}$ 左右，转移至 $30~\mathrm{mL}$ 的分液漏斗中，容器用 $5\mathrm{mL}$ 水洗涤，洗液并入分液漏斗中。加入 $15\mathrm{mL}$ 异丙醚，振摇提取，分取醚层，同时做空白对照，样品管的异丙醚层与对照管比较，应显深紫色。

# A.3 海藻酸钙含量（以氧化钙计，以干基计）的测定

# A.3.1 方法提要

将海藻酸钙试样灰化后，与酸反应形成可溶性的钙盐，利用乙二胺四乙酸二钠标准滴定溶液滴定反应产生的钙离子，折算成海藻酸钙(以氧化钙计，以干基计)的含量。

# A.3.2 试剂和材料

A.3.2.1 氢氧化钾溶液 $(2\mathrm{mol} / \mathrm{L})$ ：准确称取氢氧化钾 $112\mathrm{g}$ ，加水溶解稀释至 $1000~\mathrm{mL}$ ，混合均匀，备用。  
A.3.2.2 混合酸消化液：硝酸、高氯酸以 $4:1$ 的体积比混合均匀，备用。  
A.3.2.3 三乙醇胺溶液 $(10\%)$ ：量取 $10~\mathrm{mL}$ 三乙醇胺置于 $90~\mathrm{mL}$ 水中，混合均匀。  
A.3.2.4钙红指示剂 $(1\%)$ ：称取 $1\mathrm{g}$ 钙红指示剂 $\mathrm{(C_{21}O_7N_2SH_{14})}$ ，加 $99\mathrm{g}$ 固体氯化钠，于研钵中混匀研细，盛于棕色广口瓶中，备用。  
A.3.2.5 乙二胺四乙酸二钠标准滴定溶液 $(0.01\mathrm{mol} / \mathrm{L})$ 。

# A.3.3 仪器和设备

A.3.3.1 碱式滴定管： $50~\mathrm{mL}$   
A.3.3.2 万用电阻炉。  
A.3.3.3 容量瓶： $250~\mathrm{mL}$   
A.3.3.4 三角瓶： $250~\mathrm{mL}$

# A.3.4 分析步骤

# A.3.4.1 试样消化

取测定灰分的坩埚连同遗留残渣，小心加入混合酸消化液 $25\mathrm{mL}\pm 5\mathrm{mL}$ ，置于通风橱内万用电阻炉上小火加热，酸液过少时可补加少量混合酸消化液，继续加热消化至消化液无色透明为止。此时消化液中可能会残留剩余酸消化液，为将其蒸出，应将消化液继续加热；若消化液较少，可多次补充 $10\mathrm{mL}\pm 5\mathrm{mL}$ 水缓慢加热，蒸约 $1\mathrm{h}\sim 2\mathrm{h}$ ，即可取下；待冷却后，将坩埚内消化液小心转移至容量瓶中，并用少量水反复冲洗坩埚，并不断用 $\mathrm{pH}$ 试纸检测，直至洗涤液不再呈明显酸性为止，洗液并入容量瓶并定容。

取与消化试样相同量的混合酸消化液，按上述操作做试剂空白试验。

# A.3.4.2 试样及空白滴定

分别移取 $5\mathrm{mL}$ 试样消化液及空白于三角瓶中，加 $50~\mathrm{mL}$ 蒸馏水混合均匀，加入 $5\mathrm{mL}$ 氢氧化钾溶液，再加入 $1\mathrm{mL}$ 三乙醇胺，加 $0.1\mathrm{g}$ 钙红指示剂，充分振荡摇晃使溶液混合均匀。在不断振摇下，用乙二胺四乙酸二钠标准滴定溶液滴定，溶液由酒红色变为纯蓝色，即为终点。

# A.3.4.3 结果计算

海藻酸钙含量(以氧化钙计，以干基计)的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {c \times \left(V - V _ {0}\right) \times 56.08 \times 250}{m \times 5 \times 1000 \times \left(1 - w _ {2}\right)} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$c$ ——乙二胺四乙酸二钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

V ——滴定试样所消耗的乙二胺四乙酸二钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白所消耗的乙二胺四乙酸二钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

56.08——氧化钙的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ；

250 ——容量瓶的容积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克(g)；

5 ——移取试样消化液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算因子；

$w_{2}$ ——试样的干燥减量， $\%$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $3.0\%$ 。

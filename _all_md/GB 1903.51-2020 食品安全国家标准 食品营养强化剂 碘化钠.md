# 检验方法

# A.1 一般规定

本标准所用试剂和水在未注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品在未注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 钠离子试验

# A.2.1.1 试剂和材料

A.2.1.1.1 碳酸钾溶液： $150~\mathrm{g / L}$

A.2.1.1.2 氢氧化钾溶液： $150~\mathrm{g / L}$

A.2.1.1.3 焦锑酸钾溶液：准确称取焦锑酸钾 $2\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），在 $85~\mathrm{mL}$ 热水中溶解，迅速冷却，加入氢氧化钾溶液 $10~\mathrm{mL}$ ，放置 $24\mathrm{h}$ ，滤过，加水定容至 $100~\mathrm{mL}$ 。

# A.2.1.2 鉴定步骤

称取试样 $0.1\mathrm{g}$ ，置 $10\mathrm{mL}$ 试管中，加入 $2\mathrm{mL}$ 水溶解，加碳酸钾溶液 $2\mathrm{mL}$ ，加热至沸，应不得有沉淀生成；加焦锑酸钾溶液 $4\mathrm{mL}$ ，加热至沸；置冰水中冷却，必要时，用玻棒摩擦试管内壁，应有致密的沉淀生成。

# A.2.2 碘离子试验

# A.2.2.1 试剂和材料

A.2.2.1.1 盐酸。

A.2.2.1.2 三氯甲烷。

A.2.2.1.3 淀粉指示液： $10\mathrm{g / L}$

# A.2.2.2 鉴定步骤

称取试样 $0.1\mathrm{g}$ ，加水 $20~\mathrm{mL}$ ，加 $1\mathrm{mL}$ 盐酸；如加三氯甲烷振摇，三氯甲烷层显紫色；如加淀粉指示液，溶液显蓝色。

# A.3 碘化钠含量（以干基计）的测定

# A.3.1 方法提要

在盐酸介质溶液中，用碘酸钾标准滴定溶液进行滴定，通过三氯甲烷层颜色的消失判断终点。

# A.3.2 试剂和材料

A.3.2.1 盐酸。

A.3.2.2 三氯甲烷。  
A.3.2.3 碘酸钾标准滴定溶液： $c(1 / 6\mathrm{KIO}_3) = 0.3\mathrm{mol} / \mathrm{L}$

# A.3.3 分析步骤

准确称取预先干燥(干燥方法见A.4)的试样 $0.5\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 锥形瓶中，加入 $10~\mathrm{mL}$ 水溶解，加入 $35~\mathrm{mL}$ 盐酸，摇匀。加 $5\mathrm{mL}$ 三氯甲烷，用碘酸钾标准滴定溶液进行滴定（近终点时，剧烈振摇)，直至三氯甲烷层无色后静置 $5\mathrm{min}$ 不再呈紫色为止，即为终点。

# A.3.4 结果计算

碘化钠含量（以干基计）的质量分数 $\omega_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {V \times c \times M}{m \times 1000} \times 100 \% \tag{A.1}
$$

式中：

$V$ ——滴定试验溶液所消耗的碘酸钾标准滴定溶液的体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——碘酸钾标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——碘化钠 $(1 / 3\mathrm{NaI})$ 的摩尔质量的数值，单位为克每摩尔 $(\mathrm{g / mol})(M = 49.96)$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

1000——换算因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $5\%$ 。

# A.4 干燥减量的测定

# A.4.1 仪器和设备

A.4.1.1 万分之一电子天平。

A.4.1.2 电热恒温干燥箱：温度 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。

# A.4.2 分析步骤

用预先于 $105\mathrm{^\circ C}\pm 2\mathrm{^\circ C}$ 干燥至恒重的称量瓶准确称取 $2\mathrm{g}$ 试样（精确至 $0.0001\mathrm{g})$ ，置于电热恒温干燥箱内，在 $105\mathrm{^\circ C}\pm 2\mathrm{^\circ C}$ 下干燥 $4\mathrm{h}$ ，称量。

# A.4.3 结果计算

干燥减量的质量分数 $\varpi_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m _ {1} - m _ {0}}{m} \times 100 \% \tag{A.2}
$$

式中：

$m_{1}$ ——干燥前试样和称量瓶质量的数值，单位为克 $(\mathrm{g})$ ；

$m_0$ ——干燥后试样和称量瓶质量的数值，单位为克 $(\mathrm{g})$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $5\%$ 。

# A.5 碘酸盐（以 $\mathbf{IO}_3$ 计）的测定

# A.5.1 试剂和材料

A.5.1.1 硫酸溶液： $c(\mathrm{H}_2\mathrm{SO}_4) = 0.5\mathrm{mol / L}$   
A.5.1.2 碘酸盐标准溶液：准确称取碘酸钾 $0.4\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，置于 $1000~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。取该溶液 $1\mathrm{mL}$ ，置于 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。贮存于棕色瓶中。  
A.5.1.3 淀粉指示液： $10\mathrm{g / L}$   
A.5.1.4 无氨和无二氧化碳的水：将无氨的水倒入烧瓶中，煮沸 $15\mathrm{min}$ ，立即用装有钠石灰管的胶塞塞紧，冷却。

# A.5.2 分析步骤

准确称取试样 $1.1\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），置于 $10~\mathrm{mL}$ 容量瓶中，用无氨和无二氧化碳的水溶解并稀释至刻度，摇匀，转移至比色管；加入 $1\mathrm{mL}$ 淀粉指示液，加 $0.25\mathrm{mL}$ 硫酸溶液，混匀。其溶液的颜色不应深于标准比色液。

标准比色液配制：准确称取试样 $0.1\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，置于 $10~\mathrm{mL}$ 容量瓶中，加入 $1\mathrm{mL}$ 碘酸盐标准溶液，用无氨和无二氧化碳的水稀释至刻度，摇匀，转移至比色管；加入 $1\mathrm{mL}$ 淀粉指示液，加 $0.25\mathrm{mL}$ 硫酸溶液，混匀。

# A.6 钾的测定

# A.6.1 试剂和材料

A.6.1.1 乙酸溶液：取冰乙酸 $6\mathrm{mL}$ ，加水稀释至 $100~\mathrm{mL}$ ，摇匀。  
A.6.1.2 四苯硼钠溶液： $33.33\mathrm{g / L}$   
A.6.1.3 氯化钾溶液： $9.5\mathrm{mg / L}$

# A.6.2 分析步骤

准确称取试样 $1\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，置 $100~\mathrm{mL}$ 容量瓶中，加水溶解并稀释至刻度，摇匀，取 $4\mathrm{mL}$ ，加乙酸溶液 $1\mathrm{mL}$ ，混匀，加四苯硼钠溶液 $5\mathrm{mL}$ ，立即振摇并放置 $10\mathrm{min}$ ；如发生浑浊，与氯化钾溶液 $4\mathrm{mL}$ 同法制成的对照液比较，不得更浓。

# A.6.3 结果判定

供试液浑浊度不浓于标准对照液，即试样中的钾不大于 $0.05\%$ 。

# A.7 硝酸盐、亚硝酸盐和氨的试验

# A.7.1 试剂和材料

A.7.1.1 氢氧化钠溶液： $40\mathrm{g / L}$   
A.7.1.2 铝丝。  
A.7.1.3 石蕊试纸。

# A.7.2 分析步骤

称取试样 $1\mathrm{g}$ ，置于 $40~\mathrm{mL}$ 试管中，加 $5\mathrm{mL}$ 水溶解，然后加 $5\mathrm{mL}$ 氢氧化钠溶液，加 $0.2\mathrm{g}$ 铝丝。用脱脂棉塞住试管口，在试管口上面放置一条湿润的红色石蕊试纸。将试管置于沸水浴上加热 $15\mathrm{min}$ ，石蕊试纸应不变为蓝色。

# A.7.3 结果判定

石蕊试纸不变为蓝色，即为通过试验。

# A.8 硫代硫酸盐和钡的试验

# A.8.1 试剂和材料

A.8.1.1 硫酸溶液： $c(\mathrm{H}_2\mathrm{SO}_4) = 1\mathrm{mol / L}$

A.8.1.2 无氨和无二氧化碳的水：将无氨的水倒入烧瓶中，煮沸 $15\mathrm{min}$ ，立即用装有钠石灰管的胶塞塞紧，冷却。

# A.8.2 分析步骤

称取试样 $0.5\mathrm{g}$ ，用 $10~\mathrm{mL}$ 无氨和无二氧化碳的水溶解，加2滴硫酸溶液， $1\mathrm{min}$ 内溶液不应浑浊。

# A.8.3 结果判定

供试液 $1\mathrm{min}$ 内不浑浊，即为通过试验。

# A.9 碱度的试验

# A.9.1 试剂和材料

A.9.1.1 硫酸溶液： $c(\mathrm{H}_2\mathrm{SO}_4) = 0.05\mathrm{mol / L}$

A.9.1.2 酚酞指示溶液： $10\mathrm{g / L}$

# A.9.2 分析步骤

称取试样 $1\mathrm{g}$ 加水至 $10~\mathrm{mL}$ 溶解，加酚酞指示液1滴与硫酸溶液 $0.1\mathrm{mL}$ ，不得显红色。

# A.9.3 结果判定

供试液溶液不显红色，即为通过试验。

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准滴定溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

A.2.1 试样经加强热，即变成黄色；放冷，黄色即消失。  
A.2.2 在试样溶液中加入亚铁氰化钾溶液，即生成白色沉淀；分离，沉淀物在稀盐酸中不溶解。  
A.2.3 将试样制备成中性或碱性溶液，加入硫化钠溶液，即生成白色沉淀。

# A.3 氧化锌含量的测定

# A.3.1 试剂和材料

A.3.1.1 盐酸溶液： $9.5\% \sim 10.5\%$ 。取盐酸 $8.4~\mathrm{mL}$ ，加水使稀释成 $100~\mathrm{mL}$   
A.3.1.2 氨-氯化铵缓冲液： $\mathrm{pH}10.0$ 。取氯化铵 $5.4 \mathrm{~g}$ ，加水 $20 \mathrm{~mL}$ 溶解后，加浓氨溶液 $35 \mathrm{~mL}$ ，再加水稀释至 $100 \mathrm{~mL}$ ，即得。  
A.3.1.3 乙二胺四乙酸二钠标准滴定液: $c(\mathrm{EDTA}-2\mathrm{Na})=0.05 \mathrm{~mol} / \mathrm{L}$ 。取乙二胺四乙酸二钠 $19 \mathrm{~g}$ , 加适量的水使溶解成 $1000 \mathrm{~mL}$ , 摇匀。  
A.3.1.4 氨试液：取浓氨溶液 $400~\mathrm{mL}$ ，加水使成 $1000~\mathrm{mL}$ ，即得。  
A.3.1.5 铬黑T指示剂：取铬黑 $\mathrm{T}0.1\mathrm{g}$ ，加氯化钠 $10\mathrm{g}$ ，研磨均匀，即得。

# A.3.2 分析步骤

精确称取试样 $0.1\mathrm{g}$ ，加盐酸溶液 $2\mathrm{mL}$ 使溶解，加水 $25\mathrm{mL}$ ，加 $0.025\%$ 甲基红的乙醇溶液1滴，滴加氨试液至溶液显微黄色，加水 $25\mathrm{mL}$ 、氨-氯化铵缓冲液 $(\mathrm{pH}10.0)10\mathrm{mL}$ 与铬黑T指示剂少许，用乙二胺四乙酸二钠标准滴定溶液 $(0.05\mathrm{mol / L})$ 滴定至溶液由紫色转变为纯蓝色。 $1\mathrm{mL}$ 乙二胺四乙酸二钠滴定液 $(0.05\mathrm{mol / L})$ 相当于 $4.069\mathrm{mg}$ 的 $\mathrm{ZnO}$ 。

# A.4 碱度的测定

# A.4.1 试剂和材料

A.4.1.1 盐酸标准滴定溶液： $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$ 。取盐酸 $9.0~\mathrm{mL}$ ，加水适量使成 $1000~\mathrm{mL}$ ，摇匀。  
A.4.1.2 酚酞指示液：取酚酞 $1\mathrm{g}$ ，加乙醇 $100~\mathrm{mL}$ 使溶解，即得。

# A.4.2 分析步骤

称取试样 $1.0\mathrm{g}$ ，精确至 $0.01\mathrm{g}$ ，加新沸的热水 $10~\mathrm{mL}$ ，振摇 $5\mathrm{min}$ ，放冷，过滤，滤液加酚酞指示液

2滴，如显粉红色，加盐酸标准滴定溶液 $(0.1\mathrm{mol} / \mathrm{L})0.10\mathrm{mL}$ ，粉红色应消失。

# A.5 碳酸盐与酸中不溶物的测定

# A.5.1 试剂和材料

硫酸溶液： $9.5\% \sim 10.5\%$

# A.5.2 分析步骤

称取试样 $2.0\mathrm{g}$ ，精确至 $0.01\mathrm{g}$ ，加水 $10\mathrm{mL}$ 混合后，加硫酸溶液 $30\mathrm{mL}$ ，置水浴上加热，不应发生气泡；搅拌后，溶液应澄清。

# A.6 灼烧失重的测定

# A.6.1 仪器和设备

高温炉。

# A.6.2 分析步骤

称取试样 $1.0\mathrm{g}$ ，精确至 $0.01\mathrm{g}$ ，在 $800^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 灼烧至质量恒定，减失重量不得超过 $1.0\%$ 。

# A.7 铁盐的测定

# A.7.1 试剂和材料

A.7.1.1 盐酸溶液： $9.5\% \sim 10.5\%$

A.7.1.2 过硫酸铵。

A.7.1.3 铁标准溶液：每 $1\mathrm{mL}$ 相当于 $10~\mu \mathrm{g}$ 的Fe。

称取硫酸铁铵 $\left[\mathrm{FeNH}_{4}\left(\mathrm{SO}_{4}\right)_{2} \cdot 12 \mathrm{H}_{2} \mathrm{O}\right] 0.863 \mathrm{~g}$ , 置 $1000 \mathrm{~mL}$ 容量瓶中, 加水溶解后, 加硫酸 $2.5 \mathrm{~mL}$ , 用水稀释至刻度, 摇匀, 作为贮备液。

临用前，精确量取贮备液 $10~\mathrm{mL}$ ，置 $100~\mathrm{mL}$ 容量瓶中，加水稀释至刻度，摇匀，即得。

A.7.1.4 硫氰酸铵溶液： $30\%$ 。

# A.7.2 仪器和设备

一般的实验室仪器和纳氏比色管。

# A.7.3 分析步骤

# A.7.3.1 试样溶液的制备

称取试样 $0.40\mathrm{g}$ ，加稀盐酸 $8\mathrm{mL}$ 、水 $15\mathrm{mL}$ 与硝酸2滴，煮沸 $5\mathrm{min}$ 使溶解，放冷，加水至 $50\mathrm{mL}$ ，混匀后，取 $25\mathrm{mL}$ ，加水 $10\mathrm{mL}$ 。

# A.7.3.2 测定

取试样加水溶解至 $25~\mathrm{mL}$ ，移置 $50~\mathrm{mL}$ 纳氏比色管中，加稀盐酸 $4\mathrm{mL}$ 与过硫酸铵 $50~\mathrm{mg}$ ，用水稀释成 $35~\mathrm{mL}$ 后，加 $30\%$ 硫氰酸铵溶液 $3\mathrm{mL}$ ，再加水稀释成 $50~\mathrm{mL}$ ，摇匀；如显色，立即与一定量标准铁

溶液制成的对照溶液（取该品种项下规定量的标准铁溶液，置 $50~\mathrm{mL}$ 纳氏比色管中，加水 $25~\mathrm{mL}$ ，加稀盐酸 $4\mathrm{mL}$ 与过硫酸铵 $50~\mathrm{mg}$ ，用水稀释成 $35~\mathrm{mL}$ ，加 $30\%$ 硫氰酸铵溶液 $3\mathrm{mL}$ ，再加水稀释成 $50~\mathrm{mL}$ ，摇匀）比较，不得更深 $(0.005\%)$ 。

如供试管与对照管色调不一致时，可分别移至分液漏斗中，各加正丁醇 $20~\mathrm{mL}$ 提取，待分层后，将正丁醇层移置 $50~\mathrm{mL}$ 纳氏比色管中，再用正丁醇稀释至 $25~\mathrm{mL}$ ，比较，不得更深 $(0.005\%)$ 。

# A.8 铅盐的测定

# A.8.1 试剂和材料

A.8.1.1 冰乙酸。  
A.8.1.2 铬酸钾指示液。

# A.8.2 分析步骤

称取试样 $2.0\mathrm{g}$ ，精确至 $0.01\mathrm{g}$ ，加水 $20\mathrm{mL}$ 搅匀后，加冰乙酸 $5\mathrm{mL}$ ，置水浴上加热溶解后，放冷，过滤，滤液加铬酸钾指示液5滴，不得发生浑浊。

# A.9 砷盐的测定

# A.9.1 试剂和材料

A.9.1.1 盐酸。

A.9.1.2 锌粒：粒径 $0.5\mathrm{mm}\sim 1\mathrm{mm}$

A.9.1.3 碘化钾溶液。

A.9.1.4 氯化亚锡溶液。

A.9.1.5 砷标准溶液：每 $1\mathrm{mL}$ 相当于 $1\mu \mathrm{g}$ 的As。

称取三氧化二砷 $0.132\mathrm{g}$ ，置 $1000\mathrm{mL}$ 容量瓶中，加 $20\%$ 氢氧化钠溶液 $5\mathrm{mL}$ 溶解后，用适量的硫酸溶液中和，再加硫酸溶液 $10\mathrm{mL}$ ，用水稀释至刻度，摇匀，作为贮备液。

临用前，精密量取贮备液 $10\mathrm{mL}$ ，置 $1000\mathrm{mL}$ 量瓶中，加硫酸溶液 $10\mathrm{mL}$ ，用水稀释至刻度，摇匀，即得。

A.9.1.6 醋酸铅棉花。

A.9.1.7 溴化汞试纸。

# A.9.3 分析步骤

# A.9.3.1 试样

取本品 $1.0\mathrm{g}$ ，加盐酸 $5\mathrm{mL}$ 与水 $23\mathrm{mL}$ 使溶解。

# A.9.3.2 标准砷斑制备

精确量取标准砷溶液 $2\mathrm{mL}$ ，置A瓶中，加盐酸 $5\mathrm{mL}$ 与水 $21\mathrm{mL}$ ，再加碘化钾试液 $5\mathrm{mL}$ 与酸性氯化亚锡试液5滴，在室温放置 $10\mathrm{min}$ 后，加锌粒 $2\mathrm{g}$ ，立即按图A.1将装好的导气管C密塞于A瓶上，并将A瓶置 $25^{\circ}\mathrm{C} \sim 40^{\circ}\mathrm{C}$ 水浴中，反应 $45\mathrm{min}$ ，取出溴化汞试纸，即得。

# A.9.3.3 测定

测试时，于导气管C中装入醋酸铅棉花 $60\mathrm{mg}$ （装管高度为 $60~\mathrm{mm}\sim 80~\mathrm{mm}$ ，再于旋塞D的顶端平面上放一片溴化汞试纸(试纸大小以能覆盖孔径而不露出平面外为宜)，盖上旋塞盖E并旋紧，即得。

取试样 $1.0\mathrm{g}$ ，加盐酸 $5\mathrm{mL}$ 与水 $23\mathrm{mL}$ 使溶解，置A瓶中，按照标准砷斑的制备，自“再加碘化钾试液 $5\mathrm{mL}$ ”起操作。将生成的砷斑与标准砷斑比较，不得更深 $(0.0002\%)$ 。

# 食品安全国家标准

# 食品营养强化剂 酪蛋白磷酸肽

# 1 范围

本标准适用于以牛乳或酪蛋白制品为原料，用酶解法生产制得的食品营养强化剂酪蛋白磷酸肽。

# 2 分子结构和相对分子质量

主要有效成分是含有1个～6个磷酸丝氨酸残基的多肽，其相对分子质量为 $1000\mathrm{u}\sim 5000\mathrm{u}$ 。

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度应为分析纯或高于分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 硫酸。

A.2.1.2 硫酸铜 $(\mathrm{CuSO}_4\cdot 5\mathrm{H}_2\mathrm{O})$ 。

A.2.1.3 硫酸钾 $\mathrm{(K_2SO_4)}$ 。

A.2.1.4 混合试剂：硫酸溶液（质量分数 $25\%$ ）+钼酸铵溶液 $(25~\mathrm{g / L})$ +水十维生素C溶液 $(100~\mathrm{g / L})$ ， $1 + 1 + 2 + 1$ 。

# A.2.2 分析步骤

将酪蛋白磷酸肽含量的测定所得的干燥沉淀物(A.3.4)转移至凯氏消化管中，加入 $10~\mathrm{mL}$ 硫酸和 $1\textrm{g}$ 催化剂 $(0.1\mathrm{g}$ 硫酸铜和 $0.9\mathrm{g}$ 硫酸钾)，加热消化完全。取 $1\mathrm{mL}$ 消化液于试管中，加入 $1\mathrm{mL}$ 水，沸水浴 $10\mathrm{min}$ 。取出，冷却至室温，吸取 $100\mu \mathrm{L}$ 至另一试管中，加入 $1.9  \mathrm{mL}$ 水，再加入 $3\mathrm{mL}$ 混合试剂，于 $45^{\circ}C$ 水浴保温 $20\mathrm{min}$ 。取出，溶液应显蓝色。

# A.3 酪蛋白磷酸肽含量的测定

# A.3.1 方法提要

在试样溶液中加入一定浓度的钡离子，钡离子在酪蛋白磷酸肽分子间形成桥连，在适当的温度和 $\mathrm{pH}$ 下，加入一定浓度的乙醇即可使桥连状的酪蛋白磷酸肽分子团沉淀下来，将沉淀干燥并称重，得到产品中酪蛋白磷酸肽的含量。

# A.3.2 试剂和材料

A.3.2.1 无水乙醇：用前冷藏至 $4^{\circ}\mathrm{C}$ 。

A.3.2.2 氯化钡溶液： $100\mathrm{g / L}$

A.3.2.3 盐酸溶液：取 $18\mathrm{mL}$ 盐酸慢慢加入 $100~\mathrm{mL}$ 水中。

A.3.2.4 氢氧化钠溶液： $80~\mathrm{g / L}$

# A.3.3 仪器和设备

冷冻离心机。

# A.3.4 分析步骤

称取试样 $0.5\mathrm{g}\sim 1.5\mathrm{g}$ ，精确至 $0.0002\mathrm{g}$ ，置于一个 $50~\mathrm{mL}$ 离心管A中，加入 $15~\mathrm{mL}$ 水使其完全溶解。用盐酸溶液调节试样溶液的 $\mathrm{pH}$ 至4.6，然后置于冷冻离心机中，在约 $4^{\circ}C$ 下 $5000\mathrm{r / min}\sim 7000\mathrm{r / min}$ 离心 $30\mathrm{min}$ ，取上清液于预先在 $105^{\circ}C$ 干燥至恒重的一个 $50~\mathrm{mL}$ 离心管B中，用氢氧化钠溶液调节此上清液的 $\mathrm{pH}$ 值至6.8，控制溶液总量在 $20~\mathrm{mL}$ 以内，加入氯化钡溶液 $1.5~\mathrm{mL}$ ，再加入无水乙醇至 $50~\mathrm{mL}$ ，摇匀后于 $4^{\circ}C$ 冰箱中放置至少 $12\mathrm{h}$ 。然后，从冰箱中取出置于冷冻离心机中，在约 $4^{\circ}C$ 下 $5000\mathrm{r / min}\sim$ $7000\mathrm{r / min}$ 离心 $30\mathrm{min}$ ，弃去上清液得沉淀物。沉淀物先在 $60^{\circ}C\sim 70^{\circ}C$ 干燥箱中烘 $1\textrm{h}$ ，再升高温度至 $105^{\circ}C$ 烘至恒重。

# A.3.5 结果计算

酪蛋白磷酸肽含量的质量分数 $\omega$ 按式(A.1)计算：

$$
w = \frac {m _ {1} - m _ {2}}{m \times \left(1 - w _ {0}\right)} \times 100 \% \tag{A.1}
$$

式中：

$m_{1}$ ——恒重后离心管B和沉淀物的质量，单位为克(g)；

$m_{2}$ ——离心管B的质量，单位为克(g)；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$w_{0}$ ——试样的干燥减量， $\%$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

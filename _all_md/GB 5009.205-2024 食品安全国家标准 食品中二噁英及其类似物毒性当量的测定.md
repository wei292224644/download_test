# 前言

本标准代替GB5009.205—2013《食品安全国家标准 食品中二噁英及其类似物毒性当量的测定》。

本标准与GB5009.205—2013相比，主要变化如下：

——增加了第二法“同位素稀释-气相色谱-三重四极杆质谱法”；

——删除了世界卫生组织(WHO)1998年制定的毒性当量因子(TEF)；

——修改了第一法“同位素稀释-气相色谱-磁式高分辨质谱法”部分内容的表述和格式。

# 食品安全国家标准

# 食品中二噁英及其类似物毒性当量的测定

# 1 范围

本标准规定了食品中17种2,3,7,8-取代的多氯代二苯并二噁英、多氯代二苯并呋喃(PCDD/Fs)和12种二噁英样多氯联苯(DL-PCBs)含量及其毒性当量(TEQ)(见附录A的表A.1)的测定方法。

第一法“同位素稀释-气相色谱-磁式高分辨质谱法”适用于食品中17种PCDD/Fs和12种DL-PCBs含量及其TEQ的测定。

第二法“同位素稀释-气相色谱-三重四极杆质谱法”适用于肉及肉制品、水产动物及其制品、乳及乳制品、蛋及蛋制品和油脂中17种PCDD/Fs和12种DL-PCBs含量及其TEQ的测定。

# 第一法 同位素稀释-气相色谱-磁式高分辨质谱法

# 2 原理

试样经提取、净化和浓缩后，采用气相色谱-磁式高分辨质谱仪测定，稳定性同位素稀释法定量；以各目标化合物的毒性当量因子(TEF)与所测得的含量相乘后累加，计算样品中PCDD/Fs和DL-PCBs的TEQ。

# 3 试剂和材料

# 3.1 试剂

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

3.1.1 丙酮 $\mathrm{(C_3H_6O)}$ ：农残级。  
3.1.2 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：农残级。  
3.1.3 甲苯 $\left(\mathrm{C}_{7} \mathrm{H}_{8}\right)$ ：农残级。  
3.1.4 环己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{12}\right)$ ：农残级。  
3.1.5 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：农残级。  
3.1.6 甲醇 $\mathrm{(CH_3OH)}$ ：色谱级。  
3.1.7 正壬烷 $(\mathrm{C_9H_{20}})_{\bullet}\geqslant 99\%$   
3.1.8 乙酸乙酯 $\mathrm{(CH_3COOCH_2CH_3)}$ ：农残级。  
3.1.9 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：优级纯。  
3.1.10 浓硫酸 $\mathrm{(H_2SO_4)}$ ：优级纯。  
3.1.11 氢氧化钠 $\mathrm{(NaOH)}$ ：优级纯。  
3.1.12 硝酸银 $\mathrm{(AgNO_3)}$ ：优级纯。  
3.1.13 提取用硅藻土：堆积密度 $18.0\mathrm{g} / 100\mathrm{mL}\sim 36.0\mathrm{g} / 100\mathrm{mL}$ （如EXtrelutNT，或等效产品）。  
3.1.14 凝胶色谱填料：聚苯乙烯凝胶， $38\mu \mathrm{m}\sim 75\mu \mathrm{m}$ （如Bio-BeadsS-X3，或等效产品）。

3.1.15 硅胶： $75\mu \mathrm{m} \sim 250\mu \mathrm{m}$ 。  
3.1.16 碱性氧化铝： $63\mu \mathrm{m}\sim 200\mu \mathrm{m}$ （如EcoChrom™MPAluminaB-SuperI，或等效产品）。  
3.1.17 弗罗里土： $150\mu \mathrm{m}\sim 250\mu \mathrm{m}$   
3.1.18 活性炭： $150 \mu \mathrm{m} \sim 180 \mu \mathrm{m}$ （如Carbopack C, Supelco 10258，或等效产品）。  
3.1.19 净化用硅藻土： $26\mu \mathrm{m}$ （如 Celite 545 AW, Supelco 20199-U，或等效产品）。

# 3.2 试剂配制

3.2.1 活性硅胶：硅胶先后用甲醇和二氯甲烷清洗，待溶剂挥干后在 $600^{\circ}\mathrm{C}$ 下至少烘烤 $12\mathrm{h}$ ，临用现配。  
3.2.2 酸化硅胶(44%,质量分数):称取 $112.0\mathrm{~g}$ 活性硅胶于 $250\mathrm{~mL}$ 具塞磨口旋转烧瓶中,加入 $88.0\mathrm{~g}$ 浓硫酸,密闭后置于摇床上振荡,直至硅胶呈均匀流动状态,临用现配。  
3.2.3 碱化硅胶( $33\%$ ，质量分数)：称取 $100.0\mathrm{g}$ 活性硅胶于 $250~\mathrm{mL}$ 具塞磨口旋转烧瓶中，加入 $49.0\mathrm{g}$ $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液，密闭后置于摇床上振荡，直至硅胶呈均匀流动状态，临用现配。  
3.2.4 硝酸银硅胶：称取 $10.0\mathrm{g}$ 硝酸银于 $250~\mathrm{mL}$ 具塞磨口旋转烧瓶中，加入 $40~\mathrm{mL}$ 水使之溶解，再慢慢加入 $90.0\mathrm{g}$ 活性硅胶，密闭后置于摇床上振荡，直至硅胶呈均匀流动状态，临用现配。  
3.2.5 碱性氧化铝：取碱性氧化铝在 $600^{\circ}\mathrm{C}$ 下活化 $24\mathrm{h}$ ，临用现制。  
3.2.6 含水弗罗里土（1%，质量分数）：取适量弗罗里土装入索氏提取器中，用正己烷：二氯甲烷（1：1，体积比)提取 $24\mathrm{h}$ ，待溶剂挥干后，称取 $99.0\mathrm{g}$ ，加入 $1.0\mathrm{mL}$ 水，在密闭容器中（如具塞玻璃烧瓶或具塞玻璃三角瓶)混合均匀，临用现配。  
3.2.7 混合活性炭：称取 $9.0\mathrm{g}$ 活性炭和 $41.0\mathrm{g}$ 净化用硅藻土，在密闭容器中（如具塞玻璃烧瓶或具塞玻璃三角瓶）混和均匀，然后 $130^{\circ}\mathrm{C}$ 活化 $6\mathrm{h}$ ，临用现配。  
3.2.8 无水硫酸钠：取无水硫酸钠在 $660^{\circ}\mathrm{C}$ 下灼烧 $6\mathrm{h}$ ，临用现制。

# 5 分析步骤

# 5.1 试样制备

5.1.1 一般食品样品：干制食品粉碎均匀后，称取 $10\mathrm{g}\sim 20\mathrm{g}$ （精确到 $0.01\mathrm{g}$ )备用；乳粉称取 $15\mathrm{g}$ （精确到 $0.01\mathrm{g}$ )备用；其他样品匀质化后，称取 $50\mathrm{g}\sim 200\mathrm{g}$ （精确到 $0.01\mathrm{g}$ )，经过冷冻干燥后备用。  
5.1.2 油脂样品：直接称取 $5\mathrm{g}$ 样品（精确到 $0.01\mathrm{g}$ ）于茄形瓶中，加入PCDD/Fs和DL-PCBs同位素标记定量内标标准溶液各 $10\mu \mathrm{L}$ ，待净化。

# 5.2 提取

# 5.2.1 索氏提取

提取前，在索氏提取器(4.10)中装入一支空的提取套筒，以正己烷：二氯甲烷(1:1,体积比)为提取溶剂，以3次/h～4次/h的回流速度预提取8h后取出晾干。

将试样(5.1.1)置于陶瓷研钵中，加入无水硫酸钠研磨，制成能自由流动的粉末。转移至预先清洗的套筒中，分别加入PCDD/Fs和DL-PCBs同位素标记定量内标标准溶液各 $10~\mu \mathrm{L}$ ，用玻璃棉盖住样品，平衡 $30\mathrm{min}$ 后装入索氏提取器中，以正己烷：二氯甲烷 $(1:1$ ，体积比)为提取溶剂，以3次/h～4次/h的回流速度提取 $18\mathrm{h}\sim 24\mathrm{h}$ 。

# 5.2.2 加速溶剂提取

将试样(5.1.1)置于陶瓷研钵中，研磨后与适量硅藻土混合均匀，转移至提取池中，分别加入PCDD/Fs和DL-PCBs同位素标记定量内标标准溶液各 $10~\mu \mathrm{L}$ ，密闭后置于加速溶剂提取仪上提取，提取参考条件为：提取溶剂：正己烷：二氯甲烷（1：1，体积比）；压力： $10.3\mathrm{MPa}$ ；温度： $150^{\circ}\mathrm{C}$ ；静态提取时间： $10\mathrm{min}$ ；循环1次。

# 5.2.3 脂肪称量

准确称量茄形瓶重量 $m_{1}$ (精确至 $0.01\mathrm{g}$ )，将提取液(5.2.1或5.2.2)转移至茄形瓶中，用旋转蒸发仪

将溶剂蒸干后准确称重 $m_{2}$ （精确至 $0.01\mathrm{g}$ )，然后按式(1)计算样品中脂肪含量：

$$
X = \frac {m _ {2} - m _ {1}}{m} \times 100 \% \tag{1}
$$

式中：

$X$ ——脂肪含量；

$m_{2}$ ——溶剂蒸干后茄形瓶的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——初始茄形瓶的质量，单位为克 $(\mathrm{g})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

# 5.3 脂肪去除

# 5.3.1 酸化硅胶除脂

用 $150~\mathrm{mL}$ 正己烷溶解试样(5.1.2)或称量脂肪后的试样(5.2.3)，再加入 $50\mathrm{g}$ 酸化硅胶（ $44\%$ ，质量分数），用旋转蒸发仪在大气压下以 $60^{\circ}\mathrm{C}$ 旋转加热 $20\mathrm{min}$ 。静置 $8\mathrm{min} \sim 10\mathrm{min}$ 后，将上清液倒入茄形瓶中。再用 $50~\mathrm{mL}$ 正己烷清洗瓶中的酸化硅胶，合并上清液至该茄形瓶中。如果酸化硅胶颜色较深，则应重复上述过程，直至酸化硅胶为浅黄色。将合并后的上清液用旋转蒸发仪浓缩至 $3\mathrm{mL} \sim 5\mathrm{mL}$ ，供下一步净化用。浓缩过程中避免暴沸。

# 5.3.2 凝胶渗透色谱除脂

分别以甲苯、环己烷：乙酸乙酯(1:1,体积比)冲洗凝胶渗透色谱柱(GPC)，然后用 $5\mathrm{mL}\sim 10\mathrm{mL}$ 环己烷：乙酸乙酯(1：1，体积比)溶解试样(5.1.2)或称量脂肪后的试样(5.2.3)，用 $10~\mathrm{mL}$ 环己烷：乙酸乙酯(1：1，体积比)分两次洗涤茄形瓶，一并转入柱内。先用 $100~\mathrm{mL}$ 环己烷：乙酸乙酯(1：1，体积比)淋洗，再用 $90~\mathrm{mL}$ 环己烷：乙酸乙酯(1：1，体积比)以1滴/s～2滴/s的流速洗脱，用茄形瓶接收洗脱液，再用旋转蒸发仪浓缩至 $3\mathrm{mL}\sim 5\mathrm{mL}$ ，供下一步净化用。浓缩过程中避免暴沸。

根据实验室条件，选择使用酸化硅胶除脂和/或凝胶渗透色谱除脂。

# 5.4 净化分离

# 5.4.1 填充柱净化

# 5.4.1.1 复合硅胶柱净化

取内径 $15\mathrm{mm}$ 玻璃层析柱，底部填以适量玻璃棉，依次装入 $2\mathrm{g}$ 活性硅胶、 $5\mathrm{g}$ 碱化硅胶、 $2\mathrm{g}$ 活性硅胶、 $10\mathrm{g}$ 酸化硅胶、 $2\mathrm{g}$ 活性硅胶、 $5\mathrm{g}$ 硝酸银硅胶、 $2\mathrm{g}$ 活性硅胶和 $2\mathrm{g}$ 无水硫酸钠，干法装柱，轻敲层析柱，使其分布均匀。

先用 $150~\mathrm{mL}$ 正已烷预淋洗，当液面降至无水硫酸钠层上方约 $2\mathrm{mm}$ 时，关闭柱阀，弃去淋洗液。检查层析柱，如果出现沟流现象应重新装柱。加入除脂浓缩后的提取液，并用 $5\mathrm{mL}$ 正己烷分两次洗涤茄形瓶(5.3)，一并加入柱中，打开柱阀，当液面降至无水硫酸钠层时，加入 $400~\mathrm{mL}$ 正己烷，以1滴/s～2滴/s的流速洗脱，用茄形瓶收集洗脱液，用旋转蒸发仪浓缩至 $3\mathrm{mL}\sim 5\mathrm{mL}$ ，供下一步净化用。浓缩过程中避免暴沸。

# 5.4.1.2 碱性氧化铝柱分离

取内径 $15\mathrm{mm}$ 玻璃层析柱，底部填以适量玻璃棉，依次装入 $25\mathrm{g}$ 碱性氧化铝、 $10\mathrm{g}$ 无水硫酸钠。干法装柱，轻敲层析柱，使其分布均匀。先用 $150\mathrm{mL}$ 正已烷预淋洗，当液面降至氧化铝上方约 $2\mathrm{mm}$ 时，关闭柱阀。弃去淋洗液。检查层析柱，如果出现沟流现象应重新填柱。加入经混合硅胶柱净化的提

取液，并用 $5\mathrm{mL}$ 正己烷分两次洗涤茄形瓶(5.4.1.1)，一并加入柱中。用 $60~\mathrm{mL}$ 正己烷淋洗氧化铝柱，弃去淋洗液。用 $90~\mathrm{mL}$ 甲苯以 $1\mathrm{mL} / \mathrm{min}\sim 2\mathrm{mL} / \mathrm{min}$ 流速洗脱，用茄形瓶收集洗脱液，供DL-PCBs分析用。再用 $200~\mathrm{mL}$ 正已烷：二氯甲烷 $(1:1$ ，体积比)以1滴/s～2滴/s的流速洗脱，用茄形瓶收集洗脱液，供PCDD/Fs分析用。各洗脱液分别用旋转蒸发仪浓缩至 $1\mathrm{mL}\sim 2\mathrm{mL}$ ，再加入 $50~\mathrm{mL}$ 正己烷，浓缩至 $1\mathrm{mL}\sim 2\mathrm{mL}$ 。浓缩过程中避免暴沸。

# 5.4.1.3 碱性氧化铝柱净化

含PCDD/Fs组分洗脱液的进一步净化：取内径 $8\mathrm{mm}$ 玻璃层析柱，底部填以适量玻璃棉，依次装入 $2.5\mathrm{g}$ 碱性氧化铝和 $2\mathrm{g}$ 无水硫酸钠。干法装柱，轻敲层析柱，使其分布均匀。用 $20~\mathrm{mL}$ 正己烷预淋洗，弃去淋洗液。加入浓缩后洗脱液。用 $40~\mathrm{mL}$ 正己烷：二氯甲烷 $(98:2$ ，体积比)淋洗，弃去淋洗液。用 $30~\mathrm{mL}$ 正己烷：二氯甲烷(1：1，体积比)以1滴/s～2滴/s的流速洗脱，用茄形瓶收集洗脱液，用旋转蒸发仪浓缩至 $1\mathrm{mL}\sim 2\mathrm{mL}$ 。浓缩过程中避免暴沸。

含DL-PCBs组分洗脱液的进一步净化：取内径 $8\mathrm{mm}$ 玻璃层析柱，底部填以适量玻璃棉，依次装入 $2.5\mathrm{g}$ 碱性氧化铝和 $2\mathrm{g}$ 无水硫酸钠。干法装柱，轻敲层析柱，使其分布均匀。用 $30~\mathrm{mL}$ 正已烷：二氯甲烷 $(99:1$ ，体积比)预淋洗，弃去淋洗液。加入浓缩后的洗脱液，用 $15~\mathrm{mL}$ 正已烷：二氯甲烷 $(1:1$ 体积比)以1滴/s～2滴/s的流速洗脱，用茄形瓶收集洗脱液，用旋转蒸发仪浓缩至 $1\mathrm{mL}\sim 2\mathrm{mL}$ 。浓缩过程中避免暴沸。

# 5.4.2 全自动样品净化系统净化

将酸碱复合硅胶柱、碱性氧化铝柱和活性炭净化柱等按顺序连接在全自动样品净化系统上，按程序配好各洗脱溶液并连接好管路(参见图D.1)，将除脂浓缩后的提取液转移到全自动样品净化系统的进样管。按照洗脱程序顺序洗脱(参见表D.1)，对样品进行净化、分离，用茄形瓶分别收集含PCDD/Fs和DL-PCBs组分的洗脱液，用旋转蒸发仪浓缩至 $1\mathrm{mL}\sim 2\mathrm{mL}$ ，再加入 $50~\mathrm{mL}$ 正己烷，浓缩至 $1\mathrm{mL}\sim$ $2\mathrm{mL}$ 。浓缩过程中避免暴沸。

# 5.4.3 补充净化方法

# 5.4.3.1 活性炭柱净化

取 $10\mathrm{mL}$ 一次性玻璃移液管，切除两端，制成约 $10~\mathrm{cm}$ 长玻璃管，装入适量玻璃棉并塞紧，再装入 $0.55\mathrm{g}$ 混合活性炭，随后装入适量玻璃棉压实并塞紧。先后用如下溶液进行预淋洗： $5\mathrm{mL}$ 甲苯、 $2\mathrm{mL}$ 二氯甲烷：甲醇：甲苯 $(15:4:1$ ，体积比)溶液、 $1\mathrm{mL}$ 二氯甲烷：环己烷(1：1，体积比)溶液和 $5\mathrm{mL}$ 正己烷。如果流速超过 $0.5\mathrm{mL / min}$ ，则弃用该活性炭柱。将浓缩到约 $1\mathrm{mL}$ 的PCDD/Fs组分净化液(5.4.1.3)移入活性炭柱，然后用 $1\mathrm{mL}$ 正己烷洗涤原瓶两次，并移入活性炭柱内。分别以 $3\mathrm{mL}$ 正己烷、 $2\mathrm{mL}$ 二氯甲烷：环己烷(1：1，体积比)溶液、 $2\mathrm{mL}$ 二氯甲烷：甲醇：甲苯 $(15:4:1$ ，体积比)溶液淋洗，弃去淋洗液。倒置活性炭柱，以 $20~\mathrm{mL}$ 甲苯以1滴/s～2滴/s的流速洗脱。用茄形瓶收集洗脱液，用旋转蒸发仪浓缩至 $1\mathrm{mL}\sim 2\mathrm{mL}$ ，再加入 $50~\mathrm{mL}$ 正己烷，浓缩至 $1\mathrm{mL}\sim 2\mathrm{mL}$ ，待进一步处理。浓缩过程中避免暴沸。

# 5.4.3.2 弗罗里土柱净化

取直径为 $15 \mathrm{~mm}$ 的玻璃柱，底部填上玻璃棉后，依次装入 $2 \mathrm{~g}$ 无水硫酸钠、 $15 \mathrm{~g}$ 弗罗里士、 $2 \mathrm{~g}$ 无水硫酸钠。以正己烷湿法装柱，轻敲层析柱，使其分布均匀。用 $200 \mathrm{~mL}$ 正己烷淋洗，当液面至距无水硫

酸钠层约 $2\mathrm{mm}$ 时，关闭柱阀，弃去淋洗液。将浓缩到约 $1\mathrm{mL}$ 的PCDD/Fs组分净化液(5.4.1.3)移入该弗罗里土柱，打开柱阀。用 $200~\mathrm{mL}$ 正已烷淋洗，弃去淋洗液。用 $300~\mathrm{mL}$ 二氯甲烷以1滴/s～2滴/s的流速洗脱，用茄形瓶收集洗脱液，用旋转蒸发仪浓缩至 $1\mathrm{mL}\sim 2\mathrm{mL}$ ，待进一步处理。浓缩过程中避免暴沸。

# 5.5 浓缩

# 5.9 标准曲线的制作

# 5.9.1 相对响应因子

将PCDD/Fs校正标准溶液和DL-PCBs校正标准溶液分别按浓度由低到高的顺序注入GC-HRMS中，得到峰面积。各目标化合物的相对响应因子(RRF)按式(3)计算。式(3)适用于除1,2,3,7,8,9-HxCDD和OCDF之外的其他PCDD/Fs和DL-PCBs。

$$
\mathrm {R R F} = \frac {\left(A _ {\mathrm {n 1}} + A _ {\mathrm {n 2}}\right) \times c _ {1}}{\left(A _ {\mathrm {l l}} + A _ {\mathrm {l 2}}\right) \times c _ {\mathrm {n}}} \tag {3}
$$

式中：

$A_{\mathrm{n1}}$ 和 $A_{\mathrm{n2}}$ ——目标化合物的第一个和第二个质量数离子的峰面积；

$c_{1}$ ——定量内标的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A_{11}$ 和 $A_{12}$ ——定量内标的第一个和第二个质量数离子的峰面积；

$c_{\mathrm{n}}$ ——目标化合物的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ 。

在校正标准溶液的浓度范围内，每个化合物的RRF的相对标准偏差 $\leqslant 20\%$

# 5.9.2 响应因子

# 5.9.2.1 1,2,3,7,8,9-HxCDD和OCDF的响应因子

将PCDD/Fs校正标准溶液按浓度由低到高的顺序注入GC-HRMS中，得到峰面积。1,2,3,7,8,9-HxCDD和OCDF的响应因子(RF)按式(4)计算(1,2,3,7,8,9-HxCDD采用 $^{13}\mathrm{C}_{12}-1,2,3,6,7,8-\mathrm{Hx}-$ CDD作为定量内标，OCDF采用 $^{13}\mathrm{C}_{12}-$ OCDD作为定量内标)。

$$
\mathrm {R F} = \frac {\left(A _ {\mathrm {n l}} + A _ {\mathrm {n 2}}\right) \times c _ {1}}{\left(A _ {1 1} + A _ {1 2}\right) \times c _ {\mathrm {n}}} \tag {4}
$$

式中：

$A_{\mathrm{n1}}$ 和 $A_{\mathrm{n2}}$ ——目标化合物的第一个和第二个质量数离子的峰面积；

$c_{1}$ ——定量内标的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A_{11}$ 和 $A_{12}$ ——定量内标的第一个和第二个质量数离子的峰面积；

$c_{\mathrm{n}}$ ——目标化合物的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ 。

在校正标准溶液的浓度范围内，每个化合物的RF的相对标准偏差 $\leqslant 35\%$

# 5.9.2.2 定量内标的响应因子

定量内标的响应因子 $(\mathrm{RF}_l)$ 按式(5)计算。

$$
\mathrm {R F} _ {l} = \frac {\left(A _ {\mathrm {l l}} + A _ {\mathrm {l 2}}\right) \times c _ {\mathrm {r}}}{\left(A _ {\mathrm {r l}} + A _ {\mathrm {r 2}}\right) \times c _ {\mathrm {l}}} \tag {5}
$$

式中：

$A_{11}$ 和 $A_{12}$ ——定量内标的第一个和第二个质量数离子的峰面积；

$c_{\mathrm{r}}$ ——回收率内标的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A_{\mathrm{r1}}$ 和 $A_{\mathrm{r2}}$ ——回收率内标的第一个和第二个质量数离子的峰面积；

$c_{1}$ ——定量内标的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ 。

在校正标准溶液的浓度范围内，每个化合物的RF的相对标准偏差 $\leqslant 35\%$

# 13.9 标准曲线的制作

# 13.9.1 相对响应因子

将PCDD/Fs校正标准溶液和DL-PCBs校正标准溶液分别按浓度由低到高的顺序注入GC-MS/

MS中，得到峰面积。各目标化合物的相对响应因子(RRF)按式(13)计算。式(13)适用于除1,2,3,7，8,9-HxCDD和OCDF之外的其他PCDD/Fs和DL-PCBs。

$$
\mathrm {R R F} = \frac {\left(A _ {\mathrm {n} 1} + A _ {\mathrm {n} 2}\right) \times c _ {1}}{\left(A _ {\mathrm {l} 1} + A _ {\mathrm {l} 2}\right) \times c _ {\mathrm {n}}} \tag {13}
$$

式中：

$A_{\mathrm{n1}}$ 和 $A_{\mathrm{n2}}$ ——目标化合物的第一个和第二个离子对的峰面积；

$c_{1}$ ——定量内标的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A_{11}$ 和 $A_{12}$ ——定量内标的第一个和第二个离子对的峰面积；

$c_{\mathrm{n}}$ ——目标化合物的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ 。

在校正标准溶液的浓度范围内，每个化合物的RRF的相对标准偏差 $\leq 20\%$

# 13.9.2 响应因子

# 13.9.2.1 1,2,3,7,8,9-HxCDD和OCDF的响应因子

将PCDD/Fs校正标准溶液按浓度由低到高的顺序注入GC-MS/MS中，得到峰面积。1,2,3,7,8,9-HxCDD和OCDF的响应因子(RF)按式(14)计算(1,2,3,7,8,9-HxCDD采用 $^{13}\mathrm{C}_{12}-1,2,3,6,7,8-$ HxCDD作为定量内标，OCDF采用 $^{13}\mathrm{C}_{12}-$ OCDD作为定量内标)。

$$
\mathrm {R F} = \frac {\left(A _ {\mathrm {n} 1} + A _ {\mathrm {n} 2}\right) \times c _ {1}}{\left(A _ {1 1} + A _ {1 2}\right) \times c _ {\mathrm {n}}} \tag {14}
$$

式中：

$A_{\mathrm{n1}}$ 和 $A_{\mathrm{n2}}$ ——目标化合物的第一个和第二个离子对的峰面积；

$c_{1}$ ——定量内标的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A_{11}$ 和 $A_{12}$ ——定量内标的第一个和第二个离子对的峰面积；

$c_{\mathrm{n}}$ ——目标化合物的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ 。

在校正标准溶液的浓度范围内，每个化合物的RF的相对标准偏差 $\leq 35\%$

# 13.9.2.2 定量内标的响应因子

定量内标的响应因子按式(15)计算。

$$
\mathrm {R F} _ {1} = \frac {\left(A _ {1 1} + A _ {1 2}\right) \times c _ {\mathrm {r}}}{\left(A _ {\mathrm {r} 1} + A _ {\mathrm {r} 2}\right) \times c _ {1}} \tag {15}
$$

式中：

$A_{11}$ 和 $A_{12}$ ——定量内标的第一个和第二个离子对的峰面积；

$c_{\mathrm{r}}$ ——回收率内标的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A_{\mathrm{r1}}$ 和 $A_{\mathrm{r2}}$ ——回收率内标的第一个和第二个离子对的峰面积；

$c_{1}$ ——定量内标的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ 。

在校正标准溶液的浓度范围内，每个化合物的RF的相对标准偏差 $\leq 35\%$

# 牛奶中赛拉嗪残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了牛奶中赛拉嗪残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于牛奶中赛拉嗪残留量的定性、定量检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

牛奶中残留的赛拉嗪, 经乙腈提取并沉淀蛋白, 固相萃取柱净化, 液相色谱串联质谱仪测定, 基质匹配标准溶液外标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.3 浓氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$   
5.1.4 甲酸（HCOOH）。

# 5.2 溶液的配制

5.2.1 $2\%$ 甲酸甲醇溶液：取甲酸 $2\mathrm{mL}$ ，加甲醇稀释至 $100~\mathrm{mL}$ ，混匀，即得  
5.2.2 $0.5\%$ 氨水溶液：取浓氨水 $0.5 \mathrm{~mL}$ ，加水稀释至 $100 \mathrm{~mL}$ ，混匀，即得。  
5.2.3 $20\%$ 乙腈溶液：取乙腈 $200~\mathrm{mL}$ ，加水稀释至 $1000~\mathrm{mL}$ ，混匀，即得。  
5.2.4 流动相A： $0.1\%$ 甲酸乙腈溶液：取甲酸 $0.1\mathrm{mL}$ ，加乙腈稀释至 $100~\mathrm{mL}$ ，混匀，即得。  
5.2.5 流动相 $\mathrm{B}:0.1\%$ 甲酸溶液：取甲酸 $0.1\mathrm{mL}$ ，加水稀释至 $100~\mathrm{mL}$ ，混匀，即得。

# 5.3 标准品

赛拉嗪（Xylazine， $\mathrm{C_{12}H_{16}N_2S}$ ，CAS号：7361-61-7），含量 $\geq 98.0\%$

# 5.4 标准溶液的制备

5.4.1 标准储备液 $(1\mathrm{mg / mL})$ ：取赛拉嗪标准品约 $10\mathrm{mg}$ ，精密称定，用乙腈溶解并定容至 $10~\mathrm{mL}$ 容量瓶中摇匀，即得。 $-18^{\circ}C$ 以下保存，有效期3个月。  
5.4.2 标准中间液 $(10\mu \mathrm{g / mL})$ ：精密量取上述标准储备液 $100~\mu \mathrm{L}$ ，于 $10~\mathrm{mL}$ 容量瓶中，加乙腈稀释至刻度，摇匀，即得。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1个月。

5.4.3 标准工作液 $(100\mathrm{ng / mL})$ ：精密量取上述标准中间液 $100~\mu \mathrm{L}$ ，于 $10~\mathrm{mL}$ 容量瓶中，加 $20\%$ 乙腈溶液稀释至刻度，摇匀，即得。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1个月。

5.5 材料  
5.5.1 固相萃取柱： $\mathrm{HLB}(60\mathrm{mg} / 3\mathrm{mL})$ ，或相当者。  
5.5.2 微孔滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 天平：感量 $0.01\mathrm{g}$   
6.3 分析天平：感量 $0.00001\mathrm{g}$   
6.4 涡旋混合器。  
6.5 振荡器。  
6.6 离心机。  
6.7 氮吹仪。  
6.8 超声波仪。

# 7 试料的制备与保存

# 7.1 试料的制备

试料的制备包括：

a）取混匀后的供试样品，作为供试试料；  
b）取混匀后的空白样品，作为空白试料；  
c）取混匀后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试料 $2(\pm 0.02)\mathrm{g}$ ，加乙腈 $8\mathrm{mL}$ ，涡旋混合 $20\mathrm{s}$ ，振荡提取 $5\mathrm{min}$ ， $5^{\circ}\mathrm{C}$ 下 $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液于 $10\mathrm{mL}$ 离心管， $40^{\circ}\mathrm{C}$ 水浴氮气流下浓缩至小于 $2\mathrm{mL}$ ，加 $0.5\%$ 的氨水溶液 $4\mathrm{mL}$ ，涡旋混合，备用。

# 8.2 净化

依次用甲醇 $3\mathrm{mL}$ 、水 $3\mathrm{mL}$ 、 $0.5\%$ 氨水溶液 $3\mathrm{mL}$ 活化固相萃取柱。取备用液全部过柱，用水 $3\mathrm{mL}$ 淋洗，真空抽干，用 $2\%$ 甲酸甲醇溶液 $3\mathrm{mL}$ 洗脱。将洗脱液于 $40^{\circ}\mathrm{C}$ 水浴中氮气吹干，残余物中加入 $20\%$ 乙腈溶液 $0.5\mathrm{mL}$ ，涡旋后超声溶解 $5\mathrm{min}$ ，经 $0.2\mu \mathrm{m}$ 滤膜过滤后，供液相色谱-串联质谱仪分析。

# 8.3 基质匹配标准曲线的制备

精密量取标准工作液适量，用 $20\%$ 乙腈溶液稀释成 $0.5 \mathrm{ng} / \mathrm{mL}, 1.0 \mathrm{ng} / \mathrm{mL}, 2.0 \mathrm{ng} / \mathrm{mL}, 5.0 \mathrm{ng} / \mathrm{mL}, 10 \mathrm{ng} / \mathrm{mL}, 20 \mathrm{ng} / \mathrm{mL}, 50 \mathrm{ng} / \mathrm{mL}$ 的系列工作溶液。将空白牛奶样品经提取、净化、氮气吹干后，用上述系列标准工作液 $0.5 \mathrm{~mL}$ 溶解，制成系列浓度的基质匹配标准溶液。采用液相色谱-串联质谱仪测定，以待测物的特征离子质量色谱峰面积为纵坐标，以基质匹配标准溶液浓度为横坐标，绘制标准工作曲线，计算线性方程及回归系数。

# 8.4 测定

# 8.4.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱 $(50\mathrm{mm}\times 2.1\mathrm{mm},1.7\mu \mathrm{m})$ 或相当者；

b) 流动相：A 为 $0.1\%$ 甲酸乙腈溶液，B 为 $0.1\%$ 甲酸溶液；  
c) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ ;   
d) 进样量: $10 \mu \mathrm{L}$ ;   
e) 柱温： $30^{\circ}\mathrm{C}$ ；  
f）流动相梯度洗脱程序： $0\mathrm{min}\sim 2.5\mathrm{min}$ ，A相由 $5 \%$ 线性变化至 $50\%$ . $2.5\mathrm{min}\sim 3\mathrm{min}$ ，保持A相比例 $100\% ;3\mathrm{min}\sim 4.5\mathrm{min}$ ，保持A相比例为 $5 \%$ 。

# 8.4.2 串联质谱参考条件

a）离子源：电喷雾离子源；  
b）扫描方式：正离子扫描  
c）检测方式：多反应监测（MRM）；  
d) 定性离子对、定量离子对、锥孔电压和碰撞能量参考值见表1。

# 8.5 空白试验

取空白试料，除不加药物外，采用完全相同的测定步骤进行测定。

# 9 结果计算和表述

试料中赛拉嗪的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V}{A _ {\mathrm {S}} \times m} \dots \tag {1}
$$

式中：

$X$ ——试料中赛拉嗪残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ ——基质匹配标准溶液中赛拉嗪浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$V$ ——溶解残余物所用溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

A ——试样溶液中赛拉嗪定量离子对的峰面积；

$A_{\mathrm{S}}$ 基质匹配标准溶液中赛拉嗪定量离子对的峰面积；

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

牛奶中赛拉嗪的检测限为 $0.2\mu \mathrm{g / kg}$ ，定量限为 $0.5\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 5\mu \mathrm{g} / \mathrm{kg}$ 添加浓度范围内，回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

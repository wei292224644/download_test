# 前言

本标准代替SB/T10417—2007《酱油中乙酰丙酸的测定方法》。

本标准与SB/T10417—2007相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中乙酰丙酸的测定”；  
将适用范围扩充至酱油、饮料（包括碳酸饮料、咖啡饮品、果汁饮料、乳饮料、植物蛋白饮料）、酱类、蜂蜜、乳粉、饼干、糕点、糖果、果冻、面制品；  
——增加了试剂的级别和分子式；  
——增加了液体试样和固体试样的制备方法；  
——增加了食品中乙酰丙酸测定方法的定量限；  
——修改了色谱图。

# 食品安全国家标准

# 食品中乙酰丙酸的测定

# 1 范围

本标准规定了用气相色谱法测定食品中乙酰丙酸的方法。

本标准适用于酱油、饮料（包括碳酸饮料、咖啡饮品、果汁饮料、乳饮料、植物蛋白饮料）、酱类、蜂蜜、乳粉、饼干、糕点、糖果、果冻、面制品中乙酰丙酸的测定。

# 第一法 内标法

# 2 原理

样品经酸化后，用乙醚提取乙酰丙酸，以正庚酸作为内标物，用配有氢火焰离子化检测器的气相色谱仪进行测定，以内标法进行定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，所用水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $650^{\circ}C$ 下灼烧 $4\mathrm{h}$ ，贮于干燥器中备用。  
3.1.2 无水乙醚 $(\mathrm{C_4H_{10}O})$ 。  
3.1.3 盐酸（HCl）。  
3.1.4 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.5 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。

# 3.2 试剂配制

饱和氯化钠溶液：称取 $370\mathrm{g}$ 氯化钠于烧杯中，加入 $1000~\mathrm{mL}$ 水，搅拌使其溶解、混匀，静置备用。

# 3.3 标准品

3.3.1 正庚酸 $(\mathrm{C}_7\mathrm{H}_{14}\mathrm{O}_2)$ ：纯度 $\geqslant 99.5\%$   
3.3.2 乙酰丙酸 $\mathrm{(C_5H_8O_3)}$ ：纯度 $\geqslant 99.5\%$

# 3.4 标准溶液配制

3.4.1 正庚酸标准储备溶液 $(5.00\mathrm{mg / mL})$ ：称取 $0.5\mathrm{g}$ 正庚酸(精确至 $0.0001\mathrm{g}$ )，用乙酸乙酯溶解并定容于 $100~\mathrm{mL}$ 容量瓶中，混匀。  
3.4.2 乙酰丙酸标准储备溶液 $(5.00\mathrm{mg / mL})$ ：称取 $0.5\mathrm{g}$ 乙酰丙酸(精确至 $0.0001\mathrm{g}$ )，用乙酸乙酯溶

解并定容于 $100~\mathrm{mL}$ 容量瓶中，混匀。

3.4.3 乙酰丙酸标准系列工作溶液：分别准确吸取乙酰丙酸标准储备液(3.4.2) $0.025\mathrm{mL},0.05\mathrm{mL}$ 、 $0.1\mathrm{mL},0.5\mathrm{mL},1.0\mathrm{mL},1.5\mathrm{mL}$ 于6个 $10~\mathrm{mL}$ 容量瓶中，各加入 $1.0~\mathrm{mL}$ 正庚酸标准储备液(3.4.1)，用乙酸乙酯定容，即得标准系列工作液，相当于每毫升含 $12.5\mu \mathrm{g}$ 、 $25.0\mu \mathrm{g}$ 、 $50.0\mu \mathrm{g}$ 、 $250.0\mu \mathrm{g}$ 、 $500.0\mu \mathrm{g}$ 、 $750.0\mu \mathrm{g}$ 的乙酰丙酸。

# 4 仪器和设备

4.1 气相色谱仪：配有氢火焰离子化检测器(FID)。  
4.2 分析天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.3 旋转蒸发仪。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 液体试样

准确称取 $5\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 试样于 $100\mathrm{mL}$ 具塞试管中，加入 $10\mathrm{mL}$ 饱和氯化钠溶液、 $1.0\mathrm{mL}$ 正庚酸标准储备液、 $3.0\mathrm{mL}$ 盐酸，充分振摇 $1\mathrm{min}$ 后，加入 $50\mathrm{mL}$ 乙醚，振摇萃取 $3\mathrm{min} \sim 5\mathrm{min}$ ，静置约 $10\mathrm{min} \sim 15\mathrm{min}$ 。待分层后，吸取上层乙醚萃取液于 $250\mathrm{mL}$ 分液漏斗中，再重复萃取两次，每次用乙醚 $50\mathrm{mL}$ ，合并乙醚萃取液，用 $10\mathrm{mL}$ 饱和氯化钠溶液洗涤两次，弃去下层。乙醚层用 $30\mathrm{g}$ 无水硫酸钠脱水，在 $45^{\circ}\mathrm{C}$ 左右浓缩至近干，用乙酸乙酯定容至 $10\mathrm{mL}$ ，充分摇匀，待气相色谱进样检测。

# 5.1.2 固体试样

准确称取充分粉碎均匀的试样 $5\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 于 $100\mathrm{mL}$ 具塞试管中，加入 $10\mathrm{mL}$ 饱和氯化钠溶液、 $1.0\mathrm{mL}$ 正庚酸标准储备液、 $3.0\mathrm{mL}$ 盐酸，充分振摇 $1\mathrm{min}$ 后，加入 $50\mathrm{mL}$ 乙醚，振摇萃取 $3\mathrm{min} \sim 5\mathrm{min}$ ，静置约 $10\mathrm{min} \sim 15\mathrm{min}$ 。待分层后，吸取上层乙醚萃取液于 $250\mathrm{mL}$ 分液漏斗中，再重复萃取两次，每次用乙醚 $50\mathrm{mL}$ ，合并乙醚萃取液，用 $10\mathrm{mL}$ 饱和氯化钠溶液洗涤两次，弃去下层。乙醚层用 $30\mathrm{g}$ 无水硫酸钠脱水，在 $45^{\circ}\mathrm{C}$ 左右浓缩至近干，用乙酸乙酯定容至 $10\mathrm{mL}$ ，充分摇匀，待气相色谱进样检测。

# 5.2 气相色谱参考条件

5.2.1 色谱柱：DB-WAX石英毛细管柱 $(30\mathrm{m}\times 0.25\mathrm{mm}\times 0.25\mu \mathrm{m})$ 或相当型号色谱柱。  
5.2.2 进样口温度： $270^{\circ}C$   
5.2.3 检测器温度： $270^{\circ}C$   
5.2.4 升温程序：初始柱温 $120^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $8^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $230^{\circ}\mathrm{C}$ ，保持 $4\mathrm{min}$ 。  
5.2.5 载气：高纯氮，纯度 $\geq 99.999\%$ ，流速： $1.0~\mathrm{mL / min}$ 。  
5.2.6 氢气流速： $30~\mathrm{mL / min}$ 。  
5.2.7 空气流速： $300~\mathrm{mL / min}$   
5.2.8 尾吹气： $30~\mathrm{mL / min}$ 。  
5.2.9 进样方式：分流进样，分流比 $25:1$ 。  
5.2.10 进样体积： $1\mu \mathrm{L}$

# 5.3 标准曲线的制作

将标准系列工作溶液分别注入气相色谱仪中，测定乙酰丙酸与内标物质的相应峰面积。以乙酰丙酸峰面积/内标物峰面积为纵坐标，以乙酰丙酸质量浓度为横坐标，绘制标准曲线。

乙酰丙酸标准溶液的气相色谱图参见图A.1。

# 5.4 试样溶液的测定

将试样溶液注入气相色谱仪中，得到乙酰丙酸和内标物质的相应峰面积，根据标准曲线得到待测液中乙酰丙酸的质量浓度。

# 5.5 空白试验

用 $5.0\mathrm{mL}$ 水代替试样，按 $5.1\sim 5.4$ 的步骤做空白试验。

# 6 分析结果的表述

乙酰丙酸的含量按式(1)计算：

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——样品中乙酰丙酸含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——测定用样品中乙酰丙酸质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样品的定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

方法检出限为 $10\mathrm{mg / kg}$ ，定量限为 $25~\mathrm{mg / kg}$

# 第二法 外标法

# 9 原理

样品经酸化后，用乙醚提取乙酰丙酸，用配有氢火焰离子化检测器的气相色谱仪进行分离测定，以外标法进行定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 10.1 试剂

10.1.1 无水乙醚 $(\mathrm{C_4H_{10}O})$ 。  
10.1.2 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $650^{\circ}C$ 以下灼烧 $4\mathrm{h}$ ，贮于干燥器中备用。  
10.1.3 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。  
10.1.4 盐酸（HCl）。  
10.1.5 氯化钠 $(\mathrm{NaCl})$ 。

# 10.2 试剂配制

盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ：量取 $50~\mathrm{mL}$ 盐酸，用水稀释至 $100~\mathrm{mL}$ 。

# 10.3 标准品

乙酰丙酸 $(\mathrm{C}_5\mathrm{H}_8\mathrm{O}_3)$ ：纯度 $\geq 99.5\%$

# 10.4 标准溶液配制

10.4.1 乙酰丙酸标准储备溶液 $(5.00\mathrm{mg / mL})$ ：准确称取 $0.5\mathrm{g}$ 乙酰丙酸（精确至 $0.0001\mathrm{g}$ ，用乙酸乙酯溶解并定容于 $100~\mathrm{mL}$ 容量瓶中，混匀。  
10.4.2 乙酰丙酸标准系列工作溶液：

分别准确吸取乙酰丙酸标准储备溶液(10.4.1) $0.0\mathrm{mL}, 0.05\mathrm{mL}, 0.1\mathrm{mL}, 0.5\mathrm{mL}, 1.0\mathrm{mL}, 1.5\mathrm{mL}$ 于6个 $10\mathrm{mL}$ 容量瓶中，用乙酸乙酯定容至刻度后混匀。此标准系列工作液相当于每毫升含 $0.0\mu \mathrm{g}$ 、 $25.0\mu \mathrm{g}, 50.0\mu \mathrm{g}, 250.0\mu \mathrm{g}, 500.0\mu \mathrm{g}, 750.0\mu \mathrm{g}$ 的乙酰丙酸。

# 11 仪器和设备

11.1 气相色谱仪：配有氢火焰离子化检测器(FID)。  
11.2 分析天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
11.3 旋转蒸发仪。  
11.4 超声波清洗器。  
11.5 离心机。

# 12 分析步骤

# 12.1 试样制备

# 12.1.1 液体试样

准确称取 $5\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $1\mathrm{mL}$ 盐酸溶液，充分振摇均匀，再加入 $25~\mathrm{mL}$ 乙醚，振摇萃取 $1\mathrm{min}$ ，静置分层，以 $4000\mathrm{r / min}$ 离心 $5\mathrm{min} \sim 10\mathrm{min}$ ，吸取上层乙醚萃取液于 $250~\mathrm{mL}$ 分液漏斗中，再重复萃取两次，每次用乙醚 $20~\mathrm{mL}$ ，合并乙醚提取液，乙醚层用 $15\mathrm{g}$ 无水硫酸钠脱水，将滤液浓缩至近干，用乙酸乙酯定容至 $5.0~\mathrm{mL}$ ，充分摇匀，待气相色谱进样检测。

# 12.1.2 固体试样

准确称取充分粉碎均匀的试样 $5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $2.0\mathrm{g}$ 氯化钠，加入

1 mL盐酸溶液, 加入 $25 \mathrm{~mL}$ 乙醚, 超声波提取 $10 \mathrm{~min}$ , 以 $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min} \sim 10 \mathrm{~min}$ , 吸取上层乙醚萃取液于 $250 \mathrm{~mL}$ 分液漏斗中, 再重复萃取两次, 每次用乙醚 $20 \mathrm{~mL}$ , 合并乙醚提取液, 乙醚层用 $15 \mathrm{~g}$ 无水硫酸钠脱水, 将滤液浓缩至近干, 用乙酸乙酯定容至 $5.0 \mathrm{~mL}$ , 充分摇匀, 待气相色谱进样检测。

# 12.2 仪器参考条件

12.2.1 色谱柱：DB-FFAP石英毛细管柱 $(30\mathrm{m}\times 0.25\mathrm{mm}\times 0.25\mu \mathrm{m})$ 或相当型号色谱柱。  
12.2.2 进样口温度： $260^{\circ}C$   
12.2.3 检测器温度： $280^{\circ}C$   
12.2.4 升温程序：初始柱温 $60^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $18^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $200^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，再以 $10^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $230^{\circ}\mathrm{C}$ ，保持 $12\mathrm{min}$ 。  
12.2.5 载气：高纯氮，纯度 $\geq 99.999\%$ ，流速 $1.0~\mathrm{mL / min}$   
12.2.6 氢气流速： $30~\mathrm{mL / min}$ 。  
12.2.7 空气流速： $300~\mathrm{mL / min}$   
12.2.8 尾吹气： $30~\mathrm{mL / min}$   
12.2.9 进样方式：分流进样，分流比 $25:1$ 。  
12.2.10 进样量： $1\mu \mathrm{L}$

# 12.3 标准曲线的制作

将标准系列工作液分别注入气相色谱仪中，测定相应的峰面积，以标准工作液的质量浓度为横坐标，以响应峰面积为纵坐标，绘制标准曲线。

乙酰丙酸标准溶液的气相色谱图参见图A.2。

# 12.4 试样溶液的测定

将试样溶液注入气相色谱仪中，得到响应峰面积，根据标准曲线得到待测液中乙酰丙酸的质量浓度。

# 13 分析结果的表述

乙酰丙酸含量按式(2)计算：

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——样品中乙酰丙酸含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——测定用样品中乙酰丙酸质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样品的定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$m$ ——样品质量，单位为克 $(\mathrm{g})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

方法检出限为 $10\mathrm{mg / kg}$ ，定量限为 $25~\mathrm{mg / kg}$

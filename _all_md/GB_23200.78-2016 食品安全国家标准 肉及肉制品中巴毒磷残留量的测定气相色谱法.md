# 食品安全国家标准

# 肉及肉制品中巴毒磷残留量的测定 气相色谱法

# 1 范围

本标准规定了肉及肉制品中巴毒磷残留量的测定方法。

本标准适用于猪肉、香肠和午餐肉等肉及肉制品中巴毒磷残留量的测定，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样中残留的巴毒磷经环己烷一乙酸乙酯混合溶剂提取，提取液经凝胶渗透色谱净化，用配有火焰光度检测器的气相色谱仪测定，外标法定量。如有必要，可用气相色谱一质谱法确证。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 丙酮 $(\mathrm{C}_7\mathrm{H}_6\mathrm{O})$ ：色谱纯。  
4.1.2 环己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{12}\right)$ ：色谱纯。  
4.1.3 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。

# 4.2 溶液配制

4.2.1 环己烷—乙酸乙酯(1+1，体积比)：取 $100\mathrm{mL}$ 环己烷，加入 $100\mathrm{mL}$ 乙酸乙酯，摇匀备用。  
4.2.2 无水硫酸钠： $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，储于密封容器中备用。

# 4.3 标准品

巴毒磷标准品 $\mathrm{(Croxyphos,C_{14}H_{19}O_6P,}$ CAS号：7700-17-6）：纯度 $\geqslant 99\%$ □

# 4.4 巴毒磷标准溶液配制

准确称取适量的巴毒磷标准品，用丙酮配制成浓度为 $1.0\mathrm{mg / mL}$ 的标准储备液，再根据需要用丙酮稀释成适当浓度的标准工作溶液。标准储备液在 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中保存，有效期为12个月；标准工作液在 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 中保存，有效期为6个月。

# 5 仪器和设备

5.1 气相色谱仪：火焰光度检测器，配有磷滤光片（ $526\mathrm{nm}$ ）。  
5.2 凝胶渗透色谱仪：配有单元泵、馏分收集器。  
5.3 离心机：转速 $>3000\mathrm{r / min}$   
5.4 电子天平：感量精确至 $0.0001\mathrm{g}$ 。

5.5 均质器。  
5.6 旋转蒸发器。  
5.7 浓缩瓶： $100~\mathrm{mL}$   
5.8 有机相微孔滤膜： $0.45\mu \mathrm{m}$   
5.9 无水硫酸钠柱：柱长 $7\mathrm{cm}$ 、内径 $2\mathrm{cm}$ 的桶形漏斗，内装约 $3\mathrm{cm}$ 无水硫酸钠。

# 6 试样制备与保存

# 6.1 试样制备

将所取全部样品缩分出具有代表性样品不少于 $500\mathrm{g}$ ，取样部位按GB2763的规定执行。经捣碎机充分捣碎均匀，均分成2份，分别装入洁净容器，密封，标明标记。

# 6.2 试样保存

将试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。在制样的操作过程中, 应防止样品受到污染或发生残留物含量的变化。

# 7 分析步骤

# 7.1 提取

称取试样约 $10.0 \mathrm{~g}$ (精确至 $0.1 \mathrm{~g}$ ) 于 $50 \mathrm{~mL}$ 离心管中, 加入 $10 \mathrm{~g}$ 无水硫酸钠, 用玻璃棒搅拌使之与样品充分混合。然后, 加入 $20 \mathrm{~mL}$ 环己烷一乙酸乙酯 $(1+1$ , 体积比), 均质 $2 \mathrm{~min}$ , 以 $3000 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ 。吸取上层有机相过无水硫酸钠柱, 收集于 $100 \mathrm{~mL}$ 浓缩瓶中。残渣再用 $40 \mathrm{~mL}$ 环己烷一乙酸乙酯 $(1+1$ , 体积比)按上述步骤分 2 次提取, 合并提取液于 $40^{\circ} \mathrm{C}$ 下浓缩至约 $5 \mathrm{~mL}$ 。将提取液转移至 $10 \mathrm{~mL}$ 刻度试管中, 并用 $3 \mathrm{~mL}$ 环己烷一乙酸乙酯 $(1+1$ , 体积比)分 3 次洗涤浓缩瓶, 合并洗涤液于刻度试管中。用环己烷一乙酸乙酯 $(1+1$ , 体积比)定容至 $10 \mathrm{~mL}$ , 过 $0.45 \mu \mathrm{m}$ 滤膜, 待凝胶渗透色谱(GPC)净化。

# 7.2 净化

# 7.2.1 凝胶渗透色谱(GPC)净化

# 7.2.2 凝胶渗透色谱条件

a）凝胶净化柱：柱长 $400\mathrm{mm}$ ，内径 $25\mathrm{mm}$ ,Bio Beads S-X3，或相当者；  
b) 流动相: 环己烷-乙酸乙酯(1+1,体积比);  
c) 流速: $5.0 \mathrm{~mL} / \mathrm{min}$ ;   
d) 样品定量环: $5.0 \mathrm{~mL}$ ;   
e) 预淋洗时间：17min;

f）凝胶渗透色谱仪平衡时间： $5\mathrm{min}$

收集时间：17 min~30 min。

# 7.2.3 凝胶渗透色谱净化步骤

将 $10 \mathrm{~mL}$ 待净化液按 7.2.2 规定的条件进行净化, 将所收集的组分于室温下氮吹浓缩近干, 加入 $1.0 \mathrm{~mL}$ 丙酮溶解定容, 供气相色谱测定。

# 7.3 测定

# 7.3.1 气相色谱参考条件

a）色谱柱：HP-5，柱长 $30\mathrm{m}$ ，内径 $250~\mu \mathrm{m}$ ，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）载气：氮气，纯度 $\geqslant 99.99\%$ ，流速 $1.5\mathrm{mL / min}$   
c) 尾吹气：氮气，纯度 $\geqslant 99.99\%$ ，流速 $60~\mathrm{mL / min}$   
d) 氢气：流速 $75 \mathrm{~mL} / \mathrm{min}$ ；  
e) 空气：流速 $100\mathrm{mL / min}$

f）色谱柱温度：初始温度为 $180^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $10^{\circ}\mathrm{C / min}$ 的速率升高到 $280^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ ；  
g）进样口温度： $250^{\circ}\mathrm{C}$   
h）检测器温度： $200^{\circ}\mathrm{C}$   
i）进样量： $1\mu \mathrm{L}$   
j）进样方式：不分流进样。

# 7.3.2 色谱测定

根据样液中巴毒磷含量情况，选定峰面积与样液相近的标准工作溶液。标准工作液和样液中巴毒磷的响应值均应在仪器检测的线性范围内。标准工作液与样液应等体积参插进样测定。在上述色谱条件下，巴毒磷的保留时间约为 $6.67\mathrm{min}$ 。巴毒磷标准物质的气相色谱图参见附录A，外标法定量。

# 7.4 气相色谱一质谱确证

# 7.4.1 气相色谱一质谱参考条件

a）色谱柱：Rtx-5MS，柱长 $30\mathrm{m}$ ，内径 $250~\mu \mathrm{m}$ ，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）载气：氦气，纯度 $\geqslant 99.99\%$ ，流速 $1.5\mathrm{mL / min}$   
c） 色谱柱温度： $180^{\circ}\mathrm{C}$   
d）程序升温：初始温度为 $180^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $10^{\circ}\mathrm{C / min}$ 的速率升高到 $280^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$   
e) 进样口温度: $250^{\circ} \mathrm{C}$ ;   
f）色谱—质谱接口温度： $280^{\circ}\mathrm{C}$   
g）进样量： $1\mu \mathrm{L}$   
h）进样方式：无分流， $1\mathrm{min}$ 后开阀；  
i）电离能量： $70\mathrm{eV}$   
j）电离方式：EI；  
k）测定方式：离子监测方式；  
1) 监测离子 $(m / z)$ : 193、166、127、105;  
m）溶剂延迟： $3\mathrm{min}$

# 7.4.2 质谱确证

如果样液与标准品溶液的气相色谱图中，在相同保留时间有峰出现，则可用质谱对其确证。对标准溶液及样液均按7.4.1规定的条件进行测定。

# 7.5 空白实验

除不加试样外，均按 $7.1\sim 7.4$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中巴毒磷的含量。

$$
X _ {i} = \frac {A _ {i} \times C _ {i} \times V}{A _ {i s} \times m}. \tag {1}
$$

式中：

$X_{i}$ ——试样中巴毒磷残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A_{i}$ ——样液中巴毒磷的峰高或峰面积；

$C_i$ ——标准工作液中巴毒磷的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{\mathrm{is}}$ ——标准工作液中巴毒磷的峰高或峰面积；

$m$ ——最终样液所代表的试样质量，单位为克(g)。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录B的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录 C 的要求。

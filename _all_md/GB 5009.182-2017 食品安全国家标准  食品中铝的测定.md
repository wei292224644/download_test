# 前言

本标准代替GB/T5009.182—2003《面制食品中铝的测定》、GB/T23374—2009《食品中铝的测定电感耦合等离子体质谱法》、GB/T18932.11—2002《蜂蜜中钾、磷、铁、钙、锌、铝、钠、镁、硼、锰、铜、钡、钛、钒、镍、钴、铬含量的测定方法 电感耦合等离子体原子发射光谱(ICP-AES)法》、SN/T2208—2008《水产品中钠、镁、铝、钙、铬、铁、镍、铜、锌、砷、锶、钼、镉、铅、汞、硒的测定 微波消解-电感耦合等离子体质谱法》中铝的测定方法。

本标准与GB/T5009.182—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中铝的测定”；  
——改进了第一法分光光度法；  
——增加电感耦合等离子体质谱法为第二法；  
——增加电感耦合等离子体发射光谱法为第三法；  
——增加石墨炉原子吸收光谱法为第四法。

# 食品安全国家标准

# 食品中铝的测定

# 1 范围

本标准规定了食品中铝含量测定的分光光度法、电感耦合等离子体质谱法、电感耦合等离子体发射光谱法和石墨炉原子吸收光谱法。

本标准第一法适用于检测使用含铝食品添加剂的食品中的铝，第二法、第三法和第四法适用于检测食品中的铝。

# 第一法 分光光度法

# 2 原理

试样经处理后，在乙二胺-盐酸缓冲液中 $(\mathrm{pH}6.7\sim 7.0)$ ，聚乙二醇辛基苯醚（TritonX-100)和溴代十六烷基吡啶(CPB)的存在下，三价铝离子与铬天青S反应生成蓝绿色的四元胶束，于 $620~\mathrm{nm}$ 波长处测定吸光度值并与标准系列比较定量。

# 3 试剂和材料

除非另有说明，所用试剂均为分析纯，实验用水为 $\mathrm{GB / T6682}$ 规定的三级水。

# 3.1 试剂

3.1.1 硝酸 $\mathrm{(HNO_3)}$ ：优级纯。  
3.1.2 硫酸 $\mathrm{(H_2SO_4)}$ ：优级纯。  
3.1.3 盐酸（HCl）：优级纯。  
3.1.4 氨水 $\left(\mathrm{NH}_{3} \cdot \mathrm{H}_{2} \mathrm{O}\right)$ ：优级纯。  
3.1.5 无水乙醇 $\mathrm{(C_2H_6O)}$ ：优级纯。  
3.1.6 对硝基苯酚 $(\mathrm{C_6H_5NO_3})$ 。   
3.1.7 铬天青 $\mathrm{S(C_{23}H_{13}O_9SCl_2Na_3)}$ 。  
3.1.8 乙二胺 $(\mathrm{C}_2\mathrm{H}_8\mathrm{N}_2)$ 。  
3.1.9 聚乙二醇辛基苯醚（Triton X-100）。  
3.1.10 溴代十六烷基吡啶（CPB， $\mathrm{C}_{21}\mathrm{H}_{38}\mathrm{BrN}$ ）。  
3.1.11 抗坏血酸 $(\mathrm{C_6H_8O_6})$

# 3.2 试剂配制

3.2.1 盐酸溶液 $(1 + 1)$ ：量取 $50~\mathrm{mL}$ 盐酸与 $50~\mathrm{mL}$ 水混合均匀。  
3.2.2 硫酸溶液 $(1\%)$ ：吸取 $1\mathrm{mL}$ 硫酸缓慢加入到 $80~\mathrm{mL}$ 水中，放冷后用水稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.3 对硝基苯酚乙醇溶液 $(1\mathrm{g} / \mathrm{L})$ ：称取 $0.1\mathrm{g}$ 对硝基苯酚，溶于 $100~\mathrm{mL}$ 无水乙醇中，混匀。

3.2.4 硝酸溶液 $(5\%)$ ：量取 $5\mathrm{mL}$ 硝酸，加水定容至 $100~\mathrm{mL}$ ，混匀。  
3.2.5 硝酸溶液 $(2.5\%)$ ：量取 $2.5\mathrm{mL}$ 硝酸，加水定容至 $100~\mathrm{mL}$ ，混匀。  
3.2.6 氨水溶液 $(1 + 1)$ ：量取 $10\mathrm{mL}$ 氨水，加入 $10\mathrm{mL}$ 水中，混匀。  
3.2.7 硝酸溶液 $(2 + 98)$ ：量取 $2\mathrm{mL}$ 硝酸与 $98~\mathrm{mL}$ 水混合均匀。  
3.2.8 乙醇溶液 $(1 + 1)$ ：量取 $50~\mathrm{mL}$ 无水乙醇溶于 $50~\mathrm{mL}$ 水中，混匀。  
3.2.9 铬天青S溶液 $(1\mathrm{g / L})$ ：称取 $0.1\mathrm{g}$ 铬天青S溶于 $100~\mathrm{mL}$ 乙醇溶液 $(1 + 1)$ 中，混匀。  
3.2.10 Triton X-100 溶液 $(3\%)$ ：吸取 $3\mathrm{mL}$ Triton X-100 置于 $100\mathrm{mL}$ 容量瓶中，加水定容至刻度，混匀。  
3.2.11 CPB溶液 $(3\mathrm{g} / \mathrm{L})$ ：称取 $0.3\mathrm{g}$ CPB溶于 $15~\mathrm{mL}$ 无水乙醇中，加水稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.12 乙二胺溶液 $(1 + 2)$ ：量取 $10\mathrm{mL}$ 乙二胺缓慢加入 $20~\mathrm{mL}$ 水中，混匀。  
3.2.13 乙二胺-盐酸缓冲溶液 $(\mathrm{pH}6.7\sim 7.0)$ ：量取 $100~\mathrm{mL}$ 乙二胺沿玻璃棒缓慢加入 $200~\mathrm{mL}$ 水中，待冷却后再沿玻璃棒缓缓加入 $190~\mathrm{mL}$ 盐酸，混匀，若 $\mathrm{pH} > 7.0$ 或 $\mathrm{pH} < 6.7$ 时可分别用盐酸溶液 $(1 + 1)$ 或乙二胺溶液 $(1 + 2)$ 调节 $\mathrm{pH}$ 。  
3.2.14 抗坏血酸溶液 $(10\mathrm{g / L})$ ：称取 $1\mathrm{g}$ 抗坏血酸，用水溶解并定容至 $100~\mathrm{mL}$ ，混匀。临用时现配。

# 3.3 标准品

铝标准溶液： $1000\mathrm{mg / L}$ 。或经国家认证并授予标准物质证书的一定浓度的铝标准溶液。

# 3.4 标准溶液配制

3.4.1 铝标准中间液 $(100\mathrm{mg / L})$ ：准确吸取 $1.00~\mathrm{mL}$ 铝标准溶液 $(1000\mathrm{mg / L})$ 于 $10~\mathrm{mL}$ 容量瓶中，加硝酸溶液 $(5\%)$ 定容至刻度，混匀。  
3.4.2 铝标准使用液 $(1.00\mathrm{mg / L})$ ：准确吸取 $1.00~\mathrm{mL}$ 铝标准中间液 $(100\mathrm{mg / L})$ ，置于 $100~\mathrm{mL}$ 容量瓶中，用硝酸溶液 $(5\%)$ 稀释至刻度，混匀。

# 4 仪器和设备

注：所有玻璃仪器均需以硝酸(1+5)浸泡 $24\mathrm{h}$ 以上，用自来水反复冲洗，最后用水冲洗晾干后方可使用。

4.1 分光光度计。  
4.2 天平：感量 $1\mathrm{mg}$   
4.3 可调式控温电热炉或电热板。

4.5 恒温干燥箱。

# 5 分析步骤

# 5.1 试样制备

在采样和试样制备过程中，应注意不使试样污染，应避免使用含铝器具。

面制品、豆制品、虾味片、烘焙食品等样品粉碎均匀后，取约 $30\mathrm{g}$ 置 $85^{\circ}C$ 恒温干燥箱中干燥 $4\mathrm{h}$ 。

# 5.2 试样消解

称取试样 $0.2\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.500~\mathrm{mL}\sim 5.00~\mathrm{mL}$ ，置于硬质玻璃消化管或锥形瓶中，加入 $10~\mathrm{mL}$ 硝酸， $0.5\mathrm{mL}$ 硫酸，在可调式控温电热炉或电热板上加热，推荐条件： $100^{\circ}C$ 加热 $1\textrm{h}$ ，升至 $150~^\circ C$ 加热 $1\textrm{h}$ ，再升至 $180^{\circ}C$ 加热 $2\textrm{h}$ ，然后升至 $200^{\circ}C$ ，若变棕黑色，再补加硝酸消化，

直至管口冒白烟，消化液呈无色透明或略带黄色。取出冷却，用水转移定容至 $50~\mathrm{mL}(V_1)$ 容量瓶中，混匀备用。同时做试剂空白试验。

# 5.3 显色反应及比色测定

分别吸取 $1.00\mathrm{mL}(V_2)$ 试样消化液、空白溶液分别置于 $25~\mathrm{mL}$ 具塞比色管中，加水至 $10~\mathrm{mL}$ 刻度。另取 $25~\mathrm{mL}$ 具塞比色管7支，分别加入铝标准使用溶液 $0\mathrm{mL},0.500\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},3.00\mathrm{mL},$ $4.00~\mathrm{mL}$ 和 $5.00~\mathrm{mL}$ （该系列标准溶液中铝的质量分别为 $0\mu \mathrm{g},0.500\mu \mathrm{g},1.00\mu \mathrm{g},2.00\mu \mathrm{g},3.00\mu \mathrm{g},$ $4.00\mu \mathrm{g},5.00\mu \mathrm{g})$ ，并依次向各管中加入硫酸溶液 $(1\%)$ 1 mL，加水至 $10~\mathrm{mL}$ 刻度。

向标准管、试样管、试剂空白管中滴加1滴对硝基苯酚乙醇溶液 $(1\mathrm{g / L})$ ，混匀，滴加氨水溶液 $(1 + 1)$ 至浅黄色，滴加硝酸溶液 $(2.5\%)$ 至黄色刚刚消失，再多加 $1\mathrm{mL}$ ，加入 $1\mathrm{mL}$ 抗坏血酸溶液 $(10\mathrm{g / L})$ ，混匀后加 $3\mathrm{mL}$ 铬天青S溶液 $(1\mathrm{g / L})$ ，混匀后加 $1\mathrm{mL}$ TritonX-100溶液 $(3\%)$ ， $3\mathrm{mL}$ CPB溶液 $(3\mathrm{g / L})$ $3\mathrm{mL}$ 乙二胺-盐酸缓冲溶液，加水定容至 $25.0~\mathrm{mL}$ ，混匀，放置 $40\mathrm{min}$ 。

于 $620\mathrm{nm}$ 波长处，用 $1\mathrm{cm}$ 比色皿以空白溶液为参比测定吸光度值。以标准系列溶液中铝的质量为横坐标，以相应的吸光度值为纵坐标，并绘制标准曲线。根据试样消化液的吸光度值与标准曲线比较定量。

# 6 分析结果的表述

试样中铝含量按式(1)计算：

$$
X = \frac {\left(m _ {1} - m _ {0}\right) \times V _ {1}}{m \times V _ {2}} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中铝的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$m_{1}$ ——测定用试样消化液中铝的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_0$ ——空白溶液中铝的质量，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——试样消化液总体积，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——测定用试样消化液体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样称样量或移取体积，单位为克或毫升（g 或 mL）。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 9 原理

试样消解处理后，经石墨炉原子化，在 $257.4\mathrm{nm}$ 处测定吸光度。在一定浓度范围内铝含量与吸光度值成正比，与标准系列比较定量。

# 12 分析步骤

# 12.1 试样制备

注：在采样和试样制备过程中，应避免污染，应避免使用含铝器具。

# 12.1.1 粮食、豆类样品

样品去除杂物后，粉碎，储于塑料瓶中。

# 12.1.2 蔬菜、水果、鱼类、肉类等水分含量高的样品

样品用水洗净，晾干，取可食部分，制成匀浆，储于塑料瓶中。

# 12.1.3 饮料、酒、醋、酱油等液体样品

将样品摇匀。

# 12.1.4 面制品、豆制品、虾味片、烘焙食品等样品

样品粉碎均匀后，取约 $30\mathrm{g}$ 置 $85^{\circ}C$ 恒温干燥箱中干燥 $4\mathrm{h}$

# 12.2 试样消解

# 12.2.1 湿法消解

称取固体试样 $0.2\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.500~\mathrm{mL}\sim 5.00~\mathrm{mL}$ ，置于硬质玻璃消化管中，加入 $10~\mathrm{mL}$ 硝酸， $0.5~\mathrm{mL}$ 硫酸，在可调式控温电热炉上加热，推荐条件： $100^{\circ}C$ 加热 $1\textrm{h}$ ，升至 $150^{\circ}C$ 加热 $1\textrm{h}$ ，再升至 $180^{\circ}C$ 加热 $2\textrm{h}$ ，然后升至 $200^{\circ}C$ ，若变棕黑色，再补加硝酸消化。直至管口冒白烟，消化液呈无色透明或略带黄色。取出冷却，用水转移定容至 $25~\mathrm{mL}$ 容量瓶，混匀备用，同时做试剂空白试验。亦可采用锥形瓶，于可调式电热板上，按上述操作方法进行湿式消解。

# 12.2.2 微波消解

称取固体试样 $0.2\mathrm{g}\sim 0.8\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.500~\mathrm{mL}\sim 3.00~\mathrm{mL}$ ，置于微波消解内罐中，加入硝酸 $5\mathrm{mL}\sim 8\mathrm{mL}$ ，盖上内罐盖，然后旋紧外盖置于微波消解仪中，根据不同种类的试样设置微波炉消解系统的消解条件(具体消解条件参考附录A)，消解完毕待消解罐冷却至室温后，打开消解罐，于电热板上赶酸至近干，待降至室温后用少许水洗涤消化罐3次～4次，洗液合并于 $25~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀备用，同时做试剂空白试验。

# 12.2.3 压力罐消解

称取固体试样 $0.2\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.500~\mathrm{mL}\sim 5.00~\mathrm{mL}$ ，置于压力消解内罐中，加入硝酸 $5\mathrm{mL}\sim 8\mathrm{mL}$ ，盖上内盖，旋紧外套，置于恒温干燥箱中，消解完毕待消解罐冷却至室温后，打开压力消解罐，取出内罐，在电热板上赶酸至近干，待降至室温后用少许水洗涤消化罐3次～4次，洗液合并于 $25~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀备用，同时做试剂空白试验（具体消解条件参考附

录B)。

# 12.3 测定

# 12.3.1 仪器参考条件

根据各自仪器性能调至最佳状态。参考条件为波长 $257.4 \mathrm{~nm}$ ，狭缝 $0.5 \mathrm{~nm}$ ，灯电流 $10 \mathrm{~mA} \sim 15 \mathrm{~mA}$ ，干燥温度 $85^{\circ} \mathrm{C} \sim 120^{\circ} \mathrm{C}, 30 \mathrm{~s}$ ；灰化温度 $1000^{\circ} \mathrm{C} \sim 1200^{\circ} \mathrm{C}$ ，持续 $15 \mathrm{~s} \sim 20 \mathrm{~s}$ ，原子化温度 $2750^{\circ} \mathrm{C}$ ，持续 $4 \mathrm{~s} \sim 5 \mathrm{~s}$ ；内气流量 $0.3 \mathrm{~L} / \mathrm{min}$ ，进样量 $10 \mu \mathrm{L}$ ，原子化时停气。

# 12.3.2 标准曲线制作

按质量浓度由低到高的顺序将 $10\mu \mathrm{L}$ 标准系列溶液(可根据使用仪器选择最佳进样量)注入石墨管，原子化后测其吸光度值。以质量浓度为横坐标，吸光度值为纵坐标，制作标准曲线。

# 12.3.3 试样溶液的测定

按仪器最佳进样量将适当体积的试样消化液、空白溶液分别注入石墨炉中，测定其吸光度值，由标准曲线得到试样消化液中中铝的质量浓度。

# 13 分析结果的表述

试样中铝的含量按式(2)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中铝的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$\rho$ ——试样溶液中铝的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白溶液中铝的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$m$ ——试样称样量或移取体积，单位为克或毫升(g或 $\mathrm{mL}$ )；

$V$ ——试样消化液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数。

计算结果保留三位有效数字。

当铝含量 $\geqslant 10\mathrm{mg / kg(mg / L)}$ 时，计算结果保留三位有效数字；当铝含量 $<  10~\mathrm{mg / kg(mg / L)}$ 时，计算结果保留两位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 15 其他

当称样量为 $0.5\mathrm{g}$ (或 $0.5\mathrm{mL}$ )，定容体积为 $25\mathrm{mL}$ 时，方法的检出限为 $0.3\mathrm{mg/kg}$ （或 $0.3\mathrm{mg/L}$ ），定量限为 $0.8\mathrm{mg/kg}$ （或 $0.8\mathrm{mg/L}$ ）。

# 附录A

# 检验方法

# A.1 警示

本标准试验方法中使用的部分试剂具有毒性或腐蚀性，使用时需小心谨慎并按照相关规定操作。若溅到皮肤上应立即用水冲洗，严重者应立即治疗。在使用挥发性酸时，要在通风橱中进行。

# A.2 一般规定

除非另有说明，在分析中仅使用确认为分析纯试剂和GB/T6682中规定的一级水。试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸 $(\mathrm{HCl})$ ： $36\% \sim 38\%$   
A.3.1.2 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
A.3.1.3 三水合磷酸氢二钾 $\mathrm{(K_2HPO_4\cdot 3H_2O)}$   
A.3.1.4 $5^{\prime}$ -单磷酸尿苷标准品 $\mathrm{(C_9H_{13}N_2O_9P}$ ，CAS号：58-97-9）：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
A.3.1.5 盐酸溶液：移取 $1\mathrm{mL}$ 盐酸，溶于水并定容至 $1000~\mathrm{mL}$   
A.3.1.6 磷酸二氢钾溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $13.6\mathrm{g}$ 磷酸二氢钾，溶于水并定容至 $1000~\mathrm{mL}$ 。  
A.3.1.7 磷酸氢二钾溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $2.28\mathrm{g}$ 三水合磷酸氢二钾，溶于水并定容至 $100~\mathrm{mL}$ 。  
A.3.1.8 流动相: $0.1 \mathrm{~mol} / \mathrm{L}$ 磷酸二氢钾溶液 (A.3.1.6), 用 $0.1 \mathrm{~mol} / \mathrm{L}$ 的磷酸氢二钾溶液 (A.3.1.7) 调 $\mathrm{pH}$ 至 5.6, 过滤膜, 备用。  
A.3.1.9 微孔滤膜： $0.45\mu \mathrm{m}$ ，水系。

# A.3.2 仪器和设备

A.3.2.1 电子天平：感量为 $0.001\mathrm{g}$   
A.3.2.2 紫外-可见光分光光度计。  
A.3.2.3 高效液相色谱仪：配有紫外检测器或二极管阵列检测器。  
A.3.2.4 pH计：精度为0.01。  
A.3.2.5 石英比色皿： $1\mathrm{cm}$

# A.3.3 紫外最大吸收峰鉴别

# A.3.3.1 分析步骤

# A.3.3.1.1 试样溶液的制备

称取试样 $0.02\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ），用盐酸溶液(A.3.1.5)溶解并定容至 $1000~\mathrm{mL}$ ，混匀，作为试样溶液。

# A.3.3.1.2 测定

将试样溶液(见A.3.3.1.1)注入 $1\mathrm{cm}$ 石英比色皿，以盐酸溶液(A.3.1.5)为空白，用紫外-可见光分光光度计A.3.2.2测定其吸光度，最大吸收值应在 $(262\pm 2)\mathrm{nm}$ 处。

# A.3.4 液相色谱保留时间鉴别

# A.3.4.1 分析步骤

# A.3.4.1.1 试样溶液的制备

称取试样 $0.02\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ), 用流动相(A.3.1.8)溶解并定容至 $10\mathrm{mL}$ 。移取该溶液 $1.00\mathrm{mL}$ , 用流动相稀释并定容至 $100\mathrm{mL}$ , 混匀, 过滤膜, 作为试样溶液。

# A.3.4.1.2 对照溶液的制备

称取 $5^{\prime}$ -单磷酸尿苷标准品 $0.02\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )，用流动相(A.3.1.8)溶解并定容至 $10~\mathrm{mL}$ 。移取该溶液 $1.00~\mathrm{mL}$ ，用流动相稀释并定容至 $100~\mathrm{mL}$ ，混匀，作为对照溶液。

# A.3.4.1.3 色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱（柱长 $250~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m})$ ，或等效色谱柱。  
b）检测波长： $254~\mathrm{nm}$   
c) 流动相：同A.3.1.8。  
d) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ 。  
e) 柱温: $30^{\circ} \mathrm{C}$ 。  
f）进样体积： $20~\mu \mathrm{L}$

# A.3.4.1.4 系统适应性试验

将对照溶液（见A.3.4.1.2)注入液相色谱仪中，记录色谱图。

$5^{\prime}$ -单磷酸尿苷主峰与它最近的杂质峰的分离度 $R$ 应不小于2.0。

连续进样6次， $5^{\prime}$ -单磷酸尿苷峰面积的相对标准偏差(RSD)应不大于 $2.0\%$ 。

# A.3.4.1.5 测定

将试样溶液(见A.3.4.1.1)及对照溶液(见A.3.4.1.2)分别注入液相色谱仪中，记录色谱图。两者的主峰保留时间应一致，相对偏差 $\pm 2.5\%$ 。

# A.4 $5^{\prime}$ -单磷酸尿苷含量（以干基计）

# A.4.1 紫外分光光度法

# A.4.1.1 方法原理

采用紫外-可见光分光光度计，测定 $260~\mathrm{nm}$ 处试样溶液的吸光度值，以摩尔吸光系数计算试样中 $5^{\prime}$ -单磷酸尿苷的含量。

# A.4.1.2 试剂和材料

A.4.1.2.1 盐酸 $(\mathrm{HCl}):36\% \sim 38\%$

A.4.1.2.2 盐酸溶液：移取 $1\mathrm{mL}$ 盐酸，溶于水并定容至 $1000~\mathrm{mL}$

# A.4.1.3 仪器和设备

A.4.1.3.1 电子天平：感量为 $0.001\mathrm{g}$   
A.4.1.3.2 紫外-可见光分光光度计。  
A.4.1.3.3 石英比色皿： $1\mathrm{cm}$

# A.4.1.4 分析步骤

称取 $0.5\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 样品，用盐酸溶液(A.4.1.2.2)溶解并定容至 $1000\mathrm{mL}$ 。移取该溶液 $10.0\mathrm{mL}$ ，用盐酸溶液稀释并定容至 $250\mathrm{mL}$ ，该溶液即为试样溶液。将试样溶液注入 $1\mathrm{cm}$ 的石英比色皿中，以盐酸溶液做空白，用紫外-可见光分光光度计测定其在 $260\mathrm{nm}$ 处的吸光度。

# A.4.1.5 结果计算

试样中 $5^{\prime}$ -单磷酸尿苷的质量分数(以干基计) $\omega_{1}$ 的数值以百分比 $(\%)$ 表示，按式(A.1)计算。

$$
w _ {1} = \frac {A \times M \times V \times f}{\varepsilon \times b \times m} \times \frac {1 0 0}{1 0 0 - w} \times 1 0 0 \dots \tag {A.1}
$$

式中：

$A$ ——试样溶液在波长 $260 \mathrm{~nm}$ 下的吸光度；

$M$ —— $5^{\prime}$ -单磷酸尿苷的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M = 324.18]$

$V$ ——试样定容体积，单位为升 $(\mathrm{L})$ ；

$f$ ——试样溶液的稀释倍数 $(f = 25)$ ；

$\varepsilon$ —— $5^{\prime}$ -单磷酸尿苷在 $260 \mathrm{~nm}$ 处的摩尔吸收系数，单位为升每摩尔厘米 $\left[\mathrm{L} / (\mathrm{mol} \cdot \mathrm{cm})\right]$ （ $\varepsilon = 9900$ ）；

$b$ ——比色皿厚度，单位为厘米(cm)；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$w$ ——试样的水分含量，以 $\%$ 表示。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，保留小数点后一位。两次独立测定结果的绝对差值不超过其算术平均值的 $2\%$ 。

# A.6.5 结果计算

试样中有关物质的含量 $w_{3}$ 的数值以百分比 $(\%)$ 表示，按式(A.5)计算。

$$
w _ {3} = \frac {A _ {3} \times 2}{A _ {4} \times 1 0 0} \times 1 0 0 \tag {A.5}
$$

式中：

$A_{3}$ ——试样溶液除主峰以外的所有杂质峰的峰面积之和；

$A_{4}$ ——对照溶液中主峰的峰面积。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，保留两位有效数字。两次独立测定结果的绝对差值不超过其算术平均值的 $10\%$ 。

# A.7 乙醇

# A.7.1 方法原理

试样置于密闭的顶空瓶中，在特定温度下平衡一定时间，使试样中的乙醇挥发出来并在顶空瓶内达到动态平衡，通过气相色谱仪进行分离检测，外标法定量。

# A.7.2 试剂和材料

A.7.2.1 乙醇 $(\mathrm{C}_2\mathrm{H}_6\mathrm{O})$ ：色谱纯，纯度 $\geqslant 99.5\%$   
A.7.2.2 氢氧化钠(NaOH)。  
A.7.2.3 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $40\mathrm{g}$ 氢氧化钠(A.7.2.2)，加水稀释并定容至 $1000~\mathrm{mL}$ 。  
A.7.2.4 顶空进样瓶：体积 $20~\mathrm{mL}$

# A.7.3 仪器和设备

A.7.3.1 电子天平：感量为 $0.001\mathrm{g},0.0001\mathrm{g}$   
A.7.3.2 气相色谱仪，配顶空进样器和氢火焰离子化检测器。

# A.7.4 分析步骤

# A.7.4.1 试样溶液的制备

称取试样 $10\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ), 用氢氧化钠溶液(A.7.2.3)溶解并定容至 $100\mathrm{mL}$ 。移取 $10\mathrm{mL}$ 该溶液于顶空进样瓶, 旋紧盖子, 备用。

# A.7.4.2 对照溶液的制备

称取 $0.01\mathrm{g}$ 乙醇(精确至 $0.0001\mathrm{g}$ )，用氢氧化钠溶液(A.7.2.3)溶解并定容至 $100\mathrm{mL}$ 。移取 $10\mathrm{mL}$ 该溶液于顶空进样瓶，旋紧盖子，备用。

# A.7.4.3 色谱参考条件

a）色谱柱：毛细管柱 $(30\mathrm{m}\times 0.53\mathrm{mm}$ ，固定液为 $6 \%$ 氰丙基苯基- $94 \%$ 二甲基聚硅氧烷，薄膜厚度

为 $3.00\mu \mathrm{m}$ 或等效柱。

b）柱温：初始温度 $40^{\circ}\mathrm{C}$ ，保持 $20\mathrm{min}$ ，以 $10^{\circ}\mathrm{C / min}$ 速率升温至 $240^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ 。  
c）检测器温度： $250~\mathrm{^\circ C}$ 。  
d）进样口温度： $140~^\circ \mathrm{C}$ 。  
e) 载气: $\mathrm{N}_2$ ; 柱流量: $2.5 \mathrm{~mL} / \mathrm{min}$ 。  
f）顶空进样器条件：平衡温度 $80~\mathrm{^\circ C}$ ，平衡时间 $60\mathrm{min}$ ；定量环温度 $85~\mathrm{^\circ C}$ ；传输线温度 $90~\mathrm{^\circ C}$ 。  
g）进样时间： $1\mathrm{min}$   
h）进样量： $1.0~\mathrm{mL}$

# A.7.4.4 系统适应性试验

取对照溶液（见A.7.4.2)注入气相色谱仪中，观察色谱图。

# A.7.4.5 测定

# A.7.5 结果计算

试样中乙醇的含量 $\omega_{4}$ 的数值以毫克每千克 $(\mathrm{mg / kg})$ 表示，按式(A.6)计算。

$$
w _ {4} = \frac {A _ {5} \times C _ {3} \times V _ {1} \times 1 0 0 0}{A _ {6} \times m _ {1} \times 1 0 0 0} \quad \dots \tag {A.6}
$$

式中：

$A_{5}$ ——试样溶液乙醇的峰面积；

$C_3$ ——对照溶液中乙醇的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V_{1}$ ——试样溶液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m_{1}$ ——试样质量，单位为克(g)；

1000——换算系数；

$A_{6}$ ——对照溶液乙醇的峰面积。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，保留两位有效数字。两次独立测定结果的绝对差值不超过其算术平均值的 $10\%$ 。

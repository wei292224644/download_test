2017-06-23 实施

# 前言

本标准代替GB/T5009.189—2003《银耳中米酵菌酸的测定》。

本标准与GB/T5009.189—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中米酵菌酸的测定”；  
——修改了适用范围；  
——修改了试样制备，增加了固相萃取；  
——增加了高效液相色谱条件；  
——增加了附录A；  
——规定了方法检出限和定量限；  
——删除了薄层色谱法。

# 食品安全国家标准

# 食品中米酵菌酸的测定

# 1 范围

本标准规定了银耳及其制品、酵米面及其制品等食品中米酵菌酸的测定方法。

本标准适用于银耳及其制品、酵米面及其制品等食品中米酵菌酸的测定。

# 2 原理

试样经提取、净化、浓缩及过滤后，经高效液相色谱仪分析，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.3 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。  
3.1.4 甲酸 $\mathrm{(CH_2O_2)}$ 。  
3.1.5 盐酸（HCl）。  
3.1.6 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。  
3.1.7 碳酸氢钠 $(\mathrm{NaHCO}_3)$ 。  
3.1.8 石油醚 $\left(\mathrm{C}_{5} \mathrm{H}_{12} \mathrm{O}_{2}\right)$ ：沸程 $30^{\circ} \mathrm{C} \sim 60^{\circ} \mathrm{C}$ 。  
3.1.9 无水乙醚 $(\mathrm{C_4H_{10}O})$ 。  
3.1.10 三氯甲烷 $\mathrm{(CHCl_3)}$

# 3.2 试剂配制

3.2.1 磷酸溶液 $(45.4\%)$ ：量取 $45.4~\mathrm{mL}$ 磷酸于 $100~\mathrm{mL}$ 容量瓶中，用水稀释定容至刻度。  
3.2.2 碳酸氢钠溶液 $(40\mathrm{g / L})$ ：称取 $40\mathrm{g}$ 碳酸氢钠加水溶解，转移至 $1000~\mathrm{mL}$ 容量瓶中定容至刻度。  
3.2.3 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ：量取 $50~\mathrm{mL}$ 盐酸于 $100~\mathrm{mL}$ 容量瓶中，用水稀释定容至刻度。  
3.2.4 甲醇-氨水溶液：量取 $80~\mathrm{mL}$ 甲醇，加入 $1.0~\mathrm{mL}$ 氨水，加水定容到 $100~\mathrm{mL}$ ，混匀。  
3.2.5 甲酸-甲醇溶液 $(2\%)$ ：吸取 $2.0~\mathrm{mL}$ 甲酸于 $100~\mathrm{mL}$ 容量瓶中，用甲醇稀释定容至刻度。

# 3.3 材料

固相萃取柱：阴离子交换柱 $(60\mathrm{mg} / 3\mathrm{mL})$ 或等效品，临用前依次加 $5.0~\mathrm{mL}$ 甲醇和 $5.0~\mathrm{mL}$ 水活化，保持柱体湿润。

# 3.4 标准品

米酵菌酸 $\mathrm{(C_{28}H_{38}O_7}$ ,CAS号：11076-19-0)：纯度 $\geqslant 95\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.5 标准溶液配制

3.5.1 米酵菌酸标准储备液 $(0.1\mathrm{mg / mL})$ ：准确称取米酵菌酸标准品 $1\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ ），用甲醇溶解，转移至 $10~\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度。置于 $2^{\circ}C\sim 8^{\circ}C$ 冰箱中避光保存，有效期为6个月。  
3.5.2 米酵菌酸标准系列工作液：分别吸取米酵菌酸标准储备液用甲醇稀释定容，配制成米酵菌酸浓度分别为 $0.3 \mu \mathrm{g} / \mathrm{mL}, 0.5 \mu \mathrm{g} / \mathrm{mL}, 1.0 \mu \mathrm{g} / \mathrm{mL}, 2.0 \mu \mathrm{g} / \mathrm{mL}, 5.0 \mu \mathrm{g} / \mathrm{mL}, 10.0 \mu \mathrm{g} / \mathrm{mL}, 20.0 \mu \mathrm{g} / \mathrm{mL}, 40.0 \mu \mathrm{g} / \mathrm{mL}$ 的标准工作溶液。临用时配制。

# 4 仪器和设备

4.1 高效液相色谱仪，带二极管阵列检测器。  
4.2 天平：感量分别为 $0.01\mathrm{g}$ 和 $0.01\mathrm{mg}$ 。  
4.3 固相萃取装置。  
4.4 超声波振荡器。  
4.5 旋转蒸发仪。  
4.6 氮吹仪。  
4.7 微孔有机滤膜（孔径 $0.45\mu \mathrm{m}$ ）。  
4.8 涡旋振荡器。  
4.9 恒温水浴锅。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固相萃取法

# 5.1.1.1 提取

干试样经粉碎过 $\phi 0.425\mathrm{mm}$ 筛后，称取 $20\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )置于锥形瓶中，加入 $100~\mathrm{mL}$ 甲醇-氨水溶液；鲜(湿)试样经剪碎、匀浆后称取 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )，加入 $80~\mathrm{mL}$ 甲醇-氨水溶液。混匀，室温下避光浸泡 $1\textrm{h}$ ，置于超声波振荡器中超声提取约 $30\mathrm{min}$ ，过滤。干试样取滤液 $50~\mathrm{mL}$ ，鲜(湿)试样取滤液 $40~\mathrm{mL}$ 。置 $80^{\circ}C$ 水浴中浓缩至约 $3\mathrm{mL}$ ，待净化。

# 5.1.1.2 净化

将浓缩后的试样全部转移到已活化的固相萃取柱中，依次用 $5\mathrm{mL}$ 水和 $5\mathrm{mL}$ 甲醇淋洗，弃去流出液，抽干萃取柱，再用 $6\mathrm{mL}$ 甲酸-甲醇溶液 $(2\%)$ 洗脱，收集洗脱液，于 $40^{\circ}\mathrm{C} \pm 0.5^{\circ}\mathrm{C}$ 水浴中氮吹至干，然后加入 $0.5\mathrm{mL}$ 甲醇，涡旋溶解，混匀，经微孔有机滤膜过滤后进行HPLC分析。

# 5.1.2 液-液萃取法

# 5.1.2.1 提取

干试样经粉碎过 $\phi 0.425\mathrm{mm}$ 筛后，称取 $20\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，置于具塞锥形瓶中，加入 $20~\mathrm{mL}$ 甲

醇，室温下避光浸泡 $1\mathrm{h}$ ，再加入 $80~\mathrm{mL}$ 三氯甲烷和 $0.2\mathrm{mL}$ 磷酸溶液 $(45.4\%)$ ；鲜（湿）试样经剪碎、匀浆后称取 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），加入 $16\mathrm{mL}$ 甲醇，于室温下避光浸泡 $1\mathrm{h}$ ，再加入 $64\mathrm{mL}$ 三氯甲烷和 $0.16\mathrm{mL}$ 磷酸溶液 $(45.4\%)$ 。振荡 $30\mathrm{min}$ ，过滤，干试样取滤液 $50~\mathrm{mL}$ ，鲜（湿）试样取滤液 $40~\mathrm{mL}$ 。

# 5.1.2.2 萃取

将上述滤液分别移入 $150~\mathrm{mL}$ 分液漏斗中，加入与滤液等体积的碳酸氢钠溶液 $(40~\mathrm{g / L})$ ，振摇 $2\mathrm{min}$ ，静置待分层后，取出下层于另一分液漏斗中；再用碳酸氢钠溶液 $(40~\mathrm{g / L})$ $10~\mathrm{mL}$ 重复提取两次，轻摇；将三次提取的碳酸氢钠溶液合并，加入 $25~\mathrm{mL}$ 三氯甲烷，振摇 $2\mathrm{min}$ ，静置分层后弃去三氯甲烷，于分液漏斗中慢慢加入盐酸溶液(6mol/L)调该溶液 $\mathsf{pH}$ 至 $2\sim 3$ ，加入石油醚 $50~\mathrm{mL}$ （鲜、湿试样为 $40~\mathrm{mL}$ )，振摇 $3\mathrm{min}$ ，静置分层，取出石油醚层于旋蒸瓶中，再分别用石油醚 $30~\mathrm{mL}$ 和 $20~\mathrm{mL}$ 各提取一次（鲜、湿试样两次均为 $20~\mathrm{mL}$ )，将石油醚层并入同一瓶中，于 $40^{\circ}C\pm 0.5^{\circ}C$ 水浴中旋蒸浓缩至干，用少量甲醇分次将旋蒸瓶中提取物溶解并转移至 $5.0~\mathrm{mL}$ 玻璃离心管或浓缩瓶中，于 $40^{\circ}C\pm 0.5^{\circ}C$ 下氮吹浓缩至干，然后加入 $0.5~\mathrm{mL}$ 甲醇，涡旋溶解，混匀，经微孔有机滤膜过滤后进行HPLC分析。

# 5.2 色谱参考条件

色谱参考条件列出如下：

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱(柱长 $250~\mathrm{mm}$ ，内径 $4.6\mathrm{mm}$ ，粒度 $5\mu \mathrm{m})$ 或同等性能的色谱柱；  
b）流动相：甲醇 $+$ 水 $= 75 + 25$ ，水用冰乙酸调 $\mathrm{pH}$ 至2.5；  
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d) 柱温: $30^{\circ} \mathrm{C}$ ;   
e）检测波长： $267~\mathrm{nm}$   
f）进样量： $20~\mu \mathrm{L}$

# 5.3 标准曲线的制作

分别将 $20~\mu \mathrm{L}$ 米酵菌酸标准系列工作液注入液相色谱仪中，测定相应的峰面积，以标准工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。米酵菌酸标准溶液的色谱图见图A.1。

# 5.4 试样溶液的测定

将试样溶液注入液相色谱仪中，以保留时间定性，同时记录峰面积，根据标准曲线得到待测液中米酵菌酸的浓度。

# 6 分析结果的表述

试样中米酵菌酸含量按式(1)计算：

$$
X = \frac {\rho \times V \times 2}{m} \tag {1}
$$

式中：

$X$ ——试样中米酵菌酸的含量，单位为微克每克 $(\mu \mathrm{g} / \mathrm{g})$ ；

$\rho$ ——由标准曲线得到的试样溶液中米醇菌酸的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样溶液的浓缩定容体积，单位为毫升 $(\mathrm{mL})$ ；

2——稀释倍数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

方法检出限为 $0.005\mu \mathrm{g / g}$ ，定量限为 $0.015\mu \mathrm{g / g}$

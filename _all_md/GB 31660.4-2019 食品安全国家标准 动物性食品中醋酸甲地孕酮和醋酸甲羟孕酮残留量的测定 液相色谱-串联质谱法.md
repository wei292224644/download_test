# 动物性食品中醋酸甲地孕酮和醋酸甲羟孕酮残留量的测定

# 液相色谱-串联质谱法

# 1 范围

本标准规定了动物性食品中醋酸甲地孕酮和醋酸甲羟孕酮残留检测的制样和液相色谱-串联质谱测定方法。

本标准适用于猪、牛、羊肌肉、脂肪、肝脏、肾脏和牛奶中醋酸甲地孕酮和醋酸甲羟孕酮残留量的检测。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样中残留的醋酸甲地孕酮和醋酸甲羟孕酮经乙腈提取，正己烷除脂，混合阳离子柱净化，甲醇洗脱，液相色谱-串联质谱法测定，内标法定量。

# 4 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 乙腈（ $\mathrm{CH}_3\mathrm{CN}$ ）：色谱纯。

4.1.2 甲醇（ $\mathrm{CH}_3\mathrm{OH}$ ）。

4.1.3 甲酸（HCOOH）：色谱纯。

4.1.4 乙酸（ $\mathrm{CH}_3\mathrm{COOH}$ ）。

4.1.5 正己烷（ $\mathrm{C_6H_{14}}$ ）。

4.1.6 乙酸乙酯（ $\mathrm{CH}_3\mathrm{COOC}_2\mathrm{H}_5$ ）。

4.1.7 乙酸铵（ $\mathrm{CH}_3\mathrm{COONH}_4$ ）。

# 4.2 溶液配制

4.2.1 $0.2\mathrm{mol} / \mathrm{L}$ 乙酸铵缓冲液：取乙酸铵 $15.4\mathrm{g}$ ，加水 $900\mathrm{ml}$ 使溶解，用乙酸调 $\mathrm{pH}$ 值至5.2，加水稀释至 $1000\mathrm{mL}$ 。  
4.2.2 $2\%$ 甲酸水溶液：取甲酸 $2\mathrm{mL}$ ，加水溶解并稀释至 $100~\mathrm{mL}$ 。  
4.2.3 $0.1\%$ 甲酸水溶液：取甲酸 $1\mathrm{mL}$ ，加水溶解并稀释至 $1000~\mathrm{mL}$ 。  
4.2.4 $50\%$ 甲醇水溶液：取甲醇 $50\mathrm{mL}$ ，加水溶解并稀释至 $100~\mathrm{mL}$ 。  
4.2.5 $30\%$ 甲醇水溶液：取甲醇 $30\mathrm{mL}$ ，加水溶解并稀释至 $100~\mathrm{mL}$ 。  
4.2.6 $0.1\%$ 甲酸水乙腈溶液：取 $0.1\%$ 甲酸水溶液 $20\mathrm{mL}$ ，加乙腈溶解并稀释至 $100\mathrm{mL}$ ，混匀。

# 4.3 标准品

4.3.1 醋酸甲地孕酮（Megestrol acetate, $\mathrm{C}_{24}\mathrm{H}_{32}\mathrm{O}_4$ ,CAS:595-33-5）、醋酸甲羟孕酮（Medroxyprogesterone acetate, $\mathrm{C}_{24}\mathrm{H}_{34}\mathrm{O}_4$ ,CAS:71-58-9），含量均 $\geqslant 98.0\%$ 。  
4.3.2 内标：氘代醋酸甲地孕酮（megestrolacetate- $\mathrm{D}_3,\mathrm{C}_{24}\mathrm{H}_{29}\mathrm{D}_3\mathrm{O}_4$ ），含量 $\geqslant 98.0\%$

# 4.4 标准溶液制备

4.4.1 标准贮备液：取醋酸甲地孕酮、醋酸甲羟孕酮标准品各约 $10\mathrm{mg}$ ，精密称定，于 $100\mathrm{mL}$ 棕色量瓶中，用乙腈溶解并稀释至刻度，配制成浓度为 $0.1\mathrm{mg/mL}$ 的醋酸甲地孕酮、醋酸甲羟孕酮标准贮备液。-20℃保存，有效期6个月。  
4.4.2 内标贮备液：取氘代醋酸甲地孕酮标准品约 $10\mathrm{mg}$ ，精密称定，于 $100\mathrm{mL}$ 棕色量瓶中，用乙腈溶解并稀释至刻度，配制成 $0.1\mathrm{mg/mL}$ 的氘代醋酸甲地孕酮贮备液。-20℃保存，有效期为6个月。  
4.4.3 混合标准工作液：精密量取醋酸甲地孕酮、醋酸甲羟孕酮标准贮备液各 $0.5 \mathrm{~mL}$ ，于 $50 \mathrm{~mL}$ 棕色量瓶中，用乙腈稀释至刻度，配制成浓度均为 $1 \mu \mathrm{g} / \mathrm{mL}$ 混合标准工作液。 $4^{\circ} \mathrm{C}$ 以下避光保存，有效期1个月。  
4.4.4 内标工作液：精密量取内标贮备液 $0.5\mathrm{mL}$ ，于 $50\mathrm{mL}$ 棕色量瓶中，用乙腈稀释至刻度，配制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 的内标标准工作液。 $4^{\circ}\mathrm{C}$ 以下避光保存，有效期1个月。

# 4.4.5 标准曲线的制备

精密量取适量混合标准工作液及内标标准工作液，用流动相稀释配制成浓度为2、5、25、50、100 ng/mL的系列标准溶液（内标均为20 ng/mL）。以特征离子质量色谱峰面积比为纵坐标，标准溶液浓度为横坐标，绘制标准曲线。

# 4.5 材料

4.5.1 混合阳离子固相萃取柱： $60\mathrm{mg} / 3\mathrm{mL}$ ，或相当者。  
4.5.2 $\beta$ -盐酸葡萄糖醛苷酶/芳基硫酸酯酶。  
4.5.3 微孔滤膜： $0.22\mu \mathrm{m}$

# 5 仪器和设备

5.1 液相色谱-串联质谱仪：配电喷雾离子源。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
5.3 氮吹仪。  
5.4 旋涡混合器： $3000\mathrm{r / min}$   
5.5 超声波萃取仪。  
5.6 移液枪： $200~\mu \mathrm{L}$ ， $1\mathrm{mL}$ ， $5\mathrm{mL}$ 。  
5.7 离心机： $10000\mathrm{r / min}$   
5.8 梨形瓶： $100\mathrm{mL}$   
5.9 旋转蒸发器。  
5.10 固相萃取装置。

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。脂肪组织于 $60^{\circ}\mathrm{C}$ 水浴中融化。

——取均质后的供试样品，作为供试试料。

——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

- $18^{\circ} \mathrm{C}$ 以下保存，3个月内进行分析检测。

# 7 测定步骤

# 7.1 酶解

取试样 $2 \mathrm{~g}$ (准确到 $\pm 20 \mathrm{mg}$ ), 于 $50 \mathrm{~mL}$ 离心管中, 加内标工作液 $40 \mu \mathrm{L}$ , 加 $0.2 \mathrm{~mol} / \mathrm{L}$ 乙酸铵缓冲液 $4 \mathrm{~mL}$ , 涡旋混匀后加入 $\beta$ -盐酸葡萄糖醛苷酶/芳基硫酸酯酶 $40 \mu \mathrm{L}$ , 于 $37^{\circ} \mathrm{C}$ 下避光水浴低速振荡, 酶解 $12 \mathrm{~h}$ 。

# 7.2 提取

# 7.2.1 肌肉、肝脏、肾脏组织

试样经酶解后，加乙酸乙酯 $10\mathrm{mL}$ ，于旋涡振荡器上剧烈振荡 $10\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液至梨形瓶中。残渣加乙酸乙酯 $10\mathrm{mL}$ 重复提取1次，合并上清液， $50^{\circ}\mathrm{C}$ 旋转蒸发至干。加乙腈 $10\mathrm{mL}$ 、正己烷 $5\mathrm{ml}$ 使溶解，转至 $50\mathrm{ml}$ 离心管中，低速涡旋 $10\mathrm{s}$ ， $3000\mathrm{r / min}$

离心 $2 \mathrm{~min}$ , 弃正己烷层, 下层液于 $50^{\circ} \mathrm{C}$ 旋转蒸发至干, 加 $30 \%$ 甲醇水溶液 $3 \mathrm{~mL}$ , 溶解, 备用。

# 7.2.2 脂肪组织

试样经酶解后，加乙腈 $10\mathrm{mL}$ ，于旋涡振荡器上剧烈振荡 $0.5\mathrm{min}$ ， $50^{\circ}\mathrm{C}$ 超声提取 $10\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液移至另一 $50\mathrm{mL}$ 离心管中。残渣加乙腈 $10\mathrm{mL}$ 重复提取1次。合并上清液，加正己烷 $4\mathrm{mL}$ ，低速涡旋 $10\mathrm{s}$ ， $3000\mathrm{r / min}$ 离心 $2\mathrm{min}$ ，弃正己烷层。加正己烷 $4\mathrm{mL}$ ，再次除脂。下层 $50^{\circ}\mathrm{C}$ 旋转蒸发至干，加入 $30\%$ 甲醇水溶液 $3\mathrm{mL}$ 溶解，备用。

# 7.3 净化

混合阳离子固相萃取柱用甲醇、水各 $3\mathrm{mL}$ 活化，取备用液过柱，依次用水、 $50\%$ 甲醇溶液各 $3\mathrm{mL}$ 淋洗，抽干。用甲醇 $5\mathrm{mL}$ 洗脱，洗脱液于 $50^{\circ}\mathrm{C}$ 下氮气吹干。用 $0.2\mathrm{mL}$ 乙腈（ $80\%$ ） $-0.1\%$ 甲酸溶液溶解残余物，涡旋混匀，过 $0.22\mu \mathrm{m}$ 滤膜或 $15000\mathrm{rpm}$ 高速离心 $10\mathrm{min}$ ，取上清液，供高效液相色谱-串联质谱仪测定。

# 7.4.3 测定法

# 8 结果计算和表述

试样中待测药物的残留量按式（1）计算：

$$
X = \frac {A \times A _ {i S} ^ {\prime} \times C _ {S} \times C _ {i S} \times V}{A _ {i S} \times A _ {S} \times C _ {i S} ^ {\prime} \times m}. \tag {1}
$$

式中：

$X$ —试样中被测物质的残留量, 单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ;

$C_{s}$ ——标准工作溶液中被测物质的浓度, 单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ;

$C_{is}$ —试样溶液中内标的浓度, 单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ;

$C_{is}$ ——标准工作溶液中内标的浓度，单位为纳克每毫升（ $\mathrm{ng/mL}$ ）；

$A$ ——试样溶液中被测物质的峰面积；

$A_{is}$ ——标准工作溶液中内标的峰面积；

$A_{is}$ —试样溶液中内标的峰面积；

$A_{s}$ ——标准工作溶液中被测物质的峰面积；

$V$ —试样溶液定容体积，单位为毫升（ $\mathrm{mL}$ ）；

$m$ —试料质量，单位为克（g）。

计算结果需扣除空白值。测定结果用两次平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法的灵敏度、准确度和精密度

# 9.1 灵敏度

本方法的检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1\mu \mathrm{g / kg}$

# 9.2准确度

本方法在 $1\mu \mathrm{g / kg}\sim 5\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

# 检验方法

# A.1 警示

本试验方法中使用的部分试剂具有毒性、腐蚀性及易燃性，操作者须小心谨慎！如溅到皮肤上应立即用水冲洗，严重者应立即治疗。使用易燃品时，严禁使用明火加热。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 氨水。  
A.3.1.2 氢氧化钠溶液： $40~\mathrm{g / L}$   
A.3.1.3 硝酸银溶液： $10\mathrm{g / L}$   
A.3.1.4 酚酞指示液： $10\mathrm{g / L}$   
A.3.1.5 硝酸溶液： $6 + 94$

# A.3.2 鉴别方法

称取约 $1\mathrm{g}$ 试样，置于 $100~\mathrm{mL}$ 烧杯中，加 $10~\mathrm{mL}$ 水，1滴酚酞指示液，用氢氧化钠溶液调至中性，滴加硝酸银溶液，有黄色沉淀生成，该沉淀能溶于硝酸溶液或氨水。

# A.4 总有机碳(以C计)的测定

# A.4.1 湿化学氧化法

# A.4.1.1 方法提要

试样在酸性条件下，通入氮气吹扫去除无机碳，经过硫酸钠氧化，在 $100^{\circ}\mathrm{C}$ 下迅速生成为二氧化碳，通过红外线分析仪检测出有机碳的含量。

# A.4.1.2 试剂和材料

A.4.1.2.1 过硫酸钠溶液: $100 \mathrm{~g} / \mathrm{L}$ 。称取 $100 \mathrm{~g}$ 过硫酸钠, 溶解于适量无二氧化碳的水中, 用无二氧化碳的水稀释到 $1000 \mathrm{~mL}$ 。  
A.4.1.2.2 磷酸溶液： $5\%$ 。量取 $59~\mathrm{mL}$ 磷酸，用无二氧化碳的水稀释到 $1000~\mathrm{mL}$   
A.4.1.2.3 总碳标准储备溶液： $1\mathrm{mL}$ 溶液含碳(C) $1.0\mathrm{mg}$ 。称取 $2.1254\mathrm{g}$ 在 $120^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 干燥 $2\mathrm{h}$ 的基准邻苯二甲酸氢钾，加无二氧化碳的水溶解，移入 $1000\mathrm{mL}$ 容量瓶中，用无二氧化碳的水稀释至刻

度，摇匀。

A.4.1.2.4 总碳标准使用溶液: $1 \mathrm{~mL}$ 溶液含碳 (C) $0.1 \mathrm{mg}$ 。用移液管移取 $10 \mathrm{~mL}$ 总碳标准储备溶液（见A.4.1.2.3），置于 $100 \mathrm{~mL}$ 容量瓶中，用无二氧化碳的水稀释至刻度，摇匀。

A.4.1.2.5 无二氧化碳的水：按GB/T603规定的方法制备。

# A.4.1.3 仪器和设备

总有机碳(TOC)分析仪：使用的高纯氮符合GB/T8979的要求。

# A.4.1.4 分析步骤

# A.4.1.4.1 标准曲线的绘制

分别移取 $0.00\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},3.00\mathrm{mL},4.00\mathrm{mL}$ 的总碳标准使用溶液，置于 $100~\mathrm{mL}$ 容量瓶中，用无二氧化碳的水稀释到刻度，摇匀。将总有机碳(TOC)分析仪调至最佳试验条件，按照浓度从低到高的顺序依次进行测定，每测一个标准溶液仪器自动吸入 $5.00~\mathrm{mL}$ 对应标准溶液、 $0.50~\mathrm{mL}$ 磷酸溶液、 $1.00~\mathrm{mL}$ 过硫酸钠溶液。反应完全后，仪器自动测量标准曲线溶液的峰面积，从每个标准溶液的峰面积中减去空白溶液的峰面积，以碳的质量浓度 $(\mathrm{mg / L})$ 为横坐标，对应的峰面积为纵坐标，绘制标准曲线。

# A.4.1.4.2 测定

称取约 $3\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ 。置于 $100~\mathrm{mL}$ 容量瓶中，用无二氧化碳的水稀释至刻度，摇匀。用总有机碳(TOC)分析仪进行测定，仪器自动依次吸入 $5.00~\mathrm{mL}$ 试样溶液、 $0.50~\mathrm{mL}$ 磷酸溶液、 $1.00~\mathrm{mL}$ 过硫酸钠溶液。反应完全后，仪器自动测量试样溶液的峰面积，从标准曲线上查出试样溶液中碳的质量浓度 $(\mathrm{mg / L})$ 。

# A.4.1.5 结果计算

总有机碳（以C计)的质量分数 $\omega_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\rho \times 0.1}{m _ {1} \times 1000} \times 100 \% \tag{A.1}
$$

式中：

$\rho$ ——从标准曲线上查出的试样溶液中碳的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

0.1 ——试样溶液的体积，单位为升 $(\mathrm{L})$ ；

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.0005\%$ 。

# A.4.2 催化氧化燃烧法

# A.4.2.1 方法提要

注射样品到燃烧炉高温区，样品中总有机碳在催化剂和氧气氛围中被分解为二氧化碳。产生的二氧化碳通过干燥单元干燥后被输送到非色散红外检测器。使用标准曲线法测定样品中的总碳含量和无机碳含量。总有机碳(TOC)等于总碳(TC)减去无机碳(IC)。

# A.4.2.2 试剂和材料

A.4.2.2.1 总碳标准储备溶液： $1\mathrm{mL}$ 溶液含碳(C) $1.0\mathrm{mg}$ 。称取 $2.1254\mathrm{g}$ 在 $120^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 干燥 $2\mathrm{h}$ 的

基准邻苯二甲酸氢钾，加无二氧化碳的水溶解，移入 $1000\mathrm{mL}$ 容量瓶中，用无二氧化碳的水稀释至刻度，摇匀。

A.4.2.2.2 总碳标准使用溶液： $1\mathrm{mL}$ 溶液含碳(C) $0.1\mathrm{mg}$ 。用移液管移取 $10\mathrm{mL}$ 总碳标准储备溶液（见A.4.2.2.1)，置于 $100\mathrm{mL}$ 容量瓶中，用无二氧化碳的水稀释至刻度，摇匀。  
A.4.2.2.3 无机碳标准储备溶液： $1\mathrm{mL}$ 溶液含碳(C) $1.0\mathrm{mg}$ 。称取 $3.50\mathrm{g}$ 预先在硅胶干燥器中干燥 $2\mathrm{h}$ 的碳酸氢钠和 $4.41\mathrm{g}$ 在 $280^{\circ}\mathrm{C} \sim 290^{\circ}\mathrm{C}$ 干燥 $1\mathrm{h}$ 的基准碳酸钠，加无二氧化碳的水溶解，移入 $1000\mathrm{mL}$ 容量瓶中，用无二氧化碳的水稀释至刻度，摇匀。  
A.4.2.2.4 无机碳标准使用溶液: $1 \mathrm{~mL}$ 溶液含碳(C) $0.01 \mathrm{mg}$ 。用移液管移取 $1.00 \mathrm{~mL}$ 无机碳标准储备溶液(见 A.4.2.2.3), 置于 $100 \mathrm{~mL}$ 容量瓶中, 用无二氧化碳的水稀释至刻度, 摇匀。  
A.4.2.2.5 无二氧化碳的水：按GB/T603规定的方法制备。

# A.4.2.3 仪器和设备

总有机碳(TOC)分析仪：使用的高纯氧符合GB/T14599的要求。

# A.4.2.4 分析步骤

# A.4.2.4.1 标准曲线的绘制

分别移取 $0.00\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},3.00\mathrm{mL},4.00\mathrm{mL}$ 的总碳标准使用溶液，置于 $100~\mathrm{mL}$ 容量瓶中，用无二氧化碳的水稀释到刻度，摇匀。将总有机碳(TOC)分析仪调至最佳试验条件，吸取标准溶液测量峰面积，从每个标准溶液的峰面积中减去空白溶液的峰面积，以碳的质量浓度 $(\mathrm{mg / L})$ 为横坐标，对应的峰面积为纵坐标，绘制总碳标准曲线。

分别移取 $0.00\mathrm{mL},1.00\mathrm{mL},2.50\mathrm{mL},5.00\mathrm{mL},10.00\mathrm{mL}$ 的无机碳标准使用溶液，置于 $100~\mathrm{mL}$ 容量瓶中，用无二氧化碳的水稀释到刻度，摇匀。将总有机碳(TOC)分析仪调至最佳试验条件，吸取标准溶液测量峰面积，从每个标准溶液的峰面积中减去空白溶液的峰面积，以碳的质量浓度 $(\mathrm{mg / L})$ 为横坐标，对应的峰面积为纵坐标，绘制无机碳标准曲线。

# A.4.2.4.2 测定

称取约 $3\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ 。置于 $100~\mathrm{mL}$ 容量瓶中，用无二氧化碳的水稀释至刻度，摇匀。用总有机碳(TOC)分析仪分别测定试样溶液中总碳和无机碳产生的峰面积，分别从总碳标准曲线和无机碳标准曲线上查出试样溶液中总碳和无机碳的质量浓度 $(\mathrm{mg / L})$ 。

# A.4.2.5 结果计算

总有机碳（以C计)的质量分数 $\omega_{1}$ ，按式(A.2)计算：

$$
w _ {1} = \frac {\left(\rho_ {\mathrm {TC}} - \rho_ {\mathrm {IC}}\right) \times 0.1}{m _ {2} \times 1000} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$\rho_{\mathrm{TC}}$ ——从标准曲线上查出的试样溶液中总碳的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_{\mathrm{IC}}$ ——从标准曲线上查出的试样溶液中无机碳的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

0.1 ——试样溶液的体积，单位为升 $(\mathrm{L})$

$m_{2}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.0005\%$ 。

# A.5 氯化物（以CI计）的测定

# A.5.1 方法提要

在硝酸介质中，试样中的氯化物与加入的硝酸银形成氯化银沉淀，通过与标准比浊液比较，确定试样中氯化物的含量。

# A.5.2 试剂和材料

A.5.2.1 硝酸溶液： $1 + 2$

A.5.2.2 硝酸银溶液： $17\mathrm{g / L}$

A.5.2.3 氯化物标准溶液： $1\mathrm{mL}$ 溶液含氯(Cl) $0.010\mathrm{mg}$ 。用移液管移取 $10.00\mathrm{mL}$ 按GB/T602制备的氯化物标准溶液，置于 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。该溶液现用现配。

# A.5.3 分析步骤

称取 $5.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $25~\mathrm{mL}$ 比色管中。加水至体积约 $20~\mathrm{mL}$ ，依次加入 $2\mathrm{mL}$ 硝酸溶液和 $1\mathrm{mL}$ 硝酸银溶液，用水稀释至刻度，摇匀。放置 $10\mathrm{min}$ ，与标准比浊液比较。试样溶液的浊度不应大于标准比浊液。

标准比浊液的制备：用移液管移取 $15.00\mathrm{mL}$ 氯化物标准溶液，置于 $25~\mathrm{mL}$ 比色管中，与试样同时同样处理。

# A.6 硝酸盐（以 $\mathbf{NO}_3$ 计）的测定

# A.6.1 试剂和材料

A.6.1.1 硫酸。

A.6.1.2 氯化钠。

A.6.1.3靛蓝溶液：称取 $0.18\mathrm{g}$ 非蓝二磺酸钠 $\left(\mathrm{C}_{16} \mathrm{H}_{8} \mathrm{O}_{8} \mathrm{~N}_{2} \mathrm{~S}_{2} \mathrm{Na}_{2}\right)$ ，溶解于 $100 \mathrm{~mL}$ 水中。使用期为 $60 \mathrm{~d}$ 。

# A.6.2 分析步骤

# A.7.4.4 测定

称取 $3\mathrm{g}\sim 4\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ 。置于 $100~\mathrm{mL}$ 容量瓶中，加入 $5\mathrm{mL}$ 硝酸溶液，用水稀释至刻度，摇匀。同时制备空白试样溶液。

按照电感耦合等离子体发射光谱的操作规程，使用与标准曲线绘制时相同的仪器工作条件和待测元素测定波长，测量试样溶液和空白试样溶液中被测元素的发射光谱强度，根据试样溶液和空白试样溶液产生的发射光谱强度计算出试样溶液中各待测元素的质量浓度 $(\mathrm{mg / L})$ 。

# A.7.5 结果计算

硫酸盐的质量分数 $\varpi_{2}$ ，按式(A.3)计算：

$$
w _ {2} = \frac {\left(\rho_ {1} - \rho_ {0}\right) \times 0.1}{m _ {3} \times 1000} \times 100 \% \quad \dots \tag{A.3}
$$

式中：

$\rho_{1}$ ——由标准曲线上查得试样溶液中硫酸盐的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——由标准曲线上查得空白试样溶液中硫酸盐的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

0.1 ——试样溶液的体积，单位为升 $(\mathrm{L})$ ；

$m_{3}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

铁(Fe)[或铅(Pb)、镉(Cd)]质量分数 $w_{3}$ ，数值以毫克每千克 $(\mathrm{mg / kg})$ 表示，按式(A.4)计算：

$$
w _ {3} = \frac {\left(\rho_ {3} - \rho_ {2}\right) \times 0 . 1 \times 1 0 0 0}{m _ {3}} \quad \dots \tag {A.4}
$$

式中：

$\rho_{3}$ ——由标准曲线上查得试样溶液中待测元素的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_{2}$ ——由标准曲线上查得空白试样溶液中待测元素的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

0.1 ——试样溶液的体积，单位为升 $(\mathrm{L})$ ；

1000——换算系数；

$m_{3}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于表A.4的规定。

# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替NY/T1726—2009《蔬菜中非草隆等15种取代脲类除草剂残留量的测定液相色谱法》。与NY/T1726—2009相比，主要变化如下：

对标准名称进行了修改，增加了食品安全国家标准部分；  
根据食品安全标准的格式进行了修改；  
——规范性引用文件中增加GB2763《食品安全国家标准 食品中农药最大残留限量》标准；  
在试样制备中增加了取样部位的规定及细化了试样制备的要求；  
——增加了精密度要求。

本标准所代替标准的历次版本发布情况为：

—NY/T1726—2009。

# 食品安全国家标准

# 蔬菜中非草隆等15种取代脲类除草剂残留量的测定 液相色谱法

# 1 范围

本标准规定了蔬菜中非草隆、丁噻隆、甲氧隆、灭草隆、绿麦隆、氟草隆、异丙隆、敌草隆、绿谷隆、溴谷隆、炔草隆、环草隆、利谷隆、氯溴隆和草不隆15种取代脲类除草剂残留量液相色谱法测定方法。

本标准适用于蔬菜中上述15种取代脲类除草剂残留量的测定。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的应用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样经乙腈匀浆提取，提取液经盐析，过弗罗里硅土固相萃取柱净化。样品经浓缩后，进液相色谱，经 $\mathrm{C_{18}}$ 分离后，样品过光解反应器分解成伯胺，与OPA反应生成荧光物质。在激发波长 $350~\mathrm{nm}$ ，发射波长 $450~\mathrm{nm}$ 处测定，根据保留时间定性、外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为色谱纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN)}$   
4.1.2 甲醇（ $\mathrm{CH}_3\mathrm{OH})$   
4.1.3 丙酮 $\mathrm{(C_2H_6CO)}$   
4.1.4 正己烷 $(\mathrm{C_6H_{14}})$ 。  
4.1.5 异丙醇 $\mathrm{(C_3H_8OH)}$   
4.1.6 氯化钠（NaCl，分析纯）： $140^{\circ}\mathrm{C}$ 烘烤 $4\mathrm{h}$ ，在干燥器内冷却至室温，储于干燥器中备用。

# 4.2 溶液配制

4.2.1 四硼酸钠溶液 $(4.0\mathrm{g / L})$ ：称取 $7.5\mathrm{g}$ 四硼酸钠 $\mathrm{(Na_2B_4O_7\cdot 10H_2O)}$ ，用水溶解并稀释至1000mL。  
4.2.2 邻苯二甲醛的甲醇溶液 $(10\mathrm{g / L})$ ：称取 $1.0\mathrm{g}$ 邻苯二甲醛 $\left(\mathrm{C}_{8}\mathrm{H}_{6}\mathrm{O}_{2}\right)$ ，用甲醇溶解并稀释至 $100\mathrm{mL}$ 。  
4.2.3 硫基乙醇溶液 $(200\mathrm{g / L})$ ：称取 $20.0\mathrm{g}$ 硫基乙醇 $\left[\left(\mathrm{CH}_{3}\right)_{2}\mathrm{N} \left(\mathrm{CH}_{2}\right)_{2}\mathrm{SH} \cdot \mathrm{HCl}\right]$ ，用四硼酸钠溶液溶解稀释至 $100\mathrm{mL}$ 。  
4.2.4 柱后衍生试剂: 吸取 $5 \mathrm{~mL}$ 邻苯二甲醛溶液和 $5 \mathrm{~mL}$ 疏基乙醇溶液, 加入到 $500 \mathrm{~mL}$ 四硼酸钠溶液中。

# 4.3 标准品

15种农药标准品，纯度 $\geqslant 95\%$

# 4.4 标准溶液配制

4.4.1 单一农药标准溶液: 分别准确称取各农药标准品, 用异丙醇溶解稀释, 逐一配成 $1000 \mathrm{mg} / \mathrm{L}$ 的单一农药标准储备溶液, $-18^{\circ} \mathrm{C}$ 以下储存。使用时根据各农药的响应值, 吸取适量的标准储备液, 用乙腈稀释配制成所需的标准工作溶液。  
4.4.2 混合农药标准溶液：根据各农药的响应值，逐一吸取一定体积的各农药标准储备液分别注入同一容量瓶中，用乙腈稀释至刻度，配制成农药混合标准储备溶液。使用前，用乙腈稀释成所需的标准工作液。混合标准工作溶液质量浓度参见附录A。

# 4.5 材料

弗罗里硅土固相萃取小柱：填充物 $1000 \mathrm{mg}$ , 容积 $6 \mathrm{~mL}$

# 5 仪器

5.1 液相色谱仪：附荧光检测器。  
5.2 带紫外光解的柱后衍生装置，紫外灯，波长 $254\mathrm{nm}, 4\mathrm{W}$ ；光解反应管，长度 $3\mathrm{m}$ ，管内径 $0.5\mathrm{mm}$ 。连接示意图参见附录B。  
5.3 高速匀浆机，转速 $>10000\mathrm{r / min}$   
5.4 氮吹仪。

# 6 分析步骤

# 6.1 试样的制备

将蔬菜样品按 GB 2763 的规定取样, 对于个体较小的样品, 取样后全部处理; 对于个体较大的基本均匀样品, 可在对称轴或对称面上分割或切成小块后处理; 对于细长、扁平或组分含量在各部分有差异的样品, 可在不同部位切取小片或截成小段或处理; 取后的样品将其切碎, 充分混匀, 用四分法取样或直接放入组织捣碎机中捣碎成匀浆。匀浆放入聚乙烯瓶中于 $-20^{\circ} \mathrm{C} \sim -16^{\circ} \mathrm{C}$ 条件下保存。

# 6.2 提取

称取 $25 \mathrm{~g}$ 试样(精确到 $0.01 \mathrm{~g}$ ) 于 $150 \mathrm{~mL}$ 锥形烧瓶中, 加入 $50.0 \mathrm{~mL}$ 乙腈, 用高速匀浆机 $10000 \mathrm{r} /$ min 匀浆 $2 \mathrm{~min}$ 后用滤纸过滤。滤液收集到装有 $5 \mathrm{~g} \sim 7 \mathrm{~g}$ 氯化钠的 $100 \mathrm{~mL}$ 具塞量筒中, 收集滤液 $30 \mathrm{~mL} \sim 40 \mathrm{~mL}$ , 盖上盖子, 剧烈振荡 $1 \mathrm{~min}$ , 静置 $20 \mathrm{~min}$ , 使乙腈相和水相分层。

# 6.3 净化

吸取上述上层提取液 $10 \mathrm{~mL}$ 于 $20 \mathrm{~mL}$ 离心管中, 于 $50^{\circ} \mathrm{C}$ 用氮气缓慢吹干后, 加入 $2.0 \mathrm{~mL}$ 正己烷溶解残渣。将弗罗里硅土固相萃取小柱依次用 $5 \mathrm{~mL}$ 丙酮一正己烷溶液 $(4 + 6)$ , $5 \mathrm{~mL}$ 正己烷预淋洗条件化, 当溶剂液面到达柱吸附层表面时, 立即加入样品溶液, 用 $15 \mathrm{~mL}$ 离心管收集洗脱液, 用 $5 \mathrm{~mL}$ 丙酮一正己烷溶液 $(4 + 6)$ 洗 $20 \mathrm{~mL}$ 离心管后过弗罗里硅土柱, 并重复一次。将 $15 \mathrm{~mL}$ 离心管置于 $50^{\circ} \mathrm{C}$ 水浴中, 用氮气缓慢吹干后, 加入 $2.5 \mathrm{~mL}$ 乙腈振荡溶解残渣, 再加入 $2.5 \mathrm{~mL}$ 水, 振摇后, 过 $0.22 \mu \mathrm{m}$ 滤膜, 待测。

# 6.4 色谱参考条件

6.4.1 色谱柱： $\mathrm{C_{18}}$ 柱， $5.0 \mu \mathrm{m}, 250 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ ，或相当者。  
6.4.2 柱温： $25^{\circ}C$   
6.4.3 荧光检测器： $\lambda_{\mathrm{ex}} 350 \mathrm{~nm}, \lambda_{\mathrm{em}} 450 \mathrm{~nm}$ 。  
6.4.4 进样量： $20~\mu \mathrm{L}$

6.4.5 流速： $0.75\mathrm{mL / min}$   
6.4.6 流动相梯度：见表1。   
6.4.7 柱后衍生试剂：流速 $0.2\mathrm{mL} / \mathrm{min}$   
6.4.8 光解温度：室温。  
6.4.9 衍生温度：室温。

# 6.5 测定

按照仪器条件，对标准工作溶液和试样溶液等体积交替进样，根据保留时间定性，外标法定量。同时，做试剂空白。

# 7 结果计算

样品中被测农药残留量以质量分数 $\omega$ 计，单位以毫克每千克 $(\mathrm{mg / kg})$ 表示，按式(1)计算。

$$
\omega = \frac {c \times A _ {1} \times V _ {1} \times V}{m \times A _ {2} \times V _ {2}} \times \frac {1 0 0 0}{1 0 0 0} \tag {1}
$$

式中：

$c$ ——标准溶液中各农药的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$A_{1}$ ——样品测定液中各农药的峰面积；

$A_{2}$ ——标准溶液中各农药的峰面积；

$V$ ——样品定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——样品提取液的体积，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——分取乙腈提取液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克(g)；

1000——换算系数。

计算结果应扣除空白值，计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，保留2位有效数字。含量超 $1\mathrm{mg / kg}$ 时，表示到3位有效数字。

# 8 精密度

在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录C的要求。

在再现性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录D的要求。

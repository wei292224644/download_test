2024-03-06 实施

# 食品安全国家标准

# 动物性水产品及其制品中

# 曼氏迭宫绦虫裂头的检验

# 1 范围

本标准规定了动物性水产品及其制品中曼氏迭宫绦虫(Spirometramansoni)裂头蚴的形态学和PCR检验方法。

本标准适用于动物性水产品及其制品中曼氏迭宫绦虫裂头蚴的检验。

# 2 原理

动物性水产品及其制品中的曼氏迭宫绦虫裂头为主要寄生于蛙等水产品的肌肉内。解剖分离获得虫体，根据裂头的形态结构，初步判断裂头种类；通过扩增曼氏迭宫绦虫裂头线粒体DNA的细胞色素氧化酶 $(cox\mathrm{I})$ 基因片段并测序，进行曼氏迭宫绦虫裂头的鉴定。

# 3 仪器和设备

3.1 生物显微镜： $40\times \sim 100\times$   
3.2 体视显微镜： $7.5\times \sim 150\times$   
3.3 PCR扩增仪。  
3.4 凝胶成像系统。  
3.5 电泳仪。  
3.6 高速离心机：转速 $\geqslant 12000\mathrm{r / min}$   
3.7 微量移液器： $0.2\mu \mathrm{L}\sim 2.5\mu \mathrm{L},1\mu \mathrm{L}\sim 10\mu \mathrm{L},10\mu \mathrm{L}\sim 100\mu \mathrm{L},100\mu \mathrm{L}\sim 1000\mu \mathrm{L}$

# 4 试剂和材料

# 4.1 试剂

4.1.1 $1\mathrm{mol} / \mathrm{L}$ Tris-HCl溶液 $(\mathrm{pH}8.0)$ 。  
4.1.2 $0.5\mathrm{mol / L}$ EDTA溶液 $(\mathrm{pH}8.0)$ 。   
4.1.3 $10\%$ SDS溶液。  
4.1.4 $5\mathrm{mol} / \mathrm{L}$ NaCl溶液。  
4.1.5 $20\mathrm{mg / mL}$ 蛋白酶K。  
4.1.6 苯酚/三氯甲烷/异戊醇 $(25:24:1)$ 。  
4.1.7 三氯甲烷。  
4.1.8 $5\mathrm{U} / \mu \mathrm{L}$ 耐热DNA聚合酶。  
4.1.9 $10 \times \mathrm{PCR}$ 缓冲液。  
4.1.10 $25\mathrm{mmol / L}$ $\mathrm{MgCl_2}$ 。

4.1.11 dNTPs: dATP、dTTP、dCTP、dGTP, 每种浓度为 $2.5 \mathrm{mmol} / \mathrm{L}$ 。  
4.1.12 琼脂糖：电泳级。  
4.1.13 $50\times \mathrm{TAE}$ 缓冲液，使用前用去离子水稀释成 $1\times \mathrm{TAE}$ 缓冲液。  
4.1.14 $1 \times \mathrm{TE}$ 溶液 $(\mathrm{pH} 8.0)$ 。   
4.1.15 $10\mathrm{mg / mL}$ 溴化乙锭（EB）或其他核酸染料。  
4.1.16 $6\times$ 上样缓冲液。  
4.1.17 $100\mathrm{bp}\sim 2000\mathrm{bp}$ DNA分子量标准。  
4.1.18 引物：浓度为 $10\mu \mathrm{mol} / \mathrm{L}$

正向引物 SEF3: $5^{\prime}$ -AATTCTTTCTGCTTGTGTGT- $3^{\prime}$

反向引物 SER1:5'-ATCAACAAATAATCCACGCAC-3'

扩增曼氏迭宫绦虫裂头蚴 $\text{cox}$ I 基因片段长度为 $475 \mathrm{bp}$ 。

# 4.2 试剂配制

4.2.1 生理盐水：称取 $8.5\mathrm{gNaCl}$ 溶解于 $900~\mathrm{mL}$ 去离子水，加去离子水至 $1000~\mathrm{mL}$   
4.2.2 $1\mathrm{mol} / \mathrm{L}$ Tris-HCl溶液：称取 $121.1\mathrm{g}$ Tris(三羟甲基氨基甲烷)溶解于 $800~\mathrm{mL}$ 去离子水，用盐酸调节 $\mathrm{pH}$ 至8.0，加去离子水至 $1000~\mathrm{mL}$ 。  
4.2.3 $0.5\mathrm{mol} / \mathrm{L}$ EDTA溶液：称取 $186.1\mathrm{gNa_2EDTA\cdot 2H_2O}$ （二水合乙二胺四乙酸二钠)溶解于 $800~\mathrm{mL}$ 去离子水，搅拌溶解，用 $\mathrm{NaOH}$ 调节 $\mathrm{pH}$ 至8.0，加去离子水至 $1000~\mathrm{mL}$   
4.2.4 $5\mathrm{mol} / \mathrm{L}$ NaCl溶液：称取 $292.5\mathrm{gNaCl}$ 溶解于 $900~\mathrm{mL}$ 去离子水，加去离子水至 $1000~\mathrm{mL}$   
4.2.5 裂解液： $10\%$ SDS溶液 $100~\mathrm{mL}, 1~\mathrm{mol} / \mathrm{L}$ Tris-HCl溶液 $10~\mathrm{mL}, 0.5~\mathrm{mol} / \mathrm{L}$ EDTA溶液 $200~\mathrm{mL}, 5~\mathrm{mol} / \mathrm{L}$ NaCl溶液 $20~\mathrm{mL}$ ，加灭菌去离子水至 $1000~\mathrm{mL}$ 。  
4.2.6 $1.5\%$ 琼脂糖凝胶：琼脂糖 $1.5\mathrm{g}$ ，加入 $1\times \mathrm{TAE}$ 缓冲液至 $100~\mathrm{mL}$ ，加热至完全融化后冷却至 $60^{\circ}C\sim 70^{\circ}C$ ，加入 $10~\mathrm{mg / mL}$ 溴化乙锭 $5\mu \mathrm{L}$ ，混匀，制备凝胶。

# 5 检测方法

# 5.1 形态学方法

# 5.1.1 样品制备

用手术剪对待检的蛙等动物性水产品或其制品逐只进行解剖。去皮后按照皮下、四肢、躯干、内脏、头部的顺序先肉眼观察，后置于体视显微镜下，用解剖针分离肌肉和筋膜，若发现乳白色扁平带状虫体，将其完整取出。

# 5.1.2 镜检

用生物显微镜观察虫体形态，虫体呈白色、带状，大小 $(0.5\mathrm{cm}\sim 80\mathrm{cm})\times (0.3\mathrm{cm}\sim 1\mathrm{cm})$ 。虫体头端膨大，圆形或圆锥形，中央有一明显凹陷；体不分节，具有不规则横皱褶；后端呈钝圆形，具有很强的伸缩蠕动能力，其模式图见图A.1，其实物图见图A.2。具备上述特征的虫体，判断为疑似曼氏迭宫绦虫裂头。虫体立即用于DNA提取或一 $20^{\circ}C$ 保存备用。

# 5.2 PCR方法

# 5.2.1 DNA 抽提

挑取5.1.2分离的疑似虫体，放入 $1.5\mathrm{mL}$ 离心管中，加裂解液 $500~\mu \mathrm{L}$ ，匀浆。加入蛋白酶K $10~\mu \mathrm{L}$ $55^{\circ}C$ 水浴至虫体被完全消化 $(1\mathrm{h}\sim 3\mathrm{h})$ 。在混合液中加入苯酚/三氯甲烷/异戊醇 $(25:24:1)500~\mu \mathrm{L}$

混匀， $4^{\circ}\mathrm{C}12000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。取上层水相，加入等体积的三氯甲烷，混匀， $4^{\circ}\mathrm{C}12000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。取上层水相加入2倍体积 $-20^{\circ}\mathrm{C}$ 预冷的无水乙醇，放置 $1\mathrm{h}$ ，充分混匀， $4^{\circ}\mathrm{C}12000\mathrm{r / min}$ 离心 $15\mathrm{min}$ 。倾去上清液，加入 $75\%$ 乙醇 $700~\mu \mathrm{L}$ 冲洗沉淀， $4^{\circ}\mathrm{C}12000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃上清液，干燥后加入 $50~\mu \mathrm{L}1\times \mathrm{TE}$ 溶液溶解DNA，立即检测或 $-20^{\circ}\mathrm{C}$ 保存备用。

注：根据实验室实际情况，可使用经验证的商品化组织DNA提取试剂盒提取DNA。

# 5.2.2 PCR反应体系

在PCR管中依次加入 $10\times \mathrm{PCR}$ 缓冲液 $5.0\mu \mathrm{L},\mathrm{MgCl}_2$ 5.0 $\mu \mathrm{L}$ 、dNTPs $2.0\mu \mathrm{L}$ 引物SEF3和SER1各 $2.0\mu \mathrm{L}$ 耐热DNA聚合酶 $0.5\mu \mathrm{L}$ DNA模板 $2.5\mu \mathrm{L}$ ，加灭菌去离子水至总体积 $50~\mu \mathrm{L}$ 。每次试验需设阳性、阴性和空白对照。阳性对照用曼氏迭宫绦虫裂头蚴DNA或含有目标基因序列的质控品，阴性对照用不含虫体的动物性水产品DNA做模板，空白对照用灭菌去离子水做模板。

# 5.2.3 PCR反应条件

$94^{\circ} \mathrm{C}$ 预变性 $5 \mathrm{~min}; 94^{\circ} \mathrm{C}$ 变性 $30 \mathrm{~s}, 55^{\circ} \mathrm{C}$ 退火 $30 \mathrm{~s}, 72^{\circ} \mathrm{C}$ 延伸 $1 \mathrm{~min}$ , 扩增 35 个循环; $72^{\circ} \mathrm{C}$ 延伸 $10 \mathrm{~min}, 4^{\circ} \mathrm{C}$ 保存。

# 5.2.4 电泳

取PCR扩增产物 $10\mu \mathrm{L}$ 与 $6\times$ 上样缓冲液 $2\mu \mathrm{L}$ 混合，加样于 $1.5\%$ 琼脂糖凝胶中，其中一孔加入DNA分子量标准。 $1\times$ TAE电泳缓冲液， $5\mathrm{V / cm}$ 恒压电泳 $30\mathrm{min}\sim 40\mathrm{min}$ ，用凝胶成像系统观察和记录结果。

# 5.2.5 PCR结果判定

阳性对照出现预期大小的条带(475 bp)，阴性对照和空白对照无条带，待测样品扩增出预期大小的条带，可判定PCR结果为阳性；无扩增条带或未扩增出预期大小的条带均判为PCR结果为阴性。

取PCR结果为阳性的PCR产物进行基因序列双向测序，将测序结果与曼氏迭宫绦虫裂头蚴参考序列(见附录B)进行同源性比对。

# 6 结果报告

6.1 形态学方法检出疑似裂头、PCR结果为阳性且扩增片段基因序列与曼氏迭宫绦虫裂头参考序列同源性 $\geq 98\%$ ，报告检出曼氏迭宫绦虫裂头。  
6.2 形态学方法未检出裂头、或PCR结果为阴性、或扩增片段基因序列与曼氏迭宫绦虫裂头参考序列同源性 $< 98\%$ ，报告未检出曼氏迭宫绦虫裂头。

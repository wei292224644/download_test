# 前言

本标准代替GB/T5009.210—2008《食品中泛酸的测定》、GB5413.17—2010《食品安全国家标准婴幼儿食品和乳品中泛酸的测定》。

本标准与GB/T5009.210—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中泛酸的测定”；  
——增加了高效液相色谱方法；  
——修改了食品的前处理方法；  
——增加了检出限和定量限。

# 食品安全国家标准

# 食品中泛酸的测定

# 1 范围

本标准规定了食品中泛酸和泛酸钙的测定方法。

本标准第一法适用于食品中泛酸的测定，第二法适用于营养素补充剂类保健食品和配方食品中泛酸(钙)的测定。

# 第一法 微生物法

# 2 原理

泛酸是植物乳杆菌Lactobacillus plantarum(ATCC 8014)生长所必需的营养素，在一定控制条件下，将植物乳杆菌液接种至含有试样液的培养液中，培养一定时间后测定透光率(或吸光度值)，根据泛酸含量与透光率(或吸光度值)的标准曲线计算出试样中泛酸的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.2 试剂配制

3.2.1 乙酸溶液 $(0.2\mathrm{mol} / \mathrm{L})$ ：吸取 $11.8~\mathrm{mL}$ 冰乙酸，用水稀释至 $1000~\mathrm{mL}$ ，混匀。

3.2.2 乙酸钠溶液 $(0.2\mathrm{mol} / \mathrm{L})$ ：称取 $27.2\mathrm{g}$ 三水合乙酸钠，加水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。

3.2.3 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：吸取 $83~\mathrm{mL}$ 盐酸，加水稀释至 $1000~\mathrm{mL}$ ，混匀。

3.2.4 盐酸浸泡液：吸取 $100~\mathrm{mL}$ 盐酸与50倍水混合。

3.2.5 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $40\mathrm{g}$ 氢氧化钠，加水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。

3.2.6 氢氧化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $4\mathrm{g}$ 氢氧化钠，加水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。

3.2.7 Tris缓冲液：称取 $121.0\mathrm{g}$ 三羟甲基氨基甲烷溶于 $500~\mathrm{mL}$ 水中，用冰乙酸调 $\mathrm{pH}$ 至 $8.1\pm 0.1$ ，加水至 $1000~\mathrm{mL}$ ，混匀。贮存于 $2^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中，可保存2周。

3.2.8 生理盐水：称取 $9\mathrm{g}$ 氯化钠，加水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。临用前预先灭菌，于 $121^{\circ}\mathrm{C}$ 高压灭菌 $10\mathrm{min}$ 后备用。

3.2.9 乙醇溶液 $(20\%)$ ：量取 $200~\mathrm{mL}$ 无水乙醇与 $800~\mathrm{mL}$ 水混匀。

3.2.10 碳酸钠溶液 $(0.08\mathrm{mol} / \mathrm{L})$ ：称取 $8.5\mathrm{g}$ 碳酸钠，加水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。

3.2.11 碳酸氢钾溶液 $(0.02\mathrm{mol} / \mathrm{L})$ ：称取 $2\mathrm{g}$ 碳酸氢钾，加水溶解并稀释至 $1000~\mathrm{mL}$ 。混匀。

3.2.12 碱性磷酸酶溶液：称取 $2\mathrm{g}$ 碱性磷酸酶，加水溶解并稀释至 $100~\mathrm{mL}$ 。临用现配， $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱预冷。

# 3.2.13 鸽子肝脏提取液

3.2.13.1 活化 Dowex $1 \times 8$ ：称取 $100 \mathrm{~g}$ Dowex $1 \times 8$ 于锥形瓶中，加入 $1 \mathrm{~L}$ 盐酸溶液，置于振荡器上充分振摇 $10 \mathrm{~min}$ ，用铺有滤纸的布氏漏斗过滤。Dowex $1 \times 8$ 转回锥形瓶，再加入 $1 \mathrm{~L}$ 盐酸溶液重复振摇、过滤。Dowex $1 \times 8$ 加入 $1 \mathrm{~L}$ 水振摇 $10 \mathrm{~min}$ ，过滤，重复用水洗涤10次。逐滴加入Tris缓冲液调节 Dowex $1 \times 8 \mathrm{pH}$ 至 $8.0 \pm 0.1$ 。 $2^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 冰箱保存，2天内用完。

3.2.13.2 鸽子肝脏提取液：配制此试剂前一天将所用容器放入 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱中冷藏过夜。称取 $30\mathrm{g}$ 鸽子肝脏丙酮提取物干粉，放入研钵中，冰浴条件下分两次加入 $300\mathrm{mL}$ 碳酸氢钾溶液研磨至匀浆，转入具塞离心管中，盖好塞后充分振摇， $-20^{\circ}\mathrm{C}$ 冷冻 $10\mathrm{min}$ 后以 $3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将上清液转至 $500\mathrm{mL}$ 广口瓶中。加 $150\mathrm{g}$ 活化 Dowex $1 \times 8$ ，放入冰浴中混匀 $5\mathrm{min}$ ，将混合液倒入离心管中， $3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。将上清液移入另一 $500\mathrm{mL}$ 预冷的广口瓶中， $-20^{\circ}\mathrm{C}$ 冷冻 $10\mathrm{min}$ ，再加 $150\mathrm{g}$ 活化 Dowex $1 \times 8$ ，放入冰浴中混匀 $5\mathrm{min}$ ，将混合液倒入离心管中， $3000\mathrm{r / min}$ 离心，将上清液分装于具塞试管中（每管大约 $3\mathrm{mL}$ ）， $-20^{\circ}\mathrm{C}$ 冷冻保存。用前于 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱内化冻并保存至用时。

# 3.3 培养基

3.3.1 甲盐溶液：分别称取 $25\mathrm{g}$ 磷酸氢二钾和 $25\mathrm{g}$ 三水合磷酸二氢钾，加水溶解并稀释至 $500~\mathrm{mL}$ ，混匀。加入 $1\mathrm{mL}$ 甲苯， $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱可保存1年。  
3.3.2 乙盐溶液：分别称取 $10\mathrm{g}$ 七水合硫酸镁、 $0.5\mathrm{g}$ 氯化钠、 $0.5\mathrm{g}$ 七水合硫酸亚铁和 $0.5\mathrm{g}$ 一水合硫酸锰，加水溶解并稀释至 $500\mathrm{mL}$ 。加5滴盐酸， $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱可保存1年。  
3.3.3 琼脂培养基：按表1称取或吸取各试剂，加水至 $100~\mathrm{mL}$ ，混合，沸水浴加热至琼脂完全溶化。趁热用 $1\mathrm{mol} / \mathrm{L}$ 盐酸溶液和/或 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液调节 $\mathrm{pH}$ 至 $6.8\pm 0.1$ 。尽快分装，根据试管内径粗细加入 $3\mathrm{mL}\sim 5\mathrm{mL}$ ，液面高度不得低于 $2\mathrm{cm}$ 。塞上棉塞， $121^{\circ}\mathrm{C}$ 高压灭菌 $15\mathrm{min}$ 。取出后试管直立放

置，待冷却后于冰箱内保存，备用。

# 3.4 标准品

D-泛酸钙 $\mathrm{(C_{18}H_{32}CaN_2O_{10})}$ ：纯度 $\geqslant 99\%$ 。

# 3.5 标准溶液的配制

3.5.1 泛酸标准储备溶液 $(40.0\mu \mathrm{g} / \mathrm{mL})$ ：精确称取 $43.5\mathrm{mg}$ 预干燥至恒重的D-泛酸钙，加水溶解并转移至 $1000~\mathrm{mL}$ 容量瓶中，加 $10~\mathrm{mL}$ 乙酸溶液， $100~\mathrm{mL}$ 乙酸钠溶液，用水定容至刻度。储存于棕色瓶中，加入3滴～5滴甲苯，于 $2^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中可保存2年。  
3.5.2 泛酸标准中间液 $(1.00\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取 $25.0~\mathrm{mL}$ 泛酸标准储备溶液置于 $1000~\mathrm{mL}$ 容量瓶中，加入 $10~\mathrm{mL}$ 乙酸溶液， $100~\mathrm{mL}$ 乙酸钠溶液，用水定容至刻度。加入3滴～5滴甲苯于 $2^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中可保存1年。  
3.5.3 泛酸标准工作溶液 $(20\mathrm{ng / mL})$ ：准确吸取 $2.00~\mathrm{mL}$ 泛酸标准中间溶液置于 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。临用前现配。

# 5 菌种的制备与保存

# 5.1 菌种

植物乳杆菌 Lactobacillus plantarum (ATCC 8014)。

# 5.2 储备菌种的制备

将菌种植物乳杆菌Lactobacillus plantarum(ATCC8014)转接至琼脂培养基中，在 $37^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温箱中培养 $20\mathrm{h} \sim 24\mathrm{h}$ ，连续传种2次～3次。取出后放入 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱作为储备菌株保存。每月至少传代一次，不宜超过25代。

试验前将储备菌株接种至琼脂培养基中， $37^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温培养箱中培养 $20\mathrm{h} \sim 24\mathrm{h}$ 以活化菌株，用于接种液的制备。

注：保存数周以上的储备菌株,不能立即用作接种液制备,试验前宜连续传种 2 代 $\sim  3$ 代以保证细菌活力。

# 5.3 接种液的制备

试验前一天，取 $2\mathrm{mL}$ 泛酸标准工作溶液和 $4\mathrm{mL}$ 泛酸测定用培养液混匀，分装于2支 $5\mathrm{mL}$ 离心管中，塞好棉塞，于 $121~^\circ \mathrm{C}$ 高压灭菌 $15\mathrm{min}$ 后即为种子培养液。冷却后用接种环将活化的菌株转种至2支种子培养液中，于 $37~^\circ \mathrm{C}\pm 1~^\circ \mathrm{C}$ 恒温培养箱中培养 $20\mathrm{h}\sim 24\mathrm{h}$ 。取出后 $3000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，弃上清液，无菌操作下用已预先灭菌的生理盐水淋洗2次， $3000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，弃上清液。再加入 $3\mathrm{mL}$ 灭菌生理盐水，振荡混匀，制成接种液，立即使用。

# 6 分析步骤

注：所有操作均需避光进行

# 6.1 试样制备

谷薯类、豆类、乳粉等试样需粉碎、研磨、过筛（筛板孔径 $0.3\mathrm{mm}\sim 0.5\mathrm{mm}$ ）；肉、蛋、坚果等用匀质器制成食糜；果蔬、半固体食品等试样需匀浆混匀；液体试样用前振摇混合。 $4^{\circ}\mathrm{C}$ 冰箱可保存1周。

# 6.2 试样提取

# 6.2.1 直接提取法

形态为颗粒、粉末、片剂、液体的营养素补充剂或强化剂、预混料，添加了泛酸钙的配方食品或保健食品可采用直接提取法。

准确称取适量试样 $(m)$ ，固体试样 $0.2\mathrm{g}\sim 2\mathrm{g}$ ；液态试样 $5\mathrm{g}\sim 10\mathrm{g}$ ，精确至 $0.001\mathrm{g}$ ，转入 $100\mathrm{mL}$ 锥形瓶中，加入 $80~\mathrm{mL}$ 乙醇溶液，具塞，超声振摇提取 $4\mathrm{h}$ 以上至试样完全溶解，用水定容至 $100~\mathrm{mL}$ $(V_{1})$ 。

# 6.2.2 酶解法

一般谷薯类、肉蛋乳类、果蔬菌藻类、豆及坚果类等食品试样宜采用酶解提取法。

6.2.2.1 水解：准确称取适量试样 $(m$ ，约含 $0.02\mathrm{mg}\sim 0.2\mathrm{mg}$ 泛酸)，精确至 $0.001\mathrm{g}$ 。一般谷薯类、肉类、蛋类、豆类及其制品称取 $1\mathrm{g}\sim 5\mathrm{g}$ ；新鲜果蔬、乳及其制品称取 $5\mathrm{g}\sim 10\mathrm{g}$ 。转入至 $100\mathrm{mL}$ 锥形瓶中，加 $10\mathrm{mL}$ Tris缓冲液、 $40\mathrm{mL}$ 水，振荡混匀。于 $121^{\circ}C$ 高压条件下水解 $15\mathrm{min}$ 。冷却至室温，转移

至 $100\mathrm{mL}$ 容量瓶中，用水定容至刻度 $(V_{1})$ ，过滤。

6.2.2.2 酶解：准确吸取适量试样滤液 $(1.0\mathrm{mL}\sim 10.0\mathrm{mL},V_{2})$ 至 $25~\mathrm{mL}$ 具塞刻度试管底部，补水至 $10~\mathrm{mL}$ ，加 $5\mathrm{mL}$ Tris缓冲液。在冰浴条件下，依次小心加入预冷的 $0.1\mathrm{mL}$ 碳酸氢钠溶液、 $0.4\mathrm{mL}$ 碱性磷酸酶溶液、 $0.2\mathrm{mL}$ 鸽子肝脏提取液和 $0.4\mathrm{mL}$ 水。小心混匀试管，避免混合物黏附于试管壁，加1滴甲苯， $37^{\circ}C\pm 1^{\circ}C$ 温箱中温育过夜(8h以上)。加水至 $20~\mathrm{mL}$ ，用冰乙酸调节 $\mathrm{pH}$ 至 $4.5\pm 0.1$ ，加水定容至 $25.0~\mathrm{mL}(V_3)$ ，过滤。调节 $\mathrm{pH}$ ：吸取适量的试样酶解液 $(2.0\mathrm{mL}\sim 20.0\mathrm{mL},V_4)$ 于 $25~\mathrm{mL}$ 具塞刻度试管中，加水至 $20~\mathrm{mL}$ ，用 $0.1\mathrm{mol / L}$ 氢氧化钠溶液调节 $\mathrm{pH}$ 至 $6.8\pm 0.1$ ，用水定容至 $25.0~\mathrm{mL}(V_5)$ 。

另取一试管加入 $5\mathrm{mL}$ Tris缓冲液和 $10~\mathrm{mL}$ 水，同法加入碳酸氢钠溶液、碱性磷酸酶溶液、鸽子肝脏提取液酶液及温育过夜，并调节 $\mathrm{pH}$ 至 $6.8\pm 0.1$ ，用水定容至 $25.0~\mathrm{mL}$ 作为酶空白液。

注：以谷物、乳粉等为基质的配方食品如需计量基质本底泛酸含量，可采用酶解法提取。

# 6.3 稀释

根据试样中泛酸含量用水对试样提取液进行适当稀释 $(F)$ ，使稀释后试样提取液中泛酸浓度约为 $20\mathrm{ng / mL}$ 。

# 6.4 测定系列管制备

所用试管使用前洗刷干净，用水煮沸 $30\mathrm{min}$ ，沥干后放入盐酸浸泡液中浸泡 $2\mathrm{h}$ ，经 $170^{\circ}\mathrm{C}$ 烘 $3\mathrm{h}$ 后使用。

# 6.4.1 试样和酶空白系列管

取4支试管，分别加入 $1.0\mathrm{mL},2.0\mathrm{mL},3.0\mathrm{mL},4.0\mathrm{mL}$ 试样提取液 $(V_{x})$ ，补水至 $5.0~\mathrm{mL}$ ，加入 $5.0~\mathrm{mL}$ 泛酸测定用培养液，混匀。另取4支试管分别加入 $1.0\mathrm{mL},2.0\mathrm{mL},3.0\mathrm{mL},4.0\mathrm{mL}$ 酶空白液。

# 6.4.2 标准系列管

取试管分别加入泛酸标准工作溶液 $0.00\mathrm{mL}, 0.50\mathrm{mL}, 1.00\mathrm{mL}, 1.50\mathrm{mL}, 2.00\mathrm{mL}, 2.50\mathrm{mL}, 3.00\mathrm{mL}, 3.50\mathrm{mL}, 4.00\mathrm{mL}, 4.50\mathrm{mL}, 5.00\mathrm{mL}$ 于试管中，补水至 $5.0\mathrm{mL}$ ，相当于标准系列管中泛酸含量为 $0\mathrm{ng}, 10\mathrm{ng}, 20\mathrm{ng}, 30\mathrm{ng}, 40\mathrm{ng}, 50\mathrm{ng}, 60\mathrm{ng}, 70\mathrm{ng}, 80\mathrm{ng}, 90\mathrm{ng}, 100\mathrm{ng}$ 泛酸，再加入 $5.0\mathrm{mL}$ 泛酸测定用培养液，混匀。为保证标准曲线的线性关系，应制备2套～3套标准系列管，绘制标准曲线时，以每个标准点的均值计算。

# 6.5 灭菌

将所有测定管塞好棉塞，于 $121^{\circ}\mathrm{C}$ 高压灭菌 $15\mathrm{min}$

# 6.6 接种和培养

待测定系列管冷却至室温后，在无菌操作条件下向每支测定管滴加接种液 $50~\mu \mathrm{L}$ 。塞好棉塞，置于 $37^{\circ}C \pm 1^{\circ}C$ 恒温培养箱中培养 $16\mathrm{h} \sim 20\mathrm{h}$ ，直至达到最大混浊度，即再培养 $2\mathrm{h}$ 后透光率(或吸光度值)无明显变化。另准备一支标准0管（含 $0\mathrm{ng}$ 泛酸）不接种作为0对照管。

# 6.7 测定

将培养好的标准系列管、试样和酶空白系列管用漩涡混匀器混匀。用厚度为 $1\mathrm{cm}$ 比色杯，于 $550~\mathrm{nm}$ 处，以未接种的0对照管调节透光率为 $100\%$ （或吸光度值为0)，依次测定标准系列管、试样和酶空白系列管的透光率(或吸光度值)。如果0对照管有明显的细菌增长，或者与0对照管相比，标准0管透光率在 $90\%$ 以下(或吸光度值在0.2以上)；或标准系列管透光率最大变化量 $<  40\%$ (或吸光度值变

化量 $< 0.4$ )，说明可能有杂菌或不明来源的泛酸混入，需重做试验。

注：泛酸测定适宜的光谱范围 $540\mathrm{nm}\sim 660\mathrm{nm}$

# 6.8 分析结果表述

6.8.1 标准曲线：以标准系列管泛酸含量为横坐标，每个标准点透光率(或吸光度值)均值为纵坐标，绘制标准曲线。  
6.8.2 试样结果计算：从标准曲线查得试样和酶空白系列管中泛酸的相应含量 $(\rho_{\mathrm{x}})$ ，如果每个试样的4支试样系列管中有3支以上泛酸含量在 $10\mathrm{ng}\sim 80\mathrm{ng}$ 范围内，且按照式(1)计算每毫升试样稀释液中泛酸浓度 $(\rho)$ ，各管之间相对偏差小于 $15\%$ ，则可继续按式(2)或式(3)～式(5)进行结果计算，否则需重新取样测定。

试样稀释液中泛酸浓度按式(1)计算：

$$
\rho = \frac {\rho_ {\mathrm {x}}}{V _ {\mathrm {x}}} \tag {1}
$$

式中：

$\rho$ ——试样稀释液中泛酸浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$\rho_{\mathrm{x}}$ ——从标准曲线上查得测定系列管中泛酸含量，单位为纳克(ng)；

$V_{\mathrm{x}}$ ——制备试样系列管时吸取的试样稀释液体积，单位为毫升 $(\mathrm{mL})$ 。

采用直接提取法的试样中泛酸含量按式(2)计算：

$$
X = \frac {\bar {\rho} \times V _ {1} \times F}{m} \times \frac {1 0 0}{1 0 ^ {6}} \quad \dots \tag {2}
$$

式中：

X ——试样中泛酸含量，固态试样单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ，液态试样为毫克每百毫升 $(\mathrm{mg} / 100\mathrm{mL})$

$\bar{\rho}$ ——试样稀释液泛酸浓度平均值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V_{1}$ ——试样提取液的定容体积，单位为毫升 $(\mathrm{mL})$

$F$ ——试样提取液稀释倍数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$\frac{100}{10^6}$ ——换算系数。

采用酶解法的试样中泛酸含量按式(3)～式(5)计算：

$$
m _ {0} = \bar {\rho_ {0}} \times 2 5 \quad \dots \tag {3}
$$

$$
m _ {\mathrm {x}} = \frac {\bar {\rho} \times V _ {5} \times V _ {3} \times F}{V _ {4}} \tag {4}
$$

$$
X = \frac {\left(m _ {x} - m _ {0}\right) \times V _ {1}}{m \times V _ {2}} \times \frac {1 0 0}{1 0 ^ {6}} \tag {5}
$$

式中：

$m_{0}$ ——酶空白液中泛酸含量，单位为纳克 $(\mathrm{ng})$ ；

$\bar{\rho}_0$ ——酶空白液中泛酸浓度平均值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

25 ——酶空白液总体积，单位为毫升 $(\mathrm{mL})$ ；

$m_{\mathrm{x}}$ ——试样酶解液中泛酸含量，单位为纳克 $(\mathrm{ng})$ ；

$\bar{\rho}$ ——试样稀释液泛酸浓度平均值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V_{5}$ ——试样调 $\mathrm{pH}$ 后的定容体积，单位为毫升 $(\mathrm{mL})$

$V_{3}$ ——试样酶解液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$F$ ——试样提取液稀释倍数；

$V_{4}$ ——试样调 $\mathrm{pH}$ 时吸取的酶解液体积，单位为毫升 $(\mathrm{mL})$ ；

X ——试样中泛酸含量，固态试样单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ，液态试样为毫克每百毫升 $(\mathrm{mg} / 100\mathrm{mL})$

$V_{1}$ ——试样水解液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——试样酶解时吸取的水解液体积，单位为毫升 $(\mathrm{mL})$ ；

$\frac{100}{10^6}$ ——换算系数。

结果如以泛酸钙计量，应乘以1.087。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

一般食品在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ ；营养素补充剂和强化食品在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 第二法 高效液相色谱法

# 9 原理

利用泛酸易溶于水，在弱酸性至中性条件下 $(\mathrm{pH}5.0\sim 7.0)$ 稳定的理化性质，试样用热水在超声波振荡下提取，经 $\mathrm{C}_{18}$ 反相色谱柱分离，在紫外检测器检测 $200~\mathrm{nm}$ 波长处检测，根据色谱峰的保留时间及紫外光谱图定性，外标法定量，计算试样中泛酸含量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 盐酸（HCl）。  
10.1.2 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。  
10.1.3 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$ ：色谱纯。  
10.1.4 七水合硫酸锌 $(\mathrm{ZnSO}_4\cdot 7\mathrm{H}_2\mathrm{O})$ 。  
10.1.5 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
10.1.6 淀粉酶：酶活力 $\geqslant 1.5\mathrm{U / mg}$

# 10.2 试剂配制

10.2.1 盐酸 $(0.1\mathrm{mol} / \mathrm{L})$ ：吸取 $8.3\mathrm{mL}$ 盐酸至 $1000~\mathrm{mL}$ 容量瓶中，加水稀释定容至刻度，混匀。  
10.2.2 硫酸锌溶液 $(0.5\mathrm{mol} / \mathrm{L})$ ：称取 $14.4\mathrm{g}$ 七水合硫酸锌，加水溶解并定容至 $100~\mathrm{mL}$ 。

# 12.1 试样制备

# 12.2 试样测定液的制备

# 12.2.1 营养素补充剂类保健食品

准确称取或量取适量试样 $(m)$ ，精确至 $0.001\mathrm{g}$ ，一般固体试样 $0.2\mathrm{g} \sim 2\mathrm{g}$ ，液态试样 $10\mathrm{g} \sim 20\mathrm{g}$ ，置于 $50\mathrm{mL}$ 锥形瓶中，加入约 $30\mathrm{mL}40^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ 温水超声提取 $20\mathrm{min}$ ，用水定容至刻度 $(V)$ 。转入离心管 $3000\mathrm{r / min}$ 离心 $5\mathrm{min} \sim 10\mathrm{min}$ ，取上清液过 $0.45\mu \mathrm{m}$ 滤膜，滤液待上机测定。

# 12.2.2 配方食品

准确称取适量试样 $(m)$ ，精确至 $0.001\mathrm{g}$ ，一般固态试样约 $5\mathrm{g}$ ，液态试样约 $20\mathrm{g}$ 。置于 $100\mathrm{mL}$ 锥形瓶中，加入 $40^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ 温水至 $30\mathrm{mL}$ 。如果试样中含有淀粉，加入淀粉酶 $0.2\mathrm{g}$ ，振摇混匀，盖上瓶塞，在 $55^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 水浴条件下，振摇酶解 $120\mathrm{min} \sim 240\mathrm{min}$ 。如果试样不含淀粉，直接超声提取 $20\mathrm{min}$ 。

取出试样液，冷却至室温，用 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸调节 $\mathrm{pH}$ 至 $5.0\pm 0.1$ ，加入 $5\mathrm{mL}0.5\mathrm{mol} / \mathrm{L}$ 硫酸锌溶液，充分混合。转入 $50~\mathrm{mL}$ 容量瓶中，用水定容至刻度并充分混匀后，转入离心管 $3000\mathrm{r / min}$ 离心

$5\mathrm{min}\sim 10\mathrm{min}$ ，取上清液过 $0.45\mu \mathrm{m}$ 滤膜，滤液待上机测定。

# 12.4 参考液相色谱条件

12.4.1 色谱柱： $\mathrm{ODS - C_{18}}$ （粒径 $5\mu \mathrm{m}$ ， $250\mathrm{mm} \times 4.6\mathrm{mm}$ ）或具有同等性能的色谱柱。

12.4.3 紫外检测器：检测波长： $210~\mathrm{nm}$   
12.4.4 流动相： $0.02\mathrm{mol / L}$ 磷酸二氢钾溶液 $+$ 乙腈 $= 95 + 5$   
12.4.5 流速： $1.0~\mathrm{mL / min}$   
12.4.6 进样量： $10~\mu \mathrm{L}$ 或 $20~\mu \mathrm{L}$

# 12.5.1 标准曲线测定

按照已经确立的色谱条件，将泛酸标准工作液依次进行色谱分析(标准色谱图见附录B)，记录标准出峰时间、色谱峰高(或峰面积)。以标准工作液浓度为横坐标，峰高(或峰面积)为纵坐标，绘制标准曲线。

# 12.5.2 试样溶液的测定

取试样测定液按照色谱条件进行色谱分析，根据试样峰高(或峰面积)从标准曲线中查得相应的泛酸浓度 $(\rho)$ 。

$$

式中：

X ——试样中泛酸含量,固态试样单位为毫克每百克 $(\mathrm{mg} / 100 \mathrm{~g})$ , 液态试样为毫克每百毫升 $(\mathrm{mg} / 100 \mathrm{~mL})$ ;

$\rho$ ——从标准曲线上查得试样测定液中泛酸的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样测定液总体积，单位为毫升 $(\mathrm{mL})$ ；

$F$ ——试样测定液稀释倍数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$\frac{100}{1000}$ ——换算系数。

A.2.6 腺嘌呤-鸟嘌呤-尿嘧啶溶液：分别称取硫酸腺嘌呤、盐酸鸟嘌呤和尿嘧啶各 $0.1\mathrm{g}$ 于 $250~\mathrm{mL}$ 烧杯中，加 $75~\mathrm{mL}$ 水和 $2\mathrm{mL}$ 盐酸，加热使其完全溶解后冷却。若有沉淀产生，再加盐酸数滴，加热，如此反复直至冷却后无沉淀产生为止。用水稀释至 $100~\mathrm{mL}$ ，加3滴～5滴甲苯，贮存于棕色试剂瓶中，置 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱中可保存1年。  
A.2.7 胱氨酸-色氨酸溶液：分别称取 $4\mathrm{gL}$ -胱氨酸和 $1\mathrm{gL}$ -色氨酸或 $2\mathrm{gL}$ -色氨酸于 $800~\mathrm{mL}$ 水中，加热至 $70^{\circ}C\sim 80^{\circ}C$ ，逐滴加入 $3\mathrm{mol / L}$ 盐酸溶液，不断搅拌，直至完全溶解为止。冷却至室温后加水稀释至 $1000~\mathrm{mL}$ 。加3滴 $\sim 5$ 滴甲苯，贮存于棕色试剂瓶中，于 $2^{\circ}C\sim 4^{\circ}C$ 冰箱中可保存1年。  
A.2.8 维生素液I：分别称取 $20\mathrm{mg}$ 核黄素和 $10\mathrm{mg}$ 盐酸硫胺素，加入 $1.0\mathrm{mL}$ 生物素溶液 $(40~\mu \mathrm{g / mL})$ ，用 $0.02\mathrm{mol / L}$ 乙酸溶液溶解并定容至 $1000~\mathrm{mL}$ 。加入3滴～5滴甲苯，贮存于棕色试剂瓶中， $2^{\circ}C\sim 4^{\circ}C$ 冰箱可保存1年。  
A.2.9 维生素液Ⅱ：分别称取 $10\mathrm{mg}$ 对氨基苯甲酸、 $50\mathrm{mg}$ 尼克酸和 $40\mathrm{mg}$ 盐酸吡哆醇，溶于 $1000\mathrm{mL}$ 乙醇溶液中。加入3滴～5滴甲苯，贮存于棕色试剂瓶中， $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱可保存1年。  
A.2.10 聚山梨酯-80溶液：将 $25\mathrm{g}$ 聚山梨酯-80用乙醇溶解并稀释至 $250~\mathrm{mL}$ 。 $2^{\circ}C\sim 4^{\circ}C$ 冰箱可保存1年。  
A.2.11 泛酸测定用培养液：配制 $1000~\mathrm{mL}$ 培养液，按表A.1吸取液体试剂，混合后加水 $300~\mathrm{mL}$ ，依次加入固体试剂，煮沸搅拌 $2\mathrm{min}$ 。用 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液、 $1\mathrm{mol} / \mathrm{L}$ 盐酸溶液调节基础培养液 $\mathrm{pH}$ 至 $6.8 \pm 0.1$ 。加入乙盐溶液 $20~\mathrm{mL}$ ，用水补至 $1000~\mathrm{mL}$ 。配制时可根据培养液用量按比例增减。临用现配。

# 食品安全国家标准

# 食品中香兰素、甲基香兰素、乙基香兰素和香豆素的测定

# 1 范围

本标准规定了食品中香兰素、甲基香兰素、乙基香兰素和香豆素的测定方法。

本标准适用于婴幼儿配方食品、婴幼儿辅助食品、糕点、糖果、乳及乳制品、饮料、小麦粉中香兰素、甲基香兰素、乙基香兰素和香豆素的测定。

# 第一法 液相色谱法

# 2 原理

试样加水混匀后，加入乙腈超声提取，经液相色谱分离，紫外检测器或二极管阵列检测器检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂与材料

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 甲酸（HCOOH）：色谱纯。  
3.1.3 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.4 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.5 浓盐酸（HCl）： $12\mathrm{mol} / \mathrm{L}$ 。  
3.1.6 微孔滤膜： $0.45\mu \mathrm{m}$ ，有机相型。

# 3.2 试剂配制

3.2.1 $0.1\%$ 甲酸甲醇溶液：吸取 $1\mathrm{mL}$ 甲酸溶于甲醇并稀释至 $1000~\mathrm{mL}$ ，临用现配。  
3.2.2 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：吸取 $8.33~\mathrm{mL}$ 浓盐酸，溶于水并稀释至 $100~\mathrm{mL}$ ，临用现配。  
3.2.3 甲醇-水溶液 $(4 + 1)$ ：将甲醇和水按 $4:1$ （体积比）混合均匀。  
3.2.4 $0.5\%$ 甲酸溶液：吸取 $5\mathrm{mL}$ 甲酸，溶于水并稀释至 $1000~\mathrm{mL}$ ，临用现配。

# 3.3 标准品

3.3.1 香兰素标准品 $\left(\mathrm{C}_{8} \mathrm{H}_{8} \mathrm{O}_{3}, \mathrm{CAS}\right.$ 号：121-33-5）：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。

3.3.2 甲基香兰素标准品 $\mathrm{(C_9H_{10}O_3}$ ,CAS号：120-14-9)：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.3 乙基香兰素标准品 $\mathrm{(C_9H_{10}O_3}$ ,CAS号：121-32-4)：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.4 香豆素标准品 $\mathrm{(C_9H_6O_2}$ ，CAS号：91-64-5）：纯度 $\geq 97\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 标准储备液 $(1000\mathrm{mg / L})$ ：准确称取香兰素、甲基香兰素、乙基香兰素、香豆素各 $10\mathrm{mg}$ （精确至 $0.1\mathrm{mg})$ ，分别置于 $10~\mathrm{mL}$ 容量瓶中，用 $0.1\%$ 甲酸甲醇溶液溶解并定容至刻度，混匀。将溶液转移至棕色玻璃容器内，一 $18^{\circ}C$ 下避光保存，保存期8个月。  
3.4.2 标准混合中间液 $(10\mathrm{mg / L})$ ：吸取香兰素、甲基香兰素、乙基香兰素、香豆素的标准储备液 $(1000\mathrm{mg / L})$ 各 $1.00~\mathrm{mL}$ ，置于 $100~\mathrm{mL}$ 容量瓶中，加 $0.1\%$ 甲酸甲醇定容至刻度，混匀。将溶液转移至棕色玻璃容器中，一 $18^{\circ}C$ 下避光保存，保存期3个月。  
3.4.3 标准混合系列工作液：分别吸取标准混合中间液 $(10\mathrm{mg / L})0.2\mathrm{mL},1.0\mathrm{mL},5.0\mathrm{mL}$ ，置于 $10~\mathrm{mL}$ 容量瓶中，另分别吸取香兰素、甲基香兰素、乙基香兰素、香豆素的标准储备液 $(1000\mathrm{mg / L})0.2\mathrm{mL}$ 产 $0.5\mathrm{mL}$ 和 $1\mathrm{mL}$ ，置于 $10~\mathrm{mL}$ 容量瓶中，加甲醇-水溶液定容至刻度，混匀。标准混合系列工作液的浓度分别为 $0.2\mathrm{mg / L},1.0\mathrm{mg / L},5.0\mathrm{mg / L},20.0\mathrm{mg / L},50.0\mathrm{mg / L},100.0\mathrm{mg / L}$ ，临用现配。

# 4 仪器和设备

4.1 液相色谱仪：配有二极管阵列或紫外检测器。  
4.2 天平：感量分别为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$   
4.3 涡旋混合器。  
4.4 离心机。  
4.5 氮气浓缩装置。  
4.6 超声波发生器。

# 5.3 标准曲线的制作

将标准混合系列工作液分别注入液相色谱仪中，测定相应的峰面积，以标准系列工作液中香兰素、甲基香兰素、乙基香兰素和香豆素的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。香兰素、甲基香兰素、乙基香兰素、香豆素标准溶液的色谱图参见附录A中图A.1。

# 5.4 试样溶液的测定

将试样溶液注入液相色谱仪中，得到峰面积，根据标准曲线得到试样溶液中被测化合物的浓度。

# 6 分析结果的表述

试样中被测化合物的含量按式(1)计算。

$$
X _ {i} = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X_{i}$ ——试样中被测化合物的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中被测化合物的浓度, 单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ 。

1000——换算系数。

注：计算结果需扣除空白值，保留小数点后两位数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

本方法的检出限均为 $0.07\mathrm{mg / kg}$ ，定量限均为 $0.20~\mathrm{mg / kg}$

# 9 原理

# 13 分析结果的表述

试样中被测化合物的含量按式(2)计算。

$$
X _ {i} = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X_{i}$ ——试样溶液中被测化合物的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中被测化合物的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ 。

1000——换算系数。

注：计算结果需扣除空白值，保留小数点后两位数字。

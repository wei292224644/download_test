# 前言

本标准代替GB/T5009.125—2003《尼龙6树脂及成型品中己内酰胺的测定》、GB/T23296.20—2009《食品接触材料 高分子材料 食品模拟物中己内酰胺及己内酰胺盐的测定 气相色谱法》和SN/T2283—2009《食品接触材料 高分子材料 食品模拟物中己内酰胺及己内酰胺盐的测定 气相色谱法》。

本标准与GB/T5009.125—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 己内酰胺的测定和迁移量的测定”；

——增加了己内酰胺的测定方法。

# 食品安全国家标准

# 食品接触材料及制品

# 己内酰胺的测定和迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中已内酰胺的测定方法和迁移量的测定方法。

本标准适用于食品接触材料及制品中己内酰胺的测定和迁移量的测定。

# 己内酰胺的测定

# 2 原理

试样经水提取后，己内酰胺溶解在提取液中，经滤膜过滤后用配有紫外检测器的高效液相色谱仪进行检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

乙腈 $\mathrm{(C_2H_3N}$ ，CAS号：75-05-8）：色谱纯。

# 3.2 标准品

3.2.1 己内酰胺 $\left(\mathrm{C}_{6} \mathrm{H}_{11} \mathrm{NO}, \mathrm{CAS}\right.$ 号：105-60-2）：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.3 标准溶液配制

# 3.3.1 己内酰胺标准储备液

称取 $100\mathrm{mg}$ (精确至 $0.0001\mathrm{g}$ ) 已内酰胺, 用水溶解后, 定容至 $100\mathrm{mL}$ , 配制成浓度为 $1000\mathrm{mg/L}$ 的储备液。溶液在 $4^{\circ}\mathrm{C}$ 下避光密封储存, 有效期为 1 个月。

# 3.3.2 己内酰胺中间标准溶液

用刻度吸量管吸取 $0.05\mathrm{mL}, 0.25\mathrm{mL}, 0.5\mathrm{mL}, 1.0\mathrm{mL}, 2.5\mathrm{mL}, 5.0\mathrm{mL}$ 己内酰胺标准储备液于6个 $10\mathrm{mL}$ 容量瓶中，用水定容，得到己内酰胺浓度为 $5.0\mathrm{mg / L}, 25.0\mathrm{mg / L}, 50.0\mathrm{mg / L}, 100.0\mathrm{mg / L}, 250.0\mathrm{mg / L}, 500.0\mathrm{mg / L}$ 的中间标准溶液。溶液储存方式同3.3.1。

# 3.3.3 己内酰胺标准工作溶液

分别准确吸取 $1.0\mathrm{mL}$ 己内酰胺中间标准溶液于6支 $10\mathrm{mL}$ 具塞玻璃试管中，分别加入 $4.0\mathrm{mL}$ 水

混匀，获得己内酰胺标准工作溶液。其中，己内酰胺浓度分别为 $1.0\mathrm{mg / L},5.0\mathrm{mg / L},10.0\mathrm{mg / L}$ 、 $20.0\mathrm{mg / L},50.0\mathrm{mg / L},100.0\mathrm{mg / L}$ 。溶液储存方式同3.3.1。

# 4 仪器和设备

4.1 高效液相色谱仪：配备紫外检测器（UV）。  
4.2 冷冻研磨仪。  
4.3 分析天平：感量 $0.0001\mathrm{g}$   
4.4 具塞玻璃试管： $10~\mathrm{mL}$   
4.5 $0.45\mu \mathrm{m}$ 微孔滤膜。  
4.6 恒温水浴锅。

# 5 分析步骤

# 5.1 试样处理

先将试样用冷冻研磨仪或剪刀等其他切割工具将其破碎成粒径小于 $1\mathrm{mm}\times 1\mathrm{mm}$ 后再称量。切割样品时，不可使其发热变软。

# 5.2 试样溶液的制备

称取粉碎样品 $1.0\mathrm{g}$ （精确到 $0.0001\mathrm{g}$ ）于具塞玻璃试管中，加入 $10~\mathrm{mL}$ 水后在沸水浴中提取 $40\mathrm{min}$ ，放冷至室温，然后取上层清液置于 $25~\mathrm{mL}$ 容量瓶中。再用 $10~\mathrm{mL}$ 水按照上述方式提取1次，并合并两次清液后，用水定容至刻度，取 $1\mathrm{mL}$ 提取液通过 $0.45\mu \mathrm{m}$ 滤膜过滤后供高效液相色谱进样。若样品溶液浓度超出线性范围，使用水适当稀释，使其浓度处于线性范围内。

# 5.3 空白溶液的制备

除不加试样外，采用与5.2完全相同的分析步骤、试剂和用量。

# 5.4 色谱参考条件

5.4.1 色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6~\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ 。  
5.4.2 检测器：紫外检测波长 $210~\mathrm{nm}$   
5.4.3 流动相：乙腈-水 $(20 + 80)$ 。  
5.4.4 流速： $1.0~\mathrm{mL / min}$   
5.4.5 进样体积： $10~\mu \mathrm{L}$

# 5.5 标准曲线的制作

按照5.4所列测定条件，分别将己内酰胺标准工作溶液进液相色谱仪测定。以标准工作溶液中己内酰胺浓度为横坐标，单位为毫克每升 $(\mathrm{mg / L})$ ，以对应己内酰胺峰面积为纵坐标，制作标准工作曲线。标准色谱图参见图A.1。

# 5.6 试样溶液的测定

按照5.4所列条件，将试样溶液(5.2)及空白溶液(5.3)进液相色谱仪测定，得到己内酰胺峰面积，扣除空白值。

# 6 分析结果的表述

试样中己内酰胺含量按式(1)计算：

$$
X = \frac {\rho \times V \times f \times 1 0 ^ {- 3}}{m \times 1 0 ^ {- 3}} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中已内酰胺的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——依据校准曲线获得的试样溶液中已内酰胺的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——浓度稀释因子；

$10^{-3}$ ——单位换算因子；

$m$ ——试样称样量，单位为克 $(\mathrm{g})$ 。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过其算术平均值的 $10\%$ 。

# 8 其他

方法检出限为 $10\mathrm{mg / kg}$ 、方法定量限为 $25~\mathrm{mg / kg}$

# 己内酰胺迁移量的测定

# 9 原理

样品经浸泡后，水基、酒精、酸性食品模拟物中的己内酰胺直接通过高效液相色谱分离，采用紫外检测器进行测定，外标法定量；油基食品模拟物中己内酰胺通过乙醇-水溶液萃取后测定。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 乙腈 $\mathrm{(C_2H_3N}$ ,CAS号：75-05-8)：色谱纯。  
10.1.2 乙醇 $\mathrm{(C_2H_6O}$ ，CAS号：64-17-5）。  
10.1.3 正己烷 $\mathrm{(C_6H_{14}}$ ，CAS号：110-54-3）。  
10.1.4 水基、酸性、酒精、油基食品模拟物：所用试剂依据GB31604.1规定。

# 10.2 试剂配制

10.2.1 乙醇-水溶液 $(1 + 2)$ ：量取 $100\mathrm{mL}$ 乙醇和 $200~\mathrm{mL}$ 水，混匀。  
10.2.2 水基、酸性、酒精、油基食品模拟物：按GB5009.156操作。

# 10.3 标准品

10.3.1 己内酰胺 $\mathrm{(C_6H_{11}NO, CAS}$ 号：105-60-2）：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 10.4 标准溶液配制

# 10.4.1 己内酰胺标准储备液

称取 $100\mathrm{mg}$ (精确至 $0.0001\mathrm{g}$ ) 已内酰胺, 用水溶解后, 定容至 $100\mathrm{mL}$ , 配制成浓度为 $1000\mathrm{mg/L}$ 的储备液。溶液在 $4^{\circ}\mathrm{C}$ 下避光密封储存, 有效期为 1 个月。

# 10.4.2 己内酰胺标准溶液

用刻度吸量管吸取 $0.05\mathrm{mL}, 0.25\mathrm{mL}, 0.5\mathrm{mL}, 1.0\mathrm{mL}, 2.5\mathrm{mL}, 5.0\mathrm{mL}$ 己内酰胺储备液于 $10\mathrm{mL}$ 容量瓶中，用水定容，得到己内酰胺浓度分别为 $5.0\mathrm{mg / L}, 25.0\mathrm{mg / L}, 50.0\mathrm{mg / L}, 100.0\mathrm{mg / L}, 250.0\mathrm{mg / L}, 500.0\mathrm{mg / L}$ 的标准溶液。溶液储存方式同10.4.1。

# 10.4.3 水基、酸性、酒精类食品模拟物标准工作溶液

用刻度吸量管吸取 $1.0\mathrm{mL}$ 己内酰胺标准溶液于6支 $10~\mathrm{mL}$ 具塞玻璃试管中，分别加入 $4.0~\mathrm{mL}$ 水基食品模拟物，混匀，获得水基食品模拟物标准工作溶液。其中，己内酰胺浓度分别为 $1.0~\mathrm{mg / L}$ 、 $5.0~\mathrm{mg / L},10.0~\mathrm{mg / L},20.0~\mathrm{mg / L},50.0~\mathrm{mg / L},100.0~\mathrm{mg / L}$ 。采用同样方式，分别用酸性、酒精类食品模拟物配制同样浓度的己内酰胺标准工作溶液。

# 10.4.4 油基食品模拟物标准工作溶液

分别称取 $8.0\mathrm{g}$ 橄榄油（精确到 $0.01\mathrm{g}$ ），置于6个分液漏斗中，分别加入 $1.6\mathrm{mL}$ 己内酰胺标准溶液，混匀，使得油基模拟物中己内酰胺浓度为 $1.0\mathrm{mg/kg}$ 、 $5.0\mathrm{mg/kg}$ 、 $10.0\mathrm{mg/kg}$ 、 $20.0\mathrm{mg/kg}$ 、 $50.0\mathrm{mg/kg}$ 、 $100.0\mathrm{mg/kg}$ 。然后加入 $15\mathrm{mL}$ 正己烷，混匀，加入 $8\mathrm{mL}$ 乙醇-水溶液 $(1+2)$ ，振荡 $10\mathrm{min}$ ，静置 $30\mathrm{min}$ 使两相分层，移取 $5\mathrm{mL}$ 下层水溶液，经脱脂棉与 $0.45\mu\mathrm{m}$ 微孔滤膜过滤后待测。

# 11 仪器和设备

11.1 高效液相色谱法：配备紫外检测器（UV）。  
11.2 分析天平：感量 $0.0001\mathrm{g}$   
11.3 具塞玻璃试管： $10~\mathrm{mL}$   
11.4 $0.45\mu \mathrm{m}$ 微孔滤膜。  
11.5 恒温水浴锅。

# 12 分析步骤

# 12.1 试样迁移试验

按照GB5009.156及GB31604.1的要求，对样品进行迁移试验，得到食品模拟物试液。如果得到的食品模拟物试液不能马上进行下一步试验，应将食品模拟物试液于 $4^{\circ}\mathrm{C}$ 冰箱中避光保存。

所得食品模拟物试液应冷却或恢复至室温后进行下一步试验。

# 12.2 试液制备

# 12.2.1 水基、酸性、酒精类食品模拟物的处理

用刻度吸量管吸取迁移试验中得到的水基、酸性、酒精类食品模拟物约 $1\mathrm{mL}$ ，通过 $0.45\mu \mathrm{m}$ 滤膜过滤后待测。

# 12.2.2 油基食品模拟物的处理

称取迁移试验中得到的橄榄油 $8.0\mathrm{g}$ （精确到 $0.01\mathrm{g}$ )置于分液漏斗中，加入 $1.6~\mathrm{mL}$ 水，混匀，加入 $15~\mathrm{mL}$ 正己烷，混匀，加入 $8\mathrm{mL}$ 乙醇-水溶液 $(1 + 2)$ ，振荡 $10\mathrm{min}$ ，静置 $30\mathrm{min}$ 使两相分层，移取 $5\mathrm{mL}$ 下层水溶液，经脱脂棉与 $0.45\mu \mathrm{m}$ 微孔滤膜过滤后待测。

# 12.3 空白试液的制备

除不加试样外，采用与12.2完全相同的分析步骤、试剂和用量。

# 12.4 色谱参考条件

12.4.1 色谱柱： $\mathrm{C_{18}}$ ，柱长 $250 \mathrm{~mm}$ 、内径 $4.6 \mathrm{~mm}$ 、粒径 $5 \mu \mathrm{m}$ 。  
12.4.2 检测器：紫外检测波长 $210~\mathrm{nm}$   
12.4.3 流动相：乙腈-水 $(20 + 80)$   
12.4.4 流速： $1.0~\mathrm{mL / min}$   
12.4.5 进样体积： $10~\mu \mathrm{L}$

# 12.5 标准曲线的制作

按照12.4所列条件，分别将水基、酸性、酒精类食品模拟物标准工作溶液、油基食品模拟物标准工作溶液进液相色谱仪测定。以食品模拟物中己内酰胺浓度为横坐标，单位为“毫克每升 $(\mathrm{mg / L}$ ，水基、酸性、酒精类食品模拟物)或毫克每千克 $(\mathrm{mg / kg}$ ，油基食品模拟物）”，以己内酰胺峰面积为纵坐标，绘制标准工作曲线。

# 12.6 试样溶液的测定

按12.4所列条件，分别将模拟物溶液(12.2)、空白溶液(12.3)依次进液相色谱仪测定，得到目标物峰面积，扣除空白值。

# 13 分析结果的表述

由标准曲线得到试样溶液中己内酰胺浓度，按GB5009.156进行迁移量的计算，得到食品接触材料及制品中己内酰胺的迁移量。计算结果保留两位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过其算术平均值 $10\%$ 。

# 15 其他

水基、酸性、酒精类食品模拟物中己内酰胺的方法检出限为 $0.4\mathrm{mg / L}$ 、方法定量限为 $1.0~\mathrm{mg / L}$ ；油基食品模拟物中己内酰胺的方法检出限为 $0.4\mathrm{mg / kg}$ 、方法定量限为 $1.0~\mathrm{mg / kg}$ 。

# 前言

本标准代替GB25592—2010《食品安全国家标准 食品添加剂 硫酸铝铵》。

本标准与GB25592—2010相比，主要变化如下：

——修改了范围；  
——增加了硫酸铝铵 $\left[\mathrm{NH}_{4} \mathrm{Al}\left(\mathrm{SO}_{4}\right)_{2}\right]$ 的技术要求；  
——修改了铅指标值；  
——删除了重金属、水不溶物指标；  
——增加了干燥减量的检验方法；  
——将“氟化物（以F计）”修改为“氟(F)”;   
——将氟的检验方法修改为GB/T5009.18；  
——将铅的检验方法修改为GB5009.12或GB5009.75；  
——将砷的检验方法修改为GB5009.76或GB5009.11。

# 食品添加剂 硫酸铝铵

# 1 范围

本标准适用于以硫酸铝和硫酸铵为原料生产的食品添加剂硫酸铝铵。

# 2 分子式和相对分子质量

# 2.1 分子式

十二水合硫酸铝铵： $\mathrm{NH_4Al(SO_4)_2\cdot 12H_2O}$

硫酸铝铵： $\mathrm{NH_4Al(SO_4)_2}$

# 2.2 相对分子质量

十二水合硫酸铝铵：453.31（按2018年国际相对原子质量）

硫酸铝铵：237.13（按2018年国际相对原子质量）

# 检验方法

警示：本检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应小心谨慎！必要时，应在通风橱中进行。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603之规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 氢氧化钠溶液： $100\mathrm{g / L}$   
A.2.1.2 氯化钡溶液： $50~\mathrm{g / L}$   
A.2.1.3 盐酸溶液： $1 + 4$   
A.2.1.4 硝酸亚汞试纸：将滤纸浸入 $50~\mathrm{g / L}$ 硝酸亚汞溶液内，取出晾干。  
A.2.1.5 石蕊试纸。

# A.2.2 鉴别方法

# A.2.2.1 铝盐的鉴别

取试样水溶液，加氢氧化钠溶液，即发生白色的胶状沉淀。沉淀能在过量的氢氧化钠溶液中溶解。

# A.2.2.2 硫酸盐的鉴别

取试样水溶液，加氯化钡溶液，即发生白色沉淀。加盐酸溶液，沉淀不溶解。

# A.2.2.3 铵盐的鉴别

取试样水溶液于表面皿上，加一定量的氢氧化钠溶液后，再盖上贴有湿润的红色石蕊试纸的表面皿，能观察到石蕊试纸变蓝色。

# A.3 硫酸铝铵（以干基计）含量的测定

# A.3.1 方法提要

在弱酸性介质中，EDTA与铝形成络合物，用硝酸铅返滴定过量的EDTA，从而确定硫酸铝铵的含量。

# A.3.2 试剂和材料

A.3.2.1 盐酸溶液： $1 + 4$   
A.3.2.2 氨水溶液： $1 + 1$

A.3.2.3 乙酸-乙酸钠缓冲溶液： $\mathrm{pH}\approx 6$   
A.3.2.4 乙二胺四乙酸二钠(EDTA)标准滴定溶液： $c(\mathrm{EDTA}) = 0.05\mathrm{mol / L}$   
A.3.2.5 硝酸铅标准滴定溶液： $c[\mathrm{Pb}(\mathrm{NO}_3)_2] = 0.05\mathrm{mol / L}$   
A.3.2.6 刚果红试纸。  
A.3.2.7 二甲酚橙指示液： $2\mathrm{g / L}$

# A.3.3 分析步骤

称取约 $5\mathrm{g}$ 试样（十二水合硫酸铝铵按A.4.2研磨并经干燥；硫酸铝铵按A.5.2处理），精确至 $0.0002\mathrm{g}$ ，置于 $150~\mathrm{mL}$ 烧杯中，加入 $80~\mathrm{mL}$ 水，加热溶解，冷却后移入 $250~\mathrm{mL}$ 容量瓶中，加10滴盐酸溶液，用水稀释至刻度，摇匀(混浊时可过滤，弃去初始滤液)。此溶液为试样溶液A，用于硫酸铝铵含量的测定。

用移液管移取 $25~\mathrm{mL}$ 试样溶液A，置于 $250~\mathrm{mL}$ 锥形瓶中，再用移液管加入 $50~\mathrm{mL}$ 乙二胺四乙酸二钠标准滴定溶液。放入一小块刚果红试纸，用氨水溶液调至试纸呈紫红色 $(\mathrm{pH}\approx 6)$ ，加 $15~\mathrm{mL}$ 乙酸-乙酸钠缓冲溶液，煮沸 $3\mathrm{min}$ 。冷却后加3滴～4滴二甲酚橙指示液，用硝酸铅标准滴定溶液滴定至橙黄色为终点。

同时做空白试验。

空白试验除不加试样外，其他操作及加入试剂的种类和量(标准滴定溶液除外)与测定试验相同。

# A.3.4 结果计算

十二水合硫酸铝铵 $\left[\mathrm{AlNH}_{4}\left(\mathrm{SO}_{4}\right)_{2} \cdot 12 \mathrm{H}_{2} \mathrm{O}\right]$ 的质量分数 $\omega_{1}$ 按式(A.1)计算。

$$
w _ {1} = \frac {\left[ \left(V _ {0} - V\right) / 1000 \right] \times c \times M}{m \times (25 / 250)} \times 100 \% \dots \tag{A.1}
$$

式中：

$V_{0}$ ——滴定空白试样溶液所消耗的硝酸铅标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V$ ——滴定试样溶液所消耗的硝酸铅标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硝酸铅标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——十二水合硫酸铝铵 $\left[\mathrm{AlNH}_{4}\left(\mathrm{SO}_{4}\right)_{2} \cdot 12 \mathrm{H}_{2} \mathrm{O}\right]$ 的摩尔质量的数值，单位为克每摩尔（ $\mathrm{g/mol}$ ） $(M = 453.33)$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

25 ——移取试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

250——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

硫酸铝铵 $\mathrm{AlNH_4(SO_4)_2}$ 的质量分数 $\varpi_{2}$ 按式(A.2)计算。

$$
w _ {2} = \frac {\left[ \left(V _ {1} - V _ {2}\right) / 1000 \right] \times c \times M _ {1}}{m _ {1} \times (25 / 250)} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$V_{1}$ ——滴定空白试验溶液所消耗的硝酸铅标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴定试样溶液所消耗的硝酸铅标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硝酸铅标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{1}$ ——硫酸铝铵 $\left[\mathrm{AlNH}_{4}(\mathrm{SO}_{4})_{2}\right]$ 的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})(M = 237.14)$

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

25 ——移取试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

250——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差

值不大于 $0.2\%$ 。

# A.4 水分的测定

# A.4.1 仪器和设备

A.4.1.1 电热恒温干燥箱：控温范围为 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。  
A.4.1.2 称量瓶： $\phi 50\mathrm{mm}\times 30\mathrm{mm}$   
A.4.1.3 试验筛：R40/3系列 $\phi 200\mathrm{mm}\times 50\mathrm{mm} / 355\mu \mathrm{m},\phi 200\mathrm{mm}\times 50\mathrm{mm} / 250\mu \mathrm{m}$

# A.4.2 分析步骤

用已于 $105\mathrm{^\circ C}\pm 2\mathrm{^\circ C}$ 下干燥至质量恒定的称量瓶称量约 $1\mathrm{g}$ 研磨至通过孔径为 $250~\mu \mathrm{m}\sim 355~\mu \mathrm{m}$ 的试验筛的试样，精确至 $0.0002\mathrm{g}$ ，均匀摊开，放入硅胶干燥器内，干燥至质量恒定。

# A.4.3 结果计算

水分的质量分数 $w_{3}$ 按式(A.3)计算。

$$
w _ {3} = \frac {m _ {2} - m _ {3}}{m _ {2}} \times 100 \% \tag{A.3}
$$

式中：

$m_{2}$ ——试样干燥前的质量的数值，单位为克 $(\mathrm{g})$ ；

$m_{3}$ ——试样干燥后的质量的数值，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.05\%$ 。

# A.5 干燥减量的测定

# A.5.1 仪器和设备

A.5.1.1 电热恒温干燥箱：控温范围为 $200^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。

A.5.1.2 称量瓶： $\phi 50\mathrm{mm}\times 30\mathrm{mm}$

# A.5.2 分析步骤

称取约 $6.0\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于预先于 $200^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 的电热恒温干燥箱中干燥至质量恒定的称量瓶中，于 $200^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 下干燥 $4\mathrm{h}$ 。

# A.5.3 结果计算

干燥减量的质量分数 $\varpi_{4}$ 按式(A.4)计算。

$$
w _ {4} = \frac {m _ {4} - m _ {5}}{m _ {6}} \times 100 \% \tag{A.4}
$$

式中：

$m_{4}$ —称量瓶和试样干燥前质量，单位为克 $(\mathrm{g})$

$m_{5}$ —称量瓶和试样干燥后质量，单位为克 $(\mathrm{g})$

$m_{6}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差

值不大于 $0.5\%$ 。

# A.6 硒（Se）的测定

# A.6.1 试剂和材料

A.6.1.1 盐酸溶液： $c(\mathrm{HCl}) = 4\mathrm{mol / L}$   
A.6.1.2 氨水溶液： $1 + 2$   
A.6.1.3 盐酸羟胺。

A.6.1.4 环己烷。

A.6.1.5 硒标准溶液: $1 \mathrm{~mL}$ 溶液含有硒 $0.001 \mathrm{mg}$ 。移取 $10 \mathrm{~mL}$ 按 GB/T 602 配制的硒标准溶液, 置于 $1000 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度, 摇匀。

A.6.1.6 2,3-二氨基萘溶液：称取 $0.100\mathrm{g}2$ ，3-二氨基萘和 $0.500\mathrm{g}$ 盐酸羟胺，溶解在 $100\mathrm{mL}$ 盐酸溶液中 $(0.1\mathrm{mol / L})$ 。

# A.6.2 仪器和设备

分光光度计：带有 $1\mathrm{cm}$ 的比色皿。

# A.6.3 分析步骤

# A.6.3.1 标准溶液的制备

用移液管移取 $6\mathrm{mL}$ 础标准溶液，置于 $150~\mathrm{mL}$ 烧杯中，加 $25~\mathrm{mL}$ 盐酸溶液。

# A.6.3.2 试样溶液的制备

称取 $0.20\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $150~\mathrm{mL}$ 烧杯中，加 $25~\mathrm{mL}$ 盐酸溶液。

# A.6.3.3 测定

向上述两种溶液中分别加入 $5\mathrm{mL}$ 氨水溶液，用氨水溶液调节溶液的 $\mathrm{pH}$ 到 $2.0\pm 0.2$ ，用水稀释至 $60~\mathrm{mL}$ ，用 $10~\mathrm{mL}$ 水转移至分液漏斗中，加 $10~\mathrm{mL}$ 水，加 $0.2\mathrm{g}$ 盐酸羟胺，摇匀至溶解，立刻加 $5.0~\mathrm{mL}$ 2,3-二氨基萘溶液。塞上塞子，摇匀，在室温放置 $10\mathrm{min}$ 。加 $5.0~\mathrm{mL}$ 环己烷，用力摇 $2\mathrm{min}$ ，使其分层。去掉水相，离心分离除去痕量水，于 $380~\mathrm{nm}$ 处使用 $1\mathrm{cm}$ 比色皿测定每个萃取液的吸光度，试样溶液的吸光度不大于标准溶液的吸光度即为试样的硒含量不大于 $30.0~\mathrm{mg / kg}$

# A.7 铅(Pb)的测定

按GB5009.12或GB5009.75规定的方法进行测定，试验中所用水为GB/T6682规定的二级水。

# A.8 砷(As)的测定

按GB5009.76或GB5009.11规定的方法进行测定，试验中所用水为GB/T6682规定的二级水。

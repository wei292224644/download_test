# 前言

本标准代替GB/T5009.33-2008《食品中亚硝酸盐与硝酸盐的测定》。

本标准与GB/T5009.33-2008相比，主要变化如下：

——第一法中增加粉状婴幼儿配方食品的淋洗条件；

——删除第三法示波极谱法，增加“乳及乳制品中亚硝酸盐与硝酸盐的测定”作为第三法。

本标准所代替标准的历次版本发布情况为：

GB5009.33-1985、GB/T5009.33-1996、GB/T5009.33-2003、GB/T5009.33-2008。

# 食品安全国家标准

# 食品中亚硝酸盐与硝酸盐的测定

# 1 范围

本标准规定了食品中亚硝酸盐和硝酸盐的测定方法。

本标准适用于食品中亚硝酸盐和硝酸盐的测定。

# 第一法 离子色谱法

# 2 原理

试样经沉淀蛋白质、除去脂肪后，采用相应的方法提取和净化，以氢氧化钾溶液为淋洗液，阴离子交换柱分离，电导检测器检测。以保留时间定性，外标法定量。

# 3 试剂和材料

3.1 超纯水：电阻率 $>18.2 \mathrm{M} \Omega \cdot \mathrm{cm}$ 。  
3.2 乙酸（ $\mathrm{CH}_3\mathrm{COOH}$ ）：分析纯。  
3.3 氢氧化钾（KOH）：分析纯。  
3.4 乙酸溶液（ $3\%$ ）：量取乙酸（3.2） $3\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中，以水稀释至刻度，混匀。  
3.5 亚硝酸根离子 $\left(\mathrm{NO}_{2}^{-}\right)$ 标准溶液（ $100\mathrm{mg/L}$ ，水基体）。  
3.6硝酸根离子（ $\mathrm{NO}_3^-$ ）标准溶液（ $1000\mathrm{mg / L}$ ，水基体）。  
3.7 亚硝酸盐（以 $\mathrm{NO}_2^-$ 计，下同）和硝酸盐（以 $\mathrm{NO}_3^-$ 计，下同）混合标准使用液：准确移取亚硝酸根离子 $(\mathrm{NO}_2^-)$ 和硝酸根离子 $(\mathrm{NO}_3^-)$ 的标准溶液各 $1.0 \mathrm{~mL}$ 于 $100 \mathrm{~mL}$ 容量瓶中，用水稀释至刻度，此溶液每 $1 \mathrm{~L}$ 含亚硝酸根离子 $1.0 \mathrm{mg}$ 和硝酸根离子 $10.0 \mathrm{mg}$ 。

# 4 仪器和设备

4.1 离子色谱仪：包括电导检测器，配有抑制器，高容量阴离子交换柱， $50~\mu \mathrm{L}$ 定量环。  
4.2 食物粉碎机。  
4.3 超声波清洗器。  
4.4天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$

4.5 离心机：转速 $\geq 10000$ 转/分钟，配 $5\mathrm{mL}$ 或 $10\mathrm{mL}$ 离心管。   
4.6 $0.22\mu \mathrm{m}$ 水性滤膜针头滤器。  
4.7 净化柱：包括 $\mathrm{C_{18}}$ 柱、Ag柱和Na柱或等效柱。  
4.8 注射器： $1.0\mathrm{mL}$ 和 $2.5\mathrm{mL}$

注：所有玻璃器皿使用前均需依次用 $2\mathrm{mol} / \mathrm{L}$ 氢氧化钾和水分别浸泡 $4\mathrm{h}$ ，然后用水冲洗3次～5次，晾干备用。

# 5 分析步骤

# 5.1 试样预处理

5.1.1 新鲜蔬菜、水果：将试样用去离子水洗净，晾干后，取可食部切碎混匀。将切碎的样品用四分法取适量，用食物粉碎机制成匀浆备用。如需加水应记录加水量。  
5.1.2 肉类、蛋、水产及其制品：用四分法取适量或取全部，用食物粉碎机制成匀浆备用。  
5.1.3 乳粉、豆奶粉、婴儿配方粉等固态乳制品(不包括干酪)：将试样装入能够容纳2倍试样体积的带盖容器中，通过反复摇晃和颠倒容器使样品充分混匀直到使试样均一化。  
5.1.4 发酵乳、乳、炼乳及其他液体乳制品：通过搅拌或反复摇晃和颠倒容器使试样充分混匀。  
5.1.5 干酪：取适量的样品研磨成均匀的泥浆状。为避免水分损失，研磨过程中应避免产生过多的热量。

# 5.2 提取

5.2.1 水果、蔬菜、鱼类、肉类、蛋类及其制品等：称取试样匀浆 $5 \mathrm{~g}$ （精确至 $0.01 \mathrm{~g}$ ，可适当调整试样的取样量，以下相同），以 $80 \mathrm{~mL}$ 水洗入 $100 \mathrm{~mL}$ 容量瓶中，超声提取 $30 \mathrm{~min}$ ，每隔 $5 \mathrm{~min}$ 振摇一次，保持固相完全分散。于 $75^{\circ} \mathrm{C}$ 水浴中放置 $5 \mathrm{~min}$ ，取出放置至室温，加水稀释至刻度。溶液经滤纸过滤后，取部分溶液于 10000 转/分钟离心 $15 \mathrm{~min}$ ，上清液备用。  
5.2.2 腌鱼类、腌肉类及其它腌制品：称取试样匀浆 $2\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），以 $80~\mathrm{mL}$ 水洗入 $100~\mathrm{mL}$ 容量瓶中，超声提取 $30\mathrm{min}$ ，每 $5\mathrm{min}$ 振摇一次，保持固相完全分散。于 $75^{\circ}C$ 水浴中放置 $5\mathrm{min}$ ，取出放置至室温，加水稀释至刻度。溶液经滤纸过滤后，取部分溶液于10000转/分钟离心 $15\mathrm{min}$ ，上清液备用。  
5.2.3 乳：称取试样 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），置于 $100\mathrm{mL}$ 容量瓶中，加水 $80\mathrm{mL}$ ，摇匀，超声 $30\mathrm{min}$ ，加入 $3\%$ 乙酸溶液 $2\mathrm{mL}$ ，于 $4^{\circ}\mathrm{C}$ 放置 $20\mathrm{min}$ ，取出放置至室温，加水稀释至刻度。溶液经滤纸过滤，取上清液备用。  
5.2.4 乳粉：称取试样 $2.5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），置于 $100\mathrm{mL}$ 容量瓶中，加水 $80\mathrm{mL}$ ，摇匀，超声 $30\mathrm{min}$ ，加入 $3\%$ 乙酸溶液 $2\mathrm{mL}$ ，于 $4^{\circ}\mathrm{C}$ 放置 $20\mathrm{min}$ ，取出放置至室温，加水稀释至刻度。溶液经滤纸过滤，取上清液备用。  
5.2.5 取上述备用的上清液约 $15 \mathrm{~mL}$ , 通过 $0.22 \mu \mathrm{m}$ 水性滤膜针头滤器、 $\mathrm{C}_{18}$ 柱, 弃去前面 $3 \mathrm{~mL}$ (如果氯离子大于 $100 \mathrm{mg} / \mathrm{L}$ , 则需要依次通过针头滤器、 $\mathrm{C}_{18}$ 柱、 $\mathrm{Ag}$ 柱和 $\mathrm{Na}$ 柱, 弃去前面 $7 \mathrm{~mL}$ ), 收集后面洗脱液待测。

固相萃取柱使用前需进行活化，如使用OnGuard II RP柱（1.0mL）、OnGuard II Ag柱（1.0mL）和OnGuard II Na柱（1.0mL）<sup>1</sup>，其活化过程为：OnGuard II RP柱（1.0mL）使用前依次用10mL甲醇、15mL水通过，静置活化30min。OnGuard II Ag柱（1.0mL）和OnGuard II Na柱（1.0mL）用10

mL 水通过，静置活化 $30 \mathrm{~min}$ 。

# 5.3 参考色谱条件

5.3.1 色谱柱：氢氧化物选择性，可兼容梯度洗脱的高容量阴离子交换柱，如Dionex IonPac AS11-HC $4\mathrm{mm} \times 250\mathrm{mm}$ （带IonPac AG11-HC型保护柱 $4\mathrm{mm} \times 50\mathrm{mm}$ ）<sup>1</sup>，或性能相当的离子色谱柱。  
5.3.2 淋洗液   
5.3.2.1 一般试样：氢氧化钾溶液，浓度为 $6\mathrm{mmol / L}\sim 70\mathrm{mmol / L}$ ；洗脱梯度为 $6\mathrm{mmol / L}30\mathrm{min}$ ，70mmol/L5min， $6\mathrm{mmol / L}5\mathrm{min}$ ；流速 $1.0\mathrm{mL / min}$ 。  
5.3.2.2 粉状婴幼儿配方食品：氢氧化钾溶液，浓度为 $5\mathrm{mmol / L}\sim 50\mathrm{mmol / L}$ ；洗脱梯度为 $5\mathrm{mmol / L}33$ min， $50\mathrm{mmol / L}5\mathrm{min}$ ， $5\mathrm{mmol / L}5\mathrm{min}$ ；流速 $1.3~\mathrm{mL / min}$ 。  
5.3.3 抑制器：连续自动再生膜阴离子抑制器或等效抑制装置。  
5.3.4 检测器：电导检测器，检测池温度为 $35^{\circ}C$   
5.3.5 进样体积： $50~\mu \mathrm{L}$ （可根据试样中被测离子含量进行调整）。

# 5.4 测定

# 5.4.1 标准曲线

移取亚硝酸盐和硝酸盐混合标准使用液，加水稀释，制成系列标准溶液，含亚硝酸根离子浓度为 $0.00\mathrm{mg / L}$ 、 $0.02\mathrm{mg / L}$ 、 $0.04\mathrm{mg / L}$ 、 $0.06\mathrm{mg / L}$ 、 $0.08\mathrm{mg / L}$ 、 $0.10\mathrm{mg / L}$ 、 $0.15\mathrm{mg / L}$ 、 $0.20\mathrm{mg / L}$ ；硝酸根离子浓度为 $0.0\mathrm{mg / L}$ 、 $0.2\mathrm{mg / L}$ 、 $0.4\mathrm{mg / L}$ 、 $0.6\mathrm{mg / L}$ 、 $0.8\mathrm{mg / L}$ 、 $1.0\mathrm{mg / L}$ 、 $1.5\mathrm{mg / L}$ 、 $2.0\mathrm{mg / L}$ 的混合标准溶液，从低到高浓度依次进样。得到上述各浓度标准溶液的色谱图（图1）。以亚硝酸根离子或硝酸根离子的浓度（ $\mathrm{mg / L}$ ）为横坐标，以峰高（ $\mu S$ ）或峰面积为纵坐标，绘制标准曲线或计算线性回归方程。

# 5.4.2 样品测定

分别吸取空白和试样溶液 $50 \mu \mathrm{L}$ , 在相同工作条件下, 依次注入离子色谱仪中, 记录色谱图。根据保留时间定性, 分别测量空白和样品的峰高 ( $\mu \mathrm{S}$ ) 或峰面积。

# 6 分析结果的表述

试样中亚硝酸盐（以 $\mathrm{NO}_2$ 计）或硝酸盐（以 $\mathrm{NO}_3$ 计）含量按式（1）计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V \times f \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中亚硝酸根离子或硝酸根离子的含量, 单位为毫克每千克 $(\mathrm{mg} / \mathrm{kg})$ ;

$c$ ——测定用试样溶液中的亚硝酸根离子或硝酸根离子浓度, 单位为毫克每升 $(\mathrm{mg} / \mathrm{L})$ ;

$c_{0}$ ——试剂空白液中亚硝酸根离子或硝酸根离子的浓度, 单位为毫克每升 $(\mathrm{mg} / \mathrm{L})$ ;

$V$ ——试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样溶液稀释倍数；

$m$ ——试样取样量，单位为克（g）。

说明：试样中测得的亚硝酸根离子含量乘以换算系数1.5，即得亚硝酸盐（按亚硝酸钠计）含量；试样中测得的硝酸根离子含量乘以换算系数1.37，即得硝酸盐（按硝酸钠计）含量。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对值差不得超过算术平均值的 $10\%$ 。

# 第二法 分光光度法

# 8 原理

亚硝酸盐采用盐酸萘乙二胺法测定，硝酸盐采用镉柱还原法测定。

试样经沉淀蛋白质、除去脂肪后，在弱酸条件下亚硝酸盐与对氨基苯磺酸重氮化后，再与盐酸萘乙二胺偶合形成紫红色染料，外标法测得亚硝酸盐含量。采用镉柱将硝酸盐还原成亚硝酸盐，测得亚硝酸盐总量，由此总量减去亚硝酸盐含量，即得试样中硝酸盐含量。

# 10.6 镐柱

10.6.1 海绵状镉的制备：投入足够的锌皮或锌棒于 $500 \mathrm{~mL}$ 硫酸镉溶液 $(200 \mathrm{~g} / \mathrm{L})$ 中，经过 $3 \mathrm{~h} \sim 4 \mathrm{~h}$ ，当其中的镉全部被锌置换后，用玻璃棒轻轻刮下，取出残余锌棒，使镉沉底，倾去上层清液，以水用倾泻法多次洗涤，然后移入组织捣碎机中，加 $500 \mathrm{~mL}$ 水，捣碎约 $2 \mathrm{~s}$ ，用水将金属细粒洗至标准筛上，取 20 目～40 目之间的部分。  
10.6.2 镉柱的装填：如图 2。用水装满镉柱玻璃管，并装入 $2 \mathrm{~cm}$ 高的玻璃棉做垫，将玻璃棉压向柱底时，应将其中所包含的空气全部排出，在轻轻敲击下加入海绵状镉至 $8 \mathrm{~cm} \sim 10 \mathrm{~cm}$ 高，上面用 $1 \mathrm{~cm}$ 高的玻璃棉覆盖，上置一贮液漏斗，末端要穿过橡皮塞与镉柱玻璃管紧密连接。

如无上述镉柱玻璃管时，可以 $25\mathrm{mL}$ 酸式滴定管代用，但过柱时要注意始终保持液面在镉层之上。

当镉柱填装好后，先用 $25 \mathrm{~mL}$ 盐酸 $(0.1 \mathrm{~mol} / \mathrm{L})$ 洗涤，再以水洗两次，每次 $25 \mathrm{~mL}$ ，镉柱不用时用水封盖，随时都要保持水平面在镉层之上，不得使镉层夹有气泡。

# 12.1 亚硝酸盐含量计算

亚硝酸盐（以亚硝酸钠计）的含量按式（3）进行计算。

$$
X _ {1} = \frac {A _ {1} \times 1 0 0 0}{m \times \frac {V _ {1}}{V _ {0}} \times 1 0 0 0} \dots \tag {3}
$$

式中：

$X_{I}$ ——试样中亚硝酸钠的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A_{I}$ ——测定用样液中亚硝酸钠的质量, 单位为微克 $(\mu \mathrm{g})$ ;

$m$ ——试样质量，单位为克（g）；

$V_{I}$ ——测定用样液体积，单位为毫升（ $\mathrm{mL}$ ）；

$V_{0}$ ——试样处理液总体积，单位为毫升（ $\mathrm{mL}$ ）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

硝酸盐（以硝酸钠计）的含量按式（4）进行计算。

$$
X _ {2} = \left(\frac {A _ {2} \times 1 0 0 0}{m \times \frac {V _ {2}}{V _ {0}} \times \frac {V _ {4}}{V _ {3}} \times 1 0 0 0} - X _ {1}\right) \times 1. 2 3 2 \tag {4}
$$

式中：

$X_{2}$ ——试样中硝酸钠的含量，单位为毫克每千克 $(\mathrm{mg/kg})$ ；

$A_{2}$ ——经镉粉还原后测得总亚硝酸钠的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m$ ——试样的质量，单位为克（g）；

1.232——亚硝酸钠换算成硝酸钠的系数；

$V_{2}$ ——测总亚硝酸钠的测定用样液体积，单位为毫升（ $\mathrm{mL}$ ）；

$V_{0}$ ——试样处理液总体积，单位为毫升（ $\mathrm{mL}$ ）；

$V_{3}$ ——经镉柱还原后样液总体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{4}$ ——经镉柱还原后样液的测定用体积，单位为毫升（ $\mathrm{mL}$ ）；

$X_{I}$ ——由式（3）计算出的试样中亚硝酸钠的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 14 原理

试样经沉淀蛋白质、除去脂肪后，用镀铜镉粒使部分滤液中的硝酸盐还原为亚硝酸盐。在滤液和已还原的滤液中，加入磺胺和N-1-萘基-乙二胺二盐酸盐，使其显粉红色，然后用分光光度计在 $538\mathrm{nm}$ 波长下测其吸光度。

将测得的吸光度与亚硝酸钠标准系列溶液的吸光度进行比较，就可计算出样品中的亚硝酸盐含量和硝酸盐还原后的亚硝酸总量；从两者之间的差值可以计算出硝酸盐的含量。

# 17.2 检查柱的还原能力

每天至少要进行两次，一般在开始时和一系列测定之后。

17.2.1 用移液管将 $20~\mathrm{mL}$ 的硝酸钾标准溶液（15.14）移入还原柱顶部的贮液杯中，再立即向该贮液杯中添加 $5\mathrm{mL}$ 缓冲溶液(15.5)。用一个 $100~\mathrm{mL}$ 的容量瓶收集洗提液。洗提液的流量不应超过 $6\mathrm{mL / min}$ 。

17.2.3 当容量瓶中的洗提液接近 $100\mathrm{mL}$ 时，从柱子下取出容量瓶，用水定容至刻度，混合均匀。

17.3.3 检查镉柱的还原能力，如低于 $95\%$ ，要重复再生。

# 17.4样品的称取和溶解

17.4.1 液体乳样品：量取 $90~\mathrm{mL}$ 样品于 $500~\mathrm{mL}$ 锥形瓶中，用 $22~\mathrm{mL} 50^{\circ}\mathrm{C} \sim 55^{\circ}\mathrm{C}$ 的水分数次冲洗样品量筒，冲洗液倾入锥形瓶中，混匀。  
17.4.2 乳粉样品：在 $100\mathrm{mL}$ 烧杯中称取 $10\mathrm{g}$ 样品，准确至 $0.001\mathrm{g}$ 。用 $112\mathrm{mL}50^{\circ}\mathrm{C}\sim 55^{\circ}\mathrm{C}$ 的水将样品洗入 $500\mathrm{mL}$ 锥形瓶中，混匀。  
17.4.3 乳清粉及以乳清粉为原料生产的粉状婴幼儿配方食品样品：在 $100 \mathrm{~mL}$ 烧杯中称取 $10 \mathrm{~g}$ 样品，准确至 $0.001 \mathrm{~g}$ 。用 $112 \mathrm{~mL} 50^{\circ} \mathrm{C} \sim 55^{\circ} \mathrm{C}$ 的水将样品洗入 $500 \mathrm{~mL}$ 锥形瓶中，混匀。用铝箔纸盖好锥形瓶口，将溶好的样品在沸水中煮 $15 \mathrm{~min}$ ，然后冷却至约 $50^{\circ} \mathrm{C}$ 。

# 17.5 脂肪和蛋白的去除

17.5.1 按顺序加入 $24\mathrm{mL}$ 硫酸锌溶液（15.8.1）、 $24\mathrm{mL}$ 亚铁氰化钾溶液（15.8.2）和 $40\mathrm{mL}$ 缓冲溶液（15.5），加入时要边加边摇，每加完一种溶液都要充分摇匀。  
17.5.2 静置 $15\mathrm{min}\sim 1\mathrm{h}$ 。然后用滤纸（16.9）过滤，滤液用 $250~\mathrm{mL}$ 锥形瓶收集。

# 17.6硝酸盐还原为亚硝酸盐

17.6.1 移取 $20 \mathrm{~mL}$ 滤液于 $100 \mathrm{~mL}$ 小烧杯中, 加入 $5 \mathrm{~mL}$ 缓冲溶液 (15.5), 摇匀, 倒入镉柱顶部的贮液杯中, 以小于 $6 \mathrm{~mL} / \mathrm{min}$ 的流速过柱。洗提液（过柱后的液体）接入 $100 \mathrm{~mL}$ 容量瓶中。

17.6.2 当贮液杯快要排空时，用 $15 \mathrm{~mL}$ 水冲洗小烧杯，再倒入贮液杯中。冲洗水流完后，再用 $15 \mathrm{~mL}$ 水重复一次。当第二次冲洗水快流尽时，将贮液杯装满水，以最大流速过柱。  
17.6.3 当容量瓶中的洗提液接近 $100\mathrm{mL}$ 时，取出容量瓶，用水定容，混匀。

# 17.7 测定

17.7.1 分别移取 $20\mathrm{mL}$ 洗提液（17.6.3）和 $20\mathrm{mL}$ 滤液（17.5.2）于 $100\mathrm{mL}$ 容量瓶中，加水至约 $60\mathrm{mL}$ 。  
17.7.2 在每个容量瓶中先加入 $6 \mathrm{~mL}$ 显色液 1（15.10），边加边混；再加入 $5 \mathrm{~mL}$ 显色液 2（15.11）。小心混合溶液，使其在室温下静置 $5 \mathrm{~min}$ ，避免直射阳光。  
17.7.3 加入 $2 \mathrm{~mL}$ 显色液 3（15.12），小心混合，使其在室温下静置 $5 \mathrm{~min}$ ，避免直射阳光。用水定容至刻度，混匀。  
17.7.4 在 $15\mathrm{min}$ 内用 $538~\mathrm{nm}$ 波长，以空白试验液体为对照测定上述样品溶液的吸光度。

# 17.8 标准曲线的制作

17.8.1 分别移取（或用滴定管放出） $0\mathrm{mL}$ 、 $2\mathrm{mL}$ 、 $4\mathrm{mL}$ 、 $6\mathrm{mL}$ 、 $8\mathrm{mL}$ 、 $10\mathrm{mL}$ 、 $12\mathrm{mL}$ 、 $16\mathrm{mL}$ 和 $20\mathrm{mL}$ 亚硝酸钠标准溶液(15.13)于9个 $100\mathrm{mL}$ 容量瓶中。在每个容量瓶中加水，使其体积约为 $60\mathrm{mL}$ 。  
17.8.2 在每个容量瓶中先加入 $6 \mathrm{~mL}$ 显色液 1（15.10），边加边混；再加入 $5 \mathrm{~mL}$ 显色液 2（15.11）。小心混合溶液，使其在室温下静置 $5 \mathrm{~min}$ ，避免直射阳光。  
17.8.3 加入 $2 \mathrm{~mL}$ 显色液 3（15.12），小心混合，使其在室温下静置 $5 \mathrm{~min}$ ，避免直射阳光。用水定容至刻度，混匀。  
17.8.4 在 $15 \mathrm{~min}$ 内, 用 $538 \mathrm{~nm}$ 波长, 以第一个溶液 (不含亚硝酸钠) 为对照测定另外八个溶液的吸光度。  
17.8.5 将测得的吸光度对亚硝酸根质量浓度作图。亚硝酸根的质量浓度可根据加入的亚硝酸钠标准溶液的量计算出。亚硝酸根的质量浓度为横坐标，吸光度为纵坐标。亚硝酸根的质量浓度以 $\mu \mathrm{g} / 100 \mathrm{~mL}$ 表示。

# 18 分析结果的表述

# 18.1 亚硝酸盐含量

样品中亚硝酸根含量按式（5）计算：

$$
X = \frac {2 0 0 0 0 \times c _ {1}}{m \times V _ {1}} \tag {5}
$$

式中：

$X$ ——样品中亚硝酸根含量，单位为毫克每千克 $(\mathrm{mg/kg})$ ；

$c_{1}$ ——根据滤液（17.5.2）的吸光度（17.7.4），从标准曲线上读取的 $\mathrm{NO}_2^-$ 的浓度，单位为微克每百毫升（ $\mu \mathrm{g} / 100 \mathrm{~mL}$ ）；

$m$ ——样品的质量（液体乳的样品质量为 $90 \times 1.030 \mathrm{~g}$ ），单位为克（g）；

$V_{I}$ ——所取滤液（17.5.2）的体积（17.7.1），单位为毫升 $(\mathrm{mL})$ 。

样品中以亚硝酸钠表示的亚硝酸盐含量，按式（6）计算：

$$
\mathrm {W} \left(\mathrm {N a N O} _ {2}\right) = 1. 5 \times \mathrm {W} \left(\mathrm {N O} _ {2} ^ {-}\right) \dots \tag {6}
$$

式中：

W（ $\mathrm{NO}_2^-$ ）——样品中亚硝酸根的含量，单位为毫克每千克（ $\mathrm{mg/kg}$ ）；

W（ $\mathrm{NaNO}_2$ ）——样品中以亚硝酸钠表示的亚硝酸盐的含量，单位为毫克每千克（ $\mathrm{mg/kg}$ ）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 18.2硝酸盐含量

样品中硝酸根含量按式（7）计算：

$$
X = 1. 3 5 \times \left[ \frac {1 0 0 0 0 0 \times c _ {2}}{m \times V _ {2}} - W \left(N O _ {2} ^ {-}\right) \right] \dots \tag {7}
$$

式中：

$X$ ——样品中硝酸根含量（ $\mathrm{mg/kg}$ ）；

$c_{2}$ ——根据洗提液（17.6.3）的吸光度（17.7.4），从标准曲线上读取的亚硝酸根离子浓度，单位为微克每百毫升 $(\mu \mathrm{g} / 100 \mathrm{~mL})$ ；

$m$ ——样品的质量，单位为克（g）；

$V_{2}$ ——所取洗提液（17.6.3）的体积（17.7.1），单位为毫升（mL）；

$W(NO_{2}-)$ ——根据式（5）计算出的亚硝酸根含量。

若考虑柱的还原能力，样品中硝酸根含量按式（8）计算：

样品的硝酸根含量 $(\mathrm{mg / kg}) = 1.35\times \left[\frac{100000\times c_2}{m\times V_2} -W(NO_2^-)\right]\times \frac{100}{r}$ (8)

式中：

r——测定一系列样品后柱的还原能力。

样品中以硝酸钠计的硝酸盐的含量按式（9）计算：

$$
\mathrm {W} \left(\mathrm {N a N O} _ {3}\right) = 1. 3 7 1 \times \mathrm {W} \left(\mathrm {N O} _ {3} ^ {-}\right) \dots \tag {9}
$$

式中：

W $\left(\mathrm{NO}_{3}^{-}\right)$ ——样品中硝酸根的含量，单位为毫克每千克 $(\mathrm{mg/kg})$ ；

W（ $\mathrm{NaNO}_3$ ）——样品中以硝酸钠计的硝酸盐的含量，单位为毫克每千克（ $\mathrm{mg/kg}$ ）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 19 精密度

由同一分析人员在短时间间隔内测定的两个亚硝酸盐结果之间的差值，不应超过 $1\mathrm{mg / kg}$ 。

由同一分析人员在短时间间隔内测定的两个硝酸盐结果之间的差值，在硝酸盐含量小于 $30\mathrm{mg / kg}$ 时，不应超过 $3\mathrm{mg / kg}$ ；在硝酸盐含量大于 $30\mathrm{mg / kg}$ 时，不应超过结果平均值的 $10\%$ 。

由不同实验室的两个分析人员对同一样品测得的两个硝酸盐结果之差，在硝酸盐含量小于 $30\mathrm{mg / kg}$ 时，差值不应超过 $8\mathrm{mg / kg}$ ；在硝酸盐含量大于或等于 $30\mathrm{mg / kg}$ 时，该差值不应超过结果平均值的 $25\%$ 。

# 20 其他

本标准第一法中亚硝酸盐和硝酸盐检出限分别为 $0.2\mathrm{mg / kg}$ 和 $0.4\mathrm{mg / kg}$ ；第二法中亚硝酸盐和硝酸盐检出限分别为 $1\mathrm{mg / kg}$ 和 $1.4\mathrm{mg / kg}$ ；第三法中亚硝酸盐和硝酸盐检出限分别为 $0.2\mathrm{mg / kg}$ 和 $1.5\mathrm{mg / kg}$ 。

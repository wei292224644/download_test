# 附录A

# 检验方法

# A.1 警示

本标准的试样具有强腐蚀性，检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603之规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 氯化钡溶液： $100\mathrm{g / L}$   
A.3.1.2 盐酸溶液： $1 + 1$   
A.3.1.3 乙酸铅溶液： $80~\mathrm{g / L}$   
A.3.1.4 乙酸铵溶液： $100\mathrm{g / L}$

# A.3.2 鉴别方法

A.3.2.1 取约 $1\mathrm{g}$ 试样，溶于 $20~\mathrm{mL}$ 水，加入氯化钡溶液，即产生白色沉淀，此白色沉淀不溶于盐酸溶液。  
A.3.2.2 取约 $1\mathrm{g}$ 试样，溶于 $20~\mathrm{mL}$ 水，加入乙酸铅溶液，即产生白色沉淀，此白色沉淀溶于乙酸铵溶液。  
A.3.2.3 取约 $1\mathrm{g}$ 试样，溶于 $20~\mathrm{mL}$ 水，加入盐酸不产生沉淀。

# A.4 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 含量的测定

# A.4.1 方法提要

用氢氧化钠标准滴定溶液滴定试样溶液，根据氢氧化钠标准滴定溶液的消耗量，确定硫酸的含量。

# A.4.2 试剂

A.4.2.1 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 1\mathrm{mol} / \mathrm{L}$   
A.4.2.2 甲基橙指示液： $1\mathrm{g / L}$

# A.4.3 分析步骤

移取约 $1\mathrm{mL}$ 试样，置于已知质量的具塞小锥形瓶中，准确称其质量，精确至 $0.0002\mathrm{g}$ ，小心加入 $30~\mathrm{mL}$ 水，冷却后滴入2滴甲基橙指示液，用氢氧化钠标准滴定溶液滴定至黄色。

同时作空白试验。空白试验除不加试样外，其他操作及加入试剂的种类和量(标准滴定溶液除外)与测定试验相同。

# A.4.4 结果计算

硫酸 $\mathrm{(H_2SO_4)}$ 含量的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\left[ \left(V - V _ {0}\right) / 1000 \right] c M}{m} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V$ ——滴定试样溶液所消耗的氢氧化钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白试样溶液所消耗的氢氧化钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——氢氧化钠标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——试样的质量的数值，单位为克(g)；

1000——换算因子；

$M$ ——硫酸 $(1 / 2\mathrm{H}_2\mathrm{SO}_4)$ 摩尔质量的数值，单位为克每摩尔 $(\mathrm{g / mol})$ （ $M = 49.04$ ）。

实验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 硝酸盐（以 $\mathbf{NO}_3$ 计)的测定

# A.5.1 试剂和材料

# A.5.1.1 硫酸。

# A.5.1.2 硝酸盐 $\mathrm{(NO_3)}$ 标准溶液： $0.100\mathrm{mg / mL}$

称取 $8.022\mathrm{g}$ 已于 $105~^\circ \mathrm{C}\pm 2$ ℃干燥 $1\textrm{h}$ 的硝酸钾，置于 $500~\mathrm{mL}$ 容量瓶中，用水溶解并稀释至刻度，摇匀。移取 $5\mathrm{mL}$ 该溶液，将滴管插入 $400~\mathrm{mL}$ 硫酸液面以下，缓慢滴入，冷却至室温。转移至 $500~\mathrm{mL}$ 容量瓶，用硫酸稀释至刻度。

A.5.1.3 硫酸亚铁 $(\mathrm{FeSO}_4\cdot 7\mathrm{H}_2\mathrm{O})$ 溶液： $100~\mathrm{g / L}$ ，即配即用。

# A.5.2 分析步骤

分别于两支(A、B) $100~\mathrm{mL}$ 比色管中移入 $50~\mathrm{mL}$ 硫酸，缓慢加入 $5\mathrm{mL}$ 硫酸亚铁溶液，用玻璃棒搅拌均匀，用冰浴冷却至 $10^{\circ}\mathrm{C}\sim 15^{\circ}\mathrm{C}$ 。A比色管中加入 $10\mathrm{g}$ 试样，冷却至室温，用硫酸稀释至 $100~\mathrm{mL}$ 刻度。B比色管中边滴加 $1.0~\mathrm{mL}$ 硝酸盐标准溶液边混匀，用硫酸稀释至刻度。在白色发光背景下自上而下的观察A比色管中溶液的颜色与B比色管所呈颜色。A比色管所呈颜色不应深于B比色管所呈颜色。

# A.6 还原性物质的测定

称量 $8.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $100~\mathrm{mL}$ 烧杯中，用 $50~\mathrm{mL}$ 冰水稀释，冷却，滴加 $0.10~\mathrm{mL}$ 高锰酸钾溶液 $\left[c(1 / 5\mathrm{KMnO}_4) = 0.1\mathrm{mol} / \mathrm{L}\right]$ ，溶液所呈粉红色能保持 $5\mathrm{min}$ 。

# A.7 氯化物（以C1计）的测定

# A.7.1 试剂和材料

# A.7.1.1 $95\%$ 乙醇。

A.7.1.2 硝酸溶液： $1 + 2$   
A.7.1.3 硝酸银溶液： $17\mathrm{g / L}$   
A.7.1.4 氯化物(Cl)标准溶液： $0.010\mathrm{mg / mL}$

# A.7.2 分析步骤

称取 $1.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $50~\mathrm{mL}$ 烧杯中，加入适量的水使之溶解，全部转移至 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。用移液管移取 $10~\mathrm{mL}$ 上述试样溶液，置于 $50~\mathrm{mL}$ 比色管中，加入 $4\mathrm{mL}$ 硝酸溶液和 $2\mathrm{mL}$ 硝酸银溶液，用水稀释至刻度，轻轻摇匀。静置 $10\mathrm{min}$ 后，于黑背景下自上而下的观察与标准比浊溶液比对，所产生的浊度不应深于标准比浊溶液。

标准比浊溶液是移取 $5.00\mathrm{mL}$ 氯化物标准溶液，与试样同时同样处理。

# A.8 铁 $(\mathbf{Fe})$ 的测定

# A.8.1 试剂和材料

A.8.1.1 盐酸溶液： $1 + 3$   
A.8.1.2 其他试剂和材料同GB/T3049—2006第4章。

# A.8.2 仪器和设备

分光光度计：配有 $4\mathrm{cm}$ 比色Ⅲ。

# A.8.3 分析步骤

# A.8.3.1 工作曲线的绘制

按GB/T3049—2006中6.3的规定，使用 $4\mathrm{cm}$ 比色皿，绘制铁含量为 $10~\mu \mathrm{g}\sim 100~\mu \mathrm{g}$ 工作曲线。

# A.8.3.2 测定

称取约 $0.5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $150~\mathrm{mL}$ 烧杯中，在砂浴或可控温电炉上加热近干。以下按GB/T3049—2006中6.4规定从“必要时，加水至 $60~\mathrm{mL}$ ……”开始进行操作。同时同样处理空白试样溶液。从工作曲线上查出相应的铁的质量。

# A.8.4 结果计算

铁(Fe)的质量分数 $\omega_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {\left(m _ {1} - m _ {0}\right) / 1000}{m} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$m_{1}$ ——从工作曲线上查出的试样溶液中铁的质量的数值，单位为毫克 $(\mathrm{mg})$ ；

$m_{0}$ ——从工作曲线上查出的空白试样溶液中铁的质量的数值，单位为毫克 $(\mathrm{mg})$ ；

$m$ ——试样质量的数值，单位为克(g)；

1000 ——换算因子。

实验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.0005\%$ 。

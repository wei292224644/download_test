# 动物性食品中对乙酰氨基酚残留量的测定

# 高效液相色谱法

# 1 范围

本标准规定了动物性食品中对乙酰氨基酚残留量检测的制样和高效液相色谱测定方法。

本标准适用于猪、牛和羊的肌肉、肝脏和肾脏组织中对乙酰氨基酚残留量的检测。

# 3原理

试料中残留的乙酰氨基酚，用乙酸乙酯提取，HLB柱净化，甲醇洗脱，氮气吹干，流动相溶解，高效液相色谱测定，外标法定量。

# 4 试剂和材料

以下所用试剂，除特别注明者外均为分析纯；水为符合GB/T6682规定的一级水。

4.1 对乙酰氨基酚标准品：含量 $\geqslant 99\%$   
4.2 甲醇：色谱纯。  
4.3 乙酸乙酯  
4.4 正己烷  
4.5 HLB固相萃取柱： $60\mathrm{mg / 3mL}$ ，或相当者。  
4.6 $5\%$ 甲醇水溶液：取甲醇 $5\mathrm{mL}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
4.7 $0.05\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液：取乙酸铵 $3.15\mathrm{g}$ ，用水溶解并稀释至 $1000\mathrm{mL}$ ，滤膜过滤。  
4.8 $1\mathrm{mg / mL}$ 对乙酰氨基酚标准贮备液：精密称取对乙酰氨基酚 $10\mathrm{mg}$ ，于 $10\mathrm{mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $1\mathrm{mg / mL}$ 的对乙酰氨基酚标准贮备液。 $-20^{\circ}\mathrm{C}$ 以下保存，有效期3个月。  
4.9 $10~\mu \mathrm{g / mL}$ 对乙酰氨基酚标准工作液：精密量取 $1\mathrm{mg / mL}$ 对乙酰氨基酚标准贮备液 $1.0  \mathrm{mL}$ ，于

$100 \mathrm{~mL}$ 量瓶中, 用流动相溶解并稀释至刻度, 配制成浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 的对乙酰氨基酚标准工作液。现配现用。

# 5 仪器和设备

5.1 高效液相色谱仪：配紫外检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.3 冷冻高速离心机  
5.4 匀浆机  
5.5 旋涡混合器  
5.6 循环水真空泵  
5.7 具塞离心管： $50~\mathrm{mL}$ 。  
5.8滤膜： $0.45\mu \mathrm{m}$

# 6 试料的制备与保存

# 6.1 试料的制备

取新鲜或冷冻的空白或供试组织，去除筋膜，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-20°C以下保存。

# 7 测定步骤

# 7.1 提取

称取试料 $(5 \pm 0.05) \mathrm{~g}$ , 于具塞离心管中, 加乙酸乙酯 $20 \mathrm{~mL}$ 旋涡 $2 \mathrm{~min}$ , $6000 \mathrm{r} / \mathrm{min}$ 离心 $15 \mathrm{~min}$ , 取上清液, 于另一离心管中, 残渣中加乙酸乙酯 $10 \mathrm{~mL}$ , 重复提取一次, 合并两次上清液, $6000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ , 取上清液, 于 $45^{\circ} \mathrm{C}$ 水浴蒸发至干, 用甲醇 $3 \mathrm{~mL}$ 溶解残余物, 加正己烷 $5 \mathrm{~mL}$ , 混匀, 静置分层, 弃正己烷层液, 取下层液, 于 $45^{\circ} \mathrm{C}$ 水浴氮气吹干, 用水 $6 \mathrm{~mL}$ 溶解残余物, $10000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ , 取上清液, 备用。

# 7.2 净化

HLB 柱依次用甲醇 $3 \mathrm{~mL}$ 和水 $3 \mathrm{~mL}$ 活化，取备用液过柱，控制流速小于 $2 \mathrm{~mL} / \mathrm{min}$ 。用水 $3 \mathrm{~mL}$ 和 $5 \%$ 甲醇 $3 \mathrm{~mL}$ 淋洗，甲醇 $4 \mathrm{~mL}$ 洗脱，收集洗脱液，于 $40 \sim 45^{\circ} \mathrm{C}$ 水浴氮气吹干，用流动相 $1.0 \mathrm{~mL}$ 溶解残余物， $10000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，取上清液，供高效液相色谱测定。

# 7.3 标准曲线的制备

精密量取 $10~\mu \mathrm{g / mL}$ 对乙酰氨基酚标准工作液适量，用流动相稀释，配制成浓度为10、25、50、100、200、400、800和 $1600~\mu \mathrm{g / L}$ 的系列标准溶液，供高效液相色谱测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.4 测定

# 7.4.1 液相色谱条件

色谱柱： $\mathrm{C_{18}}$ （ $250 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ ，粒径 $5 \mu \mathrm{m}$ ），或相当者。

流动相：甲醇 $+0.05\mathrm{mol / L}$ 乙酸铵溶液（ $20 + 80$ ， $\mathrm{v / v})$

流速： $1.0\mathrm{mL / min}$

紫外检测波长： $250\mathrm{nm}$

进样量： $50~\mu \mathrm{L}$

柱温： $30^{\circ}\mathrm{C}$ 。

# 7.4.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法以峰面积计算。标准溶液及试样溶液中对乙酰氨基酚响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液空白添加试样溶液的高效液相色谱图分别见附录A。

# 7.5 空白试验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算和表述

试料中对乙酰氨基酚药物的残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算

$$
X = \frac {A \times C S \times V}{A S \times m}
$$

式中：

$X$ ——供试试料中对乙酰氨基酚药物残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

$A$ ——试样溶液中对乙酰氨基酚的峰面积；

$A_{S}$ ——标准工作液中对乙酰氨基酚的峰面积；

$C_{\mathrm{S}}$ ——标准工作液中对乙酰氨基酚的浓度, $\mu \mathrm{g} / \mathrm{L}$ ;

$V$ ——溶解残余物所用流动相体积, $\mathrm{mL}$ ;

$m$ ——供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法的灵敏度、准确度、精密度

# 9.1 灵敏度

本方法的检测限为 $3\mu \mathrm{g} / \mathrm{kg}$ ，定量限为 $10~\mu \mathrm{g} / \mathrm{kg}$

# 9.2 准确度

本方法在 $10\sim 100~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$

# 9.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

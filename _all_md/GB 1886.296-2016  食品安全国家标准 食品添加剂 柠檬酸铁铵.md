# 1 范围

本标准适用于以柠檬酸、硫酸亚铁或铁、氨为原料，经加工制得的食品添加剂柠檬酸铁铵。  
本标准同样适用于食品营养强化剂柠檬酸铁铵。

# 附录A

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。必要时，需在通风橱中进行。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。

# A.2 一般规定

本标准除另有规定外，所用试剂的纯度应在分析纯以上，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和溶液

A.3.1.1 盐酸溶液： $1 + 3$

A.3.1.2 三氯化铁溶液： $100\mathrm{g / L}$

A.3.1.3 高锰酸钾溶液：称取高锰酸钾 $0.316\mathrm{~g}$ ，加水溶解定容至 $100\mathrm{~mL}$ ，浓度约为 $0.1\mathrm{~mol/L}$ 。

A.3.1.4 硫酸汞溶液：取黄色氧化汞 $5\mathrm{g}$ ，加水 $40\mathrm{mL}$ 混合，边搅拌边加入硫酸 $20\mathrm{mL}$ ，再加水 $40\mathrm{mL}$ ，搅拌至完全溶解。

A.3.1.5 氢氧化钠溶液： $1\mathrm{mol} / \mathrm{L}$

# A.3.2 鉴别方法

A.3.2.1 试样易溶于水，不溶于乙醇。

A.3.2.2 取 $0.5 \mathrm{~g}$ 试样灼烧, 用 $5 \mathrm{~mL}$ 盐酸溶液溶解所生成的残渣, 所得溶液与三氯化铁溶液对比颜色相同。

A.3.2.3 取 $5 \mathrm{~mL} 10 \%$ 试样溶液, 分别加入 $0.3 \mathrm{~mL}$ 高锰酸钾溶液、 $4 \mathrm{~mL}$ 硫酸汞溶液, 加热混合溶液至沸腾, 有白色沉淀生成。

A.3.2.4 称取 $0.5\mathrm{g}$ 试样，溶于 $5\mathrm{mL}$ 水中，加入 $5\mathrm{mL}$ 氢氧化钠溶液，有红棕色沉淀生成，加热混合物，有氨气放出。

# A.4 柠檬酸铁铵含量（以Fe计）的测定

# A.4.1 试剂和溶液

A.4.1.1 盐酸。

A.4.1.2 碘化钾。

A.4.1.3 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$

A.4.1.4 淀粉指示液： $5\mathrm{g / L}$

# A.4.2 分析步骤

称取约 $1\mathrm{g}$ 试样，精确至 $0.1\mathrm{mg}$ ，置于 $250~\mathrm{mL}$ 碘量瓶中，加水 $25~\mathrm{mL}$ 溶解，再加入 $5\mathrm{mL}$ 盐酸和 $4\mathrm{g}$ 碘化钾，摇匀，盖塞，于暗处静置 $15\mathrm{min}$ 后，加水 $100~\mathrm{mL}$ ，用硫代硫酸钠标准滴定溶液滴定，近终点时加入 $1\mathrm{mL}$ 淀粉指示液，继续滴定至溶液蓝色消失，同时做空白试验。

# A.4.3 结果计算

柠檬酸铁铵含量(以Fe计)的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\left(V - V _ {0}\right) \times c \times M}{m \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V$ ——试样所用硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验所用硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——所用硫代硫酸钠标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——铁的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M = 55.85]$ ；

$m$ ——试样的质量，单位为克(g)；

1000——换算系数。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的允许绝对差值不大于 $10\%$ 。

# A.5 柠檬酸铁的测定

# A.5.1 试剂和溶液

亚铁氰化钾溶液： $100\mathrm{g / L}$

# A.5.2 分析步骤

称取 $1.0\mathrm{g}$ 试样，加水 $100~\mathrm{mL}$ 溶解，加入亚铁氰化钾溶液 $10~\mathrm{mL}$ ，无蓝色沉淀生成，即为通过试验。

# A.6 草酸盐的测定

# A.6.1 试剂和溶液

A.6.1.1 乙醚。

A.6.1.2 冰乙酸。

A.6.1.3 盐酸溶液： $1 + 3$

A.6.1.4 醋酸钙溶液：称取 $5.00\mathrm{g}$ 醋酸钙，加水溶解，定容至 $100~\mathrm{mL}$

# A.6.2 分析步骤

取 $1.00\mathrm{g}$ 试样，加水 $10~\mathrm{mL}$ 溶解，置于 $125~\mathrm{mL}$ 分液漏斗中，再加入 $2\mathrm{mL}$ 盐酸溶液，先后用 $50~\mathrm{mL}$ 和 $20~\mathrm{mL}$ 乙醚萃取，合并萃取液于 $150~\mathrm{mL}$ 烧杯中，加水 $10~\mathrm{mL}$ ，在水浴上挥干乙醚，在水溶液中加入1滴冰乙酸和 $1~\mathrm{mL}$ 醋酸钙溶液， $5\mathrm{min}$ 内不得产生浑浊，即为通过试验。

# A.7 硫酸盐（以 $\mathrm{SO}_4$ 计）的测定

# A.7.1 试剂和溶液

A.7.1.1 盐酸溶液： $1 + 3$

A.7.1.2 硫酸溶液： $0.005\mathrm{mol / L}$   
A.7.1.3 氯化钡溶液： $120\mathrm{g / L}$

# A.7.2 分析步骤

称取 $0.1\mathrm{g}$ 试样，精确至 $0.1\mathrm{mg}$ ，加水溶解，置于 $50~\mathrm{mL}$ 比色管中，加入 $1\mathrm{mL}$ 盐酸溶液，用水稀释至 $50~\mathrm{mL}$ ，此为试样溶液。另取一支比色管，加入 $0.60~\mathrm{mL}$ 硫酸溶液，再加入 $1\mathrm{mL}$ 盐酸溶液，用水稀释至 $50~\mathrm{mL}$ ，此为对照液。若溶液不够澄清，则将两份溶液在同样条件下过滤。分别加入氯化钡溶液 $2\mathrm{mL}$ ，混匀后放置 $10\mathrm{min}$ 。在黑色背景上从比色管上方比较两者浊度，试样溶液浊度不应超过对照液浊度，即试样中硫酸盐含量(以 $\mathrm{SO}_4$ 计)不大于 $0.3\%$ 。

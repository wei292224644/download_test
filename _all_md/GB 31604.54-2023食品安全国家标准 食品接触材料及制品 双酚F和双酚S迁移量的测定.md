# 食品安全国家标准

# 食品接触材料及制品

# 双酚F和双酚S迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中双酚F和双酚S迁移量的液相色谱和液相色谱-串联质谱测定方法。

第一法液相色谱法适用于食品接触用塑料材料及制品、食品接触用涂料及涂层、食品接触用纸和纸板材料及制品中双酚F和双酚S迁移量的测定。

第二法液相色谱-串联质谱法适用于食品接触材料及制品中双酚F和双酚S迁移量的测定。

# 第一法 液相色谱法

# 2 原理

食品接触材料及制品根据GB31604.1及GB5009.156进行迁移试验后，采用液相色谱检测，其中水性、酸性、酒精类食品模拟物过滤后直接进样； $95\%$ （体积分数）乙醇用氮气吹干后，采用乙醇-水定容，过滤后进样；油脂类食品模拟物通过甲醇-水混合液萃取，过滤后进样；异辛烷化学替代溶剂用氮气吹干后，采用甲醇-水定容，过滤后进样。外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为色谱纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 水性、酸性、酒精类、油脂类食品模拟物及化学替代溶剂：所用试剂按照GB5009.156的规定。  
3.1.2 正己烷 $\mathrm{(C_6H_{14})}$ 。  
3.1.3 甲醇 $(\mathrm{CH}_4\mathrm{O})$ 。  
3.1.4 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ 。  
3.1.5 异辛烷 $\left(\mathrm{C}_{8} \mathrm{H}_{18}\right)$ 。  
3.1.6 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ 。  
3.1.7 乙醇 $\mathrm{(C_2H_6O)}$ 。  
3.1.8 玉米油：化学纯，符合GB5009.156的要求。

# 3.2 试剂配制

3.2.1 水性、酸性、酒精类、玉米油食品模拟物及化学替代溶剂：按照GB5009.156操作。  
3.2.2 甲醇-水混合液 $(1 + 1)$ ：量取 $100~\mathrm{mL}$ 甲醇和 $100~\mathrm{mL}$ 水，混匀。  
3.2.3 乙醇-水混合液 $(1 + 1)$ ：量取 $100~\mathrm{mL}$ 乙醇和 $100~\mathrm{mL}$ 水，混匀。

# 3.3 标准品

3.3.1 双酚 $\mathrm{F}(\mathrm{C}_{13}\mathrm{H}_{12}\mathrm{O}_2$ ，CAS：620-92-8)：纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.2 双酚 $\mathrm{S}(\mathrm{C}_{12}\mathrm{H}_{10}\mathrm{O}_4\mathrm{S},\mathrm{CAS}:80 - 09 - 1)$ ：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

# 3.4.1 标准储备溶液

# 3.4.1.1 双酚F（双酚S)标准储备溶液A( $100\mathrm{mg / L}$ ，甲醇）

准确称取双酚F(双酚S)标准品 $10\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ), 用甲醇溶解并定容至 $100\mathrm{mL}$ 容量瓶中, 混匀。将溶液转移至棕色标准溶液储备瓶, 于 $4^{\circ}\mathrm{C}$ 冰箱避光保存, 有效期为3个月。

# 3.4.1.2 双酚F（双酚S)标准储备溶液 $\mathbf{B}(100\mathrm{mg / L}$ ，异辛烷）

准确称取双酚F(双酚S)标准品 $10\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ), 用 $10\mathrm{mL}$ 丙酮溶解后, 用异辛烷定容至 $100\mathrm{mL}$ 容量瓶中, 混匀。将溶液转移至棕色标准溶液储备瓶中, 于 $4^{\circ}\mathrm{C}$ 冰箱避光保存, 有效期为3个月。

# 3.4.2 标准中间溶液

# 3.4.2.1 双酚F（双酚S)标准中间溶液A( $10.0\mathrm{mg / L}$ ，甲醇）

准确吸取双酚F(双酚S)标准储备溶液A(3.4.1.1) $10.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用甲醇定容，混匀。将溶液转移至棕色标准溶液储备瓶中，于 $4^{\circ}C$ 冰箱避光保存，有效期为1个月。

# 3.4.2.2 双酚F（双酚S)标准中间溶液B(10.0mg/L，异辛烷）

准确吸取双酚F(双酚S)标准储备溶液B(3.4.1.2) $10.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用异辛烷定容，混匀。将溶液转移至棕色标准溶液储备瓶中，于 $4^{\circ}C$ 冰箱避光保存，有效期为1个月。

# 3.4.2.3 双酚F（双酚S)标准中间溶液C(1.00mg/L，甲醇-水）

准确吸取双酚F(双酚S)标准储备溶液A(3.4.1.1) $1.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用甲醇-水混合液定容，混匀。临用现配。

# 3.4.2.4 双酚S标准中间溶液 $\mathbf{D}(0.100\mathrm{mg / L}$ ，甲醇-水）

准确吸取双酚S标准中间溶液C(3.4.2.3) $10.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用甲醇-水混合液定容，混匀。临用现配。

# 3.4.3 标准工作溶液的制备

# 3.4.3.1 水性、酸性、酒精类食品模拟物标准工作溶液

准确吸取双酚F、双酚S标准中间溶液A(3.4.2.1) $0.150\mathrm{mL},0.250\mathrm{mL},0.500\mathrm{mL},2.50\mathrm{mL}$ 产 $5.00~\mathrm{mL}$ 于5个 $50~\mathrm{mL}$ 容量瓶中，用水定容，得到浓度为 $30.0~\mu \mathrm{g / L},50.0~\mu \mathrm{g / L},100~\mu \mathrm{g / L},500~\mu \mathrm{g / L},$ $1.00\mathrm{mg / L}$ 的双酚F、双酚S混合标准工作液。采用同样的方式，分别用酸性、酒精类食品模拟物配制同样浓度系列的标准工作溶液，临用现配。酒精类食品模拟物包括 $10\%$ （体积分数）乙醇、 $20\%$ （体积分数)乙醇、 $50\%$ (体积分数)乙醇，依据GB31604.1，按食品接触材料的实际使用条件进行选择。 $95\%$ （体积分数)乙醇采用 $50\%$ (体积分数)乙醇的标准工作曲线。

# 3.4.3.2 油脂类食品模拟物标准工作溶液

在5个 $10\mathrm{mL}$ 具塞试管中分别准确加入玉米油 $2\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ), 加入双酚F标准中间溶液C(3.4.2.3) $200\mu \mathrm{L}, 300\mu \mathrm{L}, 400\mu \mathrm{L}, 1.00\mathrm{mL}, 2.00\mathrm{mL}$ ; 加入双酚S标准中间溶液D(3.4.2.4) $600\mu \mathrm{L}$ , 双酚S标准中间溶液C(3.4.2.3) $100\mu \mathrm{L}, 200\mu \mathrm{L}, 400\mu \mathrm{L}, 1.00\mathrm{mL}$ , 得到双酚F浓度为 $100\mu \mathrm{g} / \mathrm{kg}$ , $150\mu \mathrm{g} / \mathrm{kg}, 200\mu \mathrm{g} / \mathrm{kg}, 500\mu \mathrm{g} / \mathrm{kg}, 1.00\mathrm{mg} / \mathrm{kg}$ , 双酚S浓度为 $30.0\mu \mathrm{g} / \mathrm{kg}, 50.0\mu \mathrm{g} / \mathrm{kg}, 100\mu \mathrm{g} / \mathrm{kg}, 200\mu \mathrm{g} / \mathrm{kg}, 500\mu \mathrm{g} / \mathrm{kg}$ 的混合标准溶液, 临用现配。分别在每个试管中再加入 $3\mathrm{mL}$ 正己烷, 混匀。在5个 $10\mathrm{mL}$ 具塞试管中分别加入 $2.20\mathrm{mL}, 2.60\mathrm{mL}, 2.40\mathrm{mL}, 1.60\mathrm{mL}, 0\mathrm{mL}$ 甲醇-水混合液, 涡旋振荡 $2\mathrm{min}$ , 静置分层。吸取下层溶液, 通过 $0.45\mu \mathrm{m}$ 聚四氟乙烯微孔滤膜过滤后供测定用。

# 3.4.3.3 异辛烷化学替代溶剂标准工作溶液

准确吸取双酚F、双酚S标准中间溶液B(3.4.2.2) $0.150\mathrm{mL},0.250\mathrm{mL},0.500\mathrm{mL},2.50\mathrm{mL}$ 产 $5.00~\mathrm{mL}$ 于5个 $50~\mathrm{mL}$ 容量瓶中，用异辛烷定容，得到浓度为 $30.0~\mu \mathrm{g / L},50.0~\mu \mathrm{g / L},100~\mu \mathrm{g / L},$ $500~\mu \mathrm{g / L},1.00~\mathrm{mg / L}$ 的双酚F、双酚S混合标准工作液。上机前需按照5.1.2.4与食品模拟物浸泡液同步处理。

# 4 仪器和设备

4.1 液相色谱仪，配紫外检测器(UV)或二极管阵列检测器(DAD)。  
4.2 涡旋振荡器。  
4.3 移液器：20 $\mu$ L、50 $\mu$ L、250 $\mu$ L、1000 $\mu$ L、5000 $\mu$ L。  
4.4 分析天平：感量分别为 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$   
4.5 恒温水浴锅。  
4.6 氮吹仪。  
4.7 聚四氟乙烯微孔滤膜： $0.45 \mu \mathrm{m}$ 。

# 5 分析步骤

# 5.1 试液的制备

# 5.1.1 迁移试验

按照GB5009.156及GB31604.1的要求，对样品进行迁移试验，得到食品模拟物试液。迁移试验所得试液如不能立即测试，应置于 $4^{\circ}\mathrm{C}$ 冰箱中避光保存，贮存期限为 $7\mathrm{d}$ 。进行下一步试验前，应将食品模拟物试液恢复至室温。

# 5.1.2 食品模拟物的处理

# 5.1.2.1 水性、酸性、酒精类[95%（体积分数）乙醇除外]食品模拟物

准确移取迁移试验中得到的水性、酸性、酒精类食品模拟物约 $2\mathrm{mL}$ ，采用 $0.45\mu \mathrm{m}$ 聚四氟乙烯微孔滤膜过滤后供测定用。

# 5.1.2.2 $95\%$ （体积分数）乙醇

准确吸取迁移试验中得到的 $95\%$ （体积分数)乙醇 $1\mathrm{mL}$ 至试管中，用氮气吹干。在试管中准确加入 $1\mathrm{mL}$ 乙醇-水混合液，旋涡混合 $2\mathrm{min}$ 。溶液采用 $0.45\mu \mathrm{m}$ 聚四氟乙烯微孔滤膜过滤后供测定用。

# 5.1.2.3 油脂类食品模拟物

准确称取迁移试验中得到的玉米油 $2\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）于 $10~\mathrm{mL}$ 试管中，加入 $3\mathrm{mL}$ 正己烷，混匀，加入 $3\mathrm{mL}$ 甲醇-水混合液，涡旋振荡 $2\mathrm{min}$ ，静置分层。吸取下层溶液，采用 $0.45\mu \mathrm{m}$ 聚四氟乙烯微孔滤膜过滤后供测定用。

# 5.1.2.4 异辛烷化学替代溶剂

准确吸取迁移试验中得到的异辛烷化学替代溶剂 $5\mathrm{mL}$ 至试管中，用氮气吹干。在试管中准确加入 $5\mathrm{mL}$ 甲醇-水混合液，旋涡混合 $2\mathrm{min}$ 。溶液采用 $0.45\mu \mathrm{m}$ 聚四氟乙烯微孔滤膜过滤后供测定用。

# 5.1.3 空白试液的制备

按5.1.1和5.1.2处理未与食品接触材料及制品接触的食品模拟物、化学替代溶剂。

# 5.3 标准曲线的制作

按照5.2所列的仪器参考条件，将标准系列工作液分别注入液相色谱仪中，测定相应的峰面积，以标准工作溶液中双酚F(双酚S)的浓度为横坐标，以对应的峰面积为纵坐标，绘制标准曲线，得到线性方程。双酚F(双酚S)标准工作溶液的色谱图参见附录A中图A.1～图A.7。

# 5.4 试液的测定

# 5.4.1 定性测定

按照5.2所列的仪器参考条件，测定食品模拟物试液和标准工作溶液，食品模拟物试液与标准工作溶液的色谱峰保留时间差异应在 $\pm 2.5\%$ 范围内。

# 5.4.2 定量测定

按照5.2所列的仪器参考条件，将试样溶液和空白试液注入液相色谱仪中，得到目标物峰面积，根据标准曲线计算试样溶液和空白试液中双酚F(双酚S)的含量 $c$ 和 $c_{0}$ ，空白试液中双酚F(双酚S)的含量 $c_{0}$ 不应高于方法检出限浓度。

# 6 分析结果的表述

# 6.1 非密封制品类食品接触材料及制品双酚F（双酚S)特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

对于除了盖子、密封圈、连接件等密封制品（以下简称密封制品）以外的食品接触材料及制品，双酚F(双酚S)特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(1)进行计算。

$$
X _ {1} = \frac {\left(c - c _ {0}\right) \times V}{1 0 0 0 \times S} \times F \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——食品接触材料及制品中双酚F(双酚S)的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样浸泡液中双酚F(双酚S)的含量，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ 或微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c_{0}$ ——空白浸泡液中双酚F(双酚S)的含量，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ 或微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克(kg)；

1000 ——换算系数；

S ——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$F$ ——在实际使用情形下，食品接触材料及制品的接触面积(S)与食品或食品模拟物体积(V)的比 $(S / V)$ ，各种液态食品密度通常以 $1\mathrm{kg} / \mathrm{L}$ 计，单位为平方分米每千克 $(\mathrm{dm}^2 /\mathrm{kg})$ 。

在实际使用情形下，当 $S / V$ 已知时， $F$ 即为实际 $S / V$ ；当 $S / V$ 未知时， $F$ 采用 $6\mathrm{dm}^2 /\mathrm{kg}$ ，即 $6\mathrm{dm}^2$ 食品接触材料及制品接触 $1\mathrm{kg}$ 食品或食品模拟物。

结果至少保留2位有效数字。

# 6.2 密封制品类食品接触材料及制品双酚F（双酚S)特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

当预期用途已知时，密封制品类食品接触材料及制品双酚F(双酚S)特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(2)计算，需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {2} = \frac {\left(c - c _ {0}\right) \times V}{1 0 0 0 \times S} \times F \quad \dots \tag {2}
$$

式中：

$X_{2}$ ——食品接触材料及制品中双酚F(双酚S)的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ 。

结果至少保留2位有效数字。

# 6.3 密封制品类食品接触材料及制品双酚F(双酚S)特定迁移量的计算（以 $\mathrm{mg}$ /件表示）

当预期用途未知时，密封制品类食品接触材料及制品双酚F(双酚S)特定迁移量以 $\mathrm{mg}/$ 件表示时，按式(3)计算，需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {3} = \frac {\left(c - c _ {0}\right) \times V}{1 0 0 0 \times n} \tag {3}
$$

式中：

$X_{3}$ ——食品接触材料及制品中双酚F(双酚S)的特定迁移量，单位为毫克每件 $(\mathrm{mg}/$ 件）；

$n$ ——浸泡用密封制品的件数，单位为件。

结果至少保留2位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过其算术平均值的 $15\%$ 。

# 8 其他

本方法在使用水性、酸性、酒精类食品模拟物、异辛烷化学替代溶剂时，食品接触材料及制品中双酚F和双酚S的检出限均为 $0.01\mathrm{mg / kg}$ ，定量限均为 $0.03\mathrm{mg / kg}$ ；使用油脂类食品模拟物时，食品接触材料及制品中双酚F检出限为 $0.03\mathrm{mg / kg}$ ，定量限为 $0.1\mathrm{mg / kg}$ ；双酚S检出限为 $0.01\mathrm{mg / kg}$ ，定量限为 $0.03\mathrm{mg / kg}$ 。

# 第二法 液相色谱-串联质谱法

# 9 原理

食品接触材料及制品根据GB31604.1及GB5009.156进行迁移试验后，采用液相色谱-串联质谱检测，其中水性、酸性、酒精类食品模拟物过滤后直接进样；油脂类食品模拟物通过甲醇-水混合液萃取，过滤后进样。异辛烷化学替代溶剂用氮气吹干后，采用甲醇-水定容，过滤后进样。外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为色谱纯，水为GB/T6682规定的一级水。

# 10.1 试剂

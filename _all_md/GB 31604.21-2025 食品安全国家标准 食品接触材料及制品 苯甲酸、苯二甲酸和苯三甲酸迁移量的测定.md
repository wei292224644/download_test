# 苯甲酸、苯二甲酸和苯三甲酸迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中苯甲酸、1,2-苯二甲酸（又称邻苯二甲酸）、1,3-苯二甲酸（又称间苯二甲酸）、1,4-苯二甲酸（又称对苯二甲酸）、1,3,5-苯三甲酸（又称均苯三甲酸）、1,2,4-苯三甲酸（又称偏苯三甲酸）、1,2,3-苯三甲酸（又称连苯三甲酸）迁移量的液相色谱测定方法。

本标准适用于食品接触用塑料材料及制品、食品接触用纸和纸板材料及制品、食品接触材料及制品用黏合剂、食品接触用涂料及涂层中苯甲酸、邻苯二甲酸、间苯二甲酸、对苯二甲酸、均苯三甲酸、偏苯三甲酸、连苯三甲酸迁移量的测定。

# 2 原理

食品接触材料及制品根据GB31604.1及GB5009.156进行迁移实验后，橄榄油和异辛烷浸泡液中各目标分析物经 $50\%$ （体积分数)乙醇溶液或乙腈溶液 $(5 + 1)$ 萃取； $95\%$ （体积分数)乙醇浸泡液将乙醇体积分数稀释至 $50\%$ 以下；其他浸泡液直接进样测定。各目标分析物经液相色谱分离，紫外检测器或二极管阵列检测器测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 冰乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
3.1.2 无水乙醇 $\mathrm{(C_2H_6O)}$ 。  
3.1.3 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ 。  
3.1.4 $95\%$ 乙醇。  
3.1.5 氢氧化钠(NaOH)。  
3.1.6 正庚烷 $(\mathrm{C_7H_{16}})$ 。  
3.1.7 异辛烷 $\mathrm{(C_8H_{18})}$   
3.1.8 磷酸 $\mathrm{(H_3PO_4)}$ ：色谱纯。  
3.1.9 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。

# 3.2 试剂配制

3.2.1 食品模拟物：按GB5009.156的要求进行配制。  
3.2.2 $0.05\%$ （体积分数)磷酸溶液：移取 $0.5\mathrm{mL}$ 磷酸，加水 $1000~\mathrm{mL}$ ，混匀。  
3.2.3 乙腈溶液 $(5 + 1)$ ：量取 $50~\mathrm{mL}$ 乙腈(分析纯)与 $10~\mathrm{mL}$ 水，混匀。

3.2.4 乙腈溶液 $(1 + 1)$ ：量取 $50~\mathrm{mL}$ 乙腈(分析纯)与 $50~\mathrm{mL}$ 水，混匀。  
3.2.5 $50\%$ （体积分数）乙醇溶液：量取 $50~\mathrm{mL}$ 无水乙醇与 $50~\mathrm{mL}$ 水，混匀。  
3.2.6 $1.0\mathrm{mol} / \mathrm{L}$ NaOH溶液：按照GB/T601的方法配制，无需标定。

# 3.3 标准品/物质

7种苯甲酸、苯二甲酸和苯三甲酸标准品（见附录A)，纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

# 3.4.1 标准储备液

# 3.4.1.1 对苯二甲酸标准储备液（ $500\mathrm{mg / L}$ ）

准确称取 $50\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ) 对苯二甲酸于具塞锥形瓶中，加入 $1\mathrm{mL}1.0\mathrm{mol} / \mathrm{L}\mathrm{NaOH}$ 溶液，超声溶解后加入少量乙腈溶液 $(1 + 1)$ 稀释后，转移至 $100\mathrm{mL}$ 容量瓶中，并用乙腈溶液 $(1 + 1)$ 定容。在 $4^{\circ}\mathrm{C} \sim 10^{\circ}\mathrm{C}$ 条件下避光保存，有效期6个月。

# 3.4.1.2 其他标准物质单标储备液（ $1000\mathrm{mg / L}$ ）

准确称取 $100\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ) 其他标准物质于具塞锥形瓶中，加入 $1\mathrm{mL}1.0\mathrm{mol} / \mathrm{L}\mathrm{NaOH}$ 溶液，超声溶解后加入少量乙腈溶液 $(1 + 1)$ 稀释后，转移至 $100\mathrm{mL}$ 容量瓶中，并用乙腈溶液 $(1 + 1)$ 定容。在 $4^{\circ}\mathrm{C} \sim 10^{\circ}\mathrm{C}$ 条件下避光保存，有效期6个月。

# 3.4.2 混合标准中间液A

准确移取 $24\mathrm{mL}$ 对苯二甲酸储备液 $(500\mathrm{mg / L})$ 、 $1.0~\mathrm{mL}$ 均苯三甲酸标准储备液 $(1000\mathrm{mg / L})$ 于 $200~\mathrm{mL}$ 容量瓶中，再分别移取其他5种标准储备液 $(1000\mathrm{mg / L})$ 各 $12\mathrm{mL}$ 于同一容量瓶中，用乙腈定容至刻度，混匀。混合标准中间液中均苯三甲酸的质量浓度为 $5\mathrm{mg / L}$ ，其余目标分析物的质量浓度均为 $60~\mathrm{mg / L}$ 。混合标准中间液置于 $4^{\circ}C\sim 10^{\circ}C$ 条件下避光保存，有效期 $15\mathrm{d}$

# 3.4.3 混合标准中间液B

准确移取 $25~\mathrm{mL}$ 混合标准中间液A于 $50~\mathrm{mL}$ 容量瓶中，用乙腈定容至刻度，混匀。混合标准中间液B中均苯三甲酸的质量浓度为 $2.5\mathrm{mg / L}$ ，其余目标分析物的质量浓度均为 $30\mathrm{mg / L}$ 。临用现配。

# 3.4.4 混合标准中间液C

准确移取 $5.0\mathrm{mL}$ 对苯二甲酸储备液 $(500\mathrm{mg / L})$ 于 $25~\mathrm{mL}$ 容量瓶中，再分别准确移取 $2.5\mathrm{mL}$ 其他6种标准物质单标储备液 $(1000\mathrm{mg / L})$ 于同一容量瓶中，用乙腈定容至刻度，混匀。混合标准中间液C中各目标分析物的质量浓度均为 $100\mathrm{mg / L}$ 。临用现配。

# 3.4.5 混合标准工作液系列

# 3.4.5.1 混合标准工作液系列D（适用于橄榄油、异辛烷中各目标分析物的定量分析）

分别准确移取 $0.125\mathrm{mL}, 0.40\mathrm{mL}, 1.0\mathrm{mL}, 2.0\mathrm{mL}, 3.0\mathrm{mL}, 4.0\mathrm{mL}$ 混合标准中间液B于6个 $10\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。所得标准系列工作液中均苯三甲酸的质量浓度分别为 $0.031\mathrm{mg / L}, 0.10\mathrm{mg / L}, 0.25\mathrm{mg / L}, 0.50\mathrm{mg / L}, 0.75\mathrm{mg / L}, 1.0\mathrm{mg / L}$ ，其余6种目标分析物的质量浓度分别为 $0.375\mathrm{mg / L}, 1.2\mathrm{mg / L}, 3.0\mathrm{mg / L}, 6.0\mathrm{mg / L}, 9.0\mathrm{mg / L}, 12\mathrm{mg / L}$ 。临用现配。当试样溶液中

# 3.4.5.2 混合标准工作液系列E（适用于其他食品模拟物、化学替代溶剂中各目标分析物的定量分析）

分别准确移取 $0.10\mathrm{mL},0.125\mathrm{mL},0.40\mathrm{mL},1.0\mathrm{mL},2.0\mathrm{mL},3.0\mathrm{mL},4.0\mathrm{mL}$ 混合标准中间液B于7个 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。所得标准系列工作液中均苯三甲酸的质量浓度分别为 $0.025\mathrm{mg / L},0.031\mathrm{mg / L},0.10\mathrm{mg / L},0.25\mathrm{mg / L},0.50\mathrm{mg / L},0.75\mathrm{mg / L},1.0\mathrm{mg / L}$ ，其余6种目标分析物的质量浓度分别为 $0.30\mathrm{mg / L},0.375\mathrm{mg / L},1.2\mathrm{mg / L},3.0\mathrm{mg / L},6.0\mathrm{mg / L},9.0\mathrm{mg / L},12.0\mathrm{mg / L}$ 。临用现配。当试样溶液中苯甲酸、邻苯二甲酸的质量浓度超出上述线性范围时，可分别准确移取 $1.0~\mathrm{mL}$ 产 $2.0~\mathrm{mL},3.0~\mathrm{mL},4.0~\mathrm{mL},5.0~\mathrm{mL},6.0~\mathrm{mL}$ 混合标准中间液C于6个 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。所得该标准系列工作液中7种目标分析物的质量浓度分别为 $10~\mathrm{mg / L},20~\mathrm{mg / L},30~\mathrm{mg / L}$ $40~\mathrm{mg / L},50~\mathrm{mg / L},60~\mathrm{mg / L}$ 。临用现配。

4.2 水浴振荡器。

# 5 分析步骤

# 5.1 迁移试验

食品接触材料及制品按照GB31604.1和GB5009.156的要求进行迁移试验。迁移试验所得浸泡液如不能立即测试，可置于 $0\% \sim 4\%$ 条件下中避光保存，保存期不超过3d。若进行下一步试验，应将浸泡液恢复至室温后使用。

# 5.2 浸泡液的处理

# 5.2.1 橄榄油浸泡液的处理

5.2.1.1 苯二甲酸、苯三甲酸待测液：准确称取 $5.0\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )迁移试验所得橄榄油浸泡液至 $25~\mathrm{mL}$ 具塞玻璃离心管中，依次加入 $5.00~\mathrm{mL}$ 正庚烷、 $4.00~\mathrm{mL}~50\%$ （体积分数)乙醇溶液， $1500~\mathrm{r / min}$ 涡旋振荡提取 $15\mathrm{min},3000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取下层乙醇溶液经滤膜过滤后，待测。  
5.2.1.2 苯甲酸待测液：准确称取 $10.0\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )迁移试验所得橄榄油浸泡液至 $25~\mathrm{mL}$ 具塞玻璃离心管中，加入 $4.00~\mathrm{mL}$ 乙腈溶液 $(5 + 1)$ ， $1500\mathrm{r / min}$ 涡旋振荡提取 $15\mathrm{min},3000\mathrm{r / min}$ 离心

$10\mathrm{min}$ ，取大约 $2\mathrm{mL}$ 上层溶液经滤膜过滤，准确移取 $1.00~\mathrm{mL}$ 滤液加入 $1.00~\mathrm{mL}$ 水混匀，待测。

# 5.2.2 异辛烷浸泡液的处理

5.2.2.1 苯二甲酸、苯三甲酸待测液：准确移取 $5.00 \mathrm{~mL}$ 迁移试验所得异辛烷浸泡液至 $25 \mathrm{~mL}$ 具塞玻璃离心管中，加入 $4.00 \mathrm{~mL} 50 \%$ （体积分数）乙醇溶液， $1500 \mathrm{r} / \mathrm{min}$ 涡旋振荡提取 $15 \mathrm{~min}, 3000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取下层乙醇溶液经滤膜过滤后，待测。  
5.2.2.2 苯甲酸待测液：准确移取 $10.0\mathrm{mL}$ 迁移试验所得异辛烷浸泡液至 $25\mathrm{mL}$ 具塞玻璃离心管中，加入 $4.00\mathrm{mL}$ 乙腈溶液 $(5 + 1)$ ， $1500\mathrm{r / min}$ 涡旋振荡提取 $15\mathrm{min}$ ， $3000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取约 $2\mathrm{mL}$ 下层溶液经滤膜过滤，准确移取 $1.00\mathrm{mL}$ 滤液加入 $1.00\mathrm{mL}$ 水混匀，待测。

# 5.2.3 $95\%$ （体积分数）乙醇浸泡液的处理

准确移取 $10.0\mathrm{mL}$ 迁移试验所得 $95\%$ （体积分数）乙醇浸泡液于玻璃容器中，旋蒸或氮吹至 $3.0\mathrm{mL}$ 左右，残液转移至 $10\mathrm{mL}$ 容量瓶中，用水洗涤玻璃容器1次～2次，洗涤液合并至容量瓶，用水定容，混匀后，取 $1\mathrm{mL} \sim 2\mathrm{mL}$ 溶液经滤膜过滤后，待测。

# 5.2.4 其他浸泡液的处理

移取 $1\mathrm{mL}\sim 2\mathrm{mL}$ 迁移试验所得浸泡液，经滤膜过滤后，待测。

# 5.6 试液的测定

# 5.6.1 定性测定

按照5.4所列的仪器参考条件对试样待测液(5.2)和混合标准工作液(3.4.5)进行测定，若试样待测液的待测物色谱峰保留时间与标准工作液中相同目标分析物的色谱峰保留时间偏差在 $\pm 0.1\mathrm{min}$ 范围内，则能判断样品中存在相应的待测物。

# 6 分析结果的表述

# 6.1 浸泡液处理过程换算因子的计算

橄榄油和异辛烷浸泡液处理过程换算因子 $f$ 按式(1)进行计算。

$$
f = \frac {V _ {0} \times n}{m} \tag {1}
$$

式中：

$f$ ——浸泡液前处理过程换算因子；

$V_{0}$ ——提取溶液体积，单位为升（L）；

$n$ ——提取溶液稀释倍数，其中苯甲酸待测液稀释倍数为2，其他均为1；

$m$ ——浸泡液的取样量，单位为升(L)或千克 $(\mathrm{kg})$ 。

# 6.2 食品接触材料及制品苯甲酸、苯二甲酸、苯三甲酸特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

食品接触材料及制品苯甲酸、苯二甲酸、苯三甲酸特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(2)进行计算：

$$
X _ {1} = \frac {\left(\rho - \rho_ {0}\right) \times f \times V}{S} \times F \quad \dots \tag {2}
$$

式中：

$X_{1}$ ——目标分析物的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——试样溶液中目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——空白试液中目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

$f$ ——浸泡液中目标分析物前处理过程换算因子，橄榄油和异辛烷浸泡液中 $f$ 按式(1)进行计算，其他浸泡液处理过程不涉及换算， $f$ 值用1表示；

$V$ ——迁移试验中试样浸泡液体积，单位为升 $(\mathrm{L})$ ；

$S$ ——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$F$ ——食品接触材料及制品在可预见使用情形下实际的接触面积与食品体积(质量)比(以下简称

实际 $S / V$ )，单位为平方分米每千克 $(\mathrm{dm}^2 /\mathrm{kg})$ 。各种液态食品通常密度以 $1\mathrm{kg / L}$ 计将其体积换算为质量来计算实际 $S / V$ 。当实际 $S / V$ 已知时， $F$ 即为可预见使用情形下的最大 $S / V$ ；当实际 $S / V$ 未知时， $F$ 采用 $6\mathrm{dm}^2 /\mathrm{kg}$ ，即 $6\mathrm{dm}^2$ 食品接触材料及制品接触 $1\mathrm{kg}$ 食品。

结果保留2位有效数字。

# 6.3 密封制品类食品接触材料及制品苯甲酸、苯二甲酸、苯三甲酸特定迁移量的计算（以 $\mathrm{mg}$ /件表示）

密封制品类食品接触材料及制品苯甲酸、苯二甲酸、苯三甲酸特定迁移量以 $\mathrm{mg}$ /件表示时，按式(3)计算，需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {2} = \frac {\left(\rho - \rho_ {0}\right) \times f \times V}{n} \tag {3}
$$

式中：

$X_{2}$ ——苯甲酸、苯二甲酸、苯三甲酸特定迁移量，单位为毫克每件 $(\mathrm{mg}/$ 件）；

$\rho$ ——试样溶液中目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——空白试液中目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

$f$ ——浸泡液中目标分析物前处理过程换算因子，橄榄油和异辛烷浸泡液中 $f$ 按式(1)进行计算，其他浸泡液处理过程不涉及换算， $f$ 值用1表示；

$V$ ——迁移试验中试样浸泡液体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

$n$ ——迁移试验中浸泡用密封制品的数量，单位为件。

结果保留2位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不应超过其算术平均值的 $10\%$ 。

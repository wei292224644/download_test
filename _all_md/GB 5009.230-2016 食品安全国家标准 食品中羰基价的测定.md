# 食品中炭基价的测定

2016-08-31 发布

2017-03-01 实施

# 前言

本标准代替GB/T5009.37—2003《食用植物油卫生标准的分析方法》中4.3炭基价的测定部分。

本标准与GB/T5009.37—2003中4.3部分相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中炭基价的测定”；

——修改了范围；

——修改了整个实验部分内容。

# 食品安全国家标准

# 食品中基价的测定

# 1 范围

本标准规定了食品中炭基价的测定。

本标准适用于油炸小食品、坚果制品、方便面、膨化食品以及食用植物油等食品中炭基价的测定。

# 2 原理

羰基化合物和2,4-二硝基苯肼的反应产物，在碱性溶液中形成褐红色或酒红色，在 $440~\mathrm{nm}$ 下，测定吸光度，计算羰基价。

# 3 试剂与材料

# 3.1 试剂

3.1.1 乙醇 $\mathrm{(C_2H_6O)}$ ：分析纯。  
3.1.2 苯 $\left(\mathrm{C}_{6} \mathrm{H}_{6}\right)$ ：光谱纯或色谱纯。  
3.1.3 2,4-二硝基苯肼 $\mathrm{(C_6H_6N_4O_4)}$ ：分析纯。  
3.1.4 三氯乙酸 $\mathrm{(C_2H_6Cl_3O_2)}$ ：分析纯。  
3.1.5 氢氧化钾(KOH)：分析纯。  
3.1.6 石油醚 $\left(\mathrm{C}_{5} \mathrm{H}_{12} \mathrm{O}_{2}\right)$ ：分析纯，沸程 $30^{\circ} \mathrm{C} \sim 60^{\circ} \mathrm{C}$ 。  
3.1.7 铝粉（Al）：分析纯。

# 3.2 试剂配制

3.2.1 精制乙醇: 取 $1000 \mathrm{~mL}$ 乙醇, 置于 $2000 \mathrm{~mL}$ 圆底烧瓶中, 加入 $5 \mathrm{~g}$ 铝粉、沸石和 $10 \mathrm{~g}$ 氢氧化钾, 连接标准磨口的回流冷凝管, 水浴中加热回流 $1 \mathrm{~h}$ , 然后用全玻璃蒸馏装置, 蒸馏并收集馏液。  
3.2.2 三氯乙酸溶液：称取 $4.3\mathrm{g}$ 固体三氯乙酸，加 $100~\mathrm{mL}$ 苯溶解。  
3.2.3 2,4-二硝基苯肼溶液：称取 $50\mathrm{mg}2$ ，4-二硝基苯肼，溶于 $100~\mathrm{mL}$ 苯中。  
3.2.4 氢氧化钾-乙醇溶液：称取 $4\mathrm{g}$ 氢氧化钾，加 $100\mathrm{mL}$ 精制乙醇使其溶解；置冷暗处过夜，取上部澄清液使用。溶液变黄褐色则应重新配制。

# 4 仪器和设备

4.1 分光光度计。  
4.2 天平：感量为 $1\mathrm{g},0.1\mathrm{mg}$   
4.3 涡旋混合器。  
4.4 旋转蒸发仪。  
4.5 鼓风式烘箱。

# 5 分析步骤

# 5.1 取样方法

称取含油脂较多的试样 $0.5\mathrm{kg}$ ，含脂肪少的试样取 $1.0\mathrm{kg}$ ，在玻璃研钵中研碎，混合均匀后，按四分法对角取样，放置广口瓶内保存于 $4^{\circ}\mathrm{C}$ 以下冰箱中。

液态油脂类试样根据试样情况取有代表性试样后，放置广口瓶内保存于 $4^{\circ}\mathrm{C}$ 以下冰箱中。

# 5.2 非油脂类试样处理

5.2.1 含油脂高的试样，如油炸花生、坚果等：称取混合均匀的试样 $50\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 带盖广口瓶中，加入 $50~\mathrm{mL}$ 石油醚，放置 $14\mathrm{h} \sim 18\mathrm{h}$ ，用快速滤纸过滤后，室温下，用旋转蒸发器旋蒸 $15\mathrm{min}$ ，减压回收溶剂，得到油脂以供测定。用前保存于 $4^{\circ}\mathrm{C}$ 以下冰箱中。  
5.2.2 含油脂中等的试样，如蛋糕、江米条等：称取混合均匀的试样 $100\mathrm{g}$ ，置于 $500~\mathrm{mL}$ 带盖广口瓶中，加入 $100~\mathrm{mL} \sim 200~\mathrm{mL}$ 石油醚，放置 $14\mathrm{h} \sim 18\mathrm{h}$ ，用快速滤纸过滤后，室温下，用旋转蒸发器旋蒸 $15\mathrm{min}$ ，减压回收溶剂，在 $50^{\circ}\mathrm{C}$ 鼓风干燥箱中挥发石油醚 $1\mathrm{h}$ ，得到油脂以供测定。用前保存于 $4^{\circ}\mathrm{C}$ 以下冰箱中。  
5.2.3 含油脂少的试样，如面包、饼干等：称取混合均匀的试样 $250\mathrm{g}\sim 300\mathrm{g}$ 于 $500~\mathrm{mL}$ 带盖广口瓶，加入适量石油醚浸泡试样，放置 $14\mathrm{h}\sim 18\mathrm{h}$ ，用快速滤纸过滤后，室温下，用旋转蒸发器旋蒸 $15\mathrm{min}$ ，减压回收溶剂，在 $50^{\circ}\mathrm{C}$ 鼓风干燥箱中挥发石油醚 $1\mathrm{h}$ ，得到油脂以供测定。用前保存于 $4^{\circ}\mathrm{C}$ 以下冰箱中。  
5.2.4 含水量较高样品，可加入适量无水硫酸钠，使样品成粒状；易结块样品，可加入4倍～6倍量的海砂，混合均匀后再提取油脂。

# 5.3 测定

称取 $0.025\mathrm{g}\sim 0.5\mathrm{g}$ 油样（精确至 $0.1\mathrm{mg})$ ：羰基价低于 $30~\mathrm{meq / kg}$ 的油样称取 $0.1\mathrm{g}$ ，羰基价 $30\mathrm{meq / kg}\sim 60\mathrm{meq / kg}$ 的油样称取 $0.05\mathrm{g}$ ，羰基价高于 $60~\mathrm{meq / kg}$ 的油样，称取 $0.025\mathrm{g}$ ；置于 $25~\mathrm{mL}$ 具塞试管中，加 $5\mathrm{mL}$ 苯溶解油样，加 $3\mathrm{mL}$ 三氯乙酸溶液及 $5\mathrm{mL}2$ ，4-二硝基苯肼溶液，仔细振摇混匀。

在 $60^{\circ}\mathrm{C}$ 水浴中加热 $30\mathrm{min}$ ，反应后取出用流水冷却至室温，沿试管壁缓慢加入 $10\mathrm{mL}$ 氢氧化钾-乙醇溶液，使成为二液层，涡旋振荡混匀后，放置 $10\mathrm{min}$ 。

以 $1\mathrm{cm}$ 比色杯，用试剂空白调节零点，于波长 $440~\mathrm{nm}$ 处测吸光度。

# 6 分析结果的表述

试样的基价按式(1)进行计算：

$$
X = \frac {A}{8 5 4 \times m} \times 1 0 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——试样的炭基价（以油脂计），单位为毫克当量每千克 $(\mathrm{meq} / \mathrm{kg})$ ；

$A$ ——测定时样液吸光度；

854 ——各种醛的毫克当量吸光系数的平均值；

$m$ ——油样质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

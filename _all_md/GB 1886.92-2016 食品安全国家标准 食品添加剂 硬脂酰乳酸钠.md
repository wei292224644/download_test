# 检验方法

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682中规定的三级水。分析中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603之规定制备。本试验所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

A.2.1 取试样 $1\mathrm{g}$ ，混入水 $25\mathrm{mL}$ 和盐酸 $5\mathrm{mL}$ ，加热，脂肪酸被释出，使油状层浮出液体表面。用水层进行钠试验，应呈阳性反应。

A.2.2 取试样 $25 \mathrm{~g}$ , 置于锥形烧瓶中, 加入 $50 \mathrm{~g}$ 浓度为 $15 \%$ 的氢氧化钾乙醇溶液, 回流 $1 \mathrm{~h}$ , 或至皂化完全。冷却, 加入 $150 \mathrm{~mL}$ 水, 混匀。溶液至完全皂化后, 加入 $60 \mathrm{~mL}$ 浓度为 $0.5 \mathrm{~mol} / \mathrm{L}$ 的硫酸溶液, 加热并不断搅拌至脂肪酸析出并呈透明状清液。用沸水洗涤脂肪酸, 至硫酸盐除尽为止, 收集于小烧杯中, 于蒸汽浴上温热至脂肪酸与水层完全分离, 并透明。冷却, 弃去水层, 将脂肪酸熔融后滤入干燥烧杯中, 在 $105^{\circ} \mathrm{C}$ 下干燥 $20 \mathrm{~min}$ 。该精制脂肪酸的凝固点按常规方法测定应不低于 $54^{\circ} \mathrm{C}$ 。

A.2.3 溶于乙醇和热油脂，能分散于温水中。

# A.3 酸值（以KOH计）的测定

# A.3.1 分析步骤

称取 $1\mathrm{g}$ 试样，置于 $125~\mathrm{mL}$ 锥形烧瓶中，加入 $25~\mathrm{mL}$ 中性乙醇（在乙醇中加入1滴酚酞试液，用 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠标准溶液中和至粉红色，并保持 $30~\mathrm{s}$ 不褪色)，加热使试样溶解，冷却至室温，再加5滴酚酞试液，然后快速用 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠标准溶液滴定至呈粉红色，并维持 $30~\mathrm{s}$ 不褪色为终点。

# A.3.2 结果计算

酸值（以KOH计） $\omega_{1}$ 毫克每克 $(\mathrm{mg / g})$ ，按式(A.1)计算：

$$
w _ {1} = \frac {M \times V \times c}{m} \tag {A.1}
$$

式中：

$M$ ——氢氧化钾的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(\mathrm{KOH}) = 56.10]$

$V$ ——滴定时消耗的 $0.1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——滴定过程中所用氢氧化钠标准溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。

# A.4 钠的测定

# A.4.1 分析步骤

# A.4.1.1 镧贮备液的制备

称取 $5.86\mathrm{g}$ 氧化镧 $\mathrm{(La_2O_3)}$ ，放入 $100~\mathrm{mL}$ 容量瓶中，用水湿润，缓慢加入 $25~\mathrm{mL}$ 盐酸，旋转至溶解后用水定容至刻度，混匀。

# A.4.1.2 钠贮备液的制备

称取预先在 $105\mathrm{^\circ C}$ 下干燥 $2\mathrm{h}$ 后的氯化钠 $1.271\mathrm{g}$ ，放入 $500~\mathrm{mL}$ 容量瓶中，用水定容至刻度后，混匀。钠贮备液每毫升含钠 $1\mathrm{mg}(1000\mathrm{mg / kgNa})$ 。

# A.4.1.3 标准溶液的制备

取3只 $100~\mathrm{mL}$ 容量瓶，各加 $10.0~\mathrm{mL}$ 镧贮备液。用微升注射器在第一只容量瓶中加入 $0.20~\mathrm{mL}$ 钠贮备液，在第二只中加入 $0.40~\mathrm{mL}$ ，在第三只中加入 $0.50~\mathrm{mL}$ ，然后各用水稀释定容后混匀。各容量瓶中每毫升分别含钠 $2.0~\mu \mathrm{g}, 4.0~\mu \mathrm{g}$ 和 $5.0~\mu \mathrm{g}$ 。需当天制备。

# A.4.1.4 试样溶液的制备

准确称取 $250\mathrm{mg}$ 试样，放入 $30~\mathrm{mL}$ 烧杯中，加 $10~\mathrm{mL}$ 热乙醇，溶解后移入 $25~\mathrm{mL}$ 容量瓶中，用乙醇洗涤烧杯2次，每次 $5\mathrm{mL}$ ，洗液并入容量瓶中，再用乙醇定容后混匀。另取镧贮备液 $2.5\mathrm{mL}$ ，放入第二只 $25~\mathrm{mL}$ 容量瓶中，用微升注射器取第一只容量瓶中的试样乙醇液 $0.25~\mathrm{mL}$ ，加入第二只容量瓶中，再用水定容后混合。

# A.4.2 分析和结果计算

用一适当的原子吸收分光光度计，按仪器规定的操作步骤测定各标准溶液和试样溶液在波长 $589~\mathrm{nm}$ 处的吸光度。按各标准溶液不同的钠浓度和吸光度绘制标准曲线，据此求出试样溶液中的钠浓度 $c(\mu \mathrm{g / mL})$ ，然后按2.5倍钠的浓度求出试样中的钠含量 $(\mathrm{mg})$ 。

# A.4.3 注意事项

在此试验中不能使用常规玻璃仪器，以免被钠污染，必要时，使用适宜的塑料(如聚乙烯)容器。

# A.5 酯值的测定

# A.5.1 分析步骤

# A.5.1.1 试样溶液的制备

A.3 酸值分析中所保留的中和液。

# A.5.1.2 测定

取酸值分析中所保留的中和液，加入 $10\mathrm{mL}$ 氢氧化钾乙醇溶液（将 $11.2\mathrm{g}$ 氢氧化钾溶于 $250~\mathrm{mL}$ 乙醇中，再用 $25~\mathrm{mL}$ 水稀释）。加入5滴酚酞指示液，装上冷凝器，回流 $2\mathrm{h}$ ，冷却后再加5滴酚酞指示液，用 $0.05\mathrm{mol} / \mathrm{L}$ 硫酸溶液滴定过量的碱。预先用 $10.0~\mathrm{mL}$ 氢氧化钾乙醇溶液做空白试验。

# A.5.2 结果计算

酯值 $\omega_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {M \times \left(V _ {0} - V _ {1}\right) \times c _ {1}}{m _ {1}} \quad \dots \tag {A.2}
$$

式中：

$M$ ——氢氧化钾的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(\mathrm{KOH}) = 56.10]$

$V_{0}$ ——滴定试样所消耗 $0.05 \mathrm{~mol} / \mathrm{L}$ 硫酸溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——滴定空白液所消耗 $0.05 \mathrm{~mol} / \mathrm{L}$ 硫酸溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——滴定过程中所使用硫酸溶液的摩尔浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。

# A.6 总乳酸量的测定

# A.6.1 分析步骤

# A.6.1.1 标准储备液的制备

称取 $1.067\mathrm{g}$ 乳酸锂 $\mathrm{(LiC_3H_5O_3)}$ ，用水溶解后定容至 $1000~\mathrm{mL}$ 。

# A.6.1.2 标准溶液的制备

分别吸取标准储备液 $1.0\mathrm{mL},2.0\mathrm{mL},4.0\mathrm{mL},6.0\mathrm{mL}$ 和 $8.0~\mathrm{mL}$ ，各放入 $100~\mathrm{mL}$ 容量瓶中，用水定容并混合。分别吸取 $1.0~\mathrm{mL}$ 上述溶液，置于5个试管中，试管中的标准液中的乳酸浓度分别为 $1\mu \mathrm{g / mL},2\mu \mathrm{g / mL},4\mu \mathrm{g / mL},6\mu \mathrm{g / mL}$ 和 $8\mu \mathrm{g / mL}$

# A.6.1.3 试样溶液的制备

准确称取 $200\mathrm{mg}$ 试样，放入 $125~\mathrm{mL}$ 锥形烧瓶中，加 $0.5\mathrm{mol} / \mathrm{L}$ 氢氧化钾乙醇溶液 $10~\mathrm{mL}$ 和水 $10~\mathrm{mL}$ ，装上空气冷凝器，缓慢回流 $45\mathrm{min}$ ，用水约 $40~\mathrm{mL}$ 洗涤冷凝器和瓶边，在蒸汽浴上加热至无乙醇气味，加 $1:2$ 稀硫酸 $6\mathrm{mL}$ ，加热至脂肪酸融化，然后冷却至约 $60^{\circ}\mathrm{C}$ ，加 $25~\mathrm{mL}$ 石油醚，缓慢旋动混合物后立即移入分液漏斗，将水层收集于 $100~\mathrm{mL}$ 容量瓶中。用水洗涤石油醚层两次，每次 $20~\mathrm{mL}$ ，洗液并入容量瓶。用水稀释至 $100~\mathrm{mL}$ 并混合。取此溶液 $1.0~\mathrm{mL}$ ，放入 $100~\mathrm{mL}$ 容量瓶中，用水定容后混合。

# A.6.2 分析和结果计算

吸取 $1.0\mathrm{mL}$ 试样溶液，置于试管中。另取 $1.0\mathrm{mL}$ 水放入试管中作为对照。对此两试管溶液和A.6.1.2中5个装有标准溶液的试管按如下操作进行处理：

a) 加入 1 滴硫酸铜试液, 迅速加入 $9.0 \mathrm{~mL}$ 硫酸, 缓慢转动, 松开试管塞, 在 $90^{\circ} \mathrm{C}$ 水浴中加热 5 min 后立即在冰水浴中冷却 $5 \mathrm{~min}$ 至 $20^{\circ} \mathrm{C}$ 以下, 加 3 滴对苯基酚试液, 立即摇动, 于 $30^{\circ} \mathrm{C}$ 水浴中加热 $30 \mathrm{~min}$ , 在这段时间内摇动两次以使试剂分散。  
b) 将试管放入 $90^{\circ} \mathrm{C}$ 水浴中维持 $90 \mathrm{~s}$ , 取出后立即放入冰水中迅速冷却至室温。盛于 $1 \mathrm{~cm}$ 吸收池中, 用适当的分光光度计在 $570 \mathrm{~nm}$ 处测定吸光度。据此对照标准曲线即可求得试样溶液中的乳酸量 $(\mu \mathrm{g})$ 。

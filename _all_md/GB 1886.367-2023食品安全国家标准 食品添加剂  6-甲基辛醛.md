# 食品添加剂 6-甲基辛醛

# 1 范围

本标准适用于以6-甲基辛醇为原料，在二氯甲烷溶剂中由催化剂2，2，6，6-四甲基哌啶氧化物催化氧化后除去溶剂，真空蒸馏制得的食品添加剂6-甲基辛醛。

# 2 分子式、结构式和相对分子质量

# 2.1 分子式

$$
\mathrm {C} _ {9} \mathrm {H} _ {1 8} \mathrm {O}
$$

# 附录A

# 6-甲基辛醛含量的测定

# A.1 仪器和设备

A.1.1 气相色谱仪：按GB/T11538—2006中第5章的规定。

A.1.2 色谱柱：按B.2中毛细管柱规定，或采用同等效能的色谱柱。

A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

试样制备：精密量取试样 $1.00\mathrm{mL}$ ，置于 $50~\mathrm{mL}$ 容量瓶中，加乙腈(分析纯)稀释至刻度，摇匀备用。

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行。

# B.2 操作条件

B.2.1 色谱柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ 。  
B.2.2 固定相： $5\%$ 苯基-甲基聚硅氧烷。  
B.2.3 膜厚: $0.25 \mu \mathrm{m}$ 。  
B.2.4 柱温升温程序：初始温度 $60^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$ ；以 $20^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $100^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ ；以 $10^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $250^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ 。  
B.2.5 进样口温度： $230^{\circ}C$ 。  
B.2.6 检测器温度： $270^{\circ}C$   
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气；恒流模式，柱流速为 $1.0~\mathrm{mL / min}$   
B.2.9 氢气流量： $40~\mathrm{mL / min}$   
B.2.10 空气流量： $400~\mathrm{mL / min}$   
B.2.11 柱前压： $67\mathrm{kPa}$ （相当于10psi）。  
B.2.12 进样量： $1.0\mu \mathrm{L}$   
B.2.13 分流比： $100:1$

# 1 范围

本标准适用于以葡萄糖酸或葡萄糖酸-δ-内酯与硫酸亚铁、葡萄糖酸与碳酸亚铁或还原铁粉为主要原料，经化学反应、加工、纯化而制得的食品营养强化剂葡萄糖酸亚铁。本标准同样适用于食品添加剂葡萄糖酸亚铁。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

$D$ -葡萄糖酸二价铁盐

# 2.2 分子式

$\mathrm{C_{12}H_{22}FeO_{14}}\bullet n\mathrm{H_2O}(n = 0$ 或2）

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂及其制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 冰乙酸。

A.2.1.2 乙醇。

A.2.1.3 苯肼。

A.2.1.4 铁氰化钾溶液： $100\mathrm{g / L}$ ，临用前制备。

# A.2.2 鉴别方法

# A.2.2.1 溶解性

易溶于热水，几乎不溶于乙醇。

# A.2.2.2 亚铁离子的鉴别

称取约 $0.5\mathrm{g}$ 试样，溶于 $100~\mathrm{mL}$ 水中，滴加铁氰化钾溶液，生成深蓝色沉淀。

# A.2.2.3 葡萄糖的鉴别

# A.2.2.3.1 方法提要

试样在乙酸介质中，与苯肼共热，生成黄色葡萄糖酰苯肼结晶。

# A.2.2.3.2 分析步骤

称取约 $0.5\mathrm{g}$ 试样，加 $10~\mathrm{mL}$ 水，溶解（必要时加热），取 $5\mathrm{mL}$ ，加 $0.7\mathrm{mL}$ 冰乙酸和 $1\mathrm{mL}$ 新蒸馏苯肼，在水浴上加热 $30\mathrm{min}$ ，放至室温，用玻璃棒摩擦试管内壁，则析出黄色结晶。

# A.3 葡萄糖酸亚铁 $\left(\mathrm{C}_{12} \mathrm{H}_{22} \mathrm{FeO}_{14}\right)$ 含量（以干基计）的测定

# A.3.1 方法提要

在酸性介质中，用硫酸铈标准滴定溶液滴定，以1，10-菲啰啉指示液指示终点。

# A.3.2 试剂和材料

A.3.2.1 锌粉。

# A.3.2.2 硫酸。

A.3.2.3 硫酸溶液： $c(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4) = 2\mathrm{mol} / \mathrm{L}$ 。量取 $60~\mathrm{mL}$ 硫酸，缓缓注入 $1000~\mathrm{mL}$ 水中，冷却，摇匀。

A.3.2.4 硫酸铈标准滴定溶液： $c(\mathrm{CeSO}_4) = 0.1\mathrm{mol / L}$

A.3.2.5 1,10-菲啰啉指示液：称取 $0.5\mathrm{g}$ 硫酸亚铁，加 $100~\mathrm{mL}$ 水溶解，加2滴硫酸溶液与 $0.5\mathrm{g}$ 1,10-菲啰啉，摇匀，溶液保存于密闭容器内。

# A.3.3 分析步骤

称取约 $1.5\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于 $500~\mathrm{mL}$ 锥形瓶中，加 $75~\mathrm{mL}$ 水和 $15~\mathrm{mL}$ 硫酸溶液溶解，加 $250~\mathrm{mg}$ 锌粉直至溶液脱色，以小烧杯盖住锥形瓶，置室温放置 $20\mathrm{min}$ 。用表面铺有薄薄一层锌粉的垂熔玻璃漏斗过滤（或真空抽滤），以 $10~\mathrm{mL}$ 硫酸溶液和 $10~\mathrm{mL}$ 水分别洗涤垂熔玻璃漏斗，合并滤液和洗涤液，以1,10-菲啰啉指示液为指示剂，迅速以硫酸铈标准滴定溶液滴定至溶液颜色由橘黄色转变为绿色。同时进行空白试验。

# A.3.4 结果计算

葡萄糖酸亚铁 $\mathrm{(C_{12}H_{22}FeO_{14})}$ 含量(以干基计)的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\frac {V _ {1} - V _ {0}}{1 000} \times c _ {1} \times M _ {1}}{m _ {1} \times \left(1 - w _ {2}\right)} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——滴定试样溶液消耗的硫酸铈标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白溶液消耗的硫酸铈标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$c_{1}$ ——硫酸铈标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{1}$ ——葡萄糖酸亚铁的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M_1(\mathrm{C}_{12}\mathrm{H}_{22}\mathrm{FeO}_{14}) = 446.15]$

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$w_{2}$ ——实测试样干燥减量的质量分数， $\%$ 。

取两次平行测定结果的算术平均值为准。两次平行测定结果的绝对差值不大于 $0.3\%$ 。

# A.4 三价铁（以 $\mathrm{Fe}^{3+}$ 计）的测定

# A.4.1 试剂和材料

A.4.1.1 碘化钾。

A.4.1.2 盐酸。

A.4.1.3 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol} / \mathrm{L}$

A.4.1.4 淀粉指示液： $10\mathrm{g / L}$

# A.4.2 分析步骤

称取 $5\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于碘量瓶中，加 $100~\mathrm{mL}$ 水、 $10~\mathrm{mL}$ 盐酸溶解，加入 $3\mathrm{g}$ 碘化钾，水封瓶塞，振摇溶解，于暗处放置约 $5\mathrm{min}$ ，起塞并用少量水冲洗塞盖，加入 $0.5\mathrm{mL}$ 淀粉指示液，迅速以硫代硫酸钠标准滴定溶液滴定至溶液蓝色消失为终点，即得。同时进行空白试验。

# A.4.3 结果计算

三价铁（以 $\mathrm{Fe}^{3+}$ 计）的质量分数 $w_{3}$ ，按式(A.2)计算：

$$
w _ {3} = \frac {\frac {V _ {3} - V _ {2}}{1 000} \times c _ {2} \times M _ {2}}{m _ {2}} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$V_{3}$ ——滴定试样溶液消耗的硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴定空白溶液消耗的硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$c_{2}$ ——硫代硫酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{2}$ ——铁的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M_2(\mathrm{Fe}) = 55.85]$

$m_{2}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

取两次平行测定结果的算术平均值为准。两次平行测定结果的绝对差值不大于 $0.1\%$ 。

# A.5 还原糖的测定

# A.5.1 试剂和材料

A.5.1.1 无水碳酸钠 $\mathrm{(Na_2CO_3)}$   
A.5.1.2 硫化氢气体：取少量硫铁矿细粉，加稀盐酸适量，将产生硫化氢气体导出。  
A.5.1.3 盐酸： $36\%$   
A.5.1.4 氨水： $25\%$   
A.5.1.5 费林溶液。  
A.5.1.6 盐酸溶液： $3\mathrm{mol} / \mathrm{L}$ 。取 $258~\mathrm{mL}~36\%$ 盐酸，加水稀释至 $1\mathrm{L}$   
A.5.1.7 氨水溶液： $6\mathrm{mol} / \mathrm{L}$ 。取 $450~\mathrm{mL}25\%$ 氨水，加水稀释至 $1\mathrm{L}$   
A.5.1.8 碳酸钠溶液： $10.5\%$ 。称取 $10.5\mathrm{g}$ 无水碳酸钠 $\mathrm{(Na_2CO_3)}$ ，加 $89.5\mathrm{g}$ 水溶解。  
A.5.1.9 乙酸铅试纸。

# A.5.2 分析步骤

称取 $0.5\mathrm{g}$ 试样，加 $10~\mathrm{mL}$ 水溶解，温热，加入 $1\mathrm{mL}$ 氨溶液使溶液呈碱性，通入硫化氢气体沉淀离子，静置 $30\mathrm{min}$ ，过滤，以 $10~\mathrm{mL}$ 水分两次洗涤，合并滤液和洗涤液，用盐酸酸化，再加入 $2\mathrm{mL}$ 盐酸溶液，煮沸，直至蒸气不使润湿的乙酸铅试纸变黑，继续煮沸使溶液体积约为 $10~\mathrm{mL}$ ，冷却，加入 $5\mathrm{mL}$ 碳酸钠溶液和 $20~\mathrm{mL}$ 水，过滤，滤液加水稀释至 $100~\mathrm{mL}$ 。吸取溶液 $5\mathrm{mL}$ ，加入 $2\mathrm{mL}$ 费林溶液，煮沸 $1\mathrm{min}$ 不应产生红色沉淀，即为通过试验。

# A.6 硫酸盐（以 $\mathrm{SO}_4$ 计）的测定

# A.6.1 试剂和材料

A.6.1.1 盐酸溶液： $1 + 3$   
A.6.1.2 氯化钡溶液： $250~\mathrm{g / L}$   
A.6.1.3 硫酸钾标准溶液： $0.1\mathrm{mg / mL}$

# A.6.2 分析步骤

称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加 $40\mathrm{mL}$ 水溶解，用盐酸溶液调节溶液至中性，用水稀释至 $50\mathrm{mL}$ ，振摇，过滤，保留滤液。取 $25\mathrm{mL}$ 滤液，作为试样溶液，置于 $50\mathrm{mL}$ 纳氏比色管中，加 $2\mathrm{mL}$ 盐酸溶液，

加水至体积约 $40~\mathrm{mL}$ ，加 $5\mathrm{mL}$ 氯化钡溶液，用水稀释至 $50~\mathrm{mL}$ ，摇匀，放置 $10\mathrm{min}$ ，在黑色背景下，轴向观察，试样溶液所呈浊度不应大于标准比浊溶液。

标准比浊溶液：量取 $5.0\mathrm{mL}$ 硫酸钾标准溶液，置于 $50~\mathrm{mL}$ 比色管中。与试样溶液同时同样处理。

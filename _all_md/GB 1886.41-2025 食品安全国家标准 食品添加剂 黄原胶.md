# 前言

本标准代替GB1886.41—2015《食品安全国家标准 食品添加剂 黄原胶》。

本标准与GB1886.41—2015《食品安全国家标准 食品添加剂 黄原胶》相比，主要变化如下：

——修改了黄原胶的范围；  
——增加了黄原胶含量指标及检验方法；  
——修改了总氮的检验方法；  
——增加了异丙醇指标及检验方法；  
——增加了砷指标及检验方法；  
——修改了微生物检验方法的前处理方法。

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 溶解性试验

称取 $1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，慢慢倾入装有 $100\mathrm{mL}$ 水的烧杯中，开启搅拌器至转速 $200\mathrm{r / min}$ ，边搅拌边加入试样， $25\mathrm{min}$ 后应可溶解。按该方法将试样加入乙醇、丙酮或乙醚中不溶解。

# A.2.2 凝胶试验

加 $300~\mathrm{mL}$ 水于 $500~\mathrm{mL}$ 烧杯中，预热至 $80^{\circ}C$ ，开启搅拌器至转速 $200~\mathrm{r / min}$ ，边搅拌边加入干燥试样和槐豆胶各 $1.5\mathrm{g}$ （精确到 $0.01\mathrm{g})$ ，至混合物形成溶液后，继续搅拌 $30\mathrm{min}$ 以上（搅拌过程中水温不低于 $60^{\circ}C)$ 。停止搅拌，在室温下至少冷却 $2\textrm{h}$ ，当温度降到低于 $40^{\circ}C$ 时，形成凝胶状物。按该方法制备 $1\%$ 的试样溶液作为对照，不加槐豆胶，无此胶状物出现。

# A.3 黄原胶含量（以干基计）的测定

# A.3.1 方法提要

黄原胶试样经低浓度的氢氧化钾和盐酸溶液处理后，先用无水乙醇沉淀，再用无水乙醇洗涤去除杂质后过滤，滤渣经干燥称重，计算黄原胶含量。

# A.3.2 试剂和材料

A.3.2.1 氢氧化钾溶液 $(0.04\mathrm{g / mL})$ ：称取 $4\mathrm{g}$ 氢氧化钾，加水溶解并定容至 $100~\mathrm{mL}$ 。  
A.3.2.2 稀盐酸溶液 $(1 + 3$ ，体积比)：吸取 $10~\mathrm{mL}$ 盐酸，加入 $30~\mathrm{mL}$ 水，混匀。  
A.3.2.3 无水乙醇。  
A.3.2.4 丙酮。  
A.3.2.5 硅藻土545。  
A.3.2.6 硝酸银试液 $(17\mathrm{g} / \mathrm{L})$ ：称取 $1.7\mathrm{g}$ 硝酸银，加水溶解并定容至 $100~\mathrm{mL}$ 。

# A.3.3 仪器和设备

A.3.3.1 离心机。  
A.3.3.2 真空干燥箱。  
A.3.3.3 玻璃过滤器（G3）。  
A.3.3.4 干燥器。  
A.3.3.5 电子天平：感量为 $0.001\mathrm{g}$

# A.3.4 分析步骤

在玻璃过滤器(G3)中加入硅藻土545，约 $0.5\mathrm{cm}$ 厚度，置于真空干燥箱内，于 $80^{\circ}\mathrm{C}, 40\mathrm{kPa} \sim$

$53\mathrm{kPa}$ 条件下干燥，取出置于干燥器中冷却后称重，并重复以上干燥、称量操作，至前后两次质量之差不超过 $2\mathrm{mg}$ ，即为恒重，质量记为 $m_{1}$ ，精确至 $0.001\mathrm{g}$ 。准确称取干燥失重测定后的试样 $0.5\mathrm{g}$ ，精确至 $0.001\mathrm{g}$ ，质量记为 $m$ 。加 $10\mathrm{mL}$ 无水乙醇使样品充分分散，再加 $10\mathrm{mL}$ 氢氧化钾溶液 $(0.04\mathrm{g} / \mathrm{mL})$ 溶解，并加水 $90\mathrm{mL}$ 。向该溶液中加 $15\mathrm{mL}$ 稀盐酸 $(1 + 3)$ 和 $300\mathrm{mL}$ 无水乙醇，并剧烈搅拌。静置 $2\mathrm{h}$ 后， $4000\mathrm{r} / \mathrm{min}$ 离心 $10\mathrm{min}$ ，除去上清液。再次加入适量无水乙醇，重复上述离心、除去上清液步骤，直至上清液中没有氯化物（倾出约 $10\mathrm{mL}$ 上清液于烧杯中，加入 $0.5\mathrm{mL}$ 硝酸银试液，无浑浊现象，则上清液中无氯化物）。用上述恒重后的玻璃过滤器过滤沉淀，使用无水乙醇洗涤。最终残留物用 $30\mathrm{mL}$ 丙酮清洗后，先将带有残留物的玻璃过滤器放置在通风橱内约 $30\mathrm{min}$ 以上，再于 $80^{\circ}\mathrm{C}, 40\mathrm{kPa} \sim 53\mathrm{kPa}$ 条件下真空干燥 $4\mathrm{h}$ 后，取出置于干燥器中冷却并干燥至恒重，质量记为 $m_{2}$ ，精确至 $0.001\mathrm{g}$ 。

# A.3.5 结果计算

黄原胶含量 $w_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {m _ {2} - m _ {1}}{m} \times 100 \% \tag{A.1}
$$

式中：

$m_{2}$ ——玻璃过滤器、硅藻土和残留物干燥后的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——玻璃过滤器和硅藻土干燥后的质量，单位为克 $(\mathrm{g})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以两次平行测定结果的算术平均值表示。结果保留到小数点后一位。在重复性条件下获得的两次独立测定结果的绝对值不大于算术平均值的 $3.0\%$ 。

# A.4 黏度的测定

# A.4.1 仪器和设备

旋转黏度计。

# A.4.2 测定条件

A.4.2.1 转子型号：3号转子。  
A.4.2.2 转子转速： $60\mathrm{r / min}$   
A.4.2.3 测定温度： $24^{\circ}C\sim 25^{\circ}C$

# A.4.3 分析步骤

# A.4.3.1 试样溶液制备

分别准确称取 $3\mathrm{g}$ 试样和氯化钾，精确至 $0.01\mathrm{g}$ ，混合后在开启搅拌器的条件下缓慢加入盛有 $294\mathrm{g}$ 蒸馏水的 $400\mathrm{mL}$ 烧杯中。避免混合物粘到搅拌叶或烧杯壁上，并以 $800\mathrm{r / min}$ 连续搅拌 $2\mathrm{h}$ ，温度保持 $24^{\circ}\mathrm{C} \sim 25^{\circ}\mathrm{C}$ ；停止搅拌，取出烧杯，用搅拌棒或其他类似物上下翻动溶液至均匀无明显气泡。

# A.4.3.2 测定

用旋转黏度计在A.4.2规定的条件下测定试样溶液的黏度值。

# A.5 剪切性能值的测定

# A.5.1 测定方法

按A.4分别测定3号转子在转速 $6\mathrm{r / min}$ 和 $60~\mathrm{r / min}$ 时的黏度值，按式(A.2)计算剪切性能值。

# A.5.2 结果计算

剪切性能值 $N$ ，按式(A.2)计算：

$$
N = \frac {\eta_ {1}}{\eta_ {2}} \tag {A.2}
$$

式中：

$\eta_{1}$ ——转速 $6 \mathrm{r} / \mathrm{min}$ 时的黏度值，单位为厘泊(cP)；

$\eta_{2}$ ——转速 $60~\mathrm{r / min}$ 时的黏度值，单位为厘泊(cP)。

计算结果保留至小数点后一位。

# A.6 干燥失重的测定

# A.6.1 方法提要

在一定温度条件下将试样烘干至恒重，计算失去的物质质量。

# A.6.2 仪器和设备

A.6.2.1 玻璃制称量瓶：内径 $60~\mathrm{mm}\sim 70~\mathrm{mm}$ ，高 $35~\mathrm{mm}$ 以下。  
A.6.2.2 电热恒温干燥箱。  
A.6.2.3 电子天平：感量为 $0.0001\mathrm{g}$

# A.6.3 分析步骤

称取 $1.0\mathrm{g}\sim 2.0\mathrm{g}$ （精确至 $0.0001\mathrm{g})$ 的试样于已烘至恒重的称量瓶中，加盖，侧摇，使试样在称量瓶中均匀分布，将载物的称量瓶放入干燥箱中，打开瓶盖，将瓶盖留在干燥箱内，在 $105~^\circ \mathrm{C}\pm 1~^\circ \mathrm{C}$ 下干燥 $2.5\mathrm{h}$ ，将带试样的称量瓶立即盖上盖子，放入干燥器中冷却至室温后称量。然后再放入干燥箱中干燥 $1\textrm{h}$ ，取出，放入干燥器内冷却至室温后称量，并重复干燥及称重操作直至恒重。根据减轻的质量和取样量计算干燥失重。

# A.6.4 结果计算

干燥失重的质量分数 $\varpi_{2}$ ，按式(A.3)计算：

$$
w _ {2} = \frac {m _ {3} - m _ {4}}{m} \times 100 \% \quad \dots \tag{A.3}
$$

式中：

$m_{3}$ ——烘干前称量瓶和试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{4}$ ——烘干后称量瓶和试样的质量，单位为克 $(\mathrm{g})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准，结果保留三位有效数字。平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.7 丙酮酸的测定

# A.7.1 方法提要

黄原胶经过盐酸酸解后释放出丙酮酸，丙酮酸与2，4-二硝基苯肼反应生成丙酮酸-2，4-二硝基苯腙，在碳酸钠溶液中显棕红色，通过测定试样溶液与丙酮酸标准溶液的吸光度值判定试样的丙酮酸含量。

# A.7.2 试剂和材料

A.7.2.1 丙酮酸。

A.7.2.2 2,4-二硝基苯肼。  
A.7.2.3 乙酸乙酯。  
A.7.2.4 $1\mathrm{mol} / \mathrm{L}$ 盐酸溶液：量取 $9\mathrm{mL}$ 盐酸，加水定容至 $100~\mathrm{mL}$ 。 $2\mathrm{mol} / \mathrm{L}$ 盐酸：吸取 $18~\mathrm{mL}$ 盐酸，加水定容至 $100~\mathrm{mL}$ 。  
A.7.2.5 $0.1\mathrm{mol} / \mathrm{L}$ 碳酸钠标准溶液：称取 $5.3\mathrm{g}$ 碳酸钠，加水溶解并定容至 $1000~\mathrm{mL}$

# A.7.3 仪器和设备

A.7.3.1 分光光度计。  
A.7.3.2 电子天平：感量为 $0.0001\mathrm{g}$ 。

# A.7.4 标准溶液制备

准确称取 $45.0\mathrm{mg}$ 丙酮酸(精确至 $0.1\mathrm{mg}$ )，移入 $500~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，混匀。取该溶液 $10\mathrm{mL}$ 置于 $50~\mathrm{mL}$ 具塞烧瓶中，吸取 $20~\mathrm{mL}~1~\mathrm{mol / L}$ 盐酸加入烧瓶中，称量烧瓶，回流加热 $3\mathrm{h}$ 。冷却至室温，并补充回流过程中所失去的水分。移取 $1\mathrm{mL}2,4-$ 二硝基苯肼的盐酸溶液（1：200，盐酸溶液为 $2\mathrm{mol / L}$ ）于 $30~\mathrm{mL}$ 分液漏斗中，加入 $2\mathrm{mL}$ 具塞烧瓶中经回流处理的溶液，混匀，置室温下 $5\mathrm{min}$ ，先用 $5\mathrm{mL}$ 乙酸乙酯萃取，弃去水层，再用 $5\mathrm{mL}$ 碳酸钠标准溶液萃取乙酸乙酯中的腙，萃取三次，收集萃取液置于 $50~\mathrm{mL}$ 容量瓶中，用碳酸钠标准溶液稀释至刻度。

# A.7.5 试样溶液制备

准确称取 $600.0\mathrm{mg}$ 试样(精确至 $0.1\mathrm{mg}$ )，移入 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，混匀。取该溶液 $10~\mathrm{mL}$ 置于 $50~\mathrm{mL}$ 具塞烧瓶中，吸取 $20~\mathrm{mL}~1~\mathrm{mol} / \mathrm{L}$ 盐酸加入烧瓶中，称量烧瓶，回流加热 $3\mathrm{h}$ 。冷却至室温，并补充回流过程中所失去的水分。移取 $1\mathrm{mL}2,4-$ 二硝基苯肼的盐酸溶液（1：200，盐酸溶液为 $2\mathrm{mol} / \mathrm{L}$ ）于 $30~\mathrm{mL}$ 分液漏斗中，加入 $2\mathrm{mL}$ 具塞烧瓶中经回流处理的溶液，混匀，置室温下 $5\mathrm{min}$ ，先用 $5\mathrm{mL}$ 乙酸乙酯萃取，弃去水层，再用 $5\mathrm{mL}$ 碳酸钠标准溶液萃取乙酸乙酯中的腙，萃取三次，收集萃取液置于 $50~\mathrm{mL}$ 容量瓶中，用碳酸钠标准溶液稀释至刻度。

# A.7.6 测定

在适宜的分光光度计上将待测样品装入 $1\mathrm{cm}$ 比色皿，以碳酸钠标准溶液为空白，测定试样溶液和标准溶液在 $375~\mathrm{nm}$ 处的吸光度值。试样溶液的吸光度不低于标准溶液的吸光度，则试样的丙酮酸含量不低于 $1.5\%$ 。

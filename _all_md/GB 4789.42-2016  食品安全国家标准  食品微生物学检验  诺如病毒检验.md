# 前言

本标准代替SN/T1635—2005《贝类中诺沃克病毒检测方法普通RT-PCR方法和实时荧光RT-PCR方法》。

本标准与SN/T1635—2005相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品微生物学检验 诺如病毒检验”；  
——标准检测范围从“贝类”扩增为“食品”；  
——修改“操作步骤”；  
——增加“质量控制要求”，可参见附录C；  
——删除“普通RT-PCR方法”。

# 1 范围

本标准规定了食品中诺如病毒(Norovirus)的实时荧光RT-PCR检测方法。

本标准适用于贝类，生食蔬菜，胡萝卜、瓜、坚果等硬质表面食品，草莓、西红柿、葡萄等软质水果等食品中诺如病毒核酸的检测。

# 5 操作步骤

# 5.1 病毒提取

注：样品处理一般应在 $4^{\circ}\mathrm{C}$ 以下的环境中进行运输。实验室接到样品后应尽快进行检测，如果暂时不能检测应将样品保存在 $-80^{\circ}\mathrm{C}$ 冰箱中，试验前解冻。样品处理和PCR反应应在单独的工作区域或房间进行。每个样品可设置 $2\sim 3$ 个平行处理。

# 5.1.1 软质水果和生食蔬菜

5.1.1.1 将 $25\mathrm{g}$ 软质水果或生食蔬菜切成约 $2.5\mathrm{cm} \times 2.5\mathrm{cm} \times 2.5\mathrm{cm}$ 的小块（如水果或蔬菜小于该体积，可不切）。  
5.1.1.2 将样品小块移至带有 $400~\mathrm{mL}$ 网状过滤袋的样品袋，加入 $40~\mathrm{mL}$ TGBE溶液（软质水果样品，

需加入30U A.niger果胶酶，或1140U A.aculeatus果胶酶)，加入 $10~\mu \mathrm{L}$ 过程控制病毒。

5.1.1.3 室温，60次/min，振荡 $20\mathrm{min}$ 。酸性软质水果需在振荡过程中，每隔 $10\mathrm{min}$ 检测 $\mathrm{pH}$ ，如 $\mathrm{pH}$ 低于9.0时，使用 $1\mathrm{mol} / \mathrm{LNaOH}$ 调 $\mathrm{pH}$ 至9.5，每调整一次 $\mathrm{pH}$ ，延长振荡时间 $10\mathrm{min}$ 。  
5.1.1.4 将振荡液转移至离心管，如体积较大，可使用2根离心管。 $10000\mathrm{r / min},4^{\circ}\mathrm{C}$ ，离心 $30\mathrm{min}$ 取上清至干净试管或三角瓶，用 $1\mathrm{mol} / \mathrm{L}$ HCl调 $\mathrm{pH}$ 至7.0。  
5.1.1.5 加入0.25倍体积 $5\times \mathrm{PEG / NaCl}$ 溶液，使终溶液浓度为 $100\mathrm{g / L}$ PEG, $0.3\mathrm{mol / LNaCl}$ 。60s摇匀， $4^{\circ}C$ ，60次/min，振荡 $60\mathrm{min}$ 。 $10000\mathrm{r / min},4^{\circ}C$ ，离心 $30\mathrm{min}$ ，弃上清。 $10000\mathrm{r / min},4^{\circ}C$ ，离心 $5\mathrm{min}$ 紧实沉淀，弃上清。  
5.1.1.6 $500\mu \mathrm{L}$ PBS 悬浮沉淀。如食品样品为生食蔬菜，可直接将悬浮液转移至干净试管，测定并记录悬浮液毫升数，用于后续 RNA 提取。如食品样品为软质水果，将悬浮液转移至耐氯仿试管中。加入 $500\mu \mathrm{L}$ 氯仿/丁醇混合液，涡旋混匀，室温静置 $5\mathrm{min}$ 。 $10000\mathrm{r / min}, 4^{\circ}\mathrm{C}$ ，离心 $15\mathrm{min}$ ，将液相部分仔细转移至干净试管，测定并记录悬浮液毫升数，用于后续 RNA 提取。

# 5.1.2 硬质表面食品

5.1.2.1 将无菌棉拭子使用PBS湿润后，用力擦拭食品表面（ $\leq 100 \mathrm{~cm}^2$ ）。记录擦拭面积。将 $10 \mu \mathrm{L}$ 过程控制病毒添加至该棉拭子。  
5.1.2.2 将棉拭子浸入含 $490 \mu \mathrm{L}$ PBS 试管中，紧贴试管一侧挤压出液体。如此重复浸入和挤压 $3 \sim 4$ 次，确保挤压出最大量的病毒，测定并记录液体毫升数，用于后续 RNA 提取。硬质食品表面过于粗糙，可能会损坏棉拭子，可使用多个棉拭子。

# 5.1.3 贝类

5.1.3.1 戴上防护手套，使用无菌贝类剥刀打开至少10个贝类。  
5.1.3.2 使用无菌剪刀、手术钳或其他等效器具在胶垫上解剖出贝类软体组织中的消化腺，置于干净培养皿中。收集 $2.0\mathrm{g}$ 。  
5.1.3.3 使用无菌刀片或等效均质器将消化腺匀浆后，转移至离心管。加入 $10~\mu \mathrm{L}$ 过程控制病毒。加入 $2.0~\mathrm{mL}$ 蛋白酶K溶液，混匀。  
5.1.3.4 使用恒温摇床或等效装置， $37^{\circ}\mathrm{C}$ ，320次/min，振荡 $60\mathrm{min}$ 。  
5.1.3.5 将试管放入水浴或等效装置， $60^{\circ}\mathrm{C}, 15\mathrm{min}$ 。室温， $3000\mathrm{r / min}, 5\mathrm{min}$ 离心，将上清液转移至干净试管，测定并记录上清液 $\mathrm{mL}$ 数，用于后续RNA提取。

# 5.2 病毒RNA提取和纯化

注：病毒RNA可手工提取和纯化，也可使用商品化病毒RNA提取纯化试剂盒。提取完成后，为延长RNA保存时间可选择性加入RNase抑制剂。操作过程中应佩戴一次性橡胶或乳胶手套，并经常更换。提取出来的RNA立即进行反应，或保存在 $4^{\circ}\mathrm{C}$ 小于 $8\mathrm{h}$ 。如果长期储存建议 $-80^{\circ}\mathrm{C}$ 保存。

# 5.2.1 病毒裂解

将病毒提取液加入离心管，加入病毒提取液等体积Trizol试剂，混匀，激烈振荡，室温放置 $5\mathrm{min}$ ，加入0.2倍体积氯仿，涡旋剧烈混匀 $30~\mathrm{s}$ （不能过于强烈，以免产生乳化层，也可用手颠倒混匀）， $12000\mathrm{r / min}$ ，离心 $5\mathrm{min}$ ，上层水相移入新离心管中，不能吸出中间层。

# 5.2.2 病毒RNA提取

离心管中加入等体积异丙醇，颠倒混匀，室温放置 $5\mathrm{min}$ ， $12000\mathrm{r / min}$ ，离心 $5\mathrm{min}$ ，弃上清，倒置于吸水纸上，沾干液体（不同样品须在吸水纸不同地方沾干）。

# 5.2.3 病毒RNA纯化

5.2.3.1 每次加入等体积 $75\%$ 乙醇，颠倒洗涤RNA沉淀2次。  
5.2.3.2 于 $4^{\circ}\mathrm{C},12000\mathrm{r / min}$ ，离心 $10\mathrm{min}$ ，小心弃上清，倒置于吸水纸上，沾干液体(不同样品须在吸水纸不同地方沾干)。或小心倒去上清液，用微量加样器将其吸干，一份样本换用一个吸头，吸头不要碰到有沉淀，室温干燥 $3\mathrm{min}$ ，不能过于干燥，以免RNA不溶。  
5.2.3.3 加入 $16 \mu \mathrm{L}$ 无 RNase 超纯水, 轻轻混匀, 溶解管壁上的 RNA, $2000 \mathrm{r} / \mathrm{min}$ , 离心 $5 \mathrm{~s}$ , 冰上保存备用。

# 5.3 质量控制

# 5.3.1 空白对照

以无RNase超纯水作为空白对照(A反应孔)。

# 5.3.2 阴性对照

以不含有诸如病毒的贝类，提取RNA，作为阴性对照（B反应孔）。

# 5.3.3 阳性对照

以外加扩增控制RNA，作为阳性对照（J反应孔）。

# 5.3.4 过程控制病毒

5.3.4.1 以食品中过程控制病毒 RNA 的提取效率表示食品中诺如病毒 RNA 的提取效率, 作为病毒提取过程控制。  
5.3.4.2 将过程控制病毒按 5.2 步骤提取和纯化 RNA。可大量提取, 分装为 $10 \mu \mathrm{L}$ 过程控制病毒的 RNA 量, $-80^{\circ} \mathrm{C}$ 保存, 每次检测时取出使用。  
5.3.4.3 将 $10\mu \mathrm{L}$ 过程控制病毒的RNA进行数次10倍梯度稀释（ $\mathrm{D} \sim \mathrm{G}$ 反应孔），加入过程控制病毒引物、探针，采用与诺如病毒实时荧光RT-PCR反应相同的反应条件确定未稀释和梯度稀释过程病毒RNA的 $C_{\mathrm{t}}$ 值。  
5.3.4.4 以未稀释和梯度稀释过程控制病毒RNA的浓度 $\lg$ 值为 $X$ 轴，以其 $C_{\mathrm{t}}$ 值为 $Y$ 轴，建立标准曲线；标准曲线 $r^2$ 应 $\geqslant 0.98$ 。未稀释过程控制病毒RNA浓度为1，梯度稀释过程控制RNA浓度分别为 $10^{-1},10^{-2},10^{-3}$ 等。  
5.3.4.5 将含过程控制病毒食品样品 RNA(C 反应孔), 加入过程控制病毒引物、探针, 采用诺如病毒实时荧光 RT-PCR 反应相同的反应体系和参数, 进行实时荧光 RT-PCR 反应, 确定 $C_{\mathrm{t}}$ 值, 代入标准曲线, 计算经过病毒提取等步骤后的过程控制病毒 RNA 浓度。  
5.3.4.6 计算提取效率，提取效率 $=$ 经病毒提取等步骤后的过程控制病毒RNA浓度 $\times 100\%$ ，即(C反应孔） $C_{t}$ 值对应浓度 $\times 100\%$ 。

# 5.3.5 外加扩增控制

5.3.5.1 通过外加扩增控制RNA，计算扩增抑制指数，作为扩增控制。  
5.3.5.2 外加扩增控制RNA分别加入含过程控制病毒食品样品RNA（H反应孔）、 $10^{-1}$ 稀释的含过程控制病毒食品样品RNA(I反应孔)、无RNase超纯水(J反应孔)，加入G I或G II型引物探针，采用附录C反应体系和参数，进行实时荧光RT-PCR反应，确定 $C_{\mathrm{t}}$ 值。  
5.3.5.3 计算扩增抑制指数，抑制指数 $=$ （含过程控制病毒食品样品RNA+外加扩增控制RNA） $C_{t}$ 值

一(无RNase超纯水 $+$ 外加扩增控制RNA) $C_{\mathrm{t}}$ 值，即抑制指数 $=$ (H反应孔) $C_{\mathrm{t}}$ 值一(J反应孔) $C_{\mathrm{t}}$ 值。如抑制指数 $\geq 2.00$ ，需比较10倍稀释食品样品的抑制指数，即抑制指数 $=$ (I反应孔) $C_{\mathrm{t}}$ 值一(J反应孔） $C_{\mathrm{t}}$ 值。

# 5.4 实时荧光RT-PCR

实时荧光RT-PCR反应体系和反应参数详见附录B。反应体系中各试剂的量可根据具体情况或不同的反应总体积进行适当调整。可采用商业化实时荧光RT-PCR试剂盒。也可增加调整反应孔，实现一次反应完成GⅠ和GⅡ型诺如病毒的独立检测。将 $18.5\mu \mathrm{L}$ 实时荧光RT-PCR反应体系添加至反应孔后，不同反应孔加入下述不同物质，检测GⅠ或GⅡ基因型诺如病毒。

A反应孔：空白对照，加入 $5\mu \mathrm{L}$ 无RNase超纯水 $+1.5\mu \mathrm{LG}$ I或GII型引物探针；

B反应孔：阴性对照，加入 $5\mu \mathrm{L}$ 阴性提取对照 $\mathrm{RNA} + 1.5\mu \mathrm{L}$ GI或GII型引物探针；

C反应孔：病毒提取过程控制1，加入 $5\mu \mathrm{L}$ 含过程控制病毒食品样品 $\mathrm{RNA} + 1.5\mu \mathrm{L}$ 过程控制病毒引物探针；

D 反应孔: 病毒提取过程控制 2 , 加入 $5 \mu \mathrm{L}$ 过程控制病毒 $\mathrm{RNA} + 1.5 \mu \mathrm{L}$ 过程控制病毒引物探针;

E 反应孔: 病毒提取过程控制 3 , 加入 $5 \mu \mathrm{L} 10^{-1}$ 倍稀释过程控制病毒 $\mathrm{RNA} + 1.5 \mu \mathrm{L}$ 过程控制病毒引物探针;

F反应孔：病毒提取过程控制4，加入 $5\mu \mathrm{L}10^{-2}$ 倍稀释过程控制病毒 $\mathrm{RNA + 1.5~\mu L}$ 过程控制病毒引物探针；

G反应孔：病毒提取过程控制5，加入 $5\mu \mathrm{L}10^{-3}$ 倍稀释过程控制病毒 $\mathrm{RNA + 1.5~\mu L}$ 过程控制病毒引物探针；

H反应孔：扩增控制1，加入 $5\mu \mathrm{L}$ 含过程控制病毒食品样品 $\mathrm{RNA} + 1\mu \mathrm{L}$ 外加扩增控制 $\mathrm{RNA}+$ $1.5\mu \mathrm{LG}\mathrm{I}$ 或GⅡ型引物探针；

I 反应孔: 扩增控制 2 , 加入 $5 \mu \mathrm{L} 10^{-1}$ 倍稀释的含过程控制病毒食品样品 $\mathrm{RNA} + 1 \mu \mathrm{L}$ 外加扩增控制 $\mathrm{RNA} + 1.5 \mu \mathrm{L} \mathrm{GI}$ 或 $\mathrm{GII}$ 型引物探针;

J反应孔：扩增控制3/阳性对照，加入 $5\mu \mathrm{L}$ 无RNase超纯水 $+1$ $\mu \mathrm{L}$ 外加扩增控制 $\mathrm{RNA} + 1.5\mu \mathrm{L}$ GI或GⅡ型引物探针；

K反应孔：样品1，加入 $5\mu \mathrm{L}$ 含过程控制病毒食品样品RNA $+1.5\mu \mathrm{L}$ GI或GII型引物探针；

L反应孔：样品2，加入 $5\mu \mathrm{L}10^{-1}$ 倍稀释的含过程控制病毒食品样品 $\mathrm{RNA + 1.5~\mu L~G~I}$ 或GII型引物探针。

# 6 结果与报告

# 6.1 检测有效性判定

6.1.1 需满足以下质量控制要求，检测方有效：空白对照阴性(A反应孔)；阴性对照阴性(B反应孔)；阳性对照(J反应孔)阳性。  
6.1.2 过程控制 $(\mathrm{C} \sim \mathrm{G}$ 反应孔)需满足：提取效率 $\geqslant 1\%$ ；如提取效率 $< 1\%$ ，需重新检测；但如提取效率 $< 1\%$ ，检测结果为阳性，也可酌情判定为阳性。  
6.1.3 扩增控制 $(\mathrm{H} \sim \mathrm{J}$ 反应孔)需满足：抑制指数 $< 2.00$ ；如抑制指数 $\geqslant 2.00$ ，需比较10倍稀释食品样品的抑制指数；如10倍稀释食品样品扩增的抑制指数 $< 2.00$ ，则扩增有效，且需采用10倍稀释食品样品RNA的 $C_{t}$ 值作为结果；10倍稀释食品样品扩增的抑制指数也 $\geqslant 2.00$ 时，扩增可能无效，需要重新检测；但如抑制指数 $\geqslant 2.00$ ，检测结果为阳性，也可酌情判定为阳性。

# 6.2 结果判定

待测样品的 $C_{\mathrm{t}}$ 值大于等于45时，判定为诺如病毒阴性；待测样品的 $C_{\mathrm{t}}$ 值小于等于38时，判定为诺如病毒阳性；待测样品的 $C_{\mathrm{t}}$ 值大于38，小于45时，应重新检测；重新检测结果大于等于45时，判定为诺如病毒阴性；小于等于38时，判定为诺如病毒阳性。

# 6.3 报告

根据检测结果，报告“检出诺如病毒基因”或“未检出诺如病毒基因”。

# 附录C

# 过程控制病毒培养及引物、探针

# C.1 概要

本标准使用过程控制病毒进行过程控制，可使用门哥病毒或其他等效，不与诺如病毒交叉反应的病毒。门哥病毒是小核糖核酸病毒科的鼠病毒。门哥病毒株MC0是一种重组病毒，与野生型门哥病毒相比缺乏poly[C]，是与野生型门哥病毒具有相似生长特性的无毒表型。门哥病毒株MC0是一种转基因生物，如果检测实验室不允许使用转基因生物，可以使用其他的过程控制病毒。也可使用商业化试剂或试剂盒中的过程控制病毒。

# C.2 培养试剂和仪器

# C.2.1 HeLa 细胞

推荐使用 Eagle 最低必需培养液 (Eagle's minimum essential medium, MEM) 培养, 并将 $2 \mathrm{mmol} / \mathrm{L}$ L-谷氨酸和 Earle's BSS 调为 $1.5 \mathrm{~g} / \mathrm{L}$ 碳酸氢钠, $0.1 \mathrm{mmol} / \mathrm{L}$ 非必需氨基酸, $1.0 \mathrm{mmol} / \mathrm{L}$ 丙酮酸钠, $1 \times$ 链霉素/青霉素液, $100 \mathrm{~mL} / \mathrm{L}$ (生长) 或 $20 \mathrm{~mL} / \mathrm{L}$ (维持) 胎牛血清。

# C.2.2 仪器

为确保细胞培养和病毒生长，需细胞培养所需的 $\mathrm{CO}_{2}$ 浓度可调培养箱，细菌培养耗材（例如培养皿）等。

# C.3 培养过程

门哥病毒培养在铺满 $80\% \sim 90\%$ 单层 HeLa（ATCC®CCL-2™）细胞中，置于 $50~\mathrm{mL} / \mathrm{L}$ $\mathrm{CO}_{2}$ 的气氛中(开放培养箱)或不可调的气氛中(封闭培养箱)，直至 $75\%$ 出现细胞病理效应。细胞培养器皿经过一个冻融循环，将培养物 $3000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。将细胞培养物离心上清留存用于过程控制。

# 附录D

# 外加扩增控制RNA制备

# D.1 概要

通过将目标DNA序列连接至合适的质粒载体上，目标序列位于RNA聚合酶启动子序列的下游序列，从而表达出外部扩增控制RNA。G I型外部扩增RNA序列位于诺如病毒(GenBank登录号m87661)的 $5291\sim 5376$ 。G II型外部扩增RNA序列位于Lordsdale病毒(GenBank登录号x86557)的 $5012\sim 5100$ 。

# D.2 试剂和设备

D.2.1 限制性酶：用于连接及相关的缓冲液。  
D.2.2 DNA 纯化试剂。  
D.2.3 体外RNA转录试剂(RNA聚合酶，NTPs，缓冲液等)。  
D.2.4 RNase-free DNase。  
D.2.5 RNA 纯化试剂。  
D.2.6 DNA凝胶电泳试剂和设备。  
D.2.7 培养箱： $37^{\circ} \mathrm{C}$ 。

# D.3 质粒DNA连接

添加 $100\mathrm{ng}\sim 500\mathrm{ng}$ 纯化的目标DNA和质粒载体加入含有合适的限制酶和缓冲液的反应体系中，限制酶和缓冲液的使用按照酶厂家推荐，并确保目标序列位于质粒中RNA聚合酶启动子序列的下游。 $37^{\circ}C$ 培养 $120\mathrm{min}$ 。DNA纯化使用DNA纯化试剂纯化。使用凝胶电泳检查连接情况，比较连接前与连接后目标DNA和质粒情况。

# D.4 外加扩增控制RNA的表达

添加连接后的质粒至转化体系。该体系按照转化体系提供厂家建议配置。使用RNA纯化试剂纯化RNA后，分装， $-80^{\circ}\mathrm{C}$ 储存，每次检测前取出备用。

# 附录E

# RNase的去除和无RNase溶液的配制

# E.1 RNase的去除

E.1.1 配制溶液用的酒精、异丙醇等应采用未开封的新品。配制溶液所用的超纯水、玻璃容器、移液器吸嘴、药匙等用具应无 RNase。操作过程中应自始自终佩戴抛弃式橡胶或乳胶手套，并经常更换，以避免将皮肤上的细菌、真菌及人体自身分泌的 RNase 染用具或带入溶液。

E.1.2 玻璃容器应在 $240^{\circ}\mathrm{C}$ 烘烤 $4\mathrm{h}$ 以去除RNase。

E.1.3 离心管、移液器吸嘴、药匙等塑料用具应用无RNase超纯水室温浸泡过夜，然后灭菌，烘干；或直接购买无RNase的相应用具。

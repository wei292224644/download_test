# 食品添加剂 麦芽酚

# 1 范围

本标准适用于由糠醛为原料制得的食品添加剂麦芽酚。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

3-羟基-2-甲基-4-吡喃酮

# 2.2 分子式

$\mathrm{C_6H_6O_3}$

# 附录A

# 麦芽酚含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按 GB/T 11538—2006 中 10.4 测定含量。试样制备：称取本品 $1 \mathrm{~g}$ 溶于 $20 \mathrm{~mL}$ 无水乙醇中，摇匀备

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行，应符合要求。食品添加剂麦芽酚气相色谱图及操作条件参见附录B。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $25\mathrm{m}\sim 50\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ 。  
B.2.2 固定相：聚乙二醇20000。  
B.2.3 膜厚: $0.25 \mu \mathrm{m}$ 。  
B.2.4 色谱炉温度： $180^{\circ}C$ 恒温。  
B.2.5 进样口温度： $250^{\circ}C$   
B.2.6 检测器温度： $200^{\circ}C$   
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 载气流速： $20~\mathrm{mL / min}\sim 30~\mathrm{mL / min}$   
B.2.10 进样量： $1\mu \mathrm{L}$   
B.2.11 分流比： $100:1$

# 前言

本标准代替GB28401—2012《食品安全国家标准 食品添加剂 磷脂》。

本标准与GB28401—2012相比，主要变化如下：

——增加了油菜籽、蛋黄为主要原料制得的食品添加剂磷脂；  
——重金属指标改为铅指标；  
——增加了商品化磷脂产品的相应描述。

# 食品安全国家标准

# 食品添加剂 磷脂

# 1 范围

本标准适用于以大豆、葵花籽、油菜籽等植物油籽料或其加工副产物为主要原料，经脱水、脱杂、脱色或脱脂等工序，或以蛋黄为主要原料经提取、精制等工序制得的食品添加剂磷脂。

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

A.2.1 取含 $0.5\%$ 样品的乙醇溶液 $2\mathrm{mL}$ 置于试管中，加 $5\%$ 氯化镉乙醇溶液1滴 $\sim 2$ 滴，即产生白色沉淀。  
A.2.2 取含 $10\%$ 样品的乙醇溶液 $2\mathrm{mL}$ 置于试管中，加硝酸铋钾溶液（取硝酸铋 $8\mathrm{g}$ ，加硝酸 $20\mathrm{mL}$ 使溶解；另取碘化钾 $27.2\mathrm{g}$ ，加水 $50\mathrm{mL}$ 使溶解，合并上述两种溶液，加水稀释成至 $100\mathrm{mL})1$ 滴～2滴，即产生砖红色沉淀。

# A.3 正己烷不溶物的测定

# A.3.1 试剂和材料

正已烷。

# A.3.2 仪器和设备

A.3.2.1 抽滤器： $500~\mathrm{mL}$   
A.3.2.2 玻璃砂芯坩埚(漏斗)：G3。

# A.3.3 分析步骤

A.3.3.1 将清洁的坩埚在 $101^{\circ}\mathrm{C} \sim 105^{\circ}\mathrm{C}$ 烘箱中烘至恒重。  
A.3.3.2 称取试样 $10.0\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，置于烧杯中，加入正己烷约 $100\mathrm{mL}$ ，用玻璃棒搅拌溶解后，通过已恒重的坩埚抽滤。  
A.3.3.3 用 $25~\mathrm{mL}$ 正己烷分两次洗涤烧杯和玻璃棒并将不溶物全部移入坩埚内，用正己烷洗涤坩埚内壁和不溶物，最后尽量抽除坩埚内残留正己烷。  
A.3.3.4 取下坩埚，用脱脂棉沾少许正己烷擦净坩埚外壁。将坩埚于 $101~\mathrm{^\circ C}\sim 105~\mathrm{^\circ C}$ 烘箱中烘干 $1\textrm{h}$ 取出，放入干燥器中冷却至室温，称重。再烘干 $20\mathrm{min}$ ，冷却，称重，直至恒重。

# A.3.4 结果计算

正己烷不溶物含量以质量分数 $\varpi_{1}$ 计，按式(A.1)计算。

$$
w _ {1} = \frac {m _ {1} - m _ {2}}{m} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$m_{1}$ ——空坩埚及不溶物的总质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——空坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准，计算结果保留两位有效数字。在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# A.4 酸价的测定

# A.4.1 试剂和材料

A.4.1.1 石油醚。

A.4.1.2 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.1\mathrm{mol / L}$

A.4.1.3 中性乙醇：质量分数为 $95\%$ 。使用前以酚酞指示液作指示剂，用氢氧化钠标准滴定溶液滴定至溶液呈微红色，并维持 $5 \mathrm{~s}$ 不褪色为终点。

A.4.1.4 酚酞指示液： $10\mathrm{g / L}$

# A.4.2 分析步骤

称取约 $2\mathrm{g}$ 混合均匀的试样，精确至 $0.001\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 锥形瓶中，加入 $50~\mathrm{mL}$ 石油醚，轻轻振摇使之溶解，然后加入 $50~\mathrm{mL}$ 中性乙醇，摇匀。加入4滴酚酞指示液，用氢氧化钠标准滴定溶液快速滴定，边滴边振摇，接近终点时，减慢滴定速度至每次1滴～2滴，至粉红色出现并维持 $5\mathrm{s}$ 不褪色即为终点。

空白试验：另取一个干净的 $250~\mathrm{mL}$ 的锥形瓶，准确加入 $50~\mathrm{mL}$ 石油醚，轻轻振摇使之溶解，然后加入 $50~\mathrm{mL}$ 中性乙醇，摇匀。加入4滴酚酞指示液，用氢氧化钠标准滴定溶液快速滴定，边滴边振摇，接近终点时，减慢滴定速度至每次1滴～2滴，至粉红色出现并维持 $5\mathrm{s}$ 不褪色即为终点。此滴定所消耗的标准滴定溶液的毫升数为 $V_{0}$ 。

# A.4.3 结果计算

酸价以氢氧化钾(KOH)的质量分数 $\omega_{2}$ 计，数值以毫克每克 $(\mathrm{mg / g})$ 表示，按式(A.2)计算。

$$
w _ {2} = \frac {\left(V - V _ {0}\right) \times c \times 5 6 . 1}{m _ {0}} \quad \dots \tag {A.2}
$$

式中：

$V$ ——滴定时消耗的氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白时消耗的氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——氢氧化钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

56.1——氢氧化钾的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{KOH}) = 56.1]$

$m_0$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准，计算结果保留至小数点后一位。在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# A.5 过氧化值的测定

# A.5.1 试剂和材料

A.5.1.1 乙酸与三氯甲烷混合液：将3份乙酸与2份三氯甲烷混合，体积比 $3:2$

A.5.1.2 饱和碘化钾溶液：新配制且不得含有游离碘和碘酸盐。确保溶液中有结晶存在，存放于避光处。在 $30~\mathrm{mL}$ 乙酸与三氯甲烷混合液中加入 $0.5\mathrm{mL}$ 饱和碘化钾溶液和2滴淀粉指示液，若溶液出现蓝色并需要硫代硫酸钠标准滴定溶液1滴以上才能消除，则需重新配制此溶液。

A.5.1.3 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.01\mathrm{mol / L}$ ，将 $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$ 的硫代硫酸钠标准滴定溶液稀释10倍制得。

A.5.1.4 淀粉指示液： $10\mathrm{g / L}$

注：上述所有试剂和水中不得含有溶解氧。

# A.5.2 仪器和设备

实验室常规仪器。使用的所有器皿不得含有还原性或氧化性物质。磨砂玻璃表面不得涂油。

# A.5.3 分析步骤

称取约 $5\mathrm{g}$ 混合均匀的试样，精确至 $0.001\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 碘瓶中，加入 $30~\mathrm{mL}$ 乙酸与三氯甲烷混合液，振摇使试样充分溶解。加入 $0.5\mathrm{mL}$ 饱和碘化钾溶液，紧密盖塞反应 $1\mathrm{min}$ ，反应过程中至少轻摇碘瓶3次。然后立即加入 $30~\mathrm{mL}$ 水和 $0.5\mathrm{mL}$ 淀粉指示液，用硫代硫酸钠标准滴定溶液滴定，边滴边振摇，临近终点时，不断振摇使所有的碘从溶剂层释放出来，逐滴添加硫代硫酸钠标准滴定溶液至溶液蓝色消失，即为终点。同时进行空白试验，当空白试验消耗的硫代硫酸钠标准滴定溶液超过 $0.1\mathrm{mL}$ 时，应更换试剂重新对试样进行测定。

# A.5.4 结果计算

过氧化值以 $w_{3}$ 计，数值以毫摩尔每千克 $(\mathrm{mmol / kg})$ 表示，按式(A.3)计算。

$$
w _ {3} = \frac {1 0 0 0 \times \left(V - V _ {0}\right) \times c}{2 \times m _ {3}} \quad \dots \tag {A.3}
$$

式中：

1000——换算因子；

$V$ ——滴定试样所消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白所消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫代硫酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

2 ——换算因子；

$m_{3}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准，计算结果保留至小数点后一位。在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

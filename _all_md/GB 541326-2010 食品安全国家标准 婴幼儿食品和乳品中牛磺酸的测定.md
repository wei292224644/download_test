# 前言

本标准第二法等同采用国际分析家学会（AOAC）AOAC 997.05 Taurine in powdered milk and powdered infant formulae。

本标准代替GB/T5413.26-1997《婴幼儿食品和乳粉牛磺酸的测定》。

本标准与GB/T5413.26-1997相比，主要变化如下：

将原标准方法OPA柱后衍生高效液相色谱法定为第一法；

——增加单磺酰氯柱前衍生高效液相色谱法为第二法；

对原标准的结构进行了修改。

——外标法定量采用标准曲线法；

——增加附录A标样的液相色谱图。

本标准附录A为资料性附录。

本标准所代替标准的历次版本发布情况为：

GB5413-1985、GB/T5413.26-1997。

# 食品安全国家标准婴幼儿食品和乳品中牛磺酸的测定

# 1 范围

本标准规定了婴幼儿食品和乳品中牛磺酸的测定方法。

本标准适用于婴幼儿食品和乳品中牛磺酸的测定。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

# 第一法 OPA柱后衍生法

# 3 原理

样品用偏磷酸溶液溶解，经超声波振荡提取、离心、微孔滤膜过滤后，通过钠离子色谱柱分离，与邻苯二甲醛（OPA）衍生反应，用荧光检测器进行检测，外标法定量。

# 4 试剂和材料

除非另有规定，所用试剂均为分析纯，水为GB/T6682规定的一级水。

4.1 偏磷酸（ $\mathrm{HPO_3}$ ）。  
4.2 柠檬酸三钠 $\mathrm{(Na_3C_6H_5O_7\cdot 2H_2O)}$ 。  
4.3 苯酚 $(\mathrm{C_6H_6O})$ 。  
4.4 硝酸 $(\mathrm{HNO}_3)$ 。  
4.5 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
4.6 硼酸 $\mathrm{(H_3BO_3)}$ 。  
4.7 氢氧化钾(KOH)。  
4.8 邻苯二甲醛 $(\mathrm{C_8H_6O_2})$ （OPA）。  
4.9 2-巯基乙醇 $\mathrm{(C_2H_6OS)}$ 。  
4.10 聚氧乙烯月桂酸醚（Brij-35）。  
4.11 牛磺酸标准品：纯度 $\geq 99\%$   
4.12 偏磷酸溶液（ $10\mathrm{g / L}$ ）：称取 $10.0\mathrm{g}$ 偏磷酸（4.1），用水溶解并定容至 $1000\mathrm{mL}$ 。

4.13 柠檬酸缓冲液：称取 $19.6 \mathrm{~g}$ 柠檬酸三钠（4.2），加 $950 \mathrm{~mL}$ 水溶解，加入 $1 \mathrm{~mL}$ 苯酚（4.3），用硝酸（4.4）调 $\mathrm{pH}$ 值至 $3.10 \sim 3.25$ ，经 $0.45 \mu \mathrm{m}$ 微孔滤膜过滤。  
4.14 柱后荧光衍生溶剂（邻苯二甲醛溶液）  
4.14.1 硼酸钾溶液（ $0.5\mathrm{mol} / \mathrm{L}$ ）：称取 $30.9\mathrm{g}$ 硼酸（4.6）， $26.3\mathrm{g}$ 氢氧化钾（4.7），用水溶解并定容至 $1000\mathrm{mL}$ 。  
4.14.2 邻苯二甲醛衍生溶液：称取 $0.60\mathrm{g}$ 邻苯二甲醛（4.8），用 $10\mathrm{mL}$ 甲醇（4.5）溶解后，加入 $0.5\mathrm{mL}2$ -巯基乙醇（4.9）和 $0.35\mathrm{g}$ Brij-35（4.10），用 $0.5\mathrm{mol / L}$ 的硼酸钾溶液（4.14.1）定容至 $1000\mathrm{mL}$ ，经 $0.45\mu \mathrm{m}$ 微孔滤膜过滤。临用前配制。  
4.15 牛磺酸标准溶液  
4.15.1 牛磺酸标准储备溶液（ $1\mathrm{mg/mL}$ ）：准确称取 $0.1000\mathrm{g}$ 牛磺酸标准品（4.11），用水溶解并定容至 $100\mathrm{mL}$ 。储备液在 $4^{\circ}\mathrm{C}$ 下可保存7天。  
4.15.2 牛磺酸标准工作液：将牛磺酸标准储备液（4.15.1）用水稀释制备一系列标准溶液，标准系列浓度为：0、5、10、15、 $20\mu \mathrm{g / mL}$ 。临用前配制。

# 5 仪器和设备

5.1 高效液相色谱仪，带有荧光检测器。  
5.2 柱后反应器。  
5.3 荧光衍生溶剂输液泵。  
5.4 超声波振荡器。  
5.5 pH计：精度为0.01。  
5.6 离心机：转速 $\geqslant 5000$ 转/分钟。   
5.7 $0.45\mu \mathrm{m}$ 微孔滤膜。  
5.8 天平：感量为 $1\mathrm{mg}$ ， $0.1\mathrm{mg}$ 。

# 6 分析步骤

# 6.1 试样的处理

准确称取固体样品 $1\mathrm{g}\sim 5\mathrm{g}$ 试样，液体样品 $5\mathrm{g}\sim 20\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，试样中含牛磺酸 $5\mu \mathrm{g}$ 以上），加 $30\mathrm{mL}$ 偏磷酸溶液(4.12)溶解，充分摇匀，移入 $100\mathrm{mL}$ 容量瓶中；放入超声波振荡器中振荡 $10\mathrm{min}\sim$ $15\mathrm{min}$ ，取出冷却至室温后，用水定容至刻度；样液在5000转/分钟条件下离心 $10\mathrm{min}$ ，取上清液经 $0.45\mu \mathrm{m}$ 微孔膜（5.7）过滤，接取中间滤液以备进样。

# 6.2 测定

# 6.2.1 参考色谱条件

色谱柱：钠离子氨基酸分析专用柱（ $250\mathrm{mm}\times 4.6\mathrm{mm}$ ）或同等性能的色谱柱。

流动相：柠檬酸缓冲液（4.13）。

流动相流速： $0.30\mathrm{mL / min}$

荧光衍生溶剂流速： $0.30\mathrm{mL / min}$

柱温： $55^{\circ}C$ 。

检测波长：激发波长： $338\mathrm{nm}$ ，发射波长： $425\mathrm{nm}$

进样量： $20~\mu \mathrm{L}$ 。

# 6.2.2 标准曲线绘制

将牛磺酸标准系列工作液（4.15.2）依次经衍生后按上述推荐色谱条件上机测定，记录色谱峰面积，色谱图参见附录A。以峰面积为纵坐标，浓度为横坐标，绘制标准曲线。

# 6.2.3 试液测定

将试液按上述推荐色谱条件上机测定，从标准曲线中查得试液相应的浓度。

# 7 分析结果的表述

# 7.1 结果计算

$$
X = \frac {c \times V \times 1 0 0}{m \times 1 0 0 0} \dots \tag {1}
$$

式中：

X ——试样中牛磺酸的含量，单位为毫克/100克（ $\mathrm{mg} / 100 \mathrm{~g}$ ）；

$c$ ——试液的进样浓度，单位为微克/毫升（ $\mu \mathrm{g} / \mathrm{mL}$ ）；

$V$ ——试样定容容积, 单位为毫升 $(\mathrm{mL})$ ;

$m$ ——试样质量，单位为克（g）。

# 7.2 结果表示

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 8 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 单磺酰氯柱前衍生法

# 9 原理

样品用水溶解，用亚铁氰化钾和乙酸锌沉淀蛋白质。取上清液用丹磺酰氯衍生反应，衍生物经 $\mathrm{C}_{18}$ 反相色谱柱分离，用紫外检测器（波长 $254\mathrm{nm}$ ）或荧光检测器（激发波长 $330\mathrm{nm}$ ，发射波长 $530\mathrm{nm}$ ）检测，外标法定量。

# 10 试剂和材料

除非另有规定，所用试剂均为分析纯，水为GB/T6682规定的一级水。

10.1 乙腈（ $\mathrm{CH}_3\mathrm{CN}$ ）：色谱纯。  
10.2 冰乙酸（CH3COOH）。  
10.3 盐酸。  
10.4 无水碳酸钠 $\mathrm{(Na_2CO_3)}$

10.5 亚铁氰化钾 $\left[\mathrm{K}_{4} \mathrm{Fe}(\mathrm{CN})_{6}\right]$ 。  
10.6 乙酸锌 $\left[\mathrm{Zn}\left(\mathrm{CH}_{3} \mathrm{COO}\right)_{2}\right]$ 。  
10.7 乙酸钠 $\mathrm{[Na(CH_3COO)]}$   
10.8 盐酸甲胺（甲胺盐酸盐） $\mathrm{(CH_3NH_2\cdot HCl)}$ 。  
10.9 丹磺酰氯（5-二甲氨基萘-1-磺酰氯）：色谱纯。

注：丹磺酰氯对光和湿敏感不稳定。

10.10 牛磺酸标准品：纯度 $\geq 99\%$   
10.11 盐酸（ $1\mathrm{mol} / \mathrm{L}$ ）：吸取 $9\mathrm{mL}$ 盐酸（10.3），用水稀释并定容到 $100\mathrm{mL}$ 。  
10.12 沉淀剂  
10. 12.1 沉淀剂 I：称取 $15.0 \mathrm{~g}$ 亚铁氰化钾（10.5），用水溶解并定容至 $100 \mathrm{~mL}$ 。该沉淀剂在室温下 3 个月内稳定。  
10.12.2 沉淀剂II：称取 $30.0\mathrm{g}$ 乙酸锌（10.6），用水溶解并定容至 $100\mathrm{mL}$ 。该沉淀剂在室温下3个月内保持稳定。  
10.13 碳酸钠缓冲液（ $\mathrm{pH}9.5$ ）（ $80 \mathrm{mmol} / \mathrm{L}$ ）：称取 $0.424 \mathrm{~g}$ 无水碳酸钠（10.4），加 $40 \mathrm{~mL}$ 水溶解，用 $1 \mathrm{~mol} / \mathrm{L}$ 盐酸（10.11）调 $\mathrm{pH}$ 值至 9.5，用水定容至 $50 \mathrm{~mL}$ 。该溶液在室温下 3 个月内稳定。  
10.14 丹磺酰氯溶液（ $1.5\mathrm{mg/mL}$ ）：称取 $0.15\mathrm{g}$ 丹磺酰氯（10.9），用乙腈（10.1）溶解并定容至 $100\mathrm{mL}$ 。临使用前配制。  
10.15 盐酸甲胺溶液（ $20\mathrm{mg/mL}$ ）：称取 $2.0\mathrm{g}$ 盐酸甲胺（10.8），用水溶解并定容至 $100\mathrm{mL}$ 。该溶液在 $4^{\circ}\mathrm{C}$ 下3个月内稳定。  
10.16 乙酸钠缓冲液（ $\mathrm{pH}4.2$ ）（ $10 \mathrm{mmol} / \mathrm{L}$ ）：称取 $0.820 \mathrm{~g}$ 乙酸钠（10.7），加 $800 \mathrm{~mL}$ 水溶解，用冰乙酸（10.2）调节 $\mathrm{pH}$ 值至 4.2，用水定容至 $1000 \mathrm{~mL}$ ，经 $0.45 \mu \mathrm{m}$ 微孔滤膜过滤。  
10.17 牛磺酸标准溶液  
10.17.1 牛磺酸标准储备溶液（ $1\mathrm{mg/mL}$ ）：称取 $0.1000\mathrm{g}$ 牛磺酸标准品（10.10），用水溶解并定容至 $100\mathrm{mL}$ 。储备液在 $4^{\circ}\mathrm{C}$ 下可保存7天。  
10.17.2 牛磺酸标准工作液（紫外检测用）：将牛磺酸标准储备液（10.17.1）用水稀释制备一系列标准溶液，标准系列浓度为：0、5、10、15、 $20\mu \mathrm{g} / \mathrm{mL}$ 。临用前配制。  
10.17.3 牛磺酸标准工作液（荧光检测用）：将牛磺酸标准储备液（10.17.2）用水稀释制备一系列标准溶液，标准系列浓度为：0、0.5、0.10、0.15、 $0.20\mu \mathrm{g / mL}$ 。临用前配制。

# 11 仪器和设备

11.1 高效液相色谱仪，带紫外检测器或二极管阵列检测器或者荧光检测器。  
11.2 pH计：精度为0.01。  
11.3 涡旋混合器。  
11.4 超声波振荡器。

11.5 离心机：转速 $\geqslant 5000$ 转/分钟。   
11.6 $0.45\mu \mathrm{m}$ 微孔滤膜。  
11.7 天平：感量 $1\mathrm{mg}$ ， $0.1\mathrm{mg}$ 。

# 12 操作步骤

# 12.1 试样的处理

# 12.1.1 试液提取

称取固体样品 $1\mathrm{g}\sim 5\mathrm{g}$ ，液体样品 $5\mathrm{g}\sim 30\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ，若用紫外检测器，试样中含牛磺酸宜在 $1\mu \mathrm{g}$ 以上，若用荧光检测器，试样中含牛磺酸宜在 $50\mu \mathrm{g}$ 以上）于 $100\mathrm{mL}$ 容量瓶中，加入 $80~\mathrm{mL}$ 温水（ $50^{\circ}C\sim 60^{\circ}C$ ）溶解，充分混匀，置超声波振荡器上振荡 $10\mathrm{min}$ ，冷却到室温。加 $1.0\mathrm{mL}$ 沉淀剂I（10.12.1)，涡旋混合， $1.0~\mathrm{mL}$ 沉淀剂II（10.12.2)，涡旋混合，用水定容至刻度，充分混匀，试液于5000转/分钟下离心 $10\mathrm{min}$ ，取上清液备用。上清液在 $4^{\circ}C$ 暗处保存放置 $24\mathrm{h}$ 稳定。

# 12.1.2 试液衍生化

吸取 $1.00\mathrm{mL}$ 上述上清液到 $10\mathrm{mL}$ 具塞玻璃试管中，加入 $1.00\mathrm{mL}$ 碳酸钠缓冲液（10.13）， $1.00\mathrm{mL}$ 丹磺酰氯溶液（10.14），充分混合，室温避光衍生反应 $2\mathrm{h}$ （ $1\mathrm{h}$ 后需摇晃1次），加入 $0.10\mathrm{mL}$ 盐酸甲胺溶液（10.15）涡旋混合，以终止反应，避光静置至沉淀完全。取上清液经 $0.45\mu \mathrm{m}$ 微孔滤膜（11.6）过滤，取滤液备用。衍生物在 $4^{\circ}\mathrm{C}$ 可避光保存 $48\mathrm{h}$ 。

另取 $1.00\mathrm{mL}$ 标准工作液（10.17.2），与试液同步进行衍生。

# 12.2 测定

# 12.2.1 参考色谱条件

色谱柱： $\mathrm{C_{18}}$ 反相色谱柱（粒径 $5\mu \mathrm{m}$ ， $250\mathrm{mm} \times 4.6\mathrm{mm}$ ）或同等性能色谱柱。

流动相：10 mmol/L 乙酸钠缓冲液（10.16）—乙腈（10.1）=70+30。

流速： $1.00\mathrm{mL / min}$

柱温：室温。

检测波长：紫外检测器或二极管阵列检测器： $254\mathrm{nm}$

或荧光检测器：激发波长： $330\mathrm{nm}$ ；发射波长： $530\mathrm{nm}$ 。

进样量： $20~\mu \mathrm{L}$

# 12.2.2 标准曲线绘制

将牛磺酸标准系列工作液（紫外检测用）（10.17.2）或牛磺酸标准系列工作液（荧光检测用）（10.17.3）的衍生物依次按上述推荐色谱条件上机测定，记录色谱峰面积，色谱图参见附录A。以峰面积为纵坐标，浓度为横坐标，绘制标准曲线。

# 12.2.3 试液测定

将试液衍生物按上述推荐色谱条件上机测定，从标准曲线中查得试液相应的浓度。

# 13 分析结果的表述

试样中牛磺酸的含量按式（2）计算：

$$
X = \frac {c \times V \times 1 0 0}{m \times 1 0 0 0} \dots \tag {2}
$$

式中：

$X$ ——试样牛磺酸的含量，单位为毫克/100克 $(\mathrm{mg} / 100 \mathrm{~g})$ ；

$c$ ——试液的进样浓度, 单位为微克/毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ;

$V$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克（g）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

本标准的定量限为：当取样量为 $10.00\mathrm{g}$ 时，第一法 $0.5\mathrm{mg} / 100\mathrm{g}$ ，第二法中紫外检测法为 $5\mathrm{mg} / 100\mathrm{g}$ ，荧光检测法为 $0.1\mathrm{mg} / 100\mathrm{g}$ 。

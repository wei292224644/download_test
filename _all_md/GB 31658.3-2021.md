# 食品安全国家标准

# 猪尿中巴氯芬残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了猪尿中巴氯芬残留量测定的制样和液相色谱串联质谱测定方法。

本文件适用于猪尿中巴氯芬残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的巴氯芬,经酸化、提取,固相萃取柱净化,液相色谱- 串联质谱法测定,内标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.3 甲酸（HCOOH）：色谱纯。  
5.1.4 盐酸（HCl）。  
5.1.5 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$   
5.1.6 氢氧化钠(NaOH)。

# 5.2 溶液配制

5.2.1 $10\%$ 盐酸溶液：取盐酸 $10~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ 。  
5.2.2 $0.1\%$ 甲酸溶液：取甲酸 $0.5 \mathrm{~mL}$ ，用水稀释至 $500 \mathrm{~mL}$ 。  
5.2.3 $0.2\%$ 甲酸溶液：取甲酸 $0.2\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$   
5.2.4 $0.2\%$ 甲酸-乙腈溶液：取 $0.2\%$ 甲酸溶液 $80~\mathrm{mL}$ ，加乙腈稀释成 $100~\mathrm{mL}$ 。  
5.2.5 $5\%$ 氨水甲醇溶液：取氨水 $5\mathrm{mL}$ ，用甲醇稀释至 $100~\mathrm{mL}$   
5.2.6 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：取氢氧化钠 $4\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。

# 5.3 标准品

5.3.1 巴氯芬（Baclofen， $\mathrm{C_{10}H_{12}ClNO_2}$ ，CAS号：1134-47-0）含量 $\geq 99.0\%$   
5.3.2 巴氯芬 $\mathrm{D_4}$ ：含量 $\geq 99.0\%$

# 5.4 标准溶液制备

5.4.1 巴氯芬标准储备液：取巴氯芬标准品约 $10\mathrm{mg}$ ，精密称定，加 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液 $1\mathrm{mL}$ 使溶

解，用甲醇稀释定容至 $10\mathrm{mL}$ 容量瓶，配成浓度为 $1\mathrm{mg / mL}$ 的巴氯芬标准储备液。 $4^{\circ}C$ 以下保存，有效期4个月。

5.4.2 巴氯芬标准中间液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：精密量取巴氯芬标准储备液 $100~\mu \mathrm{L}$ ，于 $10~\mathrm{mL}$ 容量瓶，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g} / \mathrm{mL}$ 的巴氯芬标准中间液。 $4^{\circ}\mathrm{C}$ 以下保存，有效期1个月。  
5.4.3 巴氯芬标准工作液 $(1\mu \mathrm{g / mL})$ ：精密量取巴氯芬标准中间液 $1\mathrm{mL}$ ，用甲醇稀释定容至 $10~\mathrm{mL}$ 容量瓶，配制成浓度为 $1\mu \mathrm{g / mL}$ 的巴氯芬标准工作液。现配现用。  
5.4.4 巴氯芬- $\mathrm{D_4}$ 储备液 $(1\mathrm{mg / mL})$ ：取巴氯芬- $\mathrm{D_4}$ 对照品约 $10\mathrm{mg}$ ，精密称定，加 $1\mathrm{mol / L}$ 氢氧化钠溶液 $1\mathrm{mL}$ 使溶解，用甲醇稀释定容至 $10~\mathrm{mL}$ 容量瓶刻度，配成浓度为 $1\mathrm{mg / mL}$ 的巴氯芬- $\mathrm{D_4}$ 储备液。 $4^{\circ}C$ 以下保存，有效期4个月。  
5.4.5 巴氯芬- $\mathrm{D_4}$ 中间液 $(10\mu \mathrm{g / mL})$ ：精密量取巴氯芬- $\mathrm{D_4}$ 储备液 $100~\mu \mathrm{L}$ ，于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释制成浓度为 $10~\mu \mathrm{g / mL}$ 的巴氯芬- $\mathrm{D_4}$ 中间液。 $4^{\circ}C$ 以下保存，有效期1个月。  
5.4.6 巴氯芬- $\mathrm{D_4}$ 工作液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：精密量取巴氯芬- $\mathrm{D_4}$ 中间液 $1\mathrm{mL}$ ，置于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 的巴氯芬- $\mathrm{D_4}$ 工作液。现配现用。  
5.4.7 混合标准工作液：分别精密量取巴氯芬标准工作液和巴氯芬 $\mathrm{D_4}$ 工作液适量，用 $0.2\%$ 甲酸-乙腈溶液配制成含巴氯芬 $\mathrm{D_4}10\mathrm{ng / mL}$ ，含巴氯芬分别为 $1.0~\mathrm{ng / mL},2.0~\mathrm{ng / mL},5.0~\mathrm{ng / mL},10.0~\mathrm{ng / mL},$ $25.0~\mathrm{ng / mL}$ 和 $50.0~\mathrm{ng / mL}$ 的系列混合标准工作液。现配现用。

# 5.5 材料

混合型强阳离子固相萃取柱， $60\mathrm{mg / 3mL}$ ，或相当者。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配有电喷雾电离源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 离心机： $8000\mathrm{r / min}$   
6.4 涡旋振荡器。  
6.5 固相萃取装置。  
6.6 氮吹仪。  
6.7 pH计。  
6.8 离心管： $50~\mathrm{mL}$   
6.9 滤膜： $0.22\mu \mathrm{m}$ ，水相。

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试样品，使用前恢复室温，混匀，如有混浊，离心后取上清液备用。

a）取备用的供试样品，作为供试材料；  
b）取备用的空白样品，作为空白材料；  
c）取备用的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

供试样品或空白样品应尽快置于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 8 测定步骤

# 8.1 提取

取试料 $2.0\mathrm{mL}$ ，于 $50~\mathrm{mL}$ 离心管中，加巴氯芬- $\mathrm{D_4}$ 工作液 $10~\mu \mathrm{L}$ ，用 $10\%$ 盐酸溶液调节 $\mathrm{pH}$ 至2.0，涡旋混匀， $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，备用。

# 8.2 净化

取固相萃取柱，依次用甲醇 $3\mathrm{mL}$ 、水 $3\mathrm{mL}$ 活化。备用液全部过柱，用水 $3\mathrm{mL}$ 、甲醇 $3\mathrm{mL}$ 依次淋洗，抽干 $2\mathrm{min}$ ，用 $5\%$ 氨水甲醇溶液 $6\mathrm{mL}$ 洗脱，收集洗脱液，于 $50^{\circ}\mathrm{C}$ 水浴氮气吹干，用 $0.2\%$ 甲酸-乙腈溶液 $1.0\mathrm{mL}$ 溶解，滤膜过滤，液相色谱-串联质谱测定。

# 8.3 标准曲线的制备

取 $1.0\mathrm{ng / mL},2.0\mathrm{ng / mL},5.0\mathrm{ng / mL},10.0\mathrm{ng / mL},25.0\mathrm{ng / mL}$ 和 $50.0\mathrm{ng / mL}$ 的系列混合标准工作液，供液相色谱法-串联质谱法测定。以测得特征离子质量色谱峰峰面积比为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.4 测定

# 8.4.1 色谱条件

a）色谱柱： $C_{18}(100\mathrm{mm}\times 3.0\mathrm{mm},1.8\mu \mathrm{m})$ ，或相当者；  
b） 柱温： $35^{\circ}C$   
c）进样量： $10~\mu \mathrm{L}$   
d) 流动相：A为甲醇，B为 $0.1\%$ 甲酸溶液，梯度洗脱条件见表1；  
e) 流速: $0.25 \mathrm{~mL} / \mathrm{min}$ 。

# 8.4.3 测定法

取试料溶液和标准溶液，作单点或多点校准，按内标法以色谱峰面积定量。标准溶液及试样溶液中巴氯芬及巴氯芬- $\mathrm{D_4}$ 的响应值均应在仪器检测的线性范围内。试样液中巴氯芬的保留时间与标准工作液中巴氯芬的保留时间偏差在 $\pm 2.5\%$ 以内，且检测到的相对离子丰度，应当与浓度相当的标准溶液相对丰度一致，其允许偏差应符合表3要求。在上述色谱-质谱条件下，巴氯芬及巴氯芬- $\mathrm{D_4}$ 标准溶液特征离子质量色谱图见附录A。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

猪尿液中巴氯芬残留量按公式(1)计算。

$$
X = \frac {A \times A _ {i S} ^ {\prime} \times C _ {S} \times C _ {i S} \times V}{A _ {i S} \times A _ {S} \times C _ {i S} ^ {\prime} \times V _ {m}} \tag {1}
$$

式中：

X 猪尿中巴氯芬残留量的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$C_{\mathrm{iS}}$ ——试样溶液中内标物巴氯芬 $\mathrm{D}_{\mathrm{i}}$ 浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$C_{\mathrm{S}}$ ——混合标准工作液中巴氯芬浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$C_{\mathrm{is}}^{\prime}$ ——混合标准工作液中内标物巴氯芬 $\mathrm{D}_{2}$ 浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

A ——试样溶液中巴氯芬的峰面积；

$A_{\mathrm{is}}$ ——试样溶液中内标物巴氯芬 $\mathrm{D}_{1}$ 的峰面积；

$A_{\mathrm{S}}$ ——混合标准工作液中巴氯芬的峰面积；

$A_{\mathrm{is}}^{\prime}$ ——混合标准工作液中内标物巴氯芬 $\mathrm{D}_{4}$ 峰面积；

V ——溶解残余物所用溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{m}$ ——供试猪尿液体积的数值，单位为毫升 $(\mathrm{mL})$ 。

# 10 检测方法的灵敏度、准确度、精密度

# 10.1 灵敏度

本方法的检测限为 $0.5\mathrm{ng / mL}$ ，定量限为 $1.0\mathrm{ng / mL}$

# 10.2 准确度

本方法在 $1.0\mathrm{ng / mL}\sim 25\mathrm{ng / mL}$ 添加浓度水平的回收率为 $70\% \sim 110\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

# 前言

本标准代替GB/T5009.215—2008《食品中有机锡含量的测定》。

本标准与GB/T5009.215—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中有机锡的测定”；

——标准文本格式的更新。

# 食品安全国家标准

# 食品中有机锡的测定

# 1 范围

本标准规定了食品中有机锡的气相色谱-脉冲火焰光度检测器检测方法。

本标准适用于鱼类、贝类、葡萄酒和酱油等样品中二甲基锡、三甲基锡、一丁基锡、二丁基锡、三丁基锡、一苯基锡、二苯基锡、三苯基锡的测定。

# 2 原理

分别以一甲基锡为单取代有机锡的内标，三丙基锡为二、三取代有机锡的内标，采用内标法定量。在试样中定量加入一甲基锡和三丙基锡内标，超声辅助将有机锡提取出来，有机溶剂萃取，提取后的样品溶液经凝胶渗透色谱净化、戊基格林试剂衍生、衍生化产物再经弗罗里硅土(Florisil)净化，采用气相色谱-脉冲火焰光度检测器(GC-PFPD)测定。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 正己烷 $(n - C_6H_{14})$ ：重蒸。  
3.1.2 四氢呋喃 $\mathrm{(CH_2CH_2OCH_2CH_2)}$ ：重蒸。  
3.1.3 乙酸乙酯 $\mathrm{(CH_3COOC_2H_5)}$ ：重蒸。  
3.1.4 环己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{12}\right)$ ：重蒸。  
3.1.5 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。  
3.1.6 无水乙醚 $(\mathrm{C}_2\mathrm{H}_5\mathrm{OC}_2\mathrm{H}_5)$ 。  
3.1.7 溴代正戊烷 $(n - \mathrm{C}_5\mathrm{H}_{12}\mathrm{Br})$ 。  
3.1.8 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：将无水硫酸钠置干燥箱中，于 $120^{\circ}\mathrm{C}$ 干燥 $4\mathrm{h}$ ，冷却后，密闭保存。  
3.1.9 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.10 硫酸 $\mathrm{(H_2SO_4)}$ ：优级纯。  
3.1.11 盐酸（HCl）：优级纯。  
3.1.12 氢溴酸（HBr）。  
3.1.13 乙二胺四乙酸二钠（EDTANa2）。  
3.1.14 金属钠(Na)。  
3.1.15 镁条 $(\mathrm{Mg})$ 。  
3.1.16 环庚三烯酚酮(tropolone)： $98\%$ 。  
3.1.17 弗罗里硅土(Florisil)： $0.147 \mathrm{~mm} \sim 0.25 \mathrm{~mm}$ (60目～100目)， $120^{\circ} \mathrm{C}$ 烘烤 $12 \mathrm{~h}$ 。  
3.1.18 聚苯乙烯凝胶： $0.075 \mathrm{~mm} \sim 0.147 \mathrm{~mm}$ （100 目～200 目），或同类产品。

# 3.2 试剂配制

3.2.1 环庚三烯酚酮-正己烷溶液 $(0.03\%)$ ：量取正己烷 $100~\mathrm{mL}$ ，加入 $0.03\mathrm{g}$ 环庚三烯酚酮混匀。  
3.2.2 氯化钠溶液 $(20\mathrm{g / L})$ ：称取氯化钠 $100\mathrm{g}$ ，溶于水并稀释至 $500~\mathrm{mL}$ 。  
3.2.3 饱和氯化钠溶液：在 $100~\mathrm{mL}$ 水中加入过量氯化钠，水浴使溶解，恢复室温后要求结晶析出。  
3.2.4 甲醇-水溶液 $(4 + 1)$ : 取甲醇适量, 制备甲醇-水 $(4 + 1)$ 溶液, 并根据每 $1\mathrm{mL}$ 甲醇加 $1\mu \mathrm{L}$ 盐酸的要求, 加入盐酸, 制得含盐酸的甲醇-水 $(4 + 1)$ 溶液。  
3.2.5 氢溴酸-四氢呋喃溶液 $(1 + 20)$ ：取氢溴酸和四氢呋喃各适量，按照 $1 + 20$ 的比例制备成溶液。

# 3.3 标准品

3.3.1 有机锡标准品及其标准溶液：见附录A。除二甲基锡、三丁基锡纯度为 $95\%$ 外，其他标准品纯度均大于 $97\%$ 。  
3.3.2 内标：一甲基锡(MMT)和三丙基锡(TPrT)标准品：纯度 $>98\%$

# 3.4 标准溶液配制

3.4.1 有机锡标准贮备液：准确称取有机锡的标准品适量，置于 $10\mathrm{mL}$ 烧杯中，加入甲醇-水溶液(3.2.4)溶解，转移到 $10\mathrm{mL}$ 容量瓶中，并稀释至刻度。于 $-20^{\circ}\mathrm{C}$ 冰箱保存。  
3.4.2 内标标准贮备溶液：准确称取内标适量，置于 $10 \mathrm{~mL}$ 烧杯中，加入甲醇-水溶液(3.2.4)溶解，转移到 $10 \mathrm{~mL}$ 容量瓶中，并稀释至刻度。于 $-20^{\circ} \mathrm{C}$ 冰箱保存。  
3.4.3 有机锡标准及内标中间溶液：量取有机锡标准或内标贮备液适量，用甲醇-水溶液(3.2.4)稀释100倍，质量浓度见附录A。于 $-20^{\circ}\mathrm{C}$ 冰箱保存。  
3.4.4 有机锡标准及内标工作溶液：量取有机锡标准或内标中间液适量，置于 $10\mathrm{mL}$ 容量瓶中，加入甲醇-水溶液(3.2.4)并稀释至刻度，浓度见附录A。于 $-20^{\circ}\mathrm{C}$ 冰箱保存。

# 4 仪器和设备

4.1 气相色谱仪(GC)：配脉冲火焰光度检测器(PFPD)，硫滤光片。  
4.2 色谱柱：DB-1毛细管柱或等效柱，柱长 $30\mathrm{m}$ ，膜厚 $0.25\mu \mathrm{m}$ ，内径 $0.25\mathrm{mm}$ 。  
4.3 组织匀浆器。  
4.4 振荡器。  
4.5 超声波清洗器。  
4.6 旋转蒸发仪。  
4.7 氮气浓缩器。  
4.8 三口瓶、分液漏斗和加热回流装置。  
4.9 加热磁力搅拌装置。  
4.10 玻璃层析柱。  
4.11 分析天平：感量为 $1\mathrm{mg}$   
4.12 电加热套。

# 5 分析步骤

# 5.1 戊基格林试剂的合成

5.1.1 乙醚重蒸与除水：在 $35^{\circ} \mathrm{C}$ 下，采用全玻璃蒸馏装置重蒸两次。

在蒸馏的乙醚中，加入光洁金属钠片，至不再产生明显气泡后，放置 $1\mathrm{h}\sim 2\mathrm{h}$ ，继续加入金属钠片，保存备用。

5.1.2 溴代正戊烷的重蒸：量取溴代正戊烷 $100~\mathrm{mL}$ ，置于蒸馏瓶中，在 $140^{\circ}\mathrm{C}$ 下，采用全玻璃蒸馏装置重蒸两次，馏分避光收集。  
5.1.3 镁屑的制备：取镁条，刮去表面氧化膜层，用剪刀剪成约 $0.3\mathrm{mm}$ 的碎屑，用正己烷洗涤2次～3次，挥干，保存备用。  
5.1.4 戊基溴化镁的合成：称取镁屑 $10\mathrm{g}$ ，置 $500\mathrm{mL}$ 三口瓶中，加入重蒸乙醚 $100\mathrm{mL}$ ，加入搅拌子。在搅拌下，滴加重蒸溴代正戊烷 $60\mathrm{mL}$ 和重蒸乙醚 $50\mathrm{mL}$ 的混合溶液。反应发生后，停止搅拌。当反应速度减缓后，继续滴加上述溴代正戊烷和乙醚的混合溶液，并搅拌。调节滴加速度，使反应瓶中的乙醚保持微沸状态。当反应缓慢时，开始加热，保持反应发生，继续加热回流至反应完全，得到戊基溴化镁溶液约 $200\mathrm{mL}$ ，溶液呈灰黑色混浊状态。将合成的戊基溴化镁分装至棕色小瓶中，封口，干燥器内保存。

# 5.2 试样制备

鱼去皮及刺，制成肉糜；贝类试样取可食组织制成匀浆；葡萄酒和酱油试样混合均匀。鱼贝类试样制成匀浆后，可采用冷冻干燥，制成冻干粉末，置 $-20^{\circ}\mathrm{C}$ 以下低温保存。

# 5.2.1 试样提取

# 5.2.1.1 鱼类样品

准确称取试样适量，加入乙二胺四乙酸二钠 $0.15\mathrm{g}$ 和 $20\mathrm{g / L}$ 氯化钠溶液 $5\mathrm{mL}$ ，摇匀，加入内标工作溶液 $50~\mu \mathrm{L}$ ，加入氢溴酸-四氢呋喃溶液 $(1 + 20)(3.2.5)15\mathrm{mL}$ ，超声 $5\mathrm{min}$ 。如果为湿样，加入饱和氯化钠溶液 $1\mathrm{mL}$ ，摇匀，加入内标工作溶液 $50~\mu \mathrm{L}$ ，加入氢溴酸-四氢呋喃溶液 $(1 + 20)(3.2.5)15\mathrm{mL}$ ，超声 $5\mathrm{min}$ 。

# 5.2.1.2 贝类样品

准确称取试样适量，加入 $20\%$ 氯化钠溶液 $5\mathrm{mL}$ ，摇匀，加入内标工作溶液 $50~\mu \mathrm{L}$ ，加入氢溴酸-四氢呋喃溶液(1+20)(3.2.5) $15~\mathrm{mL}$ ，超声 $5\mathrm{min}$ 。

# 5.2.1.3 葡萄酒等液体样品

量取试样 $10\mathrm{mL}$ ，加入氯化钠 $2\mathrm{g}$ ，摇匀，加入内标工作溶液 $50~\mu \mathrm{L}$ ，加入氢溴酸-四氢呋喃溶液 $(1 + 20)$ (3.2.5) $15~\mathrm{mL}$ ，超声 $5\mathrm{min}$ 。

# 5.2.2 试样溶液的净化

在试样溶液中加入含 $0.03\%$ 环庚三烯酚酮的正己烷 $25~\mathrm{mL}$ ，振荡萃取 $40\mathrm{min}$ ，离心 $10\mathrm{min}$ $(3000\mathrm{r / min})$ ，静置分层，吸取有机相转移至茄形瓶中，在残渣中加入正己烷 $10~\mathrm{mL}$ ，振荡萃取 $10\mathrm{min}$ ，离心 $10\mathrm{min}(3000\mathrm{r / min})$ ，静置分层，吸取有机相。合并至茄形瓶中，旋转蒸发浓缩至近干。

# 5.2.3 凝胶渗透色谱净化

5.2.3.1 凝胶柱的装填：取聚苯乙烯凝胶，用四氢呋喃-乙酸乙酯 $(1 + 1)$ 溶液浸泡过夜。用玻璃棉封堵内径为 $1.7\mathrm{cm}\sim 1.8\mathrm{cm}$ 的玻璃层析柱底端，湿法加入浸泡好的凝胶，凝胶自然沉降，稳定后柱长约为 $15~\mathrm{cm}$ 。  
5.2.3.2 净化：在试样提取液的浓缩残渣中加入四氢呋喃-乙酸乙酯 $(1 + 1)$ 溶液 $1\mathrm{mL}$ ，将此溶液全部转

移至层析柱上，用 $1\mathrm{mL}$ 四氢呋喃-乙酸乙酯 $(1 + 1)$ 溶液洗涤茄形瓶。待层析柱中试液溶液的液面降至接近凝胶时，将洗液转移至柱上。用四氢呋喃-乙酸乙酯 $(1 + 1)$ 溶液洗脱，弃去 $0\mathrm{mL} \sim 18\mathrm{mL}$ 流分，收集 $18\mathrm{mL} \sim 33\mathrm{mL}$ 流分，收集流出体积 $15\mathrm{mL}$ 。

# 5.2.4 戊基溴化镁格林试剂衍生

将5.2.3.2收集的净化溶液旋转蒸发浓缩近干，加入环己烷 $10\mathrm{mL}$ ，继续旋转蒸发浓缩至约 $1\mathrm{mL}$ 转移至 $10~\mathrm{mL}$ 离心管中，用环己烷洗涤茄形瓶，合并在离心管中，并定容至 $2\mathrm{mL}$ 。用 $1\mathrm{mL}$ 注射器加取上述合成的戊基溴化镁格林试剂 $0.8\mathrm{mL}$ ，涡旋振荡混匀，超声反应 $15\mathrm{min}$ 后，逐滴加入 $0.5\mathrm{mol / L}$ 硫酸约 $3\mathrm{mL}$ ，振摇，终止衍生反应，涡旋振摇，静置使上层溶液澄清。

# 5.2.5 弗罗里硅土(Florisil)柱净化

5.2.5.1 层析柱装填：用玻璃棉封堵玻璃柱底端后，从底部到顶部依次装入活化弗罗里硅土(Florisil) $1.5\mathrm{g}$ 、无水硫酸钠 $2\mathrm{g}$ ，用正己烷 $10\mathrm{mL}$ 预淋洗。  
5.2.5.2 净化：将5.2.4衍生溶液的上层有机相全部转移至弗罗里硅土柱上，当柱中溶液的液面降至无水硫酸钠层时，用正己烷洗脱，收集洗脱液 $5\mathrm{mL}$ ，在氮气流下浓缩至约 $1\mathrm{mL}$ 后，转移至另一根填以 $1.5\mathrm{g}$ 弗罗里硅土(Florisil)和 $2\mathrm{g}$ 无水硫酸钠的层析柱上，用正己烷-甲苯 $(5 + 1)$ 溶液 $10\mathrm{mL}$ 预淋洗，用正己烷-甲苯 $(5 + 1)$ 溶液洗脱，收集 $10\mathrm{mL}$ 流分。

# 5.2.6 浓缩

在氮气流下，将5.2.5.2净化的试样溶液浓缩定容至 $1.0\mathrm{mL}$ ，转移进进样小瓶中，待GC测定。

# 5.3 仪器参考条件

气相色谱参考条件：

a）色谱柱<sup>1)</sup>：DB-1柱(或等效柱)，柱长 $30\mathrm{m}$ ，膜厚 $0.25\mu \mathrm{m}$ ，内径 $0.25\mathrm{mm}$ 。  
b）采用不分流方式，进样口温度 $280^{\circ}\mathrm{C}$ 。  
c）柱温程序：开始温度为 $50~\mathrm{^\circ C}$ ，保持 $1\mathrm{min}$ ；以 $10^{\circ}C / \mathrm{min}$ 升温至 $120^{\circ}C$ ； $5^{\circ}C / \mathrm{min}$ 升温至 $200^{\circ}C$ ； $10^{\circ}C / \mathrm{min}$ 升温至 $280^{\circ}C$ ，保持 $5\mathrm{min}$ 。  
d）载气为高纯氮气（纯度 $>99.999\%$ ）。  
e）脉冲火焰光度检测器参考条件：模式：硫滤光片；温度： $350^{\circ}\mathrm{C}$ ；燃气和助燃气流速：空气 $_1$ 21 $\mathrm{mL} / \mathrm{min}$ ，氢气 $22~\mathrm{mL} / \mathrm{min}$ ，空气 $_2$ 11 $\mathrm{mL} / \mathrm{min}$ ；光电倍增管电压： $550~\mathrm{V}$ ；门槛时间： $4~\mathrm{ms}$ ；门延迟时间： $5~\mathrm{ms}$ ；激发电压： $100~\mathrm{mV}$ 。

# 5.4 标准曲线制作

准确称取不含有机锡的对应食物样品作为空白基质适量，加入 $5\mathrm{mL}20\%$ 氯化钠溶液，分别加入有机锡混合标准溶液 $0\mu \mathrm{L},10\mu \mathrm{L},30\mu \mathrm{L},50\mu \mathrm{L},100\mu \mathrm{L},200\mu \mathrm{L},400\mu \mathrm{L}$ 及内标工作溶液 $50~\mu \mathrm{L}$ ，按试样提取与净化过程要求同步操作。吸取标准系列溶液 $1\mu \mathrm{L}$ 注入气相色谱仪进行分析，得到色谱图（参见附录B.1及附录B.2），以保留时间定性。

# 5.5 试样溶液的测定

吸取试样溶液 $1\mu \mathrm{L}$ 注入气相色谱仪中，得到色谱图(参见附录B.3)，以保留时间定性。计算目标

化合物与内标的峰面积或峰高比，以标准系列溶液中目标有机锡的进样量(ng)与对应的目标有机锡与内标的峰面积或峰高比绘制线性曲线，根据线性曲线计算试样中有机锡含量。

# 6 分析结果的表述

试样中目标有机锡含量(以Sn计)的质量分数按式(1)计算：

$$
X = \frac {A \times f}{m} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中目标有机锡含量(以 $\mathrm{Sn}$ 计)，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ 或微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$A$ ——试样色谱峰与内标色谱峰的峰高比值对应的目标有机锡质量(以Sn计)，单位为纳克 $(\mathrm{ng})$ ；

$f$ ——试样稀释因子；

$m$ ——试样称样量，单位为克(g)或毫升 $(\mathrm{mL})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

本方法定量限(以Sn计)为：二甲基锡 $0.5\mu \mathrm{g} / \mathrm{kg}$ 、三甲基锡 $1.2\mu \mathrm{g} / \mathrm{kg}$ 、一丁基锡 $1.5\mu \mathrm{g} / \mathrm{kg}$ 、二丁基锡 $0.5\mu \mathrm{g} / \mathrm{kg}$ 、三丁基锡 $0.6\mu \mathrm{g} / \mathrm{kg}$ 、一苯基锡 $1.7\mu \mathrm{g} / \mathrm{kg}$ 、二苯基锡 $0.8\mu \mathrm{g} / \mathrm{kg}$ 、三苯基锡 $0.8\mu \mathrm{g} / \mathrm{kg}$ 。

# 食品添加剂 丙二醇脂肪酸酯

# 1 范围

本标准适用于以脂肪酸或油脂与1,2-丙二醇反应生成的，经过或不经过分子蒸馏等加工得到的食品添加剂丙二醇脂肪酸酯。

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度应为分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

试样不溶于水，可溶于乙醇、乙酸乙酯、三氯甲烷和其他氯化烃类。

# A.3 总单丙二醇脂肪酸酯和游离丙二醇的测定

# A.3.1 方法提要

丙二醇脂肪酸酯或游离丙二醇和硅烷化试剂反应形成挥发性硅烷化衍生物，气相色谱测定。

# A.3.2 试剂和材料

A.3.2.1 吡啶。  
A.3.2.2 $N,N$ -双三甲基硅三氟乙酰胺（BSTFA）。  
A.3.2.3 三甲基氯硅烷（TMCS）。  
A.3.2.4 十七烷（内标物）。  
A.3.2.5 1,2-丙二醇标准品。  
A.3.2.6 丙二醇单硬脂酸酯标准品。

注：不同的产品应选用对应单丙二醇脂肪酸酯标准品试验。

# A.3.3 仪器和设备

A.3.3.1 气相色谱仪，配氢火焰离子化检测器。   
A.3.3.2 天平：感量 $0.01\mathrm{mg}$

# A.3.4 参考色谱条件

A.3.4.1 色谱柱： $5\%$ 苯基- $95\%$ 甲基聚硅氧烷毛细管柱，柱长 $30 \mathrm{~m}$ ，内径 $0.25 \mathrm{~mm}$ ，涂膜厚度 $0.25 \mu \mathrm{m}$ 。或等效色谱柱。  
A.3.4.2 柱温：初温 $50~\mathrm{^\circ C}$ ，以 $10^{\circ}C / \min$ 的速率升温至 $320^{\circ}C$ ，并维持 $1\mathrm{min}$   
A.3.4.3 进样口温度： $320^{\circ}C$   
A.3.4.4 检测器温度： $350^{\circ}C$   
A.3.4.5 载气：氮气。  
A.3.4.6 载气流速： $1\mathrm{mL / min}$   
A.3.4.7 氢气： $60~\mathrm{mL / min}$   
A.3.4.8 空气： $500~\mathrm{mL / min}$

A.3.4.9 分流比： $1:50$   
A.3.4.10 进样量： $1\mu \mathrm{L}$

# A.3.5 分析步骤

# A.3.5.1 内标溶液的制备

称取 $100\mathrm{mg}$ 十七烷(内标物)，精确至 $0.0001\mathrm{g}$ ，置于 $100\mathrm{mL}$ 容量瓶中，用吡啶溶解并定容。

# A.3.5.2 标准溶液的制备

称取丙二醇单硬脂酸酯标准品和丙二醇标准品各 $100\mathrm{mg}$ ，精确至 $0.0001\mathrm{g}$ ，置于 $25\mathrm{mL}$ 容量瓶中，用内标溶液溶解并定容。取 $0.8\mathrm{mL}$ 所得溶液置于 $2.5\mathrm{mL}$ 螺旋盖样品瓶中，加 $0.1\mathrm{mL}$ TMCS和 $0.3\mathrm{mL}$ BSTFA后猛烈摇匀，置于 $70^{\circ}C$ 烘箱中反应 $20\mathrm{min}$ ，得到标准溶液。

# A.3.5.3 试样溶液的制备

称取 $100\mathrm{mg}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于 $25\mathrm{mL}$ 容量瓶中，以内标溶液定容至刻度。取 $0.8\mathrm{mL}$ 所得溶液置于 $2.5\mathrm{mL}$ 螺旋盖样品瓶中，加 $0.1\mathrm{mL}$ TMCS和 $0.3\mathrm{mL}$ BSTFA后猛烈摇匀，置于 $70^{\circ}\mathrm{C}$ 烘箱中反应 $20\mathrm{min}$ ，得到试样溶液。

# A.3.5.4 测定

在参考色谱条件下，注入标准溶液和试样溶液进行测定。丙二醇单硬脂酸酯和1,2-丙二醇标准色谱图参见图B.1。

# A.3.6 结果计算

# A.3.6.1 反应因子 $\pmb{R}$ 的计算

反应因子 $R$ 按式(A.1)计算：

$$
R = \frac {A _ {\mathrm {S}}}{A _ {\mathrm {d}}} \times \frac {m _ {\mathrm {d}}}{m _ {\mathrm {S}}} \quad \dots \tag {A.1}
$$

式中：

$A_{\mathrm{S}}$ ——标准物质峰面积；

$A_{\mathrm{d}}$ ——内标物峰面积；

$m_{\mathrm{d}}$ ——标准溶液中内标物的质量，单位为毫克 $(\mathrm{mg})$ ；

$m_{\mathrm{S}}$ ——标准溶液中标准物质的质量，单位为毫克 $(\mathrm{mg})$ 。

# A.3.6.2 待测组分质量分数的计算

总单丙二醇脂肪酸酯或游离丙二醇的质量分数 $w_{i}$ ，按式(A.2)计算：

$$
w _ {i} = \frac {A _ {i} \times m _ {\mathrm {d}}}{A _ {\mathrm {d}} \times m \times R} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$A_{i}$ ——试样溶液中待测组分峰面积；

$m_{\mathrm{d}}$ ——试样溶液中内标物的质量，单位为毫克 $(\mathrm{mg})$ ；

$A_{\mathrm{d}}$ ——试样溶液中内标物峰面积；

$m$ ——试样的质量，单位为毫克 $(\mathrm{mg})$ ；

$R$ ——反应因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

# A.4 酸值的测定

# A.4.1 试剂和材料

A.4.1.1 $95\%$ 乙醇。

A.4.1.2 氢氧化钠标准溶液： $c(\mathrm{NaOH}) = 0.1\mathrm{mol / L}$

A.4.1.3 酚酞指示液： $10\mathrm{g / L}$

A.4.1.4 中性热乙醇：在加热后（温度高于 $70^{\circ}\mathrm{C}$ ）的 $95\%$ 乙醇中加入酚酞指示液，用氢氧化钠标准溶液中和至微红色，并保持 $30\mathrm{~s}$ 不褪色。

# A.4.2 分析步骤

称取试样 $5\mathrm{g}$ ，精确至 $0.001\mathrm{g}$ ，置于一个 $500~\mathrm{mL}$ 锥形瓶中，加入 $75~\mathrm{mL} \sim 100~\mathrm{mL}$ 中性热乙醇，搅拌并继续加热，使试样完全溶解。加入 $0.5\mathrm{mL}$ 酚酞指示液，边振摇，边立即用氢氧化钠标准溶液滴定，滴定至溶液呈微红色并维持 $30~\mathrm{s}$ 不褪色即为滴定终点。

# A.4.3 结果计算

酸值(以KOH计)的质量分数 $\varpi_{1}$ ，以毫克每克 $(\mathrm{mg / g})$ 计，按式(A.3)计算：

$$
w _ {1} = \frac {V \times c \times M}{m} \quad \dots \tag {A.3}
$$

式中：

$V$ ——试样所消耗氢氧化钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——氢氧化钠标准溶液的试剂浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——氢氧化钾的摩尔质量，单位为毫克每毫摩尔 $(\mathrm{mg / mol})[M(\mathrm{KOH}) = 56.1]$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# A.5 皂质（以硬脂酸钾计）的测定

# A.5.1 试剂和材料

A.5.1.1 丙酮。

A.5.1.2 盐酸标准滴定溶液： $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$

A.5.1.3 溴酚蓝指示液： $1\mathrm{g / L}$

# A.5.2 分析步骤

称取试样 $5\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，加入 $100\mathrm{mL}$ 预先用盐酸标准滴定溶液调节至中性的丙酮中，加入 $5$ 滴～6滴溴酚蓝指示液，在水浴中温热至试样完全溶解。用盐酸标准滴定溶液进行滴定，滴定至溶液呈黄色，并维持 $30\mathrm{s}$ 不褪色即为滴定终点。

# A.5.3 结果计算

皂质（以硬脂酸钾计）的质量分数 $\varpi_{2}$ ，按式(A.4)计算：

$$
w _ {2} = \frac {V \times c \times e}{m} \quad \dots \tag {A.4}
$$

式中：

$V$ ——滴定时消耗的盐酸标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——盐酸标准溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

e——当量因子，31.0；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不得超过 $0.1\%$ 。

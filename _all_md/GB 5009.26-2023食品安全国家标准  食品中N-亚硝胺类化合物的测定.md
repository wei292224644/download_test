2023-09-06 发布

# 前言

本标准代替GB5009.26—2016《食品安全国家标准 食品中 $N$ -亚硝胺类化合物的测定》。

本标准与GB5009.26—2016相比，主要变化如下：

——增加了第一法水蒸气蒸馏-气相色谱-质谱/质谱法；  
——增加了第二法 QuEChERS-气相色谱-质谱/质谱法；  
——增加了第三法水蒸气蒸馏-液相色谱-质谱/质谱法；  
——删除了气相色谱-质谱法；  
——修改了原第二法气相色谱-热能分析仪法为第四法；  
——修改了前处理蒸馏法条件、气相色谱-热能分析仪法的检出限和定量限。

# 食品安全国家标准

# 食品中 $N$ -亚硝胺类化合物的测定

# 1 范围

本标准规定了食品中 $N\cdot$ 二甲基亚硝胺(NDMA)的测定方法。

本标准适用于肉及肉制品、水产动物及其制品中 $N$ -二甲基亚硝胺的测定。

# 第一法 水蒸气蒸馏-气相色谱-质谱/质谱法

# 2 原理

本法以 $N$ -二甲基亚硝胺- $\mathrm{D}_6$ 为内标，试样中加入内标，经水蒸气蒸馏，样品中的 $N$ -二甲基亚硝胺通过二氯甲烷吸收，液液萃取分离，采用气相色谱-质谱/质谱仪(GC-MS/MS)测定，内标法定量。

# 3 试剂与材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：色谱纯。  
3.1.2 浓硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ ： $18.4~\mathrm{mol / L}$   
3.1.3 异辛烷 $\left(\mathrm{C}_{8} \mathrm{H}_{18}\right)$ ：色谱纯。  
3.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.5 氯化钠 $\mathrm{(NaCl)}$ ：优级纯。

# 3.2 试剂配制

硫酸溶液 $(1 + 3)$ ：量取 $30~\mathrm{mL}$ 浓硫酸，缓缓倒入 $90~\mathrm{mL}$ 冷水中，一边搅拌使得充分散热，冷却后小心混匀。

# 3.3 标准品

3.3.1 $N$ -二甲基亚硝胺标准溶液 $\left(\mathrm{C}_{2} \mathrm{H}_{6} \mathrm{~N}_{2} \mathrm{O}, \mathrm{CAS}\right.$ 号：62-75-9）：质量浓度为 $1000 \mu \mathrm{g} / \mathrm{mL}$ 的 $N$ -二甲基亚硝胺甲醇溶液，或经国家认证并授予标准物质证书的标准品。  
3.3.2 $N$ -二甲基亚硝胺- $\mathrm{D}_6$ 内标标准溶液（NDMA- $\mathrm{D}_6$ ， $\mathrm{C}_2\mathrm{D}_6\mathrm{N}_2\mathrm{O}$ ,CAS号：17829-05-9）：质量浓度为 $1000~\mu \mathrm{g / mL}$ ，溶剂为甲醇。

# 3.4 标准溶液配制

3.4.1 $N$ -二甲基亚硝胺标准储备液 $(100~\mu \mathrm{g / mL})$ ：准确吸取 $1.0\mathrm{mL}N\cdot$ 二甲基亚硝胺标准溶液 $(1000\mu \mathrm{g / mL})$ ，置于 $10~\mathrm{mL}$ 容量瓶中，用二氯甲烷定容至刻度，混匀。将溶液转移至棕色玻璃容器内，

$-18^{\circ} \mathrm{C}$ 避光保存，保存期6个月。

3.4.2 $N$ -二甲基亚硝胺标准中间液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取 $1.0\mathrm{mL}N$ -二甲基亚硝胺标准储备溶液 $(100~\mu \mathrm{g / mL})$ ，置于 $100~\mathrm{mL}$ 容量瓶中，用二氯甲烷定容至刻度，混匀。将溶液转移至棕色玻璃容器内， $-18^{\circ}C$ 避光保存，保存期3个月。  
3.4.3 $N$ -二甲基亚硝胺- $\mathrm{D}_6$ 内标储备液（ $100~\mu \mathrm{g / mL}$ ）：准确吸取 $N$ -二甲基亚硝胺- $\mathrm{D}_6$ 标准溶液 $1.0~\mathrm{mL}$ ，置于 $10~\mathrm{mL}$ 容量瓶中，用二氯甲烷定容至刻度，混匀。将溶液转移至棕色玻璃容器内， $-18^{\circ}C$ 避光保存，保存期6个月。  
3.4.4 $N$ -二甲基亚硝胺- $\mathrm{D}_6$ 内标中间液（ $1\mu \mathrm{g} / \mathrm{mL}$ ）：准确吸取 $N$ -二甲基亚硝胺- $\mathrm{D}_6$ 内标储备液 $1.0\mathrm{mL}$ ，置于 $100\mathrm{mL}$ 容量瓶中，用二氯甲烷定容至刻度，混匀。将溶液转移至棕色玻璃容器内， $-18^{\circ}\mathrm{C}$ 避光保存，保存期3个月。  
3.4.5 $N$ -二甲基亚硝胺标准及内标混合系列工作液：分别准确吸取 $N$ -二甲基亚硝胺标准中间液 $(1\mu \mathrm{g} / \mathrm{mL})$ 0.1 mL、0.2 mL、0.5 mL、1.0 mL 和 2.0 mL，置于 10 mL 容量瓶中，各加内标中间液 $(1\mu \mathrm{g} / \mathrm{mL})$ 0.4 mL，用二氯甲烷定容至刻度，混匀。 $N$ -二甲基亚硝胺标准系列工作溶液质量浓度为 $10\mu \mathrm{g} / \mathrm{L}$ 、 $20\mu \mathrm{g} / \mathrm{L}$ 、 $50\mu \mathrm{g} / \mathrm{L}$ 、 $100\mu \mathrm{g} / \mathrm{L}$ 和 $200\mu \mathrm{g} / \mathrm{L}$ ，其中内标的质量浓度均为 $40\mu \mathrm{g} / \mathrm{L}$ 。临用现配。

# 5 分析步骤

# 5.1 样品前处理

# 5.1.1 试样制备

代表性样品取可食部分捣碎后，制备成均匀试样，装入洁净的容器，密封并做好标识。试样于 $-18^{\circ}\mathrm{C}$ 冷冻保存，备用。

# 5.1.2 提取

准确称取 $20\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 试样，加入 $N-$ 二甲基亚硝胺内标标准中间液 $(1\mu \mathrm{g} / \mathrm{mL})40\mu \mathrm{L}$ ，加入 $100\mathrm{mL}$ 水和 $50\mathrm{g}$ 氯化钠于蒸馏瓶(管)中，充分混匀。载样前后均需进行气密性检查。在 $250\mathrm{mL}$ 三角烧瓶中加入 $50\mathrm{mL}$ 二氯甲烷、 $0.5\mathrm{mL}$ 异辛烷，冷凝管(冷却水温度控制在 $10^{\circ}\mathrm{C} \sim 15^{\circ}\mathrm{C}$ ) 出口伸入二氯甲烷液面下，并将三角烧瓶置于冰浴中，开启蒸馏装置加热蒸馏(参考条件：自动蒸馏装置蒸汽功率设置为 $50\%$ )，收集 $200\mathrm{mL} \sim 250\mathrm{mL}$ 冷凝液(包含 $50\mathrm{mL}$ 二氯甲烷提取液)后关闭加热装置，停止蒸馏。

注：蒸汽功率设置目的为控制蒸汽量，蒸汽加热样品时不致使样品瀑沸至蒸馏管腔体外。

# 5.1.3 净化

在盛有冷凝液的三角瓶中加入 $15\mathrm{g}$ 氯化钠和 $2\mathrm{mL}$ 硫酸溶液，搅拌使氯化钠完全溶解。然后将溶液转移至 $500~\mathrm{mL}$ 分液漏斗中，振荡 $5\mathrm{min}$ ，必要时放气，静置分层后，将二氯甲烷层转移至另一圆底烧瓶中，再用 $120~\mathrm{mL}$ 二氯甲烷分3次萃取，每次 $40~\mathrm{mL}$ 萃取，合并4次二氯甲烷萃取液，总体积约为 $170~\mathrm{mL}$ 。

# 5.1.4 浓缩

将二氯甲烷萃取液用 $10\mathrm{g}$ 无水硫酸钠脱水后，于 $5^{\circ}\mathrm{C}\sim 18^{\circ}\mathrm{C}$ 水浴温度条件下旋转蒸发浓缩至 $5\mathrm{mL}\sim 10\mathrm{mL}$ ，转入试管中，控制氮吹温度在 $18^{\circ}\mathrm{C}\sim 25^{\circ}\mathrm{C}$ ，氮吹(参考条件：氮吹流速 $1\mathrm{L / min})$ 至近干 $(0.3\mathrm{mL}\sim 0.8\mathrm{mL})$ ，用二氯甲烷溶解残渣，准确定容至 $1.0~\mathrm{mL}$ ，上机检测。

注：控制氮吹流速，不致使液体飞溅管壁。

# 5.2 仪器参考条件

# 5.2.1 气相色谱条件

a）色谱柱：强极性石英毛细管WAX柱，固定相为聚乙二醇 $[30\mathrm{m}\times 0.25\mathrm{mm}$ （内径） $\times 0.25\mu \mathrm{m}$ （膜厚）]，或相当者。  
b）进样口温度： $220^{\circ}C$ 。  
c）载气：氦气，纯度 $\geqslant 99.999\%$ ；流速 $1\mathrm{mL} / \mathrm{min}$ 。  
d）进样方式：不分流进样。  
e) 进样量: $1 \mu \mathrm{L}$ 。  
f）升温程序：初始温度 $40^{\circ}\mathrm{C}$ ，以 $10^{\circ}\mathrm{C / min}$ 升至 $80^{\circ}\mathrm{C}$ ，以 $1^{\circ}\mathrm{C / min}$ 升至 $90^{\circ}\mathrm{C}$ ，再以 $30^{\circ}\mathrm{C / min}$ 升至 $240^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$ 。

# 5.2.2 质谱/质谱条件

a）离子源温度： $250~^\circ \mathrm{C}$ 。  
b）色谱与质谱接口温度： $250^{\circ}\mathrm{C}$ 。  
c）电离方式：电子轰击源（EI源）。  
d）电离能量： $70\mathrm{eV}$ 。  
e) 溶剂延迟： $6\mathrm{min}$ 。  
f） 四级杆温度： $150~^\circ \mathrm{C}$ 。  
g）监测模式：多反应监测(MRM)模式，离子对参数见表1。

# 5.4.2 定量测定

将试样溶液注入气相色谱-质谱/质谱仪中，得到 $N$ -二甲基亚硝胺的峰面积与对应同位素内标的峰面积的比值，根据标准曲线得到试样溶液中被测化合物的浓度。样液中 $N$ -二甲基亚硝胺的峰面积比值应在标准曲线的线性范围内；如果含量超过标准曲线范围，需调整称样量或定容体积重新检测，若调整定容体积，应调整内标的添加量，使最终定容后内标与标准曲线内标的质量浓度一致。

# 5.5 空白试验

用水代替试样，按5.1及5.2所述操作步骤进行，必要时，进行环境空白的测定，空白测定值需低于该方法的定量限的 $50\%$ 。

# 6 分析结果的表述

试样中 $N$ -二甲基亚硝胺含量按式(1)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中 $N$ -二甲基亚硝胺的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——试样中 $N$ -二甲基亚硝胺色谱峰与对应内标物色谱峰的峰面积比值，经标准曲线求得的对应 $N$ -二甲基亚硝胺质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白试验溶液中 $N-$ 二甲基亚硝胺色谱峰与对应内标物色谱峰的峰面积比值，经标准曲线求得的对应 $N-$ 二甲基亚硝胺质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果保留小数点后2位数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

# 8.1 方法检测限

称样量为 $20\mathrm{g}$ ，定容体积为 $1.0\mathrm{mL}$ 时，本方法检出限为 $0.30~\mu \mathrm{g / kg}$ ，定量限为 $1.00~\mu \mathrm{g / kg}$ 。

# 8.2 安全警示

本试验所涉及的对人体可能造成危害的试剂包括有机溶剂（如乙腈、正己烷）等。试验人员有可能因接触这些物质而带来风险，因而，涉及这些物质的试验操作中应在通风橱内进行，并佩戴专业的防护工具（如手套、口罩等）。

亚硝胺类化合物是强致癌物，试验结束后，随意丢弃这些物质可能会对人员及环境造成损害，同时，在处理不当时，试验用的 $N$ -二甲基亚硝胺标准品及过期标准物质也可能带来风险，因而，在试验过程中不得随意丢弃这些物质，应分别回收在密闭容器并保存在远离火源、通风良好的环境中。

# 第二法 QuEChERS-气相色谱-质谱/质谱法

# 9 原理

本法以 $N$ -二甲基亚硝胺- $\mathrm{D}_6$ 为内标，试样中加入内标，试样中的 $N$ -二甲基亚硝胺经乙腈提取，基质分散萃取填料净化，采用气相色谱-质谱/质谱仪测定，内标法定量。

# 10 试剂与材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 10.1 试剂

10.1.1 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ ：色谱纯。  
10.1.2 硫酸镁 $(\mathrm{MgSO_4})$ 。  
10.1.3 氯化钠(NaCl)：优级纯。  
10.1.4PLS-A粉末：吡咯烷酮修饰聚苯乙烯二乙烯苯聚合物粉末或EMR-Lipid萃取粉剂：增强型去脂分散粉末。  
10.1.5 微孔滤膜 $(0.22\mu \mathrm{m})$ ，有机相型。

# 10.2 标准品

同3.3。

# 26.2.2 热能分析仪条件

a）接口温度： $250~^\circ \mathrm{C}$ 。  
b）热解室温度： $500^{\circ}C$ 。  
c）真空度： $59.8\mathrm{Pa}\sim 66.5\mathrm{Pa}$   
d) 氧气压力: $2 \mathrm{psi}(13790 \mathrm{~Pa})$ 。  
e）臭氧水平：244。

# 26.3 标准曲线的制作

将 $N$ -二甲基亚硝胺标准系列工作溶液按浓度由低到高的顺序注入气相色谱-热能分析仪中进样分析，以 $N$ -二甲基亚硝胺的质量浓度为横坐标，以 $N$ -二甲基亚硝胺的峰面积为纵坐标，绘制标准曲线。 $N$ -二甲基亚硝胺的色谱图参见附录D中图D.1。

# 26.4 试样溶液的测定

将标准系列溶液和试样溶液注入 GC-TEA 中, 用保留时间定性, 测定试样溶液中 $N$ -二甲基亚硝胺的峰面积, 根据标准曲线得到试样溶液中 $N$ -二甲基亚硝胺的浓度。标准工作液和样液中 $N$ -二甲基亚硝化合物的响应值均应在仪器线性响应范围内。

# 26.5 空白试验

用水代替试样，按26.1及26.2所述操作步骤进行，必要时，进行环境空白的测定。空白测定值需低于该方法定量限的 $50\%$ 。

# 27 分析结果的表述

试样中 $N$ -二甲基亚硝胺含量按式(4)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {4}
$$

式中：

$X$ ——试样中 $N$ -二甲基亚硝胺的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——由标准曲线得到的试样中 $N-$ 二甲基亚硝胺质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——由标准曲线得到的空白试验溶液中 $N-$ 二甲基亚硝胺质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$V$ ——试液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果保留小数点后2位数字。

# 28 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 29.2 安全警示

本试验所涉及的对人体可能造成危害的试剂包括有机溶剂（如乙腈、正己烷）等。试验人员有可能因接触这些物质而带来风险，因而，涉及这些物质的试验操作中应在通风橱内进行，并佩戴专业的防护工具（如手套、口罩等）。

亚硝胺类化合物是强致癌物，试验结束后，随意丢弃这些物质可能会对人员及环境造成损害，同时，在处理不当时，试验用的 $N$ -二甲基亚硝胺标准品及过期标准物质也可能带来风险，因而，在试验过程中不得随意丢弃这些物质，应分别回收在密闭容器并保存在远离火源、通风良好的环境中。

# 食品安全国家标准

# 蜂产品中喹诺酮类药物多残留的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了蜂蜜和蜂王浆中喹诺酮类药物残留量检测的制样和液相色谱-串联质谱检测方法。

本文件适用于蜂蜜和蜂王浆中噁喹酸、萘啶酸、氟甲喹、西诺沙星、二氟沙星、沙拉沙星、氧氟沙星、恩诺沙星、洛美沙星、培氟沙星、诺氟沙星、环丙沙星、依诺沙星、达氟沙星、马波沙星、氟罗沙星、加替沙星、奥比沙星、吡哌酸和司帕沙星单个或多个药物残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语与定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的喹诺酮类药物经磷酸盐缓冲液提取，HLB固相萃取柱净化，液相色谱-串联质谱测定，基质匹配标准曲线校正，外标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.3 甲酸（CHOOH）：色谱纯。  
5.1.4 乙酸乙酯 $(\mathrm{C_4H_8O_2})$   
5.1.5 磷酸氢二钠 $(\mathrm{Na_2HPO_4}\cdot 12\mathrm{H_2O})$   
5.1.6 磷酸二氢钠 $(\mathrm{NaH_2PO_4}\cdot 2\mathrm{H_2O})$   
5.1.7 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。

# 5.2 溶液配制

5.2.1 磷酸盐缓冲液：取磷酸氢二钠 $7.16\mathrm{g}$ 、磷酸二氢钠 $3.12\mathrm{g}$ ，加水 $800\mathrm{mL}$ 使溶解，用磷酸调 $\mathrm{pH}$ 至 $3.0 \pm 0.05$ ，用水稀释至 $1000\mathrm{mL}$ 。  
5.2.2 $5\%$ 甲酸甲醇溶液：取甲酸 $5\mathrm{mL}$ ，加甲醇 $95~\mathrm{mL}$ ，混匀。  
5.2.3 $0.1\%$ 甲酸溶液：取甲酸 $0.5 \mathrm{~mL}$ ，加水稀释至 $500 \mathrm{~mL}$ ，混匀。  
5.2.4 $20\%$ 乙腈甲酸溶液：取乙腈 $20~\mathrm{mL}$ ，加入 $0.1\%$ 甲酸溶液 $80~\mathrm{mL}$ ，混匀。

# 5.3 标准品

噁喹酸、氟甲喹、西诺沙星、二氟沙星、沙拉沙星、氧氟沙星、恩诺沙星、洛美沙星、培氟沙星、诺氟沙星、

环丙沙星、萘啶酸、依诺沙星、达氟沙星、马波沙星、氟罗沙星、加替沙星、奥比沙星、吡哌酸和司帕沙星，含量均 $\geq 95\%$ ，具体信息见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液 $(1\mathrm{mg / mL})$ ：取噁喹酸、氟甲喹、西诺沙星、二氟沙星、沙拉沙星、氧氟沙星、恩诺沙星、洛美沙星、培氟沙星、诺氟沙星、环丙沙星、萘啶酸、依诺沙星、达氟沙星、马波沙星、氟罗沙星、加替沙星、奥比沙星、吡哌酸和司帕沙星对照品各 $10\mathrm{mg}$ ，精密称定，用 $0.5\mathrm{mL}$ 氨水溶解后，用甲醇稀释定容至 $10\mathrm{mL}$ 容量瓶，配制成浓度为 $1\mathrm{mg / mL}$ 的标准储备液。 $-18^{\circ}C$ 避光保存，有效期6个月。  
5.4.2 混合标准工作液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：准确量取上述标准储备液各 $1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g} / \mathrm{mL}$ 的混合标准工作液。 $4^{\circ}\mathrm{C}$ 避光保存，有效期1个月。

# 5.5 材料

5.5.1 HLB固相萃取柱： $200\mathrm{mg / 6mL}$ ，或相当者。  
5.5.2 微孔尼龙滤膜： $0.22\mu \mathrm{m}$   
5.5.3 玻璃纤维滤纸： $1.6\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 涡旋混合器。  
6.4 涡旋振荡器。  
6.5 高速冷冻离心机：转速可达 $20000\mathrm{r / min}$   
6.6 超声波清洗机。  
6.7 固相萃取装置。  
6.8 氮吹仪。

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试蜂产品，将其搅拌均匀。对有结晶的蜂蜜样品，在密闭情况下，置于不超过 $60^{\circ}\mathrm{C}$ 的水浴中解晶，待样品全部融化后搅匀，冷却至室温。制备好的试料置于样品瓶中，密封，并做上标记。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

蜂蜜样品于室温下避光保存；蜂王浆样品于 $-18^{\circ}C$ 下保存。

# 8 测定步骤

# 8.1 提取

# 8.1.1 蜂蜜

取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，于 $50~\mathrm{mL}$ 聚丙烯离心管中，加磷酸盐缓冲液 $20~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，振荡 $10\mathrm{min},10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液，备用。

# 8.1.2 蜂王浆

取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，于 $50~\mathrm{mL}$ 聚丙烯离心管中，加磷酸盐缓冲液 $10~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，振荡 $10\mathrm{min},4^{\circ}C$ 下 $10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，移取上清液至另一离心管。残渣加磷酸盐缓冲液 $10~\mathrm{mL}$ 重复提取，合并2次上清液，玻璃纤维滤纸过滤，备用。

# 8.2 净化

HLB固相萃取柱分别用甲醇 $10\mathrm{mL}$ 、水 $10\mathrm{mL}$ 活化，备用液过柱，用水 $5\mathrm{mL}$ 淋洗，抽干，依次用 $5\%$ 甲酸甲醇和乙酸乙酯各 $4\mathrm{mL}$ 洗脱，抽干，收集洗脱液， $40^{\circ}C$ 氮气吹干。用 $20\%$ 乙腈甲酸水溶液 $1.00\mathrm{mL}$ 溶解残余物，超声 $1\mathrm{min}$ ，涡旋混匀（蜂王浆样品溶液需在 $4^{\circ}C$ 下 $20000\mathrm{r / min}$ 离心 $10\mathrm{min}$ )，上清液过微孔尼龙滤膜，供液相色谱-串联质谱测定。

# 8.3 基质匹配标准曲线的制备

准确量取混合标准工作液适量，用 $20\%$ 乙腈甲酸水溶液稀释，配制成浓度为 $1\mathrm{ng / mL},2\mathrm{ng / mL}$ $5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL}$ 和 $50~\mathrm{ng / mL}$ 的系列标准溶液，各取 $1.00~\mathrm{mL}$ ，分别加于经提取、净化步骤处理的空白试料中，溶解残余物（蜂王浆基质匹配标准溶液需在 $4^{\circ}C$ 下 $20000\mathrm{r / min}$ 离心 $10\mathrm{min}$ )，上清液过微孔尼龙滤膜后，供液相色谱串联质谱测定。以各药物特征离子质量色谱峰面积为纵坐标，标准溶液浓度为横坐标，绘制基质匹配标准曲线。求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在同样测试条件下，试料溶液中喹诺酮类药物峰的保留时间与基质匹配标准溶液相应峰的保留时间相对偏差在 $\pm 2.5\%$ 以内，且检测到的相对离子丰度，应当与浓度相当的基质匹配标准溶液相对丰度一致。其允许偏差应符合表3的要求。

# 8.4.3.2 定量测定

取试料溶液和相应的基质匹配标准溶液，作单点或多点校准，以特征离子质量色谱峰面积定量，按外标法计算。基质匹配标准工作液及试料溶液中的喹诺酮类药物响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，基质匹配标准溶液特征离子质量色谱图见附录B。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试料中喹诺酮类药物残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V}{A _ {\mathrm {S}} \times m} \tag {1}
$$

式中：

$X$ ——试料中相应的喹诺酮类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ 基质标准溶液中相应的喹诺酮类药物浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$A_{\mathrm{S}}$ 基质标准溶液中相应的喹诺酮类药物峰面积；

A ——试料溶液中相应的喹诺酮类药物峰面积；

$V$ ——溶解残余物溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1.0~\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $1\mu \mathrm{g} / \mathrm{kg}\sim 5\mu \mathrm{g} / \mathrm{kg}$ 添加浓度的回收率为 $60\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 20\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

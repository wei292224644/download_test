# 食品中植酸的测定

2016-08-31 发布

2017-03-01 实施

# 前言

本标准代替GB/T5009.153—2003《植物性食品中植酸的测定》。

本标准与GB/T5009.153—2003相比，主要修改如下：

——标准名称修改为“食品安全国家标准 食品中植酸的测定”；

——标准适用范围调整为食用油脂、加工水果、肉制品、鲜虾、糖果、果蔬饮料。

# 食品安全国家标准

# 食品中植酸的测定

# 1 范围

本标准规定了食品中植酸含量的测定方法。

本标准适用于食用油脂、加工水果、肉制品、鲜虾、糖果、果蔬饮料中植酸的测定。

# 2 原理

试样用酸性溶液提取，经阴离子交换树脂吸附和解吸附，洗脱液中的植酸与三氯化铁-磺基水杨酸混合液发生褪色反应，用分光光度计在波长 $500\mathrm{nm}$ 处测定吸光度，计算试样中植酸含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 氢氧化钠。  
3.1.2 氯化钠。  
3.1.3 三氯化铁。  
3.1.4 盐酸。  
3.1.5 磺基水杨酸。  
3.2 试剂配制  
3.2.1 $30\mathrm{g / L}$ 氢氧化钠溶液：称取氢氧化钠 $30\mathrm{g}$ ，用水溶解定容至 $1000~\mathrm{mL}$ 。  
3.2.2 $0.7\mathrm{mol} / \mathrm{L}$ 氯化钠溶液：称取氯化钠 $40.91\mathrm{g}$ ，用水溶解定容至 $1000~\mathrm{mL}$ 。  
3.2.3 $0.05\mathrm{mol} / \mathrm{L}$ 氯化钠溶液：称取氯化钠 $2.92\mathrm{g}$ ，用水溶解定容至 $1000~\mathrm{mL}$ 。  
3.2.4 $1.2\%$ 盐酸溶液：量取盐酸 $33.3 \mathrm{~mL}$ ，加入 $966.7 \mathrm{~mL}$ 水溶解。  
3.2.5 硫酸钠-盐酸提取溶液：称取 $100\mathrm{g}$ 无水硫酸钠溶于 $1.2\%$ 盐酸溶液，用 $1.2\%$ 盐酸溶液定容至 $1000~\mathrm{mL}$ 。  
3.2.6 三氯化铁-磺基水杨酸反应溶液：称取 $1.5\mathrm{g}$ 三氯化铁和 $15\mathrm{g}$ 磺基水杨酸，加水溶解并定容至 $500~\mathrm{mL}$ 。使用前用水稀释15倍。  
3.3 标准品

植酸钠标准品(CAS号：14306-25-3)，纯度 $\geq 85\%$

# 3.4 标准溶液配置

3.4.1 植酸标准溶液：准确称取 $1.65\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )植酸钠标准品，用水溶解定容至 $100~\mathrm{mL}$ ，配得浓度为 $10.0\mathrm{mg / mL}$ 植酸标准储备液。使用前，用水稀释至浓度 $0.1\mathrm{mg / mL}$ 。  
3.5 阴离子交换树脂：AG1-X4( $106\mu \mathrm{m}\sim 250\mu \mathrm{m})$ ，离子交换容量： $3.5\mathrm{mmol / g}$ （干）。

# 4 仪器和设备

4.1 天平：感量 $0.01\mathrm{g}$ 。  
4.2 振荡器。  
4.3 离心机： $5000\mathrm{r / min}$   
4.4 分光光度计。  
4.5 固相萃取空柱管： $\phi 0.8\mathrm{cm}\times 10\mathrm{cm}$ 或相当者。

# 5 试样制备与保存

# 5.1 试样制备

# 5.1.1 固体样品

取有代表性可食用部分，用组织捣碎机粉碎匀浆，混合均匀后装入洁净容器内密封并做好标识。

# 5.1.2 液体样品

取有代表性的样品混合均匀后，装入洁净容器内密封并做好标识。

# 5.2 试样保存

试样于 $-18^{\circ}\mathrm{C}$ 冰箱内保存。

注：制样和样品保存过程中，应防止样品受到污染和待测物损失。

# 6 分析步骤

# 6.1 提取

称取试样 $10.0\mathrm{g}$ ，置于具塞三角瓶中，加入 $40~\mathrm{mL}$ 硫酸钠-盐酸提取溶液(3.2.5)，振荡提取 $2\mathrm{h}$ ，提取液于 $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集全部上清液并用硫酸钠-盐酸提取溶液定容至 $50~\mathrm{mL}$ ，经快速滤纸过滤后备用。

# 6.2 净化

取 $0.5\mathrm{g}$ 阴离子交换树脂(3.5)湿法装入空柱管(4.5)中，分别用 $15~\mathrm{mL}$ 氯化钠溶液(3.2.2)和 $20~\mathrm{mL}$ 水洗涤离子交换柱。取 $5\mathrm{mL}$ （鲜虾样品取 $10~\mathrm{mL}$ )滤液，加入 $1\mathrm{mL}$ （鲜虾样品加 $2\mathrm{mL}$ ）氢氧化钠溶液(3.2.1)，用水稀释至 $30~\mathrm{mL}$ （鲜虾样品至 $60~\mathrm{mL}$ )，混匀后转入活化后的离子交换柱中，再分别用 $15~\mathrm{mL}$ 水和 $15~\mathrm{mL}$ 氯化钠溶液(3.2.3)以 $1\mathrm{mL / min}$ 的流速淋洗交换柱，弃去流出液。最后用 $25~\mathrm{mL}$ 氯化钠溶液(3.2.2)洗脱，收集全部洗脱液于 $25~\mathrm{mL}$ 具塞刻度管中，定容至刻度。

注：树脂装填时上下层需放置孔径 $20\mu \mathrm{m}\sim 50\mu \mathrm{m}$ 的筛板，并压实树脂。

# 7 分析步骤

# 7.1 标准曲线的制作

准确吸取植酸标准溶液(3.4.1) $0.0\mathrm{mL},0.04\mathrm{mL},0.1\mathrm{mL},1.0\mathrm{mL},2.0\mathrm{mL},5.0\mathrm{mL}$ 于6支 $10~\mathrm{mL}$ 比

色管中, 分别用水稀释至 $5 \mathrm{~mL}$ , 制得含植酸 $0.0 \mathrm{mg} 、 0.004 \mathrm{mg} 、 0.01 \mathrm{mg} 、 0.1 \mathrm{mg} 、 0.2 \mathrm{mg} 、 0.5 \mathrm{mg}$ 的系列标准溶液, 加入 $4 \mathrm{~mL}$ 反应溶液 (3.2.6), 混匀, 静置 $20 \mathrm{~min}$ 后取部分清液倒入 $1 \mathrm{~cm}$ 比色皿中, 于 $500 \mathrm{~nm}$ 处测定吸光度, 以吸光度为纵坐标, 植酸的质量为横坐标, 绘制标准曲线或计算回归方程。

# 7.2 测定

准确吸取 $5\mathrm{mL}6.2$ 所得洗脱液于 $10~\mathrm{mL}$ 比色管中，加入反应溶液(3.2.6)4 mL，混匀，静置 $20\mathrm{min}$ 后取部分清液倒入 $1\mathrm{cm}$ 比色皿中，于 $500~\mathrm{nm}$ 处测定吸光度，并在标准曲线上查得或从回归方程计算出试液中植酸含量。

# 8 分析结果的表述

试样中植酸含量按式(1)计算：

$$
X = \frac {m _ {2} \times 2 5 \times 1 0 0 0}{m _ {1} \times 5 \times V \times 1 0 0 0} \times 5 0 \tag {1}
$$

式中：

$X$ ——试样中植酸含量，单位为克每千克 $(\mathrm{g} / \mathrm{kg})$ ；

$m_{2}$ —— $5\mathrm{mL}$ 供测定用的试液中植酸的质量，单位为毫克 $(\mathrm{mg})$ ；

25 ——洗脱液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m_{1}$ ——试样质量，单位为克 $(\mathrm{g})$ ；

5 ——供测定用的试液体积，单位为毫升 $(\mathrm{mL})$ ；

$V$ ——供净化的提取液体积，单位为毫升 $(\mathrm{mL})$ ；

50 提取液定容体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留三位有效数字。

# 9 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 10 其他

本方法的检出限为 $0.006\mathrm{g / kg}$ ，定量限为 $0.02\mathrm{g / kg}$

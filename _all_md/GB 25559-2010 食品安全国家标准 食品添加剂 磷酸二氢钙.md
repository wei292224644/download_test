# 食品安全国家标准

# 食品添加剂 磷酸二氢钙

# 1 范围

本标准适用于以磷酸氢钙、磷酸三钙、石灰乳或碳酸钙之一与热法磷酸反应制得的食品添加剂磷酸二氢钙。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

# 3 分子式和相对分子质量

# 3.1 分子式

无水磷酸二氢钙：Ca（ $\mathrm{H}_2\mathrm{PO}_4$ ）

一水磷酸二氢钙： $\mathrm{Ca(H_2PO_4)_2\cdot H_2O}$

# 3.2相对分子质量

无水磷酸二氢钙：234.05（按2007年国际相对原子质量）

一水磷酸二氢钙：252.07（按2007年国际相对原子质量）

# 检验方法

# A.1 警示

本试验方法中使用的部分试剂具有毒性或腐蚀性，操作时须小心谨慎！如溅到皮肤上应立即用水冲洗，严重者应立即治疗。使用剧毒品，应严格按照有关规定管理；使用时应避免吸入或与皮肤接触，必要时应在通风橱中进行，暴露部位有伤口的人员不能接触。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008中规定的三级水。试验中所用标准滴定溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按HG/T3696.1、HG/T3696.2、HG/T3696.3之规定制备。

# A.3 鉴别试验

A.3.1 试剂和材料

A.3.1.1 硝酸溶液： $1 + 8$   
A.3.1.2 氨水溶液： $1 + 1$   
A.3.1.3 硝酸银溶液： $20\mathrm{g / L}$   
A.3.1.4 草酸铵溶液： $33\mathrm{g / L}$

A.3.2 分析步骤

A.3.2.1 钙离子的鉴别

称取约 $0.1 \mathrm{~g}$ 试样, 加 $20 \mathrm{~mL}$ 水摇匀后过滤, 滤液中加 $5 \mathrm{~mL}$ 草酸铵溶液, 产生白色沉淀。

A.3.2.2 磷酸根的鉴别

称取约 $1 \mathrm{~g}$ 试样, 溶于 $20 \mathrm{~mL}$ 水中, 加硝酸银溶液, 生成黄色沉淀, 此沉淀溶于氨水溶液或硝酸溶液。

# A.4 磷酸二氢钙的测定

A.4.1 氧化还原滴定法（仲裁法）

A.4.1.1 方法提要

试料经盐酸溶解，加入草酸铵溶液后用氨水调节溶液的pH值，在沸水浴中煮沸，生成草酸钙沉淀，用高锰酸钾标准滴定溶液滴定草酸钙沉淀，得出结果。

A.4.1.2 试剂和材料

A. 4.1.2.1 盐酸溶液: $1 + 3$ 。  
A.4.1.2.2 草酸铵溶液： $33\mathrm{g / L}$   
A. 4.1.2.3 洗涤液：取 $10 \mathrm{~mL}$ 草酸铵溶液，用水稀释至 $1000 \mathrm{~mL}$ 。  
A.4.1.2.4 氨水溶液： $1 + 1$   
A.4.1.2.5 硫酸溶液： $1 + 6$   
A.4.1.2.6 高锰酸钾标准滴定溶液： $c$ （ $1 / 5\mathrm{KMnO}_4)\approx 0.1\mathrm{mol} / \mathrm{L}$   
A.4.1.2.7 甲基橙指示液： $1\mathrm{g / L}$

A.4.1.2.8 甲基红指示液： $1\mathrm{g / L}$

# A.4.1.3 仪器和设备

A.4.1.3.1 恒温水浴箱。  
A.4.1.3.2 玻璃砂坩埚：孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$

# A.4.1.4 分析步骤

称取约 $0.5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $250\mathrm{mL}$ 烧杯中，加入 $10\mathrm{mL}$ 盐酸溶液，使试样溶解，加3滴甲基橙指示液，煮沸 $5\mathrm{min}$ 。如有必要，在煮沸过程中加入盐酸溶液或水使试验溶液的 $\mathrm{pH}$ 值和体积保持不变。加2滴甲基红指示液， $30\mathrm{mL}$ 草酸铵溶液，接着边搅拌边滴加氨水溶液，直至溶液的粉红色刚刚消失。将烧杯置于恒温水浴箱中煮沸 $30\mathrm{min}$ ，冷却至室温。待沉淀沉降，用倾析法将上层置于清洁干燥的玻璃砂坩埚抽滤，用 $30\mathrm{mL}$ 冷的洗涤液（ $20^{\circ}\mathrm{C}$ 以下）洗涤烧杯中的沉淀，洗涤液转移至玻璃砂坩埚抽滤，重复洗涤至沉淀全部转移至玻璃砂坩埚中，最后各用 $10\mathrm{mL}$ 水洗涤两次。将玻璃砂坩埚放入烧杯中，加入 $100\mathrm{mL}$ 水和 $50\mathrm{mL}$ 硫酸溶液，用滴定管加入 $35\mathrm{mL}$ 高锰酸钾标准滴定溶液，搅拌至颜色消失，加热至大约 $70^{\circ}\mathrm{C}$ ，用高锰酸钾标准滴定溶液滴定至溶液呈粉红色，并在 $30\mathrm{s}$ 内不消失即为终点。同时进行空白试验。

空白试验除不加试样外，其他操作及加入试剂的种类和量（标准滴定溶液除外）与测定试验相同。

# A.4.1.5 结果计算

磷酸二氢钙含量以钙（Ca）的质量分数 $w_{1}$ 计，数值以%表示，按公式（A.1）计算：

$$
w _ {1} = \frac {\left[ \left(V - V _ {0}\right) / 1000 \right] c M}{m} \times 100 \% \dots \tag{A.1}
$$

式中：

$V$ ——滴定试验溶液所消耗的高锰酸钾标准滴定溶液的体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验所消耗的高锰酸钾标准滴定溶液的体积的数值，单位为毫升（ $\mathrm{mL}$ ）；

$c$ ——高锰酸钾标准滴定溶液浓度的准确数值，单位为摩尔每升（mol/L）；

$m$ ——试料质量的数值，单位为克（g）；

$M$ ——钙（1/2Ca）的摩尔质量的数值，单位为克每摩尔（ $\mathrm{g/mol}$ ）（ $M=20.039$ ）。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.4.2 硫酸锌返滴定法

# A.4.2.1 方法提要

在试验溶液中，加入过量的乙二胺四乙酸二钠标准滴定溶液，与钙离子络合，以KB混合液为指示剂，用锌标准滴定溶液滴定过量的乙二胺四乙酸二钠标准滴定溶液。

# A.4.2.2 试剂和材料

A.4.2.2.1 盐酸溶液： $1 + 1$   
A.4.2.2.2 氨-氯化铵缓冲溶液（甲）： $\mathrm{pH}\approx 10$   
A.4.2.2.3 乙二胺四乙酸二钠标准滴定溶液： $c$ （EDTA） $\approx 0.05\mathrm{mol / L}$   
A. 4.2.2.4 酸性铬蓝K-萘酚绿B混合指示液（KB指示液）。  
A.4.2.2.5 硫酸锌标准滴定溶液： $c$ （ $\mathrm{ZnSO_4\cdot 7H_2O})\approx 0.05\mathrm{mol / L}$

配制：称取 $15\mathrm{g}$ 硫酸锌，加水溶解，用水稀释至 $1000\mathrm{mL}$ ，摇匀。

标定：移取 $25.00\mathrm{mL}$ 配制的硫酸锌标准滴定溶液，置于锥形瓶中，加 $10\mathrm{mL}$ 氨-氯化铵缓冲溶液（甲）

及 $75 \mathrm{~mL}$ 水，加约 $0.02 \mathrm{~g}$ 铬黑 T 指示剂，用 $0.05 \mathrm{~mol} / \mathrm{L}$ 的乙二胺四乙酸二钠标准滴定溶液滴定至溶液由紫色变为纯蓝色，并保持 $30 \mathrm{~s}$ 不褪色，即为终点。同时进行空白试验。

空白试验除不加试样外，其他操作及加入试剂的种类和量（标准滴定溶液除外）与测定试验相同。

硫酸锌标准滴定溶液的浓度按公式（A.2）计算：

$$
c = \frac {\left(V _ {1} - V _ {0}\right) c _ {1}}{V} \dots \tag {A.2}
$$

式中：

$V_{1}$ ——滴定所消耗的乙二胺四乙酸二钠标准滴定溶液的体积的数值，单位为毫升（ $\mathrm{mL}$ ）；

$V_{0}$ ——空白试验消耗的乙二胺四乙酸二钠标准滴定溶液的体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——乙二胺四乙酸二钠标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V$ ——移取硫酸锌标准滴定溶液的体积的数值，单位为毫升（ $\mathrm{mL}$ ）。

# A.4.2.3 分析步骤

称取约 $0.5 \mathrm{~g}$ 试样，精确至 $0.0002 \mathrm{~g}$ ，置于 $100 \mathrm{~mL}$ 烧杯中，用少量水润湿，加 $5 \mathrm{~mL}$ 盐酸溶液使试样全部溶解，全部转移至 $250 \mathrm{~mL}$ 容量瓶中，用水稀释至刻度，摇匀。必要时干过滤，弃去前 $20 \mathrm{~mL}$ 滤液。

移取 $25.00 \mathrm{~mL}$ 上述试验溶液，置于 $500 \mathrm{~mL}$ 锥形瓶中，再用另一移液管移入 $25.00 \mathrm{~mL}$ 乙二胺四乙酸二钠标准滴定溶液，加 $50 \mathrm{~mL}$ 水、 $10 \mathrm{~mL}$ 氨-氯化铵缓冲溶液（甲），摇匀，放置 $5 \mathrm{~min}$ 。加约2滴～3滴KB指示液，用硫酸锌标准滴定溶液滴定至溶液由蓝色变为蓝紫色，并保持 $30 \mathrm{~s}$ 内紫色不褪色，即为终点。同时进行空白试验。

空白试验除不加试样外，其他操作及加入试剂的种类和量（标准滴定溶液除外）与测定试验相同。

# A.4.2.4 结果计算

磷酸二氢钙含量以钙（Ca）的质量分数 $w_{1}$ 计，数值以%表示，按公式（A.3）计算：

$$
w _ {1} = \frac {\left[ \left(V _ {0} - V _ {1}\right) / 1000 \right] c M}{m \times 25 / 250} \times 100 \% \dots \tag{A.3}
$$

式中：

$V_{0}$ ——空白试验所消耗的硫酸锌标准滴定溶液的体积的数值，单位为毫升（ $\mathrm{mL}$ ）；

$V_{I}$ ——滴定试验溶液所消耗的硫酸锌标准滴定溶液的体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫酸锌标准滴定溶液浓度的准确数值，单位为摩尔每升（mol/L）；

$m$ ——试料质量的数值，单位为克（g）；

$M$ ——钙（Ca）的摩尔质量的数值，单位为克每摩尔（ $\mathrm{g/mol}$ ）（ $M=40.078$ ）。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.6.2.3 分析步骤

# A.6.2.3.1 铅标准测定溶液的制备及测定

# A.8.3 分析步骤

A.8.3.1 标准工作溶液的绘制

移取氟化物标准溶液 $1.00\mathrm{mL}$ 、 $2.00\mathrm{mL}$ 、 $3.00\mathrm{mL}$ 、 $4.00\mathrm{mL}$ 、 $5.00\mathrm{mL}$ 分别置于5个 $50~\mathrm{mL}$ 容量瓶中，各加入 $4\mathrm{mL}$ 盐酸溶液，加 $25~\mathrm{mL}$ 总离子强度缓冲液，用水稀释至刻度，摇匀。

称取约 $1.5 \mathrm{~g}$ 试样，精确至 $0.01 \mathrm{~g}$ ，加入 $4 \mathrm{~mL}$ 盐酸溶液， $25 \mathrm{~mL}$ 总离子强度缓冲液，溶解，置于 $100 \mathrm{~mL}$ 容量瓶中，用水稀释至刻度，摇匀。

A.8.3.3 测定

将氟电极和甘汞电极与电位计的负端和正端联接，将电极插入盛有水的 $50 \mathrm{~mL}$ 聚乙烯塑料烧杯中，预热仪器，在磁力搅拌器上以恒速搅拌，读取平衡电位值，更换浸泡电极的水2次～3次，至电极说明书中规定的电位值后，即可进行标准工作溶液和试样溶液的电位测定。

由低至高浓度分别测定氟标准工作溶液的平衡电位。以电极电位作纵坐标，氟离子浓度（ $\mathrm{mg/mL}$ ）作横坐标，在半对数坐标纸上绘制工作曲线。

同法测定试样溶液的平衡电位, 从工作曲线上查出试样中氟离子的浓度 $(\mathrm{mg} / \mathrm{mL})$ 。

# A.8.4 结果计算

氟化物含量以氟（F）的质量分数 $w_{2}$ 计，数值以 $\mathrm{mg / kg}$ 表示，按公式（A.4）计算：

$$
w _ {2} = \frac {c \times 1 0 0}{m \times 1 0 ^ {- 3}} \dots \tag {A.4}
$$

式中：

$c$ ——从工作曲线上查得的氟离子浓度的数值, 单位为毫克每毫升 $(\mathrm{mg} / \mathrm{mL})$ ;

$m$ ——试样的质量的数值，单位为克（g）。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $10\mathrm{mg / kg}$ 。

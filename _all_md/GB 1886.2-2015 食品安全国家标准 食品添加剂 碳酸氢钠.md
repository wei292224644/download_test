# 附录A

# 检验方法

# A.1 安全提示

本试验方法中使用的部分试剂具有毒性或腐蚀性，操作时应小心谨慎。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所需标准溶液、杂质标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 钠的鉴别

用盐酸润湿铂丝，在火焰上燃烧至无色，再蘸取少许试验溶液在火焰上燃烧，火焰即呈鲜黄色。

# A.3.2 碳酸氢盐的鉴别

A.3.2.1 取试样少许，加盐酸溶液 $(1 + 2)$ 后可产生气体，该气体通入氢氧化钙溶液 $(3\mathrm{g} / \mathrm{L})$ 中有白色沉淀产生。  
A.3.2.2 在试验溶液中滴加硫酸镁溶液 $(120\mathrm{g / L})$ 时，在常温下无沉淀，煮沸后产生白色沉淀。

# A.4 总碱量（以 $\mathrm{NaHCO_3}$ 计）的测定

# A.4.1 方法提要

试样溶于水，以溴甲酚绿-甲基红作指示剂，用盐酸标准滴定溶液滴定。

# A.4.2 试剂和材料

A.4.2.1 盐酸标准滴定溶液： $c(\mathrm{HCl}) = 1\mathrm{mol} / \mathrm{L}$   
A.4.2.2 溴甲酚绿-甲基红指示液。

# A.4.3 分析步骤

称取约 $2.5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 锥形瓶中，加 $50~\mathrm{mL}$ 水使全部溶解。滴加10滴溴甲酚绿-甲基红指示液，用盐酸标准滴定溶液滴定至试验溶液由绿色变为暗红色后，煮沸 $2\mathrm{min}$ ，冷却至室温，用盐酸标准滴定溶液继续滴定至暗红色为终点。

同时进行空白试验。空白试验应与测定平行进行，并采用相同的分析步骤，取相同量的所有试剂（标准滴定溶液除外），但空白试验不加试样。

# A.4.4 结果计算

总碱量(以 $\mathrm{NaHCO_3}$ 计)的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {c \times \left(V _ {1} - V _ {0}\right) \times M}{m \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$c$ ——盐酸标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{1}$ ——滴定试验溶液所消耗的盐酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验所消耗的盐酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$M$ ——碳酸氢钠的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(\mathrm{NaHCO}_3) = 84.01]$

$m$ ——试样的质量，单位为克(g)；

1000——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于 $0.2\%$ 。

# A.5 干燥减量的测定

# A.5.1 方法提要

将试样置于真空干燥箱中放置 $4\mathrm{h}$ 后取出，测定其干燥减量。

# A.5.2 仪器和设备

A.5.2.1 称量瓶： $\Phi 50\mathrm{mm}\times 30\mathrm{mm}$

A.5.2.2 真空泵。

A.5.2.3 真空表： $-0.1\mathrm{MPa}$

A.5.2.4 真空干燥箱：温度能控制在 $40^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$

# A.5.3 分析步骤

用已于真空干燥箱中干燥至质量恒定的称量瓶，称取约 $5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ 。慢慢摇动称量瓶使试样厚度均匀，放入真空干燥箱中用真空泵抽取真空 $0.04\mathrm{MPa}$ ，并保持此真空度，在 $40^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 条件下，放置 $4\mathrm{h}$ ，取出称量。

# A.5.4 结果计算

干燥减量的质量分数 $w_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m _ {2} - m _ {3}}{m _ {1}} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$m_{2}$ ——干燥前称量瓶和试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{3}$ ——干燥后称量瓶和试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于 $0.02\%$ 。

# A.6 $\mathbf{pH}(10\mathrm{g / L}$ 水溶液）的测定

# A.6.1 仪器和设备

酸度计：精度为 $0.02\mathrm{pH}$ 单位。

# A.6.2 分析步骤

称取 $1.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $250~\mathrm{mL}$ 烧杯中。加入约 $100~\mathrm{mL}$ 无二氧化碳的水，使试样溶解，在 $10\mathrm{min}$ 内(从加水开始计时)按GB/T9724的规定进行测定。

# A.7 铵盐的测定

称取约 $1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ 。置于 $50~\mathrm{mL}$ 烧杯中，加 $10~\mathrm{mL}$ 水溶解。在加热至沸过程中无氨味。

# A.8 澄清度的测定

# A.8.1 方法提要

在室温下用定量水溶解试样，在相同条件下和标准比较。

# A.8.2 试剂和材料

A.8.2.1六次甲基四胺溶液： $100\mathrm{g / L}$ 。称取 $10.0\mathrm{g}\pm 0.1\mathrm{g}$ 预先于硅胶干燥器中干燥 $24\mathrm{h}$ 的六次甲基四胺，置于烧杯中，加少量水溶解，全部转移至 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。

A.8.2.2 硫酸联氨溶液： $10\mathrm{g / L}$ 。称取 $1.0\mathrm{g}\pm 0.1\mathrm{g}$ 预先于硅胶干燥器中干燥 $24\mathrm{h}$ 的硫酸联氨，置于烧杯中，加少量水溶解，全部转移至 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。

A.8.2.3 标准比浊溶液。用移液管移取 $25\mathrm{mL}$ 六次甲基四胺溶液和 $25\mathrm{mL}$ 硫酸联氨溶液，置于干燥的试剂瓶中，摇匀，室温下放置 $24\mathrm{h}$ ，制成标准比浊溶液A。此溶液有效期 $60\mathrm{d}$ 。用移液管移取 $10.0\mathrm{mL}$ 标准比浊溶液A，置于 $1000\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀，制成标准比浊溶液B。此溶液有效期 $1\mathrm{d}$ 。

# A.8.3 分析步骤

称取 $1.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $25~\mathrm{mL}$ 比色管中。加水溶解后，用水稀释至刻度，摇匀。放置 $5\mathrm{min}$ 后，与标准比浊溶液比较，对着黑色背景，从比色管上方观察，试验溶液的澄清度不得低于标准比浊溶液所示的澄清度。

标准比浊溶液是用移液管移取 $5.00\mathrm{mL}$ 标准比浊液B，置于 $25~\mathrm{mL}$ 比色管中，用水稀释至刻度，摇匀。

# A.9 氯化物（以CI计）的测定

# A.9.1 汞量法（仲裁法）

# A.9.1.1 方法提要

同GB/T3051—2000第3章。

# A.9.1.2 试剂和材料

同GB/T3051—2000第4章。

# A.9.1.3 仪器和设备

同GB/T3051—2000第5章。

# A.9.1.4 分析步骤

称取约 $2\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ 。置于锥形瓶中，加 $40~\mathrm{mL}$ 水溶解。滴加2滴溴酚蓝指示液，滴加硝酸溶液中和至黄色，再滴加氢氧化钠溶液至呈蓝色，再用硝酸溶液调至恰呈黄色，并过量2滴～3滴。加入 $1\mathrm{mL}$ 二苯偶氮碳酰肼指示液，用硝酸汞标准滴定溶液滴定至溶液由黄色变为紫色，即为终点。

同时进行空白试验。空白试验应与测定平行进行，并采用相同的分析步骤，取相同量的所有试剂（标准滴定溶液除外），但空白试验不加试样。

保存滴定后的废液，按GB/T3051—2000附录D要求处理。

# A.9.1.5 结果计算

氯化物（以Cl计）的质量分数 $\omega_{3}$ ，按式(A.3)计算：

$$
w _ {3} = \frac {c _ {1} \times \left(V _ {3} - V _ {2}\right) \times M}{m _ {4} \times 1000} \times 100 \% \quad \dots \tag{A.3}
$$

式中：

$c_{1}$ ——硝酸汞标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{3}$ ——滴定试验溶液所消耗的硝酸汞标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——空白试验所消耗的硝酸汞标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$M$ ——氯的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(\mathrm{Cl}) = 35.45]$

$m_{4}$ ——试样的质量，单位为克 $(\mathrm{g})$

1000——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于 $0.01\%$ 。

# A.9.2 目视比浊法

# A.9.2.1 方法提要

在酸性介质中加入硝酸银溶液，银离子与氯离子生成白色的氯化银悬浊液，与同时同样处理的标准比浊溶液进行对比。

# A.9.2.2 试剂和材料

A.9.2.2.1 $95\%$ 乙醇。  
A.9.2.2.2 硝酸溶液： $1 + 6$   
A.9.2.2.3 硝酸银溶液： $17\mathrm{g / L}$   
A.9.2.2.4 氯化物标准溶液： $1\mathrm{mL}$ 溶液含氯 $(\mathrm{Cl})0.10\mathrm{mg}$

# A.9.2.3 分析步骤

称取 $1.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $50~\mathrm{mL}$ 烧杯中，加入适量的水使之溶解，全部转移至 $100~\mathrm{mL}$ 容量

瓶中，用水稀释至刻度，摇匀。用移液管移取 $25~\mathrm{mL}$ 该试验溶液，置于 $50~\mathrm{mL}$ 比色管中，加入 $1\mathrm{mL}$ 体积分数为 $95\%$ 乙醇溶液， $3\mathrm{mL}$ 硝酸溶液和 $2\mathrm{mL}$ 硝酸银溶液，用水稀释至刻度，轻轻摇匀。静置 $10\mathrm{min}$ 后，于黑背景下与标准比浊溶液比对，所产生的浊度不得深于标准比浊溶液。

标准比浊溶液是移取 $0.50\mathrm{mL},1.50\mathrm{mL},2.00\mathrm{mL},2.50\mathrm{mL},5.00\mathrm{mL}$ 氯化物标准溶液，与试样同时同样处理。

# A.10 白度的测定

# A.10.1 仪器和设备

A.10.1.1 白度计：带有标准白度板和工作白度板，分度值为0.2。

A.10.1.2 标准白度板。

# A.10.2 分析步骤

用定期标定过的标准白度板校正工作白度板。将白度仪调整至工作状态，将试样均匀地置于粉末皿中，使试样面超过粉末皿约 $2\mathrm{cm}$ 。用光洁的玻璃板覆盖在试样的表面上，压紧试样，并稍加旋转，移去玻璃板。沿水平方向观察试样表面，应无凹凸不平、疵点和斑痕异常情况。

将试样皿置于仪器台上，测定白度值，读准至0.1；将试样皿在仪器台上旋转 $90^{\circ}$ ，测定白度值，读准至0.1；再旋转 $90^{\circ}$ ，测定白度值，读准至0.1。三次读数结果极差不得大于0.5。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于1。

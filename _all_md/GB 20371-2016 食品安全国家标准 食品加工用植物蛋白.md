# 前言

本标准代替GB/T20371—2006《食品工业用大豆蛋白》。

本标准与GB/T20371—2006相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品加工用植物蛋白”；  
——修改了范围；  
——增加了术语和定义；  
——修改了理化指标；  
——修改了卫生要求；  
——增加了真菌毒素限量；  
——修改了附录。

# 食品安全国家标准

# 食品加工用植物蛋白

# 1 范围

本标准适用于食品加工用途的植物蛋白产品。

本标准不适用于棉籽蛋白和菜籽蛋白。

# 2 术语和定义

# 2.1 植物蛋白

以植物为原料，去除或部分去除植物原料中的非蛋白成分（如水分、脂肪、碳水化合物等），蛋白质含量不低于 $40\%$ 的产品。其主要产品有豆类（如大豆、豌豆、蚕豆）蛋白、谷类（如小麦、玉米、大米、燕麦）蛋白、坚果及籽类（如花生）蛋白、薯类（如马铃薯）蛋白及其他植物类蛋白。

# 2.2 粗提蛋白

通过初级提取，部分去除植物原料中的非蛋白成分(如水分、脂肪、碳水化合物等)而制得的产品。

# 2.3 浓缩蛋白

通过提取、浓缩、分离等工艺，去除或部分去除植物原料中的非蛋白成分（如水分、脂肪、碳水化合物等）而制得的产品。包括通过提取、加热凝固等工艺制得的马铃薯凝固蛋白。

# 2.4 分离蛋白

通过提取、浓缩、分离、精制等工艺，去除或部分去除植物原料中的非蛋白成分（如水分、脂肪、碳水化合物等）而制得的产品。

# 2.5 植物水解蛋白

植物蛋白经酶适度水解制得的以蛋白质为主要成分的产品。

# 2.6 组织蛋白

以植物蛋白为原料，经挤压或纺丝工艺加工制成的、具有特定组织结构的产品。

原料应符合相应的产品标准和有关规定。

# 3.4 污染物限量和真菌毒素限量

3.4.1 污染物限量应符合GB2762中相应类别产品的规定。其中大豆蛋白应符合GB2762中豆类制品的规定，花生蛋白应符合GB2762中花生的规定，小麦蛋白应符合GB2762中面筋的规定，玉米蛋白、燕麦蛋白应符合GB2762中谷物制品的规定，马铃薯蛋白、豌豆蛋白、蚕豆蛋白应符合GB2762中蔬菜制品的规定，大米蛋白应符合GB2762中大米的规定。  
3.4.2 真菌毒素限量应符合GB2761中相应类别产品的规定。其中花生蛋白应符合GB2761中花生的规定，玉米蛋白应符合GB2761中谷物制品的规定，大米蛋白应符合GB2761中大米的规定。

# 4 其他

4.1 产品名称应标注具体植物来源，如：大豆蛋白、小麦蛋白、玉米蛋白、豌豆蛋白、马铃薯蛋白等。  
4.2 产品名称可标注具体的分类，以大豆、花生为例，如大豆粗提蛋白、大豆浓缩蛋白、大豆分离蛋白、大豆组织蛋白、花生粗提蛋白、花生浓缩蛋白、花生分离蛋白、花生组织蛋白等。  
4.3 产品名称中可包含描述产品成型后物理形态的词语，如：颗粒、碎片、粉等。  
4.4 植物水解蛋白名称应依据具体植物来源进行标注，如：大豆水解蛋白、小麦水解蛋白、玉米水解蛋白、豌豆水解蛋白、马铃薯水解蛋白等。  
4.5 大豆蛋白产品应按照下列方式标识脲酶活性和安全性文字说明：

脲酶活性为阴性；

脲酶活性为非阴性（需加热灭酶处理后方可食用的产品）。

# 附录A

# 大豆蛋白中脲酶（尿素酶）活性的检验方法

# A.1 仪器设备

A.1.1 粉碎机：粉碎时应不生强热。  
A.1.2 样品筛：孔径 $200\mu \mathrm{m}$   
A.1.3 分析天平：感量 $0.1\mathrm{mg}$   
A.1.4 恒温水浴：可控温 $30^{\circ}\mathrm{C}\pm 0.5^{\circ}\mathrm{C}$   
A.1.5 计时器。  
A.1.6 酸度计：精度0.02，附有磁力搅拌器和滴定装置。  
A.1.7 实验室常用玻璃仪器。

# A.2 试剂和溶液

试剂为分析纯，水应符合GB/T6682的规定。

# A.2.1 尿素缓冲溶液 $(\mathbf{pH}7.0\pm 0.1)$

称取 $8.95\mathrm{g}$ 磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4\cdot 12\mathrm{H}_2\mathrm{O})$ 、 $3.40\mathrm{g}$ 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$ 溶于水并稀释至 $1000~\mathrm{mL}$ ，再将 $30\mathrm{g}$ 尿素溶在此缓冲溶液中，有效期1个月。

# A.2.2 盐酸溶液 $[c(\mathrm{HCl}) = 0.1\mathrm{mol} / \mathrm{L}]$

移取 $8.3\mathrm{mL}$ 盐酸，用水稀释至 $1000~\mathrm{mL}$

# A.2.3 氢氧化钠标准溶液 $[c(\mathrm{NaOH}) = 0.1\mathrm{mol} / \mathrm{L}]$

称取 $4\mathrm{g}$ 氢氧化钠溶于水并稀释至 $1000~\mathrm{mL}$ ，按GB/T601规定的方法配制和标定。

# A.2.4 甲基红、溴甲酚绿混合乙醇溶液

称取 $0.1\mathrm{g}$ 甲基红，溶于 $95\%$ 乙醇并稀释至 $100~\mathrm{mL}$ ，再称取 $0.5\mathrm{g}$ 溴甲酚绿，溶于 $95\%$ 乙醇并稀释至 $100~\mathrm{mL}$ ，两种溶液等体积混合，储存于棕色瓶中。

# A.3 试样的制备

用粉碎机(A.1.1)将具有代表性的样品粉碎，使之全部通过样品筛(A.1.2)。对特殊样品(水分或挥发物含量较高而无法粉碎的样品)应先在实验室温度下进行预干燥，再进行粉碎，当计算结果时应将干燥失重计算在内。

# A.4 测定步骤

称取约 $0.2\mathrm{g}$ 制备好的试样(A.3)(精确至 $0.1\mathrm{mg}$ )于玻璃试管中(如活性很高可称 $0.05\mathrm{g}$ 试样)，加入 $10\mathrm{mL}$ 尿素缓冲溶液(A.2.1)，立即盖好试管盖剧烈振摇后，将试管马上置于 $30^{\circ}\mathrm{C} \pm 0.5^{\circ}\mathrm{C}$ 恒温水浴

(A.1.4)中, 计时保持 $30 \mathrm{~min} \pm 10 \mathrm{~s}$ 。要求每个试样加入尿素缓冲液的时间间隔保持一致。停止反应时再以相同的时间间隔加入 $10 \mathrm{~mL}$ 盐酸溶液 (A.2.2), 振摇后迅速冷却到 $20^{\circ} \mathrm{C}$ 。将试管内容物全部转入小烧杯中, 用 $20 \mathrm{~mL}$ 水冲洗试管数次, 以氢氧化钠标准溶液 (A.2.3) 用酸度计 (A.1.6) 滴定至 $\mathrm{pH} 4.70$ 。如果选择用指示剂, 则将试管内容物全部转入 $250 \mathrm{~mL}$ 锥形瓶中, 加入 8 滴 $\sim 10$ 滴混合指示剂 (A.2.4), 以氢氧化钠标准溶液 (A.2.3) 滴定至溶液呈蓝绿色。

另取试管做空白试验，称取约 $0.2\mathrm{g}$ 制备好的试样(A.3)(精确至 $0.1\mathrm{mg}$ )于玻璃试管中(如活性很高可称 $0.05\mathrm{g}$ 试样)，加入 $10~\mathrm{mL}$ 盐酸溶液(A.2.2)，振摇后加入 $10~\mathrm{mL}$ 尿素缓冲溶液(A.2.1)，立即盖好试管盖剧烈振摇，将试管马上置于 $30^{\circ}\mathrm{C}\pm 0.5^{\circ}\mathrm{C}$ 恒温水浴(A.1.4)中，计时保持 $30\mathrm{min}\pm 10\mathrm{s}$ 。停止反应时将试管迅速冷却到 $20^{\circ}\mathrm{C}$ 。将试管内容物全部转入小烧杯中，用 $20~\mathrm{mL}$ 水冲洗试管数次，以氢氧化钠标准溶液(A.2.3)用酸度计(A.1.6)滴定至 $\mathrm{pH}4.70$ 。如果选择用指示剂，则将试管内容物全部转入 $250~\mathrm{mL}$ 锥形瓶中，加入8滴～10滴混合指示剂(A.2.4)，以氢氧化钠标准溶液(A.2.3)滴定至溶液呈蓝绿色。

# A.5 结果的计算

A.5.1 大豆制品中尿素酶活性 $X$ ，按式(A.1)计算。若试样经粉碎前的预干燥处理后，则按式(A.2)计算：

$$
X = \frac {1 4 \times c \times \left(V _ {0} - V\right)}{3 0 \times m} \quad \dots \tag {A.1}
$$

$$
X = \frac {1 4 \times c \times \left(V _ {0} - V\right) \times (1 - w)}{3 0 \times m} \tag {A.2}
$$

式中：

$X$ ——试样的尿素酶活性，单位为活性单位每克 $(\mathrm{U} / \mathrm{g})$ ；

14——氮的摩尔质量， $M(\mathrm{N}_2) = 14\mathrm{g / mol}$

$c$ ——氢氧化钠标准滴定溶液浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{0}$ ——空白消耗氢氧化钠标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$

$V$ ——试样消耗氢氧化钠标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

30——反应时间，单位为分钟（min）；

$m$ ——试样质量，单位为克(g)；

$\varpi$ ——预干燥时试样失重的质量分数， $\%$ 。

计算结果保留到小数点后两位。

A.5.2 精密度：同一分析人员用相同方法，同时或连续两次测定活性 $\leqslant 0.2$ 时结果之差不超过平均值的 $20\%$ ，活性 $>0.2$ 时结果之差不超过平均值的 $10\%$ ，结果以算术平均值表示。

# 前言

本标准代替GB/T5009.17—2003《食品中总汞及有机汞的测定》。

本标准与GB/T5009.17—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中总汞及有机汞的测定”；  
——取消了总汞测定的二硫腙比色法, 有机汞测定的气相色谱法和冷原子吸收法;  
——增加了甲基汞测定的液相色谱-原子荧光光谱法(LC-AFS)。

# 1 范围

# 第一篇 食品中总汞的测定

# 第一法 原子荧光光谱分析法

# 2 原理

试样经酸加热消解后，在酸性介质中，试样中汞被硼氢化钾或硼氢化钠还原成原子态汞，由载气（氩气)带入原子化器中，在汞空心阴极灯照射下，基态汞原子被激发至高能态，在由高能态回到基态时，发射出特征波长的荧光，其荧光强度与汞含量成正比，与标准系列溶液比较定量。

# 3 试剂和材料

# 3.2 试剂配制

3.2.1 硝酸溶液 $(1 + 9)$ ：量取 $50~\mathrm{mL}$ 硝酸，缓缓加入 $450~\mathrm{mL}$ 水中。  
3.2.2 硝酸溶液 $(5 + 95)$ ：量取 $5\mathrm{mL}$ 硝酸，缓缓加入 $95~\mathrm{mL}$ 水中。  
3.2.3 氢氧化钾溶液 $(5\mathrm{g} / \mathrm{L})$ ：称取 $5.0\mathrm{g}$ 氢氧化钾，纯水溶解并定容至 $1000~\mathrm{mL}$ ，混匀。  
3.2.4 硼氢化钾溶液 $(5\mathrm{g} / \mathrm{L})$ ：称取 $5.0\mathrm{g}$ 硼氢化钾，用 $5\mathrm{g} / \mathrm{L}$ 的氢氧化钾溶液溶解并定容至 $1000~\mathrm{mL}$ 混匀。现用现配。  
3.2.5 重铬酸钾的硝酸溶液 $(0.5\mathrm{g} / \mathrm{L})$ ：称取 $0.05\mathrm{g}$ 重铬酸钾溶于 $100~\mathrm{mL}$ 硝酸溶液 $(5 + 95)$ 中。  
3.2.6 硝酸-高氯酸混合溶液 $(5 + 1)$ ：量取 $500~\mathrm{mL}$ 硝酸， $100~\mathrm{mL}$ 高氯酸，混匀。

# 3.3 标准品

氯化汞 $(\mathrm{HgCl_2})$ ：纯度 $\geq 99\%$

# 3.4 标准溶液配制

3.4.1 汞标准储备液 $(1.00\mathrm{mg / mL})$ ：准确称取 $0.1354\mathrm{g}$ 经干燥过的氯化汞，用重铬酸钾的硝酸溶液 $(0.5\mathrm{g / L})$ 溶解并转移至 $100~\mathrm{mL}$ 容量瓶中，稀释至刻度，混匀。此溶液浓度为 $1.00\mathrm{mg / mL}$ 。于 $4^{\circ}C$ 冰箱中避光保存，可保存2年。或购买经国家认证并授予标准物质证书的标准溶液物质。  
3.4.2 汞标准中间液 $(10\mu \mathrm{g / mL})$ ：吸取 $1.00~\mathrm{mL}$ 汞标准储备液 $(1.00\mathrm{mg / mL})$ 于 $100~\mathrm{mL}$ 容量瓶中，用重铬酸钾的硝酸溶液 $(0.5\mathrm{g / L})$ 稀释至刻度，混匀，此溶液浓度为 $10\mu \mathrm{g / mL}$ 。于 $4^{\circ}C$ 冰箱中避光保存，可保存2年。  
3.4.3 汞标准使用液 $(50\mathrm{ng / mL})$ ：吸取 $0.50~\mathrm{mL}$ 汞标准中间液 $(10\mu \mathrm{g} / \mathrm{mL})$ 于 $100~\mathrm{mL}$ 容量瓶中，用 $0.5\mathrm{g / L}$ 重铬酸钾的硝酸溶液稀释至刻度，混匀，此溶液浓度为 $50~\mathrm{ng / mL}$ ，现用现配。

# 5 分析步骤

# 5.1 试样预处理

5.1.1 在采样和制备过程中，应注意不使试样污染。  
5.1.2 粮食、豆类等样品去杂物后粉碎均匀，装入洁净聚乙烯瓶中，密封保存备用。  
5.1.3 蔬菜、水果、鱼类、肉类及蛋类等新鲜样品，洗净晾干，取可食部分匀浆，装入洁净聚乙烯瓶中，密封，于 $4^{\circ}\mathrm{C}$ 冰箱冷藏备用。

# 5.2 试样消解

# 5.2.1 压力罐消解法

称取固体试样 $0.2\mathrm{g}\sim 1.0\mathrm{g}$ （精确到 $0.001\mathrm{g})$ ，新鲜样品 $0.5\mathrm{g}\sim 2.0\mathrm{g}$ 或液体试样吸取 $1\mathrm{mL}\sim 5\mathrm{mL}$ 称量(精确到 $0.001\mathrm{g})$ ，置于消解内罐中，加入 $5\mathrm{mL}$ 硝酸浸泡过夜。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱， $140^{\circ}C\sim 160^{\circ}C$ 保持 $4\mathrm{h}\sim 5\mathrm{h}$ ，在箱内自然冷却至室温，然后缓慢旋松不锈钢外套，将消解内罐取出，用少量水冲洗内盖，放在控温电热板上或超声水浴箱中，于 $80^{\circ}C$ 或超声脱气 $2\mathrm{min}\sim 5\mathrm{min}$ 赶去棕色气体。取出消解内罐，将消化液转移至 $25~\mathrm{mL}$ 容量瓶中，用少量水分3次洗涤内罐，洗涤液合并于容量瓶中并定容至刻度，混匀备用；同时作空白试验。

# 5.2.2 微波消解法

称取固体试样 $0.2\mathrm{g}\sim 0.5\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )、新鲜样品 $0.2\mathrm{g}\sim 0.8\mathrm{g}$ 或液体试样 $1\mathrm{mL}\sim 3\mathrm{mL}$ 于消解罐中，加入 $5\mathrm{mL}\sim 8\mathrm{mL}$ 硝酸，加盖放置过夜，旋紧罐盖，按照微波消解仪的标准操作步骤进行消解（消解参考条件见附录A表A.1）。冷却后取出，缓慢打开罐盖排气，用少量水冲洗内盖，将消解罐放在控温电热板上或超声水浴箱中，于 $80^{\circ}C$ 加热或超声脱气 $2\mathrm{min}\sim 5\mathrm{min}$ ，赶去棕色气体，取出消解内罐，将消化液转移至 $25~\mathrm{mL}$ 塑料容量瓶中，用少量水分3次洗涤内罐，洗涤液合并于容量瓶中并定容至刻度，混匀备用；同时作空白试验。

# 5.2.3 回流消解法

# 5.2.3.1 粮食

称取 $1.0\mathrm{g}\sim 4.0\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )试样，置于消化装置锥形瓶中，加玻璃珠数粒，加 $45~\mathrm{mL}$ 硝酸、 $10~\mathrm{mL}$ 硫酸，转动锥形瓶防止局部炭化。装上冷凝管后，小火加热，待开始发泡即停止加热，发泡停止后，加热回流 $2\mathrm{h}$ 。如加热过程中溶液变棕色，再加 $5\mathrm{mL}$ 硝酸，继续回流 $2\mathrm{h}$ ，消解到样品完全溶解，一般呈淡黄色或无色，放冷后从冷凝管上端小心加 $20~\mathrm{mL}$ 水，继续加热回流 $10\mathrm{min}$ 放冷，用适量水冲洗冷凝管，冲洗液并入消化液中，将消化液经玻璃棉过滤于 $100~\mathrm{mL}$ 容量瓶内，用少量水洗涤锥形瓶、滤器，洗涤液并入容量瓶内，加水至刻度，混匀。同时做空白试验。

# 5.2.3.2 植物油及动物油脂

称取 $1.0\mathrm{g}\sim 3.0\mathrm{g}$ （精确到 $0.001\mathrm{g})$ 试样，置于消化装置锥形瓶中，加玻璃珠数粒，加入 $7\mathrm{mL}$ 硫酸，小心混匀至溶液颜色变为棕色，然后加 $40~\mathrm{mL}$ 硝酸。以下按5.2.3.1“装上冷凝管后，小火加热……同时做空白试验”步骤操作。

# 5.2.3.3 薯类、豆制品

# 6 分析结果的表述

试样中汞含量按式(1)计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \quad \dots \dots \dots \dots \dots \dots \dots \dots \dots \tag {1}
$$

式中：

$X$ ——试样中汞的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$c$ ——测定样液中汞含量，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{0}$ ——空白液中汞含量，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样消化液定容总体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$m$ ——试样质量，单位为克或毫升（g 或 mL）。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 第二法 冷原子吸收光谱法

# 9 原理

汞蒸气对波长 $253.7 \mathrm{~nm}$ 的共振线具有强烈的吸收作用。试样经过酸消解或催化酸消解使汞转为离子状态，在强酸性介质中以氯化亚锡还原成元素汞，载气将元素汞吹入汞测定仪，进行冷原子吸收测定，在一定浓度范围其吸收值与汞含量成正比，外标法定量。

分别吸取汞标准使用液 $(50\mathrm{ng / mL})0.00\mathrm{mL},0.20\mathrm{mL},0.50\mathrm{mL},1.00\mathrm{mL},1.50\mathrm{mL},2.00\mathrm{mL}$ 、 $2.50~\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，用硝酸溶液 $(1 + 9)$ 稀释至刻度，混匀。各自相当于汞浓度为 $0.00\mathrm{ng / mL}$ 、 $0.20\mathrm{ng / mL},0.50\mathrm{ng / mL},1.00\mathrm{ng / mL},1.50\mathrm{ng / mL},2.00\mathrm{ng / mL}$ 和 $2.50\mathrm{ng / mL}$ 。将标准系列溶液分别置于测汞仪的汞蒸气发生器中，连接抽气装置，沿壁迅速加入 $3.0~\mathrm{mL}$ 还原剂氯化亚锡 $(100~\mathrm{g / L})$ ，迅速盖紧瓶塞，随后有气泡产生，立即通过流速为 $1.0~\mathrm{L / min}$ 的氮气或经活性炭处理的空气，使汞蒸气经过氯化钙干燥管进入测汞仪中，从仪器读数显示的最高点测得其吸收值。然后，打开吸收瓶上的三通阀将产生的剩余汞蒸气吸收于高锰酸钾溶液 $(50~\mathrm{g / L})$ 中，待测汞仪上的读数达到零点时进行下一次测定。同时做空白试验。求得吸光度值与汞质量关系的一元线性回归方程。

# 12.5 试样溶液的测定

分别吸取样液和试剂空白液各 $5.0\mathrm{mL}$ 置于测汞仪的汞蒸气发生器的还原瓶中，以下按照12.4“连接抽气装置……同时做空白试验”进行操作。将所测得吸光度值，代入标准系列溶液的一元线性回归方程中求得试样溶液中汞含量。

# 13 分析结果的表述

试样中汞含量按式(2)计算：

$$
X = \frac {\left(m _ {1} - m _ {2}\right) \times V _ {1} \times 1 0 0 0}{m _ {1} \times V _ {2} \times 1 0 0 0 \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中汞含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$m_{1}$ ——测定样液中汞质量，单位为纳克 $(\mathrm{ng})$ ；

$m_{2}$ ——空白液中汞质量，单位为纳克 $(\mathrm{ng})$ ；

$V_{1}$ ——试样消化液定容总体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$m$ ——试样质量，单位为克或毫升（g 或 mL）；

$V_{2}$ ——测定样液体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留两位有效数字。

# 液相色谱-原子荧光光谱联用方法

# 16 原理

食品中甲基汞经超声波辅助 $5 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液提取后，使用 $\mathrm{C}_{18}$ 反相色谱柱分离，色谱流出液进入在线紫外消解系统，在紫外光照射下与强氧化剂过硫酸钾反应，甲基汞转变为无机汞。酸性环境下，无机汞与硼氢化钾在线反应生成汞蒸气，由原子荧光光谱仪测定。由保留时间定性，外标法峰面积定量。

# 20 分析结果的表述

$$
X = \frac {f \times \left(c - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {3}
$$

式中：

$X$ ——试样中甲基汞的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$f$ —稀释因子；

$c$ ——经标准曲线得到的测定液中甲基汞的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{0}$ ——经标准曲线得到的空白溶液中甲基汞的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

V ——加入提取试剂的体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$m$ ——试样称样量，单位为克 $(\mathrm{g})$ 。

计算结果保留两位有效数字。

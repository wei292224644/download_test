# 蜂产品中4种抗病毒类药物残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了蜂产品中4种抗病毒类药物残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于蜂蜜和蜂王浆中吗啉胍、金刚烷胺、金刚乙胺和美金刚残留量的测定。

# 4 原理

蜂蜜和蜂王浆中的残留药物用乙腈溶液提取，苯硼酸固相萃取柱净化，液相色谱-串联质谱仪测定，内标法定量。

# 5 试剂和材料

以下所用试剂，除特殊注明外均为分析纯试剂，水为符合GB/T 5682中规定的一级水。

# 5.1 试剂

5.1.1 甲酸 $(\mathrm{HCO})(\mathrm{OH})_2$ 色谱纯

5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.3 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯

5.1.4 氨水 $(\mathrm{NH_4OH})$

5.1.5 乙酸铵 $\mathrm{(CH_3COONH_2)}$

# 5.2 溶液配制

5.2.1 $0.1\%$ 甲酸溶液：取甲酸 $100~\mu \mathrm{L}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀

5.2.2 $0.5\%$ 甲酸溶液：取甲酸 $500~\mu \mathrm{L}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。

5.2.3 $1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。

5.2.4 $20\%$ 甲醇溶液：取甲醇 $20~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。

5.2.5 $80\%$ 乙腈溶液：取乙腈 $80~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。

5.2.6 乙酸铵溶液 $(0.2\mathrm{mol} / \mathrm{L},\mathrm{pH}8.5)$ ：取乙酸铵 $1.54\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ ，氨水调节 $\mathrm{pH}$ 至8.5。

5.2.7 $0.1\%$ 甲酸+甲醇溶液：取 $0.1\%$ 甲酸溶液 $10~\mathrm{mL}$ ，用甲醇稀释至 $100~\mathrm{mL}$ ，混匀。

5.2.8 $0.5\%$ 甲酸+甲醇溶液：取甲醇 $5\mathrm{mL}$ ，用 $0.5\%$ 甲酸溶液稀释至 $100~\mathrm{mL}$ ，混匀。

5.2.9 $1\%$ 甲酸 $+$ 甲醇溶液：取 $1\%$ 甲酸溶液 $40~\mathrm{mL}$ ，用甲醇稀释至 $100~\mathrm{mL}$ ，混匀。

# 5.3 标准品及其纯度

金刚烷胺、金刚乙胺、美金刚、吗啉胍、金刚烷胺- $\mathrm{d}_{15}$ 、金刚乙胺- $\mathrm{d}_4$ 、美金刚- $\mathrm{d}_6$ 和吗啉胍- $\mathrm{d}_8$ ，含量均 $\geqslant$

$99\%$ ，具体信息见附录A。

# 5.4 标准溶液配制

5.4.1 抗病毒类药物及其内标物标准储备液：分别称取吗啉胍及其同位素内标标准品约 $10\mathrm{mg}$ ，用 $0.1\%$ 甲酸-甲醇溶液溶解并定容至 $10\mathrm{mL}$ ，配制成 $1000\mu \mathrm{g} / \mathrm{mL}$ 的标准储备液。分别称取金刚烷胺、金刚乙胺和美金刚及其同位素内标标准品约 $10\mathrm{mg}$ ，用甲醇溶解后定容至 $10\mathrm{mL}$ ，配制成 $1000\mu \mathrm{g} / \mathrm{mL}$ 的标准储备液。于 $-18^{\circ}\mathrm{C}$ 以下避光保存，有效期3个月。  
5.4.2 抗病毒类药物混合标准中间液：依次吸取4种抗病毒类药物标准储备液 $100\mu \mathrm{L}$ ，于 $10~\mathrm{mL}$ 容量瓶中，用甲醇溶液定容至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的混合标准工作液。于一 $18^{\circ}C$ 以下避光保存，有效期1个月。  
5.4.3 抗病毒类药物混合标准工作液：精密量取抗病毒类药物混合标准中间液适量，用 $0.5\%$ 甲酸-甲醇溶液将其稀释成 $0.1\mu \mathrm{g} / \mathrm{L}, 0.5\mu \mathrm{g} / \mathrm{L}, 1.0\mu \mathrm{g} / \mathrm{L}, 10.0\mu \mathrm{g} / \mathrm{L}, 50.0\mu \mathrm{g} / \mathrm{L}, 100.0\mu \mathrm{g} / \mathrm{L}, 500.0\mu \mathrm{g} / \mathrm{L}$ 的系列浓度标准工作液。现用现配。  
5.4.4 抗病毒类药物同位素内标混合标准中间液：依次吸取4种抗病毒类药物同位素内标标准储备液 $100\mu \mathrm{L}$ ，于 $10~\mathrm{mL}$ 容量瓶中，用甲醇溶液定容至刻度，配制成吗啉胍- $\mathbf{d}_8$ 、金刚烷胺- $\mathbf{d}_{15}$ 、金刚乙胺- $\mathbf{d}_4$ 和美金刚- $\mathbf{d}_6$ 浓度为 $10~\mu \mathrm{g / mL}$ 的抗病毒类药物同位素内标混合标准工作液。于一 $18^{\circ}C$ 以下避光保存，有效期1个月。  
5.4.5 抗病毒类药物同位素内标混合标准工作液：吸取抗病毒类药物同位素内标混合标准中间液 $100\mu \mathrm{L}$ ，于 $10\mathrm{mL}$ 容量瓶中，用 $0.5\%$ 甲酸-甲醇溶液定容至刻度，配制成吗啉胍- $\mathrm{d}_{8}$ 、金刚烷胺- $\mathrm{d}_{15}$ 、金刚乙胺- $\mathrm{d}_{4}$ 和美金刚- $\mathrm{d}_{6}$ 浓度为 $100\mathrm{ng / mL}$ 的抗病毒类药物同位素内标混合标准工作液。现用现配。

5.5.2 水系滤膜： $0.22\mu \mathrm{m}$   
5.5.3 增强型脂质去除净化包(EMR-Lipid)或相当者。

6.2 涡旋混合器。  
6.3 高速冷冻离心机：转速 $8000\mathrm{r / min}$ 或以上，可控温度 $4^{\circ}C$ 。  
6.4 水浴振荡器。  
6.5 氮吹仪。  
6.6 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
6.7 移液器： $10\mu \mathrm{L}, 20\mu \mathrm{L}, 200\mu \mathrm{L}, 1000\mu \mathrm{L}, 5000\mu \mathrm{L}$ 。  
6.8 pH计。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试蜂产品,将其搅拌均匀。对有结晶的蜂蜜样品,在密闭情况下,置于不超过 $60^{\circ} \mathrm{C}$ 的水浴中解晶,待样品全部融化后搅匀,冷却至室温。制备好的试样置于样品瓶中,密封,并做上标记。

a）取均质后的供试样品，作为供试试样；  
b) 取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

蜂蜜样品于室温下避光保存；蜂王浆样品于 $-18^{\circ}C$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取蜂蜜试料 $(2\pm 0.05)\mathrm{g}$ 于 $50~\mathrm{mL}$ 具塞塑料离心管中，依次加入同位素内标混合标准工作液 $100~\mu \mathrm{L}$ 和 $80\%$ 乙腈溶液 $10~\mathrm{mL}$ ，振荡 $5\mathrm{min}$ ，于 $4^{\circ}C$ 下 $8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取全部上清液，加水至 $30~\mathrm{mL}$ ，用氨水调 $\mathrm{pH}$ 至 $8.5\pm 0.02$ ，备用。

称取蜂王浆试料 $(1\pm 0.05)\mathrm{g}$ 于 $50~\mathrm{mL}$ 具塞塑料离心管中，依次加入同位素内标混合标准工作液 $100\mu \mathrm{L}$ 和乙腈溶液 $6\mathrm{mL}$ ，振荡 $5\mathrm{min}$ ，于 $4^{\circ}C$ 下 $8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取全部上清液至预先加入 $3.6\mathrm{mL}$ 水活化的EMR净化包，振荡 $3\mathrm{min}$ ，于 $4^{\circ}C$ 下 $8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取全部上清液，加水至 $20~\mathrm{mL}$ ，用氨水调 $\mathsf{pH}$ 至 $8.5\pm 0.02$ ，备用。

# 8.2 净化

固相萃取柱预先依次用乙腈 $10\mathrm{mL}$ 、水 $10\mathrm{mL}$ 和 $0.2\mathrm{mol} / \mathrm{L}$ 乙酸铵 $(\mathrm{pH}8.5)10\mathrm{mL}$ 活化，备用液以 $\leqslant 1$ 滴/s的流速通过固相萃取柱，待备用液完全流出后，依次用水 $10\mathrm{mL}$ 和 $20\%$ 甲醇溶液 $10\mathrm{mL}$ 洗柱，弃去全部流出液，负压抽干，用 $8\mathrm{mL}$ 甲醇 $1\%$ 甲酸溶液洗脱，洗脱液收集于 $10\mathrm{mL}$ 离心管中，于 $40^{\circ}\mathrm{C}$ 下氮吹至近干， $1.0\mathrm{mL}0.5\%$ 甲酸-甲醇溶液复溶，经 $0.22\mu \mathrm{m}$ 滤膜过滤待测。蜂王浆的洗脱液上机前用 $0.5\%$ 甲酸水-甲醇(95+5)稀释10倍后测定。

# 8.3 基质匹配标准曲线的制备

精密量取混合标准工作液和同位素内标混合标准工作液适量，分别加入经提取和净化的空白试料残渣中， $45\%$ 水浴氮气吹干，加入 $0.5\%$ 甲酸水-甲醇溶液 $1.0\mathrm{mL}$ 涡旋溶解残余物，配制成浓度为 $0.1\mu \mathrm{g} / \mathrm{L}$ 、 $1.0\mu \mathrm{g} / \mathrm{L}, 10.0\mu \mathrm{g} / \mathrm{L}, 30.0\mu \mathrm{g} / \mathrm{L}, 100.0\mu \mathrm{g} / \mathrm{L}, 500.0\mu \mathrm{g} / \mathrm{L}$ 的蜂蜜和蜂王浆基质匹配系列混合标准溶液，内标溶液浓度为 $10\mu \mathrm{g} / \mathrm{L}$ ，蜂王浆的基质匹配标准溶液用 $0.5\%$ 甲酸水-甲醇（95+5）稀释10倍，微孔滤膜过滤，供液相色谱-串联质谱测定。以待测物的峰面积的平均值与内标的峰面积比作为纵坐标、标准溶液浓度为横坐标，绘制基质匹配标准曲线，求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在相同测试条件下，试样溶液中抗病毒类药物与其内标的保留时间之比与基质匹配标准溶液中抗病毒类药物与其内标的保留时间之比的偏差在 $1\%$ 之内；且检测到的相对离子丰度应与浓度相当的校正标准溶液相对离子丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.4.3.2 定量测定

分别取适量试样溶液和相应浓度的基质匹配标准工作液，作单点校准或多点校准，以色谱峰峰面积积分值定量。基质匹配标准工作液及试样液中药物的响应值均应在仪器检测的线性范围内，试样液进样过程中应穿插标准工作液，以便准确定量。对于试料中抗病毒类药物残留量超过仪器测定线性范围的，在提取时根据药物浓度相应增加内标工作液的添加量，使试样溶液稀释后抗病毒类药物的响应在仪器线性范围内，对应内标物浓度与标准曲线制备中的内标物浓度一致。4种抗病毒类药物及其同位素内标混合标准溶液的特征离子质量色谱图见附录B。

# 8.5 空白试验

取空白试样，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中抗病毒类药物的残留量 $X$ ，按标准曲线或公式(1)计算。

$$
X = \frac {C _ {s} \times C _ {i s} \times A _ {i} \times A ^ {\prime} {} _ {i s} \times V \times 1 0 0 0}{C ^ {\prime} {} _ {i s} \times A _ {s} \times A _ {i s} \times m \times 1 0 0 0}. \tag {1}
$$

式中：

$X$ ——试样中抗病毒类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；  
$C_{s}$ ——基质匹配标准溶液中抗病毒类药物浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；  
$C_{is}$ ——试样溶液中抗病毒类药物内标物浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A_{i}$ ——试样溶液中抗病毒类药物的峰面积；

$A_{is}^{\prime}$ ——标准溶液中抗病毒类药物内标物峰面积；

$V$ ——试样最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$C_{is}^{\prime}$ 基质匹配标准溶液中抗病毒类药物内标物浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$A_{s}$ 基质匹配标准溶液中抗病毒类药物峰面积；

$A_{is}$ ——试样溶液中抗病毒类药物内标物峰面积；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

注：计算结果以平行测定结果的算术平均值表示，含量不小于 $1\mu \mathrm{g} / \mathrm{kg}$ 的保留3位有效数字， $1\mu \mathrm{g} / \mathrm{kg}$ 以下保留至小数点后2位。

# 10 检测方法的灵敏度、正确度和精密度

# 10.1 灵敏度

本方法蜂蜜和蜂王浆中金刚烷胺、金刚乙胺、美金刚、吗啉胍检出限为 $0.3\mu \mathrm{g / kg}$ ，定量限为 $1\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法在蜂蜜中 $1\mu \mathrm{g / kg}\sim 50\mu \mathrm{g / kg}$ 添加浓度范围内，其回收率范围为 $70\% \sim 110\%$ ；在蜂王浆中 $1\mu \mathrm{g / kg}\sim 50\mu \mathrm{g / kg}$ 添加浓度范围内，其回收率范围为 $70\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

# 前言

本标准代替GB3862—2006《食品添加剂 天然薄荷脑》。

本标准与GB3862—2006相比，主要变化如下：

——“熔点 $41.5\,^{\circ}\mathrm{C} \sim 44.0\,^{\circ}\mathrm{C}$ ”改为“熔程 $41.0\,^{\circ}\mathrm{C} \sim 44.0\,^{\circ}\mathrm{C}$ ”；

——增加了“薄荷脑含量”指标；

——删除了“重金属含量”及“砷含量”要求。

# 食品安全国家标准

# 1 范围

本标准适用于以薄荷油为原料经冷冻法结晶分离制得的食品添加剂天然薄荷脑。

# 附录A

# 薄荷脑含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

试样制备：称取试样 $2\mathrm{g}$ 溶于 $1\mathrm{mL}$ 无水乙醇中，摇匀备用。

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行，应符合要求。

# B.2 操作条件

B.2.4 色谱炉温度：线性程序升温从 $100^{\circ}\mathrm{C}$ 至 $150^{\circ}\mathrm{C}$ ，速率 $2.5^{\circ}\mathrm{C} / \mathrm{min}$ ，然后在 $150^{\circ}\mathrm{C}$ 恒温 $10\mathrm{min}$ 。  
B.2.5 进样口温度： $230^{\circ}C$   
B.2.6 检测器温度： $250^{\circ}C$ 。  
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 柱前压： $103\mathrm{kPa}$   
B.2.10 进样量： $0.1\mu \mathrm{L}$   
B.2.11 分流比： $100:1$

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度均为分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 乳铁蛋白占总蛋白质量百分比的测定

# A.2.1 方法提要

采用高效液相色谱法，在选定的工作条件下，于紫外检测器 $280\mathrm{nm}$ 处通过色谱柱分离测定乳铁蛋白，面积归一化法定量。

# A.2.2 试剂和材料

A.2.2.1 水：符合 GB/T 6682 的一级水。

A.2.2.2 乙腈：色谱纯。

A.2.2.3 氯化钠溶液： $0.9\%$

A.2.2.4 $1\%$ 三氟乙酸溶液：吸取 $1\mathrm{mL}$ 三氟乙酸，加一级水定容至 $100~\mathrm{mL}$ ，过 $0.45\mu \mathrm{m}$ 滤膜。

A.2.2.5 乳铁蛋白对照品：纯度 $\geqslant 95\%$

A.2.2.6 $0.45\mu \mathrm{m}$ 微孔水相滤膜。

# A.2.3 仪器和设备

高效液相色谱仪：配备紫外检测器。

# A.2.4 参考色谱条件

A.2.4.1 色谱柱：Alltech prosphere C4 300A（ $150\mathrm{mm}\times 4.6\mathrm{mm}$ ），Alltech All-guardTM Guard（ $7.5\mathrm{mm}\times 4.6\mathrm{mm}$ ）或具有同等性能的色谱柱。

A.2.4.2 检测波长： $280~\mathrm{nm}$

A.2.4.3 流速： $1.0~\mathrm{mL / min}$

A.2.4.4 柱温： $35^{\circ}C$ 。

A.2.4.5 进样量： $50~\mu \mathrm{L}$

A.2.4.6 流动相梯度洗脱参数见表A.1。

# A.2.5 分析步骤

称取试样和乳铁蛋白对照品各 $0.1\mathrm{g}$ ，分别加入一级水（或氯化钠溶液） $10~\mathrm{mL}$ 使其溶解后，通过 $0.45\mu \mathrm{m}$ 滤膜，得到试样溶液和对照溶液。分别量取上述两种溶液 $50~\mu \mathrm{L}$ 注入高效液相色谱仪，在 $280~\mathrm{nm}$ 处进行测定，使试样溶液主峰的保留时间和对照溶液主峰的保留时间相一致（约为 $10\mathrm{min}$ ）。主峰作为乳铁蛋白峰，在主峰保留时间2倍范围内测定所有溶出的峰面积，主峰面积与所有峰面积的比值即为总蛋白中乳铁蛋白的含量。

# A.2.6 结果计算

乳铁蛋白占总蛋白含量的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {A _ {\mathrm {i}}}{A} \times 100 \% \tag{A.1}
$$

式中：

$A_{\mathrm{i}}$ ——主峰的峰面积；

$A$ ——所有溶出的峰面积。

试验结果以平行测定结果的算术平均值为准，保留3位有效数字。

# A.3 溶解度试验

# A3.1 分析步骤

称取 $0.8\mathrm{g}$ 试样，加入约 $40~\mathrm{mL}$ 的 $20^{\circ}\mathrm{C}$ 水中，充分搅拌 $2\mathrm{min}$ 以上，静止后观察。此时试样应完全溶解，溶液澄清透明，无肉眼可见杂质。

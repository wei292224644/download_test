# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 稀盐酸：量取盐酸 $234~\mathrm{mL}$ ，加水稀释至 $1000~\mathrm{mL}$ 。  
A.2.1.2 氯化钡试液：称取氯化钡的细粉 $5\mathrm{g}$ ，加水使溶解为 $100~\mathrm{mL}$ 。  
A.2.1.3 $10\%$ 磷钼酸。  
A.2.1.4 硫氰酸钾。  
A.2.1.5 硝酸钴。  
A.2.1.6 二氯甲烷。

# A.2.2 鉴别方法

A.2.2.1 称取 $0.05 \mathrm{~g}$ 试样, 加稀盐酸 $5 \mathrm{~mL}$ 和氯化钡试液 $1 \mathrm{~mL}$ , 振摇, 滤过; 在滤液中加入 $10 \%$ 磷钼酸溶液 $1 \mathrm{~mL}$ , 产生黄绿色沉淀。  
A.2.2.2 称取 $0.1\mathrm{g}$ 试样，置试管中，加入硫氰酸钾和硝酸钴各 $0.1\mathrm{g}$ ，混合后，加入二氯甲烷 $5\mathrm{mL}$ ，溶液呈蓝色。

# A.3 平均相对分子质量的测定

# A.3.1 试剂和材料

A.3.1.1 邻苯二甲酸酐。  
A.3.1.2 无水吡啶。  
A.3.1.3 氢氧化钾标准滴定溶液 $(0.5\mathrm{mol} / \mathrm{L})$ 。  
A.3.1.4 酚酞。

# A.3.2 分析步骤

称取聚乙二醇样品(精确至 $0.0001\mathrm{g}$ )，使其质量约为预期平均相对分子质量的1/160，置干燥的 $250~\mathrm{mL}$ 具塞锥形瓶中。固体样品加入无水吡啶 $25~\mathrm{mL}$ ，微温使其溶解后，准确加入邻苯二甲酸酐的吡啶溶液（取邻苯二甲酸酐 $49\mathrm{g}$ ，溶于无水吡啶 $300~\mathrm{mL}$ 中，小心摇动至溶解，放置过夜，备用） $25.0~\mathrm{mL}$ ；液体样品直接加入邻苯二甲酸酐的吡啶溶液 $25.0~\mathrm{mL}$ ；摇匀，置沸水浴中，加热 $30\mathrm{min} \sim 60\mathrm{min}$ ，取出冷却，准确加入氢氧化钾滴定液 $(0.5\mathrm{mol / L})50~\mathrm{mL}$ ，以酚酞的吡啶溶液 $(1\mathrm{g / 100~mL})$ 为指示剂，用氢氧化钾滴定液 $(0.5\mathrm{mol / L})$ 滴定到初次出现粉红色，持续 $15\mathrm{s}$ 不褪色，记录消耗的滴定液的体积，记作 $S$ 。用

同样方法以 $25.0\mathrm{mL}$ 的邻苯二甲酸酐的吡啶溶液做空白测定，记录消耗的滴定液的体积，记作 $B$ 。

# A.3.3 结果计算

样品中聚乙二醇的平均相对分子质量 $(M)$ 按式(A.1)计算。

$$
M = \frac {2 0 0 0 \times m}{c \times (B - S)} \quad \dots \tag {A.1}
$$

式中：

2000——常数，聚乙二醇和氢氧化钾的摩尔反应系数 $2 \times 1000$ （毫升和升的单位转换系数）；

$m$ —称量样品的质量，单位为克 $(\mathrm{g})$

$c$ ——氢氧化钾溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$B$ ——空白所消耗的氢氧化钾标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

S ——样品所消耗的氢氧化钾标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

# A.4 $\mathbf{pH}$ 的测定

称取本品 $1.0\mathrm{g}$ ，加水 $20~\mathrm{mL}$ 溶解后，摇匀。按GB/T6368规定方法测定。

# A.5.2 测试方法

用乌氏黏度计测定运动黏度。取符合要求的乌氏黏度计(校准并附有黏度计常数，样品流出时间不少于 $200\mathrm{~s}$ ，如测量黏度 $300\mathrm{~mm}^2/\mathrm{s} \sim 600\mathrm{~mm}^2/\mathrm{s}$ 的样品，选用黏度计毛细管内径应为 $2.00\mathrm{~mm} \pm 0.04\mathrm{~mm})$ 1支，将聚乙二醇样品（不得少于 $7\mathrm{~mL}$ ）沿洁净、干燥乌氏黏度计的管1内壁注入球A中，将黏度计垂直固定于恒温浴（本标准为 $100\mathrm{~^\circ C} \pm 0.3\mathrm{~^\circ C}$ ）中，并使液面高于球C，放置 $15\mathrm{~min}$ 后，将管口2、3各接一乳胶管，夹住管口3的胶管，自管口2处抽气，使样品的液面缓缓升至球C的中部（此过程中要排除气泡），先开放管口3，再开放管口1，使样品在管内自然下落，用秒表准确记录液面自测定线 $T_{1}$ 下降至测定线 $T_{2}$ 处的流出时间，记作 $t$ 。

# A.5.3 结果计算

聚乙二醇样品的运动黏度 $(\nu)$ 按式(A.2)计算。

$$
\nu = K \times t \quad \dots \tag {A.2}
$$

式中：

$K$ ——用已知黏度的标准液测得的黏度计常数，单位为二次方毫米每二次方秒 $(\mathrm{mm}^2/\mathrm{s}^2)$ ；

$t$ ——测得的流出时间，单位为秒(s)。

# A.7.3 参考色谱条件

A.7.3.3 温度：起始温度为 $35^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ ，以 $5^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升至 $180^{\circ}\mathrm{C}$ ，再以 $30^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升至 $230^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ （可根据具体情况调整）。进样口温度为 $150^{\circ}\mathrm{C}$ ，检测器温度为 $250^{\circ}\mathrm{C}$ 。顶空瓶平衡温度为 $70^{\circ}\mathrm{C}$ ，平衡时间为 $45\mathrm{min}$ 。

# A.7.4 分析步骤

# A.7.4.1 环氧乙烷标准贮备液的制备

量取环氧乙烷 $300\mu \mathrm{L}$ (相当于 $0.25\mathrm{g}$ 环氧乙烷), 置含 $50\mathrm{ml}$ 经过处理的聚乙二醇400(以 $60^{\circ}C$ 1 $1.5\mathrm{kPa}\sim 2.5\mathrm{kPa}$ 下旋转蒸发 $6\mathrm{h}$ , 除去挥发性成分)的 $100~\mathrm{mL}$ 量瓶中, 加入相同溶剂稀释至刻度, 摇匀, 作为环氧乙烷标准贮备液。称取 $1\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ) 冷的环氧乙烷标准贮备液, 置含 $40~\mathrm{mL}$ 经过处理的聚乙二醇400的 $50~\mathrm{mL}$ 量瓶中, 加相同溶剂稀释至刻度。准确称取 $10\mathrm{g}$ , 置含 $30~\mathrm{mL}$ 水的 $50~\mathrm{mL}$ 量瓶中, 加水稀释至刻度。量取 $10~\mathrm{mL}$ , 置 $50~\mathrm{mL}$ 量瓶中, 加水稀释至刻度, 摇匀, 作为环氧乙烷标准溶液(约 $2\mu \mathrm{g / mL}$ )。

# A.7.4.2 环氧乙烷标准贮备液的标定

取 $50\%$ 氯化镁的无水乙醇混悬液 $10~\mathrm{mL}$ ，加入 $20~\mathrm{mL}$ 乙醇盐酸滴定液 $(0.1\mathrm{mol} / \mathrm{L})$ 混匀，放置过夜。取 $5\mathrm{g}$ 环氧乙烷标准贮备液，置上述溶液中，放置 $30\mathrm{min}$ ，按照电位滴定法，用乙醇氢氧化钾滴定液 $(0.1\mathrm{mol} / \mathrm{L})$ 滴定，以电极电位突跃为滴定终点，并将滴定的结果用空白试验校正，每 $1\mathrm{mL}$ 乙醇氢氧化钾滴定液相当于 $4.404\mathrm{mg}$ 的环氧乙烷，计算，即得环氧乙烷标准贮备液的浓度。

# A.7.4.3 二氧六环标准溶液制备

称取二氧六环适量，用水制成每 $1\mathrm{mL}$ 中含 $0.1\mathrm{mg}$ 的溶液，作为二氧六环标准溶液 $(100~\mu \mathrm{g / mL})$ 。

# A.7.4.4 系统适应性试验

量取 $0.5\mathrm{mL}$ 环氧乙烷标准溶液置顶空瓶中，加入新鲜配制的 $0.001\%$ 乙醛溶液 $0.1\mathrm{mL}$ 及二氧六环标准溶液 $0.1\mathrm{mL}$ ，密封，摇匀，作为系统适用性试验溶液。色谱检查结果满足：乙醛峰和环氧乙烷峰之间的分离度不小于2.0，二氧六环峰高应为基线噪声的5倍以上。

# A.7.4.5 测定

测试方法：取本品 $1.0\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ，置顶空瓶中，准确加入超纯水 $1.0\mathrm{mL}$ ，密封，摇匀，作为供试品溶液。取本品 $1.0\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ，置顶空瓶中，准确加入 $0.5\mathrm{mL}$ 环氧乙烷标准溶液及 $0.5\mathrm{mL}$ 二氧六环标准溶液，密封，摇匀，作为对照品溶液。分别取供试品溶液及对照品溶液顶空进样，记录色谱图，按峰面积定量。重复进样至少3次，环氧乙烷峰面积的相对标准偏差不得超过 $15\%$ ，二氧六环峰面积的相对标准偏差不得超过 $10\%$ 。

# A.7.4.6 结果计算

聚乙二醇样品中环氧乙烷和二氧六环的含量 $(W_{i})$ 按式(A.3)计算。

$$
W _ {i} = \frac {m _ {i}}{\left(A _ {i s} / A _ {i} - 1\right) \times m} \times 100 \% \quad \dots \tag{A.3}
$$

式中：

$m_{i}$ ——所加入的已知含量的待测组分 $i$ 的质量，单位为克 $(\mathrm{g})$ ；

$A_{is}$ ——加入标准溶液后组分 $i$ 的色谱峰面积；

$A_{i}$ ——供试品中组分 $i$ 的色谱峰面积；

$m$ ——供试品的质量，单位为克 $(\mathrm{g})$ 。

A.8.1.3.1 色谱柱：填充不锈钢柱 $3\mathrm{mm}\times 1.5\mathrm{m}$ 色谱柱由承载 $12\%$ 山梨醇的未经处理的硅藻土填充，或其他等效色谱柱。  
A.8.1.3.2 载气：氮气或其他适合的惰性气体。

A.8.1.3.3 温度：柱温保持在 $160^{\circ}\mathrm{C}$ ，进样口温度保持在 $260^{\circ}\mathrm{C}$ ，检测器温度保持在 $280^{\circ}\mathrm{C}$ 。

# A.8.1.4 分析步骤

A.8.1.4.1 标准溶液制备: 取乙二醇与二甘醇各 $50 \mathrm{mg}$ , 置 $100 \mathrm{~mL}$ 容量瓶中, 加水稀释至刻度, 摇匀, 作为对照溶液。  
A.8.1.4.2 供试液制备：取供试品 $4.0\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )，置 $10\mathrm{mL}$ 容量瓶中，加水稀释至刻度，摇匀，作为供试品溶液。

# A.8.1.4.3 测定

将 $1.0\mu \mathrm{L}$ 标准液进样，记录其色谱图，调整操作条件使灵敏度最佳化。测定第一色谱峰(乙二醇)和第二色谱峰(二甘醇)的峰高。将 $1.0\mu \mathrm{L}$ 供试液进样，在同样操作条件下记录其色谱图，测定第一色谱峰和第二色谱峰的峰高。

# A.8.1.4.4 结果计算

聚乙二醇样品中乙二醇和二甘醇的百分含量 $(W_{i})$ 按式(A.4)计算。

$$
W _ {i} = \frac {h _ {i} \times C _ {i} \times V}{H _ {i} \times m} \times 100 \% \quad \dots \tag{A.4}
$$

式中：

$h_i$ ——供试液组分 $i$ 的峰高；

$C_i$ ——标准溶液中组分 $i$ 的浓度，单位为克每毫升 $(\mathrm{g / mL})$ ；

$V$ ——供试液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$H_{i}$ ——标准液组分 $i$ 的峰高；

$m$ ——供试品的质量，单位为克 $(\mathrm{g})$ 。

# A.8.2 平均相对分子质量高于450低于1000的聚乙二醇样品中的乙二醇和二甘醇总量的测定

# A.8.2.1 试剂和材料

A.8.2.1.1 硝酸： $0.25\mathrm{mol / L}$

A.8.2.1.2 硝酸铈铵溶液：将 $6.25\mathrm{g}$ 硝酸铈铵溶于 $100\mathrm{mL}0.25\mathrm{mol} / \mathrm{L}$ 的硝酸溶液，配置后3d内使用。

A.8.2.1.3 乙腈（色谱级）。

A.8.2.1.4 二甘醇（色谱级）。

A.8.2.1.5 二苯醚。

# A.8.2.2 仪器和设备

A.8.2.2.1 蒸馏瓶。

A.8.2.2.2 分光光度计。

# A.8.2.3 分析步骤

A.8.2.3.1 标准液的制备：将 $62.5\mathrm{mg}$ 二甘醇置于 $25~\mathrm{mL}$ 容量瓶中，并用新蒸馏的乙腈和水等体积混合的溶液溶解二甘醇并稀释至刻度，混匀。

A.8.2.3.2 供试液的制备：在 $250~\mathrm{mL}$ 蒸馏瓶中，将 $50.0\mathrm{g}$ 聚乙二醇溶于 $75~\mathrm{mL}$ 二苯醚。如遇到结晶，可预热溶液使结晶溶解。在 $0.133\mathrm{kPa} \sim 0.266\mathrm{kPa}$ 的压力下缓慢蒸馏，蒸馏液收集到有 $1\mathrm{mL}$ 刻度的 $100~\mathrm{mL}$ 接收容器中，直至收集到 $25~\mathrm{mL}$ 蒸馏液后蒸馏停止。向蒸馏液中加入 $20.0~\mathrm{mL}$ 水，用力振荡，再静置分层，冰浴该混合物使二苯醚凝固并分离固液两相。过滤分离得到的水层，收集滤液。用 $5.0~\mathrm{mL}$ 冰水洗涤二苯醚，过滤洗涤液并收集。将滤液和洗涤液混合入一个 $25~\mathrm{mL}$ 容量瓶，加水稀释至刻度，摇匀，如有必要，可将溶液加热至室温。在有玻璃塞的 $125~\mathrm{mL}$ 锥形瓶中，将该溶液与 $25.0~\mathrm{mL}$ 新蒸馏的乙腈混合均匀。

# A.8.2.3.3 测定

分别移取 $10.0\mathrm{mL}$ 标准液和供试液到装有 $15.0\mathrm{mL}$ 的硝酸铈铵溶液的 $50~\mathrm{mL}$ 烧瓶中，混匀。 $2\mathrm{min} \sim 5\mathrm{min}$ 内，使用 $1\mathrm{cm}$ 比色皿，在最大吸收波长 $450~\mathrm{nm}$ 处同时测定两种溶液的吸光度，空白对照为含 $15.0\mathrm{mL}$ 的硝酸铈铵溶液和 $10.0\mathrm{mL}$ 新蒸馏的乙腈与水等体积的混和溶液。供试品的吸光度不应超过标准溶液，即乙二醇和二甘醇的总含量不应超过 $0.25\%$ 。

# A.9 灼烧残渣的测定

# A.9.1 试剂和材料

硫酸。

# A.9.2 测试方法

取供试品 $5.0\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )，置已灼烧至恒重的坩埚中(坩埚灰化前恒重质量 $m_{1}$ )，缓缓灼烧至完全炭化，放冷；加硫酸 $0.5\mathrm{mL}\sim 1\mathrm{mL}$ 使湿润，低温加热至硫酸蒸气除尽后，在 $700~\mathrm{^\circ C}\sim 800~\mathrm{^\circ C}$ 灼烧使完全灰化，移置干燥器内，放冷，称量后，再在 $700~\mathrm{^\circ C}\sim 800~\mathrm{^\circ C}$ 灼烧至恒重（坩埚灰化后恒重质量 $m_{2}$ )，即得。

# A.9.3 结果计算

灼烧残渣的质量分数 $w$ 按式(A.5)计算。

$$
w = \frac {m _ {2} - m _ {1}}{m} \times 100 \% \tag{A.5}
$$

式中：

$m_{2}$ ——坩埚灰化后的恒重质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——坩埚灰化前的恒重质量，单位为克(g)；

$m$ ——供试品的质量，单位为克 $(\mathrm{g})$ 。

# 前言

本标准代替GB25557—2010《食品安全国家标准 食品添加剂 焦磷酸钠》。

本标准与GB25557—2010相比，主要变化如下：

——修改了范围；  
——将“氟化物（以F计）”修改为“氟(F)”;   
——将氟的检验方法修改为GB/T5009.18；  
——将铅的检验方法修改为GB5009.75或GB5009.12；  
——将砷的检验方法修改为GB5009.76或GB5009.11；  
——将重金属的检测方法修改为GB5009.74。

# 食品安全国家标准

# 食品添加剂 焦磷酸钠

# 1 范围

本标准适用于以碳酸钠(或氢氧化钠)和食品添加剂磷酸(含湿法磷酸)为原料生产的食品添加剂焦磷酸钠。

# 2 分子式和相对分子质量

# 2.1 分子式

无水焦磷酸钠： $\mathrm{Na_4P_2O_7}$

十水合焦磷酸钠： $\mathrm{Na_4P_2O_7\bullet 10H_2O}$

# 2.2 相对分子质量

无水焦磷酸钠：265.91（按2018年国际相对原子质量）

十水合焦磷酸钠：446.07（按2018年国际相对原子质量）

# 检验方法

警示：本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应小心谨慎！如溅到皮肤或眼睛上应立即用大量水冲洗，严重者应立即治疗。

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准滴定溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603之规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 盐酸。

A.2.1.2 硝酸溶液： $1 + 9$

A.2.1.3 喹钼柠酮溶液：按HG/T3696.3要求进行配制。

A.2.1.4 铂丝环。

# A.2.2 鉴别方法

# A.2.2.1 钠离子

称取 $1\mathrm{g}$ 试样，加 $20~\mathrm{mL}$ 水溶解，用盐酸润湿铂丝环后，在无色火焰上燃烧至无色，再蘸取试液在无色火焰中燃烧，火焰应呈亮黄色。

# A.2.2.2 焦磷酸根离子

A.2.2.2.1 试样溶液：将 $0.1\mathrm{g}$ 试样溶于 $100~\mathrm{mL}$ 硝酸溶液中。

A.2.2.2.2 试验溶液A：于 $30~\mathrm{mL}$ 喹钼柠酮溶液中滴入 $0.5\mathrm{mL}$ 试样溶液。

A.2.2.2.3 试验溶液B：将剩余的试样溶液于 $95\%$ 加热 $10\mathrm{min}$ ，取 $0.5\mathrm{mL}$ 此溶液滴入 $30~\mathrm{mL}$ 喹钼柠酮溶液中。

A.2.2.2.4 判定：试验溶液B立即形成黄色沉淀，试验溶液A则不出现。

# A.3 焦磷酸钠的测定

# A.3.1 方法提要

焦磷酸钠与盐酸反应生成焦磷酸二氢二钠，向溶液中加入硫酸锌，定量生成焦磷酸锌沉淀和硫酸，用氢氧化钠标准滴定溶液滴定生成的硫酸，再根据氢氧化钠标准滴定溶液的消耗量计算出焦磷酸钠的含量。

# A.3.2 试剂和材料

A.3.2.1 盐酸溶液： $1 + 10$

A.3.2.2 盐酸溶液： $1 + 100$

A.3.2.3 硫酸锌溶液： $125\mathrm{g / L}$ ，将 $125\mathrm{g}$ 硫酸锌 $(\mathrm{ZnSO_4\cdot 7H_2O})$ 溶解于水，用水稀释至 $1\mathrm{L}$ ，在 $\mathrm{pH}$ 计上，根据显示的 $\mathrm{pH}$ ，用硫酸溶液(1+500)或氢氧化钠溶液(6g/L)将 $\mathrm{pH}$ 调至3.8。

A.3.2.4 无水焦磷酸钠。

a）以工业无水焦磷酸钠为原料的制备方法：

第一次结晶：称取 $30\mathrm{g}$ 工业无水焦磷酸钠，置于 $400~\mathrm{mL}$ 烧杯中，加 $100~\mathrm{mL}$ 水，加热溶解，用中速定量滤纸过滤。将滤液在冷水浴中冷却，析出结晶，倾出溶液，用少量水洗涤结晶两次。

第二次结晶：将第一次结晶用少量水加热溶解，在冷水浴中冷却，析出结晶，倾出溶液。

第三次结晶：将第二次结晶按第二次结晶方法再结晶一次。

b）以试剂十水合焦磷酸钠为原料的制备方法：

称取 $80\mathrm{g}$ 试剂十水合焦磷酸钠，按a)中第一次和第二次结晶方法操作。

将上述方法重结晶的焦磷酸钠置于瓷坩埚中，于 $400^{\circ}\mathrm{C}$ 下灼烧至质量恒定。

A.3.2.5 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.1\mathrm{mol / L}$

标定：称取约 $0.5\mathrm{g}$ 无水焦磷酸钠(A.3.2.4)，精确至 $0.0002\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 烧杯中，加 $90~\mathrm{mL}$ 水溶解，在搅拌下加入盐酸溶液(A.3.2.1、A.3.2.2)调至溶液 $\mathrm{pH}$ 为3.8。加入 $50~\mathrm{mL}$ 硫酸锌溶液，搅拌 $5\mathrm{min}$ ，在搅拌下用氢氧化钠标准滴定溶液滴定至溶液的 $\mathrm{pH}$ 接近3.6停止滴定，搅拌 $2\mathrm{min}$ 使溶液达到平衡，继续滴定至 $\mathrm{pH}$ 为3.8，此时每加一滴后要搅拌 $30~\mathrm{s}$ 。

每毫升 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠标准滴定溶液相当于焦磷酸钠的克数 $\rho$ ，单位为克每毫升 $(\mathrm{g / mL})$ ，按式(A.1)计算。

$$
\rho = \frac {m _ {1}}{V _ {1}} \tag {A.1}
$$

式中：

$m_{1}$ —称取无水焦磷酸钠的质量，单位为克 $(\mathrm{g})$

$V_{1}$ ——在标定中消耗氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

# A.3.3 仪器和设备

A.3.3.1 电位滴定仪或 $\mathrm{pH}$ 计：分辨率为 $0.01\mathrm{mV}$ 或 $0.02\mathrm{pH}$

A.3.3.2 电磁搅拌器。

# A.3.4 分析步骤

称取约 $5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，将试样溶于水，转移至 $500\mathrm{mL}$ 容量瓶中，稀释至刻度并摇匀，必要时过滤。

用移液管移取 $50~\mathrm{mL}$ 试验溶液置于 $250~\mathrm{mL}$ 烧杯中，加 $40~\mathrm{mL}$ 水，在搅拌下慢慢加入盐酸溶液调至溶液 $\mathrm{pH}$ 为3.8，然后按A.3.2.5中所述步骤，从“加入 $50~\mathrm{mL}$ 硫酸锌溶液……”开始进行操作。

# A.3.5 结果计算

十水合焦磷酸钠 $\mathrm{(Na_4P_2O_7\cdot 10H_2O)}$ 的质量分数 $\varpi_{1}$ 按式(A.2)计算。

$$
w _ {1} = \frac {\rho \times V _ {2} \times 1.678 \times 10}{m _ {2}} \times 100 \% \quad \dots \tag{A.2}
$$

无水焦磷酸钠 $\mathrm{(Na_4P_2O_7)}$ 的质量分数 $w_{2}$ 按式(A.3)计算。

$$
w _ {2} = \frac {\rho \times V _ {2} \times 10}{m _ {2}} \times 100 \% \tag{A.3}
$$

式中：

$\rho$ ——每毫升 $0.1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠标准滴定溶液相当于焦磷酸钠的克数，单位为克每毫升 $(\mathrm{g} / \mathrm{mL})$ ；

$V_{2}$ ——滴定试验溶液所消耗的氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1.678——无水焦磷酸钠换算为十水合焦磷酸钠的系数；

10 —稀释倍数；

$m_{2}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A.4 灼烧减量的测定

# A.4.1 仪器和设备

A.4.1.1 电热恒温干燥箱：控温范围为 $110^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。

A.4.1.2 高温炉：控温范围为 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$

A.4.1.3 瓷坩埚。

# A.4.2 分析步骤

称取约 $5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 灼烧至质量恒定的瓷坩埚中，于 $110^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 下烘 $4\mathrm{h}$ ，再移入 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 的高温炉中灼烧 $30\mathrm{min}$ ，于干燥器中冷却至室温，称量。

# A.4.3 结果计算

灼烧减量的质量分数 $\varpi_{3}$ 按式(A.4)计算。

$$
w_{3} = \frac{m_{3} - m_{4}}{m_{5}}\times 100\% \tag{A.4}
$$

式中：

$m_{3}$ ——灼烧前试样和瓷坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{4}$ ——灼烧后试样和瓷坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{5}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值为：十水合焦磷酸钠不大于 $0.1\%$ ；无水焦磷酸钠不大于 $0.02\%$ 。

# A.5 水不溶物的测定

# A.5.1 仪器和设备

A.5.1.1 玻璃砂坩埚：滤板孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$   
A.5.1.2 电热恒温干燥箱：控温范围为 $105\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}$ 。

# A.5.2 分析步骤

称取约 $20\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $400~\mathrm{mL}$ 烧杯中，加 $200~\mathrm{mL}$ 水，加热溶解。趁热用 $105^{\circ}\mathrm{C}\pm$

$2^{\circ}\mathrm{C}$ 干燥至质量恒定的玻璃砂坩埚过滤，用热水洗涤至滤液呈无碱性。将玻璃砂坩埚和水不溶物置于 $105~^\circ \mathrm{C}\pm 2\mathrm{^\circ C}$ 的电热恒温干燥箱中干燥至质量恒定。

# A.5.3 结果计算

水不溶物的质量分数 $w_{4}$ 按式(A.5)计算。

$$
w _ {4} = \frac {m _ {6} - m _ {7}}{m _ {8}} \times 100 \% \tag{A.5}
$$

式中：

$m_{6}$ ——水不溶物和玻璃砂坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{7}$ ——玻璃砂坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{8}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.02\%$ 。

# A.6 $\mathbf{pH}(10\mathrm{g} / \mathrm{L}$ 溶液）的测定

# A.6.1 试剂和材料

无二氧化碳的水。

# A.6.2 仪器和设备

酸度计：分辨率为 $0.01\mathrm{pH}$ ，配有玻璃电极和饱和甘汞电极（或复合电极）。

# A.6.3 分析步骤

称取 $1.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $250~\mathrm{mL}$ 烧杯中，加 $100~\mathrm{mL}$ 无二氧化碳的水溶解。在室温下用已校准的酸度计测量试样溶液的 $\mathrm{pH}$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于0.2。

# A.7 正磷酸盐试验

# A.7.1 试剂和材料

硝酸银溶液 $(17\mathrm{g / L})$ 。

# A.7.2 分析步骤

称取 $1.0\mathrm{g}$ 研成粉末的试样，加2滴 $\sim 3$ 滴硝酸银溶液 $(17~\mathrm{g / L})$ ，不应产生明显的黄色。

# A.8 铅(Pb)的测定

按GB5009.75或GB5009.12规定的方法进行测定。试验用水为GB/T6682中规定的二级水。

# A.9 砷（As）的测定

按GB5009.76或GB5009.11规定的方法进行测定。试验用水为GB/T6682中规定的二级水。

# A.10 重金属(以Pb计)的测定

按GB5009.74规定的方法进行测定。试验用水为 $\mathrm{GB / T6682}$ 中规定的二级水。

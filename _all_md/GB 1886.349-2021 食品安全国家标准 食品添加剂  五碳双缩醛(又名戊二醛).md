# 食品添加剂 五碳双缩醛（又名戊二醛）

# 1 范围

本标准适用于由乙烯基甲醚或乙烯基乙醚与丙烯醛反应生成2-烷氧基-3,4-二氢吡喃，再经水解得到的食品添加剂五碳双缩醛（又名戊二醛）。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

五碳双缩醛（又名戊二醛）

# 2.2 分子式

$\mathrm{C}_5\mathrm{H}_8\mathrm{O}_2$

# 附录A

# 检验方法

警示：本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，按国家相关规定操作，使用时需小心谨慎。若溅到皮肤上应立即用水冲洗，严重者应立即就医。在使用挥发性酸时，应在通风橱中进行。

# A.1 一般规定

本标准所用试剂和水在未注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂和制品在未注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 溶解性

移取约 $1\mathrm{mL}$ 试样，置于烧杯中，加入 $1\mathrm{mL}$ 水，摇动时间不少于 $30~\mathrm{s}$ ，并在 $5\mathrm{min}$ 内观察样品的溶解情况。试样应易溶于水。

# A.2.2 2,4-二硝基苯腙熔点鉴别

# A.2.2.1 方法原理

五碳双缩醛与2,4-二硝基苯肼反应生成2,4-二硝基苯腙沉淀，且2,4-二硝基苯腙具有特定的熔点。

# A.2.2.2 试剂和材料

A.2.2.2.1 2,4-二硝基苯肼溶液：取 $0.8\mathrm{g}2,4-$ 二硝基苯肼，加入 $4\mathrm{mL}$ 硫酸，搅拌至完全溶解，继续搅拌，并逐滴沿壁加入 $6\mathrm{mL}$ 水，再加入 $20~\mathrm{mL}$ 乙醇，混合摇匀，过滤，取滤液即得。

A.2.2.2.2 无水乙醇。  
A.2.2.2.3 二氯乙烷。  
A.2.2.2.4 丙酮。  
A.2.2.2.5 定量滤纸。

# A.2.2.3 仪器和设备

A.2.2.3.1 水浴锅。  
A.2.2.3.2 涡旋混合器。  
A.2.2.3.3 熔点仪。

# A.2.2.4 测定步骤

# A.2.2.4.1 2,4-二硝基苯脲生成

取 $0.4\mathrm{mL}$ 试样，加入到 $20~\mathrm{mL}~2,4-$ 二硝基苯肼溶液中，摇匀，静置 $5\mathrm{min}$ ，过滤，用 $200~\mathrm{mL}$ 乙醇分多次洗涤，待乙醇自然挥干后，收集沉淀于 $50~\mathrm{mL}$ 离心管中。

# A.2.2.4.2 二氯乙烷重结晶

在 $80^{\circ}\mathrm{C}$ 水浴中，用 $20\mathrm{mL}80^{\circ}\mathrm{C}$ 的热二氯乙烷溶解沉淀，并每隔 $10\mathrm{min}$ 涡旋一次，共6次，趁热过滤。滤液经冰浴冷却至产生结晶，过滤后收集结晶于 $100~\mathrm{mL}$ 圆底烧瓶中。

# A.2.2.4.3 丙酮重结晶

在圆底烧瓶中加入 $30~\mathrm{mL}$ 丙酮， $60^{\circ}\mathrm{C}$ 水浴回流再次溶解沉淀，回流时应保持水浴液面低于圆底烧瓶中丙酮的液面，每隔 $2\mathrm{min} \sim 3\mathrm{min}$ 轻微摇晃圆底烧瓶，避免粘壁。回流 $2\mathrm{h}$ 后，趁热过滤，收集滤液于 $50~\mathrm{mL}$ 比色管中，将比色管置于冰浴中冷却重结晶，过滤，收集结晶。

# A.2.2.4.4 熔点测定

按GB/T617测定所得结晶的熔点，其熔点范围应为 $185^{\circ}\mathrm{C}\sim 195^{\circ}\mathrm{C}$ 。

# A.3 五碳双缩醛含量的测定

# A.3.1 方法原理

戊二醛的羰基可与盐酸羟胺发生肟化反应再生成戊二肟，戊二醛在生成戊二肟的同时会释放出盐酸，三乙醇胺呈弱碱性，可以中和除去盐酸，加入过量的三乙醇胺，用盐酸标准溶液进行回滴，通过同时进行空白测定的校正，便可计算出戊二醛的含量。

# A.3.2 试剂和材料

A.3.2.1 三乙醇胺溶液: $74 \mathrm{~g} / \mathrm{L}$ 。将 $74 \mathrm{~g}$ 三乙醇胺用一定量水溶解后转移至 $1 \mathrm{~L}$ 容量瓶中, 用水定容刻度后混匀。  
A.3.2.2 盐酸羟胺溶液: $35 \mathrm{~g} / \mathrm{L}, \mathrm{pH} 3.60$ 。将 $35.0 \mathrm{~g}$ 盐酸羟胺用一定量水溶解后转移至 $1 \mathrm{~L}$ 容量瓶中,用水定容刻度混匀后, 用三乙醇胺溶液调 $\mathrm{pH}$ 至 3.60。  
A.3.2.3 盐酸标准滴定溶液： $c(\mathrm{HCl}) = 0.5\mathrm{mol / L}$

# A.3.3 仪器和设备

A.3.3.1 电子天平：感量为 $0.1\mathrm{g}$ 和 $0.0001\mathrm{g}$   
A.3.3.2 磁力搅拌器。  
A.3.3.3 酸度计：精度为 $\pm 0.01$

# A.3.4 测定方法

将 $65.0 \mathrm{~mL} \mathrm{pH} 3.60$ 的盐酸羟胺溶液和 $30.8 \mathrm{~mL}$ 三乙醇胺溶液加入至 $200 \mathrm{~mL}$ 小烧杯中，在每只小烧杯放入聚四氟乙烯(或相当材质)的转子，然后将小烧杯置于磁力搅拌器上搅拌，准确加入一定量的试样（约相当于 $300 \mathrm{mg}$ 五碳双缩醛），加盖，搅拌。在室温下至少搅拌 $60 \mathrm{~min}$ ，但不超过 $90 \mathrm{~min}$ 。同时不加试样做试剂空白试验。搅拌结束后，用 $0.5 \mathrm{~mol} / \mathrm{L}$ 盐酸将试样和空白样滴定至 $\mathrm{pH} 3.60$ 为终点。

注：在中和分析过程中，搅拌速度是一个关键因素。当需要搅拌时，应确保溶液内不要因此混入气泡，且试样和空白样均始终保持同样的搅拌速度。

# A.3.5 结果计算

五碳双缩醛含量以 $\mathrm{C}_5\mathrm{H}_8\mathrm{O}_2$ 的质量分数 $\varpi_{1}$ 计，按式(A.1)计算。

$$
w _ {1} = \frac {M \times c \times \left(V _ {1} - V _ {2}\right)}{m \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$M$ 五碳双缩醛 $\left(\frac{1}{2}\mathrm{C}_5\mathrm{H}_8\mathrm{O}_2\right)$ 的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})(M = 50.06)$

$c$ ——盐酸标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{1}$ ——空白样液所耗盐酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试样液所耗盐酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品质量的数值，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果保留三位有效数字。试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# A.4 甲醇和乙醇含量的测定

# A.4.1 方法原理

试样经气相色谱分离，氢火焰离子化检测器检测，外标法定量。

# A.4.2 试剂和材料

A.4.2.1 异丙醇：色谱纯。  
A.4.2.2 甲醇标准品 $\mathrm{(CH_4O,CAS}$ 号：67-56-1)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。  
A.4.2.3 乙醇标准品 $\left(\mathrm{C}_{2} \mathrm{H}_{6} \mathrm{O}, \mathrm{CAS}\right.$ 号：64-17-5）：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# A.4.3 仪器和设备

A.4.3.1 气相色谱仪，配氢火焰离子化检测器(FID)。  
A.4.3.2 电子天平：感量为 $0.1\mathrm{mg}$

# A.4.4 气相色谱参考条件

A.4.4.1 色谱柱：聚乙二醇石英毛细管柱，柱长 $18\mathrm{m}$ ，内径 $0.53\mathrm{mm}$ ，膜厚 $1.0\mu \mathrm{m}$ ，或等效柱。  
A.4.4.2 色谱柱温度：初温 $40^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $4.0^{\circ}\mathrm{C / min}$ 升到 $130^{\circ}\mathrm{C}$ ，以 $20^{\circ}\mathrm{C / min}$ 升到 $200^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ 。  
A.4.4.3 检测器温度： $250^{\circ}C$   
A.4.4.4 进样口温度： $250^{\circ}C$   
A.4.4.5 载气流速： $1\mathrm{mL / min}$   
A.4.4.6 氢气流速： $40~\mathrm{mL / min}$   
A.4.4.7 空气流速： $400~\mathrm{mL / min}$   
A.4.4.8 进样量： $1\mu \mathrm{L}$   
A.4.4.9 分流比： $20:1$

# A.4.5 分析步骤

# A.4.5.1 标准溶液的制备

A.4.5.1.1 甲醇标准储备液: $5000 \mathrm{mg} / \mathrm{L}$ 。准确称取 $0.5 \mathrm{~g}$ 甲醇标准品, 精确至 $0.0001 \mathrm{~g}$ , 置于 $100 \mathrm{~mL}$ 容量瓶中, 用异丙醇定容至刻度, 混匀, $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 冰箱密封保存。  
A.4.5.1.2 乙醇标准储备液： $5000\mathrm{mg / L}$ 。准确称取 $0.5\mathrm{g}$ 乙醇标准品，精确至 $0.0001\mathrm{g}$ ，置于 $100\mathrm{mL}$ 容量瓶中，用异丙醇定容至刻度，混匀， $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱密封保存。  
A.4.5.1.3 混合系列标准工作液：分别吸取 $0.05 \mathrm{~mL}, 0.2 \mathrm{~mL}, 0.4 \mathrm{~mL}, 0.8 \mathrm{~mL}, 1.6 \mathrm{~mL}, 2 \mathrm{~mL}$ 甲醇标准储备液和乙醇标准储备液，于6个 $10 \mathrm{~mL}$ 容量瓶中，用异丙醇定容至刻度，依次配制成甲醇和乙醇含量为 $25 \mathrm{mg} / \mathrm{L}, 100 \mathrm{mg} / \mathrm{L}, 200 \mathrm{mg} / \mathrm{L}, 400 \mathrm{mg} / \mathrm{L}, 800 \mathrm{mg} / \mathrm{L}, 1000 \mathrm{mg} / \mathrm{L}$ 系列标准溶液，现配现用。

# A.4.5.2 样品溶液的制备

准确称取 $1\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于 $10\mathrm{mL}$ 容量瓶中，用异丙醇稀释至刻度，混匀，备用。

# A.4.6 测定

# A.4.6.1 标准曲线的绘制

将标准系列工作液分别注入气相色谱仪中，测定相应的峰面积。标准谱图参见附录B中图B.1。以标准工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。

# A.4.6.2 试样溶液的测定

将试样溶液注入气相色谱仪中，得到相应的峰面积，根据标准曲线得到待测液中甲醇或乙醇的浓度。

# A.4.7 结果计算

甲醇或乙醇含量 $w_{2}$ ，以质量分数计，按式(A.2)计算。

$$
w _ {2} = \frac {c \times V}{m \times 1000 \times 1000} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$c$ ——根据标准曲线计算得到的试样中甲醇或乙醇的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样溶液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果保留三位有效数字。实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

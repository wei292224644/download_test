# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2561—2010《进出口食品中吡啶类农药残留量的测定液相色谱一质谱/质谱法》。与SN/T2561—2010相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“进出口食品”改为“食品”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

# 食品安全国家标准

# 食品中吡啶类农药残留量的测定 液相色谱一质谱/质谱法

# 1 范围

本标准规定了食品中吡虫啉、啶虫脒、咪唑乙烟酸、氟啶草酮、啶酰菌胺、噻唑烟酸和氟硫草定7种吡啶类农药残留量检测的制样和液相色谱一质谱/质谱法。

本标准适用于大米、小麦、马铃薯、菠菜、柑橘、核桃仁、茶叶、猪肉、鱼肉、猪肝和牛奶中吡虫啉、啶虫脒、咪唑乙烟酸、氟啶草酮、啶酰菌胺、噻唑烟酸和氟硫草定残留量的测定和确证，其他食品可参照执行。

# 3 原理

试样中残留量的农药用氯化钠盐析后乙腈提取，提取液经石墨化炭黑或 $\mathrm{C_{18}}$ 固相萃取小柱净化，用液相色谱一质谱/质谱仪检测和确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：液相色谱级。  
4.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：液相色谱级。  
4.1.3 甲苯 $(\mathrm{C_7H_8})$ 。  
4.1.4 乙酸 $\mathrm{(CH_3COOH)}$   
4.1.5 氯化钠 $(\mathrm{NaCl})$ 。  
4.1.6 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置于干燥器内备用。

# 4.2 溶液配制

4.2.1 $0.1\%$ 乙酸溶液：取 $1\mathrm{mL}$ 乙酸，用水稀释并定容至 $1000~\mathrm{mL}$   
4.2.2 甲醇一水 $(1 + 1$ ，体积比)：量取 $100~\mathrm{mL}$ 甲醇，加入 $100~\mathrm{mL}$ 水，混匀备用。  
4.2.3 甲苯—乙腈酸性溶液：量取 $100\mathrm{mL}$ 甲苯，加入 $300\mathrm{mL}$ 乙腈和 $4\mathrm{mL}$ 乙酸，混匀备用。

# 4.3 标准品

4.3.1 吡虫啉、啶虫脒、咪唑乙烟酸、氟啶草酮、啶酰菌胺、噻唑烟酸和氟硫草定标准物质：纯度 $\geq 98\%$ 参见附录A。

# 4.4 标准溶液配制

4.4.1 标准储备液：分别准确称取适量的上述标准品，用甲醇配制成 $1.0\mathrm{mg / mL}$ 的标准储备液， $4^{\circ}C$ 避

光保存。

4.4.2 混合标准中间液的配制：取上述标准储备液适量，用甲醇配制成 $10\mathrm{g / mL}$ 的混合标准溶液， $4^{\circ}\mathrm{C}$ 避光保存。  
4.4.3 标准工作溶液的配制：吸取适量的混合标准中间液，用空白样品基质配制成适当浓度的标准工作溶液，使用前配制。

4.5 材料

4.5.1 $\mathrm{C_{18}}$ 固相萃取小柱： $3\mathrm{mL} / 500\mathrm{mg}$ ，或相当者。  
4.5.2 石墨化炭黑小柱： $6\mathrm{mL} / 500\mathrm{mg}$ ，或相当者。

# 5 仪器和设备

5.1 高效液相色谱—质谱/质谱仪：配电喷雾离子源(ESI)   
5.2 分析天平：感量为 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$   
5.3 涡旋振荡器。  
5.4 离心机（最大转速 $10000\mathrm{r / min}$ ）。  
5.5 氮气吹干仪。  
5.6 旋转蒸发仪。  
5.7 固相萃取装置。  
5.8 捣碎机。

# 6 试样制备与保存

6.1 取样部位

样品取样部位按GB2763的规定执行。

6.2 试样制备

在制样的操作过程中,应防止样品受到污染或发生残留物含量的变化。

6.2.1 柑橘、马铃薯和菠菜

取有代表性样品 $500\mathrm{g}$ , 切碎后 (不可用水洗) 用捣碎机加工成浆状。混匀, 装入洁净的盛样容器内, 密封并标明标记。

6.2.2 大米、小麦、核桃仁和茶叶

取有代表性样品 $500\mathrm{g}$ , 用粉碎机粉碎。混匀, 装入洁净的盛样容器内, 密封并标明标记。

6.2.3 鱼肉、猪肉和猪肝

取有代表性样品 $500\mathrm{g}$ ，用捣碎机充分捣碎均匀，装入洁净的盛样容器内，密封并标明标记。

6.3 试样保存

大米、小麦、核桃仁和茶叶于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 下保存; 柑橘、马铃薯、菠菜、猪肉、鱼肉、猪肝和牛奶于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 7 分析步骤

7.1 提取

7.1.1 大米、小麦、核桃仁和茶叶

对于大米、小麦、核桃仁样品，称取 $2\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )。对于茶叶样品，称取 $1\mathrm{g}$ 试样（精确至 $0.01\mathrm{g})$ 。将称取的试样置于 $50~\mathrm{mL}$ 离心管中，加入 $5\mathrm{mL}$ 水混匀后浸泡 $30\mathrm{min}$ ，加入 $2\mathrm{g}$ 氯化钠和 $10$ mL乙腈，于涡旋振荡器上混合 $3\mathrm{min},5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将上层清液转移至另一支离心管中。再

用 $5\mathrm{mL}$ 乙腈重复以上提取过程，合并提取液，于 $40^{\circ}\mathrm{C}$ 下氮吹浓缩至 $2\mathrm{mL}$ ，待净化。

# 7.1.2 柑橘、牛奶

称取 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）样品于 $50~\mathrm{mL}$ 离心管中，加入 $2\mathrm{g}$ 氯化钠和 $10\mathrm{mL}$ 乙腈，于涡旋振荡器上混合 $3\mathrm{min}$ 。 $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将上层清液转移至另一支离心管中。再用 $5\mathrm{mL}$ 乙腈重复以上提取过程，合并提取液，于 $40^{\circ}\mathrm{C}$ 下氮吹浓缩至 $2\mathrm{mL}$ ，待净化。

# 7.1.3 马铃薯、菠菜、猪肉、鱼肉和猪肝

称取 $2\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 样品于 $50\mathrm{mL}$ 离心管中，加入 $3\mathrm{mL}$ 水、 $2\mathrm{g}$ 氯化钠和 $10\mathrm{mL}$ 乙腈，于涡旋振荡器上混合 $3\mathrm{min}$ 。 $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将上层清液转移至另一支离心管中。再用 $5\mathrm{mL}$ 乙腈重复以上提取过程，合并提取液，于 $40^{\circ}\mathrm{C}$ 下氮吹浓缩至 $2\mathrm{mL}$ ，待净化。

# 7.2 净化

# 7.2.1 大米、小麦、马铃薯、柑橘、猪肉、鱼肉和牛奶

在 $\mathrm{C}_{18}$ 固相萃取小柱上端装入 $1\mathrm{cm}$ 高的无水硫酸钠, 先用 $5\mathrm{mL}$ 乙腈预淋洗小柱, 弃去淋洗液。然后将上述所得提取液过柱, 并用 $4\mathrm{mL}$ 乙腈洗脱, 控制流速为 $0.5\mathrm{mL} / \mathrm{min}$ , 收集所有流出液。在 $40^{\circ}\mathrm{C}$ 下氮吹浓缩至近干, 用 $1.0\mathrm{mL}$ 甲醇—水溶液溶解残渣, 过 $0.45\mu \mathrm{m}$ 有机滤膜供液相色谱—质谱/质谱仪测定。

# 7.2.2 菠菜、猪肝、核桃仁和茶叶

在石墨化炭黑固相萃取小柱上端装入 $1 \mathrm{~cm}$ 高的无水硫酸钠, 先用 $10 \mathrm{~mL}$ 甲苯一乙腈酸性溶液预淋洗小柱, 弃去淋洗液。然后加入上述提取液过柱, 用 $25 \mathrm{~mL}$ 甲苯一乙腈酸性溶液洗脱, 控制流速为 $0.5 \mathrm{~mL} / \mathrm{min}$ , 收集所有流出液。在 $40^{\circ} \mathrm{C}$ 下水浴旋转浓缩至 $1 \mathrm{~mL} \sim 2 \mathrm{~mL}$ 后, 再用氮气流吹至近干。用 $1.0 \mathrm{~mL}$ 甲醇一水溶液溶解残渣, 过 $0.45 \mu \mathrm{m}$ 有机相微孔滤膜后, 供液相色谱一质谱/质谱仪测定。

b) 流动相: 梯度洗脱程序见表 1;  
c) 流速: $0.5 \mathrm{~mL} / \mathrm{min}$ ;   
d) 柱温: $30^{\circ} \mathrm{C}$ ;   
e) 进样量: $10 \mu \mathrm{L}$ 。

# 7.4 空白试验

除不加试样外，均按7.4～7.3的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中吡啶类农药的残留量，计算结果需扣除空白。

$$
X = \frac {A \times c \times V}{A _ {s} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中吡啶类农药组分的残留量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——样液中吡啶类农药组分的峰面积；

$c$ ——标准工作溶液中吡啶类农药组分的浓度，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准工作溶液中吡啶类农药组分的峰面积；

$m$ ——最终样液所代表的试样质量，单位为克(g)。

计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 D 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 E 的要求。

# 10 定量限和回收率

# 10.1 定量限

大米、小麦、马铃薯、菠菜、柑橘、核桃仁、猪肉、鱼肉、猪肝和牛奶中吡啶类农药的定量限均为 $0.005\mathrm{mg / kg}$ ；茶叶中吡啶类农药的定量限为 $0.01\mathrm{mg / kg}$ 。

# 10.2 回收率

大米、小麦、马铃薯、菠菜、柑橘、核桃仁、茶叶、猪肉、鱼肉、猪肝和牛奶中吡啶类农药的添加水平及回收率数据参见附录F。

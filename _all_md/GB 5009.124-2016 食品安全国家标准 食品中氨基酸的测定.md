# 前言

本标准代替GB/T5009.124—2003《食品中氨基酸的测定》。

本标准与GB/T5009.124—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中氨基酸的测定”；  
扩大了适用范围；  
——增加了方法的检出限和定量限；  
——修改了结果计算的公式。

# 食品安全国家标准

# 食品中氨基酸的测定

# 1 范围

本标准规定了用氨基酸分析仪(茚三酮柱后衍生离子交换色谱仪)测定食品中氨基酸的方法。

本标准适用于食品中酸水解氨基酸的测定，包括天冬氨酸、苏氨酸、丝氨酸、谷氨酸、脯氨酸、甘氨酸、丙氨酸、缬氨酸、蛋氨酸、异亮氨酸、亮氨酸、酪氨酸、苯丙氨酸、组氨酸、赖氨酸和精氨酸共16种氨基酸。

# 2 原理

食品中的蛋白质经盐酸水解成为游离氨基酸，经离子交换柱分离后，与茚三酮溶液产生颜色反应，再通过可见光分光光度检测器测定氨基酸含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682中规定的一级水。

# 3.1 试剂

3.1.1 盐酸（HCl）：浓度 $\geq 36\%$ ，优级纯。  
3.1.2 苯酚 $\mathrm{(C_6H_5OH)}$ 。  
3.1.3 氮气：纯度 $99.9\%$   
3.1.4 柠檬酸钠 $\mathrm{(Na_3C_6H_5O_7\cdot 2H_2O)}$ ：优级纯。  
3.1.5 氢氧化钠(NaOH)：优级纯。

# 3.2 试剂配制

3.2.1 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ：取 $500~\mathrm{mL}$ 盐酸加水稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.2 冷冻剂：市售食盐与冰块按质量 $1:3$ 混合。  
3.2.3 氢氧化钠溶液 $(500\mathrm{g / L})$ ：称取 $50\mathrm{g}$ 氢氧化钠，溶于 $50~\mathrm{mL}$ 水中，冷却至室温后，用水稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.4 柠檬酸钠缓冲溶液 $\left[c\left(\mathrm{Na}^{+}\right)=0.2 \mathrm{~mol} / \mathrm{L}\right]$ ：称取 $19.6 \mathrm{~g}$ 柠檬酸钠加入 $500 \mathrm{~mL}$ 水溶解，加入 $16.5 \mathrm{~mL}$ 盐酸，用水稀释至 $1000 \mathrm{~mL}$ ，混匀，用 $6 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液或 $500 \mathrm{~g} / \mathrm{L}$ 氢氧化钠溶液调节 $\mathrm{pH}$ 至 2.2。  
3.2.5 不同 $\mathrm{pH}$ 和离子强度的洗脱用缓冲溶液：参照仪器说明书配制或购买。  
3.2.6 苷三酮溶液：参照仪器说明书配制或购买。

# 3.3 标准品

3.3.1 混合氨基酸标准溶液：经国家认证并授予标准物质证书的标准溶液。

3.3.2 16种单个氨基酸标准品：固体，纯度 $\geqslant 98\%$

# 3.4 标准溶液配制

3.4.1 混合氨基酸标准储备液 $(1\mu \mathrm{mol} / \mathrm{mL})$ ：分别准确称取单个氨基酸标准品（精确至 $0.00001\mathrm{g}$ ）于同一 $50~\mathrm{mL}$ 烧杯中，用 $8.3\mathrm{mL}6\mathrm{mol} / \mathrm{L}$ 盐酸溶液溶解，精确转移至 $250~\mathrm{mL}$ 容量瓶中，用水稀释定容至刻度，混匀(各氨基酸标准品称量质量参考值见表1)。  
3.4.2 混合氨基酸标准工作液 $(100\mathrm{nmol / mL})$ ：准确吸取混合氨基酸标准储备液 $1.0~\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，加 $\mathrm{pH}2.2$ 柠檬酸钠缓冲溶液定容至刻度，混匀，为标准上机液。

# 4 仪器和设备

4.1 实验室用组织粉碎机或研磨机。  
4.2 匀浆机。  
4.3 分析天平：感量分别为 $0.0001\mathrm{g}$ 和 $0.00001\mathrm{g}$ 。  
4.4 水解管：耐压螺盖玻璃试管或安瓿瓶，体积为 $20~\mathrm{mL}\sim 30~\mathrm{mL}$   
4.5 真空泵：排气量 $\geqslant 40\mathrm{L / min}$   
4.6 酒精喷灯。  
4.7 电热鼓风恒温箱或水解炉。  
4.8 试管浓缩仪或平行蒸发仪（附带配套 $15\mathrm{mL}\sim 25\mathrm{mL}$ 试管）。  
4.9 氨基酸分析仪：茚三酮柱后衍生离子交换色谱仪。

# 5 分析步骤

# 5.1 试样制备

固体或半固体试样使用组织粉碎机或研磨机粉碎，液体试样用匀浆机打成匀浆密封冷冻保存，分析用时将其解冻后使用。

# 5.2 试样称量

均匀性好的样品，如奶粉等，准确称取一定量试样（精确至 $0.0001\mathrm{g}$ ），使试样中蛋白质含量在 $10\mathrm{mg} \sim 20\mathrm{mg}$ 范围内。对于蛋白质含量未知的样品，可先测定样品中蛋白质含量。将称量好的样品置于水解管中。

很难获得高均匀性的试样，如鲜肉等，为减少误差可适当增大称样量，测定前再做稀释。

对于蛋白质含量低的样品，如蔬菜、水果、饮料和淀粉类食品等，固体或半固体试样称样量不大于 $2\mathrm{g}$ ，液体试样称样量不大于 $5\mathrm{g}$ 。

# 5.3 试样水解

根据试样的蛋白质含量，在水解管内加 $10\mathrm{mL}\sim 15\mathrm{mL}6\mathrm{mol / L}$ 盐酸溶液。对于含水量高、蛋白质含量低的试样，如饮料、水果、蔬菜等，可先加入约相同体积的盐酸混匀后，再用 $6\mathrm{mol / L}$ 盐酸溶液补充至大约 $10~\mathrm{mL}$ 。继续向水解管内加入苯酚3滴 $\sim 4$ 滴。

将水解管放入冷冻剂中，冷冻 $3\mathrm{min}\sim 5\mathrm{min}$ ，接到真空泵的抽气管上，抽真空(接近 $0\mathrm{Pa}$ )，然后充入氮气，重复抽真空-充入氮气3次后，在充氮气状态下封口或拧紧螺丝盖。

将已封口的水解管放在 $110^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 的电热鼓风恒温箱或水解炉内，水解 $22\mathrm{h}$ 后，取出，冷却至室温。

打开水解管，将水解液过滤至 $50~\mathrm{mL}$ 容量瓶内，用少量水多次冲洗水解管，水洗液移入同一 $50~\mathrm{mL}$ 容量瓶内，最后用水定容至刻度，振荡混匀。

准确吸取 $1.0\mathrm{mL}$ 滤液移入到 $15\mathrm{mL}$ 或 $25\mathrm{mL}$ 试管内，用试管浓缩仪或平行蒸发仪在 $40^{\circ}\mathrm{C} \sim$ $50^{\circ}\mathrm{C}$ 加热环境下减压干燥，干燥后残留物用 $1\mathrm{mL} \sim 2\mathrm{mL}$ 水溶解，再减压干燥，最后蒸干。

用 $1.0\mathrm{mL}\sim 2.0\mathrm{mL}$ $\mathrm{pH}2.2$ 柠檬酸钠缓冲溶液加入到干燥后试管内溶解，振荡混匀后，吸取溶液通过 $0.22\mu \mathrm{m}$ 滤膜后，转移至仪器进样瓶，为样品测定液，供仪器测定用。

# 5.4 测定

# 5.4.1 仪器条件

使用混合氨基酸标准工作液注入氨基酸自动分析仪，参照JJG1064—2011氨基酸分析仪检定规程及仪器说明书，适当调整仪器操作程序及参数和洗脱用缓冲溶液试剂配比，确认仪器操作条件。

# 5.4.2 色谱参考条件

a）色谱柱：磺酸型阳离子树脂；  
b）检测波长： $570~\mathrm{nm}$ 和 $440~\mathrm{nm}$

# 5.4.3 试样的测定

混合氨基酸标准工作液和样品测定液分别以相同体积注入氨基酸分析仪，以外标法通过峰面积计算样品测定液中氨基酸的浓度。

混合氨基酸标准储备液中各氨基酸的含量按式(1)计算：

$$
c _ {j} = \frac {m _ {j}}{M _ {j} \times 2 5 0} \times 1 0 0 0 \quad \dots \tag {1}
$$

式中：

$c_{j}$ ——混合氨基酸标准储备液中氨基酸 $j$ 的浓度，单位为微摩尔每毫升 $(\mu \mathrm{mol} / \mathrm{mL})$ ；

$m_{j}$ ——称取氨基酸标准品 $j$ 的质量，单位为毫克 $(\mathrm{mg})$ ；

$M_{j}$ ——氨基酸标准品 $j$ 的分子量；

250 ——定容体积，单位为毫升 $(\mathrm{mL})$

1000 ——换算系数。

结果保留4位有效数字。

# 6.2 样品中氨基酸含量的计算

样品测定液氨基酸的含量按式(2)计算：

$$
c _ {i} = \frac {c _ {s}}{A _ {s}} \times A _ {i} \tag {2}
$$

式中：

$c_{i}$ ——样品测定液氨基酸 $i$ 的含量，单位为纳摩尔每毫升 $(\mathrm{nmol} / \mathrm{mL})$ ；

$A_{i}$ ——试样测定液氨基酸 $i$ 的峰面积；

$A_{s}$ ——氨基酸标准工作液氨基酸 $s$ 的峰面积；

$c_{s}$ ——氨基酸标准工作液氨基酸 $s$ 的含量，单位为纳摩尔每毫升 $(\mathrm{nmol} / \mathrm{mL})$ 。

试样中各氨基酸的含量按式(3)计算：

$$
X _ {i} = \frac {c _ {i} \times F \times V \times M}{m \times 1 0 ^ {9}} \times 1 0 0 \quad \dots \tag {3}
$$

式中：

$X_{i}$ ——试样中氨基酸 $i$ 的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$c_{i}$ ——试样测定液中氨基酸 $i$ 的含量, 单位为纳摩尔每毫升 $(\mathrm{nmol} / \mathrm{mL})$ ;

$F$ ——稀释倍数；

V ——试样水解液转移定容的体积，单位为毫升 $(\mathrm{mL})$ ；

$M$ ——氨基酸 $i$ 的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ，各氨基酸的名称及摩尔质量见表2；

$m$ —称样量，单位为克 $(\mathrm{g})$

$10^{9}$ ——将试样含量由纳克 $(\mathrm{ng})$ 折算成克 $(\mathrm{g})$ 的系数；

100 ——换算系数。

# 食品中烟酸和烟酰胺的测定

2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替GB/T5009.89—2003《食品中烟酸的测定》、GB5413.15—2010《食品安全国家标准婴幼儿食品和乳品种烟酸和烟酰胺的测定》和GB/T9695.25—2008《肉与肉制品维生素PP含量测定》。

本标准与GB5413.15—2010相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中烟酸和烟酰胺的测定”；  
——调整了试剂顺序和格式；  
——修改并细化了适用于不同食品种类的前处理方法(第一法)；  
——增加了标准溶液浓度校正方法（第二法）；  
——重新评估了检出限，增加了定量限。

# 食品安全国家标准

# 食品中烟酸和烟酰胺的测定

# 1 范围

本标准规定了食品中烟酸和烟酰胺的测定方法。

本标准第一法为微生物法，适用于各类食品包括以天然食品为基质的强化食品中烟酸和烟酰胺总量的测定；第二法为高效液相色谱法，适用于强化食品中烟酸和烟酰胺的测定。

# 第一法 微生物法

# 2 原理

烟酸(烟酰胺)是植物乳杆菌Lactobacillus plantarum（ATCC8014)生长所必需的营养素，在一定控制条件下，利用植物乳杆菌对烟酸和烟酰胺的特异性，在含有烟酸和烟酰胺的样品中生长形成的光密度来测定烟酸和烟酰胺的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。培养基可购买符合测试要求的商品化培养基。

# 3.1 菌种

植物乳杆菌 Lactobacillus plantarum (ATCC 8014), 或其他有效标准菌株。

# 3.2 试剂

3.2.1 盐酸（HCl）。  
3.2.2 氢氧化钠(NaOH)。  
3.2.3 氯化钠 $\mathrm{(NaCl)}$ 。  
3.2.4 浓硫酸 $\mathrm{(H_2SO_4)}$ 。  
3.2.5 乙醇 $\mathrm{(C_2H_5OH)}$ 。

# 3.3 试剂配制

3.3.1 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：吸取 $83~\mathrm{mL}$ 盐酸，于 $1000~\mathrm{mL}$ 烧杯中，加 $917~\mathrm{mL}$ 水，混匀。  
3.3.2 盐酸溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：吸取盐酸溶液(3.3.1) $10~\mathrm{mL}$ ，加水溶解至 $100~\mathrm{mL}$ 。  
3.3.3 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $40\mathrm{g}$ 氢氧化钠于 $1000~\mathrm{mL}$ 烧杯中，加水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。

3.3.4 氢氧化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：吸取氢氧化钠溶液(3.3.3) $10~\mathrm{mL}$ ，加水溶解至 $100~\mathrm{mL}$ 。  
3.3.5 生理盐水 $(0.9\%)$ ：称取 $9\mathrm{g}$ 氯化钠，溶解于 $1000~\mathrm{mL}$ 水中，分装 $10~\mathrm{mL}$ 于试管中， $121^{\circ}\mathrm{C}$ 灭菌 $15\mathrm{min}$ ，备用。  
3.3.6 乙醇溶液 $(25\%)$ ：量取 $200~\mathrm{mL}$ 无水乙醇与 $800~\mathrm{mL}$ 水混匀。  
3.3.7 硫酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：于 $2000~\mathrm{mL}$ 烧杯中先注入 $700~\mathrm{mL}$ 水，吸取 $56~\mathrm{mL}$ 硫酸沿烧杯壁缓慢倒入水中，用水稀释到 $1000~\mathrm{mL}$ 。

# 3.4 培养基

3.4.1 乳酸杆菌琼脂培养基：可按A.1配制。  
3.4.2 乳酸杆菌肉汤培养基：可按A.2配制。  
3.4.3 烟酸测定用培养基：可按A.3配制。

注：一些商品化合成培养基效果良好，商品化合成培养基按标签说明进行配制。

# 3.5 标准品

烟酸 $\left(\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{NO}_{2}\right)$ ：纯度 $\geqslant 99.5\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.6 标准溶液配制

3.6.1 烟酸标准储备液 $(0.1\mathrm{mg / mL})$ : 精确称取 $50.0\mathrm{mg}$ 烟酸标准品, 用乙醇溶液溶解并移至 $500~\mathrm{mL}$ 容量瓶中, 定容, 混匀于 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冷藏。此溶液每毫升相当于 $100\mu \mathrm{g}$ 烟酸。  
3.6.2 烟酸标准中间液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取 $1.0~\mathrm{mL}$ 烟酸标准储备液(3.6.1)置于 $100~\mathrm{mL}$ 棕色容量瓶中，用乙醇溶液稀释并定容至刻度，混匀于 $2^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冷藏。此溶液每毫升相当于 $1\mu \mathrm{g}$ 烟酸。  
3.6.3 烟酸标准工作液 $(100\mathrm{ng / mL})$ ：准确吸取 $5.00~\mathrm{mL}$ 烟酸标准中间液(3.6.2)置于 $50~\mathrm{mL}$ 容量瓶中，用水稀释定容至刻度，混匀于 $2^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冷藏。此溶液每毫升相当于 $100\mathrm{ng}$ 烟酸。

# 4 仪器和设备

4.1 天平：感量分别为 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$ 。  
4.2 均质设备：用于试样的均质化。  
4.3 恒温培养箱： $36\mathrm{^\circ C}\pm 1\mathrm{^\circ C}$   
4.4 涡旋振荡器。  
4.5 压力蒸汽消毒器： $121^{\circ}\mathrm{C}(0.10\mathrm{MPa} \sim 0.12\mathrm{MPa})$ 。  
4.6 离心机：转速 $\geqslant 2000\mathrm{r / min}$   
4.7 pH计：精度 $\pm 0.01$   
4.8 分光光度计。  
4.9 超净工作台。  
4.10 超声波振荡器。

注：玻璃仪器使用前，用活性剂(月桂磺酸钠或家用洗涤剂加入到洗涤用水中即可)对硬玻璃测定管及其他必要的玻璃器皿进行清洗，清洗之后烘干，于 $170~\mathrm{^\circ C}$ 干热 $3\mathrm{h}$ 后使用。

# 5 分析步骤

# 5.1 储备菌种的制备

将菌种植物乳杆菌Lactobacillus plantarum（ATCC8014)转接至乳酸杆菌琼脂培养基中，在 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温培养箱中培养 $20\mathrm{h} \sim 24\mathrm{h}$ ，取出后放入 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱中保存。每月至少传种一次，作为储备菌株保存。

实验前将储备菌株接种至乳酸杆菌琼脂培养基中，在 $36\mathrm{^\circ C}\pm 1\mathrm{^\circ C}$ 恒温培养箱中培养 $20\mathrm{h}\sim 24\mathrm{h}$ 以活化菌株，用于接种液的制备。保存数周以上的储备菌种，不能立即用作接种液制备，实验前宜连续传种2代～3代以保证菌株活力。

# 5.2 接种液的制备

试验前一天，从乳酸杆菌琼脂培养基移取部分菌种于灭菌的 $10\mathrm{mL}$ 乳酸杆菌肉汤培养基中，于 $36^{\circ}C\pm 1^{\circ}C$ 恒温培养箱中培养 $6\mathrm{h}\sim 18\mathrm{h}$ 。在无菌条件下离心该培养液 $15\mathrm{min}$ ，倾去上清液。加入 $10~\mathrm{mL}$ 已灭菌的生理盐水重新分散细胞，于旋涡混合器上快速混合均匀，离心 $15\mathrm{min}$ ，倾去上清液。重复离心和清洗步骤三次。以第三次细胞分散液中吸取 $1\mathrm{mL}$ 加入 $10~\mathrm{mL}$ 已灭菌的生理盐水，使其充分混合均匀制成混悬液，备用。用721分光光度计，在 $550~\mathrm{nm}$ 波长下，以 $0.9\%$ 生理盐水为参比，读取该菌悬液的透光值，用 $0.9\%$ 生理盐水或第三次细胞分散液调整透光值，使其范围在 $60\% \sim 80\%$ 之间。立即使用。

# 5.3 试样制备

谷薯类、豆类、坚果(去壳)等试样需粉碎、研磨、过筛（筛板孔径 $0.3\mathrm{mm}\sim 0.5\mathrm{mm}$ ）；乳粉、米粉等试样混匀；肉、蛋、鱼、动物内脏等用打碎机制成食糜；果蔬、半固体食品等试样需匀浆混匀；液体试样用前振摇混合。如不能马上检测，于 $4^{\circ}\mathrm{C}$ 冰箱保存。

# 5.4 试样提取

准确称取约烟酸试样，一般乳类、新鲜果蔬试样 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）；谷类、豆类、坚果类、内脏、生肉、干制试样 $0.2\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )，液态试样 $5\mathrm{g}$ ；乳粉、米粉等准确称取适量试样 $2\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）；一般营养素补充剂、复合营养强化剂 $0.1\mathrm{g}\sim 0.5\mathrm{g}$ ；食品 $0.2\mathrm{g}\sim 1\mathrm{g}$ ；液体饮料或流质、半流质试样 $5\mathrm{g}\sim 10\mathrm{g}$ 于 $100~\mathrm{mL}$ 锥形瓶中，加入被检验物质干重10倍的硫酸溶液。在 $121^{\circ}C$ 下，水解 $30\mathrm{min}$ 后冷却至室温。用 $0.1\mathrm{mol / L}$ 氢氧化钠溶液调 $\mathsf{pH}$ 至 $6.0\sim 6.5$ ，再用 $0.1\mathrm{mol / L}$ 盐酸调 $\mathsf{pH}$ 至 $4.5\pm 0.1$ ，用水定容至 $100~\mathrm{mL}$ ，用无灰滤纸过滤，滤液备用。

# 5.5 稀释

根据试样中烟酸含量用水对试样提取液进行适当稀释 $(f)$ ，使稀释后试样提取液中烟酸含量在 $50.0\mathrm{ng}\sim 500.0\mathrm{ng}$ 范围内。

# 5.6 测定系列管制备

# 5.6.3 灭菌

将所有的标准系列管和试样系列管测定管塞好棉塞，于 $121^{\circ}\mathrm{C}(0.10\mathrm{MPa}\sim 0.12\mathrm{MPa})$ 高压灭菌 $5\mathrm{min}$ 。灭菌完成后，迅速冷却，备用。

# 5.7 培养

5.7.1 接种：在无菌操作条件下，将接种液转入无菌滴管，向每支测定管接种一滴。  
5.7.2 培养：将加完菌液的试管置于 $36\,^{\circ}\mathrm{C} \pm 1\,^{\circ}\mathrm{C}$ 恒温培养箱中培养 $16\,\mathrm{h} \sim 24\,\mathrm{h}$ ，直至获得最大浊度，即再培养 $2\,\mathrm{h}$ 浊度无明显变化。另准备一支标准0管(含 $0.0\,\mathrm{ng}$ 烟酸)不接种作为0对照管。

# 5.8 测定

用厚度为 $1\mathrm{cm}$ 比色杯，在波长 $550~\mathrm{nm}$ 条件下读取光密度值，将培养好的测定管用涡旋混匀器混匀。以未接种0对照管调节透光率为 $100\%$ ，然后依次测定标准系列管、试样系列管的透光率。取出最高浓度标准曲线管振荡 $5\mathrm{s}$ ，测定光密度值，放回重新培养。2h后同等条件重新测该管的光密度，如果两次光密度的绝对差结果 $\leq 2\%$ ，则取出全部检验管测定标准溶液和试样的光密度。

# 5.9 标准曲线的制作

以标准系列管烟酸含量为横坐标，光密度值为纵坐标，绘制标准曲线，也可对各个标准点做拟合曲线。各个标准点3管之间的光密度值的相对标准偏差应小于 $10\%$ ，如果某一标准点3支试样管中有2支烟酸含量落在 $50~\mathrm{ng}\sim 500~\mathrm{ng}$ 范围内，且该两管之间折合为每毫升试样提取液中烟酸含量的偏差小于 $10\%$ ，则该结果可用，如果3支试样管中烟酸含量的相对标准偏差大于 $10\%$ ，则该点舍去，不参与标准曲线的绘制。

# 6 分析结果的表述

试样结果计算：从标准曲线查得试样系列管中烟酸的相应含量 $(c)$ ，按式(1)进行结果计算。

测定液浓度按式(1)计算：

$$
X = \frac {\bar {\rho} \times V _ {1} \times f}{m} \times \frac {1 0 0}{1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中烟酸含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$\rho$ ——试样系列管折合为试样提取液中烟酸浓度平均值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V_{1}$ ——试样提取液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样提取液稀释倍数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$\frac{100}{1000}$ ——折算成每百克试样中烟酸毫克数的换算系数。

结果保留两位有效数字。

# 7 精密度

普通食品在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ ；强化食品在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 8 其他

按样品种类将待测试样稀释到线性范围内再对样品进行测试。

本标准试管法线性范围 $50\mathrm{ng / mL}\sim 500\mathrm{ng / mL}$ ；天然类等含量较低的食品试样称样量为 $5\mathrm{g}$ 时，检出限为 $0.05\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.1\mathrm{mg} / 100\mathrm{g}$ ；强化食品等含量较高的食品试样称样量为 $1\mathrm{g}$ 时，检出限为 $0.25\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.5\mathrm{mg} / 100\mathrm{g}$ 。

# 第二法 高效液相色谱法

# 9 原理

高蛋白样品经沉淀蛋白质，高淀粉样品经淀粉酶酶解，在弱酸性环境下超声波振荡提取，以 $\mathrm{C_{18}}$ 色谱柱分离，在紫外检测器检测 $261\mathrm{nm}$ 波长处检测，根据色谱峰的保留时间定性，外标法定量，计算试样中烟酸和烟酰胺含量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 盐酸（HCl）：优级纯。

10.1.2 氢氧化钠 $\mathrm{(NaOH)}$ ：优级纯。  
10.1.3 高氯酸 $\mathrm{(HClO_4)}$ ：体积分数为 $60\%$ ，优级纯。  
10.1.4 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
10.1.5 异丙醇 $\left(\mathrm{C}_{3} \mathrm{H}_{8} \mathrm{O}\right)$ ：色谱纯。  
10.1.6 庚烷磺酸钠 $\left(\mathrm{C}_{7} \mathrm{H}_{15} \mathrm{NaO}_{3} \mathrm{~S}\right)$ ：色谱纯。  
10.1.7 淀粉酶：酶活力 $\geqslant 1.5\mu /\mathrm{mg}$

# 10.2 试剂配制

10.2.1 盐酸 $(5.0\mathrm{mol} / \mathrm{L})$ ：用量筒量取 $415~\mathrm{mL}$ 盐酸，于 $1000~\mathrm{mL}$ 烧杯中，加 $585~\mathrm{mL}$ 水，混匀。  
10.2.2 盐酸 $(0.1\mathrm{mol} / \mathrm{L})$ ：用移液管吸取 $8.3\mathrm{mL}$ 盐酸，于 $1000~\mathrm{mL}$ 烧杯中，加 $991.7~\mathrm{mL}$ 水，混匀。  
10.2.3 氢氧化钠 $(5.0\mathrm{mol} / \mathrm{L})$ ：称取 $200\mathrm{g}$ 氢氧化钠(3.2.2)于烧杯中加水溶解并转移至 $1000\mathrm{mL}$ 容量瓶中，加水定容，混匀。  
10.2.4 氢氧化钠 $(0.1\mathrm{mol} / \mathrm{L})$ : 称取 $4.0\mathrm{g}$ 氢氧化钠(3.2.2)于烧杯中加水溶解并转移至 $1000\mathrm{mL}$ 容量瓶中, 加水定容, 混匀。  
10.2.5 流动相：甲醇 $70~\mathrm{mL}$ 、异丙醇 $20~\mathrm{mL}$ 、庚烷磺酸钠 $1\mathrm{g}$ ，用 $910~\mathrm{mL}$ 水溶解并混匀后，用高氯酸调 $\mathrm{pH}$ 至 $2.1 \pm 0.1$ ，经 $0.45\mu \mathrm{m}$ 膜过滤。

# 10.3 烟酸和烟酰胺标准溶液

烟酸 $\left(\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{NO}_{2}\right)$ 和烟酰胺 $\left(\mathrm{C}_{6} \mathrm{H}_{6} \mathrm{~N}_{2} \mathrm{O}\right)$ ：纯度 $>99\%$ ，或经国家认证并授予标准物质证书的标准物质。

10.3.1 烟酸和烟酰胺标准储备液 $(1.000\mathrm{mg / mL})$ ：准确称取烟酸及烟酰胺标准品各 $0.05\mathrm{g}$ （精确到 $0.1\mathrm{mg}$ )，分别置于 $100~\mathrm{mL}$ 容量瓶中，用 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶解，定容至刻度，混匀( $4^{\circ}C$ 冰箱中可保存1个月）。

注：标准储备液配制完以后，需要进行浓度校正，校正方法见附录B。

10.3.2 烟酸和烟酰胺标准混合中间液 $(100.0\mu \mathrm{g} / \mathrm{mL})$ : 准确吸取烟酸和烟酰胺标准储备液 (10.3.1) 各 $10.0 \mathrm{~mL}$ 于 $100 \mathrm{~mL}$ 容量瓶中, 加水定容至刻度, 混匀。临用前配制。  
10.3.3 烟酸和烟酰胺标准混合工作液: 分别准确吸取标准混合中间液 (10.3.2) $1.0 \mathrm{~mL} 、 2.0 \mathrm{~mL}$ 、 $5.0 \mathrm{~mL} 、 10.0 \mathrm{~mL} 、 20.0 \mathrm{~mL}$ 于 $100 \mathrm{~mL}$ 容量瓶中, 加水定容至刻度, 混匀, 得到浓度分别为 $1.0 \mu \mathrm{g} / \mathrm{mL}$ 、 $2.0 \mu \mathrm{g} / \mathrm{mL} 、 5.0 \mu \mathrm{g} / \mathrm{mL} 、 10.0 \mu \mathrm{g} / \mathrm{mL} 、 20.0 \mu \mathrm{g} / \mathrm{mL}$ 的标准混合工作液。临用前配制。

# 12 分析步骤

# 12.1 试样制备与处理

非粉状固态试样粉碎并混合均匀；液态试样摇匀。

# 12.1.1 淀粉类和含淀粉的食品（即食谷物、面包、饼干、面条、小麦粉和杂粮粉等制品）

称取混合均匀固体试样约 $5.0\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ），加入约 $25\mathrm{mL}45^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 的水，称取混合均匀液体试样约 $20.0\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $150~\mathrm{mL}$ 锥形瓶中，加入约 $0.5\mathrm{g}$ 淀粉酶，摇匀后向锥形瓶中充氮，盖上塞，置于 $50^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$ 的培养箱内培养约 $30\mathrm{min}$ ，取出冷却至室温。

注：如果条件允许，建议酶解时采用 $55^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 水浴振摇。

# 12.1.2 不含淀粉的食品（调制乳、调制乳粉、饮料类、固体饮料类、豆粉和豆浆粉等制品）

称取混合均匀固体试样约 $5.0\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ），加入约 $25\mathrm{mL}45^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 的水，称取混合均匀液体试样约 $20.0\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $150\mathrm{mL}$ 锥形瓶中，置于超声波振荡器中振荡约 $10\mathrm{min}$ 以上充分溶解，静置 $5\mathrm{min}\sim 10\mathrm{min}$ ，并冷却至室温。

# 12.1.3 提取

待试样溶液降至室温后，用 $5.0\mathrm{mol / L}$ 盐酸溶液和 $0.1\mathrm{mol / L}$ 盐酸溶液调节试样溶液的 $\mathrm{pH}$ 至 $1.7\pm 0.1$ ，放置约 $2\mathrm{min}$ 后，再用 $5.0\mathrm{mol / L}$ 氢氧化钠溶液和 $0.1\mathrm{mol / L}$ 氢氧化钠溶液调节试样溶液的 $\mathrm{pH}$ 至 $4.5\pm 0.1$ ，置于 $50~^\circ \mathrm{C}$ 水浴超声波振荡器中振荡 $10\mathrm{min}$ 以上充分提取，冷却至室温后转至 $100~\mathrm{mL}$ 容量瓶中，用水反复冲洗锥形瓶，洗液合并于 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度后混匀，经滤纸过滤。滤液再经 $0.45\mu \mathrm{m}$ 微孔滤膜加压过滤，用样品瓶收集，即为试样测定液。

注：必要时，试样测定液用水进行适当的稀释 $(f)$ ，使试样测定液中烟酸和烟酰胺浓度在 $1\mu \mathrm{g / mL}\sim 20~\mu \mathrm{g / mL}$ 范围内。

# 12.2 参考液相色谱条件

参考液相色谱条件列出如下：

a）色谱柱： $\mathrm{C_{18}}$ （粒径 $5\mu \mathrm{m},250\mathrm{mm}\times 4.6\mathrm{mm})$ 或具有同等性能的色谱柱；  
b） 柱温： $25\mathrm{^\circ C}\pm 0.5\mathrm{^\circ C}$   
c）紫外检测器：检测波长为 $261~\mathrm{nm}$   
d) 流动相: 甲醇 $70 \mathrm{~mL}$ 、异丙醇 $20 \mathrm{~mL}$ 、庚烷磺酸钠 $1 \mathrm{~g}$ , 用 $910 \mathrm{~mL}$ 水溶解并混匀后, 用高氯酸调 $\mathrm{pH}$ 至 $2.1 \pm 0.1$ , 经 $0.45 \mu \mathrm{m}$ 膜过滤;  
e) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
f）进样量： $10~\mu \mathrm{L}$ 或 $20~\mu \mathrm{L}$

# 12.3 测定

# 12.3.1 标准曲线测定

按照已经确立的色谱条件，将烟酸及烟酰胺混合标准系列测定液依次按上述推荐色谱条件进行测定（标准样品色谱图见C.1）。记录各组分的色谱峰面积或峰高，以峰面积或峰高为纵坐标，以标准测定液的浓度为横坐标，绘制标准曲线。

# 12.3.2 试样溶液的测定

将试样测定液按上述推荐色谱条件进样测定。记录各组分色谱峰面积或峰高，根据标准曲线计算出试样测定液中烟酸及烟酰胺各组分的浓度 $\rho$ 。

# 13 分析结果的表述

# 13.1 试样烟酸和烟酰胺含量计算

试样中烟酸或烟酰胺的含量，按式(2)计算：

$$
X _ {1 \text {或} 2} = \frac {\rho_ {i} \times V \times 1 0 0}{m} \tag {2}
$$

式中：

$X_{1\text{或}2}$ ——试样中烟酸或烟酰胺的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ ；

$\rho$ ——试样待测液中烟酸或烟酰胺的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

注：液态试样中烟酸或烟酰胺含量也可以微克每百毫升 $(\mu \mathrm{g} / 100\mathrm{mL})$ 为单位。

# 13.2 试样中维生素PP的总含量

试样中维生素PP的总含量X，按式(3)计算：

$$
X = X _ {1} + X _ {2} \times 1. 0 0 8 \quad \dots \tag {3}
$$

式中：

$X$ ——试样中维生素PP的总含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$

$X_{1}$ ——试样中烟酸的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ ；

$X_{2}$ ——试样中烟酰胺的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ ；

1.008——烟酰胺转化成烟酸的系数。

# 13.3 结果表述

结果保留到整数。

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

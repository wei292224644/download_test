# 食品安全国家标准

# 食品添加剂 紫甘薯色素

# 1 范围

本标准适用于以番薯属植物番薯（Ipomoea batatas Lam）中紫色的块根为原料，用含柠檬酸等酸的水或乙醇水溶液，经浸提、精制而成的食品添加剂紫甘薯色素。

# 2 化学成分、分子式、结构式和相对分子质量

# 2.1 化学成分

紫甘薯色素的主要着色成分是花色苷，其中主要花色苷配基为矢车菊素和芍药素。

# 2.2 分子式

矢车菊素： $\mathrm{C_{15}H_{11}O_6}$

芍药素： $\mathrm{C_{16}H_{13}O_6}$

# 2.3 结构式

![](images/c9acefceba03e733cd5c8397387707ca0b1a516a132655fa0e2ff89d269a2871.jpg)

矢车菊素： $\mathrm{R} = \mathrm{OH}$

芍药素： $\mathrm{R = OCH_3}$

# 2.4 相对分子质量

矢车菊素：287.23（按2013年国际相对原子质量）

芍药素：301.26（按2013年国际相对原子质量）

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯的试剂和GB/T6682中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 颜色反应

称取约 $0.1\mathrm{g}$ 试样，溶于 $100~\mathrm{mL}$ $\mathrm{pH}3.0$ 柠檬酸-磷酸氢二钠缓冲液中，溶液应呈红色～紫红色；向此溶液中逐渐滴加氢氧化钠溶液 $(40~\mathrm{g / L})$ ，至试样溶液呈碱性时，应逐渐变为紫罗兰色、蓝色、绿色。

# A.2.2 最大吸收峰

取A.3色价测定中的紫甘薯色素试样液，用分光光度计检测，在波长 $530\mathrm{nm}\pm 10\mathrm{nm}$ 范围内应有一个最大吸收峰。

# A.3 色价的测定

# A.3.1 试剂和材料

A.3.1.1 磷酸氢二钠。

A.3.1.2 柠檬酸。

# A.3.2 仪器和设备

分光光度计。

# A.3.3 分析步骤

# A.3.3.1 试剂配制

A.3.3.1.1 磷酸氢二钠溶液： $0.2\mathrm{mol} / \mathrm{L}$ ，精确称取 $71.64\mathrm{g}$ 磷酸氢二钠（ $\mathrm{Na_2HPO_4\cdot 12H_2O)}$ ，用水溶解并定容至 $1000~\mathrm{mL}$ 。  
A.3.3.1.2 柠檬酸溶液: $0.1 \mathrm{~mol} / \mathrm{L}$ , 精确称取 $21.01 \mathrm{~g}$ 柠檬酸 $\left(\mathrm{C}_{6} \mathrm{H}_{8} \mathrm{O}_{7} \cdot \mathrm{H}_{2} \mathrm{O}\right)$ , 用水溶解并定容至 $1000 \mathrm{~mL}$ 。  
A.3.3.1.3 pH 3.0 柠檬酸-磷酸氢二钠缓冲溶液: 精确量取 $4.11 \mathrm{~mL}$ 磷酸氢二钠溶液与 $15.89 \mathrm{~mL}$ 柠檬酸溶液混合, 摇匀, 用酸度计进行检测, 如果 $\mathrm{pH}$ 不为 3.0 , 则用磷酸氢二钠溶液或柠檬酸溶液调整 $\mathrm{pH}$ 至 3.0 。

# A.3.3.2 测定

称取 $0.1\mathrm{g}\sim 0.2\mathrm{g}$ 试样(精确至 $0.0001\mathrm{g}$ )，用 $\mathrm{pH}3.0$ 柠檬酸-磷酸氢二钠缓冲溶液溶解，转移至 $100~\mathrm{mL}$ 容量瓶中，加 $\mathrm{pH}3.0$ 柠檬酸-磷酸氢二钠缓冲溶液定容至刻度，摇匀。取此试样溶液置于 $1\mathrm{cm}$ 比色皿中，以 $\mathrm{pH}3.0$ 柠檬酸-磷酸氢二钠缓冲溶液做空白对照，用分光光度计在 $530~\mathrm{nm}\pm 10~\mathrm{nm}$ 范围内的最大吸收波长处测定吸光度（吸光度值应控制在 $0.3\sim 0.7$ ，否则应调整试样溶液浓度，再重新测定）。

# A.3.4 结果计算

色价 $E_{1\mathrm{cm}}^{1\%}$ （530 $\mathrm{nm}\pm 10~\mathrm{nm}$ )，按式(A.1)计算：

$$
E _ {1 \mathrm {cm}} ^ {1 \%} = \frac {A}{c} \times \frac {1}{100} \quad \dots \tag{A.1}
$$

式中：

$A$ ——试样溶液的吸光度；

$c$ ——被测试样溶液浓度，单位为克每毫升 $(\mathrm{g / mL})$ ；

100——浓度换算系数（对应色价规定换算成试样浓度为 $1\mathrm{g} / 100\mathrm{mL}$ ）。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于其算术平均值的 $2.5\%$ 。

# A.4 $\mathbf{pH}(10\mathrm{g} / \mathrm{L}$ 水溶液)的测定

将试样配成 $10\mathrm{g / L}$ 水溶液，摇匀，用酸度计测定。

# A.5 总花色苷（以矢车菊-3-葡萄糖苷计）的测定

# A.5.1 试剂和材料

A.5.1.1 氯化钾。

A.5.1.2 盐酸。

A.5.1.3 乙酸钠。

# A.5.2 仪器和设备

分光光度计。

# A.5.3 分析步骤

# A.5.3.1 试剂配制

A.5.3.1.1 pH 1.0 氯化钾缓冲溶液: $0.025 \mathrm{~mol} / \mathrm{L}$ , 精确称取 $1.86 \mathrm{~g} \mathrm{KCl}$ 置烧杯中, 加入约 $980 \mathrm{~mL}$ 水。测量 $\mathrm{pH}$ , 用 $\mathrm{HCl}$ (约 $6.3 \mathrm{~mL}$ ) 调 $\mathrm{pH}$ 至 $1.0$ (± 0.05), 转移并用水定容至 $1000 \mathrm{~mL}$ 。

A.5.3.1.2 pH 4.5 乙酸钠缓冲溶液: $0.4 \mathrm{~mol} / \mathrm{L}$ , 精确称取 $54.43 \mathrm{~g} \mathrm{CH}_{3} \mathrm{CO}_{2} \mathrm{Na} \cdot 3 \mathrm{H}_{2} \mathrm{O}$ 置烧杯中, 加入约 $960 \mathrm{~mL}$ 水。测量 $\mathrm{pH}$ , 用 $\mathrm{HCl}$ (约 $20 \mathrm{~mL}$ ) 调 $\mathrm{pH}$ 至 $4.5 (\pm 0.05)$ , 转移并用水定容至 $1000 \mathrm{~mL}$ 。

# A.5.3.2 测试液制备

准确称取一定量的试样(精确至 $0.0001\mathrm{g}$ )，用水溶解后，稀释定容至 $50~\mathrm{mL}$ ，此为待测试样的储备

液。精确吸取两份等量的储备液,分别用 $\mathrm{pH} 1.0$ 的缓冲溶液和 $\mathrm{pH} 4.5$ 的缓冲溶液稀释定容至 $50 \mathrm{~mL}$ ,此为试样液。储备液的最大取样量应不超过 $10 \mathrm{~mL}$ ,以保证不超出缓冲溶液的缓冲能力。

用 $\mathrm{pH}1.0$ 的缓冲溶液对储备液进行适当稀释，直到 $520\mathrm{nm}$ 波长下的吸光度值在分光光度计的线性范围内( $0.3\sim 0.7)$ 。采用这个稀释倍数，制备两份试样液，一份用 $\mathrm{pH}1.0$ 的缓冲溶液稀释，另一份用 $\mathrm{pH}4.5$ 的缓冲溶液稀释。

# A.5.3.3 测定

用 $1\mathrm{cm}$ 比色皿，以水为空白对照，在 $520~\mathrm{nm}$ 和 $700~\mathrm{nm}$ 波长下，测定分别经过 $\mathrm{pH}1.0$ 和 $\mathrm{pH}4.5$ 的缓冲溶液稀释的试样的吸光度，试样制备完毕后要求在 $20\mathrm{min}\sim 50\mathrm{min}$ 的时间以内测定吸光度。

注：在 $700\mathrm{nm}$ 波长下测定吸光度值是为了校正混浊对测定结果的干扰。如果稀释后的试样过于混浊，测定前采用离心或过滤的方法进行澄清处理，采用的过滤器不能吸附花色苷。

# A.5.4 结果计算

总花色苷(以矢车菊-3-葡萄糖苷计)的质量分数 $\omega$ ，按式(A.2)计算：

$$
w = \frac {A \times M \times f}{\varepsilon \times l \times 1000} \times \frac {V}{m} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

A $(A_{520\mathrm{nm}} - A_{700\mathrm{nm}})_{\mathrm{pH}1.0} - (A_{520\mathrm{nm}} - A_{700\mathrm{nm}})_{\mathrm{pH}4.5}$

$M$ ——矢车菊-3-葡萄糖苷的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ （ $M = 449.2~\mathrm{g / mol}$ ）；

$f$ ——试样储备液的稀释倍数，按A.5.3.2中方法计算；

$\varepsilon$ ——矢车菊-3-葡萄糖苷的摩尔消光系数，单位为升每摩尔厘米 $[\mathrm{L} / (\mathrm{mol}\cdot \mathrm{cm})](\varepsilon = 26900)$ ；

$l$ ——光路长，单位为厘米(cm)；

1000——体积换算系数；

$V$ ——储备液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于其算术平均值的 $5\%$ 。

注：紫甘薯中的花色苷种类不同于矢车菊-3-葡萄糖苷，因此最大吸收波长、相对分子质量和摩尔消光系数均有区别，本测定结果只是一个以矢车菊-3-葡萄糖苷计的总花色苷含量，不代表紫甘薯色素中实际的总花色苷含量。

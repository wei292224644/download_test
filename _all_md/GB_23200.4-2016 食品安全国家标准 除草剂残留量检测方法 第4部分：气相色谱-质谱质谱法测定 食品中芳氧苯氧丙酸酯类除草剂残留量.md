# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T1737.4—2010《除草剂残留量检验方法第4部分：气相色谱一质谱/质谱法测定进出口食品中芳氧苯氧丙酸酯类》。与SN/T1737.4—2010相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“进出口食品”改为“食品”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T1737.4—2010。

# 食品安全国家标准除草剂残留量检测方法

# 第4部分：气相色谱一质谱/质谱法测定食品中芳氧苯氧丙酸酯类除草剂残留量

# 1 范围

本标准规定了食品中2,4-滴丁酯、氟吡禾灵、吡氟禾草灵、炔草酯、禾草灵、氰氟草酯、噁唑禾草灵、精喹禾灵等芳氧苯氧丙酸酯类除草剂残留量的气相色谱一质谱/质谱检测方法。

本标准适用于大豆、大麦茶、粳米、胡萝卜、菠菜、青刀豆、蒜苗、草莓、蜂蜜、猪肉、鱼、禽蛋中2,4-滴丁酯、氟吡禾灵、吡氟禾草灵、炔草酯、禾草灵、氰氟草酯、噁唑禾草灵、精喹禾灵残留量的测定和确证，其他食品可参照执行。

# 2 规范性引用文件

# 3 原理

试样中残留的芳氧苯氧丙酸酯类除草剂经正己烷饱和过的乙腈(含 $1 \%$ 冰醋酸)提取, 基质分散固相萃取净化, 用气相色谱一质谱/质谱仪测定, 外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
4.1.2 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
4.1.3 丙酮 $(\mathrm{C_3H_6O})$ ：色谱纯。  
4.1.4 冰醋酸 $\mathrm{(C_2H_4O_2)}$   
4.1.5 无水乙酸钠 $\mathrm{(C_2H_3ONa)}$   
4.1.6 无水硫酸镁 $(\mathrm{MgSO_4})$

# 4.2 溶液配制

提取溶剂(含 $1 \%$ 冰醋酸的经正己烷饱和过的乙腈溶液): 加 $10 \mathrm{~mL}$ 冰醋酸到 $990 \mathrm{~mL}$ 的乙腈 (事先用正己烷饱和过)。

# 4.3 标准品

芳氧苯氧丙酸酯类除草剂标准品：2，4-滴丁酯(2，4-D butylate，CAS号：94-80-4，分子式： $\mathrm{C_{12}H_{14}C_{12}O_3}$ ，纯度 $\geqslant 97.1\%$ )；氟吡禾灵(Haloxyfop，CAS号：69806-34-4，分子式： $\mathrm{C_{15}H_{11}ClF_3NO_4}$ ，纯度 $\geqslant 99\%$ )；吡氟禾草灵(Fluazifop，CAS号：69335-91-7，分子式： $\mathrm{C_{15}H_{12}F_3NO_4}$ ，纯度 $\geqslant 99\%$ )；炔草酯

(Clodinafop-propargyl, CAS号: 105512-06-9, 分子式: $\mathrm{C_{17}H_{13}ClFNO_4}$ , 纯度 $\geqslant 99.0\%$ ); 禾草灵(Diclofop, CAS号: 51338-27-3, 分子式: $\mathrm{C_{16}H_{14}Cl_2O_4}$ , 纯度 $\geqslant 96.5\%$ ); 氰氟草酯(Cyhalofop-butyl, CAS号: 122008-85-9, 分子式: $\mathrm{C_{20}H_{20}FNO_4}$ , 纯度 $\geqslant 99.0\%$ ); 嗽唑禾草灵(Fenoxaprop, CAS号: 95617-09-7, 分子式: $\mathrm{C_{16}H_{12}ClNO_5}$ , 纯度 $\geqslant 99\%$ ); 精喹禾灵(Quizalofop-P-ethyl, CAS号: 100646-51-3, 分子式: $\mathrm{C_{19}H_{17}ClN_2O_4}$ , 纯度 $\geqslant 99.5\%$ )。

# 4.4 标准溶液配制

4.4.1芳氧苯氧丙酸酯类除草剂标准储备溶液：准确称取适量芳氧苯氧丙酸酯类除草剂的标准品，经丙酮溶解、稀释和定容后，分别配制成 $100\mu \mathrm{g / mL}$ 的单种标准储备溶液，此溶液可在 $0^{\circ}C\sim 4^{\circ}C$ 条件下避光储存6个月。  
4.4.2 混合标准中间溶液: 准确吸取每种芳氧苯氧丙酸酯类的标准储备溶液各 $5.0 \mathrm{~mL}$ 于 $50 \mathrm{~mL}$ 棕色容量瓶中, 用丙酮稀释至刻度线后配制成浓度为 $10.0 \mu \mathrm{g} / \mathrm{mL}$ 的混合标准中间溶液, 此溶液可在 $0^{\circ} \mathrm{C} \sim$ $4^{\circ} \mathrm{C}$ 条件下避光储存 3 个月。  
4.4.3 混合标准工作溶液：根据需要再用丙酮稀释成适用浓度的混合标准工作溶液，现用现配。

# 4.5 材料

4.5.1 乙二胺-N-丙基甲硅烷(PSA)填料： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}$   
4.5.2 石墨化炭黑填料： $120\mu \mathrm{m}\sim 400\mu \mathrm{m}_{\circ}$   
4.5.3 十八烷基硅烷(ODS)填料： $60\mu \mathrm{m}\sim 100\mu \mathrm{m}$   
4.5.4 滤膜： $0.45\mu \mathrm{m}$ ，有机相。

# 5 仪器和设备

5.1 气相色谱—质谱/质谱联用仪。  
5.2 振荡器。  
5.3 电子天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.4 涡旋振荡器。  
5.5 组织捣碎机。  
5.6 粉碎机。  
5.7 旋转蒸发仪。  
5.8 氮气吹干仪。  
5.9 具塞锥形瓶： $250~\mathrm{mL}$   
5.10 试管： $10~\mathrm{mL}$   
5.11 浓缩瓶： $150~\mathrm{mL}$

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按GB2763的规定执行。在取样和制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 胡萝卜、青刀豆、蒜苗、草莓、菠菜

取有代表性样品 $500\mathrm{g}$ ，切碎后，用组织捣碎机将样品加工成浆状，装入洁净容器，密封并标明标记。

# 6.2.2 大豆、大麦茶、粳米

取有代表性样品 $500\mathrm{g}$ ，切碎后，用组织捣碎机将样品加工成浆状，装入洁净容器，密封并标明标记。

# 6.2.3 猪肉、鱼肉、禽蛋

取有代表性样品 $500\mathrm{g}$ ，切碎后，用组织捣碎机将样品充分捣碎均匀，装入洁净容器，密封并标明标记。

# 6.2.4 蜂蜜

取有代表性样品 $500\mathrm{g}$ ，对无结晶的蜂蜜样品将其搅拌均匀；对有结晶析出的蜂蜜样品，在密闭情况下，将样品瓶置于不超过 $60^{\circ}\mathrm{C}$ 的水浴中温热，振荡，待样品全部融化后搅匀，迅速冷却至室温，在融化时应注意防止水分挥发。装入洁净容器，密封并标明标记。

# 6.3 试样保存

大豆、大麦茶、粳米、蜂蜜和禽蛋等试样于 $4^{\circ}\mathrm{C}$ 以下保存；胡萝卜、青刀豆、蒜苗、草莓、菠菜、猪肉、鱼肉、龙虾等试样于 $-18^{\circ}\mathrm{C}$ 以下保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 胡萝卜、青刀豆、蒜苗、草莓和菠菜

称取 $10\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )于 $250\mathrm{mL}$ 具塞锥形瓶中，加入 $15\mathrm{g}$ 无水硫酸镁、 $6\mathrm{g}$ 无水乙酸钠和 $50\mathrm{mL}$ 提取溶剂，于振荡器上振荡 $30\mathrm{min}$ ，静置 $10\mathrm{min}$ ，过滤于 $150\mathrm{mL}$ 浓缩瓶中。残渣再加入 $20\mathrm{mL}$ 提取溶剂提取一次，合并两次滤液， $40^{\circ}\mathrm{C}$ 下旋转浓缩至干。用 $2\mathrm{mL}$ 乙腈溶解残渣，待净化。

# 7.1.2 大豆、粳米、猪肉、鱼肉和禽蛋

称取 $5\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )于 $250\mathrm{mL}$ 具塞锥形瓶中，加入 $15\mathrm{g}$ 无水硫酸镁、 $6\mathrm{g}$ 无水乙酸钠和 $50\mathrm{mL}$ 提取溶剂，于振荡器上振荡 $30\mathrm{min}$ （猪肉、鱼肉、蜂蜜、禽蛋等肉制品应均质提取 $5\mathrm{min}$ ），静置 $10\mathrm{min}$ ，过滤于 $150\mathrm{mL}$ 浓缩瓶中。残渣再加入 $20\mathrm{mL}$ 提取溶剂提取一次，合并两次滤液， $40^{\circ}\mathrm{C}$ 下旋转浓缩至干。用 $2\mathrm{mL}$ 乙腈溶解残渣，待净化。

# 7.1.3 大麦茶

称取 $5\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ ) 于 $250\mathrm{mL}$ 具塞锥形瓶中，加入 $5\mathrm{mL}$ 水浸泡过夜后，再加入 $15\mathrm{g}$ 无水硫酸镁、 $6\mathrm{g}$ 无水乙酸钠和 $50\mathrm{mL}$ 提取溶剂，于振荡器上振荡 $30\mathrm{min}$ ，静置 $10\mathrm{min}$ ，过滤于 $150\mathrm{mL}$ 浓缩瓶中。残渣再加入 $20\mathrm{mL}$ 提取溶剂提取一次，合并两次滤液， $40^{\circ}\mathrm{C}$ 下旋转浓缩至干。用 $2\mathrm{mL}$ 乙腈溶解残渣，待净化。

# 7.1.4 蜂蜜

称取 $5\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ ) 于 $250\mathrm{mL}$ 具塞锥形瓶中，加入 $5\mathrm{mL}$ 水溶解后再加入 $15\mathrm{g}$ 无水硫酸镁、 $6\mathrm{g}$ 无水乙酸钠和 $50\mathrm{mL}$ 提取溶剂，于振荡器上振荡 $30\mathrm{min}$ ，静置 $10\mathrm{min}$ ，过滤于 $150\mathrm{mL}$ 浓缩瓶中。残渣再加入 $20\mathrm{mL}$ 提取溶剂提取一次，合并两次滤液， $40^{\circ}\mathrm{C}$ 下旋转浓缩至干。用 $2\mathrm{mL}$ 乙腈溶解残渣，待净化。

# 7.2 净化

# 7.2.1 胡萝卜、青刀豆、蒜苗、草莓、菠菜和大麦茶提取液

将7.1相应样品提取液转移到事先装有 $200\mathrm{mg}$ PSA填料和 $250\mathrm{mg}$ 石墨化炭黑填料的小试管中，充分涡旋 $1\mathrm{min}$ ，待色素完全消除后，过滤膜，供气相色谱一质谱测定和确证。

# 7.2.2 大豆、粳米、猪肉、鱼肉、蜂蜜和禽蛋提取液

将7.1相应样品提取液转移到事先装有 $200\mathrm{mg}$ PSA填料， $150\mathrm{mg}$ 石墨化炭黑填料和 $100\mathrm{mg}$ $\mathrm{C}_{18}$ 填料的小试管中，充分涡旋 $1\mathrm{min}$ ，过滤膜，供气相色谱一质谱测定和确证。

# 7.3 测定

# 7.3.1 气相色谱一质谱参考条件

a）色谱柱：DB-5ms弹性石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；

b）柱温：初始温度 $50^{\circ}\mathrm{C}$ （保持 $2\mathrm{min}$ ），以 $30^{\circ}\mathrm{C / min}$ 升至 $180^{\circ}\mathrm{C}$ ，再以 $5^{\circ}\mathrm{C / min}$ 升至 $280^{\circ}\mathrm{C}$ （保持 $10\mathrm{min})$   
c）进样口温度： $250^{\circ}\mathrm{C}$   
d）色谱—质谱接口温度： $280^{\circ}\mathrm{C}$   
e）载气：氦气，纯度 $\geq 99.999\%$ ，载气流速 $1\mathrm{mL / min}$   
f）进样量： $1\mu \mathrm{L}$   
g）进样方式：不分流进样， $1.2\mathrm{min}$ 后开阀；  
h）离子源：电子轰击离子源（EI源）；  
i）离子源温度： $230^{\circ}\mathrm{C}$   
j） 电子能量： $70\mathrm{eV}$   
k）溶剂延迟时间： $9.0\mathrm{min}$

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中芳氧苯氧丙酸酯类残留含量。

$$
X _ {i} = \frac {A _ {i} \times C _ {i} \times V}{A s _ {i} \times m} \tag {1}
$$

式中：

$X_{i}$ ——试样中芳氧苯氧丙酸酯类残留量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A_{i}$ ——样液中芳氧苯氧丙酸酯类的峰面积；

$C_i$ ——标准工作液中芳氧苯氧丙酸酯类的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A s_{i}$ ——标准工作液中芳氧苯氧丙酸酯类的峰面积；

$m$ ——最终样液所代表的试样质量，单位为克(g)。

计算结果需将空白值扣除，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录C的要求。

9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 D 的要求。

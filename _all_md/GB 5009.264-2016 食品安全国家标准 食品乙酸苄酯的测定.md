# 前言

本标准代替GB/T21914—2008《茶饮料中乙酸苄酯的测定 气相色谱法》。

本标准与GB/T21914—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中乙酸苄酯的测定”。

——增加了定量限。

# 食品安全国家标准

# 食品中乙酸苄酯的测定

# 1 范围

本标准规定了茶饮料中乙酸苄酯的气相色谱测定方法。

本标准适用于茶饮料中乙酸苄酯的测定。

# 2 原理

茶饮料中乙酸苄酯经提取定容后，在气相色谱柱中与内标物正十二烷及其他组分分离，用氢火焰离子化检测器检测，以内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ 。  
3.1.2 无水乙醇 $\mathrm{(C_2H_6O)}$ 。

# 3.2 标准品

3.2.1 乙酸苄酯 $\mathrm{(C_9H_{10}O_2)}$ ：含量 $\geqslant 98\%$   
3.2.2 正十二烷 $\left[\mathrm{CH}_{3}\left(\mathrm{CH}_{2}\right)_{10} \mathrm{CH}_{3}\right]$ ：含量 $\geqslant 98 \%$ 。

# 3.3 标准溶液配制

3.3.1 乙酸苄酯标准储备液 $(1000\mu \mathrm{g / mL})$ ：准确称取 $100\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )乙酸苄酯，用无水乙醇溶解并定容至 $100~\mathrm{mL}$ 。  
3.3.2 乙酸苄酯使用液 $(80\mu \mathrm{g / mL})$ ：准确吸取 $1000~\mu \mathrm{g / mL}$ 的乙酸苄酯溶液 $8.00~\mathrm{mL}$ ，用二氯甲烷定容至 $100~\mathrm{mL}$ 。临用时现配。  
3.3.3 正十二烷内标液 $(200\mu \mathrm{g / mL})$ : 准确称取 $20\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ) 正十二烷, 用二氯甲烷定容至 $100\mathrm{mL}$ 。  
3.3.4 混合标液配制： $5\mathrm{mL}$ 比色管中加入 $80~\mu \mathrm{g / mL}$ 乙酸苄酯 $4\mathrm{mL}$ 和 $200~\mu \mathrm{g / mL}$ 的正十二烷 $1\mathrm{mL}$ ，混匀，用于计算相对校正因子。

# 4 仪器和设备

4.1 气相色谱仪：配有氢火焰离子化检测器。   
4.2 氮气吹干仪。

4.3 分析天平：感量为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$

# 5 分析步骤

# 5.1 试样前处理

准确称取 $40\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )茶饮料置于 $125~\mathrm{mL}$ 分液漏斗中，加入 $20~\mathrm{mL}$ 二氯甲烷，振摇 $1\mathrm{min}$ 静置分层，二氯甲烷层缓慢滴入漏斗(底部少许脱脂棉，上面 $10\mathrm{g}$ 无水硫酸钠)脱水过滤，收集于 $50~\mathrm{mL}$ 刻度试管中，然后再向分液漏斗中加入 $20~\mathrm{mL}$ 二氯甲烷，振摇 $1\mathrm{min}$ ，静置分层，同上过滤收集，之后用少量二氯甲烷冲洗漏斗3遍，并收集。收集液用氮气吹干仪 $(30^{\circ}C)$ 浓缩至 $3\mathrm{mL}$ 左右，用少量二氯甲烷将其完全转移至 $5\mathrm{mL}$ 比色管中，加入 $1\mathrm{mL}$ 的 $200~\mu \mathrm{g / mL}$ 正十二烷后用二氯甲烷定容至 $5\mathrm{mL}$ ，混匀待测。

# 5.2 气相色谱参考条件

5.2.1 色谱柱：SE-30柱，柱长 $30\mathrm{m}$ ，内径 $0.53\mathrm{mm}$ ，膜厚 $1.0\mu \mathrm{m}$ ，或性能相当者。  
5.2.2 柱温：初始温度 $90^{\circ}\mathrm{C}$ ，恒温 $1\mathrm{min}$ ，程序升温，升温速率 $10^{\circ}\mathrm{C / min}$ ，最终温度为 $180^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ 。  
5.2.3 检测器温度： $250^{\circ}C$ 。  
5.2.4 汽化室温度： $230^{\circ}C$   
5.2.5 氮气压力： $41.4\mathrm{kPa}(6.0\mathrm{psi})$   
5.2.6 氢气流速： $45~\mathrm{mL / min}$   
5.2.7 空气流速： $450~\mathrm{mL / min}$   
5.2.8 进样量： $1\mu \mathrm{L}$

# 5.3 测定

将混合标液和制备好的试样溶液注入气相色谱仪中，得到相应的峰面积。

# 6 分析结果的表述

# 6.1 相对校正因子的计算

乙酸苄酯对正十二烷的相对校正因子 $f$ 按式(1)计算：

$$
f = \frac {m _ {1} \times A _ {2}}{m _ {2} \times A _ {1}} \tag {1}
$$

式中：

$f$ ——乙酸苄酯对正十二烷的相对校正因子；

$m_{1}$ ——混合标液中乙酸苄酯的质量，单位为微克 $(\mu \mathrm{g})$ ；

$A_{2}$ ——混合标液中正十二烷的峰面积；

$m_{2}$ ——混合标液中正十二烷的质量，单位为微克 $(\mu \mathrm{g})$ ；

$A_{1}$ ——混合标液中乙酸苄酯的峰面积。

# 6.2 试样中乙酸苄酯的含量

试样中乙酸苄酯的含量按式(2)计算：

$$
X = f \times \frac {m _ {2} \times A _ {1} \times 1 0 0 0}{m _ {1} \times A _ {2} \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中乙酸苄酯的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$f$ ——乙酸苄酯对正十二烷的相对校正因子；

$m_{2}$ ——试样溶液中正十二烷的质量，单位为微克 $(\mu \mathrm{g})$ ；

$A_{1}$ ——试样溶液中乙酸苄酯的峰面积；

1000——换算系数；

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$A_{2}$ ——试样溶液中正十二烷的峰面积。

计算结果保留到小数点后两位。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法的检出限为 $0.005\mathrm{mg / kg}$ ，定量限为 $0.015\mathrm{mg / kg}$ 。

# 前言

本标准代替GB1902—2005《食品添加剂苯甲酸钠》。

本标准与GB1902—2005相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 苯甲酸钠”；  
——删除了砷项目；  
——增加了邻苯二甲酸项目；  
——修改了氯化物指标；  
——增加了重金属的检验方法。

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 盐酸溶液： $1 + 2$   
A.2.1.2 三氯化铁溶液： $100\mathrm{g / L}$

# A.2.2 苯甲酸根的鉴别

在试样溶液 $(100\mathrm{g / L})$ 中加1滴三氯化铁溶液，生成赭色沉淀，再加盐酸溶液酸化，析出白色沉淀。

# A.2.3 钠元素的鉴别

称取约 $0.2\mathrm{g}$ 试样，溶于 $10~\mathrm{mL}$ 水中，用铂丝蘸取盐酸溶液在无色火焰上燃烧至无色，再蘸取试验溶液少许，在无色火焰上燃烧，火焰应呈鲜黄色。

# A.3 苯甲酸钠（以干基计）含量的测定

# A.3.1 方法提要

盐酸与苯甲酸钠起中和反应，用乙醚萃取反应生成的苯甲酸，根据盐酸标准滴定溶液的用量计算苯甲酸钠的含量。

# A.3.2 试剂和材料

A.3.2.1 乙醚。  
A.3.2.2 盐酸标准滴定溶液： $c(\mathrm{HCl}) = 0.5\mathrm{mol / L}$   
A.3.2.3 溴酚蓝指示液： $0.4\mathrm{g / L}$

# A.3.3 分析步骤

称取 $1.5\mathrm{g}$ A.11.1中干燥物A，精确至 $0.0001\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 锥形瓶中，加 $25~\mathrm{mL}$ 水溶解，再加 $50~\mathrm{mL}$ 乙醚和10滴溴酚蓝指示液，用盐酸标准滴定溶液滴定，边滴边将水层和乙醚层充分摇匀，当水层显示淡绿色时为终点。

# A.3.4 结果计算

苯甲酸钠（以干基计）的质量分数 $w_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {V \times c \times M}{1000 \times m} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V$ ——试样消耗盐酸标准滴定溶液(A.3.2.2)的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——盐酸标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——苯甲酸钠的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{C_7H_5O_2Na}) = 144.1]$

$m$ ——干燥物A的质量，单位为克 $(\mathrm{g})$ 。

取两次平行测定结果的算术平均值为报告结果。两次平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.4 溶液的澄清度试验

# A.4.1 试剂和材料

A.4.1.1 硝酸溶液： $1 + 23$

A.4.1.2 糊精溶液： $20~\mathrm{g / L}$

A.4.1.3 硝酸银溶液： $17\mathrm{g / L}$

A.4.1.4 盐酸标准滴定溶液： $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$

A.4.1.5 浊度标准溶液: $c(\mathrm{Cl}) = 0.01 \mathrm{mg} / \mathrm{mL}$ 。量取盐酸标准滴定溶液 $14.10 \mathrm{~mL}$ , 置于 $50 \mathrm{~mL}$ 容量瓶中, 稀释至刻度。量取该溶液 $10.0 \mathrm{~mL}$ 于 $1000 \mathrm{~mL}$ 容量瓶中, 加水稀释至刻度。

# A.4.2 分析步骤

称取 $4.5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，用水溶解并移入 $25~\mathrm{mL}$ 比色管中，加水至溶液体积为 $22~\mathrm{mL}$ ，作为试验溶液；取另一 $25~\mathrm{mL}$ 比色管，准确加入 $0.20~\mathrm{mL}$ 浊度标准溶液，加水至 $20~\mathrm{mL}$ ，加 $1\mathrm{mL}$ 硝酸溶液， $0.2~\mathrm{mL}$ 糊精溶液， $1\mathrm{mL}$ 硝酸银溶液，摇匀，避光放置 $15\mathrm{min}$ ，作为标准比浊溶液。

目视轴向观察，试验溶液的浊度不得大于标准比浊溶液的浊度。

# A.5 易氧化物试验

# A.5.1 方法提要

试样中易氧化物与高锰酸钾反应，反应完全后，试液呈高锰酸钾溶液的粉红色时为反应终点，以消耗高锰酸钾标准滴定溶液的体积数控制易氧化物的量。

# A.5.2 试剂和材料

A.5.2.1 硫酸。

A.5.2.2 高锰酸钾标准滴定溶液： $c(\frac{1}{5}\mathrm{KMnO}_4) = 0.1\mathrm{mol / L}$

# A.5.3 分析步骤

称取 $1.0\mathrm{g}$ 试样，精确至 $0.001\mathrm{g}$ 。在 $100\mathrm{mL}$ 水中加入 $1.5\mathrm{mL}$ 硫酸，边煮沸边滴加高锰酸钾标准滴定溶液至粉红色，保持 $30\mathrm{s}$ 不褪色，趁热加入已称试样，在溶液约 $70^{\circ}\mathrm{C}$ 时，用高锰酸钾标准滴定溶液滴定至粉红色，保持 $15\mathrm{s}$ 不褪色，所用高锰酸钾标准滴定溶液不应超过 $0.5\mathrm{mL}$ 。

# A.6 酸碱度的测定

# A.6.1 试剂和材料

A.6.1.1 盐酸标准滴定溶液： $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$   
A.6.1.2 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.1\mathrm{mol / L}$   
A.6.1.3 酚酞指示液： $10\mathrm{g / L}$

# A.6.2 分析步骤

称取 $1.0\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，加 $20\mathrm{mL}$ 无二氧化碳的水溶解，加2滴酚酞指示液，若试验溶液无色，加 $0.20\mathrm{mL}$ 氢氧化钠标准滴定溶液，试验溶液应呈淡红色；若试验溶液呈淡红色，加 $0.20\mathrm{mL}$ 盐酸标准滴定溶液，淡红色应消失。

# A.7 重金属（以 $\mathbf{Pb}$ 计）的测定

# A.7.1 试剂和材料

A.7.1.1 盐酸溶液： $234\rightarrow 1000$ 。本液含HCl应为 $9.5\% \sim 10.5\%$   
A.7.1.2 硫代乙酰胺溶液：取硫代乙酰胺 $4\mathrm{g}$ ，加水使溶解，定容至 $100\mathrm{mL}$ ，置于冰箱中保存。临用前取混合液（由 $40\mathrm{g/L}$ 氢氧化钠溶液 $15.0\mathrm{mL}$ 、水 $5.0\mathrm{mL}$ 和甘油 $20.0\mathrm{mL}$ 组成） $5.0\mathrm{mL}$ ，加硫代乙酰胺溶液 $1.0\mathrm{mL}$ ，置水浴上加热 $20\mathrm{s}$ ，冷却，立即使用。  
A.7.1.3 pH3.5的乙酸盐缓冲液：按GB5009.74配制。   
A.7.1.4 铅标准使用液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：按GB5009.74配制。

# A.7.2 仪器和设备

纳氏比色管： $25~\mathrm{mL}$

# A.7.3 分析步骤

A.7.3.1 A管：取纳氏比色管，加标准铅溶液 $1.0\mathrm{mL}$ 与 $\mathrm{pH}3.5$ 的乙酸盐缓冲液 $2\mathrm{mL}$ 后，加水稀释成 $25~\mathrm{mL}$ 。  
A.7.3.2 B管：取试样 $2.0\mathrm{g}$ 于烧杯中，加水 $45~\mathrm{mL}$ ，不断搅拌，滴加盐酸溶液 $5\mathrm{mL}$ ，过滤，分取滤液 $25~\mathrm{mL}$ 于纳氏比色管中。  
A.7.3.3 C管：取 $1.0\mathrm{g}$ 试样于纳氏比色管中，加水适量使溶解，再加 $1.0\mathrm{mL}$ 铅标准使用液 $(10~\mu \mathrm{g / mL})$ 与 $\mathrm{pH}3.5$ 的乙酸盐缓冲液 $2\mathrm{mL}$ 后，用水稀释成 $25~\mathrm{mL}$ 。  
A.7.3.4 再在 A、B、C 三管中分别加硫代乙酰胺试液 $2 \mathrm{~mL}$ , 摇匀, 放置 $2 \mathrm{~min}$ , 在白色背景下自上而下纵向观察, C 管的色度不浅于 A 管的色度, B 管的色度不深于 A 管的色度, 则可判定试样中重金属 (以 $\mathrm{Pb}$ 计) 含量低于 $10 \mathrm{mg} / \mathrm{kg}$ 。

# A.8 硫酸盐（以 $\mathrm{SO}_4$ 计）的测定

# A.8.1 试剂和材料

A.8.1.1 盐酸。  
A.8.1.2 氯化钡溶液： $50~\mathrm{g / L}$

A.8.1.3 硫酸盐 $\mathrm{(SO_4)}$ 标准溶液： $0.1\mathrm{mg / mL}$

# A.8.2 分析步骤

称取 $1.0\mathrm{g}$ 试样，精确至 $0.001\mathrm{g}$ ，置于烧杯中，加 $40~\mathrm{mL}$ 水溶解，加 $1.5\mathrm{mL}$ 盐酸混匀，过滤于 $50~\mathrm{mL}$ 比色管中，用水洗涤沉淀，使滤液达到 $45~\mathrm{mL}$ ，作为试验溶液。

取 $10.0\mathrm{mL}$ 硫酸盐 $\left(\mathrm{SO}_4\right)$ 标准溶液于另一支 $50~\mathrm{mL}$ 比色管中，加 $1.5\mathrm{mL}$ 盐酸，加水至 $45~\mathrm{mL}$ ，作为标准比浊溶液。

同时向试验溶液和标准比浊溶液中各加 $5\mathrm{mL}$ 氯化钡溶液，放置 $10\mathrm{min}$ ，试验溶液的浊度不应深于标准比浊溶液的浊度。

# A.9 氯化物（以Cl计）的测定

# A.9.1 方法提要

试样中含有有机氯(芳香族氯化物)和无机氯化合物，通过加入碳酸钙，在高温下灼烧，使有机氯转化为氯化钙，与无机氯一起溶入试样溶液。在酸性条件下，试样溶液中的氯离子与硝酸银溶液反应生成氯化银沉淀，其产生的浊度应不大于标准比浊溶液的浊度。

# A.9.2 试剂和材料

A.9.2.1 碳酸钙。  
A.9.2.2 硝酸溶液： $1 + 9$   
A.9.2.3 硝酸银溶液： $17\mathrm{g / L}$   
A.9.2.4 氯化物(Cl)标准溶液： $0.1\mathrm{mg / mL}$

# A.9.3 分析步骤

称取 $0.5\mathrm{g}$ 试样，精确至 $0.001\mathrm{g}$ ，置于 $30~\mathrm{mL}$ 瓷坩埚中，加 $2\mathrm{mL}$ 硝酸溶液充分混合，于 $100^{\circ}\mathrm{C}$ 干燥至无明显湿迹，加 $0.8\mathrm{g}$ 碳酸钙，用少量水润湿， $100^{\circ}\mathrm{C}$ 干燥后，电炉上低温炭化，再于 $600^{\circ}\mathrm{C}$ 的高温炉中灼烧 $10\mathrm{min}$ ，冷却后，用 $20~\mathrm{mL}$ 硝酸溶液溶解残渣，过滤于 $50~\mathrm{mL}$ 比色管中，用 $15~\mathrm{mL}$ 水洗涤瓷坩埚，将洗液并入滤液，加水至刻度，作为试验溶液。

另称取 $0.8\mathrm{g}$ 碳酸钙，精确至 $0.001\mathrm{g}$ ，加 $22.5\mathrm{mL}$ 硝酸溶液，当有不溶物时过滤，转入 $50\mathrm{mL}$ 比色管中，加 $1.00\mathrm{mL}$ 氯化物(Cl)标准溶液，用水稀释至刻度，作为标准比浊溶液。

同时在两溶液中各加 $0.5\mathrm{mL}$ 硝酸银溶液，充分摇匀，放置 $5\mathrm{min}$ ，试验溶液的浊度不得大于标准比浊溶液的浊度。

# A.10 邻苯二甲酸的测定

# A.10.1 方法提要

将试样溶解于甲醇和乙酸溶液的混合溶液中，用邻苯二甲酸作为外标物用液相色谱定量测定苯甲酸钠中邻苯二甲酸的含量。

# A.10.2 试剂和材料

A.10.2.1 甲醇。  
A.10.2.2 乙酸溶液： $1\rightarrow 100$

A.10.2.3 甲醇-乙酸溶液的混合液： $2 + 3$

A.10.2.4 邻苯二甲酸标准品：质量分数 $\geq 99.5\%$

A.10.2.5 邻苯二甲酸标准储备液: $100 \mu \mathrm{g} / \mathrm{mL}$ 。称取 $10 \mathrm{mg}$ 邻苯二甲酸标准品, 溶解于 $30 \mathrm{~mL}$ 甲醇后, 加乙酸溶液定容到 $100 \mathrm{~mL}$ 。

A.10.2.6 邻苯二甲酸标准工作液： $1.0\mu \mathrm{g / mL}$ 。取 $1.0~\mathrm{mL}$ 邻苯二甲酸标准储备液(A.10.2.5)，用甲醇-乙酸溶液的混合液定容到 $100~\mathrm{mL}$

# A.10.3 仪器和设备

液相色谱仪：配备紫外检测器。

# A.10.5 分析步骤

称取 $1\mathrm{g}$ 试样，精确到 $0.0001\mathrm{g}$ ，溶解于流动相中，并定容至 $50~\mathrm{mL}$ 作为试样测试液。分别取 $20~\mu \mathrm{L}$ 邻苯二甲酸标准工作液和试样测试液进样。

# A.10.6 结果计算

邻苯二甲酸含量的质量分数 $w_{2}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.2)计算：

$$
w _ {2} = \frac {c _ {\mathrm {S T}} \times A _ {\mathrm {S}} \times V}{A _ {\mathrm {S T}} \times m} \tag {A.2}
$$

式中：

$c_{\mathrm{ST}}$ ——邻苯二甲酸标准溶液的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$A_{\mathrm{S}}$ ——试样测试液的峰面积；

$V$ ——试样测试液的体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{\mathrm{ST}}$ ——邻苯二甲酸标准溶液的峰面积；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

取两次平行测定结果的算术平均值为测定结果，平行测定结果的相对平均偏差不得超过 $20\%$ 。

# A.11 干燥减量的测定

# A.11.1 分析步骤

称取约 $1.5\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于预先在 $105~^\circ \mathrm{C}\sim 110~^\circ \mathrm{C}$ 已恒量的称量瓶中，使试样厚度

均匀，于 $105\mathrm{^\circ C}\sim 110\mathrm{^\circ C}$ 干燥至恒量。保留干燥物A用于苯甲酸钠含量的测定。

# A.11.2 结果计算

干燥减量的质量分数 $\varpi_{3}$ ，按式(A.3)计算：

$$
w _ {3} = \frac {m - m _ {1}}{m} \times 100 \% \tag{A.3}
$$

式中：

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——干燥后试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果表示至小数点后2位。

取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.2\%$ 。

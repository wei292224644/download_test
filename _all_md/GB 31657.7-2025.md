# 1 范围

本文件规定了蜂产品中拟除虫菊酯类药物残留量检测的制样和气相色谱-串联质谱方法。

本文件适用于蜂蜜和蜂王浆中氟氯氰菊酯、氯氟氰菊酯、氯氰菊酯、溴氰菊酯、氰戊菊酯、氟胺氰菊酯残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

蜂蜜和蜂王浆中的残留药物用乙腈提取，正已烷萃取后用气相色谱-串联质谱检测，外标法定量。

# 5 试剂和材料

以下所用试剂，除特殊注明外均为分析纯试剂，求为符合GB/T6682中规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_2CN)}$ ：色谱纯

5.1.2 正己烷 $(\mathrm{C_6H_{11}})$ ：色谱纯。

5.1.3 氯化钠(NaCl)

5.1.4 无水硫酸镁 $(\mathrm{MgSO_4})$ 。

# 5.2 标准品及其纯度

氟氯氰菊酯、氯氟氰菊酯、氯氰菊酯、溴氰菊酯、氰戊菊酯和氟胺氰菊酯，含量均 $\geqslant 98\%$ ，具体信息见附录A。

# 5.3 标准溶液配制

5.3.1 拟除虫菊酯类药物单一标准储备液：分别准确称取6种拟除虫菊酯类药物标准品适量，用正己烷溶解并定容至 $10~\mathrm{mL}$ ，配制成 $100~\mathrm{mg / L}$ 的标准储备液。于 $-18^{\circ}C$ 以下避光可保存3个月。

5.3.2 拟除虫菊酯类药物混合标准工作液：分别准确吸取6种拟除虫菊酯类药物标准储备液 $100\mu \mathrm{L}$ ，于 $10~\mathrm{mL}$ 容量瓶中，用正己烷定容至刻度，配制成浓度为 $1.0\mathrm{mg / L}$ 的拟除虫菊酯类药物混合标准工作液。于一 $18^{\circ}C$ 以下避光可保存1个月。

# 5.4 材料

有机系滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 气相色谱-串联质谱仪：配电子轰击电离源。

6.2 涡旋振荡仪。  
6.3 高速冷冻离心机：转速 $8000\mathrm{r / min}$ 或以上。  
6.4 氮吹仪。  
6.5 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试蜂产品，将其搅拌均匀。对有结晶的蜂蜜样品。在密闭情况下，置于不超过 $60^{\circ}\mathrm{C}$ 的水浴中解晶，待样品全部融化后搅匀，冷却至室温。制备好的试样置于样品瓶中，密封，并做上标记。

a）取均质后的供试样品，作为供试试样；  
b) 取均质后的空白样品, 作为空白试样。

# 7.2 试样的保存

蜂蜜样品于室温下避光保存；蜂王浆样品于 $-18^{\circ} \mathrm{C}$ 下保存。

# 8 测定步骤

# 8.1 提取

称取试料 $(5.0 \pm 0.05) \mathrm{g}$ 于 $50 \mathrm{~mL}$ 具塞塑料离心管中，加入水 $5 \mathrm{~mL}$ ，振荡 $1 \mathrm{~min}$ ，加入乙腈 $10 \mathrm{~mL}$ ，振荡 $5 \mathrm{~min}$ ，再依次加入水 $10 \mathrm{~mL}$ 和氯化钠 $8 \mathrm{~g}$ ，振荡 $5 \mathrm{~min}$ ，于 $4^{\circ} \mathrm{C}$ 下 $8000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，取上层溶液于另一塑料离心管中，残渣加入乙腈 $5 \mathrm{~mL}$ 重复提取1次，合并全部上层溶液，备用。

# 8.2 净化

备用液于 $40^{\circ} \mathrm{C}$ 下氮吹至干，依次加入水 $1 \mathrm{~mL}$ 和正己烷 $1 \mathrm{~mL}$ ，拧紧离心管盖，超声 $2 \mathrm{~min}$ ，振荡 $1 \mathrm{~min}$ ，于 $4^{\circ} \mathrm{C}$ 下 $8000 \mathrm{r} / \mathrm{min}$ 离心 $2 \mathrm{~min}$ ，取上层溶液加入 $0.4 \mathrm{~g}$ 无水硫酸镁，涡旋混匀，静置，上清液转至进样瓶中待测。

# 8.3 基质匹配标准曲线的制备

精密量取混合标准工作液适量，分别加入经提取和净化的蜂蜜或蜂王浆空白试样残渣中，用正己烷配制成浓度为 $10\mu \mathrm{g} / \mathrm{L}, 25\mu \mathrm{g} / \mathrm{L}, 50\mu \mathrm{g} / \mathrm{L}, 100\mu \mathrm{g} / \mathrm{L}, 250\mu \mathrm{g} / \mathrm{L}, 500\mu \mathrm{g} / \mathrm{L}$ 的基质匹配系列混合标准溶液，供气相色谱串联质谱测定。以待测物定量离子的色谱峰面积为纵坐标、标准溶液浓度为横坐标，绘制基质匹配标准曲线，求回归方程和相关系数。

# 8.4 测定

# 8.4.1 气相色谱参考条件

a) 色谱柱: $5\%$ 苯基-甲基聚硅氧烷型非极性色谱柱 $(30 \mathrm{~m} \times 0.25 \mathrm{~mm}, 0.25 \mu \mathrm{m})$ 或相当者;   
b）升温程序： $200^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，以 $10^{\circ}\mathrm{C / min}$ 升温至 $310^{\circ}\mathrm{C}$ ，保持 $6.5\mathrm{min}$ ；  
c) 进样口温度: $310^{\circ} \mathrm{C}$ ;   
d) 进样量: $1 \mu \mathrm{L}$ ;   
e) 进样方式: 不分流进样;  
f) 溶剂延迟时间：5 min；  
g）载气：高纯氦气；  
h）柱流量： $1.5\mathrm{mL / min}$   
i) 流量控制方式：恒线速度；  
j) 连接线温度: $310^{\circ} \mathrm{C}$ 。

# 8.4.2 质谱参考条件

a）检测方式：多反应监测(MRM)；  
b）离子源温度： $200^{\circ}\mathrm{C}$

c） 电离方式：EI；  
d) 碰撞气：高纯氦气；  
e）选择离子参数设定见表1。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在相同试验条件下，试样溶液中拟除虫菊酯类药物的保留时间与基质匹配标准溶液中对应的保留时间的相对偏差在 $\pm 0.5\%$ 之内；且样品中各组分定性离子的相对丰度与浓度接近的基质匹配标准溶液中对应的定性离子的相对丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.4.3.2 定量测定

分别取适量试样溶液和相应浓度的基质匹配标准工作液, 作单点校准或多点校准, 以色谱峰峰面积积分值之和定量。基质匹配标准工作液及试料液中药物的响应值均应在仪器检测的线性范围内, 试样液进样过程中应穿插标准工作液, 以便准确定量。对于试料中拟除虫菊酯类药物残留量超过仪器测定线性范围的, 应进行适当倍数稀释后再进行分析。各拟除虫菊酯类药物混合标准溶液的特征离子质量色谱图见附录 B。

# 8.5 空白试验

取空白试样，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中拟除虫菊酯类药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {s}} \times A \times V \times 1 0 0 0}{A _ {\mathrm {s}} \times m \times 1 0 0 0} \dots \tag {1}
$$

式中：

$X$ ——试样中相应的拟除虫菊酯类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{s}}$ ——基质匹配标准溶液中相应的拟除虫菊酯类药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

A ——试样中相应的拟除虫菊酯类药物峰面积；

$V$ ——溶解残留物的正己烷体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——基质匹配标准溶液中相应的拟除虫菊酯类药物峰面积；

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

注：有异构体的药物须计算各异构体的峰面积之和。

计算结果以平行测定结果的算术平均值表示，保留3位有效数字。

# 10 检测方法的灵敏度、正确度和精密度

# 10.1 灵敏度

本方法蜂蜜和蜂王浆中氟氯氰菊酯、氯氟氰菊酯、氯氰菊酯、溴氰菊酯、氰戊菊酯、氟胺氰菊酯检出限为 $2.5\mu \mathrm{g / kg}$ ，定量限为 $5\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法在 $5\mu \mathrm{g} / \mathrm{kg}\sim 25\mu \mathrm{g} / \mathrm{kg}$ 添加浓度的回收率为 $70\% \sim 110\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 20\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

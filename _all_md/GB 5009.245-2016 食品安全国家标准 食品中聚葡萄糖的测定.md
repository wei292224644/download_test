# 食品安全国家标准

# 食品中聚葡萄糖的测定

# 1 范围

本标准规定了食品中聚葡萄糖的测定方法。

本标准适用于食品中添加的聚葡萄糖的测定。

# 2 术语和定义

下列术语和定义适用于本文件。

# 2.1

聚葡萄糖 $\left[\left(\mathrm{C}_{6} \mathrm{H}_{10} \mathrm{O}_{5}\right)_{n}\right]$

由葡萄糖、山梨糖醇和柠檬酸(或磷酸)按一定比例混合，在高温下加热聚合并精制、干燥而成的多聚体，平均聚合度12。属于可溶性膳食纤维。

# 3 原理

食品中聚葡萄糖经热水浸提、超滤离心后，滤液经酶解去除淀粉、果聚糖等干扰物后，再用离子色谱-脉冲安培检测器定量测定聚葡萄糖含量。

# 4 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 氢氧化钠溶液 $(50\%)$ ：色谱纯，离子色谱专用。  
4.1.2 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
4.1.3 三水合乙酸钠 $\mathrm{(CH_3COONa\cdot 3H_2O)}$   
4.1.4 无水乙酸钠 $\mathrm{(CH_3COONa)}$ 。  
4.1.5 果聚糖酶(fructanase)：≥2000U/mL。   
4.1.6 淀粉葡糖苷酶(Amyloglucosidase)：≥3260 U/mL(可溶性淀粉)；≥200 U/mL(对硝基酚-β-麦芽糖苷，p-NP-β maltoside)。  
4.1.7 异淀粉酶(Isoamylase)：≥1000U/mL。

# 4.2 试剂配制

4.2.1 乙酸溶液 $(0.2\mathrm{mol} / \mathrm{L})$ ：准确吸取 $1.2\mathrm{mL}$ 冰乙酸，用水稀释至 $100~\mathrm{mL}$ 。  
4.2.2 乙酸钠溶液 $(0.2\mathrm{mol} / \mathrm{L})$ ：准确称取 $2.72\mathrm{g}$ 三水合乙酸钠，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
4.2.3 乙酸盐缓冲液 $(\mathrm{pH}$ 为4.5）：将 $28~\mathrm{mL}$ 乙酸溶液 $(0.2\mathrm{mol} / \mathrm{L})$ 与 $22~\mathrm{mL}$ 乙酸钠溶液 $(0.2\mathrm{mol} / \mathrm{L})$ 混

合，用水稀释至 $100~\mathrm{mL}$

4.2.4 混合酶液：分别吸取 $680~\mu \mathrm{L}$ 果聚糖酶、 $84~\mu \mathrm{L}$ 淀粉葡糖苷酶、 $17~\mu \mathrm{L}$ 异淀粉酶混合，加乙酸盐缓冲液稀释至 $20~\mathrm{mL}$ 。混合酶液临用现配。

# 4.3 流动相

4.3.1 流动相A（含 $0.15\mathrm{mol} / \mathrm{L}$ 氢氧化钠）：准确吸取 $15.7\mathrm{mL}$ 氢氧化钠溶液 $(50\%)$ ，用预先脱气的水稀释 $2\mathrm{L}$ ，惰性气体保护。  
4.3.2 流动相B(含 $0.15\mathrm{mol} / \mathrm{L}$ 氢氧化钠， $0.50\mathrm{mol} / \mathrm{L}$ 乙酸钠)：准确称量 $41\mathrm{g}$ 无水乙酸钠（或 $68\mathrm{g}$ 三水合乙酸钠)，用流动相A溶解定容至 $1\mathrm{L}$ ，混匀，过 $0.45\mu \mathrm{m}$ 膜，脱气。

注：配制流动相时，去离子水需预先脱气 $0.5\mathrm{h}$ ，定容后的流动相混匀时倒置混匀 $5\sim 6$ 次，不可剧烈振摇。配好的流动相沿储液瓶壁缓缓倒入，脱气 $0.5\mathrm{h}$ 后惰性气体保护。

# 4.4 参考物质

聚葡萄糖 $\left[\left(\mathrm{C}_{6} \mathrm{H}_{10} \mathrm{O}_{5}\right)_{n}\right]$ ：纯度大于 $90\%$ ，CAS：68424-04-4。

注：为保证结果的准确性，如能确定添加的聚葡萄糖来源，应选用食品中添加的聚葡聚糖同源的参考物质，并达到FCC级。

# 4.5 参考物质溶液的配制

4.5.1 参考物质储备液 $(2.00\mathrm{mg / g})$ : 准确称取 $0.20\mathrm{g}$ 聚葡萄糖参考物质 (精确至 $0.0001\mathrm{g}$ ) 至已预先称重的 $100\mathrm{mL}$ 具螺口塞的试样瓶中 (精确至 $0.001\mathrm{g}$ )。加入约 $100\mathrm{g}$ 预热至 $80^{\circ}\mathrm{C}$ 的去离子水, 拧紧螺口塞, 涡旋振荡 $30\mathrm{~s}$ , 将试样瓶置于 $80^{\circ}\mathrm{C}$ 水浴 $10\mathrm{~min}$ , 每隔 $5\mathrm{~min}$ 振荡一次, 以使聚葡萄糖充分溶解, 取出试样瓶, 冷却至室温, 称重 (精确至 $0.001\mathrm{g}$ )。参考物质储备液于 $4^{\circ}\mathrm{C}$ 保存, 有效期 1 个月。  
4.5.2 参考物质中间液：分别精确称量 $10.0\mathrm{g},7.50\mathrm{g},5.00\mathrm{g},3.75\mathrm{g},2.50\mathrm{g},1.50\mathrm{g}$ 参考物质储备液（精确至 $0.0001\mathrm{g}$ )加水至 $20.0\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ，得到浓度为 $1.000\mathrm{mg / g},0.750\mathrm{mg / g},0.500\mathrm{mg / g},$ $0.375\mathrm{mg / g},0.250\mathrm{mg / g},0.150\mathrm{mg / g}$ 的参考物质中间液。  
4.5.3 参考物质工作液：取 $2\mathrm{mL}$ 离心管，称量(精确至 $0.0001\mathrm{g}$ )，精确称量参考物质中间液 $0.20\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )转移至已称重的离心管中，加入酶混合液 $0.8\mathrm{mL}$ ，称量（精确至 $0.0001\mathrm{g}$ )，振荡混合均匀，于 $50~^\circ \mathrm{C}$ 水浴 $60\mathrm{min}$ ，沸水浴 $10\mathrm{min}$ ，取出，冰浴 $5\mathrm{min}$ ，于 $10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。上清液过 $0.22\mu \mathrm{m}$ 滤膜，进样分析，得到浓度分别为 $200~\mu \mathrm{g / g}, 150~\mu \mathrm{g / g}, 100~\mu \mathrm{g / g}, 75.0~\mu \mathrm{g / g}, 50.0~\mu \mathrm{g / g},$ $30.0~\mu \mathrm{g / g}$ 的参考物质工作液系列，进样分析。以此6点参考物质工作液上机获得工作曲线，其回归方程的决定系数 $(R^2$ 值)应不小于0.995。

# 5 仪器与设备

5.1 分析天平：感量分别为 $1\mathrm{mg},0.1\mathrm{mg}$   
5.2 恒温水浴箱：控温精度 $\pm 1^{\circ}C$   
5.3 高速离心机：转速 $\geqslant 10000\mathrm{r / min}$   
5.4 涡旋振荡器。  
5.5 微量可调移液器： $20\mu \mathrm{L} \sim 200\mu \mathrm{L}, 200\mu \mathrm{L} \sim 1000\mu \mathrm{L}$ 。  
5.6 针筒式过滤器：孔径 $0.22\mu \mathrm{m}$   
5.7 超滤离心设备：100000Da分子截留，聚醚砜膜， $0.5\mathrm{mL}$ 容积。  
5.8 磁力搅拌器：带搅拌子。  
5.9 离子色谱仪：配备脉冲安培检测器、梯度泵。

# 6 操作步骤

# 6.1 试样处理

6.1.1 固体试样：研磨、粉碎，分析前密闭保存，避免水分变化。  
6.1.2 液体试样：测定前需混合摇匀。

# 6.2 聚葡萄糖提取

6.2.1 固体试样：取 $100\mathrm{mL}$ 具塞试样瓶，加入1粒磁力搅拌子，盖上螺口塞，称量(精确至 $0.001\mathrm{g}$ )，记为 $m_{1}$ 。准确称取一定量试样（约含聚葡萄糖 $0.05\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，记为 $m_{2}$ ）至试样瓶中，加 $100\mathrm{mL}$ 预热至 $80^{\circ}\mathrm{C}$ 的水，磁力搅拌 $30\mathrm{s}, 80^{\circ}\mathrm{C}$ 水浴 $10\mathrm{min}$ ，每 $5\mathrm{min}$ 磁力搅拌 $30\mathrm{s}$ 。取出试样瓶冷却至室温后称量（精确至 $0.001\mathrm{g}$ ），记为 $m_{3}$ 。  
6.2.2 液体试样（饮料、液态奶等）：取 $100\mathrm{mL}$ 具塞试样瓶，称量(精确至 $0.001\mathrm{g}$ )，记为 $m_{1}$ 。准确吸取一定量试样（约含聚葡萄糖 $0.05\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，记为 $m_{2}$ ）至试样瓶中，加 $100\mathrm{mL}$ 水，振荡混合均匀，称量（精确至 $0.001\mathrm{g}$ ），记为 $m_{3}$ 。  
6.2.3 适量吸取样液至 $2.0\mathrm{mL}$ 的离心管中， $10000\mathrm{r / min}$ 离心 $15\mathrm{min}$ ，取上清液过 $0.22\mu \mathrm{m}$ 滤膜，取 $0.45\mathrm{mL}$ 滤液加入 $1.5\mathrm{mL}$ 带有分子截留膜的离心管中，于 $10000\mathrm{r / min}$ 超滤离心 $30\mathrm{min}$ 。注意观察是否离心完全及离心液的澄清度，如截留膜上方留有样液或离心液混浊，可加大转速、延长离心时间，或增加稀释倍数 $(F)$ 。

# 6.3 酶解

6.3.1 取 $2\mathrm{mL}$ 离心管，称量（精确至 $0.0001\mathrm{g}$ ），记为 $m_{4}$ 。  
6.3.2 吸取 $0.2 \mathrm{~mL}$ 超滤离心液加入离心管中，称量（精确至 $0.0001 \mathrm{~g}$ ），记为 $m_{5}$ ；加入酶混合液 $0.8 \mathrm{~mL}$ ，称量（精确至 $0.0001 \mathrm{~g}$ ），记为 $m_{6}$ ；振荡混合均匀，于 $50^{\circ} \mathrm{C}$ 水浴 $60 \mathrm{~min}$ ，沸水浴 $10 \mathrm{~min}$ ，取出，冰浴 $5 \mathrm{~min}$ ，于 $10000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ 。上清液过 $0.2 \mu \mathrm{m}$ 滤膜，进样分析。上清液需在 $72 \mathrm{~h}$ 内检测。

# 6.4 色谱参考条件

检测条件见附录B。

# 6.5 分析结果的表述

试样中聚葡萄糖含量按式(1)计算：

$$
X = \rho \times F \times c \times \frac {m _ {6} - m _ {4}}{m _ {5} - m _ {4}} \times \frac {m _ {3} - m _ {1}}{m _ {2}} \times \frac {1 0 0}{1 0 ^ {6}} \dots \tag {1}
$$

式中：

$X$ ——试样中聚葡萄糖的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$\rho$ ——由标准曲线计算得到的聚葡萄糖的浓度，单位为微克每克 $(\mu \mathrm{g} / \mathrm{g})$ ；

$F$ ——稀释倍数；

$c$ ——聚葡萄糖参考物质的纯度， $\%$ ；

$m_{6}$ ——离心管、酶解样液、混合酶液的总质量，单位为克 $(\mathrm{g})$ ；

$m_{4}$ ——空离心管质量，单位为克 $(\mathrm{g})$

$m_{5}$ ——离心管、用于酶解样液总质量，单位为克 $(\mathrm{g})$ ；

$m_{3}$ —加水后试样瓶总质量，单位为克 $(\mathrm{g})$

$m_{1}$ ——试样瓶总质量，单位为克 $(\mathrm{g})$

$m_{2}$ ——试样质量，单位为克 $(\mathrm{g})$ ；

100 ——将结果表示为百分含量的系数；

$10^{6}$ ——微克 $(\mu \mathrm{g})$ 与克 $(\mathrm{g})$ 的转换系数。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

最小称样量为 $5\mathrm{g}$ 时，固体试样检出限为 $0.2\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.5\mathrm{mg} / 100\mathrm{g}$ ；液体试样为检出限为 $0.15\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.5\mathrm{mg} / 100\mathrm{g}$ 。

# 酶活性测定

# A.1 果聚糖酶酶活力测定方法

# A.1.1 原理

在 $\mathrm{pH}4.5$ ，温度 $40^{\circ}\mathrm{C}$ ，底物(蔗果三糖)浓度为 $10\mathrm{mmol / L}$ 的标准条件下， $1\mathrm{min}$ 释放 $1\mu \mathrm{mol}$ 果糖所需的酶量为 $1\textrm{U}$ 。

# A.1.2 试剂和溶剂

A.1.2.1 乙酸 $(0.2\mathrm{mol} / \mathrm{L})$ ：准确吸取 $12~\mathrm{mL}$ 冰乙酸，用水稀释至 $1000~\mathrm{mL}$

A.1.2.2 乙酸钠溶液 $(0.2\mathrm{mol} / \mathrm{L})$ ：准确称取 $27.2\mathrm{g}$ 三水合乙酸钠，用水溶解并稀释至 $1000~\mathrm{mL}$ 。

A.1.2.3 乙酸盐缓冲液 $(\mathrm{pH}$ 为4.5): 将 $280 \mathrm{~mL}$ 乙酸溶液 $(0.2 \mathrm{~mol} / \mathrm{L})$ 与 $220 \mathrm{~mL}$ 乙酸钠溶液 $(0.2 \mathrm{~mol} / \mathrm{L})$ 混合, 用水稀释至 $1000 \mathrm{~mL}$ 。

A.1.2.4 氢氧化钠溶液 $(2\mathrm{mol} / \mathrm{L})$ ：准确称取 $80\mathrm{g}$ 氢氧化钠，用水溶解并稀释至 $1000~\mathrm{mL}$ 。

A.1.2.5 3,5-二硝基水杨酸试剂(DNS)：将 $6.3\mathrm{g}3,5-$ 二硝基水杨酸试剂和 $262~\mathrm{mL}~2~\mathrm{mol} / \mathrm{L}$ NaOH溶液，加到 $500~\mathrm{mL}$ 含有 $185\mathrm{g}$ 酒石酸钾钠的热水溶液中，再加 $5\mathrm{g}$ 苯酚和 $5\mathrm{g}$ 亚硫酸钠，搅拌溶解，冷却后加蒸馏水定容至 $1000~\mathrm{mL}$ ，贮于棕色瓶中备用。

A.1.2.6 果糖标准液 $(1\mathrm{mg / mL})$ : 准确称取 $80^{\circ}\mathrm{C}$ 烘至恒重的分析纯果糖 $100\mathrm{mg}$ , 置于小烧杯中, 加少量乙酸盐缓冲液溶解后, 转移到 $100\mathrm{mL}$ 容量瓶中, 用乙酸盐缓冲液定容至 $100\mathrm{mL}$ , 混匀, $4^{\circ}\mathrm{C}$ 冰箱中保存备用。

A.1.2.7 蔗果三糖溶液 $(10\mathrm{mmol / L})$ ：精确称取 $5.04\mathrm{g}$ 蔗果三糖，用乙酸盐缓冲液溶解并稀释至 $1000~\mathrm{mL}$ 。

# A.1.3 仪器

A.1.3.1 分析天平。

A.1.3.2 电热恒温水浴锅。

A.1.3.3 可见光分光光度计。

A.1.3.4 计时器。

# A.1.4 分析步骤

# A.1.4.1 果糖标准曲线的绘制

吸取乙酸盐缓冲液 $4.0\mathrm{mL}$ 至 $25\mathrm{mL}$ 刻度试管中，加入DNS试剂 $5.0\mathrm{mL}$ ，沸水浴加入 $5\mathrm{min}$ ，冷却至室温，用水定容至 $25\mathrm{mL}$ ，制成标准空白样。

分别吸取果糖标准液 $1.00\mathrm{mL},2.00\mathrm{mL},3.00\mathrm{mL},4.00\mathrm{mL},5.00\mathrm{mL},6.00\mathrm{mL},7.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用乙酸盐缓冲液定容至 $100~\mathrm{mL}$ ，配制成浓度 $0.10~\mathrm{mg / mL},0.20~\mathrm{mg / mL},0.30~\mathrm{mg / mL}$ 、 $0.40~\mathrm{mg / mL},0.50~\mathrm{mg / mL},0.60~\mathrm{mg / mL},0.70~\mathrm{mg / mL}$ 果糖标准使用液，吸取果糖标准使用液各 $2.0~\mathrm{mL}$ ，分别加入到 $25~\mathrm{mL}$ 刻度试管中，再分别加入 $2.0~\mathrm{mL}$ 乙酸盐缓冲液和 $5.0~\mathrm{mL}$ DNS试剂。将各管摇匀，在沸水浴中准确加热 $5\mathrm{min}$ ，取出，冷却至室温，加水定容至 $25~\mathrm{mL}$ ，加塞后颠倒混匀，放置 $30\mathrm{min}$ ，以标准空白液为对照调零，在 $540~\mathrm{nm}$ 处测定吸光度值。以吸光度值为纵坐标，果糖含量 $(\mathrm{mg})$

为横坐标，绘制标准曲线。

# A.1.4.2 待测酶溶液的配制

称取测试酶样品，用乙酸盐缓冲液进行稀释、定容（稀释后的酶液中果聚糖酶活力在 $0.04\mathrm{U / mL}\sim$ 0.08 U/mL之间）。

# A.1.4.3 测定步骤

吸取 $10.0\mathrm{mL}$ 蔗果三糖溶液， $40^{\circ}\mathrm{C}$ 平衡 $20\mathrm{min}$ 。

吸取 $10.0\mathrm{mL}$ 待测酶液， $40^{\circ}\mathrm{C}$ 平衡 $20\mathrm{min}$ 。

吸取 $2.00\mathrm{mL}$ 平衡后待测酶液，加入到 $25~\mathrm{mL}$ 刻度试管中，再加入 $5.0~\mathrm{mL}$ DNS试剂，混匀，然后加入 $2.0~\mathrm{mL}$ 平衡后的蔗果三糖溶液， $40^{\circ}\mathrm{C}$ 平衡 $30\mathrm{min}$ ，沸水浴加热 $5\mathrm{min}$ ，冷却至室温，加水定容至 $25~\mathrm{mL}$ ，加塞后颠倒混匀，放置 $30\mathrm{min}$ ，以标准空白液为对照调零，在 $540~\mathrm{nm}$ 处测定吸光度值 $A_{\mathrm{B}}$ 。

吸取 $2.0\mathrm{mL}$ 平衡后的待测酶液，加入到 $25~\mathrm{mL}$ 刻度试管中，加入 $2.0\mathrm{mL}$ 平衡后的蔗果三糖溶液，混匀 $40^{\circ}C$ 平衡 $30\mathrm{min}$ ，然后再加入 $5.0\mathrm{mL}$ DNS试剂，混匀以终止反应，沸水浴加热 $5\mathrm{min}$ ，冷却至室温，用水定容至 $25~\mathrm{mL}$ ，加塞后颠倒混匀，放置 $30\mathrm{min}$ ，以标准空白液为对照调零，在 $540~\mathrm{nm}$ 处测定吸光度值 $A_{\mathrm{E}}$ 。

# A.1.5 酶活力计算

果聚糖酶的酶活力按式(A.1)计算：

$$
X _ {\mathrm {D}} = \frac {\left[ \left(A _ {\mathrm {E}} - A _ {\mathrm {B}}\right) \times K + C _ {0} \right]}{M \times t} \times 1 0 0 0 \tag {A.1}
$$

式中：

$X_{\mathrm{D}}$ ——待测酶液的果聚糖酶活力，单位为活力单位每毫升 $(\mathrm{U} / \mathrm{mL})$ ；

$A_{\mathrm{E}}$ ——酶反应液的吸光度；

$A_{\mathrm{B}}$ ——酶空白液的吸光度；

$K$ ——标准曲线的斜率；

$C_0$ ——标准曲线的截距；

$M$ ——果糖的摩尔质量， $M(\mathrm{C_6H_{12}O_6}) = 180.2\mathrm{g / mol}$

$t$ ——酶解反应的时间，单位为分（min）；

1000——换算系数。

# A.2 异淀粉酶酶活力测定方法

# A.2.1 原理

在 $\mathrm{pH}4.0$ ，温度 $40^{\circ}\mathrm{C}$ ，底物(牡蛎糖，oysterglycogen)浓度为 $10\mathrm{mg / mL}$ 的标准条件下， $1\mathrm{min}$ 释放出 $1\mu \mathrm{mol}$ 还原糖当量(以葡萄糖计)所需要的酶量为 $1\textrm{U}$ 。

# A.2.2 试剂和溶剂

A.2.2.1 乙酸盐缓冲液 $(\mathrm{pH}$ 为4.0): 称取 $54.4 \mathrm{~g}$ 三水和乙酸钠 $\left(\mathrm{CH}_{3} \mathrm{COONa} \cdot 3 \mathrm{H}_{2} \mathrm{O}\right)$ , 溶于水, 加 $92 \mathrm{~mL}$ 乙酸(冰醋酸), 稀释至 $1000 \mathrm{~mL}$ 。  
A.2.2.2 氢氧化钠溶液 $(2\mathrm{mol} / \mathrm{L})$ ：准确称取 $80\mathrm{g}$ 氢氧化钠，用水溶解并稀释至 $1000~\mathrm{mL}$ 。  
A.2.2.3 3,5-二硝基水杨酸试剂(DNS)：将 $6.3\mathrm{g}3,5-$ 二硝基水杨酸试剂和 $262~\mathrm{mL}$ 氢氧化钠溶液，加到 $500~\mathrm{mL}$ 含有 $185\mathrm{g}$ 酒石酸钾钠的热水溶液中，再加 $5\mathrm{g}$ 苯酚和 $5\mathrm{g}$ 亚硫酸钠，搅拌溶解，冷却后加蒸

馏水定容至 $1000~\mathrm{mL}$ ，贮于棕色瓶中备用。

A.2.2.4 葡萄糖标准液 $(1\mathrm{mg / mL})$ ：称取无水葡萄糖 $100~\mathrm{mg}$ ，加乙酸盐缓冲液溶解，定容至 $100~\mathrm{mL}$ 。  
A.2.2.5 牡蛎糖溶液 $(10\mathrm{mg / mL})$ ：精确称取 $1.0\mathrm{g}$ 牡蛎糖，用乙酸盐缓冲液溶解并稀释至 $100~\mathrm{mL}$ 。

# A.2.3 仪器

A.2.3.1 分析天平。  
A.2.3.2 电热恒温水浴锅。  
A.2.3.3 分光光度计。  
A.2.3.4 计时器。

# A.2.4 分析步骤

# A.2.4.1 葡萄糖标准曲线的绘制

吸取乙酸盐缓冲液 $4.0\mathrm{mL}$ 至 $25\mathrm{mL}$ 刻度试管中，加入DNS试剂 $5.0\mathrm{mL}$ ，沸水浴加入 $5\mathrm{min}$ ，冷却至室温，用水定容至 $25\mathrm{mL}$ ，制成标准空白样。

分别吸取葡萄糖标准液 $1.00\mathrm{mL},2.00\mathrm{mL},3.00\mathrm{mL},4.00\mathrm{mL},5.00\mathrm{mL},6.00\mathrm{mL},7.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用乙酸盐缓冲液定容至 $100~\mathrm{mL}$ ，配制成浓度 $0.10\mathrm{mg / mL},0.20\mathrm{mg / mL},0.30\mathrm{mg / mL},$ $0.40\mathrm{mg / mL},0.50\mathrm{mg / mL},0.60\mathrm{mg / mL},0.70\mathrm{mg / mL}$ 葡萄糖标准使用液，吸取葡萄糖标准使用液各 $2.0~\mathrm{mL}$ ，分别加入到 $25~\mathrm{mL}$ 刻度试管中，再分别加入 $2.0~\mathrm{mL}$ 乙酸盐缓冲液和 $5.0~\mathrm{mL}$ DNS试剂。将各管摇匀，在沸水浴中准确加热 $5\mathrm{min}$ ，取出，冷却至室温，加水定容至 $25~\mathrm{mL}$ ，加塞后颠倒混匀，放置 $30\mathrm{min}$ ，以标准空白液为对照调零，在 $540~\mathrm{nm}$ 处测定吸光度值。以吸光度值为纵坐标，葡萄糖含量(mg)为横坐标，绘制标准曲线。

# A.2.4.2 待测酶溶液的配制

称取测试酶样品，用乙酸盐缓冲液进行稀释、定容（稀释后的酶液中果聚糖酶活力在 $0.04\mathrm{U / mL}\sim$ 0.08 U/mL之间）。

# A.2.4.3 测定步骤

吸取 $10.0\mathrm{mL}$ 牡蛎糖溶液， $40^{\circ}\mathrm{C}$ 平衡 $20\mathrm{min}$ 。

吸取 $10.0\mathrm{mL}$ 待测酶液， $40^{\circ}\mathrm{C}$ 平衡 $20\mathrm{min}$ 。

吸取 $2.0\mathrm{mL}$ 平衡后待测酶液，加入到 $25~\mathrm{mL}$ 刻度试管中，再加入 $5.0\mathrm{mL}$ DNS试剂，混匀，然后加入 $2.0\mathrm{mL}$ 平衡后的牡蛎糖溶液， $40^{\circ}\mathrm{C}$ 平衡 $30\mathrm{min}$ ，沸水浴加热 $5\mathrm{min}$ ，冷却至室温，加水定容至 $25~\mathrm{mL}$ ，加塞后颠倒混匀，放置 $30\mathrm{min}$ ，以标准空白液为对照调零，在 $540~\mathrm{nm}$ 处测定吸光度值 $A_{\mathrm{B}}$ 。

吸取 $2.0\mathrm{mL}$ 平衡后的待测酶液，加入到 $25~\mathrm{mL}$ 刻度试管中，加入 $2.0\mathrm{mL}$ 平衡后的牡蛎糖溶液，混匀 $40^{\circ}C$ 平衡 $30\mathrm{min}$ ，然后再加入 $5.0\mathrm{mL}$ DNS试剂，混匀以终止反应，沸水浴加热 $5\mathrm{min}$ ，冷却至室温，加水定容至 $25~\mathrm{mL}$ ，加塞后颠倒混匀，放置 $30\mathrm{min}$ ，以标准空白液为对照调零，在 $540~\mathrm{nm}$ 处测定吸光度值 $A_{\mathrm{E}}$ 。

# A.2.5 酶活力计算

异淀粉酶的酶活力按式(A.2)计算：

$$
X _ {\mathrm {D}} = \frac {\left[ \left(A _ {\mathrm {E}} - A _ {\mathrm {B}}\right) \times K + C _ {0} \right]}{M \times t} \times 1 0 0 0 \dots \tag {A.2}
$$

式中：

$X_{\mathrm{D}}$ ——待测酶液的异淀粉酶活力，单位为活力单位每毫升 $(\mathrm{U} / \mathrm{mL})$ ；

$A_{\mathrm{E}}$ ——酶反应液的吸光度；

$A_{\mathrm{B}}$ ——酶空白液的吸光度；

$K$ ——标准曲线的斜率；

$C_0$ ——标准曲线的截距；

$M$ ——葡萄糖的摩尔质量， $M(\mathrm{C_6H_{12}O_6}) = 180.2\mathrm{g / mol}$

$t$ ——酶解反应的时间，单位为分 $(\mathrm{min})$ ；

1000——换算系数。

# A.3 淀粉葡糖苷酶酶活力测定方法

# A.3.1 以可溶性淀粉为底物的酶活力

以可溶性淀粉为底物的酶活力测定参照GB1886.174—2016《食品安全国家标准 食品添加剂食品工业用酶制剂》附录A.2 $\alpha$ -淀粉酶活力的测定。

# A.3.2 以对硝基酚- $\beta$ -麦芽糖苷为底物的酶活力

# A.3.2.1 原理

在 $\mathrm{pH}4.5$ ，温度为 $40^{\circ}\mathrm{C}$ ，底物(对硝基酚- $\beta$ -麦芽糖苷)浓度为 $10\mathrm{mmol / L}$ ，有过量 $\beta$ -葡糖苷酶存在的标准条件下，每分钟从对硝基酚- $\beta$ -麦芽糖苷释放出 $1\mu \mathrm{mol}$ 对硝基酚(p-NP)所需要的酶量为 $1\textrm{U}$ 。

# A.3.2.2 试剂和溶剂

A.3.2.2.1 乙酸 $(0.2\mathrm{mol} / \mathrm{L})$ ：准确吸取 $12~\mathrm{mL}$ 冰乙酸，用水稀释至 $1000~\mathrm{mL}$

A.3.2.2.2 乙酸钠溶液 $(0.2\mathrm{mol} / \mathrm{L})$ ：准确称取 $27.2\mathrm{g}$ 三水合乙酸钠，用水溶解并稀释至 $1000~\mathrm{mL}$ 。

A.3.2.2.3 乙酸盐缓冲液 $(\mathrm{pH}$ 为4.5）：将 $280~\mathrm{mL}$ 乙酸溶液 $(0.2\mathrm{mol} / \mathrm{L})$ 与 $220~\mathrm{mL}$ 乙酸钠溶液 $(0.2\mathrm{mol} / \mathrm{L})$ 混合，用水稀释至 $1000~\mathrm{mL}$ 。

A.3.2.2.4 碳酸钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：准确称取 $105.9\mathrm{g}$ 碳酸钠，用水溶解并稀释至 $1000~\mathrm{mL}$ 。

A.3.2.2.5 对硝基酚标准液 $(1\mathrm{mg / mL})$ ：称取对硝基酚 $100~\mathrm{mg}$ ，加乙酸盐缓冲液溶解，定容至 $100~\mathrm{mL}$ 。

A.3.2.2.6 $\beta$ -葡糖苷酶溶液：用乙酸盐缓冲液进行稀释、定容（稀释后的酶液中 $\beta$ -葡糖苷酶活力在 $2\mathrm{U} / \mathrm{mL} \sim 4\mathrm{U} / \mathrm{mL}$ 之间）。

A.3.2.2.7 对硝基酚- $\beta$ -麦芽糖苷溶液( $10\mathrm{mmol / mL}$ )：准确称取 $4.63\mathrm{g}$ 对硝基酚- $\beta$ -麦芽糖苷，用 $\beta$ -葡糖苷酶溶液溶解并稀释至 $1000~\mathrm{mL}$ 。

# A.3.2.3 仪器

A.3.2.3.1 分析天平。

A.3.2.3.2 电热恒温水浴锅。

A.3.2.3.3 分光光度计。

A.3.2.3.4 计时器。

# A.3.2.4 分析步骤

# A.3.2.4.1 对硝基苯标准曲线的绘制

吸取乙酸盐缓冲液 $2.0\mathrm{mL}$ ，加入 $1.0\mathrm{mL}$ 碳酸钠溶液，混匀，制成标准空白样。

分别吸取对硝基酚标准液 $1.00\mathrm{mL},2.00\mathrm{mL},3.00\mathrm{mL},4.00\mathrm{mL},5.00\mathrm{mL},6.00\mathrm{mL},7.00\mathrm{mL}$ ，分别用乙酸盐缓冲液定容至 $100\mathrm{mL}$ ，配制成浓度 $0.10\mathrm{mg / mL},0.20\mathrm{mg / mL},0.30\mathrm{mg / mL},0.40\mathrm{mg / mL},$ $0.50~\mathrm{mg / mL}$ 、 $0.60~\mathrm{mg / mL}$ 、 $0.70~\mathrm{mg / mL}$ 对硝基酚标准使用液，吸取对硝基酚标准使用液各 $1\mathrm{mL}$ ，分别加入到 $5\mathrm{mL}$ 刻度试管中，再分别加入 $1.0~\mathrm{mL}$ 乙酸盐缓冲液和 $1.0~\mathrm{mL}$ 碳酸钠溶液，以标准空白液为对照调零，在 $400~\mathrm{nm}$ 处测定吸光度值。以吸光度值为纵坐标，对硝基酚含量 $(\mathrm{mg})$ 为横坐标，绘制标准曲线。

# A.3.2.4.2 待测酶溶液的配制

称取测试酶样品，用乙酸盐缓冲液进行稀释、定容（稀释后的酶液中 $\beta$ -葡糖苷酶活力在 $2\mathrm{U} / \mathrm{mL}\sim$ $4\mathrm{U} / \mathrm{mL}$ 之间）。

# A.3.2.4.3 测定步骤

吸取 $10.0\mathrm{mL}$ 对硝基酚- $\beta$ -麦芽糖苷溶液， $40^{\circ}C$ 平衡 $20\mathrm{min}$

吸取 $10.0\mathrm{mL}$ 待测酶液， $40^{\circ}\mathrm{C}$ 平衡 $20\mathrm{min}$ 。

吸取 $1.0\mathrm{mL}$ 平衡后待测酶液，加入到 $5\mathrm{mL}$ 刻度试管中，再加入 $1.0\mathrm{mL}$ 碳酸钠溶液，混匀，然后加入 $1.0\mathrm{mL}$ 平衡后的对硝基酚- $\beta$ -麦芽糖苷溶液， $40^{\circ}C$ 平衡 $10\mathrm{min}$ ，室温放置 $5\mathrm{min}$ ，以标准空白液为对照调零，在 $400~\mathrm{nm}$ 处测定吸光度值 $A_{\mathrm{B}}$ 。

吸取 $1.0\mathrm{mL}$ 平衡后待测酶液，加入到 $5\mathrm{mL}$ 刻度试管中，再加入 $1.0\mathrm{mL}$ 平衡后的对硝基酚- $\beta$ -麦芽糖苷溶液，混匀， $40^{\circ}C$ 平衡 $10\mathrm{min}$ ，然后加入 $1.0\mathrm{mL}$ 碳酸钠溶液，室温放置 $5\mathrm{min}$ ，以标准空白液为对照调零，在 $400~\mathrm{nm}$ 处测定吸光度值 $A_{\mathrm{E}}$ 。

# A.3.2.5 酶活力计算

葡糖苷酶的酶活力按式(A.3)计算：

$$
X _ {\mathrm {D}} = \frac {\left[ \left(A _ {\mathrm {E}} - A _ {\mathrm {B}}\right) \times K + C _ {0} \right]}{M \times t} \times 1 0 0 0 \tag {A.3}
$$

式中：

$X_{\mathrm{D}}$ ——待测酶液的淀粉葡糖苷酶活力，单位为活力单位每毫升 $(\mathrm{U} / \mathrm{mL})$ ；

$A_{\mathrm{E}}$ ——酶反应液的吸光度；

$A_{\mathrm{B}}$ ——酶空白液的吸光度；

$K$ ——标准曲线的斜率；

$C_0$ ——标准曲线的截距；

$M$ ——对硝基酚的摩尔质量， $M(\mathrm{C_6H_5NO_3}) = 136.11\mathrm{g / mol}$

$t$ ——酶解反应的时间，单位为分 $(\mathrm{min})$ ；

1000——换算系数。

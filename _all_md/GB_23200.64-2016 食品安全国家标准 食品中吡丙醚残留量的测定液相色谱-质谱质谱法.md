# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2623—2010《进出口食品中吡丙醚残留量的检测方法液相色谱一质谱/质谱法》。与SN/T2623—2010相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“进出口食品”改为“食品”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T2623—2010。

# 食品安全国家标准

# 食品中吡丙醚残留量的测定 液相色谱一质谱/质谱法

# 1 范围

本标准规定了食品中吡丙醚残留量的制样和测定方法。

本标准适用于大米、大豆、菠菜、柠檬、茶叶、蘑菇、牛肉、猪肝和牛奶中吡丙醚残留量的检测，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样中残留的吡丙醚在醋酸钠缓冲剂作用下用酸性乙腈提取，再用PSA填料净化，液相色谱一质谱进行测定，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 无水硫酸镁 $(\mathrm{MgSO_4})$ 。  
4.1.2 无水乙酸钠 $(\mathrm{C}_2\mathrm{H}_3\mathrm{O}_2\mathrm{Na})$ 。  
4.1.3 乙酸 $\mathrm{(C_2H_4O_2)}$ ：色谱级。  
4.1.4 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ ：色谱级。  
4.1.5 N-丙基乙二胺(PSA)填料： $50\mu \mathrm{m}$ ，色谱级。  
4.1.6 甲酸 $(\mathrm{CH}_2\mathrm{O}_2)$ ：色谱级。  
4.1.7 乙酸铵 $(\mathrm{C_2H_7NO_2})$ 。

# 4.2 溶液配制

4.2.1 $0.025\%$ 甲酸水溶液(含 $5 \mathrm{mmol} / \mathrm{L}$ 乙酸铵): 准确吸取 $0.25 \mathrm{~mL}$ 甲酸和称取 $0.386 \mathrm{~g}$ 乙酸铵于 $1 \mathrm{~L}$ 容量瓶中, 用水溶解并稀释定容至 $1 \mathrm{~L}$ 。  
4.2.2 $1\%$ 乙酸乙腈溶液。

# 4.3 标准品

4.3.1 吡丙醚标准品(pyriproxyfen, $\mathrm{C_{20}H_{19}NO_3}$ ,CAS号：95737-68-1)：纯度 $>99\%$ 。

# 4.4 标准溶液配制

4.4.1 吡丙醚标准储备液：准确称取适量的吡丙醚标准品，用乙腈溶解并稀释配制成 $100\mu \mathrm{g / mL}$ 的标

准储备液， $4^{\circ}\mathrm{C}$ 以下保存。

4.4.2 吡丙醚标准工作液：根据需要，用 $10\%$ 乙腈水溶液稀释成适当浓度的标准工作液， $4^{\circ}\mathrm{C}$ 以下保存。  
4.5 材料  
4.5.1 微孔滤膜： $0.2\mathrm{mm}$ ，有机相。

# 5 仪器和设备

5.1 液相色谱—质谱/质谱仪：配有电喷雾(ESI)离子源。  
5.2 离心机： $4000\mathrm{r / min}$   
5.3 分析天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
5.4 具塞聚丙烯离心管， $2\mathrm{mL}$ 和 $50~\mathrm{mL}$   
5.5 粉碎机。

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按照 GB 2763 的规定执行。在取样和制样的操作过程中, 应防止样品受到污染或发生残留物含量的变化。

# 6.2 大米、大豆、菠菜、柠檬、茶叶、蘑菇、牛肉、猪肝

取有代表性样品约 $500 \mathrm{~g}$ , 用组织捣碎机充分捣碎混匀, 均分成两份, 分别装入洁净容器内作为试样。密封, 并标明标记。将试样置于 $-18^{\circ} \mathrm{C}$ 冷冻避光保存。

# 6.3 牛奶

取有代表性样品约 $500 \mathrm{~g}$ , 充分混匀, 均分成两份, 分别装入洁净容器内作为试样, 密封, 并标明标记。将试样置于低于 $4^{\circ} \mathrm{C}$ 冷藏避光保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 茶叶、大豆、蘑菇、大米样品

称取上述样品 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 聚丙烯塑料离心管中，加 $10\mathrm{mL}$ 水，混匀，静置 $30\mathrm{min}$ 。将离心管置于冰浴中，然后加 $6\mathrm{g}$ 无水硫酸镁和 $1.5\mathrm{g}$ 无水乙酸钠，恢复至室温后，准确加入 $15.0\mathrm{mL}$ $1\%$ 乙酸乙腈溶液，振荡提取 $4\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。

# 7.1.2 菠菜、柠檬和牛奶样品

称取 $15 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ ) 于预先称有 $6 \mathrm{~g}$ 无水硫酸镁和 $1.5 \mathrm{~g}$ 无水乙酸钠的 $50 \mathrm{~mL}$ 聚丙烯塑料离心管中, 将离心管置于冰浴中, 恢复至室温后, 准确加入 $15.0 \mathrm{~mL} 1 \%$ 乙酸乙腈溶液, 振荡提取 $4 \mathrm{~min}$ , $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ 。

# 7.1.3 牛肉、猪肝样品

称取 $15\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 聚丙烯塑料离心管中，准确加入 $15\mathrm{mL}1\%$ 乙酸乙腈溶液，振荡提取 $4\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。另取称有 $6\mathrm{g}$ 无水硫酸镁和 $1.5\mathrm{g}$ 无水乙酸钠的 $50~\mathrm{mL}$ 聚丙烯塑料离心管，加入 $10~\mathrm{mL}$ 水，溶解，待水溶液恢复到室温后，再将乙腈提取液转移至其中，振荡 $4\mathrm{min},4000\mathrm{r}/$ min离心 $5\mathrm{min}$ 。

# 7.2 净化

取 $1\mathrm{mL}$ 上述乙腈提取液于预先称有 $50\mathrm{mg}$ PSA和 $150\mathrm{mg}$ 无水硫酸镁的 $2\mathrm{mL}$ 聚丙烯塑料离心管

中，振荡提取 $1\mathrm{min}$ ，在 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。

准确吸取 $200\mu \mathrm{L}$ 上清液用水定容至 $1\mathrm{mL}$ ，混匀后过 $0.2\mu \mathrm{m}$ 滤膜。滤液供液相色谱一质谱测定。

# 7.3 测定

# 7.3.1 高效液相色谱参考条件

a）色谱柱：色谱柱：CAPCELLPAK $C_{18}$ 柱， $50\mathrm{mm}\times 2.0\mathrm{mm}$ （内径），膜厚 $3\mu \mathrm{m}$ ，或相当者；  
b) 流动相: 流动相 A 为 $0.1\%$ 甲酸乙腈溶液, 流动相 B 为 $0.025\%$ 甲酸水溶液 (含 $5 \mathrm{mmol} / \mathrm{L}$ 乙酸铵);  
c) 流速: $300 \mu \mathrm{L} / \mathrm{min}$ , 梯度洗脱程序见表 1;

d) 柱温: $35^{\circ} \mathrm{C}$ ;   
e) 进样量: $20 \mathrm{~mL}$ 。

# 7.3.2 质谱参考条件

a）离子源：电喷雾ESI，正离子；  
b) 扫描方式：多反应监测(MRM)；  
c）其他参数条件参见附录A。

# 7.3.3 定量测定

根据试样中被测物的含量,选取响应值相近的标准工作液同时进行分析。标准工作液和待测液中吡丙醚的响应值均应在仪器线性响应范围内。在上述色谱条件下的吡丙醚的参考保留时间为 $1.4 \mathrm{~min}$ , 标准溶液的多反应监测色谱图参见附录 B。

# 7.3.4 定性测定

按照液相色谱—质谱/质谱条件测定样品和标准工作溶液，如果检测的质量色谱峰保留时间与标准品一致，定性离子对的相对丰度是用相对于最强离子丰度的强度百分比表示，应当与浓度相当标准工作溶液的相对丰度一致，相对丰度允许偏差不超过表2规定的范围，则可判断样品中存在对应的被测物。

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算样品中待测药物残留量。计算结果需扣除空白值。

$$
X = \frac {A \times c \times V}{A _ {s} \times m \times 1 0 0 0} \dots \tag {1}
$$

式中：

$X$ ——试样中吡丙醚的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

A ——样液中吡丙醚的峰面积；

$c$ ——标准溶液中吡丙醚的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

V ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$

$A_{s}$ ——标准溶液中吡丙醚的峰面积；

m 最终溶液所代表试样的质量，单位为克(g)；

1000 ——换算系数。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 C 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 D 的要求。

# 10.1 定量限

本方法菠菜、柠檬、牛肉、猪肝和牛奶中的定量限为 $5.0\mu \mathrm{g} / \mathrm{kg}$ 。

本方法大豆、大米、蘑菇和茶叶中的定量限为 $15.0\mu \mathrm{g} / \mathrm{kg}$

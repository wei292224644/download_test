# 前言

本标准代替GB/T5009.16—2003《食品中锡的测定》。

本标准与GB/T5009.16—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中锡的测定”；  
——修改了标准溶液配制；  
——增加了罐头食品的试样制备方法；  
——修改了仪器测定部分的描述；  
——增加了方法定量限；  
——修改了方法检出限；  
——修改了计算公式。

# 食品安全国家标准

# 食品中锡的测定

# 1 范围

本标准规定了食品中锡的氢化物原子荧光光谱法和苯芬酮比色法的测定方法。

本标准适用于罐装固体食品、罐装饮料、罐装果酱、罐装婴幼儿配方及辅助食品中锡的测定。

# 第一法 氢化物原子荧光光谱法

# 2 原理

试样经消化后，在硼氢化钠的作用下生成锡的氢化物 $(\mathrm{SnH_4})$ ，并由载气带入原子化器中进行原子化，在锡空心阴极灯的照射下，基态锡原子被激发至高能态，在去活化回到基态时，发射出特征波长的荧光，其荧光强度与锡含量成正比，与标准系列溶液比较定量。

# 3 试剂和材料

注：除特别注明外，本方法所使用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ ：优级纯。  
3.1.2 硝酸 $\mathrm{(HNO_3)}$ ：优级纯。  
3.1.3 高氯酸 $\mathrm{(HClO_4)}$ ：优级纯。  
3.1.4 硫脲 $\mathrm{(CH_4N_2S)}$ 。  
3.1.5 抗坏血酸 $\mathrm{(C_6H_8O_6)}$ 。  
3.1.6 硼氢化钠 $\mathrm{(NaBH_4)}$   
3.1.7 氢氧化钠(NaOH)。

# 3.2 试剂配制

3.2.1 硝酸-高氯酸混合溶液 $(4 + 1)$ ：量取 $400~\mathrm{mL}$ 硝酸和 $100~\mathrm{mL}$ 高氯酸，混匀。  
3.2.2 硫酸溶液 $(1 + 9)$ ：量取 $100~\mathrm{mL}$ 硫酸倒入 $900~\mathrm{mL}$ 水中，混匀。  
3.2.3 硫脲 $(150\mathrm{g / L})+$ 抗坏血酸 $(150\mathrm{g / L})$ 混合溶液：分别称取 $15.0\mathrm{g}$ 硫脲和 $15.0\mathrm{g}$ 抗坏血酸溶于水中，并稀释至 $100~\mathrm{mL}$ ，置于棕色瓶中避光保存或临用时配制。  
3.2.4 氢氧化钠溶液 $(5.0\mathrm{g} / \mathrm{L})$ ：称取氢氧化钠 $5.0\mathrm{g}$ 溶于 $1000~\mathrm{mL}$ 水中。  
3.2.5 硼氢化钠溶液 $(7.0\mathrm{g} / \mathrm{L})$ ：称取 $7.0\mathrm{g}$ 硼氢化钠，溶于氢氧化钠溶液中，临用时配制。

# 3.3 标准品

金属锡(Sn)标准品，纯度为 $99.99\%$ 或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液的配制

3.4.1 锡标准溶液 $(1.0\mathrm{mg / mL})$ ：准确称取 $0.1\mathrm{g}$ （精确到 $0.0001\mathrm{g}$ 金属锡标准品，置于小烧杯中，加入 $10.0~\mathrm{mL}$ 硫酸，盖以表面皿，加热至锡完全溶解，移去表面皿，继续加热至发生浓白烟，冷却，慢慢加入 $50~\mathrm{mL}$ 水，移入 $100~\mathrm{mL}$ 容量瓶中，用硫酸溶液 $(1 + 9)$ 多次洗涤烧杯，洗液并入容量瓶中，并稀释至刻度，混匀。  
3.4.2 锡标准使用液 $(1.0\mu \mathrm{g / mL})$ ：准确吸取锡标准溶液 $1.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用硫酸溶液(1+9)定容至刻度。此溶液浓度为 $10.0\mu \mathrm{g / mL}$ 。准确吸取该溶液 $10.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用硫酸溶液 $(1 + 9)$ 定容至刻度。

# 4 仪器和设备

4.1 原子荧光光谱仪。  
4.2 电热板。  
4.3 电子天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$

# 5 分析步骤

# 5.1 试样制备

罐头食品全量取可食内容物制成匀浆或者均匀粉末。

# 5.2 试样消化

5.2.1 称取试样 $1.0\mathrm{g}\sim 5.0\mathrm{g}$ 于锥形瓶中，加入 $20.0~\mathrm{mL}$ 硝酸-高氯酸混合溶液 $(4 + 1)$ ，加 $1.0\mathrm{mL}$ 硫酸，3粒玻璃珠，放置过夜。次日置电热板上加热消化，如酸液过少，可适当补加硝酸，继续消化至冒白烟，待液体体积近 $1\mathrm{mL}$ 时取下冷却。用水将消化试样转入 $50~\mathrm{mL}$ 容量瓶中，加水定容至刻度，摇匀备用。同时做空白试验（如试样液中锡含量超出标准曲线范围，则用水进行稀释，并补加硫酸，使最终定容后的硫酸浓度与标准系列溶液相同）。  
5.2.2 取定容后的 5.2.1 试样 $10.0 \mathrm{~mL}$ 于 $25 \mathrm{~mL}$ 比色管中，加入 $3.0 \mathrm{~mL}$ 硫酸溶液 $(1 + 9)$ ，加入 $2.0 \mathrm{~mL}$ 硫脲 $(150 \mathrm{~g} / \mathrm{L})+$ 抗坏血酸 $(150 \mathrm{~g} / \mathrm{L})$ 混合溶液，再用水定容至 $25 \mathrm{~mL}$ ，摇匀。

# 5.3 仪器参考条件

原子荧光光谱仪分析参考条件：  
负高压： $380\mathrm{V}$   
灯电流： $70\mathrm{mA}$   
——原子化温度： $850^{\circ}C$   
炉高： $10\mathrm{mm}$   
屏蔽气流量： $1200~\mathrm{mL / min}$   
载气流量： $500~\mathrm{mL / min}$   
——测量方式：标准曲线法；  
——读数方式：峰面积；  
——延迟时间：1s；  
——读数时间：15s；  
加液时间：8s；  
——进样体积： $2.0~\mathrm{mL}$

# 5.4 标准系列溶液的配制

标准曲线：分别吸取锡标准使用液 $0.00\mathrm{mL}, 0.50\mathrm{mL}, 2.00\mathrm{mL}, 3.00\mathrm{mL}, 4.00\mathrm{mL}, 5.00\mathrm{mL}$ 于 $25\mathrm{mL}$ 比色管中，分别加入硫酸溶液 $(1 + 9)5.00\mathrm{mL}, 4.50\mathrm{mL}, 3.00\mathrm{mL}, 2.00\mathrm{mL}, 1.00\mathrm{mL}, 0.00\mathrm{mL}$ ，加入 $2.0\mathrm{mL}$ 硫脲 $(150\mathrm{g / L})+$ 抗坏血酸 $(150\mathrm{g / L})$ 混合溶液，再用水定容至 $25\mathrm{mL}$ 。该标准系列溶液浓度为： $0\mathrm{ng / mL}, 20\mathrm{ng / mL}, 80\mathrm{ng / mL}, 120\mathrm{ng / mL}, 160\mathrm{ng / mL}, 200\mathrm{ng / mL}$ 。

# 5.5 仪器测定

按照5.3设定好仪器测量最佳条件，根据所用仪器的型号和工作站设置相应的参数，点火及对仪器进行预热，预热 $30\mathrm{min}$ 后进行标准曲线及试样溶液的测定。

# 6 分析结果的表述

试样中锡含量按式(1)进行计算：

$$
X = \frac {\left(c _ {1} - c _ {0}\right) \times V _ {1} \times V _ {3}}{m \times V _ {2} \times 1 0 0 0} \quad \dots \tag {1}
$$

式中

$X$ ——试样中锡含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{1}$ ——试样消化液测定浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{0}$ ——试样空白消化液浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V_{1}$ ——试样消化液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——测定用溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——测定用所取试样消化液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数。

当计算结果小于 $10\mathrm{mg / kg}$ 时保留小数点后两位数字，大于 $10\mathrm{mg / kg}$ 时保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当取样量为 $1.0\mathrm{g}$ 时，本方法定量限为 $2.5\mathrm{mg / kg}$

# 第二法 苯芴酮比色法

# 9 原理

试样经消化后，在弱酸性溶液中四价锡离子与苯芴酮形成微溶性橙红色络合物，在保护性胶体存在下与标准系列溶液比较定量。

# 10 试剂和材料

注：除特别注明外，本方法所使用试剂均为分析纯，水为GB/T6682规定的三级水。

# 10.1 试剂

10.1.1 酒石酸 $(\mathrm{C_4H_4O_6H_2})$ 。  
10.1.2 抗坏血酸 $\mathrm{(C_6H_8O_6)}$ 。  
10.1.3 酚酞 $(\mathrm{C}_{20}\mathrm{H}_{14}\mathrm{O}_4)$ 。  
10.1.4 氨水 $(\mathrm{NH}_4\mathrm{OH})$ 。  
10.1.5 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。  
10.1.6 乙醇 $\mathrm{(C_2H_5OH)}$ 。  
10.1.7 甲醇 $\mathrm{(CH_3OH)}$ 。  
10.1.8 苯芴酮 $(\mathrm{C_{19}H_{12}O_5})$ 。  
10.1.9 动物胶（明胶）。

# 10.2 试剂配制

10.2.1 酒石酸溶液 $(100\mathrm{g / L})$ ：称取 $100\mathrm{g}$ 酒石酸溶于 $1\mathrm{L}$ 水中。  
10.2.2 抗坏血酸溶液 $(10.0\mathrm{g / L})$ ：称取 $10.0\mathrm{g}$ 抗坏血酸溶于 $1\textrm{L}$ 水，临用时配制。  
10.2.3 动物胶溶液 $(5.0\mathrm{g / L})$ ：称取 $5.0\mathrm{g}$ 动物胶溶于 $1\textrm{L}$ 水，临用时配制。  
10.2.4 氨溶液 $(1 + 1)$ ：量取 $100~\mathrm{mL}$ 氨水加入 $100~\mathrm{mL}$ 水中，混匀。  
10.2.5 硫酸溶液 $(1 + 9)$ ：量取 $10~\mathrm{mL}$ 硫酸，搅拌下缓缓倒入 $90~\mathrm{mL}$ 水中，混匀。  
10.2.6 苯芴酮溶液 $(0.1\mathrm{g / L})$ ：称取 $0.01\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )苯芴酮加少量甲醇及硫酸数滴溶解，以甲醇稀释至 $100~\mathrm{mL}$ 。  
10.2.7 酚酞指示液 $(10.0\mathrm{g / L})$ ：称取 $1.0\mathrm{g}$ 酚酞，用乙醇溶解至 $100~\mathrm{mL}$ 。

# 10.3 标准品

金属锡(Sn)标准品，纯度为 $99.99\%$ 或经国家认证并授予标准物质证书的标准物质。

# 10.4 标准溶液的配制

10.4.1 锡标准溶液 $(1.0\mathrm{mg / mL})$ ：准确称取 $0.1\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ 金属锡，置于小烧杯中，加入 $10~\mathrm{mL}$ 硫酸，盖以表面皿，加热至锡完全溶解，移去表面皿，继续加热至发生浓白烟，冷却，慢慢加入 $50~\mathrm{mL}$ 水，移入 $100~\mathrm{mL}$ 容量瓶中，用硫酸溶液 $(1 + 9)$ 多次洗涤烧杯，洗液并入容量瓶中，并稀释至刻度，混匀。  
10.4.2 锡标准使用液：吸取 $10.0\mathrm{mL}$ 锡标准溶液，置于 $100~\mathrm{mL}$ 容量瓶中，以硫酸溶液 $(1 + 9)$ 稀释至刻度，混匀。如此再次稀释至每毫升相当于 $10.0\mu \mathrm{g}$ 锡。

# 11 仪器和设备

11.1 分光光度计。  
11.2 电子天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$

# 12 分析步骤

# 12.1 试样制备

12.1.1 试样消化，同5.2.1。

12.1.2 吸取 $1.00\mathrm{mL} \sim 5.00\mathrm{mL}$ 试样消化液和同量的试剂空白溶液，分别置于 $25\mathrm{mL}$ 比色管中。于试样消化液、试剂空白液中各加 $0.5\mathrm{mL}$ 酒石酸溶液 $(100\mathrm{g / L})$ 及1滴酚酞指示液 $(100\mathrm{g / L})$ ，混匀，各加氨溶液 $(1 + 1)$ 中和至淡红色，加 $3.0\mathrm{mL}$ 硫酸溶液 $(1 + 1), 1.0\mathrm{mL}$ 动物胶溶液 $(5.0\mathrm{g / L})$ 及 $2.5\mathrm{mL}$ 抗坏血酸溶液 $(10.0\mathrm{g / L})$ ，再加水至 $25\mathrm{mL}$ ，混匀，再各加 $2.0\mathrm{mL}$ 苯芴酮溶液 $(0.1\mathrm{g / L})$ ，混匀，放置 $1\mathrm{h}$ 后测量。

# 12.2 标准曲线的制作

吸取 $0.00\mathrm{mL}, 0.20\mathrm{mL}, 0.40\mathrm{mL}, 0.60\mathrm{mL}, 0.80\mathrm{mL}, 1.00\mathrm{mL}$ 锡标准使用液（相当于 $0.00\mu \mathrm{g}$ 、 $2.00\mu \mathrm{g}, 4.00\mu \mathrm{g}, 6.00\mu \mathrm{g}, 8.00\mu \mathrm{g}, 10.00\mu \mathrm{g}$ 锡），分别置于 $25\mathrm{mL}$ 比色管中，各加 $0.5\mathrm{mL}$ 酒石酸溶液 $(100\mathrm{g} / \mathrm{L})$ 及1滴酚酞指示液 $(10.0\mathrm{g} / \mathrm{L})$ ，混匀，各加氨溶液 $(1 + 1)$ 中和至淡红色，加 $3.0\mathrm{mL}$ 硫酸溶液 $(1 + 9), 1.0\mathrm{mL}$ 动物胶溶液 $(5.0\mathrm{g} / \mathrm{L})$ 及 $2.5\mathrm{mL}$ 抗坏血酸溶液 $(10.0\mathrm{g} / \mathrm{L})$ ，再加水至 $25\mathrm{mL}$ ，混匀，再各加 $2.0\mathrm{mL}$ 苯芴酮溶液，混匀，放置 $1\mathrm{h}$ 后测量。

用 $2\mathrm{cm}$ 比色杯于波长 $490~\mathrm{nm}$ 处测吸光度，标准各点减去零管吸光值后，以标准系列溶液的浓度为横坐标，以吸光度为纵坐标，绘制标准曲线或计算直线回归方程。

# 12.3 试样溶液的测定

用 $2\mathrm{cm}$ 比色杯以标准系列溶液零管调节零点，于波长 $490~\mathrm{nm}$ 处分别对试剂空白溶液和试样溶液测定吸光度，所得吸光值与标准曲线比较或代入回归方程求出含量。

# 13 分析结果的表述

试样中锡的含量按式(2)进行计算：

$$
X = \frac {\left(m _ {1} - m _ {2}\right) \times V _ {1}}{m _ {3} \times V _ {2}} \quad \dots \tag {2}
$$

式中：

$X$ ——试样中锡的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$m_{1}$ ——测定用试样消化液中锡的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_{2}$ ——试剂空白液中锡的质量，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——试样消化液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m_{3}$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——测定用试样消化液的体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留两位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

当取样量为 $1.0\mathrm{g}$ ，取消化液为 $5.0\mathrm{mL}$ 测定时，本方法定量限为 $20\mathrm{mg / kg}$ 。

# 前言

本标准代替GB/T5009.123—2003《食品中铬的测定方法》。

本标准与GB/T5009.123—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中铬的测定方法”；  
——样品前处理增加了微波消解法和湿法消解法；  
——增加了方法定量限（LOQ）；  
基体改进剂采用磷酸二氢铵代替磷酸铵；  
——删除第二法示波极谱法。

# 食品安全国家标准

# 食品中铬的测定

# 1 范围

本标准规定了食品中铬的石墨炉原子吸收光谱测定方法。

本标准适用于各类食品中铬的含量测定。

# 2 原理

试样经消解处理后，采用石墨炉原子吸收光谱法，在 $357.9\mathrm{nm}$ 处测定吸收值，在一定浓度范围内其吸收值与标准系列溶液比较定量。

# 3 试剂和材料

注：除非另有规定，本方法所用试剂均为优级纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 硝酸 $\mathrm{(HNO_3)}$ 。

3.1.2 高氯酸 $\mathrm{(HClO_4)}$ 。

3.1.3 磷酸二氢铵 $(\mathrm{NH_4H_2PO_4})$ 。

# 3.2 试剂配制

3.2.1 硝酸溶液 $(5 + 95)$ ：量取 $50~\mathrm{mL}$ 硝酸慢慢倒入 $950~\mathrm{mL}$ 水中，混匀。  
3.2.2 硝酸溶液 $(1 + 1)$ ：量取 $250~\mathrm{mL}$ 硝酸慢慢倒入 $250~\mathrm{mL}$ 水中，混匀。  
3.2.3 磷酸二氢铵溶液 $(20\mathrm{g / L})$ ：称取 $2.0\mathrm{g}$ 磷酸二氢铵，溶于水中，并定容至 $100~\mathrm{mL}$ ，混匀。

# 3.3 标准品

重铬酸钾 $\mathrm{(K_2Cr_2O_7)}$ ：纯度 $>99.5\%$ 或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 铬标准储备液：准确称取基准物质重铬酸钾 $(110^{\circ}\mathrm{C}$ ，烘 $2\mathrm{h})$ 1.4315 g（精确至 $0.0001\mathrm{g}$ ），溶于水中，移入 $500~\mathrm{mL}$ 容量瓶中，用硝酸溶液 $(5 + 95)$ 稀释至刻度，混匀。此溶液每毫升含 $1.000\mathrm{mg}$ 铬。或购置经国家认证并授予标准物质证书的铬标准储备液。  
3.4.2 铬标准使用液：将铬标准储备液用硝酸溶液 $(5 + 95)$ 逐级稀释至每毫升含 $100\mathrm{ng}$ 铬。  
3.4.3 标准系列溶液的配制：分别吸取铬标准使用液 $(100\mathrm{ng / mL})0\mathrm{mL},0.500\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL}$ 、 $3.00~\mathrm{mL},4.00~\mathrm{mL}$ 于 $25~\mathrm{mL}$ 容量瓶中，用硝酸溶液(5+95)稀释至刻度，混匀。各容量瓶中每毫升分别含铬 $0\mathrm{ng},2.00\mathrm{ng},4.00\mathrm{ng},8.00\mathrm{ng},12.0\mathrm{ng},16.0\mathrm{ng}$ 。或采用石墨炉自动进样器自动配制。

# 4 仪器设备

注：所用玻璃仪器均需以硝酸溶液 $(1 + 4)$ 浸泡 $24\mathrm{h}$ 以上，用水反复冲洗，最后用去离子水冲洗干净。

4.1 原子吸收光谱仪，配石墨炉原子化器，附铬空心阴极灯。  
4.2 微波消解系统，配有消解内罐。  
4.3 可调式电热炉。  
4.4 可调式电热板。  
4.5 压力消解器：配有消解内罐。  
4.6 马弗炉。  
4.7 恒温干燥箱。  
4.8 电子天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$

# 5 分析步骤

# 5.1 试样的预处理

5.1.1 粮食、豆类等去除杂物后，粉碎，装入洁净的容器内，作为试样。密封，并标明标记，试样应于室温下保存。  
5.1.2 蔬菜、水果、鱼类、肉类及蛋类等水分含量高的鲜样，直接打成匀浆，装入洁净的容器内，作为试样。密封，并标明标记。试样应于冰箱冷藏室保存。

# 5.2 样品消解

# 5.2.1 微波消解

准确称取试样 $0.2\mathrm{g}\sim 0.6\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )于微波消解罐中，加入 $5\mathrm{mL}$ 硝酸，按照微波消解的操作步骤消解试样（消解条件参见A.1）。冷却后取出消解罐，在电热板上于 $140~\mathrm{^\circ C}\sim 160~\mathrm{^\circ C}$ 赶酸至 $0.5\mathrm{mL}\sim 1.0\mathrm{mL}$ 。消解罐放冷后，将消化液转移至 $10~\mathrm{mL}$ 容量瓶中，用少量水洗涤消解罐2次～3次，合并洗涤液，用水定容至刻度。同时做试剂空白试验。

# 5.2.2 湿法消解

准确称取试样 $0.5\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 于消化管中，加入 $10~\mathrm{mL}$ 硝酸、 $0.5\mathrm{mL}$ 高氯酸，在可调式电热炉上消解（参考条件： $120^{\circ}C$ 保持 $0.5\mathrm{h}\sim 1\mathrm{h}$ 、升温至 $180^{\circ}C2\mathrm{h}\sim 4\mathrm{h}$ 、升温至 $200^{\circ}C\sim 220^{\circ}C)$ 。若消化液呈棕褐色，再加硝酸，消解至冒白烟，消化液呈无色透明或略带黄色，取出消化管，冷却后用水定容至 $10~\mathrm{mL}$ 。同时做试剂空白试验。

# 5.2.3 高压消解

准确称取试样 $0.3\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 于消解内罐中，加入 $5\mathrm{mL}$ 硝酸。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱，于 $140~^\circ \mathrm{C}\sim 160~^\circ \mathrm{C}$ 下保持 $4\mathrm{h}\sim 5\mathrm{h}$ 。在箱内自然冷却至室温，缓慢旋松外罐，取出消解内罐，放在可调式电热板上于 $140~^\circ \mathrm{C}\sim 160~^\circ \mathrm{C}$ 赶酸至 $0.5\mathrm{mL}\sim 1.0\mathrm{mL}$ 。冷却后将消化液转移至 $10~\mathrm{mL}$ 容量瓶中，用少量水洗涤内罐和内盖2次～3次，合并洗涤液于容量瓶中并用水定容至刻度。同时做试剂空白试验。

# 5.2.4 干法灰化

准确称取试样 $0.5\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 于坩埚中，小火加热，炭化至无烟，转移至马弗炉中，于

$550~\mathrm{^\circ C}$ 恒温 $3\mathrm{h}\sim 4\mathrm{h}$ 。取出冷却，对于灰化不彻底的试样，加数滴硝酸，小火加热，小心蒸干，再转入 $550~^\circ \mathrm{C}$ 高温炉中，继续灰化 $1\mathrm{h}\sim 2\mathrm{h}$ ，至试样呈白灰状，从高温炉取出冷却，用硝酸溶液 $(1 + 1)$ 溶解并用水定容至 $10~\mathrm{mL}$ 。同时做试剂空白试验。

# 5.3 测定

# 5.3.1 仪器测试条件

根据各自仪器性能调至最佳状态。参考条件见A.2。

# 5.3.2 标准曲线的制作

将标准系列溶液工作液按浓度由低到高的顺序分别取 $10\mu \mathrm{L}$ （可根据使用仪器选择最佳进样量），注入石墨管，原子化后测其吸光度值，以浓度为横坐标，吸光度值为纵坐标，绘制标准曲线。

# 5.3.3 试样测定

在与测定标准溶液相同的实验条件下，将空白溶液和样品溶液分别取 $10~\mu \mathrm{L}$ （可根据使用仪器选择最佳进样量），注入石墨管，原子化后测其吸光度值，与标准系列溶液比较定量。

对有干扰的试样应注入 $5\mu \mathrm{L}$ (可根据使用仪器选择最佳进样量)的磷酸二氢铵溶液 $(20.0\mathrm{g / L})$ （标准系列溶液的制作过程应按5.3.3操作）。

# 6 分析结果的表述

试样中铬含量的计算见式(1)：

$$
X = \frac {\left(c - c _ {0}\right) \times V}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中铬的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——测定样液中铬的含量，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{0}$ ——空白液中铬的含量，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——样品消化液的定容总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品称样量，单位为克(g)；

1000 ——换算系数。

当分析结果 $\geqslant 1\mathrm{mg / kg}$ 时，保留三位有效数字；当分析结果 $<  1\mathrm{mg / kg}$ 时，保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

以称样量 $0.5\mathrm{g}$ ，定容至 $10~\mathrm{mL}$ 计算，方法检出限为 $\mathrm{0.01mg / kg}$ ，定量限为 $\mathrm{0.03mg / kg}$

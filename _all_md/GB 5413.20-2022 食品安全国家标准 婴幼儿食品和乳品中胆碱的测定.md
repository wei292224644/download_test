# 前言

本标准代替GB5413.20—2013《食品安全国家标准 婴幼儿食品和乳品中胆碱的测定》。

本标准与GB5413.20—2013相比，主要变化如下：

——删除了第二法雷氏盐分光光度法，增加了离子色谱法作为第二法；

——增加了液相色谱-串联质谱法作为第三法。

# 食品安全国家标准

# 婴幼儿食品和乳品中胆碱的测定

# 1 范围

本标准规定了婴幼儿食品和乳品中胆碱的测定方法。

本标准适用于婴幼儿食品和乳品中胆碱的测定。

# 第一法 酶比色法

# 2 原理

试样经酸水解，再经酶作用后与显色剂反应生成有色物质，在一定浓度范围内，其颜色的深浅与胆碱含量成正比，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 三羟甲基氨基甲烷 $\left[(\mathrm{CH}_2\mathrm{OH})_3\mathrm{CNH}_2\right]$   
3.1.2 苯酚 $\mathrm{(C_6H_5OH)}$ 。  
3.1.3 浓盐酸（HCl）。  
3.1.4 氢氧化钠(NaOH)。  
3.1.5 胆碱氧化酶： $\geqslant 10\mathrm{U / mg}$ ，置于一 $20^{\circ}C$ 保存。  
3.1.6 过氧化物酶： $\geq 250\mathrm{U / mg}$ ，置于 $2^{\circ}C\sim 8^{\circ}C$ 保存。  
3.1.7 4-氨基安替比林 $(\mathrm{C_{11}H_{13}N_3O})$ 。  
3.1.8 磷脂酶 $\mathrm{D}:\geqslant 60\mathrm{U / mg}$ ，置于一 $20^{\circ}C$ 保存。

# 3.2 试剂配制

3.2.1 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $85~\mathrm{mL}$ 浓盐酸注入约 $900~\mathrm{mL}$ 水中，稀释至 $1000~\mathrm{mL}$ 。  
3.2.2 盐酸溶液 $(3\mathrm{mol} / \mathrm{L})$ ：量取 $250~\mathrm{mL}$ 浓盐酸注入约 $600~\mathrm{mL}$ 水中，稀释至 $1000~\mathrm{mL}$ 。  
3.2.3 三羟甲基氨基甲烷缓冲溶液（Tris） $(0.05\mathrm{mol} / \mathrm{L})$ ：准确称取 $6.057\mathrm{g}$ 三羟甲基氨基甲烷溶于 $500~\mathrm{mL}$ 水中，用盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ 调 $\mathrm{pH}$ 至 $8.0\pm 0.2$ ，再用水定容至 $1000~\mathrm{mL}$ 。此溶液在 $4^{\circ}\mathrm{C}$ 冰箱中可保存1个月。  
3.2.4 显色剂：取 $120\mathrm{U}$ 的胆碱氧化酶、 $280\mathrm{U}$ 的过氧化物酶、 $100\mathrm{U}$ 的磷脂酶 $\mathrm{D}, 15\mathrm{mg}4-$ 氨基安替比林和 $50\mathrm{mg}$ 苯酚置于 $100\mathrm{mL}$ 的容量瓶中，用 $0.05\mathrm{mol/L}$ Tris缓冲溶液溶解并定容。临用现配。  
3.2.5 氢氧化钠溶液 $(500\mathrm{g / L})$ ：称取 $500\mathrm{g}$ 氢氧化钠，溶于水并稀释至 $1000~\mathrm{mL}$ ，采用塑料容器储存。

# 3.3 标准品

胆碱酒石酸氢盐标准品 $\mathrm{(C_9H_{19}NO_7}$ ，相对分子质量：253.25，CAS号：87-67-2）：纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 胆碱（以胆碱氢氧化物计，相对分子质量121.18)标准储备溶液 $(2500\mathrm{mg / L})$ ：准确称取在 $102^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 烘至恒重的胆碱酒石酸氢盐 $522.5\mathrm{mg}$ ，用水溶解并转移至 $100~\mathrm{mL}$ 容量瓶中定容，混匀。 $4^{\circ}\mathrm{C}$ 以下避光保存，有效期3个月。  
3.4.2 胆碱标准工作溶液 $(250\mathrm{mg / L})$ ：吸取 $10.0~\mathrm{mL}$ 胆碱标准储备溶液于 $100~\mathrm{mL}$ 容量瓶中，用水定容，混匀， $4^{\circ}C$ 以下避光保存，有效期1个月。

# 3.5 材料

3.5.1 $0.45\mu \mathrm{m}$ 水性滤膜针头滤器。  
3.5.2 注射器： $5\mathrm{mL}$ 或相当者。

# 4 仪器和设备

4.1 分析天平：感量分别为 $0.1\mathrm{mg}$ 和 $0.001\mathrm{g}$   
4.2 恒温水浴装置：温度可控制在 $70^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 和 $37^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。  
4.3 pH计：精度0.01。  
4.4 分光光度计。

# 5 分析步骤

# 5.1 样品前处理

# 5.1.1 样品预处理

准确称取 $20\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 混合均匀的液态试样于 $100\mathrm{mL}$ 锥形瓶中，加入 $3\mathrm{mol/L}$ 的盐酸溶液 $10\mathrm{mL}$ ，加塞混匀。

准确称取 $5\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 混合均匀的半固态或固态试样于 $100\mathrm{mL}$ 锥形瓶中，加入 $1\mathrm{mol/L}$ 的盐酸溶液 $30\mathrm{mL}$ ，加塞混匀。

# 5.1.2 水解

将装有试样的容器放在 $70^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 水浴中，放置 $3\mathrm{h}$ （每隔 $30\mathrm{min}$ 振摇一次），冷却至室温。用氢氧化钠溶液 $(500\mathrm{g / L})$ 调 $\mathrm{pH}$ 至 $3.5\sim 4.0$ ，转入 $50~\mathrm{mL}$ 容量瓶中，用水定容至刻度。

# 5.1.3 过滤

用滤纸过滤水解液，若滤液不澄清，用 $0.45\mu \mathrm{m}$ 水性滤膜针头滤器再次过滤。收集滤液待测。

# 5.2 测定

# 5.2.1 标准曲线的制作

分别吸取 $2.00\mathrm{mL},4.00\mathrm{mL},6.00\mathrm{mL},8.00\mathrm{mL},10.00\mathrm{mL}$ 胆碱标准工作溶液 $(250~\mathrm{mg / L})$ 于 $10~\mathrm{mL}$

的容量瓶中，用水定容至刻度，混匀，配制成浓度分别为 $50.0\mathrm{mg / L},100\mathrm{mg / L},150\mathrm{mg / L},200\mathrm{mg / L},$ $250~\mathrm{mg / L}$ 的标准系列工作溶液。准备6支比色管，一个比色管用作试剂空白(A)，加入 $100~\mu \mathrm{L}$ 水，另5支比色管对应加入标准系列工作溶液各 $100~\mu \mathrm{L}$ ，再分别加入 $3.00~\mathrm{mL}$ 显色剂，混匀，把比色管置于 $37^{\circ}C\pm 2^{\circ}C$ 水浴中保温反应 $15\mathrm{min}$

# 5.2.2 试样准备

准备2支比色管(B，C)，均加入待分析溶液 $100\mu \mathrm{L}$ ，比色管B加入 $3.00~\mathrm{mL}$ 水，比色管C加入 $3.00~\mathrm{mL}$ 显色剂，混匀，把比色管置于 $37^{\circ}C\pm 2^{\circ}C$ 水浴中保温反应 $15\mathrm{min}$

# 5.2.3 比色测定

将标准系列工作溶液及试样从水浴中取出，冷却至室温。在波长 $505\mathrm{nm}$ 处，用水作空白，用 $1\mathrm{cm}$ 微量比色皿测定吸光值。以胆碱标准溶液的浓度为横坐标，以标准溶液的吸光值减去试剂空白（比色管A)的吸光值为纵坐标，制作标准曲线。

# 6 分析结果的表述

# 6.1 净吸光值的计算

通常配制的试剂会产生轻微颜色，且由于水解作用滤液也不是无色的，为了除去这些干扰因素，必须从总吸光值中减去各自的空白值（比色管A和比色管B）。

试样净吸光值按式(1)计算：

$$
A = A _ {\mathrm {t}} - A _ {\mathrm {b}} - A _ {\mathrm {e}} \quad \dots \tag {1}
$$

式中：

$A$ ——试样净吸光值；

$A_{\mathrm{t}}$ ——总吸光值（比色管C）；

$A_{\mathrm{b}}$ ——试剂吸光值（比色管A）；

$A_{\mathrm{e}}$ ——滤液吸光值（比色管B）。

$A_{\mathrm{b}}$ 和 $A_{\mathrm{e}}$ 不应大于总吸光值的 $20\%$ 。

# 6.2 胆碱含量的计算

试样中的胆碱（以胆碱氢氧化物计）的含量按式(2)计算：

$$
X = \frac {c \times V \times f \times 1 0 0}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中的胆碱(以胆碱氢氧化物计)含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$c$ ——自标准曲线上查得净吸光值对应的胆碱的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

V ——水解液被稀释的体积(水解液定容体积, 通常为 $50 \mathrm{~mL}$ ), 单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释倍数；

100 ——换算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当固态或半固态试样称样量为 $5\mathrm{g}$ 时，方法的检出限为 $1\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $3\mathrm{mg} / 100\mathrm{g}$ ；当液态试样称样量为 $20\mathrm{g}$ 时，方法的检出限为 $0.3\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.8\mathrm{mg} / 100\mathrm{g}$ 。

# 第二法 离子色谱法

# 9 原理

试样经酸水解，固相萃取柱净化后离子色谱分离，电导检测器检测，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 浓盐酸（HCl）。

10.1.2 甲烷磺酸 $(\mathrm{CH}_4\mathrm{O}_3\mathrm{S})$ ：色谱纯。

# 10.2 试剂配制

10.2.1 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $85~\mathrm{mL}$ 浓盐酸注入约 $900~\mathrm{mL}$ 水中，稀释至 $1000~\mathrm{mL}$ 。

10.2.2 盐酸溶液 $(1.7\mathrm{mol} / \mathrm{L})$ ：量取 $145~\mathrm{mL}$ 浓盐酸注入约 $800~\mathrm{mL}$ 水中，稀释至 $1000~\mathrm{mL}$ 。

10.2.3 甲烷磺酸溶液 $(6\mathrm{mmol / L})$ ：吸取 $0.390~\mathrm{mL}$ 甲烷磺酸并稀释至 $1000~\mathrm{mL}$ 。

10.2.4 甲烷磺酸溶液( $15\mathrm{mmol / L}$ )：吸取 $0.974~\mathrm{mL}$ 甲烷磺酸并稀释至 $1000~\mathrm{mL}$

10.2.5 甲烷磺酸溶液 $(25\mathrm{mmol / L})$ ：吸取 $1.62~\mathrm{mL}$ 甲烷磺酸并稀释至 $1000~\mathrm{mL}$

# 10.3 标准品

胆碱酒石酸氢盐标准品 $\mathrm{(C_9H_{19}NO_7}$ ，相对分子质量：253.25，CAS号：87-67-2）：纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 10.4 标准溶液配制

10.4.1 胆碱(以胆碱氢氧化物计，相对分子质量121.18)标准储备液 $(2500\mathrm{mg / L})$ ：准确称取在 $102^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 烘至恒重的胆碱酒石酸氢盐 $522.5\mathrm{mg}$ ，用水溶解并转移至 $100~\mathrm{mL}$ 容量瓶中定容，混匀。 $4^{\circ}\mathrm{C}$ 以下避光保存，有效期3个月。  
10.4.2 胆碱标准工作液 $(100\mathrm{mg / L})$ ：吸取 $2.0~\mathrm{mL}$ 上述胆碱标准储备液于 $50~\mathrm{mL}$ 容量瓶中，用 $15\mathrm{mmol / L}$ 的甲烷磺酸溶液定容，混匀， $4^{\circ}C$ 以下避光保存，有效期1个月。  
10.4.3 胆碱标准系列工作液：分别吸取上述胆碱标准工作液 $0.200\mathrm{mL}, 0.500\mathrm{mL}, 1.00\mathrm{mL}, 2.50\mathrm{mL}$

$5.00\mathrm{mL}$ 于一组 $100~\mathrm{mL}$ 容量瓶中，用 $15\mathrm{mmol / L}$ 的甲烷磺酸溶液定容混匀，配得胆碱浓度为 $0.200\mathrm{mg / L},0.500\mathrm{mg / L},1.00\mathrm{mg / L},2.50\mathrm{mg / L},5.00\mathrm{mg / L}$ 的标准系列工作液。临用现配。

# 10.5 材料

10.5.1 $0.45\mu \mathrm{m}$ 水性滤膜针头滤器。  
10.5.2 净化柱： $\mathrm{C_{18}}$ 固相萃取小柱 $1.0~\mathrm{mL}$ 或相当者。  
10.5.3 注射器： $5\mathrm{mL}$ 或相当者。

# 11 仪器和设备

11.1 离子色谱仪(IC)：配电导检测器。  
11.2 分析天平：感量分别为 $0.1\mathrm{mg}$ 和 $0.001\mathrm{g}$   
11.3 电热恒温水浴装置：温度可控制在 $70^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。  
11.4 涡旋混合器。

# 12 分析步骤

# 12.1 样品前处理

# 12.1.1 试样提取

# 12.1.1.1 液态试样

准确称取 $10\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 混合均匀的液体试样于 $50~\mathrm{mL}$ 比色管中，加入 $1.7\mathrm{mol} / \mathrm{L}$ 的盐酸溶液 $15\mathrm{mL}$ ，加盖，混匀后放入 $70^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 水浴中水解 $3\mathrm{h}$ （每 $30\mathrm{min}$ 摇一次）。将水解液冷却至室温，转移至 $50~\mathrm{mL}$ 容量瓶并用水定容至刻度，混匀待用。

# 12.1.1.2 半固态或固态试样

准确称取 $2.5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）半固态或固体试样于 $50~\mathrm{mL}$ 比色管中，加入 $1\mathrm{mol} / \mathrm{L}$ 的盐酸溶液 $25~\mathrm{mL}$ ，加盖，涡旋至试样溶液中无结块，混匀后放入 $70^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 水浴中水解 $3\mathrm{h}$ （每 $30\mathrm{min}$ 摇一次）。将水解液冷却至室温，转移至 $50~\mathrm{mL}$ 容量瓶并用水定容至刻度，混匀待用。

# 12.1.2 试样净化

$\mathrm{C_{18}}$ 固相萃取小柱 $(1.0~\mathrm{mL})$ 使用前依次用 $10~\mathrm{mL}$ 甲醇、 $15~\mathrm{mL}$ 水通过，静置活化 $30\mathrm{min}$ 。用水稀释提取溶液50倍(可根据试样中胆碱的浓度适当的调整稀释倍数，不得少于10倍)，取稀释后的溶液约 $15~\mathrm{mL}$ 过 $0.45\mu \mathrm{m}$ 水性滤膜和 $\mathrm{C_{18}}$ 固相萃取小柱 $(1.0~\mathrm{mL})$ ，弃去前 $3\mathrm{mL}$ ，收集后面洗脱液待测。

# 12.2 仪器参考条件

a）离子色谱柱参数：装有羧基的高容量阳离子交换柱，如IonPac CS12A $4\mathrm{mm}\times 250\mathrm{mm}$ （带IonPac CG12A型保护柱 $4\mathrm{mm}\times 50\mathrm{mm}$ )或IonPac CS19 $4\mathrm{mm}\times 250\mathrm{mm}$ （带IonPac CG19型保护柱 $4\mathrm{mm}\times 50\mathrm{mm}$ )，或等效色谱柱。  
b) IonPac CS12A 等度淋洗: $15 \mathrm{mmol} / \mathrm{L}$ 甲烷磺酸溶液等度淋洗, 采集时间 $25 \mathrm{~min}$ 。 IonPac CS19 等度淋洗: $6 \mathrm{mmol} / \mathrm{L}$ 甲烷磺酸溶液等度淋洗, 采集时间 $25 \mathrm{~min}$ 。

c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ 。  
d）电导检测器：配抑制器或等效抑制装置。  
e) 进样量: $100 \mu \mathrm{L}$ 。

# 12.3 标准曲线的制作

将标准系列工作液分别注入离子色谱仪，测定相应的电导率峰面积或峰高。以标准系列工作液的浓度为横坐标，以电导率峰面积或峰高为纵坐标，绘制标准曲线。

胆碱标准溶液的离子色谱图参见附录A。

# 12.4 试样溶液的测定

将试样溶液注入离子色谱仪中，得到相应的电导率峰面积或峰高，根据标准曲线得到待测溶液中胆碱的浓度。

# 12.5 空白试验

不称取试样，按照12.1的步骤做空白试验。应确认不含有干扰待测组分的物质。

# 13 分析结果的表述

试样中胆碱（以胆碱氢氧化物计）的含量按式(3)计算：

$$
X = \frac {c \times V \times f \times 1 0 0}{m \times 1 0 0 0} \quad \dots \tag {3}
$$

式中：

$X$ ——试样中胆碱的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$c$ ——由标准曲线得到的试样溶液中胆碱(以胆碱氢氧化物计)的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

V —定容的体积，单位为毫升 $(\mathrm{mL})$

$f$ ——稀释倍数；

100 ——换算系数；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

当固态或半固态试样称样量为 $2.5\mathrm{g}$ 时，方法的检出限为 $2\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $6\mathrm{mg} / 100\mathrm{g}$ ；当液态试样称样量为 $10\mathrm{g}$ 时，方法的检出限为 $0.5\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $2\mathrm{mg} / 100\mathrm{g}$ 。

# 第三法 液相色谱-串联质谱法

# 16 原理

试样经酸水解后调节 $\mathrm{pH}$ 并过滤，液相色谱-串联质谱测定和确认，同位素内标法定量。

# 17 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 17.1 试剂

17.1.1 甲酸（HCOOH）：色谱纯。  
17.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
17.1.3 甲酸铵 $\mathrm{(HCOONH_4)}$ ：纯度 $\geqslant 99.9\%$   
17.1.4 浓盐酸 $(\mathrm{HCl})$ 。  
17.1.5 氢氧化钠 $(\mathrm{NaOH})$ ：纯度 $\geqslant 99.9\%$

# 17.2 试剂配制

17.2.1 甲酸铵水溶液 $(10\mathrm{mmol / L})$ ：称取 $0.63\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )甲酸铵，用水溶解后，用甲酸调节 $\mathrm{pH}$ 至 $5.0\pm 0.1$ ，转移至 $1000~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。

# 20 分析结果的表述

试样中胆碱（以胆碱氢氧化物计）的含量按式(4)计算：

$$
X = \frac {c \times V \times f \times 1 0 0}{m \times 1 0 0 0} \tag {4}
$$

式中：

$X$ ——试样中胆碱的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$c$ ——根据标准曲线计算得到的试样溶液中胆碱(以胆碱氢氧化物计)的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样经温水溶解后定容的体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释倍数，其值为1000；

100 ——换算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果保留三位有效数字。

# 21 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

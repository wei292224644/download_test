# 食品安全国家标准

# 粮谷及油籽中二氯喹啉酸残留量的测定 气相色谱法

# 1 范围

本标准规定了粮谷及油籽中二氯喹啉酸残留量的气相色谱测定方法。

本标准适用于糙米、大豆、玉米、小麦中二氯喹啉酸残留量的测定。其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB/T6682 分析实验室用水规格和试验方法。

# 3 原理

试样用丙酮提取，旋转蒸发至干后用水溶解，二氯甲烷液液萃取净化后甲酯化，然后经硅酸镁柱净化，气相色谱仪测定，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ ：色谱纯。  
4.1.2 碳酸氢钠 $\mathrm{(NaHCO_3)}$ ：色谱纯。  
4.1.3 氢氧化钠(NaOH)。  
4.1.4 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：色谱纯。  
4.1.5 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。  
4.1.6 无水硫酸钠 $(\mathrm{Na_2SO_4})$ ：经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置干燥器中备用。  
4.1.7 石油醚。  
4.1.8 乙醚 $(\mathrm{C_4H_{10}O})$ ：重蒸馏。  
4.1.9 氢氧化钾(KOH)。

# 4.2 溶液配制

4.2.1 硫酸溶液： $3\mathrm{mol} / \mathrm{L}$   
4.2.2 氢氧化钠溶液： $1\mathrm{mol} / \mathrm{L}$   
4.2.3 氢氧化钾溶液： $10\mathrm{mol} / \mathrm{L}$   
4.2.4 重氮甲烷：置 $4\mathrm{mL}$ 乙醚和 $2\mathrm{mL}$ 氢氧化钾溶液于反应管中，加入 $5\mathrm{mL}$ 乙醚溶解的 $2\mathrm{gN}$ -甲基- $\mathrm{N}$ -亚硝基-4-甲苯磺酸胺的溶液，平稳吹氮气 $5\mathrm{min}$ ，收集反应管中乙醚溶液。

# 4.3 标准品

二氯喹啉酸标准品：纯度 $\geq 99\%$

# 4.4 标准溶液配制

4.4.1 二氯喹啉酸标准储备溶液：准确称取适量的二氯喹啉酸标准品，用少量丙酮溶解，并以石油醚配制成 $1000\mu \mathrm{g} / \mathrm{mL}$ 的标准储备液，避光于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 保存。  
4.4.2 二氯喹啉酸标准工作液：根据工作需要用石油醚稀释成适用浓度的标准工作溶液。

# 4.5 材料

硅酸镁固相萃取柱： $6\mathrm{mL} / 1\mathrm{g}$ ，或相当者。

# 5 仪器和设备

5.1 气相色谱仪，配电子俘获检测器。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.3 振荡器。  
5.4 旋转蒸发仪。  
5.5 分液漏斗。  
5.6 固相萃取装置，带真空泵。

# 6 试样制备与保存

# 6.1 试样制备

取有代表性样品 $500 \mathrm{~g}$ , 取样部位按 GB 2763 的规定执行, 用粉碎机粉碎。混匀, 均分成 2 份作为试样, 装入洁净的盛样容器内, 密封并标明标记。在取样和制样的操作过程中, 应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样保存

试样于 $-5^{\circ} \mathrm{C}$ 保存。

# 7 分析步骤

# 7.1 提取

7.1.1 称取 $20 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ ) 试样于 $250 \mathrm{~mL}$ 具塞锥形瓶中, 加入 $100 \mathrm{~mL}$ 丙酮, 振荡提取 $30 \mathrm{~min}$ , 将提取液过滤于 $250 \mathrm{~mL}$ 心形瓶中, 用 $50 \mathrm{~mL}$ 丙酮分 2 次洗涤残渣, 洗涤液经过滤后合并于上述心形瓶中于 $40^{\circ} \mathrm{C}$ 水浴中减压蒸发至近干, 加入 $10 \mathrm{~mL}$ 水和 $5 \mathrm{~g}$ 碳酸氢钠。溶解后, 加入约 $2 \mathrm{~mL}$ 氢氧化钠溶液, 调 $\mathrm{pH}$ 为 $9 \pm 0.2$ 。充分混合后移入分液漏斗中, 用 $20 \mathrm{~mL}$ 水分 2 次洗涤, 溶液合并于分液漏斗中。  
7.1.2 于7.1.1的分液漏斗中加入 $100\mathrm{mL}$ 二氯甲烷，振摇 $5\mathrm{min}$ ，静置分层，弃去下层有机相，再分2次，每次用 $50\mathrm{mL}$ 二氯甲烷重复洗涤水相。于水相中加入约 $5\mathrm{mL}$ 硫酸溶液，使 $\mathrm{pH}$ 为 $2 \pm 0.2$ 。加入 $100\mathrm{mL}$ 二氯甲烷，振荡提取 $5\mathrm{min}$ ，静置分层。分出下层有机相于锥形瓶中，水相再分2次，每次用 $50\mathrm{mL}$ 二氯甲烷重复提取，合并有机相。经无水硫酸钠脱水，置于具塞心形瓶中，于 $40^{\circ}\mathrm{C}$ 水浴中旋转浓缩至近干，加入 $2\mathrm{mL}$ 丙酮以溶解残渣。

# 7.2 衍生化

于7.1.2的丙酮溶液中加入 $5\mathrm{mL}$ 重氮甲烷溶液，密封，置 $60^{\circ}\mathrm{C}$ 水浴中反应 $10\mathrm{min}$ ，蒸除溶剂，用 $5\mathrm{mL}$ 石油醚溶解。

# 7.3 净化

将硅酸镁固相萃取柱安装在固相萃取装置上，先用 $6\mathrm{mL}$ 石油醚预淋洗，弃去。将溶液(7.2)倾入到固相萃取柱中，待全部流出后，再用 $10\mathrm{mL}$ 石油醚分2次淋洗固相萃取柱，弃去。用 $10\mathrm{mL}$ 丙酮一石油醚 $(9 + 1)$ 洗脱，保持流速 $1.5\mathrm{mL} / \mathrm{min}$ ，收集全部流出液， $45^{\circ}\mathrm{C}$ 下氮气流吹至近干。用石油醚溶解并移入

容量瓶中定容至 $1.0\mathrm{mL}$ ，供气相色谱仪测定。

# 7.4 标准物质衍生化

准确移取适当浓度的二氯喹啉酸标准工作液于具塞心形瓶中，于 $40^{\circ}\mathrm{C}$ 水浴中旋转浓缩至近干，加入 $2\mathrm{mL}$ 丙酮以溶解残渣，以下按7.2操作。

# 7.5 测定

# 7.5.1 气相色谱参考条件

a）色谱柱：BP10石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径），膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度： $240^{\circ}\mathrm{C}$   
c) 进样口温度: $280^{\circ} \mathrm{C}$ ;   
d) 检测器温度: $290^{\circ} \mathrm{C}$ ;   
e) 载气：氮气，纯度 $\geq 99.995\%$ ， $10~\mathrm{mL / min}$   
f）进样量： $1\mu \mathrm{L}$   
g) 尾吹气：氮气，纯度 $\geqslant 99.995\%$ ， $50~\mathrm{mL / min}$ 。

# 7.5.2 色谱测定

根据样液中被测农药的含量情况, 选定浓度相近的标准工作溶液。标准工作液和待测样液中甲基化农药的响应值均应在仪器检测的线性范围内。对标准工作溶液与样液等体积参插进样测定。在上述色谱条件下, 二氯喹啉酸甲酯的保留时间约为 $3 \mathrm{~min}$ 。标准品的气相色谱图参见图 A.1。

# 7.6 空白实验

除不加试样外，均按 $7.1\sim 7.5$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中二氯喹啉酸农药的含量。

$$
X = \frac {h \times c \times V}{h _ {s} \times m} \tag {1}
$$

式中：

$X$ ——试样中二氯喹啉酸残留含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$h$ ——样液中二氯喹啉酸甲酯的色谱峰高，单位为毫米 $(\mathrm{mm})$ ；

$c$ ——标准工作液中二氯喹啉酸的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ —标液最终定容体积，单位为毫升 $(\mathrm{mL})$

$h_s$ ——标准工作液中二氯喹啉酸甲酯的色谱峰高，单位为毫米（ $\mathrm{mm}$ ）；

$m$ ——最终样液所代表的试样量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 B 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 C 的要求。

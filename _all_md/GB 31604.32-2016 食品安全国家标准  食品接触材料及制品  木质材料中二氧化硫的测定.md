# 食品安全国家标准

# 食品接触材料及制品 木质材料中二氧化硫的测定

# 1 范围

本标准规定了食品接触木(竹)质材料中二氧化硫滴定法和比色法的测定方法。

本标准适用于木(竹)筷、木(竹)碗、木(竹)勺、木(竹)铲中二氧化硫的测定。

# 第一法 滴定法

# 2 原理

在密闭容器中对试样进行酸化并加热蒸馏，以释放出其中的二氧化硫，释放物用乙酸铅溶液吸收。吸收后用浓盐酸酸化，再以碘标准溶液滴定，根据所消耗的碘标准溶液量计算出试样中二氧化硫的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 盐酸（HCl）。  
3.1.2 乙酸铅 $\left[\mathrm{Pb}(\mathrm{CH}_3\mathrm{COO})_2\cdot 3\mathrm{H}_2\mathrm{O}\right]$   
3.1.3 碘 $(\mathrm{I}_2)$ 。  
3.1.4 可溶性淀粉 $\left[\left(\mathrm{C}_{6} \mathrm{H}_{10} \mathrm{O}_{5}\right)_{n}\right]$

# 3.2 试剂配制

3.2.1 盐酸溶液 $(1 + 1$ ，体积比)：取盐酸 $100~\mathrm{mL}$ 小心加入 $100~\mathrm{mL}$ 水中，混匀。  
3.2.2 乙酸铅溶液 $(20\mathrm{g / L})$ ：称取 $20\mathrm{g}$ 乙酸铅，溶于少量水中并稀释至 $1000~\mathrm{mL}$ 。  
3.2.3 淀粉指示剂 $(10\mathrm{g / L})$ ：称取 $1\mathrm{g}$ 可溶性淀粉，用少许水调成糊状后，再缓缓倾入 $100~\mathrm{mL}$ 沸水中，随加随搅拌，煮沸 $2\mathrm{min}$ ，冷却至室温，备用，此溶液现用现配。

# 3.3 标准品

碘标准溶液 $\left[\left(\frac{1}{2}\mathrm{I}_2\right) = 0.1\mathrm{mol} / \mathrm{L}\right]$ ：按GB/T601配制与标定或商品化产品。

# 3.4 标准溶液配制

碘标准滴定溶液 $\left[\left(\frac{1}{2} I_{2}\right)=0.01 \mathrm{~mol} / \mathrm{L}\right]$ ：吸取 $10.0 \mathrm{~mL}$ 碘标准溶液，置于 $100 \mathrm{~mL}$ 容量瓶中，用水稀

释定容至刻度。

# 4 仪器和设备

4.1 分析天平：感量 $0.1\mathrm{mg}$   
4.2 全玻璃蒸馏器。  
4.3 酸式滴定管。  
4.4 碘量瓶。  
4.5 电刨或电锯。  
4.6 粉碎机。

# 5 分析步骤

# 5.1 试样制备

将抽取的样品，用电刨或电锯取其表面 $3\mathrm{mm}$ 厚，粉碎，装入洁净容器内。密封作为试样，标明标记。在制样的操作过程中，应防止样品受到二氧化硫的污染。

# 5.2 蒸馏

称取试样 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），置于 $500~\mathrm{mL}$ 圆底烧瓶。在圆底烧瓶中加入 $250~\mathrm{mL}$ 水，为防止样液暴沸，可加入少许玻璃珠。装上冷凝装置，冷凝管下端插入碘量瓶中的 $25~\mathrm{mL}$ 乙酸铅溶液 $(20~\mathrm{g / L})$ 中，然后在蒸馏瓶中加入 $10~\mathrm{mL}$ 盐酸溶液 $(1 + 1$ ，体积比)，立即盖上塞子，加热蒸馏。当蒸馏液约达 $200~\mathrm{mL}$ 时，使冷凝管下端离开液面，再继续蒸馏 $1\mathrm{min}$ 。用少量蒸馏水冲洗插入乙酸铅溶液 $(20~\mathrm{g / L})$ 的装置部分。在检测试样的同时做空白试验，空白试验的测定从“在圆底烧瓶中加入 $250~\mathrm{mL}$ 水”做起。

# 5.3 滴定

向取下的碘量瓶中依次加入 $10~\mathrm{mL}$ 盐酸、 $1\mathrm{mL}$ 淀粉指示剂 $(10\mathrm{g / L})$ ，摇匀后用碘标准滴定溶液 $(0.01\mathrm{mol} / \mathrm{L})$ 滴定至溶液变蓝且在 $30\mathrm{s}$ 内不褪色为止。

# 6 分析结果的表述

试样中二氧化硫的含量按式(1)计算：

$$
X = \frac {\left(V _ {1} - V _ {2}\right) \times c \times 0 . 0 3 2 0 3 \times 1 0 0 0 \times 1 0 0 0}{m} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中二氧化硫的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$V_{1}$ ——滴定试样所用碘标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴定试剂空白所用碘标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——碘标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

0.032 03—1 mL 碘标准溶液 $\left[c\left(\frac{1}{2} I_{2}\right)=1.0 \mathrm{~mol} / \mathrm{L}\right]$ 相当的二氧化硫的质量，单位为克每毫摩尔 $(\mathrm{g} / \mathrm{mmol})$ ；

1000 ——换算系数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

方法检出限为 $10\mathrm{mg / kg}$

# 第二法 比色法

# 9 原理

在密闭容器中对试样进行酸化并加热蒸馏，以释放出其中的二氧化硫，二氧化硫被四氯汞钾吸收后形成稳定的络合物，再与甲醛及盐酸副玫瑰苯胺作用生成玫瑰紫色化合物，比色定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。四氯汞钾液有剧毒性，操作时应戴橡胶手套，若溅到皮肤上，立即用水冲洗。

# 10.1 试剂

10.1.1 氯化汞（ $\mathrm{HgCl_2}$ ）。  
10.1.2 乙二胺四乙酸二钠盐 $(\mathrm{Na}_{2}\cdot$ EDTA）。  
10.1.3 氯化钾(KCl)。  
10.1.4 氨基磺酸 $(\mathrm{H}_3\mathrm{NO}_3\mathrm{S})$ 。  
10.1.5 甲醛（HCHO）：含量 $36\% \sim 38\%$   
10.1.6 盐酸副玫瑰苯胺 $\mathrm{(C_{19}H_{18}ClN_3}$ ；CASNo.569-61-9)：纯度不得低于 $95\%$ 。  
10.1.7 盐酸（HCl）。  
10.1.8 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。

# 10.2 试剂配制

10.2.1 四氯汞钾吸收液 $(0.04\mathrm{mol} / \mathrm{L})$ ：称取 $10.86\mathrm{g}$ 氯化汞、 $0.066\mathrm{g}$ 乙二胺四乙酸二钠盐、 $5.96\mathrm{g}$ 氯化钾，溶于水中并稀释至 $1000~\mathrm{mL}$ 。  
10.2.2 氨基磺酸溶液 $(6\mathrm{g} / \mathrm{L})$ ：称取 $0.6\mathrm{g}$ 氨基磺酸，溶于 $100~\mathrm{mL}$ 水中，临用配制。  
10.2.3 甲醛溶液 $(2\mathrm{g} / \mathrm{L})$ ：吸取 $1\mathrm{mL}$ 甲醛，用水稀释至 $200~\mathrm{mL}$ ，储存在具塞玻璃瓶中，临用配制。  
10.2.4 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：取盐酸 $90~\mathrm{mL}$ 用水稀释到 $1000~\mathrm{mL}$ 。  
10.2.5 盐酸溶液 $(1 + 1)$ ：取盐酸 $100~\mathrm{mL}$ 小心加入 $100~\mathrm{mL}$ 水中。  
10.2.6 磷酸溶液 $(3\mathrm{mol} / \mathrm{L})$ ：取 $20.5\mathrm{mL}$ 磷酸用水稀释到 $100~\mathrm{mL}$ 。  
10.2.7 盐酸副玫瑰苯胺储备液 $(2\mathrm{g} / \mathrm{L})$ ：精确称取 $0.2\mathrm{g}$ 盐酸副玫瑰苯胺，溶于 $100~\mathrm{mL}$ 盐酸溶液中。  
10.2.8 盐酸副玫瑰苯胺工作液 $(0.16\mathrm{g / L})$ ：取 $20~\mathrm{mL}$ 盐酸副玫瑰苯胺储备液和 $25~\mathrm{mL}$ 磷酸溶液于 $250~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，此溶液于暗处保存可放置6个月。

# 10.3 标准溶液配制

10.3.1 二氧化硫标准溶液：配制见附录A。  
10.3.2二氧化硫标准使用液：按标定计算结果，用四氯汞钾吸收液(10.2.1)将二氧化硫标准溶液(10.3.1)稀释成 $1\mathrm{mL}$ 相当于 $5\mu \mathrm{g}$ 二氧化硫，于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 下可保存1个月。

# 11 仪器和设备

11.1 紫外可见分光光度计。  
11.2 分析天平：感量 $0.1\mathrm{mg}$   
11.3 全玻璃蒸馏器。  
11.4 碘量瓶。  
11.5 电刨或电锯。  
11.6 粉碎机。  
11.7 水浴锅。

# 12 分析步骤

# 12.1 试样制备

将抽取的样品，用电刨或电锯取其表面 $3\mathrm{mm}$ 厚，粉碎，装入洁净容器内。密封作为试样，标明标记。在制样的操作过程中，应防止样品受到二氧化硫的污染。

# 12.2 蒸馏

称取试样 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）（试样量视产品二氧化硫含量高低而定），置于 $500~\mathrm{mL}$ 圆底烧瓶。在圆底烧瓶中加入 $250~\mathrm{mL}$ 水，为防止样液暴沸，可加入玻璃珠少许，装入冷凝装置，冷凝管下端应插入碘量瓶中的 $25~\mathrm{mL}$ 四氯汞钾吸收液(10.2.1)中，然后在蒸馏瓶中加入 $10~\mathrm{mL}$ 盐酸溶液(10.2.5)，立即盖上塞子，加热蒸馏。当蒸馏液约达 $200~\mathrm{mL}$ 时，使冷凝管下端离开液面，再蒸馏 $1\mathrm{min}$ 。用少量蒸馏水冲洗插入四氯汞钾吸收液(10.2.1)的装置部分。将蒸馏液转移至 $250~\mathrm{mL}$ 容量瓶中，用新煮沸放冷的水加至刻度。在检测试样的同时做空白试验，空白试验的测定从“在圆底烧瓶中加入 $250~\mathrm{mL}$ 水”做起。

# 12.3 测定

吸取 $0.5\mathrm{mL}\sim 5.0\mathrm{mL}$ 上述试样处理液于 $25~\mathrm{mL}$ 容量瓶中。另吸取 $0\mathrm{mL},0.20\mathrm{mL},0.40\mathrm{mL}$ $0.60~\mathrm{mL},0.80~\mathrm{mL},1.50~\mathrm{mL},2.00~\mathrm{mL}$ 二氧化硫标准使用液(10.3.2）（相当于 $0\mu \mathrm{g},1.0\mu \mathrm{g},2.0\mu \mathrm{g}$ $3.0\mu \mathrm{g},5.0\mu \mathrm{g},7.5\mu \mathrm{g},10\mu \mathrm{g}$ 二氧化硫)，分别置于 $25~\mathrm{mL}$ 容量瓶中。向样液、空白和标液容量瓶中各加入 $1.0~\mathrm{mL}$ 氨基磺酸溶液(10.2.2)混匀，放置 $10\mathrm{min}$ ，再准确加入 $2.0~\mathrm{mL}$ 甲醛溶液(10.2.3)和 $1.0~\mathrm{mL}$ 盐酸副玫瑰苯胺工作液(10.2.8)，并用新煮沸放冷的水加至刻度，摇匀，放在恒温水浴中 $25^{\circ}C\pm 1^{\circ}C$ 反应 $30\mathrm{min}$ ，用 $1\mathrm{cm}$ 比色杯，以零管调节零点，于波长 $548~\mathrm{nm}$ 处测吸光度，绘制标准曲线比较。

# 13 分析结果的表述

试样中二氧化硫的含量按式(2)计算：

$$
X = \frac {\left(m _ {1} - m _ {2}\right) \times 1 0 0 0}{m \times V / 2 5 0 \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中二氧化硫的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$m_{1}$ ——测定用样液中二氧化硫的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_{2}$ ——测定用空白液中二氧化硫的质量，单位为微克 $(\mu \mathrm{g})$ ；

1000——换算系数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

V 测定用样液的体积，单位为毫升 $(\mathrm{mL})$

250 ——换算系数，蒸馏液转入容量瓶中定容 $250~\mathrm{mL}$ 。

计算结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

方法检出限为 $1\mathrm{mg / kg}$ ，定量限为 $5\mathrm{mg / kg}$

# 二氧化硫标准溶液的配制

称取 $0.2\mathrm{g}$ 亚硫酸钠 $\mathrm{(NaSO_3)}$ 溶于 $250~\mathrm{mL}$ 四氯汞钾吸收液(10.2.1)中，放置过夜，上清液用定量滤纸过滤后标定，此液为二氧化硫储备液。

吸取 $20.0\mathrm{mL}$ 亚硫酸钠溶液于 $250~\mathrm{mL}$ 碘量瓶中，加 $100~\mathrm{mL}$ 水，准确加入 $20.00~\mathrm{mL}$ 碘标准溶液 $(0.100\mathrm{mol} / \mathrm{L})$ 、 $5\mathrm{mL}$ 冰乙酸，摇匀，放置于暗处， $2\mathrm{min}$ 后迅速以硫代硫酸钠标准溶液 $(0.100\mathrm{mol} / \mathrm{L})$ 滴定至淡黄色，加 $0.5\mathrm{mL}$ 淀粉指示液，继续滴至无色。另取 $100~\mathrm{mL}$ 水，准确加入 $20.00~\mathrm{mL}$ 碘标准溶液 $(0.100\mathrm{mol} / \mathrm{L})$ 、 $5\mathrm{mL}$ 冰乙酸，按统一方法做试剂空白试验。

# 前言

本标准代替GB/T5009.26—2003《食品中 $N$ -亚硝胺类的测定》。

本标准与GB/T5009.26—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中 $N$ -亚硝胺类化合物的测定”；  
——将原方法中的填充色谱柱修改为毛细管色谱柱；  
——将原方法中的气相色谱高分辨质谱仪修改为气相色谱质谱仪。

# 食品安全国家标准

# 食品中 $N$ -亚硝胺类化合物的测定

# 1 范围

本标准规定了食品中 $N\cdot$ 二甲基亚硝胺含量的测定方法。

本标准适用于肉及肉制品、水产动物及其制品中 $N\cdot$ 二甲基亚硝胺含量的测定。

# 第一法 气相色谱-质谱法

# 2 原理

试样中的 $N$ -亚硝胺类化合物经水蒸气蒸馏和有机溶剂萃取后，浓缩至一定体积，采用气相色谱-质谱联用仪进行确认和定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 二氯甲烷 $\left(\mathrm{CH}_{2} \mathrm{Cl}_{2}\right)$ : 色谱纯, 每批应取 $100 \mathrm{~mL}$ 在 $40^{\circ} \mathrm{C}$ 水浴上用旋转蒸发仪浓缩至 $1 \mathrm{~mL}$ , 在气相色谱-质谱联用仪上应无阳性响应, 如有阳性响应, 则需经全玻璃装置重蒸后再试, 直至阴性。  
3.1.2 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.3 氯化钠(NaCl)：优级纯。  
3.1.4 硫酸 $\mathrm{(H_2SO_4)}$ 。  
3.1.5 无水乙醇 $\mathrm{(C_2H_5OH)}$

# 3.2 试剂配制

硫酸溶液 $(1 + 3)$ ：量取 $30~\mathrm{mL}$ 硫酸，缓缓倒入 $90~\mathrm{mL}$ 冷水中，一边搅拌使得充分散热，冷却后小心混匀。

# 3.3 标准品

$N$ -亚硝胺标准品 $\mathrm{(C_2H_6N_2O}$ ，CAS号：62-75-9）：纯度 $\geq 98.0\%$ □

# 3.4 $N$ -亚硝胺标准溶液的制备

3.4.1 $N$ -亚硝胺 $\mathrm{(C_2H_6N_2O}$ , CAS号: 62-75-9)标准溶液: 用二氯甲烷配制成 $1\mathrm{mg/mL}$ 的溶液。  
3.4.2 $N$ -亚硝胺标准中间液 $\mathrm{(C_2H_6N_2O}$ ,CAS号：62-75-9)标准使用液：用二氯甲烷配制成 $1\mu \mathrm{g / mL}$ 的标准使用液。

# 4 仪器和设备

4.1 气相色谱-质谱联用仪。  
4.2 旋转蒸发仪。  
4.3 全玻璃水蒸气蒸馏装置或等效的全自动水蒸气蒸馏装置（见图A.1）。  
4.4 氮吹仪。  
4.5 制冰机。  
4.6 电子天平：感量为 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$ 。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 提取

水蒸馏装置蒸馏：准确称取 $200\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）试样，加入 $100~\mathrm{mL}$ 水和 $50~\mathrm{g}$ 氯化钠于蒸馏管中，充分混匀，检查气密性。在 $500~\mathrm{mL}$ 平底烧瓶中加入 $100~\mathrm{mL}$ 二氯甲烷及少量冰块用以接收冷凝液，冷凝管出口伸入二氯甲烷液面下，并将平底烧瓶置于冰浴中，开启蒸馏装置加热蒸馏，收集 $400~\mathrm{mL}$ 冷凝液后关闭加热装置，停止蒸馏。

# 5.1.2 萃取净化

在盛有蒸馏液的平底烧瓶中加入 $20\mathrm{g}$ 氯化钠和 $3\mathrm{mL}$ 的硫酸 $(1 + 3)$ ，搅拌使氯化钠完全溶解。然后将溶液转移至 $500~\mathrm{mL}$ 分液漏斗中，振荡 $5\mathrm{min}$ ，必要时放气，静置分层后，将二氯甲烷层转移至另一平底烧瓶中，再用 $150~\mathrm{mL}$ 二氯甲烷分三次提取水层，合并4次二氯甲烷萃取液，总体积约为 $250~\mathrm{mL}$ 。

# 5.1.3 浓缩

将二氯甲烷萃取液用 $10\mathrm{g}$ 无水硫酸钠脱水后，进行旋转蒸发，于 $40^{\circ}\mathrm{C}$ 水浴上浓缩至 $5\mathrm{mL} \sim 10\mathrm{mL}$ 改氮吹，并准确定容至 $1.0\mathrm{mL}$ ，摇匀后待测定。

# 5.2 气相色谱-质谱测定条件

# 5.2.1 气相色谱条件

毛细管气相色谱柱：INNOWAX石英毛细管柱(柱长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ）；进样口温度： $220^{\circ}\mathrm{C}$ ；程序升温条件：初始柱温 $40^{\circ}\mathrm{C}$ ，以 $10^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升至 $80^{\circ}\mathrm{C}$ ，以 $1^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升至 $100^{\circ}\mathrm{C}$ ，再以 $20^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升至 $240^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$ ；载气：氦气；流速： $1.0\mathrm{mL} / \mathrm{min}$ ；进样方式：不分流进样；进样体积： $1.0\mu \mathrm{L}$ 。

# 5.2.2 质谱条件

选择离子检测。 $9.9\mathrm{min}$ 开始扫描 $N$ -二甲基亚硝胺，选择离子为15.0，42.0，43.0，44.0，74.0；电子轰击离子化源(EI)，电压： $70~\mathrm{eV}$ ；离子化电流： $300~\mu \mathrm{A}$ ；离子源温度： $230^{\circ}C$ ；接口温度： $230^{\circ}C$ ；离子源真空度： $1.33\times 10^{-4}\mathrm{Pa}$ 。

# 5.3 标准曲线的制作

分别准确吸取 $N$ -亚硝胺的混合标准储备液 $(1\mu \mathrm{g} / \mathrm{mL})$ 配制标准系列的浓度为 $0.01~\mu \mathrm{g / mL}$ 、

$0.02\mu \mathrm{g / mL},0.05\mu \mathrm{g / mL},0.1\mu \mathrm{g / mL},0.2\mu \mathrm{g / mL},0.5\mu \mathrm{g / mL}$ 的混合标准系列溶液，进样分析，质谱图见图B.1，用峰面积对浓度进行线性回归，表明在给定的浓度范围内 $N$ -亚硝胺呈线性，回归方程分别为 $y$ 为峰面积， $x$ 为浓度 $(\mu \mathrm{g / mL})$ 。

# 5.4 试样溶液的测定

将试样溶液注入气相色谱-质谱联用仪中，得到某一特定监测离子的峰面积，根据标准曲线计算得到试样溶液中 $N$ -二甲基亚硝胺 $(\mu \mathrm{g} / \mathrm{mL})$ 。

# 6 分析结果的表述

试样中 $N$ -二甲基亚硝胺含量按式(1)计算：

$$
X = \frac {h _ {1}}{h _ {2}} \times \rho \times \frac {V}{m} \times 1 0 0 0 \tag {1}
$$

式中：

$X$ ——试样中 $N$ -二甲基亚硝胺的含量，单位为微克每千克或微克每升 $(\mu \mathrm{g} / \mathrm{kg}$ 或 $\mu \mathrm{g} / \mathrm{L})$ ；

$h_{1}$ ——浓缩液中该某一 $N$ -亚硝胺化合物的峰面积；

$h_{2}$ ——N-亚硝胺标准的峰面积；

$\rho$ ——标准溶液中 $N$ -亚硝胺化合物的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试液(浓缩液)的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量或体积，单位为克或毫升(g或 $\mathrm{mL}$ ）；

1000——换算系数。

结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

当取样量为 $200\mathrm{g}$ ，浓缩体积为 $1.0\mathrm{mL}$ 时，本方法的检出限为 $0.3\mu \mathrm{g / kg}$ ，定量限为 $1.0\mu \mathrm{g / kg}$ 。

# 第二法 气相色谱-热能分析仪法

# 9 原理

试样经水蒸气蒸馏，样品中的 $N$ -二甲基亚硝胺随着蒸气通过二氯甲烷吸收，再以二氯甲烷液液萃取、分离，供气相色谱-热能分析仪(GC-TEA)测定。

GC-TEA测定原理：自气相色谱柱分离后的 $N$ -二甲基亚硝胺在热解室中经特异性催化裂解产生一氧化氮(NO)基团，后者与臭氧反应生成激发态 $\mathrm{NO}^*$ 。当激发态 $\mathrm{NO}^*$ 返回基态时发射出近红外光 $(600\mathrm{nm}\sim 2800\mathrm{nm})$ ，并被光电倍增管检测( $600\mathrm{nm}\sim 800\mathrm{nm})$ 。由于特异性催化裂解，加上CTR过滤器除去杂质，使热能分析仪只能检测NO基团，而成为 $N$ -亚硝胺类化合物的特异性检测器。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 10.1 试剂

10.1.1 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：色谱纯，每批应取 $100~\mathrm{mL}$ 在 $40^{\circ}C$ 水浴上用旋转蒸发仪浓缩至 $1\mathrm{mL}$ ，在热能分析仪上应无阳性响应，如有阳性响应，则需经全玻璃装置重蒸后再试，直至阴性。  
10.1.2 氯化钠 $(\mathrm{NaCl})$ 。  
10.1.3 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
10.1.4 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。  
10.1.5 无水乙醇 $\mathrm{(C_2H_5OH)}$

# 10.2 试剂配制

硫酸溶液 $(1 + 3)$ ：量取 $30~\mathrm{mL}$ 硫酸，缓缓倒入 $90~\mathrm{mL}$ 冷水中，一边搅拌使得充分散热，冷却后小心混匀。

# 10.3 标准品

$N$ -亚硝胺标准品 $\mathrm{(C_2H_6N_2O}$ ，CAS号：62-75-9）：纯度 $\geq 98.0\%$ □

# 10.4 $N$ -亚硝胺标准溶液的制备

10.4.1 $N\cdot$ 亚硝胺 $\mathrm{(C_2H_6N_2O}$ ,CAS号：62-75-9)标准溶液：用二氯甲烷配制成 $1\mathrm{mg / mL}$ 的溶液。  
10.4.2 $N$ -亚硝胺标准中间液 $\mathrm{(C_2H_6N_2O}$ ,CAS号：62-75-9)标准使用液：用二氯甲烷配制成 $1\mu \mathrm{g / mL}$ 的标准使用液。

# 11 仪器和设备

11.1 气相色谱仪(GC)。  
11.2 热能分析仪（TEA）。  
11.3 全玻璃水蒸气蒸馏装置或等效的全自动水蒸气蒸馏装置(见图A.1)。  
11.4 旋转蒸发仪。  
11.5 氮吹仪。  
11.6 制冰机。  
11.7 电子天平：感量为 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$ 。

# 12 分析步骤

# 12.1 试样制备

# 12.1.1 提取

水蒸馏装置蒸馏：准确称取 $200\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）试样，加入 $100~\mathrm{mL}$ 水和 $50\mathrm{g}$ 氯化钠于蒸馏管中，充分混匀，检查气密性。在 $500~\mathrm{mL}$ 平底烧瓶中加入 $100~\mathrm{mL}$ 二氯甲烷及少量冰块用以接收冷凝液，冷凝管出口伸入二氯甲烷液面下，并将平底烧瓶置于冰浴中，开启蒸馏装置加热蒸馏，收集 $400~\mathrm{mL}$ 冷凝

液后关闭加热装置，停止蒸馏。

# 12.1.2 萃取净化

在盛有蒸馏液的平底烧瓶中加入 $20\mathrm{g}$ 氯化钠和 $3\mathrm{mL}$ 硫酸 $(1 + 3)$ ，搅拌使氯化钠完全溶解。然后将溶液转移至 $500~\mathrm{mL}$ 分液漏斗中，振荡 $5\mathrm{min}$ ，必要时放气，静置分层后，将二氯甲烷层转移至另一平底烧瓶中，再用 $150~\mathrm{mL}$ 二氯甲烷分三次提取水层，合并4次二氯甲烷萃取液，总体积约为 $250~\mathrm{mL}$ 。

# 12.1.3 浓缩

将二氯甲烷萃取液用 $10\mathrm{g}$ 无水硫酸钠脱水后，进行旋转蒸发，于 $40^{\circ}\mathrm{C}$ 水浴上浓缩至 $5\mathrm{mL} \sim 10\mathrm{mL}$ 后改氮吹，并准确定容至 $1.0\mathrm{mL}$ ，摇匀后待测定。

# 12.2 仪器参考条件

# 12.2.1 气相色谱条件

气相色谱条件列出如下：

a）色谱柱：VF-WAX毛细管色谱柱(柱长 $30\mathrm{m}$ ，内径为 $0.32\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ，固定液为聚乙二醇)或相当型号色谱柱；  
b）进样口温度： $120^{\circ}\mathrm{C}$   
c）程序升温条件：初始温度 $60^{\circ}\mathrm{C}$ 保留 $2\mathrm{min},6^{\circ}\mathrm{C / min}$ 升至 $150~^\circ \mathrm{C},20~^\circ \mathrm{C / min}$ 升至 $200^{\circ}C$ ，保留 $5\mathrm{min}$   
d) 载气：氮气 $(\geq 99.999\%)$ ；  
e) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
f）进样方式：不分流进样；  
g）进样体积： $2\mu \mathrm{L}$

# 12.2.2 热能分析仪条件

热能分析仪条件列出如下：

a） 接口温度： $250~\mathrm{^\circ C}$   
b）热解室温度： $500^{\circ}\mathrm{C}$   
c）真空度： $59.85\mathrm{Pa}\sim 66.5\mathrm{Pa}$   
d) 氧气压力: $13.79 \mathrm{kPa}$ ;   
e) 臭氧水平：244(22.8V)。

# 12.3 标准曲线的制作

分别准确吸取 $200\mu \mathrm{L},400\mu \mathrm{L},600\mu \mathrm{L},800\mu \mathrm{L},1000\mu \mathrm{L}N-$ 二甲基亚硝胺标准工作液（ $1\mu \mathrm{g / mL}$ 于 $1.0~\mathrm{mL}$ 容量瓶中用二氯甲烷定容至 $1.0~\mathrm{mL}$ ，其浓度分别为 $0.2\mu \mathrm{g / mL},0.4\mu \mathrm{g / mL},0.6\mu \mathrm{g / mL},$ $0.8\mu \mathrm{g / mL},1.0\mu \mathrm{g / mL}$

将上述浓度梯度 $N$ -二甲基亚硝胺标准系列工作液分别注入GC-TEA中，用保留时间定性，测定 $N$ -二甲基亚硝胺的峰面积，谱图见图C.1。以 $N$ -二甲基亚硝胺标准工作液的浓度为横坐标 $(\mu \mathrm{g / mL})$ ，以 $N$ -二甲基亚硝胺的峰面积为纵坐标，绘制标准曲线。

# 12.4 试样溶液的测定

将试样溶液注入 GC-TEA 中, 用保留时间定性, 测定 $N$ -二甲基亚硝胺的峰面积, 根据标准曲线法

计算得到试样溶液中 $N\cdot$ 二甲基亚硝胺的浓度 $(\mu \mathrm{g / mL})$ 。

# 13 分析结果的表述

试样中 $N$ -二甲基亚硝胺含量按式(2)计算：

$$
X = \frac {\rho \times V \times 1 0 0 0}{W} \tag {2}
$$

式中：

$X$ ——试样中 $N$ -二甲基亚硝胺的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——试液中 $N-$ 二甲基亚硝胺的浓度，单位为毫克每升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试液定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$W$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 15 其他

当取样量为 $200\mathrm{g}$ ，浓缩体积为 $1.0~\mathrm{mL}$ 时，本方法的检出限为 $0.15\mu \mathrm{g / kg}$ ，定量限为 $0.5\mu \mathrm{g / kg}$ 。

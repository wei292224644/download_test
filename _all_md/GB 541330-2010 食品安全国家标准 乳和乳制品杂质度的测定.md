# 食品安全国家标准

# 乳和乳制品杂质度的测定

# 1 范围

本标准规定了乳和乳制品杂质度的测定方法。

本标准适用于巴氏杀菌乳、灭菌乳、生乳、炼乳及乳粉杂质度的测定，不适用于含非乳蛋白质、淀粉类成分、不溶性有色物质及影响过滤的添加物质。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

# 3 原理

试样经过滤板过滤、冲洗，根据残留于过滤板上的可见带色杂质的数量确定杂质量。

# 4 仪器和设备

4.1 过滤设备：杂质度过滤机或配有可安放过滤板漏斗的 $2000\mathrm{mL}\sim 2500\mathrm{mL}$ 抽滤瓶。  
4.2 过滤板：直径 $32\mathrm{mm}$ ，单位面积质量为 $135\mathrm{g} / \mathrm{m}^2$ ，符合附录A的要求，过滤时通过面积的直径为 $28.6\mathrm{mm}$ 。  
4.3 杂质度标准板。  
4.4 杂质度标准板的制作方法见附录B。  
4.5 天平：感量为 $0.1\mathrm{g}$

# 5 分析步骤

液体乳样量取 $500 \mathrm{~mL}$ ；乳粉样称取 $62.5 \mathrm{~g}$ （精确至 $0.1 \mathrm{~g}$ ），用8倍水充分调和溶解，加热至 $60^{\circ} \mathrm{C}$ ；炼乳样称取 $125 \mathrm{~g}$ （精确至 $0.1 \mathrm{~g}$ ），用4倍水溶解，加热至 $60^{\circ} \mathrm{C}$ ，于过滤板上过滤，为使过滤迅速，可用真空泵抽滤，用水冲洗过滤板，取下过滤板，置烘箱中烘干，将其上杂质与标准杂质板比较即得杂质度。

当过滤板上杂质的含量介于两个级别之间时，判定为杂质含量较多的级别。

# 6 分析结果的表述

与杂质度标准比较得出的过滤板上的杂质量，即为该样品的杂质度。

# 7 精密度

按本标准所述方法对同一样品所作的两次重复测定，其结果应一致，否则应重复再测定两次。

# 附录A

# （规范性附录）

# 杂质度过滤板的检验

# A.1 试剂盒材料

A. 1.1 润湿剂： $1\%$ 溶液。使用一种湿雾剂或其他合适的润湿剂。  
A. 1.2 植物胶溶液：将 $0.75 \mathrm{~g}$ 角豆胶或其他合适的胶加到 $100 \mathrm{~mL}$ 水中，然后用搅拌器搅拌。

通过抽真空或加热处理，使溶液排除气泡。煮沸、冷却，然后加 $2 \mathrm{~mL} 40 \%$ 甲醛溶液。

为了便于在没有搅拌的情况下在水中分离，可将 $0.75\mathrm{g}$ 角豆胶溶入 $100\mathrm{mL}$ 容量瓶内的几毫升乙醇中。用水稀释至刻度，充分混合。然后按上述步骤继续操作。

A. 1.3 蔗糖溶液：将 $750 \mathrm{~g}$ 市售白砂糖溶于 $750 \mathrm{~mL}$ 水中。  
A. 1.4 精制杂质混合物: 用地面牛粪、泥土及木炭经过干燥箱 $(100^{\circ} \mathrm{C})$ 烘干, 制备混合物, 每种材料分别过筛, 收集能通过 $106 \mu \mathrm{m}$ (140 目) 但不能通过 $75 \mu \mathrm{m}$ (200 目) 的筛子成分, 做法如下:

放 $100\mathrm{g}$ 以下的牛粪或泥土， $50\mathrm{g}$ 以下的木炭，过筛。在 $106\mu \mathrm{m}$ （140目）的筛子外套装 $75\mu \mathrm{m}$ （200目）的另一个筛子，覆盖、固定接收装置。用手振荡筛子的套，以每分钟敲打120次的速度过筛。一次操作大约用量 $20\mathrm{g}$ ，以过 $106\mu \mathrm{m}$ （140目）筛子的碎片继续在 $75\mu \mathrm{m}$ （200目）的筛子上过 $5\mathrm{min}$ 。使用第二次过筛所保留的成分，按表A.1最大程度的混合均匀：

将 $2 \mathrm{~g}$ 上述混合物放入到 $100 \mathrm{~mL}$ 容量瓶中，用 $5 \mathrm{~mL}$ 润湿剂（A.1.1）润湿。加入 $46 \mathrm{~mL}$ 植物胶溶液，然后用蔗糖溶液（A.1.3）将液面加到瓶颈口。加几滴乙醇，再用蔗糖溶液加至刻度，充分混合。

将溶液倒入一个 $250 \mathrm{~mL}$ 烧杯或有螺旋盖的瓶子中，允许带进少量空气，用一个小的机械搅拌器以 200 转/分钟 $\sim 300$ 转/分钟的速度搅拌，直到杂质在明亮的反射光下均匀分布为止。搅拌时，不要使细小颗粒堆积在烧杯底部的旋涡处。用吸管（出口直径 $3 \mathrm{~mm}$ ）移取 $10 \mathrm{~mL}$ （大约相当于 $200 \mathrm{mg}$ 杂质混合物）于容量瓶中，用水定容到 $1000 \mathrm{~mL}$ 。

# A.2 仪器和设备

A.2.1 天平：感量为 $0.1\mathrm{mg}$   
A.2.2 干燥器：含有效干燥剂。  
A.2.3 干燥箱。  
A.2.4 过滤装置。  
A.2.5 过滤板。  
A.2.6 滤纸：直径 $7\mathrm{cm}$ 或 $9\mathrm{cm}$

# A.3 分析步骤

A. 3.1 将滤纸（A.2.6）放在一个布氏漏斗中，用大约 $200 \mathrm{~mL}$ 的水冲洗，然后放入 $100^{\circ} \mathrm{C}$ 烘箱中烘干至恒重。

A.3.2 将 $60 \mathrm{~mL}$ 精制杂质混合物（A.1.4）经过充分搅拌后，通过安装在过滤装置上的过滤板（A.2.5）过滤，相当于 $12 \mathrm{mg}$ 杂质。

用一个清洁的三角瓶收集滤液。将滤液转移到烧杯中，用水冲洗该三角瓶两次，将洗液全部加入到烧杯中。

A. 3.3 将滤液再一次通过固定在布氏漏斗中的经清洗、干燥，并称好质量的滤纸（A.3.1）。用水充分冲洗烧杯和滤纸，然后将滤纸放入 $100^{\circ} \mathrm{C}$ 烘箱中烘干至恒重。  
A.3.4 至少检验两个以上过滤板。

# A.4 评价

通过三个或三个以上过滤板杂质的平均量不超过 $2.8\mathrm{mg}$ 。根据精制的杂质混合物制备的标准板不应在表面下层出现杂质。

# 附录B

# （规范性附录）

# 杂质度标准板的制作

# B.1 材料

使焦粉、灰土、牛粪、木炭通过一定的筛子，然后在 $100^{\circ} \mathrm{C}$ 烘箱中烘干，并按下列比例配合混匀。焦粉占 $40 \%$ ，其中：

通过 $850\mu \mathrm{m}$ （20目）筛而不通过 $425\mu \mathrm{m}$ （40目）筛的占 $10\%$

通过 $425\mu \mathrm{m}$ （40目）筛而不通过 $250\mu \mathrm{m}$ （60目）筛的占 $30\%$

灰土占 $30\%$ ，可通过 $425\mu \mathrm{m}$ （40目）筛。

牛粪占 $20\%$ ，其中：

通过 $850\mu \mathrm{m}$ （20（20目）目筛而不通过 $425\mu \mathrm{m}$ （40目）筛的占 $2\%$

通过 $425\mu \mathrm{m}$ （40目）筛而不通过250（60目）筛的占 $8\%$

通过 $250\mu \mathrm{m}$ （60目）筛而不通过 $180\mu \mathrm{m}$ (80目)筛的占 $10\%$

木炭占 $10\%$ ，其中：

通过 $850\mu \mathrm{m}$ （20（20目）筛而不通过 $425\mu \mathrm{m}$ （40目）筛的占 $4\%$

通过 $425\mu \mathrm{m}$ （40目）筛而不通过250（60目）筛的占 $6\%$

# B.2 步骤

B.2.1 将已准备好的各种杂质混匀（总量以 $50 \mathrm{~g}$ 为宜），从中准确称取 $1.000 \mathrm{~g}$ ，直接倒入 $500 \mathrm{~mL}$ 容量瓶中，加蒸馏水 $2 \mathrm{~mL}$ 和体积分数为 $0.75 \%$ 经过过滤的阿拉伯胶液 $23 \mathrm{~mL}$ ，再以质量分数为 $50 \%$ 的经过过滤的蔗糖液加至刻度并混匀，此液杂质浓度为 $2 \mathrm{mg} / \mathrm{mL}$ 。  
B. 2.2 取浓度为 $2 \mathrm{mg} / \mathrm{mL}$ 的杂质液 $10 \mathrm{~mL}$ , 以 $500 \mathrm{~g} / \mathrm{L}$ 过滤的蔗糖液稀释至 $100 \mathrm{~mL}$ , 则此液杂质的浓度为 $0.2 \mathrm{mg} / \mathrm{mL}$ 。  
B.2.3 取浓度为 $0.2 \mathrm{mg} / \mathrm{mL}$ 的杂质液 $10 \mathrm{~mL}$ , 以 $500 \mathrm{~g} / \mathrm{L}$ 过滤的蔗糖液稀释至 $100 \mathrm{~mL}$ , 则此液杂质的浓度为 $0.02 \mathrm{mg} / \mathrm{mL}$ 。  
B.2.4 以 $500\mathrm{mL}$ 牛乳或 $62.5\mathrm{g}$ 乳粉为取样量，按表B.1制备各标准杂质板。

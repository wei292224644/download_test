# 检验方法

# A. 1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682中规定的三级水。

分析中所用标准滴定溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603之规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A. 2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 盐酸。

A.2.1.2 盐酸溶液： $1 + 1$

A.2.1.3 碳酸钠溶液： $106\mathrm{g / L}$

A.2.1.4 铁氰化钾溶液： $53~\mathrm{g / L}$

A.2.1.5 4-氨基安替比林-硼酸盐缓冲溶液： $0.1\%$

量取含 $0.618\%$ 硼酸的 $0.1 \mathrm{~mol} / \mathrm{L}$ 氯化钾溶液 $1000 \mathrm{~mL}$ 与 $0.1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液 $420 \mathrm{~mL}$ 混合，得到 $\mathrm{pH} 9.0$ 的硼酸盐缓冲溶液。

取 $1 \mathrm{~g} 4$ -氨基安替比林溶于 $1000 \mathrm{~mL}$ 硼酸盐缓冲溶液中。

A.2.1.6 乙酸氧铀锌溶液。

称取 $10\mathrm{g}$ 乙酸双氧铀，加 $5\mathrm{mL}$ 冰乙酸和 $50\mathrm{mL}$ 水，微热使溶解；称取 $30\mathrm{g}$ 乙酸锌，加 $3\mathrm{mL}$ 冰乙酸和 $30\mathrm{mL}$ 水，微热使溶解；将两溶液混合，放冷，过滤，即得。

# A.2.2 鉴别步骤

# A. 2.2.1 对羟基苯甲酸乙酯的熔点

称取 $0.5 \mathrm{~g}$ 试样, 精确至 $0.01 \mathrm{~g}$ , 加入 $50 \mathrm{~mL}$ 水使溶解, 立即加 $5 \mathrm{~mL}$ 盐酸, 振摇, 过滤, 用水充分洗涤沉淀, 在 $80^{\circ} \mathrm{C}$ 下减压干燥 $2 \mathrm{~h}$ 。按GB/T 617测定试样熔点, 熔点为 $115^{\circ} \mathrm{C} \sim 118^{\circ} \mathrm{C}$ 。

# A.2.2.2 显色反应

取 $0.01 \mathrm{~g}$ 试样，置于试管中，加 $1 \mathrm{~mL}$ 碳酸钠溶液，加热煮沸 $30 \mathrm{~s}$ ，冷却，加入 $5 \mathrm{~mL} 4$ -氨基安替比林-硼酸盐缓冲溶液和 $1 \mathrm{~mL}$ 铁氰化钾溶液，混匀，溶液变为红色。

# A.2.2.3 钠盐试验

铂丝用盐酸湿润后，蘸取样品，在无色火焰中燃烧，火焰应显鲜黄色。

称取 $1 \mathrm{~g}$ 试样, 精确至 $0.01 \mathrm{~g}$ , 用适量的水溶解, 加 $1 \mathrm{~mL}$ 盐酸溶液, 用水稀释至 $20 \mathrm{~mL}$ 。取 $1 \mathrm{~mL}$ 该试样溶液, 加 $5 \mathrm{~mL}$ 乙酸氧铀锌溶液, 摇匀, 有黄色沉淀产生。

# A. 3 对羟基苯甲酸乙酯钠含量的测定

# A.3.1 试剂和材料

A.3.1.1 冰乙酸。  
A.3.1.2 高氯酸标准滴定溶液： $c(\mathrm{HClO}_4) = 0.1\mathrm{mol / L}$

# A.3.2 分析步骤

称取约 $0.15 \mathrm{~g}$ 样品, 精确至 $0.0001 \mathrm{~g}$ , 加 $50 \mathrm{~mL}$ 冰乙酸溶解, 用高氯酸标准滴定溶液滴定, 电位滴定法指示终点。同时进行空白试验。

# A.3.3 结果计算

对羟基苯甲酸乙酯钠（干基计）的质量分数 $w_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\left(v _ {1} - v _ {2}\right) \times c \times M}{m \times 1000} \times 100 \% \dots \tag{A.1}
$$

式中：

$v_{1}$ ——空白消耗高氯酸标准滴定溶液体积的数值，单位为毫升（ $\mathrm{mL}$ ）；

$v_{2}$ ——样品消耗高氯酸标准滴定溶液体积的数值，单位为毫升（ $\mathrm{mL}$ ）；

$c$ ——高氯酸标准滴定溶液浓度的准确数值, 单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——换算为干基后样品的质量的数值，单位为克（g）；

$M$ ——对羟基苯甲酸乙酯钠的摩尔质量的数值，单位为克每摩尔（ $\mathrm{g/mol}$ ） $[M(\mathrm{C}_9\mathrm{H}_9\mathrm{NaO}_3)=188.2]$ ；1000——换算系数。

实验结果以平行测定结果的算术平均值为准（保留一位小数）。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $1.0\%$ 。

# A. 4 氯化物的测定

# A.4.1 试剂和材料

A.4.1.1 硝酸溶液： $1 + 9$   
A.4.1.2 硝酸银溶液： $17\mathrm{g / L}$   
A.4.1.3 氯化物（Cl）标准溶液： $0.01\mathrm{mg / mL}$

# A.4.2 分析步骤

称取 $2.0\mathrm{g}$ 样品，加 $40~\mathrm{mL}$ 水溶解，用硝酸溶液调节溶液至中性，用水稀释至 $50~\mathrm{mL}$ ，振摇，过滤，保留滤液。取滤液 $5.0~\mathrm{mL}$ ，作为试验溶液，置于 $50~\mathrm{mL}$ 纳氏比色管中，加 $10~\mathrm{mL}$ 硝酸溶液，加水至体积约 $40~\mathrm{mL}$ ，加 $1.0~\mathrm{mL}$ 硝酸银溶液，用水稀释至 $50~\mathrm{mL}$ ，摇匀，暗处放置 $5\mathrm{min}$ ，在黑色背景下，轴向观察，试验溶液所呈浊度不得大于标准。

标准是量取 $7.0 \mathrm{~mL}$ 氯化物（Cl）标准溶液，与试验溶液同时同样处理。

# A.5 硫酸盐的测定

# A.5.1 试剂和材料

A.5.1.1 盐酸溶液： $1 + 3$   
A.5.1.2 氯化钡溶液： $250\mathrm{g / L}$   
A.5.1.3 硫酸钾标准溶液： $0.1\mathrm{mg / mL}$

# A.5.2 分析步骤

量取A.4.2的滤液 $25 \mathrm{~mL}$ , 作为试验溶液, 置于 $50 \mathrm{~mL}$ 纳氏比色管中, 加 $2 \mathrm{~mL}$ 盐酸溶液, 加水至体积约 $40 \mathrm{~mL}$ , 加 $5 \mathrm{~mL}$ 氯化钡溶液, 用水稀释至 $50 \mathrm{~mL}$ , 摇匀, 放置 $10 \mathrm{~min}$ , 在黑色背景下, 轴向观察, 试验溶液所呈浊度不得大于标准。

标准是量取 $2.4 \mathrm{~mL}$ 硫酸钾标准溶液，与试验溶液同时同样处理。

# A. 6 杂质的测定

# A.6.1 方法提要

用高效液相色谱法，在选定的工作条件下，通过色谱柱使样品溶液中各组分分离，用紫外吸收检测器检测，外标法定量，得到样品中各杂质的含量。

# A.6.2 试剂和材料

A.6.2.1 对羟基苯甲酸乙酯钠标准品：质量分数 $\geq 95\%$   
A.6.2.2 对羟基苯甲酸标准品：质量分数 $\geq 99\%$   
A.6.2.3 甲醇。  
A.6.2.4 冰乙酸溶液： $1\rightarrow 100$

# A.6.3 仪器和设备

A.6.3.1 高效液相色谱仪(HPLC)：配有紫外光检测器。  
A. 6.3.2 进样器: 自动进样器或微量进样器, $50 \mu \mathrm{L}$ 。  
A.6.3.3 数据处理系统：色谱工作站或数据处理机。

# A.6.4 色谱分析条件

# A.6.5 分析步骤

# A.6.5.1 对羟基苯甲酸乙酯钠和对羟基苯甲酸混合溶液的制备

分别称取适量对羟基苯甲酸乙酯钠和对羟基苯甲酸，混合，用流动相溶解并稀释，制成含对羟基苯甲酸乙酯钠、对羟基苯甲酸各 $0.1\mathrm{mg / mL}$ 的混合溶液。

# A.6.5.2 样品溶液的制备

称取适量样品，用流动相溶解并稀释，制成 $1.0\mathrm{mg / mL}$ 的样品溶液。

# A.6.5.3 对照溶液的制备

准确量取样品溶液适量，用流动相稀释，制成 $10~\mu \mathrm{g / mL}$ 的对照溶液。

# A.6.5.4 测定

按高效液相色谱操作规程调节仪器，使基线平稳。取 $20~\mu \mathrm{L}$ 对羟基苯甲酸乙酯钠和对羟基苯甲酸混合溶液注入色谱仪，记录色谱图，理论板数按对羟基苯甲酸乙酯钠峰计算不低于8000，对羟基苯甲酸乙酯钠峰与对羟基苯甲酸峰的分离度应大于1.5。进样 $20~\mu \mathrm{L}$ 对照溶液，调节检测灵敏度，使羟基苯甲酸乙酯钠峰峰高约为满量程的 $20\%$ 。分别准确进样 $20~\mu \mathrm{L}$ 对照溶液和样品溶液，记录色谱图至对羟基苯甲酸乙酯钠峰保留时间的4倍。

# A.6.6 结果计算

样品溶液色谱图中如有对羟基苯甲酸峰，其峰面积不得大于对照溶液中对羟基苯甲酸乙酯钠峰面积的4倍。其他单个杂质峰面积不得大于对照溶液中对羟基苯甲酸乙酯钠峰面积的0.5倍。其他各杂质峰面积的和不得大于对照溶液中对羟基苯甲酸乙酯钠峰面积。

# A. 7 溶液的澄清度与颜色

# A.7.1 试剂和材料

# A.7.1.1 比色用重铬酸钾溶液的制备

称取 $0.4000 \mathrm{~g}$ 在 $120^{\circ} \mathrm{C}$ 干燥至恒量的基准重铬酸钾, 置于 $500 \mathrm{~mL}$ 容量瓶中, 加适量水溶解并稀释至刻度, 摇匀。该溶液含重铬酸钾 $\left(\mathrm{K}_{2} \mathrm{Cr}_{2} \mathrm{O}_{7}\right) 0.800 \mathrm{mg} / \mathrm{mL}$ 。

# A.7.1.2 比色用氯化钴溶液的制备

称取约 $32.5\mathrm{g}$ 六水合氯化钴，精确至 $0.01\mathrm{g}$ ，加适量盐酸溶液（ $1\rightarrow 40$ ）溶解，转入 $500\mathrm{mL}$ 容量瓶中，用盐酸溶液（ $1\rightarrow 40$ ）稀释至刻度。准确量取 $2.0\mathrm{mL}$ 该溶液于锥形瓶中，加 $200\mathrm{mL}$ 水摇匀，加氨水溶液至氯化钴溶液由浅红色变为绿色，加 $10\mathrm{mL}$ 乙酸-乙酸钠缓冲溶液（ $\mathrm{pH}6.0$ ），加热至 $60^{\circ}\mathrm{C}$ ，加5滴二甲酚橙指示液，用乙二胺四乙酸二钠标准滴定溶液 $[c(\mathrm{EDTA}) = 0.05\mathrm{mol / L}]$ 滴定至溶液显黄色，记录消耗硫代硫酸钠标准滴定溶液的体积数。每 $1\mathrm{mL}$ 乙二胺四乙酸二钠标准滴定溶液相当于 $11.90\mathrm{mg}$ 的六水合氯化钴（ $\mathrm{CoCl}_2\cdot 6\mathrm{H}_2\mathrm{O}$ ）。根据上述滴定结果，在剩余的氯化钴溶液中加入适量盐酸溶液（ $1\rightarrow 40$ ），配制成含六水合氯化钴（ $\mathrm{CoCl}_2\cdot 6\mathrm{H}_2\mathrm{O}$ ） $59.5\mathrm{mg / mL}$ 溶液。

# A.7.1.3 标准比色液的制备

准确量取 $12.0 \mathrm{~mL}$ 比色用氯化钴溶液、 $20.0 \mathrm{~mL}$ 比色用重铬酸钾液和 $68.0 \mathrm{~mL}$ 水，混合摇匀，作为标准贮备液。准确量取 $2.5 \mathrm{~mL}$ 标准贮备液，加 $7.5 \mathrm{~mL}$ 水，混合摇匀，得标准比色液。

# A.7.2 分析步骤

称取 $1.0 \mathrm{~g}$ 试样置于 $25 \mathrm{~mL}$ 纳氏比色管中, 加 $10 \mathrm{~mL}$ 水溶解, 溶液应澄清; 另取标准比色液 $10 \mathrm{~mL}$ , 置于另一 $25 \mathrm{~mL}$ 纳氏比色管中, 两管同置白色背景上, 自上向下透视, 或同置白色背景前, 平视观察, 试样管呈现的颜色与对照管比较, 不得更深。

# A.8 铅的测定

按GB 5009.12-2010中“石墨炉原子吸收光谱法”或“火焰原子吸收光谱法”进行。其中，试样不

需预处理，试样消解可根据实验室条件选用“干法灰化”或“湿式消解法”进行，并视样品情况将试样溶液进行适当的稀释。

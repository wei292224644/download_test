# 食品安全国家标准

# 动物性水产品及其制品中并殖吸虫的检验

# 1 范围

本标准规定了动物性水产品及其制品中并殖吸虫（Paragonimus）囊蚴的形态学和PCR的检验方法。

本标准适用于动物性水产品及其制品中并殖吸虫囊蚴的检验。

# 2 原理

动物性水产品及其制品中并殖吸虫的囊为主要寄生在蟹类、蛔蛄和沼虾等甲壳动物的肌肉组织内。捣碎沉淀分离或捣碎后使用胃蛋白酶消化动物宿主组织获得囊，根据并殖吸虫囊的形态特征，初步判断囊的种类；通过扩增并殖吸虫核糖体DNA的第二内转录间隔区(ITS2)基因片段并测序，进行并殖吸虫囊的鉴定。

# 3 仪器与设备

3.1 生物显微镜： $100 \times \sim 400 \times$ 。  
3.2 体视显微镜： $7.5\times \sim 150\times$   
3.3 PCR扩增仪。  
3.4 凝胶成像系统。  
3.5 电泳仪。  
3.6 恒温培养箱： $37^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 。  
3.7 高速离心机：转速 $\geqslant 12000\mathrm{r / min}$   
3.8 网筛：孔径 $2\mathrm{mm}(10$ 目)；孔径 $0.425\mathrm{mm}(40$ 目）。  
3.9 微量移液器： $0.2\mu \mathrm{L}\sim 2.5\mu \mathrm{L},1\mu \mathrm{L}\sim 10\mu \mathrm{L},10\mu \mathrm{L}\sim 100\mu \mathrm{L},100\mu \mathrm{L}\sim 1000\mu \mathrm{L}$   
3.10 锥形量杯： $1000~\mathrm{mL}$   
3.11 玻璃珠： $425\mu \mathrm{m}\sim 600\mu \mathrm{m}$   
3.12 陶瓷研钵（直径 $16\mathrm{cm}$ 和研磨棒。  
3.13 回旋振荡器。

# 4 试剂和材料

# 4.1 试剂

4.1.1 盐酸： $36\% \sim 38\% \mathrm{HCl}$ 溶液。  
4.1.2 生理盐水： $0.85\% \mathrm{NaCl}$ 溶液。  
4.1.3 $1\mathrm{mol} / \mathrm{L}$ Tris-HCl溶液（pH8.0）。  
4.1.4 0.5 mol/L EDTA 溶液（pH 8.0）。

4.1.5 $10\%$ SDS溶液。  
4.1.6 $5\mathrm{mol} / \mathrm{L}$ NaCl溶液。  
4.1.7 $3000\mathrm{U / mg}$ 胃蛋白酶。  
4.1.8 $20\mathrm{mg / mL}$ 蛋白酶K。  
4.1.9 苯酚/三氯甲烷/异戊醇 $(25:24:1)$ 。  
4.1.10 $5\mathrm{U} / \mu \mathrm{L}$ 耐热DNA聚合酶。  
4.1.11 $10 \times \mathrm{PCR}$ 缓冲液。  
4.1.12 $25\mathrm{mmol / L}$ $\mathrm{MgCl_2}$ 。  
4.1.13 dNTPs:dATP、dTTP、dCTP、dGTP，每种浓度为 $2.5\mathrm{mmol / L}$   
4.1.14 琼脂糖：电泳级。  
4.1.15 $50 \times \mathrm{TAE}$ 缓冲液，使用前用去离子水稀释成 $1 \times \mathrm{TAE}$ 缓冲液。  
4.1.16 $1\times \mathrm{TE}$ 溶液（pH8.0）。  
4.1.17 $10\mathrm{mg / mL}$ 溴化乙锭（EB）或其他核酸染料。  
4.1.18 $6\times$ 上样缓冲液。  
4.1.19 $100\mathrm{bp}\sim 2000\mathrm{bp}$ DNA分子量标准。  
4.1.20 引物：浓度为 $10\mu \mathrm{mol} / \mathrm{L}$

正向引物 $3\mathrm{S}:5^{\prime}$ -GGTACCGGTGGATCACTCGGCTCGTG- $3^{\prime}$

反向引物 A28: $5^{\prime}$ -GGGATCCTGGTTAGTTTCTTTTCCTCCGC-3'

扩增的并殖吸虫 ITS2 基因片段长度为 $520 \mathrm{bp}$ 。

# 4.2 试剂配制

4.2.1 生理盐水：称取 $8.5\mathrm{gNaCl}$ 溶解于 $900~\mathrm{mL}$ 去离子水，再加去离子水至 $1000~\mathrm{mL}$ 。  
4.2.2 $1\mathrm{mol} / \mathrm{L}$ Tris-HCl溶液：称取 $121.1\mathrm{g}$ Tris(三羟甲基氨基甲烷)溶解于 $800~\mathrm{mL}$ 去离子水，用盐酸调节 $\mathrm{pH}$ 至8.0，加去离子水至 $1000~\mathrm{mL}$ 。  
4.2.3 $0.5\mathrm{mol} / \mathrm{L}$ EDTA溶液：称取 $186.1\mathrm{gNa_2EDTA}\cdot 2\mathrm{H_2O}$ （二水合乙二胺四乙酸二钠)溶解于 $800~\mathrm{mL}$ 去离子水，搅拌溶解，用 $\mathrm{NaOH}$ 调节 $\mathrm{pH}$ 至8.0，加去离子水至 $1000~\mathrm{mL}$   
4.2.4 $5\mathrm{mol} / \mathrm{L}$ NaCl溶液：称取 $292.5\mathrm{gNaCl}$ 溶解于 $900~\mathrm{mL}$ 去离子水，再加去离子水至 $1000~\mathrm{mL}$   
4.2.5 胃蛋白酶消化液：取胃蛋白酶 $5\mathrm{g}$ ，溶解于 $900~\mathrm{mL}$ 生理盐水中，加盐酸 $7\mathrm{mL}$ ，混匀，再加生理盐水至 $1000~\mathrm{mL}$ ，临用现配。  
4.2.6 裂解液： $10\%$ SDS 溶液 $100 \mathrm{~mL}, 1 \mathrm{~mol} / \mathrm{L}$ Tris-HCl 溶液 $10 \mathrm{~mL}, 0.5 \mathrm{~mol} / \mathrm{L}$ EDTA 溶液 $200 \mathrm{~mL}, 5 \mathrm{~mol} / \mathrm{L} \mathrm{NaCl}$ 溶液 $20 \mathrm{~mL}$ ，加灭菌去离子水至 $1000 \mathrm{~mL}$ 。  
4.2.7 $1.5\%$ 琼脂糖凝胶：琼脂糖 $1.5\mathrm{g}$ ，加入 $1\times \mathrm{TAE}$ 缓冲液至 $100~\mathrm{mL}$ ，加热至完全融化后冷却至 $60^{\circ}C\sim 70^{\circ}C$ ，加入 $10~\mathrm{mg / mL}$ 溴化乙锭 $5\mu \mathrm{L}$ ，混匀，制备凝胶。

# 5 检测方法

# 5.1 形态学方法

# 5.1.1 捣碎法

# 5.1.1.1 样品制备

当送检的甲壳类水产品或其制品的数量较少时，对单个样品1只(个)分别检测，使用消毒过的陶瓷研钵及研磨棒捣碎样品，用生理盐水冲洗10目网筛过滤于锥形量筒内，除去粗的蟹(蝌蛄、虾)壳。滤液再用40目网筛过滤，除去细壳，加生理盐水至锥形量杯的最大刻度处，沉淀 $15\mathrm{min}$ 后，弃上清液，用生

理盐水洗涤沉淀3次～5次至上清液透明为止，全部沉渣吸入玻璃平皿，在体视显微镜下收集囊蚴。

# 5.1.1.2 镜检

将含有沉渣的玻璃平皿在体视显微镜下观察，检查挑取其中的囊蚴，在生物显微镜下进行形态鉴定。新鲜水产品中并殖吸虫囊蚴一般呈圆球形或椭圆球形，囊蚴直径 $250\mu \mathrm{m}\sim 435\mu \mathrm{m}$ ，壁厚 $8\mu \mathrm{m}\sim$ $20~\mu \mathrm{m}$ ，通常具有2层囊壁膜，后尾蚴折叠蜷曲在囊内，能看见充满黑色颗粒的排泄囊和两侧弯曲的肠管。不同种的并殖吸虫囊蚴大小有差异，如卫氏并殖吸虫囊蚴直径 $250\mu \mathrm{m}\sim 390\mu \mathrm{m}$ ，壁厚 $14\mu \mathrm{m}\sim$ $20~\mu \mathrm{m}$ ，有两层囊壁，外薄内厚，后尾蚴挤缩囊内，排泄囊稍大；斯氏并殖吸虫囊蚴，直径 $400\mu \mathrm{m}\sim$ $420\mu \mathrm{m}$ ，壁厚 $12\mu \mathrm{m}\sim 14\mu \mathrm{m}$ ，内层囊壁略薄，后尾蚴挤缩囊内，排泄囊小而黑；三平正并殖吸虫囊蚴直径 $415\mu \mathrm{m}\sim 435\mu \mathrm{m}$ ，壁厚 $8\mu \mathrm{m}\sim 12\mu \mathrm{m}$ ，内壁层厚，近两端更显增厚，体多作内壁形蜷曲于囊内，其模式图见图A.1，其实物图见图A.2。在水产品中检查到具备上述特征的囊蚴，可判定为疑似并殖吸虫囊蚴。疑似并殖吸虫囊蚴立即用于DNA提取或一 $20^{\circ}C$ 保存备用。

# 5.1.2 胃蛋白酶消化法

# 5.1.2.1 取样

当送检的甲壳类水产品或其制品数量大于20个以上，可使用胃蛋白酶消化法。分批用研磨棒在消毒过的陶瓷研钵中将样品捣碎，备用。

# 5.1.2.2 消化

取5.1.2.1样品 $200\mathrm{g}$ 或者按样品称重后的实际重量，分批按照样品与胃蛋白酶消化液 $1:5$ 的质量体积比混匀，置三角烧杯中充分混匀后，于 $37^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温培养箱放置 $4\mathrm{h} \sim 6\mathrm{h}$ ，使肌肉组织完全消化。可根据不同水产品种类以及消化时间对胃蛋白酶浓度和使用量进行调整。

# 5.1.2.3 过滤

消化后的悬液经10目网筛过滤，除去粗壳；再用40目网筛过滤，除去细壳，并用生理盐水冲洗网筛上的残留物。收集滤液置于锥形量杯内，搅拌后沉淀 $15\mathrm{min}$ 。轻轻倾去上清液，加入生理盐水至锥形量杯的最大刻度处，搅拌后再沉淀 $15\mathrm{min}$ 。通常需用生理盐水洗涤滤液中的沉淀3次～5次，直至上清液透明为止，沉淀备用。

# 5.1.2.4 镜检

全部沉淀物分次转移至玻璃平皿，在体视显微镜下去除沉淀中的杂质，分离收集疑似囊蚴，用生物显微镜观察囊蚴形态。具备5.1.1.2形态特征的囊蚴，可判定为疑似并殖吸虫囊蚴，疑似并殖吸虫囊蚴立即用于DNA提取或 $-20^{\circ}\mathrm{C}$ 保存备用。

# 5.2 PCR方法

# 5.2.1 DNA提取

取符合并殖吸虫囊形态特征的单个或多个疑似囊用生理盐水洗净，放入 $1.5\mathrm{mL}$ 离心管中，加DNA裂解液 $500~\mu \mathrm{L}$ ，再加入少量玻璃珠振荡 $5\mathrm{min}\sim 10\mathrm{min}$ ，匀浆使囊充分裂解。加入蛋白酶K $10~\mu \mathrm{L}$ ，混匀， $55^{\circ}C$ 振荡消化 $2\mathrm{h}$ 。加入苯酚/三氯甲烷/异戊醇（ $25:24:1$ ） $240~\mu \mathrm{L}$ ，混匀， $4^{\circ}C$ $12000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。吸取上清(约 $500~\mu \mathrm{L}$ ) 加入新的 $1.5\mathrm{mL}$ 离心管，加入2倍体积一 $20^{\circ}C$ 预冷的无水乙醇，充分混匀， $4^{\circ}C$ $12000\mathrm{r / min}$ 离心 $15\mathrm{min}$ 。沉淀用 $75\%$ 乙醇 $700~\mu \mathrm{L}$ 离心洗涤1次～2次，每次 $4^{\circ}C$ $12000\mathrm{r / min}$ 离心 $3\mathrm{min}$ 。倒干乙醇， $37^{\circ}C$ 干燥后加入 $1\times \mathrm{TE}$ 溶液（ $\mathrm{pH8.0}$ ） $100~\mu \mathrm{L}$ 溶解

DNA，立即用于检测或 $-20^{\circ} \mathrm{C}$ 保存备用。

注：根据实验室实际情况，可使用经验证的商品化组织DNA提取试剂盒提取DNA。

# 5.2.2 PCR反应体系

在PCR管中依次加入 $10\times \mathrm{PCR}$ 缓冲液 $5.0\mu \mathrm{L},\mathrm{MgCl}_25.0\mu \mathrm{L},$ dNTPs $2.0\mu \mathrm{L}$ 、正向引物和反向引物各 $2.0\mu \mathrm{L}$ 、耐热DNA聚合酶 $0.5\mu \mathrm{L}$ 、DNA模板 $2.5\mu \mathrm{L}$ 、加灭菌去离子水至总体积 $50~\mu \mathrm{L}$ 。每次试验需设阳性对照、空白对照。阳性对照用并殖吸虫囊蚴或成虫的DNA或者相应基因片段的质粒做模板，空白对照用灭菌去离子水做模板。

# 5.2.3 PCR反应条件

$95\mathrm{C}$ 预变性 $1\mathrm{min};94\mathrm{C}$ 变性 $50~\mathrm{s},68~\mathrm{C}$ 退火 $1\mathrm{min},68\mathrm{C}$ 延伸 $1\mathrm{min},35$ 个循环； $72\mathrm{C}$ 延伸 $7\mathrm{min};$ $4^{\circ}C$ 保存。

# 5.2.4 电泳

取PCR扩增产物 $10~\mu \mathrm{L}$ 与 $6\times$ 上样缓冲液 $2.0\mu \mathrm{L}$ 混合，加样于含溴化乙锭的 $1.5\%$ 琼脂糖凝胶中，其中的一孔加DNA分子量标准。 $1\times$ TAE缓冲液， $5\mathrm{V / cm}$ 恒压电泳 $30\mathrm{min}\sim 40\mathrm{min}$ ，用凝胶成像系统观察和记录结果。

# 5.2.5 PCR结果判定

阳性对照出现预期大小的条带(520 bp)，空白对照无条带。待测样品扩增出现预期大小条带，可判PCR结果为阳性；无扩增条带或未扩增出预期大小的条带均判为PCR结果阴性。

取PCR结果为阳性的PCR产物进行基因序列双向测序，将测序结果与并殖吸虫的参考序列（见附录B)进行同源性比对。

# 6 结果报告

6.1 形态学方法检出疑似并殖吸虫囊蚴、PCR结果为阳性且扩增片段基因序列与并殖吸虫参考序列同源性 $\geqslant 95\%$ ，报告检出并殖吸虫囊蚴。  
6.2 形态学方法鉴定未检出囊蚴、或PCR结果为阴性、或基因序列与并殖吸虫参考序列同源性 $< 95\%$ ，报告未检出并殖吸虫囊蚴。

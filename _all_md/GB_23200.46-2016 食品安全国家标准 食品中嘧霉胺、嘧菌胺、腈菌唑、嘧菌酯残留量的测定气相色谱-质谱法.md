本标准按照GB/T1.1—2009给出的规则起草。

# 食品中嘧霉胺、嘧菌胺、腈菌唑、嘧菌酯残留量的测定气相色谱一质谱法

# 1 范围

本标准规定了试样的制备方法和保存条件以及粮谷、蔬菜、水果、坚果、茶叶、畜、禽、水产品、蜂产品中嘧霉胺、嘧菌胺、腈菌唑、嘧菌酯残留量检验的气相色谱一质谱测定方法。

本标准适用于大米、茄子、苹果、板栗、茶叶、牛肉、鸡肉、鱼、蜂蜜中嘧霉胺、嘧菌胺、腈菌唑、嘧菌酯残留量的测定。其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

用丙酮或乙酸乙酯、丙酮和氯化钠水溶液提取试样中残留的嘧霉胺、嘧菌胺、腈菌唑、嘧菌酯，经液液萃取和石墨化炭黑柱/氨基柱组合柱净化，用气相色谱一质谱仪选择离子检测，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ ：农残级。  
4.1.2 乙酸乙酯 $(\mathrm{C_4H_8O_2})$   
4.1.3 乙腈 $\mathrm{(CH_3CN)}$ ：HPLC级。  
4.1.4 甲苯 $(\mathrm{C}_7\mathrm{H}_8)$ 。  
4.1.5 正己烷 $(\mathrm{C_6H_{14}})$   
4.1.6 氯化钠 $\mathrm{(NaCl)}$ 。  
4.1.7 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：于 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，储于密封容器中备用。

# 4.2 溶液配制

4.2.1 氯化钠溶液 $(300\mathrm{g / L})$ ：称取 $300\mathrm{g}$ 氯化钠用蒸馏水溶解并定容至 $1000~\mathrm{mL}$   
4.2.2 乙腈—甲苯(3+2，体积比)：取 $30~\mathrm{mL}$ 乙腈和 $20~\mathrm{mL}$ 甲苯，混合均匀。  
4.2.3 乙腈—甲苯(3+1，体积比)：取 $30~\mathrm{mL}$ 乙腈和 $10~\mathrm{mL}$ 甲苯，混合均匀。  
4.2.4 丙酮—正己烷 $(1 + 1$ ，体积比)：取 $50~\mathrm{mL}$ 丙酮和 $50~\mathrm{mL}$ 正己烷，混合均匀。  
4.2.5 乙腈饱和正己烷。  
4.2.6 正己烷饱和乙腈。

# 4.3 标准品

4.3.1 喀霉胺标准品：分子式， $\mathrm{C_{12}H_{13}N_3}$ ；CAS号：53112-28-0；纯度 $\geqslant 99\%$ 。  
4.3.2 喹菌胺标准品：分子式， $\mathrm{C_{14}H_{13}N_3}$ ；CAS号：110235-47-7；纯度 $\geq 99\%$ 。  
4.3.3 腈菌唑标准品：分子式， $\mathrm{C_{15}H_{17}ClN_4}$ ；CAS号：88671-89-0；纯度 $\geqslant 99\%$ 。  
4.3.4 喹菌酯标准品：分子式， $\mathrm{C_{22}H_{17}N_3O_5}$ ；CAS号：131860-33-8；纯度 $\geq 99\%$ 。

# 4.4 标准溶液配制

嘧霉胺、嘧菌胺、腈菌唑、嘧菌酯标准溶液：准确称取适量的嘧霉胺、嘧菌胺、腈菌唑、嘧菌酯标准品，用丙酮配制成 $100\mu \mathrm{g / mL}$ 标准储备液， $5^{\circ}C$ 以下储存，6个月以内使用。再用丙酮稀释成适当浓度的标准工作溶液， $5^{\circ}C$ 以下储存，3个月以内使用。

# 4.5 材料

4.5.1 石墨化炭黑柱/氨基柱组合柱, $6 \mathrm{~mL} / 500 \mathrm{mg}$ , 或者石墨化炭黑柱 $(6 \mathrm{~mL} / 500 \mathrm{mg})$ 与氨基柱 (3 $\mathrm{mL} / 500 \mathrm{mg}$ ) 按照从上到下串联使用。  
4.5.2 无水硫酸钠柱: $150 \mathrm{~mm} \times 10 \mathrm{~mm}$ 玻璃层析柱, 从下往上依次装入脱脂棉, $5 \mathrm{~cm}$ 高的无水硫酸钠。

# 5 仪器和设备

5.1 气相色谱—质谱联用仪，配电子轰击离子源（EI源）。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.3 涡旋振荡器。  
5.4 均质器。  
5.5 离心机： $6000\mathrm{r / min}$   
5.6 固相萃取装置  
5.7 旋转蒸发器。  
5.8 氮气吹干仪。  
5.9 离心管：玻璃， $50~\mathrm{mL}$   
5.10 水平回旋式摇床

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按照GB2763的规定执行。在取样和制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 水果、蔬菜、坚果

取约 $500\mathrm{g}$ ，用粉碎机粉碎，装入洁净容器作为试样，密封并做好标识。

# 6.2.2 动物源性食品

6.2.2.1 肉类及水产品：取样品中有代表性的部分约 $500\mathrm{g}$ ，用粉碎机粉碎，装入洁净容器作为试样，密封并做好标识。  
6.2.2.2 蜂蜜：取有代表性样品约 $500\mathrm{g}$ ，未结晶样品将其用力搅拌均匀，有结晶析出样品可将样品瓶盖塞紧后，置于不超过 $60^{\circ}\mathrm{C}$ 的水浴中，待样品全部溶化后搅匀，迅速冷却至室温。制备好的样品装入洁净容器内密封并做好标识。

# 6.2.3 粮谷、茶叶

取样品约 $500\mathrm{g}$ ，用粉碎机粉碎至全部通过20目筛，装入洁净容器作为试样，密封并做好标识。

# 6.3 试样保存

水果、蔬菜、坚果、畜、禽和水产品等试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存；粮谷、茶叶、蜂产品等试样于室温避光保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 水果、蔬菜

称取搅碎混匀的试样约 $10\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $100\mathrm{mL}$ 锥形瓶中，加 $25\mathrm{mL}$ 丙酮和 $15\mathrm{g}$ 氯化钠，在水平回旋式摇床上振摇 $30\mathrm{min}$ 。过滤到 $250\mathrm{mL}$ 分液漏斗中，用 $20\mathrm{mL}$ 丙酮分两次洗涤锥形瓶及滤渣，合并滤液于分液漏斗中，加 $25\mathrm{mL}$ 氯化钠溶液和 $30\mathrm{mL}$ 乙酸乙酯于同一分液漏斗中，振摇 $2\mathrm{min}$ ，静置分层，收集有机相。下层水相再用 $20\mathrm{mL}$ 乙酸乙酯萃取一次，合并有机相，过无水硫酸钠柱至心形瓶中。于 $45^{\circ}\mathrm{C}$ 水浴上抽真空旋转蒸发至干。

# 7.1.2 畜、禽、水产品

称取搅碎混匀的试样约 $10\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加 $25~\mathrm{mL}$ 乙酸乙酯， $15\mathrm{g}$ 氯化钠，均质 $1\mathrm{min}$ ，以 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。取上层有机相过无水硫酸钠柱转入心形瓶。再每次用 $20~\mathrm{mL}$ 乙酸乙酯分2次按上述步骤提取残渣，合并有机相于心形瓶中。于 $45^{\circ}\mathrm{C}$ 水浴上抽真空旋转蒸发至干。

# 7.1.3 粮谷、茶叶、板栗

称取搅碎混匀的试样约 $10\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 于三角瓶中, 加 $25\mathrm{mL}$ 乙酸乙酯(板栗样品加 $5\mathrm{g}$ 氯化钠), 在水平回旋式摇床上振摇 $30\mathrm{min}$ , 过滤到 $250\mathrm{mL}$ 分液漏斗中, 用 $20\mathrm{mL}$ 乙酸乙酯分2次洗涤锥形瓶及滤渣, 合并滤液于同一分液漏斗中并加入 $30\mathrm{mL}$ 氯化钠溶液, 振荡 $1\mathrm{min}$ , 液液萃取, 静置分层, 乙酸乙酯层经无水硫酸钠柱转入心型瓶中; 水相再加入 $25\mathrm{mL}$ 乙酸乙酯液液萃取, 静置分层, 弃去水相, 合并乙酸乙酯层于上述同一心形瓶中。于 $45^{\circ}\mathrm{C}$ 水浴上抽真空旋转蒸发至干。

# 7.1.4 蜂蜜

称取 $10\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 蜂蜜样品于三角瓶中, 加入 $20\mathrm{mL}$ 氯化钠溶液和 $5\mathrm{mL}$ 丙酮溶解, 在水平回旋式摇床上振摇 $30\mathrm{min}$ , 转入 $250\mathrm{mL}$ 分液漏斗中, 再分别用 $30\mathrm{mL}$ 氯化钠溶液分2次, $50\mathrm{mL}$ 乙酸乙酯分两次洗原三角瓶, 均转入同一分液漏斗, 振摇 $2\mathrm{min}$ , 静置分层(如果发生乳化, 可将上层及乳化层在 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ , 取上层转入心型瓶), 收集有机相到另一个分液漏斗。水相再用 $20\mathrm{m}$ 乙酸乙酯提取两次, 合并有机相到同一分液漏斗中, 分液漏斗中加入 $40\mathrm{mL}$ 氯化钠溶液振摇 $1\mathrm{min}$ , 静置(如果发生乳化, 可将上层及乳化层在 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ), 乙酸乙酯层过无水硫酸钠柱转入上述心形瓶, 于 $45^{\circ}\mathrm{C}$ 水浴上抽真空旋转蒸发至干。

# 7.2 净化

# 7.2.1 畜、禽、水产品、粮谷、茶叶、板栗

对7.1.2和7.1.3获得的试样残渣，加入 $40~\mathrm{mL}$ 乙腈饱和正己烷分两次溶解，转入同一 $250~\mathrm{mL}$ 分液漏斗中，分别用 $50~\mathrm{mL}$ 正己烷饱和乙腈分2次、乙腈饱和正己烷 $10~\mathrm{mL}$ 洗心型瓶，均转入上述分液漏斗中。振荡分层，乙腈层过无水硫酸钠柱转入原心型瓶，正己烷层每次再用正己烷饱和乙腈 $15~\mathrm{mL}$ 洗两次，正己烷层弃去，乙腈层过无水硫酸钠柱合并入心型瓶，于 $45^{\circ}C$ 水浴上抽真空旋转蒸发至干。

# 7.2.2 石墨化炭黑柱/氨基柱组合柱净化

用 $1\mathrm{mL}$ 乙腈一甲苯 $(3 + 1$ ，体积比)溶解7.1.1、7.1.4、7.2.1得到的试样残渣，全部转入石墨化炭黑柱一氨基柱。再用 $1\mathrm{mL}$ 乙腈一甲苯 $(3 + 1$ ，体积比)分两次洗心形瓶，并入上述石墨化炭黑柱一氨基柱，弃去全部流出液。用 $10\mathrm{mL}$ 乙腈一甲苯 $(3 + 2$ ，体积比)洗脱石墨化炭黑柱一氨基组合柱，接收全部洗脱液。于 $45^{\circ}C$ 水浴上氮气流吹干。用丙酮一正己烷 $(1 + 1$ ，体积比)定容 $1.0~\mathrm{mL}$ 。供气相色谱质谱

分析。

# 7.3 测定

# 7.3.1 气相色谱一质谱参考条件

# 7.3.1.1 色谱条件

a）色谱柱：DB-5MS， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱升温程序： $210^{\circ}\mathrm{C}$ （保持 $2\mathrm{min})\xrightarrow{30^{\circ}\mathrm{C / min}}280^{\circ}\mathrm{C}\xrightarrow{10^{\circ}\mathrm{C / min}}290^{\circ}\mathrm{C}$ （保持 $6\mathrm{min})$   
c）进样口温度： $250^{\circ}\mathrm{C}$   
d) 载气：氦气，纯度 $99.999\%$   
e) 载气流速: 恒流模式 $1 \mathrm{~mL} / \mathrm{min}$ ;  
f）进样方式：不分流；  
g）进样量： $2\mu \mathrm{L}$   
h）开阀时间：1min。

# 7.3.1.2 质谱条件

a） 接口温度： $280^{\circ}C$   
b) 离子源: 电子轰击源 (EI);  
c) 电离电压: $70 \mathrm{eV}$ ;  
d) 离子源温度: $230^{\circ} \mathrm{C}$ ;   
e) 检测方式：SIM；  
f) 溶剂延迟时间：2.5min；

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中各被测物的含量。

$$
X _ {i} = \frac {A _ {i} \times c _ {i} \times V}{A _ {s i} \times m}. \tag {1}
$$

式中：

$X_{i}$ ——式样中嘧霉胺、嘧菌胺、腈菌唑、嘧菌酯含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A_{i}$ ——样液中嘧霉胺、嘧菌胺、腈菌唑、嘧菌酯的峰面积；

$c_{i}$ ——标准溶液中嘧霉胺、嘧菌胺、腈菌唑、嘧菌酯的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{si}$ ——标准溶液中嘧霉胺、嘧菌胺、腈菌唑、嘧菌酯的峰面积；

$m$ ——最终样液所代表的试样量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 B 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 C 的要求。

# 10 定量限和回收率

# 10.1 定量限

4种农药的定量限分别为：嘧霉胺为 $0.01\mathrm{mg / kg}$ ；嘧菌胺为 $0.01\mathrm{mg / kg}$ ；腈菌唑为 $0.01\mathrm{mg / kg}$ ；嘧菌酯为 $0.005\mathrm{mg / kg}$ 。

# 10.2 回收率

当添加水平为 $0.01 \mathrm{mg} / \mathrm{kg} 、 0.05 \mathrm{mg} / \mathrm{kg} 、 0.1 \mathrm{mg} / \mathrm{kg}$ 时, 喹霉胺、嘧菌胺、腈菌唑和嘧菌酯的添加回收率参见附录 D。

# 鸡可食性组织中地克珠利残留量的测定

# 高效液相色谱法

# 1 范围

本标准规定了鸡可食性组织中地克珠利残留量检测的制样和高效液相色谱测定方法。

本标准适用于鸡的肌肉、肝脏和肾脏中地克珠利残留量的检测。

# 3 原理

试料中残留的地克珠利，用乙腈提取，正已烷除脂，减压浓缩干，反相高效液相色谱-紫外测定，外标法定量。

# 4 试剂与材料

以下所用试剂，除特殊注明者外，均为分析纯试剂；水为符合GB/T6682规定的一级水。

4.1 地克珠利对照品：含量 $\geq 99\%$   
4.2 磷酸  
4.3 正己烷  
4.4 N，N-二甲基甲酰胺  
4.5 甲醇：色谱纯。  
4.6 乙腈：色谱纯。  
4.7 $0.2\%$ 磷酸溶液：取磷酸 $2.34 \mathrm{~mL}$ ，用水溶解并稀释至 $1000 \mathrm{~mL}$ 。

4.8 $1 \mathrm{mg} / \mathrm{mL}$ 地克珠利标准贮备液：精密称取地克珠利对照品 $10 \mathrm{mg}$ ，于 $10 \mathrm{~mL}$ 量瓶中，用 N，N-二甲基甲酰胺溶解并稀释至刻度，配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的地克珠利标准贮备液。 $-20^{\circ} \mathrm{C}$ 以下保存，有效期 6 个月。  
4.9 $10 \mu \mathrm{g} / \mathrm{mL}$ 地克珠利标准工作液: 准确量取 $1 \mathrm{mg} / \mathrm{mL}$ 地克珠利标准贮备液 $1.0 \mathrm{~mL}$ , 于 $100 \mathrm{~mL}$ 量瓶中, 用 $\mathrm{N}, \mathrm{~N}-$ 二甲基甲酰胺稀释至刻度, 配制成浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 的地克珠利标准工作液, $-20^{\circ} \mathrm{C}$ 以下保存, 有效期 3 个月。

# 5 仪器与设备

5.1 高效液相色谱仪：配紫外检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 旋转蒸发仪   
5.5 均质机  
5.6 高速离心机  
5.7 聚丙烯离心管： $50~\mathrm{mL}$ 。  
5.8 茄形瓶： $50~\mathrm{mL}$

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 样品的保存

$-20^{\circ} \mathrm{C}$ 以下保存。

# 7 测定步骤

# 7.1 基质匹配标准曲线的制备

精密量取地克珠利标准工作液适量，分别添加到6份空白试料中，制得浓度为100、250、500、

1000、2500 和 $5000 \mu \mathrm{g} / \mathrm{kg}$ 的系列基质匹配标准溶液，按提取步骤操作，供高效液相色谱法测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.2 提取

称取试料 $(2 \pm 0.02)$ g，于 $50 \mathrm{~mL}$ 聚丙烯离心管中，加乙腈 $10 \mathrm{~mL}$ ，均质 $1 \mathrm{~min}$ ，振摇 $15 \mathrm{~min}$ ，6000 r/min 离心 $10 \mathrm{~min}$ ，收集乙腈提取液于 $50 \mathrm{~mL}$ 茄形瓶中，残渣再重复提取一次，合并两次提取液，正已烷 $5 \mathrm{~mL}$ ，弃正己烷层液，加正丙醇 $5 \mathrm{~mL}$ ，于 $50^{\circ} \mathrm{C}$ 减压蒸干，用流动相 $1.0 \mathrm{~mL}$ 溶解残余物，15000 r/min 离心 $10 \mathrm{~min}$ ，取上清液，供高效液相色谱仪测定。

# 7.3 测定

# 7.3.1 高效液相色谱条件

色谱柱： $C_{18}$ （ $250\mathrm{mm}\times 4.6\mathrm{mm}$ ，粒径 $5\mu \mathrm{m})$ ，或相当者。

流动相：乙腈+0.2%磷酸（57+43，v/v）。

流速： $1\mathrm{mL / min}$

波长： $278\mathrm{nm}$

柱温： $30^{\circ}\mathrm{C}$ 。

进样：进样量 $20~\mu \mathrm{L}$ 。

# 7.3.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试样溶液地克珠利响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和空白添加试样溶液的高效液相色谱图见附录A。

# 7.4 空白实验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算与表述

试料中地克珠利残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ），按下式计算

$$
X = \frac {A \times C S \times V}{A _ {\mathrm {s}} \times m}
$$

式中：

$X$ ——供试试料中地克珠利的残留量， $\mu \mathrm{g} / \mathrm{kg}$ ；

$A$ ——试样中地克珠利的峰面积；

$A_{S}$ ——标准溶液中地克珠利的峰面积；

$C s$ ——标准溶液中地克珠利的浓度, $\mu \mathrm{g} / \mathrm{L}$ ;

$V$ ——最终试样体积, $\mathrm{mL}$ ;

$m$ 供试试料质量， $\mathbf{g}$ □

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度和精密度

# 9.1 灵敏度

本方法的检测限为 $50~\mu \mathrm{g / kg}$ ，定量限为 $250~\mu \mathrm{g / kg}$ 。

# 9.2 准确度

本方法在 $250\sim 2500~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率范围为 $70\% \sim 120\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

# 1 范围

本标准适用于由刺苹婆(Sterculia urens Roxburgh)及其他苹婆属种植物，划破其树干、采取渗出的胶状分泌物，经干燥、粉碎而成的食品添加剂刺梧桐胶。

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度均为分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 溶解性

称取 $2\mathrm{g}$ 试样，加入到 $50~\mathrm{mL}$ 水中，可形成颗粒状、坚硬的、略带乳白色的凝胶，对石蕊呈酸性，不溶于乙醇。

# A.2.2 乙醇溶液中的溶胀

试样可溶胀于 $60\%$ 乙醇溶液中。

# A.2.3 颜色反应

称取 $0.2\mathrm{g}$ 试样，加入到 $20~\mathrm{mL}$ 水中，煮沸至形成黏液。加入 $5\mathrm{mL}$ 盐酸，混合液再煮沸 $5\mathrm{min}$ 。应呈现持久的红色或粉红色。

称取 $0.5\mathrm{g}$ 试样，与 $2\mathrm{mL}5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液混合，缓缓加热，应呈现棕色。

# A.2.4 沉淀试验

称取 $1\mathrm{g}$ 试样，加入到 $80~\mathrm{mL}$ 水中，混合振摇 $24\mathrm{h}$ ，形成黏液。移取 $4\mathrm{mL}$ 黏液，加入 $0.5\mathrm{mL}$ 盐酸，煮沸，再加入 $1\mathrm{mL}5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液，过滤。在滤液中加入 $3\mathrm{mL}$ 费林溶液，加热，应生成红色沉淀。

# A.3 酸不溶灰分的测定

# A.3.1 试剂和材料

盐酸溶液： $1 + 9$

# A.3.2 分析步骤

量取 $25\mathrm{mL}$ 盐酸溶液，加入到测定总灰分时所得的灰分中，煮沸 $5\mathrm{min}$ 。用过滤坩埚或无灰分滤纸过滤，用热水洗涤滤渣数次，在 $675~^\circ \mathrm{C}\pm 25~^\circ \mathrm{C}$ 下灼烧至恒重。

# A.3.3 结果计算

酸不溶灰分的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {m _ {1}}{m _ {0}} \times 100 \% \tag{A.1}
$$

式中：

$m_{1}$ ——灼烧后残渣的质量，单位为克(g)；

$m_0$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。

# A.4 淀粉或糊精的测定

# A.4.1 试剂和材料

碘溶液：称取碘 $14.0\mathrm{g}$ ，溶于含有碘化钾 $36.0\mathrm{g}$ 的 $100\mathrm{mL}$ 水溶液中，加入3滴盐酸，加水稀释至 $1000\mathrm{mL}$ 。

# A.4.2 分析步骤

配制 $10\mathrm{mg / mL}$ 的试样溶液。取适量试样溶液，煮沸后冷却，加入几滴碘溶液，无蓝色或红色出现，即为通过试验。

# 食品安全国家标准

# 食品中低聚半乳糖的测定

# 1 范围

本标准规定了食品中低聚半乳糖的液相色谱测定的方法。

本标准适用于婴幼儿配方食品（不包括特殊医学用途婴儿配方食品）、婴幼儿辅助食品、乳制品、饮料、焙烤食品中低聚半乳糖的测定。

本标准不适用于含有其他低聚还原性物质会造成干扰的食品中低聚半乳糖的测定。

# 2 原理

试样中的低聚半乳糖 $(n$ 为 $2\sim 7)$ 经水溶液提取、2-氨基苯甲酰胺衍生，用淀粉葡糖苷酶酶解去除麦芽糊精、淀粉干扰后，用高效液相色谱-荧光检测器检测。采集的色谱峰根据用相同试验条件下液相色谱串联质谱法确定的相同聚合度的低聚半乳糖的保留时间定性，采用相对应聚合度和相对分子质量的麦芽糖类聚糖标准物质外标法定量，扣除乳糖、麦芽糖含量后，得出低聚半乳糖含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

警示——氰基硼氢化钠和2-甲基吡啶-N-甲硼烷为危险物品，遇水释放易燃气体、对环境有危险，谨遵试剂使用说明规范。请穿戴个人防护装备，在通风柜下进行所有操作。

3.2.2 2-氨基苯甲酰胺衍生溶液 $(0.35\mathrm{mol} / \mathrm{L})$ ：准确称取 $476\mathrm{mg}$ 2-氨基苯甲酰胺和 $630\mathrm{mg}$ 氰基硼氢化钠或 $1070\mathrm{mg}$ 2-甲基吡啶- $N$ -甲硼烷于烧杯中，用 $10~\mathrm{mL}$ 乙酸-二甲基亚砜溶液 $(3 + 7)$ 溶解，混合均匀。  
3.2.3 乙酸铵溶液 $(0.2\mathrm{mol} / \mathrm{L},\mathrm{pH}4.5\pm 0.1)$ ：称取 $1.54\mathrm{g}$ 乙酸铵，用 $80~\mathrm{mL}$ 水溶解后，用乙酸调节 $\mathrm{pH}$ 至 $4.5\pm 0.1$ ，用水稀释至 $100~\mathrm{mL}$ 。临用现配。  
3.2.4 淀粉葡糖苷酶溶液(约 $180~\mathrm{U / mL}$ )：称取相当于活力约 $900\mathrm{U}$ 淀粉葡糖苷酶溶于 $5\mathrm{mL}$ 乙酸铵溶液 $(0.2\mathrm{mol} / \mathrm{L}\mathrm{pH}4.5\pm 0.1)$ 中，搅拌至完全溶解。临用现配。  
3.2.5 甲酸铵溶液 $(50\mathrm{mmol / L},\mathrm{pH}4.4\pm 0.1)$ ：称取 $3.15\mathrm{g}$ 甲酸铵，用 $900~\mathrm{mL}$ 水溶解后，用甲酸调节 $\mathrm{pH}$ 至 $4.4\pm 0.1$ ，用水稀释至 $1000~\mathrm{mL}$ 。临用现配。

# 5.1 样品前处理

# 5.1.1 试样制备

# 5.1.3 试样处理

称取 $1.000\mathrm{g}$ 试样于 $50~\mathrm{mL}$ 烧杯中用水溶解，转移至 $50~\mathrm{mL}$ 容量瓶中，加水定容至刻度，混匀，作为试样液，待用。吸取 $20~\mu \mathrm{L}$ 试样液于 $2\mathrm{mL}$ 具旋盖离心管中，加入 $200~\mu \mathrm{L}2$ -氨基苯甲酰胺衍生溶液，涡旋混合 $30~\mathrm{s}$ ，在 $60^{\circ}\mathrm{C}$ 水浴中保温 $120\mathrm{min}$ ，取出放至室温，加入 $1\mathrm{mL}$ 乙酸铵溶液 $(0.2\mathrm{mol / L},\mathrm{pH}4.5\pm$ 0.1)涡旋混合 $30~\mathrm{s}$ 。再移取 $0.50~\mathrm{mL}$ 混合溶液于 $2\mathrm{mL}$ 离心管中，加入 $200~\mu \mathrm{L}$ 淀粉葡糖苷酶溶液（约 $180~\mathrm{U / mL}$ ），在 $50^{\circ}\mathrm{C}$ 水浴中保温 $30\mathrm{min}$ ，取出放至室温，加入 $0.70~\mathrm{mL}$ 乙腈，混匀后 $6000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液用 $0.22~\mu \mathrm{m}$ 有机微孔滤膜过滤后，供高效液相色谱测定。

# 6 分析结果的表述

试样中低聚半乳糖含量 $X$ 按公式(1)计算。

$$
X = \frac {\left(\sum \rho_ {\mathrm {D P i}} - \rho_ {0} - \rho_ {\text {乳 糖}} - \rho_ {\text {麦 芽 糖}}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中低聚半乳糖含量，单位为克每千克 $(\mathrm{g} / \mathrm{kg})$ ；

$\rho_{\mathrm{DPi}}$ ——由标准曲线得到的不同聚合度糖衍生物的浓度，单位为微克每毫升 $(\mu \mathrm{g / mL})(i = 2\sim 7)$

$\rho_0$ ——空白试样溶液中衍生物的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$\rho_{\text{乳糖}}$ ——试样溶液中乳糖衍生物的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$\rho_{\text{麦芽糖}}$ ——试样溶液中麦芽糖衍生物的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样定容的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，保留3位有效数字。

# 7 精密度

在重复性条件下，获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

当称样量为 $1.000\mathrm{g}$ 时，检出限为 $1.0\mathrm{g / kg}$ ，定量限为 $3.0\mathrm{g / kg}$ 。

# 其他低聚还原性物质鉴别方法

# A.1 原理

试样溶液采用淀粉葡糖苷酶除麦芽糊精、淀粉干扰后， $\beta$ -半乳糖苷酶水解样品中的乳糖和低聚半乳糖后，经2-氨基苯甲酰胺衍生，用高效液相色谱-荧光检测器检测。将采集信号根据糖聚合度保留时间分段后分别峰加和，采用相对应聚合度麦芽糖类聚糖标准物质外标法分别定量后计算总的衍生物含量，扣除空白溶液和麦芽糖衍生物含量后计算其他低聚还原性物质总含量。若试样中其他低聚还原性物质总含量大于定量限，则标准方法不适用于该试样中低聚半乳糖的测定。

# A.4.2.1 试样处理

称取 $1.000\mathrm{g}$ 试样于 $50~\mathrm{mL}$ 烧杯中用水溶解，转移至 $50~\mathrm{mL}$ 容量瓶中，加水定容至刻度，混匀，作为试样液，待用。吸取 $500~\mu \mathrm{L}$ 试样液于 $2\mathrm{mL}$ 具旋盖离心管中，加入 $200~\mu \mathrm{L}$ 淀粉葡糖苷酶溶液（约 $180\mathrm{U / mL})$ ， $50~\mu \mathrm{L}\beta$ -半乳糖苷酶溶液(4000U/mL)，涡旋混合 $30~s$ ，在 $60^{\circ}C$ 水浴中保温 $120\mathrm{min}$ ，随后水浴温度升至 $100^{\circ}C$ 保温 $5\mathrm{min}$ ，取出置于 $4^{\circ}C$ 冷却 $10\mathrm{min}$ 。吸取 $20~\mu \mathrm{L}$ 混合溶液于 $2\mathrm{mL}$ 离心管中，加入 $200~\mu \mathrm{L}2$ -氨基苯甲酰胺衍生溶液，涡旋混合 $30~s$ ，在 $60^{\circ}C$ 水浴中保温 $120\mathrm{min}$ ，取出放至室温，加入 $1\mathrm{mL}$ 乙腈溶液 $(75\%)$ ，混匀后 $6000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液用 $0.22~\mu \mathrm{m}$ 有机微孔滤膜过滤后，供高效液相色谱测定。

# A.5 分析结果的表述

试样中其他低聚还原性物质含量 $W$ 按公式(A.1)计算。

$$
W = \frac {\left(\sum \rho_ {\mathrm {D P i}} - \rho_ {0} - \rho_ {\text {麦 芽 糖}}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \quad \dots \tag {A.1}
$$

式中：

$W$ ——试样中其他低聚还原性物质含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho_{\mathrm{DPi}}$ ——由标准曲线得到的不同聚合度糖衍生物的浓度，单位为微克每毫升 $(\mu \mathrm{g / mL})(i = 2\sim 7)$

$\rho_0$ ——空白试样溶液中衍生物的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$\rho$ 麦芽糖 试样溶液中麦芽糖衍生物的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$

$V$ ——试样定容的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——试样的质量，单位为克(g)。

以重复性条件下获得的2次独立测定结果的算术平均值表示，计算结果保留3位有效数字。

# A.6 鉴别规则

其他低聚还原性物质含量大于定量限 $3.0\mathrm{g / kg}$ ，则视为样品中含有其他低聚还原性物质。

# A.7 其他低聚还原性物质

本标准列举出其他常见低聚还原性物质，但不限于此，如低聚异麦芽糖、低聚木糖、聚葡萄糖、果葡糖浆、玉米糖浆、神经节苷酯、2'-岩藻糖基乳糖、3-岩藻糖基乳糖、乳糖-N-新四糖、3'-唾液酸化乳糖、6'-唾液酸化乳糖。如含有该类物质，且含量大于 $3.0\mathrm{g / kg}$ ，则视为样品中含有其他低聚还原性物质，无需再单独验证方法适用性。

如使用不同品牌的高效液相色谱仪，需将液相色谱-质谱法测定的麦芽糖衍生物的保留时间 $(t_{\text{麦芽糖-MS}})$ 与液相色谱法测定的麦芽糖衍生物的保留时间 $(t_{\text{麦芽糖-HPLC}})$ 进行校正，得出不同仪器间的相对死体积时间差 $(t_0)$ 。该数值代入液相色谱-质谱法测定的不同聚合度糖保留时间段 $(t_{\mathrm{DPi - MS}})$ 对液相色谱

法测定的不同聚合度糖保留时间段 $(t_{\mathrm{DPi - HPLC}})$ 进行校正，如果液相色谱-质谱法测定的麦芽糖衍生物的保留时间早于液相色谱法测定的麦芽糖衍生物的保留时间，则液相色谱-质谱法测定的不同聚合度糖保留时间段 $(t_{\mathrm{DPi - MS}})$ 加上两仪器间的死体积时间差 $(t_0)$ ，反之则减去两仪器间的死体积时间差 $(t_0)$ ，可得到高效液相色谱仪测定的不同聚合度糖保留时间段 $(t_{\mathrm{DPi - HPLC}})$ 。

计算液相色谱-质谱仪和高效液相色谱仪死体积时间差见公式D.1。

$$
t _ {0} = \left| t _ {\text {麦 芽 糖 － M S}} - t _ {\text {麦 芽 糖 － H P L C}} \right| \quad \dots \dots \dots \dots \dots \dots \dots \dots \tag {D.1}
$$

式中：

$t_0$ ——液相色谱-质谱仪和液相色谱仪死体积时间差，单位为分 $(\mathrm{min})$ ；

$t_\text{麦芽糖-MS}$ ——液相色谱-质谱法测定的麦芽糖衍生物的保留时间，单位为分(min)；

$t_\text{麦芽糖 - HPLC}$ ——液相色谱法测定的麦芽糖衍生物的保留时间，单位为分 $(\mathrm{min})$ 。

液相色谱-质谱法测定的不同聚合度糖保留时间段 $t_{\mathrm{DPiMS}}$ 为 $t_{\mathrm{DPi}}$ 起始 $\sim t_{\mathrm{DPi}}$ 终止。

液相色谱法测定的不同聚合度糖保留时间段 $t_{\mathrm{DPi - HPLC}}$ 为 $(t_{\mathrm{DPi}\text{起始}}\pm t_0)\sim (t_{\mathrm{DPi}\text{终止}}\pm t_0)$ 。

注：液相色谱测定的麦芽糖衍生物的保留时间稳定的前提下，该附录方法无需多次测定判别不同聚合度糖保留时间段。

# 1 范围

本文件规定了奶及奶粉中阿维菌素类药物残留检测的制样和液相色谱-串联质谱测定方法。

本文件适用于牛奶、羊奶和奶粉中阿维菌素（以阿维菌素Bla计）、伊维菌素（以22，23-二氢阿维菌素Bla计）、多拉菌素和乙酰氨基阿维菌素（以乙酰氨基阿维菌素Bla计）单个或多个药物的残留检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的阿维菌素类药物，用乙腈提取， $\mathrm{C_{48}}$ 固相萃取柱净化，液相色谱串联质谱测定，基质匹配标准溶液外标法定量。

# 5 试剂和材料

# 5.1 试剂

以下所用试剂，除特别注明外均为分析纯试剂，水为符合GB/T6682规定的一级水。

5.1.1 乙腈 $\mathrm{(CH_3CN)_2}$ 色谱纯。  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯  
5.1.3 甲酸 $(\mathrm{CH}_2\mathrm{O}_2)$ ：色谱纯  
5.1.4 三乙胺 $(\mathrm{C_6H_{15}N})$   
5.1.5 异辛烷 $(\mathrm{C_8H_{18}})$

# 5.2 溶液配制

5.2.1 $0.1\%$ 甲酸乙腈溶液：取甲酸 $1\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$ 。  
5.2.2 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$   
5.2.3 淋洗溶液：取乙腈 $30~\mathrm{mL}$ 、水 $70~\mathrm{mL}$ 和三乙胺 $20~\mu \mathrm{L}$ ，混匀。  
5.2.4 $50\%$ 乙腈溶液：取乙腈 $50~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$

# 5.3 标准品

阿维菌素(Avermectin, $\mathrm{C_{48}H_{72}O_{14}}\cdot \mathrm{C_{47}H_{70}O_{14}}$ ,CAS号：71751-41-2)，含量 $\geq 98.0\%$ ；伊维菌素(Iver-mectin, $\mathrm{C_{48}H_{74}O_{14}}$ ,CAS号：70288-86-7)，含量 $\geq 98.0\%$ ；多拉菌素（Doramectin, $\mathrm{C_{50}H_{74}O_{14}}$ ,CAS号：117704-25-3)，含量 $\geq 94.3\%$ ；乙酰氨基阿维菌素(Eprinomectin, $\mathrm{C_{50}H_{75}NO_{14}}$ ,CAS号：123997-26-2），含量 $\geq 98.0\%$ 。

# 5.4 标准溶液的制备

5.4.1 阿维菌素类药物标准储备液：精密称取阿维菌素（以阿维菌素Bla计）、伊维菌素（以22，23-二氢

阿维菌素Bla计)、多拉菌素和乙酰氨基阿维菌素(以乙酰氨基阿维菌素Bla计)标准品各约 $10\mathrm{mg}$ ，于不同的 $10\mathrm{mL}$ 容量瓶中，用乙腈溶解并定容至刻度，配制成浓度为 $1\mathrm{mg/mL}$ 的阿维菌素Bla、22，23-二氢阿维菌素Bla、多拉菌素和乙酰氨基阿维菌素Bla标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期12个月。

5.4.2 阿维菌素类药物标准工作液：精密量取 $1\mathrm{mg / mL}$ 的阿维菌素 $\mathrm{Bla}, 22, 23-$ 二氢阿维菌素 $\mathrm{Bla},$ 多拉菌素和乙酰氨基阿维菌素 $\mathrm{Bla}$ 标准储备液 $100\mu \mathrm{L}$ ，于不同的 $10\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，配制成浓度为 $10\mu \mathrm{g / mL}$ 的阿维菌素 $\mathrm{Bla}, 22, 23-$ 二氢阿维菌素 $\mathrm{Bla},$ 多拉菌素和乙酰氨基阿维菌素 $\mathrm{Bla}$ 标准工作液。 $-18^{\circ}C$ 以下保存，有效期6个月。

# 5.5 材料

5.5.1 $\mathrm{C_{18}}$ 固相萃取柱： $500\mathrm{mg / 6mL}$ ，或相当者。  
5.5.2 亲水聚四氟乙烯微孔滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配有电喷雾离子源(ESI)。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 涡旋混合器。  
6.4 高速离心机。  
6.5 氮吹仪。  
6.6 水平振荡器。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试牛奶或羊奶及奶粉，混合均匀。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

牛奶和羊奶：冷藏避光或 $-18^{\circ} \mathrm{C}$ 以下保存。

奶粉：阴凉干燥处保存。

# 8 测定步骤

# 8.1 提取

称取牛奶或羊奶 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ )或奶粉 $0.5\mathrm{g}$ （准确至 $\pm 0.01\mathrm{g}$ )于 $50~\mathrm{mL}$ 离心管中，加乙腈 $8\mathrm{mL}$ ，涡旋后 $200\mathrm{r / min}$ 水平振荡 $5\mathrm{min},8000\mathrm{r / min}$ 离心 $8\mathrm{min}$ ，转移上清液至另一 $50~\mathrm{mL}$ 离心管中，加水 $10~\mathrm{mL}$ 、三乙胺 $25~\mu \mathrm{L}$ ，混匀，备用。

# 8.2 净化与浓缩

$\mathrm{C_{18}}$ 固相萃取柱依次用乙腈 $5\mathrm{mL}$ 和淋洗溶液 $5\mathrm{mL}$ 活化，取备用液过柱，用淋洗溶液 $5\mathrm{mL}$ 淋洗，抽干；再用异辛烷 $5\mathrm{mL}$ 淋洗，抽干；用乙腈 $5\mathrm{mL}$ 洗脱，收集洗脱液于 $10\mathrm{mL}$ 试管中，于 $50^{\circ}\mathrm{C}$ 下氮气吹干。残余物中加入 $50\%$ 乙腈溶液 $1.0\mathrm{mL}$ ，充分溶解。过微孔滤膜后供液相色谱-串联质谱仪测定。

# 8.3 基质匹配标准曲线的制备

精密量取阿维菌素类标准工作液适量，用 $50\%$ 乙腈水溶液稀释成含阿维菌素B1a、22，23-二氢阿维菌素Bla、多拉菌素和乙酰氨基阿维菌素Bla浓度分别为 $0.5\mathrm{ng / mL},1\mathrm{ng / mL},2\mathrm{ng / mL},5\mathrm{ng / mL}$ $10~\mathrm{ng / mL},20~\mathrm{ng / mL},50~\mathrm{ng / mL},100~\mathrm{ng / mL}$ 和 $400~\mathrm{ng / mL}$ 的系列标准工作液（牛奶或羊奶用 $0.5\mathrm{ng / mL},1\mathrm{ng / mL},2\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL}$ 和 $50~\mathrm{ng / mL}$ 浓度点，奶粉用 $2\mathrm{ng / mL}$

$5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL},50\mathrm{ng / mL},100\mathrm{ng / mL}$ 和 $400\mathrm{ng / mL}$ 浓度点），从中各取 $1.0~\mathrm{mL}$ ，分别加入空白试料经提取、净化和吹干后的残余物中，充分溶解，作为基质匹配标准溶液，过微孔滤膜后上机测定。以特征离子质量色谱峰面积为纵坐标、基质匹配标准溶液浓度为横坐标，绘制标准曲线。

# 8.4 测定

# 8.4.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}(50~mm\times 2.1~mm,1.7~\mu m)}$ ，或性能相当者。  
b）流动相：A为 $0.1\%$ 甲酸溶液，B为 $0.1\%$ 甲酸乙腈溶液。流动相梯度： $0\mathrm{min}\sim 1\mathrm{min}$ 保持 $70\%$ B; $1\mathrm{min}\sim 3\mathrm{min}$ ， $70\%$ B线性变化到 $100\% \mathrm{B};3\mathrm{min}\sim 4\mathrm{min}$ 保持 $100\% \mathrm{B};4\mathrm{min}\sim 6\mathrm{min}$ 保持 $70\% \mathrm{B}$ 。  
c) 流速: $0.4 \mathrm{~mL} / \mathrm{min}$ 。  
d) 进样量: $5 \mu \mathrm{L}$ 。  
e） 柱温： $30^{\circ}C$

# 8.4.2 串联质谱参考条件

a）离子源：电喷雾离子源；  
b）扫描方式：正离子扫描；  
c）检测方式：多反应离子监测(MRM)；  
d）电喷雾电压： $5500\mathrm{V}$   
e) 离子源温度: $500^{\circ} \mathrm{C}$ ;   
f）辅助气1：50psi；  
g）辅助气2：50psi；  
h）气帘气：30 psi；  
i）碰撞气：Medium。  
j）待测药物定性、定量离子对和对应的去簇电压、碰撞能量参考值见表1。

# 8.4.3 测定法

试样溶液的保留时间在基质匹配标准溶液保留时间的 $\pm 2.5\%$ 之内。试样溶液中的离子相对丰度与基质匹配标准溶液中的离子相对丰度相比，符合表2的要求。

取基质匹配标准溶液和试样溶液，作单点或多点校准，按外标法，以峰面积计算。基质匹配标准溶液及试样溶液中阿维菌素类药物的峰面积均应在仪器检测的线性范围之内，超出线性范围时进行适当倍数稀释后再进行分析。基质匹配标准溶液中特征离子质量色谱图见附录A。

# 8.5 空白试验

取空白试料，除不添加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中阿维菌素类药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C \times A \times V \times 1 0 0 0}{A _ {\mathrm {s}} \times m \times 1 0 0 0} \dots \tag {1}
$$

式中：

$X$ ——试样中阿维菌素类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C$ ——基质匹配标准溶液中阿维菌素类药物浓度的数值，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$A$ ——试样溶液中阿维菌素类药物的峰面积；

$V$ ——溶解残余物所用溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ 基质匹配标准溶液中阿维菌素类药物的峰面积；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法对牛奶或羊奶的检测限为 $0.2\mu \mathrm{g / kg}$ ，定量限为 $0.5\mu \mathrm{g / kg}$ ；对奶粉的检测限为 $1\mu \mathrm{g / kg}$ ，定量限为 $4\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法牛奶中阿维菌素 Bla 在 $0.5 \mu \mathrm{g} / \mathrm{kg} \sim 5 \mu \mathrm{g} / \mathrm{kg}, 22, 23$ 二氢阿维菌素 Bla 在 $0.5 \mu \mathrm{g} / \mathrm{kg} \sim 20 \mu \mathrm{g} / \mathrm{kg}$ 、多拉菌素在 $0.5 \mu \mathrm{g} / \mathrm{kg} \sim 30 \mu \mathrm{g} / \mathrm{kg}$ 、乙酰氨基阿维菌素 Bla 在 $0.5 \mu \mathrm{g} / \mathrm{kg} \sim 40 \mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上，羊奶中阿维菌素类药物在 $0.5 \mu \mathrm{g} / \mathrm{kg} \sim 5 \mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上，以及奶粉中阿维菌素 Bla 在 $4 \mu \mathrm{g} / \mathrm{kg} \sim 40 \mu \mathrm{g} / \mathrm{kg}, 22, 23$ -二氢阿维菌素 Bla 在 $4 \mu \mathrm{g} / \mathrm{kg} \sim 160 \mu \mathrm{g} / \mathrm{kg}$ 、多拉菌素在 $4 \mu \mathrm{g} / \mathrm{kg} \sim 240 \mu \mathrm{g} / \mathrm{kg}$ 、乙酰氨基阿维菌素 Bla 在 $4 \mu \mathrm{g} / \mathrm{kg} \sim 320 \mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率均为 $60\% \sim 120\%$ 。

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

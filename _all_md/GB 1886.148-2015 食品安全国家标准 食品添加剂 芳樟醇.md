# 食品安全国家标准

# 食品添加剂 芳樟醇

# 1 范围

本标准适用于食品添加剂芳樟醇。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

3,7-二甲基-1,6-辛二烯-3-醇

# 2.2 分子式

$\mathrm{C_{10}H_{18}O}$

# 2.3 结构式

![](images/f6c5b9d992a7bb77280d5dc6b1465ee9a3e9c48e95a2a78128296a3ef5b05351.jpg)

154.25（按2007年国际相对原子质量）

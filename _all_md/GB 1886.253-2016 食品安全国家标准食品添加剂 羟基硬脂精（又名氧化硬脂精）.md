# (又名氧化硬脂精)

# 1 范围

本标准适用于以经过氢化的食用油为原料，经熔化、脱色、加热反应、过滤等步骤加工制得的食品添加剂羟基硬脂精(又名氧化硬脂精)。食品添加剂羟基硬脂精(又名氧化硬脂精)是部分氧化的硬脂酸和其他脂肪酸的甘油酯的混合物。

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度均为分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 酸值（以KOH计）的测定

# A.2.1 试剂和材料

A.2.1.1 中性乙醇溶液： $95\%$ 乙醇，加入 $2\mathrm{mL}$ 酚酞指示液，用氢氧化钾-乙醇标准滴定溶液中和至微红色，并保持 $30\mathrm{s}$ 不褪色。

A.2.1.2 氢氧化钾-乙醇标准滴定溶液： $c(\mathrm{KOH}) = 0.1\mathrm{mol / L}$

A.2.1.3 酚酞指示液： $10\mathrm{g / L}$

# A.2.2 分析步骤

称取试样 $10\mathrm{g}$ ，精确至 $0.001\mathrm{g}$ ，置于锥形瓶中，加入 $50~\mathrm{mL}$ 中性乙醇溶液，加热使试样完全溶解，用氢氧化钾-乙醇标准滴定溶液滴定至溶液呈微红色，并维持 $30~\mathrm{s}$ 不褪色，即为滴定终点。

# A.2.3 结果计算

酸值(以KOH计)的质量分数 $\omega_{1}$ ，单位为毫克每克 $(\mathrm{mg / g})$ ，按式(A.1)计算：

$$
w _ {1} = \frac {V \times c \times M}{m} \tag {A.1}
$$

式中：

$V$ ——试样消耗氢氧化钾-乙醇标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——氢氧化钾-乙醇标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——氢氧化钾的摩尔质量，单位为毫克每毫摩尔 $(\mathrm{mg / mol})[M(\mathrm{KOH}) = 56.1]$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# A.3 折射率 $(48\mathrm{C})$ 的测定

# A.3.1 仪器和设备

阿贝折光仪。

# A.3.2 分析步骤

将试样熔融后用滤纸过滤，在 $48^{\circ}\mathrm{C}$ 条件下用阿贝折光仪检测。

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2648—2010《进出口食品中啶酰菌胺残留量的测定 气相色谱一质谱法》。与SN/T2648—2010相比，主要变化如下：

# 1 范围

本标准规定了食品中啶酰菌胺残留量的气相色谱一质谱检测方法。

本标准适用于菠菜、胡萝卜、草莓、花生、板栗、茶叶、葱、鸡肉、鳝和蜂蜜中啶酰菌胺残留量的检测和确证，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB/T6682 分析实验室用水规格和试验方法。

# 3 原理

试样中啶酰菌胺用乙腈提取，经液/液分配和丙基乙二胺键合硅胶(PSA)固相萃取柱净化，用气相色谱一电子轰击源质谱测定，外标法定量。

# 4 试剂和材料

除另有规定外，所用试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ ：农药残留级。  
4.1.2 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ ：农药残留级。  
4.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：农药残留级。  
4.1.4 氯化钠 $(\mathrm{NaCl})$ ： $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置入干燥器中冷却，备用。

# 4.2 溶液配制

4.2.1 正己烷饱和乙腈：取 $100\mathrm{mL}$ 正己烷， $100\mathrm{mL}$ 乙腈于分液漏斗中，振荡混匀，取下层待用。  
4.2.2 乙腈饱和正己烷：取 $100\mathrm{mL}$ 正己烷， $100\mathrm{mL}$ 乙腈于分液漏斗中，振荡混匀，取上层待用。  
4.2.3 丙酮—正己烷(3+7，体积比)：取 $300~\mathrm{mL}$ 丙酮，加入 $700~\mathrm{mL}$ 正己烷，摇匀备用。

# 4.3 标准品

4.3.1啶酰菌胺标准物质(boscalid，CAS号： $188425 - 85 - 6$ ，分子式： $\mathrm{C_{18}H_{12}C_{12}N_2O)}$ ：纯度 $\geqslant 98.0\%$

# 4.4 标准溶液配制

4.4.1 喹酰菌胺标准溶液：准确称取适量标准品用丙酮溶解并配制成浓度为 $1.0\mathrm{mg / mL}$ 的标准储备液。根据需要再用不含啶酰菌胺的空白样品基质溶液稀释成适当浓度的标准工作溶液。保存于 $-18^{\circ}C$ 中。

# 4.5 材料

4.5.1 丙基乙二胺键合硅胶(primary secondary amine, PSA)固相萃取柱： $3\mathrm{mL}/500\mathrm{mg}$ ，或相当者。

# 5 仪器和设备

5.1 气相色谱—质谱仪：配有电子轰击源（EI）。  
5.2 固相萃取装置。  
5.3 均质器，转速 $10000\mathrm{r / min}$   
5.4 旋转蒸发器。  
5.5 氮气浓缩仪。  
5.6 具塞离心管： $50~\mathrm{mL},100~\mathrm{mL}$   
5.7 浓缩瓶： $50~\mathrm{mL},100~\mathrm{mL}$   
5.8 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按GB2763的规定执行。在取样和制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 粮谷和坚果

取有代表性样品量 $500\mathrm{g}$ , 全部磨碎并通过 $2.0\mathrm{mm}$ 圆孔筛。混匀, 装入洁净的容器内, 密闭, 标明标记。

# 6.2.2 水果和蔬菜

取有代表性样品 $500\mathrm{g}$ , 切碎后 (不可水洗), 用食品捣碎机将样品加工成浆状。混匀, 装入洁净的容器内, 密闭, 标明标记。

# 6.2.3 肉和鱼

取有代表性样品 $500\mathrm{g}$ , 充分搅碎均匀, 装入洁净容器内作为试样, 密闭, 标明标记。

# 6.2.4 蜂蜜

取有代表性样品 $500 \mathrm{~g}$ , 对无结晶的蜂蜜样品将其搅拌均匀; 对有结晶析出的蜂蜜样品, 在密闭情况下, 将样品瓶置于不超过 $60^{\circ} \mathrm{C}$ 的水浴中温热, 振荡, 待样品全部融化后搅匀, 迅速冷却至室温, 在融化时必须注意防止水分挥发。装入洁净的容器内, 密闭, 标明标记。

# 6.2.5 茶叶

取有代表性样品 $500\mathrm{g}$ , 用磨碎机全部磨碎并通过 $2.0\mathrm{mm}$ 圆孔筛。混匀, 装入洁净的容器内, 密闭, 标明标记。

# 6.3 试样保存

粮谷和坚果类试样于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 保存; 其他类试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 7 测定步骤

# 7.1 提取

称取 $10\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ ) 于 $100\mathrm{mL}$ 具塞离心管中，加入 $10\mathrm{mL}$ 水，准确加入 $40.0\mathrm{mL}$ 正己烷饱和乙腈，均质器提取 $2\mathrm{min}$ （蜂蜜需加水 $10\mathrm{mL}$ 和乙腈剧烈振荡 $20\mathrm{min}$ ），再加入 $5\mathrm{g}$ 氯化钠，剧烈振荡 $10\mathrm{min}$ ， $3000\mathrm{r/min}$ 离心 $10\mathrm{min}$ ，待净化。

# 7.2 净化

# 7.2.1 液/液分配净化

取上层提取液 $20.0\mathrm{mL}$ （菠菜、胡萝卜、草莓)或 $10.0\mathrm{mL}$ （花生、鸡肉、葱、鳕、蜂蜜、板栗和茶叶）转移至 $50~\mathrm{mL}$ 具塞离心管中，加入 $10\mathrm{mL}$ 乙腈饱和正己烷，振摇 $3\mathrm{min}$ ，静置分层，弃去上层正己烷相，再用 $10\mathrm{mL}$ 乙腈饱和正己烷重复操作一次，弃去正己烷相；下层乙腈相收集于 $100\mathrm{mL}$ 浓缩瓶中，于 $40^{\circ}\mathrm{C}$ 水浴中浓缩至近 $1\mathrm{mL}$ 。

# 7.2.2 固相萃取(SPE)净化

用 $5\mathrm{mL}$ 丙酮一正己烷预淋洗PSA柱。将样液倾入柱中，用 $10\mathrm{mL}$ 丙酮一正己烷洗脱，控制流速小于 $2\mathrm{mL} / \mathrm{min}$ 。收集全部洗脱液于 $40^{\circ}\mathrm{C}$ 水浴中浓缩至近干，氮气吹干。用丙酮一正己烷溶解并定容至 $1.0\mathrm{mL}$ ，气相色谱一质谱仪测定。

# 7.3 测定

# 7.3.1 气相色谱一质谱参考条件

a）色谱柱：HP-5MS石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度：初始温度为 $70^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$ ，以 $25^{\circ}\mathrm{C} / \mathrm{min}$ 程序升温至 $150^{\circ}\mathrm{C}$ ，以 $15^{\circ}\mathrm{C} / \mathrm{min}$ 程序升温至 $200^{\circ}\mathrm{C}$ ，再以 $10^{\circ}\mathrm{C} / \mathrm{min}$ 程序升温至 $280^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ ；  
c）进样口温度： $250^{\circ}\mathrm{C}$   
d） 色谱—质谱接口温度： $280^{\circ}\mathrm{C}$   
e) 载气：氦气，纯度 $\geqslant 99.999\%$ ，恒流模式， $1\mathrm{mL} / \mathrm{min}$   
f) 进样量: $1 \mu \mathrm{L}$ ;   
g）进样方式：无分流进样， $0.65\mathrm{min}$ 后开阀；  
h）电离方式：电子轰击  
i）离子源温度： $230^{\circ}\mathrm{C}$   
j） 四极杆温度： $150^{\circ}\mathrm{C}$   
1）测定方式：选择离子监测方式；  
m) 选择监测离子 $(m / z)$ : 定量 140, 定性 112、167、342;  
n）溶剂延迟： $4.0\mathrm{min}$

# 7.3.2 色谱测定与确证

根据样液中被测物含量情况，选定浓度相近的基质标准工作溶液，对标准工作溶液与样液等体积参插进样测定，基质标准工作溶液和待测样液中啶酰菌胺的响应值均应在仪器检测的线性范围内。

如果样液与标准工作溶液的选择离子色谱图中，在相同保留时间有色谱峰出现，并且在扣除背景后的样品质量色谱中，所选离子均出现，所选择离子的丰度比与标准品对应离子的丰度比，其值在允许范围内（允许范围见表1）。在上述色谱条件下啶酰菌胺保留时间约 $18.8\mathrm{min}$ ，其监测离子 $(m / z)$ 为 $m / z$ $140, m / z$ $112, m / z$ $167, m / z$ $342$ （其相对丰度比为 $100:33:13:27$ ）对其进行确证；根据定量离子 $m / z$ $140$ 对其进行外标法定量。在上述色谱条件下啶酰菌胺标准物的气相色谱—质谱总离子流色谱图和全扫描质谱图见图A.1和图A.2。

# 7.4 空白试验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中啶酰菌胺残留量。

$$
X = \frac {h \times c \times V}{h _ {s} \times m} \tag {1}
$$

式中：

$X$ ——试样中啶酰菌胺残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$h$ ——样液中啶酰菌胺的色谱峰高；

$c$ ——标准工作液中啶酰菌胺的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$h_s$ ——标准工作液中啶酰菌胺的色谱峰高；

$m$ ——最终样液所代表的试样质量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 B 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 C 的要求。

# 10 定量限和回收率

# 10.1 定量限

本方法啶酰菌胺的定量限为 $0.01\mathrm{mg / kg}$

# 10.2 回收率

当添加水平为 $0.01 \mathrm{mg} / \mathrm{kg}, 0.02 \mathrm{mg} / \mathrm{kg}, 0.04 \mathrm{mg} / \mathrm{kg}$ 时, 喹酰菌胺的添加回收率参见附录 D。

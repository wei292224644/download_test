# 1 范围

本文件规定了牛奶中氯前列醇残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于牛奶中氯前列醇残留量的检测。

# 4 原理

试样中残留的氯前列醇，用乙腈提取，混合阴离子交换固相萃取柱净化，液相色谱-串联质谱负离子模式测定，外标法定量。

# 5 试剂和材料

以下所用的试剂，除特别注明外均为分析纯试剂；水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{CO})_3$   
5.1.3 甲酸 $(\mathrm{CH}_2\mathrm{O}_2)$   
5.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)_2}$   
5.1.5 乙酸铵 $\mathrm{(CH_3COONH_2)}$ ：色谱纯

# 5.2 标准品

# 5.3 溶液配制

5.2.1 氯前列醇(Cloprostenol, $\mathrm{C_{22}H_{29}ClO_6}$ ,CAS号：40665-92-7)，含量 $\geq 98\%$ 。  
5.3.1 $0.1\%$ 甲酸溶液：取甲酸 $1 \mathrm{~mL}$ ，用水稀释至 $1000 \mathrm{~mL}$ ，混匀。  
5.3.2 $5\%$ 氨水溶液：取氨水 $50~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.3.3 $2\%$ 甲酸乙腈溶液：取甲酸 $20~\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.3.4 $0.1\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液：称取乙酸铵 $7.70\mathrm{g}$ ，用水溶解并稀释至 $1000\mathrm{mL}$ ，混匀。  
5.3.5 $5\mathrm{mmol / L}$ 乙酸铵溶液：取 $0.1\mathrm{mol / L}$ 乙酸铵溶液 $50~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.3.6 乙腈乙酸铵溶液：取乙腈 $30~\mathrm{mL}$ ，加 $5\mathrm{mmol / L}$ 乙酸铵溶液至 $100~\mathrm{mL}$ ，混匀。

# 5.4 标准溶液制备

5.4.1 标准储备液：取氯前列醇钠对照品约 $10\mathrm{mg}$ ，精密称定，加乙腈适量使溶解并稀释定容至 $10\mathrm{mL}$ 容量瓶，配制成浓度为 $1\mathrm{mg/mL}$ 的氯前列醇标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期3个月。

5.4.2 标准中间液：准确量取标准储备液 $0.1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶，用乙腈稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的标准中间液。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1个月。  
5.4.3 系列标准工作液：准确量取标准中间液适量，用乙酸铵乙腈稀释配制成浓度为 $5\mathrm{ng / mL}$ 、 $10\mathrm{ng / mL},20\mathrm{ng / mL},50\mathrm{ng / mL},100\mathrm{ng / mL},500\mathrm{ng / mL}$ 的系列标准工作溶液。现用现配。

5.5 材料

5.5.1 混合阴离子交换固相萃取柱： $60\mathrm{mg} / 3\mathrm{mL}$ 或其性能相当者。  
5.5.2 滤膜： $0.22\mu \mathrm{m}$ ，有机相。  
5.5.3 离心管： $10\mathrm{mL},50\mathrm{mL}$

6 仪器和设备

6.1 高效液相色谱-串联质谱仪：配有电喷雾离子源（ESI）。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$   
6.3 涡旋混合器。  
6.4 超声仪。  
6.5 冷冻离心机。  
6.6 氮吹仪。  
6.7 固相萃取装置。

7 试样的制备与保存

7.1 试样的制备

取适量新鲜或解冻的空白或供试试样，并均质。

a）取均质的供试样品，作为供试试样；  
b）取均质的空白样品，作为空白试样；  
c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下储存。

8 测定步骤

8.1 提取

取试料 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ )于 $50~\mathrm{mL}$ 离心管中，加乙腈 $5\mathrm{mL}$ ，无水硫酸钠 $2\mathrm{g}$ ，涡旋混匀，超声提取 $10\mathrm{min}$ ，于 $4^{\circ}C$ 10000r/min离心 $10\mathrm{min}$ ，取上清液于 $10\mathrm{mL}$ 离心管中，重复提取1次，合并2次上清液。 $40^{\circ}C$ 氮气吹干，残余物用 $1\mathrm{mL}$ 乙腈溶解，加 $0.1\%$ 甲酸水溶液 $3\mathrm{mL}$ ，涡旋混匀，备用。

8.2 净化

取混合阴离子交换固相萃取柱，用乙腈 $3\mathrm{mL}$ 活化，水 $3\mathrm{mL}$ 平衡。取备用液过柱，用 $5\%$ 氨水溶液 $3\mathrm{mL}$ 淋洗，抽干。用 $2\%$ 甲酸乙腈溶液 $3\mathrm{mL}$ 洗脱，收集洗脱液， $40^{\circ}\mathrm{C}$ 氮气吹干，残余物用 $1\mathrm{mL}$ 乙腈乙酸铵溶液溶解，用微孔滤膜过滤，供液相色谱-串联质谱测定。

8.3 基质匹配标准曲线的制备

准确量取氯前列醇的标准工作液各 $1\mathrm{mL}$ ，分别溶解经提取、净化及吹干后的空白试料残余物，滤膜过滤，配制成氯前列醇浓度为 $5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL},50\mathrm{ng / mL},100\mathrm{ng / mL},500\mathrm{ng / mL}$ 的系列基质匹配标准溶液，供液相色谱-串联质谱测定，以特征离子质量色谱峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

b）流动相：A为 $5\mathrm{mmol / L}$ 乙酸铵溶液；B为乙腈；

d) 进样量: $5 \mu \mathrm{L}$ ;

# 8.4.3 测定法

取试料溶液和基质匹配标准溶液，按8.4.1和8.4.2设定仪器条件操作，作单点或多点校准，外标法计算。基质匹配标准溶液及试料溶液中目标药物的特征离子质量色谱峰峰面积均应在仪器检测的线性范围之内。试料溶液中待测物质的保留时间与基质匹配标准工作液中待测物质的保留时间之比，偏差在 $\pm 2.5\%$ 以内，且试料溶液中的离子相对丰度与基质匹配标准溶液中的离子相对丰度相比，符合表3的要求，则可判定为样品中存在对应的待测物质。相关谱图见附录A。

# 8.5 空白试验

取空白试料，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算与表述

试样中氯前列醇的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times C _ {\mathrm {s}} \times V}{A _ {\mathrm {s}} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中氯前列醇残留量的数值，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——试样溶液中氯前列醇的峰面积；

$A_{\mathrm{S}}$ 基质匹配标准溶液中氯前列醇的峰面积；

$C_{\mathrm{s}}$ ——基质匹配标准溶液中氯前列醇浓度的数值，单位为纳克每毫升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样溶液最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ 供试试样质量的数值，单位为克 $(\mathrm{g})$

# 10 检测方法的灵敏度、准确度、精密度

# 10.1 灵敏度

本方法检测限为 $1.5\mu \mathrm{g / kg}$ ，定量限为 $5\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $5\mu \mathrm{g / kg}\sim 500~\mu \mathrm{g / kg}$ 添加浓度的水平上其回收率为 $80\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leq 15\%$ ，批间相对标准偏差 $\leq 15\%$

# 食品接触材料及制品

# 三乙胺和三正丁胺的测定

2016-10-19 发布

2017-04-19 实施

# 前言

本标准代替SN/T2890—2011《出口食品接触材料 高分子材料 聚碳酸酯树脂及其制品中三乙胺及三正丁胺的测定 气相色谱法》。

本标准与SN/T2890—2011相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 三乙胺和三正丁胺的测定”；  
——增加了对“试验中应避免使用塑料材质的容器及转移器具”的要求；  
——将原标准中的“具塞量筒”改为“具塞刻度离心管”。

# 食品安全国家标准

# 食品接触材料及制品三乙胺和三正丁胺的测定

# 1 范围

本标准规定了食品接触材料及制品中三乙胺和三正丁胺含量的气相色谱测定方法。

本标准适用于聚碳酸酯食品接触材料及制品中三乙胺和三正丁胺含量的测定。

# 2 原理

样品经二氯甲烷溶解后，加入丙酮使聚合物树脂沉淀，溶液中的三乙胺和三正丁胺经浓缩定容后采用气相色谱法测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯。试验中应避免使用塑料材质的容器及转移器具。

# 3.1 试剂

3.1.1 二氯甲烷 $(\mathrm{CH}_2\mathrm{Cl}_2)$ 。

3.1.2 丙酮 $\mathrm{(C_3H_6O)}$ 。

# 3.2 标准品

3.2.1 三乙胺 $\mathrm{(C_6H_{15}N,CAS}$ 号：121-44-8)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

3.2.2 三正丁胺 $\mathrm{(C_{12}H_{27}N,CAS}$ 号：102-82-9）：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.3 标准溶液配制

# 3.3.1 标准储备液

在 $50~\mathrm{mL}$ 烧杯中准确称取 $10\mathrm{mg}$ 三乙胺和 $10\mathrm{mg}$ 三正丁胺(精确至 $0.1\mathrm{mg}$ )，加入约 $30~\mathrm{mL}$ 二氯甲烷，溶解后定量转移至 $100~\mathrm{mL}$ 容量瓶内，用二氯甲烷定容并摇匀，得到标准储备液。标准储备液应于 $4^{\circ}\mathrm{C}$ 下密封避光保存，有效期为1个月。

# 3.3.2 标准工作溶液

移取标准储备液 $4.00\mathrm{mL}$ ，用二氯甲烷稀释至 $100\mathrm{mL}$ ，定容并摇匀。再将稀释后的溶液分别移取 $1.00\mathrm{mL}, 2.00\mathrm{mL}, 3.00\mathrm{mL}, 4.00\mathrm{mL}, 5.00\mathrm{mL}$ 于5个不同的 $20\mathrm{mL}$ 容量瓶中，用二氯甲烷定容并摇匀，得到浓度分别为 $0.2\mathrm{mg/L}, 0.4\mathrm{mg/L}, 0.6\mathrm{mg/L}, 0.8\mathrm{mg/L}, 1.0\mathrm{mg/L}$ 的标准工作溶液。标准工作溶液应于临用前配制。

# 4 仪器和设备

4.1 气相色谱仪：配氮磷检测器。  
4.2 分析天平或电子天平：感量 $0.1\mathrm{mg}$   
4.3 离心机： $3000\mathrm{r / min}$   
4.4 旋转蒸发仪。  
4.5 具塞刻度离心管： $2\mathrm{mL}$ 或 $5\mathrm{mL}$   
4.6 针头式有机相过滤器： $0.22\mu \mathrm{m}$

# 5 分析步骤

# 5.1 试样制备

先将试样用切割工具将其破碎成粒径小于 $5\mathrm{mm}\times 5\mathrm{mm}$ 后再称量。切割样品时，不可使其发热变软。

# 5.2 试样溶液制备

称取 $1\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ ) 置于 $200\mathrm{mL}$ 锥形瓶中，加入 $20\mathrm{mL}$ 二氯甲烷。待样品完全溶解后，边摇动锥形瓶边用分液漏斗逐滴加入 $100\mathrm{mL}$ 丙酮。滴加完毕后将瓶内溶液摇匀，离心 $10\mathrm{min}$ ，取上层清液旋转蒸发浓缩至近 $1\mathrm{mL}$ 。将浓缩液完全转移至具塞刻度离心管中，再用二氯甲烷定容至 $2.0\mathrm{mL}$ ，得到试样溶液，供气相色谱上机测定。

注1：溶液离心可分次进行，需将离心后的上层清液合并。  
注2：若试样溶液中有少量浑浊，可用针头式有机相过滤器过滤后再上气相色谱仪测定。

# 5.3 空白试验

除不加试样外，采用与5.2完全相同的分析步骤、试剂和用量，进行平行操作。

# 5.4 气相色谱参考条件

5.4.1 色谱柱：ZB-5石英毛细管柱，柱长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ ，膜厚 $5\mu \mathrm{m}$ ，或相当者；  
5.4.2 柱温：初始温度 $40^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ ；然后以 $20^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升至 $250^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ ；  
5.4.3 进样口温度： $200^{\circ}C$   
5.4.4 检测器温度： $250^{\circ}C$   
5.4.5 载气：氮气（纯度 $\geq 99.999\%$ ），流量 $1.0~\mathrm{mL / min}$ ；  
5.4.6 检测器氢气流量： $3\mathrm{mL / min}$   
5.4.7 检测器空气流量： $60~\mathrm{mL / min}$   
5.4.8 进样量： $1\mu \mathrm{L}$   
5.4.9 进样方式：不分流进样。

# 5.5 标准工作曲线的制作

按照5.4所列色谱条件，对标准工作溶液进行检测。以标准工作溶液中三乙胺和三正丁胺的浓度 $(\mathrm{mg / L})$ 为横坐标，以对应的色谱峰面积为纵坐标，进行线性回归，绘制标准工作曲线。

# 5.6 样液测定

按照5.4所列测定条件，对空白溶液和试样溶液依次进样测定，得到空白溶液和试样溶液中三乙胺

和三正丁胺的色谱峰面积。根据标准曲线得到溶液中三乙胺和三正丁胺的浓度。

# 6 分析结果的表述

样品中三乙胺和三正丁胺含量按式(1)计算：

$$
X _ {i} = \frac {\left(\rho_ {i} - \rho_ {0}\right) \times V}{m} \tag {1}
$$

式中：

$X_{i}$ ——试样中三乙胺或三正丁胺的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho_{i}$ ——试样溶液中三乙胺或三正丁胺的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——空白溶液中三乙胺或三正丁胺的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样溶液的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测试结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

三乙胺和三正丁胺的方法检出限均为 $0.2\mathrm{mg / kg}$ ，定量限均为 $0.4\mathrm{mg / kg}$ 。

# 食品安全国家标准

# 食品添加剂 乙酸芳樟酯

# 1 范围

本标准适用于以芳樟醇或脱氢芳樟醇和乙酐为原料经化学反应制得的食品添加剂乙酸芳樟酯。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

3,7-二甲基-1,6-辛二烯-3-醇乙酸酯

# 2.2 分子式

$$
\mathrm {C} _ {1 2} \mathrm {H} _ {2 0} \mathrm {O} _ {2}
$$

# 2.3 结构式

![](images/0199d8fa3f77c51a33720fa0b8802d352b00020ed59d2aa607227fd72f1d88ef.jpg)

# 2.4 相对分子质量

196.29（按2007年国际相对原子质量）

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。

面积归一化法：按GB/T11538—2006中10.4测定含量。

按GB/T11538—2006中11.4规定进行。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ 。  
B.2.2 固定相：聚乙二醇。  
B.2.3 膜厚： $0.25\mu \mathrm{m}$   
B.2.4 色谱炉温度： $115~\mathrm{^\circ C}$ 恒温 $2\mathrm{min}$ ；然后线性程序升温从 $115~\mathrm{^\circ C}\sim 195~\mathrm{^\circ C}$ ，速率 $3~^\circ \mathrm{C / min}$ 。  
B.2.5 进样口温度： $210^{\circ}C$   
B.2.6 检测器温度： $260^{\circ}C$   
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 柱前压： $0.06\mathrm{MPa}$   
B.2.10 进样量：约 $0.2\mu \mathrm{L}$   
B.2.11 分流比： $100:1$

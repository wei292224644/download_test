# 食品接触材料及制品 丙烯腈的测定和迁移量的测定

2016-10-19 发布

2017-04-19 实施

# 前言

本标准代替GB/T5009.152—2003《食品包装用苯乙烯-丙烯腈共聚物和橡胶改性的丙烯腈-丁二烯-苯乙烯树脂及其成型品中残留丙烯腈单体的测定》和GB/T23296.8—2009《食品接触材料 高分子材料 食品模拟物中丙烯腈的测定 气相色谱法》和SN/T2197—2008《食品接触材料 高分子材料 食品和食品模拟物中丙烯腈的测定 气相色谱法》。

本标准与GB/T5009.152—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 丙烯腈的测定和迁移量的测定”；  
——删除了气相色谱-氢火焰检测器法，保留气相色谱-氮磷检测器法；  
——增加了食品接触材料及制品中丙烯腈迁移量的测定。

# 食品安全国家标准

# 食品接触材料及制品 丙烯腈的测定和迁移量的测定

# 1 范围

本标准规定了食品接触材料及其制品中丙烯腈的测定方法和迁移量的测定方法。  
本标准适用于食品接触材料及其制品中丙烯腈的测定和迁移量的测定。

# 丙烯腈的测定

# 2 原理

食品接触材料及制品经 $N,N-$ 二甲基甲酰胺溶解或分散于顶空瓶中，加热使待测成分达到气液平衡，然后定量吸取顶空气进行气相色谱测定。根据保留时间定性，丙腈作为内标物，以内标法定量。

# 3 试剂和材料

本方法所用水为GB/T6682规定的一级水。试验中容器及转移器具应避免使用塑料材质。

# 3.1 试剂

$N,N-$ 二甲基甲酰胺（DMF）：分析纯，在丙烯腈和丙腈保留时间处无干扰峰。

# 3.2 标准品

3.2.1丙烯腈 $\mathrm{(C_3H_3N,CAS}$ 号：107-13-1)： $1000\mu \mathrm{g / mL}$ ，或经国家认证并授予标准物质证书的标准物质。  
3.2.2 丙腈 $\mathrm{(C_2H_5CN}$ ，CAS号：107-12-0）： $1000\mu \mathrm{g / mL}$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.3 标准溶液配制

# 3.3.1 丙烯腈标准使用液

吸取丙烯腈标准品 $0.1\mathrm{mL}, 0.2\mathrm{mL}, 0.3\mathrm{mL}, 0.4\mathrm{mL}, 0.5\mathrm{mL}$ ，分别移入 $10\mathrm{mL}$ 容量瓶中，各加入 $N, N-$ 二甲基甲酰胺稀释至刻度，混匀（每毫升相当于丙烯腈 $10\mu \mathrm{g}, 20\mu \mathrm{g}, 30\mu \mathrm{g}, 40\mu \mathrm{g}, 50\mu \mathrm{g}$ ），使用当天配制。

# 3.3.2 丙腈标准溶液

吸取丙腈标准品 $0.5\mathrm{mL}$ 移入 $10\mathrm{mL}$ 容量瓶中，加入 $N,N$ -二甲基甲酰胺稀释至刻度，混匀，得到浓度为 $50~\mu \mathrm{g / mL}$ 的丙腈标准溶液。

# 4 仪器和设备

4.1 气相色谱仪（带氮磷检测器）。  
4.2 顶空自动进样器。  
4.3 $20~\mathrm{mL}$ 顶空气测定瓶，配有表层涂聚氟乙烯硅橡胶盖及铝帽。  
4.4 分析天平：感量为 $0.1\mathrm{mg}$   
4.5 $10~\mathrm{mL}$ 容量瓶。   
4.6 $250~\mu \mathrm{L}$ 微量注射器。

# 5 分析步骤

# 5.1 试样制备

取具代表性的食品包装材料 $10\mathrm{g}$ ，用粉碎机粉碎或用剪刀剪成粒径小于或等于 $0.2\mathrm{cm}$ 的碎片，混匀，装入洁净的容器内。

# 5.2 试液制备

称取 $0.5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样装入顶空瓶中，加入 $5.0\mathrm{mL}N,N-$ 二甲基甲酰胺，立即加盖密封，用 $250~\mu \mathrm{L}$ 微量注射器在顶空瓶中加入 $200~\mu \mathrm{L}$ 丙腈标准溶液，充分振摇，使瓶中试样完全溶解或充分分散，待测。同时做空白试验。

# 5.3 仪器参考条件

# 5.3.1 顶空进样器条件

顶空进样器条件列出如下：

a）平衡时间： $30\mathrm{min}$   
b）炉温： $80^{\circ}C$   
c）压力： $138\mathrm{kPa}$   
d）加压时间： $2\mathrm{min}$   
e）进样时间： $0.04\mathrm{min}$   
f）传输线温度： $85^{\circ}C$ □

# 5.3.2 气相色谱条件

气相色谱条件列出如下：

a) 色谱柱: 交联键合聚乙二醇固定相毛细管色谱柱, 柱长 $30 \mathrm{~m}$ , 内径 $0.32 \mathrm{~mm}$ , 膜厚 $0.25 \mu \mathrm{m}$ , 或等效柱;  
b）柱温度程序： $40^{\circ}\mathrm{C}$ 保持 $4\mathrm{min},10^{\circ}\mathrm{C / min}$ 升至 $70^{\circ}\mathrm{C}$ 保持 $1\mathrm{min},15^{\circ}\mathrm{C / min}$ 升至 $170^{\circ}\mathrm{C}$   
c）进样口温度： $150~\mathrm{^\circ C}$   
d）检测器温度： $280~\mathrm{^\circ C}$   
e) 载气氮气流速: $50 \mathrm{~mL} / \mathrm{min}$ ;   
f) 氢气流速: $1.5 \mathrm{~mL} / \mathrm{min}$ ;   
g）空气流速： $145~\mathrm{mL / min}$   
h）进样口恒压进样： $100\mathrm{kPa}$

# 5.4 校准溶液配制与校准曲线制作

取5只顶空瓶分别加入 $200\mu \mathrm{L}$ 丙腈标准溶液，再依次加入丙烯腈系列标准溶液，此时校准溶液中的丙烯腈含量分别相当于 $2.0\mu \mathrm{g},4.0\mu \mathrm{g},6.0\mu \mathrm{g},8.0\mu \mathrm{g},10.0\mu \mathrm{g}$ 。气相色谱调至最佳工作状态，将装有标准系列的顶空瓶置于顶空进样器中，取液上气进气相色谱，进行测定，得到相应丙腈和丙烯腈色谱峰。以丙烯腈含量为横坐标，丙烯腈峰面积与丙腈峰面积的比值为纵坐标绘制校准曲线。

丙烯腈气相色谱图见图A.1。

# 5.5 试样溶液的测定

气相色谱调至最佳工作状态，将装有待测试液的顶空瓶置于顶空进样器中，取液上气进气相色谱，进行测定，得到相应丙烯腈与丙腈的色谱峰，根据保留时间定性，根据试样中丙烯腈与内标物丙腈峰面积比值计算定量。

# 6 分析结果的表述

试样中丙烯腈的含量按式(1)计算：

$$
X = \frac {c \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中丙烯腈的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——由标准曲线求得试样溶液中丙烯腈的含量，单位为微克 $(\mu \mathrm{g})$ ；

1000——换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

方法检出限为 $0.1\mathrm{mg / kg}$ ，定量限为 $0.2\mathrm{mg / kg}$

# 丙烯腈迁移量的测定

# 9 原理

食品接触材料及制品采用食品模拟物浸泡，丙烯腈迁移到浸泡液中，取一定量模拟物浸泡液加入顶空瓶中，加热使待测成分达到气液平衡，然后定量吸取顶空气进行气相色谱测定。根据保留时间定性，丙腈作为内标物，以内标法定量。

# 10 试剂和材料

本方法所用水为GB/T6682规定的一级水。试验中容器及转移器具应避免使用塑料材质。

# 10.1 试剂

10.1.1 $N,N$ -二甲基甲酰胺(DMF)：分析纯，在丙烯腈和丙腈保留时间处无干扰峰。  
10.1.2 水基、酸性、酒精类、油基食品模拟物：所用试剂依据GB31604.1的规定。

# 10.2 试剂配制

水基、酸性、酒精类、油基食品模拟物：按GB5009.156操作。

# 10.3 标准品

10.3.1 丙烯腈 $\mathrm{(C_3H_3N, CAS}$ 号：107-13-1)： $1000\mu \mathrm{g / mL}$ ，或经国家认证并授予标准物质证书的标准物质。  
10.3.2丙腈 $\mathrm{(C_2H_5CN}$ ,CAS号：107-12-0）： $1000\mu \mathrm{g / mL}$ ，或经国家认证并授予标准物质证书的标准物质。

# 10.4 丙烯腈标准溶液配制

丙烯腈系列标准使用液：分别吸取丙烯腈标准液 $10\mu \mathrm{L},20\mu \mathrm{L},30\mu \mathrm{L},40\mu \mathrm{L},50\mu \mathrm{L}$ 移入 $10\mathrm{mL}$ 容量瓶中，各加 $N,N$ -二甲基甲酰胺稀释至刻度，混匀(每毫升相当于丙烯腈 $1\mu \mathrm{g},2\mu \mathrm{g},3\mu \mathrm{g},4\mu \mathrm{g},5\mu \mathrm{g})$ ，使用当天配制。

# 10.5 丙腈标准溶液配制

吸取丙腈标准品 $50~\mu \mathrm{L}$ 移入 $10~\mathrm{mL}$ 容量瓶中，加入 $N,N-$ 二甲基甲酰胺稀释至刻度，混匀，得到浓度为 $5\mu \mathrm{g / mL}$ 的丙腈标准溶液。

# 11 仪器和设备

11.1 气相色谱仪（带氮磷检测器）。  
11.2 顶空自动进样器。  
11.3 $20~\mathrm{mL}$ 顶空气测定瓶，配有表层涂聚氟乙烯硅橡胶盖及铝帽。  
11.4 涡旋振荡器。  
11.5 分析天平：感量 $0.1\mathrm{mg}$   
11.6 $10\mathrm{mL}$ 容量瓶。   
11.7 $100\mu \mathrm{L}$ 微量注射器。

# 12 分析步骤

# 12.1 试液制备

# 12.1.1 水基、酸性、酒精类、油基食品模拟物

本标准食品模拟试验采用水基、酸性、酒精类、油基食品模拟物，所需试液通过迁移试验获取，可在

$4^{\circ} \mathrm{C}$ 冰箱中避光保存一周。

# 12.1.2 迁移试验

按照GB5009.156及GB31604.1的要求，对样品进行迁移试验，得到食品模拟物试液。如果得到的食品模拟物试液不能马上进行下一步试验，应将食品模拟物试液于 $4^{\circ}\mathrm{C}$ 冰箱中保存。所得食品模拟物试液应冷却或恢复至室温后进行下一步试验。

# 12.1.3 浸泡液的处理

食品模拟物配制过程参照GB/T5009.156用 $5\mathrm{mL}$ 移液器移取 $5.0~\mathrm{mL}$ 食品模拟物装入顶空瓶中，准确称量移入液体质量，立即加盖密封，用 $100\mu \mathrm{L}$ 微量注射器在顶空瓶中加入 $100\mu \mathrm{LN},N$ 二甲基甲酰胺和 $100\mu \mathrm{L}$ 丙腈标准溶液，混合均匀，待测。同时做空白试验。

# 12.2 仪器参考条件

# 12.2.1 顶空进样器条件

顶空进样器条件列出如下：

a）平衡时间： $30\mathrm{min}$   
b）炉温： $80^{\circ}C$   
c）压力： $138\mathrm{kPa}$   
d）加压时间： $2\mathrm{min}$   
e）进样时间： $0.04\mathrm{min}$   
f）传输线温度： $85^{\circ}C$ 。

# 12.2.2 气相色谱条件

气相色谱条件列出如下：

a）色谱柱：交联键合聚乙二醇固定相毛细管色谱柱，柱长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ，或等效柱；  
b）柱温度程序： $40^{\circ}\mathrm{C}$ 保持 $4\mathrm{min},10^{\circ}\mathrm{C / min}$ 升至 $70~^\circ \mathrm{C}$ 保持 $1\mathrm{min},15^{\circ}\mathrm{C / min}$ 升至 $170^{\circ}C$

c）进样口温度： $150~\mathrm{^\circ C}$   
d）检测器温度： $280^{\circ}C$   
e) 载气氮气流速: $50 \mathrm{~mL} / \mathrm{min}$ ;   
f) 氢气流速: $1.5 \mathrm{~mL} / \mathrm{min}$ ;   
g）空气流速： $145~\mathrm{mL / min}$   
h）进样口恒压进样： $100\mathrm{kPa}$

# 12.3 标准曲线制作

5只顶空瓶分别按照12.1.3的试液制备过程统一加入不含丙烯腈的食品模拟物，只将12.1.3过程中加入的 $100\mu \mathrm{L}N,N$ -二甲基甲酰胺换成加入 $100\mu \mathrm{L}$ 丙烯腈系列标准溶液，此时校准溶液中的丙烯腈含量分别相当于 $0.1\mu \mathrm{g},0.2\mu \mathrm{g},0.3\mu \mathrm{g},0.4\mu \mathrm{g},0.5\mu \mathrm{g}$ 。同试样测定，顶空法注入色谱仪，计算丙烯腈与丙腈峰面积比值。以丙烯腈含量为横坐标，丙烯腈与丙腈峰面积比值为纵坐标绘制校准曲线，根据试样中丙烯腈与内标物丙腈峰面积比值计算定量。

丙烯腈气相色谱图参见图A.1。

# 12.4 试样溶液的测定

气相色谱调至最佳工作状态，将装有待测试液的顶空瓶置于顶空进样器中，取液上气进气相色谱，进行测定，得到相应丙烯腈与丙腈的色谱峰，根据保留时间定性，并测量其相应的峰面积。

# 13 分析结果的表述

由标准曲线得到试样溶液中丙烯腈的含量，按GB5009.156进行迁移量计算，得到食品接触材料及制品中丙烯腈的迁移量。计算结果保留两位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

方法检出限为 $0.01\mathrm{mg / kg}$ ，定量限为 $0.02\mathrm{mg / kg}$

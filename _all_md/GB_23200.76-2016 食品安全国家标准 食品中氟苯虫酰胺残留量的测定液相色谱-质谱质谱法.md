本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2581—2010《进出口食品中氟虫酰胺残留量的测定液相色谱一质谱/质谱法》。与SN/T2581—2010相比，主要变化如下：

# 食品安全国家标准

# 食品中氟苯虫酰胺残留量的测定 液相色谱一质谱/质谱法

# 1 范围

本标准规定了食品中氟苯虫酰胺残留量测定的液相色谱一质谱/质谱分析方法。

本标准适用于葱、萝卜、番茄、橙、大豆、苹果、茶、核桃、鱼、猪瘦肉、猪肝和牛奶中氟苯虫酰胺残留量的检测和确证，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

试样中的氟苯虫酰胺残留用乙腈提取，提取液经石墨碳一氨基固相萃取柱或弗罗里硅土固相萃取柱净化，液相色谱一质谱/质谱测定，外标法定量。

# 4 试剂和材料

所有试剂除特殊注明外，均为分析纯，水为GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ ：色谱纯。  
4.1.2 甲苯 $(\mathrm{C}_7\mathrm{H}_8)$ ：色谱纯。  
4.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
4.1.4 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ ：色谱纯。  
4.1.5 氯化钠(NaCl)。  
4.1.6 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，在干燥器内冷却至室温，储于密封瓶中备用。

# 4.2 溶液配制

4.2.1 乙腈饱和正己烷： $250\mathrm{mL}$ 的分液漏斗中加入 $150\mathrm{mL}$ 正己烷和 $50\mathrm{mL}$ 乙腈混匀振荡 $5\mathrm{min}$ ，取上层备用。  
4.2.2 乙腈一甲苯(3+1，体积比)：用量筒量取 $350~\mathrm{mL}$ 乙腈和 $150~\mathrm{mL}$ 甲苯混匀备用。  
4.2.3 丙酮一正己烷(1+1，体积比)：用量筒量取 $200~\mathrm{mL}$ 正己烷和 $200~\mathrm{mL}$ 丙酮混匀备用。  
4.2.4 乙腈一水(7+3，体积比)：用量筒量取 $350~\mathrm{mL}$ 乙腈和 $150~\mathrm{mL}$ 水混匀备用。

# 4.3 标准品

4.3.1 氟苯虫酰胺（flubendiamide）标准品：纯度 $\geqslant 99\%$ ，CAS号：272451-65-7，分子式： $\mathrm{C_{23}H_{22}F_7IN_2O_4S}$ ，相对分子质量为682.39。

# 4.4 标准溶液配制

4.4.1 氟苯虫酰胺标准储备溶液 $(100\mu \mathrm{g} / \mathrm{mL})$ : 准确称取适量的氟苯虫酰胺标准物质, 用乙腈配成浓度为 $100\mu \mathrm{g} / \mathrm{mL}$ 的标准储备液, 该标准储备液置于 $4^{\circ} \mathrm{C}$ 中避光密封保存。  
4.4.2 氟苯虫酰胺基质标准工作液：使用前根据需要将标准储备溶液用各种样本的空白基质稀释成适当浓度的标准工作液。基质标准工作液应现用现配。

# 4.5 材料

4.5.1 石墨炭一氨基固相萃取柱: $500 \mathrm{mg} / 500 \mathrm{mg}, 5 \mathrm{~mL}$ 或相当者。加样前先用 $5 \mathrm{~mL}$ 乙腈一甲苯预淋洗柱。  
4.5.2 弗罗里硅土固相萃取柱： $5\mathrm{mL} / 1000\mathrm{mg}$ ，或相当者。加样前依次用 $2\mathrm{mL}$ 丙酮和 $4\mathrm{mL}$ 正己烷预淋洗柱。  
4.5.3 微孔滤膜： $0.2\mu \mathrm{m}$ ，通用型。

# 5 仪器和设备

5.1 液相色谱—质谱/质谱仪：配电喷雾离子源(ESI)。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 组织捣碎机。  
5.4 均质器： $10000\mathrm{r / min}$   
5.5 离心机： $4000\mathrm{r / min}$   
5.6 聚四氟乙烯塑料离心管： $50~\mathrm{mL}$ ，具塞。  
5.7 旋转蒸发仪。  
5.8 氮吹仪。

# 6 试样制备和保存

# 6.1 取样部位

样品取样部位按 GB 2763 的规定执行。在取样和制样过程中,应防止样品受到污染或发生氟苯虫酰胺残留量的变化。

# 6.2 试样制备

# 6.2.1 茶叶、大豆、核桃

取代表性样品约 $500 \mathrm{~g}$ , 经组织捣碎机粉碎并通过 $2.0 \mathrm{~mm}$ 圆孔筛, 混匀, 装入洁净容器内密封, 标明标记。

# 6.2.2 葱、萝卜、番茄、橙、苹果、鱼、猪瘦肉、猪肝

取代表性样品约 $500\mathrm{g}$ ，切碎，经组织捣碎机充分捣碎均匀，装入洁净容器内密封，标明标记。

# 6.2.3 牛奶

取代表性样品约 $500\mathrm{g}$ ，装入洁净容器内密封，标明标记。

# 6.3 试样保存

茶叶、粮谷、坚果类等试样于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存；蔬菜、水果、畜、禽类等试样于 $-18^{\circ}\mathrm{C}$ 冷冻保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 葱、萝卜、番茄、橙、大豆、苹果、茶、核桃

大豆、核桃和茶叶：称取 $2\mathrm{g}$ （精确至 $0.01\mathrm{g}$ 试样于 $50~\mathrm{mL}$ 离心管中，加 $5\mathrm{mL}$ 水放置 $0.5\mathrm{h}$ 葱、萝卜、番茄、橙、苹果：称取 $2\mathrm{g}$ （精确至 $0.01\mathrm{g}$ 试样于 $50~\mathrm{mL}$ 离心管中。加入 $20~\mathrm{mL}$ 乙腈，以 $10000\mathrm{r / min}$

均质 $1\mathrm{min}$ ，加入 $2\mathrm{g}$ 氯化钠，摇匀，并于 $4000\mathrm{r / min}$ 离心 $3\mathrm{min}$ 。吸取上层有机相于浓缩瓶中，残渣中加入 $15~\mathrm{mL}$ 乙腈，重复提取一次，合并上层有机相，在 $40^{\circ}C$ 水浴中减压浓缩至近干，待净化。

# 7.1.2 鱼、猪瘦肉、猪肝、牛奶

称取粉碎后试样 $2\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 于 $50\mathrm{mL}$ 离心管中, 加入 $2\mathrm{g}$ 无水硫酸钠, $20\mathrm{mL}$ 乙腈和 $10\mathrm{mL}$ 乙腈饱和正己烷, 以 $10000\mathrm{r / min}$ 均质 $1\mathrm{min}$ , 以 $4000\mathrm{r / min}$ 离心 $3\mathrm{min}$ , 吸取下层有机相于浓缩瓶中, 残渣中加入 $15\mathrm{mL}$ 乙腈, 重复提取一次, 合并下层有机相, 在 $40^{\circ}\mathrm{C}$ 水浴中减压浓缩至近干, 待净化。

# 7.2 净化

# 7.2.1 葱、萝卜、番茄、橙、大豆、苹果、茶叶、核桃

分3次，每次用 $2\mathrm{mL}$ 乙腈一甲苯洗涤浓缩瓶中残留物并全部转入石墨碳一氨基固相萃取柱中，然后用 $15\mathrm{mL}$ 乙腈一甲苯淋洗柱子，控制流速不超过 $2.0\mathrm{mL} / \mathrm{min}$ ，收集全部流出液，于 $40^{\circ}\mathrm{C}$ 水浴中减压浓缩，氮吹至近干，用乙腈一水溶解残渣并定容至 $2.0\mathrm{mL}$ ，过滤膜，供分析。

# 7.2.2 鱼、猪瘦肉、猪肝、牛奶

分3次，每次用 $2\mathrm{mL}$ 正己烷洗涤浓缩瓶中残留物并全部转入弗罗里硅土固相萃取柱中，待溶液全部流出固相萃取柱后加入 $2\mathrm{mL}$ 正己烷淋洗，弃去全部流出液。再以 $10\mathrm{mL}$ 丙酮一正己烷洗脱，控制流速不超过 $2.0\mathrm{mL} / \mathrm{min}$ ，收集全部洗脱液，于 $40^{\circ}\mathrm{C}$ 水浴中减压浓缩，氮吹至近干，用乙腈一水溶解残渣并定容至 $2.0\mathrm{mL}$ ，过滤膜，供分析。

# 7.3.2 色谱测定与确证

根据样液中氟苯虫酰胺含量情况，选定峰面积相近的标准工作溶液。标准工作溶液和样液中氟苯虫酰胺响应值均应在仪器检测线性范围内。如果残留量超出标准曲线范围，应用乙腈将样品提取液(7.1)进行适当稀释。标准工作溶液和样液等体积参插进样测定。在上述色谱条件下，氟苯虫酰胺的保留时间约为 $1.75\mathrm{min}$ 。

# 7.4 空白实验

除不称取样品外，均按 $7.1\sim 7.3$ 的测定条件和步骤进行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中氟苯虫酰胺的残留含量，计算结果需扣除空白值。

$$
X = \frac {A \times c \times V}{A _ {S} \times m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中氟苯虫酰胺的残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

A ——样液中氟苯虫酰胺的峰面积或峰高；

$c$ ——标准工作液中氟苯虫酰胺的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

V ——样液最终定容体积,单位为毫升 $(\mathrm{mL})$ ;

$A_{S}$ ——标准工作液中氟苯虫酰胺的峰面积或峰高；

m 最终样液所代表的试样质量，单位为克(g)；

1000 ——换算系数。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 C 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 D 的要求。

# 食品安全国家标准

# 食品中钼的测定

# 1 范围

本标准规定了食品中钼的石墨炉原子吸收光谱和电感耦合等离子体质谱测定方法。  
本标准适用于食品中钼的测定。

# 第一法 石墨炉原子吸收光谱法

# 2 原理

试样经消解处理后，经石墨炉原子化，在 $313.3\mathrm{nm}$ 处测定吸光度值，在一定浓度范围内钼的吸光度值与钼含量成正比，采用外标法进行定量分析。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 硝酸 $\mathrm{(HNO_3)}$ 。  
3.1.2 高氯酸 $\mathrm{(HClO_4)}$   
3.1.3 氯化钙 $\mathrm{(CaCl_2)}$ 。

# 3.2 试剂配制

3.2.1 硝酸溶液(5：95，体积分数)：量取 $50~\mathrm{mL}$ 硝酸缓慢加入 $950~\mathrm{mL}$ 水中，混匀。  
3.2.2 氯化钙溶液 $(1.00\mathrm{g / L})$ ：称取 $0.100\mathrm{g}$ 氯化钙，用少量水溶解后，用硝酸溶液(5：95，体积分数)定容至 $100~\mathrm{mL}$ ，混匀。  
3.2.3 硝酸溶液(1：5，体积分数)：量取 $100~\mathrm{mL}$ 硝酸缓慢加入 $500~\mathrm{mL}$ 水中，混匀。

# 3.3 标准品

钼酸铵 $\mathrm{(H_8MoN_2O_4}$ ,CAS号：13106-76-8)：纯度 $\geq 99.98\%$ 。或经国家认证并授予标准物质证书的标准溶液。

# 3.4 标准溶液配制

3.4.1 钼标准储备液 $(1000\mathrm{mg / L})$ : 准确称取 $2.0430\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ) 钼酸铵, 用 $50\mathrm{mL}$ 水溶解, 移入 $1000\mathrm{mL}$ 容量瓶, 加硝酸溶液 $(5:95$ , 体积分数) 定容至刻度, 混匀。此溶液可冷藏保存 12 个月。  
3.4.2钼标准中间液 $(10.0\mathrm{mg / L})$ ：准确吸取钼标准储备液 $(1000\mathrm{mg / L})1.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶

中，加硝酸溶液(5：95，体积分数)至刻度，混匀。此溶液可冷藏保存1个月。

3.4.3 钼标准使用液 $(1.00\mathrm{mg / L})$ ：准确吸取钼标准中间液 $(10.0\mathrm{mg / L})10.0\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加硝酸溶液(5：95，体积分数)至刻度，混匀。临用现配。

3.4.4 钼标准系列工作溶液：分别吸取钼标准使用液 $(1.00\mathrm{mg / L})0\mathrm{mL},0.10\mathrm{mL},0.25\mathrm{mL},0.50\mathrm{mL}$ 、 $1.00~\mathrm{mL}$ 和 $2.50~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加硝酸溶液(5:95，体积分数)定容至刻度，混匀。此钼标准系列溶液的质量浓度分别为 $0\mu \mathrm{g} / \mathrm{L},1.00\mu \mathrm{g} / \mathrm{L},2.50\mu \mathrm{g} / \mathrm{L},5.00\mu \mathrm{g} / \mathrm{L},10.0\mu \mathrm{g} / \mathrm{L}$ 和 $25.0\mu \mathrm{g} / \mathrm{L}$ 。临用现配。

注：可根据仪器的灵敏度及样品中钼的实际含量确定标准系列工作溶液中钼的质量浓度。

# 4 仪器和设备

所有玻璃器皿及聚四氟乙烯消解内罐均需硝酸溶液(1：5，体积分数)浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

4.1 原子吸收光谱仪：配石墨炉原子化器，附钼空心阴极灯。  
4.2 天平：感量分别为 $1\mathrm{mg}$ 和 $0.1\mathrm{mg}$   
4.3 可调式电热炉或可调式电热板。  
4.4 微波消解系统：配聚四氟乙烯消化内罐。  
4.5 压力消解罐：配聚四氟乙烯消化内罐。  
4.6 恒温干燥箱。  
4.7 样品粉碎设备：匀浆机、高速粉碎机。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固态样品

# 5.1.1.1 干样

谷物、豆类、坚果、干制品、焙烤食品等低含水量样品，取可食部分，必要时经高速粉碎机粉碎，使其混合均匀，储于塑料容器中；乳粉、面粉、蛋白粉等呈均匀状的粉状样品，摇匀。

# 5.1.1.2 鲜样

蔬菜、水果、水产品等高含水量样品，必要时洗净，晾干，取可食部分匀浆均匀，储于塑料容器中；肉类、蛋类等样品取可食部分匀浆均匀，储于塑料容器中。

# 5.1.2 液态样品

饮料、液态乳、食用油等样品摇匀。

# 5.1.3 半固态样品

搅拌均匀。

# 5.2 样品前处理

# 5.2.1 微波消解法

称取试样 $0.3\mathrm{g}\sim 0.8\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 或准确移取液体试样 $0.500\mathrm{mL}\sim 3.00\mathrm{mL}$ 于微波消解罐

中, 加入 $5 \mathrm{~mL} \sim 10 \mathrm{~mL}$ 硝酸, 按照微波消解仪的操作步骤消解试样, 仪器参考条件见附录 A 中表 A.1, 冷却后取出, 在电热板上于 $150^{\circ} \mathrm{C} \sim 170^{\circ} \mathrm{C}$ 赶酸至 $1 \mathrm{~mL}$ 左右。取下消解罐, 冷却后将消化液转移至 $10 \mathrm{~mL}$ 或 $25 \mathrm{~mL}$ 容量瓶中, 用少量水洗涤消解罐 2 次 $\sim 3$ 次, 合并洗涤液于容量瓶中并用水定容至刻度, 混匀备用。同时做空白试验。

# 5.2.2 湿式消解法

称取试样 $0.5\mathrm{g}\sim 3.0\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 或准确移取液体试样 $0.500\mathrm{mL}\sim 5.00\mathrm{mL}$ 于消化管中，加入 $10\mathrm{mL}$ 硝酸和 $0.5\mathrm{mL}$ 高氯酸，在可调式电热炉上消解（参考条件：升至 $120^{\circ}\mathrm{C}$ 保持 $0.5\mathrm{h}\sim 1\mathrm{h}$ ，升至 $180^{\circ}\mathrm{C}$ 保持 $2\mathrm{h}\sim 4\mathrm{h}$ ，升至 $200^{\circ}\mathrm{C}\sim 220^{\circ}\mathrm{C}$ 直至消解完全）。若消化液呈棕褐色，再加少量硝酸，消解至冒白烟，如消化液仍呈棕褐色，反复加入少量硝酸，使消化液呈无色透明或略带黄色，赶酸至 $1\mathrm{mL}$ 左右，取出消化管，冷却后将消化液转移至 $10\mathrm{mL}$ 或 $25\mathrm{mL}$ 容量瓶中，用少量水洗涤消化管2次～3次，合并洗涤液于容量瓶中并用水定容至刻度，混匀备用。同时做空白试验。亦可采用锥形瓶，于可调式电热板上，按上述操作方法进行湿式消解。

# 5.2.3 压力罐消解法

称取试样 $0.2\mathrm{g}\sim 1.0\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.500~\mathrm{mL}\sim 5.00~\mathrm{mL}$ 于消解罐内，加入 $5\mathrm{mL}\sim 10\mathrm{mL}$ 硝酸，盖好内盖，旋紧不锈钢外套，放入恒温干燥箱中，于 $150^{\circ}C\sim 170^{\circ}C$ 下保持 $4\mathrm{h}\sim$ $5\mathrm{h}$ 。冷却后缓慢旋松外罐，取出消解内罐，在电热板上于 $150^{\circ}C\sim 170^{\circ}C$ 赶酸至 $1\mathrm{mL}$ 左右。取下消解罐，冷却后将消化液转移至 $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 容量瓶中，用少量水洗涤消解罐2次～3次，合并洗涤液于容量瓶中并用水定容至刻度，混匀备用。同时做空白试验。

# 5.3 仪器参考条件

根据各自仪器性能调至最佳状态。仪器操作条件参见附录A中表A.2。

# 5.4 标准曲线的制作

按质量浓度由低到高的顺序分别将钼标准系列工作溶液注入石墨炉，原子化后测其吸光度值，以质量浓度为横坐标，吸光度值为纵坐标，制作标准曲线。

# 5.5 试样溶液的测定

在与测定标准溶液相同的试验条件下，将空白溶液或试样溶液（可根据试样溶液浓度进行适当稀释后上机)注入石墨炉，原子化后测定其吸光度值，与标准系列比较定量。

对于有干扰的试样应在标准曲线的制作和试样溶液的测定过程中同时注入基体改进剂。

# 6 分析结果的表述

试样中钼的含量按公式(1)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times f}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中钼的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$\rho$ ——试样溶液中钼的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白溶液中钼的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样消化液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样的稀释倍数；

$m$ ——试样的取样量或移取体积，单位为克或毫升(g或 $\mathrm{mL}$ )；

1000——换算系数。

当钼含量小于 $1.00\mathrm{mg / kg}$ （或 $\mathrm{mg / L}$ )时，计算结果保留2位有效数字；当钼含量大于或等于 $1.00\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )时，计算结果保留3位有效数字。

# 7 精密度

样品中钼元素含量大于 $1\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )时，在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ ；小于或等于 $1\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )且大于 $0.1\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )时，在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ ；小于或等于 $0.1\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )时，在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当称样量为 $0.5\mathrm{g}$ (或 $2.5\mathrm{mL}$ )、定容体积为 $25\mathrm{mL}$ 时，方法的检出限为 $0.02\mathrm{mg/kg}$ (或 $0.003\mathrm{mg/L}$ )，定量限为 $0.05\mathrm{mg/kg}$ (或 $0.01\mathrm{mg/L}$ )。

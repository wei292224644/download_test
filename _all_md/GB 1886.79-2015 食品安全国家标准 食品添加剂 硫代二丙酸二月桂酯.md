# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试剂中所用标准溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

溶解性：溶于多数有机溶剂，不溶于水。

# A.3 硫代二丙酸二月桂酯 $(\mathbf{C}_{30}\mathbf{H}_{58}\mathbf{O}_4\mathbf{S})$ 含量的测定

准确称取 $700\mathrm{mg}$ 试样，置于 $250~\mathrm{mL}$ 锥形烧瓶中，加入 $100~\mathrm{mL}$ 乙酸和 $50~\mathrm{mL}$ 乙醇。在约 $40^{\circ}\mathrm{C}$ 条件下加热混合液，直至试样溶解。然后加入 $3\mathrm{mL}$ 盐酸和4滴对乙氧基柯衣定试液（将 $50~\mathrm{mg}$ 对乙氧基柯衣定溶于 $25~\mathrm{mL}$ 水和 $25~\mathrm{mL}$ 乙醇的混合液中，加3滴盐酸，搅拌，有必要时可过滤，得澄清液体），立即用 $0.05\mathrm{mol} / \mathrm{L}$ 溴溶液滴定。待达到终点(粉红色)时，再加入4滴上述指示液后继续滴定至从红色变为浅黄色。同时进行空白试验并作必要校正。每毫升 $0.05\mathrm{mol} / \mathrm{L}$ 溴溶液相当于 $25.74\mathrm{mg}$ $\mathrm{C_{30}H_{58}O_4S}$ 。再减去在“酸度测定”(A.4)中测得的硫代二丙酸含量与2.89相乘的值，之间的差值即为 $\mathrm{C_{30}H_{58}O_4S}$ 的含量。

# A.4 酸度（以硫代二丙酸计）的测定

准确称取 $2\mathrm{g}$ 试样，置于 $250~\mathrm{mL}$ 锥形烧瓶中，加入 $50~\mathrm{mL}$ 由1份甲醇和3份苯配成的混合液，溶解混匀后，加入5滴酚酞试液，再用 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钾乙醇溶液滴定。每毫升 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钾乙醇溶液相当于 $8.91\mathrm{mg}3,3^{\prime}$ -硫代二丙酸。

# 食品安全国家标准

# 食品中氨基甲酸乙酯的测定

# 1 范围

本标准规定了啤酒、葡萄酒、黄酒、白酒等酒类以及酱油中氨基甲酸乙酯含量的气相色谱-质谱法测定。

本标准适用于啤酒、葡萄酒、黄酒、白酒等酒类以及酱油中氨基甲酸乙酯含量的测定。

# 2 原理

试样加 $\mathrm{D}_5$ -氨基甲酸乙酯内标后，经过碱性硅藻土固相萃取柱净化、洗脱，洗脱液浓缩后，用气相色谱-质谱仪进行测定，内标法定量。

# 3 试剂和材料

注：除非另有说明，本标准所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.2 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
3.1.4 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。  
3.1.5 乙醚 $\mathrm{(C_4H_{10}O)}$ ：色谱纯。  
3.1.6 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.7 碱性硅藻土固相萃取柱：填料 $4000\mathrm{mg}$ 、柱容量 $12\mathrm{mL}$ 。

# 3.2 试剂配制

3.2.1 无水硫酸钠： $450^{\circ}\mathrm{C}$ 烘烤 $4\mathrm{h}$ ，冷却后贮存于干燥器中备用。  
3.2.2 $5\%$ 乙酸乙酯-乙醚溶液：取 $5\mathrm{mL}$ 乙酸乙酯，用乙醚稀释到 $100~\mathrm{mL}$ ，混匀待用。

# 3.3 标准品

3.3.1 氨基甲酸乙酯标准品 $\mathrm{(C_3H_7O_2N}$ ，CAS：51-79-6）：纯度大于 $99.0\%$   
3.3.2 $\mathrm{D}_5$ -氨基甲酸乙酯标准品（ $\mathrm{C_3H_2D_5NO_2}$ ，CAS：73962-07-9）：纯度大于 $98.0\%$ 。

# 3.4 标准溶液配制

3.4.1 $\mathrm{D}_5$ -氨基甲酸乙酯储备液 $(1.00\mathrm{mg / mL})$ ：准确称取 $0.01\mathrm{g}$ （精确到 $0.0001\mathrm{g})\mathrm{D}_5$ -氨基甲酸乙酯标准品，用甲醇溶解、定容至 $10~\mathrm{mL},4^{\circ}C$ 以下保存。

3.4.2 $\mathrm{D}_{5}$ -氨基甲酸乙酯使用液 $(2.00\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取 $\mathrm{D}_{5}$ -氨基甲酸乙酯储备液 $(1.00\mathrm{mg / mL})0.10\mathrm{mL}$ 用甲醇定容至 $50~\mathrm{mL},4^{\circ}\mathrm{C}$ 以下保存。  
3.4.3 氨基甲酸乙酯储备液 $(1.00\mathrm{mg / mL})$ ：准确称取 $0.05\mathrm{g}$ （精确到 $0.0001\mathrm{g}$ )氨基甲酸乙酯标准品，用甲醇溶解、定容至 $50~\mathrm{mL},4^{\circ}\mathrm{C}$ 以下保存，保存期3个月。  
3.4.4 氨基甲酸乙酯中间液 $(10.0\mu \mathrm{g / mL})$ ：准确吸取氨基甲酸乙酯储备液 $(1.00\mathrm{mg / mL})1.00\mathrm{mL}$ ，用甲醇定容至 $100~\mathrm{mL},4^{\circ}C$ 以下保存，保存期1个月。  
3.4.5 氨基甲酸乙酯中间液 $(0.50\mu \mathrm{g / mL})$ ：准确吸取氨基甲酸乙酯中间液 $(10.0\mu \mathrm{g / mL})5.00\mathrm{mL}$ ，用甲醇定容至 $100~\mathrm{mL}$ ，现配现用。  
3.4.6 标准曲线工作溶液：分别准确吸取氨基甲酸乙酯中间液 $(0.50\mu \mathrm{g / mL})20.0\mu \mathrm{L}, 50.0\mu \mathrm{L}, 100.0\mu \mathrm{L}$ 、 $200.0\mu \mathrm{L}, 400.0\mu \mathrm{L}$ 和氨基甲酸乙酯标准中间液 $(10.0\mu \mathrm{g / mL})40.0\mu \mathrm{L}, 100.0\mu \mathrm{L}$ 于7个 $1\mathrm{mL}$ 容量瓶中，各加 $2.00\mu \mathrm{g / mL}$ $\mathrm{D}_5$ -氨基甲酸乙酯使用液 $100~\mu \mathrm{L}$ ，用甲醇定容至刻度，得到 $10.0\mathrm{ng / mL}$ 、 $25.0\mathrm{ng / mL}, 50.0\mathrm{ng / mL}, 100\mathrm{ng / mL}, 200\mathrm{ng / mL}, 400\mathrm{ng / mL}, 1000\mathrm{ng / mL}$ 的标准曲线工作溶液，现配现用。

# 4 仪器和设备

4.1 气相色谱-质谱仪，带电子轰击源(EI)源。  
4.2 涡旋混匀器。  
4.3 氮吹仪。  
4.4 固相萃取装置，配真空泵。  
4.5 超声波清洗机。  
4.6 马弗炉。  
4.7 天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$

# 5 分析步骤

# 5.1 试样制备

样品摇匀，称取 $2\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )样品（啤酒超声 $5\mathrm{min}$ 脱气后称量)，加 $100.0\mu \mathrm{L}2.00\mu \mathrm{g / mL}$ $\mathrm{D}_5$ -氨基甲酸乙酯使用液、氯化钠 $0.3\mathrm{g}$ (若为酱油不加氯化钠)，超声溶解、混匀后，加样到碱性硅藻土固相萃取柱上，在真空条件下，将样品溶液缓慢渗入萃取柱中，并静置 $10\mathrm{min}$ 。经 $10~\mathrm{mL}$ 正己烷淋洗后，用 $10~\mathrm{mL}5\%$ 乙酸乙酯-乙醚溶液以约 $1\mathrm{mL / min}$ 流速进行洗脱，洗脱液经装有 $2\mathrm{g}$ 无水硫酸钠的玻璃漏斗脱水后，收集于 $10~\mathrm{mL}$ 刻度试管中，室温下用氮气缓缓吹至约 $0.5\mathrm{mL}$ ，用甲醇定容至 $1.00~\mathrm{mL}$ 制成测定液，供GC/MS分析。

# 5.2 仪器参考条件

气相色谱-质谱仪分析参考条件：

毛细管色谱柱：DB-INNOWAX， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径） $\times 0.25\mu \mathrm{m}$ （膜厚)或相当色谱柱；  
进样口温度： $220^{\circ}C$   
柱温：初温 $50^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，然后以 $8^{\circ}\mathrm{C / min}$ 升至 $180^{\circ}\mathrm{C}$ ，程序运行完成后， $240^{\circ}\mathrm{C}$ 后运行 $5\mathrm{min}$ ；  
载气：氦气，纯度 $\geqslant 99.999\%$ ，流速 $1\mathrm{mL / min}$   
——电离模式：电子轰击源（EI），能量为 $70\mathrm{eV}$ ；  
四级杆温度： $150~\mathrm{^\circ C}$   
离子源温度： $230^{\circ}C$

——传输线温度： $250^{\circ}\mathrm{C}$   
——溶剂延迟：11min；  
——进样方式：不分流进样；  
——进样量： $1\mu \mathrm{L}\sim 2\mu \mathrm{L}$   
——检测方式：选择离子监测（SIM）；  
——氨基甲酸乙酯选择监测离子 $(m / z)$ : 44、62、74、89, 定量离子 62;  
—— $\mathrm{D}_5$ -氨基甲酸乙酯选择监测离子 $(m / z)64,76$ ，定量离子64。

# 5.3 定性测定

按方法条件测定标准工作溶液和试样，低浓度试样定性可以减少定容体积，试样的质量色谱峰保留时间与标准物质保留时间的允许偏差小于 $\pm 2.5\%$ ；定性离子对的相对丰度与浓度相当标准工作溶液的相对丰度允许偏差不超过表1的规定。

# 5.4 定量测定

# 5.4.1 标准曲线的制作

将氨基甲酸乙酯标准曲线工作溶液 $10.0\mathrm{ng / mL}$ 、25.0ng/mL、50.0ng/mL、100ng/mL、 $200\mathrm{ng / mL},400\mathrm{ng / mL},1000\mathrm{ng / mL}$ (内含 $200\mathrm{ng / mL}$ $\mathrm{D}_5$ -氨基甲酸乙酯)进行气相色谱-质谱仪测定，以氨基甲酸乙酯浓度为横坐标，标准曲线工作溶液中氨基甲酸乙酯峰面积与内标 $\mathrm{D}_5$ -氨基甲酸乙酯的峰面积比为纵坐标，绘制标准曲线。

# 5.4.2 试样测定

将试样溶液同标准曲线工作溶液进行测定，根据测定液中氨基甲酸乙酯的含量计算试样中氨基甲酸乙酯的含量，其中试样含低浓度的氨基甲酸乙酯时，宜采用 $10.0\mathrm{ng / mL},25.0\mathrm{ng / mL},50.0\mathrm{ng / mL},$ $100\mathrm{ng / mL},200\mathrm{ng / mL}$ 的标准曲线工作溶液绘制标准曲线；试样含高浓度氨基甲酸乙酯时，宜采用 $50.0~\mathrm{ng / mL},100~\mathrm{ng / mL},200~\mathrm{ng / mL},400~\mathrm{ng / mL},1000~\mathrm{ng / mL}$ 的标准曲线工作溶液绘制标准曲线。

# 6 分析结果的表述

试样中氨基甲酸乙酯含量按式(1)计算：

$$
X = \frac {c \times V \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——样品中氨基甲酸乙酯含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c$ ——测定液中氨基甲酸乙酯的含量，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——样品测定液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，保留3位有效数。

# 7 精密度

在重复性条件下获得的两次独立测定结果的相对偏差，当含量 $\leqslant 50~\mu \mathrm{g / kg}$ 时，不得超过算术平均值的 $15\%$ ；当含量 $>50~\mu \mathrm{g / kg}$ 时，不得超过算术平均值的 $10\%$ 。

# 8 其他

当试样取 $2\mathrm{g}$ 时，本方法氨基甲酸乙酯检出限为 $2.0~\mu \mathrm{g / kg}$ ，定量限为 $5.0~\mu \mathrm{g / kg}$ 。

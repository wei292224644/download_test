# 前言

本标准代替GB5009.17—2014《食品安全国家标准 食品中总汞及有机汞的测定》。

本标准与GB5009.17—2014相比，主要变化如下：

第一篇 食品中总汞的测定

——修改第一法的名称为原子荧光光谱法，修改了试样消解和附录的相关内容；  
——增加直接进样测汞法作为第二法，增加电感耦合等离子体质谱法作为第三法；  
——修改冷原子吸收光谱法作为第四法。

第二篇 食品中甲基汞的测定

——修改液相色谱-原子荧光光谱联用法作为第一法，修改了方法的适用范围；  
——增加液相色谱-电感耦合等离子体质谱联用法作为第二法。

# 第一篇 食品中总汞的测定

# 第一法 原子荧光光谱法

# 2 原理

试样经酸加热消解后，在酸性介质中，试样中汞被硼氢化钾或硼氢化钠还原成原子态汞，由载气（氩气)带入原子化器中，在汞空心阴极灯照射下，基态汞原子被激发至高能态，在由高能态回到基态时，发射出特征波长的荧光，其荧光强度与汞含量成正比，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 硝酸 $\mathrm{(HNO_3)}$ 。  
3.1.2 过氧化氢 $\mathrm{(H_2O_2)}$ 。  
3.1.3 硫酸 $\mathrm{(H_2SO_4)}$ 。  
3.1.4 氢氧化钾(KOH)。  
3.1.5 硼氢化钾 $\mathrm{(KBH_4)}$ ：分析纯。  
3.1.6 重铬酸钾 $\mathrm{(K_2Cr_2O_7)}$

# 3.2 试剂配制

3.2.1 硝酸溶液 $(1 + 9)$ ：量取 $50~\mathrm{mL}$ 硝酸，缓缓加入 $450~\mathrm{mL}$ 水中，混匀。  
3.2.2 硝酸溶液 $(5 + 95)$ ：量取 $50~\mathrm{mL}$ 硝酸，缓缓加入 $950~\mathrm{mL}$ 水中，混匀。  
3.2.3 氢氧化钾溶液 $(5\mathrm{g} / \mathrm{L})$ ：称取 $5.0\mathrm{g}$ 氢氧化钾，用水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.4 硼氢化钾溶液 $(5\mathrm{g} / \mathrm{L})$ ：称取 $5.0\mathrm{g}$ 硼氢化钾，用氢氧化钾溶液 $(5\mathrm{g} / \mathrm{L})$ 溶解并稀释至 $1000\mathrm{mL}$ ，混匀。临用现配。

3.2.5 重铬酸钾的硝酸溶液 $(0.5\mathrm{g} / \mathrm{L})$ ：称取 $0.5\mathrm{g}$ 重铬酸钾，用硝酸溶液 $(5 + 95)$ 溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。

注：本方法也可用硼氢化钠作为还原剂：称取 $3.5\mathrm{g}$ 硼氢化钠，用氢氧化钠溶液 $(3.5\mathrm{g} / \mathrm{L})$ 溶解并定容至 $1000~\mathrm{mL}$ ，混匀。临用现配。

# 3.3 标准品

氯化汞 $\mathrm{(HgCl_2}$ ，CAS号：7487-94-7）：纯度 $\geq 99\%$

# 3.4 标准溶液配制

3.4.1 汞标准储备液 $(1000\mathrm{mg / L})$ ：准确称取 $0.1354\mathrm{g}$ 氯化汞，用重铬酸钾的硝酸溶液 $(0.5\mathrm{g / L})$ 溶解并转移至 $100~\mathrm{mL}$ 容量瓶中，稀释并定容至刻度，混匀。于 $2^{\circ}C\sim 8^{\circ}C$ 冰箱中避光保存，有效期2年。或经国家认证并授予标准物质证书的汞标准溶液。  
3.4.2 汞标准中间液 $(10.0\mathrm{mg / L})$ ：准确吸取汞标准储备液 $(1000\mathrm{mg / L})1.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用重铬酸钾的硝酸溶液 $(0.5\mathrm{g / L})$ 稀释并定容至刻度，混匀。于 $2^{\circ}C\sim 8^{\circ}C$ 冰箱中避光保存，有效期1年。  
3.4.3 汞标准使用液 $(50.0\mu \mathrm{g} / \mathrm{L})$ : 准确吸取汞标准中间液 $(10.0\mathrm{mg / L})1.00\mathrm{mL}$ 于 $200~\mathrm{mL}$ 容量瓶中, 用重铬酸钾的硝酸溶液 $(0.5\mathrm{g / L})$ 稀释并定容至刻度, 混匀。临用现配。  
3.4.4 汞标准系列溶液：分别吸取汞标准使用液 $(50.0\mu \mathrm{g} / \mathrm{L})0.00\mathrm{mL},0.20\mathrm{mL},0.50\mathrm{mL},1.00\mathrm{mL}$ 、 $1.50~\mathrm{mL},2.00~\mathrm{mL},2.50~\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，用硝酸溶液 $(1 + 9)$ 稀释并定容至刻度，混匀，相当于汞浓度为 $0.00\mu \mathrm{g} / \mathrm{L},0.20\mu \mathrm{g} / \mathrm{L},0.50\mu \mathrm{g} / \mathrm{L},1.00\mu \mathrm{g} / \mathrm{L},1.50\mu \mathrm{g} / \mathrm{L},2.00\mu \mathrm{g} / \mathrm{L},2.50\mu \mathrm{g} / \mathrm{L}$ 。临用现配。

# 5 分析步骤

# 5.1 试样预处理

5.1.1 粮食、豆类等样品取可食部分粉碎均匀，装入洁净聚乙烯瓶中，密封保存备用。  
5.1.2 蔬菜、水果、鱼类、肉类及蛋类等新鲜样品，洗净晾干，取可食部分匀浆，装入洁净聚乙烯瓶中，密封，于 $2\,^{\circ}\mathrm{C} \sim 8\,^{\circ}\mathrm{C}$ 冰箱冷藏备用。  
5.1.3 乳及乳制品匀浆或均质后装入洁净聚乙烯瓶中，密封于 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 冰箱冷藏备用。

# 5.2 试样消解

# 5.2.1 微波消解法

称取固体试样 $0.2\mathrm{g}\sim 0.5\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ，含水分较多的样品可适当增加取样量至 $0.8\mathrm{g}$ ）或准确称取液体试样 $1.0\mathrm{g}\sim 3.0\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ，对于植物油等难消解的样品称取 $0.2\mathrm{g}\sim 0.5\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ，置于消解罐中，加入 $5\mathrm{mL}\sim 8\mathrm{mL}$ 硝酸，加盖放置 $1\mathrm{h}$ ，对于难消解的样品再加入 $0.5\mathrm{mL}\sim 1\mathrm{mL}$ 过氧化氢，旋紧罐盖，按照微波消解仪的标准操作步骤进行消解（微波消解参考条件见附录A中表A.1）。冷却后取出，缓慢打开罐盖排气，用少量水冲洗内盖，将消解罐放在控温电热板上或超声水浴箱中， $80^{\circ}\mathrm{C}$ 下加热或超声脱气 $3\mathrm{min}\sim 6\mathrm{min}$ 赶去棕色气体，取出消解内罐，将消化液转移至 $25\mathrm{mL}$ 容量瓶中，用少量水分3次洗涤内罐，洗涤液合并于容量瓶中并定容至刻度，混匀备用；同时做空白试验。

# 5.2.2 压力罐消解法

称取固体试样 $0.2\mathrm{g}\sim 1.0\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ，含水分较多的样品可适当增加取样量至 $2\mathrm{g}$ ），或准确称取液体试样 $1.0\mathrm{g}\sim 5.0\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ），对于植物油等难消解的样品称取 $0.2\mathrm{g}\sim 0.5\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ），置于消解内罐中，加入 $5\mathrm{mL}$ 硝酸，放置 $1\mathrm{h}$ 或过夜，盖好内盖，旋紧不锈钢外套，放入恒温干燥箱， $140^{\circ}\mathrm{C}\sim 160^{\circ}\mathrm{C}$ 下保持 $4\mathrm{h}\sim 5\mathrm{h}$ ，在箱内自然冷却至室温，缓慢旋松不锈钢外套，将消解内罐取出，用少量水冲洗内盖，将消解罐放在控温电热板上或超声水浴箱中， $80^{\circ}\mathrm{C}$ 下加热或超声脱气 $3\mathrm{min}\sim 6\mathrm{min}$ 赶去棕色气体。取出消解内罐，将消化液转移至 $25\mathrm{mL}$ 容量瓶中，用少量水分3次洗涤内罐，洗涤液合并于容量瓶中并定容至刻度，混匀备用；同时做空白试验。

# 5.2.3 回流消化法

# 5.2.3.1 粮食

称取 $1.0\mathrm{g}\sim 4.0\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )试样，置于消化装置锥形瓶中，加玻璃珠数粒，加 $45~\mathrm{mL}$ 硝酸、 $10~\mathrm{mL}$ 硫酸，转动锥形瓶防止局部炭化。装上冷凝管后，低温加热，待开始发泡即停止加热，发泡停止后，加热回流 $2\mathrm{h}$ 。如加热过程中溶液变棕色，再加 $5\mathrm{mL}$ 硝酸，继续回流 $2\mathrm{h}$ ，消解到样品完全溶解，一般呈淡黄色或无色，待冷却后从冷凝管上端小心加入 $20~\mathrm{mL}$ 水，继续加热回流 $10\mathrm{min}$ ，放置冷却后，用适量水冲洗冷凝管，冲洗液并入消化液中，将消化液经玻璃棉过滤于 $100~\mathrm{mL}$ 容量瓶内，用少量水洗涤锥形瓶、滤器，洗涤液并入容量瓶内，加水至刻度，混匀备用；同时做空白试验。

# 5.2.3.2 植物油及动物油脂

称取 $1.0\mathrm{g}\sim 3.0\mathrm{g}$ （精确到 $0.001\mathrm{g})$ 试样，置于消化装置锥形瓶中，加玻璃珠数粒，加入 $7\mathrm{mL}$ 硫酸，小心混匀至溶液颜色变为棕色，然后加 $40~\mathrm{mL}$ 硝酸。后续步骤同5.2.3.1“装上冷凝管后，小火加热同时做空白试验”。

# 5.2.3.3 薯类、豆制品

称取 $1.0\mathrm{g}\sim 4.0\mathrm{g}$ （精确到 $0.001\mathrm{g})$ 试样，置于消化装置锥形瓶中，加玻璃珠数粒及 $30~\mathrm{mL}$ 硝酸、 $5\mathrm{mL}$ 硫酸，转动锥形瓶防止局部炭化。后续步骤同5.2.3.1“装上冷凝管后，小火加热……同时做空白试验”。

# 5.2.3.4 肉、蛋类

称取 $0.5\mathrm{g}\sim 2.0\mathrm{g}$ （精确到 $0.001\mathrm{g})$ 试样，置于消化装置锥形瓶中，加玻璃珠数粒及 $30~\mathrm{mL}$ 硝酸、 $5\mathrm{mL}$ 硫酸，转动锥形瓶防止局部炭化。后续步骤同5.2.3.1“装上冷凝管后，小火加热……同时做空白试

验”。

# 5.2.3.5 乳及乳制品

称取 $1.0\mathrm{g}\sim 4.0\mathrm{g}$ （精确到 $0.001\mathrm{g})$ 试样，置于消化装置锥形瓶中，加玻璃珠数粒及 $30~\mathrm{mL}$ 硝酸，乳加 $10~\mathrm{mL}$ 硫酸，乳制品加 $5\mathrm{mL}$ 硫酸，转动锥形瓶防止局部炭化。后续步骤同5.2.3.1“装上冷凝管后，小火加热……同时做空白试验”。

# 5.3 测定

# 5.3.1 仪器参考条件

根据各自仪器性能调至最佳状态。光电倍增管负高压： $240\mathrm{V}$ ；汞空心阴极灯电流： $30~\mathrm{mA}$ ；原子化器温度： $200^{\circ}C$ ；载气流速： $500~\mathrm{mL / min}$ ；屏蔽气流速： $1000~\mathrm{mL / min}$ 。

# 5.3.2 标准曲线的制作

设定好仪器最佳条件，连续用硝酸溶液 $(1 + 9)$ 进样，待读数稳定之后，转入标准系列溶液测量，由低到高浓度顺序测定标准溶液的荧光强度，以汞的质量浓度为横坐标，荧光强度为纵坐标，绘制标准曲线。

注：可根据仪器的灵敏度及样品中汞的实际含量微调标准系列溶液中汞的质量浓度范围。

# 5.3.3 试样溶液的测定

转入试样测量，先用硝酸溶液 $(1 + 9)$ 进样，使读数基本回零，再分别测定处理好的试样空白和试样溶液。

# 6 分析结果的表述

试样中汞含量按式(1)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中汞的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——试样溶液中汞含量，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白液中汞含量，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

V ——试样消化液定容总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样称样量，单位为克(g)；

1000——换算系数。

当汞含量 $\geqslant 1.00\mathrm{mg / kg}$ 时，计算结果保留三位有效数字；当汞含量 $<  1.00\mathrm{mg / kg}$ 时，计算结果保留两位有效数字。

# 7 精密度

样品中汞含量大于 $1\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ ；小于或等于 $1\mathrm{mg / kg}$ 且大于 $0.1\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ ；小于或等于 $0.1\mathrm{mg / kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当样品称样量为 $0.5\mathrm{g}$ ，定容体积为 $25~\mathrm{mL}$ 时，方法检出限为 $0.003\mathrm{mg / kg}$ ，方法定量限为 $0.01\mathrm{mg / kg}$ 。

# 第二法 直接进样测汞法

# 9 原理

样品经高温灼烧及催化热解后，汞被还原成汞单质，用金汞齐富集或直接通过载气带入检测器，在 $253.7\mathrm{nm}$ 波长处测量汞的原子吸收信号，或由汞灯激发检测汞的原子荧光信号，外标法定量。

# 27 分析结果的表述

试样中甲基汞含量按式(4)计算。

$$
X = \frac {f \times \left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {4}
$$

式中：

$X$ ——试样中甲基汞的含量(以 $\mathrm{Hg}$ 计)，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$f$ ——稀释因子，2.5；

$\rho$ ——经标准曲线得到的测定液中甲基汞的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——经标准曲线得到的空白溶液中甲基汞的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——加入提取试剂的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样称样量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

当甲基汞含量 $\geqslant 1.00\mathrm{mg / kg}$ 时，计算结果保留三位有效数字；当甲基汞含量 $<  1.00~\mathrm{mg / kg}$ 时，计算结果保留两位有效数字。

# 第二法 液相色谱-电感耦合等离子体质谱联用法

# 30 原理

试样中甲基汞经超声波辅助 $5\mathrm{mol} / \mathrm{L}$ 盐酸溶液提取后，使用 $\mathrm{C_{18}}$ 反相色谱柱分离，分离后的目标化合物经过雾化由载气送入电感耦合等离子体(ICP)炬焰中，经过蒸发、解离、原子化、电离等过程，大部分转化为带正电荷的离子，经离子采集系统进入质谱仪，质谱仪根据质荷比进行分离测定。以保留时间和质荷比定性，外标法定量。

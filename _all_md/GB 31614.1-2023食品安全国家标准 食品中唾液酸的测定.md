# 食品中唾液酸的测定

2023-09-06 发布

2024-03-06 实施

# 前言

本标准代替GB/T30636—2014《燕窝及其制品中唾液酸的测定 液相色谱法》。

本标准与GB/T30636—2014相比，主要变化如下：

——修改了标准名称；  
——修改了方法的适用范围，增加了液态乳、乳粉、糕点、饮料；  
明确了燕窝及其制品中测定的目标物质为结合态唾液酸；  
——增加了液相色谱-荧光检测法和液相色谱-质谱/质谱法。

# 食品安全国家标准

# 食品中唾液酸的测定

# 1 范围

本标准规定了食品中唾液酸的测定方法。

本标准第一法液相色谱-紫外检测法适用于燕窝及其制品中结合态唾液酸的测定，第二法液相色谱-荧光检测法和第三法液相色谱-质谱/质谱法适用于液态乳、乳粉、糕点、饮料中唾液酸的测定。

# 第一法 液相色谱-紫外检测法

# 2 原理

样品在盐酸溶液中加热水解，释放出结合态唾液酸，样品试液用强阴离子交换色谱柱分离，紫外检测器或二极管阵列检测器检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂及材料

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 磷酸 $\mathrm{(H_3PO_4)}$ ：色谱纯。  
3.1.3 浓盐酸 $(\mathrm{HCl})$ ： $12\mathrm{mol} / \mathrm{L}$   
3.1.4 透析袋：可透过相对分子质量7000以下的化合物，临用前用水浸泡 $1\mathrm{h}$ 。  
3.1.5 微孔滤膜： $0.45\mu \mathrm{m}$ ，有机相型。

# 3.2 试剂配制

3.2.1 盐酸溶液 $(0.05\mathrm{mol} / \mathrm{L})$ ：吸取 $4.17~\mathrm{mL}$ 浓盐酸，溶于水并稀释至 $1000~\mathrm{mL}$ 。  
3.2.2 $0.1\%$ 磷酸溶液：吸取 $1\mathrm{mL}$ 磷酸，溶于水并稀释至 $1000~\mathrm{mL}$   
3.2.3 $0.1\%$ 磷酸溶液-乙腈 $(4 + 6)$ ：将 $0.1\%$ 磷酸溶液和乙腈按 $4:6$ 的体积比混合均匀。

# 3.3 标准品

唾液酸（ $N$ -乙酰神经氨酸， $\mathrm{C_{11}H_{19}NO_9}$ ，CAS号131-48-6）：纯度 $\geqslant 96\%$ ，或经国家认证授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 唾液酸标准储备液 $(1000\mathrm{mg / L})$ ：准确称取适量唾液酸标准品（按纯度进行折算），用水溶解并

配制成质量浓度为 $1000\mathrm{mg / L}$ 的标准储备溶液，于 $4^{\circ}C$ 下避光保存，保存期6个月。

3.4.2 唾液酸标准系列工作液：分别吸取唾液酸标准储备液 $(1000\mathrm{mg / L})0.1\mathrm{mL},0.5\mathrm{mL},1.0\mathrm{mL}$ 、 $5.0~\mathrm{mL},10.0~\mathrm{mL}$ 和 $20.0~\mathrm{mL}$ 容量瓶中，加入 $0.1\%$ 磷酸溶液-乙腈 $(4 + 6)$ 定容至刻度，混匀。唾液酸标准系列工作液的质量浓度分别为 $1.0\mathrm{mg / L},5.0\mathrm{mg / L},10.0\mathrm{mg / L},50.0\mathrm{mg / L},100.0\mathrm{mg / L}$ 和 $200.0\mathrm{mg / L}$ ，临用现配。

# 4 仪器和设备

4.1 液相色谱仪：配有紫外检测器或二极管阵列检测器。  
4.2 恒温水浴锅。  
4.3 涡旋混合器。  
4.4 分析天平：感量为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$   
4.5 匀浆机：转速不低于 $10000\mathrm{r / min}$   
4.6 烘箱。  
4.7 高速离心机。  
4.8 筛网：筛网孔径 $0.150 \mathrm{~mm}$ 。  
4.9 研钵。

# 5 分析步骤

# 5.1 样品前处理

# 5.1.1 试样制备

# 5.1.1.1 燕窝（盏、条、碎等）和固态燕窝制品

称取 $10\mathrm{g}$ 燕窝、固态燕窝制品在 $101^{\circ}\mathrm{C} \sim 105^{\circ}\mathrm{C}$ 烘箱中烘干至恒重（两次质量差不超过 $2\mathrm{mg}$ ），在干燥器中冷却后研磨，全部过100目筛(筛网孔径 $0.150\mathrm{mm}$ )后装入洁净容器，放于干燥器内，密闭备用。

# 5.1.1.2 液态燕窝制品

液态燕窝制品混匀后取 $200\mathrm{g}$ ，先低速匀浆，再缓慢升至转速 $10000\mathrm{r / min}$ ，匀浆 $3\mathrm{min}$ ，静置至泡沫消除，装入洁净容器，密闭避光，冷藏保存。

# 5.1.2 试样提取

# 5.1.2.1 燕窝（盏、条、碎等）和固态燕窝制品

唾液酸总量的测定：称取 $0.1\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )样品于 $25~\mathrm{mL}$ 具塞比色管中，加入 $10~\mathrm{mL}$ 盐酸溶液，盖上玻璃塞，涡旋混匀，置于 $80^{\circ}C$ 水浴中水解 $40\mathrm{min}$ ，在水浴过程中每 $5\mathrm{min}$ 振荡一次，取出离心管，冷却至室温。将水解液转移至 $100~\mathrm{mL}$ 容量瓶，用适量水洗涤离心管两次并转移至容量瓶中，用 $0.1\%$ 磷酸溶液-乙腈 $(4 + 6)$ 定容至刻度，混匀。移取 $2\mathrm{mL}$ 于离心管中， $15000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，取上清液过 $0.45\mu \mathrm{m}$ 微孔滤膜后，待测。

游离态唾液酸含量的测定：称取 $0.1\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )样品于 $25~\mathrm{mL}$ 具塞比色管中，加入 $10~\mathrm{mL}$ 水，涡旋振荡 $3\mathrm{min}$ ，将试样溶液转移至 $100~\mathrm{mL}$ 容量瓶中，用适量水洗涤离心管两次并转移至容量瓶中，用 $0.1\%$ 磷酸溶液-乙腈 $(4 + 6)$ 定容至刻度，混匀。移取 $2\mathrm{mL}$ 于离心管中， $15000~\mathrm{r / min}$ 离心

$3\mathrm{min}$ ，取上清液过 $0.45\mu \mathrm{m}$ 微孔滤膜后，待测。

# 5.1.2.2 液态燕窝制品

结合态唾液酸含量的测定：称取 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）样品置于透析袋中（试样体积不超过透析袋容积的 $20\%$ ），扎紧透析袋两端，置于 $1000~\mathrm{mL}$ 烧杯中在流动的自来水下透析 $24\mathrm{h}$ 。透析后，将此透析袋中的试液转移至 $25~\mathrm{mL}$ 具塞比色管中，加少量水冲洗透析袋，合并至比色管中，加入 $104\mu \mathrm{L}$ 浓盐酸，加水定容至 $25~\mathrm{mL}$ ，混匀，使水解液中盐酸的质量浓度为 $0.05\mathrm{mol / L}$ 。盖上玻璃塞，置于 $80^{\circ}\mathrm{C}$ 水浴中水解 $40\mathrm{min}$ ，取出比色管，冷却至室温。将水解液转移至 $100~\mathrm{mL}$ 容量瓶中，用适量水洗涤比色管两次并转移至容量瓶中，用 $0.1\%$ 磷酸溶液-乙腈 $(4 + 6)$ 定容至刻度，混匀。移取 $2\mathrm{mL}$ 于离心管中， $15000~\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，取上清液过 $0.45\mu \mathrm{m}$ 微孔滤膜后，待测。

# 5.2 液相色谱参考条件

液相色谱参考条件如下：

a）色谱柱：SAX强阴离子交换柱， $250\mathrm{mm}\times 4.6\mathrm{mm}$ （内径）， $5\mu \mathrm{m}$ ，或相当者。  
b） 柱温： $30^{\circ}C$   
c) 检测波长: $205 \mathrm{~nm}$ 。  
d) 流动相: $0.1\%$ 磷酸溶液-乙腈 $(40 + 60$ , 体积比)。  
e) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ 。  
f) 进样量: $10 \mu \mathrm{L}$ 。

# 5.3 标准曲线的制作

将标准系列工作液分别注入液相色谱仪中，测定相应的峰面积，以标准系列工作液中唾液酸的质量浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。唾液酸标准溶液的色谱图参见附录A中图A.1。

# 5.4 试样溶液的测定

将试样溶液注入液相色谱仪中，得到峰面积，根据标准曲线得到待测液中唾液酸的质量浓度。标准工作液和待测样液中唾液酸的响应值均应在仪器线性响应范围内。如果含量超过标准曲线线性范围，需调整称样量或定容体积重新检测。

# 6 分析结果的表述

试样中结合态唾液酸的含量按式(1)计算。

$$
X = \frac {\rho \times V}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中唾液酸的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中唾液酸的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果需扣除空白值，保留3位有效数字。

对于燕窝和固态燕窝制品，分别测定唾液酸总量和游离态唾液酸含量，结合态唾液酸含量由唾液酸

总量减去游离态唾液酸含量计算得到。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

燕窝和固态燕窝制品，当称样量为 $0.1\mathrm{g}$ 时，检出限为 $0.3\mathrm{g / kg}$ ，定量限为 $1.0\mathrm{g / kg}$ ；液态燕窝制品，当称样量为 $10\mathrm{g}$ 时，检出限为 $0.003\mathrm{g / kg}$ ，定量限为 $0.01\mathrm{g / kg}$ 。

# 第二法 液相色谱-荧光检测法

# 9 原理

样品在盐酸溶液中加热水解，释放出唾液酸，提取液经固相萃取柱净化后，与邻苯二胺反应生成荧光衍生物，样品试液用反相色谱柱分离，荧光检测器检测，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 10.1 试剂及材料

10.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
10.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
10.1.3 磷酸 $\left(\mathrm{H}_{3} \mathrm{PO}_{4}\right)$ ：色谱纯。  
10.1.4 四氢呋喃 $\left(\mathrm{C}_{4} \mathrm{H}_{8} \mathrm{O}\right)$ ：色谱纯。  
10.1.5 浓盐酸 $(\mathrm{HCl})$ ： $12\mathrm{mol} / \mathrm{L}$   
10.1.6 邻苯二胺盐酸盐 $\mathrm{(C_6H_8N_2\cdot 2HCl)}$   
10.1.7 三氯甲烷 $\mathrm{(CHCl_3)}$   
10.1.8 含极性基团的反相聚合物固相萃取柱: $60 \mathrm{mg}, 3 \mathrm{~mL}$ , 或相当者。含极性基团的反相聚合物固相萃取柱吸附剂是由亲脂性二乙烯苯和亲水性 $N$ -乙烯基吡咯烷酮两种单体按照一定比例聚合成的大孔聚合物。  
10.1.9 微孔滤膜： $0.45\mu \mathrm{m}$ ，有机相型。

# 10.2 试剂配制

10.2.1 盐酸溶液 $(0.05\mathrm{mol} / \mathrm{L})$ ：吸取 $4.17~\mathrm{mL}$ 浓盐酸，溶于水并稀释至 $1000~\mathrm{mL}$ 。  
10.2.2 乙腈-水溶液 $(1 + 9)$ ：将乙腈和水按 $1:9$ 的体积比混合均匀。  
10.2.3 邻苯二胺盐酸盐溶液 $(50\mathrm{mg / mL})$ ：称取 $2.5\mathrm{g}$ 邻苯二胺盐酸盐，加入 $50~\mathrm{mL} 0.05\mathrm{mol / L}$ 盐酸溶液溶解，混匀，现用现配。  
10.2.4 $0.2\%$ 磷酸溶液：吸取 $2\mathrm{mL}$ 磷酸，溶于水并稀释至 $1000~\mathrm{mL}$

# 10.3 标准品

唾液酸（ $N$ -乙酰神经氨酸， $\mathrm{C_{11}H_{19}NO_9}$ ，CAS号131-48-6）：纯度 $\geqslant 96\%$ ，或经国家认证授予标准物质证书的标准品。

# 10.4 标准溶液配制

10.4.1 唾液酸标准储备液 $(1000\mathrm{mg / L})$ ：准确称取适量唾液酸标准品(按纯度进行折算)，用水溶解并配制成质量浓度为 $1000~\mathrm{mg / L}$ 的标准储备溶液，于 $4^{\circ}C$ 下避光保存，保存期6个月。  
10.4.2 唾液酸标准系列工作液：分别吸取唾液酸标准储备液 $(1000\mathrm{mg / L})0.2\mathrm{mL},1.0\mathrm{mL},2.0\mathrm{mL}$ 、 $4.0~\mathrm{mL},10.0~\mathrm{mL}$ 和 $20.0~\mathrm{mL}$ 容量瓶中，用乙腈-水溶液 $(1 + 9)$ 定容至刻度，混匀。唾液酸标准系列工作液的质量浓度分别为 $2.0\mathrm{mg / L},10.0\mathrm{mg / L},20.0\mathrm{mg / L},40.0\mathrm{mg / L},100.0\mathrm{mg / L}$ 和 $200.0\mathrm{mg / L}$ ，临用现配。

# 11 仪器和设备

11.1 液相色谱仪：配有荧光检测器。  
11.2 恒温水浴锅。  
11.3 涡旋混合器。  
11.4 分析天平：感量为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。  
11.5 匀浆机：转速不低于 $10000\mathrm{r / min}$   
11.6 高速离心机。

# 12 分析步骤

# 12.1 样品前处理

# 12.1.1 试样制备

液态试样摇匀；基质均匀的半固态试样和粉状试样直接用于下一步试样提取；其他试样需匀浆或粉碎均匀。

# 12.1.2 试样提取

称取 $1\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 样品于 $25\mathrm{mL}$ 具塞比色管中（固体样品先加入 $3\mathrm{mL}$ 水，涡旋振荡 $30\mathrm{s}$ ），沿管壁缓慢加入 $10\mathrm{mL}$ 盐酸溶液，一边加一边振荡，盖上玻璃塞，涡旋混匀，置于 $80^{\circ}\mathrm{C}$ 水浴中水解 $40\mathrm{min}$ ，在水浴过程中每 $5\mathrm{min}$ 振荡一次。取出比色管，冷却至室温，用乙腈-水溶液 $(1 + 9)$ 定容至 $25\mathrm{mL}$ ，混匀，静置 $5\mathrm{min}$ 。取 $10\mathrm{mL}$ 上层溶液放入离心管中，加入 $5\mathrm{mL}$ 三氯甲烷，涡旋混匀后， $9000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液待净化。

# 12.1.3 试样净化

固相萃取柱依次用 $3\mathrm{mL}$ 甲醇和 $3\mathrm{mL}$ 水活化，负压抽干，样液以小于 $1\mathrm{mL} / \mathrm{min}$ 的流速通过固相萃取柱，收集流出液，过 $0.45\mu \mathrm{m}$ 微孔滤膜后，待衍生化。

# 12.1.4 衍生化

准确吸取 $1.0\mathrm{mL}$ 样液于 $10~\mathrm{mL}$ 具塞比色管中，准确加入 $1.0\mathrm{mL}$ 邻苯二胺盐酸盐溶液，盖上玻璃

塞，混匀，置于 $80^{\circ}\mathrm{C}$ 水浴中反应 $35\mathrm{min}$ ，取出比色管，冷却至室温，过 $0.45\mu \mathrm{m}$ 微孔滤膜后，待测。

同时做唾液酸标准系列工作溶液的衍生化，衍生后标准系列工作溶液的质量浓度分别为 $1.0\mathrm{mg / L}$ 、 $5.0\mathrm{mg / L},10.0\mathrm{mg / L},20.0\mathrm{mg / L},50.0\mathrm{mg / L}$ 和 $100.0\mathrm{mg / L}$ 。

# 12.2 液相色谱参考条件

液相色谱参考条件如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱， $250~\mathrm{mm}\times 4.6~\mathrm{mm}$ （内径）， $5\mu \mathrm{m}$ ，或相当者。  
b） 柱温： $30~\mathrm{^\circ C}$   
c）荧光检测器激发波长： $235~\mathrm{nm}$ ，发射波长： $414~\mathrm{nm}$   
d) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ 。  
e) 进样量: $10 \mu \mathrm{L}$ 。  
f) 流动相：A 相: $950 \mathrm{~mL} 0.2\%$ 磷酸溶液 $+50 \mathrm{~mL}$ 四氢呋喃; B 相: 甲醇。梯度洗脱, 见表 1。

# 12.3 标准曲线的制作

将衍生后的标准系列工作液分别注入液相色谱仪中，测定相应的峰面积，以标准系列工作液中唾液酸的质量浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。唾液酸标准溶液的色谱图参见附录B中图B.1。

# 12.4 试样溶液的测定

将衍生后的试样溶液注入液相色谱仪中，得到峰面积，根据标准曲线得到待测液中唾液酸的质量浓度。标准工作液和待测样液中唾液酸的响应值均应在仪器线性响应范围内。如果含量超过标准曲线线性范围，需调整称样量或定容体积重新检测。

# 13 分析结果的表述

试样中唾液酸的含量按式(2)计算。

$$
X = \frac {\rho \times V \times f}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中唾液酸的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中唾液酸的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

# 第三法 液相色谱-质谱/质谱法

# 16 原理

样品在盐酸溶液中加热水解，释放出唾液酸，提取液经固相萃取柱净化后，样品试液用反相色谱柱分离，液相色谱-质谱/质谱测定，外标法定量。

# 17 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 17.1 试剂及材料

17.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
17.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
17.1.3 甲酸（HCOOH）：色谱纯。  
17.1.4 浓盐酸 $(\mathrm{HCl})$ ： $12\mathrm{mol} / \mathrm{L}$   
17.1.5 三氯甲烷 $\mathrm{(CHCl_3)}$ 。  
17.1.6 含极性基团的反相聚合物固相萃取柱： $60\mathrm{mg}, 3\mathrm{mL}$ ，或相当者。含极性基团的反相聚合物固相萃取柱吸附剂是由亲脂性二乙烯苯和亲水性 $N$ -乙烯基吡咯烷酮两种单体按照一定比例聚合成的大孔聚合物。  
17.1.7 微孔滤膜： $0.22\mu \mathrm{m}$ ，有机相型。

# 17.2 试剂配制

17.2.1 盐酸溶液 $(0.05\mathrm{mol} / \mathrm{L})$ ：吸取 $4.17~\mathrm{mL}$ 浓盐酸，溶于水并稀释至 $1000~\mathrm{mL}$ 。  
17.2.2 乙腈-水溶液 $(1 + 9)$ ：将乙腈和水按 $1:9$ 的体积比混合均匀。  
17.2.3 $0.1\%$ 甲酸溶液：吸取 $1\mathrm{mL}$ 甲酸，溶于水并稀释至 $1000~\mathrm{mL}$

# 17.3 标准品

唾液酸（ $N$ -乙酰神经氨酸， $\mathrm{C_{11}H_{19}NO_9}$ ，CAS号131-48-6）：纯度 $\geqslant 96\%$ ，或经国家认证授予标准物质证书的标准品。

# 17.4 标准溶液配制

17.4.1 唾液酸标准储备液 $(1000\mathrm{mg / L})$ ：准确称取适量唾液酸标准品(按纯度进行折算)，用水溶解并配制成质量浓度为 $1000~\mathrm{mg / L}$ 的标准储备溶液，于 $4^{\circ}C$ 下避光保存，保存期6个月。  
17.4.2 唾液酸标准中间液 $(100\mathrm{mg / L})$ ：吸取标准储备液 $(1000\mathrm{mg / L})10\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加水定容至刻度，混匀，于 $4^{\circ}C$ 下避光保存，保存期1个月。  
17.4.3 唾液酸标准系列工作液：分别吸取唾液酸标准中间液 $(100\mathrm{mg / L})0.1\mathrm{mL},0.2\mathrm{mL},0.5\mathrm{mL}$ 、 $1.0~\mathrm{mL}$ 和 $2.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用乙腈-水溶液 $(1 + 9)$ 定容至刻度，混匀。唾液酸标准系列工作液的质量浓度分别为 $0.1\mathrm{mg / L},0.2\mathrm{mg / L},0.5\mathrm{mg / L},1.0\mathrm{mg / L}$ 和 $2.0\mathrm{mg / L}$ ，临用现配。

# 19.1.2 试样提取

称取 $1\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 样品于 $25\mathrm{mL}$ 具塞比色管中（固体样品先加入 $3\mathrm{mL}$ 水，涡旋振荡 $30\mathrm{s}$ ），沿管壁缓慢加入 $10\mathrm{mL}$ 盐酸溶液，一边加一边振荡，盖上玻璃塞，涡旋混匀，置于 $80^{\circ}\mathrm{C}$ 水浴中水解 $40\mathrm{min}$ ，在水浴过程中每 $5\mathrm{min}$ 振荡一次。取出比色管，冷却至室温，用乙腈-水溶液 $(1 + 9)$ 定容至 $25\mathrm{mL}$ ，混匀，静置 $5\mathrm{min}$ 。取 $10\mathrm{mL}$ 上层溶液放入离心管中，加入 $5\mathrm{mL}$ 三氯甲烷，涡旋混匀后， $9000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液待净化。

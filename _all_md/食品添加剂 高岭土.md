# 附录A

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T603规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 硫酸。

A.3.1.2 氨水溶液： $2 + 3$

# A.3.2 鉴别方法

称取 $1.0\mathrm{g}$ 试样置于蒸发皿中，加入 $10~\mathrm{mL}$ 水，缓慢加入 $5\mathrm{mL}$ 硫酸，混匀。置于电炉上蒸至近千，继续加热至冒三氧化硫的白色浓烟，取下。冷却后小心加入 $20~\mathrm{mL}$ 水，煮沸几分钟后用定性滤纸过滤。灰色的二氧化硅残留在漏斗中，在滤液中加入氨水溶液，应有凝胶状的白色氢氧化铝沉淀生成，此沉淀不溶于过量的氨水溶液。

# A.4 碳酸盐的测定

# A.4.1 试剂和材料

硫酸。

# A.4.2 分析步骤

称取 $1.0\mathrm{g}$ 试样，置于 $50~\mathrm{mL}$ 烧杯中。加入 $10~\mathrm{mL}$ 水，混匀。边冷却边缓慢加入 $5\mathrm{mL}$ 硫酸。在加酸过程中，没有气泡产生，即为通过试验。

# A.5 铁的测定

# A.5.1 试剂和材料

水杨酸钠。

# A.5.2 分析步骤

称取 $2.0\mathrm{g}$ 试样，置于研钵内，加入 $10~\mathrm{mL}$ 水研磨均匀，加入 $0.5\mathrm{g}$ 水杨酸钠，颜色不深于浅红色时，即为通过试验。

# A.6 硫化物的测定

# A.6.1 试剂和材料

A.6.1.1 盐酸溶液： $1 + 3$

A.6.1.2 乙酸铅溶液： $95\mathrm{g / L}$ 。将 $9.5\mathrm{g}$ 乙酸铅 $\left[\mathrm{Pb}(\mathrm{CH}_3\mathrm{COO})_2\cdot 3\mathrm{H}_2\mathrm{O})\right]$ 溶解于新煮过的水中，稀释至 $100~\mathrm{mL}$ 。

# A.6.2 分析步骤

称取 $1.0\mathrm{g}$ 试样，置于 $250~\mathrm{mL}$ 锥形瓶中，加入 $25~\mathrm{mL}$ 水， $15~\mathrm{mL}$ 盐酸溶液，立即用乙酸铅溶液润湿的滤纸盖在瓶口，加热至沸，并煮沸几分钟。滤纸无褐色产生，即为通过试验。

# A.7 酸溶物的测定

# A.7.1 试剂和材料

盐酸溶液： $1 + 3$

# A.7.2 仪器和设备

电热恒温干燥箱：温度能控制为 $105\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}$ 。

# A.7.3 分析步骤

称取约 $1\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ 。置于烧杯中，加入 $20~\mathrm{mL}$ 盐酸溶液，搅拌，放置 $15\mathrm{min}$ 后过滤并洗涤。滤液收集于 $50~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。移取 $25~\mathrm{mL}$ 滤液于预先于 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 干燥至质量恒定的蒸发皿中，蒸发至干，将蒸发皿置于 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 的电热恒温干燥箱中干燥至质量恒定。

# A.7.4 结果计算

酸溶物的质量分数 $\omega_{1}$ 按式(A.1)计算：

$$
w _ {1} = \frac {m _ {1} \times 50}{m \times 25} \times 100 \% \tag{A.1}
$$

式中：

$m_{1}$ ——干燥后残余物的质量，单位为克 $(\mathrm{g})$ ；

50——容量瓶的容积，单位为毫升 $(\mathrm{mL})$

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

25——移取试验溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.8 灼烧减量的测定

# A.8.1 仪器和设备

高温炉：温度能控制为 $575\,^{\circ}\mathrm{C} \pm 25\,^{\circ}\mathrm{C}$ 。

# A.8.2 分析步骤

称取约 $2\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ 。置于预先于 $575~^\circ \mathrm{C}\pm 25~^\circ \mathrm{C}$ 灼烧至质量恒定的瓷坩埚中，于 $575~^\circ \mathrm{C}\pm 25~^\circ \mathrm{C}$ 的高温炉中灼烧至质量恒定。

# A.8.3 结果计算

灼烧减量的质量分数 $\omega_{2}$ 按式(A.2)计算：

$$
w _ {2} = \frac {m - m _ {1}}{m} \times 100 \% \tag{A.2}
$$

式中：

$m$ ——试样质量，单位为克(g)；

$m_{1}$ ——灼烧后残余物的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A.9 无机砷（As）的测定

称取约 $10\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ 。置于 $250~\mathrm{mL}$ 烧瓶中，加入 $50~\mathrm{mL}$ 盐酸溶液 $(1 + 23)$ 。接上回流冷凝器，于蒸汽浴中加热 $30\mathrm{min}$ ，冷却。将上清液过滤至 $100~\mathrm{mL}$ 容量瓶中，在烧瓶中保留尽可能多的不溶物，用 $30~\mathrm{mL}$ 热水分三次洗涤残渣和烧瓶，每次滤液收集至容量瓶中，最后用 $15~\mathrm{mL}$ 热水洗涤滤纸。将滤液冷却至室温，用水稀释至刻度，摇匀。此溶液为试验溶液A，用于无机砷(As)和铅(Pb)的测定。

移取一定量试验溶液A，按GB/T5009.76或GB/T5009.11规定的方法进行测定。

# A.10 铅(Pb)的测定

移取一定量的试验溶液A(A.9)，按GB5009.12规定的方法进行测定。

# 食品安全国家标准

# 食品添加剂 甲基纤维素

# 1 范围

本标准适用于以木浆或棉浆等天然纤维素为原料，经碱化和醚化处理后，再经洗涤、干燥后制得的食品添加剂甲基纤维素。

# 2 化学名称、结构式、分子式和相对分子质量

# 2.1 化学名称

纤维素甲基醚

# 2.2 结构式

![](images/d3b53e088a7b6e314a25abbac432f9a8cfedd717915942e8692d3024f6f0f43a.jpg)  
$\mathrm{R} = \mathrm{H}$ 或 $\mathrm{CH}_3$

# 2.3 分子式

$$
\left[ \mathrm {C} _ {6} \mathrm {H} _ {7} \mathrm {O} _ {2} (\mathrm {O H}) _ {x} (\mathrm {O C H} _ {3}) _ {y} \right] _ {n}
$$

其中， $x = 1.00\sim 1.55,y = 2.00\sim 1.45,x + y = 3.00(y =$ 取代度）。

# 2.4 相对分子质量（按2011年国际原子质量）

未取代的结构单元：162.14；

取代度为1.45的结构单元：182；

取代度为2.0的结构单元：190。

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

氢氧化钠溶液或盐酸溶液： $1\mathrm{mol} / \mathrm{L}$

# A.2.2 试样液制备

在盛有 $100~\mathrm{mL}$ 水的烧杯(液面)上部加入试样 $1\mathrm{g}$ ，任其在溶液表面分散，静置，使试样变成透明的黏性溶液（约 $5\mathrm{h}$ ），振荡烧杯将残留试样打湿，用玻璃棒搅拌至试样完全溶解。

# A.2.3 鉴别方法

A.2.3.1 取 $10 \mathrm{~mL}$ 试样液, 加入同体积的氢氧化钠溶液或盐酸溶液时, 此时, 混合液应保持透明、有黏性。  
A.2.3.2 取 $10\mathrm{mL}$ 试样液，加热，溶液产生雾状和片状沉淀，冷却后又溶解。  
A.2.3.3 取 $10\mathrm{mL}$ 试样液，倾倒在玻璃板上，蒸发除去水分，形成白色薄膜。

# A.3 甲氧基 $(-\mathrm{OCH}_3)$ 含量（以干基计）的测定

# A.3.1 试剂和材料

A.3.1.1 甲苯。  
A.3.1.2 邻二甲苯。  
A.3.1.3 己二酸。  
A.3.1.4 氢碘酸。  
A.3.1.5 甲基碘。

# A.3.2 仪器和设备

气相色谱仪：配备热导检测器，或其他等效的检测器。

# A.3.3 参考色谱条件

A.3.3.1 色谱柱： $1.8\mathrm{m}\times 4\mathrm{mm}$ 玻璃柱，填料为 $125\mu \mathrm{m}\sim 150\mu \mathrm{m}$ 色谱纯硅藻土含 $10\%$ 的甲基硅酮油或其他等效色谱柱。  
A.3.3.2 柱温： $100\%$

A.3.3.3 进样口温度： $200^{\circ}C$   
A.3.3.4 检测器温度： $200^{\circ}C$   
A.3.3.5 载气：氦气。  
A.3.3.6 流速： $20~\mathrm{mL / min}$   
A.3.3.7 进样量：约 $2\mu \mathrm{L}$   
A.3.3.8 甲基碘、甲苯和邻二甲苯的色谱保留时间分别为 $3\mathrm{min},7\mathrm{min}$ 和 $13\mathrm{min}$

# A.3.4 分析步骤

注意：在通风橱中进行含有氢碘酸的操作步骤。使用护目镜、耐酸手套和其他合适的安全设备。在处理热玻璃瓶时，由于压力，故应极其小心。万一不慎接触氢碘酸，应用大量清水冲洗，并立即寻求医疗帮助。

# A.3.4.1 内标溶液的制备

称取 $2.5\mathrm{g}$ 甲苯，精确到 $0.001\mathrm{g}$ ，用邻二甲苯稀释至 $1000\mathrm{mL}$ 。

# A.3.4.2 标准溶液的制备

称取 $135\mathrm{mg}$ 己二酸，精确到 $0.001\mathrm{g}$ ，置于合适的血清瓶中，加入 $4.0\mathrm{mL}$ 氢碘酸，再加入 $4.0\mathrm{mL}$ 内标溶液，用隔膜盖封住瓶口，称重。通过隔膜，注入 $90~\mu \mathrm{L}$ 甲基碘，再称重，计算加入的甲基碘的重量。混合均匀，静置后分层。

# A.3.4.3 试样溶液的制备

称取 $0.065\mathrm{g}$ 试样，精确到 $0.001\mathrm{g}$ ，置于 $5\mathrm{mL}$ 带压紧隔膜的瓶中，加入与试样等量的己二酸，然后加入 $2\mathrm{mL}$ 内标溶液。小心吸取 $2\mathrm{mL}$ 氢碘酸，加入瓶中，盖紧瓶盖，称重。摇晃瓶 $30\mathrm{s}$ ，在 $150^{\circ}\mathrm{C}$ 条件下加热 $20\mathrm{min}$ ，然后再摇晃，再在 $150^{\circ}\mathrm{C}$ 条件下加热 $40\mathrm{min}$ 。将瓶子冷却 $45\mathrm{min}$ ，称重。若重量损失超过 $10\mathrm{mg}$ ，则弃掉，重新制备试样溶液。

# A.3.4.4 测定

吸取标准溶液上层液，在A.3.3参考色谱条件下，进行色谱分析，记录色谱图，计算得到等质量甲苯和甲基碘的相对响应因子 $F$ ，从而计算出甲氧基 $\mathrm{(-OCH_3)}$ 含量(以干基计)。

# A.3.4.5 结果计算

A.3.4.5.1 等质量甲苯和甲基碘的相对响应因子 $F$ ，按式(A.1)计算：

$$
F = \frac {Q}{A} \quad \dots \tag {A.1}
$$

式中：

$Q$ ——标准溶液中甲基碘与甲苯的质量比；

$A$ ——标准溶液色谱图中甲基碘与甲苯的峰面积比。

A.3.4.5.2 甲氧基 $(\mathrm{-OCH_3})$ （以干基计)的质量分数 $\varpi$ ，按式(A.2)计算：

$$
w = 2 \times \frac {3 1}{1 4 2} \times F \times \alpha \times \frac {m}{m _ {1}} \quad \dots \tag {A.2}
$$

式中：

31 甲氧基和甲基碘的相对分子质量比；

$F$ ——实际测得的甲苯和甲基碘的相对响应因子；

$\alpha$ ——试样溶液色谱图中甲基碘和甲苯的峰面积之比；

$m$ ——内标溶液中甲苯的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——试样质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准，保留2位有效数字。在重复性测定条件下获得的两次独立测定结果的绝对差值不超过其算术平均值的 $10\%$ 。

# A.4 黏度的测定

称取适量试样，加入事先加热至 $80^{\circ}\mathrm{C} \sim 90^{\circ}\mathrm{C}$ 的水制成 $2\mathrm{g} / 100\mathrm{mL}$ 的试样溶液，在约 $400\mathrm{r / min}$ 条件下搅拌 $10\mathrm{min} \sim 20\mathrm{min}$ ，置冰浴中冷却，继续搅拌 $40\mathrm{min}$ ，将试样溶液的温度调整至 $20^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ ，选择合适的旋转黏度计进行测定。当黏度(质量分数为 $2\%$ 水溶液) $\geqslant 20000\mathrm{mPa}\cdot \mathrm{s}$ 时，应改用质量分数为 $1\%$ 水溶液测定。

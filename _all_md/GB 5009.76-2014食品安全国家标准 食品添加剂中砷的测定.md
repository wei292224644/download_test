# 前言

本标准代替GB/T5009.76—2003《食品添加剂中砷的测定》。

本标准与GB/T5009.76—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂中砷的测定”；  
——删除了砷斑法；  
——增加了氢化物原子荧光测定法为第二法。

# 第一法 二乙氨基二硫代甲酸银比色法

# 2 原理

在碘化钾和氯化亚锡存在下，将样液中的高价砷还原为三价砷，三价砷与锌粒和酸产生的新生态氢作用，生成砷化氢气体，经乙酸铅棉花除去硫化氢干扰后，被溶于三乙醇胺-三氯甲烷中或吡啶中的二乙氨基二硫代甲酸银溶液吸收并作用，生成紫红色络合物，与标准比较定量。

# 3.2 试剂配制

3.2.1 硫酸溶液 $(1 + 1)$ ：量取 $100~\mathrm{mL}$ 硫酸慢慢加入 $100~\mathrm{mL}$ 水中，混匀，冷却后使用。  
3.2.2 硫酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $28~\mathrm{mL}$ 硫酸，慢慢加入水中，用水稀释到 $500~\mathrm{mL}$ 。  
3.2.3 盐酸溶液 $(1 + 1)$ ：量取 $100~\mathrm{mL}$ 盐酸慢慢加入 $100~\mathrm{mL}$ 水中，混匀，冷却后使用。  
3.2.4 氢氧化钠溶液 $(200\mathrm{g / L})$ ：称取 $20\mathrm{g}$ 氢氧化钠用水溶解并定容至 $100~\mathrm{mL}$ 。  
3.2.5 硝酸镁溶液 $(150\mathrm{g / L})$ ：称取 $15\mathrm{g}$ 硝酸镁用水溶解并定容至 $100~\mathrm{mL}$ 。  
3.2.6 碘化钾溶液 $(150\mathrm{g / L})$ ：称取 $15\mathrm{g}$ 碘化钾用水溶解并定容至 $100~\mathrm{mL}$ 。贮于棕色瓶内（临用前配制）。  
3.2.7 氯化亚锡溶液 $(400\mathrm{g / L})$ ：称取 $20\mathrm{g}$ 氯化亚锡，溶于 $50~\mathrm{mL}$ 盐酸溶液。  
3.2.8 吸收液A：称取 $0.25\mathrm{g}$ 二乙氨基二硫代甲酸银，研碎后用适量三氯甲烷溶解。加入 $1.0\mathrm{mL}$ 三乙醇胺，再用三氯甲烷稀释至 $100\mathrm{mL}$ 。静置后过滤于棕色瓶中，贮存于冰箱内备用。  
3.2.9 吸收液 B: 称取 $0.50 \mathrm{~g}$ 二乙氨基二硫代甲酸银, 研碎后用吡啶溶解并稀释至 $100 \mathrm{~mL}$ 。静置后过滤于棕色瓶中, 贮存于冰箱内备用。  
3.2.10 酚酞乙醇溶液 $(10\mathrm{g / L})$ ：称取 $1.0\mathrm{g}$ 酚酞溶于 $100~\mathrm{mL}$ 乙醇溶液中。  
3.2.11 乙酸铅溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 乙酸铅用水溶解并定容至 $100~\mathrm{mL}$ 。

# 3.3 标准品

# 3.4 标准溶液配制

3.4.1 砷标准储备溶液 $(0.1\mathrm{mg / mL})$ ：准确称取 $0.1320\mathrm{g}$ 于硫酸干燥器中干燥至恒重的三氧化二砷，溶于 $5\mathrm{mL}$ 氢氧化钠溶液中。溶解后，加入 $25~\mathrm{mL}$ 硫酸溶液，移入 $1000~\mathrm{mL}$ 容量瓶中，加新煮沸冷却的水稀释至刻度。  
3.4.2 砷标准使用液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：临用前取 $1.0~\mathrm{mL}$ 砷标准储备溶液，加 $1\mathrm{mL}$ 硫酸溶液于 $100~\mathrm{mL}$ 容量瓶中，加新煮沸冷却的水稀释至刻度。

# 5 样品处理

# 5.1 无机试样的制备

无机试样的“试样处理”可按相关标准规定的方法进行。

# 5.2 有机试样的制备

有机试样的“试样处理”除按相关标准规定的外，一般按下述方法进行：

a) 湿法消解: 称取 $5 \mathrm{~g}$ 试样 (精确至 $0.001 \mathrm{~g}$ ), 置于 $250 \mathrm{~mL}$ 锥形瓶中, 加 $10 \mathrm{~mL}$ 硝酸, 放置片刻 (或过夜) 后, 于电热板上加热, 待反应缓和后, 取下并放置冷却, 沿瓶壁加入 $5 \mathrm{~mL}$ 硫酸, 再继续加热至瓶中溶液开始变成棕色后, 不断滴加硝酸 (如有必要可滴加些高氯酸), 至有机质分解完全, 继续加热至生成大量的二氧化硫白色烟雾, 最后溶液应无色或微黄色。冷却后加 $20 \mathrm{~mL}$ 水煮沸, 除去残余的硝酸至产生白烟为止。如此处理两次, 放冷, 将溶液移入 $50 \mathrm{~mL}$ 容量瓶

中，用少量水洗涤锥形瓶2次～3次，将洗涤液并入容量瓶中，最后用水补至刻度。取相同量的硝酸、硫酸，同时做试剂空白试验。

b）干灰化法：称取 $5\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )于瓷坩埚中，加 $10~\mathrm{mL}$ 硝酸镁溶液，混匀，浸泡 $4\mathrm{h}$ ，可调式电炉上低温或水浴上蒸干，再加入 $1.00\mathrm{g}$ 氧化镁粉末仔细覆盖在干渣上，用可调式电炉小火加热至炭化完全，将坩埚移入马弗炉中，在 $550^{\circ}\mathrm{C}$ 以下灼烧至灰化完全，冷却后取出，加适量水湿润灰分，加入酚酞乙醇溶液数滴，再滴加 $(1 + 1)$ 盐酸溶液至酚酞红色褪去，然后将溶液移入 $50~\mathrm{mL}$ 容量瓶中（必要时过滤），用少量水洗涤坩埚3次，洗液并入容量瓶中，加水至刻度，混匀。取相同量的氧化镁、硝酸镁，同时做试剂空白试验。

# 6 测定

# 6.1 吸收液的选择

可根据分析的需要选择吸收液A或吸收液B。在测定过程中，样品、空白及标准溶液都应用同一吸收液。

# 6.2 限量试验

6.2.1 吸取一定量的试样液和砷的标准使用液（含砷量不低于 $5\mu \mathrm{g}$ ），分别置于砷发生瓶A中，补加硫酸至总量为 $5\mathrm{mL}$ ，加水至 $50~\mathrm{mL}$ 。  
6.2.2 于6.2.1各瓶中加 $3\mathrm{mL}$ 碘化钾溶液，混匀，放置 $5\mathrm{min}$ 。分别加入 $1\mathrm{mL}$ 氯化亚锡溶液，混匀，再放置 $15\mathrm{min}$ 。再各加入 $5\mathrm{g}$ 无砷金属锌，立即塞上装有乙酸铅棉花的导气管B，并使管B的尖端插入盛有 $5.0\mathrm{mL}$ 吸收液A或吸收液B的吸收管C中，室温反应 $1\mathrm{h}$ ，取下吸收管C，用三氯甲烷(吸收液A)或吡啶(吸收液B)将吸收液体积定容到 $5.0\mathrm{mL}$ 。  
6.2.3 经目视比色或用 $1\mathrm{cm}$ 比色杯，于 $515~\mathrm{nm}$ 波长(吸收液A)或 $540~\mathrm{nm}$ 波长(吸收液B)，测定吸收液的吸光度。样品液的色度或吸光度不得超过砷的标准吸收液的色度或吸光度。

# 6.3 定量测定

6.3.1 吸取 $25 \mathrm{~mL}$ (或适量) 试样液及同量的试剂空白液, 分别置于砷发生瓶 A 中, 补加硫酸至总量为 $5 \mathrm{~mL}$ , 加水至 $50 \mathrm{~mL}$ , 混匀。  
6.3.2 吸取 $0.0\mathrm{mL}, 2.0\mathrm{mL}, 4.0\mathrm{mL}, 6.0\mathrm{mL}, 8.0\mathrm{mL}, 10.0\mathrm{mL}$ 砷标准溶液 $(1.0\mathrm{mL}$ 相当于 $1.0\mu \mathrm{g}$ 砷），分别置于砷发生瓶A中，加水至 $40\mathrm{mL}$ ，再加 $10\mathrm{mL}(1 + 1)$ 硫酸溶液，混匀。  
6.3.3 向试样液、试剂空白液及砷标准液中各加 $3\mathrm{mL}$ 碘化钾溶液，混匀，放置 $5\mathrm{min}$ ，再分别加 $1\mathrm{mL}$ 氯化亚锡溶液，混匀，放置 $15\mathrm{min}$ 后，各加入 $5\mathrm{g}$ 无砷金属锌，立即塞上装有乙酸铅棉花的导气管B，并使管B的尖端插入盛有 $5.0\mathrm{mL}$ 吸收液A或吸收液B的吸收管C中，室温反应 $1\mathrm{h}$ ，取下吸收管C，用三氯甲烷(吸收液A)或吡啶(吸收液B)将吸收液体积定容到 $5.0\mathrm{mL}$ 。用 $1\mathrm{cm}$ 比色杯，于 $515\mathrm{nm}$ 波长（吸收液A）或 $540\mathrm{nm}$ 波长（吸收液B)处，用零管调节仪器零点，测吸光度，绘制标准曲线。

# 7 分析结果的表述

试样中砷含量按式(1)计算：

$$
c = \frac {\left(m _ {1} - m _ {2}\right) \times V _ {1} \times 1 0 0 0}{m \times V _ {2} \times 1 0 0 0} \tag {1}
$$

式中：

$c$ ——样品中砷的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ [或毫克每升 $(\mathrm{mg / L})]$ ；

$m_{1}$ ——试样液中砷的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_{2}$ ——试剂空白液中砷的质量，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——试样处理后定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品质量(体积)，单位为克(或毫升) $(\mathrm{g / mL})$ ；

$V_{2}$ ——测定时所取试样液体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数。

计算结果保留两位有效数字。

# 8 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 氢化物原子荧光光度法

# 9 原理

试样经消解后，加入硫脲使五价砷还原为三价砷，再加入硼氢化钠或硼氢化钾使还原成砷化氢，由氩气载入石英原子化器中分解为原子态砷，在特制砷空心阴极的发射光激发下产生原子荧光，其荧光强度在固定条件下与被测液中的砷浓度成正比，与标准系列比较定量。

# 12 分析步骤

# 12.1 试样消解

12.1.1 湿法消解：称取固体试样 $1\mathrm{g}\sim 2.5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，液体试样 $5\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，置于 $100~\mathrm{mL}$ 锥形瓶中，加硝酸 $20~\mathrm{mL}\sim 40~\mathrm{mL}$ ，硫酸 $1.25~\mathrm{mL}$ ，放置过夜。次日置于电热板上加热消解（ $\langle < 220^{\circ}C)$ 。若消解液处理至 $10~\mathrm{mL}$ 左右时仍有未分解物质或色泽变深，取下冷却，补加硝酸 $5\mathrm{mL}\sim$ $10~\mathrm{mL}$ ，再消解至 $10~\mathrm{mL}$ 左右观察，如此反复两三次，注意避免炭化。如仍不能消解完全，则加入高氯酸 $1\mathrm{mL}\sim 2\mathrm{mL}$ ，继续加热至消解完全后，再持续加热至高氯酸的白烟散尽，硫酸的白烟开始冒出。取下冷却，加水 $25~\mathrm{mL}$ ，再加热至产生硫酸白烟。取下冷却，用水将消化液转入 $25~\mathrm{mL}$ 容量瓶或比色管中，加入硫脲溶液 $(50~\mathrm{g / L})2.5~\mathrm{mL}$ ，最后用水定容至刻度并混匀，备测。同时做空白试验。  
12.1.2 干灰化法：称取 $1\mathrm{g}\sim 2.5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 于 $50~\mathrm{mL}$ 或 $100~\mathrm{mL}$ 坩埚中。加 $150~\mathrm{g / L}$ 硝酸镁溶液 $10~\mathrm{mL}$ 混匀，低温加热蒸干，将氧化镁 $1\mathrm{g}$ 仔细覆盖在干渣上，于电炉上炭化至无黑烟，移入 $550^{\circ}C$ 高温炉灼烧至灰化完全。取出放冷，小心加入盐酸溶液 $(1 + 1)10~\mathrm{mL}$ 以中和氧化镁并溶解灰分，转入 $25~\mathrm{mL}$ 容量瓶或比色管，向容量瓶或比色管中加入 $50~\mathrm{g / L}$ 硫脲 $2.5~\mathrm{mL}$ ，另用硫酸溶液 $(1 + 9)$ 分次涮洗坩埚后转出合并，定容至 $25~\mathrm{mL}$ ，混匀备测。同时做两份空白试验。

# 12.2 标准系列溶液制备

在 $25~\mathrm{mL}$ 容量瓶中依次准确加入 $1\mu \mathrm{g / mL}$ 砷标准使用液 $0\mathrm{mL},0.05\mathrm{mL},0.20\mathrm{mL},0.50\mathrm{mL}$ 、 $1.00~\mathrm{mL},2.00~\mathrm{mL}$ ，硫酸溶液 $(1 + 9)12.5~\mathrm{mL},50~\mathrm{g / L}$ 硫脲 $2.5~\mathrm{mL}$ ，加水定容至刻度，分别相当于砷浓度 $0\mathrm{ng / mL},2\mathrm{ng / mL},8\mathrm{ng / mL},20\mathrm{ng / mL},40\mathrm{ng / mL},80\mathrm{ng / mL}$ ，混匀备测。

# 12.3 测定

12.3.1 仪器参考条件(可根据不同的仪器自行设定测量条件)如下：

负高压： $400\mathrm{V}$   
——砷空心阴极灯电流： $60\mathrm{mA}$   
——原子化器：温度 $800^{\circ}\mathrm{C} \sim 850^{\circ}\mathrm{C}$   
高度 $8\mathrm{mm}$   
——氩气流速：载气 $400~\mathrm{mL / min}$   
—屏蔽气流速： $1000~\mathrm{mL / min}$   
——测量方式：标准曲线法；  
——读数方式：峰面积；  
——读数延迟时间：1s；  
——读数时间：15s；  
加液时间：5s；  
标液或样液加入体积： $2\mathrm{mL}$

12.3.2 设定好仪器最佳条件，逐步将炉温升至所需温度后，稳定 $10\mathrm{min}\sim 20\mathrm{min}$ 后开始测量。连续用标准系列的零管进样，待读数稳定之后，转入标准系列测量，绘制标准曲线。转入试样测量，分别测定试样空白和试样消化液，每测不同的试样前都应清洗进样器。

# 13 分析结果的表述

试样中砷含量按式(2)计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中砷的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样消化液测定浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{0}$ ——试样空白消化液测定浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样消化液总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克(g)；

1000——换算系数。

计算结果保留两位有效数字。

# 14 精密度

湿法消解在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

干灰化法在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

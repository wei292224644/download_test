# 动物性食品中林可霉素、克林霉素和大观霉素多残留的测定

# 气相色谱—质谱法

# 1.范围

本标准规定了动物性食品中林可霉素、克林霉素和大观霉素残留量检测的制样和气相色谱-质谱测定方法。

本标准适用于猪、牛和鸡的肌肉和肾脏以及牛奶和鸡蛋中林可霉素、克林霉素和大观霉素单个或多个药物残留量的检测。

# 3原理

试料中残留的林可霉素、克林霉素和大观霉素，用磷酸缓冲液提取， $3\%$ 三氯乙酸沉淀蛋白，离子对原理反相SPE柱净化，N,O-双（三甲基硅烷基）三氟乙酰胺衍生，气相色谱-质谱法测定，外标法定量。

# 4 试剂和材料

以下所用的试剂，除特别注明外均为分析纯试剂，水为符合GB/T6682规定的一级水。

4.1盐酸林可霉素标准品：含量 $\geq 99\%$ ；盐酸克林霉素和盐酸大观霉素标准品：含量 $\geq 95\%$ 。  
4.2 N,O-双（三甲基硅烷基）三氟乙酰胺   
4.3 乙酸：优级纯。  
4.4 甲醇：色谱纯。  
4.5 乙睛：色谱纯。  
4.6 正己烷：色谱纯。  
4.7 磷酸二氢钾：优级纯。  
4.8 磷酸：优级纯。

4.9 氢氧化钾：优级纯。  
4.10 三氯乙酸：优级纯。  
4.11 十二烷基磺酸钠：色谱纯。  
4.12 HLB固相萃取柱： $60\mathrm{mg / 3mL}$ ，或相当者。  
4.13 提取液：取磷酸二氢钾 $1.36 \mathrm{~g}$ ，用水溶解并稀释至 $1000 \mathrm{~mL}$ ，用磷酸调节 $\mathrm{pH}$ 至 4.0。  
4.14 $3\%$ 三氯乙酸溶液：取三氯乙酸 $3.0\mathrm{g}$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。  
4.15 $2\mathrm{mol} / \mathrm{L}$ 氢氧化钾溶液：取氢氧化钾 $11.6\mathrm{g}$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。  
4.16 十二烷基磺酸钠溶液：取十二烷基磺酸钠 $2.78\mathrm{g}$ ，用水溶解并稀释至 $50~\mathrm{mL}$ 。  
4.17 十二烷基磺酸钠缓冲液：取十二烷基磺酸钠 $2.78\mathrm{g}$ ，用水溶解，加乙酸 $1\mathrm{mL}$ ，用水稀释至 $500\mathrm{mL}$ 。  
4.18 $1 \mathrm{mg} / \mathrm{mL}$ 林可霉素、克林霉素和大观霉素标准贮备液：精密称取林可霉素、克林霉素和大观霉素标准品各 $10 \mathrm{mg}$ ，分别于 $10 \mathrm{~mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的林可霉素、克林霉素和大观霉素标准贮备液。 $2 \sim 8^{\circ} \mathrm{C}$ 避光保存，有效期6个月。  
4.19 $100 \mu \mathrm{g} / \mathrm{mL}$ 标准工作液: 精密量取 $1 \mathrm{mg} / \mathrm{mL}$ 林可霉素、克林霉素和大观霉素标准贮备液各 $1.0 \mathrm{~mL}$ ,分别于 $10 \mathrm{~mL}$ 量瓶中, 用甲醇稀释至刻度, 配制成浓度为 $100 \mu \mathrm{g} / \mathrm{mL}$ 的林可霉素、克林霉素和大观霉素标准工作液。 $2 \sim 8^{\circ} \mathrm{C}$ 避光保存, 有效期 1 周。

5.1 气相色谱-质谱联用仪：配EI源。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 冷冻高速离心机  
5.5 电热恒温水浴锅  
5.6 旋涡混合器  
5.7 均质机  
5.8 聚四氟乙烯离心管： $50~\mathrm{mL}$ 。

# 6 试料的制备与保存

# 6.1 试料的制备

# 6.1.1 猪、牛和鸡的肌肉、肝脏和肾脏组织

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.1.2 牛奶和鸡蛋

取适量新鲜或冷藏的空白或供试牛奶和鸡蛋，混合，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

组织试料- $20^{\circ}\mathrm{C}$ 以下保存。鸡蛋和牛奶试料 $2\sim 8^{\circ}\mathrm{C}$ 以下保存。

# 7 测定步骤

# 7.1 基质匹配标准曲线的制备

分别精密量取 $100 \mu \mathrm{g} / \mathrm{mL}$ 林可霉素、克林霉素和大观霉素标准工作液适量，用甲醇稀释，配制成浓度为 50、100、200、400、800、1600 和 $3200 \mu \mathrm{g} / \mathrm{L}$ 的系列混合标准溶液，各取 $1.0 \mathrm{~mL}$ ，分别溶解经提取、净化及吹干的空白试料残余物，充分混匀，室温吹干，加 $\mathrm{N}_2\mathrm{O}$ -双（三甲基硅烷基）三氟乙酰胺 $500 \mu \mathrm{L}$ 和乙腈 $100 \mu \mathrm{L}$ ，涡旋混合 $1 \mathrm{~min}$ ，密封，于 $75^{\circ} \mathrm{C}$ 恒温箱衍生反应 $1 \mathrm{~h}$ ，于室温氮气吹干，加正己烷 $0.5 \mathrm{~mL}$ 溶解残余物，涡旋混匀，供气相色谱-质谱测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.2 提取

# 7.2.1 肌肉和肾脏组织

称取试料（ $5 \pm 0.05$ ）g，于 $50 \mathrm{~mL}$ 聚四氟乙烯离心管中，加提取液 $10 \mathrm{~mL}$ ，振荡 $5 \mathrm{~min}$ ， $5000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取上清液于另一离心管中，残渣中加提取液 $10 \mathrm{~mL}$ ，重复提取一次，合并两次上清液，加 $3 \%$ 三氯乙酸溶液 $5 \mathrm{~mL}$ ，混匀， $8000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取上清液，用 $2 \mathrm{~mol} / \mathrm{L}$ 氢氧化钾溶液调 $\mathrm{pH}$ 至 $5.8 \pm 0.2$ ，加十二烷基磺酸钠溶液 $2 \mathrm{~mL}$ ，漩涡混匀，静置 $15 \mathrm{~min}$ ，备用。

# 7.2.2 牛奶和鸡蛋组织

称取试料（ $5 \pm 0.05$ ）g，于 $50 \mathrm{~mL}$ 聚四氟乙烯离心管中，加提取液 $10 \mathrm{~mL}$ ，振荡 $5 \mathrm{~min}$ ， $5000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取上清液于另一离心管中，残渣中加提取液 $10 \mathrm{~mL}$ ，重复提取一次，合并两次上清液，加正己烷 $5 \mathrm{~mL}$ ，混匀， $5000 \mathrm{r} / \mathrm{min}$ 离心 $15 \mathrm{~min}$ ，取下层液，用 $2 \mathrm{~mol} / \mathrm{L}$ 氢氧化钾溶液调 $\mathrm{pH}$ 至 $5.8 \pm 0.2$ ，加十二烷基磺酸钠溶液 $2 \mathrm{~mL}$ ，漩涡混匀，静置 $15 \mathrm{~min}$ ，备用。

# 7.3 净化

HLB 柱依次用甲醇 $3 \mathrm{~mL}$ 、水 $3 \mathrm{~mL}$ 和十二烷基磺酸钠缓冲液 $3 \mathrm{~mL}$ 活化，取备用液过柱，用水淋洗两次，每次 $3 \mathrm{~mL}$ ，用甲醇 $4 \mathrm{~mL}$ 洗脱，收集洗脱液，于 $40^{\circ} \mathrm{C}$ 水浴氮气吹至近干，室温吹干，加 $\mathrm{N}_{2} \mathrm{O}-$ 双（三甲基硅烷基）三氟乙酰胺 $500 \mu \mathrm{L}$ 和乙腈 $100 \mu \mathrm{L}$ ，涡旋混合 $1 \mathrm{~min}$ ，密封，于 $75^{\circ} \mathrm{C}$ 恒温箱衍生反

应 $1 \mathrm{~h}$ , 室温氮气吹干, 用正己烷 $0.5 \mathrm{~mL}$ 溶解残余物, 涡旋混匀, 供气相色谱-质谱测定。

# 8 结果计算与表述

单点校准： $C = \frac{CsA}{As}$

或基质匹配标准曲线校准：由 $As = aCs + b$

求得a和b，则 $C = \frac{A - b}{a}$

试料中林可霉素、克林霉素和大观霉素的残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ），按下式计算：

$$
X = \frac {C V}{m}
$$

式中：

$X$ ——供试试料中相应的林可霉素、克林霉素或大观霉素的残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

$C s$ ——标准溶液中相应的林可霉素、克林霉素或大观霉素浓度, $\mu g / L$ ;

C——供试试料溶液中相应的林可霉素、克林霉素或大观霉素浓度, $\mu \mathrm{g} / \mathrm{L}$ ;

As——标准溶液中相应的林可霉素、克林霉素或大观霉素的衍生物的峰面积；

$A$ ——供试试料溶液中相应的林可霉素、克林霉素或大观霉素的衍生物的峰面积；

$V$ ——溶解残余物体积， $\mathrm{mL}$ ；

$m$ ——供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度、精密度

# 9.1灵敏度

本方法的检测限，林可霉素和克林霉素为 $15\mu \mathrm{g / kg}$ ，大观霉素为 $25~\mu \mathrm{g / kg}$ 。

本方法林可霉素和克林霉素在肌肉、牛奶和鸡蛋中的定量限为 $20 \mu \mathrm{g} / \mathrm{kg}$ , 在肾脏中定量限为 $50 \mu \mathrm{g} / \mathrm{kg}$ ; 大观霉素的定量限为 $50 \mu \mathrm{g} / \mathrm{kg}$ 。

# 9.2准确度

本方法克林霉素在 $20\sim 200~\mu \mathrm{g / kg}$ 、林可霉素在 $20\sim 3000~\mu \mathrm{g / kg}$ 和大观霉素在 $50\sim 4000~\mu \mathrm{g / kg}$ 添加浓度水平上回收率为 $70\% \sim 110\%$ 。

# 9.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

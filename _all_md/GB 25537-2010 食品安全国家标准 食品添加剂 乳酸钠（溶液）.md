# 检验方法

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682—2008中规定的水。分析中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。本试验所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

a)盐酸溶液： $0.1\mathrm{mol} / \mathrm{L}$   
b) 硫酸溶液: $1 \mathrm{~mL}$ 硫酸加 $19 \mathrm{~mL}$ 水配制成。  
c）高锰酸钾试液： $0.1\mathrm{mol / L}$   
d) 吗啡啉溶液： $1 + 4$ 。  
e) 亚硝基铁氰化钠试液： $50\mathrm{g / L}$ 。

# A.2.2 分析步骤

A. 2.2.1 钠盐的鉴别: 取铂丝, 用盐酸溶液湿润后, 蔗取试样, 在无色火焰中燃烧, 火焰显亮黄色。  
A.2.2.2 乳酸盐的鉴别: 取 $5 \mathrm{~mL}$ 试样, 加入 $2 \mathrm{~mL}$ 硫酸溶液混匀, 再加 $2 \mathrm{~mL}$ 高锰酸钾试液, 加热,应有乙醛气体产生。乙醛气体的识别采用等体积的吗啡啉溶液和亚硝基铁氰化钠试液的混合液湿润过的滤纸, 滤纸与气体相接触呈蓝色。

# A.3 总乳酸钠的测定

# A.3.1 试剂和材料

a) 乙酸酐。  
b) 冰乙酸。  
c) 高氯酸标准滴定溶液: $c(\mathrm{HClO}_4) = 0.1 \, \mathrm{mol/L}$ 。  
d) 结晶紫指示液: $5 \mathrm{~g} / \mathrm{L}$ 。

# A.3.2 分析步骤

称取相当于含有约 $300\mathrm{mg}$ 乳酸钠的液体试样(精确至 $0.0001\mathrm{g}$ )，加入 $10\mathrm{mL}$ 乙酸酐和 $50\mathrm{mL}$ 冰乙酸，混合均匀，静置约 $20\mathrm{min}$ ，加1滴结晶紫指示液，用高氯酸标准滴定溶液滴定，滴定至溶液显蓝绿色，30s内不褪色，即为滴定终点，同时做空白试验对滴定结果进行校正。

# A.3.3 结果计算

乳酸钠（ $\mathrm{C_3H_5NaO_3}$ ）的含量按公式（A.1）计算：

$$
X _ {1} = \frac {\left(V _ {1} - V _ {0}\right) \times c \times 0.1121}{m} \times 100 \% \dots \tag{A.1}
$$

式中：

$X_{1}$ ——乳酸钠（ $\mathrm{C_3H_5NaO_3}$ ）的含量， $\%$ ；

$V_{1}$ ——试样滴定消耗高氯酸标准滴定溶液的体积, 单位为毫升 $(\mathrm{mL})$ ;

$V_{0}$ ——空白滴定消耗高氯酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——高氯酸标准滴定溶液的实际浓度，单位为摩尔每升（mol/L）；

0.1121——消耗 $1\mathrm{mL}1\mathrm{mol} / \mathrm{L}$ 高氯酸标准滴定溶液相当于 $0.1121\mathrm{g}$ 乳酸钠（ $\mathrm{C_3H_5NaO_3}$ ）；

$m$ ——试样质量，单位为克（g）。

检测结果以平行测定结果的算术平均值为准。平行测定结果的绝对差值不大于 $0.5\%$

# A.4 $L$ -乳酸钠占总乳酸钠的含量的测定

# A.4.1 仪器和设备

高效液相色谱仪，配紫外检测器，检测波长为 $254 \mathrm{~nm}$ 。

# A.4.2 参考色谱条件

a) 色谱柱：固定相是 $3\mu \mathrm{m}$ 的硅胶涂敷在新型光学活性体 N,N-二辛基-L(D)-丙氨酸表面的手性色谱柱， $4.6\mathrm{mm} \times 50\mathrm{mm}$ ；或其他等同性能的用于分离光学异构体的色谱柱。  
b) 流动相: $0.002 \mathrm{~mol} / \mathrm{L}$ 硫酸铜 $\left(\mathrm{CuSO}_{4} \cdot 5 \mathrm{H}_{2} \mathrm{O}\right)$ 溶液。  
c) 流速: $1.2 \mathrm{~mL} / \mathrm{min}$ 。  
d) 柱温： $40^{\circ}\mathrm{C}$ 。  
e) 进样量： $20\mu \mathrm{L}$ 。  
f) 分离度: $D$ 与 $L$ 型异构体分离度在 1.0 以上。

# A.4.3 分析步骤

# A.4.3.1 试样液的制备

称取 $1\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ ），用流动相稀释后，转移到 $100\mathrm{mL}$ 容量瓶中，加流动相定容至刻度。所得溶液用 $0.45\mu \mathrm{m}$ 滤膜过滤，滤液备用。

# A.4.3.2 测定

采用高效液相色谱仪对试样液进行测定，得到色谱图，记录 $D$ -乳酸钠及 $L$ -乳酸钠的波峰面积 $\mathrm{A}_{\mathrm{D}}$ 和 $\mathrm{A}_{\mathrm{L}}$ ，得出 $L$ -乳酸钠含量的百分比。

# A.4.4 结果计算

$L$ -乳酸钠占总乳酸钠的含量 $X_{2}$ 按公式（A.2）计算：

$$
X _ {2} = \frac {A _ {L}}{A _ {L} + A _ {D}} \times 100 \% \dots \tag{A.2}
$$

式中：

$X_{2}$ —— $L$ 乳酸钠占总乳酸钠的含量， $\%$ ；

$A_{\mathrm{L}}$ —— $L$ 乳酸钠的色谱峰面积；

$A_{\mathrm{D}}$ —— $D$ -乳酸钠的色谱峰面积。

实验结果以平行测定结果的算术平均值为准。平行测定结果的绝对差值不大于 $0.5\%$ 。

# A. 5 pH的测定

取1份试样，加入5份等体积的水，配制成待测溶液（1+5），摇匀后用pH计测定。

# A.6 氯化物的测定

# A.6.1 试剂和材料

a) 氯化物标准溶液: $0.1 \mathrm{mg} / \mathrm{mL}$ 。使用时, 将此标准溶液稀释 10 倍, 使氯化物浓度为 $0.01 \mathrm{mg} / \mathrm{mL}$ 。  
b) 硝酸溶液: $1 + 2$ 。  
c) 硝酸银溶液: $17 \mathrm{~g} / \mathrm{L}$ 。

# A.6.2 分析步骤

称取 $1.00 \mathrm{~g}$ 试样，用水溶解后，转移至 $50 \mathrm{~mL}$ 容量瓶中，加水定容至刻度，摇匀。准确吸取 $5 \mathrm{~mL}$ 此试样液，置于 $50 \mathrm{~mL}$ 比色管中，加水使成约 $35 \mathrm{~mL}$ ，加 $10 \mathrm{~mL}$ 硝酸溶液，再加 $1 \mathrm{~mL}$ 硝酸银溶液，加水稀释至 $50 \mathrm{~mL}$ ，摇匀，在暗处放置 $5 \mathrm{~min}$ ，置于黑色背景上，其浊度不得深于标准管。

标准管的制备：准确吸取 $5\mathrm{mL}$ 稀释10倍后的氯化物标准溶液与试样管同时同样处理。

# A.7 硫酸盐的测定

# A.7.1 试剂和材料

a) 盐酸溶液: $3 \mathrm{~mol} / \mathrm{L}$ 。  
b) 氯化钡溶液: $250 \mathrm{~g} / \mathrm{L}$ 。  
c) 硫酸盐标准溶液: $0.1 \mathrm{mg} / \mathrm{mL}$ 。

# A.7.2 分析步骤

称取 $2.00 \mathrm{~g}$ 试样，置于 $50 \mathrm{~mL}$ 比色管中，加适量水，加盐酸使成中性，继续加水稀释到约 $40 \mathrm{~mL}$ ，加 $2 \mathrm{~mL}$ 盐酸溶液，摇匀，加 $5 \mathrm{~mL}$ 氯化钡溶液，用水稀释至 $50 \mathrm{~mL}$ ，充分摇匀，放置 $10 \mathrm{~min}$ ，置于黑色背景上，与标准管比较，其浊度不得深于标准管。

标准管的制备：准确吸取 $1\mathrm{mL}$ 硫酸盐标准溶液与试样管同时同样处理。

# A.8 柠檬酸盐、草酸盐、磷酸盐或酒石酸盐的测定

# A.8.1 试剂和材料

a) 氨水溶液： $1 + 1$ 。  
b) 盐酸溶液: $1 + 5$ 。  
c) 氯化钙溶液： $75\mathrm{g/L}$ 。称取 $7.5\mathrm{g}$ 氯化钙（ $\mathrm{CaCl}_2 \cdot 2\mathrm{H}_2\mathrm{O}$ ），加水溶解，稀释定容至 $100\mathrm{mL}$ 。

# A.8.2 分析步骤

吸取 $5 \mathrm{~mL}$ 试样，用刚煮沸并冷却后的水稀释至 $50 \mathrm{~mL}$ 。取 $4 \mathrm{~mL}$ 稀释后的样液，用氨水溶液或盐酸溶液调整样液 $\mathrm{pH}$ 为 $7.3 \sim 7.7$ ，加入 $1 \mathrm{~mL}$ 氯化钙溶液后在沸水浴锅中加热 $5 \mathrm{~min}$ ，溶液不得产生浑浊，即为通过检验。

# A.9 还原糖的测定

于 $10\mathrm{mL}$ 热的费林溶液中，加入5滴试样，应无红色沉淀产生。

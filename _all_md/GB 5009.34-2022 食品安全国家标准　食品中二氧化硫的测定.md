# 前言

本标准代替GB5009.34—2016《食品安全国家标准 食品中二氧化硫的测定》。

本标准与GB5009.34—2016相比，主要变化如下：

——修改了原滴定法为酸碱滴定法。

——增加了分光光度法、离子色谱法。

# 食品安全国家标准

# 食品中二氧化硫的测定

# 1 范围

本标准规定了食品中二氧化硫的测定方法。

第一法酸碱滴定法适用于食品中二氧化硫的测定；第二法分光光度法，直接提取法适用于白糖及白糖制品、淀粉及淀粉制品和生湿面制品中二氧化硫的测定，充氮蒸馏提取法适用于葡萄酒及赤砂糖中二氧化硫的测定；第三法离子色谱法适用于食品中二氧化硫的测定。

# 第一法 酸碱滴定法

# 2 原理

采用充氮蒸馏法处理试样，试样酸化后在加热条件下亚硫酸盐等系列物质释放二氧化硫，用过氧化氢溶液吸收蒸馏物，二氧化硫溶于吸收液被氧化生成硫酸，采用氢氧化钠标准溶液滴定，根据氢氧化钠标准溶液消耗量计算试样中二氧化硫的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的三级水。

# 3.1 试剂

3.1.1 过氧化氢 $(\mathrm{H}_2\mathrm{O}_2):30\%$   
3.1.2 无水乙醇 $\mathrm{(C_2H_5OH)}$ 。  
3.1.3 氢氧化钠(NaOH)。  
3.1.4 甲基红 $(\mathrm{C_{15}H_{15}N_3O_2})$ 。  
3.1.5 盐酸 $(\mathrm{HCl})(\rho_{20} = 1.19\mathrm{g / mL})$ 。  
3.1.6 氮气（纯度 $>99.9\%$ ）。

# 3.2 试剂配制

3.2.1 过氧化氢溶液 $(3\%)$ : 量取质量分数为 $30\%$ 的过氧化氢 $100 \mathrm{~mL}$ , 加水稀释至 $1000 \mathrm{~mL}$ 。临用时现配。  
3.2.2 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ：量取盐酸 $(\rho_{20} = 1.19\mathrm{g / mL})50~\mathrm{mL}$ ，缓缓倾入 $50~\mathrm{mL}$ 水中，边加边搅拌。  
3.2.3 甲基红乙醇溶液指示剂 $(2.5\mathrm{g} / \mathrm{L})$ ：称取甲基红指示剂 $0.25\mathrm{g}$ ，溶于 $100~\mathrm{mL}$ 无水乙醇中。

# 3.3 标准溶液配制

3.3.1 氢氧化钠标准溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：按照GB/T601配制并标定，或经国家认证并授予标准物质证

书的标准滴定溶液。

3.3.2 氢氧化钠标准溶液 $(0.01\mathrm{mol} / \mathrm{L})$ ：移取氢氧化钠标准溶液 $(0.1\mathrm{mol} / \mathrm{L})10.0\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加无二氧化碳的水稀释至刻度。

# 4 仪器和设备

4.1 玻璃充氮蒸馏器： $500~\mathrm{mL}$ 或 $1000~\mathrm{mL}$ ，另配电热套、氮气源及气体流量计，或等效的蒸馏设备，装置原理图见附录A。  
4.2 电子天平：感量为 $0.01\mathrm{g}$ 。  
4.3 $10\mathrm{mL}$ 半微量滴定管和 $25~\mathrm{mL}$ 滴定管。  
4.4 粉碎机。  
4.5 组织捣碎机。

# 5 分析步骤

# 5.1 试样前处理

# 5.1.1 液体试样

取啤酒、葡萄酒、果酒、其他发酵酒、配制酒、饮料类试样，采样量应大于 $1\mathrm{L}$ ，对于袋装、瓶装等包装试样需至少采集3个包装（同一批次或号），将所有液体在一个容器中混合均匀后，密闭并标识，供检测用。

# 5.1.2 固体试样

取粮食加工品、固体调味品、饼干、薯类食品、糖果制品（含巧克力及制品）、代用茶、酱腌菜、蔬菜干制品、食用菌制品、其他蔬菜制品、蜜饯、水果干制品、炒货食品及坚果制品（烘炒类、油炸类、其他类）、食糖、干制水产品、熟制动物性水产制品、食用淀粉、淀粉制品、淀粉糖、非发酵性豆制品、蔬菜、水果、海水制品、生干坚果与籽类食品等试样，采样量应大于 $600\mathrm{g}$ ，根据具体产品的不同性质和特点，直接取样，充分混合均匀，或者将可食用的部分，采用粉碎机等合适的粉碎手段进行粉碎，充分混合均匀，贮存于洁净盛样袋内，密闭并标识，供检测用。

# 5.1.3 半流体试样

对于袋装、瓶装等包装试样需至少采集3个包装(同一批次或号)；对于酱、果蔬罐头及其他半流体试样，采样量均应大于 $600\mathrm{g}$ ，采用组织捣碎机捣碎混匀后，贮存于洁净盛样袋内，密闭并标识，供检测用。

# 5.2 试样测定

取固体或半流体试样 $20\mathrm{g}\sim 100\mathrm{g}$ (精确至 $0.01\mathrm{g}$ , 取样量可视含量高低而定); 取液体试样 $20\mathrm{mL}$ $(\mathrm{g})\sim 200\mathrm{mL}(\mathrm{g})$ , 将称量好的试样置于图 A.1 中圆底烧瓶 A 中, 加水 $200\mathrm{mL}\sim 500\mathrm{mL}$ 。安装好装置后, 打开回流冷凝管开关给水 (冷凝水温度 $< 15^{\circ}\mathrm{C}$ ), 将冷凝管的上端 E 口处连接的玻璃导管置于 $100\mathrm{mL}$ 锥形瓶底部。锥形瓶内加入 $3\%$ 过氧化氢溶液 $50\mathrm{mL}$ 作为吸收液 (玻璃导管的末端应在吸收液液面以下)。在吸收液中加入 3 滴 $2.5\mathrm{g/L}$ 甲基红乙醇溶液指示剂, 并用氢氧化钠标准溶液 $(0.01\mathrm{mol/L})$ 滴定至黄色即终点 (如果超过终点, 则应舍弃该吸收溶液)。开通氮气, 调节气体流量计至 $1.0\mathrm{L/min}\sim 2.0\mathrm{L/min}$ ; 打开分液漏斗 C 的活塞, 使 $6\mathrm{mol/L}$ 盐酸溶液 $10\mathrm{mL}$ 快速流入蒸馏瓶, 立刻加

热烧瓶内的溶液至沸，并保持微沸 $1.5\mathrm{h}$ ，停止加热。将吸收液放冷后摇匀，用氢氧化钠标准溶液 $(0.01\mathrm{mol} / \mathrm{L})$ 滴定至黄色且 $20\mathrm{s}$ 不褪，并同时进行空白试验。

# 6 分析结果的表述

试样中二氧化硫的含量按式(1)计算：

$$
X = \frac {\left(V - V _ {0}\right) \times c \times 0 . 0 3 2 \times 1 0 0 0 \times 1 0 0 0}{m} \tag {1}
$$

式中：

$X$ ——试样中二氧化硫含量(以 $\mathrm{SO}_2$ 计)，单位为毫克每千克 $(\mathrm{mg / kg})$ 或毫克每升 $(\mathrm{mg / L})$

$V$ ——试样溶液消耗氢氧化钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白溶液消耗氢氧化钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——氢氧化钠滴定液的摩尔浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

0.032 ——1 mL 氢氧化钠标准溶液 (1 mol/L) 相当的二氧化硫的质量(g), 单位为克每毫摩尔 $(\mathrm{g} / \mathrm{mmol})$ ;

$m$ ——试样的质量或体积，单位为克(g)或毫升(mL)。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 检出限与定量限

当用 $0.01\mathrm{mol} / \mathrm{L}$ 氢氧化钠滴定液时，固体或半流体称样量为 $35\mathrm{g}$ 时，检出限为 $1\mathrm{mg / kg}$ ，定量限为 $10\mathrm{mg / kg}$ ；液体取样量为 $50\mathrm{mL(g)}$ 时，检出限为 $1\mathrm{mg / L(mg / kg)}$ ，定量限为 $6\mathrm{mg / L(mg / kg)}$ 。

# 第二法 分光光度法

# 9 原理

样品直接用甲醛缓冲吸收液浸泡或加酸充氮蒸馏-释放的二氧化硫被甲醛溶液吸收，生成稳定的羟甲基磺酸加成化合物，酸性条件下与盐酸副玫瑰苯胺，生成蓝紫色络合物，该络合物的吸光度值与二氧化硫的浓度成正比。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的三级水。

# 10.1 试剂

10.1.1 氨基磺酸铵 $(\mathrm{H}_6\mathrm{N}_2\mathrm{O}_3\mathrm{S})$ 。  
10.1.2 乙二胺四乙酸二钠 $(\mathrm{C_{10}H_{14}N_2Na_2O_8})$ 。

10.1.3 甲醛 $(\mathrm{CH}_2\mathrm{O})$ ： $36\% \sim 38\%$ ，应不含有聚合物（没有沉淀且溶液不分层）。  
10.1.4 邻苯二甲酸氢钾 $\mathrm{(KHC_8H_4O_4)}$ 。  
10.1.5 $0.2\%$ 盐酸副玫瑰苯胺 $\left(\mathrm{C}_{19} \mathrm{H}_{18} \mathrm{ClN}_{3}\right)$ 溶液。   
10.1.6 冰乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
10.1.7 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。

# 10.2 试剂配制

10.2.1 氢氧化钠溶液 $(1.5\mathrm{mol} / \mathrm{L})$ ：称取 $6.0\mathrm{gNaOH}$ (3.1.3)，溶于水并稀释至 $100~\mathrm{mL}$ 。  
10.2.2 乙二胺四乙酸二钠溶液 $(0.05\mathrm{mol} / \mathrm{L})$ ：称取 $1.86\mathrm{g}$ 乙二胺四乙酸二钠（简称EDTA-2Na），溶于水并稀释至 $100~\mathrm{mL}$ 。  
10.2.3 甲醛缓冲吸收储备液：称取 $2.04\mathrm{g}$ 邻苯二甲酸氢钾，溶于少量水中，加入 $36\% \sim 38\%$ 的甲醛溶液 $5.5\mathrm{mL},0.05\mathrm{mol} / \mathrm{L}$ EDTA-2Na溶液 $20.0~\mathrm{mL}$ ，混匀，加水稀释并定容至 $100~\mathrm{mL}$ ，贮于冰箱中冷藏保存。  
10.2.4 甲醛缓冲吸收液：量取甲醛缓冲吸收储备液适量，用水稀释100倍。临用时现配。  
10.2.5 盐酸副玫瑰苯胺溶液 $(0.5\mathrm{g} / \mathrm{L})$ ：量取 $0.2\%$ 盐酸副玫瑰苯胺溶液 $25.0~\mathrm{mL}$ ，分别加入磷酸30 mL和盐酸(3.1.5) $12~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，摇匀，放置 $24\mathrm{h}$ ，备用（避光密封保存）。  
10.2.6 氨基磺酸铵溶液 $(3\mathrm{g} / \mathrm{L})$ ：称取 $0.30\mathrm{g}$ 氨基磺酸铵 $\mathrm{(H_6N_2O_3S)}$ 溶于水并稀释至 $100~\mathrm{mL}$ 。  
10.2.7 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ：量取盐酸(3.1.5) $50~\mathrm{mL}$ ，缓缓倾入 $50~\mathrm{mL}$ 水中边加边搅拌。

# 10.3 标准品

二氧化硫标准溶液 $(100\mu \mathrm{g / mL})$ ：具有国家认证并授予标准物质证书。

# 10.4 标准溶液配制

二氧化硫标准使用液 $(10\mu \mathrm{g / mL})$ ：准确吸取二氧化硫标准溶液 $(100~\mu \mathrm{g / mL})5.0~\mathrm{mL}$ ，用甲醛缓冲吸收液定容至 $50~\mathrm{mL}$ 。临用时现配。

# 12.2 试样处理

# 12.2.1 直接提取法

称取固体试样约 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )，加甲醛缓冲吸收液 $100~\mathrm{mL}$ ，振荡浸泡 $2\mathrm{h}$ ，过滤，取续滤液，待测。同时做空白试验。

# 12.2.2 充氮蒸馏法

称取固体或半流体试样 $10\mathrm{g}\sim 50\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，取样量可视含量高低而定）；量取液体试样

$50 \mathrm{~mL} \sim 100 \mathrm{~mL}$ , 置于图 A.1 中圆底烧瓶 A 中, 加水 $250 \mathrm{~mL} \sim 300 \mathrm{~mL}$ 。打开回流冷凝管开关给水 (冷凝水温度 $< 15^{\circ} \mathrm{C}$ ), 将冷凝管的上端 E 口处连接的玻璃导管置于 $100 \mathrm{~mL}$ 锥形瓶底部。锥形瓶内加入甲醛缓冲吸收液 $30 \mathrm{~mL}$ 作为吸收液 (玻璃导管的末端应在吸收液液面以下)。开通氮气, 使其流量计调节气体流量至 $1.0 \mathrm{~L} / \mathrm{min} \sim 2.0 \mathrm{~L} / \mathrm{min}$ , 打开分液漏斗 C 的活塞, 使 $6 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液 $10 \mathrm{~mL}$ 快速流入蒸馏瓶, 立刻加热烧瓶内的溶液至沸, 并保持微沸 $1.5 \mathrm{~h}$ , 停止加热。取下吸收瓶, 以少量水冲洗导管尖嘴, 并入吸收瓶中。将瓶内吸收液转入 $100 \mathrm{~mL}$ 容量瓶中, 甲醛缓冲吸收液定容, 待测。

# 12.3 标准曲线的制作

分别准确量取 $0.00\mathrm{mL}, 0.20\mathrm{mL}, 0.50\mathrm{mL}, 1.00\mathrm{mL}, 2.00\mathrm{mL}, 3.00\mathrm{mL}$ 二氧化硫标准使用液（相当于 $0.0\mu \mathrm{g}, 2.0\mu \mathrm{g}, 5.0\mu \mathrm{g}, 10.0\mu \mathrm{g}, 20.0\mu \mathrm{g}, 30.0\mu \mathrm{g}$ 二氧化硫），置于 $25\mathrm{mL}$ 具塞试管中，加入甲醛缓冲吸收液至 $10.00\mathrm{mL}$ ，再依次加入 $3\mathrm{g/L}$ 氨基磺酸铵溶液 $0.5\mathrm{mL}, 1.5\mathrm{mol/L}$ 氢氧化钠溶液 $0.5\mathrm{mL}, 0.5\mathrm{g/L}$ 盐酸副玫瑰苯胺溶液 $1.0\mathrm{mL}$ ，摇匀，放置 $20\mathrm{min}$ 后，用紫外可见分光光度计在波长 $579\mathrm{nm}$ 处测定标准溶液吸光度，并以质量为横坐标，吸光度为纵坐标绘制标准曲线。

# 12.4 试样溶液的测定

根据试样中二氧化硫含量，吸取试样溶液 $0.50\mathrm{mL}\sim 10.00\mathrm{mL}$ ，置于 $25~\mathrm{mL}$ 具塞试管中，按12.3“加入甲醛缓冲吸收液至 $10.00\mathrm{mL}$ …”进行操作，同时做空白试验。

# 13 分析结果的表述

试样中二氧化硫的含量按式(2)计算：

$$
X = \frac {\left(m _ {1} - m _ {0}\right) \times V _ {1} \times 1 0 0 0}{m _ {2} \times V _ {2} \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中二氧化硫含量(以 $\mathrm{SO}_2$ 计)，单位为毫克每千克 $(\mathrm{mg / kg})$ 或毫克每升 $(\mathrm{mg / L})$

$m_{1}$ ——由标准曲线中查得的测定用试液中二氧化硫的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_0$ ——由标准曲线中查得的测定用空白溶液中二氧化硫的质量，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——试样提取液/试样蒸馏液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m_{2}$ ——试样的质量或体积，单位为克(g)或毫升(mL)。

$V_{2}$ ——测定用试样提取液/试样蒸馏液的体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测试结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 检出限与定量限

当固体或半流体称样量为 $10\mathrm{g}$ ，定容体积为 $100~\mathrm{mL}$ ，取样体积为 $10~\mathrm{mL}$ 时，本方法检出限为 $1\mathrm{mg / kg}$ ，定量限为 $6\mathrm{mg / kg}$ ；液体取样量为 $10~\mathrm{mL}$ 时，定容体积为 $100~\mathrm{mL}$ ，取样体积为 $10~\mathrm{mL}$ 时，本方法检出限为 $1\mathrm{mg / L}$ ，定量限为 $6\mathrm{mg / L}$ 。

# 第三法 离子色谱法

# 16 原理

试样中亚硫酸盐系列物质经酸处理转化为二氧化硫，通过充氮-水蒸气蒸馏随水蒸气馏出，经过氧化氢溶液吸收并被氧化生成硫酸，溶液中的硫酸根离子经离子色谱柱分离，电导检测器检测，外标法定量。

# 17 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 17.1 标准品

硫酸根离子标准溶液 $(1000\mu \mathrm{g / mL})$ ，具有国家认证并授予标准物质证书。

# 17.2 材料

微孔滤膜： $0.45\mu \mathrm{m}$

18.2 离子色谱仪：配有电导检测器和抑制器。

# 19.2 试样处理

# 19.2.1 固体或半流体试样

称取固体或半流体试样 $10\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ), 置于图 B.1 中圆底烧瓶 D 中, 加水 $50\mathrm{mL}$ , 振摇, 使分散均匀; 接通氮气保护, 控制其流量为 $1.0\mathrm{L} / \mathrm{min} \sim 2.0\mathrm{L} / \mathrm{min}$ 。接通水蒸气蒸馏瓶 A, 吸收瓶 H (100 mL 纳氏比色管) 中加入 $3\%$ 过氧化氢溶液 (3.2.1) $20\mathrm{mL}$ 作为吸收液, 吸收管下端插入吸收液液面以下。D 瓶中沿瓶壁快速加入 $10\mathrm{mL}$ 盐酸溶液 (6 mol/L) (3.2.2), 迅速密塞, 开始蒸馏, 保持 A 瓶沸腾并调整蒸馏火力, 使吸收管端的馏出液的流出速率约为 $2\mathrm{mL} / \mathrm{min}$ 。蒸馏至瓶 H 中溶液总体积约为 $95\mathrm{mL}$ (时间 $30\mathrm{min} \sim 40\mathrm{min}$ ), 用水洗涤尾接管并将其转移至 $100\mathrm{mL}$ 容量瓶中, 用水定容至刻度, 摇匀, 放置 $1\mathrm{h}$ 后, 以微孔滤膜 $(0.45\mu \mathrm{m})$ 过滤, 即得待测样液。不加试样, 同法做空白试验。

# 19.2.2 液体试样

准确量取 $10\mathrm{mL}\sim 50\mathrm{mL}$ 液体试样或称取 $10\mathrm{g}\sim 50\mathrm{g}$ 液体试样(精确至 $0.01\mathrm{g}$ )，当液体取样不足 $50~\mathrm{mL}$ 时，加水将体积补足至 $50~\mathrm{mL}$ ，其他操作同19.2.1。

# 19.3 仪器参考条件

19.3.1 色谱柱：采用以烷醇季铵为功能基的乙基乙烯基苯-二乙烯基苯超大孔乳聚凝胶型聚合物树脂作为填料的高容量阴离子交换柱(填料直径 $9\mu \mathrm{m},250\mathrm{mm}\times 4\mathrm{mm}$ )或等效柱，保护柱使用相同填料的阴离子交换柱(填料直径 $13\mu \mathrm{m},50\mathrm{mm}\times 4\mathrm{mm}$ )。

19.3.2 柱温： $30^{\circ}C$ 。  
19.3.3 淋洗液： $20\mathrm{mmol / L}$ 氢氧化钾溶液（或等效淋洗液）。  
19.3.4 流速： $1.0~\mathrm{mL / min}$   
19.3.5 抑制器：阴离子抑制器，抑制电流 $50\mathrm{mA}$ （或等效抑制器）。  
19.3.6 检测器：电导检测器，检测池温度 $35^{\circ}C$   
19.3.7 进样体积： $100~\mu \mathrm{L}$

# 19.4 标准曲线的制作

准确吸取硫酸根离子标准溶液 $5.00\mathrm{mL}$ ，置 $50~\mathrm{mL}$ 容量瓶中，加水定容至刻度，从上述溶液中，准确吸取 $0.10\mathrm{mL}, 0.20\mathrm{mL}, 0.50\mathrm{mL}, 1.00\mathrm{mL}, 2.00\mathrm{mL}, 4.00\mathrm{mL}$ ，置 $10~\mathrm{mL}$ 容量瓶中，加水定容至刻度，制成分别含硫酸根离子 $1.0\mu \mathrm{g} / \mathrm{mL}, 2.0\mu \mathrm{g} / \mathrm{mL}, 5.0\mu \mathrm{g} / \mathrm{mL}, 10.0\mu \mathrm{g} / \mathrm{mL}, 20.0\mu \mathrm{g} / \mathrm{mL}, 40.0\mu \mathrm{g} / \mathrm{mL}$ 的溶液。精密吸取硫酸根离子标准使用溶液 $100~\mu \mathrm{L}$ ，从低浓度到高浓度依次进样，得到上述浓度标准溶液的色谱图，以硫酸根离子浓度为横坐标，峰面积或峰高为纵坐标绘制标准曲线，硫酸根离子标准溶液的色谱图参见附录C中图C.1。

# 19.5 试样溶液的测定

在与校准曲线相同色谱条件下注入测试液，测试液以峰面积或峰高定量，根据标准曲线得到待测液中的硫酸根离子浓度。如试样溶液浓度超过标准曲线，稀释进样。

# 20 分析结果的表述

试样中二氧化硫的含量按式(3)计算：

$$
X = \frac {\left(C - C _ {0}\right) \times V \times f}{m} \times \frac {1 0 0 0}{1 0 0 0} \times 0. 6 6 6 9 \tag {3}
$$

式中：

$X$ ——试样中二氧化硫含量(以 $\mathrm{SO}_2$ 计)，单位为毫克每千克 $(\mathrm{mg / kg})$ 或毫克每升 $(\mathrm{mg / L})$ ；

$C$ 测定用试样中硫酸根离子的含量(以 $\mathrm{SO}_4^{2-}$ 计)，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$

$C_0$ ——试剂空白溶液中硫酸根离子的含量(以 $\mathrm{SO}_4^{2-}$ 计)，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样定容的体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样溶液稀释倍数；

$m$ ——试样的质量或体积，单位为克(g)或毫升(mL)；

0.6669 ——硫酸根换算为二氧化硫的系数。

计算结果保留三位有效数字。

# 21 精密度

在重复性条件下获得的两次独立测试结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 22 检出限与定量限

当固体或半流体称样量为 $10\mathrm{g}$ ，定容体积为 $100\mathrm{mL}$ ，本方法检出限为 $2\mathrm{mg/kg}$ ，定量限为 $6\mathrm{mg/kg}$ ；液体取样量为 $10\mathrm{mL}$ 时，定容体积为 $100\mathrm{mL}$ ，本方法检出限为 $2\mathrm{mg/L}$ ，定量限为 $6\mathrm{mg/L}$ 。

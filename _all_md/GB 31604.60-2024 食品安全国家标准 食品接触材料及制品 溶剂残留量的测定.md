# 食品安全国家标准

# 食品接触材料及制品 溶剂残留量的测定

# 1 范围

本标准规定了食品接触用复合材料及制品中环己烷、甲基环己烷、丙酮、乙酸乙酯、甲醇、乙酸异丙酯、丁酮、异丙醇、乙醇、乙酸正丙酯、4-甲基-2-戊酮、正丙醇、乙酸丁酯、异丁醇、正丁醇、乙二醇甲醚醋酸酯、丙二醇甲醚醋酸酯、丙二醇甲醚、丙二醇乙醚、苯、甲苯、乙苯、邻二甲苯、对二甲苯、间二甲苯25种溶剂残留量测定的气相色谱方法、气相色谱-质谱方法。

本标准适用于食品接触用复合材料及制品中溶剂残留量的测定。

# 第一法 气相色谱法

# 2 原理

将食品接触材料及制品试样置于顶空瓶中，加热使待测组分达到液-气平衡，顶空气体经气相色谱柱分离后，采用氢火焰离子化检测器进行检测，外标法定量。

# 3 试剂和材料

# 3.1 试剂

$N,N-$ 二甲基甲酰胺 $\mathrm{(C_3H_7NO)}$ ：色谱纯。

# 3.2 标准品

环己烷、甲基环己烷、丙酮、乙酸乙酯、甲醇、乙酸异丙酯、丁酮、异丙醇、乙醇、乙酸正丙酯、4-甲基-2-戊酮、正丙醇、乙酸丁酯、异丁醇、正丁醇、乙二醇甲醚醋酸酯、丙二醇甲醚醋酸酯、丙二醇甲醚、丙二醇乙醚、苯、甲苯、乙苯、邻二甲苯、对二甲苯、间二甲苯：纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准物质，详细信息见附录A的表A.1。

# 3.3 标准溶液配制

3.3.1 单一标准物质储备液 $(200\mathrm{g / L})$ : 准确称取 $2\mathrm{g}$ (精确至 $0.1\mathrm{mg}$ ) 单一标准品, 用 $N,N-$ 二甲基甲酰胺溶解并定容至 $10\mathrm{mL}$ , 混匀, 将溶液转移至棕色玻璃容器中, 于 $-18^{\circ}C$ 下避光保存, 有效期6个月。  
3.3.2 非苯类溶剂混合标准中间液 $(10\mathrm{g / L})$ ：分别移取 $2.50~\mathrm{mL}$ 各非苯类溶剂单一标准物质储备液（ $200~\mathrm{g / L}$ ）于 $50~\mathrm{mL}$ 容量瓶中，加 $N,N-$ 二甲基甲酰胺定容至刻度，混匀，将溶液转移至棕色玻璃容器中，于 $-18^{\circ}C$ 下避光保存，有效期6个月。  
3.3.3 苯类溶剂混合标准中间液 $(10\mathrm{~g/L})$ : 分别移取 $2.50\mathrm{~mL}$ 各苯类溶剂单一标准物质储备液 $(200\mathrm{~g/L})$ 于 $50\mathrm{~mL}$ 容量瓶中, 加 $N, N-$ 二甲基甲酰胺定容至刻度, 混匀, 将溶液转移至棕色玻璃容器中, 于 $-18^{\circ}\mathrm{C}$ 下避光保存, 有效期6个月。  
3.3.4 混合标准中间液(非苯类溶剂 $5000\mathrm{mg / L}$ 、苯类溶剂 $2500\mathrm{mg / L})$ ：分别移取 $25.00~\mathrm{mL}$ 非苯类

混合标准中间液 $(10\mathrm{~g} / \mathrm{L})$ 、 $12.50\mathrm{~mL}$ 苯类混合标准中间液 $(10\mathrm{~g} / \mathrm{L})$ 于 $50\mathrm{~mL}$ 容量瓶中，加 $N,N-$ 二甲基甲酰胺定容至刻度，混匀，将溶液转移至棕色玻璃容器中。所得混合标准中间液中每种非苯类溶剂质量浓度均为 $5000\mathrm{mg} / \mathrm{L}$ 、苯类溶剂质量浓度均为 $2500\mathrm{mg} / \mathrm{L}$ 。于 $-18^{\circ}C$ 下避光保存，有效期6个月。  
3.3.5 混合标准系列中间液：分别移取 $0.20 \mathrm{~mL}, 0.40 \mathrm{~mL}, 1.0 \mathrm{~mL}, 2.0 \mathrm{~mL}, 5.0 \mathrm{~mL}, 10.0 \mathrm{~mL}$ 混合标准中间液（非苯类溶剂 $5000 \mathrm{mg} / \mathrm{L}$ 、苯类溶剂 $2500 \mathrm{mg} / \mathrm{L}$ ）于6个 $10 \mathrm{~mL}$ 容量瓶中，加 $N, N-$ 二甲基甲酰胺定容至刻度，混匀，得到6个混合标准系列中间液。6个混合标准系列中间液中每种非苯类溶剂质量浓度依次为 $100 \mathrm{mg} / \mathrm{L}, 200 \mathrm{mg} / \mathrm{L}, 500 \mathrm{mg} / \mathrm{L}, 1000 \mathrm{mg} / \mathrm{L}, 2500 \mathrm{mg} / \mathrm{L}, 5000 \mathrm{mg} / \mathrm{L}$ ，每种苯类溶剂质量浓度依次为 $50 \mathrm{mg} / \mathrm{L}, 100 \mathrm{mg} / \mathrm{L}, 250 \mathrm{mg} / \mathrm{L}, 500 \mathrm{mg} / \mathrm{L}, 1250 \mathrm{mg} / \mathrm{L}, 2500 \mathrm{mg} / \mathrm{L}$ ，临用现配。  
3.3.6 混合标准系列工作液：分别用微量进样针准确移取 $10~\mu \mathrm{L}$ 上述6个混合标准系列中间液于6个均装有空白基质的顶空瓶中，迅速封盖。所得待测体系中每种非苯类溶剂的质量分别为 $1\mu \mathrm{g},2\mu \mathrm{g}$ 、 $5\mu \mathrm{g},10\mu \mathrm{g},25\mu \mathrm{g},50\mu \mathrm{g}$ ，每种苯类溶剂的质量分别为 $0.5\mu \mathrm{g},1\mu \mathrm{g},2.5\mu \mathrm{g},5\mu \mathrm{g},12.5\mu \mathrm{g},25\mu \mathrm{g}$ 。临用现配。

# 4 仪器和设备

4.1 气相色谱仪：配备顶空进样器、氢火焰离子化检测器。   
4.2 天平：感量为 $0.1\mathrm{mg}$   
4.3 真空干燥箱：极限压力 $1\mathrm{kPa}$   
4.4 顶空瓶： $20~\mathrm{mL}$ ，可在 $125^{\circ}\mathrm{C}$ 烘干 $4\mathrm{h}$ 以上，以确保无干扰目标溶剂测定的物质。  
4.5 移液器：量程分别为 $10\mathrm{mL}, 5\mathrm{mL}, 1\mathrm{mL}, 200\mu \mathrm{L}$ 。  
4.6 微量进样针： $10~\mu \mathrm{L}$

# 5 分析步骤

# 5.1 样品保存

用铝箔紧紧包裹装入无目标溶剂的密闭容器中，在实验室保存时，置于 $-18^{\circ}\mathrm{C}$ 冰箱保存。

注：样品中溶剂残留量与样品运输、保存等环境条件有关，本标准仅规定了实验室接收后的保存条件。

# 5.2 试样制备

# 5.2.1 一般要求

样品制备应迅速。若样品有印刷图案，应在印刷画面最大且色序最多的位置取样。若取样位置对检测结果有影响，结果报出时，报出最大的检测结果，并注明取样位置。卷膜样品，在取样前除去最外部的6层～8层；多层/多个样品层叠存放时，取中间位置的样品。

# 5.2.2 复合软包装材料及制品

裁取试样 $100\mathrm{cm}^2$ （以单面计，精确到 $0.01\mathrm{cm}^2$ ），迅速卷曲后确保样品高度不超过 $5\mathrm{cm}$ ，放入顶空瓶中，加入 $10~\mu \mathrm{L}~N,N-$ 二甲基甲酰胺，封盖待测。当试样中目标溶剂残留量超出标准曲线范围时，可适当减少取样面积。

# 5.2.3 其他复合包装材料及制品

裁取试样 $100\mathrm{cm}^2$ （以单面计，精确到 $0.01\mathrm{cm}^2$ )，迅速将试样剪成 $1\mathrm{cm}\times 3\mathrm{cm}$ 左右的长条，尽快置于顶空瓶中，加入 $10~\mu \mathrm{L}~N,N-$ 二甲基甲酰胺，封盖待测。当试样中目标溶剂残留量超出标准曲线范围

或试样所占体积超过顶空瓶体积的2/3时，可适当减少取样面积。

# 5.2.4 空白基质的制备

除不加 $N,N$ -二甲基甲酰胺以外，按 $5.2.1\sim 5.2.3$ 制备试样，平行制样至少7份，分别置于敞口顶空瓶中，放入真空干燥箱中， $90~^\circ \mathrm{C}$ 真空干燥 $24\mathrm{h}$ 。若第一次真空干燥后空白基质中目标溶剂仍有检出，进行第二次真空干燥，直至空白基质中无目标溶剂检出。空白基质保存于干燥器中备用。

# 5.3 空白试验

取5.2.4制备的空白基质，加入 $10\mu \mathrm{LN},N$ -二甲基甲酰胺，封盖待测。

# 5.4 仪器参考条件

# 5.4.1 顶空进样器参考条件

顶空进样器参考条件如下：

a）平衡温度： $80^{\circ}\mathrm{C}$   
b）进样环温度： $90^{\circ}\mathrm{C}$   
c）传输线温度： $100\%$   
d）顶空平衡时间： $30\mathrm{min}$   
e）环平衡时间： $0.05\mathrm{min}$   
f）加压时间： $0.2\mathrm{min}$   
g）进样时间： $0.5\mathrm{min}$

# 5.4.2 气相色谱参考条件

气相色谱参考条件如下：

a）色谱柱：聚乙二醇毛细管柱，长度 $50\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ ，膜厚 $0.2\mu \mathrm{m}$ 或性能相当者；  
b）升温程序：初始温度 $35\%$ 保持 $10\mathrm{min}$ ，以 $3\% / \mathrm{min}$ 的升温速率升温至 $100\%$ ，再以 $10\% / \mathrm{min}$ 的速率升温至 $130\%$ 保持 $5\mathrm{min}$ ，后运行 $2\mathrm{min}$ ；  
c）进样口温度： $230^{\circ}C$   
d）分流比： $2:1$   
e) 隔垫吹扫： $10~\mathrm{mL / min}$   
f）检测器温度： $280^{\circ}C$   
g）氢气、空气、尾吹气流量分别为： $30~\mathrm{mL / min},300~\mathrm{mL / min},25~\mathrm{mL / min}$   
h）流速： $0.5 \mathrm{~mL} / \mathrm{min}$ 保持 $10 \mathrm{~min}; 0.5 \mathrm{~min}$ 匀速升至 $1 \mathrm{~mL} / \mathrm{min}$ ，保持 $1 \mathrm{~mL} / \mathrm{min}$ 至程序升温结束。

# 5.5 标准曲线的绘制

按照5.4的仪器参考条件，对混合标准系列工作液进行测定，分别以混合标准工作液中各目标溶剂的质量为横坐标，以对应的峰面积为纵坐标，绘制各目标溶剂的标准曲线，得到线性方程。混合标准工作液的色谱图参见附录B的图B.1。

# 5.6 试样溶液的测定

按照5.4的仪器参考条件，对试样(5.2)、空白试验待测样(5.3)进行测定，由标准曲线计算各目标溶剂的质量。

# 6 分析结果的表述

食品接触材料及制品中每种溶剂残留量按式(1)计算。

$$
X = \left(\frac {m}{S} - \frac {m _ {0}}{S _ {0}}\right) \times 1 0 \tag {1}
$$

式中：

$X$ ——食品接触材料及制品中某一溶剂残留量，单位为毫克每平方米 $(\mathrm{mg} / \mathrm{m}^2)$ ；

$m$ ——试样中该溶剂残留的质量，单位为微克 $(\mu \mathrm{g})$ ；

S ——试样取样面积，单位为平方厘米 $(\mathrm{cm}^2)$ ；

$m_0$ ——空白试验中该溶剂的含量，单位为微克 $(\mu \mathrm{g})$ ；

$S_{0}$ ——空白试验中试样的取样面积，单位为平方厘米 $(\mathrm{cm}^2)$ ；

10 ——换算系数。

计算结果保留2位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当取样面积为 $100\mathrm{cm}^2$ （以单面计)时，本方法对每种非苯类溶剂的检出限均为 $0.05\mathrm{mg / m^2}$ ，定量限均为 $0.1\mathrm{mg / m^2}$ 。每种苯类溶剂的检出限均为 $0.025\mathrm{mg / m^2}$ ，定量限均为 $0.05\mathrm{mg / m^2}$ 。

当取样面积为 $100\mathrm{cm}^2$ （以单面计)时，本方法对每种非苯类溶剂的检出限均为 $0.05\mathrm{mg / m^2}$ ，定量限均为 $0.1\mathrm{mg / m^2}$ 。每种苯类溶剂的检出限均为 $0.005\mathrm{mg / m^2}$ ，定量限均为 $0.01\mathrm{mg / m^2}$ 。

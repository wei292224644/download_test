# 食品安全国家标准

# 动物性水产品及其制品中华支睾吸虫的检验

# 1 范围

本标准规定了动物性水产品及其制品中华支睾吸虫(Clonorchis sinensis)囊蚴的形态学和PCR检验方法。

本标准适用于动物性水产品及其制品中华支睾吸虫囊蚴的检验。

# 2 原理

动物性水产品及其制品中的华支睾吸虫囊为主要寄生于淡水鱼和淡水虾的肌肉组织。用压片法观察囊或使用胃蛋白酶消化动物性水产品及其制品的肌肉组织获得囊。根据囊的形态特征，初步判断囊的种类。通过扩增华支睾吸虫核糖体DNA第二内转录间隔区(ITS2)基因片段并测序，进行华支睾吸虫囊的鉴定。

# 3 仪器和设备

3.1 生物显微镜： $100\times \sim 400\times$   
3.2 体视显微镜： $7.5\times \sim 150\times$   
3.3 PCR扩增仪。  
3.4 凝胶成像系统。  
3.5 电泳仪。  
3.6 恒温培养箱： $37^{\circ}C\pm 1^{\circ}C$   
3.7 高速离心机：转速 $\geqslant 12000\mathrm{r / min}$   
3.8 网筛：孔径 $0.25\mathrm{mm}$ (60目)。  
3.9 锥形量杯： $1000~\mathrm{mL}$   
3.10 微量移液器： $0.2\mu \mathrm{L}\sim 2.5\mu \mathrm{L},1\mu \mathrm{L}\sim 10\mu \mathrm{L},10\mu \mathrm{L}\sim 100\mu \mathrm{L},100\mu \mathrm{L}\sim 1000\mu \mathrm{L}$

# 4 试剂和材料

# 4.1 试剂

4.1.1 盐酸： $36\% \sim 38\% \mathrm{HCl}$ 溶液。  
4.1.2 生理盐水： $0.85\% \mathrm{NaCl}$ 溶液。  
4.1.3 $1\mathrm{mol} / \mathrm{L}$ Tris-HCl溶液 $(\mathrm{pH}8.0)$   
4.1.4 $0.5\mathrm{mol} / \mathrm{L}$ EDTA溶液 $(\mathrm{pH}8.0)$   
4.1.5 $10\%$ SDS溶液。  
4.1.6 $5\mathrm{mol} / \mathrm{L}$ NaCl溶液。

4.1.7 $3000\mathrm{U / mg}$ 胃蛋白酶。  
4.1.8 $20\mathrm{mg / mL}$ 蛋白酶K。  
4.1.9 苯酚/三氯甲烷/异戊醇 $(25:24:1)$ 。  
4.1.10 $5\mathrm{U} / \mu \mathrm{L}$ 耐热DNA聚合酶。  
4.1.11 $10 \times \mathrm{PCR}$ 缓冲液。  
4.1.12 $25\mathrm{mmol / L}$ $\mathrm{MgCl_2}$ 。  
4.1.13 dNTPs:dATP、dTTP、dCTP、dGTP，每种浓度为 $2.5\mathrm{mmol / L}$   
4.1.14 琼脂糖：电泳级。  
4.1.15 $50 \times \mathrm{TAE}$ 缓冲液：使用前用去离子水稀释成 $1 \times \mathrm{TAE}$ 缓冲液。  
4.1.16 $1\times \mathrm{TE}$ 溶液 $(\mathrm{pH}8.0)$   
4.1.17 $10\mathrm{mg / mL}$ 溴化乙锭（EB）或其他核酸染料。  
4.1.18 $6\times$ 上样缓冲液。  
4.1.19 $100\mathrm{bp}\sim 2000\mathrm{bp}$ DNA分子量标准。  
4.1.20 PCR引物：浓度为 $20\mu \mathrm{mol} / \mathrm{L}$

正向引物 CS1:5'-CGAGGGTCGGCTTATAAAC-3'

反向引物 CS2:5'-GGAAAGTTAAGCACCGACC-3'

扩增华支睾吸虫 ITS2 基因片段长度为 $315\mathrm{bp}$ 。

# 4.2 试剂配制

4.2.1 生理盐水：称取 $8.5\mathrm{gNaCl}$ 溶解于 $900~\mathrm{mL}$ 去离子水，再加去离子水至 $1000~\mathrm{mL}$   
4.2.2 $1 \mathrm{~mol} / \mathrm{L}$ Tris-HCl 溶液: 称取 $121.1 \mathrm{~g}$ Tris(三羟甲基氨基甲烷)溶解于 $800 \mathrm{~mL}$ 去离子水, 用盐酸调节 $\mathrm{pH}$ 至 8.0 , 加去离子水至 $1000 \mathrm{~mL}$ 。  
4.2.3 $0.5 \mathrm{~mol} / \mathrm{L}$ EDTA 溶液: 称取 $186.1 \mathrm{~g} \mathrm{Na}_{2} \mathrm{EDTA} \cdot 2 \mathrm{H}_{2} \mathrm{O}$ (二水合乙二胺四乙酸二钠) 溶解于 $800 \mathrm{~mL}$ 去离子水, 搅拌溶解, 用 $\mathrm{NaOH}$ 调节 $\mathrm{pH}$ 至 8.0, 加去离子水至 $1000 \mathrm{~mL}$ 。  
4.2.4 $5\mathrm{mol} / \mathrm{L}$ NaCl溶液：称取 $292.5\mathrm{gNaCl}$ 溶于 $900~\mathrm{mL}$ 去离子水中，加去离子水至 $1000~\mathrm{mL}$   
4.2.5 胃蛋白酶消化液：取胃蛋白酶 $5\mathrm{g}$ ，溶解于 $900~\mathrm{mL}$ 生理盐水中，加盐酸 $7\mathrm{mL}$ ，混匀，再加生理盐水至 $1000~\mathrm{mL}$ ，临用现配。  
4.2.6 裂解液： $10\%$ SDS溶液 $100~\mathrm{mL}, 1~\mathrm{mol} / \mathrm{L}$ Tris-HCl 溶液 $10~\mathrm{mL}, 0.5~\mathrm{mol} / \mathrm{L}$ EDTA 溶液 $200~\mathrm{mL}, 5~\mathrm{mol} / \mathrm{L}$ NaCl 溶液 $20~\mathrm{mL}$ ，加灭菌去离子水至 $1000~\mathrm{mL}$ 。  
4.2.7 $1.5\%$ 琼脂糖凝胶：琼脂糖 $1.5\mathrm{g}$ ，加入 $1\times \mathrm{TAE}$ 缓冲液至 $100~\mathrm{mL}$ ，加热至完全融化后冷却至 $60^{\circ}C\sim 70^{\circ}C$ ，加入 $10~\mathrm{mg / mL}$ 溴化乙锭 $5\mu \mathrm{L}$ ，混匀，制备凝胶。

# 5 检测方法

# 5.1 形态学方法

# 5.1.1 压片检查法

# 5.1.1.1 样品制备

取动物性水产品或其制品的肌肉组织约 $2\mathrm{g}$ ，放在两张载玻片之间，用力压薄，用棉线扎紧玻片两端。

# 5.1.1.2 镜检

将玻片放在生物显微镜下观察肌肉中的囊。华支睾吸虫囊一般为椭圆形，大小 $(90\mu \mathrm{m}\sim$

$110\mu \mathrm{m})\times (100\mu \mathrm{m}\sim 140\mu \mathrm{m})$ 。囊外有一层动物组织反应所产生的纤维层，囊壁分两层，外壁较厚，内壁较薄，囊内有一卷曲活动的幼虫，可见口吸盘和腹吸盘，腹吸盘下方为一椭圆形的充满黑色颗粒的排泄囊，其模式图见图A.1，其实物图见图A.2。在水产品中检查到具备上述特征的囊，可判定为疑似华支睾吸虫囊。囊用于DNA提取，或保存于一 $20^{\circ}C$ 。

注：未检出疑似华支睾吸虫囊蚴的样品需用胃蛋白酶消化法进行检测。

# 5.1.2 胃蛋白酶消化法

# 5.1.2.1 取样

取动物性水产品或其制品的肌肉组织或鱼体背、腹两侧及尾部等易感部位肌肉，用剪刀剪成小块，或用均质器低速搅碎。

# 5.1.2.2 消化

取5.1.2.1样品 $200\mathrm{g}$ ，按照样品与胃蛋白酶消化液 $1:5$ 的质量体积比充分搅拌混匀，于 $37^{\circ}C \pm 1^{\circ}C$ 恒温培养箱放置 $4\mathrm{h} \sim 16\mathrm{h}$ ，使肌肉完全消化。可根据不同水产品种类以及消化时间对胃蛋白酶浓度和使用量进行调整。

# 5.1.2.3 过滤

消化后的悬液用网筛过滤，并用生理盐水冲洗网筛上的残留物。收集滤液置于锥形量杯内，搅拌后静置沉淀 $20\mathrm{min}\sim 30\mathrm{min}$ 。轻轻倾去上清液，加入适量生理盐水，搅拌后再静置沉淀 $20\mathrm{min}\sim 30\mathrm{min}$ 。用生理盐水洗涤滤液中的沉淀3次 $\sim 5$ 次，直至上清液透明为止，沉淀备用。

# 5.1.2.4 镜检

全部沉淀物分次转移至玻璃平皿，在体视显微镜下去除沉淀中的杂质，分离收集疑似囊蚴，用生物显微镜观察囊蚴形态。具备5.1.1.2形态特征的囊蚴，可判定为疑似华支睾吸虫囊蚴。疑似华支睾吸虫囊蚴用于DNA提取，或保存于 $-20^{\circ}\mathrm{C}$ 。

# 5.2 PCR方法

# 5.2.1 DNA提取

取5.1.1.2或5.1.2.4分离的单个或多个疑似囊蚴放入 $1.5\mathrm{mL}$ 离心管中，加入无菌生理盐水 $50~\mu \mathrm{L}$ ，再加入裂解液 $600~\mu \mathrm{L}$ ，混匀。加入蛋白酶 $\mathrm{K}10~\mu \mathrm{L},55^{\circ}\mathrm{C}$ 振荡消化至虫体被完全消化 $(1\mathrm{h}\sim 2\mathrm{h})$ 。加苯酚/三氯甲烷/异戊醇(25：24：1) $500~\mu \mathrm{L}$ ，振荡混匀， $4^{\circ}\mathrm{C}12000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。吸取上清液加入等体积的三氯甲烷，振荡混匀， $4^{\circ}\mathrm{C}12000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。吸取上清液加入2倍体积一 $20^{\circ}\mathrm{C}$ 预冷的无水乙醇，充分混匀， $4^{\circ}\mathrm{C}12000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。弃上清液，沉淀用 $700~\mu \mathrm{L}~75\%$ 乙醇洗涤， $4^{\circ}\mathrm{C}12000\mathrm{r / min}$ 离心 $3\mathrm{min}$ 。弃上清液，沉淀干燥后用 $50~\mu \mathrm{L}~1\times \mathrm{TE}$ 溶液溶解，立即用于检测或保存于 $-20^{\circ}\mathrm{C}$ 。

注：根据实验室实际情况，可使用经验证的商品化组织DNA提取试剂盒提取DNA。

# 5.2.2 PCR反应体系

在PCR管中依次加入 $10\times \mathrm{PCR}$ 缓冲液 $5.0\mu \mathrm{L},\mathrm{MgCl}_25.0\mu \mathrm{L},$ dNTPs $2.0\mu \mathrm{L}$ 、正向引物和反向引物各 $2.0~\mu \mathrm{L}$ 、耐热DNA聚合酶 $0.5\mu \mathrm{L}$ 、DNA模板 $2.5\mu \mathrm{L}$ ，加灭菌去离子水至总体积 $50~\mu \mathrm{L}$ 。每次试验需设阳性、阴性和空白对照。阳性对照用华支睾吸虫的DNA或者相应基因片段的质粒做模板，阴性对照用不含华支睾吸虫的动物性水产品DNA做模板，空白对照用灭菌去离子水做模板。

# 5.2.3 PCR反应条件

反应程序为： $94^{\circ}\mathrm{C}$ 预变性 $3\mathrm{min}$ ；然后 $94^{\circ}\mathrm{C}$ 变性 $40\mathrm{s}, 62^{\circ}\mathrm{C}$ 退火 $30\mathrm{s}, 72^{\circ}\mathrm{C}$ 延伸 $30\mathrm{s}, 40$ 个循环； $72^{\circ}\mathrm{C}$ 延伸 $10\mathrm{min}; 4^{\circ}\mathrm{C}$ 保存。

# 5.2.4 电泳

取PCR扩增产物 $10\mu \mathrm{L}$ 与 $6\times$ 上样缓冲液 $2\mu \mathrm{L}$ 混合，加样于 $1.5\%$ 琼脂糖凝胶中，其中一孔加入DNA分子量标准。 $1\times$ TAE电泳缓冲液， $5\mathrm{V / cm}$ 恒压电泳 $30\mathrm{min}\sim 40\mathrm{min}$ ，用凝胶成像系统观察和记录结果。

# 5.2.5 PCR结果判定

阳性对照出现预期大小的条带，阴性对照和空白对照没有该条带，待测样品扩增出预期大小的条带，可判定PCR结果为阳性；无扩增条带或未扩增出预期大小的条带均判为PCR结果阴性。

取PCR结果为阳性的PCR产物进行基因双向测序，将测序结果与华支睾吸虫的参考序列(附录B)进行同源性比对。

# 6 结果报告

6.1 形态学方法检出疑似囊蚴、PCR结果为阳性且扩增片段基因序列与华支睾吸虫的参考序列同源性 $\geqslant 99\%$ ，报告检出华支睾吸虫。  
6.2 形态学方法未检出囊蚴、PCR结果为阴性或扩增片段基因序列与华支睾吸虫的参考序列同源性 $< 99\%$ ，报告未检出华支睾吸虫。

# 牛奶中阿维菌素类药物残留检测

# 高效液相色谱法

# 1 范围

本标准规定了牛奶中阿维菌素类药物残留量检测的制样和高效液相色谱测定方法。

本标准适用于牛奶中伊维菌素、阿维菌素、多拉菌素和埃普利诺菌素单个或多个药物残留量的检测。

# 3原理

试料中残留的阿维菌素类药物，用乙腈提取， $\mathrm{C}_{18}$ 柱净化，三氟乙酸酐和N-甲基咪唑衍生化，高效液相色谱-荧光法测定，外标法定量。

# 4 试剂和材料

以下所用试剂，除特别注明外均为分析纯试剂，水为符合GB/T6682规定的一级水。

4.1埃普利诺菌素、阿维菌素、伊维菌素标准品：含量 $\geq 98\%$ ；多拉菌素标准品：含量 $\geq 94.3\%$   
4.2 甲醇：色谱纯。  
4.3 乙腈：色谱纯。  
4.4 三氟乙酸酐  
4.5 三乙胺  
4.6 异辛烷   
4.7 N-甲基咪唑

4.8 $\mathrm{C}_{18}$ 固相萃取柱： $500\mathrm{mg / 6mL}$ ，或相当者。  
4.9 洗涤液：取乙腈 $30~\mathrm{mL}$ 、水 $70~\mathrm{mL}$ 和三乙胺 $20\mu \mathrm{L}$ ，混匀。  
4.10 衍生化试剂A液：取N-甲基咪唑 $1\mathrm{mL}$ 、乙腈 $1\mathrm{mL}$ ，混匀，现配现用。  
4.11 衍生化试剂B液：取三氟乙酸酐 $1\mathrm{mL}$ 、乙腈 $2\mathrm{mL}$ ，混匀，现配现用。  
4.12 $200\mu \mathrm{g} / \mathrm{mL}$ 阿维菌素类药物混合标准贮备液：精密称取埃普利诺菌素、阿维菌素、多拉菌素和伊维菌素标准品各 $10\mathrm{mg}$ 于 $50~\mathrm{mL}$ 量瓶中，用乙腈溶解并稀释至刻度，配制成浓度为 $200\mu \mathrm{g} / \mathrm{mL}$ 的阿维菌素类药物混合标准贮备液。 $2\sim 8^{\circ}C$ 以下保存，有效期6个月。  
4.13 $10\mu \mathrm{g / mL}$ 阿维菌素类药物混合标准工作液：精密量取 $200~\mu \mathrm{g / mL}$ 阿维菌素类药物混合标准贮备液 $0.5\mathrm{mL}$ ，于 $10\mathrm{mL}$ 量瓶中，用乙腈溶解并稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的阿维菌素类药物标准工作液。 $2\sim 8^{\circ}C$ 以下保存，有效期6个月。

# 5 仪器和设备

5.1 高效液相色谱仪：配荧光检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 涡旋混合仪  
5.5 离心机  
5.6 固相萃取装置  
5.7 氮吹仪   
5.8 微孔滤膜： $0.45\mu \mathrm{m}$

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或解冻的空白或供试牛奶，混合均匀。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作，作为空白添加试料。

# 6.2 试料的保存

-20°C以下保存。

# 7 测定步骤

# 7.1 标准曲线的制备

精密量取 $10\mu \mathrm{g / mL}$ 阿维菌素类药物混合标准贮备液适量，用乙腈稀释，配制成浓度为2、5、10、50、100、500和 $1000\mathrm{ng / mL}$ 的系列标准溶液，各取 $1.0\mathrm{mL}$ 于 $5\mathrm{mL}$ 试管中，于 $60^{\circ}C$ 水浴氮气吹干，依次加衍生化试剂A液 $100~\mu \mathrm{L}$ 和衍生化试剂B液 $150~\mu \mathrm{L}$ ，密闭，涡动 $10~s$ ，依次加冰醋酸和三乙胺各 $50~\mu \mathrm{L}$ ，涡动 $10~s$ ，与样品溶液同步密闭反应 $30\mathrm{min}$ ，加 $650~\mu \mathrm{L}$ 甲醇混匀，供高效液相色谱测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.2 提取

称取试料（ $5 \pm 0.05$ ） $\mathrm{g}$ 于 $50 \mathrm{~mL}$ 离心管中，加乙腈 $8 \mathrm{~mL}$ ，涡动 $1 \mathrm{~min}$ ， $4500 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取上清液。残渣中加乙腈 $8 \mathrm{~mL}$ ，重复提取一次，合并两次上清液，加水 $20 \mathrm{~mL}$ 、三乙胺 $50 \mu \mathrm{L}$ ，混匀，备用。

# 7.3 净化

$\mathrm{C}_{18}$ 柱依次用乙腈 $5 \mathrm{~mL}$ 和洗涤液 $5 \mathrm{~mL}$ 活化，取备用液过柱，自然流干，抽干 $5 \mathrm{~min}$ ，加异辛烷 $3 \mathrm{~mL}$ 洗涤，抽干 $5 \mathrm{~min}$ ，乙腈 $5 \mathrm{~mL}$ 洗脱，收集洗脱液于 $10 \mathrm{~mL}$ 试管中，于 $60^{\circ} \mathrm{C}$ 水浴氮气吹干，备用。

# 7.4衍生化

于备用试管中依次加衍生化试剂A液 $100\mu \mathrm{L}$ 、衍生化试剂B液 $150~\mu \mathrm{L}$ ，密闭，涡动 $10~\mathrm{s}$ ，依次加冰醋酸 $50~\mu \mathrm{L}$ 和三乙胺 $50~\mu \mathrm{L}$ ，涡动 $10~\mathrm{s}$ ，于室温密闭反应 $30\mathrm{min}$ ，加甲醇 $650~\mu \mathrm{L}$ 混匀。过滤，供高效液相色谱法测定。

# 7.5 测定

# 7.5.1 液相色谱参考条件

色谱柱：Symmetry $C_{18}$ （ $250\mathrm{mm} \times 4.6\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ），或相当者；

流动相：乙腈+水（90+10，v/v）；

流速：1 mL/min;

柱温： $30^{\circ}C$

激发波长： $365~\mathrm{nm}$

发射波长： $475\mathrm{nm}$

进样量： $20~\mu \mathrm{L}$

# 7.5.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试样溶液中阿维菌素类药物响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和空白添加试样溶液的高效液相色谱图见附录A。

# 7.6 空白试验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算和表述

试料中阿维菌素类药物的残留量 $(\mu \mathrm{g} / \mathrm{kg})$ ：按下式计算

$$
X = \frac {C \times V}{m}
$$

式中：

$X$ —供试试料中相应的阿维菌素类药物的残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

C—试样溶液中相应的阿维菌素类药物的浓度, $\mu \mathrm{g} / \mathrm{mL}$ ;

V—试样液总体积，mL；

$m$ ——供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度和精密度

# 9.1 灵敏度

本方法的检测限为 $1\mu \mathrm{g / kg}$ ，定量限为 $2\mu \mathrm{g / kg}$

# 9.2 准确度

本方法在 $2\sim 100~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

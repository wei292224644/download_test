# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 吸水性状鉴别

称取试样 $0.5\mathrm{g}$ ，置入盛有 $300~\mathrm{mL}$ 冷水 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 的烧杯中，轻轻晃动烧杯震动液面，使胶粒分散溶胀，形成本品特有的筋韧滑腻的絮状蛋清样胶体，滤去水分，加少许十二水硫酸铝钾，经搅拌挤压后应形成棉絮状胶纤维。

# A.2.2 红外光谱鉴别

按照GB/T6040规定的方法，分别称取样品约 $1\mathrm{mg}$ 与溴化钾约 $100\mathrm{mg}$ 在玛瑙研钵中进行研磨直至混合均匀，取少许上述混合物粉末倒入压片模中用压片器压制成透明的薄片(盐窗)，然后放到红外光谱仪上进行红外扫描，扫描范围 $4000~\mathrm{cm}^{-1}\sim 400~\mathrm{cm}^{-1}$ ，得到其红外透射光谱。根据样品的红外光谱图与标准的红外光谱图在 $4000~\mathrm{cm}^{-1}\sim 400~\mathrm{cm}^{-1}$ 范围内的吸收峰的相似程度，对化合物进行定性。红外光谱图见附录B中的图B.1。

# A.3 黏度 $(1.0\%$ 溶液） $(20^{\circ}\mathrm{C})$ 的测定

# A.3.1 仪器

A.3.1.1 黏度计：旋转黏度计。  
A.3.1.2 高脚烧杯： $500~\mathrm{mL}$   
A.3.1.3 恒温水浴：控温精度 $0.2^{\circ}C$

# A.3.2 分析步骤

称取干燥试样 $3\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ），分散于 $300~\mathrm{mL}$ 水中，搅拌溶胀约 $1\textrm{h}$ 后，边加热边搅拌至沸腾，用煮沸的水补足蒸发的水，搅匀，转入烧杯中，盖上表面皿在恒温水浴 $20^{\circ}C\pm 1^{\circ}C$ 放置 $6\mathrm{h}\sim 8\mathrm{h}$ ，按照黏度计操作说明进行黏度测定。

# A.4 $\mathbf{pH}$ 的测定

# A.4.1 仪器和设备

A.4.1.1 pH计：测定范围为 $0\sim 14$ ，最小分度值为0.02。  
A.4.1.2 离心机：最大转速为 $4000\mathrm{r / min}$

# A.4.2 分析步骤

称取 $1\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样置于烧杯中，加入 $100\mathrm{mL}25^{\circ}\mathrm{C}$ 水，振摇 $30\mathrm{min}$ ，以 $3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液用 $\mathrm{pH}$ 计测定。

# A.5 淀粉试验

# A.5.1 试剂和材料

A.5.1.1 碘化钾溶液：称取 $36.0\mathrm{g}$ 碘化钾溶于 $100~\mathrm{mL}$ 水溶液中。

A.5.1.2 碘溶液：称取 $14.0\mathrm{g}$ 碘，溶于碘化钾溶液(A.5.1.1)中，摇匀，加盐酸3滴，用水定容至 $1000~\mathrm{mL}$ 。

# A.5.2 分析步骤

称取试样 $1.0\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）于 $50~\mathrm{mL}$ 烧杯中，加水 $10~\mathrm{mL}$ ，用玻棒搅匀，煮沸后放至室温，加 $0.1\mathrm{mol} / \mathrm{L}$ 碘溶液(A.5.1.2)2滴，搅匀，不显蓝色为通过试验。

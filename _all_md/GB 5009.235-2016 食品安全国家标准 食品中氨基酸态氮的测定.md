# 前言

本标准代替GB/T5009.39—2003《酱油卫生标准的分析方法》中4.2氨基酸态氮，GB/T5009.40—2003《酱卫生标准的分析方法》中4.1氨基酸态氮，SB/T10310—1999《黄豆酱检验方法》中3.3氨基氮。

本标准与GB/T5009.39—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中氨基酸态氮的测定”；  
——扩充了第一法的适用范围；  
——修改了原标准的结构。

# 食品安全国家标准

# 食品中氨基酸态氮的测定

# 1 范围

本标准规定了酱油、酱、黄豆酱中氨基酸态氮的测定方法。

本标准第一法适用于以粮食和其副产品豆饼、麸皮等为原料酿造或配制的酱油，以粮食为原料酿造的酱类，以黄豆、小麦粉为原料酿造的豆酱类食品中氨基酸态氮的测定；第二法适用于以粮食和其副产品豆饼、麸皮等为原料酿造或配制的酱油中氨基酸态氮的测定。

# 第一法 酸度计法

# 2 原理

利用氨基酸的两性作用，加入甲醛以固定氨基的碱性，使羧基显示出酸性，用氢氧化钠标准溶液滴定后定量，以酸度计测定终点。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 甲醛 $(36\% \sim 38\%)$ ：应不含有聚合物（没有沉淀且溶液不分层）。  
3.1.2 氢氧化钠(NaOH)。  
3.1.3 酚酞 $\mathrm{(C_{20}H_{14}O_4)}$ 。  
3.1.4 乙醇 $\mathrm{(CH_3CH_2OH)}$ 。  
3.1.5 邻苯二甲酸氢钾 $\mathrm{(HOOC_6H_4COOH)}$ ：基准物质。

# 3.2 试剂配制

氢氧化钠标准滴定溶液 $[c(\mathrm{NaOH}) = 0.050\mathrm{mol / L}]$ ：经国家认证并授予标准物质证书的标准滴定溶液或配制方法如下：

a）酚酞指示液：称取酚酞 $1\mathrm{g}$ ，溶于 $95\%$ 的乙醇中，用 $95\%$ 乙醇稀释至 $100~\mathrm{mL}$ 。  
b）氢氧化钠溶液[氢氧化钠标准滴定溶液 $c(\mathrm{NaOH}) = 0.05\mathrm{mol / L}]$ ：称取 $110\mathrm{g}$ 氢氧化钠于 $250~\mathrm{mL}$ 的烧杯中，加 $100~\mathrm{mL}$ 的水，振摇使之溶解成饱和溶液，冷却后置于聚乙烯的塑料瓶中，密塞，放置数日，澄清后备用。取上层清液 $2.7~\mathrm{mL}$ ，加适量新煮沸过的冷蒸馏水至 $1000~\mathrm{mL}$ 摇匀。  
c）氢氧化钠标准滴定溶液的标定：准确称取约 $0.36\mathrm{g}$ 在 $105~^\circ \mathrm{C}\sim 110~^\circ \mathrm{C}$ 干燥至恒重的基准邻苯二甲酸氢钾，加 $80~\mathrm{mL}$ 新煮沸过的水，使之尽量溶解，加2滴酚酞指示液（ $10~\mathrm{g / L})$ ，用氢氧化钠溶液滴定至溶液呈微红色， $30~s$ 不褪色。记下耗用氢氧化钠溶液毫升数。同时做空白试验。

d) 计算：氢氧化钠标准滴定溶液的浓度按式(1)计算：

$$
c = \frac {m}{\left(V _ {1} - V _ {2}\right) \times 0 . 2 0 4 2} \quad \dots \tag {1}
$$

式中：

$c$ ——氢氧化钠标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——基准邻苯二甲酸氢钾的质量，单位为克 $(\mathrm{g})$ ；

$V_{1}$ ——氢氧化钠标准溶液的用量体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——空白实验中氢氧化钠标准溶液的用量体积，单位为毫升 $(\mathrm{mL})$ ；

0.2042——与 $1.00\mathrm{mL}$ 氢氧化钠标准滴定溶液 $[c(\mathrm{NaOH}) = 1.000\mathrm{mol / L}]$ 相当的基准邻苯二甲酸氢钾的质量，单位为克（g）。

# 4 仪器和设备

4.1 酸度计（附磁力搅拌器）。  
4.2 $10\mathrm{mL}$ 微量碱式滴定管。  
4.3 分析天平：感量 $0.1\mathrm{mg}$

# 5 分析步骤

# 5.1 酱油试样

称量 $5.0\mathrm{g}$ (或吸取 $5.0\mathrm{mL}$ ) 试样于 $50~\mathrm{mL}$ 的烧杯中, 用水分数次洗入 $100~\mathrm{mL}$ 容量瓶中, 加水至刻度, 混匀后吸取 $20.0~\mathrm{mL}$ 置于 $200~\mathrm{mL}$ 烧杯中, 加 $60~\mathrm{mL}$ 水, 开动磁力搅拌器, 用氢氧化钠标准溶液 $[c(\mathrm{NaOH}) = 0.050\mathrm{mol} / \mathrm{L}]$ 滴定至酸度计指示 $\mathrm{pH}$ 为8.2, 记下消耗氢氧化钠标准滴定溶液的毫升数, 可计算总酸含量。加入 $10.0~\mathrm{mL}$ 甲醛溶液, 混匀。再用氢氧化钠标准滴定溶液继续滴定至 $\mathrm{pH}$ 为9.2, 记下消耗氢氧化钠标准滴定溶液的毫升数。同时取 $80~\mathrm{mL}$ 水, 先用氢氧化钠标准溶液 $[c(\mathrm{NaOH}) = 0.050\mathrm{mol} / \mathrm{L}]$ 调节至 $\mathrm{pH}$ 为8.2, 再加入 $10.0~\mathrm{mL}$ 甲醛溶液, 用氢氧化钠标准滴定溶液滴定至 $\mathrm{pH}$ 为9.2, 做试剂空白试验。

# 5.2 酱及黄豆酱样品

将酱或黄豆酱样品搅拌均匀后，放入研钵中，在 $10\mathrm{min}$ 内迅速研磨至无肉眼可见颗粒，装入磨口瓶中备用。用已知重量的称量瓶称取搅拌均匀的样品 $5.0\mathrm{g}$ ，用 $50~\mathrm{mL}80^{\circ}\mathrm{C}$ 左右的蒸馏水分数次洗入 $100~\mathrm{mL}$ 烧杯中，冷却后，转入 $100~\mathrm{mL}$ 容量瓶中，用少量水分次洗涤烧杯，洗液并入容量瓶中，并加水至刻度，混匀后过滤。吸取滤液 $10.0~\mathrm{mL}$ ，置于 $200~\mathrm{mL}$ 烧杯中，加 $60~\mathrm{mL}$ 水，开动磁力搅拌器，用氢氧化钠标准溶液 $[c(\mathrm{NaOH}) = 0.050\mathrm{mol / L}]$ 滴定至酸度计指示 $\mathrm{pH}$ 为8.2，记下消耗氢氧化钠标准滴定溶液的毫升数，可计算总酸含量。加入 $10.0~\mathrm{mL}$ 甲醛溶液，混匀。再用氢氧化钠标准滴定溶液继续滴定至 $\mathrm{pH}$ 为9.2，记下消耗氢氧化钠标准滴定溶液的毫升数。同时取 $80~\mathrm{mL}$ 水，先用氢氧化钠标准溶液 $[c(\mathrm{NaOH}) = 0.050\mathrm{mol / L}]$ 调节至 $\mathrm{pH}$ 为8.2，再加入 $10.0~\mathrm{mL}$ 甲醛溶液，用氢氧化钠标准滴定溶液滴定至 $\mathrm{pH}$ 为9.2，做试剂空白试验。

# 6 分析结果的表述

试样中氨基酸态氮的含量按式(2)或式(3)进行计算：

$$
X _ {1} = \frac {\left(V _ {1} - V _ {2}\right) \times c \times 0 . 0 1 4}{m \times V _ {3} / V _ {4}} \times 1 0 0 \quad \dots \tag {2}
$$

$$
X _ {2} = \frac {\left(V _ {1} - V _ {2}\right) \times c \times 0 . 0 1 4}{V \times V _ {3} / V _ {4}} \times 1 0 0 \quad \dots \tag {3}
$$

式中：

$X_{1}$ ——试样中氨基酸态氮的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$X_{2}$ ——试样中氨基酸态氮的含量，单位为克每百毫升 $(\mathrm{g} / 100\mathrm{mL})$

$V_{1}$ ——测定用试样稀释液加入甲醛后消耗氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试剂空白实验加入甲醛后消耗氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——氢氧化钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

0.014 与 $1.00 \mathrm{~mL}$ 氢氧化钠标准滴定溶液 $[c(\mathrm{NaOH}) = 1.000 \mathrm{~mol} / \mathrm{L}]$ 相当的氮的质量，单位为克(g)；

$m$ ——称取试样的质量，单位为克 $(\mathrm{g})$ ；

V ——吸取试样的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——试样稀释液的取用量，单位为毫升 $(\mathrm{mL})$

$V_{4}$ ——试样稀释液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

100 ——单位换算系数。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 比色法

# 8 原理

在 $\mathrm{pH}$ 为4.8的乙酸钠-乙酸缓冲液中，氨基酸态氮与乙酰丙酮和甲醛反应生成黄色的3,5-二乙酸-2,6-二甲基-1,4二氢化吡啶氨基酸衍生物。在波长 $400\mathrm{nm}$ 处测定吸光度，与标准系列比较定量。

# 9 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 9.1 试剂

9.1.1 乙酸 $\mathrm{(CH_3COOH)}$ 。  
9.1.2 无水乙酸钠 $\mathrm{(CH_3COONa)}$ 或乙酸钠 $\mathrm{(CH_3COONa\cdot 3H_2O)}$ 。  
9.1.3 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。  
9.1.4 乙酰丙酮 $(\mathrm{C}_5\mathrm{H}_8\mathrm{O}_2)$ 。

# 9.2 试剂配制

9.2.1 乙酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $5.8~\mathrm{mL}$ 冰乙酸，加水稀释至 $100~\mathrm{mL}$ 。  
9.2.2 乙酸钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $41\mathrm{g}$ 无水乙酸钠或 $68\mathrm{g}$ 乙酸钠 $\mathrm{(CH_3COONa\cdot 3H_2O)}$ ，加水溶解后并稀释至 $500~\mathrm{mL}$ 。  
9.2.3 乙酸钠-乙酸缓冲液：量取 $60~\mathrm{mL}$ 乙酸钠溶液 $(1\mathrm{mol} / \mathrm{L})$ 与 $40~\mathrm{mL}$ 乙酸溶液 $(1\mathrm{mol} / \mathrm{L})$ 混合，该溶

液 $\mathrm{pH}$ 为4.8。

9.2.4 显色剂： $15\mathrm{mL}37\%$ 甲醇与 $7.8\mathrm{mL}$ 乙酰丙酮混合，加水稀释至 $100\mathrm{mL}$ ，剧烈振摇混匀（室温下放置稳定3d）。

# 9.3 标准溶液配制

9.3.1 氨氮标准储备溶液 $(1.0\mathrm{mg / mL})$ ：精密称取 $105~^\circ \mathrm{C}$ 干燥 $2\textrm{h}$ 的硫酸铵 $0.4720\mathrm{g}$ 于小烧杯中，加水溶解后移至 $100~\mathrm{mL}$ 容量瓶中，并稀释至刻度，混匀，此溶液每毫升相当于 $1.0\mathrm{mg}$ 氨氮（ $10^{\circ}C$ 下冰箱内贮存稳定1年以上）。  
9.3.2 氨氮标准使用溶液 $(0.1\mathrm{~g} / \mathrm{L})$ : 用移液管精确量取 $10 \mathrm{~mL}$ 氨氮标准储备液 $(1.0 \mathrm{mg} / \mathrm{mL})$ 于 $100 \mathrm{~mL}$ 容量瓶内, 加水稀释至刻度, 混匀, 此溶液每毫升相当于 $100 \mu \mathrm{g}$ 氨氮 $(10^{\circ} \mathrm{C}$ 下冰箱内贮存稳定 1 个月)。

# 10 仪器和设备

10.1 分光光度计。  
10.2 电热恒温水浴锅 $(100^{\circ}\mathrm{C} \pm 0.5^{\circ}\mathrm{C})$ 。  
10.3 $10\mathrm{mL}$ 具塞玻璃比色管。

# 11 分析步骤

# 11.1 试样前处理

称量 $1.00\mathrm{g}$ (或吸取 $1.0~\mathrm{mL}$ ) 试样于 $50~\mathrm{mL}$ 容量瓶中，加水稀释至刻度，混匀。

# 11.2 标准曲线的制作

精密吸取氨氮标准使用溶液 $0\mathrm{mL},0.05\mathrm{mL},0.1\mathrm{mL},0.2\mathrm{mL},0.4\mathrm{mL},0.6\mathrm{mL},0.8\mathrm{mL},1.0\mathrm{mL}$ （相当于 $\mathrm{NH}_3\text{-N} 0\mu \mathrm{g},5.0\mu \mathrm{g},10.0\mu \mathrm{g},20.0\mu \mathrm{g},40.0\mu \mathrm{g},60.0\mu \mathrm{g},80.0\mu \mathrm{g},100.0\mu \mathrm{g})$ 分别于 $10~\mathrm{mL}$ 比色管中。向各比色管分别加入 $4\mathrm{mL}$ 乙酸钠-乙酸缓冲溶液(pH4.8)及 $4\mathrm{mL}$ 显色剂，用水稀释至刻度，混匀。置于 $100^{\circ}\mathrm{C}$ 水浴中加热 $15\mathrm{min}$ ，取出，水浴冷却至室温后，移入 $1\mathrm{cm}$ 比色皿内，以零管为参比，于波长 $400~\mathrm{nm}$ 处测量吸光度，绘制标准曲线或计算线性回归方程。

# 11.3 试样的测定

精密吸取 $2\mathrm{mL}$ 试样稀释溶液于 $10~\mathrm{mL}$ 比色管中。加入 $4\mathrm{mL}$ 乙酸钠-乙酸缓冲溶液 $(\mathrm{pH}4.8)$ 及 $4\mathrm{mL}$ 显色剂，用水稀释至刻度，混匀。置于 $100^{\circ}\mathrm{C}$ 水浴中加热 $15\mathrm{min}$ ，取出，水浴冷却至室温后，移入 $1\mathrm{cm}$ 比色皿内，以零管为参比，于波长 $400\mathrm{nm}$ 处测量吸光度。试样吸光度与标准曲线比较定量或代入线性回归方程，计算试样含量。

# 12 分析结果的表述

试样中氨基酸态氮的含量按式(4)或式(5)进行计算：

$$
X _ {1} = \frac {m}{m _ {1} \times 1 0 0 0 \times 1 0 0 0 \times V _ {1} / V _ {2}} \times 1 0 0 \quad \dots \tag {4}
$$

$$
X _ {2} = \frac {m}{V \times 1 0 0 0 \times 1 0 0 0 \times V _ {1} / V _ {2}} \times 1 0 0 \quad \dots \tag {5}
$$

式中：

$X_{1}$ ——试样中氨基酸态氮的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$X_{2}$ ——试样中氨基酸态氮的含量，单位为克每百毫升 $(\mathrm{g} / 100\mathrm{mL})$

$m$ ——试样测定液中氮的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_{1}$ ——称取试样的质量，单位为克 $(\mathrm{g})$ ；

V 一吸取试样的体积，单位为毫升 $(\mathrm{mL})$

$V_{1}$ ——测定用试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试样前处理中的定容体积，单位为毫升 $(\mathrm{mL})$ ；

100、1000 ——单位换算系数。

# 13 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 14 其他

本方法的检出限为 $0.007\mathrm{~0~mg / 100~g}$ ，定量限为 $0.021\mathrm{~0~mg / 100~g}$ 。

# 检验方法

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 盐酸。

A.2.1.2 硝酸。

A.2.1.3 盐酸溶液： $1\rightarrow 1000$

A.2.1.4 溴饱和溶液。

A.2.1.5 3,5-二羟基甲苯-乙醇溶液： $1\rightarrow 10$

A.2.1.6 硫酸铁铵-盐酸溶液： $1\rightarrow 1000$

A.2.1.7 氧化镁试液。

A.2.1.8 氢氧化钠溶液： $40~\mathrm{g / L}$

A.2.1.9 氢氧化钠溶液： $1\rightarrow 25$

A.2.1.10 氨水溶液： $2 + 3$

A.2.1.11 钼酸铵溶液: 称取三氧化钼粉末 $6.5 \mathrm{~g}$ , 加入氨水溶液 $(14.5 + 14)$ 使之溶解。冷却后, 边搅拌边加入硝酸溶液 $(32 + 40)$ , 放置 $48 \mathrm{~h}$ 后, 用石棉过滤。此溶液不宜长期保存。量取此液 $5 \mathrm{~mL}$ , 加入 $2 \mathrm{~mL}$ 磷酸氢二钠溶液 $(1 \rightarrow 8)$ 时, 立即或稍加热后应有大量黄色沉淀生成, 否则此溶液不能使用。应避光保存。如有沉淀生成, 则使用上层澄清液。

# A.2.2 鉴别方法

# A.2.2.1 最大吸收峰

称取 $0.02\mathrm{g}$ 试样，加 $1000~\mathrm{mL}$ 盐酸溶液溶解制成的溶液，在波长 $277~\mathrm{nm}\sim 281~\mathrm{nm}$ 处有最大紫外吸收峰。

# A.2.2.2 焰色反应

本品在焰色反应中呈黄色。

# A.2.2.3 鉴别

A.2.2.3.1 称取 $0.03\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $100\mathrm{mL}$ 水中。取 $3\mathrm{mL}$ 此液，加 $1\mathrm{mL}$ 盐酸及 $1\mathrm{mL}$ 溴的饱和溶液，在水浴中加热 $30\mathrm{min}$ ，用空气吹去溴后，加入 $0.2\mathrm{mL}3,5-$ 二羟基甲苯-乙醇溶液，再加 $3\mathrm{mL}$ 硫酸铁铵-盐酸溶液，在水浴中加热 $20\mathrm{min}$ 时，显绿色。

A.2.2.3.2 称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $20~\mathrm{mL}$ 水中。取 $5\mathrm{mL}$ 此液，加入 $2\mathrm{mL}$ 氧化镁试液，应不产生沉淀。接着加 $7\mathrm{mL}$ 硝酸，煮沸 $10\mathrm{min}$ 后，加氢氧化钠溶液 $(40\mathrm{g / L})$ 中和后的溶液，加钼酸铵溶

液，加热时，生成黄色沉淀，再加氢氧化钠溶液 $(1\rightarrow 25)$ 或氨水溶液时，沉淀溶解。

# A.3 $5^{\prime}$ -胞苷酸二钠 $(\mathbf{C}_9\mathbf{H}_{12}\mathbf{N}_3\mathbf{Na}_2\mathbf{O}_8\mathbf{P})$ 含量（以干基计）的测定

# A.3.1 试剂和材料

盐酸溶液： $1\rightarrow 1000$

# A.3.2 分析步骤

称取 $0.5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，加盐酸溶液溶解并定容至 $1000\mathrm{mL}$ ，取 $10\mathrm{mL}$ 该液，加盐酸溶液稀释并定容至 $250\mathrm{mL}$ ，作为试样溶液。测定在波长 $280\mathrm{nm}$ 处试样溶液的吸光度 $A_0$ 。

# A.3.3 结果计算

5'-胞苷酸二钠 $\mathrm{(C_9H_{12}N_3Na_2O_8P)}$ (以干基计)含量的质量分数 $w_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {0 . 5 \times 1.446 \times A _ {0}}{m _ {1} \times \left(1 - w _ {2}\right)} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

0.5 ——换算系数；

1.446——换算系数；

$A_{0}$ ——试样溶液的吸光度；

$m_{1}$ ——试样的质量，单位为克(g)；

$w_{2}$ ——试样中水分的质量分数， $\%$ 。

# A.4 澄清度的测定

# A.4.1 试剂和材料

A.4.1.1 硝酸溶液： $1 + 2$

A.4.1.2 糊精溶液： $20\mathrm{g / L}$

A.4.1.3 硝酸银溶液： $20\mathrm{g / L}$

A.4.1.4 浊度标准溶液：含氯(Cl) $0.01\mathrm{mg / mL}$ 。量取 $14.1\mathrm{mL}\pm 0.02\mathrm{mL}c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$ 盐酸标准滴定溶液，置于 $50~\mathrm{mL}$ 容量瓶中，稀释至刻度。量取 $10\mathrm{mL}\pm 0.02\mathrm{mL}$ 溶液于 $1000~\mathrm{mL}$ 容量瓶中，加水稀释至刻度，摇匀。

# A.4.2 分析步骤

称取约 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于比色管中，加水溶解并稀释至 $25\mathrm{mL}$ ，作为试样溶液；取另一只比色管，准确加入 $0.50\mathrm{mL}$ 浊度标准溶液，加水至 $20\mathrm{mL}$ ，加 $1\mathrm{mL}$ 硝酸溶液， $0.2\mathrm{mL}$ 糊精溶液及 $1\mathrm{mL}$ 硝酸银溶液，加水至 $25\mathrm{mL}$ ，摇匀，避光放置 $15\mathrm{min}$ ，作为标准比浊溶液。

在无阳光直射情况下，轴向及侧向观察，试样溶液的浊度不大于标准比浊溶液的浊度，即为通过试验。

# A.5 吸光度比的测定

# A.5.1 试剂和材料

盐酸溶液： $1\rightarrow 1000$

# A.5.2 分析步骤

称取 $0.020\mathrm{g}$ 试样，加盐酸溶液溶解配至 $1000~\mathrm{mL}$ ，测定溶液在波长 $250~\mathrm{nm},260~\mathrm{nm}$ 和 $280~\mathrm{nm}$ 处的吸光度 $A_{1},A_{2}$ 和 $A_{3}$ ，计算 $A_{1} / A_{2}$ 和 $A_{3} / A_{2}$ 。

# A.6 其他核酸分解物的测定

# A.6.1 试剂和材料

正丙醇：氨试液：丙酮混合液 $= 6:5:2$

# A.6.2 仪器和设备

薄层板：在 $110^{\circ}\mathrm{C}$ 干燥 $1\mathrm{h}$ ，以薄层色谱用硅胶(掺入荧光剂)作为载体。

# A.6.3 分析步骤

称取 $100\mathrm{mg}$ 试样，加水溶解并稀释至 $20~\mathrm{mL}$ ，作为试样溶液。量取 $1\mu \mathrm{L}$ 试样溶液，不用对照液，以正丙醇-氨试液-丙酮混合液(6：5：2)作为展开溶剂，进行薄层色谱分析。展开溶剂顶端由原线上升约 $10~\mathrm{cm}$ 高时停止展开，风干后在暗处、紫外线(波长约 $250~\mathrm{nm}$ )下观察，只看出有一个斑点为通过试验。

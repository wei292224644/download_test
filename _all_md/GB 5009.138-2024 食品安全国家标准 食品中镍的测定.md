# 食品安全国家标准食品中镍的测定

2024-02-08发布

2024-08-08 实施

# 前言

本标准代替GB5009.138—2017《食品安全国家标准 食品中镍的测定》。

本标准与GB5009.138—2017相比，主要变化如下：

——增加了第二法 电感耦合等离子体质谱法；

——增加了第三法 电感耦合等离子体发射光谱法；

——修改了标准溶液配制、储存条件和温度、试样称量范围和精密度的描述。

# 食品安全国家标准

# 食品中镍的测定

# 1 范围

本标准规定了食品中镍的石墨炉原子吸收光谱、电感耦合等离子体质谱和电感耦合等离子体发射光谱测定方法。

本标准适用于食品中镍的测定。

# 第一法 石墨炉原子吸收光谱法

# 2 原理

试样消解处理后，经石墨炉原子化，在 $232.0\mathrm{nm}$ 处测定吸光度。在一定浓度范围内，镍的吸光度值与镍含量成正比，采用外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 硝酸 $(\mathrm{HNO}_3)$ 。  
3.1.2 高氯酸 $\mathrm{(HClO_4)}$   
3.1.3 硝酸钯 $\left[\mathrm{Pd}\left(\mathrm{NO}_{3}\right)_{2}\right]$   
3.1.4 磷酸二氢铵 $(\mathrm{NH_4H_2PO_4})$ 。

# 3.2 试剂配制

3.2.1 硝酸溶液 $(5 + 95)$ ：量取硝酸 $50~\mathrm{mL}$ ，缓慢加入到 $950~\mathrm{mL}$ 水中，混匀。  
3.2.2 硝酸溶液 $(1 + 1)$ ：量取 $500~\mathrm{mL}$ 硝酸，缓慢加入到 $500~\mathrm{mL}$ 水中，混匀。  
3.2.3 硝酸溶液 $(1 + 5)$ ：量取 $100~\mathrm{mL}$ 硝酸，缓慢加入到 $500~\mathrm{mL}$ 水中，混匀。  
3.2.4 磷酸二氢铵-硝酸钯溶液：称取 $0.02\mathrm{g}$ 硝酸钯，加少量硝酸溶液 $(1 + 1)$ 溶解后，再加入 $2\mathrm{g}$ 磷酸二氢铵，溶解后用硝酸溶液 $(1 + 1)$ 定容至 $100~\mathrm{mL}$ ，混匀。

# 3.3 标准品

金属镍(Ni，CAS号：7440-02-0)：纯度 $>99.99\%$ ，或经国家认证并授予标准物质证书的镍标准溶液。

# 3.4 标准溶液配制

3.4.1 镍标准储备液 $(1000\mathrm{mg / L})$ ：准确称取 $1\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ 金属镍，加入 $30~\mathrm{mL}$ 硝酸溶液

(1+1), 加热溶解, 移入 $1000 \mathrm{~mL}$ 容量瓶, 加水稀释至刻度, 混匀。此溶液于 $0^{\circ} \mathrm{C} \sim 5^{\circ} \mathrm{C}$ 冷藏, 有效期6个月。  
3.4.2 镍标准中间液 $(10.0\mathrm{mg / L})$ ：准确吸取镍标准储备液 $(1000\mathrm{mg / L})1.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用硝酸溶液 $(5 + 95)$ 定容至刻度，混匀。此溶液于 $0^{\circ}\mathrm{C}\sim 5^{\circ}\mathrm{C}$ 冷藏，有效期1个月。  
3.4.3 镍标准使用液 $(1.00\mathrm{mg / L})$ ：准确吸取镍标准中间液 $(10.0\mathrm{mg / L})10.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用硝酸溶液 $(5 + 95)$ 定容至刻度，混匀。此溶液于 $0^{\circ}\mathrm{C}\sim 5^{\circ}\mathrm{C}$ 冷藏，有效期1个月。  
3.4.4 镍标准系列溶液：分别吸取镍标准使用液 $(1.00\mathrm{mg / L})0\mathrm{mL},0.25\mathrm{mL},0.50\mathrm{mL},1.00\mathrm{mL}$ 、 $2.00~\mathrm{mL}$ 和 $2.50~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加硝酸溶液 $(5 + 95)$ 至刻度，混匀。此镍标准系列溶液的质量浓度分别为 $0\mu \mathrm{g} / \mathrm{L},2.50\mu \mathrm{g} / \mathrm{L},5.0\mu \mathrm{g} / \mathrm{L},10.0\mu \mathrm{g} / \mathrm{L},20.0\mu \mathrm{g} / \mathrm{L}$ 和 $25.0\mu \mathrm{g} / \mathrm{L}$ 。临用现配。

注：可根据仪器的灵敏度、样品中镍的实际含量及不同仪器型号确定标准系列溶液中镍的质量浓度。

# 4 仪器和设备

注：所有玻璃器皿及聚四氟乙烯消解内罐和内盖均需硝酸溶液 $(1 + 5)$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

4.1 原子吸收光谱仪：配石墨炉原子化器，附镍空心阴极灯。  
4.2 分析天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.3 可调式电热炉、可调式电热板。  
4.4 微波消解系统：配聚四氟乙烯消解内罐。  
4.5 压力消解罐：配聚四氟乙烯消解内罐。  
4.6 恒温干燥箱。  
4.7 马弗炉。  
4.8 粉碎机、匀浆机。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固态样品

# 5.1.1.1 干样

对于含水量低的样品，取可食部分，必要时经高速粉碎机粉碎均匀；对于固体乳制品、蛋白粉等呈均匀状的粉状样品，混匀；对于其他固体样品，取可食部分，制备为均匀试样。

# 5.1.1.2 鲜样

蔬菜、水果、水产品等高含水量样品洗净，晾干，取可食部分匀浆均匀；对于肉类、蛋类等样品，取可食部分匀浆均匀。

# 5.1.1.3 速冻及罐头食品

经解冻的速冻食品及罐头样品，取可食部分匀浆均匀。

# 5.1.2 液态样品

摇匀。

# 5.1.3 半固态样品

搅拌均匀。

# 5.2 试样消解

# 5.2.1 湿式消解法

称取固体试样 $0.5\mathrm{g}\sim 2\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.50~\mathrm{mL}\sim 3.00~\mathrm{mL}$ 于带刻度消化管中，含乙醇或二氧化碳的样品先在电热板上低温加热除去乙醇或二氧化碳，加入 $10~\mathrm{mL}$ 硝酸和 $0.5\mathrm{mL}$ 高氯酸，放数粒玻璃珠，在可调式电热炉上消解(参考条件： $120^{\circ}C$ $0.5\mathrm{h}\sim 1\mathrm{h}$ ；升至 $180^{\circ}C$ $2\mathrm{h}\sim$ $4\mathrm{h}$ ，升至 $200^{\circ}C\sim 220^{\circ}C)$ 。若消化液呈棕褐色，再加少量硝酸，消解至冒白烟，消化液呈无色透明或略带黄色，赶酸至 $1\mathrm{mL}$ 左右，停止消解，冷却后用水定容至 $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ ，混匀备用。同时做试剂空白试验。亦可采用锥形瓶，于可调式电热板上，按上述操作方法进行湿法消解。

# 5.2.2 微波消解法

称取固体试样 $0.2\mathrm{g}\sim 0.5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.50~\mathrm{mL}\sim 2.00~\mathrm{mL}$ 于微波消解罐中，含乙醇或二氧化碳的样品先在电热板上低温加热除去乙醇或二氧化碳，加入 $5\mathrm{mL}\sim 10\mathrm{mL}$ 硝酸，按照微波消解的操作步骤消解试样，消解条件参考附录A表A.1。冷却后取出消解罐，在电热板上于 $140^{\circ}C\sim 160^{\circ}C$ 赶酸至 $1\mathrm{mL}$ 左右。消解罐放冷后，将消化液转移至 $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 容量瓶中，用少量水洗涤消解罐2次～3次，合并洗涤液于容量瓶中并用水定容至刻度，混匀备用。同时做试剂空白试验。

# 5.2.3 压力罐消解法

称取固体试样 $0.2\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.50~\mathrm{mL}\sim 5.00~\mathrm{mL}$ 于消解内罐中，含乙醇或二氧化碳的样品先在电热板上低温加热除去乙醇或二氧化碳，加入 $5\mathrm{mL}$ 硝酸。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱，于 $140^{\circ}C\sim 160^{\circ}C$ 下保持 $4\mathrm{h}\sim 5\mathrm{h}$ 。冷却后缓慢旋松外罐，取出消解内罐，放在可调式电热板上于 $140^{\circ}C\sim 160^{\circ}C$ 赶酸至 $1\mathrm{mL}$ 左右。冷却后将消化液转移至 $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 容量瓶中，用少量水洗涤内罐和内盖2次～3次，合并洗涤液于容量瓶中并用水定容至刻度，混匀备用。同时做试剂空白试验。

# 5.2.4 干法灰化

称取固体试样 $0.5\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.50~\mathrm{mL}\sim 10.0~\mathrm{mL}$ 于坩埚中，小火加热，炭化至无烟，转移至马弗炉中，于 $550~\mathrm{^\circ C}$ 灰化 $3\mathrm{h}\sim 4\mathrm{h}$ 。冷却，取出，对于灰化不彻底的试样，加数滴硝酸，小火加热，小心蒸干，再转入 $550~\mathrm{^\circ C}$ 马弗炉中，继续灰化 $1\mathrm{h}\sim 2\mathrm{h}$ ，至试样呈白灰状，冷却，取出，用适量硝酸溶液 $(1 + 1)$ 溶解并用水定容至 $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 。同时做试剂空白试验。

# 5.3 测定

# 5.3.1 仪器参考条件

仪器操作条件参见附录B表B.1。

# 5.3.2 标准曲线的制作

按质量浓度由低到高的顺序分别将 $10\mu \mathrm{L}$ 镍标准系列溶液和 $5\mu \mathrm{L}$ 磷酸二氢铵-硝酸钯溶液（可根

据所使用的仪器确定最佳进样量、最佳基体改进剂)同时注入石墨炉，原子化后测其吸光度值，以质量浓度为横坐标，吸光度值为纵坐标，制作标准曲线。

# 5.3.3 试样溶液的测定

在与测定标准溶液相同的实验条件下，将 $10\mu \mathrm{L}$ 空白溶液或试样溶液（可根据试样溶液浓度进行适当稀释后上机）与 $5\mu \mathrm{L}$ 磷酸二氢铵-硝酸钯溶液（可根据所使用的仪器确定最佳进样量、最佳基体改进剂）同时注入石墨炉，原子化后测其吸光度值，根据标准曲线得到试样溶液中镍元素的浓度。

# 6 分析结果的表述

试样中镍的含量按式(1)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times f \times V}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中镍的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 或毫克每升 $(\mathrm{mg / L})$ ；

$\rho$ ——试样溶液中镍的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$f$ ——稀释因子；

$\rho_0$ ——空白溶液中镍的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样消化液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的称样量或移取体积，单位为克(g)或毫升 $(\mathrm{mL})$ ；

1000——换算系数。

当镍含量 $\geqslant 1.00\mathrm{mg / kg}$ （或 $\mathrm{mg / L}$ )时，计算结果保留3位有效数字；当镍含量 $<  1.00~\mathrm{mg / kg}$ （或 $\mathrm{mg / L})$ 时，计算结果保留2位有效数字。

# 7 精密度

样品中镍含量大于 $1\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ ；小于或等于 $1\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )且大于 $0.1\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ ；小于或等于 $0.1\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当称样量为 $0.5\mathrm{g}$ (或 $0.5\mathrm{mL}$ )、定容体积为 $10\mathrm{mL}$ 时，方法的检出限为 $0.02\mathrm{mg/kg}$ (或 $0.02\mathrm{mg/L}$ )，定量限为 $0.05\mathrm{mg/kg}$ (或 $0.05\mathrm{mg/L}$ )。

# 前言

本标准代替GB/T5009.185—2003《苹果和山楂制品中展青霉素的测定》、NY/T1650—2008《苹果及山楂制品中展青霉素的测定高效液相色谱法》、SN/T2008—2007《进出口果汁中棒曲霉毒素的检测方法高效液相色谱法》和SN/T2534—2010《进出口水果和蔬菜制品中展青霉素含量检测方法液相色谱-质谱/质谱法与高效液相色谱法》和SN/T1859—2007《饮料中棒曲霉素和5-羟甲基糠醛的测定方法液相色谱-质谱法和气相色谱-质谱法》中展青霉素部分。

本标准与GB/T5009.185—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中展青霉素的测定”；  
——增加了同位素稀释-液相色谱-串联质谱法；  
——增加了液相色谱法；  
扩大了适用范围；  
——删除了薄层色谱法。

# 食品安全国家标准

# 食品中展青霉素的测定

# 1 范围

本标准规定了食品中展青霉素的测定方法。

本标准第一法为同位素稀释-液相色谱串联质谱法，适用于苹果和山楂为原料的水果及其制品、果蔬汁类和酒类食品中展青霉素含量的测定。

本标准第二法为高效液相色谱法，适用于苹果为原料的水果及其果蔬汁类和酒类食品中展青霉素含量的测定。

# 第一法 同位素稀释-液相色谱-串联质谱法

# 2 原理

样品(浊汁、半流体及固体样品用果胶酶酶解处理)中的展青霉素经溶剂提取，展青霉素固相净化柱或混合型阴离子交换柱净化、浓缩后，经反相液相色谱柱分离，电喷雾离子源离子化，多反应离子监测检测，内标法定量。

# 3 试剂和材料

除非另有说明，本方法使用的试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.3 乙酸 $\mathrm{(CH_3COOH)}$ ：色谱纯。  
3.1.4 乙酸铵 $\mathrm{(CH_3COONH_4)}$   
3.1.5 果胶酶（液体）：活性 $\geqslant 1500\mathrm{U / g},2^{\circ}C\sim 8^{\circ}\mathrm{C}$ 避光保存。

# 3.2 试剂配制

3.2.1 乙酸溶液：取 $10\mathrm{mL}$ 乙酸加入 $250~\mathrm{mL}$ 水，混匀。  
3.2.2 乙酸铵溶液 $(5\mathrm{mmol / L})$ ：称取 $0.38\mathrm{g}$ 乙酸铵，加 $1000~\mathrm{mL}$ 水溶解。

# 3.3 标准品

3.3.1 展青霉素标准品 $\left(\mathrm{C}_{7} \mathrm{H}_{6} \mathrm{O}_{4}, \mathrm{CAS}\right.$ 号：149-29-1）：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.2 ${}^{13}\mathrm{C}_7$ -展青霉素同位素内标： $25\mu \mathrm{g / mL}$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 标准储备溶液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：用 $2\mathrm{mL}$ 乙腈溶解展青霉素标准品 $1.0\mathrm{mg}$ 后，移入 $10~\mathrm{mL}$ 的容量瓶，乙腈定容至刻度。溶液转移至试剂瓶中后，在 $-20^{\circ}\mathrm{C}$ 下冷冻保存，备用，有效期6个月。展青霉素标准溶液浓度的标定参见附录A。  
3.4.2 标准工作液 $(1\mu \mathrm{g} / \mathrm{mL})$ : 准确吸取 $100~\mu \mathrm{L}$ 经标定过的展青霉素标准储备溶液至 $10~\mathrm{mL}$ 容量瓶中, 用乙酸溶液定容至刻度。溶液转移至试剂瓶中后, 在 $4^{\circ}\mathrm{C}$ 下避光保存, 有效期3个月。  
3.4.3 ${ }^{13} \mathrm{C}_{7}$ -展青霉素同位素内标工作液 $(1 \mu \mathrm{g} / \mathrm{mL})$ : 准确移取展青霉素同位素内标 $(25 \mu \mathrm{g} / \mathrm{mL})$ $0.40 \mathrm{~mL}$ 至 $10 \mathrm{~mL}$ 容量瓶中, 用乙酸溶液定容。在 $4^{\circ} \mathrm{C}$ 下避光保存, 备用, 3 个月内有效。  
3.4.4 标准系列工作溶液：分别准确移取标准工作液适量至 $10\mathrm{mL}$ 容量瓶中，加入 $500~\mu \mathrm{L}1.0~\mu \mathrm{g / mL}$ 的同位素内标工作液，用乙酸溶液定容至刻度，配制展青霉素浓度为 $5\mathrm{ng / mL},10\mathrm{ng / mL},25\mathrm{ng / mL},$ $50~\mathrm{ng / mL},100~\mathrm{ng / mL},150~\mathrm{ng / mL},200~\mathrm{ng / mL},250~\mathrm{ng / mL}$ 系列标准溶液。

# 4 仪器和设备

4.1 液相色谱-质谱联用仪：带电喷雾离子源。  
4.2 匀浆机。  
4.3 高速粉碎机。  
4.4 组织捣碎机。  
4.5 涡旋振荡器。  
4.6 pH计：测量精度 $\pm 0.02$   
4.7 天平：感量为 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$ 。  
4.8 $50~\mathrm{mL}$ 具塞PVC离心管。  
4.9 离心机：转速 $\geq 6000\mathrm{r / min}$   
4.10 展青霉素固相净化柱(以下简称净化柱)：混合填料净化柱MycosepTM228或相当者。  
4.11 混合型阴离子交换柱：N-乙烯吡咯烷酮-二乙烯基苯共聚物基质- $\mathrm{CH}_2\mathrm{N}(\mathrm{CH}_3)_2\mathrm{C}_4\mathrm{H}_9^+$ 为填料的固相萃取柱(6 mL, $150~\mathrm{mg}$ ) 或相当者。使用前分别用 $6\mathrm{mL}$ 甲醇和 $6\mathrm{mL}$ 水预淋洗并保持柱体湿润。  
4.12 $100~\mathrm{mL}$ 梨形烧瓶。  
4.13 固相萃取装置。  
4.14 旋转蒸发仪。  
4.15 氮吹仪。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 液体样品（苹果汁、山楂汁等）

样品倒入匀浆机中混匀，取其中任意的 $100\mathrm{g}$ (或 $\mathrm{mL}$ )样品进行检测。

酒类样品需超声脱气 $1\mathrm{h}$ 或 $4^{\circ}\mathrm{C}$ 低温条件下存放过夜脱气。

# 5.1.2 固体样品（山楂片、果丹皮等）

样品用高速粉碎机将其粉碎，混合均匀后取样品 $100\mathrm{g}$ 用于检测。果丹皮等高黏度样品经液氮冻干后立即用高速粉碎机将其粉碎，混合均匀后取样品 $100\mathrm{g}$ 用于检测。

# 5.1.3 半流体（苹果果泥、苹果果酱、带果粒果汁等）

样品在组织捣碎机中捣碎混匀后，取 $100\mathrm{g}$ 用于检测。

# 5.2 试样提取及净化

# 5.2.1 混合型阴离子交换柱法

# 5.2.1.1 试样提取

# 5.2.1.1.1 澄清果汁

称取 $2\mathrm{g}$ 试样(准确至 $0.01\mathrm{g}$ )，加入 $50~\mu \mathrm{L}$ 同位素内标工作液混匀待净化。

# 5.2.1.1.2 苹果酒

称取 $1\mathrm{g}$ 试样(准确至 $0.01\mathrm{g}$ )，加入 $50~\mu \mathrm{L}$ 同位素内标工作液，加水至 $10\mathrm{mL}$ 混匀后待净化。

# 5.2.1.1.3 固体、半流体试样

称取 $1\mathrm{g}$ 试样(准确至 $0.01\mathrm{g}$ )于 $50~\mathrm{mL}$ 离心管中，加入 $50~\mu \mathrm{L}$ 同位素内标工作液，静置片刻后，再加入 $10~\mathrm{mL}$ 水与 $75~\mu \mathrm{L}$ 果胶酶混匀，室温下避光放置过夜后，加入 $10.0~\mathrm{mL}$ 乙酸乙酯，涡旋混合 $5\mathrm{min}$ ，在 $6000~\mathrm{r / min}$ 下离心 $5\mathrm{min}$ ，移取乙酸乙酯层至 $100~\mathrm{mL}$ 梨形烧瓶。再用 $10.0~\mathrm{mL}$ 乙酸乙酯提取一次，合并两次乙酸乙酯提取液，在 $40^{\circ}\mathrm{C}$ 水浴中用旋转蒸发仪浓缩至干，以 $5.0~\mathrm{mL}$ 乙酸溶液溶解残留物，待净化处理。

# 5.2.1.2 净化

将待净化液转移至预先活化好的混合型阴离子交换柱中，控制样液以约 $3\mathrm{mL / min}$ 的速度稳定过柱。上样完毕后，依次加入 $3\mathrm{mL}$ 的乙酸铵溶液、 $3\mathrm{mL}$ 水淋洗。抽干混合型阴离子交换柱，加入 $4\mathrm{mL}$ 甲醇洗脱，控制流速约 $3\mathrm{mL / min}$ ，收集洗脱液。在洗脱液中加入 $20~\mu \mathrm{L}$ 乙酸，置 $40^{\circ}\mathrm{C}$ 下用氮气缓缓吹至近干，用乙酸溶液定容至 $1.0\mathrm{mL}$ ，涡旋 $30~\mathrm{s}$ 溶解残留物， $0.22~\mu \mathrm{m}$ 滤膜过滤，收集滤液于进样瓶中以备进样。按同一操作方法做空白试验。

# 5.2.2 净化柱法

# 5.2.2.1 试样提取

# 5.2.2.1.1 液体试样

称取 $4\mathrm{g}$ 试样(准确至 $0.01\mathrm{g}$ )于 $50~\mathrm{mL}$ 离心管中，加入 $250~\mu \mathrm{L}$ 同位素内标工作液，加入 $21~\mathrm{mL}$ 乙腈，混合均匀，在 $6000~\mathrm{r / min}$ 下离心 $5\mathrm{min}$ ，待净化。

# 5.2.2.1.2 固体、半流体试样

称取 $1\mathrm{g}$ 试样(准确至 $0.01\mathrm{g}$ )于 $50~\mathrm{mL}$ 离心管中，加入 $100~\mu \mathrm{L}$ 同位素内标工作液，混匀后静置片刻，再加入 $10~\mathrm{mL}$ 水与 $150~\mu \mathrm{L}$ 果胶酶溶液混匀，室温下避光放置过夜后，加入 $10.0~\mathrm{mL}$ 乙酸乙酯，涡旋混合 $5\mathrm{min}$ ，在 $6000~\mathrm{r / min}$ 下离心 $5\mathrm{min}$ ，移取乙酸乙酯层至梨形烧瓶。再用 $10.0~\mathrm{mL}$ 乙酸乙酯提取一次，合并两次乙酸乙酯提取液，在 $40^{\circ}\mathrm{C}$ 水浴中用旋转蒸发仪浓缩至干，以 $2.0~\mathrm{mL}$ 乙酸溶液溶解残留物，再加入 $8\mathrm{mL}$ 乙腈，混匀后待净化。

# 5.2.2.2 净化

按照所使用净化柱的说明书操作，将提取液通过净化柱净化，弃去初始的 $1\mathrm{mL}$ 净化液，收集后续部分。

用吸量管准确吸取 $5.0\mathrm{mL}$ 净化液，加入 $20~\mu \mathrm{L}$ 乙酸，在 $40^{\circ}C$ 下用氮气缓缓地吹至近干，加入乙酸溶液定容至 $1\mathrm{mL}$ ，涡旋 $30\mathrm{s}$ 溶解残渣，过 $0.22\mu \mathrm{m}$ 滤膜，收集滤液于进样瓶中以备进样。按同一操作方法做空白试验。

注：上述方法的样品提取和净化部分，包括混合型阴离子交换柱净化和净化柱净化方法，可根据实际情况，选择其中一种方法即可。

# 5.4 标准曲线的制作

将标准系列工作溶液由低到高浓度进样检测，以标准系列工作溶液中展青霉素的浓度为横坐标，以展青霉素色谱峰与内标色谱峰的峰面积比值为纵坐标，绘制得到标准曲线。

# 5.5 测定

将试样溶液注入液相色谱-质谱仪中，测得相应的峰面积，由标准曲线得到试样溶液中展青霉素的浓度。

# 5.6 定性

试样中目标化合物色谱峰的保留时间与相应标准色谱峰的保留时间相比较，变化范围在 $\pm 2.5\%$ 之内。

每种化合物的质谱定性离子必须出现，至少应包括一个母离子和两个子离子，而且同一检测批次，对同一化合物，样品中目标化合物的两个子离子的相对丰度比与浓度相当的标准溶液相比，其允许偏差不超过表1规定的范围。

# 6 分析结果的表述

试样中展青霉素的含量按式(1)计算：

$$
X = \frac {\rho \times V}{m} \times f \quad \dots \tag {1}
$$

式中：

$X$ ——试样中展青霉素的含量，单位为微克每千克或微克每升 $(\mu \mathrm{g} / \mathrm{kg}$ 或 $\mu \mathrm{g} / \mathrm{L})$ ；

$\rho$ ——由标准曲线计算所得的试样溶液中展青霉素的浓度，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$V$ ——最终定容体积，单位毫升 $(\mathrm{mL})$ ；

$m$ ——试样的称样量，单位克 $(\mathrm{g})$ ；

$f$ ——稀释倍数。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 第二法 高效液相色谱法

# 9 原理

样品(浊汁、半流体及固体样品用果胶酶酶解处理)中的展青霉素经提取，展青霉素固相净化柱净化、浓缩后，液相色谱分离，紫外检测器检测。外标法定量。

# 10 试剂和材料

除非另有说明，本方法使用的试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

10.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
10.1.3 乙酸 $\mathrm{(CH_3COOH)}$ ：色谱纯。  
10.1.4 乙酸乙酯 $\mathrm{(CH_3COOCH_2CH_3)}$   
10.1.5 乙酸铵 $\mathrm{(CH_3COONH_4)}$   
10.1.6 果胶酶（液体）：活性不低于 $1500\mathrm{U / g},2^{\circ}\mathrm{C}\sim 8^{\circ}\mathrm{C}$ 避光保存。

# 10.2 试剂配制

乙酸溶液：取 $10\mathrm{mL}$ 乙酸加入 $250~\mathrm{mL}$ 水，混匀。

# 10.3 标准品

展青霉素标准品 $\mathrm{(C_7H_6O_4}$ ，CAS号：149-29-1)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 10.4 标准溶液配制

10.4.1 标准储备溶液 $(100\mu \mathrm{g / mL})$ ：用 $2\mathrm{mL}$ 乙腈溶解展青霉素标准品 $1.0\mathrm{mg}$ 后，移入 $10~\mathrm{mL}$ 的容量瓶，乙腈定容至刻度。溶液转移至试剂瓶中后，在 $-20^{\circ}C$ 下冷冻保存，备用，6个月内有效。展青霉素标准溶液浓度的标定参见附录A。  
10.4.2 标准工作液 $(1\mu \mathrm{g / mL})$ ：移取 $100~\mu \mathrm{L}$ 经标定过的展青霉素标准储备溶液，用乙酸溶液溶解并转移至 $10~\mathrm{mL}$ 容量瓶中，定容至刻度。溶液转移至试剂瓶中后，在 $4^{\circ}C$ 下避光保存，3个月内有效。  
10.4.3 标准系列工作溶液：分别准确移取标准工作液适量至 $5\mathrm{mL}$ 容量瓶中，用乙酸溶液定容至刻度，配制展青霉素浓度为 $5\mathrm{ng/mL}, 10\mathrm{ng/mL}, 25\mathrm{ng/mL}, 50\mathrm{ng/mL}, 100\mathrm{ng/mL}, 150\mathrm{ng/mL}, 200\mathrm{ng/mL}, 250\mathrm{ng/mL}$ 系列标准溶液。

除不加同位素内标外，试样提取及净化柱净化操作同5.2.2。

# 12.3 仪器参考条件

a）液相色谱柱： $\mathrm{T}_{3}$ 柱，柱长 $150~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $3.0\mu \mathrm{m}$ ，或相当者；  
b）流动相：A相：水，B相：乙腈；  
c）梯度洗脱条件： $5\%$ B(0 min~13 min)， $100\%$ B(13 min~15 min)， $5\%$ B(15 min~20 min)；  
d) 流速: $0.8 \mathrm{~mL} / \mathrm{min}$ ;   
e） 色谱柱柱温： $40^{\circ}\mathrm{C}$   
f) 进样量: $100 \mu \mathrm{L}$ ;   
g）紫外检测器条件：检测波长为 $276\mathrm{nm}$

# 12.4 标准曲线的制作

将标准系列溶液由低到高浓度依次进样检测，以标准溶液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。

# 12.5 测定

将试样溶液注入液相色谱一质谱仪中，测得相应的峰面积，由标准曲线得到试样溶液中展青霉素的浓度。

# 13 分析结果的表述

试样中展青霉素的含量按式(2)计算：

$$
X = \frac {\rho \times V}{m} \times f \quad \dots \tag {2}
$$

式中：

$X$ ——试样中展青霉素的含量，单位为微克每千克或微克每升 $(\mu \mathrm{g} / \mathrm{kg}$ 或 $\mu \mathrm{g} / \mathrm{L})$ ；

$\rho$ ——由标准曲线得到的试样溶液中展青霉素的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——最终定容体积，单位毫升 $(\mathrm{mL})$ ；

$m$ ——试样的称样量，单位克 $(\mathrm{g})$ ；

$f$ ——稀释倍数。

计算结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 15 其他

液体试样的检出限为 $6\mu \mathrm{g / kg}$ ，定量限为 $20~\mu \mathrm{g / kg}$ ；固体、半流体试样的检出限为 $12\mu \mathrm{g / kg}$ ，定量限为 $40~\mu \mathrm{g / kg}$ 。

# 展青霉素标准溶液浓度的标定

# A.1 仪器校正

测定重铬酸钾溶液的摩尔消光系数，以求出使用仪器的校正因子。准确称取 $74\mathrm{mg}$ 经干燥的重铬酸钾，用 $0.009\mathrm{mol / L}$ 硫酸溶解后并准确稀释至 $1000~\mathrm{mL}$ ，相当于 $\left[c(\mathrm{K}_2\mathrm{Cr}_2\mathrm{O}_7) = 0.25\mathrm{mmol / L}\right]$ 。再吸取 $25~\mathrm{mL}$ 此稀释液于 $50~\mathrm{mL}$ 容量瓶中，加 $0.009\mathrm{mol / L}$ 硫酸稀释至刻度，相当于 $0.125\mathrm{mmol / L}$ 溶液。再吸取 $25~\mathrm{mL}$ 此稀释液于 $50~\mathrm{mL}$ 容量瓶中，加 $0.009\mathrm{mol / L}$ 硫酸稀释至刻度，相当于 $0.0625\mathrm{mmol / L}$ 溶液。用 $1\mathrm{cm}$ 石英杯，在最大吸收峰的波长( $350~\mathrm{nm}$ 处)用 $0.009\mathrm{mol / L}$ 硫酸作空白，测得以上三种不同浓度的溶液的吸光度，以上三种浓度的摩尔消光系数按式(A.1)计算：

$$
E = \frac {A}{c} \quad \dots \tag {A.1}
$$

式中：

$E$ ——重铬酸钾溶液的摩尔消光系数；

$A$ 测得重铬酸钾溶液的吸光度；

$c$ ——重铬酸钾溶液的摩尔浓度。

取三种浓度的摩尔消光系数的平均值 $E^{\prime}$ 并与重铬酸钾的摩尔消光系数值3160比较，即求出使用仪器的校正因子，按式(A.2)进行计算：

$$
f = \frac {3 1 6 0}{E ^ {\prime}} \quad \dots \tag {A.2}
$$

式中：

$f$ ——使用仪器的校正因子；

$E^{\prime}$ ——测得的重铬酸钾摩尔消光系数平均值。

若 $f$ 大于0.95或小于1.05，则使用仪器的校正因子可略而不计。

# A.2 展青霉素标准溶液的制备

取 $1000\mu \mathrm{L}$ 储备液用氮气吹干后，立即用 $20~\mathrm{mL}$ 乙醇溶解残渣，置于 $4^{\circ}C$ 冰箱保存。该标准溶液约为 $5\mu \mathrm{g / mL}$ 。用紫外分光光度计以 $1\mathrm{cm}$ 石英比色皿测此标准溶液在 $250~\mathrm{nm}\sim 350~\mathrm{nm}$ 处的最大吸收峰的波长及该波长的吸光度值，以乙醇为参比溶液。

展青霉素标准溶液的浓度按式(A.3)进行计算：

$$
\rho = \frac {A \times M \times 1 0 0 0 \times f}{E _ {2}} \quad \dots \tag {A.3}
$$

式中：

$\rho$ ——展青霉素标准溶液的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$A$ ——在波长 $276 \mathrm{~nm}$ 处测得的吸光度值；

M ——展青霉素的相对分子质量等于154；

1000——换算系数；

$f$ ——使用仪器的校正因子；

$E_{2}$ ——展青霉素乙醇溶液在最大吸收波长 $276\mathrm{nm}$ 处的摩尔消光系数14600。

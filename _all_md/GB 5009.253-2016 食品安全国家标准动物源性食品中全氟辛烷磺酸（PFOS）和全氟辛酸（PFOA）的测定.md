# 食品安全国家标准

# 动物源性食品中全氟辛烷磺酸(PFOS)和全氟辛酸(PFOA)的测定

# 1 范围

本标准规定了动物源性食品中全氟辛烷磺酸(PFOS)和全氟辛酸(PFOA)含量的同位素稀释液相色谱-串联质谱测定方法。

本标准适用于动物源性食品中PFOS和PFOA含量的测定。

# 2 原理

试样中PFOS和PFOA经乙腈提取，通过分散固相萃取净化，用液相色谱-串联质谱仪测定，采用内标法定量。

# 3 试剂和材料

除非另有说明，所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

3.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.3 盐酸（HCl）。  
3.4 氯化钠(NaCl)。  
3.5 乙酸铵 $\mathrm{(CH_3COONH_4)}$ ：色谱纯。  
3.6 N-丙基乙二胺固相吸附剂(PSA)： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}$ ，ProElut填料，或相当者。  
3.7 $\mathrm{C_{18}}$ 吸附剂： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}$ ，ProElut填料，或相当者。  
3.8 石墨化碳黑吸附剂(GCB)： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}$ ，ProElut填料，或相当者。  
3.9 乙酸铵溶液 $(5\mathrm{mmol} / \mathrm{L})$ ：准确称 $0.3855\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）乙酸铵，用水溶解定容至 $1000~\mathrm{mL}$ 。

3.10 全氟辛烷磺酸(PFOS)和全氟辛酸(PFOA)标准品溶液：准确称取PFOA[纯度 $\geqslant 98\%$ ，CAS: $335 - 67 - 1, \mathrm{CF}_3(\mathrm{CF}_2)_6\mathrm{COOH}]$ 和PFOS(纯度 $\geqslant 99\%$ ，CAS:1763-23-1， $\mathrm{C_8HF_{17}O_3S}$ )各 $10.00\mathrm{mg}$ （精确至 $0.0001\mathrm{g})$ ，用甲醇稀释至 $100~\mathrm{mL}$ ，配置成浓度均为 $100~\mu \mathrm{g / mL}$ 的PFOA和PFOS标准溶液。一 $4^{\circ}C$ 环境下保存，有效期为6个月。

3.11 PFOA标准储备溶液：吸取PFOA标准品溶液，用甲醇稀释，配制成浓度为 $100.0\mu \mathrm{g} / \mathrm{L}$ 的PFOA标准储备溶液；PFOS标准储备溶液：吸取PFOS标准溶液，用甲醇稀释，配制成浓度为 $100.0\mu \mathrm{g} / \mathrm{L}$ 的PFOS标准储备溶液。均在 $-4^{\circ}\mathrm{C}$ 环境下保存，有效期为4个月。

注：全氟辛烷磺酸(PFOS)在溶液中以全氟辛烷磺酸离子状态存在,样品溶液以及标准溶液以全氟辛烷磺酸定量。

3.12 混合系列标准工作溶液：吸取标准储备溶液，用甲醇稀释，配制成PFOA浓度依次为 $0.05\mu \mathrm{g} / \mathrm{L}$ 、 $0.1\mu \mathrm{g} / \mathrm{L}, 0.5\mu \mathrm{g} / \mathrm{L}, 1.0\mu \mathrm{g} / \mathrm{L}, 2.0\mu \mathrm{g} / \mathrm{L}, 10.0\mu \mathrm{g} / \mathrm{L}$ ，PFOS浓度依次为 $0.1\mu \mathrm{g} / \mathrm{L}, 0.2\mu \mathrm{g} / \mathrm{L}, 1.0\mu \mathrm{g} / \mathrm{L}, 10.0\mu \mathrm{g} / \mathrm{L}, 20.0\mu \mathrm{g} / \mathrm{L}, 40.0\mu \mathrm{g} / \mathrm{L}$ 的混合系列标准工作溶液。-4℃环境下保存，有效期为2个月。  
3.13 $^{13}\mathrm{C}_4$ -PFOA标准品溶液：通过甲醇稀释配制，稀释至浓度为 $50.0\mu \mathrm{g / mL}$ 的标准品溶液；

1,2,3,4- ${}^{13}\mathrm{C}_4$ -PFOS标准品溶液：通过甲醇稀释配制，稀释至浓度为 $50.0\mu \mathrm{g / mL}$ 的标准品溶液。均在 $-4^{\circ}\mathrm{C}$ 环境下保存，有效期为6个月。  
3.14 内标混合使用溶液I：吸取 ${}^{13}\mathrm{C}_4$ -PFOA和 $1,2,3,4^{-13}\mathrm{C}_4$ -PFOS标准品溶液，用甲醇稀释，配制成 ${}^{13}\mathrm{C}_4$ -PFOA和 $1,2,3,4^{-13}\mathrm{C}_4$ -PFOS浓度分别为 $10.0\mu \mathrm{g / L}$ 和 $50.0\mu \mathrm{g / L}$ 的内标混合使用溶液。 $-4^{\circ}\mathrm{C}$ 环境下保存，有效期为4个月。  
3.15 内标混合使用溶液Ⅱ：吸取 ${}^{13}\mathrm{C}_4$ -PFOA 和 $1,2,3,4{-}^{13}\mathrm{C}_4$ -PFOS 内标混合使用溶液 I，用甲醇稀释，配制成 ${}^{13}\mathrm{C}_4$ -PFOA 和 $1,2,3,4{-}^{13}\mathrm{C}_4$ -PFOS 浓度分别为 $1.0~\mu \mathrm{g / L}$ 和 $5.0~\mu \mathrm{g / L}$ 的内标混合使用溶液。 $-4^{\circ}\mathrm{C}$ 环境下保存，有效期为 2 个月。  
3.16 含有碳 13 同位素内标的混合系列标准工作溶液: 吸取标准储备溶液以及内标混合使用溶液 I, 用甲醇稀释, 配制成 PFOA 浓度依次为 $0.05 \mu \mathrm{g} / \mathrm{L}, 0.1 \mu \mathrm{g} / \mathrm{L}, 0.5 \mu \mathrm{g} / \mathrm{L}, 1.0 \mu \mathrm{g} / \mathrm{L}, 2.0 \mu \mathrm{g} / \mathrm{L}, 10.0 \mu \mathrm{g} / \mathrm{L}$ , PFOS 浓度依次为 $0.1 \mu \mathrm{g} / \mathrm{L}, 0.2 \mu \mathrm{g} / \mathrm{L}, 1.0 \mu \mathrm{g} / \mathrm{L}, 10.0 \mu \mathrm{g} / \mathrm{L}, 20.0 \mu \mathrm{g} / \mathrm{L}, 40.0 \mu \mathrm{g} / \mathrm{L}$ , 以及 ${ }^{13} \mathrm{C}_{4}-\mathrm{PFOA}$ 和 $1,2,3,4-{}^{13} \mathrm{C}_{4}-\mathrm{PFOS}$ 浓度均分别为 $1.0 \mu \mathrm{g} / \mathrm{L}$ 和 $5.0 \mu \mathrm{g} / \mathrm{L}$ 的含有 ${ }^{13} \mathrm{C}$ 同位素内标的混合系列标准工作溶液。 $-4^{\circ} \mathrm{C}$ 环境下保存, 有效期为 2 个月。  
3.17 滤膜： $0.22\mu \mathrm{m}$ ，有机相型。

# 4 仪器和设备

4.1 液相色谱-串联质谱仪：配有电喷雾离子源（ESI）。  
4.2 氮吹仪。  
4.3 粉碎机。  
4.4 离心机，转速 $\geqslant 5000\mathrm{r / min}$   
4.5 均质机，参考转速： $3400\mathrm{r / min}\sim 24000\mathrm{r / min}$   
4.6 分析天平：感量 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$

# 5 样品制备

鱼、虾、蟹、贝类、章鱼、鸡肉、猪肉、牛肉取其可食部分（去骨，去壳），切成小块，用粉碎机制成肉糜；蛋品去壳，制成匀浆；牛奶，混匀备用。将样品置于玻璃样品瓶中，密封，标识后置于 $-18^{\circ}\mathrm{C}$ 保存。

# 6 分析步骤

# 6.1 样品前处理

# 6.1.1 提取

称取试样 $5\mathrm{g}$ （准确至 $0.01\mathrm{g}$ ）（样品使用前解冻均质），置于 $50~\mathrm{mL}$ 聚丙烯离心管中，加入内标混合使用溶液Ⅱ $400~\mu \mathrm{L}$ ，加水 $5\mathrm{mL}$ ，漩涡混合 $1\mathrm{min}$ ，加入 $10~\mathrm{mL}$ 乙腈(3.2)和 $30~\mu \mathrm{L}$ 盐酸(3.3)，震荡 $10\mathrm{min}$ 。加入 $2\mathrm{g}$ 氯化钠(3.4)，再次振摇 $10\mathrm{min}$ ，以 $5000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。移取上层乙腈溶液于另一试管中，在 $45^{\circ}C$ 水浴中氮气吹至约 $4\mathrm{mL}$ ，待净化。

# 6.1.2 净化

将上述溶液转移至装有 $100\mathrm{mg}$ PSA(3.6)、 $40\mathrm{mg}$ $\mathrm{C}_{18}(3.7)$ 和 $20\mathrm{mg}$ GCB(3.8)的 $15\mathrm{mL}$ 聚丙烯离心管中，振摇 $10\mathrm{min}$ ，以 $5000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，移取上清溶液于另一试管中，在 $45^{\circ}\mathrm{C}$ 水浴中氮气吹至干，用 $1\mathrm{mL}$ 甲醇(3.1)溶解，吸入 $1\mathrm{mL}$ 注射器，经 $0.22\mu \mathrm{m}$ 有机滤膜过滤后，待测。

# 6.2 测定

# 6.2.1 液相色谱参考条件

6.2.1.1 为降低高效液相色谱管道中引入的PFOS和PFOA的污染，液相系统中加一预柱，并对部分特氟龙材质管路替换为PEEK管路或不锈钢管路。  
6.2.1.2 分析色谱柱： $\mathrm{C_{18}}$ ， $150 \mathrm{~mm} \times 2.1 \mathrm{~mm}$ （内径）， $5 \mu \mathrm{m}$ ，或相当者。  
6.2.1.3 预吸附柱： $\mathrm{C}_{18}$ ， $50 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ （内径）， $5 \mu \mathrm{m}$ ，或相当者。  
6.2.1.4 流动相梯度洗脱程序见表1。  
6.2.1.5 流速： $200~\mu \mathrm{L / min}$   
6.2.1.6 进样量： $10~\mu \mathrm{L}$

# 6.2.4 定量测定

以各混合系列标准工作溶液中的PFOA或PFOS浓度和PFOA或PFOS与其相对应的 $^{13}\mathrm{C}$ 同位素标记物的峰面积比绘制标准曲线，按照内标法进行定量计算。在6.2.1所述的色谱条件下，PFOA和PFOS的参考保留时间约为 $7.66\mathrm{min}$ 和 $8.78\mathrm{min}$ ， ${}^{13}\mathrm{C}_4$ -PFOA和 $1,2,3,4-{}^{13}\mathrm{C}_4$ -PFOS的参考保留时间约为 $7.66\mathrm{min}$ 和 $8.79\mathrm{min}$ ，标准溶液的多反应监测(MRM)图参见附录B。

标准工作溶液和样液中待测物的响应值均应在标准曲线的线性响应范围内，如果含量超过范围，则重新取样分析，增加相应内标添加量，使内标浓度与待测液浓度相匹配，然后用甲醇稀释到适当浓度后分析。

# 6.3 空白试验

除不加试样外，均按6.1及6.2所述操作步骤进行。

# 7 质量控制

# 7.1 取样抽样

取样抽样均应采用不锈钢仪器，避免接触塑料。

# 7.2 有机溶剂和试验器皿

本标准中使用到的所有有机溶剂及试验器皿，在使用前，应进行空白试验。如本底值高于定量限，应对有机溶剂进行重蒸，或更换试验器皿，直至本底值低于定量限。

# 8 分析结果的表述

试样中PFOA和PFOS的含量按式(1)计算：

$$
X = \frac {\left(A - A _ {0}\right) \times V \times f}{m} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中PFOA或PFOS的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——试液中PFOA或PFOS色谱峰与相对应内标物色谱峰的峰面积比值，经标准曲线求得的对应的PFOA或PFOS浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$A_{0}$ ——空白溶液中PFOA或PFOS色谱峰与相对应内标物色谱峰的峰面积比值，经标准曲线求得的对应的PFOA或PFOS浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试液稀释倍数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 9 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $30\%$ 。

# 10 其他

本方法PFOA和PFOS的检出限分别为 $0.002\mu \mathrm{g / kg}$ 和 $0.02\mu \mathrm{g / kg}$ ，定量限分别为 $0.01\mu \mathrm{g / kg}$ 和 $0.1~\mu \mathrm{g / kg}$ 。

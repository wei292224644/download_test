# 食品安全国家标准

# 水产品中二甲戊灵残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了水产品中二甲戊灵残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鱼、虾、蟹、贝、海参、龟鳖类可食性组织中二甲戊灵残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T30891—2014 水产品抽样规范

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的二甲戊灵经乙腈提取，HLB固相萃取柱净化， $\mathrm{C_{18}}$ 吸附剂去除脂类杂质，液相色谱-串联质谱法测定，内标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ ：色谱纯。  
5.1.2 甲醇 $(\mathrm{CH}_4\mathrm{O})$ ：色谱纯。  
5.1.3 甲酸 $(\mathrm{CH}_2\mathrm{O}_2)$ ：色谱纯。  
5.1.4 正己烷 $(\mathrm{C_6H_{14}})$ ：色谱纯  
5.1.5 丙酮 $(\mathrm{C_3H_6O})$ ：色谱纯。  
5.1.6 氯化钠 $(\mathrm{NaCl})$

# 5.2 溶液配制

5.2.1 正己烷-丙酮溶液：取正己烷 $500~\mathrm{mL}$ 、丙酮 $500~\mathrm{mL}$ ，混匀。  
5.2.2 甲酸水溶液 $(0.1\%)$ ：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$

# 5.3 标准品

5.3.1 二甲戊灵(Pendimethalin, $\mathrm{C_{13}H_{19}N_3O_4}$ ,CAS号：40487-42-1)，含量 $\geqslant 98.0\%$ 。  
5.3.2 氰代二甲戊灵(Pendimethalin- $\mathrm{D}_5$ ， $\mathrm{C_{13}H_{14}N_3O_4D_5}$ ，CAS号：1219803-39-0)，含量 $\geq 98.0\%$

# 5.4 标准溶液配制

5.4.1 标准储备液 $(1\mathrm{mg / mL})$ ：取二甲戊灵标准品约 $10\mathrm{mg}$ ，精密称定，用乙腈溶解并稀释定容至 $10\mathrm{mL}$ 容量瓶，摇匀，即得。 $4^{\circ}C$ 避光保存，有效期6个月。  
5.4.2 内标储备液 $(1\mathrm{mg / mL})$ ：取氘代二甲戊灵标准品约 $10~\mathrm{mg}$ ，精密称定，用乙溶解并稀释定容至

$10\mathrm{mL}$ 容量瓶，摇匀，即得。 $4^{\circ}C$ 避光保存，有效期6个月。

5.4.3 标准中间液 $(1\mu \mathrm{g / mL})$ ：准确移取二甲戊灵标准储备液 $0.1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，配制成浓度为 $1\mu \mathrm{g / mL}$ 的标准中间液。 $4^{\circ}C$ 下避光保存，有效期2周。

5.4.4 内标中间液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：准确移取氘代二甲戊灵标准储备液 $0.1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，配制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 的内标中间液。 $4^{\circ}\mathrm{C}$ 下避光保存，有效期2周。

5.5 材料

5.5.1 HLB固相萃取柱： $60~\mathrm{mg / 3~mL}$ ，或相当者。  
5.5.2 $\mathrm{C_{18}}$ 吸附剂： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}_{\circ}$   
5.5.3 微孔滤膜： $0.22\mu \mathrm{m}$ ，耐有机试剂。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源(ESI源)。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$   
6.3 氮吹仪。  
6.4 均质机。  
6.5 涡旋混合器。  
6.6 离心机： $10000\mathrm{r / min}$ 或以上。  
6.7 超声波振荡器。  
6.8 旋转蒸发器。  
6.9 固相萃取装置。

# 7 试料的制备与保存

# 7.1 试料的制备

按GB/T30891—2014附录B的要求制样。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试料 $5.0\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，加内标工作液 $50~\mu \mathrm{L}$ ，混匀，依次加水 $6\mathrm{mL}$ 、氯化钠 $1\mathrm{g}$ ，乙腈 $20~\mathrm{mL},3000~\mathrm{r / min}$ 涡旋 $1\mathrm{min}$ ，超声提取 $20\mathrm{min},4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取上清液，残渣加乙腈 $20~\mathrm{mL}$ 重复提取一次，合并上清液。 $40^{\circ}C$ 旋蒸浓缩至近干，加乙腈 $3\mathrm{mL}$ 溶解残渣，备用。

# 8.2 净化

固相萃取柱用乙腈 $3\mathrm{mL}$ 活化，取备用液过柱，控制流速不超过 $1\mathrm{mL / min}$ ，用乙腈 $3\mathrm{mL}$ 淋洗，加正己烷-丙酮溶液 $8\mathrm{mL}$ 洗脱。收集洗脱液， $40^{\circ}\mathrm{C}$ 氮吹至近干，准确加入乙腈 $2\mathrm{mL}$ 复溶，加 $\mathrm{C_{18}}$ 吸附剂 $0.5\mathrm{g}$ ，以 $3000\mathrm{r / min}$ 涡旋 $1\mathrm{min}$ ，以 $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱仪测定。

# 8.3 标准曲线的制备

准确移取标准工作液、内标工作液各适量，用乙腈稀释配制成二甲戊灵浓度分别为 $2.5\mu \mathrm{g} / \mathrm{L}$ 、 $5.0\mu \mathrm{g} / \mathrm{L},10.0\mu \mathrm{g} / \mathrm{L},20.0\mu \mathrm{g} / \mathrm{L},50.0\mu \mathrm{g} / \mathrm{L},100.0\mu \mathrm{g} / \mathrm{L}$ ，氘代二甲戊灵浓度均为 $25.0\mu \mathrm{g} / \mathrm{L}$ 的系列混合

标准工作溶液，供液相色谱-串联质谱仪测定。使用时现用现配。以待测物与内标物峰面积比值为纵坐标、对应二甲戊灵浓度为横坐标，绘制标准工作曲线，求回归方程和相关系数。

# 8.4 测定

# 8.4.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 柱（ $100\mathrm{mm}\times 2.1\mathrm{mm},1.7\mu \mathrm{m})$ ，或相当者；  
b） 柱温： $40^{\circ}C$   
c）进样量： $5\mu \mathrm{L}$   
d) 流速: $0.5 \mathrm{~mL} / \mathrm{min}$ ;   
e）流动相：A为乙腈，B为甲酸水溶液 $(0.1\%)$   
f）梯度洗脱条件见表1。

# 8.4.3 测定方法

# a）定性方法

通过试样中各组分的保留时间、特征离子与相应的标准工作液中各组分的保留时间、特征离子相对照进行定性。在相同测试条件下，阳性样品保留时间与标准物质保留时间相对偏差在 $\pm 2.5\%$ 以内，且检测到的相对离子丰度，应当与浓度相当的校正标准品相对丰度一致。定量与定性碎片离子丰度比应符合表3的要求。

# b）定量方法

将试样溶液和标准工作液在相同的色谱质谱条件下进行测定，作单点或多点校准，以色谱峰面积定量，内标法计算。标准工作液和试样液中目标物的响应值均应在仪器检测线性范围内。标准溶液的多反应监测谱图见附录A。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测物残留量按标准曲线或公式(1)计算

$$
X = \frac {A \times A _ {i S} ^ {\prime} \times C _ {S} \times C _ {i S} \times V}{A _ {i S} \times A _ {S} \times C _ {i S} ^ {\prime} \times m} \tag {1}
$$

式中：

$X$ ——试样中二甲戊灵残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A_{\mathrm{S}}$ ——标准工作液中二甲戊灵的峰面积；

$A_{is}^{\prime}$ ——标准工作液中气氛代二甲戊灵的峰面积；

A ——试样溶液中二甲戊灵的峰面积；

$A_{iS}$ ——试样溶液中氘代二甲戊灵的峰面积；

$C_{\mathrm{S}}$ ——标准工作液中三甲戊灵标准品浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$C_{\mathrm{is}}^{\prime}$ ——标准工作液中氘代二甲戊灵标准品浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{iS}$ ——试样溶液中氘代二甲戊灵浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样复溶液体积的数值；单位为毫升 $(\mathrm{mL})$

$m$ 供试试样质量的数值，单位为克（g）

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

二甲戊灵的检测限为 $1.0\mu \mathrm{g / kg}$ ，定量限为 $2.0\mu \mathrm{g / kg}$

# 10.2 准确度

二甲戊灵在 $2\mu \mathrm{g} / \mathrm{kg}\sim 20\mu \mathrm{g} / \mathrm{kg}$ 添加浓度的回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

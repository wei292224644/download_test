# 食品添加剂 $\gamma$ -己内酯

# 1 范围

本标准适用于食品添加剂 $\gamma$ -己内酯。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

$\gamma$ -己内酯

# 2.2 分子式

$\mathrm{C_6H_{10}O_2}$

# 高效液相色谱法

# 1 范围

本文件规定了动物性食品中环丙氨嗪残留量检测的制样和高效液相色谱测定方法。

本文件适用于羊和禽肌肉、脂肪、肝脏及羊肾脏组织中环丙氨酸残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的环丙氨嗪，用三氯乙酸/乙腈溶液提取，混合阳离子交换固相萃取柱净化，高效液相色谱测定，外标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$   
5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.3 盐酸（HCl）。  
5.1.4 乙酸铵 $\mathrm{(CH_3CONH_3)}$   
5.1.5 乙酸 $\mathrm{(CH_3COOH)}$   
5.1.6 正己烷 $(\mathrm{C_6H_{14}})$   
5.1.7 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。  
5.1.8 三氯乙酸 $\mathrm{(CCl_3COOH)}$

# 5.2 溶液配制

5.2.1 $25\mathrm{mmol / L}$ 乙酸铵溶液：取乙酸铵 $0.19\mathrm{g}$ ，加水 $950~\mathrm{mL}$ 使溶解，用乙酸调 $\mathrm{pH}$ 至5.0，加水稀释至 $1000~\mathrm{mL}$ 。  
5.2.2 $1\%$ 三氯乙酸溶液：取三氯乙酸 $1\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$   
5.2.3 提取液：取 $1\%$ 三氯乙酸溶液 $15~\mathrm{mL}$ ，用乙腈稀释至 $100~\mathrm{mL}$   
5.2.4 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液：取盐酸 $9\mathrm{mL}$ ，用稀释至 $1000\mathrm{mL}$ 。  
5.2.5 $5\%$ 氨水甲醇溶液：取氨水 $5\mathrm{mL}$ ，用甲醇稀释至 $100~\mathrm{mL}$   
5.2.6 流动相：取 $25\mathrm{mmol / L}$ 乙酸铵溶液 $40.0~\mathrm{mL}$ ，用乙腈定容至 $1000~\mathrm{mL}$ 。

# 5.3 标准品

环丙氨嗪(Cyromazine， $\mathrm{C_6H_{10}N_6}$ ，CAS号：66215-27-8)，含量 $\geq 97.0\%$

# 5.4 标准溶液的制备

5.4.1 标准储备液 $(1\mathrm{mg / mL})$ ：精确称取相当于环丙氨嗪 $25\mathrm{mg}$ 的环丙氨嗪标准品，用甲醇溶解并稀释定容至 $25~\mathrm{mL}$ 容量瓶，配成浓度为 $1\mathrm{mg / mL}$ 标准储备液。 $-18^{\circ}C$ 以下保存，有效期1年。  
5.4.2 标准工作液 $(10\mathrm{mg / L})$ ：精密量取标准储备液 $1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 容量瓶，用甲醇稀释至刻度，配成浓度为 $10\mathrm{mg / L}$ 的环丙氨嗪标准工作液。 $-18^{\circ}C$ 以下保存，有效期6个月。

# 5.5 材料

5.5.1 混合阳离子交换固相萃取柱： $60~\mathrm{mg / 3~mL}$ ，或相当者。  
5.5.2 针头式过滤器通用型滤膜：尼龙材质，孔径 $0.45\mu \mathrm{m}$ 或性能相当者。

# 6 仪器和设备

6.1 高效液相色谱仪：配紫外检测器  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和感量 $0.01\mathrm{g}$   
6.3 氮吹仪。  
6.4 固相萃取装置。  
6.5 均质机。  
6.6 涡旋混合器。  
6.7 离心机：最大转速不低于 $10000\mathrm{r / min}$   
6.8 pH计。  
6.9 旋转蒸发仪。  
6.10 分液漏斗： $100~\mathrm{mL}$   
6.11 鸡心瓶： $100~\mathrm{mL}$   
6.12 刻度离心管：分度值 $0.1\mathrm{mL}$

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取试料 $5\mathrm{g}$ （精确至 $\pm 0.02\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管，加提取液 $15~\mathrm{mL}$ ，高速均质使均匀分散，以 $5000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液于 $100~\mathrm{mL}$ 分液漏斗中，残渣加提取液 $10~\mathrm{mL}$ ，重复提取1次，合并上清液，加正己烷 $30~\mathrm{mL}$ ，振摇 $2\mathrm{min}$ ，静置使分层。收集下层液于 $100~\mathrm{mL}$ 鸡心瓶中， $50^{\circ}C$ 水浴旋转蒸发至 $1\mathrm{mL}$ （需控制真空度防止爆沸），转至 $10~\mathrm{mL}$ 刻度离心管中，用提取液淋洗鸡心瓶2次，每次 $2\mathrm{mL}$ ，合并提取液，以 $10000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液，备用。

# 8.2 净化

固相萃取柱依次用甲醇 $3\mathrm{mL}$ 和水 $3\mathrm{mL}$ 活化，取备用液过柱，控制流速 $1\mathrm{mL / min}$ 。依次用甲醇

$3\mathrm{mL},0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液 $3\mathrm{mL}$ 、水 $3\mathrm{mL}$ 和甲醇 $3\mathrm{mL}$ 淋洗，抽干，用氨水甲醇溶液 $5\mathrm{mL}$ 洗脱。

收集洗脱液，于 $50^{\circ}\mathrm{C}$ 氮气吹干，用流动相 $1.0\mathrm{mL}$ 溶解残余物，涡旋 $30~\mathrm{s}$ ，滤膜过滤，供高效液相色谱仪测定。

# 8.3 标准曲线的制备

精密量取环丙氨嗪标准工作液（ $10\mathrm{mg / L}$ ）适量，用流动相稀释，配成浓度为 $0.05\mu \mathrm{g / mL}$ 、 $0.1\mu \mathrm{g / mL}, 0.2\mu \mathrm{g / mL}, 0.4\mu \mathrm{g / mL}, 1\mu \mathrm{g / mL}$ 和 $2\mu \mathrm{g / mL}$ 的系列标准溶液，供高效液相色谱法测定。以测得峰面积为纵坐标、对应标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.4 测定

# 8.4.1 色谱条件参考条件

a）色谱柱：HILIC（亲水反应离子交换色谱柱， $2.1\mathrm{mm}\times 150\mathrm{mm},3.5\mu \mathrm{m})$ ，或相当者；  
b) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ ;   
c）检测波长： $214~\mathrm{nm}$   
d）进样量： $20~\mu \mathrm{L}$   
e） 柱温： $30^{\circ}C$

# 8.4.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法以色谱峰面积定量。标准工作液及试样溶液中待测物响应值应在仪器检测的线性范围之内。试样中待测物峰的保留时间与标准工作液相应峰的保留时间相对偏差应在 $\pm 2.5\%$ 以内。环丙氨嗪标准溶液色谱图见附录A。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

供试试料中环丙氨嗪残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times C _ {\mathrm {S}} \times V _ {1} \times V _ {3} \times 1 0 0 0}{A _ {\mathrm {S}} \times V _ {2} \times m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ 供试试料中环丙氨嗪残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

A ——试样溶液中环丙氨嗪峰面积；

$C_{\mathrm{S}}$ ——标准溶液中环丙氨嗪浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$V_{1}$ ——提取液体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{3}$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$

$A_{\mathrm{s}}$ ——标准溶液中环丙氨嗪峰面积；

$V_{2}$ ——用于净化的提取液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——供试试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $10~\mu \mathrm{g / kg}$ ，定量限为 $25~\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $25\mu \mathrm{g / kg}\sim 600\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 110\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

# 1 范围

本文件规定了猪、牛、羊、鸡组织（肌肉、肝脏、肾脏和脂肪），鸡蛋和牛奶中氨基甲酸酯类杀虫剂（克百威、3-羟基克百威、涕灭威、涕灭威砜、涕灭威亚砜、灭多威、速灭威、灭除威、混杀威、灭虫威、灭虫威砜、灭虫威亚砜、抗蚜威、去甲基抗蚜威、乙硫苯威、异丙威、西维因、恶虫威、残杀威、仲丁威、苯氧威、茚虫威）残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、牛、羊、鸡组织（肌肉、肝脏、肾脏和脂肪），鸡蛋和牛奶中氨基甲酸酯类杀虫剂残留量的测定。

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的目标化合物经乙腈提取，正己烷除脂，固相萃取富集净化，液相色谱-串联质谱法测定，外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯  
5.1.2 甲醇 $(\mathrm{CH}_3\mathrm{OH})$   
5.1.3 甲酸（HCOOH）：色谱纯。  
5.1.4 正己烷 $(\mathrm{C_6H_{14}})$   
5.1.5 二氯甲烷 $(\mathrm{CH}_2\mathrm{Cl}_2)$   
5.1.6 氯化钠(NaCl)。  
5.1.7 无水硫酸钠 $\mathrm{(Na_2SO_4)}$

# 5.2 溶液配制

5.2.1 二氯甲烷-甲醇溶液：取二氯甲烷 $99~\mathrm{mL}$ 、甲醇 $1\mathrm{mL}$ ，混匀。  
5.2.2 $25\%$ 乙腈溶液：取乙腈 $25~\mathrm{mL}$ 水 $75~\mathrm{mL}$ ，混匀。  
5.2.3 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.4 饱和氯化钠溶液：取氯化钠 $50\mathrm{g}$ ，加水 $100~\mathrm{mL}$ ，边加边振摇，静置即得。

# 5.3 标准品

克百威、3-羟基克百威、涕灭威、涕灭威砜、涕灭威亚砜、灭多威、速灭威、灭除威、混杀威、灭虫威、灭虫威砜、灭虫威亚砜、抗蚜威、去甲基抗蚜威、乙硫苯威、异丙威、西维因、恶虫威、残杀威、仲丁威、苯氧威和茚虫威，含量均 $\geq 95\%$ ，见附录A。

# 5.4 标准溶液的配制

5.4.1 标准储备液：取各氨基甲酸酯类杀虫剂标准品约 $10\mathrm{mg}$ ，精密称定，分别加乙腈适量使溶解并稀释定容至 $10\mathrm{mL}$ 棕色容量瓶，配制成浓度均为 $1000\mu \mathrm{g} / \mathrm{mL}$ 的标准储备液。 $-18^{\circ}\mathrm{C}$ 以下避光保存，有效期6个月。  
5.4.2 混合标准中间液：分别准确移取标准储备液各 $1\mathrm{mL}$ ，于 $100\mathrm{mL}$ 棕色容量瓶，用乙腈稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的混合标准中间液。 $-18^{\circ}C$ 以下避光保存，有效期6个月。  
5.4.3 混合标准工作液：准确移取混合标准中间液适量，用 $25\%$ 乙腈溶液稀释成浓度为 $2\mu \mathrm{g} / \mathrm{L}, 5\mu \mathrm{g} / \mathrm{L}, 20\mu \mathrm{g} / \mathrm{L}, 50\mu \mathrm{g} / \mathrm{L}, 200\mu \mathrm{g} / \mathrm{L}$ 和 $500\mu \mathrm{g} / \mathrm{L}$ 的系列混合标准工作液。现用现配。

# 5.5 材料

5.5.1氨基固相萃取柱： $500\mathrm{mg / 6mL}$ ，或相当者，使用前用二氯甲烷-甲醇溶液 $4\mathrm{mL}$ 活化。  
5.5.2 针头式过滤器（通用型滤膜）：孔径 $0.22\mu \mathrm{m}$ ，有机系。

# 6 仪器和设备

6.1 液相色谱串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$   
6.3 氮吹仪。  
6.4 固相萃取装置。  
6.5 离心管：聚丙烯塑料离心管， $50~\mathrm{mL}$ 。  
6.6 离心机： $5000\mathrm{r / min}$ 或以上。  
6.7 涡旋混合器。

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存，3个月内进行分析检测。

# 8 测定步骤

# 8.1 提取

取试料 $5\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 塑料离心管，依次加入无水硫酸钠 $5\mathrm{g}$ 、乙腈 $15~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，振荡 $30\mathrm{min},5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液，残渣加乙腈 $10~\mathrm{mL}$ ，重复提取1次，合并提取液，备用。

# 8.2 净化

取备用液，加饱和氯化钠溶液 $10\mathrm{mL}$ 、正己烷 $10\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ， $5000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，取中间层乙腈溶液于 $50~\mathrm{mL}$ 塑料离心管中， $40^{\circ}\mathrm{C}$ 水浴氮吹至近干，加二氯甲烷-甲醇溶液 $2\mathrm{mL}$ 溶解残渣，过经活化的氨基固相萃取柱，收集流出液，用二氯甲烷-甲醇溶液 $2\mathrm{mL}$ 洗 $50~\mathrm{mL}$ 塑料离心管后过柱，并重复1次，收集全部流出液， $50^{\circ}\mathrm{C}$ 水浴氮吹至近干，用 $25\%$ 乙腈溶液 $1.0~\mathrm{mL}$ 溶解残渣，涡旋混匀， $0.22\mu \mathrm{m}$ 滤膜滤

过，供液相色谱-串联质谱测定。

# 8.3 标准曲线的制备

取 $2\mu \mathrm{g} / \mathrm{L},5\mu \mathrm{g} / \mathrm{L},20\mu \mathrm{g} / \mathrm{L},50\mu \mathrm{g} / \mathrm{L},200\mu \mathrm{g} / \mathrm{L}$ 和 $500~\mu \mathrm{g} / \mathrm{L}$ 的系列混合标准工作溶液，进样，供液相色谱-串联质谱测定。以定量离子对峰面积为纵坐标、标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

超过表3规定的范围，则可判定为样品中存在对应的待测物。标准工作溶液多反应监测色谱图见附录B。

# 9 结果计算和表述

试样中待测物残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V \times 1 0 0 0}{A _ {\mathrm {S}} \times m \times 1 0 0 0} \dots \tag {1}
$$

式中：

$X$ ——试料中氨基甲酸酯类杀虫剂残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ ——标准溶液中氨基甲酸酯类杀虫剂浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$A$ ——试料溶液中氨基甲酸酯类杀虫剂的峰面积；

$A_{S}$ ——标准溶液中氨基甲酸酯类杀虫剂的峰面积；

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法在猪、牛、羊、鸡组织（肌肉、肝脏、肾脏和脂肪），鸡蛋和牛奶中的检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1.0~\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法在 $1\mu \mathrm{g} / \mathrm{kg}\sim 5\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $60\% \sim 110\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

# 前言

本标准代替GB/T23296.10—2009《食品接触材料 高分子材料 食品模拟物中对苯二甲酸的测定 高效液相色谱法》和SN/T2184—2008《食品接触材料 高分子材料 食品模拟物中对苯二酸的测定 高效液相色谱法》。

本标准与GB/T23296.10—2009和SN/T2184—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 对苯二甲酸迁移量的测定”；  
——修改了食品模拟物试液的制备；  
——修改了分析结果的表述；  
——修改了检出限和定量限。

# 食品安全国家标准

# 食品接触材料及制品

# 对苯二甲酸迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中对苯二甲酸迁移量的测定方法。

本标准适用于食品接触材料及制品中对苯二甲酸迁移量的测定。

# 2 原理

食品模拟物中的对苯二甲酸通过高效液相色谱进行分离，采用紫外检测器进行检测。水基、酸性、酒精类食品模拟物直接进样测定，橄榄油介质食品模拟物通过稀碳酸氢钠溶液提取后进样测定。以邻苯二甲酸作内标，采用内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 三水合乙酸钠 $\mathrm{(C_2H_3O_2Na\cdot 3H_2O)}$   
3.1.2 碳酸氢钠 $\mathrm{(NaHCO_3)}$ 。  
3.1.3 正庚烷 $(n - \mathrm{C}_7\mathrm{H}_{16})$ 。  
3.1.4 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.5 异丙醇 $\mathrm{(C_3H_8O)}$ 。  
3.1.6 磷酸 $\mathrm{(H_3PO_4)}$ 。  
3.1.7 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.8 水基、酸性、酒精类、油基食品模拟物：所用试剂依据GB31604.1的规定。

# 3.2 试剂配制

3.2.1 水基、酸性、酒精类、油基食品模拟物：按GB5009.156操作。  
3.2.2 $1\mathrm{g / L}$ 碳酸氢钠水溶液：称取 $1\mathrm{g}$ （精确至 $0.1\mathrm{g})$ 碳酸氢钠，用水溶解并定容于 $1\textrm{L}$ 容量瓶中。  
3.2.3 乙酸钠缓冲液 $(\mathrm{pH} = 3.6)$ ：称取 $25.0\mathrm{g}$ 三水合乙酸钠溶于 $350~\mathrm{mL}$ 水中，加入 $5.0~\mathrm{mL}$ 磷酸，用约 $50~\mathrm{mL}$ 冰乙酸调节 $\mathrm{pH}$ 至3.6，加水定容至 $500~\mathrm{mL}$   
3.2.4 $50\%$ 乙酸溶液：量取 $50~\mathrm{mL}$ 冰乙酸于 $100~\mathrm{mL}$ 容量瓶中，用水定容。

# 3.3 标准品

3.3.1 对苯二甲酸 $\mathrm{(C_8H_6O_4}$ ,CAS号：100-21-0)，纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

3.3.2 邻苯二甲酸 $\mathrm{(C_8H_6O_4}$ ，CAS号：88-99-3)，纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 对苯二甲酸标准储备溶液 $(500\mathrm{mg / L})$ ：准确称取 $50~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )对苯二甲酸标准品，加入 $90~\mathrm{mL}$ 甲醇使其溶解，将溶液加热至 $50^{\circ}C$ ，持续 $1\textrm{h}$ ，使对苯二甲酸充分溶解。待溶液冷却后转移至 $100~\mathrm{mL}$ 容量瓶，用甲醇定容。有效期为6个月。  
3.4.2 对苯二甲酸标准中间溶液：分别于6只 $25~\mathrm{mL}$ 容量瓶中加入 $0\mathrm{mL},1.0\mathrm{mL},2.0\mathrm{mL},5.0\mathrm{mL}$ 、 $10.0~\mathrm{mL},20.0~\mathrm{mL}$ 对苯二甲酸标准储备溶液，再分别加入 $5\mathrm{mL}$ 邻苯二甲酸内标储备溶液，用甲醇定容。对苯二甲酸标准中间溶液的浓度分别为 $0.0\mathrm{mg / L},20\mathrm{mg / L},40\mathrm{mg / L},100\mathrm{mg / L},200\mathrm{mg / L},400\mathrm{mg / L}$ 。内标邻苯二甲酸的浓度均为 $200\mathrm{mg / L}$ 。应当天配制。  
3.4.3 水基、酸性、酒精类食品模拟物介质标准工作溶液：分别移取 $2.0 \mathrm{~mL}$ 对苯二甲酸标准中间溶液于6个 $50 \mathrm{~mL}$ 容量瓶中，用对应的食品模拟物定容。水基、酸性、酒精类食品模拟物中对苯二甲酸标准工作液浓度分别为 $0.0 \mathrm{mg} / \mathrm{L}, 0.8 \mathrm{mg} / \mathrm{L}, 1.6 \mathrm{mg} / \mathrm{L}, 4.0 \mathrm{mg} / \mathrm{L}, 8.0 \mathrm{mg} / \mathrm{L}, 16.0 \mathrm{mg} / \mathrm{L}$ ；内标浓度均为 $8.0 \mathrm{mg} / \mathrm{L}$ 。  
3.4.4 橄榄油介质标准工作溶液：分别称取橄榄油 $50\mathrm{g}$ 于6个 $250~\mathrm{mL}$ 分液漏斗中，准确移入 $2.0~\mathrm{mL}$ 对苯二甲酸标准中间溶液，混合均匀后加入 $50~\mathrm{mL}$ 正庚烷，再混匀。再加入 $20~\mathrm{mL}$ 碳酸氢钠溶液，充分振荡 $1\mathrm{min}$ ，静置 $15\mathrm{min}$ 分层。用 $100~\mathrm{mL}$ 烧杯收集下层水相，然后加入 $20~\mathrm{mL}$ 碳酸氢钠溶液重新提取上层油相。摇匀后静置，待两相分离后，用烧杯收集下层水相，合并两次水相提取溶液。用注射器或真空泵(流速 $10~\mathrm{mL / min} \sim 20~\mathrm{mL / min}$ ) 使提取液通过固相萃取 $\mathrm{C_{18}}$ 柱。收集滤液于 $50~\mathrm{mL}$ 容量瓶中，加入 $1.0~\mathrm{mL}~50\%$ 乙酸溶液，用水定容。取 $1~\mathrm{mL} \sim 2~\mathrm{mL}$ 滤液用滤膜过滤。对苯二甲酸的标准工作液浓度分别为 $0.0~\mathrm{mg / kg}, 0.8~\mathrm{mg / kg}, 1.6~\mathrm{mg / kg}, 4.0~\mathrm{mg / kg}, 8.0~\mathrm{mg / kg}, 16.0~\mathrm{mg / kg}$ ；内标浓度均为 $8.0~\mathrm{mg / kg}$ 。  
3.4.5 邻苯二甲酸内标储备溶液 $(1000\mathrm{mg / L})$ ：准确称取 $0.1\mathrm{g}$ （精确至 $0.1\mathrm{mg}$ )邻苯二甲酸标准品，用 $10~\mathrm{mL}$ 异丙醇溶解，然后将溶液转移至 $100~\mathrm{mL}$ 容量瓶中并以异丙醇定容。有效期为6个月。  
3.4.6 邻苯二甲酸内标中间溶液 $(200\mathrm{mg / L})$ ：取 $5\mathrm{mL}$ 邻苯二甲酸内标储备溶液于 $25~\mathrm{mL}$ 容量瓶中，用甲醇定容。

# 4 仪器设备

4.1 高效液相色谱仪：配紫外检测器和 $10\mu \mathrm{L}$ 定量环。  
4.2 分析天平：感量 $0.1\mathrm{mg},0.1\mathrm{g}$   
4.3 pH计。  
4.4 固相萃取 $\mathrm{C_{18}}$ 柱：十八烷基硅烷(ODS) $400\mathrm{mg}$   
4.5 水系微孔滤膜： $0.45 \mu \mathrm{m}$ 。

# 5 分析步骤

# 5.1 样品迁移试验

按照GB5009.156及GB31604.1的要求，对样品进行迁移试验，得到食品模拟物试液。如果得到的食品模拟物试液不能马上进行下一步试验，应将食品模拟物试液于 $4^{\circ}\mathrm{C}$ 冰箱中避光保存。

应将所得食品模拟物试液冷却或恢复至室温后进行下一步试验。

# 5.2 试液制备

# 5.2.1 水基、酸性、酒精类食品模拟物试液的制备

移取 $50.0\mathrm{mL}$ 从迁移试验中得到的水基食品模拟物于预先盛有 $2.0\mathrm{mL}$ 邻苯二甲酸内标中间溶液的 $100~\mathrm{mL}$ 锥形瓶中，混合均匀，取 $1\mathrm{mL}\sim 2\mathrm{mL}$ 用滤膜过滤待测。

# 5.2.2 橄榄油模拟物试液的制备

称取 $50.0\mathrm{g}$ 从迁移试验中得到的橄榄油介质食品模拟物于 $250~\mathrm{mL}$ 分液漏斗中，加入 $2.0~\mathrm{mL}$ 内标中间溶液，混合均匀后量取 $50~\mathrm{mL}$ 的正庚烷于分液漏斗中。再按3.4.4“再加入 $20~\mathrm{mL}$ 碳酸氢钠溶液……取 $1\mathrm{mL} \sim 2\mathrm{mL}$ 滤液用滤膜过滤”步骤操作，溶液待测。

# 5.3 空白试液的制备

按照5.2所述步骤处理没有与食品接触材料接触的食品模拟物。

# 5.4 测定

# 5.4.1 液相色谱测定参考条件

液相色谱测定参考条件列出如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或相当者；  
b）流动相：A： $150~\mathrm{mL}$ 乙酸钠缓冲液 $(3.2.5) + 750~\mathrm{mL}$ 水；B：甲醇；

测定水基、酸性、酒精类模拟物试液时：流动相为 $\mathrm{A} + \mathrm{B} = 80 + 20$ （体积比）；测定橄榄油模拟物试液时：流动相为 $\mathrm{A} + \mathrm{B} = 85 + 15$ （体积比）；

c) 流速: $1.5 \mathrm{~mL} / \mathrm{min}$ ;   
d） 柱温：室温；  
e) 进样量: $10 \mu \mathrm{L}$ ;   
f）紫外检测器：波长 $242\mathrm{nm}$

# 5.4.2 绘制标准工作曲线

按照5.4.1所列测定条件，对标准工作溶液依次进样测定。以标准工作溶液中对苯二甲酸浓度为横坐标，单位以“ $\mathrm{mg / L}$ 或 $\mathrm{mg / kg}$ ”表示，以对苯二甲酸/邻苯二甲酸的峰面积比值为纵坐标，分别绘制标准工作曲线。标准溶液色谱图参见附录A。

# 5.4.3 试液测定

对空白溶液和食品模拟物试液依次进样，扣除空白值，得到对苯二甲酸/邻苯二甲酸的峰面积比值。

# 6 分析结果的表述

由标准曲线得到试样溶液中对苯二甲酸的浓度，按GB5009.156进行迁移量计算，得到食品接触材料及制品中对苯二甲酸的迁移量。计算结果保留两位有效数字。

# 7 确证

当上述结果超过特定迁移限量值(SML)时，用液相色谱-串联质谱法或其他有效方法确证。液相色

谱-串联质谱法分析参考条件及色谱图参见附录B和附录C。

# 8 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 9 其他

水基、酸性、酒精类模拟物中对苯二甲酸的方法检出限为 $0.3\mathrm{mg / L}$ ；橄榄油模拟物中对苯二甲酸的方法检出限为 $0.3\mathrm{mg / kg}$ 。

水基、酸性、酒精类模拟物中对苯二甲酸的方法定量限为 $1.0\mathrm{mg / L}$ ；橄榄油模拟物中对苯二甲酸的方法定量限为 $1.0\mathrm{mg / kg}$ 。

# B.1 液相色谱条件

液相色谱条件列出如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $150~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或同等性能色谱柱；

# B.3 定性测定

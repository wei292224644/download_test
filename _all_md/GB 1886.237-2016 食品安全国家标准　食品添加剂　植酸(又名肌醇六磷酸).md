# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试剂中所用标准溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 硫酸。  
A.2.1.2 过氧化氢。  
A.2.1.3 氨水溶液： $2 + 3$   
A.2.1.4 氯化钙溶液： $75\mathrm{g / L}$   
A.2.1.5 钼酸铵溶液： $25\mathrm{g / L}$

# A.2.2 沉淀试验

取试样适量，用氨水溶液中和，加氯化钙溶液数滴，即出现白色沉淀。

# A.2.3 显色试验

取试样 $0.2\mathrm{g}$ ，加硫酸 $1\mathrm{mL}$ ，加热至完全炭化，再滴加过氧化氢至无色透明，加钼酸铵溶液即显黄色。

# A.3 植酸含量的测定

# A.3.1 方法提要

植酸经高氯酸和硝酸消解生成磷酸，在硝酸介质中，磷酸与喹钼柠酮试剂反应生成磷钼酸喹啉黄色沉淀，通过过滤、洗涤、干燥后称重，即可计算出试样中的植酸含量。

# A.3.2 试剂和材料

A.3.2.1 硝酸。

A.3.2.2 硝酸溶液： $1 + 1$

A.3.2.3 喹钼柠酮试剂：称取钼酸钠 $70\mathrm{g}$ 加水 $150~\mathrm{mL}$ 溶解，为溶液I，称取柠檬酸 $60\mathrm{g}$ 加水 $150~\mathrm{mL}$ 和硝酸 $85~\mathrm{mL}$ 溶解，在搅拌下将溶液I倒入其中，为溶液Ⅱ；量取水 $100~\mathrm{mL}$ ，加入硝酸 $35~\mathrm{mL}$ 和喹啉 $5~\mathrm{mL}$ ，将该混合液缓缓倒入溶液Ⅱ中，放置 $24\mathrm{h}$ ，过滤于 $1\mathrm{L}$ 的容量瓶中，加丙酮 $280~\mathrm{mL}$ ，用水稀释至刻度，混匀后贮存于塑料瓶中，有效期半年。

# A.3.3 仪器和设备

砂芯玻璃漏斗或坩埚：G5，滤片孔径 $2\mu \mathrm{m}\sim 4\mu \mathrm{m}40\mathrm{mL}$

# A.3.4 分析步骤

称取试样约 $0.3\mathrm{g}$ （精确至 $0.0002\mathrm{g}$ ，置于 $300~\mathrm{mL}$ 锥形瓶中，加硝酸 $10~\mathrm{mL}$ 、高氯酸 $5\mathrm{mL}$ ，在通风橱内置于电炉上加热，逐渐升温使白烟逸出直至溶液透明近干，冷却。用少量水冲洗瓶壁，将其定量转移至 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，为溶液A。

量取溶液 $\mathrm{A}20\mathrm{mL}$ 于 $300~\mathrm{mL}$ 锥形瓶中，加水 $80~\mathrm{mL}$ 、硝酸溶液 $10~\mathrm{mL}$ 和喹钼柠酮试剂 $50~\mathrm{mL}$ 于沸水中加热陈化至溶液澄清，冷却至室温。在抽滤装置上以倾泻法过滤，将沉淀定量转移至已恒重的砂芯玻璃漏斗中，用少量水多次洗涤沉淀，将该漏斗置于 $120^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 烘箱内干燥 $1\textrm{h}$ ，于干燥器内冷却至室温，称量。

# A.3.5 结果计算

植酸 $\mathrm{(C_6H_{18}O_{24}P_6)}$ 的质量分数 $\varpi$ ，按式(A.1)计算：

$$
w = \frac{m_{1}\times 0.01400\times3.552}{m_{2}\times\frac{20}{100}}\times 100\% \quad \dots \tag{A.1}
$$

式中：

$m_{1}$ ——沉淀物的质量，单位为克 $(\mathrm{g})$ ；

0.014 00——磷钼酸喹啉与磷的换算系数；

3.552 磷与植酸的换算系数；

$m_{2}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

20 ——被测试样溶液的体积；

100 ——试样溶液的总体积。

所得结果保留至1位小数。

取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.3\%$ 。

# A.4 无机磷的测定

# A.4.1 方法提要

磷钼蓝比色法：在一定酸度和还原剂存在下，植酸溶液中游离磷酸盐生成的磷钼黄被还原成磷钼蓝。用目视比色法测定无机磷的限量值。

# A.4.2 试剂和材料

A.4.2.1 氨水溶液： $2 + 3$

A.4.2.2 硫酸溶液： $1 + 6$

A.4.2.3 钼酸铵溶液： $25\mathrm{g / L}$

A.4.2.4 抗坏血酸溶液： $100\mathrm{g / L}$ ，现用现配。

A.4.2.5 磷标准溶液: $1 \mathrm{~mL}$ 含 $0.01 \mathrm{mg} \mathrm{P}$ 。取 $0.439 \mathrm{~g}$ 磷酸二氢钾, 溶于水, 移入 $1000 \mathrm{~mL}$ 容量瓶中, 稀释至刻度, 使用时将溶液稀释 10 倍。

# A.4.3 分析步骤

称取试样 $0.1\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，置于 $25~\mathrm{mL}$ 比色管中，加水 $5\mathrm{mL}$ ，用氨水溶液中和至 $\mathrm{pH} = 7$ (pH试纸检验)。加硫酸溶液 $1\mathrm{mL}$ 、钼酸铵溶液 $1\mathrm{mL}$ 、抗坏血酸溶液 $1\mathrm{mL}$ ，用水稀释至刻度，于冰浴 $(0^{\circ}\mathrm{C}\sim$ $5^{\circ}\mathrm{C})$ 中放置 $30\mathrm{min}$ 后，目视比色。试样所呈蓝色不得深于限量标准。

限量标准是取磷标准溶液(1 mL 含 0.01 mg P)2.0 mL, 加水至 5 mL, 与试样液同时同样处理。

# A.5 氯化物的测定

# A.5.1 试剂和材料

氨水溶液： $2 + 3$

# A.5.2 分析步骤

称取试样 $0.5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，置于 $25~\mathrm{mL}$ 比色管中，加水 $20~\mathrm{mL}$ ，用氨水溶液中和至 $\mathrm{pH} = 7$ (pH试纸检验)。其余按照GB/T9729的规定进行。

限量标准是取氯化物标准溶液(1 mL含 $0.1\mathrm{mgCl}$ ) $1.0\mathrm{mL}$ 加水至 $20\mathrm{mL}$ , 与试样液同时同样处理。

# A.6 硫酸盐的测定

# A.6.1 试剂和材料

氨水溶液： $2 + 3$

# A.6.2 分析步骤

称取试样 $0.5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ), 置于 $25\mathrm{mL}$ 比色管中, 加水 $20\mathrm{mL}$ , 用氨水溶液中和至 $\mathrm{pH} = 7$ (pH试纸检验)。其余按照GB/T9728的规定进行。

限量标准是取硫酸盐标准溶液 $(1\mathrm{mL}$ 含 $0.1\mathrm{mgSO}_4)1.0\mathrm{mL}$ ，加水至 $20~\mathrm{mL}$ ，与试样液同时同样处理。

# A.7 钙盐的测定

# A.7.1 方法提要

草酸钙比浊法：在乙酸介质中，微量钙离子与草酸根离子生成难溶的草酸钙悬浮体，用目视比浊法测定钙的限量值。

# A.7.2 试剂和材料

A.7.2.1 乙酸。

A.7.2.2 氨水溶液： $2 + 3$

A.7.2.3 草酸铵溶液： $40~\mathrm{g / L}$

A.7.2.4钙标准溶液： $1\mathrm{mL}$ 含 $0.1\mathrm{mgCa}$ 。准确称取于 $105~\mathrm{^\circ C}\sim 110~^\circ \mathrm{C}$ 烘干的碳酸钙 $0.25\mathrm{g}$ ，加入乙酸溶液溶解，加水稀释定容至 $1000~\mathrm{mL}$

# A.7.3 分析步骤

称取试样 $0.5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ), 置于 $25\mathrm{mL}$ 比色管中, 加水 $10\mathrm{mL}$ , 用氨水溶液中和至 $\mathrm{pH} = 7$ (pH试纸检验)。加乙酸 $1\mathrm{mL}$ , 草酸铵溶液 $5\mathrm{mL}$ , 用水稀释至刻度, 放置 $10\mathrm{min}$ 后比浊。所呈浊度不得大于标准。

限量标准是取钙标准溶液 $(1\mathrm{mL}$ 含 $0.1\mathrm{mgCa})1.0\mathrm{mL}$ ，加水至 $10~\mathrm{mL}$ ，与试样液同时同样处理。

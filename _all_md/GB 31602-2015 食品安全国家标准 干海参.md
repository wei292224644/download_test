# 食品安全国家标准

# 干海参

# 1 范围

本标准适用于干海参。

# 2 术语和定义

# 2.1 干海参

以刺参等海参为原料，经去内脏、煮制、盐渍（或不盐渍）、脱盐（或不脱盐）、干燥等工序制成的产品；或以盐渍海参为原料，经脱盐（或不脱盐）、干燥等工序制成的产品。

注：在刺参收获的季节，通常的做法是将鲜刺参煮制、盐渍，制成半成品（即盐渍海参），贮存于冷库中，做为干海参生产的原料贮备。

# 2.2 复水后干重率

干海参复水后，再烘干所得到的干物质质量的百分率。

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所有试剂的纯度应在分析纯以上，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，实验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种试剂配制时，均指水溶液。

# A.2 样品前处理

A.2.1 取至少3只干海参，放入高速粉碎机粉碎( $25000\mathrm{r / min}$ ， $10~s / \text{次} \sim 15~s / \text{次}$ )，应多次粉碎，至试样全部通过 $830\mu \mathrm{m}(20$ 目)筛，处理后的试样应密封、备用。  
A.2.2 经本方法处理的样品，主要用于蛋白质、水分、盐分等指标的检测。

# A.3 干海参的复水

# A.3.1 预浸泡

取2只～3只干海参，称重约 $10\mathrm{g}(m_{1}$ ，精确至 $0.01\mathrm{g})$ ，置于 $1000~\mathrm{mL}$ 烧杯中，倒入水（水量约为海参质量的50倍，并应浸没参体)，再盖上表面皿，室温浸泡 $24\mathrm{h}$ 。

# A.3.2 清洗

在浸泡液中剖开海参体，清洗海参体附着的泥砂，去除嘴部石灰质后，切成宽约 $5\mathrm{mm}$ 条状；将海参体、泥砂及嘴部石灰质均保留在原浸泡液中。

# A.3.3 水煮

将经A.3.2处理的试样及浸泡液于原烧杯中，盖上表面皿，大火煮沸，然后调至小火，保持沸腾继续煮 $30\mathrm{min}$ ，晾至室温后，置于 $0^{\circ}\mathrm{C} \sim 10^{\circ}\mathrm{C}$ 冰箱中，放置 $20\mathrm{h}$ 。煮沸及浸泡过程中应保持水量浸没参体。

# A.3.4 试样

A.3.4.1 将A.3.3处理的浸出液及海参体全部倒入 $1000~\mathrm{mL}$ 量筒中，定容至 $600~\mathrm{mL}$ ，混匀。  
A.3.4.2取出海参放入烧杯中，加入 $600~\mathrm{mL}$ 水，按A.3.3的方法再水煮一次、将在冰箱放置后的海参取出，用滤纸吸去表面水分，绞碎备用于污染物及兽药残留项目的检测。  
A.3.4.3 过滤浸泡液，将其全部转移至无灰滤纸中，用于含砂量的检测。  
A.3.4.4所得滤液用于水溶性总糖的检测。当测试液中糖含量高时，应用水稀释后再测；测试液中适宜的糖含量为 $30~\mu \mathrm{g / mL}\sim 70~\mu \mathrm{g / mL}$

# A.4 干海参中复水后干重率的检验方法

# A.4.1 原理

将干海参复水，去除海参体内各种水溶性物质，再将海参体烘干所得到的干物质的质量分数。

# A.4.2 仪器和设备

A.4.2.1 烧杯：高型， $1000~\mathrm{mL}$   
A.4.2.2 称量瓶。  
A.4.2.3 电热恒温干燥箱。  
A.4.2.4 干燥器：内附有效干燥剂。  
A.4.2.5 天平：感量为 $0.1\mathrm{mg}$

# A.4.3 分析步骤

A.4.3.1取整只干海参，称重 $(m_{2}$ ，精确至 $0.0001\mathrm{g})$ ，置于 $1000~\mathrm{mL}$ 烧杯中，倒入水（水量约为海参质量的50倍，并应浸没参体)，盖上表面皿，室温浸泡 $24\mathrm{h}$ ；剖开海参体，在原浸泡液中清洗参体内附着的泥砂，仔细去除嘴部石灰质。  
A.4.3.2 将洗好的海参, 切成宽约 $5 \mathrm{~mm}$ 的条, 放入洁净的烧杯中, 倒入水 (水量约为海参质量的 50 倍,并应浸没参体), 盖上表面皿, 大火煮沸, 然后调至小火, 保持微沸 $30 \mathrm{~min}$ 后, 凉至室温, 补水至原刻度,置于 $0^{\circ} \mathrm{C} \sim 5^{\circ} \mathrm{C}$ 冰箱中, 放置 $18 \mathrm{~h} \sim 20 \mathrm{~h}$ 。  
A.4.3.3 用已恒重的滤纸过滤后，再将试样切为约 $3\mathrm{mm}\times 3\mathrm{mm}$ 小块，连同滤纸置入已恒重的称量瓶中，于 $101^{\circ}\mathrm{C}\sim 105^{\circ}\mathrm{C}$ 烘箱中烘 $8\mathrm{h}$ 以上(至恒重)，于干燥器中冷却 $30\mathrm{min}$ ，称重 $(m_3$ ，精确至 $0.0001\mathrm{g})$ 。

# A.4.4 分析结果的表述

复水后干重率按式(A.1)计算，计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

$$
X _ {1} = \frac {m _ {3}}{m _ {2}} \times 1 0 0 \quad \dots \tag {A.1}
$$

式中：

$X_{1}$ ——试样中复水后干重率，单位为克每 $100$ 克 $(\mathrm{g} / 100\mathrm{g})$ ；

$m_{3}$ ——试样干燥后的质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

# A.4.5 精密度

在重复性条件下获得的两次独立测定结果的绝对偏差不得超过算术平均值的 $5\%$ 。

# A.5 干海参中含砂量的检验方法

# A.5.1 原理

将干海参浸泡清洗后，进行过滤，所得残渣灼烧后得到的干物质的质量分数。

# A.5.2 仪器和设备

A.5.2.1 坩埚。  
A.5.2.2 电炉。  
A.5.2.3 高温炉。  
A.5.2.4 天平：感量为 $0.1\mathrm{mg}$

# A.5.3 分析步骤

将A.3.4.3得到的过滤物连同无灰滤纸包好置入已干燥称重的坩埚中，将坩埚置于电炉上炭化，再

移入高温炉中， $550^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，至颜色变白。取出坩埚，在空气中冷却 $1\mathrm{min}$ 后，放入干燥器中冷却 $30\mathrm{min}$ ，称重 $(m_4$ ，精确至 $0.0001\mathrm{g})$ 。

# A.5.4 分析结果的表述

含砂量按式(A.2)计算，计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

$$
X _ {2} = \frac {m _ {4}}{m _ {1}} \times 1 0 0 \quad \dots \tag {A.2}
$$

式中：

$X_{2}$ ——试样中的含砂量，单位为克每100克 $(\mathrm{g} / 100\mathrm{g})$

$m_{4}$ ——灼烧后残渣的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

# A.5.5 精密度

在重复性条件下获得的两次独立测定结果的绝对偏差不得超过算术平均值的 $5\%$ 。

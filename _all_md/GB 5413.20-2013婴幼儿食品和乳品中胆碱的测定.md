# 前言

本标准代替GB/T5413.20—1997《婴幼儿配方食品和乳粉胆碱的测定》。

本标准与GB/T5413.20—1997相比，主要变化如下：

——修改了标准的名称；

——修改了用于酶反应的显色剂成分；

——增加了第二法雷氏盐分光光度法。

# 食品安全国家标准

# 婴幼儿食品和乳品中胆碱的测定

# 1 范围

本标准规定了婴幼儿食品和乳品中胆碱的测定方法。

本标准适用于婴幼儿食品和乳品中胆碱的测定。

# 第一法 酶比色法

# 2 原理

试样中的胆碱经酸水解后变成游离态的胆碱，再经酶氧化后与显色剂反应生成有色物质，其颜色的深浅在一定浓度范围内与胆碱含量成正比。

# 3 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 三羟甲基氨基甲烷 $\left[\left(\mathrm{CH}_{2} \mathrm{OH}\right)_{3} \mathrm{CNH}_{2}\right]$   
3.1.2 苯酚 $(\mathrm{C_6H_5OH})$   
3.1.3 浓盐酸(HCl)。  
3.1.4 氢氧化钠(NaOH)。  
3.1.5 胆碱氧化酶：置于 $-20^{\circ} \mathrm{C}$ 保存。  
3.1.6 过氧化物酶：置于 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 保存。  
3.1.7 4-氨基安替比林 $\left(\mathrm{C}_{11} \mathrm{H}_{13} \mathrm{~N}_{3} \mathrm{O}\right)$ 。  
3.1.8 磷脂酶D：置于-20℃保存。

# 3.2 试剂配制

3.2.1 盐酸 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $85~\mathrm{mL}$ 浓盐酸加水稀释至 $1000~\mathrm{mL}$ 。  
3.2.2 盐酸(3mol/L)：量取 $125\mathrm{mL}$ 浓盐酸加水稀释至 $500~\mathrm{mL}$ 。  
3.2.3 Tris缓冲溶液（0.05mol/L）： $\mathrm{pH} = 8.0\pm 0.2$

称取 $6.057 \mathrm{~g}$ 三羟甲基氨基甲烷溶入 $500 \mathrm{~mL}$ 蒸馏水中, 用 $1 \mathrm{~mol} / \mathrm{L}$ 盐酸调 $\mathrm{pH}$ 至 $8.0 \pm 0.2$ , 用蒸馏水定容至 $1000 \mathrm{~mL}$ 。此溶液在 $4^{\circ} \mathrm{C}$ 冰箱中可保存一个月。

3.2.4 用于酶反应的显色剂：取 $100 \sim 120$ 活力单位的胆碱氧化酶、 $250 \sim 280$ 活力单位的过氧化物酶、75个~100个活力单位的磷脂酶 D、 $15 \mathrm{mg} 4$ -氨基安替比林， $50 \mathrm{mg}$ 苯酚置于 $100 \mathrm{~mL}$ 的容量瓶中，用 $0.05 \mathrm{~mol} / \mathrm{L}$ Tris 缓冲溶液稀释至刻度。临用时配制。  
3.2.5 氢氧化钠溶液 $(500\mathrm{g / L})$ ：称取 $500\mathrm{g}$ 氢氧化钠，溶于水并稀释至 $1000~\mathrm{mL}$ 。

# 3.3 标准品

胆碱酒石酸氢盐标准品（ $\mathrm{C_9H_{19}NO_7}$ ）：纯度 $\geq 99\%$ 。

# 3.4 标准溶液配制

3.4.1 胆碱氢氧化物标准贮备溶液（ $2.5\mathrm{mg/mL}$ ）：称取在 $102^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 烘至恒重的胆碱酒石酸氢盐 523 mg 置于 $100\mathrm{mL}$ 容量瓶中，用蒸馏水稀释至刻度。冷藏于 $4^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 冰箱中，保存不超过 1 周。  
3.4.2 胆碱氢氧化物标准工作溶液（ $250\mu \mathrm{g} / \mathrm{mL}$ ）：吸取 $10.0\mathrm{mL}$ 标准贮备溶液于 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度。临用时配制。

# 4 仪器和设备

4.1 天平：感量为 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$ 。  
4.2 恒温水浴锅：温度可控制在 $70^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 和 $37^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。  
4.3 pH计：精度0.01。  
4.4 分光光度计。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固体试样

称取 $5\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）混合均匀的试样，于 $100~\mathrm{mL}$ 的锥形瓶中，加入 $30~\mathrm{mL}$ 盐酸溶液（3.2.1）。

# 5.1.2 液体试样

称取 $20\mathrm{g}$ （精确到 $0.01\mathrm{g}$ )混合均匀的试样，于 $100~\mathrm{mL}$ 的锥形瓶中，加入 $10~\mathrm{mL}$ 盐酸溶液（3.2.2）。

# 5.1.3 水解

将装有试样的容器放在 $70^{\circ} \mathrm{C}$ 水浴中，加塞混匀，水解 $3 \mathrm{~h}$ （每隔 $30 \mathrm{~min}$ 振摇一次），冷却。用氢氧化钠溶液调 $\mathrm{pH}$ 为 $3.5 \sim 4.0$ ，转入 $50 \mathrm{~mL}$ 容量瓶中，用蒸馏水定容至刻度。

# 5.1.4 过滤

过滤水解液（5.1.3）。滤液应是澄清的，否则，用 $0.45\mu \mathrm{m}$ 的滤膜再次过滤。滤液放在 $4^{\circ}\mathrm{C}$ 的冰箱中可以保存3d。

# 6 分析结果的表述

# 6.1 净吸光值的计算

通常新鲜配制的试剂会产生轻微颜色，且由于水解作用滤液也不是无色的，为了除去这些干扰因素，应该从总吸光值中减去各自的空白值（管A和管B）。

试样净吸光值计算如公式（1）所示：

$$
A = A _ {\mathrm {t o t}} - A _ {\mathrm {b l}} - A _ {\mathrm {e x}} \tag {1}
$$

式中：

$A$ ——试样净吸光值；

$A_{tot}$ ——总吸光值（管C）；

$A_{bl}$ ——试剂吸光值（管A）；

$A_{ex}$ ——滤液吸光值（管B）。

$A_{\mathrm{bl}}$ 和 $A_{\mathrm{ex}}$ 不应大于总吸光值的 $20\%$ ，对于标准曲线， $A_{\mathrm{ex}} = 0$ 。

# 6.2 胆碱含量的计算

在标准曲线上查出净吸光值的位置，并记下相应的浓度 $c$ ，以每 $100\mathrm{g}$ 试样中胆碱氢氧化物的毫克数表示胆碱的含量 $(X)$ ，单位为 $(\mathrm{mg} / 100\mathrm{g})$ ，试样中的胆碱氢氧化物含量按公式（2）计算：

$$
X = \frac {c \times V \times 1 0 0}{m \times 1 0 0 0} \quad \dots \tag {2}
$$

式中：

$X$ ——试样中的胆碱氢氧化物含量，单位为毫克每百克 $(\mathrm{mg} / 100 \mathrm{~g})$ ；

$c$ ——自标准曲线上查得的胆碱氢氧化物的浓度, 微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ;

$V$ ——水解液被稀释的体积（通常为 $50 \mathrm{~mL}$ ），单位为毫升（ $\mathrm{mL}$ ）；

$m$ ——试样的质量，单位为克（g）；

1000——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留整数位。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $8\%$ 。

# 8 其他

方法检出限为 $1\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $3\mathrm{mg} / 100\mathrm{g}$ 。

# 第二法 雷氏盐分光光度法

# 9 原理

试样中的胆碱用氢氧化钡-甲醇-三氯甲烷混合溶液水解抽提，经弗罗里硅土层析净化，与雷纳克铵盐溶液形成粉红色的胆碱雷纳克铵盐，用丙酮溶解洗脱，于 $526\mathrm{nm}$ 测定吸收值。在一定浓度范围内，胆碱雷纳克铵盐颜色的深浅与其含量成正比。

# 10 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 10.1 试剂

10.1.1 弗罗里硅土：100目～200目， $650^{\circ}\mathrm{C}$ 活化。

10.1.2 甲醇（ $\mathrm{CH}_3\mathrm{OH}$ ）。

10.1.3 三氯甲烷（ $\mathrm{CHCl}_3$ ）。

10.1.4 氢氧化钡 $\mathrm{[Ba(OH)_2\cdot 8H_2O]}$

10.1.5 冰乙酸（ $\mathrm{CH}_3\mathrm{COOH}$ ）。

10.1.6 乙酸甲酯（ $\mathrm{C_3H_6O_2}$ ）。

10.1.7 丙酮 $\mathrm{(CH_3COCH_3)}$

10.1.8 雷纳克铵盐 $(\mathrm{C_4H_{10}CrN_7S_4})$ 。

# 10.2 试剂配制

10.2.1 饱和氢氧化钡-甲醇-三氯甲烷溶液：称取 $6 \mathrm{~g}$ 氢氧化钡溶于 $100 \mathrm{~mL}$ 甲醇中，放入超声波中溶解后，加入 $10 \mathrm{~mL}$ 三氯甲烷，混匀。  
10.2.2 冰乙酸-甲醇溶液 $(1 + 10)$ ：1体积冰乙酸与10体积甲醇混合。  
10.2.3 雷纳克铵盐溶液：称取 $0.25\mathrm{g}$ 雷纳克铵盐，加入 $10\mathrm{mL}$ 水中，放入超声波中溶解后过滤，临用时配制。

# 10.3 标准品

胆碱酒石酸氢盐 $\mathrm{(C_9H_{19}NO_7)}$ ：纯度 $\geq 99\%$

# 10.4 标准溶液配制

胆碱酒石酸氢盐标准溶液 $(1\mathrm{g / L})$ ：准确称取在 $102^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 烘至恒重的胆碱酒石酸氢盐 $0.1000\mathrm{g}$ ，用蒸馏水定容至 $100~\mathrm{mL}$ 容量瓶中。冷藏于 $4^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 冰箱中，保存不超过一周。

# 11 仪器和设备

11.1 天平：感量为 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$ 。  
11.2 回流装置： $250\mathrm{mL}$ 磨口锥形瓶及回流装置。  
11.3 恒温水浴锅：温度可控制在 $79^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。  
11.4 层析柱：长约 $10\mathrm{cm}$ 、内径 $1\mathrm{cm}$ 的带 $50~\mathrm{mL}$ 杯口的玻璃柱。  
11.5 分光光度计。

# 12 分析步骤

# 12.1 试样制备

称取固体试样 $10\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ), 称取液体试样 $20\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 于磨口锥形瓶中, 加入 $50\mathrm{mL}$ 氢氧化钡-甲醇-三氯甲烷提取液。混合均匀后接入回流装置, 于 $79 \pm 2^{\circ}\mathrm{C}$ 的水浴锅内水解抽提 $4\mathrm{h}$ 。每隔 $1\mathrm{h}$ 震荡一次, 以避免试样结块。水解抽提结束后, 取出锥形瓶冷却至室温, 过滤。滤渣用冰乙酸-甲醇混合液洗涤3次 $\sim 4$ 次, 洗液一并收集于 $100\mathrm{mL}$ 容量瓶中。用甲醇定容至刻度, 混匀。

# 12.2 标准曲线的绘制

# 12.2.1 层析柱制备

用乳胶管连接层析柱与滴头，用少量脱脂棉堵住层析柱底部，倒入约 $5\mathrm{cm}$ 左右高的弗罗里硅土，用甲醇浸湿，备用。

# 12.2.2 层析

分别吸取胆碱标准溶液 $0\mathrm{mL}$ 、 $1.0\mathrm{mL}$ 、 $2.0\mathrm{mL}$ 、 $3.0\mathrm{mL}$ 、 $4.0\mathrm{mL}$ 、 $5.0\mathrm{mL}$ 注入层析柱中，当溶液完全进入柱床后，依次用 $5\mathrm{mL}$ 和 $10\mathrm{mL}$ 甲醇， $20\mathrm{mL}$ 乙酸甲酯洗涤层析柱。再加入 $5\mathrm{mL}$ 雷纳克铵盐溶液，用适量的冰乙酸洗去过量的雷纳克铵盐，直至层析柱上无雷纳克铵盐附着处呈现硅土原有的白色。用丙酮洗脱粉红色的胆碱雷纳克铵盐，收集于 $10\mathrm{mL}$ 的容量瓶中，用丙酮定容（如果洗脱液混浊需过 $0.45\mu \mathrm{m}$ 的滤膜）。在波长 $526\mathrm{nm}$ 处测定溶液的吸光值，以胆碱酒石酸氢盐含量为横坐标 $(m_x)$ ，吸光值为纵坐标绘制标准曲线。

# 12.3 试样的测定

吸取 $10 \mathrm{~mL}$ 试样水解液（12.1）于层析柱中，其余操作按照 12.2.2进行。自标准曲线上查得 $10 \mathrm{~mL}$ 试样水解液中胆碱酒石酸氢盐的含量。

# 13 分析结果的表述

试样中的胆碱以胆碱氢氧化物计，以毫克每百克（ $\mathrm{mg} / 100 \mathrm{~g}$ ）表示，按公式（3）计算：

$$
X = \frac {m _ {x}}{\frac {m}{1 0 0} \times V} \times 1 0 0 \times 0. 4 7 4 \dots \tag {3}
$$

式中：

$X$ ——试样中胆碱氢氧化物的含量, 单位为毫克每百克 $(\mathrm{mg} / 100 \mathrm{~g})$ ;

$m$ ——试样的质量，单位为克（g）；

$V$ ——层析时吸取试样水解液的体积, 单位为毫升 $(\mathrm{mL})$ ;

$m_{x}$ ——从标准曲线上查得胆碱酒石酸氢盐的含量，单位为毫克 $(\mathrm{mg})$ ；

0.474 胆碱酒石酸氢盐转化为胆碱氢氧化物的系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留整数位。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$

# 15 其他

方法检出限为 $2\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $5\mathrm{mg} / 100\mathrm{g}$ 。

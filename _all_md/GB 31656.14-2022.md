# 食品安全国家标准

# 水产品中27种性激素残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了水产品中27种性激素残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鱼、虾和蟹可食部分中己二烯雌酚、己烯雌酚、雌酮、己烷雌酚、雌二醇、雌三醇、炔雌醇、苯甲酸雌二醇、群勃龙、诺龙、雄烯二酮、勃地酮、睾酮、炔诺酮、美雄酮、甲基睾酮、康力龙、苯丙酸诺龙、丙酸睾酮、孕酮、21α-羟基孕酮、17α-羟基孕酮、甲羟孕酮、醋酸甲地孕酮、醋酸氯地孕酮、醋酸甲羟孕酮和左炔诺孕酮残留量的测定。

# 4 原理

试样中残留的性激素用乙酸乙酯-甲基叔丁基醚混合溶剂和乙酸乙酯依次提取，提取液经浓缩、 $\mathrm{C_{18}}$ 固相萃取柱净化，用液相色谱-串联质谱仪测定，内标法定量。

# 5 试剂与材料

以下所用的试剂，除特别注明外均为分析纯；水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ，色谱纯。  
5.1.2 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
5.1.3 甲基叔丁基醚 $(\mathrm{C}_5\mathrm{H}_{12}\mathrm{O})$ ：色谱纯。  
5.1.4 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ ：色谱纯。  
5.1.5 冰乙酸 $\mathrm{(CH_3COOH)}$   
5.1.6 三水乙酸钠 $\mathrm{(C_2H_3NaO_2\cdot 3H_2O)}$   
5.1.7 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。

# 5.2 溶液配制

5.2.1 乙酸-乙酸钠缓冲溶液：取三水乙酸钠 $21.5\mathrm{g}$ 、冰乙酸 $2.6\mathrm{mL}$ ，加水适量使溶解并稀释至 $500\mathrm{mL}$ ，混匀。  
5.2.2 $0.02\%$ 氨水：取氨水 $100~\mu \mathrm{L}$ 于 $500~\mathrm{mL}$ 水中，混匀。临用前配制。  
5.2.3 乙酸乙酯-甲基叔丁基醚溶液：取乙酸乙酯和甲基叔丁基醚等体积混合。

# 5.3 标准品

5.3.1 性激素：己二烯雌酚、己烯雌酚、雌酮、己烷雌酚、雌二醇、雌三醇、炔雌醇、苯甲酸雌二醇、群勃龙、

诺龙、雄烯二酮、勃地酮、睾酮、炔诺酮、美雄酮、甲基睾酮、康力龙、苯丙酸诺龙、丙酸睾酮、孕酮、 $21\alpha$ 羟基孕酮、 $17\alpha$ -羟基孕酮、甲羟孕酮、醋酸甲地孕酮、醋酸氯地孕酮、醋酸甲羟孕酮和左炔诺孕酮：含量 $\geqslant 97\%$ 具体见附录A。

5.3.2 内标：已烯雌酚- $\mathrm{D_8}$ 、雌酮- $\mathrm{D}_2$ 、雌二醇- ${}^{13}\mathrm{C}_2$ 、甲基睾酮- $\mathrm{D}_3$ 、孕酮- $\mathrm{D}_9$ 、左炔诺孕酮- $\mathrm{D}_6$ 和甲羟孕酮- $\mathrm{D}_3$ ：含量 $\geqslant 97\%$ ，具体见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液：取性激素类标准品各 $10\mathrm{mg}$ ，精密称定，分别用甲醇适量使溶解并定容于 $100~\mathrm{mL}$ 棕色容量瓶，配制成浓度均为 $100\mu \mathrm{g / mL}$ 的标准储备液。 $-18^{\circ}C$ 以下保存，有效期1年。  
5.4.2 混合标准中间液：准确量取各标准储备液适量，于 $100\mathrm{mL}$ 棕色容量瓶，用甲醇稀释配制成己二烯雌酚、己烯雌酚、雌酮、群勃龙、诺龙、雄烯二酮、勃地酮、睾酮、美雄酮、甲基睾酮、苯丙酸诺龙、丙酸睾酮、孕酮、 $21\alpha$ 羟基孕酮、 $17\alpha$ 羟基孕酮、甲羟孕酮、醋酸甲地孕酮、醋酸氯地孕酮和醋酸甲羟孕酮浓度均为 $1\mu \mathrm{g} / \mathrm{mL}$ ；己烷雌酚、雌二醇、左炔诺孕酮、炔诺酮、苯甲酸雌二醇和康力龙浓度均为 $2\mu \mathrm{g} / \mathrm{mL}$ ；雌三醇和炔雌醇浓度均为 $4\mu \mathrm{g} / \mathrm{mL}$ 的混合标准中间液。-18℃以下保存，有效期6个月。  
5.4.3 混合标准工作液：准确量取混合标准中间液 $10\mathrm{mL}$ ，于 $50~\mathrm{mL}$ 棕色容量瓶中，用甲醇稀释至刻度，配制成己二烯雌酚、己烯雌酚、雌酮、群勃龙、诺龙、雄烯二酮、勃地酮、睾酮、美雄酮、甲基睾酮、苯丙酸诺龙、丙酸睾酮、孕酮、 $21\alpha$ -羟基孕酮、 $17\alpha$ -羟基孕酮、甲羟孕酮、醋酸甲地孕酮、醋酸氯地孕酮和醋酸甲羟孕酮浓度均为 $0.2\mu \mathrm{g / mL}$ ；己烷雌酚、雌二醇、左炔诺孕酮、炔诺酮、苯甲酸雌二醇和康力龙浓度均为 $0.4\mu \mathrm{g / mL}$ ；雌三醇和炔雌醇浓度均为 $0.8\mu \mathrm{g / mL}$ 的混合标准工作液。 $-18^{\circ}C$ 以下保存，有效期3个月。  
5.4.4 内标标准储备液：取同位素内标物各 $10\mathrm{mg}$ ，精密称定，分别用甲醇适量使溶解并定容于 $50~\mathrm{mL}$ 棕色容量瓶，配制成浓度为 $200\mu \mathrm{g / mL}$ 的内标单标储备溶液。 $-18^{\circ}C$ 以下保存，有效期1年。  
5.4.5 混合内标工作液：分别准确量取内标储备液适量，于 $100\mathrm{mL}$ 棕色容量瓶中，用甲醇稀释至刻度，配制成雌二醇 ${}^{13}\mathrm{C}_2$ 浓度为 $2\mu \mathrm{g / mL}$ ；已烯雌酚- $\mathrm{D_8}$ 、雌酮- $\mathrm{D_2}$ 、甲基睾酮- $\mathrm{D_3}$ 、孕酮- $\mathrm{D_9}$ 、甲羟孕酮- $\mathrm{D_3}$ 和左炔诺孕酮- $\mathrm{D_6}$ 浓度均为 $1\mu \mathrm{g / mL}$ 的混合内标工作液。一 $18^{\circ}C$ 以下保存，有效期6个月。

# 5.5 材料

5.5.1 $\mathrm{C_{18}}$ 固相萃取柱： $500\mathrm{mg / 3mL}$ ，或相当者。  
5.5.2 微孔尼龙滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 高效液相色谱-串联质谱仪：配电喷雾离子源(ESI)。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 高速离心机： $10000\mathrm{r / min}$   
6.4 涡旋混合器。  
6.5 均质机。  
6.6 旋转蒸发仪。  
6.7 固相萃取装置。  
6.8 梨形瓶： $100~\mathrm{mL}$   
6.9 超声波清洗器。  
6.10 塑料离心管： $10~\mathrm{mL}$ 和 $50~\mathrm{mL}$

# 7 试样的制备与保存

# 7.1 试样的制备

按GB/T30891—2014附录B的要求制样。

a）取均质的供试样品，作为供试试样；  
b）取均质的空白样品，作为空白试样；  
c）取均质的空白样品，添加适宜浓度的标准溶液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取试料 $5 \mathrm{~g}$ (准确至 $\pm 0.05 \mathrm{~g}$ ), 于 $50 \mathrm{~mL}$ 离心管中, 准确加入 $30 \mu \mathrm{L}$ 混合内标工作液和乙酸-乙酸钠缓冲液 $10 \mathrm{~mL}$ , 涡旋 $30 \mathrm{~s}$ , 加乙酸乙酯-甲基叔丁基醚溶液 $10 \mathrm{~mL}$ , 涡旋 $1 \mathrm{~min}$ , 超声 $10 \mathrm{~min}$ , 涡旋混合 $1 \mathrm{~min}$ , 于 $8000 \mathrm{r} / \mathrm{min}$ 离心 $8 \mathrm{~min}$ , 取上层有机相于 $100 \mathrm{~mL}$ 梨形瓶; 残渣用乙酸乙酯 $10 \mathrm{~mL}$ 重复提取 1 次, 有机相合并至梨形瓶中, 于 $35^{\circ} \mathrm{C}$ 水浴旋转蒸发至干。对蒸干后有油脂残留的样品, 用氮气或空气将残留在油脂中的乙酸乙酯吹除干净。加入正己烷 $7 \mathrm{~mL}$ 溶解残留物, 备用。

# 8.2 净化

取 $\mathrm{C}_{18}$ 固相萃取柱，依次用甲醇、正己烷各 $5\mathrm{mL}$ 活化。取备用液，过柱，控制流速不超过1滴/s；用正己烷 $10\mathrm{mL}$ 淋洗，吹干；用甲醇 $10\mathrm{mL}$ 洗脱，收集洗脱液，于 $40^{\circ}\mathrm{C}$ 水浴旋转蒸发至干。准确加入甲醇 $0.85\mathrm{mL}$ ，涡旋混合 $1\mathrm{min}$ ，再准确加入水 $0.15\mathrm{mL}$ ，混合 $30\mathrm{s}, 8000\mathrm{r/min}$ ，离心 $6\mathrm{min}$ ，上清液用 $0.22\mu\mathrm{m}$ 尼龙滤膜过滤，供液相色谱串联质谱测定。

# 8.3 基质匹配标准工作曲线的制备

分别取 $10~\mu \mathrm{L}, 25~\mu \mathrm{L}, 50~\mu \mathrm{L}, 100~\mu \mathrm{L}, 250~\mu \mathrm{L}, 500~\mu \mathrm{L}$ 混合标准工作溶液于 $5\mathrm{mL}$ 玻璃离心管中，分别加混合内标工作液 $30~\mu \mathrm{L}$ ，吹干，加入经7.1和7.2步骤处理所得的基质空白溶液 $1.0~\mathrm{mL}$ ，混合 $1\mathrm{min}$ ，配制成激素浓度为 $2\mathrm{ng} / \mathrm{mL} \sim 400\mathrm{ng} / \mathrm{mL}$ 系列基质匹配标准工作液，上机测定。以标准物质特征离子色谱峰的峰面积与内标物特征离子色谱峰的峰面积比值为纵坐标、相应的浓度为横坐标，绘制基质标准工作曲线。求回归方程和相关系数。

# 8.5 测定法

# 8.5.1 定性测定

在同样的测试条件下，试料溶液中激素类药物的保留时间与标准工作液中激素类药物的保留时间之比偏差在 $\pm 2.5\%$ 以内；且检测到的离子的相对丰度，应当与浓度相当的校正标准溶液的离子相对丰度一致。其允许偏差应符合表3的要求。

# 8.5.2 定量测定

取试料溶液和基质匹配标准工作液，作单点或多点校准，按内标法定量。基质匹配标准工作液及试料溶液中目标物的响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，标准溶液特征离子质量色谱图见附录B。

# 8.6 空白试验

取空白试样，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times A _ {i S} ^ {\prime} \times C _ {S} \times C _ {i S} \times V}{A _ {i S} \times A _ {S} \times C _ {i S} ^ {\prime} \times m} \tag {1}
$$

式中：

$X$ ——试样中待测物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——试样溶液中待测物的峰面积；

$A_{iS}$ ——试样溶液中内标的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中待测物的峰面积；

$A_{i\mathrm{S}}^{\prime}$ ——标准溶液中内标的峰面积；

$C_{i\mathrm{S}}$ ——试样溶液中内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$C_{\mathrm{S}}$ ——标准溶液中待测物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$C_{iS}^{\prime}$ ——标准溶液中内标浓度的数值，单位为微克每升 $(\mu \mathrm{g / L})$

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ 供试试样质量的数值，单位为克 $(\mathrm{g})$

# 10 方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法己二烯雌酚、己烯雌酚、雌酮、群勃龙、诺龙、雄烯二酮、勃地酮、睾酮、美雄酮、甲基睾酮、苯丙酸诺龙、丙酸睾酮、孕酮、 $21\alpha$ 羟基孕酮、 $17\alpha$ 羟基孕酮、甲羟孕酮、醋酸甲地孕酮、醋酸氯地孕酮和醋酸甲羟孕酮检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1.0\mu \mathrm{g / kg}$ ；已烷雌酚、雌二醇、左炔诺孕酮、炔诺酮、苯甲酸雌二醇和康力龙的检测限为 $1.0\mu \mathrm{g / kg}$ ，定量限为 $2.0\mu \mathrm{g / kg}$ ；雌三醇和炔雌醇的检测限为 $2.0\mu \mathrm{g / kg}$ ，定量限为 $4.0\mu \mathrm{g / kg}$ 。

# 前言

本标准代替GB5009.210—2016《食品安全国家标准 食品中泛酸的测定》。

本标准与GB5009.210—2016相比，主要变化如下：

——增加了液相色谱-串联质谱法；

——修改了微生物法试样的前处理方法，并增加了微孔板测定方法。

# 食品安全国家标准

# 食品中泛酸的测定

# 1 范围

本标准规定了食品中泛酸的测定方法。

本标准第一法适用于婴幼儿配方食品（除特殊医学用途婴儿配方食品外）、婴幼儿辅助食品、乳制品、饮料、营养补充剂中泛酸的测定。

本标准第二法和第三法适用于食品中泛酸的测定。

# 第一法 液相色谱法

# 2 原理

试样用热水提取后，经 $\mathrm{C_{18}}$ 反相色谱柱分离，以保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 盐酸（HCl）。  
3.1.2 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。  
3.1.3 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$ 。  
3.1.4 七水合硫酸锌 $(\mathrm{ZnSO}_4\cdot 7\mathrm{H}_2\mathrm{O})$   
3.1.5 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.6 $\alpha$ -淀粉酶：酶活力 $\geqslant 1.5 \mathrm{U} / \mathrm{mg}$ 。

# 3.2 试剂配制

3.2.1 盐酸 $(0.1\mathrm{mol} / \mathrm{L})$ ：吸取 $8.3~\mathrm{mL}$ 盐酸至 $800~\mathrm{mL}$ 水中，加水稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.2 盐酸 $(1.0\mathrm{mol} / \mathrm{L})$ ：吸取 $8.3~\mathrm{mL}$ 盐酸至 $80~\mathrm{mL}$ 水中，加水稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.3 硫酸锌溶液 $(0.5\mathrm{mol} / \mathrm{L})$ ：称取 $14.4\mathrm{g}$ 七水合硫酸锌，加水溶解并稀释至 $100~\mathrm{mL}$ 。  
3.2.4 磷酸二氢钾溶液 $(0.02\mathrm{mol} / \mathrm{L})$ ：称取 $2.722\mathrm{g}$ 磷酸二氢钾，加 $500~\mathrm{mL}$ 水溶解，用磷酸调节 $\mathrm{pH}$ 至 $3.0\pm 0.1$ ，用水稀释至 $1000~\mathrm{mL}$ ，用 $0.45\mu \mathrm{m}$ 滤膜过滤。

# 3.3 标准品

D-泛酸钙标准品 $\mathrm{(C_{18}H_{32}CaN_2O_{10}}$ ,CAS号：137-08-6)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 泛酸标准储备液 $(500\mu \mathrm{g} / \mathrm{mL})$ ：准确称取 $136\mathrm{mg}$ D-泛酸钙标准品(精确至 $0.1\mathrm{mg}$ )，加水溶解并转入到 $250~\mathrm{mL}$ 容量瓶中，稀释至刻度，混匀。标准储备液 $-18^{\circ}\mathrm{C}$ 及以下避光保存，可保存3个月。  
3.4.2 泛酸标准中间溶液 $(100\mu \mathrm{g / mL})$ ：准确吸取标准储备溶液 $20.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加水稀释至刻度。临用现配。  
3.4.3 泛酸标准工作溶液：分别准确吸取泛酸标准中间液 $1.0\mathrm{mL},2.0\mathrm{mL},4.0\mathrm{mL},8.0\mathrm{mL},16.0\mathrm{mL}$ 和 $32.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加水稀释至刻度，得到浓度分别为 $1.0~\mu \mathrm{g / mL},2.0~\mu \mathrm{g / mL},4.0~\mu \mathrm{g / mL},$ $8.0~\mu \mathrm{g / mL},16.0~\mu \mathrm{g / mL}$ 和 $32.0~\mu \mathrm{g / mL}$ 的泛酸标准工作溶液。临用现配。

# 4 仪器和设备

4.1 天平：感量 $0.1\mathrm{mg}$ 。  
4.2 恒温振荡水浴箱：振荡频率 $100\mathrm{r / min}\pm 20\mathrm{r / min}$   
4.3 超声波振荡器。  
4.4 pH计：精度为 $\pm 0.01$   
4.5 离心机：转速 $\geq 8000\mathrm{r / min}$   
4.6 $0.45\mu \mathrm{m}$ 滤膜。  
4.7 高效液相色谱仪：带紫外检测器或二极管阵列检测器。

# 5 分析步骤

# 5.1 试样制备

固态试样粉碎并混合均匀。碳酸饮料需超声波去除二氧化碳，其他液态试样摇匀。

# 5.2 试样提取

# 5.2.1 饮料、营养素补充剂

准确称取或量取适量试样，精确至 $0.001\mathrm{g}$ ，一般固体试样 $0.2\mathrm{g} \sim 2\mathrm{g}$ ，液态试样 $10\mathrm{g} \sim 20\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 锥形瓶中，加入约 $30~\mathrm{mL}40^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ 温水超声提取 $20\mathrm{min}$ 。转入 $50~\mathrm{mL}$ 容量瓶中，用水稀释至刻度并充分混匀后，转入离心管， $8000\mathrm{r / min}$ 离心 $2\mathrm{min}$ ，取上清液过 $0.45\mu \mathrm{m}$ 滤膜，滤液待上机测定。

# 5.2.2 婴幼儿配方食品（除特殊医学用途婴儿配方食品外）、婴幼儿辅助食品、乳制品

准确称取适量试样，精确至 $0.001\mathrm{g}$ ，一般固态试样约 $5\mathrm{g}$ ，液态试样约 $20\mathrm{g}$ 。置于 $100\mathrm{mL}$ 锥形瓶中，加入约 $30\mathrm{mL}40^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 温水。不含淀粉试样，直接超声提取 $20\mathrm{min}$ 。含淀粉试样，加入 $\alpha$ -淀粉酶 $0.2\mathrm{g}$ ，振摇混匀，盖上瓶塞，在 $55^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}$ 水浴条件下振摇酶解 $30\mathrm{min}$ ，取出试样液，冷却至室温。

用 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸或 $1.0\mathrm{mol} / \mathrm{L}$ 盐酸调节 $\mathrm{pH}$ 至 $5.0 \pm 0.1$ ，加入 $5\mathrm{mL}0.5\mathrm{mol} / \mathrm{L}$ 硫酸锌溶液，充分混合。转入 $50~\mathrm{mL}$ 容量瓶中，用水稀释至刻度并充分混匀后，转入离心管， $8000\mathrm{r / min}$ 离心 $2\mathrm{min}$ ，取上清液过 $0.45\mu \mathrm{m}$ 滤膜，滤液待上机测定。

# 5.3 液相参考色谱条件

液相参考色谱条件如下：

a）色谱柱： $\mathrm{ODS - C_{18}}$ （粒径 $5\mu \mathrm{m},250\mathrm{mm}\times 4.6\mathrm{mm})$ 或具有同等性能的色谱柱。  
b） 柱温： $35\mathrm{^\circ C}\pm 0.5\mathrm{^\circ C}$   
c) 检测波长: $200 \mathrm{~nm}$ 。  
d) 流动相: $0.02 \mathrm{~mol} / \mathrm{L}$ 磷酸二氢钾溶液-乙腈 $(95 + 5)$ 。  
e) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ 。  
f）进样量： $10~\mu \mathrm{L}$ 或 $20~\mu \mathrm{L}$

# 5.4 测定

# 5.4.1 标准曲线测定

将泛酸标准工作液依次进行色谱分析(标准色谱图见附录A中图A.1)。以标准工作液浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。

# 5.4.2 试样溶液的测定

将试样测定液进行色谱分析，根据试样峰面积从标准曲线中查得相应的泛酸浓度。

# 6 分析结果的表述

试样中泛酸的含量按式(1)计算。

$$
X = \frac {c \times V \times f}{m} \times \frac {1 0 0}{1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中泛酸含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$c$ ——从标准曲线得到的试样溶液中泛酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样测定液总体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样测定液稀释倍数；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ ；

100 1000 ——换算系数。

计算结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

8.1 对于固体试样，当称样量为 $5\mathrm{g}$ ，定容至 $50~\mathrm{mL}$ 时，检出限为 $0.025\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.08\mathrm{mg} / 100\mathrm{g}$ 。  
8.2 对于液体试样，当取样量为 $20\mathrm{g}$ ，定容至 $50~\mathrm{mL}$ 时，检出限为 $0.0063\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.02\mathrm{mg} / 100\mathrm{g}$ 。

# 第二法 液相色谱-串联质谱法

# 9 原理

试样经酶解、提取、离心、过滤后，经 $\mathrm{C}_{18}$ 反相色谱柱分离，采用串联质谱多离子反应监测方式检测，稳定同位素稀释内标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
10.1.2 甲酸（HCOOH）：色谱纯。  
10.1.3 盐酸（HCl）。  
10.1.4 冰乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
10.1.5 乙酸锌 $\left[\mathrm{Zn}\left(\mathrm{CH}_{3} \mathrm{COO}\right)_{2}\right]$ 。  
10.1.6 亚铁氰化钾 $\left[\mathrm{K}_{4} \mathrm{Fe}(\mathrm{CN})_{6}\right]$ 。  
10.1.7 三羟甲基氨基甲烷 $\mathrm{(C_4H_{11}NO_3)}$   
10.1.8 碳酸氢钠 $(\mathrm{NaHCO}_3)$ 。  
10.1.9 碳酸氢钾 $\mathrm{(KHCO_3)}$   
10.1.10 甲苯 $(\mathrm{C_7H_8})$ 。  
10.1.11 $\alpha$ -淀粉酶：酶活力 $\geqslant 1.5 \mathrm{U} / \mathrm{mg}$ 。  
10.1.12 阴离子交换树脂：Dowex $1 \times 8$ 粒度 $38 \mu \mathrm{m} \sim 75 \mu \mathrm{m}$ 。  
10.1.13 碱性磷酸酶：来源于牛肠粘膜，EC 3.1.3.1，酶活力 $\geqslant 10\mathrm{U / mg}$   
10.1.14 鸽子肝脏丙酮提取物干粉：酶活力 $\geqslant 0.1\mathrm{U / g}$

# 10.2 试剂配制

10.2.1 甲酸溶液 $(0.1\%)$ ：吸取 $1\mathrm{mL}$ 甲酸，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
10.2.2 乙酸锌溶液 $(300\mathrm{g / L})$ ：称取 $30.0\mathrm{g}$ 乙酸锌，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
10.2.3 亚铁氰化钾溶液 $(150\mathrm{g / L})$ ：称取 $15.0\mathrm{g}$ 亚铁氰化钾，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
10.2.4 Tris缓冲液：称取 $121.0\mathrm{g}$ 三羟甲基氨基甲烷溶于 $500~\mathrm{mL}$ 水中，用冰乙酸调 $\mathrm{pH}$ 至 $8.1\pm$ 0.2，加水稀释至 $1000~\mathrm{mL}$ ，混匀，贮存于 $2^{\circ}\mathrm{C}\sim 8^{\circ}\mathrm{C}$ 冰箱中。  
10.2.5 碳酸氢钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $0.84\mathrm{g}$ 碳酸氢钠，加水溶解并稀释至 $100~\mathrm{mL}$ ，混匀。  
10.2.6 碱性磷酸酶溶液：称取 $0.2\mathrm{g}$ 碱性磷酸酶至研钵中，少量多次加水研磨至溶解，用水稀释至 $10~\mathrm{mL}$ 。临用现配，存于 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 冰箱预冷。  
10.2.7 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：吸取 $83~\mathrm{mL}$ 盐酸到 $800~\mathrm{mL}$ 水中，加水稀释至 $1000~\mathrm{mL}$ ，混匀。  
10.2.8 碳酸氢钾溶液 $(0.02\mathrm{mol} / \mathrm{L})$ ：称取 $2\mathrm{g}$ 碳酸氢钾，加水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。  
10.2.9 树脂活化：称取 $100 \mathrm{~g}$ 阴离子交换树脂于 $2 \mathrm{~L}$ 锥形瓶中，加入 $1 \mathrm{~L} 1 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液，置于振荡器上充分振摇 $10 \mathrm{~min}$ ，用铺有滤纸的布氏漏斗过滤。将阴离子交换树脂转回锥形瓶，再加入 $1 \mathrm{~L} 1 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液重复振摇 $10 \mathrm{~min}$ 、过滤。向阴离子交换树脂加入 $1 \mathrm{~L}$ 水振摇洗涤 $10 \mathrm{~min}$ ，过滤，重复用水洗涤

10次，转移至锥形瓶中。向阴离子交换树脂加入少量水，混匀，逐滴向阴离子交换树脂加入Tris缓冲液，调节 $\mathrm{pH}$ 至 $8.0\pm 0.1$ 。置于 $2^{\circ}C\sim 8^{\circ}C$ 冰箱保存，2d内使用。

10.2.10 鸽子肝脏提取液：配制此试剂前一天将所用容器放入 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 冰箱中冷藏过夜。称取 $30\mathrm{g}$ 鸽子肝脏丙酮提取物干粉，放入预冷的研钵中，冰浴条件下分次加入 $0.02\mathrm{mol/L}$ 碳酸氢钾溶液 $300\mathrm{mL}$ ，研磨成匀质混悬液。将混悬液转至预冷的具盖离心管中，盖紧后充分振摇后，放入 $-20^{\circ}\mathrm{C}$ 冰箱内静置 $10\mathrm{min}$ 。取出后， $3000\mathrm{r/min}$ 离心 $5\mathrm{min}$ ，将上清液转至 $500\mathrm{mL}$ 预冷的广口瓶中。向上清液中加入部分活化树脂（约 $150\mathrm{g}$ ），放入冰水浴中振摇 $5\mathrm{min}$ 。倒入预冷离心管中 $3000\mathrm{r/min}$ 离心 $5\mathrm{min}$ 。将上清液移入另一 $500\mathrm{mL}$ 预冷的广口瓶中， $-20^{\circ}\mathrm{C}$ 静置 $10\mathrm{min}$ 。再加入剩余活化树脂（约 $150\mathrm{g}$ ），放入冰浴中振摇 $5\mathrm{min}$ ，倒入离心管中， $3000\mathrm{r/min}$ 离心 $5\mathrm{min}$ 。收集上清液， $-20^{\circ}\mathrm{C}$ 静置 $10\mathrm{min}$ ，分装于预冷的具塞试管中， $-20^{\circ}\mathrm{C}$ 冷冻保存，可保存1年。用前预置于 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 冰箱内化冻并使用。

# 10.3 标准品

10.3.1 D-泛酸钙标准品 $\mathrm{(C_{18}H_{32}CaN_2O_{10}}$ ,CAS号：137-08-6)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准品。  
10.3.2 $^{13}\mathrm{C}_6$ ， ${}^{15}\mathrm{N}_{2}$ -泛酸钙（ $\mathrm{^{13}C_6C_{12}H_{32}Ca^{15}N_2O_{10}}$ ，CAS号：356786-94-2）：纯度 $\geq 97\%$ 。

# 10.4 标准溶液配制

10.4.1 泛酸标准储备液 $(500\mu \mathrm{g} / \mathrm{mL})$ ：准确称取 $136~\mathrm{mg}$ D-泛酸钙标准品(精确至 $0.1\mathrm{mg}$ )，加水溶解并转入到 $250~\mathrm{mL}$ 容量瓶中，稀释至刻度，混匀。标准储备液 $-18^{\circ}\mathrm{C}$ 及以下避光保存，可保存3个月。  
10.4.2 泛酸标准中间溶液 $(10\mu \mathrm{g} / \mathrm{mL})$ : 准确吸取标准储备液 $1\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中, 加水稀释至刻度。临用现配。  
10.4.3 内标储备液 $(50\mu \mathrm{g / mL})$ ：称取 $5\mathrm{mg}^{13}\mathrm{C}_6$ ， ${}^{15}\mathrm{N}_{2}$ -泛酸钙，加水溶解并转入到 $100~\mathrm{mL}$ 容量瓶中，稀释至刻度。内标储备液一 $18^{\circ}C$ 及以下避光保存，可保存6个月。  
10.4.4 泛酸标准系列工作液：吸取适量标准中间溶液于 $25~\mathrm{mL}$ 容量瓶中，加入 $100~\mu \mathrm{L}$ 内标储备液，用水稀释至 $25~\mathrm{mL}$ ，得到泛酸标准工作液浓度分别为 $10~\mathrm{ng / mL}, 50~\mathrm{ng / mL}, 100~\mathrm{ng / mL}, 300~\mathrm{ng / mL}, 600~\mathrm{ng / mL}, 1000~\mathrm{ng / mL}$ 和 $1500~\mathrm{ng / mL}$ 。临用现配。

# 11 仪器和设备

11.1 液相色谱-串联质谱仪：配有电喷雾离子源(ESI)。  
11.2 分析天平：感量分别为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$   
11.3 pH计：精度为 $\pm 0.01$   
11.4 离心机：转速 $\geq 8000\mathrm{r / min}$   
11.5 超声波振荡器。  
11.6 压力蒸汽消毒器： $121^{\circ}\mathrm{C}$ 。  
11.7 恒温振荡水浴箱：振荡频率 $100\mathrm{r / min}\pm 20\mathrm{r / min}$   
11.8 涡旋混合器。

# 12 分析步骤

# 12.1 样品前处理

12.1.1 谷薯类、肉蛋类、蔬果类、菌藻类、豆及坚果类等食品

准确称取适量试样(含 $0.02\mathrm{mg}\sim 0.2\mathrm{mg}$ 泛酸，精确到 $0.01\mathrm{g}$ )，转入至 $100~\mathrm{mL}$ 锥形瓶中。加入 $10~\mathrm{mL}$ Tris缓冲液、 $30~\mathrm{mL}$ 水，振荡混匀。于 $121^{\circ}C$ 高压条件下水解 $20\mathrm{min}$ 。冷却至室温，依次加入 $0.1~\mathrm{mL}$ 碳酸氢钠溶液、 $0.4~\mathrm{mL}$ 碱性磷酸酶溶液、 $0.2~\mathrm{mL}$ 鸽子肝脏提取液，混匀后，加 $100~\mu \mathrm{L}$ 甲苯，具塞。 $37^{\circ}C\pm 1^{\circ}C$ 下以 $100\mathrm{r / min}\pm 20\mathrm{r / min}$ 振幅恒温振荡 $8\mathrm{h}\sim 10\mathrm{h}$ 。转移至 $100~\mathrm{mL}$ 容量瓶中，加水至刻度，过滤。

移取 $10.0\mathrm{mL}$ 上述液体于 $50~\mathrm{mL}$ 离心管中，加入 $100\mu \mathrm{L}$ 内标储备液，加水至 $20~\mathrm{mL}$ ，涡旋 $10~\mathrm{s}$ 。分别加入 $0.4\mathrm{mL}$ 乙酸锌溶液和亚铁氰化钾溶液，加水至 $25~\mathrm{mL}$ ，涡旋 $10~\mathrm{s}$ 。静置 $10\mathrm{min} \sim 30\mathrm{min}$ 后， $8000\mathrm{r / min}$ 离心 $2\mathrm{min}$ 。上清液过 $0.22\mu \mathrm{m}$ 尼龙滤膜后进样分析。

# 13 分析结果的表述

试样中泛酸的含量按式(2)计算。

$$
X = \frac {c \times V}{m} \times \frac {1 0 0}{1 0 ^ {6}} \times f \tag {2}
$$

式中：

$X$ ——试样中泛酸的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$c$ ——由标准曲线得到的试样溶液中泛酸的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样测定液体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的取样量，单位为克(g)；

$\frac{100}{10^6}$ ——换算系数；

$f$ ——稀释因子。

计算结果保留3位有效数字。

# 第三法 微生物法

# 16 原理

泛酸是植物乳植杆菌(Lactiplantibacillus plantarum)生长所必需的营养素，在一定可控的条件下培养一段时间后，植物乳植杆菌的生长与泛酸含量呈现一定线性关系。因植物乳植杆菌生长导致的培养液浑浊度的变化可采用分光光度计或酶标仪进行透光率(或吸光度值)测定，根据培养液中泛酸含量与透光率(或吸光度值)拟合曲线方程计算出试样中泛酸的含量。

# 19 菌种的制备与保存

# 19.1 菌种

植物乳植杆菌Lactiplantibacillus plantarum（ATCC8014），或等效菌株。

# 19.2 储备菌株的制备

无菌操作条件下将菌种植物乳植杆菌-冻干菌粉转接至无菌乳酸杆菌琼脂培养基中， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温培养 $20\mathrm{h} \sim 24\mathrm{h}$ ，使菌种复活。连续传种2次～3次，将培养好的琼脂管作为植物乳植杆菌储备菌株，放入 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱中保存。

# 19.3 菌株传代与保存

将植物乳植杆菌储备菌株每月至少传代1次，培养并保存新菌株。

# 19.4 菌株活化

试验前 $2\mathrm{d}\sim 3\mathrm{d}$ ，将最新保存的菌株接种至无菌琼脂管中， $36^{\circ}C\pm 1^{\circ}C$ 培养 $20\mathrm{h}\sim 24\mathrm{h}$ ，以活化菌株，用于接种液的制备。

注：如果新菌株保存时间超过2周，试验前宜连续传种2代～3代以保证细菌活力。

# 19.5 接种液的制备

试验前一天，取 $2\mathrm{mL}$ 泛酸标准工作液 $(20~\mathrm{ng / mL})$ 和 $4\mathrm{mL}$ 泛酸测定用培养液混匀，分装于2支 $5\mathrm{mL}$ 离心管中，塞好棉塞， $121^{\circ}C$ 下高压灭菌 $15\mathrm{min}$ ，冷却至室温，此即种子培养液。

用接种环将活化菌株接种至种子培养液中， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $20\mathrm{h} \sim 24\mathrm{h}$ 。取出离心管， $3000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，弃去上清液。无菌操作下加入约 $3\mathrm{mL}$ 已预先无菌化处理的生理盐水，振荡洗涤沉淀， $3000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，弃上清液，用生理盐水重复洗涤一次，再加入约 $3\mathrm{mL}$ 无菌生理盐水，振荡混匀，制成接种液，立即使用。

注：为保证接种液中菌群数量,可另外制备 1 支种子培养液,同法接种、洗涤,加入生理盐水混匀；以生理盐水为对

照，用分光光度计在 $550~\mathrm{nm}$ 波长下测定混悬液透光率。根据此管透光率调整接种液体积，使透光率在 $60\% \sim 80\%$ 范围内。

注：所有操作避免日光、紫外线直射。

# 20.1 试样制备

谷薯类、豆类、乳粉等试样需粉碎、研磨、过筛（筛板孔径 $0.3\mathrm{mm}\sim 0.5\mathrm{mm}$ ）；肉、蛋、坚果等用匀质器制成食糜；果蔬、半固体食品等用匀浆器混匀；液体试样用前振摇混匀。 $4^{\circ}\mathrm{C}$ 冰箱可保存1周， $-20^{\circ}\mathrm{C}$ 冰箱可保存半年。试样匀质化处理过程中可加入适量水，并称量加水前后试样质量，计算质量换算因子。

# 20.2 试样提取

# 20.2.1 酶解法

谷薯类、肉蛋类、蔬果类、菌藻类、豆及坚果类等食品试样中的泛酸含量应采用酶解法。

水解：准确称取适量试样 $(m)$ ，精确至 $0.001\mathrm{g}$ ，一般固态及半固态谷薯类、肉类、蛋类、豆类、菌藻类及其制品称取 $1\mathrm{g} \sim 5\mathrm{g}$ ，新鲜水果、蔬菜等含水量较高的食品称取 $5\mathrm{g} \sim 10\mathrm{g}$ ，称样量可根据试样泛酸含量进行调整。转入至 $100\mathrm{mL}$ 锥形瓶中，加 $10\mathrm{mL}$ Tris缓冲液、 $30\mathrm{mL}$ 水，振荡混匀，于 $121^{\circ}\mathrm{C}$ 高压条件下水解 $15\mathrm{min} \sim 20\mathrm{min}$ 。

酶解：冷却至室温，依次加入 $0.1\mathrm{mL}$ 碳酸氢钠溶液、 $0.4\mathrm{mL}$ 碱性磷酸酶溶液、 $0.2\mathrm{mL}$ 鸽子肝脏提取液，小心混匀，如果试样中淀粉含量较高，导致样液较为黏稠，可另外加入 $20\mathrm{mg}\sim 40\mathrm{mg}\alpha$ -淀粉酶。加 $50~\mu \mathrm{L}$ 甲苯，具塞， $37^{\circ}C\pm 1^{\circ}C$ 下以 $80~\mathrm{r / min}\pm 20~\mathrm{r / min}$ 振幅振荡水浴 $8\mathrm{h}\sim 10\mathrm{h}$

定容、沉淀、过滤：将试样酶解液转移至 $100\mathrm{mL}$ 容量瓶中，用水定容至刻度 $(V)$ ，过滤。准确吸取适量滤液 $(1.0\mathrm{mL} \sim 10.0\mathrm{mL}, V_{1})$ 至 $25\mathrm{mL}$ 具塞刻度试管底部，加水至 $15\mathrm{mL}$ 。如果滤液澄清且滴加冰乙酸没有沉淀出现，用 $1.0\mathrm{mol/L}$ 乙酸缓冲液直接调节 $\mathrm{pH}$ 至 $6.8 \pm 0.2$ ，用水定容至 $25\mathrm{mL}(V_{2})$ ；如果滤液浑浊或滴加冰乙酸有沉淀析出，则用冰乙酸调节 $\mathrm{pH}$ 至 $4.5 \pm 0.1$ ，加水定容至 $25\mathrm{mL}(V_{2})$ ，过滤后准确吸取适量滤液 $(5.0\mathrm{mL} \sim 10.0\mathrm{mL}, V_{3})$ 至另一 $25\mathrm{mL}$ 具塞刻度试管，用Tris缓冲液或稀的氢氧化钠溶液调节 $\mathrm{pH}$ 至 $6.8 \pm 0.2$ ，用水定容至 $25\mathrm{mL}(V_{4})$ 。

另取一试管按试样提取步骤加入缓冲液、碳酸氢钠溶液、碱性磷酸酶溶液、鸽子肝脏提取液， $\alpha-$ 淀粉酶(使用时)，振荡水浴后调节 $\mathrm{pH}$ 后定容过滤，作为酶空白液。

# 21 分析结果表述

# 21.1 标准曲线

以标准系列管中泛酸浓度为横坐标，以每个标准点透光率(或吸光度值)均值为纵坐标，绘制标准曲线，拟合四参数曲线方程。

# 21.2 试样结果计算

根据试样和酶空白系列管的透光率(或吸光度值)，从标准曲线计算得到对应的泛酸浓度 $(c_{\mathrm{x}})$ ，按照式(3)计算试样稀释液中泛酸浓度 $(c)$ 。如果试样系列管中有3支以上在标准曲线范围之内，且换算为试样稀释液泛酸浓度的相对偏差 $\leqslant 15\%$ ，计为有效管。根据有效管个数按照式(4)计算泛酸稀释液质量浓度平均值 $(\overline{c})$ ，并继续按照式(5)或式(6)计算试样中泛酸含量结果。

试样稀释液中泛酸质量浓度及其平均值按式(3)、式(4)计算。

$$
c _ {i} = c _ {\mathrm {x}} \times \frac {V _ {5}}{V _ {\mathrm {x}}} \tag {3}
$$

$$
\bar {c} = \frac {\sum_ {i = 1} ^ {n} c _ {i}}{n} \tag {4}
$$

式中：

$c_{i}$ ——试样稀释液泛酸质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{\mathrm{x}}$ ——由标准曲线计算得到的试样系列管中泛酸质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V_{5}$ 制备试样系列管时最终体积与吸取的试样稀释液体积之比；

$V_{\mathrm{x}}$ 制备试样系列管时最终体积与吸取的试样稀释液体积之比；

$\overline{c}$ ——试样稀释液泛酸质量浓度均值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$\sum_{i=1}^{n} c_{i}$ ——由有效管计算的试样稀释液质量浓度和，单位为纳克每毫升 $(\mathrm{ng/mL})$ ；

$n$ ——有效管个数。

采用直接提取法的试样按式(5)计算泛酸含量。

$$
X = \frac {\bar {c} \times V \times f}{m} \times \frac {1 0 0}{1 0 ^ {6}} \tag {5}
$$

式中：

$X$ ——试样中泛酸含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$\overline{c}$ ——试样稀释液泛酸质量浓度均值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样提取液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样提取液稀释倍数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$\frac{100}{10^6}$ ——换算系数。

采用酶解法提取的试样，按式(6)计算泛酸含量。

$$
X = \frac {\left(\bar {c} - \overline {{c _ {0}}}\right) \times V}{m} \times \frac {V _ {2} \times V _ {4}}{V _ {1} \times V _ {3}} \times f \times \frac {1 0 0}{1 0 ^ {6}} \quad \dots \tag {6}
$$

式中：

$X$ ——试样中泛酸含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$\overline{c}$ ——试样稀释液中泛酸质量浓度均值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$\overline{c}_0$ ——酶空白液中泛酸质量浓度均值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样提取液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}, V_{4}$ ——调节 $\mathrm{pH}$ 后定容体积，单位为毫升（ $\mathrm{mL}$ ）；

$V_{1}, V_{3}$ ——调节 $\mathrm{pH}$ 时吸取的试样提取液体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样提取液稀释倍数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$\frac{100}{10^6}$ ——换算系数。

结果保留3位有效数字。

# B.2.2 试剂配制

B.2.2.1 氢氧化钠溶液 $(10\mathrm{mol} / \mathrm{L})$ ：称取 $40\mathrm{g}$ 氢氧化钠，用 $100~\mathrm{mL}$ 水溶解。

B.2.2.2 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $4\mathrm{g}$ 氢氧化钠，用 $100~\mathrm{mL}$ 水溶解。

B.2.2.3 盐酸溶液 $(3\mathrm{mol} / \mathrm{L})$ ：吸取 $250~\mathrm{mL}$ 盐酸，用水稀释至 $1000~\mathrm{mL}$ ，混匀。

B.2.2.4 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：按10.2.7配制。

B.2.2.5 酪蛋白液: 称取 $50 \mathrm{~g}$ 无维生素酪蛋白于 $500 \mathrm{~mL}$ 烧杯中, 加 $200 \mathrm{~mL}$ 盐酸溶液 $(3 \mathrm{~mol} / \mathrm{L})$ , $121^{\circ} \mathrm{C}$ 下高压水解 $6 \mathrm{~h}$ 。将水解物转移至蒸发皿内, 在沸水浴上蒸发至膏状。加 $200 \mathrm{~mL}$ 水使之溶解后再蒸发至膏状, 如此反复 3 次, 以除去盐酸。用 $10 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液调节 $\mathrm{pH}$ 至 $3.5 \pm 0.1$ 。加 $20 \mathrm{~g}$ 活性炭, 振摇约 $20 \mathrm{~min}$ , 过滤。重复活性炭处理 2 次 $\sim 4$ 次, 直至滤液呈淡黄色或无色。滤液加水稀释至 $1000 \mathrm{~mL}$ , 加 $3 \mathrm{~mL}$ 甲苯, 置 $2^{\circ} \mathrm{C} \sim 8^{\circ} \mathrm{C}$ 冰箱中可保存 1 年。

注：每次蒸发时不可蒸干或焦糊，以避免所含营养素破坏。可直接由试剂公司购买效力相当的酸水解无维生素酪蛋白。

B.2.2.6 腺嘌呤-鸟嘌呤-尿嘧啶溶液: 分别称取硫酸腺嘌呤、盐酸鸟嘌呤和尿嘧啶各 $0.1 \mathrm{~g}$ 于 $250 \mathrm{~mL}$ 烧杯中, 加 $75 \mathrm{~mL}$ 水和 $2 \mathrm{~mL}$ 盐酸, 加热使其完全溶解后冷却。若有沉淀产生, 再加盐酸数滴, 加热, 如此反复直至冷却后无沉淀产生为止。用水稀释至 $100 \mathrm{~mL}$ , 加 $200 \mu \mathrm{L}$ 甲苯, 贮存于棕色试剂瓶中, 于

$2^{\circ} \mathrm{C} \sim 8^{\circ} \mathrm{C}$ 冰箱中可保存1年。

B.2.2.7 胱氨酸-色氨酸溶液：分别称取 $4\mathrm{gL}$ -胱氨酸和 $1\mathrm{gL}$ -色氨酸或 $2\mathrm{gL}$ -色氨酸于 $800~\mathrm{mL}$ 水中，加热至 $70^{\circ}\mathrm{C}\sim 80^{\circ}\mathrm{C}$ ，逐滴加入 $3\mathrm{mol} / \mathrm{L}$ 盐酸溶液，不断搅拌，直至完全溶解为止。冷却至室温后加水稀释至 $1000~\mathrm{mL}$ 。加 $200~\mu \mathrm{L}$ 甲苯，贮存于棕色试剂瓶中，于 $2^{\circ}\mathrm{C}\sim 8^{\circ}\mathrm{C}$ 冰箱中可保存1年。  
B.2.2.8 维生素液I: 分别称取 $20 \mathrm{mg}$ 核黄素和 $10 \mathrm{mg}$ 盐酸硫胺素, 加入 $1.0 \mathrm{~mL}$ 生物素溶液 $(40 \mu \mathrm{g} / \mathrm{mL})$ , 用 $0.02 \mathrm{~mol} / \mathrm{L}$ 乙酸溶液溶解并稀释至 $1000 \mathrm{~mL}$ 。加入 $200 \mu \mathrm{L}$ 甲苯, 贮存于棕色试剂瓶中, 于 $2^{\circ} \mathrm{C} \sim 8^{\circ} \mathrm{C}$ 冰箱中可保存 1 年。  
B.2.2.9 维生素液Ⅱ:分别称取 $10\mathrm{mg}$ 对氨基苯甲酸、 $50\mathrm{mg}$ 尼克酸和 $40\mathrm{mg}$ 盐酸吡哆醇,溶于 $1000\mathrm{mL}$ 乙醇溶液中。加入 $200\mu \mathrm{L}$ 甲苯,贮存于棕色试剂瓶中,于 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 中冰箱可保存1年。  
B.2.2.10 聚山梨酯-80溶液：将 $25\mathrm{g}$ 聚山梨酯-80用乙醇溶解并稀释至 $250~\mathrm{mL}$ ，于 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 冰箱中可保存1年。  
B.2.2.11 甲盐溶液：同B.1.2.1。  
B.2.2.12 泛酸测定用培养液：配制 $100 \mathrm{~mL}$ 培养液，按表 B.2 各成分用量吸取液体试剂，混合后加水 $300 \mathrm{~mL}$ ，依次加入固体试剂，煮沸搅拌 $2 \mathrm{~min}$ 。用 $1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液、 $1 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液调节基础培养液 $\mathrm{pH}$ 至 $6.8 \pm 0.1$ 。加入乙盐溶液 $20 \mathrm{~mL}$ ，用水补至 $1000 \mathrm{~mL}$ 。配制时可根据基础培养液用量按比例增减。

盖上瓶盖后于 $121^{\circ}\mathrm{C}$ 高压灭菌 $15\mathrm{min}$ ，冷却至室温，临用现配。

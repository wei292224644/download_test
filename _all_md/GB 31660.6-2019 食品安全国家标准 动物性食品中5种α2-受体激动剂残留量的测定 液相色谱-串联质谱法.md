# 液相色谱-串联质谱法

# 1 范围

本标准规定了动物性食品中5种 $\alpha_{2}$ -受体激动剂的制样和液相色谱-串联质谱测定方法。

本标准适用于猪肌肉、肝脏和肾脏组织及鸡肌肉和肝脏组织中替扎尼定、赛拉嗪、溴莫尼定、安普乐定和可乐定残留量的测定。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样中残留的 $\alpha_{2}$ -受体激动剂，用碳酸钠缓冲溶液、乙酸乙酯提取，固相萃取柱净化，液相色谱-串联质谱测定，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 乙腈（ $\mathrm{CH}_3\mathrm{CN}$ ）：色谱纯。  
4.1.2 甲酸（HCOOH）：色谱纯。  
4.1.3 甲醇（ $\mathrm{CH}_3\mathrm{OH}$ ）。  
4.1.4 氨水（ $\mathrm{NH_4OH}$ ）。  
4.1.5 无水碳酸钠（ $\mathrm{Na}_2\mathrm{CO}_3$ ）。  
4.1.6 碳酸氢钠（ $\mathrm{NaHCO_3}$ ）。  
4.1.7 乙酸乙酯（ $\mathrm{CH}_3\mathrm{COOC}_2\mathrm{H}_5$ ）。

# 4.2 溶液配制

4.2.1 碳酸钠溶液：取无水碳酸钠 $10.6\mathrm{g}$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。  
4.2.2 碳酸氢钠溶液：取碳酸氢钠 $8.4\mathrm{g}$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。  
4.2.3 碳酸钠缓冲溶液：取碳酸钠溶液 $90 \mathrm{~mL}$ 、碳酸氢钠溶液 $10 \mathrm{~mL}$ , 混匀, 现用现配。  
4.2.4 甲酸溶液（0.2%）：取甲酸 $1\mathrm{mL}$ ，用水溶解并稀释至 $500\mathrm{mL}$ 。  
4.2.5 甲酸-乙腈溶液：取 $0.2\%$ 甲酸溶液 $80~\mathrm{mL}$ ，加乙腈 $20~\mathrm{mL}$ ，混匀。  
4.2.6 氨水甲醇溶液（5%）：取氨水 $5\mathrm{mL}$ ，用甲醇溶解并稀释至 $100\mathrm{mL}$ 。

# 4.3 标准品

盐酸替扎尼定（Tizanidine hydrochloride, $\mathrm{C_9H_8ClN_5S}$ HCl, CAS:64461-82-1）、盐酸赛拉嗪（Xylazine Hydrochloride, $\mathrm{C_{12}H_{16}N_2S}$ , CAS:7361-61-7）、溴莫尼定（Brimonidine, $\mathrm{C_{11}H_{10}BrN_5}$ , CAS:59803-98-4）、盐酸安普乐定（Apraclonidine hydrochloride, $\mathrm{C_9H_{10}N_4Cl_2}$ HCl,

CAS:73218-79-8）和盐酸可乐定（Clonidine hydrochloride, $\mathrm{C}_{9} \mathrm{H}_{10} \mathrm{Cl}_{3} \mathrm{~N}_{3}$ , CAS:4205-91-8）：含量均 $\geqslant 98 \%$ 。

# 4.4 标准溶液制备

4.4.1 标准贮备液：取盐酸替扎尼定、盐酸赛拉嗪、溴莫尼定、盐酸安普乐定和盐酸可乐定标准品各约 $10\mathrm{mg}$ ，精密称定，分别于 $10\mathrm{mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度均为 $1\mathrm{mg/mL}$ 的盐酸替扎尼定、盐酸赛拉嗪、溴莫尼定、盐酸安普乐定和盐酸可乐定标准贮备液。 $2\sim8^{\circ}\mathrm{C}$ 保存，有效期6个月。  
4.4.2 混合标准工作液：分别精密量取上述标准贮备液各 $1 \mathrm{~mL}$ ，于 $100 \mathrm{~mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 的混合标准工作液。于 $2 \sim 8^{\circ} \mathrm{C}$ 保存，有效期1个月。  
4.4.3 标准曲线制备：分别精密量取混合标准工作液适量，用甲酸-乙腈溶液稀释配制成浓度为0.5、1、10、50和 $100\mu \mathrm{g} / \mathrm{L}$ 系列混合标准工作液，临用现配。

# 4.5 材料

4.5.1 固相萃取MCX柱： $60\mathrm{mg / 3mL}$ ，或相当者。  
4.5.2 滤膜：有机相， $0.22\mu \mathrm{m}$

# 5 仪器和设备

5.1 液相色谱-串联质谱仪：配电喷雾电离源。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
5.3 均质机。  
5.4 离心机。  
5.5 涡旋振荡器。  
5.6 旋转蒸发仪。  
5.7 固相萃取装置。  
5.8 鸡心瓶。  
5.9 离心管： $50~\mathrm{mL}$ 。

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎并使均质。

——取均质的供试样品，作为供试试料。  
——取均质的空白样品，作为空白试料。  
——取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 样品的保存

-20℃以下保存。

# 7 测定步骤

# 7.1 提取

取试料 $2 \mathrm{~g}$ (准确至 $\pm 20 \mathrm{mg}$ ), 于 $50 \mathrm{~mL}$ 离心管中, 加碳酸钠缓冲液 $5 \mathrm{~mL}$ , 振荡混匀, 加乙酸乙酯 $10 \mathrm{~mL}$ , 充分振荡, 于 $8000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 取上清液于鸡心瓶中, 下层溶液中加乙酸乙酯 $10 \mathrm{~mL}$ 重复提取一次, 合并两次上清液。于 $55^{\circ} \mathrm{C}$ 旋转蒸发至干, 加甲酸-乙腈溶液 $4.0 \mathrm{~mL}$ 溶解残余物, 备用。

# 7.2 净化

MCX柱依次用甲醇、水各 $3 \mathrm{~mL}$ 活化。取备用液过柱, 用水 $3 \mathrm{~mL}$ 、甲醇 $3 \mathrm{~mL}$ 分别淋洗, 挤干, 氨水甲醇溶液 $6 \mathrm{~mL}$ 洗脱, 收集洗脱液, 于 $60^{\circ} \mathrm{C}$ 旋转蒸发至干, 用甲酸-乙腈溶液 $1.0 \mathrm{~mL}$ 溶解残余物, 滤过, 液相色谱-串联质谱测定。

# 7.3 测定

# 7.3.1 色谱条件

a）色谱柱： $\mathrm{C_{18}}$ （ $100\mathrm{mm}\times 3.0\mathrm{mm}$ ，粒径 $1.8\mu \mathrm{m}$ ），或相当者；  
b）柱温： $30^{\circ}\mathrm{C}$   
c）进样量： $10~\mu \mathrm{L}$   
d）流速： $0.3\mathrm{mL / min}$   
e) 流动相: A: 乙腈; B: $0.2\%$ 甲酸溶液, 梯度洗脱程序见表 1。

# 7.4 测定法

# 7.4.1 定性测定

在同样测试条件下，试样液中 $\alpha_{2}$ -受体激动剂的保留时间与标准工作液中相应 $\alpha_{2}$ -受体激动剂的保留时间之比，偏差在 $\pm 5\%$ 以内，且检测到离子的相对丰度，应当与浓度相当的校正标准溶液相对丰度一致，其允许偏差应符合表3要求。

# 7.4.2 定量测定

取试样溶液和相应的标准工作液，按外标法以峰面积定量，标准工作液及试样溶液中的 $\alpha_{2}$ 受体激动剂类响应值均应在仪器检测的线性范围内。在上述条件下， $\alpha_{2}$ 受体激动剂标准溶液特征离子质量色谱图见附录A。

# 7.5 空白试验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算和表述

试料中待测药物的残留量按式（1）计算：

$$
X = \frac {C _ {s} \times A \times V}{A _ {s} \times m} \dots \tag {1}
$$

式中：

$X$ ——供试试料中相应的 $\alpha_{2}$ 受体激动剂类药物残留量, 单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ;

$C_{s}$ ——标准溶液中相应的 $\alpha_{2}$ 受体激动剂类药物浓度, 单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ;

$A$ ——试样溶液中相应的 $\alpha_{2}$ 受体激动剂类药物的色谱峰面积；

$A_{s}$ ——标准溶液中相应的 $\alpha_{2}$ 受体激动剂类药物色谱峰面积；

$V$ ——溶解残余物所用的溶液体积, 单位为毫升 $(\mathrm{mL})$ ;

$m$ ——供试试料质量，单位为克（g）；

计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留两位有效数字。

# 9 检测方法灵敏度、准确度和精密度

# 9.1 灵敏度

本方法的检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1\mu \mathrm{g / kg}$

# 9.2 准确度

本方法在 $1\sim 100~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 100\%$

# 9.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$

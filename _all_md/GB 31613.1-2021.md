# 方法一 液相色谱-串联质谱法

# 1 范围

本文件规定了牛可食性组织中氨丙啉残留量检测的制样和液相色谱串联质谱测定方法。

本文件适用于牛肌肉、肝脏、肾脏和脂肪中氨丙啉残留量的检测。

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试料中残留的氨内啉经磷酸盐缓冲液提取（脂肪样品用乙腈提取），HLB固相萃取柱净化，液相色谱串联质谱法测定，基质校准外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.3 甲酸（HCOOH）：色谱纯。  
5.1.4 磷酸氢二钾 $(\mathrm{K_2HPO_4})$ 。  
5.1.5 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$

# 5.2 标准品

盐酸氨丙啉（Amoprolium Hydrochloride, $\mathrm{C_{14}H_{19}ClN_4\cdot HCl}$ ,CAS号：137-88-2)，含量 $\geq 98.0\%$ 。

# 5.3 溶液配制

5.3.1 磷酸盐缓冲液 $(\mathrm{pH}6.0)$ ：取磷酸氢二钾 $2.0\mathrm{g}$ 、磷酸二氢钾 $8.0\mathrm{g}$ ，加水溶解并稀释至 $1000~\mathrm{mL}$ 混匀。  
5.3.2 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.3.3 $0.1\%$ 甲酸溶液-甲醇：取 $0.1\%$ 甲酸溶液 $900~\mathrm{mL}$ 、甲醇 $100~\mathrm{mL}$ ，混匀。

# 5.4 标准溶液制备

5.4.1 标准储备液 $(100\mathrm{mg / L})$ ：取盐酸氨丙啉标准品适量(约相当于氨丙啉 $10\mathrm{mg}$ )，精密称定，用甲醇溶解，稀释定容于 $100~\mathrm{mL}$ 容量瓶，摇匀，即得。 $-18^{\circ}C$ 以下保存，有效期6个月。

5.4.2 标准工作液：准确移取标准储备液适量，用 $0.1\%$ 甲酸溶液-甲醇逐级稀释成浓度为 $5\mu \mathrm{g} / \mathrm{L}$ 、 $20\mu \mathrm{g} / \mathrm{L},50\mu \mathrm{g} / \mathrm{L},200\mu \mathrm{g} / \mathrm{L},500\mu \mathrm{g} / \mathrm{L}$ 和 $2000\mu \mathrm{g} / \mathrm{L}$ 的系列标准工作溶液。现用现配。

# 5.5 材料

HLB固相萃取小柱： $60~\mathrm{mg / 3~mL}$ ，或相当者。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 氮吹仪。  
6.4 固相萃取装置。  
6.5 均质机。  
6.6 涡旋混合器。  
6.7 聚丙烯塑料离心管： $50~\mathrm{mL}$ 。  
6.8 离心机： $8000\mathrm{r / min}$ 或以上。

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存，3个月内进行分析检测。

# 8 测定步骤

# 8.1 提取

# 8.1.1 牛肌肉、肝脏和肾脏

取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，加磷酸盐缓冲液 $10.0\mathrm{mL},10000\mathrm{r / min}$ 均质 $1\mathrm{min},8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液，残渣加磷酸盐缓冲液 $10.0~\mathrm{mL}$ ，重复提取一次，合并上清液，备用。

# 8.1.2 牛脂肪

取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，加乙腈 $10.0~\mathrm{mL},10~000~\mathrm{r / min}$ 均质 $1\mathrm{min},8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液于 $50~\mathrm{mL}$ 塑料离心管中。残渣加乙腈 $10~\mathrm{mL}$ ，重复提取一次，合并2次上清液，于 $50^{\circ}C$ 水浴中氮吹至干，用磷酸盐缓冲液 $20.0~\mathrm{mL}$ 溶解残余物，备用。

# 8.2 净化

固相萃取柱用甲醇、水各 $3\mathrm{mL}$ 活化。取 $5.0\mathrm{mL}$ 备用液过柱，待液面到达柱床表面时用水 $3\mathrm{mL}$ 淋洗，挤至近干，加甲醇 $2\mathrm{mL}$ ，洗脱，洗脱液在 $40^{\circ}\mathrm{C}$ 水浴氮气吹干。用 $0.1\%$ 甲酸溶液-甲醇 $(90:10,V/V)$ $1.0\mathrm{mL}$ 溶解残余物，过 $0.22\mu \mathrm{m}$ 滤膜，液相色谱-串联质谱仪测定。

# 8.3 基质匹配标准曲线的制备

取空白试样依次按8.1和8.2处理至氮气吹干，加系列浓度标准工作液 $1.0\mathrm{mL}$ 溶解残渣，制成浓度为 $5\mu \mathrm{g} / \mathrm{L}, 20\mu \mathrm{g} / \mathrm{L}, 50\mu \mathrm{g} / \mathrm{L}, 200\mu \mathrm{g} / \mathrm{L}, 500\mu \mathrm{g} / \mathrm{L}$ 和 $2000\mu \mathrm{g} / \mathrm{L}$ 的系列基质匹配标准溶液，供液相色谱-串联质谱仪测定。以特征离子质量色谱峰面积为纵坐标，基质匹配标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.4 测定

# 8.4.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱 $(150\mathrm{mm}\times 2.1\mathrm{mm},3.5\mu \mathrm{m})$ 或相当者；  
b）流动相： $0.1\%$ 甲酸溶液-甲醇 $(90:10,V / V)$   
c) 流速: $0.2 \mathrm{~mL} / \mathrm{min}$ ;   
d） 柱温： $30^{\circ}C$   
e）进样量： $10~\mu \mathrm{L}$

# 8.4.2 质谱参考条件

a）离子源：电喷雾(ESI)离子源；  
b）扫描方式：正离子扫描；  
c）检测方式：多反应监测（MRM）；  
d) 毛细管电压: $3000 \mathrm{~V}$ ;  
e) RF透镜电压： $0.5\mathrm{V}$   
f）离子源温度： $150^{\circ}C$   
g）脱溶剂气温度： $350\%$   
h）锥孔气流速： $50~\mathrm{L / h}$   
i）脱溶剂气流速： $1000\mathrm{L / h}$   
j）二级碰撞气：氩气；  
k）定性离子对、定量离子对、碰撞能力和锥孔电压见表1。

取试料溶液和相应基质匹配标准溶液，作单点或多点校准，按外标法以色谱峰面积定量。基质匹配标准溶液及试料溶液中目标药物的特征离子质量色谱峰峰面积均应在仪器检测的线性范围之内。试料溶液中氨丙啉的保留时间与基质匹配标准溶液中氨丙啉保留时间偏差在 $\pm 2.5\%$ 以内，且试料溶液中的相对离子丰度与基质匹配标准溶液中的相对离子丰度相比，符合表2的要求。标准溶液多反应监测色谱图见附录A。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试料中氨丙啉的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times C _ {\mathrm {S}} \times V _ {1} \times V _ {3} \times 1 0 0 0}{A _ {\mathrm {S}} \times V _ {2} \times m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试料中氨丙啉残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$A$ ——试样溶液中氨丙啉峰面积；  
$C_{\mathrm{S}}$ ——标准工作液中氨丙啉浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$   
$V_{1}$ ——磷酸盐缓冲液体积的数值，单位为毫升 $(\mathrm{mL})$   
$V_{3}$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$   
$A_{S}$ ——标准工作液中氨丙啉峰面积；  
$V_{2}$ ——过HLB固相萃取柱所用备用液体积的数值，单位为毫升 $(\mathrm{mL})$   
$m$ ——供试试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法在牛肌肉、肝脏、肾脏和脂肪中的检测限为 $10~\mu \mathrm{g / kg}$ ，定量限为 $25~\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法在 $25\mu \mathrm{g / kg}\sim 3000\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

# 15 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 15.1 试剂

15.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
15.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
15.1.3 庚烷磺酸钠 $\mathrm{(C_7H_{15}NaO_3S)}$ 。  
15.1.4 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
15.1.5 冰醋酸 $\mathrm{(CH_3COOH)}$ 。  
15.1.6 正丙醇 $\mathrm{(CH_3CH_2CH_2OH)}$   
15.1.7 氢氧化钠(NaOH)。  
15.1.8 三乙胺 $\left[\left(\mathrm{C}_{2} \mathrm{H}_{5}\right)_{3} \mathrm{~N}\right]_{\circ}$

# 15.2 标准品

盐酸氨丙啉（Amoprolium Hydrochloride, $\mathrm{C_{14}H_{19}ClN_4\cdot HCl}$ ,CAS号：137-88-2)，含量 $\geq 98.0\%$ 。

# 15.3 溶液配制

15.3.1 氢氧化钠溶液 $(5\mathrm{mol} / \mathrm{L})$ ：取氢氧化钠 $20\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ ，冷却，于塑料容器保存。

15.3.2 磷酸盐缓冲液：取磷酸二氢钾 $13.6\mathrm{g}$ ，加水溶解并稀释至 $1000\mathrm{mL}$ ，用氢氧化钠溶液调 $\mathrm{pH}$ 至7.0。

15.3.3 庚烷磺酸钠溶液：取庚烷磺酸钠 $1.2\mathrm{g}$ ，用水溶解并稀释至 $1000\mathrm{mL}$ ，再加冰醋酸 $24\mathrm{mL}$ 、三乙胺 $6\mathrm{mL}$ ，混匀。

# 15.4 标准溶液制备

15.4.1 标准储备液 $(1\mathrm{mg / mL})$ ：取盐酸氨丙啉标准品适量(约相当于氨丙啉 $10\mathrm{mg}$ )，精密称定，用甲醇溶解并稀释定容于 $10~\mathrm{mL}$ 容量瓶中，配制成浓度为 $1\mathrm{mg / mL}$ 的标准储备液。于 $-18^{\circ}C$ 以下保存，有效期6个月。

15.4.2 标准工作液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：精密量取标准储备液 $1\mathrm{mL}$ ，于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $100\mu \mathrm{g} / \mathrm{mL}$ 标准工作液。一 $18^{\circ}C$ 以下保存，有效期6个月。

15.5 材料

HLB固相萃取柱： $500\mathrm{mg / 6mL}$ ，或相当者。

17 试料的制备与保存  
17.1 试料的制备

取新鲜或冷冻的空白或供试组织，去除筋膜，绞碎，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

17.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

18 测定步骤

18.1 提取

取试料 $5\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，加乙腈 $10~\mathrm{mL},10000\mathrm{r / min}$ 均质 $1\mathrm{min},5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液于鸡心瓶中。残渣加乙腈 $10~\mathrm{mL}$ ，重复提取一次，合并2次上清液，加正丙醇 $5\mathrm{mL}$ ，于 $50^{\circ}C$ 水浴中旋转蒸发至干，用磷酸盐缓冲液 $20.0~\mathrm{mL}$ 溶解残余物，备用。

18.2 净化

固相萃取柱依次用甲醇、水各 $5\mathrm{mL}$ 活化。取备用液 $5.0\mathrm{mL} \sim 10.0\mathrm{mL}$ 过柱，用水 $5\mathrm{mL}$ 淋洗，挤干。加流动相 $4.0\mathrm{mL}$ ，洗脱，挤干，收集洗脱液，混匀，滤过，供高效液相色谱测定。

18.3 标准曲线的制备

精密量取标准工作液适量，用流动相稀释配制成浓度 $0.2\mu \mathrm{g / mL},0.5\mu \mathrm{g / mL},1\mu \mathrm{g / mL},5\mu \mathrm{g / mL}$ $10~\mu \mathrm{g / mL}$ 和 $20~\mu \mathrm{g / mL}$ 系列标准溶液，供高效液相色谱测定。以测得峰面积为纵坐标、相对应的标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。  
18.4 测定

# 18.4.1 液相色谱条件

a）色谱柱： $\mathrm{C_{18}(250~mm\times 4.6~mm,5~\mu m)}$ ，或相当者；  
b）流动相：庚烷磺酸钠溶液-甲醇-乙腈 $(60:35:5)$   
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d）检测波长： $267~\mathrm{nm}$   
e) 进样量: $20 \mu \mathrm{L}$ ;

# 18.4.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法以色谱峰面积定量。标准溶液及试样溶液中氨丙啉的响应值均应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和空白添加试样溶液的液相色谱图见附录B。

# 18.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 19 结果计算和表述

试料中氨丙啉的残留量按标准曲线或公式(2)计算。

$$
X = \frac {A \times C _ {\mathrm {S}} \times V _ {1} \times V _ {3} \times 1 0 0 0}{A _ {\mathrm {S}} \times V _ {2} \times m \times 1 0 0 0} \dots \tag {2}
$$

式中：

$X$ ——试料中氨丙啉残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$A$ ——试样溶液中氨丙啉峰面积；

$C_{\mathrm{S}}$ ——标准工作液中氨丙啉浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$V_{1}$ ——磷酸盐缓冲液体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{3}$ ——流动相洗脱液体积的数值，单位为毫升 $(\mathrm{mL})$

# 20 检测方法灵敏度、准确度和精密度

# 20.1 灵敏度

本方法的检测限为 $100\mu \mathrm{g / kg}$ ，定量限为 $250~\mu \mathrm{g / kg}$

# 20.2 准确度

本方法在牛的肌肉、肝脏、肾脏组织 $250~\mu \mathrm{g / kg}\sim 2000~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim$ $110\%$ ；在牛脂肪中组织 $250~\mu \mathrm{g / kg}\sim 4000~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$ 。

# 20.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏 $\leqslant 20\%$

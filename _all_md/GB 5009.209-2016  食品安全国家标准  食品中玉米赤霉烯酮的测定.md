# 前言

本标准代替GB/T5009.209—2008《谷物中玉米赤霉烯酮的测定》、GB/T23504—2009《食品中玉米赤霉烯酮的测定免疫亲和层析净化高效液相色谱法》、SN/T1745—2006《进出口大豆、油菜籽和食用植物油中玉米赤霉烯酮的检验方法》、SN/T1772—2006《进出口粮谷中玉米赤霉烯酮的测定免疫亲和柱-液相色谱法》，代替GB/T21982—2008《动物源食品中玉米赤霉醇、 $\beta$ -玉米赤霉醇、 $\alpha$ -玉米赤霉烯醇、 $\beta$ -玉米赤霉烯醇、玉米赤霉酮和玉米赤霉烯酮残留量检测方法液相色谱-质谱/质谱法》中玉米赤霉烯酮的检测方法。

本标准与GB/T5009.209—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中玉米赤霉烯酮的测定”；  
——增加了适用范围；  
——增加了荧光光度法作为第二法；  
——增加了固相萃取柱净化液相色谱-质谱法作为第三法。

# 食品安全国家标准

# 食品中玉米赤霉烯酮的测定

# 1 范围

本标准规定了食品中玉米赤霉烯酮的测定方法。

本标准第一法适用于粮食和粮食制品，酒类，酱油、醋、酱及酱制品，大豆、油菜籽、食用植物油中玉米赤霉烯酮的测定，第二法适用于大豆、油菜籽、食用植物油中玉米赤霉烯酮的测定，第三法适用于牛肉、猪肉、牛肝、牛奶、鸡蛋中玉米赤霉烯酮的测定。

# 第一法 液相色谱法

# 2 原理

用乙腈溶液提取试样中的玉米赤霉烯酮，经免疫亲和柱净化后，用高效液相色谱荧光检测器测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.3 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.4 氯化钾(KCl)。  
3.1.5 磷酸氢二钠 $\mathrm{(Na_2HPO_4)}$ 。  
3.1.6 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
3.1.7 吐温-20 $(\mathrm{C}_{58}\mathrm{H}_{114}\mathrm{O}_{26})$ 。  
3.1.8 盐酸（HCl）。

# 3.2 试剂配制

3.2.1 提取液：乙腈-水 $(9 + 1)$   
3.2.2 PBS 清洗缓冲液: 称取 $8.0 \mathrm{~g}$ 氯化钠、 $1.2 \mathrm{~g}$ 磷酸氢二钠、 $0.2 \mathrm{~g}$ 磷酸二氢钾、 $0.2 \mathrm{~g}$ 氯化钾, 用 $990 \mathrm{~mL}$ 水将上述试剂溶解, 用盐酸调节 $\mathrm{pH}$ 至 7.0 , 用水定容至 $1 \mathrm{~L}$ 。  
3.2.3 PBS/吐温-20缓冲液：称取 $8.0\mathrm{g}$ 氯化钠、 $1.2\mathrm{g}$ 磷酸氢二钠、 $0.2\mathrm{g}$ 磷酸二氢钾、 $0.2\mathrm{g}$ 氯化钾，用 $900~\mathrm{mL}$ 水将上述试剂溶解，用盐酸调节 $\mathrm{pH}$ 至7.0，加入 $1\mathrm{mL}$ 吐温-20，用水定容至 $1\mathrm{L}$

# 3.3 标准品

玉米赤霉烯酮 $\mathrm{(C_{18}H_{22}O_5}$ ,CAS号：17924-92-4)，纯度 $\geq 98.0\%$ 。或经国家认证并授予标准物质证

书的标准物质。

# 3.4 标准溶液配制

3.4.1 标准储备液：准确称取适量的标准品（精确至 $0.0001\mathrm{g}$ ），用乙腈溶解，配制成浓度为 $100~\mu \mathrm{g / mL}$ 的标准储备液， $-18^{\circ}\mathrm{C}$ 以下避光保存。  
3.4.2 系列标准工作液：根据需要准确吸取适量标准储备液，用流动相稀释，配制成 $10\mathrm{ng / mL},50\mathrm{ng / mL},100\mathrm{ng / mL},200\mathrm{ng / mL},500\mathrm{ng / mL}$ 的系列标准工作液， $4^{\circ}\mathrm{C}$ 避光保存。

# 3.5 材料

3.5.1 玉米赤霉烯酮免疫亲和柱：柱规格 $1\mathrm{mL}$ 或 $3\mathrm{mL}$ ，柱容量 $\geqslant 1500\mathrm{ng}$ ，或等效柱。  
3.5.2 玻璃纤维滤纸：直径 $11\mathrm{cm}$ ，孔径 $1.5\mu \mathrm{m}$ ，无荧光特性。

# 4 仪器和设备

4.1 高效液相色谱仪：配有荧光检测器。  
4.2 高速粉碎机：转速 $\geqslant 12000\mathrm{r / min}$   
4.3 均质器：转速 $\geqslant 12000\mathrm{r / min}$   
4.4 高速均质器：转速 $18000\mathrm{r / min}\sim 22000\mathrm{r / min}$   
4.5 氮吹仪。  
4.6 空气压力泵。  
4.7 玻璃注射器： $10~\mathrm{mL}$   
4.8 天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。

# 5 分析步骤

# 5.1 提取

# 5.1.1 粮食和粮食制品

称取 $40.0\mathrm{g}$ 粉碎试样(精确到 $0.1\mathrm{g}$ )于均质杯中，加入 $4\mathrm{g}$ 氯化钠和 $100\mathrm{mL}$ 提取液，以均质器(4.3)高速搅拌提取 $2\mathrm{min}$ ，定量滤纸过滤。移取 $10.0\mathrm{mL}$ 滤液加入 $40\mathrm{mL}$ 水稀释混匀，经玻璃纤维滤纸过滤至滤液澄清，滤液备用。

# 5.1.2 酱油、醋、酱及酱制品

称取 $25.0\mathrm{g}$ （精确到 $0.1\mathrm{g}$ )混匀的试样，用乙腈定容至 $100.0~\mathrm{mL}$ ，超声提取 $2\mathrm{min}$ ，定量滤纸过滤。移取 $10.0~\mathrm{mL}$ 滤液并加入 $40~\mathrm{mL}$ 水稀释混匀，经玻璃纤维滤纸过滤至滤液澄清，滤液备用。

# 5.1.3 大豆、油菜籽、食用植物油

准确称取试样 $40.0\mathrm{g}$ （准确到 $0.1\mathrm{g}$ ）（大豆需要磨细且粒度 $\leqslant 2\mathrm{mm}$ 于均质杯中，加入 $4.0\mathrm{g}$ 氯化钠和 $100\mathrm{mL}$ 提取液，以高速均质器(4.4)高速搅拌提取 $1\mathrm{min}$ ，定量滤纸过滤。移取 $10.0\mathrm{mL}$ 滤液并加入 $40\mathrm{mL}$ 水稀释，经玻璃纤维滤纸过滤至滤液澄清，滤液备用。

# 5.1.4 酒类

取脱气酒类试样(含二氧化碳的酒类使用前先置于 $4^{\circ} \mathrm{C}$ 冰箱冷藏 $30 \mathrm{~min}$ , 过滤或超声脱气)或其他

不含二氧化碳的酒类试样 $20.0 \mathrm{~g}$ (精确到 $0.1 \mathrm{~g}$ ) 于 $50 \mathrm{~mL}$ 容量瓶中, 用乙腈定容至刻度, 摇匀。移取 $10.0 \mathrm{~mL}$ 滤液并加入 $40 \mathrm{~mL}$ 水稀释混匀, 经玻璃纤维滤纸过滤至滤液澄清, 滤液备用。

# 5.2 净化

# 5.2.1 粮食和粮食制品

将免疫亲和柱连接于玻璃注射器下，准确移取 $10.0\mathrm{mL}$ （相当于 $0.8\mathrm{g}$ 样品）5.1.1中的滤液，注入玻璃注射器中。将空气压力泵与玻璃注射器连接，调节压力使溶液以1滴/s～2滴/s的流速缓慢通过免疫亲和柱，直至有部分空气进入亲和柱中。用 $5\mathrm{mL}$ 水淋洗柱子1次，流速为1滴/s～2滴/s，直至有部分空气进入亲和柱中，弃去全部流出液。准确加入 $1.5\mathrm{mL}$ 甲醇洗脱，流速约为1滴/s。收集洗脱液于玻璃试管中，于 $55^{\circ}C$ 以下氮气吹干后，用 $1.0\mathrm{mL}$ 流动相溶解残渣，供液相色谱测定。

# 5.2.2 酱油、醋、酱及酱制品，酒类

将免疫亲和柱连接于玻璃注射器下，准确移取 $10.0\mathrm{mL}5.1.2$ 或5.1.4中的滤液，注入玻璃注射器中。将空气压力泵与玻璃注射器相连接，调节压力使溶液以1滴/s～2滴/s的流速缓慢通过免疫亲和柱，直至有部分空气进入亲和柱中。依次用 $10\mathrm{mL}$ PBS清洗缓冲液和 $10\mathrm{mL}$ 水淋洗免疫亲和柱，流速为1滴/s～2滴/s，直至空气进入亲和柱中，弃去全部流出液。准确加入 $1.0\mathrm{mL}$ 甲醇洗脱，流速约为1滴/s。收集洗脱液于玻璃试管中，于 $55^{\circ}\mathrm{C}$ 以下氮气吹干后，用 $1.0\mathrm{mL}$ 流动相溶解残渣，供液相色谱测定。

# 5.2.3 大豆、油菜籽、食用植物油

将免疫亲和柱连接于玻璃注射器下，准确移取 $10.0\mathrm{mL}5.1.3$ 中的滤液，注入玻璃注射器中。将空气压力泵与玻璃注射器相连接，调节压力使溶液以1滴/s～2滴/s的流速缓慢通过免疫亲和柱，直至有部分空气进入亲和柱中。依次用 $10~\mathrm{mL}$ PBS/吐温-20缓冲液和 $10~\mathrm{mL}$ 水淋洗免疫亲和柱，流速为1滴/s～2滴/s，直至空气进入亲和柱中，弃去全部流出液。准确加入 $1.5\mathrm{mL}$ 甲醇洗脱，流速约为1滴/s。收集洗脱液于干净的玻璃试管中，于 $55^{\circ}\mathrm{C}$ 以下氮气吹干后，用 $1.0\mathrm{mL}$ 流动相溶解残渣，供液相色谱测定。

# 5.3 空白试验

不称取试样，按5.1和5.2的步骤做空白试验。应确认不含有干扰待测组分的物质。

# 5.4 测定

# 5.4.1 高效液相色谱参考条件

高效液相色谱参考条件如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $150~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒度 $4\mu \mathrm{m}$ ，或等效柱；  
b）流动相：乙腈-水-甲醇 $(46:46:8$ ，体积比）；  
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d) 检测波长: 激发波长 $274 \mathrm{~nm}$ , 发射波长 $440 \mathrm{~nm}$ ;  
e) 进样量: $100 \mu \mathrm{L}$ ;   
f) 柱温：室温。

# 5.4.2 标准曲线的制作

将系列玉米赤霉烯酮标准工作液按浓度从低到高依次注入高效液相色谱仪，得到相应的峰面积。

以目标物质的浓度为横坐标，目标物质的峰面积为纵坐标绘制标准曲线。

# 5.4.3 试样溶液的测定

将待测试样溶液注入高效液相色谱仪，得到玉米赤霉烯酮的峰面积。由标准曲线得到试样溶液中玉米赤霉烯酮的浓度。

# 6 分析结果的表述

试样中玉米赤霉烯酮的含量按式(1)计算：

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0} \times f \quad \dots \tag {1}
$$

式中：

$X$ ——试样中玉米赤霉烯酮的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——试样测定液中玉米赤霉烯酮的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样测定液的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000 单位换算常数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ ；

$f$ ——稀释倍数。

计算结果需扣除空白值，保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

本方法对粮食和粮食制品中玉米赤霉烯酮的检出限为 $5\mu \mathrm{g / kg}$ ，定量限为 $17~\mu \mathrm{g / kg}$ 。酒类中玉米赤霉烯酮的检出限为 $20~\mu \mathrm{g / kg}$ ，定量限为 $66~\mu \mathrm{g / kg}$ 。酱油、醋、酱及酱制品中玉米赤霉烯酮的检出限为 $50~\mu \mathrm{g / kg}$ ，定量限为 $165~\mu \mathrm{g / kg}$ 。大豆、油菜籽、食用植物油中玉米赤霉烯酮的检出限为 $10~\mu \mathrm{g / kg}$ ，定量限 $33~\mu \mathrm{g / kg}$ 。

# 第二法 荧光光度法

# 9 原理

用乙腈溶液提取试样中的玉米赤霉烯酮，经免疫亲和柱净化后，加入氯化铝溶液进行衍生。洗脱液通过荧光光度计测定。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一水。

10.2.3 氯化铝衍生溶液：称取 $2.5\mathrm{g}$ 氯化铝溶解于 $50~\mathrm{mL}$ 甲醇中。  
10.2.4 硫酸溶液 $(0.05\mathrm{mol} / \mathrm{L})$ ：取 $2.8\mathrm{mL}$ 硫酸，缓慢加入适量水中，冷却后定容至 $1000~\mathrm{mL}$ 。  
10.2.5 荧光光度计校准溶液：称取 $3.40\mathrm{g}$ 硫酸奎宁，用 $0.05\mathrm{mol} / \mathrm{L}$ 硫酸溶液稀释至 $100\mathrm{mL}$ ，此溶液的荧光光度计读数相当于 0.45。

# 10.3 标准品

玉米赤霉烯酮 $\left(\mathrm{C}_{18} \mathrm{H}_{22} \mathrm{O}_{5}\right)$ , CAS 号: 17924-92-4), 纯度 $\geqslant 98.0\%$ 。或经国家认证并授予标准物质证书的标准物质。

# 10.4 材料

10.4.1 玉米赤霉烯酮免疫亲和柱：柱规格 $1\mathrm{mL}$ 或 $3\mathrm{mL}$ ，柱容量 $\geqslant 1500\mathrm{ng}$ ，或等效柱。  
10.4.2 玻璃纤维滤纸：直径 $11\mathrm{cm}$ ，孔径 $1.5\mu \mathrm{m}$ ，无荧光特性。

# 11 仪器和设备

11.1 荧光光度计。  
11.2 高速均质器： $18000\mathrm{r / min}\sim 22000\mathrm{r / min}$   
11.3 空气压力泵。  
11.4 玻璃注射器： $10~\mathrm{mL}$   
11.5 天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。

# 12 分析步骤

# 12.1 提取

准确称取试样 $40.0\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）（大豆需要磨细且粒度 $\leqslant 2\mathrm{mm}$ 于均质杯中，加入 $4.0\mathrm{g}$ 氯化钠和 $100\mathrm{mL}$ 提取液(10.2.1)，以高速均质器高速搅拌提取 $1\mathrm{min}$ ，定量滤纸过滤。移取 $10.0\mathrm{mL}$ 滤液并加入 $40\mathrm{mL}$ PBS/吐温-20缓冲液稀释混匀，经玻璃纤维滤纸过滤至滤液澄清，滤液备用。

# 12.2 净化

将免疫亲和柱连接于玻璃注射器下，准确移取 $10.0\mathrm{mL}$ 滤液，注入玻璃注射器中。将空气压力泵与玻璃注射器相连接，调节压力使溶液以1滴/s～2滴/s的流速缓慢通过免疫亲和柱，直至有部分空气进入亲和柱中。依次用 $10\mathrm{mL}$ PBS/吐温-20缓冲液和 $10\mathrm{mL}$ 水淋洗免疫亲和柱，流速为1滴/s～2滴/s，直至空气进入亲和柱中，弃去全部流出液。准确加入 $1.0\mathrm{mL}$ 甲醇洗脱，流速约为1滴/s。收集洗脱液于玻璃试管中，于 $55^{\circ}C$ 以下氮气吹干后，用 $1.0\mathrm{mL}$ 流动相溶解残渣供测定。

# 12.3 空白试验

不称取试样，按12.1和12.2的步骤做空白试验。应确认不含有干扰待测组分的物质。

# 12.4 测定

# 12.4.1 测定条件

激发波长 $360\mathrm{nm}$ ，发射波长 $450\mathrm{nm}$ 。

# 12.4.2 荧光光度计校准

以 $0.05\mathrm{mol} / \mathrm{L}$ 硫酸溶液为空白调零，以荧光光度计校准溶液进行校准。

# 12.4.3 样液测定

在12.2得到的样液中加入 $1.0\mathrm{mL}$ 氯化铝衍生溶液，立即置于荧光光度计中读取玉米赤霉烯酮的浓度。

# 13 分析结果的表述

试样中玉米赤霉烯酮的含量按式(2)算：

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0} \times f \tag {2}
$$

式中：

$X$ ——试样中玉米赤霉烯酮的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——试样测定液中玉米赤霉烯酮的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样测定液的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——单位换算常数；

$m$ ——试样的称样量，单位为克(g)；

$f$ ——稀释倍数。

计算结果需扣除空白值，保留两位有效数字。

# 第三法 液相色谱-质谱法

# 16 原理

# 19 分析步骤

# 19.1 试样制备

注：在制样的操作过程中，应防止样品污染或发生残留物含量的变化。

# 19.1.1 肌肉和内脏

取 $500\mathrm{g}$ 样品，用组织捣碎机充分捣碎混匀，均分成两份，分别装入洁净容器作为试样，密封。于 $-18^{\circ}\mathrm{C}$ 以下避光保存。

# 19.1.2 牛奶

取 $500\mathrm{g}$ 样品，充分混匀，均分成两份，分别装入洁净容器作为试样，密封。于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 避光保存。

# 19.1.3 鸡蛋

取 $500\mathrm{g}$ 样品，去壳后用组织捣碎机搅拌充分混匀，均分成两份，分别装入洁净容器作为试样，密封。于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 避光保存。

# 19.2 水解

称取 $5.0\mathrm{g}$ 试样(精确至 $0.1\mathrm{g}$ ) 于 $50~\mathrm{mL}$ 具塞离心管中，加入 $10~\mathrm{mL}~0.05~\mathrm{mol / L}$ 乙酸钠缓冲溶液和 $0.025~\mathrm{mL}\beta$ -葡萄糖苷酸/硫酸酯复合酶，旋涡混匀，于 $37^{\circ}\mathrm{C}$ 水浴中振荡 $12\mathrm{h}$ 。

# 19.3 提取

水解后加入 $15\mathrm{mL}$ 无水乙醚，振荡提取 $5\mathrm{min}$ 后，以 $4000\mathrm{r / min}$ 离心 $2\mathrm{min}$ ，将上清液转移至浓缩瓶中，再用 $15\mathrm{mL}$ 无水乙醚重复提取一次，合并上清液， $40^{\circ}\mathrm{C}$ 以下旋转浓缩至近干。加入 $1\mathrm{mL}$ 三氯甲烷溶解残渣，超声波助溶 $2\mathrm{min}$ 后，转入 $10\mathrm{mL}$ 离心管中，再用 $3\mathrm{mL}0.5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液润洗浓缩瓶后转移至同一离心管中，涡旋混匀，以 $4000\mathrm{r / min}$ 离心 $2\mathrm{min}$ ，吸取上层氢氧化钠溶液。再用 $3\mathrm{mL}0.5\mathrm{mol} / \mathrm{L}$ 氢氧化钠重复润洗、萃取一次，合并氢氧化钠萃取液，加入 $1\mathrm{mL}$ 磷酸-水溶液，混匀后待净化。

# 19.4 净化

使用前依次用 $5\mathrm{mL}$ 甲醇和 $5\mathrm{mL}$ 水预淋洗固相萃取柱。

将19.3中样品提取液转入固相萃取柱。用 $5\mathrm{mL}$ 水、 $5\mathrm{mL}$ 甲醇-水溶液淋洗，弃去流出液；再用 $10~\mathrm{mL}$ 甲醇进行洗脱，收集洗脱液。整个固相萃取净化过程控制流速不超过 $2\mathrm{mL} / \mathrm{min}$ 。洗脱液在 $40^{\circ}\mathrm{C}$ 以下用氮气吹干。残留物用 $1.0~\mathrm{mL}$ 乙腈溶解，涡旋混匀后，过 $0.2\mu \mathrm{m}$ 微孔滤膜，供仪器检测。

# 19.5 基质标准溶液的制备

称取5份 $5.0\mathrm{g}$ 空白试样(精确至 $0.1\mathrm{g}$ )于 $50~\mathrm{mL}$ 具塞离心管中，分别加入相应体积的玉米赤霉烯酮标准工作溶液，配制成浓度为 $1\mu \mathrm{g / kg},5\mu \mathrm{g / kg},10\mu \mathrm{g / kg},50\mu \mathrm{g / kg}$ 和 $100\mu \mathrm{g / kg}$ 的基质标准溶液，然后按19.2、19.3、19.4操作。

# 20 分析结果的表述

$$
X = \frac {\rho \times V}{m \times 1 0 0 0} \times 1 0 0 0 \quad \dots \tag {3}
$$

式中：

$X$ ——试样中玉米赤霉烯酮的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——从基质标准曲线得到的试样测定液中玉米赤霉烯酮的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样测定液的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度应在分析纯以上，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，实验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 酶活力的测定

# A.2.1 方法原理

溶菌酶可水解细菌的细胞壁，造成藤黄微球菌的溶解而引起溶液吸光度值的降低。

一个溶菌酶活力单位定义为 $25\mathrm{^\circ C}$ ， $\mathrm{pH}6.2$ 条件下，使用藤黄微球菌悬浊液在 $450~\mathrm{nm}$ 处每分钟引起吸光度变化为0.001所需溶菌酶的量。

# A.2.2 试剂和材料

A.2.2.1 藤黄微球菌：ATCC4698或CICC10680。

A.2.2.2 $0.1\mathrm{mol} / \mathrm{L}$ 磷酸盐缓冲液：pH6.2。

称取 $11.70\mathrm{g}$ 磷酸二氢钠 $\mathrm{(NaH_2PO_4\cdot 2H_2O)}$ 、 $7.86\mathrm{g}$ 磷酸氢二钠 $\mathrm{(Na_2HPO_4\cdot 12H_2O)}$ 及 $0.372\mathrm{g}$ 乙二胺四乙酸二钠（EDTA-2Na）于无菌水中并稀释定容至 $1000~\mathrm{mL}$ 。调整缓冲溶液 $\mathrm{pH}$ 至 $6.2\pm 0.1$

注：用小份缓冲溶液检查 $\mathrm{pH}$ ，以避免缓冲溶液被污染。如果需要，通过加入更多的磷酸二氢钠溶液或磷酸氢二钠溶液调整 $\mathrm{pH}$ 。

A.2.2.3 溶菌酶标准品：蛋清溶菌酶。

A.2.2.4 底物溶液：用磷酸盐缓冲液制备 $50~\mathrm{mL}$ 藤黄微球菌悬浊液。使用前，底物于 $37^{\circ}C$ 培养 $30\mathrm{min}$ 该底物溶液室温下可稳定 $2\textrm{h}$ 。以磷酸盐缓冲液调分光光度计零点，然后测定底物溶液的吸光度， $450~\mathrm{nm}$ 下读数应为 $0.70\pm 0.1$

# A.2.3 仪器和设备

A.2.3.1 分光光度计：精度 $\pm 0.001$

A.2.3.2 pH计。

注：所用的器皿应保持无菌，保证工作环境的清洁。

# A.2.4 分析步骤

# A.2.4.1 试样溶液的制备

准确称取 $100\mathrm{mg}\pm 0.1\mathrm{mg}$ 试样，置于 $50~\mathrm{mL}$ 容量瓶中，用约 $25~\mathrm{mL}$ 磷酸盐缓冲液搅拌溶解并稀释定容，充分混匀。再转移 $3\mathrm{mL}$ 上述试样制备溶液至 $100~\mathrm{mL}$ 容量瓶中，用磷酸盐缓冲液搅拌溶解并稀释定容。

# A.2.4.2 标准溶液的制备

精确称取 $50\mathrm{mg}$ 蛋清溶菌酶标准品于 $50~\mathrm{mL}$ 容量瓶中，用约 $25~\mathrm{mL}$ 磷酸盐缓冲液搅拌溶解并稀释定容，充分混匀(如果需要，冷冻该溶液以备后续测定)。转移 $3\mathrm{mL}$ 上述标准制备溶液至 $100~\mathrm{mL}$ 容量瓶中，用磷酸盐缓冲液搅拌溶解并稀释定容。

# A.2.4.3 测定

取3份标准溶液和3份试样溶液进行测定。

$25\,^{\circ}\mathrm{C}$ 室温下，将 $1\,\mathrm{cm}$ 比色皿放入分光光度计，用磷酸盐缓冲液调整吸光度零点。吸 $2.9\,\mathrm{mL}$ 底物溶液于比色皿，最初 $450\,\mathrm{nm}$ 处吸光度应为 $0.70 \pm 0.10, 3\,\mathrm{min}$ 之内初始吸光度值变化应小于或等于0.003时，方可开始测定。吸取 $0.1\,\mathrm{mL}$ 标准溶液加入底物溶液，充分混合。记录 $3\,\mathrm{min}$ 吸光度值的变化，每 $15\,\mathrm{s}$ 记录一次吸光度值。每分钟吸光度值变化应在 $0.03 \sim 0.08$ ，若不在要求范围需调整试样溶液的浓度。重复操作测定试样溶液。

反应 $1\mathrm{min}$ 后稳定，计算时忽略最初 $1\mathrm{min}$ 的读数。

# A.2.4.4 结果计算

酶活力 $X$ ，按式(A.1)计算：

$$
X = \frac {\left(A _ {1} - A _ {2}\right)}{2 \times m \times 0 . 0 0 1} \quad \dots \tag {A.1}
$$

式中：

$A_{1}$ ——试样在 $450 \mathrm{~nm}$ 处反应 $1 \mathrm{~min}$ 时的吸光度；

$A_{2}$ ——试样在 $450 \mathrm{~nm}$ 处反应 $3 \mathrm{~min}$ 时的吸光度；

$m$ ——用于分析的试样制备溶液中的溶菌酶质量，单位为毫克 $(\mathrm{mg})$ ；

2 ——获得 $1\mathrm{min}$ 和 $3\mathrm{min}$ 吸光度读数所用的时间，单位为分钟 $(\mathrm{min})$ ；

0.001——由单位溶菌酶每分钟引起的吸光度降低的值。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# A.3 $\mathbf{pH}$ 的测定

# A.3.1 仪器

A.3.1.1 酸度计：精度 $0.01\mathrm{pH}$ ，备有玻璃电极和甘汞电极（或复合电极）。

# A.3.2 分析步骤

按仪器使用说明书调试和校正酸度计。

用无菌水配制 $15\mathrm{g / L}$ 的溶菌酶待测液（液体产品直接测定）。用水冲洗电极探头，用滤纸轻轻吸干，将电极插入待测液中，调节温度调节器，使仪器指示温度与待测液温度相同，稳定后读数。

所得结果表示至一位小数。

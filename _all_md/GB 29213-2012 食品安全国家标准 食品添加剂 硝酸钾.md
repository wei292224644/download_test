# 食品安全国家标准

# 食品添加剂 硝酸钾

# 1 范围

本标准适用于以硝酸铵、硝酸钠和氯化钾为原料，经复分解法或离子交换法等方法生产的，再经精制而得的食品添加剂硝酸钾。

# 2 化学名称、分子式和相对分子质量

# 2.1 化学名称

硝酸钾

# 2.2 分子式

$\mathrm{KNO}_3$

# 2.3 相对分子质量

101.1（按2007年国际相对原子质量）

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008中规定的三级水。本标准试验中所需标准滴定溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A. 3.1.1 乙酸。

A.3.1.2 乙醇。

A.3.1.3 酒石酸钠溶液： $100\mathrm{g / L}$

A.3.1.4 氨水溶液： $4 + 6$

A. 3.1.5 高锰酸钾溶液: $c(1 / 5 \mathrm{KMnO}_{4}) = 0.1 \mathrm{~mol} / \mathrm{L}$ 。称取 $3.161 \mathrm{~g}$ 高锰酸钾, 用适量水溶解后, 再稀释定容至 $1000 \mathrm{~mL}$ 。

# A.3.2 鉴别方法

# A.3.2.1 钾离子的鉴别

称取 $1 \mathrm{~g}$ 试样溶于 $10 \mathrm{~mL}$ 水中, 加入酒石酸钠, 生成白色晶体沉淀, 在氨试样溶液中、碱性氢氧化物和碳酸盐溶液中可被溶解。搅拌、用玻璃棒摩擦试管内壁或加入少量的乙酸或乙醇可以加速沉淀的生成。

# A.3.2.2 硝酸根离子的鉴别

A. 3.2.2.1 称取 $1 \mathrm{~g}$ 试样溶于 $10 \mathrm{~mL}$ 水中, 滴入高锰酸钾溶液中不会引起褪色反应。

A. 3.2.2.2 将试样置于铜箔上，滴加硫酸，加热时，放出棕红色的烟。

# A. 4 硝酸钾 $\left(\mathrm{KNO}_{3}\right)$ 含量的测定

# A.4.1 试剂和材料

A.4.1.1 氢氧化钠溶液： $400\mathrm{g / L}$   
A.4.1.2 硫酸溶液： $c(\mathrm{H}_2\mathrm{SO}_4) = 0.1\mathrm{mol / L}$   
A.4.1.3 氢氧化钠标准滴定溶液： $c$ （ $\mathrm{NaOH}) = 0.1\mathrm{mol / L}$   
A.4.1.4 甲基红-亚甲基蓝混合指示液。  
A.4.1.5 无氨的水。  
A.4.1.6 定氮合金：粒径不大于 $0.85\mathrm{mm}$

# A.4.2 仪器和设备

A.4.2.1 蒸馏仪器：按图A.1配备或其他具有相同蒸馏能力的定氮蒸馏仪器。   
A.4.2.2 沸石。

# A.4.3 分析步骤

称取已于 $105~\mathrm{^\circ C}\pm 2$ ℃干燥4h的 $0.4\mathrm{g}$ 样品，精确至 $0.0002\mathrm{g}$ 。在蒸馏瓶中加 $300~\mathrm{mL}$ 无氨的水，摇动使试样溶解。加入约 $3\mathrm{g}$ 定氮合金和几粒沸石，将蒸馏瓶连接于蒸馏装置上，接口处均应涂上硅脂以防止漏气。在吸收瓶中用移液管移入 $50~\mathrm{mL}$ 硫酸溶液，加4滴 $\sim 5$ 滴甲基红-亚甲基蓝混合指示液，并与蒸馏装置连接。在吸收瓶中加适量水，以保证导流管出口位于吸收液液面下约 $1.5\mathrm{cm}$ 。通过蒸馏装置的分液漏斗加入 $15~\mathrm{mL}$ 氢氧化钠溶液，在溶液将流尽时加入 $20~\mathrm{mL}\sim 30~\mathrm{mL}$ 水冲洗漏斗，剩 $3\mathrm{mL}\sim 5\mathrm{mL}$ 水时关闭活塞。静置 $10\mathrm{min}$ 后，开通冷却水，同时加热，沸腾时根据泡沫产生程度调节供热强度，避免泡沫溢出或液滴带出。蒸馏出至少 $150~\mathrm{mL}$ 馏出液后，用 $\mathsf{pH}$ 试纸检查液滴，如不呈碱性则结束蒸馏。用少量水冲洗导

流管的下端，取下吸收瓶。用氢氧化钠标准滴定溶液滴定吸收瓶中的吸收液呈灰绿色为终点。

同时进行空白试验。空白试验除不加试样外，其他操作及加入试剂的种类和量（标准滴定溶液除外）与测定试验相同。

# A.4.4 结果计算

硝酸钾（ $\mathrm{KNO}_3$ ）含量的质量分数 $w_1$ ，按式（A.1）计算：

$$
w _ {1} = \frac {\left[ \left(V _ {0} - V _ {1}\right) / 1000 \right] c M}{m} \times 100 \% \dots \tag{A.1}
$$

式中：

$V_{0}$ ——空白试验所消耗的氢氧化钠标准滴定溶液的体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——滴定试样溶液所消耗的氢氧化钠标准滴定溶液的体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——氢氧化钠标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——硝酸钾（ $\mathrm{KNO}_3$ ）的摩尔质量的数值，单位为克每摩尔(g/mol)（ $M = 101.1$ ）；

$m$ ——试样的质量的数值，单位为克(g)；

1000——换算因子。

实验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A.5 干燥减量的测定

# A.5.1 仪器和设备

A.5.1.1 称量瓶： $\phi 30\mathrm{mm}\times 25\mathrm{mm}$   
A.5.1.2 电热恒温干燥箱：能控制温度在 $105\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}$ 。

# A.5.2 分析步骤

在预先于 $105~\mathrm{^\circ C}\pm 2$ ℃下干燥至质量恒定的称量瓶中称取约 $2\mathrm{g}$ 试样，精确到 $0.0002\mathrm{g}$ ，在 $105~\mathrm{^\circ C}\pm$ $2^{\circ}C$ 下干燥 $4\mathrm{h}$ ，冷却 $30\mathrm{min}$ ，称量。

# A.5.3 结果计算

干燥减量的质量分数 $w_{2}$ ，按式（A.2）计算：

$$
w _ {2} = \frac {m _ {1} - m _ {2}}{m} \times 100 \% \dots \tag{A.2}
$$

式中：

$m_{1}$ ——试样和称量瓶的质量的数值，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——干燥后试样和称量瓶质量的数值，单位为克（g）；

$m$ ——试样的质量的数值，单位为克（g）。

实验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$

# A. 6 氯酸盐的测定

称取约 $0.1 \mathrm{~g}$ 已于 $105^{\circ} \mathrm{C} \pm 2^{\circ} \mathrm{C}$ 下干燥 $4 \mathrm{~h}$ 后的试样于烧杯中，加入 $2 \mathrm{~mL}$ 硫酸，试样保持白色，没有气味和气体产生，即为通过试验。

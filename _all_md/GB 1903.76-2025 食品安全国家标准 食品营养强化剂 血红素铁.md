# 1 范围

本标准适用于以检验检疫合格的动物血液或经离心分离得到的血球液为原料，经酶解、分离、干燥制得的食品营养强化剂血红素铁。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

血红素铁

# 2.2 分子式

$$
\mathrm {C} _ {3 4} \mathrm {H} _ {3 2} \mathrm {F e N} _ {4} \mathrm {O} _ {4}
$$

# 附录A

# 检验方法

# A.1 一般规定

本标准的检验方法中所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用溶液在未标明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 氢氧化钠。

A.2.1.2 吡啶。

A.2.1.3 次亚硫酸钠。

A.2.1.4 硝酸：色谱纯。

A.2.1.5 氨水。

A.2.1.6 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：称取 $40.0\mathrm{g}$ 氢氧化钠，用无二氧化碳的水溶解并定容至 $1\mathrm{L}$ 。

A.2.1.7 吡啶氢氧化钠溶液：吸取 $100~\mathrm{mL}$ 吡啶，加入 $30~\mathrm{mL}~1~\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液，用水稀释至 $300~\mathrm{mL}$ 。

# A.2.2 仪器和设备

A.2.2.1 分析天平：感量 $0.0001\mathrm{g}$

A.2.2.2 紫外分光光度计。

# A.2.3 鉴别方法

# A.2.3.1 呈色试验

A.2.3.1.1 称取 $0.01\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ) 血红素铁试样, 溶解于 $50\mathrm{mL}$ 的吡啶氢氧化钠溶液中, 取 $5\mathrm{mL}$ 加入 $15\mathrm{mg}$ 的次亚硫酸钠, 溶液应呈现红色。

A.2.3.1.2 称取 $0.01 \mathrm{~g}$ (精确至 $0.0001 \mathrm{~g}$ ) 血红素铁试样置于 $100 \mathrm{~mL}$ 烧杯中, 加入 $5 \mathrm{~mL}$ 硝酸, 加热, 溶液应呈现黄色, 冷却后用氨水碱化, 溶液应呈现橙黄色。

# A.2.3.2 紫外光谱法

称取 $0.01\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ) 血红素铁试样, 溶解于 $50~\mathrm{mL}$ 的吡啶氢氧化钠溶液中, 在紫外光扫描谱图中, 在 $399~\mathrm{nm}$ 波长处具有最大吸收。

# A.3 血红素铁 $\left(\mathbf{C}_{34} \mathbf{H}_{32} \mathbf{FeN}_{4} \mathbf{O}_{4}\right)$ 含量的测定

# A.3.1 方法提要

样品中的血红素铁用吡啶氢氧化钠溶解，采用液相色谱分离，紫外或二极管阵列检测器检测，保留时间定性，峰面积外标法定量。

# A.3.2 试剂和材料

除另有说明，所用试剂均为分析纯，水为GB/T6682规定的一级水。

A.3.2.1 甲醇：色谱纯。

A.3.2.2 甲酸：色谱纯。

A.3.2.3 氢氧化钠。

A.3.2.4 吡啶。

A.3.2.5 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：称取 $40.0\mathrm{g}$ 氢氧化钠，用无二氧化碳的水溶解并定容至 $1\mathrm{L}$ 。

A.3.2.6 吡啶氢氧化钠溶液：吸取 $100~\mathrm{mL}$ 吡啶，加入 $30~\mathrm{mL}~1~\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液，用水稀释至 $300~\mathrm{mL}$ 。  
A.3.2.7 羟高铁血红素 $\mathrm{(C_{34}H_{33}FeN_4O_5}$ ，CAS号：15489-90-4)标准品：含量 $\geq 98.0\%$

A.3.2.8 羟高铁血红素标准储备液：准确称取 $100^{\circ}\mathrm{C}$ 干燥 $3\mathrm{h}$ 的羟高铁血红素 $0.1028\mathrm{g}$ ，用吡啶氢氧化钠溶液(A.3.2.6)溶解并定容至 $100~\mathrm{mL}$ ，此溶液相当于血红素铁 $1\mathrm{mg / mL}$ 。在棕色试剂瓶中 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 避光保存，有效期 $7\mathrm{d}$ 。

A.3.2.9 滤膜： $0.45\mu \mathrm{m}$ ，水相。

# A.3.3 仪器和设备

A.3.3.1 高效液相色谱仪：配紫外检测器或二极管阵列检测器。  
A.3.3.2 分析天平：感量 $0.0001\mathrm{g}$   
A.3.3.3 超声波发生器：功率大于 $180\mathrm{W}$   
A.3.3.4 离心机：大于 $4000\mathrm{r / min}$   
A.3.3.5 恒温干燥箱。

# A.3.4 液相色谱参考条件

A.3.4.1 色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ 或等效色谱柱。  
A.3.4.2 流动相：甲醇： $0.1\%$ 甲酸水溶液 $= 70:30$ （体积比）。  
A.3.4.3 流速： $1.0~\mathrm{mL / min}$   
A.3.4.4 柱温： $30^{\circ}C$ 。  
A.3.4.5 进样量： $5.0\mu \mathrm{L}$   
A.3.4.6 检测波长： $399~\mathrm{nm}$

# A.3.5 分析步骤

准确称取试样 $0.1\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），用吡啶氢氧化钠溶液(A.3.2.6)溶解，避光条件下超声波助溶 $10\mathrm{min}$ ，冷却至室温，转移至 $50~\mathrm{mL}$ 容量瓶中，用吡啶氢氧化钠溶液定容至刻度，混匀，量取该溶液 $1\mathrm{mL}$ 置 $10~\mathrm{mL}$ 容量瓶中，用吡啶氢氧化钠溶液稀释至刻度， $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液，滤膜过滤，液相色谱测定。并做空白试验，除不加试样外，其他操作与测定试样相同。

# A.3.6 定性分析

依据保留时间一致性进行定性识别的方法，根据羟高铁血红素标准品的保留时间，偏差在 $\pm 2.5\%$ 以内，确定样品中血红素铁的色谱峰（见附录B）。

# A.3.7 定量分析

分别取羟高铁血红素标准储备液(A.3.2.8) $0.25\mathrm{mL}, 0.50\mathrm{mL}, 1.00\mathrm{mL}, 2.00\mathrm{mL}, 3.00\mathrm{mL}, 5.00\mathrm{mL}$ 于 $50\mathrm{mL}$ 容量瓶中，用吡啶氢氧化钠溶液(A.3.2.6)定容，配制成相当于血红素铁溶液质量浓度分别为 $5\mu \mathrm{g} / \mathrm{mL}, 10\mu \mathrm{g} / \mathrm{mL}, 20\mu \mathrm{g} / \mathrm{mL}, 40\mu \mathrm{g} / \mathrm{mL}, 60\mu \mathrm{g} / \mathrm{mL}, 100\mu \mathrm{g} / \mathrm{mL}$ 的标准溶液，液相色谱测定，以血红素铁浓度为横坐标，以色谱峰面积为纵坐标，绘制标准曲线。

用标准工作曲线对试样进行定量，标准工作溶液和试样中血红素铁的响应值均应在仪器的检测线性范围内。

在A.3.4的色谱条件下，羟高铁血红素标准品和血红素铁样品的色谱图参见附录B。

# A.3.8 结果计算

血红素铁 $\mathrm{(C_{34}H_{32}FeN_4O_4)}$ 含量的质量分数 $\varpi_{1}$ ，以 $\%$ 表示，按公式(A.1)计算：

$$
w _ {1} = \frac {\left(\rho - \rho_ {0}\right) \times V \times f}{m \times 10 ^ {6}} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$\rho$ ——试样溶液中血红素铁的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$\rho_0$ ——空白试验中血红素铁的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样定容总体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样稀释倍数；

$m$ ——试样称取质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $2.0\%$ 。

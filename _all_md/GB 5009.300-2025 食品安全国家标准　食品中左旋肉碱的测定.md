# 前言

本标准代替GB29989—2013《食品安全国家标准 婴幼儿食品和乳品中左旋肉碱的测定》。

本标准与GB29989—2013相比，主要变化如下：

——标准名称更改为《食品安全国家标准 食品中左旋肉碱的测定》；  
——更改了分光光度法；  
——增加了“第二法 柱前衍生-高效液相色谱法”；  
——增加了“第三法 液相色谱-串联质谱法”。

# 食品安全国家标准

# 食品中左旋肉碱的测定

# 1 范围

本标准规定了食品中左旋肉碱的测定方法。

第一法分光光度法适用于婴幼儿食品（不含特殊医学用途婴儿配方食品中的乳蛋白部分水解配方、乳蛋白深度水解配方和氨基酸配方）和乳品中左旋肉碱的测定。

第二法柱前衍生-高效液相色谱法适用于食品（不含特殊医学用途婴儿配方食品和特殊医学用途配方食品）中左旋肉碱的测定。

第三法液相色谱-串联质谱法适用于婴幼儿配方食品和特殊医学用途配方食品中左旋肉碱的测定。

# 第一法 分光光度法

# 2 原理

试样经过水提取，用高氯酸沉淀蛋白质后过滤。滤液中结合态的左旋肉碱经碱皂化游离出来，与乙酰辅酶A在乙酰肉碱转移酶的催化下反应生成乙酰肉碱和游离的辅酶A。游离的辅酶A和 $5,5^{\prime}$ -二硫双(2-硝基苯甲酸)反应生成黄色物质，其颜色深浅与游离的辅酶A含量成正比。因游离的辅酶A与左旋肉碱是等摩尔反应关系，可间接求出试样中左旋肉碱的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 高氯酸 $\mathrm{(HClO_4)}$   
3.1.2 氢氧化钠 $\mathrm{(NaOH)}$ 。  
3.1.3 氢氧化钾(KOH)。  
3.1.4 $5,5^{\prime}$ -二硫双（2-硝基苯甲酸） $(\mathrm{C_{14}H_8N_2O_8S_2})$ 。  
3.1.5 $N - 2$ -羟乙基哌嗪- $N - 2$ -乙烷磺酸 $(\mathrm{C_8H_{18}N_2O_4S})$ 。  
3.1.6 乙二胺四乙酸二钠 $(\mathrm{C_{10}H_{14}N_2Na_2}\cdot 2\mathrm{H_2O})$ 。  
3.1.7 乙酰辅酶A（AcetylCoA）：纯度 $\geqslant 85\%$ ，在 $2^{\circ}C\sim 8^{\circ}C$ 下保存。  
3.1.8 乙酰肉碱转移酶(CAT)：在 $2^{\circ} \mathrm{C} \sim 8^{\circ} \mathrm{C}$ 下保存。

# 3.2 试剂配制

3.2.1 高氯酸溶液 $(13\%)$ ：吸取 $13~\mathrm{mL}$ 高氯酸，用水稀释至 $100~\mathrm{mL}$ 。  
3.2.2 高氯酸溶液 $(3\%)$ ：吸取 $3\mathrm{mL}$ 高氯酸，用水稀释至 $100~\mathrm{mL}$ 。

3.2.3 氢氧化钠溶液 $(10\mathrm{mol} / \mathrm{L})$ ：称取 $40\mathrm{g}$ 氢氧化钠用水溶解，冷却后稀释至 $100~\mathrm{mL}$ 。  
3.2.4 氢氧化钾溶液 $(4\mathrm{mol} / \mathrm{L})$ ：称取 $22.4\mathrm{g}$ 氢氧化钾用水溶解，冷却后稀释至 $100~\mathrm{mL}$ 。  
3.2.5 显色储备液：分别称取 $50\mathrm{mg}5,5^{\prime}$ 二硫双(2-硝基苯甲酸)、 $5.96\mathrm{g}N - 2$ -羟乙基哌嗪- $N - 2 -$ 乙烷磺酸、 $185\mathrm{mg}$ 乙二胺四乙酸二钠溶于 $30~\mathrm{mL}$ 水中，用 $10\mathrm{mol / L}$ 氢氧化钠溶液调节 $\mathrm{pH}$ 至 $7.4\sim 7.6$ ，然后用水稀释至 $50~\mathrm{mL}$ 。 $4^{\circ}C$ 保存，有效期3个月。  
3.2.6 显色工作液：吸取 $5\mathrm{mL}$ 显色储备液用水稀释至 $25~\mathrm{mL}$   
3.2.7 乙酰辅酶A(AcetylCoA)溶液：称取 $20\mathrm{mg}$ 乙酰辅酶A溶于 $2\mathrm{mL}$ 水中。临用现配。  
3.2.8 乙酰肉碱转移酶(CAT)溶液：吸取 $100\mu \mathrm{L}$ 乙酰肉碱转移酶悬浮液，加水至 $2\mathrm{mL}$ ，混匀。临用现配。

# 3.3 标准品

左旋肉碱标准品 $\mathrm{(C_7H_{15}NO_3}$ ,CAS号：541-15-1)：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质。使用前于 $102^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 电热鼓风干燥箱中干燥 $2\textrm{h}$ 。

# 3.4 标准溶液配制

3.4.1 左旋肉碱标准储备液 $(1\mathrm{mg / mL})$ ：准确称取 $10\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ )左旋肉碱置于 $10~\mathrm{mL}$ 烧杯中，加入 $5\mathrm{mL}$ 水溶解，全部转移至 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。将溶液转移至棕色容器中， $4^{\circ}C$ 避光保存，有效期3个月。  
3.4.2 左旋肉碱标准中间液 $(80\mu \mathrm{g} / \mathrm{mL})$ ：吸取左旋肉碱标准储备液 $2.00~\mathrm{mL}$ ，置于 $25~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。将溶液转移至棕色容器中， $4^{\circ}\mathrm{C}$ 避光保存，有效期1个月。  
3.4.3 左旋肉碱系列标准工作液：分别吸取左旋肉碱标准中间液 $0.125 \mathrm{~mL}, 0.50 \mathrm{~mL}, 1.00 \mathrm{~mL}, 2.00 \mathrm{~mL}, 3.00 \mathrm{~mL}, 5.00 \mathrm{~mL}$ 置于 $25 \mathrm{~mL}$ 容量瓶中，用水定容至刻度，混匀，即得。该系列标准工作液质量浓度分别为 $0.4 \mu \mathrm{g} / \mathrm{mL}, 1.6 \mu \mathrm{g} / \mathrm{mL}, 3.2 \mu \mathrm{g} / \mathrm{mL}, 6.4 \mu \mathrm{g} / \mathrm{mL}, 9.6 \mu \mathrm{g} / \mathrm{mL}, 16 \mu \mathrm{g} / \mathrm{mL}$ 。临用现配。

# 4 仪器和设备

4.1 分析天平：感量分别为 $0.001\mathrm{g}$ 和 $0.01\mathrm{mg}$   
4.2 pH计。  
4.3 恒温水浴装置。  
4.4 分光光度计。  
4.5 电热鼓风干燥箱。

# 5 分析步骤

# 5.1 试样处理

5.1.1 固态试样：准确称取 $5.0\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )混合均匀的试样于烧杯中，用 $30~\mathrm{mL}40^{\circ}\mathrm{C}$ 温水溶解，转入 $100~\mathrm{mL}$ 容量瓶中。加入 $10~\mathrm{mL}$ 高氯酸溶液 $(13\%)$ ，混合均匀后静置 $20\mathrm{min}$ 。用水定容至刻度，混匀，用定量滤纸过滤。

取滤液 $20~\mathrm{mL}$ ，用氢氧化钾溶液 $(4\mathrm{mol} / \mathrm{L})$ 调节 $\mathrm{pH}$ 至 $12.5\sim 13.0$ 后，置于 $40^{\circ}\mathrm{C}$ 水浴中 $60\mathrm{min}$ 。冷却后先用高氯酸溶液 $(13\%)$ 将 $\mathrm{pH}$ 调至 $9\sim 10$ ，再用高氯酸溶液 $(3\%)$ 将 $\mathrm{pH}$ 调节至 $7.0\sim 7.5$ 。将样液转入 $50~\mathrm{mL}$ 容量瓶中，用水定容。混匀后置于 $4^{\circ}\mathrm{C}$ 冰箱中静置 $2\mathrm{h}$ 以上。将试样处理液从冰箱中取出放置至室温，取上清液，用 $0.45\mu \mathrm{m}$ 滤膜过滤后备用。

5.1.2 液态试样：准确称取 $25.0\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )混合均匀的试样，转入 $100~\mathrm{mL}$ 容量瓶中。后续步骤按5.1.1中“加入 $10~\mathrm{mL}$ 高氯酸溶液 $(13\%)$ ，混合均匀后静置 $20\mathrm{min}\dots$ ”处理。

# 5.2 标准曲线的绘制

吸取左旋肉碱系列标准工作液 $2.00\mathrm{mL}$ 于 $1\mathrm{cm}$ 比色皿中，加入 $0.80~\mathrm{mL}$ 显色工作液和 $100~\mu \mathrm{L}$ 乙酰辅酶A溶液，盖上比色皿盖，混合均匀后放入分光光度计中，分光光度计的波长调为 $412~\mathrm{nm},5\mathrm{min}$ 后归零。迅速加入 $100~\mu \mathrm{L}$ 乙酰肉碱转移酶溶液，混合均匀后放入分光光度计中，反应 $10\mathrm{min}$ 后记录吸光值。以左旋肉碱系列标准工作液的浓度为横坐标，以吸光值为纵坐标，制作标准曲线。

# 5.3 试样测定

取 $2.00\mathrm{mL}$ 试样处理液按5.2的步骤测定其吸光值。在标准曲线上查得试样待测液的浓度。

# 6 分析结果的表述

试样中左旋肉碱的含量按式(1)计算。

$$
X = \frac {c \times 5 0}{m \times V \times 1 0} \times 1 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——试样中左旋肉碱的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$c$ ——由标准曲线得到的试样溶液中左旋肉碱的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

50 ——样液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的取样质量，单位为克 $(\mathrm{g})$ ；

$V$ ——滤液的体积，单位为毫升 $(\mathrm{mL})$ ；

10 ——换算系数；

100 ——样液的定容体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留至小数点后2位。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不应超过算术平均值的 $10\%$ 。

# 8 其他

固态试样的方法检出限为 $0.6\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $2\mathrm{mg} / 100\mathrm{g}$ 。液态试样的方法检出限为 $0.12\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.4\mathrm{mg} / 100\mathrm{g}$ 。

# 第二法 柱前衍生-高效液相色谱法

# 9 原理

试样中的左旋肉碱经稀盐酸提取、碱皂化、乙腈沉淀蛋白后，用阳离子交换固相萃取柱净化。净化液中的左旋肉碱在氯甲酸丁酯、三乙胺的催化下与L-丙酰胺-β-萘胺发生衍生反应，衍生物用高效液相

色谱仪测定，外标法定量。

10.2.4 氢氧化钾溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $56\mathrm{g}$ 氢氧化钾，用水溶解并稀释至 $1000~\mathrm{mL}$ 。

10.2.5 $4\%$ 氨水甲醇溶液：吸取 $4\mathrm{mL}$ 氨水，加甲醇稀释至 $100~\mathrm{mL}$

# 12 分析步骤

# 12.1 试样制备

液态试样摇匀；基质均匀的半固态试样和粉状试样直接用于提取；其他试样需匀浆或粉碎均匀。

# 12.2 试样提取

12.2.1 固态试样：称取 $5.0\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）试样置于 $150~\mathrm{mL}$ 锥形瓶中，加入 $20~\mathrm{mL}$ 盐酸溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ，涡旋混匀 $1\mathrm{min}$ ，超声 $5\mathrm{min}$ 溶解试样，加入 $5\mathrm{mL}$ 氢氧化钾溶液 $(1\mathrm{mol} / \mathrm{L})$ 涡旋混匀，于 $60^{\circ}\mathrm{C}$ 水浴皂化 $30\mathrm{min}$ ，冷却至室温，加入 $5\mathrm{mL}$ 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ，涡旋混匀 $1\mathrm{min}$ ，转移至 $50~\mathrm{mL}$ 容量瓶中，用盐酸溶液 $(0.1\mathrm{mol} / \mathrm{L})$ 定容至刻度，摇匀。吸取 $1.00~\mathrm{mL}$ 提取液，加入 $9.00~\mathrm{mL}$ 乙腈，涡旋混匀 $1\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，所得上清液待净化。

注：左旋肉碱含量大于 $1000\mathrm{mg / kg}$ 的试样，试样应先用水溶解并稀释至左旋肉碱含量小于或等于 $1000\mathrm{mg / kg}$ 后，按照12.2.1中提取方法进行提取。

12.2.2 液态试样：称取 $25.0\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 试样置于 $150\mathrm{mL}$ 锥形瓶中，加入 $5\mathrm{mL}$ 氢氧化钾溶液 $(1\mathrm{mol} / \mathrm{L})$ 涡旋混匀，于 $60^{\circ}\mathrm{C}$ 水浴皂化 $30\mathrm{min}$ ，冷却至室温，加入 $5\mathrm{mL}$ 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ，涡旋混匀 $1\mathrm{min}$ ，转移至 $50\mathrm{mL}$ 容量瓶中，用盐酸溶液 $(0.1\mathrm{mol} / \mathrm{L})$ 定容至刻度，摇匀。吸取 $1.00\mathrm{mL}$ 提取液，加入 $9.00\mathrm{mL}$ 乙腈，涡旋混匀 $1\mathrm{min}$ ， $4000\mathrm{r} / \mathrm{min}$ 离心 $5\mathrm{min}$ ，所得上清液待净化。

注：左旋肉碱含量大于 $200\mathrm{mg / kg}$ 的试样，试样应先用水稀释至左旋肉碱含量小于或等于 $200\mathrm{mg / kg}$ 后，按照12.2.2中提取方法进行提取。

# 12.3 净化

吸取 $1.00\mathrm{mL}$ 待净化溶液于活化好的固相萃取柱中，以1滴/s的速度过柱，用 $3\mathrm{mL}$ 盐酸溶液 $(10\mathrm{mmol / L})$ 淋洗，弃去淋洗液，真空抽 $5\mathrm{min}$ 尽量除去柱中水分，用 $3\mathrm{mL}4\%$ 氨水甲醇溶液洗脱，收集洗脱液，于 $45^{\circ}\mathrm{C}$ 氮气浓缩至干。所得残渣待衍生。

# 12.4 衍生

在残渣中加入 $0.5\mathrm{mL}$ 衍生溶液，涡旋 $1\mathrm{min}$ ，超声 $30\mathrm{s}$ 溶解残渣，再依次加入 $0.5\mathrm{mL}$ 催化剂I和 $0.5\mathrm{mL}$ 催化剂Ⅱ，剧烈涡旋 $3\mathrm{min}$ ，室温下放置反应 $10\mathrm{min}$ 。反应完成后，加入 $2.00\mathrm{mL}$ 碳酸氢钠溶液 $(50\mathrm{mmol / L})$ ，剧烈涡旋混匀 $3\mathrm{min}$ 终止反应， $4000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，取上层水相用 $0.45\mu \mathrm{m}$ 滤膜过滤，制得试样待测液。

# 12.5 液相色谱参考条件

液相色谱参考条件如下。

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱 $(3.0\mathrm{mm}\times 150\mathrm{mm},2.7\mu \mathrm{m})$ 或性能相当者。  
b）流动相A：乙腈-甲醇 $(1 + 1)$ ，流动相B：磷酸盐缓冲溶液；洗脱比例为 $\mathrm{A}:\mathrm{B}(35:65$ ，体积比）。  
c) 流速: $0.6 \mathrm{~mL} / \mathrm{min}$ 。  
d) 柱温: $35^{\circ} \mathrm{C}$ 。  
e) 检测波长: $244 \mathrm{~nm}$ 。  
f）进样体积： $10~\mu \mathrm{L}$

# 12.6 标准曲线的制作

吸取 $1.00\mathrm{mL}$ 左旋肉碱系列标准工作液，于 $45^{\circ}\mathrm{C}$ 氮气浓缩至干后按12.4衍生化，分别注入高效液相色谱仪中，测定相应的峰面积，以系列标准工作液的浓度为横坐标，以左旋肉碱衍生物的峰面积为纵坐标，绘制标准曲线。标准溶液的色谱图参见附录A中图A.1。

# 12.7 试样溶液的测定

将试样待测液注入高效液相色谱仪中，以保留时间定性，测得待测物峰面积，根据标准曲线得到待测液中左旋肉碱的浓度。

# 13 分析结果的表述

试样中左旋肉碱的含量按式(2)计算。

$$
X = \frac {c \times V \times 1 0}{m \times 1 0 0 0} \times 1 0 0 \quad \dots \tag {2}
$$

式中：

$X$ ——试样中左旋肉碱的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$c$ ——由标准曲线得到的试样溶液中左旋肉碱的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

10 —稀释倍数；

$m$ ——试样的取样质量，单位为克(g)；

1000 ——换算系数；

100 ——换算系数。

计算结果保留至小数点后2位。

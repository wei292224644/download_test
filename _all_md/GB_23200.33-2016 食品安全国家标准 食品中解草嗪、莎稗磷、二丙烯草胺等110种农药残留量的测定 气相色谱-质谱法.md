# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2149—2008《进出口食品中解草嗪、莎稗磷、二丙烯草胺等110种农药残留量的检测方法气相色谱一质谱法》。与SN/T2149—2008相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“进出口食品”改为“食品”；  
——标准范围中增加“其他食品可参照执行”。

# 食品安全国家标准

# 食品中解草嗪、莎稗磷、二丙烯草胺等110种农药残留量的测定气相色谱一质谱法

# 1 范围

本标准规定了食品中110种农药(参见附录A)残留量检测的气相色谱一质谱法。

本标准适用于大米、糙米、大麦、小麦、玉米中110种农药残留量的测定，其他食品可参照执行。

# 3 原理

试样加水浸泡后用丙酮振荡提取，然后依次通过液一液分配、凝胶渗透色谱和固相萃取对提取液进行净化，用气相色谱一质谱仪检测，标准曲线外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 丙酮 $\mathrm{(CH_3COCH_3)}$ ：色谱纯。  
4.1.2 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ 。  
4.1.3 环己烷 $(\mathrm{C_6H_{12}})$ 。  
4.1.4 乙酸乙酯 $\mathrm{(CH_3COOC_2H_5)}$   
4.1.5 乙腈 $(\mathrm{C_2H_3N})$ 。  
4.1.6 甲苯 $(\mathrm{C}_7\mathrm{H}_8)$ 。  
4.1.7 氯化钠 $(\mathrm{NaCl})$ 。  
4.1.8 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：使用前 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，储于干燥器中，冷却后备用。

# 4.2 溶液配制

4.2.1 氯化钠溶液 $(15\% ,W / V)$ ：称取 $15\mathrm{g}$ 氯化钠，溶于 $100~\mathrm{mL}$ 水中，摇匀备用。  
4.2.2 环己烷—乙酸乙酯(1+1,体积比)：取 $100\mathrm{mL}$ 环己烷，加入 $100\mathrm{mL}$ 乙酸乙酯，摇匀备用。  
4.2.3 乙腈—甲苯(3+1,体积比)：取 $300~\mathrm{mL}$ 乙腈，加入 $100~\mathrm{mL}$ 甲苯，摇匀备用。

# 4.3 标准品

4.3.1 农药标准物质：标准物质清单参见附录A，标准物质纯度 $\geq 95\%$

# 4.4 农药标准溶液

4.4.1 标准储备溶液：准确称取适量标准品（精确至 $0.1\mathrm{mg}$ ），用丙酮溶解，配制成浓度为 $500~\mu \mathrm{g / mL}$ 的标准储备溶液， $-18^{\circ}\mathrm{C}$ 冷冻避光保存，有效期3个月。  
4.4.2 混合标准中间溶液: 将 110 种农药分成 5 个组, 分组情况参见附录 A。移取一定体积的标准储备溶液用丙酮配制成 $10 \mathrm{mg} / \mathrm{L}$ 的混合标准溶液, $4^{\circ} \mathrm{C}$ 冷藏避光保存, 有效期 1 个月。  
4.4.3 基质混合标准工作溶液：分别移取一定体积的混合标准溶液，加入经过 $7.1 \sim 7.2$ 步骤净化后的空白样品基质溶液，混匀，用丙酮定容至 $1.0 \mathrm{~mL}$ 。基质混合标准工作溶液应现用现配。

# 4.5 材料

4.5.1 助滤剂：celite 545，或相当者。  
4.5.2 以石墨化非多孔碳/酰胺丙基甲硅烷基化硅胶为填料的固相萃取柱：Envi-Carb/LC-NH $_2$ ， $500 \mathrm{mg} / 500 \mathrm{mg}, 6 \mathrm{~mL}$ ，或相当者。

# 5 仪器和设备

5.1 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.2 固相萃取装置。  
5.3 样品粉碎机：配20目样品筛。  
5.4 振荡器。  
5.5 抽滤装置。  
5.6 旋转蒸发仪。  
5.7 氮吹仪。  
5.8 涡旋振荡器。  
5.9 气相色谱—质谱仪：配有电子轰击电离源(EI)。  
5.10 凝胶渗透色谱仪：配凝胶色谱柱[Bio-beads S-X3,300 mm×25 mm(内径)]。

# 6 试样制备与保存

# 6.1 试样制备

取代表性试样 $500\mathrm{g}$ ，取样部位按GB2763的规定执行，用粉碎机粉碎并使其全部通过20目的样品筛。混合均匀，装入洁净的容器内，密封并标识。

# 6.2 试样保存

试样于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 避光保存。取样、制样及保存过程中应防止试样受到污染或者残留农药含量发生变化。

# 7 分析步骤

# 7.1 提取

称取试样 $20\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 置于锥形瓶中, 加入 $20\mathrm{mL}$ 水放置 $30\mathrm{min}$ 。加入 $80\mathrm{mL}$ 丙酮, 置于振荡器上振荡提取 $30\mathrm{min}$ 。向抽滤装置内加入适量助滤剂, 将试样及提取液转移至抽滤装置上, 减压抽滤, 再分3次, 每次 $5\mathrm{mL}$ 丙酮洗涤锥形瓶及试样残渣, 合并提取液和洗涤液, 于 $40^{\circ}\mathrm{C}$ 浓缩至 $20\mathrm{mL}$ 。将上述溶液转移至分液漏斗中, 依次向分液漏斗中加入 $50\mathrm{mL}15\% (\mathrm{W} / \mathrm{V})$ 氯化钠水溶液, $50\mathrm{mL}$ 二氯甲烷, 振荡 $5\mathrm{min}$ , 静置后收集二氯甲烷层并过无水硫酸钠脱水。再加入 $50\mathrm{mL}$ 二氯甲烷重复上述操作。

合并经无水硫酸钠脱水的二氯甲烷，于 $40^{\circ}\mathrm{C}$ 水浴旋转浓缩至近干，再用氮气吹干。用环己烷一乙酸乙酯定容至 $4.0\mathrm{mL}$ ，待净化。

# 7.2 净化

# 7.2.1 凝胶渗透色谱净化

将7.1中待净化溶液置于凝胶色谱仪上，进样体积为 $2.0\mathrm{mL}$ （相当于一半称样量的样品），环己烷一乙酸乙酯作为流动相，流速为 $3.0\mathrm{mL / min}$ ，弃去第 $0\mathrm{mL}\sim 20\mathrm{mL}$ 淋洗液，收集第 $21\mathrm{mL}\sim 70\mathrm{mL}$ 淋洗液，于 $40^{\circ}\mathrm{C}$ 水浴旋转浓缩至近干，再用氮气吹干。用 $2\mathrm{mL}$ 乙腈一甲苯溶液溶解残渣。

# 7.2.2 固相萃取净化

将Envi-Carb/LC-NH $_2$ 固相萃取柱置于固相萃取装置上。用 $10 \mathrm{~mL}$ 乙腈—甲苯溶液预淋洗。将经7.2.1净化的溶液移入固相萃取柱时即开始收集淋洗液，再分3次，每次 $1 \mathrm{~mL}$ 乙腈—甲苯溶液洗涤容器并移入固相萃取柱，并调节装置使淋洗液流速为 $2 \mathrm{~mL} / \mathrm{min}$ 。再用 $20 \mathrm{~mL}$ 乙腈—甲苯溶液洗涤固相萃取柱，合并淋洗液。于 $40^{\circ} \mathrm{C}$ 水浴旋转浓缩至近干，再用氮气吹干。用丙酮定容至 $1.0 \mathrm{~mL}$ ，供气相色谱—质谱测定。

# 7.3 测定

# 7.3.1 气相色谱一质谱参考条件

a）色谱柱：DB-5MS $(30\mathrm{m}\times 0.25\mathrm{mm}\times 0.25\mu \mathrm{m})$ 石英毛细管柱或相当者；  
b）色谱柱温度： $60^{\circ}\mathrm{C}$ 保持 $2\mathrm{min}$ ，先以 $25^{\circ}\mathrm{C / min}$ 升温至 $130^{\circ}\mathrm{C}$ ，然后以 $4^{\circ}\mathrm{C / min}$ 升温至 $180^{\circ}\mathrm{C}$ ，再以 $6^{\circ}\mathrm{C / min}$ 升温至 $300^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$   
c) 载气：氦气，纯度 $\geqslant 99.999\%$ ，流速 $1.0~\mathrm{mL / min}$   
d）进样口温度： $250^{\circ}\mathrm{C}$   
e) 进样量: $2 \mu \mathrm{L}$ ;   
f）进样方式：脉冲不分流进样，脉冲压力20psi，保持1.5min；  
g） 电子轰击电离源： $70\mathrm{eV}$   
h）离子源温度： $230^{\circ}\mathrm{C}$   
i）四极杆温度： $150^{\circ}\mathrm{C}$   
j）GC-MS接口温度： $280^{\circ}\mathrm{C}$   
k）溶剂延迟时间：7min；  
(1) 选择离子监测：每种农药选择1个定量离子、1个～3个定性离子，依据农药的保留时间分组检测。农药的保留时间、定量离子、定性离子参见附录B。每组检测离子的分组时间和驻留时间参见附录C。

# 7.3.2 定性测定

进行样品测定时，如果样品质量色谱图中出现与标准物质保留时间一致的色谱峰，且在扣除背景后，该色谱峰对应的质谱图中出现所选择的离子（离子丰度比与标准物质离子丰度比的偏差满足残留分析要求），则可以判断样品中存在这种农药。

# 7.3.3 定量测定

为减少基质对定量结果的影响，本方法采用空白样品溶液配置基质混合标准工作溶液，外标法定量测定。样品中农药响应值应在基质混合标准工作溶液线性范围内。本方法标准物质在各类粮谷基质中选择离子监测GC-MS质量色谱图参见附录D。

# 7.4 平行试验

按照 $7.1\sim 7.3$ 的规定对同一试样进行平行试验。

# 7.5 空白试验

除不加试样外，均按 $7.1\sim 7.4$ 的规定执行。

# 8 结果计算和表述

试样中农药残留量可由GC—MSD数据处理软件计算，或按式(1)计算。

$$
X _ {i} = \frac {A \times c \times V}{A _ {s} \times m \times 1 0 0 0} \dots \tag {1}
$$

式中：

$X_{i}$ ——试样中农药残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

A ——样液中农药定量离子色谱峰的峰面积；

$c$ 基质标准溶液中农药的浓度，单位为毫克每升 $(\mathrm{mg / L})$

V ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ 基质标准溶液中农药定量离子色谱峰的峰面积；

m 最终溶液相当的试样质量，单位为克(g)；

1000 ——换算系数。

计算结果需将空白值扣除，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 F 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 G 的要求。

# 10 定量限和回收率

# 10.1 定量限

采用本方法对大米、糙米、大麦、小麦和玉米 5 种食品中 110 种农药残留进行测定, 各种农药的测定低限均为 $0.01 \mathrm{mg} / \mathrm{kg}$ 。

# 10.2 回收率

采用本方法对大米、糙米、大麦、小麦和玉米 5 种食品进行添加回收实验, 添加水平为 $0.01 \mathrm{mg} / \mathrm{kg}$ 、 $0.05 \mathrm{mg} / \mathrm{kg} 、 0.1 \mathrm{mg} / \mathrm{kg}, 110$ 种农药在 5 种食品中的添加回收率参见附录 E。

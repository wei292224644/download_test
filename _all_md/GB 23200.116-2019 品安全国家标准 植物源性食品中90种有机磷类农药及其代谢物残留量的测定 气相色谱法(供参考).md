# 食品安全国家标准

# 植物源性食品中90种有机磷类农药及其代谢物残留量的测定气相色谱法

# 方法一 气相色谱双柱法

# 1 范围

本标准规定了植物源性食品中90种有机磷类农药及其代谢物(参见附录A)残留量的气相色谱测定方法。

本标准适用于植物源性食品中90种有机磷类农药及其代谢物残留量的测定。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的应用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682分析实验室用水规格和试验方法

# 3 原理

试样用乙腈提取，提取液经固相萃取或分散固相萃取净化，使用带火焰光度检测器的气相色谱仪检测，根据双柱色谱峰的保留时间定性，外标法定量。

# 4 试剂与材料

除非另有说明，在分析中仅使用分析纯的试剂，水为GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN,CAS}$ 号： $75 - 05 - 8)$   
4.1.2 丙酮 $\mathrm{(C_3H_6O}$ ，CAS号： $67 - 64 - 1$ ：色谱纯。  
4.1.3 甲苯 $\mathrm{(C_7H_8}$ ，CAS号： $108 - 88 - 3$ ）：色谱纯。  
4.1.4 无水硫酸镁 $(\mathrm{MgSO}_4$ ，CAS号： $7487 - 88 - 9)$   
4.1.5 氯化钠(NaCl，CAS号：7647—14—5)。  
4.1.6 乙酸钠 $\mathrm{(CH_3COONa,CAS}$ 号： $127 - 09 - 3)$

# 4.2 溶液配制

乙腈一甲苯溶液 $(3 + 1$ ，体积比)：量取 $100~\mathrm{mL}$ 甲苯加入 $300~\mathrm{mL}$ 乙腈中，混匀。

# 4.3 标准品

90种有机磷类农药及其代谢物标准品：参见附录A，纯度 $\geq 96\%$

# 4.4 标准溶液配制

4.4.1 标准储备溶液 $(1000\mathrm{mg / L})$ ：准确称取 $10\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )有机磷类农药及其代谢物各标准品，用丙酮溶解并分别定容到 $10~\mathrm{mL}$ 。标准储备溶液避光且低于一 $18^{\circ}C$ 保存，有效期一年。  
4.4.2 混合标准溶液（I、II、III、IV、V和VI）：详见附录A，将90种有机磷类农药及其代谢物分成6个组，分别准确吸取一定量的单个农药储备溶液于 $50~\mathrm{mL}$ 容量瓶中，用丙酮定容至刻度。混合标准溶液，避光 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存，有效期一个月。

# 4.5 材料

4.5.1固相萃取柱：石墨化炭黑填料(GCB) $500\mathrm{mg}$ /氨基填料 $(\mathrm{NH}_2)500\mathrm{mg},6\mathrm{mL}$   
4.5.2 乙二胺一N一丙基硅烷硅胶（PSA）： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}$   
4.5.3 十八烷基甲硅烷改性硅胶 $\left(\mathrm{C}_{18}\right): 40 \mu \mathrm{m} \sim 60 \mu \mathrm{m}$ 。  
4.5.4 陶瓷均质子： $2\mathrm{cm}$ （长） $\times 1\mathrm{cm}$ （外径）。  
4.5.5 微孔滤膜（有机相）： $0.22\mu \mathrm{m}\times 25\mathrm{mm}$

# 5 仪器和设备

5.1 气相色谱仪：配有双火焰光度检测器(FPD磷滤光片)。  
5.2 分析天平：感量 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$   
5.3 高速匀浆机：转速不低于 $15000\mathrm{r / min}$   
5.4 离心机：转速不低于 $4200\mathrm{r / min}$   
5.5 组织捣碎机。  
5.6 旋转蒸发仪。  
5.7 氮吹仪，可控温。  
5.8 涡旋振荡器。

# 6 试样制备

# 6.1 试样制备

蔬菜和水果的取样量按照相关标准规定执行，食用菌样品随机取样 $1\mathrm{kg}$ 。样品取样部位按GB2763的规定执行。对于个体较小的样品，取样后全部处理；对于个体较大的基本均匀样品，可在对称轴或对称面上分割或切成小块后处理；对于细长、扁平或组分含量在各部分有差异的样品，可在不同部位切取小片或截成小段后处理；取后的样品将其切碎，充分混匀，用四分法取样或直接放入组织捣碎机中捣碎成匀浆，放入聚乙烯瓶中。

取谷类样品 $500\mathrm{g}$ ，粉碎后使其全部可通过 $425\mu \mathrm{m}$ 的标准网筛，放入聚乙烯瓶或袋中。取油料作物、茶叶、坚果和调味料各 $500\mathrm{g}$ ，粉碎后充分混匀，放入聚乙烯瓶或袋中。

植物油类搅拌均匀，放入聚乙烯瓶中。

# 6.2 试样贮存

将试样按照测试和备用分别存放。于 $-20^{\circ}\mathrm{C} \sim -16^{\circ}\mathrm{C}$ 条件下保存。

# 7 分析步骤

# 7.1 提取和净化

# 7.1.1 蔬菜、水果和食用菌

称取 $20\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）试样于 $150~\mathrm{mL}$ 烧杯中，加入 $40~\mathrm{mL}$ 乙腈，用高速匀浆机 $15000\mathrm{r / min}$ 匀浆 $2\mathrm{min}$ ，提取液过滤至装有 $5\mathrm{g}\sim 7\mathrm{g}$ 氯化钠的 $100~\mathrm{mL}$ 具塞量筒中，盖上塞子，剧烈振荡 $1\mathrm{min}$ ，在室温下静置 $30\mathrm{min}$ 。

准确吸取 $10\mathrm{mL}$ 上清液于 $100~\mathrm{mL}$ 烧杯中， $80^{\circ}\mathrm{C}$ 水浴中氮吹蒸发近干，加入 $2\mathrm{mL}$ 丙酮溶解残余物，盖上铝箔，备用。

将上述备用液完全转移至 $15\mathrm{mL}$ 刻度离心管中，再用约 $3\mathrm{mL}$ 丙酮分3次冲洗烧杯，并转移至离心管，最后定容至 $5.0\mathrm{mL}$ ，涡旋 $0.5\mathrm{min}$ ，用微孔滤膜(4.5.5)过滤，待测。

# 7.1.2 油料作物和坚果

称取 $10\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）试样于 $150~\mathrm{mL}$ 烧杯中，加入 $20~\mathrm{mL}$ 水，混匀后，静置 $30\mathrm{min}$ ，再加入 $50~\mathrm{mL}$ 乙腈，用高速匀浆机 $15000\mathrm{r / min}$ 匀浆 $2\mathrm{min}$ ，提取液过滤至装有 $5\mathrm{g}\sim 7\mathrm{g}$ 氯化钠的 $100~\mathrm{mL}$ 具塞量筒中，盖上塞子，剧烈振荡 $1\mathrm{min}$ ，在室温下静置 $30\mathrm{min}$ 。

准确吸取 $8\mathrm{mL}$ 上清液于 $15~\mathrm{mL}$ 刻度离心管中，加入 $900~\mathrm{mg}$ 无水硫酸镁、 $150~\mathrm{mg}$ PSA、 $150~\mathrm{mg}$ $\mathrm{C_{18}}$ ，涡旋 $0.5\mathrm{min},4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，准确吸取 $5\mathrm{mL}$ 上清液加入到 $10~\mathrm{mL}$ 刻度离心管中， $80^{\circ}C$ 水浴中氮吹蒸发近干，准确加入 $1.00~\mathrm{mL}$ 丙酮，涡旋 $0.5\mathrm{min}$ ，用微孔滤膜(4.5.5)过滤，待测。

# 7.1.3 谷物

称取 $10\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）试样于 $150~\mathrm{mL}$ 具塞锥形瓶中，加入 $20~\mathrm{mL}$ 水浸润 $30\mathrm{min}$ ，加入 $50~\mathrm{mL}$ 乙腈，在振荡器上以转度为 $200~\mathrm{r / min}$ 振荡 $30\mathrm{min}$ ，提取液过滤至装有 $5\mathrm{g} \sim 7\mathrm{g}$ 氯化钠的 $100~\mathrm{mL}$ 具塞量筒中，盖上塞子，剧烈振荡 $1\mathrm{min}$ ，在室温下静置 $30\mathrm{min}$ 。

准确吸取 $10\mathrm{mL}$ 上清液于 $100~\mathrm{mL}$ 烧杯中， $80^{\circ}\mathrm{C}$ 水浴中氮吹蒸发近干，加入 $2\mathrm{mL}$ 丙酮溶解残余物，盖上铝箔，备用。

将上述溶液完全转移至 $10.0\mathrm{mL}$ 刻度试管中，再用 $5\mathrm{mL}$ 丙酮分3次冲洗烧杯，收集淋洗液于刻度试管中， $50^{\circ}\mathrm{C}$ 水浴氮吹蒸发近干，准确加入 $2.00\mathrm{mL}$ 丙酮，涡旋 $0.5\mathrm{min}$ ，用微孔滤膜(4.5.5)过滤，待测。

# 7.1.4 茶叶和调味料

称取 $5\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 试样于 $150~\mathrm{mL}$ 烧杯中，加入 $20~\mathrm{mL}$ 水浸润 $30\mathrm{min}$ ，加入 $50~\mathrm{mL}$ 乙腈，用高速匀浆机 $15000\mathrm{r / min}$ 高速匀浆 $2\mathrm{min}$ ，提取液过滤至装有 $5\mathrm{g} \sim 7\mathrm{g}$ 氯化钠的 $100~\mathrm{mL}$ 具塞量筒中，盖上塞子，剧烈振荡 $1\mathrm{min}$ ，在室温下静置 $30\mathrm{min}$ 。

准确吸取 $10\mathrm{mL}$ 上清液于 $100~\mathrm{mL}$ 烧杯中， $80^{\circ}\mathrm{C}$ 水浴中氮吹蒸发近干，加入 $2\mathrm{mL}$ 乙腈一甲苯溶液（3+1，体积比）溶解残余物，待净化。

将固相萃取柱(4.5.1)用 $5\mathrm{mL}$ 乙腈一甲苯溶液(4.2)预淋洗。当液面到达柱筛板顶部时，立即加入上述待净化溶液，用 $100~\mathrm{mL}$ 茄型瓶收集洗脱液，用 $2\mathrm{mL}$ 乙腈一甲苯溶液(4.2)涮洗烧杯后过柱，并重复一次。再用 $15\mathrm{mL}$ 乙腈一甲苯溶液(4.2)洗脱柱子，收集的洗脱液于 $40^{\circ}\mathrm{C}$ 水浴中旋转蒸发近干，用 $5\mathrm{mL}$ 丙酮冲洗茄型瓶并转移到 $10\mathrm{mL}$ 离心管中， $50^{\circ}\mathrm{C}$ 水浴中氮吹蒸发近干，准确加入 $1.00\mathrm{mL}$ 丙酮，涡旋混匀，用微孔滤膜(4.5.5)过滤，待测。

# 7.1.5 植物油

称取 $3\mathrm{g}$ （精确至 $0.01\mathrm{g}$ 试样于 $50\mathrm{mL}$ 塑料离心管中，加入 $5\mathrm{mL}$ 水、 $15\mathrm{mL}$ 乙腈，并加入 $6\mathrm{g}$ 无水硫酸镁、 $1.5\mathrm{g}$ 醋酸钠及1颗陶瓷均质子，剧烈振荡 $1\mathrm{min},4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。

准确吸取 $8\mathrm{mL}$ 上清液到内有 $900~\mathrm{mg}$ 无水硫酸镁、 $150~\mathrm{mg}$ PSA、 $150~\mathrm{mg}$ $\mathrm{C}_{18}$ 的 $15~\mathrm{mL}$ 离心管中，涡旋 $0.5\mathrm{min},4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，准确吸取 $5\mathrm{mL}$ 上清液放入 $10~\mathrm{mL}$ 刻度离心管中， $80^{\circ}C$ 水浴中氮吹蒸发近干，准确加入 $1.00~\mathrm{mL}$ 丙酮，涡旋 $0.5\mathrm{min}$ ，用微孔滤膜(4.5.5)过滤，待测。

# 7.2 测定

# 7.2.1 仪器参考条件

a)色谱柱：  
A柱： $50\%$ 聚苯基甲基硅氧烷石英毛细管柱， $30\mathrm{m}\times 0.53\mathrm{mm}$ （内径） $\times 1.0\mu \mathrm{m}$ ，或相当者；  
B柱： $100\%$ 聚苯基甲基硅氧烷石英毛细管柱， $30\mathrm{m}\times 0.53\mathrm{mm}$ （内径） $\times 1.5\mu \mathrm{m}$ ，或相当者；  
b)色谱柱温度： $150~\mathrm{^\circ C}$ 保持 $2\mathrm{min}$ ，然后以 $8\%$ /min程序升温至 $210^{\circ}C$ ，再以 $5\%$ /min升温至250 $^\circ \mathrm{C}$ ，保持 $15\mathrm{min}$   
c)载气：氮气，纯度 $\geqslant 99.999\%$ ，流速为 $8.4~\mathrm{mL / min}$   
d)进样口温度： $250^{\circ}C$   
e)检测器温度： $300^{\circ}\mathrm{C}$   
f)进样量： $1\mu \mathrm{L}$   
g)进样方式：不分流进样；  
h)燃气：氢气，纯度 $\geqslant 99.999\%$ ，流速为 $80~\mathrm{mL / min}$

助燃气：空气，流速为 $110~\mathrm{mL / min}$

# 7.2.2 标准曲线

将混合标准中间溶液用丙酮稀释成质量浓度为 $0.005\mathrm{mg / L},0.01\mathrm{mg / L},0.05\mathrm{mg / L},0.1\mathrm{mg / L}$ 和1

$\mathrm{mg / L}$ 的系列标准溶液, 参考色谱条件测定。以农药质量浓度为横坐标、色谱的峰面积积分值为纵坐标, 绘制标准曲线。

# 7.2.3 定性及定量

# 7.2.3.1 定性测定

以目标农药的保留时间定性。被测试样中目标农药双柱上色谱峰的保留时间与相应标准色谱峰的保留时间相比较，相差应在 $\pm 0.05\mathrm{min}$ 之内。

# 7.2.3.2 定量测定

以外标法定量。

# 7.3 试样溶液的测定

将混合标准工作溶液和试样溶液依次注入气相色谱仪中，保留时间定性，测得目标农药色谱峰面积，根据式(1)，得到各农药组分含量。待测样液中农药的响应值应在仪器检测的定量测定线性范围之内，超过线性范围，时应根据测定浓度进行适当倍数稀释后再进行分析。

# 7.4 平行试验

按 $7.1\sim 7.3$ 的规定对同一试样进行平行试验测定。

# 7.5 空白试验

除不加试料外，按 $7.1 \sim 7.4$ 的规定进行平行操作。

# 8 结果计算

试样中被测农药残留量以质量分数 $\omega$ 计，单位以毫克每千克 $(\mathrm{mg / kg})$ 表示，按式(1)计算。

$$
\omega = \frac {V _ {1} \times A \times V _ {3}}{V _ {2} \times A _ {S} \times m} \times \rho \quad \dots \tag {1}
$$

式中：

$\omega$ ——样品中被测组分含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\mathrm{V}_{1}$ ——提取溶剂总体积，单位为毫升 $(\mathrm{mL})$ ；

$\mathrm{V}_{2}$ ——提取液分取体积，单位为毫升 $(\mathrm{mL})$

$\mathrm{V}_{3}$ ——待测溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

A——待测溶液中被测组分峰面积；

$\mathrm{A_S}$ ——标准溶液中被测组分峰面积；

m——试样质量，单位为克(g)；

$\rho$ ——标准溶液中被测组分质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ 。

计算结果应扣除空白值，计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，保留2位有效数字。当结果超过 $1\mathrm{mg / kg}$ 时，保留3位有效数字。

# 9 精密度

在重复性条件下，获得的2次独立测试结果的绝对差值不得超过重复性限 $(r)$ ，参见附录B。

在再现性条件下，获得的2次独立测试结果的绝对差值不得超过再现性限 $(R)$ ，参见附录B。

# 1 范围

本文件规定了禽蛋中头孢噻呋代谢物去呋喃甲酰基头孢噻呋残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鸡蛋、鸭蛋、鹌鹑蛋中去呋喃甲酰基头孢噻呋残留量的测定，其他禽蛋的检测可参照执行。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中残留的头孢噻呋及代谢物，加入 $0.4\%$ 二硫赤藓醇溶液混匀，用 $14\%$ 碘乙酰胺溶液衍生化，生成稳定的乙酰胺衍生物，水饱和正己烷除脂，固相萃取柱净化浓缩，液相色谱-串联质谱测定，内标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水

# 5.1 试剂

5.1.1 乙腈（CHCN）：色谱纯

5.1.2 甲酸（HCOOH）：色谱纯

5.1.3 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.4 二硫赤藓醇 $(\mathrm{C_4H_5(O_2S_2)}$

5.1.5 碘乙酰胺 $\mathrm{(C_2H_4INO)}$

5.1.6 硼酸 $(\mathrm{H}_3\mathrm{BO}_3)$ 。

5.1.7 氯化钾(KCl)。

5.1.8 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$

5.1.9 正己烷 $(\mathrm{C_6H_{14}})$

5.1.10 氢氧化钾(KOH)。

# 5.2 溶液配制

5.2.1 $50\%$ 乙腈水溶液：取乙腈 $50~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ 。  
5.2.2 $90\%$ 乙腈水溶液：取乙腈 $90~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$   
5.2.3 $0.1\%$ 甲酸乙腈溶液：取甲酸 $1\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$   
5.2.4 $0.1\%$ 甲酸水溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$   
5.2.5 $45\%$ 氢氧化钾：称取 $45\mathrm{g}$ 氢氧化钾，加水溶解并稀释至 $100~\mathrm{mL}$

5.2.6 $0.05\mathrm{mol} / \mathrm{L}$ 硼酸缓冲液：称取 $19\mathrm{g}$ 硼酸和 $3.7\mathrm{g}$ 氯化钾，加水溶解并稀释至 $1000\mathrm{mL}$ ，用 $45\%$ 氢氧化钾调 $\mathrm{pH}$ 至 $9 \pm 0.05$ 。  
5.2.7 0.025 mol/L 磷酸缓冲液：称取 $3.4\mathrm{g}$ 磷酸二氢钾，加水溶解并稀释至 $1000\mathrm{mL}$ ，用 $45\%$ 氢氧化钾调 $\mathsf{pH}$ 至 $7 \pm 0.05$ 。  
5.2.8 $0.4\%$ 二硫赤藓醇溶液：称取 $1\mathrm{g}$ 二硫赤藓醇溶于 $250~\mathrm{mL}$ 硼酸缓冲液 $(0.05\mathrm{mol} / \mathrm{L})$ 中，临用现配。  
5.2.9 $14\%$ 碘乙酰胺溶液：称取 $7\mathrm{g}$ 碘乙酰胺溶于 $50~\mathrm{mL}$ 磷酸缓冲液 $(0.025\mathrm{mol} / \mathrm{L})$ 中，临用现配。  
5.2.10 水饱和正己烷：取水 $100~\mathrm{mL}$ ，加正己烷 $100~\mathrm{mL}$ ，摇匀，静置分层，取上层液。

# 5.3 标准品

5.3.1 去呋喃甲酰基头孢噻呋(Desfuroyl ceftiofur, $\mathrm{C_{14}H_{15}N_5O_5S_3}$ , CAS号: 120882-22-6), 含量 $\geqslant 95\%$ 。  
5.3.2 去呋喃甲酰基头孢噻呋- $\mathbf{D}_3$ (Desfuroyl ceftiofur- $\mathbf{D}_3$ , $\mathrm{C}_{14}\mathrm{H}_{12}\mathrm{D}_3\mathrm{N}_5\mathrm{O}_5\mathrm{S}_3$ , CAS号: 120882-22-6-unlabelled), 含量 $\geqslant 95\%$ 。

# 5.4 标准溶液制备

5.4.1 标准储备液：取去呋喃甲酰基头孢噻呋、去呋喃甲酰基头孢噻呋- $\mathrm{D}_3$ 标准品各适量（相当于有效成分约 $10\mathrm{mg}$ )，精密称定，分别用 $50\%$ 乙腈水溶解并稀释定容于 $100~\mathrm{mL}$ 容量瓶中，配制成浓度为 $100~\mu \mathrm{g / mL}$ 的去呋喃甲酰基头孢噻呋、去呋喃甲酰基头孢噻呋- $\mathrm{D}_3$ 标准储备液， $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1个月。  
5.4.2 标准工作液：分别精密量取 $100\mu \mathrm{g / mL}$ 的去呋喃甲酰基头孢噻呋、去呋喃甲酰基头孢噻呋- $\mathbf{D}_3$ 标准储备液各 $0.1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 容量瓶中，用 $50\%$ 乙腈水稀释至刻度，配制成浓度为 $100\mu \mathrm{g / L}$ 的去呋喃甲酰基头孢噻呋、去呋喃甲酰基头孢噻呋- $\mathbf{D}_3$ 标准工作液，临用现配。

# 5.5 材料

5.5.1固相萃取柱：反相混合型亲水亲脂平衡共聚物固相萃取柱， $200\mathrm{mg / 6mL}$ ，或相当者。  
5.5.2 微孔尼龙滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量分别为 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
6.3 涡旋混合器。  
6.4 冷冻离心机：转速 $10000\mathrm{r / min}$   
6.5 氮吹仪。  
6.6 固相萃取装置。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或冷藏的空白或供试禽蛋，去壳后混合均匀。

a）取均质的供试样品，作为供试试样；  
b）取均质的空白样品，作为空白试样；  
c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取禽蛋试料 $(2\pm 0.05)\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 塑料离心管，添加去呋喃甲酰基头孢噻呋- $\mathbf{D}_3$ 标准工作液 $100~\mu \mathrm{L}$ ，加 $0.4\%$ 二硫赤藓醇溶液 $10~\mathrm{mL}$ ，涡旋混匀 $2\mathrm{min}$ ，室温静置 $15\mathrm{min}$ 。加入 $14\%$ 碘乙酰胺溶液

$3\mathrm{mL}$ ，涡旋混匀 $2\mathrm{min}$ ，室温衍生 $15\mathrm{min}$ 。衍生完毕后，加水饱和正己烷 $10~\mathrm{mL}$ ，充分混匀， $4^{\circ}\mathrm{C}10000\mathrm{r / min}$ 离心 $20\mathrm{min}$ ，取下层清液加入水饱和正己烷 $10~\mathrm{mL}$ 重复萃取1次，下层清液备用。

# 8.2 净化

用 $6\mathrm{mL}$ 甲醇、 $6\mathrm{mL}$ 水活化固相萃取柱，将全部下层清液过柱，用 $6\mathrm{mL}$ 水淋洗，抽干，用 $5\mathrm{mL}90\%$ 乙腈水溶液洗脱，收集全部洗脱液。 $50^{\circ}\mathrm{C}$ 水浴氮吹至近干，用 $0.1\%$ 甲酸水溶液 $1.0\mathrm{mL}$ 溶解残余物，涡旋混匀，过 $0.22\mu \mathrm{m}$ 微孔滤膜，供液相色谱-串联质谱测定。

# 8.3 标准曲线的制备

精密量取去呋喃甲酰基头孢噻呋和去呋喃甲酰基头孢噻呋- $\mathrm{D}_3$ 标准储备液适量，分别置于 $50~\mathrm{mL}$ 塑料离心管中，加 $0.4\%$ 二硫赤藓醇溶液 $10~\mathrm{mL}$ ，涡旋混匀 $2\mathrm{min}$ ，室温静置 $15\mathrm{min}$ 。加入 $14\%$ 碘乙酰胺溶液 $3\mathrm{mL}$ ，涡旋混匀 $2\mathrm{min}$ ，室温衍生 $15\mathrm{min}$ 。衍生完毕后，加 $0.1\%$ 甲酸水溶液定容至 $20~\mathrm{mL}$ 混匀。用 $0.1\%$ 甲酸水溶液稀释配制成去呋喃甲酰基头孢噻呋浓度为 $2\mu \mathrm{g / L},4\mu \mathrm{g / L},10\mu \mathrm{g / L},20\mu \mathrm{g / L},50\mu \mathrm{g / L},$ $100~\mu \mathrm{g / L}$ ，内标浓度为 $10~\mu \mathrm{g / L}$ 的系列标准工作溶液，临用现配，供液相色谱-串联质谱测定。以测试药物和其对应内标的特征离子峰面积之比为纵坐标、标准溶液浓度为横坐标绘制标准曲线，求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在相同测试条件下，试料溶液中去呋喃甲酰基头孢噻呋与其内标去呋喃甲酰基头孢噻呋- $\mathrm{D}_3$ 的保留时间之比与标准溶液中去呋喃甲酰基头孢噻呋与其内标去呋喃甲酰基头孢噻呋- $\mathrm{D}_3$ 的保留时间之比偏差在 $1\%$ 以内；且检测到的相对离子丰度应与浓度相当的校正标准溶液相对离子丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.4.3.2 定量测定

取试料溶液和标准溶液，做单点或多点校准，按内标法定量。标准溶液及试料溶液中目标物的响应值均应在仪器检测的线性范围内。在上述液相色谱-串联质谱条件下，标准溶液特征离子质量色谱图见附录A。

# 8.5 空白试验

取空白试样，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中去呋喃甲酰基头孢噻呋药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A _ {i} \times A ^ {\prime} {} _ {i s} \times C _ {s} \times C _ {i s} \times V}{A _ {i s} \times A _ {s} \times C ^ {\prime} {} _ {i s} \times m} \tag {1}
$$

式中：

$X$ ——试样中去呋喃甲酰基头孢噻呋药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A_{i}$ ——试料溶液中去呋喃甲酰基头孢噻呋药物的峰面积；

$A_{is}^{\prime}$ ——标准溶液中去呋喃甲酰基头孢噻呋药物内标的峰面积；

$C_s$ ——标准溶液中去呋喃甲酰基头孢噻呋药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{is}$ ——试料溶液中去呋喃甲酰基头孢噻呋内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——溶解残余物所用溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{is}$ ——试料溶液中去呋喃甲酰基头孢噻呋药物内标的峰面积；

$A_{s}$ ——标准溶液中去呋喃甲酰基头孢噻呋药物的峰面积；

$C_{is}^{\prime}$ ——标准溶液中去呋喃甲酰基头孢噻呋药物内标浓度的数值，单位为微克每升 $(\mu \mathrm{g / L})$

$m$ —供试试料质量的数值，单位为克 $(\mathbf{g})$

# 10 方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法去呋喃甲酰基头孢噻呋的检出限为 $1\mu \mathrm{g / kg}$ ，定量限为 $2\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法去呋喃甲酰基头孢噻呋在 $2\mu \mathrm{g / kg}\sim 20~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 20\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

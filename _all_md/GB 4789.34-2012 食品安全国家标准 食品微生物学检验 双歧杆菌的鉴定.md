# 前言

本标准代替GB/T4789.34-2008《食品卫生微生物学检验双歧杆菌检验》。

本标准与 GB/T 4789.34-2008 相比，主要变化如下：

——修改了标准的中文名称；

——修改了培养基和试剂；

——删除了果糖-6-磷酸盐磷酸酮酶（F6PPK）测定和双歧杆菌的计数方法。

# 食品安全国家标准

# 食品微生物学检验 双歧杆菌的鉴定

# 1 范围

本标准规定了食品中双歧杆菌（Bifidobacterium）的鉴定方法。

本标准适用于食品中双歧杆菌的鉴定。

# 2 设备和材料

除微生物实验室常规灭菌及培养设备外，其他设备和材料如下：

a）恒温培养箱： $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$   
b）气相色谱仪配FID检测器；  
c）冰箱： $2^{\circ}C\sim 5^{\circ}C$   
d）天平： 感量 $0.1\mathrm{g}$   
e）无菌试管： $18\mathrm{mm}\times 180\mathrm{mm}$ 、 $15\mathrm{mm}\times 100\mathrm{mm}$   
f) 无菌吸管: $1 \mathrm{~mL}$ (具 $0.01 \mathrm{~mL}$ 刻度)、 $10 \mathrm{~mL}$ (具 $0.1 \mathrm{~mL}$ 刻度) 或微量移液器 $(200 \mu \mathrm{L} \sim 1000 \mu \mathrm{L})$ 及配套吸头;  
g）无菌锥形瓶： $500~\mathrm{mL}$ 、 $250~\mathrm{mL}$ 。

# 3 培养基和试剂

3.1 双歧杆菌培养基：见附录A中的A.1。  
3.2 PYG 液体培养基：见附录 A 中的 A.2。  
3.3 甲醇：分析纯。  
3.4 三氯甲烷：分析纯。  
3.5 硫酸：分析纯（体积比）。  
3.6 冰乙酸：分析纯（体积比）。  
3.7 乳酸：分析纯。  
3.8 乙酸标准溶液：吸取分析纯冰乙酸 $5.7 \mathrm{~mL}$ , 移入 $100 \mathrm{~mL}$ 容量瓶中, 加水至刻度, 标定, 标定方法见附录B, 此溶液浓度约为 $1 \mathrm{~mol} / \mathrm{L}$ 。  
3.9 乙酸标准使用液：将经标定的乙酸标准溶液用水稀释至 $0.01\mathrm{mol / L}$ 。  
3.10 乳酸标准溶液：吸取分析纯乳酸 $8.4 \mathrm{~mL}$ ，移入 $100 \mathrm{~mL}$ 容量瓶中，加水至刻度，标定，标定方法见附录B，此溶液浓度约为 $1 \mathrm{~mol} / \mathrm{L}$ 。  
3.11 乳酸标准使用液：将经标定的乳酸标准溶液用水稀释至 $0.01\mathrm{mol / L}$ 。

# 5 操作步骤

# 5.1样品制备

5.1.1 样品的全部制备过程均应遵循无菌操作程序。

5.1.2 以无菌操作称取 $25 \mathrm{~g}$ (mL) 样品, 置于装有 $225 \mathrm{~mL}$ 生理盐水的灭菌锥形瓶内, 制成 1:10 的样品匀液。

# 5.2 稀释及涂布培养步骤

5.2.1 用 $1 \mathrm{~mL}$ 无菌吸管或微量移液器吸取 1:10 样品匀液 $1.0 \mathrm{~mL}$ , 沿管壁缓慢注于装有 $9 \mathrm{~mL}$ 生理盐水的无菌试管中（注意吸管尖端不要触及稀释液）, 振摇试管或换用 1 支无菌吸管反复吹打使其混合均匀, 制成 1:100 的样品匀液。  
5.2.2 另取 $1\mathrm{mL}$ 无菌吸管或微量移液器吸头，按5.2.1操作顺序，做10倍递增样品匀液，每递增稀释一次，即换用1次 $1\mathrm{mL}$ 灭菌吸管或吸头。  
5.2.3 根据待鉴定菌种的活菌数，选择三个连续的适宜稀释度，每个稀释度吸取 $0.1 \mathrm{~mL}$ 稀释液，用 L 棒在双歧杆菌琼脂平板进行表面涂布，每个稀释度作两个平皿。置 $36^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 温箱内培养 $48 \mathrm{~h} \pm 2 \mathrm{~h}$ ，培养后选取单个菌落进行纯培养。

# 5.3 纯培养

挑取3个或以上的菌落接种于双歧杆菌琼脂平板，厌氧， $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 培养 $48\mathrm{h}$ 。

# 5.4 镜检及生化鉴定

# 5.4.1 涂片镜检

双歧杆菌菌体为革兰氏染色阳性，不抗酸、无芽孢，无动力，菌体形态多样，呈短杆状、纤细杆状或球形，可形成各种分支或分叉形态。

# 5.4.2 生化鉴定

过氧化氢酶试验为阴性。选取纯培养平板上的三个单个菌落，分别进行生化反应检测，不同双歧杆菌菌种主要生化反应见附录B。

# 5.5 有机酸代谢产物测定

气相色谱法测定双歧杆菌的有机酸代谢产物，见附录C。

# 6报告

根据镜检及生化鉴定的结果（5.4）、双歧杆菌的有机酸代谢产物乙酸与乳酸微摩尔之比大于1（5.5），报告双歧杆菌属的种名。

# A.1.2制法

# A. 1.2.1 半胱氨酸盐溶液

称取半胱氨酸 $0.5\mathrm{g}$ ，加入 $1.0\mathrm{mL}$ 盐酸，使半胱氨酸全部溶解，配制成半胱氨酸盐溶液。

# A.1.2.2西红柿浸出液

将新鲜的西红柿洗净后称重切碎，加等量的蒸馏水在 $100^{\circ} \mathrm{C}$ 水浴中加热，搅拌 $90 \mathrm{~min}$ ，然后用纱布过滤，校正 $\mathrm{pH} 7.0$ ，将浸出液分装后， $121^{\circ} \mathrm{C}$ 高压灭菌 $15 \mathrm{~min} \sim 20 \mathrm{~min}$ 。

# A.1.2.3培养基

将 A.1.1 所有成分加入蒸馏水中, 加热溶解, 然后加入半胱氨酸盐溶液, 校正 $\mathrm{pH} 6.8 \pm 0.2$ 。分装后 $121^{\circ} \mathrm{C}$ 高压灭菌 $15 \mathrm{~min} \sim 20 \mathrm{~min}$ 。临用时加热熔化琼脂, 冷至 $50^{\circ} \mathrm{C}$ 时使用。

# 附录C

# 气相色谱法测定双歧杆菌的有机酸代谢产物

# C. 1 双歧杆菌培养液制备

挑取双歧杆菌琼脂平板上纯培养的双歧杆菌接种于PYG液体培养基，同时用未接种菌的PYG液体培养基做空白对照，厌氧， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $48\mathrm{h}$ 。

# C. 2 标准液的配制

# C.2.1 乙酸标准溶液

准确吸取乙酸 $5.70 \mathrm{~mL}$ 加水稀释至 $100.0 \mathrm{~mL}$ , 摇匀, 进行标定, 配成约 $1.0 \mathrm{~mol} / \mathrm{L}$ 的乙酸标准溶液。标定方法为: 准确称取乙酸 $3 \mathrm{~g}$ , 加水 $15 \mathrm{~mL}$ , 酚酞指示液 2 滴, 用 $1 \mathrm{~mol} / \mathrm{mL}$ 氢氧化钠溶液滴定, 并将滴定结果用空白试验校正。 $1 \mathrm{~mL} 1 \mathrm{~mol} / \mathrm{mL}$ 氢氧化钠溶液相当于 $60.05 \mathrm{mg}$ 的乙酸。

# C.2.2 乙酸使用液

将经标定的乙酸标准溶液用水稀释至 $20.0\mathrm{mmol / L}$ 。

# C.2.3 乳酸标准溶液

准确吸取含量为 $85\% \sim 90\%$ 的乳酸 $0.84 \mathrm{~mL}$ , 加水稀释至 $100.0 \mathrm{~mL}$ , 摇匀, 配成 $1.0 \mathrm{~mol} / \mathrm{L}$ 的乳酸标准溶液。标定方法为: 准确称取乳酸 $1 \mathrm{~g}$ , 加水 $50 \mathrm{~mL}$ , 加入 $1 \mathrm{~mol} / \mathrm{mL}$ 氢氧化钠滴定液 $25 \mathrm{~mL}$ , 煮沸 $5 \mathrm{~min}$ , 加入酚酞指示液2滴, 同时用 $0.5 \mathrm{~mol} / \mathrm{mL}$ 硫酸滴定液滴定, 并将滴定结果用空白试验校正。 $1 \mathrm{~mL} 1 \mathrm{~mol} / \mathrm{mL}$ 氢氧化钠溶液相当于 $90.08 \mathrm{mg}$ 的乳酸。

# C.2.4 乳酸使用液

将乳酸标准溶液用水稀释至 $20.0\mathrm{mmol / L}$

# C.3 方法

# C.3.1乙酸的处理

取双歧杆菌培养液 $2.0 \mathrm{~mL} \sim 3.0 \mathrm{~mL}$ 放入 $10 \mathrm{~mL}$ 离心管中，加入 $0.2 \mathrm{~mL} 50 \%$ （体积比）硫酸溶液，混匀，加入 $2.0 \mathrm{~mL}$ 丙酮，混匀后加过量氯化钠，剧烈振摇 $1 \mathrm{~min}$ ，再加入 $2.0 \mathrm{~mL}$ 乙醚，振摇 $1 \mathrm{~min}$ 后，于 $3000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，将上清液转入另一试管中，下层溶液用 $2.0 \mathrm{~mL}$ 丙酮和 $2.0 \mathrm{~mL}$ 乙醚重复提取2次，合并有机相，于 $40^{\circ} \mathrm{C}$ 水浴中用氮气吹至少量溶液存在，用丙酮定容至 $1.0 \mathrm{~mL}$ ，混匀后备用。同样操作步骤处理乙酸标准和空白培养液。

# C.3.2 乳酸的处理

取双歧杆菌培养液 $2.0\mathrm{mL}\sim 3.0\mathrm{mL}$ 放入 $10\mathrm{mL}$ 比色管中， $100^{\circ}C$ 水浴 $10\mathrm{min}$ ，加入 $0.2\mathrm{mL}50\%$ （体积比）硫酸溶液，混匀，加入 $1.0\mathrm{mL}$ 甲醇，于 $58^{\circ}C$ 水浴 $30\mathrm{min}$ 后加水 $1.0\mathrm{mL}$ ，加三氯甲烷 $1.0\mathrm{mL}$ ，振摇 $3\mathrm{min}$ ， $3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取三氯甲烷层分析。同样操作步骤处理乳酸标准和空白培养液。

# C. 3.3 气相色谱条件

# C. 4 结果计算

样品培养液中乙酸或乳酸的含量按式（C.1）计算：

$$
X = \frac {\mathrm {A} _ {\text {样}} - \mathrm {A} _ {\text {空}}}{\mathrm {A} _ {\text {标}} \times \mathrm {C}} \dots \tag {C.1}
$$

式中：

$X$ ——样品培养液中乙酸或乳酸的含量, 单位为微摩尔每毫升 $(\mu \mathrm{mol} / \mathrm{mL})$ ;

A 样—样品培养液中乙酸或乳酸的峰面积；

A 空——空白培养液中乙酸或乳酸的峰面积；

A 标一乙酸标准或乳酸标准的峰面积；

C—乙酸标准或乳酸标准的浓度, 单位为微摩尔每毫升 $(\mu \mathrm{mol} / \mathrm{mL})$ 。

# C.5 允许差

相对相差 $\leqslant 15\%$

# C. 6 结果判定

如果乙酸(μmol/mL)与乳酸(μmol/mL)比值大于1，可判定为是双歧杆菌的有机酸代谢产物。

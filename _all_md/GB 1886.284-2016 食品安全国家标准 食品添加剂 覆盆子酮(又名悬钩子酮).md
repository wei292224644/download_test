# 食品安全国家标准

# 食品添加剂 覆盆子酮（又名悬钩子酮）

# 1 范围

本标准适用于由丁酮醇和苯酚、或对羟基苯甲醛(或大茴香醛)和丙酮为原料制得的食品添加剂覆盆子酮（又名悬钩子酮）。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

4-(对羟基苯基)-2-丁酮

# 2.2 分子式

$$
\mathrm {C} _ {1 0} \mathrm {H} _ {1 2} \mathrm {O} _ {2}
$$

# 2.3 结构式

![](images/1d56691f810c3655a4ace88ce085586db63745ad08df07e7d1d7128b67cf0fd6.jpg)

# 2.4 相对分子质量

164.20（按2007年国际相对原子质量）

# 附录A

# 覆盆子酮含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

试样制备：称取本品 $1\mathrm{g}$ 溶于 $10~\mathrm{mL}~95\%$ （体积分数）乙醇中，摇匀备用。

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行，应符合要求。

食品添加剂覆盆子酮气相色谱图及操作条件参见附录B。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ 。  
B.2.2 固定相： $100\%$ 甲基聚硅氧烷。  
B.2.3 膜厚: $0.25 \mu \mathrm{m}$ 。  
B.2.4 色谱炉温度： $220^{\circ}C$ 恒温。  
B.2.5 进样口温度： $250^{\circ}C$   
B.2.6 检测器温度： $280^{\circ}C$ 。  
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 柱前压： $100\mathrm{kPa}$   
B.2.10 进样量： $0.2\mu \mathrm{L}$   
B.2.11 分流比： $60:1$ 。

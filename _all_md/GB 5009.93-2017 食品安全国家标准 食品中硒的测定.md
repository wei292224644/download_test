# 前言

本标准代替GB5009.93—2010《食品安全国家标准 食品中硒的测定》、GB/T21729—2008《茶叶中硒含量的检测方法》、SN/T0860—2000《出口蘑菇罐头中硒的测定方法 荧光分光光度法》和SN/T0926—2000《进出口茶叶中硒的检验方法 荧光光度法》。

本标准与GB5009.93—2010相比，主要变化如下：

——保留氢化物原子荧光光谱法为第一法，荧光分光光度法为第二法；

——增加电感耦合等离子体质谱法为第三法。

# 食品安全国家标准

# 食品中硒的测定

# 1 范围

本标准规定了食品中硒含量测定的氢化物原子荧光光谱法、荧光分光光度法和电感耦合等离子体质谱法。

本标准适用于各类食品中硒的测定。

# 第一法 氢化物原子荧光光谱法

# 2 原理

试样经酸加热消化后，在 $6\mathrm{mol} / \mathrm{L}$ 盐酸介质中，将试样中的六价硒还原成四价硒，用硼氢化钠或硼氢化钾作还原剂，将四价硒在盐酸介质中还原成硒化氢，由载气(氩气)带入原子化器中进行原子化，在硒空心阴极灯照射下，基态硒原子被激发至高能态，在去活化回到基态时，发射出特征波长的荧光，其荧光强度与硒含量成正比，与标准系列比较定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 硝酸（ $\mathrm{HNO}_3$ ）：优级纯。  
3.1.2 高氯酸（ $\mathrm{HClO_4}$ ）：优级纯。  
3.1.3 盐酸（HCl）：优级纯。  
3.1.4 氢氧化钠（NaOH）：优级纯。  
3.1.5 过氧化氢 $\mathrm{(H_2O_2)}$ 。  
3.1.6 硼氢化钠（ $\mathrm{NaBH}_4$ ）：优级纯。  
3.1.7 铁氰化钾 $\left[\mathrm{K}_{3} \mathrm{Fe}(\mathrm{CN})_{6}\right]$ 。

# 3.2 试剂的配制

3.2.1 硝酸-高氯酸混合酸 $(9 + 1)$ ：将 $900~\mathrm{mL}$ 硝酸与 $100~\mathrm{mL}$ 高氯酸混匀。  
3.2.2 氢氧化钠溶液 $(5\mathrm{g} / \mathrm{L})$ ：称取 $5\mathrm{g}$ 氢氧化钠，溶于 $1000~\mathrm{mL}$ 水中，混匀。  
3.2.3 硼氢化钠碱溶液 $(8\mathrm{g} / \mathrm{L})$ ：称取 $8\mathrm{g}$ 硼氢化钠，溶于氢氧化钠溶液 $(5\mathrm{g} / \mathrm{L})$ 中，混匀。现配现用。  
3.2.4 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ : 量取 $50~\mathrm{mL}$ 盐酸, 缓慢加入 $40~\mathrm{mL}$ 水中, 冷却后用水定容至 $100~\mathrm{mL}$ , 混匀。  
3.2.5 铁氰化钾溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 铁氰化钾，溶于 $100~\mathrm{mL}$ 水中，混匀。

3.2.6 盐酸溶液 $(5 + 95)$ ：量取 $25~\mathrm{mL}$ 盐酸，缓慢加入 $475~\mathrm{mL}$ 水中，混匀。

# 3.3 标准品

硒标准溶液： $1000\mathrm{mg / L}$ ，或经国家认证并授予标准物质证书的一定浓度的硒标准溶液。

# 3.4 标准溶液的制备

3.4.1 硒标准中间液 $(100\mathrm{mg / L})$ ：准确吸取 $1.00~\mathrm{mL}$ 础标准溶液 $(1000\mathrm{mg / L})$ 于 $10~\mathrm{mL}$ 容量瓶中，加盐酸溶液 $(5 + 95)$ 定容至刻度，混匀。  
3.4.2 硒标准使用液 $(1.00\mathrm{mg / L})$ ：准确吸取硒标准中间液 $(100\mathrm{mg / L})1.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用盐酸溶液 $(5 + 95)$ 定容至刻度，混匀。  
3.4.3 硒标准系列溶液：分别准确吸取硒标准使用液 $(1.00\mathrm{mg / L})0\mathrm{mL},0.500\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL}$ 和 $3.00~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加入铁氰化钾溶液 $(100\mathrm{g / L})10\mathrm{mL}$ ，用盐酸溶液 $(5 + 95)$ 定容至刻度，混匀待测。此硒标准系列溶液的质量浓度分别为 $0\mu \mathrm{g / L},5.00\mu \mathrm{g / L},10.0\mu \mathrm{g / L},20.0\mu \mathrm{g / L}$ 和 $30.0\mu \mathrm{g / L}$ 。

注：可根据仪器的灵敏度及样品中硒的实际含量确定标准系列溶液中硒元素的质量浓度。

# 4 仪器和设备

注：所有玻璃器皿及聚四氟乙烯消解内罐均需硝酸溶液 $(1 + 5)$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

4.1 原子荧光光谱仪：配硒空心阴极灯。  
4.2 天平：感量为 $1\mathrm{mg}$   
4.3 电热板。  
4.4 微波消解系统：配聚四氟乙烯消解内罐。

# 5 分析步骤

# 5.1 试样制备

注：在采样和制备过程中，应避免试样污染。

# 5.1.1 粮食、豆类样品

样品去除杂物后，粉碎，储于塑料瓶中。

# 5.1.2 蔬菜、水果、鱼类、肉类等样品

样品用水洗净，晾干，取可食部分，制成匀浆，储于塑料瓶中。

# 5.1.3 饮料、酒、醋、酱油、食用植物油、液态乳等液体样品

将样品摇匀。

# 5.2 试样消解

# 5.2.1 湿法消解

称取固体试样 $0.5\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 或准确移取液体试样 $1.00~\mathrm{mL}\sim 5.00~\mathrm{mL}$ ，置于锥形瓶中，加 $10~\mathrm{mL}$ 硝酸-高氯酸混合酸 $(9 + 1)$ 及几粒玻璃珠，盖上表面皿冷消化过夜。次日于电热板上加

热，并及时补加硝酸。当溶液变为清亮无色并伴有白烟产生时，再继续加热至剩余体积为 $2\mathrm{mL}$ 左右，切不可蒸干。冷却，再加 $5\mathrm{mL}$ 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ，继续加热至溶液变为清亮无色并伴有白烟出现。冷却后转移至 $10\mathrm{mL}$ 容量瓶中，加入 $2.5\mathrm{mL}$ 铁氰化钾溶液 $(100\mathrm{g} / \mathrm{L})$ ，用水定容，混匀待测。同时做试剂空白试验。

# 5.2.2 微波消解

称取固体试样 $0.2\mathrm{g}\sim 0.8\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $1.00~\mathrm{mL}\sim 3.00~\mathrm{mL}$ ，置于消化管中，加 $10~\mathrm{mL}$ 硝酸、 $2\mathrm{mL}$ 过氧化氢，振摇混合均匀，于微波消解仪中消化，微波消化推荐条件见附录A（可根据不同的仪器自行设定消解条件）。消解结束待冷却后，将消化液转入锥形烧瓶中，加几粒玻璃珠，在电热板上继续加热至近干，切不可蒸干。再加 $5\mathrm{mL}$ 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ，继续加热至溶液变为清亮无色并伴有白烟出现，冷却，转移至 $10~\mathrm{mL}$ 容量瓶中，加入 $2.5~\mathrm{mL}$ 铁氰化钾溶液 $(100~\mathrm{g} / \mathrm{L})$ ，用水定容，混匀待测。同时做试剂空白试验。

# 5.3 测定

# 5.3.1 仪器参考条件

根据各自仪器性能调至最佳状态。参考条件为：负高压 $340\mathrm{V}$ ；灯电流 $100\mathrm{mA}$ ；原子化温度 $800^{\circ}C$ ；炉高 $8\mathrm{mm}$ ；载气流速 $500~\mathrm{mL / min}$ ；屏蔽气流速 $1000~\mathrm{mL / min}$ ；测量方式标准曲线法；读数方式峰面积；延迟时间 $1s$ ；读数时间 $15s$ ；加液时间 $8s$ ；进样体积 $2\mathrm{mL}$ 。

# 5.3.2 标准曲线的制作

以盐酸溶液 $(5 + 95)$ 为载流，硼氢化钠碱溶液 $(8\mathrm{g} / \mathrm{L})$ 为还原剂，连续用标准系列的零管进样，待读数稳定之后，将标硒标准系列溶液按质量浓度由低到高的顺序分别导入仪器，测定其荧光强度，以质量浓度为横坐标，荧光强度为纵坐标，制作标准曲线。

# 5.3.3 试样测定

在与测定标准系列溶液相同的实验条件下，将空白溶液和试样溶液分别导入仪器，测其荧光值强度，与标准系列比较定量。

# 6 分析结果的表述

试样中硒的含量按式(1)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中硒的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$\rho$ ——试样溶液中硒的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白溶液中硒的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

V ——试样消化液总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样称样量或移取体积，单位为克或毫升(g或 $\mathrm{mL}$ )；

1000 ——换算系数。

当硒含量 $\geqslant 1.00\mathrm{mg / kg}$ （或 $\mathrm{mg / L}$ )时，计算结果保留三位有效数字，当硒含量 $<  1.00~\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )时，计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当称样量为 $1\mathrm{g}$ (或 $1\mathrm{mL}$ )，定容体积为 $10~\mathrm{mL}$ 时，方法的检出限为 $0.002\mathrm{mg / kg}$ （或 $0.002\mathrm{mg / L}$ ），定量限为 $0.006\mathrm{mg / kg}$ （或 $0.006\mathrm{mg / L}$ ）。

# 第二法 荧光分光光度法

# 9 原理

将试样用混合酸消化，使硒化合物转化为无机硒 $\mathrm{Se}^{4+}$ ，在酸性条件下 $\mathrm{Se}^{4+}$ 与2,3-二氨基萘(2,3-Diaminonaphthalene，缩写为DAN)反应生成4,5-苯并苯硒脑(4,5-Benzo piaselenol)，然后用环己烷萃取后上机测定。4,5-苯并苯硒脑在波长为 $376\mathrm{nm}$ 的激发光作用下，发射波长为 $520\mathrm{nm}$ 的荧光，测定其荧光强度，与标准系列比较定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 10.1 试剂

10.1.1 盐酸（HCl）：优级纯。  
10.1.2 环己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{12}\right)$ ：色谱纯。  
10.1.3 2,3-二氨基萘 $\mathrm{(DAN,C_{10}H_{10}N_2)}$   
10.1.4 乙二胺四乙酸二钠(EDTA-2Na, $\mathrm{C_{10}H_{14}N_2Na_2O_8}$ )。  
10.1.5 盐酸羟胺 $(\mathrm{NH}_2\mathrm{OH}\cdot \mathrm{HCl})$ 。  
10.1.6 甲酚红 $(\mathrm{C}_{21}\mathrm{H}_{18}\mathrm{O}_5\mathrm{S})$ 。  
10.1.7 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ：优级纯。

# 10.2 试剂的配制

10.2.1 盐酸溶液 $(1\%)$ ：量取 $5\mathrm{mL}$ 盐酸，用水稀释至 $500~\mathrm{mL}$ ，混匀。  
10.2.2 DAN试剂 $(1\mathrm{g / L})$ ：此试剂在暗室内配制。称取DAN $0.2\mathrm{g}$ 于一带盖锥形瓶中，加入盐酸溶液 $(1\%)$ $200~\mathrm{mL}$ ，振摇约 $15\mathrm{min}$ 使其全部溶解。加入约 $40~\mathrm{mL}$ 环己烷，继续振荡 $5\mathrm{min}$ 。将此液倒入塞有玻璃棉(或脱脂棉)的分液漏斗中，待分层后滤去环己烷层，收集DAN溶液层，反复用环己烷纯化直至环己烷中荧光降至最低时为止(约纯化5次～6次)。将纯化后的DAN溶液储于棕色瓶中，加入约 $1\mathrm{cm}$ 厚的环己烷覆盖表层，于 $0^{\circ}\mathrm{C}\sim 5^{\circ}\mathrm{C}$ 保存。必要时在使用前再以环己烷纯化一次。

注：此试剂有一定毒性，使用本试剂的人员应注意防护。

10.2.3 硝酸-高氯酸混合酸 $(9 + 1)$ ：将 $900~\mathrm{mL}$ 硝酸与 $100~\mathrm{mL}$ 高氯酸混匀。  
10.2.4 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ：量取 $50~\mathrm{mL}$ 盐酸，缓慢加入 $40~\mathrm{mL}$ 水中，冷却后用水定容至 $100~\mathrm{mL}$

混匀。

10.2.5 氨水溶液 $(1 + 1)$ ：将 $5\mathrm{mL}$ 水与 $5\mathrm{mL}$ 氨水混匀。

10.2.6 EDTA混合液：

a) EDTA溶液 $(0.2\mathrm{mol} / \mathrm{L})$ ：称取EDTA- $2\mathrm{Na}37\mathrm{g}$ ，加水并加热至完全溶解，冷却后用水稀释至 $500~\mathrm{mL}$   
b）盐酸羟胺溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 盐酸羟胺溶于水中，稀释至 $100~\mathrm{mL}$ ，混匀；  
c）甲酚红指示剂 $(0.2\mathrm{g / L})$ ：称取甲酚红 $50~\mathrm{mg}$ 溶于少量水中，加氨水溶液 $(1 + 1)1$ 滴，待完全溶解后加水稀释至 $250~\mathrm{mL}$ ，混匀；  
d) 取EDTA溶液 $(0.2\mathrm{mol} / \mathrm{L})$ 及盐酸羟胺溶液 $(100\mathrm{g} / \mathrm{L})$ 各 $50~\mathrm{mL}$ , 加甲酚红指示剂 $(0.2\mathrm{g} / \mathrm{L})$ $5\mathrm{mL}$ , 用水稀释至 $1\mathrm{L}$ , 混匀。

10.2.7 盐酸溶液 $(1 + 9)$ ：量取 $100~\mathrm{mL}$ 盐酸，缓慢加入到 $900~\mathrm{mL}$ 水中，混匀。

# 10.3 标准品

硒标准溶液： $1000\mathrm{mg / L}$ ，或经国家认证并授予标准物质证书的一定浓度的硒标准溶液。

# 10.4 标准溶液的制备

10.4.1 硒标准中间液 $(100\mathrm{mg / L})$ ：准确吸取 $1.00~\mathrm{mL}$ 础标准溶液 $(1000\mathrm{mg / L})$ 于 $10~\mathrm{mL}$ 容量瓶中，加盐酸溶液 $(1\%)$ 定容至刻度，混匀。  
10.4.2 硒标准使用液 $(50.0\mu \mathrm{g} / \mathrm{L})$ ：准确吸取硒标准中间液 $(100\mathrm{mg / L})0.50\mathrm{mL}$ ，用盐酸溶液 $(1\%)$ 定容至 $1000~\mathrm{mL}$ ，混匀。  
10.4.3 硒标准系列溶液：准确吸取硒标准使用液 $(50.0\mu \mathrm{g} / \mathrm{L})0\mathrm{mL}, 0.200\mathrm{mL}, 1.00\mathrm{mL}, 2.00\mathrm{mL}$ 和 $4.00\mathrm{mL}$ ，相当于含有硒的质量为 $0\mu \mathrm{g}, 0.0100\mu \mathrm{g}, 0.0500\mu \mathrm{g}, 0.100\mu \mathrm{g}$ 及 $0.200\mu \mathrm{g}$ ，加盐酸溶液 $(1 + 9)$ 至 $5\mathrm{mL}$ 后，加入 $20\mathrm{mL}$ EDTA 混合液，用氨水溶液 $(1 + 1)$ 及盐酸溶液 $(1 + 9)$ 调至淡红橙色（ $\mathrm{pH}1.5\sim$ 2.0）。以下步骤在暗室操作：加 DAN 试剂 $(1\mathrm{g} / \mathrm{L})3\mathrm{mL}$ ，混匀后，置沸水浴中加热 $5\mathrm{min}$ ，取出冷却后，加环己烷 $3\mathrm{mL}$ ，振摇 $4\mathrm{min}$ ，将全部溶液移入分液漏斗，待分层后弃去水层，小心将环己烷层由分液漏斗上口倾入带盖试管中，勿使环己烷中混入水滴。环己烷中反应产物为 4,5-苯并苯硒脑，待测。

# 12 分析步骤

# 12.1 试样制备

同5.1。

# 12.2 试样消解

准确称取 $0.5\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 固体试样，或准确吸取液体试样 $1.00~\mathrm{mL}\sim 5.00~\mathrm{mL}$ ，置于锥

形瓶中，加 $10~\mathrm{mL}$ 硝酸-高氯酸混合酸 $(9 + 1)$ 及几粒玻璃珠，盖上表面皿冷消化过夜。次日于电热板上加热，并及时补加硝酸。当溶液变为清亮无色并伴有白烟产生时，再继续加热至剩余体积 $2\mathrm{mL}$ 左右，切不可蒸干，冷却后再加 $5\mathrm{mL}$ 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ，继续加热至溶液变为清亮无色并伴有白烟出现，再继续加热至剩余体积 $2\mathrm{mL}$ 左右，冷却。同时做试剂空白。

# 12.3 测定

# 12.3.1 仪器参考条件

根据各自仪器性能调至最佳状态。参考条件为：激发光波长 $376\mathrm{nm}$ 、发射光波长 $520\mathrm{nm}$ 。

# 12.3.2 标准曲线的制作

将硒标准系列溶液按质量由低到高的顺序分别上机测定4，5-苯并苯硒脑的荧光强度。以质量为横坐标，荧光强度为纵坐标，制作标准曲线。

# 12.3.3 试样溶液的测定

将12.2消化后的试样溶液以及空白溶液加盐酸溶液 $(1 + 9)$ 至 $5\mathrm{mL}$ 后，加入 $20~\mathrm{mL}$ EDTA混合液，用氨水溶液 $(1 + 1)$ 及盐酸溶液 $(1 + 9)$ 调至淡红橙色 $\left(\mathrm{pH}1.5\sim 2.0\right)$ 。以下步骤在暗室操作：加DAN试剂 $(1\mathrm{g / L})3\mathrm{mL}$ ，混匀后，置沸水浴中加热 $5\mathrm{min}$ ，取出冷却后，加环己烷 $3\mathrm{mL}$ ，振摇 $4\mathrm{min}$ ，将全部溶液移入分液漏斗，待分层后弃去水层，小心将环己烷层由分液漏斗上口倾入带盖试管中，勿使环己烷中混入水滴，待测。

# 13 分析结果的表述

试样中硒的含量按式(2)计算：

$$
X = \frac {m _ {1}}{F _ {1} - F _ {0}} \times \frac {F _ {2} - F _ {0}}{m} \tag {2}
$$

式中：

$X$ ——试样中硒含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$m_{1}$ ——试样管中硒的质量，单位为微克 $(\mu \mathrm{g})$

$F_{1}$ ——标准管硒荧光读数；

$F_{0}$ ——空白管荧光读数；

$F_{2}$ ——试样管荧光读数；

$m$ ——试样称样量或移取体积，单位为克或毫升（g或 $\mathrm{mL}$ ）。

当硒含量 $\geqslant 1.00\mathrm{mg / kg}$ （或 $\mathrm{mg / L}$ )时，计算结果保留三位有效数字；当硒含量 $<  1.00~\mathrm{mg / kg}$ (或 $\mathrm{mg / L}$ )时，计算结果保留两位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 前言

本标准代替GB6226—2005《食品添加剂 乳酸钙》。

本标准与GB6226—2005相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 乳酸钙”。

# 食品安全国家标准

# 食品添加剂 乳酸钙

# 1 范围

本标准适用于以淀粉、糖质原料接种乳酸杆菌发酵直接提取制得的以及乳酸与氢氧化钙(或碳酸钙)合成制得的食品添加剂乳酸钙。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

$\alpha$ -羟基丙酸钙

# 2.2 分子式

$$
\mathrm {C} _ {6} \mathrm {H} _ {1 0} \mathrm {C a O} _ {6} \bullet n \mathrm {H} _ {2} \mathrm {O} (n = 0 \sim 5)
$$

# 2.3 结构式

$$
\left[ \begin{array}{c} \mathrm {H} \\ \mathrm {H} _ {3} \mathrm {C} - \mathrm {C} - \mathrm {C O O} ^ {-} \\ \mathrm {O H} \end{array} \right] _ {2} \mathrm {C a} ^ {2 +} \cdot n \mathrm {H} _ {2} \mathrm {O}
$$

# 2.4 相对分子质量

218.22（无水物）（按2007年国际相对原子质量）

# 附录A

# 检验方法

# A.1 警示

试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和健康措施。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 硫酸。  
A.3.1.2 高锰酸钾溶液： $3.2\mathrm{g / L}$   
A.3.1.3 草酸铵溶液： $40~\mathrm{g / L}$   
A.3.1.4 乙酸溶液： $1 + 20$   
A.3.1.5 盐酸溶液： $1 + 3$

# A.3.2 鉴别方法

# A.3.2.1 乳酸盐鉴别

取约 $0.5\mathrm{g}$ 试样，溶于 $10~\mathrm{mL}$ 水，滴加硫酸使其呈酸性，加适量高锰酸钾溶液，加热，即产生乙醛气味。

# A.3.2.2 钙盐鉴别

A.3.2.2.1 取约 $0.5\mathrm{g}$ 试样，溶于 $10~\mathrm{mL}$ 水，滴加草酸铵溶液，即产生白色沉淀。分离沉淀，加入乙酸溶液，沉淀不溶解，再加入盐酸溶液，沉淀完全溶解。  
A.3.2.2.2 用盐酸湿润后的铂丝蘸取试样，在无色火焰中呈黄红色。

# A.4 乳酸钙 $\left(\mathbf{C}_6\mathbf{H}_{10}\mathbf{CaO}_6\right)$ 含量（以干基计）的测定

# A.4.1 方法提要

在碱性条件下，以试样(换算为干品)消耗络合剂乙二胺四乙酸二钠标准滴定溶液的体积计算其乳酸钙含量，用钙试剂羧酸钠指示剂的颜色变化判断滴定的终点。

# A.4.2 试剂和材料

A.4.2.1 盐酸溶液： $1 + 4$

A.4.2.2 氢氧化钠溶液： $100\mathrm{g / L}$

A.4.2.3 乙二胺四乙酸二钠标准滴定溶液： $c(\mathrm{EDTA}) = 0.05\mathrm{mol / L}$

A.4.2.4钙试剂羧酸钠指示剂：称取 $0.1\mathrm{g}$ 钙试剂羧酸钠盐，加 $10\mathrm{g}$ 在约 $110^{\circ}C$ 干燥过的氯化钠研磨，混匀。

# A.4.3 分析步骤

称取约 $0.3\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，溶于已加有 $2\mathrm{mL}$ 盐酸溶液的 $50\mathrm{mL}$ 水中，边搅拌边滴加 $15\mathrm{mL}$ 乙二胺四乙酸二钠标准滴定溶液，再加 $5\mathrm{mL}$ 氢氧化钠溶液和 $0.1\mathrm{g}$ 钙试剂羧酸钠指示剂，用乙二胺四乙酸二钠标准滴定溶液滴定至溶液呈现蓝色为终点。

# A.4.4 结果计算

乳酸钙 $\mathrm{(C_6H_{10}CaO_6)}$ 含量(以干基计)的质量分数 $w_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\frac {V _ {1}}{1 000} \times c _ {1} \times M _ {1}}{m _ {1} \times \left(1 - w _ {2}\right)} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——试样消耗乙二胺四乙酸二钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算因子；

$c_{1}$ ——乙二胺四乙酸二钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{1}$ ——乳酸钙的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M_1(\mathrm{C}_6\mathrm{H}_{10}\mathrm{CaO}_6) = 218.2]$

$m_{1}$ ——试样的质量，单位为克(g)；

$w_{2}$ ——A.5.2干燥减量的质量分数， $\%$ 。

取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.3\%$ 。

# A.5 干燥减量的测定

# A.5.1 分析步骤

称取约 $1.5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于预先在 $120^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 干燥至质量恒定的称量瓶中，铺成 $3\mathrm{mm}$ 以下的层。在 $120^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 的恒温干燥箱中干燥 $4\mathrm{h}$ ，置于干燥器中冷却 $30\mathrm{min}$ 称量。

# A.5.2 结果计算

干燥减量的质量分数 $\varpi_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m _ {2} - m _ {3}}{m _ {2}} \times 100 \% \tag{A.2}
$$

式中：

$m_{2}$ ——干燥前试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{3}$ ——干燥后试样的质量，单位为克 $(\mathrm{g})$ 。

取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.1\%$ 。

# A.6 水溶解的测定

# A.6.1 方法提要

将试样溶于水，与标准比浊溶液进行比较。

# A.6.2 试剂和材料

A.6.2.1 硝酸溶液： $1 + 2$

A.6.2.2 糊精溶液： $20~\mathrm{g / L}$

A.6.2.3 硝酸银溶液： $17\mathrm{g / L}$

A.6.2.4 浊度标准溶液：含氯(Cl) $0.01\mathrm{mg / mL}$ 。量取 $14.10~\mathrm{mL}c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$ 盐酸标准溶液，置于 $50~\mathrm{mL}$ 容量瓶中，加水稀释至刻度。量取 $10.00~\mathrm{mL}$ 该溶液，置于 $1000~\mathrm{mL}$ 容量瓶中，加水稀释至刻度。

# A.6.3 分析步骤

称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $25\mathrm{mL}$ 比色管中，加 $20\mathrm{mL}$ 水，在水浴上加热溶解，作为试样溶液；取另一只 $25\mathrm{mL}$ 比色管，加入 $0.20\mathrm{mL}$ 浊度标准溶液，加水至 $20\mathrm{mL}$ ，加 $1\mathrm{mL}$ 硝酸溶液、 $0.2\mathrm{mL}$ 糊精溶液和 $1\mathrm{mL}$ 硝酸银溶液，摇匀，避光放置 $15\mathrm{min}$ ，作为标准比浊溶液。

在无阳光直射情况下，目视轴向及侧面观察，试样溶液的浊度不得大于标准比浊溶液的浊度。

# A.7 游离酸和游离碱的测定

# A.7.1 试剂和材料

A.7.1.1 无二氧化碳的水。  
A.7.1.2 氢氧化钠标准溶液： $c(\mathrm{NaOH}) = 0.1\mathrm{mol / L}$   
A.7.1.3 酚酞指示液： $10\mathrm{g / L}$

# A.7.2 分析步骤

称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $20~\mathrm{mL}$ 无二氧化碳的水，加2滴酚酞指示液，不应有粉红色产生；加 $0.5\mathrm{mL}$ 氢氧化钠标准溶液，应呈粉红色。

# A.8 挥发性脂肪酸的测定

# A.8.1 试剂和材料

硫酸。

# A.8.2 分析步骤

称取约 $0.5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于干燥蒸发皿中，加 $1\mathrm{mL}$ 硫酸，搅拌，在水浴上加热，不应有脂肪酸气味逸出。

# A.9 氯化物（以CI计）的测定

# A.9.1 试剂和材料

A.9.1.1 硝酸溶液： $1 + 9$   
A.9.1.2 硝酸银溶液： $17\mathrm{g / L}$   
A.9.1.3 氯化物(Cl)标准溶液： $0.01\mathrm{mg / mL}$

# A.9.2 分析步骤

称取 $1.0\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），加 $40\mathrm{mL}$ 水溶解，用硝酸溶液调节溶液至中性，用水稀释至 $50\mathrm{mL}$ ，振摇，过滤，保留滤液。取滤液 $5.0\mathrm{mL}$ ，作为试样溶液，置于 $50\mathrm{mL}$ 纳氏比色管中，加 $10\mathrm{mL}$ 硝酸溶液，加水至体积约 $40\mathrm{mL}$ ，加 $1.0\mathrm{mL}$ 硝酸银溶液，用水稀释至 $50\mathrm{mL}$ ，摇匀，暗处放置 $5\mathrm{min}$ ，在黑色背景下，轴向观察，试样溶液所呈浊度不得大于标准比浊溶液。

标准比浊溶液：移取 $5.00\mathrm{mL}$ 氯化物(Cl)标准溶液，置于 $50~\mathrm{mL}$ 比色管中。与试样溶液同时同样处理。

# A.10 硫酸盐（以 $\mathrm{SO}_4$ 计）的测定

# A.10.1 试剂和材料

A.10.1.1 盐酸溶液： $1 + 3$   
A.10.1.2 氯化钡溶液： $250~\mathrm{g / L}$   
A.10.1.3 硫酸钾标准溶液： $0.1\mathrm{mg / mL}$

# A.10.2 分析步骤

称取 $1.0\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )，置于 $50~\mathrm{mL}$ 纳氏比色管中，加 $40~\mathrm{mL}$ 水溶解，用盐酸溶液调节溶液至中性，振摇，加 $2\mathrm{mL}$ 盐酸溶液，加 $5\mathrm{mL}$ 氯化钡溶液，用水稀释至 $50~\mathrm{mL}$ ，摇匀，放置 $10\mathrm{min}$ ，在黑色背景下，轴向观察，试样溶液所呈浊度不得大于标准比浊溶液。

标准比浊溶液：量取 $7.5\mathrm{mL}$ 硫酸钾标准溶液，置于 $50~\mathrm{mL}$ 比色管中。与试样溶液同时同样处理。

# A.11 氟化物（以F计）的测定

# A.11.1 方法提要

在高氯酸介质中，通过蒸汽蒸馏使氟自试样中分离，氟与茜素氨羧络合剂和硝酸镧的混合剂形成蓝色络合物。将试样溶液的颜色与标准比色溶液进行比较。

# A.11.2 试剂和材料

A.11.2.1 高氯酸。  
A.11.2.2 丙酮。  
A.11.2.3 硝酸银溶液： $17\mathrm{g / L}$

A.11.2.4 氢氧化钠溶液： $4\mathrm{g / L}$

A.11.2.5 氢氧化钠溶液： $40\mathrm{g / L}$

A.11.2.6 盐酸溶液： $1 + 10$

A.11.2.7 茜素氨羧络合剂溶液: 称取 $0.193 \mathrm{~g}$ 茜素氨羧络合剂, 加少量水及氢氧化钠溶液 $(40 \mathrm{~g} / \mathrm{L})$ 使其溶解, 加 $0.13 \mathrm{~g}$ 乙酸钠, 用乙酸溶液 $(1+16)$ 调节 $\mathrm{pH}$ 为 5.0 (用精密 $\mathrm{pH}$ 试纸检验, 呈红色), 加水稀释至 $500 \mathrm{~mL}$ , 贮于棕色瓶中, 置冰箱内 $(6^{\circ} \mathrm{C} \sim 8^{\circ} \mathrm{C})$ 保存, 出现沉淀时应重配。

A.11.2.8 缓冲溶液 $(\mathrm{pH}\approx 4.7)$ ：称取 $44\mathrm{g}$ 乙酸钠溶于 $400~\mathrm{mL}$ 水中，加 $22~\mathrm{mL}$ 冰乙酸，再缓缓加冰乙酸调节 $\mathrm{pH}$ 为4.7，然后加水稀释至 $500~\mathrm{mL}$ 。

A.11.2.9 硝酸镧溶液：称取 $0.271\mathrm{g}$ 硝酸镧，用少量乙酸溶液 $(1 + 16)$ 溶解，加水至约 $450~\mathrm{mL}$ ，用 $25~\mathrm{g / L}$ 乙酸钠溶液调节 $\mathrm{pH}$ 为5.0（用精密 $\mathrm{pH}$ 试纸检验），再加水稀释至 $500~\mathrm{mL}$ ，置冰箱内保存，生霉后应重配。

A.11.2.10 氟化物标准溶液： $0.01\mathrm{mg / mL}$ 。临用前用水将 $0.1\mathrm{mg / mL}$ 的氟化物标准溶液稀释而成。

A.11.2.11 酚酞指示液： $10\mathrm{g / L}$

# A.11.3 仪器和设备

测氟蒸馏装置示意图见图A.1。

# A.11.4 分析步骤

称取 $2.00\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 三口烧瓶中（见图A.1），加几粒玻璃珠，慢慢加入 $10~\mathrm{mL}$ 高氯酸，用约 $8\mathrm{mL}$ 水冲洗瓶壁，加4滴～6滴硝酸银溶液，瓶塞上的温度计应密塞，并将温度计的水银球插入试样溶液中，连接好水蒸气发生器及直形冷凝器，将冷凝器的末端接上玻璃弯管，并使弯管插入盛有 $10~\mathrm{mL}$ 氢氧化钠溶液(A.11.2.4)和2滴酚酞指示液的 $250~\mathrm{mL}$ 容量瓶中。水蒸气发生器中加 $500~\mathrm{mL}$ 水，滴加氢氧化钠溶液(A.11.2.5)使溶液呈碱性。打开螺丝夹，加热至近沸。关闭螺丝夹，将水蒸气通入三口烧瓶中，三口烧瓶同时加热，调节水蒸气的进入量，使温度上升后保持在 $135^{\circ}C$ $\sim$ $140^{\circ}C$ 之间。如果容量瓶中溶液褪色，补加适量的氢氧化钠溶液(A.11.2.4)直到馏出液约为 $200~\mathrm{mL}$ ，停止蒸馏，摇匀。用氢氧化钠溶液(A.11.2.4)或盐酸溶液调节至 $\mathrm{pH}$ 为7.0(用精密 $\mathrm{pH}$ 试纸检验)，再多加2滴盐酸溶液，用水稀释至刻度，摇匀，为试样溶液。移取 $25.00~\mathrm{mL}$ 试样溶液，置于 $50~\mathrm{mL}$ 比色管中，加 $5\mathrm{mL}$ 茜素氨羧络合剂溶液和 $3\mathrm{mL}$ 缓冲溶液，混匀。慢慢加入 $5\mathrm{mL}$ 硝酸镧溶液，振摇，再加 $10~\mathrm{mL}$ 丙酮，加水至 $50~\mathrm{mL}$ ，室温放置 $20\mathrm{min}$ ，与标准比色溶液比较，其颜色不得深于标准比色溶液。

标准比色溶液是移取 $0.30\mathrm{mL}$ 氟化物标准溶液置于 $50~\mathrm{mL}$ 比色管中，加入 $25~\mathrm{mL}$ 水，加 $5\mathrm{mL}$ 茜素氨羧络合剂溶液和 $3\mathrm{mL}$ 缓冲溶液，混匀。慢慢加入 $5\mathrm{mL}$ 硝酸镧溶液，振摇，再加 $10~\mathrm{mL}$ 丙酮，加水至 $50~\mathrm{mL}$ ，室温放置 $20\mathrm{min}$ 。

# A.12 镁及碱金属的测定

# A.12.1 方法提要

试样中的钙在酸性条件下与草酸生成难溶的草酸钙沉淀，过滤，滤液中的镁及碱金属和硫酸加热生成硫酸盐，用热水浸取，再过滤，滤液蒸干，灼烧后定量。

# A.12.2 试剂和材料

A.12.2.1 盐酸。  
A.12.2.2 硫酸。  
A.12.2.3 氨水。  
A.12.2.4 草酸溶液： $150~\mathrm{g / L}$   
A.12.2.5 甲基红指示液： $1\mathrm{g / L}$

# A.12.3 分析步骤

称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加 $40~\mathrm{mL}$ 水，再加 $1\mathrm{mL}$ 盐酸，煮沸 $1\mathrm{min}$ ，迅速加 $40~\mathrm{mL}$ 草酸溶液

和2滴甲基红指示液，用氨水中和至溶液刚好呈黄色，冷却至室温。移入 $100~\mathrm{mL}$ 容量瓶中，稀释至刻度，摇匀，放置过夜。用干燥滤纸过滤，取 $50~\mathrm{mL}$ 澄清滤液，置于蒸发皿中，加 $1\mathrm{mL}$ 硫酸，在水浴上蒸发至近干，在电炉上加热至硫酸蒸气逸尽。残渣用 $25~\mathrm{mL}$ 水，加热至 $80^{\circ}\mathrm{C}$ 浸取，过滤，滤液置于预先于 $800^{\circ}\mathrm{C}\pm 25^{\circ}\mathrm{C}$ 干燥至质量恒定的坩埚中，在电炉上加热蒸干，于 $800^{\circ}\mathrm{C}\pm 25^{\circ}\mathrm{C}$ 灼烧至质量恒定。残渣质量不得大于 $5.0~\mathrm{mg}$ 。

# A.13 铁(Fe)的测定

# A.13.1 试剂和材料

A.13.1.1 过硫酸铵。  
A.13.1.2 盐酸溶液： $1 + 3$   
A.13.1.3 硫氰酸铵溶液： $300\mathrm{g / L}$   
A.13.1.4 铁(Fe)标准溶液: $0.1 \mathrm{mg} / \mathrm{mL}$ 。临用前配制成 $0.01 \mathrm{mg} / \mathrm{mL}$ 硫酸铁铵标准溶液。  
A.13.1.5 标准比色溶液：取 $2.5\mathrm{mL}$ 铁(Fe)标准溶液 $(0.01\mathrm{mg / mL})$ ，置于 $50~\mathrm{mL}$ 纳氏比色管中，加 $4\mathrm{mL}$ 盐酸溶液与 $50~\mathrm{mg}$ 过硫酸铵，用水稀释成 $35~\mathrm{mL}$ 后，加 $3\mathrm{mL}$ 硫氰酸铵溶液，用水稀释至 $50~\mathrm{mL}$ 纳氏比色管。

# A.13.2 分析步骤

称取 $0.5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加 $25\mathrm{mL}$ 水，置于水浴中加热溶解，冷却后为试样溶液，加 $4\mathrm{mL}$ 盐酸溶液与 $50\mathrm{mg}$ 过硫酸铵，用水稀释成 $35\mathrm{mL}$ 后，加 $3\mathrm{mL}$ 硫氰酸铵溶液，用水稀释至 $50\mathrm{mL}$ 纳氏比色管。试样溶液所呈颜色不得深于标准比色溶液。

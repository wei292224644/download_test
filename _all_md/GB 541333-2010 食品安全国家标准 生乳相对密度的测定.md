# 食品安全国家标准

# 生乳相对密度的测定

# 1 范围

本标准规定了生乳相对密度的测定方法。

本标准适用于生乳相对密度的测定。

# 2 原理

使用密度计检测试样，根据读数经查表可得相对密度的结果。

# 3 仪器和设备

# 3.1 密度计： $20^{\circ}\mathrm{C} / 4^{\circ}\mathrm{C}$

3.2 玻璃圆筒或 $200\mathrm{mL}\sim 250\mathrm{mL}$ 量筒：圆筒高度应大于密度计的长度，其直径大小应使在沉入密度计时其周边和圆筒内壁的距离不小于 $5\mathrm{mm}$ 。

# 4 分析步骤

取混匀并调节温度为 $10^{\circ} \mathrm{C} \sim 25^{\circ} \mathrm{C}$ 的试样，小心倒入玻璃圆筒内，勿使其产生泡沫并测量试样温度。小心将密度计放入试样中到相当刻度 $30^{\circ}$ 处，然后让其自然浮动，但不能与筒内壁接触。静置 $2 \mathrm{~min} \sim 3 \mathrm{~min}$ ，眼睛平视生乳液面的高度，读取数值。根据试样的温度和密度计读数查表1换算成 $20^{\circ} \mathrm{C}$ 时的度数。

# 5 分析结果的表述

相对密度 $\left(\rho_{4}^{20}\right)$ 与密度计刻度关系式见式（1）：

$$
\rho_ {4} ^ {2 0} = \frac {\mathrm {X}}{1 0 0 0} + 1. 0 0 0 \dots \tag {1}
$$

式中：

$\rho_{4}^{20}$ ——样品的相对密度；

X——密度计读数。

当用 $20\mathrm{^\circ C / 4}$ ℃密度计，温度在 $20\mathrm{^\circ C}$ 时，将读数代入式(1)相对密度即可直接计算；不在 $20\mathrm{^\circ C}$ 时，要查表1换算成 $20\mathrm{^\circ C}$ 时度数，然后再代入式(1)计算。

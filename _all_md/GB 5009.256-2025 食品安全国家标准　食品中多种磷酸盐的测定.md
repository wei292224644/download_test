# 前言

本标准代替GB5009.256—2016《食品安全国家标准 食品中多种磷酸盐的测定》。

本标准与GB5009.256—2016相比，主要变化如下：

——更改了方法的适用范围；  
——删除了六偏磷酸盐的测定；  
——更改了方法原理的表述；  
——增加了“第二法 食品中磷酸根 $\left(\mathrm{PO}_{4}^{3-}\right)$ 总量的测定”。

# 食品安全国家标准

# 食品中多种磷酸盐的测定

# 1 范围

本标准规定了食品中以磷酸根 $\mathrm{(PO_4^{3 - })}$ 计的多种磷酸盐的测定方法。

本标准第一法适用于食品中磷酸盐、焦磷酸盐、三偏磷酸盐、三聚磷酸盐的测定，不适用于食品中添加六偏磷酸盐的测定。

本标准第二法适用于食品中以磷酸根 $\left(\mathrm{PO}_{4}^{3-}\right)$ 计的多种磷酸盐的测定。

# 第一法 食品中4种磷酸盐的测定

# 2 原理

试样经氢氧化钠溶液提取、固相萃取柱净化后，用氢氧化钾溶液淋洗，阴离子交换柱分离，电导检测器检测。以保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 氢氧化钠(NaOH)。  
3.1.2 氢氧化钾(KOH)。  
3.1.3 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

# 3.2 试剂配制

3.2.1 氢氧化钠溶液 $(10\mathrm{mmol / L})$ ：称取 $0.4\mathrm{g}$ 氢氧化钠，加水溶解并定容至 $1000~\mathrm{mL}$ 。  
3.2.2 氢氧化钠溶液 $(50\mathrm{mmol / L})$ ：称取 $2.0\mathrm{g}$ 氢氧化钠，加水溶解并定容至 $1000~\mathrm{mL}$ 。  
3.2.3 氢氧化钾溶液 $(2\mathrm{mol} / \mathrm{L})$ ：称取 $112.0\mathrm{g}$ 氢氧化钾，加水溶解并定容至 $1000~\mathrm{mL}$ 。

# 3.3 标准品

3.3.1 磷酸钠 $\mathrm{(Na_3PO_4}$ ，CAS号：7601-54-9)，纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.2 焦磷酸钠 $\mathrm{(Na_4P_2O_7}$ ,CAS号：7722-88-5)，纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.3 三偏磷酸钠 $\left[\left(\mathrm{NaPO}_{3}\right)_{3},\mathrm{CAS}\right.$ 号：7785-84-4]，纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质。

3.3.4 三聚磷酸钠 $\left(\mathrm{Na}_{5} \mathrm{P}_{3} \mathrm{O}_{10}, \mathrm{CAS}\right.$ 号：7758-29-4），纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.2 焦磷酸根标准储备溶液 $(1000\mathrm{mg / L})$ ：准确称取焦磷酸钠 $0.153\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），用 $10\mathrm{mmol / L}$ 氢氧化钠溶液溶解并定容至 $100~\mathrm{mL}$ ，此溶液每1L含焦磷酸根 $1.0\mathrm{g}$ 。常温密闭保存，有效期6个月。  
3.4.3 三偏磷酸根标准储备溶液 $(1000\mathrm{mg / L})$ ：将三偏磷酸钠在 $103~^\circ \mathrm{C}\pm 2~^\circ \mathrm{C}$ 烘箱中干燥 $3\mathrm{h}$ ，在干燥器中冷却至室温后，准确称取三偏磷酸钠标准品 $0.132\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），用 $10\mathrm{mmol / L}$ 氢氧化钠溶液溶解并定容至 $100\mathrm{mL}$ ，此溶液每 $1\mathrm{L}$ 含三偏磷酸根 $1.0\mathrm{g}$ 。常温密闭保存，有效期6个月。  
3.4.4 三聚磷酸根标准储备溶液 $(1000\mathrm{mg / L})$ ：将三聚磷酸钠在 $103^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 烘箱中干燥 $3\mathrm{h}$ ，在干燥器中冷却至室温后，准确称取三聚磷酸钠标准品 $0.148\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），用 $10\mathrm{mmol / L}$ 氢氧化钠溶液溶解并定容至 $100\mathrm{mL}$ ，此溶液每 $1\mathrm{L}$ 含三聚磷酸根 $1.0\mathrm{g}$ 。常温密闭保存，有效期6个月。  
3.4.5 磷酸根、焦磷酸根、三偏磷酸根、三聚磷酸根混合标准中间溶液 $(100\mathrm{mg / L})$ ：分别量取磷酸根、焦磷酸根、三偏磷酸根、三聚磷酸根标准储备溶液各 $10.0~\mathrm{mL}$ ，用 $10\mathrm{mmol / L}$ 氢氧化钠溶液定容至 $100~\mathrm{mL}$ 。  
3.4.6 混合标准系列工作溶液：分别量取混合标准中间溶液 $0\mathrm{mL}, 0.03\mathrm{mL}, 0.05\mathrm{mL}, 0.10\mathrm{mL}, 0.50\mathrm{mL}, 1.00\mathrm{mL}$ 和 $2.00\mathrm{mL}$ ，用 $10\mathrm{mmol/L}$ 氢氧化钠溶液定容至 $10\mathrm{mL}$ ，配制成质量浓度分别为 $0\mathrm{mg/L}, 0.30\mathrm{mg/L}, 0.50\mathrm{mg/L}, 1.00\mathrm{mg/L}, 5.00\mathrm{mg/L}, 10.0\mathrm{mg/L}$ 和 $20.0\mathrm{mg/L}$ 的混合标准系列工作溶液。临用现配。

# 6 分析结果的表述

试样中各种磷酸根含量按公式(1)计算。

$$
X _ {i} = \frac {\left(\rho_ {i} - \rho_ {0 i}\right) \times V}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X_{i}$ ——样品中第 $i$ 个磷酸根含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho_{i}$ ——样液中第 $i$ 个磷酸根测定值，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_{0i}$ ——样品空白液中第 $i$ 个磷酸根测定值，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——样品提取液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品称样质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，结果保留3位有效数字。

若分析结果需要以磷酸根含量表示，则见附录B，将聚磷酸根含量乘以换算系数 $F$ 。样品中总磷酸根为各种聚磷酸根(除六偏磷酸根外)换算的磷酸根的总和。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不应超过算术平均值的 $15\%$ 。

# 8 检出限与定量限

膨化食品、熟制坚果与籽类、谷类和淀粉类甜品、米粉、婴幼儿配方食品及辅助食品等：取样 $2.5\mathrm{g}$ ，定容至 $50~\mathrm{mL}$ ，测定前再稀释2.5倍，测定各聚磷酸根的检出限均为 $0.002\mathrm{g / kg}$ ，定量限均为 $0.005\mathrm{g / kg}$ 。

蔬菜、水果、果冻、巧克力及糖果、杂粮、小麦粉及其制品、奶粉、乳及乳制品、饮料类等：取样 $2.5\mathrm{g}$ ，定容至 $50~\mathrm{mL}$ ，测定前再稀释5倍，测定各聚磷酸根的检出限均为 $0.003\mathrm{g / kg}$ ，定量限均为 $0.01\mathrm{g / kg}$ 。

油脂、脂肪、调味料等：取样 $1.0\mathrm{g}$ ，定容至 $50~\mathrm{mL}$ ，水产、蛋、肉类及其制品等：取样 $2.5\mathrm{g}$ ，定容至 $100~\mathrm{mL}$ ，测定前再稀释2.5倍，测定各聚磷酸根的检出限均为 $0.006\mathrm{g / kg}$ ，定量限均为 $0.02\mathrm{g / kg}$ 。

# 第二法 食品中磷酸根 $\left(\mathrm{PO}_{4}^{3-}\right)$ 总量的测定

# 9 原理

试样中各种磷酸盐经提取后，在酸性条件下转化为磷酸根 $\left(\mathrm{PO}_{4}^{3-}\right)$ 离子，经阴离子交换柱分离，电导检测器检测。根据保留时间定性，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 硝酸 $(\mathrm{HNO}_3)$ ：质量浓度为 $65\%$ 。  
10.1.2 碳酸钠 $\mathrm{(Na_2CO_3)}$ 。  
10.1.3 氢氧化钠(NaOH)。  
10.1.4 氢氧化钾(KOH)。

# 10.2 试剂配制

10.2.1 硝酸溶液（体积分数 $30\%$ ）：量取硝酸 $30~\mathrm{mL}$ ，缓缓加入 $50~\mathrm{mL}$ 水中，待降至室温后稀释至 $100~\mathrm{mL}$ 。  
10.2.2 氢氧化钠溶液 $(50\mathrm{mmol / L})$ ：同3.2.2。

10.2.3 氢氧化钾溶液（ $2\mathrm{mol} / \mathrm{L}$ ）：同3.2.3。  
10.2.4 碳酸钠(15 mmol/L)-氢氧化钠(2 mmol/L)混合溶液: 分别称取 $1.590 \mathrm{~g}$ 碳酸钠和 $0.080 \mathrm{~g}$ 氢氧化钠, 于烧杯中溶解, 降至室温后转移至容量瓶中, 定容至 $1000 \mathrm{~mL}$ 。

# 10.3 标准品

磷酸钠 $\mathrm{(Na_3PO_4}$ ，CAS号：7601-54-9)，纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准品。

# 10.4 标准溶液配制

# 13 分析结果的表述

试样中磷酸盐（以磷酸根 $\mathrm{PO}_4^{3-}$ 计）的含量按公式(2)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V}{m \times 1 0 0 0} \times f \tag {2}
$$

式中：

$X$ ——样品中磷酸盐(以磷酸根 $\mathrm{PO_4}^{3-}$ 计)含量，单位为克每千克 $(\mathrm{g/kg})$ ；

$\rho$ ——样液中磷酸盐（以磷酸根 $\mathrm{PO_4}^{3-}$ 计）测定值，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ —样品空白液中磷酸盐(以磷酸根 $\mathrm{PO}_4^{3-}$ 计)测定值，单位为毫克每升 $(\mathrm{mg/L})$ ；

$V$ ——样品提取液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品称样质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数；

$f$ —稀释倍数。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，结果保留3位有效数字。

# 14 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不应超过算术平均值的 $15\%$ 。

# 15 检出限与定量限

取样量 $2\mathrm{g}$ ，按样品前处理方法操作，蔬菜罐头、饮料、糖果、调味糖浆、果冻、油脂制品（仅限植脂末）、可可制品、巧克力和巧克力制品检出限为 $0.06\mathrm{g/kg}$ ，定量限为 $0.12\mathrm{g/kg}$ ；其他食品检出限为 $0.05\mathrm{g/kg}$ ，定量限为 $0.1\mathrm{g/kg}$ 。

# 前言

本标准系首次发布。

# 食品安全国家标准

# 家禽可食性组织中乙氧酰胺苯甲酯残留量的测定 高效液相色谱法

# 1 范围

本标准规定了家禽可食性组织中乙氧酰胺苯甲酯残留量检测的制样和高效液相色谱测定方法。

本标准适用于家禽肌肉、肝脏、肾脏组织中乙氧酰胺苯甲酯残留量的检测。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样中残留的乙氧酰胺苯甲酯用乙腈提取，正己烷脱脂，无水硫酸钠脱水，浓缩，正己烷-丙酮溶解残余物，固相萃取柱净化，甲醇洗脱，高效液相色谱测定，外标法定量。

# 4 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 乙腈（ $\mathrm{CH}_3\mathrm{CN}$ ）：色谱纯。  
4.1.2 甲醇（ $\mathrm{CH}_3\mathrm{OH}$ ）：色谱纯。  
4.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ 。  
4.1.4 丙酮（ $\mathrm{CH}_3\mathrm{COCH}_3$ ）。  
4.1.5 无水硫酸钠（ $\mathrm{Na_2SO_4}$ ）。

# 4.2 标准品

乙氧酰胺苯甲酯（Ethopabate, $\mathrm{C_{12}H_{15}O_4N_5CAS}$ 号:59-06-3）：含量 $\geqslant 98.5\%$

# 4.3 标准溶液的制备

标准储备液：取乙氧酰胺苯甲酯标准品约 $10 \mathrm{mg}$ ，精密称定，于 $100 \mathrm{~mL}$ 量瓶中，加甲醇适量使溶解并稀释至刻度，摇匀。 $4^{\circ} \mathrm{C}$ 保存，有效期1周。

# 4.4 材料

硅酸镁固相萃取柱： $100\mathrm{mg/mL}$ ，或相当者。

# 5 仪器和设备

5.1 高效液相色谱仪：配紫外检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
5.3 匀浆机。  
5.4 离心机： $4000\mathrm{r / min}$   
5.5 旋转蒸发仪。  
5.6 旋涡混合器：3000r/min。  
5.7 振荡器。

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

$-20^{\circ}\mathrm{C}$ 储存备用。

# 7 测定步骤

# 7.1 提取

称取试样 $5 \mathrm{~g}$ (准确至 $\pm 20 \mathrm{mg}$ ), 置 $50 \mathrm{~mL}$ 具塞离心管中, 加乙腈 $15 \mathrm{~mL}$ , 无水硫酸钠 $10 \mathrm{~g}$ , 正己烷 $10 \mathrm{~mL}$ , 涡旋混合 $1 \mathrm{~min}$ , 振荡 $5 \mathrm{~min}$ , $4000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ 。取下层乙腈于鸡心瓶中备用。沉淀中再加入乙腈 $15 \mathrm{~mL}$ , 重新提取一次, 合并两次乙腈提取液于同一鸡心瓶中, $45^{\circ} \mathrm{C}$ 旋转蒸发至近干。加正己烷-丙酮（9:1） $5.0 \mathrm{ml}$ 使溶解, 超声 $30 \mathrm{~s}$ , 摇匀, 转移至 $10 \mathrm{~mL}$ 离心管, $4000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ , 上清液备用。

# 7.2 净化

硅酸镁固相萃取柱用甲醇 $5 \mathrm{~mL}$ 预洗。取上述上清液 $1.0 \mathrm{~mL}$ 过柱，用正己烷 $3 \mathrm{~mL}$ 淋洗，挤干。用甲醇 $1.0 \mathrm{~mL}$ 洗脱，挤干，收集洗脱液，过 $0.45 \mu \mathrm{m}$ 有机滤膜，供高效液相色谱测定。

# 7.3 标准曲线的制备

分别精密量取标准储备液适量，用甲醇稀释成浓度分别为5.0、2.5、1.0、0.5、0.25、0.10、 $0.05\mu \mathrm{g / mL}$ 的标准溶液，供高效液相色谱仪测定。临用前配制。

# 7.4 测定

# 7.4.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 柱（ $250\mathrm{mm}\times 4.6\mathrm{mm}$ ， $5\mu \mathrm{m}$ ），或相当者。  
b）流动相：乙腈-水（30:70）。  
c）流速： $1.0\mathrm{mL / min}$   
d）检测波长： $270\mathrm{nm}$   
e）进样量： $10\mu \mathrm{L}$

# 7.4.2 测定法

取试样溶液和相应浓度的标准溶液，作单点或多点校准，按外标法以峰面积计算。标准溶液和试样溶液中乙氧酰胺苯甲酯的响应值均应在仪器检测的线性范围内。在上述色谱条件下，标准溶液色谱图见附录A。

# 7.5 空白试验

除不加试样外，采用完全相同的测定步骤进行测定。

# 8 结果计算和表述

试样中乙氧酰胺苯甲酯的残留量按（1）计算：

$$
X = \frac {C _ {s} \times A \times V _ {1} \times V _ {3}}{A _ {s} \times V _ {2} \times m} \dots \tag {1}
$$

式中：

$X$ —试样中乙氧酰胺苯甲酯的残留量，单位为微克每千克（ $\mu \mathrm{g} / \mathrm{kg}$ ）；

$\mathrm{C}_{\mathrm{S}}$ —标准溶液中乙氧酰胺苯甲酯的浓度，单位为纳克每毫升（ $\mathrm{ng/mL}$ ）；

A—试样溶液中乙氧酰胺苯甲酯的峰面积；

$\mathrm{A_S}$ —标准溶液中乙氧酰胺苯甲酯的峰面积；

$\mathrm{V}_{1}$ —试样提取液浓缩近干后残余物溶解的总体积，单位为毫升（ $\mathrm{mL}$ ）；

$\mathrm{V}_{2}$ —过硅酸镁固相萃取柱所用备用液体积，单位为毫升（mL）；

$\mathrm{V}_{3}$ ——洗脱液体积，单位为毫升（ $\mathrm{mL}$ ）；

$m$ —试样的质量, 单位为克 (g) ;

计算结果需扣除空白值，以重复性条件下两次独立测定结果的算术平均值表示，保留三位有效数字。

# 9 检测方法的灵敏度、准确度和精密度

# 9.1 灵敏度

本方法在禽肌肉组织中的检测限为 $20\mu \mathrm{g / kg}$ ，定量限为 $50~\mu \mathrm{g / kg}$ ；在禽肝脏和肾脏组织中的检测限为 $50~\mu \mathrm{g / kg}$ ，定量限为 $100~\mu \mathrm{g / kg}$ 。

# 9.2 准确度

本方法在家禽肌肉组织中 $50 \sim 1000 \mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $70 \% \sim 110 \%$ ；在家禽肝脏和肾脏组织中 $100 \sim 3000 \mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $70 \% \sim 110 \%$ 。

# 9.3 精密度

本方法的批内变异系数 $\mathrm{CV} \leqslant 10\%$ ，批间变异系数 $\mathrm{CV} \leqslant 15\%$ 。

# 液相色谱-串联质谱法

# 1 范围

本文件规定了牛可食性组织中盐霉素残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于牛肌肉、肝脏、肾脏和脂肪组织中盐霉素残留量的测定。

# 2 规范性引用文件

下列文件中内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中的药物残留用乙腈提取,提取液过滤膜后用液相色谱- 串联质谱仪测定,基质匹配外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：用前在 $650^{\circ}C$ 灼烧 $4\mathrm{h}$ ，置于干燥器中冷却后备用。  
5.1.2 乙腈（CHCN）色谱纯。  
5.1.3 正己烷 $(\mathrm{C_6H_{12}})$   
5.1.4 甲酸（HCOOH）：色谱纯  
5.1.5 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

# 5.2 溶液的配制

5.2.1 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀  
5.2.2 $0.1\%$ 甲酸乙腈：取甲酸 $1\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$ ，混匀。

# 5.3 标准品

盐霉素钠(CAS号：55721-31-8)，含量 $\geq 95\%$

# 5.4 标准溶液的制备

5.4.1 标准储备液：取标准品适量（相当于有效成分 $10\mathrm{mg}$ ），精密称定，加甲醇适量溶解并稀释定容于 $10\mathrm{mL}$ 容量瓶，配制成浓度为 $1000\mu \mathrm{g} / \mathrm{mL}$ 的标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期6个月。  
5.4.2 标准中间液：准确移取标准储备液 $1\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的标准中间液。一 $18^{\circ}C$ 以下保存，有效期1个月。  
5.4.3 标准工作液：准确移取标准中间液适量，用乙腈逐级稀释成浓度为 $1.5\mu \mathrm{g} / \mathrm{L}, 5.0\mu \mathrm{g} / \mathrm{L}, 20\mu \mathrm{g} / \mathrm{L}, 50\mu \mathrm{g} / \mathrm{L}, 200\mu \mathrm{g} / \mathrm{L}$ 和 $500\mu \mathrm{g} / \mathrm{L}$ 的系列标准工作液。现用现配。

# 5.5 材料

针头式过滤器(通用型滤膜)：尼龙材质，孔径 $0.22\mu \mathrm{m}$ ，或性能相当者。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
6.3 离心管：聚丙烯塑料离心管， $50~\mathrm{mL}$ 。  
6.4 离心机：5000 r/min 或以上。  
6.5 涡旋混合器。  
6.6 振荡器。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c) 取均质后的空白样品，添加适宜浓度的标准溶液，作为空白添加试样。

# 7.2 试样的保存

-18℃以下保存。

# 8 测定步骤

# 8.1 前处理

# 8.1.1 肌肉、肝脏、肾脏

称取试料 $(5 \pm 0.05) \mathrm{g}$ , 于 $50 \mathrm{~mL}$ 离心管中, 加入无水硫酸钠 $5 \mathrm{~g}$ 后再准确加入乙腈 $10 \mathrm{~mL}$ , 涡旋混合 $1 \mathrm{~min}$ , 振荡 $10 \mathrm{~min}, 5000 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ , 收集上清液, 残渣中准确加入乙腈 $10 \mathrm{~mL}$ , 同样步骤提取 1 次, 合并上清液, 过 $0.22 \mu \mathrm{m}$ 滤膜, 供液相色谱-串联质谱测定。

# 8.1.2 脂肪

称取试料 $(5 \pm 0.05) \mathrm{g}$ , 于 $50 \mathrm{~mL}$ 离心管中, 加入正己烷 $10 \mathrm{~mL}$ 后准确加入乙腈 $20 \mathrm{~mL}$ , 涡旋混合 $2 \mathrm{~min}, 5000 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ , 收集乙腈层; 残余物中准确加入乙腈 $20 \mathrm{~mL}$ , 同样步骤萃取 1 次, 合并乙腈溶液, 过 $0.22 \mu \mathrm{m}$ 滤膜, 供液相色谱-串联质谱测定。

# 8.2 基质匹配标准曲线的制备

取5.4.3中系列标准工作液各 $50~\mu \mathrm{L}$ 用空白试样按8.1处理后得到的最终溶液稀释至 $1.0\mathrm{mL}$ ，得浓度为 $0.075\mu \mathrm{g} / \mathrm{L}, 0.25\mu \mathrm{g} / \mathrm{L}, 1.0\mu \mathrm{g} / \mathrm{L}, 2.5\mu \mathrm{g} / \mathrm{L}, 10.0\mu \mathrm{g} / \mathrm{L}$ 和 $25.0\mu \mathrm{g} / \mathrm{L}$ 的系列基质匹配标准工作溶液，供液相色谱-串联质谱测定。以定量离子对峰面积为纵坐标、标准溶液浓度为横坐标，绘制基质匹配标准曲线。求回归方程和相关系数。

# 8.3 测定

# 8.3.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱（ $100\mathrm{mm}\times 2.1\mathrm{mm},1.7\mu \mathrm{m})$ 或相当者；  
b) 流动相: $0.1\%$ 甲酸溶液 $-0.1\%$ 甲酸乙腈 $(3 + 97, V / V)$ ;  
c) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ ;   
e) 柱温: $35^{\circ} \mathrm{C}$ ;   
f) 进样量: $10 \mu \mathrm{L}$ 。

# 8.3.2 质谱参考条件

a) 离子源: 电喷雾离子源;

b）扫描方式：正离子模式；  
c）检测方式：多反应监测（MRM）；  
d) 毛细管电压: $2.0 \mathrm{kV}$ ;  
e) 锥孔电压： $30\mathrm{V}$   
f) RF透镜电压： $0.5\mathrm{V}$   
g）离子源温度： $150~\mathrm{^\circ C}$   
h）脱溶剂气温度： $500^{\circ}\mathrm{C}$   
i） 锥孔气流速： $50~\mathrm{L / h}$   
j）脱溶剂气流速： $1000\mathrm{L / h}$   
k）倍增器电压： $650\mathrm{V}$   
1）二级碰撞气：氩气；  
m）定性离子对、定量离子对、保留时间、锥孔电压和碰撞能量参考值见表1。

# 8.3.3 测定法

# 8.3.3.1 定性测定

在相同测试条件下，试料溶液中盐霉素的保留时间与基质匹配标准工作液中盐霉素的保留时间偏差在 $\pm 0.1\mathrm{min}$ 以内；且检测到的离子的相对丰度应当与浓度相当的基质匹配标准工作溶液相对丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.3.3.2 定量测定

取试料溶液和基质匹配标准工作溶液、作单点或多点校准, 外标法计算。基质匹配标准工作溶液及试料溶液中目标药物的特征离子质量色谱峰峰面积均应在仪器检测的线性范围内。在上述色谱- 质谱条件下, 牛肉基质匹配标准溶液MRM色谱图见附录A。

# 8.4 空白试验

取空白试样，除不加药物外，采用完全相同的操作步骤进行平行操作

# 9 结果计算和表述

试样中待测药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {s}} \times A \times V \times 1 0 0 0}{A _ {\mathrm {s}} \times m \times 1 0 0 0} \dots \tag {1}
$$

式中：

$X$ ——试样中待测药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$C_{\mathrm{s}}$ ——标准溶液中待测药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

A ——试料溶液中待测药物的峰面积；

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准溶液中待测药物的峰面积；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

# 10 检测方法的灵敏度、正确度和精密度

# 10.1 灵敏度

本方法的检出限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1.0~\mu \mathrm{g / kg}$

# 10.2 正确度

本方法盐霉素在 $1.0\mu \mathrm{g} / \mathrm{kg}\sim 50\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $60\% \sim 110\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 10\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

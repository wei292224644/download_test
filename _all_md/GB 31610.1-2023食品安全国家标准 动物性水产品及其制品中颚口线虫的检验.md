# 食品安全国家标准

# 动物性水产品及其制品中颚口线虫的检验

# 1 范围

本标准规定了动物性水产品及其制品中颚口线虫(Gnathostoma)幼虫的形态学和PCR检验方法。  
本标准适用于动物性水产品及其制品中颚口线虫幼虫的检验。

# 2 原理

动物性水产品及其制品中的颚口线虫幼虫主要寄生于淡水鱼类、两栖类等水产品的肌肉等组织内。用胃蛋白酶消化动物宿主组织获得虫体。根据幼虫的形态特征，判断颚口线虫属幼虫。对可疑虫体通过扩增颚口线虫线粒体细胞色素C氧化酶第I亚基 $(cox1)$ 基因片段并测序，进行常见颚口线虫幼虫的鉴定。

# 3 仪器和设备

3.1 生物显微镜： $100 \times \sim 400 \times$ 。  
3.2 体视显微镜： $7.5\times \sim 150\times$   
3.3 PCR扩增仪。  
3.4 凝胶成像系统。  
3.5 电泳仪。  
3.6 恒温培养箱： $37^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 。  
3.7 磁力搅拌器。  
3.8 高速离心机：转速 $\geqslant 12000\mathrm{r / min}$   
3.9 网筛：孔径 $2\mathrm{mm}$ (10目）。  
3.10 锥形量杯： $1000~\mathrm{mL}$ 。  
3.11 微量移液器： $0.2\mu \mathrm{L}\sim 2.5\mu \mathrm{L},1\mu \mathrm{L}\sim 10\mu \mathrm{L},10\mu \mathrm{L}\sim 100\mu \mathrm{L},100\mu \mathrm{L}\sim 1000\mu \mathrm{L}$

# 4 试剂和材料

# 4.1 试剂

4.1.1 盐酸： $36\% \sim 38\%$ HCl溶液。  
4.1.2 生理盐水： $0.85\% \mathrm{NaCl}$ 溶液。  
4.1.3 $1\mathrm{mol} / \mathrm{L}$ Tris-HCl溶液 $(\mathrm{pH}8.0)$ 。  
4.1.4 $0.5\mathrm{mol} / \mathrm{L}$ EDTA溶液 $(\mathrm{pH}8.0)$ 。   
4.1.5 $10\%$ SDS溶液。  
4.1.6 $5\mathrm{mol} / \mathrm{L}$ NaCl溶液。  
4.1.7 $3000\mathrm{U / mg}$ 胃蛋白酶。

4.1.8 $20\mathrm{mg / mL}$ 蛋白酶K。  
4.1.9 苯酚/三氯甲烷/异戊醇 $(25:24:1)$ 。  
4.1.10 $5\mathrm{U} / \mu \mathrm{L}$ 耐热DNA聚合酶。  
4.1.11 $10 \times \mathrm{PCR}$ 缓冲液。  
4.1.12 $25\mathrm{mmol / L}$ $\mathrm{MgCl_2}$ 。  
4.1.13 dNTPs:dATP、dTTP、dCTP、dGTP，每种浓度为 $2.5\mathrm{mmol / L}$   
4.1.14 琼脂糖：电泳级。  
4.1.15 $50\times \mathrm{TAE}$ 缓冲液，使用前用去离子水稀释成 $1\times \mathrm{TAE}$ 缓冲液。  
4.1.16 $1\times \mathrm{TE}$ 溶液（pH8.0）。  
4.1.17 $10\mathrm{mg / mL}$ 溴化乙锭（EB）或其他核酸染料。  
4.1.18 $6\times$ 上样缓冲液。  
4.1.19 $100\mathrm{bp}\sim 2000\mathrm{bp}$ DNA分子量标准。  
4.1.20 引物：浓度为 $10\mu \mathrm{mol} / \mathrm{L}$

正向引物 FH5: $5^{\prime}$ -TTTTTGGGCATCCTGAGGTTTA-3'

反向引物 MCO1B: $5^{\prime}$ -TAAAGAAAGAACATAATGAAAATGAGC- $3^{\prime}$

扩增颚口线虫 $\text{cox1}$ 基因片段长度为 $441\mathrm{bp}$ 。

# 4.2 试剂配制

4.2.1 生理盐水：称取 $8.5\mathrm{gNaCl}$ 溶解于 $900~\mathrm{mL}$ 去离子水，再加去离子水至 $1000~\mathrm{mL}$ 。  
4.2.2 $1\mathrm{mol} / \mathrm{L}$ Tris-HCl溶液：称取 $121.1\mathrm{g}$ Tris(三羟甲基氨基甲烷)溶解于 $800~\mathrm{mL}$ 去离子水，用盐酸调节 $\mathrm{pH}$ 至8.0，加去离子水至 $1000~\mathrm{mL}$ 。  
4.2.3 $0.5\mathrm{mol} / \mathrm{L}$ EDTA溶液：称取 $186.1\mathrm{gNa_2EDTA\cdot 2H_2O}$ （二水合乙二胺四乙酸二钠)溶解于 $800~\mathrm{mL}$ 去离子水，搅拌溶解，用 $\mathrm{NaOH}$ 调节 $\mathrm{pH}$ 至8.0，加去离子水至 $1000~\mathrm{mL}$   
4.2.4 $5\mathrm{mol} / \mathrm{L}$ NaCl溶液：称取 $292.5\mathrm{gNaCl}$ 溶解于 $900~\mathrm{mL}$ 去离子水，再加去离子水至 $1000~\mathrm{mL}$   
4.2.5 胃蛋白酶消化液：取胃蛋白酶 $5\mathrm{g}$ ，溶解于 $900~\mathrm{mL}$ 生理盐水中，加盐酸 $7\mathrm{mL}$ ，混匀，再加生理盐水至 $1000~\mathrm{mL}$ ，临用现配。  
4.2.6 裂解液： $10\%$ SDS 溶液 $100 \mathrm{~mL}, 1 \mathrm{~mol} / \mathrm{L}$ Tris-HCl 溶液 $10 \mathrm{~mL}, 0.5 \mathrm{~mol} / \mathrm{L}$ EDTA 溶液 $200 \mathrm{~mL}, 5 \mathrm{~mol} / \mathrm{L} \mathrm{NaCl}$ 溶液 $20 \mathrm{~mL}$ ，加灭菌去离子水至 $1000 \mathrm{~mL}$ 。  
4.2.7 $1.5\%$ 琼脂糖凝胶：琼脂糖 $1.5\mathrm{g}$ ，加入 $1\times \mathrm{TAE}$ 缓冲液至 $100~\mathrm{mL}$ ，加热至完全融化后冷却至 $60^{\circ}C\sim 70^{\circ}C$ ，加入 $10\mathrm{mg / mL}$ 溴化乙锭 $5\mu \mathrm{L}$ ，混匀，制备凝胶。

# 5 检测方法

# 5.1 形态学方法

# 5.1.1 取样

取动物性水产品或其制品肌肉组织，剪成小块。

# 5.1.2 消化

取5.1.1样品 $200\mathrm{g}$ 置烧杯中，按照样品与胃蛋白酶消化液 $1:5$ 的质量体积比加入消化液混匀，于 $37^{\circ}C \pm 1^{\circ}C$ 恒温培养箱应用磁力搅拌器恒温搅拌消化 $4\mathrm{h} \sim 16\mathrm{h}$ ，使肌肉完全消化。可根据不同水产品种类以及消化时间对胃蛋白酶浓度和使用量进行调整。

# 5.1.3 过滤

消化后的悬液用网筛过滤，并用生理盐水冲洗网筛上的残留物。收集滤液置于锥形量杯内，搅拌后静置沉淀 $15\mathrm{min}\sim 30\mathrm{min}$ 。轻轻倾去上清液，加入适量生理盐水，搅拌后再静置沉淀 $15\mathrm{min}\sim 30\mathrm{min}$ 。重复洗涤3次 $\sim 5$ 次，直至上清液透明为止，沉淀备用。

# 5.1.4 镜检

全部沉淀分次转移至玻璃平皿，在体视显微镜下去除沉淀中的杂质，分离收集沉淀中的虫体或含虫体的包囊（鉴定时撕开包囊，取出虫体）。淡水鱼类、两栖类等动物性水产品及其制品中的颚口线虫为第三期幼虫，虫体呈圆柱形，较粗壮。活虫体呈淡红色、淡黄色或粉红色，半透明，头部和尾部均弯向腹面，部分呈卷曲“G”状。虫体长度因虫种不同而异，为 $0.6 \mathrm{~mm} \sim 11 \mathrm{~mm}$ 。头部近似球形(称头球)，尾部逐渐变细呈锥形。食道、肠管颜色较深。

挑出具备上述特征的虫体在生物显微镜下进一步观察。颚口线虫第三期幼虫头球与虫体躯体部分分界明显。头球前端有两片向外突起的肉质唇围绕口，头球表面排列有3环或4环小钩。体表环列有横纹和小棘，体前部棘明显大而密，体后部棘渐小而疏，尾部光滑圆钝。有1对颈乳突，1个排泄孔。食道呈棒状，分肌质部和腺质部，食道周围有4个肌质管状的颈囊环绕；肠管粗大或细长，肠内充满黄褐色颗粒。颚口线虫第三期幼虫的形态特征见附录A，其模式图见图A.1，实物图见图A.2～图A.13。

# 5.1.5 结果判定

在动物性水产品及其制品中检测到具备5.1.4特征的虫体，可判断为颚口线虫属幼虫。对疑似颚口线虫属幼虫立即用于DNA提取或保存于一 $20^{\circ}\mathrm{C}$ ，用于PCR方法鉴定。

# 5.2 PCR方法

# 5.2.1 DNA提取

取5.1.5判断为疑似颚口线虫幼虫的虫体1条，用生理盐水洗净，放入 $1.5\mathrm{mL}$ 离心管中，加裂解液 $500~\mu \mathrm{L}$ 和蛋白酶K $10~\mu \mathrm{L},55~^\circ \mathrm{C}$ 振荡消化至虫体被完全消化( $1\textrm{h}\sim 3\textrm{h})$ 。在混合液中加入苯酚/三氯甲烷/异戊醇 $(25:24:1)240\mu \mathrm{L}$ ，混匀， $4^{\circ}C12000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。吸上清液(约 $500~\mu \mathrm{L}$ )加入新的 $1.5\mathrm{mL}$ 离心管，加入 $500~\mu \mathrm{L}$ 预冷的无水乙醇（一 $20^{\circ}C)$ ，充分混匀，放入一 $20^{\circ}C$ 沉淀 $1\textrm{h},4^{\circ}C$ $12000\mathrm{r / min}$ 离心 $15\mathrm{min}$ 。沉淀用 $70\%$ 乙醇 $500~\mu \mathrm{L}$ 离心洗涤2次，每次 $4^{\circ}C12000\mathrm{r / min}$ 离心 $3\mathrm{min}$ 。倒干乙醇， $37^{\circ}C$ 干燥后加入 $1\times \mathrm{TE}$ 溶液 $100~\mu \mathrm{L}$ 溶解DNA，立即用于检测或一 $20^{\circ}C$ 保存备用。

注：根据实验室实际情况，可使用经验证的商品化组织DNA提取试剂盒提取DNA。

# 5.2.2 PCR反应体系

在PCR管中依次加入 $10\times \mathrm{PCR}$ 缓冲液 $5.0\mu \mathrm{L},\mathrm{MgCl}_25.0\mu \mathrm{L},$ dNTPs $4.0\mu \mathrm{L}$ 、正向引物和反向引物各 $1.0\mu \mathrm{L}$ 、耐热DNA聚合酶 $0.4\mu \mathrm{L}$ DNA模板 $2.0\mu \mathrm{L}$ ，加灭菌去离子水至总体积 $50~\mu \mathrm{L}$ 。每次试验需设阳性对照和空白对照。阳性对照用颚口线虫DNA或含有目标基因序列的质控品，空白对照用灭菌去离子水做模板。

# 5.2.3 PCR反应条件

$94^{\circ} \mathrm{C}$ 预变性 $3 \mathrm{~min}; 94^{\circ} \mathrm{C}$ 变性 $30 \mathrm{~s}, 58^{\circ} \mathrm{C}$ 退火 $30 \mathrm{~s}, 72^{\circ} \mathrm{C}$ 延伸 $45 \mathrm{~s}, 35$ 个循环； $72^{\circ} \mathrm{C}$ 延伸 $7 \mathrm{~min}; 4^{\circ} \mathrm{C}$ 保存。

# 5.2.4 电泳

取PCR扩增产物 $10\mu \mathrm{L}$ 与 $6\times$ 上样缓冲液 $2\mu \mathrm{L}$ 混合，加样于 $1.5\%$ 琼脂糖凝胶中，其中一孔加入DNA分子量标准。在 $1\times \mathrm{TAE}$ 电泳缓冲液， $5\mathrm{V / cm}$ 恒压电泳 $30\mathrm{min}\sim 40\mathrm{min}$ ，用凝胶成像系统观察和记录结果。

# 5.2.5 PCR结果判定

阳性对照出现预期大小的条带(441bp)，空白对照无条带，待测样品扩增出预期大小的条带，可判定PCR结果为阳性；无扩增条带或未扩增出预期大小的条带均判为PCR结果阴性。

取PCR结果为阳性的PCR产物进行基因序列双向测序，将测序结果与颚口线虫参考序列(见附录B)进行同源性比对。

# 6 结果报告

6.1 形态学方法检出颚口线虫属幼虫，报告检出颚口线虫幼虫。  
6.2 形态学方法检出疑似颚口线虫属幼虫、PCR结果为阳性且扩增片段基因序列与颚口线虫任何一条参考序列同源性 $\geq 96\%$ ，报告检出颚口线虫幼虫。  
6.3 形态学方法未检出颚口线虫属幼虫、或PCR结果为阴性、或扩增片段基因序列与颚口线虫参考序列同源性 $< 96\%$ ，均报告未检出颚口线虫幼虫。

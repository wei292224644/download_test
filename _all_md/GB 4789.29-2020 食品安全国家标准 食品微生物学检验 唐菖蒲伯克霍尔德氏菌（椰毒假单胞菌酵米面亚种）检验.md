# 1 范围

本标准规定了食品中唐菖蒲伯克霍尔德氏菌(椰毒假单胞菌酵米面亚种)[Burkholderia gladioli(Pseudomonas coccovenans subsp. farinofermentans)]的检验方法。

本标准适用于食品中唐菖蒲伯克霍尔德氏菌(椰毒假单胞菌酵米面亚种)的检验。

# 5 操作步骤

# 5.1 样品处理

无菌操作称取 $25\mathrm{g(mL)}$ 样品，置入盛有 $225~\mathrm{mL}$ GVC增菌液的无菌均质袋中(鲜银耳样品取 $1\mathrm{g}$ ，用剪刀剪碎，加入盛有 $20~\mathrm{mL}$ GVC增菌液的无菌均质袋中)，用拍击式均质器拍打 $1\mathrm{min}\sim 2\mathrm{min}$ ；或置入盛有 $225~\mathrm{mL}$ GVC增菌液的无菌均质杯中，以 $8000\mathrm{r / min}\sim 10000\mathrm{r / min}$ 均质 $1\mathrm{min}\sim 2\mathrm{min}$ ；若样品为液态，振荡混匀。

# 5.2 增菌

将上述样品增菌液置 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $20\mathrm{h} \sim 24\mathrm{h}$ 。

# 5.4 初筛试验

自选择性琼脂平板上分别挑取5个以上典型或可疑菌落（低于5个全选），分区划线接种于卵黄琼脂平板， $36\mathrm{^\circ C}\pm 1\mathrm{^\circ C}$ 培养。挑取培养 $18\mathrm{h}\sim 24\mathrm{h}$ 的菌落进行革兰氏染色及氧化酶试验。对于革兰氏染色阴性、氧化酶试验阴性的菌继续培养至 $48\mathrm{h}\pm 2\mathrm{h}$ ，对于卵磷脂酶阳性，带有虹彩环的单个菌落，接种PDA平板， $36\mathrm{^\circ C}\pm 1\mathrm{^\circ C}$ 培养 $24\mathrm{h}\pm 2\mathrm{h}$

# 5.6 血清学分型（可选择项）

# 5.6.1 菌体抗原的制备

将PDA平板 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h} \pm 2\mathrm{h}$ 的培养物用无菌生理盐水洗下，沸水浴 $(100^{\circ}\mathrm{C})2\mathrm{h}$ ， $3000\mathrm{r / min}$ $10\mathrm{min}$ 离心弃上清液，再用无菌生理盐水稀释至 $5 \times 10^{8}$ CFU/mL～ $1 \times 10^{9}$ CFU/mL菌悬液，作为凝集试验用抗原。

# 5.6.2 菌体抗原的鉴定

用多价血清做玻片凝集试验，同时用生理盐水做对照。与多价血清凝集者，依次用O-Ⅲ、O-IV、O-V、O-VI、O-VII、O-VIII因子血清做试管凝集试验。根据试验结果，判定菌体抗原型。在生理盐水中自凝者不能分型。生化特征符合，但不能与以上血清凝集者，需保留菌株做进一步鉴定。

# 5.7 毒性试验

# 5.7.1 产毒培养

将初步鉴定为唐菖蒲伯克霍尔德氏菌（椰毒假单胞菌酵米面亚种)的菌株接种PDA平板， $36^{\circ}\mathrm{C}\pm 1$ $\mathrm{^\circ C}$ ，培养 $24\mathrm{h}\pm 2\mathrm{h}$ ，用灭菌接种环刮取适量菌苔，加到 $3\mathrm{mL}$ 无菌生理盐水的试管中，配成1麦氏浓度(McFarland，MCF)的菌悬液(约为 $10^{8}\mathrm{CFU / mL})$ ，用无菌吸管吸取 $0.5\mathrm{mL}$ ，滴在铺好无菌玻璃纸的直径 $150~\mathrm{mm}$ 马铃薯葡萄糖半固体平板上，用无菌L棒涂布均匀， $26^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 培养5d。取下带菌的玻璃纸，将半固体平板置于 $100^{\circ}\mathrm{C}$ 流动蒸汽灭菌 $30\mathrm{min}$ 。室温冷却后，置一 $20^{\circ}\mathrm{C}\sim -30^{\circ}\mathrm{C}$ 冰箱过夜。将冰冻好的半固体平板置室温融化，用无菌吸管吸出冻融液，经滤纸过滤至无菌试管或锥形瓶中（此为毒素粗提液）， $4^{\circ}\mathrm{C}$ 避光保存。

同时，将未接种菌苔、铺好无菌玻璃纸的直径 $150\mathrm{mm}$ 马铃薯葡萄糖半固体平板作为阴性对照，按照同样的试验方法制备阴性对照粗提液， $4^{\circ}\mathrm{C}$ 避光保存。

# 5.7.2 毒力测定

取毒素粗提液或经 $100^{\circ}\mathrm{C}$ 水浴蒸发后的5倍～10倍浓缩液，灌胃小鼠3只，每只 $0.5\mathrm{mL}$ ，观察至 $7\mathrm{d}_{\circ}$ 若菌株产生米酵菌酸，小鼠在灌胃后 $20\mathrm{min}\sim 24\mathrm{h}$ 内发病。主要症状为竖毛、萎糜不振、躁动，继而行步蹒跚、肢体麻痹、瘫软、抽搐，呈角弓反张状，呼吸急促、死亡。

取阴性对照粗提液或经 $100^{\circ}\mathrm{C}$ 水浴蒸发后的5倍 $\sim 10$ 倍浓缩液，灌胃小鼠3只，每只 $0.5\mathrm{mL}$ ，观察至7d。小鼠应健康存活。

# 5.7.3 米酵菌酸测定

毒力测定实验阳性时，分别取毒素粗提液，阴性对照粗提液，按照GB5009.189执行。

# 6 结果报告

生化试验符合且毒性试验阳性，报告检出唐菖蒲伯克霍尔德氏菌(椰毒假单胞菌酵米面亚种)；生化试验和毒性试验有一项不符合，报告未检出唐菖蒲伯克霍尔德氏菌(椰毒假单胞菌酵米面亚种)。

挑取培养物接种， $37^{\circ} \mathrm{C}$ 培养 $18 \mathrm{~h} \sim 24 \mathrm{~h}$ 。

# A.22.4 结果

牛乳中含有丰富的糖类及蛋白质，其中主要是乳糖和酪蛋白，一般细菌均能在其中生长。由于各种细菌对这些物质的分解能力不同，故可观察数种不同的反应。

a）产酸：因发酵乳糖而产酸，使指示剂变为粉红色。  
b）凝固：因产酸过多而使牛乳中的酪蛋白凝固。  
c）胨化：经凝固的酪蛋白继续水解为蛋白胨，此时牛乳培养基的上层液体变清，底部可留有未被完全胨化的酪蛋白。  
d）产碱：未能使乳糖发酵，因分解含氮物质，生成胺及氨，培养基变为碱性，指示剂变为蓝紫色。

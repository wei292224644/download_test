# 食品安全国家标准

# 食品添加剂 $\gamma$ -癸内酯

# 1 范围

本标准适用于食品添加剂 $\gamma$ -癸内酯。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

$\gamma$ -癸内酯

# 2.2 分子式

$\mathrm{C_{10}H_{18}O_2}$

![](images/526383dd9c2eed03b585557b9a33dd32d95809e8163360a37dc3c24161696b76.jpg)

170.25（按2007年国际相对原子质量）

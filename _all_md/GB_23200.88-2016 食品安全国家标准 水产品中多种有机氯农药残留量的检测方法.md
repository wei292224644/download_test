# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN0598—1996《出口水产品中多种有机氯农药残留量检验方法》。与SN0598—1996相比，主要变化如下：

——标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“出口水产品”改为“水产品”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN 0598—1996。

# 食品安全国家标准

# 水产品中多种有机氯农药残留量的检测方法

# 1 范围

本标准规定了水产品中的六六六(BHC)及异构体、六氯苯(HCB)、七氯、环氧七氯、艾氏剂、狄氏剂、异狄氏剂、滴滴涕(DDT)及异构体和类似物(DDD,DDE)残留量检验的抽样、制样和气相色谱测定方法。

本标准适用于出口鳄鱼中14种有机氯农药 $(\alpha -\mathrm{BHC},\beta -\mathrm{BHC},\gamma -\mathrm{BHC},\delta -\mathrm{BHC}$ 、六氯苯、七氯、环氧七氯、艾氏剂、狄氏剂、异狄氏剂、o,p'-DDT、p,p'-DDT、p,p'-DDD、p,p'-DDE)残留量的检验，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样经与无水硫酸钠一起研磨干燥后，用丙酮一石油醚提取农药残留，提取液经弗罗里硅土柱净化，净化后样液用配有电子俘获检测器的气相色谱仪测定，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ ：重蒸馏。  
4.1.2 石油醚：沸程 $60^{\circ}\mathrm{C} \sim 90^{\circ}\mathrm{C}$ 经氧化铝柱净化后用全玻璃蒸馏器蒸馏，收集 $60^{\circ}\mathrm{C} \sim 90^{\circ}\mathrm{C}$ 馏分。  
4.1.3 乙醚 $(\mathrm{C_4H_{10}O})$ ：重蒸馏。  
4.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，储于密封容器中备用。

# 4.2 溶液配制

乙醚一石油醚淋洗溶液 $(15 + 85)$ ：取 $150~\mathrm{mL}$ 乙醚，加入 $850~\mathrm{mL}$ 石油醚，摇匀备用。

# 4.3 标准品

有机氯农药标准品 $(\alpha -\mathrm{BHC},\beta -\mathrm{BHC},\gamma -\mathrm{BHC},\delta -\mathrm{BHC}$ 、六氯苯、七氯、环氧七氯、艾氏剂、狄氏剂、异狄氏剂、o,p'-DDT、p,p'-DDT、p,p'-DDD、p,p'-DDE)：纯度 $\geqslant 99\%$ 。

# 4.4 标准溶液配制

14种有机氯农药标准溶液：准确称取适量的每种农药标准品，分别用少量苯溶解，然后用石油醚配成浓度各为 $0.100\mathrm{mg / mL}$ 的标准储备溶液。根据需要再以石油醚配制成适用浓度的混合标准工作溶液。保存于 $4^{\circ}C$ 冰箱内。

# 4.5 材料

4.5.1 氧化铝：层析用，中性， $74\mu \mathrm{m}\sim 150\mu \mathrm{m},800^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，冷却至室温后储于密封容器中备用。使用前应在 $130^{\circ}\mathrm{C}$ 条件下干燥 $2\mathrm{h}$ 。  
4.5.2 弗罗里硅土: $150 \mu \mathrm{m} \sim 250 \mu \mathrm{m}, 650^{\circ} \mathrm{C}$ 灼烧 $4 \mathrm{~h}$ , 储于密封容器中备用。使用前应在 $130^{\circ} \mathrm{C}$ 条件下干燥 $1 \mathrm{~h}$ 。

每批弗罗里硅土用前应做淋洗曲线。

# 5 仪器与设备

5.1 气相色谱仪：配电子捕获检测器（ECD）。  
5.2 氧化铝净化柱: $300 \mathrm{~mm} \times 20 \mathrm{~mm}$ (内径) 玻璃柱, 装入氧化铝 $40 \mathrm{~g}$ , 上端装入 $10 \mathrm{~g}$ 无水硫酸钠干法装柱, 使用前用 $40 \mathrm{~mL}$ 石油醚淋洗。  
5.3 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.4 索氏提取器： $250~\mathrm{mL}$   
5.5 全玻璃重蒸馏装置。  
5.6 玻璃研钵：口径 $11.5\mathrm{cm}$ 。  
5.7 旋转蒸发器或氮气流浓缩装置：配有 $250~\mathrm{mL}$ 蒸发瓶。  
5.8 微量注射器： $10~\mu \mathrm{L}$   
5.9 脱脂棉：经过丙酮-石油醚(2+8)混合液抽提6h处理过。  
5.10 绞肉机。

# 6 试样制备与保存

将抽取的样品充分搅碎和混匀。用四分法缩分出 $1 \mathrm{~kg}$ , 均分为 2 份, 分别装入洁净容器内, 作为试样。密封, 并标明标记。将试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

在抽样和制样的操作过程中，必须防止样品受到污染或发生残留物含量的变化。

以上样品取样部位按GB2763的规定执行。

# 7 分析步骤

# 7.1 提取

称取试样 $10.0\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）于研钵中，加 $15\mathrm{g}$ 无水硫酸钠研磨几分钟，将试样制成干松粉末。装入滤纸筒内。放入索氏提取器中。在提取器的瓶中加入 $100\mathrm{mL}$ 丙酮一石油醚 $(2 + 8)$ 混合液，在水浴上提取 $6\mathrm{h}$ （回流速度每小时10次～12次）。将提取液减压或氮气流浓缩至 $5\mathrm{mL}$ 。

# 7.2 净化

将提取液全部移入弗罗里硅土净化柱中。弃去流出液。注入 $200 \mathrm{~mL}$ 乙醚一石油醚淋洗液进行洗脱。开始时, 取部分乙醚一石油醚混合液反复清洗提取瓶, 并把洗液注入净化柱中。洗脱流速为 $2 \mathrm{~mL} / \mathrm{min} \sim 3 \mathrm{~mL} / \mathrm{min}$ , 收集流出液于 $250 \mathrm{~mL}$ 蒸发瓶中。在减压或氮气流中浓缩并定容至 $10 \mathrm{~mL}$ , 供气相色谱测定。

# 7.3 测定

# 7.3.1 气相色谱参考条件

a）色谱柱：SGE毛细管柱(或等效的色谱柱)， $25\mathrm{m}\times 0.53\mathrm{mm}$ （内径），膜厚 $0.15\mu \mathrm{m}$ 。固定相：HT5(非极性)键合相；  
b）载气：氮气（纯度 $\geq 99.99\%$ ， $10~\mathrm{mL / min}$   
c) 助气：氮气（纯度 $\geq 99.99\%$ ）， $40~\mathrm{mL} / \mathrm{min}$ ；

d）柱温：程序升温控制为 $\frac{100^{\circ}\mathrm{C}}{(2\mathrm{min})}\xrightarrow{5^{\circ}\mathrm{C}/\mathrm{min}}140^{\circ}\mathrm{C}\xrightarrow{10^{\circ}\mathrm{C}/\mathrm{min}}200^{\circ}\mathrm{C}\xrightarrow{15^{\circ}\mathrm{C}/\mathrm{min}}230^{\circ}\mathrm{C}$   
e）进样口温度： $200^{\circ}\mathrm{C}$   
f）检测器温度： $300^{\circ}\mathrm{C}$   
g）进样方式：柱头进样方式。

# 7.3.2 气相色谱测定

根据样液中有机氯农药种类和含量情况，选定峰高相近的相应标准工作混合液。标准工作混合液和样液中各有机氯农药响应值均应在仪器检测线性范围内。对标准工作混合液和样液等体积穿插进样测定。在上述色谱条件下，各有机氯农药出峰顺序和保留时间见表1。14种有机氯农药标准品的色谱图参见图A.1。

# 7.4 空白试验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中有机氯农药的含量。

$$
X _ {i} = \frac {h _ {i} \times c \times V}{h _ {i s} \times m}. \tag {1}
$$

式中：

$X_{i}$ ——试样中有机氯农药残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——标准工作溶液中各有机氯农药的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$h_i$ ——样液中各有机氯农药的峰高，单位为毫米（ $\mathrm{mm}$ ）；

$h_{is}$ ——标准工作溶液中各有机氯农药的峰高，单位为毫米 $(\mathrm{mm})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——称取试样量，单位为克 $(\mathrm{g})$ 。

计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录B的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录 C 的要求。

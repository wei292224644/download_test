# 前言

本标准代替GB10343—2008《食用酒精》中的部分指标，GB10343—2008《食用酒精》中涉及到本标准的指标以本标准为准。

# 食品安全国家标准

# 食用酒精

# 1 范围

本标准适用于食用酒精。

# 2 术语和定义

# 2.1 食用酒精

以谷物、薯类、糖蜜或其他可食用农作物为主要原料，经发酵、蒸馏精制而成的，供食品工业使用的含水酒精。

# 3 技术要求

# 3.1 原料要求

原料应符合相应的产品标准和有关规定。

# 4 其他

# 4.1 包装

4.1.1 装运食用酒精应使用专用的罐、槽车和不锈钢桶，不应使用铝桶或镀锌容器包装，不应使用未做抗静电处理的容器。包装前，应对所用容器进行严格的安全、卫生检查。  
4.1.2 灌装后的罐、槽车应加铅封。使用单位收货时，应检查铅封是否完好。  
4.1.3 包装物应体外清洁，标注内容清晰可见，标签粘贴牢固。

# 4.2 运输

4.2.1 运输工具应清洁、卫生，不应与有毒、有害、有腐蚀性或有异味的物品混装混运。  
4.2.2 搬运时应轻装轻卸，严禁扔摔、撞击和剧烈震荡，应远离热源和火种。  
4.2.3 运输过程应防火、防爆、防静电、防雷电，严禁曝晒。

# 4.3 贮存

4.3.1 产品不应与有毒、有害、有腐蚀性或有异味的物品混合存放。  
4.3.2 产品应贮存于阴凉、干燥、通风的环境中，应有防高温、火种、静电、雷电的设施。在贮存区域应有醒目的“严禁火种”的警示牌。

# 附录A

# 食用酒精中醛的测定

# A.1 碘量法

# A.1.1 原理

亚硫酸氢钠与醛发生加成反应生成 $\alpha$ -羟基磺酸钠，反应式为：

$$
\begin{array}{c}\mathrm {O}\\\parallel\\\mathrm {R} - \mathrm {C} - \mathrm {H} + \mathrm {N a H S O} _ {3} \rightarrow \mathrm {R} - \mathrm {C} - \mathrm {O H}\\\mid\\\mathrm {S O} _ {3} \mathrm {N a}\end{array}
$$

用碘氧化过量的亚硫酸氢钠，反应式为：

$$
\mathrm {N a H S O} _ {3} + \mathrm {I} _ {2} + \mathrm {H} _ {2} \mathrm {O} \rightarrow \mathrm {N a H S O} _ {4} + 2 \mathrm {H I}
$$

加过量的 $\mathrm{NaHCO_3}$ ，使加成物分解，醛重新游离出来，反应式为：

$$
\begin{array}{c}\mathrm {H}\\\mid\\\mathrm {R} - \mathrm {C} - \mathrm {O H} + 2 \mathrm {N a H C O} _ {3} \rightarrow \mathrm {R C H O} + \mathrm {N a H S O} _ {3} + \mathrm {N a} _ {2} \mathrm {C O} _ {3} + \mathrm {C O} _ {2} \uparrow + \mathrm {H} _ {2} \mathrm {O}\\\mid\\\mathrm {S O} _ {3} \mathrm {N a}\end{array}
$$

用碘标准溶液滴定分解释放出来的亚硫酸氢钠。

# A.1.2 试剂和溶液

A.1.2.1 亚硫酸氢钠溶液 $(12\mathrm{g} / \mathrm{L})$

A.1.2.2 盐酸溶液 $[c(\mathrm{HCl}) = 0.1\mathrm{mol / L}]$ ：按GB/T601配制。

A.1.2.3 碘标准溶液 $\left[c\left(\frac{1}{2}\mathrm{I}_2\right) = 0.1\mathrm{mol} / \mathrm{L}\right]$ ：按GB/T601配制与标定。

A.1.2.4 碘标准滴定溶液 $\left[c\left(\frac{1}{2}\mathrm{I}_2\right) = 0.01\mathrm{mol} / \mathrm{L}\right]$ ：使用时将 $0.1\mathrm{mol} / \mathrm{L}$ 碘标准溶液准确稀释10倍。

A.1.2.5 淀粉指示液 $(10\mathrm{g / L})$ ：按GB/T603配制。

A.1.2.6 碳酸氢钠溶液 $[c(\mathrm{NaHCO}_3) = 1\mathrm{mol} / \mathrm{L}]$

# A.1.3 分析步骤

吸取试样 $15.0\mathrm{mL}$ 于 $250~\mathrm{mL}$ 碘量瓶中，加 $15~\mathrm{mL}$ 水、 $15~\mathrm{mL}$ 亚硫酸氢钠溶液、 $7\mathrm{mL}$ 盐酸溶液，摇匀，于暗处放置 $1\textrm{h}$ ，取出，用 $50~\mathrm{mL}$ 水冲洗瓶塞，以碘标准溶液滴定，接近终点时，加淀粉指示液 $0.5\mathrm{mL}$ ，改用碘标准滴定溶液滴定至淡蓝紫色出现(不计数)。加 $20~\mathrm{mL}$ 碳酸氢钠溶液，微开瓶塞，摇荡 $0.5\mathrm{min}$ (呈无色)，用碘标准滴定溶液继续滴定至蓝紫色为其终点。同时做空白试验。

# A.1.4 结果计算

试样中的醛含量按式(A.1)计算：

$$
X = \frac {\left(V _ {1} - V _ {2}\right) \times c \times 0 . 0 2 2}{1 5} \times 1 0 ^ {6} \quad \dots \tag {A.1}
$$

式中：

$X$ ——试样中的醛含量(以乙醛计)，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V_{1}$ ——试样消耗碘标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——空白消耗碘标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——碘标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

0.022——与 $1.00 \mathrm{~mL}$ 碘标准使用溶液 $\left[c\left(\frac{1}{2} \mathrm{I}_{2}\right)=1.000 \mathrm{~mol} / \mathrm{L}\right]$ 相当的以克表示的乙醛的质量。

结果保留至整数。

# A.1.5 精密度

在重复性条件下获得的两次独立测定值之差，若醛含量 $>5\mathrm{mg / L}$ ，不得超过平均值的 $5\%$ ；若醛含量 $\leqslant 5\mathrm{mg / L}$ ，不得超过平均值的 $13\%$ 。

# A.2 比色法

# A.2.1 原理

醛和亚硫酸品红作用时，发生加成反应，经分子重排后，失去亚硫酸，生成具有醌形结构的紫红色物质，其颜色的深浅与醛含量成正比。

# A.2.2 试剂和溶液

A.2.2.1 亚硫酸氢钠溶液 $(53\%)$ ：称取 $53.0\mathrm{g}$ 亚硫酸氢钠 $\mathrm{(NaHSO_3)}$ ，溶于 $100~\mathrm{mL}$ 水中。

A.2.2.2 硫酸：密度为 $1.84\mathrm{g / cm^3}$

A.2.2.3 碱性品红-亚硫酸显色剂：称取 $0.075\mathrm{g}$ 碱性品红溶于少量 $80^{\circ}C$ 水中，冷却，加水稀释至约 $75~\mathrm{mL}$ ，移入1L棕色细口瓶内，加 $50~\mathrm{mL}$ 新配制的亚硫酸氢钠溶液，加 $500~\mathrm{mL}$ 水和 $7.5~\mathrm{mL}$ 硫酸，摇匀，放置 $10\mathrm{h}\sim 12\mathrm{h}$ 至溶液褪色并具有强烈的二氧化硫气味，置于冰箱中保存。

A.2.2.4 醛标准溶液 $(1\mathrm{g / L})$ ：准确称取乙醛氨 $0.1386\mathrm{g}$ （按乙醛：乙醛氨 $= 1:1.386)$ 迅速溶于 $10^{\circ}C$ 左右的基准乙醇(无醛酒精)中，并定容至 $100~\mathrm{mL}$ 。移入棕色试剂瓶内，贮存于冰箱中。

A.2.2.5 醛标准使用溶液：吸取乙醛标准溶液 $0.30\mathrm{mL}, 0.50\mathrm{mL}, 0.80\mathrm{mL}, 1.00\mathrm{mL}, 1.50\mathrm{mL}, 2.00\mathrm{mL}, 2.50\mathrm{mL}$ 和 $3.00\mathrm{mL}$ ，分别置于已有部分基准乙醇(无醛酒精)的 $100\mathrm{mL}$ 容量瓶中，并用基准乙醇稀释至刻度。即醛含量分别为 $3\mathrm{mg/L}, 5\mathrm{mg/L}, 8\mathrm{mg/L}, 10\mathrm{mg/L}, 15\mathrm{mg/L}, 20\mathrm{mg/L}, 25\mathrm{mg/L}$ 和 $30\mathrm{mg/L}$ 。

# A.2.3 分析步骤

吸取与试样含量相近的限量指标的醛标准使用溶液及试样各 $2\mathrm{mL}$ ，分别注入 $25~\mathrm{mL}$ 比色管中，各加 $5\mathrm{mL}$ 水、 $2\mathrm{mL}$ 显色剂，加塞摇匀，放置 $20\mathrm{min}$ （室温低于 $20^{\circ}C$ 时，需放入 $20^{\circ}C$ 水浴中显色），取出比色。用 $2\mathrm{cm}$ 比色皿，在波长 $555~\mathrm{nm}$ 处，以水调零，测定其吸光度。

# A.2.4 结果计算

试样中的醛含量按式(A.2)计算：

$$
X = \frac {A _ {x}}{A} \times c \quad \dots \tag {A.2}
$$

式中：

$X$ ——试样中的醛含量（以乙醛计），单位为毫克每升 $(\mathrm{mg / L})$ ；

$A_{x}$ ——试样的吸光度；

A 醛标准使用溶液的吸光度；

$c$ ——标准使用溶液的醛含量，单位为毫克每升 $(\mathrm{mg / L})$ 。

结果保留至整数。

# A.2.5 精密度

在重复性条件下获得的两次独立测定值之差，若醛含量 $>5\mathrm{mg / L}$ ，不得超过平均值的 $5\%$ ；若醛含量 $\leqslant 5\mathrm{mg / L}$ ，不得超过 $10\%$ 。

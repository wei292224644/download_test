# 食品安全国家标准

# 食品中果聚糖的测定

# 1 范围

本标准规定了食品中果聚糖含量的离子色谱法测定方法。

本标准适用于乳及乳制品、婴幼儿配方食品、婴幼儿谷类辅助食品、固体饮料、配制酒中单独添加的低聚果糖、多聚果糖或菊粉含量的测定。

# 2 原理

试样经热水浸提，样液中的蔗糖经蔗糖酶水解成葡萄糖和果糖，葡萄糖和果糖经硼氢化钠还原成相应的糖醇，多余的硼氢化钠用乙酸中和。样液中的果聚糖经过果聚糖酶水解成果糖和葡萄糖，经离子色谱-脉冲安培检测器测定果糖含量，通过换算系数，折算得到果聚糖的含量。

# 3 试剂和材料

# 3.1 试剂

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

3.1.2 马来酸 $\mathrm{(C_4H_4O_4)}$ 。  
3.1.3 蔗糖酶：来源于酵母，酶活力 $\geq 300\mathrm{U}$   
3.1.4 硼氢化钠 $(\mathrm{NaBH}_4)$ 。  
3.1.5 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.6 三水乙酸钠 $\mathrm{(CH_3COONa\cdot 3H_2O)}$ 。  
3.1.7 果聚糖酶：来源于黑曲霉，酶活力 $\geqslant 10000\mathrm{U}$   
3.1.8 $50\%$ 氢氧化钠溶液（NaOH）：色谱纯。  
3.1.9 无水乙酸钠 $\mathrm{(CH_3COONa)}$ ：纯度 $\geqslant 99.0\%$   
3.1.10 氮气 $(\mathrm{N}_2)$ ：纯度 $\geq 99.9\%$

3.2.1 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $40\mathrm{g}$ 氢氧化钠(精确至 $0.01\mathrm{g}$ )，溶于水并稀释至 $1000\mathrm{mL}$ ，室温下可放置2个月。  
3.2.2 马来酸钠缓冲溶液 $(100\mathrm{mmol / L},\mathrm{pH}6.5)$ ：称取 $1.16\mathrm{g}$ 马来酸(精确至 $0.01\mathrm{g})$ 于 $150~\mathrm{mL}$ 烧杯中，加入约 $70~\mathrm{mL}$ 水溶解，用 $1\mathrm{mol / L}$ 氢氧化钠溶液调节 $\mathrm{pH}6.5$ ，用水稀释至 $100~\mathrm{mL}$ 。 $4^{\circ}C$ 保存，可放置3个月。  
3.2.3 蔗糖酶溶液 $(4.5\mathrm{U} / \mathrm{mL})$ ：将蔗糖酶(活力为 $300\mathrm{U}$ )溶解于 $66~\mathrm{mL}$ 马来酸钠缓冲溶液，分装到 $2\mathrm{mL}$ 的离心管中， $-20^{\circ}\mathrm{C}$ 保存，可放置6个月。使用前需测定酶活力。  
3.2.4 氢氧化钠溶液(50 mmol/L)：称取 $2\mathrm{g}$ 氢氧化钠（精确至 $0.01\mathrm{g}$ ），溶于水并稀释至 $1000\mathrm{mL}$ ，室

温下可放置2个月。

3.2.6 乙酸溶液 $(200\mathrm{mmol / L})$ ：吸取 $0.6\mathrm{mL}$ 冰乙酸，用水稀释至 $50~\mathrm{mL},4^{\circ}C$ 保存，可放置2个月。  
3.2.7 乙酸钠溶液 $(200\mathrm{mmol / L})$ ：称取 $1.36\mathrm{g}$ 三水乙酸钠(精确至 $0.01\mathrm{g}$ )，溶于水并稀释至 $50~\mathrm{mL}$ ， $4^{\circ}\mathrm{C}$ 保存，可放置2个月。  
3.2.8 乙酸钠缓冲溶液 $(\mathrm{pH}4.5)$ ：吸取 $14~\mathrm{mL}$ 乙酸溶液和 $11~\mathrm{mL}$ 乙酸钠溶液混合，并用水稀释至 $50~\mathrm{mL}$ ，用时现配。

3.2.10 氢氧化钠溶液 $(200\mathrm{mmol / L})$ : 取 $10.4\mathrm{mL}50\%$ 氢氧化钠溶液, 用水稀释至 $1000\mathrm{mL}$ , 通入氮气保护, 缓慢摇匀, 室温下可放置 $7\mathrm{d}$ 。  
3.2.11 氢氧化钠 $(150\mathrm{mmol / L})$ 乙酸钠 $(500\mathrm{mmol / L})$ 混合溶液: 称取 $41\mathrm{g}$ 无水乙酸钠 (精确至 $0.01\mathrm{g}$ ), 用约 $500\mathrm{mL}$ 水溶解, $0.22\mu \mathrm{m}$ 滤膜过滤, 脱气 $10\mathrm{min}$ , 加入 $7.8\mathrm{mL}50\%$ 氢氧化钠溶液, 并用水稀释至 $1000\mathrm{mL}$ , 通入氮气保护, 缓慢摇匀, 室温下可放置 $7\mathrm{d}$ 。

# 5.1.2 提取

准确称取 $1\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，至少含有果聚糖 $5\mathrm{mg}$ ）试样于 $150~\mathrm{mL}$ 锥形瓶中，加入 $80^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 热水约 $50~\mathrm{mL}$ ，置于 $80^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温水浴摇床中， $150~\mathrm{r / min}$ 振摇 $15\mathrm{min}$ 后，取出，冷却至室温，转移至 $100~\mathrm{mL}$ 容量瓶中，用水分三次冲洗锥形瓶，定容，摇匀，溶液经滤纸过滤或者离心，滤液或上清液依据标准曲线的线性范围确定稀释倍数，备用。

取上述备用样液 $200\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 具塞玻璃试管中，估算样液中可能的蔗糖含量，按照每毫克蔗糖加入 $300\mu \mathrm{L}$ 蔗糖酶溶液，旋涡振荡混匀，置于 $40^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 恒温水浴摇床中， $150~\mathrm{r / min}$ 振摇 $60\mathrm{min}$ 后，加入 $300\mu \mathrm{L}$ 硼氢化钠溶液，旋涡振荡混匀，置于 $40^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 恒温水浴摇床中， $150~\mathrm{r / min}$ 振摇 $30\mathrm{min}$ 后，取出，冷却至室温。加入 $750\mu \mathrm{L}$ 乙酸溶液，静置 $10\mathrm{min}$ 。估算样液中可能的果聚糖含量，按照每毫克果聚糖加入 $1.2\mathrm{mL}$ 果聚糖酶溶液，旋涡振荡混匀，置于 $40^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 恒温水浴摇床中， $150~\mathrm{r / min}$ 振摇 $30\mathrm{min}$ 后，取出，冷却至室温。转移至 $10~\mathrm{mL}$ 容量瓶中，用水分三次冲洗玻璃试管，定容，摇匀。

活化净化柱，试样溶液依次通过 $0.22\mu \mathrm{m}$ 水相滤膜和净化柱，弃去前面3倍柱体积洗脱液，收集后面洗脱液待测。同时做试剂空白试验。

# 6 分析结果的表述

试样中果聚糖含量按式(1)、式(2)、式(3)计算：

$$
X = \frac {k _ {1} \times k _ {2} \times \left(c - c _ {0}\right) \times V \times 1 0}{m \times 0 . 2 \times 1 0 0 0} \quad \dots \tag {1}
$$

$$
k _ {1} = \frac {1 8 0 + 1 6 2 \times (n - 1)}{1 8 0 \times n} \tag {2}
$$

$$
k _ {2} = \frac {n}{n - 1} \tag {3}
$$

式中：

$X$ ——试样中果聚糖的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$k_{1}$ ——对果聚糖水解过程中增加的水分子进行校正；

$k_{2}$ ——对于 $\mathrm{F}_{\mathrm{n}}$ 型果聚糖末端果糖基被还原和 $\mathrm{GF}_{\mathrm{n}}$ 型果聚糖末端葡萄糖不被检测进行校正；

$c$ ——试样测定液中果糖的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$c_{0}$ ——试剂空白测定液中果糖的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$ ；

10 ——试样酶解液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

0.2 ——用于酶解的样品溶液体积，单位为毫升 $(\mathrm{mL})$ ；

1000 单位转换系数；

$n$ ——平均聚合度，对于低聚果糖，按照 $n = 4$ 计算；对于多聚果糖，按照 $n = 23$ 计算；对于菊粉，按照 $n = 10$ 计算。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 附录B

# 酶活力测定方法

# B.1 蔗糖酶活力测定方法

# B.1.1 原理

蔗糖酶(Sucrase, EC 3.2.1.26)又称转化酶，可以从果糖末端切开蔗糖的果糖糖苷键，使蔗糖水解生成葡萄糖和果糖，葡萄糖和果糖是还原糖，其含量可通过3,5-二硝基水杨酸比色法测定，从而度量酶活力的大小。由于蔗糖酶在碱性条件下极易失活，所以可用碱终止反应。3,5-二硝基水杨酸和还原糖共热后生成棕红色氨基化合物，在最大吸收波长 $540 \mathrm{~nm}$ 处测定，在一定浓度范围内吸光值与还原糖含量呈线性关系，因此可用于还原糖含量的测定，从而测定蔗糖酶的活力大小。

蔗糖酶活力定义为：在 $37\mathrm{^\circ C}$ 、 $\mathrm{pH}6.5$ 条件下，每分钟水解产生 $1\mu \mathrm{mol}$ 葡萄糖所需的酶量(活性)定义为1个酶活力单位。

# B.1.2 试剂

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

B.1.2.1 葡萄糖标准物质 $\mathrm{(C_6H_{12}O_6)}$ ：纯度 $\geq 99.0\%$   
B.1.2.2 蔗糖 $\left(\mathrm{C}_{12} \mathrm{H}_{22} \mathrm{O}_{11}\right)$ ：纯度 $\geqslant 99.0\%$ 。  
B.1.2.3 氢氧化钠(NaOH)。  
B.1.2.4 马来酸 $\left(\mathrm{C}_{4} \mathrm{H}_{4} \mathrm{O}_{4}\right)$ 。  
B.1.2.5 二硝基水杨酸 $\left(\mathrm{C}_{7} \mathrm{H}_{4} \mathrm{~N}_{2} \mathrm{O}_{7}\right)$ 。  
B.1.2.6 酒石酸钾钠 $\mathrm{(C_4H_4KNaO_6)}$ 。

# B.1.6 计算

蔗糖酶活力按式(B.1)计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V}{V _ {\mathrm {x}} \times 3 0} \quad \dots \tag {B.1}
$$

式中：

$X$ ——蔗糖酶活力，单位为活力单位每毫升 $(\mathrm{U} / \mathrm{mL})$ ；

$c$ ——待测酶测定液中葡萄糖的摩尔浓度，单位为微摩尔每毫升 $(\mu \mathrm{mol} / \mathrm{mL})$ ；

$c_{0}$ ——试剂空白测定液中葡萄糖的摩尔浓度，单位为微摩尔每毫升 $(\mu \mathrm{mol} / \mathrm{mL})$ ；

$V$ ——酶水解液最终定容体积 $(5\mathrm{mL})$ ，单位为毫升 $(\mathrm{mL})$

$V_{\mathrm{x}}$ ——待测酶的用量，单位为毫升 $(\mathrm{mL})$

30 ——反应时间，单位为分（min）。

# B.2 果聚糖酶活力测定方法

# B.2.1 原理

果聚糖酶(Inulinase, EC 3.2.1.80)又称菊粉酶，是能够水解 $\beta-2,1$ -D-果聚糖果糖苷键的一类水解酶，使菊粉水解生成葡萄糖和果糖，葡萄糖和果糖是还原糖，其含量可通过3,5-二硝基水杨酸比色法测定，从而度量酶活力的大小。3,5-二硝基水杨酸和还原糖共热后生成棕红色氨基化合物，在最大吸收波长 $540~\mathrm{nm}$ 处测定，在一定浓度范围内吸光值与还原糖含量呈线性关系，因此可用于还原糖含量的测定，从而测定果聚糖酶的活力大小。

果聚糖酶活力定义为：在 $55~\mathrm{^\circ C}$ 、 $\mathrm{pH}4.5$ 条件下，每分钟水解产生 $1\mu \mathrm{mol}$ 果糖所需的酶量(活性)定义为1个酶活力单位。

# B.2.2 试剂

B.2.2.2 菊粉：纯度 $\geqslant 90.0\%$   
B.2.2.3 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
B.2.2.4 三水乙酸钠 $\mathrm{(CH_3COONa\cdot 3H_2O)}$ 。  
B.2.2.5 二硝基水杨酸 $\mathrm{(C_7H_4N_2O_7)}$   
B.2.2.6 酒石酸钾钠 $\mathrm{(C_4H_4KNaO_6)}$   
B.2.2.7 苯酚 $\mathrm{(C_6H_5OH)}$ 。  
B.2.2.8 亚硫酸钠 $\left(\mathrm{Na}_{2} \mathrm{SO}_{3}\right)$ 。

# 1 范围

本文件规定了猪、牛、羊、鸡组织（肌肉、肝脏、肾脏和脂肪），鸡蛋，牛奶，羊奶，猪尿和牛尿中雌激素类药物（雌三醇、雌酮、炔雌醇、 $17\alpha$ 雌二醇、 $17\beta$ 雌二醇、己烯雌酚、己烷雌酚和己二烯雌酚）残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、牛、羊、鸡组织（肌肉、肝脏、肾脏和脂肪），鸡蛋，牛奶，羊奶，猪尿和牛尿中雌激素类药物（雌三醇、雌酮、炔雌醇、17α-雌二醇、17β-雌二醇、己烯雌酚、己烷雌酚和己二烯雌酚）残留量的测定。

# 4 原理

试样中残留的药物经酶解后用乙腈提取，固相萃取柱净化，液相色谱串联质谱仪测定，内标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ 。  
5.1.3 氢氧化钠(NaOH)。  
5.1.4 正己烷 $(\mathrm{C_6H_{14}})$   
5.1.5 二氯甲烷 $\mathrm{(CH_2Cl_2)}$   
5.1.6 氯化钠 $(\mathrm{NaCl})$ 。  
5.1.7 乙酸铵 $\mathrm{(CH_3COONH_4)}$   
5.1.8 乙酸 $\mathrm{(CH_3COOH)}$   
5.1.9 $\beta$ 葡萄糖醛酸酶/芳基硫酸酯酶（ $\beta$ -glucuronidase/aryl sulfatase）： $\beta$ -葡萄糖醛酸酶 $4.5\mathrm{U/mL}$ ，芳基硫酸酯酶 $14\mathrm{U/mL}$ 。

# 5.2 溶液的配制

5.2.1 二氯甲烷-甲醇溶液：取二氯甲烷 $70~\mathrm{mL}$ 、甲醇 $30~\mathrm{mL}$ ，混合。  
5.2.2 甲醇-水溶液：取甲醇 $40~\mathrm{mL}$ 、水 $60~\mathrm{mL}$ ，混合。  
5.2.3 乙酸铵溶液 $(0.02\mathrm{mol} / \mathrm{L})$ ：取乙酸铵 $1.54\mathrm{g}$ ，加水溶解并稀释至约 $800~\mathrm{mL}$ ，调节 $\mathsf{pH}$ 至5.2，稀释至 $1000~\mathrm{mL}$ ，混匀。

5.2.4 $40\%$ 乙腈溶液：取乙腈 $200~\mathrm{mL}$ 、水 $300~\mathrm{mL}$ ，混匀。

# 5.3 标准品

雌三醇、雌酮、炔雌醇、 $17\alpha$ 雌二醇、 $17\beta$ 雌二醇、己烯雌酚、己烷雌酚、己二烯雌酚雌三醇- $\mathrm{D}_3$ 、雌酮- $\mathrm{D}_2$ 、炔雌醇- $\mathrm{D}_4$ 、 $17\alpha$ 雌二醇- $\mathrm{D}_2$ 、 $17\beta$ 雌二醇- $\mathrm{D}_2$ 、己烯雌酚- $\mathrm{D}_8$ 、己烷雌酚- $\mathrm{D}_4$ 和己二烯雌酚- $\mathrm{D}_6$ ，含量均 $\geqslant 95\%$ 见附录A。

# 5.4 标准溶液的制备

5.4.1 标准储备液：取各雌激素类药物标准品、同位素标记标准品各 $10\mathrm{mg}$ ，精密称定，分别加乙腈适量使溶解并稀释至 $100~\mathrm{mL}$ 棕色容量瓶，制成浓度均为 $100~\mu \mathrm{g / mL}$ 的标准储备液。 $-18^{\circ}C$ 保存，有效期12个月。  
5.4.2 混合标准储备液：分别准确移取雌三醇、雌酮、炔雌醇、 $17\alpha$ 雌二醇、 $17\beta$ 雌二醇、己烯雌酚、己烷雌酚和己二烯雌酚标准储备液各 $5\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 棕色容量瓶，用乙腈稀释至刻度。 $-18^{\circ}C$ 保存，有效期12个月。  
5.4.3 混合内标储备液：分别准确移取雌三醇- $\mathrm{D}_3$ 、雌酮- $\mathrm{D}_2$ 、炔雌醇- $\mathrm{D}_4$ 、 $17\alpha$ 雌二醇- $\mathrm{D}_2$ 、 $17\beta$ 雌二醇- $\mathrm{D}_2$ 、己烯雌酚- $\mathrm{D}_8$ 、己烷雌酚- $\mathrm{D}_4$ 和己二烯雌酚- $\mathrm{D}_6$ 标准储备液各 $5\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 棕色容量瓶，用乙腈稀释至刻度。 $-18^{\circ}C$ 保存，有效期12个月。  
5.4.4 混合内标工作液：准确移取混合内标储备液 $0.2\mathrm{mL}$ 于 $10~\mathrm{mL}$ 棕色容量瓶，用乙腈稀释至刻度。该溶液中各内标浓度为 $100~\mu \mathrm{g / L}$ 。现用现配。  
5.4.5 混合标准工作液：准确移取混合标准储备液、混合内标工作液适量，用 $40\%$ 乙腈溶液稀释成浓度为 $1.0~\mu \mathrm{g / L},2.0~\mu \mathrm{g / L},5.0~\mu \mathrm{g / L},20~\mu \mathrm{g / L},50~\mu \mathrm{g / L}$ 和 $200~\mu \mathrm{g / L}$ 的系列标准工作溶液，每一标准工作溶液含各内标浓度均为 $5.0~\mu \mathrm{g / L}$ 。现用现配。

# 5.5 材料

5.5.1 HLB固相萃取柱： $200\mathrm{mg / 6mL}$ ，或相当者。  
5.5.2 氨基固相萃取柱： $500\mathrm{mg / 6mL}$ ，或相当者，使用前用二氯甲烷-甲醇溶液 $6\mathrm{mL}$ 活化。  
5.5.3 针头式过滤器（通用型滤膜）：尼龙材质，孔径 $0.22\mu \mathrm{m}$ 或性能相当者。

# 6 仪器和设备

6.1 液相色谱串联质谱仪：配自动进样器和电喷雾离子源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 氮吹仪。  
6.4 固相萃取装置。  
6.5 离心管：聚丙烯塑料离心管， $50~\mathrm{mL}$   
6.6 离心机： $10000\mathrm{r / min}$ 或以上。  
6.7 pH计。  
6.8 涡旋混合器。

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存，3个月内进行分析检测。

# 8 测定步骤

# 8.1 提取

# 8.1.1 动物组织、牛奶、羊奶和鸡蛋

取试样 $5\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管，准确加入混合内标工作液 $50~\mu \mathrm{L}$ ，混合 $15\mathrm{s}$ ，加乙酸铵溶液 $10~\mathrm{mL},\beta$ 葡萄糖醛酸酶/芳基硫酸酯酶 $50~\mu \mathrm{L}$ ，涡旋 $1\mathrm{min}$ ，盖盖子， $(37\pm 1)^{\circ}C$ 振荡酶解 $12\mathrm{h}$ 。取出，冷却至室温，加氯化钠 $4\mathrm{g}$ ，振摇溶解，加乙腈 $20~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ， $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取乙腈层，加乙腈 $15~\mathrm{mL}$ ，重复提取。合并2次提取液，加正己烷 $10~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ， $10000\mathrm{r / min}$ 离心 $2\mathrm{min}$ ，取乙腈层， $40^{\circ}C$ 氮吹至干，用甲醇 $1\mathrm{mL}$ 溶解，再加水 $9\mathrm{mL}$ ，备用。

# 8.1.2 动物尿液

取试样 $5.00\mathrm{mL}$ 于 $50~\mathrm{mL}$ 离心管，用乙酸调节 $\mathrm{pH}$ 至5.2，准确加混合内标工作液 $50~\mu \mathrm{L}$ ，混合 $15\mathrm{s}$ ，加乙酸铵溶液 $10~\mathrm{mL}$ 、 $\beta$ 葡萄糖醛酸酶/芳基硫酸酯酶 $50~\mu \mathrm{L}$ ，后续提取步骤同8.1.1。

# 8.2 净化

取HLB固相萃取柱，依次用二氯甲烷-甲醇 $6\mathrm{mL}$ 、甲醇 $6\mathrm{mL}$ ，和水 $6\mathrm{mL}$ 活化，取备用液，过柱，控制流速不超过 $2\mathrm{mL} / \mathrm{min}$ ，依次用甲醇-水 $3\mathrm{mL}$ 、水 $3\mathrm{mL}$ 淋洗小柱，抽干，再将活化过的氨基固相萃取柱串联在HLB固相萃取柱下方，用二氯甲烷-甲醇 $8\mathrm{mL}$ 洗脱；收集洗脱液，取下HLB柱，再用二氯甲烷-甲醇 $2\mathrm{mL}$ 洗氨基柱，合并洗脱液， $40^{\circ}C$ 水浴氮气吹干，用 $40\%$ 乙腈溶液 $1.0\mathrm{mL}$ 溶解残渣，涡旋混匀， $0.22\mu \mathrm{m}$ 滤膜过滤，供液相色谱串联质谱测定。

# 8.3 标准曲线的制备

取 $1.0\mu \mathrm{g} / \mathrm{L},2.0\mu \mathrm{g} / \mathrm{L},5.0\mu \mathrm{g} / \mathrm{L},20\mu \mathrm{g} / \mathrm{L},50\mu \mathrm{g} / \mathrm{L}$ 和 $200\mu \mathrm{g} / \mathrm{L}$ 的系列混合标准工作溶液，供液相色谱串联质谱测定。以激素类药物与相应内标物峰面积的比值为纵坐标、标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.4.3 测定法

取混合标准工作液与试样溶液交替进样，作单点或多点校准，按内标法以色谱峰面积定量，每种雌激素选择对应的同位素内标物进行定量。试样溶液中待测物的响应值均应在仪器测定的线性范围内。试样中待测物质的保留时间与相应内标物保留时间的比值与标准溶液中待测物质的保留时间与相应内标物保留时间的比值偏差在 $\pm 2.5\%$ 之内，且样品中各组分的相对离子丰度与浓度接近的标准溶液的相对离子丰度一致，偏差不超过表3规定的范围，则可判定为样品中存在对应的待测物。标准工作溶液多反应监测色谱图见附录B。

# 9 结果计算和表述

试料中待测药物的残留量按标准曲线或按公式(1)～公式(3)计算。

$$
C = \frac {A _ {i} \times A _ {i S} ^ {\prime} \times C _ {S} \times C _ {i S}}{A _ {i S} \times A _ {S} \times C _ {i S} ^ {\prime}} \tag {1}
$$

动物性食品中的残留量：

$$
X = \frac {C \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {2}
$$

尿液中的残留量：

$$
X = \frac {C \times V}{v} \tag {3}
$$

式中：

$X$ ——试料中雌激素类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ 或微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$C$ ——试料溶液中雌激素类药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{iS}$ ——试料溶液中雌激素类药物内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$C_{\mathrm{S}}$ ——标准溶液中雌激素类药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{iS}^{\prime}$ ——标准溶液中雌激素类药物内标浓度的数值，单位为微克每升 $(\mu \mathrm{g / L})$

$A_{i}$ ——试料溶液中雌激素类药物的峰面积；

$A_{iS}$ ——试料溶液中雌激素类药物内标的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中雌激素类药物的峰面积；

$A_{iS}^{\prime}$ ——标准溶液中雌激素类药物内标的峰面积；

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ ——试料质量的数值，单位为克(g)；

$v$ ——试料体积的数值，单位为毫升 $(\mathrm{mL})$

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法在猪、牛、羊、鸡组织（肌肉、肝脏、肾脏和脂肪），鸡蛋，牛奶和羊奶中的检测限为 $0.50~\mu \mathrm{g / kg}$ 定量限为 $1.0~\mu \mathrm{g / kg}$ ；猪尿和牛尿中的检测限为 $0.50~\mu \mathrm{g / L}$ ，定量限为 $1.0~\mu \mathrm{g / L}$ 。

# 10.2 准确度

本方法在 $1\mu \mathrm{g / kg(L)}\sim 5\mu \mathrm{g / kg(L)}$ 添加浓度水平上的回收率为 $80\% \sim 130\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

# 前言

本文件按照GB/T1.1—2020《标准化工作导则第1部分：标准化文件的结构和起草规则》的规定起草。

# 食品安全国家标准

# 水产品中镇静剂类药物残留量的测定

# 液相色谱-串联质谱法

# 1 范围

本文件规定了水产品中安眠酮、硝西泮、地西泮、奥沙西泮、艾司唑仑、阿普唑仑、劳拉西泮、三唑仑残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鱼、虾、蟹、海参可食组织中安眠酮、硝西泮、地西泮、奥沙西泮、艾司唑仑、阿普唑仑、劳拉西泮、三唑仑药物残留量的测定。其他水产品的检测可参照执行。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

GB/T30891-2014水产品抽样规范

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的镇静剂类药物,用酸化乙腈提取,亲水亲脂平衡型固相萃取柱和乙二胺-N-丙基硅烷化硅胶净化,液相色谱-串联质谱测定,基质匹配标准溶液外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯

5.1.3 乙酸 $\mathrm{(CH_3COOH)}$ 。

5.1.4 甲酸（HCOOH）：色谱纯。

5.1.5 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：经 $650^{\circ}C$ 灼烧 $4\mathrm{h}$ ，冷却后储于密闭容器中备用。

# 5.2 溶液配制

5.2.1 $1\%$ 乙酸乙腈溶液：取乙酸 $10~\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.2 $60\%$ 乙腈水溶液：取乙腈 $600~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.3 复溶液：取甲酸 $1\mathrm{mL}$ ，用 $60\%$ 乙腈水溶液稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.4 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀。

# 5.3 标准物质

安眠酮、硝西泮、地西泮、奥沙西泮、艾司唑仑、阿普唑仑、劳拉西泮、三唑仑标准溶液，浓度 $1\mathrm{mg / mL}$ 具体见附录A。

# 5.4 标准溶液配制

5.4.1 $10~\mu \mathrm{g / mL}$ 混合标准工作溶液：分别精密量取 $1\mathrm{mg / mL}$ 标准溶液各 $1\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的混合标准工作溶液。 $-18^{\circ}C$ 以下避光保存，有效期3个月。  
5.4.2 $1\mu \mathrm{g / mL}$ 混合标准工作溶液：精密量取 $10~\mu \mathrm{g / mL}$ 混合标准工作溶液 $1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，配制成浓度为 $1\mu \mathrm{g / mL}$ 的混合标准工作溶液。一 $18^{\circ}C$ 以下避光保存，有效期3个月。

# 5.5 材料

5.5.1 亲水亲脂平衡型固相萃取柱： $60\mathrm{mg / 3mL}$ ，或相当者。  
5.5.2 PSA吸附剂： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}_{\circ}$   
5.5.3 聚偏氟乙烯微孔滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源(ESI源)。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和感量 $0.00001\mathrm{g}$ 。  
6.3 氮吹仪。  
6.4 均质器。  
6.5 涡旋混合器。  
6.6 离心机：8000 r/min 或以上。  
6.7 超声波振荡器。  
6.8 具塞聚丙烯塑料离心管： $50~\mathrm{mL}$ 。  
6.9 聚丙烯塑料离心管： $20~\mathrm{mL}$ 。

# 7 试样的制备与保存

# 7.1 试样的制备

按GB/T30891—2014附录B的要求制样。

a）取均质后的供试样品，作为供试试样；  
b) 取均质后的空白样品, 作为空白试样;  
c）取均质后的空白样品，添加适宜浓度的标准溶液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试料 $(5 \pm 0.05) \mathrm{g}$ , 置于 $50 \mathrm{~mL}$ 离心管中, 量取 $1 \%$ 乙酸乙腈溶液 $20 \mathrm{~mL}$ 加入离心管中, 涡旋 $30 \mathrm{~s}$ , $35^{\circ} \mathrm{C}$ 超声提取 $10 \mathrm{~min}$ , 加入无水硫酸钠 $5 \mathrm{~g}$ , 涡旋 $1 \mathrm{~min}, 8000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ , 移取上清液于 $20 \mathrm{~mL}$ 离心管中, $40^{\circ} \mathrm{C}$ 氮气吹至近干, 加入乙腈 $0.5 \mathrm{~mL}$ 溶解残渣, 加入水 $5 \mathrm{~mL}$ , 备用。

# 8.2 净化

固相萃取柱用甲醇 $5\mathrm{mL}$ 、水 $5\mathrm{mL}$ 活化，取备用液过柱，控制流速不超过 $1\mathrm{mL} / \mathrm{min}$ ，用水 $5\mathrm{mL}$ 淋洗，加乙腈 $10\mathrm{mL}$ 洗脱。收集洗脱液， $40^{\circ}\mathrm{C}$ 氮气吹干。准确加入复溶液 $1.0\mathrm{mL}$ ，加入PSA吸附剂 $0.15\mathrm{g}$ ，涡旋 $1\mathrm{min},8000\mathrm{r} / \mathrm{min}$ 离心 $10\mathrm{min}$ ，取上清液过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱仪分析。

# 8.3 基质匹配标准曲线的制备

准确移取混合标准工作溶液适量，用复溶液稀释成浓度为 $1.0\mu \mathrm{g} / \mathrm{L},5.0\mu \mathrm{g} / \mathrm{L},10.0\mu \mathrm{g} / \mathrm{L},50.0\mu \mathrm{g} /$ L、100.0μg/L的系列标准工作溶液。取空白试料，依次按8.1和8.2处理至氮气吹干，分别加入系列标

准工作溶液 $1.0\mathrm{mL}$ 溶解残渣，加入PSA吸附剂 $0.15\mathrm{g}$ ，涡旋 $1\mathrm{min}$ ， $8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液过 $0.22\mu \mathrm{m}$ 滤膜，制成浓度为 $1.0~\mu \mathrm{g} / \mathrm{L}, 5.0~\mu \mathrm{g} / \mathrm{L}, 10.0~\mu \mathrm{g} / \mathrm{L}, 50.0~\mu \mathrm{g} / \mathrm{L}, 100.0~\mu \mathrm{g} / \mathrm{L}$ 的系列基质匹配标准溶液，供液相色谱-串联质谱仪测定。使用时现用现配。以待测物特征离子峰面积为纵坐标、相应的基质匹配标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.5 测定法

# 8.5.1 定性方法

在相同测试条件下，试料溶液中各组分与校正基质匹配标准曲线中各组分的保留时间偏差在 $\pm 0.1\mathrm{min}$ 以内；且检测到的相对离子丰度应与浓度相当的基质匹配标准溶液相对离子丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.5.2 定量方法

取试料溶液和基质匹配标准溶液，作单点或多点校准，按外标法以色谱峰面积定量。基质匹配标准溶液和试料溶液中目标物的峰面积均应在仪器检测的线性范围之内。8种镇静剂类药物标准溶液特征离子质量色谱图见附录B。

# 8.6 空白试验

取空白试样，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中8种镇静剂类药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times C _ {\mathrm {s}} \times V}{A _ {\mathrm {s}} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中镇静剂类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

A ——试料溶液中镇静剂类药物的色谱峰面积；

$C_s$ ——基质匹配标准溶液中镇静剂类药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——基质匹配标准溶液中镇静剂类药物的色谱峰面积；

$m$ ——供试试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法的检出限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1.0\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法在 $1\mu \mathrm{g} / \mathrm{kg}\sim 10\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $60\% \sim 110\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

# 液相色谱一质谱/质谱法

# 1 范围

本标准规定了食品中8种二硝基苯胺类农药残留量的液相色谱一质谱/质谱法检测方法。

本标准适用于黄豆、大米、菠菜、生姜、苹果、西瓜、甘蓝、节瓜、茶叶、鸡蛋、猪肉和鸡肝中氟乐灵、二甲戊灵、氨磺乐灵、仲丁灵、氨氟乐灵、氨氟灵、甲磺乐灵和异丙乐灵8种二硝基苯胺类农药残留量的测定和确证。其他食品可参照执行。

# 3 原理

试样用乙腈振荡提取，石墨化炭黑固相萃取柱和HLB固相萃取柱净化，液相色谱一串联质谱法仪测定和确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
4.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
4.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
4.1.4 甲酸 $(\mathrm{CH}_2\mathrm{O}_2)$ ：色谱纯。  
4.1.5 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ ：优级纯。  
4.1.6 氯化钠(NaCl)：分析纯。  
4.1.7 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：分析纯，用前经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置干燥器中备用。

# 4.2 溶液配制

4.2.1 乙腈溶液 $(1 + 1$ ，体积比，含 $0.05\%$ 甲酸)：量取 $500~\mathrm{mL}$ 乙腈与 $0.5\mathrm{mL}$ 甲酸于 $1\mathrm{L}$ 容量瓶中，用水定容至 $1\mathrm{L}$ ，混匀。  
4.2.2 正己烷一丙酮溶液 $(2 + 8$ ，体积比)：取 $20~\mathrm{mL}$ 正己烷，加入 $80~\mathrm{mL}$ 丙酮，混匀。

# 4.3 标准品

4.3.1 二硝基苯胺类农药标准物质：参见表A.1。

# 4.4 标准溶液配制

4.4.1 二硝基苯胺类农药标准储备溶液：分别准确称取适量的二硝基苯胺类农药标准物质，用丙酮配制成浓度为 $1000\mathrm{mg / L}$ 的标准储备溶液，标准溶液避光于 $-18^{\circ}C$ 保存，保存期为12个月。  
4.4.2 二硝基苯胺类农药混合中间标准溶液: 吸取适量的各标准储备溶液, 用甲醇稀释成氟乐灵浓度为 $5.0 \mathrm{mg} / \mathrm{L}$ , 其他 7 种药物浓度为 $1.0 \mathrm{mg} / \mathrm{L}$ 的混合中间标准溶液, $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 避光保存, 保存期为 3 个月。  
4.4.3 基质混合标准工作溶液: 吸取适量的混合中间标准溶液, 用空白样品提取液配成氟乐灵浓度为 $0 \mu \mathrm{g} / \mathrm{L}, 50.0 \mu \mathrm{g} / \mathrm{L}, 100 \mu \mathrm{g} / \mathrm{L}, 250 \mu \mathrm{g} / \mathrm{L}, 500 \mu \mathrm{g} / \mathrm{L}$ 和 $1000 \mu \mathrm{g} / \mathrm{L}$ , 其他 7 种农药浓度为 $0 \mu \mathrm{g} / \mathrm{L}, 10.0 \mu \mathrm{g} / \mathrm{L}, 20.0 \mu \mathrm{g} / \mathrm{L}, 50.0 \mu \mathrm{g} / \mathrm{L}, 100 \mu \mathrm{g} / \mathrm{L}$ 和 $200 \mu \mathrm{g} / \mathrm{L}$ 的基质混合标准工作溶液。当天配制。

# 4.5 材料

4.5.1 石墨化炭黑固相萃取柱： $3\mathrm{mL} / 250\mathrm{mg}$ ，或相当者。用前 $3\mathrm{mL}$ 正己烷一丙酮溶液活化，保持柱体湿润。  
4.5.2Oasis $\mathrm{HLB}^{1)}$ ， $6\mathrm{mL} / 200\mathrm{mg}$ ，或相当者。使用前用 $5\mathrm{mL}$ 乙腈活化，保持柱体湿润。

# 5 仪器和设备

5.1 液相色谱—质谱/质谱仪：配有电喷雾离子源(ESI)。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.3 粉碎机。  
5.4 组织捣碎机。  
5.5 涡旋振荡器。  
5.6 固相萃取装置，带真空泵。  
5.7 氮吹浓缩仪。  
5.8 离心机：转速 $\geq 5000\mathrm{r / min}$

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按GB2763的执行。在取样和制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 黄豆、茶叶和大米

取代表性样品约 $500\mathrm{g}$ (不可用水洗), 切碎后, 用捣碎机将样品加工成浆状, 混匀。试样均分为2份, 装入洁净容器, 密封, 并标明标记。

# 6.2.2 蔬菜、水果、鸡蛋、猪肉及鸡肝

取代表性样品约 $500 \mathrm{~g}$ (不可用水洗), 切碎后, 用捣碎机将样品加工成浆状, 混匀。试样均分为 2 份, 装入洁净容器, 密封, 并标明标记。

# 6.3 试样保存

黄豆、茶叶和大米试样于常温下保存。蔬菜、水果、鸡蛋、猪肉及鸡肝试样于 $-18^{\circ}C$ 以下冷冻存放。

# 7 分析步骤

# 7.1 提取

# 7.1.1 黄豆、大米、茶叶、鸡蛋、猪肉及鸡肝

对于茶叶、大豆、板栗、玉米样品，称取 $2.5\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）。对于菠菜、蘑菇、苹果、牛肉、牛肝、鸡肉、鱼肉、牛奶样品，称取 $5\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）。将称取的试样置于 $50\mathrm{mL}$ 离心管中，加入 $6\mathrm{mL}$ 饱和氯化钠水溶液，于涡旋振荡器上混匀 $30\mathrm{s}$ ，放置 $15\mathrm{min}$ 。加入 $6\mathrm{mL}$ 丙酮一正己烷溶液，在混匀器上混匀 $2\mathrm{min}$ 。 $5000\mathrm{r/min}$ 离心 $1\mathrm{min}$ ，吸取上层提取液于另一试管中。再分别加入 $4\mathrm{mL}$ 丙酮一正己烷溶液重复提取两次，合并提取液，在 $45^{\circ}\mathrm{C}$ 下氮气流吹至约 $1\mathrm{mL}$ 待净化。

# 7.1.2 蔬菜、水果

称取约 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )样品于 $50~\mathrm{mL}$ 具塞离心管中，然后加入 $2\mathrm{g}$ 氯化钠，再加入 $15\mathrm{mL}$ 乙腈高速均质提取 $3\mathrm{min}$ ，于 $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将乙腈层转移至 $25\mathrm{mL}$ 容量瓶中。残渣再用 $10\mathrm{mL}$ 乙腈重复提取一次，合并提取液，并用乙腈定容至 $25~\mathrm{mL}$

# 7.2 净化

# 7.2.1 黄豆、大米、猪肉

吸取全部浓缩提取液于Supelclean $\mathrm{C}_{18}$ 固相萃取柱中，以约 $1.5\mathrm{mL} / \mathrm{min}$ 的流速使样液全部通过固相萃取柱，再用 $5\mathrm{mL}$ 乙腈淋洗并抽干固相萃取柱，收集全部流出液于 $1.5\mathrm{mL}$ 离心管中， $40^{\circ}\mathrm{C}$ 以下用氮气吹至近干，残渣用乙腈+水溶液定容至 $1.0\mathrm{mL}$ 。涡旋混匀后，过微孔滤膜，供液相色谱—质谱/质谱仪测定。

# 7.2.2 蔬菜、水果

精确吸取 $5 \mathrm{~mL}$ 提取液 $5.0 \mathrm{~mL}$ 于 $15 \mathrm{~mL}$ 离心管中, $40^{\circ} \mathrm{C}$ 以下用氮气吹至近干, 用 $2.0 \mathrm{~mL}$ 正己烷一丙酮溶液溶解, 转入 Envi-carb 固相萃取柱中, 以约 $1.5 \mathrm{~mL} / \mathrm{min}$ 的流速使样液全部通过固相萃取柱, 再用 $3 \mathrm{~mL}$ 正己烷一丙酮溶液淋洗并抽干固相萃取柱, 收集全部流出液于 $15 \mathrm{~mL}$ 离心管中, $40^{\circ} \mathrm{C}$ 以下用氮气吹至近干, 残渣用乙腈溶液定容至 $1.0 \mathrm{~mL}$ 。涡旋混匀后, 过微孔滤膜供液相色谱一质谱/质谱仪测定。

# 7.2.3 茶叶、鸡蛋及鸡肝

吸取茶叶、鸡蛋及鸡肝浓缩提取液于OasisHLB固相萃取柱中，以约 $1.5\mathrm{mL / min}$ 的流速使样液全部通过固相萃取柱，再用 $5\mathrm{mL}$ 乙腈淋洗并抽干固相萃取柱，收集全部流出液于 $15~\mathrm{mL}$ 离心管中， $40^{\circ}\mathrm{C}$ 以下用氮气吹至近干，用 $2.0\mathrm{mL}$ 正己烷一丙酮溶液溶解，转入Envi-carb固相萃取柱中，以约 $1.5\mathrm{mL}/$ $\mathrm{min}$ 的流速使样液全部通过固相萃取柱，再用 $3\mathrm{mL}$ 正己烷一丙酮溶液淋洗并抽干固相萃取柱，收集全部流出液于 $15~\mathrm{mL}$ 离心管中， $40^{\circ}\mathrm{C}$ 以下用氮气吹至近干，残渣用乙腈十水溶液定容至 $1.0\mathrm{mL}$ 。旋涡混匀后，过微孔滤膜，供液相色谱一质谱/质谱仪测定。

# 7.3.4 定量测定

仪器最佳工作条件下，对基质混合标准工作溶液进样，以峰面积为纵坐标，基质混合标准工作溶液浓度为横坐标绘制标准工作曲线，用标准工作曲线对样品进行定量。样品溶液中待测物的响应值均应在测定方法的线性范围内，如果超出线性响应范围，应用空白基质溶液进行适当稀释。在 $7.3.1 \sim 7.3.2$ 规定的液相色谱—质谱条件下，二硝基苯胺类农药的保留时间分别为：氨磺乐灵（ $3.21\mathrm{min}$ ）、甲磺乐灵（ $3.41\mathrm{min}$ ）、氨氟灵（ $4.06\mathrm{min}$ ）、氨氟乐灵（ $5.20\mathrm{min}$ ）、二甲戊灵（ $5.36\mathrm{min}$ ）、仲丁灵（ $5.70\mathrm{min}$ ）、氟乐灵（ $5.70\mathrm{min}$ ）和异丙乐灵（ $6.06\mathrm{min}$ ）。标准品的液相色谱—质谱/质谱多反应监测色谱图参见图C.1。

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中各二硝基苯胺类农药的含量。

$$
X = \frac {A _ {i} \times C _ {s} \times V}{A _ {s} \times m}. \tag {1}
$$

式中：

$X$ ——试样中各二硝基苯胺类农药的残留含量，单位为毫克每千克 $(\mathrm{mg / kg})$

$A_{i}$ ——样液中各二硝基苯胺类农药的峰面积；

$C_{s}$ ——标准工作液中各二硝基苯胺类农药的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{si}$ ——标准工作液中各二硝基苯胺类农药的峰面积；

$m$ ——最终样液所代表的试样量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录D的要求。

9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录 E 的要求。

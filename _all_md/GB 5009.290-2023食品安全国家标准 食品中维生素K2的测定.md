# 食品安全国家标准

# 食品中维生素 $\mathbf{K}_2$ 的测定

# 1 范围

本标准规定了食品中四烯甲萘醌、七烯甲萘醌、九烯甲萘醌含量的液相色谱测定方法。

本标准适用于乳及乳制品、特殊膳食用食品、发酵豆制品和肉及肉制品中四烯甲萘醌、七烯甲萘醌、九烯甲萘醌三种维生素 $\mathrm{K}_{2}$ 分型的测定。

# 2 原理

试样经脂肪酶、淀粉酶或蛋白酶酶解，采用正己烷提取四烯甲萘醌、七烯甲萘醌、九烯甲萘醌，提取液浓缩后经反相液相色谱分离，锌还原柱柱后衍生，荧光检测器检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 无水乙醇 $(\mathrm{C}_2\mathrm{H}_6\mathrm{O})$ 。  
3.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ 。  
3.1.4 碳酸钾 $\mathrm{(K_2CO_3)}$   
3.1.5 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：色谱纯。  
3.1.6 四氢呋喃 $\mathrm{(C_4H_8O)}$ ：色谱纯。  
3.1.7 冰乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
3.1.8 氯化锌 $(\mathrm{ZnCl}_2)$ 。  
3.1.9 无水乙酸钠 $\mathrm{(C_2H_3O_2Na)}$   
3.1.10 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
3.1.11 淀粉酶：(CAS号：9000-92-4)酶活力 $\geqslant 4000\mathrm{U / g}$   
3.1.12 脂肪酶：(CAS号：9001-62-1)酶活力 $\geqslant 40\mathrm{U / g}$   
3.1.13 蛋白酶：(CAS号：9001-73-4)酶活力 $\geq 6000\mathrm{U / mg}$   
3.1.14 氢氧化钾(KOH)。

# 3.2 试剂配制

3.2.1 $400\mathrm{g / L}$ 氢氧化钾溶液：称取 $20.0\mathrm{g}$ 氢氧化钾于 $100~\mathrm{mL}$ 烧杯中，用 $20~\mathrm{mL}$ 水溶解，冷却后，转移至 $50~\mathrm{mL}$ 容量瓶中，加水定容至刻度，混匀，储存于聚乙烯瓶中。  
3.2.2 磷酸盐缓冲溶液 $(\mathrm{pH}8.0)$ : 称取 $54.0 \mathrm{~g}$ 磷酸二氢钾于 $500 \mathrm{~mL}$ 烧杯中, 用 $300 \mathrm{~mL}$ 水溶解, 用 $400 \mathrm{~g} / \mathrm{L}$ 氢氧化钾溶液 (3.2.1) 调节 $\mathrm{pH}$ 至 $8.0 \pm 0.2$ , 转移至 $500 \mathrm{~mL}$ 容量瓶中, 加水定容至刻度, 混匀。

3.2.3 流动相：称取 $1.5\mathrm{g}$ 氯化锌、 $0.5\mathrm{g}$ 无水乙酸钠于 $1000~\mathrm{mL}$ 烧杯中，加入 $500~\mathrm{mL}$ 甲醇、 $100~\mathrm{mL}$ 二氯甲烷（或四氢呋喃）、 $0.3~\mathrm{mL}$ 冰醋酸，转移至 $1000~\mathrm{mL}$ 容量瓶中，超声溶解后，加甲醇定容至刻度，混匀，用 $0.22\mu \mathrm{m}$ 滤膜过滤。

注：增加二氯甲烷，可以缩短保留时间，最高不宜超过 $150~\mathrm{mL / L}$

3.2.4 二氯甲烷甲醇溶液（体积比 $10 + 90$ ）：准确吸取二氯甲烷 $20~\mathrm{mL}$ 于 $200~\mathrm{mL}$ 容量瓶中，用甲醇稀释并定容至刻度，混匀。

# 3.3 标准品

3.3.1 四烯甲萘醌(MK-4)标准品 $\mathrm{(C_{31}H_{40}O_2}$ ，CAS号：863-61-6)：纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.2 七烯甲萘醌(MK-7)标准品 $\mathrm{(C_{46}H_{64}O_2}$ ，CAS号：2124-57-4）：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.3 九烯甲萘醌(MK-9)标准品 $\mathrm{(C_{56}H_{80}O_2}$ ，CAS号：523-39-7)：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

# 3.4.1 标准储备溶液

3.4.1.1 MK-4标准储备溶液 $(1.0\mathrm{mg / mL})$ ：准确称取MK-4标准品 $25~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )，用二氯甲烷甲醇溶液(3.2.4)溶解并定容至 $25~\mathrm{mL}$ ，混匀。将溶液储存于棕色玻璃容器中， $-20^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 避光保存，保存期限为6个月。  
3.4.1.2 MK-7标准储备溶液 $(1.0\mathrm{mg / mL})$ ：准确称取MK-7标准品 $25~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )，用二氯甲烷甲醇溶液(3.2.4)溶解并定容至 $25~\mathrm{mL}$ ，混匀。将溶液储存于棕色玻璃容器中， $-20^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 避光保存，保存期限为6个月。  
3.4.1.3 MK-9标准储备溶液 $(0.5\mathrm{mg / mL})$ ：准确称取MK-9标准品 $25~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )，用二氯甲烷甲醇溶液(3.2.4)溶解并定容至 $50~\mathrm{mL}$ ，混匀。将溶液储存于棕色玻璃容器中， $-20^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 避光保存，保存期限为6个月。

# 3.4.2 标准中间溶液

3.4.2.1 MK-4和MK-7混合标准中间溶液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：分别准确吸取MK-4、MK-7标准储备溶液 $0.50~\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，用二氯甲烷甲醇溶液(3.2.4)稀释并定容至刻度，混匀。将溶液储存于棕色玻璃容器中， $-20^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 避光保存，保存期限为6个月。  
3.4.2.2 MK-9标准中间溶液 $(50\mu \mathrm{g / mL})$ ：准确吸取MK-9标准储备溶液 $5.00~\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，用二氯甲烷甲醇溶液(3.2.4)稀释并定容至刻度，混匀。将溶液储存于棕色玻璃容器中， $-20^{\circ}C\pm 2^{\circ}C$ 避光保存，保存期限为6个月。

# 3.4.3 标准工作溶液

分别准确吸取MK-4、MK-7混合标准中间溶液 $0.025\mathrm{mL},0.050\mathrm{mL},0.10\mathrm{mL},0.30\mathrm{mL},0.50\mathrm{mL}$ 产 $1.0~\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，准确吸取MK-9标准中间溶液 $0.010\mathrm{mL},0.020\mathrm{mL},0.040\mathrm{mL},0.060\mathrm{mL}$ $0.10~\mathrm{mL},0.20~\mathrm{mL}$ 于上述 $10~\mathrm{mL}$ 容量瓶中，用流动相(3.2.3)定容至刻度，混匀。得到MK-4、MK-7标准工作溶液质量浓度分别为 $0.025\mu \mathrm{g / mL},0.050\mu \mathrm{g / mL},0.10\mu \mathrm{g / mL},0.30\mu \mathrm{g / mL},0.50\mu \mathrm{g / mL}$ $1.0~\mu \mathrm{g / mL}$ ，MK-9标准工作溶液质量浓度分别为 $0.050~\mu \mathrm{g / mL},0.10~\mu \mathrm{g / mL},0.20~\mu \mathrm{g / mL},0.30~\mu \mathrm{g / mL}$ $0.50~\mu \mathrm{g / mL},1.0\mu \mathrm{g / mL}$ 。临用现配。

注：MK-7、MK-9低温高浓度时易过饱和，出现浑浊，可提高溶解温度至 $40^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 超声，至完全溶解，恢复室温使用；所称取的标准品质量是折合纯标准品的质量。

# 3.5 材料

一次性微孔滤膜： $0.22\mu \mathrm{m}$ ，有机相。

# 4 仪器和设备

4.1 液相色谱仪：配有荧光检测器。  
4.2 电子天平：感量分别为 $0.1\mathrm{g},1\mathrm{mg}$ 和 $0.01\mathrm{mg}$   
4.3 水平振荡器。  
4.4 涡旋混合器。  
4.5 高速离心机（转速 $\geq 6000\mathrm{r / min}$ ）。  
4.6 恒温水浴锅。  
4.7 旋转蒸发仪。  
4.8 氮气浓缩装置。  
4.9 pH计：精确至0.1。  
4.10 组织粉碎机。  
4.11 均浆器。  
4.12 超声波清洗器。

# 5 分析步骤

# 5.1 试样制备

质地均匀的粉末类样品测定前充分混匀；肉与肉制品取可食部分粉碎均质；其他固体、半固体样品粉碎均质。液态样品测定前混匀。

注：处理过程尽可能避光操作，制备成均匀试样后，避光尽快检测。

# 5.2 试样处理

警示：处理过程应避免紫外光直接照射，尽可能避光操作。

# 5.2.1 酶解

# 5.2.1.1 乳及乳制品和特殊膳食用食品

称取混匀后的粉末试样 $50\mathrm{g}$ （精确至 $0.1\mathrm{g})$ $(m_{1})$ 至 $250~\mathrm{mL}$ 烧杯中，加入 $100~\mathrm{mL}$ 水，记录加水后的溶液质量 $(m_2)$ ，充分混匀溶解；准确称取制备后的溶液 $6\mathrm{g}$ （精确至 $0.001\mathrm{g})$ $(m_3)$ 于 $50~\mathrm{mL}$ 离心管中，加入水 $11~\mathrm{mL}$ ，再加入磷酸盐缓冲液(3.2.2) $5~\mathrm{mL}$ ，混匀，于 $50^{\circ}C\pm 2^{\circ}C$ 水浴超声 $15\mathrm{min}$ ，冷却至室温后，加入脂肪酶 $0.5\mathrm{g}$ ，再加淀粉酶 $0.2\mathrm{g}$ ，加盖，涡旋 $2\mathrm{min}\sim 3\mathrm{min}$ ，混匀后，置于 $37^{\circ}C\pm 2^{\circ}C$ 恒温水浴锅中酶解 $2\textrm{h}$ 以上（每 $30\mathrm{min}$ 涡旋 $1\mathrm{min}$ )，使其充分酶解。

液体样品，称取混匀样品 $1\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.001\mathrm{g})$ $(m_{s})$ 于 $50~\mathrm{mL}$ 离心管中，加水至 $15~\mathrm{mL}$ ；提取液加入磷酸盐缓冲液(3.2.2) $5\mathrm{mL}$ ，混匀，于 $50^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 水浴超声 $15\mathrm{min}$ ，冷却至室温后，加入脂肪酶 $0.5\mathrm{g}$ ，再加淀粉酶 $0.2\mathrm{g}$ ，加盖，涡旋 $2\mathrm{min}\sim 3\mathrm{min}$ ，混匀后，置于 $37^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 恒温水浴锅中酶解 $2\textrm{h}$ 以上（每 $30\mathrm{min}$ 涡旋 $1\mathrm{min}$ )，使其充分酶解。

# 5.2.1.2 发酵豆制品

称取均质后的试样 $50\mathrm{g}$ （精确至 $0.1\mathrm{g})$ $(m_{1})$ 至 $250~\mathrm{mL}$ 烧杯中，加入 $100~\mathrm{mL}$ 水，记录加水后的溶液质量 $(m_2)$ ，匀浆器匀浆后，立即准确称取匀浆试样 $3\mathrm{g}$ （精确至 $0.001\mathrm{g})$ $(m_3)$ 于 $50~\mathrm{mL}$ 离心管中，加入水 $13~\mathrm{mL}$ ，加入磷酸盐缓冲液(3.2.2) $5~\mathrm{mL}$ ，混匀，于 $50^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 水浴超声 $15\mathrm{min}$ ，冷却至室温后，加入脂肪酶 $0.2\mathrm{g}$ ，加入蛋白酶 $0.1\mathrm{g}$ ，加盖，涡旋 $2\mathrm{min} \sim 3\mathrm{min}$ ，混匀后，置于 $37^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 恒温水浴锅中酶解 $4\mathrm{h}$ 以上（每 $30\mathrm{min}$ 涡旋 $1\mathrm{min}$ ），使其充分酶解。

# 5.2.1.3 肉及肉制品

称取均质后的试样 $50\mathrm{g}$ （精确至 $0.1\mathrm{g})$ $(m_{1})$ 至 $250~\mathrm{mL}$ 烧杯中，加入 $100~\mathrm{mL}$ 水，记录加水后的溶液质量 $(m_2)$ ，匀浆器匀浆后，立即准确称取匀浆试样 $15\mathrm{g}$ （精确至 $0.001\mathrm{g})$ $(m_3)$ 于 $50~\mathrm{mL}$ 离心管中，加入水 $5\mathrm{mL}$ ，加入磷酸盐缓冲液(3.2.2) $5\mathrm{mL}$ ，混匀，于 $50^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 水浴超声 $15\mathrm{min}$ ，冷却至室温后，加入脂肪酶 $0.2\mathrm{g}$ ，加入蛋白酶 $0.1\mathrm{g}$ ，加盖，涡旋 $2\mathrm{min} \sim 3\mathrm{min}$ ，混匀后，置于 $37^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 恒温水浴锅中酶解 $4\mathrm{h}$ 以上（每 $30\mathrm{min}$ 涡旋 $1\mathrm{min}$ ），使其充分酶解。

# 5.2.2 提取

取出酶解好的试样，分别加入 $10~\mathrm{mL}$ 无水乙醇及 $1.0\mathrm{g}$ 碳酸钾，混匀后加入 $10~\mathrm{mL}$ 正己烷， $3000\mathrm{r / min}$ 涡旋 $2\mathrm{min} \sim 3\mathrm{min}$ 或 $\geqslant 250\mathrm{r / min}$ 振荡提取 $10\mathrm{min}$ ，以 $6000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，转移上清液至 $100~\mathrm{mL}$ 旋蒸瓶中。

注：为保证提取完全,必要时可向下层溶液中再加入 ${10}\mathrm{\;{mL}}$ 正己烷,重复操作 1 次,合并上清液至上述旋蒸瓶中。

# 5.2.3 浓缩

将上述正己烷提取液于 $40^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 水浴旋蒸至干（如有残液，可用氮气轻吹至干），用 $5.0\mathrm{mL}$ 流动相(3.2.3)充分溶解浓缩物，将此溶解液用 $0.22\mu \mathrm{m}$ 滤膜过滤，此滤液为制备的试样溶液。

注：浓缩氮吹后，如果样品浓缩物常温不易溶解，可提高溶解温度至 $40\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}$ ，至完全溶解。纳豆等 MK-7 含量高的样品，可适当加大(5.2.3)定容体积，使制备的试样溶液浓度在标准曲线范围内。

# 5.3 仪器参考条件

5.3.1 色谱柱： $\mathrm{C_{18}}$ （ $5 \mu \mathrm{m}, 4.6 \mathrm{~mm} \times 150 \mathrm{~mm}$ ）或具有同等性能的色谱柱。  
5.3.2 锌还原柱： $4.6 \mathrm{~mm} \times 50 \mathrm{~mm}$ （填料：锌粉，锌粉平均粒径 $70 \mu \mathrm{m}$ 。锌粉还原柱可直接购买商品柱，此锌粉还原柱连接于色谱柱后）。  
5.3.3 荧光检测器：激发波长为 $326\mathrm{nm}$ ，发射波长为 $410\mathrm{nm}$ 。  
5.3.4 流动相：等度洗脱（按3.2.3配制）。  
5.3.5 流速： $0.8\mathrm{mL / min}$   
5.3.6 进样量： $20~\mu \mathrm{L}$   
5.3.7 标准溶液的色谱图见附录A中图A.1。

# 5.4 标准曲线的制作

分别将MK-4、MK-7、MK-9标准系列溶液按照浓度由低到高的顺序注入液相色谱仪中，以测得的峰面积为纵坐标，以MK-4、MK-7、MK-9各组分含量 $(\mu \mathrm{g} / \mathrm{mL})$ 为横坐标，制作标准曲线。

# 5.5 试样溶液的测定

将制备的试样溶液注入液相色谱仪中，得到峰面积，根据标准曲线得到制备的试样溶液中MK-4、

MK-7、MK-9各组分的含量 $(\mu \mathrm{g} / \mathrm{mL})$ 。

# 6 分析结果的表述

固体、半固体试样的质量 $(m_{\mathrm{s}})$ 按公式(1)计算。

$$
m _ {\mathrm {s}} = \frac {m _ {1} \times m _ {3}}{m _ {2}} \tag {1}
$$

式中：

$m_{\mathrm{s}}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——均质样品的质量，单位为克(g)；

$m_{3}$ ——称取的匀浆混合样品的质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——均质加水后的溶液的质量，单位为克 $(\mathrm{g})$ 。

试样中维生素 $\mathrm{K}_2$ 各组分的含量 $X(\mathrm{MK - 4},\mathrm{MK - 7},\mathrm{MK - 9})$ 按公式(2)计算。

$$
X _ {i} = \frac {c \times V \times f}{m _ {\mathrm {s}}} \times 1 0 0 0 \tag {2}
$$

式中：

$X_{i}$ ——试样中维生素 $\mathrm{K}_{2}$ 各组分的含量（分别是 $X_{\mathrm{MK - 4}}$ 、 $X_{\mathrm{MK - 7}}$ 、 $X_{\mathrm{MK - 9}}$ ），单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c$ ——试样溶液中维生素 $\mathrm{K}_2$ 各组分的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样的稀释倍数；

$m_{\mathrm{s}}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当试样的质量 $(m_{\mathrm{s}})$ 为 $2\mathrm{g}$ 时，四烯甲萘醌(MK-4)方法检出限为 $70.0~\mu \mathrm{g / kg}$ ，定量限为 $200~\mu \mathrm{g / kg}$ ；七烯甲萘醌(MK-7)方法检出限为 $70.0~\mu \mathrm{g / kg}$ ，定量限为 $200~\mu \mathrm{g / kg}$ ；九烯甲萘醌(MK-9)方法检出限为 $160~\mu \mathrm{g / kg}$ ，定量限为 $510~\mu \mathrm{g / kg}$ 。

# 蜂产品中头孢类药物残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了蜂蜜、蜂王浆和蜂王浆冻干粉中头孢氨苄、头孢拉定、头孢唑林、头孢哌酮、头孢乙腈、头孢匹林、头孢洛宁、头孢喹肟、头孢噻肟残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于蜂蜜、蜂王浆和蜂王浆冻干粉中头孢氨苄、头孢拉定、头孢唑林、头孢哌酮、头孢乙腈、头孢匹林、头孢洛宁、头孢喹肟、头孢噻肟残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中残留的头孢类药物，经磷酸盐缓冲溶液提取，亲水亲脂平衡固相萃取柱净化，液相色谱-串联质谱法测定，基质校准外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水

# 5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.3 甲酸（HCOOH）：色谱纯。

5.1.4 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$

5.1.5 氢氧化钠(NaOH)。

# 5.2 溶液配制

5.2.1 $2.5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：取氢氧化钠 $50\mathrm{g}$ ，加水溶解并稀释至 $500~\mathrm{mL}$ 。  
5.2.2 $30\%$ 乙腈溶液：取乙腈 $30~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$   
5.2.3 $0.05\mathrm{mol} / \mathrm{L}$ 磷酸盐缓冲溶液 $(\mathrm{pH} = 8.5)$ ：取磷酸二氢钾 $6.8\mathrm{g}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ ，用 $2.5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液调节 $\mathrm{pH}$ 至8.5。  
5.2.4 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。  
5.2.5 $0.1\%$ 甲酸溶液-甲醇：取 $0.1\%$ 甲酸溶液 $95~\mathrm{mL}$ ，加甲醇 $5\mathrm{mL}$ ，混匀。

# 5.3 标准品

头孢氨苄、头孢拉定、头孢唑林、头孢哌酮、头孢乙腈、头孢匹林、去乙酰基头孢匹林、头孢洛宁、头孢喹肟、头孢噻肟，含量均 $\geq 95.0\%$ ，具体信息见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液：取标准品各 $10\mathrm{mg}$ ，精密称定，加 $30\%$ 乙腈溶液适量使溶解并定容至 $25~\mathrm{mL}$ 容量瓶，配制成浓度为 $400~\mu \mathrm{g / mL}$ 的溶液。 $-18^{\circ}C$ 以下避光保存，有效期1个月。  
5.4.2 混合标准储备液：分别准确移取各标准储备液 $0.25\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶，用 $30\%$ 乙腈溶液稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的混合标准储备液。一 $18^{\circ}C$ 以下避光保存，有效期7d。  
5.4.3 混合标准工作液：准确移取混合标准储备液适量，用 $0.1\%$ 甲酸溶液-甲醇稀释成浓度为 $2.5~\mu \mathrm{g / L},5.0~\mu \mathrm{g / L},20~\mu \mathrm{g / L},100~\mu \mathrm{g / L},200~\mu \mathrm{g / L}$ 和 $500~\mu \mathrm{g / L}$ 的系列混合标准工作溶液。现配现用。  
5.5 材料  
5.5.1固相萃取小柱：亲水亲脂平衡型固相萃取柱， $500\mathrm{mg / 6mL}$ ，或相当者。  
5.5.2 滤膜：尼龙材质，孔径 $0.22\mu \mathrm{m}$ 或性能相当者。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 氮吹仪。  
6.4 固相萃取装置。  
6.5 涡旋混合器。  
6.6 离心机：最大转速 $6000\mathrm{r / min}$ 或以上。  
6.7 离心管：聚丙烯塑料离心管， $10\mathrm{mL},50\mathrm{mL}$   
6.8 pH计。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或冷藏的空白或供试蜂产品，如无结晶，将其搅拌混合均匀，有结晶，则置于不超过 $60^{\circ}\mathrm{C}$ 的水浴中温热，振荡，待样品全部融化后搅匀，并冷却至室温。蜂王浆和蜂王浆冻干粉从冷冻环境中取出，搅拌均匀后取样。

a）取均匀后的供试样品，作为供试试样；  
b）取均匀后的空白样品，作为空白试样；  
c）取均匀后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

# 8.1.1 蜂蜜

取试料 $5\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管中，加 $0.05\mathrm{mol} / \mathrm{L}$ 磷酸盐缓冲溶液 $25~\mathrm{mL}$ ，溶解样品，涡旋混匀后用 $2.5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液调节 $\mathrm{pH}$ 至8.5，备用。

# 8.1.2 蜂王浆和蜂王浆冻干粉

取蜂王浆 $2.5\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ )或蜂王浆冻干粉 $1.0\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管中，加 $0.05\mathrm{mol} / \mathrm{L}$ 磷酸盐缓冲溶液 $25~\mathrm{mL}$ ，涡旋混匀后用 $2.5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液调节 $\mathsf{pH}$ 至8.5，以 $6000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液，备用。

# 8.2 净化

取固相萃取柱，依次用甲醇 $5\mathrm{mL},0.05\mathrm{mol} / \mathrm{L}$ 磷酸盐缓冲溶液 $10~\mathrm{mL}$ 活化，取备用液，过柱，待液面

到达柱床表面时再依次用磷酸盐缓冲溶液 $3\mathrm{mL}$ 和水 $2\mathrm{mL}$ 淋洗，弃去全部流出液。用乙腈 $3\mathrm{mL}$ 洗脱，收集洗脱液于 $10\mathrm{mL}$ 离心管中，洗脱液在 $40^{\circ}\mathrm{C}$ 水浴中氮气吹干，加 $0.1\%$ 甲酸溶液-甲醇 $1.0\mathrm{mL}$ 溶解，过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱测定。

# 8.3 基质匹配标准曲线的制备

取空白试料样依次按8.1和8.2处理， $40^{\circ}\mathrm{C}$ 水浴氮气吹干，分别加系列混合标准工作溶液 $1.0\mathrm{mL}$ 溶解残渣，过 $0.22\mu \mathrm{m}$ 滤膜，制备 $2.5\mu \mathrm{g} / \mathrm{L}, 5.0\mu \mathrm{g} / \mathrm{L}, 20\mu \mathrm{g} / \mathrm{L}, 100\mu \mathrm{g} / \mathrm{L}, 200\mu \mathrm{g} / \mathrm{L}$ 和 $500\mu \mathrm{g} / \mathrm{L}$ 的系列基质匹配标准工作溶液，供液相色谱-串联质谱测定。以定量离子对峰面积为纵坐标、标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.4.3 测定法

取试料溶液和基质匹配标准溶液, 作单点或多点校准, 按外标法以色谱峰面积定量。基质匹配标准溶液及试料溶液中目标药物的特征离子质量色谱峰峰面积均应在仪器检测的线性范围之内。试料液中待测物质的保留时间与基质匹配标准工作液中待测物质的保留时间之比, 偏差在 $\pm 2.5\%$ 以内, 且试料溶液中的离子相对丰度与基质匹配标准溶液中的离子相对丰度相比, 符合表3的要求, 则可判定为样品中存在对应的待测物质。标准溶液多反应监测色谱图见附录B。

# 8.5 空白试验

取空白试料，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {s}} \times A \times V \times 1 0 0 0}{A _ {\mathrm {s}} \times m \times 1 0 0 0} \dots \tag {1}
$$

式中：

$X$ ——试样中待测药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{s}}$ ——标准溶液中待测药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$A$ ——试样溶液中待测药物的峰面积；

$A_{s}$ ——标准溶液中待测药物的峰面积；

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

注：头孢匹林残留量以头孢匹林和去乙酰基头孢匹林之和计。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法中头孢哌酮、头孢乙腈和头孢唑林3种药物在蜂蜜、蜂王浆和蜂王浆冻干粉中的检测限分别为 $2.0\mu \mathrm{g / kg},4.0\mu \mathrm{g / kg}$ 和 $10~\mu \mathrm{g / kg}$ ，定量限分别为 $4.0~\mu \mathrm{g / kg},8.0~\mu \mathrm{g / kg}$ 和 $20~\mu \mathrm{g / kg}$ ；其余头孢类药物和去乙酰基头孢匹林在蜂蜜、蜂王浆和蜂王浆冻干粉中的检测限分别为 $0.5\mu \mathrm{g / kg},1.0\mu \mathrm{g / kg}$ 和 $2.5\mu \mathrm{g / kg}$ ，定量限分别为 $1.0\mu \mathrm{g / kg},2.0\mu \mathrm{g / kg}$ 和 $5.0~\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法在 $1.0\mu \mathrm{g / kg}\sim 40~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

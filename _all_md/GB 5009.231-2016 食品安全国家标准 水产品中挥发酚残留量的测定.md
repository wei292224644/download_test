# 食品安全国家标准

# 水产品中挥发酚残留量的测定

2016-08-31 发布

2017-03-01 实施

# 前言

本标准代替SC/T3031—2006《水产品中挥发酚残留量的测定 分光光度法》。

本标准与SC/T3031—2006相比，主要变化如下：

——将处理方法中的搅拌改为恒温磁力搅拌；

——增加水蒸气蒸馏仪为处理方法中的提取装置。

# 食品安全国家标准

# 水产品中挥发酚残留量的测定

# 1 范围

本标准规定了水产品中挥发酚残留量的分光光度测定方法。

本标准适用于水产品中可食部分挥发酚残留量的测定。

# 2 原理

用碱性溶液破坏样品组织结构，在酸性条件下用水蒸气蒸馏出挥发酚类化合物，在铁氰化钾存在下，与4-氨基安替比林反应生成橙红色的安替比林染料，用三氯甲烷萃取，在 $460~\mathrm{nm}$ 波长测定吸光度定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 三氯甲烷 $\mathrm{(CHCl_3)}$   
3.1.2 氨水 $\mathrm{(NH_3\bullet H_2O)}$ 。  
3.1.3 硫酸 $\mathrm{(H_2SO_4)}$   
3.1.4 氢氧化钠(NaOH)。  
3.1.5 铁氰化钾 $(\mathrm{K}_3[\mathrm{Fe}(\mathrm{CN})_6])$ 。  
3.1.6 硫酸铜 $(\mathrm{CuSO}_4\cdot 5\mathrm{H}_2\mathrm{O})$ 。  
3.1.7 甲基橙 $(\mathrm{C_{14}H_{15}N_3NaO_3S})$ 。  
3.1.8 氯化铵 $\mathrm{(NH_4Cl)}$ 。  
3.1.9 4-氨基安替比林 $(\mathrm{C_{11}H_{13}N_3O})$ 。  
3.1.10 活性炭粉末（粒径 $0.1\mathrm{mm}\sim 0.5\mathrm{mm}$ ）。

# 3.2 试剂配制

3.2.1 无酚水：取实验用水，每升水中加入 $0.2\mathrm{g}$ 经 $200^{\circ}\mathrm{C}$ 烘干 $30\mathrm{min}$ 的活性炭粉末，充分振摇后，放置过夜，用双层中速滤纸过滤后即可使用；或向实验用水中加氢氧化钠（NaOH）使水呈强碱性，并滴加高锰酸钾溶液至紫红色，移入全玻璃蒸馏器中加热蒸馏，取馏出液备用。无酚水应贮于玻璃瓶中，取用时应避免与硅胶制品(橡皮塞或乳胶管等)接触。  
3.2.2 $50\%$ 硫酸溶液：将浓硫酸在搅拌下缓缓加入等体积的实验用水中。  
3.2.3 $10\%$ 氢氧化钠溶液：称取 $10\mathrm{g}$ 氢氧化钠溶于水中，稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.4 $8 \%$ 铁氰化钾溶液：称取 $8 \mathrm{~g}$ 铁氰化钾 $\left(\mathrm{K}_{3}\left[\mathrm{Fe}(\mathrm{CN})_{6}\right]\right)$ 溶于水中，稀释至 $100 \mathrm{~mL}$ ，混匀， $4^{\circ} \mathrm{C}$ 冷藏，可使用一周。

3.2.5 $10\%$ 硫酸铜溶液：称取 $10\mathrm{g}$ 硫酸铜 $(\mathrm{CuSO}_4\cdot 5\mathrm{H}_2\mathrm{O})$ 溶于水中，稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.6 $0.05\%$ 甲基橙指示液：称取甲基橙 $0.05\mathrm{g}$ 溶于水中，稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.7 缓冲溶液：称取 $20\mathrm{g}$ 氯化铵 $\mathrm{(NH_4Cl)}$ 溶于 $100~\mathrm{mL}$ 氨水 $(\mathrm{NH}_3\bullet \mathrm{H}_2\mathrm{O})$ 中， $\mathrm{pH}$ 约 $10.7,4^{\circ}C$ 冷藏。  
3.2.8 $2\%$ 4-氨基安替比林溶液：称取 $2\mathrm{g}4-$ 氨基安替比林 $\left(\mathrm{C}_{11}\mathrm{H}_{13}\mathrm{N}_3\mathrm{O}\right)$ 溶于水中，稀释至 $100~\mathrm{mL},4^{\circ}\mathrm{C}$ 冷藏，可使用一周。

# 3.3 标准品

苯酚标准品 $(\mathrm{C}_6\mathrm{H}_6\mathrm{O})$ ，纯度 $\geq 99\%$

# 3.4 标准溶液配制

3.4.1 标准储备溶液: 称取苯酚 $0.01 \mathrm{~g}$ (精确至 $0.1 \mathrm{mg}$ ) 于 $50 \mathrm{~mL}$ 小烧杯中, 加无酚水溶解, 再转移到 $100 \mathrm{~mL}$ 容量瓶中, 定容, 混匀, 于 $4^{\circ} \mathrm{C}$ 保存, 苯酚质量浓度为 $100 \mu \mathrm{g} / \mathrm{mL}$ , 有效期为 $7 \mathrm{~d}$ 。  
3.4.2 标准工作液：取适量苯酚标准溶液，用无酚水稀释至每毫升含 $1.00\mu \mathrm{g}$ 苯酚，配制后要在 $2\mathrm{h}$ 内使用。

# 4 仪器和设备

4.1 水蒸气蒸馏仪或 $500~\mathrm{mL}$ 玻璃水蒸气蒸馏装置（见附录A）。  
4.2 天平：感量为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$   
4.3 $500~\mathrm{mL}$ （梨形）分液漏斗。  
4.4 磁力搅拌器。  
4.5 分光光度计。  
4.6 调温电炉或调温电加热套。  
4.7 组织捣碎机。

# 5 分析步骤

# 5.1 试样制备及保存

水产品取可食部分用组织捣碎机匀浆后，搅拌均匀，分装，密闭冷藏或冷冻保存。

# 5.2 试样处理

# 5.2.1 蒸馏提取

称取预处理的样品 $10\mathrm{g}\sim 20\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ，同时做空白试验）放烧杯中，加入氢氧化钠溶液 $75~\mathrm{mL}$ ，用磁力搅拌器搅拌 $10\mathrm{min}$ 。再加 $3\sim 5$ 滴甲基橙溶液，用硫酸溶液调节 $\mathrm{pH}\leqslant 4$ （溶液呈橙红色），再加硫酸铜溶液 $5\mathrm{mL}$ ，混匀后，置于水蒸气蒸馏仪提取瓶或 $500~\mathrm{mL}$ 蒸馏瓶中，用少量无酚水洗净烧杯，洗涤液并入水蒸气蒸馏仪提取瓶或蒸馏瓶中并加数粒玻璃珠。

使用玻璃水蒸气蒸馏装置(或水蒸气蒸馏仪)进行蒸馏，应连接好水蒸气蒸馏装置，确保严密不漏气，取 $250~\mathrm{mL}$ 锥形瓶接收馏出液，中速蒸馏 $60\mathrm{min}\sim 90\mathrm{min}$ ，至馏出液约 $240~\mathrm{mL}$ 时，停止蒸馏，并用 $10~\mathrm{mL}$ 无酚水冲洗冷凝管，洗涤液并入馏出液中(蒸馏过程中保持酸性)。蒸馏液应尽量避免与橡胶制品接触(橡皮塞、橡胶管和乳胶管里含有酚类化合物添加剂)，如需使用时用硅胶管(塞)代替。

# 5.2.2 显色

将馏出液移入分液漏斗中，加 $2.0\mathrm{mL}$ 缓冲溶液，混匀。加 $1.5\mathrm{mL}4$ -氨基安替比林溶液，混匀，再加

$1.5\mathrm{mL}$ 铁氰化钾溶液，充分混匀后，放置 $10\mathrm{min}$

# 5.2.3 萃取

准确加入 $10.0\mathrm{mL}$ 三氯甲烷，密塞，剧烈振摇 $2\mathrm{min}$ ，静置分层，彻底澄清透明后（不能存有气泡），用干脱脂棉拭干分液漏斗颈管内壁，于颈管内塞入一小团干脱脂棉，将三氯甲烷通过干脱脂棉团，弃去最初滤出的数滴萃取液，直接放入光程为 $1\mathrm{cm}$ 的比色皿中。安替比林染料用三氯甲烷萃取后，需 $6\mathrm{h}$ 以内测定。

# 5.2.4 测定

于 $460~\mathrm{nm}$ 波长下，以空白三氯甲烷萃取液为参比，用分光光度计测量吸光度。

# 5.2.5 标准曲线绘制

取8个分液漏斗，分别加入 $100~\mathrm{mL}$ 水，依次加入 $0.0~\mathrm{mL}, 0.5~\mathrm{mL}, 1.0~\mathrm{mL}, 3.0~\mathrm{mL}, 5.0~\mathrm{mL},$ $7.0~\mathrm{mL}, 10.0~\mathrm{mL}, 15.0~\mathrm{mL}$ 苯酚标准溶液，再分别加无酚水至 $250~\mathrm{mL}$ ，显色、萃取后，测定吸光度绘制标准曲线。

# 6 分析结果的表述

试样中挥发酚的含量按式(1)计算：

$$
X = \frac {m _ {1} - m _ {0}}{m} \tag {1}
$$

式中：

$X$ ——试样中挥发酚的含量(以苯酚计)，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$m_{1}$ ——由标准曲线求得试样溶液中苯酚的量，单位为微克 $(\mu \mathrm{g})$ ；

$m_0$ ——空白的苯酚量，单位为微克 $(\mu \mathrm{g})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8检出限

本方法检出限： $0.05\mathrm{mg / kg}$

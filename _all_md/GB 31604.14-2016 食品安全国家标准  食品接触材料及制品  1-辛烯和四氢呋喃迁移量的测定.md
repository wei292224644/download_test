# 前言

本标准代替GB/T23296.4—2009《食品接触材料 高分子材料 食品模拟物中1-辛烯和四氢呋喃的测定 气相色谱法》。

本标准与GB/T23296.4—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 1-辛烯和四氢呋喃迁移量的测定”；

——增加了气相色谱-质谱法；  
——修改了食品模拟物试液的制备；  
——修改了分析结果的表述；  
——修改了检出限和定量限。

# 食品安全国家标准

# 食品接触材料及制品

# 1-辛烯和四氢呋喃迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中1-辛烯和四氢呋喃迁移量的测定方法。

本标准适用于食品接触材料及制品中1-辛烯和四氢呋喃迁移量的测定。

# 第一法 气相色谱法

# 2 原理

食品模拟物中1-辛烯、四氢呋喃经顶空进样后，在色谱柱中与内标物及其他组分分离，用氢火焰离子化检测器检测，以内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 $N,N$ -二甲基乙酰胺 $\mathrm{(DMAC,C_4H_9NO)}$ ：纯度 $>99.0\%$   
3.1.2 水基、酸性、酒精类、油基食品模拟物：所用试剂依据GB31604.1的规定。

# 3.2 试剂配制

3.2.1 水基、酸性、酒精类、油基食品模拟物：按GB5009.156操作。

# 3.3 标准品

3.3.1 1-辛烯 $\left(\mathrm{C}_{8} \mathrm{H}_{16}\right)$ , CAS号: $111-66-0$ , 纯度 $\geqslant 99.5\%$ , 或经国家认证并授予标准物质证书的标准物质。  
3.3.2 异辛烷 $\left(\mathrm{C}_{8} \mathrm{H}_{18}, \mathrm{CAS}\right.$ 号：540-84-1)，纯度 $\geqslant 99.5\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.3 四氢呋喃（THF， $\mathrm{C_4H_8O}$ ，CAS号：109-99-9），纯度 $\geq 98.0\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.4 四氢吡喃（THP, $\mathrm{C}_5\mathrm{H}_{10}\mathrm{O}$ , CAS号：142-68-7），纯度 $\geq 99.9\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

0.0001g)。再加入1-辛烯 $100\mathrm{mg}$ ，约 $150~\mu \mathrm{L}$ ，重新称量(精确至 $0.0001\mathrm{g}$ )。用DMAC定容。精确计算1-辛烯的浓度。于 $4^{\circ}C$ 避光保存，有效期3个月。  
3.4.2 1-辛烯标准中间溶液 $(200\mu \mathrm{g / mL})$ ：准确移取1-辛烯标准储备液 $1.0~\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用DMAC定容。保存条件同3.4.1。  
3.4.3 1-辛烯水基、酸性、酒精类食品模拟物介质标准工作溶液：用移液管分别移取 $1.0\mathrm{mL}$ 水基、酸性、酒精类食品模拟物于6个顶空瓶中，立即用隔垫和铝盖密封。用微量注射器透过隔垫依次加入 $200~\mu \mathrm{L}$ 异辛烷内标中间溶液，混匀。按附录A中表A.1用微量注射器透过隔垫向瓶中加入1-辛烯标准中间溶液及DMAC，混匀。

3.4.6 异辛烷内标中间溶液 $(120\mu \mathrm{g / mL})$ ：移取 $3.0~\mathrm{mL}$ 异辛烷内标储备液于 $50~\mathrm{mL}$ 容量瓶中，用DMAC定容。保存条件同3.4.1。  
3.4.7 水基、酸性、酒精类 THF 标准储备液 $(2.2\mathrm{mg / mL})$ ：量取 $99~\mathrm{mL}$ 水于 $100~\mathrm{mL}$ 容量瓶中，带塞称量（精确至 $0.0001\mathrm{g}$ ）。加入 $220\mathrm{mg}$ THF，约 $250~\mu \mathrm{L}$ ，重新称重（精确至 $0.0001\mathrm{g}$ ），用水定容。精确计算THF的浓度。于 $4^{\circ}\mathrm{C}$ 避光保存，有效期1个星期。  
3.4.8 水基、酸性、酒精类THF标准中间溶液 $(22\mu \mathrm{g} / \mathrm{mL})$ ：量取 $90~\mathrm{mL}$ 水于 $100~\mathrm{mL}$ 容量瓶中，再移入 $1.0~\mathrm{mL}$ 水基THF标准储备液，用水定容。保存条件同3.4.7。  
3.4.9 THF水基、酸性、酒精类食品模拟物介质标准工作溶液：分别量取 $45~\mathrm{mL}$ 水基、酸性、酒精类食品模拟物于6个 $50~\mathrm{mL}$ 容量瓶中。按表A.2加入一定体积的水基、酸性、酒精类THF标准中间溶液，用对应的食品模拟物定容。

3.4.10 水基、酸性、酒精类THP内标储备液 $(1\mathrm{mg / mL})$ ：量取 $99~\mathrm{mL}$ 水于 $100~\mathrm{mL}$ 容量瓶中，带塞称量(精确至 $0.0001\mathrm{g}$ )，加入 $100\mathrm{mg}$ THP，约 $150~\mu \mathrm{L}$ ，重新称量(精确至 $0.0001\mathrm{g}$ )，用水定容。精确计算THP的浓度。保存条件同3.4.7。  
3.4.11 水基、酸性、酒精类THP内标中间溶液 $(70\mu \mathrm{g} / \mathrm{mL})$ ：量取 $90~\mathrm{mL}$ 水于 $100~\mathrm{mL}$ 容量瓶中，再移入 $7.0~\mathrm{mL}$ 水基THP内标储备液，用水定容。保存条件同3.4.7。

3.4.14 THF橄榄油介质标准工作溶液：分别称取 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )橄榄油于6个顶空瓶中，立即用隔垫和铝盖密封。用微量注射器移取 $100~\mu \mathrm{L}$ 油性THP内标中间溶液透过隔垫分别加入各顶空瓶中。按表A.3所示，向每个顶空瓶中加入一定体积的油性THF标准中间溶液和DMAC，配制与表A.3所列浓度相近的标准工作溶液。

# 4 仪器和设备

4.1 气相色谱仪：配备自动顶空进样器和氢火焰离子化检测器(FID)。  
4.2 顶空瓶： $20~\mathrm{mL}$ ，配备铝盖和丁基橡胶或硅树脂橡胶隔垫，隔垫接触样品一面应涂有聚四氟乙烯。注：THF可用于制造内涂聚四氟乙烯的丁基橡胶隔垫，这将对THF的分析产生严重干扰。因此在试验前应对隔垫进行净化处理，用蒸馏水至少煮沸 $15\mathrm{min}$ ，再晾干。已净化未使用的隔垫，下次使用时应再次净化。  
4.3 容量瓶： $10~\mathrm{mL},25~\mathrm{mL},50~\mathrm{mL},100~\mathrm{mL}$   
4.4 微量注射器： $25\mu \mathrm{L}, 50\mu \mathrm{L}, 100\mu \mathrm{L}, 250\mu \mathrm{L}, 500\mu \mathrm{L}$ 。  
4.5 移液管： $1\mathrm{mL},2\mathrm{mL},3\mathrm{mL},5\mathrm{mL},10\mathrm{mL}$   
4.6 分析天平：感量 $0.0001\mathrm{g}, 0.01\mathrm{g}$ 。

# 5 分析步骤

# 5.1 试样制备

按照GB5009.156及GB31604.1的要求，对样品进行迁移试验，得到食品模拟物试液。如果得到的食品模拟物试液不能马上进行下一步试验，应将食品模拟物试液于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中避光保存。

应将所得食品模拟物试液冷却或恢复至室温后进行下一步试验。

# 5.2 试液制备

# 5.2.1 用于测定1-辛烯的水基、酸性、酒精类食品模拟物试液的制备

从迁移试验中移取 $1.0\mathrm{mL}$ 水基食品模拟物于顶空瓶中，立即用隔垫和铝盖密封。用微量注射器透过隔垫依次加入 $200~\mu \mathrm{L}$ 异辛烷内标中间溶液和 $200~\mu \mathrm{L}$ DMAC，混匀。

# 5.2.2 用于测定1-辛烯的橄榄油介质食品模拟物试液的制备

从迁移试验中称取 $1\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )橄榄油介质食品模拟物至顶空瓶中，以下按5.2.1“立即用隔热垫和铝盖密封……”的操作步骤进行处理。

# 5.2.3 用于测定四氢呋喃的水基、酸性、酒精类食品模拟物试液的制备

从迁移试验中移取 $10.0\mathrm{mL}$ 水基、酸性、酒精类食品模拟物于顶空瓶中，立即用隔垫和铝盖密封。用微量注射器透过隔垫加入 $100~\mu \mathrm{L}$ 水基、酸性、酒精类THP内标中间溶液，混匀。

# 5.2.4 用于测定四氢呋喃的橄榄油介质食品模拟物试液的制备

从迁移试验中称取 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )橄榄油介质食品模拟物于顶空瓶中，立即用隔垫和铝盖密封。用微量注射器透过隔垫加入 $100~\mu \mathrm{L}$ 油性THP内标中间溶液(3.4.16)，再用微量注射器迅速加入 $300~\mu \mathrm{L}$ DMAC，混匀。

# 5.3 空白试液的制备

按照5.2所述步骤处理没有与食品接触材料接触的食品模拟物。

# 5.4 测定

# 5.4.1 气相色谱测定参考条件

顶空进样器条件列出如下：

a）样品平衡时间： $60\mathrm{min}$   
b）顶空瓶温度： $70^{\circ}\mathrm{C}$   
c）进样针温度： $70^{\circ}\mathrm{C}$   
d）平衡罐转速： $500\mathrm{r / min}$

气相色谱条件列出如下：

a）色谱柱： $(5\%)$ -苯基)-甲基聚硅氧烷石英毛细管柱，柱长 $30.0\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ 或性能类似的分析柱；  
b）程序升温：测定1-辛烯时， $60^{\circ}\mathrm{C}$ 保持 $6.5\mathrm{min}$ ，再以 $30^{\circ}\mathrm{C / min}$ 的速率升温至 $240^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ ；测定四氢呋喃时， $50^{\circ}\mathrm{C}$ 保持 $5\mathrm{min}$ ；  
c）进样口温度： $150~\mathrm{^\circ C}$   
d）检测器温度： $300^{\circ}\mathrm{C}$   
e）载气：氮气；  
f）载气流速：测定1-辛烯时为 $1.0\mathrm{mL / min}$ ；测定四氢呋喃时为 $2.5\mathrm{mL / min}$   
g）进样方式：分流进样，分流比 $10:1$   
h）进样量： $250~\mu \mathrm{L}$   
i） 氢气流速： $30~\mathrm{mL / min}$   
j）空气流速： $400~\mathrm{mL / min}$

# 5.4.2 绘制标准工作曲线

按照5.4.1所列测定条件，对标准工作溶液依次进样测定。以标准工作溶液中1-辛烯或四氢呋喃浓度为横坐标，单位以“ $\mathrm{mg / L}$ 或 $\mathrm{mg / kg}$ ”表示，以1-辛烯/异辛烷或四氢呋喃/四氢吡喃的峰面积比值为纵坐标，绘制标准曲线。标准色谱图参见图B.1和图B.2。

# 6 分析结果的表述

由标准曲线得到试样溶液中1-辛烯或四氢呋喃的浓度，按GB5009.156进行迁移量计算，得到食品接触材料及制品中1-辛烯或四氢呋喃的迁移量。计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法的水基、酸性、酒精类模拟物中的1-辛烯和四氢呋喃检出限分别为 $2.0\mathrm{mg / L}$ 和 $0.1\mathrm{mg / L}$ 橄榄油模拟物中的1-辛烯和四氢呋喃检出限分别为 $2.0\mathrm{mg / kg}$ 和 $0.1\mathrm{mg / kg}$ 。

本方法的水基、酸性、酒精类模拟物中的1-辛烯和四氢呋喃定量限分别为 $6.0\mathrm{mg / L}$ 和 $0.3\mathrm{mg / L}$

橄榄油模拟物中的1-辛烯和四氢呋喃定量限分别为 $6.0\mathrm{mg / kg}$ 和 $0.3\mathrm{mg / kg}$ 。

# 第二法 气相色谱-质谱法

# 9 原理

食品模拟物中1-辛烯、四氢呋喃经顶空进样后，在色谱柱中与内标物及其他组分分离，用质谱检测器检测，以内标法定量。

# 12.4 测定

# 12.4.1 气相色谱-质谱测定参考条件

顶空进样器条件列出如下：

a）样品平衡时间： $60\mathrm{min}$   
b）顶空瓶温度： $70^{\circ}C$   
c）进样针温度： $70^{\circ}\mathrm{C}$   
d）平衡罐转速： $500\mathrm{r / min}$ □

气相色谱-质谱条件列出如下：

a) 色谱柱: $(5\%)$ -苯基)-甲基聚硅氧烷石英毛细管柱, 柱长 $30.0 \mathrm{~m}$ , 内径 $0.25 \mathrm{~mm}$ , 膜厚 $0.25 \mu \mathrm{m}$ (质谱专用柱)或性能类似的分析柱;  
b）程序升温：初始温度 $38^{\circ}\mathrm{C}$ ，保持 $7\mathrm{min}$ ， $40^{\circ}\mathrm{C} / \mathrm{min}$ 升至 $240^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ ；  
c）进样口温度： $150~\mathrm{^\circ C}$

d) 离子源温度: $230^{\circ} \mathrm{C}$ ;   
e) 载气：氦气， $0.9\mathrm{mL} / \mathrm{min}$ ；  
f）进样方式：分流进样，分流比 $10:1$   
g）进样量： $250~\mu \mathrm{L}$   
h）传输线温度： $250~\mathrm{^\circ C}$   
i） 电离方式：EI；  
j）溶剂延迟： $2.2\mathrm{min}$   
k）质谱扫描方式：选择离子模式(SIM)，8min后关闭灯丝；  
1) 异辛烷定量离子 $m / z57$ ，定性离子 $m / z58,99$ ；1-辛烯定量离子 $m / z70$ ，定性离子 $m / z83$ 、112；四氢呋喃定量离子 $m / z72$ ，定性离子 $m / z71,73$ ；四氢吡喃定量离子 $m / z85$ ，定性离子 $m / z67,86$ 。

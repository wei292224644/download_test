# 食品安全国家标准

# 食品中三价铬和六价铬的测定

# 1 范围

本标准第一篇规定了食品中六价铬的测定方法。

本标准第一篇适用于特殊医学用途配方食品和特殊医学用途婴儿配方食品中六价铬的测定。

本标准第二篇规定了食品中三价铬的测定方法。

本标准第二篇适用于特殊医学用途配方食品和特殊医学用途婴儿配方食品中三价铬的测定。

# 第一篇 食品中六价铬的测定

# 2 原理

试样用碱性溶液振荡提取六价铬，高速离心后供液相色谱-电感耦合等离子体质谱仪测定。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 硝酸 $(\mathrm{HNO}_3)$ ：浓度 $\geq 69\%$ 。  
3.1.2 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ：浓度 $\geq 25\%$   
3.1.3 氢氧化钠 $\mathrm{(NaOH)}$ 。  
3.1.4 氦气（He）：氦气 $(\geq 99.995\%)$ 。  
3.1.5 氩气（Ar）：氩气 $(\geqslant 99.995\%)$ 或液氩。

# 3.2 试剂配制

3.2.1 $60~\mathrm{mmol / L}$ 硝酸铵溶液：吸取 $3.9~\mathrm{mL}$ 硝酸于 $1\textrm{L}$ 水中，用氨水调节 $\mathrm{pH}$ 至 $7.0\pm 0.1$   
3.2.2 $0.05\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：称取 $1\mathrm{g}$ 氢氧化钠，用水溶解并定容至 $500~\mathrm{mL}$ 。

# 3.3 标准品

六价铬标准溶液： $1000\mu \mathrm{g / mL}$ ，介质为水，为经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 六价铬标准储备液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：吸取 $1.0\mathrm{mL}$ 六价铬标准溶液 $(1000\mu \mathrm{g} / \mathrm{mL})$ ，置于 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，摇匀，转移至棕色试剂瓶中，于 $4^{\circ}\mathrm{C}$ 下避光保存，有效期为1个月。  
3.4.2 六价铬标准工作液 $(100\mu \mathrm{g} / \mathrm{L})$ ：吸取 $1.0~\mathrm{mL}$ 六价铬标准储备液，置于 $100~\mathrm{mL}$ 容量瓶中，用 $60~\mathrm{mmol / L}$ 硝酸铵溶液定容至刻度，摇匀，转移至棕色试剂瓶中。临用现配。

# 4 仪器和设备

4.1 液相色谱-电感耦合等离子体质谱仪(LC-ICP-MS)：由液相色谱仪与电感耦合等离子体质谱仪（配碰撞反应池)组成。  
4.2 天平：感量为 $0.001\mathrm{g}$ 。  
4.3 pH计：精度为0.01。  
4.4 水浴振荡器。  
4.5 冷冻离心机：离心力不低于 $13100\mathrm{g}$   
4.6 水相针式滤器： $0.45\mu \mathrm{m}$ 混合纤维素滤膜。

# 5 分析步骤

# 5.1 试样制备

5.1.1 制样和样品保存过程中，应防止样品受到污染。  
5.1.2 粉状样品、液体样品混合均匀后装入洁净容器内密封并做好标识，于 $4^{\circ}\mathrm{C}$ 下保存备用。

# 5.2 样品前处理

准确称取试样 $0.25\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，置于具塞聚丙烯离心管内，加入 $5\mathrm{mL}$ 水后涡旋分散样品，再加入 $20~\mathrm{mL}$ 氢氧化钠溶液 $(0.05\mathrm{mol} / \mathrm{L})$ ， $25^{\circ}\mathrm{C}$ 水浴振荡提取 $0.5\mathrm{h}$ ，于 $4^{\circ}\mathrm{C}$ 下 $10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上层离心清液过 $0.45\mu \mathrm{m}$ 滤膜，供液相色谱-电感耦合等离子体质谱仪测定。同时做空白试验。

# 5.3 仪器参考条件

# 5.3.1 液相色谱条件

色谱柱：Dionex AG19 保护柱 $(4.1\mathrm{mm} \times 50\mathrm{mm}, 10\mu \mathrm{m}) + \mathrm{AS19}$ 分析柱 $(4.1\mathrm{mm} \times 250\mathrm{mm}, 10\mu \mathrm{m})$ ，或性能相当的阴离子交换柱。

柱温： $25^{\circ}\mathrm{C}$ 。

流动相： $60~\mathrm{mmol / L}$ 硝酸铵， $\mathrm{pH} = 7.0$

流速： $1.0~\mathrm{mL / min}$

进样量： $50~\mu \mathrm{L}$

# 5.3.2 质谱条件

见附录A。

# 5.4 标准曲线的制作

分别吸取六价铬标准工作液 $0\mu \mathrm{L},5\mu \mathrm{L},10\mu \mathrm{L},20\mu \mathrm{L},50\mu \mathrm{L},100\mu \mathrm{L},200\mu \mathrm{L}$ ，用水定容至 $1\mathrm{mL}$ 制得六价铬质量浓度分别为 $0\mathrm{ng / mL},0.5\mathrm{ng / mL},1.0\mathrm{ng / mL},2.0\mathrm{ng / mL},5.0\mathrm{ng / mL},10.0\mathrm{ng / mL}$ 、 $20.0\mathrm{ng / mL}$ 的系列标准溶液，供液相色谱-电感耦合等离子体质谱仪测定，以六价铬的峰面积为纵坐标，六价铬的浓度为横坐标，绘制标准工作曲线。

# 5.5 试样溶液的测定

在相同条件下，将空白试验液、样品溶液注入仪器进行测定，用标准工作曲线计算出测试溶液中六

价铬的浓度。

# 6 分析结果的表述

试样中六价铬含量按照公式(1)进行计算：

$$
x _ {1} = \frac {\left(\rho - \rho_ {0}\right) \times V}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$x_{1}$ ——试样中六价铬的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——从标准工作曲线得到试样溶液中六价铬的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$\rho_0$ ——空白试验溶液中六价铬的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——提取溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

计算结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

当样品称样量为 $0.25\mathrm{g}$ ，提取溶液体积为 $25~\mathrm{mL}$ 时，该方法的六价铬检出限为 $0.02\mathrm{mg / kg}$ ，定量限为 $0.05\mathrm{mg / kg}$ 。

# 第二篇 食品中三价铬的测定

# 9 原理

试样经硝酸溶液提取三价铬，加入硫代硫酸钠溶液使三价铬不转化为六价铬，调节 $\mathrm{pH}$ 至碱性，乙二胺四乙酸二钠溶液络合，高速离心后供液相色谱-电感耦合等离子体质谱仪测定三价铬。该测定条件下，三价铬和六价铬为等量转化，测得三价铬的量扣除第一部分测定的六价铬的量即为样品中三价铬含量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 硫代硫酸钠 $\mathrm{(Na_2S_2O_3\cdot 5H_2O)}$   
10.1.2 乙二胺四乙酸二钠(EDTA-2Na)。  
10.1.3 硝酸 $(\mathrm{HNO}_3)$ ：超高纯，浓度 $\geq 69\%$ 。  
10.1.4 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ：色谱纯，浓度 $\geq 25\%$   
10.1.5 氢气（He）：氦气 $(\geq 99.995\%)$ 。

10.1.6 氩气（Ar）：氩气 $(\geq 99.995\%)$ 或液氩。

# 10.2 试剂配制

10.2.1 $0.1\mathrm{mol} / \mathrm{L}$ 硫代硫酸钠溶液：称取 $26\mathrm{g}$ 硫代硫酸钠，溶于 $1000~\mathrm{mL}$ 煮沸冷却水中备用。  
10.2.2 $60~\mathrm{mmol / L}$ 硝酸铵溶液：吸取 $3.9~\mathrm{mL}$ 硝酸于 $1\textrm{L}$ 水中，用氨水调节 $\mathrm{pH}$ 至 $7.0\pm 0.1$   
10.2.3 $20 \mathrm{mmol} / \mathrm{L}$ 乙二胺四乙酸二钠溶液：称取 $3.38 \mathrm{~g}$ 乙二胺四乙酸二钠，用硝酸铵溶液（ $60 \mathrm{mmol} / \mathrm{L}$ ）溶解并加水至近 $500 \mathrm{~mL}$ ，用氨水调节 $\mathrm{pH}$ 至 $9.0 \pm 0.1$ ，转移至 $500 \mathrm{~mL}$ 容量瓶中，用水定容。  
10.2.4 $5\%$ 硝酸溶液：称取一定量经折算为 $5\mathrm{g}$ 的硝酸置于 $100~\mathrm{mL}$ 容量瓶中，加水稀释至刻度。

# 10.3 标准品

三价铬标准溶液： $1000\mu \mathrm{g / mL}$ ，介质为盐酸，为经国家认证并授予标准物质证书的标准物质。

# 10.4 标准溶液配制

10.4.1 三价铬标准储备液 $(10\mu \mathrm{g} / \mathrm{mL})$ : 吸取 $1.0 \mathrm{~mL}$ 三价铬标准溶液 $(1000\mu \mathrm{g} / \mathrm{mL})$ , 置于 $100 \mathrm{~mL}$ 容量瓶中, 用盐酸定容至刻度, 摇匀, 转移至棕色试剂瓶中, 于 $4^{\circ} \mathrm{C}$ 下避光保存, 有效期为 1 个月。  
10.4.2 三价铬标准工作液 $(1000\mu \mathrm{g} / \mathrm{L})$ ：吸取 $10.0\mathrm{mL}$ 三价铬标准储备液，置于 $100~\mathrm{mL}$ 容量瓶中，用 $60~\mathrm{mmol / L}$ 硝酸铵溶液定容至刻度，摇匀，转移至棕色试剂瓶中。临用现配。

# 11 仪器和设备

同第4章。

# 12 分析步骤

# 12.1 试样制备

同5.1。

# 12.2 样品前处理

准确称取 $0.5\mathrm{g}$ 样品（精确至 $0.001\mathrm{g}$ ），置于具塞聚丙烯离心管内，加入 $5\mathrm{mL}5\%$ 的硝酸溶液， $80^{\circ}\mathrm{C}$ 水浴振荡 $1\mathrm{h}$ ，冷却后加入 $5\mathrm{mL}$ 硫代硫酸钠溶液 $(0.1\mathrm{mol / L})$ ，用氨水调 $\mathrm{pH}$ 至 $9.0\pm 0.1$ 后加入 $15\mathrm{mL}$ 乙二胺四乙酸二钠溶液 $(20\mathrm{mmol / L})$ ， $80^{\circ}\mathrm{C}$ 水浴振荡条件络合 $1\mathrm{h}$ ，冷却至室温后加水定容至 $25\mathrm{mL}$ ，于 $4^{\circ}\mathrm{C}10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，过 $0.45\mu \mathrm{m}$ 滤膜，上机。同时做空白试验。

# 12.3 仪器参考条件

# 12.3.1 液相色谱条件

同5.3.1。

# 12.3.2 质谱条件

见附录A。

# 12.4 标准曲线的制作

分别吸取三价铬标准工作液 $0\mu \mathrm{L},12.5\mu \mathrm{L},25\mu \mathrm{L},50\mu \mathrm{L},125\mu \mathrm{L},250\mu \mathrm{L},500\mu \mathrm{L}$ ，加入 $5\mathrm{mL}$ 硫代

硫酸钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ，用氨水调 $\mathrm{pH}$ 至 $9.0 \pm 0.1$ 后加入 $15\mathrm{mL}$ 乙二胺四乙酸二钠溶液（ $20\mathrm{mmol} / \mathrm{L}$ ）， $80^{\circ}\mathrm{C}$ 水浴振荡 $1\mathrm{h}$ 后，冷却至室温后加水定容至 $25\mathrm{mL}$ ，于 $4^{\circ}\mathrm{C}$ 下 $10000\mathrm{r} / \mathrm{min}$ 离心 $10\mathrm{min}$ ，过 $0.45\mu \mathrm{m}$ 水相滤膜，制得三价铬质量浓度分别为 $0\mathrm{ng} / \mathrm{mL}, 0.5\mathrm{ng} / \mathrm{mL}, 1.0\mathrm{ng} / \mathrm{mL}, 2.0\mathrm{ng} / \mathrm{mL}, 5.0\mathrm{ng} / \mathrm{mL}, 10.0\mathrm{ng} / \mathrm{mL}, 20.0\mathrm{ng} / \mathrm{mL}$ 的系列标准溶液，供液相色谱-电感耦合等离子体质谱仪测定，以三价铬的峰面积为纵坐标，三价铬的浓度为横坐标，绘制标准工作曲线。

# 12.5 试样溶液的测定

在相同条件下，将空白试验液、样品溶液注入仪器进行测定，用标准工作曲线计算出测试溶液中三价铬的浓度。

# 13 分析结果的表述

试样中三价铬含量按照公式(2)进行计算：

$$
x = \frac {\left(\rho - \rho_ {0}\right) \times V}{m \times 1 0 0 0} - x _ {1} \quad \dots \tag {2}
$$

式中：

$x$ ——试样中三价铬的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——从标准工作曲线得到试样溶液中三价铬的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$\rho_0$ ——空白试验溶液中三价铬的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——提取溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$x_{1}$ ——试样中六价铬的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

计算结果保留3位有效数字。

注：如采用第一篇，六价铬为未检出，则 $x_{1}$ 取0。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 15 其他

当样品称样量为 $0.5\mathrm{g}$ ，提取溶液体积为 $25~\mathrm{mL}$ 时，该方法的三价铬检出限为 $0.02\mathrm{mg / kg}$ ，定量限为 $0.05\mathrm{mg / kg}$ 。

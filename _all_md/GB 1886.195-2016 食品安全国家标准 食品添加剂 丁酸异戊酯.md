# 食品安全国家标准

# 食品添加剂 丁酸异戊酯

# 1 范围

本标准适用于以丁酸和异戊醇为原料经化学反应制得的食品添加剂丁酸异戊酯。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

丁酸3-甲基丁酯、丁酸2-甲基丁酯

# 2.2 分子式

$\mathrm{C_9H_{18}O_2}$

# 2.3 结构式

丁酸3-甲基丁酯

![](images/b9b1188d0b68ad9f8859711689f1cd48da2355d3d25fafe0f85532e8478aee3a.jpg)  
丁酸2-甲基丁酯

# 2.4 相对分子质量

158.24（按2007年国际相对原子质量）

# 附录A

# 丁酸异戊酯含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行，应符合要求。食品添加剂丁酸异戊酯气相色谱图及操作条件参见附录B

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ 。  
B.2.2 固定相： $100\%$ 二甲基聚硅氧烷。  
B.2.3 膜厚: $0.25 \mu \mathrm{m}$ 。  
B.2.4 色谱炉温度：线性程序升温从 $50~\mathrm{^\circ C}$ 至 $220^{\circ}C$ ，速率 $3^{\circ}C / \min$   
B.2.5 进样口温度： $280^{\circ}C$ 。  
B.2.6 检测器温度： $280^{\circ}C$ 。  
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 柱前压： $45\mathrm{kPa}$   
B.2.10 进样量： $0.1\mu \mathrm{L}$   
B.2.11 分流比： $100:1$

# 液相色谱-质谱联用法

# 1 范围

本标准规定了鸡蛋中氟虫腈及其代谢物残留量的液相色谱-质谱联用测定方法。

本标准适用于鸡蛋中氟虫腈及其代谢物残留量的测定。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样用乙腈提取，提取液经分散固相萃取净化，液相色谱-质谱联用仪检测，外标法定量。

# 4 试剂和材料

除非另有说明，在分析中仅使用分析纯的试剂，水为GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN, CAS}$ 号：75-05-8）：色谱纯。

4.1.2 甲酸(HCOOH,CAS号:64-18-6):色谱纯。

4.1.3 甲醇 $\mathrm{(CH_3OH, CAS}$ 号：67-56-1）：色谱纯。

4.1.4 乙酸铵 $\mathrm{(CH_3COONH_4}$ ,CAS号：631-61-8)：色谱纯。

4.1.5 无水硫酸镁 $\mathrm{(MgSO_4}$ ,CAS号：7487-88-9)。

4.1.6 氯化钠(NaCl，CAS号：7647-14-5)。

4.1.7 无水硫酸钠 $\mathrm{(Na_2SO_4}$ ,CAS号：7757-82-6)。

# 4.2 溶液配制

4.2.1 甲酸溶液 $(0.1\%)$ ：取 $1\mathrm{mL}$ 甲酸用水稀释至 $1000~\mathrm{mL}$ ，摇匀。

4.2.2 乙酸铵-甲酸溶液(5mmol/L)：称取 $0.3854\mathrm{g}$ 乙酸铵，用 $0.1\%$ 甲酸溶液溶解并稀释至1000mL,摇匀。

# 4.3 标准品

氟虫腈及其代谢物标准品，参见附录A，纯度 $\geqslant 95\%$ 。

# 4.4 标准溶液配制

4.4.1 标准储备溶液 $(100\mathrm{mg / L})$ ：分别准确称取氟虫腈、氟甲腈、氟虫腈砜和氟虫腈亚砜(4.3)各 $10$ mg(精确至 $0.1\mathrm{mg}$ )，用乙腈溶解并稀释至 $100~\mathrm{mL}$ ，摇匀，制成质量浓度为 $100\mathrm{mg / L}$ 标准储备溶液，避光一 $18^{\circ}C$ 保存，有效期1年。

4.4.2 混合标准中间溶液 $(1\mathrm{mg / L})$ ：分别准确吸取氟虫腈、氟甲腈、氟虫腈砜和氟虫腈亚砜标准储备溶液(4.4.1)各 $1\mathrm{mL}$ ，用乙腈稀释至 $100\mathrm{mL}$ ，摇匀，制成 $1\mathrm{mg / L}$ 的混合标准中间溶液，避光 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存，有效期1个月。

4.5 材料

4.5.1 乙二胺-N-丙基硅烷化硅胶(PSA)： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}_{\circ}$   
4.5.2 十八烷基硅烷键合硅胶 $(\mathrm{C}_{18})$ ： $40~\mu \mathrm{m}\sim 60~\mu \mathrm{m}$   
4.5.3 微孔滤膜(有机相)： $0.22\mu \mathrm{m}$

5 仪器

5.1 液相色谱-三重四极杆质谱联用仪：配备ESI源。  
5.2 分析天平：感量 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。  
5.3 离心机：转速不低于 $5000\mathrm{r / min}$ 。  
5.4 涡旋振荡器。  
5.5 振荡器。  
5.6 组织匀浆机。

6 试样制备

6.1 试样制备

取16枚新鲜鸡蛋(约 $1\mathrm{kg}$ )，洗净去壳后用组织匀浆机充分搅拌均匀，放入聚乙烯瓶中。

6.2 试样储存

将试样按照测试和备用分别存放，于 $-20^{\circ}\mathrm{C} \sim -16^{\circ}\mathrm{C}$ 条件下保存。

7 分析步骤

7.1 提取

准确称取 $5\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ ) 置于 $50\mathrm{mL}$ 离心管中，加入 $20\mathrm{mL}$ 乙腈，涡旋混匀 $1\mathrm{min}$ ，振荡提取 $5\mathrm{min}$ ，加入 $2\mathrm{g}$ 氯化钠和 $6\mathrm{g}$ 无水硫酸钠，涡旋 $1\mathrm{min}$ ，以 $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液待净化。

7.2 净化

准确吸取 $1\mathrm{mL}$ 上清液于 $2\mathrm{mL}$ 聚丙烯离心管中，加入 $50\mathrm{mg}$ PSA 粉末、 $50\mathrm{mg}$ $\mathrm{C}_{18}$ 粉末和 $150\mathrm{mg}$ 无水硫酸镁，涡旋混合 $30\mathrm{s}$ ，以 $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液过 $0.22\mu \mathrm{m}$ 滤膜，用于测定。

7.3 仪器参考条件

7.3.1 液相色谱参考条件

a) 色谱柱: $C_{18}(2.1 \mathrm{~mm} \times 100 \mathrm{~mm}, 2.7 \mu \mathrm{m})$ , 或性能相当者;  
b） 柱温： $35^{\circ}C$   
c) 流动相: 乙酸铵-甲酸溶液(A相), 甲醇(B相);  
d) 流速: $0.4 \mathrm{~mL} / \mathrm{min}$ ;   
e) 进样量: $2 \mu \mathrm{L}$ ;   
f) 流动相及梯度洗脱条件见表1。

# 7.4 基质混合标准工作曲线

准确吸取一定量的混合标准中间溶液，用空白基质提取液逐级稀释成质量浓度为 $0.001\mathrm{mg / L}$ 、 $0.002\mathrm{mg / L},0.004\mathrm{mg / L}$ 和 $0.01\mathrm{mg / L},0.02\mathrm{mg / L}$ 的基质混合标准工作溶液，供液相色谱-质谱联用仪测定。以农药定量离子峰面积为纵坐标、农药基质标准溶液质量浓度为横坐标，绘制标准曲线。

# 7.5 定性及定量

# 7.5.1 保留时间

被测试样中目标农药色谱峰的保留时间与相应标准色谱峰的保留时间相比较，相对误差应在土 $2.5\%$ 之内。

# 7.5.2 定量离子、定性离子及子离子丰度比

在相同实验条件下进行样品测定时，如果检出的色谱峰的保留时间与标准样品相一致，并且在扣除背景后的样品质谱图中，目标化合物的质谱定量和定性离子均出现，而且同一检测批次，对同一化合物，样品中目标化合物的定性离子和定量离子的相对丰度比与质量浓度相当的基质标准溶液相比，其允许偏差不超过表3规定的范围，则可判断样品中存在目标农药。

# 7.5.3 试样溶液的测定

将基质混合标准工作溶液和试样溶液依次注入液相色谱-质谱联用仪中，保留时间和定性离子定性，测得定量离子峰面积，待测样液中农药的响应值应在仪器检测的定量测定线性范围之内，超过线性范围时应根据测定浓度进行适当倍数稀释后再进行分析。

# 7.6 平行试验

按7.1～7.3、7.5的规定对同一试样进行平行试验测定。

# 7.7 空白试验

除不加试料外，按 $7.1\sim 7.3,7.5$ 的规定进行平行操作。

# 8 结果计算

试样中各农药残留量以质量分数 $\omega$ 计，单位以毫克每千克 $(\mathrm{mg / kg})$ 表示，按式(1)计算。

$$
\omega = \frac {\rho \times A \times V}{A _ {s} \times m} \tag {1}
$$

式中：

$\omega$ ——试样中被测物残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——基质标准工作溶液中被测物的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$A$ ——试样溶液中被测物的色谱峰面积；

$A_{s}$ ——基质标准工作溶液中被测物的色谱峰面积；

$V$ ——试样溶液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样溶液所代表试样的质量，单位为克(g)。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，保留2位有效数字。含量超 $1\mathrm{mg / kg}$ 时，保留3位有效数字。

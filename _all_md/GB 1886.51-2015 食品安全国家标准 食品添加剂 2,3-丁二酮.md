# 前言

本标准代替GB23488—2009《食品添加剂 2,3-丁二酮》。

本标准与GB23488—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 2,3-丁二酮”。

# 食品安全国家标准

# 食品添加剂 2,3-丁二酮

# 1 范围

本标准适用于化学合成法制得的食品添加剂2，3-丁二酮（又名丁二酮或双乙酰）。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

2,3-丁二酮

# 2.2 分子式

$\mathrm{C_4H_6O_2}$

# 2,3-丁二酮含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $50\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ 。  
B.2.2 固定相：聚乙二醇。  
B.2.3 膜厚: $0.25 \mu \mathrm{m}$ 。  
B.2.4 色谱炉温度： $150^{\circ}C$ 恒温。  
B.2.5 进样口温度： $250^{\circ}C$ 。  
B.2.6 检测器温度： $250^{\circ}C$ 。  
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 柱前压： $60\mathrm{kPa}$   
B.2.10 进样量：约 $0.2\mu \mathrm{L}$   
B.2.11 分流比： $100:1$

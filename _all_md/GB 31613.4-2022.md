# 牛可食性组织中吡利霉素残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了牛可食性组织中吡利霉素残留检测的制样和液相色谱-串联质谱测定方法。

本文件适用于牛肌肉、肝脏、肾脏和脂肪中吡利霉素的残留检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的吡利霉素, 用 $5 \%$ 甲酸乙腈溶液提取, 固相萃取柱净化, 液相色谱-串联质谱测定, 基质匹配标准溶液外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/16682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱醇  
5.1.3 甲酸（HCOOH）：色谱纯。  
5.1.4 无水硫酸钠 $\mathrm{(Na_2SO)_4)}$   
5.1.5 氯化钠(NaCl)。

# 5.2 溶液配制

5.2.1 $5\%$ 甲酸乙腈溶液：取甲酸 $5\mathrm{mL}$ ，用乙腈稀释至 $100~\mathrm{mL}$   
5.2.2 $0.1\%$ 甲酸乙腈溶液：取甲酸 $1\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$ 。  
5.2.3 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。  
5.2.4 $80\%$ 甲醇溶液：取甲醇 $80~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ 。

# 5.3 标准品

吡利霉素(Pirlimycin, $\mathrm{C_{17}H_{31}ClN_2O_5S}$ , CAS号: 79548-73-5), 含量 $\geqslant 99.0\%$ 。

# 5.4 标准溶液的制备

5.4.1 标准储备液：精密称取吡利霉素标准品约 $10\mathrm{mg}$ ，于 $10\mathrm{mL}$ 容量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $1\mathrm{mg/mL}$ 的吡利霉素标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期12个月。  
5.4.2 标准中间液：精密量取吡利霉素标准储备液 $0.1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的吡利霉素标准中间液。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期6个月。

5.4.3 标准工作液：精密量取吡利霉素标准中间液 $0.1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用 $80\%$ 甲醇稀释至刻度，配制成浓度为 $100\mathrm{ng/mL}$ 的吡利霉素标准工作液。 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 保存，有效期6个月。

5.5 材料

5.5.1 固相萃取小柱：基质增强脂肪去除柱（ $300\mathrm{mg} / 3\mathrm{mL}$ ），或相当者。  
5.5.2 陶瓷均质子。  
5.5.3 亲水性聚四氟乙烯微孔滤膜： $0.22\mu \mathrm{m}$

6 仪器和设备

6.1 液相色谱-串联质谱仪：配有电喷雾离子源(ESI)。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$   
6.3 涡旋混合器。  
6.4 高速离心机。

7 试样的制备与保存

7.1 试样的制备

取适量新鲜或解冻的空白或供试样品，绞碎并均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

8 测定步骤

8.1 提取

称取试料 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ )于 $50~\mathrm{mL}$ 离心管中，加入陶瓷均质子，加入 $5 \%$ 甲酸乙腈溶液 $10~\mathrm{mL}$ 涡旋 $30~\mathrm{s}$ ，加入无水硫酸钠 $4\mathrm{g}$ 、氯化钠 $1\mathrm{g}$ ，涡旋 $30~\mathrm{s},4^{\circ}C$ 下 $8000~\mathrm{r / min}$ 离心 $8\mathrm{min}$ 后，快速移取上层乙腈 $2.4\mathrm{mL}$ 至另一离心管中，加水 $0.6~\mathrm{mL}$ ，混匀，备用。

8.2 净化

将备用液直接过固相萃取小柱，自然流出，收集滤液，并挤干。取滤液适量过微孔滤膜后，供液相色谱-串联质谱仪测定。

8.3 基质匹配标准曲线的制备

精密量取吡利霉素标准工作液适量，用提取净化后的空白试样溶液稀释成含药物浓度分别为 $1\mathrm{ng / mL},2\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},100\mathrm{ng / mL}$ 和 $400\mathrm{ng / mL}$ 的基质匹配系列标准溶液，从中各取 $1.0~\mathrm{mL}$ ，过微孔滤膜上机测定。以特征离子质量色谱峰面积为纵坐标、基质匹配标准溶液浓度为横坐标，绘制标准曲线。

8.4 测定

8.4.1 液相色谱参考条件

(a) 色谱柱：五氟苯基柱 $(50\mathrm{mm}\times 3.0\mathrm{mm},2.6\mu \mathrm{m})$ ，或性能相当者。  
b）流动相：A为 $0.1\%$ 甲酸溶液，B为 $0.1\%$ 甲酸乙腈溶液。流动相梯度： $0\mathrm{min}\sim 1\mathrm{min}$ 保持 $10\% \mathrm{B};1\mathrm{min}\sim 3\mathrm{min},10\% \mathrm{B}$ 线性变化到 $90\% \mathrm{B};3\mathrm{min}\sim 4\mathrm{min}$ 保持 $90\% \mathrm{B};4\mathrm{min}\sim 5\mathrm{min}$ 保持 $10\% \mathrm{B}_{\circ}$   
c) 流速: $0.4 \mathrm{~mL} / \mathrm{min}$ 。  
d) 进样量: $5 \mu \mathrm{L}$ 。

e） 柱温： $30^{\circ}C$

# 8.4.2 串联质谱参考条件

a）离子源：电喷雾离子源；  
b）扫描方式：正离子扫描；  
c）检测方式：多反应离子监测（MRM）；  
d）电喷雾电压： $5500\mathrm{V}$ □  
e) 离子源温度: $500^{\circ} \mathrm{C}$ 。  
f）辅助气 $1:50$ psi。  
g）辅助气2：50psi。  
h) 气帘气：25 psi。  
i）碰撞气：Medium。  
j）吡利霉素定性、定量离子对和对应的去簇电压、碰撞能量参考值见表1。

# 8.5 空白试验

取空白试料，除不加药物外，采用完全相同的测定步骤进行测定。

# 9 结果计算和表述

试样中吡利霉素的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C \times A \times V _ {1} \times V _ {3} \times 1 0 0 0}{A _ {\mathrm {s}} \times V _ {2} \times m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中吡利霉素残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$C$ ——基质匹配标准溶液中吡利霉素浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A$ ——试样溶液中吡利霉素的峰面积；

$A_{s}$ ——基质匹配标准溶液中吡利霉素的峰面积；

$V_{1}$ ——提取液总体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——上清液中取出溶液体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{3}$ ——最终溶液体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法对牛肌肉、肝脏、肾脏和脂肪的检测限为 $2\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法吡利霉素在牛肌肉 $10\mu \mathrm{g / kg}\sim 200\mu \mathrm{g / kg}$ 、肝脏 $10~\mu \mathrm{g / kg}\sim 2000~\mu \mathrm{g / kg}$ 、肾脏 $10~\mu \mathrm{g / kg}\sim$ $800~\mu \mathrm{g / kg}$ 、脂肪 $10~\mu \mathrm{g / kg}\sim 200~\mu \mathrm{g / kg}$ 添加浓度上的回收率为 $60\% \sim 110\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leq 15\%$ ，批间相对标准偏差 $\leq 20\%$ 。

# 食品安全国家标准

# 食品中多种磷酸盐的测定

# 1 范围

本标准规定了食品中多种磷酸盐的测定方法。

本标准适用于食品中磷酸盐、焦磷酸盐、六偏磷酸盐、三偏磷酸盐、三聚磷酸盐的测定。

# 2 原理

试样采用相应的方法提取和净化，以氢氧化钾溶液为淋洗液，阴离子交换柱分离，电导检测器检测。以保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 氢氧化钠(NaOH)。  
3.1.2 氢氧化钾(KOH)。  
3.1.3 甲醇 $\mathrm{(CH_3OH)}$ ，色谱纯。

# 3.2 试剂配制

3.2.1 氢氧化钠溶液 $(10\mathrm{mmol / L})$ ：称取 $0.4\mathrm{g}$ 氢氧化钠，溶于水并稀释定容至 $1000~\mathrm{mL}$ 。  
3.2.2 氢氧化钠溶液 $(50\mathrm{mmol / L})$ ：称取 $2.0\mathrm{g}$ 氢氧化钠，溶于水并稀释定容至 $1000~\mathrm{mL}$ 。

# 3.3 标准品

3.3.1 磷酸钠 $\mathrm{(Na_3PO_4)}$ 标准溶液 $(1000\mathrm{mg / L}$ ，水基体）。  
3.3.2 焦磷酸钠 $\mathrm{(Na_4P_2O_7)}$ 标准溶液（ $1000\mathrm{mg / L}$ ，水基体）。  
3.3.3六偏磷酸钠 $\left[\left(\mathrm{NaPO}_{3}\right)_{6}\right]$ 标准溶液（ $1000\mathrm{mg / L}$ ，水基体）。  
3.3.4 三偏磷酸钠 $\left[\mathrm{(NaPO_3)_3}\right]$ 标准品：纯度 $\geqslant 98\%$   
3.3.5 三聚磷酸钠 $\mathrm{(Na_5P_3O_{10})}$ 标准品：纯度 $\geqslant 98\%$

# 3.4 标准溶液配制

3.4.1 磷酸根标准中间(贮备)溶液 $(100\mathrm{mg / L})$ : 吸取磷酸钠标准溶液 $17.3\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中,用 $10\mathrm{mmol / L}$ 氢氧化钠溶液稀释至刻度, 此溶液每 $1\mathrm{L}$ 含磷酸根 $0.1\mathrm{g}$ 。  
3.4.2 焦磷酸根标准贮备溶液 $(100\mathrm{mg / L})$ : 吸取焦磷酸钠标准溶液 $15.3\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中, 用 $10\mathrm{mmol / L}$ 氢氧化钠溶液稀释至刻度, 此溶液每 $1\mathrm{L}$ 含焦磷酸根 $0.1\mathrm{g}$ 。  
3.4.3六偏磷酸根标准中间(贮备)溶液 $(100\mathrm{mg / L})$ ：吸取六偏磷酸钠标准溶液 $12.9~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容

量瓶中，用 $10\mathrm{mmol / L}$ 氢氧化钠溶液稀释至刻度，此溶液每 $1\textrm{L}$ 含六偏磷酸根 $\mathrm{0.1g}$ 。

3.4.4 三偏磷酸根标准中间(贮备)溶液 $(1000\mathrm{mg / L})$ ：将三偏磷酸钠在 $103^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 烘箱中干燥 $3\mathrm{h}$ ，在干燥器中冷却至室温后，准确称取三偏磷酸钠标准品 $0.132\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )，用 $10\mathrm{mmol / L}$ 氢氧化钠溶液稀释至 $100~\mathrm{mL}$ ，此溶液每 $1\textrm{L}$ 含三偏磷酸根 $1.0\mathrm{g}$ 。  
3.4.5 三聚磷酸根标准贮备溶液 $(1000\mathrm{mg / L})$ ：将三聚磷酸钠在 $103^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 烘箱中干燥 $3\mathrm{h}$ ，在干燥器中冷却至室温后，准确称取三聚磷酸钠标准品 $0.148\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），用 $10\mathrm{mmol / L}$ 氢氧化钠溶液稀释至 $100\mathrm{mL}$ ，此溶液每 $1\mathrm{L}$ 含三聚磷酸根 $1.0\mathrm{g}$ 。  
3.4.6 标准曲线工作溶液：吸取五种不同磷酸根标准贮备溶液，用 $10\mathrm{mmol / L}$ 氢氧化钠溶液稀释，制成系列标准溶液，五种不同磷酸根的标准溶液浓度见表1。

# 4 仪器和设备

4.1 离子色谱仪：包括电导检测器，配有抑制器，带梯度泵或淋洗液发生器，高容量阴离子交换柱， $100\mu \mathrm{L}$ 定量环。  
4.2 食物粉碎机。  
4.3 超声波清洗器：可进行 $80^{\circ}\mathrm{C}$ 控温， $60\mathrm{Hz}$ 。  
4.4 天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.5 离心机：转速 $\geqslant 8000\mathrm{r / min}$ ，温度可调到 $4^{\circ}C$ 。   
4.6 $0.45\mu \mathrm{m}$ 水性滤膜（带针头过滤器）。  
4.7 净化柱：包括OnGuard II RP柱、Ag柱和Na柱1)，或等效柱。  
4.8 注射器： $1.0\mathrm{mL}$ 和 $5.0~\mathrm{mL}$

注：所有玻璃器皿使用前均需要依次用 $2\mathrm{mol} / \mathrm{L}$ 氢氧化钾溶液和水分别浸泡 $4\mathrm{h}$ ，然后用水冲洗3次～5次，晾干备用。

# 5 分析步骤

# 5.1 试样制备

5.1.1 鱼、肉类及其制品：取 $500\mathrm{g}$ 鱼、肉或其制品的可食部分，捣碎混匀。冷冻保存。  
5.1.2 蔬菜、水果：取可食部分 $500\mathrm{g}$ ，洗净、晾干、切碎、混匀，用组织搅拌机制成匀浆备用。  
5.1.3 奶粉、乳制品饮料或饮料类：通过反复摇晃和颠倒容器使样品充分混匀直到使样品均一化。

5.1.4 固体油脂、脂肪等样品：取 $500\mathrm{g}$ 的样品研磨成均匀的泥浆状。为避免水分损失，研磨过程中应避免产生过多的热量。  
5.1.5 杂粮、小麦粉及其制品、果冻、巧克力、糖果、膨化食品、熟制坚果与籽类：取 $500\mathrm{g}$ 可食部分用食物粉碎机搅匀备用。  
5.1.6 调味料：粉末状调味料通过反复摇晃和颠倒容器使样品充分混匀；固体大颗粒状调味料取 $500\mathrm{g}$ 用食物粉碎机搅匀备用。  
5.1.7 谷类和淀粉类甜品、米粉、婴幼儿配方食品及辅助食品：通过反复摇晃和颠倒容器使样品充分混匀直到使样品均一化。

注：在试样制备过程中，应防止样品受到污染。

# 5.2 样品前处理

# 5.2.1 样品的提取

5.2.1.1 蔬菜、水果、果冻、巧克力及糖果、杂粮、小麦粉及其制品、奶粉、乳制品饮料或饮料类：称取 $2.5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，可适当调整试样的取样量）试样，用 $50\mathrm{mmol / L}$ 氢氧化钠溶液洗入 $50~\mathrm{mL}$ 比色管中混匀定容至刻度， $80^{\circ}\mathrm{C}$ 超声提取 $30\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振摇一次，保持固定相完全分散。冷却至室温后，溶液经滤纸过滤；取滤液于 $4^{\circ}\mathrm{C}$ 下， $8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液备用。  
5.2.1.2 膨化食品、熟制坚果与籽类、谷类和淀粉类甜品、米粉、婴幼儿配方食品及辅助食品：称取 $2.5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，可适当调整试样的取样量）试样，用 $50\mathrm{mmol/L}$ 氢氧化钠溶液洗入 $25\mathrm{mL}$ 比色管中混匀定容至刻度， $80^{\circ}\mathrm{C}$ 超声提取 $30\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振摇一次，保持固定相完全分散。冷却至室温后，溶液经滤纸过滤；取滤液于 $4^{\circ}\mathrm{C}$ 下， $8000\mathrm{r/min}$ 离心 $10\mathrm{min}$ ，取上清液备用。  
5.2.1.3 油脂、脂肪、调味料：称取 $1\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，可适当调整试样的取样量）试样，用 $50\mathrm{mmol / L}$ 氢氧化钠溶液洗入 $50~\mathrm{mL}$ 比色管中混匀定容至刻度， $80^{\circ}\mathrm{C}$ 超声提取 $30\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振摇一次，保持固定相完全分散。冷却至室温后，溶液经滤纸过滤；取滤液于 $4^{\circ}\mathrm{C}$ 下， $8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取水相清液备用。  
5.2.1.4 鱼、肉类及其制品：称取 $2.5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，可适当调整试样的取样量）试样，用 $50\mathrm{mmol / L}$ 氢氧化钠溶液洗入 $100~\mathrm{mL}$ 比色管中混匀定容至刻度， $80^{\circ}\mathrm{C}$ 超声提取 $30\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振摇一次，保持固定相完全分散。冷却至室温后，溶液经滤纸过滤；取滤液于 $4^{\circ}\mathrm{C}$ 下， $8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液备用。

# 5.2.2 样品的净化

取5.2.1中处理完成后的上清液约 $15\mathrm{mL}$ ，通过 $0.45\mu \mathrm{m}$ 水性滤膜针头过滤器、OnGuard II RP，弃去前面 $3\mathrm{mL}$ （如果氯离子大于 $100\mathrm{mg / L}$ ，则需要依次通过针头过滤器、OnGuard II RP、Ag柱和Na柱，弃去前 $7\mathrm{mL}$ )，收集后面洗脱液待测。测定前应根据样品含量对待测液进行适当稀释。

固相萃取柱使用前需进行活化，如使用OnGuard II RP柱 $(1.0\mathrm{mL})$ 、OnGuard II Ag柱 $(1.0\mathrm{mL})$ 和OnGuard II Na柱 $(1.0\mathrm{mL})$ ，其活化过程为：OnGuard II RP柱 $(1.0\mathrm{mL})$ 使用前依次用 $10~\mathrm{mL}$ 甲醇、 $15~\mathrm{mL}$ 水通过，静置活化 $30\mathrm{min}$ 。OnGuard II Ag柱 $(1.0\mathrm{mL})$ 和OnGuard II Na柱 $(1.0\mathrm{mL})$ 用 $10~\mathrm{mL}$ 水通过，静置活化 $30\mathrm{min}$

# 5.4 标准曲线的制作

将标准系列工作液从低浓度到高浓度依次进样，测定相应的电导检测器的信号值，得到各浓度标准溶液的色谱图。以标准工作液的浓度 $(\mathrm{mg / L})$ 为横坐标，以峰面积 $(\mu \mathrm{s})$ 或峰高为纵坐标，绘制标准曲线，并计算线性回归方程。参考色谱图及保留时间见附录B。

# 5.5 试样溶液的测定

将试样溶液在相同工作条件下注入离子色谱仪中，记录色谱图，以保留时间定性，测定样品的峰面积 $(\mu \mathrm{s})$ 或峰高，根据标准曲线得到待测液中被测组份的浓度。

# 5.6 空白试验

空白试验系指除不加试样外，采用完全相同的分析步骤、试剂和用量，进行平行操作。

# 6 分析结果的表述

试样中磷酸根含量按式(1)计算：

$$
X _ {i} = \frac {\left(\rho_ {i} - \rho_ {0 i}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X_{i}$ ——样品中第 $i$ 个磷酸根含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho_{i}$ ——样液中第 $i$ 个磷酸根测定值，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_{0i}$ ——样品空白液中第 $i$ 个磷酸根测定值，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——样品提取液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品称样量，单位为克(g)；

1000——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

若分析结果需要以正磷酸根含量表示，则见附录A将聚磷酸根含量乘以换算系数 $F$ 。样品中总多聚磷酸根为各种多聚磷酸根换算的正磷酸根的总和。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

本标准的检出限和定量限如下：

a）膨化食品、熟制坚果与籽类谷类和淀粉类甜品、米粉、婴幼儿配方食品及辅助食品：取样 $2.5\mathrm{g}$ ，定容 $50~\mathrm{mL}$ ，测定前再稀释2.5倍，测定各多聚磷酸根的检出限分别为正磷酸根 $1.5\mathrm{mg / kg}$ ，焦磷酸根 $1.4\mathrm{mg / kg}$ ，三聚磷酸根 $1.5\mathrm{mg / kg}$ ，三偏磷酸根 $1.6\mathrm{mg / kg}$ ，六偏磷酸根 $4.8\mathrm{mg / kg}$ ，定量限分别为正磷酸根 $5.0\mathrm{mg / kg}$ ，焦磷酸根 $5.0\mathrm{mg / kg}$ ，三聚磷酸根 $5.0\mathrm{mg / kg}$ ，三偏磷酸根 $5.0\mathrm{mg / kg}$ ，六偏磷酸根 $15\mathrm{mg / kg}$ 。  
b）蔬菜、水果、果冻、巧克力及糖果、杂粮、小麦粉及其制品、奶粉、乳制品饮料或饮料类：取样 $2.5\mathrm{g}$ ，定容 $50~\mathrm{mL}$ ，测定前再稀释5倍，测定各多聚磷酸根的检出限分别为正磷酸根 $3.0\mathrm{mg / kg}$ ，焦磷酸根 $2.8\mathrm{mg / kg}$ ，三聚磷酸根 $3.0\mathrm{mg / kg}$ ，三偏磷酸根 $3.2\mathrm{mg / kg}$ ，六偏磷酸根 $9.6\mathrm{mg / kg}$ ，定量限分别为正磷酸根 $10\mathrm{mg / kg}$ ，焦磷酸根 $10\mathrm{mg / kg}$ ，三聚磷酸根 $10\mathrm{mg / kg}$ ，三偏磷酸根 $10\mathrm{mg / kg}$ ，六偏磷酸根 $30\mathrm{mg / kg}$ 。

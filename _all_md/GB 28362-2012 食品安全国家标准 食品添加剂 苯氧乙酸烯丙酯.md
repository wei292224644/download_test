# 食品安全国家标准

# 食品添加剂 苯氧乙酸烯丙酯

# 1 范围

本标准适用于由苯氧乙酸和烯丙醇为原料制得的食品添加剂苯氧乙酸烯丙酯。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

苯氧乙酸2-丙烯酯

# 2.2 分子式

$\mathrm{C_{11}H_{12}O_3}$

# 2.3 结构式

![](images/3ecfb14429f1bd1073a13eea80375be24d2f1cb1d3206e752d8312ae18349354.jpg)

# 2.4 相对分子质量

192.22(按2007年国际相对原子质量)

# 附录A

# 苯氧乙酸烯丙酯含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。

A.1.2 柱：毛细管柱。

A. 1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

# A. 3 重复性及结果表示

按GB/T11538—2006中11.4规定执行。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ 。  
B.2.2 固定相：改性聚乙二醇。  
B.2.3 膜厚： $0.25\mu \mathrm{m}$   
B.2.4 色谱炉温度: $150^{\circ} \mathrm{C}$ 恒温 $2 \mathrm{~min}$ , 然后线性程序升温从 $150^{\circ} \mathrm{C} \sim 210^{\circ} \mathrm{C}$ , 速率 $10^{\circ} \mathrm{C} / \mathrm{min}$ , 最后在 $210^{\circ} \mathrm{C}$ 恒温 $15 \mathrm{~min}$ 。  
B.2.5 进样口温度： $280^{\circ}C$   
B.2.6 检测器温度： $280^{\circ}C$   
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 柱前压： $25\mathrm{kPa}$   
B. 2.10 进样量：约 $0.2 \mu \mathrm{L}$ 。  
B.2.11 分流比： $1 / 60$

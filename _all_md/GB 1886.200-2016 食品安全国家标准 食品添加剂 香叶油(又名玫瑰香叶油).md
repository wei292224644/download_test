# 前言

本标准代替GB11959—2008《食品添加剂 香叶(精)油》。

本标准与GB11959—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 香叶油(又名玫瑰香叶油)”;

——香叶醇含量由“ $5.0\sim 12.0$ ”改为“ $2.0\sim 12.0$ ”。

# 食品安全国家标准

# 食品添加剂 香叶油（又名玫瑰香叶油）

# 1 范围

本标准适用于用水蒸气蒸馏法从生长在中国的香叶(Pelargonium graveolens Hers.)的新鲜茎、叶中提取的食品添加剂香叶油(又名玫瑰香叶油)。

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。

面积归一化法：按GB/T11538—2006中10.4测定含量。

按GB/T11538—2006中11.4规定进行，应符合要求。食品添加剂香叶油气相色谱图及操作条件参见附录B。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $50\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ 。  
B.2.2 固定相：聚乙二醇20000。  
B.2.3 膜厚: $0.25 \mu \mathrm{m}$ 。  
B.2.4 色谱炉温度：线性程序升温从 $65~\mathrm{^\circ C}$ 至 $230^{\circ}C$ ，速率 $2^{\circ}C / \min$   
B.2.5 进样口温度： $230^{\circ}C$   
B.2.6 检测器温度： $250^{\circ}C$   
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 载气流速： $1.1\mathrm{mL / min}$   
B.2.10 进样量：约 $0.2\mu \mathrm{L}$   
B.2.11 分流比： $100:1$

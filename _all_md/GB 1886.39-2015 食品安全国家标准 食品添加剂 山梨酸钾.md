本标准代替GB13736—2008《食品添加剂 山梨酸钾》。

本标准与GB13736—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 山梨酸钾”。

# 食品安全国家标准

# 食品添加剂 山梨酸钾

# 1 范围

本标准适用于以山梨酸和碳酸钾(或氢氧化钾)用水为溶剂反应制得的食品添加剂山梨酸钾。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

反，反-2，4-己二烯酸钾

# 2.2 分子式

$\mathrm{C_6H_7KO_2}$

# 2.3 结构式

$$
\mathrm {C H} _ {3} \mathrm {C H} = \mathrm {C H C H} = \mathrm {C H C O O K}
$$

# 2.4 相对分子质量

150.22（按2007年国际相对原子质量）

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应小心谨慎。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。使用剧毒品时，应严格按照有关规定管理；使用时应避免吸入或与皮肤接触，必要时应在通风橱中进行。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 方法原理

A.3.1.1 山梨酸钾与酒石酸氢钠反应生成难溶于乙醇的酒石酸钾白色沉淀。  
A.3.1.2 山梨酸钾和溴发生加成反应而使溴的棕黄色消失。  
A.3.1.3 山梨酸钾中的不饱和基团在紫外光区有特征吸收。

# A.3.2 试剂和材料

A.3.2.1 无水乙醇。  
A.3.2.2 盐酸溶液： $1 + 3$   
A.3.2.3 酒石酸氢钠饱和溶液。  
A.3.2.4 溴饱和溶液。  
A.3.2.5 盐酸溶液： $9 + 1000$

# A.3.3 仪器和设备

紫外分光光度仪。

# A.3.4 分析步骤

A.3.4.1 称取 $1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $100\mathrm{mL}$ 水中，在 $5\mathrm{mL}$ 该试样溶液中加入 $5\mathrm{mL}$ 饱和酒石酸氢钠溶液和 $5\mathrm{mL}$ 无水乙醇，摇匀，应有白色透明的沉淀物生成。  
A.3.4.2 称取 $1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $100\mathrm{mL}$ 水中，在 $10\mathrm{mL}$ 该试样溶液中，加丙酮 $1\mathrm{mL}$ ，滴加盐酸溶液（A.3.2.2）使成弱酸性后，加溴溶液2滴摇匀时，溴的颜色应消失。  
A.3.4.3 称取 $50\mathrm{mg}$ 试样，精确至 $0.1\mathrm{mg}$ ，溶于水，稀释至 $250.0\mathrm{mL}$ ，吸取 $2.0\mathrm{mL}$ ，用盐酸溶液(A.3.2.5)稀释至 $200.0\mathrm{mL}$ ，在 $230\mathrm{nm} \sim 350\mathrm{nm}$ 进行分光光度测定，此溶液应在 $264\mathrm{nm} \pm 2\mathrm{nm}$ 处有最大吸收，最大吸收处的比吸光度应为 $1650 \sim 1900$ 。

# A.4 山梨酸钾（以 $\mathbf{C}_6\mathbf{H}_7\mathbf{KO}_2$ 计）（以干基计)的测定

# A.4.1 方法提要

干燥试样以冰乙酸为溶剂，乙酸酐为助溶剂，以结晶紫为指示剂，用高氯酸标准滴定溶液滴定，根据消耗高氯酸标准滴定溶液的体积计算山梨酸钾含量。

# A.4.2 试剂和材料

A.4.2.1 冰乙酸。  
A.4.2.2 乙酸酐。  
A.4.2.3 高氯酸标准滴定溶液： $c(\mathrm{HClO}_4) = 0.1\mathrm{mol / L}$   
A.4.2.4 结晶紫指示液： $5\mathrm{g / L}$

# A.4.3 分析步骤

A.4.3.1 称取约 $0.2\mathrm{g}$ 干燥物(A.5.2)，精确至 $0.0002\mathrm{g}$ ，置于已放有 $48\mathrm{mL}$ 冰乙酸和 $2\mathrm{mL}$ 乙酸酐的 $250\mathrm{mL}$ 碘量瓶中，温热使成溶液。冷却至室温，加2滴结晶紫指示液，用高氯酸标准滴定溶液滴定至溶液由紫色变为蓝绿色为终点。

A.4.3.2 在测定的同时，按与测定相同的步骤，对不加试样而使用相同数量的试剂溶液做空白试验。

# A.4.4 结果计算

山梨酸钾(以 $\mathrm{C_6H_7KO_2}$ 计)(以干基计)的质量分数 $\omega_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\left[ \frac {\left(V _ {1} - V _ {2}\right)}{1 000} \right] \times c \times M}{m} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——试样消耗高氯酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——空白试验消耗高氯酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$c$ ——高氯酸标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——山梨酸钾的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(\mathrm{C_6H_7KO_2}) = 150.2]$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 干燥减量的测定

# A.5.1 方法提要

测量在一定温度下试样中的可挥发性物质的排出量。

# A.5.2 分析步骤

称取 $2\mathrm{g}\sim 3\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于预先在 $105~^\circ \mathrm{C}\pm 2~^\circ \mathrm{C}$ 干燥至质量恒定的称量瓶中，铺成 $5\mathrm{mm}$ 以下的层。在 $105~^\circ \mathrm{C}\pm 2~^\circ \mathrm{C}$ 的恒温干燥箱中干燥 $3\mathrm{h}$ ，置于干燥器中冷却 $30\mathrm{min}$ 称量。保留干燥

物用于山梨酸钾含量的测定。

# A.5.3 结果计算

干燥减量的质量分数 $\varpi_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m _ {2} - m _ {1}}{m _ {2}} \times 100 \% \tag{A.2}
$$

式中：

$m_{2}$ ——干燥前试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——干燥后试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.1\%$ 。

# A.6 氯化物（以Cl计）的测定

# A.6.1 方法提要

氯化物和硝酸银在酸性(硝酸)溶液中生成氯化银白色沉淀。将试样溶于水，在酸性条件下，加入硝酸银产生白色浑浊，与标准比浊溶液进行比较，做限量试验。

# A.6.2 试剂和材料

A.6.2.1 硝酸溶液： $1 + 10$   
A.6.2.2 硝酸银溶液： $17\mathrm{g / L}$   
A.6.2.3 氯化物(Cl)标准溶液： $0.18\mathrm{mg / mL}$

# A.6.3 分析步骤

称取 $1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加水约 $30~\mathrm{mL}$ 溶解，边搅拌边加硝酸溶液 $11~\mathrm{mL}$ ，过滤，水洗，合并洗液和滤液置于 $100~\mathrm{mL}$ 比色管中，加水至 $50~\mathrm{mL}$ 为试验溶液。另取一只比色管，加入 $1\mathrm{mL} \pm 0.02\mathrm{mL}$ 氯化物(Cl)标准溶液，加 $30~\mathrm{mL}$ 水，加 $6\mathrm{mL}$ 硝酸溶液及水配至 $50~\mathrm{mL}$ 为标准比浊溶液。分别于两比色管中加入 $1\mathrm{mL}$ 硝酸银溶液，充分混匀，避开日光直射，放置 $5\mathrm{min}$ ，在黑色背景下侧面或轴向进行观察，试验溶液的浊度不得大于标准比浊溶液的浊度。

# A.7 硫酸盐（以 $\mathrm{SO}_4$ 计）的测定

# A.7.1 方法提要

硫酸盐和氯化钡在酸性(盐酸)溶液中，生成硫酸钡白色沉淀，与标准比浊溶液进行比较，做限量试验。

# A.7.2 试剂和材料

A.7.2.1 盐酸溶液： $1 + 4$   
A.7.2.2 氯化钡溶液： $50~\mathrm{g / L}$   
A.7.2.3 硫酸盐 $\mathrm{(SO_4)}$ 标准溶液： $0.19\mathrm{mg / mL}$

# A.7.3 分析步骤

称取试样 $0.5\mathrm{g}$ ，精确至 $0.01\mathrm{g}$ ，加水约 $30~\mathrm{mL}$ 溶解，边搅拌边加盐酸溶液 $3\mathrm{mL}$ ，过滤，水洗，合并洗

液和滤液于 $100~\mathrm{mL}$ 比色管中，加水至 $50~\mathrm{mL}$ ，作为试验溶液。

取 $1.0\mathrm{mL}$ 硫酸盐(SO4)标准溶液于另一支 $100~\mathrm{mL}$ 比色管中，加 $1\mathrm{mL}$ 盐酸溶液，加水至 $50~\mathrm{mL}$ ，作为标准比浊溶液。

同时向试验溶液和标准比浊溶液中各加 $5\mathrm{mL}$ 氯化钡溶液，放置 $10\mathrm{min}$ 后观察，试验溶液的浊度不得大于标准比浊溶液的浊度。

# A.8 醛（以HCHO计)的测定

# A.8.1 方法原理

甲醛与品红亚硫酸起加成反应，由于重排的结果，恢复醌型结构的紫红色物质，与标准比色溶液进行比较，做限量试验。

# A.8.2 试剂和材料

A.8.2.1 异丙醇。

A.8.2.2 盐酸溶液： $9 + 100$

A.8.2.3 甲醛标准溶液： $0.1\mathrm{mg / mL}$

A.8.2.4 品红-亚硫酸溶液：将 $0.2\mathrm{g}$ 碱性品红溶解于 $120~\mathrm{mL}$ 热水中，加入 $20~\mathrm{mL}$ 新鲜配制的 $100~\mathrm{g / L}$ 亚硫酸钠 $\left(\mathrm{NaSO}_3\cdot 7\mathrm{H}_2\mathrm{O}\right)$ 溶液及 $2\mathrm{mL}$ 盐酸，移入锥形瓶中，用棉花团塞住瓶口，置于暗处，隔日后装入棕色瓶中。

# A.8.3 分析步骤

称取试样 $1\mathrm{g}$ ，精确至 $0.01\mathrm{g}$ ，溶于 $50~\mathrm{mL}$ 异丙醇及 $30~\mathrm{mL}$ 水中，用盐酸溶液调整 $\mathrm{pH}$ 至4（用精密试纸检验），转移至 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度。取上述溶液 $10~\mathrm{mL}$ 置于 $50~\mathrm{mL}$ 比色管中，取 $1.0~\mathrm{mL}$ 甲醛标准溶液、 $4~\mathrm{mL}$ 异丙醇和 $5~\mathrm{mL}$ 水注入另一 $50~\mathrm{mL}$ 比色管中。然后分别于两比色管中加入 $1~\mathrm{mL}$ 品红-亚硫酸溶液，摇匀，放置 $30\mathrm{min}$ 后进行目视比色，试验溶液所呈的红色不得深于标准比色溶液。

# A.9 澄清度试验

# A.9.1 试剂和材料

A.9.1.1 盐酸溶液： $24\rightarrow 1000$   
A.9.1.2 氯化钴 $(\mathrm{CoCl}_2\cdot 6\mathrm{H}_2\mathrm{O})$ ：氯化钴的质量分数按GB/T605—2006进行测定。  
A.9.1.3 氯化铁 $\mathrm{(FeCl_3\cdot 6H_2O)}$ ：氯化铁的质量分数按HG/T3474—2000进行测定。  
A.9.1.4 氯化钴标准原液：称取已测定质量分数的氯化钴，用盐酸溶液溶解并稀释，使该溶液 $1\mathrm{mL}$ 中含 $59.5\mathrm{mg}$ 氯化钴。  
A.9.1.5 氯化铁标准原液：称取已测定质量分数的氯化铁，用盐酸溶液溶解并稀释，使该溶液 $1\mathrm{mL}$ 中含 $45\mathrm{mg}$ 氯化铁。  
A.9.1.6 标准比色原液：取氯化钴标准原液 $6\mathrm{mL}$ 和氯化铁标准原液 $24\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中，用盐酸溶液稀释至 $100\mathrm{mL}$ 。  
A.9.1.7 标准比色溶液：取 $12.5\mathrm{mL}$ 标准比色原液于 $100~\mathrm{mL}$ 容量瓶中，用盐酸溶液稀释至 $100~\mathrm{mL}$ ，混合均匀。

# A.9.2 分析步骤

称取 $0.2\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $10\mathrm{mL}$ 比色管中，加 $5.0\mathrm{mL}$ 水溶解，作为试验溶液；取 $5\mathrm{mL} \pm 0.02\mathrm{mL}$ 标准比色溶液置于另一只 $10\mathrm{mL}$ 比色管中，在无阳光直射情况下，轴向及侧向观察，试验溶液的颜色不得深于标准比色溶液的颜色。

# A.10.1 方法提要

以酚酞为指示剂，用规定量的硫酸标准滴定溶液中和。

# A.10.2 试剂和材料

A.10.2.1 硫酸标准滴定溶液： $c\left(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4\right) = 0.100\mathrm{mol / L}$

A.10.2.2 酚酞指示液： $10\mathrm{g / L}$

# A.10.3 分析步骤

称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于约 $20~\mathrm{mL}$ 无二氧化碳的水，加2滴酚酞指示溶液显红色，再加硫酸标准滴定溶液 $0.40~\mathrm{mL}$ ，摇匀，红色应消失。

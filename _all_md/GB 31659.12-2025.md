# 1 范围

本文件规定了禽蛋和奶中大观霉素、双氢链霉素、链霉素、卡那霉素、庆大霉素、新霉素、潮霉素B和安普霉素等氨基糖苷类药物残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于禽蛋和牛奶、羊奶中大观霉素、双氢链霉素、链霉素、卡那霉素A、庆大霉素 $\mathrm{C_1}$ 、庆大霉素 $\mathrm{C_{1a}}$ 、庆大霉素 $\mathrm{C_2 + C_{2b}}$ 、新霉素B、潮霉素B和安普霉素等氨基糖苷类药物残留量的检测，其他类型奶的检测可参照执行。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中残留的氨基糖苷类药物用三氯乙酸溶液提取，亲水-亲脂固相萃取柱净化，液相色谱-串联质谱法测定，外标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯  
5.1.3 甲酸（HCOOH）：色谱纯。  
5.1.4 异丙醇 $\mathrm{(C_3H_7OH)}$ ：色谱纯。  
5.1.5 乙酸铵 $\mathrm{(CH_3COONH_4)}$ ：色谱纯。  
5.1.6 三氯乙酸 $(\mathrm{Cl}_3\mathrm{CCOOH})$   
5.1.7 氨水 $(\mathrm{NH_4OH})$ 。

# 5.2 溶液配制

5.2.1 $5\%$ 三氯乙酸溶液：取三氯乙酸 $50~\mathrm{g}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。  
5.2.2 $5\%$ 甲醇溶液：取甲醇 $5\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$   
5.2.3 $0.1\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液：取乙酸铵 $0.77\mathrm{g}$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。  
5.2.4 $0.002\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液：取 $0.1\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液 $10~\mathrm{mL}$ ，用水稀释至 $500~\mathrm{mL}$ 。  
5.2.5 洗脱溶液：取甲酸 $20~\mathrm{mL}$ 、异丙醇 $10~\mathrm{mL},0.002\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液 $170~\mathrm{mL}$ ，混匀，用甲酸调节 $\mathrm{pH}$

至0.8。

5.2.6 流动相A：取甲酸 $5\mathrm{mL},0.1\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液 $10~\mathrm{mL}$ ，用水稀释至 $500~\mathrm{mL}$ 。  
5.2.7 流动相B：取甲酸 $5\mathrm{mL},0.1\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液 $10~\mathrm{mL}$ ，用乙腈稀释至 $500~\mathrm{mL}$ 。

# 5.3 标准品

盐酸大观霉素、硫酸卡那霉素A含量 $\geqslant 96.0\%$ ，硫酸双氢链霉素、硫酸链霉素、潮霉素B含量 $\geqslant$ $92.0\%$ ，庆大霉素 $\mathbf{C}_1$ 五乙酸盐、庆大霉素 $\mathbf{C}_{1a}$ 五乙酸盐、庆大霉素 $\mathbf{C}_2$ 五乙酸盐含量 $\geqslant 95.0\%$ ，硫酸新霉素B、硫酸安普霉素含量 $\geqslant 99.0\%$ ，具体见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液：取标准品各适量（相当于各活性成分约 $10\mathrm{mg}$ ），精密称定，用水适量使溶解并稀释定容至 $10\mathrm{mL}$ 聚丙烯容量瓶，分别制成浓度均为 $1\mathrm{mg/mL}$ 的标准储备液。于聚丙烯圆形广口瓶中 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 保存，有效期1个月。  
5.4.2 混合标准中间液I：准确移取双氢链霉素、链霉素、卡那霉素A、庆大霉素 $\mathrm{C_1}$ 、庆大霉素 $\mathrm{C_2 + C_{2a}}$ 、潮霉素B和安普霉素标准储备液各 $0.1\mathrm{mL}$ ，大观霉素、庆大霉素 $\mathrm{C_{1a}}$ 、新霉素B标准储备液各 $0.2\mathrm{mL}$ ，于 $10~\mathrm{mL}$ 聚丙烯容量瓶，用水稀释定容至刻度，配制成混合标准中间液。其中，双氢链霉素、链霉素、卡那霉素A、庆大霉素 $\mathrm{C_1}$ 、庆大霉素 $\mathrm{C_2 + C_{2a}}$ 、潮霉素B和安普霉素浓度均为 $10~\mu \mathrm{g / mL}$ ，大观霉素、庆大霉素 $\mathrm{C_{1a}}$ 、新霉素B浓度均为 $20~\mu \mathrm{g / mL}$ 。于聚丙烯圆形广口瓶中 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1周。  
5.4.3 混合标准中间液Ⅱ：准确移取潮霉素B安普霉素标准储备液各 $0.1\mathrm{mL}$ ，大观霉素、双氢链霉素、链霉素、卡那霉素A庆大霉素 $\mathbf{C}_1$ 、庆大霉素 $\mathbf{C}_{1\mathrm{a}}$ 、庆大霉素 $\mathbf{C}_2 + \mathbf{C}_{2\mathrm{a}}$ 、新霉素B标准储备液各 $0.4\mathrm{mL}$ ，于 $10\mathrm{mL}$ 聚丙烯容量瓶，用水稀释至刻度，配制成混合标准中间液。其中，潮霉素B安普霉素浓度均为 $10~\mu \mathrm{g / mL}$ ，大观霉素、双氢链霉素、链霉素、卡那霉素A庆大霉素 $\mathbf{C}_1$ 、庆大霉素 $\mathbf{C}_{1\mathrm{a}}$ 、庆大霉素 $\mathbf{C}_2 + \mathbf{C}_{2\mathrm{a}}$ 、新霉素B浓度均为 $40~\mu \mathrm{g / mL}$ 。于聚丙烯圆形广口瓶中 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1周。  
5.4.4 混合标准工作液I：准确移取混合标准中间液I适量，用洗脱溶液稀释，配制成双氢链霉素、链霉素、卡那霉素A、庆大霉素 $\mathbf{C}_1$ 、庆大霉素 $\mathrm{C_2 + C_{2a}}$ 、潮霉素B和安普霉素浓度分别为 $0.125\mu \mathrm{g / mL}$ 、 $0.25\mu \mathrm{g / mL},0.50\mu \mathrm{g / mL},0.75\mu \mathrm{g / mL},1\mu \mathrm{g / mL},2\mu \mathrm{g / mL},3\mu \mathrm{g / mL}$ ，大观霉素、庆大霉素 $\mathrm{C}_{1a}$ 、新霉素B浓度分别为 $0.25\mu \mathrm{g / mL},0.50\mu \mathrm{g / mL},1\mu \mathrm{g / mL},1.5\mu \mathrm{g / mL},2\mu \mathrm{g / mL},4\mu \mathrm{g / mL},6\mu \mathrm{g / mL}$ 的混合标准工作液。临用现配。  
5.4.5 混合标准工作液Ⅱ：准确移取混合标准中间液Ⅱ适量，用洗脱溶液稀释，配制成潮霉素B、安普霉素浓度分别为 $0.125\mu \mathrm{g / mL}, 0.25\mu \mathrm{g / mL}, 0.50\mu \mathrm{g / mL}, 0.75\mu \mathrm{g / mL}, 1\mu \mathrm{g / mL}, 2\mu \mathrm{g / mL}, 3\mu \mathrm{g / mL}$ ，大观霉素、双氢链霉素、链霉素、卡那霉素A、庆大霉素C₁、庆大霉素C₁a、庆大霉素C₂+C₂₃、新霉素B浓度分别为 $0.5\mu \mathrm{g / mL}, 1\mu \mathrm{g / mL}, 2\mu \mathrm{g / mL}, 3\mu \mathrm{g / mL}, 4\mu \mathrm{g / mL}, 8\mu \mathrm{g / mL}, 12\mu \mathrm{g / mL}$ 的混合标准工作液。临用现配。

# 7.1 试样的制备

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；

# 8 测定步骤

# 8.1 提取

称取试料 $(5\pm 0.05)\mathrm{g}$ 于 $50~\mathrm{mL}$ 聚丙烯离心管，加 $5\%$ 三氯乙酸溶液 $20~\mathrm{mL}$ 涡旋混合 $1\mathrm{min}$ ，振荡提取 $15\mathrm{min},4^{\circ}C$ 下 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液，用氨水调节 $\mathsf{pH}$ 至7.5，备用。

# 8.2 净化

固相萃取柱依次用甲醇、水各 $5\mathrm{mL}$ 活化，取备用液过柱，依次用水、 $3\%$ 甲醇溶液各 $3\mathrm{mL}$ 淋洗，抽干。用洗脱溶液 $2.0\mathrm{mL}$ 进行洗脱，收集洗脱液，抽干，涡旋，尼龙微孔滤膜过滤，供液相色谱-串联质谱仪测定。

# 8.3 基质匹配标准曲线的制备

# 8.3.1 禽蛋基质匹配标准曲线的制备

# 8.5 测定法

# 8.5.1 定性测定

在相同测试条件下，试料溶液中氨基糖苷类药物色谱峰的保留时间与基质配备标准溶液中氨基糖苷类药物色谱峰的保留时间偏差在 $\pm 0.1\mathrm{min}$ 以内；且检测到的相对离子丰度应与浓度相当的基质匹配标准溶液相对离子丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.5.2 定量测定

取试料溶液和相应的基质配备标准溶液，做单点或多点校准，按外标法以特征离子质量色谱峰面积定量，基质配备标准溶液及试料溶液中的氨基糖苷类药物响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，氨基糖苷类药物基质匹配标准溶液特征离子质量色谱图见附录B。

# 8.6 空白试验

取空白试样，除不加标准溶液外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中氨基糖苷类药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {1} \times A \times V \times 1 0 0 0}{A _ {1} \times m \times 1 0 0 0}. \tag {1}
$$

式中：

$X$ ——试样中氨基糖苷类药物残留量的数值，单位为微克/千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{s}}$ 基质匹配标准溶液中氨基糖苷类药物浓度的数值单位为纳克每毫升 $(\mathrm{ng / mL})$

A ——试料溶液中氨基糖苷类药物的色谱峰面积；

V 洗脱溶液体积的数值，单位为毫升 $(\mathrm{mL})$

# 10 方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法在禽蛋中双氢链霉素、链霉素、卡那霉素A、庆大霉素 $\mathrm{C_1}$ 、庆大霉素 $\mathrm{C_2 + C_{2a}}$ 、潮霉素B、安普霉素的检出限均为 $5\mu \mathrm{g / kg}$ ，定量限均为 $10\mu \mathrm{g / kg}$ ；大观霉素、庆大霉素 $\mathrm{C_3}$ 、新霉素B的检出限均为 $10\mu \mathrm{g / kg}$ ，定量限为 $20\mu \mathrm{g / kg}$ 。在牛羊奶中潮霉素B、安普霉素的检出限均为 $5\mu \mathrm{g / kg}$ ，定量限均为 $10\mu \mathrm{g / kg}$ ；大观霉素（双氢链霉素、链霉素、卡那霉素A、庆大霉素 $\mathrm{C_4}$ 、庆大霉素 $\mathrm{C_{1a}}$ 、庆大霉素 $\mathrm{C_2 + C_{2a}}$ 、新霉素B的检出限均为 $20\mu \mathrm{g / kg}$ ，定量限为 $10\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法禽蛋在 $10\mu \mathrm{g} / \mathrm{kg}\sim 4000\mu \mathrm{g} / \mathrm{kg}$ 牛羊奶在 $10\mu \mathrm{g} / \mathrm{kg}\sim 3000\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $60\% \sim 110\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leq 15\%$ ，批间相对标准偏差 $\leq 20\%$ 。

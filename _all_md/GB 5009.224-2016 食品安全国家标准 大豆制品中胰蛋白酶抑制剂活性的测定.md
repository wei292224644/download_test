# 食品安全国家标准

# 大豆制品中胰蛋白酶抑制剂活性的测定

# 1 范围

本标准规定了大豆制品中胰蛋白酶抑制剂活性（TIA）的测定方法。

本标准适用于大豆制品中胰蛋白酶抑制剂活性的测定。

# 2 原理

胰蛋白酶可与苯甲酰-L-精氨酸-对硝基苯胺(L-BAPA)发生反应，生成对硝基苯胺，该物质在 $410~\mathrm{nm}$ 下有特征吸收。大豆制品中的胰蛋白酶抑制剂可抑制此反应，使吸光度值下降，其下降程度与胰蛋白酶抑制活性成正比。采用分光光度计在 $410~\mathrm{nm}$ 处测定该反应前后的吸光度值，可定量分析胰蛋白酶抑制剂活性。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 盐酸（HCl）。  
3.1.2 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.3 氢氧化钠(NaOH)。  
3.1.4 氯化钙 $(\mathrm{CaCl}_2\cdot 2\mathrm{H}_2\mathrm{O})$ 。  
3.1.5 胰蛋白酶（Trypsin）： $-20^{\circ}\mathrm{C}$ 冻藏。  
3.1.6 苯甲酰-L-精氨酸-对硝基苯胺（L-BAPA）。  
3.1.7 三羟甲基氨基甲烷 $\mathrm{[NH_2C(CH_2OH)_3}$ ，Tris]。  
3.1.8 二甲亚砜 $(\mathrm{C}_2\mathrm{H}_6\mathrm{OS},\mathrm{DMOS})$ 。

# 3.2 试剂配制

3.2.1 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ：取 $50~\mathrm{mL}$ 盐酸(3.1.1)加入水中，用水稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.2 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：取 $83~\mathrm{mL}$ 盐酸(3.1.1)加入水中，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.3 盐酸溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：取 $8.3\mathrm{mL}$ 盐酸(3.1.1)加入水中，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.4 盐酸溶液 $(0.001\mathrm{mol} / \mathrm{L})$ ：取 $1\mathrm{mL}1\mathrm{mol} / \mathrm{L}$ 盐酸溶液加入水中，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.5 乙酸溶液 $(5.3\mathrm{mol} / \mathrm{L})$ ：取 $30~\mathrm{mL}$ 冰乙酸(3.1.2)加入水中，用水稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.6 氢氧化钠溶液 $(0.01\mathrm{mol} / \mathrm{L})$ ：称取 $0.40\mathrm{g}$ 氢氧化钠(3.1.3)，溶于 $500~\mathrm{mL}$ 水中，用水稀释至 $1000~\mathrm{mL}$ 。  
3.2.7 氯化钙盐酸溶液: 称取 $0.735 \mathrm{~g}$ 氯化钙(3.1.4)溶解于 $1 \mathrm{~L} 0.001 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液中, 用 $1 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液和 $0.1 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液调节 $\mathrm{pH}$ 至 $3.0 \pm 0.1$ 。

3.2.8 胰蛋白酶储备液 $(0.270\mathrm{mg / mL})$ ：将胰蛋白酶(3.1.5)放置至室温，称取胰蛋白酶 $27.0\mathrm{mg}$ 于小烧杯中，用氯化钙盐酸溶液(3.2.7)溶解，转移至 $100~\mathrm{mL}$ 容量瓶中，以氯化钙盐酸溶液(3.2.7)定容至刻度。保存于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中，最多可使用5d。  
3.2.9 胰蛋白酶使用液 $(0.0135\mathrm{mg / mL})$ ：用移液管量取 $5.0~\mathrm{mL}$ 胰蛋白酶储备液(3.2.8)于 $100~\mathrm{mL}$ 容量瓶中，用氯化钙盐酸溶液(3.2.7)定容至刻度。  
3.2.10 三羟甲基氨基甲烷-氯化钙缓冲液：称取 $6.05\mathrm{g}$ 三羟甲基氨基甲烷(3.1.7)和 $0.735\mathrm{g}$ 氯化钙(3.1.4)溶于预先加入 $900~\mathrm{mL}$ 水的带有刻度的 $1000~\mathrm{mL}$ 烧杯中，用 $1\mathrm{mol} / \mathrm{L}$ 盐酸溶液和 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液调节 $\mathrm{pH}$ 值至 $8.2\pm 0.1$ ，加水至 $1000~\mathrm{mL}$ 。  
3.2.11 L-BAPA 溶液: 称取 $60 \mathrm{mg} \mathrm{L}$ -BAPA(3.1.6)用 $1 \mathrm{~mL}$ 二甲亚砜(3.1.8)溶解, 用三羟甲基氨基甲烷-氯化钙缓冲溶液(3.2.10)转移至 $100 \mathrm{~mL}$ 容量瓶中, 稀释至刻度。用时现配。

# 4 仪器和设备

4.1 可见分光光度计。  
4.2 分析天平：感量分别为 $0.01\mathrm{g}, 0.001\mathrm{g}, 0.0001\mathrm{g}$ 。  
4.3 酸度计：精度为0.05。  
4.4 离心机：转速 $\geqslant 4000\mathrm{r / min}$   
4.5 涡旋振荡器。  
4.6 恒温水浴锅：精度为 $\pm 0.1^{\circ}C$

# 5 分析步骤

# 5.1 试样制备

对于粉末状样品，取有代表性的样品至少 $200\mathrm{g}$ ，充分混匀后备用。对于块状或颗粒状样品，取有代表性的样品至少 $200\mathrm{g}$ ，粉碎至 $425\mu \mathrm{m}$ 以下，充分混匀备用。粉碎过程中避免样品过热。

# 5.2 样品提取

称取 $1\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )制备好的试样(5.1)于 $100~\mathrm{mL}$ 锥形瓶中，加入 $50~\mathrm{mL}0.01\mathrm{mol / L}$ 氢氧化钠溶液，摇匀。用 $1\mathrm{mol / L}$ 盐酸溶液和 $0.1\mathrm{mol / L}$ 盐酸溶液调节 $\mathrm{pH}$ 至 $9.5\pm 0.1$ ，置于 $0^{\circ}\mathrm{C}\sim$ $4^{\circ}\mathrm{C}$ 冰箱中静置 $15\mathrm{h}\sim 24\mathrm{h}$ 。

将样品提取液取出放置至室温，转移至 $100\mathrm{mL}$ 容量瓶中，用水定容至刻度，摇匀。经过 $15\mathrm{min}$ 的沉淀后，根据需要对样品提取液进行稀释。稀释度取决于预期的样品TIA值。将提取液保存于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱中，可保存 $1\mathrm{d}$ 。

# 5.3 样品提取液稀释

参照附录A中表A.1的稀释方案，将样品提取液稀释三个不同稀释浓度，确保在三个抑制百分率中至少有一个TIA值的测定结果在 $40\% \sim 60\%$ 之间。

如果三个测定结果均没有在此范围内，则需要改变稀释度重新测定。

# 6 分析结果的表述

# 6.1 样品提取液的抑制率

样品提取液的抑制率按式(1)计算：

$$
i = \frac {\left(A _ {\mathrm {r}} - A _ {\mathrm {br}}\right) - \left(A _ {\mathrm {s}} - A _ {\mathrm {bs}}\right)}{\left(A _ {\mathrm {r}} - A _ {\mathrm {br}}\right)} \times 100 \% \quad \dots \tag{1}
$$

式中：

$i$ ——抑制率；

$A_{\mathrm{r}}$ ——标准溶液的吸光度；

$A_{\mathrm{br}}$ ——标准空白溶液的吸光度；

$A_{s}$ ——样品溶液的吸光度；

$A_{\mathrm{bs}}$ ——样品空白溶液的吸光度；

$100\%$ ——换算系数。

# 6.2 胰蛋白酶抑制剂活性

胰蛋白酶抑制剂活性按式(2)计算，以每克样品抑制胰蛋白酶毫克数 $(\mathrm{mg / g})$ 表示：

$$
\mathrm {T I A} = \frac {i}{100 \%} \times \frac {\rho f}{m F} \tag{2}
$$

式中：

TIA ——胰蛋白酶抑制剂活性，单位为毫克每克 $(\mathrm{mg / g})$ ；

i ——抑制百分率；

$100\%$ ——换算系数；

$\rho$ ——胰蛋白酶使用液的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$f$ —— $5.6 \times 10^{3}$ ，该数值是满足本实验胰蛋白酶活性条件下(5.4)的胰蛋白酶纯度折算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$F$ ——样品提取液的稀释度(表A.1所示理论稀释度， $\mathrm{mL} / 100\mathrm{mL}$ )。

计算结果精确至 $0.1\mathrm{mg / g}$

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法的检出限为 $0.5\mathrm{mg / g}$

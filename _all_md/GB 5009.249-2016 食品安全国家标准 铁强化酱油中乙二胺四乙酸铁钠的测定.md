# 前言

本标准代替GB/T21234—2007《铁强化酱油中乙二胺四乙酸铁钠的测定》。

本标准与GB/T21234—2007相比，主要变化如下：

——标准名称修改为“食品安全国家标准 铁强化酱油中乙二胺四乙酸铁钠的测定”；

——取消第二法 比色法；

——计算公式中含量以 $\mathrm{mg / 100mL}$ 表示。

# 食品安全国家标准

# 铁强化酱油中乙二胺四乙酸铁钠的测定

# 1 范围

本标准规定了铁强化酱油中乙二胺四乙酸铁钠的测定方法。

本标准适用于铁强化酱油中乙二胺四乙酸铁钠的测定。

# 2 原理

试样经甲醇沉淀，过滤，滤液稀释后，经反相高效液相色谱法分离，紫外检测器 $254\mathrm{nm}$ 波长处检测，由色谱峰保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。  
3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.3 四丁基氢氧化铵 $\mathrm{(C_{16}H_{36}N\bullet OH)}$ 水溶液 $(40~\mathrm{g / 100~mL})$ 。  
3.1.4 甲酸（HCOOH）。  
3.1.5 甲酸溶液 $(100\mathrm{mL} / \mathrm{L})$ ：吸取 $5\mathrm{mL}$ 甲酸，用水定容至 $50~\mathrm{mL}$ 。  
3.1.6 甲醇水溶液 $(3 + 1)$ ：量取 $750~\mathrm{mL}$ 甲醇(3.1.1)，加水 $250~\mathrm{mL}$ ，混匀。  
3.1.7 乙二胺四乙酸铁钠标准品 $\mathrm{(C_{10}H_{12}FeN_2NaO_8\cdot 3H_2O)}$ ：纯度 $\geqslant 99.0\%$

# 3.2 标准溶液的制备

3.2.1 乙二胺四乙酸铁钠标准贮备液 $(2.0\mathrm{mg / mL})$ ：称取乙二胺四乙酸铁钠标准品 $0.05\mathrm{g}$ （精确至 $0.0001\mathrm{g})$ ，置于 $25~\mathrm{mL}$ 容量瓶中，用水溶解定容。此溶液应以棕色容量瓶配制，于冰箱中冷藏避光保存，推荐15日内使用。  
3.2.2 乙二胺四乙酸铁钠标准中间液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：用甲醇水溶液(3.1.6)将 $2.5\mathrm{mL}$ 乙二胺四乙酸铁钠标准贮备液定容至 $50~\mathrm{mL}$ 。此溶液应以棕色容量瓶配制，于冰箱中冷藏避光保存。  
3.2.3 乙二胺四乙酸铁钠标准工作液：分别吸取乙二胺四乙酸铁钠标准中间液 $1.0\mathrm{mL},2.0\mathrm{mL}$ $4.0~\mathrm{mL},6.0~\mathrm{mL},8.0~\mathrm{mL},10.0~\mathrm{mL}$ 于 $50.0~\mathrm{mL}$ 棕色容量瓶中，用水定容。此浓度即为 $2.0~\mu \mathrm{g / mL}$ $4.0~\mu \mathrm{g / mL},8.0~\mu \mathrm{g / mL},12.0~\mu \mathrm{g / mL},16.0~\mu \mathrm{g / mL},20.0~\mu \mathrm{g / mL}$ ，临用时配制。

# 4 仪器与设备

4.1 高效液相色谱仪，带有紫外检测器或二极管阵列检测器。

4.2 天平：感量为 $0.1\mathrm{mg}$   
4.3 pH计：精度为0.01。

# 5 分析步骤

# 5.1 试样溶液的制备

吸取混匀后试样 $2.50~\mathrm{mL}$ ，置于 $50.0~\mathrm{mL}$ 棕色容量瓶中，加甲醇水溶液(3.1.6)稀释至刻度，摇匀。于避光处静置 $50\mathrm{min}$ 后，滤纸过滤。吸取滤液 $5.00~\mathrm{mL}$ ，置于 $50~\mathrm{mL}$ 棕色容量瓶中，用水稀释至刻度，摇匀，用 $0.45\mu \mathrm{m}$ 滤膜过滤，即得试样溶液，待上机测试用。

# 5.2 液相色谱参考条件

5.2.1 色谱柱： $\mathrm{C_8}$ 柱， $4.6 \mathrm{~mm} \times 150 \mathrm{~mm}, 5 \mu \mathrm{m}$ ，或具同等性能的色谱柱。  
5.2.2 流动相：吸取四丁基氢氧化铵水溶液 $(40\mathrm{g} / 100\mathrm{mL})3.3\mathrm{mL}$ 置于烧杯中，加水 $130~\mathrm{mL}$ ，再用甲酸溶液(3.1.5)将 $\mathrm{pH}$ 调至3.30，将此溶液转移至 $1000~\mathrm{mL}$ 容量瓶中，加 $125~\mathrm{mL}$ 甲醇(3.1.2)，用水稀释至刻度，摇匀，用 $0.45\mu \mathrm{m}$ 滤膜过滤，脱气后使用。  
5.2.3 流速： $1.0~\mathrm{mL / min}$   
5.2.4 检测波长： $254~\mathrm{nm}$   
5.2.5 进样量： $20~\mu \mathrm{L}$

# 5.3 标准曲线的绘制

将标准系列工作液(3.2.3)依次按5.2推荐色谱条件上机测定，记录色谱峰面积，色谱图参见附录A。以峰面积为纵坐标，浓度为横坐标，绘制标准曲线。

# 5.4 试样溶液的测定

将试样溶液(5.1)按5.2推荐色谱条件进样测定，从标准曲线中查得试样溶液相应的浓度。

# 6 分析结果的表述

试样中乙二胺四乙酸铁钠含量按式（1）计算：

$$
X = \frac {c _ {\mathrm {s}} \times f \times 1 0 0}{1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中乙二胺四乙酸铁钠的含量，单位为毫克每百毫升 $(\mathrm{mg} / 100\mathrm{mL})$ ；

$c_{\mathrm{s}}$ ——试样溶液的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$f$ —稀释倍数；

100 ——换算系数；

1000 ——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不超过算术平均值的 $10\%$ 。

# 8 其他

标准浓度的线性范围为 $2.0\mu \mathrm{g / mL}\sim 20.0\mu \mathrm{g / mL}$ ，定量限为 $40~\mathrm{mg / 100~mL}$

# 食品安全国家标准

# 食品添加剂 苯甲醛丙二醇缩醛

# 1 范围

本标准适用于由苯甲醛和1,2-丙二醇为原料制得的食品添加剂苯甲醛丙二醇缩醛。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

4-甲基-2-苯基-1,3-二氧戊环

# 2.2 分子式

$$
\mathrm {C} _ {1 0} \mathrm {H} _ {1 2} \mathrm {O} _ {2}
$$

164.20（按2007年国际相对原子质量）

# 苯甲醛丙二醇缩醛含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行，应符合要求。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$   
B.2.2 固定相： $5\%$ 二苯基 $95\%$ 二甲基聚硅氧烷。  
B.2.3 膜厚： $1.0~\mu \mathrm{m}$   
B.2.4 色谱炉温度： $150~\mathrm{^\circ C}$ 恒温 $3\mathrm{min}$ ；然后线性程序升温从 $150~\mathrm{^\circ C}\sim 250~\mathrm{^\circ C}$ ，速率 $10~\mathrm{^\circ C / min}$ ；最后在 $250~\mathrm{^\circ C}$ 恒温 $5\mathrm{min}$ 。  
B.2.5 进样口温度： $250~\mathrm{^\circ C}$   
B.2.6 检测器温度： $280^{\circ}C$ 。  
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 柱前压： $120\mathrm{kPa}$   
B.2.10 进样量： $0.1\mu \mathrm{L}$   
B.2.11 分流比： $50:1$

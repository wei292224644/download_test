# 液相色谱-串联质谱法

# 1 范围

本文件规定了动物性食品中赛杜霉素的超高效液相色谱-串联质谱测定方法。

本文件适用于鸡肌肉和肝脏组织中赛杜霉素残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的赛杜霉素用乙腈提取，固相萃取柱净化，超高效液相色谱串联质谱法检测，外标法定量。

# 5 试剂和材料

除特别注明者外均为分析纯试剂；水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.3 甲酸（HCOOH）：色谱纯。  
5.1.4 二氯甲烷 $\mathrm{(CH_2Cl_2)}$

# 5.2 溶液配制

5.2.1 $50\%$ 乙腈溶液：取乙腈 $50~\mathrm{mL}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ ，混匀  
5.2.2 $80\%$ 二氯甲烷甲醇溶液：取二氯甲烷 $80~\mathrm{mL}$ ，加甲醇 $20~\mathrm{mL}$ ，混匀。  
5.2.3 流动相A：取水 $100~\mathrm{mL}$ ，加甲酸 $0.1\mathrm{mL}$ ，混匀。  
5.2.4 流动相B：取乙腈 $100~\mathrm{mL}$ ，加甲酸 $0.1~\mathrm{mL}$ ，混匀。

# 5.3 标准品

赛杜霉素(Semduramicin，分子式： $\mathrm{C_{45}H_{75}O_{16}}$ ，CAS号：113378-31-7)，含量 $\geq 94.3\%$

# 5.4 标准溶液制备

5.4.1 标准储备液：精密称取相当于赛杜霉素 $10\mathrm{mg}$ 的对照品，用乙腈溶解并定容于 $10~\mathrm{mL}$ 容量瓶中，配制成浓度为 $1\mathrm{mg / mL}$ 的赛杜霉素标准储备液。 $-18^{\circ}C$ 以下保存，有效期3个月。  
5.4.2 $10~\mu \mathrm{g / mL}$ 赛杜霉素标准工作液：精密量取标准储备液 $0.1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用 $50\%$ 乙腈水溶液稀释至刻度，配制成 $10~\mu \mathrm{g / mL}$ 赛杜霉素标准工作液。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1个月。

5.4.3 $1\mu \mathrm{g / mL}$ 赛杜霉素标准工作液：精密量取 $10~\mu \mathrm{g / mL}$ 赛杜霉素标准工作液 $1.0\mathrm{mL}$ 于 $10\mathrm{mL}$ 容量瓶中，用 $50\%$ 乙腈水溶液稀释至刻度，配制成 $1\mu \mathrm{g / mL}$ 赛杜霉素标准工作液。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1个月。  
5.5 材料  
5.5.1固相萃取柱：石墨化碳黑氨基固相萃取柱： $500\mathrm{mg / 6mL}$   
5.5.2 滤膜：有机相， $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 超高效液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$   
6.3 高速离心机。  
6.4 涡旋混合器。  
6.5 水平振荡器。  
6.6 均质机。  
6.7 固相萃取装置。  
6.8 氮吹仪。

# 7 试样的制备与保存

# 7.1 试样制备

取适量新鲜或解冻的空白或供试组织，绞碎，并均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样保存

$-20^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取试料 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管内，加乙腈 $12~\mathrm{mL}$ ，涡旋混匀 $1\mathrm{min}$ ，振荡 $10\mathrm{min}$ $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ （肝脏 $8000\mathrm{r / min}$ )，取上清液，加乙腈 $12~\mathrm{mL}$ 重复提取一次，合并两次提取液，备用。

# 8.2 净化

萃取柱依次用二氯甲烷 $5\mathrm{mL}$ 和乙腈 $5\mathrm{mL}$ 活化，取备用液，过柱，控制流速 $2\mathrm{mL} / \mathrm{min}\sim 3\mathrm{mL} / \mathrm{min}$ 。抽干 $10\mathrm{min}$ ，用 $80\%$ 二氯甲烷甲醇溶液 $6\mathrm{mL}$ 洗脱，收集洗脱液， $40^{\circ}\mathrm{C}$ 水浴中氮气吹干。用 $50\%$ 乙腈水溶液溶解并稀释至 $2.0\mathrm{mL}$ ，过滤，供超高效液相色谱-串联质谱测定。

# 8.3 基质匹配标准曲线的制备

精密量取 $1\mu \mathrm{g} / \mathrm{mL}$ 赛杜霉素标准工作液 $20~\mu \mathrm{L},40~\mu \mathrm{L}$ 和 $100~\mu \mathrm{L},10~\mu \mathrm{g} / \mathrm{mL}$ 赛杜霉素标准工作液 $20~\mu \mathrm{L},100~\mu \mathrm{L}$ 和 $200~\mu \mathrm{L}$ ，分别加入6份按提取和净化处理的空白试料残渣中，用 $50\%$ 乙腈水溶液溶解并稀释至 $2\mathrm{mL}$ ，配制成浓度为 $10~\mathrm{ng / mL},20~\mathrm{ng / mL},50~\mathrm{ng / mL},100~\mathrm{ng / mL},500~\mathrm{ng / mL}$ 和 $1000~\mathrm{ng / mL}$ 的基质匹配系列标准溶液， $4^{\circ}\mathrm{C}10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液，滤膜过滤，供超高效液相色谱-串联质谱测定。以测得特征离子色谱峰面积为纵坐标，基质匹配标准溶液浓度为横坐标，绘制基质匹配标准曲线。求回归方程和相关系数。

# 8.4 测定

b) 流动相: A 为 $0.1\%$ 甲酸溶液, B 为 $0.1\%$ 甲酸乙腈溶液;  
c）梯度洗脱：梯度洗脱程序见表1；  
d) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ ;   
e） 柱温： $30^{\circ}C$   
f）进样量： $10~\mu \mathrm{L}$

# 8.5 测定法

取试料溶液和基质匹配标准溶液，作单点或多点校准，外标法计算。基质匹配标准溶液及试料溶液中赛杜霉素的特征离子质量色谱峰面积均应在仪器检测的线性范围之内。试样溶液中的相对离子丰度与基质匹配标准溶液中的相对离子丰度相比，符合表3的要求。标准溶液特征离子质量色谱图见附录A。

# 8.6 空白试验

取空白试料，除不加药物外，采用完全相同的步骤进行平行操作。

# 9 结果计算和表述

试样中赛杜霉素的残留量按公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V}{A _ {\mathrm {S}} \times m} \tag {1}
$$

式中：

$X$ ——试样中赛杜霉素残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ ——标准溶液中赛杜霉素浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

A ——试样溶液中赛杜霉素的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中赛杜霉素的峰面积；

$V$ ——试样溶液浓缩后定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量的数值，单位为克(g)。

# 10 检测方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $3\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $10~\mu \mathrm{g / kg}\sim 400~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

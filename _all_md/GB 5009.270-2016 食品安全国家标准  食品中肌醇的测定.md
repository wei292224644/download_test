2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替GB5413.25—2010《食品安全国家标准 婴幼儿食品和乳品中肌醇的测定》。

本标准与GB5413.25—2010相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中肌醇的测定”；  
——增加了食品的检出限和定量限；  
——修改了方法的适用范围；  
——修改了试样制备和样品前处理方法；  
——修改了精密度。

# 食品安全国家标准

# 食品中肌醇的测定

# 1 范围

本标准规定了食品中肌醇的测定方法。

本标准第一法适用于食品中肌醇的测定，本标准第二法适用于婴幼儿食品、乳品及饮料中肌醇的测定。

# 第一法 微生物法

# 2 原理

利用葡萄汁酵母菌(Saccharomyces uvarum)对肌醇的特异性和灵敏性，定量测定试样中待测物质的含量。在含有除待测物质以外所有营养成分的培养基中，微生物的生长与待测物质含量呈线性关系，根据透光率与标准工作曲线进行比较，即可计算出试样中待测物质的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.2 氢氧化钠(NaOH)。  
3.1.3 盐酸（HCl）。  
3.1.4 五氧化二磷 $(\mathrm{P_2O_5})$ 。  
3.1.5 葡萄汁酵母菌(Saccharomyces uvarum)，ATCC9080，或其他等效标准菌株。

# 3.2 试剂配制

3.2.1 氯化钠溶液 $(9\mathrm{g / L})$ ：称取 $9.0\mathrm{g}$ 氯化钠溶解于 $1000~\mathrm{mL}$ 水中，分装试管，每管 $10~\mathrm{mL}$ 。 $121^{\circ}\mathrm{C}$ 灭菌 $15\mathrm{min}$ 。  
3.2.2 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $82~\mathrm{mL}$ 盐酸，冷却后定容至 $1000~\mathrm{mL}$ 。  
3.2.3 盐酸溶液 $(0.44\mathrm{mol} / \mathrm{L})$ ：量取 $36.6~\mathrm{mL}$ 盐酸，冷却后定容至 $1000~\mathrm{mL}$ 。  
3.2.4 氢氧化钠溶液 $(600\mathrm{g / L})$ ：称取 $300\mathrm{g}$ 氢氧化钠溶解于水中，冷却后定容至 $500~\mathrm{mL}$ 。  
3.2.5 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $40\mathrm{g}$ 氢氧化钠溶解于水中，冷却后定容至 $1000~\mathrm{mL}$ 。

# 3.3 标准品

肌醇标准品 $\left(\mathrm{C}_{6} \mathrm{H}_{12} \mathrm{O}_{6}\right)$ ：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 肌醇标准储备液 $(0.2\mathrm{mg / mL})$ ：肌醇标准品置于五氧化二磷的干燥器中干燥 $24\mathrm{h}$ 以上，称取 $50~\mathrm{mg}$ 肌醇标准品(精确到 $0.1\mathrm{mg}$ )，用水充分溶解，定容至 $250~\mathrm{mL}$ 棕色容量瓶中，贮存在 $4^{\circ}C$ 冰箱中。  
3.4.2 肌醇标准中间液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：吸取 $5.00~\mathrm{mL}$ 肌醇标准储备液，用水定容到 $100~\mathrm{mL}$ 棕色容量瓶中，贮存在 $4^{\circ}\mathrm{C}$ 冰箱中。  
3.4.3 肌醇标准工作液 $(1\mu \mathrm{g} / \mathrm{mL}$ 和 $2\mu \mathrm{g} / \mathrm{mL})$ ：吸取 $10~\mathrm{mL}$ 肌醇标准中间液两次，分别用水定容到 $100~\mathrm{mL}$ 容量瓶和 $50~\mathrm{mL}$ 容量瓶中。该工作液需每次临用前配制。

# 3.5 材料

# 3.5.1 培养基

3.5.1.1 麦芽浸粉琼脂培养基(Malt Extract Agar)：可按附录A配制。  
3.5.1.2 肌醇测定培养基：可按附录A配制。

注：一些商品化合成培养基效果良好，商品化合成培养基按标签说明进行配制。

# 3.5.2 玻璃珠

直径约 $5\mathrm{mm}$ 。

# 4 仪器和设备

注：除微生物常规灭菌及培养设备外，其他设备和材料如下：

4.1 天平：感量为 $0.1\mathrm{mg}$ 。  
4.2 pH计：精度 $\pm 0.01$   
4.3 分光光度计。  
4.4 涡旋振荡器。  
4.5 离心机：转速 $\geq 2000\mathrm{r / min}$   
4.6 恒温培养箱： $30^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 。  
4.7 振荡培养箱： $30^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ ，振荡速度140次/min～160次/min。  
4.8 压力蒸汽消毒器： $121^{\circ}\mathrm{C}(0.10\mathrm{MPa} \sim 0.12\mathrm{MPa})$ 。

注：玻璃仪器使用前，用活性剂(月桂磺酸钠或家用洗涤剂加入到洗涤用水中即可)对硬玻璃测定管及其他必要的玻璃器皿进行清洗，清洗之后 $200^{\circ}\mathrm{C}$ 干热 $2\mathrm{h}$ 。

# 5 分析步骤

# 5.1 接种菌悬液的制备

# 5.1.1 菌种复苏

将菌株活化后接种到麦芽浸粉琼脂斜面培养基上， $30^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $16\mathrm{h} \sim 24\mathrm{h}$ 后，再转接2代～3代来增强活力，制成贮备菌种，贮于 $4^{\circ}\mathrm{C}$ 冰箱中，保存期不要超过2周。临用前接种到新的麦芽浸粉琼脂斜面培养基上。

# 5.1.2 接种菌悬液的制备

在使用的前一天将贮备菌种转接到新配制的麦芽浸粉琼脂斜面培养基上，于 $30^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养

$20 \mathrm{~h} \sim 24 \mathrm{~h}$ 。用接种环刮取菌苔到装有氯化钠溶液 $(9 \mathrm{~g} / \mathrm{L})$ 的试管中。以 $2000 \mathrm{r} / \mathrm{min}$ 离心 $15 \mathrm{~min}$ , 如此清洗3次 $\sim 4$ 次。吸取一定量的该菌液移入装有 $10 \mathrm{~mL}$ 氯化钠溶液 $(9 \mathrm{~g} / \mathrm{L})$ 的试管中, 制成接种菌悬液。

用分光光度计，以氯化钠溶液作空白， $550~\mathrm{nm}$ 波长下测定该接种菌悬液的透光率，调整加入的菌液量或者加入一定量的氯化钠溶液使该菌悬液透光率在 $60\% \sim 80\%$ 。

# 5.2 试样制备

谷物类、乳粉等试样需粉碎、研磨、过筛(筛板孔径 $0.3\mathrm{mm}\sim 0.5\mathrm{mm}$ )；肉及肉制品等用打碎机制成食糜；果蔬等试样需匀浆混匀；液体试样用前振摇混合。 $4^{\circ}C$ 冰箱保存，1周内测定。

# 5.3 试样提取

准确称取肌醇 $0.5\mathrm{mg}\sim 2.0\mathrm{mg}$ 的试样，一般乳粉、新鲜果蔬、内脏、生肉试样 $1\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）；谷类、豆类、含量较低的天然食品试样 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），一般营养素补充剂、复合营养强化剂 $0.1\mathrm{g}\sim 0.5\mathrm{g}$ ；液体饮料或流质、半流质试样 $5\mathrm{g}\sim 10\mathrm{g}$ 于 $250\mathrm{mL}$ 锥形瓶中，对于干粉试样加入 $80\mathrm{mL}$ 盐酸溶液 $(0.44\mathrm{mol} / \mathrm{L})$ ，对于液体试样加入 $100\mathrm{mL}$ 盐酸溶液 $(0.44\mathrm{mol} / \mathrm{L})$ ，混匀。

将锥形瓶以铝箔纸覆盖，在灭菌釜中 $125^{\circ}\mathrm{C}$ 水解 $1\mathrm{h}$ 。取出，冷却至室温，加入约 $2\mathrm{mL}$ 氢氧化钠溶液 $(600\mathrm{g / L})$ ，冷却。用氢氧化钠溶液 $(1\mathrm{mol / L})$ 或盐酸溶液 $(1\mathrm{mol / L})$ 调 $\mathrm{pH}$ 至5.2，转入 $250~\mathrm{mL}$ 容量瓶中，定容至刻度。混匀，过滤，收集滤液，该滤液为待测液。调整稀释度，使待测液肌醇的浓度在 $1\mu \mathrm{g / mL}\sim 10~\mu \mathrm{g / mL}$ 范围内。

# 5.6 灭菌

每支试管内加入一粒玻璃珠，盖上试管帽， $121^{\circ}\mathrm{C}$ 灭菌 $5\mathrm{min}$ （商品培养基按标签说明进行灭菌）。

# 5.7 接种

将试管固定在振荡培养箱内，用约140次/min～160次/min的振荡速度，在 $30^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 振荡培养 $22\mathrm{h} \sim 24\mathrm{h}$ 。

# 5.8 测定

注：对每支试管进行视觉检查，接种空白试管S1内培养液应是澄清的，如果出现浑浊，则结果无效。

5.8.1 从振荡培养箱内取出试管，放入灭菌釜内， $100^{\circ}\mathrm{C}$ 保持 $5\mathrm{min}$ ，使微生物停止生长。  
5.8.2 用接种空白试管 S1 作空白, 将分光光度计透光率调到 $100\%$ (或吸光度为 0), 读出接种空白试管 S2 的读数。再以接种空白试管 S2 作空白, 调节透光率为 $100\%$ (或吸光度为 0), 依次读出其他每支试管的透光率 (或吸光度)。  
5.8.3 用涡旋振荡器充分混合每一只试管(也可以加一滴消泡剂)后，立即将培养液移入比色皿内进行测定，波长为 $540 \mathrm{~nm} \sim 660 \mathrm{~nm}$ ，待读数稳定 $30 \mathrm{~s}$ 后，读出透光率，每支试管稳定时间要相同。以肌醇标准系列的浓度为横坐标，透光率为纵坐标作标准曲线。  
5.8.4 根据待测液的透光率, 由标准曲线中得到该待测液中肌醇的浓度, 再根据稀释因子和称样量计算出试样中肌醇的含量。透光率超出标准曲线管 $\mathrm{S}3 \sim \mathrm{S}10$ 范围的试样管要舍去。  
5.8.5 对每个编号的待测液的试管，用每支试管的透光率计算每毫升该编号待测液肌醇的浓度，并计算该编号待测液的肌醇浓度平均值，每支试管测得的浓度不超过其平均值的 $\pm 15\%$ ，超过者要舍去。如果有1管的测定结果不符合上述要求，则舍弃该管的测定结果，重新计算平均值；如果有2管的测定结果不符合上述要求，则需重新检验。

注：绘制标准曲线，既可读取透光率，也可读取吸光度。

# 6 分析结果表述

试样中肌醇的含量按式(1)计算：

$$
X = \frac {C _ {x}}{m} \times \frac {f}{1 0 0 0} \times 1 0 0 \tag {1}
$$

式中：

$X$ ——试样中肌醇的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$C_x$ ——5.8.5中计算所得的平均值，单位为微克 $(\mu \mathrm{g})$ ；

$m$ ——样品质量，单位为克(g)；

$f$ ——样液稀释倍数；

1000 ——换算系数；

100 ——换算系数。

计算结果保留三位有效数字。

# 7 精密度

天然食品：在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

强化食品：在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $8\%$ 。

# 8 其他

天然食品，称样量为 $5\mathrm{g}$ 时，本方法的检出限为 $2.5\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $5\mathrm{mg} / 100\mathrm{g}$ ；强化食品，称样量为 $1\mathrm{g}$ 时，本方法的检出限为 $12.5\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $25\mathrm{mg} / 100\mathrm{g}$ 。

# 第二法 气相色谱法

# 9 原理

试样中的肌醇用水和乙醇提取后，与硅烷化试剂衍生，正己烷提取，经气相色谱分离，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 10.1 试剂

10.1.1 无水乙醇 $\mathrm{(C_2H_6O)}$ 。  
10.1.2 正己烷 $(\mathrm{C_6H_{14}})$ 。  
10.1.3 $95\%$ 乙醇 $\mathrm{(C_2H_6O)}$ 。  
10.1.4 三甲基氯硅烷 $(\mathrm{C_3H_9ClSi})$ 。  
10.1.5 六甲基二硅胺烷 $\mathrm{(C_6H_{19}NSi_2)}$   
10.1.6 $N,N$ -二甲基甲酰胺 $\mathrm{(C_3H_7NO)}$

# 10.2 试剂配制

硅烷化试剂：分别吸取体积比为 $1:2$ 的三甲基氯硅烷和六甲基二硅胺烷，超声混匀。现用现配。

注： $N,N$ -二甲基甲酰胺、三甲基氯硅烷和六甲基二硅胺烷必须保证当三者混合后无白色浑浊现象时方可使用。

# 10.3 标准品

10.3.1 肌醇标准品 $\left(\mathrm{C}_{6} \mathrm{H}_{12} \mathrm{O}_{6}\right)$ , CAS号: 87-89-8): 纯度 $\geqslant 99\%$ , 或经国家认证并授予标准物质证书的标准物质。  
10.3.2 肌醇标准溶液 $(0.010\mathrm{mg / mL})$ ：称取 $100\mathrm{mg}$ （精确到 $0.1\mathrm{mg}$ )经过 $105^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 烘干 $2\mathrm{h}$ 的肌醇标准物质于 $100~\mathrm{mL}$ 容量瓶中，用 $25~\mathrm{mL}$ 水溶解完全。用 $95\%$ 的乙醇定容至刻度，混匀。取 $1\mathrm{mL}$ 此溶液于 $100~\mathrm{mL}$ 容量瓶中，用 $70\%$ 的乙醇定容至刻度，混匀。

# 11 仪器和设备

11.1 气相色谱仪：配氢火焰离子化检测器。   
11.2 分析天平：感量为 $0.1\mathrm{mg}$   
11.3 离心机：转速 $\geqslant 5000\mathrm{r / min}$   
11.4 旋转蒸发仪。  
11.5 超声波仪。  
11.6 恒温热水浴槽。  
11.7 烘箱。

# 12 分析步骤

# 12.1 试样处理与衍生

12.1.1 试样处理：固态和粉状试样研磨混合均匀后称取 $1\mathrm{g}$ （精确到 $0.1\mathrm{mg}$ ，于 $50~\mathrm{mL}$ 容量瓶中，加入 $12\mathrm{mL}40^{\circ}\mathrm{C}$ 温水溶解试样；液态试样直接称取 $12\mathrm{g}$ （精确到 $0.1\mathrm{mg}$ )于 $50~\mathrm{mL}$ 容量瓶中。上述试样超声提取 $10\mathrm{min}$ ，用 $95\%$ 乙醇定容至刻度，混匀。静置 $20\mathrm{min}$ 后，取 $10~\mathrm{mL}$ 于 $15~\mathrm{mL}$ 离心管中，以不低于 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。取上清液 $5\mathrm{mL}$ 于旋转蒸发浓缩瓶中。  
12.1.2 干燥与衍生：向浓缩瓶中加入 $10 \mathrm{~mL}$ 无水乙醇，在 $80^{\circ} \mathrm{C} \pm 5^{\circ} \mathrm{C}$ 下旋转浓缩至近干时再加入 $5 \mathrm{~mL}$ 无水乙醇继续浓缩至干燥，转移浓缩瓶至烘箱中 $100^{\circ} \mathrm{C} \pm 5^{\circ} \mathrm{C}$ 烘干 $1 \mathrm{~h}$ 。加入 $10.0 \mathrm{~mL} N, N-$ 二甲基甲酰胺，超声溶解 $5 \mathrm{~min}$ 并转移至 $25 \mathrm{~mL}$ 有螺纹盖的离心管中，加入硅烷化试剂 $3.0 \mathrm{~mL}$ 并放于 $80^{\circ} \mathrm{C} \pm 5^{\circ} \mathrm{C}$ 水浴中衍生反应 $75 \mathrm{~min}$ 。其间每隔 $20 \mathrm{~min}$ 取出振荡一次，然后取出冷却至室温。加入 $5 \mathrm{~mL}$ 正己烷，振荡混合后静置分层。取上层液 $3 \mathrm{~mL}$ 于预先加少许无水硫酸钠的带螺纹盖离心管中，振荡后以不低于 $4000 \mathrm{r} / \mathrm{min}$ 离心，此为试样测定液。

# 12.2 肌醇标准测定液的制备

分别吸取 $0.0\mathrm{mL},2.0\mathrm{mL},4.0\mathrm{mL},6.0\mathrm{mL},8.0\mathrm{mL},10.0\mathrm{mL}$ 肌醇标准溶液于浓缩瓶中，按12.1.2步骤操作。

# 12.3 测定

# 12.3.1 参考色谱条件

参考色谱条件列出如下：

a）色谱柱：填料为 $50\%$ 氰丙基-甲基聚硅氧烷的毛细管柱，柱长 $60~\mathrm{m}$ ，内径 $0.25~\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ；或同等性能的色谱柱；  
b）进样口温度： $280^{\circ}C$

# 12.3.2 标准曲线制作

分别将标准溶液测定液注入到气相色谱仪中（色谱图见图B.1），以测得的峰面积(或峰高)为纵坐标，以肌醇标准测定液中肌醇的含量 $(\mathrm{mg})$ 为横坐标制作标准曲线。

# 12.3.3 试样溶液的测定

分别将试样测定液注入到气相色谱仪中得到峰面积(或峰高)，从标准曲线中获得试样测定液中肌醇的含量 $(\mathrm{mg})$ 。

# 13 分析结果的表述

试样中肌醇含量按式(2)计算：

$$
X = \frac {C _ {s} \times f _ {i}}{m _ {i}} \times 1 0 0 \tag {2}
$$

式中：

$X$ ——试样中肌醇含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$C_s$ ——从标准曲线中获得试样测定液肌醇的含量，单位为毫克 $(\mathrm{mg})$ ；

$f_{i}$ ——试样测定液所含肌醇换算成试样中所含肌醇的系数为10；

$m_{i}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

100 ——换算系数。

计算结果保留小数点后一位。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

固体或粉末样品检出限为 $1.0\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $3.0\mathrm{mg} / 100\mathrm{g}$ 。

液体样品检出限为 $0.2\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.5\mathrm{mg} / 100\mathrm{g}$ 。

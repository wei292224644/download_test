# 前言

本标准代替GB/T5009.94-2003《植物性食品中稀土的测定》，并同时代替GB/T22290-2008《茶叶中稀土元素的测定电感耦合等离子体质谱法》，GB/T23199-2008《茶叶中稀土元素的测定电感耦合等离子体发射光谱法和电感耦合等离子体质谱法》，GB/T7630-1987<大米小麦中氧化稀土总量的测定三溴偶氮脾分光光度法》。

本标准与GB/T5009.94-2003相比，主要变化如下：

——名称由《植物性食品中稀土的测定》改为《植物性食品中稀土元素的测定》；  
——增加了电感耦合等离子体质谱法；  
——删除了分光光度三波长法。

# 食品安全国家标准

# 植物性食品中稀土元素的测定

# 1 范围

本标准规定了用电感耦合等离子体质谱法测定植物性食品中稀土元素的方法。

本标准适用于谷类粮食、豆类、蔬菜、水果、茶叶等植物性食品中钪（Sc）、钇（Y）、镧（La）、铈（Ce）、镨（Pr）、钕（Nd）、钐（Sm）、铕（Eu）、钆（Gd）、铽（Tb）、镝（Dy）、钬（Ho）、铒（Er）、铥（Tm）、镱（Yb）、镥（Lu）的测定。

# 2 原理

样品经消解处理为样品溶液，样品溶液经雾化由载气送入ICP或送入等离子体炬管中，经过蒸发、解离、原子化和离子化等过程，转化为带正电荷的离子，经离子采集系统进入质谱仪，质谱仪根据质荷比进行分离。对于一定的质荷比，质谱的信号强度与进入质谱仪的离子数成正比，即样品浓度与质谱信号强度成正比。通过测量质谱的信号强度来测定试样溶液的元素浓度。

# 3 试剂和材料

# 3.1 试剂

注：除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的一级水。

3.1.1 硝酸（ $\mathrm{HNO}_3$ ）。

3.1.2 氩气（Ar）：高纯氩气（>99.999%）或液氩。

# 3.2 试剂配制

硝酸溶液 $(5 + 95)$ ：取 $50 \mathrm{~mL}$ 硝酸，用水稀释至 $1000 \mathrm{~mL}$ 。

# 3.3 标准品

3.3.1 稀土元素贮备液（ $10\mu \mathrm{g} / \mathrm{mL}$ ）(Sc、Y、La、Ce、Pr、Nd、Sm、Eu、Gd、Tb、Dy、Ho、Er、Tm、Yb、Lu)。  
3.3.2 内标贮备液（ $10\mu \mathrm{g} / \mathrm{mL}$ ）（Rh、In、Re）。  
3.3.3 仪器调谐贮备液（10 ng/mL）（Li、Co、Ba、Tl）。

# 3.4 标准溶液配制

3.4.1 稀土元素混合标准使用溶液（ $100\mathrm{ng/mL}$ ）：取适量 $\mathrm{Sc}$ 、Y、La、Ce、Pr、Nd、Sm、Eu、Gd、Tb、Dy、Ho、Er、Tm、Yb、Lu的各元素单标标准储备溶液或元素混合标准贮备溶液，用硝酸溶液逐级稀释至浓度为 $100.0\mu\mathrm{g/L}$ 的元素混合标准使用溶液。

3.4.2 标准曲线工作液：取适量元素混合标准使用溶液，用硝酸溶液配制成浓度为 $0\mu \mathrm{g} / \mathrm{L}$ 、0.0500 $\mu \mathrm{g} / \mathrm{L}$ 、 $0.100\mu \mathrm{g} / \mathrm{L}$ 、 $0.500\mu \mathrm{g} / \mathrm{L}$ 、 $1.00\mu \mathrm{g} / \mathrm{L}$ 、 $2.00\mu \mathrm{g} / \mathrm{L}$ 的标准系列或浓度为 $0\mu \mathrm{g} / \mathrm{L}$ 、 $1.00\mu \mathrm{g} / \mathrm{L}$ 、 $2.00\mu \mathrm{g} / \mathrm{L}$ 、 $5.00\mu \mathrm{g} / \mathrm{L}$ 、 $10.0\mu \mathrm{g} / \mathrm{L}$ 、 $20.0\mu \mathrm{g} / \mathrm{L}$ 的标准系列，亦可依据样品溶液中稀土元素浓度适当调节标准系列浓度范围。

3.4.3 内标使用液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：取适量内标贮备液（ $10~\mu \mathrm{g / mL}$ ），用硝酸溶液（5+95）稀释10倍，浓度为 $1\mu \mathrm{g / mL}$ 。  
3.4.4 仪器调谐使用液（1 ng/mL）：取适量仪器调谐贮备液，用硝酸溶液（5+95）稀释10倍，浓度为1 ng/mL。

# 4 仪器和设备

4.1 电感耦合等离子体质谱仪（ICP-MS）。  
4.2 天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$ 。  
4.3 高压密闭微波消解系统，配有聚四氟乙烯高压消解罐。  
4.4 密闭高压消解器，配有消解内罐。  
4.5 恒温干燥箱（烘箱）。  
4.6 $50^{\circ}\mathrm{C} \sim 200^{\circ}\mathrm{C}$ 控温电热板。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 试样预处理

5.1.1.1 干样：谷类粮食、豆类等取可食部分，经高速粉碎机粉碎，混匀，备用。  
5.1.1.2 湿样：蔬菜、水果等取可食部分，水洗干净，晾干或纱布揩干，经匀浆器匀浆，备用。

# 5.2 试样消解

5.2.1 微波消解：称取 $0.2 \mathrm{~g} \sim 0.5 \mathrm{~g}$ （精确到 $0.001 \mathrm{~g}$ ）于高压消解罐中，加入 $5 \mathrm{~mL} \mathrm{HNO}_{3}$ ，旋紧罐盖，放置 $1 \mathrm{~h}$ ，按照微波消解仪的标准操作步骤进行消解（消解条件参见表 A.1）。冷却后取出，缓慢打开罐盖排气，将高压消解罐放入控温电热板上，于 $140^{\circ} \mathrm{C}$ 赶酸。消解罐取出放冷，将消化液转移至 $10 \mathrm{~mL} \sim 25 \mathrm{~mL}$ 容量瓶中，用少量水分 3 次洗涤罐，洗液合并于容量瓶中并定容至刻度，混匀备用；同时作试剂空白。  
5.2.2 密闭高压罐消解：称取样品 $0.5\mathrm{g}\sim 1.\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ）于消解内罐中，加入 $5\mathrm{mL}$ 硝酸浸泡过夜。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱， $140~^\circ \mathrm{C}\sim 160~^\circ \mathrm{C}$ 保持 $4\mathrm{h}\sim 6\mathrm{h}$ ，在箱内自然冷却至室温，缓慢旋松不锈钢外套，将消解内罐取出，放在控温电热板上，于 $140~^\circ \mathrm{C}$ 赶酸。消解内罐放冷后，将消化液转移至 $10\mathrm{mL}\sim 25\mathrm{mL}$ 容量瓶中，用少量水分3次洗涤罐，洗液合并于容量瓶中并定容至刻度，混匀备用；同时作试剂空白。

# 5.3 仪器参考条件

5.3.1 按照仪器标准操作规程进行仪器起始化、质量校准、氩气流量等的调试。选择合适条件，包括雾化器流速、检测器和离子透镜电压、射频入射功率等，使氧化物形成 $\mathrm{CeO / Ce < 1\%}$ 和双电荷化合物形成[70/140] $< 3\%$ 。  
5.3.2 测定参考条件：在调谐仪器达到测定要求后，编辑测定方法、干扰校正方程（校正钠（Eu）元素）及选择各待测元素同位素钪（ ${}^{45}\mathrm{Sc}$ ）、钇（ ${}^{89}\mathrm{Y}$ ）、镧（ ${}^{139}\mathrm{La}$ ）、铈（ ${}^{140}\mathrm{Ce}$ ）、镨（ ${}^{141}\mathrm{Pr}$ ）、钕（ ${}^{146}\mathrm{Nd}$ ）、钐（ ${}^{147}\mathrm{Sm}$ ）、铕（ ${}^{153}\mathrm{Eu}$ ）、钆（ ${}^{157}\mathrm{Gd}$ ）、铽（ ${}^{159}\mathrm{Tb}$ ）、镝（ ${}^{163}\mathrm{Dy}$ ）、钬（ ${}^{165}\mathrm{Ho}$ ）、铒（ ${}^{166}\mathrm{Er}$ ）、铥（ ${}^{169}\mathrm{Tm}$ ）、镱（ ${}^{172}\mathrm{Yb}$ ）、镥（ ${}^{175}\mathrm{Lu}$ ），在线引入内标使用溶液，观测内标灵敏度，使仪器产生的信号强度为 $400000 \sim 600000$ cps（仪器操作参考条件参见附录A.2）。测定脉冲模拟转换系数，符合要求后，将试剂空白、

标准系列、样品溶液依次进行测定。对各被测元素进行回归分析，计算其线性回归方程。

铸（Eu）元素校正方程采用： $[^{151}\mathrm{Eu}] = [151] - [(\mathrm{Ba}(135)\mathrm{O}) / \mathrm{Ba}(135)]\times [135]$ 。式中，[（Ba(135)O)/Ba(135)]为氧化物比，[151]、[135]分别为质量数151和135处的质谱的信号强度CPS。

# 5.4 标准曲线的制作

将标准系列工作液分别注入电感耦合等离子质谱仪中，测定相应的信号响应值，以标准工作液的浓度为横坐标，以响应值——离子每秒计数值（CPS）为纵坐标，绘制标准曲线。

# 5.5 试样溶液的测定

将试样溶液注入电感耦合等离子质谱仪中，得到相应的信号响应值，根据标准曲线得到待测液中相应元素的浓度，平行测定次数不少于两次。

# 6 分析结果的表述

试样中第i个稀土元素含量按照式（1）计算：

$$
X _ {i} = \frac {\left(C _ {i} - C _ {i o}\right) \times V}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X_{i}$ ——样品中第 $i$ 个稀土元素含量，单位为毫克每千克（ $\mathrm{mg/kg}$ ）；

$C_i$ ——样液中第 $i$ 个稀土元素测定值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{i0}$ ——样品空白液中第 $i$ 个稀土元素测定值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——样品消化液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品称样量，单位为克（g）；

1000——单位转换。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，保留3位有效数字。

若分析结果需要以氧化物含量表示，则参见附录B，将各元素含量乘以换算系数F。

# 7 精密度

样品中的钪、钇、镧、铈、钕等稀土元素含量大于 $10\mu \mathrm{g} / \mathrm{kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ ，样品中稀土元素含量小于 $10\mu \mathrm{g} / \mathrm{kg}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

本标准的检出限：取样 $0.5\mathrm{g}$ ，定容 $10\mathrm{mL}$ ，测定各稀土元素的检出限（ $\mu \mathrm{g} / \mathrm{kg}$ ）分别为Sc 0.6，Y 0.3，La 0.4，Ce 0.3，Pr 0.2，Nd 0.2，Sm 0.2，Eu 0.06，Gd 0.1，Tb 0.06，Dy 0.08，Ho 0.03，Er 0.06，Tm 0.03，Yb 0.06，Lu 0.03。定量限（ $\mu \mathrm{g} / \mathrm{kg}$ ）分别为Sc 2.1，Y 1.1，La 1.4，Ce 0.9，Pr 0.7，Nd 0.8，Sm 0.5，Eu 0.2，Gd 0.5，Tb 0.2，Dy 0.3，Ho 0.1，Er 0.2，Tm 0.1，Yb 0.2，Lu 0.1。

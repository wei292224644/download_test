# 食品安全国家标准

# 食品添加剂 香茅醛

# 1 范围

本标准适用于由香茅油或柠檬桉油为原料经单离、或以柠檬醛为原料经化学反应制得的食品添加剂香茅醛。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

# 3,7-二甲基-6-辛烯醛

# 2.2 分子式

$$
\mathrm {C} _ {1 0} \mathrm {H} _ {1 8} \mathrm {O}
$$

# 2.3 结构式

![](images/69d22a1a5c8fdeed042bf3f9ed650ec3e076ec27795c1146a5284b903a7ddce0.jpg)

# 2.4 相对分子质量

154.25（按2007年国际相对原子质量）

# 附录A

# 香茅醛含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

试样制备：量取本品 $0.05\mathrm{mL}$ 溶于 $1\mathrm{mL}95\%$ （体积分数）乙醇中，摇匀备用。或直接取样测定。

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行，应符合要求。

食品添加剂香茅醛气相色谱及操作条件参见附录B、附录C、附录D。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ 。  
B.2.2 固定相：硝基对苯二酸改性的聚乙二醇。  
B.2.3 膜厚: $0.5 \mu \mathrm{m}$ 。  
B.2.4 色谱炉温度：线性程序升温从 $80^{\circ}\mathrm{C}$ 至 $240^{\circ}\mathrm{C}$ ，速率 $4^{\circ}\mathrm{C} / \mathrm{min}$ ，然后在 $240^{\circ}\mathrm{C}$ 恒温 $10\mathrm{min}$ 。  
B.2.5 进样口温度： $250^{\circ}C$ 。  
B.2.6 检测器温度： $250^{\circ}C$ 。  
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 柱流量： $1\mathrm{mL} / \mathrm{min}$   
B.2.10 进样量： $0.2\mu \mathrm{L}$   
B.2.11 分流比： $120:1$

C.2.8 载气：氮气。  
C.2.9 柱前压： $36.5\mathrm{kPa}$   
C.2.10 进样量： $1.0\mu \mathrm{L}$   
C.2.11 分流比： $50:1$

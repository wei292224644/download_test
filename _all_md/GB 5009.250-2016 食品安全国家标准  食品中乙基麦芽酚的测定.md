# 前言

本标准代替GB/T21917—2008《饮料中乙基麦芽酚的测定方法》。

本标准与GB/T21917—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中乙基麦芽酚的测定”；  
——增加了标准的适用范围；  
——增加了不同类型食品试样的前处理方法；  
——修改了液相色谱条件；  
——增加了附录B质谱确证。

# 食品安全国家标准

# 食品中乙基麦芽酚的测定

# 1 范围

本标准规定了食品中乙基麦芽酚含量的高效液相色谱测定方法。

本标准适用于饮料、糖果、果冻、肉制品、饼干、面包、糕点、乳粉食品中乙基麦芽酚含量的测定。

# 2 原理

试样经提取、净化后，采用配有二极管阵列检测器或紫外检测器的高效液相色谱仪检测，外标法定量；阳性样品需要采用质谱法进行定性确认。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.3 磷酸二氢钠 $(\mathrm{NaH_2PO_4}\cdot 2\mathrm{H_2O})$ 。

# 3.2 磷酸二氢钠溶液配制

称取 $3.90\mathrm{g}$ 磷酸二氢钠，加水溶解并稀释至近 $1000~\mathrm{mL}$ ，经磷酸调节 $\mathrm{pH}$ 至 $4.0 \pm 0.1$ ，定容至 $1000~\mathrm{mL}$ ，经微孔滤膜过滤备用。

# 3.3 乙基麦芽酚 $(\mathrm{C_7H_8O_3})$ 标准品

纯度不小于 $99\%$ 。

# 3.4 标准溶液配制

3.4.1 标准储备液：称取乙基麦芽酚 $0.1\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），用甲醇溶解，并定容至 $100~\mathrm{mL}$ 容量瓶中，此溶液浓度为 $1\mathrm{mg / mL}$ 。  
3.4.2 标准系列工作液：分别准确吸取不同体积的标准储备液，用甲醇将其稀释成乙基麦芽酚含量分别为 $0.0\mu \mathrm{g / mL}, 0.5\mu \mathrm{g / mL}, 2.0\mu \mathrm{g / mL}, 5.0\mu \mathrm{g / mL}, 25.0\mu \mathrm{g / mL}, 100.0\mu \mathrm{g / mL}$ 的标准系列工作液。

# 4 仪器和设备

4.1 高效液相色谱仪（HPLC)：配有二极管阵列检测器(DAD)或紫外检测器(UVD)。  
4.2 超声波清洗器。

4.3 涡旋混合仪。  
4.4 水浴锅。  
4.5 离心机：转速不低于 $6000\mathrm{r / min}$   
4.6 分析天平：感量分别为 $0.01\mathrm{g}, 0.001\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 碳酸饮料、果汁饮料、乳饮料、植物蛋白饮料试样

准确称取 $10\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）（碳酸饮料需先超声 $2\mathrm{min} \sim 3\mathrm{min}$ 以除去二氧化碳后取样）于 $25\mathrm{mL}$ 具塞刻度试管中，用乙腈定容至刻度，混匀，超声 $10\mathrm{min}$ （若样品溶液浑浊时， $6000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ），取上清液，经微孔滤膜过滤，滤液进液相色谱仪分析。

# 5.1.2 糖果、果冻试样

准确称取 $2\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ ）于 $25\mathrm{mL}$ 具塞刻度试管中，加入 $20\mathrm{mL}$ 水，置于 $60^{\circ}\mathrm{C} \sim 70^{\circ}\mathrm{C}$ 水浴锅中加热 $2\mathrm{min} \sim 3\mathrm{min}$ ，加塞，剧烈振摇使其分散均匀，再置于 $60^{\circ}\mathrm{C} \sim 70^{\circ}\mathrm{C}$ 水浴锅中加热 $30\mathrm{min}$ ，取出后趁热超声 $5\mathrm{min}$ ，冷却后用甲醇定容至刻度（奶糖用乙腈定容）， $6000\mathrm{r/min}$ 离心 $10\mathrm{min}$ ，取上清液，经微孔滤膜过滤，滤液进液相色谱仪分析。

# 5.1.3 肉制品、饼干、面包、糕点试样

准确称取 $2\mathrm{g}$ 粉碎均匀试样（精确至 $0.001\mathrm{g}$ ）至 $25~\mathrm{mL}$ 具塞刻度试管中，加入 $5\mathrm{mL}$ 水，加塞，涡旋 $1\mathrm{min}$ ，使其分散均匀后，再加 $15~\mathrm{mL}$ 甲醇，加塞，摇匀，超声 $20\mathrm{min}$ ，用甲醇定容至刻度， $6000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液，微孔滤膜过滤，滤液进液相色谱仪分析。

# 5.1.4 乳粉试样

准确称取 $2\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ )至 $25~\mathrm{mL}$ 具塞刻度试管中，加入 $8\mathrm{mL}$ 水，加塞，涡旋 $1\mathrm{min}$ ，使其充分溶解后，加 $15~\mathrm{mL}$ 乙腈，加塞，摇匀，超声 $20\mathrm{min}$ ，用乙腈定容至刻度， $6000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液，经微孔滤膜过滤，滤液进液相色谱仪分析。

# 5.2 仪器参考条件

5.2.1 色谱柱： $\mathrm{C_{18}}$ 柱， $250 \mathrm{~mm} \times 4.6 \mathrm{~mm}, 5 \mu \mathrm{m}$ ，或性能相当者。  
5.2.2 流动相：A为甲醇；B为磷酸二氢钠溶液，梯度洗脱条件见表1。

# 5.3 标准曲线的制作

将标准系列工作液分别注入高效液相色谱仪中，测定相应的乙基麦芽酚的色谱峰面积，以标准工作液的浓度为横坐标，以色谱峰的峰面积为纵坐标，绘制标准曲线。

# 5.4 试样溶液的测定

将试样溶液注入高效液相色谱仪中，得到相应的乙基麦芽酚的色谱峰面积，根据标准曲线得到待测液中乙基麦芽酚的浓度。

乙基麦芽酚的标准液相色谱图参见图A.1。

# 5.5 定性确认

进行试样测定时，如果检出乙基麦芽酚的色谱峰的保留时间与标准品相一致，同时该物质的紫外吸收图谱与标准品的紫外吸收图谱相一致，则可初步确认试样中存在被测的乙基麦芽酚。阳性试样需用质谱仪进行确认试验(见附录B)。

# 6 分析结果的表述

试样中乙基麦芽酚含量按式(1)计算：

$$
X = \frac {c \times V \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中乙基麦芽酚的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——由标准曲线得出的样液中乙基麦芽酚的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当取样量为 $2\mathrm{g}$ 时，糖果、果冻、肉制品、饼干、面包、糕点、乳粉等类食品的方法检出限为 $1.5\mathrm{mg / kg}$ ，定量限为 $5\mathrm{mg / kg}$ ；当取样量为 $10\mathrm{g}$ 时，饮料的方法检出限为 $0.6\mathrm{mg / kg}$ ，定量限为 $2\mathrm{mg / kg}$ 。

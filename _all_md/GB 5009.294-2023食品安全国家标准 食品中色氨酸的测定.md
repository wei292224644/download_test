# 食品安全国家标准

# 食品中色氨酸的测定

# 1 范围

本标准规定了食品中色氨酸的测定方法。

本标准适用于婴幼儿配方食品、特殊医学用途配方食品和婴幼儿谷类辅助食品中色氨酸的测定。

# 第一法 酶水解-液相色谱法

# 2 原理

试样以蛋白酶酶解，经甲醇-水稀释后，用反相液相色谱柱分离，荧光检测器或紫外检测器检测，外标法定量。

# 3 试剂和材料

除另有规定外，所有试剂均为分析纯，水为GB/T6682中规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 乙酸 $\mathrm{(C_2H_4O_2)}$ ：色谱纯。  
3.1.3 乙酸铵 $\mathrm{(C_2H_7NO_2)}$ ：色谱纯。  
3.1.4 盐酸 $(\mathrm{HCl}):12\mathrm{mol} / \mathrm{L}$   
3.1.5 2-氨基-2-羟甲基-1,3-丙二醇 $\mathrm{(C_4H_{11}NO_3}$ ,Tris碱）。  
3.1.6 灰色链霉菌蛋白酶(XIV型；来源于灰色链霉菌)：活力 $\geqslant 3.5\mathrm{U / mg}$ 。

# 3.2 试剂配制

3.2.1 盐酸溶液 $(1.0\mathrm{mol} / \mathrm{L})$ ：量取 $83~\mathrm{mL}$ 盐酸，注入 $300~\mathrm{mL}$ 水中，混匀后加水稀释至 $1\mathrm{L}$ 。  
3.2.2 Tris缓冲液 $(0.1\mathrm{mol} / \mathrm{L},\mathrm{pH} = 8.5)$ ：称取 $6.06\mathrm{g}$ Tris碱于 $1000~\mathrm{mL}$ 烧杯中，加水 $500~\mathrm{mL}$ 溶解，用 $1.0\mathrm{mol} / \mathrm{L}$ 盐酸溶液调节溶液 $\mathrm{pH}$ 至 $8.5\pm 0.05$ 。室温下保存，保存期6个月。  
3.2.3 蛋白酶溶液 $(17.5 \mathrm{U} / \mathrm{mL})$ : 称取 $0.05 \mathrm{~g}$ 灰色链霉菌蛋白酶, 用 Tris 缓冲液溶解并定容至 $10 \mathrm{~mL}$ ,使蛋白酶质量浓度为 $17.5 \mathrm{U} / \mathrm{mL}$ , 临用现配。  
3.2.4 乙酸铵溶液 $(10\mathrm{mmol / L},\mathrm{pH} = 4.0)$ ：称取 $0.77\mathrm{g}$ 乙酸铵，加入 $900~\mathrm{mL}$ 水溶解，用乙酸调节溶液 $\mathrm{pH}$ 到 $4.0\pm 0.1$ 后加水至 $1000~\mathrm{mL}$ ，经 $0.22\mu \mathrm{m}$ 水相微孔滤膜过滤后备用。

# 3.3 标准品

L-色氨酸标准品 $\mathrm{(C_{11}H_{12}N_2O_2}$ ,CAS号：73-22-3)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书

的标准物质。

# 3.4 标准溶液配制

3.4.1 L-色氨酸标准储备液 $(1.00\mathrm{mg / mL})$ ：准确称取 $25\mathrm{mg}$ （精确至 $0.0001\mathrm{g})$ L-色氨酸标准品于烧杯中，加入 $10~\mathrm{mL}$ 水溶解并转移至 $25~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。转移至棕色玻璃容器中，于 $4^{\circ}C$ 冰箱中保存，保存期6个月。  
3.4.2 L-色氨酸标准中间液 $(100\mu \mathrm{g / mL})$ ：吸取L-色氨酸标准储备液 $1.00~\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，加水定容至刻度，混匀。转移至棕色玻璃容器中，于 $4^{\circ}C$ 冰箱中保存，保存期6个月。  
3.4.3 L-色氨酸标准系列工作液：分别移取L-色氨酸标准中间液 $20.0\mu \mathrm{L},100\mu \mathrm{L},500\mu \mathrm{L},1.00\mathrm{mL}$ 、 $2.00~\mathrm{mL},5.00~\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，各加入 $2.40~\mathrm{mL}$ 甲醇，加水定容至刻度，混匀。L-色氨酸标准系列工作液的质量浓度分别为 $0.200\mu \mathrm{g / mL},1.00\mu \mathrm{g / mL},5.00\mu \mathrm{g / mL},10.0\mu \mathrm{g / mL},20.0\mu \mathrm{g / mL}$ ， $50.0\mu \mathrm{g / mL}$ 。临用现配。

# 3.5 材料

3.5.1 螺纹带盖的离心管。  
3.5.2 微孔滤膜： $0.22\mu \mathrm{m}$ ，有机相。  
3.5.3 微孔滤膜： $0.22\mu \mathrm{m}$ ，水相。

# 4 仪器和设备

4.1 高效液相色谱仪：带有荧光检测器或紫外检测器。  
4.2 分析天平：感量分别为 $0.01\mathrm{g}, 0.001\mathrm{g}, 0.0001\mathrm{g}$ 。  
4.3 恒温培养箱： $50^{\circ}\mathrm{C} \pm 0.5^{\circ}\mathrm{C}$ 。  
4.4 超声波发生器。  
4.5 pH计：精度为 $\pm 0.01$   
4.6 涡旋混合器。

# 5 分析步骤

# 5.1 试样处理

# 5.1.1 试样制备

婴幼儿配方食品、特殊医学用途配方食品应混匀。婴幼儿谷类辅助食品应预先粉碎并混匀。

# 5.1.2 试样提取

准确称取试样 $0.2\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ）于 $50~\mathrm{mL}$ 螺纹带盖的离心管中，依次加入 $0.5\mathrm{mL}$ 蛋白酶溶液、 $3.0\mathrm{mL}$ Tris缓冲液、 $200\mu \mathrm{L}$ 甲醇，涡旋混匀后超声溶解，于 $50^{\circ}\mathrm{C}$ 恒温培养箱中酶解 $16\mathrm{h}$ 。

取出离心管，冷却至室温，加入 $12\mathrm{mL}$ 甲醇，用水将样品转移并定容至 $50~\mathrm{mL}$ 容量瓶中，混匀，经 $0.22\mu \mathrm{m}$ 有机相微孔滤膜过滤，滤液供液相色谱测定。

# 5.2 液相色谱参考条件

液相色谱参考条件如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $150~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，填料粒径 $3.5\mu \mathrm{m}$ ，或相当者。  
b） 柱温： $35^{\circ}C$   
c）流动相：甲醇 $+$ 乙酸铵溶液 $= 10 + 90$   
d) 流速: $0.8 \mathrm{~mL} / \mathrm{min}$ 。  
e）荧光检测器：激发波长为 $280~\mathrm{nm}$ ，发射波长为 $346~\mathrm{nm}$ 。紫外检测器：检测波长 $280~\mathrm{nm}$ 。  
f）进样量： $5\mu \mathrm{L}$

# 5.3 空白试验

除不加试样外，其他操作与试样的操作一致。

# 5.4 标准曲线的制作

将L-色氨酸标准系列工作液分别注入液相色谱仪中，测得相应的色谱峰面积，以标准系列工作液中L-色氨酸的质量浓度为横坐标，以峰面积为纵坐标，绘制标准曲线，L-色氨酸标准溶液的色谱图参见附录A中图A.1和图A.2。

# 5.5 试样溶液的测定

将空白溶液和试样溶液分别注入液相色谱仪中，得到峰面积，根据标准曲线得到待测溶液中色氨酸的含量。

# 6 分析结果的表述

试样中色氨酸的含量按式(1)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中色氨酸的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$\rho$ ——由标准曲线得到的试样溶液中色氨酸的含量，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$\rho_0$ ——由标准曲线得到的空白溶液中色氨酸的含量，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$ ；

100 ——换算系数；

$m$ ——试样的质量，单位为克(g)；

1000——换算系数。

计算结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当称样量为 $0.2\mathrm{g}$ 时，方法的检出限为 $1.5\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $5.0\mathrm{mg} / 100\mathrm{g}$ 。

# 第二法 碱水解-液相色谱法

# 9 原理

试样在碱性条件下水解，水解液用盐酸调至中性，经反相液相色谱柱分离，荧光检测器检测，内标法定量。

# 10 试剂和材料

除另有规定外，所有试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 中规定的一级水。

# 10.1 试剂

10.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
10.1.2 乙酸 $\mathrm{(C_2H_4O_2)}$ ：色谱纯。  
10.1.3 乙酸铵 $\mathrm{(C_2H_7NO_2)}$ ：色谱纯。  
10.1.4 氢氧化锂 $(\mathrm{LiOH}\cdot \mathrm{H}_2\mathrm{O})$ 。  
10.1.5 盐酸 $(\mathrm{HCl})$ ： $12\mathrm{mol} / \mathrm{L}$

# 10.2 试剂配制

10.2.1 盐酸溶液 $(3\mathrm{mol} / \mathrm{L})$ ：量取 $100~\mathrm{mL}$ 盐酸，注入 $300~\mathrm{mL}$ 水中，混匀。  
10.2.2 氢氧化锂溶液 $(4\mathrm{mol} / \mathrm{L})$ ：称取 $167.8\mathrm{g}$ 氢氧化锂加入适量水溶解，用水定容至 $1000~\mathrm{mL}$ 。  
10.2.3 乙酸铵溶液 $(10\mathrm{mmol / L},\mathrm{pH} = 4.0)$ ：称取 $0.77\mathrm{g}$ 乙酸铵加入 $900~\mathrm{mL}$ 水溶解，用乙酸调节溶液 $\mathrm{pH}$ 至 $4.0\pm 0.1$ ，用水定容至 $1000~\mathrm{mL}$ ，摇匀，经 $0.22\mu \mathrm{m}$ 水相微孔滤膜过滤备用。

# 10.3 标准品

10.3.1 L-色氨酸标准品 $\mathrm{(C_{11}H_{12}N_2O_2}$ ,CAS号：73-22-3)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。  
10.3.2 $\alpha^{-}$ 甲基-D-色氨酸标准品（ $\mathrm{C_{12}H_{14}N_2O_2}$ ，CAS号：110117-83-4）；纯度 $\geq 98\%$

# 10.4 标准溶液配制

10.4.1 L-色氨酸标准储备液 $(1.00\mathrm{mg / mL})$ ：准确称取 $25~\mathrm{mg}$ （精确至 $0.0001\mathrm{g})$ L-色氨酸标准品于烧杯中，加入 $10~\mathrm{mL}$ 水溶解并转移至 $25~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。转移至棕色玻璃容器中，于 $4^{\circ}C$ 冰箱中保存，保存期6个月。  
10.4.2 L-色氨酸标准中间液 $(100\mu \mathrm{g / mL})$ ：吸取L-色氨酸标准储备液 $1.00~\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。转移至棕色玻璃容器中，于 $4^{\circ}C$ 冰箱中保存，保存期6个月。  
10.4.3 内标工作液 $(1\mathrm{mg / mL})$ ：称取 $10\mathrm{mg}$ （精确至 $0.001\mathrm{g})\alpha$ -甲基-D-色氨酸标准品于烧杯中，加入 $5\mathrm{mL}$ 水溶解并转移至 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。转移至棕色玻璃容器中，于 $4^{\circ}C$ 冰箱中保存，保存期6个月。  
10.4.4 L-色氨酸标准系列工作液：分别移取L-色氨酸标准中间液 $20~\mu \mathrm{L}, 100~\mu \mathrm{L}, 500~\mu \mathrm{L}, 1.00~\mathrm{mL}$ 、 $2.00~\mathrm{mL}, 5.00~\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，各加入 $100~\mu \mathrm{L}$ 内标工作液，加水定容至刻度，混匀。L-色氨酸标准系列工作液的质量浓度分别为 $0.20~\mu \mathrm{g} / \mathrm{mL}, 1.00~\mu \mathrm{g} / \mathrm{mL}, 5.00~\mu \mathrm{g} / \mathrm{mL}, 10.0~\mu \mathrm{g} / \mathrm{mL}, 20.0~\mu \mathrm{g} / \mathrm{mL}, 50.0~\mu \mathrm{g} / \mathrm{mL}$ 。临用现配。

# 10.5 材料

10.5.1 水解管：聚四氟乙烯材质，带螺纹口外盖，内衬聚四氟乙烯密封垫。  
10.5.2 微孔滤膜： $0.22\mu \mathrm{m}$ ，水相。

# 11 仪器和设备

11.1 高效液相色谱仪：带有荧光检测器。  
11.2 分析天平：感量分别为 $0.01\mathrm{g}, 0.001\mathrm{g}, 0.0001\mathrm{g}$ 。  
11.3 电热干燥箱： $110^{\circ}\mathrm{C}\pm 0.5^{\circ}\mathrm{C}$   
11.4 超声波发生器。  
11.5 涡旋混合器。  
11.6 pH计：精度为 $\pm 0.01$

# 12 分析步骤

# 12.1.1 试样制备

婴幼儿配方食品、特殊医学用途配方食品应混匀。婴幼儿谷类辅助食品应预先粉碎并混匀。

# 12.1.2 试样提取

准确称取试样 $0.2\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ）于水解管中，依次加入 $0.5\mathrm{mL}$ 内标工作液、 $5\mathrm{mL}$ 氢氧化锂溶液，涡旋混匀后于超声波振荡器中振荡 $5\mathrm{min}$ ，置于 $110^{\circ}\mathrm{C}$ 电热干燥箱中水解 $20\mathrm{h}$ 。

取出水解管，待冷却至室温后，用盐酸溶液调 $\mathrm{pH}$ 至 $6\sim 8$ ，将水解液全部转移至 $50~\mathrm{mL}$ 容量瓶中并用水定容至刻度，混匀后经 $0.22\mu \mathrm{m}$ 微孔滤膜过滤备用。

# 12.2 液相色谱参考条件

液相色谱参考条件如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $150~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，填料粒径 $3.5\mu \mathrm{m}$ ，或相当者。  
b）流动相：A相：甲醇，B相：乙酸铵溶液，参考梯度洗脱程序见表1。  
c) 流速: $0.8 \mathrm{~mL} / \mathrm{min}$ 。  
d） 柱温： $35^{\circ}C$   
e）荧光检测器：激发波长为 $280~\mathrm{nm}$ ，发射波长为 $346~\mathrm{nm}$ 。  
f）进样量： $5\mu \mathrm{L}$

# 12.3 空白试验

除不加试样外，其他操作与试样的操作一致。

# 12.4 标准曲线的制作

将L-色氨酸标准系列工作液分别注入液相色谱仪中，测得相应的色谱峰面积，以标准系列工作液中L-色氨酸的质量浓度为横坐标，以L-色氨酸峰面积与内标峰面积的比值为纵坐标，绘制标准曲线，标准溶液的色谱图参见附录A中图A.3。

# 12.5 试样溶液的测定

将空白溶液和试样溶液分别注入液相色谱仪中，得到的空白溶液和试样溶液中目标化合物的峰面积与内标化合物峰面积的比值，根据标准曲线得到相应的色氨酸含量。

# 13 分析结果的表述

试样中色氨酸的含量按式(2)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中色氨酸的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$\rho$ ——试样中色氨酸色谱峰与对应内标物色谱峰的峰面积比值，经标准曲线求得的对应色氨酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$\rho_0$ ——空白溶液中色氨酸色谱峰与对应内标物色谱峰的峰面积比值，经标准曲线求得的对应色氨酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$ ；

100 ——换算系数；

$m$ ——试样的质量，单位为克(g)；

1000——换算系数。

计算结果保留3位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

当称样量为 $0.2\mathrm{g}$ 时，方法的检出限为 $1.5\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $5.0\mathrm{mg} / 100\mathrm{g}$ 。

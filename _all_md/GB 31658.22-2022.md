# 食品安全国家标准

# 动物性食品中 $\beta$ -受体激动剂残留量的测定

# 液相色谱-串联质谱法

# 1 范围

本文件规定了动物性食品中 $\beta$ 受体激动剂残留检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、牛、羊的肌肉、肝脏和肾脏中克仑特罗、莱克多巴胺、沙丁胺醇、西马特罗、齐帕特罗、氯丙那林、特布他林、西布特罗、马布特罗、溴布特罗、班布特罗、克仑丙罗、妥布特罗、利托君、克仑赛罗、马喷特罗、克仑潘特和羟甲基克仑特罗共18种 $\beta$ 受体激动剂单个或混合物残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T 6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的 $\beta$ 受体激动剂，酶解、高氯酸沉淀蛋白后，经乙酸乙酯、叔丁基甲醚萃取，固相萃取柱净化，液相色谱-串联质谱法测定，内标法定量。

# 5 试剂和材料

# 5.1 试剂

以下所用试剂,除特别注明外均为分析纯试剂,水为符合 GB/T 6682 规定的一级水。

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.3 甲酸（HCOOH）：色谱纯。  
5.1.4 高氯酸 $(\mathrm{HClO}_4)$ ： $70\% \sim 72\%$   
5.1.5 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。  
5.1.6 乙酸乙酯 $\mathrm{(CH_3COOC_2H_5)}$ ：色谱纯。  
5.1.7 叔丁基甲醚 $\mathrm{[CH_3OC(CH_3)_3]}$ ：色谱纯。  
5.1.8 $\beta$ -葡萄糖醛酸酶/芳基硫酸酯酶( $\beta$ -Glucuronidase/aryl sulfatase)： $30\mathrm{U} / 60\mathrm{U} / \mathrm{mL}$

# 5.2 溶液配制

5.2.1 $0.2\mathrm{mol} / \mathrm{L}$ 乙酸铵缓冲液：取乙酸铵 $15.4\mathrm{g}$ ，溶解于 $1000~\mathrm{mL}$ 水中，用乙酸调 $\mathrm{pH}$ 至5.2。  
5.2.2 $0.1\mathrm{mol} / \mathrm{L}$ 高氯酸溶液：取高氯酸 $8.7\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。  
5.2.3 $10\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：称取 $40\mathrm{g}$ 氢氧化钠，用适量水溶解冷却后，用水稀释至 $100~\mathrm{mL}$   
5.2.4 $2\%$ 甲酸溶液：取甲酸 $2\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$   
5.2.5 $5\%$ 氨化甲醇溶液：取氨水 $5\mathrm{mL}$ ，用甲醇稀释至 $100~\mathrm{mL}$ 。

5.2.6 $0.1\%$ 甲酸乙腈溶液：取甲酸 $1\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$   
5.2.7 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。  
5.2.8 甲醇-0.1%甲酸溶液 $(10 + 90, V / V)$ ：取甲醇 $10\mathrm{mL}$ 和 $0.1\%$ 甲酸溶液 $90\mathrm{mL}$ ，混匀。

# 5.3 标准品

5.3.1 $\beta$ 受体激动剂：克仑特罗、莱克多巴胺、沙丁胺醇、西马特罗、齐帕特罗、氯丙那林、特布他林、西布特罗、马布特罗、溴布特罗、班布特罗、克仑丙罗、妥布特罗、利托君、克仑赛罗、马喷特罗、克仑潘特和羟甲基克仑特罗，含量均 $\geq 98.0\%$ ，见附录A。  
5.3.2同位素内标：克仑特罗- $\mathrm{D}_9$ 、盐酸莱克多巴胺- $\mathrm{D}_6$ 、沙丁胺醇- $\mathrm{D}_3$ 、西马特罗- $\mathrm{D}_7$ 、齐帕特罗- $\mathrm{D}_7$ 、氯丙那林- $\mathrm{D}_7$ 、特布他林- $\mathrm{D}_9$ 、西布特罗- $\mathrm{D}_9$ 、盐酸马布特罗- $\mathrm{D}_9$ 、盐酸班布特罗- $\mathrm{D}_9$ 、克仑丙罗- $\mathrm{D}_7$ 、盐酸妥布特罗- $\mathrm{D}_9$ 和羟甲基克仑特罗- $\mathrm{D}_6$ ，含量均 $\geqslant 98.0\%$ 。见附录A。

# 5.4 标准溶液的配制

5.4.1 混合标准储备液：精密称取克仑特罗等标准品约 $10\mathrm{mg}$ ，用甲醇溶解并定容至 $10\mathrm{mL}$ 容量瓶中，配制成浓度为 $1.0\mathrm{mg/mL}$ 的标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期12个月。  
5.4.2 混合内标储备液：精密称取克仑特罗- $\mathrm{D}_9$ 等同位素内标约 $10\mathrm{mg}$ ，用甲醇溶解并定容至 $10~\mathrm{mL}$ 容量瓶中，配制成浓度为 $1.0\mathrm{mg / mL}$ 的内标储备液。一 $18^{\circ}C$ 以下保存，有效期12个月。  
5.4.3 混合标准工作液：精密量取 $1\mathrm{mg / mL}$ 的混合标准储备液 $100~\mu \mathrm{L}$ ，于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的标准工作液。 $-18^{\circ}C$ 以下保存，有效期6个月。  
5.4.4 混合内标工作液：精密量取 $1\mathrm{mg / mL}$ 的混合内标储备液 $100~\mu \mathrm{L}$ ，于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的内标工作液。 $-18^{\circ}C$ 以下保存，有效期6个月。

# 5.5 材料

5.5.1 混合型阳离子交换固相萃取柱： $60\mathrm{mg / 3mL}$ ，或相当者。  
5.5.2 微孔滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配有电喷雾离子源(ESI)。  
6.2 分析天平：感量 $0.00001\mathrm{g}$   
6.3 天平：感量 $0.01\mathrm{g}$   
6.4 恒温振荡水浴摇床。  
6.5 涡旋混合器。  
6.6 高速离心机。  
6.7 振荡器。  
6.8 氮吹仪。  
6.9 固相萃取装置。

# 7 试样的制备与保存

# 7.1 试样制备

取适量新鲜或解冻的空白或供试样品，并均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 酶解与提取

称取试料 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管内，加 $0.2\mathrm{mol / L}$ 乙酸铵缓冲溶液 $6\mathrm{mL},\beta$ 葡萄糖醛酸酶/芳基硫酸酯酶 $40~\mu \mathrm{L}$ ，涡旋混匀， $37^{\circ}C$ 避光水浴振荡 $16\mathrm{h}$ ，放置至室温，备用。

# 8.2 萃取、净化与浓缩

取备用液，加 $100\mathrm{ng / mL}$ 内标工作液 $100~\mu \mathrm{L}$ ，涡旋混匀， $8000\mathrm{r / min}$ 离心 $8\mathrm{min}$ ，取上清液，加 $0.1\mathrm{mol / L}$ 高氯酸溶液 $5\mathrm{mL}$ ，涡旋混匀，用高氯酸调 $\mathrm{pH}$ 至 $1.0\pm 0.2,8000\mathrm{r / min}$ 离心 $8\mathrm{min}$ 后，将上清液用 $10\mathrm{mol / LNaOH}$ 溶液调 $\mathrm{pH}$ 至 $10\pm 0.5$ 。加乙酸乙酯 $15\mathrm{mL}$ ，中速振荡 $5\mathrm{min},5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上层有机相。下层水相中再加入叔丁基甲醚 $10\mathrm{mL}$ ，中速振荡 $5\mathrm{min},5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上层有机相，合并， $50^{\circ}C$ 下氮气吹干，用 $2\%$ 甲酸溶液 $5\mathrm{mL}$ 溶解，备用。

取混合型阳离子交换固相萃取柱,依次用甲醇、 $2 \%$ 甲酸溶液各 $3 \mathrm{~mL}$ 活化,取备用液过柱,依次用 $2 \%$ 甲酸溶液、甲醇各 $3 \mathrm{~mL}$ 淋洗,抽干,用 $5 \%$ 氨化甲醇溶液 $3 \mathrm{~mL}$ 洗脱,洗脱液在 $50^{\circ} \mathrm{C}$ 下氮气吹干。

残余物中加入甲醇-0.1%甲酸溶液(10+90,V/V)0.5mL,充分溶解,过0.22μm微孔滤膜,供液相色谱-串联质谱仪测定。

# 8.3 标准曲线的制备

精密量取混合标准工作液、混合内标工作液适量，用甲醇 $0.1\%$ 甲酸溶液 $(10 + 90, V / V)$ 稀释成浓度为 $1\mathrm{ng} / \mathrm{mL}, 2\mathrm{ng} / \mathrm{mL}, 5\mathrm{ng} / \mathrm{mL}, 10\mathrm{ng} / \mathrm{mL}, 20\mathrm{ng} / \mathrm{mL}$ 和 $50\mathrm{ng} / \mathrm{mL}$ 的系列标准工作液，含内标均为 $5\mathrm{ng} / \mathrm{mL}$ ，供液相色谱-串联质谱法测定。以待测药物特征离子色谱峰的峰面积与对应内标物特征离子色谱峰的峰面积比值为纵坐标，相应的标准溶液浓度比为横坐标，绘制标准工作曲线，求回归方程和相关系数。

# 8.4 测定

# 8.4.1 液相色谱参考条件

a）色谱柱：五氟苯基柱 $(50\mathrm{mm}\times 3.0\mathrm{mm},2.6\mu \mathrm{m})$ ，或相当者。  
b) 流动相: A 相为 $0.1\%$ 甲酸溶液, B 相为 $0.1\%$ 甲酸乙腈溶液。流动相梯度为 $0 \mathrm{~min} \sim 0.5 \mathrm{~min}$ 保持 $5\% \mathrm{B}; 0.5 \mathrm{~min} \sim 5 \mathrm{~min}, 5\% \mathrm{B}$ 线性变化到 $60\% \mathrm{B}, 5 \mathrm{~min} \sim 6.5 \mathrm{~min}$ , 保持 $95\% \mathrm{B}, 6.5 \mathrm{~min} \sim 8.5 \mathrm{~min}$ 保持 $5\% \mathrm{B}$ 。  
c) 流速: $0.4 \mathrm{~mL} / \mathrm{min}$ 。  
d） 进样量： $5\mu \mathrm{L}$   
e) 柱温: $30^{\circ} \mathrm{C}$ 。

# 8.4.2 串联质谱参考条件

a）离子源：电喷雾离子源；  
b）扫描方式：正离子扫描；  
c）检测方式：多反应离子监测(MRM)；  
d）电喷雾电压： $5500\mathrm{V}$   
e) 离子源温度: $550^{\circ} \mathrm{C}$ ;   
f）辅助气1：50psi；  
g）辅助气2：60psi；  
h) 气帘气：30 psi；  
i）碰撞气：Medium。

# 9 结果计算和表述

试样中 $\beta$ -受体激动剂残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times V \times A _ {i s} ^ {\prime} \times C _ {s} \times C _ {i s}}{m \times A _ {i s} \times A _ {s} \times C _ {i s} ^ {\prime}} \times \frac {1 0 0 0}{1 0 0 0} \dots \tag {1}
$$

式中：

$X$ ——试样中 $\beta$ -受体激动剂残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

A ——试样溶液中 $\beta$ 受体激动剂的峰面积；

$A_{s}$ ——标准溶液中 $\beta$ 受体激动剂的峰面积；

$A_{is}$ ——试样溶液中 $\beta$ 受体激动剂对应内标的峰面积；

$A_{is}^{\prime}$ ——标准溶液中 $\beta$ 受体激动剂对应内标的峰面积；

$C_{\mathrm{S}}$ ——标准溶液中 $\beta$ -受体激动剂浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$C_{is}$ ——试样溶液中 $\beta$ -受体激动剂内标浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$C_{is}^{\prime}$ ——标准溶液中 $\beta$ 受体激动剂内标浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$V$ ——溶解最终残余物体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——供试试样质量的数值，单位为克(g)。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法对猪、牛、羊的肌肉、肝脏和肾脏的检测限为 $0.2\mu \mathrm{g / kg}$ ，定量限为 $0.5\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法对于猪、牛、羊的肌肉、肝脏和肾脏中 $\beta$ 受体激动剂在 $0.5\mu \mathrm{g / kg}\sim 10~\mu \mathrm{g / kg}$ 添加浓度上的回收率为 $60\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

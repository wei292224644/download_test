# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度应在分析纯以上，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。高效液相色谱试验用水应符合GB/T6682中一级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 溶解性试验

样品微溶于水和无水乙醇。

# A.2.2 红外吸收光谱试验

取样品 $2\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ), 加入溴化钾 $200\mathrm{mg}$ , 混合均匀, 在红外灯下研磨至均匀细粉并进行压片, 在 $4000\mathrm{cm}^{-1} \sim 400\mathrm{cm}^{-1}$ 范围下进行红外扫描。样品溴化钾分散物的红外光谱应与附录B中的标准红外光谱相一致。

# A.3 爱德万甜（以干基计）的测定

# A.3.1 原理

用高效液相色谱测定法，在选定的工作条件下，通过色谱柱使试样溶液中各组分分离，用紫外检测器检测，用内标法定量，计算试样中爱德万甜的含量。

A.3.2.7 水和乙腈的混合液（7：3，体积比）：准确量取 $300~\mathrm{mL}$ 乙腈(A.3.2.3)，加入容量瓶中，用水稀释，精确定容至 $1000~\mathrm{mL}$ 。

# A.3.5 分析步骤

# A.3.5.1 溶液制备

# A.3.5.1.2 标准贮备溶液

# A.3.5.1.4 试样溶液

# A.3.6 测定

# A.3.6.1 标准曲线

分别将标准溶液(A.3.5.1.3)注入色谱仪中，记录所生成色谱图中主要色谱峰的保留时间和峰面积

(爱德万甜典型液相色谱图见附录 C)。对于每种标准溶液, 计算爱德万甜峰面积与内标物苯甲酸峰面积的比率。以标准溶液浓度为横坐标, 以生成的峰面积比率为纵坐标, 绘制标准曲线。

# A.3.6.2 试样溶液的测定

将试样溶液(A.3.5.1.4)注入色谱仪中，记录所生成色谱图中主要色谱峰的保留时间和峰面积。计算爱德万甜峰值的峰面积和内标物苯甲酸的峰面积响应的比率。利用标准曲线，测定试样溶液中爱德万甜质量浓度 $(C_{\mathrm{A}})$ ，单位为毫克每毫升 $(\mathrm{mg / mL})$ 。

# A.3.7 计算

试样中爱德万甜(以干基计) $\omega_{\mathrm{A}}$ 百分比按式(A.1)计算：

$$
w _ {\mathrm {A}} = \frac {C _ {\mathrm {A}} \times V _ {\mathrm {U}}}{m _ {\mathrm {U}}} \times 1 0 0 \tag {A.1}
$$

式中：

$C_{\mathrm{A}}$ ——由标准曲线测定的试样溶液爱德万甜的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V_{\mathrm{U}}$ ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$

$m_{\mathrm{U}}$ ——试样的质量，单位为毫克 $(\mathrm{mg})$ ；

100——百分比。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得爱德万甜的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

# A.4.5.1 溶液制备

# A.4.6 测定

在参考色谱条件下，分别注入系列标准溶液（A.4.5.1.2)和试样溶液(A.4.5.1.3)，记录色谱图，测定所生成色谱图中的峰面积。按外标法用系列标准溶液作校正表。爱德万甜酸和其他相关物质典型液相色谱图见附录D。

# A.4.7 计算

爱德万甜酸(以干基计)的百分比 $w_{\mathrm{AA}}$ 按式(A.2)计算：

$$
w _ {\mathrm {A A}} = \frac {A _ {\mathrm {U}} \times C _ {\mathrm {S}}}{A _ {\mathrm {S}} \times C _ {\mathrm {U}}} \times 1 0 0 \tag {A.2}
$$

式中：

$A_{\mathrm{U}}$ ——试样溶液色谱图中爱德万甜酸的峰面积；

$C_{\mathrm{S}}$ ——爱德万甜酸标准溶液的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$A_{\mathrm{S}}$ ——标准溶液色谱图中爱德万甜酸的峰面积；

$C_{\mathrm{U}}$ ——试样溶液的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$

100 ——百分比。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得爱德万甜酸的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

其他相关物质(以干基计)的总百分比 $w_{\mathrm{Q}}$ 按式(A.3)计算：

$$
w _ {\mathrm {Q}} = \frac {A _ {\mathrm {T}} \times C _ {\mathrm {S}}}{A _ {\mathrm {S}} \times C _ {\mathrm {U}}} \times 1 0 0 \tag {A.3}
$$

式中：

$A_{\mathrm{T}}$ ——试样溶液色谱图中爱德万甜和爱德万甜酸之外其他成分的峰面积之和；

$C_{\mathrm{S}}$ ——标准溶液的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$A_{\mathrm{S}}$ ——标准溶液色谱图中爱德万甜酸的峰面积；

$C_{\mathrm{U}}$ ——试样溶液的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

100 ——百分比。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得其他相关物质的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

# A.5.5 结果计算

试样中待测组分（甲醇或乙酸乙酯)的含量 $X_{i}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.4)计算。

$$
X _ {i} = \frac {V _ {\mathrm {T}} \times C _ {\mathrm {S}}}{m _ {\mathrm {T}}} \times \frac {A _ {\mathrm {T}}}{A _ {\mathrm {S}}} \quad \dots \tag {A.4}
$$

式中：

$V_{\mathrm{T}}$ ——试样溶液的体积，单位毫升 $(\mathrm{mL})$

$C_{\mathrm{S}}$ ——标准溶液中甲醇或乙酸乙酯的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$m_{\mathrm{T}}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$A_{\mathrm{T}}$ ——试样溶液中甲醇或乙酸乙酯的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中甲醇或乙酸乙酯的峰面积。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

2016-08-31 发布

# 前言

本标准代替GB/T5009.141—2003《食品中诱惑红的测定》。

本标准与GB/T5009.141—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中诱惑红的测定”；  
——增加了试剂的级别和分子式；  
——增加了标准品。

# 食品安全国家标准

# 食品中诱惑红的测定

# 1 范围

本标准规定了汽水、硬糖、糕点、冰淇淋中诱惑红的测定方法。

本标准适用于汽水、硬糖、糕点、冰淇淋中诱惑红的测定。

# 2 原理

诱惑红在酸性条件下被聚酰胺粉吸附，而在碱性条件下解吸附，再用纸色谱法进行分离后，与标准比较定性、定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ 。  
3.1.2 石油醚：沸程 $30^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$   
3.1.3 硫酸 $\mathrm{(H_2SO_4)}$ ：优级纯。  
3.1.4 乙醇 $\mathrm{(CH_3CH_2OH)}$ 。  
3.1.5 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ：含量 $20\% \sim 25\%$   
3.1.6 柠檬酸 $\mathrm{(C_6H_8O_7\bullet H_2O)}$ 。  
3.1.7 钨酸钠 $(\mathrm{Na}_2\mathrm{WO}_4\cdot 2\mathrm{H}_2\mathrm{O})$ 。  
3.1.8 丁酮 $\mathrm{(C_4H_8O)}$ 。  
3.1.9 柠檬酸钠 $(\mathrm{C_6H_5Na_3O_7})$ 。  
3.1.10 正丁醇 $(\mathrm{C_4H_{10}O})$ 。  
3.1.11 海砂。  
3.1.12 甲酸（HCOOH）。

# 3.2 试剂配制

3.2.1 硫酸溶液（ $10\%$ ，体积分数）：将 $1 \mathrm{~mL}$ 硫酸缓慢加入至 $8 \mathrm{~mL}$ 水中，混匀，冷却，用水定容至 $10 \mathrm{~mL}$ ，混匀。  
3.2.2 乙醇-氨溶液：取 $2\mathrm{mL}$ 的氨水，加 $70\%$ （体积分数）乙醇至 $100~\mathrm{mL}$ 。  
3.2.3 乙醇溶液 $(50\%)$ ，体积分数）：量取 $50~\mathrm{mL}$ 无水乙醇与 $50~\mathrm{mL}$ 水混匀。  
3.2.4 柠檬酸溶液 $(200\mathrm{g / L})$ ：称取 $20\mathrm{g}$ 柠檬酸，加水至 $100~\mathrm{mL}$ ，溶解混匀。  
3.2.5 钨酸钠溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 钨酸钠，加水至 $100~\mathrm{mL}$ ，溶解混匀。  
3.2.6 氨水溶液（ $1\%$ ，体积分数）：量取 $1 \mathrm{~mL}$ 氨水，加水至 $100 \mathrm{~mL}$ ，混匀。

3.2.7 柠檬酸钠溶液 $(2.5\%)$ ，体积分数）：称取 $2.5\mathrm{g}$ 柠檬酸，加水至 $100~\mathrm{mL}$ ，溶解混匀。  
3.2.8 甲醇-甲酸溶液(6：4，体积分数)：量取甲醇 $60~\mathrm{mL}$ ，甲酸 $40~\mathrm{mL}$ ，混匀。  
3.2.9 展开剂1：丁酮 $+$ 丙醇 $+$ 水 $+$ 氨水 $(7 + 3 + 3 + 0.5)$ 。  
3.2.10 展开剂2：正丁醇 $+$ 无水乙醇 $+1\%$ 氨水溶液 $(6 + 2 + 3)$   
3.2.11 展开剂 $3:2.5\%$ 柠檬酸钠 + 氨水 + 乙醇 $(8+1+2)$ 。

# 3.3 标准品

诱惑红(CAS:25956-17-6)。

# 3.4 标准溶液配制

3.4.1 诱惑红标准贮备液配制：准确称取诱惑红 $0.025\mathrm{g}$ （精确到 $0.0001\mathrm{g}$ ，按诱惑红实际纯度折算为纯品后的质量），用水溶解并定容至 $25\mathrm{mL}$ ，诱惑红浓度为 $1.0\mathrm{mg/mL}$ 。  
3.4.2 诱惑红标准使用液 $(0.1\mathrm{mg / mL})$ ：吸取诱惑红的标准贮备液 $5.0~\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，加水稀释到 $50~\mathrm{mL}$ 。

# 4 仪器和设备

4.1 可见分光光度计。  
4.2 电子天平：感量为 $0.001\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
4.3 微量注射器： $10~\mu \mathrm{L},50~\mu \mathrm{L}$   
4.4 展开槽。  
4.5 电吹风机。  
4.6 离心机。  
4.7 恒温水浴锅。

# 5 分析步骤

# 5.1 试样制备

5.1.1 汽水：将样品加热去二氧化碳后，称取 $10\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ）样品于烧杯中，然后用 $20\%$ 柠檬酸调 $\mathrm{pH}$ 呈酸性，加入 $0.5\mathrm{g} \sim 1.0\mathrm{g}$ 聚酰胺粉吸附色素，将吸附色素的聚酰胺粉全部转到漏斗中过滤，用 $\mathrm{pH}4$ 的酸性热水洗涤多次（约 $200\mathrm{mL}$ ），以洗去糖等物质。若有天然色素，用甲醇-甲酸溶液洗涤1次～3次，每次 $20\mathrm{mL}$ ，至洗液无色为止。再用 $70^{\circ}\mathrm{C}$ 的水多次洗涤至流出液中性。洗涤过程应充分搅拌然后用乙醇-氨水溶液分次解吸色素，收集全部解吸液，于水浴上驱除氨，蒸发至 $2\mathrm{mL}$ 左右，转入 $5\mathrm{mL}$ 的容量瓶中，用 $50\%$ 的乙醇分次洗涤蒸发皿，洗涤液并入 $5\mathrm{mL}$ 的容量瓶中，用 $50\%$ 的乙醇定容至刻度。此液留作纸色谱用。  
5.1.2 硬糖：称取 $10\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )的已粉碎的样品，加 $30~\mathrm{mL}$ 的水，温热溶解，若样品溶液的 $\mathrm{pH}$ 较高，用柠檬酸溶液(3.2.4)调至 $\mathrm{pH}4$ 左右。加入 $0.5\mathrm{g}\sim 1.0\mathrm{g}$ 聚酰胺粉吸附色素，将吸附色素的聚酰胺粉全部转到漏斗中过滤，用 $\mathrm{pH}4$ 的酸性热水洗涤多次（约 $200~\mathrm{mL}$ ，以洗去糖等物质。若有天然色素，用甲醇-甲酸溶液洗涤1次～3次，每次 $20~\mathrm{mL}$ ，至洗液无色为止。再用 $70^{\circ}\mathrm{C}$ 的水多次洗涤至流出液中性。洗涤过程应充分搅拌然后用乙醇-氨水溶液分次解吸色素，收集全部解吸液，于水浴上驱除氨，蒸发至 $2\mathrm{mL}$ 左右，转入 $5\mathrm{mL}$ 的容量瓶中，用 $50\%$ 的乙醇分次洗涤蒸发皿，洗涤液并入 $5\mathrm{mL}$ 的容量瓶中，用 $50\%$ 的乙醇定容至刻度。此液留作纸色谱用。  
5.1.3 糕点：称取 $10\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )已粉碎的样品，加入 $30~\mathrm{mL}$ 石油醚提取脂肪，共提三次，然后用

电吹风吹干, 倒入漏斗中, 用乙醇-氨解吸色素, 解吸液于水浴上蒸发至 $20 \mathrm{~mL}$ , 加入 $1 \mathrm{~mL}$ 的钨酸钠溶液沉淀蛋白, 真空抽滤, 用乙醇-氨解吸滤纸上的诱惑红, 然后将滤液于水浴上挥去氨, 调 $\mathrm{pH}$ 呈酸性, 加入 $0.5 \mathrm{~g} \sim 1.0 \mathrm{~g}$ 聚酰胺粉吸附色素, 将吸附色素的聚酰胺粉全部转到漏斗中过滤, 用 $\mathrm{pH} 4$ 的酸性热水洗涤多次 (约 $200 \mathrm{~mL}$ ), 以洗去糖等物质。若有天然色素, 用甲醇-甲酸溶液洗涤 1 次 $\sim 3$ 次, 每次 $20 \mathrm{~mL}$ , 至洗液无色为止。再用 $70^{\circ} \mathrm{C}$ 的水多次洗涤至流出液中性。洗涤过程应充分搅拌然后用乙醇-氨水溶液分次解吸色素, 收集全部解吸液, 于水浴上驱除氨, 蒸发至 $2 \mathrm{~mL}$ 左右, 转入 $5 \mathrm{~mL}$ 的容量瓶中, 用 $50 \%$ 的乙醇分次洗涤蒸发皿, 洗涤液并入 $5 \mathrm{~mL}$ 的容量瓶中, 用 $50 \%$ 的乙醇定容至刻度。此液留作纸色谱用。

5.1.4 冰淇淋：称取 $10\mathrm{g}$ (精确到 $0.001\mathrm{g}$ ) 已均匀的试样于烧杯中，加入 $20\mathrm{g}$ 海沙 $15\mathrm{mL}$ 石油醚提取脂肪，提取2次，倾去石油醚，然后在 $50^{\circ}\mathrm{C}$ 的水浴上挥去石油醚，再加入乙醇-氨解吸液解吸诱惑红，解吸液倒入 $100\mathrm{mL}$ 的蒸发皿中，直至解吸液无色。将解吸液在水浴上挥去乙醇，使体积约为 $20\mathrm{mL}$ 时，加入 $1\mathrm{mL}$ 硫酸 $(1+10)$ ， $1\mathrm{mL}$ 钨酸钠溶液沉淀蛋白，放置 $2\mathrm{min}$ ，然后用乙醇-氨调至 $\mathrm{pH}$ 呈碱性，将溶液转入离心管中， $5000\mathrm{r/min}$ 离心 $15\mathrm{min}$ ，倾出上清液，于水浴挥去乙醇，然后用柠檬酸溶液调 $\mathrm{pH}$ 呈酸性，加入 $0.5\mathrm{g} \sim 1.0\mathrm{g}$ 聚酰胺粉吸附色素，将吸附色素的聚酰胺粉全部转到漏斗中过滤，用 $\mathrm{pH}4$ 的酸性热水洗涤多次（约 $200\mathrm{mL}$ ），以洗去糖等物质。若有天然色素，用甲醇-甲酸溶液洗涤1次～3次，每次 $20\mathrm{mL}$ ，至洗液无色为止。再用 $70^{\circ}\mathrm{C}$ 的水多次洗涤至流出液中性。洗涤过程应充分搅拌然后用乙醇-氨水溶液分次解吸色素，收集全部解吸液，于水浴上驱除氨，蒸发至 $2\mathrm{mL}$ 左右，转入 $5\mathrm{mL}$ 的容量瓶中，用 $50\%$ 的乙醇分次洗涤蒸发皿，洗涤液并入 $5\mathrm{mL}$ 的容量瓶中，用 $50\%$ 的乙醇定容至刻度。此液留作纸色谱用。

# 5.2 定性

取层析纸，在距底边 $2\mathrm{cm}$ 起始线上分别点 $3\mu \mathrm{L}\sim 10\mu \mathrm{L}$ 的样品处理液、 $1\mu \mathrm{L}$ 诱惑红标准使用液，分别挂于盛有展开剂1、展开剂2、展开剂3的展开槽中，用上行法展开，待溶剂前沿展至 $15~\mathrm{cm}$ 处，将滤纸取出空气中凉干，与标准斑比较定性。

# 5.3 定量

# 5.3.1 标准曲线的制备

吸取 $0.0\mathrm{mL}, 0.2\mathrm{mL}, 0.4\mathrm{mL}, 0.6\mathrm{mL}, 0.8\mathrm{mL}, 1.0\mathrm{mL}$ 诱惑红标准使用液，分别置于 $10\mathrm{mL}$ 比色管中，各加水稀释到刻度，浓度分别为 $0\mu \mathrm{g} / \mathrm{mL}, 2\mu \mathrm{g} / \mathrm{mL}, 4\mu \mathrm{g} / \mathrm{mL}, 6\mu \mathrm{g} / \mathrm{mL}, 8\mu \mathrm{g} / \mathrm{mL}, 10\mu \mathrm{g} / \mathrm{mL}$ 。用 $1\mathrm{mL}$ 比色杯，以零管调零点，于波长 $500\mathrm{nm}$ 处，测定吸光度，绘制标准曲线。

# 5.3.2 样品的测定

取色谱用纸，在距离底边 $2\mathrm{cm}$ 的起始线上，点 $0.20~\mathrm{mL}$ 样品处理液，从左到右点成条状。纸的右边点诱惑红的标准溶液 $1\mu \mathrm{L}$ ，依法展开，取出晾干。将样品的色带剪下，用少量热水洗涤数次，洗液移入 $10~\mathrm{mL}$ 的比色管中，加水稀释至刻度，混匀后，与标准管同时在 $500~\mathrm{nm}$ 处，测定吸光度。

# 6 分析结果的表述

试样中诱惑红含量按式(1)计算：

$$
X = \frac {A \times 1 0 0 0}{m \times \frac {V _ {2}}{V _ {1}} \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中诱惑红的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

A ——测定用样品中诱惑红的含量，单位为毫克 $(\mathrm{mg})$ ；

$V_{1}$ ——样品解吸后总体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——样品纸层析用体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

取样量为 $10\mathrm{g}$ 时，检出限为 $25\mathrm{mg / kg}$

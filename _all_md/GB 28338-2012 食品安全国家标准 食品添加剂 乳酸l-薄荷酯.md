# 食品安全国家标准

# 食品添加剂 乳酸 $l$ -薄荷酯

# 1 范围

本标准适用于由乳酸和 $l$ -薄荷脑为原料制得的食品添加剂乳酸 $l$ -薄荷酯。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

5-甲基-2-(1-甲基乙基)环己醇α-羟基丙酸酯

# 2.2 分子式

$$
\mathrm {C} _ {1 3} \mathrm {H} _ {2 4} \mathrm {O} _ {3}
$$

# 2.3 结构式

![](images/92f4abc6556e197095c0ba07901f9401491093ca5cf80f45c1d285e8109a55d3.jpg)

# 2.4 相对分子质量

228.33(按2007年国际相对原子质量)

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $15\mathrm{m}$ ，内径 $0.53\mathrm{mm}$ 。  
B.2.2 固定相：二甲基聚硅氧烷。  
B.2.3 膜厚： $0.5\mu \mathrm{m}$   
B.2.4 色谱炉温度： $160^{\circ}C$ 恒温。  
B.2.5 进样口温度： $250^{\circ}C$   
B.2.6 检测器温度： $250^{\circ}C$ 。  
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 载气流速：约 $20\mathrm{mL / min}$   
B.2.10 进样量： $0.2\mu \mathrm{L}$   
B.2.11 分流比：1/75。

# 前言

本标准代替GB4853—2008《食品级白油》。

本标准与GB4853—2008相比，主要变化如下：

——修改了低、中黏度的初馏点温度的指标要求；  
——修改了1号食品添加剂白油的 $5\%$ 蒸馏点碳数、 $5\%$ 蒸馏点温度的指标要求；  
——修改了平均相对分子质量的检验方法，并删除了并列方法中的SH/T0398检验方法；  
——修改了铅含量的检验方法。

# 附录A

# 铅的测定

# A.1 警示

本标准的检验方法中使用的试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 试剂和材料

A.2.1 盐酸。  
A.2.2 冰乙酸溶液：量取 $12\mathrm{mL}$ 的冰乙酸用水稀释至 $100~\mathrm{mL}$   
A.2.3 其他试剂同GB5009.12。

# A.3 仪器和设备

A.3.1 高温炉：温度能控制在 $450^{\circ}\mathrm{C} \sim 550^{\circ}\mathrm{C}$ 。  
A.3.2 电热恒温水浴。

# A.4 测定方法

称取约 $2\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，放于瓷坩埚中，置于电炉上在通风橱中缓慢加热至炭化，然后转入高温炉中，在 $450^{\circ}\mathrm{C} \sim 550^{\circ}\mathrm{C}$ 使其完全灰化，灰化时间应不少于 $2\mathrm{h}$ ，冷却至室温，加入 $2\mathrm{mL}$ 盐酸并在电热恒温水浴上蒸干，加入 $2\mathrm{mL}$ 冰乙酸溶液溶解残渣，全部转入 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀，此溶液作为试样溶液。

按照GB5009.12测定试样溶液中的铅。若测定结果超出曲线范围，将试样溶液稀释后进行测定并计算。

注：加热过程中应避免试样因体积膨胀、燃烧而溢出坩埚外。

# 水产品中硝呋索尔代谢物残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了水产品中硝呋索尔代谢物3，5-二硝基水杨酸肼(3，5-Dinitrosalicylicacidhydrazide，DNSAH)残留量的制样和液相色谱-串联质谱测定方法。

本文件适用于鱼、虾、蟹及鳖等水产品可食组织中3,5-二硝基水杨酸肼(DNSAH)残留量的检测，其他水产品的检测可参照执行。

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的硝呋索尔代谢物DNSAH在酸性条件下水解,用2-硝基苯甲醛衍生化,乙酸乙酯液-液萃取、浓缩及超滤净化,液相色谱-串联质谱仪测定,内标法定量。

# 5.2 溶液配制

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.2 醋酸铵 $\mathrm{(CH_3COONH_2)}$ ：色谱纯  
5.1.3 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ ：色谱纯。  
5.1.4 二甲亚砜 $(\mathrm{C_2H_6OS})$ ：色谱纯  
5.1.5 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.6 2-硝基苯甲醛 $\mathrm{(C_7H_5NO_3)}$ ：色谱纯。  
5.1.7 磷酸氢二钾 $(\mathrm{K_2HPO_4})$   
5.1.8 盐酸（HCl）。  
5.1.9 氨水 $(\mathrm{NH_4OH})$ ： $25\% \sim 28\%$   
5.2.1 $0.5\mathrm{mol} / \mathrm{L}$ 盐酸溶液：准确量取 $42~\mathrm{mL}$ 盐酸，用水稀释并定容至 $1000~\mathrm{mL}$   
5.2.2 $2\mathrm{mmol / L}$ 醋酸铵溶液：称取 $\mathrm{0.15g}$ 醋酸铵，用水溶解并稀释至 $1000~\mathrm{mL}$   
5.2.3 $0.05\mathrm{mol} / \mathrm{L}2$ -硝基苯甲醛溶液：称取 $0.076\mathrm{g}2$ -硝基苯甲醛，用二甲亚砜溶解并稀释至 $10~\mathrm{mL}$ ，现用现配。

5.2.4 1.0 mol/L 磷酸氢二钾溶液：称取 $87.1\mathrm{g}$ 磷酸氢二钾，用水溶解并定容至 $500~\mathrm{mL}$ 。  
5.2.5 $10\%$ 乙腈水溶液：准确移取 $10~\mathrm{mL}$ 乙腈，用水稀释并定容至 $100~\mathrm{mL}$ 。  
5.2.6 氨水乙腈溶液：准确移取 $35~\mu \mathrm{L}$ 氨水，用 $10\%$ 乙腈水溶液稀释并定容至 $100~\mathrm{mL}$ 。

# 5.3 标准品

5.3.1 3,5-二硝基水杨酸肼(3,5-Dinitrosalicylic acid hydrazide,DNSAH, $\mathrm{C}_7\mathrm{H}_6\mathrm{O}_6\mathrm{N}_4$ ，CAS号：955-07-7)，纯度 $\geqslant 99\%$ ，或相当者。  
5.3.2 3,5-二硝基水杨酸肼 $^{-15}$ N $_{2}$ (3,5-Dinitrosalicylic acid hydrazide $^{-15}$ N $_{2}$ ,DNSAH $^{-15}$ N $_{2}$ ,C $_{7}$ H $_{6}$ O $_{6}$ N $_{2}$ $^{15}$ N $_{2}$ , CAS号:1346598-09-1):纯度≥95%,或相当者。

# 5.4 标准溶液制备

5.4.1 标准储备液 $(1.0\mathrm{mg / mL})$ ：取DNSAH标准品约 $10\mathrm{mg}$ ，精密称定，先用 $1\mathrm{mL}$ 二甲亚砜溶解，再用甲醇稀释并定容至 $10~\mathrm{mL}$ 棕色容量瓶中，配制成浓度为 $1.0\mathrm{mg / mL}$ 标准储备液， $-18^{\circ}C$ 避光保存，有效期为6个月。  
5.4.2 标准中间液 $(1.0\mu \mathrm{g / mL})$ ：准确移取 $1.0\mathrm{mg / mL}$ 标准储备液 $0.1\mathrm{mL}$ 于 $100~\mathrm{mL}$ 棕色容量瓶中，用甲醇稀释并定容，配制成浓度为 $1.0\mu \mathrm{g / mL}$ 的标准中间液，一 $18^{\circ}C$ 避光保存，有效期3个月。  
5.4.3 标准工作液 $1(100\mathrm{ng / mL})$ ：准确移取适量 $1.0~\mu \mathrm{g / mL}$ 标准中间液，用甲醇稀释成 $100\mathrm{ng / mL}$ 标准工作液1，现用现配。  
5.4.4 标准工作液 $2(10\mathrm{ng / mL})$ ：准确移取适量 $100\mathrm{ng / mL}$ 标准工作液1，用甲醇稀释成 $10\mathrm{ng / mL}$ 标准工作液2，现用现配。  
5.4.5 内标储备液 $(0.1\mathrm{mg / mL})$ ：取 $\mathrm{DNSAH - }^{15}\mathrm{N}_2$ 内标标准品，精密称定，先用 $1\mathrm{mL}$ 二甲亚砜溶解，再用甲醇稀释并定容至 $10~\mathrm{mL}$ 棕色容量瓶中，配制成浓度为 $0.1\mathrm{mg / mL}$ 内标储备液， $-18^{\circ}C$ 避光保存，有效期为6个月。  
5.4.6 内标工作液 $(0.1\mu \mathrm{g} / \mathrm{mL})$ ：准确移取适量DNSAH- ${}^{15}\mathrm{N}_2$ 内标储备液，用甲醇逐级稀释成浓度为 $0.1\mu \mathrm{g} / \mathrm{mL}$ 的内标工作液，一 $18^{\circ}C$ 避光保存，有效期3个月。

# 5.5 材料

5.5.1 超滤离心管： $10\mathrm{ku}$ 。  
5.5.2 聚丙烯离心管： $50~\mathrm{mL}$ 、 $20~\mathrm{mL}$ 、1.5 mL。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$ 。  
6.3 离心机：转速不低于 $6000\mathrm{r / min}$ 。  
6.4 微量高速离心机：14000r/min。  
6.5 氮吹仪。  
6.6 涡旋混合器。  
6.7 pH计。

# 7 试样的制备与保存

# 7.1 试样的制备

按GB/T30891—2014附录B的要求制样。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 水解和衍生化

称取试料 $(2\pm 0.05)\mathrm{g}$ 于 $50~\mathrm{mL}$ 离心管中，加入内标工作液 $0.1\mathrm{mL}$ ，涡旋混合 $1\mathrm{min}$ ，再加入盐酸溶液 $5\mathrm{mL}$ 和2-硝基苯甲醛溶液 $0.15\mathrm{mL}$ ，涡旋混合 $1\mathrm{min}$ 后，置于恒温振荡器中 $37^{\circ}C$ 避光振荡 $16\mathrm{h}$ 。

# 8.2 提取净化

取出离心管冷却至室温，加入磷酸氢二钾溶液，调节 $\mathrm{pH}$ 至 $7.0\sim 7.5$ ，加入乙酸乙酯 $8\mathrm{mL}$ ，涡旋振荡 $10\mathrm{min},6000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上层清液转移至 $10~\mathrm{mL}$ 离心管中，于 $40^{\circ}C$ 下氮气吹干。准确加入氨水乙腈溶液 $0.5\mathrm{mL}$ ，充分涡旋振荡溶解残留物， $14000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，将清液转移至超滤离心管中， $14000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取滤液，供液相色谱-串联质谱仪分析。实验过程需要注意避免阳光直射，样品前处理完后应及时上机检测。

# 8.3 基质标准曲线的制备

取空白试料于7个 $50~\mathrm{mL}$ 离心管中，分别准确移取标准工作液 $2(10\mathrm{ng / mL})0.025\mathrm{mL},0.05\mathrm{mL}$ 、 $0.1\mathrm{mL},0.2\mathrm{mL}$ 和标准工作液 $1(100\mathrm{ng / mL})0.05\mathrm{mL},0.1\mathrm{mL}$ 和 $0.2\mathrm{mL}$ ，按8.1和8.2步骤操作，使标准溶液最终浓度分别为 $0.5\mathrm{ng / mL},1.0\mathrm{ng / mL},2.0\mathrm{ng / mL},4.0\mathrm{ng / mL},10.0\mathrm{ng / mL},20.0\mathrm{ng / mL}$ 和 $40.0\mathrm{ng / mL}$ ，内标溶液的浓度为 $20.0\mathrm{ng / mL}$ ，并按8.1测定。

# 8.5 测定法

# 8.5.1 定性测定

在相同测试条件下，试料溶液中DNSAH与其内标 $\mathrm{DNSAH - }^{15}\mathrm{N}_2$ 的保留时间之比与标准溶液中DNSAH与其内标 $\mathrm{DNSAH - }^{15}\mathrm{N}_2$ 的保留时间之比偏差在 $1\%$ 以内；且检测到的相对离子丰度应与浓度相当的校正标准溶液相对离子丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.5.2 定量测定

取试料溶液和相应的标准工作液，作单点或多点校准，按内标法以峰面积定量，标准工作液及试料溶液中目标化合物的响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，硝呋索尔代谢物DNSAH标准溶液特征离子质量色谱图见附录A。

# 8.6 空白试验

取空白试样，除不加药物外，采用完全相同的操作步骤进行平行操作。

# 9 结果计算和表述

试样中DNSAH的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A _ {i} \times A _ {i s} ^ {\prime} \times C _ {s} \times C _ {i s} \times V}{A _ {i s} \times A _ {s} \times C _ {i s} ^ {\prime} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中DNSAH残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A_{i}$ ——试料溶液中DNSAH的峰面积；

$A_{is}^{\prime}$ ——标准溶液中 $\mathrm{DNSAH^{-15}N_2}$ 的峰面积；

$C_{s}$ ——标准溶液中DNSAH浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$C_{is}$ ——试料溶液中 $\mathrm{DNSAH}^{-15}\mathrm{N}_2$ 浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{is}$ ——试料溶液中 $\mathrm{DNSAH}^{-15}\mathrm{N}_2$ 的峰面积；

$A_{s}$ ——标准溶液中DNSAH的峰面积；

$C_{is}^{\prime}$ ——标准溶液中 $\mathrm{DNSAH^{-15}N_2}$ 浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$m$ ——供试试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法的检出限为 $0.25\mu \mathrm{g / kg}$ ，定量限为 $0.5\mu \mathrm{g / kg}$

# 10.2 正确度

本方法在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 10~\mu \mathrm{g} / \mathrm{kg}$ 添加浓度的回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

# 1 范围

本文件规定了动物性可食组织中头孢噻呋残留量检测的制样和高效液相色谱测定方法。

本文件适用于猪、牛肌肉、脂肪、肝脏和肾脏中头孢噻呋残留量的测定。

# 3 术语和定义

# 4 原理

试料中残留的头孢噻呋，经二硫赤藓醇（DTE）作用，使头孢噻呋及去呋喃甲酰头孢噻呋（DFC）有关代谢物从蛋白或含硫化合物中分离，与碘乙酰胺反应，生成稳定的乙酰胺衍生物（DCA），经提取、纯化、净化，高效液相色谱测定，外标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

5.1.2 甲醇 $\mathrm{(CH_3OH)}$ 。  
5.1.3 二硫赤藓醇 $\mathrm{(C_4H_{10}O_2S_2}$ ，DTE）。  
5.1.4 碘乙酰胺 $\mathrm{(C_2H_4INO)}$   
5.1.5 无水氯化钙 $\mathrm{(CaCl_2)}$   
5.1.6 冰醋酸 $\mathrm{(CH_3COOH)}$   
5.1.7 磷酸 $(\mathrm{H}_3\mathrm{PO}_4):85\%$   
5.1.8 氯化钾(KCl)。  
5.1.9 氢氧化钾 $(\mathrm{NaOH})$ 。  
5.1.10 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
5.1.11 四硼酸钠 $(\mathrm{Na}_2\mathrm{B}_4\mathrm{O}_7\cdot 10\mathrm{H}_2\mathrm{O})$   
5.1.12 氯化钠 $(\mathrm{NaCl})$   
5.1.13 氢氧化钠（NaOH）。  
5.1.14 三氟乙酸 $\mathrm{(CF_3COOH,TFA)}$

# 5.2 标准品

盐酸头孢噻呋(Ceftiofur Hydrochloride, $\mathrm{C_{19}H_{16}N_5O_7S_3\cdot HCl}$ )，以 $\mathrm{C_{19}H_{17}N_5O_7S_3}$ 计，含量 $\geq 87.5\%$ 或相当者。

# 5.3 溶液配制

5.3.1硼酸缓冲液 $(0.05\mathrm{mol} / \mathrm{L})$ ：取四硼酸钠 $19\mathrm{g}$ 、氯化钾 $3.7\mathrm{g}$ ，加水使溶解并稀释至 $1000~\mathrm{mL}$ 。  
5.3.2 磷酸盐缓冲液 $(0.025\mathrm{mol} / \mathrm{L})$ ：取磷酸二氢钾 $3.4\mathrm{g}$ ，加水溶解并稀释至 $1000~\mathrm{mL}$ ，用 $45\%$ 氢氧化钾溶液调 $\mathrm{pH}$ 至7.0。  
5.3.3 提取液 $(0.4\%$ DTE)：取DTE $1\mathrm{g}$ ，加硼酸缓冲液适量使溶解并稀释至 $250~\mathrm{mL}$ ，现用现配。  
5.3.4 碘乙酰胺溶液 $(14\%)$ ：取碘乙酰胺 $7\mathrm{g}$ 溶于 $50~\mathrm{mL}$ 磷酸盐缓冲液中，现用现配。  
5.3.5 氢氧化钠溶液 $(0.01\mathrm{mol} / \mathrm{L})$ ：取氢氧化钠 $0.4\mathrm{g}$ ，加水使溶解并稀释至 $1000~\mathrm{mL}$ 。  
5.3.6 氯化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：取氯化钠 $5.9\mathrm{g}$ ，加水溶解并稀释至 $1000~\mathrm{mL}$ 。  
5.3.7 氯化钙溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：取无水氯化钙 $11.1\mathrm{g}$ ，加水溶解并稀释至 $1000~\mathrm{mL}$ 。  
5.3.8 磷酸溶液 $(25\%)$ ：取磷酸 $25~\mathrm{mL}$ ，加水溶解并稀释至 $100~\mathrm{mL}$ 。  
5.3.9 醋酸溶液 $(5\%)$ ：取冰醋酸 $5\mathrm{mL}$ ，加水溶解并稀释至 $100~\mathrm{mL}$ 。  
5.3.10 $\mathrm{C_{18}}$ 洗脱液：乙腈-水（ $15:85)$   
5.3.11SAX预洗液：甲醇氯化钠溶液（25：75）。  
5.3.12 SAX洗脱液：乙腈-醋酸溶液（5：95）。  
5.3.13 SCX预洗液：甲醇-氯化钙溶液（25：75）。  
5.3.14 SCX洗脱液：乙腈-氯化钠溶液(5：95)。  
5.3.15 流动相：

A(0.1%TFA水溶液)：取 $1\mathrm{mL}$ 三氟乙酸，加水至 $1000~\mathrm{mL}$

$\mathrm{B}(0.1\% \mathrm{TFA}$ 乙腈溶液）：取 $1\mathrm{mL}$ 三氟乙酸，加乙腈至 $1000~\mathrm{mL}$

# 5.4 标准溶液制备

5.4.1 标准储备液 $(100\mu \mathrm{g / mL})$ ：取盐酸头孢噻呋标准品约 $11\mathrm{mg}$ ，精密称定，加甲醇使溶解并稀释定容至 $100~\mathrm{mL}$ 容量瓶，摇匀，即得。 $-18^{\circ}C$ 保存，有效期6个月。  
5.4.2 标准工作液 $(10\mu \mathrm{g / mL})$ ：精密量取标准储备液 $5\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶，加磷酸盐缓冲液至刻度，摇匀，即得。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1个月。

# 5.5 材料

5.5.1 $\mathrm{C_{18}}$ 固相萃取柱： $1\mathrm{g} / 6\mathrm{mL}$ ，或相当者。  
5.5.2SAX固相萃取柱： $500\mathrm{mg / 10mL}$ ，或相当者。  
5.5.3 SCX固相萃取柱： $100\mathrm{mg / 10mL}$ ，或相当者。

6.3 恒温水浴振荡器。  
6.4 匀浆机。  
6.5 涡旋混合器。  
6.6 冷冻离心机  
6.7 固相萃取装置。  
6.8 pH计。  
6.9 离心管： $50~\mathrm{mL}$ 。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，加提取液 $30.0~\mathrm{mL}$ ，中速振荡 $5\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。取 $15.0~\mathrm{mL}$ 于另一 $50~\mathrm{mL}$ 离心管， $20\mathrm{r / min}\sim 30\mathrm{r / min}$ 转速， $50^{\circ}C$ 恒温水浴振摇 $15\mathrm{min}$ ，备用。

# 8.2 衍生

在上述备用液加碘乙酰胺溶液 $3\mathrm{mL}$ ，混匀，室温下放置 $30\mathrm{min}$ 。 $4^{\circ}\mathrm{C},10000\mathrm{r / min}$ 离心 $20\mathrm{min}$ 。取上清液，备用。

# 8.3 净化

8.3.1取 $\mathrm{C_{18}}$ 固相萃取柱，依次用甲醇 $4\mathrm{mL}$ 磷酸盐缓冲液 $5\mathrm{mL}$ 预洗柱。取8.2中衍生后备用液过柱。依次用磷酸缓冲液 $5\mathrm{mL}$ 氢氧化钠溶液 $3\mathrm{mL}$ 洗柱，挤干。加入 $\mathrm{C_{18}}$ 洗脱液 $3\mathrm{mL}$ ，洗脱，收集洗脱液。加水 $15\mathrm{mL}$ 稀释至总体积为 $18~\mathrm{mL}$ ，混匀，备用。  
8.3.2取SAX固相萃取柱，依次用甲醇、SAX预洗液和水各 $2\mathrm{mL}$ 预洗柱。取8.3.1备用液过柱，加水 $1\mathrm{mL}$ 洗柱，挤干。加SAX洗脱液 $3\mathrm{mL}$ ，洗脱，收集洗脱液。加水 $10\mathrm{mL}$ 使总体积为 $13~\mathrm{mL}$ ，混匀，备用。

注：脂肪组织稀释提取液经SAX固相萃取柱净化后，用 $1\mathrm{mL}$ 水洗，挤干。加SAX固相萃取柱洗脱液 $2.0\mathrm{mL}$ ，收集洗脱液，混匀，供高效液相色谱进行分析。

8.3.3取SCX固相萃取柱，依次用甲醇 $1\mathrm{mL}$ 、SCX预洗液 $2\mathrm{mL}$ 和水 $2\mathrm{mL}$ 预洗柱。取8.3.2备用液，过柱，用水 $1\mathrm{mL}$ 淋洗，挤干或真空抽干。加SCX洗脱液 $2.0\mathrm{mL}$ ，洗脱，挤干，收集洗脱液，供高效液相色谱测定。

# 8.4 标准曲线的制备

精密量取标准储备液适量，用磷酸盐缓冲液稀释制成浓度为 $0.1\mu \mathrm{g / mL},0.2\mu \mathrm{g / mL},0.4\mu \mathrm{g / mL}$ 、 $2.0~\mu \mathrm{g / mL},10.0~\mu \mathrm{g / mL}$ 和 $20.0~\mu \mathrm{g / mL}$ 的溶液，分别精密量取 $0.5\mathrm{mL}$ ，按衍生及净化步骤同法处理制成头孢噻呋浓度为 $25~\mathrm{ng / mL},50~\mathrm{ng / mL},100~\mathrm{ng / mL},500~\mathrm{ng / mL},2500~\mathrm{ng / mL}$ 和 $5000~\mathrm{ng / mL}$ 的标准溶液，供高效液相色谱测定。以峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

注：用于脂肪组织检测的标准曲线的制备：经过SAX固相萃取柱净化后，加入水 $1\mathrm{mL}$ 洗柱，挤干。加SAX洗脱液 $2.0~\mathrm{mL}$ ，收集洗脱液，混匀，供高效液相色谱测定。

# 8.5 测定

# 8.5.1 色谱条件

a）色谱柱： $\mathrm{C_{18}}$ 柱（ $250\mathrm{mm}\times 4.6\mathrm{mm},5\mu \mathrm{m})$ ，或相当者；  
b）检测波长： $266~\mathrm{nm}$   
c） 柱温： $30^{\circ}C$   
d）进样量： $40~\mu \mathrm{L}$   
e) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
f）流动相：A为 $0.1\%$ 三氟乙酸水溶液，B为 $0.1\%$ 三氟乙酸乙腈溶液；

g）流动相梯度洗脱条件见表1。

# 8.5.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校正，以色谱峰面积定量，按外标法计算。标准溶液及试样溶液中DCA的响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和试样溶液的高效液相色谱图见附录A。

# 8.6 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试料中头孢噻呋的残留量按公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V}{A _ {\mathrm {S}} \times m} \times 2 \tag {1}
$$

式中：

$X$ ——试料中头孢噻呋残留量的数值，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$

A ——试料溶液中头孢噻呋的峰面积；

$A_{\mathrm{S}}$ ——对照溶液中头孢噻呋的峰面积；

$C_{\mathrm{S}}$ ——对照溶液中头孢噻呋浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$V$ ——SCX(脂肪为SAX)洗脱液体积的数值，单位为毫升 $(\mathrm{mL})$ 。

$m$ 供试试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法在牛、猪肌肉、脂肪、肾脏和猪肝脏中的定量限为 $100\mu \mathrm{g / kg}$ ，牛肝脏中的定量限为 $500~\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法在 $100\mu \mathrm{g} / \mathrm{kg}\sim 6000\mu \mathrm{g} / \mathrm{kg}$ 添加浓度的回收率为 $80\% \sim 110\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

14 原理

16.4 匀浆机（10000r/min）。  
16.5 涡旋混合器。  
16.6 冷冻离心机。  
16.7 pH计。

17 试样的制备与保存

# 17.1 试样的制备

取适量新鲜或冷藏的空白或供试牛奶，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 17.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 18 测定步骤

# 18.1 提取

取试样 $5.0\mathrm{mL}$ ，加提取液 $5.0\mathrm{mL}$ ，充分混合，于 $20~\mathrm{r / min}\sim 30~\mathrm{r / min},50^{\circ}C$ 水浴振摇 $15\mathrm{min},4^{\circ}C$ 产 $10000\mathrm{r / min}$ 离心 $20\mathrm{min}$ ，取上清液（尽量完全转移上清液，如上清液中有漂浮的固状凝结物，将其除去），备用。

取 $\mathrm{C_{18}}$ 固相萃取柱，依次用甲醇 $5\mathrm{mL}$ 、醋酸铵溶液Ⅱ（2次，每次 $5\mathrm{mL}$ )预洗，保持湿润。备用液过柱，并保持溶液缓慢流出。用醋酸铵溶液Ⅱ洗柱2次，每次 $5\mathrm{mL}$ ，排干洗液，备用。

# 18.2 衍生

取碘乙酰胺溶液 $5\mathrm{mL}$ 于上述 $\mathrm{C_{18}}$ 柱，使溶液约在 $30\mathrm{min}$ 流完，保证衍生化反应充分进行。用醋酸铵溶液Ⅱ冲洗2次，每次 $5\mathrm{mL}$ ，再用醋酸溶液 $5\mathrm{mL}$ 洗涤，保持柱子湿润。

# 18.3 净化

SCX柱依次用甲醇、乙腈- $2\%$ 醋酸溶液 $(20:80)$ 各 $2\mathrm{mL}$ 预洗。将 $\mathrm{C_{18}}$ 柱直接放在相应的SCX柱上，用 $\mathrm{C_{18}}$ 洗脱液 $3\mathrm{mL}$ 洗脱2次，使溶液以适当速度缓慢流过SCX固相萃取柱。依次用甲醇 $1\mathrm{mL}$ 、醋酸溶液 $2\mathrm{mL}$ 淋洗SCX柱，挤干柱子和连接口的溶液。用SCX洗脱液 $2.0~\mathrm{mL}$ ，洗脱，收集洗脱液，混匀，供高效液相色谱测定。

# 18.4 标准曲线的制备

精密量取头孢噻呋标准工作液适量，用醋酸铵溶液Ⅱ稀释成浓度为 $0.02\mu \mathrm{g / mL},0.05\mu \mathrm{g / mL}$ 、 $0.1\mu \mathrm{g / mL},0.3\mu \mathrm{g / mL}$ 和 $0.6\mu \mathrm{g / mL}$ 的溶液，分别精密量取 $5.0~\mathrm{mL}$ ，按衍生及净化步骤同法处理制成浓度为 $50~\mathrm{ng / mL},125~\mathrm{ng / mL},250~\mathrm{ng / mL},750~\mathrm{ng / mL}$ 和 $1500~\mathrm{ng / mL}$ 的DCA衍生物标准溶液，供高效液相色谱测定。

# 18.5 测定

# 18.5.1 色谱条件

a）色谱柱： $\mathrm{C_{18}(250~mm\times 4.6~mm,5~\mu m)}$ ，或相当者；  
b）检测波长： $266~\mathrm{nm}$   
c） 柱温： $30^{\circ}C$   
d）进样量： $40~\mu \mathrm{L}$   
e) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
f）流动相：A为 $0.1\%$ 三氟乙酸水溶液，B为 $0.1\%$ 三氟乙酸乙腈溶液；  
g）流动相梯度洗脱条件见表2。

# 18.5.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校正，以色谱峰面积定量，按外标法计算。标准溶液及试样溶液中头孢噻呋的响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和试样溶液的高效液相色谱图见附录B。

# 18.6 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 19 结果计算和表述

试样中头孢噻呋的残留量按公式（2）计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V _ {1}}{A _ {\mathrm {S}} \times V} \tag {2}
$$

式中：

$X$ ——试样中头孢噻呋残留量的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

A ——试样溶液中头孢噻呋色谱峰的峰面积；  
$A_{\mathrm{S}}$ ——标准溶液中头孢噻呋色谱峰的峰面积；  
$V_{1}$ ——SCX洗脱液体积的数值，单位为毫升 $(\mathrm{mL})$   
$V$ ——试料中取样量的数值，单位为毫升 $(\mathrm{mL})$ 。

# 20 检测方法的灵敏度、准确度和精密度

# 20.1 灵敏度

本方法在牛奶中的定量限为 $50\mathrm{ng / mL}$

# 20.2 准确度

本方法在 $50\mathrm{ng / mL}\sim 200\mathrm{ng / mL}$ 添加浓度的回收率为 $80\% \sim 110\%$

# 20.3 精密度

本方法的批内相对标准偏差 $\leq 15\%$ ，批间相对标准偏差 $\leq 15\%$ 。

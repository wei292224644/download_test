# 前言

本标准代替GB4789.35—2016《食品安全国家标准 食品微生物学检验 乳酸菌检验》。

本标准与GB4789.35—2016相比，主要变化如下：

——增加了实时荧光PCR方法为选做方法；  
——修改了乳酸菌的定义、样品制备、培养时间；  
——修改了嗜热链球菌和乳杆菌计数方法的描述；  
——修改了部分培养基成分、储备液浓度和制备方法。

# 食品安全国家标准

# 食品微生物学检验 乳酸菌检验

# 1 范围

本标准规定了含乳酸菌食品中乳酸菌(lactic acid bacteria)的检验方法。

本标准适用于含活性乳酸菌的食品中乳酸菌的检验。

# 2 术语和定义

# 2.1 乳酸菌 lactic acid bacteria

一类可发酵糖主要产生大量乳酸的细菌的通称，不能液化明胶、不产生吲哚、革兰氏阳性、无运动、无芽孢、触酶阴性、硝酸还原酶阴性及细胞色素氧化酶阴性反应的细菌。本标准中乳酸菌主要为乳杆菌属(Lactobacillus)、双歧杆菌属(Bifidobacterium)和嗜热链球菌（Streptococcus thermophilus）。

# 6 操作步骤

# 6.1 样品制备

6.1.1 样品的全部制备过程均应遵循无菌操作程序。  
6.1.2 稀释液在试验前应在 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 条件下充分预热 $15\mathrm{min} \sim 30\mathrm{min}$ 。  
6.1.3 冷冻样品可先使其在 $2^{\circ}\mathrm{C} \sim 5^{\circ}\mathrm{C}$ 条件下解冻，时间不超过 $18\mathrm{h}$ ，也可在温度不超过 $45^{\circ}\mathrm{C}$ 的条件下解冻，时间不超过 $15\mathrm{min}$ 。  
6.1.4 固体和半固体样品：以无菌操作称取 $25\mathrm{g}$ 样品，置于装有 $225~\mathrm{mL}$ 稀释液的无菌均质杯内，于 $8000\times g\sim 10000\times g$ 均质 $1\mathrm{min}\sim 2\mathrm{min}$ ，制成 $1:10$ 样品匀液；或置于 $225~\mathrm{mL}$ 稀释液的无菌均质袋中，用拍击式均质器拍打 $1\mathrm{min}\sim 2\mathrm{min}$ 制成 $1:10$ 的样品匀液。  
6.1.5 液体样品：液体样品应先将其充分摇匀后以无菌吸管吸取样品 $25\mathrm{mL}$ 放入装有 $225\mathrm{mL}$ 稀释液的无菌锥形瓶（瓶内预置适当数量的无菌玻璃珠）或均质袋中，充分振摇或拍击式均质器拍打 $1\mathrm{min} \sim 2\mathrm{min}$ ，制成 $1:10$ 的样品匀液。  
6.1.6 经特殊技术（如包埋技术）处理的含乳酸菌食品样品应在相应技术/工艺要求下进行有效前处理。

# 6.2 稀释及培养

6.2.1 用 $1 \mathrm{~mL}$ 无菌吸管或微量移液器吸取 $1:10$ 样品匀液 $1 \mathrm{~mL}$ , 沿管壁缓慢注于装有 $9 \mathrm{~mL}$ 稀释液的无菌试管中 (注意吸管或微量移液器吸头尖端不要触及稀释液), 振摇试管或换用 1 支无菌吸管反复吹打使其混合均匀, 制成 $1:100$ 的样品匀液。  
6.2.2 另取 $1\mathrm{mL}$ 无菌吸管或微量移液器吸头，按上述操作顺序，做10倍递增样品匀液，每递增稀释一次，即换用1次 $1\mathrm{mL}$ 灭菌吸管或吸头。  
6.2.3 经特殊技术(如包埋技术)处理的含乳酸菌食品应按照相应技术/工艺要求进行稀释。

# 6.3.2 双歧杆菌计数

根据对待检样品双歧杆菌含量的估计，选择2个～3个连续的适宜稀释度，每个稀释度吸取 $1\mathrm{mL}$ 样品匀液于灭菌平皿内，每个稀释度做两个平皿。稀释液移入平皿后，将冷却至 $48^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ 的莫匹罗星锂盐和半胱氨酸盐酸盐改良MRS琼脂培养基倾注入平皿 $15\mathrm{mL} \sim 20\mathrm{mL}$ ，转动平皿使混合均匀。培养基凝固后倒置于 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 厌氧培养，根据双歧杆菌生长特性，一般选择培养 $48\mathrm{h}$ ，若菌落无生长或生长较小可选择培养至 $72\mathrm{h}$ ，培养后计数平板上的所有菌落数。从样品稀释到平板倾注要求在 $15\mathrm{min}$ 内完成。

# 6.3.3 嗜热链球菌计数

根据待检样品嗜热链球菌活菌数的估计，选择2个～3个连续的适宜稀释度，每个稀释度吸取 $1\mathrm{mL}$ 样品匀液于灭菌平皿内，每个稀释度做两个平皿。稀释液移入平皿后，将冷却至 $48^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 的MC琼脂培养基及时倾注入平皿 $15~\mathrm{mL}\sim 20~\mathrm{mL}$ ，转动平皿使混合均匀。培养基凝固后倒置于 $36^{\circ}\mathrm{C}\pm$

$1^{\circ} \mathrm{C}$ 有氧培养, 根据嗜热链球菌生长特性, 一般选择培养 $48 \mathrm{~h}$ , 若菌落无生长或生长较小可选择培养至 $72 \mathrm{~h}$ 。嗜热链球菌在 MC 琼脂培养基平板上的菌落特征为: 菌落中等偏小, 边缘整齐光滑的红色菌落, 直径 $2 \mathrm{~mm} \pm 1 \mathrm{~mm}$ , 菌落背面为粉红色。

# 6.3.4 乳杆菌计数

根据待检样品活菌总数的估计，选择2个～3个连续的适宜稀释度，每个稀释度吸取 $1\mathrm{mL}$ 样品匀液于灭菌平皿内，每个稀释度做两个平皿。稀释液移入平皿后，将冷却至 $48~^\circ \mathrm{C}\sim 50~^\circ \mathrm{C}$ 的MRS琼脂培养基倾注入平皿 $15\mathrm{mL}\sim 20\mathrm{mL}$ ，转动平皿使混合均匀。培养基凝固后倒置于 $36~^\circ \mathrm{C}\pm 1~^\circ \mathrm{C}$ 厌氧培养，根据乳杆菌生长特性，一般选择培养 $48\mathrm{h}$ ，若菌落无生长或生长较小可选择培养至 $72\mathrm{h}$ 。从样品稀释到平板倾注要求在 $15\mathrm{min}$ 内完成。

# 8 乳酸菌的鉴定（可选做）

# 8.1 第一法 生化鉴定

# 8.1.1 纯培养

挑取3个或以上单菌落，嗜热链球菌接种于MC琼脂平板，置 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 有氧培养 $48\mathrm{h}$ ，乳杆菌属接种于MRS琼脂平板，置 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 厌氧培养 $48\mathrm{h}$ 。

# 8.1.2 双歧杆菌的鉴定

按GB4789.34的规定操作。

# 8.1.3 涂片镜检

嗜热链球菌菌体镜下呈球形或球杆状，直径为 $0.5\mu \mathrm{m}\sim 2.0\mu \mathrm{m}$ ，成对或成链排列，无芽孢，革兰氏染色阳性。乳杆菌属镜下菌体形态多样，呈长杆状、弯曲杆状或短杆状，无芽孢，革兰氏染色阳性。

# 8.2 第二法 实时荧光PCR法鉴定

# 8.2.1 纯培养

同8.1.1。

# 8.2.2 DNA模板制备

使用接种环刮取MC琼脂平板或MRS琼脂平板上的菌落2个～10个，悬浮于 $200\mu \mathrm{L}$ 灭菌生理盐水中，充分混匀， $10000\times g\sim 12000\times g$ 离心 $3\mathrm{min}$ ，弃去上清。加入 $50~\mu \mathrm{L}$ DNA提取液涡旋混匀，置于 $100^{\circ}\mathrm{C}$ 水浴或者金属浴中 $10\mathrm{min}$ 后迅速冷却， $10000\times g\sim 12000\times g$ 离心 $3\mathrm{min}$ 。吸取上清液至新的PCR反应管内，作为DNA模板使用。提取后的DNA模板应置于 $4^{\circ}\mathrm{C}$ 供当天使用，否则应于一 $20^{\circ}\mathrm{C}$ 以下保存，并于1周内使用。

注：根据实验室实际情况，也可用商品化试剂盒制备DNA模板。

# 8.2.3 PCR反应体系

总反应体系体积为 $25\mu \mathrm{L}:10\times$ PCR缓冲液 $2.5\mu \mathrm{L}$ 、上下游引物（ $10~\mu \mathrm{mol / L})$ 各 $1\mu \mathrm{L}$ 、探针 $(10\mu \mathrm{mol} / \mathrm{L})0.5\mu \mathrm{L},$ dNTPs( $2.5\mu \mathrm{mol} / \mathrm{L})3\mu \mathrm{L},$ Taq DNA聚合酶 $(5\mathrm{U} / \mu \mathrm{L})0.5\mu \mathrm{L}$ 、模板DNA $1\mu \mathrm{L}$ 灭菌去离子水补足至 $25~\mu \mathrm{L}$ 。每个反应均应设置至少2个平行。

注：反应体系中各试剂的量可根据具体情况或不同的反应总体积进行适当调整。亦可选用含有PCR缓冲液、 $\mathrm{MgCl_2}$ 、dNTP和Taq酶等成分基于Taqman探针的实时荧光PCR预混液。

# 8.2.4 PCR反应条件

$50^{\circ} \mathrm{C} 5 \mathrm{~min}, 95^{\circ} \mathrm{C}$ 预变性 $3 \mathrm{~min}, 94^{\circ} \mathrm{C}$ 变性 $5 \mathrm{~s}, 60^{\circ} \mathrm{C}$ 退火延伸 $40 \mathrm{~s}$ （同时收集FAM荧光），进行40个循环。

注：PCR反应参数可根据基因扩增仪型号实时荧光PCR反应体系进行适当调整。

# 8.2.5 对照设置

检测过程(包括DNA提取)中，每个反应均应设置阳性对照、阴性对照和空白对照。其中阳性对照模板为扩增片段的阳性克隆分子DNA或阳性菌株DNA，阴性对照模板为非乳酸菌菌株DNA，空白对照模板为无菌水。

# 8.2.6 结果判读

# 8.2.6.1 对照的结果判读

阳性对照出现典型扩增曲线， $\mathrm{Ct} \leqslant 30$ ；阴性对照无典型扩增曲线或 $\mathrm{Ct} \geqslant 40$ ；空白对照无典型扩增曲线或 $\mathrm{Ct} \geqslant 40$ 。否则，结果视为无效。

# 8.2.6.2 样品的结果判读

当样品检测 $\mathrm{Ct} \geqslant 40$ 时，判定样品结果为某种乳酸菌阴性；当检测 $\mathrm{Ct} \leqslant 35$ ，可判定该样品结果为某种乳酸菌阳性；当检测 $35 < \mathrm{Ct} < 40$ 时，重复试验，若重复试验结果检测 $\mathrm{Ct} \geqslant 40$ ，则判定为某种乳酸菌阴性，否则，判定为某种乳酸菌阳性。

# 液相色谱-串联质谱法

# 1 范围

本文件规定了动物性食品中左旋咪唑残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、禽、牛、羊的肌肉、肝脏、肾脏和脂肪组织中左旋咪唑残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的左旋咪唑，在碱性条件下用乙酸乙酯提取， $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液萃取，混合型阳离子交换固相萃取柱净化，液相色谱-串联质谱测定，外标法定量。

# 5 试剂或材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.3 甲酸（HCOOH）：色谱纯。  
5.1.4 乙酸乙酯 $(\mathrm{C_4H_8O_2})$   
5.1.5 无水硫酸钠 $\mathrm{(Na_2SO_4)}$   
5.1.6 碳酸氢钠 $\mathrm{(NaHCO_3)}$   
5.1.7 碳酸钠 $\mathrm{(Na_2CO_3)}$ 。  
5.1.8 盐酸（HCl）。  
5.1.9 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。

# 5.2 溶液配制

5.2.1 碳酸氢钠饱和溶液：取水 $100~\mathrm{mL}$ ，加无水碳酸氢钠至不溶解为止，取上清液，现用现配。  
5.2.2 碳酸钠饱和溶液：取水 $100~\mathrm{mL}$ ，加碳酸钠至不溶解为止，取上清液，现用现配。  
5.2.3 碳酸盐缓冲液：取碳酸氢钠饱和溶液 $90~\mathrm{mL}$ 、碳酸钠饱和溶液 $10~\mathrm{mL}$ ，混匀。  
5.2.4 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液：取盐酸 $9\mathrm{mL}$ ，加水稀释至 $1000\mathrm{mL}$ ，混匀。  
5.2.5 $4\%$ 氨水甲醇溶液：取氨水 $4\mathrm{mL}$ ，用甲醇稀释至 $100~\mathrm{mL}$ ，混匀，现用现配。  
5.2.6 $0.1\%$ 甲酸溶液：取甲酸 $500~\mu \mathrm{L}$ ，用水稀释至 $500~\mathrm{mL}$ ，混匀。

5.2.7 $10\%$ 乙腈甲酸溶液：取乙腈 $10\mathrm{mL}$ ，用 $0.1\%$ 甲酸水溶液稀释至 $100~\mathrm{mL}$ ，混匀。

# 5.3 标准品

盐酸左旋咪唑（Levamisolehydrochloride， $\mathrm{C_{11}H_{12}N_2S\cdot HCl}$ ,CAS号：16595-80-5），含量 $\geq 99\%$

# 5.4 标准溶液制备

5.4.1 标准储备液：取盐酸左旋咪唑标准品适量(相当于左旋咪唑约 $10\mathrm{mg}$ )，精密称定，加甲醇适量使溶解并稀释定容至 $10\mathrm{mL}$ 容量瓶，配制成浓度为 $1\mathrm{mg/mL}$ 的左旋咪唑标准储备液。 $-18^{\circ}\mathrm{C}$ 保存，有效期6个月。  
5.4.2 标准工作液：精密量取标准储备液 $1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 容量瓶，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的左旋咪唑标准工作液。 $4^{\circ}\mathrm{C}$ 保存，有效期3个月。  
5.5 材料  
5.5.1 混合型阳离子交换固相萃取柱： $60~\mathrm{mg / 3~mL}$ ，或相当者。  
5.5.2 陶瓷均质子。  
5.5.3 微孔尼龙滤膜： $0.22\mu \mathrm{m}$

# 6 仪器设备

6.1 液相色谱-串联质谱仪：配电喷雾电离源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 涡旋混合器。  
6.4 涡旋振荡器。  
6.5 高速冷冻离心机：转速 $\geqslant 10000\mathrm{r / min}$   
6.6 固相萃取装置。  
6.7 氮吹仪。

# 7 试样的制备与保存

# 7.1 试样制备

取适量新鲜或解冻的空白或供试组织，绞碎，并均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准溶液，作为空白添加试样。

# 7.2 试样保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取试料 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 聚丙烯离心管中（脂肪样品需在 $60^{\circ}C$ 水浴下加热 $20\mathrm{min}$ ），加入一粒陶瓷均质子，加碳酸盐缓冲液 $0.5\mathrm{mL}$ ，无水硫酸钠 $2\mathrm{g}$ ，乙酸乙酯 $10\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，振荡 $5\mathrm{min}$ $6000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上层乙酸乙酯至另一离心管，残渣用乙酸乙酯 $10\mathrm{mL}$ 重复提取一次，合并两次提取液，加 $0.1\mathrm{mol / L}$ 盐酸溶液 $5\mathrm{mL}$ ，振荡 $5\mathrm{min}$ ， $6000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取下层水相至另一离心管中，有机相中再加 $0.1\mathrm{mol / L}$ 盐酸溶液 $5\mathrm{mL}$ 重复萃取一次，合并两次萃取液，备用。

# 8.2 净化

取混合型阳离子交换固相萃取柱，依次用甲醇 $3\mathrm{mL},0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液 $3\mathrm{mL}$ 活化。取备用液过柱，流速控制在 $1\mathrm{mL} / \mathrm{min}\sim 2\mathrm{mL} / \mathrm{min}$ ，依次用水 $3\mathrm{mL}$ 、甲醇 $3\mathrm{mL}$ 淋洗，抽干， $4 \%$ 氨水甲醇溶液 $3\mathrm{mL}$ 洗脱，抽干，收集洗脱液， $40^{\circ}C$ 下氮气吹干。用 $10\%$ 乙腈甲酸溶液 $1.0~\mathrm{mL}$ 溶解残余物，超声 $1\mathrm{min}$ ，过

0.22 $\mu$ m 微孔尼龙滤膜，供液相色谱-串联质谱仪测定。

# 8.3 基质匹配标准曲线的制备

精密量取标准工作液适量，用甲醇稀释，配制成浓度分别为 $10\mathrm{ng / mL},20\mathrm{ng / mL},50\mathrm{ng / mL}$ 、 $100\mathrm{ng / mL},200\mathrm{ng / mL}$ 和 $500\mathrm{ng / mL}$ 的系列标准溶液，各取 $100~\mu \mathrm{L}$ ，分别加于经提取、净化步骤处理的空白试料洗脱液中，于 $40^{\circ}C$ 下氮气吹干，用 $10\%$ 乙腈甲酸水溶液 $1.0\mathrm{mL}$ 溶解残余物，超声 $1\mathrm{min}$ ，混匀。配制成浓度分别为 $1\mathrm{ng / mL},2\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL}$ 和 $50\mathrm{ng / mL}$ 的基质匹配标准溶液，过微孔尼龙滤膜，供液相色谱-串联质谱仪测定。以左旋咪唑定量离子质量色谱峰面积为纵坐标，基质匹配标准溶液浓度为横坐标，绘制基质匹配标准曲线。求回归方程和相关系数。

c）进样量： $2\mu \mathrm{L}$   
d) 流速: $0.30 \mathrm{~mL} / \mathrm{min}$ ;

# 8.4.3 测定法

# 8.4.3.1 定性测定

在同样测试条件下，试料溶液中左旋咪唑的保留时间与基质匹配标准工作液中的保留时间相对偏差在 $\pm 2.5\%$ 以内，且检测到的相对离子丰度，应当与浓度相当的基质匹配标准溶液相对离子丰度一致。其允许偏差应符合表3要求。

# 8.4.3.2 定量测定

取试料溶液和相应的基质匹配标准工作液，作单点或多点校准，按外标法以峰面积定量，基质匹配标准工作液及试料溶液中的左旋咪唑响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，左旋咪唑基质匹配标准溶液的特征离子质量色谱图见附录A。

# 8.5 空白试验

除不加试料外，采用完全相同的测定步骤进行测定。

# 9 结果计算和表述

试样中待测物残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {s}} \times A \times V}{A _ {\mathrm {s}} \times m} \tag {1}
$$

式中：

$X$ ——试样中左旋咪唑残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——试样中左旋咪唑定量离子峰面积；

$A_{\mathrm{s}}$ 基质匹配标准溶液中左旋咪唑定量离子峰面积；

$C_{\mathrm{s}}$ ——基质匹配标准溶液中左旋咪唑浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——最终试样定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量的数值，单位为克（g）。

# 10 检测方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1.0\mu \mathrm{g / kg}$

# 10.2 准确度

本方法左旋咪唑在肌肉、肾脏和脂肪组织 $1.0\mu \mathrm{g / kg}\sim 10\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim$ $120\% ,10~\mu \mathrm{g / kg}\sim 20~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$ ;在肝脏组织 $1.0\mu \mathrm{g / kg}\sim 10\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\% ,10~\mu \mathrm{g / kg}\sim 200~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim$ $110\%$ 。

# 10.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

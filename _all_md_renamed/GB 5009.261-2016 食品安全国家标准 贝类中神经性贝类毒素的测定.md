2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替SN/T1573—2013《出口贝类中神经性贝类毒素检验方法 小鼠生物法》。

本标准与SN/T1573—2013相比，主要变化如下：

——标准名称修改为“食品安全国家标准 贝类中神经性贝类毒素的测定”。

# 食品安全国家标准

# 贝类中神经性贝类毒素的测定

# 1 范围

本标准规定了贝类中神经性贝类毒素(NSP)检测的小鼠生物测定方法。

本标准适用于贝类及制品中神经性贝类毒素(NSP)的检测。

# 2 原理

用乙醚提取贝类中神经性贝类毒素，提取物经减压蒸干后，再以 $1\%$ 吐温-60生理盐水为分散介质，制备 $\mathrm{NSP - 1\%}$ 吐温-60生理盐水混悬液，将该混悬液注射入腹腔，观察小鼠存活情况，计算其毒力。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 盐酸（HCl）。  
3.1.2 无水乙醚 $(\mathrm{C_4H_{10}O})$ 。  
3.1.3 吐温-60 $(\mathrm{C}_{64}\mathrm{H}_{126}\mathrm{O}_{26})$ 。  
3.1.4 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.5 次氯酸钠（NaClO）。

# 3.2 试剂配制

3.2.1 氯化钠溶液 $(0.85\%)$ ：称取 $0.85\mathrm{gNaCl}$ ，加水溶解并定容至 $100~\mathrm{mL}$ 。  
3.2.2 $1\%$ 吐温-60：称取 $1.0\mathrm{g}$ 吐温-60，用 $0.85\%$ 氯化钠溶液溶解并定容至 $100~\mathrm{mL}$ 。  
3.2.3 次氯酸钠溶液 $(5\%)$ ：称取次氯酸钠 $50\mathrm{g}$ ，用水溶解，并稀释至 $1000~\mathrm{mL}$ 。

# 3.3 标准品

短裸甲藻毒素标准品(brevetoxin-2)，纯度 $\geq 90\%$

# 3.4 材料

小白鼠：体重为 $19\mathrm{g}\sim 21\mathrm{g}$ 的健康ICR品系雄性小鼠。

# 4 仪器和设备

4.1 旋转蒸发器。  
4.2 均质器。

4.3 天平：感量为 $0.1\mathrm{g}$   
4.4 分液漏斗。  
4.5 圆底烧瓶。  
4.6 布氏漏斗。  
4.7 一次性注射器：1 mL。  
4.8 离心机：转速 $\geq 6000\mathrm{r / min}$   
4.9 金属筛网：孔径约 $2\mathrm{mm}$

# 5 分析步骤

注：橡胶手套、玻璃制品等用过的器材,以及废弃的提取液应在次氯酸钠溶液(5%)中浸泡 $1\mathrm{\;h}$ 以上,以使毒素分解。

# 5.1 试样制备

# 5.1.1 样品采集

每个分析样品至少要取10个以上贝类，并使贝肉达 $200\mathrm{g}$ 以上。

冷冻样品置于保温盒中冷冻送检，或保证其处于低温状态 $(0\,^{\circ}\mathrm{C} \sim 10\,^{\circ}\mathrm{C})$ 送检。如为带壳样品，应开壳去除水分后冷冻送检。

# 5.1.2 试样制备

# 5.1.2.1 生鲜带壳样品

用清水将贝壳外表彻底洗净，切断闭壳肌，开壳，用水淋洗内部去除泥沙及其他外来物。将闭壳肌和连接在胶合部的组织分开，取出贝肉，切勿割破肉体。开壳前不要加热或用麻醉剂。收集 $200\mathrm{g}$ 贝肉放在孔径约 $2\mathrm{mm}$ 的金属筛网上沥水 $5\mathrm{min}$ ，捡出碎壳等杂物，将贝肉均质，备用。

# 5.1.2.2 冷冻样品

在室温下，使冷冻样品呈半冷冻状态。带壳冷冻样品，按5.1.2.1方法清洗、开壳、淋洗取肉，除去贝肉外部附着的冰片，抹去水分后，室温缓化。收集 $200\mathrm{g}$ 贝肉放在孔径约 $2\mathrm{mm}$ 的金属筛网上沥水 $5\mathrm{min}$ ，将贝肉均质，备用。

# 5.1.2.3 贝类罐头

将罐头内容物沥干水分，倒入均质器充分均质，备用。

# 5.1.2.4 贝肉干制品

称取 $100\mathrm{g}$ 干制品放入足量清水中浸泡 $24\mathrm{h} \sim 48\mathrm{h}(4^{\circ}\mathrm{C}$ 冷藏)，沥干、均质、备用。

# 5.1.2.5 盐渍品

用清水洗涤，流水脱盐，沥干，均质、备用。

# 5.2 试样提取

取 $100\mathrm{g}$ 试样到 $500~\mathrm{mL}$ 烧杯中，加入 $5\mathrm{g}$ 氯化钠和 $1\mathrm{mL}$ 浓盐酸。搅拌均匀。边搅拌边加热混合物至沸腾，文火煮 $5\mathrm{min}$ ，冷却至室温。

将混合物移至 $500~\mathrm{mL}$ 离心管中，用 $50~\mathrm{mL}$ 乙醚冲洗烧杯，将冲洗液一同移到离心管中。向离心管

中再加入 $100~\mathrm{mL}$ 乙醚，盖塞，充分振摇。 $6000\mathrm{r / min}$ 离心 $15\mathrm{min}$

离心后小心倒出醚层(上层)溶液至 $1000\mathrm{mL}$ 分液漏斗中，用乙醚重复三次抽提离心管中的沉淀，三次抽提用总量为 $250~\mathrm{mL}$ 乙醚，转移醚层液体至分液漏斗中。轻轻振荡(不能生成乳浊液)，静置分层后去除水层(下层)及贝肉碎片。

将乙醚层移入 $500~\mathrm{mL}$ 的圆底烧瓶中，减压浓缩(旋转蒸发器， $35^{\circ}C\pm 1^{\circ}C$ )去除乙醚。

用 $1\%$ 吐温-60生理盐水将浓缩物在刻度试管中稀释到 $10~\mathrm{mL}$ ，充分振摇，制成均匀 $\mathrm{NSP - 1\%}$ 吐温-60生理盐水混悬液。此时 $1\mathrm{mL}$ 液量相当于预先测定的 $10\mathrm{g}$ 去壳贝肉的重量，以此悬浮液作为试验原液进行动物实验。

# 5.3 小白鼠试验

选择体重为 $19\mathrm{g}\sim 21\mathrm{g}$ 的健康ICR品系雄性小鼠6只，随机分2组：检测样品组和空白对照组（ $1\%$ 吐温-60生理盐水），3只/组。

分别取 $1\mathrm{mL}$ 待测液或 $1\%$ 吐温-60生理盐水腹腔注射。注射时若有提取液溢出，须将该只小鼠丢弃，并重新注射一只小鼠。记录注射完毕时间和小鼠停止呼吸时的死亡时间，连续观测 $930\mathrm{min}$ 。若小鼠的中位数死亡时间在 $2\mathrm{h}$ 以内，则要用 $1\%$ 吐温-60生理盐水对待测液进行稀释后重新注射，直到小鼠的死亡时间在 $2\mathrm{h} \sim 6\mathrm{h}$ 内。

# 6 分析结果的表述

# 6.1 待测样品校正鼠单位（CMU）的确定

根据待测样品的小鼠死亡时间(见表A.1)查出相应的鼠单位数MU；根据小鼠的重量(见表A.2)查出其对应的体重校正系数。同一只小鼠的鼠单位与体重校正系数相乘，即得该只小鼠的CMU。选取检测样品受试组中3只小鼠CMU的中位数，即为该样品受试组的中位数CMU，以此值进行6.2的计算。

# 6.2 毒素毒力的计算与结果表述

# 6.2.1 NSP毒力的计算

试样中NSP毒力按式(1)计算：

$$
X = \frac {\mathrm {C M U} \times 1 0}{1 0 0} \times D F \quad \dots \tag {1}
$$

式中：

$X$ ——试样中NSP毒力，单位为鼠单位每克 $(\mathrm{MU} / \mathrm{g})$ ；

CMU——检测样品受试组小鼠的中位数校正鼠单位，单位为鼠单位每毫升 $(\mathrm{MU} / \mathrm{mL})$ ；

$DF$ ——稀释倍数。

注：10—单位为毫升；100—单位为克。

# 6.2.2 结果表述

在空白对照组小鼠正常的情况下进行如下判断和表述：

若小鼠的死亡时间等于 $360\mathrm{min}$ ，则待测样品的NSP毒力即相当于 $0.2\mathrm{MU / g}$ 。

若实验组中位数死亡时间小于 $120\mathrm{min}$ ，则应对样品提取液进行稀释，再选取3只小鼠进行试验，直至得到中位数死亡时间为 $120\mathrm{min} \sim 360\mathrm{min}$ 为止，根据最后的稀释液实验结果计算样品的鼠单位毒力，报告该样品的NSP毒力为： $\times \times \times \mathrm{MU / g}$ 。

若实验组中位数死亡时间大于 $360\mathrm{min}$ ，则直接计算确定样品鼠单位毒力，报告该样品的NSP毒力为： $\times \times \times \mathrm{MU / g}$ 。

若实验组中所有小鼠在观察 $930\mathrm{min}$ 内均不死亡，则也可报告该样品的NSP毒力小于 $0.1\mathrm{MU / g}$ 。

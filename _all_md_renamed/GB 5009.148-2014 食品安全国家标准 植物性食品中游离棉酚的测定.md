# 前言

本标准代替GB/T5009.148—2003《植物性食品游离棉酚的测定》。

本标准与GB/T5009.148—2003相比，主要变化如下：

——修改了原理；  
——修改了检出限；  
——增加了附录A中样品加标液相色谱图A.2。

# 食品安全国家标准

# 植物性食品中游离棉酚的测定

# 1 范围

本标准规定了植物油或以棉籽饼为原料的其他液体食品中游离棉酚的测定方法。

本标准适用于植物油或以棉籽饼为原料的其他液体食品中游离棉酚的测定。

# 2 原理

植物油中游离棉酚经无水乙醇提取，利用高效液相色谱法检测，色谱峰保留时间定性，外标法定量。

以棉籽饼为原料的水溶性液体样品中的游离棉酚经无水乙醚提取，浓缩至干，再加入乙醇溶解，利用高效液相色谱法检测，色谱峰保留时间定性，外标法定量。

# 3 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 磷酸 $\mathrm{(H_3PO_4)}$ 。  
3.1.2 无水乙醇 $\mathrm{(CH_3CH_2OH)}$   
3.1.3 丙酮 $\mathrm{(C_3H_6O)}$ 。  
3.1.4 氮气 $(\mathrm{N}_2)$ 。  
3.1.5 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。

# 3.2 试剂配制

磷酸溶液：取 $300~\mathrm{mL}$ 水，加 $6.0~\mathrm{mL}$ 磷酸，混匀，经 $0.45\mu \mathrm{m}$ 滤膜过滤。

# 3.3 标准品

棉酚 $\left(\mathrm{C}_{32} \mathrm{H}_{34} \mathrm{O}_{10}\right)$ 标准品：纯度 $>95\%$ 。

# 3.4 标准溶液配制

3.4.1 棉酚标准储备液 $(1.0\mathrm{mg / mL})$ ：准确称取 $0.1\mathrm{g}$ （精确到 $0.0001\mathrm{g}$ )棉酚纯品，用丙酮溶解，并定容至 $100.00~\mathrm{mL}$ 。  
3.4.2 棉酚中间标准溶液 $(50\mu \mathrm{g} / \mathrm{mL})$ : 取棉酚标准储备液 $(1.0\mathrm{mg / mL})5.0\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中, 用无水乙醇定容至刻度。  
3.4.3 棉酚标准工作液：准确吸取 $1.00\mathrm{mL},2.00\mathrm{mL},5.00\mathrm{mL},8.00\mathrm{mL}$ 棉酚中间标准液于 $10~\mathrm{mL}$ 容量瓶中，用无水乙醇稀释至刻度，此溶液相应于 $5\mu \mathrm{g / mL},10\mu \mathrm{g / mL},25\mu \mathrm{g / mL},40\mu \mathrm{g / mL}$ 的标准系列溶液。

# 4 仪器和设备

4.1 高效液相色谱仪：带紫外检测器或者二极管阵列检测器。  
4.2 浓缩仪。  
4.3 离心机： $3000\mathrm{r / min}$

# 5 分析步骤

# 5.1 试样制备

5.1.1 植物油：称取油样 $1\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）于离心试管中，加入 $5\mathrm{mL}$ 无水乙醇，剧烈振摇 $2\mathrm{min}$ ，静置分层（或冰箱过夜），取上清液滤纸过滤， $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，上清液过 $0.45\mu \mathrm{m}$ 滤膜，即为试样液。  
5.1.2 以棉籽饼为原料的水溶性液体样品：称取样品 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )于离心试管中，加入 $10~\mathrm{mL}$ 无水乙醚，振摇 $2\mathrm{min}$ ，静置 $5\mathrm{min}$ ，取上层乙醚层 $5\mathrm{mL}$ ，用氮气吹干，用 $1.0\mathrm{mL}$ 无水乙醇定容，过 $0.45\mu \mathrm{m}$ 滤膜，即为试样液。

# 5.2 色谱测定

# 5.2.1 参考色谱条件

参考色谱条件如下：

——色谱柱： $\mathrm{C_{18}}$ 柱， $250\mathrm{mm}\times 4.6\mathrm{mm},5\mu \mathrm{m}$ ，或具同等性能的色谱柱；  
——流动相：甲醇：磷酸溶液 $= 85:15$   
流速： $1.0~\mathrm{mL / min}$   
柱温： $40^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$   
——测定波长： $235~\mathrm{nm}$   
进样体积： $10~\mu \mathrm{L}$

# 5.2.2 标准曲线的绘制

分别将棉酚标准工作液注入高效液相色谱仪中，记录峰高或峰面积。以峰高或者峰面积为纵坐标，以棉酚标准工作液浓度为横坐标绘制标准曲线(标准溶液液相色谱图参见附录A中的图A.1)。

# 5.2.3 试样测定

将试样液注入高效色谱仪中，记录峰高或者峰面积，根据标准曲线计算的待测液中棉酚的浓度（样品加标液相色谱图参见图A.2）。

# 6 分析结果的表述

试样5.1.1中游离棉酚含量按式(1)计算：

$$
X = \frac {5 \times c}{m} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中棉酚的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

5 ——5.1.1折合所用无水乙醇的体积，单位为毫升 $(\mathrm{mL})$

$c$ ——测定试样液中棉酚的含量，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试样5.1.2中游离棉酚含量按式(2)计算：

$$
X = \frac {2 \times c}{m} \quad \dots \tag {2}
$$

式中：

$X$ ——试样中棉酚的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

2 ——5.1.2折合所用无水乙醇的体积，单位为毫升 $(\mathrm{mL})$

$c$ ——测定试样液中棉酚的含量，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，结果保留2位有效数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

植物油体样品取样 $1.0\mathrm{g}$ 时，检出限为 $2.5\mathrm{mg / kg}$ ，定量限为 $7.5\mathrm{mg / kg}$ 。以棉籽饼为原料的水溶性液体样品取样 $10\mathrm{g}$ 时，检出限为 $0.25\mathrm{mg / kg}$ ，定量限为 $0.75\mathrm{mg / kg}$ 。

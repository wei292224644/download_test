# 前言

本标准代替GB5009.5—2016《食品安全国家标准 食品中蛋白质的测定》。

本标准与GB5009.5—2016相比，主要变化如下：

——修改了标准适用范围；  
——修改了第一法凯氏定氮法的取样量和标准滴定溶液浓度；  
——增加了第二法标准溶液和显色剂的储存条件和时间；  
——修改了第三法 燃烧法的适用范围和检出限；  
——增加了附录B燃烧法校正曲线；  
——修改了分析结果表述和附录C蛋白质折算系数表；  
——修改了精密度。

# 食品安全国家标准

# 食品中蛋白质的测定

# 1 范围

本标准规定了食品中蛋白质的测定方法。

本标准适用于食品中蛋白质的测定。

# 第一法 凯氏定氮法

# 2 原理

食品中的蛋白质在催化加热条件下被分解，产生的氨与硫酸结合生成硫酸铵。碱化蒸馏使氨游离，用硼酸吸收后以硫酸或盐酸标准滴定溶液滴定，根据硫酸或盐酸的消耗量计算氮含量，再乘以折算系数，即为蛋白质的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 硫酸铜 $(\mathrm{CuSO}_4\cdot 5\mathrm{H}_2\mathrm{O})$ 。  
3.1.2 硫酸钾 $\mathrm{(K_2SO_4)}$ 。  
3.1.3 氢氧化钠(NaOH)。  
3.1.4 硫酸 $(\mathrm{H}_2\mathrm{SO}_4) : \geqslant 98\%$ 。  
3.1.5 硼酸 $\mathrm{(H_3BO_3)}$ 。  
3.1.6 乙醇 $(\mathrm{C}_2\mathrm{H}_5\mathrm{OH})$ ： $95\%$   
3.1.7 甲基红 $\mathrm{(C_{15}H_{15}N_3O_2)}$ 。  
3.1.8 溴甲酚绿 $(\mathrm{C}_{21}\mathrm{H}_{14}\mathrm{Br}_4\mathrm{O}_5\mathrm{S})$ 。  
3.1.9 亚甲基蓝 $\mathrm{(C_{16}H_{18}ClN_3S\cdot 3H_2O)}$   
3.1.10 pH试纸：测量范围 $0\sim 14$

# 3.2 试剂配制

3.2.1 硼酸溶液 $(20\mathrm{g / L})$ ：称取 $20\mathrm{g}$ 硼酸，加水溶解后并稀释至 $1000~\mathrm{mL}$ 。  
3.2.2 氢氧化钠溶液 $(400\mathrm{g / L})$ ：称取 $40\mathrm{g}$ 氢氧化钠加水溶解后，冷却，并稀释至 $100~\mathrm{mL}$ 。  
3.2.3 硫酸标准滴定溶液 $\left[c\left(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4\right)\right]$ 或盐酸标准滴定溶液 $[c(\mathrm{HCl})]0.10\mathrm{mol / L}$ 。按照GB/T601的要求配制和标定，或经国家认证并授予标准物质证书的滴定溶液标准物质。

3.2.4 硫酸标准滴定溶液 $\left[c\left(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4\right)\right]$ 或盐酸标准滴定溶液 $[c(\mathrm{HCl})]0.05\mathrm{mol / L}$ ：用移液管吸取 $50~\mathrm{mL}~0.10~\mathrm{mol / L}$ 硫酸标准滴定溶液 $\left[c\left(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4\right)\right]$ 或盐酸标准滴定溶液 $[c(\mathrm{HCl})]$ 至容量瓶，用水稀释到 $100~\mathrm{mL}$ ，临用现配，必要时重新标定。或经国家认证并授予标准物质证书的滴定溶液标准物质。  
3.2.5 甲基红指示剂 $(1\mathrm{g / L})$ ：称取 $0.1\mathrm{g}$ 甲基红，溶于 $95\%$ 乙醇，稀释至 $100~\mathrm{mL},10^{\circ}\mathrm{C}\sim 30^{\circ}\mathrm{C}$ ，保存12个月。  
3.2.6 亚甲基蓝指示剂 $(1\mathrm{g / L})$ ：称取 $0.1\mathrm{g}$ 亚甲基蓝，溶于 $95\%$ 乙醇，稀释至 $100~\mathrm{mL},10^{\circ}\mathrm{C}\sim 30^{\circ}\mathrm{C}$ 保存12个月。  
3.2.7 溴甲酚绿指示剂 $(1\mathrm{g} / \mathrm{L})$ ：称取 $0.1\mathrm{g}$ 溴甲酚绿，溶于 $95\%$ 乙醇，稀释至 $100\mathrm{mL},10^{\circ}\mathrm{C}\sim 30^{\circ}\mathrm{C}$ 保存12个月。  
3.2.8 A混合指示液：2份甲基红乙醇溶液与1份亚甲基蓝乙醇溶液，混匀。 $10^{\circ}\mathrm{C} \sim 30^{\circ}\mathrm{C}$ 保存12个月。  
3.2.9 B混合指示液：1份甲基红乙醇溶液与5份溴甲酚绿乙醇溶液，混匀。 $10^{\circ}\mathrm{C} \sim 30^{\circ}\mathrm{C}$ 保存12个月。

# 5 分析步骤

# 5.1 凯氏定氮法

# 5.1.1 试样制备

液态样品摇匀；含有气体的蛋白饮料应采用超声波的方式脱气后再称量测定；基质均匀的半固态样品和粉状样品直接称量；其他样品需匀浆或粉碎均匀。粮食类样品至少要研磨 $200\mathrm{g}$ 样品。颗粒、块状固体样品研磨后要充分混匀，使其完全通过 $0.9\mathrm{mm}(20$ 目)孔径的筛子。产品标准中对样品处理有特殊要求的，按产品标准执行。制备好的试样应尽快测定。

# 5.1.2 水分测定

如试样结果以干基计算，应按照GB5009.3或相应产品标准中规定的方法测定水分。

# 5.1.3 试样处理

称取固体试样 $0.2\mathrm{g}\sim 2\mathrm{g}$ 、半固体试样 $2\mathrm{g}\sim 5\mathrm{g}$ ，（精确至 $0.001\mathrm{g})$ ，液体试样 $10\mathrm{mL(g)}\sim 25\mathrm{mL(g)}$ （相当于 $30~\mathrm{mg}\sim 40~\mathrm{mg}$ 氮)，分别移入干燥的 $100~\mathrm{mL},250~\mathrm{mL}$ 或 $500~\mathrm{mL}$ 定氮瓶中，加入 $0.4\mathrm{g}$ 硫酸铜、

6 g 硫酸钾及 $20 \mathrm{~mL}$ 硫酸, 轻摇后于瓶口放一小漏斗, 将瓶以 $45^{\circ}$ 角斜放置于有小孔的石棉网上。缓慢加热, 待内容物全部碳化, 泡沫完全停止后, 加大火力, 并保持瓶内液体微沸, 至液体呈蓝绿色并澄清透明后, 再继续加热 $0.5 \mathrm{~h} \sim 1 \mathrm{~h}$ 。取下定氮瓶冷却至室温, 小心加入 $20 \mathrm{~mL}$ 水, 将内容物全部转移至 $100 \mathrm{~mL}$ 容量瓶中, 并用少量水洗定氮瓶内壁, 洗液并入容量瓶中, 再加水至刻度, 混匀备用。同时进行空白试验。

# 5.1.4 测定

按附录A装好定氮蒸馏装置，向水蒸气发生器内装水至2/3处，加入数粒玻璃珠，加甲基红数滴及数毫升硫酸，至溶液成红色，以保持水呈酸性，加热煮沸水蒸气发生器内的水并保持沸腾。

向接收瓶内加入 $10.0\mathrm{mL}$ 硼酸溶液及3滴～4滴A混合指示剂或B混合指示剂，并使冷凝管的下端插入液面下，根据试样中氮含量，准确吸取 $2.0\mathrm{mL}\sim 10.0\mathrm{mL}$ 试样处理液由小玻杯注入反应室，以 $10~\mathrm{mL}$ 水洗涤小玻杯并使之流入反应室内，随后塞紧棒状玻塞。将 $10.0\mathrm{mL}$ 氢氧化钠溶液倒入小玻杯，提起玻塞使其缓缓流入反应室，立即将玻塞盖紧，并水封。夹紧螺旋夹，开始蒸馏。蒸馏 $15\mathrm{min}$ 后移动蒸馏液接收瓶，液面离开冷凝管下端，再蒸馏约 $1\mathrm{min}$ ，至用 $\mathrm{pH}$ 试纸检测馏出液为中性。用少量水冲洗冷凝管下端外部，取下蒸馏液接收瓶。尽快以硫酸或盐酸标准滴定溶液滴定至终点，如用A混合指示液，终点颜色为紫红色；如用B混合指示液，终点颜色为浅灰红色。

# 5.2 全自动或半自动凯氏定氮仪法

称取固体试样 $0.2\mathrm{g}\sim 2\mathrm{g}$ 、半固体试样 $2\mathrm{g}\sim 5\mathrm{g}$ ，（精确至 $0.001\mathrm{g})$ ，液体试样 $10\mathrm{g(mL)}\sim 25\mathrm{g(mL)}$ （相当于 $30~\mathrm{mg}\sim 40~\mathrm{mg}$ 氮)，再加入 $0.4\mathrm{g}$ 硫酸铜、 $6\mathrm{g}$ 硫酸钾及 $20~\mathrm{mL}$ 硫酸于消化炉进行消化。当消化炉温度达到 $420^{\circ}C$ 之后，继续消化至少 $1\textrm{h}$ ，此时消化管中的液体呈绿色透明状，于全自动或半自动凯氏定氮仪(使用前根据不同仪器优化分析参数，加入氢氧化钠溶液，盐酸或硫酸标准溶液以及含有混合指示剂A或B的硼酸溶液)进行试样检测。

当蛋白质含量 $\leqslant 1\mathrm{g} / 100\mathrm{g}$ 或 $1\mathrm{g} / 100\mathrm{mL}$ 时，建议使用 $0.05\mathrm{mol} / \mathrm{L}$ 的标准滴定液滴定，当蛋白质含量 $>1\mathrm{g} / 100\mathrm{g}$ 或 $1\mathrm{g} / 100\mathrm{mL}$ 时，建议使用 $0.10\mathrm{mol} / \mathrm{L}$ 的标准滴定液滴定。

注：当蛋白质含量过低且滴定体积小于 $1\mathrm{mL}$ 或蛋白质含量过高导致滴定体积大于自动凯氏定氮仪滴定管体积时，可调整称样量以减小滴定误差。当样品脂肪含量过高或糖含量过高时，可调整称样量以减小或避免消化时出现爆沸、喷溅、溢流的情况。

# 6 分析结果的表述

试样中蛋白质的含量按式(1)进行计算：

$$
X = \frac {\left(V _ {1} - V _ {2}\right) \times c \times 0 . 0 1 4 0}{m \times V _ {3} / V _ {4}} \times F \times 1 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——试样中蛋白质的含量，单位为克每百克或克每百毫升 $(\mathrm{g} / 100\mathrm{g}$ 或 $\mathrm{g} / 100\mathrm{mL})$

$V_{1}$ ——试液消耗硫酸或盐酸标准滴定液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试剂空白消耗硫酸或盐酸标准滴定液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫酸或盐酸标准滴定溶液浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

0.0140— $1.0~\mathrm{mL}$ 硫酸 $\left[c\left(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4\right) = 1.000\mathrm{mol / L}\right]$ 或盐酸 $[c(\mathrm{HCl}) = 1.000\mathrm{mol / L}]$ 标准滴定溶液相当的氮的质量，单位为克每毫摩尔（ $\mathrm{g / mmol}$ ）；

$m$ ——试样的质量，单位为克或毫升(g或 $\mathrm{mL}$ ）；

$V_{3}$ ——吸取消化液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{4}$ —消解溶液的定容体积，单位为毫升 $(\mathrm{mL})$

$F$ ——蛋白质折算系数（见附录C）；

100 ——由 $\mathrm{g / g}$ 转化为 $\mathrm{g} / 100\mathrm{g}$ 的换算系数。

蛋白质含量 $\geqslant 1\mathrm{g} / 100\mathrm{g}$ 或 $1\mathrm{g} / 100\mathrm{mL}$ 时，结果保留三位有效数字；蛋白质含量 $< 1\mathrm{g} / 100\mathrm{g}$ 或 $1\mathrm{g} / 100\mathrm{mL}$ 时，结果保留两位有效数字。

注1：分析结果以氮含量表述时，不需要乘蛋白质折算系数 $F$ 。分析结果以蛋白质含量表述时，应同时报告蛋白质折算系数。

注2：当用半自动或全自动凯氏定氮仪全部转移消化液时， $V_{3} = V_{4}$ 。

注3：以干基计算试样中蛋白质的含量需根据试样的水分含量折算。

# 7 精密度

当样品中蛋白质含量 $\leqslant 10\mathrm{g} / 100\mathrm{g}$ 或 $10\mathrm{g} / 100\mathrm{mL}$ ，在重复条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

当样品中蛋白质含量 $>10\mathrm{g} / 100\mathrm{g}$ 或 $10\mathrm{g} / 100\mathrm{mL}$ ，在重复条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 8 其他

当用 $0.05\mathrm{mol} / \mathrm{L}$ 盐酸滴定液时，称样量为 $5.0\mathrm{g}$ 或 $5.0\mathrm{mL}$ ，本方法对氮的检出限为 $0.008\mathrm{g} / 100\mathrm{g}$ 或 $0.008\mathrm{g} / 100\mathrm{mL}$ 。

# 第二法 分光光度法

# 9 原理

食品中的蛋白质在催化加热条件下被分解，分解产生的氨与硫酸结合生成硫酸铵，在 $\mathrm{pH}4.8$ 的乙酸钠-乙酸缓冲溶液中与乙酰丙酮和甲醛反应生成黄色的3,5-二乙酰-2,6-二甲基-1,4-二氢化吡啶化合物。在波长 $400\mathrm{nm}$ 下测定吸光度值，与标准曲线比较定量，结果乘以换算系数，即为蛋白质含量。

10.2.2 对硝基苯酚指示剂 $(1\mathrm{g / L})$ ：称取 $0.1\mathrm{g}$ 对硝基苯酚指示剂溶于 $20~\mathrm{mL}~95\%$ 乙醇中，加水稀释至 $100~\mathrm{mL}$ 。  
10.2.3 乙酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $5.8~\mathrm{mL}$ 乙酸，以水稀释至 $100~\mathrm{mL}$ 。  
10.2.4 乙酸钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $41\mathrm{g}$ 无水乙酸钠或 $68\mathrm{g}$ 乙酸钠，溶于水并稀释至 $500~\mathrm{mL}$ 。  
10.2.5 乙酸钠-乙酸缓冲溶液：量取 $60~\mathrm{mL}$ 乙酸钠溶液与 $40~\mathrm{mL}$ 乙酸溶液混合，该溶液 $\mathsf{pH}$ 为4.8。  
10.2.6 显色剂：分别量取 $15 \mathrm{~mL}$ 甲醛与 $7.8 \mathrm{~mL}$ 乙酰丙酮，混合后以水稀释至 $100 \mathrm{~mL}$ ，剧烈振摇混匀， $4^{\circ} \mathrm{C}$ 条件下于棕色瓶中可保存 $3 \mathrm{~d}$ 。

# 10.3 标准品

硫酸铵标准储备溶液（以氮计） $(1.0\mathrm{g / L})$ ，或经国家认证并授予标准物质证书的标准品。

# 10.4 标准溶液配制

10.4.1 硫酸铵标准储备溶液(以氮计) $(1.0\mathrm{g} / \mathrm{L})$ ：称取 $105^{\circ}\mathrm{C}$ 干燥 $2\mathrm{h}$ 的硫酸铵 $0.4720\mathrm{g}$ 加水溶解后稀释至 $100\mathrm{mL}$ ，混匀。此溶液每毫升相当于 $1.0\mathrm{mg}$ 氮。 $4^{\circ}\mathrm{C}$ 条件下可保存1个月。  
10.4.2 硫酸铵标准工作溶液 $(0.10\mathrm{mg / mL})$ ：准确吸取硫酸铵标准储备液 $(1.0\mathrm{g / L})10.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶内，以水稀释至刻度，混匀。此溶液每毫升相当于 $0.1\mathrm{mg}$ 氮。临用现配。

# 12.1 试样制备

同5.1.1。

# 12.2 水分测定

# 12.3 试样消解

称取固体试样 $0.1\mathrm{g}\sim 0.5\mathrm{g}$ 、半固体试样 $0.2\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 、液体试样 $1\mathrm{mL(g)}\sim 5\mathrm{mL}$ $(\mathrm{g})$ ，移入干燥的 $100~\mathrm{mL}$ 或 $250~\mathrm{mL}$ 定氮瓶中，加入 $0.1\mathrm{g}$ 硫酸铜、 $1\mathrm{g}$ 硫酸钾及 $5\mathrm{mL}$ 硫酸，轻摇后于瓶口放一小漏斗，将定氮瓶以 $45^{\circ}$ 角斜放置于有小孔的石棉网上。缓慢加热，待内容物全部炭化，泡沫完全停止后，加大火力，并保持瓶内液体微沸，至液体呈蓝绿色澄清透明后，再继续加热 $0.5\mathrm{h}\sim 1\mathrm{h}$ 。取下定氮瓶冷却至室温，小心加入 $20~\mathrm{mL}$ 水，将内容物全部转移至 $100~\mathrm{mL}$ 容量瓶中，并用少量水洗定氮瓶内壁，洗液并入容量瓶中，再加水至刻度，混匀备用。同时进行空白试验。

# 12.4 试样溶液的制备

量取 $2.0\mathrm{mL}\sim 5.0\mathrm{mL}$ （精确至 $0.1\mathrm{mL}$ 试样或试剂空白消化液于 $50~\mathrm{mL}$ 或 $100~\mathrm{mL}$ 容量瓶内，加1滴 $\sim 2$ 滴对硝基苯酚指示剂溶液，摇匀后滴加氢氧化钠溶液中和至黄色，再滴加乙酸溶液至溶液无色，以水稀释至刻度，混匀。

# 12.5 标准曲线的制作

吸取 $0.00\mathrm{mL}, 0.05\mathrm{mL}, 0.10\mathrm{mL}, 0.20\mathrm{mL}, 0.40\mathrm{mL}, 0.60\mathrm{mL}, 0.80\mathrm{mL}$ 和 $1.00\mathrm{mL}$ 硫酸铵标准使用溶液（相当于 $0.00\mu \mathrm{g}, 5.00\mu \mathrm{g}, 10.0\mu \mathrm{g}, 20.0\mu \mathrm{g}, 40.0\mu \mathrm{g}, 60.0\mu \mathrm{g}, 80.0\mu \mathrm{g}$ 和 $100.0\mu \mathrm{g}$ 氮），分别置于 $10\mathrm{mL}$ 比色管中。加 $4.0\mathrm{mL}$ 乙酸钠-乙酸缓冲溶液及 $4.0\mathrm{mL}$ 显色剂，以水稀释至刻度，混匀。置于 $100^{\circ}\mathrm{C}$ 水浴中加热 $15\mathrm{min}$ 。取出用水冷却至室温后，移入 $1\mathrm{cm}$ 比色杯内，以零管为参比，于波长 $400\mathrm{nm}$ 处测量吸光度值，根据标准曲线各点吸光度值绘制标准曲线或计算线性回归方程。

# 12.6 试样测定

吸取 $0.50\mathrm{mL}\sim 2.00\mathrm{mL}$ (相当于氮 $\leqslant 100~\mu \mathrm{g})$ 试样溶液和同量的试剂空白溶液，分别置于 $10~\mathrm{mL}$ 比色管中。加 $4.0\mathrm{mL}$ 乙酸钠-乙酸缓冲溶液及 $4.0\mathrm{mL}$ 显色剂，加水稀释至刻度，混匀。置于 $100^{\circ}C$ 水浴中加热 $15\mathrm{min}$ 。取出用水冷却至室温后，移入 $1\mathrm{cm}$ 比色杯内，以零管为参比，于波长 $400~\mathrm{nm}$ 处测量吸光度值，在标准曲线上查得待测液的浓度。

# 13 分析结果的表述

试样中蛋白质的含量按式(2)进行计算。

$$
X = \frac {\left(c - c _ {0}\right) \times V _ {1} \times V _ {3}}{m \times V _ {2} \times V _ {4} \times 1 0 0 0 \times 1 0 0 0} \times 1 0 0 \times F \dots \tag {2}
$$

式中：

$X$ ——试样中蛋白质的含量，单位为克每百克或克每百毫升 $(\mathrm{g} / 100\mathrm{g}$ 或 $\mathrm{g} / 100\mathrm{mL})$

$c$ ——试样测定液中氮的含量，单位为微克 $(\mu \mathrm{g})$ ；

$c_{0}$ ——试剂空白测定液中氮的含量，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——试样消化液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——试样溶液总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克或毫升（g或 $\mathrm{mL}$ ）；

$V_{2}$ ——制备试样溶液的消化液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{4}$ ——测定用试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

1000 由 $\mu \mathrm{g}$ 转化为 $\mathrm{mg}$ 的换算系数；

1000 由 $\mathrm{mg}$ 转化为 $\mathbf{g}$ 的换算系数；

100 ——由 $\mathrm{g / g}$ 转化为 $\mathrm{g / 100g}$ 的换算系数；

$F$ ——蛋白质折算系数（见附录C）；

蛋白质含量 $\geqslant 1\mathrm{g} / 100\mathrm{g}$ 或 $1\mathrm{g} / 100\mathrm{mL}$ 时，结果保留三位有效数字；蛋白质含量 $< 1\mathrm{g} / 100\mathrm{g}$ 或 $1\mathrm{g} / 100\mathrm{mL}$ 时，结果保留两位有效数字。

注1：分析结果以氮含量表述时，不需要乘蛋白质折算系数 $F$ 。分析结果以蛋白质含量表述时，应同时报告蛋白质折算系数。

注2：以干基计算试样中蛋白质的含量需根据试样的水分含量折算。

# 14 精密度

当样品中蛋白质含量 $\leqslant 10\mathrm{g} / 100\mathrm{g}$ 或 $10\mathrm{g} / 100\mathrm{mL}$ ，在重复条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

当样品中蛋白质含量 $>10\mathrm{g} / 100\mathrm{g}$ 或 $10\mathrm{g} / 100\mathrm{mL}$ ，在重复条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 15 其他

称样量为 $5.0\mathrm{g}$ 或 $5.0~\mathrm{mL}$ 时，本方法对氮的检出限为 $0.0001\mathrm{g} / 100\mathrm{g}$ 或 $0.0001\mathrm{g} / 100\mathrm{mL}$ 。

# 第三法 燃烧法

# 16 原理

试样在 $900^{\circ}\mathrm{C} \sim 1200^{\circ}\mathrm{C}$ 高温下燃烧，燃烧过程中产生混合气体，其中的碳、硫等干扰气体和盐类被吸收管吸收，氮氧化物被全部还原成氮气，形成的氮气气流通过热导检测器(TCD)进行检测。

# 17 试剂和材料

17.1 氧气 $(\mathrm{O}_2)$ ：纯度 $\geq 99.995\%$ 。  
17.2 载气 $(\mathrm{CO}_{2},\mathrm{He},\mathrm{Ar}$ 等）：纯度 $\geq 99.995\%$   
17.3 含氮标准物质：天冬氨酸 $\mathrm{(C_4H_7NO_4)}$ 、尿素 $\mathrm{(CH_4N_2O)}$ ，纯度 $\geqslant 99\%$ 。或经国家认证并授予标准物质证书的含氮标准物质。  
17.4 无氮铝箔纸、锡囊、无氮塑料囊或不锈钢坩埚。

# 18 仪器和设备

18.1 氮/蛋白质分析仪：配有热导检测器，测定范围 $0.1\mathrm{mg}\sim 200\mathrm{mg}$ 氮含量。具备校正功能。  
18.2 分析天平：感量为 $0.1\mathrm{mg}$   
18.3 匀浆机。  
18.4 粉碎机。

# 21 测定步骤

# 21.1 仪器工作条件

# 21.1.1 燃烧温度

$900^{\circ} \mathrm{C} \sim 1200^{\circ} \mathrm{C}$ 。

# 21.1.2 通氧量

使用前应根据不同仪器自行检查最佳通氧量。通氧量应控制在样品燃烧后，保留 $2\% \sim 8\%$ 残氧量。

# 21.1.3 通氧时间

根据样品量的不同以及样品燃烧难易程度不同调节通氧气时间，以保证样品完全燃烧。

# 21.1.4 仪器校正

开机后待仪器稳定后，利用优化好的仪器参数测定氮含量略高于待测样品的标准物质，进行3次重复测定得到日常校正系数 $(f)$ 。如日常校正系数大于1.1或小于0.9，或更换热导检测器后，应重新绘制校正曲线(见附录B)，待日常校正系数结果在 $0.9\sim 1.1$ 范围内后，再进行样品测定。

$$
f = \left(\frac {c}{c _ {1}} + \frac {c}{c _ {2}} + \frac {c}{c _ {3}}\right) \times \frac {1}{3} \tag {3}
$$

式中：

$f$ ——日常校正系数；

$c$ ——标准物质的理论氮含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$c_{1}, c_{2}, c_{3}$ ——标准物质三次重复测定的氮含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ 。

# 21.2 测定

根据仪器说明书要求称取 $0.1\mathrm{g}\sim 1.0\mathrm{g}$ 充分混匀的固体试样（精确至 $0.001\mathrm{g})$ ，或移取 $0.1\mathrm{mL}$ $(\mathrm{g})\sim 1.0~\mathrm{mL(g)}$ 充分混匀的液体试样，用锡箔、锡囊包裹或称量于不锈钢坩埚中，放置于样品盘上。试样经过充分燃烧，测其含量。

# 22 分析结果的表述

试样中蛋白质的含量按式(4)进行计算：

$$
X = c \times F \quad \dots \tag {4}
$$

式中：

$X$ ——试样中蛋白质的含量，单位为克每百克或克每百毫升 $(\mathrm{g} / 100\mathrm{g}$ 或 $\mathrm{g} / 100\mathrm{mL})$ ；

$c$ ——试样中氮的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$F$ ——蛋白质折算系数（见附录C）。

蛋白质含量 $\geqslant 1\mathrm{g} / 100\mathrm{g}$ 或 $1\mathrm{g} / 100\mathrm{mL}$ 时，结果保留三位有效数字；蛋白质含量 $< 1\mathrm{g} / 100\mathrm{g}$ 或 $1\mathrm{g} / 100\mathrm{mL}$ 时，结果保留两位有效数字。

注1：分析结果以氮含量表述时，不需要乘蛋白质折算系数 $F$ 。分析结果以蛋白质含量表述时，应同时报告蛋白质折算系数。

注2：以干基计算试样中蛋白质的含量需根据试样的水分含量折算。

# 23 精密度

# 前言

本标准代替GB8272—2009《食品添加剂蔗糖脂肪酸酯》。

本标准与GB8272—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 蔗糖脂肪酸酯”。

# 食品安全国家标准

# 食品添加剂 蔗糖脂肪酸酯

# 1 范围

本标准适用于以蔗糖和食用油脂或脂肪酸为主要原料经酯化并精制而成的食品添加剂蔗糖脂肪酸酯。

# 2 分子式

$$
\left(\mathrm {R C O O}\right) _ {n} \mathrm {C} _ {1 2} \mathrm {H} _ {1 2} \mathrm {O} _ {3} (\mathrm {O H}) _ {8 - n}
$$

R ——脂肪酸的烃基；

$n$ ——蔗糖的羟基酯化数。

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 氯化钠。  
A.2.1.2 无水硫酸钠。  
A.2.1.3 乙醚。  
A.2.1.4 盐酸溶液： $1 + 3$   
A.2.1.5 氢氧化钾-乙醇溶液。  
A.2.1.6 葱酮硫酸溶液： $2\mathrm{g} / \mathrm{L}$

# A.2.2 分析步骤

# A.2.2.1 试样处理

称取 $1\mathrm{g}$ 试样于 $250~\mathrm{mL}$ 锥形瓶中，加 $25~\mathrm{mL}$ 氢氧化钾-乙醇溶液，装上回流冷凝管，在水浴上加热微沸 $1\textrm{h}$ ，取下稍冷后加 $50~\mathrm{mL}$ 水，加热浓缩至约 $30~\mathrm{mL}$ ，加 $10~\mathrm{mL}$ 盐酸溶液，充分振摇，加入氯化钠使之成为饱和溶液，摇匀，移入分液漏斗中，每次用 $30~\mathrm{mL}$ 乙醚，萃取两次，将醚层与水层分离，待测。

# A.2.2.2 鉴别

醚层用 $20~\mathrm{mL}$ 氯化钠饱和溶液洗涤后，加 $2\mathrm{g}$ 无水硫酸钠脱水，再将醚层置于通风橱内的热水浴上蒸干，得白色柔软晶片。

取 $2\mathrm{mL}$ 水层于试管中，在水浴上加热赶尽乙醚，冷却后沿管壁加 $1\mathrm{mL}$ 葱酮硫酸溶液，应呈蓝-绿色。

# A.3 酸值（以KOH计)的测定

# A.3.1 方法提要

中和 $1\mathrm{g}$ 试样中游离的脂肪酸所需要的氢氧化钾的毫克数。

# A.3.2 试剂和材料

A.3.2.1 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.5\mathrm{mol / L}$   
A.3.2.2 中性热乙醇：取适量乙醇(体积分数 $95\%$ )，加热后加入1滴酚酞指示液，用 $0.5\mathrm{mol / L}$ 氢氧化

钠标准滴定溶液滴定至微红色，并保持 $30\mathrm{~s}$ 不褪色。

A.3.2.3 酚酞指示液： $10\mathrm{g / L}$

# A.3.3 分析步骤

称取约 $5\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ ），置于 $500\mathrm{mL}$ 锥形瓶中，加入 $75\mathrm{mL} \sim 100\mathrm{mL}$ 中性热乙醇使试样溶解，加 $0.5\mathrm{mL}$ 酚酞指示液，趁热边摇晃边用 $0.5\mathrm{mol/L}$ 氢氧化钠标准滴定溶液滴定至呈微红色，并维持 $30\mathrm{s}$ 不褪色为终点。

# A.3.4 结果计算

酸值(以KOH计)的质量分数 $\varpi_{1}$ ，单位为毫克每克 $(\mathrm{mg / g})$ ，按式(A.1)计算：

$$
w _ {1} = \frac {V _ {1} \times c _ {1} \times M}{m _ {1}} \quad \dots \tag {A.1}
$$

式中：

$V_{1}$ ——滴定时消耗的氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——氢氧化钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——氢氧化钾的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(\mathrm{KOH}) = 56.1]$

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以两次平行测定结果的算术平均值为准(保留一位小数)。在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# A.4 游离糖（以蔗糖计）的测定

# A.4.1 试剂和材料

A.4.1.1 正丁醇。

A.4.1.2 氯化钠溶液： $50~\mathrm{g / L}$

A.4.1.3 盐酸溶液： $6\mathrm{mol} / \mathrm{L}$ 。取 $540~\mathrm{mL}$ 盐酸，加水稀释至 $1000~\mathrm{mL}$ ，摇匀。

A.4.1.4 氢氧化钠溶液： $200\mathrm{g / L}$

A.4.1.5 葡萄糖标准溶液：称取 $1.000\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，经在 $98^{\circ}\mathrm{C} \sim 100^{\circ}\mathrm{C}$ 干燥至恒重的纯葡萄糖，加水溶解后，加入 $5\mathrm{mL}$ 盐酸，用水稀释至 $200\mathrm{mL}$ 。

A.4.1.6 费林溶液甲液。

A.4.1.7 费林溶液乙液。

A.4.1.8 甲基红指示液： $1\mathrm{g / L}$

A.4.1.9 亚甲基蓝指示液： $0.05\mathrm{g / L}$

# A.4.2 分析步骤

# A.4.2.1 试样处理

称取约 $2\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），置于锥形瓶中，加入 $40~\mathrm{mL}$ 正丁醇，在水浴上加热溶解。转入 $125~\mathrm{mL}$ 分液漏斗中，然后以 $60^{\circ}\mathrm{C} \sim 70^{\circ}\mathrm{C}$ 的氯化钠溶液每次 $10~\mathrm{mL}$ 萃取2次，分离（必要时离心），合并萃取液。加 $2.0~\mathrm{mL}~6~\mathrm{mol} / \mathrm{L}$ 盐酸溶液，在 $68^{\circ}\mathrm{C} \sim 70^{\circ}\mathrm{C}$ 水浴中加热 $15\mathrm{min}$ ，冷却后滴加甲基红指示液，用 $20\%$ 氢氧化钠溶液中和至中性，加水定容至 $50~\mathrm{mL}$ ，用干燥滤纸过滤，收集滤液供测定。

# A.4.2.2 费林溶液的标定

精密吸取费林溶液甲液、费林溶液乙液各 $5\mathrm{mL}$ ，加 $10~\mathrm{mL}$ 水，置于 $250~\mathrm{mL}$ 锥形瓶中，从滴定管中

滴加约 $9.5\mathrm{mL}$ 葡萄糖标准溶液，煮沸 $2\mathrm{min}$ ，加2滴亚甲基蓝指示液，继续滴加葡萄糖标准溶液至蓝色完全消失为终点。根据葡萄糖标准溶液消耗量计算 $10\mathrm{mL}$ 费林溶液相当的葡萄糖质量 $(m_3)$ 。

# A.4.2.3 测定

精确吸取费林溶液甲液、费林溶液乙液各 $5\mathrm{mL}$ ，准确加入 $15\mathrm{mL}$ 试样滤液（含糖量应在 $0.2\% \sim 0.5\%$ ），煮沸 $2\mathrm{min}$ ，加亚甲基蓝指示液，用葡萄糖标准溶液滴定至终点。用量为 $V_{2}$ （不得超过 $0.5\mathrm{mL} \sim 1.0\mathrm{mL}$ ，超过量应先在煮沸前加入）。

# A.4.3 结果计算

游离糖（以蔗糖计）的质量分数 $w_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m _ {3} - V _ {2} \times c _ {2}}{m _ {2} \times \frac {15}{50}} \times 0.95 \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$m_{3}$ ——费林溶液 $10\mathrm{mL}$ 相当的葡萄糖的质量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——滴定用葡萄糖标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{2}$ ——葡萄糖标准溶液的浓度，单位为克每毫升 $(\mathrm{g / mL})$ ；

$m_{2}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

15 ——测定时所取试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

50 ——试样处理后定容的体积，单位为毫升 $(\mathrm{mL})$ ；

0.95——葡萄糖换算为蔗糖的系数。

试验结果以两次平行测定结果的算术平均值为准(保留一位小数)。在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# A.5 灼烧残渣的测定

按GB/T7531规定的方法测定，称取约 $2\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，灼烧温度为 $850~^\circ \mathrm{C}\pm 25~^\circ \mathrm{C}$ 。

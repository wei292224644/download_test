# 前言

本标准代替GB/T23296.7—2009《食品接触材料塑料中表氯醇含量的测定高效液相色谱法》及SN/T2897—2011《出口食品接触材料金属材料表面涂层中环氧氯丙烷的测定气相色谱法》。

本标准与GB/T23296.7—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 环氧氯丙烷的测定和迁移量的测定”；  
——增加了适用范围；  
——修改了涂层的制样方法；  
——增加了食品接触材料及制品中环氧氯丙烷迁移量的检测方法。

# 食品安全国家标准

# 食品接触材料及制品环氧氯丙烷的测定和迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品(塑料、涂料、纸和纸板)中环氧氯丙烷的测定方法和迁移量的测定方法。

本标准适用于食品接触材料及制品(塑料、涂料、纸和纸板)中环氧氯丙烷的测定方法和环氧氯丙烷迁移量的测定方法，其中高效液相色谱法适用于食品接触材料及制品(塑料、涂料、纸制品)中环氧氯丙烷的测定，气相色谱-质谱法适用于食品接触材料及制品(塑料、涂料、纸和纸板)中环氧氯丙烷的迁移量的测定。

# 环氧氯丙烷的测定

# 2 原理

试样中环氧氯丙烷经1,4-二氧六环提取后，提取液用微蒸馏装置进行蒸馏，馏出液中的环氧氯丙烷经衍生后，其衍生物经过液相色谱分离，用荧光检测器测定环氧氯丙烷衍生物的含量，换算出试样中环氧氯丙烷的含量。采用外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。  
3.1.2 9,10-二甲氧基蒽-2-磺酸钠 $\mathrm{(C_{16}H_{13}NaO_5S)}$ ，即DAS-Na。  
3.1.3 1,4-二氧六环 $\mathrm{(C_4H_8O_2)}$ ：纯度大于 $99\%$ ，含水量应小于 $0.01\%$ 。  
3.1.4 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.5 强酸性阳离子交换树脂。  
3.1.6 氮气：纯度大于或等于 $99.9\%$   
3.1.7 冰块。

# 3.2 试剂配制

# 3.2.1 甲醇溶液（ $80\%$ ，体积分数）

量取 $400~\mathrm{mL}$ 甲醇于 $500~\mathrm{mL}$ 容量瓶中，用水定容。

# 3.2.2 9,10-二甲氧基蒽-2-磺酸试剂（DAS试剂）

用 $200\mathrm{mL}$ 甲醇溶液 $(80\%)$ ，体积分数)活化约 $50\mathrm{g}$ 预先处理好的强酸性阳离子交换树脂，然后用树脂填充内径为 $1\mathrm{cm}$ 的玻璃柱，直至树脂高度约 $20~\mathrm{cm}$ ，备用。称取 $100\mathrm{mg}$ 的DAS-Na于预先盛有 $20~\mathrm{mL}$ 甲醇的烧杯中，加热溶解，将溶液分2次～3次全部过强酸性阳离子交换树脂。用甲醇洗脱，使流出液流速约为 $0.5\mathrm{mL / min}$ ，收集 $\mathrm{pH}$ 为 $1\sim 2$ 时的洗脱液，直至 $\mathrm{pH}$ 开始上升。将所得洗脱液用氮吹仪吹干，制得DAS试剂。应避光保存，保存期为一年。

# 3.2.3 DAS-乙腈溶液 $(5\mathrm{mg / mL})$

称取 $20\mathrm{mg}$ (精确至 $0.1\mathrm{mg})$ DAS试剂溶解于 $4\mathrm{mL}$ 乙腈中。临用现配，避光保存。

# 3.3 标准品

环氧氯丙烷 $\mathrm{(C_3H_5ClO}$ ,CAS号：106-89-8)，纯度 $>99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

# 3.4.1 环氧氯丙烷标准储备液（ $1\mathrm{mg / mL}$ ）

称取 $100\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ) 环氧氯丙烷于 $100\mathrm{mL}$ 容量瓶中，用1,4-二氧六环定容。溶液于 $-20^{\circ}\mathrm{C} \sim 20^{\circ}\mathrm{C}$ 避光密封储存，有效期为3个月。

# 3.4.2 环氧氯丙烷标准使用液（ $40~\mu \mathrm{g / mL}$ ）

准确移取 $2.0\mathrm{mL}$ 的环氧氯丙烷标准储备液 $(1\mathrm{mg / mL})$ 至 $50~\mathrm{mL}$ 容量瓶中，用1，4-二氧六环定容。

# 3.4.3 环氧氯丙烷的标准中间溶液

分别移取 $0.0\mathrm{mL}, 0.5\mathrm{mL}, 1.0\mathrm{mL}, 2.0\mathrm{mL}, 3.0\mathrm{mL}, 4.0\mathrm{mL}, 5.0\mathrm{mL}$ 的环氧氯丙烷标准使用液 $(40~\mu \mathrm{g / mL})$ 于7个 $25~\mathrm{mL}$ 的容量瓶中，用1,4-二氧六环定容。标准中间液的浓度分别为 $0.0~\mu \mathrm{g / mL}$ 、 $0.8~\mu \mathrm{g / mL}, 1.6~\mu \mathrm{g / mL}, 3.2~\mu \mathrm{g / mL}, 4.8~\mu \mathrm{g / mL}, 6.4~\mu \mathrm{g / mL}, 8.0~\mu \mathrm{g / mL}$ 。

# 3.4.4 环氧氯丙烷标准工作溶液

分别移取 $10.0\mathrm{mL}1,4-$ 二氧六环于7个样品瓶中，再分别准确加入 $100~\mu \mathrm{L}$ 环氧氯丙烷标准中间溶液，加入磁力搅拌棒，用隔垫和铝盖密封。并将样品瓶标记为 $\mathrm{A_b}$ 。环氧氯丙烷标准工作溶液的浓度分别为： $0\mu \mathrm{g / L},8\mu \mathrm{g / L},16\mu \mathrm{g / L},32\mu \mathrm{g / L},48\mu \mathrm{g / L},64\mu \mathrm{g / L},80\mu \mathrm{g / L}$

# 4 仪器和设备

4.1 高效液相色谱仪：配有 $20~\mu \mathrm{L}$ 定量环和荧光检测器。  
4.2 分析天平：感量分别为 $1\mathrm{mg}$ 和 $0.1\mathrm{mg}$   
4.3 微蒸馏装置（如图A.1所示）。  
4.4 氮吹仪。  
4.5 样品瓶： $20~\mathrm{mL}$ ，配备铝盖和丁基橡胶隔垫或硅树脂橡胶隔垫，隔垫接触样品一面应涂有聚四氟乙烯。

注：如果没有合适样品瓶，可用顶空瓶代替。

4.6 移液器： $50~\mu \mathrm{L}, 100~\mu \mathrm{L}$ 。  
4.7 分析天平：感量 $0.1\mathrm{mg}$   
4.8 微孔滤膜： $0.22\mu \mathrm{m}$

注：使用的滤膜不应对环氧氯丙烷衍生物有吸附作用。

4.9 容量瓶。

# 5 分析步骤

# 5.1 试样制备

测试与食品接触的塑料制品及纸和纸板样品时，样品裁剪成若干个面积为 $2\mathrm{dm}^2$ 的小片，任取其中两片作平行试样。将 $2\mathrm{dm}^2$ 的小片切割成大小 $1\mathrm{cm}^2$ 的碎片，备用。对于涂层样品则轻轻刮下约 $0.2\mathrm{g}$ ，备用。

# 5.2 试样溶液的制备

# 5.2.1 塑料及纸制品试样提取液的制备

将备用待测的塑料或纸制品称量 $0.50\mathrm{g}$ （精确至 $0.1\mathrm{mg}$ )后，浸入 $50.0~\mathrm{mL}$ 的1,4-二氧六环中，密封。在室温下提取 $6\mathrm{h}$ 。移取 $10.0~\mathrm{mL}$ 提取液于 $20~\mathrm{mL}$ 样品瓶中，加入磁力搅拌棒，用隔垫和铝盖密封，并将顶空瓶标记为瓶 $\mathrm{A_s}$ 。

# 5.2.2 涂层提取液的制备

称取 $0.15\mathrm{g}$ (精确至 $0.1\mathrm{mg}$ ) 备用的涂层样品，浸入 $15.0\mathrm{mL}$ 的1,4-二氧六环中，密封。在室温下提取 $6\mathrm{h}$ 。移取 $10.0\mathrm{mL}$ 提取液于样品瓶中，加入磁力搅拌棒用隔垫和铝盖密封，并将样品瓶标记为瓶 $\mathrm{A}_{\mathrm{s}}$ 。

注1：本标准规定了塑料制品、纸和纸板制品及涂层制品中环氧氯丙烷含量的测定方法。预期也可用于其他任何含有环氧氯丙烷的食品接触材料及制品。  
注2：如果塑料包装试样或容器涂层较厚，应该提高温度或延长提取时间。整个提取过程应在密闭环境中进行以防止环氧氯丙烷逸出。

# 5.3 空白溶液的制备

移取 $10.0\mathrm{mL}$ 未与塑料、纸制品或涂层样品接触的1，4-二氧六环转入 $20~\mathrm{mL}$ 样品瓶中，加入磁力搅拌棒，用隔垫和铝盖密封，并将样品瓶标记为 $\mathrm{A}_0$ 。

# 5.4 1,4-二氧六环提取液的蒸馏

应用图A.1的微蒸馏装置，将1,4-二氧六环提取液进行蒸馏，收集部分馏出液。图A.1中瓶A为5.2.1、5.2.2、5.3和3.4.4中的样品瓶，其中分别含有 $10~\mathrm{mL}$ 的试样提取溶液、空白溶液和标准工作溶液。

另取 $20~\mathrm{mL}$ 样品瓶，于体积为 $3\mathrm{mL}$ 处进行标记后用隔热和铝盖密封。将密封后的样品瓶称量（精确至 $0.1\mathrm{mg}$ ），并标记为瓶B。

如图A.1所示，瓶A和瓶B通过外层套有聚四氟乙烯的不锈钢管连接，钢管刺入瓶A中的深度为 $1\mathrm{cm}\sim 2\mathrm{cm}$ ，刺入瓶B中的深度 $3\mathrm{cm}\sim 4\mathrm{cm}$ 。为避免管口堵塞，应在样品瓶的瓶塞上预先刺一个小孔，使钢管可迅速刺入瓶中。同时，在瓶B的盖子上插入注射器针头以避免瓶中压力过大。

将瓶A置于加热板上，将瓶B置于装有冰水混合物的烧杯中。加热瓶A，使其中1,4-二氧六环提取液沸腾，继续缓慢加热使瓶A中的1,4-二氧六环提取液蒸馏至瓶B中，直至瓶B中馏出液面达到 $3\mathrm{mL}$ 标记处。将瓶A从加热板上移开，移去瓶B中的注射器针头和不锈钢管。用滤纸仔细擦干瓶B。将瓶B重新称量(精确至 $0.1\mathrm{mg}$ )，根据两次称量的差值和1,4-二氧六环的密度，计算馏出液的体积。

馏出液可于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 密闭储存2周。

# 5.5 环氧氯丙烷的衍生

打开含有馏出液的瓶B铝盖，加入 $50~\mu \mathrm{L}$ DAS-乙腈溶液 $(5\mathrm{mg / mL})$ ，用隔垫和铝盖密封。将样品瓶置于 $75^{\circ}C$ 烘箱中 $17\mathrm{h}\pm 1\mathrm{h}$ 。然后将衍生溶液冷却至室温，并用 $0.22\mu \mathrm{m}$ 的微孔滤膜过滤，滤液待测。

环氧氯丙烷的DAS衍生物对光十分敏感，衍生应在避光条件下操作，衍生液可在室温下避光保存2d。

# 5.6 仪器参考条件

仪器参考条件列出如下：

a）色谱柱： $\mathrm{C_8}$ ，柱长 $250~\mathrm{mm}$ ，内径为 $4.6  \mathrm{mm}$ ，粒径为 $5\mu \mathrm{m}$ ，或相当者；  
b）流动相：乙腈和水，流动相梯度淋洗条件见表1；  
c）进样量： $2\mu \mathrm{L}$   
d) 流速: $2 \mathrm{~mL} / \mathrm{min}$ ;   
e） 柱温：室温；  
f) 荧光检测器：激发波长 $262\mathrm{nm}$ ，发射波长 $490\mathrm{nm}$ 。

# 5.7 标准曲线的制作

按照5.6的测定条件，将标准工作溶液按5.4和5.5处理后得到的衍生物溶液依次进样，测量环氧氯丙烷的DAS衍生物的峰面积(如果环氧氯丙烷的DAS衍生物的两个同分异构体出现两个峰，则将两个同分异构体的峰面积相加)。以标准工作溶液中环氧氯丙烷浓度为横坐标，以环氧氯丙烷衍生物的峰面积乘以系数K后的值为纵坐标，绘制标准曲线。环氧氯丙烷标准溶液衍生物的液相色谱图见附录B。

注：K为瓶B接收的馏出液体积除以瓶A中用于蒸馏的1,4-二氧六环提取液体积(10mL)。

# 5.8 试样的测定

将空白溶液和试样衍生物溶液依次进样测量。根据样品中环氧氯丙烷衍生物的峰面积乘以相应的

系数K后，由标准曲线得到1，4-二氧六环提取液中环氧氯丙烷的浓度。

# 6 分析结果的表述

试样中环氧氯丙烷的含量，按式(1)计算：

$$
X = \frac {\rho \times V}{m} \times 1 0 ^ {- 3} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中环氧氯丙烷的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——标准曲线得到的1,4-二氧六环提取液中环氧氯丙烷的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——用于试样提取的1,4-二氧六环溶液总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

方法的塑料制品、纸和纸板、涂层制品中环氧氯丙烷检出限为 $0.3\mathrm{mg / kg}$ ，定量限为 $0.8\mathrm{mg / kg}$ 。

# 环氧氯丙烷迁移量的测定（气相色谱-质谱法）

# 9 原理

水基和酸性两种食品模拟物中的环氧氯丙烷采用二氯甲烷萃取后气相色谱-质谱测定，外标法定量。酒精类和油基食品模拟物中的环氧氯丙烷直接用动态顶空-气相色谱-质谱法测定，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 10.1 试剂

10.1.1 配制水基、酸性、酒精类、油基食品模拟物所需试剂：依据GB31604.1的规定。

注：本方法所用油基食品模拟物为葵花籽油，使用其他油基模拟物应对方法进行优化。

10.1.2 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：重蒸馏。

10.1.3 氯化钠(NaCl)。

# 10.2 试剂配制

水基、酸性、酒精类、油基食品模拟物：按GB5009.156的规定配制。

# 10.3 标准品

环氧氯丙烷 $\mathrm{(C_3H_5ClO}$ ,CAS号：106-89-8)，纯度 $>99\%$ ，或经国家认证并授予标准物质证书的标准

物质。

# 10.4 标准溶液配制

# 10.4.1 水基、酸性和酒精类食品模拟物用环氧氯丙烷标准储备液（ $1000\mathrm{mg / L}$ ）

准确称取环氧氯丙烷标准物质 $50\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ), 用二氯甲烷定量转移至 $50\mathrm{mL}$ 容量瓶中并稀释至刻度。于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 下密封保存。

# 10.4.2 水基、酸性和酒精类食品模拟物的环氧氯丙烷标准中间液(50 mg/L)

准确移取环氧氯丙烷标准储备液 $(1000\mathrm{mg / L})5\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用二氯甲烷稀释至刻度。于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 下密封保存。

# 10.4.3 水基、酸性和酒精类食品模拟物的环氧氯丙烷标准中间液（ $10\mathrm{mg / L}$ ）

准确移取环氧氯丙烷标准储备液 $(1000\mathrm{mg / L})1\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用二氯甲烷稀释至刻度。于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 下密封保存。

# 10.4.4 水基、酸性和酒精类食品模拟物的环氧氯丙烷标准中间液( $1\mathrm{mg / L}$ ）

准确移取环氧氯丙烷标准中间液 $(10\mathrm{mg / L})5\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，用二氯甲烷稀释至刻度。于 $0^{\circ}C\sim 4^{\circ}C$ 下密封保存。

# 10.4.5 水基、酸性食品模拟物的环氧氯丙烷系列标准工作液

分别在5个 $50~\mathrm{mL}$ 容量瓶中依次加入水基、酸性和酒精类食品模拟物的环氧氯丙烷标准中间液 $(1\mathrm{mg / L})2.5\mathrm{mL},5.0\mathrm{mL}$ ，加入水基、酸性和酒精类食品模拟物的环氧氯丙烷标准中间液（ $10\mathrm{mg / L})$ $1.0  \mathrm{mL},2.5  \mathrm{mL},5.0  \mathrm{mL}$ ，用相应的食品模拟物稀释至刻度。该系列标准工作液中环氧氯丙烷的浓度分别为 $0.05  \mathrm{mg / L},0.1  \mathrm{mg / L},0.2  \mathrm{mg / L},0.5  \mathrm{mg / L},1.0  \mathrm{mg / L}$ 。

# 10.4.6 酒精类食品模拟物的环氧氯丙烷系列标准中间液

# 12.2 试液制备

# 12.2.1 水基和酸性食品模拟物试液制备

准确移取迁移试验得到的水基和酸性食品模拟物 $100~\mathrm{mL}$ ，放入 $250~\mathrm{mL}$ 分液漏斗中，加入 $5\mathrm{g}$ 氯化钠，振摇溶解，准确加入 $5.0~\mathrm{mL}$ 二氯甲烷，振摇 $1\mathrm{min}$ 。静置分层，滤纸卷成小条，擦干分液漏斗颈管内的水珠，放出下层二氯甲烷萃取液，于离心管中，盖塞。按照上述步骤，再次准确加入二氯甲烷 $5\mathrm{mL}$ ，萃取模拟物，合并萃取液于同一离心管中，记录萃取液体积为 $V_{i}$ ，备用。

# 13 分析结果的表述

# 13.1 食品模拟物试液中环氧氯丙烷浓度的计算

# 13.1.1 水基和酸性食品模拟物试液中环氧氯丙烷浓度的计算

由标准曲线求得的试样溶液中环氧氯丙烷的浓度 $\rho$ 代入式(2)得出结果即为水基和酸性食品模拟物试液中环氧氯丙烷的浓度。

$$
C = \frac {\rho \times V _ {i}}{V} \quad \dots \tag {2}
$$

式中：

$C$ ——水基和酸性食品模拟物试液中环氧氯丙烷的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho$ ——测试液中环氧氯丙烷的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V_{i}$ ——萃取液最终体积，单位为毫升 $(\mathrm{mL})$

$V$ ——萃取用食品模拟物总体积，固定值为100，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留两位有效数字。

# 13.1.2 酒精类和油基食品模拟物试液中环氧氯丙烷浓度的计算

由标准曲线求得的试样溶液中环氧氯丙烷的浓度，即为酒精类和油基食品模拟物试液中环氧氯丙烷的浓度，上机前试液经过稀释需再乘上稀释倍数。酒精类食品模拟物试液中环氧氯丙烷的含量，单位为毫克每升 $(\mathrm{mg / L})$ ，油基食品模拟物试液中环氧氯丙烷的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 。

# 13.2 环氧氯丙烷迁移量的计算

由13.1.1和13.1.2得到的食品模拟物试液中环氧氯丙烷浓度，按GB5009.156进行迁移量的计算，得到食品接触材料及制品中环氧氯丙烷的迁移量。计算结果保留两位有效数字。

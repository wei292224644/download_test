# 食品中三甲胺的测定

2016-08-31 发布

2017-03-01实施

# 前言

本标准代替GB/T5009.179—2003《火腿中三甲胺氮的测定》。

本标准与GB/T5009.179—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中三甲胺的测定”；  
——将原有分光光度法修改为第一法 顶空气相色谱-质谱联用法、第二法 顶空气相色谱法；  
——标准的适用范围扩充为水产动物及其制品和肉与肉制品；  
——将三甲胺氮改成三甲胺。

# 食品安全国家标准

# 食品中三甲胺的测定

# 1 范围

本标准规定了水产动物及其制品和肉与肉制品中三甲胺的测定方法。

本标准适用于水产动物及其制品和肉与肉制品中三甲胺的测定。

# 第一法 顶空气相色谱-质谱联用法

# 2 原理

试样经 $5\%$ 三氯乙酸溶液提取，提取液置于密封的顶空瓶中，在碱液作用下三甲胺盐酸盐转化为三甲胺，在 $40^{\circ}\mathrm{C}$ 经过 $40\mathrm{min}$ 的平衡，三甲胺在气液两相中达到动态的平衡，吸取顶空瓶内气体注入气相色谱-质谱联用仪进行检测，以保留时间 $(RT)$ 、辅助定性离子 $(m / z59$ 和 $m / z42)$ 和定量离子 $(m / z58)$ 进行定性，以外标法进行定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 氢氧化钠(NaOH)。  
3.1.2 三氯乙酸 $(\mathrm{C}_2\mathrm{HCl}_3\mathrm{O}_2)$ 。

# 3.2 试剂配制

3.2.1 $50\%$ 氢氧化钠溶液：称取 $100\mathrm{g}$ 氢氧化钠，溶于 $20^{\circ}\mathrm{C}\sim 30^{\circ}\mathrm{C}$ 的 $100~\mathrm{mL}$ 水中。  
3.2.2 $5\%$ 三氯乙酸溶液：称取 $25\mathrm{g}$ 三氯乙酸溶于水中，并定容为 $500~\mathrm{mL}$

# 3.3 标准品

三甲胺盐酸盐(CAS:593-81-7)，分子式： $(\mathrm{CH}_3)_3\mathrm{NHCl}$ ，纯度 $\geq 98\%$ ，置于干燥器中，在 $4^{\circ}\mathrm{C}$ 条件下保存。

# 3.4 标准溶液配制

3.4.1 三甲胺标准储备液：称取三甲胺盐酸盐标准品 $0.0162\mathrm{g}$ ，用 $5\%$ 三氯乙酸溶液溶解并定容至 $100\mathrm{mL}$ ，等同浓度为 $100\mu \mathrm{g} / \mathrm{mL}$ 的三甲胺标准储备液，在 $4^{\circ}\mathrm{C}$ 条件下保存。  
3.4.2 三甲胺标准使用溶液：吸取一定体积的三甲胺标准储备液用 $5\%$ 三氯乙酸溶液逐级稀释成浓度分别为 $1.0~\mu \mathrm{g / mL},2.0~\mu \mathrm{g / mL},5.0~\mu \mathrm{g / mL},10.0~\mu \mathrm{g / mL},20.0~\mu \mathrm{g / mL},40.0~\mu \mathrm{g / mL}$ 的三甲胺标准使用溶液。

# 4 仪器和设备

4.1 气相色谱-质谱联用仪，配有分流/不分流进样口和电子轰击电离源（EI源）。  
4.2 天平：感量分别为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.3 恒温水浴锅：控温精度 $\pm 2^{\circ}C$   
4.4 顶空瓶：容积 $20~\mathrm{mL}$ ，配有聚四氟乙烯硅橡胶垫和密封帽，使用前在 $120^{\circ}\mathrm{C}$ 烘烤 $2\mathrm{h}$ 。  
4.5 微量注射器： $1\mathrm{mL}$   
4.6 医用塑料注射器： $5\mathrm{mL}$   
4.7 均质机。  
4.8 绞肉机。  
4.9 低速离心机。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 试样预处理与保存

对于畜禽肉类及其肉制品，去除脂肪和皮，对于鱼和虾等动物水产及其制品，需要去鳞或去皮，所有样品取肌肉部分约 $100\mathrm{g}$ ，用绞肉机绞碎或用刀切细，混匀。制备好的试样若不立即测定，应密封在聚乙烯塑料袋中并于 $-18^{\circ}\mathrm{C}$ 冷冻保存，测定前于室温下放置解冻即可。

# 5.1.2 试样提取

称取约 $10\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )制备好的样品于 $50~\mathrm{mL}$ 的塑料离心管中，加入 $20~\mathrm{mL}~5\%$ 三氯乙酸溶液，用均质机均质 $1\mathrm{min}$ ，以 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，在玻璃漏斗加上少许脱脂棉，将上清液滤入 $50~\mathrm{mL}$ 容量瓶，残留物再分别用 $15~\mathrm{mL}$ 和 $10~\mathrm{mL}~5\%$ 三氯乙酸溶液重复上述提取过程两次，合并滤液并用 $5\%$ 三氯乙酸溶液定容至 $50~\mathrm{mL}$ 。

# 5.1.3 提取液顶空处理

准确吸取提取液 $2.0\mathrm{mL}$ 于 $20~\mathrm{mL}$ 顶空瓶中，压盖密封，用医用塑料注射器准确注入 $5.0\mathrm{mL}50\%$ 氢氧化钠溶液，备用。

# 5.1.4 标准溶液顶空处理

分别取3.4.2中各标准使用液 $2.0\mathrm{mL}$ 至 $20~\mathrm{mL}$ 顶空瓶中，压盖密封，用医用塑料注射器分别准确注入 $5.0~\mathrm{mL}50\%$ 氢氧化钠溶液，备用。

# 5.2 仪器参考条件

# 5.2.1 色谱条件

色谱条件如下：

a）石英毛细管色谱柱： $30\mathrm{m}$ （长） $\times 0.25\mathrm{mm}$ （内径） $\times 0.25\mu \mathrm{m}$ （膜厚），固定相为聚乙二醇，或其他等效的色谱柱；  
b）载气：高纯氦气；  
c) 流量 $1.0 \mathrm{~mL} / \mathrm{min}$ ; 进样口温度 $220^{\circ} \mathrm{C}$ ;

d）分流比： $10:1$   
e）升温程序： $40^{\circ}\mathrm{C}$ 保持 $3\mathrm{min}$ ，以 $30^{\circ}\mathrm{C} / \mathrm{min}$ 速率升至 $220^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ 。

# 5.2.2 质谱条件

质谱条件如下：

a）离子源：电子轰击电离源（EI源），温度： $220^{\circ}\mathrm{C}$   
b）离子化能量： $70\mathrm{eV}$   
c）传输线温度： $230~\mathrm{^\circ C}$   
d) 溶剂延迟：1.5 min；  
e）扫描方式：选择离子扫描（SIM）。

# 5.3 测定

5.3.1 顶空进样：将制备好的试样在 $40^{\circ}\mathrm{C}$ 平衡 $40\mathrm{min}$ 。在5.2色谱质谱条件下，用进样针抽取顶空瓶内液上气体 $100~\mu \mathrm{L}$ ，注入GC-MS中进行测定。  
5.3.2 定性测定：以选择离子方式采集数据，以试样溶液中三甲胺的保留时间（RT）、辅助定性离子 $(m / z 59$ 和 $m / z 42)$ 、定量离子 $(m / z 58)$ 以及辅助定性离子与定量离子的峰度比 $(Q)$ 与标准溶液的进行比较定性。试样溶液中三甲胺的辅助定性离子和定量离子峰度比 $(Q_{\text{样品}})$ 与标准溶液中三甲胺的辅助定性离子和定量离子峰度比 $(Q_{\text{标准}})$ 的相对偏差控制在 $\pm 15\%$ 以内。  
5.3.3 定量测定：采用外标法定量。以标准溶液中三甲胺的峰面积为纵坐标，以标准溶液中三甲胺的浓度为横坐标，绘制校准曲线，用校准曲线计算试样溶液中三甲胺的浓度。

# 6 分析结果的表述

6.1 试样中的三甲胺的含量按式(1)计算：

$$
X _ {1} = \frac {c \times V}{m} \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——试样中三甲胺含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——从校准曲线得到的三甲胺浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

6.2 试样中三甲胺氮的含量按式(2)计算：

$$
X _ {2} = \frac {X _ {1} \times 1 4 . 0 1}{5 9 . 1 1} \tag {2}
$$

式中：

$X_{2}$ ——试样中三甲胺氮的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$X_{1}$ ——试样中三甲胺含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

14.01——氮的相对原子质量；

59.11——三甲胺的相对分子质量。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当称样量为 $10\mathrm{g}$ ，定容体积为 $50~\mathrm{mL}$ ，顶空进样体积为 $100\mu \mathrm{L}$ ，方法的检出限为 $1.5\mathrm{mg / kg}$ ，定量限为 $5.0\mathrm{mg / kg}$ 。

# 第二法 顶空气相色谱法

# 9 原理

# 12 分析步骤

# 12.2 仪器参考条件

仪器参考条件如下：

a）石英毛细管色谱柱： $30\mathrm{m}$ （长） $\times 0.32\mathrm{mm}$ （内径） $\times 0.5\mu \mathrm{m}$ （膜厚），固定相为聚乙二醇，或其他等效的色谱柱；  
b）载气：高纯氮气；  
c) 流量： $2.5\mathrm{mL} / \mathrm{min}$   
d）进样口温度： $220^{\circ}C$   
e) 分流比： $2:1$

f）升温程序： $40^{\circ}\mathrm{C}$ 保持 $3\mathrm{min}$ ，以 $30^{\circ}\mathrm{C / min}$ 速率升至 $220^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ；

g）检测器温度： $220^{\circ}C$   
h）尾吹气（氮气)流量： $35~\mathrm{mL / min}$   
i） 氢气流量： $40~\mathrm{mL / min}$   
j）空气流量： $400~\mathrm{mL / min}$

# 12.3 测定

12.3.1 顶空进样：将制备好的试样在 $40^{\circ}\mathrm{C}$ 平衡 $40\mathrm{min}$ 。在上述色谱质谱条件下，用进样针抽取顶空瓶内液上气体 $250~\mu \mathrm{L}$ ，注入GC-FID中进行测定。  
12.3.2 定性定量：根据标准色谱图中三甲胺的保留时间进行定性分析。采用外标法进行定量分析，以标准峰面积为纵坐标，以标准溶液浓度为横坐标，绘制校准曲线，用校准曲线计算试样溶液中三甲胺的浓度。

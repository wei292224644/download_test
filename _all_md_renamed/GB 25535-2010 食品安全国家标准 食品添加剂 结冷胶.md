# 食品安全国家标准

# 食品添加剂 结冷胶

# 1 范围

本标准适用于由伊乐假单胞菌（Pseudomonas elodea）对碳水化合物进行纯种培养发酵后，经加工制成的食品添加剂结冷胶。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

# 3 分子结构和相对分子质量

# 3.1 分子结构

由一分子葡萄糖醛酸、一分子鼠李糖和两分子葡萄糖组成的基本单元重复聚合组成。

# 3.2 相对分子质量

$4 \times 10^{5} \sim 6 \times 10^{5}$ （按2007年国际相对原子质量）

# 附录A

# （规范性附录）

# 检验方法

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682—2008中规定的水。分析中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。本试验所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 溶解性试验

溶于水中，形成粘稠溶液；不溶于乙醇。

# A.2.2 钙离子凝胶试验

在 $99\mathrm{mL}$ 水中溶解 $1\mathrm{g}$ 样品，配制成 $1\%$ 溶液。用一个机械搅拌器和一个螺旋桨形的搅拌叶，搅拌该溶液 $2\mathrm{h}$ （留部分溶液在A.2.3时用），用宽孔吸管取少量溶液，将其移至 $\mathrm{CaCl}_2$ 溶液（ $100\mathrm{g/L}$ ）中，立即形成一种凝胶。

# A.2.3 钠离子凝胶试验

向A.2.2中配制的 $1\%$ 样品溶液中加入 $0.5 \mathrm{~g} \mathrm{NaCl}$ ，加热该溶液至 $80^{\circ} \mathrm{C}$ ，不停地搅拌并在 $80^{\circ} \mathrm{C}$ 条件下保持 $1 \mathrm{~min}$ ，停止加热及搅拌，冷却至室温，形成坚硬的凝胶。

# A.3 结冷胶的测定

# A.3.1 试剂和材料

a) 硅藻土：色谱纯。  
b) 无水乙醇。  
c) 乙醇溶液： $78 + 22$ 。

# A.3.2 仪器和设备

a) 玻璃过滤器。  
b) 干燥器（ $180\mathrm{mm}$ ）。

# A.3.3 分析步骤

称取约 $1.0 \mathrm{~g}$ 色谱纯的硅藻土, 置于一个玻璃过滤器中, 均匀铺平, $105^{\circ} \mathrm{C}$ 下干燥 $5 \mathrm{~h}$ , 在干燥器内冷却后准确称量。称取约 $0.2 \mathrm{~g}$ 干燥后（ $105^{\circ} \mathrm{C}, 2.5 \mathrm{~h}$ ）的试样（精确至 $0.001 \mathrm{~g}$ ），加 $50 \mathrm{~mL}$ 水，在约 $80^{\circ} \mathrm{C}$ 水浴锅中搅拌溶解 $30 \mathrm{~min}$ 。加入 $200 \mathrm{~mL}$ 预先加热至 $60^{\circ} \mathrm{C} \sim 70^{\circ} \mathrm{C}$ 的无水乙醇，混合均匀，静置 $12 \mathrm{~h}$ , 用上述装有硅藻土的玻璃过滤器过滤, 然后用乙醇溶液洗涤滤渣 5 次, 前 3 次每次洗涤用量 $20 \mathrm{~mL}$ , 后 2 次每次洗涤用量 $10 \mathrm{~mL}$ 。在 $105^{\circ} \mathrm{C}$ 下将滤渣干燥 5h, 在干燥器内冷却后准确称量。

# A.3.4 结果计算

结冷胶的含量 $X_{1}$ 按公式（A.1）计算：

$$
X _ {1} = \frac {m _ {1}}{m _ {0}} \times 100 \% \dots \tag{A.1}
$$

式中：

$X_{1}$ ——结冷胶的含量， $\%$ ；

$m_{1}$ ——残渣的质量，单位为克（g）；

$m_{0}$ ——试样的质量，单位为克（g）。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $2\%$

# 附录B

# （资料性附录）

# 异丙醇的测定

# B.1 试剂和材料

a) 异丙醇：色谱纯。  
b) 叔丁醇：色谱纯。

# B.2 仪器和设备

a) 气相色谱仪：配有氢火焰电离检测器。  
b) 色谱柱: $1.8\mathrm{m} \times 2.3\mathrm{mm}$ (内径) 不锈钢柱或等同功效柱, 内填充 $0.2\mathrm{mm} \sim 0.15\mathrm{mm}$ 的 Porapak QS 填料或其他等同物。

# B.3 参考色谱条件

a) 载气：氦气。  
b) 流速: $80 \mathrm{~mL} / \mathrm{min}$ 。  
c) 进样口温度: $200^{\circ} \mathrm{C}$ 。  
e) 柱温： $165^{\circ}\mathrm{C}$ 。  
f)检出器温度： $200^{\circ}\mathrm{C}$ 。  
g) 进样量: $5 \mu \mathrm{L}$ 。  
h) 异丙醇的保留时间约为 $2\mathrm{min}$ ，叔丁醇的保留时间约为 $3\mathrm{min}$ 。

# B. 4 分析步骤

# B.4.1 标准溶液制备

# B.4.1.1 异丙醇（IPA）标准溶液

准确称取 $500.0\mathrm{mg}$ 异丙醇，转移至一个 $50\mathrm{mL}$ 容量瓶中，加水稀释并定容至刻度，混匀。吸取 $10\mathrm{mL}$ 该溶液，移入一个 $100\mathrm{mL}$ 容量瓶中，加水稀释并定容至刻度，混匀。

# B.4.1.2 叔丁醇（TBA）标准溶液

准确称取 $500.0 \mathrm{mg}$ 叔丁醇，转移至一个 $50 \mathrm{~mL}$ 容量瓶中，加水稀释并定容至刻度，混匀。吸取 $10 \mathrm{~mL}$ 该溶液，移入一个 $100 \mathrm{~mL}$ 容量瓶中，加水稀释并定容至刻度，混匀。

# B.4.1.3 混合标准溶液

分别吸取 $4 \mathrm{~mL}$ 异丙醇（IPA）和叔丁醇（TBA）标准溶液，转移至一个 $125 \mathrm{~mL}$ 锥形瓶中，加水稀释至约 $100 \mathrm{~mL}$ ，混匀。每毫升该溶液约含有 $40 \mu \mathrm{g}$ 异丙醇和 $40 \mu \mathrm{g}$ 叔丁醇。

# B.4.2 试样液制备

在盛有 $200 \mathrm{~mL}$ 水的 $1000 \mathrm{~mL}$ 圆底蒸馏瓶中, 加入 $1 \mathrm{~mL}$ 合适的消泡剂（如Dow-Corning G-10或等同物）, 使其分散。加入准确称取的约 $5 \mathrm{~g}$ 结冷胶试样, 摇振 $1 \mathrm{~h}$ 。该瓶上连接一个分馏柱, 调节温度,使泡沫无法进入该柱, 接馏出液约 $100 \mathrm{~mL}$ 。在馏出液中加入 $4.0 \mathrm{~mL}$ 叔丁醇 (TBA) 标准溶液, 制得试样液。

# B.4.3 测定

注入 $5 \mu \mathrm{L}$ 混合标准溶液，在参考色谱条件下进行测定，记录异丙醇的峰面积值为 $A_{\mathrm{IPA}}$ ，叔丁醇的峰面积值为 $A_{\mathrm{TBA}}$ ，计算出相对校正因子 $f(f = A_{\mathrm{IPA}} / A_{\mathrm{TBA}})$ 。

同样的，注入 $5\mu \mathrm{L}$ 试样液，在参考色谱条件下进行测定，记录异丙醇的峰面积值为 $\mathbf{a}_{\mathrm{IPA}}$ ，叔丁醇的峰面积值为 $\mathbf{a}_{\mathrm{TBA}}$ 。

# B.5 结果计算

异丙醇的含量 $X$ 按公式（B.1）计算：

$$
X = \frac {a _ {I P A} \times 4 0 0 0}{f \times a _ {T B A} \times m} \dots \tag {B.1}
$$

式中：

$X$ ——异丙醇的含量，单位为毫克每千克（ $\mathrm{mg/kg}$ ）；

$a_{IPA}$ ——试样液中异丙醇色谱峰面积的平均值；

$f$ ——相对校正因子（ $f = A_{IPA} / A_{TBA}$ ）；

$a_{TBA}$ ——试样液中叔丁醇色谱峰面积的平均值；

$m$ ——称取的试样质量，单位为克（g）。

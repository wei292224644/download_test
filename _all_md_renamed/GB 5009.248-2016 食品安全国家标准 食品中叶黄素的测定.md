# 食品安全国家标准

# 食品中叶黄素的测定

2016-08-31 发布

2017-03-01实施

# 前言

本标准代替GB/T23209—2008《奶粉中叶黄素的测定 液相色谱-紫外检测法》。

本标准与GB/T23209—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中叶黄素的测定”；  
扩大适用范围，增加在冷冻饮品、米面制品、焙烤食品、果酱、果冻和饮料中叶黄素的液相色谱测定方法；  
——对原标准的原理进行了修改，把原用丙酮为溶剂提取改为用乙醚-正己烷-环已烷溶剂体系进行提取；  
——增加操作过程的注意点；  
——增加了对包括奶粉在内的脂肪含量高的样品的皂化步骤，并提供了不同前处理方法以适应各种样品基质分析需要；  
——增加了对叶黄素标准溶液浓度的校正要求；  
——对于在试验操作过程中叶黄素可能产生异构化的现象，增加了对因异构化生成的顺式叶黄素的定性与定量要求。

# 食品安全国家标准

# 食品中叶黄素的测定

# 1 范围

本标准规定了食品中叶黄素的液相色谱测定方法。

本标准适用于婴幼儿配方奶粉、乳品、冷冻饮品、米面制品、焙烤食品、果酱、果冻和饮料中叶黄素的液相色谱测定。

# 2 原理

脂肪含量高（脂肪含量以干基计不低于 $3\%$ )的食品经氢氧化钾溶液室温皂化使叶黄素游离后，再以乙醚-正己烷-环己烷 $(40 + 40 + 20$ ，体积比)提取，液相色谱法分离，紫外检测器或二极管阵列检测器检测，外标法定量。

其他食品直接以乙醚-正己烷-环己烷 $(40 + 40 + 20$ ，体积比)提取样品中叶黄素。提取液经中性氧化铝固相萃取小柱净化后，液相色谱法分离，紫外检测器或二极管阵列检测器检测，外标法定量。

样品在提取与分析过程中，反式结构的叶黄素可能发生异构化，转化为顺式叶黄素。对于转化产生的顺式叶黄素，可通过保留时间定性、峰面积加合定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 环己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{12}\right)$ ：色谱纯。  
3.1.2 乙醚 $\left[(\mathrm{C}_2\mathrm{H}_5)_2\mathrm{O}\right]$ ：色谱纯。  
3.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
3.1.4 无水乙醇 $\mathrm{(C_2H_5OH)}$ ：色谱纯。  
3.1.5 甲基叔丁基醚 $\left[\mathrm{CH}_{3} \mathrm{OCC}\left(\mathrm{CH}_{3}\right)_{3}, \mathrm{MTBE}\right]$ ：色谱纯。  
3.1.6 二丁基羟基甲苯 $\mathrm{(C_{15}H_{24}O,BHT)}$ 。  
3.1.7 氢氧化钾(KOH)。  
3.1.8 碘 $(\mathrm{I}_2)$ 。  
3.2 试剂配制  
3.2.1 $10\%$ 氢氧化钾溶液：称取 $10\mathrm{g}$ 氢氧化钾(3.1.7)，加水溶解稀释至 $100~\mathrm{mL}$   
3.2.2 $20\%$ 氢氧化钾溶液：称取 $20\mathrm{g}$ 氢氧化钾(3.1.7)，加水溶解稀释至 $100~\mathrm{mL}$   
3.2.3 萃取溶剂：称取 $1\mathrm{g}$ BHT(3.1.6)，以 $200~\mathrm{mL}$ 环己烷(3.1.1)溶解，加入 $400~\mathrm{mL}$ 乙醚(3.1.2)和 $400~\mathrm{mL}$ 正己烷(3.1.3)，混匀。  
3.2.4 $0.1\%$ BHT乙醇溶液：称取 $0.1\mathrm{g}$ BHT(3.1.6)，以 $100~\mathrm{mL}$ 乙醇(3.1.4)溶解，混匀。  
3.2.5 碘的乙醇溶液：称取 $1\mathrm{mg}$ 碘(3.1.8)，加乙醇(3.1.4)溶解稀释至 $1\mathrm{L}$ 。

# 3.3 标准品

叶黄素(CAS号：127-40-2)，纯度不低于 $98.0\%$

# 3.4 标准溶液配制

3.4.1 标准储备液 $(50\mu \mathrm{g / mL})$ ：准确称取 $5\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ )叶黄素(3.1.3)，以 $0.1\%$ BHT乙醇溶液(3.2.4)溶解并定容至 $100~\mathrm{mL}$ 。该标准储备液充氮避光置于一 $20^{\circ}C$ 或以下的冰箱中可保存六个月。

注：叶黄素标准储备液使用前需校正，具体操作见附录A。

3.4.2 标准工作液：从叶黄素标准贮备液(3.4.1)中准确移取 $0.050\mathrm{mL}, 0.100\mathrm{mL}, 0.200\mathrm{mL}, 0.400\mathrm{mL}$ 、 $1.00\mathrm{mL}$ 溶液入5个 $25\mathrm{mL}$ 棕色容量瓶中，用 $0.1\%$ BHT乙醇溶液(3.2.4)定容至刻度，得到浓度为 $0.100\mu \mathrm{g} / \mathrm{mL}, 0.200\mu \mathrm{g} / \mathrm{mL}, 0.400\mu \mathrm{g} / \mathrm{mL}, 0.800\mu \mathrm{g} / \mathrm{mL}, 2.00\mu \mathrm{g} / \mathrm{mL}$ 的系列标准工作液。标准工作液充氮避光置于 $-20^{\circ}\mathrm{C}$ 或以下的冰箱中可保存一个月。

3.5 中性氧化铝固相萃取小柱， $500\mathrm{mg} / 3\mathrm{mL}$ ，使用前以 $5\mathrm{mL}$ 萃取溶剂(3.2.3)淋洗，保持柱体湿润。  
3.6 $0.45\mu \mathrm{m}$ 滤膜，有机系。

# 4 仪器和设备

4.1 液相色谱仪，带二极管阵列检测器或紫外检测器。  
4.2 紫外可见分光光度计。  
4.3 分析天平：感量 $0.01\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。  
4.4 组织捣碎机。  
4.5 旋涡振荡器。  
4.6 振荡器。  
4.7 减压浓缩装置。  
4.8 固相萃取装置。  
4.9 离心机：转速不低于 $4500\mathrm{r / min}$

# 5 分析步骤

注：由于叶黄素对光敏感，除非另行说明，所有试验操作应在无 $500\mathrm{nm}$ 以下紫外光的黄色光源或红色光源环境中进行。

# 5.1 试样制备

将一定数量的样品按要求经过粉碎、均质、缩分后，储存于样品瓶中。制备好的试样应充氮密封后置于 $-20^{\circ}\mathrm{C}$ 或以下的冰箱中保存。

# 5.2 提取

# 5.2.1 脂肪含量高的食品（如婴幼儿配方奶粉、乳粉、冰淇淋、焙烤坚果类食品等）

准确称取 $2\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样于 $50~\mathrm{mL}$ 聚丙烯离心管中，加入约 $0.2\mathrm{g}$ BHT(3.1.6)和 $10~\mathrm{mL}$ 乙醇(3.1.4)，混匀，加入 $10~\mathrm{mL}~10\%$ 氢氧化钾溶液(3.2.1)，涡旋振荡 $1\mathrm{min}$ 混匀，室温避光振荡皂化 $30\mathrm{min}$ ，以 $10~\mathrm{mL}$ 萃取溶剂(3.2.3)避光涡旋振荡提取 $3\mathrm{min},4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，重复提取2次，合并提取液，以 $10~\mathrm{mL}$ 水洗涤， $4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ 分层，重复洗涤1次，合并有机相于室温减压浓缩至近干，以 $0.1\%$ BHT乙醇溶液(3.2.4)涡旋振荡溶解残渣并定容至 $5\mathrm{mL}$ ，过 $0.45\mu \mathrm{m}$ 滤膜

# （3.6），供液相色谱测定。

液态奶：准确称取 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )样品于 $50~\mathrm{mL}$ 聚丙烯离心管中，加入约 $0.2\mathrm{g}$ BHT(3.1.6)和 $10~\mathrm{mL}$ 乙醇(3.1.4)，混匀，加入 $2\mathrm{mL}20\%$ 的氢氧化钾溶液(3.2.2)，涡动 $1\mathrm{min}$ 混匀，室温避光振荡皂化 $30\mathrm{min}$ ，以 $10~\mathrm{mL}$ 萃取溶剂(3.2.3)避光涡旋振荡提取 $3\mathrm{min},4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，重复提取2次，合并提取液，以 $10~\mathrm{mL}$ 水洗涤， $4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ 分层，重复洗涤1次，合并有机相于室温减压浓缩至近干，以 $0.1\% \mathrm{BHT}$ 乙醇溶液(3.2.4)涡旋振荡溶解残渣并定容至 $25~\mathrm{mL}$ ，过 $0.45\mu \mathrm{m}$ 滤膜(3.6)，供液相色谱测定。

# 5.2.2 其他食品（如米、面制品、果酱等）

准确称取 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )均匀样品置于 $50~\mathrm{mL}$ 聚丙烯离心管中，以 $10~\mathrm{mL}$ 萃取溶剂(3.2.3)避光涡旋振荡提取 $3\mathrm{min},4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，重复提取2次，合并提取液，于室温减压浓缩至近干，以 $3\mathrm{mL}$ 萃取溶剂(3.2.3)涡旋振荡溶解，重复操作1次，合并萃取溶剂，混匀，待净化。

将上述溶液以约 $1\mathrm{mL} / \mathrm{min}$ 的流速过已活化的中性氧化铝固相萃取小柱(3.5)，用 $3\mathrm{mL}$ 萃取溶剂(3.2.3)洗脱，合并流出液与洗脱液，于室温减压浓缩至近干，以 $0.1\%$ BHT乙醇溶液(3.2.4)涡旋振荡溶解残渣并定容至 $10~\mathrm{mL}$ ，过 $0.45\mu \mathrm{m}$ 滤膜(3.6)，供液相色谱测定。

# 5.3 仪器参考条件

5.3.1 色谱柱：C30色谱柱， $5\mu \mathrm{m}$ ， $250~\mathrm{mm}\times 4.6~\mathrm{mm}$ (内径)或相当者；  
5.3.2 柱温： $30^{\circ}C$   
5.3.3 流动相：甲醇/水 $(88 + 12$ ，体积比，含 $0.1\%$ BHT)-甲基叔丁基醚(含 $0.1\%$ BHT)，梯度洗脱， $0\mathrm{min}\sim 18\mathrm{min}$ ，甲醇/水由 $100\%$ 变换至 $10\%$ ； $18.1\mathrm{min}$ ，甲醇/水由 $10\%$ 变换至 $100\%$ ，保留 $10\mathrm{min}$ 。  
5.3.4 流速： $1.0~\mathrm{mL / min}$   
5.3.5 检测波长： $445~\mathrm{nm}$   
5.3.6 进样量： $50~\mu \mathrm{L}$

# 5.4 标准曲线的制作

将标准系列工作液分别注入液相色谱中，测定相应的峰面积，以标准工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。

# 5.5 试样溶液的测定

待测样液中叶黄素的响应值应在仪器线性响应范围内，否则应适当稀释或浓缩。标准工作液与待测样液等体积进样。根据标准溶液色谱峰的保留时间和峰面积，对试样溶液的色谱峰根据保留时间进行定性（待测样品中化合物色谱峰的保留时间与标准溶液相比变化范围应在 $\pm 2.5\%$ 之内），外标法定量。平行测定次数不少于两次。叶黄素标准溶液液相色谱图参见图B.1。

注：由于在样品的提取与分析过程中，温度、光照等原因均可使反式结构的叶黄素发生异构化，转化为顺式叶黄素。可按以下步骤获得顺式叶黄素：以乙醇为溶剂，配制 $800\mu \mathrm{g} / \mathrm{L}$ 的叶黄素标准溶液 $50~\mathrm{mL}$ ，加入 $2\mathrm{mL}$ 碘的乙醇溶液(3.2.5)，摇匀，混合液在日光或日光灯下放置 $30\mathrm{min}$ 。可获得顺式结构的叶黄素。由此制备的含顺式结构的叶黄素在检测时可作为对照品。经光碘异构化的反式叶黄素标准溶液色谱图参见图B.2。

# 6 分析结果的表述

试样中叶黄素含量按式(1)计算：

$$
X = \frac {c \times V}{m} \times \frac {1}{F} \times 1 0 0 \tag {1}
$$

式中：

$X$ ——试样中叶黄素的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ ；

$c$ ——由标准曲线而得的样液中标准品的含量，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样品最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——称样量，单位为克(g)；

$F$ ——校正系数。可通过以下方式获得：用液相色谱分析试样溶液，将顺式与反式叶黄素色谱峰面积加合作为总峰面积，其中反式叶黄素峰面积除以总峰面积所得值为校正系数。

以重复性条件下获得的两次独立测定结果的算术平均值表示，计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

本方法的检出限：婴幼儿配方奶粉、乳粉、冰淇淋、焙烤坚果类等食品，当取样量为 $2\mathrm{g}$ 、定容体积为 $5\mathrm{mL}$ 时，检出限为 $3\mu \mathrm{g} / 100\mathrm{g}$ ；液态奶等，当取样量为 $10\mathrm{g}$ 、定容体积为 $25~\mathrm{mL}$ 时，检出限为 $3\mu \mathrm{g} / 100\mathrm{g}$ 米、面制品、果酱等食品，当取样量为 $5\mathrm{g}$ 、定容体积为 $10~\mathrm{mL}$ 时，检出限为 $3\mu \mathrm{g} / 100\mathrm{g}$ 。

本方法的定量限：婴幼儿配方奶粉、乳粉、冰淇淋、焙烤坚果类等食品，当取样量为 $2\mathrm{g}$ 、定容体积为 $5\mathrm{mL}$ 时，定量限为 $10~\mu \mathrm{g} / 100\mathrm{g}$ ；液态奶等当取样量为 $10\mathrm{g}$ 、定容体积为 $25~\mathrm{mL}$ 时，定量限为 $10~\mu \mathrm{g} / 100~\mathrm{g}$ 米、面制品、果酱等食品，当取样量为 $5\mathrm{g}$ 、定容体积为 $10~\mathrm{mL}$ 时，定量限为 $10~\mu \mathrm{g} / 100~\mathrm{g}$ 。

# 标准溶液浓度校正方法

叶黄素标准溶液配制后需要校准。取 $1\mathrm{mL}$ 标准贮备液，以乙醇定容至 $25~\mathrm{mL}$ 。移取该溶液至 $1\mathrm{cm}$ 的石英比色皿中，以乙醇为空白，以分光光度计在 $445~\mathrm{nm}$ 波长下测定吸光值 $A$ 。按式(A.1)计算标准溶液浓度。

$$
c = \frac {A}{E _ {1 \mathrm {cm}} ^ {1 \%}} \times 2500 \times F \quad \dots \tag{A.1}
$$

式中：

$c$ ——标准溶液浓度， $\mu \mathrm{g / mL}$

$A$ ——标准溶液的吸光值；

2500——转换系数；

$E_{1\mathrm{cm}}^{1\%}$ ——乙醇中叶黄素的吸光系数，为2550；

$F$ ——校正系数，可按下述方式获得：用液相色谱分析校准后的标准溶液，将顺式与反式叶黄素色谱峰面积加合作为总峰面积，其中反式叶黄素峰面积除以总峰面积所得值为校正系数。

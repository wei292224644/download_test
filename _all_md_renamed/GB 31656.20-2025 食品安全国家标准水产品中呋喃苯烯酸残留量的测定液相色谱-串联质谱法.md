# 水产品中呋喃苯烯酸残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了水产品中呋喃苯烯酸(Nifurstylenate)残留量的制样和液相色谱-串联质谱测定方法。

本文件适用于鱼、虾、蟹及鳖等水产品可食组织中呋喃苯烯酸残留量的测定，其他水产品的检测可参照执行。

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中残留的呋喃苯烯酸，用甲酸水溶液分散，以叔丁基羟基茴香醚作为保护剂，用乙腈提取，经亚铁氰化钾和乙酸锌沉淀蛋白，正已烷和 $\mathrm{C}_{16}$ 柱净化，液相色谱、串联质谱仪测定，外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂和材料

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.2 正己烷 $\mathrm{(C_6H_{14})}$ ：色谱纯。

5.1.3 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.4 乙酸铵 $\mathrm{(CH_3COONH_2)}$ ：色谱纯。

5.1.5 甲酸（HCOOH）：色谱纯。

5.1.6 氯化钠 $(\mathrm{NaCl})$ ：优级纯。

5.1.7 叔丁基羟基茴香醚 $(\mathrm{C_{11}H_{16}O_2})$ ：纯度大于 $99\%$

5.1.8 亚铁氰化钾 $\left[\mathrm{K}_{4} \mathrm{Fe}(\mathrm{CN})_{6} \cdot 3 \mathrm{H}_{2} \mathrm{O}\right]$ 。

5.1.9 乙酸锌 $(\mathrm{C_4H_6ZnO_4}\cdot 2\mathrm{H_2O})$

5.1.10 氨水 $(\mathrm{NH_4OH})$ ： $25\% \sim 28\%$

# 5.2 溶液配制

5.2.1 $50\%$ 乙腈水溶液：取乙腈 $500~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$   
5.2.2 $1\%$ 甲酸水溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$   
5.2.3 亚铁氰化钾溶液：称取亚铁氰化钾 $10.6\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
5.2.4 乙酸锌溶液：称取乙酸锌 $22.0\mathrm{g}$ ，加冰乙酸 $3\mathrm{mL}$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。

5.2.5 $50\%$ 甲醇水溶液：取甲醇 $50~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ 。  
5.2.6 0.1%甲酸溶液(含 $5\mathrm{mmol} / \mathrm{L}$ 乙酸铵)溶液：称取乙酸铵 $0.385\mathrm{g}$ 溶解于 $800~\mathrm{mL}$ 水中，再加入甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。  
5.2.7 $20\mathrm{mg / mL}$ 叔丁基羟基茴香醚溶液：称取叔丁基羟基茴香醚 $200\mathrm{mg}$ ，用乙腈溶解并稀释至 $10\mathrm{mL}$ 。

# 5.3 标准品

呋喃苯烯酸钠(SodiumNifurstylenate, $\mathrm{C_{13}H_8NNaO_5}$ ,CAS号：54992-23-3)，含量 $\geqslant 99\%$ ，或相当者。

# 5.4 标准溶液配制

5.4.1 标准储备液 $(1.0\mathrm{mg / mL})$ ：取呋喃苯烯酸钠标准品 $10\mathrm{mg}$ ，精密称定，用甲醇溶解并定容至 $10~\mathrm{mL}$ 棕色容量瓶中，配制成浓度为 $1.0\mathrm{mg / mL}$ 的标准储备液，一 $18^{\circ}C$ 避光保存，有效期6个月。  
5.4.2 标准中间液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：准确移取标准储备液 $1.0\mathrm{mL}$ ，用甲醇稀释并定容至 $100~\mathrm{mL}$ 棕色容量瓶中，配制成浓度为 $10~\mu \mathrm{g} / \mathrm{mL}$ 的标准中间液，一 $18^{\circ}C$ 避光保存，有效期6个月。  
5.4.3 标准工作液 $(0.1\mu \mathrm{g / mL})$ ：准确移取适量 $10~\mu \mathrm{g / mL}$ 标准中间液，用甲醇稀释配制成 $0.1\mu \mathrm{g / mL}$ 标准工作液，现用现配。

# 5.5 材料

5.5.1 $\mathrm{C}_{18}$ 固相萃取柱： $1000\mathrm{mg / 6mL}$   
5.5.2 $50~\mathrm{mL}$ 聚丙烯离心管。  
5.5.3 $50~\mathrm{mL}$ 梨形瓶。  
5.5.4 聚四氟乙烯微孔滤膜: $0.22 \mu \mathrm{m}$ 。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$ 。  
6.3 离心机：转速不低于 $4000\mathrm{r / min}$ 。  
6.4 超声波清洗器。  
6.5 涡旋混合器。  
6.6 旋转蒸发仪。  
6.7 固相萃取装置。

# 7 试样的制备与保存

# 7.1 试样的制备

按GB/T30891—2014附录B的要求制样。

a）取均质的供试样品，作为供试试样；  
b) 取均质的空白样品, 作为空白试样;  
c) 取均质的空白样品，添加适宜浓度的标准溶液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下避光保存。

# 8 测定步骤

# 8.1 提取

称取试料 $(4\pm 0.05)\mathrm{g}$ ，于 $50~\mathrm{mL}$ 具塞离心管中，加入 $1 \%$ 甲酸水溶液 $5\mathrm{mL}$ ，涡旋混合 $30~s$ ，加入乙腈 $20.0~\mathrm{mL},20~\mathrm{mg / mL}$ 叔丁基羟基茴香醚 $200~\mu \mathrm{L}$ ，涡旋混合 $1\mathrm{min}$ ，超声提取 $30\mathrm{min}$ 。加入亚铁氰化钾溶液 $2.5\mathrm{mL}$ 、乙酸锌溶液 $2.5\mathrm{mL}$ 、氯化钠 $10\mathrm{g}$ ，振荡提取 $3\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液待净化。

# 8.2 净化

准确移取8.1上清液 $10.0\mathrm{mL}$ 于 $50~\mathrm{mL}$ 离心管中，再加入水 $10~\mathrm{mL}$ 正己烷 $6\mathrm{mL}$ ，振荡 $1\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去上层正己烷相，下层清液备用。 $\mathrm{C_{18}}$ 固相萃取柱依次用 $10~\mathrm{mL}$ 乙腈和 $10~\mathrm{mL}$ 水活化，取备用液全部过 $\mathrm{C_{18}}$ 固相萃取柱，收集流出液于 $50~\mathrm{mL}$ 离心管中。在 $\mathrm{C_{18}}$ 固相萃取柱中再加入 $50\%$ 乙腈水溶液 $10~\mathrm{mL}$ ，收集并合并流出液。在上述离心管中加入乙腈 $10~\mathrm{mL}$ 、氯化钠 $10\mathrm{g}$ ，振荡提取 $3\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液于 $50~\mathrm{mL}$ 梨形瓶中，加入 $200\mu \mathrm{L}$ 氨水， $45^{\circ}C$ 下减压浓缩至干，用 $50\%$ 甲醇水溶液定容至 $2.0~\mathrm{mL}$ ，过 $0.22\mu \mathrm{m}$ 聚四氟乙烯微孔滤膜，置于棕色进样瓶中，供液相色谱-串联质谱仪测定。

# 8.3 标准曲线的制备

准确移取适量标准工作液，用 $50\%$ 甲醇水溶液稀释，使呋喃苯烯酸的浓度分别为 $0.5 \mathrm{ng} / \mathrm{mL}$ 、 $1.0 \mathrm{ng} / \mathrm{mL}$ 、 $2.0 \mathrm{ng} / \mathrm{mL}$ 、 $5.0 \mathrm{ng} / \mathrm{mL}$ 、 $10.0 \mathrm{ng} / \mathrm{mL}$ 和 $20.0 \mathrm{ng} / \mathrm{mL}$ ，供液相色谱-串联质谱仪测定。

# 8.4 测定

b） 柱温： $35^{\circ}C$   
c） 进样量： $3\mu \mathrm{L}$   
d) 流速: $0.4 \mathrm{~mL} / \mathrm{min}$ ;   
e) 流动相：A 为 $0.1\%$ 甲酸溶液（含 $5 \mathrm{mmol} / \mathrm{L}$ 乙酸铵），B 为乙腈，梯度洗脱程序见表 1。

# 8.5 测定法

# 8.5.1 定性测定

在相同测试条件下，试料溶液中呋喃苯烯酸与校正标准溶液中呋喃苯烯酸的保留时间偏差在 $\pm 0.1\mathrm{min}$ 以内；且检测到的相对离子丰度，应与浓度相当的校正标准溶液相对离子丰度一致。其允许偏差为 $\pm 40\%$

# 8.5.2 定量测定

取试料溶液和相应的标准溶液，作单点或多点校准，按外标法以峰面积定量，标准溶液及试料溶液中的呋喃苯烯酸响应值均应在仪器检测的线性范围内。在上述液相色谱-质谱条件下，呋喃苯烯酸标准溶液的特征离子质量色谱图见附录A。

# 8.6 空白试验

取空白试样，除不加药物外，采用完全相同的操作步骤进行平行操作。

# 9 结果计算和表述

试样中呋喃苯烯酸的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A _ {i} \times C _ {s} \times V _ {1} \times V _ {3}}{A _ {s} \times V _ {2} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中呋喃苯烯酸残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A_{i}$ ——试料溶液中呋喃苯烯酸的峰面积；

$C_{s}$ ——标准溶液中呋喃苯烯酸浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V_{1}$ ——待净化液总体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——最终试样定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准溶液中呋喃苯烯酸的峰面积；

$V_{2}$ ——吸取出用于净化的净化液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、正确度和精密度

# 10.1 灵敏度

本方法的检出限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1.0~\mu \mathrm{g / kg}$

# 10.2 正确度

本方法在 $1.0\mu \mathrm{g / kg}\sim 10~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

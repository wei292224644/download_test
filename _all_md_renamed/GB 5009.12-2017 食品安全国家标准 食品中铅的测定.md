# 前言

本标准代替GB5009.12—2010《食品安全国家标准 食品中铅的测定》、GB/T20380.3—2006《淀粉及其制品 重金属含量 第3部分：电热原子吸收光谱法测定铅含量》、GB/T23870—2009《蜂胶中铅的测定 微波消解一石墨炉原子吸收分光光度法》、GB/T18932.12—2002《蜂蜜中钾、钠、钙、镁、锌、铁、铜、锰、铬、铅、镉含量的测定方法 原子吸收光谱法》、NY/T1100—2006《稻米中铅、镉的测定 石墨炉原子吸收光谱法》，SN/T2211—2008《蜂皇浆中铅和镉的测定 石墨炉原子吸收光谱法》中铅的测定方法。

本标准与GB5009.12—2010相比，主要变化如下：

在前处理方法中，保留湿法消解和压力罐消解，删除干法灰化和过硫酸铵灰化法，增加微波消解；  
——保留石墨炉原子吸收光谱法为第一法，采用磷酸二氢铵-硝酸钯溶液作为基体改进剂；保留火焰原子吸收光谱法为第三法；保留二硫腙比色法为第四法；  
——增加电感耦合等离子体质谱法作为第二法；  
——删除氢化物原子荧光光谱法、单扫描极谱法；  
——增加了微波消解升温程序、石墨炉原子吸收光谱法和火焰原子吸收光谱法的仪器参考条件为附录。

# 食品安全国家标准

# 食品中铅的测定

# 1 范围

本标准规定了食品中铅含量测定的石墨炉原子吸收光谱法、电感耦合等离子体质谱法、火焰原子吸收光谱法和二硫腙比色法。

本标准适用于各类食品中铅含量的测定。

# 第一法 石墨炉原子吸收光谱法

# 2 原理

试样消解处理后，经石墨炉原子化，在 $283.3\mathrm{nm}$ 处测定吸光度。在一定浓度范围内铅的吸光度值与铅含量成正比，与标准系列比较定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 硝酸 $\mathrm{(HNO_3)}$ 。  
3.1.2 高氯酸 $\mathrm{(HClO_4)}$   
3.1.3 磷酸二氢铵 $(\mathrm{NH_4H_2PO_4})$ 。  
3.1.4 硝酸钯 $\left[\mathrm{Pd}\left(\mathrm{NO}_{3}\right)_{2}\right]$

# 3.2 试剂配制

3.2.1 硝酸溶液 $(5 + 95)$ ：量取 $50~\mathrm{mL}$ 硝酸，缓慢加入到 $950~\mathrm{mL}$ 水中，混匀。  
3.2.2 硝酸溶液 $(1 + 9)$ ：量取 $50~\mathrm{mL}$ 硝酸，缓慢加入到 $450~\mathrm{mL}$ 水中，混匀。  
3.2.3 磷酸二氢铵-硝酸钯溶液：称取 $0.02\mathrm{g}$ 硝酸钯，加少量硝酸溶液 $(1 + 9)$ 溶解后，再加入 $2\mathrm{g}$ 磷酸二氢铵，溶解后用硝酸溶液 $(5 + 95)$ 定容至 $100~\mathrm{mL}$ ，混匀。

# 3.3 标准品

硝酸铅 $\left[\mathrm{Pb}\left(\mathrm{NO}_{3}\right)_{2},\mathrm{CAS}\right.$ 号：10099-74-8]：纯度 $>99.99\%$ 。或经国家认证并授予标准物质证书的一定浓度的铅标准溶液。

# 3.4 标准溶液配制

3.4.1 铅标准储备液 $(1000\mathrm{mg / L})$ ：准确称取 $1.5985\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )硝酸铅，用少量硝酸溶液 $(1 + 9)$ 溶解，移入 $1000~\mathrm{mL}$ 容量瓶，加水至刻度，混匀。

3.4.2 铅标准中间液 $(1.00\mathrm{mg / L})$ ：准确吸取铅标准储备液 $(1000\mathrm{mg / L})1.00\mathrm{mL}$ 于 $1000~\mathrm{mL}$ 容量瓶中，加硝酸溶液 $(5 + 95)$ 至刻度，混匀。  
3.4.3 铅标准系列溶液：分别吸取铅标准中间液 $(1.00\mathrm{mg / L})0\mathrm{mL},0.500\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},3.00\mathrm{mL}$ 和 $4.00~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加硝酸溶液(5+95)至刻度，混匀。此铅标准系列溶液的质量浓度分别为 $0\mu \mathrm{g} / \mathrm{L},5.00\mu \mathrm{g} / \mathrm{L},10.0\mu \mathrm{g} / \mathrm{L},20.0\mu \mathrm{g} / \mathrm{L},30.0\mu \mathrm{g} / \mathrm{L}$ 和 $40.0\mu \mathrm{g} / \mathrm{L}$ 。

注：可根据仪器的灵敏度及样品中铅的实际含量确定标准系列溶液中铅的质量浓度。

# 4 仪器和设备

注：所有玻璃器皿及聚四氟乙烯消解内罐均需硝酸溶液 $(1 + 5)$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

4.1 原子吸收光谱仪：配石墨炉原子化器，附铅空心阴极灯。  
4.2 分析天平：感量 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.3 可调式电热炉。  
4.4 可调式电热板。  
4.5 微波消解系统：配聚四氟乙烯消解内罐。  
4.6 恒温干燥箱。  
4.7 压力消解罐：配聚四氟乙烯消解内罐。

# 5 分析步骤

# 5.1 试样制备

注：在采样和试样制备过程中，应避免试样污染。

# 5.1.1 粮食、豆类样品

样品去除杂物后，粉碎，储于塑料瓶中。

# 5.1.2 蔬菜、水果、鱼类、肉类等样品

样品用水洗净，晾干，取可食部分，制成匀浆，储于塑料瓶中。

# 5.1.3 饮料、酒、醋、酱油、食用植物油、液态乳等液体样品

将样品摇匀。

# 5.2 试样前处理

# 5.2.1 湿法消解

称取固体试样 $0.2\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.500~\mathrm{mL}\sim 5.00~\mathrm{mL}$ 于带刻度消化管中，加入 $10~\mathrm{mL}$ 硝酸和 $0.5~\mathrm{mL}$ 高氯酸，在可调式电热炉上消解（参考条件： $120^{\circ}C / 0.5h\sim 1h$ ；升至 $180^{\circ}C / 2h\sim 4h$ 、升至 $200^{\circ}C\sim 220^{\circ}C)$ 。若消化液呈棕褐色，再加少量硝酸，消解至冒白烟，消化液呈无色透明或略带黄色，取出消化管，冷却后用水定容至 $10~\mathrm{mL}$ ，混匀备用。同时做试剂空白试验。亦可采用锥形瓶，于可调式电热板上，按上述操作方法进行湿法消解。

# 5.2.2 微波消解

称取固体试样 $0.2\mathrm{g}\sim 0.8\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 或准确移取液体试样 $0.500~\mathrm{mL}\sim 3.00~\mathrm{mL}$ 于微波消解

罐中，加入 $5\mathrm{mL}$ 硝酸，按照微波消解的操作步骤消解试样，消解条件参考附录A。冷却后取出消解罐，在电热板上于 $140~^\circ \mathrm{C}\sim 160~^\circ \mathrm{C}$ 赶酸至 $1\mathrm{mL}$ 左右。消解罐放冷后，将消化液转移至 $10~\mathrm{mL}$ 容量瓶中，用少量水洗涤消解罐2次～3次，合并洗涤液于容量瓶中并用水定容至刻度，混匀备用。同时做试剂空白试验。

# 5.2.3 压力罐消解

称取固体试样 $0.2\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.500~\mathrm{mL}\sim 5.00~\mathrm{mL}$ 于消解内罐中，加入 $5\mathrm{mL}$ 硝酸。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱，于 $140^{\circ}C\sim 160^{\circ}C$ 下保持 $4\mathrm{h}\sim 5\mathrm{h}$ 。冷却后缓慢旋松外罐，取出消解内罐，放在可调式电热板上于 $140^{\circ}C\sim 160^{\circ}C$ 赶酸至 $1\mathrm{mL}$ 左右。冷却后将消化液转移至 $10~\mathrm{mL}$ 容量瓶中，用少量水洗涤内罐和内盖2次～3次，合并洗涤液于容量瓶中并用水定容至刻度，混匀备用。同时做试剂空白试验。

# 5.3 测定

# 5.3.1 仪器参考条件

根据各自仪器性能调至最佳状态。参考条件见附录B。

# 5.3.2 标准曲线的制作

按质量浓度由低到高的顺序分别将 $10\mu \mathrm{L}$ 铅标准系列溶液和 $5\mu \mathrm{L}$ 磷酸二氢铵-硝酸钯溶液（可根据所使用的仪器确定最佳进样量）同时注入石墨炉，原子化后测其吸光度值，以质量浓度为横坐标，吸光度值为纵坐标，制作标准曲线。

# 5.3.3 试样溶液的测定

在与测定标准溶液相同的实验条件下，将 $10~\mu \mathrm{L}$ 空白溶液或试样溶液与 $5\mu \mathrm{L}$ 磷酸二氢铵-硝酸钯溶液(可根据所使用的仪器确定最佳进样量)同时注入石墨炉，原子化后测其吸光度值，与标准系列比较定量。

# 6 分析结果的表述

试样中铅的含量按式(1)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中铅的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$\rho$ ——试样溶液中铅的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白溶液中铅的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样消化液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样称样量或移取体积，单位为克或毫升(g或 $\mathrm{mL}$ )；

1000——换算系数。

当铅含量 $\geqslant 1.00\mathrm{mg / kg}$ （或 $\mathrm{mg / L}$ )时，计算结果保留三位有效数字；当铅含量 $<  1.00~\mathrm{mg / kg}$ （或 $\mathrm{mg / L})$ 时，计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当称样量为 $0.5\mathrm{g}$ (或 $0.5\mathrm{mL}$ )，定容体积为 $10\mathrm{mL}$ 时，方法的检出限为 $0.02\mathrm{mg/kg}$ （或 $0.02\mathrm{mg/L}$ ，定量限为 $0.04\mathrm{mg/kg}$ 或 $0.04\mathrm{mg/L}$ 。

# 第三法 火焰原子吸收光谱法

# 9 原理

试样经处理后，铅离子在一定 $\mathrm{pH}$ 条件下与二乙基二硫代氨基甲酸钠（DDTC)形成络合物，经4-甲基-2-戊酮(MIBK)萃取分离，导入原子吸收光谱仪中，经火焰原子化，在 $283.3\mathrm{nm}$ 处测定的吸光度。在一定浓度范围内铅的吸光度值与铅含量成正比，与标准系列比较定量。

10.1.6 二乙基二硫代氨基甲酸钠[DDTC, $(\mathrm{C}_2\mathrm{H}_5)_2\mathrm{NCSSNa}\cdot 3\mathrm{H}_2\mathrm{O}]$

10.2.4 柠檬酸铵溶液 $(250\mathrm{g / L})$ ：称取 $25\mathrm{g}$ 柠檬酸铵，用水溶解并稀释至 $100~\mathrm{mL}$ ，混匀。  
10.2.5 溴百里酚蓝水溶液 $(1\mathrm{g} / \mathrm{L})$ ：称取 $0.1\mathrm{g}$ 溴百里酚蓝，用水溶解并稀释至 $100~\mathrm{mL}$ ，混匀。

# 12.1 试样制备

同5.1。

# 12.2 试样前处理

同5.2.1

# 12.3 测定

# 12.3.1 仪器参考条件

根据各自仪器性能调至最佳状态。参考条件参见附录C。

# 12.3.2 标准曲线的制作

分别吸取铅标准使用液 $0\mathrm{mL}, 0.250\mathrm{mL}, 0.500\mathrm{mL}, 1.00\mathrm{mL}, 1.50\mathrm{mL}$ 和 $2.00\mathrm{mL}$ （相当 $0\mu \mathrm{g}$ 、 $2.50\mu \mathrm{g}, 5.00\mu \mathrm{g}, 10.0\mu \mathrm{g}, 15.0\mu \mathrm{g}$ 和 $20.0\mu \mathrm{g}$ 铅）于 $125\mathrm{mL}$ 分液漏斗中，补加水至 $60\mathrm{mL}$ 。加 $2\mathrm{mL}$ 柠檬酸铵溶液（ $250\mathrm{g/L}$ ），溴百里酚蓝水溶液（ $1\mathrm{g/L}$ ）3滴～5滴，用氨水溶液（ $1+1$ ）调 $\mathrm{pH}$ 至溶液由黄变蓝，加硫酸铵溶液（ $300\mathrm{g/L}$ ） $10\mathrm{mL}$ ，DDTC溶液（ $1\mathrm{g/L}$ ） $10\mathrm{mL}$ ，摇匀。放置 $5\mathrm{min}$ 左右，加入 $10\mathrm{mL}$ MIBK，剧烈振摇提取 $1\mathrm{min}$ ，静置分层后，弃去水层，将MIBK层放入 $10\mathrm{mL}$ 带塞刻度管中，得到标准系列溶液。

将标准系列溶液按质量由低到高的顺序分别导入火焰原子化器，原子化后测其吸光度值，以铅的质量为横坐标，吸光度值为纵坐标，制作标准曲线。

# 12.3.3 试样溶液的测定

将试样消化液及试剂空白溶液分别置于 $125~\mathrm{mL}$ 分液漏斗中，补加水至 $60~\mathrm{mL}$ 。加 $2\mathrm{mL}$ 柠檬酸铵

溶液 $(250\mathrm{g / L})$ ，溴百里酚蓝水溶液 $(1\mathrm{g / L})3$ 滴～5滴，用氨水溶液 $(1 + 1)$ 调 $\mathrm{pH}$ 至溶液由黄变蓝，加硫酸铵溶液 $(300\mathrm{g / L})10\mathrm{mL}$ ,DDTC溶液 $(1\mathrm{g / L})10\mathrm{mL}$ ，摇匀。放置 $5\mathrm{min}$ 左右，加入 $10~\mathrm{mL}$ MIBK，剧烈振摇提取 $1\mathrm{min}$ ，静置分层后，弃去水层，将MIBK层放入 $10~\mathrm{mL}$ 带塞刻度管中，得到试样溶液和空白溶液。

将试样溶液和空白溶液分别导入火焰原子化器，原子化后测其吸光度值，与标准系列比较定量。

# 13 分析结果的表述

试样中铅的含量按式(2)计算：

$$
X = \frac {m _ {1} - m _ {0}}{m _ {2}} \tag {2}
$$

式中：

$X$ ——试样中铅的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$ ；

$m_{1}$ ——试样溶液中铅的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_0$ ——空白溶液中铅的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_{2}$ ——试样称样量或移取体积，单位为克或毫升（g或 $\mathrm{mL}$ ）。

当铅含量 $\geqslant 10.0\mathrm{mg / kg}$ （或 $\mathrm{mg / L}$ )时，计算结果保留三位有效数字；当铅含量 $<  10.0~\mathrm{mg / kg}$ （或 $\mathrm{mg / L})$ 时，计算结果保留两位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 第四法 二硫腙比色法

# 16 原理

试样经消化后，在 $\mathrm{pH}8.5\sim 9.0$ 时，铅离子与二硫腙生成红色络合物，溶于三氯甲烷。加入柠檬酸铵、氰化钾和盐酸羟胺等，防止铁、铜、锌等离子干扰。于波长 $510~\mathrm{nm}$ 处测定吸光度，与标准系列比较定量。

# 17.2 试剂配制

17.2.1 硝酸溶液 $(5 + 95)$ ：量取 $50~\mathrm{mL}$ 硝酸，缓慢加入到 $950~\mathrm{mL}$ 水中，混匀。  
17.2.2 硝酸溶液 $(1 + 9)$ ：量取 $50~\mathrm{mL}$ 硝酸，缓慢加入到 $450~\mathrm{mL}$ 水中，混匀。  
17.2.3 氨水溶液 $(1 + 1)$ ：量取 $100~\mathrm{mL}$ 氨水，加入 $100~\mathrm{mL}$ 水，混匀。  
17.2.4 氨水溶液 $(1 + 99)$ ：量取 $10\mathrm{mL}$ 氨水，加入 $990~\mathrm{mL}$ 水，混匀。  
17.2.5 盐酸溶液 $(1 + 1)$ ：量取 $100~\mathrm{mL}$ 盐酸，加入 $100~\mathrm{mL}$ 水，混匀。  
17.2.6 酚红指示液 $(1\mathrm{g / L})$ ：称取 $0.1\mathrm{g}$ 酚红，用少量多次乙醇溶解后移入 $100~\mathrm{mL}$ 容量瓶中并定容至刻度，混匀。  
17.2.7 二硫腙-三氯甲烷溶液 $(0.5\mathrm{g} / \mathrm{L})$ : 称取 $0.5\mathrm{g}$ 二硫腙, 用三氯甲烷溶解, 并定容至 $1000\mathrm{mL}$ , 混匀, 保存于 $0^{\circ}\mathrm{C} \sim 5^{\circ}\mathrm{C}$ 下, 必要时用下述方法纯化。

称取 $0.5\mathrm{g}$ 研细的二硫腙，溶于 $50~\mathrm{mL}$ 三氯甲烷中，如不全溶，可用滤纸过滤于 $250~\mathrm{mL}$ 分液漏斗中，用氨水溶液 $(1 + 99)$ 提取三次，每次 $100~\mathrm{mL}$ ，将提取液用棉花过滤至 $500~\mathrm{mL}$ 分液漏斗中，用盐酸溶液 $(1 + 1)$ 调至酸性，将沉淀出的二硫腙用三氯甲烷提取2次～3次，每次 $20~\mathrm{mL}$ ，合并三氯甲烷层，用等量水洗涤两次，弃去洗涤液，在 $50^{\circ}\mathrm{C}$ 水浴上蒸去三氯甲烷。精制的二硫腙置硫酸干燥器中，干燥备用。或将沉淀出的二硫腙用 $200~\mathrm{mL}, 200~\mathrm{mL}, 100~\mathrm{mL}$ 三氯甲烷提取三次，合并三氯甲烷层为二硫腙-三氯甲烷溶液。

17.2.8 盐酸羟胺溶液 $(200\mathrm{g / L})$ ：称 $20\mathrm{g}$ 盐酸羟胺，加水溶解至 $50~\mathrm{mL}$ ，加2滴酚红指示液 $(1\mathrm{g / L})$ ，加氨水溶液 $(1 + 1)$ ，调 $\mathrm{pH}$ 至 $8.5\sim 9.0$ （由黄变红，再多加2滴），用二硫腙-三氯甲烷溶液 $(0.5\mathrm{g / L})$ 提取至三氯甲烷层绿色不变为止，再用三氯甲烷洗二次，弃去三氯甲烷层，水层加盐酸溶液 $(1 + 1)$ 至呈酸性，加水至 $100~\mathrm{mL}$ ，混匀。  
17.2.9 柠檬酸铵溶液 $(200\mathrm{g / L})$ ：称取 $50\mathrm{g}$ 柠檬酸铵，溶于 $100~\mathrm{mL}$ 水中，加2滴酚红指示液 $(1\mathrm{g / L})$ 加氨水溶液 $(1 + 1)$ ，调 $\mathrm{pH}$ 至 $8.5\sim 9.0$ ，用二硫腙-三氯甲烷溶液 $(0.5\mathrm{g / L})$ 提取数次，每次 $10~\mathrm{mL}\sim$ $20~\mathrm{mL}$ ，至三氯甲烷层绿色不变为止，弃去三氯甲烷层，再用三氯甲烷洗二次，每次 $5\mathrm{mL}$ ，弃去三氯甲烷层，加水稀释至 $250~\mathrm{mL}$ ，混匀。  
17.2.10 氰化钾溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 氰化钾，用水溶解后稀释至 $100~\mathrm{mL}$ ，混匀。  
17.2.11 二硫腙使用液：吸取 $1.0 \mathrm{~mL}$ 二硫腙-三氯甲烷溶液 $(0.5 \mathrm{~g} / \mathrm{L})$ ，加三氯甲烷至 $10 \mathrm{~mL}$ ，混匀。用 $1 \mathrm{~cm}$ 比色杯，以三氯甲烷调节零点，于波长 $510 \mathrm{~nm}$ 处测吸光度(A)，用式(3)算出配制 $100 \mathrm{~mL}$ 二硫腙使用液（ $70 \%$ 透光率）所需二硫腙-三氯甲烷溶液 $(0.5 \mathrm{~g} / \mathrm{L})$ 的毫升数(V)。量取计算所得体积的二硫腙-三氯甲烷溶液，用三氯甲烷稀释至 $100 \mathrm{~mL}$ 。

$$
V = \frac {1 0 \times (2 - \lg 7 0)}{A} = \frac {1 . 5 5}{A} \quad \dots \tag {3}
$$

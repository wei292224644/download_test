# 食品安全国家标准

# 食品中蛋白质的测定

# 1 范围

本标准规定了食品中蛋白质的测定方法。

本标准第一法和第二法适用于各种食品中蛋白质的测定，第三法适用于蛋白质含量在 $10\mathrm{g} / 100\mathrm{g}$ 以上的粮食、豆类奶粉、米粉、蛋白质粉等固体试样的测定。

本标准不适用于添加无机含氮物质、有机非蛋白质含氮物质的食品的测定。

# 第一法 凯氏定氮法

# 2 原理

食品中的蛋白质在催化加热条件下被分解，产生的氨与硫酸结合生成硫酸铵。碱化蒸馏使氨游离，用硼酸吸收后以硫酸或盐酸标准滴定溶液滴定，根据酸的消耗量计算氮含量，再乘以换算系数，即为蛋白质的含量。

# 5 分析步骤

# 5.1 凯氏定氮法

5.1.1 试样处理：称取充分混匀的固体试样 $0.2\mathrm{g}\sim 2\mathrm{g}$ 、半固体试样 $2\mathrm{g}\sim 5\mathrm{g}$ 或液体试样 $10\mathrm{g}\sim 25\mathrm{g}$ （约当于 $30~\mathrm{mg}\sim 40~\mathrm{mg}$ 氮)，精确至 $0.001\mathrm{g}$ ，移入干燥的 $100~\mathrm{mL},250~\mathrm{mL}$ 或 $500~\mathrm{mL}$ 定氮瓶中，加入 $0.4\mathrm{g}$ 硫酸铜、6g硫酸钾及 $20~\mathrm{mL}$ 硫酸，轻摇后于瓶口放一小漏斗，将瓶以 $45^{\circ}C$ 角斜支于有小孔的石棉网上。小心加热，待内容物全部碳化，泡沫完全停止后，加强火力，并保持瓶内液体微沸，至液体呈蓝绿色并澄清透明后，再继续加热 $0.5\mathrm{h}\sim 1\mathrm{h}$ 。取下放冷，小心加入 $20~\mathrm{mL}$ 水，放冷后，移入 $100~\mathrm{mL}$ 容量瓶中，并用少量水洗定氮瓶，洗液并入容量瓶中，再加水至刻度，混匀备用。同时做试剂空白试验。  
5.1.2 测定：按图1装好定氮蒸馏装置，向水蒸气发生器内装水至2/3处，加入数粒玻璃珠，加甲基红乙醇溶液数滴及数毫升硫酸，以保持水呈酸性，加热煮沸水蒸气发生器内的水并保持沸腾。  
5.1.3 向接受瓶内加入 $10.0\mathrm{mL}$ 硼酸溶液及1滴 $\sim 2$ 滴A混合指示剂或B混合指示剂，并使冷凝管的

下端插入液面下，根据试样中氮含量，准确吸取 $2.0\mathrm{mL}\sim 10.0\mathrm{mL}$ 试样处理液由小玻杯注入反应室，以 $10~\mathrm{mL}$ 水洗涤小玻杯并使之流入反应室内，随后塞紧棒状玻塞。将 $10.0\mathrm{mL}$ 氢氧化钠溶液倒入小玻杯，提起玻塞使其缓缓流入反应室，立即将玻塞盖紧，并水封。夹紧螺旋夹，开始蒸馏。蒸馏 $10\mathrm{min}$ 后移动蒸馏液接收瓶，液面离开冷凝管下端，再蒸馏 $1\mathrm{min}$ 。然后用少量水冲洗冷凝管下端外部，取下蒸馏液接收瓶。尽快以硫酸或盐酸标准滴定溶液滴定至终点，如用A混合指示液，终点颜色为灰蓝色；如用B混合指示液，终点颜色为浅灰红色。同时做试剂空白。

# 5.2 自动凯氏定氮仪法

称取充分混匀的固体试样 $0.2\mathrm{g}\sim 2\mathrm{g}$ 、半固体试样 $2\mathrm{g}\sim 5\mathrm{g}$ 或液体试样 $10\mathrm{g}\sim 25\mathrm{g}$ （约当于 $30~\mathrm{mg}\sim$ $40~\mathrm{mg}$ 氮)，精确至 $0.001\mathrm{g}$ ，至消化管中，再加入 $0.4\mathrm{g}$ 硫酸铜、 $6\mathrm{g}$ 硫酸钾及 $20~\mathrm{mL}$ 硫酸于消化炉进行消化。当消化炉温度达到 $420^{\circ}C$ 之后，继续消化 $1\textrm{h}$ ，此时消化管中的液体呈绿色透明状，取出冷却后加入 $50~\mathrm{mL}$ 水，于自动凯氏定氮仪(使用前加入氢氧化钠溶液，盐酸或硫酸标准溶液以及含有混合指示剂A或B的硼酸溶液)上实现自动加液、蒸馏、滴定和记录滴定数据的过程。

# 6 分析结果的表述

试样中蛋白质的含量按式(1)计算：

$$
X = \frac {\left(V _ {1} - V _ {2}\right) \times c \times 0 . 0 1 4 0}{m \times V _ {3} / 1 0 0} \times F \times 1 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——试样中蛋白质的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$V_{1}$ ——试液消耗硫酸或盐酸标准滴定液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试剂空白消耗硫酸或盐酸标准滴定液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫酸或盐酸标准滴定溶液浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

0.0140—1.0mL硫酸 $[c(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4) = 1.000\mathrm{mol / L}]$ 或盐酸 $[c(\mathrm{HCl}) = 1.000\mathrm{mol / L}]$ 标准滴定溶液相当的氮的质量，单位为克(g)；

$m$ ——试样的质量，单位为克(g)；

$V_{3}$ ——吸取消化液的体积，单位为毫升 $(\mathrm{mL})$ ；

$F$ ——氮换算为蛋白质的系数，各种食品中氮转换系数见附录A；

100 —换算系数。

蛋白质含量 $\geqslant 1\mathrm{g} / 100\mathrm{g}$ 时，结果保留三位有效数字；蛋白质含量 $< 1\mathrm{g} / 100\mathrm{g}$ 时，结果保留两位有效数字。

注：当只检测氮含量时，不需要乘蛋白质换算系数F。

# 7 精密度

在重复条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 分光光度法

# 8 原理

食品中的蛋白质在催化加热条件下被分解, 分解产生的氨与硫酸结合生成硫酸铵, 在 $\mathrm{pH} 4.8$ 的乙

酸钠-乙酸缓冲溶液中与乙酰丙酮和甲醛反应生成黄色的3,5-二乙酰-2,6-二甲基-1,4-二氢化吡啶化合物。在波长 $400\mathrm{nm}$ 下测定吸光度值，与标准系列比较定量，结果乘以换算系数，即为蛋白质含量。

# 11 分析步骤

# 11.1 试样消解

称取充分混匀的固体试样 $0.1\mathrm{g}\sim 0.5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 、半固体试样 $0.2\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.001\mathrm{g})$

或液体试样 $1\mathrm{g}\sim 5\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ), 移入干燥的 $100\mathrm{mL}$ 或 $250\mathrm{mL}$ 定氮瓶中, 加入 $0.1\mathrm{g}$ 硫酸铜、 $1\mathrm{g}$ 硫酸钾及 $5\mathrm{mL}$ 硫酸, 摇匀后于瓶口放一小漏斗, 将定氮瓶以 $45^{\circ}$ 角斜支于有小孔的石棉网上。缓慢加热, 待内容物全部炭化, 泡沫完全停止后, 加强火力, 并保持瓶内液体微沸, 至液体呈蓝绿色澄清透明后, 再继续加热 $0.5\mathrm{h}$ 。取下放冷, 慢慢加入 $20\mathrm{mL}$ 水, 放冷后移入 $50\mathrm{mL}$ 或 $100\mathrm{mL}$ 容量瓶中, 并用少量水洗定氮瓶, 洗液并入容量瓶中, 再加水至刻度, 混匀备用。按同一方法做试剂空白试验。

# 11.2 试样溶液的制备

吸取 $2.00\mathrm{mL}\sim 5.00\mathrm{mL}$ 试样或试剂空白消化液于 $50~\mathrm{mL}$ 或 $100~\mathrm{mL}$ 容量瓶内，加1滴～2滴对硝基苯酚指示剂溶液，摇匀后滴加氢氧化钠溶液中和至黄色，再滴加乙酸溶液至溶液无色，用水稀释至刻度，混匀。

# 11.3 标准曲线的绘制

吸取 $0.00\mathrm{mL}, 0.05\mathrm{mL}, 0.10\mathrm{mL}, 0.20\mathrm{mL}, 0.40\mathrm{mL}, 0.60\mathrm{mL}, 0.80\mathrm{mL}$ 和 $1.00\mathrm{mL}$ 氨氮标准使用溶液（相当于 $0.00\mu \mathrm{g}, 5.00\mu \mathrm{g}, 10.0\mu \mathrm{g}, 20.0\mu \mathrm{g}, 40.0\mu \mathrm{g}, 60.0\mu \mathrm{g}, 80.0\mu \mathrm{g}$ 和 $100.0\mu \mathrm{g}$ 氮），分别置于 $10\mathrm{mL}$ 比色管中。加 $4.0\mathrm{mL}$ 乙酸钠-乙酸缓冲溶液及 $4.0\mathrm{mL}$ 显色剂，加水稀释至刻度，混匀。置于 $100^{\circ}\mathrm{C}$ 水浴中加热 $15\mathrm{min}$ 。取出用水冷却至室温后，移入 $1\mathrm{cm}$ 比色杯内，以零管为参比，于波长 $400\mathrm{nm}$ 处测量吸光度值，根据标准各点吸光度值绘制标准曲线或计算线性回归方程。

# 11.4 试样测定

吸取 $0.50\mathrm{mL}\sim 2.00\mathrm{mL}$ （约相当于氮 $< 100~\mu \mathrm{g})$ 试样溶液和同量的试剂空白溶液，分别于 $10~\mathrm{mL}$ 比色管中。加 $4.0\mathrm{mL}$ 乙酸钠-乙酸缓冲溶液及 $4.0\mathrm{mL}$ 显色剂，加水稀释至刻度，混匀。置于 $100^{\circ}C$ 水浴中加热 $15\mathrm{min}$ 。取出用水冷却至室温后，移入 $1\mathrm{cm}$ 比色杯内，以零管为参比，于波长 $400~\mathrm{nm}$ 处测量吸光度值，试样吸光度值与标准曲线比较定量或代入线性回归方程求出含量。

# 12 分析结果的表述

试样中蛋白质的含量按式(2)计算：

$$
X = \frac {\left(C - C _ {0}\right) \times V _ {1} \times V _ {3}}{m \times V _ {2} \times V _ {4} \times 1 0 0 0 \times 1 0 0 0} \times 1 0 0 \times F \dots \tag {2}
$$

式中：

$X$ ——试样中蛋白质的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$C$ ——试样测定液中氮的含量，单位为微克 $(\mu \mathrm{g})$ ；

$C_0$ ——试剂空白测定液中氮的含量，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——试样消化液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——试样溶液总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克(g)；

$V_{2}$ ——制备试样溶液的消化液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{4}$ ——测定用试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

100 ——换算系数；

$F$ ——氮换算为蛋白质的系数。

蛋白质含量 $\geqslant 1\mathrm{g} / 100\mathrm{g}$ 时，结果保留三位有效数字；蛋白质含量 $< 1\mathrm{g} / 100\mathrm{g}$ 时，结果保留两位有效数字。

# 13 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第三法 燃烧法

# 14 原理

试样在 $900^{\circ}\mathrm{C} \sim 1200^{\circ}\mathrm{C}$ 高温下燃烧，燃烧过程中产生混合气体，其中的碳、硫等干扰气体和盐类被吸收管吸收，氮氧化物被全部还原成氮气，形成的氮气气流通过热导检测器(TCD)进行检测。

15.2 天平：感量为 $0.1\mathrm{mg}$

# 16 分析步骤

按照仪器说明书要求称取 $0.1\mathrm{g}\sim 1.0\mathrm{g}$ 充分混匀的试样（精确至 $0.0001\mathrm{g})$ ，用锡箔包裹后置于样品盘上。试样进入燃烧反应炉 $(900^{\circ}C\sim 1200^{\circ}C)$ 后，在高纯氧 $(\geqslant 99.99\%)$ 中充分燃烧。燃烧炉中的产物 $\left(\mathrm{NO}_x\right)$ 被载气二氧化碳或氮气运送至还原炉 $(800^{\circ}C)$ 中，经还原生成氮气后检测其含量。

# 17 分析结果的表述

试样中蛋白质的含量按式(3)计算：

$$
X = C \times F \quad \dots \tag {3}
$$

式中：

$X$ ——试样中蛋白质的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$C$ ——试样中氮的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$F$ ——氮换算为蛋白质的系数。

结果保留三位有效数字。

# 18 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 19 其他

本方法第一法当称样量为 $5.0\mathrm{g}$ 时，检出限为 $8\mathrm{mg} / 100\mathrm{g}$

本方法第二法当称样量为 $5.0\mathrm{g}$ 时，检出限为 $0.1\mathrm{mg} / 100\mathrm{g}$

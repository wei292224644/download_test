# 蜂蜜和蜂王浆中氟胺氰菊酯残留量的测定 气相色谱法

# 1 范围

本文件规定了蜂蜜和蜂王浆中氟胺氰菊酯残留量检测的制样和气相色谱测定方法。

本文件适用于蜂蜜和蜂王浆中氟胺氰菊酯残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的氟胺氰菊酯经水溶解，用正己烷丙酮溶液提取，弗罗里硅土固相萃取柱净化，采用配置电子捕获检测器的气相色谱仪测定，外标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 丙酮 $\mathrm{(CH_3COCH_3)}$ ：色谱纯。  
5.1.2 正己烷 $\mathrm{(C_6H_{14})}$ ：色谱纯。  
5.2 溶液配制  
5.2.1 正己烷-丙酮溶液 $(1:1)$ ：取等体积的正己烷、丙酮，混匀。  
5.2.2 正己烷-丙酮溶液 $(95:5)$ ：取丙酮 $5\mathrm{mL}$ 、正己烷 $95~\mathrm{mL}$ ，混匀。  
5.3 标准品

氟胺氰菊酯(tau-fluvalinate, $\mathrm{C_{26}H_{22}ClF_3N_2O_3}$ ,CAS号：102851-06-9）：含量 $\geq 98.0\%$ 。

# 5.4 标准溶液制备

5.4.1 标准储备液：取氟胺氰菊酯对照品 $10\mathrm{mg}$ ，精密称定，用正己烷溶解并稀释定容至 $100\mathrm{mL}$ 容量瓶，配制成浓度为 $100\mu \mathrm{g} / \mathrm{mL}$ 的标准储备溶液。 $0^{\circ}\mathrm{C} \sim 5^{\circ}\mathrm{C}$ 保存，有效期6个月。  
5.4.2 标准工作液：精密量取标准储备液适量，用正己烷稀释，配制成浓度为 $10~\mu \mathrm{g / L}, 20~\mu \mathrm{g / L},$ $50~\mu \mathrm{g / L}, 100~\mu \mathrm{g / L}, 200~\mu \mathrm{g / L}$ 和 $500~\mu \mathrm{g / L}$ 系列标准工作溶液。现配现用。  
5.5 材料

弗罗里硅土固相萃取柱： $1\mathrm{g} / 6\mathrm{mL}$ ，或相当者。

# 6 仪器与设备

6.1 气相色谱仪：配备电子捕获检测器  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$

6.3 涡旋混合仪。  
6.4 离心机。  
6.5 旋转蒸发仪。  
6.6 周相萃取装置。  
6.7 氮吹仪。

# 7 试料的制备与保存

# 7.1 试料的制备

蜂蜜：将供试样品搅拌均匀；对于结晶的样品，在密闭的情况下，放置于不超过 $60^{\circ}\mathrm{C}$ 的水浴中溶解，搅拌均匀后迅速冷却至室温。分出约 $150\mathrm{g}$ 作为试料，置于样品瓶中，密封，并标明标记。

蜂王浆：将供试样品搅拌均匀；对于冷冻的供试样品，在常温下完全解冻后，搅拌均匀。分出约 $150\mathrm{g}$ 作为试料，置于样品瓶中，密封，并标明标记。

# 7.2 试料的保存

蜂蜜：常温保存。

蜂王浆： $-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取蜂蜜 $5\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g})$ [蜂王浆 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g})]$ ，于 $50~\mathrm{mL}$ 具塞聚丙烯离心管，加水 $10~\mathrm{mL}$ ，涡旋混合 $1\mathrm{min}$ ，使溶解。加正己烷-丙酮溶液 $(1:1)20~\mathrm{mL}$ ，涡旋 $3\mathrm{min},3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 取上层于 $150~\mathrm{mL}$ 鸡心瓶。用正己烷-丙酮溶液 $(1:1)20~\mathrm{mL}$ 重复提取一次，合并上层液， $40^{\circ}C$ 水浴减压浓缩至近干，加正己烷 $3\mathrm{mL}$ 使溶解，备用。

# 8.2 净化

弗罗里硅土固相萃取柱依次用正己烷-丙酮溶液（95：5）、正己烷各 $5\mathrm{mL}$ 活化，取备用液过柱，再用正己烷 $3\mathrm{mL}$ 洗涤鸡心瓶，洗液过柱，弃去此部分流出液。用正己烷-丙酮溶液 $(95:5)10\mathrm{mL}$ 洗脱，收集洗脱液， $40^{\circ}\mathrm{C}$ 水浴氮气吹至干，用正己烷 $1.0\mathrm{mL}$ 复溶，供气相色谱仪测定。

# 8.3 标准工作曲线的制备

准确吸取氟胺氰菊酯标准工作溶液适量，分别进样，以标准工作溶液浓度为横坐标，以峰面积为纵坐标，绘制标准工作曲线。

# 8.4 测定

8.4.1 气相色谱参考条件

a）色谱柱：VF-1MS( $100\%$ 二甲基聚硅氧烷)石英毛细管柱 $(30\mathrm{m}\times 0.25\mathrm{mm}\times 0.25\mathrm{m})$ ，或相当者；  
b）载气：氮气，纯度 $\geq 99.999\%$   
c）进样口温度： $280^{\circ}\mathrm{C}$   
d） 检测器温度： $300^{\circ}\mathrm{C}$   
e）柱温：初始温度 $80^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $30^{\circ}\mathrm{C / min}$ 升温至 $200^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，再以 $10^{\circ}\mathrm{C / min}$ 升温至 $280^{\circ}\mathrm{C}$ ，保持 $6\mathrm{min}$   
f）载气流速： $1.0\mathrm{mL / min}$   
g）进样量： $1\mu \mathrm{L}$   
h）进样方式：不分流进样， $1\mathrm{min}$ 后打开分流阀。

# 8.4.2 测定法

取试料溶液和相应的标准溶液，作单点或多点校准，以色谱峰面积定量，按外标法计算。标准溶液及试料溶液中氟胺氰菊酯响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液色谱图见附

录A。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试料中残留量的待测物按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V}{A _ {\mathrm {S}} \times m} \tag {1}
$$

式中：

$C$ —供试试料溶液中氟胺氰菊酯浓度的数值，单位微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$C_{\mathrm{S}}$ ——标准溶液中氟胺氰菊酯浓度的数值，单位微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$A_{\mathrm{S}}$ ——标准溶液中氟胺氰菊酯的峰面积；

$A$ 供试试料溶液中氟胺氰菊酯的峰面积；

$X$ 供试试料中氟胺氰菊酯残留量的数值，单位为微克每千克 $(\mu \mathrm{g / kg})$

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ ——供试试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本标准方法中氟胺氰菊酯在蜂蜜和蜂王浆中的检测限分别为 $2\mu \mathrm{g / kg}$ 和 $4\mu \mathrm{g / kg}$

本标准方法中氟胺氰菊酯在蜂蜜和蜂王浆中的定量限分别为 $5\mu \mathrm{g / kg}$ 和 $10~\mu \mathrm{g / kg}$

# 10.2 准确度

蜂蜜中氟胺氰菊酯在 $5\mu \mathrm{g} / \mathrm{kg}\sim 100\mu \mathrm{g} / \mathrm{kg}$ 添加水平的回收率为 $70\% \sim 120\%$

蜂王浆中氟胺氰菊酯在 $10\mu \mathrm{g / kg}\sim 100\mu \mathrm{g / kg}$ 添加水平的回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

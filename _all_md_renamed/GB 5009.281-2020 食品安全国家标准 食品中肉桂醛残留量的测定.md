# 食品安全国家标准

# 食品中肉桂醛残留量的测定

# 1 范围

本标准规定了食品中肉桂醛残留量的气相色谱-质谱测定方法。

本标准适用于水果及禽畜肉中肉桂醛残留量的测定。

# 2 原理

试样中的肉桂醛用乙酸乙酯均质提取，根据不同样品基质，提取液经N一丙基乙二胺（PSA）固相吸附剂、石墨化碳(GCB)固相吸附剂或碳十八 $(\mathrm{C}_{18})$ 固相吸附剂净化，气相色谱-质谱检测和确证，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为色谱纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ 。  
3.1.2 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ 。  
3.1.3 甲苯 $(\mathrm{C}_7\mathrm{H}_8)$ 。  
3.1.4 亚铁氰化钾 $\left[\mathrm{K}_{4} \mathrm{Fe}(\mathrm{CN})_{6} \cdot 3 \mathrm{H}_{2} \mathrm{O}\right]$ ：分析纯。  
3.1.5 乙酸锌 $\left[\mathrm{Zn}\left(\mathrm{CH}_{3} \mathrm{COO}\right)_{2} \cdot 2 \mathrm{H}_{2} \mathrm{O}\right]$ ：分析纯。  
3.1.6 无水乙酸 $\mathrm{(C_2H_4O_2)}$ ：分析纯。

# 3.2 溶液配制

3.2.1 亚铁氰化钾溶液（ $92\mathrm{g / L}$ ）：称取 $106\mathrm{g}$ 亚铁氰化钾，加入适量水溶解，用水定容至 $1000~\mathrm{mL}$ 。  
3.2.2 乙酸锌溶液（ $183\mathrm{g / L}$ ）：称取 $220\mathrm{g}$ 乙酸锌溶于少量水中，加入 $30\mathrm{mL}$ 无水乙酸，用水定容至 $1000\mathrm{mL}$ 。

# 3.3 标准品

肉桂醛（Cinnamaldehyde， $\mathrm{C_9H_8O}$ ，CAS号为104-55-2，相对分子质量为132.16）：标准物质纯度 $\geq 98\%$ 。

# 3.4 标准溶液配制

3.4.1 肉桂醛标准储备液 $(100\mathrm{mg / L})$ ：准确称取 $10\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )的肉桂醛标准物质，用乙腈溶解并定容至 $100~\mathrm{mL}$ 。于 $4^{\circ}C$ 避光保存，保存期为3个月。  
3.4.2 肉桂醛标准中间溶液 $(1\mathrm{mg / L})$ ：准确吸取肉桂醛标准储备液 $0.5\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，用乙

酸乙酯定容。于 $4^{\circ} \mathrm{C}$ 避光保存，保存期为1个月。

3.4.3 肉桂醛标准工作液：准确吸取肉桂醛标准中间液 $0.20\mathrm{mL}, 0.50\mathrm{mL}, 1.00\mathrm{mL}, 2.00\mathrm{mL}$ 和 $5.00\mathrm{mL}$ ，用乙酸乙酯定容至 $10\mathrm{mL}$ ，配制成质量浓度分别为 $0.02\mu \mathrm{g} / \mathrm{mL}, 0.05\mu \mathrm{g} / \mathrm{mL}, 0.1\mu \mathrm{g} / \mathrm{mL}, 0.2\mu \mathrm{g} / \mathrm{mL}$ 和 $0.5\mu \mathrm{g} / \mathrm{mL}$ 的标准工作溶液。临用现配。

# 3.5 材料

3.5.1 N-丙基乙二胺(PSA)固相吸附剂：粒度 $40\mu \mathrm{m}\sim 60\mu \mathrm{m}$   
3.5.2 石墨化碳黑(GCB)固相吸附剂：粒度 $120\mu \mathrm{m}\sim 400\mu \mathrm{m}$   
3.5.3 碳十八 $(\mathrm{C}_{18})$ 固相吸附剂：平均粒度 $50~\mu \mathrm{m}$   
3.5.4 有机相针筒式滤膜： $0.45\mu \mathrm{m}$

# 4 仪器和设备

4.1 气相色谱-质谱仪：配有电子轰击源（EI）。  
4.2 分析天平：感量 $0.001\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
4.3 高速均质仪：转速 $>10000\mathrm{r / min}$   
4.4 高速离心机：转速 $>8000\mathrm{r / min}$   
4.5 旋涡混合器。

# 5 分析步骤

# 5.1 试样制备

取代表性样品可食部分约 $500\mathrm{g}$ ，经捣碎机充分捣碎均匀，装入洁净容器，密封，标明标记。试样于 $-18^{\circ}\mathrm{C}$ 保存。

# 5.2 提取

5.2.1 水果：称取试样 $5\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $10~\mathrm{mL}$ 乙酸乙酯， $10000~\mathrm{r / min}$ 均质提取 $1\mathrm{min}$ 。将提取液于 $8000~\mathrm{r / min}$ 条件下离心 $5\mathrm{min}$ ，转移出上层乙酸乙酯提取液，定容至 $10~\mathrm{mL}$ ，待净化。  
5.2.2 禽畜肉：称取试样 $5\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $10~\mathrm{mL}$ 水，漩涡混匀。向提取液中分别加入铁氰化钾溶液 $2\mathrm{mL}$ 和乙酸锌溶液 $2\mathrm{mL}$ ，漩涡混匀后加入 $20~\mathrm{mL}$ 乙酸乙酯， $10000~\mathrm{r / min}$ 均质提取 $1\mathrm{min}$ 。将提取液于 $8000~\mathrm{r / min}$ 条件下离心 $5\mathrm{min}$ ，转移出上层乙酸乙酯提取液，定容至 $20~\mathrm{mL}$ ，待净化。

# 5.3 净化

5.3.1 水果：分别称取 $50\mathrm{mg}$ N-丙基乙二胺(PSA）固相吸附剂、 $10\mathrm{mg}$ 石墨化碳黑(GCB)固相吸附剂和 $10~\mu \mathrm{L}$ 甲苯于 $2\mathrm{mL}$ 具塞离心管中，加入 $1\mathrm{mL}$ 提取液，漩涡震荡 $1\mathrm{min}$ ，静置 $5\mathrm{min}$ ，将净化液通过针筒式滤膜，供气相色谱-质谱测定和确证。  
5.3.2禽畜肉：分别称取 $50\mathrm{mg}$ N-丙基乙二胺(PSA）固相吸附剂、 $10\mathrm{mg}$ 石墨化碳黑(GCB)固相吸附剂、 $100\mathrm{mg}$ 碳十八 $(\mathrm{C}_{18})$ 固相吸附剂和 $10~\mu \mathrm{L}$ 甲苯于 $2\mathrm{mL}$ 具塞离心管中，加入 $1\mathrm{mL}$ 提取液，漩涡震荡 $1\mathrm{min}$ ，静置 $5\mathrm{min}$ ，将净化液通过针筒式滤膜，供气相色谱-质谱测定和确证。

# 5.4 测定

# 5.4.1 气相色谱-质谱参考条件

气相色谱-质谱参考条件如下：

a）色谱柱： $30\mathrm{m}\times 0.25\mathrm{mm}(\mathrm{i.d.})$ ，膜厚 $0.25\mu \mathrm{m}$ ，DB-5MS石英毛细管柱，或相当者；  
b）色谱柱温度： $40^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，以 $15^{\circ}\mathrm{C / min}$ 的升温速率升至 $180^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $50^{\circ}\mathrm{C / min}$ 的升温速率升至 $300^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ ；  
c）进样口温度： $250~\mathrm{^\circ C}$   
d）色谱-质谱接口温度： $280^{\circ}\mathrm{C}$   
e) 载气：氦气，纯度 $\geqslant 99.999\%$ ，流速 $1.2~\mathrm{mL / min}$   
f）进样量： $1\mu \mathrm{L}$   
g）进样方式：不分流进样， $1.5\mathrm{min}$ 后开阀；  
h）电离方式：EI；  
i） 电离能量： $70\mathrm{eV}$   
j） 测定方式：选择离子监测方式；  
k）选择监测离子 $(m / z)$ ：见表1；  
1) 溶剂延迟：5 min；  
m）离子源温度： $280^{\circ}C$ 。

# 5.4.2 定性测定

对标准溶液及样液均按5.4.1规定的条件进行测定，如果样液与标准溶液在相同的保留时间有峰出现，则对其进行质谱确证，在扣除背景后的样品谱图中，所选择离子全部出现，同时所选择离子的离子丰度比与标准品相关离子的相对丰度一致，波动范围在表2的最大容许偏差之内，可判定样品中存在肉桂醛。被确证的样品可判定为肉桂醛阳性检出。肉桂醛标准溶液的质谱图参见附录A中图A.1。

# 5.4.3 定量测定

根据样液中肉桂醛含量的情况，选定与样液中肉桂醛浓度相近的标准工作溶液。标准工作溶液和样液中肉桂醛响应值应在仪器测定的线性范围内。标准工作溶液和样液等体积分别进样测定。在上述色谱条件下，肉桂醛的保留时间约为 $8.8\mathrm{min}$ ，标准品的总离子流色谱图参见附录A中图A.2。

# 5.5 空白试验

除不加试样外，均按上述测定步骤进行。

# 6 结果计算和表述

试样中肉桂醛的含量按式(1)计算：

$$
X = \frac {c \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中肉桂醛的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——由标准曲线得出的试样液中待测物的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样的定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——由 $\mu \mathrm{g} / \mathrm{mL}$ 转换为 $\mathrm{mg / kg}$ 的换算因子；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

结果须扣除空白值，保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

称样量为 $5.0\mathrm{g}$ 时，水果的检出限为 $0.010\mathrm{mg / kg}$ ，定量限为 $0.050\mathrm{mg / kg}$ ；禽畜肉的检出限为 $0.020\mathrm{mg / kg}$ ，定量限为 $0.10\mathrm{mg / kg}$ 。

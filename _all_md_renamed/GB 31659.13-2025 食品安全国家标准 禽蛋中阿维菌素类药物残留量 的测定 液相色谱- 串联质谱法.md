# 液相色谱-串联质谱法

# 1 范围

本文件规定了禽蛋中阿维菌素类药物残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于禽蛋中阿维菌素 $\mathrm{B}_{1\mathrm{a}}$ 、22.23-二氢阿维菌素 $\mathrm{B}_{1\mathrm{b}}$ 、多拉菌素、乙酰氨基阿维菌素 $\mathrm{B}_{1\mathrm{a}}$ 和莫昔克丁残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的阿维菌素类药物经酸化乙腈提取, 固相萃取柱净化, 液相色谱-串联质谱法测定, 外标法定量。

# 5 试剂和材料

水为符合 GB/T 6682 规定的一级水。

# 5.1 试剂

# 5.2 溶液配制

5.1.1 乙腈(CHCN)2色谱纯  
5.1.2 甲酸（HCOOH）：色谱纯  
5.1.3 乙酸铵 $\mathrm{(CH_3COONH_4)}$ ：色谱纯  
5.2.1 $1\%$ 甲酸乙腈溶液：取甲酸 $10~\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.2 $0.1\%$ 甲酸乙腈溶液：取甲酸 $\mathrm{ImL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.3 含 $10 \mathrm{mmol} / \mathrm{L}$ 乙酸铵的 $0.1\%$ 甲酸水溶液：取乙酸铵 $0.77 \mathrm{~g}$ ，用适量水充分溶解，加入甲酸 $1 \mathrm{~mL}$ 后，加水稀释至 $1000 \mathrm{~mL}$ ，混匀，现用现配。  
5.2.4 $50\%$ 乙腈水溶液：取乙腈 $500~\mathrm{mL}$ ，加水稀释至 $1000~\mathrm{mL}$ ，混匀。

# 5.3 标准品

阿维菌素 $\mathrm{B_{1a}}$ 、22，23-二氢阿维菌素 $\mathrm{B_{1a}}$ 、多拉菌素、乙酰氨基阿维菌素 $\mathrm{B_{1a}}$ 和莫昔克丁，含量均 $\geqslant 95\%$ 见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液：取阿维菌素 $\mathrm{B_{1a}}$ 、22，23-二氢阿维菌素 $\mathrm{B_{1a}}$ 、多拉菌素、乙酰氨基阿维菌素 $\mathrm{B_{1a}}$ 、莫昔克丁标准品适量（相当于各有效成分 $10\mathrm{mg}$ )，精密称定，用乙腈溶解并稀释定容于 $100~\mathrm{mL}$ 容量瓶，配制成浓度为 $100~\mu \mathrm{g / mL}$ 标准储备液，一 $18^{\circ}C$ 以下避光保存，有效期3个月。  
5.4.2 $10~\mu \mathrm{g / mL}$ 混合标准工作液：分别精密量取阿维菌素类药物的标准储备液各 $1\mathrm{mL}$ ，于 $10~\mathrm{mL}$ 容量瓶

中，用乙腈稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的混合标准工作液。 $2^{\circ}C\sim 8^{\circ}C$ 避光保存，有效期1个月。

5.4.3 $1\mu \mathrm{g / mL}$ 混合标准工作液：精密量取 $10~\mu \mathrm{g / mL}$ 混合标准工作液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，配制成浓度为 $1\mu \mathrm{g / mL}$ 的混合标准工作液，现用现配。

# 5.5 材料

5.5.1 通过型亲水亲脂平衡固相萃取小柱： $150\mathrm{mg / 3mL}$ ，或相当者。  
5.5.2 微孔尼龙滤膜： $0.22\mu \mathrm{m}$   
5.5.3 盐析包：含 $4\mathrm{g}$ 氯化钠和 $1\mathrm{g}$ 无水硫酸镁。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
6.3 离心机：转速不低于 $4000\mathrm{r / min}$   
6.4 组织匀浆机。  
6.5 涡旋混合器。  
6.6 振荡器。  
6.7 氮吹仪。  
6.8 固相萃取装置。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试禽蛋样品，洗净去壳，并均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准溶液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试料 $(5 \pm 0.05)$ g 于 $50 \mathrm{~mL}$ 离心管中, 加入纯水 $5 \mathrm{~mL}$ 并涡旋混合 $30 \mathrm{~s}$ 后, 准确加入 $1 \%$ 甲酸乙腈溶液 $10 \mathrm{~mL}$ , 振荡 $15 \mathrm{~min}$ 。加入含 $4 \mathrm{~g}$ 氯化钠和 $1 \mathrm{~g}$ 无水硫酸镁的盐析包, 涡旋 $1 \mathrm{~min}, 4000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ , 取上层乙腈相, 备用。

# 8.2 净化

取备用液 $2\mathrm{mL}$ 至固相萃取小柱，流速控制在 $1\mathrm{mL} / \mathrm{min}\sim 2\mathrm{mL} / \mathrm{min}$ ，收集流出液，并用 $1\mathrm{mL}$ 乙腈淋洗固相萃取小柱，合并流出液和淋洗液， $40^{\circ}\mathrm{C}$ 下氮气吹干。用 $50\%$ 乙腈水溶液 $1\mathrm{mL}$ 溶解残留物，超声 $1\mathrm{min}$ ，过 $0.22\mu \mathrm{m}$ 微孔尼龙滤膜，供液相色谱-串联质谱仪测定。

# 8.3 基质匹配标准曲线的制备

精密量取标准工作溶液适量，用经提取和净化的空白试料溶液稀释配制浓度为 $1\mu \mathrm{g} / \mathrm{L},5\mu \mathrm{g} / \mathrm{L}$ 、 $10~\mu \mathrm{g} / \mathrm{L},50~\mu \mathrm{g} / \mathrm{L},100~\mu \mathrm{g} / \mathrm{L}$ 和 $250~\mu \mathrm{g} / \mathrm{L}$ 的基质匹配标准溶液，过滤后供液相色谱-串联质谱测定。以各药物定量离子色谱峰面积为纵坐标、基质匹配标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在相同测试条件下，试样溶液中阿维菌素类药物与校正标准溶液中阿维菌素类药物的保留时间偏差在 $\pm 0.1\mathrm{min}$ 以内；且检测到的相对离子丰度应与浓度相当的校正标准溶液相对离子丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.4.3.2 定量测定

取试样溶液和基质匹配标准工作液，做单点或多点校准，按外标法以色谱峰面积定量，基质匹配标准工作液及试样溶液中的阿维菌素类药物响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，

阿维菌素类药物基质匹配标准溶液特征离子质量色谱图见附录B。

# 8.5 空白试验

取空白试样，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中阿维菌素类药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {s} \times A \times V _ {1} \times V _ {3}}{A _ {s} \times V _ {2} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中阿维菌素类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{s}$ ——基质匹配标准溶液中阿维菌素类药物浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A$ ——试料溶液中阿维菌素类药物的峰面积；

$V_{1}$ ——试料提取液总体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——试样最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——基质匹配标准溶液中阿维菌素类药物的峰面积；

$V_{2}$ ——净化用提取液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法的检出限为 $0.2\mu \mathrm{g / kg}$ ；莫昔克丁定量限为 $1\mu \mathrm{g / kg}$ ，阿维菌素 $\mathbf{B}_{\mathrm{la}}$ 、22，23-二氢阿维菌素 $\mathbf{B}_{\mathrm{la}}$ 、多拉菌素、乙酰氨基阿维菌素 $\mathbf{B}_{\mathrm{la}}$ 定量限为 $0.5\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 200\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 20\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

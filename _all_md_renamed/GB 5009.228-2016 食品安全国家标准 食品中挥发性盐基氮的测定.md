2016-08-31 发布

# 前言

本标准代替GB/T5009.44—2003《肉与肉制品卫生标准的分析方法》、GB/T5009.45—2003《水产品卫生标准的分析方法》、GB/T5009.47—2003《蛋与蛋制品卫生标准的分析方法》中的挥发性盐基氮测定部分，代替SC/T3032—2007《水产品中挥发性盐基氮的测定》。

本标准与GB/T5009.44—2003相比，主要修改如下：

——标准名称修改为“食品安全国家标准 食品中挥发性盐基氮的测定”；  
——修改了标准的适用范围；  
——整合了GB/T5009.47—2003《蛋与蛋制品卫生标准的分析方法》中皮蛋(松花蛋)挥发性盐基氮的测定方法；  
——增加了自动凯氏定氮仪法，作为第二法；  
——改进了微量扩散法的操作方法。

# 食品安全国家标准

# 食品中挥发性盐基氮的测定

# 1 范围

本标准规定了食品中挥发性盐基氮的测定方法。

本标准适用于以肉类为主要原料的食品、动物的鲜(冻)肉、肉制品和调理肉制品、动物性水产品和海产品及其调理制品、皮蛋(松花蛋)和咸蛋等腌制蛋制品中挥发性盐基氮的测定。

# 第一法 半微量定氮法

# 2 原理

挥发性盐基氮是动物性食品由于酶和细菌的作用，在腐败过程中，使蛋白质分解而产生氨以及胺类等碱性含氮物质。挥发性盐基氮具有挥发性，在碱性溶液中蒸出，利用硼酸溶液吸收后，用标准酸溶液滴定计算挥发性盐基氮含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 氧化镁 $(\mathrm{MgO})$ 。  
3.1.2 硼酸 $\mathrm{(H_3BO_3)}$   
3.1.3 三氯乙酸 $(\mathrm{C}_2\mathrm{HCl}_3\mathrm{O}_2)$ 。  
3.1.4 盐酸（HCl）或硫酸 $\mathrm{(H_2SO_4)}$   
3.1.5 甲基红指示剂 $\mathrm{(C_{15}H_{15}N_3O_2)}$ 。  
3.1.6 溴甲酚绿指示剂 $\mathrm{(C_{21}H_{14}Br_4O_5S)}$ 或亚甲基蓝指示剂 $\mathrm{(C_{16}H_{18}ClN_3S\cdot 3H_2O)}$ 。  
3.1.7 $95\%$ 乙醇 $\mathrm{(C_2H_5OH)}$   
3.1.8 消泡硅油。

# 3.2 试剂配制

3.2.1 氧化镁混悬液 $(10\mathrm{g} / \mathrm{L})$ ：称取 $10\mathrm{g}$ 氧化镁，加 $1000~\mathrm{mL}$ 水，振摇成混悬液。  
3.2.2 硼酸溶液 $(20\mathrm{g / L})$ ：称取 $20\mathrm{g}$ 硼酸，加水溶解后并稀释至 $1000~\mathrm{mL}$ 。  
3.2.3 三氯乙酸溶液 $(20\mathrm{g / L})$ ：称取 $20\mathrm{g}$ 三氯乙酸，加水溶解后并稀释至 $1000~\mathrm{mL}$ 。  
3.2.4 盐酸标准滴定溶液 $(0.1000\mathrm{mol} / \mathrm{L})$ 或硫酸标准滴定溶液 $(0.1000\mathrm{mol} / \mathrm{L})$ ：按照GB/T601制备。  
3.2.5 盐酸标准滴定溶液 $(0.0100\mathrm{mol} / \mathrm{L})$ 或硫酸标准滴定溶液 $(0.0100\mathrm{mol} / \mathrm{L})$ ：临用前以盐酸标准滴定溶液 $(0.1000\mathrm{mol} / \mathrm{L})$ 或硫酸标准滴定溶液 $(0.1000\mathrm{mol} / \mathrm{L})$ 配制。

3.2.6 甲基红乙醇溶液 $(1\mathrm{g / L})$ ：称取 $0.1\mathrm{g}$ 甲基红，溶于 $95\%$ 乙醇，用 $95\%$ 乙醇稀释至 $100~\mathrm{mL}$ 。  
3.2.7 溴甲酚绿乙醇溶液 $(1\mathrm{g / L})$ ：称取 $0.1\mathrm{g}$ 溴甲酚绿，溶于 $95\%$ 乙醇，用 $95\%$ 乙醇稀释至 $100~\mathrm{mL}$ 。  
3.2.8 亚甲基蓝乙醇溶液 $(1\mathrm{g / L})$ ：称取 $0.1\mathrm{g}$ 亚甲基蓝，溶于 $95\%$ 乙醇，用 $95\%$ 乙醇稀释至 $100~\mathrm{mL}$ 。  
3.2.9 混合指示液：1份甲基红乙醇溶液与5份溴甲酚绿乙醇溶液临用时混合，也可用2份甲基红乙醇溶液与1份亚甲基蓝乙醇溶液临用时混合。

# 4 仪器和设备

4.1 天平：感量为 $1\mathrm{mg}$   
4.2 搅拌机。  
4.3 具塞锥形瓶： $300~\mathrm{mL}$   
4.4 半微量定氮装置：如图A.1所示。  
4.5 吸量管： $10.0\mathrm{mL},25.0\mathrm{mL},50.0\mathrm{mL}$ 。  
4.6 微量滴定管： $10~\mathrm{mL}$ ，最小分度 $0.01~\mathrm{mL}$

# 5 分析步骤

# 5.1 半微量定氮装置

按图A.1安装好半微量定氮装置。装置使用前做清洗和密封性检查。

# 5.2 试样处理

鲜(冻)肉去除皮、脂肪、骨、筋腱，取瘦肉部分，鲜(冻)海产品和水产品去除外壳、皮、头部、内脏、骨刺，取可食部分，绞碎搅匀。制成品直接绞碎搅匀。肉糜、肉粉、肉松、鱼粉、鱼松、液体样品可直接使用。皮蛋(松花蛋)、咸蛋等腌制蛋去蛋壳、去蛋膜，按蛋：水 $= 2:1$ 的比例加入水，用搅拌机绞碎搅匀成匀浆。鲜(冻)样品称取试样 $20\mathrm{g}$ ，肉粉、肉松、鱼粉、鱼松等干制品称取试样 $10\mathrm{g}$ ，精确至 $0.001\mathrm{g}$ ，液体样品吸取 $10.0\mathrm{mL}$ 或 $25.0\mathrm{mL}$ ，置于具塞锥形瓶中，准确加入 $100.0\mathrm{mL}$ 水，不时振摇，试样在样液中分散均匀，浸渍 $30\mathrm{min}$ 后过滤。皮蛋、咸蛋样品称取蛋匀浆 $15\mathrm{g}$ （计算含量时，蛋匀浆的质量乘以 $2/3$ 即为试样质量），精确至 $0.001\mathrm{g}$ ，置于具塞锥形瓶中，准确加入 $100.0\mathrm{mL}$ 三氯乙酸溶液，用力充分振摇 $1\mathrm{min}$ ，静置 $15\mathrm{min}$ 待蛋白质沉淀后过滤。滤液应及时使用，不能及时使用的滤液置冰箱内 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冷藏备用。对于蛋白质胶质多、粘性大、不容易过滤的特殊样品，可使用三氯乙酸溶液替代水进行实验。蒸馏过程泡沫较多的样品可滴加1滴～2滴消泡硅油。

# 5.3 测定

向接收瓶内加入 $10\mathrm{mL}$ 硼酸溶液，5滴混合指示液，并使冷凝管下端插入液面下，准确吸取 $10.0~\mathrm{mL}$ 滤液，由小玻杯注入反应室，以 $10~\mathrm{mL}$ 水洗涤小玻杯并使之流入反应室内，随后塞紧棒状玻塞。再向反应室内注入 $5\mathrm{mL}$ 氧化镁混悬液，立即将玻塞盖紧，并加水于小玻杯以防漏气。夹紧螺旋夹，开始蒸馏。蒸馏 $5\mathrm{min}$ 后移动蒸馏液接收瓶，液面离开冷凝管下端，再蒸馏 $1\mathrm{min}$ 。然后用少量水冲洗冷凝管下端外部，取下蒸馏液接收瓶。以盐酸或硫酸标准滴定溶液 $(0.0100\mathrm{mol / L})$ 滴定至终点。使用1份甲基红乙醇溶液与5份溴甲酚绿乙醇溶液混合指示液，终点颜色至紫红色。使用2份甲基红乙醇溶液与1份亚甲基蓝乙醇溶液混合指示液，终点颜色至蓝紫色。同时做试剂空白。

# 6 分析结果的表述

试样中挥发性盐基氮的含量按式(1)计算：

$$
X = \frac {\left(V _ {1} - V _ {2}\right) \times c \times 1 4}{m \times \left(V / V _ {0}\right)} \times 1 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——试样中挥发性盐基氮的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ 或毫克每百毫升 $(\mathrm{mg} / 100\mathrm{mL})$ ；

$V_{1}$ ——试液消耗盐酸或硫酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试剂空白消耗盐酸或硫酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——盐酸或硫酸标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

14 ——滴定 $1.0 \mathrm{~mL}$ 盐酸 $[c(\mathrm{HCl}) = 1.000 \mathrm{~mol} / \mathrm{L}]$ 或硫酸 $[c(1 / 2 \mathrm{H}_{2} \mathrm{SO}_{4}) = 1.000 \mathrm{~mol} / \mathrm{L}]$ 标准滴定溶液相当的氮的质量，单位为克每摩尔（ $\mathrm{g} / \mathrm{mol}$ ）；

$m$ ——试样质量，单位为克(g)，或试样体积，单位为 $(\mathrm{mL})$ ；

$V$ ——准确吸取的滤液体积，单位为毫升 $(\mathrm{mL})$ ，本方法中 $V = 10$

$V_{0}$ ——样液总体积，单位为毫升 $(\mathrm{mL})$ ，本方法中 $V_{0} = 100$

100——计算结果换算为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ 或毫克每百毫升 $(\mathrm{mg} / 100\mathrm{mL})$ 的换算系数。

实验结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 自动凯氏定氮仪法

# 8 试剂和材料

# 8.1 试剂

8.1.1 氧化镁 $(\mathrm{MgO})$ 。

8.2.2 盐酸标准滴定溶液 $(0.1000\mathrm{mol} / \mathrm{L})$ 或硫酸标准滴定溶液 $(0.1000\mathrm{mol} / \mathrm{L})$ ：同3.2.4。  
8.2.3 甲基红乙醇溶液 $(1\mathrm{g / L})$ ：同3.2.6。

# 9 仪器和设备

9.1 天平：感量为 $1\mathrm{mg}$   
9.2 搅拌机。  
9.3 自动凯氏定氮仪。  
9.4 蒸馏管： $500~\mathrm{mL}$ 或 $750~\mathrm{mL}$ 。  
9.5 吸量管： $10.0~\mathrm{mL}$

# 10 分析步骤

# 10.1 仪器设定

10.1.1 标准溶液使用盐酸标准滴定溶液 $(0.1000\mathrm{mol} / \mathrm{L})$ 或硫酸标准滴定溶液 $(0.1000\mathrm{mol} / \mathrm{L})$ 。  
10.1.2 带自动添加试剂、自动排废功能的自动定氮仪，关闭自动排废、自动加碱和自动加水功能，设定加碱、加水体积为 $0\mathrm{mL}$ 。  
10.1.3 硼酸接收液加入设定为 $30~\mathrm{mL}$   
10.1.4 蒸馏设定：设定蒸馏时间 $180\mathrm{s}$ 或蒸馏体积 $200~\mathrm{mL}$ ，以先到者为准。  
10.1.5 滴定终点设定：采用自动电位滴定方式判断终点的定氮仪，设定滴定终点 $\mathrm{pH} = 4.65$ 。采用颜色方式判断终点的定氮仪，使用混合指示液， $30~\mathrm{mL}$ 的硼酸接收液滴加10滴混合指示液。

# 10.2 试样处理

鲜(冻)肉去除皮、脂肪、骨、筋腱，取瘦肉部分，鲜(冻)海产品和水产品去除外壳、皮、头部、内脏、骨刺，取可食部分，绞碎搅匀。制成品直接绞碎搅匀。肉糜、肉粉、肉松、鱼粉、鱼松、液体样品等均匀样品可直接使用。皮蛋(松花蛋)、咸蛋等腌制蛋去蛋壳、去蛋膜，按蛋：水 $= 2:1$ 的比例加入水，用搅拌机绞碎搅匀成匀浆。皮蛋、咸蛋样品称取蛋匀浆 $15\mathrm{g}$ （计算含量时，蛋匀浆的质量乘以 $2/3$ 即为试样质量），其他样品称取试样 $10\mathrm{g}$ ，精确至 $0.001\mathrm{g}$ ，液体样品吸取 $10.0\mathrm{mL}$ ，于蒸馏管内，加入 $75\mathrm{mL}$ 水，振摇，使试样在样液中分散均匀，浸渍 $30\mathrm{min}$ 。

# 10.3 测定

10.3.1 按照仪器操作说明书的要求运行仪器，通过清洗、试运行，使仪器进入正常测试运行状态，首先进行试剂空白测定，取得空白值。  
10.3.2 在装有已处理试样的蒸馏管中加入 $1\mathrm{g}$ 氧化镁，立刻连接到蒸馏器上，按照仪器设定的条件和仪器操作说明书的要求开始测定。  
10.3.3 测定完毕及时清洗和疏通加液管路和蒸馏系统。

# 11 分析结果的表述

试样中挥发性盐基氮的含量按式(2)计算：

$$
X = \frac {\left(V _ {1} - V _ {2}\right) \times c \times 1 4}{m} \times 1 0 0 \tag {2}
$$

式中：

$\mathrm{X}$ ——试样中挥发性盐基氮的含量, 单位为毫克每百克 $(\mathrm{mg} / 100 \mathrm{~g})$ 或毫克每百毫升 $(\mathrm{mg} / 100 \mathrm{~mL})$ ; $V_{1}$ ——试液消耗盐酸或硫酸标准滴定溶液的体积, 单位为毫升 $(\mathrm{mL})$ ;

$V_{2}$ ——试剂空白消耗盐酸或硫酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——盐酸或硫酸标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

14——滴定 $1.0\mathrm{mL}$ 盐酸 $[c(\mathrm{HCl}) = 1.000\mathrm{mol / L}]$ 或硫酸 $[c(1 / 2\mathrm{H}_2\mathrm{SO}_4) = 1.000\mathrm{mol / L}]$ 标准滴定溶液相当的氮的质量，单位为克每摩尔 $(\mathrm{g / mol})$

$m$ ——试样质量，单位为克(g)或试样体积，单位为 $(\mathrm{mL})$ ：

100——计算结果换算为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ 或毫克每百毫升 $(\mathrm{mg} / 100\mathrm{mL})$ 的换算系数。

实验结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 第三法 微量扩散法

# 13 原理

挥发性盐基氮可在 $37^{\circ}\mathrm{C}$ 碱性溶液中释出，挥发后吸收于硼酸吸收液中，用标准酸溶液滴定，计算挥发性盐基氮含量。

# 14 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 14.1 试剂

14.1.1 硼酸 $(\mathrm{H}_3\mathrm{BO}_3)$ 。  
14.1.2 盐酸（HCl）或硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。  
14.1.3 碳酸钾 $(\mathrm{K_2CO_3})$ 。  
14.1.4 阿拉伯胶。   
14.1.5 甘油 $(\mathrm{C_3H_8O_3})$ 。  
14.1.6 甲基红指示剂 $\left(\mathrm{C}_{15} \mathrm{H}_{15} \mathrm{~N}_{3} \mathrm{O}_{2}\right)$ 。  
14.1.7 溴甲酚绿指示剂 $\left(\mathrm{C}_{21} \mathrm{H}_{14} \mathrm{Br}_4 \mathrm{O}_5 \mathrm{~S}\right)$ 或亚甲基蓝指示剂 $\left(\mathrm{C}_{16} \mathrm{H}_{18} \mathrm{ClN}_3 \mathrm{~S} \cdot 3 \mathrm{H}_2\mathrm{O}\right)$ 。  
14.1.8 $95\%$ 乙醇 $\mathrm{(C_2H_5OH)}$

# 14.2 试剂配制

14.2.1 硼酸溶液 $(20\mathrm{g / L})$ ：同3.2.2。  
14.2.2 盐酸标准滴定溶液 $(0.010\mathrm{~mol} / \mathrm{L})$ 或硫酸标准滴定溶液 $(0.010\mathrm{~mol} / \mathrm{L})$ ：同3.2.5。  
14.2.3 饱和碳酸钾溶液：称取 $50\mathrm{g}$ 碳酸钾，加 $50~\mathrm{mL}$ 水，微加热助溶，使用上清液。  
14.2.4 水溶性胶：称取 $10\mathrm{g}$ 阿拉伯胶，加 $10~\mathrm{mL}$ 水，再加 $5\mathrm{mL}$ 甘油及 $5\mathrm{g}$ 碳酸钾，研匀。  
14.2.5 甲基红乙醇溶液 $(1\mathrm{g} / \mathrm{L})$ ：同3.2.6。  
14.2.6 溴甲酚绿乙醇溶液 $(1\mathrm{g / L})$ ：同3.2.7。  
14.2.7 亚甲基蓝乙醇溶液 $(1\mathrm{g / L})$ ：同3.2.8。  
14.2.8 混合指示液：同3.2.9。

# 15 仪器和设备

15.1 天平：感量为 $1\mathrm{mg}$   
15.2 搅拌机。  
15.3 具塞锥形瓶： $300~\mathrm{mL}$   
15.4 吸量管： $1.0\mathrm{mL},10.0\mathrm{mL},25.0\mathrm{mL},50.0\mathrm{mL}$   
15.5 扩散Ⅲ（标准型）：玻璃质，有内外室，带磨砂玻璃盖。  
15.6 恒温箱： $37^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 。  
15.7 微量滴定管： $10~\mathrm{mL}$ ，最小分度 $0.01~\mathrm{mL}$

# 16 分析步骤

# 16.1 试样处理

鲜(冻)肉去除皮、脂肪、骨、筋腱，取瘦肉部分，鲜(冻)海产品和水产品去除外壳、皮、头部、内脏、骨刺，取可食部分，绞碎搅匀。制成品直接绞碎搅匀。肉糜、肉粉、肉松、鱼粉、鱼松、液体样品可直接使用。皮蛋(松花蛋)、咸蛋等腌制蛋去蛋壳、去蛋膜，按蛋：水 $= 2:1$ 的比例加入水，用搅拌机绞碎搅匀成匀浆。鲜(冻)样品称取试样 $20\mathrm{g}$ ，肉粉、肉松、鱼粉、鱼松等干制品称取试样 $10\mathrm{g}$ ，皮蛋、咸蛋样品称取蛋匀浆 $15\mathrm{g}$ （计算含量时，蛋匀浆的质量乘以 $2/3$ 即为试样质量），精确至 $0.001\mathrm{g}$ ，液体样品吸取 $10.0\mathrm{mL}$ 或 $25.0\mathrm{mL}$ ，置于具塞锥形瓶中，准确加入 $100.0\mathrm{mL}$ 水，不时振摇，试样在样液中分散均匀，浸渍 $30\mathrm{min}$ 后过滤，滤液应及时使用，不能及时使用的滤液置冰箱内 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冷藏备用。

# 16.2 测定

将水溶性胶涂于扩散皿的边缘，在皿中央内室加入硼酸溶液 $1\mathrm{mL}$ 及1滴混合指示剂。在皿外室准确加入滤液 $1.0\mathrm{mL}$ ，盖上磨砂玻璃盖，磨砂玻璃盖的凹口开口处与扩散皿边缘仅留能插入移液器枪头或滴管的缝隙，透过磨砂玻璃盖观察水溶性胶密封是否严密，如有密封不严处，需重新涂抹水溶性胶。然后从缝隙处快速加入 $1\mathrm{mL}$ 饱和碳酸钾溶液，立刻平推磨砂玻璃盖，将扩散皿盖严密，于桌子上以圆周运动方式轻轻转动，使样液和饱和碳酸钾溶液充分混合，然后于 $37^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 温箱内放置 $2\mathrm{h}$ ，放凉至室温，揭去盖，用盐酸或硫酸标准滴定溶液 $(0.0100\mathrm{mol/L})$ 滴定。使用1份甲基红乙醇溶液与5份溴甲酚绿乙醇溶液混合指示液，终点颜色至紫红色。使用2份甲基红乙醇溶液与1份亚甲基蓝乙醇溶液混合指示液，终点颜色至蓝紫色。同时做试剂空白。

# 17 分析结果的表述

试样中挥发性盐基氮的含量按式(3)计算：

$$
X = \frac {\left(V _ {1} - V _ {2}\right) \times c \times 1 4}{m \times \left(V / V _ {0}\right)} \times 1 0 0 \tag {3}
$$

式中：

$X$ ——试样中挥发性盐基氮的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ 或毫克每百毫升 $(\mathrm{mg} / 100\mathrm{mL})$ ；

$V_{1}$ ——试液消耗盐酸或硫酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试剂空白消耗盐酸或硫酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——盐酸或硫酸标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

14——滴定 $1.0\mathrm{mL}$ 盐酸 $[c(\mathrm{HCl}) = 1.000\mathrm{mol / L}]$ 或硫酸 $[c(1 / 2\mathrm{H}_2\mathrm{SO}_4) = 1.000\mathrm{mol / L}]$ 标准滴

定溶液相当的氮的质量，单位为克每摩尔 $(\mathrm{g / mol})$

$m$ ——试样质量，单位为克(g)，或试样体积，单位为 $(\mathrm{mL})$ ；

$V$ ——准确吸取的滤液体积，单位为毫升 $(\mathrm{mL})$ ，本方法中 $V = 1$

$V_{0}$ ——样液总体积，单位为毫升 $(\mathrm{mL})$ ，本方法中 $V_{0} = 100$

100——计算结果换算为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ 或毫克每百毫升 $(\mathrm{mg} / 100\mathrm{mL})$ 的换算系数。

实验结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 18 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 19 检出限

本标准第一法中，当称样量为 $20.0\mathrm{g}$ 时，检出限为 $0.18\mathrm{mg} / 100\mathrm{g}$ ；当称样量为 $10.0\mathrm{g}$ 时，检出限为 $0.35\mathrm{mg} / 100\mathrm{g}$ ；液体样品取样 $25.0\mathrm{mL}$ 时，检出限为 $0.14\mathrm{mg} / 100\mathrm{mL}$ ；液体样品取样 $10.0\mathrm{mL}$ 时，检出限为 $0.35\mathrm{mg} / 100\mathrm{mL}$ 。

本标准第二法中，当称样量为 $10.0\mathrm{g}$ 时，检出限为 $0.04\mathrm{mg} / 100\mathrm{g}$ ；液体样品取样 $10.0\mathrm{mL}$ 时，检出限为 $0.04\mathrm{mg} / 100\mathrm{mL}$ 。

本标准第三法中，当称样量为 $20.0\mathrm{g}$ 时，检出限为 $1.75\mathrm{mg} / 100\mathrm{g}$ ；当称样量为 $10.0\mathrm{g}$ 时，检出限为 $3.50\mathrm{mg} / 100\mathrm{g}$ ；液体样品取样 $25.0\mathrm{mL}$ 时，检出限为 $1.40\mathrm{mg} / 100\mathrm{mL}$ ；液体样品取样 $10.0\mathrm{mL}$ 时，检出限为 $3.50\mathrm{mg} / 100\mathrm{mL}$ 。

# 食品安全国家标准

# 食品中 $\beta$ -阿朴-8' - 胡萝卜素醛的测定

# 1 范围

本标准规定了食品中 $\beta$ -阿朴-8'-胡萝卜素醛的液相色谱测定方法。

本标准适用于风味发酵乳、再制干酪、冷冻饮品、糖果、焙烤食品、半固体复合调味料及饮料中 $\beta$ -阿朴-8'-胡萝卜素醛的测定。

# 2 原理

样品经氢氧化钾溶液皂化后，以正己烷提取游离出的 $\beta$ -阿朴-8'-胡萝卜素醛，提取液浓缩并用乙腈复溶后，经高效液相色谱进行分离，紫外可见光检测器或二极管阵列检测器检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。  
3.1.2 无水乙醇 $\mathrm{(C_2H_6O)}$ 。  
3.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ 。  
3.1.4 氢氧化钾(KOH)。  
3.1.5 2,6-二叔丁基对甲酚 $\left(\mathrm{C}_{15} \mathrm{H}_{24} \mathrm{O}\right)$ , butylated hydroxytoluene, 简称 BHT)。  
3.1.6 甲酸 $\mathrm{(CH_2O_2)}$ ：色谱纯。

# 3.2 试剂配制

3.2.1 氢氧化钾溶液 $(200\mathrm{g / L})$ ：称取 $200\mathrm{g}$ 氢氧化钾，加水溶解稀释至 $1000~\mathrm{mL}$ 。  
3.2.2 $0.1\%$ BHT-乙腈溶液：称取 $0.1\mathrm{g}$ BHT，以 $100~\mathrm{mL}$ 乙腈溶解，混匀。  
3.2.3 $0.1\%$ 甲酸（体积分数）溶液：移取 $1\mathrm{mL}$ 甲酸，用水稀释至 $1\mathrm{L}$ 。

# 3.3 标准品

$\beta$ -阿朴-8'-胡萝卜素醛 $(\mathrm{C}_{30}\mathrm{H}_{40}\mathrm{O},\mathrm{CAS}$ 号：1107-26-2）：纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 $\beta$ -阿朴-8'-胡萝卜素醛标准储备液( $10~\mu \mathrm{g / mL}$ )：准确称取 $1\mathrm{mg}$ （精确至 $0.01\mathrm{mg})\beta$ -阿朴-8'-胡萝卜素醛标准品，用 $0.1\%$ BHT-乙腈溶液溶解并转移至 $100~\mathrm{mL}$ 棕色容量瓶中，用 $0.1\%$ BHT-乙腈溶液定容至 $100~\mathrm{mL}$ ，混匀。于 $-18^{\circ}C$ 保存，有效期3个月。

注：β-阿朴-8'-胡萝卜素醛标准储备液使用前需校正，具体操作见附录A。

3.4.2 $\beta$ -阿朴-8'-胡萝卜素醛标准系列工作液：分别吸取适量 $\beta$ -阿朴-8'-胡萝卜素醛标准储备液于 $10~\mathrm{mL}$ 棕色容量瓶中，加乙腈定容至刻度，配制成质量浓度分别为 $0.050\,0\mu\mathrm{g/mL}, 0.100\,μ\mathrm{g/mL}, 0.200\,μ\mathrm{g/mL}, 0.500\,μ\mathrm{g/mL}, 1.00\,μ\mathrm{g/mL}$ 的工作液。临用现配。

# 4 仪器和设备

4.1 液相色谱仪：配有二极管阵列检测器或紫外可见光检测器。  
4.2 天平：感量分别为 $0.01\mathrm{g}$ 和 $0.01\mathrm{mg}$ 。  
4.3 分光光度计。  
4.4 恒温振荡水浴装置。  
4.5 组织捣碎机。  
4.6 旋转蒸发仪。  
4.7 组织匀浆机。  
4.8 离心机：转速 $\geq 4000\mathrm{r / min}$   
4.9 涡旋混合器。  
4.10 超声清洗器：工作频率 $40\mathrm{kHz}$ ，功率 $500\mathrm{W}$ 。  
4.11 有机相微孔滤膜： $0.45\mu \mathrm{m}$

# 5 分析步骤

注：由于 $\beta$ -阿朴-8'-胡萝卜素醛对光敏感，除非另有说明，所有试验操作应在无 $500~\mathrm{nm}$ 以下紫外光的黄色光源或者红色光源环境中进行。

# 5.1 样品前处理

# 5.1.1 试样制备

固体样品(糖果、焙烤食品、固体饮料等)经组织捣碎机充分粉碎、均质；半固体样品(风味发酵乳、半固体复合调味料等)经组织匀浆机匀浆后混匀；液体样品(液体饮料等)直接混合均匀；冷冻饮品在 $40^{\circ}\mathrm{C}$ 水浴避光条件下融化后充分混匀。

将一定数量的样品按要求制备后，贮存于样品瓶中，避光冷藏，尽快检测。

# 5.1.2 试样处理

准确称取 $2\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样于 $50~\mathrm{mL}$ 离心管中，加入约 $0.2\mathrm{gBHT},10\mathrm{mL}$ 无水乙醇、 $10~\mathrm{mL}$ 氢氧化钾溶液，涡旋振荡 $1\mathrm{min}$ ，混匀，置于 $30^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 恒温水浴装置内避光振荡皂化 $30\mathrm{min}$ 。

将皂化液转移至分液漏斗中，并用 $10\mathrm{mL}$ 水分两次加入分液漏斗中，加入 $10\mathrm{mL}$ 正己烷，振荡萃取 $5\mathrm{min}$ ，将下层溶液转移至另一 $100~\mathrm{mL}$ 分液漏斗中，加入 $10~\mathrm{mL}$ 正己烷再次萃取，弃去下层水相溶液，合并正己烷层至 $50~\mathrm{mL}$ 蒸发瓶中，在室温条件下旋蒸浓缩至近干。准确加入 $20~\mathrm{mL}$ 乙腈溶解残渣，涡旋混匀 $30~\mathrm{s}$ ，过 $0.45\mu \mathrm{m}$ 有机滤膜，供液相色谱测定。

# 5.1.3 空白实验

除不加试样外，按照5.1.2的测定步骤进行空白实验。

# 5.2 仪器参考条件

5.2.1 色谱柱： $\mathrm{C_{18}}$ 柱， $150 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ (i.d.)，粒径 $5 \mu \mathrm{m}$ ，或性能相当的色谱柱。  
5.2.2 流动相：乙腈 $+0.1\%$ 甲酸溶液（95：5，体积比）。  
5.2.3 流速： $1.0~\mathrm{mL / min}$   
5.2.4 检测波长： $460~\mathrm{nm}$   
5.2.5 柱温： $40^{\circ}C$ 。  
5.2.6 进样量： $20~\mu \mathrm{L}$

# 5.3 标准曲线的制作

将标准系列工作液分别注入高效液相色谱仪中，测定相应的峰面积，以标准系列工作液中 $\beta$ -阿朴-8'-胡萝卜素醛的质量浓度为横坐标，以 $\beta$ -阿朴-8'-胡萝卜素醛的峰面积为纵坐标，绘制标准曲线。 $\beta$ -阿朴-8'-胡萝卜素醛标准溶液色谱图参见附录B。

# 5.4 试样溶液的测定

将试样溶液注入高效液相色谱仪中，得到待测物峰面积，代入标准曲线计算得到待测液中 $\beta$ -阿朴-8'-胡萝卜素醛的质量浓度。

待测样品中化合物色谱峰的保留时间与标准溶液相比变化范围应在 $\pm 2.5\%$ 之内。

# 6 结果计算和表述

试样中 $\beta$ -阿朴-8'-胡萝卜素醛的含量按式(1)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中 $\beta$ -阿朴-8'-胡萝卜素醛的含量，单位为克每千克 $(\mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中 $\beta$ -阿朴-8'-胡萝卜素醛的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$\rho_0$ ——由标准曲线得到的空白实验中 $\beta$ -阿朴-8'-胡萝卜素醛的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样的复溶体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的取样量，单位为克(g)；

1000——单位换算系数。

计算结果保留2位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

当称样量为 $2\mathrm{g}$ 时，检出限为 $0.0002\mathrm{g / kg}$ ，定量限为 $0.0005\mathrm{g / kg}$ 。

# 标准溶液质量浓度校正方法

$\beta$ -阿朴-8'-胡萝卜素醛标准储备液使用前需要进行浓度校正。取 $1.00~\mathrm{mL}$ 标准储备液于 $25~\mathrm{mL}$ 容量瓶中，以乙腈定容至刻度，摇匀。移取该溶液至 $1\mathrm{cm}$ 的石英比色皿中，使用分光光度计在 $460~\mathrm{nm}$ 波长下测定吸光值(A)。按式(A.1)计算标准溶液质量浓度(以乙腈作为空白测试溶液)。

$$
X = \frac {A}{E _ {1 \mathrm {cm}} ^ {1 \%}} \times 25 \times 10 ^ {4} \quad \dots \tag{A.1}
$$

式中：

$X$ ——标准溶液的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$A$ ——标准溶液的吸光值；

$E_{1\mathrm{cm}}^{1\%}$ ——乙腈中 $\beta$ -阿朴-8'-胡萝卜素醛的吸光系数，为 $2640~\mathrm{dL / g}$ ；

25 —稀释倍数；

$10^{4}$ ——转换系数 $(\mathrm{g / dL}$ 转化为 $\mu \mathrm{g / mL})$ 。

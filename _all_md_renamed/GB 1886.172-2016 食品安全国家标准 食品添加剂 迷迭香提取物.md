# 食品添加剂 迷迭香提取物

# 1 范围

本标准适用于以迷迭香(Rosmarinus officinalis L.)的茎、叶为原料，经溶剂提取或超临界二氧化碳萃取、精制等工艺生产的食品添加剂迷迭香提取物。提取溶剂为水、甲醇、乙醇、丙酮和(或)正己烷。

# 检验方法

# A.1 一般规定

本标准所用试剂和水在未注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品在未注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 总抗氧化成分（以鼠尾草酸和鼠尾草酚计）的测定

# A.2.1 方法提要

脂溶性迷迭香提取物中的主要抗氧化成分（鼠尾草酸和鼠尾草酚）在 $280\mathrm{nm}$ 检测波长下有最大吸收峰，根据鼠尾草酸标准品浓度和相应的峰面积，可得出鼠尾草酸的标准曲线公式，并根据标准曲线由试样峰面积得到对应的试样溶液浓度。同时，由于在 $280\mathrm{nm}$ 下，同浓度的鼠尾草酚的峰面积是鼠尾草酸的1.36倍。因此可以根据鼠尾草酸的标准曲线计算得到鼠尾草酚的浓度。

# A.2.2 试剂和材料

A.2.2.1 水：符合GB/T6682规定的一级水。  
A.2.2.2 丙酮：色谱纯。  
A.2.2.3 乙腈：色谱纯。  
A.2.2.4 鼠尾草酸标准品：纯度 $\geq 98\%$   
A.2.2.5 磷酸溶液： $1\rightarrow 1000$

# A.2.3 仪器和设备

高效液相色谱仪：配有紫外检测器。

# A.2.4 参考色谱条件

A.2.4.1 色谱柱： $\mathrm{C}_{18}$ 反相色谱柱（ $\phi 4.6 \mathrm{~mm} \times 250 \mathrm{~mm}, 5 \mu \mathrm{m}$ ）；或其他等效的色谱柱。  
A.2.4.2 流动相A：水十磷酸溶液。  
A.2.4.3 流动相B：乙腈十磷酸溶液。  
A.2.4.4 柱温： $30^{\circ}C$ 。  
A.2.4.5 检测波长： $280~\mathrm{nm}$   
A.2.4.6 流动相流速： $1.0~\mathrm{mL / min}$   
A.2.4.7 进样量： $10~\mu \mathrm{L}$   
A.2.4.8 梯度洗脱条件：见表A.1。

# A.2.5 分析步骤

# A.2.5.1 绘制标准曲线

以丙酮溶解鼠尾草酸标准品，配制混合梯度标准溶液，使鼠尾草酸浓度梯度在 $0.010\mathrm{mg / mL}\sim$ $1.000~\mathrm{mg / mL}$ 。在A.2.4参考色谱条件下，对标准溶液进行测定，重复进样一次。根据标准溶液浓度和峰面积，绘制标准曲线。线性关系应达到 $R^2\geq 0.99$ 。记录标准曲线的线性公式 $Y = a\times c + b$ 。其中， $c$ 是鼠尾草酸的浓度， $Y$ 是这个浓度对应的峰面积， $a$ 和 $b$ 分别为标准曲线的斜率和截距。

# A.2.5.2 试样溶液的制备

称取试样 $140\mathrm{mg}\sim 180\mathrm{mg}$ (精确至 $0.0001\mathrm{g}$ ), 溶于 $20\mathrm{mL}$ 丙酮, 定容至 $25\mathrm{mL}$ 容量瓶中, 充分混匀后经 $0.22\mu \mathrm{m}$ 微孔滤膜过滤。

# A.2.5.3 测定

在A.2.4参考色谱条件下，对试样液进行测定，重复进样一次。参考图B.1迷迭香提取物色谱图中鼠尾草酸和鼠尾草酚的相对位置确定出两者的响应峰，并且记录两者的峰面积 $Y_{1}$ 和 $Y_{2}$ 。

# A.2.6 结果计算

# A.2.6.1 鼠尾草酸的浓度 $c_{1}$

鼠尾草酸的浓度 $c_{1}$ ，单位为毫克每毫升 $(\mathrm{mg / mL})$ ，按式(A.1)计算：

$$
c _ {1} = \frac {Y _ {1} - b}{a} \tag {A.1}
$$

式中：

$Y_{1}$ ——试样溶液中的鼠尾草酸的峰面积；

$b$ ——鼠尾草酸标准曲线公式中的截距；

$a$ ——鼠尾草酸标准曲线公式中的斜率。

# A.2.6.2 鼠尾草酚的浓度 $c_{2}$

鼠尾草酚的浓度 $c_{2}$ ，单位为毫克每毫升 $(\mathrm{mg / mL})$ ，按式(A.2)计算：

$$
c _ {2} = \frac {Y _ {2} - b}{1 . 3 6 a} \tag {A.2}
$$

式中：

$Y_{2}$ ——试样溶液中的鼠尾草酚的峰面积；

$b$ ——鼠尾草酸标准曲线公式中的截距；

1.36——在 $280 \mathrm{~nm}$ 下，同浓度的鼠尾草酚的峰面积是鼠尾草酸的 1.36 倍；

$a$ ——鼠尾草酸标准曲线公式中的斜率。

# A.2.6.3 总抗氧化成分的质量分数 $w_{1}$

总抗氧化成分(以鼠尾草酸和鼠尾草酚计)的质量分数 $w_{1}$ ，按式(A.3)计算：

$$
w _ {1} = \frac {\left(c _ {1} + c _ {2}\right) \times V}{m} \times 100 \% \quad \dots \tag{A.3}
$$

式中：

$c_{1}$ ——试样溶液中鼠尾草酸的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$c_{2}$ ——试样溶液中鼠尾草酚的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为毫克 $(\mathrm{mg})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

# A.3 迷迭香酸的测定

# A.3.1 方法提要

水溶性迷迭香提取物用甲醇溶解，以甲醇-磷酸溶液为流动相，用以十八烷基硅烷为填料的液相色谱柱和紫外检测器或二极管阵列检测器，对试样中的迷迭香酸进行反相高效液相色谱分离和测定，与标准品保留时间比较定性，峰面积外标法定量。

# A.3.2 试剂和材料

A.3.2.1 甲醇：色谱纯。  
A.3.2.2 迷迭香酸标准品：纯度 $\geq 98\%$   
A.3.2.3 磷酸溶液： $0.5\mathrm{mL}$ 磷酸加入 $100~\mathrm{mL}$ 水。  
A.3.2.4 标准储备液：准确称取迷迭香酸标准品 $10\mathrm{mg}$ （精确至 $0.0001\mathrm{g}$ ），用流动相溶解并定容至 $10\mathrm{mL}$ ，混匀，置冰箱中保存。此溶液 $1\mathrm{mL}$ 含迷迭香酸 $1.0\mathrm{mg}$ 。

# A.3.3 仪器和设备

高效液相色谱仪：配有紫外检测器或二极管阵列检测器。

# A.3.4 参考色谱条件

A.3.4.1 色谱柱：以十八烷基硅烷为填充物的色谱柱 $(\varphi 4.6 \mathrm{~mm} \times 250 \mathrm{~mm}, 5 \mu \mathrm{m})$ ；或其他等效的色谱柱。  
A.3.4.2 流动相：甲醇：磷酸溶液 $= 45:55$   
A.3.4.3 流速： $0.6\mathrm{mL / min}$   
A.3.4.4 柱温： $40^{\circ}\mathrm{C}$   
A.3.4.5 进样量： $10~\mu \mathrm{L}$   
A.3.4.6 检测波长： $283~\mathrm{nm}$   
A.3.4.7 保留时间： $18\mathrm{min}$

# A.3.5 分析步骤

# A.3.5.1 试样溶液的制备

准确称取试样 $200\mathrm{mg}$ (精确至 $0.0001\mathrm{g}$ ), 用流动相溶解并定容至 $100\mathrm{mL}$ , 经 $0.45\mu \mathrm{m}$ 微孔滤膜过滤, 即得试样溶液。

# A.3.5.2 标准曲线的绘制

准确吸取标准储备液，配制成 $0\mathrm{mg / mL},0.25\mathrm{mg / mL},0.5\mathrm{mg / mL},1.0\mathrm{mg / mL}$ 的标准溶液系列，在A.3.4参考色谱条件下，进行色谱分析，根据迷迭香酸标准溶液的含量及相应色谱峰面积，以色谱峰面积为纵坐标，迷迭香酸含量为横坐标，绘制标准曲线。

# A.3.5.3 测定

准确吸取试样溶液 $10\mu \mathrm{L}$ ，在规定色谱条件下，进行色谱分析，以保留时间定性，峰面积外标法定量。

# A.3.6 结果计算

迷迭香酸的质量分数 $\varpi_{2}$ ，按式(A.4)计算：

$$
w _ {2} = \frac {c \times V}{1 0 0 0 \times m} \times 1 0 0 \% \tag{A.4}
$$

式中：

$c$ ——由标准曲线得出的试样溶液中迷迭香酸的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——质量换算系数；

$m$ ——试样的质量，单位为毫克 $(\mathrm{mg})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $2.5\%$ 。

# A.4 残留溶剂（正己烷、甲醇）的测定

# A.4.1 试剂和材料

A.4.1.1 水：GB/T6682规定的一级水。  
A.4.1.2 氯化钠溶液： $10\mathrm{g}$ 氯化钠溶解于 $100~\mathrm{mL}$ 水中。  
A.4.1.3 待测组分标准品：正己烷、甲醇，色谱纯。  
A.4.1.4 内标液： $160~\mu \mathrm{g / g}$ 正丙醇溶液。

# A.4.2 仪器和设备

气相色谱仪：配备氢火焰离子检测器(FID)和顶空进样器。

# A.4.3 参考色谱条件

A.4.3.1 色谱柱：石英毛细管柱 $(\varphi 0.15\mu \mathrm{m}\times 15\mathrm{m})$ ，涂层为 $6 \%$ 氰丙基苯基和 $94 \%$ 二甲基聚硅氧烷，涂层厚度为 $0.84\mu \mathrm{m}$ 。或同等性能的色谱柱。  
A.4.3.2 载气：氦气。

A.4.3.3 载气流速： $0.8\mathrm{mL / min}$   
A.4.3.4 柱温： $40^{\circ}\mathrm{C}$ 保持 $5\mathrm{min}$ ，以 $25^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $250^{\circ}\mathrm{C}$ ，总运行时间为 $13.4\mathrm{min}$ 。  
A.4.3.5 进样口温度： $250~\mathrm{^\circ C}$   
A.4.3.6 检测器温度： $300^{\circ}C$   
A.4.3.7 进样量： $1.0~\mathrm{mL}$

# A.4.4 参考顶空进样条件

A.4.4.1 试样加热温度： $90^{\circ}C$   
A.4.4.2 试样加热时间： $10\mathrm{min}$   
A.4.4.3 试样搅拌转速： $400\mathrm{r / min}$   
A.4.4.4 分流比： $1:50$   
A.4.4.5 注射器温度： $120^{\circ}C$   
A.4.4.6 压射速度： $1\mathrm{mL / s}$

# A.4.5.4 测定

在A.4.3和A.4.4参考操作条件下，分别对标准溶液系列和试样溶液进行色谱分析。

# A.4.6 结果计算

# A.4.6.1 平均响应因子 $f$

标准溶液的平均响应因子 $f$ ，按式（A.5）计算：

$$
f = \frac {\sum \left(\frac {c _ {\mathrm {s}}}{c _ {i}} \times \frac {R _ {i}}{R _ {\mathrm {s}}}\right)}{5} \tag {A.5}
$$

式中：

$c_{\mathrm{s}}$ ——每一标准溶液中待测组分的浓度，单位为微克每克 $(\mu \mathrm{g} / \mathrm{g})$ ；

$c_{i}$ ——每一标准溶液中正丙醇(内标物)的浓度，单位为微克每克 $(\mu \mathrm{g} / \mathrm{g})$ ；

$R_{i}$ ——每一标准溶液色谱图中正丙醇(内标物)的峰面积值；

$R_{\mathrm{s}}$ ——每一标准溶液色谱图中待测组分的峰面积值；

5 ——标准溶液系列数。

# A.4.6.2 待测组分含量 $w_{i}$

待测组分含量 $\varpi_{i}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.6)计算：

$$
w _ {i} = \frac {R _ {\mathrm {u}} \times c _ {0} \times f}{R _ {0} \times c _ {\mathrm {u}}} \tag {A.6}
$$

式中：

$R_{\mathrm{u}}$ ——试样溶液色谱图中待测组分的峰面积值；

$c_{0}$ ——试样溶液中正丙醇(内标物)的浓度，单位为微克每克 $(\mu \mathrm{g} / \mathrm{g})$ ；

$f$ ——平均响应因子；

$R_{0}$ ——试样溶液色谱图中正丙醇(内标物)的峰面积值；

$c_{\mathrm{u}}$ ——试样溶液中迷迭香提取物的浓度，单位为微克每克 $(\mu \mathrm{g} / \mathrm{g})$ 。

由式(A.6)计算得到待测组分(正己烷和甲醇)含量 $\varpi_{3}$ 和 $\varpi_{4}$ ，两者之和即为试样中残留溶剂含量。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

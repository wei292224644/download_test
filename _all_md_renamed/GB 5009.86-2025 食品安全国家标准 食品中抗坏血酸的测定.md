# 前言

本标准代替GB5413.18—2010《婴幼儿食品和乳品中维生素C的测定》和GB5009.86—2016《食品中抗坏血酸的测定》。

本标准与GB5009.86—2016相比，主要变化如下：

扩大了液相色谱法和荧光分光光度法的适用范围；

——修改了液相色谱法和荧光分光光度法检出限、定量限和标准曲线浓度范围；

——优化了液相色谱法的色谱条件。

# 食品安全国家标准

# 食品中抗坏血酸的测定

# 1 范围

本标准规定了食品中抗坏血酸的测定方法。

第一法液相色谱法适用于食品中L-抗坏血酸、D-异抗坏血酸的测定。

第二法荧光分光光度法适用于特殊膳食用食品、乳及乳制品、水果和蔬菜及其制品中抗坏血酸的测定，以及未添加D-异抗坏血酸的上述食品中L-抗坏血酸的测定。

第三法2，6-二氯靛酚滴定法适用于水果和蔬菜及其制品中还原型抗坏血酸的测定。

# 第一法 液相色谱法

# 2 原理

试样中的抗坏血酸经提取、还原和稀释后，以含有离子对试剂和还原剂的混合溶液作为流动相，经反相色谱分离，紫外检测器或二极管阵列检测器检测，以色谱峰的保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 冰乙酸 $\mathrm{(C_2H_4O_2)}$ 。  
3.1.2 三氯乙酸 $(\mathrm{C_2HCl_3O_2},\mathrm{TCA})$ 。  
3.1.3 偏磷酸 $\mathrm{(HPO_3)}$ 。  
3.1.4 $40\%$ 四丁基氢氧化铵水溶液（ $40\% \mathrm{TBAH}$ ）。  
3.1.5 乙酸铵 $(\mathrm{C}_2\mathrm{H}_7\mathrm{NO}_2)$ 。  
3.1.6 L-半胱氨酸 $(\mathrm{C_3H_7NO_2S})$ 。  
3.1.7 三(2-羧乙基)膦 $\mathrm{(C_9H_{15}O_6P}$ ，TCEP）。  
3.1.8 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。

# 3.2 试剂配制

3.2.1 乙酸铵溶液 $(500\mathrm{mmol / L},\mathrm{pH}5.4)$ ：称取乙酸铵 $19.27\mathrm{g}$ ，加水溶解并稀释至 $500~\mathrm{mL}$ ，用冰乙酸调 $\mathrm{pH}$ 至5.4。  
3.2.2 TCA溶液 $(150\mathrm{g / L})$ ：称取TCA $75~\mathrm{g}$ ，加水溶解并稀释至 $500~\mathrm{mL}$ ，混匀。  
3.2.3 偏磷酸溶液 $(200\mathrm{g / L})$ ：称取偏磷酸 $100\mathrm{g}$ ，加水溶解并稀释至 $500~\mathrm{mL}$ ，混匀。  
3.2.4 TCEP溶液 $(250\mathrm{mg / L})$ ：称取TCEP $125~\mathrm{mg}$ ，加水溶解并稀释至 $500~\mathrm{mL}$ ，混匀。  
3.2.5 L-半胱氨酸溶液 $(40\mathrm{g / L})$ ：称取L-半胱氨酸 $40~\mathrm{g}$ ，加适量水超声溶解，再加水稀释至1L，混匀。

临用现配。

3.2.6 流动相：量取水 $900 \mathrm{~mL}$ 于烧杯中，吸取 $40 \%$ TBAH 溶液 $3.25 \mathrm{~mL}$ 、称取乙酸铵 $3.85 \mathrm{~g}$ 置于水中，然后加入 TCEP $50 \mathrm{mg}$ （或 L-半胱氨酸溶液 $20 \mathrm{~mL}$ ），再加入乙腈 $10 \mathrm{~mL}$ ，混匀后用冰乙酸（约 $3.14 \mathrm{~mL}$ ）调 $\mathrm{pH}$ 至 $4.5 \sim 5.0$ ；最后加水稀释至 $1 \mathrm{~L}$ ，混匀；超声除去气泡。

# 3.3 标准品

3.3.1 L-抗坏血酸标准品 $\mathrm{(C_6H_8O_6}$ ,CAS号：50-81-7)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.2 D-异抗坏血酸标准品 $\mathrm{(C_6H_8O_6}$ ,CAS号：89-65-6)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 L-抗坏血酸标准溶液 $(50\mu \mathrm{g} / \mathrm{mL})$ ：称取L-抗坏血酸标准品 $10\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ，用TCEP溶液溶解并定容至 $10~\mathrm{mL}$ ，混匀。准确吸取上述标准溶液 $500~\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 棕色容量瓶中，用TCEP溶液定容，混匀。将溶液转移至棕色玻璃容器中，临用现配。  
3.4.2 D-异抗坏血酸标准溶液 $(50\mu \mathrm{g} / \mathrm{mL})$ ：称取D-异抗坏血酸标准品 $10\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ），用TCEP溶液溶解并定容至 $10~\mathrm{mL}$ ，混匀。准确吸取上述标准溶液 $500~\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 棕色容量瓶中，用TCEP溶液定容，混匀。将溶液转移至棕色玻璃容器中，临用现配。  
3.4.3 L-抗坏血酸或D-异抗坏血酸标准系列工作液：分别准确吸取L-抗坏血酸或D-异抗坏血酸标准溶液 $(50\mu \mathrm{g / mL})10\mu \mathrm{L},100\mu \mathrm{L},200\mu \mathrm{L},400\mu \mathrm{L},1000\mu \mathrm{L}$ 和 $2000\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 棕色容量瓶中，用流动相定容，轻摇混匀。L-抗坏血酸或D-异抗坏血酸标准系列工作液的质量浓度分别为 $0.05\mu \mathrm{g / mL}$ 、 $0.50\mu \mathrm{g / mL},1.0\mu \mathrm{g / mL},2.0\mu \mathrm{g / mL},5.0\mu \mathrm{g / mL}$ 和 $10~\mu \mathrm{g / mL}$ ，临用现配。

注：可依据实际待测样品情况，在线性范围内适当调整标准溶液浓度范围。

# 4 仪器和设备

4.1 液相色谱仪：配有紫外检测器或二极管阵列检测器。  
4.2 天平：感量分别为 $0.01\mathrm{mg}$ 和 $0.1\mathrm{mg}$   
4.3 pH计：精度为0.1。  
4.4 涡旋混合器：转速不低于 $1500\mathrm{r / min}$   
4.5 离心机：转速不低于 $5000\mathrm{r / min}$   
4.6 超声波清洗仪。

# 5 分析步骤

# 5.1 试样制备

液体样品和浆体样品摇匀；半固态样品和粉状样品混匀；胶基糖果等胶状样品需剪刀剪碎或冷冻状态粉碎混匀；其他样品需匀浆或粉碎混匀。水果和蔬菜及其制品(取可食部分)需 $10:1$ 加入 $150 \mathrm{~g} / \mathrm{L}$ 的 TCA 溶液或 $200 \mathrm{~g} / \mathrm{L}$ 的偏磷酸溶液匀浆或粉碎均匀。

注：处理过程应尽可能在避光条件下进行；所有样品制备后立即进行下一步试验。

# 5.1.1 试样称取

# 5.1.1.1 固体（含半固体）样品

称取制备后的试样 $25\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )于锥形瓶中，加入TCEP溶液 $225\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )，充分溶解混匀。称取混匀后的溶液 $2\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于 $10~\mathrm{mL}$ 容量瓶中。

# 5.1.1.2 液体（含浆体）样品

称取制备后的试样 $2\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确吸取 $2\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中。

# 5.1.2 试样提取与还原

于上述 $10\mathrm{mL}$ 容量瓶中，加入TCEP溶液(或L-半胱氨酸溶液) $4\mathrm{mL}$ ，再加入TCA溶液 $2\mathrm{mL}$ 混匀，静置反应 $5\mathrm{min}$ ，用水定容，混匀。然后转移至离心管， $5000\mathrm{r / min}$ 离心 $1\mathrm{min}$ ，此样液为还原样液。

注：氨基酸配方粉、蛋白水解配方粉、部分蛋白水解配方粉等特殊医学用途配方食品和羊乳粉应使用L-半胱氨酸作为还原剂。

# 5.1.3 试样稀释

吸取离心后的上清液 $1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，加乙酸铵溶液 $1\mathrm{mL}$ ，用流动相定容，混匀，此样液为稀释样液。取适量过 $0.22\mu \mathrm{m}$ 水系微孔滤膜至棕色进样瓶中，供液相色谱测定。

注：必要时，可改变稀释倍数，确保待测液中抗坏血酸浓度在线性范围内。

# 5.2 仪器参考条件

5.2.1 色谱柱： $\mathrm{C_{18}}$ 柱（柱长 $150~\mathrm{mm}$ ，内径 $4.6~\mathrm{mm}$ ，填料粒径 $4\mu \mathrm{m}$ ）或性能相当的色谱柱。  
5.2.2 柱温：室温。  
5.2.3 流动相：按照3.2.6配制。  
5.2.4 流速： $1.0~\mathrm{mL / min}$   
5.2.5 检测波长： $265~\mathrm{nm}$   
5.2.6 进样量： $10~\mu \mathrm{L}$

# 5.3 标准曲线的制作

将标准系列工作液分别注入液相色谱仪中，得到相应的峰面积，以标准系列工作液中L-抗坏血酸或D-异抗坏血酸的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。L-抗坏血酸或D-异抗坏血酸标准溶液的色谱图参见附录A中图A.1。

# 5.4 试样溶液的测定

将试样溶液注入液相色谱仪中，得到抗坏血酸的峰面积，根据标准曲线得到待测液中抗坏血酸的浓度。

# 6 分析结果的表述

试样中L-抗坏血酸或D-异抗坏血酸的含量按公式(1)计算：

$$
X = \frac {\rho \times M _ {0} \times V _ {1} \times V _ {3} \times f \times 1 0 0}{m _ {0} \times m \times V _ {2} \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

X ——试样中L-抗坏血酸或D-异抗坏血酸的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ 或毫克每百毫升 $(\mathrm{mg} / 100\mathrm{mL})$ ；

$\rho$ ——由标准曲线得到的试样溶液中L-抗坏血酸或D-异抗坏血酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$M_0$ ——固体(含半固体)样品加TCEP溶液溶解后的总质量，单位为克(g)；

$V_{1}$ ——还原样液的总体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——稀释样液的总体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样稀释因子；

100 ——换算系数；

$m_0$ ——固体(含半固体)样品的称样量，单位为克(g)；

$m$ ——液体(含浆体)样品或制备的样液的取样量，单位为克(g)或毫升(mL)；

$V_{2}$ ——还原样液的取样体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数。

计算结果保留3位有效数字。

注1：水果和蔬菜及其制品试样的稀释因子需按样品与保护剂质量比计算。

注2：L-抗坏血酸与D-异抗坏血酸之和为抗坏血酸含量。

注3：抗坏血酸盐的含量可根据相对分子质量换算。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

固体(含半固体)样品，取样量 $25\mathrm{g}$ 时，L-抗坏血酸和D-异抗坏血酸的检出限均为 $1.00\mathrm{mg} / 100\mathrm{g}$ ，定量限均为 $3.00\mathrm{mg} / 100\mathrm{g}$ 。

液体（含浆体）样品，取样量 $2\mathrm{g}$ （或 $2\mathrm{mL}$ 时，L-抗坏血酸和D-异抗坏血酸的检出限均为 $0.100\mathrm{mg / 100g}$ （或 $0.100\mathrm{mg / 100mL}$ )，定量限均为 $0.300\mathrm{mg / 100g}$ （或 $0.300\mathrm{mg / 100mL}$ ）。

# 第二法 荧光分光光度法

# 9 原理

试样中抗坏血酸在活性炭催化作用下，被氧化为脱氢抗坏血酸后与邻苯二胺(OPDA)反应生成有荧光的喹唔啉，其荧光强度与抗坏血酸的浓度在一定条件下成正比，以此测定试样中抗坏血酸的含量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 淀粉酶：酶活力 $1.5\mathrm{U} / \mathrm{mg}$

10.1.2 三水合乙酸钠 $\mathrm{(C_2H_3O_2Na\cdot 3H_2O)}$   
10.1.3 偏磷酸 $\mathrm{(HPO_3)}$ 。  
10.1.4 乙酸溶液 $(36\%)$ 。  
10.1.5 盐酸溶液 $(12\mathrm{mol} / \mathrm{L})$ 。  
10.1.6 硼酸 $(\mathrm{H}_3\mathrm{BO}_3)$ 。  
10.1.7 邻苯二胺 $(\mathrm{C_6H_8N_2})$ 。  
10.1.8 亚铁氰化钾 $\left[\mathrm{K}_4\mathrm{Fe}(\mathrm{CN})_6\right]$   
10.1.9 活性炭/酸性活性炭(C)。

# 10.2 试剂配制

10.2.1 偏磷酸-乙酸溶液 A: 称取偏磷酸 $15 \mathrm{~g}$ 并量取 $36 \%$ 乙酸溶液 $40 \mathrm{~mL}$ 于 $200 \mathrm{~mL}$ 水中, 溶解并稀释至 $500 \mathrm{~mL}$ 。  
10.2.2 偏磷酸-乙酸溶液 B: 称取偏磷酸 $15 \mathrm{~g}$ 并量取 $36 \%$ 乙酸溶液 $40 \mathrm{~mL}$ 于 $100 \mathrm{~mL}$ 水中, 溶解并稀释至 $250 \mathrm{~mL}$ 。  
10.2.3 乙酸钠溶液：称取三水合乙酸钠 $500\mathrm{g}$ ，用水溶解并稀释至 $1\mathrm{L}$ 。  
10.2.4硼酸-乙酸钠溶液：称取硼酸 $3\mathrm{g}$ ，用乙酸钠溶液溶解并稀释至 $100~\mathrm{mL}$ ，临用现配。  
10.2.5 邻苯二胺溶液 $(400\mathrm{mg / L})$ ：称取邻苯二胺 $40~\mathrm{mg}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ ，临用现配。  
10.2.6 盐酸溶液 $(10\%)$ ：量取 $12\mathrm{mol} / \mathrm{L}$ 的盐酸溶液 $274~\mathrm{mL}$ ，用水溶解并稀释至 $1\mathrm{L}$ 。  
10.2.7 盐酸溶液 $(1\%)$ ：吸取 $12\mathrm{mol} / \mathrm{L}$ 的盐酸溶液 $2.74~\mathrm{mL}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
10.2.8 亚铁氰化钾溶液 $(20\mathrm{g / L})$ ：称取亚铁氰化钾 $2\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
10.2.9 酸性活性炭：称取活性炭约 $200\mathrm{g}$ 于大烧杯中，加入 $10\%$ 的盐酸溶液 $1\mathrm{L}$ ，加热至沸腾，减压过滤，取下结块置于另一个大烧杯中，用玻璃棒打散活性炭结块，用水清洗（清洗过程中应不断搅拌）至滤液中无铁离子为止，在 $105^{\circ}\mathrm{C}$ 烘箱中干燥约 $10\mathrm{h}$ 后使用。

注：采用商品化酸性活性炭,可减免酸洗过程,需检验铁离子进行验收,检验方法见附录 B。

# 10.3 标准品

L-抗坏血酸标准品 $\left(\mathrm{C}_{6} \mathrm{H}_{8} \mathrm{O}_{6}, \mathrm{CAS}\right.$ 号：50-81-7）：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准品。

# 10.4 标准溶液配制（200 $\mu \mathrm{g / mL}$ ）

称取L-抗坏血酸标准品 $20\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )，用偏磷酸-乙酸溶液A溶解并定容至 $100~\mathrm{mL}$ ，临用现配。

# 11 仪器和设备

11.1 荧光分光光度计。  
11.2 天平：感量为 $0.01\mathrm{mg}, 0.1\mathrm{mg}$ 和 $0.1\mathrm{g}$ 。  
11.3 烘箱：精度 $1^{\circ}\mathrm{C}$ 。  
11.4 恒温培养箱：精度 $1^{\circ} \mathrm{C}$ 。

# 12 分析步骤

处理过程应尽可能在避光条件下进行。

# 12.1 试样制备

液体样品和浆体样品摇匀；半固态样品和粉状样品混匀；其他样品需匀浆或粉碎均匀；水果和蔬菜及其制品(取可食部分)需 $1:1$ 加入偏磷酸-乙酸溶液匀浆或粉碎均匀。

# 12.1.1 特殊膳食用食品和乳及乳制品

# 12.1.1.1 含淀粉试样

称取制备后的固体(含半固体)试样 $5\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 或称取制备后的液体(含浆体)试样 $10\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 于碘量瓶中, 加入淀粉酶 $0.1\mathrm{g}$ , 固体(含半固体)试样加入 $45^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ 的温水 $50\mathrm{mL}$ , 液体(含浆体)试样加入 $45^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ 的温水 $40\mathrm{mL}$ , 混合均匀后, 充氮气排除瓶中空气, 盖上瓶塞, 置于 $45^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温培养箱内 $30\mathrm{min}$ , 取出冷却至室温, 用偏磷酸-乙酸溶液 B 转移并定容至 $100\mathrm{mL}$ 。

注：淀粉酶根据活力单位大小调整用量。

# 12.1.1.2 不含淀粉试样

称取制备后的固体(含半固体)试样 $5\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ), 用偏磷酸-乙酸溶液 A 溶解并定容至 $100\mathrm{mL}$ 。或称取制备后的液体(含浆体)试样 $10\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ), 用偏磷酸-乙酸溶液 A 溶解并定容至 $100\mathrm{mL}$ 。

# 12.1.2 水果和蔬菜及其制品

称取匀浆或粉碎后含保护剂的试样 $20\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）（相当于 $10\mathrm{g}$ 样品），用偏磷酸-乙酸溶液A稀释并定容至 $100~\mathrm{mL}$ 。

注：必要时，可改变稀释倍数，确保待测液中抗坏血酸浓度在线性范围内。

# 12.2 反应液的制备

# 12.2.1 氧化液制备

将上述标准溶液 $(200\mu \mathrm{g / mL})$ 和试样溶液分别转移至锥形瓶中，加入酸性活性炭 $2\mathrm{g}$ ，剧烈振摇 $1\mathrm{min}$ ，过滤至滤液澄清(弃去约 $5\mathrm{mL}$ 初滤液)。即为标准氧化液和试样氧化液。

# 12.2.2 空白液制备

分别准确吸取标准氧化液和试样氧化液 $5\mathrm{mL}$ 于 $50~\mathrm{mL}$ 具塞比色管中，加入硼酸-乙酸钠溶液 $5\mathrm{mL}$ ，室温静置 $30\mathrm{min}$ ，用水定容；然后再分别准确吸取定容后的溶液 $2\mathrm{mL}$ 于 $10~\mathrm{mL}$ 具塞比色管中，加入邻苯二胺溶液 $5\mathrm{mL}$ ，混匀，室温避光放置 $60\mathrm{min}$ 。即为标准空白液和试样空白液。

# 12.2.3 标准工作液制备

准确吸取标准氧化液 $5\mathrm{mL}$ 于 $50~\mathrm{mL}$ 具塞比色管中，加入乙酸钠溶液 $5\mathrm{mL}$ ，室温静置 $30\mathrm{min}$ ，用水定容；分别准确吸取定容后的溶液 $0.05\mathrm{mL}, 0.50\mathrm{mL}, 1.25\mathrm{mL}, 2.50\mathrm{mL}, 5.00\mathrm{mL}$ 和 $10.00\mathrm{mL}$ 于 $10\mathrm{mL}$ 容量瓶中，用水定容。然后分别准确吸取 $2\mathrm{mL}$ 溶液至 $10\mathrm{mL}$ 具塞比色管中，加入邻苯二胺溶液 $5\mathrm{mL}$ ，混匀，室温避光放置 $60\mathrm{min}$ 。即为标准工作液，质量浓度分别为 $1.00\mu \mathrm{g} / \mathrm{mL}, 10.0\mu \mathrm{g} / \mathrm{mL}, 25.0\mu \mathrm{g} / \mathrm{mL}, 50.0\mu \mathrm{g} / \mathrm{mL}, 100\mu \mathrm{g} / \mathrm{mL}$ 和 $200\mu \mathrm{g} / \mathrm{mL}$ 。

注：可依据实际待检测样品情况，在线性范围内适当调整标准溶液浓度范围；样品溶液与标准品溶液同步操作后，不再考虑反应液引起的样品溶液稀释倍数变化和标准溶液浓度的变化。

# 12.2.4 试样待测液制备

准确吸取试样氧化液 $5\mathrm{mL}$ 于 $50~\mathrm{mL}$ 具塞比色管中，加入乙酸钠溶液 $5\mathrm{mL}$ ，室温静置 $30\mathrm{min}$ ，用水定容；然后准确吸取定容后的溶液 $2\mathrm{mL}$ 于 $10~\mathrm{mL}$ 具塞比色管中，加入邻苯二胺溶液 $5\mathrm{mL}$ ，混匀，室温避光放置 $60\mathrm{min}$ 。即为试样待测液。

# 12.3 测定

# 12.3.1 标准曲线的制作

将标准空白液和标准工作液依次转入比色皿中，于激发波长 $350\mathrm{nm}$ 、发射波长 $430\mathrm{nm}$ 条件下分别测定其荧光响应值。以标准工作液荧光响应值分别减去标准空白液荧光响应值为纵坐标，对应的抗坏血酸浓度为横坐标，绘制标准曲线。

# 12.3.2 试样溶液的测定

将试样空白液和试样待测液依次转入比色皿中，于激发波长 $350\mathrm{nm}$ 、发射波长 $430\mathrm{nm}$ 条件下分别测定其荧光响应值。试样待测液荧光响应值减去试样空白液荧光响应值后在标准曲线上查得对应抗坏血酸的浓度。

注1：比色皿放入比色室后，应在3s内完成荧光值测定，避免荧光淬灭对测定结果的影响。  
注2：脱氢抗坏血酸与硼酸可形成复合物而不与OPDA反应，以此排除试样中荧光杂质产生的干扰，此方法不适用于试样空白液响应值远高于待测物响应值的样品。

# 13 分析结果的表述

试样中抗坏血酸的含量按公式(2)计算：

$$
X = \frac {c \times V \times f \times 1 0 0}{m \times 1 0 0 0} \quad \dots \tag {2}
$$

$X$ ——试样中抗坏血酸的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$c$ ——由标准曲线查得的试样测定液中抗坏血酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样稀释因子；

100 ——换算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

计算结果保留3位有效数字。

注：水果和蔬菜及其制品试样的稀释因子需按样品与保护剂质量比计算。

# 14 精密度

在重复性条件下获得两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

固体(含半固体)样品，取样量 $5\mathrm{g}$ 时，检出限为 $3.00\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $10.0\mathrm{mg} / 100\mathrm{g}$ 。

液体(含浆体)样品，取样量 $10\mathrm{g}$ 时，检出限为 $1.50\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $5.00\mathrm{mg} / 100\mathrm{g}$ 。

# 第三法 2,6-二氯靛酚滴定法

# 16 原理

用蓝色的碱性染料2,6-二氯靛酚标准溶液对含还原型抗坏血酸的试样酸性浸出液进行氧化还原滴定，2,6-二氯靛酚被还原为无色，当到达滴定终点时，多余的2,6-二氯靛酚在酸性介质中显浅红色，由2,6-二氯靛酚的消耗量计算样品中还原型抗坏血酸的含量。

# 17 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 17.1 试剂

17.1.1 偏磷酸 $(\mathrm{HPO}_3) : \geqslant 38\%$   
17.1.2 草酸 $(\mathrm{C_2H_2O_4})$ 。  
17.1.3 碳酸氢钠 $\mathrm{(NaHCO_3)}$ 。  
17.1.4 2,6-二氯靛酚(2,6-二氯靛酚钠盐， $\mathrm{C_{12}H_6Cl_2NNaO_2}$ ）。  
17.1.5 白陶土(或高岭土)：对抗坏血酸无吸附性。

# 17.2 试剂配制

17.2.1 偏磷酸溶液 $(20\mathrm{g / L})$ ：称取偏磷酸 $20\mathrm{g}$ ，用水溶解并定容至 $1\mathrm{L}$ 。

17.2.2 草酸溶液 $(20\mathrm{g / L})$ ：称取草酸 $20\mathrm{g}$ ，用水溶解并定容至 $1\mathrm{L}$ 。

17.2.3 2,6-二氯靛酚(2,6-二氯靛酚钠盐)溶液。

称取碳酸氢钠 $52\mathrm{mg}$ 溶解在 $200~\mathrm{mL}$ 热蒸馏水中，然后称取2,6-二氯靛酚 $50\mathrm{mg}$ 溶解在上述碳酸氢钠溶液中。冷却并用水定容至 $250~\mathrm{mL}$ ，过滤至棕色瓶内，于 $4^{\circ}\mathrm{C}$ 环境中保存。每次使用前，用L-抗坏血酸标准溶液标定其滴定度。

标定方法：准确吸取L-抗坏血酸标准溶液 $1\mathrm{mL}$ 于 $50~\mathrm{mL}$ 锥形瓶中，加入偏磷酸溶液或草酸溶液 $10~\mathrm{mL}$ ，摇匀，用2,6-二氯靛酚溶液滴定至粉红色，保持 $15\mathrm{s}$ 不褪色为止。同时另取偏磷酸溶液或草酸溶液 $10~\mathrm{mL}$ 做空白试验。2,6-二氯靛酚溶液的滴定度按公式(3)计算：

$$
T = \frac {c \times V}{V _ {1} - V _ {0}} \tag {3}
$$

式中：

$T$ ——2,6-二氯靛酚溶液的滴定度，即每毫升2,6-二氯靛酚溶液相当于抗坏血酸的毫克数，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$c$ ——L-抗坏血酸标准溶液的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V$ ——吸取L-抗坏血酸标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——滴定L-抗坏血酸标准溶液所消耗2,6-二氯靛酚溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白所消耗2，6-二氯靛酚溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

# 17.3 标准品

L-抗坏血酸标准品 $\mathrm{(C_6H_8O_6}$ ,CAS号：50-81-7)：纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的

标准品。

# 17.4 标准溶液配制

L-抗坏血酸标准溶液 $(1.0\mathrm{mg / mL})$ ：称取L-抗坏血酸标准品 $100\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )，溶于偏磷酸溶液或草酸溶液并定容至 $100~\mathrm{mL}$ ，临用现配。

# 18 分析步骤

整个检测过程应在避光条件下进行。

# 18.1 试液制备

称取具有代表性样品的可食部分 $100\mathrm{g}$ ，放入粉碎机中，加入偏磷酸溶液或草酸溶液 $100\mathrm{g}$ ，迅速捣成匀浆。准确称取匀浆样品 $10\mathrm{g} \sim 40\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）于烧杯中，用偏磷酸溶液或草酸溶液将样品转移至 $100\mathrm{mL}$ 容量瓶，并稀释至刻度，摇匀后过滤。若滤液有颜色，可按每克样品加 $0.4\mathrm{g}$ 白陶土脱色后再过滤。

# 18.2 滴定

准确吸取滤液 $10\mathrm{mL}$ 于 $50~\mathrm{mL}$ 锥形瓶中，用标定过的2,6-二氯靛酚溶液滴定，直至溶液呈粉红色 $15\mathrm{s}$ 不褪色为止。同时做空白试验。

注：此方法不适用于脱色后本底颜色干扰滴定终点观察的样品。

# 19 结果计算

试样中还原型抗坏血酸的含量按公式(4)计算：

$$
X = \frac {\left(V - V _ {0}\right) \times T \times A}{m} \times 1 0 0 \tag {4}
$$

式中：

$X$ ——试样中还原型抗坏血酸的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$V$ ——滴定试样所消耗2，6-二氯靛酚溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白所消耗2，6-二氯靛酚溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$T$ ——2,6-二氯靛酚溶液的滴定度，即每毫升2,6-二氯靛酚溶液相当于还原型抗坏血酸的毫克数 $(\mathrm{mg / mL})$ ；

$A$ ——稀释倍数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

计算结果保留3位有效数字。

# 20 精密度

在重复性条件下获得的两次独立测定结果的绝对差值，含量 $>20\mathrm{mg} / 100\mathrm{g}$ 时，不得超过算术平均值的 $2\%$ ；含量 $\leqslant 20\mathrm{mg} / 100\mathrm{g}$ 时，不得超过算术平均值的 $5\%$ 。

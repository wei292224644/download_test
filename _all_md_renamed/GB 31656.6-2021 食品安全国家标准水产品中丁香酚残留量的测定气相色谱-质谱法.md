# 食品安全国家标准

# 水产品中丁香酚残留量的测定 气相色谱-质谱法

# 1 范围

本文件规定了水产品中丁香酚残留量检测的制样和气相色谱质谱测定方法。

本文件适用于鱼、虾可食组织中丁香酚残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

GB/T30891—2014 水产品抽样规范

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中残留的丁香酚经乙腈提取，固相萃取柱净化，正己烷除脂，气相色谱质谱法测定，内标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.2 正己烷 $(\mathrm{C_6H_{14}})$ ：色谱纯。

5.1.3 乙酸乙酯 $\mathrm{(CH_3COOCH_2CH_3)}$ ：色谱纯。

5.1.4 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ ：色谱纯。

# 5.2 溶液配制

5.2.1 甲醇溶液：取甲醇 $10\mathrm{mL}$ ，加水至 $100~\mathrm{mL}$

5.2.2 乙腈饱和正己烷：取正己烷 $200~\mathrm{mL}$ 于 $250~\mathrm{mL}$ 分液漏斗，加适量乙腈，剧烈振摇，待分配平衡后，弃去下层乙腈层，即得。

# 5.3 标准品

5.3.1 丁香酚(Eugenol, $\mathrm{C_{10}H_{12}O_2}$ ,CAS号：97-53-0)，含量 $\geqslant 99\%$ 。

5.3.2 氘代丁香酚(Eugenol- $\mathrm{d}_3$ ， $\mathrm{C_{10}H_9D_3O_2}$ ,CAS号：1335401-17-6)，含量 $\geq 99\%$ 。

# 5.4 标准溶液配制

5.4.1 丁香酚标准储备液：取丁香酚标准品 $10\mathrm{mg}$ ，精密称定，用乙酸乙酯溶解并定容于 $100\mathrm{mL}$ 棕色容量瓶，制成浓度为 $100\mu \mathrm{g} / \mathrm{mL}$ 标准储备液。 $-18^{\circ}\mathrm{C}$ 以下避光保存，有效期2个月。

5.4.2 内标储备液：取氘代丁香酚标准品 $10\mathrm{mg}$ ，精密称定，用乙酸乙酯溶解并定容于 $100~\mathrm{mL}$ 棕色容量瓶，制成浓度为 $100~\mu \mathrm{g / mL}$ 的内标储备液。 $-18^{\circ}C$ 以下避光保存，有效期2个月。

5.4.3 丁香酚标准工作液：准确移取丁香酚标准储备液适量，用乙酸乙酯稀释为浓度 $1\mu \mathrm{g / mL}$ 的标准工

作液。 $4^{\circ} \mathrm{C}$ 下避光保存，有效期1个月。

5.4.4 内标工作液：取内标储备液适量，用乙酸乙酯稀释浓度约为 $1\mu \mathrm{g / mL}$ 的标准工作液。 $4^{\circ}C$ 下避光保存，有效期1个月。

5.5 材料

5.5.1固相萃取柱 $(\mathrm{C_{18}}):300~\mathrm{mg / 3~mL}$ ，或相当者。  
5.5.2 具塞塑料离心管： $50~\mathrm{mL}$   
5.5.3 尼龙微孔滤膜： $0.22\mu \mathrm{m}$

6 仪器与设备

6.1 气相色谱-串联质谱仪：配电子轰击电离离子源。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和感量 $0.00001\mathrm{g}$   
6.3 氮吹仪。  
6.4 涡旋混合器。  
6.5 离心机： $4000\mathrm{r / min}$   
6.6 超声波振荡器  
6.7 均质机。

7 试料的制备与保存

7.1 试料的制备

按GB/T30891—2014中附录B的要求制样。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

8 测定步骤

8.1 提取

取试料约 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，于 $50~\mathrm{mL}$ 具塞塑料离心管中，加内标工作液 $0.1\mathrm{mL}$ 、乙腈 $10~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，超声 $10\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。取上层液于另一 $50~\mathrm{mL}$ 具塞塑料离心管中，残渣用乙腈 $10~\mathrm{mL}$ 重复提取一次，合并上清液。加乙腈饱和正己烷 $10~\mathrm{mL}$ ，脱脂，涡旋， $1\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取乙腈层，按上述方法重复脱脂一次。 $45^{\circ}C$ 水浴氮气吹至近干，加甲醇水溶液 $10~\mathrm{mL}$ 使溶解，备用。

8.2 净化

固相萃取柱依次用甲醇、水各 $3\mathrm{mL}$ 活化。取备用液过柱，加水 $3\mathrm{mL}$ 淋洗，抽干 $5\mathrm{min}$ ，加甲醇 $3\mathrm{mL}$ 洗脱，取洗脱液，氮气吹干。加乙酸乙酯 $1.0\mathrm{mL}$ 使溶解， $0.22\mu \mathrm{m}$ 滤膜过滤，气相色谱质谱测定。

8.3 标准曲线的制备

精密量取丁香酚标准工作液、内标工作液适量，用乙酸乙酯稀释成浓度为 $4\mu \mathrm{g} / \mathrm{L},10\mu \mathrm{g} / \mathrm{L},40\mu \mathrm{g} / \mathrm{L}$ $100~\mu \mathrm{g} / \mathrm{L},400~\mu \mathrm{g} / \mathrm{L},800~\mu \mathrm{g} / \mathrm{L}$ 系列标准工作液。标准工作液中氘代丁香酚的含量均为 $100~\mu \mathrm{g} / \mathrm{L}$ 。现用现配。取系列标准工作液进样，以丁香酚和氘代丁香酚峰面积比值为纵坐标、对应丁香酚浓度为横坐标，绘制标准工作曲线。求回归方程和相关系数。

8.4 测定

8.4.1 气相色谱参考条件

a）进样针： $10.0\mu \mathrm{L}$ ，或相当者；  
b）色谱柱：毛细管柱（填料： $5\%$ -苯基甲基聚硅氧烷，如HP-5，规格： $30\mathrm{m}\times 250\mu \mathrm{m}\times 0.25\mu \mathrm{m})$ ，或相当者；  
c）进样方式：不分流进样；  
d）载气：氦气，纯度 $99.999\%$ ，流速 $1.0~\mathrm{mL / min}$   
e) 进样量: $2.0 \mu \mathrm{L}$ ;   
f）进样口温度： $280^{\circ}C$   
g）柱温：起始温度 $70^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ；以 $40^{\circ}\mathrm{C / min}$ 升温至 $120^{\circ}\mathrm{C}$ ，以 $10^{\circ}\mathrm{C / min}$ 升温至 $170^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ 。

# 8.4.2 质谱参考条件

a）离子源：EI源；  
b）电子能量： $70\mathrm{eV}$   
c）离子源温度： $280^{\circ}C$   
d）四级杆温度： $150^{\circ}\mathrm{C}$   
e) 接口温度: $280^{\circ} \mathrm{C}$ ;   
f）后运行温度： $280^{\circ}\mathrm{C},3\mathrm{min}$   
g）溶剂延迟： $4.0\mathrm{min}$   
h）测定方式：多反应(MRM)监测模式，定性离子对、定量离子对和保留时间参考值详见表1。

# 8.4.3 测定法

取试料溶液和标准溶液，作单点或多点校准，以色谱峰面积定量，内标法计算。标准溶液及试料溶液中目标化合物的特征离子质量色谱峰面积均应在仪器检测的线性范围之内。试料溶液中丁香酚的特征离子质量色谱峰面积均应在仪器检测的线性范围内。试样溶液中的相对离子丰度与标准溶液中的相对离子丰度相比，符合表2的要求。标准溶液特征离子质量色谱图见附录A。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times A _ {i S} ^ {\prime} \times C _ {S} \times C _ {i S} \times V}{A _ {i S} \times A _ {S} \times C _ {i S} ^ {\prime} \times m} \tag {1}
$$

式中：

$X$ ——试样中丁香酚残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\varLambda$ ——试样溶液中丁香酚的峰面积；

$A_{iS}$ ——试样溶液中内标的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中丁香酚的峰面积；

$A_{iS}^{\prime}$ ——标准溶液中内标的峰面积；

$C_{\mathrm{is}}$ ——试样溶液中内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$C_{\mathrm{S}}$ ——标准溶液中丁香酚浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$C_{i\mathrm{S}}^{\prime}$ ——标准溶液中内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $2.5\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $10~\mu \mathrm{g / kg}\sim 200~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$

# 前言

本标准代替GB/T5009.74—2003《食品添加剂中重金属限量试验》。

本标准与GB/T5009.74—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂中重金属限量试验”；

——增加了压力消解罐消解法。

# 食品安全国家标准

# 食品添加剂中重金属限量试验

# 1 范围

本标准规定了食品添加剂中重金属的限量试验方法。

本标准适用于食品添加剂中重金属的限量试验。

# 2 原理

在弱酸性 $(\mathrm{pH}3\sim 4)$ 条件下，试样中的重金属离子与硫化氢作用，生成棕黑色，与同法处理的铅标准溶液比较，做限量试验。

# 3 试剂和材料

注：除非另有说明，本标准所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 硝酸 $(\mathrm{HNO}_3)$ 。  
3.1.2 硫酸 $\mathrm{(H_2SO_4)}$ 。  
3.1.3 盐酸（HCl）。  
3.1.4 氨水 $\mathrm{(NH_3\bullet H_2O)}$ 。  
3.1.5 乙酸铵 $(\mathrm{C}_2\mathrm{H}_7\mathrm{NO}_2)$ 。  
3.1.6 酚酞 $(\mathrm{C}_2\mathrm{OH}_{14}\mathrm{O}_4)$ 。  
3.1.7 过氧化氢 $\mathrm{(H_2O_2)}$ 。  
3.1.8 硫化氢 $(\mathrm{H}_2\mathrm{S})$ 。  
3.1.9 高氯酸 $\mathrm{(HClO_4)}$

# 3.2 试剂配制

3.2.1 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ：量取 $50~\mathrm{mL}$ 盐酸，用水稀释至 $100~\mathrm{mL}$ 。  
3.2.2 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $8.3\mathrm{mL}$ 盐酸，用水稀释至 $100~\mathrm{mL}$ 。  
3.2.3 氨溶液 $(6\mathrm{mol} / \mathrm{L})$ ：量取 $40~\mathrm{mL}$ 氨水，用水稀释至 $100~\mathrm{mL}$ 。  
3.2.4 氨溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $6.7~\mathrm{mL}$ 氨水，用水稀释至 $100~\mathrm{mL}$ 。  
3.2.5 pH3.5的乙酸盐缓冲液：称取 $25.0\mathrm{g}$ 乙酸铵溶于 $25~\mathrm{mL}$ 水中，加 $45~\mathrm{mL}6~\mathrm{mol / L}$ 盐酸溶液，用稀盐酸或稀氨水调节 $\mathrm{pH}$ 至3.5，用水稀释至 $100~\mathrm{mL}$ 。  
3.2.6 酚酞乙醇溶液 $(1\%)$ ：称取 $1.0\mathrm{g}$ 酚酞溶于 $100~\mathrm{mL}$ 乙醇溶液中。  
3.2.7 硫化氢饱和溶液：硫化氢气体通入不含二氧化碳的水中（如流速为 $80~\mathrm{mL / min}$ 左右时，通气 $1\textrm{h}$ 。此溶液临用前制备）。  
3.2.8 硝酸溶液 $(1\%)$ ：取 $1\mathrm{mL}$ 硝酸加水稀释至 $100~\mathrm{mL}$

# 3.3 标准品

高纯硝酸铅 $\left[\mathrm{Pb}\left(\mathrm{NO}_{3}\right)_{2}\right]$ ，纯度为 $99.99\%$ 或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 铅标准储备液 $(1\mathrm{mg / mL})$ ：称取 $0.1598\mathrm{g}$ 高纯硝酸铅 $\left[\mathrm{Pb}(\mathrm{NO}_3)_2\right]$ ，溶于 $10~\mathrm{mL}$ 硝酸溶液（ $1\%$ ）中，定量移入 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度。  
3.4.2 铅标准使用液 $(10\mu \mathrm{g / mL})$ ：吸取铅标准储备液 $(1\mathrm{mg / mL})1.0\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加水至刻度。

# 4 仪器

注：所用玻璃仪器均需以硝酸溶液 $(1 + 4)$ 浸泡 $24\mathrm{h}$ 以上，用水反复冲洗，最后用去离子水冲洗干净。

4.1 $50~\mathrm{mL}$ 纳氏比色管。  
4.2 电热板。  
4.3 可调式电炉。  
4.4 马弗炉。  
4.5 电子天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.6 水浴锅。  
4.7 压力消解罐。

# 5 样品处理

# 5.1 无机试样的制备

无机试样的“试样处理”可按各产品质量规格的要求进行溶解或消化等前处理。试验应在无元素污染的通风柜中进行。

# 5.2 有机试样的制备

有机试样的“试样处理”除按各产品质量规格的要求外，一般可按下述方法进行：

a）湿法消解：称取 $5.00\mathrm{g}$ 试样，置于 $25~\mathrm{mL}$ 锥形瓶中，加 $10~\mathrm{mL} \sim 15~\mathrm{mL}$ 硝酸浸润试样，放置片刻(或过夜)后，于电热板上加热，待反应缓和后取下放冷，沿瓶壁加入 $5\mathrm{mL}$ 硫酸，再继续加热至瓶中溶液开始变成棕色，不断滴加硝酸(如有必要可滴加些高氯酸)至有机质分解完全，继续加热至生成大量的二氧化硫白色烟雾，最后溶液应呈无色或微黄色。冷却后加 $20~\mathrm{mL}$ 水，煮沸除去残余的硝酸至产生白烟为止。如此处理两次，放冷。将溶液移入 $50~\mathrm{mL}$ 容量瓶中，用水洗涤锥形瓶，将洗涤液并入容量瓶中，加水定容至刻度，混匀。每 $10~\mathrm{mL}$ 溶液相当于 $1.0\mathrm{g}$ 样品。取同样量的硝酸、硫酸，同时做试剂空白试验。  
b）干法消解：称取 $5.00\mathrm{g}$ 试样，置于硬质玻璃蒸发皿或石英坩埚中，加入适量硫酸浸润试样，于电炉上小火炭化后，加 $2\mathrm{mL}$ 硝酸和5滴硫酸，小心加热直到白色烟雾挥尽，移入马弗炉中，于 $500^{\circ}C$ 灰化完全，冷却后取出，加 $2\mathrm{mL}$ 盐酸溶液 $(6\mathrm{mol / L})$ 湿润残渣，于水浴上慢慢蒸发至干。用1滴浓盐酸湿润残渣，并加 $10~\mathrm{mL}$ 水，于沸水浴上再次加热 $2\mathrm{min}$ ，将溶液移入 $50~\mathrm{mL}$ 容量瓶中，如有必要应过滤，用少量水洗涤坩埚和滤器，洗滤液一并移入容量瓶中，定容后混匀，每 $10~\mathrm{mL}$ 该溶液相当于 $1.0\mathrm{g}$ 试样。在试样灰化同时，另取一坩埚，同时做试剂空白试验。  
c）压力消解罐消解法：按压力消解罐使用说明书称取适量试样(精确至 $0.001\mathrm{g}$ )于聚四氟乙烯

内罐，加硝酸 $2\mathrm{mL}\sim 4\mathrm{mL}$ 浸泡过夜。再加过氧化氢 $2\mathrm{mL}\sim 3\mathrm{mL}$ （总量不能超过罐容积的1/3)。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱， $120^{\circ}C\sim 140^{\circ}C$ 保持 $3\mathrm{h}\sim 4\mathrm{h}$ ，在箱内自然冷却至室温，用滴管将消化液洗入或过滤入（视消化后试样的盐分而定）容量瓶中，用水少量多次洗涤罐，洗液合并于容量瓶中并定容至刻度，混匀备用；同时作试剂空白。

# 6 测定

6.1 A管(标准管)：吸取含铅量相当于指定的重金属限量的铅标准使用液（不低于 $10\mu \mathrm{g}$ 铅）于 $50~\mathrm{mL}$ 纳氏比色管中（如试样经处理，应同时吸取与试样液等量的试剂空白液），加水至 $25~\mathrm{mL}$ ，混匀，加1滴酚酞指示液，用稀盐酸 $(6\mathrm{mol} / \mathrm{L})$ 或稀氨水 $(1\mathrm{mol} / \mathrm{L})$ 调节 $\mathrm{pH}$ 至中性（酚酞红色刚褪去），加入 $5\mathrm{mL}$ $\mathrm{pH}3.5$ 的乙酸盐缓冲液，混匀，备用。  
6.2 B管(样品管)：取一支与A管所配套的纳氏比色管，加入 $10\mathrm{mL}\sim 20\mathrm{mL}$ （或适量)试样液，加水至 $25~\mathrm{mL}$ ，混匀，加1滴 $1 \%$ 酚酞指示液，用稀盐酸 $(6\mathrm{mol} / \mathrm{L})$ 或稀氨水 $(1\mathrm{mol} / \mathrm{L})$ 调节 $\mathsf{pH}$ 至中性（酚酞红色刚褪去)，加入 $5\mathrm{mL}$ pH3.5的乙酸盐缓冲液，混匀，备用。  
6.3 C管：取一支与A、B管所配套的纳氏比色管，加入与B管等量的相同的试样液，再加入与A管等量的铅标准使用液 $(10\mu \mathrm{g} / \mathrm{mL})$ ，加水至 $25~\mathrm{mL}$ ，混匀，加1滴 $1\%$ 酚酞指示液，用稀盐酸(6mol/L)或稀氨水 $(1\mathrm{mol} / \mathrm{L})$ 调节 $\mathsf{pH}$ 至中性(酚酞红色刚褪去)，加入 $5\mathrm{mL}$ pH3.5的乙酸盐缓冲液，混匀，备用。  
6.4 向各管中加入 $10 \mathrm{~mL}$ 新鲜制备的硫化氢饱和溶液，并加水至 $50 \mathrm{~mL}$ 刻度，混匀，于暗处放置 $5 \mathrm{~min}$ 后，在白色背景下观察，B 管的色度不得深于 A 管的色度，C 管的色度应与 A 管的色度相当或深于 A 管的色度，则可判定为样品中重金属含量（以铅计）低于 A 管（标准管）对应的重金属含量（以铅计）。

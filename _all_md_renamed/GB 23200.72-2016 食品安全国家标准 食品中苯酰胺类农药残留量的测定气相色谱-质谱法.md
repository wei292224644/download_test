本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T3143—2012《出口食品中苯酰胺类农药残留量的测定气相色谱一质谱法》。与SN/T3143—2012相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；

# 食品安全国家标准

# 食品中苯酰胺类农药残留量的测定 气相色谱一质谱法

# 1 范围

本标准规定了食品中25种苯酰胺类农药残留量(参见附录A)的气相色谱—质谱检测方法。

本标准适用于玉米、菠菜、蘑菇、苹果、大豆、板栗、茶叶、牛肉、牛肝、鸡肉、鱼肉、牛奶中25种苯酰胺类农药残留量的测定和确证，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

试样用丙酮一正己烷振荡提取，石墨化炭黑固相萃取柱或中性氧化铝固相萃取柱净化，气相色谱一质谱仪测定和确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
4.1.2 丙酮 $\mathrm{(CH_3COCH_3)}$ ：色谱纯。  
4.1.3 氯化钠 $\mathrm{(NaCl)}$ 。  
4.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置干燥器中备用。

# 4.2 溶液配制

4.2.1 丙酮—正己烷溶液 $(1 + 2$ ，体积比)：取 $100~\mathrm{mL}$ 丙酮，加入 $200~\mathrm{mL}$ 正己烷，摇匀备用。  
4.2.2 丙酮—正己烷溶液 $(1 + 1$ ，体积比)：取 $200~\mathrm{mL}$ 丙酮，加入 $200~\mathrm{mL}$ 正己烷，摇匀备用。

# 4.3 标准品

4.3.1 苯酰胺类农药标准物质：参见附录A。

# 4.4 标准溶液配制

4.4.1 苯酰胺类农药标准储备溶液：分别准确称取适量的苯酰胺类农药标准物质，用丙酮配制成 $1000\mu \mathrm{g / mL}$ 标准储备液，标准溶液避光于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 保存，保存期为6个月。  
4.4.2 苯酰胺类农药标准工作溶液：根据检测要求，分别量取上述各标准储备液于同一容量瓶中，用正己烷稀释到刻度配制成适当浓度的标准工作溶液，标准溶液避光于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 保存，保存期为一个月。

# 4.5 材料

4.5.1 石墨化炭黑固相萃取柱： $6\mathrm{mL} / 500\mathrm{mg}$ ，或相当者。

4.5.2 中性氧化铝固相萃取柱： $6\mathrm{mL} / 500\mathrm{mg}$ ，或相当者。

# 5 仪器和设备

5.1 气相色谱—质谱联用仪，配电子轰击离子源（EI源）。  
5.2 组织捣碎机。  
5.3 粉碎机。  
5.4 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.5 涡旋振荡器。  
5.6 固相萃取装置，带真空泵。  
5.7 离心机： $6000\mathrm{r / min}$   
5.8 离心管： $50~\mathrm{mL}$   
5.9 刻度试管： $15\mathrm{mL}$   
5.10 微量注射器： $10~\mu \mathrm{L}$

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按 GB 2763 的规定执行。在取样和制样的操作过程中,应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 水果、蘑菇或蔬菜

取有代表性样品 $500\mathrm{g}$ , 切碎后(不可用水洗), 用捣碎机将样品加工成浆状。混匀, 装入洁净的盛样容器内, 密封并标明标记。

# 6.2.2 茶叶、大豆、板栗、玉米

取有代表性样品 $500\mathrm{g}$ , 用粉碎机粉碎。混匀, 装入洁净的盛样容器内, 密封并标明标记。

# 6.2.3 牛奶

将样品搅拌均匀, 分出 $500 \mathrm{~g}$ 作为试样。制备好的试样均分成 2 份, 分别装入样品瓶中, 密封, 并标明标记。

# 6.2.4 牛肉、牛肝、鱼肉、鸡肉

取有代表性样品 $500\mathrm{g}$ ，经捣碎机充分捣碎均匀，装入洁净的盛样容器内，密封并标明标记。

# 6.3 试样保存

茶叶、大豆、板栗、粮谷类等试样于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 保存；牛奶、蘑菇、水果蔬菜类和肉及肉制品类等试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 7 分析步骤

# 7.1 提取

对于茶叶、大豆、板栗、玉米样品，称取 $2.5\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）。对于菠菜、蘑菇、苹果、牛肉、牛肝、鸡肉、鱼肉、牛奶样品，称取 $5\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）。将称取的试样置于 $50\mathrm{mL}$ 离心管中，加入 $6\mathrm{mL}$ 饱和氯化钠水溶液，于涡旋振荡器上混匀 $30\mathrm{s}$ ，放置 $15\mathrm{min}$ 。加入 $6\mathrm{mL}$ 丙酮—正己烷溶液，在涡旋振荡器上混匀 $2\mathrm{min}$ 。 $5000\mathrm{r/min}$ 离心 $1\mathrm{min}$ ，吸取上层提取液于另一试管中。再分2次，每次加入 $4\mathrm{mL}$ 丙酮—正己烷溶液重复提取，合并提取液，在 $45^{\circ}\mathrm{C}$ 下氮气流吹至约 $1\mathrm{mL}$ 待净化。

# 7.2 净化

# 7.2.1 茶叶、菠菜、蘑菇、苹果

将石墨化炭黑固相萃取柱(柱内填约 $1\mathrm{cm}$ 高的无水硫酸钠层)安装在固相萃取的真空抽滤装置上，先用 $6\mathrm{mL}$ 丙酮一正己烷(4.2.2)预淋洗萃取柱，弃去全部预淋洗液。将提取液加入到石墨化炭黑固相萃取柱中，待提取液全部流出后，再用 $8\mathrm{mL}$ 丙酮一正己烷混合液洗脱萃取柱，保持流速 $1.5\mathrm{mL} / \mathrm{min}$ ，收集全部流出液， $45^{\circ}\mathrm{C}$ 下氮气流吹至近干。用正己烷定容至 $0.5\mathrm{mL}$ ，供GC-MS测定。

# 7.2.2 玉米、大豆、板栗、牛肉、牛肝、鸡肉、鱼肉、牛奶

将中性氧化铝固相萃取柱(柱内填约 $1\mathrm{cm}$ 高的无水硫酸钠层)安装在固相萃取的真空抽滤装置上，先用 $6\mathrm{mL}$ 丙酮一正己烷混合液预淋洗萃取柱，弃去全部预淋洗液。将提取液加入到中性氧化铝固相萃取柱中，待提取液全部流出后，再用 $5\mathrm{mL}$ 丙酮一正己烷混合液洗脱萃取柱，保持流速 $1.5\mathrm{mL} / \mathrm{min}$ ，收集全部流出液， $45^{\circ}\mathrm{C}$ 下氮气流吹至近干。用正己烷定容至 $0.5\mathrm{mL}$ ，供GC-MS测定。

# 7.3 测定

# 7.3.1 气相色谱一质谱参考条件

a）色谱柱：HP-1701MS石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径），膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度： $100^{\circ}\mathrm{C}$ （保持 $1\mathrm{min})\xrightarrow{10^{\circ}\mathrm{C / min}}280^{\circ}\mathrm{C}$ （保持 $11\mathrm{min})$   
c) 进样口温度: $250^{\circ} \mathrm{C}$ ;   
d） 色谱—质谱接口温度： $280^{\circ}C$   
e) 载气：氦气，纯度 $\geqslant 99.995\%$ ，流速 $1.0~\mathrm{mL / min}$   
f）进样量： $1\mu \mathrm{L}$   
g）进样方式：无分流进样， $1\mathrm{min}$ 后开阀；  
h） 电离方式：EI；  
i）电离能量： $70\mathrm{eV}$   
j）检测方式：选择离子监测方式（SIM）；  
k）监测离子 $(m / z)$ ：参见附录A；  
1) 溶剂延迟: $8 \mathrm{~min}$ 。

# 7.3.2 色谱测定与确证

根据样液中苯酰胺类农药的含量情况，选定峰面积相近的标准工作溶液，对标准工作液和样液等体积参插进样，测定标准工作溶液和样液中苯酰胺类农药的响应值均应在仪器检测的线性范围内。

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中各苯酰胺类农药的含量。

$$
X = \frac {A _ {i} \times C _ {s i} \times V}{A _ {s i} \times m}. \tag {1}
$$

式中：

$X$ ——试样中各苯酰胺类农药的残留含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A_{i}$ ——样液中各苯酰胺类农药的峰面积；

$C_{s}$ ——标准工作液中各苯酰胺类农药的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准工作液中各苯酰胺类农药的峰面积；

$m$ ——最终样液所代表的试样量，单位为克(g)。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录D的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 E 的要求。

# 10 定量限和回收率

# 10.1 定量限

本方法苯酰胺类农药的定量限为 $0.01\mathrm{mg / kg}$

# 10.2 回收率

当添加水平为 $0.01 \mathrm{mg} / \mathrm{kg}, 0.05 \mathrm{mg} / \mathrm{kg}, 0.2 \mathrm{mg} / \mathrm{kg}$ 时, 25 种苯酰胺类农药的添加回收率参见附录 F。

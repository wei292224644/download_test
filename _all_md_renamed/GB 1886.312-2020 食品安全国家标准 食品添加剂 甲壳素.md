# 食品安全国家标准

# 食品添加剂 甲壳素

# 1 范围

本标准适用于以虾壳、蟹壳、鲨壳、鱿鱼骨、乌鱼骨等为主要原料，经脱钙、脱蛋白等工艺加工制得的食品添加剂甲壳素。

# 2 分子式和结构式

# 2.1 分子式

$$
\left(\mathrm {C} _ {8} \mathrm {H} _ {1 3} \mathrm {N O} _ {5}\right) _ {n}
$$

$$
n: \text {聚 合 度}, 1 0 0 0 \sim 3 0 0 0
$$

# 2.2 结构式

$$
n = 1 0 0 0 \sim 3 0 0 0
$$

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度应在分析纯以上，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 $1\%$ 乙酸溶液：量取 $1.0~\mathrm{mL}$ 乙酸，用水定容至 $100~\mathrm{mL}$

A.2.1.2 硫酸。

A.2.1.3 盐酸。

A.2.1.4 $0.2\%$ 葱酮硫酸溶液：称取 $0.2\mathrm{g}$ 葱酮，精确至 $0.01\mathrm{g}$ ，溶于 $100~\mathrm{mL}$ 硫酸中，现配现用。

A.2.1.5 氢氧化钠溶液： $1\mathrm{mol} / \mathrm{L}$

A.2.1.6 柠檬酸溶液： $0.1\mathrm{mol} / \mathrm{L}$

A.2.1.7 柠檬酸三钠溶液： $0.1\mathrm{mol} / \mathrm{L}$

A.2.1.8 柠檬酸-柠檬酸三钠缓冲液 $(\mathrm{pH}5.0)$ ：分别量取 $16.4~\mathrm{mL}$ 柠檬酸溶液 $(0.1\mathrm{mol / L})$ 与 $23.6~\mathrm{mL}$ 柠檬酸三钠溶液 $(0.1\mathrm{mol / L})$ ，混匀。若溶液 $\mathsf{pH}$ 发生偏差，需调至 $\mathrm{pH}5.0$ 。

A.2.1.9 乙二醇单甲醚。

A.2.1.10 苷三酮溶液：称取 $1.0\mathrm{g}$ 苷三酮，精确至 $0.01\mathrm{g}$ ，加入 $25\mathrm{mL}$ 乙二醇单甲醚和 $25\mathrm{mL}$ 柠檬酸-柠檬酸三钠缓冲液，混匀。

# A.2.2 鉴别方法

A.2.2.1 称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加入 $200\mathrm{mL}$ 水，混合，试样应不溶解。

A.2.2.2 称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加入 $200\mathrm{mL}1\%$ 乙酸溶液，混合，试样应不溶解。

A.2.2.3 称取 $0.2\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加入 $5\mathrm{mL}0.2\%$ 葱酮硫酸溶液和 $1\mathrm{mL}$ 水，水浴加热，试样溶液应呈蓝色至绿色。

A.2.2.4 称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加入 $10\mathrm{mL}$ 盐酸，置于 $90^{\circ}\mathrm{C} \sim 100^{\circ}\mathrm{C}$ 水浴中放置 $4\mathrm{h}$ ，过滤，滤液用氢氧化钠溶液中和，作为试样溶液。吸取 $1\mathrm{mL}$ 试样溶液，加入 $1\mathrm{mL}$ 苄三酮溶液，加热，溶液应呈紫红色。

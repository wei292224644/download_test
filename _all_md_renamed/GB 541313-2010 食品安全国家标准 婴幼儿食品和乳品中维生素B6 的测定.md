# 前言

本标准代替GB/T5413.13-1997《婴幼儿配方食品和乳粉维生素 $\mathrm{B}_{6}$ 的测定》。

本标准与GB/T5413.13-1997相比，主要变化如下：

——标准名称改为《婴幼儿食品和乳品中维生素B6的测定》；

——完善了维生素 $\mathrm{B}_{6}$ 含量的计算。

本标准附录A为资料性附录。

本标准所代替标准的历次版本发布情况为：

GB5413-1985、GB/T5413.13-1997。

# 食品安全国家标准

# 婴幼儿食品和乳品中维生素 $\mathrm{B}_{6}$ 的测定

# 1 范围

本标准规定了婴幼儿食品和乳品中维生素 $\mathbf{B}_6$ 的测定方法。

本标准适用于婴幼儿食品和乳品中维生素 $\mathrm{B}_{6}$ 的测定。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

# 3 原理

试样经热水提取等前处理后，经 $\mathrm{C}_{18}$ 色谱柱分离，荧光检测器检测，外标法定量测定维生素 $\mathrm{B}_{6}$ （吡哆醇、吡哆醛、吡哆胺）的含量。

# 4 试剂和材料

除非另有规定，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

4.1 淀粉酶：酶活力 $\geqslant 1.5\mathrm{U / mg}$   
4.2 辛烷磺酸钠：优级纯。  
4.3 冰乙酸：优级纯。  
4.4 甲醇：色谱纯。  
4.5 三乙胺：色谱纯。  
4.6 盐酸溶液（5.0 mol/L，0.1 mol/L）。  
4.7 氢氧化钠溶液（ $5.0\mathrm{mol/L}$ ， $0.1\mathrm{mol/L}$ ）。  
4.8 标准溶液  
4.8.1 维生素 $\mathrm{B}_{6}$ （吡哆醇、吡哆醛、吡哆胺）标准贮备液（ $1\mathrm{mg/mL}$ ）：称取 $0.05\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）吡哆醇、吡哆醛、吡哆胺标准品，分别置于 $50\mathrm{mL}$ 容量瓶中，用水溶解定容。  
4.8.2 维生素 $\mathrm{B}_{6}$ （吡哆醇、吡哆醛、吡哆胺）混合标准中间液（ $20\mu \mathrm{g} / \mathrm{mL}$ ）：分别准确吸取吡哆醇、吡哆醛、吡哆胺标准贮备液（4.8.1） $1\mathrm{mL}$ 至 $50\mathrm{mL}$ 容量瓶中，用水定容。  
4.8.3 维生素 $\mathrm{B}_{6}$ （吡哆醇、吡哆醛、吡哆胺）混合标准系列测定液：分别准确吸取维生素 $\mathrm{B}_{6}$ 混合标准中间液（4.8.2） $0.0\mathrm{mL}$ 、 $1.0\mathrm{mL}$ 、 $2.0\mathrm{mL}$ 、 $3.0\mathrm{mL}$ 、 $5.0\mathrm{mL}$ ，至 $100\mathrm{mL}$ 容量瓶中，用水定容。该标准系列浓度分别为 $0.00\mu\mathrm{g/mL}$ 、 $0.20\mu\mathrm{g/mL}$ 、 $0.40\mu\mathrm{g/mL}$ 、 $0.60\mu\mathrm{g/mL}$ 、 $1.00\mu\mathrm{g/mL}$ 。临用前配制。

# 5 仪器和设备

5.1 超声波振荡器。

5.2 高效液相色谱仪，带荧光检测器。  
5.3 天平：感量为 $0.1\mathrm{mg}$   
5.4 pH计：精度为0.01。  
5.5 培养箱： $30^{\circ}\mathrm{C}\sim 80^{\circ}\mathrm{C}$

# 6 分析步骤

# 6.1 试样预处理

6.1.1 含淀粉的试样: 称取混合均匀固体试样约 $5.0 \mathrm{~g}$ (精确至 $0.0001 \mathrm{~g}$ ), 加入约 $25 \mathrm{~mL} 45^{\circ} \mathrm{C} \sim 50^{\circ} \mathrm{C}$ 的水, 或称取混合均匀液体试样约 $20.0 \mathrm{~g}$ (精确至 $0.0001 \mathrm{~g}$ ) 于 $150 \mathrm{~mL}$ 锥形瓶中。加入约 $0.5 \mathrm{~g}$ 淀粉酶 (4.1), 混匀后向锥形瓶中充氮, 盖上瓶塞, 置 $50^{\circ} \mathrm{C} \sim 60^{\circ} \mathrm{C}$ 培养箱内约 $30 \mathrm{~min}$ 。取出冷却至室温。  
6.1.2 不含淀粉的试样：称取混合均匀固体试样约 $5.0\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），加入约 $25\mathrm{mL}45^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 的水，或称取混合均匀液体试样约 $20.0\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）于 $150\mathrm{mL}$ 锥形瓶中。混匀并充分溶解，静置 $5\mathrm{min}\sim 10\mathrm{min}$ ，冷却至室温。

# 6.2 待测液的制备

6.2.1 用盐酸溶液（4.6）调节试样溶液的 $\mathrm{pH}$ 值至 $1.7 \pm 0.1$ ，放置约 $1 \mathrm{~min}$ 。再用氢氧化钠溶液（4.7）调节试样溶液的 $\mathrm{pH}$ 值至 $4.5 \pm 0.1$ 。  
6.2.2 将试样溶液转移至 $50 \mathrm{~mL}$ 容量瓶中，用水冲洗锥形瓶。洗液合并于 $50 \mathrm{~mL}$ 容量瓶中，用水定容至 $50 \mathrm{~mL}$ 。  
6.2.3 把上述容量瓶放入超声波振荡器中，超声振荡约 $10\mathrm{min}$ 。  
6.2.4 另取 $50 \mathrm{~mL}$ 三角瓶，上面放入三角漏斗和滤纸，把超声波振荡的试样溶液倒入其中，自然过滤。滤液再经 $0.45 \mu \mathrm{m}$ 微孔滤膜加压过滤，用试管收集，即为试样待测液。

# 6.3 参考色谱条件

色谱柱： $C_{18}$ 柱（粒径 $5\mu \mathrm{m}$ ， $150\mathrm{mm}\times 4.6\mathrm{mm}$ ）或同等性能的色谱柱。

流动相：甲醇（4.4） $50~\mathrm{mL}$ ，辛烷磺酸钠（4.2） $2.0~\mathrm{g}$ ，三乙胺（4.5） $2.5~\mathrm{mL}$ ，用水溶解并定容到 $1000~\mathrm{mL}$ 后，用冰乙酸（4.3）调pH至 $3.0 \pm 0.1$ ，再经 $0.45~\mu\mathrm{m}$ 微孔滤膜加压过滤。

流速： $1.0\mathrm{mL / min}$

检测波长：激发波长 $293\mathrm{nm}$ ，发射波长 $395\mathrm{nm}$ 。

进样量： $10~\mu \mathrm{L}$

# 6.4 定量分析

# 6.4.1 标准曲线绘制

将维生素 $\mathrm{B}_{6}$ 混合标准系列测定液（4.8.3）依次进行色谱测定（其标准样品色谱图参见附录A中图A.1）。记录各组分的色谱峰面积或峰高，以峰面积或峰高为纵坐标，以标准测定液的浓度为横坐标，绘制标准曲线。

# 6.4.2 试样测定

将试样待测液（6.2.4）进行色谱测定。记录各组分色谱峰面积或峰高，根据标准曲线计算出试样待测液中维生素 $\mathrm{B}_6$ 各组分的浓度 $c_{i}$ 。

# 7 分析结果的表述

# 7.1 试样中维生素 $\mathrm{B}_{6}$ 各组分的含量的计算

试样中维生素 $\mathrm{B}_{6}$ 各组分的含量按式（1）计算：

$$
X _ {i} = \frac {c _ {i} \times V \times 1 0 0}{m} \dots \tag {1}
$$

式中：

$X_{i}$ ——试样中组分的含量, 单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$ ;

$c_{i}$ ——试样待测液中维生素 $\mathrm{B}_{6}$ 各组分的浓度, 单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ;

$m$ ——试样的质量，单位为克（g）；

$V$ ——试样溶液的体积，单位为毫升（ $\mathrm{mL}$ ）。

# 7.2 试样中维生素 $\mathrm{B}_6$ 的含量的计算

试样中维生素 $\mathrm{B}_{6}$ 的含量按式（2）计算：

$$
X = X _ {\text {醇}} + X _ {\text {醛}} \times 1. 0 1 2 + X _ {\text {胺}} \times 1. 0 0 6 \dots \tag {2}
$$

式中：

$X$ ——试样中维生素 $\mathrm{B}_{6}$ 含量, 单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$ ;

$X_{\text{醇}}$ ——试样中吡哆醇含量，单位为微克每百克（ $\mu \mathrm{g} / 100 \mathrm{~g}$ ）；

$X_{\text {醛 }}$ ——试样中吡哆醛含量, 单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$ ;

$X_{\text {胶}}$ ——试样中吡哆胺含量, 单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$ ;

1.012——吡哆醛的含量换算成吡哆醇的系数；

1.006——吡哆胺的含量换算成吡哆醇的系数。

注：试样中维生素 $\mathrm{B}_6$ 含量以吡哆醇计。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 8 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 9 其他

本标准检出限为：吡哆醇 $1.5\mu \mathrm{g} / 100\mathrm{g}$ 、吡哆醛 $1.3\mu \mathrm{g} / 100\mathrm{g}$ 、吡哆胺 $1.6\mu \mathrm{g} / 100\mathrm{g}$ 。

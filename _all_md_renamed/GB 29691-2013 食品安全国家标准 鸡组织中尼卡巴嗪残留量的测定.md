# 高效液相色谱法

# 1 范围

本标准规定了鸡可食性组织中尼卡巴嗪标识残留物4, 4'-二硝基均二苯脲残留量检测的制样和高效液相色谱测定方法。

本标准适用于鸡肌肉、肝脏和肾脏组织中尼卡巴嗪标识残留物 $4,4^{\prime}$ -二硝基均二苯脲残留量的测定。

# 3原理

试料中残留的4,4'-二硝基均二苯脲，用乙腈提取，正己烷除脂， $\mathbf{C}_{18}$ 柱净化，乙腈水溶液洗脱，高效液相色谱测定，外标法定量。

# 4 试剂和材料

以下所用试剂，除特别注明者外，均为分析纯试剂，水为符合GB/T6682规定的一级水。

4.1 4,4’-二硝基均二苯脲对照品：含量 $\geqslant 99\%$ 。  
4.2 N,N-二甲基甲酰胺。  
4.3 乙腈：色谱纯。  
4.4 甲醇：色谱纯。  
4.5 正己烷：色谱纯。  
4.6 正丙醇。  
4.7 $\mathrm{C_{18}}$ 固相萃取柱： $500\mathrm{mg / 3mL}$ ，或相当者。  
4.8 洗脱液：取乙腈 $70 \mathrm{~mL}$ ，用水溶解并稀释至 $100 \mathrm{~mL}$ 。  
4.9 $1\mathrm{mg / mL}$ 4,4'-二硝基均二苯脲标准贮备液: 精密称取4,4'-二硝基均二苯脲对照品25 mg, 于25 mL

的容量瓶中，用二甲基甲酰胺溶解并稀释至刻度，配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的 $4, 4^{\prime}$ -二硝基均二苯脲标准贮备液。 $-20^{\circ} \mathrm{C}$ 以下保存，有效期6个月。

4.10 $100 \mu \mathrm{g} / \mathrm{mL} 4, 4'$ -二硝基均二苯脲标准工作液: 精密量取 $1 \mathrm{mg} / \mathrm{mL} 4, 4'$ -二硝基均二苯脲标准贮备液 $1.0 \mathrm{~mL}$ , 于 $10 \mathrm{~mL}$ 量瓶中, 用甲醇溶解并稀释至刻度, 配制成浓度为 $100 \mu \mathrm{g} / \mathrm{mL}$ 的 $4, 4'$ -二硝基均二苯脲标准工作液。2-4°C以下保存, 有效期一周。

# 5 仪器和设备

5.1 高效液相色谱仪：配紫外检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 均质机  
5.5 旋转蒸发仪   
5.6 旋涡混合器  
5.7 超声波清洗器  
5.8 离心机  
5.9 鸡心瓶： $50~\mathrm{mL}$ 。  
5.10 具塞离心管： $10\mathrm{mL}$ 、 $50\mathrm{mL}$ 。  
5.11 滤膜：有机相， $0.45 \mu \mathrm{m}$ 。

# 6 试料制备与保存

# 6.1 试料的制备

取适量新鲜或冷冻的空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

$-20^{\circ} \mathrm{C}$ 以下保存。

# 7 测定步骤

# 7.1 提取

称取试料（ $2.0 \pm 0.02$ ）g，于 $50 \mathrm{~mL}$ 离心管中，加乙腈 $10 \mathrm{~mL}$ ，超声 $5 \mathrm{~min}$ ， $4000 \mathrm{r} / \mathrm{min}$ 离心 $12 \mathrm{~min}$ ，取上清液，于 $50 \mathrm{~mL}$ 鸡心瓶中。残渣中加乙腈 $10 \mathrm{~mL}$ ，重复提取一次，合并两次上清液，加正丙醇 $3 \mathrm{~mL}$ ， $60^{\circ} \mathrm{C}$ 旋转蒸近干。加乙腈 $0.5 \mathrm{~mL}$ 、正己烷 $1 \mathrm{~mL}$ ，涡旋 $3 \mathrm{~min}$ ，溶解，移至 $10 \mathrm{~mL}$ 的具塞离心管中。用乙腈 $0.5 \mathrm{~mL}$ 和正己烷 $1 \mathrm{~mL}$ 重复涡旋溶解一次。合并两次溶液至 $10 \mathrm{~mL}$ 的具塞离心管中，加入正己烷 $2 \mathrm{~mL}$ ，漩涡混合 $3 \mathrm{~min}$ ， $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，弃上层正己烷液。再加正己烷 $2 \mathrm{~mL}$ ，重复提取一次， $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，取下层液，备用。

# 7.2 净化

$\mathrm{C}_{18}$ 柱用乙腈 $10 \mathrm{~mL}$ 活化，取备用液过柱，自然流干，收集滤液。用洗脱液 $4 \mathrm{~mL}$ 洗脱，收集洗脱液，合并滤液和洗脱液，于 $60^{\circ} \mathrm{C}$ 旋转蒸发近干，用流动相 $2.0 \mathrm{~mL}$ 溶解残余物，滤膜过滤，供高效液相色谱测定。

# 7.3 标准曲线的制备

精密量取 $100 \mu \mathrm{g} / \mathrm{mL} 4, 4'$ -二硝基均二苯脲标准工作液适量，用流动相稀释，配制成浓度为31.25、62.5、125、250、500、1000、2000、4000和 $8000 \mu \mathrm{g} / \mathrm{L}$ 的系列标准溶液，供高效液相色谱测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.4 测定

# 7.4.1 色谱条件

色谱柱： $\mathrm{C}_{18}$ （ $250 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ ，粒径 $5 \mu \mathrm{m}$ ），或相当者；

流动相：乙腈+水（58+42，v/v）；

流速： $1.0\mathrm{mL / min}$

检测器：紫外检测器；

波长： $340\mathrm{nm}$

柱温： $30^{\circ}\mathrm{C}$

进样量： $20~\mu \mathrm{L}$ 。

# 7.4.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试样溶液中 $4,4^{\prime}$ -二硝基均二苯脲响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和空白试料添加试样溶液的高效液相色谱图分别见附录A。

# 7.5 空白实验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果的计算与表达

试料中4, 4’-二硝基均二苯脲的残留量 $(\mu \mathrm{g} / \mathrm{kg})$ ：按下式计算

式中：

$$
X = \frac {\mathrm {A} \times \mathrm {C s} \times \mathrm {v}}{\mathrm {A s} \times m}
$$

$X$ ——供试试料中 $4, 4'$ -二硝基均二苯脲的残留量， $\mu \mathrm{g} / \mathrm{kg}$ ；

A ——试样溶液中 $4,4^{\prime}$ -二硝基均二苯脲的峰面积；

$\mathrm{A}_{\mathrm{S}}$ ——标准溶液中 $4, 4'$ -二硝基均二苯脲的峰面积;

$\mathrm{C}_{\mathrm{s}}$ ——标准溶液中 $4, 4'$ -二硝基均二苯脲的浓度， $\mu \mathrm{g} / \mathrm{L}$ ;

V——溶解残余物的流动相体积，mL；

$m$ ——供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留至小数点后三位。

# 9 检测方法的灵敏度、准确度、精密度

# 9.1 灵敏度

本方法的检测限为 $20 \mu \mathrm{g} / \mathrm{kg}$ , 定量限为 $100 \mu \mathrm{g} / \mathrm{kg}$ 。

# 9.2 准确度

本方法在 $100\sim 400~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

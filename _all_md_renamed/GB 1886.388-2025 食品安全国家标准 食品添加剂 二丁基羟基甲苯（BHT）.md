# 前言

本标准代替GB1900—2010《食品添加剂 二丁基羟基甲苯(BHT)》。

本标准与GB1900—2010相比，主要变化如下：

——更改了范围；  
——增加了气味；  
——增加了主含量指标及检验方法；  
——将“熔点（初熔）”更改为“熔程”；  
——将“重金属(以Pb计)”更改为“铅(Pb)”;   
——更改了鉴别试验。

# 1 范围

本标准适用于以对甲酚、异丁烯为原料，用酸性催化剂(浓硫酸或对甲苯磺酸)进行烷基化反应的生成物，经精馏、重结晶制得的食品添加剂二丁基羟基甲苯(BHT)。

# 检验方法

# A.1 警示

试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和防护措施。

# A.2 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682中规定的三级水。

试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。

# A.3 鉴别试验

# A.3.1 紫外分光光度法

# A.3.1.1 试剂和材料

无水乙醇：优级纯。

# A.3.1.2 仪器和设备

A.3.1.2.1 紫外-可见分光光度计：带 $1\mathrm{cm}$ 比色皿。  
A.3.1.2.2 分析天平：感量 $0.0001\mathrm{g}$

# A.3.1.3 分析步骤

# A.3.1.3.1 试样溶液的制备

称取约 $0.1\mathrm{g}$ 试样，准确至 $0.0001\mathrm{g}$ ，置于 $100\mathrm{mL}$ 容量瓶中，加无水乙醇适量，置超声水浴中超声 $2\mathrm{min}$ ，摇匀，放冷，定容至 $100\mathrm{mL}$ 。精密量取 $1.00\mathrm{mL}$ 上述溶液，置 $100\mathrm{mL}$ 容量瓶中，加无水乙醇稀释至刻度，摇匀，即得。

# A.3.1.3.2 测定

以无水乙醇作参比，用 $1\mathrm{cm}$ 比色皿，试样溶液在 $230~\mathrm{nm}\sim 320~\mathrm{nm}$ 波长范围内扫描，在 $278~\mathrm{nm}$ 波长处应有最大吸收。

# A.3.2 气相色谱法

# A.3.2.1 试剂和材料

二丁基羟基甲苯(CAS:128-37-0,分子式: $\mathrm{C_{15}H_{24}O)}$ 标准品:纯度 $\geqslant 99.0\%$ 。

# A.3.2.2 仪器和设备

A.3.2.2.1 气相色谱仪：配备氢火焰离子化检测器。   
A.3.2.2.2 分析天平：感量 $0.0001\mathrm{g}$

# A.3.2.3 分析步骤

称取 $0.5\mathrm{g}$ 二丁基羟基甲苯标准品(精确至 $0.0001\mathrm{g})$ ，置于 $10~\mathrm{mL}$ 容量瓶中，用无水乙醇溶解并定

容, 摇匀, 得到二丁基羟基甲苯的标样溶液。按 A.4 规定的方法, 在同样的条件下测定标样溶液和试样溶液, 记录试样溶液的色谱图。试样溶液中二丁基羟基甲苯的色谱峰保留时间应与标样溶液基本一致, 相对偏差 $2.5\%$ 之内。二丁基羟基甲苯标准溶液的气相色谱图见附录 B。

# A.4 二丁基羟基甲苯的测定

# A.4.1 方法提要

试样经无水乙醇溶解，在选定的工作条件下通过气相色谱柱，使样品中各组分分离，用火焰离子化检测器检测，采用面积归一法定量。

# A.4.2 试剂和材料

A.4.2.1 无水乙醇：优级纯。

A.4.2.2 氮气：体积分数不小于 $99.99\%$ ，经硅胶或5A分子筛干燥，净化。

A.4.2.3 氢气：体积分数不小于 $99.99\%$ ，经硅胶或5A分子筛干燥，净化。

A.4.2.4 空气：无油，经硅胶或5A分子筛干燥，净化。

# A.4.3 仪器和设备

A.4.3.1 气相色谱仪：配备氢火焰离子化检测器。

A.4.3.2 分析天平：感量 $0.0001\mathrm{g}$

# A.4.4 分析步骤

# A.4.4.1 试样溶液的制备

称取约 $0.5\mathrm{g}$ 试样(精确至 $0.0001\mathrm{g}$ )，置于 $10~\mathrm{mL}$ 容量瓶中，用无水乙醇溶解并定容，摇匀，即得。

# A.4.4.2 参考气相色谱条件

A.4.4.2.1 色谱柱：弱极性毛细管柱（键合， $5\%$ 苯基 $+95\%$ 甲基聚硅氧烷毛细管柱），长 $30~\mathrm{m}$ ，内径 $0.32~\mathrm{mm}$ ，膜厚 $1.00~\mu \mathrm{m}$ ，或性能相当者。

A.4.4.2.2 检测器温度： $250~\mathrm{^\circ C}$

A.4.4.2.3 进样口温度： $230^{\circ}C$

A.4.4.2.4 柱温： $150^{\circ}\mathrm{C}$ 保持 $1.0\mathrm{min},10^{\circ}\mathrm{C / min}$ 升温至 $210^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$

A.4.4.2.5 进样量： $0.6\mu \mathrm{L}$

A.4.4.2.6 分流比： $1:50$

A.4.4.2.7 载气：高纯氮气。

A.4.4.2.8 柱前压： $0.06\mathrm{MPa}\sim 0.1\mathrm{MPa}$

A.4.4.2.9 尾吹气流速： $20~\mathrm{mL / min}$

A.4.4.2.10 氢气流速： $30~\mathrm{mL / min}$

A.4.4.2.11 空气流速： $300~\mathrm{mL / min}$

# A.4.4.3 测定

在仪器的最佳条件下，吸取试样溶液注入气相色谱仪中，记录色谱图，按面积归一法定量。

# A.4.5 结果计算

二丁基羟基甲苯的含量的质量分数 $\varpi$ 按式(A.1)计算：

$$
w = \frac {A}{\sum A _ {i}} \times 100 \% \tag{A.1}
$$

式中：

$A$ ——试样溶液中二丁基羟基甲苯的峰面积；

$A_{i}$ ——试样溶液中组分 $i$ 的峰面积。

测定结果以两次平行测定的算术平均值表示，两次平行测定结果的绝对差值不大于 $0.2\%$

# A.5 游离酚（以对甲酚计）的测定

# A.5.1 试剂和材料

A.5.1.1 溴标准溶液： $c(1 / 2\mathrm{Br}_2) = 0.1\mathrm{mol} / \mathrm{L}$   
A.5.1.2 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$   
A.5.1.3 氢氧化钠溶液： $25\mathrm{g / L}$   
A.5.1.4 盐酸溶液： $1 + 1$   
A.5.1.5 碘化钾溶液： $100\mathrm{g / L}$   
A.5.1.6 淀粉指示液： $10\mathrm{g / L}$

# A.5.2 仪器和设备

分析天平：感量 $0.0001\mathrm{g}$

# A.5.3 分析步骤

称取约 $10\mathrm{g}$ 试样（精确至 $0.0001\mathrm{g}$ ），置于 $250~\mathrm{mL}$ 锥形烧瓶中，加 $50~\mathrm{mL}$ 氢氧化钠溶液，于 $65^{\circ}\mathrm{C}$ 下振荡不少于 $3\mathrm{min}$ ，过滤至 $250~\mathrm{mL}$ 容量瓶中，滤渣用水洗涤数次，合并滤液于容量瓶中，用水稀释至刻度。吸取 $100.00~\mathrm{mL}$ 于 $500~\mathrm{mL}$ 碘量瓶中，精密加入 $10.00~\mathrm{mL}$ 溴标准溶液，加 $10~\mathrm{mL}$ 盐酸溶液，迅速加盖充分摇匀，用 $5~\mathrm{mL}$ 碘化钾溶液封口，在 $15^{\circ}\mathrm{C}$ 以下静置 $15\mathrm{min}$ ，将碘化钾溶液放入碘量瓶中，加盖振荡，用水封口，放置 $5\mathrm{min}$ ，用硫代硫酸钠标准滴定溶液滴定至浅黄色，加 $5~\mathrm{mL}$ 淀粉指示液，继续滴定至蓝色消失为终点。同时做空白对照试验。

# A.5.4 结果计算

游离酚(以对甲酚计)的含量的质量分数 $\varpi$ 按式(A.2)计算：

$$
w = \frac {\left(V _ {2} - V _ {1}\right) \times c \times M}{m \times 1000 \times 6 \times (100 / 250)} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$V_{2}$ ——空白消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——试样消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫代硫酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

M 对甲酚 $\mathrm{(CH_3C_6H_4OH)}$ 的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$

$m$ ——试样质量，单位为克(g)；

1000 ——换算系数；

6 ——换算系数；

100/250——换算系数。

测定结果以两次平行测定的算术平均值表示，两次平行测定结果的绝对差值不大于 $0.002\%$ 。

# A.6 硫酸盐（以 $\mathbf{SO}_4^{2 - }$ 计）的测定

# A.6.1 试剂和材料

A.6.1.1 盐酸溶液： $1 + 10$

A.6.1.2 氯化钡溶液： $50~\mathrm{g / L}$ ，现用现配。

A.6.1.3 硫酸盐 $(\mathrm{SO}_4^{2-})$ 标准溶液: $0.1 \mathrm{mg/mL}$ 。

# A.6.2 分析步骤

称取约 $5\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），置于 $50~\mathrm{mL}$ 烧杯中，加 $30~\mathrm{mL}$ 新煮沸冷却的水，边加热边搅拌至试样溶解，搅拌并冷却至室温，过滤至 $50~\mathrm{mL}$ 比色管中，分别加入盐酸溶液和氯化钡溶液各 $5\mathrm{mL}$ ，用新煮沸冷却的水稀释至 $50~\mathrm{mL}$ ，摇匀，放置 $10\mathrm{min}$ 。另取 $1.00~\mathrm{mL}$ 硫酸盐标准溶液，与试样同时同样处理，以此作为标准。试样溶液所呈浊度不得大于标准，即为 $\leqslant 0.002\%$ 。

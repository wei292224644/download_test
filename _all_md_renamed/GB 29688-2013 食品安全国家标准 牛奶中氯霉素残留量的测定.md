# 1 范围

本标准规定了牛奶中氯霉素残留量检测的制样和液相色谱-串联质谱测定方法。

本标准适用于牛奶中氯霉素残留量的检测。

# 3 原理

试料中残留的氯霉素，用乙酸乙酯提取，正己烷除脂， $\mathrm{C}_{18}$ 柱净化，液相色谱-串联质谱测定，内标法定量。

# 4 试剂和材料

以下所用的试剂，除特别注明者外均为分析纯试剂；水为符合GB/T 6682规定的一级水。

4.1 氯霉素标准品：含量 $\geqslant 97\%$   
4.2 内标物：氘代氯霉素标准品：含量为 $100\mu \mathrm{g} / \mathrm{mL}$ （作为内标物标准贮备液）。  
4.3 甲醇：色谱纯。  
4.4 乙腈：色谱纯。  
4.5 乙酸乙酯  
4.6 氯化钠   
4.7 正己烷

4.8 $\mathrm{C_{18}}$ 固相萃取柱： $500\mathrm{mg / 3mL}$ ，或相当者。  
4.9 $4\%$ 氯化钠溶液：取氯化钠 $4\mathrm{g}$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。  
4.10 $100 \mu \mathrm{g} / \mathrm{mL}$ 氯霉素标准贮备液: 精密称取氯霉素标准品 $10 \mathrm{mg}$ , 于 $100 \mathrm{~mL}$ 量瓶中, 用甲醇溶解并稀释至刻度, 配制成浓度为 $100 \mu \mathrm{g} / \mathrm{mL}$ 的氯霉素标准贮备液。 $-20^{\circ} \mathrm{C}$ 以下保存, 有效期 1 年。  
4.11 $100 \mu \mathrm{g} / \mathrm{L}$ 氯霉素标准工作溶液 精密量取 $100 \mu \mathrm{g} / \mathrm{mL}$ 氯霉素标准贮备溶液 $100 \mu \mathrm{L}$ , 于 $100 \mathrm{~mL}$ 量瓶中, 用 $50 \%$ 乙腈溶解并稀释至刻度, 配制成浓度为 $100 \mu \mathrm{g} / \mathrm{L}$ 的标准工作液。 $2 \sim 8^{\circ} \mathrm{C}$ 保存, 有效期 1 个月。  
4.12 $20 \mu \mathrm{g} / \mathrm{L}$ 氖代氯霉素标准工作溶液 精密量取氘代氯霉素标准品 $20 \mu \mathrm{L}$ , 于 $1000 \mathrm{~mL}$ 量瓶中, 用 $50 \%$ 乙腈溶解并稀释至刻度, 配制成浓度为 $20 \mu \mathrm{g} / \mathrm{L}$ 的氘代氯霉素标准工作液。 $2 \sim 8^{\circ} \mathrm{C}$ 保存, 有效期 3 个月。

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或冷藏的空白或供试牛奶，混合，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-20°C以下保存。

# 7 测定步骤

# 7.1 提取

取试料（ $10 \pm 0.05$ ）g，于 $50 \mathrm{~mL}$ 离心管中，加氘代氯霉素内标工作液 $250 \mu \mathrm{L}$ ，再加乙酸乙酯 $20 \mathrm{~mL}$ ，振荡 $15 \mathrm{~min}$ ， $6000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取乙酸乙酯层液于鸡心瓶中。再加乙酸乙酯 $20 \mathrm{~mL}$ 重复提取一次，合并两次提取液于鸡心瓶中，于 $45^{\circ} \mathrm{C}$ 水浴旋转蒸发至干。用 $4 \%$ 氯化钠溶液 $5 \mathrm{~mL}$ 溶解残留物，加正己烷 $5 \mathrm{~mL}$ 振荡混合 $1 \mathrm{~min}$ ，静置分层，弃正己烷液。再加正己烷 $5 \mathrm{~mL}$ ，重复提取一次。取下层液备用。

# 7.2 净化

$\mathrm{C}_{18}$ 柱依次用甲醇 $5 \mathrm{~mL}$ 和水 $5 \mathrm{~mL}$ 活化，取备用液过柱，控制流速 $1$ 滴/3～4s，用水 $5 \mathrm{~mL}$ 淋洗，抽干，用甲醇 $5 \mathrm{~mL}$ 洗脱，收集洗脱液，于 $50^{\circ} \mathrm{C}$ 氮气吹干。用 $50 \%$ 乙腈 $1.0 \mathrm{~mL}$ 溶解残余物，涡旋混匀，滤膜过滤，供液相色谱-串联质谱测定。

# 7.3 标准曲线的制备

精密量取 $100 \mu \mathrm{g} / \mathrm{L}$ 氯霉素标准工作溶液和 $20 \mu \mathrm{g} / \mathrm{L}$ 氘代氯霉素内标工作溶液适量，用流动相稀释，配制成氯霉素浓度为 0.10、0.25、0.50、1.0、2.0、5.0 $\mu \mathrm{g} / \mathrm{L}$ ，氘代氯霉素浓度为 $5 \mu \mathrm{g} / \mathrm{L}$ 的系列标准溶液，供液相色谱-串联质谱仪测定。以特征离子质量色谱峰面积为纵坐标，标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.4 测定

# 7.4.1 液相色谱条件

色谱柱： $C_{18}$ （ $150\mathrm{mm}\times 2.1\mathrm{mm}$ ，粒径 $5\mu \mathrm{m})$ ，或相当者；

柱温： $30^{\circ}\mathrm{C}$ ；

流速： $0.2\mathrm{mL / min}$

进样量： $20~\mu \mathrm{L}$

运行时间：8min；

流动相：乙腈+水（50+50，v/v）；

# 7.4.2 质谱条件

电离模式：ESI;

扫描方式：负离子扫描；

检测方式：多反应检测；

电离电压： $2.8\mathrm{kV}$

源温： $120^{\circ}\mathrm{C}$

雾化温度： $350^{\circ}\mathrm{C}$

锥孔气流速： $50\mathrm{L / h}$

雾化气流速： $450\mathrm{L / h}$

数据采集窗口：8min；

驻留时间：0.3s；

定性、定量离子及对应的锥孔电压和碰撞电压见表1

# 8 结果计算和表述

计算 $\mathrm{A}_{321 / 151.6} / \mathrm{A}_{325.8 / 156.6}$ 峰面积比值，标准曲线校准。

由标准曲线方程： $\mathrm{A_s / A'_is = a\times c_s / c'_is + b}$

求得a和b，则 $c_{\mathrm{isA}}^{\prime}$

$$
c = - (- b)
$$

$$
\begin{array}{c c} \text {a} & \mathrm {A} _ {\mathrm {i s}} \\ \hline \end{array}
$$

试料中氯霉素残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算：

$$
X = \frac {C \times V}{m}
$$

式中：

X——供试试料中氯霉素的残留量， $\mu \mathrm{g} / \mathrm{kg}$ ；

$A_{s}$ ——对照溶液中氯霉素的峰面积；

$A_{iS}^{\prime}$ ——对照溶液中内标氘代氯霉素的峰面积；

$c_{s}$ ——对照溶液中内标氘代氯霉素的浓度, $\mathrm{ng} / \mathrm{mL}$ ;

$c_{is}^{\prime}$ ——对照溶液中氯霉素的浓度， $\mathrm{ng / mL}$

$c$ ——供试溶液中氯霉素的浓度, $\mathrm{ng} / \mathrm{mL}$ ;

$c_{is}$ ——试样中氘代氯霉素的浓度, $\mathrm{ng} / \mathrm{mL}$ ;

$A$ ——试样中氯霉素的峰面积；

$A_{iS}$ ——试样中内标氘代氯霉素的峰面积；

$V$ ——溶解残余物的体积, $\mathrm{mL}$ ;

$m$ ——供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度、精密度

# 9.1 灵敏度

本方法检测限为 $0.01\mu \mathrm{g / kg}$ ，定量限为 $0.1\mu \mathrm{g / kg}$

# 9.2 准确度

本方法在 $0.02\sim 0.10~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $50\% \sim 120\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 17\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

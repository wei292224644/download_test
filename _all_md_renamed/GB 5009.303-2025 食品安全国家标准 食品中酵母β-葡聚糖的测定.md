# 食品安全国家标准

# 食品中酵母 $\pmb{\beta}$ - 葡聚糖的测定

# 1 范围

本标准规定了食品中不溶性酵母 $\beta$ -葡聚糖的测定方法。

本标准适用于乳及乳制品、蛋白粉、糖果、饮料中不溶性酵母 $\beta$ -葡聚糖的测定。

# 2 原理

试样经蛋白酶和/或脂肪酶去除脂肪、蛋白质后，通过离心将酵母 $\beta$ -葡聚糖进行沉淀。沉淀后的酵母 $\beta$ -葡聚糖在多种酶的作用下水解为葡萄糖。利用葡萄糖氧化酶(GOD)在有氧条件下催化葡萄糖氧化，生成D-葡萄糖醛酸- $d$ -内酯和过氧化氢，过氧化氢经过氧化物酶(POD)催化与4-氨基安替比林和对羟基苯甲酸生成红色醌亚胺(GODPOD法)。利用分光光度计在 $510~\mathrm{nm}$ 波长处测定醌亚胺的吸光度。最后通过酵母 $\beta$ -葡聚糖和葡萄糖的转化系数(0.9)，计算得出样品中酵母 $\beta$ -葡聚糖的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 碱性蛋白酶，液体，酶活力 $\geq 240000\mathrm{U / mL}$   
3.1.2 中性蛋白酶，液体，酶活力 $\geq 80000\mathrm{U / mL}$   
3.1.3 酸性蛋白酶，液体，酶活力 $\geq 50000\mathrm{U / mL}$   
3.1.4 脂肪酶，液体，酶活力 $\geq 20000\mathrm{U / mL}$   
3.1.5 乙二胺四乙酸四钠二水合物 $\mathrm{(C_{10}H_{12}N_2Na_4O_8\cdot 2H_2O)}$ 。  
3.1.6 三羟甲基氨基甲烷(tris) $(\mathrm{C_4H_{11}NO_3})$ 。  
3.1.7 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.8 无水乙酸钠 $\mathrm{(CH_3COONa)}$ 。  
3.1.9 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.10 氢氧化钾(KOH)。  
3.1.11 氢氧化钠(NaOH)。  
3.1.12 盐酸（HCl）。  
3.1.13 溶壁酶，酶活力 $\geq 200\mathrm{U / mg}$   
3.1.14 $\beta - (1,6)$ - 葡聚糖酶，酶活力 $\geqslant 2\mathrm{U} / \mathrm{mg}$   
3.1.15 $\beta-$ (1,3)-葡聚糖酶和葡萄糖苷酶混合酶，酶活力分别 $\geqslant 100\mathrm{U / mL}$ 和 $20~\mathrm{U / mL}$ 。  
3.1.16 葡萄糖氧化酶，酶活力 $\geqslant 400\mathrm{U}$   
3.1.17 过氧化物酶，酶活力 $\geqslant 1000\mathrm{U}$   
3.1.18 4-氨基安替比林 $(\mathrm{C_{11}H_{13}N_3O})$ 。

3.1.19 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$ 。  
3.1.20 对羟基苯甲酸 $(\mathrm{C}_7\mathrm{H}_6\mathrm{O}_3)$ 。  
3.1.21 叠氮化钠 $(\mathrm{NaN}_3)$ 。  
3.1.22 滤膜：孔径 $0.45\mu \mathrm{m}$

# 3.2 试剂配制

3.2.1 蛋白酶混合溶液：分别吸取 $2.0 \mathrm{~mL}$ 碱性蛋白酶和中性蛋白酶于 $5 \mathrm{~mL}$ 离心管中，混匀后， $4^{\circ} \mathrm{C}$ 保存。临用现配。  
3.2.2 氢氧化钾溶液 $(2.0\mathrm{mol} / \mathrm{L})$ ：称取 $11.2\mathrm{g}$ 氢氧化钾，加水溶解并稀释至 $100~\mathrm{mL}$ ，混匀后， $4^{\circ}\mathrm{C}$ 冷藏保存。  
3.2.3 氢氧化钠溶液 $(1.0\mathrm{mol} / \mathrm{L})$ ：称取 $4.0\mathrm{g}$ 氢氧化钠，加水充分溶解后，定容至 $100~\mathrm{mL}$ ，并混匀。  
3.2.4 盐酸溶液 $(1.0\mathrm{mol} / \mathrm{L})$ ：称取 $9\mathrm{mL}$ 盐酸至盛有 $80~\mathrm{mL}$ 水的 $100~\mathrm{mL}$ 容量瓶中，充分混匀后，加水定容并混匀。  
3.2.5 乙酸钠缓冲溶液 $\mathrm{A}(0.2\mathrm{mol} / \mathrm{L})$ ：分别称取 $5.25\mathrm{g}$ 无水乙酸钠和 $2.16\mathrm{g}$ 冰乙酸至 $400~\mathrm{mL}$ 水中，氢氧化钠溶液或冰乙酸调节 $\mathrm{pH}$ 至 $5.0\pm 0.05$ ，加水定容至 $500~\mathrm{mL}$ ，并混匀。  
3.2.6 乙酸钠缓冲溶液 $\mathrm{B}(1.2\mathrm{mol} / \mathrm{L})$ ：分别称取 $4.96\mathrm{g}$ 无水乙酸钠和 $32.32\mathrm{g}$ 冰乙酸至 $400~\mathrm{mL}$ 水中，氢氧化钠溶液或冰乙酸调节 $\mathrm{pH}$ 至 $3.8\pm 0.05$ ，加水定容至 $500~\mathrm{mL}$ ，并混匀。  
3.2.7 缓冲溶液 $\mathrm{C}(\mathrm{pH}7.5)$ : 溶解 $1.212\mathrm{g}$ 三羟甲基氨基甲烷、 $1.169\mathrm{g}$ 氯化钠以及 $0.416\mathrm{g}$ 乙二胺四乙酸四钠二水合物于 $90\mathrm{mL}$ 水中。用盐酸溶液或氢氧化钠溶液调节 $\mathrm{pH}$ 至 $7.5 \pm 0.05$ ，加水定容至 $100\mathrm{mL}$ ，并混匀。使用前，移取适量以水稀释10倍。临用现配。  
3.2.8 溶壁酶溶液 $(5\mathrm{U} / \mu \mathrm{L})$ : 称取适量溶壁酶至缓冲溶液 C 中, 使溶壁酶最终浓度为 $5\mathrm{U} / \mu \mathrm{L}$ 。临用现配。  
3.2.9 $\beta-$ (1,6)-葡聚糖酶溶液：溶解适量 $\beta-$ (1,6)-葡聚糖酶于乙酸钠缓冲溶液A中，终浓度为 $1\mathrm{U}/$ $300~\mu \mathrm{L}$ 的悬浊液。临用现配。  
3.2.10 混合酶溶液：移取适量 $\beta-(1,3)$ -葡聚糖酶和 $\beta$ -葡萄糖苷酶混合酶，加入适量乙酸钠缓冲溶液 A 稀释混匀，使最终浓度分别为 $20 \mathrm{U} / \mathrm{mL}$ 和 $4 \mathrm{U} / \mathrm{mL}$ 。临用现配。使用期间应置于冰水浴中保存。  
3.2.11 葡萄糖氧化酶-过氧化物酶(GODPOD)缓冲液: 向 $200 \mathrm{~mL}$ 容量瓶中加入 $160 \mathrm{~mL}$ 左右水, 随后分别加入 $27.2 \mathrm{~g}$ 磷酸二氢钾, $8.4 \mathrm{~g}$ 氢氧化钠和 $6.0 \mathrm{~g}$ 对羟基苯甲酸, 搅拌使其完全溶解后, 调节 $\mathrm{pH}$ 至 $7.4 \pm 0.05$ 。最后加入 $0.8 \mathrm{~g}$ 叠氮化钠, 溶解后加水定容, 临用现配。使用前应进行稀释, 量取 GODPOD 缓冲液 $48 \mathrm{~mL}$ 至 $1000 \mathrm{~mL}$ 容量瓶中, 加水定容至刻度, 充分混匀。临用现配。  
3.2.12GODPOD工作液：称取 $400\mathrm{U}$ 葡萄糖氧化酶和 $1000\mathrm{U}$ 过氧化物酶和 $31.3\mathrm{mg}4$ -氨基安替比林于 $1000~\mathrm{mL}$ 容量瓶中，加入 $20~\mathrm{mL}$ 稀释后的GODPOD缓冲液，轻轻晃动使充分溶解，再以GODPOD缓冲液定容，混匀后 $4^{\circ}\mathrm{C}$ 避光保存。临用现配。

注： $3.2.8\sim 3.2.12$ 也可使用商品化试剂或试剂盒。

# 3.3 标准品

3.3.1 不溶性酵母 $\beta$ -葡聚糖参考物质 $\left[(\mathrm{C}_{6} \mathrm{H}_{10} \mathrm{O}_{5})_{n}, \mathrm{CAS}\right.$ 号：9012-72-0]，纯度 $\geqslant 70\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.2 无水葡萄糖 $\left(\mathrm{C}_{6} \mathrm{H}_{12} \mathrm{O}_{6}\right)$ , CAS号:50-99-7),纯度 $\geqslant 99\%$ ,或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 葡萄糖标准储备溶液 $(2.00\mathrm{g / L})$ ：称取 $0.200\mathrm{g}$ （精确至 $0.1\mathrm{mg}$ )经过 $98^{\circ}\mathrm{C}\sim 100^{\circ}\mathrm{C}$ 干燥 $2\textrm{h}$ 的无

水葡萄糖，加水溶解并定容至 $100~\mathrm{mL}$ ，摇匀，将溶液转移至试剂瓶。临用现配。

3.4.2 葡萄糖系列标准工作溶液：分别量取 $0.6 \mathrm{~mL}, 1.0 \mathrm{~mL}, 2.5 \mathrm{~mL}, 4.0 \mathrm{~mL}$ 和 $5.0 \mathrm{~mL}$ 葡萄糖标准储备溶液至 $10 \mathrm{~mL}$ 容量瓶中，加水定容并混匀。其葡萄糖质量浓度分别为 $0.12 \mathrm{~g} / \mathrm{L}, 0.20 \mathrm{~g} / \mathrm{L}, 0.50 \mathrm{~g} / \mathrm{L}, 0.80 \mathrm{~g} / \mathrm{L}$ 和 $1.00 \mathrm{~g} / \mathrm{L}$ 。临用现配。

# 4 仪器和设备

4.1 分光光度计：可测定 $510\mathrm{nm}$ 波长下的吸光度，配有 $1\mathrm{cm}$ 比色皿。  
4.2 恒温水浴装置： $40^{\circ}\mathrm{C} \sim 80^{\circ}\mathrm{C}$ ，精度为 $1.0^{\circ}\mathrm{C}$ 。  
4.3 天平：感量分别为 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$ 。  
4.4 移液器：量程为 $1.00\mathrm{mL}$ 和 $200~\mu \mathrm{L}$   
4.5 pH计：精度为0.01。  
4.6 涡旋混合器。  
4.7 离心机：转速可达到 $8000\mathrm{r / min}$   
4.8 干燥箱。  
4.9 圆孔筛：筛孔 $2.0 \mathrm{~mm}$ 。

# 5 试样的制备

# 5.1 固体样品

取固体样品 $5.0\mathrm{g}\sim 50.0\mathrm{g}$ ，研钵研磨粉碎后， $2.0\mathrm{mm}$ 圆孔筛过筛后保存。

注：软糖等无法粉碎的样品，应尽量剪碎。

# 6 分析步骤

# 6.1 基质对照样品的制备

称取 $3\mathrm{mg}\sim 10\mathrm{mg}$ 酵母 $\beta$ -葡聚糖参考物质(精确至 $0.0001\mathrm{g}$ ，与待测样品中酵母 $\beta$ -葡聚糖含量偏差不大于 $20\%$ )于尖底离心管中，加入 $10.0\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 相应的空白样品，充分混匀。

# 6.2 沉淀

# 6.2.1 液体样品

分别称取 $10.0\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，或相当于 $3\mathrm{mg} \sim 10\mathrm{mg}$ 酵母 $\beta$ -葡聚糖的样品质量）混匀后的液体样品和制备的相应基质对照样品于 $15\mathrm{mL}$ 尖底离心管中，各加入 $200\mu \mathrm{L}$ 蛋白酶混合溶液（酸奶等酸性样品加入 $200\mu \mathrm{L}$ 酸性蛋白酶）， $40^{\circ}\mathrm{C}$ 孵育 $2\mathrm{h}$ 后冷却至室温。静置 $10\mathrm{min}$ 后， $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，用滴管缓慢吸除上层脂肪和中间酶解液，保留沉淀。

# 6.2.2 固体样品

分别称取 $10.0\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，或相当于 $3\mathrm{mg} \sim 10\mathrm{mg}$ 酵母 $\beta$ -葡聚糖的样品质量）5.1处理后的固体样品和制备的相应基质对照样品于 $50~\mathrm{mL}$ 离心管中，各加入 $20~\mathrm{mL}$ 水充分溶解。再加入 $500~\mu \mathrm{L}$ 蛋白酶混合溶液混匀， $40^{\circ}\mathrm{C}$ 孵育 $2\mathrm{h}$ 后冷却至室温。静置 $10\mathrm{min}$ 后， $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，用滴管缓慢吸除上层脂肪和中间酶解液，保留沉淀。

注：对于离心后没有出现酵母 $\beta$ -葡聚糖沉淀的样品或脂肪含量 $\geq 10\%$ 的样品，蛋白酶孵育 $2\textrm{h}$ 后再加入 $500~\mu \mathrm{L}$ 脂

肪酶， $40\mathrm{^\circ C}$ 孵育 $2\mathrm{h}$ 。或称取 $10.0\mathrm{g}$ 样品后，参考GB5009.88—2023中6.1.3去除样品中多余脂肪后，再用蛋白酶将蛋白质水解。

# 6.3 沉淀清洗

向6.2处理后的沉淀中加入 $5\mathrm{mL}$ 水，充分分散，静置 $10\mathrm{min}$ 后， $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，用滴管缓慢吸除上清液，沉淀清洗需重复3次以上，直至上清液按照6.4的方法无葡萄糖检出，则底部沉淀可以进行下一步酶解。

# 6.4 上清液的测定

移取适量经6.3步骤处理后的上清液 $0.45\mu \mathrm{m}$ 滤膜过滤后，移取 $100~\mu \mathrm{L}$ 至 $5\mathrm{mL}$ 离心管中，各加入 $3\mathrm{mL}$ GODPOD工作液， $40^{\circ}C$ 水浴 $20\mathrm{min}$ 。移出水浴锅，冷却至室温后，转移至 $1\mathrm{cm}$ 比色皿中，以GODPOD工作液为空白，用分光光度计在 $510~\mathrm{nm}$ 处测量吸光度，并记录。用标准曲线计算试样溶液中葡萄糖的浓度。

# 6.5 酶解

# 6.5.1 样品和基质对照样品

向6.3处理后的样品和基质对照样品中加入 $400\mu \mathrm{L}$ 冷的氢氧化钾溶液，并冰水浴 $20\mathrm{min}$ 。冰水浴期间多次短时涡旋至全部沉淀分散且无可见结块后，加入 $1.6\mathrm{mL}$ 乙酸钠缓冲溶液B，随后加入 $500\mu \mathrm{L}$ 溶壁酶溶液，记录此时总体积分别为 $V_{1}$ 和 $V_{1}^{\prime}$ 。将反应液在 $50~^\circ \mathrm{C}$ 恒温水浴锅中孵育 $12\mathrm{h}\sim 18\mathrm{h}$ 后，冷却至室温。分别移取 $150\mu \mathrm{L}(V_{2}$ 和 $V_{2}^{\prime})$ 该酶解液至 $2\mathrm{mL}$ 离心管中，加入 $300\mu \mathrm{L}\beta -$ (1,6)-葡聚糖酶溶液， $80~^\circ \mathrm{C}$ 孵育 $15\mathrm{min}$ 后，冷却至室温。再加入 $300\mu \mathrm{L}$ 混合酶溶液，记录总体积分别为 $V_{3}$ 和 $V_{3}^{\prime}$ ， $40~^\circ \mathrm{C}$ 孵育 $1\mathrm{h}$ 后，冷却至室温。移取适量溶液，经 $0.45\mu \mathrm{m}$ 滤膜过滤后进行检测。

# 6.5.2 酶空白溶液

移取 $200\mu \mathrm{L}$ 氢氧化钾溶液至 $5\mathrm{mL}$ 离心管中，加入 $0.8\mathrm{mL}$ 乙酸钠缓冲溶液B，混匀后移取 $120~\mu \mathrm{L}$ 加入 $2\mathrm{mL}$ 离心管中，加入 $30~\mu \mathrm{L}$ 溶壁酶溶液，混匀。 $50^{\circ}C$ 孵育 $12\mathrm{h}\sim 18\mathrm{h}$ 后，冷却至室温。再加入 $300~\mu \mathrm{L}\beta -(1,6)$ -葡聚糖酶溶液， $80^{\circ}C$ 孵育 $15\mathrm{min}$ 后，冷却至室温。最后加入 $300~\mu \mathrm{L}$ 混合酶溶液， $40^{\circ}C$ 孵育 $1\textrm{h}$ ，冷却至室温。移取适量溶液，经 $0.45\mu \mathrm{m}$ 滤膜过滤后进行检测。

# 6.6 测定

# 6.6.1 标准曲线的绘制

分别移取 $100\mu \mathrm{L}$ 葡萄糖系列标准溶液至 $5\mathrm{mL}$ 离心管中，各加入 $3\mathrm{mL}$ GODPOD工作液， $40^{\circ}\mathrm{C}$ 水浴 $20\mathrm{min}$ 。移出水浴，冷却至室温后，移取适量溶液经 $0.45\mu \mathrm{m}$ 滤膜过滤后转移至 $1\mathrm{cm}$ 比色皿中，以GODPOD工作液为空白，用分光光度计在 $510\mathrm{nm}$ 处测量吸光度，并记录。以吸光度为纵坐标，以系列标准溶液的浓度为横坐标，绘制标准曲线。

# 6.6.2 样品溶液的测定

分别移取 $100\mu \mathrm{L}$ 经6.5步骤处理后的样品溶液、基质对照样品溶液、酶空白溶液，至 $5\mathrm{mL}$ 离心管中，各加入 $3\mathrm{mL}$ GODPOD工作液， $40^{\circ}\mathrm{C}$ 水浴 $20\mathrm{min}$ 。移出水浴，冷却至室温后，转移至 $1\mathrm{cm}$ 比色皿中，以GODPOD工作液为空白，用分光光度计在 $510\mathrm{nm}$ 处测量吸光度，并记录。用标准曲线计算试样溶液中葡萄糖的浓度。可根据具体试样进行稀释 $(f)$ 。

# 7 分析结果的表达

试样中酵母 $\beta$ -葡聚糖的含量按公式(1)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times f _ {1} / \left[ \left(\rho_ {\mathrm {S}} - \rho_ {0}\right) \times f _ {2} \right] \times \rho_ {\mathrm {S}} ^ {\prime} \times 0 . 9 \times 1 0 0 0}{m / V _ {1} \times V _ {2} / V _ {3} \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——食品中酵母 $\beta$ -葡聚糖的含量，单位为克每千克 $(\mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——样品酶解液最终的葡萄糖含量的测定值，单位为克每升 $(\mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——酶空白溶液的葡萄糖含量的测定值，单位为克每升 $(\mathrm{g / L})$ ；

$f_{1}$ ——样品酶解液的稀释倍数；

$\rho_{\mathrm{s}}$ ——基质对照样品酶解液的葡萄糖含量的测定值，单位为克每升 $(\mathrm{g} / \mathrm{L})$ ；

$f_{2}$ ——基质对照样品酶解液的稀释倍数；

$\rho_{\mathrm{s}}^{\prime}$ ——基质对照样品酶解后理论葡萄糖含量的计算值, 单位为克每升 $(\mathrm{g} / \mathrm{L})$ ；

0.9 ——葡萄糖换算为酵母 $\beta$ -葡聚糖的系数；

1000—克与千克的换算系数；

$m$ ——样品质量，单位为克(g)；

$V_{1}$ ——溶壁酶酶解后酶解液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——从溶壁酶酶解液中移取的酶解液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——完成所有酶解后的终体积，单位为毫升 $(\mathrm{mL})$ ；

1000——毫升与升的换算系数。

公式(1)中对照品酶解后理论葡萄糖含量的计算值 $\rho_{\mathrm{S}}^{\prime}$ 按公式(2)计算：

$$
\rho_ {\mathrm {S}} ^ {\prime} = \frac {m _ {\mathrm {S}} \times P \times V _ {2} ^ {\prime} \times 1 0 0 0}{0 . 9 \times V _ {1} ^ {\prime} \times V _ {3} ^ {\prime} \times 1 0 0 \times 1 0 0 0} \tag {2}
$$

式中：

$\rho_{\mathrm{s}}^{\prime}$ 基质对照样品酶解后理论葡萄糖含量的计算值，单位为克每升 $(\mathrm{g / L})$ ；

$m_{\mathrm{S}}$ ——酵母 $\beta$ -葡聚糖参考物质对照品的质量，单位为毫克 $(\mathrm{mg})$ ；

$P$ ——酵母 $\beta$ -葡聚糖参考物质的纯度(对照品说明书标识的纯度), 单位为克每百克 $(\mathrm{g} / 100 \mathrm{~g})$ ;

$V_{2}^{\prime}$ ——从溶壁酶酶解液中移取的酶解液体积，单位为毫升 $(\mathrm{mL})$ ；

1000——毫升与升的换算系数。

0.9 ——葡萄糖换算为酵母 $\beta$ -葡聚糖的系数；

$V_{1}^{\prime}$ ——溶壁酶酶解后酶解液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}^{\prime}$ ——完成所有酶解后的终体积，单位为毫升 $(\mathrm{mL})$ ；

100 ——参考物质纯度单位转化系数；

1000——毫克与克的换算系数。

当样品与对照品按照6.5操作过程，二者的 $V_{1}$ 和 $V_{1}^{\prime}, V_{2}$ 和 $V_{2}^{\prime}, V_{3}$ 和 $V_{3}^{\prime}, f_{1}$ 和 $f_{2}$ 完全相同时，公式(1)可简化为公式(3)：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times m _ {\mathrm {S}} \times P \times 1 0 0 0}{\left(\rho_ {\mathrm {S}} - \rho_ {0}\right) \times m \times 1 0 0 \times 1 0 0 0} \tag {3}
$$

式中：

$X$ ——食品中酵母 $\beta$ -葡聚糖的含量，单位为克每千克 $(\mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——样品酶解液最终的葡萄糖含量的测定值，单位为克每升 $(\mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——酶空白溶液的葡萄糖含量的测定值，单位为克每升 $(\mathrm{g / L})$ ；

$m_{\mathrm{S}}$ ——酵母 $\beta$ -葡聚糖参考物质的质量，单位为毫克 $(\mathrm{mg})$ ；

P 酵母 $\beta$ -葡聚糖参考物质的纯度(依据试剂厂家提供的纯度报告),单位为克每百克（g/ $100\mathrm{g})$

1000—克与千克的换算系数；

$\rho_{\mathrm{s}}$ ——酵母 $\beta$ -葡聚糖基质对照样品酶解液的葡萄糖含量的测定值，单位为克每升 $(\mathrm{g} / \mathrm{L})$ ；

$m$ ——样品质量，单位为克(g)；

100 ——对照品纯度单位转换系数；

1000——毫克与克的换算系数。

计算结果保留三位有效数字。

# 8 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

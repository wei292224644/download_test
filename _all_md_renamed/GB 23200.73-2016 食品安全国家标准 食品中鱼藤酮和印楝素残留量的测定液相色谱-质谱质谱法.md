本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T3264—2012《出口食品中鱼藤酮和印楝素残留量的检测方法液相色谱一质谱/质谱法》。与SN/T3264—2012相比，主要变化如下：

# 食品安全国家标准

# 食品中鱼藤酮和印楝素残留量的测定 液相色谱一质谱/质谱法

# 1 范围

本标准规定了食品中鱼藤酮和印楝素残留量的液相色谱一质谱/质谱检测方法。

本标准适用于大米、花椰菜、苹果、木耳、茶叶、蜂蜜、猪肝、鱼肉、虾肉、鸡肉、牛奶中鱼藤酮和印楝素残留量的测定和确证，其他食品可参照执行。

# 2 规范性引用文件

# 3 原理

试样中残留的鱼藤酮和印楝素采用乙腈提取，提取液经氯化钠盐析后经正己烷除脂，以聚苯乙烯一二乙烯基苯一吡咯烷酮聚合物填料的固相萃取小柱净化，液相色谱一质谱/质谱仪检测及确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
4.1.2 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
4.1.3 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。  
4.1.4 甲酸（HCOOH）：色谱纯。  
4.1.5 氯化钠 $(\mathrm{NaCl})$ 。  
4.1.6 乙酸铵 $\mathrm{(CH_3COONH_4)}$   
4.1.7 碳酸氢钠 $\mathrm{(NaHCO_3)}$

# 4.2 溶液配制

4.2.1 饱和碳酸氢钠溶液：称取一定量碳酸氢钠溶于水中至饱和。  
4.2.2 $5\mathrm{mmol / L}$ 乙酸铵缓冲液：称取 $0.38\mathrm{g}$ 乙酸铵溶于 $800~\mathrm{mL}$ 水中，加入 $2\mathrm{mL}$ 甲酸，以水定容至 $1000~\mathrm{mL}$ 。

# 4.3 标准品

标准物质(鱼藤酮：英文名 Rotenone，分子式 $\mathrm{C_{23}H_{22}O_6}$ ，CAS号：83-79-4，相对分子质量394.42；印楝素：英文名Azadirachtin，分子式 $\mathrm{C_{35}H_{44}O_{16}}$ ，CAS号：11141-17-6，相对分子质量720.71）：纯度 $\geq 98\%$ 。

# 4.4 标准溶液配制

4.4.1 鱼藤酮和印楝素标准储备液 $(100\mathrm{mg / L})$ : 准确称取 $0.0100\mathrm{g}$ 鱼藤酮和印楝素标准物质, 用甲醇溶解并定容至 $100\mathrm{mL}$ , 该标准储备液于 $4^{\circ}\mathrm{C}$ 避光保存不超过 1 个月。  
4.4.2 鱼藤酮和印楝素标准工作液：根据需要取适量标准储备液，以 $20\%$ 乙腈水溶液稀释成适当浓度的标准工作液，临用现配。

# 4.5 材料

4.5.1 聚苯乙烯一二乙烯基苯一吡咯烷酮聚合物填料的固相萃取柱： $3\mathrm{mL}/60\mathrm{mg}$ 。使用前依次以 $3\mathrm{mL}$ 甲醇、 $3\mathrm{mL}$ 水预处理。  
4.5.2 滤膜： $0.22\mu \mathrm{m}$ ，有机相。

# 5 仪器和设备

5.1 液相色谱—质谱/质谱联用仪，配电喷雾(ESD)源。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.3 离心机： $4500\mathrm{r / min}$ ，配有 $50~\mathrm{mL}$ 的具塞塑料离心管。  
5.4 粉碎机。  
5.5 组织捣碎机。  
5.6 涡旋振荡器。  
5.7 超声波清洗器。  
5.8 固相萃取装置。  
5.9 氮吹仪。

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按 GB 2763 的规定执行。在取样和制样过程中,应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 水果蔬菜类

取有代表性样品 $500 \mathrm{~g}$ , 切碎后 (不可水洗), 用组织捣碎机将样品加工成浆状, 混匀, 分成 2 份, 装入洁净容器内, 密封并标识。

# 6.2.2 茶叶、粮谷与坚果类

取有代表性样品 $500\mathrm{g}$ ，用粉碎机粉碎并通过直径 $2.0\mathrm{mm}$ 的筛，混匀，分成2份，装入洁净容器内，密封并标识。

# 6.2.3 肉及肉制品

取有代表性样品 $500\mathrm{g}$ ，切碎后用组织捣碎机将样品加工成浆状，混匀，分成2份，装入洁净容器内，密封并标识。

# 6.2.4 蜂蜜

取有代表性样品 $500\mathrm{g}$ ，对于无结晶的蜂蜜样品，将其搅拌均匀。对有结晶的样品，在密闭情况下，置于不超过 $60^{\circ}\mathrm{C}$ 的水浴中温热，振荡，待样品全部融化后搅匀，迅速冷却至室温，分成2份，装入洁净样品瓶中，密封并标识。

# 6.2.5 牛奶

取有代表性的样品 $500\mathrm{g}$ ，充分混匀，分为2份，装入洁净样品瓶中，密封并标识。

# 6.3 试样保存

茶叶、果酒、蜂蜜、牛奶、粮谷于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存，蔬菜、水果、肉及肉制品于 $-18^{\circ}\mathrm{C}$ 保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 茶叶、谷物及坚果等样品

称取 $1\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样, 置于 $50\mathrm{mL}$ 具塞塑料离心管中, 加入 $5\mathrm{mL}$ 饱和碳酸氢钠溶液振摇后避光浸泡 $10\mathrm{min}$ , 加入 $15\mathrm{mL}$ 乙腈, 涡动 $30\mathrm{s}$ 后超声提取 $10\mathrm{min}, 4500\mathrm{r} / \mathrm{min}$ 离心 $3\mathrm{min}$ , 移出有机相, 残渣再加入 $10\mathrm{mL}$ 乙腈重复提取 1 次, 合并提取液, 加入约 $3\mathrm{g}$ 氯化钠盐析, $4500\mathrm{r} / \mathrm{min}$ 离心 $3\mathrm{min}$ 离心, 取上清液, 加入 $2\mathrm{mL}$ 经乙腈饱和后的正己烷, 振摇 $1\mathrm{min}, 4500\mathrm{r} / \mathrm{min}$ 离心 $3\mathrm{min}$ 离心, 弃去正己烷层, 乙腈层于 $45^{\circ}\mathrm{C}$ 减压旋转蒸发至近干, 以 $5\mathrm{mL}20\%$ 甲醇水溶解残渣, 按7.2步骤净化。

# 7.1.2 蔬菜与水果样品

称取 $2\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）试样，置于 $50~\mathrm{mL}$ 具塞塑料离心管中，加入约 $2\mathrm{g}$ 碳酸氢钠和 $15\mathrm{mL}$ 乙腈，振摇均匀后超声提取 $10\mathrm{min},4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，移出有机相，残渣再加入 $10\mathrm{mL}$ 乙腈重复提取1次，合并提取液，加入约 $3\mathrm{g}$ 氯化钠盐析， $4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，取上清液，于 $45^{\circ}C$ 减压旋转蒸发至近干，以 $5\mathrm{mL}20\%$ 甲醇水溶解残渣，按7.2步骤净化。

# 7.1.3 蜂蜜

称取 $2\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样置于 $50\mathrm{mL}$ 具塞塑料离心管中，加入 $5\mathrm{mL}$ 饱和碳酸氢钠溶液振摇，加入 $15\mathrm{mL}$ 乙腈，涡动 $30\mathrm{s}$ 后超声提取 $10\mathrm{min},4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，移出有机相，残渣再加入 $10\mathrm{mL}$ 乙腈重复提取1次，合并提取液，加入约 $3\mathrm{g}$ 氯化钠盐析， $4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，取上清液，于 $45^{\circ}\mathrm{C}$ 减压旋转蒸发至近干，以 $5\mathrm{mL}20\%$ 甲醇水溶解残渣，按7.2步骤净化。

# 7.1.4 牛奶、果汁及葡萄酒等

称取 $2\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）试样置于 $50~\mathrm{mL}$ 具塞塑料离心管中，加入约 $2\mathrm{g}$ 碳酸氢钠和 $15~\mathrm{mL}$ 乙腈，涡动 $30~\mathrm{s}$ 后超声提取 $10\mathrm{min},4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，移出有机相，残渣再加入 $10~\mathrm{mL}$ 乙腈重复提取1次，合并提取液，加入约 $3\mathrm{g}$ 氯化钠盐析， $4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，取上清液，于 $45^{\circ}C$ 减压旋转蒸发至近干，以 $5\mathrm{mL}20\%$ 甲醇水溶解残渣，按7.2的步骤净化。

# 7.1.5 鱼、肉及肉制品

称取 $2\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样置于 $50\mathrm{mL}$ 具塞塑料离心管中，加入 $5\mathrm{mL}$ 饱和碳酸氢钠溶液振摇，加入 $15\mathrm{mL}$ 乙腈，涡动 $30\mathrm{s}$ 后超声提取 $10\mathrm{min},4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，移出有机相，残渣再加入 $10\mathrm{mL}$ 乙腈重复提取1次，合并提取液，加入约 $3\mathrm{g}$ 氯化钠盐析， $4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，取上清液，加入 $5\mathrm{mL}$ 正己烷，振摇 $1\mathrm{min},4500\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，弃去正己烷层，乙腈层于 $45^{\circ}\mathrm{C}$ 减压旋转蒸发至近干，以 $5\mathrm{mL}20\%$ 甲醇水溶解残渣，按7.2步骤净化。

# 7.2 净化

将样品提取液上柱，用 $5\mathrm{mL}$ 水淋洗，弃去全部淋洗液，抽干，以 $5\mathrm{mL}$ 乙腈洗脱，保持流速约为 $1\mathrm{mL} / \mathrm{min}$ ，收集洗脱液，于 $45^{\circ}\mathrm{C}$ 氮吹至近干，以 $20\%$ 乙腈水溶液定容 $1\mathrm{mL}$ ，过 $0.22\mu \mathrm{m}$ 滤膜，供测定。

# 7.3 测定

# 7.3.1 液相色谱参考条件

a）色谱柱：Phenomenex Luna $\mathrm{C_{18}}$ 柱， $150\mathrm{mm}\times 2.0\mathrm{mm}$ （内径）， $3\mu \mathrm{m}$ ，或相当者；  
b） 柱温： $35^{\circ}C$   
c）流动相：乙腈—5mmol/L乙酸铵缓冲液 $(35 + 65$ ，体积比)；  
d) 流速: $400 \mu \mathrm{L} / \mathrm{min}$ ;   
e) 进样量: $10 \mu \mathrm{L}$ 。

# 7.3.2 质谱参考条件

a）离子源：电喷雾源(ESI)，正离子模式；  
b) 扫描方式：多反应监测(MRM)；

其他参考质谱条件参见A.1。

# 7.3.3 色谱测定与确证

根据试样中被测物的含量情况,选取响应值适宜的标准工作液进行色谱分析。标准工作液和待测样液中鱼藤酮和印楝素的响应值均应在仪器线性响应范围内。按式(1)进行结果计算。在本方法条件下,鱼藤酮和印楝素的保留时间约为 $5.6 \mathrm{~min}$ 和 $4.2 \mathrm{~min}$ , 鱼藤酮和印楝素标准品的多反应监测 (MRM) 色谱图参见附录 B。

定性标准进行样品测定时，如果检出的色谱峰保留时间与标准样品一致，并且在扣除背景后的样品谱图中，各定性离子的相对丰度与浓度接近的同样条件下得到的标准溶液谱图相比，最大允许相对偏差不超过表1中规定的范围，则可判断样品中存在对应的被测物。

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用数据处理软件或按式(1)计算试样中鱼藤酮和印楝素药物的残留量,计算结果需扣除空白值：

$$
X = c \times \frac {V}{m} \tag {1}
$$

式中：

$X$ ——试样中鱼藤酮或印楝素残留量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c$ ——从标准工作曲线得到的鱼藤酮和印楝素溶液浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——样品溶液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——最终样液所代表的试样质量，单位为克(g)。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录C的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录 D 的要求。

# 10 定量限和回收率

# 10.1 定量限

本方法鱼藤酮和印楝素的定量限分别为为 $0.0005\mathrm{mg / kg}$ 和 $0.002\mathrm{mg / kg}$ 。

# 10.2 回收率

当添加水平为 $0.0005\mathrm{mg / kg}, 0.001\mathrm{mg / kg}, 0.005\mathrm{mg / kg}$ 时，鱼藤酮和印楝素的添加回收率参见附录E。

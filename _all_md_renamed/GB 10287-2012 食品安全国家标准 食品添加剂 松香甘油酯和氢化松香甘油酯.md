# 前言

本标准代替GB10287—1988《食品添加剂松香甘油酯和氢化松香甘油酯》。

本标准与GB10287—1988相比，主要变化如下：

——修改了产品主要成分及命名，增加了主要成分的结构式（见第3章）；  
——增加了感官要求（见表1）；  
——修改了理化指标中“溶解度”改为“溶解性”（见表2）；  
——修改了酸值，由 $3.0\sim 9.0\mathrm{mgKOH / g}$ 改为 $\leqslant 9.0~\mathrm{mg / g}$   
——修改了氢化松香甘油酯软化点，由 $78.0\sim 88.0$ 改为 $78.0\sim 90.0$ （见表2）；  
——修改了松香甘油酯相对密度，由 $1.080\sim 1.090$ 改为 $1.060\sim 1.090$ （见表2）；  
——修改了氢化松香甘油酯相对密度，由 $1.060\sim 1.070$ 改为 $1.060\sim 1.090$ （见表2）；  
——修改了总砷(As)限量指标，由 $0.0002\%$ 改为 $1.0 \mathrm{mg} / \mathrm{kg}$ ，测试方法改为引用GB/T5009.11（见表2）；  
——修改了重金属(Pb)限量指标，由 $0.002\%$ 改为 $10.0 \mathrm{mg} / \mathrm{kg}$ ，测试方法改为引用GB/T5009.74（见表2）；  
——修改了比重，改为相对密度并修改了计算公式（见表2、附录A.4）；  
——修改了酸值、灰分的测定方法，改为直接引用（见表2）。

# 氢化松香甘油酯

# 1 范围

本标准适用于以特、一级脂松香为原料，与甘油酯化反应而制得的食品添加剂松香甘油酯；及以氢化松香为原料，与甘油酯化反应制得的食品添加剂氢化松香甘油酯。

# 2 术语和定义

下列术语和定义适用于本文件。

# 2.1 氢化松香甘油酯

以普通氢化松香(主要成分为二氢枞酸)或高度氢化松香(主要成分为四氢枞酸和二氢枞酸)为原料，与甘油酯化反应，经水蒸气吹蒸处理而制得。

# 附录A

# 检验方法

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682—2008中规定的三级水。

试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603之规定制备。试验方法中所用溶液在未指明溶剂时，均指水溶液。

# A.2 鉴别试验 红外光谱法

# A.2.1 仪器和设备

红外光谱仪。

# A.2.2 试样制备

溴化钾压片法。将试样研成粉末，加入适量溴化钾中，混合研磨均匀，用压片机压成均匀透明薄膜。

# A.2.3 鉴别试验

将试样放于红外光谱仪中测试，得到的红外光谱图与附录B中松香甘油酯的红外光谱图（见图B.1)和氢化松香甘油酯的红外光谱图(见图B.2)应一致。

# A.3 溶解性的测定

将试样与甲苯按 $1:1$ (质量比)溶解，注入清洁、干燥、透明的玻璃试管中，在漫射光下以横向目视的方式进行观察。液体应清澈透明、无杂质、无悬浮物。

# A.4 相对密度的测定

# A.4.1 分析步骤

A.4.1.1 取约 $10\mathrm{g}$ 块状试样，除去表面碎屑，检查应无裂纹及气泡，表面光洁。

A.4.1.2 取一根直径小于 $0.15 \mathrm{~mm}$ 金属丝称，精确至 $0.001 \mathrm{~g}$ 。将金属丝在酒精灯上加热，趁热插入块状试样深度约 $2 \mathrm{~mm} \sim 3 \mathrm{~mm}$ ，冷却，用该金属丝将试样挂在天平一端，称量，精确至 $0.001 \mathrm{~g}$ 。

A.4.1.3 将悬挂着的试样浸入 $25^{\circ} \mathrm{C} \pm 0.5^{\circ} \mathrm{C}$ 的水中恒温 $0.5 \mathrm{~h} \sim 1 \mathrm{~h}$ 后，取出。随即浸入盛有 $25^{\circ} \mathrm{C} \pm 2^{\circ} \mathrm{C}$ 水的烧杯中（烧杯支于三角架上，不与称盘接触）。试样上端距液面不少于 $1 \mathrm{~cm}$ ，试样表面不应附有气泡。迅速称量，精确至 $0.001 \mathrm{~g}$ 。

# A.4.2 结果计算

相对密度以 $d_{25}^{25}$ 表示，按式（A.1）计算：

$$
d _ {2 5} ^ {2 5} = \frac {\rho_ {2 5} (m - m _ {2})}{(m - m _ {1})} \tag {A.1}
$$

式中：

$m$ ——试样和金属丝在空气中的质量，单位为克(g)；

$m_{1}$ ——试样和金属丝在水中的质量，单位为克 $(\mathrm{g})$

$m_{2}$ ——金属丝在空气中的质量，单位为克 $(\mathrm{g})$ ；

$\rho_{25}$ ——水在 $25^{\circ} \mathrm{C}$ 时的相对密度。

两次平行测定的绝对差值应不大于0.005，取其算术平均值为报告值。

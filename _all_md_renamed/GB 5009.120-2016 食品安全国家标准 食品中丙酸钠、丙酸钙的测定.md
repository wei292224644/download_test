# 食品中丙酸钠、丙酸钙的测定

2016-08-31 发布

2017-03-01 实施

# 前言

本标准代替GB/T5009.120—2003《食品中丙酸钠、丙酸钙的测定》、GB/T23382—2009《食品中丙酸钠、丙酸钙的测定 高效液相色谱法》。

本标准与GB/T5009.120—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中丙酸钠、丙酸钙的测定”；  
——增加了液相色谱法；  
——气相色谱柱由填充柱改为毛细管柱，色谱条件也相应变动。

# 食品安全国家标准

# 食品中丙酸钠、丙酸钙的测定

# 1 范围

本标准规定了豆类制品、生湿面制品、面包、糕点、醋、酱油中丙酸钠、丙酸钙的测定方法。  
本标准适用于豆类制品、生湿面制品、面包、糕点、醋、酱油中丙酸钠、丙酸钙的测定。

# 第一法 液相色谱法

# 2 原理

试样中的丙酸盐通过酸化转化为丙酸，经超声波水浴提取或水蒸汽蒸馏，收集后调 $\mathrm{pH}$ ，经高效液相色谱测定，外标法定量其中丙酸的含量。样品中的丙酸钠和丙酸钙以丙酸计，需要时可根据相应参数分别计算丙酸钠和丙酸钙的含量。

# 3 试剂与材料

除非另有说明，所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 磷酸 $\mathrm{(H_3PO_4)}$   
3.1.2 磷酸氢二铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{HPO}_{4}\right]$ 。  
3.1.3 硅油。

# 3.2 试剂配制

3.2.1 磷酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：在 $50~\mathrm{mL}$ 水中加入 $53.5\mathrm{mL}$ 磷酸，混匀后，加水定容至 $1000~\mathrm{mL}$ 。  
3.2.2 磷酸氢二铵溶液 $(1.5\mathrm{g} / \mathrm{L})$ ：称取磷酸氢二铵 $1.5\mathrm{g}$ ，加水溶解定容至 $1000~\mathrm{mL}$ 。

# 3.3 标准品

3.3.1 丙酸标准品 $\mathrm{(C_3H_6O_2)}$ ，CAS：79-09-4，纯度 $\geqslant 97.0\%$   
3.3.2 丙酸标准贮备液 $(10\mathrm{mg / mL})$ ：精确称取 $250.0\mathrm{mg}$ 丙酸标准品于 $25~\mathrm{mL}$ 容量瓶中，加水至刻度， $4^{\circ}C$ 冰箱中保存，有效期为6个月。

# 4 仪器和设备

4.1 高效液相色谱(HPLC)仪：配有紫外检测器或二极管阵列检测器。  
4.2 天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$   
4.3 超声波水浴。

4.4 离心机：转速不低于 $4000\mathrm{r / min}$   
4.5 组织捣碎机。  
4.6 具塞塑料离心管： $50~\mathrm{mL}$   
4.7 水蒸汽蒸馏装置： $500~\mathrm{mL}$ 。  
4.8 鼓风干燥箱。  
4.9 pH计。

# 5 分析步骤

# 5.1 样品制备

固体样品经组织捣碎机捣碎混匀后备用(面包样品需运用鼓风干燥箱, $37^{\circ} \mathrm{C}$ 下干燥 $2 \mathrm{~h} \sim 3 \mathrm{~h}$ 进行风干, 置于组织捣碎机中磨碎); 液体样品摇匀后备用。

# 5.2 试样处理

5.2.1 蒸馏法(适用于豆类制品、生湿面制品、醋、酱油等样品): 样品均质后, 准确称取 $25 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ ), 置于 $500 \mathrm{~mL}$ 蒸馏瓶中, 加入 $100 \mathrm{~mL}$ 水, 再用 $50 \mathrm{~mL}$ 水冲洗容器, 转移到蒸馏瓶中, 加 $1 \mathrm{~mol} / \mathrm{L}$ 磷酸溶液 $20 \mathrm{~mL}, 2$ 滴 $\sim 3$ 滴硅油, 进行水蒸气蒸馏, 将 $250 \mathrm{~mL}$ 容量瓶置于冰浴中作为吸收液装置, 待蒸馏至约 $240 \mathrm{~mL}$ 时取出, 在室温下放置 $30 \mathrm{~min}$ , 用 $1 \mathrm{~mol} / \mathrm{L}$ 磷酸溶液调 $\mathrm{pH}$ 为 3 左右, 加水定容至刻度, 摇匀, 经 $0.45 \mu \mathrm{m}$ 微孔滤膜过滤后, 待液相色谱测定。  
5.2.2 直接浸提法（适用于面包、糕点）：准确称取 $5 \mathrm{~g}$ （精确至 $0.01 \mathrm{~g}$ 试样至 $100 \mathrm{~mL}$ 烧杯中，加水 $20 \mathrm{~mL}$ ，加入 $1 \mathrm{~mol} / \mathrm{L}$ 磷酸溶液 $0.5 \mathrm{~mL}$ ，混匀，经超声浸提 $10 \mathrm{~min}$ 后，用 $1 \mathrm{~mol} / \mathrm{L}$ 磷酸溶液调 $\mathrm{pH}$ 为3左右，转移试样至 $50 \mathrm{~mL}$ 容量瓶中，用水定容至刻度，摇匀。将试样全部转移至 $50 \mathrm{~mL}$ 具塞塑料离心管中，以不低于 $4000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取上清液，经 $0.45 \mu \mathrm{m}$ 微孔滤膜过滤后，待液相色谱测定。

# 6 仪器参考条件

6.1 色谱柱： $\mathrm{C_{18}}$ 柱， $4.6 \mathrm{~mm} \times 250 \mathrm{~mm}, 5 \mu \mathrm{m}$ 或等效色谱柱。  
6.2 流动相： $1.5 \mathrm{~g} / \mathrm{L}$ 磷酸氢二铵溶液，用 $1 \mathrm{~mol} / \mathrm{L}$ 磷酸溶液调 $\mathrm{pH}$ 为 $2.7 \sim 3.5$ （使用时配制）；经 $0.45 \mu \mathrm{m}$ 微孔滤膜过滤。  
6.3 流速： $1.0~\mathrm{mL / min}$   
6.4 柱温： $25^{\circ}C$ 。  
6.5 进样量： $20~\mu \mathrm{L}$   
6.6 波长： $214~\mathrm{nm}$   
6.7 色谱柱清洗参考条件：实验结束后，用 $10\%$ 甲醇清洗 $1\mathrm{h}$ ，再用 $100\%$ 甲醇清洗 $1\mathrm{h}$ 。

# 7 标准曲线绘制

7.1 蒸馏法：准确吸取标准储备液 $0.5\mathrm{mL}, 1.0\mathrm{mL}, 2.5\mathrm{mL}, 5.0\mathrm{mL}, 7.5\mathrm{mL}, 10.0\mathrm{mL}, 12.5\mathrm{mL}$ 置于 $500\mathrm{mL}$ 蒸馏瓶中，其他操作同5.2.1样品前处理，其丙酸标准溶液的最终浓度分别为 $0.02\mathrm{mg/mL}$ 、 $0.04\mathrm{mg/mL}$ 、 $0.1\mathrm{mg/mL}$ 、 $0.2\mathrm{mg/mL}$ 、 $0.3\mathrm{mg/mL}$ 、 $0.4\mathrm{mg/mL}$ 、 $0.5\mathrm{mg/mL}$ ，经 $0.45\mu\mathrm{m}$ 微孔滤膜过滤，浓度由低到高进样，以浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。  
7.2 直接浸提法：准确吸取 $5.0\mathrm{mL}$ 标准储备液于 $50~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，配制成浓度为

$1.0\mathrm{mg / mL}$ 标准工作液。再准确吸取标准工作液 $0.2\mathrm{mL},0.5\mathrm{mL},1.0\mathrm{mL},2.0\mathrm{mL},3.0\mathrm{mL},4.0\mathrm{mL},$ $5.0~\mathrm{mL}$ 至 $10~\mathrm{mL}$ 容量瓶中，分别加入 $1\mathrm{mol} / \mathrm{L}$ 磷酸 $0.2\mathrm{mL}$ ，用水定容至 $10  \mathrm{mL}$ ，混匀。其丙酸标准溶液的最终浓度分别为 $0.02\mathrm{mg / mL},0.05\mathrm{mg / mL},0.1\mathrm{mg / mL},0.2\mathrm{mg / mL},0.3\mathrm{mg / mL},0.4\mathrm{mg / mL},$ $0.5\mathrm{mg / mL}$ ，经 $0.45\mu \mathrm{m}$ 微孔滤膜过滤，浓度由低到高进样，以浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。

# 8 试样溶液的测定

处理后的样液同标准系列同样进机测试。根据标准曲线计算样品中的丙酸浓度。

待测样液中丙酸响应值应在标准曲线线性范围内，超出浓度线性范围则应稀释后再进样分析。

# 9 分析结果的表述

样品中丙酸钠(钙)含量(以丙酸计)，按式(1)计算：

$$
X = \frac {c \times V \times 1 0 0 0}{m \times 1 0 0 0} \times f \quad \dots \tag {1}
$$

式中：

$X$ ——样品中丙酸钠(钙)含量(以丙酸计)，单位为克每千克 $(\mathrm{g} / \mathrm{kg})$ ；

$c$ ——由标准曲线得出的样液中丙酸的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V$ ——样液最后定容体积, 单位为毫升 $(\mathrm{mL})$ ;

$m$ ——样品质量，单位为克 $(\mathrm{g})$ ；

$f$ ——稀释倍数。

试样中测得的丙酸含量乘以换算系数1.2967，即得丙酸钠的含量；

试样中测得的丙酸含量乘以换算系数1.2569，即得丙酸钙含量。

计算结果保留三位有效数字。

# 10 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 11 其他

取样为 $25\mathrm{g}$ ，定容体积为 $250~\mathrm{mL}$ 时，丙酸的检出限为 $0.03\mathrm{g / kg}$ ，定量限为 $0.10\mathrm{g / kg}$ 。

# 第二法 气相色谱法

# 12 原理

试样中的丙酸盐通过酸化转化为丙酸，经水蒸气蒸馏收集后直接进气相色谱，用氢火焰离子化检测器检测，以保留时间定性，外标法定量其中丙酸的含量。样品中的丙酸钠和丙酸钙以丙酸计，需要时，可根据相应参数分别计算丙酸钠和丙酸钙的含量。

# 13 试剂和材料

注：除非另有说明，所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 13.1 试剂

13.1.1 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。  
13.1.2 甲酸 $\mathrm{(CH_2O_2)}$ 。  
13.1.3 硅油。

# 13.2 试剂配制

13.2.1 磷酸溶液 $(10 + 90)$ ：取 $10\mathrm{mL}$ 磷酸加水至 $100~\mathrm{mL}$ 。  
13.2.2 甲酸溶液 $(2 + 98)$ ：取 $1\mathrm{mL}$ 甲酸加水至 $50~\mathrm{mL}$

# 13.3 标准品

丙酸标准品 $\mathrm{(C_3H_6O_2)}$ ，CAS：79-09-4，纯度 $\geq 97.0\%$

# 13.4 标准溶液配制

13.4.1 丙酸标准贮备液 $(10\mathrm{mg / mL})$ ：精确称取 $250.0\mathrm{mg}$ 丙酸标准品于 $25~\mathrm{mL}$ 容量瓶中，加水至刻度， $4^{\circ}C$ 冰箱中保存，有效期为6个月。  
13.4.2 丙酸标准使用液：将贮备液用水稀释成 $10~\mu \mathrm{g / mL}\sim 250~\mu \mathrm{g / mL}$ 的标准系列，临用现配。

# 14 仪器和设备

14.1 气相色谱仪：带氢火焰离子化检测器。   
14.2 天平：感量为 $0.0001\mathrm{g}$ 和感量为 $0.01\mathrm{g}$ 。  
14.3 水蒸气蒸馏装置。  
14.4 鼓风干燥箱。

# 15 分析步骤

# 15.1 样品制备

同5.1。

# 15.2 试样处理

样品均质后，准确称取 $25\mathrm{g}$ ，（面包样品需运用鼓风干燥箱， $37^{\circ}\mathrm{C}$ 下干燥 $2\mathrm{h} \sim 3\mathrm{h}$ 进行风干，置于研钵中磨碎），置于 $500~\mathrm{mL}$ 蒸馏瓶中，加入 $100~\mathrm{mL}$ 水，再用 $50~\mathrm{mL}$ 水冲洗容器，转移到蒸馏瓶中，加 $10~\mathrm{mL}$ 磷酸溶液，2滴～3滴硅油，进行水蒸气蒸馏，蒸馏速度为2滴/s～3滴/s，将 $250~\mathrm{mL}$ 容量瓶置于冰浴中作为吸收液装置，待蒸馏近 $250~\mathrm{mL}$ 时取出，在室温下放置 $30\mathrm{min}$ ，加水至刻度。混匀，供气相色谱分析用。

# 15.3 仪器参考条件

15.3.1 色谱柱：聚乙二醇(PEG)石英毛细管柱，柱长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ ，膜厚度 $0.5\mu \mathrm{m}$ 或同等性能

的色谱柱。

15.3.2 载气：氮气，纯度 $>99.99\%$   
15.3.3 载气流速： $1\mathrm{mL} / \mathrm{min}$   
15.3.4 进样口温度： $250^{\circ}C$   
15.3.5 分流比： $10:1$   
15.3.6 检测器温度： $250^{\circ}C$   
15.3.7 柱温箱温度： $125^{\circ}\mathrm{C}$ 保持 $5\mathrm{min}$ ，然后以 $15^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升到 $180^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ 。  
15.3.8 进样量： $1\mu \mathrm{L}$

# 15.4 标准曲线的制作

取标准系列中各种浓度的标准使用液 $10\mathrm{mL}$ ，加 $0.5\mathrm{mL}$ 甲酸溶液，混匀。将其分别注入气相色谱仪中，测定相应的峰面积或峰高，以标准工作液的浓度为横坐标，以响应值(峰面积或峰高)为纵坐标，绘制标准曲线。

# 15.5 试样溶液的测定

吸取 $10\mathrm{mL}$ 制备的试样溶液(15.1)于试管中，加入 $0.5\mathrm{mL}$ 甲酸溶液，混匀，同标准系列同样进机测试。根据标准曲线计算样品中的丙酸浓度。

# 16 分析结果的表述

样品中丙酸钠(钙)含量(以丙酸计)，按式(2)计算：

$$
X = \frac {c}{m} \times \frac {V}{1 0 0 0} \tag {2}
$$

式中：

$X$ ——样品中丙酸钠(钙)含量(以丙酸计)，单位为克每千克 $(\mathrm{g} / \mathrm{kg})$ ；

$c$ ——由标准曲线得出的样液中丙酸的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$m$ ——样品质量，单位为克 $(\mathrm{g})$ ；

$V$ ——样液最终定容体积，单位为毫升（ $\mathrm{mL}$ ）；

1000 $\mu \mathrm{g} / \mathrm{g}$ 换算至 $\mathrm{g / kg}$ 的系数。

试样中测得的丙酸含量乘以换算系数1.2967，即得丙酸钠的含量；

试样中测得的丙酸含量乘以换算系数1.2569，即得丙酸钙含量。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 17 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 18 其他

取样为 $25\mathrm{g}$ ，定容体积为 $250~\mathrm{mL}$ 时，丙酸的检出限为 $0.03\mathrm{g / kg}$ ，定量限为 $0.10\mathrm{g / kg}$ 。

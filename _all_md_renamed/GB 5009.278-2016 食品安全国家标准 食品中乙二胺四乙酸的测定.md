# 食品中乙二胺四乙酸盐的测定

2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替SN/T1018—2001《出口食品罐头中乙二胺四乙酸含量检验方法》。

本标准与SN/T1018—2001相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中乙二胺四乙酸盐的测定”；  
——增加了标准适用范围；  
——增加了复合调味料中乙二胺四乙酸二钠钙的检测方法。

# 食品安全国家标准

# 食品中乙二胺四乙酸盐的测定

# 1 范围

本标准规定了食品中乙二胺四乙酸盐的液相色谱测定方法。

本标准适用于果酱、果脯、腌渍的蔬菜罐头、除番茄沙司外的蔬菜泥(酱)、坚果与籽类罐头、八宝粥罐头、复合调味料、饮料中乙二胺四乙酸二钠和复合调味料中乙二胺四乙酸二钠钙的测定。本标准不适用于铁强化酱油中乙二胺四乙酸铁钠的测定。

# 2 原理

需测定乙二胺四乙酸二钠的试样用水提取，提取液加入三氯化铁络合，经混合型阴离子(PXA)固相萃取柱净化，液相色谱检测，外标法定量。测定复合调味料时，先测出乙二胺四乙酸盐的三氯化铁络合物总量后，再测定乙二胺四乙酸二钠钙含量，根据两者结果确定乙二胺四乙酸二钠的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 盐酸（HCl）。  
3.1.3 磷酸 $\mathrm{(H_3PO_4)}$ 。  
3.1.4 甲酸 $\mathrm{(CH_2O_2)}$ ：色谱纯。  
3.1.5 四丁基溴化铵 $\mathrm{(C_{16}H_{36}NBr)}$ 。  
3.1.6 乙酸钠 $\mathrm{(CH_3COONa)}$ 。  
3.1.7 三氯化铁 $(\mathrm{FeCl}_3\cdot 6\mathrm{H}_2\mathrm{O})$ 。

# 3.2 试剂配制

3.2.1 三氯化铁溶液：称取 $0.54\mathrm{g}$ 三氯化铁，溶于 $90~\mathrm{mL}$ 水中，加入 $0.10~\mathrm{mL}$ 盐酸，转移到 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。  
3.2.2 $5\%$ 甲酸甲醇水溶液：取 $5 \mathrm{~mL}$ 甲酸、 $20 \mathrm{~mL}$ 甲醇用水定容至 $100 \mathrm{~mL}$ 。  
3.2.3 四丁基溴化铵-乙酸钠混合溶液 $(\mathrm{pH}4.0)$ : 称取 $6.45 \mathrm{~g}$ 四丁基溴化铵、 $2.46 \mathrm{~g}$ 乙酸钠, 加 $1000 \mathrm{~mL}$ 水超声溶解, 加入磷酸调节 $\mathrm{pH}$ 至 $4.0 \pm 0.1$ 。  
3.2.4 四丁基溴化铵-乙酸钠混合溶液 $(\mathrm{pH}2.5)$ ：称取 $6.45\mathrm{g}$ 四丁基溴化铵、 $2.46\mathrm{g}$ 乙酸钠，加 $1000\mathrm{mL}$ 水超声溶解，加入磷酸调节 $\mathrm{pH}$ 至 $2.5 \pm 0.1$ 。

# 3.3 标准品

3.3.1 乙二胺四乙酸二钠标准品 $\mathrm{(C_{10}H_{14}N_2O_8Na_2\cdot 2H_2O}$ ,CAS号：6381-92-6）：纯度 $\geqslant 99\%$ 。

3.3.2 乙二胺四乙酸二钠钙标准品 $\mathrm{(C_{10}H_{14}N_2O_8Na_2Ca}$ ，CAS号：62-33-9)：纯度 $\geqslant 99\%$ 。

# 3.4 标准溶液配制

3.4.1 乙二胺四乙酸二钠标准溶液：准确称取适量的乙二胺四乙酸二钠标准品，用水溶解并定容至 $100~\mathrm{mL}$ 棕色容量瓶中，得浓度为 $10~\mathrm{mg / mL}$ 的标准储备溶液，此溶液转移至储液瓶中 $4^{\circ}C$ 下储存。  
3.4.2 乙二胺四乙酸二钠标准系列溶液：吸取1体积乙二胺四乙酸二钠标准溶液加9体积水稀释成浓度为 $1\mathrm{mg / mL}$ 的标准中间溶液。再用水将标准中间溶液稀释为 $0.5~\mu \mathrm{g / mL},2.0~\mu \mathrm{g / mL},5.0~\mu \mathrm{g / mL},$ $10.0~\mu \mathrm{g / mL},50.0~\mu \mathrm{g / mL}$ 的系列标准工作液。  
3.4.3 乙二胺四乙酸二钠钙标准溶液：准确称取适量的乙二胺四乙酸二钠钙标准品，用水溶解并定容至 $100~\mathrm{mL}$ 棕色容量瓶中，得浓度为 $10~\mathrm{mg / mL}$ 的标准储备溶液，此溶液转移至储液瓶中 $4^{\circ}C$ 下储存。  
3.4.4 乙二胺四乙酸二钠钙标准系列溶液: 吸取 1 体积乙二胺四乙酸二钠钙标准溶液加入 9 体积水稀释成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的标准中间溶液。再用水将乙二胺四乙酸二钠钙标准中间溶液稀释为 $1.0 \mu \mathrm{g} / \mathrm{mL}, 10.0 \mu \mathrm{g} / \mathrm{mL}, 20.0 \mu \mathrm{g} / \mathrm{mL}, 50.0 \mu \mathrm{g} / \mathrm{mL}, 100.0 \mu \mathrm{g} / \mathrm{mL}$ 的系列标准工作液。

# 4 仪器和设备

4.1 液相色谱仪：配紫外检测器。  
4.2 电子天平：感量为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。  
4.3 离心机：转速 $\geqslant 7500\mathrm{r / min}$   
4.4 涡旋振荡器。  
4.5 超声波清洗机。  
4.6 pH计：精度为0.01。  
4.7 24孔固相萃取装置。  
4.8 微孔滤膜： $0.45\mu \mathrm{m}$   
4.9 混合阴离子(PXA)固相萃取柱： $150\mathrm{mg} / 6\mathrm{mL}$ 或相当者。PXA主要成分是含亲水基团的聚苯乙烯/二乙烯基苯共聚物上键合季铵基团。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固体样品

样品用粉碎机粉碎，混合均匀后装入洁净容器内密封并做好标识。含糖量高的样品，需要经过冷冻处理后，再进行粉碎。试样于 $4^{\circ}\mathrm{C}$ 下保存。

# 5.1.2 液体样品

含果粒样品需榨汁机匀浆混合均匀后，装入洁净容器内密封并做好标识。碳酸饮料取 $500\mathrm{g}$ 于 $1000\mathrm{mL}$ 烧杯中，在 $70^{\circ}\mathrm{C}$ 水浴中边加热边搅拌，去除部分二氧化碳，冷却至室温后，装入洁净容器内密封并做好标识。试样于 $4^{\circ}\mathrm{C}$ 下保存。

# 5.2 测定

# 5.2.1 乙二胺四乙酸盐总量测定

# 5.2.1.1 提取

# 5.2.1.1.1 与水不互溶的样品

称取试样 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ), 置于 $50~\mathrm{mL}$ 离心管中, 加入 $25~\mathrm{mL}$ 水, 涡旋混匀, 超声 $20\mathrm{min}$ , $7500\mathrm{r / min}$ 离心 $5\mathrm{min}$ , 取上清液置于 $50~\mathrm{mL}$ 玻璃比色管中。剩余残渣重复提取一次, 离心后合并上清液, 加水定容至 $50~\mathrm{mL}$ 。吸取 $5\mathrm{mL}$ 定容液于 $15~\mathrm{mL}$ 离心管中, 待络合。

# 5.2.1.1.2 与水互溶的样品

称取试样 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 玻璃比色管中，加入 $25~\mathrm{mL}$ 水，涡旋混匀，超声提取 $20\mathrm{min}$ 后，加水定容至 $50~\mathrm{mL}$ 。吸取 $5\mathrm{mL}$ 上清液于 $15~\mathrm{mL}$ 离心管中，待络合。

# 5.2.1.2 络合

向上述提取溶液中加入 $0.5\mathrm{mL}$ 三氯化铁溶液，涡旋混匀 $1\mathrm{min}$ ，超声 $20\mathrm{min}$ ， $7500\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，待净化。

# 5.2.1.3 净化

依次用 $5 \mathrm{~mL}$ 甲醇、 $5 \mathrm{~mL}$ 水活化 PXA 柱, 将待净化液全部上样过柱, 依次用 $5 \mathrm{~mL}$ 水、 $5 \mathrm{~mL}$ 甲醇淋洗, 抽干, 用 $5 \mathrm{~mL} 5 \%$ 甲酸甲醇水溶液洗脱, 抽干, 收集洗脱液定容至 $5 \mathrm{~mL}$ , 过滤, 供液相色谱仪测定。

# 5.2.2 乙二胺四乙酸二钠钙测定（仅指复合调味料）

# 5.2.2.1 提取

见5.2.1.1。

# 5.2.2.2 净化

依次用 $5\mathrm{mL}$ 甲醇、 $5\mathrm{mL}$ 水活化PXA柱，将待净化液全部上样过柱，依次用 $5\mathrm{mL}$ 水、 $5\mathrm{mL}$ 甲醇淋洗，抽干，用 $5\mathrm{mL}5\%$ 甲酸甲醇水溶液洗脱，抽干，收集洗脱液定容至 $5\mathrm{mL}$ ，过滤，供液相色谱仪测定。

# 5.3 仪器参考条件

a）色谱柱：Plus- $\mathrm{C_{18}}$ 柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5.0~\mu \mathrm{m}$ ，或相当色谱柱；  
b) 检测乙二胺四乙酸二钠络合物的流动相: 甲醇: 四丁基溴化铵-乙酸钠混合溶液 $(\mathrm{pH}4.0)=15:85$   
c）检测乙二胺四乙酸二钠钙的流动相：甲醇：四丁基溴化铵-乙酸钠混合溶液 $(\mathrm{pH}2.5) = 15:85$   
d) 流速: $0.8 \mathrm{~mL} / \mathrm{min}$ ;   
e） 柱温： $35~\mathrm{^\circ C}$   
f) 波长: $254 \mathrm{~nm}$ ;  
g）进样量： $10~\mu \mathrm{L}$

# 5.4 标准曲线的制作

# 5.4.1 乙二胺四乙酸盐总量测定

分别吸取 $0\mu \mathrm{L},25\mu \mathrm{L},100\mu \mathrm{L},250\mu \mathrm{L},500\mu \mathrm{L},2500\mu \mathrm{L}$ 浓度为 $1\mathrm{mg / mL}$ 的乙二胺四乙酸二钠标准中间溶液于 $50~\mathrm{mL}$ 比色管中，加入 $25~\mathrm{mL}$ 水和 $1.0~\mathrm{mL}$ 三氯化铁溶液，涡旋混匀 $1\mathrm{min}$ ，超声 $20\mathrm{min}$ ，冷却至室温后，用水定容至刻度，络合所得标准系列工作液浓度分别为 $0.0\mu \mathrm{g / mL},0.5\mu \mathrm{g / mL},$ $2.0\mu \mathrm{g / mL},5.0\mu \mathrm{g / mL},10.0\mu \mathrm{g / mL},50.0\mu \mathrm{g / mL}$ 。再将其注入液相色谱仪中，测定相应的峰面积，以标准工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线，乙二胺四乙酸二钠络合物的标准色谱图见图A.1。

# 5.4.2 乙二胺四乙酸二钠钙含量测定

将乙二胺四乙酸二钠钙系列标准工作液分别注入液相色谱仪中，测定相应的峰面积，以标准工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线，乙二胺四乙酸二钠钙的标准色谱图见图A.2。

# 5.5 试样溶液的测定

将试样溶液注入液相色谱仪中，以保留时间定性，同时记录峰面积，根据标准曲线得到待测液中乙二胺四乙酸二钠络合物或乙二胺四乙酸二钠钙的浓度。

# 5.6 空白试验

除不加试样外，按与试样相同的步骤操作。

# 6 分析结果的表述

# 6.1 除复合调味料外其他试样

试样中乙二胺四乙酸盐(以乙二胺四乙酸二钠计)含量按式(1)计算：

$$
X _ {1} = \frac {\rho_ {1} \times V}{m _ {1}} \times \frac {1 0 0 0}{1 0 0 0} \tag {1}
$$

式中：

$X_{1}$ ——试样中乙二胺四乙酸二钠的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho_{1}$ ——试样溶液中乙二胺四乙酸二钠的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——被测试样总体积，单位为毫升 $(\mathrm{mL})$ ；

$m_{1}$ ——称取试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果扣除空白值。计算结果保留三位有效数字。

# 6.2 复合调味料

试样中乙二胺四乙酸盐的含量应分为乙二胺四乙酸二钠和乙二胺四乙酸二钠钙分别表述。其中乙二胺四乙酸二钠钙含量按式(2)计算：

$$
X _ {2} = \frac {\rho_ {2} \times V}{m _ {2}} \times \frac {1 0 0 0}{1 0 0 0} \tag {2}
$$

式中：

$X_{2}$ ——试样中乙二胺四乙酸二钠钙的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho_{2}$ ——试样溶液中乙二胺四乙酸二钠钙的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——被测试样总体积，单位为毫升 $(\mathrm{mL})$ ；

$m_{2}$ ——称取试样的质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

计算结果扣除空白值。计算结果保留三位有效数字。

乙二胺四乙酸二钠含量按式(3)计算：

$$
X _ {3} = \left(X _ {1} \times \frac {2 9 2 . 2 4}{3 3 6 . 2 1} - X _ {2} \times \frac {2 9 2 . 2 4}{3 7 6 . 2 1}\right) \times \frac {3 3 6 . 2 1}{2 9 2 . 2 4}. \tag {3}
$$

式中：

$X_{3}$ ——试样中乙二胺四乙酸二钠的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$X_{1}$ ——试样中乙二胺四乙酸盐络合物计算所得乙二胺四乙酸二钠的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

292.24——乙二胺四乙酸的相对分子质量；

336.21——乙二胺四乙酸二钠的相对分子质量；

$X_{2}$ ——试样中乙二胺四乙酸二钠钙的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

376.21——乙二胺四乙酸二钠钙的相对分子质量。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 8 其他

本方法的检出限为 $0.01\mathrm{g / kg}$ ，定量限为 $0.03\mathrm{g / kg}$

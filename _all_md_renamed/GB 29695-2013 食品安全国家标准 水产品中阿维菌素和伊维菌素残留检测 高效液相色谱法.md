# 高效液相色谱法

# 1 范围

本标准规定了水产品中阿维菌素和伊维菌素残留量的制样和高效液相色谱测定方法。

本标准适用于鱼的可食性组织中阿维菌素和伊维菌素残留量的检测。

# 3 原理

试料中残留的阿维菌素和伊维菌素，用乙腈提取，正己烷除脂，碱性氧化铝柱净化，N-甲基咪唑和三氟乙酸酐衍生化，高效液相色谱-荧光法测定，外标法定量。

# 4 试剂与材料

以下所用的试剂，除特别注明外均为分析纯试剂；水为符合GB/T6682规定的一级水。

4.1 阿维菌素标准品： $\mathrm{B_{la}}$ 含量 $\geqslant 94\%$ 。伊维菌素标准品： $\mathrm{B_{la}}$ 含量 $\geqslant 92\%$   
4.2 乙腈：色谱纯。  
4.3 甲醇：色谱纯。  
4.4 正己烷  
4.5 三氟乙酸酐  
4.6 N-甲基咪唑   
4.7 冰乙酸：优级纯。  
4.8 无水硫酸钠： $650^{\circ}\mathrm{C}$ 烘 $4\mathrm{h}$ ，干燥器中冷却备用。  
4.9 碱性氧化铝固相萃取柱： $1\mathrm{g} / 3\mathrm{mL}$ ，或相当者。  
4.10 衍生化试剂A: 取N-甲基咪唑 $1 \mathrm{~mL}$ 、乙腈 $1 \mathrm{~mL}$ , 混匀, 现配现用。  
4.11 衍生化试剂B：取三氟乙酸酐 $1 \mathrm{~mL}$ 、乙腈 $2 \mathrm{~mL}$ ，混匀，现配现用。

4.12 $0.4\%$ 乙酸溶液：取冰乙酸 $0.4 \mathrm{~mL}$ ，用水溶解并稀释至 $100 \mathrm{~mL}$ 。  
4.13 $100 \mu \mathrm{g} / \mathrm{mL}$ 阿维菌素和伊维菌素混合标准贮备液: 精密称取阿维菌素和伊维菌素标准品各 $10 \mathrm{mg}$ ,于 $100 \mathrm{~mL}$ 量瓶中, 用甲醇溶解并稀释至刻度, 配制成浓度为 $100 \mu \mathrm{g} / \mathrm{mL}$ 的阿维菌素和伊维菌素混合标准贮备液。- $18^{\circ} \mathrm{C}$ 以下保存, 有效期6个月。  
4.14 $10 \mu \mathrm{g} / \mathrm{mL}$ 阿维菌素和伊维菌素混合标准工作液: 精密量取 $100 \mu \mathrm{g} / \mathrm{mL}$ 阿维菌素和伊维菌素混合标准贮备液 $10 \mathrm{~mL}$ , 于 $100 \mathrm{~mL}$ 容量瓶中, 用甲醇稀释至刻度, 配制成浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 阿维菌素和伊维菌素混合标准工作液。 $2 \sim 8^{\circ} \mathrm{C}$ 保存, 有效期3个月。

# 5 仪器

5.1 高效液相色谱仪：配荧光检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 旋涡混合器  
5.5 超声波清洗器  
5.6 离心机  
5.7 氮吹仪   
5.8 旋转蒸发仪   
5.9 固相萃取装置  
5.10 具塞玻璃离心管： $5\mathrm{mL}$ 。  
5.11 无水硫酸钠柱：玻璃层析柱（ $\varphi 10 \mathrm{~mm} \times 150 \mathrm{~mm}$ ，2号砂芯）中装填 $5 \mathrm{~g}$ 无水硫酸钠。  
5.12 茄形瓶： $50~\mathrm{mL}$ 。  
5.13 具塞聚丙烯离心管： $50~\mathrm{mL}$ 。  
5.14 滤膜： $0.45\mu \mathrm{m}$ 。

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-18 ℃ 以下保存。

# 7 测定步骤

# 7.1 标准曲线的制备

精密量取 $1\mu \mathrm{g / mL}$ 阿维菌素和伊维菌素混合标准工作液适量，用甲醇稀释，配制成浓度为0.1、0.2、0.5、1和 $2\mu \mathrm{g / mL}$ 的系列标准溶液，各取 $0.1\mathrm{mL}$ 于 $5\mathrm{mL}$ 离心管中，于 $45\sim 55^{\circ}C$ 水浴氮气吹干，按衍生化步骤处理。供高效液相色谱测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.2 提取

称取试料（ $5 \pm 0.05$ ）g，于 $50 \mathrm{~mL}$ 聚丙烯离心管中，加乙腈 $15 \mathrm{~mL}$ ，旋涡混合 $1 \mathrm{~min}$ ，超声 $30 \mathrm{~min}$ ， $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，取上清液于另一 $50 \mathrm{~mL}$ 聚丙烯离心管中。残渣中加入乙腈 $10 \mathrm{~mL}$ ，重复提取一次，合并两次提取液。在提取液中加正己烷 $10 \mathrm{~mL}$ ，充分涡旋混合 $1 \mathrm{~min}$ ， $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，弃上层正己烷层。提取液中再加正己烷 $10 \mathrm{~mL}$ ，重复提取一次，弃正己烷层，备用。

# 7.2 净化

取备用液，依次过用乙腈 $10\mathrm{mL}$ 活化的无水硫酸钠柱和碱性氧化铝固相萃取柱，收集滤液。待滤液流至将尽，用乙腈 $10\mathrm{mL}$ 冲洗离心管，转至无水硫酸钠柱和碱性氧化铝固相萃取柱，吹干，合并滤液于茄形瓶中，于 $40\sim 50^{\circ}\mathrm{C}$ 旋转蒸发至干。用乙腈 $3\mathrm{mL}$ 分2次溶解残余物，并转至 $5\mathrm{mL}$ 具塞玻璃离心管中， $45\sim 55^{\circ}\mathrm{C}$ 水浴氮气吹干。

# 7.3 衍生化

于上述 $5 \mathrm{~mL}$ 离心管中加衍生化试剂 $\mathrm{A} 100 \mu \mathrm{L}$ ，盖紧塞子，旋涡混合 $30 \mathrm{~s}$ ，再加衍生化试剂 B 150 $\mu \mathrm{L}$ ，盖紧塞子，旋涡混合 $30 \mathrm{~s}$ ，密封避光，室温下衍生 $15 \mathrm{~min}$ ，加甲醇 $750 \mu \mathrm{L}$ ，旋涡混合 $30 \mathrm{~s}$ ，过滤，供高效液相色谱测定。

# 7.4 测定

# 7.4.1 色谱条件

色谱柱： $\mathrm{C_{18}}$ （ $250 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ ，粒径 $5 \mu \mathrm{m}$ ），或相当者；

流速： $1.5\mathrm{mL / min}$

进样量： $20\mu \mathrm{L}$

柱温： $35~\mathrm{^\circ C}$

检测器：激发波长 $365 \mathrm{~nm}$ ，发射波长 $475 \mathrm{~nm}$ 。

# 7.4.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试样溶液中阿维菌素和伊维菌素响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和空白添加试样溶液的高效液相色谱图见附录A。

# 7.5 空白实验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算

试料中阿维菌素或伊维菌素的残留量 $(\mu \mathrm{g} / \mathrm{kg})$ ：按下式计算

$$
X = \frac {C \times V}{m}
$$

式中：

$X$ ——供试试料中相应的阿维菌素或伊维菌素残留量， $\mu \mathrm{g} / \mathrm{kg}$ ；

C——试样溶液中相应的阿维菌素或伊维菌素的浓度, ng/mL;

$V$ ——试样溶液总体积, $\mathrm{mL}$ ;

$m$ 供试试料质量， $\mathrm{g}$

注：计算结果需扣除空白值，测定结果用平行测定后的算术平均值表示，保留三位有效数字。

# 9 方法的灵敏度、准确度和精密度

# 9.1 灵敏度

本方法检测限为 $2\mu \mathrm{g / kg}$ ，定量限为 $4\mu \mathrm{g / kg}$

# 9.2 准确度

本方法在 $4\sim 20~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

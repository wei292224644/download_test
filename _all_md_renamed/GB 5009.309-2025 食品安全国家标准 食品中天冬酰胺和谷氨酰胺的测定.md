# 食品安全国家标准

# 食品中天冬酰胺和谷氨酰胺的测定

# 1 范围

本标准规定了食品中天冬酰胺和谷氨酰胺测定的柱前衍生-高效液相色谱法和液相色谱-串联质谱法。

第一法中直接提取法适用于特殊膳食用食品中游离态天冬酰胺和游离态谷氨酰胺的测定，酶解提取法适用于特殊膳食用食品中天冬酰胺总量和谷氨酰胺总量的测定。

第二法适用于特殊膳食用食品中天冬酰胺总量和谷氨酰胺总量的测定。

# 第一法 柱前衍生-高效液相色谱法

# 2 原理

试样经直接提取或灰色链霉菌蛋白酶酶解，天冬酰胺和谷氨酰胺与丹磺酰氯进行衍生反应，衍生物经 $\mathrm{C_{18}}$ 柱分离，紫外检测器检测，外标法定量。

# 3 试剂和材料

除另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.2 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。  
3.1.3 丹磺酰氯 $(\mathrm{C_{12}H_{12}ClNO_2S})$ 。  
3.1.4 无水碳酸钠 $\mathrm{(Na_2CO_3)}$ 。  
3.1.5 无水磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4)$ 。  
3.1.6 盐酸甲胺 $\mathrm{(CH_5N\cdot HCl)}$   
3.1.7 三羟甲基氨基甲烷（Tris碱， $\mathrm{C_4H_{11}O_3N}$ ）。  
3.1.8 灰色链霉菌蛋白酶(XIV型；来源于灰色链霉菌)：酶活力 $\geqslant 3.5\mathrm{U / mg}$ 。  
3.1.9 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ ：色谱纯，含量 $\geq 85\%$   
3.1.10 浓盐酸（HCl）。

# 3.2 试剂配制

3.2.1 丹磺酰氯溶液 $(2.0\mathrm{g} / \mathrm{L})$ ：称取 $0.2\mathrm{g}$ 丹磺酰氯，用乙腈溶解并定容至 $100~\mathrm{mL}$ ，现配现用。  
3.2.2 盐酸溶液 $(1.0\mathrm{mol} / \mathrm{L})$ ：吸取 $8.3~\mathrm{mL}$ 浓盐酸，加水稀释至 $100~\mathrm{mL}$ 。  
3.2.3 Tris缓冲液 $(0.05\mathrm{mol} / \mathrm{L})$ ：称取 $6.06\mathrm{g}$ 三羟甲基氨基甲烷于 $1000~\mathrm{mL}$ 烧杯中，加水 $800~\mathrm{mL}$ 溶解，用 $1.0\mathrm{mol} / \mathrm{L}$ 盐酸溶液调节 $\mathrm{pH}$ 至 $8.0\pm 0.1$ ，用水稀释至 $1000~\mathrm{mL}$ 。 $4^{\circ}\mathrm{C}$ 保存，有效期1个月。

3.2.4 蛋白酶溶液 $(52.5\mathrm{U} / \mathrm{mL})$ ：称取 $0.15\mathrm{g}$ 灰色链霉菌蛋白酶，用Tris缓冲液溶解并定容至 $10~\mathrm{mL}$ 。现配现用。  
3.2.5 碳酸钠缓冲溶液 $(60\mathrm{mmol / L})$ ：称取 $0.318\mathrm{g}$ 无水碳酸钠，加 $40~\mathrm{mL}$ 水溶解，用 $1.0\mathrm{mol / L}$ 盐酸溶液调 $\mathrm{pH}$ 至 $9.5\pm 0.1$ ，用水定容至 $50~\mathrm{mL}$ 。现配现用。  
3.2.6 盐酸甲胺溶液 $(20\mathrm{g / L})$ ：称取 $2.0\mathrm{g}$ 盐酸甲胺，用水溶解并定容至 $100~\mathrm{mL}$ 。 $4^{\circ}C$ 保存，有效期3个月。  
3.2.7 磷酸氢二钠溶液 $(10\mathrm{mmol / L})$ ：称取无水磷酸氢二钠 $1.42\mathrm{g}$ ，用 $800~\mathrm{mL}$ 水溶解，用磷酸调节 $\mathrm{pH}$ 至 $6.5\pm 0.1$ ，用水稀释至 $1000~\mathrm{mL}$ 。  
3.2.8 $20\%$ 甲醇溶液：量取 $20~\mathrm{mL}$ 甲醇，加水稀释至 $100~\mathrm{mL}$

# 3.3 标准品

3.3.1 天冬酰胺标准品 $\mathrm{(C_4H_8N_2O_3}$ ,CAS号：70-47-3）：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.2 谷氨酰胺标准品 $\left(\mathrm{C}_{5} \mathrm{H}_{10} \mathrm{~N}_{2} \mathrm{O}_{3}\right)$ , CAS号: 56-85-9): 纯度 $\geqslant 99\%$ , 或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 天冬酰胺标准储备液 $(2.00\mathrm{mg / mL})$ ：准确称取 $100\mathrm{mg}$ （精确至 $0.0001\mathrm{g}$ )天冬酰胺标准品，用 $20\%$ 甲醇溶液溶解并定容至 $50~\mathrm{mL}$ 。 $4^{\circ}C$ 避光保存，有效期3个月。  
3.4.2 谷氨酰胺标准储备液 $(2.00\mathrm{mg / mL})$ ：准确称取 $100\mathrm{mg}$ （精确至 $0.0001\mathrm{g}$ )谷氨酰胺标准品，用 $20\%$ 甲醇溶液溶解并定容至 $50~\mathrm{mL}$ 。 $4^{\circ}C$ 避光保存，有效期3个月。  
3.4.3 混合标准中间液 $(20.0\mu \mathrm{g / mL})$ ：分别准确吸取天冬酰胺和谷氨酰胺标准储备液各 $1.00~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。现配现用。  
3.4.4 混合标准系列工作液：分别准确吸取适量混合标准中间液 $(20.0\mu \mathrm{g / mL})$ 于 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度，获得质量浓度分别为 $0\mu \mathrm{g / mL},0.500\mu \mathrm{g / mL},1.00\mu \mathrm{g / mL},2.00\mu \mathrm{g / mL},5.00\mu \mathrm{g / mL},$ $10.0\mu \mathrm{g / mL}$ 和 $20.0\mu \mathrm{g / mL}$ 的标准系列工作液。现配现用。

# 3.5 材料

3.5.1 螺纹带盖玻璃瓶： $20~\mathrm{mL}$   
3.5.2 有机微孔滤膜： $0.22\mu \mathrm{m}$

# 4 仪器和设备

4.1 高效液相色谱仪：配紫外检测器或二极管阵列检测器。  
4.2 电子天平：感量 $0.001\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
4.3 涡旋混合器。  
4.4 恒温水浴装置： $37^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 。  
4.5 pH计：精度0.01。  
4.6 离心机：转速 $\geqslant 5000\mathrm{r / min}$

# 5 分析步骤

# 5.1 试样制备

非粉状固体试样粉碎并混合均匀，粉状固体试样或液体试样摇匀。

# 5.2 试样提取

# 5.2.1 直接提取法

固体试样称取 $5.0\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于 $50~\mathrm{mL}$ 干燥烧杯中，加适量温水 $(40^{\circ}\mathrm{C}\sim 45^{\circ}\mathrm{C})$ 溶解，冷却至室温后转移并定容至 $25~\mathrm{mL}$ 。准确移取 $1.00~\mathrm{mL}$ 于 $150~\mathrm{mL}$ 锥形瓶中，加入约 $25~\mathrm{mL}$ 水涡旋混匀，用 $1.0\mathrm{mol / L}$ 盐酸溶液调节 $\mathsf{pH}$ 至 $4.5\pm 0.1$ ，然后转移至 $250~\mathrm{mL}$ 容量瓶中，用水定容至刻度，摇匀，离心或者过滤得到待衍生试样溶液。

液体试样称取 $1.0\mathrm{g}\sim 5.0\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 于 $150~\mathrm{mL}$ 锥形瓶中，后续操作步骤同固体试样。

# 5.2.2 酶解提取法

固体试样称取 $5.0\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于 $50~\mathrm{mL}$ 干燥烧杯中，加适量温水 $(40^{\circ}\mathrm{C}\sim 45^{\circ}\mathrm{C})$ 溶解，冷却至室温后转移并定容至 $25~\mathrm{mL}$ 。准确移取 $1.00~\mathrm{mL}$ 于 $20~\mathrm{mL}$ 螺纹带盖玻璃瓶中，依次加入 $0.5~\mathrm{mL}$ 蛋白酶溶液、 $3.0~\mathrm{mL}$ Tris缓冲液、 $0.2~\mathrm{mL}$ 甲醇，涡旋混匀，于 $37^{\circ}\mathrm{C}$ 恒温水浴酶解 $16\mathrm{h}$ 。取出玻璃瓶，冷却至室温，将酶解液全部转移至 $250~\mathrm{mL}$ 容量瓶中，用水定容至刻度，摇匀。离心或者过滤得到待衍生试样溶液。

液体试样称取 $0.5\mathrm{g}\sim 1.0\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 于 $20~\mathrm{mL}$ 螺纹带盖玻璃瓶中，后续操作步骤同固体试样。

# 5.3 衍生反应

准确吸取待衍生试样溶液 $1.00\mathrm{mL}$ 至 $15\mathrm{mL}$ 离心管中，加入 $1.00\mathrm{mL}$ 碳酸钠缓冲溶液和 $1.00\mathrm{mL}$ 丹磺酰氯溶液，充分混合，室温避光反应 $2\mathrm{h}$ （反应约 $1\mathrm{h}$ 时取出振摇均匀)，加入 $0.10\mathrm{mL}$ 盐酸甲胺溶液涡旋混合，终止反应，避光静置至沉淀完全，经 $0.22\mu \mathrm{m}$ 滤膜过滤，待测。

另准确吸取 $1.00\mathrm{mL}$ 混合标准系列工作液，与待衍生试样溶液同步进行衍生。

# 5.4 仪器参考条件

色谱参考条件如下：

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱 $(250\mathrm{mm}\times 4.6\mathrm{mm},5\mu \mathrm{m})$ 或性能相当者；  
b）流动相A为 $10\mathrm{mmol / L}$ 磷酸氢二钠溶液，流动相B为甲醇，梯度洗脱条件见表1；  
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d） 柱温： $35~^\circ C$   
e) 检测波长: $247 \mathrm{~nm}$ ;   
f) 进样量: $10 \mu \mathrm{L}$ 。

# 5.5 标准曲线的制作

将衍生后的混合标准系列工作液分别注入高效液相色谱仪中，以混合标准系列工作液中天冬酰胺或谷氨酰胺的质量浓度为横坐标，以天冬酰胺衍生物或谷氨酰胺衍生物的峰面积为纵坐标，分别绘制标准曲线。标准溶液衍生后的色谱图见附录A中图A.1。

# 5.6 试样溶液的测定

将衍生后的待测液注入高效液相色谱仪中，得到相应的峰面积，根据标准曲线得到试样溶液中天冬酰胺或谷氨酰胺的质量浓度。待测液中目标化合物的响应值应在标准曲线线性范围内，超过线性范围则应对待衍生试样溶液进行稀释，重新按样品衍生分析步骤处理后再进样分析。

# 5.7 空白试验

除不加试样外，其他操作与样品的操作一致。

# 6 分析结果的表述

# 6.1 游离态天冬酰胺和游离态谷氨酰胺

直接提取法试样中游离态天冬酰胺、游离态谷氨酰胺的含量按公式(1)计算：

$$
X = \frac {\rho \times V \times f}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中游离态天冬酰胺、游离态谷氨酰胺的含量，单位为毫克每克 $(\mathrm{mg / g})$ ；

$\rho$ ——由标准曲线得到的试样溶液中天冬酰胺、谷氨酰胺的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释因子；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

1000——单位换算系数。

计算结果保留3位有效数字。

# 6.2 天冬酰胺总量和谷氨酰胺总量

酶解提取法试样中天冬酰胺总量、谷氨酰胺总量按公式(2)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times f}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中天冬酰胺总量、谷氨酰胺总量，单位为毫克每克 $(\mathrm{mg / g})$ ；  
$\rho$ ——由标准曲线得到的试样溶液中天冬酰胺、谷氨酰胺的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；  
$\rho_0$ ——由标准曲线得到的空白试验溶液中天冬酰胺、谷氨酰胺的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；  
$V$ ——试样最终定容体积，单位为毫升 $(\mathrm{mL})$ ；  
$f$ ——稀释因子；  
$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

1000——单位换算系数。

计算结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当固体试样称样量为 $5.0\mathrm{g}$ 时，方法检出限均为 $0.2\mathrm{mg / g}$ ，定量限均为 $0.6\mathrm{mg / g}$ ；当液体试样称样量为 $1.0\mathrm{g}$ 时，方法检出限均为 $0.04\mathrm{mg / g}$ ，定量限均为 $0.12\mathrm{mg / g}$ 。

# 第二法 液相色谱-串联质谱法

# 9 原理

试样经灰色链霉菌蛋白酶酶解，稀释后用液相色谱-串联质谱法测定，同位素内标法定量。

# 13 分析结果的表述

试样中天冬酰胺总量、谷氨酰胺总量按式(3)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times f}{m \times 1 0 0 0 \times 1 0 0 0} \tag {3}
$$

式中：

$X$ ——试样中天冬酰胺总量、谷氨酰胺总量，单位为毫克每克 $(\mathrm{mg / g})$ ；

$\rho$ ——由标准曲线得到的试样溶液中天冬酰胺、谷氨酰胺的质量浓度，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$\rho_0$ ——由标准曲线得到的空白试验溶液中天冬酰胺、谷氨酰胺的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释因子；

$m$ ——试样质量，单位为克(g)；

1000——单位换算系数。

计算结果保留3位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

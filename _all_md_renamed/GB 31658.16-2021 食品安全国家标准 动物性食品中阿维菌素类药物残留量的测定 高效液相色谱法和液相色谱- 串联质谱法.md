# 方法一 高效液相色谱法

# 1 范围

本文件规定了动物性食品中阿维菌素、伊维菌素、多拉菌素和乙酰氨基阿维菌素的制样和高效液相色谱测定方法。

本文件适用于猪、牛、羊的肌肉、肝脏、肾脏和脂肪组织中阿维菌素、伊维菌素、多拉菌素和乙酰氨基阿维菌素残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T 6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的阿维菌素类药物经乙腈提取， $\mathrm{C_{18}}$ 固相萃取柱净化，荧光衍生化，高效液相色谱法测定，外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯  
5.1.3 正己烷 $\mathrm{(C_6H_{14})}$ ：色谱纯。  
5.1.4 乙酸 $\mathrm{(CH_3COOH)}$ ：色谱纯。  
5.1.5 三氟乙酸酐 $\mathrm{(C_4F_6O_3)}$ ：色谱纯。  
5.1.6 三乙胺 $(\mathrm{C_6H_{15}N})$   
5.1.7 1-N-甲基咪唑 $(\mathrm{C_4H_6N_2})$ 。

# 5.2 溶液配制

5.2.1 衍生化试剂A液：取1-N-甲基咪唑 $5\mathrm{mL}$ 、乙腈 $5\mathrm{mL}$ ，混匀。

5.2.2 衍生化试剂B液：取三氟乙酸酐 $5\mathrm{mL}$ 、乙腈 $10\mathrm{mL}$ ，混匀。

5.2.3 洗涤液：取乙腈 $80~\mathrm{mL}$ 、水 $120~\mathrm{mL}$ 、三乙胺 $0.2~\mathrm{mL}$ ，混匀。

# 5.3 标准品

阿维菌素、伊维菌素、多拉菌素、乙酰氨基阿维菌素含量均 $\geq 95\%$ ，具体见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液 $(1\mathrm{mg / mL})$ ：取阿维菌素、伊维菌素、多拉菌素、乙酰氨基阿维菌素标准品各适量（相当于各活性成分约 $10~\mathrm{mg}$ )，精密称定，于 $10~\mathrm{mL}$ 容量瓶中，加乙腈溶解并稀释至刻度，配制成浓度为 $1\mathrm{mg / mL}$ 的标准储备液。一 $18^{\circ}C$ 以下避光保存，有效期6个月。  
5.4.2 混合标准工作液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：精密量取标准储备液各 $0.1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的混合标准工作液。一 $18^{\circ}C$ 以下避光保存，有效期3个月。  
5.4.3 混合标准工作液 $(1\mu \mathrm{g / mL})$ ：精密量取 $10~\mu \mathrm{g / mL}$ 的混合标准工作液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，配制成浓度为 $1\mu \mathrm{g / mL}$ 的混合标准工作液。一 $18^{\circ}C$ 以下避光保存，有效期3个月。

# 5.5 材料

5.5.1固相萃取柱：三键键合 $\mathrm{C_{18}}$ ， $500~\mathrm{mg / 6~mL}$ ，或相当者。  
5.5.2 微孔尼龙滤膜： $0.2\mu \mathrm{m}$

# 6 仪器和设备

6.1 高效液相色谱仪：配荧光检测器。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 离心机。  
6.4 组织匀浆机。  
6.5 涡旋混合器。  
6.6 固相萃取装置。  
6.7 氮吹仪。

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管中，加乙腈 $8\mathrm{mL}$ ，涡旋 $2\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。收集上清液，残渣中加乙腈 $8\mathrm{mL}$ ，重复提取1次。合并2次提取液，混匀。取提取液 $4\mathrm{mL}$ ，加水6 $\mathrm{mL}$ 和三乙胺 $10~\mu \mathrm{L}$ ，混匀，备用。

# 8.2 净化

固相萃取柱依次用乙腈、洗涤液各 $4\mathrm{mL}$ 活化，取备用液过柱，依次用洗涤液 $5\mathrm{mL}$ 和水 $3\mathrm{mL}$ 淋洗，抽干，用正己烷 $4\mathrm{mL}$ 淋洗，抽干，用乙腈 $8\mathrm{mL}$ 洗脱。收集洗脱液， $40^{\circ}\mathrm{C}$ 水浴氮气吹干。

# 8.3 衍生化

加衍生化试剂A液 $100\mu \mathrm{L}$ 、衍生化试剂B液 $150~\mu \mathrm{L}$ 、乙酸 $50~\mu \mathrm{L}$ 、三乙胺 $50~\mu \mathrm{L}$ ，涡旋 $30\mathrm{s},20^{\circ}\mathrm{C}$ 避光静置 $15\mathrm{min}$ 。加甲醇 $650~\mu \mathrm{L}$ ，涡旋 $10\mathrm{s},20^{\circ}\mathrm{C}$ 避光静置 $15\mathrm{min}$ ，过滤后供高效液相色谱仪测定。

# 8.4 标准曲线的制备

精密量取混合标准工作液适量，用甲醇稀释，配制成阿维菌素类药物浓度为 $2.5\mu \mathrm{g} / \mathrm{L}, 5\mu \mathrm{g} / \mathrm{L}$

12.5 $\mu \mathrm{g} / \mathrm{L}, 25 \mu \mathrm{g} / \mathrm{L}, 50 \mu \mathrm{g} / \mathrm{L}, 125 \mu \mathrm{g} / \mathrm{L}$ 和 $250 \mu \mathrm{g} / \mathrm{L}$ 的系列标准工作液， $40^{\circ} \mathrm{C}$ 水浴氮气吹干，按8.3衍生化步骤处理，供高效液相色谱仪测定。以测得峰面积为纵坐标、标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.5 测定

液相色谱参考条件如下。

a）色谱柱： $\mathrm{C_{18}}$ （ $250\mathrm{mm}\times 4.6\mathrm{mm},5\mu \mathrm{m})$ ，或相当者；  
b）流动相：乙腈：水（90：10，体积比）；  
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d）检测波长：激发波长： $365~\mathrm{nm}$ ，发射波长： $475~\mathrm{nm}$   
e） 柱温： $35^{\circ}C$   
f）进样量： $50~\mu \mathrm{L}$

# 8.6 测定法

取试样溶液和标准溶液，作单点或多点校准，按外标法以色谱峰面积定量。试样溶液及标准溶液中阿维菌素类药物的峰面积应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液的高效液相色谱图见附录B。

# 8.7 空白试验

取空白试料，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中阿维菌素类药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times C _ {S} \times V _ {1} \times V _ {3}}{A _ {S} \times V _ {2} \times m}. \tag {1}
$$

式中：

$X$ ——试样中阿维菌素类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ ——标准溶液中阿维菌素类药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

A ——试样溶液中阿维菌素类药物的色谱峰面积；

$A_{\mathrm{S}}$ ——标准溶液中阿维菌素类药物的色谱峰面积；

$V_{1}$ ——提取用乙腈总体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——合并后所取提取液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——试样最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——供试试样质量的数值，单位为克（g）。

# 10 方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $1.5\mu \mathrm{g / kg}$ ，定量限为 $5\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $5\mu \mathrm{g} / \mathrm{kg}\sim 100\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

# 18.4.4 定量测定

按18.4.1和18.4.2设定仪器条件，作单点或多点校准，按外标法计算试样中药物的残留量。基质匹配标准溶液及试样溶液中目标药物的特征离子质量色谱峰面积均应在仪器检测的线性范围之内。标准溶液特征离子质量色谱图见附录B。

# 18.5 空白试验

取空白试料，采用完全相同的测定步骤进行平行操作。

# 19 结果计算和表述

试样中阿维菌素类药物的残留量按标准曲线或公式(2)计算。

$$
X = \frac {A \times C _ {\mathrm {S}} \times V _ {1} \times V _ {3}}{A _ {\mathrm {S}} \times V _ {2} \times m} \dots \tag {2}
$$

式中：

$X$ ——试样中阿维菌素类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ ——基质匹配标准溶液中阿维菌素类药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$A$ ——试样溶液中阿维菌素类药物的色谱峰面积；

$A_{\mathrm{S}}$ ——基质匹配标准溶液中阿维菌素类药物的色谱峰面积；

$V_{1}$ ——提取用乙腈总体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——合并后所取提取液体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{3}$ ——试样最终定容体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ ——供试试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 20 方法灵敏度、准确度和精密度

# 20.1 灵敏度

本方法的检测限为 $0.2\mu \mathrm{g / kg}$ ，定量限为 $1\mu \mathrm{g / kg}$

# 20.2 准确度

本方法在 $1\mu \mathrm{g / kg}\sim 100\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$

# 20.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

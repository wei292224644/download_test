# 食品安全国家标准

# 食品添加剂 硫酸镁

# 1 范围

本标准适用于以天然碳酸镁(菱镁矿)和硫酸、硫镁钒及海水为原料生产的食品添加剂硫酸镁。

# 2 化学名称、分子式和相对分子质量

# 2.1 化学名称

无水硫酸镁

一水合硫酸镁

三水合硫酸镁

七水合硫酸镁

硫酸镁干燥品

# 2.2 分子式

无水硫酸镁： $\mathrm{MgSO_4}$

一水合硫酸镁： $\mathrm{MgSO_4\cdot H_2O}$

三水合硫酸镁： $\mathrm{MgSO_4\cdot 3H_2O}$

七水合硫酸镁： $\mathrm{MgSO_4\cdot 7H_2O}$

硫酸镁干燥品： $\mathrm{MgSO_4\cdot nH_2O}$ （ $n$ 是水合作用的平均值，在 $2\sim 3$ 之间）

# 2.3 相对分子质量

无水硫酸镁：120.37（按2007年国际相对原子质量）

一水合硫酸镁：138.38（按2007年国际相对原子质量）

三水合硫酸镁：174.41（按2007年国际相对原子质量）

七水合硫酸镁：246.47（按2007年国际相对原子质量）

# 附录A

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008规定的三级水。试验中所需标准滴定溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 氯化铵溶液： $200\mathrm{g / L}$

A.3.1.2 碳酸铵溶液： $200\mathrm{g / L}$

A.3.1.3 磷酸钠溶液： $60~\mathrm{g / L}$

A.3.1.4 氨水溶液： $2 + 3$

A.3.1.5 氯化钡溶液： $100\mathrm{g / L}$

A.3.1.6 盐酸溶液： $1 + 1$

A.3.1.7 乙酸铅溶液： $80~\mathrm{g / L}$

A.3.1.8 乙酸铵溶液： $100\mathrm{g / L}$

# A.3.2 鉴别方法

# A.3.2.1 镁离子鉴别

取约 $0.5\mathrm{g}$ 试样，溶于 $10~\mathrm{mL}$ 水，加 $5\mathrm{mL}$ 氯化铵溶液、 $5\mathrm{mL}$ 碳酸铵溶液，搅拌，不产生沉淀，再加入 $5\mathrm{mL}$ 磷酸钠溶液，产生白色结晶沉淀，加入 $10~\mathrm{mL}$ 氨水溶液白色结晶沉淀不溶解。

# A.3.2.2 硫酸根离子鉴别

A.3.2.2.1 取约 $0.5\mathrm{g}$ 试样，溶于 $10~\mathrm{mL}$ 水，加入氯化钡溶液，即产生白色沉淀，此白色沉淀不溶于盐酸溶液。  
A.3.2.2.2 取约 $0.5\mathrm{g}$ 试样，溶于 $10~\mathrm{mL}$ 水，加入乙酸铅溶液，即产生白色沉淀，此白色沉淀溶于乙酸铵溶液。  
A.3.2.2.3 取约 $0.5\mathrm{g}$ 试样，溶于 $10~\mathrm{mL}$ 水，加入盐酸不产生沉淀。

# A.4 硫酸镁 $(\mathrm{MgSO}_4)$ 含量的测定

# A.4.1 方法提要

在试样溶液中加入 $\mathrm{pH}\approx 10$ 的氨-氯化铵缓冲溶液甲，以铬黑T为指示剂，用乙二胺四乙酸二钠

(EDTA)标准滴定溶液滴定镁含量。

# A.4.2 试剂和材料

A.4.2.1 氨-氯化铵缓冲溶液甲： $\mathrm{pH}\approx 10$   
A.4.2.2 乙二胺四乙酸二钠标准滴定溶液： $c(\mathrm{EDTA}) = 0.05\mathrm{mol / L}$   
A.4.2.3 铬黑T指示剂。

# A.4.3 分析步骤

称取约 $1.5\mathrm{g}$ 按照A.9灼烧后的试样，精确至 $0.0002\mathrm{g}$ ，置于 $200~\mathrm{mL}$ 烧杯中，加入 $25~\mathrm{mL}$ 水溶解，然后转移至 $250~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。用移液管移取 $25~\mathrm{mL}$ 试样溶液，置于 $250~\mathrm{mL}$ 锥形瓶中，加 $25~\mathrm{mL}$ 水，加入 $10~\mathrm{mL}$ 氨-氯化铵缓冲溶液甲，少量铬黑T指示剂，用乙二胺四乙酸二钠标准滴定溶液滴定至溶液由紫红色变为纯蓝色为终点。

同时同样做空白试验，空白试样溶液除不加试样外，其他加入试剂的种类和量(标准滴定溶液除外)与试样溶液相同。

# A.4.4 结果计算

硫酸镁 $(\mathrm{MgSO_4})$ 含量的质量分数 $w_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\left(V _ {1} - V _ {0}\right) \times c \times M / 1000}{m \times 25 / 250} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——滴定试样溶液消耗乙二胺四乙酸二钠(EDTA)标准滴定溶液的体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白试样溶液所消耗的乙二胺四乙酸二钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——乙二胺四乙酸二钠(EDTA)标准滴定溶液的浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——试样的质量的数值，单位为克 $(\mathrm{g})$ ；

$M$ ——硫酸镁 $(\mathrm{MgSO}_4)$ 的摩尔质量的数值，单位为克每摩尔 $(\mathrm{g/mol})(M = 120.37)$ ；

25 ——移取试样溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

250 ——容量瓶的容积的数值，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算因子。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 铅(Pb)的测定

# A.5.1 试剂和材料

A.5.1.1 三氯甲烷。  
A.5.1.2 盐酸。  
A.5.1.3 硝酸。  
A.5.1.4 氢氧化钠溶液： $250~\mathrm{g / L}$   
A.5.1.5 吡咯烷二硫代甲酸铵(APDC)溶液： $20\mathrm{g / L}$ 。使用前过滤。  
A.5.1.6 铅(Pb)标准溶液: $0.005 \mathrm{mg} / \mathrm{mL}$ 。  
A.5.1.7 水：符合GB6682—2008中二级水的规定。

# A.5.2 仪器和设备

原子吸收分光光度计：配有铅空心阴极灯。

# A.5.3 分析步骤

# A.5.3.1 玻璃仪器的清洗

均以硝酸溶液 $(1 + 5)$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

# A.5.3.2 试样溶液的制备

称取约 $5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $150~\mathrm{mL}$ 烧杯中，加 $30~\mathrm{mL}$ 水，加入 $1\mathrm{mL}$ 盐酸。加热煮沸几分钟，冷却，用水稀释至 $100~\mathrm{mL}$ 。用氢氧化钠溶液调整溶液 $\mathrm{pH}$ 为 $1.0\sim 1.5$ 。将此溶液转移至 $250~\mathrm{mL}$ 分液漏斗中，用水稀释至约 $200~\mathrm{mL}$ 。加 $2\mathrm{mL}$ 吡咯烷二硫代氨基甲酸铵（APDC）溶液，混合。分别用 $20~\mathrm{mL}$ 三氯甲烷萃取两次，收集萃取物于 $50~\mathrm{mL}$ 烧杯中，蒸汽浴蒸发至干（此操作必须在通风橱中进行），于残渣中加入 $3\mathrm{mL}$ 硝酸，继续蒸发至近干。加入 $0.5\mathrm{mL}$ 硝酸和 $10~\mathrm{mL}$ 水，加热直至溶液体积约 $3\mathrm{mL}\sim 5\mathrm{mL}$ 。转移至 $10~\mathrm{mL}$ 容量瓶，用水稀释至刻度，摇匀。

# A.5.3.3 空白试样溶液的制备

取 $30~\mathrm{mL}$ 水置于 $150~\mathrm{mL}$ 烧杯中，加入 $1\mathrm{mL}$ 盐酸。以下操作同A.5.3.2中“加热煮沸几分钟，冷却，用水稀释至 $100~\mathrm{mL}$ …转移至 $10~\mathrm{mL}$ 容量瓶，用水稀释至刻度，摇匀”。

# A.5.3.4 标准溶液的制备

移取 $2.00\mathrm{mL}$ 铅标准溶液置于 $150~\mathrm{mL}$ 烧杯中，加入 $1\mathrm{mL}$ 盐酸。以下操作同A.5.3.2中“加热煮沸几分钟，冷却，用水稀释至 $100~\mathrm{mL}$ …转移至 $10~\mathrm{mL}$ 容量瓶，用水稀释至刻度，摇匀”。

# A.5.3.5 分析步骤

将试样溶液、标准溶液导入原子吸收分光光度计中，于波长 $283.3\mathrm{nm}$ 处，用空气-乙炔火焰，以空白试样溶液调零，测量吸光度。试样溶液的吸收值不应高于标准溶液吸收值。

# A.6 硒 $(\mathrm{Se})$ 的测定

# A.6.1 试剂和材料

A.6.1.1 盐酸羟胺。

A.6.1.2 环己烷。

A.6.1.3 盐酸溶液： $1 + 1$

A.6.1.4 氨水溶液： $1 + 1$

A.6.1.5 2,3-二氨基萘溶液：称取 $0.1\mathrm{g}2,3-$ 二氨基萘和 $0.5\mathrm{g}$ 盐酸羟胺，溶解在 $0.1\mathrm{mol / L}$ 的盐酸溶液中，制成 $100~\mathrm{mL}$ 溶液。

A.6.1.6 硒(Se)标准溶液： $0.01\mathrm{mg / mL}$

# A.6.2 仪器和设备

分光光度计，配有光程为 $1\mathrm{cm}$ 的比色皿。

# A.6.3 分析步骤

称取约 $1\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $150~\mathrm{mL}$ 烧杯中，加 $25~\mathrm{mL}$ 水， $5\mathrm{mL}$ 盐酸溶液，搅拌，加热煮沸 $15\mathrm{min}$ ，冷却。用氨水溶液调节 $\mathrm{pH}$ 为 $2\pm 0.2$ （用精密 $\mathrm{pH}$ 试纸检查），加入 $0.2\mathrm{g}$ 盐酸羟胺，搅拌至溶解，立即加入 $5\mathrm{mL}2,3-$ 二氨基萘溶液，混匀。盖上表面皿，在室温下静置 $100\mathrm{min}$ 。然后全部转移至 $125~\mathrm{mL}$ 分液漏斗中，补加水至约 $60~\mathrm{mL}$ 。用移液管加 $10~\mathrm{mL}$ 环己烷，用力振荡 $2\mathrm{min}$ ，静置分层后，弃去水层。用 $1\mathrm{cm}$ 的比色管，在波长约 $380~\mathrm{nm}$ 处，以水作参比，测定吸光度。其吸光度不应大于标准。

标准是用移液管移取 $3\mathrm{mL}$ 硒标准溶液，与试样同时同样处理。

# A.7 氯化物（以CI计）的测定

称取 $1.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $50~\mathrm{mL}$ 烧杯中，加 $10~\mathrm{mL}$ 水使试样溶解。然后全部转移至 $50~\mathrm{mL}$ 比色管中，加 $2\mathrm{mL}$ 硝酸银溶液 $(17~\mathrm{g / L})$ 和 $1\mathrm{mL}$ 硝酸溶液 $(1 + 2)$ ，再加水至刻度，摇匀。置于暗处放置 $10\mathrm{min}$ ，与标准比浊溶液比较，其浊度不应深于标准。

标准比浊溶液是移取 $3.00\mathrm{mL}$ 氯(Cl)标准溶液 $(0.1\mathrm{mg / mL})$ ，置于 $50~\mathrm{mL}$ 比色管中。与试样溶液同时同样处理。

# A.8 铁 $(\mathbf{Fe})$ 的测定

# A.8.1 试剂和材料

A.8.1.1 铁(Fe)标准溶液: $0.01 \mathrm{mg} / \mathrm{mL}$ 。  
A.8.1.2 其余同GB/T3049—2006中第4章。

# A.8.2 仪器和设备

分光光度计：配有光程为 $4\mathrm{cm}$ 的比色皿。

# A.8.3 分析步骤

# A.8.3.1 工作曲线的绘制

按GB/T3049—2006中6.3的规定，使用 $4\mathrm{cm}$ 比色皿，绘制铁含量为 $0.01\mathrm{mg}\sim 0.1\mathrm{mg}$ 工作曲线。

# A.8.3.2 测定

称取约 $10\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $150~\mathrm{mL}$ 烧杯中，加 $50~\mathrm{mL}$ 水、 $5\mathrm{mL}$ 盐酸溶液，加热煮沸 $4\mathrm{min}$ ，取下冷却至室温。全部转移至 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。

用移液管移取 $10\mathrm{mL}$ 试样溶液，置于 $100~\mathrm{mL}$ 容量瓶中，以下按GB/T3049—2006中6.4从“必要时，加水至 $60~\mathrm{mL}$ …”开始进行操作。

同时做空白试验，空白试样溶液除不加试样外，其他操作和加入的试剂与试样溶液相同。

根据测得的吸光度，从工作曲线上查出相应的铁的质量 $(\mathrm{mg})$ 。

# A.8.4 结果计算

铁(Fe)含量的质量分数 $w_{2}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m _ {1} - m _ {0}}{m \times (1 0 / 1 0 0) / 1 0 0 0} \tag {A.2}
$$

式中：

$m_{1}$ ——从工作曲线上查出的试样溶液中铁的质量的数值，单位为毫克 $(\mathrm{mg})$ ；

$m_{0}$ ——从工作曲线上查出的空白试样溶液中铁的质量的数值，单位为毫克 $(\mathrm{mg})$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

10 ——移取试样溶液的体积的数值，单位为毫升 $(\mathrm{mL})$ ；

100 ——容量瓶的容积的数值，单位为毫升 $(\mathrm{mL})$ ；

1000——换算因子。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $2\mathrm{mg / kg}$ 。

# A.9 灼烧减量的测定

# A.9.1 仪器和设备

A.9.1.1 电热恒温干燥箱：控制温度 $105\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}$ 。

A.9.1.2 高温炉：控制温度 $450^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 。

# A.9.2 分析步骤

称取约 $1\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于在 $450~^\circ \mathrm{C}\pm 25~^\circ \mathrm{C}$ 下质量恒定的瓷坩埚中，于 $105~^\circ \mathrm{C}\pm 2~^\circ \mathrm{C}$ 电热恒温干燥箱中烘 $2\mathrm{h}$ ，再移入 $450~^\circ \mathrm{C}\pm 25~^\circ \mathrm{C}$ 的高温炉中灼烧至质量恒定。

# A.9.3 结果计算

灼烧减量的质量分数 $w_{3}$ ，按式(A.3)计算：

$$
w _ {3} = \frac {m _ {1} - m _ {2}}{m} \times 100 \% \tag{A.3}
$$

式中：

$m_{1}$ ——灼烧前试样和瓷坩埚的质量的数值，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——灼烧后试样和瓷坩埚的质量的数值，单位为克 $(\mathrm{g})$ ；

$m$ ——试样的质量的数值，单位为克 $(\mathrm{g})$ 。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.05\%$ 。

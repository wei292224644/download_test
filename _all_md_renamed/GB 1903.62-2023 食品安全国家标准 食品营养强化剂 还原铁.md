# 1 范围

本标准适用于以铁磷或铁精矿粉为原料，通过煤基直接还原法或气基直接还原法生产得到的食品营养强化剂还原铁。

# 2 化学名称、分子式和相对分子质量

# 2.1 化学名称

还原铁

# 2.2 分子式

Fe

# 2.3 相对分子质量

55.85（按2018年国际相对原子质量）

# 附录A

# 检验方法

警告：试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和防护措施。

# A.1 一般规定

本标准所用试剂和水在未注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品，在未注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验筛的规格按GB/T6003.1的规定选用。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 盐酸。  
A.2.1.2 氢氧化钠。  
A.2.1.3 铁氰化钾。  
A.2.1.4 盐酸溶液 $(2 + 8)$ ：量取 $20~\mathrm{mL}$ 盐酸，用水稀释并定容至 $100~\mathrm{mL}$ 。  
A.2.1.5 铁氰化钾溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 铁氰化钾，用水溶解并定容至 $100~\mathrm{mL}$ 。  
A.2.1.6 氢氧化钠溶液 $(40\mathrm{g / L})$ ：称取 $4\mathrm{g}$ 氢氧化钠，用水溶解并定容至 $100~\mathrm{mL}$ 。

# A.2.2 仪器和设备

电子天平：感量为 $0.01\mathrm{g}$ 。

# A.2.3 鉴别方法

将少量试样溶于盐酸溶液(A.2.1.4)中，有气体逸出，停止后加入铁氰化钾溶液(A.2.1.5)，有黑蓝色沉淀生成。此沉淀加入氢氧化钠溶液(A.2.1.6)后，生成绿白色沉淀，颜色很快转变为绿色，振荡后变为棕色。

# A.3 铁(Fe)含量的测定

# A.3.1 方法提要

试样在隔绝空气条件下加酸溶解后，以1,10-菲啰啉-亚铁作指示剂，用硫酸铈标准滴定溶液滴定，溶液由红色变为淡蓝色为滴定终点，根据所消耗的硫酸铈标准溶液的量计算铁含量。

# A.3.2 试剂和材料

A.3.2.1 硫酸。

A.3.2.2 硫酸溶液 $(3 + 100)$ ：量取 $3\mathrm{mL}$ 硫酸，缓慢倒入盛有 $100~\mathrm{mL}$ 水的烧杯中，不断搅拌，混匀，冷却后使用。  
A.3.2.3 1,10-菲啰啉-亚铁指示剂：称取 $0.7\mathrm{g}$ 硫酸亚铁 $\mathrm{(FeSO_4\cdot 7H_2O)}$ 溶于 $70~\mathrm{mL}$ 水中，加2滴硫酸，加 $1.5\mathrm{g}1$ ，10-菲啰啉 $(\mathrm{C_{12}H_8N_2\cdot H_2O})$ 或 $1.76\mathrm{g}1$ ，10-菲啰啉盐酸盐 $(\mathrm{C_{12}H_8N_2\cdot HCl\cdot H_2O})$ ，溶解

后，加水稀释至 $100~\mathrm{mL}$ 。临用前制备。

A.3.2.4 硫酸铈标准滴定溶液： $c[\mathrm{Ce}(\mathrm{SO}_4)_2] = 0.1\mathrm{mol / L}$

# A.3.3 仪器和设备

A.3.3.1 电子天平：感量分别为 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。

A.3.3.2 电热板。

A.3.3.3 本生阀装置：锥形瓶上带一个塞子，该塞子上插入一根连有一短段橡胶管的玻璃管，橡胶管的侧面有一狭长的切口，其另一端插入一段玻璃棒，使气体能够逸出，而空气不能进入。本生阀装置如图 A.1 所示。

# A.3.3.4 分析步骤

称取约 $0.2\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于本生阀装置（A.3.3.3）的锥形瓶，加 $50~\mathrm{mL}$ 硫酸溶液(A.3.2.2)，迅速盖上瓶塞。在蒸汽浴上加热锥形瓶，使试样完全溶解，冷却，加 $50~\mathrm{mL}$ 沸腾并冷却的水稀释，加2滴1,10-菲啰啉-亚铁指示剂(A.3.2.3)，用硫酸铈标准滴定溶液(A.3.2.4)滴定，溶液由红色变为淡蓝色为终点。

同时进行空白试验。空白试验除不加试样外，其他与试验溶液相同。

# A.3.4 结果计算

铁(Fe)的质量分数 $w_{1}$ 按式(A.1)计算。

$$
w _ {1} = \frac {\left(V _ {1} - V _ {2}\right) \times c \times M}{m _ {1} \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——滴定试样溶液所消耗的硫酸铈标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴定空白溶液所消耗的硫酸铈标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫酸铈标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——铁(Fe)的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})(M = 55.85)$ ；

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$

1000——换算系数。

试验结果以平行测定结果的算术平均值为准，在重复性条件下获得的2次独立测定结果的绝对差值不大于 $1\%$ 。

# A.4 酸不溶物的测定

# A.4.1 方法提要

试样经硫酸溶液溶解后，过滤、洗涤、烘干、称重，测定酸不溶物含量。

# A.4.2 试剂和材料

A.4.2.1 硫酸。  
A.4.2.2 氯化钡。  
A.4.2.3 硫酸溶液 $(1 \mathrm{~mol} / \mathrm{L})$ : 量取 $54 \mathrm{~mL}$ 硫酸, 缓慢倒入盛有约 $600 \mathrm{~mL}$ 水的烧杯中, 不断搅拌, 等溶液冷却至室温后转移至 $1 \mathrm{~L}$ 容量瓶中, 用少量水冲洗烧杯壁 2 次, 倒入容量瓶, 加水定容至刻度, 混匀。  
A.4.2.4 氯化钡溶液 $(122\mathrm{g} / \mathrm{L})$ ：称取 $12.2\mathrm{g}$ 氯化钡，用水溶解并定容至 $100~\mathrm{mL}$ 。

# A.4.3 仪器和设备

A.4.3.1 电子天平：感量分别为 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
A.4.3.2 玻璃砂芯坩埚：滤板孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$   
A.4.3.3 电热恒温干燥箱：精度 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。  
A.4.3.4 本生阀装置：见A.3.3.3。

# A.4.4 分析步骤

称取约 $1.0\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于本生阀装置(A.3.3.3)的锥形瓶中，加入 $25\mathrm{mL}$ 硫酸溶液(A.4.2.3)，迅速盖上瓶塞，沸水浴上加热，直至气体逸尽。用预先于 $105~^\circ \mathrm{C}\pm 2~^\circ \mathrm{C}$ 干燥至恒重(前后2次质量差不超过 $2\mathrm{mg}$ )的玻璃砂芯坩埚过滤。用水洗涤至滤液经氯化钡溶液(A.4.2.4)检验无硫酸盐反应，置于电热恒温干燥箱中，于 $105~^\circ \mathrm{C}\pm 2~^\circ \mathrm{C}$ 下干燥至恒重(前后2次质量差不超过 $2\mathrm{mg}$ )，冷却，称量。

# A.4.5 结果计算

酸不溶物以质量分数 $\varpi_{2}$ 计，按式(A.2)计算。

$$
w _ {2} = \frac {m _ {3} - m _ {4}}{m _ {2}} \times 100 \% \tag{A.2}
$$

式中：

$m_{3}$ ——干燥后玻璃砂芯坩埚和不溶物的质量，单位为克 $(\mathrm{g})$ ；

$m_{4}$ ——玻璃砂芯坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ —称取试样的质量，单位为克 $(\mathrm{g})$

试验结果以平行测定结果的算术平均值为准，在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# A.5 铅的测定

# A.5.1 试剂和材料

A.5.1.1 水：GB/T6682规定的一级水。  
A.5.1.2 硝酸：优级纯。  
A.5.1.3 过氧化氢：优级纯。  
A.5.1.4 硝酸溶液 $(5 + 95)$ ：量取 $5\mathrm{mL}$ 硝酸，用水稀释并定容室 $100~\mathrm{mL}$ 。  
A.5.1.5 过氧化氢溶液 $(30 + 70)$ ：量取 $30~\mathrm{mL}$ 过氧化氢，用水稀释并定容至 $100~\mathrm{mL}$ 。

# A.5.2 仪器和设备

电子天平：感量为 $0.0001\mathrm{g}$ 。

# A.5.3 分析步骤

称取 $0.2\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，加入 $30~\mathrm{mL}$ 硝酸溶液（A.5.1.4），逐滴加入过氧化氢溶液（A.5.1.5）直至试样完全溶解，用硝酸溶液(A.5.1.4)定容至 $50~\mathrm{mL}$ 。按GB5009.12或GB5009.75规定的“测定”方法进行测定。

# A.6 总碑的测定

# A.6.1 试剂和材料

A.6.1.1 水：GB/T6682规定的一级水。  
A.6.1.2 盐酸：优级纯。  
A.6.1.3 过氧化氢：优级纯。  
A.6.1.4 盐酸溶液：量取 $20~\mathrm{mL}$ 盐酸，用水稀释并定容至 $100~\mathrm{mL}$

# A.6.2 仪器和设备

电子天平：感量为 $0.0001\mathrm{g}$ 。

# A.6.3 分析步骤

称取 $0.5\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，缓慢加入 $20~\mathrm{mL}$ 盐酸溶液(A.6.1.4)，逐滴加入过氧化氢直至试样完全溶解，用水定容至 $50~\mathrm{mL}$ 。按GB5009.11或GB5009.76规定的“测定”方法进行测定。

# A.7 总汞的测定

# A.7.1 试剂和材料

A.7.1.1 水：GB/T6682规定的一级水。  
A.7.1.2 硝酸：优级纯。  
A.7.1.3 过氧化氢：优级纯。

A.7.1.4 硝酸溶液：量取 $5\mathrm{mL}$ 硝酸，用水稀释并定容至 $100~\mathrm{mL}$   
A.7.1.5 过氧化氢溶液：量取 $30~\mathrm{mL}$ 过氧化氢，用水稀释并定容至 $100~\mathrm{mL}$

# A.7.2 仪器和设备

电子天平：感量为 $0.0001\mathrm{g}$ 。

# A.7.3 分析步骤

称取 $0.2\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，加入 $30~\mathrm{mL}$ 硝酸溶液（A.7.1.4），再逐滴加入过氧化氢溶液(A.7.1.5)直至试样完全溶解，用硝酸溶液(A.7.1.4)定容至 $50~\mathrm{mL}$ 。按GB5009.17规定的“测定”方法进行测定。

# A.8 细度的测定

# A.8.1 方法提要

试样在不同规格的筛子上进行筛分，不同颗粒的试样彼此分离。根据筛余物残留量计算出试样的细度。

# A.8.2 仪器和设备

A.8.2.1 电子天平：感量为 $0.001\mathrm{g}$   
A.8.2.2 振筛机：摇动频率 $270\mathrm{r / min}\sim 300\mathrm{r / min}$ ，振动频率140次 $\mathrm{/min}\sim 160$ 次/min。  
A.8.2.3 试验筛：100目 $(\phi 200\times 50 - 0.15 / 0.10)$ 、325目 $(\phi 200\times 50 - 0.045 / 0.032)$ 。

# A.8.3 分析步骤

称取约 $100\mathrm{g}$ 试样，精确至 $0.001\mathrm{g}$ ，倒入试验筛中，盖上筛盖，固定在振筛机上，开启振筛机，振筛 $15\mathrm{min}$ ，将筛余物缓慢倒入已称重的玻璃纸上，用干燥软毛刷轻轻刷扫，最后轻敲收集筛余物，准确称量。

# A.8.4 结果计算

细度的质量分数 $\omega_{3}$ 按式(A.3)计算。

$$
w _ {3} = \frac {m _ {5} - m _ {6}}{m _ {5}} \times 100 \% \tag{A.3}
$$

式中：

$m_{5}$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

$m_{6}$ —筛余物的质量的数值，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准，在重复性条件下获得的2次独立测定结果的绝对差值不大于 $0.5\%$ 。

# 食品添加剂 叶绿素铜钾盐

# 1 范围

本标准适用于蚕沙或草类、紫花苜蓿、荨麻、菠菜等植物为原料提取叶绿素，或者直接以叶绿素为原料，经皂化、铜代等步骤加工制得的食品添加剂叶绿素铜钾盐。使用溶剂为丙酮、二氯甲烷、甲醇、乙醇、异丙醇、正己烷和(或)石油醚(沸程为 $90^{\circ}\mathrm{C} \sim 120^{\circ}\mathrm{C}$ )。

# 2 分子式和相对分子质量

# 2.1 分子式

叶绿酸铜二钾 $\mathrm{(C_{34}H_{30}O_5N_4CuK_2)}$

叶绿酸铜三钾 $\mathrm{(C_{34}H_{31}O_6N_4CuK_3)}$

# 2.2 相对分子质量

叶绿酸铜二钾：716.38（按2016年国际相对原子质量）

叶绿酸铜三钾：772.48（按2016年国际相对原子质量）

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水在未注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品在未注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 溶解性

溶于水，几乎不溶于低分子醇、酮类和乙醚，不溶于氯代烷烃、烃类和固定油类。

# A.2.2 最大吸收波长

取A.3.3.1叶绿素铜钾盐含量测定中的试样溶液，在 $405\mathrm{nm}\pm 3\mathrm{nm}$ 和 $630\mathrm{nm}\pm 3\mathrm{nm}$ 的两个波长范围内均有最大吸收峰。

# A.2.3 铜钾离子试验

取 $1\mathrm{g}$ 试样，置于 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 下灼烧至恒重的坩埚中，缓缓加热直至试样完全碳化。将碳化的试样冷却，用 $0.5\mathrm{mL} \sim 1\mathrm{mL}$ 硫酸润湿残渣，继续加热至硫酸蒸汽逸尽，并在 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 的高温炉中灼烧残渣至恒重。在残渣中加入 $10\mathrm{mL}$ 盐酸溶液 $(1 + 3)$ ，在水浴上加热溶解，过滤后补充水至 $10\mathrm{mL}$ ，作为试样溶液，进行如下试验：

a) 取 $5 \mathrm{~mL}$ 试样溶液, 加入 $10 \%$ 氨水溶液, 呈蓝色。  
b) 取 $5 \mathrm{~mL}$ 试样溶液, 加入 $0.5 \mathrm{~mL}$ 的 $0.1 \%$ 二乙基二硫代氨基甲酸钠溶液, 产生红褐色沉淀。  
c）取试样溶液做焰色试验，呈紫色。

# A.3 叶绿素铜钾盐含量（以干基计）的测定

# A.3.1 试剂和材料

A.3.1.1 $0.15\mathrm{mol} / \mathrm{L}$ 磷酸氢二钠溶液：称取 $53.7\mathrm{g}$ 磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4\cdot 12\mathrm{H}_2\mathrm{O})$ ，加水溶解，稀释并定容至 $1000~\mathrm{mL}$ 。  
A.3.1.2 $0.15\mathrm{mol} / \mathrm{L}$ 磷酸二氢钾溶液：称取 $20.4\mathrm{g}$ 磷酸二氢钾（ $\mathrm{KH_2PO_4}$ ），加水溶解，稀释并定容至 $1000~\mathrm{mL}$ 。  
A.3.1.3 磷酸盐缓冲液 $(\mathrm{pH}7.5):0.15\mathrm{mol/L}$ 磷酸氢二钠溶液与 $0.15\mathrm{mol/L}$ 磷酸二氢钾溶液 $21:4$ 混合。

# A.3.2 仪器和设备

分光光度计。

# A.3.3 分析步骤

# A.3.3.1 试样溶液的制备

称取 $0.1\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，加水溶解并稀释定容至 $100~\mathrm{mL}$ 。取 $1\mathrm{mL}$ 上述溶液，用磷酸盐缓冲液 $(\mathrm{pH}7.5)$ 稀释定容至 $100~\mathrm{mL}$ 。

# A.3.3.2 测定

取试样溶液置于 $1\mathrm{cm}$ 比色皿中，以磷酸盐缓冲液 $(\mathrm{pH}7.5)$ 做空白对照，用分光光度计在 $405~\mathrm{nm}\pm$ $3\mathrm{nm}$ 波长范围内的最大吸收波长处测定吸光度。吸光度应控制在 $0.3\sim 0.7$ ，否则应调整试样溶液浓度，再重新测定吸光度。

# A.3.4 结果计算

叶绿素铜钾盐含量（以干基计）的质量分数 $\omega_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {A \times 10 ^ {4}}{565 \times 100 \times m \times \left(1 - w _ {\mathrm {o}}\right)} \times 100 \% \dots \tag{A.1}
$$

式中：

$A$ ——试样溶液的吸光度值；

$10^{4}$ ——体积换算系数；

565—1 cm 比色皿中浓度为 $1\%$ (100 mL 溶剂含 1 g 叶绿素铜钾盐) 的溶液消光值；

100——浓度换算系数；

$m$ ——试样的质量，单位为克(g)；

$w_{\circ}$ ——试样的干燥减量， $\%$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $1.0\%$ 。

注：按照GB5009.3直接干燥法测定干燥减量，干燥温度和时间分别为 $105^{\circ}\mathrm{C}$ 和 $2\mathrm{h}$

# A.4 总铜 $(\mathbf{Cu})$ 的测定

# A.4.1 试剂和材料

同GB5009.13。

# A.4.2 仪器和设备

同GB5009.13。

# A.4.3 试样处理

# A.4.3.1 微波消解

称取 $0.2\mathrm{g}\sim 0.5\mathrm{g}$ ，精确至 $0.0002\mathrm{g}$ ，于微波消解罐中，加入 $5\mathrm{mL}$ 硝酸，按照微波消解的操作步骤消解试样，消解条件参考B.1。冷却后取出消解罐，于 $120^{\circ}\mathrm{C}\sim 140^{\circ}\mathrm{C}$ 赶酸至 $1\mathrm{mL}$ 左右。消解罐放冷后，将消化液转移至 $25~\mathrm{mL}$ 容量瓶中，用少量二级水洗涤消解罐2次～3次，合并洗涤液于容量瓶中，用二级水定容至刻度，混匀备用。同时做空白试验。

# A.4.3.2 干法灰化

称取 $0.1\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，于坩埚中，小火加热，炭化至无烟，转移至 $550^{\circ}\mathrm{C}$ 马弗炉中，灰化 $3\mathrm{h} \sim 4\mathrm{h}$ ，冷却后取出。对于灰化不彻底的试样，加数滴硝酸，小火加热，小心蒸干，再转入 $550^{\circ}\mathrm{C}$ 马弗炉中，继续灰化 $1\mathrm{h} \sim 2\mathrm{h}$ ，至试样呈白灰状，冷却后取出。用适量硝酸溶液 $(1 + 1)$ 溶解并用水定容至 $10\mathrm{mL}$ ，混匀备用。同时做空白试验。

# A.4.4 分析步骤

除试样处理外，其余步骤按GB5009.13规定的方法测定。

# A.5 游离铜 $(\mathbf{Cu})$ 的测定

# A.5.1 试样处理

称取 $0.1\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，加约 $50~\mathrm{mL}$ 水溶解，用 $1\mathrm{mol} / \mathrm{L}$ 盐酸溶液调节 $\mathrm{pH}$ 至3.0，用水定容至 $100~\mathrm{mL}$ ，过滤。

# A.5.2 分析步骤

除试样处理外，其余步骤按GB5009.13规定的方法测定。

# A.6 碱性染料的测定

配制 $5\mathrm{mL}$ 的 $0.05\%$ 试样溶液，加 $1\mathrm{mL}$ 的 $1\mathrm{mol} / \mathrm{L}$ 盐酸溶液，再加 $5\mathrm{mL}$ 正己烷，完全混合后分离。正己烷层不深于淡绿色，即为通过试验。

# A.7 溶剂残留（二氯甲烷、甲醇、异丙醇、正己烷、正庚烷)的测定

# A.7.1 试剂和材料

A.7.1.1 水：符合GB/T6682规定的一级水。  
A.7.1.2 待测组分标准品：二氯甲烷、甲醇、异丙醇、正己烷、正庚烷，色谱纯。  
A.7.1.3 内标物：3-甲基-2-戊酮，色谱纯。  
A.7.1.4 $N$ -甲基吡咯烷酮。

# A.7.2 仪器和设备

气相色谱仪：配备氢火焰离子化检测器(FID)和顶空进样器。

# A.7.3 参考色谱条件

A.7.3.1 色谱柱：石英毛细管柱 $(\Phi 0.53\mathrm{mm}\times 30\mathrm{m})$ ，涂层为二甲基聚硅氧烷，厚度为 $5\mu \mathrm{m}$ ，或同等性能的色谱柱。  
A.7.3.2 载气：氮气。  
A.7.3.3 载气流速： $5\mathrm{mL / min}$   
A.7.3.4 柱温： $35^{\circ}\mathrm{C}$ 保持 $5\mathrm{min}$ ，以 $5^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $90^{\circ}\mathrm{C}$ ，保持 $6\mathrm{min}$ 。  
A.7.3.5 进样口温度： $140^{\circ}C$

A.7.3.6 检测器温度： $300^{\circ}\mathrm{C}$ 。  
A.7.3.7 进样量： $1.0~\mathrm{mL}$

# A.7.4 参考顶空进样条件

A.7.4.1 试样加热温度： $80^{\circ}C$   
A.7.4.2 试样加热时间： $10\mathrm{min}$   
A.7.4.3 注射器温度： $90^{\circ}C$   
A.7.4.4 传质温度： $100^{\circ}\mathrm{C}$ 。

# A.7.5 分析步骤

# A.7.5.1 内标溶液的制备

量取约 $50.0\mathrm{mL}$ 水至一个 $50~\mathrm{mL}$ 容量瓶中定容，称量容量瓶，精确至 $0.0001\mathrm{g}$ 。移取 $15\mu \mathrm{L}3$ -甲基-2-戊酮，将其注入容量瓶，混匀，再称量容量瓶，精确至 $0.01\mathrm{g}$ 。

# A.7.5.2 空白溶液的制备

分别量取 $5.0\mathrm{mL}$ 水和 $1.0\mathrm{mL}$ 内标溶液，移入顶空瓶中，移取 $20~\mu \mathrm{L}N$ -甲基吡咯烷酮，将其注入顶空瓶，封盖， $60^{\circ}\mathrm{C}$ 加热 $10\mathrm{min}$ 并剧烈振摇，混匀。

# A.7.5.3 标准溶液的制备

分别量取 $5.0\mathrm{mL}$ 水和 $1.0\mathrm{mL}$ 内标溶液，移入顶空瓶中，称取 $10\mathrm{mg}$ 待测组分标准品（对每个溶剂分别分析），用N-甲基吡咯烷酮溶解稀释定容至 $100~\mathrm{mL}$ ，移取 $20~\mu \mathrm{L}$ 注入顶空瓶，封盖， $60^{\circ}\mathrm{C}$ 加热 $10\mathrm{min}$ 并剧烈振摇，混匀。

# A.7.5.4 试样溶液的制备

称取 $0.2\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于顶空瓶中，加入 $5.0\mathrm{mL}$ 水和 $1.0\mathrm{mL}$ 内标溶液，移取 $20~\mu \mathrm{L}$ N-甲基吡咯烷酮，将其注入顶空瓶，封盖， $60^{\circ}\mathrm{C}$ 加热 $10\mathrm{min}$ 并剧烈振摇，混匀。

# A.7.5.5 测定

在A.7.3和A.7.4参考操作条件下，分别对空白溶液、标准溶液和试样溶液顶空处理后进行色谱分析。

# A.7.6 结果计算

# A.7.6.1 校准因子 $f_{i}$

校准因子 $f_{i}$ 按式(A.2)计算：

$$
f _ {i} = \frac {m _ {i} \times 5 0}{m _ {\mathrm {s}} \times \left(A _ {\mathrm {f}} - A _ {\mathrm {g}}\right)} \tag {A.2}
$$

式中：

$m_{i}$ ——标准溶液中待测组分的质量，单位为毫克 $(\mathrm{mg})$ ；

50——质量换算系数；

$m_{\mathrm{s}}$ ——标准溶液中内标物的质量，单位为毫克 $(\mathrm{mg})$ ；

$A_{\mathrm{f}}$ ——标准溶液色谱图中待测组分峰面积与内标物峰面积的比值；

$A_{\mathrm{g}}$ ——空白溶液色谱图中待测组分峰面积与内标物峰面积的比值。

# A.7.6.2 待测组分含量

试样中待测组分(二氯甲烷、甲醇、异丙醇、正己烷、正庚烷)的含量 $w_{i}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.3)计算：

$$
w _ {i} = \frac {A _ {i} \times m _ {0} \times f _ {i} \times 1 0 0 0}{m \times 5 0} \quad \dots \tag {A.3}
$$

式中：

$A_{i}$ ——试样溶液色谱图中待测组分峰面积与内标物峰面积的比值；

$m_{0}$ ——内标溶液中内标物的质量，单位为毫克 $(\mathrm{mg})$ ；

$f_{i}$ ——校准因子；

1000——质量换算系数；

$m$ ——试样的质量，单位为克(g)；

50 体积换算系数。

由式(A.3)计算得到二氯甲烷的含量为 $w_{2}$ ，甲醇、异丙醇、正己烷、正庚烷的含量分别为 $w_{3}, w_{4}, w_{5}, w_{6}$ ，四者之和即为试样中溶剂残留（甲醇、异丙醇、正己烷、正庚烷总和）的含量。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $2.0\%$ 。

# 前言

本标准代替GB/T4789.9-2008《食品卫生微生物学检验空肠弯曲菌检验》。

本标准与 GB/T 4789.9-2008 相比，主要变化如下：

——修改了标准的中文名称；  
——修改了范围；  
——修改了设备和材料；  
——修改了培养基和试剂；  
——修改了样品处理；  
——删除了药物敏感性试验；  
——删除了第二法 全自动酶联荧光免疫分析仪筛选法。

# 食品安全国家标准

# 食品微生物学检验 空肠弯曲菌检验

# 1 范围

本标准规定了食品中空肠弯曲菌（Campylobacter jejuni）的检验方法。

本标准适用于食品中空肠弯曲菌的检验。

# 2 设备和材料

除微生物实验室常规灭菌及培养设备外，其他设备和材料如下：

a) 恒温培养箱： $25^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 、 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 、 $42^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ ；  
b) 冰箱: $2^{\circ} \mathrm{C} \sim 5^{\circ} \mathrm{C}$ ;   
c) 恒温振荡培养箱: $36^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 、 $42^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ ;  
d) 天平：感量 $0.1\mathrm{g}$   
e) 均质器与配套均质袋；  
f) 振荡器；  
g) 无菌吸管： $1\mathrm{mL}$ （具 $0.01\mathrm{mL}$ 刻度）、 $10\mathrm{mL}$ （具 $0.1\mathrm{mL}$ 刻度）或微量移液器及吸头；  
h) 无菌锥形瓶: 容量 $100 \mathrm{~mL} 、 200 \mathrm{~mL} 、 2000 \mathrm{~mL}$ ;  
i) 无菌培养皿：直径 $90 \mathrm{~mm}$ ；  
j) pH计或pH比色管或精密pH试纸；  
k) 水浴装置： $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 、 $100^{\circ}\mathrm{C}$ ；  
1) 微需氧培养装置：提供微需氧条件（ $5\%$ 氧气、 $10\%$ 二氧化碳和 $85\%$ 氮气）；  
m) 过滤装置及滤膜（ $0.22\mu \mathrm{m}$ 、 $0.45\mu \mathrm{m}$ ）；  
n) 显微镜：10倍～100倍，有相差功能；  
o) 离心机：离心速度 $\geqslant 20000\mathrm{g}$   
p) 比浊仪；  
q) 微生物生化鉴定系统。

# 5 操作步骤

# 5.1 样品处理

# 5.1.1 一般样品

取 $25\mathrm{g}$ (mL) 样品（水果、蔬菜、水产品为 $50\mathrm{g}$ ）加入盛有 $225\mathrm{mL}$ Bolton肉汤的有滤网的均质袋中（若为无滤网均质袋可使用无菌纱布过滤），用拍击式均质器均质 $1\mathrm{min} \sim 2\mathrm{min}$ ，经滤网或无菌纱布过滤，将滤过液进行培养。

# 5.1.2 整禽等样品

用 $200\mathrm{mL}0.1\%$ 的蛋白胨水中充分冲洗样品的内外部，并振荡 $2\mathrm{min}\sim 3\mathrm{min}$ ，经无菌纱布过滤至 $250\mathrm{mL}$ 离心管中， $16000\mathrm{g}$ 离心 $15\mathrm{min}$ 后弃去上清，用 $10\mathrm{mL}0.1\%$ 蛋白胨水悬浮沉淀，吸取 $3\mathrm{mL}$ 于 $100\mathrm{mL}$ Bolton肉汤中进行培养。

# 5.1.3 贝类

取至少12个带壳样品，除去外壳后将所有内容物放到均质袋中，用拍击式均质器均质 $1\mathrm{min}\sim 2\mathrm{min}$ 取 $25\mathrm{g}$ 样品至 $225~\mathrm{mL}$ Bolton肉汤中(1:10稀释)，充分震荡后再转移 $25~\mathrm{mL}$ 于 $225~\mathrm{mL}$ Bolton肉汤中(1:100稀释)，将1:10和1:100稀释的Bolton肉汤同时进行培养。

# 5.1.4 蛋黄液或蛋浆

取 $25\mathrm{g}$ （mL）样品于 $125\mathrm{mL}$ Bolton肉汤中并混匀（1:6稀释)，再转移 $25\mathrm{mL}$ 于 $100\mathrm{mL}$ Bolton肉汤中并混匀（1:30稀释)，同时将1:6和1:30稀释的Bolton肉汤进行培养。

# 5.1.5 鲜乳、冰淇淋、奶酪等

若为液体乳制品取 $50\mathrm{g}$ ；若为固体乳制品取 $50\mathrm{g}$ 加入盛有 $50\mathrm{mL}0.1\%$ 蛋白胨水的有滤网均质袋中，用拍击式均质器均质 $15\mathrm{s}\sim 30\mathrm{s}$ ，保留过滤液。必要时调整 $\mathsf{pH}$ 值至 $7.5\pm 0.2$ ，将液体乳制品或滤过液以 $20000\mathrm{g}$ 离心 $30\mathrm{min}$ 后弃去上清，用 $10\mathrm{mL}$ Bolton肉汤悬浮沉淀（尽量避免带入油层)，再转移至 $90~\mathrm{mL}$ Bolton肉汤进行培养。

# 5.1.6 需表面涂拭检测的样品

无菌棉签擦拭检测样品的表面（面积至少 $100\mathrm{cm}^2$ 以上），将棉签头剪落到 $100\mathrm{mL}$ Bolton肉汤中进行培养。

# 5.1.7 水样

将 $4\mathrm{L}$ 的水（对于氯处理的水，在过滤前每升水中加入 $5\mathrm{mL}1\mathrm{mol} / \mathrm{L}$ 硫代硫酸钠溶液）经 $0.45\mu \mathrm{m}$ 滤膜过滤，把滤膜浸没在 $100~\mathrm{mL}$ Bolton肉汤中进行培养。

# 5.2 预增菌与增菌

在微需氧条件下， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $4\mathrm{h}$ ，如条件允许配以 $100\mathrm{r / min}$ 的速度进行振荡。必要时测定增菌液的 $\mathrm{pH}$ 值并调整至 $7.4 \pm 0.2$ ， $42^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 继续培养 $24\mathrm{h} \sim 48\mathrm{h}$ 。

# 5.3 分离

将 $24\mathrm{h}$ 增菌液、 $48\mathrm{h}$ 增菌液及对应的1:50稀释液分别划线接种于Skirrow血琼脂与mCCDA琼脂平板上，微需氧条件下 $42^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h} \sim 48\mathrm{h}$ 。另外可选择使用空肠弯曲菌显色平板作为补充。

观察 $24\mathrm{h}$ 培养与 $48\mathrm{h}$ 培养的琼脂平板上的菌落形态，mCCDA琼脂平板上的可疑菌落通常为淡灰色，有金属光泽、潮湿、扁平，呈扩散生长的倾向。Skirrow血琼脂平板上的第一型可疑菌落为灰色、扁平、湿润有光泽，呈沿接种线向外扩散的倾向；第二型可疑菌落常呈分散凸起的单个菌落，边缘整齐、发亮。空肠弯曲菌显色培养基上的可疑菌落按照说明进行判定。

# 5.4.1.2 形态观察

挑取可疑菌落进行革兰氏染色，镜检。

# 5.4.1.3 动力观察

挑取可疑菌落用 $1 \mathrm{~mL}$ 布氏肉汤悬浮，用相差显微镜观察运动状态。

# 5.4.1.4 氧化酶试验

用铂/铱接种环或玻璃棒挑取可疑菌落至氧化酶试剂润湿的滤纸上，如果在 $10\mathrm{~s}$ 内出现紫红色、紫罗兰或深蓝色为阳性。

# 5.4.1.5 微需氧条件下 $25^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 生长试验

挑取可疑菌落，接种到哥伦比亚血琼脂平板上，微需氧条件下 $25^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $44\mathrm{h} \pm 4\mathrm{h}$ ，观察细菌生长情况。

# 5.4.1.6 有氧条件下 $42^{\circ}C \pm 1^{\circ}C$ 生长试验

挑取可疑菌落，接种到哥伦比亚血琼脂平板上，有氧条件下 $42^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 培养 $44 \mathrm{~h} \pm 4 \mathrm{~h}$ ，观察细菌生长情况。

# 5.4.2 空肠弯曲菌的鉴定

# 5.4.2.1 过氧化氢酶试验

挑取菌落，加到干净玻片上的 $3\%$ 过氧化氢溶液中，如果在 $30 \mathrm{~s}$ 内出现气泡则判定结果为阳性。

# 5.4.2.2 马尿酸钠水解试验

挑取菌落，加到盛有 $0.4\mathrm{mL}1\%$ 马尿酸钠的试管中制成菌悬液。混合均匀后在 $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 水浴中温育 $2\mathrm{h}$ 或 $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 培养箱中温育 $4\mathrm{h}$ 。沿着试管壁缓缓加入 $0.2\mathrm{mL}$ 苄三酮溶液，不要振荡，在 $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 的水浴或培养箱中再温育 $10\mathrm{min}$ 后判读结果。若出现深紫色则为阳性；若出现淡紫色或没有颜色变化则为阴性。

# 5.4.2.3 呃哚乙酸酯水解试验

挑取菌落至吲哚乙酸酯纸片上，再滴加一滴灭菌水。如果吲哚乙酸酯水解，则在 $5\mathrm{min}\sim 10\mathrm{min}$ 内出现深蓝色；若无颜色变化则表示没有发生水解。空肠弯曲菌的鉴定结果见表2。

# A.3.3.2 制法

当基础培养基的温度为 $45\%$ 左右时，无菌加入绵羊血，混匀。校正 $\mathrm{pH}$ 至 $7.3 \pm 0.2$ （ $25\%$ ）。倾注 $15\mathrm{mL}$ 完全培养基于无菌平皿中，静置至培养基凝固。制备的平板未干燥时在室温放置不得超过 $4\mathrm{h}$ ，或在 $4\%$ 左右冷藏不得超过 $7\mathrm{d}$ 。

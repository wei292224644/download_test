# 前言

本标准代替GB/T21912—2008《食品中二氧化钛的测定》。

本标准与GB/T21912—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中二氧化钛的测定”。  
——修改了标准的适用范围。  
——增加了试样的制备内容。  
——删除了第二法中微波消解前处理方式。  
对附录A进行了修改。

# 食品安全国家标准

# 食品中二氧化钛的测定

# 1 范围

本标准规定了食品中二氧化钛的电感耦合等离子体-原子发射光谱和二安替比林甲烷比色测定方法。

本标准适用于面粉、果酱、凉果类、甘草制品、脱水马铃薯、油炸坚果与籽类、糖果（含胶基糖果）、可可制品、巧克力和巧克力制品(包括代可可脂巧克力及制品)以及糖果、糖果和巧克力制品包衣、装饰糖果、顶饰、甜汁、调味糖浆、蛋黄酱、沙拉酱、固体饮料类、果冻、膨化食品、饮料浑浊剂、魔芋凝胶制品中二氧化钛的测定。

# 第一法 电感耦合等离子体-原子发射光谱法(ICP-AES)

# 2 原理

试样经酸消解后，用电感耦合等离子体-原子发射光谱仪进行分析，采用标准曲线外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 高氯酸 $\mathrm{(HClO_4)}$ ：优级纯。  
3.1.2 硫酸 $\mathrm{(H_2SO_4)}$ ：优级纯。  
3.1.3 硝酸 $\left(\mathrm{HNO}_{3}\right)$ ：优级纯。  
3.1.4 硫酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{SO}_{4}\right]$ 。

# 3.2 试剂配制

3.2.1 硫酸溶液 $(5 + 95)$ ：量取 $50~\mathrm{mL}$ 硫酸，缓慢加入 $950~\mathrm{mL}$ 水中，混匀。  
3.2.2 混合酸[高氯酸 + 硝酸 $(1 + 9)]$ ：量取 $100\mathrm{mL}$ 高氯酸，缓慢加入 $900~\mathrm{mL}$ 硝酸中，混匀。

# 3.3 标准品

二氧化钛 $\mathrm{(TiO_2)}$ ：基准试剂或光谱纯。

# 3.4 标准溶液配制

3.4.1 钛标准储备液 $(1000\mu \mathrm{g} / \mathrm{mL})$ : 称取 $0.167\mathrm{g}$ 二氧化钛, 加 $5\mathrm{g}$ 硫酸铵, 加 $10\mathrm{mL}$ 硫酸, 加热溶解, 冷却, 移入 $100\mathrm{mL}$ 容量瓶中, 稀释至刻度, 混匀。或使用经国家认证并授予标准物质证书的标准溶液。  
3.4.2 钛标准使用液 $(10.0\mu \mathrm{g / mL})$ ：吸取 $1.00~\mathrm{mL}$ 钛标准储备液于 $100~\mathrm{mL}$ 容量瓶中，用硫酸溶液

$(5 + 95)$ 稀释至刻度。

3.4.3 钛标准系列工作液：吸取 $0.000 \mathrm{~mL}, 0.500 \mathrm{~mL}, 2.00 \mathrm{~mL}, 5.00 \mathrm{~mL}, 10.0 \mathrm{~mL}, 20.0 \mathrm{~mL}$ 钛标准使用液，分别置于 $100 \mathrm{~mL}$ 容量瓶中，用硫酸溶液（ $5 + 95$ ）稀释至刻度，配成浓度为 $0.000 \mu \mathrm{g} / \mathrm{mL}$ 、 $0.0500 \mu \mathrm{g} / \mathrm{mL}, 0.200 \mu \mathrm{g} / \mathrm{mL}, 0.500 \mu \mathrm{g} / \mathrm{mL}, 1.00 \mu \mathrm{g} / \mathrm{mL}, 2.00 \mu \mathrm{g} / \mathrm{mL}$ 的钛系列标准工作液。

# 4 仪器和设备

4.1 电感耦合等离子体-原子发射光谱仪。  
4.2 微波消解仪。  
4.3 分析天平：感量为 $0.1\mathrm{mg}$

# 5 试样制备

# 5.1 固体和半固体样品

取有代表性可食用部分，用组织捣碎机粉匀浆，混合均匀后装入洁净容器内密封并做好标识。

# 5.2 液体样品

取有代表性的样品混合均匀后，装入洁净容器内密封并做好标识。

注：制样和样品保存过程中，应防止样品受到污染和待测物损失。

# 5 分析步骤

# 5.1 试样处理

# 5.1.1 普通湿法消解

称取试样约 $5\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 于锥形瓶或高型烧杯中，放入数粒玻璃珠，加入 $15\mathrm{mL} \sim 20\mathrm{mL}$ 混合酸，盖上表面皿，在电炉上缓慢消解至溶液澄清，在消解过程中若出现碳化后的黑色，在盖着表面皿的情况下小心滴加硝酸，直至溶液澄清为止。继续加热至溶液剩余约 $2\mathrm{mL} \sim 3\mathrm{mL}$ ，冷却，加入 $1\mathrm{g}$ 硫酸铵和 $5\mathrm{mL}$ 硫酸，煮沸至澄清，继续煮至高氯酸白烟被赶尽，取下冷却，转移至 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，混匀，备用。

# 5.1.2 微波消解

称取试样 $0.2\mathrm{g}\sim 0.5\mathrm{g}$ （精确到 $0.0001\mathrm{g})$ 于微波消解罐中，加 $2.5\mathrm{mL}$ 硝酸和 $2.5\mathrm{mL}$ 硫酸，设置合适的微波消解条件进行消解，消解结束后，消解罐自然冷却至室温，将消解液转移至 $50~\mathrm{mL}$ 容量瓶中，用水少量多次洗涤消解罐，洗液合并于容量瓶中，用水定容至刻度，混匀。消解液应为澄清溶液，如消解后有沉淀无法消解，应重新用普通湿法消解进行处理。

注：温控式微波消解工作条件可参考附录A。

# 5.2 空白试验

除不加试样外，按5.1.1或5.1.2进行空白试验。

# 5.3 标准曲线的绘制

将电感耦合等离子体-原子发射光谱仪调至最佳条件，测定钛标准系列工作液(3.4.3)的发射光强

度。以钛浓度为横坐标，发射光强度为纵坐标绘制标准曲线。

注：电感耦合等离子体-原子发射光谱仪工作条件可参考附录A。

# 5.4 测定

在与测定标准溶液相同的实验条件下，测定试样溶液和空白溶液的发射光强度。由标准曲线(5.3)和试样溶液的发射光强度求得试样溶液和空白溶液中钛的浓度。若试样溶液中钛浓度过高，可适当稀释。

# 6 分析结果的表述

试样中二氧化钛的含量按式(1)计算：

$$
X = \frac {\left(c _ {1} - c _ {0}\right) \times V \times f \times 1 0 0 0}{m \times 1 0 0 0} \times 1. 6 6 8 1 \quad \dots \tag {1}
$$

式中：

$X$ ——试样中二氧化钛的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{1}$ ——由标准曲线得到的试样溶液中钛的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$c_{0}$ ——由标准曲线得到的空白溶液中钛的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

V ——试样溶液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样溶液的稀释倍数；

$m$ ——试样质量，单位为克(g)；

1.668 1—— $1\mathrm{g}$ 的钛相当于 $1.6681\mathrm{g}$ 二氧化钛。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

以称样量 $0.5\mathrm{g}$ ，定容至 $50~\mathrm{mL}$ 计算，方法检出限(LOD)为 $0.3\mathrm{mg / kg}$ ，定量限(LOQ)为 $1.0\mathrm{mg / kg}$ 。

# 第二法 二安替比林甲烷比色法

# 9 原理

试样经酸消解后，在强酸介质中钛与二安替比林甲烷形成黄色络合物，于紫外分光光度计 $420\mathrm{nm}$ 波长处测量其吸光度，采用标准曲线法定量。加入抗坏血酸消除三价铁的干扰。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 10.1 试剂

10.1.1 高氯酸 $\mathrm{(HClO_4)}$ ：优级纯。

10.1.2 硫酸 $\left(\mathrm{H}_{2} \mathrm{SO}_{4}\right)$ ：优级纯。  
10.1.3 硝酸 $\mathrm{(HNO_3)}$ ：优级纯。  
10.1.4 盐酸（HCl）：优级纯。  
10.1.5 硫酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{SO}_{4}\right]$ 。  
10.1.6 抗坏血酸 $\mathrm{(C_6H_8O_6)}$   
10.1.7 二安替比林甲烷 $\left(\mathrm{C}_{23} \mathrm{H}_{24} \mathrm{~N}_{4} \mathrm{O}_{2}\right)$ 。

# 10.2 溶液配制

10.2.1 混合酸[高氯酸 + 硝酸 $(1+9)]$ : 量取 $100 \mathrm{~mL}$ 高氯酸, 缓慢加入 $900 \mathrm{~mL}$ 硝酸中, 混匀。  
10.2.2 盐酸溶液 $(1 + 1)$ ：量取 $100~\mathrm{mL}$ 盐酸，缓慢加入 $100~\mathrm{mL}$ 水中，混匀。  
10.2.3 盐酸溶液 $(1 + 23)$ ：量取 $10\mathrm{mL}$ 盐酸，缓慢加入 $230~\mathrm{mL}$ 水中，混匀。  
10.2.4 硫酸溶液 $(2 + 98)$ ：量取 $20~\mathrm{mL}$ 硫酸，缓慢加入 $980~\mathrm{mL}$ 水中，混匀。  
10.2.5 抗坏血酸溶液 $(2\%)$ ：称取 $2\mathrm{g}$ 抗坏血酸，用水溶解并稀释至 $100~\mathrm{mL}$ ，现配现用。  
10.2.6 二安替比林甲烷溶液 $(5\%)$ ：称取 $5\mathrm{g}$ 二安替比林甲烷，用盐酸溶液 $(1 + 23)(10.2.3)$ 溶解并稀释至 $100~\mathrm{mL}$ 。

# 10.3 标准品

二氧化钛 $\mathrm{(TiO_2)}$ ：基准试剂或光谱纯。

# 10.4 标准溶液配制

10.4.1 钛标准储备液 $(1000\mu \mathrm{g} / \mathrm{mL})$ ：称取 $0.167\mathrm{g}$ 二氧化钛，加 $5\mathrm{g}$ 硫酸铵，加 $10~\mathrm{mL}$ 硫酸，加热溶解，冷却，移入 $100~\mathrm{mL}$ 容量瓶中，稀释至刻度，混匀。或使用经国家认证并授予标准物质证书的标准溶液。  
10.4.2 钛标准使用液 $(10.0\mu \mathrm{g / mL})$ ：吸取 $1.00~\mathrm{mL}$ 钛标储备液(10.4.1)于 $100~\mathrm{mL}$ 容量瓶中，用硫酸溶液 $(2 + 98)$ 稀释至刻度。

# 11 仪器和设备

11.1 紫外分光光度计。  
11.2 微波消解仪。  
11.3 分析天平：感量为 $1\mathrm{mg}$

# 12 试样制备

# 12.1 固体和半固体样品

取有代表性可食用部分，用组织捣碎机粉匀浆，混合均匀后装入洁净容器内密封并做好标识。

# 12.2 液体样品

取有代表性的样品混合均匀后，装入洁净容器内密封并做好标识。

注：制样和样品保存过程中，应防止样品受到污染和待测物损失。

# 13 分析步骤

# 13.1 试样处理

称取试样约 $5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于锥形瓶或高型烧杯中，放入数粒玻璃珠，加入 $15\mathrm{mL} \sim 20\mathrm{mL}$ 混合酸，盖上表面皿，在电炉上缓慢消解至溶液澄清，继续加热至溶液剩余约 $2\mathrm{mL} \sim 3\mathrm{mL}$ ，冷却，加入 $1\mathrm{g}$ 硫酸铵和 $5\mathrm{mL}$ 硫酸，煮沸至澄清。在消解过程中若出现碳化后的黑色，在盖着表面皿的情况下小心滴加浓硝酸，直至溶液澄清为止。继续煮至高氯酸白烟被赶尽，取下冷却，转移至 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，混匀，备用。

# 13.2 空白试验

除不加试样外，按13.1进行空白试验。

# 13.3 显色

移取适量定容后的溶液于 $50~\mathrm{mL}$ 容量瓶中，加入 $5\mathrm{mL}$ 抗坏血酸溶液(10.2.5)，摇匀，再依次加入 $14\mathrm{mL}$ 盐酸溶液 $(1 + 1)(10.2.2),6\mathrm{mL}$ 二安替比林甲烷溶液（10.2.6），用水稀释至刻度，摇匀，放置 $40\mathrm{min}$ ，待测。

注：溶液移取体积根据试样中钛元素的含量而定。

# 13.4 标准系列工作液的配制

吸取 $0.000\mathrm{mL}, 0.500\mathrm{mL}, 1.00\mathrm{mL}, 2.50\mathrm{mL}, 5.00\mathrm{mL}, 10.0\mathrm{mL}$ 钛标准使用液(10.14)，分别置于 $50~\mathrm{mL}$ 容量瓶，加入 $5\mathrm{mL}$ 抗坏血酸溶液(10.2.5)，摇匀，再依次加入 $14\mathrm{mL}$ 盐酸溶液 $(1 + 1)(10.2.2)$ 、 $6\mathrm{mL}$ 二安替比林甲烷溶液(10.2.6)，用水稀释至刻度，摇匀，放置 $40\mathrm{min}$ ，此标准系列工作液中钛的浓度依次为 $0.000\mu \mathrm{g} / \mathrm{mL}, 0.100\mu \mathrm{g} / \mathrm{mL}, 0.200\mu \mathrm{g} / \mathrm{mL}, 0.500\mu \mathrm{g} / \mathrm{mL}, 1.00\mu \mathrm{g} / \mathrm{mL}, 2.00\mu \mathrm{g} / \mathrm{mL}$ ，待测。

# 13.5 标准曲线的绘制

以显色后的标准空白溶液为参比，用 $1\mathrm{cm}$ 比色皿，于 $420~\mathrm{nm}$ 波长处，用紫外分光光度计测定显色后的标准系列工作液的吸光度。以标准系列工作液的浓度为横坐标，相应的吸光度为纵坐标，绘制标准曲线。

# 13.6 测定

在与测定标准溶液相同的实验条件下，测定显色后的试样溶液和空白溶液的吸光度。由标准曲线(13.5)和空白溶液、试样溶液的吸光度求得试样溶液和空白溶液中钛的浓度。

# 14 分析结果的表述

试样中二氧化钛的含量按式(2)计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V _ {1} \times 5 0 \times 1 0 0 0}{m \times V _ {2} \times 1 0 0 0} \times 1. 6 6 8 1 \dots \tag {2}
$$

式中：

$X$ ——试样中二氧化钛的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

c 由标准曲线得到的显色后试样溶液中钛的浓度，单位为微克每毫升 $(\mu \mathrm{g / mL})$

$c_{0}$ ——由标准曲线得到的显色后空白溶液中钛的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V_{1}$ ——试样消解后初次定容的体积，单位为毫升 $(\mathrm{mL})$ ；

50 ——显色后试样溶液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克(g)；

$V_{2}$ ——显色时移取试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1.668 1—— $1\mathrm{g}$ 的钛相当于 $1.6681\mathrm{g}$ 二氧化钛。

计算结果保留两位有效数字。

# 15 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 16 检出限及定量限

以称样量 $0.5\mathrm{g}$ ，定容至 $50~\mathrm{mL}$ 计算，方法检出限(LOD)为 $1.5\mathrm{mg / kg}$ ，定量限(LOQ)为 $5.0\mathrm{mg / kg}$ 。

# 1 范围

本文件规定了动物性食品中10种利尿药残留量的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、牛、羊、鸡、鸭的肌肉、鸡蛋和牛奶中乙酰唑胺、4-氨基-6-氯苯-1,3-二磺酰胺、氯噻嗪、氢氯噻嗪、氯噻酮、呋塞米、卡氟噻嗪、氨苯蝶啶、螺内酯、坎利酮10种利尿药残留量的测定。

# 4 原理

试样经乙腈-水溶液提取，正己烷除脂，反相混合型亲水亲脂固相萃取柱净化，液相色谱-串联质谱仪测定，内标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为色谱纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $(\mathrm{CH}_3\mathrm{CN})$ 。  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ 。  
5.1.3 正己烷 $(\mathrm{C_6H_{14}})$   
5.1.4 二甲基亚砜 $(\mathrm{C}_2\mathrm{H}_6\mathrm{OS})$ 。  
5.1.5 乙酸铵 $\mathrm{(CH_3COONH_4)}$

# 5.2 溶液配制

5.2.1 $90\%$ 乙腈水溶液：取乙腈 $900~\mathrm{mL}$ ，加水稀释至 $1000~\mathrm{mL}$   
5.2.2 $50\%$ 乙腈水溶液：取乙腈 $50~\mathrm{mL}$ ，加水稀释至 $100~\mathrm{mL}$ 。  
5.2.3 $5\mathrm{mmol / L}$ 乙酸铵缓冲溶液：取乙酸铵 $0.385\mathrm{g}$ ，加水溶解，稀释至 $1000~\mathrm{mL}$ 。  
5.2.4 $90\%$ 乙腈甲醇溶液：取甲醇 $10~\mathrm{mL}$ ，用乙腈稀释至 $100~\mathrm{mL}$ ，混匀。  
5.2.5 乙腈饱和正己烷溶液：取相同体积的乙腈和正己烷，置于分液漏斗中，振荡，静置分层，取上层溶液。

# 5.3 标准品

5.3.1 利尿药标准品：乙酰唑胺、4-氨基-6-氯苯-1,3-二磺酰胺、氯噻嗪、氢氯噻嗪、氯噻酮、呋塞米、苄氟噻嗪、氨苯蝶啶、螺内酯、坎利酮，纯度均 $\geq 98.0\%$ 。或经国家认证并授予标准物质证书的标准品。标准品

信息见附录A中的表A.1。

5.3.2同位素内标标准品：乙酰唑胺- $\mathrm{D}_3$ 、4-氨基-6-氯苯-1，3-二磺酰胺- ${}^{15}\mathrm{N}_2$ 、氯噻酮- $\mathrm{D}_4$ 、氢氯噻嗪- ${}^{13}\mathrm{CD}_2$ 、氯噻嗪- ${}^{13}\mathrm{C}^{15}\mathrm{N}_2$ 、呋塞米- $\mathrm{D}_5$ 、卞氟噻嗪- $\mathrm{D}_5$ 、氨苯蝶啶- $\mathrm{D}_5$ 、坎利酮- $\mathrm{D}_6$ ，纯度均 $\geqslant 98.0\%$ 。或经国家认证并授予标准物质证书的标准品。同位素内标信息见附录A中的表A.1。

# 5.4 标准溶液制备

5.4.1 标准储备液 $(0.1\mathrm{mg / mL})$ ：分别称取利尿药标准品各约 $10\mathrm{mg}$ ，精密称定，用甲醇溶解并定容至 $100~\mathrm{mL}$ 棕色容量瓶中，混匀，配制成浓度为 $0.1\mathrm{mg / mL}$ 的利尿药标准储备液。氨苯蝶啶标准品需先用 $1\mathrm{mL}$ 二甲基亚砜溶解，再用甲醇稀释并定容至刻度。 $-20^{\circ}C$ 保存，有效期6个月。

5.4.2 内标储备液 $(0.1\mathrm{mg / mL})$ ：分别称取利尿药内标标准品各约 $10\mathrm{mg}$ ，精密称定，用甲醇溶解并定容至 $100~\mathrm{mL}$ 棕色容量瓶中，混匀，配制成浓度为 $0.1\mathrm{mg / mL}$ 的利尿药内标标准储备液。氨苯蝶啶- $\mathrm{D}_5$ 标准品需先用 $1\mathrm{mL}$ 二甲基亚砜溶解，再用甲醇稀释并定容至刻度。 $-20^{\circ}C$ 保存，有效期6个月。

5.4.3 混合标准工作液：精密量取氯噻嗪、氢氯噻嗪、苄氟噻嗪和坎利酮的标准储备液 $(0.1\mathrm{mg / mL})$ 各 $1\mathrm{mL}$ ，氯噻酮、乙酰唑胺、呋塞米、螺内酯、4-氨基-6-氯苯-1，3-二磺酰胺、氨苯蝶啶的标准储备液 $(0.1\mathrm{mg / mL})$ 各 $2.5\mathrm{mL}$ ，于同一 $100~\mathrm{mL}$ 棕色容量瓶中，用甲醇定容至刻度，配制成的混合标准工作液中氯噻嗪、氢氯噻嗪、苄氟噻嗪和坎利酮的浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ ；氯噻酮、乙酰唑胺、呋塞米、螺内酯和4-氨基-6-氯苯-1，3-二磺酰胺、氨苯蝶啶的浓度为 $2.5\mu \mathrm{g} / \mathrm{mL}$ 。 $4^{\circ}C$ 以下避光保存，有效期1个月。

5.4.4 内标工作液：精密量取内标储备液 $(0.1\mathrm{mg / mL})$ 各 $1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 棕色容量瓶中，用甲醇定容至刻度，配制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 的内标标准工作液。 $4^{\circ}\mathrm{C}$ 以下避光保存，有效期1个月。

# 5.4.5 混合标准系列工作液

精确吸取利尿药混合标准工作液 $10.0\mu \mathrm{L},20.0\mu \mathrm{L},50.0\mu \mathrm{L},100\mu \mathrm{L},200\mu \mathrm{L},500\mu \mathrm{L}$ 以及内标工作液 $200~\mu \mathrm{L}$ ，分别置于 $10~\mathrm{mL}$ 棕色容量瓶中，用 $50\%$ 乙腈水溶液定容至刻度。其中氯噻嗪、氢氯噻嗪、苄氟噻嗪和坎利酮的浓度依次为 $1.00\mathrm{ng / mL},2.00\mathrm{ng / mL},5.00\mathrm{ng / mL},10.0\mathrm{ng / mL},20.0\mathrm{ng / mL},$ $50.0\mathrm{ng / mL}$ ；氯噻酮、乙酰唑胺、呋塞米、螺内酯、4-氨基-6-氯苯-1，3-二磺酰胺、氨苯蝶啶的浓度依次为 $2.50\mathrm{ng / mL},5.00\mathrm{ng / mL},12.5\mathrm{ng / mL},25.0\mathrm{ng / mL},50.0\mathrm{ng / mL},125\mathrm{ng / mL}$ ，标准系列工作液中内标溶液的浓度为 $20.0\mathrm{ng / mL}$ 。现用现配。

# 5.5 材料

5.5.1固相萃取柱：通过式反相混合型亲水亲脂平衡共聚物固相萃取柱， $200\mathrm{mg / 6mL}$ ，或性能相当者。  
5.5.2 有机滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量分别为 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 氮吹浓缩仪。  
6.4 涡旋混合器。  
6.5 超声波发生器。  
6.6 离心机：转速不低于 $8000\mathrm{r / min}$   
6.7 固相萃取装置。  
6.8 均质机。

# 7 试样的制备与保存

# 7.1 试样制备

取适量新鲜或解冻的空白或供试肌肉组织，绞碎，并均质。

取适量新鲜或冷藏的空白或供试鸡蛋，去壳后混合均匀。

取适量新鲜或冷藏的空白或供试牛奶，混合均匀。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样保存

猪肉、牛肉、羊肉、鸡肉、鸭肉和鸡蛋试样于 $-18^{\circ} \mathrm{C}$ 以下保存，牛奶试样于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 保存。

# 8 分析步骤

# 8.1 提取

# 8.1.1 猪肉、牛肉、羊肉、鸡肉、鸭肉

称取试料 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 离心管中，加入 $40.0\mu \mathrm{L}$ 内标工作液，加入 $90\%$ 乙腈水溶液 $8.0~\mathrm{mL}$ ，于涡旋振荡器上剧烈振荡 $2\mathrm{min}$ 。超声提取 $15\mathrm{min} / 8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液至离心管中。残渣加 $90\%$ 乙腈水溶液 $8.0~\mathrm{mL}$ 重复提取1次，合并上清液。加入 $5\mathrm{mL}$ 乙腈饱和正己烷溶液，低速涡旋 $10\mathrm{s},8000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，取下层清液，用 $90\%$ 乙腈水溶液稀释至 $20.0~\mathrm{mL}$ 备用。

# 8.1.2 鸡蛋、牛奶

称取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管中，加入 $40.0\mu \mathrm{L}$ 内标工作液，加入乙腈 $8.0~\mathrm{mL}$ ，于涡旋振荡器上剧烈振荡 $2\mathrm{min}$ 。超声提取 $15\mathrm{min},8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液至离心管中。残渣加乙腈 $8.0~\mathrm{mL}$ 重复提取1次，合并上清液。加入 $5\mathrm{mL}$ 乙腈饱和正己烷溶液，低速涡旋 $10\mathrm{s},8000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，取下层清液，用乙腈稀释至 $20.0~\mathrm{mL}$ 备用。

# 8.2 净化

取备用液 $10.0 \mathrm{~mL}$ 过固相萃取柱, 以不大于 $1 \mathrm{~mL} / \mathrm{min}$ 的速率通过固相萃取柱, 接收流出的样液, 并用 $1 \mathrm{~mL} 90 \%$ 乙腈甲醇溶液洗脱, 接收全部洗脱液, 于 $40^{\circ} \mathrm{C}$ 下氮气吹干。用 $1.0 \mathrm{~mL} 50 \%$ 乙腈水溶液溶解残余物, 涡旋混匀, 过 $0.22 \mu \mathrm{m}$ 滤膜, 供液相色谱串联质谱仪测定。

# 9 结果计算和表述

试样中待测药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times A _ {i s} ^ {\prime} \times C _ {s} \times C _ {i s} \times V _ {1} \times V _ {2} \times 1 0 0 0}{A _ {i s} \times A _ {s} \times C _ {i s} ^ {\prime} \times m \times V _ {3} \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中被测物质残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{s}}$ ——标准工作溶液中被测物质浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$C_{is}$ ——试样溶液中内标浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$C_{is}^{\prime}$ ——标准工作溶液中内标浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

A ——试样溶液中被测物质的峰面积；

$A_{is}^{\prime}$ ——标准工作溶液中内标的峰面积；

$A_{is}$ ——试样溶液中内标的峰面积；

$A_{\mathrm{s}}$ ——标准工作溶液中被测物质的峰面积；

$V_{1}$ ——试样提取液总体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——净化后残余物溶解体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——试样提取液过柱净化体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量的数值，单位为克(g)；

1000 ——换算系数。

# 10 方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法中氯噻嗪、氢氯噻嗪、苄氟噻嗪和坎利酮的检出限为 $1.00\mu \mathrm{g / kg}$ ；定量限为 $2.00\mu \mathrm{g / kg}$ 。乙酰唑胺、4-氨基-6-氯苯-1,3-二磺酰胺、氯噻酮、呋塞米、螺内酯、氨苯蝶啶的检出限为 $2.50\mu \mathrm{g / kg}$ ；定量限为 $5.00\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法中氯噻嗪、氢氯噻嗪、卡氟噻嗪和坎利酮在 $2\mu \mathrm{g} / \mathrm{kg}\sim 20~\mu \mathrm{g} / \mathrm{kg}$ 的添加浓度水平时，回收率为 $60\% \sim 120\%$

乙酰唑胺、4-氨基-6-氯苯-1,3-二磺酰胺、氯噻酮、呋塞米、螺内酯、氨苯蝶啶在 $5\mu \mathrm{g} / \mathrm{kg}\sim 50~\mu \mathrm{g} / \mathrm{kg}$ 的添加浓度水平时，回收率为 $60\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

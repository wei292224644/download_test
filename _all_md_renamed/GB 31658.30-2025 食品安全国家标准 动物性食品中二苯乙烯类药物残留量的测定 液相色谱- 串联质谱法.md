# 1 范围

本文件规定了猪、牛、羊、鸡等动物组织（肌肉、肝脏、肾脏和脂肪）和禽蛋、牛奶中己烯雌酚、己烷雌酚和己二烯雌酚残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、牛、羊、鸡组织(肌肉、肝脏、肾脏和脂肪)和禽蛋、牛奶中己烯雌酚、己烷雌酚和己二烯雌酚残留量的测定。

# 4 原理

试样中残留的药物经酶解后用乙腈提取(脂肪样品先经乙腈提取,吹干复溶后再酶解),加入正己烷和乙酸乙酯后进行液-液-液三相体系净化,取中间层氨吹复溶后通过碳酸钠溶液液-液萃取和硅胶柱固相萃取进行净化,液相色谱、串联质谱仪测定、基质匹配内标法定量。

# 5 试剂与材料

以下所用的试剂，除特别注明外均为分析纯试剂，水为符合GB16382规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 氢氧化钠(NaOH)。  
5.1.3 正己烷 $(\mathrm{C_6H_{14}})$ 。  
5.1.4 乙酸乙酯 $\mathrm{(CH_3COOC_2H_7)}$   
5.1.5 乙酸铵 $\mathrm{(CH_3COONH_4)}$   
5.1.6 乙酸 $\mathrm{(CH_3COOH)}$   
5.1.7 碳酸钠 $\mathrm{(Na_2CO_3)}$ 。  
5.1.8 $\beta$ -葡萄糖醛苷酸酶/芳基硫酸酯酶（ $\beta$ -glucuronidase/aryl sulfatase）： $\beta$ -葡萄糖醛苷酸酶 $4.5\mathrm{U/mL}$ ，芳基硫酸酯酶 $14\mathrm{U/mL}$ ，或相当者。

# 5.2 溶液的配制

5.2.1 乙酸铵溶液 $(0.02\mathrm{mol} / \mathrm{L})$ ：称取乙酸铵 $1.54\mathrm{g}$ ，加水约 $800~\mathrm{mL}$ 溶解，用乙酸调节 $\mathrm{pH}$ 至5.2，加水至 $1000~\mathrm{mL}$ ，混匀。  
5.2.2 $10\%$ 碳酸钠溶液：取碳酸钠 $50~\mathrm{g}$ ，加水 $450~\mathrm{mL}$ 使其完全溶解。  
5.2.3 $40\%$ 乙腈溶液：取乙腈 $200~\mathrm{mL}$ ，加水至 $500~\mathrm{mL}$ ，混匀。  
5.2.4 乙酸乙酯+正己烷(15:85)：取 $75~\mathrm{mL}$ 乙酸乙酯、正己烷 $425~\mathrm{mL}$ ，混匀。

5.2.5 乙酸乙酯+正己烷(30：70)：取 $150~\mathrm{mL}$ 乙酸乙酯、正己烷 $350~\mathrm{mL}$ ，混匀。

# 5.3 标准品

己烯雌酚、己烷雌酚、己二烯雌酚、己烯雌酚- $\mathrm{D}_8$ 、己烷雌酚- $\mathrm{D}_4$ 和己二烯雌酚- $\mathrm{D}_6$ ，含量均 $\geqslant 95\%$ ，见附录A。

# 5.4 标准溶液的制备

5.4.1 标准储备液：取二苯乙烯类药物各标准品适量（相当于有效成分 $10\mathrm{mg}$ ），精密称定，用乙腈溶解并稀释定容于 $100~\mathrm{mL}$ 棕色容量瓶，配制成浓度均为 $100~\mu \mathrm{g / mL}$ 的标准储备液。 $-18^{\circ}C$ 以下保存，有效期12个月。  
5.4.2 同位素内标储备液：取二苯乙烯类药物各同位素内标标准品适量（相当于有效成分 $1\mathrm{mg}$ ），精密称定，用乙腈溶解并稀释定容于 $10~\mathrm{mL}$ 棕色容量瓶，配制成浓度均为 $100~\mu \mathrm{g / mL}$ 的同位素内标储备液。-18℃以下保存，有效期12个月。  
5.4.3 混合标准储备液：分别准确移取己烯雌酚、己烷雌酚和己二烯雌酚标准储备液各 $0.5\mathrm{mL}$ 于 $10\mathrm{mL}$ 棕色容量瓶，用乙腈稀释至刻度，配制成浓度为 $5.0~\mu \mathrm{g / mL}$ 的混合标准储备液。一 $18^{\circ}C$ 以下保存，有效期12个月。  
5.4.4 混合内标储备液：分别准确移取己烯雌酚- $\mathrm{D}_{8}$ 、己烷雌酚- $\mathrm{D}_{4}$ 和己二烯雌酚- $\mathrm{D}_{6}$ 标准储备液各 $0.5\mathrm{mL}$ 于 $10\mathrm{mL}$ 棕色容量瓶，用乙腈稀释至刻度，配制成浓度为 $5.0~\mu \mathrm{g / mL}$ 的混合内标储备液。 $-18^{\circ}C$ 以下保存，有效期12个月。  
5.4.5 混合内标工作液：准确移取混合内标储备液 $0.2\mathrm{mL}$ 于 $10\mathrm{mL}$ 棕色容量瓶中，用乙腈稀释至刻度。该溶液中各内标浓度为 $100~\mu \mathrm{g / L}$ 。现用现配。  
5.4.6 混合标准工作液：准确移取混合标准储备液、混合内标工作液适量，用 $40\%$ 乙腈溶液稀释成浓度为 $1.0 \mu \mathrm{g} / \mathrm{L}, 2.0 \mu \mathrm{g} / \mathrm{L}, 5.0 \mu \mathrm{g} / \mathrm{L}, 20.0 \mu \mathrm{g} / \mathrm{L}, 50.0 \mu \mathrm{g} / \mathrm{L}$ 和 $200.0 \mu \mathrm{g} / \mathrm{L}$ 的系列标准工作溶液，每一标准工作溶液含各内标浓度均为 $10 \mu \mathrm{g} / \mathrm{L}$ 。现用现配。

# 5.5 材料

5.5.1 硅胶固相萃取小柱： $500\mathrm{mg} / 3\mathrm{mL}$ ，或相当者。  
5.5.2 针头式过滤器(通用型滤膜)：尼龙材质，孔径 $0.22\mu \mathrm{m}$ ，或性能相当者。

# 6 仪器和设备

6.1 液相色谱串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
6.3 氮吹仪。  
6.4 恒温振荡器。  
6.5 固相萃取装置。  
6.6 离心管：聚丙烯塑料离心管， $50~\mathrm{mL}$ 。  
6.7 离心机：9000 r/min 或以上。  
6.8 pH计。  
6.9 涡旋混合器。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并均质。

取新鲜或解冻的空白或供试牛奶，混合均匀。

取适量新鲜或冷藏的空白或供试禽蛋，去壳后混合均匀。

a）取均质后的供试样品，作为供试试样；

b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准溶液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

# 8.1.1 肌肉、肝脏、肾脏、鸡蛋、牛奶

称取试料 $(5\pm 0.05)\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管，准确加入混合内标工作液 $100~\mu \mathrm{L}$ ，混合 $15\mathrm{s}$ ，加乙酸铵溶液 $10~\mathrm{mL},$ $\beta$ 葡萄糖醛酸酶/芳基硫酸酯酶 $50~\mu \mathrm{L}$ ，涡旋 $1\mathrm{min}$ ， $(37\pm 1)^{\circ}C$ 振荡酶解 $12\mathrm{h}$ 。酶解后以 $9000\mathrm{r / min}$ 离心 $2\mathrm{min}$ ，收集提取液，然后在残渣中加入乙腈 $10~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，6000r/min离心 $2\mathrm{min}$ ，取乙腈层，残渣再加入乙腈 $10~\mathrm{mL}$ 重复提取，合并提取液。

# 8.1.2 脂肪

称取试料 $(5\pm 0.05)\mathrm{g}$ ，准确加入混合内标工作液 $100~\mu \mathrm{L}$ ，混合 $15\mathrm{s}$ ，加入正己烷 $15~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ 再加入乙腈 $10~\mathrm{mL}$ 后涡旋 $1\mathrm{min},6000\mathrm{r / min}$ 离心 $2\mathrm{min}$ ，取乙晴层，样品中再加入乙腈 $10~\mathrm{mL}$ ，重复萃取，合并乙腈层， $50^{\circ}C$ 氮气吹干，加入乙酸铵溶液 $10~\mathrm{mL}$ ，甘菊糖醋酸酶/芳基硫酸酯酶 $50~\mu \mathrm{L}$ ，涡旋 $1\mathrm{min}$ ， $(37\pm 1)^{\circ}C$ 振荡酶解 $12\mathrm{h}$ ，后续操作步骤同8.1.1中酶后处理步骤。

# 8.2 净化

提取液中加入正己烷 $8\mathrm{mL}$ 和乙酸乙酯 $2\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ， $6000\mathrm{r}\cdot \mathrm{min}$ 离心 $2\mathrm{min}$ ，取中间乙腈层， $50^{\circ}\mathrm{C}$ 氮吹至干，用乙酸乙酯 $10\mathrm{mL}$ 溶解残渣，转移至 $50\mathrm{mL}$ 离心管，加入 $10\%$ 碳酸钠溶液 $2\mathrm{mL}$ ，涡旋混合 $1\mathrm{min}$ ，以 $6000\mathrm{r}\cdot \mathrm{min}$ 离心 $2\mathrm{min}$ ，弃去下层水相，再加入 $10\%$ 碳酸钠溶液 $2\mathrm{mL}$ ，重复上述步骤后将乙酸乙酯转移至 $10\mathrm{mL}$ 试管， $50^{\circ}\mathrm{C}$ 氮气吹干。

用乙酸乙酯+正己烷(15：85)溶液2mL溶解残渣，转移至已依次用正己烷5mL、乙酸乙酯5mL活化并抽干处理后的硅胶柱上，待溶液过柱后再用正己烷5mL淋洗，弃去正己烷淋洗液，抽干柱子，用乙酸乙酯+正己烷（30：70）溶液5mL洗脱，收集洗脱液， $50^{\circ}C$ 氢气吹干，用 $10\%$ 乙腈溶液1.0mL溶解残渣，涡旋混匀， $0.22\mu \mathrm{m}$ 滤膜过滤，供液相色谱-串联质谱测定。

# 8.3 基质匹配标准曲线的制备

取空白试料依次按8.1和8.2处理， $50^{\circ}\mathrm{C}$ 水浴氮气吹干，加入混合标准工作液 $1.0\mathrm{mL}$ 溶解残渣，过 $0.22\mu \mathrm{m}$ 滤膜，得内标浓度均为 $10~\mu \mathrm{g / L}$ 的 $1.0\mu \mathrm{g / L}, 2.0\mu \mathrm{g / L}, 3.0\mu \mathrm{g / L}, 20.0\mu \mathrm{g / L}, 50.0\mu \mathrm{g / L}$ 和 $200.0\mu \mathrm{g / L}$ 的系列基质匹配标准工作溶液，供液相色谱-串联质谱测定。以定量离子对峰面积与内标物离子对峰面积的比值为纵坐标、标准溶液浓度为横坐标，绘制基质匹配标准曲线。求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在同样测试条件下，试料溶液中待测物质的保留时间与相应内标物保留时间的比值与标准溶液中待测物质的保留时间与相应内标物保留时间的比值相对偏差在 $1.0\%$ 以内；且检测到的离子的相对丰度应当与浓度相当的基质匹配标准工作溶液相对丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.4.3.2 定量测定

取试料溶液和基质匹配标准工作溶液，做单点或多点校准，内标法计算。基质匹配标准工作溶液及试样溶液中目标药物的特征离子质量色谱峰峰面积均应在仪器检测的线性范围内。在上述色谱-质谱条件下，羊肉添加样品液相色谱-串联质谱多反应监测(MRM)色谱图见附录B。

# 8.5 空白试验

取空白试样，除不加药物外，采用完全相同的操作步骤进行平行操作。

# 9 结果计算和表述

试样中待测药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A _ {i} \times A ^ {\prime} {} _ {i s} \times C _ {s} \times C _ {i s} \times V}{A _ {i s} \times A _ {s} \times C ^ {\prime} {} _ {i s} \times m} \tag {1}
$$

式中：

$X$ ——试样中二苯乙烯类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A_{i}$ ——试料溶液中二苯乙烯类药物的峰面积；  
$A^{\prime}_{i\mathrm{s}}$ ——基质匹配标准溶液中二苯乙烯类药物内标的峰面积；  
$C_{s}$ ——基质匹配标准溶液中二苯乙烯类药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；  
$C_{is}$ ——试料溶液中二苯乙烯类药物内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；  
$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{is}$ ——试料溶液中二苯乙烯类药物内标的峰面积；

$A_{s}$ ——基质匹配标准溶液中二苯乙烯类药物的峰面积；

$C_{is}^{\prime}$ ——基质匹配标准溶液中二苯乙烯类药物内标浓度的数值，单位为微克每升 $(\mu \mathrm{g / L})$

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$ 。

己烯雌酚结果计算为顺反式2个异构体之和。

# 10 检测方法的灵敏度、正确度和精密度

# 10.1 灵敏度

本方法在猪、牛、羊、鸡组织(肌肉、肝脏、肾脏和脂肪)和禽蛋、牛奶中的检出限为 $0.50 \mu \mathrm{g} / \mathrm{kg}$ , 定量限为 $1.0 \mu \mathrm{g} / \mathrm{kg}$ 。

# 10.2 正确度

本方法在 $1\mu \mathrm{g} / \mathrm{kg}\sim 50~\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $80\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

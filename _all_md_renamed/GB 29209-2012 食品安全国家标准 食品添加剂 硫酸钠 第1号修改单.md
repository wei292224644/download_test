# 食品安全国家标准

# 食品添加剂 硫酸钠

# 1 范围

本标准适用于以天然芒硝等为原料生产的食品添加剂硫酸钠。

# 2 化学名称、分子式和相对分子质量

# 2.1 化学名称

无水硫酸钠

十水合硫酸钠

# 2.2 分子式

无水硫酸钠： $\mathrm{Na_2SO_4}$

十水合硫酸钠： $\mathrm{Na_2SO_4\cdot 10H_2O}$

# 2.3 相对分子质量

无水硫酸钠：142.04（按2007年国际相对原子质量）

十水合硫酸钠：322.20（按2007年国际相对原子质量）

# 附录A

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008中规定的三级水。试验中所用杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T602和GB/T603的规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸。

A.3.1.2 碳酸钾溶液： $150~\mathrm{g / L}$

A.3.1.3 焦锑酸钾溶液：称取 $2\mathrm{g}$ 焦锑酸钾溶于 $95~\mathrm{mL}$ 热水中，迅速冷却，加 $50~\mathrm{mL}$ 氢氧化钾溶液 $(50~\mathrm{g / L})$ 和 $1\mathrm{mL}$ 氢氧化钠溶液 $(85~\mathrm{g / L})$ ，放置 $24\mathrm{h}$ 后，过滤，加水至 $150~\mathrm{mL}$ 。

A.3.1.4 氯化钡 $(\mathrm{BaCl}_2\cdot 2\mathrm{H}_2\mathrm{O})$ 溶液： $122~\mathrm{g / L}$

A.3.1.5 乙酸铅溶液： $95\mathrm{g / L}$

A.3.1.6 铂丝。

# A.3.2 鉴别方法

# A.3.2.1 钠离子鉴别

A.3.2.1.1 称取约 $0.1 \mathrm{~g}$ 试样溶于 $2 \mathrm{~mL}$ 水中, 加入 $2 \mathrm{~mL}$ 碳酸钾溶液, 加热至沸, 应无沉淀生成。加入 $4 \mathrm{~mL}$ 焦锑酸钾溶液, 加热至沸。置于冰水中冷却, 必要时用玻璃棒摩擦试管内壁, 应有白色沉淀生成。

A.3.2.1.2 取铂丝，用盐酸溶液润湿后在无色火焰中燃烧至无色，再蘸取少许试样溶液在无色火焰上燃烧，火焰应呈亮黄色。

# A.3.2.2 硫酸根离子鉴别

A.3.2.2.1 取约 $0.5 \mathrm{~g}$ 试样, 溶于 $10 \mathrm{~mL}$ 水, 加入氯化钡溶液, 即产生白色沉淀, 此白色沉淀不溶于盐酸溶液。  
A.3.2.2.2 取约 $0.5\mathrm{g}$ 试样，溶于 $10~\mathrm{mL}$ 水，加入乙酸铅溶液，即产生白色沉淀，此白色沉淀溶于乙酸铵溶液。  
A.3.2.2.3 取约 $0.5\mathrm{g}$ 试样，溶于 $10~\mathrm{mL}$ 水，加入盐酸不产生沉淀。

# A.4 硫酸钠 $\left(\mathrm{Na}_{2}\mathrm{SO}_{4}\right)$ 含量的测定

# A.4.1 方法提要

用水溶解试样，在酸性条件下加入氯化钡，与试液中的硫酸根离子生成硫酸钡沉淀，经过滤、洗涤、

干燥、灰化、灼烧后称量计算含量。

# A.4.2 试剂和材料

A.4.2.1 盐酸溶液： $1 + 1$   
A.4.2.2 氯化钡 $(\mathrm{BaCl}_2\cdot 2\mathrm{H}_2\mathrm{O})$ 溶液： $122~\mathrm{g / L}$   
A.4.2.3 硝酸银溶液： $20\mathrm{g / L}$   
A.4.2.4 慢速定量滤纸。

# A.4.3 仪器和设备

A.4.3.1 电热恒温干燥箱： $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。  
A.4.3.2 高温炉：能控制温度在 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 。

# A.4.4 分析步骤

称取约 $5\mathrm{g}$ 于 $105~^\circ \mathrm{C}\pm 2$ ℃下干燥 $4\mathrm{h}$ 后的试样，精确至 $0.0002\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 烧杯中，加热溶解。过滤到 $500~\mathrm{mL}$ 容量瓶中，用水洗涤至无硫酸根离子为止（用氯化钡溶液检验）。冷却，用水稀释至刻度，摇匀。用移液管移取 $25~\mathrm{mL}$ 试样溶液置于 $500~\mathrm{mL}$ 烧杯中，加入 $5\mathrm{mL}$ 盐酸溶液， $270~\mathrm{mL}$ 水，加热至微沸。在搅拌下滴加 $10~\mathrm{mL}$ 氯化钡溶液，时间约需 $1.5\mathrm{min}$ 。继续搅拌并微沸 $2\mathrm{min}\sim 3\mathrm{min}$ ，然后盖上表面皿，保持微沸 $5\mathrm{min}$ 。再把烧杯放到沸水浴上保持 $2\mathrm{h}$ 。将烧杯冷却至室温，用慢速定量滤纸过滤。用温水洗涤沉淀至无氯离子为止（取 $5\mathrm{mL}$ 洗涤液，加 $5\mathrm{mL}$ 硝酸银溶液混匀，放置 $5\mathrm{min}$ 不出现浑浊）。将沉淀连同滤纸转移至已于 $800~^\circ \mathrm{C}\pm 25$ ℃下灼烧至质量恒定的瓷坩埚中，低温灰化，在 $800~^\circ \mathrm{C}\pm$ $25~^\circ \mathrm{C}$ 灼烧至质量恒定。

# A.4.5 结果计算

硫酸钠 $\mathrm{(Na_2SO_4)}$ 含量的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\left(m _ {1} - m _ {2}\right) \times 0.6086}{m} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$m_{1}$ 灼烧后硫酸钡及瓷坩埚的质量的数值，单位为克(g)；

$m_{2}$ ——瓷坩埚的质量的数值，单位为克(g)；

$m$ ——试样的质量的数值，单位为克 $(\mathrm{g})$ ；

0.6086 硫酸钡换算为硫酸钠的系数。

实验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A.5 铅(Pb)的测定

# A.5.1 试剂和材料

A.5.1.1 三氯甲烷。  
A.5.1.2 盐酸。  
A.5.1.3 硝酸。  
A.5.1.4 氢氧化钠溶液： $250~\mathrm{g / L}$   
A.5.1.5 吡咯烷二硫代甲酸铵(APDC)溶液： $20~\mathrm{g / L}$ 。使用前过滤。  
A.5.1.6 铅(Pb)标准溶液: $0.01 \mathrm{mg} / \mathrm{mL}$ 。  
A.5.1.7 水：符合GB/T6682—2008中二级水的规定。

# A.5.2 仪器和设备

原子吸收分光光度计：配有铅空心阴极灯。

# A.5.3 分析步骤

# A.5.3.1 玻璃仪器的清洗

均以硝酸溶液 $(1 + 2)$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

# A.5.3.2 铅标准溶液的制备

移取 $2.00\mathrm{mL}$ 铅标准溶液置于 $150~\mathrm{mL}$ 烧杯中，加入 $1\mathrm{mL}$ 盐酸。加热煮沸 $5\mathrm{min}$ ，冷却至室温，用水稀释至 $100~\mathrm{mL}$ 。用氢氧化钠溶液调整溶液 $\mathrm{pH}$ 为 $1.0\sim 1.5$ （用精密 $\mathrm{pH}$ 试纸检验）。将此溶液转移至 $250~\mathrm{mL}$ 分液漏斗中，用水稀释至约 $200~\mathrm{mL}$ 。加 $2\mathrm{mL}$ 吡咯烷二硫代氨基甲酸铵(APDC)溶液，混合。用三氯甲烷萃取两次，每次加入 $20~\mathrm{mL}$ ，收集萃取液（即有机相）于 $50~\mathrm{mL}$ 烧杯中，在蒸汽浴上蒸发至干（此操作必须在通风橱中进行），于残渣中加入 $3\mathrm{mL}$ 硝酸，继续蒸发至近干。加入 $0.5\mathrm{mL}$ 硝酸和 $10~\mathrm{mL}$ 水，加热直至溶液体积约 $3\mathrm{mL}\sim 5\mathrm{mL}$ 。转移至 $10~\mathrm{mL}$ 容量瓶，用水稀释至刻度，摇匀。

# A.5.3.3 试样溶液的制备

称取约 $10\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $150~\mathrm{mL}$ 烧杯中，加 $30~\mathrm{mL}$ 水，加 $1\mathrm{mL}$ 盐酸。盖上表面皿加热煮沸 $5\mathrm{min}$ ，冷却。用水稀释至 $100~\mathrm{mL}$ 。用氢氧化钠溶液调节溶液 $\mathrm{pH}$ 为 $1.0\sim 1.5$ （用精密 $\mathrm{pH}$ 试纸检验）。以下操作同A.5.3.2中“将此溶液转移至 $250~\mathrm{mL}$ 分液漏斗中……转移至 $10~\mathrm{mL}$ 容量瓶，用水稀释至刻度，摇匀”。

# A.5.3.4 测定

选用空气-乙炔火焰，于波长 $283.3\mathrm{nm}$ 处，用水调零，测定铅标准溶液和试样溶液的吸光度。试样溶液的吸光度不应大于铅标准溶液的吸光度。

# A.6 硒 $(\mathrm{Se})$ 的测定

# A.6.1 试剂和材料

A.6.1.1 盐酸羟胺。

A.6.1.2 环己烷。

A.6.1.3 盐酸溶液： $1 + 2$

A.6.1.4 氨水溶液： $1 + 2$

A.6.1.5 2,3-二氨基萘溶液：称取 $0.1\mathrm{g}2,3-$ 二氨基萘和 $0.5\mathrm{g}$ 盐酸羟胺，溶解在 $0.1\mathrm{mol / L}$ 的盐酸溶液中，制成 $100~\mathrm{mL}$ 溶液。

A.6.1.6 硒(Se)标准溶液： $0.001\mathrm{mg / mL}$

# A.6.2 仪器和设备

分光光度计：带有光程为 $1\mathrm{cm}$ 的比色皿。

# A.6.3 分析步骤

# A.6.3.1 试样溶液的制备

称取约 $0.2\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $150~\mathrm{mL}$ 烧杯中，加 $25~\mathrm{mL}$ 盐酸溶液，搅拌，加热煮沸

$15\mathrm{min}$ 。加 $25~\mathrm{mL}$ 水，冷却。用氨水溶液调节溶液 $\mathrm{pH}$ 为 $2\pm 0.2$ （用精密 $\mathrm{pH}$ 试纸检查)，加入 $0.2\mathrm{g}$ 盐酸羟胺，搅拌至溶解，立即加入 $5\mathrm{mL}2,3-$ 二氨基萘溶液，混匀。盖上表面皿，在室温下静置 $100\mathrm{min}$ 。然后全部转移至分液漏斗中，补加水至约 $60~\mathrm{mL}$ 。加 $5.0~\mathrm{mL}$ 环己烷，用力振荡 $2\mathrm{min}$ ，静置分层后，弃去水层，收集萃取液。

# A.6.3.2 标准溶液的制备

移取 $6.00\mathrm{mL}$ 硒标准溶液置于 $150~\mathrm{mL}$ 烧杯中，以下操作同A.6.3.1“加 $25~\mathrm{mL}$ 盐酸溶液收集萃取液”。

# A.6.3.3 空白试样溶液的制备

空白试样溶液除不加试样外，其余加入试剂的种类和量与试样溶液相同。

# A.6.3.4 测定

用 $1\mathrm{cm}$ 的比色皿，在波长约 $380~\mathrm{nm}$ 处，以空白试样溶液为参比，测定吸光度，试样溶液的吸光度不大于标准溶液的吸光度。

# A.7 干燥减量的测定

# A.7.1 仪器和设备

电热恒温干燥箱：温度能控制在 $105\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}$ 。

# A.7.2 分析步骤

用已于 $105\mathrm{^\circ C}\pm 2\mathrm{^\circ C}$ 烘干至质量恒定的称量瓶称取 $1\mathrm{g}\sim 2\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，于 $105\mathrm{^\circ C}\pm 2\mathrm{^\circ C}$ 电热恒温干燥箱中烘 $4\mathrm{h}$ 。

# A.7.3 结果计算

干燥减量的质量分数 $\varpi_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m - m _ {1}}{m} \times 100 \% \tag{A.2}
$$

式中：

$m$ ——试样的质量的数值，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——干燥后试样的质量的数值，单位为克 $(\mathrm{g})$ 。

实验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值无水硫酸钠不大于 $0.2\%$ ，十水合硫酸钠不大于 $1.0\%$ 。

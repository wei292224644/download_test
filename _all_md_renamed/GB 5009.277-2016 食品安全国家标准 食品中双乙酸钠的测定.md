# 前言

本标准代替GB/T23383—2009《食品中双乙酸钠的测定 高效液相色谱法》。

本标准与GB/T23383—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中双乙酸钠的测定”；

——修改了标准的适用范围。

# 食品安全国家标准

# 食品中双乙酸钠的测定

# 1 范围

本标准规定了食品中双乙酸钠的液相色谱测定方法。

本标准适用于豆干类、豆干再制品、原粮、粉圆、糕点、预制肉制品、熟肉制品、熟制水产品（可直接食用）、固体复合调味料、膨化食品中双乙酸钠的测定。本标准不适用于调味品、液体复合调味料及添加过乙酸的食品的测定。

# 2 原理

试样中双乙酸钠酸化后转化为乙酸，经超声波水浴提取或水蒸气蒸馏，收集后调 $\mathrm{pH}$ ，经高效液相色谱测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，色谱用水为GB/T6682规定的一级水，其他用水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。  
3.1.2 磷酸氢二铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{HPO}_{4}\right]$ 。  
3.1.3 硅油 $\left[\left(\mathrm{C}_{2} \mathrm{H}_{6} \mathrm{OSi}\right)_{n}\right]$ 。

# 3.2 试剂配制

3.2.1 磷酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：在 $500~\mathrm{mL}$ 水中加入 $53.5~\mathrm{mL}$ 磷酸，混匀后加水定容至 $1000~\mathrm{mL}$ 。  
3.2.2 磷酸氢二铵溶液 $(1.5\mathrm{g} / \mathrm{L})$ ：称取磷酸氢二铵 $1.5\mathrm{g}$ ，加水溶解定容至 $1000~\mathrm{mL}$ 。

# 3.3 标准品

双乙酸钠标准品 $\left[\left(\mathrm{CH}_{3} \mathrm{COO}\right)_{2} \mathrm{HNa}, \mathrm{CAS}\right.$ 号：126-96-5]：纯度 $\geqslant 99 \%$ 。

# 3.4 标准溶液配制

3.4.1 标准储备液 $(10\mathrm{mg / mL})$ ：准确称取双乙酸钠标准品 $1\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），用水定容至 $100~\mathrm{mL}$ 。该标准储备液置于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱内保存，保存期为3个月。  
3.4.2 标准工作液：准确吸取 $5.0\mathrm{mL}$ 标准储备液于 $50~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，配制成浓度为 $1.0\mathrm{mg / mL}$ 标准工作液。该标准工作液置于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱内保存，保存期为1个月。

# 4 仪器和设备

4.1 高效液相色谱（HPLC)仪：配有紫外检测器或二极管阵列检测器。

4.2 分析天平：感量为 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$   
4.3 组织捣碎机。  
4.4 水蒸气蒸馏装置： $500~\mathrm{mL}$ 。  
4.5 离心机：转速 $\geq 4000\mathrm{r / min}$   
4.6 具塞塑料离心管： $50~\mathrm{mL}$   
4.7 超声波清洗器。  
4.8 pH计。  
4.9 $0.45\mu \mathrm{m}$ 水相微孔滤膜。  
4.10 精密 $\mathrm{pH}$ 试纸： $\mathrm{pH}0.5\sim 5.0$

# 5 分析步骤

# 5.1 试样制备

固体样品取 $500\mathrm{g}$ ，经组织捣碎机捣碎混匀后备用；液体样品摇匀后备用。

# 5.2 试样处理

# 5.2.1 蒸馏法

准确称取 $25\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样，置于 $500\mathrm{mL}$ 蒸馏瓶中，加入 $100\mathrm{mL}$ 水，再用 $50\mathrm{mL}$ 水冲洗容器，转移到蒸馏瓶中，加磷酸溶液 $(1\mathrm{mol} / \mathrm{L})20\mathrm{mL},2$ 滴～3滴硅油，进行水蒸气蒸馏，将 $250\mathrm{mL}$ 容量瓶置于冰浴中作为吸收液接收装置，待蒸馏约 $240\mathrm{mL}$ 时取出，在室温下放置 $30\mathrm{min}$ ，用 $1\mathrm{mol} / \mathrm{L}$ 磷酸溶液调 $\mathrm{pH}$ 为3左右，加水定容，摇匀，经 $0.45\mu \mathrm{m}$ 水相微孔滤膜过滤后，供液相色谱测定。

# 5.2.2 直接浸提法（仅适用于馒头、花卷）

准确称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样至 $100\mathrm{mL}$ 烧杯中，加水 $20\mathrm{mL}$ ，加入 $1\mathrm{mol/L}$ 磷酸溶液 $0.5\mathrm{mL}$ ，混匀，经超声浸提 $10\mathrm{min}$ 后，用磷酸溶液 $(1\mathrm{mol/L})$ 调 $\mathrm{pH}$ 为3左右，转移试样至 $50\mathrm{mL}$ 容量瓶中，用水定容至刻度，摇匀。将试样全部转移至 $50\mathrm{mL}$ 具塞塑料离心管中，以不低于 $4000\mathrm{r/min}$ 离心 $10\mathrm{min}$ ，取上清液，经 $0.45\mu\mathrm{m}$ 水相微孔滤膜过滤后，供液相色谱测定。

# 5.3 仪器参考条件

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或等效色谱柱；  
b） 柱温： $25^{\circ}C$   
c) 流动相：用磷酸溶液 $(1\mathrm{mol} / \mathrm{L})$ 调节磷酸氢二铵溶液 $(1.5\mathrm{g} / \mathrm{L})\mathrm{pH}$ 为 $2.7\sim 3.5$ （临用现配），经 $0.45\mu \mathrm{m}$ 水相微孔滤膜过滤；  
d) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
e）波长： $214~\mathrm{nm}$   
f) 进样量: $20 \mu \mathrm{L}$ ;   
g）色谱柱清洗参考条件：试验结束后，用 $10\%$ 甲醇清洗 $1\mathrm{h}$ ，再用 $100\%$ 甲醇清洗 $1\mathrm{h}$ 。

# 5.4 标准曲线的制作

# 5.4.1 蒸馏法

准确吸取双乙酸钠标准储备液 $1.0\mathrm{mL},2.5\mathrm{mL},5.0\mathrm{mL},7.5\mathrm{mL},10.0\mathrm{mL},12.5\mathrm{mL}$ 置于 $500~\mathrm{mL}$

蒸馏瓶中，其他操作同5.2.1，其双乙酸钠标准溶液最终浓度分别为 $0.04\mathrm{mg / mL},0.1\mathrm{mg / mL},0.2\mathrm{mg / mL}$ 、 $0.3\mathrm{mg / mL},0.4\mathrm{mg / mL},0.5\mathrm{mg / mL}$ ，经 $0.45\mu \mathrm{m}$ 水相微孔滤膜过滤后分别注入液相色谱仪中，测定相应的峰面积，以标准工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。

# 5.4.2 直接浸提法

准确吸取标准工作液 $0.5\mathrm{mL},1.0\mathrm{mL},2.0\mathrm{mL},3.0\mathrm{mL},4.0\mathrm{mL},5.0\mathrm{mL}$ 至 $10~\mathrm{mL}$ 容量瓶中，分别加入磷酸溶液 $(1\mathrm{mol} / \mathrm{L})0.2\mathrm{mL}$ ，用水定容至 $10~\mathrm{mL}$ 。其双乙酸钠标准溶液浓度分别为 $0.05\mathrm{mg / mL}$ 、 $0.1\mathrm{mg / mL}$ 、 $0.2\mathrm{mg / mL}$ 、 $0.3\mathrm{mg / mL}$ 、 $0.4\mathrm{mg / mL}$ 、 $0.5\mathrm{mg / mL}$ ，经 $0.45\mu \mathrm{m}$ 水相微孔滤膜过滤后分别注入液相色谱仪中，测定相应的峰面积，以标准工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。

# 5.5 试样溶液的测定

将试样溶液注入液相色谱仪中，得到峰面积，根据标准曲线得到待测液中双乙酸钠的浓度。双乙酸钠标准溶液液相色谱图参见图A.1。

# 6 分析结果的表述

试样中双乙酸钠含量按式(1)计算：

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0} \times f \quad \dots \tag {1}
$$

式中：

$X$ ——试样中双乙酸钠的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho$ ——由标准曲线求出样液中双乙酸钠的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——样品质量，单位为克 $(\mathrm{g})$ ；

$f$ ——稀释倍数。

计算结果保留三位有效数字。

注：若在GB2760中未规定添加双乙酸钠的食品中检出双乙酸钠时或在GB2760中规定允许添加双乙酸钠的食品中检出值超出限量范围时，则需要对该食品样品中乙酸的本底值进行测定，具体按附录B操作。在计算该食品中双乙酸钠的含量时，应扣减乙酸本底的含量。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

# 8.1 蒸馏法检出限

当样品称样量为 $25.0\mathrm{g}$ ，定容体积为 $250~\mathrm{mL}$ 时，方法检出限为： $0.030~\mathrm{g / kg}$ ；方法定量限为： $0.100~\mathrm{g / kg}$ 。

# 8.2 直接浸提法

当样品称样量为 $5.0\mathrm{g}$ ，定容体积为 $50~\mathrm{mL}$ 时，方法检出限为： $0.015\mathrm{g / kg}$ ；方法定量限为： $0.05\mathrm{g / kg}$ 。

# 食品营养强化剂 葡萄糖酸镁

# 1 范围

本标准适用于以葡萄糖酸与氧化镁(或碳酸镁),或者葡萄糖酸-δ-内酯与氧化镁(或碳酸镁)为原料,经化学合成、精制而得的食品营养强化剂葡萄糖酸镁。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

D-葡萄糖酸镁

# 2.2 分子式

$$
\mathrm {C} _ {1 2} \mathrm {H} _ {2 2} \mathrm {M g O} _ {1 4} \cdot n \mathrm {H} _ {2} \mathrm {O} (n = 0 \text {或} 2)
$$

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂及其制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 冰乙酸。  
A.2.1.2 乙醇。  
A.2.1.3 苯肼。  
A.2.1.4 氯化铵溶液： $200\mathrm{g / L}$   
A.2.1.5 碳酸铵溶液： $200\mathrm{g / L}$   
A.2.1.6 磷酸钠溶液： $60~\mathrm{g / L}$   
A.2.1.7 氨水溶液： $2 + 3$

# A.2.2 鉴别方法

# A.2.2.1 溶解性

易溶于水，微溶于乙醇。

# A.2.2.2 镁离子的鉴别

取约 $0.5\mathrm{g}$ 试样，溶于 $10~\mathrm{mL}$ 水，加 $5\mathrm{mL}$ 氯化铵溶液、 $5\mathrm{mL}$ 碳酸铵溶液，搅拌，不产生沉淀，再加入 $5\mathrm{mL}$ 磷酸钠溶液，产生白色结晶沉淀，加入 $10~\mathrm{mL}$ 氨水溶液白色结晶沉淀不溶解。

# A.2.2.3 葡萄糖的鉴别

# A.2.2.3.1 方法提要

试样在乙酸介质中，与苯肼共热，生成黄色葡萄糖酰苯肼结晶。

# A.2.2.3.2 分析步骤

称取约 $0.5\mathrm{g}$ 试样，加 $10~\mathrm{mL}$ 水，溶解（必要时加热）。取 $5\mathrm{mL}$ ，加 $0.7\mathrm{mL}$ 冰乙酸和 $1\mathrm{mL}$ 新蒸馏苯肼，沸水浴加热 $30\mathrm{min}$ ，放至室温，用玻璃棒摩擦试管内壁，则析出黄色结晶。

# A.3 葡萄糖酸镁 $\left(\mathbf{C}_{12}\mathbf{H}_{22}\mathbf{MgO}_{14}\right)$ 含量（以干基计）的测定

# A.3.1 方法提要

在试样溶液中加入氨-氯化铵缓冲溶液，以铬黑T为指示剂，用乙二胺四乙酸二钠标准滴定溶液滴

定镁含量。

# A.3.2 试剂和材料

A.3.2.1 氨-氯化铵缓冲溶液： $\mathrm{pH}\approx 10$   
A.3.2.2 乙二胺四乙酸二钠标准滴定溶液： $c(\mathrm{EDTA}) = 0.05\mathrm{mol / L}$   
A.3.2.3 铬黑T指示液 $(5\mathrm{g} / \mathrm{L})$

# A.3.3 分析步骤

称取约 $0.6\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 锥形瓶中，加 $50~\mathrm{mL}$ 水溶解，加入 $10~\mathrm{mL}$ 氨-氯化铵缓冲溶液，5滴铬黑T指示液，用乙二胺四乙酸二钠标准滴定溶液滴定至溶液由紫红色变为纯蓝色。同时进行空白试验。

# A.3.4 结果计算

葡萄糖酸镁 $\mathrm{(C_{12}H_{22}MgO_{14})}$ 含量(以干基计)的质量分数 $\varpi_1$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\left(V _ {1} - V _ {0}\right) \times c _ {1} \times M _ {1}}{m _ {1} \times \left(1 - w _ {2}\right) \times 1000} \times 100 \% \dots \tag{A.1}
$$

式中：

$V_{1}$ ——滴定试样溶液消耗的乙二胺四乙酸二钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白溶液消耗的乙二胺四乙酸二钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——乙二胺四乙酸二钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{1}$ ——葡萄糖酸镁的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M_1(\mathrm{C_{12}H_{22}MgO_{14}}) = 414.60]$

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$w_{2}$ ——实测试样水分的质量分数， $\%$ ；

1000——换算因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A. 4 水分的测定

# A.4.1 试剂

A.4.1.1 甲醇：色谱纯。或无水甲醇： $500~\mathrm{mL}$ 甲醇中加入 $50\mathrm{g}5\mathrm{A}$ 分子筛。  
A.4.1.2 甲酰胺：如试剂中水的质量分数大于 $0.05\%$ ，于 $500~\mathrm{mL}$ 甲酰胺中加入 $50\mathrm{g}5\mathrm{A}$ 分子筛，塞上瓶塞，放置过夜，吸取上层清液使用。  
A.4.1.3 提取溶剂：把甲醇(A.4.1.1)和甲酰胺(A.4.1.2)按 $1 + 1$ （体积比)混合，摇匀。  
A.4.1.4 卡尔·费休试剂。

# A.4.2 分析步骤

称取约 $0.1\mathrm{g}$ 样品，精确至 $0.0001\mathrm{g}$ ，加入到适量的提取溶剂(A.4.1.3)中，滴定前搅拌 $30\mathrm{min}$ 。按GB/T606测定其中的水分含量。

# A.5 还原物质（以D-葡萄糖计)的测定

# A.5.1 方法提要

还原糖将二价铜离子还原成氧化亚铜，剩余的二价铜离子在酸性条件下与碘离子反应生成定量的碘，以硫代硫酸钠标准滴定溶液滴定生成的碘，从而计算出样品中还原糖的含量。

# A.5.2 试剂和材料

# A.5.2.1 碱性柠檬酸铜溶液

溶液A：称取 $173\mathrm{g}$ 柠檬酸钠(枸橼酸钠)和 $100\mathrm{g}$ 无水碳酸钠，加温水溶解并稀释至 $700~\mathrm{mL}$ （若溶液显浑浊，过滤使澄清）。

溶液B：称取 $17.3\mathrm{g}$ 硫酸铜结晶，加水溶解并稀释至 $100~\mathrm{mL}$

临用前取 $100\mathrm{mL}$ 溶液B，在不断振摇下，缓缓加入 $700~\mathrm{mL}$ 溶液A，冷却后，加水定容至 $1000~\mathrm{mL}$ 。

A.5.2.2 碘标准溶液： $c(\frac{1}{2}\mathrm{I}_2) = 0.05\mathrm{mol / L}$

A.5.2.3 乙酸溶液： $10\%$ 。量取 $96~\mathrm{mL}$ 乙酸(冰醋酸)，稀释至 $1000~\mathrm{mL}$   
A.5.2.4 盐酸溶液： $3\mathrm{mol} / \mathrm{L}$ 。取 $258~\mathrm{mL}$ 盐酸，加水稀释至 $1\mathrm{L}$   
A.5.2.5 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$   
A.5.2.6 淀粉指示液： $10\mathrm{g / L}$

# A.5.3 分析步骤

称取 $1.0\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 碘量瓶中，加 $10~\mathrm{mL}$ 水溶解，加入 $25~\mathrm{mL}$ 碱性柠檬酸铜溶液，瓶口用小表面皿盖住，微沸 $5\mathrm{min}$ 后，迅速冷却至室温，加 $25~\mathrm{mL}$ 乙酸溶液，摇匀，准确加入 $10.0~\mathrm{mL}$ 碘标准溶液，密塞，摇匀，放置 $10\mathrm{min}$ ，加入 $10~\mathrm{mL}$ 盐酸溶液，再加 $3~\mathrm{mL}$ 淀粉指示液，立即用硫代硫酸钠标准滴定溶液滴定至溶液显亮蓝色。同时进行空白试验。

# A.5.4 结果计算

还原物质（以D-葡萄糖计）的质量分数 $\varpi_{3}$ ，按式(A.2)计算：

$$
w_{3} = \frac{\left(V_{2} - V_{3}\right)\times c_{2}\times M_{2}}{m_{2}\times 1000}\times 100\% \quad \dots \tag{A.2}
$$

式中：

$V_{2}$ ——滴定空白溶液消耗的硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——滴定试样溶液消耗的硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{2}$ ——硫代硫酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{2}$ ——还原糖 $\left(\frac{3}{20} \mathrm{C}_{6} \mathrm{H}_{12} \mathrm{O}_{6}\right)$ 的摩尔质量，单位为克每摩尔 $(\mathrm{g/mol})$ ， $M_{2}\left(\frac{3}{20} \mathrm{C}_{6} \mathrm{H}_{12} \mathrm{O}_{6}\right)=27$

$m_{2}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.6 氯化物（以CI计）的测定

# A.6.1 试剂和材料

A.6.1.1 硝酸溶液： $1 + 9$   
A.6.1.2 硝酸银溶液： $17\mathrm{g / L}$   
A.6.1.3 氯化物(Cl)标准溶液： $0.01\mathrm{mg / mL}$

# A.6.2 分析步骤

称取 $1.0\mathrm{g}$ 样品，精确至 $0.01\mathrm{g}$ ，加 $40~\mathrm{mL}$ 水溶解，用硝酸溶液调节溶液至中性，用水稀释至 $50~\mathrm{mL}$ ，振摇，过滤，保留滤液。取 $5.0~\mathrm{mL}$ 滤液，作为试样溶液，置于 $50~\mathrm{mL}$ 纳氏比色管中，加 $10~\mathrm{mL}$ 硝酸溶液，加水至体积约 $40~\mathrm{mL}$ ，加 $1.0~\mathrm{mL}$ 硝酸银溶液，用水稀释至 $50~\mathrm{mL}$ ，摇匀，暗处放置 $5\mathrm{min}$ ，在黑色背景下，轴向观察，试样溶液所呈浊度不得大于标准比浊溶液。

标准比浊溶液：量取 $5.0\mathrm{mL}$ 氯化物标准溶液，置于 $50~\mathrm{mL}$ 比色管中。与试样溶液同时同样处理。

# A.7 硫酸盐（以 $\mathrm{SO}_4$ 计）的测定

# A.7.1 试剂和材料

A.7.1.1 盐酸溶液： $1 + 3$   
A.7.1.2 氯化钡溶液： $250~\mathrm{g / L}$   
A.7.1.3 硫酸钾标准溶液： $0.1\mathrm{mg / mL}$

# A.7.2 分析步骤

称取 $2.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加 $40~\mathrm{mL}$ 水溶解，用盐酸溶液调节溶液至中性，用水稀释至 $50~\mathrm{mL}$ ，振摇，过滤，保留滤液。取 $25~\mathrm{mL}$ 滤液，作为试样溶液，置于 $50~\mathrm{mL}$ 纳氏比色管中，加 $2\mathrm{mL}$ 盐酸溶液，加水至体积约 $40~\mathrm{mL}$ ，加 $5\mathrm{mL}$ 氯化钡溶液，用水稀释至 $50~\mathrm{mL}$ ，摇匀，放置 $10\mathrm{min}$ ，在黑色背景下，轴向观察，试样溶液所呈浊度不得大于标准比浊溶液。

标准比浊溶液：量取 $5.0\mathrm{mL}$ 硫酸钾标准溶液，置于 $50~\mathrm{mL}$ 比色管中。与试样溶液同时同样处理。

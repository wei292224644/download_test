# 液相色谱-串联质谱法

# 1 范围

本标准规定了猪组织和尿液中赛庚啶和可乐定的制样和液相色谱-串联质谱测定方法。

本标准适用于猪肌肉、肝脏、肾脏及尿液中赛庚啶和可乐定残留量的检测。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

组织试样中残留的赛庚啶或可乐定，用酸性乙腈提取，液相色谱-串联质谱测定，内标法定量。尿液中残留的赛庚啶或可乐定，经酸化、固相萃取柱净化，液相色谱-串联质谱测定，内标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 甲醇（ $\mathrm{CH}_3\mathrm{OH}$ ）：色谱纯。  
4.1.2 乙腈（ $\mathrm{CH}_3\mathrm{CN}$ ）：色谱纯。  
4.1.3 甲酸（HCOOH）：色谱纯。  
4.1.4 氨水（ $\mathrm{NH_4OH}$ ）。  
4.1.5 盐酸（HCl）。  
4.1.6 硫酸镁（ $\mathrm{Mg_2SO_4}$ ）。  
4.1.7 氯化钠（NaCl）。

# 4.2 溶液配制

4.2.1 $1\mathrm{mol} / \mathrm{L}$ 盐酸溶液：取盐酸 $90~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。

4.2.2 $1\%$ 甲酸乙腈溶液：取甲酸 $5 \mathrm{~mL}$ ，用乙腈稀释至 $500 \mathrm{~mL}$ 。  
4.2.3 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。  
4.2.4 乙腈-0.1%甲酸溶液：取 $0.1\%$ 甲酸溶液 $80~\mathrm{mL}$ ，加乙腈至 $100~\mathrm{mL}$ ，混匀。  
4.2.5 $5\%$ 氨水甲醇溶液：取氨水 $5\mathrm{mL}$ ，用甲醇稀释至 $100\mathrm{mL}$ 。

# 4.3 标准品

4.3.1 盐酸赛庚啶（Cyproheptadine hydrochloride, $\mathrm{C}_{21}\mathrm{H}_{21}\mathrm{N}$ HCl, CAS号:969-33-5）、盐酸可乐定（Clonidine hydrochloride, $\mathrm{C}_{9}\mathrm{H}_{10}\mathrm{Cl}_{3}\mathrm{N}_{3}$ , CAS号:4205-91-8）：含量均 $\geqslant 98.0\%$ 。  
4.3.2 内标：盐酸二苯拉林（Diphenylpyraline, $\mathrm{C}_{9} \mathrm{H}_{13} \mathrm{NO} \mathrm{HCl}, \mathrm{CAS}$ 号:61-76-7）、盐酸可乐定 $-\mathrm{D}_{4}$ （Clonidine-D4 hydrochloride, $\mathrm{C}_{9} \mathrm{H}_{6} \mathrm{D}_{4} \mathrm{Cl}_{3} \mathrm{~N}_{3}$ ）：含量均 $\geqslant 98 \%$ 。

# 4.4 标准溶液制备

4.4.1 标准储备液（ $1\mathrm{mg/mL}$ ）：取盐酸赛庚啶、盐酸可乐定、盐酸二苯拉林和盐酸可乐定- $\mathrm{D}_4$ 标准品各约 $10\mathrm{mg}$ ，精密称定，分别于 $10\mathrm{mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度分别为 $1\mathrm{mg/mL}$ 的盐酸赛庚啶、盐酸可乐定、盐酸二苯拉林和盐酸可乐定- $\mathrm{D}_4$ 标准贮备液。 $4^{\circ}\mathrm{C}$ 下保存，有效期6个月。  
4.4.2 混合标准工作液（ $10\mu \mathrm{g} / \mathrm{mL}$ ）：分别精密量取盐酸赛庚啶和盐酸可乐定标准贮备液各 $1\mathrm{mL}$ ，于 $100\mathrm{mL}$ 量瓶中，用甲醇溶解并稀释至刻度，制成浓度为 $10\mu \mathrm{g} / \mathrm{mL}$ 的盐酸赛庚啶和盐酸可乐定标准工作液。 $4^{\circ}\mathrm{C}$ 下保存，有效期3个月。  
4.4.3 混合内标中间液：分别精密量取盐酸二苯拉林储备液 $100 \mu \mathrm{L}$ 、盐酸可乐定- $\mathrm{D}_{4}$ 储备液 $1 \mathrm{~mL}$ ，于 $100 \mathrm{~mL}$ 量瓶中，用甲醇稀释至刻度，配制成含盐酸二苯拉林 $1 \mu \mathrm{g} / \mathrm{mL}$ 和盐酸可乐定- $\mathrm{D}_{4} 10 \mu \mathrm{g} / \mathrm{mL}$ 的混合内标中间液。于 $4^{\circ} \mathrm{C}$ 下保存，有效期1个月。  
4.4.4 混合内标工作液：精密量取混合内标中间液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 量瓶中，用乙腈- $0.1\%$ 甲酸溶液稀释至刻度，配制成含盐酸二苯拉林 $100\mathrm{ng/mL}$ 和盐酸可乐定- $\mathrm{D}_41\mu\mathrm{g/mL}$ 的混合内标工作液，临用现配。  
4.4.5 标准曲线制备：精密量取混合标准工作液和混合内标工作液适量，用乙腈- $0.1\%$ 甲酸溶液稀释成赛庚啶和可乐定浓度为0.5、1、10、50和 $100~\mu \mathrm{g / L}$ 系列混合标准工作液，其中含盐酸二苯拉林的浓度和盐酸可乐定- $\mathrm{D_4}$ 的浓度分别为 $5\mu \mathrm{g / L}$ 和 $50~\mu \mathrm{g / L}$ ，临用现配。

# 4.5 材料

4.5.1 $\mathrm{C_{18}}$ 粉：碳载量25.0-28.0。  
4.5.2 混合型强阳离子固相萃取柱， $60\mathrm{mg} / 3\mathrm{mL}$ 或相当者。

4.5.3 滤膜： $0.22\mu \mathrm{m}$ ，水相。

# 5 仪器和设备

5.1 液相色谱一串联质谱仪：配电喷雾电离源。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
5.3 均质机。  
5.4 冷冻离心机： $8000\mathrm{r / min}$ 以上。  
5.5 涡旋振荡器。  
5.6 氮吹仪。  
5.7 固相萃取装置。  
5.8 pH计。  
5.9 离心管： $50~\mathrm{mL}$

# 6 试样的制备与保存

# 6.1 试料的制备

6.1.1 猪肌肉、肝脏、肾脏

取适量新鲜或解冻的空白或供试组织，绞碎并使均质。

——取均质的供试样品，作为供试试料。  
——取均质的空白样品，作为空白试料。  
——取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.1.2 猪尿样

取适量新鲜或解冻的空白或供试猪尿，使用前恢复室温，混匀，如有混浊，离心后取上清液备用。

——取备用的供试样品，作为供试材料。  
——取备用的空白样品，作为空白材料。  
——取备用的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-20°C以下保存。

# 7 测定步骤

# 7.1 提取

# 7.1.1 组织样品

取试料 $2 \mathrm{~g}$ (准确至 $\pm 20 \mathrm{mg}$ ), 于 $50 \mathrm{~mL}$ 离心管中, 加 $50 \mu \mathrm{L}$ 混合内标工作液, 猪肌肉试样加 $\mathrm{C}_{18}$ 粉 $0.5 \mathrm{~g}$ , 肝脏、肾脏加硫酸镁 $3.0 \mathrm{~g}$ 、氯化钠 $2.0 \mathrm{~g}$ , 振荡混匀, 加 $1 \%$ 甲酸乙腈 $10 \mathrm{~mL}$ , 充分振荡, 于 $8000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 取上清液, 于离心管中, $50^{\circ} \mathrm{C}$ 水浴氮气吹干。加乙腈- $0.1 \%$ 甲酸溶液 $1.0 \mathrm{~mL}$ 使溶解, $0.22 \mu \mathrm{m}$ 滤膜滤过, 液相色谱-串联质谱仪测定。

# 7.1.2 尿样

取试料 $5.0 \mathrm{~mL}$ ，于 $50 \mathrm{~mL}$ 离心管中，加 $50 \mu \mathrm{L}$ 混合内标工作液，用 $1 \mathrm{~mol} / \mathrm{L}$ 盐酸调 $\mathrm{pH}$ 小于2，涡旋混匀， $8000 \mathrm{r} / \mathrm{min} 4^{\circ} \mathrm{C}$ 离心 $10 \mathrm{~min}$ ，将全部上清液转移至另一 $50 \mathrm{~mL}$ 离心管中，备用。

# 7.2 净化

尿样：取固相萃取柱，依次用甲醇、水各 $3\mathrm{mL}$ 活化。取备用液全部过柱，用 $0.1\%$ 甲酸溶液 $3\mathrm{mL}$ 和甲醇 $3\mathrm{mL}$ 依次淋洗，抽干 $2\mathrm{min}$ ，用 $5\%$ 氨水甲醇溶液 $3\mathrm{mL}$ 洗脱，收集洗脱液，于 $50^{\circ}\mathrm{C}$ 水浴氮气吹干，用乙腈 $-0.1\%$ 甲酸溶液 $1.0\mathrm{mL}$ 溶解， $0.22\mu\mathrm{m}$ 滤膜滤过，液相色谱-串联质谱仪测定。

c）进样量： $10~\mu \mathrm{L}$   
d）流速： $0.3\mathrm{mL / min}$

# 7.4 测定法

# 7.4.1 定性测定

在同样测试条件下，试样液中赛庚啶和可乐定的保留时间与标准工作液中相应的保留时间之比，偏差在 $\pm 5\%$ 以内，且检测到的离子的相对丰度，应当与浓度相当的校正标准溶液相对丰度一致。其允许偏差应符合表3要求。

# 7.4.2 定量测定

取试样溶液和标准溶液，作单点校准，按内标法定量。标准溶液及试样溶液中的化合物组份及内标的响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，赛庚啶、可乐定及内标标准溶液特征离子质量色谱图见附录A。

# 8 结果计算和表述

试样中待测药物的残留量按式（1）～（2）计算：

组织样品：

$$
X = \frac {A \times A _ {i S} ^ {\prime} \times C _ {S} \times C _ {i S} \times V}{A _ {i S} \times A _ {S} \times C _ {i S} ^ {\prime} \times m}. \tag {1}
$$

尿液样品：

$$
X = \frac {A \times A _ {i S} ^ {\prime} \times C _ {S} \times C _ {i S} \times V}{A _ {i S} \times A _ {S} \times C _ {i S} ^ {\prime} \times V _ {\text {皿}}} \tag {2}
$$

式中：

$X-$ 试样中赛庚啶或可乐定残留量, 单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ 或微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ;

$\mathrm{C}_{\mathrm{iS}}$ - 试样溶液中赛庚啶或可乐定对应内标物浓度, 单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ;

$\mathrm{C}_{\mathrm{S}}$ - 混合标准工作液中赛庚啶或可乐定浓度, 单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ;

$\mathrm{C}_{\mathrm{iS}}^{\prime}$ - 混合标准工作液中赛庚啶或可乐定对应内标物浓度，单位为微克每升（ $\mu \mathrm{g} / \mathrm{L}$ ）；

A-试样溶液中赛庚啶或可乐定峰面积；

$\mathrm{A}_{\mathrm{iS}}$ 一试样溶液中赛庚啶或可乐定对应内标物峰面积；

$A_{S}$ 一混合标准工作液中赛庚啶或可乐定峰面积；

$A_{iS}^{\prime}$ - 混合标准工作液中赛庚啶或可乐定对应内标物峰面积；

$V$ —溶解残余物所用溶液体积, 单位为毫升 $(\mathrm{mL})$ ;

$m$ 一供试样品质量，单位为克（g）。

$V_{m}$ -供试尿样体积, 单位为毫升 $(\mathrm{mL})$ ;

计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度和精密度

# 9.1 灵敏度

本方法的猪肌肉、肝脏和肾脏中赛庚啶和可乐定检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1\mu \mathrm{g / kg}$ 猪尿中赛庚啶和可乐定检测限为 $0.5\mu \mathrm{g / L}$ ，定量限为 $1\mu \mathrm{g / L}$ 。

# 9.2 准确度

本方法在猪肉、猪肝和猪肾中 $1\sim 10~\mu \mathrm{g / kg}$ 、猪尿中 $1\sim 10~\mu \mathrm{g / L}$ 添加浓度水平上的回收率 $60\% \sim 120\%$ 。

# 9.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 30\%$

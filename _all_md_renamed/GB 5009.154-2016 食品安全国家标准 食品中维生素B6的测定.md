# 前言

本标准代替GB/T5009.154—2003《食品中维生素 $\mathrm{B}_{6}$ 的测定》、GB5413.13—2010《食品安全国家标准婴幼儿食品和乳品中维生素 $\mathrm{B}_{6}$ 的测定》。

本标准与GB/T5009.154—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中维生素 $\mathrm{B}_{6}$ 的测定”；

——增加了高效液相色谱法。

# 食品安全国家标准

# 食品中维生素 $\mathbf{B}_{6}$ 的测定

# 1 范围

本标准规定了食品中维生素 $\mathrm{B}_{6}$ 的测定方法。

本标准第一法为高效液相色谱法，适用于添加了维生素 $\mathrm{B}_{6}$ 的食品测定；第二法为微生物法，适用于各类食品中维生素 $\mathrm{B}_{6}$ 的测定。

# 第一法 高效液相色谱法

# 2 原理

试样经提取等前处理后，经 $\mathrm{C_{18}}$ 色谱柱分离，高效液相色谱-荧光检测器检测，外标法定量测定维生素 $\mathrm{B}_{6}$ （吡哆醇、吡哆醛、吡哆胺)的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 含淀粉的试样

a）固体试样：称取混合均匀的固体试样约 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，于 $150~\mathrm{mL}$ 锥形瓶中，加入约25 mL $45^{\circ}C\sim 50^{\circ}C$ 的水，混匀。加入约 $0.5\mathrm{g}$ 淀粉酶，混匀后向锥形瓶中充氮，盖上瓶塞，置 $50^{\circ}C\sim 60^{\circ}C$ 培养箱内约 $30\mathrm{min}$ 。取出冷却至室温。  
b) 液体试样: 称取混合均匀的液体试样约 $20 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ ) 于 $150 \mathrm{~mL}$ 锥形瓶中, 混匀。加入约 $0.5 \mathrm{~g}$ 淀粉酶, 混匀后向锥形瓶中充氮, 盖上瓶塞, 置 $50^{\circ} \mathrm{C} \sim 60^{\circ} \mathrm{C}$ 培养箱内约 $30 \mathrm{~min}$ 。取出冷却至室温。

# 5.1.2 不含淀粉的试样

a）固体试样：称取混合均匀的固体试样约 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），于 $150~\mathrm{mL}$ 锥形瓶中，加入约25

$\mathrm{mL}45^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 的水，混匀。静置 $5\mathrm{min}\sim 10\mathrm{min}$ ，冷却至室温。

b）液体试样：称取混合均匀的液体试样约 $20\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）于 $150~\mathrm{mL}$ 锥形瓶中。静置 $5\mathrm{min}$ $\sim 10\mathrm{min}$ 。

# 5.1.3 待测液的制备

用盐酸溶液，调节上述试样溶液的 $\mathrm{pH}$ 至 $1.7\pm 0.1$ ，放置约 $1\mathrm{min}$ 。再用氢氧化钠溶液调节试样溶液的 $\mathrm{pH}$ 至 $4.5\pm 0.1$ 。把上述锥形瓶放入超声波振荡器中，超声振荡约 $10\mathrm{min}$ 。将试样溶液转移至 $50~\mathrm{mL}$ 容量瓶中，用水冲洗锥形瓶。洗液合并于 $50~\mathrm{mL}$ 容量瓶中，用水定容至 $50~\mathrm{mL}$ 。另取 $50~\mathrm{mL}$ 锥形瓶，上面放入漏斗和滤纸，把定容后的试样溶液倒入其中，自然过滤。滤液再经 $0.45\mu \mathrm{m}$ 微孔滤膜过滤，用试管收集，转移 $1\mathrm{mL}$ 滤液至进样瓶作为试样待测液。

注：操作过程应避免强光照射。

# 5.2 仪器参考条件

仪器参考条件列出如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $150~\mathrm{mm}$ ，柱内径 $4.6  \mathrm{mm}$ ，柱填料粒径 $5\mu \mathrm{m}$ ，或相当者；  
b）流动相：甲醇 $50~\mathrm{mL}$ 、辛烷磺酸钠 $2.0\mathrm{g}$ 、三乙胺 $2.5\mathrm{mL}$ ，用水溶解并定容到 $1000~\mathrm{mL}$ 后，用冰乙酸调 $\mathrm{pH}$ 至 $3.0\pm 0.1$ ，过 $0.45\mu \mathrm{m}$ 微孔滤膜过滤；

c) 流速: $1 \mathrm{~mL} / \mathrm{min}$ ;   
d) 柱温: $30^{\circ} \mathrm{C}$ ;   
e）检测波长：激发波长 $293\mathrm{nm}$ ，发射波长 $395\mathrm{nm}$   
f）进样体积： $10~\mu \mathrm{L}$

# 5.3 标准曲线的制作

将维生素 $\mathrm{B}_{6}$ 混合标准系列工作液分别注入高效液相色谱仪中，测定各组分的峰面积，以相应标准工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。

# 5.4 试样溶液的测定

将试样溶液注入高效液相色谱仪中，得到各组分相应的峰面积，根据标准曲线得到待测试样溶液中维生素 $\mathrm{B}_{6}$ 各组分的浓度。

# 6 分析结果的表述

试样中维生素 $\mathrm{B}_{6}$ 各组分的含量按式(1)计算：

$$
X _ {i} = \frac {\rho \times V}{m} \times \frac {1 0 0}{1 0 0 0} \tag {1}
$$

式中：

$X_{i}$ ——试样中维生素 $\mathrm{B}_{6}$ 各组分的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$\rho$ ——根据标准曲线计算得到的试样中维生素 $\mathrm{B}_{6}$ 各组分的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

V ——试样溶液的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

100 ——换算为 100 克样品中含量的换算系数；

1000——将浓度单位 $\mu \mathrm{g} / \mathrm{mL}$ 换算为 $\mathrm{mg / mL}$ 的换算系数。

试样中维生素 $\mathrm{B}_{6}$ 的含量按式(2)计算：

$$
X = X _ {\text {醇}} + X _ {\text {醛}} \times 1. 0 1 2 + X _ {\text {胺}} \times 1. 0 0 6 \quad \dots \tag {2}
$$

式中：

$X$ ——试样中维生素 $\mathrm{B}_{6}$ （以吡哆醇计）的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$

$X_{\text{醇}}$ ——试样中吡哆醇的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$X_{\text{醛}}$ ——试样中吡哆醛的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

1.012 ——吡哆醛的含量换算成吡哆醇的系数；

$X_{\text{胺}}$ ——试样中吡哆胺的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

1.006 ——吡哆胺的含量换算成吡哆醇的系数。

结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

当取样量为 $5.00\mathrm{g}$ 时，方法检出限为：吡哆醇 $0.02\mathrm{mg} / 100\mathrm{g}$ ，吡哆醛 $0.02\mathrm{mg} / 100\mathrm{g}$ ，吡哆胺 $0.02\mathrm{mg} / 100\mathrm{g}$ ；方法定量限为：吡哆醇 $0.05\mathrm{mg} / 100\mathrm{g}$ ，吡哆醛 $0.05\mathrm{mg} / 100\mathrm{g}$ ，吡哆胺 $0.05\mathrm{mg} / 100\mathrm{g}$ 。

# 第二法 微生物法

# 9 原理

食品中某一种细菌的生长必须要有某一种维生素的存在，卡尔斯伯(Saccharomyces Carlsbrgensis)酵母菌在有维生素 $\mathrm{B}_6$ 存在的条件下才能生长，在一定条作下维生素 $\mathrm{B}_6$ 的量与其生长呈正比关系。用比浊法测定该菌在试样液中生长的浑浊度，与标准曲线相比较得出试样中维生素 $\mathrm{B}_6$ 的含量。

# 12.1 菌种的制备及保存（避光处理）

12.1.1 菌种复壮：卡尔斯伯酵母(Saccharomyces carlsbergensis)，ATCC#9080菌种或等效菌种冻干品，加入约 $0.5\mathrm{mL}$ YM肉汤培养基或生理盐水复溶，取几滴复溶的菌液分别接种2支装有 $10\mathrm{mLYM}$ 肉汤培养基的试管中，于 $30^{\circ}C$ 水浴振荡培养 $20\mathrm{h}\sim 24\mathrm{h}$   
12.1.2 月储备菌种制备：将菌种复壮培养液划线接种于YM肉汤琼脂培养基(传代培养基)斜面上，于 $30\%$ 培养 $20\mathrm{h}\sim 24\mathrm{h}$ ，于 $2\% \sim 8\%$ 冰箱内保存，此菌种为第一代月储备菌种；以后每月将上一代的月储备菌种划线接种于YM肉汤琼脂培养基(传代培养基)斜面，于 $30\%$ 培养 $20\mathrm{h}\sim 24\mathrm{h}$ ，于 $2\% \sim 8\%$ 冰箱内保存，有效期一个月，此菌种为当月储备菌种。  
12.1.3 周储备菌种制备：每周从当月储备菌种接种于 YM 肉汤琼脂培养基(传代培养基)斜面，于 $30^{\circ} \mathrm{C}$ 培养 $20 \mathrm{~h} \sim 24 \mathrm{~h}$ ，于 $2^{\circ} \mathrm{C} \sim 8^{\circ} \mathrm{C}$ 冰箱内保存，有效期 $7 \mathrm{~d}$ 。保存数星期以上的菌种，不能立即用作制备接种液之用，一定要在使用前每天移种一次，连续 $2 \mathrm{~d} \sim 3 \mathrm{~d}$ ，方可使用，否则生长不好。  
12.1.4 接种菌悬液制备：在维生素 $\mathrm{B}_{6}$ 测定实验前一天，将周储备菌种转接于 $10\mathrm{mLYM}$ 肉汤培养基（种子培养液）中，可同时制备2管，于 $30^{\circ}C$ 振荡培养 $20\mathrm{h}\sim 24\mathrm{h}$ ，得到测定用的种子培养液，从月储备菌种到种子培养液总代数不超过5代。将该种子培养液于 $3000\mathrm{r / min}$ 下离心 $10\mathrm{min}$ ，倾去上清液；用 $10~\mathrm{mL}$ 生理盐水洗涤，离心，倾去上清液，用生理盐水重复洗涤2次；再加 $10~\mathrm{mL}$ 消毒过的生理盐水，将离心管置于涡旋混匀器上充分混合，使菌种成为混悬液，将此菌悬液倒入已消毒的注射器内，立即使用。

# 12.2 试样处理

12.2.1 称取试样 $0.5 \mathrm{~g} \sim 10 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ , 其中维生素 $\mathrm{B}_{6}$ 含量不超过 $10 \mathrm{ng}$ ) 放入 $100 \mathrm{~mL}$ 锥形瓶中, 加 $72 \mathrm{~mL} 0.22 \mathrm{~mol} / \mathrm{L}$ 硫酸溶液。放入高压釜 $121^{\circ} \mathrm{C}$ 下水解 $5 \mathrm{~h}$ , 取出冷却, 用 $10.0 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液和 $0.5 \mathrm{~mol} / \mathrm{L}$ 硫酸溶液调 $\mathrm{pH}$ 至 4.5, 用溴甲酚绿做指示剂 (指示剂由黄-黄绿色), 将锥形瓶内的溶液转移到 $100 \mathrm{~mL}$ 容量瓶中, 用蒸馏水定容至 $100 \mathrm{~mL}$ , 滤纸过滤, 保存滤液于冰箱内备用 (有效期不超过 $36 \mathrm{~h}$ )。

注：整个试样处理过程需要注意避光操作。

12.2.2 标准曲线的制备：3组试管各加 $0.00\mathrm{mL}, 0.02\mathrm{mL}, 0.04\mathrm{mL}, 0.08\mathrm{mL}, 0.12\mathrm{mL}$ 和 $0.16\mathrm{mL}$ 吡哆醇工作液，再加吡哆醇Y培养基补至 $5.00\mathrm{mL}$ ，混匀，加棉塞。  
12.2.3 试样管的制备：在试管中分别加入 $0.05\mathrm{mL},0.10\mathrm{mL},0.20\mathrm{mL}$ 样液，再加入吡哆醇Y培养基补至 $5.00~\mathrm{mL}$ ，用棉塞塞住试管，将制备好的标准曲线和试样测定管放入高压釜 $121^{\circ}C$ 下高压灭菌 $10\mathrm{min}$ ，冷至室温备用。  
12.2.4 接种和培养：每管种一滴接种液，于 $30^{\circ}\mathrm{C} \pm 0.5^{\circ}\mathrm{C}$ 恒温箱中培养 $18\mathrm{h} \sim 22\mathrm{h}$ 。

# 12.3 测定

将培养后的标准管和试样管从恒温箱中取出后，用分光光度计于 $550\mathrm{nm}$ 波长下，以标准管的零管调零，测定各管的吸光度值。以标准管维生素 $\mathrm{B}_6$ 所含的浓度为横坐标，吸光度值为纵坐标，绘制维生素 $\mathrm{B}_6$ 标准工作曲线，用试样管得到的吸光度值，在标准曲线上查到试样管维生素 $\mathrm{B}_6$ 的含量。

# 13 分析结果的表述

试样提取液中维生素 $\mathrm{B}_6$ 的浓度按式(3)计算：

$$
\rho = \frac {\rho_ {1} + \rho_ {2} + \rho_ {3}}{3} \tag {3}
$$

式中：

$\rho$ ——试样提取液中维生素 $\mathrm{B}_{6}$ 的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$\rho_{i}$ ——各试样测定管中维生素 $\mathrm{B}_{6}$ 的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ 。

试样中维生素 $\mathrm{B}_{6}$ 的含量按式(4)计算：

$$
X = \frac {\rho \times V \times 1 0 0}{m \times 1 0 ^ {6}} \tag {4}
$$

式中：

$X$ ——试样中维生素 $\mathrm{B}_{6}$ （以吡哆醇计）的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$

$\rho$ ——试样提取液中维生素 $\mathrm{B}_6$ 的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样提取液的定容体积与稀释体积总和，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$\frac{100}{10^6}$ ——折算成每 $100\mathrm{g}$ 试样中维生素 $\mathrm{B}_6$ 的毫克数。

计算结果保留到小数点后两位。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

当取样量为 $1.00\mathrm{g}$ 时，定量限为 $0.002\mathrm{mg} / 100\mathrm{g}$

# 食品添加剂 丁烷

# 1 范围

本标准适用于由天然气经分馏后用油吸收或冷冻制得的食品添加剂丁烷。

# 2 分子式、结构式和相对分子质量

# 2.1 分子式

$\mathrm{C_4H_{10}}$

# 2.2 结构式

$$
\mathrm {C H} _ {3} - \mathrm {C H} _ {2} - \mathrm {C H} _ {2} - \mathrm {C H} _ {3}
$$

# 2.3 相对分子质量

58.12（按2007年国际相对原子质量）

# 检验方法

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682中规定的三级水。

试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 红外光谱的测定

用红外吸收分光光度法，将试样谱图与对照谱图(见附录B)比较，两者应基本一致。

# A.2.2 蒸气压的测定

按照GB/T22024—2008中4.6的方法进行测定。试样在 $21^{\circ}\mathrm{C}$ 的蒸气压(绝对压力)应为 $213\mathrm{kPa}$ 。

# A.3 硫化物的测定

将试样钢瓶阀打开，调节形成适当流量的气流，不要将气流直对面部，将反射部分气流对着鼻子，嗅其味。试样无硫化物的特异性气味即为通过试验。

# A.4 残渣酸度的测定

取高沸点残渣测定后试样，加 $10~\mathrm{mL}$ 水溶解，加2滴 $\sim 3$ 滴甲基橙指示液 $(1\mathrm{g / L})$ ，无粉红色或红色产生即为通过试验。

# 食品添加剂 聚氧丙烯氧化乙烯甘油醚

# 1 范围

本标准适用于以环氧乙烷、环氧丙烷、丙三醇为原料，经聚合、中和、精制而成的食品添加剂聚氧丙烯氧化乙烯甘油醚。

$n_1, n_2, n_3$ ：环氧丙烷聚合度， $n_1 + n_2 + n_3 > 3$ 。

# 检验方法

# A.1 一般规定

本标准所用试剂和水在未注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品在未注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

取适量试样，按照GB/T6040测定红外吸收光谱，应在 $2870~\mathrm{cm}^{-1}$ （ $\pm 10~\mathrm{cm}^{-1}$ )及 $1110~\mathrm{cm}^{-1}$ $(\pm 10~\mathrm{cm}^{-1})$ 波数处显示吸收峰。

# A.3 环氧乙烷及环氧丙烷的测定

# A.3.1 方法提要

试样用N-甲基-2-吡咯烷酮(NMP)按 $1:10$ 比例稀释，缓和试样对氧化物响应因子的基体效应。将一部分稀释液顶空注射到气相色谱仪里，分离氧化物，采用火焰离子化检测器进行检测。以溶解在NMP中的环氧乙烷(EO)或环氧丙烷(PO)为标准物确定响应因子，根据峰面积采用外标法定量。

# A.3.2 试剂和材料

A.3.2.1 干冰。

A.3.2.2 N-甲基-2-吡咯烷酮(NMP)。  
A.3.2.3 环氧乙烷(EO)标准品：纯度 $\geqslant 99\%$   
A.3.2.4 环氧丙烷(PO)标准品：纯度 $\geq 99\%$

# A.3.3 仪器和设备

气相色谱仪：配备火焰离子化检测器，或其他等效的检测器和顶空进样器。

# A.3.4 参考色谱条件

A.3.4.1 色谱柱：石英毛细管柱 $(\phi 0.32\mathrm{mm}\times 60\mathrm{m})$ ，涂层为 $14\%$ 氰丙基苯基-二甲基聚硅氧烷，厚度为 $1.5\mu \mathrm{m}$ ，或同等性能的色谱柱。  
A.3.4.2 柱温： $35\%$ 下保持 $12\mathrm{min}$ ，以 $5^{\circ}C / \mathrm{min}$ 升温至 $105~^\circ C$ ，再以 $35~^\circ C / \mathrm{min}$ 升温至 $240~^\circ C$ ，保持 $10\mathrm{min}$ 。  
A.3.4.3 进样口温度： $200^{\circ}C$   
A.3.4.4 检测器温度： $275~\mathrm{^\circ C}$   
A.3.4.5 检测器载气：氦气。  
A.3.4.6 载气流速： $5\mathrm{mL / min}$ 下保持 $1\mathrm{min}$ ，以 $40~\mathrm{mL / min}$ 降速至 $2\mathrm{mL} / \mathrm{min}$ ，保持 $39\mathrm{min}$   
A.3.4.7 氢气流速： $40~\mathrm{mL / min}$

A.3.4.8 空气流速： $450~\mathrm{mL / min}$   
A.3.4.9 分流比： $6:1$

# A.3.5 参考顶空进样条件

A.3.5.1 定量环规格： $3\mathrm{mL}$   
A.3.5.2 顶空瓶压力： $107\mathrm{kPa}$   
A.3.5.3 载气压力： $207\mathrm{kPa}$   
A.3.5.4 炉温： $100^{\circ}C$   
A.3.5.5 定量环温度： $200^{\circ}C$ □  
A.3.5.6 传输线温度： $200^{\circ}C$ □  
A.3.5.7 循环时间： $45\mathrm{min}$   
A.3.5.8 进样时间： $0.5\mathrm{min}$   
A.3.5.9 定量环平衡时间： $0.15\mathrm{min}$   
A.3.5.10 定量环填充时间： $0.10\mathrm{min}$   
A.3.5.11 顶空瓶平衡时间： $30\mathrm{min}$   
A.3.5.12 顶空瓶加压时间： $0.2\mathrm{min}$

# A.3.6 分析步骤

# A.3.6.1 环氧乙烷标准储备液的制备

取 $40~\mathrm{mL}$ 玻璃小瓶，去皮重，加入 $25.0\mathrm{g}\pm 0.5\mathrm{gN}$ -甲基-2-吡咯烷酮，封盖，称量。使用经干冰冷却的 $1\mathrm{mL}$ 注射器抽取 $0.8\mathrm{mL}$ 同样冷却(约 $5\mathrm{min}$ )的环氧乙烷标准品。通过刺穿隔片注入 $40~\mathrm{mL}$ 小瓶中，完全混合（混合振摇时不要让混合物触及瓶盖内表面)，冷却，称量，计算出加入环氧乙烷标准品的质量。取下 $40~\mathrm{mL}$ 小瓶的瓶盖，加入适量N-甲基-2-吡咯烷酮，使液面距离小瓶顶端约 $1\mathrm{cm}$ 处，使用另一未刺穿并称量的新瓶盖封盖，回暖至室温，擦拭干瓶外的凝结水，得到环氧乙烷标准储备液。对封盖的 $40~\mathrm{mL}$ 小瓶和弃用的瓶盖进行称量，计算出加入N-甲基-2-吡咯烷酮的质量。

# A.3.6.2 环氧丙烷标准储备液的制备

取 $40~\mathrm{mL}$ 玻璃小瓶，去皮重，加入 $25.0\mathrm{g}\pm 0.5\mathrm{gN}$ -甲基-2-吡咯烷酮，封盖，称量。使用经干冰冷却的 $1\mathrm{mL}$ 注射器抽取 $0.8\mathrm{mL}$ 环氧丙烷标准品，通过刺穿隔片注入 $40~\mathrm{mL}$ 小瓶中，完全混合（混合振摇时不要让混合物触及瓶盖内表面），冷却，称量，计算出加入环氧丙烷标准品的质量。取下 $40~\mathrm{mL}$ 小瓶的瓶盖，加入适量N-甲基-2-吡咯烷酮，使液面距离小瓶顶端约 $1\mathrm{cm}$ 处，使用另一未刺穿并称量的新瓶盖封盖，回暖至室温，擦拭干瓶外的凝结水，得到环氧丙烷标准储备液。对封盖的 $40~\mathrm{mL}$ 小瓶和弃用的瓶盖进行称量，计算出加入N-甲基-2-吡咯烷酮的质量。

# A.3.6.3 环氧乙烷或环氧丙烷标准溶液的制备

取 $40~\mathrm{mL}$ 玻璃小瓶，去皮重，加入 $25.0\mathrm{g}\pm 0.5\mathrm{gN}$ -甲基-2-吡咯烷酮，封盖，称量。使用经干冰冷却的 $1\mathrm{mL}$ 注射器抽取 $0.1\mathrm{mL}$ 同样冷却的环氧乙烷或环氧丙烷标准储备液加入，冷却，称重，计算出加入环氧乙烷或环氧丙烷标准储备液的质量。取下瓶盖，加入适量N-甲基-2-吡咯烷酮，使液面距离小瓶顶端约 $1\mathrm{cm}$ 处，使用另一未刺穿并称量的新瓶盖封盖，回暖至室温，擦拭干瓶外的凝结水，得到环氧乙烷或环氧丙烷标准溶液。对封盖的小瓶和弃用的瓶盖进行称量，计算出加入N-甲基-2-吡咯烷酮的质量。

# A.3.6.4 顶空瓶中校正标准液的制备

取 $20~\mathrm{mL}$ 顶空瓶，注入 $0.9\mathrm{mL}$ N-甲基-2-吡咯烷酮，封盖，去皮重。使用经干冰冷却的 $1\mathrm{mL}$ 注射

器抽取 $0.1 \mathrm{~mL}$ 同样冷却的环氧乙烷或环氧丙烷标准溶液加入，得到顶空瓶中校正标准液，称量，计算出校正标准液的质量。重复以上步骤4次，共制得5份校正标准液。

# A.3.6.5 试样液的制备

取 $20~\mathrm{mL}$ 顶空瓶，注入 $0.9\mathrm{mL}$ N-甲基-2-吡咯烷酮，封盖，去皮重。使用经干冰冷却的 $1\mathrm{mL}$ 注射器抽取 $0.1\mathrm{mL}$ 试样加入，得到试样液，称量，计算出加入试样的质量。

# A.3.6.6 测定

将5份校正标准液和试样液放入顶空进样盘中，在A.3.4和A.3.5参考操作条件下，对每个顶空瓶进行色谱分析。环氧乙烷的相对保留时间约为 $6.4\mathrm{min}$ ，环氧丙烷的相对保留时间约为 $9.2\mathrm{min}$ （见B.1）。

# A.3.7 结果计算

# A.3.7.1 校正系数

校正系数 $R_{f}$ 按式(A.1)计算：

$$
R _ {f} = \frac {m _ {0} \times m _ {1}}{m _ {2} \times m _ {3}} \times \frac {m _ {\mathrm {s}}}{A _ {\mathrm {s}}} \tag {A.1}
$$

式中：

$m_0$ ——环氧乙烷或环氧丙烷标准品的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——标准溶液中加入标准储备液的质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——标准储备液中加入N-甲基-2-吡咯烷酮的质量，单位为克 $(\mathrm{g})$ ；

$m_{3}$ ——标准溶液中加入N-甲基-2-吡咯烷酮的质量，单位为克(g)；

$m_{\mathrm{s}}$ ——校正标准液的质量，单位为克 $(\mathrm{g})$ ；

$A_{\mathrm{s}}$ ——校正标准液中环氧乙烷或环氧丙烷标准品的峰面积。

# A.3.7.2 环氧乙烷或环氧丙烷

环氧乙烷或环氧丙烷的含量 $w_{i}$ 以毫克每千克 $(\mathrm{mg / kg})$ 计，按式(A.2)计算：

$$
w _ {i} = \frac {R _ {f} \times A _ {i}}{m _ {i}} \times 1 0 ^ {6} \tag {A.2}
$$

式中：

$R_{f}$ ——校正系数；

$A_{i}$ ——试样液中环氧乙烷或环氧丙烷的峰面积；

$m_{i}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$10^{6}$ ——质量换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得环氧乙烷的2次独立测定结果的绝对差值不大于算术平均值的 $12.1\%$ ，在重复性条件下获得环氧丙烷的2次独立测定结果的绝对差值不大于算术平均值的 $7.3\%$ 。

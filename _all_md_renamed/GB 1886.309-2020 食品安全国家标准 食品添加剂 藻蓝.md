# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有标明其他要求时，均指分析纯试剂和符合GB/T6682规定的三级水。试验中所用溶液在未注明何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 溶解性

易溶于水，呈蓝色澄清液，不溶于无水乙醇、乙醚、石油醚等有机溶剂。称取藻蓝试样适量（粉末产品 $0.05\mathrm{g}$ ，液体产品 $0.3\mathrm{g}$ ），溶于 $10\mathrm{mL}$ 水中，缓慢加入 $4.5\mathrm{g}$ 硫酸铵固体粉末，混匀至完全溶解，静置生成蓝色沉淀。

# A.2.2 稳定性

用磷酸盐缓冲液 $(\mathrm{pH}6.8)$ 配制的 $0.5\%$ 试样液，应呈蓝色，并伴随着红色荧光发生，经 $90^{\circ}\mathrm{C}$ 加热处理 $30\mathrm{min}$ 后，荧光消失。

# A.2.3 最大吸收波长

# A.2.3.1 试剂和材料

磷酸盐缓冲液： $0.1\mathrm{mol / L},\mathrm{pH}6.8$ 。按下列方法进行配制：

a) A 液 $(0.2\mathrm{mol} / \mathrm{L}$ 磷酸二氢钠水溶液): $\mathrm{NaH_2PO_4} \cdot 2\mathrm{H_2O}31.20\mathrm{g}$ , 加水溶解并定容至 $1000\mathrm{mL}$ ;  
b）B液 $(0.2\mathrm{mol} / \mathrm{L}$ 磷酸氢二钠水溶液）： $\mathrm{Na_2HPO_4\cdot 7H_2O53.61g}$ （或 $\mathrm{Na_2HPO_4\cdot 12H_2O}$ $71.63\mathrm{g}$ 或 $\mathrm{Na_2HPO_4\cdot 2H_2O35.61g}$ ，加水溶解并定容至 $1000~\mathrm{mL}$   
c) 取 $51.0 \mathrm{~mL}$ A 液与 $49.0 \mathrm{~mL}$ B 液混合摇匀, 加入水定容至 $200 \mathrm{~mL}$ , 得到 $0.1 \mathrm{~mol} / \mathrm{L}$ 磷酸盐缓冲液 $(\mathrm{pH} 6.8)$ 。

# A.2.3.2 仪器和设备

紫外可见光分光光度计。

# A.2.3.3 分析步骤

称取试样适量（粉末产品 $0.03\mathrm{g}$ ，液体产品 $0.2\mathrm{g}$ ），以磷酸盐缓冲液溶解并定容于 $100~\mathrm{mL}$ 容量瓶中，充分溶解或用磷酸盐缓冲液适当稀释后，使吸光值在 $0.3\sim 0.7$ （检测范围），用分光光度计扫描测定，该试样溶液在波长 $(620\pm 5)\mathrm{nm}$ 有最大吸收峰(参见附录B)。

# A.3 色价的测定

# A.3.1 试剂和材料

磷酸盐缓冲液：pH6.8。

# A.3.2 仪器设备

紫外可见光分光光度计。

# A.3.3 分析步骤

称取试样适量（粉末产品 $0.03\mathrm{g}\sim 0.1\mathrm{g}$ ，液体产品 $0.2\mathrm{g}\sim 0.6\mathrm{g}$ )，精确至 $0.0001\mathrm{g}$ ，以磷酸盐缓冲液 $\left(\mathrm{pH}6.8\right)$ 溶解，并定容于 $100~\mathrm{mL}$ 容量瓶中，摇匀。若溶液颜色太深，可用磷酸盐缓冲液适当稀释并定容，使溶液吸光度达到 $0.3\sim 0.7$ ，此溶液为待测试样液。以磷酸盐缓冲液作空白对照，于 $1\mathrm{cm}$ 比色皿，在试样最大吸收波长 $(620\pm 5)\mathrm{nm}$ 处测定试样液的吸光度 $(A)$ 。

# A.3.4 结果计算

色价 $E_{1\mathrm{cm}}^{1\%}(620 \pm 5)\mathrm{nm}$ 按式(A.1)计算：

$$
E _ {1 \mathrm {cm}} ^ {1 \%} (620 \pm 5) \mathrm {nm} = \frac {A \times f}{m} \quad \dots \tag{A.1}
$$

式中：

$A$ ——待测试样液的吸光度读数，要求在 $0.3\sim 0.7$

$f$ ——称取试样定容 $100\mathrm{mL}$ 之后的稀释倍数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

测定结果以平行测定结果的算术平均值为准。

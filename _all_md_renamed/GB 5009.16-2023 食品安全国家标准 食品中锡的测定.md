# 前言

本标准代替GB5009.16—2014《食品安全国家标准 食品中锡的测定》。

本标准与GB5009.16—2014相比，主要变化如下：

——增加了电感耦合等离子体质谱法为第二法；  
——增加了电感耦合等离子体发射光谱法为第三法；  
——删除了苯芬酮比色法；  
——修改了第一法氢化物原子荧光光谱法的适用范围，增加了方法检出限。

# 食品安全国家标准

# 食品中锡的测定

# 1 范围

本标准规定了食品中锡的氢化物原子荧光光谱、电感耦合等离子体质谱和电感耦合等离子体发射光谱测定方法。

本标准适用于食品中锡的测定。

# 第一法 氢化物原子荧光光谱法

# 2 原理

试样经消解后，在硼氢化钠(或硼氢化钾)的作用下生成锡的氢化物 $(\mathrm{SnH}_4)$ ，并由载气带入原子化器中进行原子化，在锡空心阴极灯的照射下，基态锡原子被激发至高能态，在去活化回到基态时，发射出特征波长的荧光，其荧光强度与锡含量成正比，与标准系列溶液比较定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的二级水。

# 3.1 试剂

3.1.1 硫酸 $\mathrm{(H_2SO_4)}$ ：优级纯。  
3.1.2 硝酸 $\mathrm{(HNO_3)}$ ：优级纯。  
3.1.3 高氯酸 $\mathrm{(HClO_4)}$ ：优级纯。  
3.1.4 硫脲 $\mathrm{(CH_4N_2S)}$ 。  
3.1.5 抗坏血酸 $\mathrm{(C_6H_8O_6)}$ 。  
3.1.6 硼氢化钠 $\mathrm{(NaBH_4)}$ 或硼氢化钾（ $\mathrm{KBH_4}$ ）。  
3.1.7 氢氧化钠(NaOH)或氢氧化钾(KOH)。

# 3.2 试剂配制

3.2.1 硝酸-高氯酸混合溶液 $(4 + 1)$ ：量取 $400~\mathrm{mL}$ 硝酸和 $100~\mathrm{mL}$ 高氯酸，混匀。  
3.2.2 硫酸溶液 $(1 + 9)$ ：量取 $100~\mathrm{mL}$ 硫酸，缓慢加入 $900~\mathrm{mL}$ 水中，混匀。  
3.2.3 硫脲+抗坏血酸溶液：称取 $15.0\mathrm{g}$ 硫脲和 $15.0\mathrm{g}$ 抗坏血酸，用水溶解并稀释至 $100\mathrm{mL}$ ，混匀。临用现配。  
3.2.4 氢氧化钠溶液 $(5\mathrm{g} / \mathrm{L})$ ：称取 $5.0\mathrm{g}$ 氢氧化钠，用水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.5 硼氢化钠溶液 $(7\mathrm{g} / \mathrm{L})$ ：称取 $7.0\mathrm{g}$ 硼氢化钠，用氢氧化钠溶液 $(5\mathrm{g} / \mathrm{L})$ 溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。临用现配。

注：本方法也可用硼氢化钾作为还原剂(用氢氧化钾溶液溶解配制)，并可根据仪器的灵敏度调整还原剂中硼氢化钠或硼氢化钾的浓度 $(7\mathrm{g / L}\sim 20\mathrm{g / L})$ 。

# 3.3 标准品

金属锡(Sn)标准品：纯度为 $\geq 99.99\%$ 。或经国家认证并授予标准物质证书的一定浓度的锡标准溶液。

# 3.4 标准溶液配制

3.4.1 锡标准溶液 $(1.00\mathrm{mg / mL})$ ：准确称取 $0.1000\mathrm{g}$ 金属锡标准品，置于小烧杯中，加入 $10.0~\mathrm{mL}$ 硫酸，盖以表面皿，加热至锡完全溶解，移去表面皿，继续加热至出现浓白烟，冷却，慢慢加入 $50~\mathrm{mL}$ 水，移入 $100~\mathrm{mL}$ 容量瓶中，用硫酸溶液 $(1 + 9)$ 多次洗涤烧杯，洗液并入容量瓶中，并稀释至刻度，混匀。  
3.4.2 锡标准中间液 $(10.0\mathrm{mg / L})$ ：准确吸取锡标准溶液 $(1.00\mathrm{mg / mL})1.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用硫酸溶液 $(1 + 9)$ 定容至刻度，混匀。于 $0^{\circ}\mathrm{C}\sim 5^{\circ}\mathrm{C}$ 保存，有效期4周。  
3.4.3 锡标准使用液 $(1.00\mathrm{mg / L})$ ：准确吸取锡标准中间液 $(10.0\mathrm{mg / L})10.0\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用硫酸溶液 $(1 + 9)$ 定容至刻度，混匀。于 $0^{\circ}\mathrm{C}\sim 5^{\circ}\mathrm{C}$ 保存，有效期4周。  
3.4.4 锡标准系列溶液：分别吸取锡标准使用液 $(1.00\mathrm{mg / L})0\mathrm{mL},0.500\mathrm{mL},2.00\mathrm{mL},3.00\mathrm{mL},4.00\mathrm{mL}$ 、 $5.00~\mathrm{mL}$ 于 $25~\mathrm{mL}$ 容量瓶中，分别加入硫酸溶液 $(1 + 9)5.00~\mathrm{mL},4.50~\mathrm{mL},3.00~\mathrm{mL},2.00~\mathrm{mL},1.00~\mathrm{mL}$ 、 $0.00~\mathrm{mL}$ ，加入硫脲+抗坏血酸溶液 $2.0~\mathrm{mL}$ ，再用水定容至 $25~\mathrm{mL}$ 。此锡标准系列溶液的质量浓度分别为 $0\mu \mathrm{g / L},20.0\mu \mathrm{g / L},80.0\mu \mathrm{g / L},120\mu \mathrm{g / L},160\mu \mathrm{g / L},200\mu \mathrm{g / L}$ 。临用现配。

注：应确保每份标准系列溶液中硫酸的体积分数为 $2\%$ ，在此前提下，可根据仪器的灵敏度及样品中锡的实际含量确定标准系列溶液中锡元素的质量浓度范围。

# 4 仪器和设备

4.1 原子荧光光谱仪：配锡空心阴极灯。  
4.2 可调式控温电热板：控温范围为 $90^{\circ}\mathrm{C} \sim 250^{\circ}\mathrm{C}$   
4.3 石墨消解装置：控温范围为 $90^{\circ}\mathrm{C} \sim 250^{\circ}\mathrm{C}$ ，配玻璃消解管（容量 $\geqslant 50~\mathrm{mL}$ ）。  
4.4 电子天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.5 样品粉碎设备：匀浆机、高速粉碎机。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固态样品

# 5.1.1.1 干样

豆类、谷物、菌类、茶叶、干制水果、焙烤食品等低含水量样品，取可食部分，必要时经高速粉碎机粉碎均匀；固体乳制品、蛋白粉、面粉等呈均匀状的粉状样品，摇匀。

# 5.1.1.2 鲜样

蔬菜、水果、水产品等高含水量样品，必要时洗净晾干，取可食部分匀浆均匀；肉类、蛋类等样品，取可食部分匀浆均匀。

# 5.1.1.3 速冻食品

解冻后，取可食部分匀浆均匀。

# 5.1.1.4 罐头食品

全量取可食用内容物(包括可食用液体部分)制成匀浆或者均匀粉末。

# 5.1.2 液态样品

软饮料、液态调味品等样品，摇匀。

# 5.1.3 半固态样品

搅拌均匀。

# 5.2 试样消解

5.2.1 称取试样 $1\mathrm{g}\sim 5\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 或准确移取液体试样 $1.00\mathrm{mL}\sim 5.00\mathrm{mL}$ 于玻璃消解器皿中，含乙醇或二氧化碳的试样于 $90^{\circ}\mathrm{C}$ 加热 $30\mathrm{min}$ 除去乙醇或二氧化碳。加入 $10.0\mathrm{mL}$ 硝酸-高氯酸混合溶液 $(4 + 1)$ ，加入 $1.0\mathrm{mL}$ 硫酸，于电热板上或石墨消解装置上消解。消解过程中消解液若变棕黑色，应立即取下放冷，适当补加硝酸或硝酸-高氯酸混合溶液 $(4 + 1)$ ，再继续消解。当消解液冒白烟并呈无色透明或略带黄色时，继续加热至剩余体积为 $1\mathrm{mL}$ 左右，取下冷却，用水定容至 $50\mathrm{mL}$ ，混匀备用，得到试样溶液。根据试样溶液中锡的含量，需要时将试样溶液用水稀释至适当浓度，并在稀释过程中补加一定量的硫酸溶液 $(1 + 9)$ ，使最终试样溶液的硫酸浓度与标准系列溶液相同。同时做空白试验。

5.2.2 分别准确移取 $10.0 \mathrm{~mL}$ 空白溶液和试样溶液，置于 $25 \mathrm{~mL}$ 容量瓶中，加入 $3.0 \mathrm{~mL}$ 硫酸溶液 $(1 + 9)$ ，加入 $2.0 \mathrm{~mL}$ 硫脲+抗坏血酸溶液，再用水定容至 $25 \mathrm{~mL}$ ，混匀，得到空白测定液和试样测定液。

# 5.3 仪器参考条件

负高压 $380\mathrm{V}$ ；灯电流 $70~\mathrm{mA}$ ；原子化温度 $850^{\circ}C$ ；屏蔽气流量 $1200~\mathrm{mL / min}$ ；载气流量 $500~\mathrm{mL / min}$ ；测量方式为标准曲线法；读数方式为峰面积；延迟时间 $1\mathrm{s}$ ；读数时间 $15\mathrm{s}$ ；加液时间 $8\mathrm{s}$ 进样体积 $2.0~\mathrm{mL}$ 。

# 5.4 标准曲线的制作

仪器预热稳定后，将标准系列溶液注入原子荧光光谱仪中，测定锡的荧光强度，以标准系列溶液中锡的质量浓度为横坐标、荧光强度为纵坐标，绘制标准曲线。

# 5.5 试样溶液的测定

将空白测定液和试样测定液分别注入原子荧光光谱仪中，测定锡的荧光强度，根据标准曲线得到溶液中锡的质量浓度。

# 6 分析结果的表述

试样中锡的含量按式(1)进行计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times f \times 2 5}{m \times 1 0 . 0 \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中锡的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 或毫克每升 $(\mathrm{mg / L})$ ；

$\rho$ ——试样测定液中锡的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白测定液中锡的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样溶液稀释倍数；

25 ——试样测定液定容体积，单位为毫升 $(\mathrm{mL})$

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

10.0 ——试样测定液所移取的试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当取样量为 $1\mathrm{g}$ （或 $1\mathrm{mL}$ ），试样溶液定容体积为 $50~\mathrm{mL}$ ，方法的检出限为 $0.8\mathrm{mg / kg}$ （或 $0.8\mathrm{mg / L})$ ，方法的定量限为 $2.5\mathrm{mg / kg}$ 或 $2.5\mathrm{mg / L})$ 。

# 第二法 电感耦合等离子体质谱法

# 9 原理

试样经硝酸 + 盐酸消解, 其中的锡在盐酸的作用下溶解并维持稳定, 由电感耦合等离子体质谱仪测定, 以锡元素特定质量数 (质荷比, $m / z$ ) 定性, 以锡元素和内标元素质谱信号响应值的比值与锡元素的浓度成正比进行定量分析。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 硝酸 $\mathrm{(HNO_3)}$ ：优级纯或更高纯度。  
10.1.2 盐酸（HCl）：优级纯或更高纯度。  
10.1.3 氩气（Ar）：氩气 $(\geq 99.995\%)$ 或液氩。  
10.1.4 氦气（He）：氦气 $(\geq 99.995\%)$ 。

# 10.2 试剂配制

10.2.1 盐酸溶液 $(5 + 95)$ ：量取 $50~\mathrm{mL}$ 盐酸，缓慢加入 $950~\mathrm{mL}$ 水中，混匀。  
10.2.2 硝酸-盐酸混合溶液 $(5 + 1 + 94)$ ：量取 $50~\mathrm{mL}$ 硝酸，缓慢加入 $940~\mathrm{mL}$ 水中，混匀，再缓慢加入 $10~\mathrm{mL}$ 盐酸，混匀。

# 10.3 标准品

10.3.1 锡标准溶液 $(1000\mathrm{mg / L})$ ：采用经国家认证并授予标准物质证书的锡标准溶液。

10.3.2 内标元素贮备液 $(1000\mathrm{mg / L})$ ：采用经国家认证并授予标准物质证书的铑(Rh)、铼(Re)等任意一种单元素或多元素内标标准贮备液。

# 10.4 标准溶液配制

10.4.1 锡标准中间液 $(10.0\mathrm{mg / L})$ ：准确吸取锡标准溶液 $(1000\mathrm{mg / L})1.00\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用盐酸溶液 $(5 + 95)$ 定容至刻度，混匀。于 $0^{\circ}\mathrm{C}\sim 5^{\circ}\mathrm{C}$ 保存，有效期4周。  
10.4.2 锡标准使用液 $(1.00\mathrm{mg / L})$ ：准确吸取锡标准中间液 $(10.0\mathrm{mg / L})10.0\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用盐酸溶液 $(5 + 95)$ 定容至刻度，混匀。于 $0^{\circ}\mathrm{C}\sim 5^{\circ}\mathrm{C}$ 保存，有效期4周。  
10.4.3 锡标准系列溶液：分别吸取锡标准使用液 $(1.00\mathrm{mg / L})0\mathrm{mL},0.250\mathrm{mL},0.500\mathrm{mL},1.00\mathrm{mL}$ 、 $2.00~\mathrm{mL}$ 和 $5.00~\mathrm{mL}$ 容量瓶中，用硝酸-盐酸混合溶液 $(5 + 1 + 94)$ 定容至刻度，混匀。此锡标准系列溶液的质量浓度分别为 $0\mu \mathrm{g} / \mathrm{L},2.50\mu \mathrm{g} / \mathrm{L},5.00\mu \mathrm{g} / \mathrm{L},10.0\mu \mathrm{g} / \mathrm{L},20.0\mu \mathrm{g} / \mathrm{L}$ 和 $50.0\mu \mathrm{g} / \mathrm{L}$ 。临用现配。

注：可根据仪器的灵敏度及样品中锡的实际含量确定标准系列溶液中锡元素的质量浓度范围。

10.4.4 内标使用液：吸取适量内标元素贮备液，用硝酸-盐酸混合溶液 $(5 + 1 + 94)$ 逐级稀释配成适当浓度的内标使用液。内标溶液可采用手动定量加入标准系列及样品溶液中，也可由仪器在线加入，内标与样品溶液混合后，内标的参考质量浓度约为 $10\mu \mathrm{g} / \mathrm{L}\sim 100\mu \mathrm{g} / \mathrm{L}$ 。

# 11 仪器和设备

11.1 电感耦合等离子体质谱仪。  
11.2 分析天平：感量 $1\mathrm{mg}$   
11.3 可调式控温电热炉或电热板：控温范围为 $80^{\circ}\mathrm{C} \sim 100^{\circ}\mathrm{C}$ 。  
11.4 微波消解系统：配聚四氟乙烯消解内罐。  
11.5 样品粉碎设备：匀浆机、高速粉碎机。

# 12 分析步骤

# 12.1 试样制备

同5.1。

# 12.2 试样消解

称取试样 $0.2\mathrm{g}\sim 0.5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，含水分较多的试样可适当增加取样量至 $1\mathrm{g}$ 或准确移取液体试样 $0.500~\mathrm{mL}\sim 3.00~\mathrm{mL}$ 于微波消解内罐中，含乙醇或二氧化碳的试样于 $90^{\circ}C$ 加热 $30\mathrm{min}$ 除去乙醇或二氧化碳。加入 $5\mathrm{mL}\sim 7\mathrm{mL}$ 硝酸，室温下加盖放置过夜，加入 $1\mathrm{mL}$ 盐酸并尽快旋紧罐盖，按照微波消解仪标准操作步骤进行消解（消解参考条件见附录A表A.1）。冷却后取出，缓慢打开罐盖排气，用少量水冲洗内盖，将消解罐放在可调式控温电热炉或电热板上，于 $90^{\circ}C$ 加热 $30\mathrm{min}$ ，用水定容至 $50~\mathrm{mL}$ ，混匀备用。同时做空白试验。

# 12.3 仪器参考条件

锡的同位素选择 $^{118}\mathrm{Sn}$ ，内标元素选择 $^{103}\mathrm{Rh}$ 或 $^{185}\mathrm{Re}$ ；射频功率 $1550\mathrm{W}$ ；等离子体气流量 $15.0\mathrm{L/min}$ ；载气流量 $1.00\mathrm{L/min}$ ；辅助气流量 $0.90\mathrm{L/min}$ ；采样深度 $10\mathrm{mm}$ ；分析模式为普通模式或碰撞/反应池模式；重复2次～3次；冲洗时间 $60\mathrm{s}$ 。

注：锡的部分化合物容易水解形成不溶物残留在管路内，为了避免水解并减少冲洗时间，宜采用硝酸-盐酸混合溶液 $(5 + 1 + 94)$ 或盐酸溶液(体积分数 $1\% \sim 5\%$ )冲洗管路。

# 12.4 标准曲线的制作

将锡标准系列溶液分别注入电感耦合等离子体质谱仪中，测定锡元素和内标元素的信号响应值，以锡元素的质量浓度为横坐标，锡元素与内标元素信号响应值的比值为纵坐标，绘制标准曲线。

# 12.5 试样溶液的测定

将空白溶液和试样溶液分别注入电感耦合等离子体质谱仪中，测定锡元素和内标元素的信号响应值，根据标准曲线得到溶液中锡元素的质量浓度。如试样溶液中锡含量超出标准曲线范围，则用硝酸-盐酸混合溶液 $(5 + 1 + 94)$ 进行稀释。

注：如内标采用手动定量加入，试样稀释液应补加内标元素，使得试样稀释液中的内标元素浓度与标准系列溶液的内标元素浓度一致。

# 13 分析结果的表述

试样中锡的含量按式(2)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times f}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中锡的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 或毫克每升 $(\mathrm{mg / L})$ ；

$\rho$ ——试样溶液中锡的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白溶液中锡的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样溶液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样溶液稀释倍数；

$m$ ——试样称样量或移取体积，单位为克(g)或毫升(mL)；

1000——换算系数。

计算结果保留三位有效数字。

# 14 精密度

样品中锡含量大于 $1\mathrm{mg / kg}$ (或 $1\mathrm{mg / L}$ )时，在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ ；小于或等于 $1\mathrm{mg / kg}$ (或 $1\mathrm{mg / L}$ )时，在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 第三法 电感耦合等离子体发射光谱法

# 16 原理

试样经硝酸 + 盐酸消解, 其中的锡在盐酸的作用下溶解并维持稳定, 由电感耦合等离子体发射光谱

仪测定，以锡元素的特征谱线波长定性，以锡元素谱线强度响应值与元素浓度成正比进行定量分析。

# 17 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的一级水。

# 17.1 试剂

17.1.1 硝酸 $\mathrm{(HNO_3)}$ ：优级纯或更高纯度。  
17.1.2 盐酸（HCl）：优级纯或更高纯度。  
17.1.3 氩气（Ar）：氩气 $(\geqslant 99.995\%)$ 或液氩。

# 17.2 试剂配制

17.2.1 盐酸溶液 $(5 + 95)$ ：量取 $50~\mathrm{mL}$ 盐酸，缓慢加入 $950~\mathrm{mL}$ 水中，混匀。  
17.2.2 硝酸-盐酸混合溶液 $(5 + 1 + 94)$ ：量取 $50~\mathrm{mL}$ 硝酸，缓慢加入 $940~\mathrm{mL}$ 水中，混匀，再缓慢加入 $10~\mathrm{mL}$ 盐酸，混匀。

# 17.3 标准品

锡标准溶液 $(1000\mathrm{mg / L})$ ：采用经国家认证并授予标准物质证书的锡标准溶液。

# 17.4 标准溶液配制

17.4.1 锡标准使用液 $(10.0\mathrm{mg / L})$ : 准确吸取锡标准溶液 $(1000\mathrm{mg / L})1.00\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中,用盐酸溶液 $(5 + 95)$ 定容至刻度, 混匀。于 $0^{\circ}\mathrm{C} \sim 5^{\circ}\mathrm{C}$ 保存, 有效期4周。  
17.4.2 锡标准系列溶液：分别吸取锡标准使用液 $(10.0\mathrm{mg / L})0\mathrm{mL},0.250\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},5.00\mathrm{mL}$ 和 $10.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用硝酸-盐酸混合溶液 $(5 + 1 + 94)$ 定容至刻度，混匀。此锡标准系列溶液的质量浓度分别为 $0\mathrm{mg / L},0.0250\mathrm{mg / L},0.100\mathrm{mg / L},0.200\mathrm{mg / L},0.500\mathrm{mg / L}$ 和 $1.00\mathrm{mg / L}$ 。临用现配。

注：可根据仪器的灵敏度及样品中锡的实际含量确定标准系列溶液中锡元素的质量浓度范围。

18.2 分析天平：感量 $1\mathrm{mg}$   
18.3 可调式控温电热炉或电热板：控温范围为 $80^{\circ}\mathrm{C} \sim 100^{\circ}\mathrm{C}$ 。

# 19.1 试样制备

同5.1。

# 19.2 试样消解

同12.2。

# 19.3 仪器参考条件

锡的分析谱线波长选择 $189.92 \mathrm{~nm}$ ；观测方式为轴向；射频功率 $1.2 \mathrm{kW}$ ；等离子体气流量 $12 \mathrm{~L} / \mathrm{min}$ ；雾化气流量 $0.7 \mathrm{~L} / \mathrm{min}$ ；辅助气流量 $1 \mathrm{~L} / \mathrm{min}$ ；重复 2 次～3 次；冲洗时间 $30 \mathrm{~s}$ 。

注：锡的部分化合物容易水解形成不溶物残留在管路内，为了避免水解并减少冲洗时间，宜采用硝酸-盐酸混合溶液 $(5 + 1 + 94)$ 或盐酸溶液(体积分数 $1\% \sim 5\%$ )冲洗管路。

# 19.4 标准曲线的制作

将锡标准系列溶液注入电感耦合等离子体发射光谱仪中，测定锡元素分析谱线的强度响应值，以锡元素的质量浓度为横坐标、其分析谱线强度响应值为纵坐标，绘制标准曲线。

# 19.5 试样溶液的测定

将空白溶液和试样溶液分别注入电感耦合等离子体发射光谱仪中，测定锡元素分析谱线的强度响应值，根据标准曲线得到溶液中锡元素的质量浓度。如试样溶液中锡含量超出标准曲线范围，则用硝酸-盐酸混合溶液 $(5 + 1 + 94)$ 进行稀释。

# 20 分析结果的表述

试样中锡的含量按式(3)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times f}{m} \quad \dots \tag {3}
$$

式中：

$X$ ——试样中锡的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 或毫克每升 $(\mathrm{mg / L})$ ；

$\rho$ ——试样溶液中锡的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——空白溶液中锡的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样溶液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样溶液稀释倍数；

$m$ ——试样称样量或移取体积，单位为克(g)或毫升(mL)；

计算结果保留三位有效数字。

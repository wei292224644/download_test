# 食品中生物素的测定

2016-08-31 发布

2017-03-01 实施

# 前言

本标准代替GB5413.19—2010《婴幼儿食品和乳品中游离生物素的测定》。

本标准与GB5413.19—2010相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中生物素的测定”；  
——修改了标准曲线管和待测液管制备的表述方式；  
——修改了测定步骤表达方式；  
——增加了谷薯类、肉类、新鲜果蔬、藻类试样、蛋类、豆类、坚果类、内脏、强化生物素的食品处理过程；  
——删除了仪器和设备中通用玻璃器皿的描述。

# 食品安全国家标准

# 食品中生物素的测定

# 1 范围

本标准规定了食品中生物素的测定方法。

本标准适用于食品中生物素的测定。

# 2 原理

生物素是植物乳杆菌(Lactobacillus plantarum)生长所必需的营养素。在生物素测定培养基中，植物乳杆菌的生长与待测试样中生物素含量呈线性关系，根据透光率与标准工作曲线进行比较，即可计算出试样中待测物质的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 无水乙醇 $\mathrm{(C_2H_6O)}$   
3.1.2 氢氧化钠(NaOH)。  
3.1.3 盐酸（HCl）。  
3.1.4 柠檬酸盐。  
3.1.5 $\alpha$ -淀粉酶： $\geqslant 1.5 \mathrm{U} / \mathrm{mg}$ 。  
3.1.6 木瓜蛋白酶： $\geqslant 5\mathrm{U / mg}$   
3.1.7 硫酸 $\mathrm{(H_2SO_4)}$ 。

# 3.2 试剂配制

3.2.1 乙醇溶液 $(50\%)$ ：量取 $500~\mathrm{mL}$ 无水乙醇与 $500~\mathrm{mL}$ 水混匀。  
3.2.2 氢氧化钠溶液 $(0.5\mathrm{mol} / \mathrm{L})$ ：称取 $20\mathrm{g}$ 氢氧化钠，溶于 $1000~\mathrm{mL}$ 水中，混匀。  
3.2.3 氯化钠溶液 $(0.85\%)$ ：称取 $8.5\mathrm{g}$ 氯化钠，加水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.4 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：吸取 $83~\mathrm{mL}$ 盐酸，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.5 柠檬酸盐缓冲液 $(\mathrm{pH}4.5)$ ：称取 $1.5\mathrm{g}$ 柠檬酸至一个 $100~\mathrm{mL}$ 带磁力搅拌器的烧杯中，加入约 $50~\mathrm{mL}$ 蒸馏水至溶解，再加入 $12~\mathrm{mL}$ 的 $\mathrm{NaOH}(1\mathrm{mol / L})$ ，调节 $\mathrm{pH}$ 至4.5(用 $0.1\mathrm{mol / L}$ HCl)，将溶液转入 $100~\mathrm{mL}$ 容量瓶中，并用蒸馏水定容。该缓冲液可在 $2^{\circ}\mathrm{C}\sim 8^{\circ}\mathrm{C}$ 储存3d。  
3.2.6 蛋白酶-淀粉酶液：分别称取 $200\mathrm{mg}$ 木瓜蛋白酶和 $\alpha$ -淀粉酶，加入 $20~\mathrm{mL}$ 水研磨至匀浆， $3000\mathrm{r / min}$ 离心 $5\mathrm{min}\sim 10\mathrm{min}$ 。现用现配。  
3.2.7 硫酸溶液 $(3\%)$ ：量取 $30~\mathrm{mL}$ 硫酸加入到 $1000~\mathrm{mL}$ 水混匀。

# 3.3 标准品

生物素(d-Biotin或Vitamin H)标准品 $\mathrm{(C_{10}H_{16}N_2O_3S)}$ ：纯度 $\geqslant 99\%$ 。

# 3.4 标准溶液配制

3.4.1 生物素标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：精确称取 $100\mathrm{mg}$ 生物素标准品，用乙醇溶液 $(50\%)$ 溶解并转移至 $1000~\mathrm{mL}$ 容量瓶中，定容至刻度。储存于棕色瓶中，于 $2^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱保存12个月。  
3.4.2 生物素标准中间液 $(1.0\mu \mathrm{g} / \mathrm{mL})$ : 准确吸取 $1.00\mathrm{mL}$ 生物素标准储备液置于 $100\mathrm{mL}$ 棕色容量瓶中, 用乙醇溶液 $(50\%)$ 稀释并定容至刻度, 混匀后储存于瓶中 $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱保存6个月。  
3.4.3 生物素标准工作液 $(10\mathrm{ng / mL})$ ：准确吸取 $1.00~\mathrm{mL}$ 生物素标准中间液置于 $100~\mathrm{mL}$ 容量瓶中，用水稀释定容至刻度，混匀。临用前现配。  
3.4.4 标准使用工作液 $(10\mathrm{ng / mL})$ : 分两个浓度, 高浓度溶液的浓度为 $0.2\mathrm{ng / mL}$ ; 低浓度溶液的浓度为 $0.1\mathrm{ng / mL}$ 。从工作液中吸取两次各 $5\mathrm{mL}$ , 用水分别定容到 $250\mathrm{mL}$ 和 $500\mathrm{mL}$ 。

# 3.5 培养基

3.5.1 乳酸杆菌琼脂培养基：可按附录A配制。  
3.5.2 乳酸杆菌肉汤培养基：可按附录A配制。  
3.5.3 生物素测定用培养基：可按附录A配制。

注：一些商品化合成培养基效果良好，商品化合成培养基按标签说明进行配制。

# 4 仪器和设备

4.1 天平：感量 $0.1\mathrm{mg}$   
4.2 恒温培养箱： $37^{\circ}C\pm 1^{\circ}C$   
4.3 压力蒸汽消毒器： $121^{\circ}\mathrm{C}(0.10\mathrm{mPa}\sim 0.12\mathrm{mPa})$ 。  
4.4 淕涡振荡器。  
4.5 离心机：转速 $\geqslant 2000\mathrm{r / min}$   
4.6 分液器： $0\mathrm{mL}\sim 10\mathrm{mL}$   
4.7 可调式电炉。  
4.8 pH计：精度 $\pm 0.01$   
4.9 分光光度计。  
4.10 超净工作台。

注：玻璃仪器使用前，用活性剂(月桂磺酸钠或家用洗涤剂加入到洗涤用水中即可)对硬玻璃测定管及其他必要的玻璃器皿进行清洗，清洗之后 $200^{\circ}\mathrm{C}$ 干热 $2\mathrm{h}$ 。

# 5 菌种的制备与保存

# 5.1 菌种

植物乳杆菌 Lactobacillus plantarum (ATCC 8014)。

# 5.2 储备菌种的制备

5.2.1 将菌种植物乳杆菌 Lactobacillus plantarum (ATCC 8014) 转接至乳酸杆菌琼脂培养基中, 在

$37^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 恒温培养箱中培养 $20 \mathrm{~h} \sim 24 \mathrm{~h}$ , 取出后放入 $2^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 冰箱中保存。每月至少传种一次, 作为储备菌株保存。

5.2.2 将储备菌株接种至乳酸杆菌琼脂培养基中，在 $37^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温培养箱中培养 $20\mathrm{h} \sim 24\mathrm{h}$ 以活化菌株，用于接种液的制备。保存数周以上的储备菌种，不能立即用作接种液制备，试验前应连续传种2代～3代以保证细菌活力。

# 5.3 接种液的制备

用接种环将活化的菌株转种至已灭菌的乳酸杆菌肉汤中，于 $37^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温培养箱中培养 $16\mathrm{h} \sim 20\mathrm{h}$ 。取出后将菌悬液离心弃去上清液，再用氯化钠溶液 $(0.85\%)$ 振匀并离心弃去上清液，如此三次。最后用氯化钠溶液 $(0.85\%)$ 稀释至透光率 $80\%$ 。

# 6 分析步骤

# 6.1 试样制备

谷薯类、豆类、乳粉等试样需粉碎、研磨、过筛（筛板孔径 $0.3\mathrm{mm}\sim 0.5\mathrm{mm}$ ）；肉、蛋、鱼、坚果等用打碎机制成食糜；果蔬、半固体食品等试样需匀浆混匀；液体试样用前振摇混合。4℃冰箱保存，1周内测定。

# 6.2 试样提取

6.2.1 薯类、肉类、乳类、新鲜果蔬、藻类试样、蛋类、豆类、坚果类、动物内脏等天然食品。准确称取适量均质样品 $(m)$ （约含 $0.2\mu \mathrm{g}\sim 0.5\mu \mathrm{g}$ 生物素），精确至 $0.001\mathrm{g}$ ，至一个 $50~\mathrm{mL}$ 锥形瓶中，加入 $30~\mathrm{mL}$ 柠檬酸缓冲液振摇后于 $121^{\circ}C$ 高压水解 $15\mathrm{min}$ 。样品取出后迅速冷却至室温，加入 $1\mathrm{mL}$ 蛋白酶-淀粉酶溶液置于 $36^{\circ}C\pm 1^{\circ}C$ 恒温培养箱内温育酶解 $16\mathrm{h}\sim 20\mathrm{h},95^{\circ}C$ 水浴中加热 $30\mathrm{min}$ ，然后迅速冷却至室温，转至 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度 $(V_{1})$ 。  
6.2.2 婴幼儿配方食品、谷物类等制品（包括原生和添加的生物素）：准确称取适量样品 $(m)$ （约含 $0.2\mu \mathrm{g}\sim 0.5\mu \mathrm{g}$ 生物素），精确至 $0.001\mathrm{g}$ ，至一个 $250~\mathrm{mL}$ 锥形瓶中加入硫酸溶液 $100~\mathrm{mL}$ ， $121^{\circ}C$ 水解 $30\mathrm{min}$ ，冷却后用氢氧化钠溶液调节 $\mathrm{pH}$ 至 $4.5\pm 0.2$ ，转到 $250~\mathrm{mL}$ 容量瓶中，用水定容，充分混合。用滤纸过滤，弃去最初的几毫升。吸取滤液 $5\mathrm{mL}$ ，加入约 $20~\mathrm{mL}$ 水，用氢氧化钠溶液调 $\mathrm{pH}$ 为 $6.8\pm 0.2$ ，转至 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度 $(V_{1})$ 。  
6.2.3 强化生物素的饮料或维生素预混料等样品：液体饮料加 $5\mathrm{mL}\sim 10\mathrm{mL}$ 样品至 $100~\mathrm{mL}$ 锥形瓶中，加 $50~\mathrm{mL}$ 水，混匀，转入 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度 $(V_{1})$ ；维生素预混料：准确称取适量样品 $(m)$ ，精确至 $0.001\mathrm{g}$ ，到 $500~\mathrm{mL}$ 锥形瓶中，加入约 $300~\mathrm{mL}$ 水，混匀。调整 $\mathrm{pH}$ 到 $8.0\pm 0.2$ ，转入 $1000~\mathrm{mL}$ 容量瓶中，用水定容至刻度 $(V_{1})$ 。

# 6.3 稀释

根据试样中生物素含量用水对试样提取液进行适当稀释，使稀释后试样提取液中生物素含量在 $0.01\mathrm{ng / mL}\sim 0.1\mathrm{ng / mL}$ 范围内。

# 6.4 测定系列管制备

# 6.4.1 试样系列管

取4支试管，分别加入 $1.0\mathrm{mL},2.0\mathrm{mL},3.0\mathrm{mL},4.0\mathrm{mL}$ 试样提取液，补水至 $5.0~\mathrm{mL}$ ，加入 $5.0~\mathrm{mL}$ 生物素测定用培养液，混匀。每个梯度做3个平行。

# 6.4.2 标准系列管

取试管分别加入标准使用工作液低浓度 $0.0\mathrm{mL}$ （未接种空白）、 $0.0\mathrm{mL}$ （接种空白）、 $1.0\mathrm{mL}$ 、 $2.0\mathrm{mL}$ 、 $3.0\mathrm{mL}$ 、 $4.0\mathrm{mL}$ 、 $5.00\mathrm{mL}$ 、和高浓度 $3.0\mathrm{mL}$ 、 $4.0\mathrm{mL}$ 、 $5.0\mathrm{mL}$ ，补水至 $5.00\mathrm{mL}$ ，相当于标准系列管中生物素含量为 $0.00\mathrm{ng}$ 、 $0.00\mathrm{ng}$ 、 $0.1\mathrm{ng}$ 、 $0.2\mathrm{ng}$ 、 $0.3\mathrm{ng}$ 、 $0.4\mathrm{ng}$ 、 $0.5\mathrm{ng}$ 、 $0.6\mathrm{ng}$ 、 $0.8\mathrm{ng}$ 、 $1.00\mathrm{ng}$ 。加 $5.0\mathrm{mL}$ 生物素测定用培养液，混匀。每个梯度做3个平行，绘制标准曲线时，以每点均值计算。

# 6.5 培养

6.5.1 灭菌：所有的试管盖上试管帽，放入灭菌釜内， $121^{\circ}\mathrm{C}(0.10\mathrm{mPa}\sim 0.12\mathrm{mPa})$ 灭菌 $5\mathrm{min}$ 。  
6.5.2 接种和培养：试管快速冷却至室温，在无菌操作条件下，将接种液转入无菌针管，向每支测定管接种一滴(约 $50\mu \mathrm{L}$ )其中标准曲线管中未接种空白和样品空白除外。置于 $37^{\circ}C\pm 1^{\circ}C$ 恒温培养箱中培养 $19\mathrm{h}\sim 20\mathrm{h}$ ，直至获得最大混浊度，即再培养 $2\mathrm{h}$ 透光率无明显变化。

# 6.6 测定

将培养好的测定管用漩涡混匀器混匀。用厚度为 $1\mathrm{cm}$ 比色杯，于 $550~\mathrm{nm}$ 处，以接种空白管调节透光率为 $100\%$ ，然后依次测定标准系列管、试样系列管吸光值。如果未接种空白对照管有明显的细菌增长，说明可能有杂菌混入，需重做试验。

注：试样提取液也可采用预先包埋了菌种的微生物法生物素试剂盒测定，效果相当。

# 6.7 分析结果表述

6.7.1 标准曲线：以标准系列管生物素含量为横坐标，吸光值为纵坐标，绘制标准曲线。  
6.7.2 结果计算：从标准曲线查得样液相应含量 $(c_{x})$ ，如果每个试样的3个测试管中有2个值落在 $0.01\mathrm{ng}\sim 0.10\mathrm{ng}$ 范围内，且每个测试管之间吸光值偏差小于 $10\%$ ，则按下式进行结果计算。

测定液浓度按式(1)计算：

$$
c = \frac {c _ {x}}{V _ {x}} \tag {1}
$$

式中：

$c$ ——样液生物素浓度，单位为纳克每毫升（ $\mathrm{ng / mL}$ ）；

$c_{x}$ ——从标准曲线上查得待测样液生物素含量，单位为纳克（ng）；

$V_{\bar{x}}$ ——制备系列管时吸取的试样提取液体积，单位为毫升（ $\mathrm{mL}$ ）。

样品中生物素含量按式(2)计算：

$$
X = \frac {\bar {c} \times f}{m} \times \frac {1 0 0}{1 0 0 0} \tag {2}
$$

式中：

$X$ ——样品中生物素含量，单位为微克每百克或毫升（ $\mu \mathrm{g} / 100 \mathrm{~g}$ 或 $\mathrm{mL}$ ）；

$\overline{c}$ ——有效测试管试样中生物素浓度平均值，单位为纳克每毫升（ $\mathrm{ng / mL}$ ）；

$f$ ——样液稀释倍数；

$m$ ——样品质量，单位为克（g）；

100 ——换算系数；

1000 ——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法线性范围为 $0.01\mu \mathrm{g / mL}\sim 0.1\mu \mathrm{g / mL}$ ，检出限为 $2.0~\mu \mathrm{g} / 100\mathrm{g}$ ，定量限为 $4.0~\mu \mathrm{g} / 100\mathrm{g}$ 。

# A.1 乳酸杆菌琼脂培养基

# A.1.1 成分

胨化乳 $15.0\mathrm{g}$ ，酵母浸膏 $5.0\mathrm{g}$ ，葡萄糖 $10.0\mathrm{g}$ ，番茄汁 $100\mathrm{mL}$ ，磷酸二氢钾 $2.0\mathrm{g}$ ，聚山梨糖单油酸酯 $1.0\mathrm{g}$ ，加水至 $1000\mathrm{mL}$ ，调节 $\mathrm{pH}$ 至 $6.8 \pm 0.2(25^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C})$ 。

# A.1.2 制法

在A.1.1中加入 $10.0\mathrm{g}$ 琼脂，加热煮沸，使琼脂溶化。混合均匀后分装试管，每管 $10~\mathrm{mL}$ 。 $121^{\circ}\mathrm{C}$ 高压灭菌 $15\mathrm{min}$ ，备用。

# A.2 乳酸杆菌肉汤培养基

# A.2.1 成分

胨化乳 $15.0\mathrm{g}$ ，酵母浸膏 $5.0\mathrm{g}$ ，葡萄糖 $10.0\mathrm{g}$ ，番茄汁 $100\mathrm{mL}$ ，磷酸二氢钾 $2.0\mathrm{g}$ ，聚山梨糖单油酸酯 $1.0\mathrm{g}$ ，加水至 $1000\mathrm{mL}$ ，调节 $\mathrm{pH}$ 至 $6.8 \pm 0.2$ （ $25^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ ）。

# A.2.2 制法

将A.2.1成分加热煮沸，混合均匀后分装试管，每管 $10~\mathrm{mL}$ 。 $121^{\circ}\mathrm{C}$ 高压灭菌 $15\mathrm{min}$ 。

# A.3 生物素测定用培养基

# A.3.1 成分

维生素测定用酪蛋白氨基酸 $12.0\mathrm{g}$ ，葡萄糖 $40.0\mathrm{g}$ ，乙酸钠 $20.0\mathrm{g}$ ，L-胱氨酸 $0.2\mathrm{g}$ ，DL-色氨酸 $0.2\mathrm{g}$ ，硫酸腺嘌呤 $20.0\mathrm{mg}$ ，盐酸鸟嘌呤 $20.0\mathrm{mg}$ ，尿嘧啶 $20.0\mathrm{mg}$ ，盐酸硫胺素 $2.0\mathrm{mg}$ ，核黄素 $2.0\mathrm{mg}$ ，烟酸 $2.0\mathrm{mg}$ ，泛酸钙 $2.0\mathrm{mg}$ ，盐酸吡哆醇 $4.0\mathrm{mg}$ ， $\rho^{-}$ 氨基苯甲酸 $200.0\mu \mathrm{g}$ ，磷酸氢二钾 $1.0\mathrm{g}$ ，磷酸二氢钾 $1.0\mathrm{g}$ ，硫酸镁 $0.4\mathrm{g}$ ，氯化钠 $20.0\mathrm{mg}$ ，硫酸亚铁 $20.0\mathrm{mg}$ ，硫酸锰 $20.0\mathrm{mg}$ ，加水至 $1000\mathrm{mL}$ ， $\mathrm{pH}6.8 \pm 0.2(25^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C})$ 。

# A.3.2 制法

将A.3.1的成分溶解于水中，调节 $\mathrm{pH}$ ，备用。

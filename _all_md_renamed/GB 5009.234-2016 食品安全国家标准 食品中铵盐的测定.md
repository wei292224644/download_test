2016-08-31 发布

2017-03-01实施

# 前言

本标准代替GB/T5009.39—2003《酱油卫生标准的分析方法》中4.9铵盐。

本标准与GB/T5009.39—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中铵盐的测定”。

# 食品安全国家标准

# 食品中铵盐的测定

# 1 范围

本标准规定了酱油中铵盐的测定方法。

本标准适用于酱油中铵盐的测定。

# 2 原理

试样在碱性溶液中加热蒸馏，使氨游离出来，被硼酸溶液吸收，然后用盐酸标准溶液滴定计算含量。

# 3 试剂和材料

# 3.1 试剂

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

3.1.1 氧化镁 $(\mathrm{MgO})$ 。  
3.1.2 硼酸 $\mathrm{(H_3BO_3)}$ 。  
3.1.3 甲基红 $(\mathrm{C_{15}H_{15}N_3O_2})$ 。  
3.1.4 溴甲酚绿 $(\mathrm{C_{15}H_{14}Br_4O_5S})$ 。  
3.1.5 乙醇 $\mathrm{(C_2H_5OH)}$ 。

# 3.2 试剂配制

3.2.1 硼酸溶液 $(20\mathrm{g / L})$ ：称取 $20.0\mathrm{g}$ 硼酸，加水溶解，稀释至 $1000~\mathrm{mL}$ 。  
3.2.2 盐酸标准滴定液 $[c(\mathrm{HCl})] = 0.100\mathrm{mol / L}$   
3.2.3 混合指示液：甲基红-乙醇溶液 $(2\mathrm{g} / \mathrm{L})$ 1份与溴甲酚绿-乙醇溶液 $(2\mathrm{g} / \mathrm{L})$ 5份，临用新配。

# 4 仪器和设备

滴定管。

# 5 分析步骤

准确吸取 $2\mathrm{mL}$ 混匀试样，置于 $500~\mathrm{mL}$ 蒸馏瓶中，加 $150~\mathrm{mL}$ 蒸馏水及 $1\mathrm{g}$ 氧化镁，连接好蒸馏装置，并使冷凝管下端连接弯管伸入接收瓶液面下，吸收瓶内盛有 $10~\mathrm{mL}$ 硼酸溶液 $(20~\mathrm{g / L})$ 及2滴混合指示剂，加热蒸馏，溶液沸腾约 $30\mathrm{min}$ 即可，用少量水冲洗弯管，以盐酸标准滴定液 $(0.100\mathrm{mol / L})$ 滴定至终点。取同量水、氧化镁、硼酸溶液按同一方法作空白实验。

# 6 分析结果的表述

试样中铵盐的含量按式(1)计算：

$$
X = \frac {\left(V _ {1} - V _ {2}\right) \times c \times 0 . 0 1 7}{V _ {3}} \times 1 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——酱油中铵盐含量，单位为克每一百毫升 $(\mathrm{g} / 100\mathrm{mL})$

$V_{1}$ ——测定用试样消耗盐酸标准滴定液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试剂空白消耗盐酸标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——盐酸标准滴定液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

0.017——与 $1.00 \mathrm{~mL}$ 盐酸标准滴定液 $[c(\mathrm{HCl})] = 0.100 \mathrm{~mol} / \mathrm{L}$ 相当的铵盐（以氨计）的质量，单位为克(g)；

$V_{3}$ ——试样体积，单位为毫升 $(\mathrm{mL})$

100 ——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，保留2位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

方法检出限 $0.0030\mathrm{g} / 100\mathrm{mL}$

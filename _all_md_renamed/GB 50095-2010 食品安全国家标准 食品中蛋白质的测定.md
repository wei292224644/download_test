# 前言

本标准代替GB/T5009.5-2003《食品中蛋白质的测定》、GB/T14771-1993《食品中蛋白质的测定方法》和GB/T5413.1-1997《婴幼儿配方食品和乳粉蛋白质的测定》。

本标准与GB/T5009.5-2003相比主要修改如下：

——在第一法中增加了自动蛋白质测定仪的方法；  
——增加了燃烧法，作为第三法；  
——修改了换算系数；  
——对计算结果的有效数字规定进行了修改；  
——增加 $\mathrm{pH}$ 计对滴定终点的判定。

本标准所代替标准的历次版本发布情况为：

GB/T5009.5-1985、GB/T5009.5-2003;  
GB5413.1-1985、GB/T5413.1-1997；  
GB/T14771-1993。

# 食品安全国家标准

# 食品中蛋白质的测定

# 1 范围

本标准规定了食品中蛋白质的测定方法。

本标准第一法和第二法适用于各种食品中蛋白质的测定，第三法适用于蛋白质含量在 $10\mathrm{g} / 100\mathrm{g}$ 以上的粮食、豆类、奶粉、米粉、蛋白质粉等固体试样的筛选测定。

本标准不适用于添加无机含氮物质、有机非蛋白质含氮物质的食品测定。

# 第一法 凯氏定氮法

# 2 规范性引用性文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

# 3 原理

食品中的蛋白质在催化加热条件下被分解，产生的氨与硫酸结合生成硫酸铵。碱化蒸馏使氨游离，用硼酸吸收后以硫酸或盐酸标准滴定溶液滴定，根据酸的消耗量乘以换算系数，即为蛋白质的含量。

# 4 试剂和材料

除非另有规定，本方法中所用试剂均为分析纯，水为GB/T6682规定的三级水。

4.1 硫酸铜（ $\mathrm{CuSO_4 \cdot 5H_2O}$ ）。  
4.2 硫酸钾（ $\mathrm{K}_2\mathrm{SO}_4$ ）。  
4.3 硫酸（ $\mathrm{H}_2\mathrm{SO}_4$ 密度为 $1.84 \mathrm{~g} / \mathrm{L}$ ）。  
4.4硼酸（ $\mathrm{H}_3\mathrm{BO}_3$ ）。  
4.5 甲基红指示剂（ $\mathrm{C_{15}H_{15}N_3O_2}$ ）。  
4.6 溴甲酚绿指示剂（ $\mathrm{C_{21}H_{14}Br_4O_5S}$ ）。  
4.7 亚甲基蓝指示剂（ $\mathrm{C_{16}H_{18}ClN_3S\cdot 3H_2O}$ ）。  
4.8 氢氧化钠（NaOH）。  
4.9 $95\%$ 乙醇（ $\mathrm{C}_2\mathrm{H}_5\mathrm{OH}$ ）。  
4.10硼酸溶液（ $20\mathrm{g / L}$ ）：称取 $20\mathrm{g}$ 硼酸，加水溶解后并稀释至 $1000\mathrm{mL}$ 。

4.11 氢氧化钠溶液（ $400\mathrm{g / L}$ ）：称取 $40\mathrm{g}$ 氢氧化钠加水溶解后，放冷，并稀释至 $100\mathrm{mL}$ 。  
4.12 硫酸标准滴定溶液（0.0500 mol/L）或盐酸标准滴定溶液（0.0500 mol/L）。  
4.13 甲基红乙醇溶液（ $1\mathrm{g/L}$ ）：称取 $0.1\mathrm{g}$ 甲基红，溶于 $95\%$ 乙醇，用 $95\%$ 乙醇稀释至 $100\mathrm{mL}$ 。  
4.14 亚甲基蓝乙醇溶液（ $1\mathrm{g/L}$ ）：称取 $0.1\mathrm{g}$ 亚甲基蓝，溶于 $95\%$ 乙醇，用 $95\%$ 乙醇稀释至 $100\mathrm{mL}$ 。  
4.15 溴甲酚绿乙醇溶液（ $1\mathrm{g/L}$ ）：称取 $0.1\mathrm{g}$ 溴甲酚绿，溶于 $95\%$ 乙醇，用 $95\%$ 乙醇稀释至 $100\mathrm{mL}$ 。  
4.16 混合指示液：2份甲基红乙醇溶液（4.13）与1份亚甲基蓝乙醇溶液（4.14）临用时混合。也可用1份甲基红乙醇溶液（4.13）与5份溴甲酚绿乙醇溶液（4.15）临用时混合。

# 6.1 凯氏定氮法

6.1.1 试样处理：称取充分混匀的固体试样 $0.2\mathrm{g}\sim 2\mathrm{g}$ 、半固体试样 $2\mathrm{g}\sim 5\mathrm{g}$ 或液体试样 $10\mathrm{g}\sim 25\mathrm{g}$ （约相当于 $30\mathrm{mg}\sim 40\mathrm{mg}$ 氮)，精确至 $0.001\mathrm{g}$ ，移入干燥的 $100\mathrm{mL}$ 、 $250\mathrm{mL}$ 或 $500\mathrm{mL}$ 定氮瓶中，加入 $0.2\mathrm{g}$ 硫酸铜（4.1）、 $6\mathrm{g}$ 硫酸钾（4.2）及 $20\mathrm{mL}$ 硫酸（4.3），轻摇后于瓶口放一小漏斗，将瓶以 $45^{\circ}$ 角斜支于有小孔的石棉网上。小心加热，待内容物全部炭化，泡沫完全停止后，加强火力，并保持瓶内液体微沸，至液体呈蓝绿色并澄清透明后，再继续加热 $0.5\mathrm{h}\sim 1\mathrm{h}$ 。取下放冷，小心加入 $20\mathrm{mL}$ 水。放冷后，移入 $100\mathrm{mL}$ 容量瓶中，并用少量水洗定氮瓶，洗液并入容量瓶中，再加水至刻度，混匀备用。同时做试剂空白试验。  
6.1.2 测定：按图1装好定氮蒸馏装置，向水蒸气发生器内装水至2/3处，加入数粒玻璃珠，加甲基红乙醇溶液（4.13）数滴及数毫升硫酸（4.3），以保持水呈酸性，加热煮沸水蒸气发生器内的水并保持沸腾。  
6.1.3 向接收瓶内加入 $10.0 \mathrm{~mL}$ 硼酸溶液 (4.10) 及 1 滴 $\sim 2$ 滴混合指示液 (4.16), 并使冷凝管的下端插入液面下, 根据试样中氮含量, 准确吸取 $2.0 \mathrm{~mL} \sim 10.0 \mathrm{~mL}$ 试样处理液由小玻杯注入反应室, 以 $10 \mathrm{~mL}$ 水洗涤小玻杯并使之流入反应室内, 随后塞紧棒状玻塞。将 $10.0 \mathrm{~mL}$ 氢氧化钠溶液 (4.11) 倒入小玻杯, 提起玻塞使其缓缓流入反应室, 立即将玻塞盖紧, 并加水于小玻杯以防漏气。夹紧螺旋夹, 开始蒸馏。蒸馏 $10 \mathrm{~min}$ 后移动蒸馏液接收瓶, 液面离开冷凝管下端, 再蒸馏 $1 \mathrm{~min}$ 。然后用少量水冲洗冷凝管下端外部, 取下蒸馏液接收瓶。以硫酸或盐酸标准滴定溶液 (4.12) 滴定至终点, 其中 2 份甲基红乙醇溶液 (4.13)与 1 份亚甲基蓝乙醇溶液 (4.14) 指示剂, 颜色由紫红色变成灰色, $\mathrm{pH} 5.4$ ; 1 份甲基红乙醇溶液 (4.13)与 5 份溴甲酚绿乙醇溶液 (4.15) 指示剂, 颜色由酒红色变成绿色, $\mathrm{pH} 5.1$ 。同时作试剂空白。

# 6.2 自动凯氏定氮仪法

称取固体试样 $0.2\mathrm{g}\sim 2\mathrm{g}$ 、半固体试样 $2\mathrm{g}\sim 5\mathrm{g}$ 或液体试样 $10\mathrm{g}\sim 25\mathrm{g}$ （约相当于 $30\mathrm{mg}\sim 40\mathrm{mg}$ 氮），精确至 $0.001\mathrm{g}$ 。按照仪器说明书的要求进行检测。

# 7 分析结果的表述

试样中蛋白质的含量按式（1）进行计算。

$$
X = \frac {\left(V _ {1} - V _ {2}\right) \times c \times 0 . 0 1 4 0}{m \times V _ {3} / 1 0 0} \times F \times 1 0 0 \tag {1}
$$

式中：

$X$ ——试样中蛋白质的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$V_{I}$ ——试液消耗硫酸或盐酸标准滴定液的体积，单位为毫升（ $\mathrm{mL}$ ）；

$V_{2}$ ——试剂空白消耗硫酸或盐酸标准滴定液的体积, 单位为毫升 $(\mathrm{mL})$ ;

$V_{3}$ ——吸取消化液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫酸或盐酸标准滴定溶液浓度，单位为摩尔每升（mol/L）；

0.0140— $1.0~\mathrm{mL}$ 硫酸 $\left[\mathrm{c} (1 / 2\mathrm{H}_2\mathrm{SO}_4) = 1.000\mathrm{mol} / \mathrm{L}\right]$ 或盐酸 $\left[\mathrm{c} (\mathrm{HCl}) = 1.000\mathrm{mol} / \mathrm{L}\right]$ 标准滴定溶液相当的氮的质量，单位为克（g）；

$m$ ——试样的质量，单位为克（g）；

$F$ ——氮换算为蛋白质的系数。一般食物为 6.25；纯乳与纯乳制品为 6.38；面粉为 5.70；玉米、高粱为 6.24；花生为 5.46；大米为 5.95；大豆及其粗加工制品为 5.71；大豆蛋白制品为 6.25；肉与肉制品为 6.25；大麦、小米、燕麦、裸麦为 5.83；芝麻、向日葵为 5.30；复合配方食品为 6.25。

以重复性条件下获得的两次独立测定结果的算术平均值表示，蛋白质含量 $\geqslant 1\mathrm{g} / 100\mathrm{g}$ 时，结果保留三位有效数字；蛋白质含量 $< 1\mathrm{g} / 100\mathrm{g}$ 时，结果保留两位有效数字。

# 8 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 分光光度法

# 9 原理

食品中的蛋白质在催化加热条件下被分解，分解产生的氨与硫酸结合生成硫酸铵，在 $\mathrm{pH}4.8$ 的乙酸钠-乙酸缓冲溶液中与乙酰丙酮和甲醛反应生成黄色的3,5-二乙酰-2,6-二甲基-1,4-二氢化吡啶化合物。在波长 $400\mathrm{nm}$ 下测定吸光度值，与标准系列比较定量，结果乘以换算系数，即为蛋白质含量。

10.1 硫酸铜（ $\mathrm{CuSO_4 \cdot 5H_2O}$ ）。  
10.2 硫酸钾 $\left(\mathrm{K}_{2} \mathrm{SO}_{4}\right)$ 。  
10.3 硫酸（ $\mathrm{H}_2\mathrm{SO}_4$ 密度为 $1.84 \mathrm{~g} / \mathrm{L}$ ）：优级纯。  
10.4 氢氧化钠（NaOH）。  
10.5 对硝基苯酚 $\left(\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{NO}_{3}\right)$ 。   
10.6 乙酸钠（ $\mathrm{CH}_3\mathrm{COONa}\cdot 3\mathrm{H}_2\mathrm{O}$ ）。  
10.7 无水乙酸钠（ $\mathrm{CH}_3\mathrm{COONa}$ ）。  
10.8 乙酸（ $\mathrm{CH}_3\mathrm{COOH}$ ）：优级纯。  
10.9 $37\%$ 甲醛（HCHO）。  
10. 10 乙酰丙酮 $\left(\mathrm{C}_{5} \mathrm{H}_{8} \mathrm{O}_{2}\right)$ 。  
10.11 氢氧化钠溶液（ $300\mathrm{g/L}$ ）：称取 $30\mathrm{g}$ 氢氧化钠加水溶解后，放冷，并稀释至 $100\mathrm{mL}$ 。  
10.12 对硝基苯酚指示剂溶液（ $1\mathrm{g/L}$ ）：称取 $0.1\mathrm{g}$ 对硝基苯酚指示剂溶于 $20\mathrm{mL}95\%$ 乙醇中，加水稀释至 $100\mathrm{mL}$ 。  
10.13 乙酸溶液（ $1\mathrm{mol} / \mathrm{L}$ ）：量取 $5.8\mathrm{mL}$ 乙酸（10.8），加水稀释至 $100\mathrm{mL}$ 。  
10.14 乙酸钠溶液（ $1\mathrm{mol} / \mathrm{L}$ ）：称取 $41\mathrm{g}$ 无水乙酸钠（10.7）或 $68\mathrm{g}$ 乙酸钠（10.6），加水溶解后并稀释至 $500\mathrm{mL}$ 。  
10.15 乙酸钠-乙酸缓冲溶液：量取 $60 \mathrm{~mL}$ 乙酸钠溶液（10.14）与 $40 \mathrm{~mL}$ 乙酸溶液（10.13）混合，该溶液 $\mathrm{pH} 4.8$ 。  
10.16 显色剂: $15 \mathrm{~mL}$ 甲醛 (10.9) 与 $7.8 \mathrm{~mL}$ 乙酰丙酮 (10.10) 混合, 加水稀释至 $100 \mathrm{~mL}$ , 剧烈振摇混匀（室温下放置稳定 3d）。  
10.17 氨氮标准储备溶液（以氮计）（ $1.0 \mathrm{~g} / \mathrm{L}$ ）：称取 $105^{\circ} \mathrm{C}$ 干燥 $2 \mathrm{~h}$ 的硫酸铵 $0.4720 \mathrm{~g}$ 加水溶解后移于 $100 \mathrm{~mL}$ 容量瓶中，并稀释至刻度，混匀，此溶液每毫升相当于 $1.0 \mathrm{mg}$ 氮。  
10.18 氨氮标准使用溶液（ $0.1\mathrm{g/L}$ ）：用移液管吸取 $10.00\mathrm{mL}$ 氨氮标准储备液（10.17）于 $100\mathrm{ml}$ 容

量瓶内，加水定容至刻度，混匀，此溶液每毫升相当于 $0.1 \mathrm{mg}$ 氮。

# 12 分析步骤

# 12.1 试样消解

称取经粉碎混匀过40目筛的固体试样 $0.1\mathrm{g}\sim 0.5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）、半固体试样 $0.2\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）或液体试样 $1\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ），移入干燥的 $100\mathrm{mL}$ 或 $250\mathrm{mL}$ 定氮瓶中，加入 $0.1\mathrm{g}$ 硫酸铜、 $1\mathrm{g}$ 硫酸钾及 $5\mathrm{mL}$ 硫酸（10.3），摇匀后于瓶口放一小漏斗，将定氮瓶以 $45^{\circ}$ 角斜支于有小孔的石棉网上。缓慢加热，待内容物全部炭化，泡沫完全停止后，加强火力，并保持瓶内液体微沸，至液体呈蓝绿色澄清透明后，再继续加热半小时。取下放冷，慢慢加入 $20\mathrm{mL}$ 水，放冷后移入 $50\mathrm{mL}$ 或 $100\mathrm{mL}$ 容量瓶中，并用少量水洗定氮瓶，洗液并入容量瓶中，再加水至刻度，混匀备用。按同一方法做试剂空白试验。

# 12.2 试样溶液的制备

吸取 $2.00\mathrm{mL} \sim 5.00\mathrm{mL}$ 试样或试剂空白消化液于 $50\mathrm{mL}$ 或 $100\mathrm{mL}$ 容量瓶内，加1滴～2滴对硝基苯酚指示剂溶液（10.12），摇匀后滴加氢氧化钠溶液（10.11）中和至黄色，再滴加乙酸溶液（10.13）至溶液无色，用水稀释至刻度，混匀。

# 12.3 标准曲线的绘制

吸取 $0.00\mathrm{mL}$ 、 $0.05\mathrm{mL}$ 、 $0.10\mathrm{mL}$ 、 $0.20\mathrm{mL}$ 、 $0.40\mathrm{mL}$ 、 $0.60\mathrm{mL}$ 、 $0.80\mathrm{mL}$ 和 $1.00\mathrm{mL}$ 氨氮标准使用溶液（相当于 $0.00\mu \mathrm{g}$ 、 $5.00\mu \mathrm{g}$ 、 $10.0\mu \mathrm{g}$ 、 $20.0\mu \mathrm{g}$ 、 $40.0\mu \mathrm{g}$ 、 $60.0\mu \mathrm{g}$ 、 $80.0\mu \mathrm{g}$ 和 $100.0\mu \mathrm{g}$ 氮），分别置于 $10\mathrm{mL}$ 比色管中。加 $4.0\mathrm{mL}$ 乙酸钠-乙酸缓冲溶液（10.15）及 $4.0\mathrm{mL}$ 显色剂（10.16)，加水稀释至刻度，混匀。置于 $100^{\circ}C$ 水浴中加热 $15\mathrm{min}$ 。取出用水冷却至室温后，移入 $1\mathrm{cm}$ 比色杯内，以零管为参比，于波长 $400\mathrm{nm}$ 处测量吸光度值，根据标准各点吸光度值绘制标准曲线或计算线性回归方程。

# 12.4 试样测定

吸取 $0.50\mathrm{mL}\sim 2.00\mathrm{mL}$ （约相当于氮 $< 100~\mu \mathrm{g}$ ）试样溶液和同量的试剂空白溶液，分别于 $10\mathrm{mL}$ 比色管中。以下按12.3自“加 $4\mathrm{mL}$ 乙酸钠-乙酸缓酸溶液（pH4.8）及 $4\mathrm{mL}$ 显色剂……”起操作。试样吸光度值与标准曲线比较定量或代入线性回归方程求出含量。

# 13 分析结果的表述

试样中蛋白质的含量按式（2）进行计算。

$$
X = \frac {\left(c - c _ {0}\right)}{m \times \frac {V _ {2}}{V _ {1}} \times \frac {V _ {4}}{V _ {3}} \times 1 0 0 0 \times 1 0 0 0} \times 1 0 0 \times F \dots \tag {2}
$$

式中：

$X$ ——试样中蛋白质的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$c$ ——试样测定液中氮的含量, 单位为微克 $(\mu \mathrm{g})$ ;

$c_{0}$ ——试剂空白测定液中氮的含量, 单位为微克 $(\mu \mathrm{g})$ ;

$V_{I}$ ——试样消化液定容体积, 单位为毫升 $(\mathrm{mL})$ ;

$V_{2}$ ——制备试样溶液的消化液体积，单位为毫升（ $\mathrm{mL}$ ）；

$V_{3}$ ——试样溶液总体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{4}$ ——测定用试样溶液体积, 单位为毫升 $(\mathrm{mL})$ ;

$m$ ——试样质量，单位为克（g）；

$F$ ——氮换算为蛋白质的系数。一般食物为6.25；纯乳与纯乳制品为6.38；面粉为5.70；玉米、高粱为6.24；花生为5.46；大米为5.95；大豆及其粗加工制品为5.71；大豆蛋白制品为6.25；肉与肉制品为6.25；大麦、小米、燕麦、裸麦为5.83；芝麻、向日葵为5.30；复合配方食品为6.25。

# 第三法 燃烧法

# 15 原理

试样在 $900^{\circ} \mathrm{C} \sim 1200^{\circ} \mathrm{C}$ 高温下燃烧，燃烧过程中产生混合气体，其中的碳、硫等干扰气体和盐类被吸收管吸收，氮氧化物被全部还原成氮气，形成的氮气气流通过热导检测仪（TCD）进行检测。

# 16 仪器和设备

16.1 氮/蛋白质分析仪。  
16.2天平：感量为 $0.1\mathrm{mg}$

# 17 分析步骤

按照仪器说明书要求称取 $0.1 \mathrm{~g} \sim 1.0 \mathrm{~g}$ 充分混匀的试样（精确至 $0.0001 \mathrm{~g}$ ），用锡箔包裹后置于样品盘上。试样进入燃烧反应炉（ $900^{\circ} \mathrm{C} \sim 1200^{\circ} \mathrm{C}$ ）后，在高纯氧（ $\geqslant 99.99\%$ ）中充分燃烧。燃烧炉中的产物（ $\mathrm{NOx}$ ）被载气 $\mathrm{CO}_{2}$ 运送至还原炉（ $800^{\circ} \mathrm{C}$ ）中，经还原生成氮气后检测其含量。

# 18 分析结果的表述

试样中蛋白质的含量按式（3）进行计算。

$$
X = C \times F \dots \tag {3}
$$

式中：

$X$ ——试样中蛋白质的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

C——试样中氮的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$F$ ——氮换算为蛋白质的系数。一般食物为6.25；纯乳与纯乳制品为6.38；面粉为5.70；玉米、高粱为6.24；花生为5.46；大米为5.95；大豆及其粗加工制品为5.71；大豆蛋白制品为6.25；肉与肉制品为6.25；大麦、小米、燕麦、裸麦为5.83；芝麻、向日葵为5.30；复合配方食品为6.25。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

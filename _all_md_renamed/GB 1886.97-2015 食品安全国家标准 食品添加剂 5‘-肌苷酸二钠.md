# 检验方法

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 $5^{\prime}$ -肌苷酸二钠含量（以干基计）的测定

# A.2.1 试剂和材料

盐酸溶液： $0.01\mathrm{mol / L}$

# A.2.2 仪器和设备

紫外分光光度计。

# A.2.3 分析步骤

# A.2.3.1 试样溶液的制备

A.2.3.1.1 称取试样 $0.5\mathrm{g}$ （精确至 $0.1\mathrm{mg}$ ），用盐酸溶液(A.2.1)溶解并定容至 $500~\mathrm{mL}$ ，摇匀。

A.2.3.1.2 吸取溶液(A.2.3.1.1) $5 \mathrm{~mL}$ , 用盐酸溶液(A.2.1)稀释并定容至 $250 \mathrm{~mL}$ , 摇匀, 作为试样溶液备用。

# A.2.3.2 测定

将试样溶液(A.2.3.1.2)注入 $10\mathrm{mm}$ 石英比色皿中，以盐酸溶液(A.2.1)作空白，于紫外分光光度计 $250~\mathrm{nm}$ 处测定吸光度。

# A.2.4 结果计算

$5^{\prime}$ -肌苷酸二钠含量(以干基计)的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {A}{310} \times \frac {250}{m} \times \frac {100}{100 - w _ {2}} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$A$ ——250 nm 处的吸光度；

310 $5^{\prime}$ -肌苷酸二钠的比吸光度；

250 ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$w_{2}$ ——样品的水分质量分数， $\%$ 。

在重复性条件下获得的两次独立测定结果的绝对差值不得大于算术平均值的 $1\%$ 。

# A.3 $\mathbf{pH}$ 的测定

# A.3.1 仪器和设备

$\mathrm{pH}$ 计。

# A.3.2 分析步骤

称取试样 $1.0\mathrm{g}$ ，加入 $20~\mathrm{mL}$ 水溶解，用 $\mathrm{pH}$ 计测定溶液 $\mathrm{pH}$ 。

# A.3.3 允许差

在重复性条件下获得的两次独立测定结果之差不得超过0.2。

# A.4 其他核苷酸的测定

# A.4.1 试剂和材料

A.4.1.1 展开溶液：饱和硫酸铵溶液：水 $= 3:7$

A.4.1.2 滤纸：层析滤纸。

# A.4.2 仪器和设备

A.4.2.1 紫外分析仪。  
A.4.2.2 微量注射器。

# A.4.3 分析步骤

称取试样 $1.0\mathrm{g}$ ，加水溶解并定容至 $100~\mathrm{mL}$ ，在层析滤纸上点样 $10.0\mu \mathrm{L}$ ，于展开溶液中展开约 $30~\mathrm{cm}$ ，晾干(或烘干)滤纸后，在紫外分析仪 $254~\mathrm{nm}$ 处观察。

# A.4.4 判定

若只呈现一个斑点，则为 $5^{\prime}$ -肌苷酸二钠，判定为合格，否则判定为不合格。

# A.5 紫外吸光度比值的测定

# A.5.1 仪器和设备

紫外分光光度计。

# A.5.2 分析步骤

将试样溶液(A.2.3.1.2)注入 $10\mathrm{mm}$ 石英比色皿，以 $0.01\mathrm{mol} / \mathrm{L}$ 盐酸溶液(A.2.1)作空白，分别测定在波长 $250\mathrm{nm}, 260\mathrm{nm}, 280\mathrm{nm}$ 下的吸光度。

# A.5.3 结果计算

紫外吸光度比值 $X_{1}(250\mathrm{nm} / 260\mathrm{nm})$ 和 $X_{2}(280\mathrm{nm} / 260\mathrm{nm})$ 分别按式(A.2)和(A.3)计算：

$$
X _ {1} = \frac {A _ {1}}{A _ {2}} \quad \dots \tag {A.2}
$$

$$
X _ {2} = \frac {A _ {3}}{A _ {2}} \quad \dots \tag {A.3}
$$

式中：

$A_{1}$ ——在波长 $250 \mathrm{~nm}$ 下的吸光度；

$A_{2}$ ——在波长 $260\mathrm{nm}$ 下的吸光度；

$A_{3}$ ——在波长 $280\mathrm{nm}$ 下的吸光度。

在重复性条件下获得的两次独立测定结果的绝对差值不得大于算术平均值的 $1\%$ 。

# A.6 最大吸收波长的测定

# A.6.1 仪器和设备

紫外分光光度计。

# A.6.2 分析步骤

将试样溶液(A.2.3.1.2)注入 $10\mathrm{mm}$ 比色皿，以 $0.01\mathrm{mol / L}$ 盐酸溶液(A.2.1)作空白，在 $220~\mathrm{nm}\sim$ $320~\mathrm{nm}$ 波长测定吸收度。

# A.7 透光率的测定

# A.7.1 仪器和设备

分光光度计。

# A.7.2 分析步骤

称取试样 $1\mathrm{g}$ (精确至 $0.1\mathrm{mg}$ ) 溶于 $20\mathrm{mL}$ 水中，搅匀。然后，将试样液置于 $10\mathrm{mm}$ 比色皿中，以水作为空白对照，于 $430\mathrm{nm}$ 下测定试样液的透光率，记录读数。

# A.7.3 允许差

在重复性条件下获得的两次独立测定结果的绝对差值不得大于算术平均值的 $0.5\%$ 。

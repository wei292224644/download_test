# 食品安全国家标准

# 肉及肉制品中西玛津残留量的检测方法

# 1 范围

本标准规定了肉及肉制品中西玛津残留量的气相色谱检测方法。

本标准适用于牛肉中西玛津残留量的检验，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样经二氯甲烷一甲醇混合溶液提取，提取液经弗罗里硅土柱净化，用配有氮磷检测器的气相色谱仪测定，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置干燥器中备用。  
4.1.2 弗罗里硅土：层析用， $0.075 \mathrm{~mm} \sim 0.150 \mathrm{~mm}$ ，在 $650^{\circ} \mathrm{C}$ 灼烧 $4 \mathrm{~h}$ ，用前于 $130^{\circ} \mathrm{C}$ 烘 $5 \mathrm{~h}$ ，冷却后储于密闭容器中，备用。

# 4.2 溶液配制

4.2.1 丙酮—石油醚混合溶液 $(15 + 85)$ ：取 $150~\mathrm{mL}$ 丙酮，加入 $850~\mathrm{mL}$ 石油醚，摇匀备用。  
4.2.2 二氯甲烷—甲醇混合溶液 $(9 + 1)$ ：取 $900~\mathrm{mL}$ 二氯甲烷，加入 $100~\mathrm{mL}$ 甲醇，摇匀备用。

# 4.3 标准品

西玛津标准品：纯度 $\geq 98\%$

# 4.4 标准溶液配制

西玛津标准溶液：准确称取适量的西玛津标准品，用丙酮配制成 $100\mu \mathrm{g / mL}$ 标准储备液。根据需要，再用丙酮稀释成适用浓度的标准工作溶液。

# 5 仪器和设备

5.1 气相色谱仪，配氮磷检测器。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.3 旋转蒸发器。  
5.4 弗罗里硅土柱：玻璃柱， $40\mathrm{cm}\times 1.5\mathrm{cm}$ （内径），自上而下依次填装 $2\mathrm{cm}$ 高无水硫酸钠、 $10\mathrm{g}$ 弗罗里硅土、 $2\mathrm{cm}$ 高无水硫酸钠。使用前，用丙酮一石油醚混合液 $(15 + 85)$ 预淋洗。

# 5.5 超声波水浴。

# 6 试样制备与保存

# 6.1 试样制备

从所取全部样品中取出有代表性样品约 $1\mathrm{kg}$ ，样品取样部位按GB2763的规定执行，经均质机绞碎，混匀，均分成2份，分别装入洁净容器内作为试样。密封并标明标记。在抽样及制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样保存

试样于 $-18^{\circ}C$ 以下冷冻保存。

# 7 分析步骤

# 7.1 提取

称取试样约 $30 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ )。加 $30 \mathrm{~g}$ 无水硫酸钠, 研磨成粉状, 转入锥形瓶中。加入 $100 \mathrm{~mL}$ 二氯甲烷—甲醇 $(9 + 1)$ 混合液, 超声振荡 $20 \mathrm{~min}$ 。用布式漏斗抽滤, 保留滤液。残渣再用 $100 \mathrm{~mL}$ 二氯甲烷—甲醇 $(9 + 1)$ 混合液, 如上述操作。合并滤液于旋转蒸发器的蒸发瓶中。于 $50^{\circ} \mathrm{C}$ 水浴中减压蒸至近干。

# 7.2 净化

用 $20 \mathrm{~mL}$ 丙酮一石油醚 $(15 + 85)$ 分3次将上述残渣溶解，溶液注入弗罗里硅土柱中净化。用 $100 \mathrm{~mL}$ 丙酮一石油醚 $(15 + 85)$ 洗脱，控制流速每秒2滴 $\sim 3$ 滴，流出液接收于旋转蒸发液的蒸发瓶中。于 $60^{\circ} \mathrm{C}$ 水浴中减压蒸至近干。用丙酮定容至 $5.0 \mathrm{~mL}$ ，供GC测定。

# 7.3 测定

# 7.3.1 气相色谱参考条件

a）色谱柱：HP-1.5m×0.53mm(内径)，膜厚2.65μm；  
b）色谱柱温度： $155^{\circ}C$   
c) 进样口温度: $240^{\circ} \mathrm{C}$ ;   
d) 检测器温度: $250^{\circ} \mathrm{C}$ ;   
e) 载气：氮气，纯度 $\geq 99.99\%$ ，流速 $40~\mathrm{mL / min}$   
f) 氢气：流速 $2\mathrm{mL} / \mathrm{min}$   
g) 空气：流速 $140\mathrm{mL / min}$   
h) 进样量: $2 \mu \mathrm{L}$ ;   
i）进样方式：无分流进样。

# 7.3.2 色谱测定

根据样液中西玛津的含量情况, 选定峰面积相近的标准工作溶液。标准工作溶液和样液中西玛津的响应值均应在仪器检测线性范围内。标准工作溶液和样液等体积参插进样测定。在上述色谱条件下, 西玛津色谱峰保留时间为 $2.2 \mathrm{~min}$ 。西玛津标准品色谱图参见图 A. 1。

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

数据处理按式(1)计算。

$$
X = \frac {h \times c \times V}{h _ {s} \times m} \tag {1}
$$

式中：

$X$ ——试样中西玛津的残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$h$ ——样液中西玛津的峰高，单位为毫米（ $\mathrm{mm}$ ）；

$c$ ——标准工作液中西玛津的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$h_{s}$ ——标准工作液中西玛津的峰高，单位为毫米（mm）；

$m$ ——最终样液所代表的试样量，单位为克(g)。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 B 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 C 的要求。

# 10 定量限和回收率

# 10.1 定量限

本方法西玛津的定量限为 $0.02\mathrm{mg / kg}$

# 10.2 回收率

当添加水平为 $0.02\mathrm{mg / kg},0.05\mathrm{mg / kg},0.1\mathrm{mg / kg}$ 时，西玛津的添加回收率为 $89.6\% \sim 91.4\%$

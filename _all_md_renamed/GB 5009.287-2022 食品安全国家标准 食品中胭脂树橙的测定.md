# 食品安全国家标准

# 食品中胭脂树橙的测定

# 1 范围

本标准规定了食品中胭脂树橙(红木素、降红木素)的液相色谱测定方法。

本标准适用于干酪和再制干酪及其类似品、人造黄油及其类似制品、植脂末、冷冻饮品、果酱、巧克力和巧克力制品、糖果、粮食和粮食制品、焙烤食品、西式火腿类、肉灌肠类、复合调味料、饮料类、果冻、膨化食品中红木素和降红木素的测定。

# 2 原理

试样用氨水乙醇溶液提取，经石油醚脱脂，乙酸酸化后采用二氯甲烷萃取，液相色谱法分离，二极管阵列检测器或紫外检测器检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.2 无水乙醇 $\mathrm{(C_2H_6O)}$ ：色谱纯。  
3.1.3 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。  
3.1.4 石油醚：沸程为 $30^{\circ}\mathrm{C} \sim 60^{\circ}\mathrm{C}$ 。  
3.1.5 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：色谱纯。  
3.1.6 2,6-二叔丁基-4-甲基苯酚 $\mathrm{(C_{15}H_{24}O}$ ，简称BHT）。  
3.1.7 浓氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ，浓度含量为 $25\% \sim 28\%$   
3.1.8 冰乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。

# 3.2 试剂配制

3.2.1 BHT-甲醇溶液：称取 $0.5\mathrm{g}$ BHT(3.1.6)，用 $500~\mathrm{mL}$ 甲醇溶解。  
3.2.2 BHT-乙醇溶液：称取 $0.5\mathrm{g}$ BHT(3.1.6)，用 $500~\mathrm{mL}$ 无水乙醇溶解。  
3.2.3 $70\%$ 乙腈水溶液：量取 $350~\mathrm{mL}$ 乙腈和 $150~\mathrm{mL}$ 水，混合均匀，加入 $0.5\mathrm{g}$ BHT(3.1.6)，振荡溶解。  
3.2.4 $5\%$ 氨水乙醇溶液：量取 $50~\mathrm{mL}$ 氨水和 $950~\mathrm{mL}$ 无水乙醇，混合均匀，加入 $1\mathrm{g}$ BHT(3.1.6)，振荡溶解。  
3.2.5 BHT-石油醚溶液：称取 $0.5\mathrm{g}$ BHT(3.1.6)，用 $500~\mathrm{mL}$ 石油醚溶解。  
3.2.6 BHT-二氯甲烷溶液：称取 $0.5\mathrm{g}$ BHT(3.1.6)，用 $500~\mathrm{mL}$ 二氯甲烷溶解。  
3.2.7 $2\%$ 乙酸水溶液：量取 $20~\mathrm{mL}$ 冰乙酸，加到 $980~\mathrm{mL}$ 水中，混合均匀。

# 3.3 标准品

3.3.1 顺式构型降红木素标准品（Norbixin， $\mathrm{C_{24}H_{28}O_4}$ ，CAS号：542-40-5）：纯度 $\geqslant 98.0\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.2 顺式构型红木素标准品（Bixin， $\mathrm{C}_{25}\mathrm{H}_{30}\mathrm{O}_4$ ，CAS号：6983-79-5）：纯度 $\geqslant 98.0\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 标准储备液 $(100\mathrm{mg / L})$ ：准确称取经折算含量的降红木素标准品(3.3.1) $10.0\mathrm{mg}$ 于 $100~\mathrm{mL}$ 棕色容量瓶中，加入BHT-甲醇溶液(3.2.1)溶解并定容至刻度，混匀；准确称取经折算含量的红木素标准品(3.3.2) $10.0\mathrm{mg}$ 于 $100~\mathrm{mL}$ 棕色容量瓶中，加入BHT-乙醇溶液(3.2.2)溶解并定容至刻度，混匀。标准储备液于一 $18^{\circ}C$ 下避光保存，保存期6个月。  
3.4.2 混合标准工作溶液：分别吸取降红木素和红木素标准储备液(3.4.1) $0.05\mathrm{mL}, 0.1\mathrm{mL}, 0.2\mathrm{mL},$ $0.5\mathrm{mL}, 1.0\mathrm{mL}, 2.0\mathrm{mL}$ 和 $5.0\mathrm{mL}$ 于 $10\mathrm{mL}$ 棕色容量瓶中，加入 $70\%$ 乙腈水溶液(3.2.3)定容至刻度，混匀。降红木素和红木素混合标准系列工作液的浓度分别为 $0.5\mathrm{mg/L}, 1.0\mathrm{mg/L}, 2.0\mathrm{mg/L},$ $5.0\mathrm{mg/L}, 10\mathrm{mg/L}, 20\mathrm{mg/L}$ 和 $50\mathrm{mg/L}$ 。现配现用。  
3.4.3 含顺、反构型的降红木素和红木素溶液：准确移取 $5.0 \mathrm{~mL}$ 的降红木素和红木素标准储备液(3.4.1)，用BHT-乙醇溶液(3.2.2)稀释定容至 $50 \mathrm{~mL}$ 无色透明容量瓶中，置于阳光下直射 $2 \mathrm{~h}$ ，获得含顺、反构型的降红木素和红木素溶液。该溶液用于降红木素和红木素各异构体的定性。现配现用。

# 3.5 材料

3.5.1 硅藻土，商品名为 Celite 545 或者相当产品。  
3.5.2 $0.45\mu \mathrm{m}$ 滤膜：聚四氟乙烯材质。

# 4 仪器和设备

4.1 液相色谱仪：配有二极管阵列检测器或紫外检测器。  
4.2 电子天平：感量为 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
4.3 涡旋混合器：转速不小于 $3000\mathrm{r / min}$   
4.4 离心机：转速不小于 $3000\mathrm{r / min}$   
4.5 氮气浓缩装置。

# 5 分析步骤

# 5.1 样品前处理

# 5.1.1 试样制备

固态样品和半固态样品需粉碎或匀浆成均匀的粉状样品和黏稠状样品；液态样品摇匀。制备好的试样于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 避光保存。

# 5.1.2 试样提取

# 5.1.2.1 粉状样品

称取试样 $0.5\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 于 $50~\mathrm{mL}$ 具塞塑料离心管中，加入 $20~\mathrm{mL}~5\%$ 氨水乙醇溶液

(3.2.4)，涡旋振荡 $2\mathrm{min},8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，转移上清液于另一支 $50~\mathrm{mL}$ 离心管内，重复提取1次，合并提取液至同一离心管内，采用氮气浓缩装置 $40^{\circ}C$ 下浓缩至体积约 $10~\mathrm{mL}$ ，待净化。

# 5.1.2.2 黏稠状样品

称取试样 $0.5\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 于 $50~\mathrm{mL}$ 具塞塑料离心管中，加入 $2\mathrm{g}\sim 5\mathrm{g}$ 硅藻土，采用玻璃棒将样品和硅藻土充分混匀，搅拌至疏松颗粒状，加入 $20~\mathrm{mL}~5\%$ 氨水乙醇溶液(3.2.4)之后的步骤同5.1.2.1。

# 5.1.2.3 液态样品

称取试样 $0.5\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 于 $50~\mathrm{mL}$ 具塞塑料离心管中，加入 $5\mathrm{mL}5\%$ 氨水乙醇溶液(3.2.4)，待净化。

# 5.1.3 试样净化

在试样提取液中加入 $2\mathrm{mL}$ 氨水，用水稀释至约 $20~\mathrm{mL}$ ，加入 $20~\mathrm{mL}$ BHT-石油醚溶液(3.2.5)，涡旋振荡 $1\mathrm{min}$ ，离心弃去上层溶液，重复加入 $20~\mathrm{mL}$ BHT-石油醚溶液(3.2.5)，涡旋振荡 $1\mathrm{min}$ ，离心弃去上层溶液。在下层溶液中加入 $2\mathrm{mL}$ 冰乙酸， $25~\mathrm{mL}$ BHT-二氯甲烷溶液(3.2.6)，涡旋振荡 $1\mathrm{min}$ ，离心后弃去上层溶液，在下层二氯甲烷溶液中加入 $1\mathrm{mL}$ BHT-乙醇溶液(3.2.2)，采用氮气浓缩装置 $35^{\circ}C$ 下浓缩至体积约 $1\mathrm{mL}$ ，用BHT-乙醇溶液(3.2.2)辅助转移残留溶液至 $5\mathrm{mL}$ 棕色容量瓶中定容并摇匀，经 $0.45\mu \mathrm{m}$ 滤膜(3.5.2)过滤至进样小瓶中，供液相色谱仪测定。

注：前处理过程要注意避光。

# 5.2 仪器参考条件

5.2.1 色谱柱： $\mathrm{C_{18}}$ 色谱柱，柱长 $150~\mathrm{mm}$ ，内径 $4.6~\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或等效色谱柱。  
5.2.2 流动相：乙腈 $+2\%$ 乙酸水溶液 $= 70 + 30$   
5.2.3 流速： $1.0~\mathrm{mL / min}$   
5.2.4 柱温： $30^{\circ}C$ 。  
5.2.5 进样量： $10~\mu \mathrm{L}$   
5.2.6 检测波长： $458~\mathrm{nm}$

# 5.3 标准曲线的制作

将3.4.2配制的混合标准工作溶液分别注入液相色谱仪中（顺式降红木素和顺式红木素标准溶液的液相色谱图参见附录A中图A.1)，以保留时间定性，测定相应的峰面积，以混合标准工作溶液的质量浓度为横坐标，峰面积为纵坐标，绘制标准曲线，得到顺式降红木素和顺式红木素的标准曲线回归方程。

将3.4.3配制的含顺、反构型的降红木素和红木素溶液分别注入液相色谱仪中（含顺、反构型的降红木素和红木素溶液的液相色谱图参见附录A中图A.2和图A.3)，确定降红木素和红木素各异构体保留时间。

# 5.4 试样溶液的测定

将试样溶液注入液相色谱仪中，以保留时间定性，得到各异构体的峰面积，根据标准曲线采用外标法定量。反式构型的浓度按对应顺式构型的标准曲线回归方程计算，顺反构型结果合并计算。若试样溶液中降红木素和红木素的质量浓度不在标准曲线范围内，则应进行适当稀释后测定。

# 6 分析结果的表述

试样中红木素或降红木素的含量按式(1)计算：

$$
X _ {i} = \frac {\rho_ {i} \times V}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X_{i}$ ——试样中红木素或降红木素的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho_{i}$ ——由标准曲线得到的试样溶液中红木素或降红木素的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

1000 ——由 $\mathrm{mg/kg}$ 转换为 $\mathrm{g/kg}$ 的换算因子。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当按取样量为 $5\mathrm{g}$ ，定容体积为 $5\mathrm{mL}$ 时，本方法的检出限：降红木素为 $0.5\mathrm{mg / kg}$ ，红木素为 $1.0\mathrm{mg / kg}$ ；定量限：降红木素为 $1.0\mathrm{mg / kg}$ ，红木素为 $2.0\mathrm{mg / kg}$ 。

# 前言

本标准代替SN/T3641—2013《出口水产品中4-己基间苯二酚残留量检测方法》。

本标准与SN/T3641—2013相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中4-己基间苯二酚残留量的测定”；

——修改了前处理部分操作。

# 食品安全国家标准

# 食品中4-己基间苯二酚残留量的测定

# 1 范围

本标准规定了虾和螃蟹中4-己基间苯二酚残留量的测定方法。

本标准适用于虾和螃蟹中4-己基间苯二酚残留量的测定。

# 2 原理

样品经乙酸乙酯提取，乙腈饱和正己烷除脂，采用高效液相色谱分离、荧光检测器检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。

3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

3.1.3 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。

3.1.4 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。

3.1.5 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：将无水硫酸钠置于马弗炉内于 $400^{\circ}\mathrm{C}$ 下灼烧 $4\mathrm{h}$ 后，置于干燥器皿内备用。

# 3.2 试剂配制

3.2.1 乙腈饱和正己烷溶液：在正己烷中加入一定量的乙腈，振摇、混匀，混合液清晰分层后，上层即为乙腈饱和正己烷。上层液备用。

3.2.2 甲醇-乙腈-水混合溶液 $(1 + 2 + 1$ ，体积比)：分别量取 $100~\mathrm{mL}$ 甲醇、 $200~\mathrm{mL}$ 乙腈和 $100~\mathrm{mL}$ 水，在 $1\textrm{L}$ 玻璃烧杯中混匀，备用。

# 3.3 标准品

4-己基间苯二酚标准品 $\mathrm{(C_{12}H_{18}O_2}$ ,CAS号：136-77-6）：纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 4-己基间苯二酚标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：准确称取 $10\mathrm{mg}$ （精确至 $0.01\mathrm{mg})$ 4-己基间苯二酚标准品，用甲醇溶解并定容至 $100~\mathrm{mL}$ ，混匀。将溶液转移至玻璃容器中，于一 $18^{\circ}C$ 保存，有效期6个月。

3.4.2 4-己基间苯二酚标准中间液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：吸取标准储备液(3.4.1) $1.00~\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，加甲醇定容。临用现配。  
3.4.3 4-己基间苯二酚标准系列工作液：分别吸取4-己基间苯二酚标准中间液 $0.02\mathrm{mL},0.05\mathrm{mL}$ 、 $0.10\mathrm{mL},0.50\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，加甲醇-乙腈-水混合溶液定容至刻度。4-己基间苯二酚标准系列工作液的质量浓度分别为 $0.0200\mu \mathrm{g / mL},0.0500\mu \mathrm{g / mL},0.100\mu \mathrm{g / mL}$ $0.500\mu \mathrm{g / mL},1.00\mu \mathrm{g / mL},2.00\mu \mathrm{g / mL}$ 。临用现配。

# 4 仪器和设备

4.1 液相色谱仪：配有荧光检测器。  
4.2 天平：感量为 $0.01\mathrm{g}$ 及 $0.01\mathrm{mg}$   
4.3 旋转蒸发仪。  
4.4 涡旋混合器。  
4.5 超声波发生器。  
4.6 离心机：转速可达到 $4000\mathrm{r / min}$   
4.7 组织捣碎机。  
4.8 均质器。  
4.9 浓缩瓶。  
4.10 有机相微孔滤膜： $0.45\mu \mathrm{m}$

# 5 分析步骤

# 5.1 试样制备

取代表性样品的可食部分约 $500\mathrm{g}$ ，经组织捣碎机粉碎混匀后，装入洁净的容器中作为试样，密封并做好标识。试样于 $-18^{\circ}\mathrm{C}$ 保存。

# 5.2 试样提取

称取 $2\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 样品于 $50~\mathrm{mL}$ 离心管中，加入 $20~\mathrm{mL}$ 乙酸乙酯和 $3\mathrm{g}$ 无水硫酸钠，均质 $30~\mathrm{s}$ 后，以 $4000~\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，将上清液全部转移至 $100~\mathrm{mL}$ 浓缩瓶中，残渣再次加入 $20~\mathrm{mL}$ 乙酸乙酯，均质 $30~\mathrm{s}$ ，以 $4000~\mathrm{r / min}$ 离心 $3\mathrm{min}$ 后，将上清液与第一次获得的上清液合并于同一浓缩瓶中，于 $35^{\circ}\mathrm{C}$ 水浴中蒸发至近干。

# 5.3 试样净化

向残渣准确加入 $2\mathrm{mL}$ 甲醇-乙腈-水混合溶液复溶，超声 $10\mathrm{s}$ ，涡旋 $30\mathrm{s}$ ，残渣溶解后，加入 $2\mathrm{mL}$ 乙腈饱和正己烷溶液，涡旋 $30\mathrm{s}$ ，转移至 $15\mathrm{mL}$ 离心管后以 $4000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，弃去正己烷层后向离心管中再次加入 $2\mathrm{mL}$ 乙腈饱和正己烷溶液涡旋 $30\mathrm{s}$ ，以 $4000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，弃去上层正己烷层，取下层过 $0.45\mu \mathrm{m}$ 有机微孔滤膜过滤，滤液待上机。

# 5.4 空白试验

除不加试样外，均按5.2和5.3测定步骤进行。

# 5.5 测定

# 5.5.1 仪器参考条件

a）色谱柱： $\mathrm{C_{18}}$ 柱， $250\mathrm{mm}\times 4.6\mathrm{mm}$ (i.d.)，粒径 $5\mu \mathrm{m}$   
b）流动相：乙腈 $+$ 水 $= 65 + 35$   
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d）检测波长：激发波长 $280\mathrm{nm}$ ，发射波长 $310\mathrm{nm}$ ；  
e） 柱温： $35^{\circ}C$   
f）进样量： $10~\mu \mathrm{L}$

# 5.5.2 标准曲线的制作

将标准系列工作液分别注入高效液相色谱仪中，测定相应的峰面积，以标准系列工作液中4-己基间苯二酚的质量浓度为横坐标，以4-己基间苯二酚的峰面积为纵坐标，绘制标准曲线。4-己基间苯二酚标准溶液的色谱图参见附录A。

# 5.5.3 试样溶液的测定

将试样溶液注入高效液相色谱仪中，通过保留时间定性，得到待测物峰面积，根据标准曲线得到待测液中4-己基间苯二酚的质量浓度。

# 6 结果计算和表述

试样中4-己基间苯二酚的含量按公式(1)计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中4-己基间苯二酚的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——由标准曲线得到的试样溶液中4-己基间苯二酚的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$c_{0}$ ——由标准曲线得到的空白试验中4-己基间苯二酚的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$

$V$ ——测试样品的复溶体积，单位为毫升 $(\mathrm{mL})$ ；

1000——单位换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

计算结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当虾和螃蟹称样量为 $2.00\mathrm{g}$ 时，检出限为 $0.0200\mathrm{mg / kg}$ ，定量限为 $0.0500\mathrm{mg / kg}$ 。

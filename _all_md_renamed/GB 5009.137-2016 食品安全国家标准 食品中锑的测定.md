# 前言

本标准代替GB/T5009.137—2003《食品中锑的测定》。

本标准与GB/T5009.137—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中锑的测定”；  
——样品前处理增加了压力罐消解法；  
——还原剂增加了硫脲-抗坏血酸。

# 食品安全国家标准

# 食品中锑的测定

# 1 范围

本标准规定了食品中锑的氢化物原子荧光光谱测定方法。

本标准适用于食品中锑的测定。

# 2 原理

试样经酸加热消解后，在酸性介质中，试样中的锑与硼氢化钠或硼氢化钾反应生成挥发性的锑氢化物，以氩气为载气，将锑氢化物导入电热石英原子化器中原子化，在锑空心阴极灯照射下，基态锑原子被激发至高能态，再由高能态回到基态时，发射出特征波长的荧光，其荧光强度与锑含量成正比，根据标准系列进行定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 硝酸（ $\mathrm{HNO}_3$ ）。  
3.1.2 过氧化氢 $\mathrm{(H_2O_2)}$ 。  
3.1.3 盐酸（HCl）。  
3.1.4 硫酸 $\mathrm{(H_2SO_4)}$ 。  
3.1.5 高氯酸（ $\mathrm{HClO_4}$ ）。  
3.1.6 硫脲 $\left[\left(\mathrm{NH}_{2}\right)_{2} \mathrm{CS}\right]$ ：分析纯。  
3.1.7 碘化钾(KI)：分析纯。  
3.1.8 抗坏血酸 $\mathrm{(C_6H_8O_6)}$ ：分析纯。  
3.1.9 硼氢化钾 $\mathrm{(KBH_4)}$ 或硼氢化钠（ $\mathrm{NaBH}_4$ ）。  
3.1.10 氢氧化钾(KOH)或氢氧化钠（NaOH）。

# 3.2 试剂的配制

3.2.1 硝酸-高氯酸混合酸 $(10 + 1)$ ：分别量取硝酸 $500~\mathrm{mL}$ 与高氯酸 $50~\mathrm{mL}$ ，混匀。  
3.2.2 盐酸溶液 $(1 + 9)$ ：量取 $50~\mathrm{mL}$ 盐酸，加入到 $450~\mathrm{mL}$ 水中，混匀。  
3.2.3 硫脲-抗坏血酸溶液：分别称取 $10\mathrm{g}$ 硫脲、 $10\mathrm{g}$ 抗坏血酸，溶于 $100\mathrm{mL}$ 水中，混匀。  
3.2.4 硫脲-碘化钾溶液：分别称取 $2\mathrm{g}$ 硫脲、 $10\mathrm{g}$ 碘化钾，溶于 $100\mathrm{mL}$ 水中，混匀。  
3.2.5 氢氧化钾溶液 $(2\mathrm{g} / \mathrm{L})$ ：称取 $1\mathrm{g}$ 氢氧化钾，溶于 $500~\mathrm{mL}$ 水中，混匀，临用现配。该溶液中的氢氧化钾也可用氢氧化钠代替。  
3.2.6 硼氢化钾碱溶液 $(20\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 硼氢化钾，溶于 $500~\mathrm{mL}$ 氢氧化钾溶液 $(2\mathrm{g / L})$ 中，混匀，临

用现配。该溶液中的硼氢化钾也可用等摩尔数的硼氢化钠代替。

# 3.3 标准品

锑标准溶液： $1000\mathrm{mg / L}$ 。或其他经国家认证并授予标准物质证书的一定浓度的锑标准溶液。

# 3.4 标准溶液的配制

3.4.1 锑标准中间液 $(100\mathrm{mg / L})$ ：准确吸取 $1\mathrm{mL}$ 锑标准溶液 $(1000\mathrm{mg / L})$ 于 $10~\mathrm{mL}$ 容量瓶中，加水定容至刻度，混匀。  
3.4.2 锑标准使用液 $(1.00\mathrm{mg / L})$ ：准确吸取 $1\mathrm{mL}$ 锑标准中间液 $(100\mathrm{mg / L})$ 于 $100~\mathrm{mL}$ 容量瓶中，加水定容至刻度，混匀。  
3.4.3 锑标准系列溶液: 分别准确吸取锑标准使用液 $(1.00 \mathrm{mg} / \mathrm{L}) 0 \mathrm{~mL}, 0.100 \mathrm{~mL}, 0.200 \mathrm{~mL}, 0.400 \mathrm{~mL}, 1.00 \mathrm{~mL}$ 和 $2.00 \mathrm{~mL}$ 于 $100 \mathrm{~mL}$ 容量瓶中, 加入少量水稀释后, 加入 $10 \mathrm{~mL}$ 盐酸溶液 $(1+9)$ , $10 \mathrm{~mL}$ 硫脲-碘化钾溶液或硫脲-抗坏血酸溶液, 加水定容至刻度, 混匀。此锑标准系列溶液的质量浓度为 $0 \mu \mathrm{g} / \mathrm{L}, 1.00 \mu \mathrm{g} / \mathrm{L}, 2.00 \mu \mathrm{g} / \mathrm{L}, 4.00 \mu \mathrm{g} / \mathrm{L}, 10.0 \mu \mathrm{g} / \mathrm{L}, 20.0 \mu \mathrm{g} / \mathrm{L}$ 。放置 $30 \mathrm{~min}$ 后测定。

注：可根据仪器的灵敏度及样品中锑的实际含量确定标准系列溶液中锑元素的质量浓度范围。

# 4 仪器和设备

注：所有玻璃器皿及四氟乙烯消解内罐均需硝酸溶液 $(1 + 5)$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

4.1 原子荧光光谱仪，配锑空心阴极灯。  
4.2 天平：感量为 $1\mathrm{mg}$   
4.3 可调式电热板。  
4.4 可调式电炉。  
4.5 微波消解系统：配聚四氟乙烯消解罐。  
4.6 恒温干燥箱。

# 5 分析步骤

# 5.1 试样制备

注：在采样和试样制备过程中，应避免污染。

# 5.1.1 粮食、豆类样品

样品去除杂物后，粉碎，储于塑料瓶中。

# 5.1.2 蔬菜、水果、鱼类、肉类等水分含量高的样品

样品用水洗净，晾干，取可食部分，制成匀浆，储于塑料瓶中。

# 5.1.3 饮料、酒、醋、酱油等液体样品

将样品摇匀。

# 5.2 试样消解

# 5.2.1 湿法消解

准确称取固体试样 $0.5\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 或准确移取液体试样 $1.00~\mathrm{mL}\sim 5.00~\mathrm{mL}$ ，置于

$50~\mathrm{mL}\sim 100~\mathrm{mL}$ 消化容器中(锥形瓶)，加入硝酸-高氯酸混合酸 $(10 + 1)5\mathrm{mL}\sim 10\mathrm{mL}$ 浸泡放置过夜。次日，置于电热板上加热消解，如消解过程溶液色泽较深，稍冷后补加少量硝酸，继续消解，消解至冒白烟，消化液呈无色透明或略带黄色，加入 $20~\mathrm{mL}$ 水，再继续加热赶酸至 $0.5\mathrm{mL}\sim 1\mathrm{mL}$ 止，冷却后用少量水转入 $10~\mathrm{mL}$ 容量瓶中，加入 $2\mathrm{mL}$ 盐酸溶液 $(1 + 9)$ ，用水定容至刻度。准确吸取试样消化液 $5.00~\mathrm{mL}$ ，加入硫脲-碘化钾溶液或硫脲-抗坏血酸溶液 $1\mathrm{mL}$ ，用水稀释定容至 $10~\mathrm{mL}$ ，摇匀，放置 $30\mathrm{min}$ 后测定。同时做试剂空白试验。

# 5.2.2 微波消解

准确称取固体试样 $0.2\mathrm{g}\sim 0.8\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $1.00~\mathrm{mL}\sim 3.00~\mathrm{mL}$ ，置于微波消解罐中，加硝酸 $5\mathrm{mL}$ 、过氧化氢 $1\mathrm{mL}$ 。微波消解程序可以根据仪器型号调至最佳条件，推荐条件可参见附录A。消解完毕，待消解罐冷却后打开，加入 $20~\mathrm{mL}$ 水，加热赶酸至 $0.5\mathrm{mL}\sim 1\mathrm{mL}$ 止，用少量水分三次冲洗消解罐，将溶液转移至 $10~\mathrm{mL}$ 容量瓶中，加入 $2\mathrm{mL}$ 盐酸溶液 $(1 + 9)$ ，用水定容至刻度。准确吸取试样消化液 $5.00~\mathrm{mL}$ ，加入硫脲-碘化钾溶液或硫脲-抗坏血酸溶液 $1\mathrm{mL}$ ，用水稀释定容至 $10~\mathrm{mL}$ ，摇匀，放置 $30\mathrm{min}$ 后测定。同时做试剂空白试验。

# 5.2.3 压力罐消解

准确称取固体试样 $0.2\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $1.00~\mathrm{mL}\sim 5.00~\mathrm{mL}$ ，置于聚四氟乙烯内罐中，加硝酸 $2\mathrm{mL}\sim 4\mathrm{mL}$ 浸泡过夜。再补加硝酸 $2\mathrm{mL}\sim 4\mathrm{mL}$ 。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱， $140^{\circ}C\sim 160^{\circ}C$ 保持 $4\mathrm{h}\sim 5\mathrm{h}$ ，在箱内自然冷却至室温，开盖取出内罐，加入 $20~\mathrm{mL}$ 水，加热赶酸至 $0.5\mathrm{mL}\sim 1\mathrm{mL}$ 止，用少量水分三次冲洗消解罐，将溶液转移至 $10~\mathrm{mL}$ 容量瓶中，加入 $2\mathrm{mL}$ 盐酸溶液 $(1 + 9)$ ，用水定容至刻度。准确吸取试样消化液 $5.00~\mathrm{mL}$ ，加入硫脲-碘化钾溶液或硫脲抗坏血酸溶液 $1\mathrm{mL}$ ，用水稀释定容至 $10~\mathrm{mL}$ ，摇匀，放置 $30\mathrm{min}$ 后测定。同时做试剂空白试验。

# 5.3 仪器参考条件

调整仪器性能至最佳状态，仪器参考条件：光电倍增管电压， $300\mathrm{V}$ ；空心阴极灯电流， $60~\mathrm{mA}$ ；原子化器高度， $8\mathrm{mm}$ ；载气流速， $300~\mathrm{mL / min}$ 。根据各自仪器性能调至最佳状态。

# 5.4 标准曲线的制作

设定好仪器最佳条件，将炉温升至所需温度后，稳定 $20\mathrm{min}\sim 30\mathrm{min}$ 开始测量。以盐酸溶液 $(5\%)$ 为载流，硼氢化钾碱溶液 $(20~\mathrm{g / L})$ 为还原剂，连续用标准系列溶液的零管进样，待读数稳定之后，锑标准系列溶液按浓度由低到高的顺序分别导入仪器，测定荧光值。以锑标准系列溶液的质量浓度为横坐标，相应的荧光值为纵坐标，绘制标准曲线。

注：如有自动进样装置，也可用程序自动稀释来配制标准系列。

# 5.5 试样溶液测定

在与测定标准溶液系列相同的实验条件下，将空白溶液和试样溶液分别导入仪器，测定荧光值，与标准系列比较定量。

# 6 分析结果的表述

试样中锑的含量按式(1)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中锑的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$\rho$ ——试样溶液中锑的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——空白溶液中锑的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样消化液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样称样量或移取体积，单位为克或毫升(g或 $\mathrm{mL}$ )；

1000——换算系数。

当锑含量 $\geqslant 1.00\mathrm{mg / kg}$ （或 $\mathrm{mg / L}$ )时，计算结果保留三位有效数字，当锑含量 $<  1.00~\mathrm{mg / kg}$ （或 $\mathrm{mg / L})$ 时，计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当称样量为 $0.5\mathrm{g}$ (或 $0.5\mathrm{mL}$ )，定容体积为 $10\mathrm{mL}$ 时，本方法的检出限为 $0.01\mathrm{mg/kg}$ (或 $0.01\mathrm{mg/L}$ )，定量限为 $0.04\mathrm{mg/kg}$ (或 $0.04\mathrm{mg/L}$ )。

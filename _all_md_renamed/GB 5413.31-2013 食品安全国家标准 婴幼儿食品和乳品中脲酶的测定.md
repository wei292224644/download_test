# 前言

本标准代替GB/T5413.31—1997《婴幼儿配方食品和乳粉脲酶的定性检验》。

本标准与GB/T5413.31—1997相比，主要变化如下：

——修改了标准的名称；  
——增加了尿素溶液的贮存条件；  
——增加了纳氏试剂的贮存条件；  
——增加了判定结果的时限规定。

# 食品安全国家标准

# 婴幼儿食品和乳品中脲酶的测定

# 1 范围

本标准规定了婴幼儿食品和乳品中脲酶的测定方法。

本标准适用于婴幼儿食品和乳品中脲酶的定性检验。

# 2 原理

脲酶在适当酸碱度和温度条件下，催化尿素转化成碳酸铵。碳酸铵在碱性条件下生成氢氧化铵，与纳氏试剂中的碘化钾汞复盐作用生成棕色的碘化双汞铵。

# 3 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 尿素（ $\mathrm{H}_2\mathrm{NCONH}_2$ ）。  
3.1.2 钨酸钠（ $\mathrm{Na_2WO_4 \cdot 2H_2O}$ ）。  
3.1.3 酒石酸钾钠（ $\mathrm{C_4H_4O_6KNa\cdot 4H_2O}$ ）。  
3.1.4 硫酸（ $\mathrm{H}_2\mathrm{SO}_4$ ）。  
3.1.5 磷酸氢二钠（ $\mathrm{Na_2HPO_4}$ ）。  
3.1.6 磷酸二氢钾（ $\mathrm{KH_2PO_4}$ ）。  
3.1.7 碘化汞（ $\mathrm{HgI_2}$ ）。  
3.1.8 碘化钾（KI）。  
3.1.9 氢氧化钠（NaOH）。

# 3.2 试剂配制

3.2.1 尿素溶液（ $10\mathrm{~g/L}$ ）：称取尿素 $5\mathrm{~g}$ ，溶解于 $500\mathrm{~mL}$ 水中。保存于棕色试剂瓶中，冰箱中冷藏，有效期为1个月。  
3.2.2 钨酸钠溶液（ $100\mathrm{g / L}$ ）：称取钨酸钠 $50\mathrm{g}$ ，溶解于 $500\mathrm{mL}$ 水中。  
3.2.3 酒石酸钾钠溶液（ $20\mathrm{g / L}$ ）：称取酒石酸钾钠 $10\mathrm{g}$ ，溶解于 $500\mathrm{mL}$ 水中。  
3.2.4 硫酸溶液（ $50~\mathrm{mL / L}$ ）：吸取硫酸 $25~\mathrm{mL}$ ，溶解于 $500~\mathrm{mL}$ 水中。  
3.2.5 磷酸氢二钠溶液：称取无水磷酸氢二钠 $9.47\mathrm{g}$ ，溶于 $1000~\mathrm{mL}$ 水中。  
3.2.6 磷酸二氢钾溶液：称取磷酸二氢钾 $9.07\mathrm{g}$ ，溶于 $1000\mathrm{mL}$ 水中。  
3.2.7 中性缓冲溶液：取磷酸氢二钠溶液 $611 \mathrm{~mL}$ ，磷酸二氢钾溶液 $389 \mathrm{~mL}$ ，两种溶液混合均匀。  
3.2.8 碘化汞-碘化钾混合溶液：称取红色碘化汞 $55\mathrm{g}$ ，碘化钾 $41.25\mathrm{g}$ ，溶于 $250~\mathrm{mL}$ 水中。

3.2.9 纳氏试剂: 称取氢氧化钠 $144 \mathrm{~g}$ 溶于 $500 \mathrm{~mL}$ 水中, 充分溶解并冷却后, 再缓慢地移入 $1000 \mathrm{~mL}$ 的容量瓶中, 加入碘化汞-碘化钾混合溶液 $250 \mathrm{~mL}$ , 加水稀释至刻度, 摇匀, 转入试剂瓶内, 静置后, 用上清液。此试剂需棕色瓶保存, 冰箱中冷藏, 有效期为1个月。

# 4 仪器和设备

4.1 电子天平：感量为 $0.01\mathrm{g}$   
4.2 旋涡振荡器。  
4.3 恒温水浴锅： $40^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$

# 5 分析步骤

取试管甲、乙两支，各称入 $0.10\mathrm{g}$ 试样，再吸入 $1\mathrm{mL}$ 水，振摇 $0.5\mathrm{min}$ （约100次）。然后分别吸入 $1\mathrm{mL}$ 中性缓冲溶液。向甲管（样品管）吸入 $1\mathrm{mL}$ 尿素溶液，再向乙管（空白对照管）吸入 $1\mathrm{mL}$ 水。两管摇匀后，置于 $40^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 水浴中保温 $20\mathrm{min}$ 。从水浴中取出两管后，各吸入 $4\mathrm{mL}$ 水，摇匀，再吸入 $1\mathrm{mL}$ 钨酸钠溶液，摇匀，吸入 $1\mathrm{mL}$ 硫酸溶液，摇匀，过滤，收集滤液备用。取上述滤液 $2\mathrm{mL}$ ，分别吸入到二支 $25\mathrm{mL}$ 具塞的比色管中。再各吸入 $15\mathrm{mL}$ 水， $1\mathrm{mL}$ 酒石酸钾钠溶液，和 $2\mathrm{mL}$ 纳氏试剂，最后用水定容至 $25\mathrm{mL}$ ，摇匀。 $5\mathrm{min}$ 内观察结果。

# 7 检出限

该方法为定性法，检出限为 $0.7\mathrm{U}$ 。

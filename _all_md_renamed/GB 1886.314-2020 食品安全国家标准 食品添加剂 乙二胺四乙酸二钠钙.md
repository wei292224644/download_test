# 食品添加剂 乙二胺四乙酸二钠钙

# 1 范围

本标准适用于由乙二胺四乙酸二钠与钙(一般用碳酸钙或天然贝壳)螯合形成的食品添加剂乙二胺四乙酸二钠钙。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

乙二胺四乙酸二钠钙

# 2.2 分子式

$$
\mathrm {C} _ {1 0} \mathrm {H} _ {1 2} \mathrm {C a N} _ {2} \mathrm {N a} _ {2} \mathrm {O} _ {8} \cdot 2 \mathrm {H} _ {2} \mathrm {O}
$$

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按照GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 方法提要

用火焰试验鉴别样品中的钠离子；用草酸试验鉴别样品中的钙离子；乙二胺四乙酸二钠钙中残留乙二胺四乙酸离子可使氯化铁溶液褪色，鉴别乙二胺四乙酸离子；红外光谱是根据组成物质的离子基团在红外光范围内（中红外： $400~\mathrm{cm}^{-1}\sim 4000~\mathrm{cm}^{-1}$ )的吸收谱带，对乙二胺四乙酸二钠钙进行定性分析。

# A.2.2 试剂和材料

A.2.2.1 氢氧化铵溶液 $(6\mathrm{mol} / \mathrm{L})$ ：量取 $400~\mathrm{mL}$ 的 $28\%$ 氢氧化铵溶液用水稀释至 $1000~\mathrm{mL}$ 。  
A.2.2.2 草酸铵溶液 $(0.5\mathrm{mol} / \mathrm{L})$ ：取 $3.5\mathrm{g}$ 一水合草酸铵溶于水中至 $100~\mathrm{mL}$ 。  
A.2.2.3 硫氰酸铵溶液 $(0.02\mathrm{g / mL})$ ：称取 $2\mathrm{g}$ 硫氰酸铵溶于水中至 $100~\mathrm{mL}$ 。  
A.2.2.4 盐酸溶液 $(3\mathrm{mol} / \mathrm{L})$ ：量取 $226~\mathrm{mL}$ 的盐酸用水稀释至 $1000~\mathrm{mL}$ 。  
A.2.2.5 氯化铁溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $9\mathrm{g}$ 的六水氯化铁溶于水至 $100~\mathrm{mL}$ 。  
A.2.2.6 甲基红溶液：称取 $100\mathrm{mg}$ 甲基红溶于 $100~\mathrm{mL}$ 乙醇，必要时过滤。

# A.2.3 仪器和设备

红外分光光度计。

# A.2.4 鉴别方法

A.2.4.1 称取 $0.2\mathrm{g}$ 试样，配制试样溶液 $(50\mathrm{mg / mL})$ ，取铂丝，用盐酸溶液湿润后，蘸取试样溶液，在无色火焰中燃烧，火焰显亮黄色。  
A.2.4.2 取 A.2.4.1 中的试样溶液 $(50\mathrm{mg / mL}) 5\mathrm{mL}$ , 加入 2 滴甲基红溶液作为指示剂, 用氢氧化铵溶液调至中性。然后用盐酸溶液将试样溶液滴定至酸性, 加入草酸铵溶液即产生白色沉淀物草酸钙, 草酸钙不溶于醋酸, 溶于盐酸。用盐酸溶液湿润后, 蘸取试样溶液, 在无色火焰中燃烧, 火焰显黄红色。  
A.2.4.3 取 $5 \mathrm{~mL}$ 水加入试管中，加入 2 滴硫氰酸铵溶液和 2 滴氯化铁溶液，得到深红色溶液，再加入 $50 \mathrm{mg}$ 的样品，混合后，深红色消失。  
A.2.4.4 乙二胺四乙酸二钠钙的红外光谱图见附录B。

# A.3 乙二胺四乙酸二钠钙 $(\mathrm{C_{10}H_{12}CaN_2Na_2O_8}\cdot 2\mathrm{H_2O})$ 含量的测定

# A.3.1 方法提要

乙二胺四乙酸二钠钙在酸性条件下，以二苯卡巴腙为指示剂，硝酸汞溶液滴定至紫红色即为终点，

以消耗硝酸汞的量计算乙二胺四乙酸二钠钙含量。

# A.3.2 试剂和材料

A.3.2.1 硝酸汞溶液。  
A.3.2.2 硝酸溶液。  
A.3.2.3 硫酸铁铵溶液：称取 $8\mathrm{g}$ 十二水硫酸铁铵溶于水至 $100~\mathrm{mL}$   
A.3.2.4 硫氰酸铵标准滴定溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $8\mathrm{g}$ 硫氰酸铵溶于水至 $1000~\mathrm{mL}$ ，根据GB/T601要求标定。  
A.3.2.5 $6 \%$ 乙酸溶液：取冰乙酸 $60~\mathrm{mL}$ 或乙酸 $166~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。  
A.3.2.6 二苯卡巴腙溶液：称取 $1\mathrm{g}$ 二苯卡巴腙溶于乙醇至 $100~\mathrm{mL}$ ，贮存于棕色瓶中。

# A.3.3 分析步骤

A.3.3.1 硝酸汞溶液 $(0.1\mathrm{mol} / \mathrm{L})$ 制备：取硝酸汞约 $35\mathrm{g}$ ，溶于 $1\%$ 硝酸溶液中，再加水稀释到 $1000~\mathrm{mL}$ 。

按如下方式进行标定：精确量取此溶液 $20~\mathrm{mL}$ ，移入锥形瓶中，加硝酸 $2\mathrm{mL}$ 和硫酸铁铵溶液 $2\mathrm{mL}$ 。冷却至 $20^{\circ}\mathrm{C}$ 以下，用硫氰酸铵 $(0.1\mathrm{mol} / \mathrm{L})$ 滴定至棕色，且 $30\mathrm{s}$ 不褪色。

硝酸汞溶液的浓度 $c$ ，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ，按式(A.1)计算：

$$
c = \frac {1}{2} \times \frac {c ^ {\prime} \times V _ {1}}{2 0} \quad \dots \tag {A.1}
$$

式中：

1/2 —— $1\mathrm{mol}$ 硫氰酸铵相当于 $1 / 2\mathrm{mol}$ 硝酸汞；

$c^{\prime}$ ——硫氰酸铵标准滴定溶液浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{1}$ ——消耗硫氰酸铵标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

20 ——硝酸汞溶液体积，单位为毫升 $(\mathrm{mL})$ 。

A.3.3.2 测试：精确称取样品约 $1.2\mathrm{g}$ ，于 $250~\mathrm{mL}$ 的烧杯中，加水 $75~\mathrm{mL}$ 溶解，加稀醋酸溶液 $25~\mathrm{mL}$ 和二苯卡巴腙溶液 $1.0~\mathrm{mL}$ 。用 $0.1\mathrm{mol} / \mathrm{L}$ 硝酸汞溶液缓慢滴定至紫红色。

# A.3.4 结果计算

乙二胺四乙酸二钠钙含量 $x$ ，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ，按式(A.2)计算：

$$
x = \frac {c \times V _ {2} \times 0.4103}{w} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$c$ ——硝酸汞溶液的浓度，单位摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{2}$ ——滴定过程中消耗的硝酸汞溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

0.4103——每毫升硝酸汞 $(1\mathrm{mol} / \mathrm{L})$ 相当于本品 $\mathrm{(C_{10}H_{12}CaN_2Na_2O_8\cdot 2H_2O)0.4103g}$

$\mathcal{W}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。

# A.4 镁螯合物转移试验

# A.4.1 方法提要

样品中残留的乙二胺四乙酸游离单体在碱性条件下，优先与镁离子形成稳定络合物，当过量的镁离子与酸性酪黑T络合时，生成深酒红色络合物。

# A.4.2 试剂和材料

A.4.2.1 浓氨水： $28\%$ 氨水。  
A.4.2.2 酸性媒介黑试液：称取酸性酪黑T约 $200\mathrm{mg}$ 和盐酸羟氨 $2\mathrm{g}$ 溶于甲醇至 $50~\mathrm{mL}$ ，过滤，贮存于遮光容器中，2周内使用。  
A.4.2.3 醋酸镁溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：精确称取 $1.1\mathrm{g}$ 四水醋酸镁溶于水至 $100~\mathrm{mL}$ 。

# A.4.3 分析步骤

# A.4.3.1 缓冲溶液制备

精确称取氯化铵 $67.5\mathrm{g}$ ，溶于 $200~\mathrm{mL}$ 水后，加入 $570~\mathrm{mL}$ 浓氨水，并加水稀释至 $1000~\mathrm{mL}$ 。

# A.4.3.2 测试

准确称量 $1\mathrm{g}$ 样品，溶解在 $5\mathrm{mL}$ 的水中，添加缓冲溶液 $5\mathrm{mL}$ ，再加入酸性媒介黑试液5滴，以醋酸镁溶液滴定至呈深酒红颜色。醋酸镁溶液需要量 $\leqslant 2\mathrm{mL}$ 为通过。

# A.5 氨基三乙酸的测定

# A.5.1 方法提要

氨基三乙酸与铜离子络合，在离子对色谱条件下与乙二胺四乙酸二钠钙基线分离，通过试样加标溶液定性，并确定限量。

# A.5.2 试剂和材料

硝酸铜溶液 $(10\mathrm{mg / mL})$ ：称取 $1\mathrm{g}$ 硝酸铜溶于水至 $100~\mathrm{mL}$ 。

# A.5.3 仪器和设备

高效液相色谱仪：配备紫外检测器，检测波长为 $254\mathrm{nm}$

# A.5.4 色谱参考条件

A.5.4.1 色谱柱： $4.6\mathrm{mm}\times 15\mathrm{cm}$ 柱， $5\mu \mathrm{m}$ ；C18填料或其他等效的色谱柱。

A.5.4.2 流速： $2\mathrm{mL} / \mathrm{min}$

A.5.4.3 进样量： $20~\mu \mathrm{L}$

A.5.4.4 氨基三乙酸和乙二胺四乙酸二钠钙的色谱保留时间分别约为 $3.5\mathrm{min}$ 和 $11.9\mathrm{min}$ ，氨基三乙酸和乙二胺四乙酸二钠钙的标准色谱图见附录C。

# A.5.5 分析步骤

# A.5.5.1 流动相的制备

量取 $10\mathrm{mL}$ 的 $25\%$ 四丁基氢氧化铵溶液于 $200~\mathrm{mL}$ 水中，用磷酸 $(1\mathrm{mol} / \mathrm{L})$ 调整溶液 $\mathrm{pH}$ 至 $7.5 \pm 0.1$ 。将此溶液转移至 $1000~\mathrm{mL}$ 容量瓶中，加入 $90~\mathrm{mL}$ 甲醇，再加水稀释至刻度，混匀，用 $0.5\mu \mathrm{m}$ 薄膜过滤，脱气后备用。

# A.5.5.2 标准贮备溶液的制备

准确称取 $100\mathrm{mg}$ 氨基三乙酸，转移至 $10~\mathrm{mL}$ 容量瓶中，加入氨水 $0.5\mathrm{mL}$ ，混合。加水稀释至刻

度，浓度为 $10\mathrm{mg / mL}$

# A.5.5.3 标准使用溶液的制备

准确量取 $100\mu \mathrm{L}$ 标准贮备溶液(A.5.5.2)至 $100~\mathrm{mL}$ 容量瓶中，用硝酸铜溶液稀释至刻度，浓度为 $10~\mu \mathrm{g / mL}$ 。

# A.5.5.4 试样溶液的制备

准确称取 $1.0\mathrm{g}$ 试样转移至 $100~\mathrm{mL}$ 容量瓶中，用硝酸铜溶液稀释至刻度，混匀，如果需要可进行超声溶解。

# A.5.5.5 试样加标溶液的制备

准确称取 $1.0\mathrm{g}$ 试样转移至 $100~\mathrm{mL}$ 容量瓶中，加入 $100~\mu \mathrm{L}$ 标准贮备溶液，用硝酸铜溶液稀释至刻度，混匀，如果需要可进行超声溶解。

# A.5.5.6 系统适应性试验

将标准溶液重复3次注入色谱仪，记录峰响应值。标准试样的3次测定结果的相对标准偏差 $\leqslant 2.0\%$ ，氨基三乙酸和乙二胺四乙酸二钠钙的分离度 $\geqslant 4.0$ 。

# A.5.5.7 测定

在A.5.4参考色谱条件下，分别取等体积 $(20~\mu \mathrm{L})$ 试样加标溶液和试样溶液进行色谱分析，记录色谱图，计算氨基三乙酸的响应值。

# A.5.6 结果判定

试样溶液中氨基三乙酸色谱峰的响应值应不超过试样加标溶液和试样溶液所得氨基三乙酸色谱峰的响应值之差。

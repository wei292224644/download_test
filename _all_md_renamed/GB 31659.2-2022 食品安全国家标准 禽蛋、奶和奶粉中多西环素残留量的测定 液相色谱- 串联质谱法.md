# 食品安全国家标准

# 禽蛋、奶和奶粉中多西环素残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了禽蛋、奶和奶粉中多西环素残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鸡蛋、鸭蛋、鹅蛋、牛奶粉、羊奶粉、生奶和羊奶中多西环素残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的多西环素，经Meltvaline $\mathrm{Na_2EDTA}$ 缓冲液提取，HLB柱净化，液相色谱-串联质谱法测定，外标法定量。

# 5 试剂和材料

# 5.1 试剂

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯  
5.1.3 甲酸（HCOOH）：色谱纯  
5.1.4 一水柠檬酸 $(\mathrm{C_6H_8O_7}\cdot \mathrm{H_2O})$   
5.1.5 十二水磷酸氢二钠 $\mathrm{(Na_2HPO_4\cdot 12H_2O)}$ 。  
5.1.6 二水乙二胺四乙酸二钠 $(\mathrm{C_{10}H_{14}N_2Na_2O_8}\cdot 2\mathrm{H_2O})$   
5.1.7 氢氧化钠(NaOH)。

# 5.2 溶液配制

5.2.1 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：取氢氧化钠 $4\mathrm{g}$ ，加水溶解并稀释至 $100~\mathrm{mL}$ ，混匀。  
5.2.2 Mcllvaine- $\mathrm{Na_2}$ EDTA缓冲液：取一水柠檬酸 $12.9\mathrm{g}$ 、十二水磷酸氢二钠 $27.6\mathrm{g}$ 、二水乙二胺四乙酸二钠 $37.2\mathrm{g}$ ，加水 $900~\mathrm{mL}$ 使溶解，用氢氧化钠溶液调 $\mathrm{pH}$ 至 $4.0\pm 0.5$ ，加水稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.3 $0.1\%$ 甲酸溶液：取甲酸 $500~\mu \mathrm{L}$ ，用水稀释至 $500~\mathrm{mL}$ ，混匀。  
5.2.4 $5\%$ 甲醇溶液：取甲醇 $5\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。  
5.2.5 $30\%$ 甲醇溶液：取甲醇 $30~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。

# 5.3 标准品

盐酸多西环素（doxycycline hydrochloride， $\mathrm{C}_{22}\mathrm{H}_{24}\mathrm{N}_2\mathrm{O}_8$ ·HCl，CAS号：10592-13-9），含量 $\geqslant 98.7\%$ 。

# 5.4 标准溶液制备

5.4.1 标准储备液：取盐酸多西环素适量（相当于多西环素 $10\mathrm{mg}$ ），精密称定，加甲醇适量使溶解并定容至 $10\mathrm{mL}$ 容量瓶，配制成浓度为 $1\mathrm{mg/mL}$ 的标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期1个月。  
5.4.2 标准中间液I：准确量取标准储备液 $0.1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶，用 $30\%$ 甲醇稀释至刻度，混匀，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的标准中间液I。现用现配。  
5.4.3 标准中间液Ⅱ：准确量取中间液 $\mathrm{I}1.0\mathrm{mL}$ ，于 $10~\mathrm{mL}$ 容量瓶中，用 $30\%$ 甲醇稀释至刻度，混匀，配制成浓度为 $1000\mathrm{ng / mL}$ 的标准中间液Ⅱ。现用现配。  
5.4.4 系列标准工作液：分别准确量取标准中间液 $\mathrm{II}0.1\mathrm{mL}, 0.2\mathrm{mL}, 0.5\mathrm{mL}, 1.0\mathrm{mL}, 2.0\mathrm{mL}$ 于 $10\mathrm{mL}$ 容量瓶中，用 $30\%$ 甲醇稀释至刻度，混匀，配制成浓度分别为 $10\mathrm{ng/mL}, 20\mathrm{ng/mL}, 50\mathrm{ng/mL}, 100\mathrm{ng/mL}, 200\mathrm{ng/mL}$ 的系列标准工作液。现用现配。

# 5.5 材料

5.5.1 固相萃取柱：亲水亲脂平衡型固相萃取柱 $60\mathrm{mg / 3mL}$   
5.5.2 微孔尼龙滤膜： $0.22\mu \mathrm{m}$   
5.5.3 定性快速滤纸。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源(ESI)。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 涡旋混合器。  
6.4 涡旋振荡器。  
6.5 高速冷冻离心机：转速可达 $14000\mathrm{r / min}$   
6.6 固相萃取装置。  
6.7 氮吹仪。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜的空白或供试禽蛋，去壳并均质。

取适量新鲜或解冻的空白或供试奶，混合均匀。

取适量新鲜的空白或供试奶粉，混合均匀。

a）取均质的供试样品，作为供试试样；  
b）取均质的空白样品，作为空白试样；  
c）取均质的空白样品，添加适宜浓度的标准溶液，作为空白添加试样。

# 7.2 试样的保存

禽蛋、奶一 $18^{\circ}C$ 以下保存，奶粉常温避光保存。

# 8 测定步骤

# 8.1 提取

禽蛋、奶：取试料 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 聚丙烯离心管中，加入Mcllvaine- $\mathrm{Na_2EDTA}$ 缓冲液 $8\mathrm{mL}$ ，振荡 $10\mathrm{min},4^{\circ}C$ 下 $14000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，吸取上层液体于另一 $50~\mathrm{mL}$ 离心管中。用Mcllvaine- $\mathrm{Na_2EDTA}$ 缓冲液重复提取2次，每次 $8\mathrm{mL}$ ，合并提取液， $4^{\circ}C$ 下 $14000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液备用(奶试样上清液经滤纸过滤后备用）。

奶粉：取试料 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 聚丙烯离心管中，加入Mcllvaine- $\mathrm{Na_2}$ EDTA缓冲液 $10~\mathrm{mL}$ ，振荡 $10\mathrm{min},4^{\circ}C$ 下 $14000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，吸取上清液于另一 $50~\mathrm{mL}$ 离心管。用Mcllvaine-

$\mathrm{Na_2EDTA}$ 缓冲液重复提取2次，每次 $10~\mathrm{mL}$ ，合并提取液， $4^{\circ}\mathrm{C}$ 下 $14000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，上清液过滤纸后备用。

# 8.2 净化

取固相萃取柱，依次用甲醇、水各 $3\mathrm{mL}$ 活化。取备用液，过柱，用水 $3\mathrm{mL},5\%$ 甲醇 $3\mathrm{mL}$ 淋洗，抽干，加甲醇 $3\mathrm{mL}$ 洗脱，抽干，收集洗脱液，于 $40^{\circ}C$ 下氮气吹至液体小于 $1\mathrm{mL}$ ，加 $30\%$ 甲醇至 $1.0\mathrm{mL}$ ，涡旋匀，过微孔尼龙滤膜，供液相色谱-串联质谱仪测定。

# 8.3 基质匹配标准曲线的制备

准确移取系列标准工作液各 $100\mu \mathrm{L}$ 于经 $8.1\sim 8.2$ 步骤处理所得空白洗脱液中， $40^{\circ}C$ 下氮气吹至液体小于 $1\mathrm{mL}$ ，加 $30\%$ 甲醇至 $1.0\mathrm{mL}$ ，涡旋混匀，配制成浓度为 $1.0\mathrm{ng / mL},2.0\mathrm{ng / mL},5.0\mathrm{ng / mL},$ $10.0\mathrm{ng / mL},20.0\mathrm{ng / mL}$ 的系列基质匹配标准溶液，过微孔尼龙滤膜，供液相色谱-串联质谱仪测定。以多西环素特征离子质量色谱峰面积为纵坐标、标准溶液浓度为横坐标，绘制基质匹配标准曲线。

# 8.4 测定

# 8.4.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $100\mathrm{mm}$ ，内径 $2.1\mathrm{mm}$ ，粒径 $1.7\mu \mathrm{m}$ ，或相当者；  
b) 流动相：A为乙腈，B为 $0.1\%$ 甲酸溶液；  
c) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ ;   
d) 进样量: $10 \mu \mathrm{L}$ ;   
e) 柱温: $30^{\circ} \mathrm{C}$ ;   
f）流动相梯度洗脱程序见表1。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在同样测试条件下，试料溶液中多西环素的保留时间与基质匹配标准工作液中多西环素的保留时间相对偏差在 $\pm 2.5\%$ 以内，且检测到的离子的相对丰度，应当与浓度相当的基质匹配标准溶液相对丰度一致。其允许偏差应符合表3的要求。

# 8.4.3.2 定量测定

取试料溶液和基质匹配标准工作液，作单点或多点校准，按外标法以峰面积定量，基质匹配标准工作液及试料溶液中的多西环素响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，空白鸡蛋基质匹配标准溶液特征离子质量色谱图见附录A。

# 8.5 空白试验

取空白试料, 除不加药物外, 采用完全相同的测定步骤进行测定。

# 9 结果计算和表述

试样中多西环素的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {s}} \times A _ {i} \times V}{A _ {\mathrm {s}} \times m} \tag {1}
$$

式中：

$X$ ——试样中多西环素残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{s}}$ 基质匹配标准溶液中多西环素浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$A_{i}$ ——试样溶液中多西环素峰面积；

$V$ ——溶解残余物所用溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ 基质匹配标准溶液中多西环素峰面积；

$m$ ——试样质量的数值，单位为克（g）。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $1\mu \mathrm{g / kg}$ ，定量限为 $2\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $2\mu \mathrm{g} / \mathrm{kg}\sim 10~\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

# 前言

本标准代替GB5009.43—2016《食品安全国家标准味精中氨酸钠(谷氨酸钠)的测定》。

本标准与GB5009.43—2016相比，主要变化如下：

——修改了第一法化学指示剂法滴定终点；

——删除了第三法酸度计法。

# 食品安全国家标准

# 味精中谷氨酸钠的测定

# 1 范围

本标准规定了味精中谷氨酸钠的测定方法。

本标准适用于味精中谷氨酸钠的测定。

# 第一法 高氯酸非水溶液滴定法

# 2 原理

谷氨酸钠的碱度在乙酸溶液中显著增强，用高氯酸标准溶液滴定样品中的谷氨酸钠，采用电位滴定法测定时，以电位突跃为依据判定滴定终点；采用化学指示剂法测定时，以 $\alpha$ -萘酚苯基甲醇为指示剂，滴定样品溶液至浅绿色为其终点，通过消耗高氯酸标准滴定溶液的量计算味精中谷氨酸钠的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 甲酸 $(\mathrm{CH}_2\mathrm{O}_2)\geqslant 99\%$   
3.1.2 乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)\geqslant 99\%$   
3.1.3 高氯酸 $\mathrm{(HClO_4)}$   
3.1.4 $\alpha$ -萘酚苯基甲醇 $(\mathrm{C}_{27}\mathrm{H}_{18}\mathrm{O}_2)$ 。

# 3.2 试剂配制

$\alpha^{-}$ 萘酚苯基甲醇-乙酸指示液 $(2\mathrm{g} / \mathrm{L})$ ：称取 $0.1\mathrm{g}\alpha^{-}$ 萘酚苯基甲醇，用乙酸溶解并稀释至 $50~\mathrm{mL}$ ，避光保存，有效期为60天。

# 3.3 标准溶液

高氯酸标准滴定溶液 $\left[c\left(\mathrm{HClO}_4\right) = 0.1\mathrm{mol} / \mathrm{L}\right]$ ：按GB/T601配制与标定或购买有证书标准滴定溶液。

# 4 仪器和设备

4.1 电位滴定仪（精度 $\leqslant 0.2\mathrm{mV}$ )，具备动态滴定模式或等量滴定模式。  
4.2 非水相 $\mathrm{pH}$ 电极，采用 $\mathrm{Ag / AgCl}$ 为内参比电极；内参比电解液为 $2\mathrm{mol} / \mathrm{L}$ 氯化锂乙醇溶液或 $0.4\mathrm{mol} / \mathrm{L}$ 四乙基溴化铵乙二醇溶液。

4.3 磁力搅拌器。  
4.4 分析天平：感量 $0.1\mathrm{mg}$   
4.5 酸式滴定管。  
4.6 超声清洗器。  
4.7 粉碎机。  
4.8 温度计。

# 5 分析步骤

# 5.1 试样制备

取 $100\mathrm{g}$ 的味精样品用粉碎机磨碎均匀。

# 5.2 试样前处理

称取试样 $0.15\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )至 $100~\mathrm{mL}$ 烧杯中，加甲酸 $3\mathrm{mL}$ ，超声至完全溶解，再加乙酸 $40~\mathrm{mL}$ ，摇匀，制备成待测试液，可采用电位滴定法或化学指示剂法测定。

# 5.2.1 电位滴定法测定

# 5.2.1.1 参考条件

参考条件如下：

a）样品测定：采用动态或等量滴定模式；  
b）试剂空白测定：采用动态或等量滴定模式。

# 5.2.1.2 样品测定

将盛有待测试液的烧杯置于磁力搅拌器上，插入电极和滴定管，使电极隔膜完全浸没到被滴定的溶液中，打开电极上部的密封塞，开启磁力搅拌器，启动滴定方法，用高氯酸标准滴定溶液进行滴定，以出现明显突跃点为滴定终点，记录该点所对应的消耗高氯酸标准滴定溶液的体积 $(V_{1})$ ，同时做空白试验，记录消耗高氯酸标准滴定溶液的体积 $(V_{0})$ ；谷氨酸钠电位滴定曲线图见附录A，另取一个烧杯倒入少量高氯酸标准滴定溶液，采用温度计测定其溶液温度并记录。

# 5.2.2 化学指示剂法测定

在盛有待测试液烧杯中加入 $\alpha$ -萘酚苯基甲醇-乙酸指示液10滴，开动磁力搅拌器，用高氯酸标准滴定溶液滴定试样液，当颜色变为浅绿色即为滴定终点，记录消耗高氯酸标准滴定溶液的体积 $(V_{1})$ ，同时做空白试验，记录消耗高氯酸标准滴定溶液的体积 $(V_{0})$ ，另取一个烧杯倒入少量高氯酸标准滴定溶液，采用温度计测定其溶液温度并记录。

# 6 分析结果的表述

# 6.1 高氯酸标准溶液浓度的校正

若滴定试样时高氯酸标准溶液的温度，与标定高氯酸标准溶液时温度之差超过 $4^{\circ}\mathrm{C}$ 时，则应重新标定高氯酸标准溶液的浓度；若不超过 $4^{\circ}\mathrm{C}$ ，则按式(1)加以校正。

$$
c _ {1} = \frac {c _ {0}}{1 + 0 . 0 0 1 1 \times \left(t _ {1} - t _ {0}\right)} \quad \dots \tag {1}
$$

式中：

$c_{1}$ ——滴定试样时高氯酸溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$c_{0}$ —标定时高氯酸溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$

$t_1$ ——滴定试样时高氯酸溶液的温度，单位为摄氏度 $(^{\circ}\mathrm{C})$ ；

$t_0$ —标定时高氯酸溶液的温度，单位为摄氏度 $(^{\circ}\mathrm{C})$

0.001 1——高氯酸标准滴定溶液每改变 $1^{\circ}\mathrm{C}$ 时的体积膨胀系数，单位为每摄氏度 $(\mathrm{^\circ C^{-1}})$ 。

# 6.2 计算

样品中谷氨酸钠的含量按式(2)计算。

$$
X = \frac {0 . 0 9 3 5 7 \times \left(V _ {1} - V _ {0}\right) \times c _ {1}}{m} \times 1 0 0 \tag {2}
$$

式中：

$X$ ——样品中谷氨酸钠的含量（含1分子结晶水）， $\%$ ；

0.09357—1.00mL高氯酸标准溶液 $[c(\mathrm{HClO}_4) = 1.000\mathrm{mol / L}]$ 相当于谷氨酸钠 $(\mathrm{C_5H_8NNaO_4}\cdot \mathrm{H_2O})$ 的质量，单位为克每摩尔 $(\mathrm{g / mol})$

$V_{1}$ ——试样消耗高氯酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗高氯酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——高氯酸标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——试样质量，单位为克(g)；

100 —换算系数。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过其算术平均值的 $0.5\%$ 。

# 第二法 旋光法

# 8 原理

谷氨酸钠分子结构中含有一个不对称碳原子，具有光学活性，能使偏振光面旋转一定角度，用旋光仪测定旋光度，根据旋光度换算谷氨酸钠的含量。

# 9 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，为GB/T6682规定的二级水。

盐酸（HCl）。

# 10 仪器和设备

10.1 旋光仪：精度 $\pm 0.010^{\circ}$ ，备有钠光灯（钠光谱D线 $589.3\mathrm{nm}$ ）。  
10.2 分析天平：感量 $0.1\mathrm{mg}$

# 11 分析步骤

# 11.1 试样制备

同5.1。

# 11.2 试样前处理

称取试样 $10\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ，加少量水溶解并转移至 $100~\mathrm{mL}$ 容量瓶中，加盐酸 $20~\mathrm{mL}$ ，混匀并冷却至 $20^{\circ}C$ ，用水定容并摇匀。

# 11.3 试样溶液的测定

于 $20^{\circ}\mathrm{C} \pm 0.5^{\circ}\mathrm{C}$ ，用标准旋光角校正仪器；将11.2试液置于旋光管中（不得有气泡），观测其旋光度，同时记录旋光管中试液的温度。

# 12 分析结果的表述

样品中谷氨酸钠的含量按式(3)计算。

$$
X = \frac {\frac {\alpha}{L \times c}}{2 5 . 1 6 + 0 . 0 4 7 (2 0 - T)} \times 1 0 0 \quad \dots \tag {3}
$$

式中：

$X$ ——样品中谷氨酸钠的含量(含1分子结晶水),单位为质量百分比 $(\%)$ ;

$\alpha$ ——实测试液的旋光度，单位为度 $(^{\circ})$

$L$ ——旋光管长度(液层厚度), 单位为分米 $(\mathrm{dm})$ ;

$c$ ——1 mL 试样液中味精的质量，单位为克每毫升 $(\mathrm{g} / \mathrm{mL})$ ；

25.16—谷氨酸钠的比旋光度 $[\alpha]_{\mathrm{D}}^{20}$ ，单位为度 $(^{\circ})$ ；

0.047 温度校正系数；

20 测试液的设定温度，单位为摄氏度 $(^{\circ}\mathrm{C})$

$T$ ——测试液的实测温度，单位为摄氏度 $(^{\circ}\mathrm{C})$ ；

100 ——换算系数。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留3位有效数字。

# 13 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过其算术平均值的 $0.5\%$ 。

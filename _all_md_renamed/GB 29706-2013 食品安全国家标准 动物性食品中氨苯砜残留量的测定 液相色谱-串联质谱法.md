# 液相色谱-串联质谱法

# 1 范围

本标准规定了动物性食品中氨苯砜残留量检测的制样和液相色谱-串联质谱测定方法。

本标准适用于猪和牛的肌肉及肝组织中氨苯砜残留量的检测。

# 3原理

试料中残留的氨苯砜，用乙腈萃取，正己烷除脂，MCX柱净化，液相色谱-串联质谱测定，外标法定量。

# 4 试剂和材料

以下所用的试剂，除特别注明外均为分析纯试剂，水为符合GB/T6682规定的一级水。

4.1 氨苯砜标准品：含量 $\geq 98\%$   
4.2 乙腈：色谱纯。  
4.3 正己烷  
4.4 浓氨水  
4.5 盐酸  
4.6 MCX固相萃取柱： $60\mathrm{mg / 3mL}$ ，或相当者。  
4.7 $1\mathrm{mol} / \mathrm{L}$ 盐酸溶液：取浓盐酸 $8.4\mathrm{mL}$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。  
4.8 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液：取浓盐酸 $0.84~\mathrm{mL}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
4.9 5 mmol/L 乙酸铵溶液：取乙酸铵 $0.385\mathrm{g}$ ，用水溶解并稀释至 $1000\mathrm{mL}$ ，滤膜过滤。  
4.10 $5\%$ 氨化乙腈：取浓氨水 $5 \mathrm{~mL}$ ，用乙腈溶解并稀释至 $100 \mathrm{~mL}$ 。  
4.11 $1 \mathrm{mg} / \mathrm{mL}$ 氨苯砜标准贮备液：精密称取氨苯砜 $10 \mathrm{mg}$ ，于 $10 \mathrm{~mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的氨苯砜标准贮备液。-20°C 避光保存，有效期 6 个月。  
4.12 $10 \mu \mathrm{g} / \mathrm{mL}$ 氨苯砜标准工作液：精密量取 $1 \mathrm{mg} / \mathrm{mL}$ 氨苯砜标准贮备液 $0.1 \mathrm{~mL}$ ，于 $10 \mathrm{~mL}$ 量瓶中，用甲醇稀释至刻度，配制成浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 的氨苯砜标准工作液。-20°C 避光保存，

有效期6个月。

4.13 $100 \mathrm{ng} / \mathrm{mL}$ 氨苯砜标准工作液: 精密量取 $10 \mu \mathrm{g} / \mathrm{mL}$ 氨苯砜标准工作液 $0.1 \mathrm{~mL}$ , 于 $10 \mathrm{~mL}$ 量瓶中, 用甲醇稀释至刻度, 配制成浓度为 $100 \mathrm{ng} / \mathrm{mL}$ 的氨苯砜标准工作液。 $-20^{\circ} \mathrm{C}$ 避光保存, 有效期 6 个月。

# 5 仪器和设备

5.1 液相色谱-串联质谱仪：配有ESI源。  
5.2 分析天平：感量 $0.00001\mathrm{g}$   
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 离心机  
5.5 氮气吹干装置  
5.6 旋涡混合器  
5.7 固相萃取装置  
5.8 超声波清洗机  
5.9 均质机  
5.10 滤膜： $0.22\mu \mathrm{m}$   
5.11 离心管： $50~\mathrm{mL}$

# 6 试料的制备与保存

# 6.1 试料的制备

取新鲜或冷冻空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 样品的保存

-20℃以下保存。

# 7 测定步骤

# 7.1 提取

称取试料（ $2 \pm 0.05$ ）g，于离心管中，加乙腈 $10 \mathrm{~mL}$ ，旋涡混合 $2 \mathrm{~min}$ ，加正己烷 $5 \mathrm{~mL}$ ，漩涡混合 $1 \mathrm{~min}$ ，超声 $5 \mathrm{~min}$ ， $5000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，弃上层正己烷液。取乙腈液于另一离心管中，加

$1\mathrm{mol} / \mathrm{L}$ 盐酸溶液 $10~\mathrm{mL}$ ，旋涡混匀，备用。

# 7.2 净化

MCX 柱依次用甲醇 $5 \mathrm{~mL}$ 和水 $5 \mathrm{~mL}$ 活化, 取备用液过柱, 控制流速小于 $1 \mathrm{~mL} / \mathrm{min}$ , 用 $0.1 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液 $5 \mathrm{~mL}$ 和甲醇 $5 \mathrm{~mL}$ 淋洗, 抽干, 用 $5 \%$ 氨化乙腈 $5 \mathrm{~mL}$ 洗脱, 于 $40^{\circ} \mathrm{C}$ 水浴氮气吹干, 用流动相 $1.0 \mathrm{~mL}$ 溶解残余物, 旋涡混匀, 供液相色谱-串联质谱测定。

# 7.3 标准曲线的制备

精密量取 $100 \mathrm{ng} / \mathrm{mL}$ 氨苯砜标准工作液适量, 用甲醇稀释, 配制成浓度分别为 0.5、1、2、5、10 和 $20 \mu \mathrm{g} / \mathrm{L}$ 的系列标准溶液。供液相色谱-串联质谱测定。以特征离子质量色谱峰面积为纵坐标, 对照溶液浓度为横坐标, 绘制标准曲线, 求回归方程和相关系数。

# 7.4 测定

# 7.4.1 液相色谱条件

色谱柱： $\mathrm{C_{18}}$ （ $150 \mathrm{~mm} \times 2.1 \mathrm{~mm}$ ，粒径 $5 \mu \mathrm{m}$ ），或相当者；

流动相：乙腈+5 mmol/L 乙酸铵（50+50，v/v）；

柱温： $40^{\circ}\mathrm{C}$ ；

进样量： $20~\mu \mathrm{L}$ 。

# 7.4.2 质谱条件

离子源：ESI 源；

扫描方式：正离子扫描；

检测方式：选择反应监测；

喷雾电压： $4800\mathrm{V}$

离子传输管温度： $350^{\circ}\mathrm{C}$

鞘气压力：40arb;

辅助气压力：5arb;

# 7.4.3 测定

# 7.4.3.1 定性测定

通过试样色谱图的保留时间与相应标准品的保留时间、各色谱峰的特征离子与相应浓度标准溶液各色谱峰的特征离子相对照定性。试样与标准品保留时间的相对偏差不大于 $5\%$ ；试样特征离子的相对丰度与浓度相当混合标准溶液的相对丰度一致，相对丰度偏差不超过表2的规定，则可判断试样中存在相应的被测物。

# 7.4.3.2 定量测定

取试样溶液和标准溶液，按外标法，以峰面积定量，标准溶液及试样溶液中的氨苯砜响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，氨苯砜标准溶液、空白组织试样和空白组织添加试样溶液中特征离子质量色谱图见附录A。

# 7.5 空白试验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算与表述

试料中氨本砜的残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算

$$
X = \frac {A \times C S \times V}{A S \times \mathfrak {m}}
$$

式中：

$X$ ——供试试料中氨本砜残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

A—试样溶液中氨本砜的峰面积；

$A_{S}$ ——标准工作液中氨本砜的峰面积；

$\mathrm{C}_{\mathrm{S}}$ ——标准工作液中氨本砜的浓度, $\mu \mathrm{g} / \mathrm{L}$ ;

$V$ ——溶解残余物所用流动相的体积, $\mathrm{mL}$ ;

$m$ —供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度、精密度

# 9.1 灵敏度

本方法的检测限为 $0.1\mu \mathrm{g / kg}$ ，定量限为 $0.5\mu \mathrm{g / kg}$

# 9.2 准确度

本方法在 $0.5\sim 2~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$

# 9.3 精密度

本方法批内相对标准偏差 $\leq 30\%$ ，批间相对标准偏差 $\leq 30\%$ 。

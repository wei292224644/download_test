# 附录A

# 检验方法

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682—2008中规定的三级水。试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603之规定制备；所用溶液在未指明溶剂时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 苟满三酮溶液： $20~\mathrm{g / L}$

A.2.1.2 铂丝。

# A.2.2 鉴别方法

# A.2.2.1 莳满三酮试验

称取约 $1\mathrm{g}$ 试样，精确至 $0.1\mathrm{g}$ ，溶于 $1000~\mathrm{mL}$ 水中，取此溶液 $5\mathrm{mL}$ ，加 $1\mathrm{mL}$ 苷满三酮溶液，加热至沸，约 $3\mathrm{min}$ 后显紫色。

# A.2.2.2 钙盐反应

取铂丝蘸取试样少许，在无色火焰上燃烧，火焰呈砖红色。

# A.3 钙含量的测定

# A.3.1 方法提要

本法以少量镁离子为辅助指示剂，即镁离子与铬黑T生成稳定的络合物指示颜色变化，用乙二胺四乙酸二钠(EDTA)标准滴定溶液滴定，根据乙二胺四乙酸二钠(EDTA)标准滴定溶液消耗的量计算钙含量。

# A.3.2 试剂和材料

A.3.2.1 三乙醇胺。  
A.3.2.2 硫酸镁溶液： $120\mathrm{g / L}$   
A.3.2.3 氨-氯化铵缓冲液： $\mathrm{pH} = 10.0$   
A.3.2.4 乙二胺四乙酸二钠(EDTA)标准滴定溶液： $c(\mathrm{EDTA}) = 0.05\mathrm{mol / L}$   
A.3.2.5 铬黑T指示液： $5\mathrm{g / L}$

# A.3.3 分析步骤

取硫酸镁溶液2滴，加氨-氯化铵缓冲液 $15\mathrm{mL}$ ，加铬黑T指示液2滴，用EDTA标准溶液滴定至纯蓝色；另取试样 $0.35\mathrm{g}$ ，精确至 $0.0002\mathrm{g}$ ，加入水 $30\mathrm{mL}$ 使其溶解，加入三乙醇胺5滴，并与上述滴定

的硫酸镁溶液合并，再用EDTA标准溶液滴定，至颜色由紫红色变为纯蓝色即为终点。同时做空白实验。

# A.3.4 结果计算

钙(以 $\mathrm{Ca}$ 计)含量的质量分数 $\omega$ ，按式(A.1)计算：

$$
w = \frac {\left(V - V _ {0}\right) \times c \times M}{1 000 \times m} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

V ——试样消耗EDTA标准溶液滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗EDTA标准溶液滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$

$c$ ——EDTA标准溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——钙的摩尔质量的数值，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{Ca}) = 40.1]$

$m$ ——试样质量的数值，单位为克(g)；

1000——换算因子。

取两次平行测定结果的算术平均值为报告结果。两次平行测定结果的绝对差值不大于 $0.3\%$ 。

# A.4 L-天门冬氨酸含量的测定

# A.4.1 方法提要

天门冬氨酸钙与过量硫酸铜反应，生成可溶的天门冬氨酸铜，使其与碘化钾反应，用硫代硫酸钠标准溶液滴定，根据硫代硫酸钠标准滴定溶液的消耗量，计算L-天门冬氨酸含量。

# A.4.2 试剂和材料

A.4.2.1 碘化钾。

A.4.2.2 冰乙酸。

A.4.2.3 硫酸铜溶液： $0.1\mathrm{mol / L}$

A.4.2.4 氢氧化钠溶液： $1\mathrm{mol} / \mathrm{L}$

A.4.2.5 氢氧化钠溶液： $0.1\mathrm{mol} / \mathrm{L}$

A.4.2.6 硫代硫酸钠标准滴定液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.02\mathrm{mol / L}$

A.4.2.7 硼酸-氯化钾缓冲溶液 $(\mathrm{pH}9.0)$ : 称取 $3.09 \mathrm{~g}$ 硼酸, 加 $0.1 \mathrm{~mol} / \mathrm{L}$ 氯化钾 $500 \mathrm{~mL}$ , 再加 $210 \mathrm{~mL}$ 的 $0.1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠。

A.4.2.8 淀粉指示液： $1\mathrm{g / L}$

A.4.2.9 酚酞指示液： $5\mathrm{g} / \mathrm{L}$

# A.4.3 分析步骤

称取适量试样（约相当天门冬氨酸 $\leqslant 100\mathrm{mg}$ ），精确至 $0.0002\mathrm{g}$ ，加水溶解并定容 $30~\mathrm{mL}$ 体积。将试样溶液置于 $150~\mathrm{mL}$ 锥形瓶中，加入硫酸铜溶液 $5\mathrm{mL}$ ，酚酞指示液2滴，用 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液调节至溶液显蓝紫色，加入硼酸-氯化钾缓冲溶液 $30~\mathrm{mL}$ ，摇匀。静置 $1\mathrm{h}$ 后过滤，用 $30~\mathrm{mL}$ 硼酸-氯化钾缓冲溶液洗涤滤器及沉淀，合并洗液。加 $1.5\mathrm{mL}$ 冰乙酸及碘化钾 $7.0\mathrm{g}$ ，振摇溶解，用硫代硫酸钠标准溶液滴定，近终点时，加入淀粉指示液 $2\mathrm{mL}$ ，继续滴定至蓝色消失。同时做空白实验。

# A.4.4 结果计算

L-天门冬氨酸含量的质量分数 $w_{1}$ ，按式(A.2)计算：

$$
w _ {1} = \frac {\left(V - V _ {0}\right) c \times M}{1 000 \times m} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$V$ ——试样消耗硫代硫酸钠标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗硫代硫酸钠标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫代硫酸钠标准溶液浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——L-天门冬氨酸的摩尔质量的数值，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{C_4H_7O_4N}) = 133.1]$

$m$ ——试样质量的数值，单位为克(g)；

1000——换算因子。

取两次平行测定结果的算术平均值为报告结果。两次平行测定结果的绝对差值不大于 $0.3\%$ 。

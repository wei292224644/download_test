# 前言

本标准代替GB/T5009.212—2008《贝类中腹泻性贝类毒素的测定》、SC/T3024—2004《腹泻性贝类毒素的测定生物法》、SN/T2269—2009《进出口贝肉中大田软海绵酸的检测液相色谱-串联质谱法》、SN/T2131.2—2010《进出口贝类腹泻性贝类毒素检验方法第2部分：小鼠生物法》、SN/T1996—2007《贝类中腹泻性贝类毒素检验方法酶联免疫吸附法》、SN/T2131.1—2008《进出口贝类腹泻性贝类毒素检测方法第1部分：荧光磷酸酶抑制法》。

本标准与GB/T5009.212—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 贝类中腹泻性贝类毒素的测定”；  
——增加了酶联免疫吸附法；  
——增加了液相色谱-串联质谱法。

# 食品安全国家标准

# 贝类中腹泻性贝类毒素的测定

# 1 范围

本标准规定了贝类中腹泻性贝类毒素测定的小鼠生物法，酶联免疫吸附法和液相色谱-串联质谱法。

本标准中小鼠生物法和酶联免疫吸附法适用于贝类及其制品中腹泻性贝类毒素的测定，液相色谱-串联质谱法适用于贝类可食部分及其制品(不包括盐渍制品)中腹泻性贝类毒素大田软海绵酸(OA)、鳍藻毒素-1(DTX-1)和鳍藻毒素-2(DTX-2)的测定。

# 小鼠生物法

# 2 原理

用丙酮提取贝类中腹泻性贝类毒素(DSP)，经无水乙醚分配，减压蒸干后，再以含 $1\%$ 吐温-60的生理盐水为分散介质，制成DSP混悬液。将该混悬液注射入小鼠腹腔，观察小鼠存活情况，计算其毒力。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 丙酮 $\mathrm{(C_3H_6O)}$   
3.1.2 无水乙醚 $(\mathrm{C_4H_{10}O})$ 。  
3.1.3 吐温-60（ $\mathrm{C}_{64}\mathrm{H}_{126}\mathrm{O}_{26}$ ）。  
3.1.4 氯化钠 $(\mathrm{NaCl})$ 。

# 3.2 试剂配制

3.2.1 氯化钠溶液 $(0.85\%)$ ：称取 $0.85\mathrm{gNaCl}$ ，加水溶解并定容至 $100~\mathrm{mL}$ 。  
3.2.2 吐温-60(1%)：称取 $1.0\mathrm{g}$ 吐温-60，用氯化钠溶液 $(0.85\%)$ 溶解并定容至 $100~\mathrm{mL}$ 。

# 3.3 材料

3.3.1 小鼠：体重为 $16\mathrm{g}\sim 20\mathrm{g}$ 的健康ICR品系雄性小鼠。  
3.3.2 金属筛网：孔径约 $2\mathrm{mm}$

# 4 仪器和设备

4.1 旋转蒸发器。

4.2 均质器。  
4.3 天平：感量为 $0.1\mathrm{g}$ 。

# 5 分析步骤

注：为避免毒素的危害，应戴手套进行操作。移液管及移液器吸头等用过的器材、废弃的提取液等应在次氯酸钠溶液 $(5\%)$ 浸泡 $1\textrm{h}$ 以上以使毒素分解。对于动物实验过程中产生的污水、废弃物及动物尸体处理，应参照GB14925执行。

# 5.1 样品采集

采取足够的贝类样品个数，并使贝肉达 $400\mathrm{g}$ 以上。远离实验室不能及时送检的样品，除了在常温下品质不会发生变化的，应将样品置于保温盒中冷冻送检，或采取必要措施保证其处于低温状态 $(0^{\circ}\mathrm{C}\sim 10^{\circ}\mathrm{C})$ 送检。如为带壳样品，应开壳，去除水分后冷冻送检。

# 5.2 试样制备

# 5.2.1 生鲜带壳样品

用清水彻底洗净贝类样品外表，切断闭壳肌，开壳，用清水淋洗内部去除泥沙及其他异物，取出贝肉。严禁以加热或药物方法开壳。注意不要破坏闭壳肌以外的组织，尤其是中肠腺（中肠腺又称消化盲囊，组织呈暗绿色或褐绿色）。将去壳贝肉置于孔径约 $2\mathrm{mm}$ 的金属筛网上，沥水 $5\mathrm{min}$ 。将贝肉剪碎。

# 5.2.2 冷冻样品

在室温下使冷冻样品融化呈半冷冻状态。带壳冷冻的样品按5.2.1方法清洗、开壳、淋洗取肉，除去贝肉外部附着的冰片，抹去水分。将贝肉剪碎。

# 5.3 试样提取

称取 $200\mathrm{g}$ 剪碎的贝肉试样置于均质杯中，按体积比加3倍量丙酮后均质 $2\mathrm{min}$ 以上。将均质好的物质倒入布氏漏斗中抽滤，收集滤液。分别用残渣2倍量的丙酮再清洗残渣两次，滤液与上述滤液合并。将滤液移入 $500~\mathrm{mL}$ 的圆底烧瓶中， $56^{\circ}C\pm 1^{\circ}C$ 下，减压浓缩去除丙酮直至在液体表面分离出油状物。

用 $100\mathrm{mL}\sim 200\mathrm{mL}$ 无水乙醚溶解油状物，倒入分液漏斗内，再用少量无水乙醚清洗圆底烧瓶，合并倒入分液漏斗内，以少量的水洗下粘壁部分，轻轻振荡(不能生成乳浊液)，静置分层后去除水层（下层）。

用相当乙醚半量的水洗乙醚层两次，去除水层，再将乙醚层移入 $250~\mathrm{mL}$ 或 $500~\mathrm{mL}$ 的圆底烧瓶中，于 $35^{\circ}C \pm 1^{\circ}C$ 减压浓缩去除乙醚。用少量无水乙醚将浓缩物移入 $50~\mathrm{mL}$ 或 $100~\mathrm{mL}$ 圆底烧瓶中，再次减压浓缩去除乙醚。

用 $1\%$ 吐温-60的生理盐水将全部浓缩物转移至刻度试管中稀释到 $10~\mathrm{mL}$ ，充分振摇，制成均匀混悬液。 $1\mathrm{mL}$ 该混悬液相当于 $20\mathrm{g}$ 试样，以此混悬液为试验原液。

以试验原液注射小鼠，当 $24\mathrm{h}$ 内2只或3只小鼠死亡时，需振荡试验原液使成均匀混悬液，用 $1\%$ 吐温-60生理盐水按表1逐步稀释成4倍或16倍的稀释液，充分混匀，按表1注射小鼠。

# 5.4 小鼠试验

选择 $16\mathrm{g}\sim 20\mathrm{g}$ 健康ICR雄性小鼠6只，随机分为实验组和溶剂对照组两组，每组3只。实验组将腹腔注射试验原液或其稀释液，溶剂对照组将腹腔注射 $1\%$ 吐温-60生理盐水。

分别取 $1\mathrm{mL}$ 待测液或 $1\%$ 吐温-60生理盐水腹腔注射小鼠。注射过程中若有提取液溢出，须将该只小鼠丢弃，并重新注射一只小鼠。仔细观察并记录死亡时间。死亡时间计算从注射完毕开始至小鼠停止呼吸(小鼠呼出最后一口气止)为止。存活动物应连续观察 $24\mathrm{h}$ 。观察时限 $24\mathrm{h}$ 内，在溶剂对照组小鼠正常的情况下，实验组若出现2只或3只小鼠死亡，则按表1进一步实验，以确定一组3只小鼠中死亡2只或2只以上的最小染毒量或最大稀释度。

# 6 分析结果的表述

观察时限 $24\mathrm{h}$ 内，在溶剂对照组小鼠正常情况下，若实验组无小鼠死亡或仅有1只小鼠死亡，则报告样品中DSP毒力为： $\mathrm{<  }0.05\mathrm{MU / g}$

# 酶联免疫吸附法

# 7 原理

根据竞争性酶联免疫反应，游离的腹泻性贝类毒素与其酶标记物竞争腹泻性贝类毒素抗体。没有被结合的酶标记物在洗涤步骤中被除去。将酶底物和显色剂加入到孔中并且孵育。结合的酶标记物将无色的发色剂转化为蓝色的产物。加入反应终止液后使颜色由蓝转变为黄色。用酶标仪在 $450~\mathrm{nm}$ 波长下测量微孔溶液的吸光度值，试样中的腹泻性贝类毒素含量与吸光度值成反比，按绘制的标准曲线定量计算。

# 8 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 8.1 试剂

8.1.1 甲醇 $\mathrm{(CH_3OH)}$ 。  
8.1.2 十二水合磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4\cdot 12\mathrm{H}_2\mathrm{O})$ 。

8.1.3 氯化钠 $(\mathrm{NaCl})$ 。  
8.1.4 氯化钾(KCl)。  
8.1.5 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
8.1.6 吐温-20 $(\mathrm{C}_{58}\mathrm{H}_{114}\mathrm{O}_{26})$ 。  
8.1.7 牛血清白蛋白（BSA）。  
8.1.8 酶标记物。  
8.1.9 过氧化氢 $\left(\mathrm{H}_{2} \mathrm{O}_{2}\right)$ 。  
8.1.10 3,3,5,5-四甲基联苯胺 $\mathrm{(TMB,C_{16}H_{20}N_2)}$   
8.1.11 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。

# 8.2 试剂配制

8.2.1 甲醇溶液 $(90\%)$ ：量取 $90~\mathrm{mL}$ 甲醇，加入 $10~\mathrm{mL}$ 水，混合均匀。  
8.2.2 磷酸盐缓冲液(PBS溶液，pH7.4)：分别称取磷酸二氢钾 $0.20\mathrm{g}$ 、十二水合磷酸氢二钠 $2.90\mathrm{g}$ 、氯化钠 $8.00\mathrm{g}$ 、氯化钾 $0.20\mathrm{g}$ ，加水溶解并定容至 $1000\mathrm{mL}$ 。  
8.2.3 酶标记物稀释液：称取 $1.0\mathrm{g}$ BSA，加PBS溶液溶解并定容至 $1000~\mathrm{mL}$   
8.2.4 酶标记物工作液：用酶标记物稀释液将酶标记物稀释至工作浓度。  
8.2.5 洗脱液：吸取 $0.5\mathrm{mL}$ 吐温-20加入PBS溶液中，并稀释至 $1000~\mathrm{mL}$   
8.2.6 硫酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：吸取 $53.2~\mathrm{mL}$ 硫酸，缓缓加至 $900~\mathrm{mL}$ 水中，并用水稀释至 $1000~\mathrm{mL}$ 。

# 8.3 标准品

大田软海绵酸 $(\mathrm{OA},\mathrm{C}_{44}\mathrm{H}_{68}\mathrm{O}_{13}$ ，CAS号78111-17-8)标准溶液。

# 8.4 标准溶液配制

标准系列工作液：将大田软海绵酸标准溶液用PBS溶液稀释，配制成浓度分别为 $0\mu \mathrm{g} / \mathrm{L},5\mu \mathrm{g} / \mathrm{L}$ 、 $10~\mu \mathrm{g} / \mathrm{L},25~\mu \mathrm{g} / \mathrm{L},50~\mu \mathrm{g} / \mathrm{L},100~\mu \mathrm{g} / \mathrm{L},150~\mu \mathrm{g} / \mathrm{L}$ 和 $200~\mu \mathrm{g} / \mathrm{L}$ 的DSP标准系列工作液。现用现配。

# 8.5 材料

包被有腹泻性贝类毒素抗体的微孔板。

注：商业化试剂盒若评价技术参数达到本标准的要求则也适合于本标准，参见附录A。

# 9 仪器和设备

9.1 酶标仪。

9.2 均质器。

9.3 离心机：转速 $\geq 6000\mathrm{r / min}$

# 10 分析步骤

# 10.1 样品采集

同5.1。

# 10.2 试样制备

同5.2。

# 10.3 试样提取

将剪碎的试样均质，准确称取 $10\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ），加入 $50~\mathrm{mL}$ 甲醇溶液 $(90\%)$ ，均质 $1\mathrm{min} \sim 2\mathrm{min}, 6000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，转移出上清液，根据上清液体积加入2倍体积的PBS溶液，混合均匀，吸取 $50~\mu \mathrm{L}$ 试样稀释液进行测定。

# 10.4 测定

将包被有腹泻性贝类毒素抗体的微孔条插入微孔架并做好标记，其中包括空白对照孔、标准液孔和样液孔，分别做平行孔。向空白对照孔加入 $50~\mu \mathrm{L}$ PBS溶液，标准液孔加入 $50~\mu \mathrm{L}$ 腹泻性贝类毒素标准系列工作液，样液孔加入 $50~\mu \mathrm{L}$ 样液。加入 $50~\mu \mathrm{L}$ 腹泻性贝类毒素酶标记物至每个微孔，迅速充分混合， $22^{\circ}C\sim 25^{\circ}C$ 避光孵育 $10\mathrm{min}$ 。孵育结束后，倒去孔中液体，每个微孔注入 $250~\mu \mathrm{L}$ 洗脱液冲洗，翻转微孔板，倾去孔内液体，再重复以上洗板操作4次，在吸水纸上拍干。每孔加 $50~\mu \mathrm{L}$ 过氧化氢和TMB，充分混合，室温避光孵育 $6\mathrm{min}$ 。每孔加入 $50~\mu \mathrm{L}$ 硫酸溶液(1mol/L)迅速混匀，终止反应，在 $10\mathrm{min}$ 内测量并记录 $450~\mathrm{nm}$ 波长下的吸光度值。

若提取液经测定后的质量浓度超出标准曲线的线性范围，应适当稀释后重新测定。

# 10.5 标准曲线的制作

以腹泻性贝类毒素标准工作液质量浓度以10为底的对数值为横坐标，以式(1)计算的标准液的百分比吸光度值为纵坐标，绘制标准曲线。

腹泻性贝类毒素标准液或样液的百分比吸光度值按式(1)计算：

$$
A = \frac {S}{S _ {0}} \times 100 \% \tag{1}
$$

式中：

$A$ ——百分比吸光度值；

S——腹泻性贝类毒素标准液或样液的平均吸光度值；

$S_{0}$ —— $0\mu \mathrm{g} / \mathrm{L}$ 的腹泻性贝类毒素标准液的平均吸光度值。

# 11 分析结果的表述

试样中腹泻性贝类毒素的含量按式(2)计算：

$$
X = \frac {\rho \times V \times f}{m} \tag {2}
$$

式中：

$X$ ——试样中腹泻性贝类毒素的含量，单位为微克每克 $(\mu \mathrm{g} / \mathrm{g})$ ；

$\rho$ ——由标准曲线得到的试样待测液中腹泻性贝类毒素的质量浓度, 单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ;

$V$ ——试样提取液的体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释倍数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

注：任何腹泻性贝类毒素含量大于 $16\mu \mathrm{g} / 100\mathrm{g}$ 的样品即被认为是有害的，对人类食用不安全。

# 12 其他

本方法的定量限为 $10\mu \mathrm{g / kg}$

# 液相色谱-串联质谱法

# 13 原理

试样经甲醇提取，碱性条件下水解释放出酯化态腹泻性贝类毒素，液相色谱分离，串联质谱法测定，以基质标准曲线进行外标法定量。

# 14 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的一级水。

14.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
14.1.3 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。  
14.1.4 氢氧化钠(NaOH)。  
14.1.5 盐酸（HCl）  
14.1.6 甲酸铵 $\mathrm{(NH_4COOH)}$ ：色谱纯。  
14.1.7 甲酸（HCOOH）：色谱纯。

# 14.2 试剂配制

14.2.1 甲醇溶液 $(30\%)$ ：量取 $30~\mathrm{mL}$ 甲醇，用水稀释至 $100~\mathrm{mL}$   
14.2.2 甲醇溶液 $(20\%)$ ：量取 $20~\mathrm{mL}$ 甲醇，用水稀释至 $100~\mathrm{mL}$ 。

14.2.4 氢氧化钠溶液 $(2.5\mathrm{mol} / \mathrm{L})$ ：准确称取 $50\mathrm{g}$ 氢氧化钠，用水溶解并稀释至 $500~\mathrm{mL}$ 。  
14.2.5 盐酸溶液 $(2.5\mathrm{mol} / \mathrm{L})$ ：准确量取 $104.5\mathrm{mL}$ 盐酸，用水稀释至 $500~\mathrm{mL}$ 。  
14.2.6 流动相A[甲酸铵溶液(2mmol/L)]：准确称取 $126\mathrm{mg}$ 甲酸铵，用 $50~\mathrm{mL}$ 水将其全部溶解，加入 $2\mathrm{mL}$ 甲酸，加水定容至 $1000~\mathrm{mL}$ ，室温下可保存 $48\mathrm{h}$ 。

# 17 分析结果的表述

17.1 试样中腹泻性贝类毒素的含量按式(3)计算：

$$
X _ {i} = \frac {\rho_ {i} \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {3}
$$

式中：

$X_{i}$ ——试样中腹泻性贝类毒素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho_{i}$ ——由基质标准曲线得到的试样溶液中腹泻性贝类毒素的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算因子；

$m$ ——与 $1\mathrm{mL}$ 试样提取液相当的试样质量，单位为克 $(\mathrm{g})$ 。

计算结果应扣除空白值。计算结果保留三位有效数字。

# 17.2 总毒力计算

试样中腹泻性贝类毒素总毒力按式(4)计算：

$$
O A _ {\mathrm {e q}} = \sum_ {i = 1} ^ {n} X _ {i} \cdot r _ {i} \quad \dots \tag {4}
$$

式中：

$OA_{\mathrm{eq}}$ ——腹泻性贝类毒素总毒力，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$X_{i}$ ——各种腹泻性贝类毒素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$r_i$ ——毒性因子。

注：试样中各种腹泻性贝类毒素的含量需按照附录B中的毒性因子(TEFs)，统一转换为总毒力 $(OA_{\mathrm{eq}})$ 。

# 18 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

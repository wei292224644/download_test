# 方法一

# 1 范围

本标准规定了牛奶中喹诺酮类药物残留量检测的制样和高效液相色谱测定方法。

本标准适用于牛奶中环丙沙星、达氟沙星、恩诺沙星、沙拉沙星和二氟沙星单个或多个药物残留量检测。

GB/T6682 分析实验室用水规则和试验方法

# 3 原理

试料中残留的喹诺酮类药物，用乙腈提取，旋转蒸发至近干，流动相溶解。高效液相色谱-荧光测定，外标法定量。

# 4 试剂与材料

以下所用的试剂，除特别注明者外均为分析纯试剂；水为符合GB/T6682规定的一级水。

4.1达氟沙星、恩诺沙星、盐酸环丙沙星、盐酸沙拉沙星和盐酸二氟沙星对照品 含量 $\geq 99.0\%$   
4.2 磷酸  
4.3 氢氧化钠  
4.4 乙腈：色谱纯。  
4.5 三乙胺  
4.6 氢氧化钠饱和溶液：取氢氧化钠适量，加水振摇使成饱和溶液，冷却后，置聚乙烯塑料瓶中，静置，澄清。  
4.7 $5 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液：取氢氧化钠饱和溶液 $28 \mathrm{~mL}$ ，用水溶解并稀释至 $100 \mathrm{~mL}$ 。

4.8 $0.03 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液：取 $5 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液 $0.6 \mathrm{~mL}$ ，用水溶解并稀释至 $100 \mathrm{~mL}$ 。  
4.9 $0.05\mathrm{mol} / \mathrm{L}$ 磷酸三乙胺溶液：取磷酸 $3.4\mathrm{mL}$ ，用水溶解并稀释至 $1000\mathrm{mL}$ 。用三乙胺调 $\mathrm{pH}$ 至2.4。  
4.10 喹诺酮类药物混合标准贮备液：精密称取达氟沙星对照品 $10 \mathrm{mg}$ ，恩诺沙星、环丙沙星、沙拉沙星和二氟沙星对照品各 $50 \mathrm{mg}$ ，于 $50 \mathrm{~mL}$ 量瓶中，用 $0.03 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液溶解并稀释至刻度，配制成达氟沙星浓度为 $0.2 \mathrm{mg} / \mathrm{mL}$ ，环丙沙星、恩诺沙星、沙拉沙星和二氟沙星浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的喹诺酮类药物混合标准贮备液。 $2 \sim 8^{\circ} \mathrm{C}$ 保存，有效期3个月。  
4.11 喹诺酮类药物混合标准工作液: 精密量取喹诺酮类药物混合标准贮备液 $1.0 \mathrm{~mL}$ , 于 $100 \mathrm{~mL}$ 量瓶中,用流动相稀释, 配制成达氟沙星浓度为 $2 \mu \mathrm{g} / \mathrm{mL}$ , 环丙沙星、恩诺沙星、沙拉沙星和二氟沙星浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 的喹诺酮类药物混合标准工作液。 $2 \sim 8^{\circ} \mathrm{C}$ 保存, 有效期1周。

# 5 仪器设备

5.1 高效液相色谱仪：配荧光检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3天平：感量 $0.01\mathrm{g}$   
5.4 振荡器  
5.5 离心机  
5.6 聚四氟乙烯离心管： $50~\mathrm{mL}$ 。  
5.7鸡心瓶： $25~\mathrm{mL}$   
5.8 滤膜： $0.45 \mu \mathrm{m}$

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或冷藏的空白或供试牛奶，混合均匀。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-20℃以下保存。

# 7测定步骤

# 7.1 提取

称取试料 $(2 \pm 0.05) \mathrm{~g}$ , 于 $50 \mathrm{~mL}$ 离心管中, 加磷酸 $100 \mu \mathrm{L}$ , 乙腈 $4 \mathrm{~mL}$ , 涡旋混匀, 中速振荡 $5 \mathrm{~min}$ , $10000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ , 取上清液于另一离心管中, 加正己烷 $5 \mathrm{~mL}$ , 涡旋 $1 \mathrm{~min}$ , 静置, 取下层清液于 $25 \mathrm{~mL}$ 鸡心瓶中。残渣中加乙腈 $4 \mathrm{~mL}$ , 重复提取一次, 上清液经同一份正己烷分配, 合并两次提取液, 于 $50^{\circ} \mathrm{C}$ 旋转蒸发至仅剩余不易蒸干的黄色油滴。用流动相 $1.0 \mathrm{~mL}$ 溶解残余物, 滤膜过滤, 供高效液相色谱法测定。

# 7.2标准曲线制备

精密量取喹诺酮类药物混合标准工作液适量，用流动相稀释，配制成浓度环丙沙星、恩诺沙星、沙拉沙星和二氟沙星为5、10、50、100、300和 $500\mu \mathrm{g} / \mathrm{L}$ ，达氟沙星浓度为1、2、10、20、60和 $100\mu \mathrm{g} / \mathrm{L}$ 的系列标准溶液，供高效液相色谱测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.3 测定

# 7.3.1 色谱条件

色谱柱： $\mathrm{C}_{18}$ （ $250 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ ，粒径 $5 \mu \mathrm{m}$ ），或相当者；

流动相： $0.05\mathrm{mol} / \mathrm{L}$ 磷酸溶液-三乙胺+乙腈（ $90 + 10$ ，v/v），滤膜过滤；

流速： $1.8\mathrm{mL / min}$

检测波长：激发波长 $280 \mathrm{~nm}$ ；发射波长 $450 \mathrm{~nm}$

柱温： $30^{\circ}\mathrm{C}$ ；

进样量： $20~\mu \mathrm{L}$

# 7.3.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试样溶液中环丙沙星、达氟沙星、恩诺沙星、沙拉沙星和二氟沙星响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和空白组织添加试样溶液的高效液相色谱图见附录A。

# 7.4 空白试验

除不加试料外，采用完全相同的测定步骤进行平行操作。

# 8 结果计算和表述

试料中喹诺酮类药物残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ），按下式计算：

$$
\mathrm {X} = \frac {C \times V}{m}
$$

式中：

$X$ ——供试试料中相应的喹诺酮类药物残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

C——试样溶液中相应的喹诺酮类药物浓度, ng/mL;

$V$ ——溶解残渣所用流动相体积, $\mathrm{mL}$ ;

$m$ ——供试试料质量， $\mathrm{g}$ 。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度、精密度

# 9.1 灵敏度

本方法环丙沙星、恩诺沙星、沙拉沙星和二氟沙星的检测限为 $5 \mu \mathrm{g} / \mathrm{kg}$ , 定量限为 $10 \mu \mathrm{g} / \mathrm{kg}$ ; 达氟沙星的检测限为 $1 \mu \mathrm{g} / \mathrm{kg}$ , 定量限为 $2 \mu \mathrm{g} / \mathrm{kg}$ 。

# 9.2 准确度

本方法在 $10\sim 100~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 100\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

# 方法二

# 1 范围

本标准规定了牛奶中11种喹诺酮类药物残留量检测的制样和高效液相色谱测定方法。

本标准适用于牛奶中恩诺沙星、环丙沙星、达氟沙星、沙拉沙星、二氟沙星、诺氟沙星、氧氟沙星、培氟沙星、洛美沙星、氟甲喹和噁喹酸单个或多个药物残留检测。

# 3 原理

试料中残留的喹诺酮类药物，用 $10\%$ 三氯乙酸-乙腈提取，反相聚合物SPE柱净化，流动相溶解，高效液相-荧光法测定，外标法定量。

# 4 试剂和材料

以下所有试剂，除特别注明者外均为分析纯试剂；水为符合GB/T6682规定的一级水。

4.1 恩诺沙星、盐酸环丙沙星、甲磺酸达氟沙星、沙拉沙星、二氟沙星、诺氟沙星、氧氟沙星、甲磺酸培氟沙星、盐酸洛美沙星、氟甲喹和噁喹酸对照品：含量 $\geqslant 98.0\%$ 。

4.2 乙腈：色谱纯。  
4.3 甲醇：色谱纯。  
4.4 无水乙醇  
4.5 乙酸  
4.6 柠檬酸   
4.7 乙酸铵  
4.8 三乙胺  
4.9 氢氧化钠

# 4.10 三氯乙酸

4.10 SPE反相聚合物柱：Strata-X填料， $60\mathrm{mg} / 3\mathrm{mL}$ ，或相当者。  
4.11 $10\%$ 乙酸溶液：取乙酸 $10~\mathrm{mL}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
4.12 $0.5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：取氢氧化钠 $2\mathrm{g}$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。  
4.13 $10\%$ 三氯乙酸溶液：取三氯乙酸 $100 \mathrm{~g}$ ，用水溶解并稀释至 $1000 \mathrm{~mL}$ 。  
4.14 乙腈- $10\%$ 三氯乙酸溶液：取乙腈 $10 \mathrm{~mL}$ ，用 $10\%$ 三氯乙酸溶液溶解并稀释至 $100 \mathrm{~mL}$ 。  
4.15 $10\%$ 甲醇水溶液：取甲醇 $10 \mathrm{~mL}$ ，用水溶解并稀释至 $100 \mathrm{~mL}$ 。  
4.16 柠檬酸/乙酸铵缓冲液：取柠檬酸 $10.56 \mathrm{~g}$ 、乙酸铵 $7.87 \mathrm{~g}$ ，用水溶解并稀释至 $1000 \mathrm{~mL}$ ，用三乙胺调 $\mathrm{pH}$ 至 4.0，滤膜过滤。  
4.17 乙腈一柠檬酸/乙酸铵缓冲液：取乙腈 $8\mathrm{mL}$ ，用柠檬酸/乙酸铵缓冲液溶解并稀释至 $100~\mathrm{mL}$ 。  
4.18 $1 \mathrm{mg} / \mathrm{mL}$ 诺氟沙星、氧氟沙星、恩诺沙星、沙拉沙星和二氟沙星标准贮备液：精密称取诺氟沙星、氧氟沙星、恩诺沙星、沙拉沙星和二氟沙星对照品各适量，分别于 $10 \mathrm{~mL}$ 量瓶中，用乙酸溶液 200 $\mu \mathrm{L}$ 溶解，用甲醇稀释至刻度，配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的喹诺酮类药物标准贮备液。-20°C以下保存，有效期6个月。  
4.19 $1 \mathrm{mg} / \mathrm{mL}$ 环丙沙星、洛美沙星、培氟沙星和达氟沙星标准贮备液：精密称取盐酸环丙沙星、盐酸洛美沙星、甲磺酸培氟沙星和甲磺酸达氟沙星对照品各适量，分别于 $10 \mathrm{~mL}$ 量瓶中，用水 $200 \mu \mathrm{L}$ 溶解，用甲醇稀释至刻度，配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的喹诺酮类药物标准贮备液。-20°C以下保存，有效期6个月。  
4.20 $1\mathrm{mg / mL}$ 嗪喹酸和氟甲喹标准贮备液：精密称取噁喹酸和氟甲喹对照品适量，分别于 $10~\mathrm{mL}$ 量瓶中，用氢氧化钠溶液 $400~\mu \mathrm{L}$ 溶解，用甲醇稀释至刻度，配制成浓度为 $1\mathrm{mg / mL}$ 的喹诺酮类药物标准贮备液。-20℃以下保存，有效期6个月。  
4.21 喹诺酮类药物混合标准工作液：精密量取喹诺酮类药物标准贮备液各适量，于量瓶中，用水溶解并稀释至刻度，配制成诺氟沙星、环丙沙星、氧氟沙星、培氟沙星、洛美沙星、恩诺沙星、沙拉沙星、二氟沙星和噁唑酸浓度为 $8 \mu \mathrm{g} / \mathrm{mL}$ ，氟甲喹浓度为 $4 \mu \mathrm{g} / \mathrm{mL}$ ，达氟沙星浓度为 $2.4 \mu \mathrm{g} / \mathrm{mL}$ 的混合标准工作液。 $2 \sim 8^{\circ} \mathrm{C}$ 保存，有效期1个月。

# 6 试样的制备与保存

# 6.1 试样的制备

取适量新鲜或冷藏的空白或供试牛奶，混合均匀。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

# 6.2 试料的保存

-20°C以下保存。

# 7 测定步骤

# 7.1 标准曲线的制备

精密量取混合标准工作液适量，用水稀释，漩涡混匀，使达氟沙星浓度为15、30、60、120、240和 $480\mu \mathrm{g} / \mathrm{L}$ ，氟甲喹浓度为25、50、100、200、400和 $800\mu \mathrm{g} / \mathrm{L}$ ，其他喹诺酮类药物浓度为50、100、200、400、800和 $1600\mu \mathrm{g} / \mathrm{L}$ 的系列标准溶液，供高效液相色谱测定。以峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.2 提取

称取试料（ $1 \pm 0.02$ ）g，于 $15 \mathrm{~mL}$ 聚丙烯离心管中，加乙腈- $10\%$ 三氯乙酸溶液 $5 \mathrm{~mL}$ ，漩涡混匀，超声 $5 \mathrm{~min}$ ，于 $4^{\circ} \mathrm{C} 8000 \mathrm{r} / \mathrm{min}$ 离心 $6 \mathrm{~min}$ ，取上清液，备用。

# 7.3 净化

SPE柱依次用甲醇 $3 \mathrm{~mL}$ 和水 $3 \mathrm{~mL}$ 活化，取备用液过柱，控制流速 $1 \mathrm{~mL} / \mathrm{min}$ ，用 $10 \%$ 甲醇水溶液 $3 \mathrm{~mL}$ 淋洗，抽干，用甲醇 $3 \mathrm{~mL}$ 洗脱，抽干。收集洗脱液，于 $60 \sim 70^{\circ} \mathrm{C}$ 氮气吹干，用乙腈一柠檬酸/乙酸铵缓冲液 $500 \mu \mathrm{L}$ 溶解残余物，涡旋混匀，于 $4^{\circ} \mathrm{C}$ 以下 $12000 \mathrm{r} / \mathrm{min}$ 离心 $6 \mathrm{~min}$ ，取上清液，供高效液相色谱测定。

# 8 结果计算和表述

# 8.1 标准曲线校准

将标准曲线的浓度和对应峰面积进行回归分析，然后按下式计算：

$$
X = \frac {A - b}{a}
$$

式中：

$X-$ 供试试料中相应的喹诺酮类药物残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

$A$ ——供试试料中相应的喹诺酮药物处理后试样溶液中被测喹诺酮类药物的色谱峰面积；

$b$ ——标准曲线回归方程中截距；

$a$ ——标准曲线回归方程中斜率。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 8.2 单点校准

试料中喹诺酮药物残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算

$$
X _ {i} = C _ {s} \times \frac {A _ {i}}{A _ {s}} \times \frac {V}{m}
$$

式中：

$X_{i}$ ——供试试料中相应的喹诺酮类药物的残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

$\mathrm{C}_{\mathrm{s}}$ ——标准溶液中相应的喹诺酮类药物的浓度， $\mathrm{ng/mL}$

$A_{i}$ ——试样溶液中相应的喹诺酮类药物的色谱峰面积；

$A_{s}$ ——标准溶液中相应的喹诺酮类药物的色谱峰面积；

$V$ ——溶解残余物所用乙腈一柠檬酸/乙酸铵缓冲液体积, $\mathrm{mL}$ 。

$m$ ——供试试料质量，g；

# 9 检测方法灵敏度、准确度与精密度

# 9.1 灵敏度

本方法诺氟沙星、氧氟沙星、环丙沙星、培氟沙星、洛美沙星、恩诺沙星、沙拉沙星、二氟沙星和噁喹酸检测限为 $25 \mu \mathrm{g} / \mathrm{kg}$ , 定量限为 $50 \mu \mathrm{g} / \mathrm{kg}$ ; 达氟沙星的检测限为 $7.5 \mu \mathrm{g} / \mathrm{kg}$ , 定量限为 $15 \mu \mathrm{g} / \mathrm{kg}$ ; 氟甲喹检测限为 $12.5 \mu \mathrm{g} / \mathrm{kg}$ , 定量限为 $25 \mu \mathrm{g} / \mathrm{kg}$ 。

# 9.2 准确度

本方法诺氟沙星、氧氟沙星、环丙沙星、培氟沙星、洛美沙星、恩诺沙星、沙拉沙星、二氟沙星、噁喹酸在 $50 \sim 200 \mu \mathrm{g} / \mathrm{kg}$ , 达氟沙星在 $15 \sim 60 \mu \mathrm{g} / \mathrm{kg}$ , 氟甲喹在 $25 \sim 100 \mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $60 \% \sim 110 \%$ 。

# 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对准标准偏差 $\leqslant 20\%$ 。

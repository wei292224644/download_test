# 食品营养强化剂 L-蛋氨酸(L-甲硫氨酸)

# 1 范围

本标准适用于以可食用的淀粉质或糖质为主要原料，经发酵、提取、精制制得的；或以DL-蛋氨酸为原料，经酶法拆分、精制制得的食品营养强化剂L-蛋氨酸(L-甲硫氨酸)。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

L-2-氨基-4-(甲硫基)丁酸

# 2.2 分子式

$$
\mathrm {C} _ {5} \mathrm {H} _ {1 1} \mathrm {N O} _ {2} \mathrm {S}
$$

# 附录A

# 检验方法

# A.1 安全提示

本试验方法中使用的部分试剂具有毒性或腐蚀性，操作时需小心谨慎。若溅到皮肤上应立即用水冲洗，严重者应立即治疗。在使用挥发酸时，要在通风橱中进行。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及其制品，在没有注明其他要求时，分别按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

采用溴化钾压片法，按照GB/T6040进行红外光谱试验，试样的红外光谱图应与L-蛋氨酸的对照谱图(见附录B)一致。

# A.4 L-蛋氨酸含量（以干基计）的测定

# A.4.1 方法提要

试样以甲酸为助溶剂，以冰乙酸为溶剂，用高氯酸标准滴定溶液滴定，根据消耗高氯酸标准滴定溶液的体积计算L-蛋氨酸的含量。

# A.4.2 试剂和材料

A.4.2.1 冰乙酸。

A.4.2.2 无水甲酸。

A.4.2.3 高氯酸标准滴定溶液： $c(\mathrm{HClO}_4) = 0.1\mathrm{mol / L}$

A.4.2.4 结晶紫指示液： $5\mathrm{g / L}$

# A.4.3 仪器和设备

A.4.3.1 电位滴定仪。

A.4.3.2 滴定管。

A.4.3.3 电子天平：感量为 $0.0001\mathrm{g}$ 。

# A.4.4 分析步骤

称取 $105\mathrm{^{\circ}C}$ 干燥至恒重的试样 $0.14\mathrm{g}$ （精确至 $0.0001\mathrm{g})$ ，置于干燥的烧杯或锥形瓶中，加 $3\mathrm{mL}$ 无水甲酸和 $50~\mathrm{mL}$ 冰乙酸溶解。用高氯酸标准滴定溶液滴定，用电位滴定仪滴定至终点。或选用指示剂，加2滴结晶紫指示液，溶液由紫色变成蓝绿色为滴定终点。同时做空白试验。

# A.4.5 结果计算

L-蛋氨酸含量（以干基计）的质量分数 $\varpi_{1}$ 按公式(A.1)计算：

$$
w _ {1} = \frac {\left(V _ {1} - V _ {2}\right) \times c \times M}{m _ {1} \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——滴定试样溶液消耗高氯酸标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴定空白溶液消耗高氯酸标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——高氯酸标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——L-蛋氨酸的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ （ $M = 149.21$ ）；

$m_{1}$ ——试样的质量，单位为克(g)；

1000——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $0.3\%$ 。

# A.5 透光率的测定

# A.5.1 仪器和设备

A.5.1.1 分光光度计。

A.5.1.2 电子天平：感量为 $0.01\mathrm{g}$

# A.5.2 分析步骤

称取 $0.5\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )，加 $20~\mathrm{mL}$ 水溶解。用 $1\mathrm{cm}$ 比色皿，以水为空白对照，在 $430~\mathrm{nm}$ 波长处测定试样溶液的透光率，记录读数。试验结果以平行测定结果的算术平均值为准。

# A.6 比旋光度的测定

# A.6.1 试剂和材料

盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ：量取 $500~\mathrm{mL}$ 盐酸，加水稀释至 $1000~\mathrm{mL}$ 。

# A.6.2 仪器和设备

A.6.2.1 旋光仪。

A.6.2.2 电子天平：感量为 $0.0001\mathrm{g}$

# A.6.3 分析步骤

称取 $105\mathrm{^\circ C}$ 干燥至恒重的试样 $1\mathrm{g}$ （精确至 $0.0001\mathrm{g})$ ，用盐酸溶液溶解定容至 $50~\mathrm{mL}$ ，按照GB/T613的规定进行测定。

# A.6.4 结果计算

比旋光度 $\alpha_{\mathrm{m}}(20^{\circ}\mathrm{C},\mathrm{D})$ ，以“ $(^{\circ})\cdot \mathrm{dm}^{2}\cdot \mathrm{kg}^{-1}$ ”表示，按公式(A.2)计算：

$$
\alpha_ {\mathrm {m}} (2 0 ^ {\circ} \mathrm {C}, \mathrm {D}) = \frac {\alpha}{l \times \rho} \tag {A.2}
$$

式中：

$\alpha$ ——在 $20^{\circ}\mathrm{C}$ 时测得的旋光度，单位为度 $(^{\circ})$

$l$ ——旋光管的长度，单位为分米（dm）；

$\rho$ ——溶液中 $\mathrm{L}$ -蛋氨酸的质量浓度，单位为克每毫升 $(\mathrm{g / mL})$ 。

# A.7 灼烧残渣的测定

# A.7.1 试剂和材料

浓硫酸。

# A.7.2 仪器和设备

A.7.2.1 高温炉。  
A.7.2.2 瓷坩埚。  
A.7.2.3 干燥器。  
A.7.2.4 电子天平：感量为 $0.0001\mathrm{g}$

# A.7.3 分析步骤

称取 $2\mathrm{g}$ 试样（精确至 $0.0001\mathrm{g}$ ），置于已在 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 下炽灼至恒重后的坩埚中，缓缓炽灼至完全碳化，冷却至室温。加浓硫酸 $1\mathrm{mL} \sim 2\mathrm{mL}$ 使之湿润，低温加热直至硫酸蒸汽除尽后，在 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 下炽灼使之完全灰化，待炉温降至 $200^{\circ}\mathrm{C}$ 左右时，取出，移至干燥器内，冷却 $30\mathrm{min}$ ，称量。再在 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 下重复炽灼至恒重。

# A.7.4 结果计算

灼烧残渣的质量分数 $\varpi_{2}$ ，按公式(A.3)计算：

$$
w _ {2} = \frac {m _ {4} - m _ {2}}{m _ {3} - m _ {2}} \times 100 \% \quad \dots \tag{A.3}
$$

式中：

$m_{2}$ ——恒重坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{3}$ ——灼烧前坩埚和试样的质量，单位为克(g)；

$m_{4}$ ——灼烧至恒重后坩埚和残渣的质量，单位为克(g)。

试验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# A.8 氯化物（以C1计）的测定

# A.8.1 试剂和材料

A.8.1.1 硝酸。  
A.8.1.2 硝酸溶液 $(V + V):1 + 9$   
A.8.1.3 硝酸银溶液： $17\mathrm{g / L}$   
A.8.1.4 氯化物标准溶液：按GB/T602配制成质量浓度为 $0.1\mathrm{mg / mL}$ 的储备液。临用前，用水稀释至每 $1\mathrm{mL}$ 标准溶液中相当于含 $0.01\mathrm{mg}$ 氯离子。

# A.8.2 分析步骤

称取 $0.30\mathrm{g}$ 试样，置于 $50~\mathrm{mL}$ 纳氏比色管中，加适量水及 $10~\mathrm{mL}$ 硝酸溶液使其溶解，加 $1\mathrm{mL}$ 硝酸银溶液，用水稀释至 $50~\mathrm{mL}$ ，摇匀，于暗处放置 $5\mathrm{min}$ ，在黑色背景下，从上往下观察，所呈浊度与标准比浊溶液比较。

标准比浊溶液：量取 $6\mathrm{mL}$ 氯化物标准溶液，置于 $50~\mathrm{mL}$ 比色管中。与试样溶液同时同样处理。

# A.8.3 结果判定

试样溶液所呈浊度不深于标准比浊溶液，即试样中的氯化物 $\leqslant 0.02\%$ 。

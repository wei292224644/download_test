# 附录A

# 检验方法

# A. 1 一般规定

本标准除另有规定外，所用试剂的纯度应在分析纯以上，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，实验用水应符合GB/T6682-2008中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

在2~3滴试样中加入 $0.5 \mathrm{~g}$ 硫酸氢钾，加热，有丙烯醛样的气味产生。

# A.3 色泽的测定

# A.3.1 试剂和材料

A.3.1.1 六水合三氯化铁（ $\mathrm{FeCl}_36\mathrm{H}_2\mathrm{O})$   
A.3.1.2 盐酸。  
A.3.1.3 碘化钾。  
A.3.1.4 盐酸溶液： $1 + 39$   
A.3.1.5 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$   
A.3.1.6 淀粉指示液： $10\mathrm{g / L}$

# A.3.2 仪器和设备

A.3.2.1 纳氏比色管： $50~\mathrm{mL}$   
A.3.2.2 比色管架：通常比色管托板为白色，最好安装日光灯照明，提高观察颜色的效果。

# A.3.3 分析步骤

# A. 3.3.1 三氯化铁比色储备液的制备

称取 $55.0 \mathrm{~g}$ 六水合三氯化铁，溶于盐酸溶液中，配制成 $1000 \mathrm{~mL}$ 的溶液。吸取该溶液 $10 \mathrm{~mL}$ ，置于一个 $250 \mathrm{~mL}$ 的碘量瓶中，加入 $15 \mathrm{~mL}$ 水、 $5 \mathrm{~mL}$ 盐酸和 $3 \mathrm{~g}$ 碘化钾，摇匀后静止 $15 \mathrm{~min}$ 。加入 $100 \mathrm{~mL}$ 水稀释，加入淀粉指示液，用硫代硫酸钠标准滴定溶液滴定游离碘。同时进行空白试验，进行必要的修正。消耗的每毫升 $0.1 \mathrm{~mol} / \mathrm{L}$ 的硫代硫酸钠标准滴定溶液相当于 $27.03 \mathrm{mg}$ 的六水合三氯化铁。通过加入盐酸溶液来调整溶液的最终体积，使每毫升溶液含有 $45.0 \mathrm{mg}$ 六水合三氯化铁，得到三氯化铁比色储备液。

# A.3.3.2 三氯化铁标准比色液的制备

移取 $0.4 \mathrm{~mL}$ 三氯化铁比色储备液，置于纳氏比色管中，加水稀释至 $50 \mathrm{~mL}$ ，每毫升溶液含有 $0.36 \mathrm{mg}$ 六水合三氯化铁。

# A.3.3.3 测定

将试样注入 $50~\mathrm{mL}$ 纳氏比色管中至刻度线，与三氯化铁标准比色液一起置于比色管架上。各管外套一黑纸筒，避免侧面光的影响。比较试样和三氯化铁标准比色液的颜色。比色时正对着白色背景，从上向下观测。

# A.3.4 结果判定

试样颜色不深于三氯化铁标准比色液的颜色，即为通过试验。

# A. 4 脂肪酸与酯类的测定

# A.4.1 试剂和材料

A.4.1.1 氢氧化钠溶液： $0.5\mathrm{mol} / \mathrm{L}$   
A.4.1.2 盐酸标准滴定溶液： $c(\mathrm{HCl}) = 0.5\mathrm{mol / L}$   
A.4.1.3 酚酞指示液： $10\mathrm{g / L}$

# A.4.2 仪器和设备

A.4.2.1 圆底烧瓶： $250~\mathrm{mL}$   
A.4.2.2 回流冷凝器。

# A.4.3 分析步骤

移取 $40.0 \mathrm{~mL}$ 试样，置于圆底烧瓶中，加入 $50 \mathrm{~mL}$ 新煮沸的水和 $5.0 \mathrm{~mL}$ 氢氧化钠溶液。将装有混合液的烧瓶与回流冷凝器相连，加热至沸腾，煮沸 $5 \mathrm{~min}$ ，冷却后，加入几滴酚酞指示液，立即用盐酸标准滴定溶液滴定，滴定至溶液红色刚好消失即为滴定终点。

# A.4.4 结果判定

盐酸标准滴定溶液消耗的体积超过 $4 \mathrm{~mL}$ , 即为通过试验。

# A.5 氯化物（以Cl计）的测定

# A.5.1 试剂和材料

A.5.1.1 吗啉。  
A.5.1.2 硝酸。  
A.5.1.3 盐酸溶液： $0.01\mathrm{mol / L}$   
A.5.1.4 硝酸银溶液： $0.1\mathrm{mol} / \mathrm{L}$

# A.5.2 仪器和设备

A.5.2.1 纳氏比色管： $50~\mathrm{mL}$   
A.5.2.2 圆底烧瓶： $100~\mathrm{mL}$   
A.5.2.3 回流冷凝器。

# A.5.3 分析步骤

# A.5.3.1 试样溶液的制备

称取 $5.0\mathrm{g}$ 试样，置于圆底烧瓶中，加入 $15~\mathrm{mL}$ 吗啉。将装有混合液的烧瓶与回流冷凝器相连，温和回流 $3\mathrm{h}$ 。冷却后，用 $10\mathrm{mL}$ 水洗涤冷凝器，洗涤液接入烧瓶中，小心用硝酸酸化烧瓶中的溶液。将溶液转移到纳氏比色管中，加入 $0.5\mathrm{mL}$ 硝酸银溶液，用水稀释至 $50~\mathrm{mL}$ 并混匀。

# A.5.3.2 对照液的制备

用 $0.40 \mathrm{~mL}$ 盐酸溶液代替试样，不进行加热和回流，其他步骤同试样溶液的制备（A.5.3.1）。

# A.5.3.3 测定

比较试样溶液和对照液的浊度。试样溶液的浊度不得超过对照液的浊度，相当于氯化物（以Cl计）含量不得超过 $0.003\%$ 。

# A. 6 易炭化合物的测定

# A.6.1 试剂和材料

A.6.1.1 六水合氯化钴（ $\mathrm{CoCl}_26\mathrm{H}_2\mathrm{O}$ ）。  
A.6.1.2 六水合三氯化铁（ $\mathrm{FeCl}_36\mathrm{H}_2\mathrm{O}$ ）。  
A.6.1.3 碘化钾。  
A. 6.1.4 盐酸。  
A.6.1.5 硫酸。  
A.6.1.6 盐酸溶液： $25 + 975$   
A.6.1.7 过氧化氢溶液：质量分数为 $3\%$   
A.6.1.8 氢氧化钠溶液： $200\mathrm{g / L}$   
A.6.1.9 硫酸溶液： $1 + 4$   
A.6.1.10 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$   
A.6.1.11 淀粉指示液： $10\mathrm{g / L}$

# A.6.2 仪器和设备

具塞比色管： $25\mathrm{mL}$ ，使用前用硫酸清洗干净。

# A.6.3 分析步骤

# A.6.3.1 氯化钴比色储备液的制备

称取 $65.0\mathrm{g}$ 六水合氯化钴，溶于盐酸溶液中，配制成 $1000~\mathrm{mL}$ 的溶液。吸取该溶液 $5\mathrm{mL}$ ，置于一个 $250~\mathrm{mL}$ 的碘量瓶中，加入 $5\mathrm{mL}$ 过氧化氢溶液和 $15\mathrm{mL}$ 氢氧化钠溶液，煮沸 $10\mathrm{min}$ ，冷却，再加入 $2\mathrm{g}$ 碘化钾和 $20~\mathrm{mL}$ 硫酸溶液。当沉淀物溶解后，用硫代硫酸钠标准滴定溶液滴定游离碘。此滴定过程对空气氧化十分敏感，应用二氧化碳进行保护。消耗的每毫升 $0.1\mathrm{mol/L}$ 的硫代硫酸钠标准滴定溶液相当于 $23.79\mathrm{mg}$ 的六水合氯化钴。通过加入盐酸溶液来调整溶液的最终体积，使每毫升溶液含有 $59.5\mathrm{mg}$ 六水合氯化钴，得到氯化钴比色储备液。

# A.6.3.2 氯化铁比色储备液的制备

称取 $55.0 \mathrm{~g}$ 六水合三氯化铁，溶于盐酸溶液中，配制成 $1000 \mathrm{~mL}$ 的溶液。吸取该溶液 $10 \mathrm{~mL}$ ，置于一个 $250 \mathrm{~mL}$ 的碘量瓶中，加入 $15 \mathrm{~mL}$ 水、 $5 \mathrm{~mL}$ 盐酸和 $3 \mathrm{~g}$ 碘化钾，摇匀后静止 $15 \mathrm{~min}$ 。加入 $100 \mathrm{~mL}$ 水稀释，加入淀粉指示液，用硫代硫酸钠标准滴定溶液滴定游离碘。同时进行空白试验，进行必要的修正。消耗的每毫升 $0.1 \mathrm{~mol} / \mathrm{L}$ 的硫代硫酸钠标准滴定溶液相当于 $27.03 \mathrm{mg}$ 的六水合三氯化铁。通过加入盐酸溶液来调整溶液的最终体积，使每毫升溶液含有 $45.0 \mathrm{mg}$ 六水合三氯化铁，得到三氯化铁比色储备液。

# A.6.3.3 标准比色液的制备

分别移取 $0.4 \mathrm{~mL}$ 氯化钴比色储备液、 $3.0 \mathrm{~mL}$ 氯化铁比色储备液和 $6.6 \mathrm{~mL}$ 水，同置于一支具塞比色管中，混合均匀。

# A.6.3.4 试样溶液的制备

分别移取 $5\mathrm{mL}$ 试样和 $5\mathrm{mL}$ 硫酸，同置于另一支具塞比色管中，控制温度 $18^{\circ}\mathrm{C} \sim 20^{\circ}\mathrm{C}$ ，轻轻混合 $1\mathrm{min}$ ，静置 $1\mathrm{h}$ 。

# A.6.3.5 测定

在白色背景下，比较试样溶液和标准比色液的颜色。

# A.6.4 结果判定

试样溶液的颜色不深于标准比色液，即为通过试验。

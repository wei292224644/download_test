# 水产品中甲氧苄啶残留量的测定

# 高效液相色谱法

# 1 范围

本标准规定了水产品中甲氧苄啶残留量的制样和高效液相色谱测定方法。

本标准适用于鱼（包括鳗鲡）、虾、蟹和龟鳖可食组织中甲氧苄啶残留量的检测。

# 3 原理

试料中残留的甲氧苄啶，用三氯甲烷和酸性甲醇溶液提取，二氯甲烷反萃取，MCX固相萃取柱净化，高效液相色谱-紫外测定，外标法定量。

# 4 试剂和材料

以下所用的试剂，除特别注明外均为分析纯试剂；水为符合GB/T6682规定的一级水。

4.1 甲氧苄啶标准品：含量 $\geqslant 98\%$   
4.2 三氯甲烷：色谱纯。  
4.3 甲醇：色谱纯。  
4.4 高氯酸：优级纯。  
4.5 硫酸：优级纯。  
4.6 二氯甲烷：色谱纯。  
4.7 氢氧化钠：优级纯。

4.8 乙酸：优级纯。  
4.9 氨水  
4.10 MCX阳离子固相萃取柱： $60\mathrm{mg} / 3\mathrm{mL}$ ，或相当者。  
4.11 $0.5\%$ 高氯酸溶液：取高氯酸 $5 \mathrm{~mL}$ ，用水溶解并稀释至 $1000 \mathrm{~mL}$ 。  
4.12 $0.1 \mathrm{~mol} / \mathrm{L}$ 硫酸溶液：取硫酸 $5.4 \mathrm{~mL}$ ，用水溶解并稀释至 $1000 \mathrm{~mL}$ 。  
4.13 $2\mathrm{~mol/L}$ 氢氧化钾：取氢氧化钾 $112.2\mathrm{~g}$ ，用水溶解并稀释至 $1000\mathrm{~mL}$ 。  
4.14 $5\%$ 乙酸溶液：取乙酸 $50~\mathrm{mL}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。  
4.15 $5\%$ 氨水甲醇溶液：取氨水 $5 \mathrm{ml}$ ，用甲醇溶解并稀释 $100 \mathrm{~mL}$ 。  
4.16 $100 \mu \mathrm{g} / \mathrm{mL}$ 甲氧苄啶标准贮备液：精密称取甲氧苄啶标准品 $10 \mathrm{mg}$ ，于 $100 \mathrm{~mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $100 \mu \mathrm{g} / \mathrm{mL}$ 的甲氧苄啶标准贮备液。-4℃以下避光保存，有效期3个月。  
4.17 $10 \mu \mathrm{g} / \mathrm{mL}$ 甲氧苄啶标准工作液：精密量取甲氧苄啶标准贮备液 $1.0 \mathrm{~mL}$ ，于 $10 \mathrm{~mL}$ 量瓶中，用甲醇稀释至刻度，配制成浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 的甲氧苄啶标准工作液。-4℃以下避光保存，有效期3个月。

# 5 仪器

5.1 高效液相色谱仪：配紫外检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 均质机  
5.5 旋涡混合器  
5.6 离心机  
5.7 旋转蒸发器  
5.8 固相萃取装置  
5.9 茄形瓶   
5.10 具塞离心管： $50~\mathrm{mL}$   
5.11 分液漏斗： $150~\mathrm{mL}$   
5.12 滤膜： $0.45\mu \mathrm{m}$ 。

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-18 ℃ 以下保存。

# 7 测定步骤

# 7.1 提取

称取试料（ $5 \pm 0.05$ ）g，于 $50 \mathrm{~mL}$ 具塞离心管中，加三氯甲烷 $15 \mathrm{~mL}$ 、甲醇 $14 \mathrm{~mL}$ 、 $0.1 \mathrm{~mol} / \mathrm{L}$ 硫酸溶液 $6 \mathrm{~mL}$ ，旋涡混合 $2 \mathrm{~min}$ ， $4000 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ ，取上清液于 $150 \mathrm{~mL}$ 分液漏斗中。残渣中加甲醇 $14 \mathrm{~mL}$ 和 $0.1 \mathrm{~mol} / \mathrm{L}$ 硫酸溶液 $6 \mathrm{~mL}$ ，重复提取一次，合并两次上清液于分液漏斗中，加 $2 \mathrm{~mol} / \mathrm{L}$ 氢氧化钾溶液 $2 \mathrm{~mL}$ 、二氯甲烷 $30 \mathrm{~mL}$ ，振摇 $2 \mathrm{~min}$ ，静置分层，取下层液于茄形瓶中，分液漏斗中加二氯甲烷 $30 \mathrm{~mL}$ 重复提取一次，合并两次下层液，于 $40^{\circ} \mathrm{C}$ 旋转蒸发至近干，用 $5 \%$ 乙酸溶液 $6 \mathrm{~mL}$ 溶解残余物，备用。

# 7.2 净化

MCX阳离子交换柱依次用甲醇 $6 \mathrm{~mL}$ 和 $5 \%$ 乙酸溶液 $6 \mathrm{~mL}$ 活化，取备用液过柱，用 $5 \%$ 乙酸溶液 $6 \mathrm{~mL}$ 和甲醇 $6 \mathrm{~mL}$ 淋洗，用氨水甲醇溶液 $15 \mathrm{~mL}$ 洗脱，于 $40^{\circ} \mathrm{C}$ 旋转蒸发至近干，用流动相 $1.0 \mathrm{~mL}$ 溶解残余物，滤膜过滤，供高效液相色谱测定。

# 7.3 标准曲线的制备

精密量取甲氧苄啶标准工作液适量，用流动相稀释，配制成浓度为50、200、500、1000、2000、 $5000\mu \mathrm{g} / \mathrm{L}$ 系列标准溶液，供高效液相色谱测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.4 测定

# 7.4.1 色谱条件

色谱柱：ZORBAX- $\mathrm{C_{18}}$ 柱（ $250\mathrm{mm}\times 4.6\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ），或相当者。

流动相：甲醇+0.5%高氯酸溶液（30+70，v/v）；

流速： $1.0\mathrm{mL / min}$

柱温： $35^{\circ}C$

进样量： $20\mu \mathrm{L}$

检测波长： $230~\mathrm{nm}$

# 7.4.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试样溶液中甲氧苄啶响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和空白添加试样溶液的高效液相色谱图见附录A。

# 7.5 空白试验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算

试料中甲氧苄啶的残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算

$$
X = \frac {C \times V}{m}
$$

式中：

$X$ ——供试试料中甲氧苄啶的残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

C—试样溶液中甲氧苄啶的浓度, $\mu \mathrm{g} / \mathrm{mL}$ ;

$V$ ——溶解残余物体积， $\mathrm{mL}$ 。

$m$ ——供试试料质量，g；

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 方法灵敏度、准确度和精确度

# 9.1 灵敏度

本方法的定量限为 $20\mu \mathrm{g / kg}$

# 9.2 准确度

本方法在 $20\sim 1000~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$

# 9.3 方法精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

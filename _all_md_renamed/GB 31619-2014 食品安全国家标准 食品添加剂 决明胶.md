# 食品添加剂 决明胶

# 1 范围

本标准适用于以决明(Cassia obtusifolia或Cassia tora)植物的种子胚乳为原料，经萃取加工而成的食品添加剂决明胶。主要含半乳甘露聚糖，即包含甘露糖线性主链和半乳糖侧链的聚合物。

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度应在分析纯以上，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 溶解性试验

不溶于乙醇。分散于冷水中，形成胶状溶液。

# A.2.2 凝胶试验

A.2.2.1 在试样溶液中加入足量的硼酸钠 $\mathrm{(Na_2B_4O_7\cdot 10H_2O)}$ 试液 $(20~\mathrm{g / L})$ ，使溶液的 $\mathrm{pH}$ 超过9，溶液形成凝胶。  
A.2.2.2 称取 $1.5\mathrm{g}$ 试样和 $1.5\mathrm{g}$ 黄原胶，混合均匀。在快速搅拌下，将混合物加入到盛有 $300~\mathrm{mL}$ $80^{\circ}\mathrm{C}$ 水的 $400~\mathrm{mL}$ 烧杯中。搅拌至混合物溶解，继续搅拌 $30\mathrm{min}$ （搅拌时溶液温度保持在 $60^{\circ}\mathrm{C}$ 以上）。停止搅拌并让混合物在室温下冷却至少 $2\mathrm{h}$ 。在温度降至 $40^{\circ}\mathrm{C}$ 以下后，形成结实、有黏弹性的胶体。而单独的 $10\mathrm{g / L}$ 试样对照液或黄原胶对照液均不形成此凝胶。

# A.2.3 $\mathbf{pH}$

$10\mathrm{g / L}$ 试样溶液的 $\mathrm{pH}$ 应为 $5.5\sim 8.0$

# A.3 半乳甘露聚糖的测定

半乳甘露聚糖的质量分数 $\omega_{1}$ 按式(A.1)计算：

$$
w _ {1} = 100 \% - w _ {2} - w _ {3} - w _ {4} - w _ {5} - w _ {6} \quad \dots \tag{A.1}
$$

式中：

$\omega_{2}$ ——干燥减量的质量分数， $\%$ ；

$w_{3}$ ——灰分的质量分数， $\%$ ；

$w_{4}$ ——酸不溶物的质量分数， $\%$ ；

$w_{5}$ ——蛋白质的质量分数， $\%$ ；

$\varpi_{6}$ ——脂肪的质量分数， $\%$ 。

# A.4 酸不溶物的测定

# A.4.1 试剂和材料

A.4.1.1 硫酸。

A.4.1.2 助滤剂：硅藻土，经 $105\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}, 3\,\mathrm{h}$ 干燥处理。

# A.4.2 仪器和设备

A.4.2.1 过滤坩埚（经 $105\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}, 3\,\mathrm{h}$ 干燥处理）。  
A.4.2.2 干燥器。

# A.4.3 分析步骤

称取 $2.0\mathrm{g}$ 试样，溶于一盛有 $150~\mathrm{mL}$ 水和 $1.5\mathrm{mL}$ 硫酸的 $250~\mathrm{mL}$ 烧杯中。用表面皿盖上烧杯，在蒸气浴上加热 $6\mathrm{h}$ ，加热过程中随时补充蒸发损失掉的水分。加热完成后，称取干燥处理后的助滤剂 $500\mathrm{mg}$ ，加入到试样溶液中，用已称重的过滤坩埚进行过滤。用热水洗涤滤渣数次，然后将坩埚连同滤渣在 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 下干燥 $3\mathrm{h}$ ，在干燥器内冷却后称重。

# A.4.4 结果计算

酸不溶物的质量分数 $w_{4}$ 按式(A.2)计算：

$$
w _ {4} = \frac {m _ {1} - m _ {2} - m _ {3}}{m _ {4}} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$m_{1}$ ——干燥后坩埚连同滤渣的总质量，单位为克(g)；

$m_{2}$ ——助滤剂的质量，单位为克 $(\mathrm{g})$

$m_{3}$ —— 坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{4}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

# A.5 淀粉试验

# A.5.1 试剂和材料

碘溶液：称取碘 $14.0\mathrm{g}$ ，溶于含有碘化钾 $36.0\mathrm{g}$ 的 $100\mathrm{mL}$ 水溶液中，加入3滴盐酸，加水稀释至 $1000\mathrm{mL}$ 。

# A.5.2 分析步骤

称取试样 $1.0\mathrm{g}$ ，分散于 $10~\mathrm{mL}$ 水中。加入碘溶液，无蓝色出现，即为通过试验。

# A.6 葱醌的测定

# A.6.1 方法提要

用乙腈提取试样中的蒽醌，通过高效液相色谱法进行测定。

注：试样和对照品应避光保存。

# A.6.2 试剂和材料

A.6.2.1 葱醌对照品：大黄素(EMO)(纯度 $\geqslant 90\%$ )、芦荟大黄素(AEM)(纯度 $\geqslant 95\%$ )和大黄素甲醚(PHY)(纯度 $\geqslant 98.0\%$ )，或者1,8-二羟基-3-甲氧基-6-甲基-蒽醌、大黄酸(RHE)(纯度 $\geqslant 95\%$ )和大黄根酸(CHR)(纯度 $\geqslant 98\%$ )。  
A.6.2.2 内标对照品：1，8-二羟基蒽醌（纯度 $\geq 96\%$ ）。  
A.6.2.3 甲醇：色谱纯。  
A.6.2.4 乙腈：色谱纯。

A.6.2.5 三氟乙酸。  
A.6.2.6 碳酸氢钠溶液： $2\mathrm{g} / \mathrm{L}$   
A.6.2.7 乙腈/碳酸氢钠溶液：乙腈和碳酸氢钠溶液的体积比为 $60:40$ 。  
A.6.2.8 缓冲溶液： $\mathrm{pH}9.0$

# A.6.3 仪器和设备

高效液相色谱仪，配二极管阵列检测器（波长 $435\mathrm{nm}$ ）。

# A.6.4 参考色谱条件

A.6.4.1 色谱柱： $\mathrm{C_{18}}$ 色谱柱， $250 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ ，粒度 $5 \mu \mathrm{m}$ 。或其他等同分离效果的色谱柱和色谱条件。  
A.6.4.2 流动相： $\mathrm{A + B}$ ；A为 $0.1\%$ 三氟乙酸溶液，B为乙腈。  
A.6.4.3 运行时间： $60\mathrm{min}$   
A.6.4.4 梯度：见表A.1。

表 A. 1 流动相的浓度配比  

<table><tr><td>时间/min</td><td>流动相A/%</td><td>流动相B/%</td></tr><tr><td>0</td><td>86</td><td>14</td></tr><tr><td>10</td><td>86</td><td>14</td></tr><tr><td>15</td><td>80</td><td>20</td></tr><tr><td>25</td><td>80</td><td>20</td></tr><tr><td>55</td><td>20</td><td>80</td></tr><tr><td>60</td><td>0</td><td>100</td></tr></table>

A.6.4.5 流速： $1\mathrm{mL} / \mathrm{min}$

A.6.4.6 进样量： $50~\mu \mathrm{L}$

# A.6.5 分析步骤

# A.6.5.1 标准贮备溶液 $(100\mathrm{mg / L})$ 的制备

称取3种蒽醌对照品和内标对照品各 $1\mathrm{mg}\pm 0.01\mathrm{mg}$ ，分别用约 $5\mathrm{mL}$ 甲醇将对照品分别转移至 $10~\mathrm{mL}$ 的容量瓶中，超声处理 $15\mathrm{min}$ 后，加甲醇稀释至刻度。

此4份标准贮备溶液在 $4^{\circ}\mathrm{C}$ 下贮存于棕色瓶中（此条件下溶液可稳定2周）。

# A.6.5.2 混合标准溶液 $(10\mathrm{mg / L})$ 的制备

3份蒽醌标准贮备溶液各吸取 $1\mathrm{mL}$ ，置于一个 $10~\mathrm{mL}$ 的容量瓶中，加甲醇稀释至刻度。

# A.6.5.3 标准工作溶液的制备

取5个 $10\mathrm{mL}$ 的容量瓶，分别加入 $5\mathrm{mL},2\mathrm{mL},1\mathrm{mL},0.5\mathrm{mL}$ 和 $0\mathrm{mL}$ 混合标准溶液，再分别加入 $1\mathrm{mL}$ 内标标准贮备溶液，混合后，分别加甲醇稀释至刻度。

# A.6.5.4 试样溶液的制备

称取约 $0.4\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于一个 $50~\mathrm{mL}$ 圆底烧瓶中。加入 $20~\mathrm{mL}$ 三氟乙酸，在 $70^{\circ}\mathrm{C}$ 下

加热回流 $4\mathrm{h}$ 。将试样冷却至室温，并用旋转蒸发器蒸发至干。加入 $3\mathrm{mL}$ 乙腈/碳酸氢钠溶液，超声处理 $30\mathrm{min}$ 。将溶液转移至一个离心管中，在 $5000\mathrm{r / min}$ 下离心 $30\mathrm{min}$ 。用事先经 $\mathrm{pH}9.0$ 的缓冲溶液中和过的萃取柱(Merck,NT1或其他等效柱)过滤上清液。吸取 $900~\mu \mathrm{L}$ 过滤后的试样溶液，置于一个 $2.5\mathrm{mL}$ 的小瓶中，加入 $100~\mu \mathrm{L}$ 内标标准贮备溶液，充分混匀。

# A.6.5.5 标准曲线的绘制

在A.6.4参考色谱条件下，分别对各个标准工作溶液和内标标准贮备溶液进行色谱分析，记录色谱图中各蒽醌及内标的峰面积。以各蒽醌与内标的峰面积比值对各个标准工作溶液浓度 $(\mathrm{mg / L})$ 作标准曲线。

# A.6.5.6 测定

在A.6.4参考色谱条件下，分别对试样溶液和内标标准贮备溶液进行色谱分析，记录色谱图中各蒽醌及内标的峰面积。计算各蒽醌与内标的峰面积比值，根据标准曲线，得到各蒽醌的浓度。

# A.6.6 结果计算

各蒽醌的含量 $\omega_{7}$ 以毫克每千克 $(\mathrm{mg / kg})$ 计，按式(A.3)计算：

$$
w _ {7} = \frac {c \times 3 \times 1 0 0 0}{1 0 0 0 \times 0 . 9 \times m _ {5}} \tag {A.3}
$$

式中：

$c$ ——根据标准曲线得到的试样溶液中各蒽醌的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

3 ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$

1000——质量换算系数；

1000——体积换算系数；

0.9 ——取样体积，单位为毫升 $(\mathrm{mL})$ ；

$m_{5}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

由式(A.3)计算得到的各蒽醌的含量之和即为试样中蒽醌的含量。

# A.7 异丙醇的测定

# A.7.1 试剂和材料

A.7.1.1 异丙醇：色谱纯。

A.7.1.2 叔丁醇：色谱纯。

# A.7.2 仪器和设备

气相色谱仪，配有火焰离子化检测器。

# A.7.3 参考色谱条件

A.7.3.1 色谱柱：填料为 $0.150\mathrm{mm}\sim 0.180\mathrm{mm}(80$ 目 $\sim 100$ 目)硅烷化的乙基乙烯苯与二乙烯苯共聚物或其他等同物质， $1.8\mathrm{m}\times 3.2\mathrm{mm}$ （内径）。或其他等同分离效果的色谱柱和色谱条件。

A.7.3.2 载气：氦气或氮气。  
A.7.3.3 流速： $80~\mathrm{mL / min}$   
A.7.3.4 进样口温度： $200^{\circ}C$   
A.7.3.5 柱温： $165~\mathrm{^\circ C}$   
A.7.3.6 检测器温度： $200^{\circ}C$

A.7.3.7 进样量： $5\mu \mathrm{L}$

# A.7.4 分析步骤

# A.7.4.1 异丙醇标准溶液的制备

称取 $100\mathrm{mg}$ 异丙醇，置于一个装有约 $90~\mathrm{mL}$ 水的 $100~\mathrm{mL}$ 容量瓶中，加水稀释至 $100~\mathrm{mL}$ ，混匀。

# A.7.4.2 叔丁醇标准溶液的制备

称取 $100\mathrm{mg}$ 叔丁醇，置于一个装有约 $90~\mathrm{mL}$ 水的 $100~\mathrm{mL}$ 容量瓶中，加水稀释至 $100~\mathrm{mL}$ ，混匀。

# A.7.4.3 混合标准溶液的制备

吸取异丙醇和叔丁醇标准溶液各 $4\mathrm{mL}$ ，置于一个 $100~\mathrm{mL}$ 容量瓶中，加水稀释至 $100~\mathrm{mL}$ ，混匀。该溶液含异丙醇和叔丁醇各 $40~\mu \mathrm{g / mL}$ 。

# A.7.4.4 试样溶液的制备

在一个盛有 $200~\mathrm{mL}$ 水的 $1000~\mathrm{mL}$ 圆底蒸馏烧瓶中，加入 $1\mathrm{mL}$ 合适的消泡剂，使其分散。加入准确称量的约 $5\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ ），振荡 $1\mathrm{h}$ 。将此烧瓶与分馏柱相连，调节温度，使泡沫不进入柱子，接馏出液约 $95~\mathrm{mL}$ 。在馏出液中加入 $4\mathrm{mL}$ 叔丁醇标准溶液，加水补充至 $100~\mathrm{mL}$ ，即得试样溶液。

# A.7.4.5 测定

在A.7.3参考色谱条件下，分别对混合标准溶液和试样溶液进行色谱分析。记录各色谱图中异丙醇和叔丁醇的峰面积值。

# A.7.5 结果计算

# A.7.5.1 响应因子的计算

响应因子 $f$ 按式(A.4)计算：

$$
f = \frac {A _ {\mathrm {I P A}}}{A _ {\mathrm {T B A}}} \quad \dots \tag {A.4}
$$

式中：

$A_{\mathrm{IPA}}$ ——混合标准溶液色谱图中异丙醇的峰面积值；

$A_{\mathrm{TBA}}$ ——混合标准溶液色谱图中叔丁醇的峰面积值。

# A.7.5.2 异丙醇含量的计算

异丙醇含量 $w_{8}$ 以毫克每千克 $(\mathrm{mg / kg})$ 计，按式(A.5)计算：

$$
w _ {8} = \frac {S _ {\mathrm {I P A}} \times 4 0 0 0}{f \times S _ {\mathrm {T B A}} \times m _ {6}} \quad \dots \tag {A.5}
$$

式中：

$S_{\mathrm{IPA}}$ ——试样溶液色谱图中异丙醇的峰面积值；

4000——换算系数；

$f$ ——响应因子；

$S_{\mathrm{TBA}}$ ——试样溶液色谱图中叔丁醇的峰面积值；

$m_{6}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

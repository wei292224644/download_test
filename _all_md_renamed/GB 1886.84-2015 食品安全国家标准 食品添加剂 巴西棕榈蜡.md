# 食品添加剂 巴西棕榈蜡

# 1 范围

本标准适用于以巴西棕榈蜡(Copernia cerifera)的叶和叶芽为原料，经提取精制制得的食品添加剂巴西棕榈蜡。

# 2 分子式和相对分子质量

# 2.1 分子式

$$
\mathrm {C} _ {7} \mathrm {H} _ {5} \mathrm {H g N O} _ {3}
$$

# 2.2 相对分子质量

351.70（按2007年国际相对原子质量）

# 检验方法

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 溶解性

不溶于水，微溶于沸腾乙醇，溶于乙醚。

# A.2.2 熔程

$$
8 0 ^ {\circ} \mathrm {C} \sim 8 6 ^ {\circ} \mathrm {C} 。
$$

# A.3 酸值（以KOH计）的测定

# A.3.1 试剂和材料

A.3.1.1 氢氧化钠标准滴定溶液： $c_{1}(\mathrm{NaOH}) = 0.1\mathrm{mol / L}$

A.3.1.2 酚酞指示液： $10\mathrm{g / L}$

# A.3.2 分析步骤

称取约 $10\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，溶于预先用氢氧化钠标准滴定溶液中和（以酚酞指示液指示的）的 $50~\mathrm{mL}$ 中性乙醇中。若不能完全溶解，可缓慢加热回流，以使之完全溶解。再加酚酞指示液 $1\mathrm{mL}$ ，用氢氧化钠标准滴定溶液滴定至溶液呈粉红色，并振摇 $10\mathrm{s}$ 不褪色。

# A.3.3 结果计算

酸值（以KOH计） $w_{1}$ ，单位为毫克每克 $(\mathrm{mg / g})$ ，按式(A.1)计算：

$$
w _ {1} = \frac {V _ {1} \times c _ {1} \times M _ {1}}{m _ {1}} \quad \dots \tag {A.1}
$$

式中：

$V_{1}$ ——氢氧化钠标准滴定溶液(A.3.1.1)的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——氢氧化钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{\mathrm{r}}$ ——氢氧化钾的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $(M = 56.109)$

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

# A.4 皂化值（以KOH计）的测定

# A.4.1 试剂和材料

A.4.1.1 无水乙醇。

A.4.1.2 氢氧化钾乙醇溶液： $40~\mathrm{g / L}$   
A.4.1.3 盐酸标准滴定溶液： $c_{2}(\mathrm{HCl}) = 0.5\mathrm{mol / L}$   
A.4.1.4 酚酞指示液： $10\mathrm{g / L}$

# A.4.2 仪器和设备

空气冷凝管：长度大于 $65~\mathrm{cm}$

# A.4.3 分析步骤

称取适量试样，精确至 $0.0001\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 磨口锥形瓶中，试样加入量一般按试样经皂化后所需 $0.5\mathrm{mol} / \mathrm{L}$ 盐酸标准滴定溶液的量约占对照样的 $45\% \sim 50\%$ 为宜。加入 $50~\mathrm{mL} \pm 0.02~\mathrm{mL}$ 氢氧化钾乙醇溶液，连接空气冷凝管，加热回流 $1\mathrm{h}$ ，稍冷后用 $10~\mathrm{mL}$ 无水乙醇淋洗冷凝管，取下锥形瓶，加入5滴酚酞指示液，用盐酸标准滴定溶液滴定至溶液的红色刚刚消失，加热试液至沸。若出现粉红色，继续滴定至红色消失即为终点。

在测定的同时，用等量的氢氧化钾乙醇溶液做空白试验。

# A.4.4 结果计算

皂化值（以KOH计） $\omega_{2}$ ，单位为毫克每克 $(\mathrm{mg / g})$ ，按式(A.2)计算：

$$
w _ {2} = \frac {\left(V _ {0} - V _ {2}\right) \times c _ {2} \times M _ {2}}{m _ {2}} \tag {A.2}
$$

式中：

$V_{0}$ ——空白试验消耗盐酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试料消耗盐酸标准滴定溶液(A.4.1.3)的体积，单位为毫升 $(\mathrm{mL})$

$c_{2}$ ——盐酸标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_2$ ——氢氧化钾的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $(M = 56.109)$

$m_{2}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

# A.5 灼烧残渣的测定

# A.5.1 试剂和材料

硫酸溶液： $6\rightarrow 100$

# A.5.2 仪器和设备

瓷坩埚或铂坩埚： $50~\mathrm{mL} \sim 100~\mathrm{mL}$ 。

# A.5.3 分析步骤

称取约 $2\mathrm{g}$ 试样，精确至 $0.001\mathrm{g}$ ，置于已恒重的瓷坩埚或铂坩埚中。用电加热板缓慢加热，使其挥发至无辛辣气味释放。加入 $10\mathrm{mL}$ 硫酸溶液以湿润全部试样，缓慢加热使硫酸逸尽。置于高温炉中，在 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 灼烧 $15\mathrm{min}$ 或以上，放入干燥器中冷却，称量。

# A.6 不皂化物的测定

# A.6.1 试剂和材料

A.6.1.1 石油醚。

A.6.1.2 中性乙醇：加酚酞指示液数滴于一定量的温热乙醇中，用氢氧化钠溶液中和至刚开始出现微粉红色。  
A.6.1.3 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.02\mathrm{mol / L}$

# A.6.2 仪器和设备

A.6.2.1 真空干燥箱。  
A.6.2.2 具塞量筒：标有 $40~\mathrm{mL},80~\mathrm{mL}$ 和 $130~\mathrm{mL}$ 的刻度，高约 $30~\mathrm{cm}$ ，直径 $3.5~\mathrm{cm}$ 。

# A.6.3 分析步骤

称取 $5.0\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $250\mathrm{mL}$ 烧瓶中，加入由 $2\mathrm{g}$ 氢氧化钾和 $40\mathrm{mL}$ 乙醇组成的溶液，装上回流冷凝器，小心煮沸回流 $1\mathrm{h}$ ，至完全皂化为止。

将烧瓶中的内容物移入量筒，用一定量的乙醇洗涤烧瓶，使量筒中的容量达 $40~\mathrm{mL}$ 。再用煮沸并冷却后的水洗涤烧瓶，洗液并入量筒，并加至总容量达 $80~\mathrm{mL}$ 。用石油醚数毫升小心洗涤烧杯，将洗液加入量筒，冷却量筒中的内容物至室温，再加石油醚至 $130~\mathrm{mL}$ 。加塞，强烈振摇量筒 $1\mathrm{min}$ 以上，静置后应分成透明两层。尽可能完全地将上层石油醚层移入 $500~\mathrm{mL}$ 分液漏斗中。另用石油醚如上萃取六次以上，每次用石油醚 $50~\mathrm{mL}$ 并均经强烈振摇。集中所有萃取液，每次用 $10\%$ 乙醇 $25~\mathrm{mL}$ 洗涤石油醚，一直到洗出液对酚酞呈中性为止，弃去洗涤液，将石油醚萃取液放入已知重量的烧杯中，用 $10~\mathrm{mL}$ 石油醚淋洗分液漏斗，洗液并入烧杯。

将盛有石油醚萃取液的烧杯放在蒸汽浴上蒸发至干，在 $75^{\circ}\mathrm{C} \sim 80^{\circ}\mathrm{C}, 26664\mathrm{Pa}(200\mathrm{mmHg})$ 真空下干燥至质量恒定，或在 $100^{\circ}\mathrm{C}$ 下干燥 $30\mathrm{min}$ 。放入干燥器中冷却，称量，为不皂化物的未校准值。

将蒸发残渣溶于 $50~\mathrm{mL}$ 温的中性乙醇中，以酚酞做指示剂，用氢氧化钠标准滴定溶液滴定。计算脂肪酸（以油酸计）的量。每毫升 $c(\mathrm{NaOH}) = 0.02 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠相当于脂肪酸（以油酸计） $5.659 \mathrm{mg}$ 。

# A.6.4 结果计算

从未校准的不皂化物量中减去脂肪酸的量，即为不皂化物的量。

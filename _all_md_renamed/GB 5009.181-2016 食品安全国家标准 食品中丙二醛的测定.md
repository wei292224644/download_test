# 前言

本标准代替GB/T5009.181—2003《猪油中丙二醛的测定》。

本标准与GB/T5009.181—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中丙二醛的测定”；  
——保留原标准方法分光光度法为第二法，扩大了方法的适用范围；  
——增加了高效液相色谱法为第一法，扩大了方法的适用范围；  
——增加了附录A。

# 食品安全国家标准

# 食品中丙二醛的测定

# 1 范围

本标准规定了食品中丙二醛的测定方法。

本标准第一法适用于食品中丙二醛的测定；本标准第二法适用于动植物油脂中丙二醛的测定。

# 第一法 高效液相色谱法

# 2 原理

试样先用酸液提取，再将提取液与硫代巴比妥酸(TBA)作用生成有色化合物，采用高效液相色谱-二极管阵列检测器测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 三氯乙酸 $(\mathrm{C}_2\mathrm{HCl}_3\mathrm{O}_2)$ 。  
3.1.3 乙二胺四乙酸二钠 $(\mathrm{C_{10}H_{14}N_2Na_2O_8}\cdot 2\mathrm{H_2O})$   
3.1.4 硫代巴比妥酸（TBA） $(\mathrm{C_4H_4N_2O_2S})$ 。

# 3.2 试剂配制

3.2.1 乙酸铵溶液 $(0.01\mathrm{mol} / \mathrm{L})$ ：称取 $0.77\mathrm{g}$ 乙酸铵，加水溶解，定容至 $1000~\mathrm{mL}$ ，经 $0.45\mu \mathrm{m}$ 过滤器过滤。  
3.2.2 三氯乙酸混合液：准确称取 $37.50\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ) 三氯乙酸(3.1.2)及 $0.50\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）乙二胺四乙酸二钠(3.1.3)，用水溶解，稀释至 $500~\mathrm{mL}$ 。  
3.2.3 硫代巴比妥酸(TBA)水溶液: 准确称取 $0.288 \mathrm{~g}$ (精确至 $0.001 \mathrm{~g}$ ) 硫代巴比妥酸(3.1.4)溶于水中, 并稀释至 $100 \mathrm{~mL}$ (如不易溶解, 可加热超声至全部溶解, 冷却后定容至 $100 \mathrm{~mL}$ ), 相当于 $0.02 \mathrm{~mol} / \mathrm{L}$ 。

# 3.3 标准品

1,1,3,3-四乙氧基丙烷(又名丙二醛乙缩醛， $\mathrm{C_{11}H_{24}O_4}$ ，CAS号：122-31-6)：纯度 $\geqslant 97\%$ 。

# 3.4 标准溶液配制

3.4.1丙二醛标准储备液 $(100\mu \mathrm{g / mL})$ ：准确移取 $0.315\mathrm{g}$ （精确至 $0.001\mathrm{g})1,1,3,3 -$ 四乙氧基丙烷至

$1000~\mathrm{mL}$ 容量瓶中，用水溶解后稀释至 $1000~\mathrm{mL}$ ，置于冰箱 $4^{\circ}\mathrm{C}$ 储存。有效期3个月。  
3.4.2 丙二醛标准使用溶液 $(1.00\mu \mathrm{g} / \mathrm{mL})$ ：准确移取丙二醛标准储备液(3.4.1) $1.0~\mathrm{mL}$ ，用三氯乙酸混合液(3.2.2)稀释至 $100~\mathrm{mL}$ ，置于冰箱 $4^{\circ}\mathrm{C}$ 储存。有效期2周。  
3.4.3 丙二醛标准系列溶液：准确移取丙二醛标准使用液 $0.10\mathrm{mL},0.50\mathrm{mL},1.0\mathrm{mL},1.5\mathrm{mL},2.5\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，加三氯乙酸混合液(3.2.2)定容至刻度，该标准溶液系列浓度为 $0.01\mu \mathrm{g / mL}$ 、 $0.05\mu \mathrm{g / mL},0.10\mu \mathrm{g / mL},0.15\mu \mathrm{g / mL},0.25\mu \mathrm{g / mL}$ ，现配现用。

# 4 仪器和设备

4.1 高效液相色谱：配有二极管阵列检测器。  
4.2 天平：感量为 $0.0001\mathrm{g}, 0.01\mathrm{g}$ 。  
4.3 恒温振荡器。  
4.4 恒温水浴锅。  
4.5 水相针式滤器， $0.45\mu \mathrm{m}$ 聚醚砜滤膜。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 提取

称取均匀的样品 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），置入 $100~\mathrm{mL}$ 具塞锥形瓶中，准确加入 $50~\mathrm{mL}$ 三氯乙酸混合液，摇匀，加塞密封，置于恒温振荡器上 $50^{\circ}C$ 振摇 $30\mathrm{min}$ ，取出，冷却至室温，用双层定量慢速滤纸过滤，弃去初滤液，续滤液备用。

# 5.1.2 衍生化

准确移取5.1.1滤液和标准系列溶液各 $5\mathrm{mL}$ 分别置于 $25~\mathrm{mL}$ 具塞比色管内，加入 $5\mathrm{mL}$ 硫代巴比妥酸(TBA)水溶液，加塞，混匀，置于 $90^{\circ}C$ 水浴内反应 $30\mathrm{min}$ ，取出，冷却至室温，取适量上层清液过滤膜上机分析。

# 5.2 液相色谱参考条件

色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $150~\mathrm{mm}$ ，内径 $4.6~\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或性能相当者。

流动相： $0.01\mathrm{mol} / \mathrm{L}$ 乙酸铵：甲醇 $= 70:30$ （体积比）。

柱温： $30^{\circ}\mathrm{C}$ 。

流速： $1.0~\mathrm{mL / min}$

进样量： $10~\mu \mathrm{L}$

检测波长： $532\mathrm{nm}$ 。

# 5.3 测定

分别吸取标准系列工作液(3.4.3)和待测试样的衍生溶液注入高效液相色谱仪中，测定相应的峰面积，以标准工作液的浓度为横坐标，以峰面积响应值为纵坐标，绘制标准曲线。根据标准曲线得到待测液中丙二醛的浓度，丙二醛标准品的液相色谱图参见附录A。

# 6 分析结果的表述

试样中丙二醛的含量按式(1)计算：

$$
X = \frac {c \times V \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中丙二醛含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

c 从标准系列曲线中得到的试样溶液中丙二醛的浓度，单位为微克每毫升 $(\mu \mathrm{g / mL})$

$V$ ——试样溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——最终试样溶液所代表的试样质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 分光光度法

# 8 原理

丙二醛经三氯乙酸溶液提取后，与硫代巴比妥酸(TBA)作用生成粉红色化合物，测定其在 $532\mathrm{nm}$ 波长处的吸光度值，与标准系列比较定量。

# 9 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 9.1 试剂

9.1.1 三氯乙酸 $(\mathrm{C}_2\mathrm{HCl}_3\mathrm{O}_2)$ 。  
9.1.2 乙二胺四乙酸二钠 $\mathrm{(C_{10}H_{14}N_2Na_2O_8\cdot 2H_2O)}$   
9.1.3 硫代巴比妥酸（TBA） $(\mathrm{C_4H_4N_2O_2S})$ 。

# 9.2 试剂配制

9.2.1 三氯乙酸混合液：准确称取 $37.50\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ) 三氯乙酸及 $0.50\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）乙二胺四乙酸二钠，用水溶解，稀释至 $500~\mathrm{mL}$ 。  
9.2.2 硫代巴比妥酸(TBA)水溶液：准确称取 $0.288\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )硫代巴比妥酸溶于水中，并稀释至 $100~\mathrm{mL}$ (如不易溶解，可加热超声至全部溶解，冷却后定容至 $100~\mathrm{mL}$ )，相当于 $0.02\mathrm{mol} / \mathrm{L}$ 。

# 9.3 标准品

1,1,3,3-四乙氧基丙烷(又名丙二醛乙缩醛， $\mathrm{C_{11}H_{24}O_4}$ ，CAS号：122-31-6)：纯度 $\geqslant 97\%$ 。

# 9.4 标准溶液配制

9.4.1 丙二醛标准储备液 $(100\mu \mathrm{g / mL})$ ：准确移取 $0.315\mathrm{g}$ （精确至 $0.001\mathrm{g})1,1,3,3 -$ 四乙氧基丙烷至 $1000~\mathrm{mL}$ 容量瓶中，用水溶解后稀释至 $1000~\mathrm{mL}$ ，置于冰箱 $4^{\circ}C$ 储存。有效期3个月。  
9.4.2 丙二醛标准使用溶液 $(1.00\mu \mathrm{g} / \mathrm{mL})$ ：准确移取丙二醛标准储备液 $1.0~\mathrm{mL}$ ，用三氯乙酸混合液稀释至 $100~\mathrm{mL}$ ，置于冰箱 $4^{\circ}\mathrm{C}$ 储存。有效期2周。  
9.4.3丙二醛标准系列溶液：准确移取丙二醛标准使用液 $0.10\mathrm{mL},0.50\mathrm{mL},1.0\mathrm{mL},1.5\mathrm{mL},2.5\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，加三氯乙酸混合液定容至刻度，该标准溶液系列浓度为 $0.01~\mu \mathrm{g / mL},0.05~\mu \mathrm{g / mL}$ $0.10~\mu \mathrm{g / mL},0.15~\mu \mathrm{g / mL},0.25~\mu \mathrm{g / mL}$ ，现配现用。

# 10 仪器和设备

10.1 分光光度计。  
10.2 天平：感量为 $0.0001\mathrm{g}, 0.01\mathrm{g}$ 。  
10.3 恒温振荡器。  
10.4 恒温水浴锅。

# 11 分析步骤

# 11.1 试样制备

称取样品 $5\mathrm{g}$ （精确到 $0.01\mathrm{g}$ )置入 $100~\mathrm{mL}$ 具塞锥形瓶中，准确加入 $50~\mathrm{mL}$ 三氯乙酸混合液，摇匀，加塞密封，置于恒温振荡器上 $50^{\circ}C$ 振摇 $30\mathrm{min}$ ，取出，冷却至室温，用双层定量慢速滤纸过滤，弃去初滤液，续滤液备用。

准确移取上述滤液和标准系列溶液各 $5\mathrm{mL}$ 分别置于 $25~\mathrm{mL}$ 具塞比色管内，另取 $5\mathrm{mL}$ 三氯乙酸混合液作为样品空白，分别加入 $5\mathrm{mL}$ 硫代巴比妥酸(TBA)水溶液，加塞，混匀，置于 $90^{\circ}C$ 水浴内反应 $30\mathrm{min}$ ，取出，冷却至室温。

# 11.2 测定

以样品空白调节零点，于 $532\mathrm{nm}$ 处 $1\mathrm{cm}$ 光径测定样品溶液和标准系列溶液(9.4.3)的吸光度值，以标准系列溶液的质量浓度为横坐标，吸光度值为纵坐标，绘制标准曲线。

# 12 分析结果的表述

试样中丙二醛含量，按式(2)计算：

$$
X = \frac {c \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中丙二醛含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——从标准系列曲线中得到的试样溶液中丙二醛的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——最终试样溶液所代表的试样质量，单位为克(g)；

1000 ——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 13 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 14 其他

第一法检出限为 $0.03\mathrm{mg / kg}$ ，定量限为 $0.10\mathrm{mg / kg}$ ；第二法检出限为 $0.05\mathrm{mg / kg}$ ，定量限为 $0.10\mathrm{mg / kg}$ 。

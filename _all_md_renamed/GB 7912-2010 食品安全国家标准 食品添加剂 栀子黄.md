2010-12-21 发布

2011-02-21 实施

# 前言

本标准代替GB7912—1987《食品添加剂 栀子黄（粉末、浸膏）》。

本标准与GB7912—1987相比，主要变化如下：

——修改了色价、干燥失重、砷、铅等指标；

——增加了栀子苷指标；

——取消了重金属和灼烧残渣的要求。

本标准的附录A为规范性附录。

本标准所代替标准的历次版本发布情况为：

—GB7912-1987。

# 食品安全国家标准

# 食品添加剂 栀子黄

# 1 范围

本标准适用于以茜草科植物栀子（Gardenia jasminoides Ellis）的果实为原料，经提取、精制而成，可用糊精稀释的粉末、浸膏或液态的食品添加剂栀子黄。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

# 3 分子式和相对分子质量

# 3.1 分子式

藏花素： $\mathrm{C_{44}H_{64}O_{24}}$

藏花酸： $\mathrm{C_{20}H_{24}O_4}$

# 3.2 相对分子质量

藏花素：977.21（按2007年国际相对原子质量）

藏花酸：328.35（按2007年国际相对原子质量）

# 检验方法

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T 6682—2008中规定的水。分析中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T 601、GB/T 602、GB/T 603的规定制备。本试验所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 最大吸收波长

取A.3.2色价测定中的栀子黄试样液，用分光光度计检测，在波长 $440\mathrm{nm}$ 附近应有最大吸收峰。

# A.2.2 颜色反应

取 $0.5 \mathrm{~g}$ 样品, 加入 $2 \mathrm{~mL}$ 硫酸, 即由深青色慢慢变为紫色, 最后变为褐色。

# A.2.3 薄层层析

# A.2.3.1 试验方法

固定相采用微结晶纤维素薄层板，薄层板的制法：称取 $10\mathrm{g}$ 微结晶纤维素SF加水 $35~\mathrm{mL}$ 成悬浊液，均匀涂布于平滑且厚度均匀的玻璃板上，涂厚为 $0.35\mathrm{mm}$ 以下，在 $60^{\circ}\mathrm{C}\sim 80^{\circ}\mathrm{C}$ 下，烘 $20\mathrm{min}$ 。移动相为异戊醇:丙酮:水=5:6:5（体积比）。

# A.2.3.2 点样试验液的制备

称取一定量的试样，配成浓度为 $50\mathrm{g / L}$ 的水溶液作为点样试验液。

# A.2.3.3 点样

在干燥的微结晶纤维素薄层板上，用毛细管点样。点离板下端 $2\mathrm{cm}$ ，点间距离 $1\mathrm{cm}$ ，点直径约 $3\mathrm{mm}$ ，点样时用冷风吹干。

# A.2.3.4 展开与观察

将点样的薄层板放入展开槽中展开，待展开高度为 $15 \mathrm{~cm}$ 时，取出、风干。观察应有两个黄色斑点： $R_{f}$ 约为0.6是藏花素； $R_{f}$ 约为0.9是藏花酸。

# A.3 色价的测定

# A.3.1 仪器和设备

分光光度计。

# A.3.2 分析步骤

称取约 $0.15\mathrm{g}$ 粉末试样（精确至 $0.0002\mathrm{g}$ ）或称取约 $1\mathrm{g}$ 浸膏或液体试样（精确至 $0.0002\mathrm{g}$ ），用水溶解，转移至 $100\mathrm{mL}$ 容量瓶中，加水定容至刻度，摇匀。然后再吸取 $10\mathrm{mL}$ 试样液，转移至 $100\mathrm{mL}$ 容量瓶中，加水定容至刻度，摇匀。取此试样液置于 $1\mathrm{cm}$ 比色皿中，以水做空白对照，用分光光度计在（ $440\mathrm{nm} \pm 5\mathrm{nm}$ ）范围内的最大吸收波长处测定吸光度。（吸光度应控制在 $0.3 \sim 0.7$ 之间，否则应调整试样液浓度，再重新测定吸光度。）

# A.3.3 结果计算

色价按公式（A.1）计算：

$$
E _ {1 c m} ^ {1 \%} (440 \pm 5) n m = \frac {A}{c} \times \frac {1}{100} \dots \tag{A.1}
$$

式中：

$E_{1cm}^{1\%}(440nm \pm 5nm)$ ——试样液浓度为 $1\%$ ，用 $1cm$ 比色皿，在 $(440nm \pm 5nm)$ 范围内的最大吸收波长处测得的色价；

$A$ ——实际测定试样液的吸光度；

$c$ ——被测试样液的浓度，单位为克每毫升（ $\mathrm{g/mL}$ ）。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

# A.4 栀子苷的测定

# A.4.1 试剂和材料

a) 乙腈：色谱纯。  
b) 栀子苷标准品：质量分数 $\geq 99\%$

# A.4.2 仪器和设备

高效液相色谱仪：配紫外检测器（检测波长 $238\mathrm{nm}$ ）。

# A.4.3 参考色谱条件

a) 色谱柱：ODSC18， $4.6\mathrm{mm}\times 25\mathrm{cm}$ ，粒度 $5\mu \mathrm{m}$ ；或其他等效的色谱柱。  
b) 流动相: 乙腈 : 水 $= 15:85$ ; 将 $150 \mathrm{~mL}$ 色谱纯乙腈与 $850 \mathrm{~mL}$ 水混合均匀后, 用 $0.45 \mu \mathrm{m}$ 滤膜过滤, 超声脱气后备用。  
c) 柱温： $40^{\circ}\mathrm{C}$ 。  
d) 流速: $0.7 \mathrm{~mL} / \mathrm{min}$ 。  
e) 进样量: $10 \mu \mathrm{L}$ 。

# A.4.4 分析步骤

# A.4.4.1 栀子苷标准曲线的制备

称取约 $0.01 \mathrm{~g}$ 桔子苷标准品（精确至 $0.0001 \mathrm{~g}$ ），用流动相（乙腈水溶液）溶解并定容至 $50 \mathrm{~mL}$ ，得到标样贮存液 A。吸取 $0.25 \mathrm{~mL} 、 0.75 \mathrm{~mL} 、 1.25 \mathrm{~mL} 、 2.0 \mathrm{~mL} 、 2.5 \mathrm{~mL}$ 贮存液 A，分别用流动相（乙腈水溶液）稀释并定容至 $50 \mathrm{~mL}$ ，得到 5 个标样。在 A.4.3 参考色谱条件下，对梯度浓度的标样进行测定，重复实验两次，得到标样平均峰面积值。以标样峰面积为纵坐标，标样的栀子苷质量浓度 $(\mathrm{g} / \mathrm{mL})$ 为横坐标，做标准曲线。

# A.4.4.2 试样液的制备

称取适量试样（精确至 $0.0001 \mathrm{~g}$ ），用流动相（乙腈水溶液）溶解并定容至 $25 \mathrm{~mL}$ ，所得溶液用 $0.45 \mu \mathrm{m}$ 滤膜过滤，滤液备用。

# A.4.4.3 测定

在 A.4.3 参考色谱条件下, 对试样液进行测定, 根据栀子苷标准品的保留时间定性。重复进样一次, 得到栀子苷平均峰面积值。根据标样峰面积和标样的栀子苷质量浓度之间的线性关系, 得到试样液中栀子苷的质量浓度 $(\mathrm{g} / \mathrm{mL})$ 。若试样液中栀子苷浓度 $(\mathrm{g} / \mathrm{mL})$ 不在标准曲线范围内, 则应调整试样液的浓度或者重新设计标准曲线。

# A.4.5 结果计算

试样中栀子苷的含量 $X_{1}$ 按公式（A.2）计算：

$$
X _ {1} = \frac {c _ {1}}{c _ {2}} \times 100 \% \dots \tag{A.2}
$$

式中：

$X_{1}$ ——试样中栀子苷的含量， $\%$ ；

$c_{1}$ ——根据标准曲线求得的栀子苷浓度，单位为克每毫升（ $\mathrm{g/mL}$ ）；

$c_{2}$ ——试样液的浓度，单位为克每毫升（ $\mathrm{g/mL}$ ）。

最后将上述计算结果换算成以色价10计的栀子苷含量。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.1\%$ 。

# 前言

本文件按照GB/T1.1—2025《标准化工作导则第1部分：标准化文件的结构和起草规则》的规定起草。

# 食品安全国家标准禽蛋中氯霉素残留量的测定液相色谱-串联质谱法

# 1 范围

本文件规定了禽蛋中氯霉素残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鸡蛋、鸭蛋、鹅蛋、鹌鹑蛋中氯霉素残留量的测定，其他禽蛋的检测可参照执行。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中残留的氯霉素经乙晴提取，乙二胺- $N$ 丙基硅烷化硅胶（PSA）和十八烷基硅烷键合硅胶 $(\mathrm{C}_{18})$ 分散固相萃取净化，液相色谱-串联质谱仪测定，内标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯  
5.1.2 甲醇 $\mathrm{(CH_3OH)}_2$ 色谱纯  
5.1.3 甲酸（HCOOH）：色谱纯。  
5.1.4 氯化钠 $(\mathrm{NaCl})$ 。  
5.2 溶液配制  
5.2.1 $50\%$ 乙腈溶液：取乙腈 $50~\mathrm{mL}$ 加水稀释至 $100~\mathrm{mL}$   
5.3 标准品  
5.3.1 氯霉素（Chloramphenicol， $\mathrm{C_{11}H_{12}Cl_2N_2O_5}$ ，CAS号：56-75-7），纯度 $\geq 99.0\%$ ，或经国家认证并授予标准物质证书的相当者。  
5.3.2氯霉素- $\mathrm{D}_5$ (Chloramphenicol- $\mathrm{D}_5$ ， $\mathrm{C}_{11}\mathrm{H}_7\mathrm{D}_5\mathrm{Cl}_2\mathrm{N}_2\mathrm{O}_5$ ,CAS号：202480-68-0)，纯度 $\geq 98.0\%$ 。

# 5.4 标准溶液制备

5.4.1 氯霉素标准储备液：取氯霉素标准品适量（相当于活性成分约 $10\mathrm{mg}$ ），精密称定，用甲醇适量使溶解并稀释定容至 $10\mathrm{mL}$ 容量瓶中，配制成浓度为 $1\mathrm{mg/mL}$ 标准储备液， $-18^{\circ}\mathrm{C}$ 以下避光保存，有效期12个月。  
5.4.2 内标储备液：取氯霉素- $\mathrm{D}_5$ 标准品约 $1\mathrm{mg}$ ，精密称定，用甲醇适量使溶解并稀释定容至 $10~\mathrm{mL}$ 容量瓶中，配制成浓度为 $100\mu \mathrm{g / mL}$ 内标储备液， $-18^{\circ}C$ 以下避光保存，有效期12个月。

5.4.3 氯霉素标准中间液：精密量取氯霉素标准储备液 $1\mathrm{mL}$ ，于 $100\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 标准中间液， $-18^{\circ}C$ 以下避光保存，有效期6个月。  
5.4.4 内标中间液：精密量取内标储备液 $1\mathrm{mL}$ ，于 $100\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 内标中间液， $-18^{\circ}\mathrm{C}$ 以下避光保存，有效期6个月。  
5.4.5 氯霉素标准工作液：精密量取氯霉素标准中间液 $100\mu \mathrm{L}$ ，于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $100\mathrm{ng / mL}$ 的标准工作液， $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 避光保存，有效期7d。  
5.4.6 内标工作液：精密量取内标中间液 $200\mu \mathrm{L}$ ，于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释定容至刻度，配制成浓度为 $20\mathrm{ng / mL}$ 的内标工作液， $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 避光保存，有效期7d。

# 5.5 材料

5.5.1 乙二胺- $N$ -丙基硅烷化硅胶(PSA)：粒径 $40\mu \mathrm{m}$   
5.5.2 十八烷基硅烷键合硅胶 $\left(\mathrm{C}_{18}\right)$ ：粒径 $40\mu \mathrm{m}$ 。  
5.5.3 尼龙微孔滤膜： $0.22\mu \mathrm{m}$ ，或相当者。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量分别为 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
6.3 氮气吹干仪。  
6.4 涡旋混合器。  
6.5 快速振荡器。  
6.6 匀浆机。  
6.7 离心机：离心力不小于 $7100\mathrm{g}$ （或转速不低于 $8000\mathrm{r / min}$ ）。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜的空白或供试禽蛋，去壳，于匀浆机充分混匀。

a）取混匀后的供试样品，作为供试试样；  
b）取混匀后的空白样品，作为空白试样；  
c）取混匀后的空白样品，添加适宜浓度的标准工作溶液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试料 $(10 \pm 0.05) \mathrm{g}$ , 于 $50 \mathrm{~mL}$ 离心管中, 加入 $100 \mu \mathrm{L} 20 \mathrm{ng} / \mathrm{mL}$ 的内标工作溶液、 $16 \mathrm{~mL}$ 乙腈、 $5.0 \mathrm{~g}$ 氯化钠, 涡旋混匀 $1 \mathrm{~min}$ , 剧烈振荡 $10 \mathrm{~min}, 7100 \mathrm{~g}$ 离心 $10 \mathrm{~min}$ , 取乙腈层备用。

# 8.2 净化

取备用液至混有 $600\mathrm{mg}$ PSA和 $600\mathrm{mgC_{18}}$ 的 $50~\mathrm{mL}$ 离心管中，剧烈振荡 $10\mathrm{min},7100\times g$ 离心 $10\mathrm{min}$ 。取上清液于 $40^{\circ}C$ 氮气吹干，加入 $2\mathrm{mL}50\%$ 乙腈溶液，涡旋 $30~s$ ，过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱仪测定。

# 8.3 标准曲线的制备

精确量取氯霉素和氯霉素- $\mathrm{D}_5$ 标准工作液适量，用 $50\%$ 乙腈溶液稀释，配制成氯霉素浓度为 $0.5\mu \mathrm{g} / \mathrm{L}$ 、 $1.0\mu \mathrm{g} / \mathrm{L},2.5\mu \mathrm{g} / \mathrm{L},5.0\mu \mathrm{g} / \mathrm{L},10.0\mu \mathrm{g} / \mathrm{L},25.0\mu \mathrm{g} / \mathrm{L}$ ，氯霉素- $\mathrm{D}_5$ 浓度均为 $1\mu \mathrm{g} / \mathrm{L}$ 系列标准溶液，临用现配，供液相色谱-串联质谱测定。以氯霉素和氯霉素- $\mathrm{D}_5$ 定量离子对的峰面积比为纵坐标、氯霉素标准

溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在相同测试条件下，试样溶液中氯霉素与其同位素内标（氯霉素- $\mathrm{D}_5$ )的保留时间之比与标准溶液中氯霉素与其同位素内标(氯霉素- $\mathrm{D}_5$ )的保留时间之比偏差在 $1\%$ 以内；且检测到的相对离子丰度应当与浓度相当的校正标准溶液相对离子丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.4.3.2 定量测定

取试样溶液和相应的标准溶液，作单点或多点校准，按内标法定量。试样溶液及标准溶液中氯霉素与氯霉素- $\mathrm{D}_5$ 的响应值均应在仪器检测的线性范围之内。对于试料中氯霉素残留量超过仪器测定线性范围的，在提取时根据药物浓度相应增加内标工作液的添加量，使试样溶液稀释后氯霉素的响应在仪器线性范围内，对应氯霉素- $\mathrm{D}_5$ 浓度与标准曲线制备中的氯霉素- $\mathrm{D}_5$ 浓度一致。在上述色谱-质谱条件下，氯霉素和

氯霉素- $\mathrm{D}_{5}$ 标准溶液特征离子对质量色谱图见附录A。

# 8.5 空白试验

称取空白试样，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中氯霉素的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {s} \times C _ {i s} \times A _ {i} \times A ^ {\prime} {} _ {i s} \times V}{C ^ {\prime} {} _ {i s} \times A _ {s} \times A _ {i s} \times m} \tag {1}
$$

式中：

$X$ ——试样中氯霉素残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{s}$ ——标准溶液中氯霉素浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{i_5}$ ——试样溶液中氯霉素-D5浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$A_{i}$ ——试样溶液中氯霉素的峰面积；

$A_{i5}^{\prime}$ ——标准溶液中氯霉素- $\mathrm{D}_5$ 的峰面积；

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$C_{is}^{\prime}$ ——标准溶液中氯霉素- $\mathrm{D}_5$ 浓度的数值，单位为微克每升 $(\mu \mathrm{g / L})$

$A_{s}$ ——标准溶液中氯霉素的峰面积；

$A_{is}$ ——试样溶液中氯霉素- $\mathrm{D}_5$ 的峰面积；

$m$ ——供试试样质量的数值，单位为克 $(\mathrm{g})$ 。

注：计算结果以平行测定结果的算术平均值表示，含量不小于 $1\mu \mathrm{g} / \mathrm{kg}$ 的保留3位有效数字， $1\mu \mathrm{g} / \mathrm{kg}$ 以下保留至小数点后2位。

# 10 方法的灵敏度、正确度和精密度

# 10.1 灵敏度

本方法氯霉素检出限为 $0.05\mu \mathrm{g / kg}$ ；定量限为 $0.1\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法在 $0.1\mu \mathrm{g} / \mathrm{kg}\sim 1.0\mu \mathrm{g} / \mathrm{kg}$ 的添加浓度水平时，氯霉素回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

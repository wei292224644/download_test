# 前言

本标准代替GB4789.9—2014《食品安全国家标准 食品微生物学检验 空肠弯曲菌检验》。

本标准与GB4789.9—2014相比，主要变化如下：

——修改了标准的名称；  
——修改了标准的范围；  
——增加了实时荧光PCR鉴定方法和基质辅助激光解吸电离飞行时间质谱鉴定方法；  
——修改了设备和材料、培养基和试剂、检验程序、操作步骤；  
——修改了附录。

# 空肠弯曲菌和结肠弯曲菌检验

# 1 范围

本标准规定了食品中空肠弯曲菌(Campylobacter jejuni)和结肠弯曲菌(Campylobacter coli)的检验方法。

本标准适用于食品中空肠弯曲菌和结肠弯曲菌的检验。

# 2 设备和材料

除微生物实验室常规灭菌、培养及分子检测设备外，其他设备和材料如下。

2.1 冰箱： $2^{\circ}\mathrm{C}\sim 8^{\circ}\mathrm{C}, - 18^{\circ}\mathrm{C}\sim -20^{\circ}\mathrm{C}$   
2.2 恒温培养箱： $25^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}, 36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}, 42^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 。  
2.3 拍击式均质器。  
2.4 无菌带滤网均质袋。  
2.5 电子天平：感量 $0.1\mathrm{g}$ ，感量 $0.01\mathrm{g}$ ，感量 $0.001\mathrm{g}$ 。  
2.6 恒温水浴或金属浴装置： $36^{\circ} \mathrm{C} \sim 100^{\circ} \mathrm{C}$ 。  
2.7 振荡器。  
2.8 无菌培养皿：直径 $60~\mathrm{mm},90~\mathrm{mm}$   
2.9 无菌亲水性滤膜：直径 $47\mathrm{mm}$ ，孔径 $0.45\mu \mathrm{m}$ 。  
2.10 微需氧培养装置：提供微需氧条件（5%氧气、10%二氧化碳和85%氮气）。  
2.11 显微镜： $10\times \sim 1000\times$   
2.12 低温高速离心机： $8000g\sim 20000g,4^{\circ}C$   
2.13 pH计或 $\mathrm{pH}$ 比色管或精密 $\mathrm{pH}$ 试纸。  
2.14 实时荧光PCR仪。  
2.15 实时荧光PCR反应管。  
2.16 微量移液器及吸头： $0.1\mu \mathrm{L} \sim 2.5\mu \mathrm{L}, 0.5\mu \mathrm{L} \sim 10\mu \mathrm{L}, 2\mu \mathrm{L} \sim 20\mu \mathrm{L}, 20\mu \mathrm{L} \sim 200\mu \mathrm{L}, 100\mu \mathrm{L} \sim 1000\mu \mathrm{L}$ 。  
2.17 无菌离心管： $50~\mathrm{mL}, 100~\mathrm{mL}$ 或更大容量。  
2.18 螺口管： $10~\mathrm{mL}$   
2.19 无菌接种环： $1\mu \mathrm{L},10\mu \mathrm{L}$   
2.20 无菌吸管：1 mL（具 $0.01\mathrm{mL}$ 刻度）、 $10~\mathrm{mL}$ （具 $0.1\mathrm{mL}$ 刻度）。  
2.21 微生物生化鉴定系统。  
2.22 基质辅助激光解吸电离飞行时间质谱仪（MALDI-TOF MS）。

# 5 操作步骤

# 5.1 样品处理

# 5.1.1 一般样品

取 $25\mathrm{g(mL)}$ 样品[水果、蔬菜、水产品(贝类除外)取 $50\mathrm{g}$ ] 加至装有 $100\mathrm{mL}$ 无菌 $0.1\%$ 蛋白胨水的有滤网的均质袋中，用拍击式均质器均质 $1\mathrm{min} \sim 2\mathrm{min}$ (或者充分揉搓、振荡 $5\mathrm{min} \sim 10\mathrm{min}$ ) 后经滤网过滤至离心管中，滤过液以 $12000\mathrm{g}4^{\circ}\mathrm{C}$ 离心 $15\mathrm{min}$ 后弃去上清(可多次离心富集)，用 $1\mathrm{mL}$ 无菌生理盐水重悬沉淀，取 $1\mathrm{mL}$ 的悬浮液加至盛有 $9\mathrm{mL}$ 弯曲菌增菌液中增菌培养。

# 5.1.2 整禽等样品

用 $200\mathrm{mL}\sim 400\mathrm{mL}$ 无菌 $0.1\%$ 蛋白胨水充分揉搓样品内外部，振荡 $5\mathrm{min}\sim 10\mathrm{min}$ 后用无菌纱布或通过有滤网的均质袋过滤至离心管中，滤过液以 $12000g4^{\circ}C$ 离心 $15\mathrm{min}$ 后弃去上清（可多次离心富集），用 $1\mathrm{mL}$ 的无菌生理盐水重悬沉淀，取 $1\mathrm{mL}$ 的悬浮液加至盛有 $9\mathrm{mL}$ 弯曲菌增菌液中增菌培养。

# 5.1.3 贝类（已去壳）

取 $100\mathrm{g}\sim 200\mathrm{g}$ 样品放入无菌均质袋中，用拍击式均质器均质 $1\mathrm{min}\sim 2\mathrm{min}$ ，取 $25\mathrm{g}$ 内容物加至盛有 $100\mathrm{mL}$ 无菌 $0.1\%$ 蛋白胨水的有滤网均质袋中，用拍击式均质器均质 $1\mathrm{min}\sim 2\mathrm{min}$ （或者充分振荡 $5\mathrm{min}\sim 10\mathrm{min}$ ）后经滤网过滤至离心管中，滤过液以 $12000\mathrm{g}4^{\circ}\mathrm{C}$ 离心 $15\mathrm{min}$ 后弃去上清（可多次离心富集），用 $1\mathrm{mL}$ 无菌生理盐水重悬沉淀，取 $1\mathrm{mL}$ 的悬浮液加至盛有 $9\mathrm{mL}$ 弯曲菌增菌液中增菌培养。

# 5.1.4 液态蛋清或全蛋混合物

取 $25\mathrm{g(mL)}$ 样品于 $75~\mathrm{mL}$ 无菌 $0.1\%$ 蛋白胨水中，充分混匀样品，将样品移至离心管中， $20000g4^{\circ}C$ 离心 $15\mathrm{min}$ 后弃去上清（可多次离心富集），用 $1\mathrm{mL}$ 无菌生理盐水重悬沉淀，取 $1\mathrm{mL}$ 的悬浮液加至盛有 $9\mathrm{mL}$ 弯曲菌增菌液中增菌培养。

# 5.1.5 鲜乳和其他乳制品

若为液体乳制品，直接取 $50\mathrm{g}$ ；若为固体乳制品，切碎后取 $50\mathrm{g}$ 加至盛有 $100\mathrm{mL}$ 的无菌 $0.1\%$ 蛋白胨水的有滤网均质袋中，用拍击式均质器均质 $15\mathrm{s} \sim 30\mathrm{s}$ 后经滤网过滤至离心管中。将液体乳制品或滤过液以 $20000\mathrm{g}4^{\circ}\mathrm{C}$ 离心 $30\mathrm{min}$ 后弃去上清(可多次离心富集)，用 $1\mathrm{mL}$ 无菌生理盐水重悬沉淀（避免带入脂质层），取 $1\mathrm{mL}$ 的沉淀悬浮液加至 $9\mathrm{mL}$ 的弯曲菌增菌液中增菌培养。

# 5.1.6 需表面涂拭检测的样品

用无菌生理盐水浸湿的无菌拭子擦拭检测样品的表面（面积至少 $100\mathrm{cm}^2$ ），将拭子头剪落至盛有 $9\mathrm{mL}$ 的弯曲菌增菌液中增菌培养。

# 5.1.7 水样

将 $4\mathrm{L}$ 的水（对于氯处理的水，离心前每升水中加入 $5\mathrm{mL}$ 浓度为 $1\mathrm{mol} / \mathrm{L}$ 的硫代硫酸钠溶液）， $12000g$ 4℃离心 $30\mathrm{min}$ 后弃去上清(可多次离心富集)，用 $1\mathrm{mL}$ 无菌生理盐水重悬沉淀，取 $1\mathrm{mL}$ 沉淀悬浮液加至盛有 $9\mathrm{mL}$ 的弯曲菌增菌液中增菌培养。

# 5.2 增菌

松开装有增菌液的螺口管管口，微需氧条件下， $42^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h} \pm 2\mathrm{h}$ 。

# 5.3 分离

# 5.3.1 贴膜

哥伦比亚血琼脂平板和卡玛丽血琼脂平板平衡至室温后，用无菌镊子沿边缘部分夹取 $0.45\mu \mathrm{m}$ 的无菌滤膜，分别贴在两种平板表面(在平板中央部位)，使滤膜与培养基表面充分贴合。

# 5.3.2 加样

取 $300\mu \mathrm{L}$ 的增菌液，分成4次～6次(滴成4滴～6滴，液滴尽量不要融合)滴加于滤膜表面（液滴位置要在滤膜边缘内)，静置等待滤膜上的液滴充分透过滤膜(静置 $45\mathrm{min}\sim 60\mathrm{min}$ ，微需氧或普通气体环境)，无菌操作，揭掉滤膜。将平板翻转，微需氧条件下， $42^{\circ}C\pm 1^{\circ}C$ 培养 $24\mathrm{h}\sim 48\mathrm{h}$

# 5.3.3 观察

培养 $24\mathrm{h}\sim 48\mathrm{h}$ 后，观察两种平板上的菌落形态。卡玛丽血琼脂平板上的可疑菌落通常为淡灰

色、不溶血、表面湿润有金属光泽; 哥伦比亚血琼脂平板上的可疑菌落通常为灰白色、不溶血、表面湿润有光泽。两种血琼脂平板上的可疑菌落都可呈现分散凸起的菌落, 边缘整齐、发亮。若 $48 \mathrm{~h}$ 培养后细菌生长缓慢或菌落很小, 可延长培养时间至 $72 \mathrm{~h}$ , 若培养 $48 \mathrm{~h}$ 后无疑似菌落生长则判定结果为阴性。

# 5.4 鉴定

# 5.4.1 弯曲菌属的鉴定

# 5.4.1.1 纯培养

挑取至少5个可疑菌落(少于5个全部挑取)，分别接种至哥伦比亚血琼脂平板上进行单菌落纯培养，微需氧条件， $42^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h} \sim 48\mathrm{h}$ 。按照 $5.4.1.2 \sim 5.4.1.4$ 进行鉴定，结果符合表1的可疑菌落鉴定为弯曲菌属。

# 5.4.1.2 形态观察

挑取可疑菌落进行革兰氏染色，镜检。

# 5.4.1.3 氧化酶试验

挑取可疑菌落(不可使用含铁接种环)至氧化酶试剂润湿的滤纸上，如果在 $10\mathrm{~s}$ 内出现紫红色、紫罗兰或深蓝色判定结果为阳性。

# 5.4.1.4 有氧条件下 $25^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 生长试验

挑取可疑菌落，接种至哥伦比亚血琼脂平板上，有氧条件下 $25^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $48\mathrm{h}$ ，观察细菌生长情况。

# 5.4.2 空肠弯曲菌和结肠弯曲菌的鉴定

对于5.4.1中弯曲菌属的可疑菌落，可通过生化试验或者实时荧光PCR或者基质辅助激光解吸电离飞行时间质谱的方法进行空肠弯曲菌和结肠弯曲菌的鉴定。

# 5.4.2.1 生化试验

# 5.4.2.1.1 过氧化氢酶试验

挑取单菌落纯培养物，加至干净玻片上的 $3\%$ 过氧化氢溶液中，如果在 $30~\mathrm{s}$ 内出现气泡则判定结果为阳性。

# 5.4.2.1.2 马尿酸钠水解试验

挑取单菌落纯培养物，加至盛有 $0.4 \mathrm{~mL} 1 \%$ 马尿酸钠的试管中制成菌悬液。混合均匀后 $36^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 水浴或金属浴 $2 \mathrm{~h}$ 或 $36^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 恒温培养箱温育 $4 \mathrm{~h}$ （期间可取出混匀）。沿着试管壁缓缓加入 $0.2 \mathrm{~mL}$

茚三酮溶液(期间不可摇晃或者振荡试管),在 $36^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 的水浴或金属浴中再温育 $10 \mathrm{~min}$ 后判读结果。若出现深紫色则判定结果为阳性;若出现淡紫色或无颜色变化则判定结果为阴性。

# 5.4.2.1.3 呃哚乙酸酯水解试验

挑取单菌落纯培养物，加至吲哚乙酸酯纸片上，再滴加一滴无菌水。如果吲哚乙酸酯水解，则在 $5\mathrm{min}\sim 10\mathrm{min}$ 内出现深蓝色，判定结果为阳性；若无颜色变化则表示未发生水解，判定结果为阴性。鉴定结果见表2。

# 5.4.2.2 实时荧光PCR

# 5.4.2.2.1 试验环境与过程控制

PCR试验环境条件和过程控制应按GB/T27403规定的要求执行。

# 5.4.2.2.2 DNA 模板制备

挑取5.4.1中弯曲菌属可疑菌落的纯培养物加入盛有 $500\mu \mathrm{L}$ 去离子水的离心管中，水浴或者金属浴 $100^{\circ}C$ 保持 $10\mathrm{min}$ ，随后取出于室温冷却 $2\mathrm{min},8000g\sim 13000g$ 离心 $10\mathrm{min}$ ，吸取上清液至新的无菌离心管内，作为DNA模板使用。提取后的DNA模板应置于 $4^{\circ}C$ 供当天使用，否则应于一 $20^{\circ}C$ 以下保存，并于1周内使用。

# 5.4.2.2.5 实时荧光PCR反应程序

预变性： $94^{\circ}\mathrm{C}, 30\mathrm{~s}$ ；变性： $94^{\circ}\mathrm{C}, 5\mathrm{~s}$ ，退火延伸： $60^{\circ}\mathrm{C}, 30\mathrm{~s}, 40$ 个循环。分别以荧光基团FAM（或者VIC/HEX/CY5)标记空肠弯曲菌的探针CJP、以荧光基团VIC(或者FAM/CY5/HEX)标记结肠弯曲菌的探针CCP，在 $60^{\circ}\mathrm{C}$ 延伸阶段采集荧光信号。

# 5.4.2.2.6 实时荧光PCR对照设置

PCR反应使用空肠弯曲菌参比菌株和结肠弯曲菌参比菌株的DNA模板作为阳性对照，使用海鸥弯曲菌参比菌株和乌普萨拉弯曲菌参比菌株的DNA模板作为阴性对照，以灭菌去离子水作为空白对照。

# 5.4.2.2.7 实时荧光PCR结果判定

# 5.4.2.2.7.1 结果有效性原则

同一次实验中需同时满足下列要求，则实时荧光PCR实验结果有效；否则实时荧光PCR实验结果无效，需重新进行实验：

a）空白对照：无荧光对数增长，相应的 $\mathrm{Ct}$ 值 $>40.0$ （或者无 $\mathrm{Ct}$ 值）；  
b）阴性对照：无荧光对数增长，相应的 $\mathrm{Ct}$ 值 $>40.0$ （或者无 $\mathrm{Ct}$ 值）；  
c）阳性对照：空肠弯曲菌的 $cj0415$ 基因和结肠弯曲菌的 $zupT$ 基因均有荧光对数增长，且荧光通道出现典型的扩增曲线，相应的 $\mathrm{Ct}$ 值 $< 30.0$ 。

# 5.4.2.2.7.2 实时荧光PCR结果判定

在符合实验结果有效性的原则下，若：

a）检测基因的 $\mathrm{Ct}$ 值 $< 30.0$ ，则判定为该基因阳性；  
b）检测基因的 $\mathrm{Ct}$ 值 $\geq 40.0$ ，则判定为该基因阴性；  
c）检测基因的 $\mathrm{Ct}$ 值 $\geq 30.0$ 且 $< 40.0$ ，应重新进行实时荧光PCR扩增实验。再次扩增后 $\mathrm{Ct}$ 值 $< 40.0$ ，有典型的扩增曲线，且对照实验结果均正常，则可判定为该基因阳性。

# 1 范围

本文件规定了水产品中喹诺酮类药物残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鱼类肌肉和皮等可食组织，虾、蟹、贝、蚌、鳖、海参可食组织中噁唑酸、氟甲喹、诺氟沙星、依诺沙星、环丙沙星、培氟沙星、洛美沙星、达氟沙星、恩诺沙星、氧氟沙星、沙拉沙星、司帕沙星、二氟沙星、西诺沙星、萘啶酸、奥比沙星、马波沙星、氟罗沙星、吡哌酸残留量的检测。其他水产品的检测可参照执行。

# 4 原理

试料中残留的喹诺酮类药物，用酸化乙腈提取，正已烷脱脂，亲水亲脂固相萃取柱净化，液相色谱-串联质谱法测定，基质校准内标法定量。

5.2.3 $0.05\mathrm{mol} / \mathrm{L}$ 磷酸盐缓冲溶液(pH7.0)：取二水合磷酸二氢钠1.56g，加入氢氧化钠溶液 $79~\mathrm{mL}$ 混匀，用水稀释至 $200~\mathrm{mL}$ 。

5.2.6 0.1%甲酸水溶液(含 $0.002\mathrm{mol} / \mathrm{L}$ 乙酸铵); 取乙酸铵 $1.54\mathrm{g}$ , 用水溶解并稀释至 $1000\mathrm{mL}$ 。从中取 $100\mathrm{mL}$ 乙酸铵溶液, 取甲酸 $1\mathrm{mL}$ , 再用水稀释至 $1000\mathrm{mL}$ 。  
5.2.7 复溶液：取乙腈 $15\mathrm{mL}$ ，取 $0.1\%$ 甲酸水溶液（含 $0.002\mathrm{mol} / \mathrm{L}$ 乙酸铵） $85\mathrm{mL}$ ，混合均匀。

5.4.1 标准储备液：取噁喹酸、氟甲喹、诺氟沙星、依诺沙星、环丙沙星、培氟沙星、洛美沙星、达氟沙星、恩诺沙星、氧氟沙星、沙拉沙星、司帕沙星、二氟沙星、西诺沙星、萘啶酸、奥比沙星、马波沙星、氟罗沙星、吡哌酸标准品各适量(相当于各活性成分 $10\mathrm{mg}$ )，精密称定，在加入 $200~\mu \mathrm{L}$ 甲酸条件下，用乙腈溶解稀释定容至 $100~\mathrm{mL}$ 棕色容量瓶，配制成浓度为 $100~\mu \mathrm{g / mL}$ 标准储备液。 $-18^{\circ}C$ 以下避光保存，有效期6个月。  
5.4.2 混合标准中间液：分别精密量取标准储备液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 棕色量瓶中，用乙腈稀释至刻度，配制成浓度为 $10\mu \mathrm{g} / \mathrm{mL}$ 标准中间液。 $-18^{\circ}\mathrm{C}$ 以下避光保存，有效期3个月。  
5.4.3 混合标准工作液：精密量取标准中间液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 棕色量瓶中，用乙腈稀释至刻度，配制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 混合标准工作液。 $4^{\circ}\mathrm{C}$ 以下避光保存，有效期1个月。  
5.4.4 内标标准储备液：取诺氟沙星- $\mathrm{D}_5$ 、环丙沙星- $\mathrm{D}_8$ 、恩诺沙星- $\mathrm{D}_5$ 各 $10\mathrm{mg}$ ，精密称定，在加入 $200\mu \mathrm{L}$ 甲酸条件下，用乙腈溶解稀释定容至 $100\mathrm{mL}$ 棕色容量瓶，配制成浓度为 $100\mu \mathrm{g} / \mathrm{mL}$ 内标标准储备液。一 $18^{\circ}\mathrm{C}$ 以下避光保存，有效期6个月。  
5.4.5 混合内标标准中间液：分别精密量取内标标准储备液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 棕色量瓶中，用乙腈稀释至刻度，配制成浓度为 $10\mu \mathrm{g} / \mathrm{mL}$ 内标标准中间液。 $-18^{\circ}\mathrm{C}$ 以下避光保存，有效期3个月。  
5.4.6 混合内标标准工作液：精密量取内标标准中间液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 棕色量瓶中，用乙腈稀释至刻度，配制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 混合内标标准工作液。 $4^{\circ}\mathrm{C}$ 以下避光保存，有效期1个月。

# 8 测定步骤

# 8.1 提取

称取试料 $(5\pm 0.05)\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管，加入内标标准工作溶液 $25~\mu \mathrm{L}$ 涡旋混合 $30~\mathrm{s}$ ，避光放置 $10\mathrm{min}$ ；加入无水硫酸钠 $5\mathrm{g}$ ，涡旋混匀，再加入酸化乙腈 $20~\mathrm{mL}$ ，于涡旋混合器上以 $2000\mathrm{r / min}$ 涡旋 $1\mathrm{min}$ ，超声 $8\mathrm{min}$ ，以 $3500\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液，于 $100~\mathrm{mL}$ 离心管内。残渣再加酸化乙腈 $20~\mathrm{mL}$ ，重复提取1次，合并2次上清液于 $100~\mathrm{mL}$ 离心管，备用。

# 8.2 净化

# 8.2.1 鳞、大黄鱼类等油脂含量高、虾蟹等色素含量高的水产品

$100~\mathrm{mL}$ 离心管中加入乙腈饱和正己烷 $30~\mathrm{mL}$ , 剧烈振摇 $2\mathrm{min}$ , 静置, 取下层清液于梨形瓶中, $40^{\circ}C$ 下旋蒸至干。加入磷酸盐缓冲液 $4\mathrm{mL}$ 溶解残余物, 振荡 $1\mathrm{min}$ , 残余物溶解液转移至 $15\mathrm{mL}$ 离心管。再重复提取残余物 2 次, 溶解液转移至同一离心管。合并 3 次残余物溶解液, $3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ , 上清液备用。

固相萃取柱依次用甲醇 $3\mathrm{mL}$ 、水 $3\mathrm{mL}$ 、磷酸盐缓冲液 $3\mathrm{mL}$ 活化，取备用液过柱，流干，流速控制为每秒1滴；水 $3\mathrm{mL}$ 淋洗，抽干，依次用氨化甲醇 $6\mathrm{mL}$ 、甲醇 $4\mathrm{mL}$ 洗脱，收集洗脱液于 $15\mathrm{mL}$ 离心管中， $50^{\circ}\mathrm{C}$ 下氮吹至干。复溶液 $1.0\mathrm{mL}$ 溶解残余物，过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱测定。

# 8.2.2 其他类水产品

$100~\mathrm{mL}$ 离心管中加入乙腈饱和正己烷 $30~\mathrm{mL}$ ，剧烈振摇 $2\mathrm{min}$ ，静置，取下层清液于梨形瓶中； $40^{\circ}\mathrm{C}$ 下旋蒸至干，复溶液 $1.0~\mathrm{mL}$ 溶解残余物，加乙腈饱和正己烷 $1\mathrm{mL}$ ，转至 $5\mathrm{mL}$ 离心管中，涡旋混合 $30~\mathrm{s}$ ， $13000~\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，取下层清液过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱测定。

# 8.3 基质匹配标准曲线的制备

分别取适量混合标准工作液于 $5\mathrm{mL}$ 玻璃离心管中，加入内标标准工作溶液 $25~\mu \mathrm{L},50^{\circ}C$ 下氮吹至干，分别加入经8.1和8.2步骤处理所得的未加入内标的基质空白液 $1.0\mathrm{mL}$ ，旋涡混匀 $1\mathrm{min}$ ，过 $0.22\mu \mathrm{m}$ 滤膜，配制成喹诺酮类药物浓度为 $2.5\mathrm{ng / mL},5.0\mathrm{ng / mL},10.0\mathrm{ng / mL},25.0\mathrm{ng / mL},50.0\mathrm{ng / mL},100.0\mathrm{ng / mL}$ 和 $200.0\mathrm{ng / mL}$ 的系列基质匹配标准溶液；现用现配。以目标物和内标物的特征离子质量色谱峰面积比值为纵坐标、基质匹配标准溶液质量浓度为横坐标，绘制基质匹配标准曲线。求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在同样测试条件下，试料溶液中喹诺酮类药物和其内标物的保留时间与基质匹配标准工作液中喹诺酮类药物和其内标物的保留时间之比偏差在 $1\%$ 以内；且检测到的相对离子丰度应与浓度相当的基质匹配标准工作液相对离子丰度一致，其允许偏差为 $-40\%$ 。

# 8.4.3.2 定量测定

取试料溶液和基质匹配标准工作液, 作单点或多点校准, 以色谱峰面积定量, 按内标法计算, 其中诺氟沙星、依诺沙星、培氟沙星、氧氟沙星、马波沙星、氟罗沙星、吡哌酸以诺氟沙星- $\mathrm{D}_{5}$ 为内标; 环丙沙星、洛美沙星、达氟沙星、沙拉沙星、二氟沙星、奥比沙星以环丙沙星- $\mathrm{D}_{5}$ 为内标; 西诺沙星、司帕沙星、噁喹酸、氟甲喹、恩诺沙星、萘啶酸以恩诺沙星- $\mathrm{D}_{5}$ 为内标。基质匹配标准工作液及试料溶液中喹诺酮类药物的响应值均应在仪器检测的线性范围之内。对于试料中喹诺酮残留量超过仪器测定线性范围的, 在提取时根据药物浓度相应增加混合内标标准工作液的添加量, 使试料溶液稀释后喹诺酮的响应在仪器检测的线性范围内, 对应内标物浓度与基质匹配标准曲线制备中的内标物浓度一致。在上述色谱-质谱条件下, 喹诺酮类药物标准溶液的特征离子质量色谱图见附录B。

# 8.5 空白试验

取空白试样，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测药物的残留量按标准曲线法或公式(1)计算。

$$
X = \frac {A _ {i} \times A _ {i s} ^ {\prime} \times C _ {s} \times C _ {i s} \times V}{A _ {i} \times A _ {s} \times C _ {i s} ^ {\prime} \times m}. \tag {1}
$$

式中：

$X$ ——试样中被测组分残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$A_{i}$ ——试料溶液中被测组分的峰面积；

$A_{is}^{\prime}$ ——基质匹配标准溶液中被测组分对应内标的峰面积；

$C_{s}$ ——基质匹配标准溶液中被测组分浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$C_{is}$ ——试料溶液中内标浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$V$ ——试料溶液定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{is}$ ——试料溶液中被测组分对应内标的峰面积；

$A_{s}$ ——基质匹配标准溶液中被测组分的峰面积；

$C_{is}^{\prime}$ ——基质匹配标准溶液中内标浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$ 。

注：计算结果以平行测定结果的算术平均值表示，含量不小于 $1\mu \mathrm{g} / \mathrm{kg}$ 的保留3位有效数字， $1\mu \mathrm{g} / \mathrm{kg}$ 以下保留至小数点后2位。

# 10 方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法的检出限为 $0.5\mu \mathrm{g / kg}$ ；定量限为 $1.0~\mu \mathrm{g / kg}$ 。

# 10.2 正确度

标准添加浓度 $1.0\mu \mathrm{g / kg}\sim 10\mu \mathrm{g / kg}$ ，回收率为 $60\% \sim 120\%$ ；标准添加浓度 $10~\mu \mathrm{g / kg}\sim 100~\mu \mathrm{g / kg}$ 回收率 $70\% \sim 120\%$ ；标准添加浓度 $100~\mu \mathrm{g / kg}\sim 500~\mu \mathrm{g / kg}$ ，回收率为 $80\% \sim 110\%$ 。

# 10.3 精密度

# 10.3.1 重复性精密度

标准添加浓度 $1.0\mu \mathrm{g / kg}\sim 10\mu \mathrm{g / kg}$ ，相对标准偏差 $\leqslant 20\%$ ；标准添加浓度 $10~\mu \mathrm{g / kg}\sim 100~\mu \mathrm{g / kg}$ ，相对标准偏差 $\leqslant 15\%$ ；标准添加浓度 $100\mu \mathrm{g / kg}\sim 500\mu \mathrm{g / kg}$ ，相对标准偏差 $\leqslant 10\%$ 。

# 10.3.2 再现性精密度

标准添加浓度 $1.0\mu \mathrm{g / kg}\sim 10\mu \mathrm{g / kg}$ ，相对标准偏差 $\leqslant 30\%$ ；标准添加浓度 $10~\mu \mathrm{g / kg}\sim 100~\mu \mathrm{g / kg}$ ，相对标准偏差 $\leqslant 20\%$ ；标准添加浓度 $100\mu \mathrm{g / kg}\sim 500\mu \mathrm{g / kg}$ ，相对标准偏差 $\leqslant 15\%$ 。

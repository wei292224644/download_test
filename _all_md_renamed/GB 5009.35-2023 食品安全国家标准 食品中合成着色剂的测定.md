# 前言

本标准代替GB5009.35—2016《食品安全国家标准 食品中合成着色剂的测定》、GB5009.141—2016《食品安全国家标准 食品中诱惑红的测定》、GB/T9695.6—2008《肉制品 胄脂红着色剂测定》、GB/T21916—2008《水果罐头中合成着色剂的测定 高效液相色谱法》。

本标准与GB5009.35—2016相比，主要变化如下：

——增加了靛蓝、诱惑红、酸性红和喹啉黄4种合成着色剂；  
——增加和明确了适用基质种类；  
——修改了样品前处理方法和仪器条件；  
——修改了检出限和定量限。

# 食品安全国家标准

# 食品中合成着色剂的测定

# 1 范围

本标准规定了食品中合成着色剂的液相色谱测定方法。

本标准适用于食品中11种合成着色剂（柠檬黄、新红、苋菜红、靛蓝、胭脂红、日落黄、诱惑红、亮蓝、酸性红、喹啉黄和赤藓红)的测定。

# 2 原理

试样中的合成着色剂用乙醇氨水溶液提取，经固相萃取净化后，用配有二极管阵列检测器的高效液相色谱仪测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.2 甲醇 $(\mathrm{CH_4O})$ 。  
3.1.3 石油醚：沸程 $30^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$   
3.1.4 无水乙醇 $\mathrm{(C_2H_6O)}$ 。  
3.1.5 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ：含量 $20\% \sim 25\%$   
3.1.6 乙酸铵 $\mathrm{(C_2H_7O_2N)}$ ：色谱纯。  
3.1.7 甲酸 $\mathrm{(CH_2O_2)}$ ：含量 $98\%$

# 3.2 试剂配制

3.2.1 乙醇氨水溶液：量取无水乙醇 $700~\mathrm{mL}$ ，加入 $4\mathrm{mL}$ 氨水，用水稀释至 $1\mathrm{L}$ ，混匀。  
3.2.2 $5\%$ 甲醇水溶液：移取甲醇(3.1.2)5 mL，用水稀释并定容至 $100~\mathrm{mL}$ ，混匀。  
3.2.3 $2\%$ 氨水甲醇溶液：移取 $2\mathrm{mL}$ 氨水，用甲醇(3.1.2)稀释至 $100~\mathrm{mL}$   
3.2.4 乙酸铵溶液 $(20\mathrm{mmol / L})$ ：称取 $1.54\mathrm{g}$ 乙酸铵，加水溶解并稀释至 $1000~\mathrm{mL}$ 。  
3.2.5 乙酸铵缓冲溶液， $\mathrm{pH} = 9.0$ ：乙酸铵溶液(3.2.4)加氨水调 $\mathrm{pH}$ 至 $9.0$ 。  
3.2.6 $2\%$ 甲酸水溶液：移取甲酸 $2\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ 。

# 3.3 标准品

3.3.1 柠檬黄(CAS号：1934-21-0)：纯度 $\geqslant 95.0\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.2 新红(CAS号：220658-76-4)：纯度 $\geqslant 95.0\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.3 苎莱红(CAS号：915-67-3)：纯度 $\geq 95.0\%$ ，或经国家认证并授予标准物质证书的标准品。

3.3.4 靛蓝(CAS号：860-22-0)：纯度 $\geq 90.0\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.5 胀脂红(CAS号：2611-82-7)：纯度 $\geqslant 95.0\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.6 日落黄(CAS号：2783-94-0)：纯度 $\geqslant 90.0\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.7 诱惑红(CAS号：25956-17-6)：纯度 $\geqslant 95.0\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.8 亮蓝(CAS号：3844-45-9)：纯度 $\geqslant 95.0\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.9 酸性红(CAS号：3567-69-9)：纯度 $\geqslant 90.0\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.10 喹啉黄(CAS号：8004-92-0)：纯度 $\geq 95.0\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.11 赤藓红(CAS号：16423-68-0)：纯度 $\geqslant 90.0\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 标准储备液 $(1.0\mathrm{mg / mL})$ ：准确称取按其纯度折算为 $100\%$ 质量的柠檬黄、新红、苋菜红、胭脂红、日落黄、诱惑红、亮蓝、酸性红、喹啉黄和赤藓红各 $100\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )，加水溶解并分别置于 $100~\mathrm{mL}$ 容量瓶中，定容至刻度，摇匀，得到浓度为 $1.0\mathrm{mg / mL}$ 的标准储备液。标准储备液可于 $4^{\circ}C$ 下避光保存6个月，靛蓝标准溶液临用现配。  
3.4.2 混合标准中间液 $(50.0\mu \mathrm{g / mL})$ ：吸取上述标准储备液(3.4.1)和靛蓝标准溶液 $(1.0\mathrm{mg / mL})$ 各 $5.00~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀，得到混合标准中间液（各合成着色剂浓度均为 $50.0\mu \mathrm{g / mL})$ ，临用现配。  
3.4.3 标准系列工作液：吸取混合标准中间液(3.4.2) $0.2\mathrm{mL}, 0.5\mathrm{mL}, 1.0\mathrm{mL}, 2.0\mathrm{mL}, 5.0\mathrm{mL}$ 和 $10.0\mathrm{mL}$ 于 $50\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀，得到标准系列工作液。浓度分别为 $0.2\mu \mathrm{g} / \mathrm{mL}$ 、 $0.5\mu \mathrm{g} / \mathrm{mL}, 1.0\mu \mathrm{g} / \mathrm{mL}, 2.0\mu \mathrm{g} / \mathrm{mL}, 5.0\mu \mathrm{g} / \mathrm{mL}$ 和 $10.0\mu \mathrm{g} / \mathrm{mL}$ 。

# 4 仪器和设备

4.1 高效液相色谱仪：带二极管阵列检测器。  
4.2 天平：感量分别为 $1\mathrm{mg}$ 和 $0.1\mathrm{mg}$   
4.3 pH计：精度为0.01。  
4.4 电动搅拌器：转速范围为 $30\mathrm{r / min}\sim 2000\mathrm{r / min}$   
4.5 涡旋混合器。  
4.6 超声波发生器或恒温摇床：超声功率不小于 $700\mathrm{W}$ ，控温范围为 $20^{\circ}\mathrm{C} \sim 80^{\circ}\mathrm{C}$ ；摇床转速范围为 $10\mathrm{r / min} \sim 500\mathrm{r / min}$ 。  
4.7 高速离心机：转速不小于 $15000\mathrm{r / min}$   
4.8 固相萃取装置。  
4.9 氮气浓缩装置。  
4.10 WAX混合型弱阴离子交换反相吸附或等效固相萃取柱， $150\mathrm{mg / 6mL}$   
4.11 针筒过滤器，PVDF(聚偏氟乙烯)或PTFE(聚四氟乙烯)滤膜，孔径为 $0.45\mu \mathrm{m}$ 。

# 5 分析步骤

# 5.1 样品前处理

# 5.1.1 试样制备

液体试样和粉状固体试样应分别混合均匀，半固体试样取固液共存物进行匀浆混和，固体试样（带核蜜饯凉果需先去核，取可食部分)经电动搅拌器粉碎等方式混合均匀，密封，制备好的试样在 $-18^{\circ}C$

以下避光保存，备用。

# 5.1.2 试样提取

# 5.1.2.1 液体类试样（饮料、配制酒、调制乳、调味糖浆、风味发酵乳等）、冷冻饮品（风味冰、冰棍类）

准确称取试样 $2\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )，冷冻饮品可先温水浴加热融化再称样，置于 $50~\mathrm{mL}$ 具塞离心管中，加入适量乙醇氨水溶液(3.2.1)，涡旋 $1\mathrm{min},5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，并用乙醇氨水溶液(3.2.1)定容至 $50~\mathrm{mL}$ ，即得提取液；准确吸取上清液 $10~\mathrm{mL},50^{\circ}C$ 下氮气浓缩至 $3\mathrm{mL}$ 左右，分2次～3次共加入 $10~\mathrm{mL}5\%$ 甲醇水溶液溶解，作为待净化液。

# 5.1.2.2 固体类试样（加工水果、腌渍的蔬菜、糖果、酱及酱制品、香辛料、果冻、杂粮粉及其制品、面糊、淀粉及淀粉类制品、胶原蛋白肠衣、即食谷物、谷类和淀粉类甜品、糕点上彩装、蛋卷、焙烤食品馅料及表面用挂浆等）

准确称取试样 $2\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ），置于 $50~\mathrm{mL}$ 具塞离心管中，先加入适量水（ $2\mathrm{mL}\sim 5\mathrm{mL}$ ） $50^{\circ}C$ 水浴加热混匀样品，加入 $25~\mathrm{mL}$ 乙醇氨水溶液(3.2.1)，涡旋 $1\mathrm{min}$ ， $50^{\circ}C$ 超声或振摇（速率 $\geqslant$ $250~\mathrm{r / min})$ 提取 $20\mathrm{min},8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液置于 $50~\mathrm{mL}$ 容量瓶中，每次加入约 $5\mathrm{mL}\sim$ $10~\mathrm{mL}$ 乙醇氨水溶液(3.2.1)重复提取操作至上清液无明显颜色，离心后合并上清液，用乙醇氨水溶液(3.2.1)定容至 $50~\mathrm{mL}$ ，即得提取液。准确吸取提取液 $10~\mathrm{mL},50^{\circ}C$ 氮气浓缩至 $3\mathrm{mL}$ 左右，分2次～3次共加入 $10~\mathrm{mL}5\%$ 甲醇水溶液溶解，作为待净化液。

# 5.1.2.3 含油量较大的试样（可可制品、巧克力和巧克力制品、调制乳粉、调制奶油粉、调制炼乳、膨化食品、加工坚果与籽类、熟制豆类、糕点、熟肉制品、复合调味料和冰淇淋、雪糕等）

准确称取试样 $2\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 具塞离心管中，加入 $20~\mathrm{mL}$ 石油醚，涡旋 $1\mathrm{min}$ ，超声或振摇(速率 $\geq 250~\mathrm{r / min}$ )提取 $10\mathrm{min},8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去上清液，油脂含量较高的可重复提取一次，弃去上清液，加入 $25~\mathrm{mL}$ 乙醇氨水溶液（3.2.1），涡旋 $1\mathrm{min},50^{\circ}\mathrm{C}$ 超声或振摇(速率 $\geq$ $250~\mathrm{r / min}$ )提取 $20\mathrm{min},8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ (若离心后提取液仍然浑浊，可转入高速离心机专用管， $15000\mathrm{r / min}$ 离心 $5\mathrm{min}$ )，取上清液置于 $50~\mathrm{mL}$ 容量瓶中，按照5.1.2.2中“每次加入约 $5\mathrm{mL}\sim 10\mathrm{mL}$ 乙醇氨水溶液……作为待净化液”操作。

# 5.1.3 试样净化

# 5.1.3.1 活化

依次用 $6\mathrm{mL}$ 甲醇(3.1.2)和 $6\mathrm{mL}$ 水活化固相萃取柱(4.10)，保持柱体湿润。

# 5.1.3.2 上样

活化后立即将5.1.2中的待净化液以 $2\mathrm{s}\sim 3\mathrm{s}1$ 滴的流速加载到固相萃取柱上。

# 5.1.3.3 淋洗

依次用 $6\mathrm{mL}2\%$ 甲酸水溶液和 $6\mathrm{mL}$ 甲醇(3.1.2)淋洗固相萃取柱，弃去淋洗液，真空抽 $2\mathrm{min}$ 至柱体近干。

# 5.1.3.4 洗脱

用 $6\mathrm{mL}2\%$ 氨化甲醇溶液洗脱，分两次加入，每次 $3\mathrm{mL}$ ，流速低于 $2\mathrm{s}\sim 3\mathrm{s}1$ 滴，收集洗脱液，于

$50~\mathrm{^\circ C}$ 氮气浓缩至近干，准确加入 $2\mathrm{mL}$ $\mathrm{pH}$ 为9.0的乙酸铵缓冲溶液(3.2.5)溶解，溶液用针筒过滤器，孔径 $0.45\mu \mathrm{m}$ 的滤膜过滤，弃去2滴～5滴初滤液，取续滤液作为待测液。

# 5.2 仪器参考条件

5.2.1 色谱柱： $\mathrm{C_{18}}$ 柱， $4.6 \mathrm{~mm} \times 250 \mathrm{~mm}, 5 \mu \mathrm{m}$ ，或同等性能色谱柱。  
5.2.2 进样量： $10~\mu \mathrm{L}$   
5.2.3 柱温： $30^{\circ}C$   
5.2.4 二极管阵列检测器波长范围： $400\mathrm{nm}\sim 800\mathrm{nm}$ ，检测波长： $415~\mathrm{nm}$ （柠檬黄、喹啉黄）， $520~\mathrm{nm}$ （新红、苋菜红、胭脂红、日落黄、诱惑红、酸性红和赤藓红）， $610~\mathrm{nm}$ （靛蓝、亮蓝）。

# 5.3 标准曲线的制作

将标准系列工作液分别注入液相色谱仪中，测定相应物质的峰面积，以标准系列工作液中该物质的浓度为横坐标，以该物质峰面积的响应值为纵坐标，绘制标准曲线。11种标准物质溶液的高效液相色谱图参见附录A。

注：亮蓝通常呈现为两个峰，喹啉黄通常呈现为四个峰，应根据标准溶液出峰情况，以各色谱峰的峰面积之和计算含量。

# 5.4 试样溶液的测定

将试样溶液注入液相色谱仪中，得到对应的峰面积，根据标准曲线计算得到待测液中的各物质浓度。

# 6 分析结果的表述

试样中合成着色剂的含量按式(1)计算。

$$
X = \frac {c \times V _ {1} \times V _ {2}}{V _ {3} \times m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中合成着色剂的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$c$ ——由标准曲线计算得到的待测液中合成着色剂浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V_{1}$ ——样品经净化洗脱后的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——样品提取液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——用于净化分取的样品提取液体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

计算结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法中，当样品取样量为 $2\mathrm{g}$ ，定容体积为 $2\mathrm{mL}$ 时，柠檬黄、新红、胭脂红、日落黄、喹啉黄、赤藓红的检出限均为 $0.5\mathrm{mg/kg}$ ，定量限均为 $1.5\mathrm{mg/kg}$ ，苋菜红、诱惑红、亮蓝、酸性红、靛蓝的检出限均为 $0.3\mathrm{mg/kg}$ ，定量限均为 $1.0\mathrm{mg/kg}$ 。

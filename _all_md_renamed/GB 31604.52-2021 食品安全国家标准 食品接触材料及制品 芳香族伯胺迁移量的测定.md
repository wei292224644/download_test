# 食品安全国家标准

# 食品接触材料及制品

# 芳香族伯胺迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中29种芳香族伯胺迁移量的测定方法。

本标准适用于食品接触材料及制品中芳香族伯胺迁移量的测定。

# 2 原理

食品接触材料及制品根据GB31604.1及GB5009.156进行迁移试验后，采用液相色谱-质谱/质谱法进行检测。其中 $4\%$ 乙酸、 $10\%$ 乙醇、 $20\%$ 乙醇、 $50\%$ 乙醇食品模拟物及化学替代溶剂 $95\%$ 乙醇经过滤后直接进样；化学替代溶剂异辛烷通过甲醇-水溶液萃取后取水层过滤进样。峰面积外标法定量。

# 3 试剂和材料

水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 酸性、酒精类食品模拟物及化学替代溶剂：所用试剂依据GB5009.156的规定。  
3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.3 甲酸（HCOOH）：色谱纯。  
3.1.4 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.5 冰醋酸 $\mathrm{(CH_3COOH)}$ ：优级纯。  
3.1.6 异辛烷 $\left(\mathrm{C}_{8} \mathrm{H}_{18}\right)$ ：色谱纯。

# 3.2 试剂配制

3.2.1 酸性、酒精类食品模拟物及化学替代溶剂：按GB5009.156配制。  
3.2.2 $0.1\%$ 甲酸：取 $1\mathrm{mL}$ 甲酸(3.1.3)于容量瓶中，加水定容至 $1\mathrm{L}$ 。  
3.2.3 甲醇-水提取液：量取 $50~\mathrm{mL}$ 甲醇(3.1.2)和 $50~\mathrm{mL}$ 水，吸取 $0.4~\mathrm{mL}$ 冰醋酸(3.1.5)，混匀。

# 3.3 标准品

29种芳香族伯胺：标准品（见附录A)，纯度 $\geq 98\%$ ；或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 29种芳香族伯胺标准储备溶液 $(1000\mathrm{mg / L})$ ：准确称取29种芳香族伯胺标准品各 $25~\mathrm{mg}$ （精确到 $0.1\mathrm{mg}$ )，分别用甲醇溶解后转移至 $25~\mathrm{mL}$ 棕色容量瓶中，并用甲醇定容至刻度，于 $4^{\circ}C$ 冰箱中避光密闭保存，保存期为1个月。  
3.4.2芳香族伯胺标准中间溶液 $\mathrm{A}(1\mathrm{mg / L}$ ，甲醇)：分别吸取29种芳香族伯胺标准储备溶液 $10~\mu \mathrm{L}$ 于

$10~\mathrm{mL}$ 棕色容量瓶中，用甲醇定容至刻度，于 $4^{\circ}\mathrm{C}$ 冰箱中避光密闭保存，保存期为1个月。

3.4.3芳香族伯胺标准中间溶液 $\mathrm{B}(1\mathrm{mg / L}$ ，异辛烷)：分别吸取29种芳香族伯胺标准储备溶液 $10~\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 棕色容量瓶中，用异辛烷定容至刻度，于 $4^{\circ}C$ 冰箱中避光密闭保存，保存期为1个月。

# 3.4.4 标准工作溶液的制备

# 3.4.4.1 异辛烷标准工作液

准确吸取芳香族伯胺标准中间溶液 $\mathrm{B}(3.4.3)30\mu \mathrm{L},50\mu \mathrm{L},100\mu \mathrm{L},300\mu \mathrm{L},500\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 容量瓶中，用异辛烷定容，得到芳香族伯胺浓度分别为 $3.0~\mu \mathrm{g / L},5.0~\mu \mathrm{g / L},10.0~\mu \mathrm{g / L},30.0~\mu \mathrm{g / L},50.0~\mu \mathrm{g / L}$ 的标准工作溶液。移取 $2\mathrm{mL}$ 异辛烷标准工作溶液至试管中，加入 $2\mathrm{mL}$ 甲醇-水提取液(3.2.3)，振荡 $30~s$ ，静置分层后，取下层水溶液，用尼龙滤膜过滤后供测定用。

# 3.4.4.2 其他食品模拟物、化学替代溶剂的标准工作溶液

准确吸取芳香族伯胺标准中间溶液A(3.4.2) $30~\mu \mathrm{L},50~\mu \mathrm{L},100~\mu \mathrm{L},300~\mu \mathrm{L},500~\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 容量瓶中，用 $10\%$ 乙醇定容，得到芳香族伯胺浓度分别为 $3.0~\mu \mathrm{g / L},5.0~\mu \mathrm{g / L},10.0~\mu \mathrm{g / L},30.0~\mu \mathrm{g / L},$ $50.0~\mu \mathrm{g / L}$ 的标准工作溶液。采用同样方式，分别用 $4 \%$ 乙酸、 $20 \%$ 乙醇、 $50 \%$ 乙醇食品模拟物及化学替代溶剂 $95 \%$ 乙醇配制同样浓度系列的芳香族伯胺标准工作溶液。

# 4 仪器和设备

4.1 液相色谱串联质谱仪：配电喷雾离子源（ESI）。  
4.2 分析天平：感量 $0.0001\mathrm{g}$ 。  
4.3 微量注射器： $10~\mu \mathrm{L},100~\mu \mathrm{L},1000~\mu \mathrm{L}$   
4.4 恒温干燥箱或恒温水浴装置。  
4.5 涡旋混合器。  
4.6 针式尼龙过滤器：孔径 $0.22\mu \mathrm{m}$

# 5 分析步骤

# 5.1 食品模拟物的制备

# 5.1.1 总则

本标准迁移试验采用 $4\%$ 乙酸、 $10\%$ 乙醇、 $20\%$ 乙醇、 $50\%$ 乙醇食品模拟物，及化学替代溶剂 $95\%$ 乙醇、异辛烷。植物油不适用于芳香族伯胺迁移量的测定。

# 5.1.2 迁移试验

迁移试验的条件选择及操作步骤按照GB31406.1和GB5009.156的规定。

# 5.1.3 试液制备

# 5.1.3.1 化学替代溶剂异辛烷试液

移取 $2\mathrm{mL}$ 迁移试验后得到的异辛烷至试管中，加入 $2\mathrm{mL}$ 甲醇-水提取液(3.2.3)，涡旋振荡 $30~\mathrm{s}$ 静置分层后，取下层水溶液，用针式尼龙过滤器过滤后供测定用。

# 5.1.3.2 其他食品模拟物和化学替代溶剂 $95\%$ 乙醇试液

移取迁移试验后得到的食品模拟物或 $95\%$ 乙醇替代溶剂约 $1\mathrm{mL}$ ，用针式尼龙过滤器过滤后，供测定用。

# 5.1.3.3 空白试液的制备

未与食品接触材料及制品接触的食品模拟物和化学替代溶剂按5.1.3.1和5.1.3.2处理即得空白试液。

# 6 分析结果的表述

# 6.1 非密封制品类食品接触材料及制品单种芳香族伯胺特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

对于除了盖子、密封圈、连接件等密封制品（以下简称密封制品）以外的食品接触材料及制品，单种芳香族伯胺特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(1)进行计算。

$$
X _ {1} = \frac {\left(c - c _ {0}\right) \times V}{S} \times \frac {S _ {0}}{m _ {1}} \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——单种芳香族伯胺的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试液中芳香族伯胺的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

$c_{0}$ ——空白试液中芳香族伯胺的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——迁移试验中试样浸泡液体积或质量，单位为升 $(\mathrm{L})$ ；

S——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$S_{0}$ ——样品实际使用中与食品接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$m_{1}$ ——样品实际接触固态食品的质量，或实际接触液态食品的体积所对应的食品质量，单位为千克 $(\mathrm{kg})$ ；各种液态食品按密度为 $1\mathrm{kg} / \mathrm{L}$ 将其体积换算为相应的质量。

结果至少保留2位有效数字。

# 6.2 密封制品类食品接触材料及制品单种芳香族伯胺特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

当预期用途已知时，对于密封制品类食品接触材料及制品，单种芳香族伯胺特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(2)进行计算。

$$
X _ {2} = \frac {\left(c - c _ {0}\right) \times V}{S} \times \frac {S _ {0}}{m _ {2}} \quad \dots \tag {2}
$$

式中：

$X_{2}$ ——单种芳香族伯胺的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$S_{0}$ ——密封制品实际使用中与食品接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$m_{2}$ ——密封制品实际使用容器盛装固态食品的质量，或实际接触液态食品的体积所对应的食品质量，单位为千克 $(\mathrm{kg})$ ；各种液态食品按密度为 $1\mathrm{kg} / \mathrm{L}$ 将其体积换算为相应的质量。

结果至少保留2位有效数字。

# 6.3 密封制品类食品接触材料及制品单种芳香族伯胺特定迁移量的计算（以 $\mathrm{mg}$ /件表示）

当预期用途未知时，密封制品类食品接触材料及制品中单种芳香族伯胺特定迁移量以 $\mathrm{mg}/$ 件表示

时，按式(3)计算，需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {3} = \frac {\left(c - c _ {0}\right) \times V}{n} \quad \dots \tag {3}
$$

式中：

$X_{3}$ ——单种芳香族伯胺特定迁移量，单位为毫克每件 $(\mathrm{mg}/$ 件）；

$n$ ——浸泡用密封制品的数量，单位为件。

结果至少保留2位有效数字。

# 6.4 芳香族伯胺迁移量总量的计算

芳香族伯胺迁移量总量为每一种芳香族伯胺的迁移量之和。如果某种芳香族伯胺没有检测信号（即低于3倍背景噪音），应记录为“未检出”或“ND”，其值以零处理。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过其算术平均值的 $10\%$ 。

# 8 其他

本方法对 $4\%$ 乙酸、 $10\%$ 乙醇、 $20\%$ 乙醇、 $50\%$ 乙醇食品模拟物及化学替代溶剂 $95\%$ 乙醇、异辛烷中单种芳香族伯胺的检出限为 $1.0~\mu \mathrm{g / kg}$ ，定量限为 $3.0~\mu \mathrm{g / kg}$ 。

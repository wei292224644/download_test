# 前言

本标准代替GB6781—2007《食品添加剂 乳酸亚铁》。

本标准与GB6781—2007相比，主要变化如下：

——修改了标准名称；  
——增加了乳酸亚铁二水合物；  
——修改了感官要求；  
——删除了干燥失重及其检测方法；  
——增加了水分及其检测方法。

# 食品安全国家标准

# 1 范围

本标准适用于乳酸钠与硫酸亚铁，或乳酸钠与氯化亚铁，或乳酸钙(乳酸铵)与硫酸亚铁（氯化亚铁），或乳酸与铁粉反应制得的食品营养强化剂乳酸亚铁。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

2-羟基丙酸亚铁

# 2.2 分子式

$\mathrm{C_6H_{10}FeO_6}\cdot n\mathrm{H}_2\mathrm{O}(n = 2$ 或3）

# 2.3 结构式

$$
\left[ \begin{array}{c c} & \\ & \mathrm {C O O ^ {-}} \\ & | \\ & \mathrm {C H O H} \\ & | \\ & \mathrm {C H _ {3}} \\ & \end{array} \right] _ {2} \quad \mathrm {F e ^ {2 +}} \cdot n \mathrm {H _ {2} O}
$$

# 2.4 相对分子质量

270.02（二水合物）（按2018年国际相对原子质量）

288.03（三水合物）（按2018年国际相对原子质量）

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及其制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 乙醇。  
A.2.1.2 高锰酸钾溶液： $3.2\mathrm{g / L}$   
A.2.1.3 硫酸溶液： $1 + 20$   
A.2.1.4 铁氰化钾溶液： $100\mathrm{g / L}$   
A.2.1.5 盐酸溶液： $1 + 3$   
A.2.1.6 吗啉溶液： $1 + 4$   
A.2.1.7 亚硝基铁氰化钠溶液： $50~\mathrm{g / L}$   
A.2.1.8 氢氧化钠溶液： $43\mathrm{g / L}$

# A.2.2 鉴别方法

# A.2.2.1 溶解性

略溶于水，几乎不溶于乙醇。

# A.2.2.2 乳酸盐的鉴别

称取约 $1.0\mathrm{g}$ 试样，溶于 $50~\mathrm{mL}$ 水中，制得试样溶液。取 $5\mathrm{mL}$ 试样溶液，加入 $2\mathrm{mL}$ 硫酸溶液混匀，再加 $2\mathrm{mL}$ 高锰酸钾溶液，加热，应有乙醛气体产生。乙醛气体的识别采用等体积吗啉溶液和亚硝基铁氰化钠溶液的混合液湿润过的滤纸，滤纸与气体相接触呈蓝色。

# A.2.2.3 亚铁盐的鉴别

取 $10\mathrm{mL}$ A.2.2.2中的试样溶液，加入 $2\mathrm{mL}$ 铁氰化钾溶液，应生成深蓝色沉淀，加盐酸溶液，沉淀不溶解；若加氢氧化钠溶液，沉淀应溶解。试样溶液与氢氧化钠溶液混合，产生微绿白色沉淀，接着很快变成绿色，摇晃后应呈棕色。

# A.3 乳酸亚铁 $\left(\mathbf{C}_6\mathbf{H}_{10}\mathbf{FeO}_6\right)$ 含量（以干基计）的测定

# A.3.1 方法提要

在酸性介质中，用硫酸铈标准滴定溶液滴定，以1，10-菲啰啉-亚铁指示液指示终点。

# A.3.2 试剂和材料

A.3.2.1 磷酸。  
A.3.2.2 硫酸溶液： $1 + 15$   
A.3.2.3 硫酸铈标准滴定溶液： $c(\mathrm{CeSO}_4) = 0.1\mathrm{mol / L}$   
A.3.2.4 1,10-菲啰啉-亚铁指示液。

# A.3.3 分析步骤

称取 $0.8\mathrm{g}$ 试样（精确到 $0.0001\mathrm{g}$ ），置于 $300\mathrm{mL}$ 的烧瓶中，加入 $160\mathrm{mL}$ 硫酸溶液和 $5\mathrm{mL}$ 磷酸，必要时冷却至室温。加入1滴1,10-菲啰啉-亚铁指示液，立即用 $0.1\mathrm{mol} / \mathrm{L}$ 的硫酸铈标准滴定溶液滴定至红色消失为终点。同时进行空白试验。

# A.3.4 结果计算

乳酸亚铁（ $\mathrm{C_6H_{10}FeO_6}$ ）含量（以干基计)的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\left(V _ {1} - V _ {0}\right) \times c _ {1} \times M _ {1}}{m _ {1} \times \left(1 - w _ {2}\right) \times 1000} \times 100 \% \dots \tag{A.1}
$$

式中：

$V_{1}$ ——滴定试样溶液消耗的硫酸铈标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白溶液消耗的硫酸铈标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——硫酸铈标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{1}$ ——乳酸亚铁的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M_{1}(\mathrm{C}_{6}\mathrm{H}_{10}\mathrm{FeO}_{6}) = 233.96]$

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$\omega_{2}$ ——实测试样水分的质量分数， $\%$ ；

1000——换算系数。

取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不超过算术平均值的 $2\%$ 。

# A.4 水分的测定

# A.4.1 试剂

A.4.1.1 无水甲醇（ $\mathrm{CH_4O}$ ）：优级纯。  
A.4.1.2 甲酰胺：优级纯。  
A.4.1.3 提取溶剂：把无水甲醇(A.4.1.1)和甲酰胺(A.4.1.2)按 $1 + 1$ （体积比）混合，摇匀。  
A.4.1.4 卡尔·费休试剂。

# A.4.2 仪器和设备

A.4.2.1 卡尔·费休水分测定仪。  
A.4.2.2 电子天平：感量为 $0.1\mathrm{mg}$

# A.4.3 分析步骤

称取 $0.1\mathrm{g}$ 样品（精确至 $0.0001\mathrm{g}$ ），加入到适量的提取溶剂(A.4.1.3)中，在搅拌下用卡尔·费休试剂滴定至终点。按GB/T606测定其中的水分含量。

# A.5 三价铁（以 $\mathrm{Fe}^{3+}$ 计）的测定

# A.5.1 方法提要

在酸性介质中，试样中的三价铁与碘化钾作用生成游离碘，用硫代硫酸钠标准溶液滴定，以此计算试样中的铁含量。

# A.5.2 试剂和材料

A.5.2.1 碘化钾。  
A.5.2.2 盐酸。  
A.5.2.3 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$   
A.5.2.4 淀粉指示液： $10\mathrm{g / L}$

# A.5.3 分析步骤

称取 $5\mathrm{g}$ 试样(精确至 $0.0001\mathrm{g}$ )，置于 $250~\mathrm{mL}$ 碘量瓶中，加 $100~\mathrm{mL}$ 水、 $10~\mathrm{mL}$ 盐酸溶解，必要时冷却至室温，加入 $3\mathrm{g}$ 碘化钾，水封瓶塞，振摇溶解，于暗处放置约 $5\mathrm{min}$ 。用硫代硫酸钠标准滴定溶液滴定，近终点时，加 $2\mathrm{mL}$ 淀粉指示液，迅速以硫代硫酸钠标准滴定溶液滴定至溶液蓝色消失为终点。同时进行空白试验。

# A.5.4 结果计算

三价铁（以 $\mathrm{Fe}^{3+}$ 计）的质量分数 $\omega_{3}$ ，按式(A.2)计算：

$$
w_{3} = \frac{\left(V_{3} - V_{2}\right)\times c_{2}\times M_{2}}{m_{2}\times1000}\times 100\% \quad \dots \tag{A.2}
$$

式中：

$V_{3}$ ——滴定试样溶液消耗的硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴定空白溶液消耗的硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{2}$ ——硫代硫酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{2}$ ——铁的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M_2(\mathrm{Fe}) = 55.85]$

$m_{2}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不超过算术平均值的 $10\%$ 。

# A.6 氯化物（以CI计）的测定

# A.6.1 试剂和材料

A.6.1.1 硝酸溶液： $1 + 9$   
A.6.1.2 硝酸银溶液： $17\mathrm{g / L}$   
A.6.1.3 氯化物(Cl)标准溶液：按GB/T602配制后，稀释至每 $1\mathrm{mL}$ 相当于 $0.01\mathrm{mg}$ 氯离子。

# A.6.2 分析步骤

称取 $0.1\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )，置于 $50~\mathrm{mL}$ 纳氏比色管中，加适量水及 $10~\mathrm{mL}$ 硝酸溶液使其溶

解，加 $1\mathrm{mL}$ 硝酸银溶液，用水稀释至 $50~\mathrm{mL}$ ，摇匀，于暗处放置 $5\mathrm{min}$ ，在黑色背景下，轴向观察，所呈浊度与标准比浊溶液比较。

标准比浊溶液：量取 $10\mathrm{mL}$ 氯化物标准溶液，置于 $50~\mathrm{mL}$ 比色管中。与试样溶液同时同样处理。

# A.6.3 结果判定

试样溶液所呈浊度不得深于标准比浊溶液，即试样中的氯化物不大于 $0.1\%$ 。

# A.7 硫酸盐（以 $\mathbf{SO}_4$ 计）的测定

# A.7.1 试剂和材料

A.7.1.1 盐酸溶液： $1 + 4$   
A.7.1.2 氯化钡溶液： $250~\mathrm{g / L}$   
A.7.1.3 硫酸盐 $\left(\mathrm{SO}_{4}\right)$ 标准溶液：按 $\mathrm{GB / T602}$ 配制后，稀释至每 $1\mathrm{mL}$ 相当于 $0.01\mathrm{mg}$ 硫酸根离子。

# A.7.2 分析步骤

称取 $0.1\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），置于 $50~\mathrm{mL}$ 纳氏比色管中，加适量水及 $2\mathrm{mL}$ 盐酸溶液使其溶解，加 $5\mathrm{mL}$ 氯化钡溶液，用水稀释至 $50~\mathrm{mL}$ ，摇匀，于暗处放置 $10\mathrm{min}$ ，在黑色背景下，轴向观察，所呈浊度与标准比浊溶液比较。

标准比浊溶液：量取 $10\mathrm{mL}$ 硫酸盐标准溶液，置于 $50~\mathrm{mL}$ 比色管中。与试样溶液同时同样处理。

# A.7.3 结果判定

试样溶液所呈浊度不得深于标准比浊溶液，即试样中的硫酸盐不大于 $0.1\%$ 。

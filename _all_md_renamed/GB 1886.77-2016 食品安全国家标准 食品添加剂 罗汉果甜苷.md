# 食品添加剂 罗汉果甜苷

# 1 范围

本标准适用于以罗汉果[Siraitia grosvenorii(Swingle)C.Jeffrey]为原料，经水煮提取、浓缩、干燥等工艺精制而成的食品添加剂罗汉果甜苷。

# 2.1 分子式

$$
\mathrm {C} _ {6 0} \mathrm {H} _ {1 0 2} \mathrm {O} _ {2 9}
$$

# 附录A

# 检验方法

# A.1 安全提示

本标准试验方法中使用的部分试剂具有毒性或腐蚀性，按相关规定操作，操作时应小心谨慎。若溅到皮肤上应立即用水冲洗，严重者应立即治疗。在使用挥发性酸时，要在通风柜中进行。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 罗汉果甜苷V对照品：纯度大于等于 $98\%$

A.3.1.2 硫酸。

A.3.1.3 乙酸酐。

A.3.1.4 薄层层析硅胶板：使用前 $110^{\circ}\mathrm{C}$ 活化 $1\mathrm{h}$

A.3.1.5 展开剂：甲醇：乙酸丁酯：水 $= 15:15:4$

A.3.1.6 甲醇溶液：量取甲醇 $70~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度。

A.3.1.7 硫酸溶液： $10\%$ ， $1\rightarrow 10$

A.3.1.8 罗汉果甜苷 V 对照品溶液: 称取 $5 \mathrm{mg} \sim 10 \mathrm{mg}$ 罗汉果甜苷 V 对照品溶于 $1 \mathrm{~mL} \sim 3 \mathrm{~mL}$ 甲醇溶液中, 制成参比液。

# A.3.2 鉴别方法

# A.3.2.1 颜色反应

在 $5\mathrm{mg}\sim 10\mathrm{mg}$ 试样(已干燥)中加入 $2\mathrm{mL}$ 乙酸酐，温热 $2\mathrm{min}$ ，缓慢加入 $0.5\mathrm{mL}$ 硫酸，两层分界面呈现红棕色。

# A.3.2.2 薄层色谱

称取 $0.05\mathrm{g}\sim 0.1\mathrm{g}$ 试样于 $1\mathrm{mL}\sim 3\mathrm{mL}$ 甲醇溶液中，振摇溶解，制成测试液。

用微量点样器分别吸取 $2\mu \mathrm{L}$ 测试液和对照品溶液点样于薄层层析板的点样线上（点样线距硅胶板底部 $2.0~\mathrm{cm}$ ，每次点样均需吹干），放入层析缸(事先放入展开剂)中展开。当溶剂前沿展开至距离点样线约 $10~\mathrm{cm}$ 时，取出薄层板。吹干溶剂，均匀地喷洒硫酸溶液， $105^{\circ}\mathrm{C}$ 加热 $10\mathrm{min}$ 显色。

罗汉果甜苷V显色后呈深紫色，测试液展开后所得的点中，应有一个点的 $R_{f}$ 值及颜色均与罗汉果甜苷V相同。

# A.4 罗汉果甜苷 $\mathbf{V}$ 含量的测定

# A.4.1 方法提要

试样中罗汉果甜苷V用甲醇溶液提取，提取液经过滤后，用配有二极管阵列检测器或紫外检测器的液相色谱仪进行测定。

# A.4.2 试剂和材料

A.4.2.1 罗汉果甜苷V对照品：纯度大于等于 $98\%$   
A.4.2.2 乙腈：色谱纯。  
A.4.2.3 水：符合GB/T6682规定的一级水。  
A.4.2.4 磷酸溶液： $0.05\% ,0.5\rightarrow 1000$   
A.4.2.5 甲醇溶液：量取色谱纯甲醇 $70~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用水(A.4.2.3)稀释至刻度。  
A.4.2.6 罗汉果甜苷 V 对照品溶液: 称取于 $105^{\circ} \mathrm{C}$ 下干燥至恒重的罗汉果甜苷 V 对照品 $10 \mathrm{mg}$ (精确至 $0.0001 \mathrm{~g}$ ) 于 $10 \mathrm{~mL}$ 容量瓶中, 用甲醇溶液溶解并稀释至刻度, 混匀。

# A.4.3 仪器和设备

高效液相色谱仪，配二极管阵列检测器或紫外检测器。

# A.4.4 参考色谱条件

A.4.4.1 色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $150~\mathrm{mm}$ ，内径 $4.6~\mathrm{mm}$ ，内装 $\mathrm{C_{18}}$ 填充物，粒径 $5\mu \mathrm{m}$ ，或是同等分离效果的色谱柱。  
A.4.4.2 流动相：磷酸溶液(A)-乙腈(B)，梯度洗脱程序见表3。  
A.4.4.3 流速： $0.6\mathrm{mL / min}$   
A.4.4.4 检测波长： $203\mathrm{nm}$   
A.4.4.5 进样量： $10~\mu \mathrm{L}$   
A.4.4.6 柱温： $30^{\circ}C$ 。

# A.4.5 分析步骤

# A.4.5.1 试样溶液的制备

称取于 $105\mathrm{^\circ C}$ 下干燥至恒重的试样 $0.05\mathrm{g}\sim 0.1\mathrm{g}$ （精确至 $0.0001\mathrm{g})$ ，置于 $25~\mathrm{mL}$ 容量瓶中，加入 $15~\mathrm{mL}$ 甲醇溶液，超声使之完全溶解，继续加入甲醇溶液定容至刻度，用 $0.45\mu \mathrm{m}$ 微孔滤膜过滤。

# A.4.5.2 测定

在规定色谱条件下，取对照品溶液和试样溶液各 $10~\mu \mathrm{L}$ 分别注入液相色谱仪，进行色谱分析，以保留时间定性，峰面积定量。

# A.4.6 结果计算

罗汉果甜苷V含量的质量分数 $\varpi$ ，按式(A.1)计算：

$$
w = \frac {A \times c \times V}{A _ {\mathrm {s}} \times m \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$A$ ——试样溶液中罗汉果甜苷 $\mathrm{V}$ 的峰面积；

$c$ ——罗汉果甜苷V对照品溶液的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$

$V$ ——试样溶液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{\mathrm{s}}$ ——对照品溶液中罗汉果甜苷V的峰面积；

$m$ ——试样的质量，单位为克(g)；

1000——换算系数。

计算结果保留小数点后两位。

试验结果以两次平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值与算术平均值的比值不大于 $5.0\%$ 。

# 前言

本标准代替GB5009.227—2016《食品安全国家标准 食品中过氧化值的测定》。

本标准与GB5009.227—2016相比，主要变化如下：

——修改了第一法“指示剂滴定法”的范围；

——增加了植脂奶油和粉末油脂制品的试样制备。

# 食品安全国家标准

# 食品中过氧化值的测定

# 1 范围

本标准规定了食品中过氧化值的测定方法。

第一法适用于食品中过氧化值的测定。

第二法适用于食用动植物油脂和人造奶油中过氧化值的测定。

# 第一法 指示剂滴定法

# 2 原理

经制备的油脂试样在三氯甲烷-冰乙酸溶液中溶解，其中的过氧化物与碘化钾反应生成碘，用硫代硫酸钠标准滴定溶液滴定析出的碘。用过氧化物相当于碘的质量分数或 $1\mathrm{kg}$ 样品中活性氧的毫摩尔数表示过氧化值的量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的三级水。

# 3.1 试剂

3.1.1 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.2 三氯甲烷 $\mathrm{(CHCl_3)}$   
3.1.3 碘化钾(KI)。  
3.1.4 石油醚：沸程为 $30^{\circ}\mathrm{C} \sim 60^{\circ}\mathrm{C}$ 。

石油醚的确认：取 $100~\mathrm{mL}$ 石油醚于旋蒸瓶中，在不高于 $40^{\circ}C$ 的水浴中，用旋转蒸发仪减压蒸干。用 $30~\mathrm{mL}$ 三氯甲烷-冰乙酸溶液分次洗涤旋蒸瓶，合并洗涤液于 $250~\mathrm{mL}$ 碘量瓶中。准确加入 $1.00~\mathrm{mL}$ 碘化钾饱和溶液，塞紧瓶盖，并轻轻振摇 $0.5\mathrm{min}$ ，在暗处放置 $3\mathrm{min}$ ，加 $1.0~\mathrm{mL}$ 淀粉指示剂后混匀，若无蓝色出现，此石油醚可用于试样制备；如加 $1.0~\mathrm{mL}$ 淀粉指示剂混匀后有蓝色出现，则需更换试剂。

3.1.5 无水硫酸钠 $(\mathrm{Na}_2\mathrm{SO}_4)$ 。  
3.1.6 可溶性淀粉。  
3.1.7 丙酮 $\mathrm{(CH_3COCH_3)}$ 。  
3.1.8 淀粉酶(CAS号：9000-92-4)：酶活力 $\geq 2000\mathrm{U / g}$   
3.1.9 木瓜蛋白酶(CAS号：9001-73-4)：酶活力 $\geqslant 6000\mathrm{U / mg}$   
3.1.10 硫代硫酸钠 $\mathrm{(Na_2S_2O_3\cdot 5H_2O)}$

# 3.2 试剂配制

3.2.1 三氯甲烷-冰乙酸溶液 $(2 + 3)$ ：将三氯甲烷和冰乙酸按 $2:3$ 的体积比混合均匀。

3.2.2 淀粉指示剂 $(10\mathrm{g / L})$ ：称取 $1\mathrm{g}$ 可溶性淀粉，加入约 $5\mathrm{mL}$ 水使其呈糊状，在搅拌下将 $95~\mathrm{mL}$ 沸水加到糊状物中，再煮沸 $1\mathrm{min}\sim 2\mathrm{min}$ ，冷却。临用现配。  
3.2.3 碘化钾饱和溶液：称取约 $16\mathrm{g}$ 碘化钾，加入 $10\mathrm{mL}$ 适量新煮沸冷却的水，摇匀后贮于棕色瓶中，盖塞，于避光处保存备用，应确保溶液中有饱和碘化钾结晶存在，若超过5.2中空白体积要求时应重新配制。

# 3.3 标准溶液配制

3.3.1 硫代硫酸钠标准滴定溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：按照GB/T5009.1要求进行配制和标定，或经国家认证并授予标准物质证书的标准滴定溶液。  
3.3.2 硫代硫酸钠标准滴定溶液 $(0.01\mathrm{mol} / \mathrm{L})$ ：由 $0.1\mathrm{mol} / \mathrm{L}$ 硫代硫酸钠标准滴定溶液以新煮沸冷却的水稀释而成。临用现配。  
3.3.3 硫代硫酸钠标准滴定溶液 $(0.002\mathrm{mol} / \mathrm{L})$ ：由 $0.01\mathrm{mol} / \mathrm{L}$ 硫代硫酸钠标准滴定溶液以新煮沸冷却的水稀释而成。临用现配。

# 4 仪器和设备

4.1 天平：感量分别为 $0.01\mathrm{g}, 0.001\mathrm{g}, 0.0001\mathrm{g}$ 。  
4.2 电热恒温干燥箱。  
4.3 旋转蒸发仪：配棕色旋蒸瓶。  
4.4 恒温水浴振荡器。  
4.5 高速冷冻离心机：转速 $\geqslant 5000\mathrm{r / min}$   
4.6 顶置搅拌器。  
4.7 滴定管：容量 $10\mathrm{mL}$ ，最小刻度 $0.05\mathrm{mL}$ 。  
4.8 滴定管：容量 $25~\mathrm{mL}$ 或 $50~\mathrm{mL}$ ，最小刻度 $0.1~\mathrm{mL}$ 。

注：本方法中使用的所有器皿不得含有还原性或氧化性物质。磨砂玻璃表面不得涂油。

# 5 分析步骤

# 5.1 试样制备

样品制备过程应避免强光，制备后油脂密闭保存，尽快测定。

# 5.1.1 动植物油脂

对液态样品，振摇装有试样的密闭容器，充分混匀后直接取样；对固态样品，选取有代表性的试样置于密闭容器中混匀后取样。如有必要，将盛有固态试样的容器置于恒温水浴中，加温至开始融化，在该温度下待其全部融化后，振摇混匀，趁试样为液态时立即取样测定。

# 5.1.2 油脂制品

# 5.1.2.1 食用氢化油、起酥油、代可可脂

同5.1.1。

# 5.1.2.2 人造奶油

将样品置于密闭容器中，于 $60^{\circ}\mathrm{C} \sim 70^{\circ}\mathrm{C}$ 的电热恒温干燥箱中加热至融化，振摇混匀后，继续加热

至破乳分层并将油层通过快速定性滤纸过滤到烧杯中,烧杯中滤液为待测试样,待测试样应澄清。趁待测试样为液态时立即取样测定。

# 5.1.2.3 植脂奶油

取有代表性样品于烧杯中，加入约5倍样品体积的石油醚，使用顶置搅拌器搅拌 $2\mathrm{min}$ 使混合均匀。边搅拌边加入约样品1.6倍质量的无水硫酸钠，继续搅拌混合 $5\mathrm{min}$ ，取下静置 $5\mathrm{min}$ 使石油醚分层（如发生乳化现象，将烧杯顶部覆盖一层保鲜膜，置于不高于 $40^{\circ}\mathrm{C}$ 水浴保温 $10\mathrm{min}$ ，使石油醚分层）。将上清液倒出，向烧杯中加入约2倍样品体积的石油醚，重复以上搅拌、静置操作，将石油醚合并，过滤，将滤液转入棕色旋蒸瓶中，在不高于 $40^{\circ}\mathrm{C}$ 的水浴中，用旋转蒸发仪减压蒸干石油醚，残留物即为待测试样，提取量不少于 $5\mathrm{g}$ 。

# 5.1.2.4 粉末油脂制品

称取代表性样品于棕色碘量瓶中，每 $1\mathrm{g}$ 样品加入 $0.02\mathrm{g}$ 木瓜蛋白酶和 $0.02\mathrm{g}$ 淀粉酶。加入2倍样品体积的水混匀，盖塞。将碘量瓶置于 $50~^\circ \mathrm{C}$ 恒温水浴振荡器，60次 $/ \mathrm{min} \sim 100$ 次/min振荡 $30\mathrm{min}$ ，取出冷却。加入与水等体积的丙酮，混匀。加入3倍样品体积的石油醚振摇提取 $1\mathrm{min}$ ，将其转入分液漏斗静置 $30\mathrm{min}$ 分层，弃去下层。若出现乳化现象，可高速冷冻离心分层（ $5000\mathrm{r / min}, 15^{\circ}\mathrm{C}, 3\mathrm{min}$ ），再将有机相转入分液漏斗。加入与石油醚等体积的水洗涤有机相，弃去下层，将上层有机相转入装有无水硫酸钠漏斗进行过滤。将滤液转入棕色旋蒸瓶中，在不高于 $40~^\circ \mathrm{C}$ 的水浴中，用旋转蒸发仪减压蒸干石油醚，残留物即为待测试样，提取量不少于 $5\mathrm{g}$ 。

注：仅含蛋白质壁材样品可只加木瓜蛋白酶；仅含碳水化合物壁材样品可只加淀粉酶。

# 5.1.3 植物性食品及其制品（经油炸、膨化、烘烤、调制、炒制、蒸煮等加工工艺制成）及动物性食品制品（经速冻、干制、腌制、油炸等加工工艺制成）

从所取全部样品中取出有代表性样品的可食部分，除去其中不含油脂部分，含水分较多的速冻调理肉样品可用纱布将水沥干再进行制备。破碎并充分混匀，置于广口瓶中，加入2倍～3倍样品体积的石油醚，摇匀，充分混合，封口后静置浸提 $12\mathrm{h}$ 以上，必要时超声 $5\mathrm{min}\sim 10\mathrm{min}$ 。经装有无水硫酸钠的漏斗过滤，取滤液，在不高于 $40^{\circ}\mathrm{C}$ 的水浴中，用旋转蒸发仪减压蒸干石油醚，残留物即为待测试样，提取量不少于 $5\mathrm{g}$ 。

# 5.2 试样的测定

应避免在阳光直射下进行试样测定。称取制备的试样 $2\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，置于 $250~\mathrm{mL}$ 碘量瓶中，加入 $30~\mathrm{mL}$ 三氯甲烷-冰乙酸溶液，轻轻振摇试样至完全溶解。准确加入 $1.00~\mathrm{mL}$ 碘化钾饱和溶液，塞紧瓶盖，并轻轻振摇 $0.5\mathrm{min}$ ，在暗处放置 $3\mathrm{min}$ 。取出加 $100~\mathrm{mL}$ 水，摇匀后立即用硫代硫酸钠标准滴定溶液(过氧化值估计值在 $0.15\mathrm{g} / 100\mathrm{g}$ 及以下时，用 $0.002\mathrm{mol} / \mathrm{L}$ 标准滴定溶液；过氧化值估计值大于 $0.15\mathrm{g} / 100\mathrm{g}$ 时，用 $0.01\mathrm{mol} / \mathrm{L}$ 标准滴定溶液)滴定析出的碘，滴定至淡黄色时，加 $1\mathrm{mL}$ 淀粉指示剂，继续滴定并强烈振摇至溶液蓝色消失为终点。同时进行空白试验。空白试验所消耗硫代硫酸钠标准滴定溶液体积 $V_{0}$ 不得超过 $0.1~\mathrm{mL}$ 。

# 6 分析结果的表述

6.1 用过氧化物相当于碘的质量分数表示过氧化值时，按式(1)计算。

$$
X _ {1} = \frac {\left(V - V _ {0}\right) \times c \times 0. 1 2 6 9}{m} \times 1 0 0 \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——过氧化值，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$

$V$ ——试样消耗的硫代硫酸钠标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗的硫代硫酸钠标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ —硫代硫酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$

0.1269——与 $1.00\mathrm{mL}$ 硫代硫酸钠标准滴定溶液 $\left[c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3)=1.000\mathrm{mol/L}\right]$ 相当的碘的质量，单位为克每毫摩尔 $(\mathrm{g} / \mathrm{mmol})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

100 ——折算为 $100\mathrm{g}$ 试样的换算系数。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，结果保留2位有效数字。

6.2 用 $1\mathrm{kg}$ 样品中活性氧的毫摩尔数表示过氧化值时，按式(2)计算。

$$
X _ {2} = \frac {\left(V - V _ {0}\right) \times c \times 1 0 0 0}{2 \times m} \quad \dots \tag {2}
$$

式中：

$X_{2}$ ——过氧化值，单位为毫摩尔每千克 $(\mathrm{mmol / kg})$ ；

V ——试样消耗的硫代硫酸钠标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗的硫代硫酸钠标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ —硫代硫酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$

1000 物质的量由摩尔折算为毫摩尔的换算系数；

2 —硫代硫酸钠和活性氧之间的转换系数；

$m$ ——试样质量，单位为克(g)。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，结果保留2位有效数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 电位滴定法

# 8 原理

经制备的油脂试样在异辛烷-冰乙酸溶液中溶解，试样中过氧化物与碘化钾反应生成碘，反应后用硫代硫酸钠标准滴定溶液滴定析出的碘，用电位滴定仪确定滴定终点。用过氧化物相当于碘的质量分数或 $1\mathrm{kg}$ 样品中活性氧的毫摩尔数表示过氧化值的量。

# 9 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 9.1 试剂

9.1.1 冰乙酸 $\mathrm{(CH_3COOH)}$

9.1.2 异辛烷 $(\mathrm{C_8H_{18}})$

9.1.3 碘化钾(KI)。  
9.1.4 硫代硫酸钠 $\mathrm{(Na_2S_2O_3\cdot 5H_2O)}$

# 9.2 试剂配制

9.2.1 异辛烷-冰乙酸溶液 $(2 + 3)$ ：将异辛烷和冰乙酸按 $2:3$ 的体积比混合均匀。  
9.2.2 碘化钾饱和溶液：同3.2.3。

# 9.3 标准溶液配制

同3.3。

# 10 仪器和设备

10.1 天平：感量分别为 $0.01\mathrm{g}, 0.001\mathrm{g}, 0.0001\mathrm{g}$ 。  
10.2 电热恒温干燥箱。  
10.3 电位滴定仪：精度为 $\pm 2\mathrm{mV}$   
10.4 磁力搅拌器。

注：本方法中使用的所有器皿不得含有还原性或氧化性物质。磨砂玻璃表面不得涂油。

# 11 分析步骤

# 11.1 试样制备

同5.1.1与5.1.2.2。

# 11.2 试样的测定

称取制备的试样 $5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于电位滴定仪的滴定杯中，加入 $50~\mathrm{mL}$ 异辛烷-冰乙酸溶液，轻轻振摇使试样完全溶解。如果试样溶解性较差（如硬脂或动物脂肪），可先向滴定杯中加入 $20~\mathrm{mL}$ 异辛烷，轻轻振摇使样品溶解，再加入 $30~\mathrm{mL}$ 冰乙酸后混匀。

向滴定杯中准确加入 $1.00\mathrm{mL}$ 碘化钾饱和溶液，开动磁力搅拌器，在合适的搅拌速度下反应 $60~\mathrm{s} \pm 1~\mathrm{s}$ 。立即向滴定杯中加入 $30~\mathrm{mL} \sim 100~\mathrm{mL}$ 水，插入电极和滴定头，设置好滴定参数，运行滴定程序，进行滴定并观察滴定曲线和电位变化，硫代硫酸钠标准滴定溶液加液量一般控制在 $0.05~\mathrm{mL}/$ 滴～ $0.2~\mathrm{mL}/$ 滴。到达滴定终点后，记录滴定终点消耗的标准溶液体积。每完成一个样品的滴定后，需将搅拌器或搅拌磁子、滴定头和电极浸入异辛烷中清洗表面的油脂。

同时进行空白试验。进行滴定并观察滴定曲线和电位变化，硫代硫酸钠标准滴定溶液加液量一般控制在 $0.005\mathrm{mL}$ /滴。到达滴定终点后，记录滴定终点消耗的标准溶液体积 $V_{0}$ 。空白试验所消耗 $0.01\mathrm{mol} / \mathrm{L}$ 硫代硫酸钠标准滴定溶液体积 $V_{0}$ 不得超过 $0.1\mathrm{mL}$

注1：要保证样品混合均匀又不会产生气泡影响电极响应，可根据仪器说明书的指导，选择一个合适的搅拌速度。

注2：可根据仪器进行加水量的调整，加水量会影响起始电位，但不影响测定结果。被滴定相位于下层，更大量的水有利于相转化，加水量越大，滴定起点和滴定终点间的电位差异越大，滴定曲线上的拐点更明显。

注3：应避免在阳光直射下进行试样测定。

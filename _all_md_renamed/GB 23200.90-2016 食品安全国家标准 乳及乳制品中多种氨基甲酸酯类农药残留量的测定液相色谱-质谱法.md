# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T3156—2012《乳及乳制品中多种氨基甲酸酯类农药残留量测定方法液相色谱一串联质谱法》。与SN/T3156—2012相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；

——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

# 1 范围

本标准规定了乳和乳制品中杀线威、灭多威、抗蚜威、涕灭威、速灭威、噁虫威、克百威、甲萘威、呋线威、异丙威、乙霉威、仲丁威、残杀威和甲硫威14种氨基甲酸酯类农药残留量的液相色谱一质谱检测方法。

本标准适用于纯奶、酸奶、奶粉、奶酪和果奶中杀线威、灭多威、抗蚜威、涕灭威、速灭威、噁虫威、克百威、甲萘威、呋线威、异丙威、乙霉威、仲丁威、残杀威和甲硫威残留量的测定和确证，其他食品可参照执行。

# 3 原理

试样用乙腈提取，提取液经固相萃取柱净化后，甲醇洗脱，用液相色谱一串联质谱仪检测和确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ ：残留级。  
4.1.2 甲醇 $\mathrm{(CH_4O)}$ ：残留级。  
4.1.3 甲酸 $\mathrm{(CH_2O_2)}$ ：优级纯。  
4.1.4 甲醇—水溶液 $(3 + 2$ ，体积比)：准确量取 $60~\mathrm{mL}$ 甲醇和 $40~\mathrm{mL}$ 水，混合后备用。  
4.1.5 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：分析纯，经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，储于密封容器中备用。  
4.1.6 氯化钠(NaCl)：分析纯。

# 4.2 标准品

农药标准物质：杀线威、灭多威、抗蚜威、涕灭威、速灭威、噁虫威、克百威、甲萘威、呋线威、异丙威、乙霉威、仲丁威、残杀威和甲硫威等，纯度均 $\geq 98.5\%$ 。标准物质信息参见附录A。

# 4.3 标准溶液配制

# 4.3.1 标准储备溶液

分别准确称取适量的各种氨基甲酸酯标准物质，用甲醇配制成浓度为 $100\mu \mathrm{g / mL}$ 的标准储备溶液， $-18^{\circ}C$ 保存。

# 4.3.2 混合标准中间溶液

(1)分别准确吸取适量的氨基甲酸酯类农药标准储备溶液,用甲醇配制成浓度为 $5 \mu \mathrm{g} / \mathrm{mL}$ 的混合标准

中间溶液， $-18^{\circ}\mathrm{C}$ 保存。

# 4.3.3 基质混合标准工作溶液

根据需要吸取适量的混合标准中间溶液，用空白样品基质溶液配制成适当浓度的混合标准工作溶液， $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱中保存，使用前配制。

# 4.4 材料

# 4.4.1 微孔过滤膜

$13\mathrm{mm}\times 0.22\mu \mathrm{m}$ ，有机系。

# 4.4.2 $\mathbf{C}_{18}$ 固相萃取柱

ENVI™-18¹) 6 mL/1000 mg, 或相当者。

# 5 仪器和设备

5.1 液相色谱—串联质谱仪：配备电喷雾离子源(ESI)。  
5.2 天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 均质器。  
5.4 离心机： $4000\mathrm{r / min}$   
5.5 旋转蒸发器。  
5.6 聚丙烯离心管： $50~\mathrm{mL}$   
5.7 烧瓶： $100~\mathrm{mL}$

# 6 试样保存

纯奶、酸奶、果奶、奶酪等于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存；奶粉密封常温保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 纯奶、酸奶、果奶、奶酪等

称取试样 $5\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 于 $50\mathrm{mL}$ 离心管中，加入 $20\mathrm{mL}$ 乙腈以及 $5\mathrm{g}$ 无水硫酸钠，均质提取 $1\mathrm{min}$ ，加入 $2\mathrm{g}$ 氯化钠，振荡，于 $4000\mathrm{r / min}$ 离心 $3\mathrm{min}$ 。吸取上清液，残渣再用 $10\mathrm{mL}$ 乙腈重复提取 1 次，合并上清液，于 $40^{\circ}\mathrm{C}$ 水浴中旋转蒸发浓缩至近干，加 $5\mathrm{mL}$ 甲醇溶解，待净化。

# 7.1.2 奶粉等

称取试样 $3 \mathrm{~g}$ (精确到 $0.01 \mathrm{~g}$ ) 于 $50 \mathrm{~mL}$ 离心管中, 加入 $5 \mathrm{~mL}$ 水, $30^{\circ} \mathrm{C}$ 水浴振荡 $10 \mathrm{~min}$ 。再加入 $20 \mathrm{~mL}$ 乙腈以及 $5 \mathrm{~g}$ 无水硫酸钠, 均质提取 $1 \mathrm{~min}$ , 加入 $2 \mathrm{~g}$ 氯化钠, 振荡, 于 $4000 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ 。吸取上清液, 残渣再用 $10 \mathrm{~mL}$ 乙腈重复提取 1 次, 合并上清液, 于 $40^{\circ} \mathrm{C}$ 水浴中旋转蒸发浓缩至近干, 加 $3 \mathrm{~mL}$ 甲醇溶解, 待净化。

# 7.2 净化

用 $10\mathrm{mL}$ 甲醇活化ENVI-18固相萃取柱后，弃去活化液，吸取 $2\mathrm{mL}$ 提取液上样。用 $15\mathrm{mL}$ 甲醇进行洗脱(流速不超过 $1\mathrm{mL / min}$ )。收集上样液和全部洗脱液于 $100~\mathrm{mL}$ 烧瓶中，于 $40^{\circ}\mathrm{C}$ 水浴中旋转蒸发浓缩至近干。用氮气吹干，甲醇一水溶液 $(3 + 2$ ，体积比)溶解并定容至 $1.0\mathrm{mL}$ ，过滤膜，供液相色谱一串联质谱仪测定和确证。

# 7.3 测定

# 7.3.1 液相色谱参考条件

a）色谱柱：Thermo Hypersil GOLD $\mathrm{C_{18}}$ ， $150\mathrm{mm}\times 2.1\mathrm{mm}$ （内径)，膜厚 $5\mu \mathrm{m}$ ，或相当者；  
b） 柱温： $30^{\circ}\mathrm{C}$   
c) 流速: $0.25 \mathrm{~mL} / \mathrm{min}$ ;   
d）进样量： $10~\mu \mathrm{L}$   
e）流动相及梯度洗脱条件见表1。

# 7.3.2 质谱参考条件

# 7.3.3 色谱测定与确证

根据样液中被测物含量情况，选定浓度相近的基质混合标准工作溶液。基质混合标准工作溶液和待测样液中14种氨基甲酸酯类农药的响应值均应在仪器检测的线性范围内。基质混合标准工作溶液与样液等体积参插进样测定。14种氨基甲酸酯类农药标准品的多反应监测(MRM)色谱图参见附录B。

标准溶液及样液均按7.3.1和7.3.2规定的条件进行测定。如果样液中与标准溶液相同的保留时间有峰出现，则对其进行确证。经确证分析被测物质量色谱峰保留时间与标准物质相一致，并且在扣除背景后的样品谱图中，所选择的离子均出现；同时，所选择离子的丰度比与标准样物质相关离子的相对丰度一致，相似度在允许偏差之内(见表2)，则可判定样品中存在对应的被测物。采用基质溶液标准曲线外标法定量。

# 7.4 空白试验

除不称取试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理软件或按(1)式计算试样中14种氨基甲酸酯类农药残留量。

$$
X _ {i} = \frac {A _ {i} \times c _ {i} \times V \times 1 0 0 0}{A _ {i s} \times m \times 1 0 0 0} \dots \tag {1}
$$

式中：

$X_{i}$ ——试样中农药 $i$ 的残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A_{i}$ ——样液中氨基甲酸酯类农药 $i$ 的峰面积(或峰高)；

$c_{i}$ ——标准工作液中农药 $i$ 的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

V ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{\mathrm{is}}$ ——标准工作液中农药 $i$ 的峰面积(或峰高)；

m 最终样液所代表的试样质量，单位为克（g）；

1000 ——换算系数。

计算结果应扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率),应符合附录 C 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 D 的要求。

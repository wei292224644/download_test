本标准适用于以蔗糖为原料，用氯原子选择性取代三个羟基而制得的食品添加剂三氯蔗糖。

# 检验方法

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682—2008中规定的水。分析中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。本试验所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

A.2.1 在检测三氯蔗糖含量试验中, 试样液在液相色谱图中的主峰保留时间应与标准溶液中三氯蔗糖的保留时间相同。  
A.2.2 在检测相关物质试验中，试样液在薄层色谱图中主色斑的 $R_{f}$ 值应与标准溶液主色斑的 $R_{f}$ 值相同。

# A.3 三氯蔗糖的测定

# A.3.1 试剂和材料

a）乙腈：色谱纯。  
b）三氯蔗糖标准品：质量分数 $\geq 98.0\%$

# A.3.2 仪器和设备

高效液相色谱仪：配备示差折光检测器，或其他等效的检测器。

# A.3.3 参考色谱条件

a）色谱柱： $\mathrm{C_{18}}$ 反相色谱柱， $8\mathrm{mm}\times 10\mathrm{cm}$ ，粒度 $5\mu \mathrm{m}$ ；或其他等效的色谱柱。  
b）流动相：将 $150~\mathrm{mL}$ 乙腈与 $850~\mathrm{mL}$ 水混合均匀后，用 $0.45\mu \mathrm{m}$ 滤膜过滤，超声脱气后备用。  
c） 柱温：室温。  
d) 流动相流速: $1.5 \mathrm{~mL} / \mathrm{min}$ 。  
e）进样量： $20~\mu \mathrm{L}$   
f）三氯蔗糖保留时间： $9\mathrm{min}$ 左右，为确保获得所需的保留时间，必要时可以调整流动相的比例。

注：系统适用性为重复注入标准溶液两次，所得响应面积的相对误差小于 $2.0\%$ 。

# A.3.4 分析步骤

# A.3.4.1 标准溶液的制备

称取约 $0.025\mathrm{g}$ 三氯蔗糖标准品（精确至 $0.0001\mathrm{g}$ ），用流动相溶解，移入 $25\mathrm{mL}$ 容量瓶中，加流动相定容至刻度，所得溶液用 $0.45\mu \mathrm{m}$ 滤膜过滤，滤液备用。

# A.3.4.2 试样液的制备

称取约 $0.025\mathrm{g}$ 试样(精确至 $0.0001\mathrm{g}$ )，用流动相溶解，移入 $25~\mathrm{mL}$ 容量瓶中，加流动相定容至刻度，所得溶液用 $0.45\mu \mathrm{m}$ 滤膜过滤，滤液备用。

# A.3.5 测定

在A.3.3参考色谱条件下，分别对标准溶液和试样液进行测定，进样量为 $20~\mu \mathrm{L}$ ，重复进样一次，计算出其主峰面积平均值。

# A.3.6 结果计算

三氯蔗糖的含量 $X_{1}$ 按公式(A.1)计算：

$$
X _ {1} = \frac {A _ {\mathrm {U}} \times m _ {\mathrm {S}} \times P}{A _ {\mathrm {S}} \times m _ {\mathrm {U}} \times \left(1 - X _ {0}\right)} \times 1 0 0 \quad \dots \tag {A.1}
$$

式中：

$X_{1}$ ——试样中三氯蔗糖的含量， $\%$ ；

$A_{\mathrm{U}}$ ——试样液色谱主峰面积的平均值；

$m_{\mathrm{S}}$ ——称取的三氯蔗糖标准品质量，单位为克 $(\mathrm{g})$ ；

$P$ 三氯蔗糖标准品中标示的三氯蔗糖的含量， $\%$

$A_{\mathrm{S}}$ ——标准溶液色谱主峰面积的平均值；

$m_{\mathrm{U}}$ ——称取的试样质量，单位为克 $(\mathrm{g})$

$X_0$ ——实测试样的水分含量， $\%$ 。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $2\%$ 。

# A.4 比旋光度的测定

A.4.1 称取约 $1\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ ），用水溶解，转移至 $100\mathrm{mL}$ 容量瓶中，加水定容至刻度，摇匀。测定温度为 $20^{\circ}\mathrm{C} \pm 0.5^{\circ}\mathrm{C}$ 。

比旋光度 $\alpha_{m}(20^{\circ}\mathrm{C},\mathrm{D})$ 数值以 $(^{\circ})\cdot \mathrm{dm}^{2}\cdot \mathrm{kg}^{-1}$ 表示，按公式(A.2)计算：

$$
\alpha_ {m} (2 0 ^ {\circ} \mathrm {C}, \mathrm {D}) = \frac {\alpha}{l \rho_ {\alpha}} \tag {A.2}
$$

式中：

$\alpha$ 测得的旋光角，单位为度 $(^{\circ})$

$l$ ——旋光管的长度，单位为分米（dm）；

$\rho_{\alpha}$ ——溶液中有效组分的质量浓度，单位为克每毫升 $(\mathrm{g / mL})$ 。

A.4.2 其他按GB/T613—2007的规定进行。

# A.5 水解产物的测定

# A.5.1 试剂和材料

a) $p$ -茴香胺。  
b）邻苯二甲酸。

c）甲醇。  
d）甘露糖醇。  
e) 果糖。

# A.5.2 仪器和设备

薄层层析板：涂有 $0.25\mathrm{mm}$ 厚度的Merk硅胶60或其他等效物质。

# A.5.3 分析步骤

# A.5.3.1 显色剂的制备

将 $1.23\mathrm{g}p$ -茴香胺和 $1.66\mathrm{g}$ 邻苯二甲酸溶于 $100~\mathrm{mL}$ 甲醇中。将溶液存放在暗处并冷藏，如果溶液退色则已失效。

# A.5.3.2 标准溶液A的制备

称取 $10\mathrm{g}$ 甘露糖醇(精确至 $0.001\mathrm{g}$ )，用水溶解后，移入 $100\mathrm{mL}$ 容量瓶中，用水定容至刻度。

# A.5.3.3 标准溶液B的制备

分别称取 $10\mathrm{g}$ 甘露糖醇（精确至 $0.001\mathrm{g}$ ）和 $0.04\mathrm{g}$ 果糖（精确至 $0.0001\mathrm{g}$ ），移入 $100\mathrm{mL}$ 容量瓶中，用水定容至刻度。

# A.5.3.4 试样液的制备

称取 $2.5\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ ），用 $5\mathrm{mL}$ 甲醇溶解后，转移至 $10\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度。

# A.5.4 测定

取标准溶液A、标准溶液B和试样液各 $5\mu \mathrm{L}$ ，分别在同一块薄层层析板的不同位置进行点样，将每份溶液分5次(每次 $1\mu \mathrm{L})$ 点于板上，每次点样之间要待样点干燥后再继续点，3份样点的面积要基本相同，点样完毕后用显色剂喷雾后于 $100^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 烘箱中加热 $15\mathrm{min}$ ，加热后立即在阴暗背景下观察层析板，试样液的色斑呈色不应深于标准溶液B的色斑呈色。

注：如果标准溶液A的点样点变黑，说明层析板加热时间过长，需重新制备。

# A.6 相关物质的测定

# A.6.1 试剂和材料

a）三氯蔗糖标准品：质量分数 $\geq 98.0\%$   
b）乙腈。  
c）硫酸。  
d) 甲醇。  
e) 氯化钠溶液: $50 \mathrm{~g} / \mathrm{L}$ 。

# A.6.2 仪器和设备

薄层层析板：涂有 $0.2\mathrm{mm}$ 厚度的 $\mathrm{C_{18}}$ 烷基修饰的硅胶或其他等效物质。

# A.6.3 分析步骤

# A.6.3.1 展开剂的制备

展开剂：氯化钠溶液：乙腈 $= 70:30$ （体积比）。

# A.6.3.2 显色剂的制备

配制 $15\%$ 硫酸的甲醇溶液（体积分数）。

# A.6.3.3 标准溶液的制备

称取 $0.5\mathrm{g}$ 三氯蔗糖标准品(精确至 $0.001\mathrm{g}$ )，溶解于 $5.0\mathrm{mL}$ 甲醇中，此为溶液C。吸取 $0.5\mathrm{mL}$ 的溶液C，用甲醇定容至 $100\mathrm{mL}$ ，此为溶液D。

# A.6.3.4 试样液的制备

称取 $1.0\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ ），溶解于 $10.0\mathrm{mL}$ 甲醇中。

# A.6.4 测定

取溶液C、溶液D和试样液各 $5\mu \mathrm{L}$ ，点于薄层层析板底部，将层析板置于盛有展开剂的层析缸中，待展开的溶剂前沿移至 $15~\mathrm{cm}$ 时后取出薄层层析板，放置，待展开剂挥发干净后用显色剂喷雾，然后于 $125^{\circ}C\pm 2^{\circ}C$ 烘箱中加热 $10\mathrm{min}$ 。试样液主色斑的 $R_{f}$ 值应与溶液C主色斑的 $R_{f}$ 值相同，且试样液中其他色斑的呈色应不深于溶液D主色斑的呈色。

$R_{f}$ 值——试样展开后斑点到原点的距离与溶剂前沿到原点的距离的比值。

# A.7 甲醇的测定

# A.7.1 试剂和材料

a）甲醇标准品：色谱纯。  
b）正丙醇：色谱纯。  
c）吡啶：色谱纯。

# A.7.2 仪器和设备

气相色谱仪：配有氢火焰离子化检测器。

# A.7.3 参考色谱条件

a）色谱柱： $2.1\mathrm{m}\times 4\mathrm{mm}$ (内径)玻璃柱，内填充 $0.2\mathrm{mm}\sim 0.15\mathrm{mm}$ 聚苯乙烯型色谱固定相；或其他等效色谱柱。  
b）载气：氮气或氦气。  
c） 柱温： $150~\mathrm{^\circ C}$ 。  
d）进样口温度： $200^{\circ}C$ 。  
e）检测器温度： $250~\mathrm{^\circ C}$ 。  
f) 流速: $20 \mathrm{~mL} / \mathrm{min}$ 。  
g）进样量： $1\mu \mathrm{L}$

注：系统适用性为重复注入标准溶液两次，所得响应面积的相对误差小于 $2.0\%$ 。

# A.7.4 分析步骤

# A.7.4.1 内标溶液的制备

准确吸取 $1.0\mathrm{mL}$ 正丙醇，置于 $100~\mathrm{mL}$ 容量瓶中，用吡啶定容至刻度，摇匀。移取 $5\mathrm{mL}$ 该溶液，置于 $500~\mathrm{mL}$ 容量瓶中，用吡啶定容至刻度，摇匀。

# A.7.4.2 标准溶液的制备

准确吸取 $2.0\mathrm{mL}$ 甲醇，置于 $100~\mathrm{mL}$ 容量瓶中，用内标溶液定容至刻度，摇匀。移取 $1.0\mathrm{mL}$ 该溶液，置于 $100~\mathrm{mL}$ 容量瓶中，用内标溶液定容至刻度，摇匀。

# A.7.4.3 试样液的制备

称取约 $2\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )，用内标溶液溶解，移入 $10~\mathrm{mL}$ 容量瓶中，用内标溶液定容至刻度，摇匀。

# A.7.5 测定

在A.7.3参考色谱条件下，分别对标准溶液和试样液进行测定，进样量为 $1\mu \mathrm{L}$ ，重复进样一次，计算出每次进样甲醇峰面积与内标物峰面积的比值。

# A.7.6 结果计算

甲醇的含量 $X_{2}$ 按公式(A.3)计算：

$$
X _ {2} = \frac {R _ {\mathrm {U}} \times 0.00158}{R _ {\mathrm {S}} \times m _ {\mathrm {U}}} \times 100 \% \quad \dots \tag{A.3}
$$

式中：

$X_{2}$ ——试样中甲醇的含量， $\%$ ；

$R_{\mathrm{U}}$ ——两次进样试样液中甲醇与内标物(正丙醇)峰面积之比的平均值；

0.00158 ——标准溶液中甲醇的浓度 $\times$ 甲醇的密度 $\times$ 试样液的体积 $(2 \times 10^{-4} \times 0.79 \times 10)$ ；

$R_{\mathrm{S}}$ ——两次进样标准溶液中甲醇与内标物(正丙醇)峰面积之比的平均值；

$m_{\mathrm{U}}$ —称取的试样质量，单位为克 $(\mathrm{g})$ 。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# 动物性食品中拟除虫菊酯类药物残留量的测定

# 气相色谱-质谱法

# 1 范围

本文件规定了动物性食品中拟除虫菊酯类药物残留量检测的制样和气相色谱-质谱测定方法。

本文件适用于牛、羊、猪的肌肉、脂肪和肝脏中溴氰菊酯、联苯菊酯、氟氰戊菊酯、氟胺氰菊酯、七氟菊酯和氰戊菊酯单个或多个药物残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的拟除虫菊酯类药物,用乙腈提取,固相萃取柱净化,气相色谱质谱法测定,外标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 正己烷 $\mathrm{(C_6H_{10})}$ ：色谱纯。  
5.1.3 氯化钠(NaCl)。  
5.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)}$   
5.1.5 苯 $(\mathrm{C_6H_6})$ ：色谱纯。  
5.1.6 丙酮 $\mathrm{(CH_3COCH_3)}$ ：色谱纯。

# 5.2 标准品

溴氰菊酯含量 $\geq 99.7\%$ ，联苯菊酯含量 $\geq 98\%$ ，氟氰戊菊酯含量 $\geq 87.5\%$ ，氟胺氰菊酯含量 $\geq 94\%$ ，七氟菊酯含量 $\geq 98\%$ ，氰戊菊酯含量 $\geq 99\%$ ，见附录A。

# 5.3 标准溶液制备

5.3.1 标准储备液：取溴氰菊酯、联苯菊酯、氟氰戊菊酯、氟胺氰菊酯、七氟菊酯和氰戊菊酯标准品各 $10\mathrm{mg}$ ，精密称定，加苯适量使溶解，用丙酮稀释定容至 $100\mathrm{mL}$ 容量瓶，摇匀，制成溴氰菊酯、联苯菊酯、氟氰戊菊酯、氟胺氰菊酯、七氟菊酯和氰戊菊酯标准储备液。 $-18^{\circ}\mathrm{C}$ 保存，有效期3个月。  
5.3.2 $10~\mu \mathrm{g / mL}$ 混合标准工作液：分别精密量取上述标准储备液各 $1\mathrm{mL}$ ，于 $10~\mathrm{mL}$ 容量瓶，用丙酮稀释刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 混合标准工作液。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1个月。  
5.3.3 $1.0\mu \mathrm{g / mL}$ 混合标准工作液：精密量取 $10~\mu \mathrm{g / mL}$ 混合标准工作液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶，用丙酮稀释至刻度，配制成浓度为 $1.0\mu \mathrm{g / mL}$ 混合标准工作液。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1个月。

# 5.4 材料

中性氧化铝固相萃取柱： $500\mathrm{mg / 6mL}$ ，或相当者。

# 6 仪器与设备

6.1 气相色谱-质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 高速离心机。  
6.4 涡旋混合器。  
6.5 水平振荡器。  
6.6 组织匀浆机。  
6.7 固相萃取装置。  
6.8 氮吹仪。

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试料 $5\mathrm{g}$ （准确到 $\pm 0.02\mathrm{g}$ ，于 $50~\mathrm{mL}$ 聚丙烯离心管，加氯化钠 $4\mathrm{g}$ 、乙腈 $25~\mathrm{mL}$ ，匀浆 $1\mathrm{min}$ ，振荡 $15\mathrm{min}$ ，以 $6000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液于另一离心管中，残渣中加乙腈 $15~\mathrm{mL}$ 重复提取1次，合并上清液，加无水硫酸钠 $4\mathrm{g}$ ，振荡，于 $4^{\circ}C$ 、 $1000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液，备用。

# 8.2 净化

取中性氧化铝固相萃取柱，用乙腈分2次活化，每次 $5\mathrm{mL}$ 。取备用液过柱，用乙腈 $10\mathrm{mL}$ 淋洗。收集全部淋洗液， $50^{\circ}\mathrm{C}$ 旋转蒸发至干，加正己烷 $1.0\mathrm{mL}$ 使溶解，过滤，供气相色谱-质谱测定。

# 8.3 基质匹配标准曲线的制备

精密量取 $1.0\mu \mathrm{g / mL}$ 混合标准工作液 $0.01\mathrm{mL},0.02\mathrm{mL},0.05\mathrm{mL},10\mu \mathrm{g / mL}$ 拟除虫菊酯混合标准工作液 $0.01\mathrm{mL},0.05\mathrm{mL}$ 和 $0.10\mathrm{mL}$ ，分别加入6份经样品前处理步骤处理的空白试料浓缩液中，加正己烷稀释至 $1\mathrm{mL}$ ，制得浓度为 $10\mathrm{ng / mL},20\mathrm{ng / mL},50\mathrm{ng / mL},100\mathrm{ng / mL},500\mathrm{ng / mL},1000\mathrm{ng / mL}$ 的基质匹配系列标准溶液，过滤，供气相色谱-质谱测定。以测得的特征离子质量色谱峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制基质匹配标准曲线。求回归方程和相关系数。

# 8.4 测定

# 8.4.1 气相色谱参考条件

a）色谱柱：DB-1( $100\%$ 二甲基聚硅氧烷)毛细管柱： $30~\mathrm{m}\times 0.25~\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温：起始柱温 $70^{\circ}\mathrm{C}$ ，以 $30^{\circ}\mathrm{C / min}$ 升至 $250^{\circ}\mathrm{C}$ ，以 $3^{\circ}\mathrm{C / min}$ 升至 $274^{\circ}\mathrm{C}$ ，以 $20^{\circ}\mathrm{C / min}$ 升至 $294^{\circ}\mathrm{C}$ ，以 $30^{\circ}\mathrm{C / min}$ 升至 $300^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$   
c）载气：高纯氦气，柱流速 $1.3\mathrm{mL} / \mathrm{min}$   
d) 分流：不分流进样；

e) 进样量: $1 \mu \mathrm{L}$ ;   
f）进样口温度： $280^{\circ}C$   
g） 接口温度： $270^{\circ}C$ □

# 8.4.2 质谱参考条件

a）离子源：NCI源；  
b）反应气： $\mathrm{CH}_4$ （纯度大于 $99.99\%$ ）；  
c）离子源温度： $150~\mathrm{^\circ C}$   
d）电离电压： $0.98\mathrm{kV}$   
e）四极杆质量分析器温度： $150~\mathrm{^\circ C}$   
f）电子能量： $235\mathrm{eV}$   
g）灯丝电流： $44.5\mu \mathrm{A}$   
h）溶剂延迟时间： $5.0\mathrm{min}$   
i）数据采集方式：选择离子监测方式，具体参数见表1。

# 8.4.3 测定法

取试料溶液和基质匹配标准溶液，作单点或多点校准，按外标法以色谱峰面积定量。基质匹配标准溶液及试料溶液中目标药物的特征离子质量色谱峰面积均应在仪器检测的线性范围之内。试料溶液中拟除虫菊酯类药物的特征离子质量色谱峰面积均应在仪器检测的线性范围内。试样溶液中的相对离子丰度与基质匹配标准溶液中的相对离子丰度相比，符合表2的要求。标准溶液中特征离子质量色谱图分别见附录B。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试料中待测物的残留量按公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V}{A _ {\mathrm {S}} \times m} \tag {1}
$$

式中：

$X$ 供试试料中相应的拟除虫菊酯类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$C_{\mathrm{S}}$ 基质匹配溶液中相应的拟除虫菊酯类药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$A_{\mathrm{S}}$ 基质匹配溶液中相应的拟除虫菊酯类药物峰面积；

$A$ 供试试料溶液中相应的拟除虫菊酯类药物峰面积；

$V$ ——浓缩后定容体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ 供试试料质量的数值，单位为克 $(\mathrm{g})$

# 10 检测方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $3\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $10~\mu \mathrm{g / kg}\sim 500~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 20\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

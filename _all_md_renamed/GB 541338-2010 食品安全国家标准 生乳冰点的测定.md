# 食品安全国家标准生乳冰点的测定

# 1 范围

本标准规定了热敏电阻冰点仪测定生乳冰点的基准方法。

本标准适用于生乳冰点的测定。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

# 3 术语和定义

生乳冰点 FPD (freezing point depression)

使用本标准规定测得的数值为原料乳的冰点，单位以摄氏千分之一度（ $\mathrm{m}^{\circ}\mathrm{C}$ ）表示。

# 4 原理

样品管中放入一定量的乳样，置于冷阱中，于冰点以下制冷。当被测乳样制冷到 $-3\,^{\circ}\mathrm{C}$ 时，进行引晶，结冰后通过连续释放热量，使乳样温度回升至最高点。并在短时间内保持恒定，为冰点温度平台，该温度即为该乳样的冰点值。

# 5 试剂和材料

除非另有说明，本方法所用试剂均为分析纯或以上规格，水为GB/T6682规定的一级水。

5.1 氯化钠（NaCl）：磨细后置于干燥炉中， $130^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 干燥 $24\mathrm{h}$ 以上，于干燥器中冷却至室温。

5.2 乙二醇（ $\mathrm{C_2H_6O_2}$ ）。

# 5.3 校准液

选择两种不同冰点的氯化钠标准溶液，氯化钠标准溶液与被测牛奶样品的冰点值相近，且所选择的两份氯化钠标准溶液的冰点值之差不得少于 $100 \mathrm{~m}^{\circ} \mathrm{C}$ ，见表1。

5.3.1 校准液A（ $20^{\circ}\mathrm{C} \sim 25^{\circ}\mathrm{C}$ 室温下）：称取 $6.731\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）氯化钠（5.1），溶于少量水中，定容至 $1000\mathrm{mL}$ 容量瓶中。其冰点值为 $-0.400^{\circ}\mathrm{C}$ 。  
5.3.2 校准液B： $20^{\circ} \mathrm{C}$ 室温下：称取 $9.422 \mathrm{~g}$ （精确至 $0.0001 \mathrm{~g}$ ）氯化钠（5.1），溶于少量水中，定容至 $1000 \mathrm{~mL}$ 容量瓶中。其冰点值为 $-0.557^{\circ} \mathrm{C}$ 。

# 5.4 冷却液

准确量取 $330 \mathrm{~mL}$ 乙二醇（5.2）于 $1000 \mathrm{~mL}$ 容量瓶中，用水定容至刻度并摇匀，其体积比分数为 $33 \%$ 。

# 6 仪器和设备

# 6.1 天平：感量为 $0.1\mathrm{mg}$ 。

6.2 热敏电阻冰点仪：带有热敏电阻控制的冷却装置（冷阱），热敏电阻探头，搅拌器和引晶装置（见图1）及温度显示仪。

# 6.2.1 检测装置，温度传感器和相应的电子线路

温度传感器为直径为 $1.60 \mathrm{~mm} \pm 0.4 \mathrm{~mm}$ 的玻璃探头，在 $0^{\circ} \mathrm{C}$ 时的电阻在 $3 \Omega$ 和 $30 \mathrm{k} \Omega$ 之间。当探头在测量位置时，热敏电阻的顶部应位于样品管的中轴线，且顶部离内壁与管底保持相等距离（见图1）。温度传感器和相应的电子线路在 $-400 \mathrm{~m}^{\circ} \mathrm{C}$ 至 $-600 \mathrm{~m}^{\circ} \mathrm{C}$ 之间测量分辨率为 $1 \mathrm{~m}^{\circ} \mathrm{C}$ 或更好。

# 6.2.2 搅拌金属棒：耐腐蚀，在冷却过程中搅拌测试样品。

搅拌金属棒应根据相应仪器的安放位置来调整振幅。正常搅拌时金属棒不得碰撞玻璃传感器或样品管壁。

6.2.3 引晶装置：操作时，测试样品达到 $-3.0^{\circ}C$ 时启动引晶的机械振动装置  
在引晶时使搅拌金属棒在 $1\mathrm{s}\sim 2\mathrm{s}$ 内加大振幅，使其碰撞样品管壁。  
6.3 样品管：硼硅玻璃，长度 $50.5\mathrm{mm} \pm 0.2\mathrm{mm}$ ，外部直径为 $16.0\mathrm{mm} \pm 0.2\mathrm{mm}$ ，内部直径为 $13.7\mathrm{mm} \pm 0.3\mathrm{mm}$ 。  
6.4 称量瓶。  
6.5 容量瓶： $1000~\mathrm{mL}$   
6.6 烘箱：温度可控制在 $150^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 。  
6.7 干燥器。  
6.8 移液器： $1\mathrm{mL}\sim 5\mathrm{mL}$

# 7 分析步骤

# 7.1 试样制备

测试样品要保存在 $0^{\circ}\mathrm{C}$ 到 $6^{\circ}\mathrm{C}$ 的冰箱中，样品抵达实验室时立即检测效果最好。测试前样品温度到达室温，且测试样品和氯化钠标准溶液测试时的温度应一致。

# 7.2 仪器预冷

开启冰点仪，等待冰点仪传感探头升起后，打开冷阱盖，按生产商规定加入相应体积冷却液（5.4），盖上盖子，冰点仪进行预冷。预冷 $30\mathrm{min}$ 后，开始测量。

# 7.3 常规仪器校准

# 7.3.1 A校准

用移液器分别吸取 $2.20 \mathrm{~mL}$ 校准液A（5.3.1），依次放入三个样品管中，在启动后的冷阱中插入装有校准液A（5.3.1）的样品管。当重复测量值在 $-0.400^{\circ} \mathrm{C} \pm 0.0020^{\circ} \mathrm{C}$ 校准值时，完成校准。

# 7.3.2 B校准

用移液器分别吸取 $2.20 \mathrm{~mL}$ 校准液 B（5.3.2），依次放入三个样品管中，在启动后的冷阱中插入装有校准液 B（5.3.2）的样品管。当重复测量值在 $-0.557^{\circ} \mathrm{C} \pm 0.0020^{\circ} \mathrm{C}$ 校准值时，完成校准。

# 7.4 样品测定

将样品 $2.20 \mathrm{~mL}$ 转移到一个干燥清洁的样品管中，将待测样品管放到仪器上的测量孔中。冰点仪的显示器显示当前样品温度，温度呈下降趋势，测试样品达到 $-3.0^{\circ} \mathrm{C}$ 时启动引晶的机械振动，搅拌金属棒开始振动引晶，温度上升，当温度不再发生变化时，冰点仪停止测量，传感头升起，显示温度即为样品冰点值。

测试结束后，应保证探头和搅拌金属棒清洁、干燥，必要时，可用柔软洁净的纱布仔细擦拭。

如果引晶在达到 $-3.0^{\circ} \mathrm{C}$ 之前发生，则该测定作废，需重新取样。测定结束后，移走样品管，并用水冲洗温度传感器和搅拌金属棒并擦拭干净。

每一样品至少进行两次平行测定，绝对差值 $\leqslant 4\mathrm{m}^{\circ}\mathrm{C}$ 时，可取平均值作为结果。

# 8 分析结果的表述

如果常规校准检查的结果证实仪器校准的有效性，则取两次测定结果的平均值，保留三位有效数字。

# 9 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不超过 $4 \mathrm{~m}^{\circ} \mathrm{C}$ 。

# 10 其他

本标准的方法检出限为 $2\mathrm{m}^{\circ}\mathrm{C}$ 。

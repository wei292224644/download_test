# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2322—2009《进出口食品中乙草胺残留量检测方法》。与SN/T2322—2009相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“进出口食品”改为“食品”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T2322—2009。

# 食品安全国家标准

# 食品中乙草胺残留量的检测方法

# 1 范围

本标准规定了食品中乙草胺残留量的气相色谱测定、气相色谱一质谱法确证和制样方法。

本标准适用于花生、大豆、玉米、小麦、洋葱、海菜、鸡肉、猪肉、腰果和松茸中乙草胺残留量的测定，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

试样中的乙草胺残留物用乙腈或乙酸乙酯提取，经凝胶渗透色谱仪(GPC)和(或)硅胶固相萃取柱(SPE)净化，用气相色谱法(GC)测定和气相色谱一质谱法(GC-MS)确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 正己烷 $(\mathrm{C_6H_{14}})$ 。  
4.1.2 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ 。  
4.1.3 环己烷 $(\mathrm{C_6H_{12}})$ 。  
4.1.4 乙腈 $(\mathrm{C_2H_3N})$ 。  
4.1.5 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ 。  
4.1.6 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，储于密封容器中备用。

# 4.2 溶液配制

4.2.1 乙酸乙酯—正己烷(3+97，体积比)：取 $30~\mathrm{mL}$ 乙酸乙酯，加入 $970~\mathrm{mL}$ 正己烷，摇匀备用。  
4.2.2 乙酸乙酯—正己烷(5+95，体积比)：取 $50~\mathrm{mL}$ 乙酸乙酯，加入 $950~\mathrm{mL}$ 正己烷，摇匀备用。  
4.2.3 乙酸乙酯一环己烷 $(50 + 50$ ，体积比)：取 $500~\mathrm{mL}$ 乙酸乙酯，加入 $500~\mathrm{mL}$ 正己烷，摇匀备用。

# 4.3 标准品

4.3.1 乙草胺标准品(acetochlor, $\mathrm{C_{14}H_{20}ClNO_2}$ ,CAS号：34256-82-1)：纯度 $\geqslant 98\%$

# 4.4 标准溶液配制

4.4.1 乙草胺标准溶液：称取适量的乙草胺标准品，用丙酮配成浓度为 $100\mu \mathrm{g / mL}$ 的标准储备液。根据需要将标准储备溶液用正己烷稀释成适当浓度的标准工作液。标准储备液和标准工作液在 $0^{\circ}C\sim$ $4^{\circ}C$ 条件下避光储存。有效期6个月。

# 5 仪器与设备

5.1 气相色谱仪：配微池电子捕获检测器 $(\mu -\mathrm{ECD})$   
5.2 气相色谱—质谱仪：配电子轰击源（EI）。  
5.3 凝胶渗透色谱仪。  
5.4 旋转蒸发仪。  
5.5 离心机： $5000\mathrm{r / min}$   
5.6 高速均质器。  
5.7 硅胶固相萃取柱： $6\mathrm{mL} / 1000\mathrm{mg}$ ，或相当者；使用前用 $10\mathrm{mL}$ 正己烷淋洗。  
5.8 离心管： $100~\mathrm{mL}$ ，塑料和玻璃。  
5.9 滤膜： $0.45\mu \mathrm{m}$ ，有机相。  
5.10 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按GB2763的规定执行。

# 6.2 蔬菜类、食用菌类

6.2.1 取洋葱等有代表性辛辣蔬菜样品 $500 \mathrm{~g}$ , 将其切成小块。装入洁净的容器内, 密闭并标明标记。 $-18^{\circ} \mathrm{C}$ 以下保存。  
6.2.2 取有代表性蔬菜样品(洋葱等辛辣蔬菜除外)和松茸等有代表性食用菌样品 $500 \mathrm{~g}$ , 将其用捣碎机加工成浆状。混匀, 装入洁净的容器内, 密闭并标明标记。 $-18^{\circ} \mathrm{C}$ 以下保存。

# 6.3 海菜

取有代表性海菜样品 $500 \mathrm{~g}$ , 切碎后, 用捣碎机加工成浆状。混匀, 装入洁净的容器内, 密闭并标明标记。 $-18^{\circ} \mathrm{C}$ 以下保存。

# 6.4 粮谷和坚果类

取花生、玉米、小麦、大豆、腰果等有代表性样品 $500\mathrm{g}$ , 用粉碎机粉碎, 混匀, 装入洁净的容器内, 密闭并标明标记。 $4^{\circ}\mathrm{C}$ 以下保存。

# 6.5 动物源类

取鸡肉、猪肉等有代表性样品 $500\mathrm{g}$ ，用绞肉机绞碎，混匀，装入洁净的容器内，密闭并标明标记。 $-18^{\circ}C$ 以下保存。

# 7 测定步骤

# 7.1 提取

# 7.1.1 粮谷、坚果

称取试样 $10\mathrm{g}$ (精确至 $0.1\mathrm{g}$ ) 于 $100\mathrm{mL}$ 塑料离心管中, 加入 $50\mathrm{mL}$ 乙腈, 在匀浆器上匀浆 $2\mathrm{min}$ 。以 $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ , 将上清液倒入浓缩瓶中, 浓缩至干, 用 $10\mathrm{mL}$ 乙酸乙酯一环己烷溶液溶解残渣, 过膜, 供凝胶渗透色谱(GPC)净化。

# 7.1.2 禽肉组织

称取试样 $10\mathrm{g}$ (精确到 $0.1\mathrm{g}$ ) 于 $100\mathrm{mL}$ 塑料离心管中，加入 $5\mathrm{g}$ 无水硫酸钠，搅拌均匀，加入 $50\mathrm{mL}$ 乙腈，在匀浆器上匀浆 $2\mathrm{min}$ 。以 $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，将上清液用无水硫酸钠脱水后倒入浓缩瓶中，浓缩至干，用 $10\mathrm{mL}$ 乙酸乙酯一环己烷溶液溶解残渣，过膜，供凝胶渗透色谱(GPC)净化。

# 7.1.3 洋葱

称取试样 $5\mathrm{g}$ (精确至 $0.1\mathrm{g}$ ) 于 $100\mathrm{mL}$ 玻璃离心管中，在 $700\mathrm{W}$ 消解功率下微波消解 $40\mathrm{s}$ 。恢复至室温，加入 $50\mathrm{mL}$ 乙酸乙酯，在匀浆器上匀浆 $2\mathrm{min}$ 。以 $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，将上清液倒入浓缩瓶中，浓缩至干，用 $1\mathrm{mL}$ 正己烷溶解残渣，供固相萃取柱净化。

# 7.1.4 海菜、松茸

称取试样 $5\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）于 $100~\mathrm{mL}$ 玻璃离心管中，加入 $50~\mathrm{mL}$ 乙酸乙酯，在匀浆器上匀浆 $2\mathrm{min}$ 。以 $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，将上清液倒入浓缩瓶中，浓缩至干，用 $1\mathrm{mL}$ 正己烷溶解残渣，供固相萃取柱净化。

# 7.2 净化

# 7.2.1 凝胶渗透色谱净化条件

a）色谱柱： $\mathrm{S - X_320cm\times 2cm}$ （内径），或相当者；  
b）流动相：环己烷—乙酸乙酯 $(1 + 1$ ，体积比)；  
c) 流速: $5.0 \mathrm{~mL} / \mathrm{min}$ ;   
d) 进样量: $5.0 \mathrm{~mL}$ ;   
e) 淋洗体积: $40 \mathrm{~mL}$ ;   
f）收集体积： $30~\mathrm{mL}$   
g) 清洗体积: $10 \mathrm{~mL}$ 。

# 7.2.2 凝胶渗透色谱净化

将7.1中粮谷、坚果和禽肉组织的提取液过滤膜注入凝胶渗透色谱仪中，收集 $40\mathrm{mL}\sim 70\mathrm{mL}$ 流出液，在 $40^{\circ}\mathrm{C}$ 水浴上减压浓缩至干，用 $1\mathrm{mL}$ 正己烷溶解残渣，供固相萃取柱净化。

# 7.2.3 硅胶固相萃取柱净化

将7.1中洋葱、海菜、松茸提取液和7.2.2中净化液注入硅胶固相萃取柱中，再分2次，每次 $1\mathrm{mL}$ 正己烷润洗盛样液容器，注入硅胶柱中，依次用 $10\mathrm{mL}$ 正己烷、 $5\mathrm{mL}$ 乙酸乙酯一正己烷溶液和 $5\mathrm{mL}$ 乙酸乙酯一正己烷溶液淋洗，再用 $15\mathrm{mL}$ 乙酸乙酯一正己烷溶液洗脱，收集洗脱液于 $40^{\circ}\mathrm{C}$ 水浴减压浓缩至干，用正己烷定容至 $5\mathrm{mL}$ ，供气相色谱测定和气相色谱一质谱确证。

# 7.3 测定

# 7.3.1 气相色谱参考条件

a）色谱柱：DB-5石英毛细管柱， $30\mathrm{m}\times 0.32\mathrm{mm}$ （内径），膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度： $60^{\circ}\mathrm{C}(1\mathrm{min})\xrightarrow{15^{\circ}\mathrm{C / min}}260^{\circ}\mathrm{C}(5\mathrm{min}),300^{\circ}\mathrm{C}$ （后运行 $15\mathrm{min}$ ）；  
c）进样口温度： $250^{\circ}\mathrm{C}$   
d) 检测器温度: $325^{\circ} \mathrm{C}$ ;   
e) 载气：氦气，纯度 $\geqslant 99.995\%$ ， $1.8~\mathrm{mL / min}$   
f）进样量： $1\mu \mathrm{L}$   
g）进样方式：无分流进样， $0.75\mathrm{min}$ 后开阀。

# 7.3.2 气相色谱一质谱参考条件

a）色谱柱：DB-5MS石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度： $60^{\circ}\mathrm{C}(1\mathrm{min})\xrightarrow{15^{\circ}\mathrm{C}/\mathrm{min}}260^{\circ}\mathrm{C}(5\mathrm{min}),300^{\circ}\mathrm{C}$ （后运行 $15\mathrm{min}$ ）；  
c）进样口温度： $250^{\circ}\mathrm{C}$   
d） 色谱一质谱接口温度： $280^{\circ}\mathrm{C}$   
e) 载气：氦气，纯度 $\geqslant 99.995\%$ ，1.0 mL/min;  
f) 进样量: $1 \mu \mathrm{L}$ ;

g）进样方式：无分流进样， $0.75\mathrm{min}$ 后开阀；  
h） 电离方式：EI；  
i）电离能量： $70\mathrm{eV}$   
j）测定方式：选择离子监测方式(SIM)；  
k）监测离子 $(m / z)$ ：定量162，定性174、234、269；  
1) 溶剂延迟： $3\mathrm{min}$

# 7.3.3 气相色谱测定

根据样液中乙草胺含量情况，选定浓度相近的标准工作溶液。标准工作溶液和样液中乙草胺响应值均应在仪器检测线性范围内。标准工作溶液和样液等体积穿插进样测定。在上述色谱条件下，乙草胺的保留时间约为 $12.3\mathrm{min}$ 。标准品的色谱图参见图A.1。

# 7.3.4 气相色谱一质谱确证

乙草胺标准溶液及样液均按7.3.2规定的条件进行测定，如果样液与标准溶液相同的保留时间有峰出现，则对其进行质谱确证。经确证分析被测物SIM色谱峰保留时间与标准样品一致，并且在扣除背景后的样品谱图中，所选择的离子均出现；选择离子 $m / z162,174,234,269$ （其丰度比为 $100:54:$ $24:8)$ 与标准样品相关离子的相对丰度一致，相似度在允许偏差之内（见表1），则可判定样品中存在乙草胺。乙草胺标准品总离子流色谱图、选择离子质谱图和全扫描质谱图参见图A.2、图A.3和图A.4。

# 7.4 空白试验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中乙草胺的含量，计算结果需将空白值扣除。

$$
X = \frac {A \times c \times V}{A _ {S} \times m}. \tag {1}
$$

式中：

$X$ ——试样中乙草胺的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——标准工作液中乙草胺的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$A$ ——样品溶液中乙草胺的峰面积；

$A_{S}$ ——标准工作溶液中乙草胺的峰面积；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——最终样液所代表的试样量，单位为克 $(\mathrm{g})$ 。

计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录B的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录 C 的要求。

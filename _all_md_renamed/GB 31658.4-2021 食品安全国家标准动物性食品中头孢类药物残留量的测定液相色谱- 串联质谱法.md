# 动物性食品中头孢类药物残留量的测定

# 液相色谱-串联质谱法

# 1 范围

本文件规定了猪、牛肌肉、肝脏、肾脏和脂肪以及牛奶中头孢类药物残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、牛肌肉、肝脏、肾脏和脂肪以及牛奶中头孢类药物（头孢氨苄、头孢拉定、头孢唑林、头孢哌酮、头孢乙腈、头孢匹林、头孢洛宁、头孢喹肟、头孢噻肟)残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的头孢类药物，经乙腈水、磷酸盐缓冲液提取，固相萃取柱净化，液相色谱-串联质谱法测定，外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水

# 5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.3 甲酸（HCOOH）：色谱纯。

5.1.4 磷酸二氢钾 $(\mathrm{KH}_2\mathrm{PO}_4)$ 。

5.1.5 氢氧化钠 $\mathrm{(NaOH)}$

5.1.6 氯化钠 $(\mathrm{NaCl})$ 。

# 5.2 溶液配制

5.2.1 氢氧化钠溶液：取氢氧化钠 $50\mathrm{g}$ ，加水溶解并稀释至 $500~\mathrm{mL}$ 。

5.2.2 乙腈-水溶液 $(15:2,V / V)$

5.2.3 $30\%$ 乙腈溶液：取乙腈 $30~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$

5.2.4 $0.05\mathrm{mol} / \mathrm{L}$ 磷酸盐缓冲液 $(\mathrm{pH} = 8.5)$ ：取磷酸二氢钾 $6.8\mathrm{g}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ ，用氢氧化钠溶液调节 $\mathrm{pH}$ 至 $8.5\pm 0.1$ 。

5.2.5 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。

# 5.3 标准品

头孢氨苄、头孢拉定、头孢唑林、头孢哌酮、头孢乙腈、头孢匹林、去乙酰基头孢匹林，头孢洛宁、头孢唑肟、头孢噻肟标准品，含量均 $\geq 95.0\%$ ，见附录A。

# 5.4 标准溶液的制备

5.4.1 混合标准储备液：取头孢氨苄、头孢拉定、头孢唑林、头孢哌酮、头孢乙腈、头孢匹林、去乙酰基头孢匹林、头孢洛宁、头孢喹肟、头孢噻肟标准品各 $10\mathrm{mg}$ ，精密称定，用 $30\%$ 乙腈溶液溶解并稀释定容至 $25\mathrm{mL}$ 容量瓶，配制成浓度为 $400~\mu \mathrm{g / mL}$ 的溶液。 $-18^{\circ}C$ 避光保存，有效期1个月。  
5.4.2 混合标准工作液：精密量取混合标准储备液适量，分别于 $10~\mathrm{mL}$ 容量瓶，用水稀释至刻度。配制成浓度均为 $0.05\mu \mathrm{g / mL}, 0.50\mu \mathrm{g / mL}, 2.0\mu \mathrm{g / mL}$ 和 $20\mu \mathrm{g / mL}$ 的混合标准工作液。 $4^{\circ}\mathrm{C}$ 避光保存，有效期 $7\mathrm{d}$ 。  
5.5 材料  
5.5.1 HLB固相萃取柱， $500\mathrm{mg / 6mL}$ ，或相当者。  
5.5.2 滤膜：尼龙材质，孔径 $0.22\mu \mathrm{m}$ ，或性能相当者。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 旋转蒸发仪。  
6.4 固相萃取装置。  
6.5 均质器。  
6.6 涡旋混合器。  
6.7 聚丙烯塑料离心管： $10\mathrm{mL}, 50\mathrm{mL}$ 。  
6.8 离心机： $5000\mathrm{r / min}$ 或以上。  
6.9 鸡心瓶： $50~\mathrm{mL}$

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存，3个月内进行分析检测。

# 8 测定步骤

# 8.1 提取

取试料 $2\mathrm{g}$ （准确到 $\pm 0.02\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管，加乙腈-水溶液 $10\mathrm{mL}$ 或乙腈 $10\mathrm{mL}$ （仅适用于牛奶）， $10000\mathrm{r / min}$ 匀质 $1\mathrm{min},5000\mathrm{r / min}$ 离心 $2\mathrm{min}$ ，收集上清液，于鸡心瓶中。残渣用上述溶液 $10\mathrm{mL}$ 重复提取1次，合并提取液， $40^{\circ}C$ 水浴中旋转蒸发，除去乙腈（如有泡沫，加饱和氯化钠水溶液 $4\mathrm{mL}$ )，立即加磷酸盐缓冲溶液 $25~\mathrm{mL}$ ，备用。

# 8.2 净化

取固相萃取柱，用甲醇 $5\mathrm{mL}$ 磷酸盐缓冲液 $10~\mathrm{mL}$ 活化。备用液过柱，待液面到达柱床表面时用磷酸盐缓冲溶液 $3\mathrm{mL}$ 淋洗，用水 $2\mathrm{mL}$ 淋洗，乙腈 $3\mathrm{mL}$ 洗脱于 $10~\mathrm{mL}$ 离心管中， $40^{\circ}\mathrm{C}$ 水浴氮气吹干，加水 $1.0~\mathrm{mL}$ 溶解，过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱测定。

# 8.3 基质匹配标准曲线的制备

精密量取混合标准工作液适量，用经提取净化后的空白样品溶液稀释配制浓度为 $5\mu \mathrm{g} / \mathrm{L},10\mu \mathrm{g} / \mathrm{L}$ $20~\mu \mathrm{g} / \mathrm{L},50~\mu \mathrm{g} / \mathrm{L},100~\mu \mathrm{g} / \mathrm{L}$ 和 $200~\mu \mathrm{g} / \mathrm{L}$ 的系列基质校准工作溶液，过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱测定。以定量离子对峰面积为纵坐标、标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

c) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ ;   
d） 柱温： $35^{\circ}C$   
e) 进样量: $10 \mu \mathrm{L}$ 。

# 8.4.3 测定法

取试料溶液和基质匹配标准溶液，作单点或多点校准，按外标法以色谱峰面积定量。基质匹配标准溶液及试料溶液中目标药物的特征离子质量色谱峰峰面积均应在仪器检测的线性范围之内。试料液中头孢类药物的保留时间与基质匹配标准工作液中头孢类药物的保留时间之比，偏差在 $\pm 2.5\%$ 以内，且试料溶液中的相对离子丰度与基质匹配标准溶液中的相对离子丰度相比，符合表3的要求。基质匹配标准溶液多反应监测色谱图见附录B。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V \times 1 0 0 0}{A _ {\mathrm {S}} \times m \times 1 0 0 0} \dots \tag {1}
$$

式中：

$X$ ——试料中待测药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ ——标准溶液中待测药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$A$ ——试料溶液中待测药物的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中待测药物的峰面积；

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法在猪、牛的肌肉、肝脏、肾脏和脂肪及牛奶中的检测限为 $2.0\mu \mathrm{g / kg}$ ，定量限为 $5.0\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法在 $5.0\mu \mathrm{g / kg}\sim 2000\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 110\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

# 前言

本标准代替GB/T5009.91—2003《食品中钾、钠的测定》、GB/T15402—1994《水果、蔬菜及其制品钠、钾含量的测定》、NY82.18—1988《果汁测定方法钾和钠的测定》，以及GB5413.21—2010《食品安全国家标准婴幼儿食品和乳品中钙、铁、锌、钠、钾、镁、铜和锰的测定》、GB/T18932.11—2002《蜂蜜中钾、磷、铁、钙、锌、铝、钠、镁、硼、锰、铜、钡、钛、钒、镍、钴、铬含量的测定方法电感耦合等离子体原子发射光谱(ICP-AES)法》、GB/T18932.12—2002《蜂蜜中钾、钠、钙、镁、锌、铁、铜、锰、铬、铅、镉含量的测定方法原子吸收光谱法》和NY/T1653—2008《蔬菜、水果及制品中矿质元素的测定电感耦合等离子体发射光谱法》中有关钾、钠的检测部分。

本标准与以上标准相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中钾、钠的测定”；  
——修改了火焰原子吸收光谱法作为第一法，火焰原子发射光谱法作为第二法，电感耦合等离子体发射光谱法作为第三法，电感耦合等离子体质谱法作为第四法；  
——修改了试样制备部分内容；  
——修改了试样消解部分内容；  
——增加了方法检出限及定量限；  
——增加了附录。

# 食品安全国家标准

# 食品中钾、钠的测定

# 1 范围

本标准规定了食品中钾、钠的火焰原子吸收光谱法、火焰原子发射光谱法、电感耦合等离子体发射光谱法和电感耦合等离子体质谱法四种测定方法。

本标准适用于食品中钾、钠的测定。

# 第一法 火焰原子吸收光谱法

# 2 原理

试样经消解处理后，注入原子吸收光谱仪中，火焰原子化后钾、钠分别吸收 $766.5\mathrm{nm}, 589.0\mathrm{nm}$ 共振线，在一定浓度范围内，其吸收值与钾、钠含量成正比，与标准系列比较定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 硝酸 $\mathrm{(HNO_3)}$ 。  
3.1.2 高氯酸 $\mathrm{(HClO_4)}$   
3.1.3 氯化铯 $(\mathrm{CsCl})$ 。

# 3.2 试剂配制

3.2.1 混合酸[高氯酸 + 硝酸 $(1+9)]$ : 取 $100 \mathrm{~mL}$ 高氯酸, 缓慢加入 $900 \mathrm{~mL}$ 硝酸中, 混匀。  
3.2.2 硝酸溶液 $(1 + 99)$ ：取 $10\mathrm{mL}$ 硝酸，缓慢加入 $990~\mathrm{mL}$ 水中，混匀。  
3.2.3 氯化铯溶液 $(50\mathrm{g / L})$ ：将 $5.0\mathrm{g}$ 氯化铯溶于水，用水稀释至 $100~\mathrm{mL}$ 。

# 3.3 标准品

3.3.1 氯化钾标准品(KCl)：纯度大于 $99.99\%$   
3.3.2 氯化钠标准品 $(\mathrm{NaCl})$ ：纯度大于 $99.99\%$

# 3.4 标准溶液配制

3.4.1 钾、钠标准储备液 $(1000\mathrm{mg / L})$ : 将氯化钾或氯化钠于烘箱中 $110^{\circ}\mathrm{C} \sim 120^{\circ}\mathrm{C}$ 干燥 $2\mathrm{h}$ 。精确称取 $1.9068\mathrm{g}$ 氯化钾或 $2.5421\mathrm{g}$ 氯化钠, 分别溶于水中, 并移入 $1000\mathrm{mL}$ 容量瓶中, 稀释至刻度, 混匀,

贮存于聚乙烯瓶内， $4\,^{\circ}\mathrm{C}$ 保存，或使用经国家认证并授予标准物质证书的标准溶液。

3.4.2 钾、钠标准工作液 $(100\mathrm{mg / L})$ ：准确吸取 $10.0~\mathrm{mL}$ 钾或钠标准储备溶液于 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，贮存于聚乙烯瓶中， $4^{\circ}\mathrm{C}$ 保存。  
3.4.3 钾、钠标准系列工作液：准确吸取 $0\mathrm{mL}, 0.1\mathrm{mL}, 0.5\mathrm{mL}, 1.0\mathrm{mL}, 2.0\mathrm{mL}, 4.0\mathrm{mL}$ 钾标准工作液于 $100\mathrm{mL}$ 容量瓶中，加氯化铯溶液 $4\mathrm{mL}$ ，用水定容至刻度，混匀。此标准系列工作液中钾质量浓度分别为 $0\mathrm{mg/L}, 0.100\mathrm{mg/L}, 0.500\mathrm{mg/L}, 1.00\mathrm{mg/L}, 2.00\mathrm{mg/L}, 4.00\mathrm{mg/L}$ ，亦可依据实际样品溶液中钾浓度，适当调整标准溶液浓度范围。准确吸取 $0\mathrm{mL}, 0.5\mathrm{mL}, 1.0\mathrm{mL}, 2.0\mathrm{mL}, 3.0\mathrm{mL}, 4.0\mathrm{mL}$ 钠标准工作液于 $100\mathrm{mL}$ 容量瓶中，加氯化铯溶液 $4\mathrm{mL}$ ，用水定容至刻度，混匀。此标准系列工作液中钠质量浓度分别为 $0\mathrm{mg/L}, 0.500\mathrm{mg/L}, 1.00\mathrm{mg/L}, 2.00\mathrm{mg/L}, 3.00\mathrm{mg/L}, 4.00\mathrm{mg/L}$ ，亦可依据实际样品溶液中钠浓度，适当调整标准溶液浓度范围。

# 4 仪器和设备

4.1 原子吸收光谱仪，配有火焰原子化器及钾、钠空心阴极灯。  
4.2 分析天平：感量为 $0.1\mathrm{mg}$ 和 $1.0\mathrm{mg}$   
4.3 分析用钢瓶乙炔气和空气压缩机。  
4.4 样品粉碎设备：匀浆机、高速粉碎机。  
4.5 马弗炉。  
4.6 可调式控温电热板。  
4.7 可调式控温电热炉。  
4.8 微波消解仪，配有聚四氟乙烯消解内罐。  
4.9 恒温干燥箱。  
4.10 压力消解罐，配有聚四氟乙烯消解内罐。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固态样品

# 5.1.1.1 干样

豆类、谷物、菌类、茶叶、干制水果、焙烤食品等低含水量样品，取可食部分，必要时经高速粉碎机粉碎均匀；对于固体乳制品、蛋白粉、面粉等呈均匀状的粉状样品，摇匀。

# 5.1.1.2 鲜样

蔬菜、水果、水产品等高含水量样品必要时洗净，晾干，取可食部分匀浆均匀；对于肉类、蛋类等样品取可食部分匀浆均匀。

# 5.1.1.3 速冻及罐头食品

经解冻的速冻食品及罐头样品，取可食部分匀浆均匀。

# 5.1.2 液态样品

软饮料、调味品等样品摇匀。

# 5.1.3 半固态样品

搅拌均匀。

# 5.2 试样消解

# 5.2.1 微波消解法

称取 $0.2\mathrm{g}\sim 0.5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 试样于微波消解内罐中，含乙醇或二氧化碳的样品先在电热板上低温加热除去乙醇或二氧化碳，加入 $5\mathrm{mL}\sim 10\mathrm{mL}$ 硝酸，加盖放置 $1\textrm{h}$ 或过夜，旋紧外罐，置于微波消解仪中进行消解（消解条件参见表A.1）。冷却后取出内罐，置于可调式控温电热炉上，于 $120~\mathrm{^\circ C}\sim$ $140^{\circ}C$ 赶酸至近干，用水定容至 $25~\mathrm{mL}$ 或 $50~\mathrm{mL}$ ，混匀备用。同时做空白试验。

# 5.2.2 压力罐消解法

称取 $0.3\mathrm{g}\sim 1\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 试样于聚四氟乙烯压力消解内罐中，含乙醇或二氧化碳的样品先在电热板上低温加热除去乙醇或二氧化碳，加入 $5\mathrm{mL}$ 硝酸，加盖放置 $1\textrm{h}$ 或过夜，旋紧外罐，置于恒温干燥箱中进行消解（消解条件参见表A.1）。冷却后取出内罐，置于可调式控温电热板上，于 $120~\mathrm{^\circ C}\sim$ $140~\mathrm{^\circ C}$ 赶酸至近干，用水定容至 $25~\mathrm{mL}$ 或 $50~\mathrm{mL}$ ，混匀备用。同时做空白试验。

# 5.2.3 湿式消解法

称取 $0.5\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )试样于玻璃或聚四氟乙烯消解器皿中，含乙醇或二氧化碳的样品先在电热板上低温加热除去乙醇或二氧化碳，加入 $10~\mathrm{mL}$ 混合酸，加盖放置 $1\textrm{h}$ 或过夜，置于可调式控温电热板或电热炉上消解，若变棕黑色，冷却后再加混合酸，直至冒白烟，消化液呈无色透明或略带黄色，冷却，用水定容至 $25~\mathrm{mL}$ 或 $50~\mathrm{mL}$ ，混匀备用。同时做空白试验。

# 5.2.4 干式消解法

称取 $0.5\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )试样于坩埚中，在电炉上微火炭化至无烟，置于 $525~^\circ \mathrm{C}\pm 25~^\circ \mathrm{C}$ 马弗炉中灰化 $5\mathrm{h}\sim 8\mathrm{h}$ ，冷却。若灰化不彻底有黑色炭粒，则冷却后滴加少许硝酸湿润，在电热板上干燥后，移入马弗炉中继续灰化成白色灰烬，冷却至室温取出，用硝酸溶液溶解，并用水定容至 $25~\mathrm{mL}$ 或 $50~\mathrm{mL}$ ，混匀备用。同时做空白试验。

# 5.3 仪器参考条件

优化仪器至最佳状态，仪器的主要条件参见表A.2。

# 5.4 标准曲线的制作

分别将钾、钠标准系列工作液注入原子吸收光谱仪中，测定吸光度值，以标准工作液的浓度为横坐标，吸光度值为纵坐标，绘制标准曲线。

# 5.5 试样溶液的测定

根据试样溶液中被测元素的含量，需要时将试样溶液用水稀释至适当浓度，并在空白溶液和试样最终测定液中加入一定量的氯化铯溶液，使氯化铯浓度达到 $0.2\%$ 。于测定标准曲线工作液相同的实验条件下，将空白溶液和测定液注入原子吸收光谱仪中，分别测定钾或钠的吸光值，根据标准曲线得到待测液中钾或钠的浓度。

# 6 分析结果的表述

试样中钾、钠含量按式(1)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times f \times 1 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中被测元素含量，单位为毫克每百克或毫克每百毫升 $(\mathrm{mg} / 100\mathrm{g}$ 或 $\mathrm{mg} / 100\mathrm{mL})$

$\rho$ ——测定液中元素的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——测定空白试液中元素的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——样液体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——样液稀释倍数；

100、1000——换算系数；

$m$ ——试样的质量或体积，单位为克或毫升(g或 $\mathrm{mL}$ )。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

以取样量 $0.5\mathrm{g}$ ，定容至 $25~\mathrm{mL}$ 计，本方法钾的检出限为 $0.2\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.5\mathrm{mg} / 100\mathrm{g}$ ；钠的检出限为 $0.8\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $3\mathrm{mg} / 100\mathrm{g}$ 。

# 第二法 火焰原子发射光谱法

# 9 原理

试样经消解处理后，注入火焰光度计或原子吸收光谱仪中，火焰原子化后分别测定钾、钠的发射强度。钾发射波长为 $766.5\mathrm{nm}$ ，钠发射波长为 $589.0\mathrm{nm}$ ，在一定浓度范围内，其发射值与钾、钠含量成正比，与标准系列比较定量。

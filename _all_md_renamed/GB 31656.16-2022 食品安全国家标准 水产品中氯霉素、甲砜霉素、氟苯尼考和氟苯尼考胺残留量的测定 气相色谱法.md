# 食品安全国家标准

# 水产品中氯霉素、甲砜霉素、氟苯尼考和氟苯尼考胺残留量的测定气相色谱法

# 1 范围

本文件规定了水产品中氯霉素、甲砜霉素、氟苯尼考和氟苯尼考胺残留量测定的制样和气相色谱测定方法。

本文件适用于鱼、虾、鳖、贝、海参等水产品可食组织中氯霉素、甲砜霉素、氟苯尼考及氟苯尼考胺残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

GB/T30891—2014 水产品抽样规范

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的氯霉素、甲砜霉素、氟苯尼考和氟苯尼考胺,在碱性条件下用乙酸乙酯提取,正己烷去脂,固相萃取柱净化,硅烷化试剂衍生化,气相色谱法检测。外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
5.1.4 乙酸 $\mathrm{(CH_3COOH)}$ ：色谱纯。  
5.1.5 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ ：色谱纯。  
5.1.6 甲苯 $(\mathrm{C}_7\mathrm{H}_8)$ ：色谱纯。  
5.1.7 浓氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。  
5.1.8 N，O-双(三甲基硅烷基)三氟乙酰胺(BSTFA)： $\mathrm{C_8H_{18}F_3NOSi_2}$   
5.1.9 三甲基氯硅烷（TMCS）： $\mathrm{C_3H_9ClSi}$

# 5.2 溶液配制

5.2.1 乙酸乙酯-氨水：取浓氨水 $2\mathrm{mL}$ ，用乙酸乙酯稀释至 $100~\mathrm{mL}$ 。现用现配。  
5.2.2 甲醇-氨水：取浓氨水 $2\mathrm{mL}$ ，用甲醇稀释至 $100~\mathrm{mL}$ 。现用现配。  
5.2.3 $5\%$ 乙酸溶液：取乙酸 $5\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$

5.2.4 衍生化试剂：取N，O-双(三甲基硅烷基)三氟乙酰胺(BSTFA)与三甲基氯硅烷（TMCS)，按体积比 $99:1$ 混合。现用现配。

# 5.3 标准品

氯霉素、甲砜霉素、氟苯尼考、氟苯尼考胺，含量均 $\geq 98\%$ 。具体见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液：取氯霉素、甲砜霉素、氟苯尼考、氟苯尼考胺标准品各 $15\mathrm{mg}$ ，精密称定，分别加乙腈适量使溶解并稀释定容至 $100\mathrm{mL}$ 棕色容量瓶，配制成浓度均为 $150\mu \mathrm{g} / \mathrm{mL}$ 的标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期3个月。  
5.4.2 混合标准工作液：分别准确量取氯霉素标准储备液 $1\mathrm{mL}$ ，甲砜霉素、氟苯尼考和氟苯尼考胺各 $10\mathrm{mL}$ 于 $50\mathrm{mL}$ 容量瓶，用乙腈稀释至刻度，制成氯霉素浓度为 $3\mu \mathrm{g} / \mathrm{mL}$ ，甲砜霉素、氟苯尼考和氟苯尼考胺浓度均为 $30\mu \mathrm{g} / \mathrm{mL}$ 的混合标准工作液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期1个月。

# 5.5 材料

5.5.1 MCX固相萃取柱： $60~\mathrm{mg / 3mL}$ ，或相当者。  
5.5.2 移液器： $20\mu \mathrm{L}\sim 200\mu \mathrm{L},200\mu \mathrm{L}\sim 1000\mu \mathrm{L}_{\circ}$   
5.5.3 鸡心瓶： $100~\mathrm{mL}$   
5.5.4 聚乙烯离心管： $15\mathrm{mL},50\mathrm{mL}$ ，具塞。  
5.5.5 玻璃离心管： $10~\mathrm{mL}$ ，具塞。  
5.5.6 棕色反应瓶：规格 $2\mathrm{mL}$

# 6 仪器和设备

6.1 气相色谱仪，配 $^{63}\mathrm{Ni}$ 电子捕获检测器。  
6.2 均质机。  
6.3 旋转蒸发仪。  
6.4 离心机。  
6.5 涡旋混合器。  
6.6 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.7 氮吹仪。  
6.8 超声波清洗器。  
6.9 固相萃取装置。

# 7 试样的制备与保存

# 7.1 试样的制备

按GB/T30891—2014附录B的要求制样。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

-18℃以下保存。

# 8 测定步骤

# 8.1 提取

取试料 $5\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，置 $50~\mathrm{mL}$ 具塞玻璃离心管中，加乙酸乙酯-氨水 $20~\mathrm{mL}$ ，立即涡旋

1 min, 超声 $15\mathrm{min}$ , $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ , 取上清液于 $100\mathrm{mL}$ 鸡心瓶中, 重复提取 1 次, 合并上清液, 加入 $5\%$ 乙酸溶液 $3\mathrm{mL}$ , 振荡混匀, $40^{\circ}\mathrm{C}$ 旋转蒸发至约 $1.5\mathrm{mL}$ , 备用。

# 8.2 净化

取备用液于 $10~\mathrm{mL}$ 离心管，用 $5\%$ 乙酸 $1.5\mathrm{mL}$ 洗涤鸡心瓶，洗液并于离心管。加正己烷 $5\mathrm{mL}$ ，涡旋混合 $1\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取水层，用正己烷 $5\mathrm{mL}$ 重复脱脂1次，取水层，备用。

取MCX固相萃取柱，依次用甲醇 $3\mathrm{mL}$ 、水 $3\mathrm{mL}$ 活化，取经正己烷脱脂后的备用液，过柱，用 $5\%$ 乙酸溶液 $2\mathrm{mL}$ 淋洗，弃去淋洗液，用甲醇-氨水 $6\mathrm{mL}$ 洗脱，收集洗脱液， $65^{\circ}C$ 氮气吹至近干，再依次用甲醇 $1\mathrm{mL}$ 、乙酸乙酯 $1\mathrm{mL}$ 洗涤离心管壁，氮气吹至近干，备用。

# 8.3 衍生

# 8.3.1 试料的衍生化

取吹干的提取物，加乙腈 $100\mu \mathrm{L}$ 复溶，移至 $2\mathrm{mL}$ 棕色反应瓶中，再用 $100\mu \mathrm{L}$ 清洗并合并至棕色反应瓶中，加衍生化试剂 $100\mu \mathrm{L}$ ，盖塞并涡旋混合 $10\mathrm{s}$ ， $60^{\circ}\mathrm{C}$ 烘箱中反应 $30\mathrm{min}$ 。冷却至室温， $50^{\circ}\mathrm{C} \sim 55^{\circ}\mathrm{C}$ 氮气吹干，准确加入甲苯 $1.0\mathrm{mL}$ ，涡旋混匀，供气相色谱仪测定。

# 8.3.2 标准工作液的衍生化

取氯霉素、甲砜霉素、氟苯尼考和氟苯尼考胺混合标准工作液适量于 $2\mathrm{mL}$ 棕色反应瓶中，置 $50^{\circ}C$ 氮吹仪吹干。以下按8.3.1的步骤操作。

# 8.4 标准曲线的制备

分别精密量取氯霉素、甲砜霉素、氟苯尼考、氟苯尼考胺标准工作液适量，用乙腈稀释氯霉素浓度为 $1.0\mu \mathrm{g} / \mathrm{L}, 1.5\mu \mathrm{g} / \mathrm{L}, 5.0\mu \mathrm{g} / \mathrm{L}, 15.0\mu \mathrm{g} / \mathrm{L}, 50.0\mu \mathrm{g} / \mathrm{L}, 150.0\mu \mathrm{g} / \mathrm{L}$ ，甲砜霉素、氟苯尼考、氟苯尼考胺浓度均为 $10.0\mu \mathrm{g} / \mathrm{L}, 15.0\mu \mathrm{g} / \mathrm{L}, 50.0\mu \mathrm{g} / \mathrm{L}, 150.0\mu \mathrm{g} / \mathrm{L}, 500.0\mu \mathrm{g} / \mathrm{L}, 1500.0\mu \mathrm{g} / \mathrm{L}$ 的系列标准溶液，经衍生化后，供气相色谱测定。以测得峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.5 测定

# 8.5.1 色谱条件

a）色谱柱：DB-5MS石英毛细管柱 $(30\mathrm{m}\times 0.25\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m})$ ，或相当者；  
b）进样方式：不分流进样；  
c）载气：氮气，纯度 $99.999\%$   
d）柱流速： $1.0\mathrm{mL / min}$ ，尾吹为 $25~\mathrm{mL / min}$   
e）进样口温度 $230^{\circ}\mathrm{C}$ ，检测器温度 $310^{\circ}\mathrm{C}$   
f）柱温程序：起始 $80^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，以 $30^{\circ}\mathrm{C / min}$ 升温至 $280^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ ；  
g）进样量： $1.0\mu \mathrm{L}$

# 8.5.2 测定法

取试料溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试料溶液中氯霉素、甲砜霉素、氟苯尼考和氟苯尼考胺响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液气相色谱图见附录B。

# 8.5.3 空白试验

取空白试样，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测物的残留量，按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {s}} \times A \times V}{A _ {\mathrm {s}} \times m} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中待测物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{s}}$ ——标准溶液中待测物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$A$ ——试样溶液中待测物色谱峰面积；

$A_{\mathrm{s}}$ ——标准溶液中待测物色谱峰面积；

$V$ ——试样溶液最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——供试试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法氯霉素的检测限为 $0.1\mu \mathrm{g / kg}$ ，甲砜霉素、氟苯尼考和氟苯尼考胺的检测限均为 $1.0~\mu \mathrm{g / kg}$ 氯霉素的定量限为 $0.3\mu \mathrm{g / kg}$ ，甲砜霉素、氟苯尼考和氟苯尼考胺的定量限均为 $3.0~\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法氯霉素在 $0.3\mu \mathrm{g / kg}\sim 30.0\mu \mathrm{g / kg}$ 添加水平上的回收率 $70\% \sim 120\%$ ；甲砜霉素、氟苯尼考和氟苯尼考胺在 $3.0\mu \mathrm{g / kg}\sim 300\mu \mathrm{g / kg}$ 添加水平上的回收率均为 $70\% \sim 120\%$ 。

# 10.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

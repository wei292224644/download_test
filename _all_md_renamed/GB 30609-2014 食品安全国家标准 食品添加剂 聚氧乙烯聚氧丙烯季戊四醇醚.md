# 食品安全国家标准

# 食品添加剂 聚氧乙烯聚氧丙烯季戊四醇醚

# 1 范围

本标准适用于以环氧丙烷、环氧乙烷、季戊四醇等多元醇为主原料，在催化剂存在下聚合而成的食品添加剂聚氧乙烯聚氧丙烯季戊四醇醚(PPE)。

# 2 结构式和平均相对分子质量

# 2.1 结构式

$\mathrm{C}[\mathrm{CH}_2\mathrm{O}(\mathrm{C}_3\mathrm{H}_6\mathrm{O})_m(\mathrm{C}_2\mathrm{H}_4\mathrm{O})_n\mathrm{H}]_4$

其中，氧化丙烯聚合度 $m$ 为 $10\sim 20$ ；氧化乙烯聚合度 $n$ 为 $0\sim 5$ 。

# 2.2 平均相对分子质量

$3000\sim 5000$ （按2007年国际相对原子质量）。

# 1 范围

本标准适用于以苯甲醛、辛醛为原料经化学反应制得的食品添加剂 $\alpha$ -己基肉桂醛。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

$\alpha$ -己基- $\beta$ -苯基丙烯醛

# 2.2 分子式

$$
\mathrm {C} _ {1 5} \mathrm {H} _ {2 0} \mathrm {O}
$$

216.32（按2007年国际相对原子质量）

# 附录A

# $\pmb{\alpha}$ -己基肉桂醛含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006或GB/T11539—2008中第5章的规定。   
A.1.2 柱：毛细管柱或填充柱。  
A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006或GB/T11539—2008中10.4测定含量。

# A.3 重复性及结果表示

按GB/T11538—2006或GB/T11539—2008中11.4规定进行。

# B.2 操作条件

B.2.1 柱：填充柱，长 $3\mathrm{m}\sim 4\mathrm{m}$ ，内径 $3\mathrm{mm}\sim 4\mathrm{mm}$   
B.2.2 固定相：聚乙二醇， $5\% \sim 10\%$ 涂于经酸洗、二甲基二氧硅烷处理的 $180~\mu \mathrm{m} \sim 250~\mu \mathrm{m}$ (60 目～80 目)白色担体上。  
B.2.3 色谱炉温度： $190~\mathrm{^\circ C}$ 恒温。  
B.2.4 进样口温度： $220^{\circ}C$ 。  
B.2.5 检测器温度： $200^{\circ}C$ 。  
B.2.6 检测器：氢火焰离子化检测器。   
B.2.7 载气：氮气。  
B.2.8 载气流速： $20~\mathrm{mL / min}\sim 30~\mathrm{mL / min}$   
B.2.9 进样量：约 $0.2\mu \mathrm{L}$

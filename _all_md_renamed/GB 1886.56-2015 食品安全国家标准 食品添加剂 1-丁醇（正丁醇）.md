# 附录A

# 检验方法

# A.1 警示

试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和防护措施。

# A.2 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682中规定的三级水。试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备；所用溶液除另有说明外，均为水溶液。

# A.3 鉴别试验

# A.3.1 溶解性

溶于水，微溶于乙醇和乙醚。

# A.3.2 密度

按GB/T4472的规定进行测定。 $20^{\circ}\mathrm{C}$ 时的密度应为 $\mathrm{0.809~g / cm^3\sim 0.811~g / cm^3}$ □

# A.4 酸度（以乙酸计）的测定

# A.4.1 试剂和材料

A.4.1.1 氢氧化钾乙醇标准滴定溶液： $c(\mathrm{KOH}) = 0.02\mathrm{mol / L}$

A.4.1.2 酚酞指示液： $10\mathrm{g / L}$

# A.4.2 分析步骤

称取约 $60\mathrm{g}$ 试样（约 $74~\mathrm{mL}$ ，精确至 $0.001\mathrm{g}$ ，加2滴 $\sim 3$ 滴酚酞指示液，用氢氧化钾乙醇标准滴定溶液滴定至溶液呈粉红色，保持 $15\mathrm{s}$ 不褪色即为终点。

# A.4.3 结果计算

酸度（以乙酸计）的质量分数 $\omega_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {V _ {1} \times c _ {1} \times M}{1000 \times m} \times 100 \% \tag{A.1}
$$

式中：

$V_{1}$ ——氢氧化钾乙醇标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——氢氧化钾乙醇标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——乙酸的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(\mathrm{CH}_3\mathrm{COOH}) = 60.05]$

1000——换算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

取两次平行测定结果的算术平均值为报告结果。两次平行测定结果的绝对差值不大于这两个测定值的算术平均值的 $20\%$ 。

# A.5 醛（以乙醛计）的测定

# A.5.1 试剂和材料

A.5.1.1 无醛的乙醇。  
A.5.1.2 碱性品红-亚硫酸溶液。  
A.5.1.3 乙醛杂质测定用标准溶液： $1\mathrm{mg / mL}$

# A.5.2 分析步骤

量取 $6.2\mathrm{mL}$ 试样，置于比色管中，加 $6\mathrm{mL}$ 无醛的乙醇及 $6\mathrm{mL}$ 水，摇匀，冷却至 $15^{\circ}\mathrm{C} \sim 20^{\circ}\mathrm{C}$ ，加 $2\mathrm{mL}$ 碱性品红-亚硫酸溶液，在 $15^{\circ}\mathrm{C} \sim 20^{\circ}\mathrm{C}$ 放置 $15\mathrm{min}$ ，所呈红色不深于标准，即为乙醛含量符合指标要求。

标准是同时取 $9.7\mathrm{mL}$ 水及 $2.5\mathrm{mL}$ 乙醛杂质测定用标准溶液，加 $6\mathrm{mL}$ 无醛的乙醇，摇匀，冷却至 $15^{\circ}\mathrm{C} \sim 20^{\circ}\mathrm{C}$ ，加 $2\mathrm{mL}$ 碱性品红-亚硫酸溶液，在 $15^{\circ}\mathrm{C} \sim 20^{\circ}\mathrm{C}$ 放置 $15\mathrm{min}$ 。

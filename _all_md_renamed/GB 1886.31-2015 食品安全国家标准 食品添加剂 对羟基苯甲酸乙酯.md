本标准与GB8850—2005相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 对羟基苯甲酸乙酯”。

# 检验方法

# A.1 安全提示

试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和健康措施。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所需标准溶液、杂质标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 氢氧化钠溶液： $40~\mathrm{g / L}$   
A.3.1.2 氯化钡溶液： $250~\mathrm{g / L}$   
A.3.1.3 氢氧化钠溶液： $80~\mathrm{g / L}$   
A.3.1.4 硫酸溶液： $1 + 17$   
A.3.1.5 碘溶液： $12.7\mathrm{g / L}$

# A.3.2 鉴别方法

A.3.2.1 称取约 $0.5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加 $10\mathrm{mL}$ 氢氧化钠溶液（A.3.1.1）煮沸 $30\mathrm{min}$ ，蒸发至约 $5\mathrm{mL}$ ，冷却，加硫酸溶液使成酸性，过滤，水洗沉淀至滤液用氯化钡溶液检定不发生浑浊，然后将沉淀物在 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 干燥 $1\mathrm{h}$ ，按GB/T617测定熔点，其终熔温度应在 $213^{\circ}\mathrm{C} \sim 217^{\circ}\mathrm{C}$ 范围内。  
A.3.2.2 称取 $0.5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加 $5\mathrm{mL}$ 氢氧化钠溶液（A.3.1.3），置于沸水浴中加热 $5\mathrm{min}$ ，加 $6\mathrm{mL}$ 硫酸溶液，冷却，过滤，滤液中加 $5\mathrm{mL}$ 氢氧化钠溶液（A.3.1.3），温热至 $50^{\circ}\mathrm{C}$ ，然后加 $5\mathrm{mL}$ 碘溶液，出现三碘甲烷的气味，接着生成沉淀。

# A.4 对羟基苯甲酸乙酯 $(\mathbf{C}_9\mathbf{H}_{10}\mathbf{O}_3)$ 含量（以干基计）的测定

# A.4.1 方法原理

试样与一定量的氢氧化钠进行皂化反应，过量的氢氧化钠用硫酸标准滴定溶液滴定，用 $\mathrm{pH}$ 计指示终点。根据消耗硫酸标准滴定溶液的体积，计算对羟基苯甲酸乙酯的含量。

# A.4.2 试剂和材料

A.4.2.1 氢氧化钠溶液： $40~\mathrm{g / L}$   
A.4.2.2 缓冲溶液： $25^{\circ}\mathrm{C}$ 时， $\mathrm{pH}$ 为4.01和6.86的缓冲溶液，按GB/T9724中的4.2和4.3配制。

A.4.2.3 硫酸标准滴定溶液： $c(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4) = 1\mathrm{mol / L}$

# A.4.3 仪器和设备

$\mathrm{pH}$ 计：精度为 $\mathrm{pH}0.1$

# A.4.4 分析步骤

A.4.4.1 按GB/T9724进行 $\mathrm{pH}$ 计调试。

A.4.4.2 称取约 $2\mathrm{g}$ 在 $80^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 干燥 $2\mathrm{h}$ 后的试样，精确至 $0.0002\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 的碘量瓶中，准确加入 $40~\mathrm{mL}$ 氢氧化钠溶液，缓缓加热至沸，回流 $1\mathrm{h}$ ，冷却至室温，定量转移至烧杯中，移到事先用 $\mathrm{pH} = 6.86$ 缓冲溶液调好的 $\mathrm{pH}$ 计上，用硫酸标准滴定溶液滴定至 $\mathrm{pH}$ 为6.50，并稳定 $30~\mathrm{s}$ ，即为终点。

A.4.4.3 在测定的同时，按与测定相同的步骤，对不加试样而使用相同数量的试剂溶液做空白试验。

# A.4.5 结果计算

对羟基苯甲酸乙酯 $\mathrm{(C_9H_{10}O_3)}$ 含量的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\frac {V _ {1} - V _ {2}}{1000} \times c \times M}{m} \times 100 \% \tag{A.1}
$$

式中：

$V_{1}$ ——空白试验消耗硫酸标准滴定溶液(A.4.2.3)的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试样消耗硫酸标准滴定溶液(A.4.2.3)的体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——毫升和升的转换系数；

$c$ ——硫酸标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——对羟基苯甲酸乙酯的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(\mathrm{C_9H_{10}O_3}) = 166.2]$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 熔点的测定

# A.5.1 毛细管法（仲裁法）

按GB/T617的规定进行。以终熔温度作为测定结果。取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.3^{\circ}\mathrm{C}$ 。

# A.5.2 数字熔点仪法

# A.5.2.1 仪器和设备

数字熔点仪：控温精度为 $\pm 0.5\,^{\circ}\mathrm{C}$ 。

# A.5.2.2 分析步骤

按熔点仪的操作说明书进行调节、校正熔点仪。调节仪器温度至 $112^{\circ}\mathrm{C}$ 。将烘干的试样研细装入洁净干燥的熔点管中，取长约 $800\mathrm{mm}$ 的干燥空心玻璃管直立于玻璃或瓷板上，将装有试样的熔点管在其中投落15次以上，直到熔点管上样品紧缩至 $3\mathrm{mm} \sim 5\mathrm{mm}$ 高或适当高度，待仪器的温度稳定后，将装有试样的熔点管插入进样孔，调节零点，以 $1^{\circ}\mathrm{C / min}$ 的速率升温，数分钟后仪器显示试样的初熔和终

熔温度。以终熔温度作为测定结果。

取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.5\%$ 。

# A.6 游离酸（以对羟基苯甲酸计）的测定

# A.6.1 试剂和材料

A.6.1.1 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.1\mathrm{mol / L}$

A.6.1.2 甲基红指示液： $1\mathrm{g / L}$

# A.6.2 分析步骤

称取 $0.75\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $15\mathrm{mL}$ 约 $80^{\circ}\mathrm{C}$ 热水中加热 $1\mathrm{min}$ ，冷却，过滤，取 $10\mathrm{mL}$ 滤液，加2滴甲基红指示液，用氢氧化钠标准滴定溶液滴定至黄色，稳定 $30\mathrm{s}$ 即为终点，消耗氢氧化钠标准滴定溶液的体积不应超过 $0.20\mathrm{mL}$ ，即为游离酸质量分数小于等于 $0.55\%$ 。

# A.7 硫酸盐（以 $\mathrm{SO}_4$ 计）的测定

# A.7.1 试剂和材料

A.7.1.1 盐酸溶液： $1 + 3$

A.7.1.2 氯化钡溶液： $250~\mathrm{g / L}$

A.7.1.3 硫酸盐 $\mathrm{(SO_4)}$ 标准溶液： $0.1\mathrm{mg / mL}$

A.7.1.4 标准溶液的配置：取 $0.96 \mathrm{~mL}$ 硫酸盐 $\left(\mathrm{SO}_{4}\right)$ 标准溶液于 $50 \mathrm{~mL}$ 比色管中，加 $5 \mathrm{~mL}$ 盐酸溶液及 $3 \mathrm{~mL}$ 氯化钡溶液，摇匀，放置 $10 \mathrm{~min}$ 。

# A.7.2 分析步骤

称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加 $100\mathrm{mL}$ 热水，充分振摇混合，加热 $5\mathrm{min}$ ，冷却，加水至 $100\mathrm{mL}$ ，过滤，取滤液 $40\mathrm{mL}$ 于 $50\mathrm{mL}$ 比色管中，加 $5\mathrm{mL}$ 盐酸溶液及 $3\mathrm{mL}$ 氯化钡溶液，摇匀，放置 $10\mathrm{min}$ ，所呈浊度不应大于标准，即为硫酸盐（以 $\mathrm{SO}_4$ ）质量分数小于等于 $0.024\%$ 。

# A.8 干燥减量的测定

# A.8.1 分析步骤

称取 $2\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于已预先于 $80^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 干燥至质量恒定的直径 $45\mathrm{mm}$ 高 $25\mathrm{mm}$ 的称量瓶中，分布均匀，于 $80^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 干燥 $2\mathrm{h}$ ，取出，冷却至室温，称量。

# A.8.2 结果计算

干燥减量的质量分数 $\varpi_{2}$ ，按式（A.2）计算：

$$
w _ {2} = \frac {m _ {1} - m _ {2}}{m _ {1}} \times 100 \% \tag{A.2}
$$

式中：

$m_{1}$ ——干燥前试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——干燥后试样的质量，单位为克 $(\mathrm{g})$ 。

取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.05\%$ 。

# A.9 灼烧残渣的测定

按GB/T9741的规定进行。测定时，称取 $5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ 。灼烧温度为 $500^{\circ}\mathrm{C} \pm 50^{\circ}\mathrm{C}$ 。取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.01\%$ 。

# A.10 重金属(以Pb计)的测定

# A.10.1 试剂和材料

A.10.1.1 丙酮。

A.10.1.2 硫化钠溶液：称取硫化钠 $5\mathrm{g}$ ，精确至 $0.01\mathrm{g}$ ，用水 $10\mathrm{mL}$ 和甘油 $30\mathrm{mL}$ 混合溶解。此溶液使用期限为两周。

A.10.1.3 乙酸溶液： $1\rightarrow 20$

A.10.1.4 铅(Pb)标准溶液： $10~\mu \mathrm{g / mL}$

# A.10.2 分析步骤

称取 $2.0\mathrm{g}$ 试样，精确至 $0.001\mathrm{g}$ ，于 $50~\mathrm{mL}$ 比色管中，加 $25~\mathrm{mL}$ 丙酮溶解，加 $2\mathrm{mL}$ 乙酸溶液，用水稀释至 $50~\mathrm{mL}$ ，加2滴硫化钠溶液，摇匀，于暗处放置 $10\mathrm{min}$ ，所呈颜色不应深于标准。

标准溶液的配置是吸取 $2.00\mathrm{mL}$ 铅(Pb)标准溶液，用水稀释至 $50~\mathrm{mL}$ ，加2滴硫化钠溶液，摇匀。

# 前言

本标准代替GB/T23296.11—2009《食品接触材料塑料中环氧乙烷和环氧丙烷含量的测定气相色谱法》。

本标准与GB/T23296.11—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 塑料中环氧乙烷和环氧丙烷的测定”；

——增加了“顶空气相色谱-质谱法”测定环氧乙烷与环氧丙烷。

# 食品安全国家标准

# 食品接触材料及制品

# 塑料中环氧乙烷和环氧丙烷的测定

# 1 范围

本标准规定了食品接触材料及制品中环氧乙烷和环氧丙烷的测定方法。

本标准适用于塑料食品接触材料及制品中环氧乙烷和环氧丙烷的测定。

# 第一法 顶空气相色谱-质谱法

# 2 原理

样品中环氧乙烷或环氧丙烷经 $N,N-$ 二甲基乙酰胺(DMAC)提取后，采用顶空进样，在色谱柱中环氧乙烷或环氧丙烷与内标物乙醚及其他组分分离，用质谱检测器检测，以内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

$N,N-$ 二甲基乙酰胺（简称DMAC， $\mathrm{C_4H_9NO}$ ，CAS号：127-19-5）：色谱纯，色谱图上与环氧乙烷或环氧丙烷有相同保留时间的杂质峰面积不得超过 $1\%$ 。

# 3.2 标准品

3.2.1 环氧乙烷 $\mathrm{(C_2H_4O}$ ,CAS号：75-21-8)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.2.2 环氧丙烷 $\mathrm{(C_3H_6O}$ ,CAS号：75-56-9)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.2.3 乙醚 $\mathrm{(C_4H_{10}O,CAS}$ 号：60-29-7)：纯度 $\geqslant 99.5\%$ ，或经国家认证并授予标准物质证书的标准物质。色谱图上与环氧乙烷或环氧丙烷有相同保留时间的杂质峰面积不得超过 $1\%$ 。

# 3.3 标准溶液配制

# 3.3.1 环氧乙烷标准储备液

称取 $50\mathrm{mg}$ (精确到 $0.0001\mathrm{g}$ ) 环氧乙烷，用DMAC溶解后，定容至 $50~\mathrm{mL}$ ，配制成浓度为 $1\mathrm{mg / mL}$ 的储备液。溶液在 $4^{\circ}\mathrm{C}$ 下避光密封储存，有效期为2个月。

# 3.3.2 环氧丙烷标准储备液

称取 $50\mathrm{mg}$ (精确到 $0.0001\mathrm{g}$ ) 环氧丙烷，用DMAC溶解后，定容至 $50~\mathrm{mL}$ ，配制成浓度为 $1\mathrm{mg / mL}$ 的储备液。溶液储存条件同3.3.1。

# 3.3.3 环氧乙烷、环氧丙烷混标中间溶液

量取 $5\mathrm{mL}$ DMAC于 $50~\mathrm{mL}$ 容量瓶中，用刻度吸量管分别吸取 $2.0~\mathrm{mL}$ 环氧乙烷标准储备液(3.3.1)及 $1.0~\mathrm{mL}$ 环氧丙烷标准储备液(3.3.2)于该容量瓶中，用DMAC定容，获得环氧乙烷、环氧丙烷混标中间溶液。环氧乙烷、环氧丙烷浓度分别为 $40.0~\mu \mathrm{g / mL},20.0~\mu \mathrm{g / mL}$ 。溶液储存条件同3.3.1。

# 3.3.4 环氧乙烷、环氧丙烷混标溶液

用刻度吸量管分别吸取 $5.0\mathrm{mL}$ DMAC于6个 $10~\mathrm{mL}$ 容量瓶中，分别用微量注射器吸取 $25~\mu \mathrm{L}$ 、 $50~\mu \mathrm{L},100~\mu \mathrm{L},250~\mu \mathrm{L},500~\mu \mathrm{L},1000~\mu \mathrm{L}$ 环氧乙烷、环氧丙烷混标中间溶液(3.3.3)加入上述6个容量瓶中，用DMAC定容。获得环氧乙烷的浓度分别为 $0.1~\mu \mathrm{g / mL},0.2~\mu \mathrm{g / mL},0.4~\mu \mathrm{g / mL},1.0~\mu \mathrm{g / mL}$ $2.0~\mu \mathrm{g / mL},4.0~\mu \mathrm{g / mL}$ ，环氧丙烷的浓度分别为 $0.05~\mu \mathrm{g / mL},0.1~\mu \mathrm{g / mL},0.2~\mu \mathrm{g / mL},0.5~\mu \mathrm{g / mL}$ $1.0~\mu \mathrm{g / mL},2.0~\mu \mathrm{g / mL}$ 。溶液储存条件同3.3.1。

# 3.3.5 乙醚内标储备液

称取 $50\mathrm{mg}$ (精确至 $0.0001\mathrm{g}$ ) 乙醚于预先盛有 $5\mathrm{mL}$ DMAC的 $50\mathrm{mL}$ 容量瓶中，用DMAC定容，配制成浓度为 $1\mathrm{mg / mL}$ 的储备液。溶液储存条件同3.3.1。

# 3.3.6 乙醚内标中间溶液

量取 $5\mathrm{mL}$ DMAC于 $10~\mathrm{mL}$ 容量瓶中，再用刻度吸量管吸取 $2.5  \mathrm{mL}$ 的乙醚内标储备液(3.3.5)于该容量瓶中，用DMAC定容，获得乙醚内标中间溶液。其浓度为 $250.0~\mu \mathrm{g / mL}$ 。溶液储存条件同3.3.1。

# 4 仪器和设备

4.1 气相色谱-质谱联用仪：配备电子轰击离子源(EI)和顶空自动进样器。  
4.2 微量注射器： $10~\mu \mathrm{L},100~\mu \mathrm{L},1000~\mu \mathrm{L}$   
4.3 顶空瓶： $20~\mathrm{mL}$ ，配备铝盖和丁基橡胶或硅树脂橡胶隔垫，隔垫接触样品一面应涂有聚四氟乙烯。  
4.4 分析天平：感量 $0.0001\mathrm{g}$ 。  
4.5 机械振荡器。

# 5 分析步骤

# 5.1 试样处理

可溶于DMAC的试样直接称量；不溶于DMAC的试样，使用冷冻研磨仪或剪刀等切割工具将其破碎成粒径小于 $1\mathrm{mm}\times 1\mathrm{mm}$ 后，尽快称量。切割样品时，不可使其发热变软。

# 5.2 试样溶液的制备

称取 $1.0\mathrm{g}\pm 0.1\mathrm{g}$ 试样(5.1)于顶空瓶中，用刻度吸量管吸取 $5.0~\mathrm{mL}$ DMAC于顶空瓶中，用微量注射器加入 $10~\mu \mathrm{L}$ 的乙醚内标中间溶液(3.3.6)后，尽快用盖密封。对于可溶于DMAC的试样，用机械振荡器以每两秒一次的频率振荡顶空瓶直至试样溶解；对于不溶于DMAC的试样，用机械振荡器以每两

秒一次的频率振荡顶空瓶 $4\mathrm{h}$ 。若样品浓度超出线性范围，可在加入内标溶液前用DMAC适当稀释，使其浓度处于线性范围内。

# 5.3 空白溶液的制备

除不加试样外，采用与5.2完全相同的分析步骤、试剂和用量。

# 5.4 仪器参考条件

# 5.4.1 顶空进样器条件

顶空进样器条件列出如下：

a）样品平衡时间：可溶于DMAC的样品为 $30\mathrm{min}$ ，不溶于DMAC的样品为 $60\mathrm{min}$   
b）顶空瓶温度： $100^{\circ}\mathrm{C}$   
c） 定量环温度： $105~\mathrm{^\circ C}$   
d) 传输线温度: $110^{\circ} \mathrm{C}$ 。

# 5.4.2 气相色谱条件

气相色谱条件列出如下：  
a）色谱柱：键合苯乙烯-二乙烯苯的PLOT柱，柱长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ ，膜厚 $20~\mu \mathrm{m}$   
b）进样口温度： $200^{\circ}\mathrm{C}$   
c）进样方式：分流进样，分流比为 $1:1$   
d）柱温： $60^{\circ}\mathrm{C}$ 下恒温 $1\mathrm{min}$ ，以 $10^{\circ}\mathrm{C / min}$ 升至 $100^{\circ}\mathrm{C}$ ，以 $20^{\circ}\mathrm{C / min}$ 升至 $180^{\circ}\mathrm{C}$ 并恒温 $2\mathrm{min}$ ，以 $30^{\circ}\mathrm{C / min}$ 升至 $250^{\circ}\mathrm{C}$ 并恒温 $4\mathrm{min}$ ；  
e) 载气：氦气。

# 5.4.3 质谱条件

质谱条件列出如下：

a）质谱接口温度： $280^{\circ}C$   
b）四级杆温度： $150~\mathrm{^\circ C}$   
c） 电离方式：EI；  
d）电离能量： $70\mathrm{eV}$   
e）测定方式：选择离子监测模式，监测离子范围： $m / z15 \sim m / z100$ 。环氧乙烷的特征离子为15、29、43、44，其中44为定量离子；环氧丙烷的特征离子为28、31、43、58，其中58为定量离子；乙醚的特征离子为31、45、59、74，其中74为定量离子。

# 5.5 标准曲线的制作

用刻度吸量管分别吸取 $5.0\mathrm{mL}$ 环氧乙烷、环氧丙烷混标溶液(3.3.4)于6个顶空瓶中，再用微量注射器分别加入 $10~\mu \mathrm{L}$ 乙醚内标中间溶液(3.3.6)，尽快用盖密封。按照5.4所列测定条件，分别将上述溶液依次通过顶空气相色谱-质谱仪测定。分别以溶液中环氧乙烷(或环氧丙烷)浓度为横座标，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ，以对应环氧乙烷(或环氧丙烷)峰面积与乙醚峰面积之比为纵座标，分别绘制标准曲线。标准色谱图参见A.1。

# 5.6 试样溶液的测定

将试样溶液和空白溶液注入顶空气相色谱-质谱仪测定，得到环氧乙烷(或环氧丙烷)与乙醚峰面积之比，由标准曲线得到试样溶液中环氧乙烷(或环氧丙烷)的浓度。

# 5.7 定性

在相同的实验条件下进行样品测定时，若试样溶液中待测物色谱峰的保留时间与相应的标准品色谱峰的保留时间的偏差在 $\pm 2.5\%$ 以内，并且在扣除背景后的样品质谱中，所选择的离子均出现，且样品谱图中定性离子的相对丰度与浓度接近的标准溶液谱图中对应定性离子的相对丰度进行比较，偏差不超过表1规定的范围，则可判定为样品中存在该种待测物。环氧乙烷、环氧丙烷与乙醚的特征离子及其丰度比见表2。

# 6 分析结果的表述

试样中环氧乙烷(或环氧丙烷)含量按式(1)计算：

$$
X _ {i} = \frac {\rho_ {i} \times V \times f \times 1 0 ^ {- 3}}{m \times 1 0 ^ {- 3}} \tag {1}
$$

式中：

$X_{i}$ ——试样中残留环氧乙烷(或环氧丙烷)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho_{i}$ ——依据校准曲线获得提取液中环氧乙烷(或环氧丙烷)的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样溶液体积，单位为每毫升 $(\mathrm{mL})$ ；

$f$ ——浓度稀释因子；

$10^{-3}$ ——单位换算因子；

$m$ ——试样称样量，单位为克 $(\mathrm{g})$ 。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过其算术平均值的 $10\%$ 。

# 8 其他

环氧乙烷检出限为 $0.2\mathrm{mg / kg}$ ，定量限为 $0.5\mathrm{mg / kg}$ ；环氧丙烷检出限为 $0.1\mathrm{mg / kg}$ ，定量限为 $0.3\mathrm{mg / kg}$ 。

# 第二法 顶空气相色谱法

# 9 原理

样品中环氧乙烷或环氧丙烷经 $N,N-$ 二甲基乙酰胺(DMAC)提取后，采用顶空进样，在色谱柱中环氧乙烷或环氧丙烷与内标物乙醚及其他组分分离，用氢火焰离子化检测器检测。按标准加入法绘制标准曲线，以内标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

$N,N-$ 二甲基乙酰胺 $\mathrm{(C_4H_9NO}$ ,CAS号：127-19-5)：色谱纯，色谱图上与环氧乙烷或环氧丙烷有相同保留时间的杂质峰面积不得超过 $1\%$ 。

# 10.2 标准品

10.2.1 环氧乙烷 $\mathrm{(C_2H_4O}$ ,CAS号：75-21-8)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。  
10.2.2 环氧丙烷 $\left(\mathrm{C}_{3} \mathrm{H}_{6} \mathrm{O}, \mathrm{CAS}\right.$ 号：200-879-2）：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。  
10.2.3 乙醚 $\left(\mathrm{C}_{4} \mathrm{H}_{10} \mathrm{O}, \mathrm{CAS}\right.$ 号：60-29-7)：纯度 $\geqslant 99.5\%$ ，或经国家认证并授予标准物质证书的标准物质。色谱图上与环氧乙烷或环氧丙烷有相同保留时间的杂质峰面积不得超过 $1 \%$ 。

# 10.3 标准溶液配制

# 10.3.1 环氧乙烷标准储备液

称取 $50\mathrm{mg}$ (精确到 $0.0001\mathrm{g}$ ) 环氧乙烷，用DMAC溶解后，定容至 $50~\mathrm{mL}$ ，配制成浓度为 $1\mathrm{mg / mL}$ 的储备液。溶液在 $4^{\circ}\mathrm{C}$ 下避光密封储存，有效期为2个月。

# 10.3.2 环氧丙烷标准储备液

称取 $50\mathrm{mg}$ (精确到 $0.0001\mathrm{g}$ ) 环氧丙烷，用DMAC溶解后，定容至 $50~\mathrm{mL}$ ，配制成浓度为 $1\mathrm{mg / mL}$ 的储备液。溶液在 $4^{\circ}\mathrm{C}$ 下避光密封储存，有效期为2个月。

# 10.3.3 环氧乙烷、环氧丙烷混标中间溶液

分别量取 $5\mathrm{mL}$ DMAC于6个 $10~\mathrm{mL}$ 容量瓶中，用微量注射器分别吸取 $0\mu \mathrm{L},40\mu \mathrm{L},100\mu \mathrm{L}$

$200\mu \mathrm{L},300\mu \mathrm{L},400\mu \mathrm{L}$ 环氧乙烷标准储备液(10.3.1)于上述6个容量瓶中，再用微量注射器分别吸取 $0\mu \mathrm{L},40\mu \mathrm{L},100\mu \mathrm{L},200\mu \mathrm{L},300\mu \mathrm{L},400\mu \mathrm{L}$ 环氧丙烷标准储备液(10.3.2)于上述6个容量瓶中，用DMAC定容至刻度，获得环氧乙烷、环氧丙烷混标中间溶液。其中，环氧乙烷的浓度分别为 $0\mu \mathrm{g / mL}$ 产 $4\mu \mathrm{g / mL},10\mu \mathrm{g / mL},20\mu \mathrm{g / mL},30\mu \mathrm{g / mL},40\mu \mathrm{g / mL}$ ，环氧丙烷的浓度分别为 $0\mu \mathrm{g / mL},4\mu \mathrm{g / mL}$ $10~\mu \mathrm{g / mL},20~\mu \mathrm{g / mL},30~\mu \mathrm{g / mL},40~\mu \mathrm{g / mL}$ 。溶液储存条件同10.3.1。

# 10.3.4 乙醚内标储备液

称取 $50\mathrm{mg}$ (精确到 $0.0001\mathrm{g}$ ) 乙醚，用DMAC溶解后，定容至 $50~\mathrm{mL}$ ，配制成浓度为 $1\mathrm{mg / mL}$ 的储备液。溶液储存条件同10.3.1。

# 10.3.5 乙醚内标中间溶液

量取 $5\mathrm{mL}$ DMAC于 $10~\mathrm{mL}$ 容量瓶中，再用刻度吸量管吸取 $3.0~\mathrm{mL}$ 乙醚内标储备液(10.3.4)于该容量瓶中，用DMAC定容，获得乙醚内标中间液。其中，乙醚为 $300~\mu \mathrm{g / mL}$ 。溶液储存条件同10.3.1。

# 11 仪器和设备

11.1 气相色谱仪：配备氢火焰离子化检测器(FID)和顶空自动进样器。   
11.2 微量注射器： $10\mu \mathrm{L}, 100\mu \mathrm{L}, 1000\mu \mathrm{L}$ 。  
11.3 顶空瓶：20 mL，配备铝盖和丁基橡胶或硅树脂橡胶隔垫，隔垫接触样品一面应涂有聚四氟乙烯。  
11.4 分析天平：感量 $0.0001\mathrm{g}$ 。  
11.5 机械振荡器。

# 12 分析步骤

# 12.1 试样处理

可溶于DMAC的试样直接称量；不溶于DMAC的试样，使用冷冻研磨仪或剪刀等切割工具将其破碎成粒径小于 $1\mathrm{mm}\times 1\mathrm{mm}$ 后，尽快称量。切割样品时，不可使其发热变软。

# 12.2 标准加入法标准工作溶液制备

分别称取6份质量为 $0.200\mathrm{g}\pm 0.005\mathrm{g}$ 试样于6个顶空瓶中，用刻度吸量管分别吸取 $1.0~\mathrm{mL}$ DMAC于6个顶空瓶中。分别用微量注射器准确加入 $10~\mu \mathrm{L}$ 环氧乙烷、环氧丙烷混标中间溶液(10.3.3)于6个顶空瓶中，再分别用微量注射器准确加入 $10~\mu \mathrm{L}$ 的乙醚内标中间溶液(10.3.5)后，尽快用盖密封。添加到试样中环氧乙烷、环氧丙烷的标准工作溶液浓度均分别为 $0\mathrm{mg / kg},0.2\mathrm{mg / kg}$ 产 $0.5\mathrm{mg / kg},1.0\mathrm{mg / kg},1.5\mathrm{mg / kg},2.0\mathrm{mg / kg}$ ，内标乙醚含量为 $1.5\mathrm{mg / kg}$ 。对于可溶于DMAC的试样，用机械振荡器以每两秒一次的频率振荡顶空瓶直至试样溶解；对于不溶于DMAC的试样，用机械振荡器以每两秒一次的频率振荡顶空瓶 $4\mathrm{h}$

注：标准工作溶液是由添加到试样中环氧乙烷(或环氧丙烷)总量除以试样质量得到，单位为 $\mathrm{mg / kg}$ 。

# 13 分析结果的表述

# 13.1 图表测定法

根据外延法，将图1中标准曲线回延至 $x$ 轴，其在 $x$ 轴上的截距 $-c$ 的绝对值与试样中环氧乙烷（或环氧丙烷)的浓度值相等，从标准曲线图上可直接读取试样中环氧乙烷(或环氧丙烷)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 。

# 13.2 回归参数计算法

根据回归参量 $a, b$ ，试样中的环氧乙烷(或环氧丙烷)的含量，按式(3)计算：

$$
c _ {i} = \frac {b}{a} \quad \dots \tag {3}
$$

式中：

$c_{i}$ ——试样中环氧乙烷(或环氧丙烷)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$b$ ——回归曲线的截距；

$a$ ——回归曲线的斜率。

计算结果保留两位有效数字。

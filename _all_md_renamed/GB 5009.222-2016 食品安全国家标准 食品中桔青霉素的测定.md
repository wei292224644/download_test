# 前言

本标准代替GB/T5009.222—2008《红曲类产品中桔青霉素的测定》、SN/T2426—2010《进出口粮谷中桔霉素含量检测方法液相色谱法》和SN/T2916—2011《出口食品中桔霉素的测定方法免疫亲和柱净化-高效液相色谱法》。

本标准与GB/T5009.222—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中桔青霉素的测定”；  
——增加了净化步骤；  
——增加了适用范围。

# 食品安全国家标准

# 食品中桔青霉素的测定

# 1 范围

本标准规定了食品中桔青霉素的测定方法。

本标准第一法适用于大米、玉米、辣椒、红曲类产品中桔青霉素的测定，第二法适用于大米、大麦、燕麦、小麦中桔青霉素的测定。

# 第一法 免疫亲和柱净化-高效液相色谱法

# 2 原理

试样中的桔青霉素用甲醇-水提取，提取液经过滤、稀释后，用免疫亲和柱净化，采用液相色谱结合荧光检测器测定桔青霉素的含量，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB6682规定的二级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.3 磷酸 $\mathrm{(H_3PO_4)}$ ：色谱纯。  
3.1.4 冰乙酸 $\mathrm{(C_2H_4O_2)}$ ：色谱纯。  
3.1.5 氢氧化钠(NaOH)。  
3.1.6 吐温 $20(\mathrm{C}_{58}\mathrm{H}_{114}\mathrm{O}_{26})$ 。  
3.1.7 氯化钠 $\mathrm{(NaCl)}$ 。  
3.1.8 磷酸氢二钠 $\mathrm{(Na_2HPO_4)}$ 。  
3.1.9 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
3.1.10 氯化钾(KCl)。

# 3.2 试剂配制

3.2.1 氢氧化钠溶液 $(2\mathrm{mol} / \mathrm{L})$ ：称取 $80.0\mathrm{g}$ 固体氢氧化钠，溶于 $1\mathrm{L}$ 水中。  
3.2.2 磷酸溶液 $(10\mathrm{mmol / L},\mathrm{pH}7.5)$ ：准确移取 $1.376~\mathrm{mL}$ 磷酸加水定容至 $2000~\mathrm{mL}$ ，用 $2\mathrm{mol / L}$ 的氢氧化钠溶液调节 $\mathrm{pH}$ 至7.5。  
3.2.3 磷酸溶液 $(10\mathrm{mmol / L},\mathrm{pH}2.5)$ ：准确移取 $1.376~\mathrm{mL}$ 磷酸加水定容至 $2000~\mathrm{mL}$ ，用 $2\mathrm{mol / L}$ 的氢氧化钠溶液调节 $\mathrm{pH}$ 至2.5。  
3.2.4 PBS缓冲溶液：称取 $8.0\mathrm{g}$ 氯化钠、 $1.2\mathrm{g}$ 磷酸氢二钠、 $0.2\mathrm{g}$ 磷酸二氢钾、 $0.2\mathrm{g}$ 氯化钾，用水溶解，

调节 $\mathrm{pH}$ 至7.0，用水定容至 $1000~\mathrm{mL}$   
3.2.5 $0.1\%$ 吐温20-PBS溶液：准确移取 $1\mathrm{mL}$ 吐温20以PBS缓冲溶液定容至 $1000~\mathrm{mL}$   
3.2.6 $0.1\%$ 磷酸溶液：准确移取 $1\mathrm{mL}$ 磷酸加水定容至 $1000~\mathrm{mL}$   
3.2.7 洗脱液I：甲醇-10mmol/L磷酸溶液 $(\mathrm{pH}2.5)(70 + 30)$ 。  
3.2.8 洗脱液Ⅱ：甲醇-0.1%磷酸溶液 $(70 + 30)$

# 3.3 标准品

桔青霉素 $\mathrm{(C_{13}H_{14}O_5}$ ,CAS号：518-75-2)，纯度 $\geq 99.0\%$ 。或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 标准储备液：准确称取一定量的桔青霉素标准品，以甲醇溶解并定容至 $10.0\mathrm{mL}$ 作为标准储备液，浓度为 $100~\mu \mathrm{g / mL}$ ，于 $4^{\circ}\mathrm{C}$ 下保存。  
3.4.2 标准中间液：准确移取 $1.0 \mathrm{~mL}$ 桔青霉素标准储备液于 $10 \mathrm{~mL}$ 容量瓶中，用甲醇定容，浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ ，于 $4^{\circ} \mathrm{C}$ 下保存。  
3.4.3 基质标准工作液：根据需要，取适量的标准中间液，用空白样品提取液配成不同浓度的基质标准工作液，现用现配。

# 3.5 材料

3.5.1 桔青霉素免疫亲和柱：柱体积 $3\mathrm{mL}$ ，最大柱容量 $20\mathrm{ng}$ ，或等效柱。  
3.5.2 玻璃纤维滤纸：直径 $11\mathrm{cm}$ ，孔径 $1.5\mu \mathrm{m}$ 。

# 4 仪器和设备

4.1 高效液相色谱仪，配荧光检测器。  
4.2 分析天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$   
4.3 高速均质器： $\geqslant 12000\mathrm{r / min}$   
4.4 混匀器。

# 5 分析步骤

# 5.1 提取

# 5.1.1 大米、玉米、辣椒

称取经充分粉碎均质试样 $10.0\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）于 $150~\mathrm{mL}$ 具塞锥形瓶中，加入 $50~\mathrm{mL}$ 甲醇-水 $(70 + 30)$ 提取液，以高速均质器高速均质提取 $2\mathrm{min}$ ，过滤提取液，移取 $1.0~\mathrm{mL}$ 滤液，置于另一干净的容器中，加入 $49~\mathrm{mL}~10~\mathrm{mmol / L}$ 磷酸溶液 $\left(\mathrm{pH}7.5\right)$ 稀释、混匀；以玻璃纤滤纸过滤待免疫亲和柱净化。

# 5.1.2 红曲及其制品

称取经充分粉碎均质试样 $1.0\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）于 $50~\mathrm{mL}$ 具塞锥形瓶中，加入 $20~\mathrm{mL}$ 甲醇-水 $(70 + 30)$ 提取液，以高速均质器高速均质提取 $2\mathrm{min}$ ，过滤提取液，移取 $1.0~\mathrm{mL}$ 滤液，置于另一干净的容器中，加入 $39~\mathrm{mL}$ PBS缓冲溶液稀释、混匀；以玻璃纤滤纸过滤待免疫亲和柱净化。

# 5.2 净化

# 5.2.1 大米、玉米、辣椒

将免疫亲和柱连接于 $10~\mathrm{mL}$ 玻璃针筒下，准确移取 $10.0~\mathrm{mL}$ （相当于 $0.04\mathrm{g}$ 试样）上述澄清滤液过免疫亲和柱，以1滴/s～2滴/s的流速全部通过亲和柱；加入 $5\mathrm{mL}10\mathrm{mmol / L}$ 磷酸 $\left(\mathrm{pH}7.5\right)$ 以1滴/s～2滴/s的流速淋洗柱子，直至空气进入到亲和柱中，弃去全部流出液。准确加入 $1.0~\mathrm{mL}$ 洗脱液I进行洗脱，洗脱流速为1滴/s～2滴/s。收集全部洗脱液于玻璃试管中，供检测用。

# 5.2.2 红曲及其制品

将免疫亲和柱连接于 $10~\mathrm{mL}$ 玻璃针筒下，准确移取 $10.0~\mathrm{mL}$ 上述澄清滤液过免疫亲和柱，以1滴/s～2滴/s的流速全部通过亲和柱；加入 $10~\mathrm{mL}0.1\%$ 吐温20-PBS溶液，以1滴/s～2滴/s的流速淋洗柱子，直至空气进入到亲和柱中，弃去全部流出液。准确加入 $1.0~\mathrm{mL}$ 洗脱液Ⅱ（红曲及其制品）进行洗脱，洗脱流速为1滴/s～2滴/s。收集全部洗脱液于玻璃试管中，供检测用。

# 5.3 空白试验

除不加试样外，均按上述操作步骤进行。

# 5.4 测定

# 5.4.1 液相色谱参考条件

# 5.4.1.1 大米、玉米、辣椒

液相色谱分析条件列出如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $150~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或等效柱；  
b） 柱温： $30^{\circ}C$   
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d）进样量： $50~\mu \mathrm{L}$   
e）检测条件：激发波长 $350~\mathrm{nm}$ ，发射波长 $500~\mathrm{nm}$   
f）流动相A液：乙腈，流动相B液： $10\mathrm{mmol / L}$ 磷酸 $(\mathrm{pH}2.5)$ 。

流动相及梯度洗脱条件见表1。

# 5.4.2 标准曲线的制作

配制 $0.0\mathrm{ng / mL},1.0\mathrm{ng / mL},2.0\mathrm{ng / mL},5.0\mathrm{ng / mL},10.0\mathrm{ng / mL},20.0\mathrm{ng / mL}6$ 个浓度的基质标准工作液。在仪器最佳工作条件下，用基质标准工作溶液分别进样，以相应的桔青霉素的色谱峰的峰面积为纵坐标，以基质标准工作溶液中桔青霉素的浓度为横坐标，绘制标准曲线。

桔青霉素标准的色谱图参见图A.1和图A.2。

# 5.4.3 试样溶液的测定

将试样溶液注入高效液相色谱仪，测定相应的峰面积。由标准曲线得到试样溶液中桔青霉素的浓度。

# 6 分析结果的表述

试样中桔青霉素含量按式(1)计算：

$$
X = \frac {\rho \times V \times f}{m} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中桔青霉素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——样液中桔青霉素的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——样液稀释倍数；

$m$ ——样液所代表的试样量，单位为克 $(\mathrm{g})$ 。

计算结果需扣除空白值。计算结果保留两位有效数字。

# 7 精密度

在重复条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法中大米、玉米、辣椒粉样品的检出限和定量限分别为 $8\mu \mathrm{g / kg}$ 和 $25~\mu \mathrm{g / kg}$ ；红曲及其制品的检出限和定量限分别为 $25~\mu \mathrm{g / kg}$ 和 $80~\mu \mathrm{g / kg}$ 。

# 第二法 $C_{18}$ 固相萃取小柱净化-高效液相色谱法

# 9 原理

用乙腈-异丙醇-水的混合溶液提取试样中桔青霉素， $\mathrm{C_{18}}$ 固相萃取小柱净化，用配荧光检测器的液相色谱仪测定，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB6682规定的二级水。

# 10.1 试剂

10.1.1 异丙醇 $\left(\mathrm{C}_{3} \mathrm{H}_{8} \mathrm{O}\right)$ ：色谱纯。  
10.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
10.1.3 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ ：优级纯。  
10.1.4 甲醇 $\mathrm{(CH_3OH)}$ 。

# 10.2 试剂配制

10.2.1 提取溶剂：乙腈-异丙醇-水 $(35 + 10 + 55)$ ，用磷酸调 $\mathrm{pH}$ 为1.5。  
10.2.2 磷酸溶液 $(0.08\mathrm{mol} / \mathrm{L})$ ：取 $5.6~\mathrm{mL}$ 磷酸，以水定容至 $1\mathrm{L}$ 。  
10.2.3 流动相：乙腈-异丙醇-磷酸溶液 $(35 + 10 + 55)$ 。

# 10.3 标准品

桔青霉素 $\left(\mathrm{C}_{13} \mathrm{H}_{14} \mathrm{O}_{5}\right)$ , CAS 号: 518-75-2), 纯度 $\geqslant 99 \%$ 。或经国家认证并授予标准物质证书的标准物质。

# 10.4 标准溶液配制

10.4.1 桔青霉素标准储备液：称取适量桔青霉素标准物质，用乙腈溶解并定容至 $1.0 \mathrm{mg} / \mathrm{mL}, 0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 保存。  
10.4.2 桔青霉素标准工作液：根据需要用流动相将标准储备液稀释成 $25\mathrm{ng / mL}, 50\mathrm{ng / mL}, 100\mathrm{ng / mL},$ $500\mathrm{ng / mL}, 1000\mathrm{ng / mL}$ 的标准工作溶液。

# 10.5 材料

10.5.1 $\mathrm{C_{18}}$ 固相萃取柱：填料 $500\mathrm{mg}$ ，柱体积 $3\mathrm{mL}$ ，或等效柱。  
10.5.2 玻璃纤维滤纸：直径 $11\mathrm{cm}$ ，孔径 $1.5\mu \mathrm{m}$ 。

# 11 仪器和设备

11.1 高效液相色谱仪：配有荧光检测器。  
11.2 振荡器。  
11.3 离心机： $\geqslant 6500\mathrm{r / min}$   
11.4 真空固相萃取装置。  
11.5 氮吹仪。  
11.6 分析天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。

# 12 分析步骤

# 12.1 提取

取样品 $500\mathrm{g}$ ，用粉碎机粉碎并通过 $830~\mu \mathrm{m}$ 圆孔筛，混匀，分成2份，装入洁净容器内，密封保存。称取试样约 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加 $10~\mathrm{mL}$ 提取溶剂(10.2.1)，在振荡器上振荡提取 $30\mathrm{min}$ 。以 $3500\mathrm{r / min}$ 离心 $4\mathrm{min}$ ，上清液转入另一离心管中。在残渣中再加入 $5\mathrm{mL}$ 提取溶剂(10.2.1)，重复上述操作，合并上清液。在提取液中加水至 $40~\mathrm{mL}$ ，并用磷酸调 $\mathrm{pH}$ 为1.5，过玻璃纤维滤纸，待净化。

# 12.2 净化

将上述溶液过预淋洗好的 $\mathrm{C_{18}}$ 固相萃取柱，用 $5\mathrm{mL}$ 水淋洗柱子。待淋洗液全部流出柱子后，减压抽干 $3\mathrm{min}$ 。用 $10~\mathrm{mL}$ 甲醇以 $1.0~\mathrm{mL / min}$ 的速度洗脱，收集全部洗脱液，在 $40^{\circ}C$ 下， $\mathbf{N}_2$ 吹干，再以 $1.0~\mathrm{mL}$ 流动相溶解，过 $0.22\mu \mathrm{m}$ 的有机相微孔滤膜，供液相色谱测定。

# 12.3 空白试验

除不加试样外，均按上述操作步骤进行。

# 12.4 测定

# 12.4.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或等效者；  
b）流动相：乙腈-异丙醇-磷酸溶液 $(35 + 10 + 55)$   
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d）进样量： $50~\mu \mathrm{L}$   
e) 柱温: $28^{\circ} \mathrm{C}$ ;   
f）检测波长：激发波长 $331\mathrm{nm}$ ，发射波长 $500\mathrm{nm}$ 。

# 12.4.2 色谱测定

根据样液中被测桔青霉素含量情况，选定峰面积相近的标准工作溶液。标准工作液和样液中桔青霉素响应值均应在仪器检测线性范围内。对标准工作液和样液等体积参插进样测定。在上述色谱条件下，桔青霉素保留时间约为 $9.1\mathrm{min}$ ，标准物质色谱图见图A.3。

# 13 分析结果的表述

试样中桔青霉素含量按式(2)计算：

$$
X = \frac {A \times \rho \times V}{A _ {\mathrm {s}} \times m} \quad \dots \tag {2}
$$

式中：

$X$ ——试样中桔青霉素含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——样液中桔青霉素的峰面积；

$\rho$ ——标准工作液中桔青霉素的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准工作液中桔青霉素的峰面积；

$m$ ——最终样液所代表的试样量，单位为克 $(\mathrm{g})$ 。

计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，结果保留两位有效数字。

# 14 精密度

在重复条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

方法检出限为 $3\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$

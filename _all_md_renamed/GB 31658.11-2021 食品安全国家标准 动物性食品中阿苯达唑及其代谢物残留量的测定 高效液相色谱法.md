# 1 范围

本文件规定了动物性食品中阿苯达唑及其代谢物残留量检测的制样和高效液相色谱测定方法。

本文件适用于猪、牛、羊、鸡的肌肉、肝脏、肾脏、脂肪组织和牛奶中阿苯达唑、阿苯达唑砜、阿苯达唑亚砜、阿苯达唑-2-氨基砜残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语与定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的阿苯达唑及其代谢物经碱化、乙酸乙酯提取（脂肪试料采用盐酸水-乙腈溶液提取，牛奶试料采用乙腈和乙酸乙酯提取），酸性氧化铝固相萃取柱及混合型阳离子交换固相萃取柱净化，高效液相色谱-荧光检测器检测，外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水

# 5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯  
5.1.3 二甲基亚砜 $(\mathrm{C}_2\mathrm{H}_6\mathrm{OS})$   
5.1.4 乙酸乙酯 $(\mathrm{C_4H_8O_2})$   
5.1.5 正己烷 $(\mathrm{C_6H_{14}})$   
5.1.6 无水硫酸钠 $\mathrm{(Na_2SO_4)}$   
5.1.7 碳酸钠 $\mathrm{(Na_2CO_3)}$   
5.1.8 盐酸（HCl）。  
5.1.9 氢氧化钠(NaOH)。  
5.1.10 焦亚硫酸钠 $(\mathrm{Na}_{2}\mathrm{S}_{2}\mathrm{O}_{5})$ 。  
5.1.11 乙二胺四乙酸二钠 $(\mathrm{C_{10}H_{14}N_2Na_2O_8\cdot 2H_2O})$   
5.1.12 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。  
5.1.13 乙酸铵 $\mathrm{(CH_3COONH_4)}$

5.1.14 冰醋酸 $\mathrm{(CH_3COOH)}$

5.2 溶液配制

5.2.1 $2\%$ 碳酸钠溶液：取碳酸钠 $2.0\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$

5.2.2 $0.2\mathrm{mol} / \mathrm{L}$ 盐酸溶液：取盐酸 $1.8~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ 。

5.2.3 $0.2\mathrm{mol} / \mathrm{L}$ 盐酸乙腈溶液：取 $0.2\mathrm{mol} / \mathrm{L}$ 盐酸溶液 $10~\mathrm{mL}$ ，加乙腈 $90~\mathrm{mL}$ ，混匀。

5.2.4 $10\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：取氢氧化钠 $4.0\mathrm{g}$ ，用水溶解并稀释至 $10\mathrm{mL}$ ，冷却至室温。

5.2.5 $0.004\mathrm{g / mL}$ 焦亚硫酸钠溶液：取焦亚硫酸钠 $0.4\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。

5.2.6 $0.1\mathrm{mol} / \mathrm{L}$ 乙二胺四乙酸二钠溶液：取乙二胺四乙酸二钠 $3.36\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。

5.2.7 $10\%$ 氨化乙腈溶液：取氨水 $10~\mathrm{mL}$ ，加乙腈 $90~\mathrm{mL}$ ，混匀。

5.2.8 $0.5\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液：取 $1.93\mathrm{g}$ 乙酸铵，用水溶解并稀释至 $500~\mathrm{mL}$ ，用冰醋酸调节 $\mathrm{pH}$ 至5.0，混匀。

5.2.9 复溶液：取乙腈 $10\mathrm{mL}$ 、甲醇 $10\mathrm{mL}$ ，加 $0.5\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液 $80~\mathrm{mL}$ ，混匀。

5.2.10 乙腈-二甲基亚砜(9：1)：取乙腈 $90~\mathrm{mL}$ 、二甲基亚砜 $10~\mathrm{mL}$ ，混匀。

5.3 标准品

5.3.1 阿苯达唑(Albendazole, $\mathrm{C_{12}H_{15}N_3O_2S}$ , CAS号:54965-21-8):含量 $\geq 99.0\%$ 。

5.3.2 阿苯达唑砜（Albendazole Sulfone, $\mathrm{C_{12}H_{15}N_3O_4S}$ , CAS号：75184-71-3）：含量 $\geq 98.3\%$ 。

5.3.3 阿苯达唑亚砜(Albendazole Sulfoxide, $\mathrm{C_{12}H_{15}N_3O_3S}$ ,CAS号：54029-12-8)：含量 $\geq 98.5\%$ 。

5.3.4 阿苯达唑-2-氨基砜（Albendazole-2-aminosulfone， $\mathrm{C_{10}H_{13}N_3O_2S}$ ，CAS号：80983-34-2）：含量 $\geq 90.0\%$ 。

5.4 标准溶液的制备

5.4.1 标准储备液：取阿苯达唑、阿苯达唑砜、阿苯达唑亚砜和阿苯达唑-2-氨基砜标准品各约 $10\mathrm{mg}$ ，精密称定，用乙腈-二甲基亚砜(9：1)溶解并稀释定容至 $100~\mathrm{mL}$ 容量瓶，配制成浓度为 $100~\mu \mathrm{g / mL}$ 的阿苯达唑、阿苯达唑砜、阿苯达唑亚砜和阿苯达唑-2-氨基砜标准储备液。 $-18^{\circ}C$ 以下保存，有效期1个月。

5.4.2 混合标准中间液：分别准确移取阿苯达唑标准储备液 $2\mathrm{mL}$ 、阿苯达唑砜标准储备液 $0.2\mathrm{mL}$ 、阿苯达唑亚砜标准储备液 $2\mathrm{mL}$ 和阿苯达唑-2-氨基砜标准储备液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，配制成阿苯达唑浓度为 $20~\mu \mathrm{g / mL}$ 、阿苯达唑砜浓度为 $2\mu \mathrm{g / mL}$ 、阿苯达唑亚砜浓度为 $20~\mu \mathrm{g / mL}$ 和阿苯达唑-2-氨基砜浓度为 $10~\mu \mathrm{g / mL}$ 的混合标准中间液。现用现配。

5.5 材料

5.5.1 酸性氧化铝固相萃取柱： $1\mathrm{g} / 3\mathrm{mL}$ ，填料为酸性氧化铝，或相当者。

5.5.2 混合型阳离子交换固相萃取柱： $150\mathrm{mg / 6mL}$ ，或相当者。

5.5.3 尼龙微孔滤膜： $0.22\mu \mathrm{m}$

6 仪器和设备

6.1 高效液相色谱仪：配荧光检测器  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和感量 $0.00001\mathrm{g}$   
6.3 组织匀浆器。  
6.4 涡旋混合器。  
6.5 涡旋振荡器。  
6.6 离心机：转速可达 $10000\mathrm{r / min}$   
6.7 固相萃取装置。  
6.8 氮吹仪。

# 7 试料的制备与保存

# 7.1 试料的制备

# 7.1.1 组织

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

# 7.1.2 牛奶

取适量新鲜或解冻的空白或供试牛奶样品，混合均匀。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

# 8.1.1 肌肉、肝脏、肾脏

取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 离心管，加 $2\%$ 的碳酸钠溶液 $0.5\mathrm{mL}$ 、二甲基亚砜 $1\mathrm{mL}$ 产 $0.004\mathrm{g / mL}$ 焦亚硫酸钠溶液 $1\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ 。加乙酸乙酯 $10~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，振荡 $5\mathrm{min}$ ， $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液。残渣加入乙酸乙酯 $10~\mathrm{mL}$ 重复提取1次，合并上清液， $40^{\circ}C$ 氮吹至近干。残渣加入正己烷 $5\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，再加入 $0.2\mathrm{mol / L}$ 盐酸-乙腈溶液 $2\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，静置分层，弃去上层正己烷，取下层溶液备用。

# 8.1.2 脂肪

取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 离心管， $70^{\circ}C$ 水浴加热使其融化，加 $2\%$ 的碳酸钠溶液 $0.5\mathrm{mL},0.2\mathrm{mol / L}$ 盐酸水-乙腈溶液 $5\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，振荡 $5\mathrm{min}$ ， $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液。残渣在 $70^{\circ}C$ 水浴加热使其融化，加 $0.2\mathrm{mol / L}$ 盐酸水-乙腈溶液 $5\mathrm{mL}$ 重复提取，合并上清液。加正己烷 $5\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，静置分层，弃去上层正己烷，加入正己烷 $5\mathrm{mL}$ 重复操作，取下层溶液备用。

# 8.1.3 牛奶

取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 离心管中，加 $10\mathrm{mol / L}$ 氢氧化钠溶液 $50~\mu \mathrm{L},0.004\mathrm{g / mL}$ 焦亚硫酸钠溶液 $0.5\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，加乙腈 $10\mathrm{mL}$ 、乙酸乙酯 $5\mathrm{mL}$ ，涡旋，振荡 $5\mathrm{min}$ ， $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，移取上清液。加无水硫酸钠 $5\mathrm{g}$ ，振荡 $5\mathrm{min}$ ， $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液， $40^{\circ}C$ 氮吹至近干。残渣加正己烷 $5\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，再加入 $0.2\mathrm{mol / L}$ 盐酸水-乙腈溶液 $2\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，静置分层，弃去上层正己烷，取下层溶液备用。

# 8.2 净化

酸性氧化铝固相萃取柱依次用 $0.2\mathrm{mol} / \mathrm{L}$ 盐酸水-乙腈溶液 $3\mathrm{mL},0.1\mathrm{mol} / \mathrm{L}$ 乙二胺四乙酸二钠溶液 $3\mathrm{mL}$ 活化，取备用液过柱，收集流出液，用 $0.2\mathrm{mol} / \mathrm{L}$ 盐酸-乙腈溶液 $1\mathrm{mL}$ 淋洗，合并流出液，加 $0.2\mathrm{mol} / \mathrm{L}$ 盐酸溶液 $8\mathrm{mL}$ ，混匀。再通过依次用甲醇 $3\mathrm{mL},0.1\mathrm{mol} / \mathrm{L}$ 乙二胺四乙酸二钠溶液 $3\mathrm{mL}$ 、水 $3\mathrm{mL},0.2\mathrm{mol} / \mathrm{L}$ 盐酸溶液 $3\mathrm{mL}$ 活化的混合型阳离子交换固相萃取柱，用 $0.2\mathrm{mol} / \mathrm{L}$ 盐酸溶液 $3\mathrm{mL}$ 、甲醇 $3\mathrm{mL}$ 淋洗，抽干 $1\mathrm{min},10\%$ 氨化乙腈溶液 $5\mathrm{mL}$ 洗脱，抽干，收集洗脱液， $40^{\circ}C$ 水浴氮气吹干。用复溶液 $1.0\mathrm{mL}$ 溶解残余物，过尼龙微孔滤膜后，供高效液相色谱测定（上机溶液应在 $72\mathrm{h}$ 内完成测定）。

# 8.3 标准曲线的制备

准确移取混合标准中间液适量，用复溶液稀释阿苯达唑和阿苯达唑亚砜浓度为 $0.05\mu \mathrm{g / mL}$

$0.1\mu \mathrm{g / mL},0.2\mu \mathrm{g / mL},0.5\mu \mathrm{g / mL},1\mu \mathrm{g / mL},2\mu \mathrm{g / mL}$ ，阿苯达唑砜浓度为 $0.005~\mu \mathrm{g / mL}$ 、 $0.01\mu \mathrm{g / mL},0.02\mu \mathrm{g / mL},0.05\mu \mathrm{g / mL},0.1\mu \mathrm{g / mL},0.2\mu \mathrm{g / mL}$ ，阿苯达唑-2-氨基砜浓度为 $0.025\mu \mathrm{g / mL},0.05\mu \mathrm{g / mL},0.1\mu \mathrm{g / mL},0.2\mu \mathrm{g / mL},0.5\mu \mathrm{g / mL},1\mu \mathrm{g / mL}$ 的系列标准工作溶液，供高效液相色谱测定。以待测物峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.4.2 测定法

取试料溶液和相应的标准溶液，作单点或多点校准，按外标法以色谱峰面积定量。标准工作液及试料溶液中阿苯达唑及其代谢物响应值应在仪器检测的线性范围之内。试料中阿苯达唑及其代谢物峰的保留时间与标准工作液相应峰的保留时间相对偏差应在 $\pm 2.5\%$ 以内。阿苯达唑及其代谢物标准溶液色谱图见附录A。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试料中待测物残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V}{A _ {\mathrm {S}} \times m} \tag {1}
$$

式中：

$X$ ——试料中相应的阿苯达唑或各代谢物残留量的数值，单位为毫克每千克 $(\mathrm{mg / kg})$

A ——试料中相应的阿苯达唑或各代谢物的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中相应的阿苯达唑或各代谢物的峰面积；

$C_{\mathrm{S}}$ ——标准溶液中相应的阿苯达唑或各代谢物浓度的数值，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

V——最终试料定容体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ ——供试试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法阿苯达唑、阿苯达唑砜、阿苯达唑亚砜、阿苯达唑-2-氨基砜在猪、牛、羊、鸡的肌肉、肝脏、肾脏、脂肪组织和牛奶中检测限均为 $0.02\mathrm{mg / kg}$ ，定量限均为 $0.05\mathrm{mg / kg}$ 。

# 10.2 准确度

本方法阿苯达唑、阿苯达唑砜、阿苯达唑亚砜、阿苯达唑-2-氨基砜在猪、牛、羊、鸡的肌肉、肝脏、肾脏、脂肪组织和牛奶 $0.05\mathrm{mg / kg}\sim 10\mathrm{mg / kg}$ 添加浓度的回收率为 $60\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

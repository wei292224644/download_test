# 1 范围

本标准规定了食品中产志贺毒素大肠埃希氏菌(Shiga toxin-producing Escherichia coli, STEC)的检验方法。

本标准适用于食品中产志贺毒素大肠埃希氏菌的检验。

# 2 设备和材料

除微生物实验室常规灭菌及培养设备外，其他设备和材料如下。

# 5 操作步骤

# 5.1 样品制备

# 5.1.1 样品保存

样品采集后应尽快送往实验室进行检测，如不能及时检测，应将待检样品根据样品贮存要求进行保存，无特殊要求的，应置于 $2\mathrm{^\circ C}\sim 5\mathrm{^\circ C}$ 冰箱中冷藏， $24\mathrm{h}$ 内完成检测。

# 5.1.2 固态或半固态样品

以无菌操作称取检样 $25\mathrm{g}$ ，加入装有 $225\mathrm{mL}$ EC(或mTSB)的无菌带滤网均质袋内，用拍击式均质器均质 $1\mathrm{min} \sim 2\mathrm{min}$ ；或放入盛有 $225\mathrm{mL}$ EC(或mTSB)的无菌均质杯中，用旋转刀片式均质器以 $8000\mathrm{r / min} \sim 10000\mathrm{r / min}$ 均质 $1\mathrm{min} \sim 2\mathrm{min}$ ，制成 $1:10$ 的样品匀液。

# 5.1.3 液态样品

以无菌操作量取检样 $25~\mathrm{mL}$ ，加入装有 $225~\mathrm{mL}$ EC(或mTSB)的无菌均质袋或无菌锥形瓶中，充分混匀制成 $1:10$ 的样品匀液。

# 5.2 增菌

将5.1制备的样品匀液 $44^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 下培养 $18\mathrm{h} \sim 24\mathrm{h}$ 。同时设置对照试验，以携带 $stx1$ 和/或 $stx2$ 的大肠埃希氏菌标准菌株在EC(或mTSB)中培养为阳性对照，以不携带 $stx1$ 和 $stx2$ 的大肠埃希氏菌标准菌株在EC(或mTSB)中培养为阴性对照，并做培养基空白对照。

# 5.3 $stx$ 毒力基因初筛

# 5.3.1 试验环境与过程控制

实时荧光定量PCR试验环境条件和过程控制应按GB/T27403—2008规定的要求执行。

# 5.3.2 DNA模板制备

混匀后吸取 $1\mathrm{mL}$ 增菌液(油脂含量高的样品不混匀，在距离液面下 $1\mathrm{cm}$ 处吸取)置于 $1.5\mathrm{mL}$ EP管中， $500g$ 离心 $1\mathrm{min}$ 。取上清液(避免带入底层沉淀物)至新的EP管中， $10000g$ 离心 $5\mathrm{min}$ ，弃去上清液。向沉淀中加入 $500\mu\mathrm{L}$ 无菌生理盐水，重悬， $10000g$ 离心 $3\mathrm{min}$ ，弃去上清液。向沉淀中加入 $100\mu\mathrm{L}$ 的 $1\times \mathrm{TE}$ 缓冲液，重悬(若样本为生肉制品，则用 $5\%$ 的Chelex-100溶液替代TE)。 $99^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 加热 $15\mathrm{min}$ ，室温冷却 $2\mathrm{min}$ ， $10000g$ 离心 $4\mathrm{min}$ ，取上清液作为DNA模板。 $4\mathrm{h}$ 内完成分析，若不能及时分析则于 $-20^{\circ}\mathrm{C}$ 短暂保存。

注：也可用商品化DNA提取试剂盒按其说明书要求提取制备DNA模板。

# 5.3.3.4 实时荧光定量PCR反应程序

预变性： $95^{\circ}\mathrm{C}10\mathrm{min}$ ；变性： $95^{\circ}\mathrm{C}15\mathrm{s}$ ；退火： $60^{\circ}\mathrm{C}40\mathrm{s}$ ；延伸： $72^{\circ}\mathrm{C}1\mathrm{min}40$ 个循环。在延伸阶段于3条荧光检测通道中，分别采集已标记的荧光信号。

# 5.3.3.5 结果有效性原则

若同一批次实验中同时满足下列要求，则实时荧光定量PCR实验结果有效；否则实时荧光定量PCR实验结果无效，需重新进行实验。

a) 空白对照: $stx1$ 、 $stx2$ 和 $16 \mathrm{~S} \mathrm{rDNA}$ 基因均无荧光对数增长 (16S rDNA 基因可能在 35 个循环后出现较弱的荧光对数增长);  
b）阴性对照： $stx1$ 和 $stx2$ 均无荧光对数增长；16S rDNA基因有荧光对数增长，相应的 $Ct$ 值 $< 30$   
c）阳性对照：stx1、stx2和16S rDNA基因均有荧光对数增长，相应的 $\mathrm{Ct}$ 值 $< 30$ 。

# 5.3.3.6 实时荧光定量PCR结果判定

在符合5.3.3.5结果有效性原则的情况下，待检样品进行检测时，若：

a) $stx1$ 和 $stx2$ 的 $\mathrm{Ct}$ 值 $\geq 40$ ，则判定为 $stx$ 阴性；  
b） $stx1$ 或 $stx2$ 的 $\mathrm{Ct}$ 值 $< 35$ ，16S rDNA 基因的 $\mathrm{Ct}$ 值 $< 35$ ，则判定为 $stx$ 阳性；  
c） $stx1$ 和 $stx2$ 的 $\mathrm{Ct}$ 值均 $\geq 35$ 且 $< 40$ ，应重新进行实时荧光定量PCR扩增实验。再次扩增后 $stx1$ 或 $stx2$ 的 $\mathrm{Ct}$ 值 $< 40,16\mathrm{S}$ rDNA基因的 $\mathrm{Ct}$ 值 $< 35$ ，判定为 $stx$ 阳性；若 $stx1$ 和 $stx2$ 的 $\mathrm{Ct}$ 值 $\geq 40$ ，则判定为 $stx$ 阴性。

判定为 $st.x$ 阴性的样品，按流程报告结果为“ $25\mathrm{g(mL)}$ 样品中未检出产志贺毒素大肠埃希氏菌”。

判定为 $stx$ 阳性的样品增菌液按5.4、5.5的规定分别进行酸处理及免疫磁珠富集处理。

# 5.4 增菌液酸处理

吸取 $450\mu \mathrm{L}5.3$ 中 $stx$ 阳性的样品增菌液至 $1.5\mathrm{mL}$ 无菌EP管， $10000g$ 离心 $2\mathrm{min}$ ，弃上清，菌体沉淀重悬于 $450~\mu \mathrm{L}$ 的E-buffer溶液中，加入 $25~\mu \mathrm{L}$ 的 $1\mathrm{mol / L}$ 盐酸 $(\mathrm{pH}2.0\sim 2.5)$ ，室温条件振荡 $1\textrm{h}$ 用 $10~\mu \mathrm{L}$ 无菌接种环划线接种于STEC显色培养基或麦康凯琼脂培养基平板， $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 培养 $18\mathrm{h}\sim$ $24\mathrm{h}$ 。

吸取上述酸处理液 $100\mu \mathrm{L}$ 至 $900\mu \mathrm{L}$ E-buffer溶液进行稀释，涡旋混匀后用 $10~\mu \mathrm{L}$ 无菌接种环划线接种于STEC显色培养基或麦康凯琼脂培养基平板， $36^{\circ}C\pm 1^{\circ}C$ 培养 $18\mathrm{h}\sim 24\mathrm{h}$ 。

# 5.5 7种重要血清型STEC免疫磁珠富集处理

5.5.1 5.3 中判定为 $stx$ 阳性的样品增菌液除进行 5.4 酸处理外, 还需同时进行免疫磁珠富集, 按生产商提供的使用说明对 7 种重要血清型 (O26、O45、O103、O111、O121、O145、O157) 的 STEC 进行了富集处理, 并设置具有相应血清型的标准菌株作为阳性对照组。当生产商使用说明与下面的描述存在偏差时, 按生产商使用说明进行操作, 每个样品换用 1 根吸管避免交叉污染。  
5.5.2 将微量离心管按样品和标准菌株编号，每个样品使用7个微量离心管(7种免疫磁珠各1管)，置于磁板架上。将7种STEC免疫磁珠悬液试剂轻柔振荡混匀后，分别加至相应编号的微量离心管中。  
5.5.3 吸取 $stx$ 阳性增菌液 $8 \mathrm{~mL}$ 加入 $15 \mathrm{~mL}$ 离心管中, $10000 g$ 离心 $3 \mathrm{~min}$ , 弃去上清液, 加入 $3 \mathrm{~mL}$ 无菌 PBS 重悬沉淀并涡旋混匀, $10000 g$ 离心 $3 \mathrm{~min}$ , PBS 洗菌步骤重复 $2 \sim 3$ 遍后弃去上清液, 菌体沉淀备用。  
5.5.4 向 5.5.3 菌体沉淀中加入 $800 \mu \mathrm{L}$ 无菌 PBS 缓冲液重悬, 涡旋混匀后分别吸取 $100 \mu \mathrm{L}$ 菌悬液至

5.5.2 加有免疫磁珠的 7 个微量离心管中, 并盖好离心管盖。每个样品更换加样吸头, 标准菌株必须与样品分开进行实验, 以避免交叉污染。每个样品均需进行 7 种血清型免疫磁珠富集。  
5.5.5 结合：颠倒混匀数次。 $37^{\circ} \mathrm{C}$ 条件下，在混匀仪上孵育 $60 \mathrm{~min}$ ，速度设置宜轻柔，使目标菌体与免疫磁珠充分接触。  
5.5.6 捕获：孵育结束后，将磁板插入磁板架中捕获磁珠，在 $3\mathrm{min}$ 内不断倾斜磁板架，确保悬液中与盖子上的免疫磁珠全部被收集，在微量离心管管壁中间位置可见圆形或椭圆形棕色聚集物。  
5.5.7 磁珠纯化：小心将离心管盖打开，从免疫磁珠聚集物的对侧轻轻移取上清液。当液面接近免疫磁珠聚集物附近时，应缓慢吸液直至液面远离聚集物附近，确保免疫磁珠聚集物不被吸走。如吸取的上清液内含有磁珠，则将其放回微量离心管，并重复5.5.6步骤。此步骤为纯化关键步骤，需要谨慎完成。  
5.5.8 洗涤：向去除液体后的微量离心管中加入 $1\mathrm{mL}$ 无菌PBS缓冲液，涡旋混匀，重复5.5.6和5.5.7步骤的要求回收磁珠。  
5.5.9 重复步骤5.5.8一次。  
5.5.10 免疫磁珠重悬：移去磁板，将免疫磁珠聚集物重新悬浮于 $100\mu \mathrm{L}$ 无菌PBS缓冲液。  
5.5.11 重悬混匀后的免疫磁珠悬液涂布或划线接种于STEC显色培养基或麦康凯琼脂培养基平板， $36\mathrm{^\circ C}\pm 1\mathrm{^\circ C}$ 培养 $18\mathrm{h}\sim 24\mathrm{h}$

# 5.6 STEC菌株的分离和复核鉴定

# 5.6.1 分离

观察STEC显色培养基或麦康凯平板上的菌落生长情况及菌落形态。在STEC显色培养基上的菌落特征按产品说明书进行判定；在麦康凯琼脂培养基平板上，典型大肠埃希氏菌为桃红色菌落和无色或淡粉色菌落。分别挑取经酸处理或磁珠富集处理的典型菌落(均不少于5个，少于5个全部挑取)，划线接种于TSA琼脂平板进行再次分离纯化。

# 5.6.2 $stx$ 毒力基因鉴定

# 5.6.2.1 DNA提取

使用接种环挑取TSA琼脂上培养 $18\mathrm{h}\sim 24\mathrm{h}$ 的可疑单菌落，悬浮于 $50~\mu \mathrm{L}$ 无菌水中，充分混匀制成菌悬液， $99^{\circ}C\pm 1$ ℃加热 $15\mathrm{min}$ ，室温冷却 $2\mathrm{min}$ ， $16000g$ 离心 $4\mathrm{min}$ ，取上清液作为DNA模板。

# 5.6.2.2 $stx$ 毒力基因和16S rDNA检测

$stx$ 毒力基因和16S rDNA的检测及结果判定按5.3.3的要求进行。

# 5.6.2.3 菌株生化鉴定

$stx$ 毒力基因和 $16\mathrm{S}$ rDNA检测为阳性的菌株，进行大肠埃希氏菌典型生化鉴定。三糖铁斜面产酸或不产酸，底层产酸，靛基质阳性， $\mathrm{H}_2\mathrm{S}$ 阴性和尿素酶阴性的培养物为大肠埃希氏菌。三糖铁斜面底层不产酸，或 $\mathrm{H}_2\mathrm{S},\mathrm{KCN}$ 、尿素酶有任一项为阳性的培养物，均非大肠埃希氏菌。

菌株鉴定可选择全自动微生物生化鉴定系统、微生物生化鉴定试剂条(盒)或基质辅助激光解析电离飞行时间质谱仪进行鉴定，按仪器或试剂盒的使用说明进行操作及判定。

# 5.7 血清学试验（选做项目）

条件允许的实验室，可对分离到的 $stx$ 阳性大肠埃希氏菌进行血清学试验。按照生产商提供的使用说明进行O抗原和H抗原鉴定。

# 6 结果与报告

初筛 $stx$ 阴性，报告“ $25\mathrm{g(mL)}$ 样品中未检出产志贺毒素大肠埃希氏菌”。

初筛 $stx$ 阳性，但未分离到 $stx$ 阳性大肠埃希氏菌，报告“ $25\mathrm{g}(\mathrm{mL})$ 样品中未检出产志贺毒素大肠埃希氏菌”。

初筛 $stx$ 阳性，且分离到 $stx$ 阳性菌株，经鉴定为大肠埃希氏菌，报告“ $25\mathrm{g}(\mathrm{mL})$ 样品中检出产志贺毒素大肠埃希氏菌”。

# 1 范围

本标准适用于以海绵铁为原料，与一氧化碳反应合成五羰基铁，五羰基铁经热分解制得的食品添加剂羰基铁粉。

# 2 化学名称、分子式和相对分子质量

# 2.1 化学名称

羰基铁

# 2.2 分子式

Fe

# 2.3 相对分子质量

55.85（按2007年国际相对原子质量）

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A. 3.1.1 盐酸溶液: $2 + 8$ 。  
A.3.1.2 铁氰化钾溶液： $100\mathrm{g / L}$   
A.3.1.3 氢氧化钠溶液： $40\mathrm{g / L}$

# A.3.2 鉴别方法

将少量试样溶于盐酸溶液中，有氢气逸出，停止后加入铁氰化钾溶液，生成不溶于盐酸溶液的黑蓝色沉淀。此沉淀可被氢氧化钠溶液分解，生成绿白色沉淀，颜色很快转变为绿色，振荡后变为棕色。

# A.4 铁（Fe）含量的测定

# A.4.1 方法提要

试样在隔绝空气条件下加酸溶解后，以1,10-菲啰啉-亚铁作指示剂，用硫酸铈标准滴定溶液滴定，溶液由红色变为淡蓝色，根据所消耗的硫酸铈标准溶液的量计算铁含量。

# A.4.2 试剂和材料

A.4.2.1 硫酸溶液： $3 + 100$   
A.4.2.2 硫酸铈标准滴定溶液： $c[\mathrm{Ce}(\mathrm{SO}_4)_2] = 0.1\mathrm{mol / L}$   
A.4.2.3 1,10-菲啰啉-亚铁指示液。

# A.4.3 分析步骤

称取约 $0.2 \mathrm{~g}$ 试样, 精确至 $0.0002 \mathrm{~g}$ , 置于 $250 \mathrm{~mL}$ 锥形瓶, 加 $50 \mathrm{~mL}$ 硫酸溶液, 盖上连有本生阀的瓶塞 (本生阀的制作方法是在塞子上插入一根连有一短段橡胶管的玻璃管, 橡胶管的侧面有一狭长的切口, 其另一端插入一段玻璃棒, 使气体能够逸出, 而空气不能进入)。在蒸汽浴上加热锥形瓶, 使试样溶解, 冷却, 加 $50 \mathrm{~mL}$ 沸腾并冷却的水稀释, 加 2 滴 1,10-菲啰啉-亚铁指示液, 用硫酸铈标准滴定溶液滴定至红色变为淡蓝色, 即为终点。

同时进行空白试验。空白试样溶液除不加试样外，其他操作和加入的试剂与试样溶液相同。

# A.4.4 结果计算

铁（Fe）含量的质量分数 $w_{1}$ ，按式（A.1）计算：

$$
w _ {1} = \frac {\left(V _ {1} - V _ {0}\right) c M}{m \times 1000} \times 100 \% \dots \tag{A.1}
$$

式中：

$V_{1}$ ——滴定试样溶液所消耗的硫酸铈标准滴定溶液的体积的数值，单位为毫升（ $\mathrm{mL}$ ）；

$V_{0}$ ——滴定空白试样溶液所消耗的硫酸铈标准滴定溶液的体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫酸铈标准滴定溶液的浓度的准确数值, 单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ;

$m$ ——试样质量的数值，单位为克（g）；

$M$ ——铁（Fe）的摩尔质量的数值，单位为克每摩尔（ $M = 55.85$ ）；

1000——换算因子。

实验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$

# A.5 砷（As）的测定

称取 $1.00 \mathrm{~g} \pm 0.01 \mathrm{~g}$ 试样，置于锥形瓶中，加入 $25 \mathrm{~mL}$ 硫酸溶液（ $3+100$ ），蒸汽浴上加热至氢气停止逸出。冷却，加 $4 \mathrm{~mL}$ 盐酸，以下操作按照 GB/T 5009.76 规定的方法进行测定。

# A.6 铅（Pb）的测定

# A.6.1 试剂和材料

A. 6.1.1 盐酸。

A.6.1.2 硝酸。

A.6.1.3 甲基异丁基甲酮。

A.6.1.4 盐酸溶液： $4 + 5$

A.6.1.5 硝酸溶液： $1 + 1$

A. 6.1.6 抗坏血酸-碘化钠溶液: 称取 $20 \mathrm{~g}$ 抗坏血酸和 $38.5 \mathrm{~g}$ 碘化钠, 加水溶解, 转移到 $200 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度, 摇匀。

A. 6.1.7 三正辛基氧膦溶液: 称取 $5.0 \mathrm{~g}$ 三正辛基氧膦转移到 $100 \mathrm{~mL}$ 容量瓶中, 用甲基异丁基甲酮溶解并稀释至刻度, 摇匀。

警告：三正辛基氧磷溶液具有刺激性，避免与眼睛、皮肤和衣服接触。

A.6.1.8 铅（Pb）标准溶液： $0.002\mathrm{mg / mL}$

A.6.1.9 水：符合GB/T6682—2008中二级水的规定。

# A.6.2 仪器和设备

原子吸收分光光度计：配有铅空心阴极灯。

# A.6.3 分析步骤

# A.6.3.1 玻璃仪器的清洗

所有配制溶液的玻璃容器用前应在硝酸溶液中浸泡 $30\mathrm{min}$ ，并用自来水反复冲洗，最后用水冲洗干净。

# A.6.3.2 试样溶液的制备

称取 $1.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $50~\mathrm{mL}$ 烧杯中，盖上表面皿，缓慢地加入 $8\mathrm{mL}$ 盐酸、 $2\mathrm{mL}$ 硝酸，待反应平静后，在蒸汽浴上蒸发至干，冷却。加入 $10~\mathrm{mL}$ 盐酸溶液溶解残渣，如果需要可适当加热。溶解后，用 $10~\mathrm{mL}$ 水冲洗转移到 $125~\mathrm{mL}$ 分液漏斗中，加入 $20~\mathrm{mL}$ 抗坏血酸-碘化钠溶液、 $5\mathrm{mL}$ 三正辛基氧膦溶液，振摇 $30~\mathrm{s}$ ，静置分层，弃去水相，将有机相放入 $10~\mathrm{mL}$ 具塞刻度管中，备用。

# A.6.3.3 标准参比溶液的制备

取 $2 \mathrm{~mL}$ 铅标准溶液于 $50 \mathrm{~mL}$ 烧杯中，盖上表面皿，缓慢地加入 $8 \mathrm{~mL}$ 盐酸、 $2 \mathrm{~mL}$ 硝酸，在蒸汽浴上蒸发至干，冷却。加入 $10 \mathrm{~mL}$ 盐酸溶液溶解残渣，用 $10 \mathrm{~mL}$ 水冲洗转移到 $125 \mathrm{~mL}$ 分液漏斗中，加入 $20 \mathrm{~mL}$ 抗坏血酸-碘化钠溶液、 $5 \mathrm{~mL}$ 三正辛基氧膦溶液，振摇 $30 \mathrm{~s}$ ，静置分层，弃去水相，将有机相放

入 $10\mathrm{mL}$ 具塞刻度管中，备用。

# A.6.4 测定

在波长 $283.3 \mathrm{~nm}$ 处，使用空气-乙炔火焰，将原子吸收分光光度计调至最佳工作状态，以甲基异丁基甲酮调零，分别测量标准参比溶液、试样溶液的吸光度，试样溶液的吸光度不应大于标准参比溶液的吸光度。

# A.7 汞（Hg）的测定

称取 $2.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $250~\mathrm{mL}$ 烧杯中，加入 $20~\mathrm{mL}$ 硝酸溶液（1+2），在蒸汽浴上消解 $45\mathrm{min}$ 加 $5\mathrm{mL}$ 盐酸溶液（1+3），继续在蒸汽浴上加热直到试样完全溶解。冷却至室温，过滤，其他操作按照GB/T5009.17规定的方法进行测定。

# A.8 酸不溶物的测定

# A.8.1 试剂和材料

A.8.1.1 硫酸溶液： $3 + 100$

A.8.1.2 氯化钡（ $\mathrm{BaCl_2\cdot 2H_2O}$ ）溶液： $122~\mathrm{g / L}$

# A.8.2 仪器和设备

A.8.2.1 电热恒温干燥箱：温度可控制在 $105\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}$ 。  
A.8.2.2 玻璃砂芯坩埚：滤板孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$

# A.8.3 分析步骤

称取 $1.00 \mathrm{~g} \pm 0.01 \mathrm{~g}$ 试样，置于 $100 \mathrm{~mL}$ 烧杯中，加 $25 \mathrm{~mL}$ 硫酸溶液。蒸汽浴上加热，直至氢气逸尽。用预先于 $105^{\circ} \mathrm{C} \pm 2^{\circ} \mathrm{C}$ 干燥至质量恒定的玻璃砂芯坩埚过滤。用水洗涤至滤液无硫酸盐反应（用氯化钡溶液检验），置于电热恒温干燥箱中，于 $105^{\circ} \mathrm{C} \pm 2^{\circ} \mathrm{C}$ 下干燥 $1 \mathrm{~h}$ ，冷却，称量。

# A.8.4 结果计算

酸不溶物的质量分数 $w_{2}$ , 按式 (A.2) 计算:

$$
w _ {2} = \frac {m _ {2} - m _ {1}}{m} \times 100 \% \dots \tag{A.2}
$$

式中：

$m_{1}$ ——玻璃砂芯坩埚的质量的数值，单位为克（g）；

$m_{2}$ ——玻璃砂芯坩埚和不溶物的质量的数值，单位为克（g）；

$m$ ——试样质量的数值，单位为克（g）。

实验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.1\%$ 。

# A.9 细度的测定

# A.9.1 仪器和设备

A.9.1.1 振筛机：摇动频率 $270\mathrm{r / min}\sim 300\mathrm{r / min}$ ，振动频率140次/min～160次/min。  
A.9.1.2 试验筛： $\phi 200\times 50 - 0.075 / 0.05$ ， $\phi 200\times 50 - 0.045 / 0.032$ GB/T6003.1—1997。

# A.9.2 分析步骤

将试验筛按筛孔尺寸由小至大、自下而上安装好。称取约 $100\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，倒入最上层试验筛中，盖上筛盖，固定在振筛机上，开启振筛机，振筛 $15\mathrm{min}$ ，将筛余物缓慢倒入已称重的玻璃纸上，用干燥软毛刷轻轻刷扫，最后轻敲收集筛余物，准确称量。

# A.9.3 结果计算

细度的质量分数 $w_{3}$ , 按式（A.3）计算:

$$
w _ {3} = \frac {m - m _ {1}}{m} \times 100 \% \dots \tag{A.3}
$$

式中：

$m$ ——试样质量的数值，单位为克（g）；

$m_{1}$ ——筛余物的质量的数值，单位为克（g）。

实验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $2\%$ 。

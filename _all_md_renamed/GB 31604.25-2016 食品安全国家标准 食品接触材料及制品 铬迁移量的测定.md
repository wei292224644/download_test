# 前言

本标准代替GB/T5009.80—2003《食品容器内壁聚四氟乙烯涂料卫生标准的分析方法》、GB/T5009.81—2003《不锈钢食具容器卫生标准的分析方法》、SN/T2597—2010《食品接触材料 高分子材料铅、镉、铬、砷、锑、锗迁移量的测定 电感耦合等离子体原子发射光谱法》和SN/T2829—2011《出口食品接触材料金属材料 食品模拟物中重金属含量的测定 电感耦合等离子体发射光谱法》中铬迁移量的测定。

本标准与GB/T5009.81—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 铬迁移量的测定”；  
——增加了电感耦合等离子体质谱法；  
——增加了电感耦合等离子体发射光谱法。

# 食品安全国家标准

# 食品接触材料及制品 铬迁移量的测定

# 1 范围

本标准规定了食品接触材料及其制品在食品模拟物中浸泡后铬迁移量测定的石墨炉原子吸收光谱法、电感耦合等离子体质谱法、电感耦合等离子体发射光谱法和二苯碳酰二肼比色法。

本标准适用于食品接触材料及其制品中铬迁移量的测定。

# 第一法 石墨炉原子吸收光谱法

# 2 原理

采用食品模拟物浸泡食品接触材料及制品中预期与食品接触的部分，浸泡液经石墨炉原子化，在 $357.9\mathrm{nm}$ 处测定的吸收值在一定浓度范围内与铬含量成正比，与标准系列比较定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为 $\mathrm{GB / T6682}$ 规定的二级水。

# 3.1 试剂

3.1.1 硝酸 $(\mathrm{HNO}_3)$ 。  
3.1.2 磷酸二氢铵 $(\mathrm{NH_4H_2PO_4})$ 。  
3.1.3 配制食品模拟物所需试剂：依据GB31604.1的规定。

# 3.2 试剂配制

3.2.1 食品模拟物：按照GB5009.156的规定配制。   
3.2.2 硝酸溶液 $(5 + 95)$ ：吸取 $5\mathrm{mL}$ 硝酸，加至 $950~\mathrm{mL}$ 水，混匀。  
3.2.3 磷酸二氢铵溶液 $(20\mathrm{g / L})$ ：称取 $2.0\mathrm{g}$ 磷酸二氢铵，用水溶解，定容至 $100~\mathrm{mL}$ 。

# 3.3 标准品

重铬酸钾 $\mathrm{(K_2Cr_2O_7}$ ，CAS号：7778-50-9）：纯度 $>99.99\%$ ，或经国家认证并授予标准物质证书的一定浓度的铬标准溶液。

# 3.4 标准溶液配制

3.4.1 铬标准储备液 $(1000\mathrm{mg / L})$ ：准确称取 $1.4315\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )重铬酸钾，溶于水中，移入 $500~\mathrm{mL}$ 容量瓶中，用硝酸溶液 $(5 + 95)$ 稀释至刻度。混匀。  
3.4.2 铬标准中间液 $(1.00\mathrm{mg / L})$ ：吸取铬标准储备液 $1.00~\mathrm{mL}$ 于 $1000~\mathrm{mL}$ 容量瓶中，加硝酸溶液

$(5 + 95)$ 至刻度，混匀。

3.4.3 铬标准系列溶液：分别吸取铬标准中间液 $0\mathrm{mL}, 0.200\mathrm{mL}, 0.400\mathrm{mL}, 0.800\mathrm{mL}, 1.20\mathrm{mL}$ 、 $1.60\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中，加相应食品模拟物至刻度，混匀。使标准溶液系列浓度分别为 $0\mu \mathrm{g} / \mathrm{L}$ 、 $2.00\mu \mathrm{g} / \mathrm{L}, 4.00\mu \mathrm{g} / \mathrm{L}, 8.00\mu \mathrm{g} / \mathrm{L}, 12.0\mu \mathrm{g} / \mathrm{L}, 16.0\mu \mathrm{g} / \mathrm{L}$ 。

注：可根据仪器的灵敏度、线性范围及浸泡液中铬的实际浓度确定标准溶液系列中铬的具体浓度。若所选食品模拟物为中性或碱性，则需添加适量硝酸使该溶液中硝酸浓度约为 $5\%$ （体积分数）。

# 4 仪器和设备

注：所有玻璃器皿均需硝酸溶液 $(1 + 5)$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

4.1 石墨炉原子吸收光谱仪：配石墨炉原子化器、铬空心阴极灯。  
4.2 分析天平：感量 $0.1\mathrm{mg}$

# 5 分析步骤

# 5.1 试样的预处理

根据待测样品的预期用途和使用条件，按照GB5009.156和GB31604.1规定的迁移试验方法及试验条件进行迁移试验。浸泡液经充分混匀后，取部分浸泡试液用于分析。若浸泡试液为中性或碱性，则添加适量硝酸使试液中硝酸浓度约为 $5\%$ （体积分数）。同时做试样空白试验。

# 5.2 测定

# 5.2.1 仪器参考条件

仪器参考条件见表A.1。

# 5.2.2 标准曲线制作

分别取 $10~\mu \mathrm{L}$ 铬标准系列溶液和 $5\mu \mathrm{L}$ 磷酸二氢铵溶液 $(20\mathrm{g / L})$ （可根据所使用的仪器确定最佳进样量)同时注入石墨炉中测定其吸光值，以标准系列浓度为横坐标，对应的吸光值为纵坐标制作标准曲线。

# 5.2.3 试样溶液测定

分别取 $10~\mu \mathrm{L}$ 空白溶液、试样浸泡液和 $5\mu \mathrm{L}$ 磷酸二氢铵溶液 $(20\mathrm{g / L})$ （可根据所使用的仪器确定最佳进样量）同时注入石墨炉中，原子化后测其吸光度值，与标准系列比较定量。

# 6 分析结果的表述

由标准曲线得到试样溶液中铬的浓度，扣除空白值后按GB5009.156进行迁移量计算，得到食品接触材料及制品中铬的迁移量。计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

方法检出限为 $0.4\mu \mathrm{g} / \mathrm{L}$ ，定量限为 $1.5\mu \mathrm{g} / \mathrm{L}$ 。

# 第四法 二苯碳酰二肼比色法

# 9 原理

以高锰酸钾氧化低价铬为高价铬 $(\mathrm{Cr}^{6+})$ ，加氢氧化钠沉淀铁，加焦磷酸钠隐蔽剩余铁等，利用二苯碳酰二肼与铬生成红色络合物，与标准系列比较定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的三级水。

# 10.1 试剂

10.1.1 硝酸 $\mathrm{(HNO_3)}$ ：优级纯。  
10.1.2 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。  
10.1.3 高锰酸钾 $\mathrm{(KMnO_4)}$ 。  
10.1.4 尿素 $\left[\mathrm{CO}\left(\mathrm{NH}_{2}\right)_{2}\right]$ 。  
10.1.5 亚硝酸钠 $(\mathrm{NaNO}_2)$ 。  
10.1.6 氢氧化钠(NaOH)。  
10.1.7 焦磷酸钠 $(\mathrm{Na}_4\mathrm{P}_2\mathrm{O}_7\cdot 10\mathrm{H}_2\mathrm{O})$   
10.1.8 二苯碳酰二肼 $(\mathrm{C_{13}H_{14}N_4O})$ 。  
10.1.9 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ 。  
10.1.10 配制食品模拟物所需试剂：依据GB31604.1的规定。

# 10.2 试剂配制

10.2.1 食品模拟物：按照GB5009.156的规定配制。   
10.2.2 硝酸溶液 $(5 + 95)$ ：吸取 $50~\mathrm{mL}$ 硝酸，加入 $950~\mathrm{mL}$ 水，混匀。  
10.2.3 硫酸溶液 $(2.5\mathrm{mol} / \mathrm{L})$ ：量取 $70~\mathrm{mL}$ 硫酸边搅拌边加入水中，放冷后用水稀释至 $500~\mathrm{mL}$ 。  
10.2.4 高锰酸钾溶液 $(3\mathrm{g} / \mathrm{L})$ ：称取 $0.3\mathrm{g}$ 高锰酸钾加水溶解，定容至 $100~\mathrm{mL}$ 。  
10.2.5 尿素溶液 $(200\mathrm{g / L})$ ：称取 $20.0\mathrm{g}$ 尿素，用水溶解，定容至 $100~\mathrm{mL}$ 。

10.2.6 亚硝酸钠溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 亚硝酸钠，用水溶解，定容至 $100~\mathrm{mL}$ 。  
10.2.7 饱和氢氧化钠溶液：称取 $110\mathrm{g}$ 氢氧化钠，溶于 $100~\mathrm{mL}$ 无二氧化碳的水中摇匀，注入聚乙烯容器中，密闭放至溶液澄清。  
10.2.8 焦磷酸钠溶液 $(50\mathrm{g / L})$ ：称取 $5\mathrm{g}$ 焦磷酸钠，用水溶解，定容至 $100~\mathrm{mL}$ 。  
10.2.9 二苯碳酰二肼溶液：称取 $0.5\mathrm{g}$ 二苯碳酰二肼溶于 $50~\mathrm{mL}$ 丙酮，加水 $50~\mathrm{mL}$ ，临用时配制，保存于棕色瓶中，如溶液颜色变深则不能使用。

# 10.3 标准品

重铬酸钾 $\mathrm{(K_2Cr_2O_7}$ ，CAS号：7778-50-9）：纯度 $>99.99\%$ ，或经国家认证并授予标准物质证书的一定浓度的铬标准溶液。

# 10.4 标准溶液配制

10.4.1 铬标准储备液 $(1000\mathrm{mg / L})$ ：准确称取 $1.4315\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )重铬酸钾，溶于水中，移入 $500~\mathrm{mL}$ 容量瓶中，用硝酸溶液 $(5 + 95)$ 稀释至刻度。混匀。  
10.4.2 铬标准使用液 $(10.0\mathrm{mg / L})$ ：吸取铬标准储备液 $1.00~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加相应食品模拟物至刻度。

# 11 仪器和设备

注：所有玻璃器皿均需硝酸溶液 $(1 + 5)$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

11.1 分光光度计。  
11.2 分析天平：感量 $0.1\mathrm{mg}$

# 12 分析步骤

# 12.1 试样的预处理

同5.1。

# 12.2 测定

# 12.2.1 标准曲线的制作

吸取铬标准使用液 $0\mathrm{mL}, 0.250\mathrm{mL}, 0.500\mathrm{mL}, 1.00\mathrm{mL}, 1.50\mathrm{mL}, 2.00\mathrm{mL}, 2.50\mathrm{mL}, 3.00\mathrm{mL}$ ，分别移入 $50.0\mathrm{mL}$ 容量瓶中，加相应食品模拟物至刻度。将标准系列溶液移入 $100\mathrm{mL}$ 锥形瓶中，加玻璃珠2粒，硫酸溶液 $(2.5\mathrm{mol} / \mathrm{L})$ ，高锰酸钾溶液 $(3\mathrm{g} / \mathrm{L})$ 数滴，混匀，加热煮沸至约 $30\mathrm{mL}$ [微红色消失，再加高锰酸钾溶液 $(3\mathrm{g} / \mathrm{L})$ 呈微红色]，放冷，加 $25.0\mathrm{mL}$ 尿素溶液 $(200\mathrm{g} / \mathrm{L})$ ，混匀，滴加亚硝酸钠溶液 $(100\mathrm{g} / \mathrm{L})$ 至微红色消失，加饱和氢氧化钠溶液呈碱性 $(\mathrm{pH} = 9)$ ，放置 $2\mathrm{h}$ 后过滤，滤液加水至 $100\mathrm{mL}$ ，混匀。取此液 $20.0\mathrm{mL}$ 于 $25\mathrm{mL}$ 比色管中，加 $1\mathrm{mL}$ 硫酸溶液 $(2.5\mathrm{mol} / \mathrm{L}), 1\mathrm{mL}$ 焦磷酸钠溶液 $(50\mathrm{g} / \mathrm{L})$ ，混匀，加 $2\mathrm{mL}$ 二苯碳酰二肼溶液，加水至 $25.0\mathrm{mL}$ ，混匀。放置 $5\mathrm{min}$ ，于 $540\mathrm{nm}$ 处测定吸光度。以吸光度为纵坐标，铬质量为横坐标，制作标准曲线。

# 12.2.2 试样溶液测定

取试样浸泡液 $50.0\mathrm{mL}$ 放入 $100~\mathrm{mL}$ 锥形瓶中，加玻璃珠2粒，硫酸溶液 $(2.5\mathrm{mol} / \mathrm{L})$ ，高锰酸钾溶液 $(3\mathrm{g} / \mathrm{L})$ 数滴，混匀，加热煮沸至约 $30~\mathrm{mL}$ [微红色消失，再加高锰酸钾溶液 $(3\mathrm{g} / \mathrm{L})$ 呈微红色]，放冷，

加 $25.0~\mathrm{mL}$ 尿素溶液 $(200\mathrm{g / L})$ ，混匀，滴加亚硝酸钠溶液 $(100\mathrm{g / L})$ 至微红色消失，加饱和氢氧化钠溶液呈碱性 $(\mathrm{pH} = 9)$ ，放置 $2\textrm{h}$ 后过滤，滤液加水至 $100~\mathrm{mL}$ ，混匀。取此液 $20.0~\mathrm{mL}$ 于 $25~\mathrm{mL}$ 比色管中，加 $1\mathrm{mL}$ 硫酸溶液 $(2.5\mathrm{mol / L})$ ， $1\mathrm{mL}$ 焦磷酸钠溶液 $(50~\mathrm{g / L})$ ，混匀，加 $2\mathrm{mL}$ 二苯碳酰二肼溶液，加水至 $25.0~\mathrm{mL}$ ，混匀。放置 $5\mathrm{min}$ ，于 $540~\mathrm{nm}$ 处测定吸光度。同时做试样空白试验。

# 13 分析结果的表述

# 13.1 食品模拟物试液中铬的浓度

食品模拟物试液中铬的浓度按式(1)计算：

$$
X = \frac {m _ {1} - m _ {0}}{V _ {1}} \tag {1}
$$

式中：

$X$ ——食品模拟物试液中铬的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$m_{1}$ ——测定所准确吸取的试样浸泡液中铬的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_0$ ——测定所准确吸取的空白液中铬的质量，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——所取试样浸泡液体积，单位为毫升 $(\mathrm{mL})$ 。

# 13.2 铬迁移量的计算

由13.1得到的食品模拟物试液中铬的含量，按GB5009.156进行迁移量的计算，得到食品接触材料及制品中铬的迁移量。计算结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 15 其他

方法检出限为 $0.008\mathrm{mg / L}$ ，定量限为 $0.025\mathrm{mg / L}$

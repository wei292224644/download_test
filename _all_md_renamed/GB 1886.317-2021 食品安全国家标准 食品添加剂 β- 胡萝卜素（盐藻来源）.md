# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 最大吸收波长

用分光光度计测定，试样的环己烷溶液 $(5\mathrm{mg / L})$ 在 $448~\mathrm{nm}\sim 457~\mathrm{nm}$ 和 $474~\mathrm{nm}\sim 486~\mathrm{nm}$ 处有最大吸收值。

# A.2.2 颜色反应

将试样甲苯溶液（约 $400\mu \mathrm{g / mL}$ )滴在滤纸上，使用 $20\%$ 三氯化锑甲苯溶液喷雾， $2\mathrm{min}\sim 3\mathrm{min}$ 后变为蓝色。

# A.3 总天然类胡萝卜素含量的测定

# A.3.1 方法原理

$\beta$ -胡萝卜素是共扼双键的化合物，在波长 $455~\mathrm{nm}\pm 1~\mathrm{nm}$ 处有最大吸收值，可依此来测定总类胡萝卜素含量。试样溶于三氯甲烷中并用己烷稀释到合适的浓度，最终的稀释在环己烷中进行，测定吸光度，并使用吸收度系数计算浓度。

# A.3.2 试剂和材料

A.3.2.1 三氯甲烷。

A.3.2.2 环己烷。

A.3.2.3 己烷。

# A.3.3 仪器和设备

分光光度计。

# A.3.4 分析步骤

# A.3.4.1 溶液A的制备

称取约 $0.5\mathrm{g}$ 的三份试样，精确到 $0.001\mathrm{g}$ ，到 $100\mathrm{mL}$ 容量瓶（使用具有低光化性质的玻璃器皿）。添加约 $10\mathrm{mL}$ 三氯甲烷溶解试样，混合均匀。透光观察，确保试样完全溶解；以己烷稀释至刻度，混合均匀，即得溶液A。当测定商品化的产品时，根据产品形式不同，需增加相应的前处理步骤，以保证其中的类胡萝卜素完全溶出。

# A.3.4.2 溶液B的制备

吸取 $2.0\mathrm{mL}$ 溶液A到 $50~\mathrm{mL}$ 容量瓶，以己烷稀释至刻度，混合均匀，即得溶液B。

# A.3.4.3 溶液C的制备

吸取 $2.0\mathrm{mL}$ 溶液B到 $50~\mathrm{mL}$ 容量瓶，以环己烷稀释至刻度，混合均匀，即得溶液C。

# A.3.4.4 测定

在最大吸收波长处测定溶液C的吸光度，以环己烷作为空白对照调零。吸光度应控制在 $0.3\sim 0.7$ 之间，否则应调整试样液(溶液C)浓度，再重新测定吸光度。使用吸光系数进行计算，不需要标准品。

# A.3.5 结果计算

总天然类胡萝卜素的质量分数 $w$ 按式(A.1)计算。

$$
w = \frac {A \times 6 2 5 0 0}{2 5 0 0 \times m} \quad \dots \tag {A.1}
$$

式中：

A ——溶液C在 $455 \mathrm{~nm}$ 附近的最大吸光度；

62500——以 $\mathrm{mL}$ 为单位的稀释因子；

2500 $\beta$ -胡萝卜素在环己烷中的百分吸光系数 $(1\% ,1\mathrm{cm})$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

结果保留三位有效数字。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $1.5\%$ 。

# A.4 d-芳烯含量的测定

# A.4.1 方法原理

加入内标物的试样用二氯甲烷溶解后，采用气相色谱法进行分析，内标法测定d-苄烯的含量。

# A.4.2 试剂和材料

A.4.2.1 正己烷。  
A.4.2.2 二氯甲烷。  
A.4.2.3 d-烯标准品：纯度 $\geq 95\%$   
A.4.2.4 $\gamma$ -松油烯标准品：纯度 $\geqslant 96\%$

# A.4.3 仪器和设备

A.4.3.1 气相色谱仪：具有氢火焰离子检测器(FID)。  
A.4.3.2 毛细管色谱柱：聚乙二醇强极性固定相，柱长 $30\mathrm{m}$ ，内径 $0.53\mathrm{mm}$ ，膜厚 $1.0\mu \mathrm{m}$ ，或其他等效色谱柱。  
A.4.3.3 参考色谱条件。  
A.4.3.3.1 进样器温度： $150^{\circ}C$   
A.4.3.3.2 检测器温度： $200^{\circ}C$   
A.4.3.3.3 柱温： $80^{\circ}\mathrm{C}$ 保持 $5\mathrm{min}$ ，以 $10^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $150^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ ；总运行时间 $15\mathrm{min}$ 。

A.4.3.3.4 载气：高纯氮气，恒压 $12\mathrm{kPa}$   
A.4.3.3.5 分流比： $5:1$ ，或不分流。  
A.4.3.3.6 进样体积： $2.0\mu \mathrm{L}$ （分流）， $0.5\mu \mathrm{L}$ （不分流）。

# A.4.4 分析步骤

# A.4.4.1 试验溶液的制备

# A.4.4.1.1 $\gamma$ -松油烯内标储备液

精确称取 $1.0\mathrm{g}\gamma$ -松油烯标准品，精确至 $0.0001\mathrm{g}$ ，正己烷溶解，移入 $100~\mathrm{mL}$ 容量瓶中，正己烷定容至刻度，摇匀，备用。

# A.4.4.1.2 d-芳烯标准储备液

精确称取 $0.5\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，d-芳烯标准品，正己烷溶解，移入 $50~\mathrm{mL}$ 容量瓶中，正己烷定容至刻度，摇匀，备用。

# A.4.4.1.3 d-芳烯标准工作液

精确移取 $5.00\mathrm{mL}$ d-苄烯标准储备液于 $50~\mathrm{mL}$ 容量瓶中，正己烷定容至刻度，摇匀，备用。

# A.4.4.1.4 系列标准工作液

精确移取 $1.00\mathrm{mL}\gamma$ -松油烯内标储备液和 $10.00~\mathrm{mL}$ d-苄烯标准储备液于 $25~\mathrm{mL}$ 容量瓶中，用正己烷稀释至刻度，摇匀，备用；精确移取 $2.00~\mathrm{mL}\gamma$ -松油烯内标储备液和 $5.00~\mathrm{mL}$ d-苄烯标准储备液于 $50~\mathrm{mL}$ 容量瓶中，用正己烷稀释至刻度，摇匀，备用；精确移取 $2.00~\mathrm{mL}\gamma$ -松油烯内标储备液和 $5.00~\mathrm{mL}$ d-苄烯标准工作液于 $50~\mathrm{mL}$ 容量瓶中，用正己烷稀释至刻度，摇匀，备用；精确移取 $2.00~\mathrm{mL}\gamma$ -松油烯内标储备液和 $2.00~\mathrm{mL}$ d-苄烯标准工作液于 $50~\mathrm{mL}$ 容量瓶中，用正己烷稀释至刻度，摇匀，备用；空白液，精确移取 $2.00~\mathrm{mL}\gamma$ -松油烯内标储备液于 $50~\mathrm{mL}$ 容量瓶中，用正己烷稀释至刻度，摇匀，备用。配制成分别含有d-苄烯 $\mathrm{0mg / mL,0.04mg / mL,0.1mg / mL,1.0mg / mL,4.0mg / mL}$ 且其中分别含有 $0.4\mathrm{mg / mL}\gamma$ -松油烯的系列标准工作液。

# A.4.4.2 系统稳定性试验

A.4.4.2.1 气相色谱运行两次空白液，不应出现d-芳烯色谱峰。

A.4.4.2.2 气相色谱运行5次含有d-苄烯 $1.0\mathrm{mg / mL}$ 和 $0.4\mathrm{mg / mL}\gamma$ -松油烯的系列标准工作液，变异系数应不超过 $1.00\%$ 。d-苄烯和 $\gamma$ -松油烯的保留时间相差应不小于 $0.5\mathrm{min}$

# A.4.4.3 试样溶液的制备

用锥形瓶精确称取 $1.0\mathrm{g}\sim 2.5\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，精确移取 $1.00\mathrm{mL}\gamma$ -松油烯内标储备液至锥形瓶中，混合均匀。加入 $10~\mathrm{mL}$ 二氯甲烷，混合均匀。将混合后的溶液移入 $25~\mathrm{mL}$ 容量瓶中，二氯甲烷定容至刻度，摇匀。

# A.4.4.4 测定

在A.4.3.3参考色谱条件下，吸取试样溶液注入色谱仪，记录所得的试样溶液中d-苄烯的峰面积和内标物质峰面积。

# A.4.5 结果计算

A.4.5.1 响应因子 $f_{1}$ 按式(A.2)计算。

$$
f _ {1} = \frac {A _ {2} \times c _ {1}}{A _ {1} \times c _ {2}} \quad \dots \tag {A.2}
$$

式中：

$A_{2}$ —— $\gamma$ -松油烯内标物质色谱峰面积；

$c_{1}$ ——d-芦烯标准液浓度， $\%$ ；

$A_{1}$ ——d-芋烯标准物质峰面积；

$c_{2}$ —— $\gamma$ -松油烯内标液浓度， $\%$ 。

A.4.5.2 d-芋烯的质量分数 $w$ 按式(A.3)计算。

$$
w = \frac {A _ {4} \times m _ {1} \times f _ {1}}{A _ {3} \times m} \tag {A.3}
$$

式中：

$A_{4}$ ——试样溶液中d-芦烯峰面积；

$m_{1}$ ——试样液中 $\gamma$ -松油烯内标物质质量，单位为克(g)；

$f_{1}$ ——响应因子；

$A_{3}$ ——试样溶液中 $\gamma$ -松油烯内标物质色谱峰面积；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

A.4.5.3 系统稳定性试验中变异系数 $CofV$ 按式(A.4)计算。

$$
C o f V = \frac {1 0 0}{\bar {X}} \sqrt {\frac {\sum_ {i = 1} ^ {N} \left(X _ {i} - \bar {X}\right) ^ {2}}{N - 1}} \quad \dots \tag {A.4}
$$

式中：

$\overline{X}$ ——稳定性试验测量平均数；

$X_{i}$ ——稳定性试验中单次测量值；

$N$ ——稳定性试验测量次数。

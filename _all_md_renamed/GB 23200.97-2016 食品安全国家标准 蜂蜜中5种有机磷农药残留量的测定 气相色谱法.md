# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T1593—2005《进出口蜂蜜中五种有机磷农药残留量检验方法 气相色谱法》。与SN/T1593—2005相比，主要变化如下：

——标准文本格式修改为食品安全国家标准文本格式；

标准名称“进出口蜂蜜”改为“蜂蜜”；

——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T1593—2005。

# 食品安全国家标准蜂蜜中5种有机磷农药残留量的测定 气相色谱法

# 1 范围

本标准规定了蜂蜜中敌百虫、皮蝇磷、毒死蜱、马拉硫磷、蝇毒磷农药残留量检测的气相色谱测定方法。

本标准适用于蜂蜜中敌百虫、皮蝇磷、毒死蜱、马拉硫磷、蝇毒磷农药残留量的测定和确证，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

蜂蜜加水稀释后，用乙酸乙酯提取样品中有机磷农药，低温浓缩，用配有火焰光度检测器的气相色谱仪测定，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。  
4.1.2 氯化钠 $(\mathrm{NaCl})$ 。

# 4.2 标准品

有机磷类标准品：敌百虫(trichlorphon，CAS号：52-68-6)，皮蝇磷(fenchlorphos，CAS号：299-84-3)，毒死蜱(chlorpyrifos，CAS号：2921-88-2)，马拉硫磷(malathion，CAS号：121-75-5)，蝇毒磷(coumaphos，CAS号：56-72-4)，纯度 $\geqslant 97\%$ 。

# 4.3 标准溶液配制

敌百虫、皮蝇磷、毒死蜱、马拉硫磷、蝇毒磷标准储备溶液：分别称取适量标准品，分别用乙酸乙酯溶解定容至 $100\mathrm{mL}$ ，溶液浓度为 $100\mu \mathrm{g} / \mathrm{mL}$ ，存放于 $4^{\circ}\mathrm{C}$ 冰箱。临用前，根据需要用乙酸乙酯稀释至适当浓度，作为混合标准工作液。

# 5 仪器和设备

5.1 气相色谱仪，配有火焰光度检测器（磷滤光片 $525\mathrm{nm}$ ）。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.3 旋转蒸发器。  
5.4 离心机： $4000\mathrm{r / min}$

# 5.5 混匀器。

# 6 试样制备与保存

对未结晶的样品，将其用力搅拌均匀，取样部位按GB2763的规定执行；对有结晶析出的样品，可将样品瓶盖塞紧后，置于不超过 $60^{\circ}\mathrm{C}$ 的水浴中温热，待样品全部融化后搅匀，迅速冷却至室温。在融化时，必须防止水分挥发。制备好的试样均分成2份，分别装入样品瓶中，密封，并标明标记。

在制样的操作过程中，应防止样品污染或发生残留物含量的变化。将试样于室温下保存。

# 7 分析步骤

# 7.1 提取

称取 $10 \mathrm{~g}$ 均匀试样(精确到 $0.01 \mathrm{~g}$ ), 置于 $50 \mathrm{~mL}$ 离心管中, 加入 $15 \mathrm{~mL}$ 水和 $5 \mathrm{~g}$ 氯化钠, 混匀。再加入 $15 \mathrm{~mL}$ 乙酸乙酯, 在混匀器上混匀 $2 \mathrm{~min}$ , 离心 $5 \mathrm{~min}(3000 \mathrm{r} / \mathrm{min})$ 。用尖嘴吸管将上层乙酸乙酯溶液移至 $100 \mathrm{~mL}$ 浓缩瓶中, 水相再加入 $15 \mathrm{~mL}$ 乙酸乙酯。重复上述操作, 合并 2 次提取液, 用旋转蒸发器在 $40^{\circ} \mathrm{C}$ 水浴减压浓缩至近干。加入 $8 \mathrm{~mL}$ 乙酸乙酯溶解, 定量转移至 $10 \mathrm{~mL}$ 离心管中, 在 $40^{\circ} \mathrm{C}$ 以下水浴中用平缓氮气流吹至干。准确加入 $1 \mathrm{~mL}$ 乙酸乙酯, 混匀, 供气相色谱测定。

# 7.2 测定

# 7.2.1 气相色谱参考条件

a) 色谱柱：石英毛细管柱，DB-1701, $30\mathrm{m}\times 0.25\mathrm{mm}$ (内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b) 载气: 氮气 (纯度大于 $99.999\%$ ); 载气流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ; 尾吹气流速: $30 \mathrm{~mL} / \mathrm{min}$ ; 氢气流速: $75 \mathrm{~mL} / \mathrm{min}$ , 空气流速: $100 \mathrm{~mL} / \mathrm{min}$ ;  
c) 柱温: 初始温度 $50^{\circ} \mathrm{C}$ 保持 $2 \mathrm{~min}$ , 以 $15^{\circ} \mathrm{C} / \mathrm{min}$ 升至 $180^{\circ} \mathrm{C}$ 保持 $1 \mathrm{~min}$ , 以 $8^{\circ} \mathrm{C} / \mathrm{min}$ 升至 $290^{\circ} \mathrm{C}$ 保持 $10 \mathrm{~min}$ ;  
d) 进样口温度: $250^{\circ} \mathrm{C}$ ;   
e) 检测器温度: $250^{\circ} \mathrm{C}$ ;   
f）进样方式：不分流进样；  
g) 进样量: $2 \mu \mathrm{L}$ ;   
h）开阀时间：1.5min。

# 7.2.2 色谱测定

根据样品中被测有机磷农药的含量，选定峰面积相近的标准工作溶液。标准工作溶液和样液中各种有机磷的响应值均应在仪器的线性范围内。标准工作溶液和样液等体积穿插进样测定。在上述色谱条件下，敌百虫的保留时间约为 $6.0 \mathrm{~min}$ ，皮蝇磷的保留时间约为 $17.8 \mathrm{~min}$ ，毒死蜱的保留时间约为 $18.5 \mathrm{~min}$ ，马拉硫磷的保留时间约为 $18.9 \mathrm{~min}$ ，蝇毒磷的保留时间约为 $27.8 \mathrm{~min}$ 。标准品的色谱图参见图A.1。

# 7.3 空白实验

除不加试样外，均按 $7.1\sim 7.2$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中有机磷类农药的含量。

$$
X _ {i} = \frac {A _ {i} \times C _ {s i} \times V}{A _ {s i} \times m}. \tag {1}
$$

式中：

$X_{i}$ ——试样中各有机磷类农药的残留含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A_{i}$ ——样液中各有机磷类农药的峰面积；

$C_{s}$ ——标准工作液中有机磷农药的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$

$A_{s}$ ——标准工作液中各有机磷农药的峰面积；

$m$ ——最终样液所代表的试样量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录B的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 C 的要求。

# 10 定量限和回收率

# 10.1 定量限

本方法有机磷类农药定量限均为 $0.01\mathrm{mg / kg}$

# 10.2 回收率

回收率(在不同添加浓度范围内)如下：

a）敌百虫：

1) 添加浓度在 $0.01\mathrm{mg / kg}$ 时，回收率为 $70\% \sim 85\%$ ，精密度为 $6.0\%$   
2）添加浓度在 $0.02\mathrm{mg / kg}$ 时，回收率为 $70\% \sim 90\%$ ，精密度为 $9.2\%$   
(3) 添加浓度在 $0.1 \mathrm{mg} / \mathrm{kg}$ 时, 回收率为 $81\% \sim 89\%$ , 精密度为 $3.2\%$ .

b）皮蝇磷：

1) 添加浓度在 $0.01 \mathrm{mg} / \mathrm{kg}$ 时, 回收率为 $73\% \sim 99\%$ , 精密度为 $10.0\%$ ;  
2）添加浓度在 $0.02\mathrm{mg / kg}$ 时，回收率为 $75\% \sim 95\%$ ，精密度为 $7.7\%$   
3) 添加浓度在 $0.1 \mathrm{mg} / \mathrm{kg}$ 时, 回收率为 $80\% \sim 97\%$ , 精密度为 $6.4\%$ .

c）毒死蜱：

1) 添加浓度在 $0.01\mathrm{mg / kg}$ 时，回收率为 $84\% \sim 102\%$ ，精密度为 $6.3\%$   
2）添加浓度在 $0.02\mathrm{mg / kg}$ 时，回收率为 $80\% \sim 100\%$ ，精密度为 $6.7\%$   
3）添加浓度在 $0.1\mathrm{mg / kg}$ 时，回收率为 $84\% \sim 106\%$ ，精密度为 $8.5\%$ 。

d) 马拉硫磷：

1）添加浓度在 $0.01\mathrm{mg / kg}$ 时，回收率为 $84\% \sim 106\%$ ，精密度为 $7.6\%$   
2）添加浓度在 $0.02\mathrm{mg / kg}$ 时，回收率为 $80\% \sim 105\%$ ，精密度为 $10.0\%$   
3）添加浓度在 $0.1\mathrm{mg / kg}$ 时，回收率为 $81\% \sim 103\%$ ，精密度为 $7.6\%$ 。

e) 蝎毒磷：

1）添加浓度在 $0.01\mathrm{mg / kg}$ 时，回收率为 $74\% \sim 96\%$ ，精密度为 $10.0\%$   
2）添加浓度在 $0.02\mathrm{mg / kg}$ 时，回收率为 $75\% \sim 100\%$ ，精密度为 $10.4\%$   
3）添加浓度在 $0.1\mathrm{mg / kg}$ 时，回收率为 $80\% \sim 102\%$ ，精密度为 $7.9\%$ 。

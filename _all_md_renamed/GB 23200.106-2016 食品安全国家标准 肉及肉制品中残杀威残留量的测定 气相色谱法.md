# 食品安全国家标准

# 肉及肉制品中残杀威残留量的测定 气相色谱法

# 1 范围

本标准规定了肉及肉制品中残杀威残留量检验的气相色谱测定方法。

本标准适用于猪肉、鸡肉和牛肉中残杀威残留量的检测，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

试样用环己烷一乙酸乙酯(1+1,体积比)均质提取，提取液用凝胶渗透色谱净化，浓缩、定容后用气相色谱测定，外标法定量。

# 4 试剂和材料

除另有规定外，所用试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 环己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{12}\right)$ ：色谱纯。

4.1.2 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。

4.1.3 正己烷 $\mathrm{(C_6H_{14})}$ ：色谱纯。

4.1.4 甲苯 $(\mathrm{C}_7\mathrm{H}_8)$ ：色谱纯。

4.1.5 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：用前在 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，储于干燥器中，冷却后备用。

# 4.2 试剂配制

环己烷—乙酸乙酯混合溶剂(1+1，体积比)：取 $100\mathrm{mL}$ 环己烷，加入 $100\mathrm{mL}$ 乙酸乙酯，摇匀备用。

# 4.3 标准品

残杀威标准物质(propoxur；分子式 $\mathrm{C_{11}H_{15}NO_3}$ ；CAS号： $114 - 26 - 1$ )：纯度 $\geqslant 99\%$ 。

# 4.4 标准溶液配制

4.4.1 残杀威标准储备溶液: 准确称取适量的残杀威标准物质 (精确至 $0.1 \mathrm{mg}$ ), 用甲苯配成浓度为 $1.0 \mathrm{mg} / \mathrm{mL}$ 的标准储备液。保存于 $4^{\circ} \mathrm{C}$ 冰箱内。

4.4.2 残杀威标准工作溶液：根据检测要求，分别吸取上述标准储备液用正己烷稀释成适当浓度的标准工作溶液。保存于 $4^{\circ}\mathrm{C}$ 冰箱内。

# 4.5 材料

4.5.1 滤膜： $0.45\mu \mathrm{m}$ ，有机相。

4.5.2 无水硫酸钠柱：在筒形漏斗（内径 $1.5\mathrm{cm}$ ），内装约 $5\mathrm{cm}$ 高的无水硫酸钠。

# 5 仪器和设备

5.1 气相色谱仪：配有氮磷检测器(NPD)。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 凝胶渗透色谱仪。  
5.4 离心机。  
5.5 旋转蒸发仪。  
5.6 氮气吹干仪。  
5.7 均质器。

# 6 试样制备和保存

# 6.1 试样的制备

样品用绞肉机绞碎，样品取样部位按GB2763的规定执行，充分混匀，用四分法缩分至不少于 $500\mathrm{g}$ ，装入清洁容器内，密封后，标明标记。

# 6.2 试样的保存

将试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 7 分析步骤

# 7.1 提取

称取 $10 \mathrm{~g}$ 试样(精确至 $0.01 \mathrm{~g}$ ) 于盛有 $20 \mathrm{~g}$ 无水硫酸钠的 $100 \mathrm{~mL}$ 离心管中, 加入 $30 \mathrm{~mL}$ 环己烷一乙酸乙酯混合溶剂, 用均质器在 $15000 \mathrm{r} / \mathrm{min}$ 均质提取 $1.5 \mathrm{~min}$ 后, 在 $3000 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ 。上清液通过装有无水硫酸钠的筒形漏斗, 收集于 $100 \mathrm{~mL}$ 梨形瓶中。残渣用 $30 \mathrm{~mL}$ 环己烷一乙酸乙酯混合溶剂同上提取操作, 合并 2 次提取液, 用少量的环己烷一乙酸乙酯混合溶剂洗涤无水硫酸钠柱。将提取液于 $40^{\circ} \mathrm{C}$ 水浴旋转蒸发至约 $2 \mathrm{~mL}$ , 待净化。

# 7.2 净化

# 7.2.1 凝胶渗透色谱仪器条件

a）净化柱： $400\mathrm{mm}\times 25\mathrm{mm}$ ，内装BIO-BeadsS-X3填料，或相当者。  
b) 流动相: 环己烷-乙酸乙酯(1+1,体积比)。  
c) 流速: $5 \mathrm{~mL} / \mathrm{min}$ 。  
d) 进样量: $5 \mathrm{~mL}$ 。  
e）开始收集时间：1200s。  
f）结束收集时间： $1800\mathrm{s}$

# 7.2.2 净化

将浓缩的提取液(7.1)转移至 $10\mathrm{mL}$ 刻度离心试管中，并用环己烷一乙酸乙酯混合溶剂多次洗涤梨形瓶，合并洗液并定容至刻度。在 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 后，将上清液用 $0.45\mu \mathrm{m}$ 有机相滤膜滤入凝胶渗透色谱仪进样瓶中。用凝胶渗透色谱仪净化，收集 $1200\mathrm{s}\sim 1800\mathrm{s}$ 的流出液。将流出液用氮气吹干仪吹干，加入 $1.0\mathrm{mL}$ 正己烷溶解，混匀，供气相色谱仪测定。

# 7.3 测定

# 7.3.1 色谱测定参考条件

a）色谱柱：HP-5， $30\mathrm{m}\times 0.32\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者。  
b）色谱柱温度：初始温度 $100^{\circ}\mathrm{C}$ ，保持 $0.5\mathrm{min}$ ；以 $10^{\circ}\mathrm{C / min}$ 的速度升至 $280^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ 。

c) 载气 $(\mathrm{N}_2)$ : $1.5 \mathrm{~mL} / \mathrm{min}$ 。  
d) 氢气 $\left(\mathrm{H}_{2}\right): 3.0 \mathrm{~mL} / \mathrm{min}$ 。  
e) 空气: $60 \mathrm{~mL} / \mathrm{min}$ 。  
f）检测器温度： $280^{\circ}C$ □  
g）进样口温度： $250^{\circ}C$   
h）进样体积： $1\mu \mathrm{L}$   
i）进样方式：无分流进样， $1.5\mathrm{min}$ 后打开分流阀。

# 7.3.2 色谱测定

根据试样中残杀威的含量，选取适宜浓度的标准工作溶液与待测样液等体积进样。标准工作液和待测样液中残杀威的响应值应在仪器响应线性范围内。在上述色谱条件下，残杀威的保留时间约为 $11.1\mathrm{min}$ ，根据峰面积外标法定量。标准品色谱图参见附录A。

# 7.4 空白试验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中残杀威的残留含量。

$$
X = \frac {A \times c _ {s} \times V}{A _ {s} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中残杀威残留含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

A ——试样溶液中残杀威色谱峰的峰面积；

$c_{s}$ ——标准工作溶液中残杀威的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积, 单位为毫升 $(\mathrm{mL})$ ;

$A_{s}$ ——标准工作溶液中残杀威色谱峰的峰面积；

$m$ ——最终样液所代表的试样质量, 单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 B 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 C 的要求。

# 10 定量限和回收率

# 10.1 定量限

本方法残杀威的定量限为 $0.01\mathrm{mg / kg}$

# 10.2 回收率

本方法在 $0.01\mathrm{mg / kg},0.02\mathrm{mg / kg},0.05\mathrm{mg / kg}$ 添加水平时的加标回收率参见附录D。

# 1 范围

本文件规定了水产品中硝基呋喃类代谢物3-氨基-2-噁唑烷基酮(3-amino-2-oxazolidinone，AOZ)、5-吗啉甲基-3-氨基-2-噁唑烷基酮(5-morpholinomethyl-3-amino-2-oxazolidinone，AMOZ)、1-氨基-2-内酰脲(1-aminohydantoin，AHD)和氨基脲(semicarbazide，SEM)残留量的制样和液相色谱-串联质谱测定方法。

本文件适用于鱼、海参及鳖等水产品可食组织中硝基呋喃类代谢物AOZ、AMOZ、AHD和SEM残留量的测定，以及虾和蟹等甲壳类可食组织中AOZ、AMOZ和AHD的测定。

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的硝基呋喃类蛋白结合态代谢物在酸性条件下水解，经2-硝基苯甲醛衍生化，用乙酸乙酯液液萃取，高速离心净化，液相色谱串联质谱测定，内标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ ：色谱纯。  
5.1.2 乙酸铵 $\mathrm{(CH_3COONH_4)}$ 色谱纯。  
5.1.3 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ ：色谱纯。  
5.1.4 二甲亚砜 $\left[\left(\mathrm{CH}_{3}\right)_{2} \mathrm{SO}\right]$ ：色谱纯  
5.1.5 2-硝基苯甲醛 $\mathrm{(C_7H_5NO_3)}$ ：色谱纯。  
5.1.6 无水磷酸氢二钾 $(\mathrm{K_2HPO_4})$   
5.1.7 盐酸（HCl）。

# 5.2 溶液配制

5.2.1 $0.5\mathrm{mol} / \mathrm{L}$ 盐酸溶液：取盐酸 $42~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.2 $0.002\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液：取乙酸铵 $0.15\mathrm{g}$ ，用水溶解并稀释至 $1000\mathrm{mL}$ ，混匀。

5.2.3 $0.05\mathrm{mol} / \mathrm{L}$ 2-硝基苯甲醛溶液：取2-硝基苯甲醛 $0.076\mathrm{g}$ ，用二甲亚砜溶解并稀释至 $10~\mathrm{mL}$ ，混匀，现用现配。  
5.2.4 $1.0\mathrm{mol} / \mathrm{L}$ 磷酸氢二钾溶液：取无水磷酸氢二钾 $87.1\mathrm{g}$ ，用水溶解并稀释至 $500~\mathrm{mL}$ ，混匀。  
5.2.5 $5\%$ 甲醇溶液：取甲醇 $5\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。

# 5.3 标准品

5.3.1 硝基呋喃类代谢物：AOZ、AMOZ、AHD·HCl和SEM·HCl，含量均 $\geq 99.0\%$ ，具体内容见附录A。

5.3.2 内标： $\mathrm{AOZ - D_4}$ 、AMOZ- $\mathrm{D}_5$ 、 $\mathrm{AHD^{-13}C_3}$ 和SEM·HC- $^{13}\mathrm{C},^{15}\mathrm{N}_2$ ，含量均 $\geqslant 99.0\%$ ，具体内容见附录A。

# 5.4 标准溶液配制

5.4.1 标准储备液：取AOZ、AMOZ、AHD·HCl、SEM·HCl标准品各适量（相当于活性成分 $10\mathrm{mg}$ ），精密称定，加甲醇适量使溶解并定容至 $10~\mathrm{mL}$ 棕色容量瓶，配制成浓度为 $1.0\mathrm{mg / mL}$ 标准储备液。 $-18^{\circ}C$ 避光保存，有效期6个月。  
5.4.2 混合标准中间液：分别准确量取标准储备液适量，用甲醇逐级稀释，配制成浓度为 $1.0\mu \mathrm{g / mL}$ 混合标准中间液。 $2^{\circ}C\sim 8^{\circ}C$ 避光保存，有效期1个月。  
5.4.3 混合标准工作液：分别准确量取混合标准中间液各 $1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 和 $100~\mathrm{mL}$ 棕色容量瓶，用甲醇稀释至刻度，配制成浓度为 $100~\mu \mathrm{g / L}$ 和 $10~\mu \mathrm{g / L}$ 混合标准工作液。现用现配。  
5.4.4同位素内标标准储备液：取 $\mathrm{AOZ - D_4}$ 、AMOZ- $\mathrm{D}_5$ 、AHD- $^{13}\mathrm{C}_3$ 和SEM·HC1- $^{13}\mathrm{C},^{15}\mathrm{N}_2$ 标准品各适量（相当于活性成分 $10\mathrm{mg})$ ，精密称定，加甲醇适量使溶解并稀释定容至 $10~\mathrm{mL}$ 棕色容量瓶，配制成浓度为 $1.0\mathrm{mg / mL}$ 内标标准储备液。一 $18^{\circ}C$ 避光保存，有效期6个月。  
5.4.5 混合内标标准工作液：分别准确量取同位素内标标准储备液适量，用甲醇逐级稀释，配制成浓度为 $100\mu \mathrm{g} / \mathrm{L}$ 混合内标标准工作液。 $-18^{\circ}\mathrm{C}$ 避光保存，有效期3个月。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾电离源  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$   
6.3 离心机： $6000\mathrm{r / min}$   
6.4 高速离心机： $14000\mathrm{r / min}$   
6.5 涡旋混合器。  
6.6 恒温振荡器。  
6.7 氮吹仪。

# 7 试料的制备与保存

# 7.1 试料的制备

按GB/T30891—2014附录B的要求制样。

a）取均质的供试料品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 水解与衍生化

取试料 $2\mathrm{g}$ （精确至 $\pm 0.01\mathrm{g}$ ，于 $50~\mathrm{mL}$ 聚丙烯离心管中，准确加入混合内标标准工作液 $50~\mu \mathrm{L}$ ，涡旋混合 $1\mathrm{min}$ ，加盐酸溶液 $5\mathrm{mL},$ 2-硝基苯甲醛溶液 $0.15\mathrm{mL}$ ，涡旋混合 $1\mathrm{min}$ ，置于恒温振荡器中， $37^{\circ}C$ 避光振荡 $16\mathrm{h}$

# 8.2 提取净化

取出离心管冷却至室温，用磷酸氢二钾溶液调 $\mathrm{pH}$ 至 $7.0\sim 7.5$ ，加乙酸乙酯 $8\mathrm{mL}$ ，涡旋振荡 $30~s$ $6000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液至 $10~\mathrm{mL}$ 玻璃离心管， $40^{\circ}C$ 氮气吹干。加 $5 \%$ 甲醇溶液 $1.0\mathrm{mL}$ 溶解残

留物，再将溶液转移至 $1.5\mathrm{mL}$ 离心管中，以 $14000\mathrm{r / min}$ 的转速离心 $10\mathrm{min}$ ，取清液过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱分析。

# 8.3 标准曲线的制备

分别准确移取 $10~\mu \mathrm{g / L}$ 和 $100~\mu \mathrm{g / L}$ 混合标准工作液各 $0.05\mathrm{mL},0.1\mathrm{mL},0.2\mathrm{mL}$ 于6个 $50~\mathrm{mL}$ 离心管中，除不加试料外，按7.1和7.2步骤操作，使最终浓度分别为 $0.5~\mu \mathrm{g / L},1.0~\mu \mathrm{g / L},2.0~\mu \mathrm{g / L},$ $5.0~\mu \mathrm{g / L},10~\mu \mathrm{g / L},20~\mu \mathrm{g / L}$ ，按7.4测定。以测得特征离子质量色谱峰外标和内标峰面积比值为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.5 测定法

# 8.5.1 定性测定

在同样测试条件下，试料溶液中硝基呋喃类代谢物的保留时间与标准溶液中硝基呋喃类代谢物的保留时间之比，偏差在 $\pm 2.5\%$ 以内，且检测到的相对离子丰度，应当与浓度相当的校正标准溶液相对丰度一致。其允许偏差应符合表3的要求。

# 8.5.2 定量测定

取试料溶液和相应的标准溶液，作单点或多点校准，以色谱峰面积定量，按内标法计算，标准溶液及试料溶液中的硝基呋喃类代谢物响应值均应在仪器检测的线性范围内。在上述液相色谱-质谱条件下，硝基呋喃类代谢物标准溶液的液相色谱-质谱图见附录B。

# 8.6 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试料中硝基呋喃类代谢物的残留量按标准曲线法或公式(1)计算。

$$
X = \frac {A _ {i} \times A _ {i S} ^ {\prime} \times C _ {S} \times C _ {i S} \times V}{A _ {i S} \times A _ {S} \times C _ {i S} ^ {\prime} \times m}. \tag {1}
$$

式中：

$X$ ——试料中硝基呋喃类代谢物的残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{i\mathrm{S}}$ ——试料溶液中硝基呋喃类代谢物内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{\mathrm{S}}$ ——对照溶液中硝基呋喃类代谢物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{is}^{\prime}$ —对照溶液中硝基呋喃类代谢物内标浓度的数值，单位为微克每升 $(\mu \mathrm{g / L})$

$A_{i}$ ——试料溶液中硝基呋喃类代谢物的峰面积；

$A_{iS}$ ——试料溶液中硝基呋喃类代谢物内标的峰面积；

$A_{\mathrm{S}}$ ——对照溶液中硝基呋喃类代谢物的峰面积；

$A_{iS}^{\prime}$ ——对照溶液中硝基呋喃类代谢物内标的峰面积；

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法AOZ、SEM、AMOZ、AHD的检测限均为 $0.5\mu \mathrm{g / kg}$ ，定量限均为 $1.0\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $1.0\mu \mathrm{g / kg}\sim 10~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

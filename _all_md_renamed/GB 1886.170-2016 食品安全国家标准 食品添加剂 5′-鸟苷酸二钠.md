# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 $5^{\prime}$ -鸟苷酸二钠含量（以干基计）、紫外吸光度比值的测定

# A.2.1 试剂和材料

盐酸溶液：量取 $0.9\mathrm{mL}$ 盐酸，缓慢注入 $1000~\mathrm{mL}$ 水中。

# A.2.2 仪器和设备

紫外分光光度计。

# A.2.3 分析步骤

# A.2.3.1 试样溶液的制备

称取试样 $0.5\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ), 用适量盐酸溶液溶解并定容至 $1000\mathrm{mL}$ , 摇匀。吸取上述溶液 $10\mathrm{mL}$ , 用盐酸溶液稀释并定容至 $250\mathrm{mL}$ , 摇匀, 作为试样溶液备用。

# A.2.3.2 测定

将试样溶液注入 $1\mathrm{cm}$ 石英比色皿中，以盐酸溶液作参比溶液，用紫外分光光度计在 $250~\mathrm{nm},260$ $\mathrm{nm},280~\mathrm{nm}$ 处测定吸光度。

# A.2.4 结果计算

# A.2.4.1 $5^{\prime}$ -鸟苷酸二钠含量（以干基计）的计算

$5^{\prime}$ -鸟苷酸二钠含量(以干基计)的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {A}{289 . 8} \times \frac {250}{m} \times \frac {100}{100 - w _ {2}} \times 100 \% \dots \tag{A.1}
$$

式中：

$A$ ——试样溶液在 $260 \mathrm{~nm}$ 处测得的吸光度；

289.8——5'-鸟苷酸二钠的比吸光度；

250 体积换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ ；

100 ——换算因子；

$w_{2}$ ——试样的干燥减量， $\%$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $1\%$ 。

# A.2.4.2 紫外吸光度比值的计算

紫外吸光度比值(250/260)和(280/260)分别以 $X_{1}$ 和 $X_{2}$ 计，按式(A.2)和式(A.3)计算：

$$
X _ {1} = \frac {A _ {1}}{A} \quad \dots \tag {A.2}
$$

$$
X _ {2} = \frac {A _ {2}}{A} \quad \dots \tag {A.3}
$$

式中：

$A_{1}$ ——试样溶液在 $250 \mathrm{~nm}$ 处测得的吸光度；

$A$ ——试样溶液在 $260 \mathrm{~nm}$ 处测得的吸光度；

$A_{2}$ ——试样溶液在 $280~\mathrm{nm}$ 处测得的吸光度。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $1\%$ 。

# A.3 透光率( $5 \%$ 水溶液)的测定

# A.3.1 仪器和设备

分光光度计。

# A.3.2 分析步骤

称取试样 $1\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ), 溶于 $20\mathrm{mL}$ 水中, 摇匀。将上述试样溶液置于 $1\mathrm{cm}$ 比色皿中, 以水作空白对照, 用分光光度计于 $430\mathrm{nm}$ 下测定试样溶液的透光率, 记录读数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $0.5\%$ 。

# A.4 $\mathbf{pH}(5\%$ 水溶液)的测定

# A.4.1 仪器和设备

酸度计。

# A.4.2 分析步骤

称取试样 $1\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ), 溶于 $20~\mathrm{mL}$ 水中, 用酸度计测定上述试样溶液的 $\mathrm{pH}$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于0.2。

# A.5 其他核苷酸的测定

# A.5.1 试剂和材料

A.5.1.1 展开溶液：叔丁醇 + 饱和硫酸铵 + 0.025 mol/L 氨水 = 3 + 160 + 40。

A.5.1.2 层析滤纸。

# A.5.2 仪器和设备

A.5.2.1 紫外分析仪。

A.5.2.2 微量注射器。

# A.5.3 分析步骤

称取试样 $1.0\mathrm{g}$ ，加水溶解并定容至 $100\mathrm{mL}$ ，用层析滤纸点样 $10.0\mu \mathrm{L}$ ，于展开溶液中展开约 $30\mathrm{cm}$ ，晾干（或烘干）滤纸，在紫外分析仪 $254\mathrm{nm}$ 处观察。

# A.5.4 结果判定

若只呈现一个斑点，判定为通过试验。

# A.6 氨基酸的测定

# A.6.1 试剂和材料

茚三酮溶液：取茚三酮 $0.5\mathrm{g}$ ，加乙醇溶解并定容至 $100~\mathrm{mL}$ 。

# A.6.2 仪器和设备

水浴锅。

# A.6.3 分析步骤

称取试样 $0.1\mathrm{g}$ ，加水溶解并定容至 $100~\mathrm{mL}$ ，摇匀后吸取试样液 $5\mathrm{mL}$ ，加入茚三酮溶液 $1\mathrm{mL}$ ，于沸水浴中加热 $3\mathrm{min}$ 。

# A.6.4 结果判定

溶液不显色，判定为通过试验。

# A.7 铵盐的测定

# A.7.1 试剂和材料

A.7.1.1 氧化镁。

A.7.1.2 红色石蕊试纸。

# A.7.2 分析步骤

称取试样 $0.1\mathrm{g}$ 于试管中，加入 $50~\mathrm{mg}$ 氧化镁及 $1\mathrm{mL}$ 水，用水湿润红色石蕊试纸，将其挂入试管中，将试管于沸水浴加热 $5\mathrm{min}$ 。

# A.7.3 结果判定

石蕊试纸不显色，判定为通过试验。

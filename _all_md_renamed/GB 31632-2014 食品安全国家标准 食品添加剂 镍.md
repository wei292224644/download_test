# 食品安全国家标准

# 食品添加剂 镍

# 1 范围

本标准适用于食品添加剂镍。食品添加剂镍分为海绵镍和负载型镍，海绵镍是以镍、铝合金粉为主要原料，经混合、研磨、活化、洗涤制得；负载型镍是以二氧化硅等为载体，以镍盐为活性中心，经洗涤、干燥、活化、成型制得。

# 2 化学名称、分子式和相对分子质量

# 2.1 化学名称

镍

# 2.2 分子式

Ni

# 2.3 相对分子质量

58.69（按2011年国际相对原子质量）

# 3 分类

食品添加剂镍根据用途的不同分为两类：海绵镍，多用于胺和多元醇的催化加氢；负载型镍，多用于食用油的催化加氢。

# 附录A

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用制剂及制品，在没有注明其他要求时均按GB/T603规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸。

A.3.1.2 溴水。

A.3.1.3 氨水。

A.3.1.4 丁二酮肟乙醇溶液： $10\mathrm{g / L}$ ，称取 $1\mathrm{g}$ 丁二酮肟，溶解于 $100~\mathrm{mL}$ 的 $95\%$ 乙醇中。

# A.3.2 鉴别方法

# A.3.2.1 海绵镍

称取约 $0.1\mathrm{g}$ 试样溶于约 $2\mathrm{mL}$ 盐酸中，用水稀释至 $20~\mathrm{mL}$ 。取 $5\mathrm{mL}$ 试液于比色管中，加几滴溴水，用氨水调至微碱性，加入 $2\mathrm{mL} \sim 3\mathrm{mL}$ 的丁二酮肟乙醇溶液，产生强烈的红色，并有沉淀产生。

# A.3.2.2 负载型镍

取 $5 \mathrm{~mL}$ 试样溶液(A.4.2.4.1)置于试管中，加几滴溴水，用氨水调至微碱性，加入 $2 \mathrm{~mL} \sim 3 \mathrm{~mL}$ 的丁二酮肟乙醇溶液，产生强烈的红色，并有沉淀产生。

# A.4 镍(Ni)含量的测定

# A.4.1 方法——适用于海绵镍

# A.4.1.1 方法提要

试样加酸溶解，在氨性介质中，镍与丁二酮肟乙醇溶液生成红色丁二酮肟镍的沉淀，然后洗净沉淀烘干称重。

# A.4.1.2 试剂和材料

A.4.1.2.1 无水乙醇。

A.4.1.2.2 酒石酸。  
A.4.1.2.3 氨水。  
A.4.1.2.4 氮气。  
A.4.1.2.5 盐酸溶液： $1 + 1$   
A.4.1.2.6 丁二酮肟乙醇溶液： $10\mathrm{g / L}$ ，称取 $1\mathrm{g}$ 丁二酮肟，溶解于 $100~\mathrm{mL}$ 的 $95\%$ 乙醇中。  
A.4.1.2.7 硝酸银溶液： $17\mathrm{g / L}$

# A.4.1.3 仪器和设备

A.4.1.3.1 电热恒温水浴。  
A.4.1.3.2 玻璃砂坩埚：滤板孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$   
A.4.1.3.3 电热恒温干燥箱： $120^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$

# A.4.1.4 分析步骤

# A.4.1.4.1 干燥试样的制备

称取约 $5\mathrm{g}$ 湿试样，置于 $20~\mathrm{mL}$ 烧杯中，加 $10~\mathrm{mL}$ 无水乙醇，摇动烧杯，静置，待乙醇澄清后倾去，重复以上步骤5次。将试样转移至预先于 $100^{\circ}\mathrm{C} \sim 105^{\circ}\mathrm{C}$ 的电热恒温干燥箱中干燥至质量恒定的 $30~\mathrm{mL}$ 的圆底烧瓶中，在约 $60^{\circ}\mathrm{C}$ 电热恒温水浴上真空加热干燥 $5\mathrm{h}$ ，用氮气将烧瓶恢复到常压并冷却至室温，称量烧瓶和干燥试样的质量。

警示：海绵镍干燥时易自燃，处理时应小心谨慎。

# A.4.1.4.2 试样溶液的制备

在 $200~\mathrm{mL}$ 烧杯中加入 $30~\mathrm{mL}$ 水，加入上述干燥试样，用 $50~\mathrm{mL}$ 的盐酸溶液冲洗烧瓶，洗液并入烧杯，缓慢加热烧杯使试样溶解，然后冷却至室温，用中速滤纸过滤溶液至 $250~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，得到试样溶液。

# A.4.1.4.3 测定

移取 $5\mathrm{mL}$ 试样溶液置 $200~\mathrm{mL}$ 烧杯中，加 $2\mathrm{g}$ 酒石酸和 $100~\mathrm{mL}$ 水，加热到约 $80^{\circ}\mathrm{C}$ ，加 $30~\mathrm{mL}$ 的丁二酮肟乙醇溶液，加氨水至溶液呈微碱性，将此混合物置于蒸汽浴上加热 $20\mathrm{min}$ ，用预先于 $120^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 的电热恒温干燥箱中干燥至质量恒定的玻璃砂坩埚过滤，用热水洗涤至滤液中不含氯离子（用硝酸银溶液检验），将玻璃砂坩埚置于 $120^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 电热恒温干燥箱中干燥至质量恒定。

# A.4.1.5 结果计算

海绵镍(Ni)含量的质量分数 $w_{1}$ 按式(A.1)计算：

$$
w _ {1} = \frac {\left(m _ {3} - m _ {2}\right) \times 250 \times 0.2032}{\left(m _ {1} - m _ {0}\right) \times 5} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$m_{3}$ ——玻璃砂坩埚和沉淀物质量，单位为克(g)；

$m_{2}$ ——玻璃砂坩埚的质量，单位为克 $(\mathrm{g})$ ；

250 ——试样溶液的总体积，单位为毫升 $(\mathrm{mL})$ ；

0.2032——丁二酮肟镍换算成镍的系数；

$m_{1}$ ——烧瓶和干燥试样的质量，单位为克 $(\mathrm{g})$ ；

$m_0$ ——烧瓶的质量，单位为克 $(\mathrm{g})$ ；

# 5 ——分取试样溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

试验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A.4.2 方法二——适用于负载型镍

# A.4.2.1 方法提要

试样经灰化后加酸溶解，在氨性介质中，镍与丁二酮肟乙醇溶液生成红色丁二酮肟镍的沉淀，然后洗净沉淀烘干称重。

# A.4.2.2 试剂和材料

A.4.2.2.1 酒石酸。

A.4.2.2.2 氨水。

A.4.2.2.3 盐酸。

A.4.2.2.4 丁二酮肟乙醇溶液： $10\mathrm{g / L}$ ，称取 $1\mathrm{g}$ 丁二酮肟，溶解于 $100~\mathrm{mL}$ 的 $95\%$ 乙醇中。

A.4.2.2.5 硝酸银溶液： $17\mathrm{g / L}$

# A.4.2.3 仪器和设备

A.4.2.3.1 电热恒温水浴。

A.4.2.3.2 玻璃砂坩埚：滤板孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$

A.4.2.3.3 高温炉： $650^{\circ}\mathrm{C}\pm 20^{\circ}\mathrm{C}$

A.4.2.3.4 电热恒温干燥箱： $120^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$

# A.4.2.4 分析步骤

# A.4.2.4.1 试样溶液的制备

在 $100 \mathrm{~mL}$ 瓷坩埚中放入一半无灰滤纸的纸浆。称取约 $2 \mathrm{~g}$ 试样，精确至 $0.0002 \mathrm{~g}$ ，放于纸浆上部。置于电炉上缓慢升温，使硬脂酸酯缓慢地融入纸浆，使有机物质缓慢燃烧、炭化，再将瓷坩埚放入高温炉中，在 $650^{\circ} \mathrm{C} \pm 20^{\circ} \mathrm{C}$ 下继续加热 $2 \mathrm{~h}$ ，至碳化完全，冷却，加 $20 \mathrm{~mL}$ 盐酸，全部移入 $200 \mathrm{~mL}$ 烧杯中，在蒸汽浴上小心蒸发至干，冷却，加 $20 \mathrm{~mL}$ 盐酸，温热助溶（试样中含有硅不能完全溶解），移入 $500 \mathrm{~mL}$ 容量瓶中，用水稀释至刻度摇匀。

# A.4.2.4.2 测定

用中速滤纸干过滤，移取 $50~\mathrm{mL}$ 滤液置于 $500~\mathrm{mL}$ 烧杯中，用水稀释至 $250~\mathrm{mL}$ ，按照上述海绵镍的分析步骤A.4.1.4.3从“加 $2\mathrm{g}$ 酒石酸和 $100~\mathrm{mL}$ 水……”开始进行操作。

# A.4.2.5 结果计算

负载型镍(Ni)含量的质量分数 $\omega_{2}$ 按式(A.2)计算：

$$
w _ {2} = \frac {\left(m _ {1} - m _ {0}\right) \times 500 \times 0.2032}{m \times 50} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$m_{1}$ ——玻璃砂坩埚和沉淀物的质量，单位为克(g)；

$m_0$ ——玻璃砂坩埚的质量，单位为克 $(\mathrm{g})$ ；

500 ——试样溶液的总体积，单位为毫升 $(\mathrm{mL})$ ；

0.2032——丁二酮肟镍换算成镍的系数；

$m$ ——试样的质量，单位为克(g)；

50 ——分取试样溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

试验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

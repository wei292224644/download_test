2016-12-23 发布

# 前言

本标准代替GB/T5009.6—2003《食品中脂肪的测定》、GB/T9695.1—2008《肉与肉制品游离脂肪含量测定》、GB5413.3—2010《食品安全国家标准婴幼儿食品和乳品中脂肪的测定》、GB/T9695.7—2008《肉与肉制品总脂肪含量测定》、GB/T14772—2008《食品中粗脂肪的测定》、GB/T5512—2008《粮油检验粮食中粗脂肪含量测定》、GB/T15674—2009《食用菌中粗脂肪含量的测定》、GB/T22427.3—2008《淀粉总脂肪测定》、GB/T10359—2008《油料饼粕含油量的测定第1部分：己烷(或石油醚)提取法》。

本标准与GB/T5009.6—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中脂肪的测定”；  
——修改了肉制品、淀粉的酸水解及抽提步骤；  
——增加了碱水解法、盖勃法。

# 食品安全国家标准

# 食品中脂肪的测定

# 1 范围

本标准规定了食品中脂肪含量的测定方法。

本标准第一法适用于水果、蔬菜及其制品、粮食及粮食制品、肉及肉制品、蛋及蛋制品、水产及其制品、焙烤食品、糖果等食品中游离态脂肪含量的测定。

本标准第二法适用于水果、蔬菜及其制品、粮食及粮食制品、肉及肉制品、蛋及蛋制品、水产及其制品、焙烤食品、糖果等食品中游离态脂肪及结合态脂肪总量的测定。

本标准第三法适用于乳及乳制品、婴幼儿配方食品中脂肪的测定。

本标准第四法适用于乳及乳制品、婴幼儿配方食品中脂肪的测定。

# 第一法 索氏抽提法

# 2 原理

脂肪易溶于有机溶剂。试样直接用无水乙醚或石油醚等溶剂抽提后，蒸发除去溶剂，干燥，得到游离态脂肪的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

# 3.2 材料

3.1.1 无水乙醚 $(\mathrm{C_4H_{10}O})$ 。  
3.1.2 石油醚 $\left(\mathrm{C}_{n} \mathrm{H}_{2 n + 2}\right)$ ：石油醚沸程为 $30^{\circ} \mathrm{C} \sim 60^{\circ} \mathrm{C}$ 。  
3.2.1 石英砂。  
3.2.2 脱脂棉。

# 4 仪器和设备

4.1 索氏抽提器。  
4.2 恒温水浴锅。  
4.3 分析天平：感量 $0.001\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。

4.4 电热鼓风干燥箱。  
4.5 干燥器：内装有效干燥剂，如硅胶。  
4.6 滤纸筒。  
4.7 蒸发皿。

# 5 分析步骤

# 5.1 试样处理

5.1.1 固体试样：称取充分混匀后的试样 $2\mathrm{g}\sim 5\mathrm{g}$ ，准确至 $0.001\mathrm{g}$ ，全部移入滤纸筒内。  
5.1.2 液体或半固体试样：称取混匀后的试样 $5\mathrm{g}\sim 10\mathrm{g}$ ，准确至 $0.001\mathrm{g}$ ，置于蒸发皿中，加入约 $20\mathrm{g}$ 石英砂，于沸水浴上蒸干后，在电热鼓风干燥箱中于 $100^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 干燥 $30\mathrm{min}$ 后，取出，研细，全部移入滤纸筒内。蒸发皿及粘有试样的玻璃棒，均用沾有乙醚的脱脂棉擦净，并将棉花放入滤纸筒内。

# 5.2 抽提

将滤纸筒放入索氏抽提器的抽提筒内，连接已干燥至恒重的接收瓶，由抽提器冷凝管上端加入无水乙醚或石油醚至瓶内容积的三分之二处，于水浴上加热，使无水乙醚或石油醚不断回流抽提（6次/h～8次/h)，一般抽提 $6\mathrm{h}\sim 10\mathrm{h}$ 。提取结束时，用磨砂玻璃棒接取1滴提取液，磨砂玻璃棒上无油斑表明提取完毕。

# 5.3 称量

取下接收瓶, 回收无水乙醚或石油醚, 待接收瓶内溶剂剩余 $1 \mathrm{~mL} \sim 2 \mathrm{~mL}$ 时在水浴上蒸干, 再于 $100^{\circ} \mathrm{C} \pm 5^{\circ} \mathrm{C}$ 干燥 $1 \mathrm{~h}$ , 放干燥器内冷却 $0.5 \mathrm{~h}$ 后称量。重复以上操作直至恒重 (直至两次称量的差不超过 $2 \mathrm{mg}$ )。

# 6 分析结果的表述

试样中脂肪的含量按式(1)计算：

$$
X = \frac {m _ {1} - m _ {0}}{m _ {2}} \times 1 0 0 \tag {1}
$$

式中：

$X$ ——试样中脂肪的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$m_{1}$ ——恒重后接收瓶和脂肪的含量，单位为克(g)；

$m_0$ ——接收瓶的质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

100——换算系数。

计算结果表示到小数点后一位。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 酸水解法

# 8 原理

食品中的结合态脂肪必须用强酸使其游离出来，游离出的脂肪易溶于有机溶剂。试样经盐酸水解后用无水乙醚或石油醚提取，除去溶剂即得游离态和结合态脂肪的总含量。

# 9 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 9.1 试剂

9.1.1 盐酸（HCl）。  
9.1.2 乙醇 $\mathrm{(C_2H_5OH)}$ 。  
9.1.3 无水乙醚 $(\mathrm{C_4H_{10}O})$ 。  
9.1.4 石油醚 $\left(\mathrm{C}_{n} \mathrm{H}_{2 n + 2}\right)$ ：沸程为 $30^{\circ} \mathrm{C} \sim 60^{\circ} \mathrm{C}$ 。  
9.1.5 碘 $(\mathrm{I}_2)$ 。  
9.1.6 碘化钾(KI)。

# 9.2 试剂的配制

9.2.1 盐酸溶液 $(2\mathrm{mol} / \mathrm{L})$ ：量取 $50~\mathrm{mL}$ 盐酸，加入到 $250~\mathrm{mL}$ 水中，混匀。  
9.2.2 碘液 $(0.05\mathrm{mol} / \mathrm{L})$ ：称取 $6.5\mathrm{g}$ 碘和 $25\mathrm{g}$ 碘化钾于少量水中溶解，稀释至 $1\mathrm{L}$ 。

# 9.3 材料

9.3.1 蓝色石蕊试纸。  
9.3.2 脱脂棉。  
9.3.3 滤纸：中速。

# 10 仪器和设备

10.1 恒温水浴锅。  
10.2 电热板：满足 $200^{\circ}\mathrm{C}$ 高温。  
10.3 锥形瓶。  
10.4 分析天平：感量为 $0.1\mathrm{g}$ 和 $0.001\mathrm{g}$ 。  
10.5 电热鼓风干燥箱。

# 11 分析步骤

# 11.1 试样酸水解

# 11.1.1 肉制品

称取混匀后的试样 $3\mathrm{g}\sim 5\mathrm{g}$ ，准确至 $0.001\mathrm{g}$ ，置于锥形瓶 $(250~\mathrm{mL})$ 中，加入 $50~\mathrm{mL}~2~\mathrm{mol / L}$ 盐酸溶

液和数粒玻璃细珠, 盖上表面皿, 于电热板上加热至微沸, 保持 $1 \mathrm{~h}$ , 每 $10 \mathrm{~min}$ 旋转摇动 1 次。取下锥形瓶, 加入 $150 \mathrm{~mL}$ 热水, 混匀, 过滤。锥形瓶和表面皿用热水洗净, 热水一并过滤。沉淀用热水洗至中性 (用蓝色石蕊试纸检验, 中性时试纸不变色)。将沉淀和滤纸置于大表面皿上, 于 $100^{\circ} \mathrm{C} \pm 5^{\circ} \mathrm{C}$ 干燥箱内干燥 $1 \mathrm{~h}$ , 冷却。

# 11.1.2 淀粉

根据总脂肪含量的估计值，称取混匀后的试样 $25\mathrm{g}\sim 50\mathrm{g}$ ，准确至 $0.1\mathrm{g}$ ，倒入烧杯并加入 $100~\mathrm{mL}$ 水。将 $100~\mathrm{mL}$ 盐酸缓慢加到 $200~\mathrm{mL}$ 水中，并将该溶液在电热板上煮沸后加入样品液中，加热此混合液至沸腾并维持 $5\mathrm{min}$ ，停止加热后，取几滴混合液于试管中，待冷却后加入1滴碘液，若无蓝色出现，可进行下一步操作。若出现蓝色，应继续煮沸混合液，并用上述方法不断地进行检查，直至确定混合液中不含淀粉为止，再进行下一步操作。

将盛有混合液的烧杯置于水浴锅 $(70^{\circ}\mathrm{C} \sim 80^{\circ}\mathrm{C})$ 中 $30\mathrm{min}$ ，不停地搅拌，以确保温度均匀，使脂肪析出。用滤纸过滤冷却后的混合液，并用干滤纸片取出粘附于烧杯内壁的脂肪。为确保定量的准确性，应将冲洗烧杯的水进行过滤。在室温下用水冲洗沉淀和干滤纸片，直至滤液用蓝色石蕊试纸检验不变色。将含有沉淀的滤纸和干滤纸片折叠后，放置于大表面皿上，在 $100^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 的电热恒温干燥箱内干燥 $1\mathrm{h}$ 。

# 11.1.3 其他食品

11.1.3.1 固体试样：称取约 $2\mathrm{g}\sim 5\mathrm{g}$ ，准确至 $0.001\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 试管内，加入 $8\mathrm{mL}$ 水，混匀后再加 $10~\mathrm{mL}$ 盐酸。将试管放入 $70^{\circ}\mathrm{C}\sim 80^{\circ}\mathrm{C}$ 水浴中，每隔 $5\mathrm{min}\sim 10\mathrm{min}$ 以玻璃棒搅拌1次，至试样消化完全为止，约 $40\mathrm{min}\sim 50\mathrm{min}$ 。  
11.1.3.2 液体试样：称取约 $10\mathrm{g}$ ，准确至 $0.001\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 试管内，加 $10~\mathrm{mL}$ 盐酸。其余操作同11.1.3.1。

# 11.2 抽提

# 11.2.1 肉制品、淀粉

将干燥后的试样装入滤纸筒内，其余抽提步骤同5.2。

# 11.2.2 其他食品

取出试管，加入 $10\mathrm{mL}$ 乙醇，混合。冷却后将混合物移入 $100~\mathrm{mL}$ 具塞量筒中，以 $25~\mathrm{mL}$ 无水乙醚分数次洗试管，一并倒入量筒中。待无水乙醚全部倒入量筒后，加塞振摇 $1\mathrm{min}$ ，小心开塞，放出气体，再塞好，静置 $12\mathrm{min}$ ，小心开塞，并用乙醚冲洗塞及量筒口附着的脂肪。静置 $10\mathrm{min} \sim 20\mathrm{min}$ ，待上部液体清晰，吸出上清液于已恒重的锥形瓶内，再加 $5\mathrm{mL}$ 无水乙醚于具塞量筒内，振摇，静置后，仍将上层乙醚吸出，放入原锥形瓶内。

# 11.3 称量

同5.3。

# 12 分析结果的表述

同6。

# 13 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第三法 碱水解法

# 14 原理

用无水乙醚和石油醚抽提样品的碱(氨水)水解液，通过蒸馏或蒸发去除溶剂，测定溶于溶剂中的抽提物的质量。

# 15 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 15.1 试剂

15.1.1 淀粉酶：酶活力 $\geqslant 1.5\mathrm{U / mg}$   
15.1.2 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ：质量分数约 $25\%$

注：可使用比此浓度更高的氨水。

15.1.3 乙醇 $\mathrm{(C_2H_5OH)}$ ：体积分数至少为 $95\%$   
15.1.4 无水乙醚 $(\mathrm{C_4H_{10}O})$ 。  
15.1.5 石油醚 $(\mathrm{C}_n\mathrm{H}_{2n + 2})$ ：沸程为 $30^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$   
15.1.6 刚果红 $\left(\mathrm{C}_{32} \mathrm{H}_{22} \mathrm{~N}_{6} \mathrm{Na}_{2} \mathrm{O}_{6} \mathrm{~S}_{2}\right)$ 。  
15.1.7 盐酸（HCl）。  
15.1.8 碘 $(\mathrm{I}_2)$ 。

# 15.2 试剂配制

15.2.1 混合溶剂：等体积混合乙醚和石油醚，现用现配。  
15.2.2 碘溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取碘 $12.7\mathrm{g}$ 和碘化钾 $25\mathrm{g}$ ，于水中溶解并定容至 $1\mathrm{L}$ 。  
15.2.3 刚果红溶液：将 $1\mathrm{g}$ 刚果红溶于水中，稀释至 $100~\mathrm{mL}$

注：可选择性地使用。刚果红溶液可使溶剂和水相界面清晰，也可使用其他能使水相染色而不影响测定结果的溶液。

15.2.4 盐酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ：量取 $50~\mathrm{mL}$ 盐酸缓慢倒入 $40~\mathrm{mL}$ 水中，定容至 $100~\mathrm{mL}$ ，混匀。

# 16 仪器和设备

16.1 分析天平：感量为 $0.0001\mathrm{g}$ 。  
16.2 离心机：可用于放置抽脂瓶或管，转速为 $500\mathrm{r / min}\sim 600\mathrm{r / min}$ ，可在抽脂瓶外端产生 $80\mathrm{g}\sim 90\mathrm{g}$ 的重力场。  
16.3 电热鼓风干燥箱。  
16.4 恒温水浴锅。  
16.5 干燥器：内装有效干燥剂，如硅胶。  
16.6 抽脂瓶：抽脂瓶应带有软木塞或其他不影响溶剂使用的瓶塞（如硅胶或聚四氟乙烯）。软木塞应

先浸泡于乙醚中，后放入 $60~\mathrm{^\circ C}$ 或 $60~^\circ \mathrm{C}$ 以上的水中保持至少 $15\mathrm{min}$ ，冷却后使用。不用时需浸泡在水中，浸泡用水每天更换1次。

注：也可使用带虹吸管或洗瓶的抽脂管(或烧瓶)，但操作步骤有所不同，见附录A中规定。接头的内部长支管下端可成勺状。

# 17 分析步骤

# 17.1 试样碱水解

# 17.1.1 巴氏杀菌乳、灭菌乳、生乳、发酵乳、调制乳

称取充分混匀试样 $10\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）于抽脂瓶中。加入 $2.0\mathrm{mL}$ 氨水，充分混合后立即将抽脂瓶放入 $65~^\circ \mathrm{C}\pm 5~^\circ \mathrm{C}$ 的水浴中，加热 $15\mathrm{min}\sim 20\mathrm{min}$ ，不时取出振荡。取出后，冷却至室温。静置 $30~\mathrm{s}$ 。

# 17.1.2 乳粉和婴幼儿食品

称取混匀后的试样，高脂乳粉、全脂乳粉、全脂加糖乳粉和婴幼儿食品约 $1\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），脱脂乳粉、乳清粉、酪乳粉约 $1.5\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），其余操作同17.1.1。

# 17.1.2.1 不含淀粉样品

加入 $10\mathrm{mL}65^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}$ 的水，将试样洗入抽脂瓶的小球，充分混合，直到试样完全分散，放入流动水中冷却。

# 17.1.2.2 含淀粉样品

将试样放入抽脂瓶中，加入约 $0.1\mathrm{g}$ 的淀粉酶，混合均匀后，加入 $8\mathrm{mL} \sim 10\mathrm{mL}45^{\circ}\mathrm{C}$ 的水，注意液面不要太高。盖上瓶塞于搅拌状态下，置 $65^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 水浴中 $2\mathrm{h}$ ，每隔 $10\mathrm{min}$ 摇混1次。为检验淀粉是否水解完全可加入2滴约 $0.1\mathrm{mol/L}$ 的碘溶液，如无蓝色出现说明水解完全，否则将抽脂瓶重新置于水浴中，直至无蓝色产生。抽脂瓶冷却至室温。

其余操作同17.1.1。

# 17.1.3 炼乳

脱脂炼乳、全脂炼乳和部分脱脂炼乳称取约 $3\mathrm{g}\sim 5\mathrm{g}$ 、高脂炼乳称取约 $1.5\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )，用 $10~\mathrm{mL}$ 水，分次洗入抽脂瓶小球中，充分混合均匀。其余操作同17.1.1。

# 17.1.4 奶油、稀奶油

先将奶油试样放入温水浴中溶解并混合均匀后，称取试样约 $0.5\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )，稀奶油称取约 $1$ g 于抽脂瓶中，加入 $8\mathrm{mL} \sim 10\mathrm{mL}$ 约 $45^{\circ}\mathrm{C}$ 的水。再加 $2\mathrm{mL}$ 氨水充分混匀。其余操作同 17.1.1。

# 17.1.5 干酪

称取约 $2\mathrm{g}$ 研碎的试样(精确至 $0.0001\mathrm{g}$ )于抽脂瓶中，加 $10~\mathrm{mL}6\mathrm{mol / L}$ 盐酸，混匀，盖上瓶塞，于沸水中加热 $20\mathrm{min}\sim 30\mathrm{min}$ ，取出冷却至室温，静置 $30~\mathrm{s}$ 。

# 17.2 抽提

17.2.1 加入 $10 \mathrm{~mL}$ 乙醇，缓和但彻底地进行混合，避免液体太接近瓶颈。如果需要，可加入2滴刚果红溶液。  
17.2.2 加入 $25~\mathrm{mL}$ 乙醚，塞上瓶塞，将抽脂瓶保持在水平位置，小球的延伸部分朝上夹到摇混器上，按

约100次/min振荡 $1\mathrm{min}$ ，也可采用手动振摇方式。但均应注意避免形成持久乳化液。抽脂瓶冷却后小心地打开塞子，用少量的混合溶剂冲洗塞子和瓶颈，使冲洗液流入抽脂瓶。

17.2.3 加入 $25~\mathrm{mL}$ 石油醚，塞上重新润湿的塞子，按17.2.2所述，轻轻振荡 $30~\mathrm{s}$ 。  
17.2.4 将加塞的抽脂瓶放入离心机中，在 $500\mathrm{r / min}\sim 600\mathrm{r / min}$ 下离心 $5\mathrm{min}$ ，否则将抽脂瓶静置至少 $30\mathrm{min}$ ，直到上层液澄清，并明显与水相分离。  
17.2.5 小心地打开瓶塞，用少量的混合溶剂冲洗塞子和瓶颈内壁，使冲洗液流入抽脂瓶。

# 17.3 称量

合并所有提取液，既可采用蒸馏的方法除去脂肪收集瓶中的溶剂，也可于沸水浴上蒸发至干来除掉溶剂。蒸馏前用少量混合溶剂冲洗瓶颈内部。将脂肪收集瓶放入 $100^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 的烘箱中干燥 $1\mathrm{h}$ ，取出后置于干燥器内冷却 $0.5\mathrm{h}$ 后称量。重复以上操作直至恒重（直至两次称量的差不超过 $2\mathrm{mg}$ ）。

# 18 分析结果的表述

试样中脂肪的含量按式(2)计算：

$$
X = \frac {\left(m _ {1} - m _ {2}\right) - \left(m _ {3} - m _ {4}\right)}{m} \times 1 0 0 \tag {2}
$$

式中：

$X$ ——试样中脂肪的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$m_{1}$ ——恒重后脂肪收集瓶和脂肪的质量，单位为克(g)；

$m_{2}$ ——脂肪收集瓶的质量，单位为克 $(\mathrm{g})$ ；

$m_{3}$ ——空白试验中，恒重后脂肪收集瓶和抽提物的质量，单位为克(g)；

$m_{4}$ ——空白试验中脂肪收集瓶的质量，单位为克 $(\mathrm{g})$ ；

$m$ ——样品的质量，单位为克(g)；

100——换算系数。

结果保留3位有效数字。

# 19 精密度

当样品中脂肪含量 $\geqslant 15\%$ 时，两次独立测定结果之差 $\leqslant 0.3\mathrm{g} / 100\mathrm{g}$

当样品中脂肪含量在 $5\% \sim 15\%$ 时，两次独立测定结果之差 $\leqslant 0.2\mathrm{g} / 100\mathrm{g}$

当样品中脂肪含量 $\leqslant 5\%$ 时，两次独立测定结果之差 $\leqslant 0.1\mathrm{g} / 100\mathrm{g}$ 。

# 附录A

# 使用带虹吸管或洗瓶的抽脂管的操作步骤

# A.1 试样碱水解

# A.1.1 巴氏杀菌、灭菌乳、生乳、发酵乳、调制乳

称取充分混匀样品 $10\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于抽脂管底部。加入 $2\mathrm{mL}$ 氨水，与管底部已稀释的样品彻底混合。将抽脂管放入 $65~^\circ \mathrm{C}\pm 5~^\circ \mathrm{C}$ 的水浴中，加热 $15\mathrm{min}\sim 20\mathrm{min}$ ，偶尔振荡样品管，然后冷却至室温。

# A.1.2 乳粉及乳基婴幼儿食品

称取混匀后的样品高脂乳粉、全脂乳粉、全脂加糖乳粉和乳基婴幼儿配方食品：约 $1\mathrm{g}$ ，脱脂乳粉、乳清粉、酪乳粉：约 $1.5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ），于抽脂管底部，加入 $10\mathrm{mL}65^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 的水，充分混合，直到样品完全分散，放入流动水中冷却。其余操作同A.1.1。

# A.1.3 炼乳

脱脂炼乳称取约 $10\mathrm{g}$ 、全脂炼乳和部分脱脂炼乳称取约 $3\mathrm{g} \sim 5\mathrm{g}$ ；高脂炼乳称取约 $1.5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )，于抽脂管底部。加入 $10\mathrm{mL}$ 水，充分混合均匀。其余操作同A.1.1。

# A.1.4 奶油、稀奶油

先将奶油样品放入温水浴中溶解并混合均匀后，奶油称取约 $0.5\mathrm{g}$ 样品，稀奶油称取 $1\mathrm{g}$ 于抽脂管底部（精确至 $0.001\mathrm{g}$ ）。其余操作同A.1.1。

# A.1.5 干酪

称取约 $2\mathrm{g}$ 研碎的样品（精确至 $0.001\mathrm{g}$ ）。加水 $9\mathrm{mL}$ 、氨水 $2\mathrm{mL}$ ，用玻璃棒搅拌均匀后微微加热使酪蛋白溶解，用盐酸中和后再加盐酸 $10\mathrm{mL}$ ，加海砂 $0.5\mathrm{g}$ ，盖好玻璃盖，以文火煮沸 $5\mathrm{min}$ ，冷却后将烧杯内容物移入抽脂管底部，用 $25\mathrm{mL}$ 无水乙醚冲洗烧杯，洗液并入抽脂管中。

# A.2 抽提

A.2.1 加入 $10\mathrm{mL}$ 无水乙醇，在管底部轻轻彻底地混合，必要时加入两滴刚果红溶液。

A.2.2 加入 $25 \mathrm{~mL}$ 无水乙醚, 加软木塞 (已被水饱和), 或用水浸湿的其他瓶塞, 上下反转 $1 \mathrm{~min}$ , 不要过度 (避免形成持久性乳化液)。必要时, 将管子放入流动的水中冷却, 然后小心地打开软木塞, 用少量的混合溶剂 (使用洗瓶) 冲洗塞子和管颈, 使冲洗液流入管中。

A.2.3 加入 $25~\mathrm{mL}$ 石油醚，加塞（塞子重新用水润湿），按A.2.2所述轻轻振荡 $30~\mathrm{s}$ 。

A.2.4 将加塞的管子放入离心机中，在 $500\mathrm{r / min}\sim 600\mathrm{r / min}$ 下离心 $1\mathrm{min}\sim 5\mathrm{min}$ 。或静置至少 $30\mathrm{min}$ ，直到上层液澄清，并明显地与水相分离，冷却。

A.2.5 小心地打开软木塞，用少量混合溶剂洗塞子和管颈，使冲洗液流入管中。

A.2.6 将虹吸管或洗瓶接头插入管中，向下压长支管，直到距两相界面的上方 $4\mathrm{mm}$ 处，内部长支管应与管轴平行。

小心地将上层液移入含有沸石的脂肪收集瓶中，也可用金属皿。避免移入任何水相。用少量混合

溶剂冲洗长支管的出口，收集冲洗液于脂肪收集瓶中。

A.2.7 松开管颈处的接头，用少量的混合溶剂冲洗接头和内部长支管的较低部分，重新插好接头，将冲洗液移入脂肪收集瓶中。

用少量的混合溶剂冲洗出口，冲洗液收集于瓶中，必要时，按17.3所述，通过蒸馏或蒸发去除部分溶剂。

A.2.8 再松开管颈处的接头，微微抬高接头，加入 $5\mathrm{mL}$ 乙醇，用乙醇冲洗长支管，如A.2.1所述混合。  
A.2.9 重复 $\mathrm{A}.2.2\sim \mathrm{A}.2.7$ 步骤进行第2次抽提，但仅用 $15~\mathrm{mL}$ 乙醚和 $15~\mathrm{mL}$ 石油醚，抽提之后，在移开管接头时，用乙醚冲洗内部长支管。  
A.2.10 重复 $\mathrm{A}.2.2\sim \mathrm{A}.2.7$ 步骤，不加乙醇，进行第3次抽提，仅用 $15~\mathrm{mL}$ 无水乙醚和 $15~\mathrm{mL}$ 石油醚。注：如果产品中脂肪的质量分数低于 $5\%$ ，可省略第3次抽提。  
A.2.11 以下按17.3所述进行。

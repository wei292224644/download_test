2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替GB/T22222—2008《食品中木糖醇、山梨醇、麦芽糖醇的测定 高效液相色谱法》。

本标准与GB/T22222—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中木糖醇、山梨醇、麦芽糖醇、赤藓糖醇的测定”；  
——修改了样品前处理方法；  
——增加了第二法高效液相色谱-蒸发光散射检测法；  
——增加了食品中赤藓糖醇的检测。

# 食品安全国家标准

# 食品中木糖醇、山梨醇、麦芽糖醇、赤藓糖醇的测定

# 1 范围

本标准规定了口香糖、饼干、糕点、面包、饮料中木糖醇、山梨醇、麦芽糖醇、赤藓糖醇的高效液相色谱-示差折光检测和蒸发光散射检测测定方法。

本标准适用于口香糖、饼干、糕点、面包、饮料中木糖醇、山梨醇、麦芽糖醇、赤藓糖醇含量的测定。

# 第一法 高效液相色谱-示差折光检测法

# 2 原理

试样经沉淀蛋白质后过滤，上清液进高效液相色谱仪，经氨基色谱柱或阳离子交换色谱柱分离，示差折光检测器检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 三氯乙酸 $(\mathrm{CCl}_3\mathrm{COOH})$ 。  
3.1.3 无水碳酸钠 $\mathrm{(Na_2CO_3)}$ 。

# 3.2 试剂配制

3.2.1 三氯乙酸溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 三氯乙酸，加水溶解并定容至 $100~\mathrm{mL}$ 。  
3.2.2 碳酸钠溶液 $(21.2\mathrm{g} / \mathrm{L})$ ：称取 $2.12\mathrm{g}$ 碳酸钠，加水溶解并定容至 $100~\mathrm{mL}$ ，现用现配。

# 3.3 标准品

3.3.1 木糖醇 $\left(\mathrm{C}_{5} \mathrm{H}_{12} \mathrm{O}_{5}\right)$ , CAS 号: 87-99-0), 纯度 $\geqslant 99\%$ , 或经国家认证并授予标准物质证书的标准物质。  
3.3.2 山梨醇 $\left(\mathrm{C}_{6} \mathrm{H}_{14} \mathrm{O}_{6}\right)$ , CAS 号: 50-70-4), 纯度 $\geqslant 99\%$ , 或经国家认证并授予标准物质证书的标准物质。  
3.3.3麦芽糖醇 $\mathrm{(C_{12}H_{24}O_{11}}$ ，CAS号：585-88-6)，纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.4 赤藓糖醇 $\mathrm{(C_4H_{10}O_4}$ ,CAS号：149-32-6)，纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准

物质。

# 3.4 标准溶液配制

3.4.1 标准储备液 $(40\mathrm{mg / mL})$ ：分别称取 $400~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ）木糖醇、山梨醇、麦芽糖醇、赤藓糖醇标准品，加水定容至 $10~\mathrm{mL}$ ，放置 $4^{\circ}\mathrm{C}$ 密封可贮藏1个月。  
3.4.2 标准工作液：分别准确移取各种糖醇标准储备液 $40~\mu \mathrm{L},60~\mu \mathrm{L},80~\mu \mathrm{L},100~\mu \mathrm{L},120~\mu \mathrm{L},150~\mu \mathrm{L}$ 加水定容至 $1\mathrm{mL}$ ，配制成质量浓度分别为 $1.6\mathrm{mg / mL},2.4\mathrm{mg / mL},3.2\mathrm{mg / mL},4.0\mathrm{mg / mL},4.8\mathrm{mg / }$ $\mathrm{mL},6.0\mathrm{mg / mL}$ 的混合系列标准工作溶液。

# 4 仪器和设备

4.1 高效液相色谱仪：具有示差折光检测器。  
4.2 色谱柱：氨基色谱柱（内径 $4.6 \mathrm{~mm}$ ，柱长 $250 \mathrm{~mm}$ ，粒径 $5 \mu \mathrm{m}$ ）或阳离子交换色谱柱（内径 $6.5 \mathrm{~mm}$ ，柱长 $300 \mathrm{~mm}$ ）。  
4.3 食品粉碎机。  
4.4 分析天平：感量为 $0.1\mathrm{mg},0.01\mathrm{g}$   
4.5 高速离心机：转速 $\geq 9500\mathrm{r / min}$   
4.6 超声波清洗机：工作频率 $40\mathrm{kHz}$ ，功率 $500\mathrm{W}$ 。

# 5 分析步骤

# 5.1 试样制备及前处理

# 5.1.1 口香糖

取口香糖样品至少 $20\mathrm{g}$ ，用刀片切成小碎块，置于密闭的容器内混匀。准确称取 $2\mathrm{g}$ 左右切碎的样品，置于 $50~\mathrm{mL}$ 离心管中，加入 $40~\mathrm{mL}$ 水，混匀后置于 $80^{\circ}\mathrm{C}$ 水浴锅中加热 $20\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振荡混匀，取出后 $9000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取 $8\mathrm{mL}$ 上清液置于 $10~\mathrm{mL}$ 容量瓶中，加水定容、摇匀， $0.22\mu \mathrm{m}$ 滤膜过滤后，上机测试。

# 5.1.2 饮料

对非蛋白饮料类，取样品至少 $200~\mathrm{mL}$ ，充分混匀，置于密闭的容器内。称取 $10\mathrm{g}$ 饮料于 $50~\mathrm{mL}$ 容量瓶中，加水定容至 $50~\mathrm{mL}$ ，摇匀， $0.22\mu \mathrm{m}$ 滤膜过滤后，上机测试。

对蛋白饮料类，取样品至少 $200\mathrm{g}$ ，置于密闭的容器内混匀。称取样品 $5\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 容量瓶中，加入 $35~\mathrm{mL}$ 水，摇匀后超声 $30\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振荡混匀，取出后 $9000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。

沉淀蛋白质：上清液中加入三氯乙酸溶液 $(100\mathrm{g / L})5\mathrm{mL}$ ，摇匀后室温放置 $30\mathrm{min},9500\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取 $8\mathrm{mL}$ 上清液于 $10~\mathrm{mL}$ 容量瓶中并加水定容，摇匀后取滤液 $850~\mu \mathrm{L}$ ，加入碳酸钠溶液(21.2 $\mathrm{g / L})150~\mu \mathrm{L}$ ，摇匀中和；或取 $10~\mathrm{mL}$ 上清液加入 $20~\mathrm{mL}$ 乙腈，摇匀后室温放置 $30\mathrm{min},9500\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，上清定容至 $50~\mathrm{mL}$ ，摇匀。 $0.22\mu \mathrm{m}$ 的微孔滤膜过滤，上机测试。

注：对糖醇含量较低，经乙腈沉淀稀释后低于检出限的样品，应采用三氯乙酸沉淀。对赤藓糖醇含量较低 $(\leqslant 1\%)$ 的样品，应采用乙腈沉淀。其他情况两种方法均可。

# 5.1.3 饼干、糕点、面包

取样品至少 $200\mathrm{g}$ ，用粉碎机粉碎，置于密闭的容器内混匀。称取粉碎的样品 $1\mathrm{g}\sim 5\mathrm{g}$ ，置于 $50~\mathrm{mL}$

离心管中，加入 $40~\mathrm{mL}$ 水，摇匀后超声 $30\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振荡混匀，取出后 $9000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。

沉淀蛋白质：上清液中加入三氯乙酸溶液 $5\mathrm{mL}$ ，摇匀后室温放置 $30\mathrm{min}$ ， $9500\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取 $8\mathrm{mL}$ 上清液于 $10\mathrm{mL}$ 容量瓶中并加水定容，摇匀后取滤液 $850~\mu \mathrm{L}$ ，加入碳酸钠溶液 $150~\mu \mathrm{L}$ ，摇匀中和；或取 $10\mathrm{mL}$ 上清液加入 $20\mathrm{mL}$ 乙腈，摇匀后室温放置 $30\mathrm{min}$ ， $9500\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，上清定容至 $50~\mathrm{mL}$ ，摇匀。 $0.22~\mu \mathrm{m}$ 滤膜过滤后，上机测试。

注：对糖醇含量较低，经乙腈沉淀稀释后低于检出限的样品，应采用三氯乙酸沉淀。对赤藓糖醇含量较低 $(\leqslant 1\%)$ 的样品，应采用乙腈沉淀。其他情况两种方法均可。

# 5.2 仪器参考条件

# 5.2.1 氨基色谱柱的仪器条件

氨基色谱柱的仪器条件列出如下：

a）色谱柱：氨基柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或等效柱；  
b） 柱温： $30^{\circ}C$   
c）流动相：乙腈：水 $= 80:20$   
d) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
e) 进样量: $20 \mu \mathrm{L}$ ;   
f）检测池温度： $30^{\circ}C$ □

# 5.2.2 阳离子交换色谱柱的仪器条件

阳离子交换色谱柱的仪器条件列出如下：

a）色谱柱：阳离子交换柱，柱长 $300\mathrm{mm}$ ，内径 $6.5\mathrm{mm}$ ，或等效柱；  
b） 柱温： $80^{\circ}C$   
c）流动相：水或与色谱柱匹配的酸性水溶液；  
d) 流速: $0.5 \mathrm{~mL} / \mathrm{min}$ ;   
e) 进样量: $20 \mu \mathrm{L}$ ;   
f）检测池温度： $50^{\circ}C$ □

# 5.3 标准曲线的制作

将 $20~\mu \mathrm{L}$ 标准系列工作液分别注入高效液相色谱仪中，在5.2所述色谱条件下测定标准溶液的响应值(峰面积)，以标准工作液的浓度为横坐标，以响应值(峰面积)为纵坐标，绘制标准曲线。

# 5.4 试样溶液的测定

将 $20\mu \mathrm{L}$ 试样溶液注入高效液相色谱仪中，在5.2所述色谱条件下测定试样的响应值（峰面积），通过各个糖醇的色谱峰的保留时间定性。根据峰面积由标准曲线得到试样溶液中木糖醇、山梨醇、麦芽糖醇、赤藓糖醇的浓度。

# 6 分析结果的表述

试样中糖醇含量按式(1)计算：

$$
X = \frac {\rho \times V}{m \times 1 0 0 0} \times 1 0 0 \tag {1}
$$

式中：

$X$ ——试样中木糖醇、山梨醇、麦芽糖醇、赤藓糖醇的含量， $\%$ ；

$\rho$ ——由标准曲线获得的试样溶液中木糖醇、山梨醇、麦芽糖醇、赤藓糖醇的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V$ ——水溶液总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法对木糖醇、山梨醇、麦芽糖醇、赤藓糖醇的检出限均为 $0.4\mathrm{g} / 100\mathrm{g}$ ，定量限均为 $1.3\mathrm{g} / 100\mathrm{g}$ 。

# 12.1.3 饼干、糕点、面包

取样品至少 $200\mathrm{g}$ ，用粉碎机粉碎，置于密闭的容器内混匀。称取粉碎的样品 $1\mathrm{g} \sim 5\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 离心管中，加入 $40~\mathrm{mL}$ 水，摇匀后超声 $30\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振荡混匀，取出后 $9000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。

沉淀蛋白质：上清液中加入三氯乙酸溶液 $5\mathrm{mL}$ ，摇匀后室温放置 $30\mathrm{min}$ ， $9500\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取 $8\mathrm{mL}$ 上清液于 $10\mathrm{mL}$ 容量瓶中并加水定容，摇匀后取滤液 $850~\mu \mathrm{L}$ ，加入碳酸钠溶液 $150~\mu \mathrm{L}$ ，摇匀中和；或取 $10\mathrm{mL}$ 上清液加入 $20\mathrm{mL}$ 乙腈，摇匀后室温放置 $30\mathrm{min}$ ， $9500\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，上清定容至 $50~\mathrm{mL}$ ，摇匀。 $0.22~\mu \mathrm{m}$ 滤膜过滤后，稀释后上机测试。

注：对糖醇含量较低，经乙腈沉淀稀释后低于检出限的样品，应采用三氯乙酸沉淀。对赤藓糖醇含量较低 $(\leqslant 1\%)$ 的样品，应采用乙腈沉淀。其他情况两种方法均可。

# 12.2 仪器参考条件

仪器参考条件列出如下：

a）色谱柱：氨基柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或等效柱；  
b） 柱温： $30^{\circ}C$   
c）流动相：乙腈：水 $= 80:20$   
d) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
e) 进样量: $10 \mu \mathrm{L}$ ;   
f）漂移管温度： $83.5\%$   
g）雾化气流速： $2.1\mathrm{L / min}$

# 12.3 标准曲线的制作

将 $10\mu \mathrm{L}$ 标准系列工作液分别注入高效液相色谱仪中，在所述色谱条件下测定标准溶液的响应值（峰面积），以标准工作液的浓度的以10为底的对数值为横坐标，以响应值(峰面积)的以10为底的对数值为纵坐标，绘制标准曲线。

# 12.4 试样溶液的测定

将 $10\mu \mathrm{L}$ 试样溶液注入高效液相色谱仪中，在所述色谱条件下测定试样的响应值(峰面积)，通过各个糖醇的色谱峰在色谱图中的保留时间，确认样品中的糖醇，根据峰面积的以10为底的对数值由标准曲线计算得到试样溶液中木糖醇、山梨醇、麦芽糖醇、赤藓糖醇的浓度。

# 13 分析结果的表述

试样中糖醇含量按式(2)计算：

$$
X = \frac {\rho \times V}{m \times 1 0 0 0} \times 1 0 0 \tag {2}
$$

式中：

$X$ ——试样中木糖醇、山梨醇、麦芽糖醇、赤藓糖醇的含量， $\%$ ；

$\rho$ ——由标准曲线获得的试样溶液中木糖醇、山梨醇、麦芽糖醇、赤藓糖醇的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V$ ——水溶液总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

计算结果保留两位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

本方法对木糖醇、山梨糖醇、麦芽糖醇、赤藓糖醇的检出限分别为 $0.01\mathrm{g} / 100\mathrm{g},0.02\mathrm{g} / 100\mathrm{g}$ 产 $0.03\mathrm{g} / 100\mathrm{g}$ 和 $0.04\mathrm{g} / 100\mathrm{g}$ ，定量限分别为 $0.03\mathrm{g} / 100\mathrm{g},0.05\mathrm{g} / 100\mathrm{g},0.07\mathrm{g} / 100\mathrm{g}$ 和 $0.12\mathrm{g} / 100\mathrm{g}$ 。

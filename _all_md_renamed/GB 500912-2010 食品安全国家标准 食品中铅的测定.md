# 第一法 石墨炉原子吸收光谱法

# 3 原理

试样经灰化或酸消解后，注入原子吸收分光光度计石墨炉中，电热原子化后吸收 $283.3\mathrm{nm}$ 共振线，在一定浓度范围，其吸收值与铅含量成正比，与标准系列比较定量。

# 4 试剂和材料

除非另有规定，本方法所使用试剂均为分析纯，水为GB/T6682规定的一级水。

4.1 硝酸：优级纯。  
4.2 过硫酸铵。  
4.3 过氧化氢（ $30\%$ ）  
4.4 高氯酸：优级纯。  
4.5 硝酸（1+1）：取 $50~\mathrm{mL}$ 硝酸慢慢加入 $50~\mathrm{mL}$ 水中。  
4.6 硝酸（ $0.5\mathrm{mol} / \mathrm{L}$ ）：取 $3.2\mathrm{mL}$ 硝酸加入 $50\mathrm{mL}$ 水中，稀释至 $100\mathrm{mL}$ 。  
4.7 硝酸（1mol/L）：取 $6.4\mathrm{mL}$ 硝酸加入 $50~\mathrm{mL}$ 水中，稀释至 $100~\mathrm{mL}$ 。  
4.8 磷酸二氢铵溶液（ $20\mathrm{g / L}$ ）：称取 $2.0\mathrm{g}$ 磷酸二氢铵，以水溶解稀释至 $100\mathrm{mL}$ 。  
4.9 混合酸：硝酸十高氯酸（ $9+1$ ）。取9份硝酸与1份高氯酸混合。  
4.10 铅标准储备液：准确称取 $1.000 \mathrm{~g}$ 金属铅（ $99.99\%$ ），分次加少量硝酸（4.5），加热溶解，总量不超过 $37 \mathrm{~mL}$ ，移入 $1000 \mathrm{~mL}$ 容量瓶，加水至刻度。混匀。此溶液每毫升含 $1.0 \mathrm{mg}$ 铅。  
4.11 铅标准使用液：每次吸取铅标准储备液 $1.0 \mathrm{~mL}$ 于 $100 \mathrm{~mL}$ 容量瓶中，加硝酸（4.6）至刻度。如此

经多次稀释成每毫升含 $10.0\mathrm{ng}$ ， $20.0\mathrm{ng}$ ， $40.0\mathrm{ng}$ ， $60.0\mathrm{ng}$ ， $80.0\mathrm{ng}$ 铅的标准使用液。

# 5 仪器和设备

5.1 原子吸收光谱仪，附石墨炉及铅空心阴极灯。  
5.2 马弗炉。  
5.3 天平：感量为 $1\mathrm{mg}$ 。  
5.4 干燥恒温箱。  
5.5 瓷坩埚。  
5.6 压力消解器、压力消解罐或压力溶弹。  
5.7 可调式电热板、可调式电炉。

# 6 分析步骤

# 6.1 试样预处理

6.1.1 在采样和制备过程中，应注意不使试样污染。  
6.1.2 粮食、豆类去杂物后，磨碎，过20目筛，储于塑料瓶中，保存备用。  
6.1.3 蔬菜、水果、鱼类、肉类及蛋类等水分含量高的鲜样，用食品加工机或匀浆机打成匀浆，储于塑料瓶中，保存备用。  
6.2 试样消解（可根据实验室条件选用以下任何一种方法消解）

6.2.1 压力消解罐消解法：称取 $1 \mathrm{~g} \sim 2 \mathrm{~g}$ 试样（精确到 $0.001 \mathrm{~g}$ ，干样、含脂肪高的试样 $< 1 \mathrm{~g}$ ，鲜样 $< 2 \mathrm{~g}$ 或按压力消解罐使用说明书称取试样）于聚四氟乙烯内罐，加硝酸（4.1） $2 \mathrm{~mL} \sim 4 \mathrm{~mL}$ 浸泡过夜。再加过氧化氢（4.3） $2 \mathrm{~mL} \sim 3 \mathrm{~mL}$ （总量不能超过罐容积的 $1 / 3$ ）。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱， $120^{\circ} \mathrm{C} \sim 140^{\circ} \mathrm{C}$ 保持 $3 \mathrm{~h} \sim 4 \mathrm{~h}$ ，在箱内自然冷却至室温，用滴管将消化液洗入或过滤入（视消化后试样的盐分而定） $10 \mathrm{~mL} \sim 25 \mathrm{~mL}$ 容量瓶中，用水少量多次洗涤罐，洗液合并于容量瓶中并定容至刻度，混匀备用；同时作试剂空白。  
6.2.2 干法灰化：称取 $1 \mathrm{~g} \sim 5 \mathrm{~g}$ 试样（精确到 $0.001 \mathrm{~g}$ ，根据铅含量而定）于瓷坩埚中，先小火在可调式电热板上炭化至无烟，移入马弗炉 $500^{\circ} \mathrm{C} \pm 25^{\circ} \mathrm{C}$ 灰化 $6 \mathrm{~h} \sim 8 \mathrm{~h}$ ，冷却。若个别试样灰化不彻底，则加 $1 \mathrm{~mL}$ 混合酸（4.9）在可调式电炉上小火加热，反复多次直到消化完全，放冷，用硝酸（4.6）将灰分溶解，用滴管将试样消化液洗入或过滤入（视消化后试样的盐分而定） $10 \mathrm{~mL} \sim 25 \mathrm{~mL}$ 容量瓶中，用水少量多次洗涤瓷坩埚，洗液合并于容量瓶中并定容至刻度，混匀备用；同时作试剂空白。  
6.2.3 过硫酸铵灰化法：称取 $1\mathrm{g}\sim 5\mathrm{g}$ 试样（精确到 $0.001\mathrm{g}$ ）于瓷坩埚中，加 $2\mathrm{mL}\sim 4\mathrm{mL}$ 硝酸（4.1）浸泡 $1\mathrm{h}$ 以上，先小火炭化，冷却后加 $2.00\mathrm{g}\sim 3.00\mathrm{g}$ 过硫酸铵（4.2）盖于上面，继续炭化至不冒烟，转入马弗炉， $500^{\circ}\mathrm{C}\pm 25^{\circ}\mathrm{C}$ 恒温 $2\mathrm{h}$ ，再升至 $800^{\circ}\mathrm{C}$ ，保持 $20\mathrm{min}$ ，冷却，加 $2\mathrm{mL}\sim 3\mathrm{mL}$ 硝酸（4.7），用滴管将试样消化液洗入或过滤入（视消化后试样的盐分而定） $10\mathrm{mL}\sim 25\mathrm{mL}$ 容量瓶中，用水少量多次洗涤瓷坩埚，洗液合并于容量瓶中并定容至刻度，混匀备用；同时作试剂空白。  
6.2.4 湿式消解法：称取试样 $1 \mathrm{~g} \sim 5 \mathrm{~g}$ （精确到 $0.001 \mathrm{~g}$ ）于锥形瓶或高脚烧杯中，放数粒玻璃珠，加 $10 \mathrm{~mL}$ 混合酸（4.9），加盖浸泡过夜，加一小漏斗于电炉上消解，若变棕黑色，再加混合酸，直至冒白烟，消化液呈无色透明或略带黄色，放冷，用滴管将试样消化液洗入或过滤入（视消化后试样的盐分而定） $10 \mathrm{~mL} \sim 25 \mathrm{~mL}$ 容量瓶中，用水少量多次洗涤锥形瓶或高脚烧杯，洗液合并于容量瓶中并定容至

刻度，混匀备用；同时作试剂空白。

# 6.3 测定

6.3.1 仪器条件：根据各自仪器性能调至最佳状态。参考条件为波长 $283.3\mathrm{nm}$ ，狭缝 $0.2\mathrm{nm} \sim 1.0\mathrm{nm}$ ，灯电流 $5\mathrm{mA} \sim 7\mathrm{mA}$ ，干燥温度 $120^{\circ}\mathrm{C}$ ， $20\mathrm{s}$ ；灰化温度 $450^{\circ}\mathrm{C}$ ，持续 $15\mathrm{s} \sim 20\mathrm{s}$ ，原子化温度： $1700^{\circ}\mathrm{C} \sim 2300^{\circ}\mathrm{C}$ ，持续 $4\mathrm{s} \sim 5\mathrm{s}$ ，背景校正为氘灯或塞曼效应。  
6.3.2 标准曲线绘制：吸取上面配制的铅标准使用液 $10.0\mathrm{ng / mL}$ （或 $\mu \mathrm{g} / \mathrm{L}$ ）， $20.0\mathrm{ng / mL}$ （或 $\mu \mathrm{g} / \mathrm{L}$ ）， $40.0\mathrm{ng / mL}$ （或 $\mu \mathrm{g} / \mathrm{L}$ ）， $60.0\mathrm{ng / mL}$ （或 $\mu \mathrm{g} / \mathrm{L}$ ）， $80.0\mathrm{ng / mL}$ （或 $\mu \mathrm{g} / \mathrm{L}$ ）各 $10~\mu \mathrm{L}$ ，注入石墨炉，测得其吸光值并求得吸光值与浓度关系的一元线性回归方程。  
6.3.3 试样测定：分别吸取样液和试剂空白液各 $10~\mu \mathrm{L}$ ，注入石墨炉，测得其吸光值，代入标准系列的一元线性回归方程中求得样液中铅含量。  
6.3.4 基体改进剂的使用：对有干扰试样，则注入适量的基体改进剂磷酸二氢铵溶液（4.8）（一般为 $5\mu \mathrm{L}$ 或与试样同量）消除干扰。绘制铅标准曲线时也要加入与试样测定时等量的基体改进剂磷酸二氢铵溶液。

# 7 分析结果的表述

试样中铅含量按式（1）进行计算。

$$
X = \frac {\left(c _ {1} - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0}. \tag {1}
$$

式中：

$X$ ——试样中铅含量，单位为毫克每千克或毫克每升（ $\mathrm{mg/kg}$ 或 $\mathrm{mg/L}$ ）；

$c_{l}$ ——测定样液中铅含量，单位为纳克每毫升（ $\mathrm{ng/mL}$ ）；

$c_{0}$ ——空白液中铅含量，单位为纳克每毫升（ $\mathrm{ng/mL}$ ）；

$V$ ——试样消化液定量总体积，单位为毫升（ $\mathrm{mL}$ ）；

$m$ ——试样质量或体积, 单位为克或毫升 (g 或 mL)。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 8 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 第二法 氢化物原子荧光光谱法

# 9 原理

试样经酸热消化后，在酸性介质中，试样中的铅与硼氢化钠 $\mathrm{(NaBH_4)}$ 或硼氢化钾 $\mathrm{(KBH_4)}$ 反应生成挥发性铅的氢化物 $\mathrm{(PbH_4)}$ 。以氩气为载气，将氢化物导入电热石英原子化器中原子化，在特制铅空心阴极灯照射下，基态铅原子被激发至高能态；在去活化回到基态时，发射出特征波长的荧光，其荧光强度与铅含量成正比，根据标准系列进行定量。

# 10 试剂和材料

10.1 硝酸+高氯酸混合酸 $(9 + 1)$ ：分别量取硝酸 $900~\mathrm{mL}$ ，高氯酸 $100~\mathrm{mL}$ ，混匀。

10.2 盐酸 $(1 + 1)$ ：量取 $250~\mathrm{mL}$ 盐酸倒入 $250~\mathrm{mL}$ 水中，混匀。  
10.3 草酸溶液 $(10\mathrm{g / L})$ ：称取 $1.0\mathrm{g}$ 草酸，加入溶解至 $100\mathrm{mL}$ ，混匀。  
10.4 铁氰化钾 $\left[\mathrm{K}_{3} \mathrm{Fe}(\mathrm{CN})_{6}\right]$ 溶液 $(100 \mathrm{~g} / \mathrm{L})$ ：称取 $10.0 \mathrm{~g}$ 铁氰化钾，加水溶解并稀释至 $100 \mathrm{~mL}$ ，混匀。  
10.5 氢氧化钠溶液 $(2\mathrm{g} / \mathrm{L})$ ：称取 $2.0\mathrm{g}$ 氢氧化钠，溶于 $1\mathrm{L}$ 水中，混匀。  
10.6 硼氢化钠 $\left(\mathrm{NaBH}_{4}\right)$ 溶液 $(10 \mathrm{~g} / \mathrm{L})$ : 称取 $5.0 \mathrm{~g}$ 硼氢化钠溶于 $500 \mathrm{~mL}$ 氢氧化钠溶液 $(2 \mathrm{~g} / \mathrm{L})$ 中, 混匀, 临用前配制。  
10.7 铅标准储备液 $(1.0\mathrm{mg / mL})$ 。  
10.8 铅标准使用液 $(1.0\mu \mathrm{g / mL})$ ：精确吸取铅标准储备液(10.7)，逐级稀释至 $1.0\mu \mathrm{g / mL}$ 。

# 11 仪器和设备

11.1 原子荧光光度计。  
11.2 铅空心阴极灯。  
11.3 电热板。  
11.4天平：感量为 $1\mathrm{mg}$

# 12 分析步骤

# 12.1 试样消化

湿消解：称取固体试样 $0.2\mathrm{g}\sim 2\mathrm{g}$ 或液体试样 $2.00\mathrm{g}$ （或 $\mathrm{mL}$ ）～ $10.00\mathrm{g}$ （或 $\mathrm{mL}$ ）（均精确到 $0.001\mathrm{g}$ ），置于 $50~\mathrm{mL}\sim 100~\mathrm{mL}$ 消化容器中（锥形瓶），然后加入硝酸+高氯酸混合酸（10.1） $5\mathrm{mL}\sim 10\mathrm{mL}$ 摇匀浸泡，放置过夜。次日置于电热板上加热消解，至消化液呈淡黄色或无色(如消解过程色泽较深，稍冷补加少量硝酸，继续消解)，稍冷加入 $20~\mathrm{mL}$ 水再继续加热赶酸，至消解液 $0.5\mathrm{mL}\sim 1.0\mathrm{mL}$ 止，冷却后用少量水转入 $25~\mathrm{mL}$ 容量瓶中，并加入盐酸（10.2） $0.5\mathrm{mL}$ ，草酸溶液（10.3） $0.5\mathrm{mL}$ ，摇匀，再加入铁氰化钾溶液（10.4） $1.00~\mathrm{mL}$ ，用水准确稀释定容至 $25~\mathrm{mL}$ ，摇匀，放置 $30\mathrm{min}$ 后测定。同时做试剂空白。

# 12.2 标准系列制备

在 $25\mathrm{mL}$ 容量瓶中，依次准确加入铅标准使用液（10.8） $0.00\mathrm{mL}$ 、 $0.125\mathrm{mL}$ 、 $0.25\mathrm{mL}$ 、 $0.50\mathrm{mL}$ 、 $0.75\mathrm{mL}$ 、 $1.00\mathrm{mL}$ 、 $1.25\mathrm{mL}$ （各相当于铅浓度 $0.0\mathrm{ng/mL}$ 、 $5.0\mathrm{ng/mL}$ 、 $10.0\mathrm{ng/mL}$ 、 $20.0\mathrm{ng/mL}$ 、 $30.0\mathrm{ng/mL}$ 、 $40.0\mathrm{ng/mL}$ 、 $50.0\mathrm{ng/mL}$ ），用少量水稀释后，加入 $0.5\mathrm{mL}$ 盐酸（10.2）和 $0.5\mathrm{mL}$ 草酸溶液（10.3）摇匀，再加入铁氰化钾溶液（10.4） $1.0\mathrm{mL}$ ，用水稀释至该度，摇匀。放置 $30\mathrm{min}$ 后待测。

# 12.3测定

# 12.3.1 仪器参考条件

负高压：323 V；铅空心阴极灯灯电流：75 mA；原子化器：炉温 750 ℃～800 ℃，炉高 8 mm；氩气流速：载气 800 mL/min；屏蔽气：1000 mL/min；加还原剂时间：7.0 s；读数时间：15.0 s；延迟时间：0.0 s；测量方式：标准曲线法；读数方式：峰面积；进样体积：2.0 mL。

# 12.3.2 测量方式

设定好仪器的最佳条件，逐步将炉温升至所需温度，稳定 $10\mathrm{min} \sim 20\mathrm{min}$ 后开始测量：连续用标准系列的零管进样，待读数稳定之后，转入标准系列的测量，绘制标准曲线，转入试样测量，分别测定试

样空白和试样消化液，试样测定结果按式（2）计算。

# 13 分析结果的表述

试样中铅含量按式（2）进行计算。

$$
X = \frac {\left(c _ {1} - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {2}
$$

试中：

$X$ ——试样中铅含量，单位为毫克每千克或毫克每升（ $\mathrm{mg/kg}$ 或 $\mathrm{mg/L}$ ）；

$c_{I}$ ——试样消化液测定浓度，单位为纳克每毫升（ $\mathrm{ng/mL}$ ）；

$c_{0}$ ——试剂空白液测定浓度，单位为纳克每毫升（ $\mathrm{ng/mL}$ ）；

$V$ ——试样消化液定量总体积，单位为毫升（ $\mathrm{mL}$ ）；

$m$ ——试样质量或体积，单位为克或毫升（g 或 mL）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第三法 火焰原子吸收光谱法

# 15 原理

试样经处理后，铅离子在一定 $\mathrm{pH}$ 条件下与二乙基二硫代氨基甲酸钠（DDTC）形成络合物，经4-甲基-2-戊酮萃取分离，导入原子吸收光谱仪中，火焰原子化后，吸收 $283.3\mathrm{nm}$ 共振线，其吸收量与铅含量成正比，与标准系列比较定量。

# 16 试剂和材料

16.1 混合酸：硝酸-高氯酸（9+1）。  
16.2 硫酸铵溶液（ $300\mathrm{g / L}$ ）：称取 $30\mathrm{g}$ 硫酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{SO}_{4}\right]$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。  
16.3 柠檬酸铵溶液（ $250\mathrm{g / L}$ ）：称取 $25\mathrm{g}$ 柠檬酸铵，用水溶解并稀释至 $100\mathrm{mL}$ 。  
16.4 溴百里酚蓝水溶液（ $1\mathrm{g/L}$ ）。  
16.5 二乙基二硫代氨基甲酸钠（DDTC）溶液（ $50\mathrm{g/L}$ ）：称取 $5\mathrm{g}$ 二乙基二硫代氨基甲酸钠，用水溶解并加水至 $100\mathrm{mL}$ 。  
16.6 氨水（1+1）。  
16.7 4-甲基-2-戊酮（MIBK）。  
16.8 铅标准溶液：操作同 10.7 和 10.8。配制铅标准使用液为 $10\mu \mathrm{g} / \mathrm{mL}$ 。  
16.9 盐酸（1+11）：取 $10\mathrm{mL}$ 盐酸加入 $110\mathrm{mL}$ 水中，混匀。

16.10 磷酸溶液（1+10）：取 $10\mathrm{mL}$ 磷酸加入 $100\mathrm{mL}$ 水中，混匀。

# 17 仪器和设备

17.1原子吸收光谱仪火焰原子化器，其余同5.2，5.3，5.4，5.5，5.6和5.7。  
17.2 天平：感量为 $1\mathrm{mg}$ 。

# 18 分析步骤

# 18.1 试样处理

18.1.1 饮品及酒类: 取均匀试样 $10 \mathrm{~g} \sim 20 \mathrm{~g}$ (精确到 $0.01 \mathrm{~g}$ ) 于烧杯中 (酒类应先在水浴上蒸去酒精),于电热板上先蒸发至一定体积后, 加入混合酸 (16.1) 消化完全后, 转移、定容于 $50 \mathrm{~mL}$ 容量瓶中。  
18.1.2 包装材料浸泡液可直接吸取测定。

# 第四法 二硫腙比色法

# 21 原理

试样经消化后，在 $\mathrm{pH}8.5\sim 9.0$ 时，铅离子与二硫腙生成红色络合物，溶于三氯甲烷。加入柠檬酸铵、氰化钾和盐酸羟胺等，防止铁、铜、锌等离子干扰，与标准系列比较定量。

22.10 二硫腙-三氯甲烷溶液（ $0.5\mathrm{g/L}$ ）：保存冰箱中，必要时用下述方法纯化。

称取 $0.5 \mathrm{~g}$ 研细的二硫腙, 溶于 $50 \mathrm{~mL}$ 三氯甲烷中, 如不全溶, 可用滤纸过滤于 $250 \mathrm{~mL}$ 分液漏斗中, 用氨水 $(1+99)$ 提取三次, 每次 $100 \mathrm{~mL}$ , 将提取液用棉花过滤至 $500 \mathrm{~mL}$ 分液漏斗中, 用盐酸 $(1+1)$ 调至酸性, 将沉淀出的二硫腙用三氯甲烷提取 2 次 $\sim 3$ 次, 每次 $20 \mathrm{~mL}$ , 合并三氯甲烷层, 用等量水洗涤两次, 弃去洗涤液, 在 $50^{\circ} \mathrm{C}$ 水浴上蒸去三氯甲烷。精制的二硫腙置硫酸干燥器中, 干燥备用。或将沉淀出的二硫腙用 $200 \mathrm{~mL}, 200 \mathrm{~mL}, 100 \mathrm{~mL}$ 三氯甲烷提取三次, 合并三氯甲烷层为二硫腙溶液。

$$

# 第五法 单扫描极谱法

# 27 原理

试样经消解后，铅以离子形式存在。在酸性介质中， $\mathrm{Pb}^{2+}$ 与 $\mathrm{I}^{-}$ 形成的 $\mathrm{PbI}_4^{2-}$ 络离子具有电活性，在滴汞电极上产生还原电流。峰电流与铅含量呈线性关系，以标准系列比较定量。

# 28 试剂和材料

28.1 底液：称取 $5.0 \mathrm{~g}$ 碘化钾， $8.0 \mathrm{~g}$ 酒石酸钾钠， $0.5 \mathrm{~g}$ 抗坏血酸于 $500 \mathrm{~mL}$ 烧杯中，加入 $300 \mathrm{~mL}$ 水溶解后，再加入 $10 \mathrm{~mL}$ 盐酸，移入 $500 \mathrm{~mL}$ 容量瓶中，加水至刻度。（在冰箱中可保存 2 个月）  
28.2铅标准贮备溶液（ $1.0\mathrm{mg / mL}$ ）：准确称取 $0.1000\mathrm{g}$ 金属铅（含量 $99.99\%$ ）于烧杯中加 $2\mathrm{mL}(1 + 1)$ 硝酸溶液，加热溶解，冷却后定量移入 $100~\mathrm{mL}$ 容量瓶并加水至刻度，混匀。  
28.3 铅标准使用溶液（ $10.0\mu \mathrm{g} / \mathrm{mL}$ ）：临用时，吸取铅标准贮备溶液 $1.00\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中，加水至刻度，混匀。  
28.4 混合酸：硝酸-高氯酸（4+1），量取 $80 \mathrm{~mL}$ 硝酸，加入 $20 \mathrm{~mL}$ 高氯酸，混匀。

# 29 仪器和设备

29.1 极谱分析仪。  
29.2 带电子调节器万用电炉。  
29.3天平：感量为 $1\mathrm{mg}$

# 30 分析步骤

# 30.1 极谱分析参考条件

单扫描极谱法(SSP法)。选择起始电位为-350mV，终止电位-850mV，扫描速度300mV/s，三电极，二次导数，静止时间5s及适当量程。于峰电位（Ep）-470mV处，记录铅的峰电流。

# 30.2 标准曲线绘制

准确吸取铅标准使用溶液 $0\mathrm{mL}$ ， $0.05\mathrm{mL}$ ， $0.10\mathrm{mL}$ ， $0.20\mathrm{mL}$ ， $0.30\mathrm{mL}$ ， $0.40\mathrm{mL}$ (相当于含 $0\mu \mathrm{g}$ ， $0.5\mu \mathrm{g}$ ， $1.0\mu \mathrm{g}$ ， $2.0\mu \mathrm{g}$ ， $3.0\mu \mathrm{g}$ ， $4.0\mu \mathrm{g}$ 铅)于 $10\mathrm{mL}$ 比色管中，加底液至 $10.0\mathrm{mL}$ ，混匀。将各管溶液依次移入电解池，置于三电极系统。按上述极谱分析参考条件测定，分别记录铅的峰电流。以含量为横坐标，其对应的峰电流为纵坐标，绘制标准曲线。

# 30.3 试样处理

粮食、豆类等水分含量低的试样，去杂物后磨碎过20目筛；蔬菜、水果、鱼类、肉类等水分含量高的新鲜试样，用均浆机均浆，储于塑料瓶。

30.3.1 试样处理（除食盐、白糖外，如粮食、豆类、糕点、茶叶、肉类等）：称取 $1\mathrm{g} \sim 2\mathrm{g}$ 试样（精确至 $0.1\mathrm{g}$ ）于 $50\mathrm{mL}$ 三角瓶中,加入 $10\mathrm{mL} \sim 20\mathrm{mL}$ 混合酸，加盖浸泡过夜。置带电子调节器万用电炉上的低档位加热。若消解液颜色逐渐加深，呈现棕黑色时，移开万用电炉，冷却，补加适量硝酸，继续加热消解。待溶液颜色不再加深，呈无色透明或略带黄色，并冒白烟，可高档位驱赶剩余酸液，至近干，在低档位加热得白色残渣，待测。同时作一试剂空白。

30.3.2 食盐、白糖：称取试样 $2.0\mathrm{g}$ 于烧杯中，待测。

# 30.3.3 液体试样

称取 $2 \mathrm{~g}$ 试样（精确至 $0.1 \mathrm{~g}$ ）于 $50 \mathrm{~mL}$ 三角瓶中（含乙醇、二氧化碳的试样应置于 $80^{\circ} \mathrm{C}$ 水浴上驱赶）。加入 $1 \mathrm{~mL} \sim 10 \mathrm{~mL}$ 混合酸，于带电子调节器万用电炉上的低档位加热，以下步骤按30.3.1“试样处理”项下操作，待测。

# 30.4 试样测定

于上述待测试样及试剂空白瓶中加入 $10.0 \mathrm{~mL}$ 底液, 溶解残渣并移入电解池。以下按 30.2 “标准曲线绘制”项下操作, 极谱图参见附录 A。分别记录试样及试剂空白的峰电流, 用标准曲线法计算试样中铅含量。

# 31 分析结果的表述

试样中铅含量按式（6）进行计算。

$$
X = \frac {\left(A - A _ {0}\right) \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {6}
$$

式中：

$X$ ——试样中铅的含量，单位为毫克每千克或毫克每升（ $\mathrm{mg/kg}$ 或 $\mathrm{mg/L}$ ）；

$A$ ——由标准曲线上查得测定样液中铅的质量, 单位为微克 $(\mu \mathrm{g})$ ;

$A_{0}$ ——由标准曲线上查得试剂空白液中铅质量, 单位为微克 $(\mu \mathrm{g})$ ;

m——试样质量或体积，单位为克或毫升（g 或 mL）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 32 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5.0\%$ 。

# 33 其他

本标准检出限：石墨炉原子吸收光谱法为 $0.005\mathrm{mg / kg}$ ；氢化物原子荧光光谱法固体试样为0.005 $\mathrm{mg / kg}$ ，液体试样为 $0.001\mathrm{mg / kg}$ ；火焰原子吸收光谱法为 $0.1\mathrm{mg / kg}$ ；比色法为 $0.25\mathrm{mg / kg}$ 。单扫描极谱法为 $0.085\mathrm{mg / kg}$ 。

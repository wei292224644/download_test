# 食品中滑石粉的测定

2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替GB/T21913—2008《食品中滑石粉的测定》。

本标准与GB/T21913—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中滑石粉的测定”；

——增加了微波消化方法进行试样处理的内容。

# 食品安全国家标准

# 食品中滑石粉的测定

# 1 范围

本标准规定了食品中滑石粉测定的原子吸收光谱方法。

本标准适用于食品中滑石粉的测定。

# 2 原理

滑石粉主要成分是天然的水合硅酸镁 $(3\mathrm{MgO}\cdot 4\mathrm{SiO}_2\cdot \mathrm{H}_2\mathrm{O})$ 。用硝酸 $+$ 高氯酸(或硝酸 $+$ 过氧化氢)消化试样，过滤，将滑石粉与其他含镁物质分离。滑石粉与氢氟酸反应生成溶于水的镁盐，原子吸收光谱法测定镁含量，根据镁含量计算滑石粉含量。

# 3 试剂和材料

警告——氢氟酸剧毒，应注意安全防护，避免接触皮肤或吸入。

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的二级水。

# 3.1 试剂

3.1.1 硝酸 $\mathrm{(HNO_3)}$ 。  
3.1.2 盐酸（HCl）。  
3.1.3 高氯酸 $\mathrm{(HClO_4)}$   
3.1.4 过氧化氢 $\mathrm{(H_2O_2)}$ 。  
3.1.5 氢氟酸（HF）。  
3.1.6 氯化锶 $(\mathrm{SrCl}_2)$ ：优级纯。

# 3.2 试剂配制

3.2.1 混合酸：硝酸 $+$ 高氯酸 $(4 + 1)$   
3.2.2 氯化锶溶液 $(15\mathrm{g} / \mathrm{L})$ ：称取氯化锶 $15\mathrm{g}$ ，加入 $100~\mathrm{mL}$ 水和 $45~\mathrm{mL}$ 盐酸溶解，用水稀释定容至 $1000~\mathrm{mL}$ 。

# 3.3 标准品

氧化镁 $(\mathrm{MgO},\mathrm{CAS}$ 号：1309-48-4）：纯度 $\geq 99.9\%$ ，或镁元素溶液标准物质。

# 3.4 标准溶液配制

# 3.4.1 镁元素标准贮备液

将氧化镁在 $750^{\circ} \mathrm{C}$ 条件下高温加热 $2 \mathrm{~h}$ , 冷却。称取氧化镁 $1.658 \mathrm{~g}$ 于烧杯中, 加入 $50 \mathrm{~mL}$ 水和 $45 \mathrm{~mL}$ 盐酸溶解, 用水定容至 $1000 \mathrm{~mL}$ , 混匀, 其中镁元素浓度为 $1000 \mathrm{mg} / \mathrm{L}$ 。贮备液于 $4^{\circ} \mathrm{C}$ 条件下

保存，保存期1年。

# 3.4.2 镁元素标准中间液

准确吸取镁元素标准贮备液或镁元素溶液标准物质，用氯化锶溶液稀释，配制成 $100\mathrm{mg / L}$ 的镁元素标准中间液，于 $4^{\circ}C$ 条件下保存，保存期3个月。

# 3.4.3 镁元素标准工作液

分别准确吸取 $0\mathrm{mL},0.5\mathrm{mL},1\mathrm{mL},2\mathrm{mL},4\mathrm{mL},5\mathrm{mL}$ 镁元素标准中间液于 $100~\mathrm{mL}$ 容量瓶中，用氯化锶溶液定容，配制成浓度为 $0\mathrm{mg / L},0.5\mathrm{mg / L},1\mathrm{mg / L},2\mathrm{mg / L},4\mathrm{mg / L},5\mathrm{mg / L}$ 的镁元素标准工作液。

# 4 仪器和设备

4.1 火焰原子吸收分光光度计（配镁空心阴极灯或无极放电灯）。  
4.2 天平：感量为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。  
4.3 微波消解仪（配聚四氟乙烯内罐）。  
4.4 聚四氟乙烯坩埚： $250~\mathrm{mL}$ ，带盖。  
4.5 可调温加热板。  
4.6 粉碎机。  
4.7 研钵。

# 5 分析步骤

# 5.1 试样制备

5.1.1 挂面、腐竹等于试样，用粉碎机粉碎，混匀，装入洁净容器，密封。  
5.1.2 湿面等湿试样，烘干后用粉碎机粉碎，混匀，装入洁净容器，密封。  
5.1.3 瓜子、干果等带壳试样，用粉碎机带壳粉碎，混匀，备用。  
5.1.4 凉果、糖果、胶姆糖等高含糖、柔软、黏着试样，用冰箱冷冻后研钵研碎，混匀，装入洁净容器，密封。  
5.1.5 梅子等带核试样，去核后粉碎，混匀，装入洁净容器，密封。

# 5.2 试样消解处理

# 5.2.1 湿消解法

称取试样 $0.5\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ，放入聚四氟乙烯增蜗中，加入 $15~\mathrm{mL}$ 混合酸，盖上盖，电热板上加热消化 $1\mathrm{h}\sim 2\mathrm{h}$ ，至消化液无色透明，如果酸液过少时，可适当补加混合酸后继续加热消化。将增蜗取下，冷却，用 $20~\mathrm{mL}$ 水冲洗增蜗，将增蜗中的固体残渣完全转移到定量滤纸上，过滤，用 $100~\mathrm{mL}$ 水冲洗滤纸及滤渣。将滤纸及固体残渣放入聚四氟乙烯增蜗中，加入 $10~\mathrm{mL}$ 混合酸和 $3\mathrm{mL}$ 氢氟酸，盖上盖，电热板上加热消化 $1\mathrm{h}\sim 2\mathrm{h}$ ，至消化液无色透明，如果酸液过少时，适当补加混合酸后继续加热消化。待增蜗中的液体接近干时，取下冷却，用氯化锶溶液冲洗增蜗，将消化液转移至 $100~\mathrm{mL}$ 容量瓶中，用氯化锶溶液定容，混匀，该溶液为试样溶液，备用。滑石粉含量高的试样溶液应适当稀释后上机测定，滑石粉含量 $10\mathrm{g / kg}$ 左右的试样稀释10倍。

# 5.2.2 微波消解法

称取试样 $0.5\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ), 放入消解罐中, 加入 $7\mathrm{mL}$ 硝酸和 $1\mathrm{mL}$ 过氧化氢, 浸泡过夜或进行预消解, 用微波消解仪进行微波消解, 微波消解参考条件见附录 A。冷却后打开消解罐, 用 $20\mathrm{mL}$ 水冲洗消解罐, 将罐内液体和固体残渣完全转移到定量滤纸上, 过滤, 用 $100\mathrm{mL}$ 水冲洗滤纸及滤渣。将滤纸及残渣放入消解罐, 加热除去滤纸中的水分。消解罐中加入 $7\mathrm{mL}$ 硝酸、 $1\mathrm{mL}$ 过氧化氢和 $3\mathrm{mL}$ 氢氟酸, 用微波消解仪进行微波消解。冷却后打开消解罐, 加热赶酸, 待消解罐中的液体接近干时, 取下冷却, 用氯化锶溶液冲洗消解罐, 将消化液转移至 $100\mathrm{mL}$ 容量瓶中, 用氯化锶溶液定容, 混匀, 该溶液为试样溶液, 备用。滑石粉含量高的试样溶液应适当稀释后上机测定, 滑石粉含量 $10\mathrm{g/kg}$ 左右的试样稀释10倍。

# 5.3 空白试验

试样空白试验与试样消解处理同步进行。用待测试样做空白试验，按照与试样消解处理一样的步骤进行消解处理，只是将滤纸及纸上固体残渣进行消解时不加入氢氟酸，试样中的滑石粉不发生化学反应。试样空白试验的取样量、定容体积和稀释倍数与试样消解处理相同。

# 5.4 仪器参考条件

a）波长： $285.2\mathrm{nm}$   
b）光谱带宽： $0.5\mathrm{nm}$   
c）灯电流： $4.0\mathrm{mA}$   
d）空气流量： $13.5\mathrm{L / min}$   
e) 乙炔流量： $2.0\mathrm{L / min}$   
f) 燃烧器高度： $7.0\mathrm{mm}$

# 5.5 标准曲线的制作

仪器预热稳定后，调节好仪器，用氯化锶溶液调零，将镁元素标准工作液分别注入原子吸收光谱仪进行测定，以浓度为横坐标、吸光度值为纵坐标，制作镁元素标准曲线。

# 5.6 试样溶液的测定

将制备的空白溶液和试样溶液分别注入原子吸收光谱仪测定镁元素吸光度值，根据镁元素标准曲线得到试样溶液或空白溶液中镁元素的浓度。

# 6 分析结果的表述

试样中滑石粉的含量按式(1)计算：

$$
X = \frac {\left(\rho_ {1} - \rho_ {0}\right) \times V \times f \times 5 . 2 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中滑石粉含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho_{1}$ ——试样溶液中镁元素的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——空白溶液中镁元素的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样溶液稀释倍数；

5.20 ——镁元素换算为滑石粉（相对分子质量379)的系数；

$m$ ——试样质量，单位为克(g)；

1000——计量单位换算系数。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当取样量为 $0.5\mathrm{g}$ ，定容体积为 $100~\mathrm{mL}$ 时，检出限为 $2\mathrm{g / kg}$ ，定量限为 $6\mathrm{g / kg}$ 。

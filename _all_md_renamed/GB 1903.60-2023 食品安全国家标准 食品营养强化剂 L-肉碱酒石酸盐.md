# 前言

本标准代替GB25550—2010《食品安全国家标准 食品添加剂 L-肉碱酒石酸盐》。

本标准的附录A和附录B为规范性附录。

本标准与GB25550—2010相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品营养强化剂 L-肉碱酒石酸盐”；  
——修改了L-肉碱的鉴别试验；  
——增加了酒石酸的鉴别试验；  
——修改了L-肉碱和酒石酸的检测方法，增加了电位滴定方法；  
——删除了重金属指标，增加了铅指标及其检验方法；  
——修改了总碑的检测方法。

# 检验方法

警告：本试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和防护措施。

# A.1 一般规定

本标准所用试剂和水在未注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂和制品在未注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 L-肉碱的鉴别试验（化学法）

# A.2.1.1 试剂和材料

A.2.1.1.1 盐酸（HCl）。  
A.2.1.1.2 氢氧化钾（KOH）。  
A.2.1.1.3 高锰酸钾 $\mathrm{(KMnO_4)}$ 。  
A.2.1.1.4 硫氰酸铬铵 $\mathrm{(C_4H_{10}CrN_7S_4\bullet H_2O)}$   
A.2.1.1.5 石蕊试纸。  
A.2.1.1.6 盐酸溶液： $1\mathrm{mol} / \mathrm{L}$ 。取 $8.3~\mathrm{mL}$ 盐酸，用水稀释至 $100~\mathrm{mL}$   
A.2.1.1.7 硫氰酸铬铵试液： $5.0\mathrm{g / L}$ 。取 $0.5\mathrm{g}$ 硫氰酸铬铵，用水溶解至 $100~\mathrm{mL}$

# A.2.1.2 分析步骤

A.2.1.2.1 称取试样约 $0.1 \mathrm{~g}$ , 加水 $4 \mathrm{~mL}$ , 振摇使溶解, 加 $1 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液 $1 \mathrm{~mL}$ , 摇匀, 加硫氰酸铬铵试液 2 滴, 应生成紫红色沉淀。  
A.2.1.2.2 称取试样约 $0.5\mathrm{g}$ ，加 $5\mathrm{mL}$ 水溶解，加氢氧化钾 $2\mathrm{g}$ 、高锰酸钾数粒，加热时放出氨（能使湿润的红色石蕊试纸变蓝）。

# A.2.2 酒石酸盐的鉴别试验（化学法）

# A.2.2.1 试剂和材料

A.2.2.1.1 硫酸 $\mathrm{(H_2SO_4)}$ 。  
A.2.2.1.2 盐酸（HCl）。  
A.2.2.1.3 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
A.2.2.1.4 过氧化氢 $\mathrm{(H_2O_2)}$   
A.2.2.1.5 氢氧化钠(NaOH)。  
A.2.2.1.6 硫酸亚铁 $\mathrm{(FeSO_4)}$   
A.2.2.1.7 间苯二酚 $\mathrm{(C_6H_6O_2)}$ 。  
A.2.2.1.8 溴化钾(KBr)。  
A.2.2.1.9 盐酸溶液： $0.24\mathrm{mol / L}$ 。取 $2\mathrm{mL}$ 浓盐酸，用水稀释至 $100~\mathrm{mL}$   
A.2.2.1.10 乙酸溶液：取 $1\mathrm{mL}$ 冰乙酸，用水稀释至 $4\mathrm{mL}$   
A.2.2.1.11 过氧化氢溶液：取 $10\mathrm{mL}30\%$ 过氧化氢，用水稀释至 $100\mathrm{mL}$

A.2.2.1.12 氢氧化钠溶液： $40\mathrm{g / L}$ 。取 $4\mathrm{g}$ 氢氧化钠，用水溶解至 $100~\mathrm{mL}$   
A.2.2.1.13 硫酸亚铁溶液： $80~\mathrm{g / L}$ 。取 $8\mathrm{g}$ 硫酸亚铁，用水溶解至 $100~\mathrm{mL}$   
A.2.2.1.14 间苯二酚溶液： $20\mathrm{g / L}$ 。取 $2\mathrm{g}$ 间苯二酚，用水溶解至 $100\mathrm{mL}$ 。  
A.2.2.1.15 溴化钾溶液： $100\mathrm{g / L}$ 。取 $10\mathrm{g}$ 溴化钾，用水溶解至 $100~\mathrm{mL}$

# A.2.2.2 分析步骤

A.2.2.2.1 称取试样约 $5.0 \mathrm{~g}$ , 用少量水溶解, 用氢氧化钠溶液或者盐酸溶液调 $\mathrm{pH}$ 至中性, 加水定容到 $100 \mathrm{~mL}$ , 为试样溶液 A。  
A.2.2.2.2 加2滴乙酸溶液于试样溶液A中，再依次加1滴硫酸亚铁溶液、2滴～3滴过氧化氢及过量氢氧化钠溶液后，L-肉碱酒石酸盐溶液为无色澄清溶液。  
A.2.2.2.3 在预先加有 2 滴～3 滴间苯二酚及 2 滴～3 滴溴化钾溶液的 $5 \mathrm{~mL}$ 硫酸中加入 2 滴～3 滴试样溶液 A, 溶液颜色先缓慢变为橘黄色, 于水浴上加热 $5 \mathrm{~min} \sim 10 \mathrm{~min}$ 后, L-肉碱酒石酸盐溶液颜色变为蓝色。冷却后倒入 $100 \mathrm{~mL}$ 水中, L-肉碱酒石酸盐溶液为红色。

# A.2.3 L-肉碱酒石酸盐的鉴别（红外光谱法）

采用红外光谱溴化钾压片法，分别取试样和对照品 $1\mathrm{mg}\sim 2\mathrm{mg}$ 及溴化钾，溴化钾：样品 $= 100:1$ ，分别混合研磨均匀，放入压片机中压片。将压片放入红外光谱仪中得到红外谱图。其谱图应与附录B中图B.1一致。

# A.3 L-肉碱含量的测定

# A.3.1 结晶紫指示剂法（第一法）

# A.3.1.1 原理

试样以冰乙酸为溶剂，以结晶紫为指示剂，用高氯酸标准滴定溶液滴定，根据消耗高氯酸标准滴定溶液的体积计算L-肉碱的含量。

# A.3.1.2 试剂和材料

A.3.1.2.1 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
A.3.1.2.2 结晶紫 $(\mathrm{C}_{25}\mathrm{H}_{30}\mathrm{ClN}_3)$ 。  
A.3.1.2.3 高氯酸标准滴定溶液： $c(\mathrm{HClO}_4) = 0.1\mathrm{mol / L}$   
A.3.1.2.4 结晶紫指示液： $5\mathrm{g / L}$ 。取 $0.5\mathrm{g}$ 结晶紫，用冰乙酸定容至 $100~\mathrm{mL}$

# A.3.1.3 仪器和设备

电子天平：感量为 $0.0001\mathrm{g}$ 。

# A.3.1.4 分析步骤

A.3.1.4.1 称取干燥试样 $0.1 \mathrm{~g}$ , 精确至 $0.0001 \mathrm{~g}$ , 加冰乙酸 $20 \mathrm{~mL}$ 溶解, 加 1 滴结晶紫指示液, 用高氯酸标准滴定溶液滴定至溶液显纯蓝色。  
A.3.1.4.2 在测定的同时，按相同的步骤对不加试样而使用相同数量的试剂溶液做空白试验。

# A.3.2 电位滴定仪法（第二法）

# A.3.2.1 方法提要

试样以冰乙酸为溶剂，用高氯酸标准滴定溶液滴定，根据消耗高氯酸标准滴定溶液的体积计算L-肉

碱的含量。

# A.3.2.2 试剂和材料

A.3.2.2.1 冰乙酸 $\mathrm{(CH_3COOH)}$   
A.3.2.2.2 高氯酸标准滴定溶液： $c(\mathrm{HClO}_4) = 0.1\mathrm{mol / L}$

# A.3.2.3 仪器和设备

A.3.2.3.1 电位滴定仪：配置玻璃电极。  
A.3.2.3.2 电子天平：感量为 $0.0001\mathrm{g}$

# A.3.2.4 分析步骤

A.3.2.4.1 称取干燥试样 $0.1\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，加 $20\mathrm{mL}$ 冰乙酸溶解，用高氯酸标准滴定溶液进行电位滴定。  
A.3.2.4.2 在测定的同时，按相同的步骤对不加试样而使用相同数量的试剂溶液做空白试验。

# A.3.3 结果计算

L-肉碱 $\mathrm{(C_7H_{15}NO_3}$ ，以干基计)的质量分数 $w_{1}$ 按式(A.1)计算。

$$
w _ {1} = \frac {\left(V _ {1} - V _ {2}\right) \times c _ {1} \times M}{m \times 1000} \times 100 \% \tag{A.1}
$$

式中

$V_{1}$ ——试样消耗高氯酸标准滴定溶液(A.3.1.4.1或A.3.2.4.1)体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——空白消耗高氯酸标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——高氯酸标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——L-肉碱的摩尔质量的数值，单位为克每摩尔 $(\mathrm{g / mol})(M = 161.2)$ ；

$m$ ——干燥试样的质量数值，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的2次独立测定结果的绝对差值不大于算术平均值的 $0.5\%$ ，结果保留3位有效数字。

# A.4 酒石酸的测定

# A.4.1 酚酞指示剂法（第一法）

# A.4.1.1 原理

以酚酞为指示剂，用氢氧化钠标准滴定溶液滴定干燥试样的水溶液，根据氢氧化钠标准滴定溶液的用量计算酒石酸的含量。

# A.4.1.2 试剂和材料

A.4.1.2.1 乙醇 $\mathrm{(CH_3CH_2OH)}$   
A.4.1.2.2 酚酞 $\mathrm{(C_{20}H_{14}O_4)}$ 。  
A.4.1.2.3 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.1\mathrm{mol / L}$   
A.4.1.2.4 酚酞指示液： $10\mathrm{g / L}$ 。取 $1.0\mathrm{g}$ 酚酞，用乙醇定容至 $100~\mathrm{mL}$

# A.4.1.3 仪器和设备

电子天平：感量为 $0.0001\mathrm{g}$

# A.4.1.4 分析步骤

A.4.1.4.1 称取干燥试样 $0.3 \mathrm{~g}$ , 精确至 $0.0001 \mathrm{~g}$ , 置于 $250 \mathrm{~mL}$ 锥形瓶中, 加入新煮沸并冷却的水 $50 \mathrm{~mL}$ 使试样溶解, 加 2 滴酚酞指示液, 用氢氧化钠标准滴定溶液滴定至溶液由无色变为粉红色。

A.4.1.4.2 在测定的同时，按相同的步骤对不加试样而使用相同数量的试剂溶液做空白试验。

# A.4.2 电位滴定仪法（第二法）

# A.4.2.1 原理

用氢氧化钠标准滴定溶液滴定干燥试样的水溶液，根据氢氧化钠标准滴定溶液的用量计算酒石酸的含量。

# A.4.2.2 试剂和材料

A.4.2.2.1 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.1\mathrm{mol / L}$

# A.4.2.3 仪器和设备

A.4.2.3.1 电位滴定仪：配置玻璃电极。

A.4.2.3.2 电子天平：感量为 $0.0001\mathrm{g}$

# A.4.2.4 分析步骤

A.4.2.4.1 称取干燥试样 $0.3\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，置于 $250\mathrm{mL}$ 锥形瓶中，加入新煮沸并冷却的水 $50\mathrm{mL}$ 使试样溶解，用氢氧化钠标准滴定溶液进行电位滴定。

A.4.2.4.2 在测定的同时，按相同的步骤对不加试样而使用相同数量的试剂溶液做空白试验。

# A.4.3 结果计算

酒石酸 $\mathrm{(C_4H_6O_6}$ ，以干基计)的质量分数 $\omega_{2}$ 按式(A.2)计算。

$$
w _ {2} = \frac {\left(V _ {4} - V _ {3}\right) \times c _ {2} \times M}{m \times 1000} \times 100 \% \tag{A.2}
$$

式中：

$V_{4}$ ——试样消耗氢氧化钠标准滴定溶液(A.4.1.4.1或A.4.2.4.1)体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——空白消耗氢氧化钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c_{2}$ ——氢氧化钠滴定液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——酒石酸的摩尔质量的数值，单位为克每摩尔 $(\mathrm{g / mol})(M = 75.04)$ ；

$m$ —干燥试样质量的数值，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的2次独立测定结果的绝对差值不大于算术平均值的 $0.5\%$ ，结果保留3位有效数字。

# A.5 干燥减量的测定

# A.5.1 仪器和设备

A.5.1.1 电子天平：感量为 $0.0001\mathrm{g}$ 。

A.5.1.2 干燥箱：温度可保持在 $105\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}$ 。

A.5.1.3 干燥器：内盛适当干燥剂。

A.5.1.4 称量瓶：扁形带盖，容量加入试样后，试样厚度小于 $5\mathrm{mm}$ 。

# A.5.2 分析步骤

按GB/T6284进行。测定时，称取 $1\mathrm{g}\sim 2\mathrm{g}$ 实验室样品，精确至 $0.0001\mathrm{g}$ ，干燥温度为 $105~^\circ \mathrm{C}\pm$ $2^{\circ}C$ ，干燥 $5\mathrm{h}$ 至恒重。

取两次平行测定结果的算术平均值为报告结果。两次平行测定结果的绝对差值不大于这2个测定值的算术平均值的 $20\%$ ，结果保留至小数点后2位。

# A.6 灼烧残渣的测定

# A.6.1 仪器和设备

A.6.1.1 电子天平：感量为 $0.0001\mathrm{g}$ 。  
A.6.1.2 高温炉：温度可保持在 $750^{\circ}\mathrm{C} \pm 50^{\circ}\mathrm{C}$ 。  
A.6.1.3 坩埚或蒸发皿。

# A.6.2 试剂和材料

浓硫酸。

# A.6.3 分析步骤

按GB/T9741进行。测定时，称取 $2\mathrm{g}$ 实验室样品，精确至 $0.0001\mathrm{g}$ ，灼烧温度为 $(750\pm 50)$ ℃。

取两次平行测定结果的算术平均值为报告结果。2次平行测定结果的绝对差值不大于这2个测定值的算术平均值的 $20\%$ ，结果保留至小数点后2位。

# A.7 $\mathbf{pH}$ 的测定

按GB/T9724进行。测定时，称取干燥试样约 $10\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，加约 $20\mathrm{mL}$ 无二氧化碳的水溶解并稀释至 $100\mathrm{mL}$ 后用 $\mathrm{pH}$ 计进行测定。

取两次平行测定结果的算术平均值为报告结果。2次平行测定结果的绝对差值不大于 $10\%$ ，结果保留2位有效数字。

# A.8 比旋光度的测定

取适量干燥试样，精确至 $0.0001\mathrm{g}$ ，用水溶解并定量稀释制成每毫升中约含 $100\mathrm{mg}$ 的溶液，按GB/T613进行测定。比旋光度 $\alpha_{\mathrm{m}}(20^{\circ}\mathrm{C},D)$ 数值以“（°）·dm²·kg⁻¹”表示，按式(A.3)计算，结果保留3位有效数字。

$$
\alpha_ {\mathrm {m}} (2 0 ^ {\circ} \mathrm {C}, D) = \frac {\alpha}{l \rho_ {\alpha}} \tag {A.3}
$$

式中：

$\alpha$ 测得的旋光角，单位为度 $(^{\circ})$

$l$ ——旋光管的长度，单位为分米（dm）；

$\rho_{a}$ ——溶液中有效组分的质量浓度，单位为克每毫升 $(\mathrm{g / mL})$ 。

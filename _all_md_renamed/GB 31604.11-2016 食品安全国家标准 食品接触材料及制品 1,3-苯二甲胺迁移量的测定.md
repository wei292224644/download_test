# 1,3-苯二甲胺迁移量的测定

2016-10-19 发布

2017-04-19 实施

# 前言

本标准代替GB/T23296.25—2009《食品接触材料 高分子材料 食品模拟物中1,3-苯二甲胺的测定 高效液相色谱法》及SN/T2550—2010《食品接触材料 高分子材料 食品模拟物中1,3-苯二甲胺的测定 高效液相色谱法》。

本标准与GB/T23296.25—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 1,3-苯二甲胺迁移量的测定”。

# 食品安全国家标准

# 食品接触材料及制品 1,3-苯二甲胺迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中1,3-苯二甲胺迁移量的高效液相色谱测定方法。  
本标准适用于食品接触材料及制品中1,3-苯二甲胺迁移量的测定。

# 2 原理

食品接触材料进行迁移试验后，对水基、酸性、酒精类食品模拟物中1,3-苯二甲胺进行荧光胺衍生，然后通过高效液相色谱进行分离，采用荧光检测器检测，外标法定量；油性模拟物中1,3-苯二甲胺经 $4\%$ 乙酸溶液萃取后采用同样方法检测。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水且用氮气饱和。

# 3.1 试剂

3.1.1 水基、酸性、酒精类、油基食品模拟物：所用试剂依据GB31604.1的规定。  
3.1.2 甲醇 $\mathrm{(CH_4O, CAS}$ 号：67-56-1)：色谱纯。  
3.1.3 四氢呋喃 $\mathrm{(C_4H_8O}$ ,CAS号：109-99-9)：色谱纯。  
3.1.4 丙酮 $\mathrm{(C_3H_6O}$ ,CAS号：67-64-1）。  
3.1.5 庚烷 $\mathrm{(C_7H_{16}}$ ，CAS号：142-82-5）。  
3.1.6 乙酸 $\mathrm{(C_2H_4O_2}$ ，CAS号：64-19-7）。  
3.1.7 荧光胺 $\mathrm{(C_{17}H_{10}O_4}$ ，CAS号：38183-12-9)：纯度 $\geq 98\%$   
3.1.8 十水合四硼酸钠 $\mathrm{(Na_2B_4O_7\cdot 10H_2O}$ ,CAS号：1330-43-4）。  
3.1.9 氢氧化钠(NaOH，CAS号：1310-73-2)。  
3.1.10 氮气： $99.999\%$

# 3.2 试剂配制

3.2.1 水基、酸性、酒精类、油基食品模拟物：按GB5009.156操作。  
3.2.2 四氢呋喃溶液（体积分数为 $90\%$ ）：量取 $90~\mathrm{mL}$ 四氢呋喃于 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度。  
3.2.3 荧光胺溶液 $(2\mathrm{mg / mL})$ ：称取 $50~\mathrm{mg}$ （精确至 $0.0001\mathrm{g}$ )荧光胺，用丙酮溶解后，定容至 $25~\mathrm{mL}$ 该溶液在 $5^{\circ}C$ 下避光储存，有效期一周。  
3.2.4 氢氧化钠溶液 $(5\mathrm{mol} / \mathrm{L})$ ：称取 $20\mathrm{g}$ 氢氧化钠，用水溶解后，定容至 $100~\mathrm{mL}$ 。  
3.2.5硼酸缓冲溶液 $(0.15\mathrm{mol} / \mathrm{L},\mathrm{pH} = 9.2)$ ：称取 $14.3\mathrm{g}$ 十水四硼酸钠，用水溶解后，定容至 $250~\mathrm{mL}$ 。在使用溶液前，如果由于温度下降而导致沉淀则需要重新溶解。

3.2.6 乙酸溶液（质量浓度为 $4\%$ ）：称取乙酸 $20\mathrm{g}$ ，用水溶解后，定容至 $500\mathrm{mL}$ 。  
3.2.7硼酸缓冲溶液 $(0.02\mathrm{mol} / \mathrm{L},\mathrm{pH} = 9.2)$ ：吸取 $133~\mathrm{mL}$ 的 $0.15\mathrm{mol} / \mathrm{L}$ 硼酸缓冲溶液(3.2.5)，用水稀释定容至 $1\textrm{L}$ 。  
3.2.8 高效液相色谱流动相[硼酸缓冲液：水：甲醇(18:37:45)]：吸取 $180~\mathrm{mL}$ 的 $0.02\mathrm{mol}$ 硼酸缓冲溶液(3.2.7)、 $370~\mathrm{mL}$ 水、 $450~\mathrm{mL}$ 甲醇混合。

# 3.3 标准品

1,3-苯二甲胺标准品 $\mathrm{(C_8H_{12}N_2}$ ，CAS号：1477-55-0)：纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

# 3.4.1 1,3-苯二甲胺标准储备液

称取 $50\mathrm{mg}$ (精确至 $0.0001\mathrm{g})$ 1,3-苯二甲胺，用水溶解后，定容至 $50~\mathrm{mL}$ ，浓度为 $1000\mathrm{mg / L}$ 。该溶液在 $5^{\circ}C$ 下避光储存，有效期3个月。

# 3.4.2 用于水基(酸性、酒精类)食品模拟物的1,3-苯二甲胺标准工作溶液

用刻度吸量管吸取 $0.20\mathrm{mL}$ 的1，3-苯二甲胺标准储备液(3.4.1)于 $100~\mathrm{mL}$ 容量瓶中，用水基（酸性、酒精类)食品模拟物稀释至刻度，获得浓度为 $2.0\mathrm{mg / L}$ 的用于水基(酸性、酒精类)食品模拟物的1，3-苯二甲胺标准工作溶液。该溶液在 $5^{\circ}C$ 下避光储存，有效期为1个月。

# 3.4.3 用于油基模拟物的1,3-苯二甲胺标准工作溶液

用刻度吸量管吸取 $2.0\mathrm{mL}$ 的1,3-苯二甲胺标准储备液(3.4.1)于 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，得浓度为 $20.0\mathrm{mg / L}$ 的溶液。吸取 $10~\mathrm{mL}$ 该溶液于 $100~\mathrm{mL}$ 容量瓶中，用四氢呋喃溶液(3.2.2)稀释至刻度，得浓度为 $2.0\mathrm{mg / L}$ 的用于油基模拟物的1,3-苯二甲胺标准工作溶液。该溶液在 $5^{\circ}C$ 下避光储存，有效期为1个月。

# 3.4.4 水基（酒精类）食品模拟物的1，3-苯二甲胺标准工作溶液

分别用刻度吸量管吸取 $0.5\mathrm{mL},2.0\mathrm{mL},3.0\mathrm{mL},4.0\mathrm{mL},5.0\mathrm{mL}$ 的1，3-苯二甲胺标准工作溶液(3.4.2)于5个 $100~\mathrm{mL}$ 容量瓶中，用相应的食品模拟物（水基、酒精食品模拟物)定容至刻度，得到浓度分别为 $0.010\mathrm{mg / L},0.040\mathrm{mg / L},0.060\mathrm{mg / L},0.080\mathrm{mg / L},0.10\mathrm{mg / L}$ 标准工作溶液。用刻度吸量管分别吸取上述溶液 $2.0\mathrm{mL}$ 于5个 $5\mathrm{mL}$ 试管中，分别加入 $0.4\mathrm{mL}$ 硼酸缓冲溶液(3.2.5)，充分混匀。分别加入 $300~\mu \mathrm{L}$ 荧光胺溶液(3.2.3)衍生试剂，振荡 $1\mathrm{min}$ ，静置 $10\mathrm{min}$ 。衍生溶液过 $0.45~\mu \mathrm{m}$ 滤膜后供高效液相色谱进样。

# 3.4.5 酸性食品模拟物的1,3-苯二甲胺标准工作溶液

分别用刻度吸量管吸取 $0.5\mathrm{mL},2.0\mathrm{mL},3.0\mathrm{mL},4.0\mathrm{mL},5.0\mathrm{mL}$ 酸性食品模拟物中的1，3-苯二甲胺标准工作溶液(3.4.2)于5个 $100~\mathrm{mL}$ 容量瓶中，用酸性食品模拟物定容至刻度，得到浓度分别为 $0.010\mathrm{mg / L},0.040\mathrm{mg / L},0.060\mathrm{mg / L},0.080\mathrm{mg / L},0.10\mathrm{mg / L}$ 标准工作溶液。准确吸取上述溶液中各 $10.0~\mathrm{mL}$ 于5个 $25~\mathrm{mL}$ 烧杯中，在第一个烧杯中滴加 $5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液(3.2.4)，调节 $\mathsf{pH}$ 至9.2，计算所需氢氧化钠溶液的体积，准确至 $0.01~\mathrm{mL}$ 。在另外四个烧杯中，准确加入上述滴加体积的氢氧化钠溶液(3.2.4)，充分混匀(各溶液 $\mathsf{pH}$ 应在 $8.0\sim 9.9$ 范围内)。以下按3.4.4中“用刻度吸量管分别吸取上述溶液 $2.0~\mathrm{mL}$ …供高效液相色谱进样”操作。

# 3.4.6 油基食品模拟物标准工作溶液

称取 $20.0\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )油基模拟物，置于5个 $125~\mathrm{mL}$ 分液漏斗中，分别用刻度吸量管吸取加入 $0.2\mathrm{mL},0.4\mathrm{mL},0.6\mathrm{mL},0.8\mathrm{mL},1.0\mathrm{mL}$ 的1,3-苯二甲胺标准工作溶液(3.4.3)于该5个 $125~\mathrm{mL}$ 分液漏斗中。分别准确加入 $0.8\mathrm{mL},0.6\mathrm{mL},0.4\mathrm{mL},0.2\mathrm{mL}$ 和 $0\mathrm{mL}$ 的四氢呋喃溶液(3.2.2)和 $5.0~\mathrm{mL}$ 庚烷，充分混匀，再加入 $20.0~\mathrm{mL}$ 乙酸溶液(3.2.6)，振荡 $5\mathrm{min}$ ，静置 $15\mathrm{min}$ ，待两相完全分离后，收集水相萃取物，得到浓度分别为 $0.02\mathrm{mg / kg},0.04\mathrm{mg / kg},0.06\mathrm{mg / kg},0.08\mathrm{mg / kg},0.10\mathrm{mg / kg}$ 油基食品模拟物标准工作溶液。准确移取上述工作溶液中各 $10~\mathrm{mL}$ 于5个 $25~\mathrm{mL}$ 烧杯中，在第一个烧杯中滴加 $5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液(3.2.4)，调节 $\mathsf{pH}$ 至9.2，计算所需氢氧化钠溶液的体积，准确至 $0.01~\mathrm{mL}$ 。在另外四个烧杯中，准确加入上述滴加体积的氢氧化钠溶液(3.2.4)，充分混匀（各溶液 $\mathsf{pH}$ 应在 $8.0\sim 9.9$ 范围内）。以下按3.4.4中“用刻度吸量管分别吸取上述溶液 $2.0~\mathrm{mL}$ …供高效液相色谱进样”操作。

# 4 仪器和设备

4.1 高效液相色谱仪：配荧光检测器（FLD）。  
4.2 分析天平：感量为 $0.0001\mathrm{g}$   
4.3 涡旋混匀器。  
4.4 机械振荡器。  
4.5 $0.45\mu \mathrm{m}$ 微孔滤膜。  
4.6 恒温水浴。

# 5 分析步骤

# 5.1 试样迁移试验

按照GB5009.156及GB31604.1的要求，对样品进行迁移试验，得到食品模拟物试液。如果得到的食品模拟物试液不能马上进行下一步试验，应将食品模拟物试液于 $4^{\circ}\mathrm{C}$ 冰箱中避光保存。

所得食品模拟物试液应冷却或恢复至室温后进行下一步试验。

# 5.2 试液制备

# 5.2.1 水基和酒精类食品模拟物的制备

用刻度吸量管吸取从迁移试验中得到的水基(酒精类)食品模拟物 $2.0\mathrm{mL}$ 于 $5\mathrm{mL}$ 试管中，加入 $0.4\mathrm{mL}$ 硼酸缓冲液，充分混匀。再加入 $300\mu \mathrm{L}$ 荧光胺溶液衍生试剂，振荡 $1\mathrm{min}$ ，静置 $10\mathrm{min}$ 。衍生溶液过 $0.45\mu \mathrm{m}$ 滤膜后供高效液相色谱进样。

# 5.2.2 酸性食品模拟物的制备

用刻度吸量管吸取从迁移试验中获得的 $10.0\mathrm{mL}$ 酸性食品模拟物于 $25~\mathrm{mL}$ 烧杯中，滴加入 $5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液(3.2.4)，体积同3.4.5中氢氧化钠溶液体积，充分混匀（溶液 $\mathrm{pH}$ 应在 $8.0\sim 9.9$ 范围内）。用刻度吸量管吸取上述溶液 $2.0\mathrm{mL}$ 于 $5\mathrm{mL}$ 试管中，以下按5.2.1中“加入 $0.4\mathrm{mL}$ 硼酸缓冲液供高效液相色谱进样”操作。

# 5.2.3 油基食品模拟物的制备

称取 $20.0\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 橄榄油模拟物置于 $125~\mathrm{mL}$ 分液漏斗中，分别加入 $1\mathrm{mL}$ 的四氢呋喃溶

液和 $5.0\mathrm{mL}$ 庚烷。充分混匀后，加入 $20~\mathrm{mL}$ 乙酸溶液，振荡 $5\mathrm{min}$ ，静置 $15\mathrm{min}$ ，待两相完全分离后，收集水相液体。用刻度吸量管吸取上述溶液中 $10.0\mathrm{mL}$ 于 $25~\mathrm{mL}$ 烧杯中，滴加入 $5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液(3.2.4)，体积同3.4.6中氢氧化钠溶液体积，充分混匀（溶液 $\mathrm{pH}$ 应在 $8.0\sim 9.9$ 范围内）。用刻度吸量管吸取上述溶液 $2.0\mathrm{mL}$ 于 $5\mathrm{mL}$ 试管中，以下按5.2.1中“加入 $0.4\mathrm{mL}$ 硼酸缓冲液……供高效液相色谱进样”操作。

# 5.3 空白溶液的制备

除不加试样外，采用与5.2完全相同的分析步骤、试剂和用量。

# 5.4 仪器参考条件

5.4.1 色谱柱： $\mathrm{C_{18}}$ ，柱长 $150~\mathrm{mm}$ ，内径 $4.6~\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ 。  
5.4.2 流动相：高效液相色谱流动相[硼酸缓冲液：水：甲醇 $(18:37:45)]$ 。  
5.4.3 流速： $1.0~\mathrm{mL / min}$   
5.4.4 柱温： $30^{\circ}C$ 。  
5.4.5 荧光检测器：激发波长 $394\mathrm{nm}$ ，发射波长 $480\mathrm{nm}$ 。  
5.4.6 进样量： $20~\mu \mathrm{L}$

# 5.5 绘制标准工作曲线

按5.4所列测定条件，分别将水基、酸性、酒精类、油基食品模拟物标准工作溶液(3.4.4、3.4.5、3.4.6)进高效液相色谱仪器测定。以食品模拟物中1,3-苯二甲胺浓度为横坐标，单位为“ $\mathrm{mg / L}$ （水基、酸性、酒精类食品模拟物）或 $\mathrm{mg / kg}$ （油基食品模拟物）”，以1.3-苯二甲胺峰面积为纵坐标，绘制标准工作曲线。标准溶液色谱图见图A.1。

# 5.6 试样溶液的测定

按5.4所列条件，分别将模拟物溶液(5.2)、空白溶液(5.3)依次进液相色谱仪测定，得到目标物峰面积，扣除空白值。

# 6 分析结果的表述

由标准曲线得到试样溶液中1,3-苯二甲胺的浓度，按GB5009.156进行迁移量计算，得到食品接触材料及制品中1,3-苯二甲胺的迁移量。计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

水基、酸性、酒精类食品模拟物中1，3-苯二甲胺的方法检出限为 $0.004\mathrm{mg / L}$ 、定量限为 $0.01\mathrm{mg / L}$ 油基食品模拟物中1，3-苯二甲胺的方法检出限为 $0.006\mathrm{mg / kg}$ 、定量限为 $0.02\mathrm{mg / kg}$ 。

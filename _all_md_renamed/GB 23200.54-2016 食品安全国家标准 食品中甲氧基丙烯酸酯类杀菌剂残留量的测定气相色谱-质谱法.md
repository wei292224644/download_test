# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2237—2008《进出口食品中甲氧基丙烯酸酯类杀菌剂残留量检测方法气相色谱一质谱法》。与SN/T2237—2008相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；

——标准名称和范围中“进出口食品”改为“食品”；

——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

# 1 范围

本标准规定了食品中11种甲氧基丙烯酸酯类杀菌剂残留量(参见附录A)的制样和测定方法(气相色谱一质谱法)。

本标准适用于苹果、梨、葡萄、甘蓝、黄瓜、蘑菇、橙汁、茶饮料、大米、板栗、牛肉、猪肉、鸡肉、鸡蛋和牛奶等食品中11种甲氧基丙烯酸酯类[包括嘧菌酯、醚菌胺、嘧螨酯、氟嘧菌酯、醚菌酯、苯氧菌酯(Z、E两种构型)、肟醚菌胺、啶氧菌酯、吡唑醚菌酯和肟菌酯]杀菌剂残留量的测定,其他食品可参照执行。

# 3 原理

试样用有机溶剂超声提取，凝胶渗透色谱(GPC)系统净化，洗脱液浓缩并定容后，气相色谱质谱仪测定，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 甲苯 $(\mathrm{C}_7\mathrm{H}_8)$ ：色谱纯。

4.1.2 环己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{12}\right)$ ：色谱纯。

4.1.3 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ 。

4.1.4 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ ：色谱纯。

4.1.5 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。

4.1.6 三氯甲烷 $\mathrm{(CHCl_3)}$

4.1.7 氯化钠(NaCl)。

4.1.8 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：用前经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置于干燥器中冷却后备用。

# 4.2 溶液配制

4.2.1 环己烷—乙酸乙酯(1+1，体积比)：取 $100\mathrm{mL}$ 环己烷，加入 $100\mathrm{mL}$ 乙酸乙酯，摇匀备用。

# 4.3 标准品

4.3.1 甲氧基丙烯酸酯类杀菌剂标准物质：参见附录A。

# 4.4 标准溶液配制

4.4.1 标准储备液：准确称取杀菌剂标准品 $50\mathrm{mg}$ (精确到 $0.1\mathrm{mg}$ ), 用甲苯溶解, 转移至 $50\mathrm{mL}$ 棕色容量瓶中并定容至刻度。该溶液浓度为 $1\mathrm{mg/mL}$ 。 $-18^{\circ}\mathrm{C}$ 以下避光保存, 保存期为 12 个月。  
4.4.2 标准中间液：吸取 $1.00\mathrm{mL}$ 标准储备液移入 $100\mathrm{mL}$ 棕色容量瓶中，用环己烷一乙酸乙酯定容，该溶液浓度为 $10~\mu \mathrm{g / mL}$ 。 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 避光保存，保存期为1个月。  
4.4.3 标准工作液：吸取一定量标准中间液，用环己烷一乙酸乙酯稀释至所需浓度，使用前配制。

# 5 仪器和设备

5.1 气相色谱质谱联用仪(GC-MS)，配有电子轰击离子源(EI源)。  
5.2 凝胶色谱净化系统(GPC)。  
5.3 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.4 均质器：最大转速 $20000\mathrm{r / min}$   
5.5 离心机：最大转速 $5000\mathrm{r / min}$   
5.6 涡旋振荡器。  
5.7 鸡心瓶： $100~\mathrm{mL}$   
5.8 塑料离心管： $50~\mathrm{mL}$   
5.9 氮吹仪。

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按GB2763的规定执行。在取样和制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 水果和蔬菜类

抽取苹果、蘑菇、黄瓜等水果或蔬菜样品 $500 \mathrm{~g}$ , 将其切碎后, 用捣碎机将样品加工成浆状, 混匀, 均分成两份作为试样, 分装入洁净的样品瓶内, 密闭并标明标记。

# 6.2.2 粮谷和坚果类

抽取大米等粮谷和板栗等坚果类样品 $500\mathrm{g}$ , 将其用粉碎机粉碎并通过 $2.0\mathrm{mm}$ 圆孔筛, 混匀, 均分成两份作为试样, 分装入洁净的盛样袋内, 密闭并标明标记。

# 6.2.3 肉类及动物内脏

抽取牛肉、鸡肉、猪肉等样品 $500\mathrm{g}$ ，切碎后，用捣碎机将样品加工成浆状，混匀，均分成两份作为试样，分装入洁净的样品瓶内，密闭并标明标记。

# 6.2.4 蛋类

抽取蛋类样品 $500 \mathrm{~g}$ , 去壳, 用捣碎机将样品加工成浆状, 混匀, 均分成两份作为试样, 分装入洁净的样品瓶内, 密闭并标明标记。

# 6.2.5 饮料和奶类

抽取饮料和奶类样品 $500\mathrm{g}$ ，混匀，均分成两份作为试样，分装入洁净的样品瓶内，密闭并标明标记。

# 6.3 试样保存

粮谷类、坚果类等试样于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 保存；新鲜水果、蔬菜类、肉、蛋、奶类和饮料等试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 水果、蔬菜、饮料类样品

将制备好的试样在室温下解冻后，放置 $30\mathrm{min}$ 。称取 $15\mathrm{g}$ 混匀的试样（精确至 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 塑料离心管中，加入 $5\mathrm{g}$ 氯化钠，涡旋混匀 $1\mathrm{min}$ ，加入 $30~\mathrm{mL}$ 环己烷一乙酸乙酯，再涡旋混匀 $1\mathrm{min}$ ，超声波提取 $10\mathrm{min}$ ，以 $4000\mathrm{r / min}$ 的转速离心 $10\mathrm{min}$ 。取上清液过填装有 $5\mathrm{g}$ 无水硫酸钠的漏斗，分取 $20~\mathrm{mL}$ 滤液(相当于 $10\mathrm{g}$ 样品)放入鸡心瓶中。 $40^{\circ}\mathrm{C}$ 以下旋转浓缩至近干。

# 7.1.2 粮谷类样品、坚果类样品

取出制备好的试样，室温放置 $30\mathrm{min}$ 。称取 $15\mathrm{g}$ 混匀的试样(精确至 $0.01\mathrm{g}$ )于 $50~\mathrm{mL}$ 塑料离心管中，加入 $5\mathrm{g}$ 氯化钠，再加入 $30~\mathrm{mL}$ 乙腈，涡旋混匀 $1\mathrm{min}$ ，超声提取 $30\mathrm{min}$ ，以 $4000\mathrm{r / min}$ 的转速离心 $10\mathrm{min}$ 。取上清液过填装有 $2\mathrm{g}$ 无水硫酸钠的漏斗，分取 $20~\mathrm{mL}$ 滤液(相当于 $10\mathrm{g}$ 样品)放入鸡心瓶中。 $40^{\circ}\mathrm{C}$ 以下旋转浓缩至近干。

# 7.1.3 肉类和动物内脏

将制备好的试样在室温下解冻后，放置 $30\mathrm{min}$ 。准确称取 $15\mathrm{g}$ 混匀的试样（精确至 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 塑料离心管中，加入 $5\mathrm{g}$ 氯化钠涡旋混匀 $1\mathrm{min}$ ，静置 $10\mathrm{min}$ ，再加入 $30\mathrm{mL}$ 乙腈，超声提取 $30\mathrm{min}$ ，以 $4000\mathrm{r / min}$ 的转速离心 $10\mathrm{min}$ 。取上清液过填装有 $5\mathrm{g}$ 无水硫酸钠的漏斗，分取 $20\mathrm{mL}$ 滤液（相当于 $10\mathrm{g}$ 样品）放入鸡心瓶中。 $40^{\circ}\mathrm{C}$ 以下旋转浓缩至近干。

# 7.1.4 蛋类、奶类样品

将制备好的试样在室温下解冻后，放置 $30\mathrm{min}$ 。称取 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )混匀的试样于 $50~\mathrm{mL}$ 塑料离心管中，加入 $5\mathrm{mL}$ 甲醇、 $10\mathrm{g}$ 无水硫酸钠以及 $20~\mathrm{mL}$ 三氯甲烷，涡旋混匀 $1\mathrm{min}$ ，振荡 $20\mathrm{min}$ ，以 $4000\mathrm{r / min}$ 的转速离心 $10\mathrm{min}$ 。用移液器吸取下层三氯甲烷相，过填有 $5\mathrm{g}$ 无水硫酸钠的漏斗，收集于鸡心瓶中。再加入 $20~\mathrm{mL}$ 三氯甲烷重复上述步骤进行二次提取。用 $20~\mathrm{mL}$ 乙腈冲洗漏斗和无水硫酸钠若干次。合并滤液于鸡心瓶中， $40^{\circ}C$ 以下水浴减压浓缩至近干。

# 7.2 净化

7.2.1 用 $4\mathrm{mL}$ 环己烷一乙酸乙酯溶解鸡心瓶中残留物，转移至GPC进样瓶中，供GPC净化用。  
7.2.2 GPC条件：采用biobeads(S-X3)凝胶净化柱，流动相为环己烷乙酸乙酯，流速 $4\mathrm{mL / min}$ ，检测波长 $254~\mathrm{nm}$ 。进样体积为 $2\mathrm{mL}$ 。在第 $15\mathrm{min}\sim$ 第 $20\mathrm{min}$ 时间段内收集洗脱液至鸡心瓶中，或根据淋洗曲线确定收集时间段。  
7.2.3 浓缩：将鸡心瓶中收集的洗脱液在 $40^{\circ}\mathrm{C}$ 以下浓缩至近干。  
7.2.4溶解：用 $1.0\mathrm{mL}$ 环己烷一乙酸乙酯溶解鸡心瓶内残留物，转移到GC进样瓶中，供GC-MS检测。

# 7.3 测定

# 7.3.1 气相色谱分析参考条件

a）色谱柱：HP-5ms石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度程序： $160^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，以 $15^{\circ}\mathrm{C / min}$ 上升至 $280^{\circ}\mathrm{C}$ ，保持 $4\mathrm{min}$ ，再以 $10^{\circ}\mathrm{C / min}$ 上升至 $300^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ ；  
c）载气程序流速： $0.8\mathrm{mL / min}$ 保持 $10\mathrm{min}$ ，以 $0.5\mathrm{mL / min}$ 上升至 $1.5\mathrm{mL / min}$ ，保持 $12\mathrm{min}$   
d) 进样口温度: $280^{\circ} \mathrm{C}$ ;   
e) 质谱仪接口温度: $280^{\circ} \mathrm{C}$ ;   
f）载气：高纯氦气，纯度 $\geq 99.999\%$   
g）进样方式：无分流进样， $1.5\mathrm{min}$ 后开阀；  
h）进样量： $2.0\mu \mathrm{L}$   
i）GC-MS接口温度： $280^{\circ}C$

# 7.3.2 质谱分析参考条件

a）电离方式：EI；

b）电离能量： $70\mathrm{eV}$   
c）四极杆温度： $150^{\circ}\mathrm{C}$   
d) 离子源温度: $230^{\circ} \mathrm{C}$ ;   
e）检测方式：选择离子模式(SIM)；  
f) 监测离子 $(m / z)$ : 11种杀菌剂的选择监测离子见表1;  
g）溶剂延迟： $6\mathrm{min}$

# 7.3.4 定量测定

采用外标法定量测定。根据杀菌剂的含量情况,选取峰面积相近的标准工作溶液一起进行色谱质谱分析。标准工作溶液和待测样液中杀菌剂的响应值均应在检测的线性范围内。对标准工作溶液和待测样液等体积穿插进样测定。

# 7.4 空白实验

除不称取试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算与表述

用气相色谱质谱仪工作站处理软件中的外标法，或按式(1)计算样品中各杀菌剂的含量。

$$
X = \frac {\left(A - A _ {o}\right) \times C _ {s} \times V}{A _ {s} \times m} \tag {1}
$$

式中：

$X$ ——样品中杀菌剂的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A$ ——样液中杀菌剂的色谱峰面积；

$A_{o}$ ——空白实验中杀菌剂的色谱峰面积；

$A_{s}$ ——标准溶液中杀菌剂的色谱峰面积；

$C_s$ ——标准溶液中杀菌剂的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——最终样液所代表的样品量，单位为克(g)。

计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 食品营养强化剂 葡萄糖酸铜

# 1 范围

本标准适用于由葡萄糖酸与氧化铜或葡萄糖酸盐与铜盐反应，结晶而得的食品营养强化剂葡萄糖酸铜产品；该产品也可由葡萄糖电解成葡萄糖酸后，用碳酸铜中和而制成。

# 2 分子式、结构式和相对分子质量

# 2.1 分子式

$$
\mathrm {C} _ {1 2} \mathrm {H} _ {2 2} \mathrm {C u O} _ {1 4}
$$

# 检验方法

# A.1 安全提示

本标准试验方法中使用的部分试剂具有毒性或腐蚀性，使用时需小心谨慎并按相关规定操作。若溅到皮肤上应立即用水冲洗，严重者应立即治疗。在使用挥发性酸时，应在通风橱中进行。

# A.2 一般规定

除非另有说明，在分析中仅使用确认为分析纯试剂和GB/T6682中规定的三级水。试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸。

A.3.1.2 氨水溶液 $(6\mathrm{mol} / \mathrm{L})$ ：取浓度为 $28\%$ 氢氧化铵液 $400~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。该溶液中氨含量为 $9.5\% \sim 10.5\%$ 。

A.3.1.3 亚铁氰化钾溶液 $(0.1\mathrm{g / mL})$ ：称取 $1\mathrm{g}$ 亚铁氰化钾 $\left[\mathrm{K}_4\mathrm{Fe}(\mathrm{CN})_6\cdot 3\mathrm{H}_2\mathrm{O}\right]$ ，溶解于 $10~\mathrm{mL}$ 水中。该溶液现用现配。

A.3.1.4 乙酸。

A.3.1.5 苯肼：临用前蒸馏。

A.3.1.6 铁片：光亮未生锈。

# A.3.2 鉴别方法

# A.3.2.1 铜离子 $(\mathrm{Cu}^{2+})$ 鉴别

# A.3.2.1.1 试样溶液

称取 $5.0\mathrm{g}$ 试样，加水溶解并定容至 $100~\mathrm{mL}$

# A.3.2.1.2 分析步骤

A.3.2.1.2.1 取适量试样溶液，加入少量盐酸酸化，放入一块铁片，铁片表面很快形成红色的金属铜薄膜。  
A.3.2.1.2.2 取适量试样溶液，加入过量氨水溶液，生成蓝色沉淀，随后沉淀溶解，溶液变为深蓝色。  
A.3.2.1.2.3 取适量试样溶液，加入亚铁氰化钾溶液，生成红棕色沉淀，沉淀不溶于稀硝酸。

# A.3.2.2 葡萄糖酸根鉴别

# A.3.2.2.1 方法提要

试样在乙酸介质中，与苯肼共热，生成黄色葡萄糖酰苯肼结晶。

# A.3.2.2.2 试验步骤

称取葡萄糖酸铜试样 $1.0\mathrm{g}$ ，用温水溶解，定容至 $10\mathrm{mL}$ ，作为测试液。取测试液 $5\mathrm{mL}$ ，加入 $0.7\mathrm{mL}$ 冰乙酸和 $1\mathrm{mL}$ 苯肼，水浴加热 $30\mathrm{min}$ ，放至室温，用玻璃棒摩擦试管内壁，则析出黄色结晶。

# A.3.2.3 红外吸收光谱鉴别

采用溴化钾压片法，按照GB/T6040的规定进行试验，试样的红外光谱图应与对照谱图（对照谱图见附录B)一致。

# A.4 葡萄糖酸铜（以 $\mathbf{C}_{12}\mathbf{H}_{22}\mathbf{CuO}_{14}$ 计）的测定

# A.4.1 方法提要

在微酸性条件下，试样溶液中加入过量碘化钾与二价铜定量反应，以淀粉为指示剂，用硫代硫酸钠标准溶液滴定析出的碘。

# A.4.2 试剂和材料

A.4.2.1 碘化钾。

A.4.2.2 乙酸。

A.4.2.3 硫氰酸铵。

A.4.2.4 硫代硫酸钠标准溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol} / \mathrm{L}$

A.4.2.5 淀粉指示液： $10\mathrm{g / L}$

# A.4.3 分析步骤

称取约 $1.5\mathrm{g}$ 试样于 $250~\mathrm{mL}$ 锥形瓶中（精确至 $0.001\mathrm{g}$ ），加入 $100~\mathrm{mL}$ 水，使其溶解。加 $2\mathrm{mL}$ 乙酸及 $5\mathrm{g}$ 碘化钾，充分混匀。用硫代硫酸钠标准溶液滴定至浅黄色。加 $2.0\mathrm{g}$ 硫氰酸铵，混匀，再加 $3\mathrm{mL}$ 淀粉指示液，继续滴定至乳白色。同时做空白试验。

# A.4.4 结果计算

葡萄糖酸铜的质量分数 $\omega_{1}$ ，数值以 $\%$ 表示，按式(A.1)计算：

$$
w _ {1} = \frac {\left(V - V _ {0}\right) \times c \times M}{m _ {1} \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V$ ——滴定试样溶液所消耗硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定试样空白溶液所消耗硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫代硫酸钠标准溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——葡萄糖酸铜的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(\mathrm{C_{12}H_{22}CuO_{14}}) = 453.84]$

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

试验结果在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 还原性物质（以 $\mathbf{D} - \mathbf{C}_6\mathbf{H}_{12}\mathbf{O}_6$ 计）的测定

# A.5.1 方法提要

在沸腾条件下，还原性物质与过量的葡萄糖酸铜或碱性柠檬酸铜溶液反应生成相当量的氧化亚铜沉淀，冷却。加入盐酸溶解氧化亚铜沉淀后，准确加入过量的碘标准溶液。亚铜离子与相当量的碘发生氧化还原反应，过量的碘用硫代硫酸钠标准溶液滴定。通过计算亚铜离子消耗的碘量，从而计算出试样中还原性物质的含量。

# A.5.2 试剂和材料

# A.5.2.1 碱性柠檬酸铜溶液

称取 $173\mathrm{g}$ 柠檬酸钠 $\mathrm{(C_6H_5Na_3O_7\cdot 2H_2O)}$ 和 $117\mathrm{g}$ 碳酸钠 $\mathrm{(Na_2CO_3\cdot H_2O)}$ ，加入约 $700~\mathrm{mL}$ 水加热溶解，必要时用滤纸过滤。称取 $17.3\mathrm{g}$ 硫酸铜 $\mathrm{(CuSO_4\cdot 5H_2O)}$ 于另一烧杯中，用约 $100~\mathrm{mL}$ 水溶解后，缓慢加入上述溶液，加入过程中不断搅拌，冷却，再加水至 $1000~\mathrm{mL}$ ，混匀。

A.5.2.2 碘标准溶液： $c\left(\frac{1}{2}\mathrm{I}_2\right) = 0.1\mathrm{mol} / \mathrm{L}$   
A.5.2.3 硫代硫酸钠标准溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$   
A.5.2.4 淀粉指示液： $10\mathrm{g / L}$   
A.5.2.5 乙酸溶液： $0.6\mathrm{mol / L}$   
A.5.2.6 盐酸溶液： $3\mathrm{mol} / \mathrm{L}$

# A.5.3 分析步骤

称取约 $1\mathrm{g}$ 试样于 $250~\mathrm{mL}$ 碘容量瓶中(精确至 $0.001\mathrm{g}$ )，用 $10~\mathrm{mL}$ 水溶解。加入 $25.0~\mathrm{mL}$ 碱性柠檬酸铜溶液(A.5.2.1)，在瓶口盖上小表面皿，准确微沸 $5\mathrm{min}$ ，迅速冷却至室温。加入 $25.0~\mathrm{mL}$ 乙酸溶液(A.5.2.5)，摇匀，准确加入 $10.0~\mathrm{mL}$ 碘标准溶液(A.5.2.2)，盖紧瓶塞，摇匀，放置 $10\mathrm{min}$ 。加入 $10~\mathrm{mL}$ 盐酸溶液，再加入 $3.0~\mathrm{mL}$ 淀粉指示液(A.5.2.4)，立即用硫代硫酸钠标准溶液(A.5.2.3)滴定至终点。

# A.5.4 结果计算

还原性物质(以 $\mathrm{D - C_6H_{12}O_6}$ 计)的质量分数 $\omega_{2}$ ，数值以\%表示，按式(A.2)计算：

$$
w _ {2} = \frac {\left(V _ {1} c _ {1} - V _ {2} c _ {2}\right) \times 27}{m _ {2} \times 1000} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$V_{1}$ ——准确加入的碘标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——碘标准溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{2}$ ——滴定试样溶液所消耗的硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{2}$ ——硫代硫酸钠标准溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

27 由经验测得的用于D-葡萄糖的换算系数，单位为克每摩尔 $(\mathrm{g / mol})$

$m_{2}$ ——试样质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

# A.6 总碑（以As计)的测定（氢化物原子荧光光度法）

称取约 $0.5\mathrm{g}$ 试样于烧杯中(精确至 $0.001\mathrm{g}$ )，加 $10~\mathrm{mL}$ 水溶解，转移至 $25~\mathrm{mL}$ 容量瓶中，再加入浓

盐酸 $1.25\mathrm{mL}$ ，用水定容至刻度，摇匀。取上述溶液 $10.0\mathrm{mL}$ ，置于 $100\mathrm{mL}$ 容量瓶，加入 $50\mathrm{g/L}$ 硫脲溶液 $10\mathrm{mL}$ ，不断摇动至白色沉淀完全溶解，用 $5.0\%$ 盐酸溶液（体积分数）定容至刻度，摇匀，作为试样溶液。以下操作按GB5009.76第二法进行。

# A.7 铅(Pb)的测定

# A.7.1 石墨炉原子吸收法

称取约 $0.25\mathrm{g}$ 试样于 $25~\mathrm{mL}$ 容量瓶中（精确至 $0.001\mathrm{g}$ ），加入 $15~\mathrm{mL}$ 水超声溶解后加入2滴浓硝酸，定容至刻度，作为试样溶液。以下操作按GB5009.12第一法进行。

# A.8 氯化物（以C1计）的测定

# A.8.1 试剂和材料

A.8.1.1 硝酸。  
A.8.1.2 硝酸银溶液： $0.10\mathrm{mol / L}$   
A.8.1.3 氯化物（以Cl计）标准溶液： $0.02\mathrm{mol / L}$

# A.8.2 分析步骤

称取约 $1\mathrm{g}$ 试样于 $50~\mathrm{mL}$ 比色管中（精确至 $0.1\mathrm{g}$ ），加 $30~\mathrm{mL} \sim 40~\mathrm{mL}$ 水溶解。分别加入 $1\mathrm{mL}$ 硝酸， $1\mathrm{mL}$ 硝酸银溶液，定容至 $50~\mathrm{mL}$ ，混匀。作为试样溶液，置无阳光直射处 $5\mathrm{min}$ 。取 $1.0\mathrm{mL}$ 氯化物（以 $\mathrm{Cl}$ 计）标准溶液于 $50~\mathrm{mL}$ 比色管中，加入 $1\mathrm{mL}$ 硝酸， $1\mathrm{mL}$ 硝酸银溶液，定容至 $50~\mathrm{mL}$ ，作为标准溶液。试样溶液的浊度不得深于标准溶液的浊度，即氯化物含量小于或等于 $0.07\%$ 。

# A.9 硫酸盐（以 $\mathrm{SO}_4$ 计）的测定

# A.9.1 试剂和材料

A.9.1.1 盐酸溶液： $3\mathrm{mol} / \mathrm{L}$   
A.9.1.2 氯化钡溶液：取 $12\mathrm{g}$ 氯化钡，用 $100~\mathrm{mL}$ 水溶解。  
A.9.1.3 硫酸盐（以 $\mathrm{SO}_4$ 计）标准溶液： $0.01\mathrm{mol} / \mathrm{L}$

# A.9.2 分析步骤

称取约 $2\mathrm{g}$ 试样于 $50~\mathrm{mL}$ 比色管中（精确至 $0.1\mathrm{g}$ ），加 $30~\mathrm{mL} \sim 40~\mathrm{mL}$ 水溶解。分别加入 $1\mathrm{mL}$ 盐酸溶液， $3\mathrm{mL}$ 氯化钡溶液，定容至 $50~\mathrm{mL}$ ，混匀。放置 $10\mathrm{min}$ ，作为试样溶液。取 $1.0\mathrm{mL}$ 硫酸盐（以 $\mathrm{SO}_4$ 计）的标准溶液于 $50~\mathrm{mL}$ 比色管中，加入 $1\mathrm{mL}$ 盐酸溶液， $3\mathrm{mL}$ 氯化钡溶液，定容至 $50~\mathrm{mL}$ ，混匀，放置 $10\mathrm{min}$ ，作为标准溶液。试样溶液的浊度不得深于标准溶液的浊度，为硫酸盐小于或等于 $0.05\%$ 。

# 前言

本标准代替GB1886.180—2016《食品安全国家标准 食品添加剂 $\beta$ -环状糊精》。

本标准与GB1886.180—2016相比，主要变化如下：

——删除了感官要求中的滋味项目内容；

——修改了 $\beta$ -环状糊精含量(以干基计)、水分的试验方法；

——增加了微生物限量要求。

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在未注明其他要求时，均指分析纯试剂和符合GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品，在未注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 比旋光度试验

称取试样 $1.00\mathrm{g}$ 溶解于 $100~\mathrm{mL}$ 水中，按GB/T613规定的方法测定，比旋光度应在 $+160^{\circ} \sim +164^{\circ}$ 。

# A.2.2 色谱试验

在 $\beta$ -环状糊精含量测定试验中，试样溶液色谱图的主峰保留时间应和标准溶液色谱图的主峰保留时间相一致。

# A.3 $\beta$ -环状糊精含量（以干基计）的测定

# A.3.1 试剂和材料

A.3.1.1 甲醇：色谱纯。  
A.3.1.2 水：GB/T6682规定的一级水。  
A.3.1.3 β-环状糊精标准品：纯度 $\geqslant 99.5\%$

# A.3.2 仪器和设备

高效液相色谱仪：配备示差折光检测器。

# A.3.3 参考色谱条件

A.3.3.1 色谱柱：以十八烷基硅烷键合硅胶为填充剂的 $10\mu \mathrm{m}$ 多孔硅胶色谱柱（ $\Phi 4.6\mathrm{mm} \times 25\mathrm{cm}$ ），或其他等效的色谱柱。

A.3.3.2 流动相：水：甲醇 $= 85:15$   
A.3.3.3 柱温： $30^{\circ}C$ 。  
A.3.3.4 检测器温度： $35^{\circ}C$   
A.3.3.5 流速： $1.5\mathrm{mL / min}$   
A.3.3.6 进样量： $10~\mu \mathrm{L}$

# A.3.4 分析步骤

# A.3.4.1 标准溶液的制备

称取 $\beta$ -环状糊精标准品 $0.05\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，置于 $10\mathrm{mL}$ 容量瓶中，用流动相溶解并定容至刻度，混匀。色谱分析前用 $0.45\mu \mathrm{m}$ 微孔滤膜过滤。

# A.3.4.2 试样溶液的制备

称取试样 $0.05\mathrm{g}$ ，精确至 $0.0001\mathrm{g}$ ，置于 $10\mathrm{mL}$ 容量瓶中，用流动相溶解并定容至刻度，混匀。色谱分析前用 $0.45\mu \mathrm{m}$ 微孔滤膜过滤。

# A.3.5 测定

在A.3.3参考色谱条件下，分别对标准溶液和试样溶液进行测定。系统适用性为重复注入标准溶液5次，所得响应面积的相对标准偏差小于 $2.0\%$ 。

# A.3.6 结果计算

$\beta$ -环状糊精含量(以干基计)的质量分数 $\omega_{1}$ 按式(A.1)计算。

$$
w _ {1} = \frac {A _ {\mathrm {u}} c _ {\mathrm {s}} \times 100}{A _ {\mathrm {s}} c _ {\mathrm {u}} \times (100 - w)} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$A_{\mathrm{u}}$ ——试样溶液中 $\beta$ -环状糊精的峰面积值；

$c_{\mathrm{s}}$ ——标准溶液的浓度，单位为克每毫升 $(\mathrm{g / mL})$ ；

100 ——质量换算系数；

$A_{s}$ ——标准溶液中 $\beta$ -环状糊精的峰面积值；

$c_{\mathrm{u}}$ ——试样溶液的浓度，单位为克每毫升 $(\mathrm{g / mL})$ ；

$w$ ——试样的水分，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.5\%$ 。

# A.4 还原糖的测定

# A.4.1 试剂和材料

A.4.1.1 碱性柠檬酸铜试液：称取 $2.5\mathrm{g}$ 硫酸铜、 $5.0\mathrm{g}$ 柠檬酸、 $14.4\mathrm{g}$ 无水碳酸钠，用水溶解并定容 $100\mathrm{mL}$ 。

A.4.1.2 乙酸溶液： $2 + 81$

A.4.1.3 碘标准溶液： $c\left(\frac{1}{2}\mathrm{I}_2\right) = 0.05\mathrm{mol / L}$

A.4.1.4 盐酸溶液： $6 + 94$

A.4.1.5 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.05\mathrm{mol / L}$

A.4.1.6 淀粉指示液： $10\mathrm{g / L}$

# A.4.2 分析步骤

称取 $1\mathrm{g}$ 试样，加入 $25~\mathrm{mL}$ 水，缓慢加热溶解，冷却，加入 $20~\mathrm{mL}$ 碱性柠檬酸铜试液和几颗玻璃珠。加热溶液约 $4\mathrm{min}$ 后溶液开始沸腾，保持煮沸 $3\mathrm{min}$ 。迅速冷却后，加入 $100~\mathrm{mL}$ 乙酸溶液和 $20.0~\mathrm{mL}$ 碘标准溶液，边摇晃溶液，边加入 $25~\mathrm{mL}$ 盐酸溶液。当沉淀溶解后，用硫代硫酸钠标准滴定溶液滴定多余的碘液。在近滴定终点，加入 $2\mathrm{mL}$ 淀粉指示液。滴定所需的硫代硫酸钠标准滴定溶液体积应大于等于 $12.8~\mathrm{mL}$ ，即还原糖含量小于等于 $1.0\%$ 。

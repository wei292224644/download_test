# 食品安全国家标准

# 食品中丙烯酰胺的测定

2014-12-01 发布

2015-05-01 实施

# 前言

本标准代替GB/T5009.204—2005《食品中丙烯酰胺含量的测定方法气相色谱一质谱（GC-MS）法》。

本标准与GB/T5009.204—2005相比，主要变化如下：

——增加第一法 稳定性同位素稀释的液相色谱-质谱/质谱法；

——气相色谱-质谱法将外标法改为稳定性同位素稀释法。

# 食品安全国家标准食品中丙烯酰胺的测定

# 1 范围

本标准规定了食品中丙烯酰胺的测定方法。

本标准适用于热加工（如煎、炙烤、焙烤等）食品中丙烯酰胺的测定。

# 第一法 稳定性同位素稀释的液相色谱-质谱/质谱法

# 2 原理

本标准应用稳定性同位素稀释技术，在试样中加入 ${}^{13}\mathrm{C}_3$ 标记的丙烯酰胺内标溶液，以水为提取溶剂，经过固相萃取柱或基质固相分散萃取净化后，以液相色谱-质谱/质谱的多反应离子监测(MRM)或选择反应监测(SRM)进行检测，内标法定量。

# 3 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲酸 (HCOOH)：色谱纯。  
3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.3 正己烷 $(\mathrm{n - C_6H_{14}})$ ：分析纯，重蒸后使用。  
3.1.4 乙酸乙酯 $(\mathrm{CH}_3\mathrm{COOC}_2\mathrm{H}_5)$ ：分析纯，重蒸后使用。  
3.1.5 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $400^{\circ}\mathrm{C}$ ，烘烤 $4\mathrm{h}$ 。  
3.1.6 硫酸铵 $\mathrm{[(NH_4)_2SO_4]}$   
3.1.7 硅藻土：ExtrelutTM20或相当产品。

# 3.2 标准品

3.2.1 丙烯酰胺 $\mathrm{(CH_2 = CHCONH_2)}$ 标准品（纯度 $>99\%$ ）。  
3.2.2 $^{13}\mathrm{C}_{3}$ -丙烯酰胺（ ${}^{13}\mathrm{CH}2 = {}^{13}\mathrm{CH}{}^{13}\mathrm{CONH}_2$ ）标准品（纯度>98%）。

# 3.3 标准溶液的配制

# 3.3.1 丙烯酰胺标准溶液的配制

3.3.1.1 丙烯酰胺标准储备溶液（ $1000\mathrm{mg / L}$ ）：准确称取丙烯酰胺标准品，用甲醇溶解并定容，使丙烯酰胺浓度为 $1000\mathrm{mg / L}$ ，置 $-20^{\circ}\mathrm{C}$ 冰箱中保存。  
3.3.1.2 丙烯酰胺中间溶液（ $100\mathrm{mg / L}$ ）：移取丙烯酰胺标准储备溶液 $1\mathrm{mL}$ ，加甲醇稀释至 $10\mathrm{mL}$ ，使丙烯酰胺浓度为 $100\mathrm{mg / L}$ ，置 $-20^{\circ}\mathrm{C}$ 冰箱中保存。  
3.3.1.3 丙烯酰胺工作溶液 I（ $10\mathrm{mg/L}$ ）：移取丙烯酰胺中间溶液 $1\mathrm{mL}$ ，用 $0.1\%$ 甲酸溶液稀释至

$10 \mathrm{~mL}$ , 使丙烯酰胺浓度为 $10 \mathrm{mg} / \mathrm{L}$ 。临用时配制。

3.3.1.4 丙烯酰胺工作溶液II（ $1\mathrm{mg / L}$ ）：移取丙烯酰胺工作溶液I $1\mathrm{mL}$ ，用 $0.1\%$ 甲酸溶液稀释至 $10\mathrm{mL}$ ，使丙烯酰胺浓度为 $1\mathrm{mg / L}$ 。临用时配制。

# 3.3.2 $^{13}\mathrm{C}_{3}$ -丙烯酰胺内标溶液

3.3.2.1 ${}^{13}\mathrm{C}_{3}$ -丙烯酰胺内标储备溶液（ $1000\mathrm{mg / L}$ ）：准确称取 ${}^{13}\mathrm{C}_{3}$ -丙烯酰胺标准品，用甲醇溶解并定容，使 ${}^{13}\mathrm{C}_{3}$ -丙烯酰胺浓度为 $1000\mathrm{mg / L}$ ，置 $-20^{\circ}\mathrm{C}$ 冰箱保存。  
3.3.2.2 内标工作溶液（ $10\mathrm{mg / L}$ ）：移取内标储备溶液 $1\mathrm{mL}$ ，用甲醇稀释至 $100\mathrm{mL}$ ，使 ${}^{13}\mathrm{C}_{3}$ -丙烯酰胺浓度为 $10\mathrm{mg / L}$ ，置 $-20^{\circ}\mathrm{C}$ 冰箱保存。

# 3.3.3 标准曲线工作溶液

取6个 $10\mathrm{mL}$ 容量瓶，分别移取 $0.1\mathrm{mL}$ 、 $0.5\mathrm{mL}$ 、 $1\mathrm{mL}$ 丙烯酰胺工作溶液 $\mathrm{II}(1\mathrm{mg / L})$ 和 $0.5\mathrm{mL}$ 、 $1\mathrm{mL}$ 和 $3\mathrm{mL}$ 丙烯酰胺工作溶液I（ $10\mathrm{mg / L}$ ）与内标工作溶液（ $10\mathrm{mg / L}$ ） $0.1\mathrm{mL}$ ，用 $0.1\%$ 甲酸溶液稀释至刻度。标准系列溶液中丙烯酰胺的浓度分别为 $10~\mu \mathrm{g / L}$ 、 $50~\mu \mathrm{g / L}$ 、 $100~\mu \mathrm{g / L}$ 、 $500~\mu \mathrm{g / L}$ 、 $1000~\mu \mathrm{g / L}$ 、 $3000~\mu \mathrm{g / L}$ ，内标浓度为 $100~\mu \mathrm{g / L}$ 。临用时配制。

# 4 仪器和设备

4.1 液相色谱-质谱/质谱联用仪（LC-MS/MS）。  
4.2 HLB固相萃取柱： $6\mathrm{mL}$ 、 $200\mathrm{mg}$ ，或相当产品。  
4.3 Bond Elut-Accucat固相萃取柱： $3\mathrm{mL}$ 、 $200\mathrm{mg}$ ，或相当产品。  
4.4 组织粉碎机。  
4.5 旋转蒸发仪。  
4.6 氮气浓缩器。  
4.7 振荡器。  
4.8 玻璃层析柱：柱长 $30\mathrm{cm}$ ，柱内径 $1.8\mathrm{cm}$ 。  
4.9 涡旋混合器。  
4.10 超纯水装置。  
4.11 分析天平：感量为 $0.1\mathrm{mg}$ 。  
4.12 离心机：转速 $\leqslant 10000\mathrm{r / m}$

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 样品提取

取 $50 \mathrm{~g}$ 试样, 经粉碎机粉碎, $-20^{\circ} \mathrm{C}$ 冷冻保存。准确称取试样 $1 \mathrm{~g} \sim 2 \mathrm{~g}$ (精确到 $0.001 \mathrm{~g}$ ), 加入 $10 \mathrm{mg} / \mathrm{L}^{13} \mathrm{C}_{3}$ -丙烯酰胺内标工作溶液 $10 \mu \mathrm{L}$ (或 $20 \mu \mathrm{L}$ ), 相当于 $100 \mathrm{ng}$ (或 $200 \mathrm{ng}$ ) 的 $^{13} \mathrm{C}_{3}$ -丙烯酰胺内标, 再加入超纯水 $10 \mathrm{~mL}$ , 振摇 $30 \mathrm{~min}$ 后, 于 $4000 \mathrm{r} / \mathrm{m}$ 离心 $10 \mathrm{~min}$ , 取上清液待净化。

# 5.1.2 样品净化

注：任选下列一种方法进行净化。

5.1.2.1 基质固相分散萃取方法（选择1）：在试样提取的上清液中加入硫酸铵 $15\mathrm{g}$ ，振荡 $10\mathrm{min}$ ，使其充分溶解，于 $4000\mathrm{r / m}$ 离心 $10\mathrm{min}$ ，取上清液 $10\mathrm{mL}$ ，备用。如上清液不足 $10\mathrm{mL}$ ，则用饱和硫酸铵补足。取洁净玻璃层析柱，在底部填少许玻璃棉并压紧，依次填装 $10\mathrm{g}$ 无水硫酸钠、 $2\mathrm{g}$ 硅藻土。称取 $5\mathrm{g}$ 硅藻土 $\mathrm{Extrelut}^{\mathrm{TM}}$ 20与上述试样上清液搅拌均匀后，装入层析柱中。用 $70\mathrm{mL}$ 正己烷淋洗，控制流

速为 $2 \mathrm{~mL} / \mathrm{min}$ , 弃去正己烷淋洗液。用 $70 \mathrm{~mL}$ 乙酸乙酯洗脱丙烯酰胺, 控制流速为 $2 \mathrm{~mL} / \mathrm{min}$ , 收集乙酸乙酯洗脱溶液, 并在 $45^{\circ} \mathrm{C}$ 水浴中减压旋转蒸发至近干, 用乙酸乙酯洗涤蒸发瓶残渣三次 (每次 $1 \mathrm{~mL}$ ), 并将其转移至已加入 $1 \mathrm{~mL} 0.1 \%$ 甲酸溶液的试管中, 涡旋振荡。在氮气流下吹去上层有机相后, 加入 $1 \mathrm{~mL}$ 正己烷, 涡旋振荡, 于 $3500 \mathrm{r} / \mathrm{m}$ 离心 $5 \mathrm{~min}$ , 取下层水相经 $0.22 \mu \mathrm{m}$ 水相滤膜过滤, 待 LC-MS/MS 测定。5.1.2.2 固相萃取柱净化（选择 2）: 在试样提取的上清液中加入 $5 \mathrm{~mL}$ 正己烷, 振荡萃取 $10 \mathrm{~min}$ , 于 $10000 \mathrm{r} / \mathrm{m}$ 离心 $5 \mathrm{~min}$ , 除去有机相, 再用 $5 \mathrm{~mL}$ 正己烷重复萃取一次, 迅速取水相 $6 \mathrm{~mL}$ 经 $0.45 \mu \mathrm{m}$ 水相滤膜过滤, 待进行 HLB 固相萃取柱净化处理。HLB 固相萃取柱使用前依次用 $3 \mathrm{~mL}$ 甲醇、 $3 \mathrm{~mL}$ 水活化。取上述滤液 $5 \mathrm{~mL}$ 上 HLB 固相萃取柱, 收集流出液, 并用 $4 \mathrm{~mL} 80 \%$ 的甲醇水溶液洗脱, 收集全部洗脱液, 并与流出液合并待进行 Bond Elut-Accucat 固相萃取柱净化; Bond Elut-Accucat 固相萃取柱依次用 $3 \mathrm{~mL}$ 甲醇、 $3 \mathrm{~mL}$ 水活化后, 将 HLB 固相萃取柱净化的全部洗脱液上样, 在重力作用下流出, 收集全部流出液, 在氮气流下将流出液浓缩至近干, 用 $0.1 \%$ 甲酸溶液定容至 $1.0 \mathrm{~mL}$ , 待 LC-MS/MS 测定。

# 5.3 标准曲线的绘制

将标准系列工作液分别注入液相色谱-质谱/质谱系统，测定相应的丙烯酰胺及其内标的峰面积，以各标准系列工作液的丙烯酰胺进样浓度（ $\mu \mathrm{g} / \mathrm{L}$ ）为横坐标，以丙烯酰胺（ $\mathrm{m} / \mathrm{z} 55$ ）和 ${}^{13}\mathrm{C}_{3}$ 丙烯酰胺内标(m/z58)的峰面积比为纵坐标，绘制标准曲线。

# 5.4 试样溶液的测定

将试样溶液注入液相色谱-质谱/质谱系统中，测得丙烯酰胺（m/z55）和 ${}^{13}\mathrm{C}_3$ 丙烯酰胺内标(m/z58)的峰面积比，根据标准曲线得到待测液中丙烯酰胺进样浓度（ $\mu \mathrm{g / L}$ ），平行测定次数不少于两次。

# 5.5 质谱分析

# 6 分析结果的表述

试样中丙烯酰胺含量按公式（1）内标法计算：

$$
X = \frac {A \times f}{M} \tag {1}
$$

式中：

$X$ ——试样中丙烯酰胺的含量, 单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ;

$A$ ——试样中丙烯酰胺（ $\mathrm{m/z}55$ ）色谱峰与 ${}^{13}\mathrm{C}_3$ 丙烯酰胺内标（ $\mathrm{m/z}58$ ）色谱峰的峰面积比值对应的丙烯酰胺质量，单位为纳克（ng）；

$f$ ——试样中内标加入量的换算因子（内标为 $10\mu \mathrm{L}$ 时 $\mathrm{f} = 1$ 或内标为 $20\mu \mathrm{L}$ 时 $\mathrm{f} = 2$ ）；

$M$ ——加入内标时的取样量，单位为克（g）。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字（或小数点后1位）。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$

# 8 其他

方法定量限为 $10\mu \mathrm{g / kg}$

# 第二法 稳定性同位素稀释的气相色谱-质谱法

# 9 原理

本标准应用稳定性同位素稀释技术，在试样中加入 ${}^{13}\mathrm{C}_3$ 标记的丙烯酰胺内标溶液，以水为提取溶剂，试样提取液采用基质固相分散萃取净化、溴试剂衍生后，采用气相色谱-串联质谱仪的多反应离子监测(MRM)或气相色谱-质谱仪的选择离子监测(SIM)进行检测，内标法定量。

# 10 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为超纯水。

# 10.1 试剂

10.1.1 正己烷 $\left(\mathrm{n}-\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：分析纯，重蒸后使用。

10.1.2 乙酸乙酯 $(\mathrm{CH}_3\mathrm{COOC}_2\mathrm{H}_5)$ ：分析纯，重蒸后使用。

10.1.3 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $400^{\circ}\mathrm{C}$ ，烘烤 $4\mathrm{h}$

10.1.4 硫酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{SO}_{4}\right]$ 。

10.1.5 硫代硫酸钠 $(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3\cdot 5\mathrm{H}_2\mathrm{O})$

10.1.6 溴 $(\mathrm{Br}_2)$ 。

10.1.7 氢溴酸 (HBr)：含量 $>48.0\%$

10.1.8 溴化钾 (KBr)。

10.1.9 超纯水，电导率（ $25^{\circ}\mathrm{C}$ ） $\leqslant 0.01 \mathrm{~mS} / \mathrm{m}$ 。

10.1.10 溴试剂。

10.1.11 硅藻土：ExtrelutTM20或相当产品。

# 10.2 试剂配制

10.2.1 饱和溴水：量取 $100 \mathrm{~mL}$ 超纯水，置于 $200 \mathrm{~mL}$ 的棕色试剂瓶中，加入 $8 \mathrm{~mL}$ 溴， $4^{\circ} \mathrm{C}$ 避光放置 $8 \mathrm{~h}$ ，上层为饱和溴水溶液。

10.2.2 溴试剂: 称取溴化钾 $20.0 \mathrm{~g}$ , 加超纯水 $50 \mathrm{~mL}$ , 使完全溶解, 再加入 $1.0 \mathrm{~mL}$ 氢溴酸和 $16.0 \mathrm{~mL}$ 饱和溴水, 摇匀, 用超纯水稀释至 $100 \mathrm{~mL}, 4^{\circ} \mathrm{C}$ 避光保存。

10.2.3 硫代硫酸钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ : 称取硫代硫酸钠 $2.48\mathrm{g}$ , 加超纯水 $50\mathrm{mL}$ , 使完全溶解, 用超纯水稀释至 $100\mathrm{mL}, 4^{\circ}\mathrm{C}$ 避光保存。

10.2.4 饱和硫酸铵溶液：称取 $80\mathrm{g}$ 硫酸铵晶体，加入超纯水 $100\mathrm{mL}$ ，超声溶解，室温放置。

# 10.3 标准品

10.3.1 丙烯酰胺 $\mathrm{(CH_2 = CHCONH_2)}$ 标准品：纯度 $>99\%$   
10.3.2 $^{13}\mathrm{C}_{3}$ -丙烯酰胺（ $\mathrm{^{13}CH_2 = ^{13}CH^{13}CONH_2}$ ）标准品：纯度 $>98\%$

# 10.4 标准溶液的配制

10.4.1 丙烯酰胺及其内标溶液：同3.3.1和3.3.2。

10.4.2 标准曲线工作溶液：取5个 $10\mathrm{mL}$ 容量瓶，分别移取 $0.1\mathrm{mL}$ 、 $0.5\mathrm{mL}$ 、 $2\mathrm{mL}$ 丙烯酰胺工作溶液II（ $1\mathrm{mg / L}$ ）和 $0.5\mathrm{mL}$ 及 $1\mathrm{mL}$ 丙烯酰胺工作溶液I（ $1\mathrm{mg / L}$ ）与 $0.5\mathrm{mL}$ 内标工作溶液（ $1\mathrm{mg / L}$ ），用超纯水稀释至刻度。标准系列溶液中丙烯酰胺浓度分别为 $10~\mu \mathrm{g / L}$ 、 $50~\mu \mathrm{g / L}$ 、 $200~\mu \mathrm{g / L}$ 、 $500~\mu \mathrm{g / L}$ 、

$1000\mu \mathrm{g} / \mathrm{L}$ ，内标浓度为 $50~\mu \mathrm{g} / \mathrm{L}$ 。临用时配制。

# 12 分析步骤

# 12.1 试样制备

# 12.1.1 样品提取

取 $50\mathrm{g}$ 试样，经粉碎机粉碎， $-20^{\circ}\mathrm{C}$ 冷冻保存。准确称取试样 $2\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ，加入 $10.0\mathrm{mg / L}$ $^{13}\mathrm{C}_{3}$ -丙烯酰胺内标溶液 $10~\mu \mathrm{L}$ （或 $20~\mu \mathrm{L}$ )，相当于 $100\mathrm{ng}$ （或 $200\mathrm{ng}$ ）的 $^{13}\mathrm{C}_{3}$ -丙烯酰胺内标，再加入超纯水 $10\mathrm{mL}$ ，振荡 $30\mathrm{min}$ 后，于 $4000\mathrm{r / m}$ 离心 $10\mathrm{min}$ ，取上清液备用。

# 12.1.2 样品净化

在试样提取的上清液中加入硫酸铵 $15 \mathrm{~g}$ , 振荡 $10 \mathrm{~min}$ , 使其充分溶解, 于 $4000 \mathrm{r} / \mathrm{m}$ 离心 $10 \mathrm{~min}$ , 取上清液 $10 \mathrm{~mL}$ , 备用。如上清液不足 $10 \mathrm{~mL}$ , 则用饱和硫酸铵补足。取洁净玻璃层析柱, 在底部填少许玻璃棉, 压紧, 依次填装无水硫酸钠 $10 \mathrm{~g} 、 \mathrm{Extrelut}^{\mathrm{TM}} 20$ 硅藻土 $2 \mathrm{~g}$ 。称取 $5 \mathrm{~g} \mathrm{Extrelut}^{\mathrm{TM}} 20$ 硅藻土与上述备用的试样上清液搅拌均匀后, 装入层析柱中。用 $70 \mathrm{~mL}$ 正己烷淋洗, 控制流速为 $2 \mathrm{~mL} / \mathrm{min}$ , 弃去正己烷淋洗液。用 $70 \mathrm{~mL}$ 乙酸乙酯洗脱, 控制流速为 $2 \mathrm{~mL} / \mathrm{min}$ , 收集乙酸乙酯洗脱溶液, 并在 $45^{\circ} \mathrm{C}$ 水浴下减压旋转蒸发至近干, 用乙酸乙酯洗涤蒸发瓶残渣三次 (每次 $1 \mathrm{~mL}$ ), 并将其转移至已加入 $1 \mathrm{~mL}$ 超纯水的试管中, 涡旋振荡。在氮气流下吹去上层有机相后, 加入 $1 \mathrm{~mL}$ 正己烷, 涡旋振荡, 于 $3500 \mathrm{r} / \mathrm{m}$ 离心 $5 \mathrm{~min}$ , 取下层水相备用衍生。

# 12.1.3 衍生

试样的衍生：在试样提取液中加入溴试剂 $1\mathrm{mL}$ ，涡旋振荡， $4^{\circ}\mathrm{C}$ 放置至少 $1\mathrm{h}$ 后，加入 $0.1\mathrm{mol/L}$ 硫代硫酸钠溶液约 $100\mu \mathrm{L}$ ，涡旋振荡除去剩余的衍生剂；加入 $2\mathrm{mL}$ 乙酸乙酯，涡旋振荡 $1\mathrm{min}$ ，于 $4000\mathrm{r/m}$ 离心 $5\mathrm{min}$ ，吸取上层有机相转移至加有 $0.1\mathrm{g}$ 无水硫酸钠的试管中，加入乙酸乙酯 $2\mathrm{mL}$ 重复萃取，合并有机相；静置至少 $0.5\mathrm{h}$ ，转移至另一试管，在氮气流下吹至近干，加 $0.5\mathrm{mL}$ 乙酸乙酯溶解残渣（注意：根据仪器的灵敏度，调整溶解残渣的乙酸乙酯体积，通常情况下，采用串联质谱仪检测，其使用量为 $0.5\mathrm{mL}$ ，采用单级质谱仪检测，其使用量为 $0.1\mathrm{mL}$ ）。

标准系列溶液的衍生：量取标准系列溶液各 $1.0\mathrm{mL}$ ，按照上述试样衍生方法同步操作。

# 13 仪器参考条件

# 13.1 色谱条件

色谱柱：DB-5 ms 柱（ $30\mathrm{m}\times 0.25\mathrm{mmI.D.}\times 0.25\mu \mathrm{m}$ ）或等效柱。

进样口温度： $120^{\circ}\mathrm{C}$ 保持 $2\mathrm{min}$ ，以 $40^{\circ}\mathrm{C / min}$ 速度升至 $240^{\circ}\mathrm{C}$ ，并保持 $5\mathrm{min}$ 。

色谱柱程序温度： $65\,^{\circ}\mathrm{C}$ 保持 $1\,\mathrm{min}$ ，以 $15\,^{\circ}\mathrm{C/min}$ 速度升至 $200\,^{\circ}\mathrm{C}$ ，再以 $40\,^{\circ}\mathrm{C/min}$ 的速度升至 $240\,^{\circ}\mathrm{C}$ ，并保持 $5\,\mathrm{min}$ 。

载气：高纯氦气（纯度 $>99.999\%$ ），柱前压为 $69 \mathrm{~mPa}$ ，相当于 $10 \mathrm{psi}$ 。

不分流进样，进样体积 $1\mu \mathrm{L}$ 。

# 13.2 质谱参数

检测方式：选择离子扫描（SIM）采集。

电离模式：电子轰击源（EI），能量为 $70 \mathrm{eV}$ 。

传输线温度： $250^{\circ}\mathrm{C}$ 。

离子源温度： $200^{\circ}\mathrm{C}$ 。

溶剂延迟：6min；质谱采集时间： $6\mathrm{min}\sim 12\mathrm{min}$

丙烯酰胺监测离子为 $\mathrm{m / z}$ 106、133、150和152，定量离子为 $\mathrm{m / z}$ 150。

${}^{13}\mathrm{C}_{3}$ -丙烯酰胺内标监测离子为 $\mathrm{m / z}$ 108、136、153和155，定量离子为 $\mathrm{m / z}$ 155。

# 13.3 标准曲线的制作

将衍生的标准系列工作液分别注入气相色谱-质谱系统，测定相应的丙烯酰胺及其内标的峰面积，以各标准系列工作液的丙烯酰胺进样浓度（ $\mu \mathrm{g} / \mathrm{L}$ ）为横坐标，以丙烯酰胺及其内标 ${}^{13}\mathrm{C}_3$ 丙烯酰胺定量离子质量色谱图上测得的峰面积比为纵坐标，绘制线性曲线。

# 13.4 试样溶液的测定

将衍生的试样溶液注入气相色谱-质谱系统中，得到丙烯酰胺和内标 ${}^{13}\mathrm{C}_3$ 丙烯酰胺的峰面积比，根据标准曲线得到待测液中丙烯酰胺进样浓度（ $\mu \mathrm{g} / \mathrm{L}$ ），平行测定次数不少于两次。

# 13.5 质谱分析

分别将试样和标准系列工作液注入气相色谱-质谱仪中，记录总离子流图和质谱图（见附录A中图A.3~图A.4）及丙烯酰胺和内标的峰面积，以保留时间及碎片离子的丰度定性，要求所检测的丙烯酰胺色谱峰信噪比(S/N)大于3，被测试样中目标化合物的保留时间与标准溶液中目标化合物的保留时间一致，同时被测试样中目标化合物的相应监测离子丰度比与标准溶液中目标化合物的色谱峰丰度比一致，允许的偏差见表1。

# 14 分析结果的表述

采用内标法，按公式（1）计算试样中丙烯酰胺含量。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字（或小数点后1位）。

# 15 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 前言

本标准代替GB1907—2003《食品添加剂亚硝酸钠》。

本标准与GB1907—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 亚硝酸钠”。

# 食品安全国家标准

# 食品添加剂 亚硝酸钠

# 1 范围

本标准适用于食品添加剂碳酸钠吸收二氧化氮气体制得的亚硝酸钠。

# 2 化学名称、分子式和相对分子质量

# 2.1 化学名称

亚硝酸钠

# 2.2 分子式

$\mathrm{NaNO}_2$

# 2.3 相对分子质量

69.00（按2007年国际相对原子质量）

# 附录A

# 检验方法

# A.1 安全提示

试验中所用部分试剂具有腐蚀性，原料具有毒性，操作时应小心。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所需标准溶液、杂质标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 冰乙酸。  
A.3.1.2 盐酸溶液： $1 + 3$   
A.3.1.3 硫酸亚铁溶液： $80~\mathrm{g / L}$   
A.3.1.4 硝酸银溶液： $17\mathrm{g / L}$   
A.3.1.5 铂丝：一端弯成直径约 $4\mathrm{mm}$ 的小环，另一端绕在玻璃棒上。

# A.3.2 鉴别方法

# A.3.2.1 亚硝酸根的鉴别

A.3.2.1.1 取适量试样溶液，加盐酸溶液后加热，应放出红棕色气体。  
A.3.2.1.2 取 $1\mathrm{mL}$ 试样溶液 $(3\mathrm{g} / \mathrm{L})$ ，加冰乙酸呈酸性后，加新配制的硫酸亚铁溶液应显棕色。

# A.3.2.2 钠离子鉴别

称量 $1\mathrm{g}$ 试样，加 $20~\mathrm{mL}$ 水溶解。用铂丝环蘸盐酸，在无色火焰上燃烧至无色。再蘸取试验溶液在无色火焰上燃烧，火焰应呈鲜黄色。

# A.4 亚硝酸钠 $\mathrm{(NaNO_2)}$ 含量（以干基计）的测定

# A.4.1 方法提要

在酸性介质中，用高锰酸钾氧化亚硝酸钠，根据高锰酸钾标准滴定溶液的消耗量计算出亚硝酸钠含量。

# A.4.2 试剂和材料

A.4.2.1 硫酸溶液： $1 + 5$ 。加热硫酸溶液至 $70^{\circ}\mathrm{C}$ 左右，滴加高锰酸钾标准滴定溶液至溶液呈微红色为止。冷却，备用。

A.4.2.2 高锰酸钾标准滴定溶液： $c\left(\frac{1}{5}\mathrm{KMnO}_4\right) = 0.1\mathrm{mol / L}$

A.4.2.3 草酸钠标准滴定溶液: $c\left(\frac{1}{2}\mathrm{Na}_2\mathrm{C}_2\mathrm{O}_4\right) = 0.1 \mathrm{~mol} / \mathrm{L}$ 。称取约 $6.7 \mathrm{~g}$ 草酸钠, 溶解于 $300 \mathrm{~mL}(1+29)$ 硫酸溶液 (配制方法同 A.4.2.1) 中, 用水稀释至 $1000 \mathrm{~mL}$ , 摇匀。用高锰酸钾标准滴定溶液标定。

# A.4.3 分析步骤

称取 $2.5\mathrm{g}\sim 2.7\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $500~\mathrm{mL}$ 容量瓶中，加水使其溶解，并稀释至刻度，摇匀。在 $250~\mathrm{mL}$ 锥形瓶中，用滴定管滴加约 $40~\mathrm{mL}$ 高锰酸钾标准滴定溶液。加入 $10~\mathrm{mL}$ 硫酸溶液，用移液管移取 $25~\mathrm{mL}$ 试验溶液，加热至约 $40^{\circ}\mathrm{C}$ 。用移液管加入 $10~\mathrm{mL}$ 草酸钠标准滴定溶液，加热至 $70^{\circ}\mathrm{C}\sim 80^{\circ}\mathrm{C}$ ，继续用高锰酸钾标准滴定溶液滴定至溶液呈粉红色并保持 $30~\mathrm{s}$ 不消失为止。

# A.4.4 结果计算

亚硝酸钠 $\mathrm{(NaNO_2)}$ 含量(以干基计)的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w_{1} = \frac{\left(c_{1}\times V_{1} - c_{2}\times V_{2}\right)\times M}{m\times\frac{25}{500}\times\left(1 - \frac{w_{2}}{100}\right)\times 1000}\times 100\% \quad \dots \tag{A.1}
$$

式中：

$c_{1}$ ——高锰酸钾标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{1}$ ——加入和滴定试验溶液所消耗的高锰酸钾标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{2}$ ——草酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{2}$ ——移取草酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$\omega_{2}$ ——按A.5测定的干燥减量的含量；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

M 与 $1.00\mathrm{mL}$ 高锰酸钾标准滴定溶液 $\left[c\left(\frac{1}{5}\mathrm{KMnO}_4\right) = 1.000\mathrm{mol / L}\right]$ 相当的以克表示的亚硝酸钠的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $\left[M\left(\frac{1}{2}\mathrm{NaNO}_3\right) = 34.50\right]_{\circ}$

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于 $0.2\%$ 。

# A.5 干燥减量的测定

# A.5.1 仪器和设备

A.5.1.1 称量瓶： $\Phi 500\mathrm{mm}\times 30\mathrm{mm}$

A.5.1.2 电烘箱：温度能控制在 $105\,^{\circ}\mathrm{C} \sim 110\,^{\circ}\mathrm{C}$ 。

# A.5.2 分析步骤

用预先于 $105\mathrm{^\circ C}\sim 110\mathrm{^\circ C}$ 下干燥的称量瓶称取约 $5\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，于 $105\mathrm{^\circ C}\sim 110\mathrm{^\circ C}$ 电烘箱中干燥至恒重。

# A.5.3 结果计算

干燥减量的质量分数 $\varpi_{2}$ ，按式（A.2）计算：

$$
w _ {2} = \frac {m _ {1} - m _ {2}}{m _ {1}} \times 100 \% \tag{A.2}
$$

式中：

$m_{2}$ ——干燥后试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于 $0.005\%$ 。

# A.6 水不溶物含量（以干基计）的测定

# A.6.1 试剂和材料

A.6.1.1 盐酸。  
A.6.1.2 淀粉-碘化钾试纸。

# A.6.2 仪器和设备

A.6.2.1 坩埚式过滤器：滤板孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$   
A.6.2.2 电烘箱：温度能控制在 $105^{\circ}\mathrm{C}\sim 110^{\circ}\mathrm{C}$

# A.6.3 分析步骤

称取约 $100\mathrm{g}$ 试样，精确至 $0.1\mathrm{g}$ ，置于 $500~\mathrm{mL}$ 烧杯中，加 $300~\mathrm{mL}$ 水，加热溶解。用预先于 $105^{\circ}\mathrm{C} \sim 110^{\circ}\mathrm{C}$ 下干燥的玻璃砂坩埚过滤，用热水洗至无亚硝酸根离子为止（取 $20~\mathrm{mL}$ 洗涤液，加两滴盐酸，用淀粉-碘化钾试纸检查）。置于 $105^{\circ}\mathrm{C} \sim 110^{\circ}\mathrm{C}$ 电烘箱中干燥至恒重。

# A.6.4 结果计算

水不溶物含量(以干基计)的质量分数 $\varpi_{3}$ ，按式(A.3)计算：

$$
w _ {3} = \frac {\left(m _ {3} - m _ {4}\right)}{m _ {5} \times \left(1 - \frac {w _ {2}}{100}\right)} \times 100 \% \tag{A.3}
$$

式中：

$m_{3}$ ——玻璃砂坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{4}$ ——水不溶物和玻璃砂坩埚的质量，单位为克 $(\mathrm{g})$ ；

$w_{2}$ ——按A.5测定的干燥减量的含量；

$m_{5}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于 $0.005\%$ 。

# A.7 铅(Pb)的测定

# A.7.1 方法提要

同GB/T9723—2007第4章。

# A.7.2 试剂和材料

A.7.2.1 盐酸。

A.7.2.2 铅标准溶液: $1 \mathrm{~mL}$ 溶液含有 $0.010 \mathrm{mg}$ 铅。按 GB/T 602 之规定配制后, 用移液管移取 $10 \mathrm{~mL}$ 溶液, 置于 $100 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度, 摇匀。此溶液使用前配制。

# A.7.3 仪器和设备

同GB/T9723—2007第6章。

# A.7.4 分析步骤

# A.7.4.1 工作曲线的绘制

按GB/T9723—2007中7.2的规定进行操作。

在一系列 $50~\mathrm{mL}$ 容量瓶中，分别加入 $0\mathrm{mL},0.5\mathrm{mL},1.0\mathrm{mL},1.5\mathrm{mL},2.0\mathrm{mL},3.0\mathrm{mL}$ 铅标准溶液，用水稀释至刻度，摇匀。

将仪器调整至最佳工作条件，在 $283.3\mathrm{nm}$ 波长下，以水调零，测量各溶液的吸光度。以铅离子浓度为横坐标，对应的吸光度为纵坐标，绘制工作曲线。

# A.7.4.2 测量

称取 $1\mathrm{g}$ 样品，精确至 $0.01\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 容量瓶中，摇匀。与标准溶液同时测定。

# A.7.5 结果计算

铅(Pb)的质量分数 $w_{4}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.4)计算：

$$
w _ {4} = \frac {m _ {6} \times 1 0 0 0}{m _ {7}} \tag {A.4}
$$

式中：

$m_{6}$ ——从标准曲线上查得铅的质量，单位为毫克 $(\mathrm{mg})$ ；

$m_{7}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于 $1\mathrm{mg / kg}$ 。

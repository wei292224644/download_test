# 乳及乳制品中多种有机氯农药残留量的测定

# 气相色谱一质谱/质谱法

# 1 范围

本标准规定了乳及乳制品中多种有机氯农药残留量气相色谱一质谱/质谱法。

本标准适用于液态奶、奶粉、酸奶(半固态)、冰激凌、奶糖等乳及乳制品中 $\alpha$ -六六六、 $\beta$ -六六六、林丹、 $\delta$ -六六六、o,p'-滴滴涕、p,p'-滴滴涕、o,p'-滴滴伊、p,p'-滴滴伊、o,p'-滴滴滴、p,p'-滴滴滴、甲氧滴滴涕、七氯、环氧七氯、艾氏剂、狄氏剂、异狄氏剂醛、异狄氏剂酮、顺式一氯丹、反式一氯丹、氧化氯丹、 $\alpha$ -硫丹、 $\beta$ -硫丹、硫丹硫酸盐、六氯苯、四氯硝基苯、五氯硝基苯、五氯苯胺、甲基五氯苯基硫醚、灭蚁灵30种有机氯农药残留量的测定和确证，其他食品可参照执行。

# 3 原理

试样中的有机氯农药残留用正己烷一丙酮(1+1,体积比)溶液提取，提取液经浓缩后，经凝胶渗透色谱和弗罗里硅土柱净化，用气相色谱一质谱/质谱仪测定和确证，外标峰面积法定量。

# 4 试剂和材料

除另有规定外，所用试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
4.1.2 丙酮 $(\mathrm{C_3H_6O})$ ：色谱纯。  
4.1.3 二氯甲烷 $(\mathrm{C_2H_2Cl_2})$ ：色谱纯。  
4.1.4 环己烷 $(\mathrm{C_6H_{12}})$ 。  
4.1.5 乙酸乙酯 $(\mathrm{C_4H_8O_2})$   
4.1.6 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，在干燥器内冷却至室温，储于密封瓶中备用。  
4.1.7 氯化钠 $(\mathrm{NaCl})$ 。

# 4.2 溶液配制

4.2.1 提取液：取适量正己烷和丙酮，按体积比 $1:1$ 进行混合。  
4.2.2 凝胶渗透色谱洗脱液：取适量环己烷和乙酸乙酯，按体积比 $1:1$ 进行混合。  
4.2.3 固相萃取洗脱液：取适量正己烷和二氯甲烷，按体积比 $5:95$ 进行混合。

# 4.3 标准品

农药标准物质：纯度 $\geq 95\%$

# 4.4 标准溶液配制

4.4.1 标准储备溶液：准确称取适量的各标准物质，用正己烷配制成浓度为 $100\mu \mathrm{g / mL}$ 的标准储备液。此溶液在 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 避光保存。  
4.4.2 标准中间溶液: 取适量的各种标准储备溶液, 配制成 $2 \mu \mathrm{g} / \mathrm{mL}$ 的混合标准工作溶液。此溶液在 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 避光保存。  
4.4.3 标准工作溶液：取适量的各种标准储备溶液，配制成适当浓度的混合标准工作溶液。标准工作液现用现配。

# 4.5 材料

4.5.1 弗罗里硅土固相萃取小柱： $6\mathrm{mL} / 1\mathrm{g}$ ，或相当者。使用前，用 $5\mathrm{mL}$ 正己烷活化。  
4.5.2 微孔滤膜： $0.45\mu \mathrm{m}$ ，有机系。

# 5 仪器和设备

5.1 气相色谱—质谱/质谱仪：配电子轰击源（EI）。  
5.2 凝胶渗透色谱仪。  
5.3 电子天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.4 涡旋振荡器。  
5.5 离心机：最大转速可达 $5000\mathrm{r / min}$   
5.6 旋转蒸发仪。  
5.7 氮吹仪。  
5.8 具塞离心管：聚四氟乙烯， $50~\mathrm{mL}$

# 6 试样制备与保存

液态奶、酸奶、冰激凌：取有代表性样约 $100 \mathrm{~g}$ , 装入洁净容器作为试样, 密封并做好标识, 于 $0^{\circ} \mathrm{C} \sim$ $4^{\circ} \mathrm{C}$ 冰箱内保存。

奶粉、奶糖：取有代表性样约 $100\mathrm{g}$ ，装入洁净容器作为试样，密封并做好标识，于常温下干燥保存。

在制样操作过程中必须防止样品受到污染或发生残留物含量的变化。

样品取样部位按GB2763的规定执行。

# 7 分析步骤

# 7.1 提取

准确称取 $10 \mathrm{~g}$ 试样(精确到 $0.01 \mathrm{~g}$ ) 于 $50 \mathrm{~mL}$ 具塞离心管中(奶粉、奶糖加 $10 \mathrm{~mL}$ 水溶解), 加入 $5 \mathrm{~g}$ 氯化钠, 再加入 $10 \mathrm{~mL}$ 提取液。用涡旋振荡器振荡 $1 \mathrm{~min}, 4000 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ , 将有机相转移至 $100 \mathrm{~mL}$ 旋蒸瓶中。残渣再分别用 $10 \mathrm{~mL}$ 提取液提取 2 次, 离心合并有机相, 在 $40^{\circ} \mathrm{C}$ 下旋转蒸发浓缩至近干。用 $10 \mathrm{~mL}$ 环己烷一乙酸乙酯混合溶液充分溶解残渣, 过 $0.45 \mu \mathrm{m}$ 滤膜, 待净化。

# 7.2 净化

# 7.2.1 将凝胶渗透色谱净化

# 7.2.1.1 参考条件

a）净化柱： $400\mathrm{mm}\times 25\mathrm{mm}$ （内径），内装Bio-Beads，S-X3, $38\mu \mathrm{m}\sim 75\mu \mathrm{m}$ 填料，或性能相当者；  
b）流动相：环己烷—乙酸乙酯 $(1 + 1$ ，体积比）；  
c) 流速: $5 \mathrm{~mL} / \mathrm{min}$ ;   
d) 进样量: $5 \mathrm{~mL}$ ;

e）开始收集时间： $10\mathrm{min}$   
f）结束收集时间： $22\mathrm{min}$

# 7.2.1.2 净化步骤

将待净化溶液转移至 $10\mathrm{mL}$ 试管中，用凝胶渗透色谱仪净化，收集 $10\mathrm{min} \sim 22\mathrm{min}$ 的淋洗液，在 $40^{\circ}\mathrm{C}$ 下减压浓缩至约 $2\mathrm{mL}$ ，待弗罗里硅土固相萃取柱净化。

# 7.2.2 固相萃取净化

将上述样液转移到已活化的弗罗里硅土固相萃取柱内，收集流出液，用 $8\mathrm{mL}$ 二氯甲烷一正己烷溶液洗脱。收集洗脱液于 $40^{\circ}\mathrm{C}$ 旋转蒸发浓缩至近干，用 $1\mathrm{mL}$ 正己烷溶解残渣，过 $0.45\mu \mathrm{m}$ 滤膜，供测定。

# 7.3 测定

# 7.3.1 仪器参考条件

a）色谱柱：TR-35ms， $30\mathrm{m}\times 0.25\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ，或性能相当者；  
b）柱温： $55^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，以 $40^{\circ}\mathrm{C} / \mathrm{min}$ 速率升至 $140^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ ，以 $2^{\circ}\mathrm{C} / \mathrm{min}$ 速率升至 $210^{\circ}\mathrm{C}$ ，以 $10^{\circ}\mathrm{C} / \mathrm{min}$ 速率升至 $280^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ ；  
c）进样口温度： $250^{\circ}\mathrm{C}$   
d) 离子源温度: $250^{\circ} \mathrm{C}$ ;   
e) 传输线温度: $250^{\circ} \mathrm{C}$ ;   
f）离子源：电子轰击离子源；  
g）测定方式：选择反应监测模式(SRM)；  
h) 监测离子 $(m / z)$ : 各种有机氯农药的定性离子对、定量离子对、碰撞能量及离子丰度比参见表 A. 1;  
i）载气：氦气，纯度 $\geqslant 99.999\%$   
j）流速： $1.2\mathrm{mL / min}$   
k）进样方式：不分流；  
1) 进样量: $1 \mu \mathrm{L}$ ;   
m） 电离能量： $70\mathrm{eV}$

# 7.3.2 色谱测定与确证

按照7.3.1气相色谱—质谱/质谱条件测定样液和标准工作溶液，外标法测定样液中的有机氯农药残留量。样品中待测物残留量应在标准曲线范围之内，如果残留量超出标准曲线范围，应进行适当稀释。在上述色谱条件下，各种有机氯质量色谱峰保留时间见表A.1。标准品的总离子流色谱图和多反应监测色谱图见图A.1和图A.2。

# 7.4 空白试验

除不称取试样外，按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用数据处理软件中的外标法或绘制标准曲线，按式(1)计算样品中有机氯农药的残留量。

$$
X _ {i} = \frac {c _ {i} \times V \times 1 0 0 0}{m \times 1 0 0 0}. \tag {1}
$$

式中：

$X_{i}$ ——试样中 $i$ 组分农药的残留量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c_{i}$ 由标准曲线得到的样液中 $i$ 组分农药的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——最终样液所代表的试样质量，单位为克(g)；

1000 ——换算系数。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 B 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 C 的要求。

10 定量限和回收率  
10.1 定量限

本方法中各种有机氯农药的定量限均为 $0.8\mu \mathrm{g} / \mathrm{kg}$

10.2 回收率

方法的平均回收率为 $62.2\% \sim 116.8\%$

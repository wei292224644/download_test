# 高效液相色谱法

# 1 范围

本文件规定了猪和牛的肌肉、脂肪、肝脏、肾脏及牛奶样品中头孢喹肟残留量检测的制样和高效液相色谱测定方法。

本文件适用于猪和牛的肌肉、脂肪、肝脏、肾脏及牛奶样品中头孢喹肟残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中残留的头孢喹衍经磷酸高氯酸钠缓冲液和乙腈提取，亲水亲脂平衡固相萃取柱净化，高效液相色谱测定，外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯；水为符合GB/T6082规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯

5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.3 甲酸（HCOOH）。

5.1.4 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。

5.1.5 正己烷 $(\mathrm{C_6H_{14}})$ 。

5.1.6 正丙醇 $(\mathrm{C}_3\mathrm{H}_8\mathrm{O})$ 。

5.1.7 一水合高氯酸钠 $\mathrm{(NaClO_4\cdot H_2O)}$

5.1.8 三乙胺 $\left[\mathrm{N}\left(\mathrm{CH}_{2} \mathrm{CH}_{3}\right)_{3}\right]$ 。

# 5.2 标准品

头孢喹肟（cefquinome， $\mathrm{C_{23}H_{24}N_6O_5S_2}$ ，CAS号：84957-30-2）：含量 $\geq 82\%$

# 5.3 溶液配制

5.3.1 $50\%$ 乙腈溶液：取乙腈 $50~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。  
5.3.2 $5\%$ 乙腈溶液：取乙腈 $5\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。  
5.3.3 $12\%$ 乙腈溶液：取乙腈 $12~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，混匀。  
5.3.4 高氯酸钠-磷酸缓冲液：取磷酸 $3.4\mathrm{mL}$ ，缓慢加适量水稀释，再加入一水合高氯酸钠 $3.45\mathrm{g}$ ，用三

乙胺调节 $\mathrm{pH}$ 至3.6，用水稀释至 $1000\mathrm{mL}$ ，混匀。

5.3.5 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.3.6 $0.2\%$ 甲酸溶液：取甲酸 $2\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.3.7 组织样品洗脱液：取 $0.1\%$ 甲酸溶液 $88~\mathrm{mL}$ 和乙腈 $12~\mathrm{mL}$ ，混匀。  
5.3.8 牛奶样品洗脱液：取 $0.2\%$ 甲酸溶液 $88~\mathrm{mL}$ 和乙腈 $12~\mathrm{mL}$ ，混匀。

# 5.4 标准溶液配制

5.4.1头孢喹肟标准储备液 $(1\mathrm{mg / mL})$ ：称取头孢喹肟标准品约 $10~\mathrm{mg}$ ，精密称定，用 $50\%$ 乙腈溶液溶解并稀释定容于 $10~\mathrm{mL}$ 量瓶中。配制成浓度为 $1\mathrm{mg / mL}$ 的标准储备液， $4^{\circ}C$ 保存，有效期1个月。  
5.4.2头孢喹肟标准工作液 $(2\mu \mathrm{g / mL})$ ：准确量取标准储备液 $0.1\mathrm{mL}$ 于 $50~\mathrm{mL}$ 量瓶中，用 $50\%$ 乙腈溶液稀释至刻度，配制成浓度为 $2\mu \mathrm{g / mL}$ 的标准工作液。 $4^{\circ}C$ 保存，有效期1个月。

# 5.5 材料

5.5.1 亲水亲脂平衡型固相萃取柱： $60~\mathrm{mg / 3~mL}$ ，或相当者。  
5.5.2 有机微孔滤膜: $0.45 \mu \mathrm{m}$ 。

# 6 仪器设备

6.1 高效液相色谱仪：配紫外检测器或二极管阵列检测器。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
6.3 振荡器。  
6.4 离心机：转速 $\geqslant 10000\mathrm{r / min}$ 。   
6.5 旋转蒸发仪。  
6.6 氮吹仪。  
6.7 茄形瓶。  
6.8 涡旋混合器。  
6.9 玻璃试管： $10~\mathrm{mL}$

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

取适量新鲜或解冻的空白或供试奶样品，混合均匀。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后空白样品，添加适宜浓度的对照溶液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取肝脏、肾脏试料 $(2 \pm 0.05) \mathrm{g}$ , 肌肉、脂肪试料 $(4 \pm 0.05) \mathrm{g}$ , 牛奶 $(5 \pm 0.05) \mathrm{g}$ 于 $50 \mathrm{~mL}$ 离心管中, 加入高氯酸钠-磷酸缓冲液 $2 \mathrm{~mL}$ , 乙腈 $10 \mathrm{~mL}$ , 涡旋混匀, 振荡 $5 \mathrm{~min}, 5000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 取上清液于茄形瓶中, 再按上述步骤重复提取 1 次, 合并 2 次上清液, 加正丙醇 $5 \mathrm{~mL}$ , 在 $40^{\circ} \mathrm{C}$ 下旋转蒸发至近干。残渣中加水 $10 \mathrm{~mL}$ 使溶解, 再加正己烷 $5 \mathrm{~mL}$ , 涡旋混匀, 转移至 $50 \mathrm{~mL}$ 离心管中, $4^{\circ} \mathrm{C}$ 下 $10000 \mathrm{r} / \mathrm{min}$ 离心 $8 \mathrm{~min}$ , 取下层清液备用。

# 8.2 净化

固相萃取柱依次用甲醇 $2\mathrm{mL}$ 和水 $2\mathrm{mL}$ 活化，备用液全部过柱，再依次用水 $3\mathrm{mL}$ 和 $5\%$ 乙腈溶液 $3\mathrm{mL}$ 洗涤，抽干后用洗脱液 $2.0\mathrm{mL}$ 洗脱，收集于 $10\mathrm{mL}$ 玻璃试管中，混匀后过 $0.45\mu \mathrm{m}$ 微孔滤膜，供高效液相色谱仪测定。

# 8.3 标准曲线的制备

精密量取 $2\mu \mathrm{g} / \mathrm{mL}$ 头孢喹肟标准工作液适量，用流动相稀释并制成浓度为 $10~\mathrm{ng / mL},25~\mathrm{ng / mL}$ $50~\mathrm{ng / mL},100~\mathrm{ng / mL},200~\mathrm{ng / mL},500~\mathrm{ng / mL}$ 的系列标准工作液，供高效液相色谱测定。以测得峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.4 测定

# 8.4.1 色谱条件

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱（ $250\mathrm{mm}\times 4.6\mathrm{mm}$ ，粒径 $3\mu \mathrm{m})$ ，或相当者；  
b）流动相：组织样品为 $0.1\%$ 甲酸溶液：乙腈 $(88:12,V / V)$ ，牛奶样品为 $0.2\%$ 甲酸溶液：乙腈 $(88:12,V / V)$   
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d） 检测波长： $270~\mathrm{nm}$   
e） 柱温： $35^{\circ}C$   
f） 进样量： $50\mu \mathrm{L}$

# 8.4.2 测定法

取试料溶液和相应的标准溶液，做单点或多点校准，以头孢喹肟的保留时间定性，被测试样中头孢喹肟色谱峰保留时间与相应标准溶液色谱峰的保留时间相比，相差应在 $\pm 0.1\mathrm{min}$ 内。按外标法以峰面积计算。标准溶液及试料溶液中头孢喹肟响应值应在仪器检测的线性范围之内。在上述色谱条件下，头孢喹肟标准溶液的高效液相色谱图见附录A。

# 8.5 空白试验

取空白试样，除不加标准溶液外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中头孢唑肟的残留量按标准曲线或公式(1)计算

$$
X = \frac {A \times C _ {\mathrm {s}} \times V - 1 0 0 0}{A _ {\mathrm {s}} \times m ^ {2} \times 1 0 0 0}. \tag {1}
$$

式中：

$X$ ——试样中相应的头孢唑肟残留量的数值；单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

A ——试料溶液中相应的头孢喹肟色谱峰面积；

$C_{\mathrm{s}}$ ——标准溶液中相应的头孢唑肟浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

V 洗脱溶液体积的数值，单位为毫升 $(\mathrm{mL})$

$A_{s}$ ——标准溶液中相应的头孢喹肟色谱峰面积；

$m$ —供试试料质量的数值，单位为克 $(\mathrm{g})$

1000——换算系数。

# 10 检测方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法在猪、牛的肌肉、脂肪组织中的检出限为 $15\mu \mathrm{g / kg}$ ，定量限为 $25~\mu \mathrm{g / kg}$ ；在猪、牛的肝脏、肾脏组织中的检出限为 $25~\mu \mathrm{g / kg}$ ，定量限为 $50~\mu \mathrm{g / kg}$ ；在牛奶中的检出限为 $5\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法在猪、牛肌肉、脂肪组织 $25\mu \mathrm{g / kg}\sim 100\mu \mathrm{g / kg}$ 添加浓度的回收率为 $60\% \sim 100\%$ ；在猪、牛肝

脏、肾脏组织 $50~\mu \mathrm{g / kg}\sim 400~\mu \mathrm{g / kg}$ 添加浓度的回收率为 $60\% \sim 100\%$ ；在牛奶 $10~\mu \mathrm{g / kg}\sim 40~\mu \mathrm{g / kg}$ 添加浓度的回收率为 $60\% \sim 100\%$ 。

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

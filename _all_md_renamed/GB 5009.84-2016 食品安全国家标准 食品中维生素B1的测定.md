# 前言

本标准代替GB/T5009.84—2003《食品中硫胺素(维生素 $\mathbf{B}_1$ )的测定》、GB5413.11—2010《食品安全国家标准婴幼儿食品和乳品中维生素 $\mathbf{B}_1$ 的测定》、GB/T7628—2008《谷物中维生素 $\mathbf{B}_1$ 的测定》、GB/T9695.27—2008《肉与肉制品维生素 $\mathbf{B}_1$ 含量测定》。

本标准与GB/T5009.84—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中维生素 $\mathrm{B}_{1}$ 的测定”；  
——增加了高效液相色谱法，作为第一法，将荧光光度法作为第二法；  
——修改了检出限的表达，增加了方法的定量限；  
——增加了人造沸石预处理中氯离子的定性鉴别方法；  
——增加了溴甲酚绿为指示剂时，溶液颜色的变化特征；  
——删去了图1（反应瓶)和图2（盐基交换管)结构图；  
——增加了人造沸石以湿重表示时的称取量。

# 食品安全国家标准

# 食品中维生素 $\mathbf{B}_{1}$ 的测定

# 1 范围

本标准规定了高效液相色谱法、荧光光度法测定食品中维生素 $\mathrm{B}_{1}$ 的方法。

本标准适用于食品中维生素 $\mathrm{B}_{1}$ 含量的测定。

# 第一法 高效液相色谱法

# 2 原理

样品在稀盐酸介质中恒温水解、中和，再酶解，水解液用碱性铁氰化钾溶液衍生，正丁醇萃取后，经 $\mathrm{C_{18}}$ 反相色谱柱分离，用高效液相色谱-荧光检测器检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 正丁醇 $\mathrm{(CH_3CH_2CH_2CH_2OH)}$   
3.1.2 铁氰化钾 $\left[\mathrm{K}_{3} \mathrm{Fe}(\mathrm{CN})_{6}\right]$ 。  
3.1.3 氢氧化钠(NaOH)。  
3.1.4 盐酸（HCl）。  
3.1.5 乙酸钠 $\mathrm{(CH_3COONa\cdot 3H_2O)}$   
3.1.6 冰乙酸 $\mathrm{(CH_3COOH)}$   
3.1.7 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.8 五氧化二磷 $\left(\mathrm{P}_{2} \mathrm{O}_{5}\right)$ 或者氯化钙 $\left(\mathrm{CaCl}_{2}\right)$ 。  
3.1.9 木瓜蛋白酶：应不含维生素 $\mathrm{B}_{1}$ ，酶活力 $\geqslant 800\mathrm{U}$ （活力单位）/ $\mathrm{mg}$ 。  
3.1.10 淀粉酶：应不含维生素 $\mathrm{B}_{1}$ ，酶活力 $\geqslant 3700\mathrm{U / g}$ 。

# 3.2 试剂配制

3.2.1 铁氰化钾溶液 $(20\mathrm{g} / \mathrm{L})$ ：称取 $2\mathrm{g}$ 铁氰化钾，用水溶解并定容至 $100~\mathrm{mL}$ ，摇匀。临用前配制。  
3.2.2 氢氧化钠溶液 $(100\mathrm{g / L})$ ：称取 $25\mathrm{g}$ 氢氧化钠，用水溶解并定容至 $250~\mathrm{mL}$ ，摇匀。  
3.2.3 碱性铁氰化钾溶液：将 $5\mathrm{mL}$ 铁氰化钾溶液与 $200~\mathrm{mL}$ 氢氧化钠溶液混合，摇匀。临用前配制。  
3.2.4 盐酸溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：移取 $8.5\mathrm{mL}$ 盐酸，加水稀释至 $1000~\mathrm{mL}$ ，摇匀。  
3.2.5 盐酸溶液 $(0.01\mathrm{mol} / \mathrm{L})$ ：量取 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液 $50~\mathrm{mL}$ ，用水稀释并定容至 $500~\mathrm{mL}$ ，摇匀。  
3.2.6 乙酸钠溶液 $(0.05\mathrm{mol} / \mathrm{L})$ : 称取 $6.80\mathrm{g}$ 乙酸钠, 加 $900\mathrm{mL}$ 水溶解, 用冰乙酸调 $\mathrm{pH}$ 为 $4.0 \sim 5.0$ 之间, 加水定容至 $1000\mathrm{mL}$ 。经 $0.45\mu \mathrm{m}$ 微孔滤膜过滤后使用。

3.2.7 乙酸钠溶液 $(2.0\mathrm{mol} / \mathrm{L})$ ：称取 $27.2\mathrm{g}$ 乙酸钠，用水溶解并定容至 $100~\mathrm{mL}$ ，摇匀。  
3.2.8 混合酶溶液：称取 $1.76\mathrm{g}$ 木瓜蛋白酶、 $1.27\mathrm{g}$ 淀粉酶，加水定容至 $50~\mathrm{mL}$ ，涡旋，使呈混悬状液体，冷藏保存。临用前再次摇匀后使用。

# 3.3 标准品

维生素 $\mathrm{B}_{1}$ 标准品：盐酸硫胺素（ $\mathrm{C_{12}H_{17}ClN_4OS\cdot HCl)}$ ），CAS：67-03-8，纯度 $\geq 99.0\%$

# 3.4 标准溶液配制

3.4.1 维生素 $\mathrm{B}_{1}$ 标准储备液 $(500\mu \mathrm{g} / \mathrm{mL})$ ：准确称取经五氧化二磷或者氯化钙干燥 $24\mathrm{h}$ 的盐酸硫胺素标准品 $56.1\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ )，相当于 $50\mathrm{mg}$ 硫胺素、用 $0.01\mathrm{mol} / \mathrm{L}$ 盐酸溶液溶解并定容至 $100\mathrm{mL}$ ，摇匀。置于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱中，保存期为3个月。  
3.4.2 维生素 $\mathrm{B}_{1}$ 标准中间液（ $10.0\mu \mathrm{g} / \mathrm{mL}$ ）：准确移取 $2.00\mathrm{mL}$ 标准储备液，用水稀释并定容至 $100\mathrm{mL}$ ，摇匀。临用前配制。  
3.4.3 维生素 $\mathrm{B}_{1}$ 标准系列工作液：吸取维生素 $\mathrm{B}_{1}$ 标准中间液 $0\mu \mathrm{L},50.0\mu \mathrm{L},100\mu \mathrm{L},200\mu \mathrm{L},400\mu \mathrm{L}$ $800~\mu \mathrm{L},1~000~\mu \mathrm{L}$ ，用水定容至 $10~\mathrm{mL}$ ，标准系列工作液中维生素 $\mathrm{B}_{1}$ 的浓度分别为 $0\mu \mathrm{g / mL},0.0500\mu \mathrm{g / mL}$ $0.100~\mu \mathrm{g / mL},0.200~\mu \mathrm{g / mL},0.400\mu \mathrm{g / mL},0.800~\mu \mathrm{g / mL},1.00~\mu \mathrm{g / mL}$ 。临用时配制。

# 4 仪器和设备

4.1 高效液相色谱仪，配置荧光检测器。  
4.2 分析天平：感量为 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$   
4.3 离心机：转速 $\geqslant 4000\mathrm{r / min}$   
4.4 pH计：精度0.01。  
4.5 组织捣碎机（最大转速不低于 $10000\mathrm{r / min}$ ）。  
4.6 电热恒温干燥箱或高压灭菌锅。

# 5 分析步骤

# 5.1 试样的制备

5.1.1 液体或固体粉末样品：将样品混合均匀后，立即测定或于冰箱中冷藏。  
5.1.2 新鲜水果、蔬菜和肉类：取 $500\mathrm{g}$ 左右样品（肉类取 $250\mathrm{g}$ ），用匀浆机或者粉碎机将样品均质后，制得均匀性一致的匀浆，立即测定或者于冰箱中冷冻保存。  
5.1.3 其他含水量较低的固体样品：如含水量在 $15\%$ 左右的谷物，取 $100\mathrm{g}$ 左右样品，用粉碎机将样品粉碎后，制得均匀性一致的粉末，立即测定或者于冰箱中冷藏保存。

# 5.2 试样溶液的制备

# 5.2.1 试液提取

称取 $3\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )固体试样或者 $10\mathrm{g}\sim 20\mathrm{g}$ 液体试样于 $100~\mathrm{mL}$ 锥形瓶中（带有软质塞子），加 $60~\mathrm{mL}~0.1~\mathrm{mol / L}$ 盐酸溶液，充分摇匀，塞上软质塞子，高压灭菌锅中 $121^{\circ}\mathrm{C}$ 保持 $30\mathrm{min}$ 。水解结束待冷却至 $40^{\circ}\mathrm{C}$ 以下取出，轻摇数次；用 $\mathsf{pH}$ 计指示，用 $2.0\mathrm{mol / L}$ 乙酸钠溶液调节 $\mathsf{pH}$ 至4.0左右，加入 $2.0~\mathrm{mL}$ (可根据酶活力不同适当调整用量)混合酶溶液，摇匀后，置于培养箱中 $37^{\circ}\mathrm{C}$ 过夜(约 $16\mathrm{h})$ ；将酶解液全部转移至 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，摇匀，离心或者过滤，取上清液备用。

# 5.2.2 试液衍生化

准确移取上述上清液或者滤液 $2.0\mathrm{mL}$ 于 $10\mathrm{mL}$ 试管中，加入 $1.0\mathrm{mL}$ 碱性铁氰化钾溶液，涡旋混匀后，准确加入 $2.0\mathrm{mL}$ 正丁醇，再次涡旋混匀 $1.5\mathrm{min}$ 后静置约 $10\mathrm{min}$ 或者离心，待充分分层后，吸取正丁醇相(上层)经 $0.45\mu \mathrm{m}$ 有机微孔滤膜过滤，取滤液于 $2\mathrm{mL}$ 棕色进样瓶中，供分析用。若试液中维生素 $\mathrm{B}_{1}$ 浓度超出线性范围的最高浓度值，应取上清液稀释适宜倍数后，重新衍生后进样。

另取 $2.0\mathrm{mL}$ 标准系列工作液，与试液同步进行衍生化。

注1：室温条件下衍生产物在 $4\mathrm{h}$ 内稳定。

注2：5.2.1和5.2.2操作过程应在避免强光照射的环境下进行。

注3：辣椒干等样品，提取液直接衍生后测定时，维生素 $\mathbf{B}_1$ 的回收率偏低。提取液经人造沸石净化后，再衍生时维生素 $\mathbf{B}_1$ 的回收率满足要求。故对于个别特殊样品，当回收率偏低时，样品提取液应净化后再衍生，具体操作步骤见第二法12.1.3部分。

# 5.3 仪器参考条件

5.3.1 色谱柱： $\mathrm{C_{18}}$ 反相色谱柱（粒径 $5\mu \mathrm{m}$ ， $250\mathrm{mm} \times 4.6\mathrm{mm}$ ）或相当者。  
5.3.2 流动相： $0.05\mathrm{mol / L}$ 乙酸钠溶液-甲醇 $(65 + 35)$ 。  
5.3.3 流速： $0.8\mathrm{mL / min}$   
5.3.4 检测波长：激发波长 $375\mathrm{nm}$ ，发射波长 $435\mathrm{nm}$ 。  
5.3.5 进样量： $20~\mu \mathrm{L}$

# 5.4 标准曲线的制作

将标准系列工作液衍生物注入高效液相色谱仪中，测定相应的维生素 $\mathrm{B}_{1}$ 峰面积，以标准工作液的浓度 $(\mu \mathrm{g} / \mathrm{mL})$ 为横坐标，以峰面积为纵坐标，绘制标准曲线。

# 5.5 试样溶液的测定

按照5.3的色谱条件，将试样衍生物溶液注入高效液相色谱仪中，得到维生素 $\mathrm{B}_{1}$ 的峰面积，根据标准曲线计算得到待测液中维生素 $\mathrm{B}_{1}$ 的浓度。

# 6 分析结果的表述

试样中维生素 $\mathrm{B}_{1}$ （以硫胺素计）含量按式(1)计算：

$$
X = \frac {c \times V \times f}{m \times 1 0 0 0} \times 1 0 0 \tag {1}
$$

式中：

$X$ ——试样中维生素 $\mathrm{B}_{1}$ (以硫胺素计)的含量, 单位为毫克每百克 $(\mathrm{mg} / 100 \mathrm{~g})$ ;

$c$ ——由标准曲线计算得到的试液(提取液)中维生素 $\mathrm{B}_{1}$ 的浓度, 单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$

$V$ ——试液(提取液)的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试液（上清液）衍生前的稀释倍数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

注：试样中测定的硫胺素含量乘以换算系数1.121，即得盐酸硫胺素的含量。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当称样量为 $10.0\mathrm{g}$ 时，按照本标准方法的定容体积，食品中维生素 $\mathrm{B}_{1}$ 的检出限为 $0.03\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.10\mathrm{mg} / 100\mathrm{g}$ 。

# 第二法 荧光分光光度法

# 9 原理

硫胺素在碱性铁氰化钾溶液中被氧化成噻嘧色素，在紫外线照射下，噻嘧色素发出荧光。在给定的条件下，以及没有其他荧光物质干扰时，此荧光之强度与噻嘧色素量成正比，即与溶液中硫胺素量成正比。如试样中含杂质过多，应经过离子交换剂处理，使硫胺素与杂质分离，然后以所得溶液用于测定。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 10.1 试剂

10.1.1 正丁醇 $\mathrm{(CH_3CH_2CH_2CH_2OH)}$   
10.1.2 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $560^{\circ}C$ 烘烤 $6\mathrm{h}$ 后使用。  
10.1.3 铁氰化钾 $\left[\mathrm{K}_{3} \mathrm{Fe}(\mathrm{CN})_{6}\right]$ 。  
10.1.4 氢氧化钠(NaOH)。  
10.1.5 盐酸（HCl）。  
10.1.6 乙酸钠 $\mathrm{(CH_3COONa\cdot 3H_2O)}$   
10.1.7 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
10.1.8 人造沸石。  
10.1.9 硝酸银 $(\mathrm{AgNO}_3)$ 。  
10.1.10 溴甲酚绿 $(\mathrm{C}_{21}\mathrm{H}_{14}\mathrm{Br}_4\mathrm{O}_5\mathrm{S})$ 。  
10.1.11 五氧化二磷 $\left(\mathrm{P}_{2} \mathrm{O}_{5}\right)$ 或者氯化钙 $\left(\mathrm{CaCl}_{2}\right)$ 。  
10.1.12 氯化钾(KCl)。  
10.1.13 淀粉酶：不含维生素 $\mathrm{B}_{1}$ ，酶活力 $\geqslant 3700\mathrm{U / g}$ 。  
10.1.14 木瓜蛋白酶：不含维生素 $\mathrm{B}_{1}$ ，酶活力 $\geqslant 800\mathrm{U}$ （活力单位）/ $\mathrm{mg}$ 。

# 10.2 试剂配制

10.2.1 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液：移取 $8.5\mathrm{mL}$ 盐酸，用水稀释并定容至 $1000~\mathrm{mL}$ ，摇匀。  
10.2.2 $0.01\mathrm{mol} / \mathrm{L}$ 盐酸溶液：量取 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液 $50~\mathrm{mL}$ ，用水稀释并定容至 $500~\mathrm{mL}$ ，摇匀。  
10.2.3 $2\mathrm{mol} / \mathrm{L}$ 乙酸钠溶液：称取 $272\mathrm{g}$ 乙酸钠，用水溶解并定容至 $1000~\mathrm{mL}$ ，摇匀。  
10.2.4 混合酶液：配制同3.2.8。  
10.2.5 氯化钾溶液 $(250\mathrm{g / L})$ ：称取 $250\mathrm{g}$ 氯化钾，用水溶解并定容至 $1000~\mathrm{mL}$ ，摇匀。  
10.2.6 酸性氯化钾 $(250\mathrm{g / L})$ ：移取 $8.5\mathrm{mL}$ 盐酸，用 $250~\mathrm{g / L}$ 氯化钾溶液稀释并定容至 $1000~\mathrm{mL}$ ，摇匀。

10.2.7 氢氧化钠溶液 $(150\mathrm{g / L})$ ：称取 $150\mathrm{g}$ 氢氧化钠，用水溶解并定容至 $1000~\mathrm{mL}$ ，摇匀。  
10.2.8 铁氰化钾溶液 $(10\mathrm{g / L})$ ：称取 $1\mathrm{g}$ 铁氰化钾，用水溶解并定容至 $100~\mathrm{mL}$ ，摇匀，于棕色瓶内保存。  
10.2.9 碱性铁氰化钾溶液：移取 $4\mathrm{mL}10\mathrm{g / L}$ 铁氰化钾溶液(10.2.8)，用 $150~\mathrm{g / L}$ 氢氧化钠溶液稀释至 $60~\mathrm{mL}$ ，摇匀。用时现配，避光使用。  
10.2.10 乙酸溶液：量取 $30~\mathrm{mL}$ 冰乙酸，用水稀释并定容至 $1000~\mathrm{mL}$ ，摇匀。  
10.2.11 $0.01\mathrm{mol} / \mathrm{L}$ 硝酸银溶液：称取 $0.17\mathrm{g}$ 硝酸银，用 $100~\mathrm{mL}$ 水溶解后，于棕色瓶中保存。  
10.2.12 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：称取 $0.4\mathrm{g}$ 氢氧化钠，用水溶解并定容至 $100~\mathrm{mL}$ ，摇匀。  
10.2.13 溴甲酚绿溶液 $(0.4\mathrm{g} / \mathrm{L})$ ：称取 $0.1\mathrm{g}$ 溴甲酚绿，置于小研钵中，加入 $1.4\mathrm{mL}0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液研磨片刻，再加入少许水继续研磨至完全溶解，用水稀释至 $250~\mathrm{mL}$ 。  
10.2.14 活性人造沸石：称取 $200\mathrm{g}0.25\mathrm{mm}$ (40目) $\sim 0.42\mathrm{mm}$ (60目)的人造沸石于 $2000\mathrm{mL}$ 试剂瓶中，加入10倍于其体积的接近沸腾的热乙酸溶液，振荡 $10\mathrm{min}$ ，静置后，弃去上清液，再加入热乙酸溶液，重复一次；再加入5倍于其体积的接近沸腾的热 $250\mathrm{g / L}$ 氯化钾溶液，振荡 $15\mathrm{min}$ ，倒出上清液；再加入乙酸溶液，振荡 $10\mathrm{min}$ ，倒出上清液；反复洗涤，最后用水洗直至不含氯离子。

氯离子的定性鉴别方法：取 $1\mathrm{mL}$ 上述上清液(洗涤液)于 $5\mathrm{mL}$ 试管中，加入几滴 $0.01\mathrm{mol} / \mathrm{L}$ 硝酸银溶液，振荡，观察是否有浑浊产生，如果有浑浊说明还含有氯离子，继续用水洗涤，直至不含氯离子为止。将此活性人造沸石于水中冷藏保存备用。使用时，倒入适量于铺有滤纸的漏斗中，沥干水后称取约 $8.0\mathrm{g}$ 倒入充满水的层析柱中。

# 10.3 标准品

盐酸硫胺素 $\mathrm{(C_{12}H_{17}ClN_4OS\cdot HCl)}$ ，CAS：67-03-8，纯度 $\geqslant 99.0\%$

# 10.4 标准溶液配制

10.4.1 维生素 $\mathrm{B}_{1}$ 标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：准确称取经氯化钙或者五氧化二磷干燥 $24\mathrm{h}$ 的盐酸硫胺素 $112.1\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ，相当于硫胺素为 $100\mathrm{mg}$ ，用 $0.01\mathrm{mol} / \mathrm{L}$ 盐酸溶液溶解，并稀释至 $1000~\mathrm{mL}$ ，摇匀。于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱避光保存，保存期为3个月。  
10.4.2 维生素 $\mathrm{B}_{1}$ 标准中间液（ $10.0\mu \mathrm{g} / \mathrm{mL}$ ）：将标准储备液用 $0.01\mathrm{mol} / \mathrm{L}$ 盐酸溶液稀释10倍，摇匀，在冰箱中避光保存。  
10.4.3 维生素 $\mathrm{B}_{1}$ 标准使用液（ $0.100\mu \mathrm{g} / \mathrm{mL}$ ）：准确移取维生素 $\mathrm{B}_{1}$ 标准中间液 $1.00\mathrm{mL}$ ，用水稀释、定容至 $100\mathrm{mL}$ ，摇匀。临用前配制。

# 11 仪器和设备

11.1 荧光分光光度计。  
11.2 离心机：转速 $\geqslant 4000\mathrm{r / min}$   
11.3 pH计：精度0.01。  
11.4 电热恒温箱。  
11.5 盐基交换管或层析柱 $(60\mathrm{mL},300\mathrm{mm}\times 10\mathrm{mm}$ 内径）。  
11.6 天平：感量为 $0.01\mathrm{g}$ 和 $0.01\mathrm{mg}$ 。

# 12 分析步骤

# 12.1 试样制备

# 12.1.1 试样预处理

用匀浆机将样品均质成匀浆，于冰箱中冷冻保存，用时将其解冻混匀使用。干燥试样取不少于 $150\mathrm{g}$ ，将其全部充分粉碎后备用。

# 12.1.2 提取

准确称取适量试样（估计其硫胺素含量约为 $10\mu \mathrm{g}\sim 30\mu \mathrm{g}$ ，一般称取 $2\mathrm{g}\sim 10\mathrm{g}$ 试样)，置于 $100~\mathrm{mL}$ 锥形瓶中，加入 $50~\mathrm{mL}0.1~\mathrm{mol / L}$ 盐酸溶液，使得样品分散开，将样品放入恒温箱中于 $121^{\circ}C$ 水解 $30\mathrm{min}$ ，结束后，凉至室温后取出。用 $2\mathrm{mol / L}$ 乙酸钠溶液调 $\mathrm{pH}$ 为 $4.0\sim 5.0$ 或者用 $0.4\mathrm{g / L}$ 溴甲酚绿溶液为指示剂，滴定至溶液由黄色转变为蓝绿色。

酶解：于水解液中加入 $2\mathrm{mL}$ 混合酶液，于 $45~^\circ \mathrm{C}\sim 50~^\circ \mathrm{C}$ 温箱中保温过夜(16h)。待溶液凉至室温后，转移至 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀、过滤，即得提取液。

# 12.1.3 净化

装柱：根据待测样品的数量，取适量处理好的活性人造沸石，经滤纸过滤后，放在烧杯中。用少许脱脂棉铺于盐基交换管柱(或层析柱)的底部，加水将棉纤维中的气泡排出，关闭柱塞，加入约 $20~\mathrm{mL}$ 水，再加入约 $8.0~\mathrm{g}$ （以湿重计，相当于干重 $1.0~\mathrm{g} \sim 1.2~\mathrm{g}$ ) 经预先处理的活性人造沸石，要求保持盐基交换管中液面始终高过活性人造沸石。活性人造沸石柱床的高度对维生素 $\mathbf{B}_{1}$ 测定结果有影响，高度不低于 $45~\mathrm{mm}$ 。

样品提取液的净化：准确加入 $20~\mathrm{mL}$ 上述提取液于上述盐基交换管柱(或层析柱)中，使通过活性人造沸石的硫胺素总量约为 $2\mu \mathrm{g}\sim 5\mu \mathrm{g}$ ，流速约为1滴/s。加入 $10~\mathrm{mL}$ 近沸腾的热水冲洗盐基交换柱，流速约为1滴/s，弃去淋洗液，如此重复三次。于交换管下放置 $25~\mathrm{mL}$ 刻度试管用于收集洗脱液，分两次加入 $20~\mathrm{mL}$ 温度约为 $90^{\circ}C$ 的酸性氯化钾溶液，每次 $10~\mathrm{mL}$ ，流速为1滴/s。待洗脱液凉至室温后，用 $250~\mathrm{g / L}$ 酸性氯化钾定容，摇匀，即为试样净化液。

标准溶液的处理：重复上述操作，取 $20~\mathrm{mL}$ 维生素 $\mathrm{B}_{1}$ 标准使用液 $(0.1\mu \mathrm{g / mL})$ 代替试样提取液，同上用盐基交换管(或层析柱)净化，即得到标准净化液。

# 12.1.4 氧化

将 $5\mathrm{mL}$ 试样净化液分别加入A、B两支已标记的 $50~\mathrm{mL}$ 离心管中。在避光条件下将 $3\mathrm{mL}150\mathrm{g / L}$ 氢氧化钠溶液加入离心管A，将 $3\mathrm{mL}$ 碱性铁氰化钾溶液(10.2.9)加入离心管B，涡旋 $15\mathrm{s}$ ；然后各加入 $10~\mathrm{mL}$ 正丁醇，将A、B管同时涡旋 $90~\mathrm{s}$ 。静置分层后吸取上层有机相于另一套离心管中，加入 $2\mathrm{g}\sim 3\mathrm{g}$ 无水硫酸钠，涡旋 $20~\mathrm{s}$ ，使溶液充分脱水，待测定。

用标准的净化液代替试样净化液重复12.1.4的操作。

# 12.2 测定

# 12.2.1 荧光测定条件

激发波长： $365~\mathrm{nm}$ ；发射波长： $435~\mathrm{nm}$ ；狭缝宽度： $5\mathrm{nm}$

# 12.2.2 依次测定下列荧光强度

a）试样空白荧光强度（试样反应管A）；  
b）标准空白荧光强度（标准反应管A）；  
c）试样荧光强度（试样反应管B)；  
d）标准荧光强度（标准反应管B）。

# 13 分析结果的表述

试样中维生素 $\mathrm{B}_{1}$ （以硫胺素计）的含量按式(2)计算：

$$
X = \frac {\left(U - U _ {\mathrm {b}}\right) \times c \times V}{\left(S - S _ {\mathrm {b}}\right)} \times \frac {V _ {1} \times f}{V _ {2} \times m} \times \frac {1 0 0}{1 0 0 0} \quad \dots \tag {2}
$$

式中：

$X$ ——试样中维生素 $\mathrm{B}_{1}$ （以硫胺素计）的含量，单位为毫克每 $100\mathrm{克}(\mathrm{mg} / 100\mathrm{g})$ 。

$U$ ——试样荧光强度；

$U_{\mathrm{b}}$ ——试样空白荧光强度；

S ——标准管荧光强度；

$S_{\mathrm{b}}$ ——标准管空白荧光强度；

$c$ ——硫胺素标准使用液的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——用于净化的硫胺素标准使用液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——试样水解后定容得到的提取液之体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试样用于净化的提取液体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样提取液的稀释倍数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

注：试样中测定的硫胺素含量乘以换算系数1.121，即得盐酸硫胺素的含量。

维生素 $\mathrm{B}_{1}$ 标准在 $0.2\mu \mathrm{g} \sim 10\mu \mathrm{g}$ 之间呈线性关系, 可以用单点法计算结果, 否则用标准工作曲线法。以重复性条件下获得的两次独立测定结果的算术平均值表示, 结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

检出限为 $0.04\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.12\mathrm{mg} / 100\mathrm{g}$ 。

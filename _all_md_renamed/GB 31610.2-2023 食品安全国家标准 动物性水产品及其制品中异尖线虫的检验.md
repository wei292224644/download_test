# 食品安全国家标准

# 动物性水产品及其制品中异尖线虫的检验

# 1 范围

本标准规定了动物性水产品及其制品中异尖线虫(Anisakidae)幼虫的形态学和PCR检验方法。  
本标准适用于动物性水产品及其制品中异尖线虫幼虫的检验。

# 2 原理

动物性水产品及其制品中的异尖线虫幼虫主要寄生于海洋鱼类、甲壳类、软体类动物的肌肉和脏器表面等组织内。解剖分离或用胃蛋白酶消化动物性水产品及其制品的肌肉或脏器等组织获得虫体，根据幼虫的形态特征，判断异尖线虫科幼虫。对可疑虫体通过扩增异尖线虫核糖体DNA内转录间隔区(ITS)基因片段并测序，进行常见异尖线虫幼虫的鉴定。

# 3 仪器和设备

3.1 生物显微镜： $100\times \sim 400\times$   
3.2 体视显微镜： $7.5\times \sim 150\times$   
3.3 PCR扩增仪。  
3.4 凝胶成像系统。  
3.5 电泳仪。  
3.6 恒温培养箱： $37^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 。  
3.7 高速离心机：转速 $\geqslant 12000\mathrm{r / min}$   
3.8 网筛：孔径 $2\mathrm{mm}$ (10目)。  
3.9 锥形量杯： $1000~\mathrm{mL}$   
3.10 微量移液器： $0.2\mu \mathrm{L}\sim 2.5\mu \mathrm{L},1\mu \mathrm{L}\sim 10\mu \mathrm{L},10\mu \mathrm{L}\sim 100\mu \mathrm{L},100\mu \mathrm{L}\sim 1000\mu \mathrm{L}$

# 4 试剂和材料

# 4.1 试剂

4.1.1 盐酸： $36\% \sim 38\%$ HCl溶液。  
4.1.2 生理盐水： $0.85\%$ NaCl溶液。  
4.1.3 $1\mathrm{mol} / \mathrm{L}$ Tris-HCl溶液 $(\mathrm{pH}8.0)$   
4.1.4 $0.5\mathrm{mol} / \mathrm{L}$ EDTA溶液 $(\mathrm{pH}8.0)$ 。   
4.1.5 $10\%$ SDS溶液。  
4.1.6 $5\mathrm{mol} / \mathrm{L}$ NaCl溶液。  
4.1.7 $3000\mathrm{U / mg}$ 胃蛋白酶。  
4.1.8 $20\mathrm{mg / mL}$ 蛋白酶K。

4.1.9 苯酚/三氯甲烷/异戊醇 $(25:24:1)$ 。  
4.1.10 $5\mathrm{U} / \mu \mathrm{L}$ 耐热DNA聚合酶。  
4.1.11 $10 \times \mathrm{PCR}$ 缓冲液。  
4.1.12 $25\mathrm{mmol / L}$ $\mathrm{MgCl_2}$ 。  
4.1.13 dNTPs:dATP、dTTP、dCTP、dGTP，每种浓度为 $2.5\mathrm{mmol / L}$   
4.1.14 琼脂糖：电泳级。  
4.1.15 $50 \times \mathrm{TAE}$ 缓冲液：使用前用去离子水稀释成 $1 \times \mathrm{TAE}$ 缓冲液。  
4.1.16 $1\times \mathrm{TE}$ 溶液 $(\mathrm{pH}8.0)$   
4.1.17 $10\mathrm{mg / mL}$ 溴化乙锭（EB）或其他核酸染料。  
4.1.18 $6\times$ 上样缓冲液。  
4.1.19 $100\mathrm{bp}\sim 2000\mathrm{bp}$ DNA分子量标准。  
4.1.20 PCR引物：浓度为 $20\mu \mathrm{mol} / \mathrm{L}$

正向引物 NC5: $5^{\prime}$ -GTAGGTGAACCTGCGGAAGGATCATT- $3^{\prime}$

反向引物 NC2:5'-TTAGTTTCTTTTCCTCCGCT-3'

扩增异尖线虫 ITS 基因片段长度为 $800\mathrm{bp}\sim 1100\mathrm{bp}$ 。

# 4.2 试剂配制

4.2.1 生理盐水：称取 $8.5\mathrm{gNaCl}$ 溶解于 $900~\mathrm{mL}$ 去离子水，再加去离子水至 $1000~\mathrm{mL}$   
4.2.2 $1\mathrm{mol} / \mathrm{L}$ Tris-HCl溶液：称取 $121.1\mathrm{g}$ Tris(三羟甲基氨基甲烷)溶解于 $800~\mathrm{mL}$ 去离子水，用盐酸调节 $\mathrm{pH}$ 至8.0，加去离子水至 $1000~\mathrm{mL}$ 。  
4.2.3 $0.5\mathrm{mol} / \mathrm{L}$ EDTA溶液：称取 $186.1\mathrm{gNa_2EDTA}\cdot 2\mathrm{H_2O}$ （二水合乙二胺四乙酸二钠)溶解于 $800~\mathrm{mL}$ 去离子水，搅拌溶解，用 $\mathrm{NaOH}$ 调节 $\mathrm{pH}$ 至8.0，加去离子水至 $1000~\mathrm{mL}$   
4.2.4 $5\mathrm{mol} / \mathrm{L}$ NaCl溶液：称取 $292.5\mathrm{gNaCl}$ 溶解于 $900~\mathrm{mL}$ 去离子水，再加水至 $1000~\mathrm{mL}$   
4.2.5 胃蛋白酶消化液：取胃蛋白酶 $5\mathrm{g}$ ，溶解于 $900~\mathrm{mL}$ 生理盐水中，加盐酸 $7\mathrm{mL}$ ，混匀，再加生理盐水至 $1000~\mathrm{mL}$ ，临用现配。  
4.2.6 裂解液： $10\%$ SDS溶液 $100~\mathrm{mL}, 1~\mathrm{mol} / \mathrm{L}$ Tris-HCl 溶液 $10~\mathrm{mL}, 0.5~\mathrm{mol} / \mathrm{L}$ EDTA 溶液 $200~\mathrm{mL}, 5~\mathrm{mol} / \mathrm{L}$ NaCl 溶液 $20~\mathrm{mL}$ ，加灭菌去离子水至 $1000~\mathrm{mL}$ 。  
4.2.7 $1.5\%$ 琼脂糖凝胶：琼脂糖 $1.5\mathrm{g}$ ，加入 $1\times \mathrm{TAE}$ 缓冲液至 $100~\mathrm{mL}$ ，加热至完全融化后冷却至 $60^{\circ}C\sim 70^{\circ}C$ ，加入 $10~\mathrm{mg / mL}$ 溴化乙锭 $5\mu \mathrm{L}$ ，混匀，制备凝胶。

# 5 检测方法

# 5.1 形态学方法

# 5.1.1 直接解剖观察法

# 5.1.1.1 样品制备

取动物性水产品或其制品，剖解，肉眼或体视显微镜观察腹腔、消化道、肠系膜、肝、生殖腺等器官及肌肉等组织。动物性水产品及其制品中的异尖线虫为第三期幼虫，虫体多以包囊形式寄生于宿主体内，包囊往往是一种圆形或长形套，紧包住虫体，若发现黄白色、无色微透明或乳白色的虫体或包囊，用镊子轻轻取出虫体或从包囊内分离出虫体，将虫体置于加有生理盐水的培养皿中，清除虫体周围杂物。

# 5.1.1.2 镜检

用生物显微镜观察分离到的虫体形态, 异尖线虫第三期幼虫的虫体呈黄白色、乳白色或无色微透

明，外形呈长纺锤形，体长 $10\mathrm{mm}\sim 30\mathrm{mm}$ 不等，头端较尾端尖细，头部有唇块，在腹侧有一明显的钻齿，中肠部体宽为 $430~\mu \mathrm{m}\sim 550~\mu \mathrm{m}$ ，无侧翼；活虫肉眼可见距头端 $2\mathrm{mm}\sim 3\mathrm{mm}$ 处有一个 $1\mathrm{mm}$ 的白点，即为长形小胃，显微镜下观察呈深色不透明，异尖线虫第三期幼虫模式图见图A.1，实物图见图A.2～图A.10。具备以上特征的虫体，可判定为异尖线虫科幼虫。对疑似异尖线虫科幼虫虫体立即用于DNA提取或 $-20^{\circ}C$ 保存备用。

注：如直接解剖观察法未检出异尖线虫科幼虫，则需用蛋白酶消化法进行检测。

# 5.1.2 胃蛋白酶消化法

# 5.1.2.1 取样

如果用直接解剖法未检出异尖线虫科幼虫，需采集动物性水产品或其制品的肌肉或脏器等组织用剪刀剪成小块。

# 5.1.2.2 消化

取处理好的样品 $200\mathrm{g}$ ，按照样品与胃蛋白酶消化液1：5的质量体积比加入消化液混匀，于 $37^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温培养箱放置 $4\mathrm{h} \sim 16\mathrm{h}$ ，使肌肉或脏器等组织完全消化，可根据不同水产品种类以及消化时间对胃蛋白酶浓度和使用量进行调整。

# 5.1.2.3 过滤

消化后的悬液用网筛过滤，并用生理盐水冲洗网筛上的残留物，用肉眼或放大镜观察残留物是否附着可疑虫体。收集滤液置于锥形量杯内，搅拌后沉淀 $15\mathrm{min}\sim 30\mathrm{min}$ 。轻轻倾去上清液，加入适量生理盐水，搅拌后再沉淀 $15\mathrm{min}\sim 30\mathrm{min}$ 。重复洗涤3次 $\sim 5$ 次，直至上清液透明为止，沉淀备用。

# 5.1.2.4 镜检

全部沉淀物分次转移至玻璃平皿，在体视显微镜下去除沉淀中的杂质，分离收集疑似虫体，用生物显微镜观察分离到的虫体形态。具备5.1.1.2特征的虫体，可判定为异尖线虫科幼虫。对疑似异尖线虫科幼虫虫体立即用于DNA提取或 $-20^{\circ}\mathrm{C}$ 保存备用。

# 5.2 PCR方法

# 5.2.1 DNA提取

取5.1.1.2或5.1.2.4判断为疑似异尖线虫科幼虫的虫体，放入 $1.5\mathrm{mL}$ 离心管中，加入无菌生理盐水 $200~\mu \mathrm{L}$ ，匀浆后加裂解液 $500~\mu \mathrm{L}$ 、蛋白酶K $10~\mu \mathrm{L}$ ，混匀， $55^{\circ}C$ 温浴至虫体被完全消化（ $1\mathrm{h}\sim 3\mathrm{h}$ ）。加入苯酚/三氯甲烷/异戊醇(25：24：1) $500~\mu \mathrm{L}$ ，混匀， $4^{\circ}C$ $12000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。吸取上清液加入等体积的三氯甲烷，混匀， $4^{\circ}C$ $12000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。吸取上清液加入0.8倍体积的异丙醇，充分混匀， $4^{\circ}C$ $12000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，弃上清。加入 $75\%$ 乙醇 $700~\mu \mathrm{L}$ 冲洗沉淀， $4^{\circ}C$ $12000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃上清液，干燥后加入 $50~\mu \mathrm{L}1\times \mathrm{TE}$ 溶液溶解DNA，立即用于检测或 $-20^{\circ}C$ 保存备用。

注：根据实验室实际情况，可使用经验证的商品化组织DNA提取试剂盒提取DNA。

# 5.2.2 PCR反应体系

在PCR管中依次加入 $10\times \mathrm{PCR}$ 缓冲液 $5.0\mu \mathrm{L},\mathrm{MgCl}_25.0\mu \mathrm{L},$ dNTPs $2.0\mu \mathrm{L}$ 、正向引物和反向引物各 $1.0~\mu \mathrm{L}$ 、耐热DNA聚合酶 $0.5\mu \mathrm{L}$ 、DNA模板 $2.5\mu \mathrm{L}$ ，加灭菌去离子水至总体积 $50~\mu \mathrm{L}$ 。每次试验需设阳性和空白对照。阳性对照用异尖线虫DNA或含有目标基因序列的质控品，空白对照用灭菌去离子水做模板。

# 5.2.3 PCR反应条件

$95\mathrm{C}$ 预变性 $4\mathrm{min};94\mathrm{C}$ 变性 $30~\mathrm{s},55~\mathrm{C}$ 退火 $30~\mathrm{s},72~\mathrm{C}$ 延伸 $30~\mathrm{s}$ ，进行35个循环； $72~\mathrm{C}$ 延伸 $10\mathrm{min};4\mathrm{C}$ 保存。

# 5.2.4 电泳

取PCR扩增产物 $10\mu \mathrm{L}$ 与 $6\times$ 上样缓冲液 $2\mu \mathrm{L}$ 混合，加样于 $1.5\%$ 琼脂糖凝胶中，其中一孔加入DNA分子量标准， $1\times$ TAE电泳缓冲液， $5\mathrm{V / cm}$ 恒压电泳 $30\mathrm{min}\sim 40\mathrm{min}$ ，用凝胶成像系统观察和记录结果。

# 5.2.5 PCR结果判定

阳性对照出现预期大小的条带 $(800\mathrm{bp}\sim 1100\mathrm{bp})$ ，空白对照无条带，待测样品扩增出预期大小的条带，可判PCR结果为阳性；无扩增条带或未扩增出预期大小的条带均判为PCR结果阴性。

取PCR结果为阳性的PCR产物进行基因序列双向测序，将测序结果与异尖线虫ITS参考序列（见附录B)进行同源性比对。

# 6 结果报告

6.1 形态学方法检出异尖线虫科幼虫，报告检出异尖线虫幼虫。  
6.2 形态学方法检出疑似异尖线虫科幼虫、PCR结果为阳性且扩增片段基因序列与任何一条异尖线虫参考序列同源性 $\geq 95\%$ ，报告检出异尖线虫幼虫。  
6.3 形态学方法未检出异尖线虫科幼虫、或PCR结果为阴性、或扩增片段基因序列与异尖线虫参考序列同源性均 $< 95\%$ ，报告未检出异尖线虫幼虫。

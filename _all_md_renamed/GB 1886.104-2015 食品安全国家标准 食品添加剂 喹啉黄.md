# 食品安全国家标准

# 食品添加剂 喹啉黄

# 1 范围

本标准适用于由2-(2-喹啉基)-1,3-茚二酮或三分之二2-(2-喹啉基)-1,3-茚二酮和三分之一[2-(6-甲基喹啉基)]-1,3-茚二酮的混合物磺化而制得的食品添加剂喹啉黄。

# 2 分子式和相对分子质量

# 2.1 分子式

$\mathrm{C_{18}H_9NNa_2O_8S_2}$ （主要成分）

# 2.2 相对分子质量

477.38（主要成分，按2007年国际相对原子质量）

# 3 产品分类

# 3.1 I型

2-(2-喹啉基)-1,3-茚二酮磺化而制得的食品添加剂喹啉黄。

# 3.2 II型

三分之二2-(2-喹啉基)-1，3-茚二酮和三分之一[2-(6-甲基喹啉基)]-1，3-茚二酮的混合物磺化而制得的食品添加剂喹啉黄。

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品在没有注明其他规定时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 喹啉黄的测定

# A.2.1 分光光度比色法

# A.2.1.1 方法提要

将喹啉黄试样用磷酸盐缓冲溶液 $(\mathrm{pH}\approx 7)$ 溶解，稀释定容后，在最大吸收波长处（ $415~\mathrm{nm}$ )，测其吸光度计算其含量。

# A.2.1.2 试剂和材料

磷酸盐缓冲溶液： $\mathrm{pH} \approx 7$ 。称取 $0.68 \mathrm{~g}$ 磷酸二氢钾，加 $29.1 \mathrm{~mL} 0.1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液，用水稀释至 $100 \mathrm{~mL}$ 。

# A.2.1.3 仪器和设备

A.2.1.3.1 分光光度计。

A.2.1.3.2 比色皿： $10\mathrm{mm}$

# A.2.1.4 分析步骤

# A.2.1.4.1 试样溶液的配制

称取约 $0.25\mathrm{g}$ 喹啉黄试样（精确至 $0.0001\mathrm{g}$ ），溶于适量磷酸盐缓冲溶液中，移入 $1000\mathrm{mL}$ 容量瓶中，加磷酸盐缓冲溶液稀释至刻度，摇匀。吸取 $10\mathrm{mL}$ ，移入 $250\mathrm{mL}$ 容量瓶中，用磷酸盐缓冲溶液稀释至刻度，摇匀，备用。

# A.2.1.4.2 测定

将喹啉黄试样溶液置于 $10\mathrm{mm}$ 比色皿中，同在最大吸收波长处（ $415~\mathrm{nm}$ ）用分光光度计测定其吸光度，用磷酸盐缓冲溶液作参比液。

# A.2.1.5 结果计算

喹啉黄的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {A \times 25 \times 1000}{865 \times m _ {1} \times 100} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

A ——试样溶液的吸光度值；

25 ——试样溶液的稀释倍数；

1000 ——试样溶液定容的体积，单位为毫升 $(\mathrm{mL})$ ；

865 在磷酸盐缓冲溶液中 $415\mathrm{nm}$ 波长处的百分吸光系数 $E_{1\mathrm{cm}}^{1\%}$

$m_{1}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

100 ——换算系数。

计算结果表示到小数点后两位。

平行测定结果的绝对差值不大于 $1.0\%$ ，取其算术平均值作为测定结果。

# A.2.2 高效液相色谱法

# A.2.2.1 方法提要

本方法用于由2-(2-喹啉基)-1,3-茚二酮磺化而制得的喹啉黄，用峰面积归一化法测定2-(2-喹啉基)-茚满基-1,3-二酮二磺酸二钠盐、2-(2-喹啉基)-茚满基-1,3-二酮单磺酸钠盐和2-(2-喹啉基)-茚满基-1,3-二酮三磺酸三钠盐的含量。

# A.2.2.2 试剂和材料

A.2.2.2.1 水：GB/T602中规定的二级水。

A.2.2.2.2 甲醇。

A.2.2.2.3 氢氧化钠溶液： $1\mathrm{mol} / \mathrm{L}$

A.2.2.2.4 乙酸溶液： $1\mathrm{mol} / \mathrm{L}$

A.2.2.2.5 乙酸钠缓冲溶液： $\mathrm{pH}\approx 4.6$ 。氢氧化钠溶液、乙酸溶液与水的体积比 $= 5:10:35$

# A.2.2.3 仪器和设备

高效液相色谱仪：配备紫外检测器，或其他等效的检测器，配有梯度洗脱功能。

# A.2.2.4 参考色谱条件

A.2.2.4.1 色谱柱： $\mathrm{C_{18}}$ 色谱柱， $250\mathrm{mm}\times 4.6\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ；或其他等效的色谱柱。

# A.2.2.4.2 流动相

流动相A：乙酸钠缓冲溶液与水体积比 $= 1:10$ ，流动相B：甲醇与流动相A的体积比 $= 80:20$ 。

A.2.2.4.3 流动相流速： $1.0~\mathrm{mL / min}$

A.2.2.4.4 检测波长： $254~\mathrm{nm}$

A.2.2.4.5 进样量： $20~\mu \mathrm{L}$

A.2.2.4.6 梯度洗脱：见表A.1梯度洗脱程序。

# A.2.2.5 分析步骤

# A.2.2.5.1 试样溶液的配制

称取约 $0.5\mathrm{g}$ 唑啉黄试样（精确至 $0.0001\mathrm{g}$ ），加流动相A溶液溶解并定容于 $100\mathrm{mL}$ 容量瓶中，摇匀备用。

# A.2.2.5.2 测定

在A.2.2.4参考色谱条件下，对试样溶液进行色谱分析，确定试样溶液色谱图中各组分对应的峰的保留时间及峰面积 $A_{i}$ ，用面积归一化法定量。

# A.2.2.6 结果计算

2-(2-喹啉基)-茚满基-1,3-二酮二磺酸二钠盐、2-(2-喹啉基)-茚满基-1,3-二酮单磺酸钠盐和2-(2-喹啉基)-茚满基-1,3-二酮三磺酸三钠盐的质量分数 $\varpi_{i}$ ，按式(A.2)计算：

$$
w _ {i} = \frac {A _ {i}}{\sum A _ {i}} \times 100 \% \tag{A.2}
$$

式中：

$A_{i}$ ——试样溶液色谱图中 $i$ 组分的峰面积；

$\sum A_{i}$ ——试样溶液色谱图中 $i$ 组分的峰面积之和。

计算结果表示到小数点后两位。

平行测定结果的绝对差值不大于 $1.0\%$ ，取其算术平均值作为测定结果。

# A.3 干燥减量、氯化物（以NaCl计）及硫酸盐（以 $\mathrm{Na_2SO_4}$ 计）总量的测定

# A.3.1 干燥减量的测定

# A.3.1.1 分析步骤

称取约 $2\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ ），置于已在 $135~^\circ \mathrm{C}\pm 2~^\circ \mathrm{C}$ 恒温干燥箱恒量的称量瓶中，在 $135~^\circ \mathrm{C}\pm$ $2^{\circ}C$ 恒温干燥箱中烘至恒量。

# A.3.1.2 结果计算

干燥减量的质量分数 $\omega_{2}$ ，按式(A.3)计算：

$$
w _ {2} = \frac {m _ {2} - m _ {3}}{m _ {2}} \times 100 \% \tag{A.3}
$$

式中：

$m_{2}$ ——干燥前试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{3}$ ——干燥后试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果表示到小数点后一位。

平行测定结果的绝对差值不大于 $0.2\%$ ，取其算术平均值作为测定结果。

# A.3.2 氯化物（以NaCl计）的测定

# A.3.2.1 试剂和材料

A.3.2.1.1 活性炭：767针型。

A.3.2.1.2 硝基苯。

A.3.2.1.3 硝酸溶液： $1 + 1$

A.3.2.1.4 硝酸银溶液： $0.1\mathrm{mol} / \mathrm{L}$

A.3.2.1.5 硫酸铁铵溶液：称取约 $14\mathrm{g}$ 硫酸铁铵，溶于 $100~\mathrm{mL}$ 水中，过滤，加 $10~\mathrm{mL}$ 硝酸，贮存于棕色瓶中。

A.3.2.1.6 硫氰酸铵标准滴定溶液： $\mathrm{c(NH_4CNS) = 0.1mol / L}$

# A.3.2.2 分析步骤

# A.3.2.2.1 试样溶液的配制

称取约 $2\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )，溶于 $150~\mathrm{mL}$ 水中，加约 $15\mathrm{g}$ 活性炭，温和煮沸 $2\mathrm{min}\sim 3\mathrm{min}$ ，加入 $1\mathrm{mL}$ 硝酸溶液，不断摇动均匀，放置 $30\mathrm{min}$ (其间不时摇动)。用干燥滤纸过滤。如滤液有色，则再加 $5\mathrm{g}$ 活性炭，不时摇动下放置 $1\mathrm{h}$ ，再用干燥滤纸过滤（如仍有色则更换活性炭重复操作至滤液无色）。每次以 $10~\mathrm{mL}$ 水洗活性炭三次，滤液合并移至 $200~\mathrm{mL}$ 容量瓶中，加水至刻度，摇匀。用于氯化物和硫酸盐含量的测定。

# A.3.2.2.2 测定

移取 $50~\mathrm{mL}$ 试样溶液，置于 $500~\mathrm{mL}$ 锥形瓶中，加 $2\mathrm{mL}$ 硝酸溶液和 $10~\mathrm{mL}$ 硝酸银溶液（氯化物含量多时要多加些）及 $5\mathrm{mL}$ 硝基苯，剧烈摇动至氯化银凝结，加入 $1\mathrm{mL}$ 硫酸铁铵溶液，用硫氰酸铵标准滴定溶液滴定过量的硝酸银到终点并保持 $1\mathrm{min}$ ，同时以同样方法做一空白试验。

# A.3.2.3 结果计算

氯化物（以NaCl计)的质量分数 $\varpi_{3}$ ，按式(A.4)计算：

$$
w _ {3} = \frac {\frac {V _ {1} - V _ {0}}{1 000} \times c _ {1} \times M _ {1}}{m _ {4} \times \frac {50}{200}} \times 100 \% \quad \dots \tag{A.4}
$$

式中：

$V_{1}$ ——滴定空白溶液耗用硫氰酸铵标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定试样溶液耗用硫氰酸铵标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$c_{1}$ ——硫氰酸铵标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{1}$ ——氯化钠的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $\left[\mathrm{M}_{1}(\mathrm{NaCl}) = 58.4\right]$

$m_{4}$ ——试样的质量，单位为克 $(\mathrm{g})$

50 ——试样消耗的体积，单位为毫升 $(\mathrm{mL})$

200 ——试样定容的体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果表示到小数点后一位。

平行测定结果的绝对差值不大于 $0.3\%$ ，取其算术平均值作为测定结果。

# A.3.3 硫酸盐（以 $\mathrm{Na_2SO_4}$ 计）的测定

# A.3.3.1 试剂和材料

A.3.3.1.1 氢氧化钠溶液： $2\mathrm{g / L}$

A.3.3.1.2 盐酸溶液： $1 + 1999$

A.3.3.1.3 氯化钡标准滴定溶液： $c\left(\frac{1}{2}\mathrm{BaCl}_2\right) = 0.1\mathrm{mol / L}$ （配制方法见附录B）。

A.3.3.1.4 酚酞指示液： $10\mathrm{g / L}$

A.3.3.1.5 玫瑰红酸钠指示液：称取 $0.1\mathrm{g}$ 玫瑰红酸钠，溶于 $10~\mathrm{mL}$ 水中（现用现配）。

# A.3.3.2 分析步骤

吸取 $25~\mathrm{mL}$ 试样溶液(A.3.2.2.1)，置于 $250~\mathrm{mL}$ 锥形瓶中，加1滴酚酞指示液，滴加氢氧化钠溶液呈粉红色，然后滴加盐酸溶液至粉红色消失，摇匀，溶解后在不断摇动下用氯化钡标准滴定溶液滴定，以玫瑰红酸钠指示液作外指示液，反应液与指示液在滤纸上交汇处呈现玫瑰红色斑点并保持 $2\mathrm{min}$ 不褪色为终点。

同时做空白试验。

# A.3.3.3 结果计算

硫酸盐(以 $\mathrm{Na_2SO_4}$ 计)的质量分数 $\omega_{4}$ ，按式(A.5)计算：

$$
w _ {4} = \frac {\frac {V _ {2} - V _ {3}}{1000} \times c _ {2} \times \frac {M _ {2}}{2}}{m _ {5} \times \frac {25}{200}} \times 100 \% \quad \dots \tag{A.5}
$$

式中：

$V_{2}$ ——滴定试样溶液耗用氯化钡标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——滴定空白溶液耗用氯化钡标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$c_{2}$ ——氯化钡标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M_{2}$ ——硫酸钠的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M_2(\mathrm{Na}_2\mathrm{SO}_4) = 142.04]$

2 ——换算系数；

$m_{5}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

25 ——试样消耗的体积，单位为毫升 $(\mathrm{mL})$

200 ——试样定容的体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果表示到小数点后一位。

平行测定结果的绝对差值不大于 $0.2\%$ ，取其算术平均值作为测定结果。

A.6.3.1 可见分光光度计。

A.6.3.2 $40~\mathrm{mm}$ 比色Ⅲ。

# A.6.4 分析步骤

# A.6.4.1 试样萃取溶液的配制

称取约 $2.0\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ ) 于 $150~\mathrm{mL}$ 烧杯中，加 $100~\mathrm{mL}$ 水和 $5\mathrm{mL}$ 氢氧化钠溶液 $(40~\mathrm{g / L})$ ，在温水浴中搅拌至完全溶解。将此溶液移入分液漏斗中，少量水洗净烧杯。每次以 $50~\mathrm{mL}$ 乙酸乙酯萃取两次，合并萃取液。以 $10~\mathrm{mL}$ 氢氧化钠溶液 $(4~\mathrm{g / L})$ 洗涤乙酸乙酯萃取液，除去痕量色素。再每次以 $10~\mathrm{mL}$ 盐酸溶液 $(1 + 3)$ 对乙酸乙酯溶液反萃取三次。合并该盐酸萃取液，然后用水稀释至 $100~\mathrm{mL}$ ，摇匀。此溶液为试样萃取溶液。

# A.6.4.2 标准溶液的制备

分别吸取 $5\mathrm{mL},10\mathrm{mL},15\mathrm{mL},20\mathrm{mL},25\mathrm{mL}$ 苯胺标准溶液至 $100~\mathrm{mL}$ 容量瓶中，用盐酸溶液 $(1 + 10)$

稀释至刻度，混合均匀，此为标准溶液。

# A.6.4.3 重氮化偶合溶液的制备

吸取 $10\mathrm{mL}$ 试样萃取溶液，移入透明洁净的试管中，浸入盛有冰水混合物的烧杯内冷却 $10\mathrm{min}$ 。在试管中加入 $1\mathrm{mL}$ 溴化钾溶液及 $0.5\mathrm{mL}$ 亚硝酸钠溶液，稍用力摇匀后仍置于冰水浴中冷却 $10\mathrm{min}$ ，进行重氮化反应。另取一个 $25\mathrm{mL}$ 容量瓶移入 $1\mathrm{mL}$ 盐溶液和 $10\mathrm{mL}$ 碳酸钠溶液。将上述试管中的苯胺重氮盐溶液加至盛有R盐溶液的容量瓶中，边加边略振摇容量瓶，用少许水洗净试管一并加入容量瓶中，再以水定容。充分混匀后在暗处放置 $15\mathrm{min}$ 。

# A.6.4.4 标准重氮化偶合溶液的制备

分别吸取 $10\mathrm{mL}$ A.6.4.2系列标准溶液，其余步骤同A.6.4.3。

# A.6.4.5 参比溶液的制备

吸取 $10\mathrm{mL}$ 盐酸溶液 $(1 + 10)$ 、 $10~\mathrm{mL}$ 碳酸钠溶液及 $1\mathrm{mL}$ R盐溶液于 $25~\mathrm{mL}$ 容量瓶中，用水定容。

# A.6.4.6 测定

将A.6.4.4中一系列标准重氮化偶合溶液分别置于比色皿中，在 $510~\mathrm{nm}$ 波长处用分光光度计测定各自的吸光度，以A.6.4.5作参比溶液，绘制成标准曲线。

试样重氮化偶合溶液置于比色皿中，在 $510\mathrm{nm}$ 波长处用分光光度计测定吸光度 $A_{\mathrm{a}}$ ，以A.6.4.5作参比溶液，根据标准曲线求出苯胺的质量。

$m_{8}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

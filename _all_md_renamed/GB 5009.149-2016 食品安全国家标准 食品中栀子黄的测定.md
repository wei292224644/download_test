# 食品中栀子黄的测定

2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替GB5009.149—2003《食品中栀子黄的测定》。

本标准与GB5009.149—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中栀子黄的测定”；  
——原标准第一法高效液相色谱法测定栀子苷修改为高效液相色谱法测定藏花素和藏花酸；  
——改进了样品前处理分析方法；  
扩大了方法适用范围；  
——删除了第二法 薄层色谱法。

# 食品安全国家标准

# 食品中栀子黄的测定

# 1 范围

本标准规定了食品中栀子黄的代表性成分：藏花素、藏花酸的测定方法。

本标准适用于冰淇淋、蜜饯、腌菜、干杏仁、巧克力、糕点、熟肉、酱油、果汁、配制酒、果冻、薯片中藏花素和藏花酸的测定。

# 2 原理

试样用甲醇超声提取后，通过 $\mathrm{C_{18}}$ 反相色谱柱分离，用高效液相色谱/可见光检测器于 $440~\mathrm{nm}$ 下检测栀子黄的主要显色成分藏花素和藏花酸，以保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.3 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.4 乙酸铵 $\mathrm{(CH_3COONH_4)}$ 。

# 3.2 试剂配制

3.2.1 乙酸-乙酸铵溶液 $(\mathrm{pH} = 4)$ ：准确称取 $0.77\mathrm{g}$ 乙酸铵置于 $1\mathrm{L}$ 容量瓶内，加 $900~\mathrm{mL}$ 水溶解，用冰乙酸调节 $\mathrm{pH} = 4.0$ ，加水定容至 $1\mathrm{L}$ ，混匀，经 $0.45\mu \mathrm{m}$ 微孔滤膜过滤后使用。

# 3.3 标准品

3.3.1 藏花素标准品 $\mathrm{(C_{44}H_{64}O_{24}}$ ，CAS号：42553-65-1)，纯度 $\geq 90.0\%$   
3.3.2 藏花酸标准品 $\mathrm{(C_{20}H_{24}O_4}$ ，CAS号：27876-94-4)，纯度 $\geq 90.0\%$

# 3.4 标准溶液配制

3.4.1 藏花素标准储备液：准确称取 $5.00 \mathrm{mg}$ (精确至 $0.01 \mathrm{mg}$ ) 藏花素标准品，用甲醇溶解，转移到 $10 \mathrm{~mL}$ 容量瓶中，用甲醇定容，混匀，藏花素浓度为 $0.5 \mathrm{mg} / \mathrm{mL}$ ，于 $4^{\circ} \mathrm{C}$ 保存。  
3.4.2 藏花酸标准储备液：准确称取 $1.00\mathrm{mg}$ (精确至 $0.01\mathrm{mg}$ ) 藏花酸标准品，用甲醇溶解，转移到 $10\mathrm{mL}$ 容量瓶中，用甲醇定容，混匀，藏花酸浓度为 $0.1\mathrm{mg/mL}$ ，于 $4^{\circ}\mathrm{C}$ 保存。  
3.4.3 藏花素、藏花酸混合标准曲线工作液: 分别均吸取藏花素标准储备液、藏花酸标准储备液 $0.05 \mathrm{~mL} 、 0.1 \mathrm{~mL} 、 0.2 \mathrm{~mL} 、 0.5 \mathrm{~mL} 、 1.0 \mathrm{~mL}$ 于 $10 \mathrm{~mL}$ 容量瓶中, 用甲醇定容, 混匀, 藏花素系列标准工作液浓度分

别均为 $2.5\mu \mathrm{g / mL},5\mu \mathrm{g / mL},10\mu \mathrm{g / mL},25\mu \mathrm{g / mL},50\mu \mathrm{g / mL}$ ，藏花酸系列标准工作液浓度分别均为 $0.5\mu \mathrm{g / mL},1\mu \mathrm{g / mL},2\mu \mathrm{g / mL},5\mu \mathrm{g / mL},10\mu \mathrm{g / mL}$ ，于 $4^{\circ}C$ 保存。

# 4 仪器和设备

4.1 高效液相色谱仪，配置可见检测器。  
4.2 分析天平：感量为 $0.01\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。  
4.3 离心机：转速 $\geq 5000\mathrm{r / min}$   
4.4 酸度计。  
4.5 超声波清洗仪。  
4.6 食品粉碎机。  
4.7 有机相型微孔滤膜：孔径 $0.45\mu \mathrm{m}$

# 5 分析步骤

# 5.1 试样制备及保存

# 5.1.1 液体样品

将饮料、酒类、酱油等样品摇匀分装，密闭常温或冷藏保存。

# 5.1.2 半固态样品

对果冻等样品取可食部分匀浆后，搅拌均匀，分装，密闭冷藏或冷冻保存。

# 5.1.3 固体样品

饼干、糕点、熟肉制品、可可制品等低含水量样品，经高速粉碎机粉碎、分装，于室温下避光密闭保存。

# 5.2 试样处理

# 5.2.1 液体样品

称取 $2\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样(若试样中含二氧化碳应先超声除去)置于烧杯中，用适量甲醇溶解并转移至 $25\mathrm{mL}$ 容量瓶中，加甲醇定容，摇匀。吸取 $1\mathrm{mL}$ 溶液过 $0.45\mu \mathrm{m}$ 有机滤膜，待测。

# 5.2.2 半固体试样及固体试样

称取 $2\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样于 $50~\mathrm{mL}$ 离心管中，准确量取 $25~\mathrm{mL}$ 甲醇溶液加入其中，超声 $20\mathrm{min}$ 后涡旋 $2\mathrm{min},4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，吸取 $1\mathrm{mL}$ 上清液过 $0.45\mu \mathrm{m}$ 有机滤膜，待测。

# 5.2.3 酱油

称取 $1\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样置于烧杯中，用适量甲醇溶解并转移至 $50~\mathrm{mL}$ 容量瓶中，加甲醇定容，摇匀。吸取 $1\mathrm{mL}$ 溶液过 $0.45\mu \mathrm{m}$ 有机滤膜，待测。

注：5.2过程应在避免强光照射的环境下进行。

# 5.3 仪器参考条件

仪器参考条件列出如下：

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或同等性能的色谱柱；  
b）流动相：A：乙酸-乙酸铵缓冲溶液，B：乙腈，梯度洗脱见表1；  
c）进样量： $10~\mu \mathrm{L}$   
d) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
e) 检测波长: $440 \mathrm{~nm}$ 。

# 5.4 标准曲线的制作

将混合标准工作液分别注入高效液相色谱仪中，测定藏花素和藏花酸相应的峰面积，以标准工作液的浓度为横坐标，以色谱峰面积为纵坐标，绘制标准曲线(标准图谱见图A.1)。

# 5.5 试样溶液的测定

将试样溶液注入高效液相色谱仪中，得到藏花素和藏花酸的峰面积，根据标准曲线得到待测液中藏花素和藏花酸的浓度。

# 6 分析结果的表述

试样中藏花素和藏花酸的含量按式(1)计算：

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \times F \quad \dots \tag {1}
$$

式中：

$X$ ——试样中藏花素或藏花酸的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho$ ——由标准曲线求得试样溶液中藏花素或藏花酸的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样品溶液定容体积，单位为 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——最终样液代表的试样质量，单位为克(g)；

$F$ ——标准品的纯度折算系数。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

方法检出限(LOD)：酱油样品当称样量为 $1.0\mathrm{g}$ 时，藏花素为 $50~\mathrm{mg / kg}$ ，藏花酸为 $10.0~\mathrm{mg / kg}$ ，其他样品取样量为 $2.0\mathrm{g}$ 时，藏花素为 $12.5~\mathrm{mg / kg}$ ，藏花酸为 $2.5~\mathrm{mg / kg}$ 。

方法定量限(LOQ)：酱油样品当称样量为 $1.0\mathrm{g}$ 时，藏花素为 $250\mathrm{mg / kg}$ ，藏花酸为 $50.0\mathrm{mg / kg}$ ，其他样品取样量为 $2.0\mathrm{g}$ 时，藏花素为 $62.5\mathrm{mg / kg}$ ，藏花酸为 $12.5\mathrm{mg / kg}$ 。

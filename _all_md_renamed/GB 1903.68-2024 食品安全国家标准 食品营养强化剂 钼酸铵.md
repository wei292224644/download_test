钼酸铵；四水合钼酸铵

$$
\left(\mathrm {N H} _ {4}\right) _ {6} \mathrm {M o} _ {7} \mathrm {O} _ {2 4} \cdot 4 \mathrm {H} _ {2} \mathrm {O}
$$

# 2.3 相对分子质量

1235.92（按2018年国际相对原子质量）

# 附录A

# 检验方法

# A.1 安全提示

试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和健康措施。

# A.2 一般规定

本标准所用试剂和水在未注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂及制品，在未注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 氨水。

A.3.1.2 硝酸。

A.3.1.3 磷酸氢二钠溶液： $120\mathrm{g / L}$

A.3.1.4 氨水溶液 $(10\%)$ ：量取 $40.56~\mathrm{mL}$ 氨水，稀释至 $100~\mathrm{mL}$ 。

# A.3.2 仪器和设备

A.3.2.1 垂熔玻璃滤器：孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$ 。  
A.3.2.2 电子天平：感量 $0.01\mathrm{g}$ 。

# A.3.3 鉴别方法

称取 $0.6\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )，加水 $1.4\mathrm{mL}$ 和 $1.45\mathrm{mL}$ 氨水(A.3.1.1)使其溶解，放至室温，缓慢加入已冷却的硝酸(A.3.1.2) $3.2\mathrm{mL}$ 和水 $4\mathrm{mL}$ 的混合溶液，混匀。放置 $24\mathrm{h} \sim 48\mathrm{h}$ 后，用垂熔玻璃滤器(A.3.2.1)过滤，取滤液 $5\mathrm{mL}$ ，加磷酸氢二钠溶液(A.3.1.3) $2\mathrm{mL}$ ，应产生黄色沉淀。加入过量的氨水溶液(A.3.1.4)，沉淀应溶解。

# A.4 不溶性杂质的测定

# A.4.1 方法提要

用水溶解试样，将不溶物滤出，用水洗涤滤渣，使之与试样主体完全分离，烘干后用天平称出水不溶性杂质的质量。

# A.4.2 仪器和设备

A.4.2.1 玻璃砂芯坩埚：孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$   
A.4.2.2 电热恒温干燥箱：温度可控制在 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。  
A.4.2.3 电子天平：感量 $0.0001\mathrm{g}$ 。

# A.4.3 分析步骤

称取 $20\mathrm{g}$ 试样(精确至 $0.0001\mathrm{g}$ )，加 $200~\mathrm{mL}$ 水使其溶解，加热煮沸，覆盖表面皿，在沸水浴上加

热 $1\mathrm{h}$ 。用预先于 $105~\mathrm{^\circ C}\pm 2$ ℃干燥至质量恒定的玻璃砂芯坩埚(A.4.2.1)趁热过滤，并用热水反复洗涤后置于电热恒温干燥箱(A.4.2.2)中， $105~\mathrm{^\circ C}\pm 2$ ℃下干燥 $2\mathrm{h}$ 。

# A.4.4 结果计算

不溶性杂质的含量 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {m _ {2} - m _ {1}}{m _ {3}} \times 100 \% \tag{A.1}
$$

式中：

$m_{2}$ ——不溶性杂质与玻璃砂芯坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——玻璃砂芯坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{3}$ ——试样质量，单位为克 $(\mathrm{g})$ 。

取平行测定结果的算术平均值为测定结果，两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# A.5 氯化物的测定

# A.5.1 方法提要

在硝酸介质中，氯离子与银离子生成难溶的氯化银。当氯离子含量较低时，在一定时间内氯化银呈悬浮体，使溶液浑浊，可用于氯化物的目视比浊法测定。

# A.5.2 试剂和材料

A.5.2.1 硝酸。

A.5.2.2 硝酸银溶液： $17\mathrm{g / L}$

A.5.2.3 氯化物标准溶液 $(0.1\mathrm{mg / mL})$ ：按GB/T602配制。

# A.5.3 仪器和设备

电子天平：感量 $0.0001\mathrm{g}$ 。

# A.5.4 分析步骤

称取 $1\mathrm{g}$ 样品（精确至 $0.0001\mathrm{g}$ ），用水定容至 $25\mathrm{mL}$ ，此液为试样溶液。

量取氯化物标准溶液(A.5.2.3) $0.2\mathrm{mL}$ ，用水定容至 $25~\mathrm{mL}$ ，此液为对照溶液。

在试样溶液和对照溶液中分别加入 $5.0\mathrm{mL}$ 硝酸(A.5.2.1)及 $1.0\mathrm{mL}$ 硝酸银溶液(A.5.2.2)，摇匀，暗处放置 $10\mathrm{min}$ 。在黑色背景下，比较两者浊度，试样溶液浊度不超过对照溶液浊度，为试样中氯化物浓度不大于 $0.002\%$ 。

# A.6 硫酸盐的测定

# A.6.1 方法提要

在盐酸介质中，钡离子与硫酸根离子生成难溶的硫酸钡。当硫酸根含量低时，在一定时间内硫酸钡呈悬浮体，使溶液浑浊，可用于硫酸盐的目视比浊法测定。

# A.6.2 试剂和材料

A.6.2.1 盐酸溶液 $(9\%)$ ：量取 $25~\mathrm{mL}$ 盐酸，稀释至 $100~\mathrm{mL}$

A.6.2.2 氯化钡溶液： $120\mathrm{g / L}$

A.6.2.3 硫酸盐 $\left(\mathrm{SO}_{4}\right)$ 标准溶液：按 GB/T 602 配制后，稀释至每 $1 \mathrm{~mL}$ 相当于 $0.05 \mathrm{mg}$ 硫酸根离子。

# A.6.3 仪器和设备

电子天平：感量 $0.0001\mathrm{g}$ 。

# A.6.4 分析步骤

称取 $0.25\mathrm{g}$ 试样(精确至 $0.0001\mathrm{g}$ )，置于 $50~\mathrm{mL}$ 具塞比色管中，加入 $30~\mathrm{mL} \sim 40~\mathrm{mL}$ 水溶解，此液为试样溶液。必要时用盐酸中和至石蕊试纸显中性。

取另一根 $50~\mathrm{mL}$ 具塞比色管，量取硫酸盐 $\mathrm{(SO_4)}$ 标准溶液(A.6.2.3) $1.0~\mathrm{mL}$ ，加入 $30~\mathrm{mL} \sim 40~\mathrm{mL}$ 水溶解，此液为对照溶液。必要时用盐酸中和至石蕊试纸显中性。

在试样溶液和对照溶液中分别加入盐酸溶液(A.6.2.1) $1.0\mathrm{mL}$ 、氯化钡试液(A.6.2.2) $3.0\mathrm{mL}$ ，加水稀释至 $50~\mathrm{mL}$ ，混匀，放置 $10\mathrm{min}$ 。在黑色背景下，比较两者浊度，试样溶液浊度不超过对照溶液浊度，为试样中硫酸盐(以 $\mathrm{SO}_4^{2-}$ 计)浓度不大于 $0.02\%$ 。

# A.7 镁及碱金属的测定

# A.7.1 试剂和材料

A.7.1.1 碳酸钠。

A.7.1.2 氢氧化钠溶液： $100\mathrm{g / L}$   
A.7.1.3 氨水溶液 $(1.6\%)$ ：量取 $67.6~\mathrm{mL}$ 氨水，稀释至 $1000~\mathrm{mL}$ 。  
A.7.1.4 无灰滤纸。

# A.7.2 仪器和设备

A.7.2.1 高温炉：温度可控制在 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$   
A.7.2.2 电子天平：感量 $0.0001\mathrm{g}$ 。

# A.7.3 分析步骤

称取 $5\mathrm{g}$ 试样（精确至 $0.0001\mathrm{g}$ ），加入 $50~\mathrm{mL}$ 水使其溶解，过滤。在滤液中加入 $0.5\mathrm{g}$ 碳酸钠(A.7.1.1)和 $25~\mathrm{mL}$ 氢氧化钠溶液(A.7.1.2)。加热使其微沸 $5\mathrm{min}$ ，冷却后用无灰滤纸过滤，并用氨水溶液(A.7.1.3)分次洗涤杯中残渣和滤纸，直至洗净为止。将滤纸连同残渣移入预先灼烧至 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 恒重的坩埚内，炭化后 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 下灼烧至恒重。

# A.7.4 结果计算

镁及碱金属的含量 $\varpi_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m _ {5} - m _ {4}}{m _ {6}} \times 100 \% \tag{A.2}
$$

式中：

$m_{5}$ ——坩埚和灰分的质量，单位为克 $(\mathrm{g})$ ；

$m_{4}$ ——坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{6}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

取平行测定结果的算术平均值为测定结果，两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# A.8 硝酸盐的测定

# A.8.1 方法提要

在硫酸介质中，靛兰胭脂红被硝酸盐氧化成黄色的靛崧而使蓝色消褪，用目视比色法测定。

# A.8.2 试剂和材料

A.8.2.1 氯化钠溶液： $0.5\mathrm{g} / 100\mathrm{mL}$   
A.8.2.2 硫酸溶液：量取 $192~\mathrm{mL}$ 硫酸，缓慢加入约 $400~\mathrm{mL}$ 水中，冷却至室温后加水稀释至 $1000~\mathrm{mL}$ 。  
A.8.2.3 龙兰胭脂红溶液：称取 $1\mathrm{g}$ 龙兰胭脂红，加硫酸溶液(A.8.2.2)稀释至 $1000~\mathrm{mL}$

# A.8.3 仪器和设备

电子天平：感量 $0.01\mathrm{g}$ 。

# A.8.4 分析步骤

称取 $1\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )，加入氯化钠溶液(A.8.2.1) $10~\mathrm{mL}$ ，混匀。加入 $0.10~\mathrm{mL}$ 龄兰胭脂红溶液(A.8.2.3)， $5\mathrm{min}$ 内溶液所显蓝色不消失为通过试验。

# A.9 磷酸盐

# A.9.1 方法提要

在酸性硝酸溶液介质中，正磷酸盐与钼酸铵反应，生成黄色的磷钼杂多酸，可用目视比色法测定。

# A.9.2 试剂和材料

A.9.2.1 氨水。

A.9.2.2 氨水溶液 $(5\%)$ ：量取 $20.28~\mathrm{mL}$ 氨水，稀释至 $100~\mathrm{mL}$ 。

A.9.2.3 磷酸盐储备溶液 $(1\mathrm{mg / mLPO_4}^{3 - })$ ：称取在 $105~^\circ \mathrm{C}\pm 2~^\circ \mathrm{C}$ 干燥 $4\mathrm{h}$ 的磷酸二氢钾 $1.433\mathrm{g}$ ，溶于水中，并稀释至 $1\mathrm{L}$ 。

A.9.2.4 磷酸盐工作溶液 $(0.1\mathrm{mg / mLPO_4}^{3 - })$ ：取上述储备溶液，用水准确稀释10倍。

A.9.2.5 硝酸铁溶液：称取 $10\mathrm{g}$ 九水合硝酸铁，用水溶解后稀释至 $100~\mathrm{mL}$

A.9.2.6 氨水溶液 $(2.5\%)$ ：量取 $10.14~\mathrm{mL}$ 氨水，稀释至 $100~\mathrm{mL}$ 。

A.9.2.7 硝酸溶液 $(18\%)$ ：量取 $26.7~\mathrm{mL}$ 硝酸，稀释至 $100~\mathrm{mL}$

A.9.2.8 钼酸铵溶液: 称取钼酸细粉 $6.5 \mathrm{~g}$ , 加水 $14 \mathrm{~mL}$ 、氨水 $14.5 \mathrm{~mL}$ 的混合溶液溶解, 放冷, 缓缓加入已冷却的硝酸 $(32 \mathrm{~mL})$ 和水 $(40 \mathrm{~mL})$ 的混合溶液, 并不断搅拌。静置 $48 \mathrm{~h}$ 后过滤。

注：钼酸铵溶液(A.9.2.8)放置后会变质而不适于使用。在 $5\mathrm{mL}$ 钼酸铵溶液(A.9.2.8)中加入 $2\mathrm{mL}$ 磷酸氢二钠溶液(A.3.1.3)，室温下或微热后没有立即产生大量黄色沉淀，则钼酸铵溶液(A.9.2.8)没有变质。钼酸铵溶液(A.9.2.8)放置在暗处保存，若贮存时产生少量沉淀，可取上清液用于测定。

# A.9.3 仪器和设备

电子天平：感量 $0.01\mathrm{g}$ 。

# A.9.4 分析步骤

试样溶液：称取试样 $20\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )，加入氨水溶液(A.9.2.2)溶解并稀释至 $100~\mathrm{mL}$ 。

对照溶液：量取磷酸盐工作溶液(A.9.2.4) $1.0~\mathrm{mL}$ ，加入氨水溶液(A.9.2.2)溶解并稀释至 $100~\mathrm{mL}$ 。

分别在对照溶液和试样溶液中加入 $3.5\mathrm{mL}$ 硝酸铁溶液(A.9.2.5)，放置 $15\mathrm{min}$ 。缓慢加热(但不要沸腾)使沉淀凝聚，过滤，并用氨水溶液(A.9.2.6)洗涤滤渣数次，充分洗净。用 $60~\mathrm{mL}$ 温热的硝酸溶液(A.9.2.7)溶解滤纸上的残留物，收集该滤液至 $250~\mathrm{mL}$ 具塞烧瓶中。加入 $13~\mathrm{mL}$ 氨水(A.9.2.1)，水浴加热至 $40^{\circ}\mathrm{C}$ 。加入 $50~\mathrm{mL}$ 钼酸铵溶液(A.9.2.8)，剧烈振摇 $5\mathrm{min}$ ，然后在 $40^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 水浴中放置 $2\mathrm{h}$ 。

试样溶液中产生的黄色沉淀不多于对照溶液，为试样中磷酸盐不大于 $5\mathrm{mg / kg}$ 。

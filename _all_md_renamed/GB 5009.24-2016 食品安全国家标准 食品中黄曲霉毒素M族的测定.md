# 前言

本标准代替GB5413.37—2010《食品安全国家标准乳和乳制品中黄曲霉毒素 $\mathbf{M}_1$ 的测定》、GB5009.24—2010《食品安全国家标准食品中黄曲霉毒素 $\mathbf{M}_1$ 和 $\mathbf{B}_1$ 的测定》、GB/T23212—2008《牛奶和奶粉中黄曲霉毒素 $\mathrm{B_1,B_2,G_1,G_2,M_1,M_2}$ 的测定高效液相色谱法-荧光检测法》和SN/T1664—2005《牛奶和奶粉中黄曲霉毒素 $\mathrm{M_1,B_1,B_2,G_1,G_2}$ 含量的测定》。

本标准与GB5413.37—2010相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中黄曲霉毒素M族的测定”；  
——增加了方法适用范围；  
——增加了对黄曲霉毒素 $\mathrm{M}_2$ 的检测；  
——修改了酶联免疫法，并修改第三法名称为酶联免疫吸附筛查法；  
——修改了液相色谱-质谱联用法；  
——修改了液相色谱法的前处理方法；  
——删除了免疫层析净化荧光分光度法。

# 食品安全国家标准

# 食品中黄曲霉毒素M族的测定

# 1 范围

本标准规定了食品中黄曲霉毒素 $\mathrm{M}_{1}$ 和黄曲霉毒素 $\mathrm{M}_{2}$ （以下简称AFT $\mathrm{M}_{1}$ 和AFT $\mathrm{M}_{2}$ )的测定方法。

第一法为同位素稀释液相色谱-串联质谱法，适用于乳、乳制品和含乳特殊膳食用食品中 $\mathrm{AFTM_1}$ 和 $\mathrm{AFTM_2}$ 的测定。

第二法为高效液相色谱法，适用范围同第一法。

第三法为酶联免疫吸附筛查法，适用于乳、乳制品和含乳特殊膳食用食品中 $\mathrm{AFTM_1}$ 的筛查测定。

# 第一法 同位素稀释液相色谱-串联质谱法

# 2 原理

试样中的黄曲霉毒素 $\mathrm{M}_{1}$ 和黄曲霉毒素 $\mathrm{M}_{2}$ 用甲醇-水溶液提取，上清液用水或磷酸盐缓冲液稀释后，经免疫亲和柱净化和富集，净化液浓缩、定容和过滤后经液相色谱分离，串联质谱检测，同位素内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.3 乙酸铵 $\mathrm{(CH_3COONH_4)}$ 。  
3.1.4 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.5 磷酸氢二钠 $\mathrm{(Na_2HPO_4)}$ 。  
3.1.6 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
3.1.7 氯化钾(KCl)。  
3.1.8 盐酸（HCl）。  
3.1.9 石油醚 $\left(\mathrm{C}_{n} \mathrm{H}_{2 n+2}\right)$ ：沸程为 $30^{\circ} \mathrm{C} \sim 60^{\circ} \mathrm{C}$ 。

# 3.2 试剂配制

3.2.1 乙酸铵溶液 $(5\mathrm{mmol} / \mathrm{L})$ ：称取 $0.39\mathrm{g}$ 乙酸铵，溶于 $1000~\mathrm{mL}$ 水中，混匀。  
3.2.2 乙腈-水溶液 $(25 + 75)$ ：量取 $250~\mathrm{mL}$ 乙腈加入 $750~\mathrm{mL}$ 水中，混匀。  
3.2.3 乙腈-甲醇溶液 $(50 + 50)$ ：量取 $500~\mathrm{mL}$ 乙腈加入 $500~\mathrm{mL}$ 甲醇中，混匀。

3.2.4 磷酸盐缓冲溶液(以下简称PBS)：称取 $8.00\mathrm{g}$ 氯化钠、 $1.20\mathrm{g}$ 磷酸氢二钠（或 $2.92\mathrm{g}$ 十二水磷酸氢二钠）、 $0.20\mathrm{g}$ 磷酸二氢钾、 $0.20\mathrm{g}$ 氯化钾，用 $900\mathrm{mL}$ 水溶解后，用盐酸调节 $\mathrm{pH}$ 至7.4，再加水至 $1000\mathrm{mL}$ 。

# 3.3 标准品

3.3.1 AFT $\mathrm{M}_1$ 标准品（ $\mathrm{C}_{17}\mathrm{H}_{12}\mathrm{O}_7$ ，CAS：6795-23-9）：纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.2 AFT $\mathrm{M}_2$ 标准品 $(\mathrm{C}_{17}\mathrm{H}_{14}\mathrm{O}_7,\mathrm{CAS}:6885 - 57 - 0)$ ：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.3 $^{13}\mathrm{C}_{17} - \mathrm{AFT}\mathrm{M}_1$ 同位素溶液（ $\mathrm{^{13}C_{17}H_{14}O_7})$ ： $0.5\mu \mathrm{g / mL}$

# 3.4 标准溶液配制

3.4.1 标准储备溶液 $(10\mu \mathrm{g / mL})$ ：分别称取 $\mathrm{AFTM_1}$ 和 $\mathrm{AFTM_21mg}$ （精确至 $0.01\mathrm{mg})$ ，分别用乙腈溶解并定容至 $100~\mathrm{mL}$ 。将溶液转移至棕色试剂瓶中，在一 $20^{\circ}C$ 下避光密封保存。临用前进行浓度校准(校准方法参见附录A)。  
3.4.2 混合标准储备溶液 $(1.0\mu \mathrm{g / mL})$ ：分别准确吸取 $10~\mu \mathrm{g / mL}$ AFT $\mathbf{M}_1$ 和AFT $\mathbf{M}_2$ 标准储备液 $1.00~\mathrm{mL}$ 于同一 $10~\mathrm{mL}$ 容量瓶中，加乙腈稀释至刻度，得到 $1.0~\mu \mathrm{g / mL}$ 的混合标准液。此溶液密封后避光 $4^{\circ}C$ 保存，有效期3个月。  
3.4.3 混合标准工作液 $(100\mathrm{ng / mL})$ ：准确吸取混合标准储备溶液 $(1.0~\mu \mathrm{g / mL})1.00~\mathrm{mL}$ 至 $10~\mathrm{mL}$ 容量瓶中，乙腈定容。此溶液密封后避光 $4^{\circ}C$ 下保存，有效期3个月。  
3.4.4 $50\mathrm{ng / mL}$ 同位素内标工作液 $1(^{13}\mathrm{C}_{17} - \mathrm{AFTM}_1)$ ：取 $\mathrm{AFTM_1}$ 同位素内标 $(0.5\mu \mathrm{g / mL})1\mathrm{mL}$ ，用乙腈稀释至 $10~\mathrm{mL}$ 。在 $-20^{\circ}C$ 下保存，供测定液体样品时使用。有效期3个月。  
3.4.5 $5\mathrm{ng / mL}$ 同位素内标工作液 $2(\mathrm{^{13}C_{17} - AFTM_1})$ ：取 $\mathrm{AFTM_1}$ 同位素内标 $(0.5\mu \mathrm{g / mL})100\mu \mathrm{L}$ ，用乙腈稀释至 $10~\mathrm{mL}$ 。在 $-20^{\circ}\mathrm{C}$ 下保存，供测定固体样品时使用。有效期3个月。  
3.4.6 标准系列工作溶液：分别准确吸取标准工作液 $5\mu \mathrm{L},10\mu \mathrm{L},50\mu \mathrm{L},100\mu \mathrm{L},200\mu \mathrm{L},500\mu \mathrm{L}$ 至 $10~\mathrm{mL}$ 容量瓶中，加入 $100~\mu \mathrm{L}~50~\mathrm{ng / mL}$ 的同位素内标工作液，用初始流动相定容至刻度，配制AFT $\mathbf{M}_1$ 和AFT $\mathbf{M}_2$ 的浓度均为 $0.05  \mathrm{ng / mL},0.1  \mathrm{ng / mL},0.5  \mathrm{ng / mL},1.0  \mathrm{ng / mL},2.0  \mathrm{ng / mL},5.0  \mathrm{ng / mL}$ 的系列标准溶液。

# 4 仪器和设备

4.1 天平：感量 $0.01\mathrm{g}, 0.001\mathrm{g}$ 和 $0.00001\mathrm{g}$ 。  
4.2 水浴锅：温控 $50^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。  
4.3 涡旋混合器。  
4.4 超声波清洗器。  
4.5 离心机： $\geqslant 6000\mathrm{r / min}$   
4.6 旋转蒸发仪。  
4.7 固相萃取装置（带真空泵）。  
4.8 氮吹仪。  
4.9 液相色谱-串联质谱仪：带电喷雾离子源。  
4.10 圆孔筛： $1\mathrm{mm}\sim 2\mathrm{mm}$ 孔径。  
4.11 玻璃纤维滤纸：快速，高载量，液体中颗粒保留 $1.6\mu \mathrm{m}$   
4.12一次性微孔滤头：带 $0.22\mu \mathrm{m}$ 微孔滤膜（所选用滤膜应采用标准溶液检验确认无吸附现象，方可

使用)。

4.13 免疫亲和柱：柱容量 $\geqslant 100\mathrm{ng}$ （柱容量、回收率、柱回收率验证方法参见附录B）。

注：对于每个批次的亲和柱在使用前需进行质量验证。

# 5 分析步骤

使用不同厂商的免疫亲和柱，在样品的上样、淋洗和洗脱的操作方面可能略有不同，应该按照供应商所提供的操作说明书要求进行操作。

警示：整个分析操作过程应在指定区域内进行。该区域应避光（直射阳光），具备相对独立的操作台和废弃物存放装置。在整个实验过程中，操作者应按照接触剧毒物的要求采取相应的保护措施。

# 5.1 样品提取

# 5.1.1 液态乳、酸奶

称取 $4\mathrm{g}$ 混合均匀的试样(精确到 $0.001\mathrm{g}$ )于 $50~\mathrm{mL}$ 离心管中，加入 $100\mu \mathrm{L}^{13}\mathrm{C}_{17} - \mathrm{AFT}\mathrm{M}_1$ 内标溶液 $(5\mathrm{ng / mL})$ 振荡混匀后静置 $30\mathrm{min}$ ，加入 $10~\mathrm{mL}$ 甲醇，涡旋 $3\mathrm{min}$ 。置于 $4^{\circ}\mathrm{C},6000\mathrm{r / min}$ 下离心 $10\mathrm{min}$ 或经玻璃纤维滤纸过滤，将适量上清液或滤液转移至烧杯中，加 $40~\mathrm{mL}$ 水或PBS稀释，备用。

# 5.1.2 乳粉、特殊膳食用食品

称取 $1\mathrm{g}$ 样品（精确到 $0.001\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $100\mu \mathrm{L}^{13}\mathrm{C}_{17} - \mathrm{AFTM}_1$ 内标溶液（ $5\mathrm{ng}/$ mL)振荡混匀后静置 $30\mathrm{min}$ ，加入 $4\mathrm{mL}50^{\circ}\mathrm{C}$ 热水，涡旋混匀。如果乳粉不能完全溶解，将离心管置于 $50^{\circ}\mathrm{C}$ 的水浴中，将乳粉完全溶解后取出。待样液冷却至 $20^{\circ}\mathrm{C}$ 后，加入 $10~\mathrm{mL}$ 甲醇，涡旋 $3\mathrm{min}$ 。置于 $4^{\circ}\mathrm{C},6000\mathrm{r / min}$ 下离心 $10\mathrm{min}$ 或经玻璃纤维滤纸过滤，将适量上清液或滤液转移至烧杯中，加 $40~\mathrm{mL}$ 水或PBS稀释，备用。

# 5.1.3 奶油

称取 $1\mathrm{g}$ 样品（精确到 $0.001\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $100\mu \mathrm{L}^{13}\mathrm{C}_{17}\mathrm{-AFTM}_1$ 内标溶液（ $5\mathrm{ng}/$ mL)振荡混匀后静置 $30\mathrm{min}$ ，加入 $8\mathrm{mL}$ 石油醚，待奶油溶解，再加 $9\mathrm{mL}$ 水和 $11\mathrm{mL}$ 甲醇，振荡 $30\mathrm{min}$ ，将全部液体移至分液漏斗中。加入 $0.3\mathrm{g}$ 氯化钠充分摇动溶解，静置分层后，将下层移到圆底烧瓶中，旋转蒸发至 $10~\mathrm{mL}$ 以下，用PBS稀释至 $30~\mathrm{mL}$

# 5.1.4 奶酪

称取 $1\mathrm{g}$ 已切细、过孔径 $1\mathrm{mm}\sim 2\mathrm{mm}$ 圆孔筛混匀样品（精确到 $0.001\mathrm{g}$ )于 $50~\mathrm{mL}$ 离心管中，加 $100\mu \mathrm{L}^{13}\mathrm{C}_{17} - \mathrm{AFT}\mathrm{M}_1$ 内标溶液 $(5\mathrm{ng / mL})$ 振荡混匀后静置 $30\mathrm{min}$ ，加入 $1\mathrm{mL}$ 水和 $18~\mathrm{mL}$ 甲醇，振荡 $30\mathrm{min}$ ，置于 $4^{\circ}\mathrm{C},6000\mathrm{r / min}$ 下离心 $10\mathrm{min}$ 或经玻璃纤维滤纸过滤，将适量上清液或滤液转移至圆底烧瓶中，旋转蒸发至 $2\mathrm{mL}$ 以下，用PBS稀释至 $30~\mathrm{mL}$ 。

# 5.2 净化

# 5.2.1 免疫亲和柱的准备

将低温下保存的免疫亲和柱恢复至室温。

# 5.2.2 净化

免疫亲和柱内的液体放弃后，将上述样液移至 $50~\mathrm{mL}$ 注射器筒中，调节下滴流速为 $1\mathrm{mL / min}\sim$

$3 \mathrm{~mL} / \mathrm{min}$ 。待样液滴完后, 往注射器筒内加入 $10 \mathrm{~mL}$ 水, 以稳定流速淋洗免疫亲和柱。待水滴完后,用真空泵抽干亲和柱。脱离真空系统, 在亲和柱下放置 $10 \mathrm{~mL}$ 刻度试管, 取下 $50 \mathrm{~mL}$ 的注射器筒, 加入 $2 \times 2 \mathrm{~mL}$ 乙腈(或甲醇)洗脱亲和柱, 控制 $1 \mathrm{~mL} / \mathrm{min} \sim 3 \mathrm{~mL} / \mathrm{min}$ 下滴速度, 用真空泵抽干亲和柱, 收集全部洗脱液至刻度试管中。在 $50^{\circ} \mathrm{C}$ 下氮气缓缓地将洗脱液吹至近干, 用初始流动相定容至 $1.0 \mathrm{~mL}$ , 涡旋 $30 \mathrm{~s}$ 溶解残留物, $0.22 \mu \mathrm{m}$ 滤膜过滤, 收集滤液于进样瓶中以备进样。

注：全自动(在线)或半自动(离线)的固相萃取仪器可优化操作参数后使用。为防止黄曲霉毒素M破坏，相关操作在避光(直射阳光)条件下进行。

# 5.6 标准曲线的制作

在5.3、5.4液相色谱-串联质谱仪分析条件下，将标准系列溶液由低到高浓度进样检测，以 $\mathrm{AFT~M_1}$ 和 $\mathrm{AFT~M_2}$ 色谱峰与内标色谱峰 $^{13}\mathrm{C}_{17}-\mathrm{AFT~M_1}$ 的峰面积比值-浓度作图，得到标准曲线回归方程，其线性相关系数应大于0.99。

# 5.7 试样溶液的测定

取5.2下处理得到的待测溶液进样，内标法计算待测液中目标物质的质量浓度，按第6章计算样品中待测物的含量。

# 5.8 空白试验

不称取试样，按5.1和5.2的步骤做空白实验。应确认不含有干扰待测组分的物质。

# 6 分析结果的表述

试样中 $\mathrm{AFTM_1}$ 或 $\mathrm{AFTM_2}$ 的残留量按式(1)计算：

$$
X = \frac {\rho \times V \times f \times 1 0 0 0}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中 $\mathrm{AFTM_1}$ 或 $\mathrm{AFTM_2}$ 的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——进样溶液中 $\mathrm{AFTM_1}$ 或 $\mathrm{AFTM_2}$ 按照内标法在标准曲线中对应的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

V ——样品经免疫亲和柱净化洗脱后的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——样液稀释因子；

1000——换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

# 酶联免疫试剂盒的质量判定方法

选取牛奶或其他阴性样品，根据所购酶联免疫试剂盒的检出限，在阴性基质中添加3个浓度水平的 $\mathrm{AFT~M_1}$ 标准溶液 $(0.1~\mu \mathrm{g / kg},0.3~\mu \mathrm{g / kg},0.5~\mu \mathrm{g / kg})$ 。按照说明书操作方法，用读数仪度数，做三次平行实验。针对每个加标浓度，回收率在 $50\% \sim 120\%$ 容许范围内的该批次产品方可使用。

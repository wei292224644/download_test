# 附录A

# 检验方法

# A.1 警示

本检验方法中使用的部分试剂具有毒性或腐蚀性，操作者应小心谨慎。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和符合GB/T6682规定的三级水。试验中所用标准溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定配制。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸。

A.3.1.2 氨水溶液： $2 + 3$

A.3.1.3 硝酸银氨溶液: 称取 $1 \mathrm{~g}$ 硝酸银, 加 $20 \mathrm{~mL}$ 水溶解, 在不断搅拌下加氨水溶液至最初形成的沉淀全部溶解, 过滤。

A.3.1.4 硝酸溶液： $1 + 9$

A.3.1.5钼酸铵溶液：称取 $7.5\mathrm{g}$ 钼酸铵，用 $20~\mathrm{mL}$ 水溶解，在搅拌下缓慢地加入 $30~\mathrm{mL}$ 硝酸溶液和 $30~\mathrm{mL}$ 水，静置 $24\mathrm{h}$ ，用玻璃砂坩埚过滤。

# A.3.2 鉴别方法

# A.3.2.1 钠离子鉴别方法

取铂丝，用盐酸润湿后，先在无色火焰上燃烧至无色，再蘸取试样在无色火焰中燃烧，火焰即显鲜黄色。

# A.3.2.2 磷酸盐鉴别方法

A.3.2.2.1 方法一：取试样的中性溶液，加硝酸银氨溶液即产生黄色沉淀，在氨水溶液中或在硝酸溶液中均易溶解。  
A.3.2.2.2 方法二：取试样溶液，加钼酸铵溶液和硝酸溶液即产生黄色沉淀，分离，沉淀能在氨溶液中溶解。

# A.4 总磷酸盐（以 $\mathbf{P}_2\mathbf{O}_5$ 计）的测定

# A.4.1 方法提要

在酸性溶液中试样全部水解为正磷酸盐,加入喹钼柠酮溶液后生成磷钼酸喹啉沉淀,过滤、洗涤、

称量。

# A.4.2 试剂和材料

A.4.2.1 硝酸溶液： $1 + 1$   
A.4.2.2 喹钼柠酮溶液：按HG/T3696.3进行配制。

# A.4.3 仪器和设备

玻璃砂坩埚：滤板孔径为 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$

# A.4.4 分析步骤

# A.4.4.1 试样溶液的制备

称取约 $2\mathrm{g}$ 试样(精确至 $0.0002\mathrm{g}$ )，置于 $100~\mathrm{mL}$ 烧杯中加水溶解，全部转移到 $500~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。必要时过滤。此试样溶液用于总磷酸盐、非活性磷酸盐的测定。

# A.4.4.2 测定

用移液管移取 $10\mathrm{mL}$ 试样溶液，置于 $400~\mathrm{mL}$ 高型烧杯中，加入 $15\mathrm{mL}$ 硝酸溶液和 $70~\mathrm{mL}$ 水，微沸 $15\mathrm{min}$ ，趁热加入 $50~\mathrm{mL}$ 喹钼柠酮溶液，盖上表面皿，保温 $10\mathrm{min}$ （在加入试剂和加热过程中，不得使用明火，不得搅拌，以免凝结成块），冷却至室温。用已在 $180^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}$ 下干燥约 $45\mathrm{min}$ 至恒量的玻璃砂坩埚以倾泻法过滤，在烧杯中洗涤沉淀三次，每次用水 $15~\mathrm{mL}$ ，将沉淀移入玻璃砂坩埚中，继续用水洗涤（所用洗涤水共约 $150~\mathrm{mL}$ ），于 $180^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}$ 下干燥约 $45\mathrm{min}$ 或于 $250^{\circ}\mathrm{C}\pm 5^{\circ}\mathrm{C}$ 下干燥约 $30\mathrm{min}$ ，至恒量，在干燥器中冷却，称重。

# A.4.5 结果计算

总磷酸盐（以 $\mathrm{P_2O_5}$ 计）的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {m _ {1} \times 0.03207}{m _ {2} \times \frac {10}{500}} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$m_{1}$ ——磷钼酸喹啉沉淀的质量，单位为克(g)；

0.032 07——磷钼酸喹啉与五氧化二磷的换算系数；

$m_{2}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$\frac{10}{500}$ ——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A.5 非活性磷酸盐（以 $\mathbf{P}_2\mathbf{O}_5$ 计）的测定

# A.5.1 方法提要

在试样溶液中加入氯化钡，与六偏磷酸钠生成沉淀，过滤。在滤液中加入酸，使其余磷酸盐水解为正磷酸盐，加入喹钼柠酮溶液后生成磷钼酸喹啉沉淀，过滤、洗涤、称量。

# A.5.2 试剂和材料

A.5.2.1 硝酸溶液： $1 + 1$   
A.5.2.2 喹钼柠酮溶液：按HG/T3696.3进行配制。  
A.5.2.3 氯化钡 $(\mathrm{BaCl}_2\cdot 2\mathrm{H}_2\mathrm{O})$ 溶液： $250~\mathrm{g / L}$

# A.5.3 仪器和设备

玻璃砂坩埚：滤板孔径为 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$

# A.5.4 分析步骤

用移液管移取 $50~\mathrm{mL}$ 试样溶液(A.4.4.1)，置于 $100~\mathrm{mL}$ 容量瓶中。在不断摇动下加入 $30~\mathrm{mL}$ 氯化钡溶液，充分摇动使沉淀完全，用水稀释至刻度，摇匀，干过滤。用移液管移取 $50~\mathrm{mL}$ 滤液，置于400 $\mathrm{mL}$ 高型烧杯中，加入 $15~\mathrm{mL}$ 硝酸溶液和 $35~\mathrm{mL}$ 水，微沸 $15\mathrm{min}$ ，趁热加入 $20~\mathrm{mL}$ 喹钼柠酮溶液，微沸 $1\mathrm{min}$ ，冷却至室温。

用已恒量的玻璃砂坩埚以倾析法过滤，在烧杯中洗涤沉淀三次，每次用水 $15\mathrm{mL}$ ，将沉淀移入玻璃砂坩埚中，继续用水洗涤（所用洗涤水共约 $150~\mathrm{mL}$ ），于 $180^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 下干燥约 $45\mathrm{min}$ 或于 $250^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 下干燥约 $30\mathrm{min}$ ，至恒量，在干燥器中冷却，称重。

# A.5.5 结果计算

非活性磷酸盐(以 $\mathrm{P}_2\mathrm{O}_5$ 计)的质量分数 $\varpi_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m _ {3} \times 0.03207}{m _ {4} \times \frac {50}{500}} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$m_{3}$ ——磷钼酸喹啉沉淀的质量，单位为克(g)；

0.032 07——磷钼酸喹啉与五氧化二磷的换算系数；

$m_{4}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

50 一换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A.6 水不溶物的测定

# A.6.1 仪器和设备

玻璃砂坩埚：滤板孔径为 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$

# A.6.2 分析步骤

称取约 $30\mathrm{g}$ 研磨后的试样（精确至 $0.01\mathrm{g}$ ），置于 $400\mathrm{mL}$ 烧杯中，加入 $200\mathrm{mL}$ 水，加热至沸腾，充分溶解，趁热用已于 $105~^\circ \mathrm{C}\sim 110~^\circ \mathrm{C}$ 下干燥至恒重的玻璃砂坩埚过滤，用热水洗涤10次（每次用水 $20~\mathrm{mL}$ ），在 $105~^\circ \mathrm{C}\sim 110~^\circ \mathrm{C}$ 下干燥至恒量。

# A.6.3 结果计算

水不溶物的质量分数 $\varpi_{3}$ ，按式(A.3)计算：

$$
w _ {3} = \frac {m _ {5} - m _ {6}}{m _ {7}} \times 100 \% \tag{A.3}
$$

式中：

$m_{5}$ ——水不溶物和玻璃砂坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{6}$ ——玻璃砂坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{7}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.01\%$ 。

# A.7 铁 $(\mathbf{Fe})$ 的测定

# A.7.1 方法提要

用抗坏血酸将试样溶液中三价铁离子还原成二价铁离子，在 $\mathrm{pH}$ 为 $2\sim 9$ 时，二价铁离子与邻菲啰啉生成橙红色络合物，用分光光度计在最大吸收波长 $510~\mathrm{nm}$ 处测量吸光度。

# A.7.2 试剂和材料

按照GB/T3049—2006中的第4章。

# A.7.3 仪器和设备

按照GB/T3049—2006中的第5章。

# A.7.4 分析步骤

# A.7.4.1 工作曲线的绘制

按GB/T3049—2006中6.3的规定，使用 $4\mathrm{cm}$ 或 $5\mathrm{cm}$ 吸收池及相应的铁标准溶液，绘制工作曲线。

# A.7.4.2 试样溶液的制备

称取约 $3\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），置于 $250~\mathrm{mL}$ 烧杯中，用少量水润湿，加盖表面皿后沿烧杯壁缓慢加入 $10~\mathrm{mL}$ 盐酸，使试样完全溶解。加入 $50~\mathrm{mL}$ 水，加热微沸 $15\mathrm{min}$ ，冷却，全部转移入 $250~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。准确移取 $25~\mathrm{mL}$ 此溶液置于 $100~\mathrm{mL}$ 容量瓶中，备用。

# A.7.4.3 空白溶液的制备

除不加试样外，加入的其他试剂量与试样溶液的制备完全相同，并同时处理。

# A.7.4.4 试样测定

在装有试样溶液和空白溶液的容量瓶中，加水至 $60~\mathrm{mL}$ ，以下操作按GB/T3049—2006中6.4的规定，从“用氨水溶液或盐酸溶液调节 $\mathrm{pH}$ 为 $2$ ”开始，至“测定两个试液的吸光度”为止。

选用与A.7.4.1中规格相同的吸收池（ $4\mathrm{cm}$ 或 $5\mathrm{cm}$ 吸收池），按GB/T3049—2006中6.4的规定测量吸光度，从工作曲线上查出试样溶液和空白溶液中铁的质量。

# A.7.5 结果计算

铁(Fe)的质量分数 $\varpi_{4}$ ，按式(A.4)计算：

$$
w _ {4} = \frac {m _ {8} - m _ {9}}{m _ {10} \times \frac {25}{250} \times 10 ^ {6}} \times 100 \% \quad \dots \tag{A.4}
$$

式中：

$m_{8}$ ——从工作曲线上查得的试样溶液中铁的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_{9}$ ——从工作曲线上查得的空白溶液中铁的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_{10}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$\frac{25}{250}$ ——换算系数；

$10^{6}$ ——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.001\%$ 。

# A.8 $\mathbf{pH}$ 的测定

# A.8.1 仪器和设备

酸度计：精度 $0.02\mathrm{pH}$ 。配有饱和甘汞电极和玻璃电极。

# A.8.2 分析步骤

称取 $1.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $250~\mathrm{mL}$ 烧杯中，用 $100~\mathrm{mL}$ 不含二氧化碳的水溶解。在室温下用酸度计，以饱和甘汞电极作参比电极，以玻璃电极作测量电极测定溶液的 $\mathrm{pH}$ 。

# A.9 氟化物（以F计）的测定

# A.9.1 方法提要

试样溶解后，在 $\mathrm{pH}5.5\sim 6.0$ 的酸性介质中，以饱和甘汞电极为参比电极，以氟离子选择电极为测量电极，用工作曲线法测量氟含量。

# A.9.2 试剂和材料

A.9.2.1 盐酸溶液： $1 + 4$   
A.9.2.2 调节总离子强度缓冲溶液（TISAB）：临用时配制。  
A.9.2.3 氟标准溶液： $1\mathrm{mL}$ 含氟 $(\mathrm{F})10~\mu \mathrm{g}$

# A.9.3 仪器和设备

A.9.3.1 氟电极。  
A.9.3.2 甘汞电极。  
A.9.3.3 电位计。  
A.9.3.4 磁力搅拌器。

# A.9.4 分析步骤

# A.9.4.1 氟标准工作溶液的配制

在5个 $50~\mathrm{mL}$ 容量瓶中，用移液管分别加入 $1.00~\mathrm{mL},2.00~\mathrm{mL},3.00~\mathrm{mL},4.00~\mathrm{mL},5.00~\mathrm{mL}$ 氟标准溶液，准确加入 $4\mathrm{mL}$ 盐酸溶液，加 $25~\mathrm{mL}$ 调节总离子强度缓冲溶液（TISAB)，用水稀释至刻度，摇匀。

# A.9.4.2 试样溶液的配制

称取 $1\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），准确加入 $4\mathrm{mL}$ 盐酸溶液和 $25\mathrm{mL}$ 总离子强度缓冲溶液溶解，置于 $50\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。

# A.9.4.3 测定

将氟电极和甘汞电极与电位计的负端和正端连接，将电极插入盛有水的 $50~\mathrm{mL}$ 的聚乙烯烧杯中，预热仪器，在磁力搅拌器上恒速搅拌，读取平衡电位值。更换浸泡电极的水，至电极说明书中规定的电位值后，即可进行标准工作溶液和试样溶液的电位测定。

由低至高浓度分别测定氟标准工作溶液的平衡电位。以电极电位作纵坐标，氟离子浓度 $(\mu \mathrm{g} / \mathrm{mL})$ 作横坐标，在半对数坐标纸上绘制标准曲线。

同样方法测定试样溶液的平衡电位，从标准曲线上查出试样的氟含量。

# A.9.5 结果计算

氟(F)的质量分数 $\omega_{5}$ ，按式(A.5)计算：

$$
w_{5} = \frac{m_{11}}{m_{12}\times 10^{6}}\times 100\% \tag{A.5}
$$

式中：

$m_{11}$ ——从工作曲线上查得的试样溶液中氟的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_{12}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$10^{6}$ ——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.01\%$ 。

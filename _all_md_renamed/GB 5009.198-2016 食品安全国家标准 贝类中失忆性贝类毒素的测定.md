2016-12-23 发布

# 前言

本标准代替GB/T5009.198—2003《贝类 记忆丧失性贝类毒素软骨藻酸的测定》、SN/T1070—2002《进出口贝类中记忆丧失性贝类毒素检验方法》、SN/T1867—2007《进出口贝类中软骨藻酸的检测方法 液相色谱-串联质谱法》、SN/T2663—2010《贝类中失忆性贝类毒素检验方法 酶联免疫吸附法》。

本标准与GB/T5009.198—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 贝类中失忆性贝类毒素的测定”；  
——修改了固相萃取条件；  
——改变了流动相；  
——增加了酶联免疫吸附法；  
——增加了液相色谱-串联质谱法。

# 食品安全国家标准

# 贝类中失忆性贝类毒素的测定

# 1 范围

本标准规定了贝类中失忆性贝类毒素测定的酶联免疫吸附法、液相色谱法和液相色谱-串联质谱法。

本标准中酶联免疫吸附法适用于贝类及其制品中失忆性贝类毒素的测定，液相色谱法和液相色谱-串联质谱法适用于贝类及其制品(不包括盐渍制品)中失忆性贝类毒素软骨藻酸(DA)的测定。

# 酶联免疫吸附法

# 2 原理

本方法测定基础是竞争性酶联免疫反应，酶标板上包被有针对软骨藻酸抗体的捕捉抗体，加入抗软骨藻酸抗体、标准液或样品溶液及软骨藻酸酶标记物，游离的失忆性贝类毒素与软骨藻酸酶标记物竞争软骨藻酸抗体，同时软骨藻酸抗体与捕捉抗体连接。没有结合的酶标记物在洗涤步骤中被除去。将酶基质和显色剂加入到微孔中并且孵育。结合的酶标记物将无色的发色剂转化为蓝色的产物。加入反应终止液后使颜色由蓝转变为黄色。在 $450\mathrm{nm}$ 测量微孔溶液的吸光度值，试样中失忆性贝类毒素的含量与吸光度值成反比，按绘制的标准曲线定量计算。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。  
3.1.2 十二水合磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4\cdot 12\mathrm{H}_2\mathrm{O})$   
3.1.3 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.4 氯化钾(KCl)。  
3.1.5 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
3.1.6 吐温-20 $(\mathrm{C}_{58}\mathrm{H}_{114}\mathrm{O}_{26})$ 。  
3.1.7 抗DA抗体。  
3.1.8 牛血清白蛋白（BSA）。  
3.1.9 DA酶标记物。  
3.1.10 过氧化氢 $(\mathrm{H}_2\mathrm{O}_2)$ 。  
3.1.11 3,3,5,5-四甲基联苯胺 $\mathrm{(TMB,C_{16}H_{20}N_2)}$   
3.1.12 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。

# 3.2 试剂配制

3.2.1 PBS溶液(pH7.4)：分别称取磷酸二氢钾 $0.20\mathrm{g}$ 、十二水合磷酸氢二钠 $2.90\mathrm{g}$ 、氯化钠 $8.00\mathrm{g}$ 、氯化钾 $0.20\mathrm{g}$ ，加水溶解并定容至 $1000\mathrm{mL}$ 。  
3.2.2 抗体稀释液：称取 $1.0\mathrm{g}$ BSA，加PBS溶液 $(\mathrm{pH}7.4)$ 溶解并定容至 $1000~\mathrm{mL}$ 。  
3.2.3 抗DA抗体工作液：抗DA抗体用抗体稀释液稀释至工作浓度。  
3.2.4 DA酶标记物工作液：用抗体稀释液将DA酶标记物稀释至工作浓度。  
3.2.5 洗脱液：吸取 $0.5\mathrm{mL}$ 吐温-20，用PBS溶液 $(\mathrm{pH}7.4)$ 稀释至 $1000~\mathrm{mL}$   
3.2.6 硫酸溶液 $(6\mathrm{mol} / \mathrm{L})$ ：吸取 $319.2\mathrm{mL}$ 硫酸，缓缓加至 $600~\mathrm{mL}$ 水中，并用水稀释至 $1000~\mathrm{mL}$ 。

# 3.3 标准品

软骨藻酸 $\mathrm{(DA,C_{15}H_{21}NO_6}$ ，CAS号14277-97-5)标准溶液。

# 3.4 标准溶液配制

DA标准系列工作液：准确吸取适量DA标准溶液用PBS溶液稀释并定容，配制成质量浓度分别为 $0\mathrm{ng / mL},0.5\mathrm{ng / mL},1.0\mathrm{ng / mL},2.0\mathrm{ng / mL},5.0\mathrm{ng / mL}$ 和 $10.0\mathrm{ng / mL}$ 的DA标准系列工作液。现用现配。

# 3.5 材料

3.5.1 包被有DA捕捉抗体的微孔板。

注：商业化试剂盒若评价技术参数达到本标准的要求则也适合于本标准，参见附录A。

3.5.2 水相微孔滤膜： $0.45\mu \mathrm{m}$

# 4 仪器和设备

4.1 酶标仪。  
4.2 天平：感量为 $0.01\mathrm{g}$ 。  
4.3 均质器。  
4.4 离心机：转速 $\geq 6000\mathrm{r / min}$

# 5 分析步骤

# 5.1 样品采集

至少采集10个贝类样品，并使贝肉达 $200\mathrm{g}$ 以上。冷冻样品置于保温盒中冷冻送检，或保证其处于低温状态 $(0^{\circ}\mathrm{C}\sim 10^{\circ}\mathrm{C})$ 送检。如为带壳样品，应开壳，去除水分后冷冻送检。

# 5.2 试样制备

# 5.2.1 生鲜带壳样品

用清水将贝类样品外表彻底洗净，切断闭壳肌，开壳，用水淋洗内部去除泥沙及其他外来物。将闭壳肌和连接在胶合部的组织分开，取出贝肉，切勿割破肉体。开壳前不要加热或用麻醉剂。收集 $100\mathrm{g}$ 贝肉置于孔径约 $2\mathrm{mm}$ 的金属筛网上，沥水 $5\mathrm{min}$ ，检出碎壳等杂物，将贝肉均质，备用。

# 5.2.2 冷冻样品

在室温下，使冷冻样品呈半冷冻状态。带壳冷冻样品，按5.2.1方法清洗、开壳、淋洗取肉，除去贝肉外部附着的冰片，抹去水分，室温缓化。将 $100\mathrm{g}$ 解冻的贝肉均质，备用。

# 5.2.3 贝类罐头

将贝类罐头内容物沥干水分，充分均质，备用。

# 5.2.4 贝肉干制品

称取 $100\mathrm{g}$ 贝肉干制品放入一定量水中浸泡 $24\mathrm{h} \sim 48\mathrm{h}(4^{\circ}\mathrm{C}$ 冷藏)，沥干、均质、备用。

# 5.2.5 盐渍制品

用清水洗涤，流水脱盐，沥干，均质，备用。

# 5.3 试样提取

称取 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ 试样，加入 $10~\mathrm{mL}$ 水，涡旋振荡 $1\mathrm{min}$ ，加入 $20~\mathrm{mL}$ 甲醇，涡旋振荡 $1\mathrm{min}$ $6000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，移取上清液，用 $0.45\mu \mathrm{m}$ 的水相微孔滤膜过滤，得到试样提取液，用PBS溶液(pH7.4)稀释100倍，得到试样稀释液。吸取 $50~\mu \mathrm{L}$ 试样稀释液进行测定。

# 5.4 测定

将已包被捕捉抗体的微孔条插入微孔架并做好标记，其中包括空白对照孔、标准液孔和样液孔，分别做平行孔。向空白对照孔加入 $150~\mu \mathrm{L}$ PBS溶液(pH7.4)，向标准液孔加入 $50~\mu \mathrm{L}$ 失忆性贝类毒素标准系列工作液，向样液孔加入 $50~\mu \mathrm{L}$ 样液。向上述所有微孔中加入 $100~\mu \mathrm{LDA}$ 酶标记物工作液，轻轻混合，再加入 $100~\mu \mathrm{LDA}$ 抗体工作液，迅速充分混合 $1\mathrm{min}$ ，用粘胶纸封住微孔以防溶液挥发， $4^{\circ}\mathrm{C}$ 孵育 $2\mathrm{h}$ 。孵育结束后，倒去孔中液体，每个微孔注入 $300~\mu \mathrm{L}$ 洗脱液冲洗，翻转微孔板，倾去孔内液体，再重复以上洗板操作2次，在吸水纸上拍干。每孔加 $150~\mu \mathrm{L}$ 过氧化氢和TMB充分混合，室温避光孵育 $30\mathrm{min}$ 。每孔加入 $50~\mu \mathrm{L}$ 硫酸溶液(6mol/L)迅速混匀，终止反应，在 $30\mathrm{min}$ 内测量并记录 $450~\mathrm{nm}$ 波长下的吸光度值。若试样稀释液经测定超出标准曲线的线性范围，可扩大稀释倍数后重新进行测定。

# 5.5 标准曲线的制作

以失忆性贝类毒素标准系列工作液的质量浓度的以10为底的对数值为横坐标，以按式(1)计算的标准液的百分比吸光度值为纵坐标，绘制标准曲线。

失忆性贝类毒素标准液(或样液)的百分比吸光度值按式(1)计算：

$$
A = \frac {S - S _ {1}}{S _ {0} - S _ {1}} \times 100 \% \tag{1}
$$

式中：

$A$ ——百分比吸光度值；

S——失忆性贝类毒素标准工作液或样液的平均吸光度值；

$S_{1}$ ——空白对照孔的平均吸光度值；

$S_{0}$ —— $0\mu \mathrm{g} / \mathrm{L}$ 的失忆性贝类毒素标准工作液的平均吸光度值。

# 6 分析结果的表述

试样中失忆贝类毒素的含量按式(2)计算：

$$
X = \frac {\rho \times V \times f}{m} \tag {2}
$$

式中：

$X$ ——试样中失忆性贝类毒素的含量，单位为微克每克 $(\mu \mathrm{g} / \mathrm{g})$ ；

$\rho$ ——由标准曲线得到的试样待测液中失忆性贝类毒素的质量浓度，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$V$ ——试样提取液的体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释倍数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

当测定值 $< 20~\mu \mathrm{g / g}$ 时，则报告失忆性贝类毒素含量 $< 20~\mu \mathrm{g / g}$ 。

当测定值 $\geqslant 20~\mu \mathrm{g / g}$ 时，则报告实际测定结果。

注：任何失忆性贝类毒素含量大于 $20~\mu \mathrm{g / g}$ 的样品即被认为是有害的，人类食用不安全。

# 7 其他

本方法的定量限为 $1\mu \mathrm{g / g}$

# 液相色谱法

# 8 原理

试样经甲醇溶液 $(50\%)$ 提取，强阴离子固相萃取柱净化，液相色谱分离，二极管阵列或紫外检测器检测，外标法定量。

# 9 试剂和材料

除非另有说明，本方法所用试剂均为色谱纯，水为GB/T6682规定的一级水。

# 9.1 试剂

9.1.1 乙腈 $\mathrm{(CH_3CN)}$ 。  
9.1.2 甲醇 $\mathrm{(CH_3OH)}$   
9.1.3 甲酸（HCOOH）。  
9.1.4 冰乙酸 $\mathrm{(CH_3COOH)}$

# 9.2 试剂配制

9.2.1 乙腈溶液 $(10\%)$ ：量取 $10~\mathrm{mL}$ 乙腈，用水稀释至 $100~\mathrm{mL}$ 。  
9.2.2 甲醇溶液 $(50\%)$ ：量取 $50~\mathrm{mL}$ 甲醇，用水稀释至 $100~\mathrm{mL}$ 。  
9.2.3 甲酸溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：吸取 $3.81~\mathrm{mL}$ 甲酸，用水稀释至 $1000~\mathrm{mL}$ 。  
9.2.4 乙酸溶液 $(0.1\%)$ ：吸取 $1\mathrm{mL}$ 冰乙酸，用水稀释至 $1000~\mathrm{mL}$ 。

# 9.3 标准品

软骨藻酸标准品 $\mathrm{(DA,C_{15}H_{21}NO_6}$ ，CAS号14277-97-5）：纯度 $\geqslant 90\%$

# 9.4 标准溶液配制

9.4.1 软骨藻酸标准储备液 $(150\mu \mathrm{g / mL})$ ：准确称取 $8.3\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ )软骨藻酸标准品，用乙腈溶液 $(10\%)$ 溶解并定容至 $50~\mathrm{mL}$ 。该软骨藻酸标准储备液浓度已经采用标准品的纯度进行换算，置于 $4^{\circ}C$ 避光保存，有效期6个月。  
9.4.2 软骨藻酸标准系列工作液：分别吸取适量软骨藻酸储备液 $(150\mu \mathrm{g / mL})$ ，用乙腈溶液 $(10\%)$ 分别稀释成浓度为 $0.3\mu \mathrm{g / mL}, 0.6\mu \mathrm{g / mL}, 3\mu \mathrm{g / mL}, 15\mu \mathrm{g / mL}, 30\mu \mathrm{g / mL}$ 的标准系列工作液。标准系列工作液于 $4^{\circ}\mathrm{C}$ 避光保存，有效期3个月。

# 9.5 材料

9.5.1 强阴离子固相萃取柱： $500\mathrm{mg} / 3\mathrm{mL}$ ，或性能相当者。使用前分别用 $6\mathrm{mL}$ 甲醇、 $3\mathrm{mL}$ 水和 $3\mathrm{mL}$ 甲醇溶液（ $50\%$ ）活化。  
9.5.2 微孔滤膜： $0.22\mu \mathrm{m}$ ，水相。

# 10 仪器和设备

10.1 液相色谱仪：配有二极管阵列或紫外检测器。  
10.2 分析天平：感量为 $0.01\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。  
10.3 均质器：转速 $\geqslant 10000\mathrm{r / min}$   
10.4 涡旋振荡器。  
10.5 离心机：转速 $\geq 8000\mathrm{r / min}$   
10.6 固相萃取装置。  
10.7 真空泵。

# 11 分析步骤

# 11.1 样品采集

同5.1。

# 11.2 试样制备

同 $5.2.1\sim 5.2.4$

# 11.3 试样提取

称取 $5\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）试样于 $50~\mathrm{mL}$ 具塞离心管中，加入 $10~\mathrm{mL}$ 甲醇溶液 $(50\%)$ ，涡旋混匀 $1\mathrm{min}$ ，超声提取 $5\mathrm{min}$ ，以 $8000\mathrm{r / min}$ 离心 $15\mathrm{min}$ ，移出上清液至 $25~\mathrm{mL}$ 容量瓶中，残渣再加入 $10~\mathrm{mL}$ 甲醇溶液 $(50\%)$ 重复提取一次，合并上清液，以甲醇溶液 $(50\%)$ 定容至 $25~\mathrm{mL}$ 。

# 11.4 试样净化

准确吸取 $5\mathrm{mL}$ 提取液移入已活化过的强阴离子固相萃取柱上，待液体以 $1\mathrm{mL} / \mathrm{min}$ 的流速流出后，再依次用 $5\mathrm{mL}$ 乙腈溶液 $(10\%)$ ， $0.5\mathrm{mL}$ 甲酸溶液 $(0.1\mathrm{mol} / \mathrm{L})$ 淋洗，保持抽气 $2\mathrm{min}$ ，弃去流出液，最后用 $3\mathrm{mL}$ 甲酸溶液 $(0.1\mathrm{mol} / \mathrm{L})$ 洗脱，保持抽气 $2\mathrm{min}$ ，收集洗脱液（ $3\mathrm{mL}$ 洗脱液相当于 $1\mathrm{g}$ 试样），过 $0.22\mu \mathrm{m}$ 水相微孔滤膜后，供液相色谱测定。

# 11.5 空白试验

除不加试样外，采用与试样相同的操作步骤，得到空白溶液。

# 11.6 仪器参考条件

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $150~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $3\mu \mathrm{m}$ ，或性能相当者；  
b）流动相：乙腈 $+$ 乙酸溶液 $(0.1\%)(13 + 87)$   
c) 流速: $1 \mathrm{~mL} / \mathrm{min}$ ;   
d） 柱温： $35^{\circ}C$   
e) 进样量: $10 \mu \mathrm{L}$ ;   
f） 测定波长： $242\mathrm{nm}$

# 11.7 标准曲线的制作

将标准系列工作液分别注入液相色谱仪中，测定相应的峰面积，以标准工作液的质量浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。软骨藻酸标准溶液的液相色谱图参见图B.1。

# 11.8 试样溶液的测定

将试样溶液注入液相色谱仪中，得到峰面积，根据标准曲线得到试样溶液中软骨藻酸的质量浓度。

# 12 分析结果的表述

试样中软骨藻酸的含量按式(3)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V}{m} \tag {3}
$$

式中：

$X$ ——试样中软骨藻酸的含量，单位为微克每克 $(\mu \mathrm{g} / \mathrm{g})$ ；

$\rho$ ——由标准曲线得到的试样溶液中软骨藻酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$\rho_0$ ——由标准曲线得到的空白溶液中软骨藻酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——洗脱液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——与洗脱液相当的试样质量，单位为克 $(\mathrm{g})$ 。

计算结果保留两位有效数字。

# 13 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 14 其他

本方法的检出限为 $0.3\mu \mathrm{g / g}$ ，定量限为 $1.0~\mu \mathrm{g / g}$

# 液相色谱-串联质谱法

# 15 原理

试样经甲醇溶液 $(50\%)$ 提取，强阴离子固相萃取柱净化，液相色谱-串联质谱检测，外标法定量。

# 16 试剂和材料

除非另有说明，本方法所用试剂均为色谱纯，水为GB/T6682规定的一级水。

# 16.1 试剂

16.1.1 甲醇 $\mathrm{(CH_3OH)}$ 。  
16.1.2 乙腈 $\mathrm{(CH_3CN)}$ 。  
16.1.3 甲酸（HCOOH）：分析纯。  
16.1.4 甲酸铵 $\mathrm{(HCOONH_4)}$ ：分析纯。

# 16.2 试剂配制

16.2.1 乙腈溶液 $(10\%)$ ：量取 $50~\mathrm{mL}$ 乙腈，用水稀释至 $500~\mathrm{mL}$ 。  
16.2.2 甲醇溶液 $(50\%)$ ：量取 $500~\mathrm{mL}$ 甲醇，用水稀释至 $1000~\mathrm{mL}$ 。  
16.2.3 甲酸铵缓冲溶液 $(2\mathrm{mmol} / \mathrm{L})$ ：称取 $0.126\mathrm{g}$ 甲酸铵，加入 $200~\mu \mathrm{L}$ 甲酸，用水溶解并稀释至 $1000~\mathrm{mL}$ 。

# 16.3 标准品

# 19 分析结果的表述

试样中软骨藻酸的含量按式(4)计算：

$$
X = \frac {\rho \times V}{m \times 1 0 0 0} \tag {4}
$$

式中：

$X$ ——试样中软骨藻酸的含量，单位为微克每克 $(\mu \mathrm{g} / \mathrm{g})$ ；

$\rho$ ——由标准工作曲线得到的试样溶液中软骨藻酸的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——洗脱液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——与洗脱液相当的试样质量，单位为克(g)；

1000 ——换算因子。

计算结果保留三位有效数字。计算结果应扣除空白值。

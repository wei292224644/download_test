# 前言

本标准代替GB30613—2014《食品安全国家标准 食品添加剂 磷酸氢二铵》。

本标准与GB30613—2014相比，主要变化如下：

——修改了范围；  
——将“无机砷(以As计)”修改为“砷（As）”；  
——将磷酸氢二铵含量检验方法修改为甲醛容量法；  
——将铅含量检验方法修改为GB5009.75或GB5009.12。

# 检验方法

警示：本检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应小心谨慎！必要时，应在通风橱中进行。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 硝酸溶液： $1 + 8$   
A.2.1.2 氨水溶液： $1 + 1$   
A.2.1.3 氢氧化钠溶液： $40~\mathrm{g / L}$   
A.2.1.4 硝酸银溶液： $17\mathrm{g / L}$   
A.2.1.5 红色石蕊试纸。

# A.2.2 鉴别方法

# A.2.2.1 磷酸根离子的鉴别

称取约 $1.0\mathrm{g}$ 试样，溶于 $20~\mathrm{mL}$ 水中，用硝酸溶液调节 $\mathrm{pH}$ 至中性。加入硝酸银溶液，可生成黄色沉淀，此沉淀溶于氨水溶液或硝酸溶液。

# A.2.2.2 铵根离子的鉴别

称取约 $1.0\mathrm{g}$ 试样，溶于 $20~\mathrm{mL}$ 水中，加入氢氧化钠溶液，释放出的氨气可使湿润的红色石蕊试纸变蓝。加热可促进试样分解。

# A.3 磷酸氢二铵 $\left[(\mathrm{NH}_4)_2\mathrm{HPO}_4\right]$ 含量的测定

# A.3.1 方法提要

试样溶解后，与甲醛作用生成相当于铵盐含量的酸，用氢氧化钠标准滴定溶液滴定，以酸度计指示终点，根据消耗氢氧化钠标准滴定溶液的体积，计算磷酸氢二铵含量。

# A.3.2 试剂和材料

A.3.2.1 甲醛溶液： $1 + 1$   
A.3.2.2 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 1\mathrm{mol} / \mathrm{L}$   
A.3.2.3 无二氧化碳的水。

# A.3.3 仪器和设备

A.3.3.1 酸度计：分辨率为 $0.01\mathrm{pH}$ ，配有玻璃电极和饱和甘汞电极（或复合电极）。  
A.3.3.2 电磁搅拌器：配有搅拌转子。

# A.3.4 分析步骤

称取约 $1.8\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $150~\mathrm{mL}$ 的烧杯中。加入 $50~\mathrm{mL}$ 无二氧化碳的水，置于电磁搅拌器上，放入搅拌转子，搅拌至试样溶解，加 $40~\mathrm{mL}$ 新制备的甲醛溶液，搅拌均匀后放置 $30\mathrm{min}$ 。将已校准的酸度计的电极放入试样溶液中，用氢氧化钠标准滴定溶液滴定至 $\mathrm{pH}8.9$ 为终点，记录消耗氢氧化钠标准滴定溶液的体积。

同时做空白试验。除不加试样外，其他加入试剂的种类和量(标准滴定溶液除外)与试样溶液相同。

# A.3.5 结果计算

磷酸氢二铵 $\left[\mathrm{(NH_4)_2HPO_4}\right]$ 含量的质量分数 $w_{1}$ 按式(A.1)计算。

$$
w _ {1} = \frac {c \times \left(V _ {1} - V _ {2}\right) \times M}{m \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$c$ ——氢氧化钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{1}$ ——滴定试样溶液所消耗的氢氧化钠标准滴定溶液的体积,单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴定空白试样溶液所消耗的氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$M$ ——磷酸氢二铵 $\left[1 / 2\left(\mathrm{NH}_{4}\right)_{2} \mathrm{HPO}_{4}\right]$ 的摩尔质量，单位为克每摩尔（g/mol）（ $M = 66.03$ ）；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A.4 $\mathbf{pH}(10\mathrm{g} / \mathrm{L}$ 水溶液）的测定

# A.4.1 试剂和材料

无二氧化碳的水。

# A.4.2 仪器和设备

酸度计：分辨率为 $0.01\mathrm{pH}$ ，配有玻璃电极和饱和甘汞电极（或复合电极）。

# A.4.3 分析步骤

称取 $1.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $100~\mathrm{mL}$ 烧杯中，用无二氧化碳的水溶解，移入 $100~\mathrm{mL}$ 容量瓶中，用无二氧化碳的水稀释至刻度，摇匀。倒入干燥烧杯中，用已校准的酸度计测量试样溶液的 $\mathrm{pH}$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于0.1。

# A.5 铅(Pb)的测定

按GB5009.75或GB5009.12规定的方法进行测定，试验中所用水为GB/T6682规定的二级水。

# A.6 砷（As）的测定

按GB5009.76或GB5009.11规定的方法进行测定，试验中所用水为GB/T6682规定的二级水。

# A.7 重金属（以 $\mathbf{Pb}$ 计）的测定

按GB5009.74规定的方法进行测定，试验中所用水为 $\mathrm{GB / T6682}$ 规定的二级水。

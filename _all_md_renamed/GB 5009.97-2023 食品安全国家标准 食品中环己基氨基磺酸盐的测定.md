# 前言

本标准代替GB5009.97—2016《食品安全国家标准 食品中环己基氨基磺酸钠的测定》。

本标准与GB5009.97—2016相比，主要变化如下：

——将标准名称改为《食品安全国家标准 食品中环己基氨基磺酸盐的测定》；  
——修改了适用范围；  
——修改了前处理方法；  
——修改了仪器条件及参数；  
——修改了第三法液相色谱-质谱/质谱法的定量方式为内标法。

# 食品安全国家标准

# 食品中环己基氨基磺酸盐的测定

# 1 范围

本标准规定了食品中环己基氨基磺酸盐的测定方法。

第一法气相色谱法适用于食品（蒸馏酒、发酵酒、配制酒、料酒及其他含乙醇的食品除外）中环己基氨基磺酸盐的测定。

第二法液相色谱法适用于食品中环己基氨基磺酸盐的测定。

第三法液相色谱-质谱/质谱法适用于食品中环己基氨基磺酸盐的测定。

# 第一法 气相色谱法

# 2 原理

试样中的环己基氨基磺酸盐经水提取，在硫酸介质中与亚硝酸钠反应，生成环己醇亚硝酸酯和环己醇，用正庚烷萃取后，用气相色谱-氢火焰离子化检测器测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 正庚烷 $\left[\mathrm{CH}_{3}\left(\mathrm{CH}_{2}\right)_{5} \mathrm{CH}_{3}\right]$ ：色谱纯。  
3.1.2 石油醚：沸程为 $30^{\circ}\mathrm{C} \sim 60^{\circ}\mathrm{C}$ 。  
3.1.3 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ ， $95.0\% \sim 98.0\%$ （质量分数），密度： $1.84~\mathrm{g / mL}$   
3.1.4 亚硝酸钠 $(\mathrm{NaNO}_2)$   
3.1.5 淀粉酶：酶活力 $\geqslant 1500\mathrm{U / g}$   
3.1.6 无水乙醇 $\mathrm{(CH_3CH_2OH)}$ 。

# 3.2 试剂配制

3.2.1 亚硝酸钠溶液 $(50\mathrm{g} / \mathrm{L})$ ：称取 $50\mathrm{g}$ 亚硝酸钠，溶于水并稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.2 硫酸溶液 $(200\mathrm{g / L})$ ：量取 $108~\mathrm{mL}$ 硫酸，缓慢加入 $800~\mathrm{mL}$ 水中，不断搅拌避免局部过热，冷却后加水稀释至 $1000~\mathrm{mL}$ ，混匀。

# 3.3 标准品

环己基氨基磺酸钠标准品 $\mathrm{(C_6H_{12}NSO_3Na)}$ ：纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 环己基氨基磺酸标准储备液 $(6\mathrm{mg / mL})$ : 准确称取适量环己基氨基磺酸钠标准品 (精确至 $0.1\mathrm{mg}$ ) 于 $100\mathrm{mL}$ 容量瓶中, 用水溶解并定容至刻度, 混匀 (环己基氨基磺酸钠折算为环己基氨基磺酸的换算系数为 0.8907)。将溶液转移至棕色玻璃容器中, $4^{\circ}\mathrm{C}$ 下避光保存, 有效期 12 个月。  
3.4.2 环己基氨基磺酸标准中间液 $(1200\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取环己基氨基磺酸标准储备液 $10~\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。将溶液转移至棕色玻璃容器中， $4^{\circ}\mathrm{C}$ 下避光保存，有效期6个月。

# 3.5 材料

3.5.1 离心管： $50~\mathrm{mL}$ 具螺纹盖刻度离心管。  
3.5.2 有机相微孔滤膜： $0.45\mu \mathrm{m}$ 孔径。

# 4 仪器和设备

4.1 气相色谱仪：配有氢火焰离子化检测器(FID)。  
4.2 分析天平：感量 $1\mathrm{mg},0.1\mathrm{mg}$   
4.3 涡旋混合器。  
4.4 高速离心机。  
4.5 超声波发生器。  
4.6 粉碎机。  
4.7 研磨机。  
4.8 匀浆机。  
4.9 恒温水浴装置。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 液体试样

直接摇匀（含二氧化碳的试样，超声除去气泡后摇匀）。

# 5.1.2 非均匀的液态、半固态试样

取适量试样，经匀浆机匀浆后混匀。

# 5.1.3 巧克力、巧克力制品、奶酪、黄油等试样

取适量试样， $-18^{\circ}\mathrm{C}$ 冷冻后，粉碎机粉碎并搅拌均匀。

# 5.1.4 固态试样

取适量试样，用研磨机或粉碎机粉碎并搅拌均匀。

# 5.2 试样提取

# 5.2.1 巧克力、奶油、奶酪、乳粉、调味面制品、油腐乳、油豆豉、肉及肉制品、水产品罐头等含较高油脂试样

称取 $2\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ ) 于离心管中，加入 $20~\mathrm{mL}$ 石油醚，涡旋 $5\mathrm{min}$ ，超声 $10\mathrm{min}$ ， $5000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，弃去石油醚层，再加入 $20~\mathrm{mL}$ 石油醚提取1次，弃去石油醚层， $60^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 水浴挥去残留石油醚，加入 $20~\mathrm{mL}$ 水（巧克力、奶油、奶酪、乳粉等试样需在水浴锅中 $60^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 加热 $20\mathrm{min}$ ），涡旋 $5\mathrm{min}$ ，超声提取 $30\mathrm{min}$ ，混匀后放至室温备用。

# 5.2.2 果冻、糖果、米粉、淀粉制品等试样

称取 $2\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )于离心管中，加入 $20~\mathrm{mL}$ 水（米粉、淀粉制品等试样需再加入 $0.2\mathrm{g}$ 淀粉酶)，混合均匀后， $60^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 水浴加热 $20\mathrm{min}$ ，涡旋 $5\mathrm{min}$ ，超声提取 $10\mathrm{min}$ ，混匀后放至室温备用。

# 5.2.3 其他固体、半固体试样

称取 $2\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ ）于离心管中，加入 $20~\mathrm{mL}$ 水，涡旋 $5\mathrm{min}$ ，超声提取 $30\mathrm{min}$ ，混匀后放至室温备用。

# 5.2.4 液体试样

称取 $2\mathrm{g}$ 试样（精确至 $0.001\mathrm{g}$ ）于离心管中，加水至 $20\mathrm{mL}$ ，涡旋 $5\mathrm{min}$ ，超声提取 $10\mathrm{min}$ ，混匀后放至室温备用。

# 5.3 衍生化

将装有试样提取液的离心管置于冰浴 $10\mathrm{min}$ 后，依次加入 $10~\mathrm{mL}$ 正庚烷、 $5\mathrm{mL}$ 亚硝酸钠溶液 $(50~\mathrm{g / L})$ 、 $5\mathrm{mL}$ 硫酸溶液 $(200~\mathrm{g / L})$ ，混匀，于冰浴中放置 $30\mathrm{min}$ ，其间振摇3次～5次，取出后涡旋 $3\mathrm{min}$ ，于 $4^{\circ}C$ 条件下 $9000\mathrm{r / min}$ 离心 $3\mathrm{min}$ （如出现乳化现象，可缓慢滴加无水乙醇，同时轻摇离心管，直至破乳，于 $4^{\circ}C$ 条件下 $9000\mathrm{r / min}$ 离心 $3\mathrm{min}$ )，取上清液过有机相微孔滤膜，供上机测定。

# 5.4 标准系列工作液的制备及衍生化

分别准确移取 $0.05\mathrm{mL}, 0.10\mathrm{mL}, 0.25\mathrm{mL}, 0.50\mathrm{mL}, 1.00\mathrm{mL}, 2.5\mathrm{mL}, 5.0\mathrm{mL}$ 环己基氨基磺酸标准中间液 $(1200\mu \mathrm{g} / \mathrm{mL})$ 于离心管中，用水稀释至 $20\mathrm{mL}$ ，按5.3测定步骤衍生化。此时正庚烷中待测物的浓度分别相当于 $6\mu \mathrm{g} / \mathrm{mL}, 12\mu \mathrm{g} / \mathrm{mL}, 30\mu \mathrm{g} / \mathrm{mL}, 60\mu \mathrm{g} / \mathrm{mL}, 120\mu \mathrm{g} / \mathrm{mL}, 300\mu \mathrm{g} / \mathrm{mL}, 600\mu \mathrm{g} / \mathrm{mL}$ ，临用现配。

# 5.5 仪器参考条件

5.5.1 色谱柱：内涂 $50\%$ 苯基 $-50\%$ 甲基聚硅氧烷的中等极性毛细管柱， $30\mathrm{m} \times 0.32\mathrm{mm} \times 0.25\mu \mathrm{m}$ 或同等性能的色谱柱。  
5.5.2 柱温升温程序：初温 $50~\mathrm{^\circ C}$ 保持 $3\mathrm{min}$ ， $10^{\circ}\mathrm{C / min}$ 升温至 $70^{\circ}\mathrm{C}$ 保持 $0.5\mathrm{min}$ ， $30^{\circ}\mathrm{C / min}$ 升温至 $220^{\circ}\mathrm{C}$ 保持 $3\mathrm{min}$ 。  
5.5.3 进样口温度： $230^{\circ}C$   
5.5.4 进样量： $1\mu \mathrm{L}$   
5.5.5 进样方式：分流进样，分流比为 $1:5$   
5.5.6 检测器：氢火焰离子化检测器(FID)，温度 $260^{\circ}\mathrm{C}$ 。

5.5.7 载气：高纯氮气，流速为 $2.0\mathrm{mL} / \mathrm{min}$   
5.5.8 氢气： $32~\mathrm{mL / min}$ ；空气： $300~\mathrm{mL / min}$

# 5.6 标准曲线的制作

经衍生化处理的标准系列工作液注入气相色谱仪中，测得相应衍生物的峰面积，以标准系列工作溶液中环己基氨基磺酸的浓度为横坐标，以环己基氨基磺酸衍生物环己醇亚硝酸酯和环己醇两峰面积之和为纵坐标，绘制标准曲线。

环己基氨基磺酸衍生物气相色谱图参见附录A中图A.1。

# 5.7 试样溶液的测定

经衍生化处理的试样溶液注入气相色谱仪中，环己基氨基磺酸衍生物环己醇亚硝酸酯和环己醇两峰保留时间定性，测得峰面积加和作为测定面积计算，根据标准曲线得到试样溶液中待测物的浓度，计算样品中待测物的含量。衍生化处理的试样溶液中待测物响应若超出线性范围，应用正庚烷稀释后再进样分析。

# 5.8 空白试验

除不加试样外，均按5.2和5.3测定步骤进行。

# 6 分析结果的表述

试样中环己基氨基磺酸含量按式(1)计算。

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \times f \quad \dots \tag {1}
$$

式中：

$X$ ——试样中环己基氨基磺酸的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho$ ——由标准曲线计算出待测溶液中环己基氨基磺酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——加入正庚烷体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克(g)；

$f$ ——稀释倍数。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

当取样量为 $2\mathrm{g}$ 时，本方法检出限为 $0.01\mathrm{g / kg}$ ，定量限为 $0.03\mathrm{g / kg}$ 。

# 第三法 液相色谱-质谱/质谱法

# 16 原理

试样中的环己基氨基磺酸盐经提取液提取后，液相色谱-质谱/质谱仪测定，保留时间及相对离子丰度定性，同位素稀释内标法定量。

# 17 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 17.1 试剂

17.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
17.1.2 甲酸（HCOOH）：色谱纯。  
17.1.3 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
17.1.4 二氯甲烷 $(\mathrm{CH}_2\mathrm{Cl}_2)$ 。

# 17.2 试剂配制

17.2.1 含 $0.1\%$ 甲酸的乙腈水溶液：吸取 $1.0 \mathrm{~mL}$ 甲酸，加入 $300 \mathrm{~mL}$ 乙腈，用水稀释至 $1000 \mathrm{~mL}$ ，混匀。  
17.2.2 含 $0.1\%$ 甲酸的水溶液：吸取 $1.0 \mathrm{~mL}$ 甲酸，用水稀释至 $1000 \mathrm{~mL}$ ，混匀。  
17.2.3 含 $0.1\%$ 甲酸的甲醇溶液：吸取 $1.0 \mathrm{~mL}$ 甲酸，用甲醇稀释至 $1000 \mathrm{~mL}$ ，混匀。

# 17.3 标准品

17.3.1 同3.3。  
17.3.2 环己基氨基磺酸钠- $\mathrm{D}_4(\mathrm{C}_6\mathrm{H}_8\mathrm{D}_4\mathrm{NSO}_3\mathrm{Na})$ 同位素内标物：纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准品。

# 17.4 标准溶液配制

17.4.1 环己基氨基磺酸标准储备液 $(5\mathrm{mg / mL})$ ：准确称取适量环己基氨基磺酸钠标准品（精确至 $0.1\mathrm{mg}$ ）于 $100~\mathrm{mL}$ 容量瓶中，用水溶解并定容至刻度，混匀(环己基氨基磺酸钠折算为环己基氨基磺酸的换算系数为0.8907)。将溶液转移至棕色玻璃容器中， $4^{\circ}\mathrm{C}$ 下避光保存，有效期12个月。  
17.4.2 环己基氨基磺酸同位素内标储备液 $(100\mu \mathrm{g / mL})$ ：准确称取适量环己基氨基磺酸钠- $\mathrm{D_4}$ （精确至 $0.1\mathrm{mg}$ )于 $100~\mathrm{mL}$ 容量瓶中，用水溶解并定容至刻度，混匀(环己基氨基磺酸钠- $\mathrm{D_4}$ 折算为环己基氨基磺酸- $\mathrm{D_4}$ 的换算系数为0.8928)。将溶液转移至棕色玻璃容器中， $4^{\circ}C$ 下避光保存，有效期12个月。  
17.4.3 环己基氨基磺酸标准中间液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取环己基氨基磺酸标准储备液 $0.20~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。将溶液转移至棕色玻璃容器中， $4^{\circ}\mathrm{C}$ 下避光保存，有效期1个月。  
17.4.4 环己基氨基磺酸同位素内标中间液 $(5\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取环己基氨基磺酸同位素内标储备液 $5.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。将溶液转移至棕色玻璃容器中， $4^{\circ}\mathrm{C}$ 下避光保存，有效期1个月。  
17.4.5 环己基氨基磺酸标准系列工作液: 分别准确吸取环己基氨基磺酸标准中间液 $10 \mu \mathrm{L}, 20 \mu \mathrm{L}, 50 \mu \mathrm{L}, 100 \mu \mathrm{L}, 200 \mu \mathrm{L}, 500 \mu \mathrm{L}, 1000 \mu \mathrm{L}$ 于 $10 \mathrm{~mL}$ 容量瓶中, 再分别加入环己基氨基磺酸同位素内标中间液 $100 \mu \mathrm{L}$ , 用含 $0.1 \%$ 甲酸的乙腈水溶液定容至刻度。环己基氨基磺酸浓度分别为 $10 \mathrm{ng} / \mathrm{mL}, 20 \mathrm{ng} / \mathrm{mL}, 50 \mathrm{ng} / \mathrm{mL}, 100 \mathrm{ng} / \mathrm{mL}, 200 \mathrm{ng} / \mathrm{mL}, 500 \mathrm{ng} / \mathrm{mL}, 1000 \mathrm{ng} / \mathrm{mL}$ , 标准系列工作液中环己基氨基磺酸同位素内标的浓度为 $50 \mathrm{ng} / \mathrm{mL}$ 。临用现配。

# 17.5 材料

17.5.1 离心管： $50~\mathrm{mL}$ 具螺纹盖刻度离心管。  
17.5.2 有机相微孔滤膜： $0.22\mu \mathrm{m}$ 孔径。

# 18 仪器和设备

18.1 液相色谱-质谱/质谱仪，配有电喷雾(ESI)离子源。  
18.2 分析天平：感量 $1\mathrm{mg},0.1\mathrm{mg}$   
18.3 涡旋混合器。  
18.4 高速离心机。

18.5 超声波发生器。  
18.6 粉碎机。  
18.7 研磨机。  
18.8 匀浆机。  
18.9 恒温水浴装置。

# 19 分析步骤

# 19.1 试样制备

同5.1。

# 19.2 试样提取

# 19.2.1 巧克力、奶油、奶酪、乳粉、调味面制品、油腐乳、油豆鼓、肉及肉制品、水产品罐头等含较高油脂试样

称取 $2\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )于离心管中，准确加入 $200\mu \mathrm{L}$ 环己基氨基磺酸同位素内标中间液，加入 $20\mathrm{mL}$ 含 $0.1\%$ 甲酸的乙腈水溶液（巧克力、奶油、奶酪、乳粉等试样需 $60^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 水浴加热 $20\mathrm{min}$ ，涡旋 $5\mathrm{min}$ ，超声提取 $30\mathrm{min}$ 后放至室温，再加入 $10\mathrm{mL}$ 二氯甲烷缓慢振摇数次， $9000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液过有机相微孔滤膜，供上机测定。

# 19.2.2 果冻、糖果、米粉、淀粉制品等试样

称取 $2\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )于离心管中，准确加入 $200\mu \mathrm{L}$ 环己基氨基磺酸同位素内标中间液，加入 $20\mathrm{mL}$ 含 $0.1\%$ 甲酸的乙腈水溶液，混匀， $60^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 水浴加热 $20\mathrm{min}$ ，涡旋 $5\mathrm{min}$ ，超声提取 $10\mathrm{min}$ ，再加入 $10\mathrm{mL}$ 二氯甲烷缓慢振摇数次， $9000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液过有机相微孔滤膜，供上机测定。

# 19.2.3 其他固体、半固体试样

称取 $2\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )于离心管中，准确加入 $200~\mu \mathrm{L}$ 环己基氨基磺酸同位素内标中间液，加入 $20~\mathrm{mL}$ 含 $0.1\%$ 甲酸的乙腈水溶液，涡旋 $5\mathrm{min}$ ，超声提取 $30\mathrm{min},9000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取样液过有机相微孔滤膜，供上机测定。

# 19.2.4 液体试样

称取 $2\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ ) 于离心管中，准确加入 $200\mu \mathrm{L}$ 环己基氨基磺酸同位素内标中间液，用含 $0.1\%$ 甲酸的乙腈水溶液稀释至 $20\mathrm{mL}$ 刻度，混匀，涡旋 $5\mathrm{min}$ ，超声提取 $10\mathrm{min}$ ， $9000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取样液过有机相微孔滤膜，供上机测定。

# 19.4 定性测定

# 19.4.1 保留时间

试样中目标化合物色谱峰的保留时间与相应标准色谱峰的保留时间相比较，变化范围应在 $\pm 2.5\%$ 之内。

# 19.4.2 定量离子、定性离子及相对离子丰度比

环己基氨基磺酸2个子离子的相对丰度比 $(k)$ 与浓度相当标准工作溶液的环己基氨基磺酸的2个子离子的相对丰度比相比，其最大允许偏差不超过表3规定的范围，则可判定为试样中存在环己基氨基酸。

# 19.5 标准曲线的制作

将标准系列工作液注入液相色谱-质谱/质谱仪中，以被测组分浓度与内标浓度比值为横坐标，以被测组分色谱峰面积与内标色谱峰面积比值为纵坐标，绘制标准曲线。

# 19.6 试样溶液的测定

将试样溶液同标准曲线工作液进样测定，根据标准曲线得到试样溶液中待测物的浓度，计算样品中待测物的含量。

# 19.7 空白试验

除不加试样外，均按19.2测定步骤进行。

# 20 分析结果的表述

试样中环己基氨基磺酸含量按式(3)计算。

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \quad \dots \tag {3}
$$

式中：

$X$ ——试样中环己基氨基磺酸的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由标准曲线计算出待测溶液中环己基氨基磺酸的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样中提取溶剂总体积或定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，结果保留3位有效数字。

# 21 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 22 其他

当取样量 $2\mathrm{g}$ 时，本方法检出限为 $0.03\mathrm{mg / kg}$ ，定量限为 $0.10\mathrm{mg / kg}$ 。

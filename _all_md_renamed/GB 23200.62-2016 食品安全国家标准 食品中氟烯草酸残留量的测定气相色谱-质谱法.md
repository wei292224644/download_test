# 1 范围

本标准规定了食品中氟烯草酸残留量的气相色谱一质谱法。

本标准适用于玉米、芹菜、苹果、花生、茶叶、牛肉、鸡肝、鱼、蜂蜜、大豆和牛奶中氟烯草酸残留量的检测和确证，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

样品中残留的氟烯草酸用乙酸乙酯或乙腈提取。提取后的有机相蒸干，残渣用乙酸乙酯一环己烷(1+1)溶解后用凝胶渗透色谱仪(GPC)净化，洗脱液蒸干定容，用气相色谱一质谱仪选择离子检测，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ ：农残级。  
4.1.2 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ 。  
4.1.3 环己烷 $(\mathrm{C_6H_{12}})$ 。  
4.1.4 乙腈 $\mathrm{(CH_3CN)}$ ：液相色谱级。  
4.1.5 正己烷 $(\mathrm{C_6H_{14}})$ 。  
4.1.6 氯化钠 $\mathrm{(NaCl)}$ 。  
4.1.7 无水硫酸钠：于 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，储于密封容器中备用。

# 4.2 溶液配制

4.2.1 乙腈饱和正己烷。  
4.2.2 正己烷饱和乙腈。  
4.2.3 乙酸乙酯一环己烷 $(1 + 1$ ，体积比)：将乙酸乙酯和环己烷等体积混合均匀，备用。  
4.2.4 氯化钠溶液： $200\mathrm{g / L}$ ，取 $200\mathrm{gNaCl}$ ，定容至 $1\mathrm{L}$

# 4.3 标准品

氟烯草酸标准品：英文名称flumiclorac-pentyl，CAS号：87546-18-7，纯度 $\geqslant 99\%$

# 4.4 标准溶液配制

4.4.1 氟烯草酸标准储备溶液：准确称取适量的氟烯草酸标准品，用丙酮配制成 $100\mu \mathrm{g / mL}$ 标准储备液。  
4.4.2 氟烯草酸标准中间溶液: $1 \mu \mathrm{g} / \mathrm{mL}$ 。准确吸取 $1 \mathrm{~mL}$ 标准储备溶液放入 $100 \mathrm{~mL}$ 容量瓶中, 用丙酮定容至刻度, 该溶液在 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 保存。  
4.4.3 基质标准工作溶液：根据需要，用空白样品按照样品处理步骤得到的提取液，配制成不同浓度的基质标准工作溶液，基质标准工作溶液要现用现配。

# 4.5 材料

4.5.1 凝胶渗透色谱柱：填料为S-X3BioBeads,22g。  
4.5.2 无水硫酸钠柱：在 $150\mathrm{mm}\times 10\mathrm{mm}$ （内径）层析柱中，依次加入脱脂棉和 $5\mathrm{cm}$ 无水硫酸钠。

# 5 仪器和设备

5.1 气相色谱—质谱联用仪：配电子轰击离子源。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 均质器。  
5.4 离心机。  
5.5 旋转蒸发器。  
5.6 凝胶渗透色谱（以下简称GPC）。  
5.7 涡旋振荡器。  
5.8 离心管： $50~\mathrm{mL}$ ，聚丙烯。

# 6 试样的制备与保存

# 6.1 取样部位

样品取样部位按 GB 2763 的规定执行。在取样和制样操作过程中必须防止样品受到污染或使残留物含量发生变化。

# 6.2 水果、蔬菜

取样品 $500 \mathrm{~g}$ , 用粉碎机粉碎, 装入洁净容器内作为试样, 密封并做好标识, 于 $18^{\circ} \mathrm{C}$ 冰箱内保存。

# 6.3 坚果

取样品 $500\mathrm{g}$ ，用粉碎机粉碎，装入洁净容器内作为试样，密封并做好标识，于 $4^{\circ}C$ 冰箱内保存。

# 6.4 粮谷、茶叶

取样品 $500 \mathrm{~g}$ , 用粉碎机粉碎至全部通过 20 目筛, 装入洁净容器内作为试样, 密封并做好标识, 室温保存。

# 6.5 肉类及水产品

取样品 $500\mathrm{g}$ ，用粉碎机粉碎，装入洁净容器作为试样，密封并做好标识，于 $-18^{\circ}C$ 冰箱内保存。

# 6.6 蜂蜜

取有代表性样品 $500\mathrm{g}$ ，未结晶样品将其用力搅拌均匀，有结晶析出样品可将样品瓶盖塞紧后，置于不超过 $60^{\circ}\mathrm{C}$ 的水浴中，待样品全部溶化后搅匀，迅速冷却至室温。制备好的样品装入洁净容器内密封并做好标识，室温保存。

# 6.7 奶

取样品 $500\mathrm{g}$ ，混匀，装入洁净容器内作为试样，密封并做好标识， $-18^{\circ}C$ 保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 水果、蔬菜

称取试样 $10\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $10\mathrm{g}$ 无水硫酸钠和 $20~\mathrm{mL}$ 乙酸乙酯，20000 $\mathrm{r / min}$ 均质提取 $2\mathrm{min},4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，上层有机相通过无水硫酸钠柱过滤入鸡心瓶中。残渣用 $20~\mathrm{mL}$ 乙酸乙酯重复提取一次，离心过滤后，合并2次提取液，将提取液于 $45^{\circ}C$ 水浴用旋转蒸发器浓缩至近干，待GPC净化。

# 7.1.2 茶叶、粮谷、大豆、花生

称取试样 $10\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $150\mathrm{mL}$ 具塞三角瓶中，加入 $50\mathrm{mL}$ 乙腈， $20000\mathrm{r / min}$ 均质提取 $2\mathrm{min}$ ，上层有机相过滤入分液漏斗中。残渣用 $50\mathrm{mL}$ 乙腈重复提取一次，过滤后，合并提取液于分液漏斗中。

# 7.1.3 畜、禽、水产品

称取试样 $10\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $10\mathrm{g}$ 氯化钠，混匀，再加 $20~\mathrm{mL}$ 乙腈，20000 $\mathrm{r / min}$ 均质 $2\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。取上层有机相过无水硫酸钠柱转入分液漏斗。再用 $20~\mathrm{mL}$ 乙腈分2次按上述步骤提取残渣，合并有机相于分液漏斗中。

# 7.1.4 蜂蜜

称取 $10\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 蜂蜜样品于 $150\mathrm{mL}$ 三角瓶中, 加入 $20\mathrm{ml}$ 水溶解, 在平板摇床上振摇 $30\mathrm{min}$ , 转入 $250\mathrm{mL}$ 分液漏斗中, 分别用 $30\mathrm{mL}$ 水和 $50\mathrm{mL}$ 乙酸乙酯清洗原三角瓶, 洗液均转入分液漏斗中, 振摇 $2\mathrm{min}$ , 静置分层(如果发生乳化, 可将上层及乳化层以 $4000\mathrm{r/min}$ 的速度离心 $5\mathrm{min}$ , 取上层转入分液漏斗), 有机相过无水硫酸钠柱于鸡心瓶中。下层水相加入乙酸乙酯2次, 每次 $20\mathrm{mL}$ , 轻轻振摇 $1\mathrm{min}$ , 静置分层, 上层转入上述鸡心瓶中, 于 $45^{\circ}\mathrm{C}$ 水浴用旋转蒸发器浓缩至近干, 待GPC净化。

# 7.1.5 牛奶

称取 $10\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 牛奶样品于 $50\mathrm{mL}$ 具塞离心管中, 加入 $20\mathrm{mL}$ 乙腈, 涡旋振荡 $1\mathrm{min}$ , 沉降蛋白, $4000\mathrm{r} / \mathrm{min}$ 离心 $5\mathrm{min}$ , 上层清液过滤入鸡心瓶中, 于 $45^{\circ}\mathrm{C}$ 水浴上抽真空缓慢旋转蒸发至 $10\mathrm{mL}$ , 溶液转入 $250\mathrm{mL}$ 分液漏斗中, 并加入 $10\mathrm{mL}$ 氯化钠溶液, 混匀, 加入 $20\mathrm{mL}$ 乙酸乙酯, 振摇 $2\mathrm{min}$ , 静置分层。乙酸乙酯层转入 $150\mathrm{mL}$ 鸡心瓶中, 下层水相加入 $20\mathrm{mL}$ 乙酸乙酯液液萃取, 弃去水相, 乙酸乙酯层合并入上述心型瓶中, 于 $45^{\circ}\mathrm{C}$ 水浴旋转蒸发器浓缩至近干, 待GPC净化。

# 7.2 净化

# 7.2.1 畜、禽、水产品、花生、粮谷、茶叶、大豆

对7.1.2和7.1.3获得的有机相溶液，加入 $40\mathrm{ml}$ 乙腈饱和正己烷，振荡 $2\mathrm{min}$ ，静置分层，乙腈层转入鸡心瓶，再用正己烷饱和乙腈洗正己烷层两次，每次 $15\mathrm{mL}$ ，乙腈层合并入同一鸡心瓶中，于 $45^{\circ}\mathrm{C}$ 水浴上抽真空旋转蒸发至干。用乙酸乙酯一环己烷定容 $10\mathrm{mL}$ ，待GPC净化。

# 7.2.2 水果、蔬菜、蜂蜜、牛奶

用乙酸乙酯一环己烷溶解7.1.1、7.1.4和7.1.5获得的残渣，并定容至 $10~\mathrm{mL}$ ，以下按7.2.3操作。

# 7.2.3 凝胶渗透色谱净化

# 7.2.3.1 条件

a）净化柱： $400\mathrm{mm}$ （长） $\times 25\mathrm{mm}$ （内径），内装BIO-BeadsS-X3填料或相当者。  
b）流动相：乙酸乙酯一环己烷 $(1 + 1)$   
c) 流速: $4.7 \mathrm{~mL} / \mathrm{min}$ 。  
d) 进样量: $5 \mathrm{~mL}$ 。  
e) 开始收集时间：9.5min。  
f) 结束收集时间：14.0min。

# 7.2.3.2 净化

将7.2.1和7.2.2获得的溶液，进样 $5\mathrm{mL}$ ，用凝胶渗透色谱仪净化，收集 $9.5\mathrm{min}\sim 14\mathrm{min}$ 的馏分于鸡心瓶中，并在 $45^{\circ}\mathrm{C}$ 水浴旋转蒸发至干，用丙酮定容至 $1\mathrm{mL}$ ，供气相色谱一质谱仪测定。

# 7.3 空白基质溶液的制备

分别称取已知各种基质的阴性样品 $10\mathrm{g}$ ，按7.1和7.2操作。

# 7.4 测定

# 7.4.1 气相色谱一质谱参考条件

a）色谱柱：DB-5MS, $30\mathrm{m}$ （长） $\times 0.25\mathrm{mm}$ （内径），膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱升温程序： $200^{\circ}\mathrm{C}$ （保持 $1\mathrm{min}$ )，以 $10^{\circ}\mathrm{C / min}$ 的速率升温到 $280^{\circ}\mathrm{C}$ （保持 $12\mathrm{min}$ ）；  
c）进样口温度： $280^{\circ}\mathrm{C}$   
d) 载气：氦气，纯度 $99.999\%$   
e) 载气流速: 恒流模式 $1 \mathrm{~mL} / \mathrm{min}$ ;  
f）进样方式：不分流；  
g） 进样量： $2\mu \mathrm{L}$   
h) 接口温度: $280^{\circ} \mathrm{C}$ ;   
i）离子源：电子轰击源(EI)；  
j) 电离电压: $70 \mathrm{eV}$ ;  
k）离子源温度： $230^{\circ}C$   
1) 检测方式: SIM;

# 7.4.2 定量测定

对样液及标准工作液等体积穿插进样测定。实际应用的标准工作液和待测样液中, 氟烯草酸的响应值均应在仪器的线性范围之内。在上述气相色谱一质谱条件下, 氟烯草酸的保留时间约为 $15.9 \mathrm{~min}$ 。标准品气相色谱一质谱图参见附录 A。

# 7.5 空白试验

除不加样品外，均按 $7.1\sim 7.4$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理仪或按式(1)计算。

$$
X = \frac {A \times c \times V}{A _ {s} \times m}. \tag {1}
$$

式中：

$X$ ——试样中氟烯草酸残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A$ ——样液中氟烯草酸的峰面积；

$c$ ——标准溶液中氟烯草酸的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准溶液中氟烯草酸的峰面积；

$m$ ——最终样液所代表的试样量，单位为克(g)。

计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 B 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 C 的要求。

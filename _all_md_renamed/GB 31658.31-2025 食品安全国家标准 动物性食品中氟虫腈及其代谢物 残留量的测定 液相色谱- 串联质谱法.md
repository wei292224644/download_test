# 液相色谱-串联质谱法

# 1 范围

本文件规定了动物性食品中氟虫腈及其代谢物残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、牛、羊、鸡的肌肉、肝脏、肾脏、脂肪，以及鸡蛋、牛奶中氟虫腈、氟甲腈、氟虫腈砜和氟虫腈硫醚残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中残留的药物，用酸化乙腈提取，经固相萃取柱净化后，液相色谱-串联质谱法测定，基质匹配外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 冰乙酸 $(\mathrm{CN}_3\mathrm{COOH})_2$ 色谱纯

5.1.2 甲酸（HCOOH）：色谱纯

5.1.3 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.4 无水硫酸镁 $(\mathrm{MgSO}_4)$

# 5.2 溶液配制

5.2.1 $0.2\%$ 乙酸乙腈溶液：取冰乙酸 $2\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$

5.2.2 $0.1\%$ 甲酸乙腈溶液：取甲酸 $1\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$

5.2.3 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$

5.2.4 $50\%$ 酸化乙腈溶液：取 $0.1\%$ 甲酸乙腈溶液 $500~\mathrm{mL}$ ，与 $0.1\%$ 甲酸溶液 $500~\mathrm{mL}$ 混合。

# 5.3 标准品

氟虫腈、氟甲腈、氟虫腈砜、氟虫腈硫醚含量均 $\geq 97.0\%$ 。具体见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液 $(100\mu \mathrm{g / mL})$ ：取氟虫腈、氟甲腈、氟虫腈砜和氟虫腈硫醚标准品各约 $10\mathrm{mg}$ ，精密称定，于 $100~\mathrm{mL}$ 容量瓶中，用乙腈溶解并稀释至刻度，配制成浓度均为 $100~\mu \mathrm{g / mL}$ 的标准储备液，一 $18^{\circ}C$ 以下避光保存，有效期12个月。  
5.4.2 混合标准工作液 $(100\mathrm{ng / mL})$ ：精密量取标准储备液各 $0.1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 容量瓶中，用 $50\%$ 酸

化乙腈溶液稀释至刻度，配制成浓度为 $100\mathrm{ng / mL}$ 混合标准工作液， $4^{\circ}\mathrm{C}$ 避光保存，有效期3个月。

# 5.5 材料

5.5.1 通过式反相混合型亲水亲脂平衡固相萃取柱： $60\mathrm{mg / 3mL}$ ，或相当者。  
5.5.2 微孔尼龙滤膜: $0.22 \mu \mathrm{m}$ 。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
6.3 氮吹仪。  
6.4 固相萃取装置。  
6.5 涡旋混合器。  
6.6 水平振荡器。  
6.7 组织匀浆机。  
6.8 低温高速离心机：转速不低于 $8000\mathrm{r / min}$

# 7 试样的制备与保存

# 7.1 试样的制备

# 7.1.1 组织

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作溶液，作为空白添加试样。

# 7.1.2 鸡蛋

取适量新鲜或冷藏的空白或供试鸡蛋，去壳后混合均匀。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作溶液，作为空白添加试样。

# 7.1.3 牛奶

取适量新鲜或解冻的空白或供试牛奶，混合均匀。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作溶液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试料 $(2 \pm 0.05) \mathrm{~g}$ , 于 $50 \mathrm{~mL}$ 离心管, 加 $0.2 \%$ 乙酸乙腈溶液 $10 \mathrm{~mL}$ , 加无水硫酸镁 $1 \mathrm{~g}$ , 涡旋混匀, 振荡 $10 \mathrm{~min}$ , 于 $4^{\circ} \mathrm{C}, 8000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ , 取上清液, 备用。

# 8.2 净化

取备用液 $2.0\mathrm{mL}$ 过固相萃取柱，挤干，收集全部流出液于离心管中， $40^{\circ}\mathrm{C}$ 水浴氮气吹干。加入 $50\%$ 酸化乙腈溶液 $1.0\mathrm{mL}$ 溶解残余物，涡旋，微孔滤膜过滤，供液相色谱-串联质谱仪测定。

# 8.3 基质匹配标准曲线的制备

精密量取混合标准工作液适量，分别加入6份经提取和净化的空白试料残渣中， $40^{\circ}\mathrm{C}$ 水浴氮气吹干，加入 $50\%$ 酸化乙腈溶液 $1.0~\mathrm{mL}$ 涡旋溶解残余物，配制成浓度为 $0.2\mathrm{ng / mL},0.4\mathrm{ng / mL},1.0\mathrm{ng / mL},$ $2.0\mathrm{ng / mL},5.0\mathrm{ng / mL}$ 和 $10.0\mathrm{ng / mL}$ 的基质匹配系列标准溶液，微孔滤膜过滤，供液相色谱-串联质谱仪测定。以测得特征离子峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

c）进样量： $2\mu \mathrm{L}$   
d) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ ;   
e) 流动相：A为 $0.1\%$ 甲酸溶液，B为 $0.1\%$ 甲酸乙腈溶液，梯度洗脱程序见表1。

# 8.4.3 定性测定

在相同测试条件下，试料溶液中氟虫腈及其代谢物与基质匹配标准溶液中氟虫腈及其代谢物的保留时间偏差在 $\pm 0.1\mathrm{min}$ 以内；且检测到的相对离子丰度应与浓度相当的基质匹配标准溶液相对离子丰度

一致，其允许偏差为 $\pm 40\%$ 。

# 8.4.4 定量测定

取试料溶液和基质匹配标准溶液，作单点或多点校准，外标法计算。基质匹配标准溶液及试料溶液中目标物响应值均应在仪器检测的线性范围内。氟虫腈及其代谢物基质(鸡蛋)匹配标准溶液特征离子质量色谱图见附录B。

# 8.4.5 空白试验

取空白试料，除不加药物外，采用完全相同的步骤进行平行操作。

# 9 结果计算和表述

试样中待测物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times C _ {\mathrm {s}} \times V _ {1} \times V _ {3}}{A _ {\mathrm {s}} \times m \times V _ {2}} \dots \tag {1}
$$

式中：

$X$ ——试样中氟虫腈或各代谢物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——试料溶液中氟虫腈或各代谢物的色谱峰面积；

$C_{s}$ ——基质匹配标准溶液中氟虫腈或各代谢物浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V_{1}$ ——加入提取液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——基质匹配标准溶液中氟虫腈或各代谢物的色谱峰面积；

$m$ 供试试样质量的数值，单位为克 $(\mathbf{g})$

$V_{2}$ ——过柱备用液体积的数值，单位为毫升 $(\mathrm{mL})$ 。

# 10 检测方法的灵敏度、正确度和精密度

# 10.1 灵敏度

本方法氟虫腈、氟甲腈、氟虫腈砜和氟虫腈硫醚在猪、牛、羊、鸡的肌肉、肝脏、肾脏、脂肪，以及鸡蛋、牛奶中，检出限均为 $0.5\mu \mathrm{g / kg}$ ，定量限均为 $1.0~\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法氟虫腈、氟甲腈、氟虫腈砜和氟虫腈硫醚在猪、牛、羊、鸡的肌肉、肝脏、肾脏、脂肪，以及鸡蛋、牛奶， $1.0\mu \mathrm{g / kg}\sim 50~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

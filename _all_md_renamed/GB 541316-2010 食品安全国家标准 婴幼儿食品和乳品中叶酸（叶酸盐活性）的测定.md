# 前言

本标准代替GB/T5413.16-1997《婴幼儿配方食品和乳粉叶酸（叶酸盐活性）测定》。

本标准与GB/T5413.16-1997相比，主要变化如下：

对磷酸盐缓冲液作了调整；

——增加了米粉的处理方法；

——增加了光密度法测定步骤。

本标准所代替标准的历次版本发布情况为：

GB5413-1985、GB/T5413.16-1997。

# 食品安全国家标准婴幼儿食品和乳品中叶酸（叶酸盐活性）的测定

# 1 范围

本标准规定了婴幼儿食品和乳品中叶酸（叶酸盐活性）的测定方法。

本标准适用于婴幼儿食品和乳品中叶酸（叶酸盐活性）的测定。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

# 3 原理

利用干酪乳杆菌（Lactobacillus casei）ATCC 7469 对叶酸的特异性，在含有叶酸的样品中生长产生的酸度和形成的光密度来测定叶酸的含量。

# 4 试剂和材料

除非另有规定，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

4.1 鸡胰腺：称取 $100\mathrm{mg}$ 干燥的鸡胰腺，加 $20\mathrm{mL}$ 蒸馏水，搅拌 $15\mathrm{min}$ ，离心 $10\mathrm{min}$ （3000转/分钟），取上清液，临用前配制。  
4.2 $0.9\%$ 生理盐水：称取 $9.0 \mathrm{~g}$ 氯化钠溶解于 $1000 \mathrm{~mL}$ 水中，分装于具塞试管中，每管 $10 \mathrm{~mL}, 121^{\circ} \mathrm{C}$ 灭菌 $15 \mathrm{~min}$ 。每周准备一次。  
4.3 磷酸盐缓冲液   
4.3.1 磷酸盐缓冲液 I（ $0.05 \, \text{mol/L}$ ）：称取 $5.85 \, \text{g}$ 磷酸二氢钾， $1.22 \, \text{g}$ 磷酸氢二钾，用 $1000 \, \text{mL}$ 水溶解。临用前按 $0.5 \, \text{g/100 mL}$ 加入抗坏血酸。  
4.3.2 磷酸盐缓冲液II（用于谷物及谷物制品前处理）：称取 $14.2\mathrm{~g}$ 磷酸氢二钠，用 $1000\mathrm{~mL}$ 水溶解。临用前按 $1.0\mathrm{~g} / 100\mathrm{~mL}$ 加入抗坏血酸，用氢氧化钠溶液A（4.16）调 $\mathrm{pH}$ 至 $7.8 \pm 0.1$ 。  
4.3.3 磷酸盐缓冲液 III（用于谷物及谷物制品测试）：称取 $14.2 \mathrm{~g}$ 磷酸氢二钠，用 $1000 \mathrm{~mL}$ 水溶解。临用前按 $1.0 \mathrm{~g} / 100 \mathrm{~mL}$ 加入抗坏血酸，用氢氧化钠溶液 A（4.16）调 $\mathrm{pH}$ 至 $6.8 \pm 0.1$ 。  
4.3.4 磷酸盐缓冲液IV（ $0.1\mathrm{mol} / \mathrm{L}$ ）（用于谷物及谷物制品标准溶液制备）：溶解 $13.61\mathrm{g}$ 磷酸二氢钾于水中稀释到 $1000\mathrm{mL}$ 。用氢氧化钾溶液（4.10）调 $\mathrm{pH}$ 至 $7.0 \pm 0.1$ 。  
4.4 叶酸标准品。  
4.5 氨水（ $10.8\%$ ）。  
4.6 甲苯（ $\mathrm{C_7H_8}$ ）。  
4.7 抗坏血酸（ $\mathrm{C_6H_8O_6}$ ）。  
4.8 菌株：干酪乳杆菌（Lactobacillus casei）ATCC 7469。

# 4.9 培养基

4.9.1 乳酸杆菌琼脂培养基：胨化乳 $15\mathrm{g}$ ，酵母浸膏 $5\mathrm{g}$ ，葡萄糖 $10\mathrm{g}$ ，番茄汁 $100\mathrm{mL}$ ，磷酸二氢钾 $2\mathrm{g}$ ，聚山梨糖单油酸酯 $1\mathrm{g}$ ，琼脂 $10\mathrm{g}$ ，加蒸馏水至 $1000\mathrm{mL}$ ，调 $\mathrm{pH}$ 至 $6.8 \pm 0.2$ （ $20^{\circ}\mathrm{C} \sim 25^{\circ}\mathrm{C}$ ）。 $121^{\circ}\mathrm{C}$ 高压灭菌 $15\mathrm{min}$ ，备用。  
4.9.2 乳酸杆菌肉汤培养基：胨化乳 $15\mathrm{g}$ ，酵母浸膏 $5\mathrm{g}$ ，葡萄糖 $10\mathrm{g}$ ，番茄汁 $100\mathrm{mL}$ ，磷酸二氢钾 $2\mathrm{g}$ ，聚山梨糖单油酸酯 $1\mathrm{g}$ ，加蒸馏水至 $1000\mathrm{mL}$ ，调 $\mathrm{pH}$ 至 $6.8 \pm 0.2$ （ $20^{\circ}\mathrm{C} \sim 25^{\circ}\mathrm{C}$ ）。 $121^{\circ}\mathrm{C}$ 高压灭菌 $15\mathrm{min}$ ，备用。  
4.9.3 叶酸测定用培养基：酪蛋白胨 $10\mathrm{g}$ ，葡萄糖 $40\mathrm{g}$ ，乙酸钠 $40\mathrm{g}$ ，磷酸氢二钾 $1\mathrm{g}$ ，磷酸二氢钾 $1\mathrm{g}$ ，DL-色氨酸 $0.2\mathrm{g}$ ，L-天门冬氨酸 $0.6\mathrm{g}$ ，L-半胱氨酸盐酸盐 $0.5\mathrm{g}$ ，硫酸腺嘌呤 $10\mathrm{mg}$ ，盐酸鸟嘌呤 $10\mathrm{mg}$ ，尿嘧啶 $10\mathrm{mg}$ ，黄嘌呤 $20\mathrm{mg}$ ，聚山梨糖 $0.1\mathrm{g}$ ，谷光甘肽 $5\mathrm{mg}$ ，硫酸镁 $0.4\mathrm{g}$ ，氯化钠 $20\mathrm{mg}$ ，硫酸亚铁 $20\mathrm{mg}$ ，硫酸锰 $15\mathrm{mg}$ ，核黄素 $1\mathrm{mg}$ ，p-氨基苯甲酸 $2\mathrm{mg}$ ，维生素 $\mathrm{B}_64\mathrm{mg}$ ，盐酸硫胺素 $400\mu \mathrm{g}$ ，泛酸钙 $800\mu \mathrm{g}$ ，烟酸 $800\mu \mathrm{g}$ ，生物素 $20\mu \mathrm{g}$ ，加蒸馏水至 $1000\mathrm{mL}$ ，调 $\mathrm{pH}$ 至 $6.7 \pm 0.1$ （ $20^{\circ}C \sim 25^{\circ}C$ ）。

注：市售商业化合成培养基效果更稳定。

4.10 氢氧化钾溶液（ $4\mathrm{mol} / \mathrm{L}$ ）：称取 $224\mathrm{g}$ 氢氧化钾于 $1000\mathrm{mL}$ 烧杯中，用 $400\mathrm{mL}$ 水溶解，冷却至室温后，转移至 $1000\mathrm{mL}$ 容量瓶中，用水定容。  
4.11 木瓜蛋白酶溶液： $1\mathrm{g}$ 蛋白酶（活力 $\geqslant 6000\mathrm{U / mg}$ ， $\mathrm{pH}6.0 \pm 0.1$ ， $40^{\circ}\mathrm{C}$ ）溶于 $100\mathrm{mL}$ 磷酸盐缓冲液 I（4.3.1）中。临用前配制。  
4.12 $\alpha$ -淀粉酶溶液： $1\mathrm{g}\alpha$ -淀粉酶（ $1.5\mathrm{U} / \mathrm{mg}$ ）溶于 $100\mathrm{mL}$ 磷酸盐缓冲液I（4.3.1）中。临用前配制。  
4.13 $0.22\mu \mathrm{m}$ 灭菌滤膜。  
4.14 标准溶液的制备

4.14.1 叶酸标准贮备液（ $500\mu \mathrm{g} / \mathrm{mL}$ ）：称取 $55\mathrm{mg}\sim 56\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ）叶酸标准品（4.4），用 $50\mathrm{mL}$ 蒸馏水转入 $100\mathrm{mL}$ 容量瓶中，加 $2\mathrm{mL}$ 氨水（4.5）。溶液制备后，按式（1）计算溶液的体积，要求贮备液中叶酸盐的浓度为 $500\mu \mathrm{g} / \mathrm{mL}$ ：

$$
\text {贮 备 液 体 积 (m L)} = \frac {m \times 1 0 0 0 \times c}{1 0 0 \times 5 0 0}
$$

$$
\text {或 简 化 为 ： 贮 备 液 体 积 (m L)} = \frac {m \times c}{5 0} \dots \tag {1}
$$

式中：

$$
m — — \text {叶 酸 标 准 品 的 质 量 ， 单 位 为 毫 克 (m g)};
$$

$$
c — — \text {叶 酸 标 准 品 的 纯 度 ， 单 位 为 克 每 百 克} (\mathrm {g} / 1 0 0 \mathrm {g}) 。
$$

用水稀释溶液至刻度，用吸管加水至计算要求的体积，充分混合，放入棕色试剂瓶中 $2\mathrm{^{\circ}C}\sim 4\mathrm{^{\circ}C}$ 冰箱冷藏，保存期为4个月。

4.14.2 叶酸标准中间液（ $50\mu \mathrm{g} / \mathrm{mL}$ ）：吸取 $10\mathrm{mL}$ 叶酸标准贮备液（4.14.1）于 $100\mathrm{mL}$ 棕色容量瓶中，用水定容至刻度，充分混匀， $2^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱冷藏，保存期为1个月。  
4.14.3 叶酸标准工作液（0.05 ng/mL，0.1 ng/mL）：吸取1mL叶酸标准中间液（4.14.2）于100mL棕色容量瓶中，用水定容至刻度，混合。再吸该液1mL于100mL棕色容量瓶中，定容，混合。从上液中分别吸取5mL于250mL和500mL棕色容量瓶中，用磷酸盐缓冲液I（4.3.1）定容到刻度，混匀，即为高浓度标准工作液（0.1ng/mL）和低浓度标准工作液（0.05ng/mL）。临用前配制。  
4.15 盐酸（ $1\mathrm{mol} / \mathrm{L}$ ）：量取 $83.0\mathrm{mL}$ 盐酸溶于水中，冷却后定容至 $1000\mathrm{mL}$ 。

4.16 氢氧化钠溶液 A（ $4\mathrm{mol} / \mathrm{L}$ ）：称取 $160\mathrm{g}$ 氢氧化钠于 $1000\mathrm{mL}$ 烧杯中，用 $400\mathrm{mL}$ 水溶解，冷却至室温后，转移至 $1000\mathrm{mL}$ 容量瓶中，用水定容。  
4.17 氢氧化钠溶液 B（0.1 mol/L）：吸取 2.5 mL 氢氧化钠溶液 A（4.16）转移至 100 mL 容量瓶中用水定容。  
4.18 氢氧化钠标准滴定溶液 $(0.1\mathrm{mol} / \mathrm{L}\pm 0.0002\mathrm{mol} / \mathrm{L})$ ：称取 $4\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）氢氧化钠用水稀释至 $1000~\mathrm{mL}$ ，用邻苯二甲酸氢钾标定。保存此溶液的容器要密封，以防二氧化碳渗入。  
4.18.1 氢氧化钠标准溶液的标定：称取约 $0.18 \mathrm{~g}$ （精确至 $0.0001 \mathrm{~g}$ ）于 $105^{\circ} \mathrm{C} \sim 110^{\circ} \mathrm{C}$ 烘至恒重的邻苯二甲酸氢钾，用 $50 \mathrm{~mL}$ 除二氧化碳的水溶于锥形瓶中，加两滴 $5 \mathrm{~g} / \mathrm{L}$ 的酚酞指示剂，用配好的氢氧化钠溶液滴定至粉红色，同时作空白实验。按式（2）计算氢氧化钠标准溶液的浓度：

$$
c = \frac {\mathrm {m}}{\left(V _ {1} - V _ {2}\right) \times 0 . 2 0 4 2} \tag {2}
$$

式中：

$c$ ——氢氧化钠的浓度，单位为摩尔每升（mol/L）；

m——称取的邻苯二甲酸氢钾的质量，单位为克（g）；

$V_{1}$ ——氢氧化钠溶液的用量，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——空白试验氢氧化钠溶液的用量，单位为毫升（ $\mathrm{mL}$ ）。

4.18.2 酚酞溶液：取 $0.5 \mathrm{~g}$ 酚酞溶于 $75 \mathrm{~mL}$ 体积分数为 $95 \%$ 的乙醇中，加入 $20 \mathrm{~mL}$ 水，再加入氢氧化钠溶液（4.18），直至加入一滴立即变成粉红色，再用水定容至 $100 \mathrm{~mL}$ 。  
4.19 溴麝香草酚蓝指示剂：称取 $0.1 \mathrm{~g}$ 溴麝香草酚蓝于研钵中，加入 $1.6 \mathrm{~mL}$ 氢氧化钠溶液 B（4.17）研磨，加少许水至完全溶解，转移至 $250 \mathrm{~mL}$ 容量瓶中用水定容。

# 5 仪器和设备

5.1 pH计：精度为0.01。  
5.2 离心机：转速 $\geqslant 2000$ 转/分钟。   
5.3 分光光度仪。  
5.4 天平：感量为 $0.1\mathrm{mg}$   
5.5 生化培养箱： $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$   
5.6 滴定管：分刻度值为 $0.1\mathrm{mL}$   
5.7 涡旋振荡器。

# 6 分析步骤

# 6.1 测试菌液的制备

6.1.1 干酪乳杆菌（Lactobacillus casei）ATCC 7469 冻干菌粉转入乳酸杆菌肉汤培养基（4.9.2）中，36 $^\circ \mathrm{C} \pm 1$ ℃ 培养 24 h 后，转接至乳酸杆菌琼脂培养基（4.9.1）试管中，再 $36^{\circ} \mathrm{C} \pm 1$ ℃ 培养 24 h。培养好的乳酸杆菌琼脂培养基（4.9.1）试管的培养物作为贮备菌种。

6.1.2 从贮备菌种培养基上分别转接到三个乳酸杆菌琼脂培养基(4.9.1)试管中, 放入培养箱中 $36^{\circ} \mathrm{C} \pm 1$ ℃培养 $24 \mathrm{~h}$ 。每月转接一次, 作为月接种管贮于冰箱中。每月定期从月接种管中重新接种 3 个转接管保存新菌株。  
6.1.3 从月接种的培养管中的一支再接种一支乳酸杆菌琼脂培养基(4.9.1)试管， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h}$ ，作为日接种管每日测定用。  
6.1.4 从日接种管中接种一管乳酸杆菌肉汤培养基（4.9.2）， $36^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 培养 $24 \mathrm{~h}$ 。在无菌条件下离心该培养液 $10 \mathrm{~min}$ （2000 转/分钟），弃去上清液。用 $10 \mathrm{~mL}$ 生理盐水（4.2）振荡洗涤菌体，离心 $10 \mathrm{~min}$ （2000 转/分钟），弃去上清液，用 $10 \mathrm{~mL}$ 生理盐水（4.2）振荡清洗。如前离心操作，弃去上清液。再加 $10 \mathrm{~mL}$ 生理盐水（4.2），混匀。吸 $1 \mathrm{~mL}$ 该菌悬液于 $10 \mathrm{~mL}$ 生理盐水（4.2）中，混匀制成测试菌液。  
6.1.5 以生理盐水（4.2）做对照，用分光光度计，于 $550~\mathrm{nm}$ 波长下，测测试菌液（6.1.4）的光密度值，此值应在 $60\% \sim 80\%$ 之间。

# 6.2 试样的制备

# 6.2.1 乳制品

称取 $2\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）试样（约含叶酸 $5\mu \mathrm{g}$ ）于 $100\mathrm{mL}$ 烧杯中，用 $25\mathrm{mL} \sim 30\mathrm{mL}$ 水复原样品，转入 $100\mathrm{mL}$ 容量瓶中，用水定容至刻度，溶液中叶酸的质量浓度大约为 $0.05\mu \mathrm{g / mL}$ 。吸取 $1\mathrm{mL}$ 该样液和 $1\mathrm{mL}$ 鸡胰腺（4.1）于一个 $180\mathrm{mm} \times 15\mathrm{mm}$ 的带螺旋盖的试管中，充分混合。加 $18\mathrm{mL}$ 含抗坏血酸的磷酸缓冲液I（4.3.1），再加 $1\mathrm{mL}$ 甲苯（4.6）。同时制备±空白对照管，吸 $1\mathrm{mL}$ 蒸馏水和 $1\mathrm{mL}$ 鸡胰腺（4.1）于空白管中，加 $18\mathrm{mL}$ 含抗坏血酸的磷酸缓冲液I（4.3.1）及 $1\mathrm{mL}$ 甲苯（4.6）。在 $37^{\circ}C$ 下，样品管和空白管保温 $16\mathrm{h}$ 后，于 $100^{\circ}C$ 水浴加热 $5\mathrm{min}$ 。用磷酸盐缓冲液I（4.3.1）作适当稀释，得到浓度约为 $0.1\mathrm{ng / mL}$ 的叶酸盐溶液。

若确定样品中强化叶酸与原生叶酸相比所占比例很大，则可以用 $1\mathrm{mL}$ 样液加 $19\mathrm{mL}$ 含抗坏血酸的磷酸缓冲液I（4.3.1）于 $100^{\circ}\mathrm{C}$ 水浴加热 $5\mathrm{min}$ ，再用磷酸盐缓冲液I（4.3.1）稀释，得到浓度约为0.1ng/mL的叶酸盐溶液。

# 6.2.2 谷物及谷物制品

称取大约含 $1\mu \mathrm{g}$ 叶酸的试样于 $150~\mathrm{mL}$ 三角烧瓶中。加 $20\mathrm{mLpH}7.8$ 磷酸缓冲溶液II（4.3.2)，混匀后加 $50~\mathrm{mL}$ 水和 $1.0~\mathrm{mL}$ 甲苯（4.6)。加盖后 $121^{\circ}\mathrm{C}15\mathrm{min}$ 灭菌，然后迅速冷却。加 $1\mathrm{mL}$ 木瓜蛋白酶溶液（4.11），于 $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 保温 $3\mathrm{h}$ 后 $100^{\circ}\mathrm{C}$ 加热 $3\mathrm{min}$ ，冷却。加 $1\mathrm{mL}\alpha$ -淀粉酶溶液（4.12）， $36^{\circ}\mathrm{C}$ $\pm 1^{\circ}\mathrm{C}$ 保温 $2\mathrm{h}$ 后加 $4\mathrm{mL}$ 鸡胰腺(4.1)，加盖， $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 保温 $16\mathrm{h}$ 后 $100^{\circ}\mathrm{C}$ 加热 $3\mathrm{min}$ ，冷却。用 $1\mathrm{mol / L}$ 盐酸（4.15）调 $\mathsf{pH}$ 至4.5，用水稀释定容到 $100\mathrm{mL}$ 。过滤得到澄清滤液，然后吸取 $1\mathrm{mL}$ 澄清滤液用磷酸盐缓冲液III（4.3.3）定容至 $100~\mathrm{mL}$ ，得到浓度约为 $0.1\mathrm{ng / mL}$ 的叶酸盐溶液。

若确定样品中强化叶酸与原生叶酸相比所占比例很大则可以直接在样品中加 $20~\mathrm{mL}~0.05~\mathrm{mol / L}$ 含抗坏血酸的磷酸缓冲液II（4.3.2）和 $50~\mathrm{mL}$ 水，于 $121^{\circ}\mathrm{C}$ 灭菌 $15\mathrm{min}$ ，然后吸取 $1\mathrm{mL}$ 澄清滤液，再用 $0.05\mathrm{mol / L}$ 磷酸盐缓冲液III（4.3.3）稀释，得到浓度约为 $0.1\mathrm{ng / mL}$ 的叶酸盐溶液。

# 6.5 灭菌

将标准曲线管和试样管 $121^{\circ} \mathrm{C}$ 灭菌 $5 \mathrm{~min}$ ，迅速冷却到室温（商品化培养基按标签说明进行灭菌）。

注：保证加热和冷却过程中条件均匀，灭菌管数过多或距离太近，在灭菌锅中都可产生不良影响。

# 6.6 接种

无菌条件下每管中均加入一滴（约 $50~\mu \mathrm{L}$ ）菌悬液（6.1.4），加盖，充分振荡混匀所有试管（标准曲线未接种空白管S1除外）。

# 6.7 培养

6.7.1 酸度法: $36^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 培养 $72 \mathrm{~h}$ 。对每支试管进行目测检查, 未接种管内培养液应是澄清的, 标准曲线管和试样管中培养液的浊度应有梯度。未接种管中若出现混浊, 则测定无效。  
6.7.2 光密度法：在 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ ，培养 $16\mathrm{h} \sim 24\mathrm{h}$ 。其他同6.7.1。

# 6.8 测定

# 6.8.1 酸度法

6.8.1.1 用 $10 \mathrm{~mL}$ 水将未接种空白管 S1 和接种空白管 S2 的培养物转至三角烧瓶中, 以溴麝香草酚蓝 (4.19) 作指示剂, 或用 $\mathrm{pH}$ 计以 $\mathrm{pH} 6.8 \pm 0.2$ 为滴定终点用氢氧化钠标准滴定溶液 (4.18) 滴定标准曲线未接种空白管 S1 和接种空白管 S2。记录下消耗的氢氧化钠标准滴定溶液体积。

注：如果接种空白滴定反应消耗的氢氧化钠标准滴定溶液体积数等于或高于未接种空白水平的 $1.5 \mathrm{~mL}$ ，则测定结果无效。

6.8.1.2 用 $10 \mathrm{~mL}$ 水将标准曲线管和试样管中的培养物转至三角烧瓶中, 以溴麝香草酚蓝 (4.19) 作指示剂, 或用 $\mathrm{pH}$ 计以 $\mathrm{pH} 6.8 \pm 0.2$ 为滴定终点用氢氧化钠标准滴定溶液 (4.18.1) 滴定标准曲线管和试样管的培养物。记录下消耗的氢氧化钠标准滴定溶液体积。

注：通常标准曲线管 S7 消耗的 $0.1 \mathrm{~mol} / \mathrm{L}$ 的氢氧化钠标准滴定溶液体积数在 $8 \mathrm{~mL} \sim 12 \mathrm{~mL}$ 之间。

# 6.8.2 光密度法

以接种空白管（表1中试管号S2）作对照，取出最高浓度标准曲线管S7，振荡 $5\mathrm{s}$ ，在波长 $550\mathrm{nm}$ 下读取光密度值，放回重新培养。 $2\mathrm{h}$ 后同等条件重新测该管的光密度，如果两次光密度的绝对差结果 $\leqslant 2\%$ ，则取出全部检验管测定标准溶液和试样的光密度。

# 6.9 标准曲线的绘制

以标准曲线管叶酸含量作横坐标，以消耗氢氧化钠标准滴定溶液的毫升数或光密度值为纵坐标绘制标准曲线。

# 6.10 试样管中叶酸含量的计算

按照13.8每个试样管测定的消耗氢氧化钠标准滴定溶液的毫升数或光密度值，从标准曲线中查得对应的叶酸含量。每一编号的三个试样管应计算管中每毫升测定液叶酸的含量，并与其平均值相比较。相对偏差小于 $15\%$ 的试管为有效试管，无效试样管应舍去，有效试样管总数应大于所有试样管总数的2/3。重新计算每一编号的有效试样管中每毫升测定液叶酸含量的平均值，以此平均值计算全部编号试样管的总平均值 $C_x$ 。

注：样品管中叶酸含量低于 $0.05\mathrm{ng}$ ，高于 $0.5\mathrm{ng}$ 的值应舍去。

# 7 分析结果的表述

试样中叶酸含量按式（3）计算：

$$
X = \left[ \left(C _ {x} \times D\right) - E B \right] \times \frac {1 0 0}{1 0 0 0 m} \dots \tag {3}
$$

式中：

$X$ ——试样中叶酸含量, 单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$ ;

$C_x$ ——6.10中计算所得的总平均值，单位为纳克（ng）；

$D$ ——样品在处理后的稀释因子；

$EB$ ——鸡胰腺空白管中叶酸含量，单位为纳克每毫升（ $\mathrm{ng/mL}$ ）；

$m$ ——样品的质量或体积，单位为克（g）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 8 精密度

在重复性条件下获得的两次独立测试结果的绝对差值不超过算术平均值的 $10\%$ 。

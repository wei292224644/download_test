# 前言

本标准代替GB/T20499—2006《食品包装用聚氯乙烯膜中己二酸二(2-乙基)己酯迁移量的测定》、GB/T20500—2006《聚氯乙烯膜中己二酸二(2-乙基)己酯与己二酸二正辛酯含量的测定》的“己二酸二(2-乙基)己酯含量测定方法”和SN/T2826—2011《食品接触材料 高分子材料 食品模拟物中己二酸酯类增塑剂的测定 气相色谱-质谱法》的“己二酸二(2-乙基)己酯迁移量测定方法”。

本标准与GB/T20499—2006、GB/T20500—2006相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 己二酸二(2-乙基)己酯的测定和迁移量的测定”；

——删除了“己二酸二(2-乙基)己酯的气相色谱测定方法”。

# 食品安全国家标准

# 食品接触材料及制品 己二酸二

# (2-乙基)己酯的测定和迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中己二酸二(2-乙基)己酯(DEHA)的测定方法与迁移量的测定方法。

本标准适用于聚氯乙烯制品中己二酸二(2-乙基)己酯的测定及迁移量的测定。

# 己二酸二（2-乙基）己酯的测定

# 2 原理

用四氢呋喃溶解试样，加入甲醇沉淀其中的聚合物，过滤后，DEHA留在滤液中，采用气相色谱-质谱仪测定，外标法定量。

# 3 试剂与材料

# 3.1 试剂

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

3.1.1 四氢呋喃 $\mathrm{(C_4H_8O}$ ,CAS号：109-99-9）。  
3.1.2 甲醇 $\mathrm{(CH_4O, CAS}$ 号：67-56-1）。  
3.1.3 正己烷 $\mathrm{(C_6H_{12}}$ ，CAS号：110-54-3)：色谱纯。

# 3.2 标准品

己二酸二(2-乙基)己酯 $\mathrm{(C_{22}H_{42}O_4}$ ，CAS号：103-23-1）：纯度 $\geq 99.8\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.3 标准溶液配制

# 3.3.1 DEHA标准储备液

称取DEHA $100\mathrm{mg}$ （精确至 $0.0001\mathrm{g})$ ，用正己烷溶解后，定容至 $50~\mathrm{mL}$ ，获得DEHA浓度为 $2\mathrm{mg / mL}$ 的标准储备液。溶液在冰箱中储存，有效期为1个月。

# 3.3.2 DEHA标准中间溶液

用刻度吸量管吸取 $2.5\mathrm{mL}$ 的DEHA标准储备液(3.3.1)至 $50~\mathrm{mL}$ 容量瓶中，用正己烷稀释至刻度，获得DEHA标准中间溶液，其浓度为 $100.0\mathrm{mg / L}$ 。溶液储存方式同3.3.1。

# 3.3.3 DEHA标准工作溶液

分别用刻度吸量管及微量注射器吸取 $50~\mu \mathrm{L}, 0.2~\mathrm{mL}, 0.5~\mathrm{mL}, 1.0~\mathrm{mL}, 2.0~\mathrm{mL}, 5.0~\mathrm{mL}$ 的DEHA标准中间溶液(3.3.2)置于6个 $10~\mathrm{mL}$ 容量瓶中，用正己烷稀释、定容，充分摇匀，获得浓度分别为 $0.5\mathrm{mg / L},2.0\mathrm{mg / L},5.0\mathrm{mg / L},10.0\mathrm{mg / L},20.0\mathrm{mg / L},50.0\mathrm{mg / L}$ 的DEHA标准工作溶液。溶液储存方式同3.3.1。

# 4 仪器与设备

4.1 气相色谱-质谱联用仪：配电子轰击离子源（EI）。  
4.2 分析天平（精度为 $0.1\mathrm{mg}$ ）。  
4.3 超声波清洗器。  
4.4 氮吹仪。  
4.5 微量注射器： $100~\mu \mathrm{L}$   
4.6 试管： $50~\mathrm{mL}$

注：实验所用的玻璃器皿，都经过甲醇淋洗，通风晾干后待用。

# 5 分析步骤

# 5.1 试样处理

取样品，用剪刀裁成 $0.5\mathrm{cm}\times 0.5\mathrm{cm}$ 样片，混合均匀。

# 5.2 试样溶液的制备

称取 $0.1\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )样品于 $50~\mathrm{mL}$ 试管中，加入 $5\mathrm{mL}$ 四氢呋喃，超声提取 $30\mathrm{min}$ ，待样品全部溶解后，缓慢加入 $25~\mathrm{mL}$ 甲醇，沉淀聚氯乙烯高聚物，滤纸过滤后分别用 $5\mathrm{mL}$ 甲醇冲洗沉淀三次，合并滤液于 $50~\mathrm{mL}$ 容量瓶中，用甲醇定容并混合均匀，取 $100\mu \mathrm{L}$ 滤液室温下氮气吹干，用正己烷定容至 $1.0~\mathrm{mL}$ ，待测。若样品浓度超出标准工作溶液线性范围，可使用正己烷适当稀释后测定。

# 5.3 空白溶液的制备

除不加试样外，采用与5.2完全相同的分析步骤、试剂和用量。

# 5.4 仪器参考条件

# 5.4.1 色谱条件

色谱条件列出如下：

a）色谱柱：固定相为 $(5\%)$ 二苯基- $(95\%)$ 二甲基亚芳基硅氧烷共聚物，柱长 $30~\mathrm{m}$ ，内径 $0.32~\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ；  
b）柱温程序：初始温度为 $90^{\circ}\mathrm{C}$ ，然后以 $15^{\circ}\mathrm{C / min}$ 的速率升至 $280^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ ；  
c）进样口温度： $280^{\circ}C$   
d) 载气：氦气 $1.5 \mathrm{~mL} / \mathrm{min}$ ；  
e）进样方式：分流方式，分流比为 $50:1$   
f）进样量： $0.2\mu \mathrm{L}$

# 5.4.2 质谱条件

质谱条件列出如下：

a）质谱接口温度： $250~\mathrm{^\circ C}$   
b）离子源温度： $250~\mathrm{^\circ C}$   
c）离子化方式：EI；  
d) 离子化能量: $70 \mathrm{eV}$ ;   
e）测定方式：选择离子监测模式，DEHA的监测离子范围 $(m / z):40\sim 370$ ，DEHA的特征离子为129、147、112、71，其中129为定量离子；  
f）溶剂延迟： $5\mathrm{min}$

# 5.5 绘制标准工作曲线

按照5.4所列测定条件，分别将标准工作溶液(3.3.3)依次注入气相色谱-质谱仪中，以标准工作溶液中浓度为横坐标，单位为毫克每升 $(\mathrm{mg / L})$ ，以对应DEHA峰面积为纵坐标，绘制标准曲线。标准色谱图参见图A.1。

# 5.6 试样溶液的测定

# 5.6.1 定性测定

按照5.4所列测定条件，将试样溶液(5.2)、空白溶液(5.3)进样气相色谱-质谱仪测定。在相同的实验条件下进行样品测定时，若试样溶液中待测物的色谱峰的保留时间与相应标准的色谱峰的保留时间的偏差在 $\pm 2.5\%$ 以内，并且在扣除背景后的样品质谱中，所选择的离子均出现，且样品谱图中定性离子的相对丰度与浓度接近的标准溶液谱图中对应定性离子的相对丰度进行比较，偏差不超过表1规定的范围，则可判定为样品中存在该种待测物。DEHA的特征离子及其丰度比见表2，DEHA质谱图见图A.2。

# 5.6.2 定量测定

进行试样及空白溶液测定时，扣除空白值，得到DEHA峰面积。

# 6 分析结果的表述

试样中DEHA含量按式(1)计算：

$$
X = \frac {\rho \times V \times f \times 10 ^ {- 6}}{m} \times 100 \% \tag{1}
$$

式中：

$X$ ——试样中DEHA百分比含量， $\%$ ；

$\rho$ ——从工作曲线上查得试样溶液中DEHA浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释因子；

$10^{-6}$ ——单位换算因子；

$m$ ——所称样品的质量，单位为克 $(\mathrm{g})$ 。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过其算术平均值的 $10\%$ 。

# 8 其他

方法检出限为 $0.1\%$ 、定量限为 $0.25\%$ 。

# 己二酸二（2-乙基）己酯迁移量的测定

# 9 原理

试样经浸泡试验后，己二酸二(2-乙基)己酯迁移到食品模拟物中，经液液萃取后，将提取液注入气相色谱质谱仪中检测，外标法定量。

# 10 试剂与材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 水基、酸性、酒精类、油基食品模拟物：所用试剂依据GB31604.1规定。  
10.1.2 正己烷 $\mathrm{(C_6H_{14}}$ ，CAS号：110-54-3)：色谱纯。  
10.1.3 甲醇 $\mathrm{(CH_3OH, CAS}$ 号：67-56-1）：色谱纯。  
10.1.4 乙酸乙酯 $\mathrm{(C_4H_8O_2}$ ，CAS号：141-78-6）：色谱纯。

# 10.2 试剂配制

水基、酸性、酒精类、油基食品模拟物：按GB5009.156操作。

# 10.3 标准品

己二酸二(2-乙基)己酯（简称 $\mathrm{DEHA},\mathrm{C}_{22}\mathrm{H}_{42}\mathrm{O}_4$ ，CAS号：103-23-1)：纯度 $\geqslant 99.8\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 10.4 标准溶液配制

# 10.4.1 DEHA标准储备液（用于水基、酸性、酒精类食品模拟物分析）

称取DEHA标准品 $100\mathrm{mg}$ (精确至 $0.0001\mathrm{g}$ )，用正己烷溶解后，定容至 $50~\mathrm{mL}$ ，获得DEHA浓度为 $2\mathrm{mg / mL}$ 的标准储备液。溶液在冰箱中储存，有效期为1个月。

# 10.4.2 DEHA标准中间溶液（用于水基、酸性、酒精类食品模拟物分析）

用刻度吸量管吸取 $2.5\mathrm{mL}$ DEHA标准储备液(10.4.1)于 $50~\mathrm{mL}$ 容量瓶中，用正己烷稀释至刻度，获得DEHA标准中间溶液，其浓度为 $100.0\mathrm{mg / L}$ 。溶液储存方式同10.4.1。

# 10.4.3 DEHA标准工作溶液（用于水基、酸性、酒精类食品模拟物分析）

分别用刻度吸量管及微量进样器吸取 $50~\mu \mathrm{L}, 0.2~\mathrm{mL}, 0.5~\mathrm{mL}, 1.0~\mathrm{mL}, 2.0~\mathrm{mL}, 5.0~\mathrm{mL}$ 的DEHA标准中间溶液(10.4.2)置于6个 $10~\mathrm{mL}$ 容量瓶中，用正己烷稀释、定容，充分摇匀，获得浓度应分别为 $0.5\mathrm{mg / L},2.0\mathrm{mg / L},5.0\mathrm{mg / L},10.0\mathrm{mg / L},20.0\mathrm{mg / L},50.0\mathrm{mg / L}$ 的DEHA标准工作溶液。溶液储存方式同10.4.1。

# 10.4.4 DEHA标准储备液（用于油基食品模拟物分析）

称取DEHA $50\mathrm{mg}$ （精确至 $0.0001\mathrm{g}$ )至 $50~\mathrm{mL}$ 烧杯中，加入油基食品模拟物 $50~\mathrm{g}$ （精确到 $0.0001\mathrm{g})$ ，溶解混匀，该标准储备液浓度为 $1000~\mathrm{mg / kg}$ 。溶液在冰箱中储存，有效期为1个月。

# 10.4.5 DEHA标准中间溶液（用于油基食品模拟物分析）

称取 $1.0\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 的DEHA标准储备液(10.4.4)于 $10~\mathrm{mL}$ 容量瓶中，再准确称取 $9.0\mathrm{g}$ （精确到 $0.0001\mathrm{g}$ )油基食品模拟物加入该容量瓶中，混匀，获得DEHA标准中间溶液，其浓度为 $100\mathrm{mg / kg}$ 。溶液储存方式同10.4.4。

# 10.4.6 DEHA标准工作溶液（用于油基食品模拟物分析）

分别称取 $0.2\mathrm{g},0.5\mathrm{g},1.0\mathrm{g},2.0\mathrm{g},4.0\mathrm{g},5.0\mathrm{g}$ （精确到 $0.01\mathrm{g})$ DEHA标准中间溶液（10.4.5）置于6个 $10~\mathrm{mL}$ 容量瓶中，再分别加入油基食品模拟物 $9.8\mathrm{g},9.5\mathrm{g},9.0\mathrm{g},8.0\mathrm{g},6.0\mathrm{g},5.0\mathrm{g}$ （精确到 $0.0001\mathrm{g})$ ，摇匀，DEHA标准工作溶液。溶液浓度分别为 $2\mathrm{mg / kg},5\mathrm{mg / kg},10\mathrm{mg / kg},20\mathrm{mg / kg},$ $40~\mathrm{mg / kg},50~\mathrm{mg / kg}$ 。分别准确称取上述溶液 $2.0\mathrm{g}$ （精确到 $0.0001\mathrm{g})$ 至6个 $10~\mathrm{mL}$ 具塞离心管中，准确加入 $2.0~\mathrm{mL}$ 甲醇，用涡旋振荡器振荡提取 $10\mathrm{min}$ ，静置至少 $30\mathrm{min}$ 使两相分层后离心，吸取上层萃取液供气相色谱-质谱分析。

# 11 仪器和设备

11.1 气相色谱-质谱联用仪：配电子轰击离子源（EI）。  
11.2 氮吹仪。  
11.3 离心机。  
11.4 涡旋振荡器。  
11.5 离心管： $10~\mathrm{mL}$   
11.6 恒温水浴。

# 12 分析步骤

# 12.1 试样迁移试验

按照GB5009.156及GB31604.1的要求，对样品进行迁移试验，得到食品模拟物试液。如果得到的食品模拟物试液不能马上进行下一步试验，应将食品模拟物试液于 $4^{\circ}\mathrm{C}$ 冰箱中避光保存。

所得食品模拟物试液应冷却或恢复至室温后进行下一步试验。

# 12.2 试液制备

# 12.2.1 水基、酸性、酒精类食品模拟物的制备

用刻度吸量管吸取 $2.0\mathrm{mL}$ 经过迁移试验的模拟物浸泡液于 $10\mathrm{mL}$ 离心管中，准确加入 $2.0\mathrm{mL}$ 乙酸乙酯，振荡提取 $10\mathrm{min}$ 后， $4500\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，移取乙酸乙酯层萃取液；以 $2.0\mathrm{mL}$ 乙酸乙酯再提取一次，合并两次乙酸乙酯萃取液，用氮气室温吹干后用正己烷定容至 $2.0\mathrm{mL}$ ，供仪器检测。若选用的食品模拟物中乙醇含量超过 $20\%$ ，则首先需要使用氮吹仪除去过量乙醇后，再按照上述方式进行处理。

# 12.2.2 油基食品模拟物的制备

称取 $2.0\mathrm{g}$ （精确到 $0.01\mathrm{g}$ )经过迁移试验的油基食品模拟物浸泡液于 $10~\mathrm{mL}$ 具塞离心管中，准确加入 $2.0  \mathrm{mL}$ 甲醇，以涡旋振荡器振荡提取 $10\mathrm{min}$ ，静置至少 $30\mathrm{min}$ 使两相分层后离心，吸取上层萃取液供气相色谱-质谱分析。

# 12.3 空白溶液的制备

除不加试样外，采用与12.2完全相同的分析步骤、试剂和用量。

# 12.4 仪器参考条件

# 12.4.1 气相色谱条件

气相色谱条件列出如下：

a) 色谱柱: 固定相为 $(5\%)$ 二苯基- $(95\%)$ 二甲基亚芳基硅氧烷共聚物, 柱长 $30 \mathrm{~m}$ , 内径为 $0.32 \mathrm{~mm}$ , 膜厚 $0.25 \mu \mathrm{m}$ ;  
b）柱温程序：初始温度为 $90^{\circ}\mathrm{C}$ ，然后以 $15^{\circ}\mathrm{C / min}$ 的速率升至 $280^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ ；  
c）进样口温度： $280^{\circ}C$   
d) 载气：氮气 $1.5 \mathrm{~mL} / \mathrm{min}$ ；  
e）进样方式：分流方式，分流比为 $50:1$   
f）进样量： $0.2\mu \mathrm{L}$

# 12.4.2 质谱条件

质谱条件列出如下：

a）质谱接口温度： $250~\mathrm{^\circ C}$   
b）离子源温度： $250~\mathrm{^\circ C}$   
c）离子化方式：EI；  
d) 离子化能量: $70 \mathrm{eV}$ ;   
e）测定方式：选择离子监测模式，DEHA的监测离子范围 $(m / z):40\sim 370$ ，DEHA的特征离子为129、147、112、71，其中129为定量离子；

f）溶剂延迟： $5\mathrm{min}$

# 12.5 绘制标准工作曲线

按照12.4所列测定条件，分别将对水基、酸性、酒精类、油基食品模拟物标准工作溶液（10.4.3、10.4.6)进行检测，以标准溶液工作溶液中DEHA浓度为横坐标，单位为“毫克每升 $(\mathrm{mg / L})$ （水基、酸性、酒精类食品模拟物)或毫克每千克 $(\mathrm{mg / kg})$ （油基食品模拟物）”，以对应DEHA峰面积为纵坐标，绘制标准曲线。

# 12.6 试样溶液的测定

# 12.6.1 定性测定

按照12.4所列测定条件，将试样溶液(12.2)、空白溶液(12.3)进样气相色谱-质谱仪测定。定性方法同5.6.1。

# 12.6.2 定量测定

进行试样及空白溶液测定时，扣除空白值，得到DEHA峰面积。

# 13 分析结果的表述

由标准曲线得到试样溶液中DEHA的浓度，按GB5009.156进行迁移量计算，得到食品接触材料及制品中DEHA的迁移量。计算结果保留两位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

水基、酸性、酒精类食品模拟物中DEHA的方法检出限为 $0.2\mathrm{mg / L}$ ，定量限为 $0.5\mathrm{mg / L}$ ；油基食品模拟物中DEHA的方法检出限为 $0.8\mathrm{mg / kg}$ ，定量限为 $2\mathrm{mg / kg}$ 。

# 食品安全国家标准

# 水果、蔬菜及其制品中甲酸的测定

# 1 范围

本标准规定了水果、蔬菜及其制品中甲酸的测定方法。

本标准适用于采用重量法测定水果、蔬菜及其制品中甲酸含量。

# 2 原理

试样中的甲酸被蒸馏出来用碳酸钡(或碳酸钙)吸收，生成甲酸钡(或甲酸钙)。甲酸钡(或甲酸钙)定量地将氯化汞还原为氯化亚汞，然后根据氯化亚汞的质量计算出甲酸含量。

# 3 试剂

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 碳酸钡 $\mathrm{(BaCO_3)}$ 或碳酸钙（ $\mathrm{CaCO_3}$ ）。  
3.1.2 氯化汞（ $\mathrm{HgCl_2}$ ）。  
3.1.3 氯化钠(NaCl)。  
3.1.4 乙酸钠 $\mathrm{(CH_3COONa)}$ 。  
3.1.5 盐酸(HCl)：浓度为 $37\%$ （质量分数），密度为 $1.19~\mathrm{g / mL}$   
3.1.6 酒石酸 $(\mathrm{C_4H_6O_6})$ 。  
3.1.7 乙醇 $\mathrm{(CH_3CH_2OH)}$   
3.1.8 乙醚 $(\mathrm{CH}_3\mathrm{OCH}_3)$ 。

# 3.2 试剂配制

3.2.1 氯化汞-氯化钠混合溶液：称取 $100\mathrm{g}$ 氯化汞和 $30\mathrm{g}$ 氯化钠，加水溶解后稀释至 $1\mathrm{L}$ ，混匀。  
3.2.2 $500\mathrm{g / L}$ 乙酸钠溶液：称取 $500\mathrm{g}$ 乙酸钠，加水溶解后稀释至1L，混匀。  
3.2.3 $10\%$ 盐酸溶液：量取 $240~\mathrm{mL}$ 盐酸，以水稀释至 $1\mathrm{L}$ ，混匀。

# 4 仪器和设备

4.1 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
4.2 蒸馏装置：如图1所示。  
4.3 烘箱： $100^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$   
4.4 水浴锅。  
4.5 干燥器：内附有效干燥剂。

4.6 坩埚式过滤器，G4。  
4.7 可调式电炉： $500\mathrm{W}\sim 1000\mathrm{W}$   
4.8 电炉：2000W。  
4.9 锥形瓶：容积为 $2000~\mathrm{mL}$ 和 $500~\mathrm{mL}$   
4.10 回流冷凝管。  
4.11 搅拌机。

# 5 分析步骤

# 5.1 试样制备

准确称取 $25\mathrm{g}\sim 50\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 均匀试样移入图1的烧瓶A中(甲酸含量小于 $0.15\mathrm{g})$ 并加水至总体积为 $100~\mathrm{mL}$ ，必要时将试样在水中浸泡 $1\textrm{h}\sim 2\textrm{h}$ 。

# 5.2 蒸馏

5.2.1 在图1的烧瓶A中加入 $0.5\mathrm{g}\sim 1\mathrm{g}$ 酒石酸，称取 $2\mathrm{g}$ 碳酸钡(或碳酸钙)至上图的烧瓶B中并加水至 $100~\mathrm{mL}$ 。

5.2.2 如图1所示，将烧瓶A、B连接于蒸气发生器和冷凝器之间，同时加热蒸气发生器和烧瓶A、B。控制蒸馏装置中的电炉温度和蒸气量，使烧瓶A、B中溶液体积变化不超过 $5\mathrm{mL}$ 。待锥形瓶中馏出液为 $1000\mathrm{mL} \sim 1500\mathrm{mL}$ 时，停止加热，弃去馏出液。  
5.2.3 用 $\Phi 15\mathrm{cm}$ 快速定量滤纸将烧瓶中的热溶液过滤于 $500~\mathrm{mL}$ 锥形瓶中，并用热水洗涤图1烧瓶B3次～4次，每次洗涤液均经滤纸过滤后收集，使得锥形瓶中溶液最终约为 $250~\mathrm{mL}$ ，将滤液转移至蒸发皿中并置于电炉上加热，待液体蒸发至约 $100~\mathrm{mL}$ 后再转移于另一 $500~\mathrm{mL}$ 锥形瓶中。  
5.2.4 在锥形瓶中加入 $10 \mathrm{~mL} 500 \mathrm{~g} / \mathrm{L}$ 乙酸钠溶液、 $2 \mathrm{~mL} 10 \%$ 盐酸溶液以及 $25 \mathrm{~mL}$ 氯化汞氯化钠混合液，安装回流冷凝管于水浴锅中沸水浴回流 $2 \mathrm{~h}$ ，取出冷却（甲酸钡定量将氯化汞还原生成氯化亚汞沉淀）。  
5.2.5 将坩埚式过滤器洗净、烘干，于干燥器中平衡 $0.5\mathrm{h}$ 后，称重，精确至 $0.0002\mathrm{g}$ 。将沉淀全部用冷水洗入该过滤器中，抽滤，并用乙醇和乙醚分别洗涤沉淀。  
5.2.6 将装有氯化亚汞沉淀的坩埚式过滤器置于 $100^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 的烘箱内烘 $1\mathrm{h}$ ，取出放入干燥器中冷却后，称重，精确至 $0.0002\mathrm{g}$ 。

# 6 分析结果的表述

试样中甲酸含量按式(1)计算：

$$
X = \frac {\left(m _ {2} - m _ {1}\right) \times 0 . 0 9 7 5}{m} \times 1 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——试样中甲酸的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$m_{2}$ ——坩埚式过滤器加氯化亚汞沉淀的质量，单位为克(g)；

$m_{1}$ ——坩埚式过滤器的质量，单位为克 $(\mathrm{g})$ ；

0.0975——氯化亚汞折算成甲酸的换算系数；

$m$ —称样量，单位为克 $(\mathrm{g})$

100 —换算系数。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留小数点后三位。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 食品添加剂 硫酸锰

# 1 范围

本标准适用于以软锰矿、菱锰矿或金属锰为原料制得的食品添加剂硫酸锰。

# 2 化学名称、分子式和相对分子质量

# 2.1 化学名称

一水合硫酸锰

# 2.2 分子式

$\mathrm{MnSO}_4 \cdot \mathrm{H}_2\mathrm{O}$

# 2.3 相对分子质量

169.02（按2007年国际相对原子质量）

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603之规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 硫化铵。

A.3.1.2 乙酸。

A.3.1.3 氨水。

A.3.1.4 乙酸铵。

A.3.1.5 氯化钡溶液： $120\mathrm{g / L}$

A.3.1.6 盐酸溶液： $1 + 1$

A.3.1.7 乙酸铅溶液： $95\mathrm{g / L}$

# A.3.2 鉴别方法

# A.3.2.1 锰离子鉴别

取约 $1 \mathrm{~g}$ 试样, 溶于 $10 \mathrm{~mL}$ 水, 加入硫化铵, 产生浅橙色沉淀, 此沉淀溶于乙酸。

# A.3.2.2 硫酸根离子鉴别

A. 3.2.2.1 取约 $1 \mathrm{~g}$ 试样, 溶于 $10 \mathrm{~mL}$ 水, 加入氯化钡溶液, 即产生白色沉淀, 此白色沉淀不溶于盐酸溶液。  
A. 3.2.2.2 取约 $1 \mathrm{~g}$ 试样, 溶于 $10 \mathrm{~mL}$ 水, 加入乙酸铅溶液, 即产生白色沉淀, 此白色沉淀溶于乙酸铵溶液。  
A.3.2.2.3 取约 $1\mathrm{g}$ 试样，溶于 $10\mathrm{mL}$ 水，加入盐酸不产生沉淀。

# A.4 硫酸锰（ $\mathrm{MnSO_4\cdot H_2O}$ ）含量的测定

# A.4.1 方法提要

在pH为10的氨-氯化铵缓冲溶液中，以铬黑T为指示剂，用乙二胺四乙酸二钠（EDTA）标准溶液滴定。

# A.4.2 试剂和材料

A.4.2.1 盐酸羟胺溶液： $100\mathrm{g / L}$   
A.4.2.2 氨-氯化铵缓冲溶液甲： $\mathrm{pH}\approx 10$   
A.4.2.3 乙二胺四乙酸二钠（EDTA）标准滴定溶液： $c(\mathrm{EDTA}) = 0.05\mathrm{mol / L}$   
A. 4.2.4 铬黑T指示液: $5 \mathrm{~g} / \mathrm{L}$ 。

# A.4.3 分析步骤

称取约 $0.25 \mathrm{~g}$ 试样，精确至 $0.0002 \mathrm{~g}$ ，置于 $250 \mathrm{~mL}$ 锥形瓶中，加 $100 \mathrm{~mL}$ 水溶解，加入 $10 \mathrm{~mL}$ 盐酸羟胺溶液、 $25 \mathrm{~mL}$ 氨-氯化铵缓冲溶液甲和5滴铬黑T指示液，加热至沸并保持 $15 \mathrm{~min}$ ，用乙二胺四乙酸二钠（EDTA）标准滴定溶液滴定至溶液由紫红色变为纯蓝色，即为终点。

同时同样做空白试验，空白试样溶液除不加试样外，其他加入试剂的种类和量（标准滴定溶液除外）与试样溶液相同。

# A.4.4 结果计算

硫酸锰（ $\mathrm{MnSO_4 \cdot H_2O}$ ）含量的质量分数 $w_{1}$ ，按式（A.1）计算：

$$
w _ {1} = \frac {\left(V - V _ {0}\right) c M}{m \times 1000} \times 100 \% \dots \tag{A.1}
$$

式中：

$V$ ——滴定试样溶液消耗的乙二胺四乙酸二钠标准滴定溶液体积的数值, 单位为毫升 $(\mathrm{mL})$ ;

$V_{0}$ ——滴定空白试样溶液消耗的乙二胺四乙酸二钠标准滴定溶液体积的数值，单位为毫升（mL）

$c$ ——乙二胺四乙酸二钠（EDTA）标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——试样质量的数值，单位为克（g）；

$M$ ——硫酸锰的摩尔质量的数值，单位为克每摩尔（ $\mathrm{g/mol}$ ） $[M(\mathrm{MnSO}_4 \cdot \mathrm{H}_2\mathrm{O}) = 169.02]$ ;

1000——换算因子。

实验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$

# A.5 铅（Pb）的测定

# A.5.1 试剂和材料

A.5.1.1 三氯甲烷。

A.5.1.2 盐酸。

A.5.1.3 硝酸。

A.5.1.4 氢氧化钠溶液： $250\mathrm{g / L}$

A.5.1.5 吡咯烷二硫代甲酸铵（APDC）溶液： $20\mathrm{g / L}$ 。使用前过滤。

A.5.1.6 铅（Pb）标准溶液： $0.01\mathrm{mg / mL}$

A.5.1.7 水：符合GB/T6682—2008中二级水的规定。

# A.5.2 仪器和设备

原子吸收分光光度计：配有铅空心阴极灯。

# A.5.3 分析步骤

# A.5.3.1 玻璃仪器的清洗

均以硝酸溶液（1+2）浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

# A.5.3.2 铅标准溶液的制备

移取 $2.00 \mathrm{~mL}$ 铅标准溶液置于 $150 \mathrm{~mL}$ 烧杯中，加入 $1 \mathrm{~mL}$ 盐酸。加热煮沸 $5 \mathrm{~min}$ ，冷却，用水稀释至 $100 \mathrm{~mL}$ 。用氢氧化钠溶液调整溶液 $\mathrm{pH}$ 为 $1.0 \sim 1.5$ （用精密 $\mathrm{pH}$ 试纸检查）。将此溶液转移至 $250 \mathrm{~mL}$ 分液漏斗中，用水稀释至约 $200 \mathrm{~mL}$ 。加 $2 \mathrm{~mL}$ 吡咯烷二硫代氨基甲酸铵（APDC）溶液，混合。用三氯甲烷萃取两次，每次加入 $20 \mathrm{~mL}$ ，收集萃取液（即有机相）于 $50 \mathrm{~mL}$ 烧杯中，在蒸汽浴上蒸发至干（此操作必须在通风橱中进行），于残渣中加入 $3 \mathrm{~mL}$ 硝酸，继续蒸发至近干。加入 $0.5 \mathrm{~mL}$ 硝酸和 $10 \mathrm{~mL}$ 水，加热直至溶液体积约 $3 \mathrm{~mL} \sim 5 \mathrm{~mL}$ 。转移至 $10 \mathrm{~mL}$ 容量瓶，用水稀释至刻度，摇匀。

# A.5.3.3 试样溶液的制备

称取约 $5 \mathrm{~g}$ 试样，精确至 $0.01 \mathrm{~g}$ ，置于 $150 \mathrm{~mL}$ 烧杯中，加 $30 \mathrm{~mL}$ 水，加入 $1 \mathrm{~mL}$ 盐酸。盖上表面皿加热煮沸 $5 \mathrm{~min}$ 。冷却，用水稀释至 $100 \mathrm{~mL}$ 。用氢氧化钠溶液调整溶液 $\mathrm{pH}$ 为 $1.0 \sim 1.5$ （用精密 $\mathrm{pH}$ 试纸检验）。以下操作同 A.5.3.2 中“将此溶液转移至 $250 \mathrm{~mL}$ 分液漏斗中……转移至 $10 \mathrm{~mL}$ 容量瓶，用水稀释至刻度，摇匀”。

# A.5.3.4 测定

选用空气-乙炔火焰，于波长 $283.3 \mathrm{~nm}$ 处，用水调零，测定铅标准溶液和试样溶液的吸光度。试样溶液的吸光度不应大于铅标准溶液的吸光度。

# A.6 硒（Se）的测定

# A.6.1 试剂和材料

A. 6.1.1 盐酸羟胺。

A.6.1.2 环己烷。

A.6.1.3 盐酸溶液： $1 + 2$

A.6.1.4 氨水溶液： $1 + 1$

A. 6.1.5 2,3-二氨基萘溶液: 称取 $0.1 \mathrm{~g} 2,3$ -二氨基萘和 $0.5 \mathrm{~g}$ 盐酸羟胺, 溶解在 $0.1 \mathrm{~mol} / \mathrm{L}$ 的盐酸溶液中, 制成 $100 \mathrm{~mL}$ 溶液。

A.6.1.6 硒（Se）标准溶液： $0.001\mathrm{mg / mL}$

# A.6.2 仪器和设备

分光光度计：带有光程为 $1\mathrm{cm}$ 的比色皿。

# A.6.3 分析步骤

# A.6.3.1 试样溶液的制备

称取约 $0.2 \mathrm{~g}$ 试样, 精确至 $0.0002 \mathrm{~g}$ , 置于 $150 \mathrm{~mL}$ 烧杯中, 加入 $25 \mathrm{~mL}$ 盐酸溶液, 搅拌, 加热煮沸 $15 \mathrm{~min}$ 。加 $25 \mathrm{~mL}$ 水, 冷却。用氨水溶液调节溶液 $\mathrm{pH}$ 为 $2 \pm 0.2$ (用精密 $\mathrm{pH}$ 试纸检查), 加入 $0.2 \mathrm{~g}$ 盐酸羟胺, 搅拌至溶解, 立即加入 $5 \mathrm{~mL} 2,3$ -二氨基萘溶液, 混匀。盖上表面皿, 在室温下静置 $100 \mathrm{~min}$ 。然后全部转移至分液漏斗中, 补加水至约 $60 \mathrm{~mL}$ 。加 $5.0 \mathrm{~mL}$ 环己烷, 用力振荡 $2 \mathrm{~min}$ , 静置分层后, 弃去水层, 收集萃取液。

# A.6.3.2 标准溶液的制备

移取 $6.00 \mathrm{~mL}$ 硒标准溶液置于 $150 \mathrm{~mL}$ 烧杯中，以下操作同A.6.3.1“加 $25 \mathrm{~mL}$ 盐酸溶液……收集萃取液”。

# A.6.3.3 空白试样溶液的制备

空白试样溶液除不加试样外，其余加入试剂的种类和量与试样溶液相同。

# A.6.3.4 测定

用 $1\mathrm{cm}$ 的比色皿，在波长约 $380~\mathrm{nm}$ 处，以空白试样溶液为参比，测定吸光度，试样溶液的吸光度不应大于标准溶液的吸光度。

# A.7 灼烧减量

# A.7.1 仪器和设备

A.7.1.1 电热恒温干燥箱：控制温度在 $110^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。

A.7.1.2 高温炉：能控制温度在 $450^{\circ}C\pm 25^{\circ}C$

# A.7.2 分析步骤

称取约 $1 \mathrm{~g}$ 试样，精确至 $0.01 \mathrm{~g}$ ，置于在 $450^{\circ} \mathrm{C} \pm 25^{\circ} \mathrm{C}$ 下质量恒定的瓷坩埚中，于 $110^{\circ} \mathrm{C} \pm 2^{\circ} \mathrm{C}$ 电热恒温干燥箱中烘 $1 \mathrm{~h}$ ，再移入 $450^{\circ} \mathrm{C} \pm 25^{\circ} \mathrm{C}$ 的高温炉中灼烧至质量恒定。

# A.7.3 结果计算

灼烧减量的质量分数 $w_{2}$ , 按式 (A.2) 计算:

$$
w _ {2} = \frac {m _ {1} - m _ {0}}{m} \times 100 \% \dots \tag{A.2}
$$

式中：

$m_{1}$ ——灼烧前试样和瓷坩埚的质量的数值，单位为克（g）；

$m_{0}$ ——灼烧后试样和瓷坩埚的质量的数值，单位为克（g）；

$m$ ——试样质量的数值，单位为克（g）。

实验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.1\%$ 。

# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T1740—2006《进出口食品中四螨嗪残留量的检测方法气相色谱串联质谱法》。与SN/T1740—2006相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；

标准名称中“进出口食品”改为“食品”；

标准范围中增加“其他食品可参照执行”。

# 食品安全国家标准

# 食品中四螨嗪残留量的测定 气相色谱一质谱法

# 1 范围

本标准规定了食品中四螨嗪残留量(参见附录A)的气相色谱一质谱检测方法。

本标准适用于柑橘、苹果、菠菜、西蓝花、牛肝、鸡肾中四螨嗪残留量的测定。其他食品可参照执行。

# 3 原理

试样用水一丙酮(1+4)振荡提取，经二氯甲烷液一液分配，以凝胶色谱柱净化，再经弗罗里硅土(Florisil)固相柱净化，洗脱液浓缩并溶解定容后，供气相色谱一质谱仪检测，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 丙酮 $\mathrm{(CH_3COCH_3}$ ,CAS号：67-64-1）：色谱纯。

4.1.2 二氯甲烷 $\mathrm{(CH_2Cl_2}$ ，CAS号：75-09-2）：色谱纯。

4.1.3 环己烷 $\mathrm{(C_6H_{12}}$ ，CAS号： $110 - 82 - 7)$ ：色谱纯。

4.1.4 乙酸乙酯 $\mathrm{(C_4H_8O_2}$ ，CAS号：141-78-6）：色谱纯。

4.1.5 正已烷 $\mathrm{(C_6H_{14}}$ ，CAS号： $110 - 54 - 3$ ：色谱纯。

4.1.6 无水硫酸钠 $\mathrm{(Na_2SO_4}$ ,CAS号：15124-09-1)： $650^{\circ}C$ 灼烧 $4\mathrm{h}$ ，储于密封容器中备用。

# 4.2 溶液配制

硫酸钠水溶液 $(20\mathrm{g} / \mathrm{L})$ ：取 $20\mathrm{g}$ 无水硫酸钠，用水定容至 $1000~\mathrm{mL}$ ，摇匀备用。

# 4.3 标准品

四螨嗪标准物质参见附录A。

# 4.4 标准溶液配制

4.4.1 四螨嗪标准储备溶液：准确称取适量的四螨嗪标准物质，用丙酮配制成浓度为 $100\mu \mathrm{g / mL}$ 的标准储备溶液，标准储备溶液避光于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 保存，保存期为6个月。

4.4.2 四螨嗪标准工作溶液：根据检测要求，分别吸取上述标准储备溶液(4.4.1)于容量瓶中，用丙酮稀释到刻度配制成适当质量浓度的标准工作溶液，标准工作溶液避光于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存，保存期为1个月。

# 4.5 材料

4.5.1 弗罗里硅土(Florisil)固相萃取柱： $1.0\mathrm{g}$ ，或相当者。

# 5 仪器与设备

5.1 气相色谱—质谱联用仪：配质量选择检测器。  
5.2 凝胶色谱仪：配单元泵和馏分收集器。  
5.3 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.4 旋转蒸发器。  
5.5 无水硫酸钠柱： $7.5\mathrm{cm}\times 1.5\mathrm{cm}$ （内径），内装 $5\mathrm{cm}$ 高无水硫酸钠。  
5.6 具塞锥形瓶： $250~\mathrm{mL}$   
5.7 分液漏斗： $250~\mathrm{mL}$   
5.8 浓缩瓶： $50~\mathrm{mL},250~\mathrm{mL}$   
5.9 滤膜： $0.45\mu \mathrm{m}$   
5.10 振荡器。

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按照GB 2763的规定执行。在抽样和制样的操作过程中,应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 水果或蔬菜类

抽取水果或蔬菜样品 $500 \mathrm{~g}$ , (不可用水洗涤), 切碎后, 依次用食品捣碎机将样品加工成浆状。混匀, 均分成 2 份作为试样, 分装入洁净的盛样袋内, 密闭并标明标记。

# 6.2.2 肉及肉制品类

取出有代表性样品约 $1 \mathrm{~kg}$ , 经捣碎机充分捣碎均匀, 均分成 2 份, 分别装入洁净容器内作为试样。密封并标明标记

# 6.3 试样保存

试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 7 分析步骤

# 7.1 提取

称取试样约 $20\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）于 $250\mathrm{mL}$ 具塞锥形瓶中，加入 $100\mathrm{mL}$ 水一丙酮 $(1 + 4)$ ，振荡提取 $30\mathrm{min}$ ，将提取液抽滤于 $250\mathrm{mL}$ 浓缩瓶中。残渣再用 $50\mathrm{mL}$ 丙酮重复提取一次，合并滤液，于 $40^{\circ}\mathrm{C}$ 水浴中旋转浓缩至约 $20\mathrm{mL}$ 。

将浓缩提取液转移至 $250 \mathrm{~mL}$ 分液漏斗中, 加入 $150 \mathrm{~mL}$ 硫酸钠水溶液和 $50 \mathrm{~mL}$ 二氯甲烷, 振摇 3 min, 静置分层, 收集二氯甲烷相。水相再分 2 次, 每次用 $50 \mathrm{~mL}$ 二氯甲烷重复提取, 合并二氯甲烷相。经无水硫酸钠柱脱水, 收集于 $250 \mathrm{~mL}$ 浓缩瓶中, 于 $40^{\circ} \mathrm{C}$ 水浴中旋转浓缩至近干, 加入 $5 \mathrm{~mL}$ 环己烷一乙酸乙酯 $(1+1)$ 溶解残渣, 并用 $0.45 \mu \mathrm{m}$ 滤膜过滤, 待净化。

# 7.2 净化

# 7.2.1 凝胶色谱净化(GPC)

# 7.2.1.1 凝胶色谱条件

a）凝胶净化柱： $700\mathrm{mm}\times 25\mathrm{mm}$ ,BioBeadsS-X3，或相当者；  
b）流动相：环己烷—乙酸乙酯 $(1 + 1)$

c) 流速: $5.0 \mathrm{~mL} / \mathrm{min}$ ;   
d) 样品定量环: $5.0 \mathrm{~mL}$ ;   
e) 预淋洗体积: $50 \mathrm{~mL}$ ;   
f) 洗脱体积: $150 \mathrm{~mL}$ ;   
g）收集体积： $95\mathrm{mL}\sim 135\mathrm{mL}$

# 7.2.1.2 凝胶色谱净化步骤

将 $5\mathrm{mL}$ 待净化液按7.2.1.1规定的条件进行净化，合并馏分收集器中的收集液于 $50~\mathrm{mL}$ 浓缩瓶中，于 $40^{\circ}\mathrm{C}$ 水浴中旋转浓缩至近干，加入 $2\mathrm{mL}$ 正己烷一乙酸乙酯 $(4 + 1)$ 以溶解残渣，待净化。

# 7.2.2 固相萃取净化(SPE)

使用前用 $5\mathrm{mL}$ 正己烷预淋洗弗罗里硅土(Florisil)固相萃取柱，将样液倾入柱中，然后用 $8\mathrm{mL}$ 正己烷一乙酸乙酯 $(4 + 1)$ 进行洗脱。收集全部洗脱液于 $50~\mathrm{mL}$ 浓缩瓶中，于 $40^{\circ}\mathrm{C}$ 水浴中旋转浓缩至干。用丙酮溶解并定容至 $2\mathrm{mL}$ ，供气相色谱一质谱测定和确证。

# 7.3 测定

# 7.3.1 气相色谱一质谱参考条件

a）色谱柱：DB-5MS石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度： $50^{\circ}\mathrm{C}$ （保持 $2\mathrm{min})\xrightarrow{30^{\circ}\mathrm{C / min}}180^{\circ}\mathrm{C}$ （保持 $1\mathrm{min})\xrightarrow{10^{\circ}\mathrm{C / min}}270^{\circ}\mathrm{C}$ （保持 $10\mathrm{min})$   
c) 进样口温度: $280^{\circ} \mathrm{C}$ ;   
d）色谱一质谱接口温度： $270^{\circ}\mathrm{C}$   
e) 载气：氦气，纯度 $\geqslant 99.999\%$ ，流速 $1.2\mathrm{mL / min}$   
f）进样量： $1\mu \mathrm{L}$   
g）进样方式：无分流进样， $1.5\mathrm{min}$ 后开阀；  
h）电离方式：EI；  
i）电离能量： $70\mathrm{eV}$   
j）测定方式：选择离子监测方式；  
k）监测离子 $(m / z)$ ：参见附录A；  
1) 溶剂延迟： $5\mathrm{min}$

# 7.3.2 色谱测定与确证

7.3.2.1 根据样液中四螨嗪的含量情况，选定峰面积相近的标准工作溶液，对标准工作溶液与样液等体积参插进样测定，标准工作溶液和待测样液中四螨嗪的响应值均应在仪器检测的线性范围内。  
7.3.2.2 在相同实验条件下, 样液中待测物质的质量色谱保留时间与标准工作溶液相同并且在扣除背景后的样品质量色谱中所选离子均出现, 经过对比所选择离子的丰度与标准品对应离子的丰度比其值在允许范围内 (允许范围见表 1) 则可判定样品中存在对应的待测物。在 7.3.1 规定的色谱条件下, 四螨嗪的保留时间及其监测离子 $(m / z)$ 丰度比参见附录 A。四螨嗪标准物的气相色谱—质谱选择离子色谱图和质谱图参见附录 B 和附录 C。

# 7.4 空白试验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中四螨嗪的残留量。

$$
X _ {i} = \frac {A _ {i} \times C _ {s i} \times V}{A _ {s i} \times m}. \tag {1}
$$

式中：

$X_{i}$ ——试样中四螨嗪的残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A_{i}$ ——样液中四螨嗪的峰面积；

$C_{s}$ ——标准工作液中四螨嗪的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准工作液中四螨嗪的峰面积；

$m$ ——最终样液所代表的试样质量, 单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 D 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 E 的要求。  
10 定量限和回收率  
10.1 定量限

本方法四螨嗪的定量限为 $0.010\mathrm{mg / kg}$

10.2 回收率

当添加水平为 $0.010 \mathrm{mg} / \mathrm{kg}, 0.050 \mathrm{mg} / \mathrm{kg}, 0.200 \mathrm{mg} / \mathrm{kg}$ 时, 四螨嗪在不同基质中的添加回收率参见附录 F。

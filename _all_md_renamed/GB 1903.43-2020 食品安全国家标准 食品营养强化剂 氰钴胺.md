# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或者腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

本标准除另有规定外，所用试剂的纯度应为分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，实验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 丙酮。  
A.3.1.2 三氯甲烷。  
A.3.1.3 乙醚。  
A.3.1.4 盐酸。  
A.3.1.5 乙酸钠。  
A.3.1.6 溴化钾。   
A.3.1.7 氟化钠。  
A.3.1.8 硫酸氢钾。  
A.3.1.9 次磷酸。  
A.3.1.10 酚酞指示剂。  
A.3.1.11 乙酸 $(1\mathrm{mol} / \mathrm{L})$ 。  
A.3.1.12 氢氧化钠溶液（ $100\mathrm{g/L}$ ）：称取 $10\mathrm{g}$ 氢氧化钠，加水溶解并定容至 $100\mathrm{mL}$ 。  
A.3.1.13 氢氧化钠溶液 $(20\mathrm{g} / \mathrm{L})$ ：称取 $1\mathrm{g}$ 氢氧化钠，加水溶解并定容至 $50~\mathrm{mL}$ 。  
A.3.1.14 1-亚硝基-2-萘酚-3,6-二磺酸钠溶液 $(2\mathrm{g} / \mathrm{L})$ ：称取 $0.1\mathrm{g}$ 1-亚硝基-2-萘酚-3,6-二磺酸钠，缓缓加水溶解并定容至 $50~\mathrm{mL}$ 。  
A.3.1.15 硫酸溶液 $(1 + 7)$ ：取 $70~\mathrm{mL}$ 水，缓缓注入 $10~\mathrm{mL}$ 硫酸，混匀。  
A.3.1.16 硫酸亚铁铵饱和溶液：称取 $42\mathrm{g}$ 六水合硫酸亚铁铵，加水溶解并定容至 $100~\mathrm{mL}$

# A.3.2 仪器和设备

A.3.2.1 天平：感量 $0.0001\mathrm{g}$   
A.3.2.2 坩埚。  
A.3.2.3 蒸馏装置。  
A.3.2.4 紫外可见分光光度计。  
A.3.2.5 红外光谱仪。

# A.3.3 鉴别方法

# A.3.3.1 溶解性

略溶于水和乙醇，不溶于丙酮、三氯甲烷或乙醚。

# A.3.3.2 紫外鉴别

准确称取 $30\mathrm{mg}$ 试样于 $1000~\mathrm{mL}$ 容量瓶中，用水定容至刻度，采用紫外-可见分光光度计测定，在吸收谱上应出现最大峰 $278\mathrm{nm}\pm 1\mathrm{nm},361\mathrm{nm}\pm 1\mathrm{nm}$ 和 $550\mathrm{nm}\pm 2\mathrm{nm}$ 。 $361\mathrm{nm}$ 波长处的吸光度与 $278\mathrm{nm}$ 波长处的吸光度的比值应为 $1.70\sim 1.90$ 。 $361\mathrm{nm}$ 波长处的吸光度与 $550\mathrm{nm}$ 波长处的吸光度的比值应为 $3.15\sim 3.40$

# A.3.3.3 钴原子鉴别

取约 $1\mathrm{mg}$ 试样，加硫酸氢钾约 $50~\mathrm{mg}$ ，置坩埚中，灼烧至熔融，放冷，用玻璃棒捣碎，加 $3\mathrm{mL}$ 水，煮沸使溶解，加酚酞指示剂1滴，混匀，滴加氢氧化钠溶液(A.3.1.12)至显淡红色后，加 $500~\mathrm{mg}$ 乙酸钠、 $0.5\mathrm{mL}$ 乙酸和 $0.5\mathrm{mL}$ 1-亚硝基-2-萘酚-3,6-二磺酸钠溶液，立即显现红色或橙红色，加盐酸 $0.5\mathrm{mL}$ ，煮沸 $1\mathrm{min}$ ，颜色应不消失。

# A.3.3.4 氰离子鉴别

移取约 $5\mathrm{mg}$ 氰钴胺试样于盛有 $5\mathrm{mL}$ 水的 $50~\mathrm{mL}$ 蒸馏烧瓶中，蒸馏烧瓶连接一个短的、水冷却的垂直冷凝装置，冷凝装置出口端浸入到盛有 $1\mathrm{mL}$ 氢氧化钠溶液(A.3.1.13)的测试管中。样品溶解于蒸馏烧瓶中后，加入 $2.5\mathrm{mL}$ 次磷酸，连接好冷凝装置，缓慢加热煮沸 $10\mathrm{min}$ ，收集 $1\mathrm{mL}$ 蒸馏液于测试管中。测试管中加入4滴冷的、硫酸亚铁铵饱和溶液，轻摇，加入 $30\mathrm{mg}$ 氟化钠，加热至沸腾，立即加入几滴硫酸溶液至溶液变澄清，再加入3滴～5滴硫酸溶液，数分钟内溶液变成蓝色至蓝绿色。

# A.3.3.5 红外鉴别

将溴化钾压片试样的谱图与氰钴胺标准品的红外光谱图（见附录B中图B.1)比较，两者应基本一致。

# A.4 氰钴胺含量

# A.4.1 仪器和设备

A.4.1.1 天平：感量 $0.0001\mathrm{g}$ 。

A.4.1.2 紫外分光光度计。

# A.4.2 分析步骤

精确称取约 $0.025\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )干燥试样于 $1000~\mathrm{mL}$ 容量瓶中，用水溶解并定容至刻度，混匀得到试样溶液。用紫外可见分光光度计测定。以水作空白，采用 $1\mathrm{cm}$ 比色皿在 $361~\mathrm{nm}$ 处测定试样溶液吸光度值。

# A.4.3 结果计算

氰钴胺含量（以干基计)的质量分数 $w_{1}$ ，按式(A.1)计算，以 $\%$ 表示。

$$
w _ {1} = \frac {A}{207} \times \frac {V}{m} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$A$ —氰钴胺紫外吸光值；

207—氰钴胺的标准百分吸收系数 $(E_{1\mathrm{cm}}^{1\%})$

$V$ ——试样定容的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量（以干基计），单位为克 $(\mathrm{g})$ 。

实验结果以平行测定结果的算术平均值为准，保留一位小数。

在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $2.0\%$ 。

# A.5 干燥减量

# A.5.1 方法原理

干燥减量是指试样在 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 温度下干燥后，失去水分及其他挥发性物质的质量与试样质量之比，用百分数表示。

# A.5.2 仪器和设备

A.5.2.1 天平：感量 $0.0001\mathrm{g}$

A.5.2.2 称量瓶。

A.5.2.3 真空干燥箱。

# A.5.3 分析步骤

称取 $50\mathrm{mg}$ （精确至 $0.0001\mathrm{g}$ ）试样，置于已干燥恒重 $(105^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C},30\mathrm{min})$ 的称量瓶中。将试样在称量瓶中均匀铺开。在 $105^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 、真空压力低于 $2.67\mathrm{kPa}$ 条件下干燥至恒重。将称量瓶置于干燥器中冷却至室温，称量。

# A.5.4 结果计算

试样干燥减量的质量分数 $w_{2}$ ，按式（A.2）计算，以 $\%$ 表示。

$$
w _ {2} = \frac {m _ {1} - m _ {2}}{m _ {1} - m _ {0}} \times 100 \% \tag{A.2}
$$

式中：

$m_{1}$ ——干燥前带有样品的称量瓶的质量，单位为克(g)；

$m_{2}$ ——干燥后带有样品的称量瓶的质量，单位为克 $(\mathrm{g})$ ；

$m_0$ ——干燥后称量瓶的质量，单位为克 $(\mathrm{g})$ 。

计算结果保留2位有效数字。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $1.0\%$ 。

# A.6 有关物质

# A.6.1 方法原理

试样用流动相溶解后，采用液相色谱仪检测，峰面积比较法定量。

# A.6.2 试剂和材料

A.6.2.1 氯胺T。

A.6.2.2 甲醇。  
A.6.2.3 磷酸氢二钠。  
A.6.2.4 磷酸。  
A.6.2.5 盐酸。  
A.6.2.6 磷酸氢二钠溶液： $0.028 \, \mathrm{mol/L}$ 。称取 $10.03 \, \mathrm{g}$ 十二水合磷酸氢二钠，加水溶解并定容至 $1000 \, \mathrm{mL}$ 。  
A.6.2.7 盐酸溶液： $0.05\mathrm{mol / L}$ 。取 $4.5~\mathrm{mL}$ 盐酸，加水至 $1000~\mathrm{mL}$ ，混匀。  
A.6.2.8 氯胺T溶液： $0.1\%$ 。量取 $\mathrm{0.1g}$ 氯胺T，加水溶解并定容至 $100~\mathrm{mL}$

# A.6.3 仪器和设备

A.6.3.1 天平：感量 $0.0001\mathrm{g}$ 。  
A.6.3.2 高效液相色谱仪：配有紫外检测器。

# A.6.4 分析步骤

A.6.4.1 试样溶液：称取试样 $10\mathrm{mg}$ 于 $10~\mathrm{mL}$ 容量瓶中，用流动相溶解并定容至刻度，混匀。  
A.6.4.2 对照溶液：精密量取 $1\mathrm{mL}$ 试样溶液于 $100~\mathrm{mL}$ 量瓶中，用流动相稀释至刻度，混匀。

A.6.4.3 系统适用性溶液：称取试样 $25\mathrm{mg}$ 于 $25~\mathrm{mL}$ 容量瓶中，加 $10~\mathrm{mL}$ 水溶解样品，加入 $5\mathrm{mL}$ $0.1\%$ 氯胺T溶液、 $0.5\mathrm{mL}0.05\mathrm{mol / L}$ 盐酸溶液，用水稀释至刻度，混匀，静置 $5\mathrm{min}$ 。准确量取 $1\mathrm{mL}$ 上述溶液于 $10~\mathrm{mL}$ 量瓶中，用流动相稀释至刻度，混匀。

A.6.4.4 灵敏度溶液：准确量取 $1\mathrm{mL}$ 对照溶液于 $10~\mathrm{mL}$ 容量瓶中，用流动相稀释至刻度，混匀。

# A.6.5 参考色谱条件

A.6.5.1 色谱柱： $\mathrm{C_{18}}$ 色谱柱（柱长 $250~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m})$ ，或其他等效色谱柱。  
A.6.5.2 流动相：取 $260~\mathrm{mL}$ 甲醇和 $740~\mathrm{mL}$ 磷酸氢二钠溶液(A.6.2.6)混匀，用磷酸调节 $\mathrm{pH}$ 至3.5。  
A.6.5.3 柱温： $25^{\circ}C$ 。  
A.6.5.4 检测器波长： $361~\mathrm{nm}$   
A.6.5.5 进样量： $10~\mu \mathrm{L}$   
A.6.5.6 运行时间：氰钴胺主峰保留时间的3倍。

# A.6.6 系统适用性试验

采用新制备的溶液，且避免强光照射。按照液相色谱仪操作规程操作，取系统适用性溶液（A.6.4.3）和灵敏度溶液(A.6.4.4)分别注入液相色谱仪中，观察色谱图。

系统适用性溶液中应出现氰钴胺峰与一个降解产物峰(相对保留时间约为1.4)，二者的分离度应大于2.5，灵敏度溶液中主峰的信噪比应大于3。

# A.6.7 测定

系统适用性合格后再进行试样检测。依次注入试样溶液(A.6.4.1)、对照溶液(A.6.4.2)进行测定。

# A.6.8 结果计算

试样中有关物质的含量 $w_{3}$ ，按式(A.3)计算，以 $\%$ 表示。

$$
w _ {3} = \frac {A _ {1}}{A _ {2} \times 100} \times 100 \% \tag{A.3}
$$

式中：

$A_{1}$ ——试样溶液除主峰以外的所有杂质峰面积之和；

$A_{2}$ ——对照溶液主峰面积。

# A.6.9 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不超过其算术平均值的 $10\%$ 。

# A.7 丙酮

# A.7.1 试剂和材料

丙酮：色谱纯。

# A.7.2 仪器和设备

A.7.2.1 天平：感量 $0.001\mathrm{g}$ 。

A.7.2.2 气相色谱仪：配有氢火焰离子化检测器(FID)和顶空进样器。

# A.7.3 参考色谱条件

A.7.3.1 色谱柱：毛细管柱（柱长 $60\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ ，膜厚 $1.6\mu \mathrm{m}$ ，固定液为 $6\%$ 氰丙基苯基- $94\%$ 二甲基聚硅氧烷），或其他等效色谱柱。

A.7.3.2 载气：氮气（纯度大于 $99.99\%$ ）。

A.7.3.3 载气流量： $1.5\mathrm{mL / min}$

A.7.3.4 柱温： $40^{\circ}\mathrm{C}$ 保持 $10\mathrm{min}$ 后，以 $20^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升温至 $240^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$ 。

A.7.3.5 进样口温度： $140~\mathrm{^\circ C}$

A.7.3.6 检测器温度： $250~\mathrm{^\circ C}$ □

A.7.3.7 检测器气体流量：空气， $400~\mathrm{mL / min}$ ；氢气， $60~\mathrm{mL / min}$ ；尾吹气， $25~\mathrm{mL / min}$ 。

A.7.3.8 进样量： $1.0~\mathrm{mL}$

A.7.3.9 分流比： $1:50$

# A.7.4 参考顶空条件

A.7.4.1 顶空瓶平衡温度： $80^{\circ}C$

A.7.4.2 平衡时间： $45\mathrm{min}$

A.7.4.3 定量环温度： $90^{\circ}C$

A.7.4.4 传输线温度： $100^{\circ}\mathrm{C}$

# A.7.5 分析步骤

# A.7.5.1 空白溶液制备

移取 $5.0\mathrm{mL}$ 水，置于顶空瓶中，迅速压紧瓶盖。

# A.7.5.2 标准溶液制备

称取 $0.1\mathrm{g}$ 丙酮(精确至 $0.001\mathrm{g}$ ) 于已预加 $80~\mathrm{mL}$ 水的 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，摇匀，得到 $1000\mathrm{mg / L}$ 的丙酮标准储备液。吸取适量的丙酮标准储备液于 $10~\mathrm{mL}$ 容量瓶中，用水配制成 $10~\mathrm{mg / L},20~\mathrm{mg / L},50~\mathrm{mg / L},100~\mathrm{mg / L},200~\mathrm{mg / L}$ 的丙酮标准使用溶液，取上述系列溶液各 $5.0~\mathrm{mL}$

分别置于顶空瓶中，迅速压紧瓶盖。

# A.7.5.3 试样溶液制备

称取试样 $0.1\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于 $10~\mathrm{mL}$ 容量瓶中，用水溶解后定容至刻度，摇匀，得到试样溶液。移取该溶液 $5.0~\mathrm{mL}$ 置于顶空瓶中，迅速压紧瓶盖，备用。

# A.7.5.4 测定

在参考条件下(A.7.3和A.7.4)，分别对空白溶液、标准系列溶液和试样溶液进行测定，记录丙酮的峰面积值。丙酮的气相色谱图参见附录C中的图C.1。以标准系列溶液色谱图中丙酮的峰面积值为Y轴，以对应溶剂浓度 $(\mathrm{mg / L})$ 为 $X$ 轴，绘制标准曲线，得到丙酮标准曲线。根据试样溶液色谱图中丙酮的峰面积值，从标准曲线求得试样溶液中丙酮的浓度 $(\mathrm{mg / L})$ 。

# A.7.6 结果计算

试样中丙酮的含量 $\varpi_{4}$ ，按式(A.4)计算。

$$
w _ {4} = \frac {c \times V}{m} \times \frac {1}{1000} \times \frac {1}{1000} \times 100 \% \dots \tag{A.4}
$$

式中：

$c$ ——从标准曲线求得的试样溶液中丙酮的浓度，单位为毫克每升 $(\mathrm{mg / L})$ 。

$V$ ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

# A.7.7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不超过其算术平均值的 $10\%$ 。

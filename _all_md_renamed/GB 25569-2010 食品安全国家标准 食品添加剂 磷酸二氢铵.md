# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时须小心谨慎！如溅到皮肤上应立即用大量水冲洗，严重者应立即治疗。试验中所用易燃品，操作时不应使用明火。

# A.2 一般规定

本标准的检验方法中所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008中规定的三级水。试验中所用标准滴定溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按HG/T3696.1、HG/T3696.2、HG/T3696.3之规定制备。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 硝酸溶液： $1 + 8$

A.3.1.2 氨水溶液： $1 + 1$

A.3.1.3 氢氧化钠溶液： $40\mathrm{g / L}$

A.3.1.4 硝酸银溶液： $17\mathrm{g / L}$

A.3.1.5 红色石蕊试纸。

# A.3.2 分析步骤

# A.3.2.1 磷酸根的鉴别

称取约 $1 \mathrm{~g}$ 试样, 溶于 $20 \mathrm{~mL}$ 水中, 加硝酸银溶液, 生成黄色沉淀, 此沉淀溶于氨水溶液或硝酸溶液。

# A.3.2.2 铵离子的鉴别

称取约 $0.5\mathrm{g}$ 试样，用氢氧化钠溶液溶解样品，释放出的氨气，可使湿润的红色石蕊试纸变蓝。加热可促进分解。

# A. 4 磷酸二氢铵的测定

# A.4.1 重量法（仲裁法）

# A.4.1.1 方法提要

在酸性介质中，喹钼柠酮与试验溶液中的磷酸根反应生成磷钼酸喹啉沉淀，沉淀经过滤、烘干、称重，计算试样中磷酸二氢铵含量。

# A.4.1.2 试剂和材料

A.4.1.2.1 硝酸溶液： $1 + 1$

A.4.1.2.2 喹钼柠酮溶液。

# A.4.1.3 仪器和设备

A.4.1.3.1 玻璃砂芯坩埚：孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$

A.4.1.3.2 电热恒温干燥箱：温度能控制在 $180^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$

# A.4.1.4 分析步骤

# A.4.1.4.1 试验溶液的制备

称取约 $1 \mathrm{~g}$ 试样, 精确至 $0.0002 \mathrm{~g}$ 。置于 $100 \mathrm{~mL}$ 烧杯中, 加少量水溶解。移入 $500 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度, 摇匀, 干过滤(弃去最初 $20 \mathrm{~mL}$ 滤液)。

# A.4.1.4.2 测定

移取 $20.00 \mathrm{~mL}$ 试验溶液，置于 $250 \mathrm{~mL}$ 烧杯中。加 $10 \mathrm{~mL}$ 硝酸溶液，加水至总体积约 $100 \mathrm{~mL}$ ，加入 $50 \mathrm{~mL}$ 喹钼柠酮溶液，盖上表面皿，在水浴中加热至烧杯内的物质达到 $75^{\circ} \mathrm{C} \pm 5^{\circ} \mathrm{C}$ （在通风橱中进行），保温30s(在加入试剂和加热过程中，不得使用明火，不得搅拌，以免凝结成块)。冷却，在冷却过程中搅拌3次～4次，用预先在 $180^{\circ} \mathrm{C} \pm 2^{\circ} \mathrm{C}$ 下烘干 $45 \mathrm{~min}$ 的玻璃砂芯坩埚抽滤。先将上层清液过滤，以倾析法用洗瓶冲洗沉淀6次，每次用水约 $30 \mathrm{~mL}$ ，最后将沉淀移入玻璃砂芯坩埚中过滤，再用水洗涤沉淀4次。将玻璃砂芯坩埚连同沉淀置于电热恒温干燥箱中，从温度稳定开始计时，在 $180^{\circ} \mathrm{C} \pm 2^{\circ} \mathrm{C}$ 下干燥 $45 \mathrm{~min}$ 。取出稍冷后，置于干燥器中冷却至室温，称重。

同时做空白试验，空白试验除不加试样外，其他加入的试剂种类和量与试验溶液完全相同，并与试验溶液同样处理。

# A.4.1.5 结果计算

磷酸二氢铵含量以磷酸二氢铵 $\left(\mathrm{NH}_{4} \mathrm{H}_{2} \mathrm{PO}_{4}\right)$ 的质量分数 $w_{l}$ 计, 数值以%表示, 按公式 (A.1) 计算:

$$
w _ {1} = \frac {\left(m _ {1} - m _ {2}\right) \times 0.05198}{m \times (20 / 500)} \times 100 \% \dots \tag{A.1}
$$

式中：

$m_{I}$ ——试验溶液中生成磷钼酸喹啉沉淀的质量的数值，单位为克（g）；

$m_{2}$ ——空白试验溶液中生成磷钼酸喹啉沉淀的质量的数值，单位为克（g）；

$m$ ——试料的质量的数值，单位为克（g）；

0.05198——磷钼酸喹啉换算成磷酸氢二铵的系数。

取平行测定结果的算术平均值为测定结果。两次平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.4.2 容量法

# A.4.2.1 方法提要

试样溶解后用氢氧化钠标准滴定溶液滴定，以 $\mathrm{pH}$ 计指示终点，根据消耗氢氧化钠标准滴定溶液的体积，计算磷酸二氢铵含量。

# A.4.2.2 试剂和材料

A.4.2.2.1 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.1\mathrm{mol} / \mathrm{L}$

A.4.2.2.2 无二氧化碳的水。

# A.4.2.3 仪器和设备

A. 4.2.3.1 pH计：分度值为0.02。  
A.4.2.3.2 电磁搅拌器：配有搅拌转子。

# A.4.2.4 分析步骤

称取约 $0.5 \mathrm{~g}$ 试样，精确至 $0.0002 \mathrm{~g}$ ，置于 $150 \mathrm{~mL}$ 的烧杯中。加入 $50 \mathrm{~mL}$ 无二氧化碳的水，置于电磁搅拌器上，放入搅拌转子，搅拌至样品完全溶解。将已校准的 $\mathrm{pH}$ 计的电极放入试验溶液中，用氢氧化钠标准滴定溶液滴定至 $\mathrm{pH}$ 为8.0时，记录消耗氢氧化钠标准滴定溶液的体积。

# A.4.2.5 结果计算

磷酸二氢铵含量以磷酸二氢铵（ $\mathrm{NH_4H_2PO_4}$ ）的质量分数 $w_{1}$ 计，数值以%表示，按公式（A.2）计算：

$$
w _ {1} = \frac {c V M / 1000}{m} \times 100 \% \dots \tag{A.2}
$$

式中：

$V$ ——滴定试验溶液所消耗的氢氧化钠标准滴定溶液的体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——氢氧化钠标准滴定溶液的浓度的准确数值，单位为摩尔每升（mol/L）；

$m$ ——试料的质量的数值，单位为克（g）；

$M$ ——磷酸二氢铵 $\left(\mathrm{NH}_{4} \mathrm{H}_{2} \mathrm{PO}_{4}\right)$ 的摩尔质量的数值，单位为克每摩尔（ $\mathrm{g/mol}$ ）（ $M=115.0$ ）。

取平行测定结果的算术平均值为测定结果。两次平行测定结果的绝对差值不大于 $0.3\%$

# A.5 氟化物的测定

# A.5.1 试剂和材料

A.5.1.1 盐酸溶液： $1 + 11$   
A.5.1.2 乙酸钠溶液： $c(\mathrm{CH}_3\mathrm{COONa}\cdot 3\mathrm{H}_2\mathrm{O})\approx 3\mathrm{mol / L}$

称取 $204\mathrm{g}$ 乙酸钠（ $\mathrm{CH}_3\mathrm{COONa}\cdot 3\mathrm{H}_2\mathrm{O})$ ，溶于 $300\mathrm{mL}$ 水中，加乙酸溶液（1+16）调节 $\mathsf{pH}$ 至7.0，加水稀释至 $500\mathrm{mL}$ 。

A.5.1.3 柠檬酸钠溶液： $c(\mathrm{Na}_3\mathrm{C}_6\mathrm{H}_5\mathrm{O}_7\cdot 2\mathrm{H}_2\mathrm{O})\approx 0.75\mathrm{mol / L}$

称取 $110\mathrm{g}$ 柠檬酸钠（ $\mathrm{Na}_3\mathrm{C}_6\mathrm{H}_5\mathrm{O}_7\cdot 2\mathrm{H}_2\mathrm{O}$ ）溶于 $300\mathrm{mL}$ 水中，加 $14\mathrm{mL}$ 高氯酸，再加水稀释至 $500\mathrm{mL}$ 。

A.5.1.4 总离子强度缓冲剂：

乙酸钠溶液与柠檬酸钠溶液等体积混合，该溶液使用前配制。

A.5.1.5 氟化物标准溶液： $1\mathrm{mL}$ 溶液含氟（F） $0.001\mathrm{mg}$

用移液管移取 $1\mathrm{mL}$ 按HG/T3696.2配制的氟化物标准溶液，置于 $1000\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。该溶液使用前配制。

# A.5.2 仪器和设备

A.5.2.1 氟离子选择电极。

A.5.2.2 饱和甘汞电极。

A.5.2.3 电磁搅拌器：配有聚乙烯转子。

A.5.2.4 pH计：分度值0.01。

# A.5.3 分析步骤

# A.5.3.1 仪器的调整

将氟离子选择电极和饱和甘汞电极与测量仪器的负端与正端相连接，电极插入盛有水的塑料杯中，在电磁搅拌中（使用聚乙烯转子），读取平衡电位值，更换2次～3次水后，待电位达到电极说明书中规定的电位值后，即可进行电位测定。

# A.5.3.2 试验溶液的制备

称取约 $1 \mathrm{~g}$ 试样, 精确至 $0.01 \mathrm{~g}$ 。置于 $50 \mathrm{~mL}$ 烧杯中, 加入 $10 \mathrm{~mL}$ 水溶解, 转移至 $50 \mathrm{~mL}$ 容量瓶中, 加 $25 \mathrm{~mL}$ 总离子强度缓冲剂, $10 \mathrm{~mL}$ 盐酸溶液, 加水稀释至刻度, 摇匀。倒入 $25 \mathrm{~mL}$ 塑料烧杯中测定电极电位。

# A.5.3.3 工作曲线的绘制

分别移取 $1.00 \mathrm{~mL} 、 5.00 \mathrm{~mL} 、 8.00 \mathrm{~mL} 、 10.00 \mathrm{~mL} 、 15.00 \mathrm{~mL}$ 氟化物标准溶液（相当于氟的含量分别为 $0.001 \mathrm{mg} 、 0.005 \mathrm{mg} 、 0.008 \mathrm{mg} 、 0.010 \mathrm{mg} 、 0.015 \mathrm{mg}$ ）于 5 只 $50 \mathrm{~mL}$ 容量瓶中，于各个容量瓶中分别加 $25 \mathrm{~mL}$

总离子强度缓冲剂， $10\mathrm{mL}$ 盐酸溶液，加水稀释至刻度，摇匀。倒入 $50\mathrm{mL}$ 清洁、干燥的塑料烧杯中测定电极电位。

以电极电位为纵坐标，氟的质量（mg）为横坐标，在半对数坐标纸上绘制工作曲线，根据试样的电位值在曲线上查得氟离子的质量。

# A.5.4 结果计算

氟化物含量以氟（F）的质量分数 $w_{2}$ 计，数值以 $\mathrm{mg / kg}$ 表示，按公式（A.3）计算：

$$
w _ {2} = \frac {m _ {1}}{m / 1 0 0 0} \dots \tag {A.3}
$$

式中：

$m_{1}$ ——根据测得的试验溶液电位值从工作曲线上查得的氟的质量的数值，单位为毫克 $(\mathrm{mg})$ ；

$m$ ——试料的质量的数值，单位为克（g）。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $2\mathrm{mg / kg}$ 。

# A.6 砷的测定

# A.6.1 二乙氨基二硫代甲酸银比色法（仲裁法）

# A.6.1.1 试剂和材料

同GB/T5009.76—2003第一法二乙氨基二硫代甲酸银比色法第3章。

# A.6.1.2 分析步骤

称取约 $1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ 。置于砷发生瓶中，加水至约 $40\mathrm{mL}$ ，溶解后加 $10\mathrm{mL}$ 硫酸溶液（ $1+1$ ），以下按 GB/T5009.76—2003 第一法中 6.3 中“吸取 0，2.00，4.00，6.00，8.00， $10.00\mathrm{mL}$ ……砷的标准系列也应同法处理，以对标准曲线进行校正” 进行操作。

# A.6.1.3 结果计算

砷含量以砷(As)的质量分数 $w_{3}$ 计, 数值以 $\mathrm{mg} / \mathrm{kg}$ 表示, 按公式(A.4)计算:

$$
w _ {3} = \frac {\left(m _ {1} - m _ {0}\right) / 1 0 0 0}{m / 1 0 0 0} \dots \tag {A.4}
$$

式中：

$m_{1}$ ——从工作曲线上查出的试验溶液中砷的质量数值，单位为毫克（ $\mathrm{mg}$ ）；

$m_{0}$ ——从工作曲线上查出的空白试验溶液中砷的质量数值，单位为毫克（mg）；

$m$ ——试样的质量数值，单位为克（g）。

取平行测定结果的算术平均值为测定结果。两次平行测定结果的绝对差值不大于 $1\mathrm{mg / kg}$ 。

# A.7 重金属的测定

# A.7.1 试剂和材料

A.7.1.1 盐酸溶液： $1 + 1$   
A.7.1.2 氨水溶液： $2 + 3$

标准比色溶液的制备：移取 $2\mathrm{mL}$ 铅标准溶液于 $50\mathrm{mL}$ 比色管中，加水至 $20\mathrm{mL}$ ，加 $5\mathrm{mL}$ 乙酸盐缓冲溶液， $10\mathrm{mL}$ 硫化氢饱和溶液，用水稀释至刻度，摇匀，于暗处放置 $5\mathrm{min}$ 。与样品同时处理。

# A.8 铅的测定

# A.8.1 石墨炉原子吸收分光光度法（仲裁法）

# A.8.1.1 试剂和材料

A. 8.1.1.1 硝酸溶液: 约 $0.5 \mathrm{~mol} / \mathrm{L}$ ;

取 $32\mathrm{mL}$ 硝酸加入水中，稀释至 $1000\mathrm{mL}$

A. 8.1.1.2 铅标准溶液： $1 \mathrm{~mL}$ 溶液含铅（Pb） $100 \mathrm{mg}$ ;

用移液管移取 $1 \mathrm{~mL}$ 按 HG/T 3696.2 配制的铅标准溶液, 置于 $100 \mathrm{~mL}$ 容量瓶中, 用硝酸溶液稀释至刻度, 摇匀。移取 $5 \mathrm{~mL}$ 稀释后溶液于 $500 \mathrm{~mL}$ 容量瓶, 用硝酸溶液稀释至刻度, 摇匀。该溶液使用前配制。

A.8.1.1.3 二级水：符合GB/T6682—2008的规定。

# A.8.1.3.1 试验溶液的制备

# A.8.1.3.2 工作曲线绘制

分别移取铅标准溶液 $0.00\mathrm{mL}$ 、 $5.00\mathrm{mL}$ 、 $10.00\mathrm{mL}$ 、 $20.00\mathrm{mL}$ 于4只 $50\mathrm{mL}$ 的容量瓶中，加硝酸溶液稀释至刻度，摇匀。此系列溶液浓度分别为 $0\mathrm{ng / mL}$ 、 $10\mathrm{ng / mL}$ 、 $20\mathrm{ng / mL}$ 、 $40\mathrm{ng / mL}$ 。各吸取 $10\mu \mathrm{L}$ ，注入石墨炉，测得其吸光度，以铅的浓度为横坐标，对应的吸光度为纵坐标，绘制工作曲线。

# A.8.1.3.3 测定

在波长 $283.3 \mathrm{~nm}$ 处将仪器调至最佳工作状态，分别移取试验溶液和空白试验溶液 $10 \mu \mathrm{L}$ ，注入石墨炉，测得其吸光度，从工作曲线上查得相应铅的含量。

# A.8.1.4 结果计算

铅含量以铅(Pb)的质量分数 $w_{4}$ 计，数值以 $\mathrm{mg / kg}$ 表示，按公式(A.5)计算：

$$
w _ {4} = \frac {\left(c _ {1} - c _ {0}\right) / 1 0 ^ {6} \times 2 5}{m / 1 0 ^ {3}} \dots \tag {A.5}
$$

式中：

$c_{1}$ ——从工作曲线上查出的试验溶液中铅的含量的数值，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$\mathrm{c}_{0}$ ——从工作曲线上查出的空白试验溶液中铅的含量的数值，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$m$ ——试料的质量的数值，单位为克(g)；

25——试料定容体积，单位为毫升（mL）。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $1\mathrm{mg / kg}$ 。

# A.8.2.3.1 试验溶液的制备

称取约 $5 \mathrm{~g}$ 样品, 精确至 $0.01 \mathrm{~g}$ , 置于 $150 \mathrm{~mL}$ 烧杯中, 加 $30 \mathrm{~mL}$ 水, 加入 $1 \mathrm{~mL}$ 盐酸。加热煮沸几分钟, 冷却, 稀释至 $100 \mathrm{~mL}$ , 用氢氧化钠溶液调整溶液 $\mathrm{pH}$ 至 $1.0 \sim 1.5$ 。将此溶液转移至 $250 \mathrm{~mL}$ 分液漏斗中, 用水稀释至约 $200 \mathrm{~mL}$ 。加 $2 \mathrm{~mL}$ 吡咯烷二硫代氨基甲酸铵（APDC）溶液, 混合。用三氯甲烷萃取两次, 每次加入 $20 \mathrm{~mL}$ , 收集萃取物于 $50 \mathrm{~mL}$ 烧杯中, 汽浴蒸发至干 (此操作必须在通风橱中进行), 于残渣中加入 $3 \mathrm{~mL}$ 硝酸, 继续蒸发至近干。加入 $0.5 \mathrm{~mL}$ 硝酸和 $10 \mathrm{~mL}$ 水, 加热直至溶液体积约为 $3 \mathrm{~mL} \sim 5 \mathrm{~mL}$ 。转移至 $10 \mathrm{~mL}$ 容量瓶, 用水稀释至刻度, 摇匀。

# A.8.2.3.2 空白试验溶液的制备

取 $30 \mathrm{~mL}$ 水置于 $150 \mathrm{~mL}$ 烧杯中，加入 $1 \mathrm{~mL}$ 盐酸。加热煮沸几分钟，冷却，稀释至 $100 \mathrm{~mL}$ 。用氢氧化钠溶液调整溶液 $\mathrm{pH}$ 至 $1.0 \sim 1.5$ 。以下操作同 A.8.2.3.1 中“将此溶液转移至 $250 \mathrm{~mL}$ 分液漏斗……转移至 $10 \mathrm{~mL}$ 容量瓶，用水稀释至刻度，摇匀”。

# A.8.2.3.3 标准溶液的制备

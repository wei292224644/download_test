# 前言

本标准代替GB31604.46—2016《食品安全国家标准 食品接触材料及制品 游离酚的测定和迁移量的测定》。

本标准与GB31604.46—2016相比，主要变化如下：

——修改了范围；  
——修改了第一部分中检出限和定量限；  
——修改了第二部分中分析步骤和结果表述；  
——增加了第三部分 苯酚迁移量的测定。

# 游离酚的测定和迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品游离酚的测定和迁移量的测定方法。

本标准第一部分适用于食品接触用环氧酚醛树脂涂料中游离酚的测定；第二部分适用于食品接触材料及制品中游离酚迁移量的测定；第三部分适用于食品接触材料及制品中苯酚迁移量的测定。

# 第一部分 游离酚的测定

# 2 原理

试样中游离酚与溴反应生成溴代苯酚，剩余的溴与碘化钾作用，析出的碘用硫代硫酸钠滴定，根据硫代硫酸钠溶液消耗量计算出游离酚的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 盐酸（HCl）：质量分数 $36\% \sim 38\%$   
3.1.2 三氯甲烷 $\mathrm{(CHCl_3)}$ 。  
3.1.3 乙醇 $(\mathrm{C}_2\mathrm{H}_6\mathrm{O})$ 。  
3.1.4 碘化钾(KI)。  
3.1.5 可溶性淀粉 $\left[\left(\mathrm{C}_{6} \mathrm{H}_{10} \mathrm{O}_{5}\right)_{n}\right]$ 。  
3.1.6 溴水 $(\mathrm{Br}_2)$ ：含量 $\geq 3\%$ 。  
3.1.7 无水碳酸钠 $\mathrm{(Na_2CO_3)}$ 。  
3.1.8 溴化钾(KBr)。

# 3.2 试剂配制

3.2.1 碘化钾溶液 $(100\mathrm{g / L})$ ：称取 $1.00\mathrm{g}$ 碘化钾，用水溶解并稀释至 $10~\mathrm{mL}$ 。  
3.2.2 淀粉指示液 $(10\mathrm{g / L})$ ：称取 $1.0\mathrm{g}$ 可溶性淀粉，加 $5\mathrm{mL}$ 水调至糊状，在搅拌下将糊状物加到 $90~\mathrm{mL}$ 沸腾的水中，煮沸 $1\mathrm{min}\sim 2\mathrm{min}$ ，冷却，稀释至 $100~\mathrm{mL}$ ，使用期为2周。

# 3.3 标准品

3.3.1 五水合硫代硫酸钠 $\mathrm{(Na_2S_2O_3\cdot 5H_2O}$ ,CAS号：10102-17-7)：工作基准试剂，或经国家认证并授予标准物质证书的标准物质。

3.3.2 溴酸钾 $\mathrm{(KBrO_3}$ ,CAS号：7758-01-2)：工作基准试剂，或经国家认证并授予标准物质证书的标准物质。  
3.3.3 溴容量分析用标准溶液：经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

# 3.4.1 硫代硫酸钠标准滴定溶液 $\left[c\left(\mathrm{Na}_{2} \mathrm{~S}_{2} \mathrm{O}_{3}\right)=0.1 \mathrm{~mol} / \mathrm{L}\right]$

称取 $26\mathrm{g}$ 硫代硫酸钠 $\mathrm{(Na_2S_2O_3\cdot 5H_2O)}$ ，加 $0.2\mathrm{g}$ 无水碳酸钠，溶于 $1000~\mathrm{mL}$ 水中，缓缓煮沸 $10\mathrm{min}$ ，冷却，置于棕色试剂瓶中，避光保存2周后过滤，使用前按照GB/T5009.1标定。也可用相同浓度的硫代硫酸钠容量分析用标准滴定溶液。

# 3.4.2 溴标准溶液 $\left[c\left(\frac{1}{2}\mathbf{Br}_2\right) = 0.1\mathbf{mol} / \mathbf{L}\right]$

称取 $3.0\mathrm{g}$ 溴酸钾及 $25.0\mathrm{g}$ 溴化钾，溶于 $1000~\mathrm{mL}$ 水中，摇匀，使用前按照GB/T5009.1标定。也可用相同浓度的溴容量分析用标准溶液。

# 4 仪器和设备

4.1 天平：感量分别为 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$ 。  
4.2 水蒸气蒸馏装置：示意图见附录A，或自动水蒸气蒸馏仪。

# 5 分析步骤

# 5.1 试样蒸馏

称取 $1.0\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 食品接触用环氧酚醛树脂涂料试样，放入蒸馏瓶内，加入 $20\mathrm{mL}$ 乙醇溶解（水基改性环氧树脂涂料，则加入 $20\mathrm{mL}$ 水），再加入 $50\mathrm{mL}$ 水，然后用水蒸气蒸馏出游离酚，馏出溶液收集于 $500\mathrm{mL}$ 锥形瓶中，在 $40\mathrm{min} \sim 50\mathrm{min}$ 内馏出蒸馏液 $300\mathrm{mL} \sim 400\mathrm{mL}$ ，最后取5滴新蒸出样液，加1滴～2滴溴水，如无白色沉淀，证明酚已蒸完，即可停止蒸馏，蒸馏液全部转入 $500\mathrm{mL}$ 容量瓶中，用水稀释至刻度，充分摇匀，备用。

# 5.2 滴定

移取 $100~\mathrm{mL}$ 蒸馏液，置于 $500~\mathrm{mL}$ 具塞锥形瓶中，加入 $25~\mathrm{mL}$ 溴标准溶液、 $5~\mathrm{mL}$ 盐酸，在室温下盖上塞子避光反应 $15\mathrm{min}$ ，取出后加入 $10~\mathrm{mL}$ 碘化钾溶液，再立即避光放置 $10\mathrm{min}$ ，取出后加入 $1~\mathrm{mL}$ 三氯甲烷，用硫代硫酸钠标准滴定溶液滴定至淡黄色，加入 $1~\mathrm{mL}$ 淀粉指示液，继续滴定至蓝色消褪为终点。

# 5.3 空白试验

取 $20~\mathrm{mL}$ 乙醇并加水稀释至 $500~\mathrm{mL}$ ，然后移取 $100~\mathrm{mL}$ 稀释后的溶液（水基改性环氧树脂涂料，则移取 $100~\mathrm{mL}$ 水），按照5.2的方法进行测定。

# 6 分析结果的表述

样品中游离酚的含量按式(1)计算。

$$
X = \frac {1 5 . 6 8 \times \left(V _ {1} - V _ {2}\right) \times c \times V _ {3}}{m \times V _ {4}} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中游离酚的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

15.68 与 $1.0 \mathrm{~L}$ 硫代硫酸钠标准溶液 $\left[c\left(\mathrm{Na}_{2} \mathrm{~S}_{2} \mathrm{O}_{3}\right)=1.000 \mathrm{~mol} / \mathrm{L}\right]$ 相当的苯酚的质量，单位为克每摩尔 $(\mathrm{g} / \mathrm{mol})$ ；

$V_{1}$ ——试剂空白滴定消耗的硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴定试样消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫代硫酸钠标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{3}$ ——蒸馏液定容的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品的质量，单位为克 $(\mathrm{g})$ ；

$V_{4}$ ——滴定时移取蒸馏液的体积，单位为毫升 $(\mathrm{mL})$ 。

结果保留2位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 8 其他

本方法的检出限为 $0.4\mathrm{g / kg}$ ，定量限为 $2.0\mathrm{g / kg}$

# 第二部分 游离酚迁移量的测定

# 9 原理

在碱性条件 $(\mathrm{pH}9.0\sim 10.5)$ 下，游离酚在铁氰化钾催化下与4-氨基安替吡啉反应，形成安替吡啉染料，水中安替吡啉染料经三氯甲烷萃取后，用分光光度计在 $460~\mathrm{nm}$ 下测定试液的吸光度值，与标准系列比较得出浸泡液中游离酚含量，进而计算得出游离酚迁移量。其他食品模拟物、替代溶剂中安替吡啉染料，用分光光度计在 $500~\mathrm{nm}$ 下测定试液的吸光度值，与标准系列比较得出浸泡液中游离酚含量，进而计算得出游离酚迁移量。

# 10.4.3 标准中间液（ $10\mathrm{mg / L}$ ）

# 12 分析步骤

# 12.1 迁移试验

食品接触材料及制品按照GB31604.1和GB5009.156的要求进行迁移试验。迁移试验所得浸泡液如不能立即测试，可置于室温避光中保存，保存期不超过 $3\mathrm{d}$ 。若进行下一步试验，应将浸泡液恢复至室温后使用。

# 12.2 浸泡液前处理

# 12.2.1 水浸泡液的处理

移取 $200~\mathrm{mL}$ 浸泡液置于分液漏斗中，依次加入 $1.0~\mathrm{mL}$ 氯化铵缓冲溶液、 $1.0~\mathrm{mL}$ 4-氨基安替吡啉溶液、 $1.0~\mathrm{mL}$ 铁氰化钾溶液，每加入一种试剂，要充分摇匀，放置 $10\mathrm{min}$ ，加入 $10.0~\mathrm{mL}$ 三氯甲烷，振摇 $2\mathrm{min}$ ，静置分层后将三氯甲烷层经无水硫酸钠过滤到具塞比色管中待测。

若浸泡液有颜色或出现浑浊，取 $200~\mathrm{mL}$ 浸泡液于蒸馏瓶中，加入数粒玻璃珠，加热蒸馏，待蒸馏出总体积的 $90\%$ 左右停止蒸馏，稍冷后，向蒸馏瓶中加入 $25~\mathrm{mL}$ 水，继续蒸馏，直到收集 $200~\mathrm{mL}$ 的馏出液为止。移取全部蒸馏液置于分液漏斗中，按上段“依次加入 $1.0~\mathrm{mL}$ 氯化铵缓冲溶液”之后的操作进行处理得到待测液。

# 12.2.2 $4\%$ （体积分数）乙酸浸泡液的处理

移取 $50.0\mathrm{mL}$ 浸泡液置于三角瓶中，加入 $7.20\mathrm{g}$ 碳酸钠，轻摇至完全溶解，依次加入 $1.0\mathrm{mL}$ 4-氨基安替吡啉溶液、 $1.0\mathrm{mL}$ 铁氰化钾溶液，每加入一种试剂，要充分摇匀，放置 $10\mathrm{min}$ ，待测。

若浸泡液有颜色或出现浑浊，取 $50.0\mathrm{mL}$ 浸泡液于蒸馏瓶中，加入数粒玻璃珠，加热蒸馏，待蒸馏出总体积的 $90\%$ 左右停止蒸馏，稍冷后，向蒸馏瓶中加入 $10~\mathrm{mL}$ 水，继续蒸馏，直到收集 $50~\mathrm{mL}$ 的馏出液为止。移取全部蒸馏液置于三角瓶中，按上段“加入 $7.20\mathrm{g}$ 碳酸钠”之后的操作进行处理得到待测液。

# 12.2.3 $10\%$ （体积分数）乙醇浸泡液、 $20\%$ （体积分数）乙醇浸泡液的处理

移取 $50.0\mathrm{mL}$ 浸泡液置于三角瓶中，依次加入 $1.0\mathrm{mL}$ 氯化铵缓冲溶液、 $1.0\mathrm{mL}4$ -氨基安替吡啉溶

液、 $1.0 \mathrm{~mL}$ 铁氰化钾溶液，每加入一种试剂，要充分摇匀，放置 $10 \mathrm{~min}$ ，待测。

若浸泡液有颜色或出现浑浊，取 $50.0\mathrm{mL}$ 浸泡液于蒸馏瓶中，加入数粒玻璃珠，加热蒸馏，待蒸馏出总体积的 $90\%$ 左右停止蒸馏，稍冷后，向蒸馏瓶中加入 $10~\mathrm{mL}$ 水，继续蒸馏，直到收集 $50~\mathrm{mL}$ 的馏出液为止。移取全部蒸馏液置于三角瓶中，按上段“依次加入 $1.0~\mathrm{mL}$ 氯化铵缓冲溶液”之后的操作进行处理得到待测液。

# 12.2.4 $50\%$ （体积分数）乙醇浸泡液、 $95\%$ （体积分数）乙醇（化学替代溶剂)浸泡液的处理

# 13 分析结果的表述

# 13.1 非密封制品类食品接触材料及制品游离酚(以苯酚计)的迁移量的计算(以 $\mathrm{mg / kg}$ 表示)

非密封制品类食品接触材料及制品游离酚(以苯酚计)迁移量以 $\mathrm{mg / kg}$ 表示时，按式(2)进行计算。

$$
X _ {1} = \frac {\left(c - c _ {0}\right) \times V}{S} \times k \times F \quad \dots \tag {2}
$$

式中：

$X_{1}$ ——游离酚(以苯酚计)的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样浸泡液中游离酚(以苯酚计)的含量, 单位为毫克每升 $(\mathrm{mg} / \mathrm{L})$ ；

$c_{0}$ ——空白浸泡液中游离酚(以苯酚计)的含量，单位为毫克每升 $(\mathrm{mg / L})$

$V$ ——试样浸泡液体积，单位为升 $(\mathrm{L})$ ；

$S$ ——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$k$ ——显色反应前浸泡液的稀释倍数，其中 $50\%$ （体积分数）乙醇溶液、 $95\%$ （体积分数）乙醇溶液稀释倍数均为 5，其他食品模拟物浸泡液、化学替代溶剂浸泡液中稀释倍数均为 1；

$F$ ——食品接触材料及制品在可预见使用情形下实际的接触面积与食品体积(质量)之间的比值 $(S / V)$ ，各种液态食品密度通常以 $1\mathrm{kg} / \mathrm{L}$ 计，单位为平方分米每千克 $(\mathrm{dm}^2 /\mathrm{kg})$ 。当实际 $S / V$ 已知时， $F$ 即为可预见使用情形下的最大 $S / V$ ；当实际 $S / V$ 未知时， $F$ 采用 $6\mathrm{dm}^2 /\mathrm{kg}$ ，即 $6\mathrm{dm}^2$ 食品接触材料及制品接触 $1\mathrm{kg}$ 食品。

结果保留2位有效数字。

# 13.2 密封制品类食品接触材料及制品游离酚(以苯酚计)迁移量的计算(以 $\mathrm{mg / kg}$ 表示)

对于密封制品类食品接触材料及制品，游离酚(以苯酚计)迁移量以 $\mathrm{mg / kg}$ 表示时，按式(3)进行计算。

$$
X _ {2} = \frac {\left(c - c _ {0}\right) \times V}{S} \times k \times \frac {S _ {0}}{V _ {3}} \quad \dots \tag {3}
$$

式中：

$X_{2}$ ——游离酚(以苯酚计)的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样浸泡液中游离酚(以苯酚计)的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

$c_{0}$ ——空白浸泡液中游离酚(以苯酚计)的含量，单位为毫克每升 $(\mathrm{mg / L})$

$V$ ——试样浸泡液的体积，单位为升 $(\mathrm{L})$ ；

$S$ ——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$k$ ——显色反应前浸泡液的稀释倍数，其中 $50\%$ （体积分数）乙醇溶液、 $95\%$ （体积分数）乙醇溶液稀释倍数均为5，其他食品模拟物浸泡液、化学替代溶剂浸泡液中稀释倍数均为1；

$S_{0}$ ——密封制品实际使用中与食品接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$V_{3}$ ——密封制品实际使用容器盛装食品的质量，单位为千克 $(\mathrm{kg})$ ，各种液态食品按密度为 $1\mathrm{kg} / \mathrm{L}$ 将其体积换算为相应的质量。

结果保留2位有效数字。

# 13.3 密封制品类食品接触材料及制品游离酚(以苯酚计)迁移量的计算(以 $\mathbf{mg}$ /件表示）

密封制品类食品接触材料及制品游离酚(以苯酚计)迁移量以 $\mathrm{mg}/$ 件表示时，按式(4)计算，需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {3} = \frac {\left(c - c _ {0}\right) \times V}{n} \times k \quad \dots \tag {4}
$$

式中：

$X_{3}$ —游离酚(以苯酚计)特定迁移量，单位为毫克每件 $(\mathrm{mg / }$ 件）；

$c$ ——试样浸泡液中游离酚(以苯酚计)的含量, 单位为毫克每升 $(\mathrm{mg} / \mathrm{L})$ ；

$c_{0}$ ——空白浸泡液中游离酚(以苯酚计)的含量,单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样浸泡液的体积，单位为升 $(\mathrm{L})$ ；

$n$ ——浸泡用密封制品的数量，单位为件；

$k$ ——显色反应前浸泡液的稀释倍数, 其中 $50\%$ (体积分数) 乙醇溶液、 $95\%$ (体积分数) 乙醇溶液稀释倍数均为 5 , 其他食品模拟物浸泡液、化学替代溶剂浸泡液中稀释倍数均为 1 。

结果保留2位有效数字。

$X_{4}$ ——苯酚的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样浸泡液中苯酚的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白浸泡液中苯酚的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

S——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$F$ ——食品接触材料及制品在可预见使用情形下实际的接触面积与食品体积(质量)比 $(S / V)$ ，各种液态食品密度通常以 $1\mathrm{kg} / \mathrm{L}$ 计，单位为平方分米每千克 $(\mathrm{dm}^2 /\mathrm{kg})$ 。当实际 $S / V$ 已知时， $F$ 即为可预见使用情形下的最大 $S / V$ ；当实际 $S / V$ 未知时， $F$ 采用 $6\mathrm{dm}^2 /\mathrm{kg}$ ，即 $6\mathrm{dm}^2$ 食品接触材料及制品接触 $1\mathrm{kg}$ 食品。

结果保留2位有效数字。

# 20.2 密封制品类食品接触材料及制品苯酚特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

对于密封制品类食品接触材料及制品，苯酚特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(6)进行计算。

$$
X _ {5} = \frac {\left(c - c _ {0}\right) \times V}{S} \times \frac {S _ {0}}{V _ {3}} \quad \dots \tag {6}
$$

式中：

$X_{5}$ ——苯酚的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样浸泡液中苯酚的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白浸泡液中苯酚的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

S——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$S_{0}$ ——密封制品实际使用中与食品接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$V_{3}$ ——密封制品实际使用容器盛装食品的质量，单位为千克 $(\mathrm{kg})$ ，各种液态食品按密度为 $1\mathrm{kg} / \mathrm{L}$ 将其体积换算为相应的质量。

结果保留2位有效数字。

# 20.3 密封制品类食品接触材料及制品苯酚特定迁移量的计算（以 $\mathbf{mg}$ /件表示）

密封制品类食品接触材料及制品苯酚特定迁移量以 $\mathrm{mg}/$ 件表示时，按式(7)计算，需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {6} = \frac {\left(c - c _ {0}\right) \times V}{n} \tag {7}
$$

式中：

$X_{6}$ ——苯酚的特定迁移量，单位为毫克每件 $(\mathrm{mg}/$ 件）；

$c$ ——试样浸泡液中苯酚的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白浸泡液中苯酚的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

$n$ ——浸泡用密封制品的数量，单位为件。

结果保留2位有效数字。

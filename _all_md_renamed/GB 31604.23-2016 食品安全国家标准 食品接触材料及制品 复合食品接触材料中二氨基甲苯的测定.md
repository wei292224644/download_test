# 前言

本标准代替GB/T5009.119—2003《复合食品包装袋中二氨基甲苯的测定》。

本标准与GB/T5009.119—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 复合食品接触材料中二氨基甲苯的测定”；  
——修改了迁移试验条件；  
——修改了衍生试剂的选择；  
——修改了色谱柱的选择；  
——增加了气相色谱-质谱法。

# 复合食品接触材料中二氨基甲苯的测定

# 1 范围

本标准规定了复合食品包装袋中二氨基甲苯的测定方法。

本标准适用于复合食品包装袋中二氨基甲苯的测定。

# 第一法 气相色谱法

# 2 原理

试样中二氨基甲苯用 $4\%$ 乙酸溶液浸出，将浸出液冷却后，在碱性条件下经二氯甲烷提取后，加七氟丁酸酐衍生化，然后将衍生物注入带有电子捕获检测器的气相色谱仪测定，以保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的三级水。

# 3.1 试剂

3.1.1 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：色谱纯。  
3.1.2 叔丁基甲醚 $\left(\mathrm{C}_{5} \mathrm{H}_{12} \mathrm{O}\right)$ ：色谱纯。  
3.1.3 七氟丁酸酐 $\left(\mathrm{C}_{8} \mathrm{~F}_{14} \mathrm{O}_{3}\right)$ ：色谱纯。  
3.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.5 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.6 冰乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
3.1.7 氢氧化钠 $(\mathrm{NaOH})$ 。

# 3.2 试剂配制

3.2.1 乙酸溶液 $(4\%)$ ：称取 $40\mathrm{g}$ 冰乙酸溶于 $960~\mathrm{mL}$ 水中。  
3.2.2 碳酸氢钠溶液 $(20\mathrm{g} / \mathrm{L})$ ：称取 $2\mathrm{g}$ 碳酸氢钠溶于水中，定容至 $100~\mathrm{mL}$ 。  
3.2.3 氢氧化钠溶液 $(500\mathrm{g / L})$ ：称取 $50\mathrm{g}$ 氢氧化钠溶于水中，定容至 $100~\mathrm{mL}$ 。

# 3.3 标准品

2,4-二氨基甲苯 $\mathrm{(C_7H_{10}N_2}$ ,CAS号：95-80-7)，纯度 $\geq 99\%$ 。或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 二氨基甲苯标准储备溶液：称取二氨基甲苯 $10\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ )于 $50~\mathrm{mL}$ 小烧杯中，加二氯甲烷溶解，用二氯甲烷转移到 $100~\mathrm{mL}$ 容量瓶中，定容，混匀，于 $4^{\circ}C$ 保存，其中二氨基甲苯的质量浓度为 $100\mu \mathrm{g / mL}$ 。  
3.4.2 二氨基甲苯标准工作溶液：吸取标准储备溶液 $0.50\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，用二氯甲烷定容至刻度，混匀，于 $4^{\circ}\mathrm{C}$ 保存，其中二氨基甲苯的质量浓度为 $1\mu \mathrm{g / mL}$

# 4 仪器和设备

4.1 气相色谱仪：带电子捕获检测器。  
4.2 分析天平：感量为 $0.01\mathrm{mg}$ 和 $0.01\mathrm{g}$   
4.3 氮吹仪。  
4.4 pH计：精度为0.1。

# 5 分析步骤

# 5.1 试样制备

5.1.1 未装过食品的包装袋：用水洗3次，淋干，按 $2\mathrm{mL} / \mathrm{cm}^2$ 计算装入乙酸溶液，热封口。  
5.1.2 装过食品的包装袋：剪口，将食品全部移出，用清水冲至无污物，再用水冲洗3次，淋干后按 $2\mathrm{mL} / \mathrm{cm}^2$ 计算装入乙酸溶液，热封口。

# 5.2 迁移试验

将上述5.1.1或5.1.2热封口后的包装袋(使用温度为 $60~\mathrm{^\circ C}\sim 120~\mathrm{^\circ C}$ )，置于预先调至 $120~^\circ \mathrm{C}\pm 5~^\circ \mathrm{C}$ 的烘箱内，恒温 $40\mathrm{min}$ ，取出自然放冷至室温，剪开封口，将提取液移入干燥的烧杯中备用。

使用温度低于 $60^{\circ}\mathrm{C}$ 的包装袋，置于预先调至 $60^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 的烘箱内，恒温 $2\mathrm{h}$ ，取出自然放冷至室温，剪开封口，将水移入干燥的烧杯中备用。

# 5.3 衍生过程

量取试样 $50.0\mathrm{mL}$ ，置于分液漏斗中，用氢氧化钠溶液调节 $\mathrm{pH}$ 为8.0，混匀，再加入 $10\mathrm{g}$ 氯化钠混匀，然后用 $10\mathrm{mL}$ 二氯甲烷分别萃取两次，每次萃取 $5\mathrm{min}$ ，静置 $10\mathrm{min}$ 。合并两次萃取液，经无水硫酸钠脱水后在 $40^{\circ}\mathrm{C}$ 下氮吹至近干，加入 $2\mathrm{mL}$ 二氯甲烷，混匀，再加入 $100\mu\mathrm{L}$ 七氟丁酸酐，轻轻混匀，置于室温下进行衍生化反应 $15\mathrm{min}$ 。

将上述反应液移入 $60~\mathrm{mL}$ 分液漏斗中，用 $2\mathrm{mL}$ 二氯甲烷分数次洗净浓缩瓶，洗液并入分液漏斗中，加入 $5\mathrm{mL}$ 碳酸氢钠溶液，轻轻摇动 $2\mathrm{min}$ ，静置 $5\mathrm{min}$ ，将二氯甲烷层移入到 $10\mathrm{mL}$ 试管中，在 $40^{\circ}\mathrm{C}$ 下氮吹至近干，用叔丁基甲醚溶解并定容至 $5.00~\mathrm{mL}$ ，注入气相色谱仪分析。

# 5.4 二氨基甲苯标准测定液的制备

分别吸取适量的二氨基甲苯工作液，加入 $100\mu \mathrm{L}$ 七氟丁酸酐，轻轻混匀，置于室温下进行衍生化反应 $15\mathrm{min}$ 。将上述反应液移入 $60~\mathrm{mL}$ 分液漏斗中，用 $2\mathrm{mL}$ 二氯甲烷分数次洗净浓缩瓶，洗液并入分液漏斗中，加入 $5\mathrm{mL}$ 碳酸氢钠溶液，轻轻摇动 $2\mathrm{min}$ ，静置 $5\mathrm{min}$ ，将二氯甲烷层移入到 $10~\mathrm{mL}$ 试管中，在 $40^{\circ}\mathrm{C}$ 下氮吹至近干，用叔丁基甲醚溶解并定容至 $5\mathrm{mL}$ 。

# 5.5 气相色谱参考条件

a）色谱柱：HP-5MS，柱长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ 或同等性能的色谱柱；  
b）柱温度程序：初始温度 $60~\mathrm{^\circ C}$ 保持 $2\mathrm{min}$ ，以 $15^{\circ}C / \mathrm{min}$ 升温至 $240^{\circ}C$ ，保持 $5\mathrm{min}$   
c）进样口温度： $200^{\circ}C$   
d）载气：氮气，纯度 $\geqslant 99.999\%$ ， $1.0~\mathrm{mL / min}$ ；尾吹气， $30~\mathrm{mL / min}$   
e）检测器：电子捕获检测器(ECD)，温度 $300^{\circ}\mathrm{C}$   
f）进样方式：不分流进样；  
g）进样量： $1.0\mu \mathrm{L}$

# 5.6 试样溶液的测定

将试样溶液注入气相色谱仪中，色谱图见图A.1，得到峰高或峰面积，以标准工作液的浓度为横坐标，以色谱峰高或峰面积为纵坐标，绘制标准曲线。根据标准曲线得到待测液中二氨基甲苯的浓度。

# 6 分析结果的表述

试样中二氨基甲苯的含量按式(1)计算：

$$
X = \frac {\rho \times V _ {2} \times 1 0 0 0}{V _ {1} \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中二氨基甲苯的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho$ ——由标准曲线求得试样溶液中二氨基甲苯的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V_{2}$ ——最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——试液体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数。

结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 检出限及定量限

当取样量 $50.0\mathrm{mL}$ ，定容体积为 $5.00~\mathrm{mL}$ 时，本方法的检出限为 $0.0005\mathrm{mg / L}$ ，定量限为 $0.001\mathrm{mg / L}$ 。

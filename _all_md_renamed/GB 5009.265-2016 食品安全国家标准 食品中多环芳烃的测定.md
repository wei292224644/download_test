# 食品安全国家标准

# 食品中多环芳烃的测定

# 1 范围

本标准规定了食品中多环芳烃(萘、苊、芴、菲、蒽、荧蒽、芘、苯并[a]蒽、萜、苯并[b]荧蒽、苯并[k]荧蒽、苯并[a]芘、茚并[1,2,3-c,d]芘、二苯并[a,h]蒽和苯并[g,h,i]芘的液相色谱测定方法和食品中多环芳烃(萘、苊烯、苊、芴、菲、蒽、荧蒽、芘、苯并[a]蒽、萜、苯并[b]荧蒽、苯并[k]荧蒽、苯并[a]芘、茚并[1,2,3-c,d]芘、二苯并[a,h]蒽和苯并[g,h,i]芘的气相色谱-质谱测定方法。

本标准适用于食品中多环芳烃含量的测定。

# 第一法 高效液相色谱法

# 2 原理

试样中的多环芳烃用有机溶剂提取，提取液浓缩至近干，溶剂溶解，用PSA（N-丙基乙二胺)和 $\mathrm{C_{18}}$ 固相萃取填料净化或用弗罗里硅土固相萃取柱净化。经浓缩定容后，通过高效液相色谱分离，测定各种多环芳烃在不同激发波长和发射波长处的荧光强度，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
3.1.3 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：色谱纯。  
3.1.4 硅藻土：色谱纯。  
3.1.5 硫酸镁 $(\mathrm{MgSO}_4)$ ：优级纯。  
3.1.6 $N$ -丙基乙二胺(PSA)：粒径 $40~\mu \mathrm{m}$   
3.1.7 封尾 $\mathrm{C}_{18}$ 固相萃取填料：粒径 $40\mu \mathrm{m}\sim 63\mu \mathrm{m}$   
3.1.8 弗罗里硅土固相萃取柱： $500\mathrm{mg}, 3\mathrm{mL}$ 。  
3.1.9 有机相型微孔滤膜： $0.22 \mu \mathrm{m}$ 。

# 3.2 试剂配制

3.2.1 正己烷-二氯甲烷混合溶液 $(1 + 1)$ ：量取 $500~\mathrm{mL}$ 正己烷，加入二氯甲烷 $500~\mathrm{mL}$ ，混匀。  
3.2.2 乙腈饱和的正己烷：量取 $800~\mathrm{mL}$ 正己烷，加入 $200~\mathrm{mL}$ 乙腈，振摇混匀后，静置分层，上层正己烷层即为乙腈饱和的正己烷。

# 3.3 标准品

多环芳烃(萘、苊烯、苊、芴、菲、蒽、荧蒽、芘、苯并[a]蒽、萜、苯并[b]荧蒽、苯并[k]荧蒽、苯并[a]芘、茚并[1,2,3-c,d]芘、二苯并[a,h]蒽和苯并[g,h,i]芘有证标准溶液 $(200\mu \mathrm{g / mL})$ ，于一 $18^{\circ}C$ 下保存。

警告——多环芳烃是已知的致癌、致畸、致突变的物质，并且致癌性随着苯环数的增加而增加，测定时应特别注意安全防护。测定应在通风柜中进行并戴手套，尽量减少暴露。

# 3.4 标准溶液配制

3.4.1 多环芳烃标准中间液 $(1000\mathrm{ng / mL})$ ：吸取多环芳烃标准溶液 $0.5\mathrm{mL}$ ，用乙腈定容至 $100~\mathrm{mL}$ 。在 $-18^{\circ}C$ 下保存。  
3.4.2 多环芳烃标准系列工作液: 分别吸取多环芳烃标准中间液 $0.10 \mathrm{~mL}, 0.50 \mathrm{~mL}, 1.0 \mathrm{~mL}, 2.0 \mathrm{~mL}, 5.0 \mathrm{~mL}, 10.0 \mathrm{~mL}$ , 用乙腈定容至 $100 \mathrm{~mL}$ , 得到质量浓度为 $1 \mathrm{ng} / \mathrm{mL}, 5 \mathrm{ng} / \mathrm{mL}, 10 \mathrm{ng} / \mathrm{mL}, 20 \mathrm{ng} / \mathrm{mL}, 50 \mathrm{ng} / \mathrm{mL}, 100 \mathrm{ng} / \mathrm{mL}$ 的标准系列工作液。

# 4 仪器和设备

4.1 高效液相色谱仪，带荧光检测器。  
4.2 电子天平：感量为 $0.01\mathrm{g}$ 。  
4.3 冷冻离心机：转速 $\geqslant 4500\mathrm{r / min}$   
4.4 涡旋振荡器。  
4.5 超声波振荡器。  
4.6 粉碎机。  
4.7 均质器。  
4.8 氮吹仪。  
4.9 旋转蒸发仪。

# 5 分析步骤

# 5.1 试样制备

5.1.1 干样: 取样品约 $500 \mathrm{~g}$ , 经粉碎机粉碎、混匀, 分装于洁净盛样袋中, 密封标识后于 $-18^{\circ} \mathrm{C}$ 冷冻保存。  
5.1.2 湿样：取样品约 $500\mathrm{g}$ ，将其可食部分先切碎，经均质器充分搅碎均匀，分装于洁净盛样袋中，密封标识后于 $-18^{\circ}\mathrm{C}$ 冷冻保存。

# 5.2 试样提取

# 5.2.1 粮谷或水分少的食品

称取 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 试样于 $50~\mathrm{mL}$ 具塞玻璃离心管A中，按以下步骤处理：

a) 加入 $10 \mathrm{~mL}$ 正己烷, 涡旋振荡 $30 \mathrm{~s}$ 后, 放入 $40^{\circ} \mathrm{C}$ 水浴超声 $30 \mathrm{~min}$ ; 以 $4500 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ,吸取上清液于玻璃离心管 B 中; 离心管 A 下层用 $10 \mathrm{~mL}$ 正己烷重复提取 1 次, 提取液合并于离心管 B 中, 氮吹 (温度控制在 $35^{\circ} \mathrm{C}$ 以下) 除去溶剂, 吹至近干。  
b）在离心管B中，加入 $4\mathrm{mL}$ 乙腈，涡旋混合 $30~s$ ，再加入 $900~\mathrm{mg}$ 硫酸镁、 $100~\mathrm{mg}$ PSA和 $100~\mathrm{mg}$

$\mathrm{C}_{18}$ 填料, 涡旋混合 $30 \mathrm{~s}$ , 以 $4500 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ , 取上清液于 $10 \mathrm{~mL}$ 玻璃刻度离心管 C 中, 离心管 B 下层再用 $2 \mathrm{~mL}$ 乙腈重复提取 1 遍, 合并提取液于离心管 C 中, 氮吹蒸发溶剂至近 $1 \mathrm{~mL}$ , 用乙腈定容至 $1 \mathrm{~mL}$ , 混匀后, 过 $0.22 \mu \mathrm{m}$ 有机相型微孔滤膜, 制得试样待测液。

# 5.2.2 水产品、肉类和蔬菜等食品

称取 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ 试样于 $50~\mathrm{mL}$ 具塞玻璃离心管A中，加 $1\mathrm{g}\sim 5\mathrm{g}$ 硅藻土，用玻棒搅匀，以下按5.2.1中a）、b)步骤处理，制得试样待测液。

# 5.2.3 含油脂高的食品或动植物油脂

称取 $1\mathrm{g}\sim 4\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 试样于 $50~\mathrm{mL}$ 具塞玻璃离心管A中，按以下步骤处理：

a）加入 $20~\mathrm{mL}$ 乙腈和 $10~\mathrm{mL}$ 乙腈饱和的正己烷，涡旋振荡 $30~\mathrm{s}$ 后，放入 $40^{\circ}\mathrm{C}$ 水浴超声 $30\mathrm{min}$ ；摇匀后，以 $4500~\mathrm{r / min}$ 冷冻 $(-4^{\circ}\mathrm{C})$ 离心 $5\mathrm{min}$ ，吸取下层乙腈层于 $100~\mathrm{mL}$ 鸡心瓶中，离心管A中溶液用 $20~\mathrm{mL}$ 乙腈重复提取1次，提取液合并于鸡心瓶中， $35^{\circ}\mathrm{C}$ 减压旋转蒸发至近干。加入 $5\mathrm{mL}$ 正己烷，涡旋振荡 $30~\mathrm{s}$ 溶解。  
b）依次用 $5\mathrm{mL}$ 二氯甲烷和 $10\mathrm{mL}$ 正己烷活化弗罗里硅土固相萃取柱，将a)获得的 $5\mathrm{mL}$ 提取样液全部移入弗罗里硅土固相萃取柱，再用 $5\mathrm{mL}$ 正己烷洗涤鸡心瓶并入柱中，用 $8\mathrm{mL}$ 正己烷-二氯甲烷混合溶液洗脱，收集所有流出物于 $20\mathrm{mL}$ 玻璃离心管B中。氮吹(温度控制在 $35^{\circ}C$ 以下)除去溶剂，吹至近干，加入 $0.5\mathrm{mL}$ 乙腈涡旋振荡 $10~s$ ，继续氮吹至除尽正己烷-二氯甲烷，用乙腈定容至 $1\mathrm{mL}$ ，混匀后，过 $0.22\mu \mathrm{m}$ 有机相型微孔滤膜，制得试样待测液。

# 5.4 标准曲线的制作

将标准系列工作液分别注入液相色谱仪中，测得相应的峰面积，以标准工作液的质量浓度为横坐标、以峰面积为纵坐标，绘制标准曲线。标准溶液的液相色谱图参见图A.1。

# 5.5 试样溶液的测定

将试样待测液注入液相色谱仪中，以保留时间定性，测得相应的峰面积，根据标准曲线得到试样待测液中多环芳烃的质量浓度。如果试样待测液中被测物质的响应值超出仪器检测的线性范围，可适当稀释后测定。

# 5.6 空白试验

空白试验除不加试样外，采用与试样完全相同的分析步骤。

# 6 分析结果的表述

试样中多环芳烃的含量 $X_{i}$ 按式(1)计算：

$$
X _ {i} = \frac {\rho_ {i} \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X_{i}$ ——试样中多环芳烃的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho_{i}$ ——依据标准曲线计算得到的试样待测液中多环芳烃 $i$ 的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样待测液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000 单位转换；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

含量大于等于 $10\mu \mathrm{g / kg}$ 时，保留三位有效数字；含量小于 $10~\mu \mathrm{g / kg}$ 时，保留两位有效数字。

注：计算结果应扣除空白值。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 第二法 气相色谱-质谱法

# 9 原理

试样中的多环芳烃用有机溶剂提取，提取液浓缩至近干，用PSA（N-丙基乙二胺)和 $\mathrm{C_{18}}$ 固相萃取填料净化或用弗罗里硅土固相萃取柱净化，经浓缩定容后，用气相色谱-质谱联用仪进行测定，外标法定量。

同第3章。

11.2 其余同 $4.2\sim 4.9$

# 12.4 气相色谱-质谱参考条件

a）色谱柱：DB-5MS，柱长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ，或同等性能的色谱柱。  
b）柱温度程序：初始温度 $90^{\circ}\mathrm{C}$ ，以 $20^{\circ}\mathrm{C / min}$ 升温至 $220^{\circ}\mathrm{C}$ ，再以 $5^{\circ}\mathrm{C / min}$ 升温至 $320^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$   
c）进样口温度： $250~\mathrm{^\circ C}$   
d） 色谱-质谱接口温度： $280^{\circ}C$   
e) 离子源温度: $230^{\circ} \mathrm{C}$ ;   
f) 载气：氦气，纯度 $\geq 99.999\%$ ，1.0 mL/min；  
g） 电离方式：EI；  
h） 电离能量： $70\mathrm{eV}$   
i) 质量扫描范围： $50\mathrm{amu}\sim 450\mathrm{amu}$   
j） 测定方式：选择离子监测方式；  
k）进样方式：不分流进样，2.0min后开阀；  
1) 进样量: $1.0 \mu \mathrm{L}$ ;   
m）溶剂延迟： $3\mathrm{min}$

# 12.5 标准曲线的制作

将标准系列工作液分别注入气相色谱-质谱仪中，测得相应的峰面积，以标准工作液的质量浓度为横坐标、以峰面积为纵坐标，绘制标准曲线。

# 12.6 试样溶液的测定

将试样待测液注入气相色谱-质谱仪中，测得相应的峰面积，根据标准曲线得到试样待测液中多环芳烃的质量浓度。如果试样待测液中被测物质的响应值超出仪器检测的线性范围，可适当稀释后测定。

# 12.7 定性

# 13 分析结果的表述

试样中多环芳烃的含量 $X_{i}$ 按式(2)计算：

$$
X _ {i} = \frac {\rho_ {i} \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X_{i}$ ——试样中多环芳烃的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho_{i}$ ——依据标准曲线计算得到的试样待测液中多环芳烃 $i$ 的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样待测液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000 单位转换；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

含量大于等于 $10\mu \mathrm{g / kg}$ 时，保留三位有效数字；含量小于 $10~\mu \mathrm{g / kg}$ 时，保留两位有效数字。

注：计算结果应扣除空白值。

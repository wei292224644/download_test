# 食品安全国家标准

# 食品中氯酸盐和高氯酸盐的测定

# 1 范围

本标准规定了食品中氯酸盐和高氯酸盐的液相色谱-串联质谱测定方法。

本标准适用于蔬菜及其制品、水果及其制品、谷物及其制品、肉及肉制品、水产品、蛋及蛋制品、乳及乳制品、调味品、饮料类、婴幼儿配方食品和婴幼儿辅助食品、茶叶中氯酸盐和高氯酸盐的测定。

# 2 原理

试样中的氯酸盐和高氯酸盐用酸化的乙腈水混合溶液提取，固相萃取柱净化，采用高效液相色谱分离，串联质谱检测，同位素内标法定量。

# 3 试剂和材料

除非另有说明，本方法所有试剂均为色谱级，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ 。

3.1.2 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。

3.1.3 甲酸（HCOOH）。

# 3.2 试剂配制

3.2.1 $0.1\%$ 甲酸水溶液：取 $1\mathrm{mL}$ 甲酸，用水稀释定容至 $1000~\mathrm{mL}$ ，混匀。  
3.2.2 乙腈-甲酸水溶液：取 $60~\mathrm{mL}$ 乙腈，用 $0.1\%$ 甲酸水溶液定容至 $100~\mathrm{mL}$ ，混匀。

# 3.3 标准品

3.3.1 氯酸钠 $\left(\mathrm{NaClO}_{3},\mathrm{CAS}\right.$ 号：7775-09-9）：纯度 $>99\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.2 高氯酸钠 $\mathrm{(NaClO_4}$ ，CAS号：7601-89-0）：纯度 $>99\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.3 $^{18}\mathrm{O}_3$ -氯酸盐（氯酸盐同位素内标）： $100\mu \mathrm{g / mL}$ （以 ${}^{18}\mathrm{O}_3$ -氯酸根计）。  
3.3.4 $^{18}\mathrm{O}_4$ -高氯酸盐(高氯酸盐同位素内标)： $100\mu \mathrm{g / mL}$ （以 $^{18}\mathrm{O}_4$ -高氯酸根计）。

# 3.4 标准溶液配制

3.4.1 氯酸盐标准储备液 $(1.0\mathrm{mg / mL}$ ，以氯酸根计）：准确称取氯酸钠 $1.2801\mathrm{g}$ ，用水溶解并定容至 $1000~\mathrm{mL}$ ，混匀。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期为1年。  
3.4.2 高氯酸盐标准储备液 $(1.0\mathrm{mg / mL}$ ，以高氯酸根计)：准确称取高氯酸钠 $1.2300\mathrm{g}$ ，用水溶解并定

容至 $1000~\mathrm{mL}$ ，混匀。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期为1年。

注：氯酸盐和高氯酸盐标准储备液可采用经国家认证并授予标准物质证书的标准品。

3.4.3 高氯酸盐标准储备中间液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取 $10.0~\mathrm{mL}$ 高氯酸盐标准储备液，置于 $100~\mathrm{mL}$ 容量瓶中，用水稀释定容至刻度，混匀，将溶液转移至试剂瓶中。 $2^{\circ}\mathrm{C}\sim 8^{\circ}\mathrm{C}$ 保存，有效期为1年。  
3.4.4 氯酸盐和高氯酸盐混合标准中间液：分别吸取 $1.00\mathrm{mL}$ 氯酸盐标准储备液和 $1.00\mathrm{mL}$ 高氯酸盐标准储备中间液，置于同一 $100\mathrm{mL}$ 容量瓶中，用水稀释并定容至刻度，摇匀，制成氯酸盐、高氯酸盐质量浓度分别为 $10~\mu \mathrm{g / mL}$ 和 $1.0~\mu \mathrm{g / mL}$ 的混合标准中间液，将溶液转移至试剂瓶中。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期为3个月。  
3.4.5 氯酸盐和高氯酸盐混合标准使用液 I：吸取 $1.00 \mathrm{~mL}$ 氯酸盐和高氯酸盐混合标准中间液，置于 $10 \mathrm{~mL}$ 容量瓶中，用水稀释并定容至刻度，摇匀，制成氯酸盐、高氯酸盐质量浓度分别为 $1000 \mathrm{ng} / \mathrm{mL}$ 和 $100 \mathrm{ng} / \mathrm{mL}$ 的混合标准使用液，将溶液转移至试剂瓶中。 $2^{\circ} \mathrm{C} \sim 8^{\circ} \mathrm{C}$ 保存，有效期为 1 个月。  
3.4.6 氯酸盐和高氯酸盐混合标准使用液Ⅱ：吸取 $1.00 \mathrm{~mL}$ 氯酸盐和高氯酸盐混合标准使用液I，置于 $10 \mathrm{~mL}$ 容量瓶中，用水稀释并定容至刻度，摇匀，制成氯酸盐、高氯酸质量盐浓度分别为 $100 \mathrm{ng} / \mathrm{mL}$ 和 $10 \mathrm{ng} / \mathrm{mL}$ 的混合标准使用液，将溶液转移至试剂瓶中。 $2^{\circ} \mathrm{C} \sim 8^{\circ} \mathrm{C}$ 保存，有效期为1个月。  
3.4.7 氯酸盐和高氯酸盐同位素内标混合标准中间液：吸取 $0.50 \mathrm{~mL}$ 氯酸盐同位素内标和 $50 \mu \mathrm{L}$ 高氯酸盐同位素内标，置于同一 $10 \mathrm{~mL}$ 容量瓶中，用水稀释并定容至刻度，摇匀，制成氯酸盐内标、高氯酸盐内标质量浓度分别为 $5.0 \mu \mathrm{g} / \mathrm{mL}$ 和 $0.50 \mu \mathrm{g} / \mathrm{mL}$ 的混合标准中间液。 $2^{\circ} \mathrm{C} \sim 8^{\circ} \mathrm{C}$ 保存，有效期为3个月。  
3.4.8 氯酸盐和高氯酸盐同位素内标混合标准使用液：吸取 $2.00 \mathrm{~mL}$ 氯酸盐和高氯酸盐同位素内标混合标准中间液，置于 $10 \mathrm{~mL}$ 容量瓶中，用水稀释并定容至刻度，摇匀，制成氯酸盐内标和高氯酸盐内标质量浓度分别为 $1.0 \mu \mathrm{g} / \mathrm{mL}$ 和 $0.10 \mu \mathrm{g} / \mathrm{mL}$ 的混合标准使用液。 $2^{\circ} \mathrm{C} \sim 8^{\circ} \mathrm{C}$ 保存，有效期为3个月。  
3.4.9 氯酸盐和高氯酸盐标准系列液：分别吸取 $0\mu \mathrm{L},250\mu \mathrm{L},500\mu \mathrm{L}$ 氯酸盐和高氯酸盐混合标准使用液Ⅱ， $100\mu \mathrm{L},500\mu \mathrm{L},1000\mu \mathrm{L},5000\mu \mathrm{L}$ 氯酸盐和高氯酸盐混合标准使用液I及 $200\mu \mathrm{L}$ 同位素内标混合标准使用液，用水稀释并定容至 $10~\mathrm{mL}$ 。氯酸盐质量浓度依次为 $0\mathrm{ng / mL},2.5\mathrm{ng / mL},$ $5.0\mathrm{ng / mL},10\mathrm{ng / mL},50\mathrm{ng / mL},100\mathrm{ng / mL}$ 和 $500\mathrm{ng / mL}$ ，高氯酸盐质量浓度依次为 $0\mathrm{ng / mL},$ $0.25\mathrm{ng / mL},0.50\mathrm{ng / mL},1.0\mathrm{ng / mL},5.0\mathrm{ng / mL},10\mathrm{ng / mL}$ 和 $50\mathrm{ng / mL}$ ，标准工作液中氯酸盐同位素内标溶液质量浓度为 $20\mathrm{ng / mL}$ ，高氯酸盐同位素内标溶液浓度为 $2.0\mathrm{ng / mL}$ 。临用现配。

# 3.5 材料

3.5.1 固相萃取柱：石墨化炭黑柱 $(500\mathrm{mg},6\mathrm{mL})$ 或性能相当者，临用前依次用 $6\mathrm{mL}$ 甲醇活化、 $6\mathrm{mL}$ 乙腈-甲酸水溶液淋洗平衡，上样前排出柱体中残留的溶液。  
3.5.2 $\mathrm{Ag / H}$ 柱(1g, $2.5\mathrm{mL}$ )或性能相当者，临用前用 $10~\mathrm{mL}$ 水淋洗，静置活化 $30\mathrm{min}$ 。  
3.5.3 聚丙烯刻度离心管： $50~\mathrm{mL}$   
3.5.4 再生纤维素滤膜： $0.22\mu \mathrm{m}$

# 4 仪器和设备

4.1 液相色谱-串联质谱仪：配有电喷雾离子源。  
4.2 电子天平：感量分别为 $0.01\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.3 涡旋混合器。  
4.4 匀浆机。  
4.5 粉碎机。  
4.6 离心机：最高转速 $\geqslant 10000\mathrm{r / min}$   
4.7 超声波提取仪。

4.8 加液器。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固态样品

5.1.1.1 干样：谷物、茶叶、干制水果和蔬菜、焙烤食品等低含水量样品，取可食部分，必要时经高速粉碎机粉碎均匀；对于固体乳制品、面粉等呈均匀状的粉状样品，摇匀。  
5.1.1.2 鲜样：蔬菜、水果、水产品等高含水量样品取可食部分，洗净，晾干表面水分，匀浆均匀；对于肉类、蛋类等样品取可食部分匀浆均匀。  
5.1.2 液态样品：饮料类、调味品等样品摇匀。  
5.1.3 半固态样品：搅拌均匀。

# 5.2 试样提取

# 5.2.1 包装饮用水

称取 $1\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )，加入 $20.0\mu \mathrm{L}$ 氯酸盐和高氯酸盐同位素内标混合标准使用液，涡旋振荡 $10\mathrm{s}$ ，液相色谱-串联质谱仪测定。

# 5.2.2 其他样品

固体样品或调味品称取 $1\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），液体或半固体样品（调味品除外）称取 $2\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )，置于 $50~\mathrm{mL}$ 离心管中，加入 $40.0\mu \mathrm{L}$ 氯酸盐和高氯酸盐同位素内标混合标准中间液，振荡混合后静置 $30\mathrm{min}$ 。加入 $0.1\%$ 甲酸水溶液 $4.0~\mathrm{mL}$ ，具塞后振摇混匀，涡旋混匀 $30~\mathrm{s}$ ，超声提取 $15\mathrm{min}$ ，加入乙腈 $6.0~\mathrm{mL}$ ，超声提取 $30\mathrm{min}$ ， $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液待净化。

# 5.3 试样净化

# 5.3.1 高盐样品（酱油、蚝油、椒盐等）

吸取 $2.0\mathrm{mL}$ 上清液过石墨化炭黑固相萃取柱，弃去流出液，再吸取 $5.0\mathrm{mL}$ 上清液过石墨化炭黑固相萃取柱，用离心管收集流出液。再用 $\mathrm{Ag / H}$ 小柱净化，弃去前 $3\mathrm{mL}$ ，收集之后的流出液过再生纤维素滤膜，液相色谱-串联质谱仪测定。

# 5.3.2 其他样品

吸取 $2.0\mathrm{mL}$ 上清液过石墨化炭黑固相萃取柱，弃去流出液，随后再吸取 $2.0\mathrm{mL}$ 上清液过石墨化炭黑固相萃取柱，流出液过再生纤维素滤膜，液相色谱-串联质谱仪测定。

# 5.4 液相色谱串联质谱参考条件

# 5.4.1 液相色谱条件如下：

a）色谱柱：填料表面带有正电荷的五氟苯基柱（柱长 $100\mathrm{mm}$ ，内径 $2.1\mathrm{mm}$ ，填料粒径 $1.7\mu \mathrm{m}$ ），或性能相当者。  
b) 流动相：A 相为乙腈，B 相为 $0.1\%$ 甲酸水溶液。梯度洗脱程序见附录 A 中表 A.1。  
c) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ 。  
d) 柱温： $35^{\circ} \mathrm{C}$ 。  
e) 进样量: $3 \mu \mathrm{L}$ 。

# 5.4.2 质谱条件如下：

a）离子化模式：电喷雾电离负离子模式（ESI-）。  
b）扫描方式：多反应监测(MRM)扫描。  
c）质谱调谐参数和定性、定量离子见附录A中表A.2。   
d) 氯酸盐和高氯酸盐的多反应监测(MRM)离子通道图参见附录B中图B.1。

# 5.5 标准曲线的制作

将氯酸盐和高氯酸盐标准系列溶液经液相色谱-串联质谱仪测定，获得相应的峰面积，以标准系列工作溶液中氯酸盐和高氯酸盐的浓度为横坐标，峰面积比为纵坐标，绘制标准曲线。

# 5.6 定性测定

被测组分选择两组离子，每组离子含一个母离子和一个子离子，在相同试验条件下，样品中待测物质的保留时间与标准溶液中对应的保留时间偏差在 $2.5\%$ 之内；所监测的离子对的信噪比均需大于或等于 $3(S / N \geqslant 3)$ 。且样品中被测组分定性离子的相对丰度与浓度接近的标准溶液中对应的定性离子的相对丰度进行比较，偏差不超过表1规定的范围，则可判定为样品中存在对应的待测物。

# 5.7 定量测定

将处理得到的试样溶液经液相色谱-串联质谱仪分析，得到氯酸盐和高氯酸盐峰面积与对应的同位素内标物峰面积比值，根据标准曲线得到待测液中氯酸盐和高氯酸盐的含量。

标准工作溶液和样液中待测物的响应值均应在标准曲线的线性响应范围内，如果含量超过范围，则重新取样分析，增加内标添加量进行处理，并用水稀释到内标浓度和待测液浓度相匹配后分析。

# 5.8 空白试验

除不称取试样外，均按照上述测定步骤进行。

如本底值高于0.5倍定量限，应对有机溶剂进行重蒸，或更换试剂和试验器皿，直至本底值低于0.5倍定量限。

# 6 分析结果的表述

试样中氯酸盐或高氯酸盐的含量按式(1)计算。

$$
X = \frac {\left(c - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中待测组分的含量，以氯酸根或高氯酸根计，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c$ ——由标准曲线计算得到的试样溶液中待测组分的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{0}$ ——由标准曲线计算得到的空白试样溶液中待测组分的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样提取中加入提取液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——单位换算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果保留3位有效数字。

# 7 精密度

在重复条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

包装饮用水中氯酸盐的检出限为 $2.00\mu \mathrm{g / kg}$ ，定量限为 $5.00~\mu \mathrm{g / kg}$ ；高氯酸盐的检出限为 $0.200\mu \mathrm{g / kg}$ ，定量限为 $0.500\mu \mathrm{g / kg}$ 。

其他样品称样量为 $1\mathrm{g}$ 时，试样中氯酸盐的检出限为 $20.0~\mu \mathrm{g / kg}$ ，定量限为 $50.0~\mu \mathrm{g / kg}$ ；高氯酸盐的检出限为 $2.00~\mu \mathrm{g / kg}$ ，定量限为 $5.00~\mu \mathrm{g / kg}$ 。

其他样品称样量为 $2\mathrm{g}$ 时，试样中氯酸盐的检出限为 $10.0~\mu \mathrm{g / kg}$ ，定量限为 $25.0~\mu \mathrm{g / kg}$ ；高氯酸盐的检出限为 $1.00~\mu \mathrm{g / kg}$ ，定量限为 $2.50~\mu \mathrm{g / kg}$ 。

# 1 范围

本标准规定了水产品中创伤弧菌(Vibrio vulnificus)的检验方法。

本标准适用于鱼、虾、蟹、贝类等水产品中创伤弧菌的检验。

# 2 设备和材料

除微生物实验室常规灭菌及培养设备外，其他设备和材料如下：

# 5 操作步骤

# 5.1 样品制备

5.1.1 新鲜样品采集后应于 $3\mathrm{h}$ 内完成检验，若不能在规定时间内完成，则将样品置于 $7^{\circ}\mathrm{C} \sim 10^{\circ}\mathrm{C}$ 条件下保存（因创伤弧菌在 $4^{\circ}\mathrm{C}$ 条件下极易形成活而不可培养的状态，因此样品勿放 $4^{\circ}\mathrm{C}$ 条件下），并尽可能在 $24\mathrm{h}$ 内完成检验；冷冻样品应在不超过 $45^{\circ}\mathrm{C}$ 的温热条件下解冻，解冻时间不超过 $15\mathrm{min}$ 。  
5.1.2 取样: 鱼类和头足类取其表面组织、肠和鳃。贝类则取全部内容物 (包括贝肉和体液)。甲壳类取整个动物或其中心部分 (包括肠和鳃)。如为带壳贝类或硬壳甲壳类, 则应先用流动自来水冲洗外壳并用滤纸吸干表面水分, 然后无菌操作打开外壳, 按上述要求取相应部分。  
5.1.3 以无菌操作取上述处理后的样品 $25\mathrm{g}$ ，加入PNCC增菌液 $225~\mathrm{mL}$ ，用旋转刀片式均质器以 $8000\mathrm{r / min}$ 均质 $1\mathrm{min}$ ，或用拍击式均质器均质 $2\mathrm{min}$ ，充分混匀制备成 $1:10$ 的样品匀液。如无均质器，则将样品放入无菌乳钵中充分研磨，取磨碎后的样品 $25\mathrm{g}$ 转入 $500~\mathrm{mL}$ 无菌锥形瓶中，加入PNCC增菌液 $225~\mathrm{mL}$ ，充分振荡混匀，制备成 $1:10$ 的样品匀液。

# 5.2 增菌

将5.1.3制备的 $1:10$ 样品匀液于 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $18\mathrm{h} \pm 1\mathrm{h}$ 。

# 5.3 PCR检测

PCR实验环境条件和过程控制应参照GB/T27403《实验室质量控制规范 食品分子生物学检测》规定执行，下同。

# 5.3.1 DNA模板的制备

在距离PNCC增菌液的液面下 $1\mathrm{cm}$ 处吸取 $1\mathrm{mL}$ 置于 $1.5\mathrm{mL}$ EP管中， $9000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，弃去上清液。向沉淀中加入 $1\mathrm{mL}$ PBS将其悬浮并充分清洗后， $9000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，弃去上清液，按此步骤反复清洗沉淀2次～3次，弃去最后一遍上清液，加入 $1\mathrm{mL}$ 无菌超纯水， $100^{\circ}\mathrm{C}$ 煮沸 $10\mathrm{min}$ ，继而 $12000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液用于PCR分析。若不能及时分析则于 $-20^{\circ}\mathrm{C}$ 保存备用。

也可以用商品化的DNA提取试剂盒按其说明书要求提取制备DNA模板。

# 5.3.2.4 PCR反应程序

预变性： $94^{\circ}\mathrm{C}, 5\mathrm{min}$ ；变性： $94^{\circ}\mathrm{C}, 1\mathrm{min}$ ；退火： $62^{\circ}\mathrm{C}, 1\mathrm{min}$ ；延伸： $72^{\circ}\mathrm{C}, 1\mathrm{min}$ ；循环数：30；终延伸： $72^{\circ}\mathrm{C}, 10\mathrm{min}$ ；电泳检测。若不能立即检测则 $4^{\circ}\mathrm{C}$ 保存备用。

# 5.3.3 电泳

凝胶电泳检测PCR扩增产物。用 $0.5\times \mathrm{TBE}$ 缓冲液配制 $1.5\%$ 的琼脂糖凝胶（含 $\mathrm{EB}0.5\mu \mathrm{g / mL}$ 或Goldview $5\mu \mathrm{L} / 100~\mathrm{mL}$ 或Gelred $5\mu \mathrm{L} / 50~\mathrm{mL}$ 等DNA染料)，取 $5\mu \mathrm{L}$ PCR扩增产物与 $1\mu \mathrm{L}6\times$ 核酸电泳上样缓冲液混合后，点样，同时有一孔加入DNA分子量标准(范围为 $100\mathrm{bp}\sim 1000\mathrm{bp})$ 。根据以下公式：电泳槽正负极的距离 $(\mathrm{cm})\times 5\mathrm{V / cm}$ 计算并设置电压，使用 $0.5\times \mathrm{TBE}$ 缓冲液恒压电泳，根据溴酚蓝的移动位置确定电泳时间，使用凝胶成像系统或紫外检测仪观察和记录结果。

# 5.3.4 结果判定

质控系统：阴性对照和空白对照均未出现扩增条带，阳性对照出现预期大小(519 bp)的扩增条带，则检测系统正常。否则，任一种对照如果出现非上述正常结果，应重做实验，同时排除污染因素。

阳性结果：在质控系统正常的情况下，待测样品出现预期大小(519 bp)的扩增条带，判定PCR结果为阳性。

阴性结果：在质控系统正常的情况下，待测样品未出现预期大小(519 bp)的扩增条带，判定PCR结果为阴性。

# 5.4 分离

用 $10\mu \mathrm{L}$ 接种环在距离PNCC增菌液的液面下 $1\mathrm{cm}$ 沾取一环增菌液，分别划线接种于CC和mCPC平板，于 $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 条件下培养 $18\mathrm{h}\pm 1\mathrm{h}$ 。典型的创伤弧菌在CC和mCPC平板上的形态为：圆形、扁平，光照下呈透明或中心不透明但边缘透明的黄色至橘黄色菌落，菌落直径 $1\mathrm{mm}\sim 2\mathrm{mm}$ ，菌落周围可出现(或不出现)黄色晕圈。

# 5.5 分纯培养

从CC平板和mCPC平板上各挑取至少5个创伤弧菌可疑菌落（少于5个时全选），分别接种于 $3\%$ 氯化钠胰蛋白胨大豆琼脂平板， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $18\mathrm{h} \pm 1\mathrm{h}$ 后用于后续鉴定。创伤弧菌在 $3\%$ 氯化钠胰蛋白胨大豆琼脂平板上的菌落形态为圆形、乳白色、湿润、隆起、直径 $1\mathrm{mm} \sim 2\mathrm{mm}$ 。

# 5.6 鉴定

创伤弧菌可采用菌落特征结合生化特性或PCR方法进行鉴定。

# 5.6.1 菌落特征及生化特性

# 5.6.1.1 初步鉴定

该步骤用于创伤弧菌的初步鉴定，进行以下4项鉴定实验时，挑取的菌落应来自分纯后的同一个单菌落。

a）革兰氏染色镜检：从5.5中 $3\%$ 氯化钠胰蛋白胨大豆琼脂平板上挑取创伤弧菌疑似菌落进行革兰氏染色并镜检。创伤弧菌为革兰氏阴性，显微镜下菌体为棒状、弧状、卵圆状等多种形态，无芽胞。  
b）氧化酶试验：用接种环从5.5中 $3\%$ 氯化钠胰蛋白胨大豆琼脂平板上挑取创伤弧菌疑似菌落适量，涂布在用氧化酶试剂润湿（如无菌滤纸)或滴有氧化酶试剂的白色或无色载体上（如载玻片）。如果涂菌部位在 $10\mathrm{~s}$ 之内变紫色(偶有蓝紫色)，即为氧化酶试验阳性，不变色为氧化酶试验阴性。创伤弧菌为氧化酶试验阳性。  
c）三糖铁试验：用接种针从5.5中 $3\%$ 氯化钠胰蛋白胨大豆琼脂平板上挑取创伤弧菌疑似菌落适量，转种于 $3\%$ 氯化钠三糖铁琼脂斜面并穿刺底层（注意接种针不要触及试管底部）， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h}$ 观察结果。创伤弧菌在 $3\%$ 氯化钠三糖铁琼脂斜面中生长时试管底层变黄，无气泡，斜面颜色不变黄（偶有斜面颜色变黄现象）。  
d）嗜盐性试验：用接种针从5.5中 $3\%$ 氯化钠胰蛋白胨大豆琼脂平板上挑取创伤弧菌疑似菌落，分别接种于含 $0\% ,3\% ,6\% ,8\%$ 和 $10\%$ 氯化钠的胰蛋白胨水中， $36^{\circ}C\pm 1^{\circ}C$ 培养 $24\mathrm{h}$ ，观察液体的混浊情况。创伤弧菌在含 $0\% ,8\%$ 和 $10\%$ 氯化钠的胰蛋白胨水中不生长或微弱生长，胰蛋白胨水澄清透亮或稍混浊，而在含 $3 \%$ 和 $6 \%$ 氯化钠的胰蛋白胨水中生长旺盛，胰蛋白胨水混浊。

# 5.6.1.2 确证实验

将初步鉴定为创伤弧菌的疑似菌落接种在 $3\%$ 氯化钠胰蛋白胨大豆琼脂平板上， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $18\mathrm{h} \pm 1\mathrm{h}$ 后，取纯培养物分别接种于含 $3\%$ 氯化钠的赖氨酸脱羧酶试验培养基和MR-VP培养基中， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h} \sim 48\mathrm{h}$ 后观察结果；另取少许纯培养物接种于 $3\%$ 氯化钠三糖铁琼脂斜面， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $18\mathrm{h}$ 后用于ONPG试验。也可选择生化鉴定试剂盒进行鉴定。

如果待测菌株出现 $665\mathrm{bp}$ 大小的条带，则判定该株创伤弧菌为血清E型；如果待测菌株出现 $344\mathrm{bp}$ 大小的条带，则判定该株创伤弧菌为生物Ⅱ型。

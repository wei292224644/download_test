# 食品安全国家标准

# 食品中磷的测定

# 1 范围

本标准规定了食品中磷含量测定的分光光度法和电感耦合等离子体发射光谱法。

本标准第一法、第三法适用于各类食品中磷的测定，第二法适用于婴幼儿食品和乳品中磷的测定。

# 第一法 钼蓝分光光度法

# 2 原理

试样经消解，磷在酸性条件下与钼酸铵结合生成磷钼酸铵，此化合物被对苯二酚、亚硫酸钠或氯化亚锡、硫酸肼还原成蓝色化合物钼蓝。钼蓝在 $660~\mathrm{nm}$ 处的吸光度值与磷的浓度成正比。用分光光度计测定试样溶液的吸光度，与标准系列比较定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 硫酸 $\left(\mathrm{H}_{2} \mathrm{SO}_{4}\right)$ ：优级纯。  
3.1.2 高氯酸 $\mathrm{(HClO_4)}$ ：优级纯。  
3.1.3 硝酸 $\mathrm{(HNO_3)}$ ：优级纯。  
3.1.4 盐酸（HCl）：优级纯。  
3.1.5 对苯二酚 $\mathrm{(C_6H_6O_2)}$ 。  
3.1.6 无水亚硫酸钠 $\mathrm{(Na_2SO_3)}$   
3.1.7 钼酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{6} \mathrm{Mo}_{7} \mathrm{O}_{24} \cdot 4 \mathrm{H}_{2} \mathrm{O}\right]$ 。  
3.1.8 氯化亚锡 $(\mathrm{SnCl}_2\cdot 2\mathrm{H}_2\mathrm{O})$ 。  
3.1.9 硫酸肼 $(\mathrm{NH}_2\mathrm{NH}_2\bullet \mathrm{H}_2\mathrm{SO}_4)$

# 3.2 试剂的配制

3.2.1 硫酸溶液 $(15\%)$ ：量取 $15~\mathrm{mL}$ 硫酸，缓慢加入到 $80~\mathrm{mL}$ 水中，冷却后用水稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.2 硫酸溶液 $(5\%)$ ：量取 $5\mathrm{mL}$ 硫酸，缓慢加入到 $90~\mathrm{mL}$ 水中，冷却后用水稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.3 硫酸溶液 $(3\%)$ ：量取 $3\mathrm{mL}$ 硫酸，缓慢加入到 $90~\mathrm{mL}$ 水中，冷却后用水稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.4 盐酸溶液 $(1 + 1)$ ：量取 $500~\mathrm{mL}$ 盐酸，加入 $500~\mathrm{mL}$ 水，混匀。  
3.2.5钼酸铵溶液 $(50~\mathrm{g / L})$ ：称取 $5\mathrm{g}$ 钼酸铵，加硫酸溶液 $(15\%)$ 溶解，并稀释至 $100~\mathrm{mL}$ ，混匀。  
3.2.6 对苯二酚溶液 $(5\mathrm{g} / \mathrm{L})$ ：称取 $0.5\mathrm{g}$ 对苯二酚于 $100~\mathrm{mL}$ 水中，使其溶解，并加入一滴硫酸，混匀。  
3.2.7 亚硫酸钠溶液 $(200\mathrm{g / L})$ ：称取 $20\mathrm{g}$ 无水亚硫酸钠溶解于 $100~\mathrm{mL}$ 水中，混匀。临用时配制。

3.2.8 氯化亚锡-硫酸肼溶液：称取 $0.1\mathrm{g}$ 氯化亚锡， $0.2\mathrm{g}$ 硫酸肼，加硫酸溶液 $(3\%)$ 并用其稀释至 $100~\mathrm{mL}$ 。此溶液置棕色瓶中，贮于 $4^{\circ}\mathrm{C}$ 可保存1个月。

# 3.3 标准品

磷酸二氢钾 $\mathrm{(KH_2PO_4}$ ,CAS号7778-77-0):纯度 $>99.99\%$ 。或经国家认证并授予标准物质证书的一定浓度的磷标准溶液。

# 3.4 标准溶液的制备

3.4.1 磷标准储备液 $(100.0\mathrm{mg / L})$ : 准确称取在 $105^{\circ}\mathrm{C}$ 下干燥至恒重的磷酸二氢钾 $0.4394\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ) 置于烧杯中, 加入适量水溶解并转移至 $1000\mathrm{mL}$ 容量瓶中, 加水定容至刻度, 混匀。  
3.4.2 磷标准使用液 $(10.0\mathrm{mg / L})$ ：准确吸取 $10~\mathrm{mL}$ 磷标准储备液 $(100.0\mathrm{mg / L})$ ，置于 $100~\mathrm{mL}$ 容量瓶中，加水稀释至刻度，混匀。

# 4 仪器和设备

4.1 分光光度计。  
4.2 可调式电热板或可调式电热炉。  
4.3 马弗炉。  
4.4 分析天平：感量 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$

# 5 分析步骤

# 5.1 试样制备

在采样和试样制备过程中，应避免污染。

# 5.1.1 粮食、豆类

样品去除杂物后，粉碎，储于塑料瓶中。

# 5.1.2 蔬菜、水果、鱼类、肉类等样品

样品用水洗净，晾干，取可食部分，制成匀浆，储于塑料瓶中。

# 5.1.3 饮料、酒、醋、酱油、食用植物油、液态乳等液体样品

将样品摇匀。

# 5.2 试样前处理

# 5.2.1 湿法消解

称取试样 $0.2\mathrm{g}\sim 3\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确吸取液体试样 $0.500~\mathrm{mL}\sim 5.00~\mathrm{mL}$ 于带刻度消化管中，加入 $10~\mathrm{mL}$ 硝酸， $1\mathrm{mL}$ 高氯酸， $2\mathrm{mL}$ 硫酸，在可调式电热炉上消解（参考条件： $120^{\circ}C / 0.5\mathrm{h}\sim 1\mathrm{h}$ 升至 $180^{\circ}C / 2\mathrm{h}\sim 4\mathrm{h}$ 升至 $200^{\circ}C\sim 220^{\circ}C)$ 。若消化液呈棕褐色，再加硝酸，消解至冒白烟，消化液呈无色透明或略带黄色。消化液放冷，加 $20~\mathrm{mL}$ 水，赶酸。放冷后转移至 $100~\mathrm{mL}$ 容量瓶中，用水多次洗涤消化管，合并洗液于容量瓶中，加水至刻度，混匀。作为试样测定溶液。同时做试剂空白试验。亦可采用锥形瓶，于可调式电热板上，按上述操作方法进行湿法消解。

# 5.2.2 干法灰化

称取试样 $0.5\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液体试样 $0.500~\mathrm{mL}\sim 10.0~\mathrm{mL}$ ，在火上灼烧成炭分，再于 $550~\mathrm{^\circ C}$ 下成灰分，直至灰分呈白色为止(必要时，可在加入浓硝酸润湿蒸干后再灰化)，加 $10~\mathrm{mL}$ 盐酸溶液 $(1 + 1)$ ，在水浴上蒸干。再加 $2\mathrm{mL}$ 盐酸溶液 $(1 + 1)$ ，用水分数次将残渣完全洗入 $100~\mathrm{mL}$ 容量瓶中，并用水稀释至刻度，摇匀。同时做试剂空白试验。

# 5.3 测定

注：可任选苯二酚、亚硫酸钠还原法或氯化亚锡、硫酸肼还原法。

# 5.3.1 对苯二酚、亚硫酸钠还原法

# 5.3.1.1 标准曲线的制作

准确吸取磷标准使用液 $0\mathrm{mL}, 0.500\mathrm{mL}, 1.00\mathrm{mL}, 2.00\mathrm{mL}, 3.00\mathrm{mL}, 4.00\mathrm{mL}, 5.00\mathrm{mL}$ ，相当于含磷量 $0\mu \mathrm{g}, 5.00\mu \mathrm{g}, 10.0\mu \mathrm{g}, 20.0\mu \mathrm{g}, 30.0\mu \mathrm{g}, 40.0\mu \mathrm{g}, 50.0\mu \mathrm{g}$ ，分别置于 $25\mathrm{mL}$ 具塞试管中，依次加入 $2\mathrm{mL}$ 钼酸铵溶液（ $50\mathrm{g/L}$ ）摇匀，静置。加入 $1\mathrm{mL}$ 亚硫酸钠溶液（ $200\mathrm{g/L}$ ）、 $1\mathrm{mL}$ 对苯二酚溶液（ $5\mathrm{g/L}$ ），摇匀。加水至刻度，混匀。静置 $0.5\mathrm{h}$ 后，用 $1\mathrm{cm}$ 比色杯，在 $660\mathrm{nm}$ 波长处，以零管作参比，测定吸光度，以测出的吸光度对磷含量绘制标准曲线。

# 5.3.1.2 试样溶液的测定

准确吸取试样溶液 $2.00\mathrm{mL}$ 及等量的空白溶液，分别置于 $25~\mathrm{mL}$ 具塞试管中，加入 $2\mathrm{mL}$ 钼酸铵溶液 $(50\mathrm{g / L})$ 摇匀，静置。加入 $1\mathrm{mL}$ 亚硫酸钠溶液 $(200\mathrm{g / L})$ 、 $1\mathrm{mL}$ 对苯二酚溶液 $(5\mathrm{g / L})$ ，摇匀。加水至刻度，混匀。静置 $0.5\mathrm{h}$ 后，用 $1\mathrm{cm}$ 比色杯，在 $660~\mathrm{nm}$ 波长处，测定其吸光度，与标准系列比较定量。

# 5.3.2 氯化亚锡、硫酸肼还原法

# 5.3.2.1 标准曲线的制作

准确吸取磷标准使用液 $0\mathrm{mL}, 0.500\mathrm{mL}, 1.00\mathrm{mL}, 2.00\mathrm{mL}, 3.00\mathrm{mL}, 4.00\mathrm{mL}, 5.00\mathrm{mL}$ ，相当于含磷量 $0\mu \mathrm{g}, 5.00\mu \mathrm{g}, 10.0\mu \mathrm{g}, 20.0\mu \mathrm{g}, 30.0\mu \mathrm{g}, 40.0\mu \mathrm{g}, 50.0\mu \mathrm{g}$ ，分别置于 $25\mathrm{mL}$ 具塞试管中，各加约 $15\mathrm{mL}$ 水， $2.5\mathrm{mL}$ 硫酸溶液（ $5\%$ ）， $2\mathrm{mL}$ 钼酸铵溶液（ $50\mathrm{g / L}$ ）， $0.5\mathrm{mL}$ 氯化亚锡-硫酸肼溶液，各管均补加水至 $25\mathrm{mL}$ ，混匀。在室温放置 $20\mathrm{min}$ 后，用 $1\mathrm{cm}$ 比色杯，在 $660\mathrm{nm}$ 波长处，以零管作参比，测定其吸光度，以吸光度对磷含量绘制标准曲线。

# 5.3.2.2 试样溶液的测定

准确吸取试样溶液 $2.00\mathrm{mL}$ 及等量的空白溶液，分别置于 $25~\mathrm{mL}$ 比色管中，各加约 $15~\mathrm{mL}$ 水， $2.5\mathrm{mL}$ 硫酸溶液 $(5\%)$ ， $2\mathrm{mL}$ 钼酸铵溶液 $(50~\mathrm{g / L})$ ， $0.5\mathrm{mL}$ 氯化亚锡-硫酸肼溶液。各管均补加水至 $25~\mathrm{mL}$ ，混匀。在室温放置 $20\mathrm{min}$ 后，用 $1\mathrm{cm}$ 比色杯，在 $660~\mathrm{nm}$ 波长处，分别测定其吸光度，与标准系列比较定量。

# 6 分析结果的表述

试样中磷的含量按式(1)计算：

$$
X = \frac {\left(m _ {1} - m _ {0}\right) \times V _ {1}}{m \times V _ {2}} \times \frac {1 0 0}{1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中磷含量，单位为毫克每百克或毫克每百毫升 $(\mathrm{mg} / 100\mathrm{g}$ 或 $\mathrm{mg} / 100\mathrm{mL})$

$m_{1}$ ——测定用试样溶液中磷的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_0$ ——测定用空白溶液中磷的质量，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——试样消化液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样称样量或移取体积，单位为克(g或 $\mathrm{mL}$ )；

$V_{2}$ ——测定用试样消化液的体积，单位为毫升 $(\mathrm{mL})$ ；

100 ——换算系数；

1000——换算系数。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 8 其他

当取样量 $0.5\mathrm{g}$ (或 $0.5\mathrm{mL}$ )，定容至 $100\mathrm{mL}$ 时，检出限为 $20\mathrm{mg} / 100\mathrm{g}$ （或 $20\mathrm{mg} / 100\mathrm{mL}$ ），定量限为 $60\mathrm{mg} / 100\mathrm{g}$ （或 $60\mathrm{mg} / 100\mathrm{mL}$ ）。

# 第二法 钠钼黄分光光度法

# 9 原理

试样经消解，磷在酸性条件下与钒钼酸铵生成黄色络合物钒钼黄。钒钼黄的吸光度值与磷的浓度成正比。于 $440\mathrm{nm}$ 测定试样溶液中钒钼黄的吸光度值，与标准系列比较定量。

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的三级水。

# 10.1 试剂

10.1.1 高氯酸 $\left(\mathrm{HClO}_4\right)$ ：优级纯。  
10.1.2 硝酸 $\mathrm{(HNO_3)}$ ：优级纯。  
10.1.3 硫酸 $\left(\mathrm{H}_{2} \mathrm{SO}_{4}\right)$ ：优级纯。  
10.1.4 钼酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{6} \mathrm{Mo}_{7} \mathrm{O}_{24} \cdot 4 \mathrm{H}_{2} \mathrm{O}\right]$ 。  
10.1.5 偏钒酸铵 $(\mathrm{NH_4VO_3})$ 。  
10.1.6 氢氧化钠(NaOH)。  
10.1.7 2,6-二硝基酚或2，4-二硝基酚 $\left[\mathrm{C}_{6} \mathrm{H}_{3} \mathrm{OH}\left(\mathrm{NO}_{2}\right)_{2}\right]$ 。

# 10.2 试剂的配制

10.2.1 钒钼酸铵试剂：

A液：称取 $25\mathrm{g}$ 钼酸铵，溶于 $400~\mathrm{mL}$ 水中。

B液：称取 $1.25\mathrm{g}$ 偏钒酸铵溶于 $300~\mathrm{mL}$ 沸水中，冷却后加 $250~\mathrm{mL}$ 硝酸。将A液缓慢加至B液中，不断搅匀，并用水稀释至1L，混匀，贮于棕色瓶中。  
10.2.2 氢氧化钠溶液 $(6\mathrm{mol} / \mathrm{L})$ ：称取 $240\mathrm{g}$ 氢氧化钠，溶于 $1000~\mathrm{mL}$ 水中，混匀。  
10.2.3 氢氧化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $4\mathrm{g}$ 氢氧化钠，溶于 $1000~\mathrm{mL}$ 水中，混匀。  
10.2.4 硝酸溶液 $(0.2\mathrm{mol} / \mathrm{L})$ ：吸取 $12.5~\mathrm{mL}$ 硝酸，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
10.2.5 二硝基酚指示剂 $(2\mathrm{g} / \mathrm{L})$ ：称取 $0.2\mathrm{g}2,6-$ 二硝基酚或2，4-二硝基酚溶于 $100~\mathrm{mL}$ 水中，混匀。

# 10.3 标准品

磷酸二氢钾 $\mathrm{(KH_2PO_4}$ ,CAS号7778-77-0):纯度 $>99.99\%$ 。或经国家认证并授予标准物质证书的一定浓度的磷标准溶液。

# 10.4 标准溶液的制备

磷标准储备液 $(50.00\mathrm{mg / L})$ ：精确称取在 $105~^\circ \mathrm{C}$ 下干燥至恒量的磷酸二氢钾 $0.2197\mathrm{g}$ （精确至 $0.0001\mathrm{g})$ ，溶于 $400~\mathrm{mL}$ 水中，移入 $1\textrm{L}$ 容量瓶，并加水至刻度，混匀。置聚乙烯瓶贮于 $4^{\circ}C$ 保存。

# 12.1 试样制备

同5.1。

# 12.3 标准曲线的制作

准确吸取磷标准储备液 $0\mathrm{mL},2.50\mathrm{mL},5.00\mathrm{mL},7.50\mathrm{mL},10.0\mathrm{mL},15.0\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，加入 $10~\mathrm{mL}$ 钒钼酸铵试剂，用水定容至刻度。该系列标准溶液中磷的质量浓度分别为 $0\mathrm{mg / L}$ 、 $2.50\mathrm{mg / L}$ 、 $5.00\mathrm{mg / L}$ 、 $7.50\mathrm{mg / L}$ 、 $10.0\mathrm{mg / L}$ 、 $15.0\mathrm{mg / L}$ 。在 $25^{\circ}C\sim 30^{\circ}C$ 下显色 $15\mathrm{min}$ 。用 $1\mathrm{cm}$ 比色杯，以零管作参比，于 $440~\mathrm{nm}$ 测定吸光度值。以吸光度值为纵坐标，磷的质量浓度为横坐标，制作标准曲线。

# 12.4 试样溶液的测定

准确吸取试样溶液 $10\mathrm{mL}$ 及等量的空白溶液于 $50~\mathrm{mL}$ 容量瓶中，加少量水后，加2滴二硝基酚指示剂 $(2\mathrm{g} / \mathrm{L})$ ，先用氢氧化钠溶液 $(6\mathrm{mol} / \mathrm{L})$ 调至黄色，再用硝酸溶液 $(0.2\mathrm{mol} / \mathrm{L})$ 调至无色，最后用氢氧化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ 调至微黄色。加入 $10\mathrm{mL}$ 钠钼酸铵试剂，用水定容至刻度。于 $440~\mathrm{nm}$ 测定其吸光度值，与标准系列比较定量。

# 13 分析结果的表述

试样中磷的含量按式(2)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times V _ {2}}{m \times V _ {1} \times 1 0 0 0} \times 1 0 0 \tag {2}
$$

式中：

$X$ ——试样中磷的含量，单位为毫克每百克或毫克每百毫升 $(\mathrm{mg} / 100\mathrm{g}$ 或 $\mathrm{mg} / 100\mathrm{mL})$

$\rho$ ——测定用试样溶液中磷的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——测定用空白溶液中磷的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样消化液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试样比色液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样称样量或移取体积，单位为克(g或 $\mathrm{mL}$ ）；

$V_{1}$ ——测定用试样消化液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

100 ——换算系数。

计算结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 15 其他

当取样量 $0.5\mathrm{g}$ (或 $0.5\mathrm{mL}$ )，定容至 $100\mathrm{mL}$ 时，检出限为 $20\mathrm{mg} / 100\mathrm{g}$ （或 $20\mathrm{mg} / 100\mathrm{mL}$ ），定量限为 $60\mathrm{mg} / 100\mathrm{g}$ （或 $60\mathrm{mg} / 100\mathrm{mL}$ ）。

# 前言

本标准代替GB5413.31—2013《食品安全国家标准婴幼儿食品和乳品中脲酶的测定》、GB/T5009.183—2003《植物蛋白饮料中脲酶的定性测定》、GB/T5009.186—2003《乳酸菌饮料中脲酶的定性测定》、GB/T30885—2014《植物蛋白饮料豆奶和豆奶饮料》附录A和GB20371—2016《食品安全国家标准 食品加工用植物蛋白》附录A检测标准。

与GB5413.31—2013相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中脲酶的测定”；  
——修改标准适用范围；  
——修改纳氏试剂显色法为第一法；  
——增加了滴定法为标准第二法。

# 食品安全国家标准

# 食品中脲酶的测定

# 1 范围

本标准规定了食品中脲酶活性的测定方法。

第一法纳氏试剂显色法适用于含大豆成分的饮料、婴幼儿配方食品、婴幼儿辅助食品、乳制品、特殊医学用途配方食品、大豆蛋白肽及豆制品中脲酶活性的定性测定。

第二法滴定法适用于大豆蛋白、豆粉与豆奶粉中脲酶活性的定量测定。

# 第一法 纳氏试剂显色法

# 2 原理

脲酶在 $\mathrm{pH}7.0,40^{\circ}\mathrm{C}$ 条件下，催化尿素转化成碳酸铵，碳酸铵在碱性条件下生成氢氧化铵，再与纳氏试剂中的碘化钾汞复盐作用生成棕色的碘化双汞铵，根据显色情况定性判断样品中脲酶活性。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 尿素 $\mathrm{(H_2NCONH_2)}$ 。  
3.1.2 二水合钨酸钠 $(\mathrm{Na}_2\mathrm{WO}_4\cdot 2\mathrm{H}_2\mathrm{O})$ 。  
3.1.3 四水合酒石酸钾钠 $\mathrm{(C_4H_4O_6KNa\cdot 4H_2O)}$   
3.1.4 硫酸 $\mathrm{(H_2SO_4)}$ 。  
3.1.5 磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4)$ 。  
3.1.6 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$ 。  
3.1.7 红色碘化汞（ $\mathrm{HgI_2}$ ）。  
3.1.8 碘化钾(KI)。  
3.1.9 氢氧化钠(NaOH)。  
3.1.10 石墨化炭黑(GCB)固相萃取柱： $300\mathrm{mg},6\mathrm{mL}$ 。

# 3.2 试剂配制

3.2.1 尿素溶液 $(10\mathrm{g / L})$ ：称取尿素 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )，用水溶解并稀释至 $500~\mathrm{mL}$ ，混匀，保存于棕色试剂瓶中，置于 $2^{\circ}\mathrm{C}\sim 8^{\circ}\mathrm{C}$ 冰箱中避光保存，有效期2周。  
3.2.2 钨酸钠溶液 $(89.0\mathrm{g / L})$ ：称取二水合钨酸钠 $50.0\mathrm{g}$ ，用水溶解并稀释至 $500~\mathrm{mL}$ ，混匀。  
3.2.3 酒石酸钾钠溶液 $(14.9\mathrm{g / L})$ ：称取四水合酒石酸钾钠 $10.0\mathrm{g}$ ，用水溶解并稀释至 $500~\mathrm{mL}$ ，混匀。

3.2.4 硫酸溶液 $(5 + 95)$ ：量取 $25~\mathrm{mL}$ 硫酸，搅拌条件下，缓缓加入 $475~\mathrm{mL}$ 水中，冷却后混匀。  
3.2.5 缓冲溶液 $(\mathrm{pH}7.0)$ ：称取磷酸氢二钠 $5.79\mathrm{g}$ ，用 $200~\mathrm{mL}$ 水溶解，另称取磷酸二氢钾 $3.53\mathrm{g}$ ，用 $200~\mathrm{mL}$ 水溶解后，移入上述溶液中，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.6 纳氏试剂：称取氢氧化钠 $14.4\mathrm{g}$ ，用 $50~\mathrm{mL}$ 水充分溶解并冷却。称取红色碘化汞 $5.5\mathrm{g}$ ，碘化钾 $4.125\mathrm{g}$ ，溶于 $25~\mathrm{mL}$ 水中，然后分次、少量缓缓移至上述氢氧化钠溶液中，加入过程中需不断摇匀。转移完毕后，加水稀释至 $100~\mathrm{mL}$ ，混匀，转入棕色试剂瓶内，静置后，取上清液使用。于 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 冰箱中保存，有效期为1个月。或使用满足要求的商品化纳氏试剂溶液。

# 4 仪器和设备

4.1 电子天平：感量分别为 $0.01\mathrm{g}$ 和 $0.001\mathrm{g}$ 。  
4.2 涡旋振荡器。  
4.3 恒温水浴锅：可控温至 $40^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 。  
4.4 具塞比色管： $25~\mathrm{mL}$

# 5 分析步骤

# 5.1 试样制备

液态样品摇匀，固态样品粉碎均匀，半固态样品搅拌均匀。

# 5.2 试样分析

液态和半固态样品：根据试样固形物含量，各称取相当于 $0.10\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）固形物/干物质的试样，分别置于甲、乙两支 $25~\mathrm{mL}$ 比色管中，加入 $1\mathrm{mL}$ 水，振摇 $30~\mathrm{s}$ ，然后分别加入 $1\mathrm{mL}$ 缓冲溶液。向甲管(样品管)加入 $1\mathrm{mL}$ 尿素溶液，再向乙管(样品空白对照管)加入 $1\mathrm{mL}$ 水。摇匀后，置于 $40^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 水浴中保温 $20\mathrm{min}$ 。从水浴中取出两管，各加入 $4\mathrm{mL}$ 水，摇匀，再加入 $1\mathrm{mL}$ 钨酸钠溶液，摇匀，加入 $1\mathrm{mL}$ 硫酸溶液，摇匀，中速定性滤纸过滤，收集滤液备用。取上述滤液 $2\mathrm{mL}$ ，分别加入至两支 $25~\mathrm{mL}$ 具塞比色管中，各加入 $15~\mathrm{mL}$ 水， $1\mathrm{mL}$ 酒石酸钾钠溶液和 $2\mathrm{mL}$ 纳氏试剂，用水定容至 $25~\mathrm{mL}$ ，摇匀。 $5\mathrm{min}$ 内观察结果。

固体样品：各称取 $0.10\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）试样，分别置于甲、乙两支 $25~\mathrm{mL}$ 比色管中，加入 $1\mathrm{mL}$ 水，振摇 $30~\mathrm{s}$ ，然后按上述相同操作进行。

注1：固形物/干物质按照附录A测定。

注2：若样品溶液有颜色，在加入硫酸溶液摇匀后，取 $6\mathrm{mL}$ 溶液直接通过石墨化碳黑固相萃取柱净化，弃去前 $2\mathrm{mL}$ 流出液，收集中间段流出液 $2\mathrm{mL}$ ，然后进行显色操作。

# 7 其他

该方法为定性法，当称样量为 $0.10\mathrm{g}$ 或相当于 $0.10\mathrm{g}$ 固形物含量时，检出限均为 $0.005\mathrm{U / g}$

# 第二法 滴定法

# 8 原理

脲酶在 $\mathrm{pH}7.0,30^{\circ}\mathrm{C}$ 条件下，催化尿素转化成碳酸铵，碳酸铵经过量盐酸中和，再用氢氧化钠标准溶液回滴，根据氢氧化钠溶液的消耗体积计算脲酶含量。

# 9 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 9.1 试剂

9.1.1 尿素 $\mathrm{(H_2NCONH_2)}$ 。  
9.1.2 氢氧化钠(NaOH)。  
9.1.3 盐酸（HCl）。  
9.1.4 磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4)$ 。  
9.1.5 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
9.1.6 甲基红 $(\mathrm{C_{15}H_{15}N_3O_2})$ 。  
9.1.7 溴甲酚绿 $(\mathrm{C}_{21}\mathrm{H}_{14}\mathrm{Br}_4\mathrm{O}_5\mathrm{S})$ 。  
9.1.8 $95\%$ 乙醇 $\mathrm{(C_2H_5OH)}$ 。

# 9.2 试剂配制

9.2.1 尿素缓冲溶液 $(\mathrm{pH}7.0 \pm 0.1)$ : 称取 $3.55 \mathrm{~g}$ 磷酸氢二钠、 $3.40 \mathrm{~g}$ 磷酸二氢钾溶于水并稀释至 $1000 \mathrm{~mL}$ 。称取 $30 \mathrm{~g}$ 尿素, 溶于上述缓冲溶液中, 混匀, 转移至棕色试剂瓶, 于 $2^{\circ} \mathrm{C} \sim 8^{\circ} \mathrm{C}$ 冰箱中避光保存, 有效期 1 个月。  
9.2.2 盐酸溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：移取 $8.3~\mathrm{mL}$ 盐酸，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
9.2.3 氢氧化钠标准溶液 $(0.05\mathrm{mol} / \mathrm{L})$ ：按GB/T601规定的方法配制 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠标准溶液，

再按GB/T601规定的方法稀释为 $0.05\mathrm{mol / L}$

9.2.4 甲基红、溴甲酚绿混合乙醇溶液：称取 $0.1\mathrm{g}$ 甲基红，用 $95\%$ 乙醇溶解并稀释至 $100~\mathrm{mL}$ ，混匀。再称取 $0.5\mathrm{g}$ 溴甲酚绿，用 $95\%$ 乙醇溶解并稀释至 $100~\mathrm{mL}$ ，混匀。两种溶液等体积混合，贮存于棕色瓶中，有效期1个月。

# 10 仪器和设备

10.1 电子天平：感量分别为 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
10.2 酸度计：精度0.02，配有磁力搅拌器和滴定装置。  
10.3 恒温水浴：可控温 $30^{\circ}\mathrm{C} \pm 0.5^{\circ}\mathrm{C}$ 。  
10.4 粉碎机：粉碎时应不产生强热。  
10.5 计时器。  
10.6 滴定管： $25~\mathrm{mL}$   
10.7 烧杯： $100~\mathrm{mL}$   
10.8 单标线吸量管： $10~\mathrm{mL}$   
10.9 温度计：量程 $0\,^{\circ}\mathrm{C} \sim 100\,^{\circ}\mathrm{C}$ 。

# 11 分析步骤

# 11.1 试样制备

用粉碎机将样品粉碎、混匀。

注：样品粉碎过程中控制温度不超过 $70^{\circ}\mathrm{C}$ 。

# 11.2 试样分析

称取约 $0.2\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ )于玻璃试管中，每个试样以相同时间间隔加入 $10.0~\mathrm{mL}$ 尿素缓冲溶液，立即盖好试管盖剧烈振摇后，将试管置于 $30^{\circ}\mathrm{C} \pm 0.5^{\circ}\mathrm{C}$ 恒温水浴中，计时保持 $30\mathrm{min} \pm 10\mathrm{s}$ 。取出试管后马上以相同的时间间隔加入 $10.0~\mathrm{mL}$ 盐酸溶液，振摇后冷却至 $20^{\circ}\mathrm{C}$ 。将试管内容物全部转入 $100~\mathrm{mL}$ 烧杯中，用 $20~\mathrm{mL}$ 水冲洗试管数次，以氢氧化钠标准溶液滴定至酸度计指示 $\mathrm{pH}4.70$ 。如果选择使用指示剂，则将试管内容物全部转入 $250~\mathrm{mL}$ 锥形瓶中，加入8滴～10滴甲基红、溴甲酚绿混合指示剂，以氢氧化钠标准溶液滴定至溶液呈蓝绿色，并保持 $30~\mathrm{s}$ 不褪色。

另取试管做空白试验，称取相同质量的试样，每个试样以相同时间间隔加入 $10.0\mathrm{mL}$ 盐酸溶液，振摇后加入 $10~\mathrm{mL}$ 尿素缓冲溶液，立即盖好试管盖剧烈振摇后，将试管置于 $30^{\circ}\mathrm{C}\pm 0.5^{\circ}\mathrm{C}$ 恒温水浴中，计时保持 $30\mathrm{min}\pm 10\mathrm{s}$ 。取出试管后将试管冷却至 $20^{\circ}\mathrm{C}$ ，然后按照相同步骤进行测定。

注：若滴定前酸度计指示 $\mathrm{pH} \geq 4.70$ 或加入指示剂后溶液显示蓝绿色，可称取 $0.05\mathrm{g}$ 试样进行试样分析。

# 12 分析结果的表述

脲酶活性以每分钟每克试样催化尿素产生氮的毫克数表示，按式(1)计算。

$$
X = \frac {1 4 \times c \times \left(V _ {0} - V\right)}{3 0 \times m} \tag {1}
$$

式中：

$X$ ——试样的脲酶活性，单位为活性单位每克 $(\mathrm{U} / \mathrm{g})$ ；

$c$ ——氢氧化钠标准滴定溶液浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{0}$ ——空白消耗氢氧化钠标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$V$ ——试样消耗氢氧化钠标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

30 ——反应时间，单位为分 $(\mathrm{min})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

14——氮的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$

计算结果保留两位有效数字。

# 13 精密度

脲酶活性 $\leqslant 0.10\mathrm{U / g}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $30\%$ ； $0.10\mathrm{U / g} < \text{脲酶活性} \leqslant 0.20\mathrm{U / g}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ ；脲酶活性 $>0.20\mathrm{U / g}$ 时，在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 14 其他

当称样量为 $0.2\mathrm{g}$ 时，本方法定量限为 $0.02\mathrm{U / g}$

# 附录A

# 固形物含量测定方法

# A.1 仪器和设备

A.1.1 恒温干燥箱。  
A.1.2 干燥器：内盛干燥剂。  
A.1.3 电子天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
A.1.4 扁型称量皿: $70 \mathrm{~mm} \times 35 \mathrm{~mm}$ 。  
A.1.5 海砂。  
A.1.6 恒温水浴锅。

# A.2 分析步骤

准确称取 $10.0\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )制备的均匀试样于已恒重的盛有适量海砂的称量皿中，在水浴上蒸发至干，取下称量皿，擦干附着的水分，放入恒温干燥箱内，在 $100^{\circ}\mathrm{C}\sim 105^{\circ}\mathrm{C}$ 下烘干 $2\mathrm{h}$ ，取出移入干燥器内冷却， $30\mathrm{min}$ 后称量。

# A.3 分析结果的表述

试样中固形物含量按式(A.1)计算：

$$
X _ {1} = \frac {m _ {2} - m _ {1}}{m} \times 1 0 0 \quad \dots \tag {A.1}
$$

式中：

$X_{1}$ ——试样中固形物的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

$m_{2}$ ——烘干后试样加海砂和称量皿的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——海砂和称量皿的质量，单位为克 $(\mathrm{g})$ ；

$m$ —称取试样质量，单位为克 $(\mathrm{g})$ 。

# A.4 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 食品安全国家标准

# 水产品中有机磷类药物残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了水产品中辛硫磷、巴胺磷、倍硫磷、马拉硫磷、二嗪农、敌百虫、敌敌畏、甲基吡啶磷、蝇毒磷残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鱼、海参、蟹和虾等水产品可食部分中辛硫磷、巴胺磷、倍硫磷、马拉硫磷、二嗪农、敌百虫、敌敌畏、甲基吡啶磷和蝇毒磷残留量的检测。

# 2 规范性引用文件

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中有机磷类药物的残留经酸化乙腈提取，经PSA吸附剂净化，液相色谱-串联质谱法测定，内标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.2 乙酸 $\mathrm{(CH_3COOH)}$ ：优级纯。

5.1.3 甲酸（HCOOH）：色谱纯。

5.1.4 无水乙酸钠 $\mathrm{(CH_3COONa)}$ ，优级纯。

# 5.2 溶液配制

5.2.1 定容液：取乙腈 $150~\mathrm{mL}$ ，加水 $350~\mathrm{mL}$ 、甲酸 $0.5~\mathrm{mL}$ ，混匀。

5.2.2 $1\%$ 乙酸乙腈：取乙腈 $990~\mathrm{mL}$ ，加乙酸 $10~\mathrm{mL}$ ，混匀。

# 5.3 标准品

5.3.1 辛硫磷、巴胺磷、倍硫磷、马拉硫磷、二嗪农、敌百虫、敌敌畏、甲基吡啶磷和蝇毒磷：含量均 $\geqslant 98.0\%$ ，具体见附录A。

5.3.2 倍硫磷 $\mathrm{D}_6$ 、马拉硫磷 $\mathrm{D}_{10}$ 、二嗪农 $\mathrm{D}_{10}$ 、敌敌畏 $\mathrm{D}_6$ 、甲基吡啶磷 $\mathrm{D}_6$ 和蝇毒磷 $\mathrm{D}_{10}$ ：浓度均为 $100\mathrm{mg / L}$ ，具体见附录A。

# 5.4 标准溶液的制备

5.4.1 标准储备液：在避光条件下，取有机磷类药物标准各适量（相当于各待测组分 $10\mathrm{mg}$ ），精密称定，分别于 $10\mathrm{mL}$ 棕色量瓶中，加乙腈(5.1.1)适量使溶解并稀释至刻度。分别配成浓度为 $1\mathrm{mg/mL}$ 的标准储备溶液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期6个月。

5.4.2 混合标准中间液：分别量取标准储备液(5.4.1)适量，用乙腈(5.1.1)稀释，配成浓度分别为：巴胺磷 $10\mathrm{mg / L}$ 、马拉硫磷 $10\mathrm{mg / L}$ 、二嗪农 $10\mathrm{mg / L}$ 、敌百虫 $10\mathrm{mg / L}$ 、敌敌畏 $10\mathrm{mg / L}$ 、甲基吡啶磷 $10\mathrm{mg / L}$ ；辛硫磷 $20\mathrm{mg / L}$ 、倍硫磷 $20\mathrm{mg / L}$ 和蝇毒磷 $20\mathrm{mg / L}$ 的混合标准中间液。 $0^{\circ}C\sim 4^{\circ}C$ 避光保存，有效期1个月。  
5.4.3 混合标准工作液：准确量取混合标准中间液(5.4.2)适量，用乙腈(5.1.1)稀释，配成浓度分别为：巴胺磷 $1\mathrm{mg / L}$ 、马拉硫磷 $1\mathrm{mg / L}$ 、二嗪农 $1\mathrm{mg / L}$ 、敌百虫 $1\mathrm{mg / L}$ 、敌敌畏 $1\mathrm{mg / L}$ 、甲基吡啶磷 $1\mathrm{mg / L}$ ；辛硫磷 $2\mathrm{mg / L}$ 、倍硫磷 $2\mathrm{mg / L}$ 和蝇毒磷 $2\mathrm{mg / L}$ 的混合标准工作液。 $0^{\circ}C \sim 4^{\circ}C$ 避光保存，有效期1周。  
5.4.4 混合内标中间液：准确吸取内标标准物质溶液(5.3.2)，用乙腈(5.1.1)稀释配制成马拉硫磷- $\mathrm{D}_{10}$ 、二嗪农- $\mathrm{D}_{10}$ 、敌敌畏- $\mathrm{D}_6$ 和甲基吡啶磷- $\mathrm{D}_6$ 均为 $5\mathrm{mg / L}$ ，倍硫磷- $\mathrm{D}_6$ 和蝇毒磷- $\mathrm{D}_{10}$ 均为 $10\mathrm{mg / L}$ 的混合内标中间液， $0^{\circ}C\sim 4^{\circ}C$ 避光保存，有效期1个月。  
5.4.5 混合内标工作液：准确吸取内标中间液(5.4.4)，用乙腈(5.1.1)稀释配成马拉硫磷- $\mathrm{D}_{10}$ 、二嗪农 $\mathrm{D}_{10}$ 、敌敌畏- $\mathrm{D}_6$ 和甲基吡啶磷- $\mathrm{D}_6$ $0.5\mathrm{mg / L}$ ，倍硫磷- $\mathrm{D}_6$ 和蝇毒磷- $\mathrm{D}_{10}1\mathrm{mg / L}$ 的混合内标工作液， $0^{\circ}\mathrm{C}\sim$ $4^{\circ}\mathrm{C}$ 避光保存，有效期2周～3周。

# 5.5 材料

5.5.1 PSA(N-丙基乙二胺)吸附剂：颗粒直径为 $40\mu \mathrm{m}\sim 60\mu \mathrm{m}$   
5.5.2 PVDF（聚偏氟乙烯)滤膜：孔径为 $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱串联质谱仪：配电喷雾离子源  
6.2 电子天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 均质机。  
6.4 离心机：转速不低于 $8000\mathrm{r / min}$   
6.5 超声波清洗器。  
6.6 涡旋混合器。  
6.7 氮吹仪。

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

-18℃以下保存。

# 8 测定步骤

# 8.1 提取

取试样 $5\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，加混合内标工作液 $100~\mu \mathrm{L}(5.4.5)$ 、 $1\%$ 乙酸乙腈(5.2.2) $25.0~\mathrm{mL}$ 无水乙酸钠 $1\mathrm{g}(5.1.4)$ ，涡旋混合 $2\mathrm{min}$ ，超声 $30\mathrm{min},8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ；取上清液，备用。

# 8.2 净化

备用液加PSA吸附剂 $500\mathrm{mg}$ ，涡旋振荡 $1\mathrm{min},8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ；取上清液 $5.0\mathrm{mL},45^{\circ}C$ 下氮气吹干，准确量取定容液 $1.0\mathrm{mL}(5.2.1)$ 溶解残渣，加PSA吸附剂 $200\mathrm{mg}$ ，涡旋 $1\mathrm{min},5000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液过膜后，供液相色谱-串联质谱测定。实验过程需要注意避免阳光直射，样品前处理完

成后应及时上机检测。

# 8.3 标准曲线的制备

准确量取适量混合标准工作液(5.4.3)及混合内标工作液 $20~\mu \mathrm{L}$ (5.4.5)，用定容液(5.2.1)稀释成巴胺磷、马拉硫磷、二嗪农、敌百虫、敌敌畏和甲基吡啶磷浓度分别为 $10~\mu \mathrm{g / L}, 20~\mu \mathrm{g / L}, 50~\mu \mathrm{g / L}, 100~\mu \mathrm{g / L},$ $200~\mu \mathrm{g / L}, 500~\mu \mathrm{g / L}$ ，辛硫磷、倍硫磷和蝇毒磷浓度分别为 $20~\mu \mathrm{g / L}, 40~\mu \mathrm{g / L}, 100~\mu \mathrm{g / L}, 200~\mu \mathrm{g / L},$ $400~\mu \mathrm{g / L}, 1000~\mu \mathrm{g / L}$ 系列标准溶液，供液相色谱-串联质谱测定。以各标准峰面积与内标峰面积的比值为纵坐标，以各标准溶液浓度为横坐标，绘制标准曲线。

# 8.4.3 测定法

# a） 定性测定

在同样测试条件下，阳性样品的保留时间与标准工作液中待测物的保留时间偏差在 $\pm 2.5\%$ 以内，且检测到的相对离子丰度，应当与浓度相当的校正标准溶液相对离子丰度一致。基峰与次强碎片离子丰度比应符合表3的要求。

# b） 定量测定

取试样溶液和相应的标准溶液等体积进样测定，按内标法以标准曲线对样品进行定量。其中辛硫磷、倍硫磷以倍硫磷 $-\mathrm{D}_6$ 为内标；巴胺磷、马拉硫磷以马拉硫磷- $\mathrm{D}_{10}$ 为内标；二嗪农以二嗪农- $\mathrm{D}_{10}$ 为内标；敌百虫、敌敌畏以敌敌畏- $\mathrm{D}_6$ 为内标；甲基吡啶磷以甲基吡啶磷- $\mathrm{D}_6$ 为内标；蝇毒磷以蝇毒磷- $\mathrm{D}_{10}$ 为内标。标准溶液、空白样品及加标样品的总离子流图参见附录A。

# 9 结果计算和表述

试样中有机磷类药物残留量按公式(1)计算。

$$
X = \frac {A \times A _ {i S} ^ {\prime} \times C _ {S} \times C _ {i S} \times V _ {1} \times V _ {3}}{A _ {i S} \times A _ {S} \times C _ {i S} ^ {\prime} \times V _ {2} \times m} \tag {1}
$$

式中：

$C_{iS}$ ——试样溶液中内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$C_{\mathrm{S}}$ ——标准溶液中待测药物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$C_{i5}^{\prime}$ ——标准溶液中内标浓度的数值，单位为微克每升 $(\mu \mathrm{g / L})$

$A_{i}$ ——试样溶液中待测药物的峰面积；

$A_{iS}$ ——试样溶液中内标的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中待测药物的峰面积；

$A_{iS}^{\prime}$ ——标准溶液中内标的峰面积；

$V_{1}$ 提取液体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——净化的上清液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——溶解残渣体积的数值，单位为毫升 $(\mathrm{mL})$ 。

# 10 方法的检测限、定量限、准确度和精密度

# 10.1 检测限和定量限

本方法的检测限：巴胺磷、马拉硫磷、二嗪农、敌百虫、敌敌畏、甲基吡啶磷均为 $5\mu \mathrm{g / kg}$ ，辛硫磷、倍硫磷和蝇毒磷均为 $10~\mu \mathrm{g / kg}$ 。定量限：巴胺磷、马拉硫磷、二嗪农、敌百虫、敌敌畏、甲基吡啶磷均为 $10~\mu \mathrm{g}/$ $\mathrm{kg}$ ，辛硫磷、倍硫磷和蝇毒磷均为 $20~\mu \mathrm{g / kg}$

# 10.2 准确度

巴胺磷、马拉硫磷、二嗪农、敌百虫、敌敌畏、甲基吡啶磷在 $10\mu \mathrm{g / kg}\sim 500\mu \mathrm{g / kg}$ 添加浓度范围内，回收率为 $70\% \sim 110\%$ ；辛硫磷、倍硫磷和蝇毒磷在 $20~\mu \mathrm{g / kg}\sim 1000~\mu \mathrm{g / kg}$ 添加浓度范围内，回收率为 $70\% \sim 110\%$ 。

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

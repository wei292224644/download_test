# 苯乙烯和乙苯的测定

2016-10-19 发布

# 前言

本标准代替GB/T5009.98—2003《食品容器及包装材料用不饱和聚酯树脂及其玻璃钢制品卫生标准的分析方法》中“4.4乙苯类化合物”部分和GB/T5009.59—2003《食品包装用聚苯乙烯树脂卫生标准的分析方法》中“6苯乙烯及乙苯等挥发成分”部分。

本标准与GB/T5009.59—2003中“6苯2烯及乙苯等挥发成分”相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 苯乙烯和乙苯的测定”；  
——修改了色谱柱类型；  
——增加了不溶性样品的测定方法。

# 食品安全国家标准

# 食品接触材料及制品苯乙烯和乙苯的测定

# 1 范围

本标准规定了食品接触材料及制品中苯乙烯和乙苯含量的气相色谱测定方法。

本标准适用于聚苯乙烯制品、不饱和聚酯树脂及其玻璃钢制品中苯乙烯和乙苯的测定。

# 2 原理

试样经二硫化碳提取后，进样气相色谱，在色谱柱中苯乙烯、乙苯与内标物正十二烷及其他组分分离，用氢火焰离子化检测器检测，以内标法定量。

# 3 试剂和材料

# 3.1 试剂

二硫化碳 $(\mathrm{CS}_2$ ，CAS号：75-15-0)：色谱纯。

# 3.2 标准品

3.2.1 苯乙烯 $\left(\mathrm{C}_{8} \mathrm{H}_{8}, \mathrm{CAS}\right.$ 号：100-42-5）：纯度大于 $99.5\%$ ，或经国家认证并授予标准物质证书的标准物质。

3.2.2 乙苯 $\left(\mathrm{C}_{8} \mathrm{H}_{10}, \mathrm{CAS}\right.$ 号：100-41-4）：纯度大于 $99.5\%$ ，或经国家认证并授予标准物质证书的标准物质。

3.2.3 正十二烷 $\left(\mathrm{C}_{12} \mathrm{H}_{26}, \mathrm{CAS}\right.$ 号：112-40-3）：纯度大于 $99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.3 标准溶液配制

# 3.3.1 苯乙烯标准储备液

称取 $200\mathrm{mg}$ （精确至 $0.0001\mathrm{g}$ ) 苯乙烯，用二硫化碳溶解后，定容至 $10~\mathrm{mL}$ ，配制成浓度为 $20\mathrm{mg / mL}$ 的储备液。溶液应于 $4^{\circ}C$ 避光密封储存，有效期为1周。

# 3.3.2 乙苯标准储备液

称取 $200\mathrm{mg}$ （精确至 $0.0001\mathrm{g})$ 乙苯，用二硫化碳溶解后，定容至 $10~\mathrm{mL}$ ，配制成浓度为 $20~\mathrm{mg / mL}$ 的储备液。应于 $4^{\circ}C$ 避光密封储存，有效期为1周。

# 3.3.3 苯乙烯、乙苯混合中间液

用刻度吸量管分别吸取 $0.25\mathrm{mL}$ 的苯乙烯标准储备液及乙苯标准储备液至预先盛有 $5\mathrm{mL}$ 二硫化

碳的 $10 \mathrm{~mL}$ 容量瓶中, 用二硫化碳稀释、定容, 获得苯乙烯、乙苯混合中间液。其中苯乙烯及乙苯浓度分别为 $500.0 \mu \mathrm{g} / \mathrm{mL}, 500.0 \mu \mathrm{g} / \mathrm{mL}$ 。溶液储存条件同 3.3.1。

# 3.3.4 正十二烷内标储备液

称取 $250\mathrm{mg}$ (精确至 $0.0001\mathrm{g}$ ) 正十二烷，用二硫化碳溶解后，定容至 $10\mathrm{mL}$ ，配制成浓度为 $25\mathrm{mg/mL}$ 的储备液。溶液储存条件同3.3.1。

# 3.3.5 正十二烷内标中间液

用刻度吸量管吸取 $0.50\mathrm{mL}$ 的正十二烷内标储备液至预先盛有 $5\mathrm{mL}$ 二硫化碳的 $10\mathrm{mL}$ 容量瓶中，用二硫化碳定容，配制成浓度为 $1250~\mu \mathrm{g / mL}$ 的中间液。溶液储存条件同3.3.1。

# 3.3.6 苯乙烯、苯乙烯及正十二烷混标工作液

用微量注射器及刻度吸量管分别移取 $50~\mu \mathrm{L},250~\mu \mathrm{L},500~\mu \mathrm{L},1.0~\mathrm{mL},2.5~\mathrm{mL},5.0~\mathrm{mL}$ 苯乙烯与乙苯混合中间液于6个已加入 $5\mathrm{mL}$ 二硫化碳的 $25~\mathrm{mL}$ 容量瓶中，再在每个容量瓶中加入 $1.0~\mathrm{mL}$ 正十二烷内标中间液，用二硫化碳定容摇匀，获得工作溶液。苯乙烯及乙苯的浓度均分别为 $1.0~\mu \mathrm{g / mL}$ 、 $5.0~\mu \mathrm{g / mL},10.0~\mu \mathrm{g / mL},20.0~\mu \mathrm{g / mL},50.0~\mu \mathrm{g / mL},100.0~\mu \mathrm{g / mL}$ ，内标浓度为 $50~\mu \mathrm{g / mL}$ 。溶液储存条件同3.3.1。

# 4 仪器和设备

4.1 气相色谱仪：配备氢火焰离子化检测器(FID)。  
4.2 分析天平：感量 $0.0001\mathrm{g}$ 。  
4.3 超声波清洗机。  
4.4 冷冻研磨仪。  
4.5 锥形瓶： $25~\mathrm{mL}$

# 5 分析步骤

# 5.1 试样处理

可溶于二硫化碳的试样直接称量；不溶于二硫化碳的试样，先使用冷冻研磨仪或剪刀等切割工具将其破碎成粒径小于 $1\mathrm{mm}\times 1\mathrm{mm}$ 后再称量。切割样品时，不可使其发热变软。

# 5.2 试样溶液的制备

对于可溶于二硫化碳的试样，称取样品 $0.5\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ）试样于 $25~\mathrm{mL}$ 容量瓶中，移取 $10~\mathrm{mL}$ 二硫化碳于容量瓶中，并加入 $1.0~\mathrm{mL}$ 内标中间液。静置直至试样溶解后，用二硫化碳定容至刻度。对于不溶于二硫化碳的试样，称取样品 $0.5\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ）试样于 $25~\mathrm{mL}$ 锥形瓶中，移取 $10~\mathrm{mL}$ 二硫化碳于锥形瓶中。封盖后用超声波清洗机提取 $20\mathrm{min}$ ，取上层清液于 $25~\mathrm{mL}$ 容量瓶中。然后，以同样方法用 $10~\mathrm{mL}$ 二硫化碳重复提取一次，合并两次上层清液于 $25~\mathrm{mL}$ 容量瓶中，加入 $1.0~\mathrm{mL}$ 内标中间溶液并定容。若样品浓度超出线性范围，需重新提取，并在加入内标溶液前用二硫化碳适当稀释，使其浓度处于线性范围内。不加试样，按照同样方法处理获得空白提取液。

# 5.3 空白溶液的制备

除不加试样外，采用与5.2完全相同的分析步骤、试剂和用量。

# 5.4 仪器参考条件

5.4.1 色谱柱：固定相为聚乙二醇，柱长 $30\mathrm{m}$ 、内径 $0.32\mathrm{mm}$ 、膜厚 $0.5\mu \mathrm{m}$ 。  
5.4.2 进样口温度： $250^{\circ}C$   
5.4.3 柱温：始温 $50~\mathrm{^\circ C}$ 下保持恒温 $1\mathrm{min}$ ，以 $10^{\circ}\mathrm{C / min}$ 速率升温至 $140^{\circ}\mathrm{C}$ ，再以 $20^{\circ}\mathrm{C / min}$ 速率升温至 $220^{\circ}\mathrm{C}$ ，恒温 $10\mathrm{min}$ 。  
5.4.4 进样方式：分流进样，分流比为 $2:1$   
5.4.5 载气：氮气，纯度大于 $99.999\%$ ，流量 $1.5~\mathrm{mL / min}$ 。  
5.4.6 检测器：氢火焰离子化检测器。  
5.4.7 进样量： $1\mu \mathrm{L}$   
5.4.8 检测器温度： $300^{\circ}C$   
5.4.9 氢气流量： $30~\mathrm{mL / min}$ ，纯度 $\geqslant 99.999\%$   
5.4.10 空气流量： $300~\mathrm{mL / min}$ ，纯度 $\geqslant 99.999\%$

# 5.5 标准曲线的制作

按照5.4所列条件，将苯乙烯、苯乙烯及正十二烷混标工作液进样气相色谱仪测定。以标准溶液工作溶液中苯乙烯浓度(或乙苯浓度)为横坐标（单位为微克每毫升 $\mu \mathrm{g / mL}$ )，以对应苯乙烯峰面积(或乙苯峰面积)与内标物正十二烷峰面积之比为纵坐标，分别绘制标准曲线。苯乙烯、乙苯及正十二烷标准溶液的色谱图参见图A.1。

# 5.6 试样溶液的测定

按照5.4所列条件，将试样溶液、空白溶液进气相色谱仪测定，扣除空白值，得到苯乙烯(或乙苯)峰面积与正十二烷面积的比值。

# 6 分析结果的表述

试样中苯乙烯(或乙苯)含量按式(1)计算：

$$
X _ {i} = \frac {\rho_ {i} \times V \times f \times 1 0 ^ {- 3}}{m \times 1 0 ^ {- 3}} \quad \dots \tag {1}
$$

式中：

$X_{i}$ ——试样中残留苯乙烯(或乙苯)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho_{i}$ ——依据校准曲线获得的试样溶液中苯乙烯（或乙苯）质量浓度，单位为微克每毫升 $(\mu \mathrm{g / mL})$ ；

$V$ ——试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——浓度稀释因子；

$m$ ——试样称样量，单位为克(g)；

$10^{-3}$ ——单位换算因子。

计算结果保留至小数点后一位。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过其算术平均值的 $10\%$ 。

# 8 其他

苯乙烯及乙苯方法检出限均为 $20\mathrm{mg / kg}$ ；苯乙烯及乙苯方法定量限均为 $50~\mathrm{mg / kg}$ 。

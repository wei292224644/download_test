# 前言

本标准代替GB17779—1999《食品添加剂L-苏糖酸钙》。

本标准与GB17779—1999相比，主要变化如下：

——删除了重金属指标及试验方法，增加了铅指标及试验方法；

将砷指标由 $0.3\mathrm{mg / kg}$ 修改为 $3\mathrm{mg / kg}$

——增加了氯化物指标及试验方法。

本标准的附录A和附录B为规范性附录。

本标准所代替标准的历次版本发布情况为：

GB17779-1999。

# 检验方法

# A.1 安全提示

本标准试验方法中使用的部分试剂具有毒性或腐蚀性，按相关规定操作，使用时需小心谨慎。若溅到皮肤上应立即用水冲洗，严重者应立即治疗。在使用挥发性酸时，要在通风橱中进行。

# A.2 一般规定

本标准所用试剂除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682—2008中规定的三级水。

试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603之规定制备。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸。

A.3.1.2 盐酸溶液： $1 + 3$

A.3.1.3 草酸铵试液： $35\mathrm{g / L}$

A.3.1.4 三氯化铁试液： $90\mathrm{g / L}$ （现用现配）。

A.3.1.5 甲基红试液： $0.5\mathrm{g / L}$

A.3.2 鉴别试验

# A.3.2.1 草酸铵沉淀试验

# A.3.2.1.1 方法原理

钙离子与草酸根反应，生成草酸钙沉淀，草酸钙不溶于乙酸，但可溶于盐酸。

# A.3.2.1.2 分析步骤

称取约 $1 \mathrm{~g}$ 实验室样品, 精确至 $0.01 \mathrm{~g}$ , 溶于 $50 \mathrm{~mL}$ 水中, 加2滴甲基红指示液, 滴加盐酸至恰成酸性; 加草酸铵溶液, 即生成白色沉淀; 分离, 沉淀不溶于乙酸, 但可溶于盐酸。

# A.3.2.2 三氯化铁鉴别

# A.3.2.2.1 方法原理

L-苏糖酸钙与三氯化铁试液反应，生成黄绿色复合物。

# A.3.2.2.2 分析步骤

称取约 $0.1 \mathrm{~g}$ 实验室样品，精确至 $0.01 \mathrm{~g}$ ，加 $5 \mathrm{~mL}$ 水溶解后，加三氯化铁溶液1滴，呈黄绿色。

# A.3.2.3 红外光吸收谱鉴别

红外光谱图（光谱图见附录B），其特征峰波数为 $3100\mathrm{cm}^{-1}$ 、 $1599\mathrm{cm}^{-1}$ 、 $1433\mathrm{cm}^{-1}$ 、 $1383\mathrm{cm}^{-1}$ 、 $1262\mathrm{cm}^{-1}$ 、 $1223\mathrm{cm}^{-1}$ 、 $1022\mathrm{cm}^{-1}$ 。

# A.4 L-苏糖酸钙的测定

# A.4.1 方法原理

在碱性条件下, 以钙紫红素为指示剂, 用乙二胺四乙酸二钠标准滴定液滴定样品 (换算为干品) 水溶液, 根据二乙胺四乙酸二钠标准滴定液的用量, 以 $\mathrm{C}_{8} \mathrm{H}_{14} \mathrm{CaO}_{10}$ 计的含量。

# A.4.2 试剂和材料

A.4.2.1 氢氧化钠溶液： $c$ （ $\mathrm{NaOH}) = 1\mathrm{mol / L}$   
A. 4.2.2 乙二胺四乙酸二钠标准滴定溶液: $c(\mathrm{EDTA}) = 0.05 \mathrm{~mol} / \mathrm{L}$ 。  
A.4.2.3 钙紫红素指示剂：取 $0.1\mathrm{g}$ 钙紫红素，加 $10\mathrm{g}$ 无水硫酸钠，研磨均匀。

# A.4.3 分析步骤

称取约 $0.48 \mathrm{~g}$ 实验室样品, 精确至 $0.0001 \mathrm{~g}$ , 加 $100 \mathrm{~mL}$ 水, 微热溶解, 放至室温, 加 $15 \mathrm{~mL}$ 氢氧化钠溶液, $0.1 \mathrm{~g}$ 钙紫红素指示剂, 用乙二胺四乙酸二钠标准滴定溶液滴定至溶液由紫红色转变为纯蓝色。

在测定的同时，按与测定相同的步骤，对不加实验室样品而相同数量的试剂溶液做空白试验。

# A.4.4 结果计算

L-苏糖酸钙（以 $\mathrm{C_8H_{14}CaO_{10}}$ 计）的质量分数 $w_{1}$ ，数值以%表示，按公式(A.1)计算：

$$
w _ {1} = \frac {\left(V - V _ {0}\right) \times c \times M}{m \times \left(1 - w _ {2}\right) \times 1000} \times 100 \% \dots \tag{A.1}
$$

式中：

$V$ ——实验室样品消耗乙二胺四乙酸二钠标准滴定液（A.4.2.2）体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗二乙胺四乙酸二钠标准滴定液（A.4.2.2）体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——乙二胺四乙酸二钠标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——实验室样品质量的数值，单位为克（g）；

$w_{2}$ ——A.6测得的L-苏糖酸钙干燥减量的质量分数，数值以%表示；

$M$ ——L-苏糖酸钙的摩尔质量的数值，单位为克每摩尔（ $\mathrm{g/mol}$ ）（ $M=310.27$ ）。

取两次测定结果的算术平均值为测定结果，两次测定的绝对差值不大于 $0.2\%$

# A.5 比旋光度的测定

# A.5.1 分析步骤

称取约 $1\mathrm{g}$ 实验室样品，准确至 $0.0001\mathrm{g}$ ，加水微加热溶解，放至室温后，移至 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀，其他按GB/T613—2007规定的方法进行。

# A.5.2 结果计算

比旋光度 $\alpha_{\mathrm{m}}(20^{\circ}\mathrm{C},\mathrm{D})$ 数值以 $(^{\circ})\cdot \mathrm{dm}^{2}\cdot \mathrm{kg}^{-1}$ 表示，按公式(A.2)计算：

$$
\alpha_ {m} (2 0 ^ {\circ} C, D) = \frac {\alpha}{l \rho_ {\alpha}} \quad \dots \tag {A.2}
$$

式中：

$a$ 测得的旋光角，单位为度(°）；

$l$ ——旋光管的长度,单位为分米(dm);

$\rho_{a}$ ——溶液中有效组分的质量浓度,单位为克每毫升 $(\mathrm{g / mL})$ 。

# A.6 干燥减量的测定

# A.6.1 方法原理

干燥减量是指样品在规定条件下经干燥后所减失的质量，根据所减失的质量和取样量计算实验室样品干燥减量的百分率。

# A.6.2 仪器和设备

恒温干燥箱。

# A.6.3 测定方法

称取约 $1\mathrm{g}$ 实验室样品，精确至 $0.0001\mathrm{g}$ ，置与实验室样品相同条件下干燥至恒重的扁形称量瓶中，在 $80^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 干燥 $2\mathrm{h}$ 。

# A.6.4 结果计算

L-苏糖酸钙干燥减量以质量分数 $w_{2}$ 计，数值以%表示，按公式(A.3)计算

$$
\mathcal {W} _ {2} = \frac {m _ {3} - m _ {1}}{m _ {2} - m _ {1}} \times 100 \% \dots \tag{A.3}
$$

式中：

$m_{1}$ ——称量瓶的质量的数值，单位为克（g）；

$m_{2}$ ——称量瓶和干燥前实验室样品质量的数值，单位为克（g）；

$m_{3}$ ——称量瓶和干燥后实验室样品的质量的数值，单位为克（g）。

取两次测定结果的算术平均值为测定结果，两次测定的绝对差值不大于 $0.05\%$

# A.7 氯化物的测定

# A.7.1 试剂和材料

A.7.1.1 硝酸溶液： $25\rightarrow 100$ 。

A.7.1.2 硝酸银溶液： $17\mathrm{g / L}$

A.7.1.3 氯化物（Cl）标准溶液： $0.01\mathrm{g / L}$

# A.7.2 分析步骤

称取约 $0.4 \mathrm{~g} \pm 0.01 \mathrm{~g}$ 实验室样品，置于 $100 \mathrm{~mL}$ 容量瓶中，加入 $20 \mathrm{~mL}$ 硝酸溶解，用水稀释至刻度，摇匀，即为实验室样品溶液。量取 $5 \mathrm{~mL} \pm 0.025 \mathrm{~mL}$ 实验室样品溶液置于 $25 \mathrm{~mL}$ 比色管中，同时取 $10 \mathrm{~mL}$ 氯化物标准溶液置于另一支 $25 \mathrm{~mL}$ 比色管中，其他按GB/T9729进行测定。

2022-06-30 发布

# 前言

本标准代替GB5413.14—2010《食品安全国家标准婴幼儿食品和乳品中维生素 $\mathrm{B}_{12}$ 的测定》。

本标准与GB5413.14—2010相比，主要变化如下：

——增加了第一法液相色谱法；  
——增加了第二法液相色谱-质谱法；  
——删除了规范性引用文件；  
——修改了原微生物法为第三法。

# 食品安全国家标准

# 食品中维生素 $\mathbf{B}_{12}$ 的测定

# 1 范围

本标准规定了食品中维生素 $\mathrm{B}_{12}$ 的测定方法。

第一法液相色谱法适用于婴幼儿食品、乳及乳制品、肉及肉制品中维生素 $\mathrm{B}_{12}$ 的测定。

第二法液相色谱-质谱法适用于婴幼儿食品、乳及乳制品、肉及肉制品、即食谷物、烘焙食品、果冻、饮料中维生素 $\mathrm{B}_{12}$ 的测定。

第三法微生物法适用于婴幼儿食品、乳及乳制品中维生素 $\mathrm{B}_{12}$ 的测定。

# 第一法 液相色谱法

# 2 原理

试样经酶解后，用氰化钾(或氰化钠)溶液将钴胺素异构体（羟钴胺素、甲钴胺素和5-脱氧腺苷钴胺素等）转化为氰钴胺素。样液通过免疫亲和柱净化、浓缩后，反相液相色谱柱分离，紫外检测器检测，外标法定量。

# 3 试剂和材料

除非另有说明，所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 无水乙酸钠 $\mathrm{(CH_3COONa)}$ 。  
3.1.2 乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.3 甲醇 $\mathrm{(CH_3OH)}$ ：色谱级。  
3.1.4 乙腈 $\mathrm{(CH_3CN)}$ ：色谱级。  
3.1.5 三氟乙酸 $\mathrm{(CF_3COOH)}$ ：色谱级。  
3.1.6 氰化钾或氰化钠(KCN/NaCN)。  
3.1.7 胃蛋白酶(CAS号：9001-75-6，活力 $\geqslant 400\mathrm{U / mg})$ 。  
3.1.8 淀粉酶（活力 $\geqslant 50\mathrm{U / mg}$ ）。  
3.1.9 乙醇 $\mathrm{(C_2H_6O)}$ 。

# 3.2 试剂配制

3.2.1 乙醇溶液 $(25\%)$ ：量取 $250~\mathrm{mL}$ 乙醇，加水稀释至 $1000~\mathrm{mL}$ ，混匀。  
3.2.2 乙酸钠缓冲液 $(0.25\mathrm{mol} / \mathrm{L})$ ：称取 $20.5\mathrm{g}$ 无水乙酸钠，用 $950~\mathrm{mL}$ 水溶解并用乙酸调 $\mathrm{pH}$ 至4.0 $\pm 0.1$ ，用水稀释至 $1000~\mathrm{mL}$ 。

3.2.3 氰化钾(或氰化钠)溶液 $(10\mathrm{mg / mL})$ ：称取 $1.0\mathrm{g}$ 氰化钾(或氰化钠)固体，加适量水溶解，并用水稀释至 $100~\mathrm{mL}$ 。

注：氰化钾(或氰化钠)为剧毒化学品；操作者须带防护用具，在通风橱内进行氰化钾(或氰化钠)溶液的配制或使用。

3.2.4 三氟乙酸溶液 $(0.04\%)$ ，体积比）：量取 $500~\mathrm{mL}$ 水，加入 $200~\mu \mathrm{L}$ 三氟乙酸，混匀。

# 3.3 标准品

维生素 $\mathrm{B}_{12}$ (氰钴胺素)标准品 $(\mathrm{C}_{63} \mathrm{H}_{88} \mathrm{CoN}_{14} \mathrm{O}_{14} \mathrm{P}, \mathrm{CAS}$ 号：68-19-9)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 维生素 $\mathrm{B}_{12}$ 标准储备液 $(1\mathrm{mg / mL})$ ：称取 $10\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ )维生素 $\mathrm{B}_{12}$ 标准品于 $50\mathrm{mL}$ 烧杯中，用乙醇溶液溶解后，转移至 $10~\mathrm{mL}$ 容量瓶中，用乙醇溶液定容至刻度，摇匀，转移至棕色试剂瓶，于一 $20^{\circ}C$ 下避光保存，有效期6个月。  
3.4.2 维生素 $\mathrm{B}_{12}$ 标准工作液 $(10\mu \mathrm{g / mL})$ ：吸取 $1.00~\mathrm{mL}$ 维生素 $\mathrm{B}_{12}$ 标准储备液，置于 $100~\mathrm{mL}$ 容量瓶中，用乙醇溶液定容至刻度。转移至棕色试剂瓶中，于 $4^{\circ}C$ 下避光保存，有效期1个月。  
3.4.3 维生素 $\mathrm{B}_{12}$ 标准系列工作溶液：准确吸取标准工作液 $30~\mu \mathrm{L},60~\mu \mathrm{L},90~\mu \mathrm{L},150~\mu \mathrm{L}$ 和 $200~\mu \mathrm{L}$ ，分别置于 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度。其浓度分别为： $30~\mathrm{ng / mL},60~\mathrm{ng / mL},90~\mathrm{ng / mL},$ $150~\mathrm{ng / mL}$ 和 $200~\mathrm{ng / mL}$ 。临用现配。

# 3.5 材料

3.5.1 维生素 $\mathrm{B}_{12}$ 免疫亲和柱，柱容量 $\geq 800\mathrm{ng}$ ，柱回收率 $\geq 85\%$ （验证方法参见附录A）。  
3.5.2 玻璃纤维滤纸。  
3.5.3 微孔滤膜：水相， $0.22\mu \mathrm{m}$

# 4 仪器和设备

4.1 液相色谱仪：带紫外检测器。  
4.2 天平：感量为 $0.01\mathrm{g}, 0.001\mathrm{g}$ 和 $0.00001\mathrm{g}$ 。  
4.3 pH计：精度0.01。  
4.4 水浴恒温振荡器：温度范围 $25^{\circ} \mathrm{C} \sim 100^{\circ} \mathrm{C}$ 。  
4.5 超声波清洗器。  
4.6 离心机：转速 $\geqslant 10000\mathrm{r / min}$

# 5 分析步骤

注：操作过程应避免紫外光照，并尽可能避光操作。

# 5.1 样品前处理

# 5.1.1 试样制备

固体样品粉碎、研磨，或用绞肉机制成食糜，均质混匀。液体样品试验前振摇混匀。

# 5.1.2 样品提取

称取 $5\mathrm{g}\sim 10\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )于 $150~\mathrm{mL}$ 具塞锥形瓶中，依次加入 $30~\mathrm{mL}$ 乙酸钠缓冲液、 $0.2\mathrm{g}$ 胃蛋白酶、 $0.05\mathrm{g}$ 淀粉酶和 $2\mathrm{mL}$ 氰化钾(或氰化钠)溶液，充分混合。将样品溶液放入水浴恒温振荡器，在 $37^{\circ}C$ ，酶解 $30\mathrm{min}$ (肉类样品酶解 $10\mathrm{h}\sim 16\mathrm{h}$ )，酶解后转入 $100^{\circ}C$ 水浴中，保持 $30\mathrm{min}$ ，取出冷却至室温。将样品溶液转移至 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，摇匀。移取上述溶液 $40~\mathrm{mL}$ 至 $50~\mathrm{mL}$ 离心管中，在 $10000\mathrm{r / min}$ 下离心 $10\mathrm{min}$ ，取上清液用玻璃纤维滤纸过滤后备用。

注：对于以氰钴铵素为营养强化剂，且本底中不含有其他形式钴胺素的样品，在样品提取时可不使用氰化钾（或氰化钠）。

# 5.1.3 净化

将免疫亲和柱连接至固相萃取装置上，弃去免疫亲和柱内的缓冲液后，移取适量上述滤液（其中含维生素 $\mathrm{B}_{12}$ 为 $10\mathrm{ng} \sim 500\mathrm{ng}$ ）过柱，调节过柱速度为 $2\mathrm{mL} / \mathrm{min} \sim 3\mathrm{mL} / \mathrm{min}$ 。待样液完全过柱后，用 $10\mathrm{mL}$ 水以稳定流速淋洗免疫亲和柱，抽干。在免疫亲和柱下放置 $10\mathrm{mL}$ 玻璃试管，用 $3\mathrm{mL}$ 甲醇分3次洗脱，收集全部洗脱液，在 $60^{\circ}\mathrm{C}$ 以下用氮气流缓缓吹至近干，用 $0.04\%$ 三氟乙酸溶液定容至 $1.0\mathrm{mL}$ ，涡旋 $30\mathrm{s}$ 溶解残留物， $0.22\mu \mathrm{m}$ 滤膜过滤，待测。

注：氰化钾(或氰化钠)废液严禁与酸混合，可加入氢氧化钠调节 $\mathrm{pH} > 10$ ，再加入高锰酸钾粉末（按废液质量的 $3\%$ 加入），使氰化物分解，放置 $24\mathrm{h}$ 后排放。

# 5.2 液相色谱参考条件

5.2.1 色谱柱： $\mathrm{C_{18}}$ 柱（柱长 $150~\mathrm{mm}$ ，柱内径 $4.6~\mathrm{mm}$ ，填料粒径 $2.5\mu \mathrm{m}$ ），或相当者。  
5.2.2 流动相：A相， $0.04\%$ 三氟乙酸溶液；B相，乙腈。  
5.2.3 梯度洗脱： $0\mathrm{min}\sim 6.0\mathrm{min},90\% \mathrm{A};6.0\mathrm{min}\sim 8.5\mathrm{min},90\% \sim 0\% \mathrm{A};8.5\mathrm{min}\sim 14.0\mathrm{min},$ $90\%$ A。  
5.2.4 流速： $0.8\mathrm{mL / min}$   
5.2.5 柱温： $40^{\circ}\mathrm{C}$ 。  
5.2.6 进样体积： $100\mu \mathrm{L}$   
5.2.7 检测波长： $361~\mathrm{nm}$

# 5.3 标准曲线的制作

将标准系列溶液由低浓度到高浓度依次注入液相色谱仪中，测定相应的色谱峰面积，以维生素 $\mathrm{B}_{12}$ 的峰面积为纵坐标，以维生素 $\mathrm{B}_{12}$ 的浓度为横坐标绘制标准曲线。液相色谱图见附录B中图B.1。

# 5.4 样品测定

将待测样液注入液相色谱仪中，得到待测溶液中维生素 $\mathrm{B}_{12}$ 的峰面积，根据标准曲线得到测定液中维生素 $\mathrm{B}_{12}$ 的浓度。待测样液中维生素 $\mathrm{B}_{12}$ 的响应值应在标准曲线的线性范围内，超过线性范围则应稀释上机溶液，或调整取样量重新按样品分析步骤处理后再进样分析。

# 5.5 空白试验

不称取试样，按样品分析步骤操作，应不含有干扰待测组分的物质。

# 6 分析结果的表述

试样中维生素 $\mathrm{B}_{12}$ （以氰钴胺素计）的含量按式(1)计算：

$$
X = \frac {\rho \times V _ {1} \times V _ {3} \times 1 0 0}{m \times V _ {2} \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中维生素 $\mathrm{B}_{12}$ 的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ ；

$\rho$ ——由标准曲线得到的试样溶液中维生素 $\mathrm{B}_{12}$ 的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V_{1}$ ——样品提取液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——试样过柱洗脱液的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

100 ——换算系数；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——上柱溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

当取样量为 $5.00\mathrm{g}$ 时，婴幼儿食品和乳品、肉与肉制品的检出限为 $0.2\mu \mathrm{g} / 100\mathrm{g}$ ，定量限为 $0.5\mu \mathrm{g} / 100\mathrm{g}$ 。

# 12.1 样品前处理

# 12.1.1 试样制备

固体样品需粉碎、研磨，或用绞肉机制成食糜，均质混匀。液体样品测定前振摇混匀。

# 12.1.2 提取

# 12.1.2.1 婴幼儿食品和乳品、肉与肉制品、烘焙食品、即食谷物

注：氰化钾(或氰化钠)废液严禁与酸混合，可加入氢氧化钠调节 $\mathrm{pH} > 10$ ，再加入高锰酸钾粉末（按废液质量的 $3\%$ 加入），使氰化物分解，放置 $24\mathrm{h}$ 后排放。

# 12.4 标准曲线的制作

将标准系列溶液由低浓度到高浓度依次注入液相色谱-串联质谱仪中，测定相应的色谱峰面积，以

维生素 $\mathrm{B}_{12}$ 与其同位素内标的峰面积比值为纵坐标，以维生素 $\mathrm{B}_{12}$ 的浓度为横坐标绘制标准曲线。

# 12.5 样品测定

将待测样液注入液相色谱-串联质谱仪中，得到待测溶液中维生素 $\mathrm{B}_{12}$ 与其同位素内标的峰面积比值，根据标准曲线得到待测液中维生素 $\mathrm{B}_{12}$ 的浓度。待测液中维生素 $\mathrm{B}_{12}$ 的响应值应在标准曲线的线性范围内，超过线性范围则应减少取样量，重新按样品分析步骤处理后再进样分析。

# 12.6 空白试验

# 13 分析结果的表述

试样中维生素 $\mathrm{B}_{12}$ （以氰钴胺素计）的含量按式(2)计算：

$$
X = \frac {\rho \times V \times 1 0 0}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中维生素 $\mathrm{B}_{12}$ 的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ ；

$\rho$ ——由标准曲线得到的试样溶液中维生素 $\mathrm{B}_{12}$ 的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样过柱洗脱液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

100 ——换算系数；

$m$ ——试样的取样量，单位为克(g)；

1000 ——换算系数。

计算结果保留两位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 第三法 微生物法

# 16 原理

维生素 $\mathrm{B}_{12}$ 是莱士曼氏乳酸杆菌生长所必需的营养素。在一定条件下，莱士曼氏乳酸杆菌的生长与维生素 $\mathrm{B}_{12}$ 的含量成对应关系。根据维生素 $\mathrm{B}_{12}$ 的含量与透光率(或吸光度值)的标准工作曲线，计算出试样中维生素 $\mathrm{B}_{12}$ 的含量。

# 19.5 灭菌

在19.3和19.4中所有需培养的试管内分别加入一粒玻璃珠，盖上试管帽， $121^{\circ}\mathrm{C}$ 灭菌 $5\mathrm{min}$ （商品培养基按标签说明进行灭菌）。

# 19.6 接种

将灭菌后的试管迅速冷却至 $30^{\circ}\mathrm{C}$ 以下。除标准曲线管中空白S1管外，用移液器分别向上述试管

中加入1滴(约 $50~\mu \mathrm{L}$ )测试菌液。

# 19.7 培养

将试管于 $36\mathrm{^\circ C}\pm 1\mathrm{^\circ C}$ 培养 $19\mathrm{h}\sim 20\mathrm{h}$ 。培养结束后，对每支试管进行目测检查，未接种试管S1内培养液应是澄清的，如果S1管出现浑浊，则测定无效，需重做试验。

# 19.8 测定

19.8.1 以接种空白管做对照，测定最高浓度标准曲线试管的透光率， $2\mathrm{h}$ 后重新测定。两次结果透光率差值若小于 $2\%$ ，则取出全部检验管测其透光率。  
19.8.2 用未接种空白试管(S1)作空白，将分光光度计透光率调到 $100\%$ （或吸光度为0)，读出接种空白试管(S2)的读数。再以接种空白试管(S2)为空白，调节透光率为 $100\%$ （或吸光度为0)，依次读出其他每支试管的透光率(或吸光度)。  
19.8.3 用涡旋振荡器充分混合每一支试管内培养物（也可以加一滴消泡剂）后，立即将培养液移入比色皿内进行测定，波长 $550 \mathrm{~nm}$ 。待读数稳定 $30 \mathrm{~s}$ 后，读出透光率，每支试管的稳定时间要相同。各标准曲线浓度点3管吸光度值的相对标准偏差应小于 $15 \%$ 。如果3支试管吸光度的相对标准偏差大于或等于 $15 \%$ ，则舍去该标准曲线浓度点（舍去的标准曲线浓度点数量不得超过一个）。以标准系列管中维生素 $\mathrm{B}_{12}$ 浓度为横坐标，透光率为纵坐标，绘制标准曲线。  
19.8.4 根据待测液的透光率, 从标准工作曲线中计算该待测液中维生素 $\mathrm{B}_{12}$ 的浓度, 根据稀释因子和称样量计算出试样中维生素 $\mathrm{B}_{12}$ 的含量。透光率超出标准曲线管 S3～S10 范围的测试值要舍去。  
19.8.5 用每支试管的透光率计算每毫升每个编号待测液中维生素 $\mathrm{B}_{12}$ 的浓度，并计算该编号待测液的维生素 $\mathrm{B}_{12}$ 浓度平均值，每支试管测得的浓度不得超过该平均值的 $15\%$ ，超过者要舍去。如果符合该要求的管数少于所有的四个编号的待测液的总管数的 $2/3$ ，用于计算试样含量的数据是不充分的，需要重新检验。如果符合要求的管数超过原来管数的 $2/3$ ，重新计算每一个编号的有效试样管中每毫升测定液中维生素 $\mathrm{B}_{12}$ 含量的平均值，以此平均值计算全部编号试样管的总平均值为 $\rho$ 。用式(3)计算试样中维生素 $\mathrm{B}_{12}$ 的含量 $X$ 。

注：绘制标准曲线，既可读取透光率 $(T\%)$ ，也可读取吸光度 $(A)$ 。

# 20 分析结果的表述

试样中维生素 $\mathrm{B}_{12}$ （以氰钴胺素计）的含量按式(3)计算：

$$
X = \frac {\rho \times f \times 1 0 0}{m \times 1 0 0 0} \tag {3}
$$

式中：

$X$ ——样品中维生素 $\mathrm{B}_{12}$ 的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ ；

$\rho$ ——有效试样管中维生素 $\mathrm{B}_{12}$ 含量的总平均值，单位为纳克 $(\mathrm{ng})$ ；

$f$ ——样液稀释倍数；

100 ——换算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

1000 ——换算系数。

结果保留两位有效数字。

# 食品安全国家标准

# 1 范围

本文件规定了动物性食品中酰胺醇类药物及其代谢物残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、鸡、牛、羊的肌肉、肝脏、肾脏、脂肪组织，以及鸡蛋、牛奶、羊奶中氯霉素、甲砜霉素、氟苯尼考和氟苯尼考胺残留量的测定。

# 2 规范性引用文件

# 4 原理

试样中残留的氯霉素、甲砜霉素、氟苯尼考和氟苯尼考胺用 $2\%$ 氨化乙酸乙酯溶液提取，正己烷脱脂，氨化乙酸乙酯反萃取，液相色谱-中联质谱法测定，内标法定量。

5.2.5 $10\mathrm{mmol / L}$ 甲酸铵溶液：取甲酸铵 $0.63\mathrm{g}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。

# 5.3 标准品

氯霉素、甲砜霉素、氟苯尼考、氟苯尼考胺，含量均 $\geq 99\%$ ，氯霉素- $\mathrm{D}_5$ 、甲砜霉素- $\mathrm{D}_3$ 、氟苯尼考- $\mathrm{D}_3$ 、氟苯尼考胺- $\mathrm{D}_3$ 内标，含量均 $\geq 98\%$ ，具体见附录A。

# 5.4 标准溶液制备

5.4.1 标准储备液：取氯霉素、甲砜霉素、氟苯尼考、氟苯尼考胺标准品各适量（相当于各活性成分约 $10\mathrm{mg}$ ），精密称定，分别加甲醇适量使溶解并稀释定容至 $100\mathrm{mL}$ 容量瓶中，配制成浓度为 $100\mu \mathrm{g} / \mathrm{mL}$ 的标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期12个月。  
5.4.2 内标储备液：取氯霉素 $-\mathrm{D}_{5}$ 、甲砜霉素 $-\mathrm{D}_{3}$ 、氟苯尼考 $-\mathrm{D}_{3}$ 、氟苯尼考胺 $-\mathrm{D}_{3}$ 内标各适量（相当于各活性成分约 $1\mathrm{mg}$ ），精密称定，分别加甲醇适量使溶解并稀释定容至 $10\mathrm{mL}$ 容量瓶中，配制成浓度为 $100\mu \mathrm{g} / \mathrm{mL}$ 的内标储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期12个月。  
5.4.3 混合标准中间液：分别精密量取氯霉素标准储备液 $0.1\mathrm{mL}$ ，甲砜霉素、氟苯尼考、氟苯尼考胺标准储备液各 $0.5\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成氯霉素浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ ，甲砜霉素、氟苯尼考、氟苯尼考胺浓度为 $5\mu \mathrm{g} / \mathrm{mL}$ 混合标准中间液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期3个月。  
5.4.4 混合内标中间液：分别精密量取氯霉素- $\mathrm{D}_5$ 内标储备液 $0.1\mathrm{mL}$ ，甲砜霉素- $\mathrm{D}_3$ 、氟苯尼考- $\mathrm{D}_3$ 、氟苯尼考胺- $\mathrm{D}_3$ 内标储备液各 $0.5\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成氯霉素- $\mathrm{D}_5$ 浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ ，甲砜霉素- $\mathrm{D}_3$ 、氟苯尼考- $\mathrm{D}_3$ 、氟苯尼考胺- $\mathrm{D}_3$ 浓度为 $5\mu \mathrm{g} / \mathrm{mL}$ 混合内标中间液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期3个月。  
5.4.5 混合内标工作液：取混合内标中间液，用 $20\%$ 甲醇溶液稀释成氯霉素- $\mathrm{D}_5$ 浓度为 $10~\mathrm{ng / mL}$ ，甲砜霉素- $\mathrm{D}_3$ 、氟苯尼考- $\mathrm{D}_3$ 、氟苯尼考胺- $\mathrm{D}_3$ 浓度为 $50~\mathrm{ng / mL}$ 混合内标工作液，现配现用。

# 8 测定步骤

# 8.1 提取

取试料 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 离心管中，加混合内标工作液 $100~\mu \mathrm{L}$ ，涡旋混匀，再加 $2 \%$ 氨化乙酸乙酯溶液 $10~\mathrm{mL}$ （牛奶、羊奶样品需另加无水硫酸钠 $3\mathrm{g}$ )，涡旋 $30~\mathrm{s}$ ，涡旋振荡 $10\mathrm{min}$ $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。上清液转入另一 $50~\mathrm{mL}$ 离心管中，残渣中加 $2 \%$ 氨化乙酸乙酯溶液 $10~\mathrm{mL}$ ，重复提取一次。合并两次提取液，于 $50^{\circ}C$ 氮气吹干，待净化。

# 8.2 净化

取待净化残渣，加 $4\%$ 氯化钠溶液 $3\mathrm{mL}$ ，涡旋使溶解，再加 $4\%$ 氯化钠饱和的正己烷 $5\mathrm{mL}$ ，涡旋 $30~s$ $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去上层正己烷层，用 $4\%$ 氯化钠饱和的正己烷重复脱脂一次。加 $2\%$ 氨化乙酸乙酯溶液 $5\mathrm{mL}$ ，涡旋振荡 $5\mathrm{min}$ ， $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上层有机相。用 $2\%$ 氨化乙酸乙酯溶液 $5\mathrm{mL}$ 重复萃取一次，合并有机相， $50^{\circ}C$ 氮气吹干，加 $20\%$ 甲醇溶液 $1.0\mathrm{mL}$ ，涡旋 $30~s$ ，过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱仪测定。

# 8.3 标准曲线的制备

精密量取混合标准中间液和混合内标工作液适量，用 $20\%$ 甲醇溶液稀释，配制成氯霉素浓度分别为 $0.2\mu \mathrm{g} / \mathrm{L},0.5\mu \mathrm{g} / \mathrm{L},1\mu \mathrm{g} / \mathrm{L},2\mu \mathrm{g} / \mathrm{L},5\mu \mathrm{g} / \mathrm{L},10\mu \mathrm{g} / \mathrm{L}$ ，甲砜霉素、氟苯尼考、氟苯尼考胺浓度分别为 $1\mu \mathrm{g} / \mathrm{L},2.5\mu \mathrm{g} / \mathrm{L},5\mu \mathrm{g} / \mathrm{L},10\mu \mathrm{g} / \mathrm{L},25\mu \mathrm{g} / \mathrm{L},50\mu \mathrm{g} / \mathrm{L}$ ，氯霉素- $\mathbf{D}_5$ 浓度均为 $1\mu \mathrm{g} / \mathrm{L}$ ，甲砜霉素- $\mathbf{D}_3$ 、氟苯尼考- $\mathbf{D}_3$ 、氟苯尼考胺- $\mathbf{D}_3$ 浓度均为 $5\mu \mathrm{g} / \mathrm{L}$ 的系列标准溶液，临用现配，供液相色谱-串联质谱仪测定。以定量离子峰面积比为纵坐标、浓度为横坐标绘制标准曲线，求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在同样测试条件下, 试料溶液中酰胺醇类药物及其代谢物的保留时间与标准工作液中酰胺醇类药物及其代谢物的保留时间相对偏差在 $\pm 2.5\%$ 以内, 且检测到的相对离子丰度, 应当与浓度相当的校正标准溶液相对离子丰度一致。其允许偏差应符合表3要求。

# 8.4.3.2 定量测定

取试料溶液和相应的标准溶液, 作单点或多点校准, 按内标法以色谱峰面积比定量。标准溶液及试样溶液中酰胺醇类药物及其代谢物与其相应内标峰面积比均应在仪器检测的线性范围内。对于残留量超出仪器线性范围的, 在提取时根据药物浓度相应增加内标工作液的添加量, 使试样溶液稀释后酰胺醇类药物及其代谢物的浓度在曲线范围之内, 对应内标浓度与标准工作液一致。标准溶液特征离子质量色谱图见附录 B。

# 9 结果计算和表述

试样中酰胺醇类药物及其代谢物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {s}} \times C _ {\mathrm {i s}} \times A _ {\mathrm {i}} \times A ^ {\prime} {} _ {\mathrm {i s}} \times V}{C ^ {\prime} {} _ {\mathrm {i s}} \times A _ {\mathrm {s}} \times A _ {\mathrm {i s}} \times m} \tag {1}
$$

式中：

$X$ ——试样中酰胺醇类药物及其代谢物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$C_{\mathrm{is}}$ ——试样溶液中酰胺醇类药物及其代谢物内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；  
$C_{s}$ ——标准溶液中酰胺醇类药物及其代谢物浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；  
$C_{\mathrm{is}}^{\prime}$ ——标准溶液中酰胺醇类药物及其代谢物内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$A_{\mathrm{i}}$ ——试样溶液中酰胺醇类药物及其代谢物的峰面积；

$A_{\mathrm{is}}$ ——试样溶液中酰胺醇类药物及其代谢物内标的峰面积；

$A_{s}$ ——标准溶液中酰胺醇类药物及其代谢物的峰面积；

$A_{\mathrm{is}}^{\prime}$ ——标准溶液中酰胺醇类药物及其代谢物内标的峰面积；

$V$ ——溶解残渣的 $20\%$ 甲醇溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法氯霉素的检测限为 $0.1\mu \mathrm{g / kg}$ ，定量限为 $0.2\mu \mathrm{g / kg}$ ；甲砜霉素、氟苯尼考、氟苯尼考胺的检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法氯霉素在 $0.2\mu \mathrm{g} / \mathrm{kg}\sim 1\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平，甲砜霉素在 $1\mu \mathrm{g} / \mathrm{kg}\sim 100\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平，氟苯尼考、氟苯尼考胺在 $1\mu \mathrm{g} / \mathrm{kg}\sim 6000\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率均为 $70\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

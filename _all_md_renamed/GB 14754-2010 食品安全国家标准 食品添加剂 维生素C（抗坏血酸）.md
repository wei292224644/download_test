# 前言

本标准代替GB14754—1993《食品添加剂维生素C（抗坏血酸)》。

本标准与GB14754—1993相比，主要变化如下：

——取消了亚甲蓝溶液鉴别试验；

——增加了红外光谱鉴别；

——维生素C含量测定中碘标准滴定溶液的浓度由 $0.1 \mathrm{~mol} / \mathrm{L}$ 修改为 $0.05 \mathrm{~mol} / \mathrm{L}$ ;

——增加了铅指标及试验方法；

——增加了铁指标及试验方法；

——增加了铜指标及试验方法。

本标准的附录A和附录B为规范性附录。

本标准所代替标准的历次版本发布情况为：

—GB14754—1993。

# 检验方法

# A.1 安全提示

本标准试验方法中使用的部分试剂具有毒性或腐蚀性，按相关规定操作，操作时需小心谨慎。若溅到皮肤上应立即用水冲洗，严重者应立即治疗。在使用挥发性酸时，要在通风橱中进行。

# A.2 一般规定

本标准所用试剂除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682—2008中规定的三级水。

试验方法中所需标准滴定溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603之规定制备。

# A.3 鉴别

A.3.1 试剂和材料

A.3.1.1 活性炭。

A.3.1.2 吡咯。

A.3.1.3 硝酸银溶液： $17.5\mathrm{g / L}$

A.3.1.4 三氯乙酸溶液： $50~\mathrm{g / L}$

A.3.2 鉴别试验

A.3.2.1 硝酸银反应试验

A.3.2.1.1 方法原理

维生素C分子中有二烯醇基，具强还原性，可被硝酸银氧化为去氢抗坏血酸，同时产生黑色银沉淀。

A.3.2.1.2 分析步骤

称取约 $0.1\mathrm{g}$ 实验室样品，精确至 $0.01\mathrm{g}$ ，溶于 $5\mathrm{mL}$ 水中，加 $0.5\mathrm{mL}$ 硝酸银溶液，即生成银的黑色沉淀。

A.3.2.2 糖类性质反应试验

A.3.2.2.1 方法原理

维生素C具有糖类性质的反应，可在三氯乙酸或盐酸存在下水解、脱羧、生成戊糖，再失水，转变为糖醛，加入吡咯，加热至 $50^{\circ}\mathrm{C}$ 即发生蓝色。

A.3.2.2.2 分析步骤

称取约 $0.015 \mathrm{~g}$ 实验室样品, 溶于 $15 \mathrm{~mL}$ 三氯乙酸溶液中, 加 $200 \mathrm{mg}$ 活性炭, 猛烈振摇 $1 \mathrm{~min}$ , 反复过滤至澄清, 在 $5 \mathrm{~mL}$ 滤液中加 $1 \mathrm{~mL}$ 吡咯, 振摇使溶解后, 加热至 $50^{\circ} \mathrm{C}$ 即发生蓝色。

A.3.2.2.3 红外光吸收谱鉴别

采用溴化钾压片法，按照GB/T 6040进行试验，实验室样品的红外光谱应与对照的图谱一致（对

照图谱见附录B）。

# A. 4 维生素C的测定

# A.4.1 方法提要

维生素C具有较强的还原性，可被碘定量氧化。以淀粉为指示剂，用碘标准滴定液滴定样品水溶液，根据碘标准滴定液的用量，计算以 $\mathrm{C_6H_8O_6}$ 计的维生素C含量。

# A.4.2 试剂和材料

A.4.2.1 硫酸溶液： $57\rightarrow 1000$   
A.4.2.2 碘标准滴定溶液： $c(\mathrm{I}_2) = 0.05\mathrm{mol / L}$   
A.4.2.3 淀粉指示液： $10\mathrm{g / L}$

# A.4.3 分析步骤

称取约 $0.2\mathrm{g}$ 实验室样品，精确至 $0.0002\mathrm{g}$ ，置于 $250\mathrm{mL}$ 碘容量瓶中，加 $20\mathrm{mL}$ 无二氧化碳的水及 $25\mathrm{mL}$ 硫酸溶液使溶解，立即用碘标准滴定溶液滴定，近终点时，加 $1\mathrm{mL}$ 淀粉指示液，滴至溶液显蓝色，保持30s不褪色为终点。

同时做空白试验，除不加试样外，其他步骤与样品测定相同。

# A.4.4 结果计算

维生素C（以 $\mathrm{C}_{6} \mathrm{H}_{8} \mathrm{O}_{6}$ 计）的质量分数 $w_{1}$ ，数值以%表示，按式(A.1)计算：

$$
w _ {1} = \frac {\left(V - V _ {0}\right) \times c \times M}{m \times 1000} \times 100 \% \dots \tag{A.1}
$$

式中：

$V$ ——实验室样品消耗碘标准滴定溶液的体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗碘标准滴定溶液的体积的数值，单位为毫升（ $\mathrm{mL}$ ）；

$c$ ——碘标准滴定溶液浓度的准确数值，单位为摩尔每升（ $\mathrm{mol} / \mathrm{L}$ ）；

M——维生素C摩尔质量的数值，单位为克每摩尔（g/mol）（M=176.12）；

$m$ ——实验室样品质量的数值，单位为克（g）；

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.3\%$

# A.5 比旋光度的测定

# A.5.1 分析步骤

称取约 $5 \mathrm{~g}$ 实验室样品，精确至 $0.0002 \mathrm{~g}$ ，置于 $50 \mathrm{~mL}$ 容量瓶中，加水溶解并稀释至刻度，摇匀。其他按 GB/T 613-2007 规定的方法进行。

# A.5.2 结果计算

比旋光度 $a_{m}(20^{\circ}\mathrm{C},D)$ 按式(A.2)计算：

$$
\alpha_ {m} (2 0 ^ {\circ} C, D) = \frac {\alpha}{l \rho_ {\alpha}} \tag {A.2}
$$

式中：

$\alpha$ 测得的旋光角，单位为度 $(^{\circ})$

$l$ ——旋光管的长度,单位为分米(dm);

$\rho_{\alpha}$ —— 溶液中有效组分的质量浓度,单位为克每毫升 $(\mathrm{g} / \mathrm{mL})$ 。

# A.6 灼烧残渣的测定

# A.6.1 试剂和材料

硫酸。

# A.6.2 分析步骤

称取约 $1 \mathrm{~g}$ 实验室样品，精确至 $0.0002 \mathrm{~g}$ ，置于已在 $700^{\circ} \mathrm{C} \sim 800^{\circ} \mathrm{C}$ 灼烧至恒重的坩埚中，先用小火缓缓加热至完全碳化，冷却至室温后，加硫酸 $0.5 \mathrm{~mL} \sim 1 \mathrm{~mL}$ 使其湿润，低温加热至硫酸蒸汽除尽后移入高温炉中，在 $700^{\circ} \mathrm{C} \sim 800^{\circ} \mathrm{C}$ 中灼烧至恒重。

# A.6.3 结果计算

灼烧残渣的质量分数 $w_{2}$ , 数值以 $\%$ 表示, 按式(A.3)计算

$$
w _ {2} = \frac {m _ {1} - m _ {2}}{m} \times 100 \% \dots \tag{A.3}
$$

式中：

$m_{1}$ ——残渣和空坩埚质量的数值，单位为克（g）；

$m_{2}$ ——空坩埚质量的数值，单位为克（g）；

$m$ —实验室样品质量的数值，单位为克（g）。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.02\%$

# A.7 砷的测定

称取 $1.0 \mathrm{~g} \pm 0.01 \mathrm{~g}$ 实验室样品，加 $10 \mathrm{~mL}$ 水溶解作为试样液；量取 $3 \mathrm{~mL} \pm 0.025 \mathrm{~mL}$ （含砷 $3.0 \mu \mathrm{g}$ ）砷（As）标准溶液制备限量标准。其他按GB/T 5009.76—2003砷斑法的规定进行。

# A.8 重金属的测定

# A.8.1 试剂和材料

A.8.1.1 硝酸。

A.8.1.2 甘油。

A.8.1.3 乙酸铵。

A.8.1.4 硝酸铅。

A.8.1.5 硫代乙酰胺。

A.8.1.6 盐酸溶液： $c$ （ $\mathrm{HCl}) = 2\mathrm{mol} / \mathrm{L}$

A.8.1.7 氨水溶液： $c(\mathrm{NH}_3\bullet \mathrm{H}_2\mathrm{O}) = 5\mathrm{mol} / \mathrm{L}$

A.8.1.8 氢氧化钠溶液： $c$ （ $\mathrm{NaOH}) = 1\mathrm{mol} / \mathrm{L}$

A.8.1.9 盐酸溶液： $c$ （HCl） $= 7\mathrm{mol / L}$

A.8.1.10 乙酸盐缓冲液（pH3.5）：

称取 $25 \mathrm{~g}$ 乙酸铵, 精确至 $0.01 \mathrm{~g}$ , 加 $25 \mathrm{~mL}$ 水溶解后, 加 $38 \mathrm{~mL} 7 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液, 用 $2 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液或 $5 \mathrm{~mol} / \mathrm{L}$ 氨水溶液准确调节 $\mathrm{pH}$ 至 $3.5$ (pH 计), 用水稀释至 $100 \mathrm{~mL}$ 。

# A.8.1.11 铅标准溶液：

称取 $0.160 \mathrm{~g}$ 硝酸铅，精确至 $0.0002 \mathrm{~g}$ ，置于 $1000 \mathrm{~mL}$ 容量瓶中，加 $5 \mathrm{~mL}$ 硝酸与 $50 \mathrm{~mL}$ 水溶解后，用水稀释至刻度，摇匀，作为贮备液。临用前，移取 $10 \mathrm{~mL} \pm 0.02 \mathrm{~mL}$ 贮备液，置于 $100 \mathrm{~mL}$ 容量瓶中，加水稀释至刻度，摇匀，即得（每 $1 \mathrm{~mL}$ 相当于 $10 \mu \mathrm{g}$ 的 $\mathrm{Pb}$ ）。配置与贮存用的玻璃仪器均不得含铅。

# A.8.1.12 硫代乙酰胺试液：

称取 $4\mathrm{~g}$ 硫代乙酰胺，精确至 $0.01\mathrm{~g}$ ，加水使溶解成 $100\mathrm{~mL}$ ，置冰箱中保存。临用前取 $5.0\mathrm{~mL}$ 混合液（由 $1\mathrm{~mol/L} 15\mathrm{~mL}$ 氢氧化钠溶液、 $5.0\mathrm{~mL}$ 水及 $20\mathrm{~mL}$ 甘油组成），加上述 $1.0\mathrm{~mL}$ 硫代乙酰胺溶液，置水浴上加热 $20\mathrm{~s}$ ，冷却，立即使用。

# A.8.2 分析步骤

按《中华人民共和国药典》2005年版二部附录VIII H 重金属检查法第一法进行。具体方法如下：

取 $25 \mathrm{~mL}$ 纳氏比色管两支，甲管中加入 $1 \mathrm{~mL} \pm 0.01 \mathrm{~mL}$ （含铅 $10.0 \mu \mathrm{g}$ ）铅（Pb）标准溶液与 $2 \mathrm{~mL}$ 乙酸盐缓冲液后，加水稀释成 $25 \mathrm{~mL}$ ；另称取 $1 \mathrm{~g} \pm 0.01 \mathrm{~g}$ 实验室样品，置于纳氏比色管乙管中，加 $15 \mathrm{~mL}$ 水溶解，加 $2 \mathrm{~mL}$ 乙酸盐缓冲液（ $\mathrm{pH} 3.5$ ），用水稀释成 $25 \mathrm{~mL}$ ，若该溶液带颜色，可在甲管中滴加少量的稀焦糖溶液或其他无干扰的有色溶液，使之与乙管一致；再在甲乙两管中分别加硫代乙酰胺试液各 $2 \mathrm{~mL}$ ，摇匀，放置 $2 \mathrm{~min}$ ，同置白纸上，自上向下透视，乙管中显出的颜色与甲管比较，不得更深。

# 前言

本标准代替GB/T5009.208—2008《食品中生物胺含量的测定》、GB/T20768—2006《鱼和虾中有毒生物胺的测定液相色谱-紫外检测法》、SN/T2209—2008《进出口水产品中有毒生物胺的检测方法高效液相色谱法》，及GB/T5009.45—2003《水产品卫生标准的分析方法》中组胺检测部分。

本标准与GB/T5009.208—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中生物胺的测定”；  
——增加了分光光度法；  
——增加了章鱼胺；  
——修改了酒类和醋酱油的测定；  
——修改了流动相和洗脱梯度；  
——修改了样品前处理方法；  
——适用范围删除了乳制品。

# 食品安全国家标准

# 食品中生物胺的测定

# 第一法 液相色谱法

# 1 范围

本标准规定了食品中色胺、β-苯乙胺、腐胺、尸胺、组胺、章鱼胺、酪胺、亚精胺和精胺含量的测定方法。

本标准适用于酒类（葡萄酒、啤酒、黄酒等）、调味品（醋和酱油）、水产品（鱼类及其制品、虾类及其制品）、肉类中生物胺的测定。

# 2 原理

水产品（鱼类及其制品、虾类及其制品）、肉类：试样用 $5\%$ 三氯乙酸提取，正已烷去除脂肪，三氯甲烷-正丁醇 $(1 + 1)$ 液液萃取净化后，丹磺酰氯衍生， $\mathrm{C_{18}}$ 色谱柱分离，高效液相色谱-紫外检测器检测，内标法定量。

酒类（葡萄酒、啤酒、黄酒等）、调味品（醋和酱油）：试样用丹磺酰氯衍生， $\mathrm{C_{18}}$ 色谱柱分离，高效液相色谱-紫外检测器检测，内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.4 标准溶液配制

注：标准溶液的配制浓度以各生物胺单体计算,称取标准品时应对其中的盐酸盐进行折算。所有标准溶液配制好

后均需转移至密闭棕色容器中，避光贮存。

3.4.1生物胺标准储备溶液的配制：准确称取各种生物胺标准品适量，分别置于 $10~\mathrm{mL}$ 小烧杯中，用 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液溶解后转移至 $10~\mathrm{mL}$ 容量瓶中，定容至刻度，混匀，配制成浓度为 $1000\mathrm{mg / L}$ （以各种生物胺单体计)的标准储备溶液，置一 $20^{\circ}C$ 冰箱储存。保存期为6个月。  
3.4.2 生物胺标准混合使用液的配制：分别吸取 $1.0\mathrm{mL}$ 各生物胺单组分标准储备溶液，置于同一个 $10~\mathrm{mL}$ 容量瓶中，用 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸稀释至刻度，混匀，配制成生物胺标准混合使用液（ $100\mathrm{mg / L}$ ）。保存期为3个月。  
3.4.3生物胺标准系列溶液的配制：分别吸取 $0.10\mathrm{mL},0.25\mathrm{mL},0.50\mathrm{mL},1.0\mathrm{mL},1.50\mathrm{mL},2.50\mathrm{mL}$ 、 $5.0~\mathrm{mL}$ 生物胺标准混合使用液 $(100\mathrm{mg / L})$ ，置于 $10~\mathrm{mL}$ 容量瓶中，用 $0.1\mathrm{mol / L}$ 盐酸溶液稀释至刻度，混匀，使浓度分别为 $1.0\mathrm{mg / L},2.5\mathrm{mg / L},5.0\mathrm{mg / L},10.0\mathrm{mg / L},15.0\mathrm{mg / L},25.0\mathrm{mg / L},50.0\mathrm{mg / L}$ ，临用现配。  
3.4.4 内标标准储备溶液的配制：准确称取内标标准品适量，置于 $10\mathrm{mL}$ 容量瓶中，用 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液溶解后稀释至刻度，混匀，配制成浓度为 $10\mathrm{mg / mL}$ 的内标标准储备溶液，置一 $20^{\circ}\mathrm{C}$ 冰箱储存。保存期为6个月。  
3.4.5 内标标准中间使用液的配制：吸取 $1.0\mathrm{mL}$ 内标标准储备溶液于 $10~\mathrm{mL}$ 容量瓶中，用 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸稀释至刻度，混匀，作为内标使用液（ $1.0\mathrm{mg / mL}$ ），存期为3个月。  
3.4.6 内标标准使用液的配制：吸取 $1.0\mathrm{mL}$ 内标标准中间使用液于 $10~\mathrm{mL}$ 容量瓶中，用 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸稀释至刻度，混匀，作为内标使用液（ $100\mathrm{mg / L}$ ），临用现配。

# 5 分析步骤

# 5.1 水产品和肉类

# 5.1.1 试样制备

取水产品及肉类样品的可食部分约 $500\mathrm{g}$ ，充分匀质，均分成两份装入洁净容器中，密封，于 $-20^{\circ}\mathrm{C}$ 保存。

# 5.1.2 提取

准确称取已经绞碎或匀浆后的水产品、肉类 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），置于 $100~\mathrm{mL}$ 具塞锥形瓶中，加入 $500\mu \mathrm{L}$ 内标使用液（ $1.0\mathrm{mg / mL}$ )与样品充分混匀，加入 $20~\mathrm{mL}~5\%$ 三氯乙酸溶液和振荡提取 $30\mathrm{min}$ 转移至 $50~\mathrm{mL}$ 具塞离心管中， $5000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，转移上清液至 $50~\mathrm{mL}$ 容量瓶中，残渣用 $20~\mathrm{mL}$

$5\%$ 三氯乙酸溶液再提取一次，合并上清液，用 $5\%$ 三氯乙酸稀释至刻度，待净化。

# 5.1.3 净化

5.1.3.1 除脂肪：移取上述试样提取液 $10\mathrm{mL}$ 于 $25\mathrm{mL}$ 具塞试管中，加入 $0.5\mathrm{g}$ 氯化钠涡旋振荡至氯化钠完全溶解后加入 $10\mathrm{mL}$ 正己烷，涡旋振荡 $5\mathrm{min}$ ，静置分层后弃去上层有机相，下层试样溶液加入 $10\mathrm{mL}$ 正己烷再除脂一次。

5.1.3.2 萃取：移取 $5\mathrm{mL}$ 上述除脂肪后的试样溶液于 $10~\mathrm{mL}$ 具塞离心管中，用 $5\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液（几滴）调节 $\mathrm{pH}$ 至12.0左右。加入 $5\mathrm{mL}$ 的正丁醇/三氯甲烷 $(1 + 1)$ 混合溶液，涡旋振荡 $5\mathrm{min},5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取出，静置分层后，转移上层水相于另一个 $10~\mathrm{mL}$ 具塞离心管中，再萃取一次，合并萃取液，用正丁醇/三氯甲烷 $(1 + 1)$ 稀释至刻度。取 $5\mathrm{mL}$ 萃取液加入 $200\mu \mathrm{L}$ 盐酸 $(1\mathrm{mol} / \mathrm{L})$ ，混匀后 $40^{\circ}\mathrm{C}$ 水浴下氮气吹干，加入 $1\mathrm{mL}$ 盐酸 $(0.1\mathrm{mol} / \mathrm{L})$ 涡旋振荡，使残留物完全溶解，待衍生。

# 5.1.4 衍生

5.1.4.1 试样的衍生：在上述待衍生的试样溶液中依次加入 $1\mathrm{mL}$ 饱和碳酸氢钠溶液、 $100\mu \mathrm{L}$ 氢氧化钠溶液（ $1\mathrm{mol} / \mathrm{L}$ ）、 $1\mathrm{mL}$ 衍生试剂，涡旋混匀 $1\mathrm{min}$ 后置于 $60^{\circ}\mathrm{C}$ 恒温水浴中衍生 $15\mathrm{min}$ ，取出，分加入 $100\mu \mathrm{L}$ 谷氨酸钠溶液，振荡混匀， $60^{\circ}\mathrm{C}$ 恒温反应 $15\mathrm{min}$ 。取出，冷却至室温，于每个离心管中加入 $1\mathrm{mL}$ 水，涡旋混合 $1\mathrm{min}$ ， $40^{\circ}\mathrm{C}$ 水浴下氮吹除去丙酮(约 $1\mathrm{mL}$ )，加入 $0.5\mathrm{g}$ 氯化钠涡旋振荡至氯化钠完全溶解后加入 $5\mathrm{mL}$ 乙醚，涡旋振荡 $2\mathrm{min}$ ，静置分层后，吸出上层有机相(乙醚层)，再萃取一次，合并乙醚萃取液， $40^{\circ}\mathrm{C}$ 水浴下氮气吹干。加入 $1\mathrm{mL}$ 乙腈涡旋振荡使残留物完全溶解， $0.22\mu \mathrm{m}$ 滤膜针头滤器过滤于进样小瓶，待测定。

5.1.4.2 标准的衍生：分别移取 $1\mathrm{mL}$ 生物胺标准系列溶液，置于 $10~\mathrm{mL}$ 具塞试管中，依次加入 $250~\mu \mathrm{L}$ 内标使用液（ $100\mathrm{mg / L}$ ），以下操作同试样的衍生步骤。

# 5.2 酒类及调味品（醋和酱油）

5.2.1 试样的衍生：准确量取 $1.0\mathrm{mL}$ 样品于 $15\mathrm{mL}$ 塑料离心管中，依次加入 $250~\mu \mathrm{L}$ 内标溶液（ $100\mathrm{mg / L}$ ）、 $1\mathrm{mL}$ 饱和碳酸氢钠溶液、 $100~\mu \mathrm{L}$ 氢氧化钠溶液（ $1\mathrm{mol / L}$ ）、 $1\mathrm{mL}$ 衍生试剂，涡旋混匀 $1\mathrm{min}$ 后置于 $60^{\circ}\mathrm{C}$ 恒温水浴锅中衍生 $15\mathrm{min}$ ，取出，分别加入 $100~\mu \mathrm{L}$ 谷氨酸钠溶液，振荡混匀， $60^{\circ}\mathrm{C}$ 恒温反应 $15\mathrm{min}$ ，取出冷却至室温，于每个离心管中加入 $1\mathrm{mL}$ 超纯水，涡旋混合 $1\mathrm{min}$ ， $40^{\circ}\mathrm{C}$ 水浴下氮吹除去丙酮（约 $1\mathrm{mL}$ ），加入 $0.5\mathrm{g}$ 氯化钠涡旋振荡至完全溶解，再加入 $5\mathrm{mL}$ 乙醚，涡旋振荡 $2\mathrm{min}$ ，静置分层后，转移上层有机相（乙醚层）于 $15\mathrm{mL}$ 离心管中，水相（下层）再萃取一次，合并两次乙醚萃取液， $40^{\circ}\mathrm{C}$ 水浴下氮气吹干。加入 $1\mathrm{mL}$ 乙腈振荡混匀，使残留物溶解， $0.22~\mu \mathrm{m}$ 滤膜针头滤器过滤，待测定。

5.2.2 标准的衍生：同5.1.4.2。

# 5.4 标准曲线的制作

将 $20~\mu \mathrm{L}$ 系列混合标准工作液的衍生液分别注入高效液相色谱仪，测得目标化合物的峰面积，以系列混合标准工作液的浓度为横坐标，以目标化合物的峰面积与内标的峰面积的比值为纵坐标，绘制标准曲线。标准溶液的色谱图见图A.1。

# 5.5 试样溶液的测定

将试样的衍生溶液注入高效液相色谱仪中，测得峰面积，以保留时间定性。根据标准曲线得到待测液中各目标化合物的浓度。

# 6 分析结果的表述

试样中生物胺含量按式(1)计算：

$$
X = \frac {c \times V \times f}{m} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中被测组分的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$c$ ——试样溶液中被测组分的浓度，单位为微克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释倍数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 分光光度法

# 9 范围

本标准规定了水产品中组胺含量的测定方法。

本标准适用于水产品(鱼类及其制品、虾类及其制品)中组胺的测定。

# 10 原理

以三氯乙酸为提取溶液，振摇提取，经正戊醇萃取净化，组胺与偶氮试剂发生显色反应后，分光光度计检测，外标法定量。

# 11 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 11.1 标准及试剂

11.1.1 磷酸组胺 $(\mathrm{C}_5\mathrm{H}_9\mathrm{N}_3\cdot 2\mathrm{H}_3\mathrm{PO}_4)$   
11.1.2 正戊醇 $(\mathrm{C}_5\mathrm{H}_{12}\mathrm{O})$ 。  
11.1.3 三氯乙酸 $(\mathrm{C}_2\mathrm{HCl}_3\mathrm{O}_2)$ 。  
11.1.4 碳酸钠 $\mathrm{(Na_2CO_3)}$   
11.1.5 氢氧化钠(NaOH)。  
11.1.6 盐酸（ $\mathrm{HCl}$ ， $37\%$ ）。  
11.1.7 对硝基苯胺 $(\mathrm{C_6H_6N_2O_2})$ 。  
11.1.8 亚硝酸钠 $\mathrm{(NaNO_2)}$ 。

# 11.2 试剂配制

11.2.1 组胺标准储备液：在 $100^{\circ}\mathrm{C}(\pm 5^{\circ}\mathrm{C})$ 下将磷酸组胺标准干燥 $2\mathrm{h}$ 后，称取 $0.2767\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 于 $50~\mathrm{mL}$ 烧杯中，用适量水完全溶解后转移至 $100~\mathrm{mL}$ 容量瓶中，定容至刻度。此溶液每毫升相当于 $1.0\mathrm{mg}$ 组胺。置一 $20^{\circ}\mathrm{C}$ 冰箱储存。保存期为6个月。  
11.2.2 磷酸组胺标准使用液：吸取 $1.0\mathrm{mL}$ 组胺标准溶液于 $50~\mathrm{mL}$ 容量瓶中，用水定容至刻度。此溶液每毫升相当于 $20.0\mu \mathrm{g}$ 组胺。临用现配。  
11.2.3 $100\mathrm{g / L}$ 三氯乙酸溶液：称取 $50\mathrm{g}$ 三氯乙酸于 $250~\mathrm{mL}$ 烧杯中，用适量水完全溶解后转移至 $500~\mathrm{mL}$ 容量瓶中，定容至刻度。保存期为6个月。  
11.2.4 $50\mathrm{g / L}$ 碳酸钠溶液：称取 $5\mathrm{g}$ 碳酸钠于 $100~\mathrm{mL}$ 烧杯中，用适量水完全溶解后转移至 $100~\mathrm{mL}$ 容量瓶中，定容至刻度。保存期为6个月。  
11.2.5 $250\mathrm{g / L}$ 氢氧化钠溶液：称取 $25\mathrm{g}$ 氢氧化钠于 $100~\mathrm{mL}$ 烧杯中，用适量水完全溶解后转移至 $100~\mathrm{mL}$ 容量瓶中，定容至刻度。保存期为3个月。  
11.2.6 (1+11)盐酸溶液：吸取 $5\mathrm{mL}$ 盐酸于 $100~\mathrm{mL}$ 烧杯中，加水 $55\mathrm{mL}$ ，混匀。保存期为6个月。  
11.2.7 偶氮试剂：

甲液（对硝基苯胺）：称取 $0.5\mathrm{g}$ 对硝基苯胺，加 $5\mathrm{mL}$ 盐酸溶液溶解后，再加水稀释至 $200\mathrm{mL}$ ，置冰箱中，临用现配。

乙液（亚硝酸钠溶液）：称取 $0.5\mathrm{g}$ 亚硝酸钠，加入 $100~\mathrm{mL}$ 水溶解混匀，临用现配。

吸取 $5\mathrm{mL}$ 甲液， $40~\mathrm{mL}$ 乙液混合即为偶氮试剂，临用现配。

# 12 分析步骤

# 12.1 试样制备

取鲜活水产品的可食部分约 $500\mathrm{g}$ 代表性样品，用组织捣碎机充分捣碎，均分成两份分别装入洁净容器中，密封，并标明标记。 $-20^{\circ}C$ 保存。

# 12.2 试样的分析

# 12.2.1 试样提取

准确称取已经绞碎均匀的试样 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，置于 $100~\mathrm{mL}$ 具塞锥形瓶中，加入 $20~\mathrm{mL}~10\%$ 三氯乙酸溶液浸泡 $2\mathrm{h}\sim 3\mathrm{h}$ ，振荡 $2\mathrm{min}$ 混匀，滤纸过滤，准确吸取 $2.0~\mathrm{mL}$ 滤液于分液漏斗中，逐滴加入氢氧化钠溶液调节 $\mathrm{pH}$ 在 $10\sim 12$ 之间，加入 $3\mathrm{mL}$ 正戊醇振摇提取 $5\mathrm{min}$ ，静置分层，将正戊醇提取液(上层)转移至 $10~\mathrm{mL}$ 刻度试管中。正戊醇提取三次，合并提取液，并用正戊醇稀释至刻度。吸取 $2.0~\mathrm{mL}$ 正戊醇提取液于分液漏斗中，加入 $3\mathrm{mL}$ 盐酸溶液振摇提取，静置分层，将盐酸提取液(下层)转移至 $10~\mathrm{mL}$ 刻度试管中。提取三次，合并提取液，并用盐酸溶液稀释至刻度。

# 12.2.2 测定

分别吸取 $0,0.20,0.40,0.60,0.80,1.0\mathrm{mL}$ 组胺标准使用液（相当于 $0,4.0,8.0,12,16,20\mu \mathrm{g}$ 组胺）及 $2.0\mathrm{mL}$ 试样提取液于 $10\mathrm{mL}$ 比色管中，加水至 $1\mathrm{mL}$ ，再加入 $1\mathrm{mL}$ 盐酸溶液，混匀。加入 $3\mathrm{mL}$ 碳酸钠溶液， $3\mathrm{mL}$ 偶氮试剂。加水至刻度，混匀，放置 $10\mathrm{min}$ 。将“0”管溶液转移至 $1\mathrm{cm}$ 比色皿，分光光度计波长调至 $480\mathrm{nm}$ ，调节吸光度为“0”后，依次测试系列标准溶液及试样溶液吸光度，以吸光度A为纵轴，组胺的质量为横轴绘制标准曲线。

# 13 结果计算

试样中组胺的含量按式(2)计算：

$$
X = \frac {m _ {1} V _ {1} \times 1 0 \times 1 0}{m _ {2} \times 2 \times 2 \times 2} \times \frac {1 0 0}{1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中组胺的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$m_{1}$ ——试样中组胺的吸光度值对应的组胺质量，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——加入三氯乙酸溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

10——第一个是正戊醇提取液的体积，单位为毫升 $(\mathrm{mL})$ ；第二个是盐酸提取液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m_{2}$ ——取样量，单位为克 $(\mathrm{g})$ ；

2 第一个是三氯乙酸提取液的体积，单位为毫升 $(\mathrm{mL})$ ；第二个是正戊醇提取液的体积，单位为毫升 $(\mathrm{mL})$ ；第三个是盐酸提取液的体积，单位为毫升 $(\mathrm{mL})$ ；

100 ——换算系数；

1000——换算系数。

计算结果保留小数点后一位。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

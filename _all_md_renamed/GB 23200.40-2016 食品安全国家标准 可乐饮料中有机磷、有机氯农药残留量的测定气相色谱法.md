本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T1984—2007《进出口可乐饮料中有机磷、有机氯农药残留量检测方法气相色谱法》。与SN/T1984—2007相比，主要变化如下：

——标准文本格式修改为食品安全国家标准文本格式；

# 食品安全国家标准

# 可乐饮料中有机磷、有机氯农药残留量的测定 气相色谱法

# 1 范围

本标准规定了可乐饮料中11种有机磷、有机氯农药残留量的气相色谱测定方法。

本标准适用于可乐饮料中敌敌畏、毒死蜱、马拉硫磷、对硫磷、七氯、六氯苯、六六六及其异构体（ $\alpha-$ 六六六、 $\beta-$ 六六六、 $\gamma-$ 六六六、 $\delta-$ 六六六）、五氯硝基苯11种有机磷、有机氯农药残留量的检测。其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

3.1 试样中有机磷残留经乙酸乙酯萃取, 旋转蒸发浓缩, 固相萃取柱净化, 使用气相色谱火焰光度检测器测定, 外标法定量。  
3.2 试样中有机氯残留经乙酸乙酯萃取,旋转蒸发浓缩,磺化净化,使用气相色谱电子俘获检测器测定,外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
4.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
4.1.3 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：优级纯。  
4.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置于干燥器中备用。  
4.1.5 氯化钠(NaCl)。  
4.1.6 氢氧化钠(NaOH)。  
4.1.7 浓硫酸 $\mathrm{(H_2SO_4)}$ ：优级纯。

# 4.2 溶液配制

4.2.1 甲醇—水溶液 $(5 + 95$ ，体积比)：量取 $5\mathrm{mL}$ 甲醇与 $95~\mathrm{mL}$ 水混合。  
4.2.2 甲醇—水溶液 $(10 + 90$ ，体积比)：量取 $10\mathrm{mL}$ 甲醇与 $90~\mathrm{mL}$ 水混合。  
4.2.3 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称 $20\mathrm{g}$ 氢氧化钠，用水溶解并定容至 $500~\mathrm{mL}$ 。

# 4.3 标准品

农药标准物质：参见附录A。

# 4.4 标准溶液配制

# 4.4.1 有机磷农药标准溶液

4.4.1.1 有机磷农药标准储备溶液: 准确称取适量的敌敌畏、毒死蜱、马拉硫磷、对硫磷, 分别用丙酮溶解并定容至棕色容量瓶中, 浓度相当于 $1000 \mathrm{mg} / \mathrm{L}$ , 储备液于 $-18^{\circ} \mathrm{C}$ 以下保存。  
4.4.1.2 有机磷混合标准中间溶液: 准确吸取适量敌敌畏、毒死蜱、马拉硫磷、对硫磷标准储备溶液于棕色容量瓶中, 用丙酮定容至刻度, 各种有机磷农药标准溶液的浓度为 $10 \mathrm{mg} / \mathrm{L}$ , 此中间溶液于 $-18^{\circ} \mathrm{C}$ 以下保存。  
4.4.1.3 有机磷混合标准工作溶液：用乙酸乙酯将混合标准中间溶液按需要逐级稀释，配制为4种有机磷农药混合标准工作溶液，混合标准工作溶液在 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存。

# 4.4.2 有机氯农药标准溶液

4.4.2.1 有机氯农药混合标准中间溶液: 移取适量体积的七氯、六氯苯、六六六混合标准溶液、五氯硝基苯标准溶液于棕色容量瓶中, 用正己烷定容至刻度, 各种有机氯农药标准溶液的浓度为 $10 \mathrm{mg} / \mathrm{L}$ , 此中间溶液于 $-18^{\circ} \mathrm{C}$ 以下保存。  
4.4.2.2 有机氯混合标准工作溶液：用正己烷将混合标准中间溶液按需要逐级稀释，配制为有机氯农药混合标准工作溶液，混合标准工作溶液于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 保存。

# 4.5 材料

HLB固相萃取小柱： $3\mathrm{mL} / 60\mathrm{mg}$ ，或相当者。

# 5 仪器和设备

5.1气相色谱仪，配电子俘获检测器(ECD)和火焰光度检测器(FPD)磷滤光片，波长 $525\mathrm{nm}$   
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.3 离心机： $5000\mathrm{r / min}$   
5.4 吹氮浓缩仪。  
5.5 固相萃取装置？  
5.6 旋转蒸发装置。  
5.7 分液漏斗： $500~\mathrm{mL}$ 。  
5.8 磨口玻璃圆底烧瓶： $500~\mathrm{mL}$   
5.9 玻璃砂芯漏斗。  
5.10 聚丙烯具塞离心管： $15~\mathrm{mL}$   
5.11 涡旋振荡器。

# 6 分析步骤

# 6.1 提取

取可乐样品于烧杯中, 放置 $60 \mathrm{~min}$ , 并用玻璃棒搅拌排气。准确称取 $150 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ ) 可乐样品, 加入 $1 \mathrm{~mol} / \mathrm{L}$ 的氢氧化钠溶液调节溶液 $\mathrm{pH}$ 至 7 左右。将调节至中性的可乐样品转移至 $500 \mathrm{~mL}$ 分液漏斗中, 加入 $15 \mathrm{~g}$ 氯化钠和 $100 \mathrm{~mL}$ 乙酸乙酯, 剧烈振荡 $2 \mathrm{~min}$ 并不时排气, 静置 $10 \mathrm{~min}$ 后, 取上层有机相, 过预先填充的无水硫酸钠柱 (在玻璃砂芯漏斗中装入 $15 \mathrm{~g}$ 左右无水硫酸钠, 并用 $20 \mathrm{~mL}$ 乙酸乙酯淋洗), 收集于 $500 \mathrm{~mL}$ 圆底烧瓶中。在分液漏斗中分 2 次加入 $200 \mathrm{~mL}$ 乙酸乙酯, 每次 $100 \mathrm{~mL}$ , 重复以上提取步骤, 合并提取液于 $40^{\circ} \mathrm{C}$ 水浴下旋转蒸发至 $3 \mathrm{~mL} \sim 4 \mathrm{~mL}$ , 转移至 $15 \mathrm{~mL}$ 离心管中, 用 $9 \mathrm{~mL}$ 正己烷分 3 次洗涤圆底烧瓶, 合并洗涤液于 $15 \mathrm{~mL}$ 离心管中, $40^{\circ} \mathrm{C}$ 水浴下吹氮至近干。

# 6.2 净化

# 6.2.1 有机磷农药测定样品溶液净化

HLB固相萃取柱使用前分别用乙酸乙酯 $3\mathrm{mL}$ 淋洗一次， $3\mathrm{mL}$ 甲醇和 $3\mathrm{mL}$ 水预处理2次，保持柱体湿润。将6.1中提取液用 $5\mathrm{mL}10\%$ 甲醇水溶液溶解，以1滴/s速率过HLB固相萃取柱，弃去流出液，再用 $10\mathrm{mL}5\%$ 甲醇水溶液润洗离心管并上HLB柱，弃去从HLB柱流出的润洗液，真空排干小柱 $10\mathrm{min}$ 。用 $9\mathrm{mL}$ 乙酸乙酯进行洗脱，收集洗脱液于 $15\mathrm{mL}$ 离心管中，加入 $3\mathrm{g}$ 无水硫酸钠，振荡 $3\mathrm{min}$ $5000\mathrm{r / min}$ 条件下离心 $2\mathrm{min}$ ，将溶液转移至另一离心管中，于 $40^{\circ}\mathrm{C}$ 水浴中吹氮浓缩至近干，用乙酸乙酯定容至 $1.0\mathrm{mL}$ ，供气相色谱FPD测定。

# 6.2.2 有机氯农药测定样品溶液净化

将6.1中提取液用 $1\mathrm{mL}$ 正己烷溶解，加入 $0.5\mathrm{mL}$ 浓硫酸，手动轻摇混匀， $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 后，取上层有机相进气相色谱ECD测定，如净化效果不充分，可再加入浓硫酸净化一次。

# 6.3 测定

# 6.3.1 气相色谱电子俘获检测仪器参考条件

a）色谱柱：DB-5石英毛细管柱： $30\mathrm{m}\times 0.32\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度： $80^{\circ}\mathrm{C}$ （保持 $1\mathrm{min})\xrightarrow{30^{\circ}\mathrm{C / min}}180^{\circ}\mathrm{C}\xrightarrow{3^{\circ}\mathrm{C / min}}205^{\circ}\mathrm{C}$ （保持 $4\mathrm{min})\xrightarrow{2^{\circ}\mathrm{C / min}}210^{\circ}\mathrm{C}$ （保持 $1\mathrm{min})$ ： $280^{\circ}\mathrm{C}$ （保持 $1\mathrm{min})$   
c) 进样口温度: $250^{\circ} \mathrm{C}$ ;   
d) 检测器温度: $300^{\circ} \mathrm{C}$ ;   
e）载气：氮气，纯度 $99.999\%$ ，恒压 $0.135\mathrm{MPa}$   
f）进样量： $1\mu \mathrm{L}$   
g）进样方式：分流进样，分流比 $12:1$

# 6.3.2 气相色谱火焰光度检测仪器参考条件

a）色谱柱：DB-1701石英毛细管柱： $30\mathrm{m}\times 0.32\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度： $90^{\circ}\mathrm{C}$ （保持 $1\mathrm{min})\xrightarrow{30^{\circ}\mathrm{C / min}}210^{\circ}\mathrm{C}$ （保持 $8\mathrm{min})\xrightarrow{10^{\circ}\mathrm{C / min}}240^{\circ}\mathrm{C}:280^{\circ}\mathrm{C}$ （保持 $5\mathrm{min})$   
c) 进样口温度: $250^{\circ} \mathrm{C}$ ;   
d) 检测器温度: $250^{\circ} \mathrm{C}$ ;   
e) 载气：氮气，纯度 $99.999\%$ ，恒压0.071MPa；  
f）进样量： $1\mu \mathrm{L}$   
g）进样方式：不分流进样， $1.0\mathrm{min}$ 后打开分流阀，吹扫流量 $50~\mathrm{mL / min}$

# 6.3.3 色谱测定与确证

待测溶液分别进气相色谱电子俘获检测和气相色谱火焰光度检测, 根据样液中被测有机磷或有机氯残留的含量情况, 选定峰面积相近的标准工作溶液。标准工作溶液和样液中有机磷或有机氯残留的响应值均应在仪器的检测线性范围内。对混合标准工作溶液和样液等体积分组分参差进样测定, 外标法定量。在 6.3.1 与 6.3.2 色谱条件下, 有机磷与有机氯农药的保留时间参见附录 A。 $10 \mu \mathrm{g} / \mathrm{L}$ 有机磷和有机氯混合标准工作溶液色谱图参见图 B.1、图 B.2。

# 6.4 空白实验

除不加试样外，均按 $6.1\sim 6.3$ 的规定执行。

# 7 结果计算和表述

按式(1)计算试样中有机磷或有机氯农药化合物的含量。

$$
X = \frac {A \times C _ {s} \times V}{A _ {s} \times m}. \tag {1}
$$

式中：

$X$ ——样品中有机磷或有机氯农药化合物含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A$ ——样液中有机磷或有机氯农药化合物峰面积；

$A_{s}$ ——标准工作溶液中有机磷或有机氯农药化合物峰面积；

$C_s$ ——标准工作溶液中有机磷或有机氯农药化合物浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$m$ ——称取的试样量，单位为克(g)；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 8 精密度

8.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录C的要求。  
8.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 D 的要求。

# 9 定量限和回收率

# 9.1 定量限

本方法有机磷、有机氯农药定量限均为 $0.0001\mathrm{mg / kg}$

# 9.2 回收率

当添加水平为 ${0.0001}\mathrm{{mg}}/\mathrm{{kg}}$ 、 ${0.0002}\mathrm{{mg}}/\mathrm{{kg}}$ 、 ${0.0005}\mathrm{{mg}}/\mathrm{{kg}}$ 时,有机磷、有机氯农药化合物的添加回收率参见附录 ${\mathrm{E}}_{\mathrm{a}}$

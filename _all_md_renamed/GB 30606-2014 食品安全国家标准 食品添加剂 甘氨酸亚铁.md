# 食品添加剂 甘氨酸亚铁

# 1 范围

本标准适用于由甘氨酸和还原铁粉反应，经喷雾干燥制得的食品添加剂甘氨酸亚铁。

# 2 分子式、结构式和相对分子质量

# 2.1 分子式

$$
\mathrm {F e} \left(\mathrm {O O C C H} _ {2} \mathrm {N H} _ {2}\right) _ {2} \cdot 2 \mathrm {H} _ {2} \mathrm {O} 。
$$

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。

试验中所用标准溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 盐酸溶液： $1 + 3$   
A.2.1.2 铁氰化钾溶液： $100\mathrm{g / L}$ ，临用前配制。  
A.2.1.3 氢氧化钠溶液： $43\mathrm{g / L}$   
A.2.1.4 亚硝酸钠溶液： $100\mathrm{g / L}$

# A.2.2 鉴别方法

# A.2.2.1 二价铁的鉴别

称取约 $0.1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $10\mathrm{mL}$ 水中，向此溶液中滴加铁氰化钾溶液，即产生深蓝色的沉淀。该沉淀不溶于盐酸溶液中，但溶于氢氧化钠溶液中。另称取约 $0.1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $20\mathrm{mL}$ 水中，滴加氢氧化钠溶液即产生青白色沉淀，摇动后颜色变为青绿色，然后变为棕褐色。

# A.2.2.2 氨基的鉴别

称取约 $0.1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $10\mathrm{mL}$ 水中，加入5滴盐酸溶液和新配制的亚硝酸钠溶液 $1\mathrm{mL}$ 产生无色气体。

# A.2.2.3 红外光谱鉴别

用红外吸收分光光度法，用溴化钾压片法制备试样，将试样谱图与对照谱图（见附录B)比较，两者应基本一致。

# A.3 二价铁含量的测定

# A.3.1 方法提要

在酸性介质中，试样以1,10-菲啰啉-亚铁为指示剂，用硫酸铈标准滴定溶液滴定，计算样品中的二价铁含量。

# A.3.2 试剂和材料

A.3.2.1 硫酸。  
A.3.2.2 硫酸铈标准滴定溶液： $c[\mathrm{Ce(SO_4)_2}] = 0.1\mathrm{mol / L}$

# A.3.2.3 1,10-菲啰啉-亚铁指示液。

# A.3.3 分析步骤

称取约 $1\mathrm{g}$ 于干燥减量测定条件下干燥后的试样，精确至 $0.0001\mathrm{g}$ ，置于已加有 $150~\mathrm{mL}$ 水和 $10~\mathrm{mL}$ 硫酸的 $300~\mathrm{mL}$ 锥形瓶中，溶解。加入1滴1,10-菲啰啉-亚铁指示液，立即用硫酸铈标准滴定溶液滴定至浅黄色为终点。

在测定的同时，按与测定相同的步骤，对不加试样而使用相同数量的试剂溶液做空白试验。

# A.3.4 结果计算

二价铁含量（以干基计）的质量分数 $w_{1}(\%)$ 按式(A.1)计算：

$$
w _ {1} = \frac {\left(V _ {1} - V _ {0}\right) \times c \times M}{m \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——试样消耗硫酸铈标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗硫酸铈标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫酸铈标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——铁的摩尔质量数值，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{Fe}) = 55.85]$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

1000——换算因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.4 氮的测定

# A.4.1 方法提要

用凯氏定氮法测定样品中的氮含量。

# A.4.2 试剂和材料

A.4.2.1 硫酸钾。  
A.4.2.2 五水硫酸铜。  
A.4.2.3 硫酸。  
A.4.2.4 氢氧化钠溶液： $300\mathrm{g / L}$   
A.4.2.5 硼酸溶液： $20\mathrm{g / L}$   
A.4.2.6 盐酸标准滴定溶液： $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$   
A.4.2.7 甲基红-次甲基蓝混合指示液。

# A.4.3 分析步骤

# A.4.3.1 自动定氮仪定氮法（仲裁法）

称取约 $0.3\mathrm{g}$ 于干燥减量测定条件下干燥后的试样，精确至 $0.0001\mathrm{g}$ ，置于自动定氮仪消化管中，加入 $3.0\mathrm{g}$ 硫酸钾， $0.15\mathrm{g}$ 五水硫酸铜，沿瓶壁缓缓加入 $10\mathrm{mL}$ 硫酸。置于消化装置上，于 $400^{\circ}\mathrm{C}$ 消化 $40\mathrm{min}$ ，冷却。置于自动定氮仪装置上，消化管中加入 $60\mathrm{mL}$ 氢氧化钠溶液，吸收瓶中加入 $50\mathrm{mL}$ 硼酸溶液和8滴甲基红-次甲基蓝混合指示液，蒸馏至馏出液呈中性。用盐酸标准滴定溶液滴定至灰紫色

为终点。

在测定的同时，按与测定相同的步骤，对不加试样而使用相同数量的试剂溶液做空白试验。

# A.4.3.2 直接蒸馏法

按HG/T4103中的直接蒸馏法进行测定。测定时称取 $0.4\mathrm{g}$ 于干燥减量测定条件下干燥后的试样。

# A.4.4 结果计算

氮含量（以干基计）的质量分数 $w_{2}(\%)$ 按式(A.2)计算：

$$
w _ {2} = \frac {\left(V _ {1} - V _ {0}\right) \times c \times M}{m \times 1000} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$V_{1}$ ——试样消耗盐酸标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗盐酸标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——盐酸标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——氮的摩尔质量数值，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{N}) = 14.01]$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

1000——换算因子。

试验结果以平行测定结果的算术平均值为准(保留一位小数)。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 三价铁的测定

# A.5.1 方法提要

在酸性介质中，试样以淀粉为指示剂，用硫代硫酸钠标准滴定溶液滴定，计算试样中的三价铁含量。

# A.5.2 试剂和材料

A.5.2.1 盐酸。

A.5.2.2 碘化钾。

A.5.2.3 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$

A.5.2.4 淀粉指示液： $10\mathrm{g / L}$

# A.5.3 分析步骤

称取约 $5\mathrm{g}$ 于干燥减量测定条件下干燥后的试样，精确至 $0.0001\mathrm{g}$ ，放入已加有 $100~\mathrm{mL}$ 水和 $10~\mathrm{mL}$ 盐酸的 $250~\mathrm{mL}$ 碘量瓶中，溶解。加入 $3\mathrm{g}$ 碘化钾，盖上瓶塞，轻轻摇动，于暗处放置 $5\mathrm{min}$ ，加入 $2\mathrm{mL}$ 淀粉指示液，用硫代硫酸钠标准滴定溶液滴定至蓝色消失为终点。

在测定的同时，按与测定相同的步骤，对不加试样而使用相同数量的试剂溶液做空白试验。

# A.5.4 结果计算

三价铁含量的质量分数 $\omega_{3}(\%)$ 按式(A.3)计算：

$$
w _ {3} = \frac {\left(V _ {1} - V _ {0}\right) \times c \times M}{m \times 1000} \times 100 \% \quad \dots \tag{A.3}
$$

式中：

$V_{1}$ ——试样消耗硫代硫酸钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗硫代硫酸钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫代硫酸钠标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——铁的摩尔质量数值，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{Fe}) = 55.85]$ ；

$m$ ——试样质量的数值，单位为克(g)；

1000——换算因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.1\%$ 。

# A.6 总铁的测定

# A.6.1 方法提要

试样经过消化处理后，以淀粉作指示剂，用硫代硫酸钠标准滴定溶液滴定，计算试样中的总铁含量。

# A.6.2 试剂和材料

A.6.2.1 硝酸。  
A.6.2.2 盐酸。  
A.6.2.3 碘化钾。  
A.6.2.4 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$   
A.6.2.5 淀粉指示液： $10\mathrm{g / L}$

# A.6.3 分析步骤

称取约 $0.5\mathrm{g}$ 于干燥减量测定条件下干燥后的试样，精确至 $0.0001\mathrm{g}$ ，置于消化器皿中，加入 $5\mathrm{mL}$ 硝酸，混合均匀，盖上表面皿，或者使用蒸汽回收装置，在 $95^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 保持不沸腾的情况下加热 $30\mathrm{min} \sim 40\mathrm{min}$ ，如果在规定时间内还有棕色烟气溢出，则表明硝酸没有将样品完全消化，再重新加入 $2\mathrm{mL}$ 硝酸，继续在上述条件下加热 $15\mathrm{min} \sim 20\mathrm{min}$ ，直至没有棕色烟气溢出，继续加热消化样品直到体积减少至大约 $3\mathrm{mL}$ ，确保在整个过程中器皿底部都被样品消化液覆盖，取下消化器皿，将其完全冷却，加入 $2\mathrm{mL}$ 盐酸，继续在 $95^{\circ}\mathrm{C} \pm 5^{\circ}\mathrm{C}$ 保持不沸腾的情况下加热 $15\mathrm{min} \sim 20\mathrm{min}$ ，加入 $2\mathrm{mL}$ 水使其沸腾。冷却至室温后，加入 $50\mathrm{mL}$ 水， $3\mathrm{g}$ 碘化钾，摇匀，于暗处放置 $5\mathrm{min}$ ，用硫代硫酸钠标准滴定溶液滴定，近终点时加入 $2\mathrm{mL}$ 淀粉指示液，继续滴定至蓝色消失为终点。

# A.6.4 结果计算

总铁含量的质量分数 $w_{4}(\%)$ 按式(A.4)计算：

$$
w _ {4} = \frac {V \times c \times M}{m \times 1000} \times 100 \% \tag{A.4}
$$

式中：

V ——试样消耗硫代硫酸钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫代硫酸钠标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——铁的摩尔质量数值，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{Fe}) = 55.85]$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

1000——换算因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.5\%$ 。

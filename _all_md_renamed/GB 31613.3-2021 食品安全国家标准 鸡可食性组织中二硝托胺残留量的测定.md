# 方法一 液相色谱法

# 1 范围

本文件规定了鸡肌肉、肝脏、肾脏和脂肪中二硝托胺及其代谢物残留量检测的制样和液相色谱测定方法。

本文件适用于鸡肌肉、肝脏、肾脏和脂肪中二硝托胺及其代谢物残留量的检测。

# 2 规范性引用文件

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试样中残留的二硝托胺及其代谢物经磷酸盐缓冲液提取，HLB固相萃取柱净化，液相色谱分离，二极管矩阵检测器检测，外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 磷酸二氢钾 $(\mathrm{KH}_2\mathrm{PO}_4)_3$   
5.1.2 磷酸氢二钾 $(\mathrm{K_2HPO_4})$ 。  
5.1.3 氢氧化钠 $(\mathrm{NaOH})$   
5.1.4 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.5 甲酸（HCOOH）：色谱纯。  
5.1.6 盐酸（HCl）。

# 5.2 溶液的配制

# 5.3 标准品

5.2.1 $1.0\mathrm{mol} / \mathrm{L}$ 盐酸溶液：取盐酸 $21.2\mathrm{mL}$ ，用水稀释至 $250~\mathrm{mL}$ ，摇匀。  
5.2.2 $1.0\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：取氢氧化钠 $40\mathrm{g}$ ，加水适量使溶解并稀释至 $1\mathrm{L}$ ，混匀。  
5.2.3 磷酸盐缓冲液 $(\mathrm{pH}6.0)$ ：取磷酸氢二钾 $2.0\mathrm{g}$ 、磷酸二氢钾 $8.0\mathrm{g}$ ，加水适量使溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.4 磷酸盐缓冲液 $(\mathrm{pH}4.0)$ ：取磷酸盐缓冲液 $(\mathrm{pH}6.0)$ ，用 $1.0\mathrm{mol / L}$ 盐酸溶液调 $\mathrm{pH}$ 至4.0，即得。  
5.2.5 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.6 $0.1\%$ 甲酸-乙腈：取 $0.1\%$ 甲酸溶液 $800~\mathrm{mL}$ 、乙腈 $200~\mathrm{mL}$ ，混匀。

二硝托胺、二硝托胺代谢物(3-ANOT)，含量均 $\geq 95\%$ ，见附录A。

# 5.4 标准溶液的制备

5.4.1 标准储备液：取二硝托胺、3-ANOT标准品各 $10\mathrm{mg}$ ，精密称定，用适量乙腈溶解并定容至 $100~\mathrm{mL}$ 容量瓶，摇匀。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期6个月。  
5.4.2 混合标准储备液：分别精密量取2种标准储备液各 $50~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，摇匀，即得。 $-18^{\circ}C$ 以下保存，有效期6个月。  
5.4.3 混合标准工作液：分别精密量取混合标准储备液适量，用 $0.1\%$ 甲酸-乙腈稀释制成含3-ANOT和二硝托胺浓度为 $25~\mu \mathrm{g / L},50~\mu \mathrm{g / L},100~\mu \mathrm{g / L},200~\mu \mathrm{g / L},500~\mu \mathrm{g / L},1000~\mu \mathrm{g / L}$ 和 $2000~\mu \mathrm{g / L}$ 的混合标准工作液。现用现配。

# 5.5 材料

5.5.1 HLB固相萃取柱： $60~\mathrm{mg / 3~mL}$ ，或相当者。  
5.5.2 滤膜：孔径 $0.22\mu \mathrm{m}$ ，有机系。

# 6 仪器和设备

6.1 液相色谱仪：配二极管矩阵检测器  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 固相萃取装置。  
6.4 均质机。  
6.5 离心管：聚丙烯塑料离心管， $50~\mathrm{mL}$   
6.6 离心机： $8000\mathrm{r / min}$ 或以上。  
6.7 pH计。

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存，3个月内进行分析检测。

# 8 测定步骤

# 8.1 提取

取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管，加磷酸盐缓冲液（ $\mathrm{pH}6.0)10.0\mathrm{mL}$ [鸡肝脏、肾脏加磷酸盐缓冲液 $(\mathrm{pH4.0})]$ ， $10000\mathrm{r / min}$ 均质 $1\mathrm{min},8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液，残渣加相应磷酸盐缓冲液 $10.0~\mathrm{mL}$ ，重复处理，合并上清液。准确移取上清液 $8\mathrm{mL}$ ，用氢氧化钠溶液调 $\mathrm{pH}$ 至 $7.0,8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液，备用。

# 8.2 净化

固相萃取柱依次用甲醇 $3\mathrm{mL}$ 、水 $3\mathrm{mL}$ 活化。取备用液，过柱，待液面到达柱床表面时用水 $3\mathrm{mL}$ 淋洗，抽干，用 $0.1\%$ 甲酸-乙腈 $4\mathrm{mL}$ 洗脱，收集洗脱液，过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱仪测定。

# 8.3 标准曲线的制备

分别精密量取混合标准工作液适量，注入液相色谱，记录色谱图。以各组分峰面积为纵坐标、相对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.4 测定

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱（ $100\mathrm{mm}\times 2.1\mathrm{mm},1.7\mu \mathrm{m})$ ，或相当者；  
b）流动相：A为 $0.1\%$ 甲酸溶液，B为乙腈，梯度洗脱条件见表1；

# 8.4.2 测定法

取试样溶液和标准溶液, 作单点或多点校准, 按外标法以色谱峰峰面积定量。标准溶液及试样溶液中目标药物的峰面积均应在仪器检测的线性范围之内。试样液中二硝托胺和3-ANOT的保留时间与标准工作液中二硝托胺和3-ANOT保留时间偏差在 $\pm 2.5\%$ 以内。标准溶液液相色谱图见附录B中的图B.1。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中二硝托胺的残留量按标准曲线或公式(1)计算

$$
X = \frac {A \times C _ {S} \times V _ {1} \times V _ {3} \times 1 0 0 0}{A _ {S} \times V _ {2} \times m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中二硝托胺或3-ANOT残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ ——标准溶液中二硝托胺或3-ANOT浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

A ——试样溶液中二硝托胺或3-ANOT的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中二硝托胺或3-ANOT的峰面积；

$V_{1}$ 提取液总体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——用于固相萃取净化上清液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——洗脱液体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法在鸡肌肉、肝脏、肾脏和脂肪中的检测限为 $150~\mu \mathrm{g / kg}$ ，定量限为 $500~\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法在 $500\mu \mathrm{g / kg}\sim 12000\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

# 19 结果计算和表述

试样中二硝托胺的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times C _ {\mathrm {S}} \times V _ {1} \times V _ {3} \times 1 0 0 0}{A _ {\mathrm {S}} \times V _ {2} \times m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中二硝托胺或3 ANOT 残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ ——标准溶液中二硝托胺或3-ANOT浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$A$ ——试样溶液中二硝托胺或3-ANOT的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中二硝托胺或3-ANOT的峰面积；

$V_{1}$ ——提取液总体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——用于固相萃取净化上清液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——洗脱液体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ ——试样质量的数值，单位为 $\mathrm{g}$ 。

# 20 检测方法的灵敏度、准确度和精密度

# 20.1 灵敏度

本方法在鸡肌肉、肝脏、肾脏和脂肪中的检测限为 $50~\mu \mathrm{g / kg}$ ，定量限为 $100~\mu \mathrm{g / kg}$ 。

# 20.2 准确度

本方法在 $100\mu \mathrm{g / kg}\sim 12000\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$

# 20.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

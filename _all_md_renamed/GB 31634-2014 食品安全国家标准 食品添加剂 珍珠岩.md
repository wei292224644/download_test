# 附录A

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。

# A.2 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。本标准试验中所需杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T602、GB/T603规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸溶液： $1 + 3$   
A.3.1.2 氨水溶液： $2 + 3$   
A.3.1.3 氢氧化钠溶液： $40~\mathrm{g / L}$   
A.3.1.4 酒石酸氢钠溶液： $100\mathrm{g / L}$ 。现用现配。  
A.3.1.5 甲基红指示液。  
A.3.1.6 石蕊试纸。

# A.3.2 鉴别方法

# A.3.2.1 试样溶液A的制备

称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $100\mathrm{mL}$ 烧杯中，加入 $25\mathrm{mL}$ 的盐酸溶液混合，盖上表面皿置于水浴中加热 $15\mathrm{min}$ ，冷却至室温并过滤。用氨水溶液调节滤液 $\mathrm{pH} = 4.5$ ，并用石蕊试纸检测。滤液为试样溶液A用于铝、钾、钠的鉴别。

# A.3.2.2 铝的鉴别

取 $5\mathrm{mL}$ 试样溶液A(A.3.2.1)，滴加氨水溶液，生成白色胶状沉淀，加入过量的氨水溶液，沉淀不溶解。滴加氢氧化钠溶液生成白色胶状沉淀，加入过量时沉淀溶解。

# A.3.2.3 钾的鉴别

取 $5\mathrm{mL}$ 试样溶液A(A.3.2.1)，滴加酒石酸氢钠溶液，会缓慢出现白色结晶沉淀。该溶液可溶于氨水和氢氧化钠溶液中。用玻璃棒搅拌或加入少量的乙酸或乙醇可以加速沉淀的生成。

# A.3.2.4 钠的鉴别

用盐酸润湿铂丝，在火焰上燃烧至无色，再蘸取少许试样溶液在火焰上燃烧，火焰即呈深黄色。

# A.3.2.5 硅的鉴别

用铂丝环蘸取少量硫酸铵钠晶体，在火焰中使晶体熔融，形成熔融珠体。让熔融珠体接触样品，硅浮现在熔融珠体表面，形成一种不透明的网状的结构。

# A.4 水可溶物的测定

# A.4.1 仪器和设备

A.4.1.1 蒸发皿： $100~\mathrm{mL}$   
A.4.1.2 滤膜过滤器：与过滤膜配套。  
A.4.1.3 过滤膜：材质为聚四氟乙烯或其他材质的过滤膜，孔径为 $0.45\mu \mathrm{m}$   
A.4.1.4 真空泵与过滤设备。  
A.4.1.5 电热恒温干燥箱： $105^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$

# A.4.2 试样溶液B的制备

称取 $10.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ 。置于 $250~\mathrm{mL}$ 烧杯中，加入 $100~\mathrm{mL}$ 水，在水浴中加热 $2\mathrm{h}$ ，期间搅拌，并补水至原体积。冷却后，用滤膜过滤器过滤，如滤液浑浊，再次抽滤。用水清洗烧杯和滤渣至滤液至 $100~\mathrm{mL}$ ，此溶液为溶液B，用于测定溶液水可溶物和 $\mathrm{pH}$ 。

# A.4.3 分析步骤

移取 $50~\mathrm{mL}$ 试样溶液B至预先于 $105~\mathrm{^\circ C}\pm 2~\mathrm{^\circ C}$ 下干燥至质量恒定的蒸发皿中，在 $105~\mathrm{^\circ C}\pm 2~\mathrm{^\circ C}$ 下干燥 $2\mathrm{h}$ ，冷却 $30\mathrm{min}$ ，称量。

# A.4.4 结果计算

水可溶物的质量分数 $\varpi_{1}$ 按式(A.1)计算：

$$
w _ {1} = \frac {\left(m _ {1} - m _ {2}\right) \times 100}{m \times 50} \times 100 \% \tag{A.1}
$$

式中：

$m_{1}$ ——干燥后残渣和蒸发皿的质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——干燥后蒸发皿质量，单位为克 $(\mathrm{g})$ ；

100——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克(g)；

50 ——移取试样溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.01\%$ 。

# A.5 $\mathbf{pH}$ 的测定

# A.5.1 仪器、设备

$\mathrm{pH}$ 计：精度0.02。

# A.5.2 分析步骤

取适量的试样溶液B，按GB/T9724的规定测定其 $\mathrm{pH}$

# A.6 干燥减量的测定

# A.6.1 仪器和设备

A.6.1.1 称量瓶： $\phi 30\mathrm{mm}\times 25\mathrm{mm}$   
A.6.1.2 电热恒温干燥箱： $105^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$

# A.6.2 分析步骤

在预先于 $105\mathrm{^\circ C}\pm 2\mathrm{^\circ C}$ 下干燥至质量恒定的称量瓶中称取约 $2\mathrm{g}$ 试样，精确到 $0.0002\mathrm{g}$ ，在 $105\mathrm{^\circ C}\pm 2\mathrm{^\circ C}$ 下干燥 $2\mathrm{h}$ ，冷却 $30\mathrm{min}$ ，称量。

# A.6.3 结果计算

干燥减量的质量分数 $\omega_{2}$ 按式(A.2)计算：

$$
w _ {2} = \frac {m _ {1} - m _ {2}}{m} \times 100 \% \tag{A.2}
$$

式中：

$m_{1}$ ——试样和称量瓶的质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——干燥后试样和称量瓶质量，单位为克 $(\mathrm{g})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.7 灼烧减量的测定

# A.7.1 仪器和设备

A.7.1.1 瓷坩埚: $50 \mathrm{~mL}$ 。  
A.7.1.2 高温炉： $1000^{\circ}\mathrm{C}\pm 50^{\circ}\mathrm{C}$

# A.7.2 分析步骤

在预先于 $1000^{\circ}\mathrm{C} \pm 50^{\circ}\mathrm{C}$ 下灼烧至质量恒定的瓷坩埚中称取约 $0.25\mathrm{g}$ 试样，精确到 $0.0002\mathrm{g}$ ，在 $1000^{\circ}\mathrm{C} \pm 50^{\circ}\mathrm{C}$ 下灼烧 $2\mathrm{h}$ ，冷却 $30\mathrm{min}$ ，称量。

# A.7.3 结果计算

灼烧减量的质量分数 $w_{3}$ 按式(A.3)计算：

$$
w _ {3} = \frac {m _ {1} - m _ {2}}{m} \times 100 \% \tag{A.3}
$$

式中：

$m_{1}$ ——试样和瓷坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——灼烧后试样和瓷坩埚的质量，单位为克 $(\mathrm{g})$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.5\%$ 。

# A.8 无机砷（以As计)的测定

# A.8.1 试剂和材料

盐酸溶液： $1 + 3$

# A.8.2 试样溶液C的制备

称取 $10\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ 。置于 $250~\mathrm{mL}$ 烧杯中，加入 $50~\mathrm{mL}$ 盐酸溶液，盖上表面皿，缓慢加热至沸，并保持沸腾 $15\mathrm{min}$ ，冷却，静置使不溶物沉降。将溶液过滤至 $100~\mathrm{mL}$ 容量瓶中，用热水洗涤烧杯三次（尽量使不溶物留在烧杯内），每次 $10~\mathrm{mL}$ ，最后用 $15~\mathrm{mL}$ 热水洗涤滤纸，滤液并入 $100~\mathrm{mL}$ 容量瓶中。冷却后用水稀释至刻度，摇匀。此溶液定为试样溶液C，用于无机砷、铅含量的测定。

# A.8.3 分析步骤

移取 $10.00\mathrm{mL}$ 试样溶液C(A.8.2)，按GB/T5009.11或GB/T5009.76规定的方法进行测定。

# A.9 铅(Pb)的测定

移取 $10.00\mathrm{mL}$ 试样溶液C(A.8.2)，按照GB5009.12中第一法石墨炉原子吸收光谱法或第二法氢化物原子荧光光谱法进行测定。

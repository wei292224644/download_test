# 前言

本标准代替GB/T5009.86—2003《蔬菜、水果及其制品中总抗坏血酸的测定(荧光法和2,4-二硝基苯肼法)》、GB/T5009.159—2003《食品中还原型抗坏血酸的测定》和GB6195—1986《水果、蔬菜维生素C含量测定法(2,6-二氯靛酚滴定法)》。

本标准与GB/T5009.86—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中抗坏血酸的测定”；  
——扩大了方法的适用范围；  
——增加了高效液相色谱法及相关方法的定量限；  
——删除了2，4-二硝基苯肼法；  
——增加了2，6-二氯靛酚滴定法；  
——按照GB/T20001.4—2015对原标准的结构进行了修改。

# 食品安全国家标准

# 食品中抗坏血酸的测定

# 1 范围

本标准规定了高效液相色谱法、荧光法、2,6-二氯靛酚滴定法测定食品中抗坏血酸的方法。

本标准第一法适用于乳粉、谷物、蔬菜、水果及其制品、肉制品、维生素类补充剂、果冻、胶基糖果、八宝粥、葡萄酒中的 $\mathrm{L}(+)$ -抗坏血酸、 $\mathrm{D}(-)$ -抗坏血酸和 $\mathrm{L}(+)$ -抗坏血酸总量的测定。第二法适用于乳粉、蔬菜、水果及其制品中 $\mathrm{L}(+)$ -抗坏血酸总量的测定。第三法适用于水果、蔬菜及其制品中 $\mathrm{L}(+)$ -抗坏血酸的测定。

# 2 术语和定义

2.1 抗坏血酸：一种具有抗氧化性质的有机化合物。又称为“维生素C”，是人体必需的营养素之一。  
2.2 $\mathrm{L}(+)$ -抗坏血酸：左式右旋光抗坏血酸。具有强还原性，对人体具有生物活性。  
2.3 D(-)-抗坏血酸：又称异抗坏血酸。具有强还原性，但对人体基本无生物活性。  
2.4 L(+) - 脱氢抗坏血酸: L(+) - 抗坏血酸极易被氧化为 L(+) - 脱氢抗坏血酸, L(+) - 脱氢抗坏血酸亦可被还原为 L(+) - 抗坏血酸。通常称为脱氢抗坏血酸。  
2.5 L(+) - 抗坏血酸总量: 将试样中 $\mathrm{L}(+)$ -脱氢抗坏血酸还原成的 $\mathrm{L}(+)$ -抗坏血酸或将试样中 $\mathrm{L}(+)$ -抗坏血酸氧化成的 $\mathrm{L}(+)$ -脱氢抗坏血酸后测得的 $\mathrm{L}(+)$ -抗坏血酸总量。

# 第一法 高效液相色谱法

# 3 原理

试样中的抗坏血酸用偏磷酸溶解超声提取后，以离子对试剂为流动相，经反相色谱柱分离，其中 $\mathrm{L}(+)$ -抗坏血酸和 $\mathrm{D}(-)$ -抗坏血酸直接用配有紫外检测器的液相色谱仪(波长 $245~\mathrm{nm}$ )测定；试样中的 $\mathrm{L}(+)$ -脱氢抗坏血酸经 $\mathrm{L}-$ 半胱氨酸溶液进行还原后，用紫外检测器(波长 $245~\mathrm{nm}$ )测定 $\mathrm{L}(+)$ -抗坏血酸总量，或减去原样品中测得的 $\mathrm{L}(+)$ -抗坏血酸含量而获得 $\mathrm{L}(+)$ -脱氢抗坏血酸的含量。以色谱峰的保留时间定性，外标法定量。

# 4 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 偏磷酸 $(\mathrm{HPO_3})_n$ ：含量（以 $\mathrm{HPO_3}$ 计） $\geqslant 38\%$   
4.1.2 磷酸三钠 $(\mathrm{Na}_3\mathrm{PO}_4\cdot 12\mathrm{H}_2\mathrm{O})$ 。  
4.1.3 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$

4.1.4 磷酸 $(\mathrm{H}_3\mathrm{PO}_4):85\%$   
4.1.5 L-半胱氨酸 $\mathrm{(C_3H_7NO_2S)}$ ：优级纯。  
4.1.6 十六烷基三甲基溴化铵 $\left(\mathrm{C}_{19} \mathrm{H}_{42} \mathrm{BrN}\right)$ ：色谱纯。  
4.1.7 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

# 4.2 试剂配制

4.2.1 偏磷酸溶液 $(200\mathrm{g / L})$ ：称取 $200\mathrm{g}$ （精确至 $0.1\mathrm{g}$ )偏磷酸(4.1.1)，溶于水并稀释至 $1\mathrm{L}$ ，此溶液保存于 $4^{\circ}C$ 的环境下可保存一个月。  
4.2.2 偏磷酸溶液 $(20\mathrm{g / L})$ ：量取 $50~\mathrm{mL}~200~\mathrm{g / L}$ 偏磷酸溶液，用水稀释至 $500~\mathrm{mL}$ 。  
4.2.3 磷酸三钠溶液 $(100\mathrm{g / L})$ ：称取 $100\mathrm{g}$ （精确至 $0.1\mathrm{g}$ )磷酸三钠，溶于水并稀释至 $1\mathrm{L}$ 。  
4.2.4 L-半胱氨酸溶液 $(40\mathrm{g / L})$ ：称取 $4\mathrm{gL}$ -半胱氨酸，溶于水并稀释至 $100~\mathrm{mL}$ 。临用时配制。

# 4.3 标准品

4.3.1 L(+) - 抗坏血酸标准品 $\left(\mathrm{C}_{6} \mathrm{H}_{8} \mathrm{O}_{6}\right)$ ：纯度 $\geqslant 99 \%$ 。  
4.3.2 D(-)-抗坏血酸(异抗坏血酸)标准品 $\mathrm{(C_6H_8O_6)}$ ：纯度 $\geq 99\%$ 。

# 4.4 标准溶液配制

4.4.1 L(+) - 抗坏血酸标准贮备溶液 $(1.000\mathrm{mg / mL})$ : 准确称取 $\mathrm{L}(+)$ - 抗坏血酸标准品 $0.01\mathrm{g}$ (精确至 $0.01\mathrm{mg}$ ), 用 $20\mathrm{g / L}$ 的偏磷酸溶液定容至 $10\mathrm{mL}$ 。该贮备液在 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 避光条件下可保存一周。  
4.4.2 D(-)-抗坏血酸标准贮备溶液 $(1.000\mathrm{mg / mL})$ ：准确称取D(-)-抗坏血酸标准品 $0.01\mathrm{g}$ （精确至 $0.01\mathrm{mg}$ )，用 $20~\mathrm{g / L}$ 的偏磷酸溶液定容至 $10~\mathrm{mL}$ 。该贮备液在 $2^{\circ}C\sim 8^{\circ}C$ 避光条件下可保存一周。  
4.4.3 抗坏血酸混合标准系列工作液：分别吸取 $\mathrm{L}(+)$ -抗坏血酸和 $\mathrm{D}(-)$ -抗坏血酸标准贮备液 $0\mathrm{mL}$ ， $0.05\mathrm{mL}, 0.50\mathrm{mL}, 1.0\mathrm{mL}, 2.5\mathrm{mL}, 5.0\mathrm{mL}$ ，用 $20\mathrm{g/L}$ 的偏磷酸溶液定容至 $100\mathrm{mL}$ 。标准系列工作液中 $\mathrm{L}(+)$ -抗坏血酸和 $\mathrm{D}(-)$ -抗坏血酸的浓度分别为 $0\mu\mathrm{g/mL}, 0.5\mu\mathrm{g/mL}, 5.0\mu\mathrm{g/mL}, 10.0\mu\mathrm{g/mL}, 25.0\mu\mathrm{g/mL}, 50.0\mu\mathrm{g/mL}$ 。临用时配制。

# 5 仪器和设备

5.1 液相色谱仪：配有二极管阵列检测器或紫外检测器。  
5.2 pH计：精度为0.01。  
5.3 天平：感量为 $0.1\mathrm{g}, 1\mathrm{mg}, 0.01\mathrm{mg}$ 。  
5.4 超声波清洗器。  
5.5 离心机：转速 $\geqslant 4000\mathrm{r / min}$   
5.6 均质机。  
5.7 滤膜： $0.45\mu \mathrm{m}$ 水相膜。  
5.8 振荡器。

# 6 分析步骤

整个检测过程尽可能在避光条件下进行。

# 6.1 试样制备

6.1.1 液体或固体粉末样品：混合均匀后，应立即用于检测。

6.1.2 水果、蔬菜及其制品或其他固体样品：取 $100\mathrm{g}$ 左右样品加入等质量 $20\mathrm{g / L}$ 的偏磷酸溶液，经均质机均质并混合均匀后，应立即测定。

# 6.2 试样溶液的制备

称取相对于样品约 $0.5\mathrm{g}\sim 2\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )混合均匀的固体试样或匀浆试样，或吸取 $2\mathrm{mL}\sim$ $10~\mathrm{mL}$ 液体试样[使所取试样含 $\mathrm{L}(+)$ -抗坏血酸约 $0.03\mathrm{mg}\sim 6\mathrm{mg}]$ 于 $50~\mathrm{mL}$ 烧杯中，用 $20~\mathrm{g / L}$ 的偏磷酸溶液(4.2.2)将试样转移至 $50~\mathrm{mL}$ 容量瓶中，震摇溶解并定容。摇匀，全部转移至 $50~\mathrm{mL}$ 离心管中，超声提取 $5\mathrm{min}$ 后，于 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液过 $0.45\mu \mathrm{m}$ 水相滤膜，滤液待测[由此试液可同时分别测定试样中 $\mathrm{L}(+)$ -抗坏血酸和 $\mathrm{D}(-)$ -抗坏血酸的含量]。

# 6.3 试样溶液的还原

准确吸取 $20~\mathrm{mL}$ 上述离心后的上清液于 $50~\mathrm{mL}$ 离心管中，加入 $10~\mathrm{mL}40~\mathrm{g / L}$ 的 $\mathrm{L}$ -半胱氨酸溶液(4.2.4)，用 $100~\mathrm{g / L}$ 磷酸三钠溶液调节 $\mathrm{pH}$ 至 $7.0\sim 7.2$ ，以200次/min振荡 $5\mathrm{min}$ 。再用磷酸调节 $\mathrm{pH}$ 至 $2.5\sim 2.8$ ，用水将试液全部转移至 $50~\mathrm{mL}$ 容量瓶中，并定容至刻度。混匀后取此试液过 $0.45\mu \mathrm{m}$ 水相滤膜后待测[由此试液可测定试样中包括脱氢型的 $\mathrm{L}(+)$ -抗坏血酸总量]。

若试样含有增稠剂，可准确吸取 $4\mathrm{mL}$ 经 $\mathrm{L}-$ 半胱氨酸溶液还原的试液，再准确加入 $1\mathrm{mL}$ 甲醇，混匀后过 $0.45\mu \mathrm{m}$ 滤膜后待测。

# 6.4 仪器参考条件

6.4.1 色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6~\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或同等性能的色谱柱。  
6.4.2 检测器：二极管阵列检测器或紫外检测器。  
6.4.3 流动相：A: $6.8\mathrm{g}$ 磷酸二氢钾和 $0.91\mathrm{g}$ 十六烷基三甲基溴化铵，用水溶解并定容至 $1\mathrm{L}$ （用磷酸调 $\mathrm{pH}$ 至 $2.5\sim 2.8)$ ；B: $100\%$ 甲醇。按 $\mathrm{A:B} = 98:2$ 混合，过 $0.45\mu \mathrm{m}$ 滤膜，超声脱气。  
6.4.4 流速： $0.7\mathrm{mL / min}$   
6.4.5 检测波长： $245~\mathrm{nm}$   
6.4.6 柱温： $25^{\circ}C$   
6.4.7 进样量： $20~\mu \mathrm{L}$

# 6.5 标准曲线制作

分别对抗坏血酸混合标准系列工作溶液进行测定，以 $\mathrm{L}(+)$ -抗坏血酸[或D(-)-抗坏血酸]标准溶液的质量浓度 $(\mu \mathrm{g / mL})$ 为横坐标， $\mathrm{L}(+)$ -抗坏血酸[或D(-)-抗坏血酸]的峰高或峰面积为纵坐标，绘制标准曲线或计算回归方程。 $\mathrm{L}(+)$ -抗坏血酸、D(-)-抗坏血酸标准色谱图参见附录A中图A.1。

# 6.6 试样溶液的测定

对试样溶液进行测定，根据标准曲线得到测定液中 $\mathrm{L}(+)$ -抗坏血酸[或D(-)-抗坏血酸]的浓度 $(\mu \mathrm{g / mL})$ 。

# 6.7 空白试验

空白试验系指除不加试样外，采用完全相同的分析步骤、试剂和用量，进行平行操作。

# 7 分析结果的表述

试样中 $\mathrm{L}(+)$ -抗坏血酸[或D(-)-抗坏血酸]的含量和 $\mathrm{L}(+)$ -抗坏血酸总量以毫克每百克表示，

按式(1)计算：

$$
X = \frac {\left(c _ {1} - c _ {0}\right) \times V}{m \times 1 0 0 0} \times F \times K \times 1 0 0 \quad \dots \tag {1}
$$

式中：

X ——试样中 $\mathrm{L}(+)$ -抗坏血酸[或 $\mathrm{D}(-)$ -抗坏血酸、 $\mathrm{L}(+)$ -抗坏血酸总量]的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$c_{1}$ ——样液中 $\mathrm{L}(+)$ -抗坏血酸[或 $\mathrm{D}(-)$ -抗坏血酸]的质量浓度，单位为微克每毫升 $(\mu \mathrm{g / mL})$ ；

$c_{0}$ ——样品空白液中 $\mathrm{L}(+)$ -抗坏血酸[或 $\mathrm{D}(-)$ -抗坏血酸]的质量浓度，单位为微克每毫升 $(\mu \mathrm{g / mL})$ ；

$V$ ——试样的最后定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——实际检测试样质量，单位克 $(\mathrm{g})$ ；

1000——换算系数（由 $\mu \mathrm{g} / \mathrm{mL}$ 换算成 $\mathrm{mg / mL}$ 的换算因子）；

$F$ ——稀释倍数（若使用6.3还原步骤时，即为2.5）；

$K$ ——若使用6.3中甲醇沉淀步骤时，即为1.25；

100 ——换算系数（由 $\mathrm{mg / g}$ 换算成 $\mathrm{mg / 100g}$ 的换算因子）。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 8 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 9 其他

固体样品取样量为 $2\mathrm{g}$ 时， $\mathrm{L}(+)$ -抗坏血酸和 $\mathrm{D}(-)$ -抗坏血酸的检出限均为 $0.5\mathrm{mg / 100g}$ ，定量限均为 $2.0~\mathrm{mg / 100~g}$ 。液体样品取样量为 $10\mathrm{g}$ (或 $10~\mathrm{mL}$ )时， $\mathrm{L}(+)$ -抗坏血酸和 $\mathrm{D}(-)$ -抗坏血酸的检出限均为 $0.1\mathrm{mg / 100g}$ （或 $0.1\mathrm{mg / 100mL}$ )，定量限均为 $0.4\mathrm{mg / 100g}$ 或 $0.4\mathrm{mg / 100mL}$ 。

# 第二法 荧光法

# 10 原理

试样中 $\mathrm{L}(+)$ -抗坏血酸经活性炭氧化为 $\mathrm{L}(+)$ -脱氢抗坏血酸后，与邻苯二胺(OPDA)反应生成有荧光的喹啉（quinoxaline)，其荧光强度与 $\mathrm{L}(+)$ -抗坏血酸的浓度在一定条件下成正比，以此测定试样中 $\mathrm{L}(+)$ -抗坏血酸总量。

注：L(+) - 脱氢抗坏血酸与硼酸可形成复合物而不与OPDA反应，以此排除试样中荧光杂质产生的干扰。

# 11 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 11.1 试剂

11.1.1 偏磷酸 $(\mathrm{HPO}_3)_n$ ：含量（以 $\mathrm{HPO}_3$ 计） $\geqslant 38\%$   
11.1.2 冰乙酸 $\mathrm{(CH_3COOH)}$ ：浓度约为 $30\%$

11.1.3 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ ：浓度约为 $98\%$ 。  
11.1.4 乙酸钠 $\mathrm{(CH_3COONa)}$ 。  
11.1.5 硼酸 $(\mathrm{H}_3\mathrm{BO}_3)$ 。  
11.1.6 邻苯二胺 $(\mathrm{C_6H_8N_2})$ 。  
11.1.7 百里酚蓝 $(\mathrm{C}_{27}\mathrm{H}_{30}\mathrm{O}_5\mathrm{S})$ 。  
11.1.8 活性炭粉。

# 11.2 试剂的配制

11.2.1 偏磷酸-乙酸溶液：称取 $15\mathrm{g}$ 偏磷酸，加入 $40~\mathrm{mL}$ 冰乙酸及 $250~\mathrm{mL}$ 水，加温，搅拌，使之逐渐溶解，冷却后加水至 $500~\mathrm{mL}$ 。于 $4^{\circ}\mathrm{C}$ 冰箱可保存 $7\mathrm{d} \sim 10\mathrm{d}$ 。  
11.2.2 硫酸溶液 $(0.15\mathrm{mol} / \mathrm{L})$ ：取 $8.3~\mathrm{mL}$ 硫酸，小心加入水中，再加水稀释至 $1000~\mathrm{mL}$ 。  
11.2.3 偏磷酸-乙酸-硫酸溶液：称取 $15\mathrm{g}$ 偏磷酸，加入 $40~\mathrm{mL}$ 冰乙酸，滴加 $0.15\mathrm{mol} / \mathrm{L}$ 硫酸溶液至溶解，并稀释至 $500~\mathrm{mL}$ 。  
11.2.4 乙酸钠溶液 $(500\mathrm{g / L})$ ：称取 $500\mathrm{g}$ 乙酸钠，加水至 $1000~\mathrm{mL}$   
11.2.5硼酸-乙酸钠溶液：称取 $3\mathrm{g}$ 硼酸，用 $500~\mathrm{g / L}$ 乙酸钠溶液溶解并稀释至 $100~\mathrm{mL}$ 。临用时配制。  
11.2.6 邻苯二胺溶液 $(200\mathrm{mg / L})$ ：称取 $20~\mathrm{mg}$ 邻苯二胺，用水溶解并稀释至 $100~\mathrm{mL}$ ，临用时配制。  
11.2.7 酸性活性炭：称取约 $200\mathrm{g}$ 活性炭粉 $(75\mu \mathrm{m} \sim 177\mu \mathrm{m})$ ，加入 $1\mathrm{L}$ 盐酸 $(1 + 9)$ ，加热回流 $1\mathrm{h} \sim 2\mathrm{h}$ 过滤，用水洗至滤液中无铁离子为止，置于 $110^{\circ}\mathrm{C} \sim 120^{\circ}\mathrm{C}$ 烘箱中干燥 $10\mathrm{h}$ ，备用。

检验铁离子方法：利用普鲁士蓝反应。将 $20\mathrm{g / L}$ 亚铁氰化钾与 $1\%$ 盐酸等量混合，将上述洗出滤液滴入，如有铁离子则产生蓝色沉淀。

11.2.8 百里酚蓝指示剂溶液 $(0.4\mathrm{mg / mL})$ ：称取 $0.1\mathrm{g}$ 百里酚蓝，加入 $0.02\mathrm{mol / L}$ 氢氧化钠溶液约 $10.75~\mathrm{mL}$ ，在玻璃研钵中研磨至溶解，用水稀释至 $250~\mathrm{mL}$ 。（变色范围：pH等于1.2时呈红色；pH等于2.8时呈黄色；pH大于4时呈蓝色）。

# 11.3 标准品

$\mathrm{L}(+)$ -抗坏血酸标准品 $(\mathrm{C}_6\mathrm{H}_8\mathrm{O}_6)$ ：纯度 $\geq 99\%$ 。

# 11.4 标准品的配制

11.4.1 L(+) - 抗坏血酸标准溶液 $(1.000\mathrm{mg / mL})$ ：称取 $\mathrm{L}(+)$ - 抗坏血酸 $0.05\mathrm{g}$ （精确至 $0.01\mathrm{mg}$ ），用偏磷酸-乙酸溶液溶解并稀释至 $50~\mathrm{mL}$ ，该贮备液在 $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 避光条件下可保存一周。  
11.4.2 L(+) - 抗坏血酸标准工作液 $(100.0\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取 $\mathrm{L}(+)$ - 抗坏血酸标准液 $10~\mathrm{mL}$ ，用偏磷酸-乙酸溶液稀释至 $100~\mathrm{mL}$ ，临用时配制。

# 12 仪器和设备

荧光分光光度计：具有激发波长 $338\mathrm{nm}$ 及发射波长 $420\mathrm{nm}$ 。配有 $1\mathrm{cm}$ 比色皿。

# 13 分析步骤

整个检测过程应在避光条件下进行。

# 13.1 试液的制备

称取约 $100\mathrm{g}$ (精确至 $0.1\mathrm{g}$ ) 试样, 加 $100\mathrm{g}$ 偏磷酸-乙酸溶液, 倒入捣碎机内打成匀浆, 用百里酚蓝

指示剂测试匀浆的酸碱度。如呈红色，即称取适量匀浆用偏磷酸-乙酸溶液稀释；若呈黄色或蓝色，则称取适量匀浆用偏磷酸-乙酸-硫酸溶液稀释，使其 $\mathrm{pH}$ 为1.2。匀浆的取用量根据试样中抗坏血酸的含量而定。当试样液中抗坏血酸含量在 $40~\mu \mathrm{g / mL}\sim 100~\mu \mathrm{g / mL}$ 之间，一般称取 $20\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 匀浆，用相应溶液稀释至 $100~\mathrm{mL}$ ，过滤，滤液备用。

# 13.2 测定

13.2.1 氧化处理：分别准确吸取 $50~\mathrm{mL}$ 试样滤液及抗坏血酸标准工作液于 $200~\mathrm{mL}$ 具塞锥形瓶中，加入 $2\mathrm{g}$ 活性炭，用力振摇 $1\mathrm{min}$ ，过滤，弃去最初数毫升滤液，分别收集其余全部滤液，即为试样氧化液和标准氧化液，待测定。  
13.2.2 分别准确吸取 $10\mathrm{mL}$ 试样氧化液于两个 $100~\mathrm{mL}$ 容量瓶中，作为“试样液”和“试样空白液”。  
13.2.3 分别准确吸取 $10~\mathrm{mL}$ 标准氧化液于两个 $100~\mathrm{mL}$ 容量瓶中，作为“标准液”和“标准空白液”。  
13.2.4 于“试样空白液”和“标准空白液”中各加 $5\mathrm{mL}$ 硼酸-乙酸钠溶液，混合摇动 $15\mathrm{min}$ ，用水稀释至 $100\mathrm{mL}$ ，在 $4^{\circ}\mathrm{C}$ 冰箱中放置 $2\mathrm{h} \sim 3\mathrm{h}$ ，取出待测。  
13.2.5 于“试样液”和“标准液”中各加 $5\mathrm{mL}$ 的 $500\mathrm{g / L}$ 乙酸钠溶液，用水稀释至 $100~\mathrm{mL}$ ，待测。

# 13.3 标准曲线的制备

准确吸取上述“标准液” $\left[\mathrm{L}(+)\right.$ -抗坏血酸含量 $10\mu \mathrm{g} / \mathrm{mL}\left[0.5\mathrm{mL},1.0\mathrm{mL},1.5\mathrm{mL},2.0\mathrm{mL}\right.$ ，分别置于 $10~\mathrm{mL}$ 具塞刻度试管中，用水补充至 $2.0  \mathrm{mL}$ 。另准确吸取“标准空白液” $2\mathrm{mL}$ 于 $10~\mathrm{mL}$ 带盖刻度试管中。在暗室迅速向各管中加入 $5\mathrm{mL}$ 邻苯二胺溶液，振摇混合，在室温下反应 $35\mathrm{min}$ ，于激发波长 $338~\mathrm{nm}$ 、发射波长 $420~\mathrm{nm}$ 处测定荧光强度。以“标准液”系列荧光强度分别减去“标准空白液”荧光强度的差值为纵坐标，对应的 $\mathrm{L}(+)$ -抗坏血酸含量为横坐标，绘制标准曲线或计算直线回归方程。

# 13.4 试样测定

分别准确吸取 $2\mathrm{mL}$ “试样液”和“试样空白液”于 $10~\mathrm{mL}$ 具塞刻度试管中，在暗室迅速向各管中加入 $5\mathrm{mL}$ 邻苯二胺溶液，振摇混合，在室温下反应 $35\mathrm{min}$ ，于激发波长 $338~\mathrm{nm}$ 、发射波长 $420~\mathrm{nm}$ 处测定荧光强度。以“试样液”荧光强度减去“试样空白液”的荧光强度的差值于标准曲线上查得或回归方程计算测定试样溶液中 $\mathrm{L}(+)$ -抗坏血酸总量。

# 14 结果计算

试样中 $\mathrm{L}(+)$ -抗坏血酸总量，结果以毫克每百克表示，按式(2)计算：

$$
X = \frac {c \times V}{m} \times F \times \frac {1 0 0}{1 0 0 0} \quad \dots \tag {2}
$$

式中：

$X$ ——试样中 $\mathrm{L}(+)$ - 抗坏血酸的总量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$c$ ——由标准曲线查得或回归方程计算的进样液中 $\mathrm{L}(+)$ -抗坏血酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

V 荧光反应所用试样体积，单位为毫升 $(\mathrm{mL})$

$m$ ——实际检测试样质量，单位克 $(\mathrm{g})$ ；

$F$ ——试样溶液的稀释倍数；

100 ——换算系数；

1000——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 15 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 16 其他

当样品取样量为 $10\mathrm{g}$ 时， $\mathrm{L}(+)$ -抗坏血酸总量的检出限为 $0.044\mathrm{mg} / 100\mathrm{g}$ ，定量限为 $0.7\mathrm{mg} / 100\mathrm{g}$ 。

# 第三法 2,6-二氯靛酚滴定法

# 17 原理

用蓝色的碱性染料2,6-二氯靛酚标准溶液对含 $\mathrm{L}(+)$ -抗坏血酸的试样酸性浸出液进行氧化还原滴定，2,6-二氯靛酚被还原为无色，当到达滴定终点时，多余的2,6-二氯靛酚在酸性介质中显浅红色，由2,6-二氯靛酚的消耗量计算样品中 $\mathrm{L}(+)$ -抗坏血酸的含量。

# 18 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 18.1 试剂

18.1.1 偏磷酸 $(\mathrm{HPO}_3)_n$ ：含量（以 $\mathrm{HPO}_3$ 计） $\geqslant 38\%$ 。  
18.1.2 草酸 $(\mathrm{C_2H_2O_4})$ 。  
18.1.3 碳酸氢钠 $(\mathrm{NaHCO_3})$ 。  
18.1.4 2,6-二氯靛酚(2,6-二氯靛酚钠盐， $\mathrm{C_{12}H_6Cl_2NNaO_2}$ ）。  
18.1.5 白陶土（或高岭土）：对抗坏血酸无吸附性。

# 18.2 试剂的配制

18.2.1 偏磷酸溶液 $(20\mathrm{g} / \mathrm{L})$ ：称取 $20\mathrm{g}$ 偏磷酸，用水溶解并定容至 $1\mathrm{L}$ 。  
18.2.2 草酸溶液 $(20\mathrm{g / L})$ ：称取 $20\mathrm{g}$ 草酸，用水溶解并定容至 $1\mathrm{L}$ 。  
18.2.3 2,6-二氯靛酚(2,6-二氯靛酚钠盐)溶液：称取碳酸氢钠 $52\mathrm{mg}$ 溶解在 $200~\mathrm{mL}$ 热蒸馏水中，然后称取2,6-二氯靛酚 $50~\mathrm{mg}$ 溶解在上述碳酸氢钠溶液中。冷却并用水定容至 $250~\mathrm{mL}$ ，过滤至棕色瓶内，于 $4^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 环境中保存。每次使用前，用标准抗坏血酸溶液标定其滴定度。

标定方法：准确吸取 $1\mathrm{mL}$ 抗坏血酸标准溶液于 $50~\mathrm{mL}$ 锥形瓶中，加入 $10~\mathrm{mL}$ 偏磷酸溶液或草酸溶液，摇匀，用2,6-二氯靛酚溶液滴定至粉红色，保持 $15\mathrm{s}$ 不褪色为止。同时另取 $10~\mathrm{mL}$ 偏磷酸溶液或草酸溶液做空白试验。2,6-二氯靛酚溶液的滴定度按式(3)计算：

$$
T = \frac {c \times V}{V _ {1} - V _ {0}} \tag {3}
$$

式中：

$T$ ——2,6-二氯靛酚溶液的滴定度，即每毫升2,6-二氯靛酚溶液相当于抗坏血酸的毫克数，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$c$ ——抗坏血酸标准溶液的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V$ ——吸取抗坏血酸标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——滴定抗坏血酸标准溶液所消耗2，6-二氯靛酚溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白所消耗2，6-二氯靛酚溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

# 18.3 标准品

$\mathrm{L}(+)$ -抗坏血酸标准品 $(\mathrm{C}_6\mathrm{H}_8\mathrm{O}_6)$ ：纯度 $\geq 99\%$ 。

# 18.4 标准溶液的配制

$\mathrm{L}(+)$ -抗坏血酸标准溶液 $(1.000\mathrm{mg / mL})$ ：称取 $100\mathrm{mg}$ （精确至 $0.1\mathrm{mg})\mathrm{L}(+)$ -抗坏血酸标准品，溶于偏磷酸溶液或草酸溶液并定容至 $100~\mathrm{mL}$ 。该贮备液在 $2^{\circ}C\sim 8^{\circ}C$ 避光条件下可保存一周。

# 19 测定

整个检测过程应在避光条件下进行。

19.1 试液制备：称取具有代表性样品的可食部分 $100\mathrm{g}$ ，放入粉碎机中，加入 $100\mathrm{g}$ 偏磷酸溶液或草酸溶液，迅速捣成匀浆。准确称取 $10\mathrm{g} \sim 40\mathrm{g}$ 匀浆样品（精确至 $0.01\mathrm{g}$ ）于烧杯中，用偏磷酸溶液或草酸溶液将样品转移至 $100\mathrm{mL}$ 容量瓶，并稀释至刻度，摇匀后过滤。若滤液有颜色，可按每克样品加 $0.4\mathrm{g}$ 白陶土脱色后再过滤。  
19.2 滴定：准确吸取 $10\mathrm{mL}$ 滤液于 $50~\mathrm{mL}$ 锥形瓶中，用标定过的2,6-二氯靛酚溶液滴定，直至溶液呈粉红色 $15\mathrm{s}$ 不褪色为止。同时做空白试验。

# 20 结果计算

试样中 $\mathrm{L}(+)$ -抗坏血酸含量按式(4)计算：

$$
X = \frac {\left(V - V _ {0}\right) \times T \times A}{m} \times 1 0 0 \tag {4}
$$

式中：

$X$ ——试样中 $\mathrm{L}(+)$ - 抗坏血酸含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ；

$V$ ——滴定试样所消耗2,6-二氯靛酚溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白所消耗2，6-二氯靛酚溶液的体积，单位为毫升 $(\mathrm{mL})$

$T$ ——2,6-二氯靛酚溶液的滴定度，即每毫升2,6-二氯靛酚溶液相当于抗坏血酸的毫克数 $(\mathrm{mg / mL})$ ；

$A$ ——稀释倍数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 21 精密度

在重复性条件下获得的两次独立测定结果的绝对差值，在 $\mathrm{L}(+)$ -抗坏血酸含量大于 $20\mathrm{mg} / 100\mathrm{g}$ 时不得超过算术平均值的 $2\%$ 。在 $\mathrm{L}(+)$ -抗坏血酸含量小于或等于 $20\mathrm{mg} / 100\mathrm{g}$ 时不得超过算术平均值的 $5\%$ 。

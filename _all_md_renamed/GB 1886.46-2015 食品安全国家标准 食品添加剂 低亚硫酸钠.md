# 附录A

# 检验方法

# A.1 安全提示

本试验方法中使用的部分试剂具有毒性或腐蚀性，操作时应小心谨慎。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所需标准溶液、杂质标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸溶液： $1 + 1$

A.3.1.2 硫酸铜溶液： $50~\mathrm{g / L}$

A.3.1.3 高锰酸钾溶液： $3.5\mathrm{g / L}$

A.3.1.4 焦锑酸钾溶液：称取 $2\mathrm{g}$ 焦锑酸钾，加 $100~\mathrm{mL}$ 水溶解，煮沸约 $5\mathrm{min}$ 后，迅速冷却，加 $10~\mathrm{mL}$ 氢氧化钠溶液，放置 $24\mathrm{h}$ 后，过滤。

# A.3.2 鉴别方法

A.3.2.1 低亚硫酸盐的鉴别方法如下：

a）取 $10\mathrm{mL}10\mathrm{g / L}$ 试验溶液，加 $2\mathrm{mL}$ 硫酸铜溶液时，显灰黑色；  
b）取 $10\mathrm{mL}10\mathrm{g / L}$ 试验溶液，加 $1\mathrm{mL}$ 高锰酸钾溶液时，溶液的颜色随即消失。

A.3.2.2 钠盐的鉴别方法如下：

a）用铂丝蘸取盐酸烧至无色，再蘸取 $10\mathrm{g / L}$ 试验溶液，在酒精灯上燃烧呈黄色；  
b）取适量 $50\mathrm{g / L}$ 试验溶液，加入焦锑酸钾溶液，生成白色晶状沉淀（用玻璃棒摩擦试管内壁时，可加速沉淀的生成）。

# A.4 低亚硫酸钠的测定

# A.4.1 方法提要

低亚硫酸钠和中性甲醛作用，生成亚硫酸氢钠甲醛和次硫酸氢钠甲醛，次硫酸氢钠甲醛和碘作用，根据碘标准滴定溶液的消耗量确定低亚硫酸钠的含量。

# A.4.2 试剂和材料

A.4.2.1 盐酸溶液： $1 + 11$

A.4.2.2 中性甲醛溶液: $1 + 1$ 。取 $100 \mathrm{~mL}$ 甲醛溶液和 $100 \mathrm{~mL}$ 水, 置于 $400 \mathrm{~mL}$ 烧杯中, 搅拌均匀, 加数滴 $10 \mathrm{~g} / \mathrm{L}$ 酚酞指示液, 用 $100 \mathrm{~g} / \mathrm{L}$ 氢氧化钠溶液中和至溶液呈微红色, 再用盐酸溶液调节至微红色刚好褪色。

A.4.2.3 可溶性淀粉溶液： $10\mathrm{g / L}$   
A.4.2.4 碘标准滴定溶液： $c\left(\frac{1}{2}\mathrm{I}_2\right)\approx 0.1\mathrm{mol / L}$

# A.4.3 分析步骤

称取 $1\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于预先盛有 $20~\mathrm{mL}$ 中性甲醛溶液的烧杯中，搅拌至完全溶解，全部转移至 $250~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。移取 $25~\mathrm{mL}$ 该溶液于 $250~\mathrm{mL}$ 锥形瓶中，加 $4\mathrm{mL}$ 盐酸溶液，用碘标准滴定溶液滴定，近终点时加入 $3\mathrm{mL}$ 淀粉溶液，继续滴定至溶液呈浅蓝色在 $30~\mathrm{s}$ 内不消失即为终点。同时同样处理空白试验溶液。

# A.4.4 结果计算

低亚硫酸钠的质量分数 $\omega_{1}$ ，按式（A.1）计算：

$$
w _ {1} = \frac {\frac {V - V _ {0}}{1 000} \times c \times M}{m \times \frac {25}{250}} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

V ——滴定试验溶液所消耗的碘标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验所消耗的碘标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$c$ ——碘标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——低亚硫酸钠的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $\left[M\left(\frac{1}{4}\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_4\right) = 43.53\right]$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

25 ——移取试验溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

250 ——试验溶液的总体积，单位为毫升 $(\mathrm{mL})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于 $0.5\%$ 。

# A.5 乙二胺四乙酸二钠的测定

# A.5.1 方法提要

亚砷酸将铬酸根离子还原为铬离子，铬离子与乙二胺四酸二钠生成紫色络合物，可用于乙二胺四乙酸二钠的检验。

# A.5.2 试剂和材料

A.5.2.1 乙酸。  
A.5.2.2 铬酸钾溶液： $5\mathrm{g / L}$   
A.5.2.3 氢氧化钠溶液： $25\mathrm{g / L}$   
A.5.2.4 亚砷酸溶液： $10\mathrm{g / L}$ 。称取 $1\mathrm{g}$ 亚砷酸(三氧化二砷)，加入 $30~\mathrm{mL}$ 氢氧化钠溶液，加热溶解。

冷却后，缓慢加入乙酸配至 $100~\mathrm{mL}$

# A.5.3 分析步骤

称取 $0.50\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于试管中，加入 $5\mathrm{mL}$ 水溶解、加 $2\mathrm{mL}$ 铬酸钾溶液、 $2\mathrm{mL}$ 亚砷酸溶液，在水浴中加热 $2\mathrm{min}$ 不显紫色为合格。

# A.6 重金属（以Pb计）的测定

# A.6.1 方法提要

在弱酸性溶液中，重金属离子与二价硫离子生成有色硫化物沉淀，重金属含量较低时，生成稳定的暗色悬浮液，可用于重金属的目视比色法测定。

# A.6.2 试剂和材料

A.6.2.1 盐酸。

A.6.2.2 乙酸溶液： $1 + 19$

A.6.2.3 硫化钠溶液。

A.6.2.4 铅标准溶液: $1 \mathrm{~mL}$ 溶液含铅(Pb) $0.010 \mathrm{mg}$ 。移取 $10.00 \mathrm{~mL}$ 铅标准溶液, 置于 $100 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度, 摇匀。现用现配。

A.6.2.5 标准比色溶液：移取 $2.00\mathrm{mL}$ 铅标准溶液，置于 $50~\mathrm{mL}$ 比色管中，加入 $2\mathrm{mL}$ 乙酸溶液，与试验溶液同时同样处理。

# A.6.3 分析步骤

# A.6.3.1 试验溶液A的制备

称取 $10.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $300~\mathrm{mL}$ 烧杯中，加 $60~\mathrm{mL}$ 热水溶解，加 $10~\mathrm{mL}$ 盐酸在可调电炉上蒸发至干。小心加强热至硫自燃。在残留物中加 $30~\mathrm{mL}$ 热水和 $10~\mathrm{mL}$ 盐酸，再在可调电炉上蒸发至干。将残留物加水溶解，用水稀释至约 $30~\mathrm{mL}$ ，过滤，滤液及洗涤液放入 $50~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。此溶液为试验溶液A。该溶液用于重金属含量的测定、铅含量的测定中双硫腙比色法、镉含量的测定中比色法和锌含量的测定中目视比色法。

# A.6.3.2 测定

用移液管移取 $10\mathrm{mL}$ 试验溶液A，置于 $50~\mathrm{mL}$ 比色管中，加入 $2\mathrm{mL}$ 乙酸溶液，滴加2滴硫化钠溶液，用水稀释至刻度，摇匀。放置 $5\mathrm{min}$ 后，试验溶液的颜色不得深于标准比色溶液。

# A.7 锌（Zn）的测定

# A.7.1 原子吸收分光光度法（仲裁法）

# A.7.1.1 方法提要

试料经处理后，在原子吸收光谱仪上测定吸光度。在一定浓度范围内，锌的吸光度与其含量成正比。

# A.7.1.2 试剂和材料

A.7.1.2.1 盐酸溶液： $1 + 4$

A.7.1.2.2 锌标准溶液： $1\mathrm{mL}$ 溶液含锌 $(Zn)0.05\mathrm{mg}$ 。移取 $50.00\mathrm{mL}$ 的锌标准溶液，置于 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。该溶液有效期为1个月。

A.7.1.2.3 水：符合GB/T6682规定的二级水。

# A.7.1.3 仪器和设备

原子吸收分光光度计。

# A.7.1.4 测定步骤

# A.7.1.4.1 工作曲线的绘制

用移液管分别移取 $0.00\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},3.00\mathrm{mL}$ 锌标准溶液，置于4个 $100~\mathrm{mL}$ 容量瓶中，加入 $5\mathrm{mL}$ 盐酸溶液，用水稀释至刻度，摇匀。使用乙炔-空气火焰，在波长 $213.8\mathrm{nm}$ 处将原子吸收分光光度计调至最佳工作状态，用水调零，测量吸光度。从每个标准溶液的吸光度中减去空白溶液的吸光度，以锌的质量为横坐标，对应的吸光度为纵坐标，绘制工作曲线。

# A.7.1.4.2 测定

用移液管移取 $5\mathrm{mL}$ 试验溶液B，置于 $100~\mathrm{mL}$ 容量瓶中，加入 $5\mathrm{mL}$ 盐酸溶液，用水稀释至刻度，摇匀。以下按A.10.1.4.1条从“使用乙炔-空气火焰，从测量吸光度开始”进行操作。同时同样处理空白试验溶液。从工作曲线上查出相应的锌的质量。

# A.7.1.5 结果计算

锌(Zn)的质量分数 $w_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m _ {3} - m _ {1}}{m _ {2} \times \frac {5}{100} \times 10 ^ {6}} \times 100 \% \tag{A.2}
$$

式中：

$m_{3}$ ——从工作曲线上查出的试验溶液中锌的质量的数值，单位为微克 $(\mu \mathrm{g})$ ；

$m_{1}$ ——从工作曲线上查出的空白试验溶液中锌的质量的数值，单位为微克 $(\mu \mathrm{g})$ ；

$m_{2}$ ——试料的质量的数值，单位为克 $(\mathrm{g})$ ；

5 ——移取试验溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

100——试验溶液的总体积，单位为毫升 $(\mathrm{mL})$ ；

$10^{6}$ ——转换系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果绝对差值不大于 $0.002\%$ 。

# A.7.2 目视比色法

# A.7.2.1 方法提要

在酸性介质中，锌离子与亚铁氰化钾溶液生成白色沉淀，可用于锌的目视比色法测定。

# A.7.2.2 试剂和材料

A.7.2.2.1 氨水溶液： $2 + 3$

A.7.2.2.2 盐酸溶液： $1 + 3$

A.7.2.2.3 亚铁氰化钾溶液： $100\mathrm{g / L}$ 。称取 $10\mathrm{g}$ 亚铁氰化钾，精确至 $0.01\mathrm{g}$ ，用水稀释至 $100~\mathrm{mL}$ ，该溶

液现用现配。

A.7.2.2.4 锌标准溶液： $1\mathrm{mL}$ 溶液含有锌 $0.10\mathrm{mg}$

# A.7.2.3 分析步骤

用移液管移取 $5\mathrm{mL}$ 试验溶液A，置于 $50~\mathrm{mL}$ 烧杯中，加入 $0.1\mathrm{mL}$ 氨水溶液，过滤，滤液放入 $25~\mathrm{mL}$ 比色管中，用水稀释至 $20~\mathrm{mL}$ ，加 $5\mathrm{mL}$ 盐酸溶液，摇匀，加 $0.1\mathrm{mL}$ 亚铁氰化钾溶液，放置 $15\mathrm{min}$ 其浊度不得大于标准比浊溶液。

标准比浊液是用移液管移取 $3.00\mathrm{mL}$ 锌标准溶液，置于 $25~\mathrm{mL}$ 比色管中，用水稀释至 $20~\mathrm{mL}$ ，加 $5\mathrm{mL}$ 盐酸溶液，摇匀，加 $0.1\mathrm{mL}$ 亚铁氰化钾溶液，放置 $15\mathrm{min}$ ，与试验溶液同时同样处理。

# A.8 澄清度的测定

# A.8.1 方法提要

将连二亚硫酸钠溶解于中性甲醛溶液中，在避免阳光直接照射的情况下，通过观察连二亚硫酸钠溶液中不溶物的含量来判别澄清度。

# A.8.2 试剂和材料

A.8.2.1 中性甲醛溶液： $1 + 1$

A.8.2.2 硝酸溶液： $1 + 2$

A.8.2.3 糊精溶液： $20\mathrm{g / L}$

A.8.2.4 硝酸银溶液： $20~\mathrm{g / L}$

A.8.2.5 盐酸标准溶液： $c(\mathrm{HCl})\approx 0.1\mathrm{mol / L}$

A.8.2.6 氯化物标准溶液: $1 \mathrm{~mL}$ 溶液含氯 (Cl) $0.01 \mathrm{mg}$ 。移取 $10.00 \mathrm{~mL}$ 的氯化物标准溶液, 置于 $100 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度, 摇匀。该溶液现用现配。

# A.8.3 分析步骤

# A.8.3.1 标准参比溶液（微浊）的制备

用移液管移取 $6\mathrm{mL}$ 氯化物标准溶液，置于 $25~\mathrm{mL}$ 比色管中，加水至约 $20~\mathrm{mL}$ ，加入 $1\mathrm{mL}$ 硝酸溶液， $0.2\mathrm{mL}$ 糊精溶液和 $1\mathrm{mL}$ 硝酸银溶液，用水稀释至刻度，摇匀。在避开阳光直接照射的情况下放置 $15\mathrm{min}$ 。

# A.8.3.2 测定

称取 $0.50\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于预先盛有 $10~\mathrm{mL}$ 中性甲醛溶液的 $25~\mathrm{mL}$ 比色管中，完全溶解后，用水稀释至刻度，摇匀。放置 $5\mathrm{min}$ ，在无阳光直射情况下，自上向下观察，试验溶液的浊度不得大于标准比浊溶液所示的浊度。

试验时，应控制标准参比溶液放置 $15\mathrm{min}$ 和试验溶液放置 $5\mathrm{min}$ 同时到点对比。

# A.9 甲酸盐（以HCHO计）的测定

# A.9.1 方法提要

在盐酸介质中，镁粉将甲酸盐还原为甲醛，甲醛与铬变酸发生特效反应，可用于甲酸盐的目视比色法测定。

# A.9.2 试剂和材料

A.9.2.1 镁粉。  
A.9.2.2 硫酸。  
A.9.2.3 盐酸溶液： $1 + 1$   
A.9.2.4 铬变酸溶液： $c(\mathrm{C_{10}H_6Na_2S_2O_8\bullet 2H_2O}) = 10~\mathrm{g / L}$ 。称取 $0.5\mathrm{g}$ 铬变酸，加 $50~\mathrm{mL}$ 硫酸摇混，离心分离，使用上层澄清液，临用时配制。  
A.9.2.5 甲醛标准溶液: $1 \mathrm{~mL}$ 溶液含甲醛 $0.002 \mathrm{mg}$ 。移取 $2.00 \mathrm{~mL}$ 甲醛标准溶液 $(1 \mathrm{~g} / \mathrm{L})$ , 置于 $1000 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度, 摇匀。现用现配。  
A.9.2.6 标准比色溶液：移取 $2.50~\mathrm{mL}$ 甲醛溶液，加入 $5\mathrm{mL}$ 盐酸溶液，接着两次加少量镁粉，加至约 $0.3\mathrm{g}$ ，几乎看不出气泡产生后，盖上表面皿，放置 $2\mathrm{h}$ 。全部转移至 $50~\mathrm{mL}$ 比色管中，加 $2\mathrm{mL}$ 硫酸， $0.5\mathrm{mL}$ 铬变酸，在沸水浴上加热 $10\mathrm{min}$ 后，用水稀释至刻度，摇匀。

# A.9.3 分析步骤

称取 $0.100\mathrm{g}\pm 0.001\mathrm{g}$ 试样，置于 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。用移液管移取 $10~\mathrm{mL}$ 该溶液，置于烧杯中，加入 $5\mathrm{mL}$ 盐酸溶液，接着两次加少量镁粉，加至约 $0.3\mathrm{g}$ ，几乎看不出气泡产生后，盖上表面皿，放置 $2\mathrm{h}$ 。全部转移至 $50~\mathrm{mL}$ 比色管中，加 $2\mathrm{mL}$ 硫酸， $0.5\mathrm{mL}$ 铬变酸，在沸水浴上加热 $10\mathrm{min}$ 后，用水稀释至刻度，摇匀。其颜色不得深于标准比色溶液。

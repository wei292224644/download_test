# 1 范围

本标准适用于维生素C经磷酸化后和氧化镁反应，制得的食品营养强化剂维生素C磷酸酯镁。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

\mathrm {C} _ {1 2} \mathrm {H} _ {1 2} \mathrm {O} _ {1 8} \mathrm {P} _ {2} \mathrm {M g} _ {3} \cdot 1 0 \mathrm {H} _ {2} \mathrm {O}
$$

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度应为分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 盐酸溶液： $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$   
A.2.1.2 氢氧化钠溶液： $c(\mathrm{NaOH}) = 0.1\mathrm{mol / L}$   
A.2.1.3 镁溶液：称取 $50\mathrm{mg}$ 镁，加入氢氧化钠溶液 $(2\mathrm{mol} / \mathrm{L})$ ，溶解后定容至 $100~\mathrm{mL}$ 。  
A.2.1.4 三氯化铁溶液：称取 $10\mathrm{g}$ 三氯化铁，加入盐酸溶液，溶解后定容至 $100~\mathrm{mL}$

# A.2.2 鉴别方法

A.2.2.1 称取约 $50\mathrm{mg}$ 试样，加入 $5\mathrm{mL}$ 盐酸溶液，再加三氯化铁溶液2滴，试样液应显酒红色。  
A.2.2.2取适量试样，加入盐酸溶液制成每毫升含 $20~\mu \mathrm{g}$ 试样的溶液，使用分光光度计测定，该溶液在 $237~\mathrm{nm}$ 处有最大吸收峰；另取适量试样，加入氢氧化钠溶液制成每毫升含 $20~\mu \mathrm{g}$ 试样的溶液，使用分光光度计测定，该溶液在 $261~\mathrm{nm}$ 处有最大吸收峰。  
A.2.2.3 称取约 $50 \mathrm{mg}$ 试样, 加水配成约 $0.1 \mathrm{~mol} / \mathrm{L}$ 的试样液, 加入氢氧化钠溶液调节至 $\mathrm{pH} 10$ , 再加入镁溶液 2 滴, 应出现蓝色沉淀。

# A.3 维生素C磷酸酯镁含量的测定

# A.3.1 高效液相色谱法

# A.3.1.1 试剂和材料

A.3.1.1.1 乙酸钠。  
A.3.1.1.2 乙二胺四乙酸二钠。  
A.3.1.1.3 正己胺。  
A.3.1.1.4 冰乙酸。  
A.3.1.1.5 甲醇溶液： $2 + 98$   
A.3.1.1.6 维生素C磷酸酯镁对照品：纯度 $\geq 98\%$

# A.3.1.2 仪器和设备

A.3.1.2.1 高效液相色谱仪：配紫外检测器。  
A.3.1.2.2 色谱柱：以十八烷基硅烷键合硅胶为填充剂的不锈钢柱 $(\phi 4.6 \mathrm{~mm} \times 15 \mathrm{~cm}$ ，填料粒径 $3 \mu \mathrm{m}$ 或其他等效的色谱柱。

# A.3.1.3 参考色谱条件

A.3.1.3.1 流动相：称取 $6.56\mathrm{g}$ 乙酸钠、 $0.0375\mathrm{g}$ 乙二胺四乙酸二钠和 $0.37\mathrm{mL}$ 正己胺，置于 $1000\mathrm{mL}$ 容量瓶中，加甲醇溶液使溶解后定容。再用冰乙酸将该溶液调节 $\mathrm{pH}$ 至5.0。

A.3.1.3.2 流速： $0.8\mathrm{mL / min}$

A.3.1.3.3 进样量： $5\mu \mathrm{L}$

A.3.1.3.4 检测波长： $200\mathrm{nm}$

# A.3.1.4 分析步骤

# A.3.1.4.1 试样溶液的制备

称取 $0.6\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ) 试样, 用流动相溶解, 移入 $200\mathrm{mL}$ 容量瓶中, 定容至刻度。色谱分析前用 $0.45\mu \mathrm{m}$ 微孔滤膜过滤, 滤液备用。

# A.3.1.4.2 对照溶液的制备

称取 $0.6\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ) 维生素C磷酸酯镁对照品，用流动相溶解，移入 $200\mathrm{mL}$ 容量瓶中，定容至刻度。色谱分析前用 $0.45\mu \mathrm{m}$ 微孔滤膜过滤，滤液备用。

# A.3.1.4.3 测定

在A.3.1.3参考色谱条件下，吸取试样溶液和对照溶液分别注入色谱仪，记录所得的试样溶液中维生素C磷酸酯镁的峰面积和对照溶液中维生素C磷酸酯镁的峰面积。

# A.3.1.5 结果计算

维生素C磷酸酯镁含量的质量分数 $w_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {A _ {\mathrm {i}} m _ {\mathrm {s}}}{A _ {\mathrm {s}} m _ {\mathrm {i}}} \times 100 \% \tag{A.1}
$$

式中：

$A_{\mathrm{i}}$ ——试样溶液中维生素C磷酸酯镁的峰面积；

$m_{\mathrm{s}}$ ——对照溶液中维生素C磷酸酯镁的质量，单位为克 $(\mathrm{g})$ ；

$A_{\mathrm{s}}$ ——对照溶液中维生素C磷酸酯镁的峰面积；

$m_{\mathrm{i}}$ ——试样溶液中维生素C磷酸酯镁的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。

# A.3.2 紫外分光光度法

# A.3.2.1 试剂和材料

盐酸溶液： $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$ 。

# A.3.2.2 仪器和设备

紫外分光光度计。

# A.3.2.3 分析步骤

称取约 $0.1\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，加入盐酸溶液溶解，置于 $100\mathrm{mL}$ 容量瓶中，定容至刻度，摇匀。从上述溶液中量取 $1.0\mathrm{mL}$ 置于 $50\mathrm{mL}$ 容量瓶中，再加入盐酸溶液稀释定容，摇匀。用紫外分光光

度计在 $237\mathrm{nm}$ 下测定吸光度。同时做空白试验。

# A.3.2.4 结果计算

维生素C磷酸酯镁含量的质量分数 $w_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {A \times 5000}{0.7627 \times m _ {1} \times 324} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$A$ ——试样液的吸光度；

5000 ——样品的稀释倍数；

0.7627——摩尔质量换算系数；

$m_{1}$ ——试样的质量，单位为克(g)；

324 ——试样液的百分吸收系数。

试验结果以平行测定结果的算术平均值为准。

# A.4 氯化物（以Cl计）的测定

# A.4.1 试剂和材料

A.4.1.1 硝酸溶液： $1 + 4$

A.4.1.2 硝酸银溶液： $17\mathrm{g / L}$

A.4.1.3 氯化钠标准溶液：称取 $1.65\mathrm{g}$ 氯化钠，置于 $1000~\mathrm{mL}$ 容量瓶中，加水使其溶解并稀释至刻度，摇匀( $1\mathrm{mL}$ 相当于 $1\mathrm{mg}$ 的 $\mathrm{Cl}$ )。

# A.4.2 仪器和设备

$50~\mathrm{mL}$ 纳氏比色管。

# A.4.3 分析步骤

# A.4.3.1 试样溶液的制备

称取 $1\mathrm{g}$ 试样置于纳氏比色管中，加水溶解至 $25~\mathrm{mL}$ ，再加 $10~\mathrm{mL}$ 硝酸溶液，加水至体积约 $40~\mathrm{mL}$ ，摇匀。

# A.4.3.2 对照溶液的制备

量取 $3.5\mathrm{mL}$ 氯化钠标准溶液置于纳氏比色管中，加水溶解至 $25~\mathrm{mL}$ ，再加 $10~\mathrm{mL}$ 硝酸溶液，加水至体积约 $40~\mathrm{mL}$ ，摇匀。

# A.4.3.3 测定

在试样液与对照溶液中分别加入 $1.0\mathrm{mL}$ 硝酸银溶液，用水稀释定容至 $50~\mathrm{mL}$ ，摇匀，在暗处放置 $5\mathrm{min}$ ，同置黑色背景上，从比色管上方向下观察所产生的浑浊。

# A.4.4 结果判定

试样溶液的浑浊度浅于对照溶液，即为通过试验 $(\leqslant 0.35\%)$ 。

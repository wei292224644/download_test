# 前言

本标准代替GB/T23813—2009《食品中1,2-丙二醇的测定》、NY/T1662—2008《乳与乳制品中1，2-丙二醇的测定气相色谱法》。

本标准与GB/T23813—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中1,2-丙二醇的测定”；  
——增加了奶片的前处理方法；  
——修改了乳粉、黄油、奶油、液体乳的前处理方法；  
——增加了方法的定量限。

# 食品安全国家标准

# 食品中 1,2-丙二醇的测定

# 1 范围

本标准规定了用气相色谱法和气相色谱-质谱法测定食品中1,2-丙二醇含量的方法。

本标准第一法适用于糕点、膨化食品、奶油、干酪、豆制品、奶片、生湿面制品、冷冻饮品、液体乳、植物蛋白饮料、乳粉、黄油、奶油中1,2-丙二醇含量的测定；第二法适用于糕点、膨化食品、干酪、豆制品、奶片、生湿面制品中1,2-丙二醇含量的测定。

# 第一法 气相色谱法

# 2 原理

试样中1,2-丙二醇用无水乙醇提取，提取液过滤后，采用气相色谱法测定。保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 无水乙醇 $\mathrm{(C_2H_5OH)}$   
3.1.2 海砂。  
3.1.3 乙腈 $\mathrm{(CH_3CN)}$ 。  
3.1.4 正己烷 $\left[\mathrm{CH}_{3}\left(\mathrm{CH}_{2}\right)_{4} \mathrm{CH}_{3}\right]$ 。

# 3.2 试剂配制

3.2.1 正己烷饱和乙腈溶液：取相同体积的乙腈和正己烷，置于分液漏斗中，振荡，静置分层，取下层溶液。  
3.2.2 乙腈饱和正己烷溶液：取相同体积的乙腈和正己烷，置于分液漏斗中，振荡，静置分层，取上层溶液。

# 3.3 1,2-丙二醇标准品 $\left(\mathrm{CH}_{2} \mathrm{OHCHOHCH}_{3}\right)$

纯度 $\geqslant 99.9\%$

# 3.4 标准溶液配制

3.4.1 1,2-丙二醇标准储备溶液 $(10.0\mathrm{mg / mL})$ ：准确称取1，2-丙二醇标准样品 $1\mathrm{g}$ （精确到 $0.0001\mathrm{g}$ ），用无水乙醇溶解并转移至 $100~\mathrm{mL}$ 容量瓶中，定容至刻度，此溶液1，2-丙二醇质量浓度为 $10.0\mathrm{mg / mL}$ 。

贮存于 $4^{\circ} \mathrm{C}$ 冰箱中，有效期3个月。

3.4.2 1,2-丙二醇标准系列工作溶液：准确吸取1，2-丙二醇标准储备溶液，用无水乙醇逐级稀释，配制成质量浓度为 $0.00\mu \mathrm{g / mL},2.00\mu \mathrm{g / mL},5.00\mu \mathrm{g / mL},10.0\mu \mathrm{g / mL},20.0\mu \mathrm{g / mL},50.0\mu \mathrm{g / mL}$ 的1，2-丙二醇标准系列溶液。临用时配制。

# 4 仪器和设备

4.1 气相色谱仪(GC)：配有氢火焰离子化检测器(FID)。  
4.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
4.3 粉碎机。  
4.4 旋涡混合器。  
4.5 回旋振荡器。  
4.6 离心机：转速 $\geq 8000\mathrm{r / min}$

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 糕点、膨化食品、干酪、豆制品、奶片

试样用粉碎机粉碎，准确称取混匀试样 $5\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）至 $100\mathrm{mL}$ 具塞锥形瓶中，加入 $50.0\mathrm{mL}$ 无水乙醇，涡旋混匀 $2\mathrm{min}$ 后振荡提取 $40\mathrm{min}$ ，静置 $1\mathrm{h}$ 后用 $0.45\mu \mathrm{m}$ 有机相滤膜过滤，所得滤液进气相色谱仪分析。

# 5.1.2 生湿面制品

试样用粉碎机粉碎，准确称取混匀试样 $2\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于研钵中，加适量的海砂（海砂与样品质量比约 $3:1$ 到 $4:1$ )研磨成干粉状，全部转移至 $100~\mathrm{mL}$ 具塞锥形瓶中，加入 $50.0~\mathrm{mL}$ 无水乙醇，涡旋混匀 $2\mathrm{min}$ 后振荡提取 $40\mathrm{min}$ ，静置 $1\mathrm{h}$ 后用 $0.45\mu \mathrm{m}$ 有机相滤膜过滤，所得滤液进气相色谱仪分析。

# 5.1.3 冷冻饮品

固体试样置于干燥烧杯中解冻，待融化后用玻棒捣碎并搅拌均匀。液体试样放至室温摇匀。准确称取混匀试样 $10\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 具塞比色管中，用无水乙醇定容，涡旋混匀 $2\mathrm{min}$ ，静置 $1\textrm{h}$ （必要时以 $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ )后用 $0.45\mu \mathrm{m}$ 有机相滤膜过滤，所得滤液进气相色谱仪分析。

# 5.1.4 液体乳、植物蛋白饮料

准确称取混匀试样 $10\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 具塞比色管中，用无水乙醇定容，涡旋混匀 $2\mathrm{min}$ 静置 $1\textrm{h}$ （必要时以 $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ )后用 $0.45\mu \mathrm{m}$ 有机相滤膜过滤，所得滤液进气相色谱仪分析。

# 5.1.5 乳粉

准确称取混匀试样 $2\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 具塞比色管中，用 $8\mathrm{mL}40^{\circ}\mathrm{C}$ 水溶解混匀，用无水乙醇定容，涡旋混匀 $2\mathrm{min}$ ，静置 $1\mathrm{h}$ （必要时以 $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ )后用 $0.45\mu \mathrm{m}$ 有机相滤膜过滤，所得滤液进气相色谱仪分析。

# 5.1.6 黄油、奶油

准确称取混匀试样 $2\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 具塞离心管中，用 $6\mathrm{mL}$ 正己烷溶解混匀，加入 $20~\mathrm{mL}$ 正己烷饱和乙腈溶液，超声波处理 $30\mathrm{min}$ 后静置分层，转移上层溶液并用 $20~\mathrm{mL}$ 正己烷饱和乙腈溶液再次萃取，合并乙腈层于 $100~\mathrm{mL}$ 离心管中，在离心管中加入 $2\mathrm{mL}$ 乙腈饱和正己烷溶液，剧烈振荡 $3\mathrm{min}$ ，以 $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去上层正己烷层，再加入 $2\mathrm{mL}$ 乙腈饱和正己烷溶液，剧烈振荡 $3\mathrm{min}$ ，以 $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去上层正己烷层，下层转移至 $50~\mathrm{mL}$ 容量瓶用乙腈定容，用 $0.45\mu \mathrm{m}$ 有机相滤膜过滤，所得滤液进气相色谱仪分析。

# 5.2 仪器参考条件

5.2.1 色谱柱：键合/交联聚乙二醇固定相石英毛细管色谱柱， $60\mathrm{m}\times 0.25\mathrm{mm},0.25\mu \mathrm{m}$ ，或相当色谱柱。  
5.2.2 载气：高纯氮；恒流模式，柱流速 $1.0\mathrm{mL / min}$   
5.2.3 采用程序升温：柱初始温度 $80^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $20^{\circ}\mathrm{C / min}$ 速率升温至 $160^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$ ，再以 $15^{\circ}\mathrm{C / min}$ 速率升温至 $220^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ 。  
5.2.4 进样口温度： $230^{\circ}C$   
5.2.5 检测器温度： $240^{\circ}\mathrm{C}$   
5.2.6 氢气流量： $40~\mathrm{mL / min}$   
5.2.7 空气流量： $350~\mathrm{mL / min}$   
5.2.8 进样量： $1\mu \mathrm{L}$   
5.2.9 进样方式：分流进样，分流比 $10:1$

# 5.3 标准曲线的制作

将标准系列工作液分别注入气相色谱仪中，测定相应的1,2-丙二醇的色谱峰面积，以标准工作液的质量浓度为横坐标，以色谱峰的峰面积为纵坐标，绘制标准曲线。

# 5.4 试样溶液的测定

将试样溶液注入气相色谱仪中，得到相应的1,2-丙二醇的色谱峰面积，根据标准曲线得到待测液中1,2-丙二醇的质量浓度。

1,2-丙二醇的标准气相色谱图参见图A.1。

# 6 分析结果的表述

试样中1，2-丙二醇的含量按式(1)计算：

$$
X = c \times \frac {V}{m \times 1 0 0 0} \times \frac {1 0 0 0}{1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中1,2-丙二醇的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$c$ ——从标准工作曲线上查出的试样溶液中1,2-丙二醇的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$

$V$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克(g)；

1000——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当取样量为 $5\mathrm{g}$ 时，本方法检出限为 $0.01\mathrm{g / kg}$ ，定量限为 $0.03\mathrm{g / kg}$ 。

# 第二法 气相色谱-质谱法

# 9 原理

试样中1,2-丙二醇用无水乙醇提取，提取液过滤后，用气相色谱-质谱联用仪测定，采用选择离子监测扫描模式(SIM)，用化合物的保留时间和特征碎片的质荷比定性，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 10.1 试剂

10.1.1 无水乙醇 $\mathrm{(C_2H_5OH)}$   
10.1.2 海砂。  
10.1.3 无水硫酸钠 $\left(\mathrm{Na}_{2} \mathrm{SO}_{4}\right)$ ：于 $650^{\circ} \mathrm{C}$ 灼烧 $4 \mathrm{~h}$ ，储于密闭干燥器中备用。

# 10.2 1,2-丙二醇标准品 $\left(\mathrm{CH}_{2} \mathrm{OHCHOHCH}_{3}\right)$

纯度 $\geqslant 99.9\%$

# 10.3 标准溶液配制

10.3.1 1,2-丙二醇标准储备溶液 $(10.0\mathrm{mg / mL})$ ：准确称取1，2-丙二醇标准样品 $1\mathrm{g}$ （精确到 $0.0001\mathrm{g}$ ），用无水乙醇溶解并转移至 $100~\mathrm{mL}$ 容量瓶中，定容至刻度，此溶液1，2-丙二醇质量浓度为 $10.0\mathrm{mg / mL}$ 。贮存于 $4^{\circ}C$ 冰箱中，有效期3个月。  
10.3.2 1,2-丙二醇标准系列工作溶液：准确吸取1，2-丙二醇标准储备液，用无水乙醇逐级稀释，配制成质量浓度为 $0.0\mu \mathrm{g / mL},0.200\mu \mathrm{g / mL},0.500\mu \mathrm{g / mL},1.00\mu \mathrm{g / mL},2.00\mu \mathrm{g / mL},10.0\mu \mathrm{g / mL}$ 的1，2-丙二醇标准系列溶液。临用时配制。

# 11 仪器和设备

11.1 气相色谱-质谱联用仪(GC-MS)：EI源。  
11.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
11.3 粉碎机。  
11.4 旋涡混合器。

11.5 回旋振荡器。

# 12 分析步骤

# 12.1 试样制备

# 12.1.1 糕点、膨化食品、干酪、豆制品、奶片

试样用粉碎机粉碎，准确称取混匀试样 $5\mathrm{g}$ （精确到 $0.01\mathrm{g}$ )至 $100~\mathrm{mL}$ 具塞锥形瓶中，加入 $50~\mathrm{mL}$ 无水乙醇，涡旋混匀 $2\mathrm{min}$ 后振荡提取 $40\mathrm{min}$ ，静置 $1\textrm{h}$ 后通过无水硫酸钠(约 $2\mathrm{g}$ )过滤，所得滤液进气相色谱-质谱联用仪分析。

# 12.1.2 生湿面制品

试样用粉碎机粉碎，准确称取混匀试样 $2\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于研钵中，加适量的海砂（海砂与样品质量比约 $3:1$ 到 $4:1$ )研磨成干粉状，全部转移至 $100~\mathrm{mL}$ 具塞锥形瓶中，加入 $50~\mathrm{mL}$ 无水乙醇，涡旋混匀 $2\mathrm{min}$ 后振荡提取 $40\mathrm{min}$ ，静置 $1\mathrm{h}$ 后通过无水硫酸钠(约 $2\mathrm{g}$ )过滤，所得滤液进气相色谱-质谱联用仪分析。

# 12.2 仪器参考条件

12.2.1 色谱柱：键合/交联聚乙二醇固定相石英毛细管色谱柱， $60\mathrm{m}\times 0.25\mathrm{mm},0.25\mu \mathrm{m}$ ，或相当色谱柱。  
12.2.2 载气：高纯氦；恒流模式，柱流速 $1.0~\mathrm{mL / min}$   
12.2.3 采用程序升温，柱初始温度 $80^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $20^{\circ}\mathrm{C / min}$ 速率升温至 $160^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$ ，再以 $15^{\circ}\mathrm{C / min}$ 速率升温至 $220^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ 。  
12.2.4 进样口温度： $230^{\circ}C$   
12.2.5 检测器温度： $240^{\circ}C$   
12.2.6 进样量： $1\mu \mathrm{L}$   
12.2.7 进样方式：分流进样，分流比 $10:1$ 。  
12.2.8 电离方式：电子轰击源（EI）。  
12.2.9 电离能量： $70\mathrm{eV}$ 。  
12.2.10 离子源温度： $230^{\circ}C$   
12.2.11 四极杆温度： $150^{\circ}C$   
12.2.12 溶剂延迟： $8\mathrm{min}$   
12.2.13 扫描方式：采用选择离子监测扫描模式(SIM)，选择离子 $m / z 31, m / z 45, m / z 61$ ，定量离子 $m / z 45$ 。

# 12.3 定性

在12.2仪器条件下，试样待测液和1,2-丙二醇标准品的目标化合物在相同保留时间处 $(\pm 0.5\%)$ 出现，并且对应质谱碎片离子的质荷比与标准品的质谱图一致，可定性目标分析物。

# 12.4 标准曲线的制作

将标准系列工作液分别注入气相色谱-质谱联用仪中，测定相应的1,2-丙二醇的色谱峰面积，以标准工作液的质量浓度为横坐标，以 $m / z$ 45基峰的峰面积为纵坐标，绘制标准曲线。

# 12.5 试样溶液的测定

将试样溶液注入气相色谱-质谱联用仪中，得到相应的1,2-丙二醇的 $m / z$ 45基峰峰面积，根据标准曲线得到待测液中1,2-丙二醇的质量浓度。

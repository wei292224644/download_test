# 前言

本标准代替GB25577—2010《食品安全国家标准 食品添加剂二氧化钛》。

本标准与GB25577—2010相比，主要变化如下：

——修改了范围；  
——删除了感官要求中的气味要求及检验方法；  
——修改了砷指标值；  
——修改了水溶物检验方法；  
——删除了重金属指标及检验方法；  
——增加了铅指标及检验方法；  
——将砷的检验方法修改为GB5009.76或GB5009.11。

本标准适用于以钛铁矿为原料采用硫酸法或以金红石矿(或富集的钛铁矿)为原料采用氯化法制得的二氧化钛，再经过精制提纯生产的食品添加剂二氧化钛。

# 检验方法

警示：检验方法中使用的部分试剂具有毒性、腐蚀性及易燃性，操作者应小心谨慎！必要时，应在通风橱中进行。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。使用易燃品时，严禁使用明火加热。

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603之规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

# A.2.1.1 硫酸。

A.2.1.2 $30\%$ 过氧化氢。

# A.2.2 鉴别方法

称取约 $0.5\mathrm{g}$ 试样，置于 $250~\mathrm{mL}$ 烧杯中，加 $5\mathrm{mL}$ 硫酸，慢慢加热，直至硫酸出现烟雾，冷却。小心地用水慢慢稀释到 $100~\mathrm{mL}$ ，过滤。然后取 $5\mathrm{mL}$ 滤液，加入数滴 $30\%$ 过氧化氢，滤液出现橙红色。

# A.3 二氧化钛含量的测定

称取 $0.20\mathrm{g}\pm 0.01\mathrm{g}$ 按A.4干燥后的试样，精确至 $0.0001\mathrm{g}$ ，其他操作同GB/T1706—2006的7.1。

# A.4 干燥减量的测定

同GB/T1706—2006的7.2，干燥 $3\mathrm{h}$

# A.5 灼烧减量（以干基计）的测定

# A.5.1 仪器和设备

高温炉：控温范围为 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 。

# A.5.2 分析步骤

称取 $2\mathrm{g}$ 按A.4干燥后的试样，精确至 $0.0002\mathrm{g}$ ，置于 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 下灼烧至质量恒定的瓷坩埚中，于高温炉中 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 下灼烧至质量恒定。

# A.5.3 结果计算

灼烧减量的质量分数 $w_{1}$ 按式(A.1)计算。

$$
w _ {1} = \frac {m _ {1} - m _ {2}}{m _ {1}} \times 100 \% \tag{A.1}
$$

式中：

$m_{1}$ ——称取A.4干燥后试样的质量，单位为克 $(\mathrm{g})$

$m_{2}$ ——灼烧后试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.02\%$ 。

# A.6 盐酸溶解物的测定

# A.6.1 试剂和材料

盐酸溶液： $1 + 19$

# A.6.2 仪器和设备

A.6.2.1 蒸发皿： $150~\mathrm{mL}$

A.6.2.2 高温炉：控温范围为 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$

A.6.2.3 中速定性滤纸。

# A.6.3 分析步骤

称取约 $5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 烧杯中，加 $100~\mathrm{mL}$ 盐酸溶液，混匀。水浴上加热至沸腾 $30\mathrm{min}$ ，用双层中速定性滤纸过滤，滤渣用盐酸溶液洗涤3次，每次用 $10~\mathrm{mL}$ 。将滤液和洗液收集于 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 下灼烧至质量恒定的蒸发皿中，在水浴上蒸干后，再于 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 下灼烧至质量恒定。

# A.6.4 结果计算

盐酸溶解物的质量分数 $w_{2}$ 按式(A.2)计算。

$$
w _ {2} = \frac {m _ {4}}{m _ {3}} \times 100 \% \tag{A.2}
$$

式中：

$m_{4}$ ——残渣的质量，单位为克 $(\mathrm{g})$ ；

$m_{3}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.01\%$ 。

# A.7 水溶物的测定

# A.7.1 试剂和材料

A.7.1.1 碳酸铵溶液： $40~\mathrm{g / L}$

A.7.1.2 水：新鲜二次蒸馏水或去离子水， $\mathrm{pH}$ 为 $6 \sim 7$ 。

A.7.1.3 中速定性滤纸。

# A.7.2 仪器和设备

A.7.2.1 蒸发皿： $150~\mathrm{mL}$   
A.7.2.2 电热恒温干燥箱：控温范围为 $105\,^{\circ}\mathrm{C} \pm 2\,^{\circ}\mathrm{C}$ 。

# A.7.3 分析步骤

称取约 $10\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $300~\mathrm{mL}$ 烧杯中，加 $150~\mathrm{mL}$ 水，用电炉煮沸并保持 $5\mathrm{min}$ ，冷却至室温后，在不断搅拌下加入 $25~\mathrm{mL}$ 碳酸铵溶液，静置 $10\mathrm{min}$ ，如悬浊液无明显沉淀现象，补加 $25~\mathrm{mL}$ 碳酸铵溶液。转移至 $250~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀后，倒入 $250~\mathrm{mL}$ 烧杯中，放置约 $2\mathrm{h}$ 至上层清液澄清后，将上层清液经过双层中速定性滤纸干过滤。弃去 $10~\mathrm{mL} \sim 20~\mathrm{mL}$ 初滤液，移取 $100.0~\mathrm{mL}$ 澄清滤液，转移至 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 下干燥至质量恒定的蒸发皿中，将蒸发皿置于水浴上蒸发至干，再于 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 下干燥至质量恒定。

# A.7.4 结果计算

水溶物的质量分数 $w_{3}$ 按式(A.3)计算。

$$
w_{3} = \frac{m_{6}}{m_{5}\times\frac{100}{250}}\times 100\% \quad \dots \tag{A.3}
$$

式中：

$m_{6}$ ——残余物的质量，单位为克 $(\mathrm{g})$ ；

$m_{5}$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.01\%$ 。

# A.8 铅(Pb)的测定

按GB5009.75或GB5009.12规定的方法进行测定，试验中所用水为GB/T6682规定的二级水。

# A.9 砷（As）的测定

称取 $10.0\mathrm{g}\pm 0.1\mathrm{g}$ 试样，置于 $250~\mathrm{mL}$ 烧杯中，加 $50~\mathrm{mL}$ 盐酸溶液 $(1 + 19)$ ，加热至沸，再缓缓地煮沸 $15\mathrm{min}$ 。用离心分离使不溶物沉降。用定性滤纸过滤上层清液，将用过的烧杯和残渣用热水洗涤3次，每次用热水 $10~\mathrm{mL}$ ，用同一滤纸过滤。再用 $10~\mathrm{mL}\sim 15~\mathrm{mL}$ 的热水冲洗滤纸，将洗液和滤液合并，冷却后转移至 $100~\mathrm{mL}$ 容量瓶中，加水至刻度，摇匀。

移取 $25.0\mathrm{mL}$ 试样溶液，按GB5009.76或GB5009.11规定的方法进行测定，试验中所用水为 $\mathrm{GB / T~6682}$ 规定的二级水。

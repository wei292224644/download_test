# 食品添加剂 3-甲硫基丙醛

# 1 范围

本标准适用于由丙烯醛为原料制得的食品添加剂3-甲硫基丙醛。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

3-甲硫基丙醛

# 2.2 分子式

$\mathrm{C_4H_8OS}$

104.17(按2007年国际相对原子质量)

# 3-甲硫基丙醛含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。

A.1.2 柱：毛细管柱。

A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定执行。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.53\mathrm{mm}$ 。  
B.2.2 固定相：(5%苯基)甲基聚硅氧烷。  
B.2.3 膜厚： $1.2\mu \mathrm{m}$   
B. 2.4 色谱炉温度: $80^{\circ} \mathrm{C}$ 恒温 $2.5 \mathrm{~min}$ , 然后线性升温从 $80^{\circ} \mathrm{C} \sim 240^{\circ} \mathrm{C}$ , 速率 $10^{\circ} \mathrm{C} / \mathrm{min}$ 。  
B.2.5 进样口温度： $250^{\circ}C$   
B.2.6 检测器温度： $250^{\circ}C$   
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 柱前压： $20\mathrm{kPa}$   
B. 2.10 进样量: $0.2 \mu \mathrm{L}$ 。  
B.2.11 分流比： $1 / 100$

# 前言

本标准代替GB5413.35—2010《食品安全国家标准 婴幼儿食品和乳品中 $\beta$ -胡萝卜素的测定》、GB/T5009.83—2003《食品中胡萝卜素的测定》和NY/T82.15—1988《果汁测定方法 $\beta$ -胡萝卜素的测定》。

本标准与GB5413.35—2010相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中胡萝卜素的测定”；  
——增加了普通食品的前处理方法；  
——增加了需要区分 $\alpha$ -胡萝卜素、 $\beta$ -胡萝卜素的色谱条件；  
——修改了胡萝卜素的结果表达。

# 食品安全国家标准

# 食品中胡萝卜素的测定

# 1 范围

本标准规定了食品中胡萝卜素的测定方法。

本标准色谱条件一适用于食品中 $\alpha$ -胡萝卜素、 $\beta$ -胡萝卜素及总胡萝卜素的测定，色谱条件二适用于食品中 $\beta$ -胡萝卜素的测定。

# 2 原理

试样经皂化使胡萝卜素释放为游离态，用石油醚萃取二氯甲烷定容后，采用反相色谱法分离，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 $\alpha$ -淀粉酶：酶活力 $\geqslant 1.5 \mathrm{U} / \mathrm{mg}$ 。  
3.1.2 木瓜蛋白酶：酶活力 $\geqslant 5\mathrm{U / mg}$   
3.1.3 氢氧化钾(KOH)。  
3.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.5 抗坏血酸 $\mathrm{(C_6H_8O_6)}$ 。  
3.1.6 石油醚：沸程 $30^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$   
3.1.7 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.8 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。  
3.1.9 三氯甲烷 $\mathrm{(CHCl_3)}$ ：色谱纯。  
3.1.10 甲基叔丁基醚 $\left[\mathrm{CH}_{3} \mathrm{OC}\left(\mathrm{CH}_{3}\right)_{3}\right]$ ：色谱纯。  
3.1.11 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：色谱纯。  
3.1.12 无水乙醇 $\mathrm{(C_2H_6O)}$ ：优级纯。  
3.1.13 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
3.1.14 2,6-二叔丁基-4-甲基苯酚 $\mathrm{(C_{15}H_{24}O,BHT)}$

# 3.2 试剂配制

氢氧化钾溶液：称固体氢氧化钾 $500\mathrm{g}$ ，加入 $500~\mathrm{mL}$ 水溶解。临用前配制。

# 3.3 标准品

3.3.1 $\alpha$ -胡萝卜素（ $\mathrm{C_{40}H_{56}}$ ，CAS号：7488-99-5）：纯度 $\geq 95\%$ ，或经国家认证并授予标准物质证书的标

准物质。

3.3.2 $\beta$ -胡萝卜素 $(\mathrm{C}_{40}\mathrm{H}_{56},\mathrm{CAS}$ 号：7235-40-7）：纯度 $\geqslant 95\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 $\alpha$ -胡萝卜素标准储备液 $(500\mu \mathrm{g} / \mathrm{mL})$ ：准确称取 $\alpha$ -胡萝卜素标准品 $50\mathrm{mg}$ （精确到 $0.1\mathrm{mg}$ ），加入 $0.25\mathrm{g}$ BHT，用二氯甲烷溶解，转移至 $100\mathrm{mL}$ 棕色容量瓶中定容至刻度。于 $-20^{\circ}\mathrm{C}$ 以下避光储存，使用期限不超过3个月。标准储备液用前需进行标定，具体操作见附录A。  
3.4.2 $\alpha$ -胡萝卜素标准中间液（ $100~\mu \mathrm{g / mL}$ ）：由 $\alpha$ -胡萝卜素标准储备液中准确移取 $10.0~\mathrm{mL}$ 溶液于 $50~\mathrm{mL}$ 棕色容量瓶中，用二氯甲烷定容至刻度。  
3.4.3 $\beta$ -胡萝卜素标准储备液 $(500~\mu \mathrm{g / mL})$ ：准确称取 $\beta$ -胡萝卜素标准品 $50~\mathrm{mg}$ （精确到 $0.1~\mathrm{mg}$ )，加入 $0.25\mathrm{g}$ BHT，用二氯甲烷溶解，转移至 $100~\mathrm{mL}$ 棕色容量瓶中定容至刻度。于 $-20^{\circ}\mathrm{C}$ 以下避光储存，使用期限不超过3个月。标准储备液用前需进行标定，具体操作见附录A。

注： $\beta$ -胡萝卜素标准品主要为全反式(all-E) $\beta$ -胡萝卜素，在储存过程中受到温度、氧化等因素的影响，会出现部分全反式 $\beta$ -胡萝卜素异构化为顺式 $\beta$ -胡萝卜素的现象，如9-顺式(9Z)- $\beta$ -胡萝卜素、13-顺式(13Z)- $\beta$ -胡萝卜素、15-顺式(15Z)- $\beta$ -胡萝卜素等。如果采用色谱条件一进行 $\beta$ -胡萝卜素的测定，应按照附录B确认 $\beta$ -胡萝卜素异构体保留时间，并计算全反式 $\beta$ -胡萝卜素标准溶液色谱纯度。

3.4.4 $\beta$ -胡萝卜素标准中间液 $(100~\mu \mathrm{g / mL})$ : 从 $\beta$ -胡萝卜素标准储备液中准确移取 $10.0~\mathrm{mL}$ 溶液于 $50~\mathrm{mL}$ 棕色容量瓶中, 用二氯甲烷定容至刻度。  
3.4.5 $\alpha$ -胡萝卜素、 $\beta$ -胡萝卜素混合标准工作液（色谱条件一用）：准确移取 $\alpha$ -胡萝卜素标准中间液 $0.50~\mathrm{mL}, 1.00~\mathrm{mL}, 2.00~\mathrm{mL}, 3.00~\mathrm{mL}, 4.00~\mathrm{mL}, 10.00~\mathrm{mL}$ 溶液至6个 $100~\mathrm{mL}$ 棕色容量瓶，分别加入 $3.00~\mathrm{mL}, \beta$ -胡萝卜素中间液，用二氯甲烷定容至刻度，得到 $\alpha$ -胡萝卜素浓度分别为 $0.5~\mu \mathrm{g} / \mathrm{mL}$ 、 $1.0~\mu \mathrm{g} / \mathrm{mL}, 2.0~\mu \mathrm{g} / \mathrm{mL}, 3.0~\mu \mathrm{g} / \mathrm{mL}, 4.0~\mu \mathrm{g} / \mathrm{mL}, 10.00~\mu \mathrm{g} / \mathrm{mL}, \beta$ -胡萝卜素浓度均为 $3.0~\mu \mathrm{g} / \mathrm{mL}$ 的系列混合标准工作液。  
3.4.6 $\beta$ -胡萝卜素标准工作液(色谱条件二用)：从 $\beta$ -胡萝卜素标准中间液中分别准确移取 $0.50~\mathrm{mL}$ 、 $1.00~\mathrm{mL},2.00~\mathrm{mL},3.00~\mathrm{mL},4.00~\mathrm{mL},10.00~\mathrm{mL}$ 溶液至6个 $100~\mathrm{mL}$ 棕色容量瓶。用二氯甲烷定容至刻度，得到浓度为 $0.5~\mu \mathrm{g / mL},1.0~\mu \mathrm{g / mL},2.0~\mu \mathrm{g / mL},3.0~\mu \mathrm{g / mL},4.0~\mu \mathrm{g / mL},10~\mu \mathrm{g / mL}$ 的系列标准工作液。

# 5 分析步骤

注：整个实验操作过程应注意避光。

# 5.1 试样制备

谷物、豆类、坚果等试样需粉碎、研磨、过筛(筛板孔径 $0.3\mathrm{mm}\sim 0.5\mathrm{mm}$ )；蔬菜、水果、蛋、藻类等试样用匀质器混匀；固体粉末状试样和液体试样用前振摇或搅拌混匀。 $4^{\circ}C$ 冰箱可保存1周。

# 5.2 试样处理

# 5.2.1 普通食品试样

# 5.2.1.1 预处理

蔬菜、水果、菌藻类、谷物、豆类、蛋类等普通食品试样准确称取混合均匀的试样 $1\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，油类准确称取 $0.2\mathrm{g}\sim 2\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，转至 $250~\mathrm{mL}$ 锥形瓶中，加入 $1\mathrm{g}$ 抗坏血酸、 $75~\mathrm{mL}$ 无水乙醇，于 $60^{\circ}C\pm 1^{\circ}C$ 水浴振荡 $30\mathrm{min}$ 。

如果试样中蛋白质、淀粉含量较高 $(>10\%)$ ，先加入 $1\mathrm{g}$ 抗坏血酸、 $15\mathrm{mL}45^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ 温水、 $0.5\mathrm{g}$ 木瓜蛋白酶和 $0.5\mathrm{g}\alpha$ -淀粉酶，盖上瓶塞混匀后，置 $55^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温水浴箱内振荡或超声处理 $30\mathrm{min}$ 后，再加入 $75\mathrm{mL}$ 无水乙醇，于 $60^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 水浴振荡 $30\mathrm{min}$ 。

# 5.2.1.2 皂化

加入 $25\mathrm{mL}$ 氢氧化钾溶液，盖上瓶塞。置于已预热至 $53~^\circ \mathrm{C}\pm 2$ ℃恒温振荡水浴箱中，皂化 $30\mathrm{min}$ 取出，静置，冷却到室温。

# 5.2.2 添加 $\beta$ -胡萝卜素的食品试样

# 5.2.2.1 预处理

固体试样：准确称取 $1\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，置于 $250~\mathrm{mL}$ 锥形瓶中，加入 $1\mathrm{g}$ 抗坏血酸，加 $50~\mathrm{mL}45^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 温水混匀。加入 $0.5\mathrm{g}$ 木瓜蛋白酶和 $0.5\mathrm{g}\alpha$ -淀粉酶(无淀粉试样可以不加 $\alpha$ -淀粉酶)，盖上瓶塞，置 $55^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 恒温水浴箱内振荡或超声处理 $30\mathrm{min}$ 。

液体试样：准确称取 $5\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，置于 $250~\mathrm{mL}$ 锥形瓶中，加入 $1\mathrm{g}$ 抗坏血酸。

# 5.2.2.2 皂化

取预处理后试样，加入 $75~\mathrm{mL}$ 无水乙醇，摇匀，再加入 $25~\mathrm{mL}$ 氢氧化钾溶液，盖上瓶塞。置于已预热至 $53^{\circ}C \pm 2^{\circ}C$ 恒温振荡水浴箱中，皂化 $30\mathrm{min}$ 。取出，静置，冷却到室温。

注：如皂化不完全可适当延长皂化时间至 $1\mathrm{h}$

# 5.3 试样萃取

将皂化液转入 $500~\mathrm{mL}$ 分液漏斗中，加入 $100~\mathrm{mL}$ 石油醚，轻轻摇动，排气，盖好瓶塞，室温下振荡 $10\mathrm{min}$ 后静置分层，将水相转入另一分液漏斗中按上述方法进行第二次提取。合并有机相，用水洗至近中性。弃水相，有机相通过无水硫酸钠过滤脱水。滤液收入 $500~\mathrm{mL}$ 蒸发瓶中，于旋转蒸发器上 $40^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 减压浓缩，近干。用氮气吹干，用移液管准确加入 $5.0~\mathrm{mL}$ 二氯甲烷，盖上瓶塞，充分溶解提取物。经 $0.45\mu \mathrm{m}$ 膜过滤后，弃出初始约 $1\mathrm{mL}$ 滤液后收集至进样瓶中，备用。

注：必要时可根据待测样液中胡萝卜素含量水平进行浓缩或稀释，使待测样液中 $\alpha$ -胡萝卜素和/或 $\beta$ -胡萝卜素浓度在 $0.5\mu \mathrm{g} / \mathrm{mL}\sim 10\mu \mathrm{g} / \mathrm{mL}$ 范围内。

b）流动相：A相：甲醇：乙腈：水 $= 73.5:24.5:2$

B相：甲基叔丁基醚；

# 5.4.1.2 绘制 $\alpha$ -胡萝卜素标准曲线、计算全反式 $\beta$ -胡萝卜素响应因子

将 $\alpha$ -胡萝卜素、 $\beta$ -胡萝卜素混合标准工作液注入HPLC仪中(色谱图见图C.1)，根据保留时间定性，测定 $\alpha$ -胡萝卜素、 $\beta$ -胡萝卜素各异构体峰面积。

$\alpha^{-}$ 胡萝卜素根据系列标准工作液浓度及峰面积，以浓度为横坐标，峰面积为纵坐标绘制标准曲线，计算回归方程。

$\beta$ -胡萝卜素根据标准工作液标定浓度、全反式 $\beta$ -胡萝卜素6次测定峰面积平均值、全反式 $\beta$ -胡萝卜素色谱纯度( $CP$ ，计算方法见附录B)，按式(1)计算全反式 $\beta$ -胡萝卜素响应因子。

$$
R F = \frac {\bar {A} _ {\text {a l l} - E}}{\rho \times C P} \quad \dots \tag {1}
$$

式中：

$RF$ ——全反式 $\beta$ -胡萝卜素响应因子，单位为峰面积毫升每微克 $(\mathrm{AU} \cdot \mathrm{mL} / \mu \mathrm{g})$ ；

$\overline{A}_{\mathrm{all - E}}$ ——全反式 $\beta$ -胡萝卜素标准工作液色谱峰峰面积平均值，单位为峰面积（AU）；

$\rho$ —— $\beta$ -胡萝卜素标准工作液标定浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$CP$ ——全反式 $\beta$ -胡萝卜素的色谱纯度， $\%$ 。

# 5.4.1.3 试样测定

在相同色谱条件下，将待测液注入液相色谱仪中，以保留时间定性，根据峰面积采用外标法定量。 $\alpha$ -胡萝卜素根据标准曲线回归方程计算待测液中 $\alpha$ -胡萝卜素浓度， $\beta$ -胡萝卜素根据全反式 $\beta$ -胡萝卜素响应因子进行计算。

# 5.4.2 色谱条件二（适用食品中 $\beta$ -胡萝卜素的测定）

# 5.4.2.1 参考色谱条件

参考色谱条件列出如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或等效柱；  
b）流动相：三氯甲烷：乙腈：甲醇 $= 3:12:85$ ，含抗坏血酸 $0.4\mathrm{g / L}$ ，经 $0.45\mu \mathrm{m}$ 膜过滤后备用；

# 5.4.2.3 试样测定

在相同色谱条件下，将待测试样液分别注入液相色谱仪中，进行HPLC分析，以保留时间定性，根据峰面积外标法定量，根据标准曲线回归方程计算待测液中 $\beta$ -胡萝卜素的浓度。

注：本色谱条件适用于 $\alpha$ -胡萝卜素含量较低(小于总胡萝卜素 $10\%$ )的食品试样中 $\beta$ -胡萝卜素的测定。

# 6 分析结果的表述

# 6.1 色谱条件一

试样中 $\alpha$ -胡萝卜素含量按式(2)计算：

$$
X _ {\alpha} = \frac {\rho_ {\alpha} \times V \times 1 0 0}{m} \tag {2}
$$

式中：

$X_{\alpha}$ ——试样中 $\alpha$ -胡萝卜素的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$

$\rho_{\alpha}$ ——从标准曲线得到的待测液中 $\alpha$ -胡萝卜素浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$

$V$ ——试样液定容体积，单位为毫升 $(\mathrm{mL})$

100——将结果表示为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ 的系数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

试样中 $\beta$ -胡萝卜素含量按式(3)计算：

$$
X _ {\beta} = \frac {\left(A _ {\text {a l l -} E} + A _ {9 Z} + A _ {1 3 Z} \times 1 . 2 + A _ {1 5 Z} \times 1 . 4 + A _ {x Z}\right) \times V \times 1 0 0}{R F \times m} \dots \tag {3}
$$

式中：

$X_{\beta}$ ——试样中 $\beta$ -胡萝卜素的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ ；

$A_{\mathrm{all - E}}$ ——试样待测液中全反式 $\beta$ 胡萝卜素峰面积，单位为峰面积（AU）；

$A_{9Z}$ ——试样待测液中9-顺式 $\beta$ -胡萝卜素的峰面积，单位为峰面积（AU）；

$A_{13Z}$ ——试样待测液中13-顺式 $\beta$ -胡萝卜素的峰面积，单位为峰面积（AU）；

1.2 ——13-顺式- $\beta$ -胡萝卜素的相对校正因子；

$A_{15Z}$ ——试样待测液中15-顺式 $\beta$ -胡萝卜素的峰面积，单位为峰面积（AU）；

1.4 ——15-顺式- $\beta$ -胡萝卜素的相对校正因子；

$A_{xZ}$ ——试样待测液中其他顺式 $\beta$ -胡萝卜素的峰面积，单位为峰面积（AU）；

$V$ ——试样液定容体积，单位为毫升 $(\mathrm{mL})$ ；

100 ——将结果表示为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ 的系数；

RF 全反式 $\beta$ -胡萝卜素响应因子, 单位为峰面积毫升每微克 $(\mathrm{AU} \cdot \mathrm{mL} / \mu \mathrm{g})$

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

注1：由于 $\beta$ -胡萝卜素各异构体百分吸光系数不同(见附录D)，所以在 $\beta$ -胡萝卜素计算过程中，需采用相对校正因子对结果进行校正。

注2：如果试样中其他顺式 $\beta$ -胡萝卜素含量较低，可不进行计算。

试样中总胡萝卜素含量按式(4)计算：

$$
X _ {\text {总}} = X _ {\alpha} + X _ {\beta} \quad \dots \tag {4}
$$

式中：

$X_{\text{总}}$ ——试样中总胡萝卜素的含量，单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$ ；

$X_{\alpha}$ ——试样中 $\alpha$ -胡萝卜素的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$

$X_{\beta}$ ——试样中 $\beta$ -胡萝卜素的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ 。

注：必要时， $\alpha-$ 胡萝卜素， $\beta-$ 胡萝卜素可转化为微克视黄醇当量 $(\mu \mathrm{gRE})$ 进行表示。

计算结果保留三位有效数字。

# 6.2 色谱条件二

试样中 $\beta$ -胡萝卜素含量按式(5)计算：

$$
X _ {\beta} = \frac {\rho_ {\beta} \times V \times 1 0 0}{m} \tag {5}
$$

式中：

$X_{\beta}$ ——试样中 $\beta$ -胡萝卜素的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ ；

$\rho_{\beta}$ ——从标准曲线得到的待测液中 $\beta$ -胡萝卜素浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样液定容体积，单位为毫升 $(\mathrm{mL})$ ；

100——将结果表示为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ 的系数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

注：结果中包含全反式 $\beta$ -胡萝卜素、9-顺式- $\beta$ -胡萝卜素、13-顺式- $\beta$ -胡萝卜素、15-顺式- $\beta$ -胡萝卜素、其他顺式异构体；不排除可能有部分 $\alpha$ -胡萝卜素。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

试样称样量为 $5\mathrm{g}$ 时， $\alpha$ -胡萝卜素、 $\beta$ -胡萝卜素检出限均为 $0.5\mu \mathrm{g} / 100\mathrm{g}$ ，定量限均为 $1.5\mu \mathrm{g} / 100\mathrm{g}$ 。

# 标准溶液浓度标定方法

# A.1 $\pmb{\alpha}$ -胡萝卜素标准储备液的标定

$\alpha$ -胡萝卜素标准储备液(浓度约为 $500~\mu \mathrm{g / mL})10~\mu \mathrm{L}$ ，注入含 $3.0~\mathrm{mL}$ 正己烷的比色皿中，混匀。比色杯厚度为 $1\mathrm{cm}$ ，以正己烷为空白，入射光波长为 $444~\mathrm{nm}$ ，测定其吸光度值，平行测定3次，取均值。

溶液浓度按式(A.1)计算：

$$
X = \frac {A}{E} \times \frac {3 . 0 1}{0 . 0 1} \quad \dots \tag {A.1}
$$

式中：

$X$ —— $\alpha$ -胡萝卜素标准储备液的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

A $\alpha$ -胡萝卜素标准储备液的紫外吸光值；

$E$ —— $\alpha$ -胡萝卜素在正己烷中的比吸光系数为0.2725；

$\frac{3.01}{0.01}$ ——测定过程中稀释倍数的换算系数。

# A.2 $\pmb{\beta}$ -胡萝卜素标准储备液的标定

取 $\beta$ -胡萝卜素标准储备液(浓度约为 $500~\mu \mathrm{g / mL})10~\mu \mathrm{L}$ , 注入含 $3.0~\mathrm{mL}$ 正己烷的比色皿中, 混匀。比色杯厚度为 $1\mathrm{cm}$ , 以正己烷为空白, 入射光波长为 $450~\mathrm{nm}$ , 测定其吸光度值, 平行测定3次, 取均值。

溶液浓度按式(A.2)计算：

$$
X = \frac {A}{E} \times \frac {3 . 0 1}{0 . 0 1} \quad \dots \tag {A.2}
$$

式中：

$X$ —— $\beta$ -胡萝卜素标准储备液的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

A - $\beta$ -胡萝卜素标准储备液的紫外吸光值；

$E$ —— $\beta$ -胡萝卜素在正己烷中的比吸光系数为0.2620；

$\frac{3.01}{0.01}$ ——测定过程中稀释倍数的换算系数。

B.2.1 碘乙醇溶液 $(0.05\mathrm{mol} / \mathrm{L})$ ：吸取 $5\mathrm{mL}$ 碘溶液，用乙醇稀释至 $50~\mathrm{mL}$ ，混匀。

# B.3 $\pmb{\beta}$ -胡萝卜素异构体保留时间的确认

分别取 $\beta$ -胡萝卜素标准中间液（ $100~\mu \mathrm{g / mL}$ ）和异构化 $\beta$ -胡萝卜素溶液，按照色谱条件一注入HPLC仪进行色谱分析。根据 $\beta$ -胡萝卜素标准中间液的色谱图确认全反式 $\beta$ -胡萝卜素的保留时间；对比 $\beta$ -胡萝卜素标准中间液和异构化 $\beta$ -胡萝卜素溶液色谱图中各峰面积变化，以及与全反式 $\beta$ -胡萝卜素的位置关系确认顺式 $\beta$ -胡萝卜素异构体的保留时间：全反式 $\beta$ -胡萝卜素前较大的色谱峰为13-顺式- $\beta$ -胡萝卜素，紧邻全反式 $\beta$ -胡萝卜素后较大的色谱峰为9-顺式- $\beta$ -胡萝卜素，13-顺式- $\beta$ -胡萝卜素前是15-顺式- $\beta$ -胡萝卜素，另外可能还有其他较小的顺式结构色谱峰，色谱图见图C.1。

# B.4 全反式 $\pmb{\beta}$ -胡萝卜素标准液色谱纯度的计算

取 $\beta$ -胡萝卜素标准工作液 $(3\mu \mathrm{g} / \mathrm{mL})$ ，按照色谱条件一进行HPLC分析，重复进样6次。计算全反式 $\beta$ -胡萝卜素色谱峰的峰面积、全反式与上述各顺式结构的峰面积总和，全反式 $\beta$ -胡萝卜素色谱纯度按式(B.1)计算。

$$
C P = \frac {\bar {A} _ {\text {all - E}}}{\bar {A} _ {\text {sum}}} \times 100 \% \tag{B.1}
$$

式中：

$CP$ ——全反式 $\beta$ -胡萝卜素色谱纯度， $\%$ ；

$\overline{A}_{\mathrm{all - E}}$ 一全反式 $\beta$ 胡萝卜素色谱峰峰面积平均值，单位为峰面积（AU）；

$\overline{A}_{\mathrm{sum}}$ ——全反式 $\beta$ -胡萝卜素及各顺式结构峰面积总和平均值，单位为峰面积（AU）。

# 食品安全国家标准

# 食品中双酚A、双酚F和双酚S的测定

# 1 范围

本标准规定了食品中双酚A、双酚F和双酚S的液相色谱-串联质谱测定方法。

本标准适用于肉及肉制品、乳及乳制品、水产品、谷物、蛋类、蔬菜、水果、饮料、婴幼儿配方食品和婴幼儿谷类辅助食品中双酚A、双酚F和双酚S的测定。

# 2 原理

试样中的双酚A、双酚F和双酚S，用乙腈或甲醇-水溶液提取，免疫亲和柱净化，液相色谱-串联质谱法测定，内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.3 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.4 磷酸氢二钠 $\mathrm{(Na_2HPO_4)}$   
3.1.5 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$ 。  
3.1.6 氯化钾(KCl)。  
3.1.7 盐酸（HCl）：优级纯。

# 3.2 试剂配制

3.2.1 盐酸溶液 $(10 + 90)$ ：量取 $10\mathrm{mL}$ 盐酸，加入 $90~\mathrm{mL}$ 水中，混匀。  
3.2.2 磷酸盐缓冲溶液(以下简称PBS)：称取 $8.0\mathrm{g}$ 氯化钠、 $1.2\mathrm{g}$ 磷酸氢二钠、 $0.2\mathrm{g}$ 磷酸二氢钾、 $0.2\mathrm{g}$ 氯化钾，用 $900\mathrm{mL}$ 水溶解，用盐酸溶液 $(10 + 90)$ 调节 $\mathrm{pH}$ 至 $7.4 \pm 0.1$ ，再加水稀释至 $1000\mathrm{mL}$ 。  
3.2.3 甲醇-水溶液 $(80 + 20)$ ：量取甲醇 $80~\mathrm{mL}$ ，加入水 $20~\mathrm{mL}$ ，混匀。  
3.2.4 甲醇-水溶液 $(40 + 60)$ ：量取甲醇 $40~\mathrm{mL}$ ，加入水 $60~\mathrm{mL}$ ，混匀。

# 3.3 标准品

3.3.1 双酚A标准品 $\mathrm{(C_{15}H_{16}O_2}$ ，CAS：80-05-7）：纯度 $\geqslant 99.0\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.2 双酚F标准品 $\mathrm{(C_{13}H_{12}O_2}$ ，CAS：620-92-8）：纯度 $\geq 99.0\%$ ，或经国家认证并授予标准物质证书的标准品。

3.3.3 双酚S标准品 $\mathrm{(C_{12}H_{10}O_4S,CAS:80 - 09 - 1)}$ ：纯度 $\geqslant 99.0\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.4 ${}^{13}\mathrm{C}_{12}$ -双酚A同位素内标（ $\mathrm{C}_3{}^{13}\mathrm{C}_{12}\mathrm{H}_{16}\mathrm{O}_2$ ，CAS：263261-65-0）： $100\mathrm{mg / L}$ ，或纯度 $\geqslant 98.0\%$ 的固体标准品。  
3.3.5 $^{13}\mathrm{C}_{12}$ -双酚F同位素内标 $(\mathrm{C}^{13}\mathrm{C}_{12}\mathrm{H}_{12}\mathrm{O}_2)$ ： $100~\mathrm{mg / L}$ ，或纯度 $\geq 98.0\%$ 的固体标准品。  
3.3.6 $^{13}\mathrm{C}_{12}$ -双酚S同位素内标 $(^{13}\mathrm{C}_{12}\mathrm{H}_{10}\mathrm{O}_4\mathrm{S})$ ： $100~\mathrm{mg / L}$ ，或纯度 $\geq 98.0\%$ 的固体标准品。

# 3.4 标准溶液配制

3.4.1 标准储备液 $(100\mathrm{mg / L})$ : 准确称取双酚A、双酚F和双酚S标准品各 $10\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ), 分别用甲醇溶解并定容至 $100\mathrm{mL}$ , 混匀。将溶液转移至棕色玻璃试剂瓶中, $-18^{\circ}\mathrm{C}$ 下避光保存, 有效期为 12 个月。  
3.4.2 标准中间液 $(1\mathrm{mg / L})$ ：准确吸取双酚A、双酚F和双酚S的标准储备液各 $0.10~\mathrm{mL}$ ，分别置于 $10~\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度，混匀。将溶液转移至棕色玻璃试剂瓶中，一 $18^{\circ}C$ 下避光保存，有效期为6个月。  
3.4.3 标准混合使用液（双酚A和双酚F： $100\mu \mathrm{g} / \mathrm{L}$ ；双酚S： $10~\mu \mathrm{g} / \mathrm{L})$ ：准确吸取双酚A和双酚F的标准中间液各 $1.00~\mathrm{mL}$ 、双酚S的标准中间液 $0.10~\mathrm{mL}$ ，置于 $10~\mathrm{mL}$ 容量瓶中，用甲醇-水溶液(40+60)定容至刻度，混匀。临用现配。  
3.4.4 同位素内标中间液 $(1\mathrm{mg / L})$ ：准确吸取 ${}^{13}\mathrm{C}_{12}$ -双酚A标准溶液、 ${}^{13}\mathrm{C}_{12}$ -双酚F标准溶液和 ${}^{13}\mathrm{C}_{12}$ -双酚S标准溶液各 $0.10~\mathrm{mL}$ ，分别置于 $10~\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度，混匀。将溶液转移至棕色玻璃试剂瓶中， $-18^{\circ}\mathrm{C}$ 下避光保存，有效期为6个月。  
3.4.5 同位素内标混合使用液 $\left({}^{13}\mathrm{C}_{12}-\right.$ 双酚A和 ${}^{13}\mathrm{C}_{12}-$ 双酚F: $100\mu \mathrm{g / L};{}^{13}\mathrm{C}_{12}-$ 双酚S: $10\mu \mathrm{g / L})$ : 准确吸取 ${}^{13}\mathrm{C}_{12}-$ 双酚A和 ${}^{13}\mathrm{C}_{12}-$ 双酚F中间液各 $1.00~\mathrm{mL},{}^{13}\mathrm{C}_{12}-$ 双酚S中间液 $0.10~\mathrm{mL}$ , 置于 $10~\mathrm{mL}$ 容量瓶中, 用甲醇-水溶液 $(40 + 60)$ 定容至刻度, 混匀。临用现配。  
3.4.6 标准系列工作液：分别准确吸取标准混合使用液 $0.10\mathrm{mL}, 0.20\mathrm{mL}, 0.50\mathrm{mL}, 0.80\mathrm{mL}$ 、 $1.00\mathrm{mL}, 2.00\mathrm{mL}$ 于 $10\mathrm{mL}$ 容量瓶中，加入 $0.50\mathrm{mL}$ 同位素内标混合使用液，用甲醇-水溶液 $(40 + 60)$ 定容至刻度，混匀。标准系列工作液中双酚A和双酚F的质量浓度分别为 $1.00\mu \mathrm{g} / \mathrm{L}, 2.00\mu \mathrm{g} / \mathrm{L}$ 、 $5.00\mu \mathrm{g} / \mathrm{L}, 8.00\mu \mathrm{g} / \mathrm{L}, 10.0\mu \mathrm{g} / \mathrm{L}, 20.0\mu \mathrm{g} / \mathrm{L}$ ，双酚S的质量浓度分别为 $0.100\mu \mathrm{g} / \mathrm{L}, 0.200\mu \mathrm{g} / \mathrm{L}, 0.500\mu \mathrm{g} / \mathrm{L}, 0.800\mu \mathrm{g} / \mathrm{L}, 1.00\mu \mathrm{g} / \mathrm{L}, 2.00\mu \mathrm{g} / \mathrm{L}$ ， ${}^{13}\mathrm{C}_{12}$ -双酚A和 ${}^{13}\mathrm{C}_{12}$ -双酚F的质量浓度为 $5.00\mu \mathrm{g} / \mathrm{L}$ ， ${}^{13}\mathrm{C}_{12}$ -双酚S的质量浓度为 $0.500\mu \mathrm{g} / \mathrm{L}$ 。临用现配。

# 3.5 材料

3.5.1 双酚A、双酚F和双酚S复合免疫亲和柱：各化合物柱容量 $\geq 200\mathrm{ng}$ ，柱回收率 $\geq 90\%$ （验证方法参见附录A）。

注：对于不同批次的免疫亲和柱在使用前需进行质量验证。

3.5.2 聚丙烯离心管： $2\mathrm{mL}$ 和 $50~\mathrm{mL}$   
3.5.3 玻璃氮吹管： $15~\mathrm{mL}$

# 4 仪器和设备

4.1 液相色谱-串联四极杆质谱仪：配电喷雾离子源。  
4.2 电子天平：感量为 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
4.3 匀浆机。  
4.4 高速粉碎机。

4.5 超声波振荡器。  
4.6 涡旋混合器。  
4.7 固相萃取装置（带真空泵）。  
4.8 高速冷冻离心机：最高转速 $\geqslant 10000\mathrm{r / min}$   
4.9 氮吹仪。  
4.10 pH计：分度值为0.1单位。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固态样品

5.1.1.1 干样：谷物等固态样品，取可食部分，经高速粉碎机粉碎均匀；乳粉、婴幼儿配方食品和婴幼儿谷类辅助食品等粉状样品混合均匀。  
5.1.1.2 鲜样：蔬菜、水果、肉及肉制品、蛋类、水产品等样品，取可食部分匀浆均匀。  
5.1.2 液态样品：碳酸饮料超声 $10\mathrm{min}$ 充分排气；液态乳和非碳酸饮料直接摇匀。  
5.1.3 半固态样品：发酵乳等半固态样品搅拌均匀。

# 5.2 试样提取

# 5.2.1 肉及肉制品、水产品、固态及半固态乳制品（乳粉、发酵乳等）、婴幼儿配方食品

称取 $1\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），置于 $50~\mathrm{mL}$ 聚丙烯离心管中，加入 $50~\mu \mathrm{L}$ 同位素内标混合使用液，振荡混合后静置 $30\mathrm{min}$ 。加入 $2\mathrm{mL}$ 水，涡旋振荡 $30~\mathrm{s}$ ，再加入 $5\mathrm{mL}$ 乙腈涡旋混合。混合液超声提取 $15\mathrm{min},4^{\circ}\mathrm{C}$ 下 $10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取上清液置于玻璃氮吹管中， $40^{\circ}\mathrm{C}$ 下用氮气缓缓吹至约 $2\mathrm{mL}$ ，加入 $8\mathrm{mL}$ PBS混匀，待净化。

# 5.2.2 蛋、乳及液态乳制品

称取 $1\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )，置于 $50~\mathrm{mL}$ 聚丙烯离心管中，加入 $50~\mu \mathrm{L}$ 同位素内标混合使用液，振荡混合后静置 $30\mathrm{min}$ 。加入 $5\mathrm{mL}$ 乙腈，涡旋混合后超声提取 $15\mathrm{min},4^{\circ}\mathrm{C}$ 下 $10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取上清液置于玻璃氮吹管中， $40^{\circ}C$ 下用氮气缓缓吹至约 $2\mathrm{mL}$ ，加入 $8\mathrm{mL}$ PBS混匀，待净化。

# 5.2.3 谷物及婴幼儿谷类辅助食品

称取 $1\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），置于 $50~\mathrm{mL}$ 聚丙烯离心管中，加入 $50~\mu \mathrm{L}$ 同位素内标混合使用液，振荡混合后静置 $30\mathrm{min}$ 。加入 $10~\mathrm{mL}$ 甲醇-水溶液 $(80 + 20)$ ，涡旋振荡 $30~\mathrm{s}$ ，超声提取 $15\mathrm{min},4^{\circ}\mathrm{C}$ 下 $10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取上清液置于玻璃氮吹管中， $40^{\circ}\mathrm{C}$ 下用氮气缓缓吹至约 $2\mathrm{mL}$ ，加入 $8\mathrm{mL}$ PBS 混匀，待净化。

# 5.2.4 蔬菜、水果

称取 $1\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），置于 $50~\mathrm{mL}$ 聚丙烯离心管中，加入 $100\mu \mathrm{L}$ 盐酸溶液 $(10 + 90)$ 混匀，再加入 $50~\mu \mathrm{L}$ 同位素内标混合使用液，振荡混合后静置 $30\mathrm{min}$ 。加入 $5\mathrm{mL}$ 乙腈，涡旋振荡 $30~\mathrm{s}$ ，超声提取 $15\mathrm{min}$ ， $4^{\circ}\mathrm{C}$ 下 $10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取上清液置于玻璃氮吹管中， $40^{\circ}\mathrm{C}$ 下用氮气缓缓吹至约 $2\mathrm{mL}$ ，加入 $8\mathrm{mL}$ PBS混匀，待净化。

# 5.2.5 饮料

称取 $1\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )，置于 $2\mathrm{mL}$ 聚丙烯离心管中，加入 $50~\mu \mathrm{L}$ 同位素内标混合使用液，振荡混合后静置 $30\mathrm{min}$ 。 $10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取上清液至 $50~\mathrm{mL}$ 聚丙烯离心管中，加入 $10~\mathrm{mL}$ PBS混匀，待净化。

注：澄清试样(如纯净水、矿泉水等)可不离心,直接加入 PBS 混匀。

# 5.3 试样净化

将低温下保存的免疫亲和柱恢复至室温，然后让柱内原有液体自然流尽。取5.2中试样提取液全部过柱，自然流干，弃去全部流出液，再用 $10~\mathrm{mL}$ 水淋洗免疫亲和柱。待水滴完后，用真空泵抽干免疫亲和柱。加入 $1\mathrm{mL}$ 甲醇洗脱，收集洗脱液于玻璃氮吹管中， $40^{\circ}\mathrm{C}$ 下用氮气缓缓吹干。准确加入 $0.40~\mathrm{mL}$ 甲醇，涡旋混合 $10~\mathrm{s}$ ，再准确加入 $0.60~\mathrm{mL}$ 水，涡旋混合后转移至 $2\mathrm{mL}$ 聚丙烯离心管中， $10000~\mathrm{r / min}$ 下离心 $5\mathrm{min}$ ，取上清液供液相色谱-串联质谱仪测定。

# 5.5 标准曲线的制作

将标准系列工作液经液相色谱-串联质谱仪测定，获得相应的峰面积，以标准系列工作液中双酚A、双酚F和双酚S与其对应的同位素内标的峰面积比为纵坐标，以标准系列工作液的浓度为横坐标，绘制标准曲线。

# 5.6 定性测定

被测试样中被测物色谱峰的保留时间与相应标准溶液色谱峰的保留时间相比较，变化范围应在 $\pm 2.5\%$ 之内。

被测物的两个质谱定性离子均出现，且同一检测批次，对同一化合物，试样中被测物的两个定性离子的相对丰度比与质量浓度相当的标准溶液相比，其偏差不超过表3规定的范围，则可判断试样中存在对应的被测物。

# 5.7 定量测定

将试样溶液注入液相色谱-串联质谱仪中，得到被测物定量离子峰面积与对应同位素内标定量离子峰面积的比值，根据标准曲线得到试样溶液中被测物的浓度。试样溶液中被测物的响应值应在标准曲线的线性范围内，超过线性范围说明样品含量过高，可以适当减少取样量重新测定。

# 5.8 空白试验

除不称取试样外，均按上述测定步骤进行。

空白试验溶液中各被测物的浓度应尽量控制在方法定量限对应浓度（双酚A和双酚 $\mathrm{F}1\mu \mathrm{g / L}$ ，双酚S $0.1\mu \mathrm{g / L})$ 以下，如果高于定量限，应尝试更换试剂和实验器皿。

# 6 分析结果的表述

试样中双酚A、双酚F和双酚S的含量按照公式(1)进行计算：

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中被测物的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——由标准曲线计算得到的试样溶液中被测物的质量浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样量，单位为克(g)；

1000——换算系数。

计算结果保留三位有效数字。

注：必要时计算结果应扣除空白值。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

# 免疫亲和柱验证方法

准确吸取双酚A、双酚F和双酚S的标准中间液各 $625\mu \mathrm{L}$ ，置于同一 $25~\mathrm{mL}$ 容量瓶中，用PBS定容至刻度，充分混匀后用于免疫亲和柱上样。分别取同一批次3根免疫亲和柱，每根柱的上样量为 $8\mathrm{mL}$ 。按5.3中步骤进行上样、淋洗和洗脱，收集洗脱液，用初始流动相稀释定容至 $10~\mathrm{mL}$ ，上液相色谱-串联质谱仪测定。用 $20.0~\mu \mathrm{g / L}$ 的溶剂标准工作溶液做单点校准，按照公式(A.1)计算上机试样中双酚A、双酚F和双酚S的含量。

$$
\rho = \frac {\rho_ {\mathrm {s}} \times A}{A _ {\mathrm {s}}} \quad \dots \tag {A.1}
$$

式中：

$\rho$ ——上机试样中被测物的含量，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_{\mathrm{s}}$ ——标准工作溶液中被测物的含量，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$A$ ——上机试样中被测物定量离子的峰面积；

$A_{\mathrm{s}}$ ——标准工作溶液中被测物定量离子的峰面积。

结果判定：上机试样中双酚A、双酚F和双酚S的含量均 $\geqslant 18.0~\mu \mathrm{g / L}$ ，为可使用商品。

# 食品添加剂 偏酒石酸

# 1 范围

本标准适用于以酒石酸为原料，经 $168\mathrm{^\circ C}\sim 172\mathrm{^\circ C}$ 烘烤 $3.5\mathrm{h}$ ，或加热至 $180\mathrm{^\circ C}\sim 190\mathrm{^\circ C}$ ，然后自然冷却制得的食品添加剂偏酒石酸。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

偏酒石酸

# 2.2 分子式

$\mathrm{C_8H_8O_{10}}$

# 检验方法

警示：试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和防护措施。

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682中规定的三级水。试验方法中所用溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603之规定制备。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 硫酸溶液： $c(\mathrm{H}_2\mathrm{SO}_4) = 0.0025\mathrm{mol / L}$

A.2.1.2 微孔滤膜： $0.22\mu \mathrm{m}$

# A.2.2 仪器和设备

A.2.2.1 高效液相色谱仪，二极管阵列检测器。

A.2.2.2 分析天平：感量 $0.0001\mathrm{g}$

A.2.2.3 微量进样器： $10~\mu \mathrm{L}$

# A.2.3 参考色谱条件

A.2.3.1 色谱柱：Rezex ROA-Organic Acid, $300\mathrm{mm}\times 7.8\mathrm{mm},8\mu \mathrm{m}$ ，或者满足条件的其他色谱柱。  
A.2.3.2 柱温： $40^{\circ}C$ 。  
A.2.3.3 流动相： $0.0025\mathrm{mol} / \mathrm{L}$ 硫酸溶液。  
A.2.3.4 流速： $0.4\mathrm{mL / min}$   
A.2.3.5 进样量： $10~\mu \mathrm{L}$   
A.2.3.6 检测波长： $210~\mathrm{nm}$

# A.2.4 分析步骤

# A.2.4.1 0.0025mol/L硫酸溶液配制

在 $1000~\mathrm{mL}$ 烧杯中加入 $800~\mathrm{mL}$ 超纯水，缓慢加入 $136~\mu \mathrm{L}~98\%$ 的浓硫酸，用玻璃棒缓慢搅匀，待溶液冷却至室温后，将所配制溶液移入 $1000~\mathrm{mL}$ 容量瓶中定容至 $1000~\mathrm{mL}$ 。

# A.2.4.2 溶液配制

称取 $0.010\mathrm{g}$ 的待测样品，放入 $50~\mathrm{mL}$ 的烧杯中，加入 $30~\mathrm{mL}$ 的 $0.0025\mathrm{mol} / \mathrm{L}$ 硫酸溶液，搅拌至样品完全溶解后移入 $100~\mathrm{mL}$ 容量瓶中，用 $0.0025\mathrm{mol} / \mathrm{L}$ 硫酸溶液定容至 $100~\mathrm{mL}$ 。

# A.2.4.3 测定

取 $10\mathrm{mL}$ 配制好的样品溶液过 $0.22\mu \mathrm{m}$ 的滤膜后，取 $10~\mu \mathrm{L}$ 处理后的样品溶液注入液相色谱仪，进行高效液相色谱分析，记录色谱峰保留时间，根据保留时间判断是否是偏酒石酸。偏酒石酸在给定的

色谱条件和试剂条件下，保留时间为 $10.0\mathrm{min}$ 。见图A.1。

# A.3 总酸量（以干基计）和脱羧度的测定

# A.3.1 试剂和材料

A.3.1.1 溴百里酚蓝溶液： $4.0\mathrm{g / L}$   
A.3.1.2 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 1.0\mathrm{mol / L}$   
A.3.1.3 硫酸溶液： $c(\mathrm{H}_2\mathrm{SO}_4) = 0.5\mathrm{mol / L}$

# A.3.2 分析步骤

称取 $2.000\mathrm{g}$ 样品，加 $100~\mathrm{mL}$ 去离子水加热溶解备用。取新配制的偏酒石酸溶液 $50~\mathrm{mL}$ 放入 $250~\mathrm{mL}$ 锥形瓶中，滴加3滴溴百里酚蓝指示剂，用 $1.0\mathrm{mol / L}$ 的氢氧化钠溶液滴定至溶液呈蓝绿色，稳定 $30~\mathrm{s}$ 不变色，记录氢氧化钠的消耗体积 $V_{1}$ 。继续加入 $20.0\mathrm{mL}1.0\mathrm{mol / L}$ 的氢氧化钠溶液，盖上瓶塞，在室温下静置 $2\textrm{h}$ ，然后用 $0.5\mathrm{mol / L}$ 的硫酸滴定过量的氢氧化钠至溶液呈蓝绿色，记录硫酸的消耗量 $V_{2}$ 。

在测定前需要将测试样品在 $120^{\circ}\mathrm{C}$ 烘箱中烘干至恒重后再进行称取。

# A.3.3 结果计算

每消耗 $1.0\mathrm{mL}1.0\mathrm{mol / L}$ 的氢氧化钠溶液，相当于溶液中含有 $0.075\mathrm{g}$ 酒石酸。

待测样品中总酸量(包括游离的和脱羧酯化的酸) $\omega$ 按式(A.1)计算。

$$
w = \left[0.075 \times \left(V_{1} + 20.0 - V_{2}\right)\right] / m \times 100\% \quad \dots \tag{A.1}
$$

式中：

$w$ ——总酸量， $\%$ （保留三位有效数字）；

0.075 ——换算系数；

$V_{1}$ ——第一次滴定所用氢氧化钠的体积，单位为毫升 $(\mathrm{mL})$ （读数精确到 $0.1\mathrm{mL}$ ）；

20.0 ——第一次滴定后氢氧化钠溶液的加入量，单位为毫升 $(\mathrm{mL})$ （读数精确到 $0.1\mathrm{mL}$ ）；

$V_{2}$ ——第二次滴定所用硫酸的体积，单位为毫升 $(\mathrm{mL})$ （读数精确到 $0.1\mathrm{mL}$ ）；

$m$ ——所称取的样品质量，单位为克 $(\mathrm{g})$ 。

待测样品的脱羧度 $T$ 按式(A.2)计算。

$$
T = (20.0 - V _ {2}) / \left(V _ {1} + 20.0 - V _ {2}\right) \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$T$ ——脱羧度， $\%$ （保留三位有效数字）；

20.0 ——第一次滴定后氢氧化钠溶液的加入量，单位为毫升 $(\mathrm{mL})$ （读数精确到 $0.1~\mathrm{mL}$ ）；

$V_{2}$ ——第二次滴定所用硫酸的体积，单位为毫升 $(\mathrm{mL})$ （读数精确到 $0.1\mathrm{mL}$ ）；

$V_{1}$ ——第一次滴定所用氢氧化钠的体积，单位为毫升 $(\mathrm{mL})$ （读数精确到 $0.1\mathrm{mL}$ ）。

上述测定均取两次平行测定结果的算术平均值为报告结果。两次平行测定结果的绝对差值不大于 $0.2\%$

# A.4 草酸盐试验

# A.4.1 试剂和材料

A.4.1.1 氨水溶液： $10\%$

A.4.1.2 硫酸钙饱和溶液。

# A.4.2 分析步骤

称取 $1.00\mathrm{g}$ 样品，溶于 $10~\mathrm{mL}$ 水中，用氨水溶液中和至中性，再加 $10~\mathrm{mL}$ 硫酸钙饱和溶液，摇匀后观察，不得出现浑浊。

# A.5 硫酸盐试验

# A.5.1 试剂和材料

A.5.1.1 盐酸溶液： $1 + 4$

A.5.1.2 氯化钡溶液：称取 $12.0\mathrm{g}\mathrm{BaCl}_2\cdot 2\mathrm{H}_2\mathrm{O}$ 溶于 $100~\mathrm{mL}$ 水中。

# A.5.2 分析步骤

称取 $1.00\mathrm{g}$ 样品，溶于 $100~\mathrm{mL}$ 水中，加热溶解。取 $10~\mathrm{mL}$ 该溶液置于试管中，加入3滴盐酸溶液及 $1\mathrm{mL}$ 氯化钡溶液，摇匀后观察，不得出现浑浊。

# 动物性食品中氯苯胍残留量的测定

# 液相色谱-串联质谱法

# 1 范围

本文件规定了动物性食品中氯苯胍残留量检测的制样和液相色谱-串联质谱的检测方法。

本文件适用于鸡的肌肉、肝脏、肾脏和皮+脂组织中氯苯胍残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的氯苯胍用 $0.1\%$ 甲酸乙腈提取，固相萃取柱净化，液相色谱串联质谱法测定，内标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水

# 5.1 试剂

5.1.1 乙腈 $(\mathrm{CH}_3\mathrm{CN})$ ：色谱纯。

5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.3 甲酸（HCOOH）：色谱纯。

5.1.4 正己烷 $(\mathrm{C_6H_{14}})$ 。

5.1.5 二甲亚砜 $(\mathrm{C}_2\mathrm{H}_6\mathrm{OS})$

# 5.2 溶液配制

5.2.1 $0.1\%$ 甲酸乙腈溶液：取甲酸 $0.1\mathrm{mL}$ ，用乙腈稀释至 $100~\mathrm{mL}$

5.2.2 $85\%$ 乙腈溶液：取乙腈 $85~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$

5.2.3 $0.1\%$ 甲酸水溶液：取甲酸 $0.1\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$

5.2.4 $0.3\%$ 甲酸水溶液：取甲酸 $0.3\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$

5.2.5 $50\%$ 乙腈甲酸水溶液：取乙腈 $50~\mathrm{mL}$ ，用 $0.3\%$ 甲酸水溶液稀释至 $100~\mathrm{mL}$ 。

# 5.3 标准品

5.3.1盐酸氯苯胍（Robenidinehydrochloride， $\mathrm{C_{15}H_{14}Cl_3N_5}$ ，CAS号：25875-50-7）：含量 $\geq 99.0\%$

5.3.2 盐酸氯苯胍 $\mathrm{D_8}$ （Robenidine- $\mathrm{D_8}$ hydrochloride, $\mathrm{C_{15}H_6C_{13}D_8N_5}$ , CAS号：1173097-77-2）：含量 $\geq$ $99.0\%$ 。

# 5.4 标准溶液制备

5.4.1 氯苯胍标准储备液：取盐酸氯苯胍标准品适量（相当于氯苯胍 $10\mathrm{mg}$ ），精密称定，加甲醇适量使溶解并稀释定容至 $10\mathrm{mL}$ 容量瓶，配制成浓度为 $1\mathrm{mg/mL}$ 的标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期6个月。  
5.4.2 氯苯胍标准工作液：准确量取氯苯胍标准储备液 $1\mathrm{mL}$ ，于 $100\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的氯苯胍标准工作液。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期3个月。  
5.4.3 氯苯胍- $\mathrm{D_8}$ 内标储备液：取盐酸氯苯胍- $\mathrm{D_8}$ 标准品适量（相当于氯苯胍- $\mathrm{D_8}10\mathrm{mg}$ ），加二甲亚砜适量使溶解，用甲醇稀释定容至 $10~\mathrm{mL}$ 容量瓶，配制成浓度为 $1\mathrm{mg / mL}$ 的氯苯胍- $\mathrm{D_8}$ 内标储备液。一 $18^{\circ}C$ 以下保存，有效期6个月。  
5.4.4 氯苯胍- $\mathrm{D_8}$ 内标中间溶液：准确量取氯苯胍- $\mathrm{D_8}$ 内标储备液 $1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的氯苯胍- $\mathrm{D_8}$ 内标中间溶液。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1个月。  
5.4.5 氯苯胍- $\mathrm{D}_{8}$ 内标工作溶液：准确量取氯苯胍- $\mathrm{D}_{8}$ 内标中间溶液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 的氯苯胍- $\mathrm{D}_{8}$ 内标工作液。现用现配。

# 5.5 材料

5.5.1 HLB固相萃取柱： $200\mathrm{mg / 3mL}$ ，或相当者。  
5.5.2 中性氧化铝固相萃取柱： $500\mathrm{mg / 3mL}$ ，或相当者。  
5.5.3 尼龙微孔滤膜： $0.22\mu \mathrm{m}$   
5.5.4 离心管： $50~\mathrm{mL}$   
5.5.5 刻度试管： $10~\mathrm{mL}$ ，分度值 $0.1\mathrm{mL}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 低温高速离心机：转速可达 $8000\mathrm{r / min}$   
6.4 组织匀浆器。  
6.5 涡旋混合器。  
6.6 固相萃取装置。  
6.7 氮吹仪。

# 7 试料的制备与保存

# 7.1 试料的制备

取新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作溶液，作为空白添加试料。

# 7.2 样品的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取试料 $2\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管，准确加氯苯胍- $\mathrm{D_8}$ 内标工作溶液 $100~\mu \mathrm{L}$ ，加 $0.1\%$ 甲酸乙腈溶液 $5\mathrm{mL}$ ，涡旋混合，中速振荡 $10\mathrm{min},8000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液，再加水 $1\mathrm{mL}$ ，混匀，作

为备用液1。

# 8.2 净化

取中性氧化铝固相萃取柱，用乙腈 $2\mathrm{mL}$ 活化，取备用液1过柱，收集全部流出液于 $10~\mathrm{mL}$ 试管中，用 $85\%$ 乙腈溶液 $2\mathrm{mL}$ 淋洗，合并流出液，于 $40^{\circ}C$ 氮气吹至体积小于 $2\mathrm{mL}$ ，再加水 $8\mathrm{mL}$ ，混匀，作为备用液2。

取HLB固相萃取柱，依次用甲醇 $3\mathrm{mL}$ 、水 $3\mathrm{mL}$ 活化，取备用液2过柱，用水 $3\mathrm{mL}$ 淋洗，抽干 $30~s$ 正己烷 $3\mathrm{mL}$ 淋洗，抽干 $10\mathrm{min}$ ，甲醇 $5\mathrm{mL}$ 洗脱，收集洗脱液，于 $40^{\circ}C$ 氮气吹干，用 $50\%$ 乙腈甲酸水溶液 $1.0\mathrm{mL}$ 溶解残余物，涡旋，尼龙微孔滤膜过滤，供液相色谱-串联质谱测定。

# 8.3 标准曲线的制备

精密量取氯苯胍标准工作溶液、氯苯胍- $\mathrm{D_8}$ 内标工作溶液适量，用 $50\%$ 乙腈甲酸水溶液稀释，配制成氯苯胍浓度为 $5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL},50\mathrm{ng / mL},100\mathrm{ng / mL},200\mathrm{ng / mL},500\mathrm{ng / mL}$ 和氯苯胍 $\mathrm{D_8}$ 内标均为 $100~\mathrm{ng / mL}$ 的系列标准溶液，供液相色谱-串联质谱测定。以特征离子质量色谱峰面积为纵坐标、相对应的标准溶液浓度为横坐标，绘制标准曲线，求回归方程和相关系数。

# 8.5 测定法

# 8.5.1 定性测定

在同样测试条件下，试料溶液中氯苯胍的保留时间与标准工作液中氯苯胍的保留时间相对偏差在 $\pm 2.5\%$ 以内，且检测到的相对离子丰度，应当与浓度相当的校正标准溶液相对离子丰度一致。其允许偏差应符合表3的要求。

# 8.5.2 定量测定

取试料溶液和相应的标准工作液，作单点或多点校准，按内标法以特征离子质量色谱峰面积定量，标准工作液及试料溶液中的氯苯胍响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，氯苯胍标准溶液特征离子质量色谱图见附录A。

# 8.6 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试料中氯苯胍残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times A _ {\mathrm {i S}} \times C _ {\mathrm {S}} \times C _ {\mathrm {i S}} \times V}{A _ {\mathrm {i S}} \times A _ {\mathrm {S}} \times C _ {\mathrm {i S}} ^ {\prime} \times m} \dots \tag {1}
$$

式中：

$X$ 供试试料中氯苯胍残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$C_{iS}$ ——试料溶液中氯苯胍 $\mathrm{D}_{\mathrm{s}}$ 内标浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$C_{\mathrm{S}}$ ——标准溶液中氯苯胍浓度的数值，单位为纳克每毫升 $(\mathrm{mg / mL})$ ；

$C_{iS}^{\prime}$ ——标准溶液中氯苯胍 $\mathrm{D}_{\mathrm{g}}$ 内标浓度的数值，单位为纳克每毫升（ $\mathrm{ng / m}$ ）

A ——试料溶液中氯苯胍的峰面积：

$A_{iS}$ ——试料溶液中氯苯胍 $\mathrm{D}_6$ 内标的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中氯苯胍的峰面积；

$A_{iS}^{\prime}$ ——标准溶液中氯苯胍 $\mathrm{D}_{8}$ 内标的峰面积；

V ——溶解残余物所用溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ 供试试料质量的数值，单位为克 $(\mathrm{g})$

# 10 检测方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $5\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在肌肉、肝脏和肾脏组织 $10\mu \mathrm{g / kg}\sim 200\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$ ；在皮+脂组织 $10~\mu \mathrm{g / kg}\sim 400~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$ 。

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 20\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

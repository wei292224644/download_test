# 食品安全国家标准

# 食品中壬基酚的测定

# 1 范围

本标准规定了食品中4-壬基酚的液相色谱-串联质谱测定方法。

本标准适用于食品中4-壬基酚的测定。

# 2 原理

试样中的4-壬基酚用乙腈或乙腈溶液提取，采用高效液相色谱分离，串联质谱仪检测，同位素内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为色谱纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。  
3.1.2 乙腈 $\mathrm{(CH_3CN)}$ 。  
3.1.3 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ：浓度 $\geq 25\%$

# 3.2 试剂配制

3.2.1 $70\%$ 乙腈溶液：量取乙腈 $700~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。  
3.2.2 $0.1\%$ 氨水溶液：移取氨水 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。

# 3.3 标准品

# 3.3.1 4-壬基酚 $(\mathbf{C}_{15}\mathbf{H}_{24}\mathbf{O})$ 标准品

4-壬基酚 $\mathrm{(C_{15}H_{24}O, CAS}$ 号：84852-15-3)，纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.3.2 同位素内标（3，6，3-壬基酚-13C6）

3,6,3-壬基酚 $\mathrm{^{13}C_6}$ 标准品（ $\mathrm{^{13}C_6C_9H_{24}O}$ ，CAS号：1173020-38-6），纯度 $\geqslant 99\%$ 。

# 3.4 标准溶液配制

3.4.1 4-壬基酚标准储备液 $(100\mathrm{mg / L})$ ：准确称取 $10\mathrm{mg}4-$ 壬基酚标准品（精确至 $0.00001\mathrm{g}$ )，置于 $10~\mathrm{mL}$ 小烧杯中，用甲醇溶解，转移至 $100~\mathrm{mL}$ 棕色容量瓶中，用甲醇定容至刻度，混匀，一 $18^{\circ}C$ 下保存，保存期6个月。  
3.4.2 同位素内标标准储备液 $(100\mathrm{mg / L})$ ：准确称取 $10\mathrm{mg}3,6,3-$ 壬基酚- ${}^{13}\mathrm{C}_6$ 标准品（精确至

0.000 01 g), 置于 $10 \mathrm{~mL}$ 小烧杯中, 用甲醇溶解, 转移至 $100 \mathrm{~mL}$ 棕色容量瓶中, 用甲醇定容至刻度, 混匀, $-18^{\circ} \mathrm{C}$ 下保存, 保存期 6 个月。  
3.4.3 4-壬基酚标准中间液 $(1.0\mathrm{mg / L})$ ：准确移取 $1.0~\mathrm{mL}$ 4-壬基酚标准储备液（ $100\mathrm{mg / L}$ )，置于 $100~\mathrm{mL}$ 棕色容量瓶中，用甲醇定容至刻度，混匀，一 $18^{\circ}C$ 下保存，保存期3个月。  
3.4.4 同位素内标工作液 $(1.0\mathrm{mg / L})$ ：准确移取 $1.0~\mathrm{mL}$ 同位素内标标准储备液（ $100\mathrm{mg / L})$ ，置于 $100~\mathrm{mL}$ 棕色容量瓶中，用甲醇定容至刻度，混匀，一 $18^{\circ}C$ 下保存，保存期3个月。  
3.4.5 4-壬基酚标准系列工作液：准确移取 $2.50 \mathrm{~mL}$ 4-壬基酚标准中间液 $(1.0 \mathrm{mg} / \mathrm{L})$ ，置于 $25 \mathrm{~mL}$ 棕色容量瓶中，用甲醇稀释并定容至刻度，摇匀，然后分别吸取 $100 \mu \mathrm{L}, 250 \mu \mathrm{L}, 500 \mu \mathrm{L}, 1.25 \mathrm{~mL}, 2.50 \mathrm{~mL}, 5.00 \mathrm{~mL}$ 以及同位素内标中间液 $(1.0 \mathrm{mg} / \mathrm{L}) 125 \mu \mathrm{L}$ ，置于 $25 \mathrm{~mL}$ 棕色容量瓶中，用乙腈定容至刻度，混匀。4-壬基酚质量浓度依次为 $0.40 \mathrm{ng} / \mathrm{mL}, 1.00 \mathrm{ng} / \mathrm{mL}, 2.00 \mathrm{ng} / \mathrm{mL}, 5.00 \mathrm{ng} / \mathrm{mL}, 10.0 \mathrm{ng} / \mathrm{mL}, 20.0 \mathrm{ng} / \mathrm{mL}$ ，标准工作液中同位素内标溶液的质量浓度为 $5.0 \mathrm{ng} / \mathrm{mL}$ 。现用现配。

# 3.5 材料

3.5.1 聚四氟乙烯离心管： $5\mathrm{mL}$ 和 $10~\mathrm{mL}$

# 4 仪器和设备

4.1 液相色谱-串联质谱仪：配有电喷雾离子源。  
4.2 分析天平：感量分别为 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
4.3 涡旋混合器。  
4.4 高速冷冻离心机： $4^{\circ}\mathrm{C}$ ，转速不低于 $10000\mathrm{r / min}$ 。  
4.5 匀浆机。  
4.6 粉碎机。  
4.7 超声波清洗仪。

# 5 试样制备与保存

液态试样摇匀，基质均匀的半固态试样和粉状试样摇匀，其他试样需匀浆或粉碎均匀。

# 6 分析步骤

# 6.1 试样前处理

# 6.1.1 液态或半固态食品

称取 $1\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )于 $5\mathrm{mL}$ 聚四氟乙烯离心管中，加入 $25.0~\mu \mathrm{L}$ 同位素内标工作液 $(1.0\mathrm{mg / L})$ ，涡旋 $10~s$ 后，加入乙腈 $3.0~\mathrm{mL}$ ，具塞后涡旋振摇 $5\mathrm{s}$ ，超声提取 $10\mathrm{min},4^{\circ}\mathrm{C}$ 条件下 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，吸取全部上层溶液置于 $5\mathrm{mL}$ 容量瓶，用乙腈定容至刻度，混匀，若试液仍浑浊，可再离心1次，取上清液经液相色谱-串联质谱仪测定。

# 6.1.2 植物油脂和奶油

称取 $1\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ ) 于 $10\mathrm{mL}$ 聚四氟乙烯离心管中（奶油样品先置于 $60^{\circ}\mathrm{C}$ 水浴加热约 $5\mathrm{min}$ ，并不时轻摇以溶解脂肪），加入 $25.0\mu \mathrm{L}$ 同位素内标工作液 $(1.0\mathrm{mg / L})$ ，涡旋 $10\mathrm{s}$ 后，加入乙腈 $4.0\mathrm{mL}$ ，涡旋振摇 $2\mathrm{min}$ ，静置 $30\mathrm{min}$ ，待分层后， $4^{\circ}\mathrm{C}$ 条件下 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，吸取全部上层溶

液置于 $5 \mathrm{~mL}$ 容量瓶，用乙腈定容至刻度，混匀，若试液仍浑浊，可再离心1次，取上清液经液相色谱-串联质谱仪测定。

# 6.1.3 固体样品

称取 $1\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ ) 于 $10\mathrm{mL}$ 聚四氟乙烯离心管中，依次加入 $25.0\mu \mathrm{L}$ 同位素内标工作液 $(1.0\mathrm{mg / L})$ 和 $70\%$ 乙腈溶液 $5.0\mathrm{mL}$ （肉类及水产样品加入 $5.0\mathrm{mL}$ 乙腈），涡旋 $30\mathrm{s}$ 后，具塞超声提取 $30\mathrm{min}$ ，冷却至室温后， $4^{\circ}\mathrm{C}$ 条件下 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，吸取全部上层溶液置于 $5\mathrm{mL}$ 容量瓶，用乙腈定容至刻度，混匀，若试液仍浑浊，可再离心1次，取上清液经液相色谱-串联质谱仪测定。

# 6.1.4 乳粉和婴幼儿配方食品

称取 $1\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ )于 $5\mathrm{mL}$ 玻璃刻度管中，依次加入 $25.0~\mu \mathrm{L}$ 同位素内标工作液 $(1.0\mathrm{mg / L})$ 和 $50^{\circ}C$ 左右热水约 $1.5\mathrm{mL}$ ，涡旋 $10s$ 待充分溶解后，超声提取 $10\mathrm{min}$ ，加入乙腈 $3.5\mathrm{mL}$ 具塞后剧烈震摇 $5s$ ，吸取全部上层溶液转移到 $5\mathrm{mL}$ 聚四氟乙烯离心管中， $4^{\circ}C$ 条件下 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，吸取全部上层溶液置于 $5\mathrm{mL}$ 容量瓶，用乙腈定容至刻度，混匀，若试液仍浑浊，可再离心1次，取上清液经液相色谱-串联质谱仪测定。

# 6.2 仪器参考条件

# 6.2.1 液相色谱参考条件：

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱 $(4.6\mathrm{mm}\times 50\mathrm{mm},1.8\mu \mathrm{m})$ ，配预柱 $(4.6\mathrm{mm}\times 5\mathrm{mm},1.8\mu \mathrm{m})$ ，或性能相当者；  
b）流动相： $0.1\%$ 氨水溶液 $+$ 甲醇 $= 10 + 90$ （体积比）；  
c) 流速: $0.35 \mathrm{~mL} / \mathrm{min}$ ;   
d） 柱温： $30^{\circ}C$   
e）进样量： $5\mu \mathrm{L}$

# 6.2.2 质谱参考条件：

a）离子化模式：电喷雾电离负离子模式（ESI-）；  
b）质谱扫描方式：多反应监测(MRM)；  
c) 干燥气温度: $340^{\circ} \mathrm{C}$ (氮气);   
d) 干燥气流速: $8 \mathrm{~L} / \mathrm{min}$ (氮气);  
e) 雾化器压力：40 psi；  
f）毛细管电压： $4000\mathrm{V}$   
g）定性离子对、定量离子对、碎裂电压及碰撞能量见表1。

注：1 psi=0.006 894 8 MPa。

# 6.3 标准曲线的制作

将4-壬基酚标准系列工作液分别注入液相色谱-串联质谱仪中，测得相应的峰面积，以标准系列工

作溶液中4-壬基酚的浓度与同位素内标的浓度比为横坐标，以4-壬基酚与同位素内标峰面积比为纵坐标，绘制标准曲线。

# 6.4 试样溶液的测定

# 6.4.1 定性测定

在相同试验条件下，样品中待测物质的保留时间与标准溶液中对应的保留时间偏差在 $\pm 2.5\%$ 之内；且样品中被测组分定性离子的相对丰度与浓度接近的标准溶液中对应的定性离子的相对丰度进行比较，偏差不超过表2规定的范围，则可判定为样品中存在对应的待测物。

# 6.4.2 定量测定

将试样溶液注入液相色谱-串联质谱仪中，得到4-壬基酚与对应的同位素内标物峰面积比值，根据标准曲线得到待测液中4-壬基酚的浓度。

# 6.5 空白试验

除不称取试样外，均按照上述测定步骤进行。

# 6.6 质量控制

本标准中使用到的所有试剂和实验器皿，在使用前，应进行空白试验。如空白值高于1.0倍检出限，应对试剂重蒸，或更换试剂和实验器皿，确保空白值低于1.0倍检出限。

# 7 分析结果的表述

试样中4-壬基酚的含量按公式(1)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中4-壬基酚的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中4-壬基酚的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$\rho_0$ ——由标准曲线得到的空白试验试样溶液中4-壬基酚的质量浓度，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

V ——试样前处理中定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

计算结果以重复条件下获得的两次独立测定结果的算术平均值表示，结果保留3位有效数字。

# 8 精密度

在重复性条件下，获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 9 其他

当称样量为 $1\mathrm{g}$ ，定容体积为 $5\mathrm{mL}$ 时，检出限为 $2.0~\mu \mathrm{g / kg}$ ，定量限为 $5.0~\mu \mathrm{g / kg}$ 。

# 前言

本标准代替GB/T5009.18—2003《食品中氟的测定》。

本标准与GB/T5009.18—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中氟的测定”；  
——修改了标准的适用范围；  
——增加了第一法离子色谱法；  
——删除了原第一法扩散-氟试剂比色法和第二法灰化蒸馏——氟试剂比色法；  
——调整了原第三法氟离子选择电极法为第二法，修改了方法线性范围，增加了方法定量限。

# 食品安全国家标准

# 食品中氟的测定

# 1 范围

本标准规定了食品中氟的测定方法。

本标准第一法适用于除食盐、砖茶外食品中氟含量的测定，第二法适用于砖茶中氟含量的测定。

# 第一法 离子色谱法

# 2 原理

试样中的氟经碱固定，高温灰化后，转化为盐的形式，在中性或弱碱性条件下，用离子交换色谱柱分离，电导检测器测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为GB/T6682规定的一级水，所用器皿均为塑料材质(聚乙烯或聚丙烯材质)。

# 3.1 试剂

3.1.1 氢氧化钾(KOH)。  
3.1.2 氢氧化钠(NaOH)。  
3.1.3 碳酸钠 $\mathrm{(Na_2CO_3)}$ 。  
3.1.4 碳酸氢钠 $(\mathrm{NaHCO}_3)$ 。

# 3.2 试剂配制

3.2.1 淋洗液（碳酸钠 $8.0 \mathrm{mmol} / \mathrm{L}$ 和碳酸氢钠 $0.25 \mathrm{mmol} / \mathrm{L}$ ）：称取 $0.848 \mathrm{~g}$ 碳酸钠和 $0.021 \mathrm{~g}$ 碳酸氢钠，溶于适量水中，定容至 $1000 \mathrm{~mL}$ 。  
3.2.2 氢氧化钠溶液 $(100\mathrm{g / L})$ ：称取 $10.0\mathrm{g}$ 氢氧化钠，溶于适量水中，定容至 $100~\mathrm{mL}$ 。  
3.2.3 氢氧化钾饱和溶液：称取 $110\mathrm{g}$ 氢氧化钾，加入 $100\mathrm{mL}$ 水溶解，混匀。

# 3.3 标准品

氟化钠(NaF，CAS号：7681-49-4)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的氟标准溶液。

# 3.4 标准溶液配制

3.4.1 氟标准储备溶液 $(1000\mathrm{mg / L})$ ：准确称取 $0.2210\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ) 经 $95^{\circ}C\sim 105^{\circ}C$ 干燥至恒重的氟化钠于烧杯中，用水溶解后，转移至 $100~\mathrm{mL}$ 容量瓶中，加水定容，混匀。 $0^{\circ}C\sim 4^{\circ}C$ 密封避光保存，有效期6个月。

3.4.2 氟标准中间溶液 $(10.0\mathrm{mg / L})$ ：准确移取 $1.00~\mathrm{mL}$ 氟标准储备溶液 $(1000\mathrm{mg / L})$ ，置于 $100~\mathrm{mL}$ 容量瓶中，加水定容，混匀。 $0^{\circ}C\sim 4^{\circ}C$ 密封避光保存，有效期1个月。

3.4.3 氟标准系列工作溶液：分别准确移取氟标准中间溶液 $(10.0\mathrm{mg / L})0.000\mathrm{mL},0.0100\mathrm{mL}$ 、 $0.0250\mathrm{mL},0.0500\mathrm{mL},0.250\mathrm{mL},1.00\mathrm{mL},5.00\mathrm{mL}$ 和 $10.00~\mathrm{mL}$ ，置于 $50~\mathrm{mL}$ 容量瓶中，加水定容，混匀。氟标准系列工作溶液的浓度分别为 $0.000\mathrm{mg / L},0.00200\mathrm{mg / L},0.00500\mathrm{mg / L},$ $0.0100\mathrm{mg / L},0.0500\mathrm{mg / L},0.200\mathrm{mg / L},1.00\mathrm{mg / L},2.00\mathrm{mg / L}$ 。临用现配。

注：可根据仪器的灵敏度及试样中氟的实际含量确定标准系列工作溶液中氟的质量浓度范围。

# 3.5 材料

3.5.1 强酸型阳离子交换树脂：H型 $(2.5\mathrm{mL})$   
3.5.2 注射器： $2.5\mathrm{mL}$ 或 $5.0~\mathrm{mL}$   
3.5.3 $0.45\mu \mathrm{m}$ 水性滤膜针头滤器。  
3.5.4 离子净化柱：Ag型(1mL)和Na型(1mL)或等效柱。

# 4 仪器和设备

4.1 离子色谱仪：配电导检测器带抑制器。  
4.2 天平：感量分别为 $0.0001\mathrm{g}$ 和 $0.001\mathrm{g}$ 。  
4.3 镍坩埚 $(50~\mathrm{mL})$ 。  
4.4 可调式电热板或可调式电炉。  
4.5 马弗炉。  
4.6 高速粉碎机。  
4.7 匀浆机。

# 5 分析步骤

注：所有塑料器皿及镍坩埚均需用氢氧化钠溶液 $(100\mathrm{g / L})$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

# 5.1 试样制备

# 5.1.1 固态试样

# 5.1.1.1 干样

粮食、豆类等低含水量试样，经高速粉碎机粉碎均匀；对于特殊医学用途配方食品、玉米粉等呈均匀状的粉状试样，混匀。

# 5.1.1.2 鲜样

蔬菜、水果等高含水量试样，必要时洗净，晾干，取可食部分匀浆均匀；对于肉类、水产品、蛋类等试样，取可食部分匀浆均匀。

# 5.1.2 半固态或固液混合试样

若试样为均一体系，试样充分混合均匀；若试样为非均一体系，匀浆均匀。

# 5.1.3 液态试样

试样充分混合均匀。

# 5.2 试样前处理

# 5.2.1 试样提取

准确称取固态、半固态或固液混合试样 $0.5\mathrm{g}\sim 1.0\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )或准确移取液态试样 $1.00~\mathrm{mL}\sim 2.00~\mathrm{mL}$ 置于镍坩埚中，加入 $2\mathrm{mL}$ 氢氧化钾饱和溶液，完全覆盖于试样表面，置于可调式电热板或电炉缓慢加热，至试样完全碳化，移入马弗炉中。 $500^{\circ}C$ 灰化 $10\mathrm{min}$ ，冷却（特殊医学用途配方食品需用 $600^{\circ}C$ 灰化 $10\mathrm{min}$ )。冷却后在镍坩埚中加入 $20~\mathrm{mL}$ 水溶解试样，转移至 $50~\mathrm{mL}$ 容量瓶，加水定容，混匀，滤纸过滤后备用。同时做空白试验。

# 5.2.2 试样净化

强酸型阳离子交换树脂、 $\mathrm{Ag}$ 型离子净化柱、 $\mathrm{Na}$ 型离子净化柱(或等效净化柱)使用前，用 $10 \mathrm{~mL}$ 水通过，静置 $30 \mathrm{~min}$ 进行活化。取上述滤液约 $12 \mathrm{~mL}$ 依次通过强酸型阳离子交换树脂、 $\mathrm{Ag}$ 型离子净化柱、 $\mathrm{Na}$ 型离子净化柱、 $0.45 \mu \mathrm{m}$ 水性滤膜针头滤器，弃去最初 $8 \mathrm{~mL}$ ，收集其后流出液，待测。

# 5.3 仪器参考条件

# 5.3.1 氢氧根淋洗体系

# 5.4 标准曲线的制作

将标准系列工作溶液分别注入离子色谱仪中，测定相应的氟离子峰高或峰面积，以标准工作溶液的氟离子浓度为横坐标，以氟离子峰高或峰面积为纵坐标，绘制标准曲线。氟标准溶液的色谱图见附录A中图A.1和图A.2。

# 5.5 试样溶液的测定

将试样溶液注入离子色谱仪中，得到相应的氟离子峰高或峰面积，根据标准曲线得到试样溶液中氟离子的浓度。

# 6 分析结果的表述

试样中氟的含量按式(1)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中氟的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$\rho$ ——由标准曲线得到的试样溶液中氟离子的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——由标准曲线得到的空白溶液中氟离子的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——试样的称样质量或移取体积，单位为克或毫升(g或 $\mathrm{mL}$ )。

结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当取样量为 $0.5\mathrm{g}$ (或 $1\mathrm{mL}$ )，定容体积为 $50~\mathrm{mL}$ 时，方法检出限为 $0.2\mathrm{mg / kg}$ （或 $0.1\mathrm{mg / L}$ ，定量限为 $0.5\mathrm{mg / kg}$ （或 $0.25\mathrm{mg / L}$ ）。

# 第二法 氟离子选择电极法

# 9 原理

试样中的氟经水提取，在一定的总离子强度下，用氟离子选择电极测定，溶液中氟离子电位值与氟离子浓度的对数值在一定浓度范围内呈线性关系，与标准系列比较定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水，所用器皿均为塑料材质(聚乙烯或聚丙烯材质)。

# 10.1 试剂

10.1.1 乙酸钠 $\mathrm{(CH_3COONa\cdot 3H_2O)}$   
10.1.2 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
10.1.3 柠檬酸钠 $\mathrm{(Na_3C_6H_5O_7\cdot 2H_2O)}$ 。  
10.1.4 高氯酸 $(\mathrm{HClO}_4)$ 。  
10.1.5 盐酸（HCl）。

# 10.2 试剂配制

10.2.1 乙酸钠溶液 $(3\mathrm{mol} / \mathrm{L})$ ：称取 $204\mathrm{g}$ 乙酸钠，溶于 $300~\mathrm{mL}$ 水中，加乙酸 $(1\mathrm{mol} / \mathrm{L})$ 调节 $\mathrm{pH}$ 至7.0，加水定容至 $500~\mathrm{mL}$ 。  
10.2.2 柠檬酸钠溶液 $(0.75\mathrm{mol} / \mathrm{L})$ ：称取 $110\mathrm{g}$ 柠檬酸钠，溶于 $300~\mathrm{mL}$ 水中，加 $14~\mathrm{mL}$ 高氯酸，加水定容至 $500~\mathrm{mL}$ 。  
10.2.3 总离子强度缓冲剂：乙酸钠溶液 $(3\mathrm{mol} / \mathrm{L})$ 与柠檬酸钠溶液 $(0.75\mathrm{mol} / \mathrm{L})$ 等量混合。临用现配。  
10.2.4 盐酸溶液 $(1 + 11)$ ：量取 $10~\mathrm{mL}$ 盐酸，加入 $110~\mathrm{mL}$ 水，混匀。  
10.2.5 乙酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $3\mathrm{mL}$ 冰乙酸，溶于适量水中，定容至 $50~\mathrm{mL}$ 。

# 10.3 标准品

同3.3。

# 10.4 标准溶液配制

10.4.1 氟标准储备溶液 $(1000\mathrm{mg / L})$ ：见3.4.1。  
10.4.2 氟标准中间溶液 $(50.0\mathrm{mg / L})$ ：准确移取 $5.00~\mathrm{mL}$ 氟标准储备溶液 $(1000\mathrm{mg / L})$ 至 $100~\mathrm{mL}$ 容量瓶，加水定容，混匀。 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 密封避光保存，有效期2个月。  
10.4.3 氟标准系列工作溶液: 分别准确移取 $0.200 \mathrm{~mL} 、 0.500 \mathrm{~mL} 、 1.00 \mathrm{~mL} 、 2.00 \mathrm{~mL} 、 5.00 \mathrm{~mL} 、 8.00 \mathrm{~mL} 、 10.0 \mathrm{~mL}$ 氟标准中间溶液 $(50.0 \mathrm{mg} / \mathrm{L})$ 置于 $50 \mathrm{~mL}$ 的容量瓶中, 然后分别加入 $25 \mathrm{~mL}$ 总离子强度缓冲剂和 $10 \mathrm{~mL}$ 盐酸溶液 $(1+11)$ , 加水定容, 混匀。氟标准系列工作溶液的浓度分别为 $0.200 \mathrm{mg} / \mathrm{L} 、 0.500 \mathrm{mg} / \mathrm{L} 、 1.00 \mathrm{mg} / \mathrm{L} 、 2.00 \mathrm{mg} / \mathrm{L} 、 5.00 \mathrm{mg} / \mathrm{L} 、 8.00 \mathrm{mg} / \mathrm{L}$ 和 $10.0 \mathrm{mg} / \mathrm{L}$ 。临用现配。

# 11 仪器和设备

11.1 电极：氟离子选择电极和饱和甘汞电极，或等效复合氟离子选择性电极。  
11.2 酸度计：精度为0.01，或与之相当的离子计或电位仪。  
11.3 磁力搅拌器。  
11.4 天平：感量分别为 $0.0001\mathrm{g}$ 和 $0.001\mathrm{g}$ 。  
11.5 高速粉碎机。  
11.6 水浴锅。  
11.7 筛网：粒径 $\leqslant 425\mu \mathrm{m}$ （筛孔 $\geqslant 40$ 目）。

# 12 分析步骤

# 12.1 试样制备

将砖茶分成4份～8份，在每份不同处取样，混匀，取样品量不低于 $20\mathrm{g}$ ，用高速粉碎机粉碎均匀，粒径达 $425\mu \mathrm{m}$ 以下（相当于40目以上）。

# 12.2 试样前处理

称取 $0.2\mathrm{g}\sim 0.5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )制备好的试样于 $50~\mathrm{mL}$ 具塞三角瓶中，加 $10~\mathrm{mL}$ 水，充分混匀置于沸水浴中 $15\mathrm{min}$ ，取出冷却至室温。用 $25~\mathrm{mL}$ 总离子强度缓冲剂分次转移至 $50~\mathrm{mL}$ 容量瓶中，加入 $10~\mathrm{mL}$ 盐酸溶液 $(1 + 11)$ ，加水定容，混匀，静置备用。同时做空白试验。

# 12.3 标准曲线的制作

将氟离子选择电极和饱和甘汞电极分别与酸度计的负端与正端相连接。电极插入盛有水的 $50~\mathrm{mL}$ 烧杯中，置于磁力搅拌器上恒速搅拌，待达到平衡电位后，即可进行电位值的测定。从低浓度到高浓度依次测定标准工作溶液的电位值，以氟离子浓度对数值为横坐标、电位值为纵坐标，绘制标准曲线。

# 12.4 试样溶液的测定

取试样溶液不少于 $20~\mathrm{mL}$ 于 $50~\mathrm{mL}$ 烧杯中(若溶液浑浊，可过滤或离心后取上清液)，进行试样溶液电位值的测定，根据标准曲线计算试样溶液中氟离子的浓度。

# 13 分析结果的表述

试样中氟的含量按式(2)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中氟的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中氟离子的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——由标准曲线得到的空白溶液中氟离子的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——试样的称样质量，单位为克 $(\mathrm{g})$ 。

结果保留3位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

当取样量为 $0.2\mathrm{g}$ ，定容体积为 $50~\mathrm{mL}$ 时，定量限为 $50~\mathrm{mg / kg}$ 。

以每 $1\mathrm{min}$ 电位值变化小于 $1\mathrm{mV}$ 作为平衡电位。

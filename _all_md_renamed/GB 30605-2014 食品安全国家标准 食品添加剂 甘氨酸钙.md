# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。

试验中所用标准溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 乙酸。  
A.2.1.2 盐酸。  
A.2.1.3 草酸铵溶液： $35\mathrm{g / L}$   
A.2.1.4 亚硝酸钠溶液： $100\mathrm{g / L}$   
A.2.1.5 盐酸溶液： $1 + 3$   
A.2.1.6 甲基红指示液： $1\mathrm{g / L}$

# A.2.2 鉴别方法

# A.2.2.1 钙的鉴别

称取约 $0.5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $10\mathrm{mL}$ 水中，加入2滴甲基红指示液，用盐酸溶液中和并滴加至溶液呈酸性，滴加草酸铵溶液即产生白色沉淀。该沉淀不溶于乙酸，但溶于盐酸。

# A.2.2.2 氨基的鉴别

称取约 $0.1 \mathrm{~g}$ 试样, 精确至 $0.01 \mathrm{~g}$ , 溶于 $10 \mathrm{~mL}$ 水中, 加入盐酸溶液 $1 \mathrm{~mL}$ 和新配制的亚硝酸钠溶液 $1 \mathrm{~mL}$ 产生无色气体。

# A.2.2.3 红外光谱鉴别

用红外吸收分光光度法，采用溴化钾压片法制备试样，将试样谱图与对照谱图(见附录B)比较，两者应基本一致。

# A.3 甘氨酸钙含量的测定

# A.3.1 方法提要

试样以钙羧酸为指示剂，用乙二胺四乙酸二钠标准滴定溶液滴定，计算样品中的甘氨酸钙含量。

# A.3.2 试剂和材料

A.3.2.1 氢氧化钠溶液： $1\mathrm{mol} / \mathrm{L}$

A.3.2.2 乙二胺四乙酸二钠(EDTA)标准滴定溶液： $c(\mathrm{EDTA}) = 0.1\mathrm{mol / L}$   
A.3.2.3 钙羧酸指示剂：称取 $1\mathrm{g}$ 钙羧酸指示剂和 $100\mathrm{g}$ 氯化钠混合研细。

# A.3.3 分析步骤

称取约 $0.5\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 锥形瓶中，加入 $100~\mathrm{mL}$ 水，溶解，加入 $15~\mathrm{mL}$ 氢氧化钠溶液，加入 $0.1\mathrm{g}$ 钙羧酸指示剂，用乙二胺四乙酸二钠标准滴定溶液滴定至蓝色为终点。

在测定的同时，按与测定相同的步骤，对不加试样而使用相同数量的试剂溶液做空白试验。

# A.3.4 结果计算

甘氨酸钙含量的质量分数 $w_{1}$ 按式(A.1)计算：

$$
w _ {1} = \frac {\left(V _ {1} - V _ {0}\right) \times c \times M}{m \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——试样消耗EDTA标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{0}$ ——空白试验消耗EDTA标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——EDTA标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——甘氨酸钙的摩尔质量数值，单位为克每摩尔 $(\mathrm{g / mol})\{M[\mathrm{Ca(OOCCH_2NH_2)_2}\cdot \mathrm{H_2O}] = 206.21\}$

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

1000——换算因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.3\%$ 。

# A.4 氮的测定

# A.4.1 方法提要

用凯氏定氮法测定样品中的氮含量。

# A.4.2 试剂和材料

A.4.2.1 硫酸钾。  
A.4.2.2 五水硫酸铜。  
A.4.2.3 硫酸。  
A.4.2.4 氢氧化钠溶液： $300\mathrm{g / L}$   
A.4.2.5 硼酸溶液： $20\mathrm{g / L}$   
A.4.2.6 盐酸标准滴定溶液： $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$   
A.4.2.7 甲基红-次甲基蓝混合指示液。

# A.4.3 分析步骤

# A.4.3.1 自动定氮仪定氮法（仲裁法）

称取约 $0.25\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于自动定氮仪消化管中，加入 $3.0\mathrm{g}$ 硫酸钾， $0.15\mathrm{g}$ 五水硫酸铜，沿瓶壁缓缓加入 $10\mathrm{mL}$ 硫酸。置于消化装置上，于 $400^{\circ}\mathrm{C}$ 消化 $40\mathrm{min}$ ，冷却。置于自动定氮仪装置上，消化管中加入 $60\mathrm{mL}$ 氢氧化钠溶液，吸收瓶中加入 $50\mathrm{mL}$ 硼酸溶液和8滴甲基红-次甲基蓝混合

指示液，蒸馏至馏出液中性。用盐酸标准滴定溶液滴定至灰紫色为终点。

在测定的同时，按与测定相同的步骤，对不加试样而使用相同数量的试剂溶液做空白试验。

# A.4.3.2 直接蒸馏法

按HG/T4103中直接蒸馏法进行测定。测定时称取 $0.3\mathrm{g}$ 试样。

# A.4.4 结果计算

氮含量（以干基计）的质量分数 $w_{2}$ 按式(A.2)计算：

$$
w _ {2} = \frac {\left(V _ {1} - V _ {0}\right) \times c \times M}{m \times 1000} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$V_{1}$ ——试样消耗盐酸标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗盐酸标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——盐酸标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——氮的摩尔质量数值，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{N}) = 14.01]$ ；

$m$ ——试样质量的数值，单位为克(g)；

1000——换算因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

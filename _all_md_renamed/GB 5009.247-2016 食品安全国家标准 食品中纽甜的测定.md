# 前言

本标准代替GB/T23378—2009《食品中纽甜的测定方法 高效液相色谱法》。

本标准与GB/T23378—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中纽甜的测定”。

——适用范围更改为饮料、蜜饯、糕点、炒货、酱腌菜、糖果、果酱、果冻、复合调味料中纽甜的测定。

# 食品安全国家标准

# 食品中纽甜的测定

# 1 范围

本标准规定了食品中纽甜含量的高效液相色谱测定方法。

本标准适用于饮料、蜜饯、糕点、炒货、酱腌菜、糖果、果酱、果冻、复合调味料食品中纽甜的测定。

# 2 原理

试样经混合提取液提取，固相萃取柱净化后，采用高效液相色谱仪测定，保留时间定性，峰面积外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 辛烷磺酸钠 $\left(\mathrm{C}_{8} \mathrm{H}_{17} \mathrm{NaO}_{3} \mathrm{~S}\right)$ ：色谱纯。  
3.1.3 磷酸 $\mathrm{(H_3PO_4)}$   
3.1.4 甲酸 $\mathrm{(CH_3COOH)}$ ：色谱纯。  
3.1.5 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.6 三乙胺 $\mathrm{(C_6H_{15}N)}$ ：色谱纯。  
3.2 试剂配制  
3.2.1 混合提取液：分别吸取 $0.8\mathrm{mL}$ 甲酸和 $2.5\mathrm{mL}$ 三乙胺，加水定容至 $1000\mathrm{mL},\mathrm{pH}$ 约4.5。  
3.2.2 离子对试剂缓冲液：称取 $2.00\mathrm{g}$ 辛烷磺酸钠，用 $500~\mathrm{mL}$ 水溶解，加入 $1.0~\mathrm{mL}$ 磷酸，加水定容至 $1000~\mathrm{mL}$ 。  
3.3 标准品

纽甜 $\mathrm{(C_{20}H_{30}N_2O_5}$ ，CAS号：165450-17-9)，纯度 $\geq 99.0\%$

# 3.4 标准溶液配制

3.4.1 标准储备液：准确称取 $0.1000\mathrm{g}$ 纽甜标准品，加混合提取液溶解定容至 $100~\mathrm{mL}$ ，此溶液纽甜含量为 $1.00\mathrm{mg / mL}$ 。  
3.4.2 标准工作液：分别吸取适量纽甜标准储备液，用混合提取液配制成浓度分别为 $0.2\mu \mathrm{g / mL}$ 、 $1.0\mu \mathrm{g / mL},5.0\mu \mathrm{g / mL},10.0\mu \mathrm{g / mL},50.0\mu \mathrm{g / mL},100.0\mu \mathrm{g / mL}$ 的系列标准工作液。  
3.5 $\mathrm{C_{18}}$ 固相萃取柱： $6\mathrm{mL},500\mathrm{mg}$ ，或相当者，使用前依次用 $5\mathrm{mL}$ 甲醇、 $10~\mathrm{mL}$ 水活化。  
3.6 $0.45\mu \mathrm{m}$ 滤膜，有机系。

# 4 仪器和设备

4.1 液相色谱仪，配有紫外检测器或二极管阵列检测器。  
4.2 超声波清洗仪。  
4.3 分析天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$   
4.4 组织捣碎机。  
4.5 旋涡振荡器。  
4.6 氮吹仪。  
4.7 固相萃取装置。  
4.8 离心机：转速 $\geqslant 4000\mathrm{r / min}$

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固态样品

称取 $10\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 粉碎均匀试样于 $50~\mathrm{mL}$ 具塞塑料离心管中，加入 $30~\mathrm{mL}$ 混合提取液，旋涡振荡 $10\mathrm{min}$ ，超声 $30\mathrm{min}$ 后，用混合提取液定容至刻度，若溶液混浊，以不低于 $4000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ 后，过滤待用。

# 5.1.2 液态样品

准确量取 $10.0\mathrm{mL}$ 试样于 $50~\mathrm{mL}$ 具塞塑料离心管中，加入 $30~\mathrm{mL}$ 混合提取液，振荡混匀。超声 $15\mathrm{min}$ 后，用混合提取液定容至刻度，若溶液混浊，以不低于 $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 后，过滤待用。

注：含气样品如碳酸饮料、汽水等先微温，搅拌去除试样中的二氧化碳或超声脱气后，再准确量取试样。

# 5.2 试样净化

吸取 $10.0\mathrm{mL}$ 的过滤液以 $1\mathrm{mL} / \mathrm{min}\sim 2\mathrm{mL} / \mathrm{min}$ 的流速通过固相萃取柱，待滤液完全流出后，用 $5\mathrm{mL}$ 的混合提取液以 $1\mathrm{mL} / \mathrm{min}\sim 2\mathrm{mL} / \mathrm{min}$ 的流速淋洗萃取柱，弃去全部流出液，用 $5\mathrm{mL}$ 甲醇以 $1\mathrm{mL} / \mathrm{min}$ 的流速洗脱，洗脱液在 $40^{\circ}\mathrm{C}$ 水浴中用氮吹仪浓缩，用混合提取液定容至 $2.0~\mathrm{mL}$ ，经 $0.45\mu \mathrm{m}$ 滤膜过滤后作为待测液供液相色谱仪分析。

# 5.3 仪器参考条件

5.3.1 色谱柱： $\mathrm{C_{18}}$ 色谱柱， $5 \mu \mathrm{m}$ ， $250 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ （内径）或相当者。  
5.3.2 柱温： $30^{\circ}C$ 。  
5.3.3 流动相：A相：乙腈；B相：离子对试剂缓冲液。梯度洗脱程序见表1。  
5.3.4 流速： $1.0~\mathrm{mL / min}$   
5.3.5 检测波长： $218~\mathrm{nm}$   
5.3.6 进样量： $50~\mu \mathrm{L}$

# 5.4 标准曲线的制作

将标准系列工作液分别注入液相色谱仪中，测定相应的峰面积，以标准工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线(纽甜标准色谱图见图A.1)。

# 5.5 试样溶液的测定

将试样溶液注入液相色谱仪中，以保留时间定性，同时记录峰面积，根据标准曲线得到待测液中纽甜的浓度，试样溶液中纽甜含量的响应值均应在仪器的检测线性范围内。

# 6 分析结果的表述

试样中纽甜的含量按式(1)计算：

$$
X = \frac {c \times V \times V _ {1} \times 1 0 0 0}{m \times V _ {2} \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中纽甜含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 或毫克每升 $(\mathrm{mg / L})$ ；

$c$ ——试样溶液中纽甜的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样品洗脱液浓缩定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——试样提取液定容体积，单位为毫升 $(\mathrm{mL})$

$m$ ——样品质量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——吸取过滤液体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本标准方法的定量限为 $0.2\mathrm{mg / kg}$

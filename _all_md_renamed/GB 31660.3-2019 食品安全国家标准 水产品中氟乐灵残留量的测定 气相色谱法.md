# 食品安全国家标准

# 水产品中氟乐灵残留量的测定 气相色谱法

# 1 范围

本标准规定了水产品中氟乐灵残留量检测的制样和气相色谱测定方法。

本标准适用于鱼、虾、蟹、鳖、贝类等水产品的可食组织中氟乐灵残留量的检测。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的，凡是注日期的引用文件，仅所注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法。

# 3 原理

试样中氟乐灵残留经丙酮提取，正己烷液液萃取，弗罗里硅土柱净化后，气相色谱电子捕获检测器测定，以外标法定量。

# 4 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 丙酮（ $\mathrm{CH}_3\mathrm{COCH}_3$ ）：色谱纯。

4.1.2 正己烷（ $\mathrm{C_6H_{14}}$ ）：色谱纯。

4.1.3 二氯甲烷（ $\mathrm{CH}_2\mathrm{Cl}_2$ ）：色谱纯。

4.1.4 无水硫酸钠（ $\mathrm{Na}_2\mathrm{SO}4$ ）。

# 4.2 溶液配制

4.2.1 $2\%$ 硫酸钠溶液：称取无水硫酸钠 $2\mathrm{g}$ ，加水溶解并稀释至 $100\mathrm{mL}$ 。

4.2.2 $10\%$ 二氯甲烷正己烷溶液：取二氯甲烷 $10 \mathrm{~mL}$ ，加正己烷溶解并稀释至 $100 \mathrm{~mL}$ ，混匀。

# 4.3 标准品

氟乐灵（Trifluralin $\mathrm{C_{13}H_{16}F_3N_3O_4}$ CAS号：1582-09-9），含量 $\geq 98.0\%$

# 4.4 标准溶液的制备

4.4.1 标准贮备液（ $100\mu \mathrm{g} / \mathrm{mL}$ ）：取氟乐灵约 $10\mathrm{mg}$ ，精密称定，于 $100\mathrm{mL}$ 棕色量瓶中，用正己烷溶解并稀释至刻度，配制成浓度为 $100\mu \mathrm{g} / \mathrm{mL}$ 标准贮备液。 $4^{\circ}\mathrm{C}$ 以下避光保存，有效期6个月。  
4.4.2 标准工作液（ $1\mu \mathrm{g} / \mathrm{mL}$ ）：精密量取标准贮备液 $1\mathrm{mL}$ ，于 $100\mathrm{mL}$ 棕色量瓶中，用正己烷溶解并稀释至刻度，配制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 氟乐灵标准工作液。 $4^{\circ}\mathrm{C}$ 以下避光保存，有效期3个月。  
4.4.3 标准工作液（ $0.1\mu \mathrm{g} / \mathrm{mL}$ ）：精密量取 $1\mu \mathrm{g} / \mathrm{mL}$ 标准贮备液 $1\mathrm{mL}$ ，于 $10\mathrm{mL}$ 棕色量瓶中，用正己烷溶解并稀释至刻度，配制成浓度为 $0.1\mu \mathrm{g} / \mathrm{mL}$ 氟乐灵标准工作液。 $4^{\circ}\mathrm{C}$ 以下避光保存，有效期2个周。

# 4.5材料

弗罗里硅土固相萃取柱： $1\mathrm{g} / 6\mathrm{mL}$ ，或相当者。

# 5 仪器和设备

5.1 气相色谱仪：配电子捕获检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
5.3 氮吹仪。  
5.4 均质机。  
5.5 旋涡混合器。  
5.6 离心机： $4000\mathrm{r / min}$   
5.7 超声波振荡器。  
5.8 旋转蒸发器。  
5.9 固相萃取装置。  
5.10 具塞聚丙烯离心管： $50~\mathrm{mL}$ 和 $100\mathrm{mL}$ 。  
5.11 玻璃离心管： $10~\mathrm{mL}$   
5.12 棕色鸡心瓶： $100~\mathrm{mL}$

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

- $18^{\circ} \mathrm{C}$ 以下保存，3个月内进行分析检测。

# 7 测定步骤

# 7.1 提取

取试样 $2\mathrm{g}$ （准确至 $\pm 20\mathrm{mg}$ ，于 $50~\mathrm{mL}$ 具塞聚丙烯离心管中，加丙酮 $10~\mathrm{mL}$ ，旋涡 $1\mathrm{min}$ $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液，残渣加丙酮 $10~\mathrm{mL}$ ，重复提取一次，合并上清液，加正己烷 $30~\mathrm{mL}$ 、 $2\%$ 硫酸钠溶液 $10~\mathrm{mL}$ ，旋涡 $1\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液于 $100~\mathrm{mL}$ 棕色鸡心瓶中，下层液体再加正己烷 $20~\mathrm{mL}$ 重复提取一次，合并上清液，于 $40^{\circ}C$ 旋转蒸发至近干，加正己烷 $2~\mathrm{mL}$ 使溶解，转移至 $10~\mathrm{mL}$ 玻璃离心管，鸡心瓶用正己烷 $2~\mathrm{mL}$ 洗涤一次，洗涤液合并入 $10~\mathrm{mL}$ 玻璃离心管，用氮气吹至约 $1~\mathrm{mL}$ ，备用。

# 7.2 净化

固相萃取柱用二氯甲烷 $5\mathrm{mL}$ 预洗，吹干，再用正己烷 $5\mathrm{mL}$ 淋洗；取备用液过柱，用正己烷 $3\mathrm{mL}$ 分三次洗玻璃离心管，洗液一并上柱，弃流出液；用 $10\%$ 二氯甲烷正己烷溶液 $5\mathrm{mL}$ 洗脱，收集洗脱液，氮气吹至近干。准确加正己烷 $5\mathrm{mL}$ 溶解残余物，供气相色谱测定。

# 7.3 标准曲线的制备

精密量取氟乐灵标准工作液 $(0.1\mu \mathrm{g / mL})$ 适量，用正己烷稀释，配制成浓度为0.25、1.0、5.0、10、 $20~\mathrm{ng / mL}$ 的系列标准工作溶液；现用现配。以峰面积为纵坐标，标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.4 测定

# 7.4.1 色谱参考条件

a) 色谱柱: HP-5ms石英毛细管柱 $(30 \mathrm{~m} \times 0.25 \mathrm{~mm} \times 0.25 \mu \mathrm{m})$ , 或相当者;   
b）载气：高纯氮气，纯度 $\geq 99.999\%$ ；流速： $1.2~\mathrm{mL / min}$   
c）进样方式：无分流进样；  
e）进样量： $1\mu \mathrm{L}$   
f）进样口温度： $230^{\circ}\mathrm{C}$   
g）柱温：初始柱温 $70^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $30^{\circ}\mathrm{C / min}$ 升至 $185^{\circ}\mathrm{C}$ ，保持 $2.5\mathrm{min}$ ，再以 $25^{\circ}\mathrm{C / min}$ 升至 $280^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ ；  
h）检测器：ECD；检测器温度： $300^{\circ}\mathrm{C}$ 。

# 7.4.2 测定法

在7.4.1规定的色谱条件下，以标准工作溶液浓度为横坐标，以峰面积为纵坐标，绘制标准工作曲线，作单点或多点校准，按外标法计算试样中药物的残留量，标准工作液和试样液中待测物的响应值均应在仪器检测线性范围内。在上述色谱-质谱条件下，标准溶液色谱图见附录A。

# 7.5 空白试验

除不加试料外，均按上述测定步骤进行。

# 8 结果计算和表述

试样中氟乐灵的残留量按式（1）计算：

$$
X = \frac {C _ {s} \times A \times V}{A _ {s} \times m} \dots \tag {1}
$$

式中：

$X$ —试样中氟乐灵的残留量，单位为微克每千克（ $\mu \mathrm{g} / \mathrm{kg}$ ）；

$C_{s}$ ——标准溶液中氟乐灵的浓度，单位为纳克每毫升（ $\mathrm{ng/mL}$ ）；

A—试样溶液中氟乐灵峰面积；

$\mathrm{A_s}$ 一标准溶液中氟乐灵峰面积；

$V$ —试样溶液定容体积, 单位为毫升 $(\mathrm{mL})$ ;

$m$ —试料质量，单位为克（g）。

计算结果需扣除空白值。测定结果用两次平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法的灵敏度、准确度和精密度

# 9.1 灵敏度

本方法检测限为 $0.5\mu \mathrm{g / kg}$ ；定量限为 $1.0~\mu \mathrm{g / kg}$ 。

# 9.2 准确度

氟乐灵在 $1\mu \mathrm{g} / \mathrm{kg}\sim 10\mu \mathrm{g} / \mathrm{kg}$ 添加浓度的回收率为 $70\% \sim 110\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

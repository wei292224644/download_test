# 1 范围

本标准适用于以食品添加剂磷酸和液氨为原料生产的食品添加剂磷酸氢二铵。

# 2 分子式和相对原子量

# 2.1 分子式

$(\mathrm{NH}_4)_2\mathrm{HPO}_4$

# 2.2 相对分子质量

132.03（按2011年国际相对原子质量）。

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，操作时应采取适当的安全和防护措施。必要时，需在通风橱中进行。如溅到皮肤上应立即用水冲洗，严重者应立即治疗。使用剧毒品时，应严格按照有关规定管理；使用时应避免吸入或与皮肤接触，必要时应在通风橱中进行。对于暴露部位有伤口的人员不能接触。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T602、GB/T603之规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 硝酸溶液： $1 + 8$   
A.3.1.2 氨水溶液： $1 + 1$   
A.3.1.3 氢氧化钠溶液： $40\mathrm{g / L}$   
A.3.1.4 硝酸银溶液： $17\mathrm{g / L}$   
A.3.1.5 红色石蕊试纸。

# A.3.2 鉴别方法

# A.3.2.1 磷酸根离子的鉴别

称取约 $1.0\mathrm{g}$ 试样，溶于 $20~\mathrm{mL}$ 水中，用硝酸溶液调节至中性。加入硝酸银溶液，可生成黄色沉淀，此沉淀溶于氨水溶液或硝酸溶液。

# A.3.2.2 铵根离子鉴别

称取约 $1.0\mathrm{g}$ 试样，溶于 $20~\mathrm{mL}$ 水中，加入氢氧化钠溶液，释放出的氨气可使湿润的红色石蕊试纸变蓝。加热可促进分解。

# A.4 磷酸氢二铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{HPO}_{4}\right]$ 含量的测定

# A.4.1 方法提要

在酸性条件下，以喹钼柠酮做沉淀剂使试样溶液中的磷酸根全部形成磷钼酸喹啉沉淀，沉淀经过滤、烘干、称重后确定试样中磷酸氢二铵含量。

# A.4.2 试剂和材料

# A.4.2.1 硝酸溶液： $1 + 1$

A.4.2.2 喹钼柠酮溶液：称取 $70\mathrm{g}$ 钼酸钠，溶于 $150~\mathrm{mL}$ 水中，为溶液1。称取 $60\mathrm{g}$ 柠檬酸，溶于 $150~\mathrm{mL}$ 水和 $85~\mathrm{mL}$ 硝酸的混合液中，为溶液2。边搅拌边将溶液1缓慢加入溶液2中，为溶液3。在 $35~\mathrm{mL}$ 硝酸和 $100~\mathrm{mL}$ 水的混合液中加入 $5\mathrm{mL}$ 喹啉，为溶液4。将溶液4加入溶液3中，搅拌均匀，放置 $24\mathrm{h}$ ，过滤，滤液中加入 $280~\mathrm{mL}$ 丙酮，用水稀释至 $1000~\mathrm{mL}$ ，混匀。

警告：此溶液应保存于聚乙烯瓶中。此溶液中含丙酮，不应靠近火焰使用。操作中如要加热或煮沸应在通风橱中进行。

# A.4.3 仪器和设备

A.4.3.1 玻璃砂芯坩埚：孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$   
A.4.3.2 电热恒温干燥箱：温度能控制为 $180^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。

# A.4.4 分析步骤

# A.4.4.1 试样溶液的制备

称取约 $1\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ 。置于 $100\mathrm{mL}$ 烧杯中，加少量水溶解。移入 $250\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。干过滤，弃去初始 $20\mathrm{mL}$ 滤液，保留滤液作为试样溶液。

# A.4.4.2 测定

用移液管移取 $10\mathrm{mL}$ 试样溶液，置于 $300~\mathrm{mL}$ 烧杯中。加 $10\mathrm{mL}$ 硝酸溶液，加水至总体积约为 $100~\mathrm{mL}$ ，盖上表面皿，在水浴中加热至烧杯内的物质达到 $75^{\circ}C\pm 5^{\circ}C$ ，加入 $50~\mathrm{mL}$ 喹钼柠酮溶液（在通风橱中进行），保温 $30~s$ （在加入试剂和加热过程中，不得使用明火，不得搅拌，以免凝结成块）。冷却，在冷却过程中搅拌3次～4次。将沉淀抽滤于预先在 $180^{\circ}C\pm 2^{\circ}C$ 下干燥至质量恒定的玻璃砂芯坩埚中。先将上层清液过滤，以倾析法用洗瓶冲洗沉淀6次，每次用水约 $30~\mathrm{mL}$ ，最后将沉淀移入玻璃砂芯坩埚中过滤，再用水洗涤沉淀4次。将玻璃砂芯坩埚连同沉淀置于电热恒温干燥箱中，从温度稳定开始计时，在 $180^{\circ}C\pm 2^{\circ}C$ 下干燥 $45\mathrm{min}$ 。取出稍冷后，置于干燥器中冷却至室温，称重。

同时做空白试验，空白试验除不加试样外，其他加入的试剂种类和量与试样溶液完全相同，并与试样溶液同样处理。

# A.4.5 结果计算

磷酸氢二铵 $\left[\mathrm{(NH_4)_2HPO_4}\right]$ 含量的质量分数 $\varpi_{1}$ 按式(A.1)计算：

$$
w _ {1} = \frac {\left(m _ {1} - m _ {2}\right) \times 0.05966 \times 250}{m \times 10} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$m_{1}$ ——试样溶液中生成磷钼酸喹啉沉淀的质量，单位为克(g)；

$m_{2}$ ——空白试样溶液中生成磷钼酸喹啉沉淀的质量，单位为克(g)；

0.05966 磷钼酸喹啉换算成磷酸氢二铵的系数；

250 ——容量瓶的容积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

10 ——移取试样溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差

值不大于 $0.3\%$ 。

# A.5 pH(10 g/L 水溶液)的测定

# A.5.1 试剂和材料

无二氧化碳的水。

# A.5.2 仪器和设备

酸度计：精度 $0.02\mathrm{pH}$ 。

# A.5.3 分析步骤

称取 $1.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $150~\mathrm{mL}$ 烧杯中，加 $100~\mathrm{mL}$ 无二氧化碳的水溶解试样。用已经校对好的酸度计进行测定。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于0.1。

# A.6 铅(Pb)的测定

# A.6.1 试剂和材料

A.6.1.1 盐酸。

A.6.1.2 硝酸。

A.6.1.3 三氯甲烷。

A.6.1.4 氢氧化钠溶液： $250~\mathrm{g / L}$

A.6.1.5 吡咯烷二硫代氨基甲酸铵(APDC)溶液: $20 \mathrm{~g} / \mathrm{L}$ , 溶解 $2 \mathrm{~g}$ 吡咯烷二硫代氨基甲酸铵(APDC)于 $100 \mathrm{~mL}$ 水中。使用前过滤。该溶液现用现配。

A.6.1.6 铅(Pb)标准溶液: $0.005 \mathrm{mg} / \mathrm{mL}$ , 用移液管移取 $5 \mathrm{~mL}$ 按 GB/T 602 配制的铅标准溶液, 置于 $100 \mathrm{~mL}$ 容量瓶中, 用水稀释至刻度, 摇匀。该溶液现用现配。

A.6.1.7 精密 $\mathrm{pH}$ 试纸： $0.5\sim 5.0$

A.6.1.8 水：符合GB/T6682—2008中二级水的规定。

# A.6.2 仪器和设备

A.6.2.1 分液漏斗： $250~\mathrm{mL}$

A.6.2.2 原子吸收分光光度计：配有铅空心阴极灯。

# A.6.3 分析步骤

# A.6.3.1 试样溶液的制备

称取 $5.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $150~\mathrm{mL}$ 烧杯中，加 $30~\mathrm{mL}$ 水溶解，加 $1\mathrm{mL}$ 盐酸。加热煮沸几分钟，冷却，用水稀释至约 $100~\mathrm{mL}$ ，用氢氧化钠溶液调整溶液 $\mathrm{pH}$ 为 $1.0\sim 1.5$ （用精密 $\mathrm{pH}$ 试纸检验）。将此溶液转移至 $250~\mathrm{mL}$ 分液漏斗中，用水稀释至约 $200~\mathrm{mL}$ 。加 $2\mathrm{mL}$ 吡咯烷二硫代氨基甲酸铵（APDC）溶液，混匀。分别用 $20~\mathrm{mL}$ 三氯甲烷萃取两次，收集萃取物于 $50~\mathrm{mL}$ 烧杯中，汽浴蒸发至干（此操作应在通风橱中进行），加入 $3\mathrm{mL}$ 硝酸，于汽浴中蒸发至近干。加入 $0.5\mathrm{mL}$ 硝酸和 $10~\mathrm{mL}$ 水，蒸发至溶液体积约为 $3\mathrm{mL}\sim 5\mathrm{mL}$ 。转移至 $10~\mathrm{mL}$ 容量瓶，用水稀释至刻度，摇匀。

# A.6.3.2 标准溶液的制备

移取 $4.00\mathrm{mL}$ 铅标准溶液置于 $150~\mathrm{mL}$ 烧杯中。以下操作同A.6.3.1中从“加 $30~\mathrm{mL}$ 水溶解……”开始，至“转移至 $10~\mathrm{mL}$ 容量瓶，用水稀释至刻度，摇匀”为止。

# A.6.3.3 测定

在 $283.3\mathrm{nm}$ 处，使用空气-乙炔火焰，以水调零，测定吸光度，试样溶液的吸光度不应高于标准溶液的吸光度。

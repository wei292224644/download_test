# 1 范围

本标准适用于以适宜的发酵用营养物质接种未经基因修饰的酵母菌种，经发酵培养将培养基中含有的亚硒酸钠转化为有机态硒，再经分离、干燥制得的食品营养强化剂富硒酵母。

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度均为分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 有机硒占总硒质量分数的测定

# A.2.1 试剂和材料

同GB5009.93—2010第一法。

# A.2.2 仪器和设备

同GB5009.93—2010第一法。

# A.2.3 无机硒测定的试样制备

称取约 $1\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 容量瓶中，加入约 $25~\mathrm{mL}$ 水，超声溶解 $10\mathrm{min}$ ，使试样充分溶解，加水定容至刻度，摇匀。将定容后的溶液置于 $5000\mathrm{r / min}$ 的离心机中，离心 $10\mathrm{min}$ ，吸取 $10.0~\mathrm{mL}$ 离心上清液于 $15~\mathrm{mL}$ 离心管中，加盐酸 $2.0~\mathrm{mL}$ 、铁氰化钾溶液 $1.0~\mathrm{mL}$ ，混匀待测。同时做空白试验。

# A.2.4 分析步骤

按照GB5009.93氢化物原子荧光光谱法测定无机硒含量，再由之前得到的总硒含量减去无机硒含量得到有机硒含量，从而计算出有机硒的百分含量。

# A.2.5 结果计算

A.2.5.1 无机硒含量 $X_{1}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.1)计算：

$$
X _ {1} = \frac {\left(c - c _ {0}\right) \times 5 0 \times 1 0 0 0}{m _ {1} \times 1 0 0 0 \times 1 0 0 0} \tag {A.1}
$$

式中：

$c$ ——试样提取液测定浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{0}$ ——无机硒空白测定浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

50 ——试样的定容体积，单位为毫升 $(\mathrm{mL})$

1000——换算系数；

$m_{1}$ ——无机硒测定时称取的试样质量，单位为克 $(\mathrm{g})$ 。

结果保留3位有效数字。

A.2.5.2 有机硒含量 $X_{2}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.2)计算：

$$
X _ {2} = X - X _ {1} \quad \dots \tag {A.2}
$$

式中：

$X$ ——试样中总硒的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$X_{1}$ ——试样中无机硒的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 。

A.2.5.3 有机硒占总硒含量的质量分数 $\omega_{1}$ ，按式(A.3)计算：

$$
w _ {1} = \frac {X _ {2}}{X} \times 100 \% \tag{A.3}
$$

式中：

$X_{2}$ ——试样中有机硒的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$X$ ——试样中总硒的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 。

试验结果以平行测定结果的算术平均值为准。结果保留至小数点后1位。

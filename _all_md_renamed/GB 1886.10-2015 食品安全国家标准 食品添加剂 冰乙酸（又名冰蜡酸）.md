# 附录A

# 检验方法

# A.1 警示

试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和防护措施。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试剂中所用标准溶液、杂质测定用标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603之规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸。  
A.3.1.2 硫酸。  
A.3.1.3 乙醇 $(95\%)$ 。  
A.3.1.4 氢氧化钠溶液： $40~\mathrm{g / L}$   
A.3.1.5 三氯化铁溶液： $50~\mathrm{g / L}$

# A.3.2 分析步骤

A.3.2.1 取 $1 \mathrm{~mL}$ 试样, 加氢氧化钠溶液中和, 并加三氯化铁溶液即呈深红色, 煮沸, 即发生红棕色沉淀, 再加盐酸, 则溶解成黄色溶液。  
A.3.2.2 取少许试样，加硫酸与少量乙醇，加热，即发生乙酸乙酯的特有香气。

# A.4 乙酸含量的测定

# A.4.1 方法提要

以酚酞为指示剂，用氢氧化钠标准滴定溶液滴定，根据消耗氢氧化钠标准滴定溶液的体积计算乙酸的含量。

# A.4.2 试剂和材料

A.4.2.1 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.5\mathrm{mol / L}$   
A.4.2.2 酚酞指示液： $10\mathrm{g / L}$

# A.4.3 分析步骤

称取 $1\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 具塞锥瓶中，锥瓶中预先装有新煮沸并冷却的 $80~\mathrm{mL}$ 水，加2滴酚酞指示液，用氢氧化钠标准滴定溶液滴定至溶液呈淡粉红色，保持 $30~\mathrm{s}$ 不褪色为终点。

# A.4.4 结果计算

乙酸含量的质量分数 $w_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {V \times c \times M}{m} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V$ ——氢氧化钠标准滴定溶液(A.4.2.1)的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——氢氧化钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——乙酸的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ ， $[M(\mathrm{CH}_3\mathrm{COOH}) = 0.06005]$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 高锰酸钾试验

# A.5.1 方法提要

试样中还原物质与高锰酸钾作用，在规定时间内，溶液中出现的粉红色不应消失。

# A.5.2 试剂和材料

高锰酸钾溶液： $3.1\mathrm{g / L}$

# A.5.3 分析步骤

量取 $2\mathrm{mL}$ 试样，置于具塞比色管中。加入 $10~\mathrm{mL}$ 水溶解，于 $20^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 水浴中加入 $0.1\mathrm{mL}$ 高锰酸钾溶液，摇匀，溶液粉红色在 $30\mathrm{min}$ 内不应消失。

# A.6 蒸发残渣的测定

# A.6.1 方法提要

在水浴上将试样蒸干后，并在烘箱中规定温度规定时间干燥，称量出残渣的质量。

# A.6.2 分析步骤

称取 $50\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ 。置于预先恒重的 $100~\mathrm{mL}$ 蒸发皿中，在水浴上蒸干后，置于 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 的恒温干燥箱中干燥 $2\mathrm{h}$ 。保留残渣用于重金属含量的测定。

# A.6.3 结果计算

蒸发残渣的质量分数 $\varpi_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {m _ {1}}{m} \times 100 \% \tag{A.2}
$$

式中：

$m_{1}$ ——残渣的质量，单位为克(g)；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

取两次平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于这两个测定值的算术平均值的 $20\%$ 。

# A.7 游离矿酸的测定

# A.7.1 原理

试样中有游离矿酸(硫酸、硝酸、盐酸)存在时，氢离子浓度增大，可改变指示剂的颜色。

# A.7.2 试剂和材料

百里草酚蓝试纸：取 $0.1\mathrm{g}$ 百里草酚蓝，溶于 $50~\mathrm{mL}$ 乙醇中，加 $6\mathrm{mL}$ 氢氧化钠溶液 $(4\mathrm{g / L})$ ，加水至 $100~\mathrm{mL}$ 。将滤纸浸透此液后晾干，备用。

# A.7.3 分析步骤

用毛细管或玻璃棒沾少许试样，点在百里草酚蓝试纸上，待试纸晾干（约 $5\mathrm{min}\sim 10\mathrm{min}$ )后，观察试纸变化情况，若试纸出现紫色斑点，表示有游离矿酸存在。以未检出游离矿酸为合格。

# 前言

本标准代替GB5009.259—2016《食品安全国家标准 食品中生物素的测定》。

本标准与GB5009.259—2016相比，主要变化如下：

——增加了第一法 液相色谱-串联质谱法，微生物法调整为第二法；  
——第二法 微生物法增加了微孔板培养法；  
——修改了标准的适用范围、样品类别和前处理方法；  
——修改了标准曲线和样品提取液制备的表述方式；  
——修改了检出限、定量限和线性范围。

# 食品安全国家标准

# 食品中生物素的测定

# 1 范围

本标准规定了食品中生物素的测定方法。

第一法 液相色谱-串联质谱法适用于调制乳粉、特殊膳食用食品中生物素的测定。

第二法 微生物法适用于食品中生物素的测定。

# 第一法 液相色谱-串联质谱法

# 2 原理

试样经溶解提取，对含淀粉试样经淀粉酶酶解后，经蛋白沉淀、离心过滤后， $\mathrm{C}_{18}$ 反相色谱柱分离，采用液相色谱-串联质谱多离子反应监测方式检测，同位素稀释内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂和材料

3.1.1 甲酸（HCOOH）：色谱纯。  
3.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.3 乙醇 $\mathrm{(CH_3CH_2OH)}$ ：色谱纯。  
3.1.4 甲酸铵 $\mathrm{(HCOONH_4)}$ ：色谱纯。  
3.1.5 高氯酸 $\mathrm{(HClO_4)}$ ： $70\% \sim 72\%$   
3.1.6 氢氧化钠(NaOH)：纯度 $\geqslant 99.9\%$   
3.1.7 淀粉酶：Taka-淀粉酶，CAS：9001-19-8，酶活力 $\geqslant 100\mathrm{U / mg}$

# 3.2 试剂配制

3.2.1 $0.1\%$ 甲酸-10 mmol/L 甲酸铵水溶液：称取 $0.63 \mathrm{~g}$ 甲酸铵，用 $100 \mathrm{~mL}$ 水溶解后，转移至 $1000 \mathrm{~mL}$ 试剂瓶中，加入 $1 \mathrm{~mL}$ 甲酸，以水稀释至 $1000 \mathrm{~mL}$ ，摇匀备用。  
3.2.2 氢氧化钠溶液 $(2\mathrm{mol} / \mathrm{L})$ ：称取 $8.00\mathrm{g}$ 氢氧化钠于烧杯中，加入 $100~\mathrm{mL}$ 水溶解，摇匀备用。  
3.2.3 乙醇溶液 $(50\%)$ ：准确量取 $500~\mathrm{mL}$ 乙醇于 $1000~\mathrm{mL}$ 试剂瓶中，以水稀释至 $1000~\mathrm{mL}$ ，摇匀备用。

# 3.3 标准品

3.3.1生物素标准品 $(\mathrm{C_{10}H_{16}N_2O_3S})$ ：CAS：58-85-5，纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书

的标准物质。

3.3.2 生物素- $\mathrm{D_4}$ （ $\mathrm{C_{10}D_4H_{12}N_2O_3S)}$ ：CAS：1217850-77-5，纯度 $\geq 98\%$

# 3.4 标准溶液配制

3.4.1 生物素标准储备液 $(100\mu \mathrm{g / mL})$ ：根据纯度换算精确称取 $10.00\mathrm{mg}$ 生物素标准品（精确至 $0.01\mathrm{mg}$ )，用乙醇溶液 $(50\%)$ 溶解、定容至 $100~\mathrm{mL}$ 。溶液转移至棕色玻璃瓶中，一 $20^{\circ}C$ 下密封保存，有效期3个月。  
3.4.2 生物素标准中间液 $(10\mu \mathrm{g / mL})$ ：准确吸取 $5.00~\mathrm{mL}$ 生物素标准储备液 $(100\mu \mathrm{g / mL})$ 至 $50~\mathrm{mL}$ 容量瓶，用乙醇溶液 $(50\%)$ 定容至 $50~\mathrm{mL}$ 。溶液转移至棕色玻璃瓶中， $-20^{\circ}C$ 下密封保存，有效期3个月。  
3.4.3 生物素标准中间液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取 $1.00~\mathrm{mL}$ 生物素标准中间液 $(10\mu \mathrm{g} / \mathrm{mL})$ 至 $10~\mathrm{mL}$ 容量瓶，用乙醇溶液 $(50\%)$ 定容至 $10~\mathrm{mL}$ 。溶液转移至棕色玻璃瓶中，一 $20^{\circ}C$ 下密封保存，有效期3个月。  
3.4.4 生物素标准工作液 $(100\mathrm{ng / mL})$ ：准确吸取 $1.00~\mathrm{mL}$ 标准中间液 $(1\mu \mathrm{g} / \mathrm{mL})$ ，用流动相A定容至 $10~\mathrm{mL}$ 。溶液转移至棕色玻璃瓶， $4^{\circ}C$ 下密封保存，有效期1个月。  
3.4.5 生物素标准工作液 $(10\mathrm{ng / mL})$ ：准确吸取 $1.00~\mathrm{mL}$ 生物素标准工作液 $(100\mathrm{ng / mL})$ ，用流动相A定容至 $10~\mathrm{mL}$ 。临用现配。

# 3.5 同位素内标溶液配制

3.5.1 生物素 $-\mathrm{D}_{4}$ 同位素内标储备液 $(100 \mu \mathrm{g} / \mathrm{mL})$ : 准确称取 $10.00 \mathrm{mg}$ (精确至 $0.01 \mathrm{mg}$ ) 生物素内标, 以乙醇溶液 $(50 \%)$ 溶解、定容至 $100 \mathrm{~mL}$ 。溶液转移至棕色玻璃瓶中, $-20^{\circ} \mathrm{C}$ 下密封保存, 有效期 3 个月。  
3.5.2 生物素- $\mathrm{D}_4$ 同位素内标工作液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取 $1.00~\mathrm{mL}$ 标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ ，用流动相A定容于 $100~\mathrm{mL}$ 。临用现配。

# 3.6 标准系列工作液配制

分别准确吸取适量生物素标准工作液于 $10~\mathrm{mL}$ 容量瓶中，准确加入生物素- $\mathrm{D_4}$ 同位素内标工作液 $(1\mu \mathrm{g / mL})150\mu \mathrm{L}$ ，用流动相A定容至刻度，使标准系列生物素的质量浓度分别为 $1.0  \mathrm{ng / mL}$ 、 $2.0\mathrm{ng / mL},5.0\mathrm{ng / mL},10\mathrm{ng / mL},15\mathrm{ng / mL},20\mathrm{ng / mL}$ 和 $25\mathrm{ng / mL}$ ，每个系列溶液中同位素的质量浓度为 $15~\mathrm{ng / mL}$ 。临用现配。

# 4 仪器和设备

4.1 液相色谱-串联质谱仪。  
4.2 天平：感量分别为 $0.001\mathrm{g}$ 和 $0.01\mathrm{mg}$   
4.3 恒温水浴锅： $25^{\circ} \mathrm{C} \sim 100^{\circ} \mathrm{C}$ 。  
4.4 pH计：精度0.01。  
4.5 超声波振荡器。  
4.6 涡旋混合器： $600\mathrm{r / min}\sim 3200\mathrm{r / min}$   
4.7 高速离心机： $\geqslant 10000\mathrm{r / min}$   
4.8 $2\mathrm{mm}$ 孔径试验筛（选用）。  
4.9 恒温培养箱： $25^{\circ}\mathrm{C} \sim 100^{\circ}\mathrm{C}$ 。

# 5 分析步骤

# 5.1 样品制备

# 5.1.1 固体样品

采样量需大于 $0.5\mathrm{kg}$ 。非均匀样品用高速粉碎机将其粉碎至全部通过 $2\mathrm{mm}$ 孔径试验筛，混合均匀后缩分至 $100\mathrm{g}$ ，储存于广口瓶中，密封保存，供检测用。均匀性样品直接混匀，供检测用。

# 5.1.2 半固体样品

采样量需大于 $0.5\mathrm{kg}$ 。需至少采集3个包装(同一批次)，将所有样品在一个容器中匀浆混匀后，其中任意的 $100\mathrm{g}$ 储存于广口瓶中，密封保存，供检测用。

# 5.1.3 液体样品

采样量需大于 $0.5\mathrm{L}$ 。需至少采集3个包装(同一批次)，将所有样品在一个容器中混匀后，其中任意的 $100~\mathrm{mL}$ 储存于广口瓶中，密封保存，供检测用。

# 5.2 样品前处理

# 5.2.1 不含淀粉试样

准确称取样品 $2\mathrm{g}\sim 5\mathrm{g}$ (精确至 $0.001\mathrm{g}$ )，置于 $50~\mathrm{mL}$ 离心管中，加入 $750~\mu \mathrm{L}$ 同位素内标工作液后，加入 $30~\mathrm{mL}$ 温水 $(35^{\circ}C\sim 40^{\circ}C)$ ，振荡混匀，超声提取 $15\mathrm{min}$ 。取出后迅速冷却至室温，用高氯酸调节 $\mathrm{pH}$ 约为1.6。在 $4^{\circ}\mathrm{C}$ 下，以 $8500~\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，经玻璃纤维过滤后用氢氧化钠溶液调节 $\mathrm{pH}$ 为 $4.6\pm 0.1$ ，样液转移至 $50~\mathrm{mL}$ 容量瓶，用水定容至刻度，混匀后移取 $1.5\mathrm{mL}$ 提取液至 $2\mathrm{mL}$ 离心管中，在 $10000~\mathrm{r / min}$ 下离心 $10\mathrm{min}$ ，样品溶液经 $0.22\mu \mathrm{m}$ 水系滤膜过滤，上机分析。

# 5.2.2 含淀粉试样

准确称取样品 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，置于 $50~\mathrm{mL}$ 离心管中，加入 $1 \%$ 样品量的淀粉酶， $750~\mu \mathrm{L}$ 同位素内标工作液后，加入 $30~\mathrm{mL}$ 温水 $(35^{\circ}C\sim 40^{\circ}C)$ ，振荡混匀，放置 $50~^\circ C\sim 60~^\circ C$ 培养箱内约 $30\mathrm{min}$ ，取出后超声提取 $15\mathrm{min}$ 。取出后迅速冷却至室温，用高氯酸调节 $\mathrm{pH}$ 约为1.6。在 $4^{\circ}C$ 下，以 $8500\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，经玻璃纤维过滤后用氢氧化钠溶液调节 $\mathrm{pH}$ 为 $4.6\pm 0.1$ ，样液转移至 $50~\mathrm{mL}$ 容量瓶，用水定容至刻度，混匀后移取 $1.5\mathrm{mL}$ 提取液至 $2\mathrm{mL}$ 离心管中，在 $10000\mathrm{r / min}$ 下离心 $10\mathrm{min}$ ，样品溶液经 $0.22\mu \mathrm{m}$ 水系滤膜过滤，上机分析。

# 5.5 标准曲线的制作

将生物素标准系列工作液浓度由低到高依次注入液相色谱-串联质谱仪中，以生物素浓度与生物素内标浓度的比值为横坐标，以生物素与生物素内标峰面积比为纵坐标，绘制生物素的标准曲线。

# 6 分析结果的表述

试样中生物素的含量按式(1)计算。

$$
X = \frac {\rho \times V}{m} \times \frac {1 0 0}{1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中生物素的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ ；

$\rho$ ——根据标准曲线计算得到的试样中生物素的质量浓度，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$V$ ——试样溶液的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克(g)；

100 ——试样中量以每100克计算的换算系数；

1000 ——试样中 $\mathrm{ng}$ 转换成 $\mu \mathrm{g}$ 的换算系数。

结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

当称样量为 $5.0\mathrm{g}$ 时，定容体积为 $50~\mathrm{mL}$ ，方法的检出限为 $0.300\mu \mathrm{g} / 100\mathrm{g}$ ，定量限为 $1.00\mu \mathrm{g} / 100\mathrm{g}$ 。

# 第二法 微生物法

# 9 原理

生物素是植物乳植杆菌(Lactiplantibacillus plantarum)生长所必需的营养素，在生物素测定培养基中，植物乳植杆菌的生长与生物素的含量呈相关性，根据生物素含量与吸光度的标准曲线计算出试样中生物素含量。

# 10 试剂和材料

除非另有规定，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水或一级水。

# 10.1 菌株

植物乳植杆菌(Lacti plantibacillus plantarum)[原植物乳杆菌(Lactobacillus plantarum)]ATCC 8014,或等效菌株。

# 10.2 培养基

10.2.1 乳酸杆菌琼脂培养基：见附录B中B.1。  
10.2.2 乳酸杆菌肉汤培养基：见附录B中B.2。  
10.2.3 生物素测定用培养基：见附录B中B.3。

注：可使用商品化的合成培养基，按照说明书操作。

# 10.3 试剂

10.3.1 无水乙醇 $\mathrm{(C_2H_5OH)}$ 。  
10.3.2 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ ： $95\% \sim 98\%$   
10.3.3 氢氧化钠(NaOH)。  
10.3.4 氯化钠 $(\mathrm{NaCl})$ 。

# 10.4 试剂配制

10.4.1 乙醇溶液 $(50\%)$ ：量取 $500~\mathrm{mL}$ 无水乙醇，加入水中，定容至 $1000~\mathrm{mL}$ 。  
10.4.2 硫酸溶液 $\mathrm{A}(3.0\mathrm{mol} / \mathrm{L})$ ：量取 $163.2~\mathrm{mL}$ 硫酸，加入水中，定容至 $1000~\mathrm{mL}$ 。  
10.4.3 硫酸溶液 $\mathrm{B}(1.0\mathrm{mol} / \mathrm{L})$ ：量取 $54.4~\mathrm{mL}$ 硫酸，加入水中，定容至 $1000~\mathrm{mL}$ 。  
10.4.4 硫酸溶液 $\mathrm{C}(0.5\mathrm{mol} / \mathrm{L})$ ：量取 $27.2~\mathrm{mL}$ 硫酸，加入水中，定容至 $1000~\mathrm{mL}$ 。  
10.4.5 氢氧化钠溶液 $\mathrm{A}(10\mathrm{mol} / \mathrm{L})$ ：称取 $400\mathrm{g}$ 氢氧化钠，加入水中，溶解定容至 $1000~\mathrm{mL}$ 。  
10.4.6 氢氧化钠溶液 $\mathrm{B}(0.1\mathrm{mol} / \mathrm{L})$ ：吸取 $10~\mathrm{mL}$ 氢氧化钠溶液A( $10\mathrm{mol} / \mathrm{L})$ ，加水定容至 $1000~\mathrm{mL}$ 。  
10.4.7 无菌生理盐水：称取 $8.5\mathrm{g}$ 氯化钠溶于 $1000~\mathrm{mL}$ 蒸馏水中，分装于具塞试管，每管 $10~\mathrm{mL}$ ， $121^{\circ}\mathrm{C}$ 下高压灭菌 $15\mathrm{min}$ 。

注：配制硫酸溶液时，要在通风橱中配制，戴手套注意安全，将浓硫酸缓慢注入水中，并不断搅拌。

# 10.5 标准品

生物素标准品 $\mathrm{(C_{10}H_{16}N_2O_3S)}$ ：CAS号为58-85-5，纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 10.6 标准溶液配制

10.6.1生物素标准储备液 $(100\mu \mathrm{g / mL})$ ：将生物素标准品置于含五氧化二磷的干燥器中，干燥过夜。按纯度称取，使生物素含量为 $50~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )，用乙醇溶液 $(50\%)$ 溶解定容至 $500~\mathrm{mL}$ 。  
10.6.2 生物素标准中间液 $(1\mu \mathrm{g} / \mathrm{mL})$ : 吸取 $1.00 \mathrm{~mL}$ 生物素标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ 至 $100 \mathrm{~mL}$ 容量瓶, 用乙醇溶液 $(50\%)$ 定容。  
10.6.3 生物素标准使用液 $(10\mathrm{ng / mL})$ ：吸取 $1.00~\mathrm{mL}$ 标准中间液 $(1\mu \mathrm{g} / \mathrm{mL})$ 至 $100~\mathrm{mL}$ 容量瓶，用乙醇溶液 $(50\%)$ 定容。  
10.6.4 生物素标准曲线工作液: 分两个浓度, 高浓度溶液的浓度为 $0.2 \mathrm{ng} / \mathrm{mL}$ ; 低浓度溶液的浓度为 $0.1 \mathrm{ng} / \mathrm{mL}$ 。从使用液中 $(10 \mathrm{ng} / \mathrm{mL})$ 吸取 2 次各 $5.00 \mathrm{~mL}$ , 用水分别定容到 $250 \mathrm{~mL}$ 和 $500 \mathrm{~mL}$ 。

注：所有标准溶液要用棕色试剂瓶冷藏于冰箱内 $(2\mathrm{C}\sim 8\mathrm{C})$ ，标准储备液保存期12个月，中间液保存期6个月，标准使用液临用现配。

11.2 离心机： $3000\mathrm{r / min}\sim 5000\mathrm{r / min}$

# 12 分析步骤

# 12.1 测试菌悬液制备

12.1.1 用接种针将植物乳植杆菌菌株穿刺转接至乳酸杆菌琼脂培养基中， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $20\mathrm{h} \sim 24\mathrm{h}$ ，取出后放入冰箱中冷藏保存，可保存1个月。每月至少传种1次，作为储备菌株保存。  
12.1.2 将储备菌株接种至乳酸杆菌琼脂培养基中， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $20\mathrm{h} \sim 24\mathrm{h}$ 以活化菌株，用于接种液的制备。保存数周以上的储备菌种，不能立即用作接种液制备，试验前应连续传种2代～3代以保证菌株活力。  
12.1.3 将 $24\mathrm{h}$ 内活化后的菌株转种至已灭菌的乳酸杆菌肉汤中， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $16\mathrm{h} \sim 20\mathrm{h}$ 。取出后将菌悬液离心弃去上清液，加入 $10\mathrm{mL}$ 生理盐水，用涡旋混合器振荡该悬液，以 $3000\mathrm{r / min} \sim 5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去上清液。重复前述操作清洗2次～3次后，再加 $10\mathrm{mL}$ 生理盐水，充分混合均匀。吸取适量该菌悬液于 $10\mathrm{mL}$ 生理盐水中，混匀制成测试菌悬液。  
12.1.4 以生理盐水做空白，用分光光度计于 $550 \mathrm{~nm}$ 波长下测定测试菌悬液透光率 $T(\%)$ ，调整上述菌液加入量，使测试菌悬液透光率在 $60\% \sim 80\%$ 。

注：块状、颗粒状试样需粉碎；乳粉、米粉等粉状试样需混匀；果蔬、肉、蛋、鱼、动物内脏等需制成食糜；半固体食品等试样需匀浆混匀；液体试样用前振摇混合。

$2\mathrm{g}\sim 3\mathrm{g}$ ；一般营养素补充剂或其他食品 $0.2\mathrm{g}\sim 1\mathrm{g}$ 。

12.2.2 液体饮料或流质、半流质试样：称取试样 $5\mathrm{g}\sim 10\mathrm{g}$ (或用单刻度移液管吸取适量体积)，置于 $250~\mathrm{mL}$ 锥形瓶中。特殊运动饮料称取样品后不需处理，称样后直接定容至 $100\mathrm{mL}(V)$ ，按12.2.4进行稀释。  
12.2.3 根据试样基质加入 $50~\mathrm{mL}$ 硫酸溶液：特殊膳食用食品、调制乳粉等强化食品加入硫酸溶液 $\mathrm{C}(0.5\mathrm{mol} / \mathrm{L})$ ；植物源食品加入硫酸溶液 $\mathrm{B}(1.0\mathrm{mol} / \mathrm{L})$ ；动物源食品加入硫酸溶液 $\mathrm{A}(3.0\mathrm{mol} / \mathrm{L})$ 。

对每个编号的试样提取液的培养管，用每支试管的吸光度计算每毫升该编号提取液生物素的浓度，并计算该编号提取液的生物素浓度平均值，每支试管测得的该浓度不得超过该平均值的 $\pm 15\%$ ，超过者要舍去。如果符合该要求的管数少于所有的4个编号的提取液的总管数的2/3，用于计算试样含量的数据是不充分的，需要重新检验。如果符合要求的管数不少于原来管数的2/3，重新计算每一编号的有效试样管中每毫升提取液中生物素含量的平均值，以此平均值计算全部编号试样管的总平均值为 $\rho$ ，按式(2)根据稀释倍数和称样量计算出试样中生物素的含量。

注：可使用与本标准检测原理相同且经等效验证的生物素检测试剂盒。

# 13 分析结果的表述

试样提取液生物素的质量浓度按式(2)计算。

$$
\rho_ {x} = \frac {c _ {x}}{V _ {x}} \tag {2}
$$

式中：

$\rho_{x}$ ——试样提取液生物素的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{x}$ ——从标准曲线上查得试样系列管中生物素的质量浓度, 单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ;

$V_{x}$ ——制备试样系列管时吸取的试样提取液体积，单位为毫升 $(\mathrm{mL})$ 。

试样中生物素的含量按式（3）计算。

$$
X = \frac {\rho \times V}{m} \times \frac {V _ {1}}{V _ {2}} \times f \times \frac {1 0 0}{1 0 0 0} \tag {3}
$$

式中：

$X$ ——试样中生物素的含量，单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ 或微克每百毫升 $(\mu \mathrm{g} / 100\mathrm{mL})$

$\rho$ ——计算所得的试样提取液中生物素质量浓度的总平均值，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$V$ ——试样提取液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量或体积，单位为克(g)或毫升(mL)；

$V_{1}$ ——过滤前的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——过滤后吸取滤液的体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样提取液稀释倍数；

100 ——换算系数；

1000 ——换算系数。

结果保留3位有效数字。

注1：对于不需要提取的样品，式中去除 $V_{1}$ 和 $V_{2}$

注2：如果样品中生物素含量较高，可适当进一步稀释；如果含量较低可适当提高样品称样量或降低稀释倍数。

<table><tr><td>胨化乳</td><td>15.0 g</td></tr><tr><td>酵母浸膏</td><td>5.0 g</td></tr><tr><td>葡萄糖</td><td>10.0 g</td></tr><tr><td>番茄汁</td><td>100 mL</td></tr><tr><td>磷酸二氢钾</td><td>2.0 g</td></tr><tr><td>聚山梨糖单油酸酯</td><td>1.0 g</td></tr><tr><td>蒸馏水</td><td>1 000 mL</td></tr></table>

# B.2.2 制法

将各成分加入蒸馏水中，加热溶解，搅拌的同时保持煮沸状态 $2\mathrm{min}\sim 3\mathrm{min}$ ，调节 $\mathrm{pH}$ 为 $6.8\pm 0.2$ （ $20^{\circ}\mathrm{C}\sim 25^{\circ}\mathrm{C}$ ），混合均匀后分装试管，每管 $10~\mathrm{mL}$ 。 $121^{\circ}\mathrm{C}$ 下高压灭菌 $15\mathrm{min}$ 。

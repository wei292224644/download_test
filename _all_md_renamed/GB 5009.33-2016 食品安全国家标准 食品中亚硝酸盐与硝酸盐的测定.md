# 前言

本标准代替GB5009.33—2010《食品安全国家标准 食品中亚硝酸盐与硝酸盐的测定》、NY/T1375—2007《植物产品中亚硝酸盐与硝酸盐的测定 离子色谱法》、NY/T1279—2007《蔬菜、水果中硝酸盐的测定 紫外分光光度法》、SN/T3151—2012《出口食品中亚硝酸盐和硝酸盐的测定 离子色谱法》。

本标准与GB5009.33—2010相比，主要变化如下：

——合并原第二法、第三法为第二法；

——增加了蔬菜、水果中硝酸盐的测定的紫外分光光度法。

# 食品中亚硝酸盐与硝酸盐的测定

# 1 范围

本标准规定了食品中亚硝酸盐和硝酸盐的测定方法。

本标准适用于食品中亚硝酸盐和硝酸盐的测定。

# 第一法 离子色谱法

# 2 原理

试样经沉淀蛋白质、除去脂肪后，采用相应的方法提取和净化，以氢氧化钾溶液为淋洗液，阴离子交换柱分离，电导检测器或紫外检测器检测。以保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙酸 $\mathrm{(CH_3COOH)}$ 。

3.1.2 氢氧化钾(KOH)。

# 3.2 试剂配制

3.2.1 乙酸溶液 $(3\%)$ ：量取乙酸 $3\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，以水稀释至刻度，混匀。

3.2.2 氢氧化钾溶液 $(1\mathrm{mol} / \mathrm{L})$ : 称取 $6\mathrm{g}$ 氢氧化钾, 加入新煮沸过的冷水溶解, 并稀释至 $100\mathrm{mL}$ , 混匀。

# 3.3 标准品

3.3.1 亚硝酸钠 $\mathrm{(NaNO_2}$ ,CAS号：7632-00-0)：基准试剂，或采用具有标准物质证书的亚硝酸盐标准溶液。  
3.3.2 硝酸钠 $\mathrm{(NaNO_3}$ ,CAS号：7631-99-4)：基准试剂，或采用具有标准物质证书的硝酸盐标准溶液。

# 3.4 标准溶液的制备

3.4.1 亚硝酸盐标准储备液 $(100\mathrm{mg / L}$ ，以 $\mathrm{NO}_2^-$ 计，下同)：准确称取 $0.1500\mathrm{g}$ 于 $110^{\circ}\mathrm{C}\sim 120^{\circ}\mathrm{C}$ 干燥至恒重的亚硝酸钠，用水溶解并转移至 $1000~\mathrm{mL}$ 容量瓶中，加水稀释至刻度，混匀。  
3.4.2 硝酸盐标准储备液 $(1000\mathrm{mg / L}$ ，以 $\mathrm{NO}_3^-$ 计，下同）：准确称取 $1.3710\mathrm{g}$ 于 $110^{\circ}\mathrm{C}\sim 120^{\circ}\mathrm{C}$ 干燥至恒重的硝酸钠，用水溶解并转移至 $1000~\mathrm{mL}$ 容量瓶中，加水稀释至刻度，混匀。  
3.4.3 亚硝酸盐和硝酸盐混合标准中间液：准确移取亚硝酸根离子 $\left(\mathrm{NO}_{2}^{-}\right)$ 和硝酸根离子 $\left(\mathrm{NO}_{3}^{-}\right)$ 的标准储备液各 $1.0\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，此溶液每升含亚硝酸根离子 $1.0\mathrm{mg}$ 和硝酸

根离子 $10.0\mathrm{mg}$ 。

3.4.4 亚硝酸盐和硝酸盐混合标准使用液：移取亚硝酸盐和硝酸盐混合标准中间液，加水逐级稀释，制成系列混合标准使用液，亚硝酸根离子浓度分别为 $0.02\mathrm{mg / L},0.04\mathrm{mg / L},0.06\mathrm{mg / L},0.08\mathrm{mg / L},$ $0.10\mathrm{mg / L},0.15\mathrm{mg / L},0.20\mathrm{mg / L}$ ；硝酸根离子浓度分别为 $0.2\mathrm{mg / L},0.4\mathrm{mg / L},0.6\mathrm{mg / L},0.8\mathrm{mg / L},$ $1.0\mathrm{mg / L},1.5\mathrm{mg / L},2.0\mathrm{mg / L}$ 。

# 4 仪器和设备

4.1 离子色谱仪：配电导检测器及抑制器或紫外检测器，高容量阴离子交换柱， $50~\mu \mathrm{L}$ 定量环。  
4.2 食物粉碎机。  
4.3 超声波清洗器。  
4.4 分析天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.5 离心机：转速 $\geqslant 10000\mathrm{r / min}$ ，配 $50~\mathrm{mL}$ 离心管。   
4.6 $0.22\mu \mathrm{m}$ 水性滤膜针头滤器。  
4.7 净化柱：包括 $\mathrm{C_{18}}$ 柱、Ag柱和Na柱或等效柱。  
4.8 注射器： $1.0\mathrm{mL}$ 和 $2.5\mathrm{mL}$

注：所有玻璃器皿使用前均需依次用 $2\mathrm{mol} / \mathrm{L}$ 氢氧化钾和水分别浸泡 $4\mathrm{h}$ ，然后用水冲洗3次～5次，晾干备用。

# 5 分析步骤

# 5.1 试样预处理

5.1.1 蔬菜、水果：将新鲜蔬菜、水果试样用自来水洗净后，用水冲洗，晾干后，取可食部切碎混匀。将切碎的样品用四分法取适量，用食物粉碎机制成匀浆，备用。如需加水应记录加水量。  
5.1.2 粮食及其他植物样品：除去可见杂质后，取有代表性试样 $50\mathrm{g}\sim 100\mathrm{g}$ ，粉碎后，过 $0.30\mathrm{mm}$ 孔筛，混匀，备用。  
5.1.3 肉类、蛋、水产及其制品：用四分法取适量或取全部，用食物粉碎机制成匀浆，备用。  
5.1.4 乳粉、豆奶粉、婴儿配方粉等固态乳制品（不包括干酪）：将试样装入能够容纳2倍试样体积的带盖容器中，通过反复摇晃和颠倒容器使样品充分混匀直到使试样均一化。  
5.1.5 发酵乳、乳、炼乳及其他液体乳制品：通过搅拌或反复摇晃和颠倒容器使试样充分混匀。  
5.1.6 干酪：取适量的样品研磨成均匀的泥浆状。为避免水分损失，研磨过程中应避免产生过多的热量。

# 5.2 提取

5.2.1 蔬菜、水果等植物性试样：称取试样 $5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，可适当调整试样的取样量，以下相同），置于 $150~\mathrm{mL}$ 具塞锥形瓶中，加入 $80~\mathrm{mL}$ 水， $1\mathrm{mL}1\mathrm{mol} / \mathrm{L}$ 氢氧化钾溶液，超声提取 $30\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振摇1次，保持固相完全分散。于 $75^{\circ}\mathrm{C}$ 水浴中放置 $5\mathrm{min}$ ，取出放置至室温，定量转移至 $100~\mathrm{mL}$ 容量瓶中，加水稀释至刻度，混匀。溶液经滤纸过滤后，取部分溶液于 $10000\mathrm{r / min}$ 离心 $15\mathrm{min}$ ，上清液备用。  
5.2.2肉类、蛋类、鱼类、及其制品等：称取试样匀浆 $5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，置于 $150~\mathrm{mL}$ 具塞锥形瓶中，加入 $80~\mathrm{mL}$ 水，超声提取 $30\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振摇1次，保持固相完全分散。于 $75^{\circ}C$ 水浴中放置 $5\mathrm{min}$ ，取出放置至室温，定量转移至 $100~\mathrm{mL}$ 容量瓶中，加水稀释至刻度，混匀。溶液经滤纸过滤后，取部分溶液于 $10000\mathrm{r / min}$ 离心 $15\mathrm{min}$ ，上清液备用。

5.2.3 腌鱼类、腌肉类及其他腌制品：称取试样匀浆 $2\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ），置于 $150~\mathrm{mL}$ 具塞锥形瓶中，加入 $80~\mathrm{mL}$ 水，超声提取 $30\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振摇1次，保持固相完全分散。于 $75^{\circ}\mathrm{C}$ 水浴中放置 $5\mathrm{min}$ ，取出放置至室温，定量转移至 $100~\mathrm{mL}$ 容量瓶中，加水稀释至刻度，混匀。溶液经滤纸过滤后，取部分溶液于 $10000\mathrm{r / min}$ 离心 $15\mathrm{min}$ ，上清液备用。  
5.2.4 乳：称取试样 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），置于 $100\mathrm{mL}$ 具塞锥形瓶中，加水 $80\mathrm{mL}$ ，摇匀，超声 $30\mathrm{min}$ ，加入 $3\%$ 乙酸溶液 $2\mathrm{mL}$ ，于 $4^{\circ}\mathrm{C}$ 放置 $20\mathrm{min}$ ，取出放置至室温，加水稀释至刻度。溶液经滤纸过滤，滤液备用。  
5.2.5 乳粉及干酪：称取试样 $2.5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，置于 $100~\mathrm{mL}$ 具塞锥形瓶中，加水 $80~\mathrm{mL}$ ，摇匀，超声 $30\mathrm{min}$ ，取出放置至室温，定量转移至 $100~\mathrm{mL}$ 容量瓶中，加入 $3\%$ 乙酸溶液 $2\mathrm{mL}$ ，加水稀释至刻度，混匀。于 $4^{\circ}\mathrm{C}$ 放置 $20\mathrm{min}$ ，取出放置至室温，溶液经滤纸过滤，滤液备用。  
5.2.6 取上述备用溶液约 $15 \mathrm{~mL}$ , 通过 $0.22 \mu \mathrm{m}$ 水性滤膜针头滤器、 $\mathrm{C}_{18}$ 柱, 弃去前面 $3 \mathrm{~mL}$ (如果氯离子大于 $100 \mathrm{mg} / \mathrm{L}$ , 则需要依次通过针头滤器、 $\mathrm{C}_{18}$ 柱、 $\mathrm{Ag}$ 柱和 $\mathrm{Na}$ 柱, 弃去前面 $7 \mathrm{~mL}$ ), 收集后面洗脱液待测。

固相萃取柱使用前需进行活化， $\mathrm{C_{18}}$ 柱 $(1.0~\mathrm{mL})$ 、Ag柱 $(1.0~\mathrm{mL})$ 和Na柱 $(1.0~\mathrm{mL})$ ，其活化过程为： $\mathrm{C_{18}}$ 柱 $(1.0~\mathrm{mL})$ 使用前依次用 $10~\mathrm{mL}$ 甲醇、 $15~\mathrm{mL}$ 水通过，静置活化 $30\mathrm{min}$ 。Ag柱 $(1.0~\mathrm{mL})$ 和Na柱 $(1.0~\mathrm{mL})$ 用 $10~\mathrm{mL}$ 水通过，静置活化 $30\mathrm{min}$ 。

# 5.4.1 标准曲线的制作

将标准系列工作液分别注入离子色谱仪中，得到各浓度标准工作液色谱图，测定相应的峰高 $(\mu \mathrm{S})$ 或峰面积，以标准工作液的浓度为横坐标，以峰高 $(\mu \mathrm{S})$ 或峰面积为纵坐标，绘制标准曲线（亚硝酸盐和硝酸盐标准色谱图见图A.1）。

# 5.4.2 试样溶液的测定

将空白和试样溶液注入离子色谱仪中，得到空白和试样溶液的峰高 $(\mu \mathrm{S})$ 或峰面积，根据标准曲线得到待测液中亚硝酸根离子或硝酸根离子的浓度。

# 6 分析结果的表述

试样中亚硝酸离子或硝酸根离子的含量按式(1)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times f \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中亚硝酸根离子或硝酸根离子的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——测定用试样溶液中的亚硝酸根离子或硝酸根离子浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——试剂空白液中亚硝酸根离子或硝酸根离子的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样溶液稀释倍数；

1000——换算系数；

$m$ ——试样取样量，单位为克 $(\mathrm{g})$ 。

试样中测得的亚硝酸根离子含量乘以换算系数1.5，即得亚硝酸盐(按亚硝酸钠计)含量；试样中测得的硝酸根离子含量乘以换算系数1.37，即得硝酸盐(按硝酸钠计)含量。

结果保留2位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 分光光度法

# 9 原理

亚硝酸盐采用盐酸萘乙二胺法测定，硝酸盐采用镉柱还原法测定。

试样经沉淀蛋白质、除去脂肪后，在弱酸条件下，亚硝酸盐与对氨基苯磺酸重氮化后，再与盐酸萘乙二胺偶合形成紫红色染料，外标法测得亚硝酸盐含量。采用镉柱将硝酸盐还原成亚硝酸盐，测得亚硝酸盐总量，由测得的亚硝酸盐总量减去试样中亚硝酸盐含量，即得试样中硝酸盐含量。

11.6.1 海绵状镉的制备：镉粒直径 $0.3\mathrm{mm}\sim 0.8\mathrm{mm}$

将适量的锌棒放入烧杯中，用 $40\mathrm{g / L}$ 硫酸镉溶液浸没锌棒。在 $24\mathrm{h}$ 之内，不断将锌棒上的海绵状镉轻轻刮下。取出残余锌棒，使镉沉底，倾去上层溶液。用水冲洗海绵状镉2次～3次后，将镉转移至搅拌器中，加 $400~\mathrm{mL}$ 盐酸 $(0.1\mathrm{mol} / \mathrm{L})$ ，搅拌数秒，以得到所需粒径的镉颗粒。将制得的海绵状镉倒回烧杯中，静置 $3\mathrm{h}\sim 4\mathrm{h}$ ，期间搅拌数次，以除去气泡。倾去海绵状镉中的溶液，并可按下述方法进行镉粒镀铜。

# 11.6.2 镉粒镀铜：

# 13.1 亚硝酸盐含量计算

亚硝酸盐（以亚硝酸钠计）的含量按式(3)计算：

$$
X _ {1} = \frac {m _ {2} \times 1 0 0 0}{m _ {3} \times \frac {V _ {1}}{V _ {0}} \times 1 0 0 0} \tag {3}
$$

式中：

$X_{1}$ ——试样中亚硝酸钠的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$m_{2}$ ——测定用样液中亚硝酸钠的质量，单位为微克 $(\mu \mathrm{g})$ ；

1000——转换系数；

$m_{3}$ ——试样质量，单位为克 $(\mathrm{g})$

$V_{1}$ ——测定用样液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——试样处理液总体积，单位为毫升 $(\mathrm{mL})$ 。

结果保留2位有效数字。

# 13.2 硝酸盐含量的计算

硝酸盐（以硝酸钠计）的含量按式(4)计算：

$$
X _ {2} = \left(\frac {m _ {4} \times 1 0 0 0}{m _ {5} \times \frac {V _ {3}}{V _ {2}} \times \frac {V _ {5}}{V _ {4}} \times 1 0 0 0} - X _ {1}\right) \times 1. 2 3 2 \dots \tag {4}
$$

式中：

$X_{2}$ ——试样中硝酸钠的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$m_{4}$ ——经镉粉还原后测得总亚硝酸钠的质量，单位为微克 $(\mu \mathrm{g})$ ；

1000——转换系数；

$m_{5}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$V_{3}$ ——测总亚硝酸钠的测定用样液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试样处理液总体积，单位为毫升 $(\mathrm{mL})$

$V_{5}$ ——经镉柱还原后样液的测定用体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{4}$ ——经镉柱还原后样液总体积，单位为毫升 $(\mathrm{mL})$ ；

$X_{1}$ ——由式(3)计算出的试样中亚硝酸钠的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

1.232——亚硝酸钠换算成硝酸钠的系数。

结果保留2位有效数字。

# 16 原理

用 $\mathrm{pH}9.6\sim 9.7$ 的氨缓冲液提取样品中硝酸根离子，同时加活性炭去除色素类，加沉淀剂去除蛋白质及其他干扰物质，利用硝酸根离子和亚硝酸根离子在紫外区 $219~\mathrm{nm}$ 处具有等吸收波长的特性，测定提取液的吸光度，其测得结果为硝酸盐和亚硝酸盐吸光度的总和，鉴于新鲜蔬菜、水果中亚硝酸盐含量甚微，可忽略不计。测定结果为硝酸盐的吸光度，可从工作曲线上查得相应的质量浓度，计算样品中硝酸盐的含量。

# 19 分析步骤

# 19.1 试样制备

选取一定数量有代表性的样品，先用自来水冲洗，再用水清洗干净，晾干表面水分，用四分法取样，切碎，充分混匀，于组织捣碎机中匀浆(部分少汁样品可按一定质量比例加入等量水)，在匀浆中加1滴正辛醇消除泡沫。

# 19.2 提取

称取 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )匀浆试样(如制备过程中加水，应按加水量折算)于 $250~\mathrm{mL}$ 锥形瓶中，加水 $100~\mathrm{mL}$ ，加入 $5\mathrm{mL}$ 氨缓冲溶液 $(\mathrm{pH} = 9.6\sim 9.7)$ ， $2\mathrm{g}$ 粉末状活性炭。振荡(往复速度为200次/min) $30\mathrm{min}$ 。定量转移至 $250~\mathrm{mL}$ 容量瓶中，加入 $2\mathrm{mL}150~\mathrm{g / L}$ 亚铁氰化钾溶液和 $2\mathrm{mL}300~\mathrm{g / L}$ 硫酸锌溶液，充分混匀，加水定容至刻度，摇匀，放置 $5\mathrm{min}$ ，上清液用定量滤纸过滤，滤液备用。同时做空白实验。

# 19.3 测定

根据试样中硝酸盐含量的高低，吸取上述滤液 $2\mathrm{mL}\sim 10\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，加水定容至刻度，混匀。用 $1\mathrm{cm}$ 石英比色皿，于 $219~\mathrm{nm}$ 处测定吸光度。

# 19.4 标准曲线的制作

将标准曲线工作液用 $1\mathrm{cm}$ 石英比色皿，于 $219~\mathrm{nm}$ 处测定吸光度。以标准溶液质量浓度为横坐标，吸光度为纵坐标绘制工作曲线。

# 20 结果计算

硝酸盐（以硝酸根计）的含量按式(5)计算：

$$
X = \frac {\rho \times V _ {6} \times V _ {8}}{m _ {6} \times V _ {7}} \tag {5}
$$

式中：

$X$ ——试样中硝酸盐的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由工作曲线获得的试样溶液中硝酸盐的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V_{6}$ ——提取液定容体积，单位为毫升 $(\mathrm{mL})$

$V_{8}$ ——待测液定容体积，单位为毫升 $(\mathrm{mL})$

$m_{6}$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$V_{7}$ ——吸取的滤液体积，单位为毫升 $(\mathrm{mL})$ 。

结果保留2位有效数字。

# 21 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 22 其他

第三法中硝酸盐检出限为 $1.2\mathrm{mg / kg}$

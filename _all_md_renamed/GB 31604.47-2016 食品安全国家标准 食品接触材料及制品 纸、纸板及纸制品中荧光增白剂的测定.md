# 食品安全国家标准

# 食品接触材料及制品 纸、纸板及纸制品中荧光增白剂的测定

# 1 范围

本标准规定了食品用纸、纸板及纸制品中荧光增白剂的测定方法。

本标准适用于食品用纸、纸板及纸制品中荧光增白剂的测定。

# 2 原理

由于荧光增白剂在吸收近紫外光(波长范围在 $300\mathrm{nm}\sim 400\mathrm{nm}$ 之间)后，分子中的电子会从基态跃迁，然后在极短时间内又回到基态，同时发射出蓝色或紫色荧光（波长范围在 $420\mathrm{nm}\sim 480\mathrm{nm}$ 之间）。因此，在波长 $365~\mathrm{nm}$ 紫外灯照射下，通过观察试样是否有明显荧光现象来定性测定试样中是否含有荧光增白剂。如果试样出现多处不连续小斑点状荧光或试样有荧光现象但不明显时，可用碱性提取液提取，然后将提取液调节为酸性，再用纱布吸附提取液中的荧光增白剂，在波长 $365~\mathrm{nm}$ 紫外灯下，观察纱布是否有明显荧光现象，来确证试样中是否含有荧光增白剂。

# 3 试剂和材料

除非另有规定，所用试剂均为分析纯，水为GB/T6682规定的一级水。所用试剂和材料在紫外灯下应无荧光现象。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 三乙胺 $\left[\left(\mathrm{CH}_{3} \mathrm{CH}_{2}\right)_{3} \mathrm{~N}\right]$ 。  
3.1.3 氢氧化钠(NaOH)：优级纯。  
3.1.4 盐酸。

# 3.2 试剂配制

3.2.1 盐酸溶液 $(10\%)$ ，体积分数）：用量筒或移液管按 $9:1$ 的体积比分别量取水和盐酸，放入合适的容器中，混匀即可。  
3.2.2 乙腈溶液（ $40\%$ ，体积分数）：用量筒按 $40:60$ 的体积比分别量取乙腈和水，放入合适的容器中，混匀即可。  
3.2.3 碱性提取液：用量筒或移液管按 $40:60:1$ 的体积比分别量取乙腈、水和三乙胺，放入合适的容器中，混匀即可。

# 3.3 标准品

荧光增白剂220标准品 $\mathrm{(C_{40}H_{40}N_{12}O_{16}S_4Na_4}$ ，简称C.I.220，CAS号：16470-24-9)，纯度大于 $95\%$ 。

# 3.4 标准溶液配制

3.4.1 标准储备液 $(1.00\mathrm{mg / mL})$ ：于避光条件下，准确称取C.I.220标准品约 $10~\mathrm{mg}$ （精确到 $0.1\mathrm{mg}$ ）于烧杯中，用碱性提取液(3.2.3)溶解，转移至 $10~\mathrm{mL}$ 棕色容量瓶中，并定容至刻度，在一 $18^{\circ}C$ 以下于黑暗处保存，有效期为 $90~\mathrm{d}$ 。  
3.4.2 标准工作液 $(40.0\mu \mathrm{g / mL})$ ：将标准储备液用乙腈溶液 $(40\%)$ ，体积分数)逐级稀释成 $40.0~\mu \mathrm{g / mL}$ 的标准工作液，于 $4^{\circ}C$ 左右避光保存，有效期为 $15\mathrm{d}$ 。

# 3.5 材料

3.5.1纱布：尺寸约 $5\mathrm{cm}\times 5\mathrm{cm}$   
3.5.1 玻璃棉。

# 4 仪器和设备

注：所有直接接触试样的仪器与设备在紫外灯下应无荧光现象。

4.1 紫外灯：波长 $365~\mathrm{nm}$   
4.2 剪刀。  
4.3 直角三角板。  
4.4 超声波清洗器。  
4.5 高速粉碎机，转速 $\geqslant 10000\mathrm{r / min}$   
4.6 旋转蒸发仪。  
4.7 恒温水浴锅。  
4.8 pH计：精度为0.1。  
4.9 分析天平：感量分别为 $1\mathrm{mg}$ 和 $0.1\mathrm{mg}$   
4.10 鸡心瓶： $250~\mathrm{mL}$   
4.11 具塞锥形瓶： $250~\mathrm{mL}$   
4.12 玻璃漏斗。  
4.13 玻璃表面皿。  
4.14 托盘。  
4.15 离心机：转速 $\geq 3500\mathrm{r / min}$

# 5 分析步骤

# 5.1 试样制备

对于食品用纸或纸板，如食品包装纸、糖果纸、冰棍纸等，从试样中随机取5张，用剪刀和直角三角板裁剪成 $100\mathrm{cm}^2$ 大小。

对于食品用纸制品，如纸杯、纸碗、纸桶、纸盒、纸碟、纸盘、纸袋等，从试样中随机取2个同批次的产品，用剪刀和直角三角板将待测纸层裁剪成 $100\mathrm{cm}^2$ 。

对于需要确证实验的试样，称取 $10\mathrm{g}$ （精确到 $1\mathrm{mg}$ )，剪成约 $5\mathrm{mm}\times 5\mathrm{mm}$ 的纸屑，再用高速粉碎机(转速在 $10000\mathrm{r / min}$ )粉碎至棉絮状，备用。如不能立即检测，应用干净的聚乙烯塑料袋盛放，在室温下避光保存。

# 5.2 荧光增白剂的直接测定

于暗室或暗箱内，打开紫外灯的电源开关，检测波长选择 $365\mathrm{nm}$ 。将制作好的 $100~\mathrm{cm}^2$ 试样置于紫外灯光源下约 $20~\mathrm{cm}$ 处，观察试样是否有明显的蓝色或紫色荧光。如试样出现多处不连续小斑点状荧光，或试样有荧光现象但不明显时，则需继续执行5.3操作步骤，进行确证实验。

# 5.3 荧光增白剂的确证

# 5.3.1 标准对照纱布的制备

称取5.1操作步骤中粉碎均匀的空白纸样 $2.0\mathrm{g}$ （精确至 $1\mathrm{mg}$ ）于 $250~\mathrm{mL}$ 锥形瓶中，加入 $40.0\mu \mathrm{g / mL}$ C.I.220标准溶液 $0.5\mathrm{mL}$ ，相当于纸样中C.I.220含量为 $10\mathrm{mg / kg}$ ，于避光状态下(要求照度小于20Lux)加入 $100~\mathrm{mL}$ 碱性提取液(乙腈、水和三乙胺的体积比为 $40:60:1$ )，于 $50^{\circ}C$ 下超声提取 $40\mathrm{min}$ 。提取结束后冷却至室温，将提取液通过装有少许玻璃棉(要求不含荧光物质)的玻璃漏斗过滤到鸡心瓶中，或者采用离心的方式(3500r/min的转速离心 $5\mathrm{min}$ )获得澄清的提取液。将提取液在 $50^{\circ}C$ 下减压浓缩至约 $40~\mathrm{mL}\sim 50~\mathrm{mL}$ ，将浓缩液转移至 $250~\mathrm{mL}$ 烧杯中，用水洗涤鸡心瓶后，洗液也一并转入 $250~\mathrm{mL}$ 烧杯中，用盐酸溶液 $(10\%)$ ，体积分数)调 $\mathrm{pH}$ 为 $3\sim 5$ ，并加水定容至约 $100~\mathrm{mL}$ 。然后将一块规格为 $5\mathrm{cm}\times 5\mathrm{cm}$ 的纱布浸没于提取液中，在 $40^{\circ}C$ 水浴吸附 $30\mathrm{min}$ 。用镊子取出纱布后，用手挤去大部分液体后，将纱布叠成四层，每层面积约为 $2.5\mathrm{cm}\times 2.5\mathrm{cm}$ ，放于玻璃表面皿中。

# 5.3.2 试样提取与吸附

称取5.1操作步骤中粉碎均匀的试样 $2.0\mathrm{g}$ （精确至 $1.0~\mathrm{mg}$ ）于 $250~\mathrm{mL}$ 锥形瓶中，其余操作按照5.3.1步骤中“于避光状态下……，放于玻璃表面皿中”进行。每个试样进行两次平行试验。

# 5.3.3 空白试验

以 $2.0\mathrm{g}$ 水代替试样，其余操作按照5.3.2操作步骤进行。

# 5.3.4 荧光增白剂的测定

于暗室或暗箱内，打开紫外灯的电源开关，检测波长选择 $365~\mathrm{nm}$ 。将放置标准对照纱布、试样纱布及空白试验纱布的表面皿一起置于紫外灯光源下约 $20~\mathrm{cm}$ 处，观察试样纱布是否比空白试验纱布有明显的蓝色或紫色荧光。

# 6 结果判定

荧光增白剂的直接测定实验：对于食品用纸或纸板，如果5张中任何一张的荧光面积大于 $5\mathrm{cm}^2$ ，则判定该试样中荧光增白剂为阳性，否则判定该试样中荧光增白剂为阴性；对于食品用纸制品，2个同批次的产品中任何一个的荧光面积大于 $5\mathrm{cm}^2$ ，则判定该试样中荧光增白剂为阳性，否则判定该试样中荧光增白剂为阴性。

荧光增白剂确证实验：如果试样的两个平行试验均无明显荧光现象，则判定该试样中荧光增白剂为阴性，如两个平行试验均有明显荧光现象，则判定该试样中荧光增白剂为阳性，如只有一个试样纱布有明显荧光现象，需要重新进行两个平行试验，如重新试验后两个平行试验均无明显荧光现象，则判定该试样中荧光增白剂为阴性，否则判定该试样中荧光增白剂为阳性。

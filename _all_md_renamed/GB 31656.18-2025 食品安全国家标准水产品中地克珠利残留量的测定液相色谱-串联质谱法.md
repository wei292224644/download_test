# 液相色谱-串联质谱法

# 1 范围

本文件规定了水产品中地克珠利残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鱼、虾、蟹及鳖等水产品可食组织中地克珠利残留量的测定。其他水产品的检测可参照执行。

# 3 术语和定义

本文件中没有需要界定的术语和定义

# 4 原理

试样中残留的地克珠利, 用乙腈提取, 无水硫酸钠脱水, 正己烷去脂, $\mathrm{C}_{18}$ 吸附剂净化, 液相色谱-串联质谱仪测定, 内标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6582规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $(\mathrm{CH}_3\mathrm{CN})_2$ 色谱纯

5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。

5.1.4 醋酸铵 $\mathrm{(CH_3COONH_3)}$ ：色谱纯。

5.1.5 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：经 $650^{\circ}C$ 灼烧 $4\mathrm{h}$ ，置于干燥器内备用。

5.1.6 $N,N$ -二甲基甲酰胺 $\mathrm{(C_3H_7NO)}$ ：色谱纯。

# 5.2 溶液配制

5.2.1 甲醇水溶液：取甲醇 $80~\mathrm{mL}$ 和水 $20~\mathrm{mL}$ ，混匀。

5.2.2 $5\mathrm{mmol / L}$ 醋酸铵溶液：称取醋酸铵 $0.38\mathrm{g}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。

# 5.3 标准品

5.3.1 地克珠利(Diclazuril, $\mathrm{C_{17}H_{9}Cl_3N_4O_2}$ ,CAS号：101831-37-2)，含量 $\geq 98\%$ ，或相当者。

5.3.2 地克珠利- $\mathrm{d}_4$ (Diclazuril- $\mathrm{d}_4$ ， $\mathrm{C}_{17}\mathrm{H}_{5}\mathrm{D}_{4}\mathrm{Cl}_{3}\mathrm{N}_{4}\mathrm{O}_{2}$ ,CAS号：1632495-80-7)，含量 $\geq 98\%$ ，或相当者。

# 5.4 标准溶液制备

5.4.1 标准储备液 $(1.0\mathrm{mg / mL})$ ：取地克珠利标准品约 $10\mathrm{mg}$ ，精密称定，先用 $2\mathrm{mL}N,N-$ 二甲基甲酰胺溶解，再用甲醇稀释并定容至 $10~\mathrm{mL}$ ，配制成 $1.0\mathrm{mg / mL}$ 标准储备液，一 $18^{\circ}C$ 避光保存，有效期

6个月。

5.4.2 标准中间液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：准确移取适量 $1.0\mathrm{mg / mL}$ 地克珠利标准储备液，用甲醇稀释，配制成 $10~\mu \mathrm{g / mL}$ 标准中间液，一 $18^{\circ}C$ 避光保存，有效期6个月。  
5.4.3 标准工作液 $(1\mu \mathrm{g / mL})$ ：准确移取适量 $10~\mu \mathrm{g / mL}$ 地克珠利标准中间液，用甲醇稀释，配制成 $1\mu \mathrm{g / mL}$ 标准工作液，一 $18^{\circ}C$ 避光保存，有效期3个月。  
5.4.4 内标储备液 $(0.1\mathrm{mg / mL})$ ：取地克珠利- $\mathbf{d}_4$ 内标标准品，精密称定，先用 $1\mathrm{mL}N,N-$ 二甲基甲酰胺溶解，再用甲醇稀释并定容至 $10~\mathrm{mL}$ ，配制成 $0.1\mathrm{mg / mL}$ 内标储备液，一 $18^{\circ}C$ 避光保存，有效期6个月。  
5.4.5 内标中间液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：准确移取适量 $0.1\mathrm{mg / mL}$ 地克珠利内标储备液，用甲醇稀释，配制成 $10~\mu \mathrm{g / mL}$ 内标中间液，一 $18^{\circ}C$ 避光保存，有效期6个月。  
5.4.6 内标工作液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：准确移取适量 $10~\mu \mathrm{g / mL}$ 地克珠利内标中间液，用甲醇稀释，配制成 $1\mu \mathrm{g / mL}$ 内标工作液，一 $18^{\circ}C$ 避光保存，有效期3个月。

5.5 材料

5.5.1 $\mathrm{C}_{18}$ 吸附剂：层析用， $40\mu \mathrm{m}\sim 63\mu \mathrm{m}$   
5.5.2 聚丙烯离心管： $50~\mathrm{mL}$   
5.5.3 梨形瓶： $50~\mathrm{mL}$   
5.5.4 有机相微孔滤膜: $0.22 \mu \mathrm{m}$ 。

6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$ 。  
6.3 离心机：转速不低于 $6000\mathrm{r / min}$ 。  
6.4 微量高速离心机：14000r/min。  
6.5 涡旋混合器。  
6.6 旋转蒸发仪。  
6.7 超声波清洗仪。

7 试样的制备与保存

7.1 试样的制备

按GB/T30891—2014附录B的要求制样。

a）取均质后的供试样品，作为供试试样；  
b) 取均质后的空白样品, 作为空白试样;  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

8 测定步骤

8.1 提取

称取试料 $(2 \pm 0.05) \mathrm{~g}$ 于 $50 \mathrm{~mL}$ 具塞离心管中, 准确加入 $1 \mu \mathrm{g} / \mathrm{mL}$ 内标标准工作液 $50 \mu \mathrm{L}$ , 涡旋混合 $30 \mathrm{~s}$ , 避光放置 $10 \mathrm{~min}$ 。加入乙腈 $10 \mathrm{~mL}$ , 涡旋混合 $1 \mathrm{~min}$ , 再加入无水硫酸钠 $2 \mathrm{~g}$ , 涡旋混合 $1 \mathrm{~min}$ , 超声提取 $10 \mathrm{~min}, 6000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 将上清液转移至 $50 \mathrm{~mL}$ 梨形瓶中。取乙腈 $10 \mathrm{~mL}$ 加入 $50 \mathrm{~mL}$ 离心管中重复提取 1 次, 合并提取液, $40^{\circ} \mathrm{C}$ 减压浓缩至干。准确加入甲醇水溶液 $1.0 \mathrm{~mL}$ 定容, 涡旋混匀, 充分溶解残余物, 待净化。

# 8.2 净化

在上述待净化液中，先加入正己烷 $2\mathrm{mL}$ ，充分涡旋混合 $1\mathrm{min},6000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去正己烷层，再加入 $\mathrm{C_{18}}$ 吸附剂 $50~\mathrm{mg}$ ，充分涡旋混合 $1\mathrm{min},10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液过 $0.22\mu \mathrm{m}$ 有机相微孔滤膜后，供液相色谱-串联质谱仪测定。

# 8.3 标准曲线的制备

准确移取适量 $1\mu \mathrm{g / mL}$ 地克珠利标准工作液，用甲醇水溶液逐级稀释，使外标溶液浓度分别为 $1\mathrm{ng / mL},2\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL},50\mathrm{ng / mL}$ ，内标标准溶液的浓度为 $50~\mathrm{ng / mL}$ ，供液相色谱-串联质谱仪测定。

# 8.5 测定法

# 8.5.1 定性测定

在相同测试条件下，试样溶液中地克珠利与其内标地克珠利- $\mathrm{d}_4$ 的保留时间之比与标准溶液中地克珠利与其内标地克珠利- $\mathrm{d}_4$ 的保留时间之比偏差在 $1\%$ 以内；且检测到的相对离子丰度，应与浓度相当的校正标准溶液相对离子丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.5.2 定量测定

取试样溶液和相应的标准工作液，作单点或多点校准，按内标法以峰面积定量，标准工作液及试样溶液中的目标化合物响应值均应在仪器检测的线性范围之内。对于试料中地克珠利残留量超过仪器测定线性范围的，在提取时根据药物浓度相应增加内标工作液的添加量，使试样溶液稀释后地克珠利的响应在仪器线性范围内，对应地克珠利- $\mathrm{d}_4$ 浓度与标准曲线制备中的地克珠利- $\mathrm{d}_4$ 浓度一致。在上述色谱-质谱条件下，地克珠利和地克珠利- $\mathrm{d}_4$ 标准溶液特征离子质量色谱图见附录A。

# 8.6 空白试验

取空白试样，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中地克珠利的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A _ {i} \times A _ {i s} ^ {\prime} \times C _ {s} \times C _ {i s} \times V}{A _ {i s} \times A _ {s} \times C _ {i s} ^ {\prime} \times m} \times f \tag {1}
$$

式中：

$X$ ——试样中地克珠利残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A_{i}$ ——试样溶液中地克珠利的峰面积；

$A_{i_3}^{\prime}$ ——标准溶液中地克珠利- $\mathrm{d}_4$ 的峰面积；

$C_{s}$ ——标准溶液中地克珠利浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$C_{i}$ ——试样溶液中地克珠利- $\mathrm{d}_4$ 浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{is}$ ——试样溶液中地克珠利- $\mathrm{d}_4$ 的峰面积；

$A_{s}$ ——标准溶液中地克珠利的峰面积；

$C_{is}^{\prime}$ ——标准溶液中地克珠利- $\mathrm{d}_4$ 浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$

$m$ 供试试样质量的数值，单位为克 $(\mathbf{g})$

$f$ ——稀释倍数。

注：计算结果以平行测定结果的算术平均值表示，保留3位有效数字。

# 10 方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法的检出限：鱼、虾和鳖为 $1.0\mu \mathrm{g / kg}$ ，蟹为 $2.5\mu \mathrm{g / kg}$ ；本方法的定量限：鱼、虾和鳖为 $2.0\mu \mathrm{g / kg}$ 蟹为 $5.0\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法在 $2\mu \mathrm{g} / \mathrm{kg} < X \leqslant 10\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$ ；在 $10\mu \mathrm{g} / \mathrm{kg} < X \leqslant 100\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$ ；在 $100\mu \mathrm{g} / \mathrm{kg} < X \leqslant 1000\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $80\% \sim 110\%$ 。

# 10.3 精密度

本方法当 $2\mu \mathrm{g / kg} < X\leqslant 10~\mu \mathrm{g / kg}$ 时，重复性相对标准偏差 $\leqslant 20\%$ ，再现性相对标准偏差 $\leqslant 30\%$ ；当 $10~\mu \mathrm{g / kg} < X\leqslant 100~\mu \mathrm{g / kg}$ 时，重复性相对标准偏差 $\leqslant 15\%$ ，再现性相对标准偏差 $\leqslant 20\%$ ；当 $100~\mu \mathrm{g / kg} < X\leqslant 1000~\mu \mathrm{g / kg}$ 时，重复性相对标准偏差 $\leqslant 10\%$ ，再现性相对标准偏差 $\leqslant 15\%$ 。

# 食品安全国家标准水产品中二硫氰基甲烷残留量的测定 气相色谱法

# 1 范围

本文件规定了水产品中二硫氰基甲烷残留量测定的制样和气相色谱测定方法。

本文件适用于鱼、虾、蟹、鳖等水产品可食组织中二硫氰基甲烷残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

GB/T30891—2014 水产品抽样规范

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的二硫氧基甲烷，用氯甲烷和正已烷混合液提取，去脂，中性氧化铝固相萃取柱净化，气相色谱测定，外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}_2$ 色谱纯。

5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。

5.1.4 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：色谱纯。

5.1.5 丙酮 $\mathrm{(CH_3COCH_3)}$ ：色谱纯。

5.1.6 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。

# 5.2 溶液配制

二氯甲烷-正己烷 $(1:1,V / V)$ ：分别取二氯甲烷和正已烷 $250~\mathrm{mL}$ ，混匀。现用现配。

# 5.3 标准品

二硫氰基甲烷(Methylenebisthiocyanate, $\mathrm{C_3H_2N_2S_2}$ ,CAS号：6317-18-6)，含量 $\geqslant 99\%$ 。

# 5.4 标准溶液制备

5.4.1 标准储备液：准确称取二硫氰基甲烷标准品 $10\mathrm{mg}$ ，用乙腈适量使溶解并稀释定容至 $50~\mathrm{mL}$ 棕色容量瓶，配制成浓度为 $200~\mu \mathrm{g / mL}$ 的标准储备液。 $-18^{\circ}C$ 以下保存，有效期3个月。  
5.4.2 标准工作液：精确量取二硫氰基甲烷标准储备液适量，用二氯甲烷稀释，配制成浓度为 $0.05\mu \mathrm{g / mL}, 0.1\mu \mathrm{g / mL}, 0.5\mu \mathrm{g / mL}, 1.0\mu \mathrm{g / mL}, 2.0\mu \mathrm{g / mL}, 5.0\mu \mathrm{g / mL}$ 和 $10.0\mu \mathrm{g / mL}$ 的系列标准工作液。现用现配。

# 5.5 材料

中性氧化铝固相萃取柱，填料 $60\mathrm{mg / 3mL}$ ，或相当者。

# 6 仪器和设备

6.1 气相色谱仪，配脉冲式火焰光度(PFPD)检测器。  
6.2 均质机。  
6.3 旋转蒸发仪。  
6.4 离心机。  
6.5 涡旋混合器。  
6.6 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.7 氮吹仪。

# 7 试样的制备与保存

# 7.1 试样的制备

按GB/T30891—2014附录B的要求制样。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 8 测定步骤

# 8.1 提取

取试料 $5\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 具塞玻璃离心管中，加二氯甲烷-正己烷 $(1:1,V / V)$ $15~\mathrm{mL}$ ，立即涡旋 $1\mathrm{min}$ ，超声提取 $10\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，用玻璃胶头滴管取上清液于鸡心瓶中，重复提取1次，合并2次上清液，混匀， $40^{\circ}C$ 氮气吹至 $0.5\mathrm{mL}\sim 1.0\mathrm{mL}$ ，备用。

# 8.2 净化和浓缩

取中性氧化铝固相萃取柱，上层添加无水硫酸钠 $1.0\mathrm{g}$ ，将备用液过柱并收集，用二氯甲烷-正己烷 $(1:1,V / V)$ $2.0\mathrm{mL}$ 洗涤8.1盛装提取液的鸡心瓶，过柱，并收集在 $15\mathrm{mL}$ 玻璃试管中，再用二氯甲烷 $8\mathrm{mL}$ 洗脱小柱，且收集在同一玻璃试管中， $40^{\circ}\mathrm{C}$ 氮气吹干，加二氯甲烷-正己烷 $(1:1,V / V)1.0\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ 使其完全溶解，供气相色谱仪测定。

# 8.3 标准曲线的制备

精密量取系列标准工作液适量，供气相色谱测定。以测得峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.4 测定

# 8.4.1 色谱条件

a）色谱柱：HP-5MS石英毛细管柱 $(30\mathrm{m}\times 0.32\mathrm{mm}$ ，粒径 $0.25\mu \mathrm{m})$ ，或相当者；  
b）进样方式：不分流进样；  
c）载气：氮气，纯度 $99.999\%$   
d） 柱流速： $1.0~\mathrm{mL / min}$   
e）进样口温度 $230^{\circ}\mathrm{C}$ ，检测器温度 $310^{\circ}\mathrm{C}$   
f）柱温程序：初始柱温 $50^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ ，以 $25^{\circ}\mathrm{C / min}$ 升温至 $180^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$   
g）进样量： $1.0\mu \mathrm{L}$

# 8.4.2 测定法

取试料溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试料溶液中二硫氰基甲烷响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液气相色谱图见附录A。

# 8.4.3 空白试验

取空白试料，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {s}} \times A \times V}{A _ {\mathrm {s}} \times m} \tag {1}
$$

式中：

$X$ ——试样中二硫氰基甲烷残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{s}}$ ——标准溶液中二硫氰基甲烷浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$A$ ——试样溶液中二硫氰基甲烷的色谱峰面积；

$A_{s}$ ——标准溶液中二硫氰基甲烷的色谱峰面积；

$V$ ——试样最终定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——供试试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $5\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $10~\mu \mathrm{g / kg}\sim 500~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 110\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

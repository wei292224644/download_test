# 前言

本标准代替GB5413.24—2010《食品安全国家标准婴幼儿食品和乳品中氯的测定》、GB/T12457—2008《食品中氯化钠的测定》、GB/T15667—1995《水果、蔬菜及其制品氯化物含量的测定》、GB/T9695.8—2008《肉与肉制品氯化物含量的测定》、GB/T22427.12—2008《淀粉及其衍生物氯化物测定》，以及GB/T5009.44—2003《肉与肉制品卫生标准的分析方法》中“14.2食盐”的测定。

本标准将对以上标准进行整合，主要修改如下：

——标准名称修改为“食品安全国家标准 食品中氯化物的测定”；  
——根据氯化物测定原理整合成三个方法：电位滴定法、佛尔哈德法（间接沉淀滴定法）、银量法（摩尔法或直接滴定法）；  
——增加超声处理步骤。

# 食品安全国家标准

# 食品中氯化物的测定

# 1 范围

本标准规定了食品中氯化物含量的电位滴定法、佛尔哈德法(间接沉淀滴定法)、银量法(摩尔法或直接滴定法)测定方法。

本标准的电位滴定法适用于各类食品中氯化物的测定。

本标准的佛尔哈德法(间接沉淀滴定法)和银量法(摩尔法或直接滴定法)不适用于深颜色食品中氯化物的测定。

# 第一法 电位滴定法

# 2 原理

试样经酸化处理后，加入丙酮，以玻璃电极为参比电极，银电极为指示电极，用硝酸银标准滴定溶液滴定试液中的氯化物。根据电位的“突跃”，确定滴定终点。以硝酸银标准滴定溶液的消耗量，计算食品中氯化物的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 亚铁氰化钾 $\left[\mathrm{K}_4\mathrm{Fe}(\mathrm{CN})_6\cdot 3\mathrm{H}_2\mathrm{O}\right]$   
3.1.2 乙酸锌 $\left[\mathrm{Zn}\left(\mathrm{CH}_{3} \mathrm{CO}_{2}\right)_{2}\right]$ 。  
3.1.3 硝酸银 $(\mathrm{AgNO_3})$ 。  
3.1.4 冰乙酸 $\mathrm{(CH_3COOH)}$   
3.1.5 硝酸 $\mathrm{(HNO_3)}$ 。  
3.1.6 丙酮 $\mathrm{(CH_3COCH_3)}$ 。

# 3.2 标准品

基准氯化钠（NaCl），纯度 $\geq 99.8\%$

# 3.3 试剂配制

3.3.1 沉淀剂I：称取 $106\mathrm{g}$ 亚铁氰化钾，加水溶解并定容到 $1\mathrm{L}$ ，混匀。  
3.3.2 沉淀剂Ⅱ：称取 $220\mathrm{g}$ 乙酸锌，溶于少量水中，加入 $30~\mathrm{mL}$ 冰乙酸，加水定容到1L，混匀。  
3.3.3 硝酸溶液 $(1 + 3)$ ：将1体积的硝酸加入到3体积水中，混匀。

# 3.4 标准溶液配制及标定

3.4.1 氯化钠基准溶液 $(0.01000\mathrm{mol} / \mathrm{L})$ : 称取 $0.5844\mathrm{g}$ (精确至 $0.1\mathrm{mg}$ ) 经 $500^{\circ}\mathrm{C} \sim 600^{\circ}\mathrm{C}$ 灼烧至恒重的基准试剂氯化钠, 于小烧杯中, 用少量水溶解, 转移到 $1000\mathrm{mL}$ 容量瓶中, 稀释至刻度, 摇匀。  
3.4.2 硝酸银标准滴定溶液 $(0.02\mathrm{mol} / \mathrm{L})$ ：称取 $3.40\mathrm{g}$ 硝酸银(精确至 $0.01\mathrm{g}$ )于小烧杯中，用少量硝酸溶解，转移到 $1000~\mathrm{mL}$ 棕色容量瓶中，用水定容至刻度，摇匀，避光贮存，或转移到棕色瓶中。或购买经国家认证并授予标准物质证书的硝酸银标准滴定溶液。  
3.4.3 标定(二级微商法)：吸取 $10.00\mathrm{mL}0.01000\mathrm{mol / L}$ 氯化钠基准溶液于 $50~\mathrm{mL}$ 烧杯中，加入 $0.2\mathrm{mL}$ 硝酸溶液及 $25~\mathrm{mL}$ 丙酮。将玻璃电极和银电极浸入溶液中，启动电磁搅拌器。从酸式滴定管滴入 $V^{\prime}\mathrm{mL}$ 硝酸银标准滴定溶液（所需量的 $90\%$ )，测量溶液的电位值 $(E)$ 。继续滴入硝酸银标准滴定溶液，每滴入 $1\mathrm{mL}$ 立即测量溶液电位值 $(E)$ 。接近终点和终点后，每滴入 $0.1\mathrm{mL}$ ，测量溶液的电位值 $(E)$ 。继续滴入硝酸银标准滴定溶液，直至溶液电位数值不再明显改变。记录每次滴入硝酸银标准滴定溶液的体积和电位值。  
3.4.4 滴定终点的确定：根据滴定记录(3.4.3)，以硝酸银标准滴定溶液的体积 $(V^{\prime})$ 和电位值 $(E)$ ，按表A.1示例，以列表方式计算 $\Delta E, \Delta V$ 、一级微商和二级微商。或电位滴定仪自动滴定、记录硝酸银标准滴定溶液的体积和电位值。

当一级微商最大、二级微商等于零时，即为滴定终点，按式(1)计算滴定到终点时硝酸银标准滴定溶液的体积 $(V_{1})$ ：

$$
V _ {1} = V _ {a} + \left(\frac {a}{a - b} \times \Delta V\right) \tag {1}
$$

式中：

$V_{1}$ ——滴定到终点时消耗硝酸银标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{a}$ ——在 $a$ 时消耗硝酸银标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$a$ ——二级微商为零前的二级微商值；

$b$ ——二级微商为零后的二级微商值；

$\Delta V$ —— $a$ 与 $b$ 之间的体积差，单位为毫升 $(\mathrm{mL})$ 。

# 3.4.5 硝酸银标准滴定溶液的浓度按式(2)计算：

$$
c = \frac {1 0 \times c _ {1}}{V _ {1}} \tag {2}
$$

式中：

$c$ ——硝酸银标准滴定溶液浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$c_{1}$ ——氯化钠基准溶液浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{1}$ ——滴定终点时消耗硝酸银标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 粉末状、糊状或液体样品

取有代表性的样品至少 $200\mathrm{g}$ ，充分混匀，置于密闭的玻璃容器内。

# 5.1.2 块状或颗粒状等固体样品

取有代表性的样品至少 $200\mathrm{g}$ ，用粉碎机粉碎或用研钵研细，置于密闭的玻璃容器内。

# 5.1.3 半固体或半液体样品

取有代表性的样品至少 $200\mathrm{g}$ ，用组织捣碎机捣碎，置于密闭的玻璃容器内。

# 5.2 试样溶液制备

# 5.2.1 婴幼儿食品、乳品

称取混合均匀的试样 $10\mathrm{g}$ （精确至 $1\mathrm{mg}$ )于 $100~\mathrm{mL}$ 具塞比色管中，加入 $50~\mathrm{mL}$ 约 $70^{\circ}C$ 热水，振荡分散样品，水浴中沸腾 $15\mathrm{min}$ ，并不时摇动，取出，超声处理 $20\mathrm{min}$ ，冷却至室温，依次加入 $2\mathrm{mL}$ 沉淀剂I和 $2\mathrm{mL}$ 沉淀剂Ⅱ，每次加后摇匀。用水稀释至刻度，摇匀，在室温静置 $30\mathrm{min}$ 。用滤纸过滤，弃去最初滤液，取部分滤液测定。必要时也可用离心机于 $5000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取部分上清液测定。

# 5.2.2 蛋白质、淀粉含量较高的蔬菜制品、淀粉制品

称取约 $5\mathrm{g}$ 试样(精确至 $1\mathrm{mg}$ ) 于 $100\mathrm{mL}$ 具塞比色管中，加适量水分散，振摇 $5\mathrm{min}$ (或用涡旋振荡器振荡 $5\mathrm{min}$ )，超声处理 $20\mathrm{min}$ ，依次加入 $2\mathrm{mL}$ 沉淀剂 I 和 $2\mathrm{mL}$ 沉淀剂 II，每次加后摇匀。用水稀释至刻度，摇匀，在室温静置 $30\mathrm{min}$ 。用滤纸过滤，弃去最初滤液，取部分滤液测定。

# 5.2.3 一般蔬菜制品、腌制品

称取约 $10\mathrm{g}$ 试样（精确至 $1\mathrm{mg}$ )于 $100~\mathrm{mL}$ 具塞比色管中，加入 $50~\mathrm{mL}70^{\circ}\mathrm{C}$ 热水，振摇 $5\mathrm{min}$ （或用涡旋振荡器振荡 $5\mathrm{min}$ )，超声处理 $20\mathrm{min}$ ，冷却至室温，用水稀释至刻度，摇匀，用滤纸过滤，弃去最初滤液，取部分滤液测定。

# 5.2.4 调味品

称取约 $5\mathrm{g}$ 试样(精确至 $1\mathrm{mg}$ ) 于 $100\mathrm{mL}$ 具塞比色管中，加入 $50\mathrm{mL}$ 水，必要时， $70^{\circ}\mathrm{C}$ 热水浴中加热溶解 $10\mathrm{min}$ ，振摇分散，超声处理 $20\mathrm{min}$ ，冷却至室温，用水稀释至刻度，摇匀，用滤纸过滤，弃去最初滤液，取部分滤液测定。

# 5.2.5 肉禽及水产制品

称取约 $10\mathrm{g}$ 试样(精确至 $1\mathrm{mg}$ ) 于 $100\mathrm{mL}$ 具塞比色管中，加入 $50\mathrm{mL}70^{\circ}\mathrm{C}$ 热水，振荡分散样品，水浴中煮沸 $15\mathrm{min}$ ，并不断摇动，取出，超声处理 $20\mathrm{min}$ ，冷却至室温，依次加入 $2\mathrm{mL}$ 沉淀剂I和 $2\mathrm{mL}$ 沉淀剂Ⅱ。每次加入沉淀剂充分摇匀，用水稀释至刻度，摇匀，在室温静置 $30\mathrm{min}$ 。用滤纸过滤，弃去最初滤液，取部分滤液测定。

# 5.2.6 鲜(冻)肉类、灌肠类、酱卤肉类、肴肉类、烧烤肉和火腿类

炭化浸出法：称取 $5\mathrm{g}$ 试样（精确至 $1\mathrm{mg}$ ）于瓷坩埚中，小火炭化完全，炭化成分用玻璃棒轻轻研碎，然后加 $25\mathrm{mL} \sim 30\mathrm{mL}$ 水，小火煮沸，冷却，过滤于 $100\mathrm{mL}$ 容量瓶中，并用热水少量多次洗涤残渣及滤器，洗液并入容量瓶中，冷至室温，加水至刻度，取部分滤液测定。

灰化浸出法：称取 $5\mathrm{g}$ 试样（精确至 $1\mathrm{mg}$ ）于瓷坩埚中，先小火炭化，再移入高温炉中，于 $500^{\circ}\mathrm{C} \sim$ $550^{\circ}\mathrm{C}$ 灰化，冷却，取出，残渣用 $50~\mathrm{mL}$ 热水分数次浸渍溶解，每次浸渍后过滤于 $100~\mathrm{mL}$ 容量瓶中，冷至室温，加水至刻度，取部分滤液测定。

# 5.3 测定

移取 $10.00\mathrm{mL}$ 试液 $(5.2)(V_{2})$ ，于 $50~\mathrm{mL}$ 烧杯中，加入 $5\mathrm{mL}$ 硝酸溶液和 $25~\mathrm{mL}$ 丙酮。将玻璃电极和银电极浸入溶液中，启动电磁搅拌器。从酸式滴定管滴入 $V^{\prime}\mathrm{mL}$ 硝酸银标准滴定溶液（所需量的 $90\%$ )，测量溶液的电位值 $(E)$ 。继续滴入硝酸银标准滴定溶液，每滴入 $1\mathrm{mL}$ 立即测量溶液电位值 $(E)$ 。接近终点和终点后，每滴入 $0.1\mathrm{mL}$ ，测量溶液的电位值 $(E)$ 。继续滴入硝酸银标准滴定溶液，直至溶液电位数值不再明显改变。记录每次滴入硝酸银标准滴定溶液的体积和电位值。以硝酸银标准滴定溶液的体积 $(V^{\prime})$ 和电位值 $(E)$ ，用列表方式计算 $\Delta E$ 、 $\Delta V$ 、一级微商和二级微商。按式(1)计算滴定终点时消耗硝酸银标准滴定溶液的体积 $(V_{3})$ 。或电位滴定仪自动滴定、记录硝酸银标准滴定溶液的体积和电位值。同时做空白试验，记录消耗硝酸银标准滴定溶液的体积 $(V_{0}^{\prime})$ 。

# 6 分析结果表述

食品中氯化物的含量按式(3)计算：

$$
X _ {1} = \frac {0 . 0 3 5 5 \times c \times \left(V _ {3} - V _ {0} ^ {\prime}\right) \times V}{m \times V _ {2}} \times 1 0 0 \quad \dots \tag {3}
$$

式中：

$X_{1}$ ——试样中氯化物的含量（以 $\mathrm{Cl}^{-}$ 计），\%；

0.0355——与 $1.00\mathrm{mL}$ 硝酸银标准滴定溶液 $[c(\mathrm{AgNO}_3) = 1.000\mathrm{mol / L}]$ 相当的氯的质量，单位为克 $(\mathrm{g})$

$c$ ——硝酸银标准滴定溶液浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{0}^{\prime}$ ——空白试验时消耗的硝酸银标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——用于滴定的滤液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——滴定试液时消耗的硝酸银标准滴定溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$V$ —样品定容体积，单位为毫升 $(\mathrm{mL})$

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

当氯化物含量 $\geq 1\%$ 时，结果保留三位有效数字；当氯化物含量 $< 1\%$ 时，结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测试结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 8 其他

以称样量 $10\mathrm{g}$ ，定容至 $100\mathrm{mL}$ 计算，方法定量限(LOQ)为 $0.008\%$ （以 $\mathrm{Cl}^-$ 计）。

# 第二法 佛尔哈德法（间接沉淀滴定法）

# 9 原理

样品经水或热水溶解、沉淀蛋白质、酸化处理后，加入过量的硝酸银溶液，以硫酸铁铵为指示剂，用硫氰酸钾标准滴定溶液滴定过量的硝酸银。根据硫氰酸钾标准滴定溶液的消耗量，计算食品中氯化物的含量。

# 10 试剂和材料

除非另有规定，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

10.1.3 硝酸 $(\mathrm{HNO}_3)$ 。

10.1.4 硝酸银 $(\mathrm{AgNO}_3)$ 。

10.1.5 乙醇 $\mathrm{(CH_3CH_2OH)}$ ：纯度 $\geqslant 95\%$

# 10.2 标准品

基准氯化钠（NaCl），纯度 $\geqslant 99.8\%$

# 10.3 试剂配制

10.4.1 硝酸银标准滴定溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $17\mathrm{g}$ 硝酸银，溶于少量硝酸中，转移到 $1000\mathrm{mL}$ 棕色容量瓶中，用水稀释至刻度，摇匀，转移到棕色试剂瓶中储存。或购买有证书的硝酸银标准滴定溶液。  
10.4.2 硫氰酸钾标准滴定溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $9.7\mathrm{g}$ 硫氰酸钾，溶于水中，转移到 $1000~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。或购买经国家认证并授予标准物质证书的硫氰酸钾标准滴定溶液。

# 前言

本标准代替SN/T1954—2007《进出口冰鲜肉中二氧化氯残留量的检验方法分光光度法》。

本标准与SN/T1954—2007相比，主要变化如下：

——修改了检测方法及检测方法的适用范围；  
——增加试样分类；  
——修改试样处理条件；  
——修改检出限，增加定量限。

# 食品安全国家标准

# 食品中二氧化氯的测定

# 1 范围

本标准规定了蔬菜、水果、畜禽肉、水产品中二氧化氯的分光光度检测方法。

本标准适用于蔬菜、水果、畜禽肉、水产品中二氧化氯的测定。

# 2 原理

试料中的二氧化氯用磷酸盐缓冲溶液提取，经冷冻离心，纤维滤纸过滤，以甘氨酸作掩蔽剂，消除溶液中 $\mathrm{Cl}_2$ 、 $\mathrm{ClO}^{-}$ 等物质的假阳性干扰，加入 $N,N$ -二乙基-对苯二胺(DPD)显色剂与二氧化氯显色，采用分光光度法在 $552~\mathrm{nm}$ 处测定其最大吸光度，从而确定食品中二氧化氯的含量。

# 3 试剂和材料

除非另有说明，所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的二级水。

# 3.1 试剂

3.1.1 甘氨酸 $(\mathrm{C}_2\mathrm{H}_5\mathrm{NO}_2)$ 。  
3.1.2 乙二胺四乙酸二钠(EDTA) $(\mathrm{C_{10}H_{14}N_2Na_2O_8})$ 。  
3.1.3 碘化钾(KI)：纯度 $99.5\%$   
3.1.4 亚氯酸钠 $\mathrm{(NaClO_2)}$   
3.1.5 乙酸 $\mathrm{(CH_3COOH)}$   
3.1.6 硫酸 $\mathrm{(H_2SO_4)}$   
3.1.7 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.8 氢氧化钠 $(\mathrm{NaOH})$ 。  
3.1.9 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
3.1.10 $N,N$ -二乙基-对苯二胺（DPD）。  
3.1.11 硫代硫酸钠 $\mathrm{(Na_2S_2O_3)}$ 。  
3.1.12 可溶性淀粉。  
3.1.13 重铬酸钾 $(\mathrm{Cr_2K_2O_7})$ 。  
3.1.14 高纯氮气。

# 3.2 试剂配制

3.2.1 DPD 显色剂 $(1 \mathrm{~g} / \mathrm{L})$ : 称取 $1 \mathrm{~g}$ DPD(3.1.10) 溶于含 $8 \mathrm{~mL}$ 硫酸溶液 $(1 + 3)$ 和 $200 \mathrm{mg}$ EDTA 的无氯二级水 (3.2.2) 中, 并用无氯二级水稀释至 $1000 \mathrm{~mL}$ , 储于具玻塞的棕色玻璃瓶中, 置于暗处。  
3.2.2 无氧化性氯二级水：二级水中加入亚硫酸钠(3.1.4)，将氧化性氯还原为氯离子（以DPD检查不显色），再进行蒸馏，所得水为无氧化性氯二级水。

3.2.3 甘氨酸溶液 $(10\%)$ ：取 $10\mathrm{g}$ 甘氨酸(3.1.1)，用水溶解，定容至 $100~\mathrm{mL}$ 。  
3.2.4 氢氧化钠溶液 $(2\mathrm{mol} / \mathrm{L})$ ：称取 $80\mathrm{g}$ 氢氧化钠，用水溶解，定容至 $1\mathrm{L}$ 。  
3.2.5 氢氧化钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $4\mathrm{g}$ 氢氧化钠，用水溶解，定容至 $1\mathrm{L}$ 。  
3.2.6 硫酸溶液 $(1 \mathrm{~mol} / \mathrm{L})$ : 量取 $98.4 \%$ 的硫酸 (3.1.6) $54 \mathrm{~mL}$ , 缓慢地加入约 $300 \mathrm{~mL}$ 蒸馏水中, 冷却以后, 转移到容量瓶中, 把容器用约 $20 \mathrm{~mL}$ 蒸馏水洗涤 3 次, 也转移到容量瓶中, 用水定容至 $1 \mathrm{~L}$ 。  
3.2.7 磷酸二氢钾溶液 $[c(\mathrm{KH}_2\mathrm{PO}_4) = 0.1\mathrm{mol / L}]$ : $13.61\mathrm{g}$ 磷酸二氢钾溶液(3.1.9)，用水溶解，定容至 $1\mathrm{L}$ 。  
3.2.8 磷酸盐缓冲液 $(\mathrm{pH} = 6.5)$ ：取 $500~\mathrm{mL}~0.1~\mathrm{mol / L~KH_2PO_4}$ 和 $81~\mathrm{mL}~0.1~\mathrm{mol / L~NaOH}$ ，用水稀释至 $1\mathrm{L}$ 。  
3.2.9 亚氯酸钠饱和溶液：取适量亚氯酸钠(3.1.4)于烧杯内，加少量水，搅拌使成为饱和溶液（亚氯酸钠的溶解度相当高，按所需用量配制）。  
3.2.10 硫代硫酸钠标准溶液的配制 $\left[c\left(\mathrm{Na}_{2}\mathrm{S}_{2}\mathrm{O}_{3}\right)=0.1000 \mathrm{~mol} / \mathrm{L}\right]$ ：称取 $16 \mathrm{~g}$ 无水硫酸钠(3.1.7)，溶于 $1 \mathrm{~L}$ 水中，加热煮沸 $10 \mathrm{~min}$ ，冷却，避光两周后过滤备用。  
3.2.11 硫代硫酸钠标准溶液的标定 $\left[c\left(\mathrm{Na}_{2}\mathrm{S}_{2}\mathrm{O}_{3}\right)=0.1000 \mathrm{~mol} / \mathrm{L}\right]$ ：称取 $0.15 \mathrm{~g}$ 于 $120^{\circ} \mathrm{C}$ 烘至恒重的基准重铬酸钾(3.1.13)，准确至 $0.0001 \mathrm{~g}$ ，置于碘量瓶中，加入 $25 \mathrm{~mL}$ 水，溶解后加入 $2 \mathrm{~g}$ 碘化钾及 $20 \mathrm{~mL} 20 \%$ 硫酸溶液，摇匀，于暗处放置 $10 \mathrm{~min}$ 。加 $150 \mathrm{~mL}$ 水，用配制好的硫代硫酸钠标准溶液（3.2.10）滴定，近终点时加入 $3 \mathrm{~mL}$ 淀粉指示液 $(5 \mathrm{~g} / \mathrm{L})$ （3.1.12），继续滴定至溶液由蓝色变为亮绿色。同时做空白试验。  
3.2.12 淀粉指示剂 $(5\mathrm{g} / \mathrm{L})$ ：称取 $5.0\mathrm{g}$ 淀粉放入 $50~\mathrm{mL}$ 烧杯，量取 $1\mathrm{L}$ 蒸馏水，先用数滴把淀粉调成糊状，再取约 $900~\mathrm{mL}$ 水加热至微沸时，倒入糊状淀粉，再用剩余蒸馏水冲洗 $50~\mathrm{mL}$ 烧杯3次，洗液倒入烧杯，然后再加入1滴 $10\%$ 盐酸，微沸 $3\mathrm{min}$ 。  
3.2.13 淀粉溶液 $(1\%)$ : 称取 $1 \mathrm{~g}$ 可溶性淀粉, 用少许水将其调成糊状, 用 $100 \mathrm{~mL}$ 沸水将其溶解, 再加热煮沸至澄清。

# 3.3 标准品

# 3.3.1 二氧化氯标准储备溶液的制备

在A瓶中放入 $300~\mathrm{mL}$ 纯水，将A瓶一端玻璃与空气压缩机相接，另一玻璃管与B瓶相连。B瓶为高强度硼硅玻璃瓶，瓶口有三根玻璃管；第一根插至离瓶底 $5\mathrm{mm}$ 处，用以引进空气；第二根上接带刻度的圆柱形分液漏斗，下端伸至液面下；第三根下端离开液面，上端与C瓶相接。溶解 $10\mathrm{g}$ 亚氯酸钠于 $750~\mathrm{mL}$ 纯水中并倒入B瓶中，在分液漏斗中装有 $20~\mathrm{mL}$ 硫酸溶液。C瓶装有亚氯酸钠饱和溶液的洗气塔。D瓶为 $2\mathrm{L}$ 硼硅玻璃收集瓶，瓶中装有 $1500~\mathrm{mL}$ 纯水，用以吸收所发生的二氧化氯，余气由排气管排出。整套装置应放入通风橱内。二氧化氯的发生及吸收装置见图1。

启动空气压缩机，使空气均匀地通过整个装置。每隔 $5\mathrm{min}$ 由分液漏斗加入 $5\mathrm{mL}$ 硫酸溶液，加完最后一次硫酸溶液后，空气流量要持续 $30\mathrm{min}$ 。

所获得的黄色二氧化氯储备溶液放入棕色瓶中密塞于冷藏箱中保存。其质量浓度约为 $250\mathrm{mg / L}\sim$ $600~\mathrm{mg / L}$ ，相当于 $500~\mathrm{mg / L}\sim 1200~\mathrm{mg / L}$ 有效氯（ $\mathrm{Cl}_2$ ）。

# 3.3.2 二氧化氯标准储备溶液的标定

向 $250~\mathrm{mL}$ 碘量瓶内加入 $100~\mathrm{mL}$ 纯水、 $1\mathrm{g}$ 碘化钾及 $5\mathrm{mL}$ 乙酸，摇动碘量瓶使碘化钾完全溶解。加入 $10.00~\mathrm{mL}$ 二氧化氯标准溶液，在暗处放置 $5\mathrm{min}$ 。用 $0.1\mathrm{mol} / \mathrm{L}$ 硫代硫酸钠标准溶液(3.2.10)滴定至溶液呈淡黄色时，加入 $1\mathrm{mL}$ 淀粉溶液，继续滴定至终点。按以上步骤加入相同量的试剂（仅不加二氧化氯)，用硫代硫酸钠标准溶液滴定至蓝色消失，记录空白用量。

二氧化氯标准储备溶液的浓度按式(1)计算：

$$
\rho \left(\mathrm {C l O} _ {2}\right) = \frac {c \times \left(V _ {1} - V _ {0}\right) \times 1 3 . 4 9}{V _ {2}} \quad \dots \tag {1}
$$

式中：

$\rho (\mathrm{ClO}_2)$ 二氧化氯标准储备溶液的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$

$c$ ——硫代硫酸钠标准溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{1}$ ——滴定二氧化氯所用硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——滴定空白所用硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ —二氧化氯体积，单位为毫升 $(\mathrm{mL})$

13.49 与 $1.00\mathrm{mL}$ 硫代硫酸钠标准溶液 $[c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1000\mathrm{mol / L}](3.2.10)$ 相当的以毫克表示的二氧化氯的质量。

# 3.4 二氧化氯标准溶液配制

$1\mathrm{mL}$ 溶液含二氧化氯 $(\mathrm{ClO}_2)0.25\mathrm{mg}$ ，根据测定的二氧化氯储备液的浓度，吸取一定量二氧化氯标准储备液，用无氯二级水(3.2.2)进行稀释。该溶液即用即配。

# 4 仪器和设备

4.1 紫外-可见分光光度计。  
4.2 均质器： $\geqslant 8000\mathrm{r / min}$   
4.3 离心机： $\geqslant 10000\mathrm{r / min}$   
4.4 具塞锥形瓶： $100~\mathrm{mL}$   
4.5 滤膜： $0.22\mu \mathrm{m}$ 。  
4.6 移液管： $1.0\mathrm{mL},2.0\mathrm{mL},5.0\mathrm{mL},10.0\mathrm{mL}$   
4.7 分析天平。

# 5 分析步骤

# 5.1 试样制备

从所取全部样品中取出有代表性样品约 $1\mathrm{kg}$ ，经捣碎机充分捣碎均匀，均分成两份，分别装入洁净容器内作为试样。密封并加贴标签。

# 5.2 试样保存

将试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。在抽样及制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 5.3 提取

# 5.3.1 水果及蔬菜

称取试样 $1.00\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，加入 $20~\mathrm{mL}$ 磷酸盐缓冲溶液 $(\mathrm{pH}6.5)(3.2.8)$ ， $8000\mathrm{r / min}$ 均质提取 $3\mathrm{min}$ ，于高速离心机中 $10000\mathrm{r / min}$ 冷冻离心 $10\mathrm{min}$ ，取出，用纤维滤纸过滤于 $10~\mathrm{mL}$ 具塞比色管中，供分光光度计测定。

# 5.3.2 畜禽肉及水产品

称取试样 $1.00\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）于 $100~\mathrm{mL}$ 离心管中，加入 $50~\mathrm{mL}$ 磷酸盐缓冲溶液（pH6.5）(3.2.8)， $8000\mathrm{r / min}$ 均质提取 $3\mathrm{min}$ ，于高速离心机中 $10000\mathrm{r / min}$ 冷冻离心 $10\mathrm{min}$ ，取出，用纤维滤纸过滤于 $10~\mathrm{mL}$ 具塞比色管中，供分光光度计测定。

# 5.4 测定

# 5.4.1 标准曲线的绘制

向一系列 $10~\mathrm{mL}$ 具塞比色管中加入一定量的二氧化氯标准使用溶液，使各管中的浓度相当于 $0.00\mathrm{mg / L},0.05\mathrm{mg / L},0.10\mathrm{mg / L},0.50\mathrm{mg / L},1.00\mathrm{mg / L},2.00\mathrm{mg / L},5.00\mathrm{mg / L}$ 的二氧化氯标准溶液。

分别加入 $1.0 \mathrm{~mL}$ 磷酸盐缓冲液(3.2.8)和 $1.0 \mathrm{~mL}$ DPD溶液(3.2.1)、 $1.0 \mathrm{~mL}$ 甘氨酸溶液，定容至刻度摇匀，在 $60 \mathrm{~s}$ 内，用 $1 \mathrm{~cm}$ 比色皿在 $552 \mathrm{~nm}$ 处测定吸光度，以标准工作液的浓度为横坐标，以响应值（吸收值）为纵坐标，绘制标准曲线，见附录A。

# 5.4.2 样品中二氧化氯的测定

取 $5\mathrm{mL}$ 滤液于 $10~\mathrm{mL}$ 具塞比色管中，向滤液中加入 $1\mathrm{mL}$ 甘氨酸溶液(3.2.3)混合，加入 $1\mathrm{mL}$ 磷酸盐缓冲液(3.2.8)和 $1.0\mathrm{mL}$ DPD溶液(3.2.1)，用水定容至刻度 $10~\mathrm{mL}$ 摇匀。立即在 $60~\mathrm{s}$ 内，用 $1\mathrm{cm}$ 比色皿在 $552~\mathrm{nm}$ 处测定吸光度，根据标准曲线求出二氧化氯的浓度。

注：整个操作过程需避光。

# 6 分析结果的表述

试样中二氧化氯含量按式(2)计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V}{m} \tag {2}
$$

式中：

$X$ ——试样中 $\mathrm{ClO}_2$ 含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——样液中 $\mathrm{ClO}_2$ 的浓度，单位为毫克每升 $(\mathrm{mg / L})$

$c_{0}$ ——试剂空白中 $\mathrm{ClO_2}$ 的浓度，单位为毫克每升 $(\mathrm{mg / L})$

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——最终样液所代表的试样质量，单位为克 $(\mathrm{g})$ 。

计算结果保留小数点后两位。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

方法检出限为 $0.65\mathrm{mg / kg}$ ，水果及蔬菜定量限为 $2.00\mathrm{mg / kg}$ ，肉及水产品定量限为 $5.00\mathrm{mg / kg}$ 。

# 食品安全国家标准

# 食品添加剂 皂树皮提取物

# 1 范围

本标准适用于以皂树(Quillajasaponaria Molina)的树皮、树干或枝条为原料，磨碎后使用水溶剂提取法提取出来经净化、精制等工艺生产的食品添加剂皂树皮提取物。

# 2 产品分类

皂树皮提取物按照不同皂素含量范围分为1型和2型两种产品类型。1型是以树皮、树干或枝条为原料，通过水溶剂提取等一般精制方法获得的。2型是以树皮、树干或枝条为原料，经水溶剂提取法提取出来后通过色谱分离或超滤方法获得的。

# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度应在分析纯以上，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。高效液相色谱试验用水应符合GB/T6682中一级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 可溶性试验

试样应具有较强水溶性，但不溶于乙醇、丙酮、甲醇和丁醇。

# A.2.2 泡沫试验

称取 $0.5\mathrm{g}$ 粉末试样溶解在 $9.5\mathrm{g}$ 水中或量取 $1\mathrm{mL}$ 液体试样混合在 $9\mathrm{mL}$ 水中，加入装有 $350~\mathrm{mL}$ 水的 $1000~\mathrm{mL}$ 量筒中，量筒加盖后用力振摇30次后静置， $30\mathrm{min}$ 后记录泡沫体积，1型泡沫体积应达到 $150~\mathrm{mL}$ 以上，2型泡沫体积应达到 $260~\mathrm{mL}$ 以上。

# A.2.3 高效液相色谱法

按照皂素含量高效液相色谱测定法(A.3)，试样主峰的保留时间应与皂素标准品的皂素主要组分的主峰一致。

# A.2.4 吸光度试验（仅限粉末试样）

观察A.2.2中粉末试样的溶液，溶液不应出现浑浊。将该溶液置于在 $520~\mathrm{nm}$ 波长下测定其吸光度，1型吸光度应小于1.2，2型吸光度应小于0.7。

# A.3 皂素含量（以干基计）的测定

# A.3.1 方法原理

使用高效液相色谱将皂素的主要组分QS-7、QS-17、QS-18和QS-21分离，皂树皮提取物中皂素含量以QS-7、QS-17、QS-18和QS-21总量计算。

# A.3.2 试剂和材料

A.3.2.1 皂树皮皂素标准品(或类似皂素含量已知的皂素标准品，CAS号：68990-67-0)。  
A.3.2.2 三氟乙酸。  
A.3.2.3 乙腈（色谱级）。  
A.3.2.4 $0.22\mu \mathrm{m}$ 孔径的过滤膜，水相。

# A.3.3 仪器和设备

高效液相色谱仪：配备紫外检测器。

# A.3.4 参考色谱条件

A.3.4.1 色谱柱： $\mathrm{C_4}$ 色谱柱(长 $4.6\mathrm{mm}\times 250\mathrm{mm}$ ，孔径 $300\mathrm{A}$ ，粒径 $5\mu \mathrm{m})$ 或其他等效色谱柱。

A.3.4.2 柱温：室温（ $30^{\circ}\mathrm{C}$ ）。  
A.3.4.3 流动相 $\mathrm{A}:0.15\%$ 三氟乙酸水溶液。  
A.3.4.4 流动相 $\mathrm{B}:0.15\%$ 三氟乙酸乙腈溶液。  
A.3.4.5 流速： $1.0~\mathrm{mL / min}$   
A.3.4.6 检测波长： $220~\mathrm{nm}$   
A.3.4.7 进样体积： $20~\mu \mathrm{L}$   
A.3.4.8 流动相梯度洗脱程序见表A.1（比例可根据实际情况调节）。

# A.3.5 分析步骤

# A.3.5.1 试样溶液的制备

# A.3.5.1.1 粉末试样

准确称取 $0.5\mathrm{g}$ 试样，精确至 $0.001\mathrm{g}$ ，在 $9.5\mathrm{g}$ 水中溶解，定容至 $10\mathrm{mL}$ 后过 $0.22\mu \mathrm{m}$ 滤膜。

# A.3.5.1.2 液体试样

准确称取 $1.0\mathrm{g}$ 试样，精确至 $0.001\mathrm{g}$ ，用 $9\mathrm{mL}$ 水稀释，定容至 $10\mathrm{mL}$ 后过 $0.22\mu \mathrm{m}$ 滤膜。

# A.3.5.2 标准溶液的制备

准确称取 $1.5\mathrm{g}$ 皂素标准品，精确至 $0.001\mathrm{g}$ ，加水，定容至 $100\mathrm{mL}$ ，使用 $0.22\mu \mathrm{m}$ 孔径的过滤膜进行过滤。

# A.3.5.3 测定

在A.3.4参考色谱条件下，分别对标准溶液和试样溶液进行色谱分析，记录所得标准溶液中4个主要皂素(QS-7、QS-17、QS-18和QS-21)峰面积的总和 $A_{\mathrm{S}}$ 和试样溶液中4个主要皂素峰面积的总和 $A_{i}$ 。

# A.3.6 结果计算

# A.3.6.1 试样溶液中的皂素浓度计算

根据上述样品制备方法制备的溶液中皂素的质量浓度 $C_i$ ，单位为毫克每毫升 $(\mathrm{mg / mL})$ ，按式(A.1)计算：

$$
C _ {i} = \frac {A _ {i}}{A _ {\mathrm {S}}} \times C _ {\mathrm {S}} \quad \dots \tag {A.1}
$$

式中：

$A_{i}$ ——试样中4个主要皂素组分QS-7、QS-17、QS-18和QS-21峰面积的总和，如附录B中示意图所示（皂素的主峰将在多酚主峰出现后出峰，见图B.2）；

$A_{\mathrm{S}}$ ——标准品中4个主要皂素峰面积的总和；

$C_{\mathrm{S}}$ ——标准品的皂素质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ 。

# A.3.6.2 试样中皂素（以干基计）的质量分数

试样中皂素(以干基计)的质量分数 $\varpi_{1}$ ，按式(A.2)计算：

$$
w _ {1} = \frac {C _ {i} \times V _ {i}}{m _ {i}} \times 100 \% \tag{A.2}
$$

式中：

$C_i$ ——试样溶液中的皂素质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V_{i}$ ——制备的试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m_{i}$ ——试样干燥减量后的质量，单位为毫克 $(\mathrm{mg})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $2.0\%$ 。

# A.4 $\mathbf{pH}$ 的测定

用水配制成 $4\%$ 质量分数的皂树皮提取物待测液，按GB/T9724规定的方法测定。

# A.5 丹宁酸（以干基计）的测定

# A.5.1 试剂和材料

A.5.1.1 乙酸。

A.5.1.2 聚乙烯聚吡咯烷酮。

# A.5.2 仪器和设备

A.5.2.1 电热恒温干燥箱。

A.5.2.2 离心机。

# A.5.3 分析步骤

称取 $3.0\mathrm{g}$ 粉末试样，或含有 $3.0\mathrm{g}$ 固形物（使用干燥减量数值换算）的液体试样，精确至 $0.01\mathrm{g}$ ，溶解在 $250~\mathrm{mL}$ 水中，用乙酸调整 $\mathrm{pH}$ 为3.5，量取 $25~\mathrm{mL}$ 溶液，在 $105^{\circ}\mathrm{C}$ 的温度条件下，干燥 $5\mathrm{h}$ ，冷却，称重。量取 $50~\mathrm{mL}$ 溶液与 $360~\mathrm{mg}$ 聚乙烯聚吡咯烷酮混合，在室温下搅拌 $30\mathrm{min}$ ，然后以 $3000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。收集上层清液，并在 $105^{\circ}\mathrm{C}$ 的温度条件下干燥 $5\mathrm{h}$ ，冷却，称重。

# A.5.4 结果计算

丹宁酸(以干基计)的质量分数 $w$ ，按式(A.3)计算：

$$
w = \frac {m _ {1} - \frac {m _ {2}}{2}}{m _ {1}} \times 100 \% \tag{A.3}
$$

式中：

$m_{1}$ ——加入聚乙烯聚吡咯烷酮前的溶液干燥后的质量，单位为克 $(\mathrm{g})$ ；

$m_{2}$ ——加入聚乙烯聚吡咯烷酮后的溶液干燥后的质量，单位为克 $(\mathrm{g})$ ；

# 2 ——换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.5\%$ 。

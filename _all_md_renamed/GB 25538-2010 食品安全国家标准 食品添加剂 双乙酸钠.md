# 检验方法

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682—2008中规定的水。分析中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。本试验所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

A. 2.1 取铂丝, 蘸取盐酸, 在无色火焰上燃烧呈无色后, 蘸取试样, 在无色火焰中燃烧, 火焰即显亮黄色。  
A. 2.2 将试样配成 $100 \mathrm{mg} / \mathrm{mL}$ 的水溶液, 与硫酸及乙醇共热时, 产生乙酸乙酯的特殊气味。试样的中性水溶液遇氯化铁溶液 $(90 \mathrm{~g} / \mathrm{L})$ , 呈深红色。

# A.3 游离乙酸的测定

# A.3.1 试剂和材料

a) 氢氧化钠标准滴定溶液: $c(\mathrm{NaOH}) = 1 \mathrm{~mol} / \mathrm{L}$ 。  
b) 酚酞指示液： $10\mathrm{g/L}$ 。

# A.3.2 分析步骤

称取约 $4 \mathrm{~g}$ 试样（精确至 $0.0001 \mathrm{~g}$ ），置于 $250 \mathrm{~mL}$ 锥形瓶中，加入 $50 \mathrm{~mL}$ 水，使试样完全溶解。加入2滴酚酞指示液，用氢氧化钠标准滴定溶液滴定至微红色，保持30s不退色即为终点。记下所消耗氢氧化钠标准滴定溶液的毫升数。

# A.3.3 结果计算

游离乙酸的含量按公式（A.1）计算。

$$
X _ {1} = \frac {V _ {1} \times c \times 0.06005}{\mathrm {m} _ {1} \times \left(1 - X _ {0}\right)} \times 100 \% \dots \tag{A.1}
$$

式中：

$X_{1}$ ——游离乙酸的含量（以干基计）， $\%$ ；

$V_{1}$ ——试样消耗氢氧化钠标准滴定溶液的体积, 单位为毫升 $(\mathrm{mL})$ ;

$c$ ——氢氧化钠标准滴定溶液的实际浓度，单位为摩尔每升（mol/L）；

0.06005——与 $1.00 \mathrm{~mL}$ 氢氧化钠标准滴定溶液 $[c(\mathrm{NaOH}) = 1.00 \mathrm{~mol} / \mathrm{L}]$ 相当的以克表示的乙酸的质量；

$m_{1}$ ——试样的质量，单位为克（g）。

$X_{0}$ ——实测试样中水分的含量， $\%$

检测结果以平行测定结果的算术平均值为准。两次平行测定结果之差不大于 $0.2\%$

# A.4 乙酸钠的测定

# A.4.1 自动电位滴定仪法（方法一）

# A.4.1.1 试剂和材料

a)邻苯二甲酸氢钾：基准试剂。  
b) 冰乙酸。  
c) 高氯酸标准滴定溶液: $c(\mathrm{HClO}_4)=0.1 \mathrm{~mol} / \mathrm{L}$ , 按GB/T 601—2002中规定的方法(方法一)配制,按下法标定:

称取约 $0.36 \mathrm{~g}$ (精确至 $0.0001 \mathrm{~g}$ ) 于 $105^{\circ} \mathrm{C} \sim 110^{\circ} \mathrm{C}$ 的烘箱中干燥至恒重的基准试剂邻苯二甲酸氢钾，置于干燥的滴定杯中，加入 $40 \mathrm{~mL}$ 冰乙酸，搅拌，溶解。用配制好的高氯酸标准滴定溶液滴定，用电位滴定仪确定滴定终点。

平行测定两次结果的算术平均值为测定结果。同时记录高氯酸标准滴定溶液的温度值 $t$ 。

电位滴定仪按公式（A.2）自动计算高氯酸标准滴定溶液的浓度：

$$
c _ {\left(H C l O _ {4}\right)} = \frac {m _ {2} \times 1 0 0 0}{V _ {2} \times M} \dots \tag {A.2}
$$

式中：

$c_{\left(\mathrm{HClO}_{4}\right)}$ ——高氯酸标准滴定溶液的标定浓度, 单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ;

$m_{2}$ ——邻苯二甲酸氢钾的质量，单位为克（g）；

$V_{2}$ ——消耗高氯酸标准滴定溶液的体积，单位为毫升（ $\mathrm{mL}$ ）；

$M$ ——邻苯二甲酸氢钾的摩尔质量，单位为克每摩尔（ $\mathrm{g/mol}$ ） $[M(KHC_{8}H_{4}O_{4})=204.22]$ 。

两次平行测定结果之差不大于 $0.2\%$

# A.4.1.2 仪器和设备

a) 电位滴定仪：自动电位滴定仪或相当精度的其他仪器。  
b) 复合玻璃电极: 非水相酸碱滴定玻璃电极, 使用 $\mathrm{Ag} / \mathrm{AgCl}$ 或甘汞电极作为参考, 测量范围 $\pm 2050.0 \mathrm{mV}$ , 分辨率 $0.1 \mathrm{mV}$ , 最大允许误差 $0.1 \%$ 。或相当精度的其他电极。

# A.4.1.3 分析步骤

# A.4.1.3.1 准备

使用高氯酸标准滴定溶液时的温度应与标定时的温度相同，若温度不相同，应将高氯酸标准滴定溶液的浓度按公式（A.3）修正到使用温度下的浓度，并输入仪器中。

$$
c _ {1} \left(\mathrm {H C l O} _ {4}\right) = \frac {\mathrm {c}}{1 + 0 . 0 0 1 1 \times \left(t _ {1} - t\right)} \dots \tag {A.3}
$$

式中：

$c_{1}$ ——使用时高氯酸标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$c$ ——标定温度下高氯酸标准滴定溶液的浓度, 单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ;

0.0011——高氯酸标准滴定溶液每改变 $1^{\circ} \mathrm{C}$ 时的体积膨胀系数；

$t_1$ ——使用时高氯酸标准滴定溶液的温度，单位为摄氏度（ $^{\circ}\mathrm{C}$ ）；

$t$ ——标定高氯酸标准滴定溶液的温度，单位为摄氏度（ $^{\circ}\mathrm{C}$ ）。

# A.4.1.3.2 测定

称取约 $0.25 \mathrm{~g}$ 试样（精确至 $0.0001 \mathrm{~g}$ ），置于干燥的滴定杯中，加入 $40 \mathrm{~mL}$ 冰乙酸，搅拌，溶解，

此为试样液。用高氯酸标准滴定溶液滴定试样液，用电位滴定仪确定滴定终点。

# A.4.1.4 结果计算

乙酸钠的含量按公式（A.4）计算：

$$
X _ {4} = \frac {V _ {4} \times c _ {4} \times 0.08203}{m _ {4} \times \left(1 - X _ {0}\right)} \times 100 \% \dots \tag{A.4}
$$

式中：

$X_{4}$ ——乙酸钠的含量（以干基计）， $\%$ ；

$V_{4}$ ——滴定试样消耗高氯酸标准滴定溶液的体积，单位为毫升（ $\mathrm{mL}$ ）；

$c_{4}$ ——高氯酸标准滴定溶液的实际浓度，单位为摩尔每升（ $\mathrm{mol} / \mathrm{L}$ ）；

0.08203——与 $1.00 \mathrm{~mL}$ 高氯酸标准滴定溶液 $[c(\mathrm{HClO}_{4}) = 1.000 \mathrm{~mol} / \mathrm{L}]$ 相当的以克表示的乙酸钠的质量；

$m_{4}$ ——试样的质量, 单位为克 $(\mathrm{g})$ ;

$X_0$ ——实测试样中水分的含量， $\%$ 。

取平行测定结果的算术平均值为测定结果，两次平行测定结果之差不大于 $0.2\%$

# A. 4.2 pH计法（方法二）

# A.4.2.1 试剂和材料

a) 冰乙酸。  
b) 高氯酸标准滴定溶液: $c\left(\mathrm{HClO}_{4}\right)=0.1 \mathrm{~mol} / \mathrm{L}$ , 按 GB/T 601-2002 中规定的方法 (方法一) 配制并标定。

# A.4.2.2 仪器和设备

a) pH计。  
b) 玻璃电极及甘汞电极。  
c) 磁力搅拌器。

# A.4.2.3 分析步骤

称取约 $0.48 \mathrm{~g}$ 试样（精确至 $0.0001 \mathrm{~g}$ ），置于一个 $200 \mathrm{~mL}$ 的干燥烧杯中，加入 $50 \mathrm{~mL}$ 冰乙酸，置于磁力搅拌器上搅拌溶解，此为试样液。将 $\mathrm{pH}$ 计的玻璃电极及甘汞电极浸入试样液的适当位置，用高氯酸标准滴定溶液滴定，接近滴定终点前后 $1 \mathrm{~mL} \sim 2 \mathrm{~mL}$ 内，每滴加 $0.1 \mathrm{~mL}$ 高氯酸标准滴定溶液，就测一次电动势，记下消耗高氯酸标准滴定溶液的毫升数和相应的电动势，用二次微商法计算滴定终点时消耗的高氯酸标准滴定溶液的体积。

注：滴定过程应在通风橱中进行。

# A.4.2.4 结果计算

乙酸钠的含量按公式（A.5）计算。

$$
X _ {5} = \frac {V _ {5} \times c _ {5} \times 0.08203}{m _ {5} \times \left(1 - X _ {0}\right)} \times 100 \% \dots \tag{A.5}
$$

式中：

$X_{5}$ ——乙酸钠的含量（以干基计）， $\%$ ；

$V_{5}$ ——滴定试样消耗高氯酸标准滴定溶液的体积, 单位为毫升 $(\mathrm{mL})$ ;

$c_{5}$ ——高氯酸标准滴定溶液的实际浓度，单位为摩尔每升（mol/L）；

0.08203——与 $1.00 \mathrm{~mL}$ 高氯酸标准滴定溶液 $[c(\mathrm{HClO}_{4}) = 1.000 \mathrm{~mol} / \mathrm{L}]$ 相当的以克表示的乙酸钠的质量；

$m_{5}$ ——试样的质量，单位为克（g）。

$X_{0}$ ——实测试样中水分的含量， $\%$ 。

检测结果以平行测定结果的算术平均值为准。两次平行测定结果之差不大于 $0.2\%$

# A. 5 pH的测定

称取约 $5 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ ) 试样于 $100 \mathrm{~mL}$ 烧杯中, 加 $50 \mathrm{~mL}$ 无二氧化碳的水溶解, 混匀, 用 $\mathrm{pH}$ 计测定。

# A.6 易氧化物的测定

# A.6.1 试剂和材料

a) 高锰酸钾标准滴定溶液: $c(1/5\mathrm{KMnO}_4)=0.1\mathrm{mol/L}$ 。  
b) 硫酸溶液： $1\mathrm{mol} / \mathrm{L}$ 。

# A.6.2 分析步骤

称取约 $1 \mathrm{~g}$ 试样（精确到 $0.0001 \mathrm{~g}$ ），置于一个锥形瓶中，加入 $50 \mathrm{~mL}$ 水溶解，再加入 $10 \mathrm{~mL}$ 硫酸溶液。将溶液在电炉上加热至 $70^{\circ} \mathrm{C} \sim 80^{\circ} \mathrm{C}$ ，用高锰酸钾标准滴定溶液滴定，使溶液呈微红色，保持 $15 \mathrm{~s}$ 不褪色，即为滴定终点。记下高锰酸钾标准滴定溶液消耗的体积（ $\mathrm{mL}$ ）。

# A.6.3 结果计算

易氧化物（以甲酸计）的含量按公式（A.6）计算：

$$
X _ {6} = \frac {c _ {6} \times V _ {6} \times 0.02301}{m _ {6}} \times 100 \% \dots \tag{A.6}
$$

式中：

$X_{6}$ ——易氧化物的含量， $\%$ ；

$c_{6}$ ——高锰酸钾标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{6}$ ——高锰酸钾标准滴定溶液滴定消耗的体积，单位为毫升（ $\mathrm{mL}$ ）；

0.02301——与 $1.00 \mathrm{~mL}$ 高锰酸钾标准滴定溶液 $[c(1 / 5 \mathrm{KMnO}_{4}) = 1.000 \mathrm{~mol} / \mathrm{L}]$ 相当的以克表示的甲酸的质量；

$m_{6}$ ——试样的质量，单位为克（g）。

检测结果以平行测定结果的算术平均值为准。两次平行测定结果之差不大于 $0.05\%$

# 乳和乳制品杂质度的测定

2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替GB5413.30—2010《食品安全国家标准 乳和乳制品杂质度的测定》。

本标准与GB5413.30—2010相比，主要变化如下：

——增加了杂质度过滤板技术要求；  
——简化了附录A的检验步骤，并将附录中测量杂质损失量修改为测量杂质残留量；  
——将附录B中的杂质度参考标准板制作修改为液体乳和乳粉类两种标准板制作方法；  
——重新确定了杂质组成成分及颗粒度的大小。

# 食品安全国家标准

# 乳和乳制品杂质度的测定

# 1 范围

本标准规定了乳和乳制品杂质度的测定方法。

本标准适用于生鲜乳、巴氏杀菌乳、灭菌乳、炼乳及乳粉杂质度的测定，不适用于添加影响过滤的物质及不溶性有色物质的乳和乳制品。

# 2 原理

生鲜乳、液体乳、用水复原的乳粉类样品经杂质度过滤板过滤，根据残留于杂质度过滤板上直观可见非白色杂质与杂质度参考标准板比对确定样品杂质的限量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

3.1 杂质度过滤板：直径 $32\mathrm{mm}$ 、质量 $135\mathrm{mg}\pm 15\mathrm{mg}$ 、厚度 $0.8\mathrm{mm}\sim 1.0\mathrm{mm}$ 的白色棉质板，应符合附录A的要求。杂质度过滤板按附录A进行检验。  
3.2 杂质度参考标准板：杂质度参考标准板的制作方法见附录B。

# 4 仪器和设备

4.1 天平：感量为 $0.1\mathrm{g}$ 。  
4.2 过滤设备：杂质度过滤机或抽滤瓶，可采用正压或负压的方式实现快速过滤（每升水的过滤时间为 $10\mathrm{s}\sim 15\mathrm{s}$ ）。安放杂质度过滤板后的有效过滤直径为 $28.6\mathrm{mm}\pm 0.1\mathrm{mm}$ 。

# 5 分析步骤

# 5.1 样品溶液的制备

5.1.1 液体乳样品充分混匀后，用量筒量取 $500~\mathrm{mL}$ 立即测定。  
5.1.2 准确称取 $62.5\mathrm{g}\pm 0.1\mathrm{g}$ 乳粉样品于 $1000~\mathrm{mL}$ 烧杯中，加入 $500~\mathrm{mL}40^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 的水，充分搅拌溶解后，立即测定。

# 5.2 测定

将杂质度过滤板放置在过滤设备上，将制备的样品溶液倒入过滤设备的漏斗中，但不得溢出漏斗，过滤。用水多次洗净烧杯，并将洗液转入漏斗过滤。分次用洗瓶洗净漏斗过滤，滤干后取出杂质度过滤板，与杂质度标准板比对即得样品杂质度。

# 6 分析结果的表述

过滤后的杂质度过滤板与杂质度参考标准板比对得出的结果，即为该样品的杂质度。

当杂质度过滤板上的杂质量介于两个级别之间时，应判定为杂质量较多的级别。如出现纤维等外来异物，判定杂质度超过最大值。

# 7 精密度

按本标准所述方法对同一样品做两次测定，其结果应一致。

# 附录A

# 杂质度过滤板的检验

# A.1 试剂和材料

# A.1.1 试剂

A.1.1.1 无水乙醇 $\mathrm{(C_2H_5OH)}$ 。  
A.1.1.2 甲醛（HCHO）。  
A.1.1.3 角豆胶：生化试剂。  
A.1.1.4 蔗糖。

# A.1.2 试剂配制

A.1.2.1 甲醛溶液 $(40\%)$ ：量取 $40~\mathrm{mL}$ 甲醛到 $100~\mathrm{mL}$ 容量瓶中，用水定容至 $100~\mathrm{mL}$ ，过滤备用。  
A.1.2.2角豆胶溶液：称取 $0.75\mathrm{g}\pm 0.01\mathrm{g}$ 角豆胶至 $250~\mathrm{mL}$ 烧杯中，加 $2\mathrm{mL}$ 无水乙醇润湿，再加 $50~\mathrm{mL}$ 水，充分混合。缓慢加热排除气泡后，煮沸，使角豆胶充分溶解后，冷却。加 $2\mathrm{mL}$ 已过滤的 $40\%$ 甲醛溶液，混匀后转入 $100~\mathrm{mL}$ 容量瓶，用水定容。  
A.1.2.3 蔗糖溶液：称取 $750\mathrm{g}\pm 0.1\mathrm{g}$ 蔗糖于 $1000~\mathrm{mL}$ 烧杯中，加水 $750~\mathrm{mL}$ 充分溶解，过滤备用。

# A.1.3 材料

杂质：用地面灰土经过恒温干燥箱 $(100^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C})$ 烘干，用标准筛收集颗粒大小为 $75\mu \mathrm{m} \sim 106\mu \mathrm{m}$ 的灰土成分，然后烘干至恒重。

# A.2 仪器和设备

A.2.1 天平：感量分别为 $0.1\mathrm{g}$ 和 $0.1\mathrm{mg}$   
A.2.2 标准筛。  
A.2.3 干燥器：含有效干燥剂。  
A.2.4 恒温干燥箱；精度为 $\pm 1\,^{\circ}\mathrm{C}$ 。  
A.2.5 过滤设备：同4.2。

# A.3 检验步骤

A.3.1 杂质溶液制备：称取 $2.00 \mathrm{~g} \pm 0.001 \mathrm{~g}$ 杂质加入 $250 \mathrm{~mL}$ 烧杯中，用 $5 \mathrm{~mL}$ 无水乙醇润湿。加入 $46 \mathrm{~mL}$ 角豆胶溶液，再加 $40 \mathrm{~mL}$ 蔗糖溶液，充分混合后，转入 $100 \mathrm{~mL}$ 容量瓶加蔗糖溶液定容，充分混匀。移取 $10 \mathrm{~mL}$ （相当于 $200 \mathrm{mg}$ 杂质）于 $1000 \mathrm{~mL}$ 容量瓶中，用水定容，充分混匀。  
A.3.2 将杂质度过滤板，放入 $100^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温干燥箱中烘干至恒重，记录质量 $N_{1}$ 。  
A.3.3 将杂质度过滤板放置在过滤设备上，准确移取 $60~\mathrm{mL}$ （相当于 $12\mathrm{mg}$ 杂质）经过充分混匀的杂质溶液，过滤，用水洗净移液器，洗液一并过滤，用 $200~\mathrm{mL}40^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 的水分多次清洗过滤板，滤干后取下杂质度过滤板，在 $100^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 恒温干燥箱中烘干至恒重，记录质量 $N_{2}$ 。

# A.4 评价

A.4.1 $M = N_{2} - N_{1}, M$ 应 $\geqslant 10\mathrm{mg}$ 。并且用锋利的刀片将杂质度过滤板上表层切下，查看余下部分不应出现杂质。  
A.4.2 每千片检验10片，不足1000片按1000片计。

# 杂质度参考标准板的制作

# B.1 试剂和材料

# B.1.1 试剂

B.1.1.3 牛粪和焦粉：分别收集牛粪和焦粉，粉碎后 $100^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 恒温干燥箱中烘干。

# B.3 液体乳参考标准杂质板制作步骤

# B.3.1 液体乳杂质参考标准液的配制

B.3.1.1 分别准确称取 $500.0 \mathrm{mg}$ 牛粪 A、B、C 于 3 个 $100 \mathrm{~mL}$ 烧杯中。加水 $2 \mathrm{~mL}$ , 加阿拉伯胶溶液 $23 \mathrm{~mL}$ , 充分混匀后, 用蔗糖溶液转入 $500 \mathrm{~mL}$ 容量瓶中并定容, 充分混匀直到杂质均匀分布, 得到浓度为 $1.0 \mathrm{mg} / \mathrm{mL}$ 的牛粪杂质参考标准液 $\mathrm{a}_{0}, \mathrm{~b}_{0}, \mathrm{c}_{0}$ 。  
B.3.1.2 分别吸取牛粪杂质参考标准液 $\mathrm{a}_0, \mathrm{b}_0, \mathrm{c}_0$ 各 $100\mathrm{mL}$ 于 $500\mathrm{mL}$ 容量瓶中，用蔗糖溶液稀释并

定容，得浓度为 $0.2\mathrm{mg / mL}$ 的牛粪杂质参考标准中间液 $\mathrm{a}_1,\mathrm{b}_1,\mathrm{c}_1$ 。

B.3.1.3 分别吸取牛粪杂质参考标准中间液 $\mathrm{a}_{1},\mathrm{b}_{1},\mathrm{c}_{1}$ 各 $10~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用蔗糖溶液稀释并定容，得浓度为 $0.02\mathrm{mg / mL}$ 的牛粪杂质参考标准工作液 $\mathrm{a}_2,\mathrm{b}_2,\mathrm{c}_2$ 。

# B.3.2 液体乳参考标准杂质板的制作

B.3.2.1 量取 $100 \mathrm{~mL}$ 蔗糖溶液，在已放置好杂质度过滤板的过滤设备上过滤，用 $100 \mathrm{~mL} 40^{\circ} \mathrm{C} \pm 2^{\circ} \mathrm{C}$ 的水分多次清洗过滤板，晾干，此杂质板为液体乳中杂质相对含量 $0 \mathrm{mg} / \mathrm{kg}$ 的杂质度参考标准板 $\mathrm{A}_{1}$ 。  
B.3.2.2 准确吸取 $6.25 \mathrm{~mL}$ 牛粪杂质参考标准工作液 $\mathrm{c}_{2}$ 于 $100 \mathrm{~mL}$ 容量瓶中，用蔗糖溶液稀释并定容，混匀后并在已放置好杂质度过滤板的过滤设备上过滤，用水洗净容量瓶，洗液一并过滤。再用 $100 \mathrm{~mL} 40^{\circ} \mathrm{C} \pm 2^{\circ} \mathrm{C}$ 的水分多次清洗过滤板，晾干，此杂质板为液体乳中杂质相对含量 $2 \mathrm{mg} / 8 \mathrm{~L}$ 的杂质度参考标准板 $\mathrm{A}_{2}$ 。  
B.3.2.3 准确吸取 $12.5 \mathrm{~mL}$ 牛粪杂质参考标准工作液 $\mathrm{b}_{2}$ 于 $100 \mathrm{~mL}$ 容量瓶中，用蔗糖溶液稀释并定容，混匀后并在已放置好杂质度过滤板的过滤设备上过滤，用水洗净容量瓶，洗液一并过滤。再用 $100 \mathrm{~mL} 40^{\circ} \mathrm{C} \pm 2^{\circ} \mathrm{C}$ 的水分多次清洗过滤板，晾干，此杂质板为液体乳中杂质相对含量 $4 \mathrm{mg} / 8 \mathrm{~L}$ 的杂质度参考标准板 $\mathrm{A}_{3}$ 。  
B.3.2.4 准确吸取 $18.75 \mathrm{~mL}$ 牛粪杂质参考标准工作液 $\mathrm{a}_{2}$ 于 $100 \mathrm{~mL}$ 容量瓶中, 用蔗糖溶液稀释并定容, 混匀后并在已放置好杂质度过滤板的过滤设备上过滤, 用水洗净容量瓶, 洗液一并过滤。再用 $100 \mathrm{~mL} 40^{\circ} \mathrm{C} \pm 2^{\circ} \mathrm{C}$ 的水分多次清洗过滤板, 晾干, 此杂质板为液体乳中杂质相对含量 $6 \mathrm{mg} / 8 \mathrm{~L}$ 的杂质度参考标准板 $\mathrm{A}_{4}$ 。

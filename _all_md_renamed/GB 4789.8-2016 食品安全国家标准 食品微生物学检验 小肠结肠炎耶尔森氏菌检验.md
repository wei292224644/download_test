# 前言

本标准代替GB/T4789.8—2008《食品卫生微生物学检验 小肠结肠炎耶尔森氏菌检验》。

本标准与GB/T4789.8—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品微生物学检验 小肠结肠炎耶尔森氏菌检验”；  
——修改了典型菌落的形态描述；  
——删除了生化鉴定中的商品化名称；  
——描述了血清鉴定的方法。

# 1 范围

本标准规定了食品中小肠结肠炎耶尔森氏菌(Yersinia enterocolitica)的检验方法。  
本标准适用于食品中小肠结肠炎耶尔森氏菌的检验。

# 2 设备和材料

除微生物实验室常规灭菌及培养设备外，其他设备和材料如下：

2.1 冰箱： $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 。  
2.2 恒温培养箱： $26^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}, 36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 。  
2.3 显微镜：10倍～100倍。

2.7 灭菌吸管： $1\mathrm{mL}$ （具 $0.01~\mathrm{mL}$ 刻度）、 $10~\mathrm{mL}$ （具 $0.1\mathrm{mL}$ 刻度）。  
2.8 锥形瓶： $200~\mathrm{mL},500~\mathrm{mL}$   
2.9 灭菌平皿：直径 $90~\mathrm{mm}$   
2.10 微生物生化鉴定试剂盒或微生物生化鉴定系统。

# 5 操作步骤

# 5.1 增菌

以无菌操作取 $25\mathrm{g}$ (或 $25~\mathrm{mL}$ )样品放入含有 $225~\mathrm{mL}$ 改良磷酸盐缓冲液增菌液的无菌均质杯或均质袋内，以 $8000\mathrm{r / min}$ 均质 $1\mathrm{min}$ 或拍击式均质器均质 $1\mathrm{min}$ 。液体样品或粉末状样品，应振荡混匀。均质后于 $26^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 增菌 $48\mathrm{h}\sim 72\mathrm{h}$ 。增菌时间长短可根据对样品污染程度的估计来确定。

# 5.2 碱处理

除乳与乳制品外，其他食品的增菌液 $0.5\mathrm{mL}$ 与碱处理液 $4.5\mathrm{mL}$ 充分混合 $15\mathrm{s}$ 。

# 5.3 分离

将乳与乳制品增菌液或经过碱处理的其他食品增菌液分别接种于CIN-1琼脂平板和改良Y琼脂平板， $26\mathrm{^\circ C}\pm 1\mathrm{^\circ C}$ 培养 $48\mathrm{h}\pm 2\mathrm{h}$ 。典型菌落在CIN-1上为深红色中心，周围具有无色透明圈(红色牛眼状菌落)，菌落大小为 $1\mathrm{mm}\sim 2\mathrm{mm}$ ，在改良Y琼脂平板上为无色透明、不黏稠的菌落。

# 5.4 改良克氏双糖试验

分别挑取5.3中的可疑菌落3个～5个，分别接种于改良克氏双糖铁琼脂，接种时先在斜面划线，再于底层穿刺， $26\,^{\circ}\mathrm{C} \pm 1\,^{\circ}\mathrm{C}$ 培养 $24\, \mathrm{h}$ ，将斜面和底部皆变黄且不产气的培养物做进一步的生化鉴定。

# 5.5 尿素酶试验和动力观察

用接种环挑取一满环5.4得到的可疑培养物，接种到尿素培养基中，接种量应足够大，振摇几秒钟， $26^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $2\mathrm{h} \sim 4\mathrm{h}$ 。将尿素酶试验阳性菌落分别接种于两管半固体培养基中，于 $26^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 和 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h}$ 。将在 $26^{\circ}\mathrm{C}$ 有动力而 $36^{\circ}\mathrm{C}$ 无动力的可疑菌培养物划线接种营养琼脂平板，进行纯化培养，用纯化物进行革兰氏染色镜检和生化试验。

# 5.6 革兰氏染色镜检

将纯化的可疑菌进行革兰染色。小肠结肠炎耶尔森氏菌呈革兰氏阴性球杆菌，有时呈椭圆或杆状，大小为 $(0.8\mu \mathrm{m}\sim 3.0\mu \mathrm{m})\times 0.8\mu \mathrm{m}$

# 5.8 血清型鉴定（选做项目）

除进行生化鉴定外，可选择做血清型鉴定。在洁净的载玻片上加一滴O因子血清，将待试培养物混入其内，使成为均一性混浊悬液，将玻片轻轻摇动 $0.5\mathrm{min}\sim 1\mathrm{min}$ ，在黑色背景下观察反应。如在 $2\mathrm{min}$ 内出现比较明显的小颗粒状凝集者，即为阳性反应，反之则为阴性，另用生理盐水作对照试验，以检查有无自凝现象；具体操作方法可按GB4789.4中沙门氏菌O因子血清分型方法进行。

# 6 结果与报告

综合以上及生化特征报告结果，报告 $25\mathrm{g}$ (或 $25~\mathrm{mL}$ )样品中检出或未检出小肠结肠炎耶尔森氏菌。

# A.6.2 制法

# A.6.3 试验方法

从琼脂斜面上挑取培养物接种，于 $26^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $18\mathrm{h} \sim 24\mathrm{h}$ ，观察结果。鸟氨酸脱羧酶阳性者由于产碱，培养基呈紫色。阴性者无碱性产物，但因葡萄糖产酸而使培养基变为黄色。对照管为黄色。

# A.8 缓冲葡萄糖蛋白胨水[甲基红(MR)和V-P试验用]

# A.8.1 成分

磷酸氢二钾 $5.0\mathrm{g}$

蒸馏水 $1000\mathrm{mL}$

# A.8.2 制法

溶化后校正 $\mathrm{pH}$ 至7.0，分装试管，每管 $1\mathrm{mL},121^{\circ}C$ 高压灭菌 $15\mathrm{min}$

# A.8.3 甲基红(MR)试验

自琼脂斜面挑取少量培养物接种本培养基中，于 $26\mathrm{^\circ C}\pm 1\mathrm{^\circ C}$ 培养 $2\mathrm{d}\sim 5\mathrm{d}$ ，哈夫尼亚菌则应在 $22\mathrm{C}\sim 25\mathrm{C}$ 培养。滴加甲基红试剂一滴，立即观察结果。鲜红色为阳性，黄色为阴性。甲基红试剂配法： $10\mathrm{mg}$ 甲基红溶于 $30~\mathrm{mL}~95\%$ 乙醇中，然后加入 $20~\mathrm{mL}$ 蒸馏水。

# A.8.4 V-P试验

用琼脂培养物接种本培养基中，于 $26^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $2\mathrm{d} \sim 4\mathrm{d}$ 。哈夫尼亚菌则应在 $22^{\circ}\mathrm{C} \sim 25^{\circ}\mathrm{C}$ 培养。加入 $6\% \alpha$ -萘酚-乙醇溶液 $0.5\mathrm{mL}$ 和 $40\%$ 氢氧化钾溶液 $0.2\mathrm{mL}$ ，充分振摇试管，观察结果。阳性反应立刻或于数分钟内出现红色，如为阴性，应放在 $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $4\mathrm{h}$ 再进行观察。

# A.10.2 制法

将A.10.1中成分于蒸馏水中溶解，校正 $\mathrm{pH}$ 至 $6.8\pm 0.2$ 。不要加热，过滤除菌，无菌分装于灭菌小试管中，每管约为 $3\mathrm{mL}$

# A.10.3 试验方法

挑取琼脂培养物接种在尿素培养基， $26^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h}$ 。尿素酶阳性者由于产碱而使培养基变为红色。

# 前言

本标准代替GB/T5009.75—2003《食品添加剂中铅的测定》。

本标准与GB/T5009.75—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂中铅的测定”；  
——增加了石墨炉原子吸收光谱法；  
——样品前处理增加了微波消解法。

# 食品安全国家标准

# 食品添加剂中铅的测定

# 1 范围

本标准规定了食品添加剂中铅的限量试验和定量试验方法。

本标准适用于食品添加剂中铅的限量试验和定量试验。

# 第一法 二苯基硫巴腙（双硫腙）比色法

# 2 原理

试样经处理加入柠檬酸氢二铵、氰化钾和盐酸羟胺等，消除铁、铜、锌等离子干扰，在 $\mathrm{pH}8.5\sim 9.0$ 时，铅离子与二苯基硫巴腙(双硫腙)生成红色络合物，用三氯甲烷提取，与标准系列比较做限量试验或定量试验。

# 3 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 硝酸 $\mathrm{(HNO_3)}$ 。  
3.1.2 硫酸 $\mathrm{(H_2SO_4)}$ 。  
3.1.3 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。  
3.1.4 盐酸（HCl）。  
3.1.5 酚红 $\mathrm{(C_{19}H_{14}O_5S)}$ 。  
3.1.6 三氯甲烷 $\mathrm{(CHCl_3)}$   
3.1.7 柠檬酸氢二铵 $\mathrm{(C_6H_{14}N_2O_7)}$ 。  
3.1.8 盐酸羟胺 $(\mathrm{NH}_2\mathrm{OH}\cdot \mathrm{HCl})$ 。  
3.1.9 氰化钾(KCN)。  
3.1.10 二苯基硫巴腙（双硫腙） $(\mathrm{C_{13}H_{12}N_4S})$ 。  
3.1.11 乙醇 $\mathrm{(C_2H_6O)}$ 。  
3.1.12 高氯酸 $\mathrm{(HClO_4)}$

# 3.2 试剂配制

3.2.1 硝酸溶液 $(1 + 1)$ ：取 $50~\mathrm{mL}$ 硝酸慢慢加入 $50~\mathrm{mL}$ 水中。  
3.2.2 氨溶液 $(1 + 1)$ ：取1份氨水与1份水混合。如含铅，应用全玻璃蒸馏器重蒸馏。  
3.2.3 酚红指示液 $(1\mathrm{g / L}$ 乙醇溶液）：称取 $100~\mathrm{mg}$ 酚红，加 $100~\mathrm{mL}$ 乙醇溶解（必要时过滤）。  
3.2.4 柠檬酸氢二铵溶液 $(500\mathrm{g / L})$ ：称取 $100\mathrm{g}$ 柠檬酸氢二铵，溶于 $100~\mathrm{mL}$ 水中，加两滴酚红指示液，

加氨溶液 $(1 + 1)$ 调节 $\mathrm{pH}$ 至 $8.5\sim 9.0$ （由黄变红，再多加两滴)，用双硫腙使用液提取数次，每次 $10~\mathrm{mL}\sim$ $20~\mathrm{mL}$ ，至三氯甲烷层至绿色且不变为止，弃去三氯甲烷层，再用三氯甲烷洗涤两次，每次 $5\mathrm{mL}$ ，弃去三氯甲烷层，加水稀释至 $200~\mathrm{mL}$ 。

3.2.5 氨溶液 $(1 + 99)$ ：取1份氯水与99份水混合。  
3.2.6 盐酸羟胺溶液 $(200\mathrm{g / L})$ ：称取 $20\mathrm{g}$ 盐酸羟胺，加 $40~\mathrm{mL}$ 水溶解，加两滴酚红指示液，加氨溶液 $(1 + 1)$ 调节 $\mathrm{pH}$ 至 $8.5\sim 9.0$ （由黄变红，再多加两滴)，用双硫腙三氯甲烷溶液提取数次，每次 $10~\mathrm{mL}\sim$ $20~\mathrm{mL}$ ，至三氯甲烷层绿色不变为止，再用三氯甲烷洗两次，每次 $5\mathrm{mL}$ ，弃去三氯甲烷层，水层加盐酸 $(1 + 1)$ 呈酸性，加水至 $100~\mathrm{mL}$ 。  
3.2.7 氰化钾溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 氰化钾，用水溶解并定容至 $100~\mathrm{mL}$ 。

# 警告：氰化钾(KCN)为危险化学品，应采取相应的防护措施。

3.2.8 双硫腙储备液 $(0.05\%$ 三氯甲烷溶液)：称取 $0.5\mathrm{g}$ 研细的双硫腙，溶于 $50~\mathrm{mL}$ 三氯甲烷中，如有残渣，可用滤纸过滤于 $250~\mathrm{mL}$ 分液漏斗中，用氨溶液 $(1 + 99)$ 提取3次，每次 $100~\mathrm{mL}$ ，将提取液用脱脂棉过滤至 $500~\mathrm{mL}$ 分液漏斗中，用盐酸溶液 $(1 + 1)$ 调至酸性，将沉淀出的双硫腙用 $200~\mathrm{mL}, 200~\mathrm{mL},$ $100~\mathrm{mL}$ 三氯甲烷分别提取3次，合并三氯甲烷层为双硫腙储备溶液。保存于冰箱中。  
3.2.9 双硫腙使用液：吸取 $1.0\mathrm{mL}$ 双硫腙储备溶液，加 $9.0\mathrm{mL}$ 三氯甲烷，混匀。用 $1\mathrm{cm}$ 比色杯，以三氯甲烷调节零点，于波长 $510\mathrm{nm}$ 处测吸光度 $(A)$ ，用式(1)算出配制 $100\mathrm{mL}$ 双硫腙使用液（ $70\%$ 透光率）所需双硫腙储备溶液的毫升数 $(V)$ ：

$$
V = \frac {1 0 \times (2 - \lg 7 0)}{A} \tag {1}
$$

式中：

V ——双硫腙储备溶液的用量，单位为毫升 $(\mathrm{mL})$ ；

lg70——双硫腙使用液（ $70\%$ 透光率）；

$A$ ——吸光度值。

3.2.10 硝酸溶液 $(1\%)$ ：取 $1\mathrm{mL}$ 硝酸，加水稀释至 $100~\mathrm{mL}$

# 3.3 标准品

高纯硝酸铅 $\left[\mathrm{Pb}\left(\mathrm{NO}_{3}\right)_{2}\right]$ ：纯度为 $99.99\%$ 或经国家认证并授予标准物质证书的标准物质。

# 3.4 铅标准储备溶液配制

铅标准储备溶液 $(1\mathrm{mg / mL})$ ：精密称取 $0.1598\mathrm{g}$ 硝酸铅标准品，加 $10~\mathrm{mL}~1\%$ 硝酸溶液，溶解后定量移入 $100~\mathrm{mL}$ 容量瓶中，加水稀释至刻度。

# 3.5 铅标准使用溶液配制

铅标准使用溶液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：吸取铅标准储备溶液 $1.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加水稀释至刻度。

# 4 仪器和设备

注：所用玻璃仪器均需以硝酸溶液 $(1 + 4)$ 浸泡 $24\mathrm{h}$ 以上，用水反复冲洗，最后用去离子水冲洗干净。

4.1 分光光度计。  
4.2 $125~\mathrm{mL}$ 分液漏斗。  
4.3 $250~\mathrm{mL}$ 凯氏烧瓶或 $250~\mathrm{mL}$ 锥形瓶。  
4.4 电子天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$

# 5 样品处理

# 5.1 无机试样的制备

无机试样的“试样处理”按照相应标准方法进行测定。

# 5.2 有机式样的制备

有机试样的“试样处理”除已有相应标准的，按照相应标准方法进行测定外，一般按下述方法进行：a）湿法消解：称取 $5.000\mathrm{g}$ 试样，置于 $250~\mathrm{mL}$ 锥形瓶中，加 $10~\mathrm{mL}$ 硝酸，放置片刻(或过夜)后，加热，待反应缓和后，取下冷却，沿瓶壁加入 $5\mathrm{mL}$ 硫酸，再继续加热，至瓶中溶液开始变成棕色，不断滴加硝酸（如有必要可滴加些高氯酸)，至有机质分解完全，继续加热，至生成大量的二氧化硫白色烟雾，最后溶液应呈无色或微黄色。冷却后加 $20~\mathrm{mL}$ 水煮沸，除去残余的硝酸至产生白烟为止。如此处理两次，放冷，将溶液移入 $50~\mathrm{mL}$ 容量瓶中，用少量水分次洗涤锥形瓶2次～3次，将洗液一并移入容量瓶中，加水至刻度，混匀备用。取相同量的硝酸、硫酸，同时做试剂空白试验。  
b）干灰化法：本法用于不适合湿法消解的试样。称取 $5.000\mathrm{g}$ 试样于瓷坩埚中，加入适量硫酸湿润试样，小心炭化后，加 $2\mathrm{mL}$ 硝酸和5滴硫酸，小心加热直到白色烟雾挥尽，移入高温炉中，于 $500^{\circ}\mathrm{C}$ 灰化完全。冷却后取出。加 $1\mathrm{mL}$ 硝酸溶液 $(1 + 1)$ ，加热使灰分溶解，将试样液转移到 $50~\mathrm{mL}$ 容量瓶中（必要时过滤），并用少量水洗涤坩埚，洗液一并移入容量瓶中，加水至刻度，混匀备用。取一坩埚，同时做试剂空白试验。

# 6 测定

# 6.1 限量试验

吸取适量试样液及铅的标准使用溶液（含铅量不低于 $5\mu \mathrm{g}$ ），分别置于 $125~\mathrm{mL}$ 分液漏斗中，各加硝酸溶液 $(1\%)$ 至 $20~\mathrm{mL}$ 。

向试样液及铅的标准使用溶液 $(1\mathrm{mg / mL})$ 中各加入 $1\mathrm{mL}$ 柠檬酸氢二铵溶液 $(500~\mathrm{g / L})$ 、 $1\mathrm{mL}$ 盐酸羟胺溶液 $(200~\mathrm{g / L})$ 和两滴酚红指示液，用氨溶液 $(1 + 1)$ 调至红色，再各加 $2\mathrm{mL}$ 氰化钾溶液 $(100~\mathrm{g / L})$ ，混匀后，加入 $5.0~\mathrm{mL}$ 双硫腙使用液，剧烈振摇 $1\mathrm{min}$ ，静置分层后，三氯甲烷层经脱脂棉滤入 $1\mathrm{cm}$ 比色杯中，于波长 $510~\mathrm{nm}$ 处，以三氯甲烷调节零点，测定吸光度或进行目视比色，试样液的吸光度或色度不应大于铅的标准液的吸光度或色度。

若试样经处理，则铅限量标准也应同法处理。

# 6.2 定量测定

吸取 $10.0\mathrm{mL}$ （或适量）试样液和同量的试剂空白液，分别置于 $125~\mathrm{mL}$ 分液漏斗中，各加 $1\%$ 硝酸溶液至 $20~\mathrm{mL}$ 。

吸取铅标准使用溶液 $0.0\mathrm{mL}, 0.1\mathrm{mL}, 0.3\mathrm{mL}, 0.5\mathrm{mL}, 0.7\mathrm{mL}, 1.0\mathrm{mL}$ （分别相当于 $0\mu \mathrm{g}, 1\mu \mathrm{g}, 3\mu \mathrm{g}, 5\mu \mathrm{g}, 7\mu \mathrm{g}, 10\mu \mathrm{g}$ 铅），分别置于 $125\mathrm{mL}$ 的分液漏斗中，各加硝酸溶液（ $1\%$ ）至 $20\mathrm{mL}$ 。向试样液、试剂空白液及铅标准溶液中各加入 $1\mathrm{mL}$ 柠檬酸氢二铵溶液 $(500\mathrm{g / L})$ ， $1\mathrm{mL}$ 盐酸羟胺溶液 $(200\mathrm{g / L})$ 和两滴酚红指示液，用氨溶液 $(1 + 1)$ 调至红色，再各加入 $2\mathrm{mL}$ 氰化钾溶液 $(100\mathrm{g / L})$ ，混匀，各加 $5.0\mathrm{mL}$ 双硫腙使用液，剧烈振摇 $1\mathrm{min}$ ，静置分层后，三氯甲烷经脱脂棉滤入 $1\mathrm{cm}$ 比色杯中，于波长 $510\mathrm{nm}$ 处，以零管调节零点，测定吸光度，绘制标准曲线。

# 7 分析结果的表述

试样中铅含量按式(2)计算：

$$
c = \frac {\left(m _ {1} - m _ {2}\right) \times 1 0 0 0}{m \times \frac {V _ {2}}{V _ {1}} \times 1 0 0 0} \tag {2}
$$

式中：

$c$ ——试样中铅的含量，单位为毫克每千克(或毫克每升)[ $\mathrm{mg/kg}$ (或 $\mathrm{mg/L}$ ]；

$m_{1}$ ——试样液中铅的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_{2}$ ——试剂空白液中铅的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m$ ——试样质量（体积），单位为克（或毫升） $\left[\mathrm{g}(\mathrm{mL})\right]$

$V_{2}$ ——测定时所取试样液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——试样处理后定容体积，单位为毫升 $(\mathrm{mL})$

1000——换算系数。

# 8 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 9 其他

本方法检出限为 $0.25\mathrm{mg / kg}$

# 第二法 石墨炉原子吸收光谱法

# 10 原理

试样经灰化或酸消解后，注入原子吸收分光光度计石墨炉中，电热原子化后吸收 $283.3\mathrm{nm}$ 共振线，在一定浓度范围，其吸收值与铅含量成正比，与标准系列比较定量。

# 11 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 11.1 试剂

11.1.1 硝酸 $\mathrm{(HNO_3)}$ ：优级纯。  
11.1.2 $30\%$ 过氧化氢 $\mathrm{(H_2O_2)}$   
11.1.3 高氯酸 $\mathrm{(HClO_4)}$ ：优级纯。  
11.1.4 磷酸二氢铵 $(\mathrm{NH_4H_2PO_4})$ 。

# 11.2 试剂配制

11.2.1 硝酸溶液 $(1 + 1)$ ：取 $50~\mathrm{mL}$ 硝酸慢慢加入 $50~\mathrm{mL}$ 水中。

11.2.2 硝酸溶液 $(0.5\mathrm{mol} / \mathrm{L})$ ：取 $3.2\mathrm{mL}$ 硝酸加入 $50~\mathrm{mL}$ 水中，稀释至 $100~\mathrm{mL}$ 。  
11.2.3 硝酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：取 $6.4~\mathrm{mL}$ 硝酸加入 $50~\mathrm{mL}$ 水中，稀释至 $100~\mathrm{mL}$ 。  
11.2.4 磷酸二氢铵溶液 $(20\mathrm{g / L})$ ：称取 $2.0\mathrm{g}$ 磷酸二氢铵，以水溶解稀释至 $100~\mathrm{mL}$ 。  
11.2.5 混合酸：硝酸 $+$ 高氯酸 $(9 + 1)$ 。取9份硝酸与1份高氯酸混合。

# 11.3 标准品

高纯金属铅(Pb)：纯度为 $99.99\%$ 或经国家认证并授予标准物质证书的标准物质。

# 11.4 标准溶液配制

11.4.1 铅标准储备液 $(1.0\mathrm{mg / mL})$ ：精确称取 $1.0000\mathrm{g}$ 金属铅 $(99.99\%)$ ，分次加少量硝酸溶液 $(1 + 1)$ ，加热溶解，总量不超过 $37~\mathrm{mL}$ ，移入 $1000~\mathrm{mL}$ 容量瓶，加水至刻度，混匀。  
11.4.2 铅标准使用液：每次吸取铅标准储备液 $1.0 \mathrm{~mL}$ 于 $100 \mathrm{~mL}$ 容量瓶中，加硝酸 $(0.5 \mathrm{~mol} / \mathrm{L})$ 或硝酸 $(1 \mathrm{~mol} / \mathrm{L})$ 至刻度。经多次稀释成每毫升含 $10.0 \mathrm{ng}, 20.0 \mathrm{ng}, 40.0 \mathrm{ng}, 60.0 \mathrm{ng}, 80.0 \mathrm{ng}$ 铅的标准使用液。

# 12 仪器和设备

注：所用玻璃仪器均需以硝酸溶液 $(1 + 4)$ 浸泡 $24\mathrm{h}$ 以上，用水反复冲洗，最后用去离子水冲洗干净。

12.1 原子吸收分光光度计（附石墨炉及铅空心阴极灯）。  
12.2 马弗炉。  
12.3 干燥恒温箱。  
12.4 瓷坩埚。  
12.5 压力消解器、压力消解罐或压力溶弹。  
12.6 可调式电热板、可调式电炉。  
12.7 电子天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
12.8 微波消解仪、微波消解罐。

# 13 分析步骤

# 13.1 试样制备

13.1.1 压力消解罐消解法：称取试样 $1.000\mathrm{g}\sim 2.000\mathrm{g}$ ，（按压力消解罐使用说明书称取试样）于聚四氟乙烯内罐，加硝酸 $2\mathrm{mL}\sim 4\mathrm{mL}$ 浸泡过夜。再加 $30\%$ 过氧化氢 $2\mathrm{mL}\sim 3\mathrm{mL}$ （总量不能超过罐容积的 $1 / 3)$ 。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱， $120^{\circ}C\sim 140^{\circ}C$ 保持 $3\mathrm{h}\sim 4\mathrm{h}$ ，在箱内自然冷却至室温，将消化液洗入或过滤入(视消化后试样的盐分而定) $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 容量瓶中，用水少量多次洗涤罐，洗涤液合并于容量瓶中并定容至刻度，混匀备用；同时做试剂空白试验。  
13.1.2 微波消解：称取 $0.500\mathrm{g}\sim 2.000\mathrm{g}$ 试样于消解罐中，加 $10.0~\mathrm{mL}$ 硝酸、 $2.0~\mathrm{mL}$ 过氧化氢 $(30\%)$ ，混合均匀，于微波消化仪中消化，消化推荐条件见附录A。消解完成后，自然冷却至室温，开盖赶酸，将消化液洗入或过滤入（视消化后试样的盐分而定） $10~\mathrm{mL}\sim 25~\mathrm{mL}$ 容量瓶中，用水少量多次洗涤罐，洗液合并于容量瓶中并定容至刻度，混匀备用；同时做试剂空白试验。  
13.1.3 干法灰化：称取试样 $1.000\mathrm{g}\sim 5.000\mathrm{g}$ （根据铅含量而定)于瓷坩埚中，先小火在可调式电热板上炭化至无烟，移入马弗炉 $500^{\circ}C$ 灰化 $6\mathrm{h}\sim 8\mathrm{h}$ 时，冷却。若个别试样灰化不彻底，则加 $1\mathrm{mL}$ 混合酸在可调式电炉上小火加热，反复多次直到消化完全，取下放冷，用硝酸溶液 $(0.5\mathrm{mol / L})$ 将灰分溶解，将试样消化液洗入或过滤入(视消化后试样的盐分而定) $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 容量瓶中，用水少量多次洗涤瓷

坩埚，洗液合并于容量瓶中并定容至刻度，混匀备用；同时做试剂空白试验。

13.1.4 湿式消解法：称取试样 $1.000\mathrm{g}\sim 5.000\mathrm{g}$ 于锥形瓶或高脚烧杯中，放数粒玻璃珠，加 $10~\mathrm{mL}$ 混合酸(或再加 $1\mathrm{mL}\sim 2\mathrm{mL}$ 硝酸)，加盖浸泡过夜，加一小漏斗电炉上消解，若变棕黑色，再加混合酸直至冒白烟，消化液呈无色透明或略带黄色，取下放冷，将消化液洗入或过滤入（视消化后试样的盐分而定） $10~\mathrm{mL}$ 或 $25~\mathrm{mL}$ 容量瓶中，用水少量多次洗涤，洗涤液合并于容量瓶中，定容至刻度，混匀备用；同时做试剂空白试验。

# 13.2 测定

13.2.1 仪器条件：根据各自仪器性能调至最佳状态。参考条件：波长 $283.3\mathrm{nm}$ ，狭缝 $0.7\mathrm{nm}$ ，灯电流 $8\mathrm{mA}$ ，干燥温度 $120^{\circ}\mathrm{C}, 20\mathrm{s}$ ；灰化温度 $800^{\circ}\mathrm{C}$ ，持续 $15\mathrm{s} \sim 20\mathrm{s}$ ；原子化温度 $1600^{\circ}\mathrm{C} \sim 1800^{\circ}\mathrm{C}$ ，持续 $4\mathrm{s} \sim 5\mathrm{s}$ ；背景校正为氘灯或塞曼效应。  
13.2.2 标准曲线绘制：吸取11.4.2配制的铅标准使用液 $10.0\mathrm{ng / mL},20.0\mathrm{ng / mL},40.0\mathrm{ng / mL},$ $60.0~\mathrm{ng / mL},80.0~\mathrm{ng / mL}$ (或 $\mu \mathrm{g} / \mathrm{L})$ 各 $10~\mu \mathrm{L}$ ，注入石墨炉，测得其吸光值并求得吸光值与浓度关系的一元线性回归方程。  
13.2.3 试样测定：分别吸取样液和试剂空白液各 $10\mu \mathrm{L}$ ，注入石墨炉，测得其吸光值，代入标准系列的一元线性回归方程中求得样液中铅含量。  
13.2.4 基体改进剂的使用：为了减少背景干扰需注入适量的基体改进剂磷酸二氢铵溶液 $(20\mathrm{g / L})$ 一般为 $5\mu \mathrm{L}$ 或与试样同量，绘制铅标准曲线时也要加入与试样测定时等量的基体改进剂磷酸二氢铵溶液。测定样品时注意背景校正器的校正能力，如不能消除背景吸收的干扰，则考虑使用化学法进行测定。

# 14 分析结果的表述

试样中铅含量按式(3)计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {3}
$$

式中：

$X$ ——试样中铅含量，单位为微克每千克(或微克每升) $[\mu \mathrm{g} / \mathrm{kg}$ 或 $\mu \mathrm{g} / \mathrm{L})]$

$c$ ——试样消化液测定浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{0}$ ——试样空白消化液测定浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样消化液总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量(体积)，单位为克或(毫升)[g(mL)]；

1000 ——换算系数。

计算结果表示保留两位有效数字。

2025-03-16 发布

2025-09-16 实施

# 前言

本标准代替GB5009.120—2016《食品安全国家标准 食品中丙酸钠、丙酸钙的测定》。

本标准与GB5009.120—2016相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中丙酸及其盐的测定”；  
扩大了方法的适用范围；  
——删除了面包样品制备中烘干步骤；  
——增加了“第一法 高效液相色谱法”中浸提法的净化步骤及计算公式；  
——补充了浸提法的检出限和定量限；  
——修改了气相色谱法标准曲线的制作过程。

# 食品安全国家标准

# 食品中丙酸及其盐的测定

# 1 范围

本标准规定了食品中丙酸及其盐的液相色谱和气相色谱测定方法。

本标准适用于粮食和粮食制品、焙烤食品、油炸食品、肉制品、调味品、豆类制品、罐头中丙酸及其盐的测定。

# 第一法 高效液相色谱法

# 2 原理

试样中的丙酸盐通过酸化转化为丙酸。经水蒸气蒸馏，收集蒸馏液并调 $\mathrm{pH}$ 后定容，用高效液相色谱仪测定；或用超声波提取后，提取液调 $\mathrm{pH}$ 后定容，经固相萃取柱净化，用高效液相色谱仪测定，根据色谱峰的保留时间定性，外标法定量检测丙酸的含量。样品中的丙酸及其盐以丙酸计。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。  
3.1.2 磷酸氢二铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{HPO}_{4}\right]$ 。  
3.1.3 硅油。  
3.1.4 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

# 3.2 试剂配制

3.2.1 磷酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：在 $50~\mathrm{mL}$ 水中加入 $53.5\mathrm{mL}$ 磷酸，混匀后，加水定容至 $1000~\mathrm{mL}$ 。  
3.2.2 磷酸氢二铵溶液 $(1.5\mathrm{g} / \mathrm{L})$ ：称取磷酸氢二铵 $1.5\mathrm{g}$ ，加水溶解并定容至 $1000~\mathrm{mL}$ 。  
3.2.3 甲醇-水溶液 $(10 + 90)$ ：将甲醇和水按 $10:90$ 的体积比混合均匀。  
3.2.4 甲醇-水溶液 $(40 + 60)$ ：将甲醇和水按 $40:60$ 的体积比混合均匀。

# 3.3 标准品

丙酸标准品 $\mathrm{(C_3H_6O_2)}$ ，CAS号：79-09-4，纯度 $\geq 99.0\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液的配制

3.4.1 丙酸标准储备液 $(10.0\mathrm{mg / mL})$ ：称取 $250.0\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )丙酸标准品，用水溶解并转移至

$25~\mathrm{mL}$ 容量瓶中，用水定容至刻度， $4^{\circ}C$ 冰箱中保存，有效期为6个月。

3.4.2 丙酸标准中间液 $(1.00\mathrm{mg / mL})$ : 准确吸取 $5.00~\mathrm{mL}$ 标准储备液于 $50~\mathrm{mL}$ 容量瓶中, 用水稀释至刻度, $4^{\circ}\mathrm{C}$ 冰箱中保存, 有效期为 3 个月。

# 3.5 材料

3.5.1固相萃取柱： $500\mathrm{mg / 6mL}$ ，聚苯乙烯/二乙烯苯亲水亲脂平衡型填料或等效柱。  
3.5.2 水相微孔滤膜和有机微孔滤膜： $0.45 \mu \mathrm{m}$ 。

# 4 仪器和设备

4.1 高效液相色谱仪：配有紫外检测器或二极管阵列检测器。  
4.2 天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$   
4.3 超声波发生器。  
4.4 离心机：转速不低于 $4000\mathrm{r / min}$   
4.5 组织捣碎机。  
4.6 水蒸气蒸馏装置： $500~\mathrm{mL}$ 。  
4.7 pH计：精度为0.01。  
4.8 固相萃取装置。

# 5 分析步骤

# 5.1 样品前处理

# 5.1.1 样品制备

固体样品和半固体样品经组织捣碎机捣碎混匀后备用；液体样品摇匀后备用。

# 5.1.2 试样处理

# 5.1.2.1 蒸馏法

准确称取 $25\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样, 置于 $500\mathrm{mL}$ 蒸馏瓶中, 加入 $100\mathrm{mL}$ 水, 再用 $50\mathrm{mL}$ 水冲洗容器, 转移到蒸馏瓶中, 加 $1\mathrm{mol/L}$ 磷酸溶液 $20\mathrm{mL}, 2$ 滴 $\sim 3$ 滴硅油, 进行水蒸气蒸馏, 将 $250\mathrm{mL}$ 容量瓶置于冰浴中作为吸收液装置, 待蒸馏至约 $240\mathrm{mL}$ 时取出, 在室温下放置 $30\mathrm{min}$ , 用 $1\mathrm{mol/L}$ 磷酸溶液调 $\mathrm{pH}$ 为 3 左右, 加水定容至刻度, 摇匀, 经 $0.45\mu\mathrm{m}$ 水相微孔滤膜过滤后, 待液相色谱测定。

# 5.1.2.2 浸提法（适用于面包、糕点）

准确称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样至于 $100\mathrm{mL}$ 烧杯中，加水约 $25\mathrm{mL}$ ，加入 $1\mathrm{mol/L}$ 磷酸溶液 $0.50\mathrm{mL}$ ，混匀，经超声浸提 $10\mathrm{min}$ 后，用 $1\mathrm{mol/L}$ 磷酸溶液调 $\mathrm{pH}$ 为3左右。转移全部试样至 $50\mathrm{mL}$ 容量瓶中，用水定容至刻度，摇匀。将试样转移至 $50\mathrm{mL}$ 离心管中，以不低于 $4000\mathrm{r/min}$ 离心 $10\mathrm{min}$ ，待净化。

固相萃取柱依次用 $5\mathrm{mL}$ 甲醇和 $5\mathrm{mL}$ 水活化，保持柱体湿润。立即取 $5.0\mathrm{mL}$ 待净化液加载至固相萃取柱上。用 $5.0\mathrm{mL}$ 甲醇-水溶液 $(10 + 90)$ 淋洗，真空抽至柱体近干。再加入 $5.0\mathrm{mL}$ 甲醇-水溶液 $(40 + 60)$ 洗脱，收集洗脱液并定容至 $5.0\mathrm{mL}$ ，混匀，经 $0.45\mu \mathrm{m}$ 有机微孔滤膜过滤后，待液相色谱测定。

# 5.2 仪器参考条件

5.2.1 色谱柱：耐水型 $\mathrm{C_{18}}$ 柱， $4.6\mathrm{mm}\times 250\mathrm{mm},5\mu \mathrm{m}$ 或同等性能的色谱柱。

5.2.2 流动相： $1.5\mathrm{g / L}$ 磷酸氢二铵溶液，用 $1\mathrm{mol / L}$ 磷酸溶液调 $\mathrm{pH}$ 为 $2.7\sim 3.5$   
5.2.3 流速： $1.0~\mathrm{mL / min}$   
5.2.4 柱温： $25^{\circ}C$ 。  
5.2.5 进样量： $20~\mu \mathrm{L}$   
5.2.6 波长： $214~\mathrm{nm}$

# 5.3 标准工作曲线的制作

# 5.3.1 蒸馏法

准确吸取标准储备液 $0.250\mathrm{mL}, 0.500\mathrm{mL}, 1.00\mathrm{mL}, 2.50\mathrm{mL}, 5.00\mathrm{mL}, 12.5\mathrm{mL}$ 置于 $500\mathrm{mL}$ 蒸馏瓶中，其他操作同5.1.2.1样品前处理，其丙酸标准溶液的最终浓度分别为 $0.010\mathrm{mg/mL}$ 、 $0.020\mathrm{mg/mL}$ 、 $0.040\mathrm{mg/mL}$ 、 $0.100\mathrm{mg/mL}$ 、 $0.200\mathrm{mg/mL}$ 、 $0.500\mathrm{mg/mL}$ ，经 $0.45\mu\mathrm{m}$ 水相微孔滤膜过滤，浓度由低到高进样，以浓度为横坐标，以峰面积为纵坐标，绘制标准曲线，丙酸标准溶液的色谱图参见附录A图A.1。

# 5.3.2 浸提法

准确吸取标准中间液 $0.100\mathrm{mL}, 0.200\mathrm{mL}, 0.400\mathrm{mL}, 1.00\mathrm{mL}, 2.00\mathrm{mL}, 5.00\mathrm{mL}$ 至 $10\mathrm{mL}$ 容量瓶中，分别加入 $1\mathrm{mol} / \mathrm{L}$ 磷酸 $0.20\mathrm{mL}$ ，用水定容至 $10\mathrm{mL}$ ，混匀。其丙酸标准溶液的最终浓度分别为 $0.0100\mathrm{mg} / \mathrm{mL}, 0.0200\mathrm{mg} / \mathrm{mL}, 0.0400\mathrm{mg} / \mathrm{mL}, 0.100\mathrm{mg} / \mathrm{mL}, 0.200\mathrm{mg} / \mathrm{mL}, 0.500\mathrm{mg} / \mathrm{mL}$ ，经 $0.45\mu \mathrm{m}$ 水相微孔滤膜过滤，浓度由低到高进样，以浓度为横坐标，以峰面积为纵坐标，绘制标准曲线，丙酸标准溶液的色谱图参见附录A中图A.2。

# 5.4 试样溶液的测定

处理后的样液同标准系列同样注入液相色谱仪测试，根据标准曲线计算样品中的丙酸浓度。待测样液中丙酸响应值应在标准曲线线性范围内，超出浓度线性范围则应稀释后再进样分析。

# 6 分析结果的表述

6.1 蒸馏法样品中丙酸及其盐含量(以丙酸计)按式(1)进行计算。

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0} \times f \quad \dots \tag {1}
$$

式中：

$X$ ——样品中丙酸及其盐含量(以丙酸计)，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho$ ——由标准曲线得出的样液中丙酸的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$m$ ——样品质量，单位为克 $(\mathrm{g})$ ；

$f$ ——稀释倍数。

6.2 浸提法样品中丙酸及其盐含量(以丙酸计)按式(2)进行计算。

$$
X = \frac {\rho \times V _ {1} \times V _ {2} \times 1 0 0 0}{V _ {3} \times m \times 1 0 0 0} \times f \tag {2}
$$

式中：

$X$ ——样品中丙酸及其盐含量(以丙酸计)，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho$ ——由标准曲线得出的样液中丙酸的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V_{1}$ ——试样提取液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试样净化后定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$V_{3}$ ——用于净化试样的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品质量，单位为克 $(\mathrm{g})$ ；

$f$ ——稀释倍数。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

蒸馏法取样为 $25\mathrm{g}$ ，定容体积为 $250~\mathrm{mL}$ 时，丙酸的检出限为 $0.03\mathrm{g / kg}$ ，定量限为 $0.10\mathrm{g / kg}$ ；浸提法取样为 $5\mathrm{g}$ 时，丙酸的检出限为 $0.03\mathrm{g / kg}$ ，定量限为 $0.10\mathrm{g / kg}$ 。

# 第二法 气相色谱法

# 9 原理

试样中的丙酸盐通过酸化转化为丙酸，经水蒸气蒸馏，收集蒸馏液，用气相色谱氢火焰离子化检测器检测，根据色谱峰的保留时间定性，外标法定量检测丙酸的含量。样品中的丙酸及其盐以丙酸计。

# 10 试剂和材料

除非另有说明，所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 10.1 试剂

10.1.1 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。  
10.1.2 甲酸 $(\mathrm{CH}_2\mathrm{O}_2)$ 。  
10.1.3 硅油。

# 10.2 试剂配制

10.2.1 磷酸溶液 $(10 + 90)$ ：将磷酸和水按 $10:90$ 的体积比混合均匀。  
10.2.2 甲酸溶液 $(2 + 98)$ ：将甲酸和水按 $2:98$ 的体积比混合均匀。

# 10.3 标准品

丙酸标准品 $\mathrm{(C_3H_6O_2)}$ ，CAS号：79-09-4，纯度 $\geqslant 99.0\%$ ，或经国家认证并授予标准物质证书的标准品。

# 10.4 标准溶液配制

丙酸标准储备液 $(10.0\mathrm{mg / mL})$ ：精确称取 $250~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ）丙酸标准品，加水溶解并转移

至于 $25~\mathrm{mL}$ 容量瓶中，用水定容至刻度， $4^{\circ}\mathrm{C}$ 冰箱中保存，有效期为6个月。

# 11 仪器和设备

11.1 气相色谱仪：配有氢火焰离子化检测器。   
11.2 天平：感量为 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
11.3 水蒸气蒸馏装置： $500~\mathrm{mL}$ 。  
11.4 组织捣碎机。

# 12.1.2 试样处理

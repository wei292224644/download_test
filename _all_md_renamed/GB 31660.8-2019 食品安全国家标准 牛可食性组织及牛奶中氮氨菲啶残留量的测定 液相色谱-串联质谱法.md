# 食品安全国家标准

# 牛可食性组织中氮氨菲啶残留量的测定 液相色谱-串联质谱法

# 1 范围

本标准规定了牛肌肉、脂肪、肝脏、肾脏及牛奶中氮氨菲啶残留量检测的制样和液相色谱-串联质谱测定方法。

本标准适用于牛肌肉、脂肪、肝脏和肾脏及牛奶中氮氨菲啶残留量的检测。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试料中残留的氮氨菲啶，用乙腈、甲酸铵-甲醇溶液提取，正己烷脱脂，液相色谱-串联质谱测定，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 甲醇（ $\mathrm{CH}_3\mathrm{OH}$ ）：色谱纯。  
4.1.2 乙腈（ $\mathrm{CH}_3\mathrm{CN}$ ）：色谱纯。  
4.1.3 甲酸（HCOOH）：色谱纯。  
4.1.4 正己烷（ $\mathrm{C_6H_{14}}$ ）。  
4.1.5 无水硫酸钠（ $\mathrm{Na}_2\mathrm{SO}_4$ ）。  
4.1.6 甲酸铵（ $\mathrm{HCOONH_4}$ ）。

# 4.2 溶液配制

4.2.1 $0.25\mathrm{mol} / \mathrm{L}$ 甲酸铵-甲醇溶液：取甲酸铵 $15.8\mathrm{g}$ ，用甲醇溶解并稀释至 $1000\mathrm{mL}$ 。  
4.2.2 $0.1\%$ 甲酸溶液：取甲酸 $1.0 \mathrm{~mL}$ ，用水溶解并稀释至 $1000 \mathrm{~mL}$ 。  
4.2.3 $80\%$ 甲醇溶液：取甲醇 $400\mathrm{mL}$ ，用水溶解并稀释至 $500\mathrm{mL}$ 。  
4.2.4 提取液：取乙腈和 $0.25\mathrm{mol} / \mathrm{L}$ 甲酸铵-甲醇溶液按1：1（体积比）混匀。

# 4.3 标准品

盐酸氮氨菲啶（Isometamidium Chloride, $\mathrm{C_{28}H_{26}ClN_7}$ CAS号:34301-55-8），含量 $\geq 95.0\%$

# 4.4 标准溶液的制备

4.4.1 标准贮备液：取氮氨菲啶标准品 $10\mathrm{mg}$ ，精密称定，于 $10~\mathrm{mL}$ 棕色量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $1\mathrm{mg / mL}$ 的标准贮备液。 $-20^{\circ}C$ 以下保存。  
4.4.2 $10 \mu \mathrm{g} / \mathrm{mL}$ 氮氨菲啶标准工作液：精密量取标准贮备液 $100 \mu \mathrm{L}$ ，于 $10 \mathrm{~mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ 的氮氨菲啶标准工作液。-20℃以下保存。  
4.4.3 $100 \mathrm{ng} / \mathrm{mL}$ 氮氨菲啶标准工作液：精密量取 $10 \mu \mathrm{g} / \mathrm{L}$ 的氮氨菲啶标准工作液 $1 \mathrm{~mL}$ ，于 $100 \mathrm{~mL}$ 量瓶中，用 $80 \%$ 甲醇溶解并稀释至刻度，配制成浓度为 $100 \mathrm{ng} / \mathrm{mL}$ 的氮氨菲啶标准工作液。 $2 \sim 8^{\circ} \mathrm{C}$ 保存。

# 4.5 材料

滤膜： $0.22\mu \mathrm{m}$

# 5 仪器和设备

5.1 液相色谱-串联质谱仪：配电喷雾离子源。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
5.3 淚涡振荡器。  
5.4 振荡器。  
5.5 组织匀浆机。  
5.6 冷冻离心机。  
5.7 旋转蒸发仪。  
5.8 鸡心瓶： $50~\mathrm{mL}$   
5.9 离心管： $50~\mathrm{mL}$

# 6 试料的制备与保存

# 6.1 试料的制备

# 6.1.1 牛奶

取适量新鲜或冷藏的空白或供试牛奶，混合均匀。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

# 6.1.2 肌肉、脂肪、肝脏和肾脏组织

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。

——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

# 6.2 试料的保存

-18°C以下保存。

# 7 测定步骤

# 7.1 提取

称取试料 $5\mathrm{g}$ （准确至 $\pm 20\mathrm{mg}$ ），于 $50~\mathrm{mL}$ 离心管中，加无水硫酸钠 $2\mathrm{g}$ ，再加提取液 $10~\mathrm{mL}$ ，涡旋混匀，振荡 $5\mathrm{min}$ ，6000 r/min离心 $10\mathrm{min}$ ，取上层液于鸡心瓶中。残渣中加提取液 $10~\mathrm{mL}$ 重复提取一次，合并两次提取液于鸡心瓶中，于 $45^{\circ}C$ 水浴旋转蒸发至干。备用。

# 7.2 净化

加 $80\%$ 甲醇溶液 $2 \mathrm{~mL}$ 于鸡心瓶中，充分涡旋溶解残余物，加正己烷 $2 \mathrm{~mL}$ 振荡混合 $1 \mathrm{~min}$ ，转移至 $10 \mathrm{~mL}$ 离心管中， $6000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，弃上层正己烷液。再加正己烷 $2 \mathrm{~mL}$ ，重复提取一次。取下层清液，滤过，供液相色谱-串联质谱测定。

# 7.3 基质匹配标准曲线的制备

精密量取 $100 \mathrm{ng} / \mathrm{mL}$ 氮氨菲啶标准工作溶液或 $10 \mu \mathrm{g} / \mathrm{mL}$ 氮氨菲啶标准工作溶液适量，于经提取、蒸干后的空白试料残余物中，用适量 $80 \%$ 甲醇溶液溶解并稀释至 $2.0 \mathrm{~mL}$ ，使氮氨菲啶浓度为20、50、100、200、500、 $1000 \mathrm{ng} / \mathrm{mL}$ ，滤过，制成系列基质匹配标准工作溶液，供液相色谱-串联质谱仪测定。以特征离子峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.4 测定

# 7.4.1 液相色谱条件

a）色谱柱： $\mathrm{BEHC}_{18}(100\mathrm{mm}\times 2.1\mathrm{mm},1.7\mu \mathrm{m})$ ，或相当者；  
b）柱温： $30^{\circ}\mathrm{C}$   
c）流速： $0.2\mathrm{mL / min}$   
d）进样量： $10~\mu \mathrm{L}$   
e）流动相： $\mathrm{A} + \mathrm{B} = 50 + 50$ ，其中A相为乙腈（含 $0.1\%$ 甲酸），B相为水（含 $0.1\%$ 甲酸）。

# 7.4.2 质谱条件

a）电离模式：ESI;  
b）扫描方式：正离子扫描；  
c）检测方式：多反应检测；

d）电离电压： $2.8\mathrm{kV}$   
e）源温： $110^{\circ}\mathrm{C}$   
f）雾化温度： $350^{\circ}\mathrm{C}$   
g）锥孔气流速： $50\mathrm{L / h}$   
h）雾化气流速： $550\mathrm{L / h}$   
i）驻留时间：0.3s；  
j）定性、定量离子及对应的锥孔电压和碰撞电压见表1。

取试样溶液和相应的基质匹配标准溶液，作单点或多点校准，按外标法计算。基质匹配标准溶液及试样溶液中氮氨菲啶的响应值均应在仪器检测的线性范围之内。试样溶液中的离子相对丰度与基质匹配标准溶液中的离子相对丰度相比，符合表2的规定。在上述色谱-质谱条件下，氮氨菲啶基质匹配标准溶液特征离子质量色谱图见附录A。

# 7.5 空白试验

除不加试料外，采用完全相同的测定步骤进行平行操作。

# 8 结果计算和表述

试料中氮氨菲啶的残留量按式（1）计算：

$$
X = \frac {C _ {s} \times A \times V}{A _ {s} \times m}. \tag {1}
$$

式中：

X—供试试料中氮氨菲啶的残留量，单位为微克每千克（ $\mu \mathrm{g} / \mathrm{kg}$ ）；

$C_{s}$ 一基质匹配标准溶液中氮氨菲啶的浓度, 单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ;

$A_{s}$ 一基质匹配标准溶液中氮氨菲啶的峰面积；

$A$ —试样溶液中氮氨菲啶的峰面积；

$V$ —溶解残余物所用溶液体积, 单位为毫升 $(\mathrm{mL})$ ;

m一供试试料质量，单位为克（g）。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度、精密度

# 9.1 灵敏度

本方法的检测限为 $5\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$

# 9.2 准确度

本方法在 $10\sim 1500~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 110\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 17\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

把 $4\mathrm{mg}\sim 8\mathrm{mg}$ 试样溶于 $1\mathrm{mL}$ 甲醇中，然后加入几滴 $25\%$ 的二甲胺水溶液，溶液从淡红色变成红色而不被还原。

# A.3 特丁基对苯二酚、特丁基对苯醌、2,5一二特丁基氢醌及氢醌的测定

# A.3.1 试剂和材料

A.3.1.1 丙酮。  
A.3.1.2 氢醌标准品。已知纯度。  
A.3.1.3 特丁基对苯二酚标准品：纯度 $\geq 99\%$   
A.3.1.4 特丁基对苯醌标准品：纯度 $\geqslant 99\%$   
A.3.1.5 2,5-二特丁基氢醌标准品：纯度 $\geqslant 99\%$

# A.3.2 仪器和设备

气相色谱仪：配有氢火焰离子化检测器和自动积分仪。

# A.3.3 参考色谱条件

A.3.3.1 色谱柱：HP-5弹性石英毛细管柱，柱长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ ，涂层厚度 $0.25\mu \mathrm{m}$ ；或其他等效的色谱柱。

A.3.3.2 气流速度：载气为高纯氮气，线速为 $30~\mathrm{cm / s}$ 。  
A.3.3.3 温度：柱温 $220^{\circ}C$ ，进样口 $250^{\circ}C$ ，检测器 $300^{\circ}C$ 。  
A.3.3.4 分流比：20:1。  
A.3.3.5 进样量： $1\mu \mathrm{L}$

# A.3.4 分析步骤

# A.3.4.1 标准溶液的制备

分别称取 $10\mathrm{mg}$ 氢醌、特丁基对苯二酚、特丁基对苯醌和2,5一二特丁基氢醌标准品，用丙酮溶解，分别转移至 $10\mathrm{mL}$ 容量瓶中，稀释定容至刻度，摇匀。

# A.3.4.2 试样液的制备

称取 $0.2\mathrm{g}$ 试样，用丙酮溶解，转移至 $10\mathrm{mL}$ 容量瓶中，稀释定容至刻度，摇匀。

# A.3.4.3 测定

在A.3.3参考色谱条件下，对各标准溶液进行气相色谱分析，确定各标准品的保留时间，再注入试样液 $1\mu \mathrm{L}$ ，进行色谱分析。

# A.3.5 结果计算

采用面积归一法分别算出特丁基对苯二酚、特丁基对苯醌、2,5一二特丁基氢醌及氢醌的含量。

实验结果以平行测定结果的算术平均值为准，特丁基对苯二酚测定结果的相对偏差不超过 $0.2\%$ ，其他物质测定结果的相对偏差不超过 $2\%$ 。

# A.4 甲苯的测定

# A.4.1 试剂和材料

A.4.1.1 辛醇。

A.4.1.2 甲苯。

# A.4.2 仪器和设备

气相色谱仪：配有氢火焰离子化检测器。

# A.4.3 参考色谱条件

A. 4.3.1 色谱柱：由不锈钢制成的填充柱，柱长 $3.66 \mathrm{~m}$ ，外径 $3.18 \mathrm{~mm}$ ，填料为SE-30型的硅树酯和硅藻土型保温砖S（Diatoport S） $(180 \mu \mathrm{m} \sim 250 \mu \mathrm{m})$ ，按重量计，其比值为 $10:100$ ；或其他等效的色谱柱。

A. 4.3.2 载气为氮气，气流速度： $25 \mathrm{~mL} / \mathrm{min}$ 。

A.4.3.3 温度：柱温 $70^{\circ}C$ ，进样口 $275^{\circ}C$ ，检测器 $300^{\circ}C$ 。

A.4.3.4 分流比： $50:1$

A. 4.3.5 进样量为 $1 \mu \mathrm{L}$ 。

# A.4.4 分析步骤

# A.4.4.1 试样液的制备

称取约 $2 \mathrm{~g}$ 试样, 精确至 $0.0002 \mathrm{~g}$ , 转移至 $10 \mathrm{~mL}$ 容量瓶中, 用辛醇溶解, 并稀释定容至刻度, 摇匀, 计准确浓度为 $\mathrm{c}_{\mathrm{S}}$ 。

# A.4.4.2 标准溶液的制备

制备每毫升含 $50\mu \mathrm{g}$ 甲苯的辛醇溶液，计准确浓度为 $\mathbf{c}_{\mathbb{R}}$

# A.4.4.3 测定

在A.4.3参考色谱条件下，分别对标准溶液和试样液进行色谱分析。在标准溶液的色谱图上测量甲苯的峰高（ $H_{R}$ ），其它峰对分析无影响。同样的，在试样液的色谱图上测量甲苯的峰高（ $H_{S}$ ）。

# A.4.5 结果计算

甲苯含量以甲苯的质量分数 $w$ 计, 数值以毫克每千克 $(\mathrm{mg} / \mathrm{kg})$ 表示, 按公式 (A.1) 计算:

$$
w = \left(H _ {S} / H _ {R}\right) \times \left(c _ {R} / c _ {S}\right) \dots \tag {A.1}
$$

式中：

$H_{S}$ ——试样液色谱图中甲苯峰高的数值；

$H_{R}$ ——标准溶液色谱图中甲苯峰高的数值；

$c_{R}$ ——标准溶液中甲苯的浓度的数值, 单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ 。

$c_{S}$ ——试样溶液的浓度的数值，单位为克每毫升（ $\mathrm{g/mL}$ ）。

实验结果以平行测定结果的算术平均值为准，平行测定结果的相对偏差不大于 $10\%$ 。

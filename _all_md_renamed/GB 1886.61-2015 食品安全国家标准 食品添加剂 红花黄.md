# 附录A

# 检验方法

# A.1 一般规定

本标准除另有规定外，所用试剂的纯度均为分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602和GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 溶解性试验

易溶于水，微溶于乙醇，几乎不溶于乙醚。

# A.2.2 最大吸收峰

# A.2.2.1 试剂和材料

柠檬酸-磷酸氢二钠缓冲溶液 $(\mathrm{pH}5.0)$ ：称取 $21\mathrm{g}$ 柠檬酸 $\mathrm{(C_6H_8O_7\cdot H_2O)}$ ，溶于 $1000~\mathrm{mL}$ 水中，此为溶液A。称取 $71.6\mathrm{g}$ 磷酸氢二钠 $\mathrm{(Na_2HPO_4\cdot 12H_2O)}$ ，溶于 $1000~\mathrm{mL}$ 水中，此为溶液B。将溶液A与溶液B混合 $(97 + 103)$ 。如果混合液 $\mathrm{pH}$ 不为5.0，则用溶液A或溶液B调整 $\mathrm{pH}$ 至5.0。

# A.2.2.2 测定

称取质量相当于 $0.1\mathrm{g}$ 、色价为10的试样，用柠檬酸-磷酸氢二钠缓冲溶液 $(\mathrm{pH}5.0)$ 溶解并稀释至 $100~\mathrm{mL}$ ，此试样液在 $400~\mathrm{nm}\sim 408~\mathrm{nm}$ 有最大吸收峰。

# A.2.3 颜色反应

在试样液(A.2.2.2)中加入氢氧化钠溶液 $(40\mathrm{g / L})$ ，使溶液变为碱性，试样液色泽则变为橙色。

# A.2.4 薄层色谱

称取质量相当于 $1\mathrm{g}$ 、色价为10的试样，用 $1\mathrm{mL}$ 水溶解，加入 $10\mathrm{mL}$ 甲醇，混匀，在约 $3000\mathrm{r / min}$ 下离心 $10\mathrm{min}$ ，取上清液为试样液。吸取试样液 $20~\mu \mathrm{L}$ ，点样于微晶纤维素薄层色谱板（预先在 $60^{\circ}\mathrm{C} \sim$ $80^{\circ}\mathrm{C}$ 下活化 $20\mathrm{min}$ ）上，干燥，置于展开剂（正丁醇：乙酸：水 $= 4:1:2$ ）中展开。当溶剂前沿上升至距原点约 $10\mathrm{cm}$ 时，取出薄板，在空气中干燥。薄板上至少有2个黄色斑点， $R_{f}$ 值应在 $0.2 \sim 0.5$ 之间。

# A.3 色价 $E_{1\%}^{1}$ （400～408）nm的测定

# A.3.1 仪器和设备

分光光度计。

# A.3.2 试剂和材料

柠檬酸-磷酸氢二钠缓冲溶液 $(\mathrm{pH}5.0)$ ：称取 $21\mathrm{g}$ 柠檬酸 $\mathrm{(C_6H_8O_7\cdot H_2O)}$ ，溶于 $1000~\mathrm{mL}$ 水中，此为溶液A。称取 $71.6\mathrm{g}$ 磷酸氢二钠 $\mathrm{(Na_2HPO_4\cdot 12H_2O)}$ ，溶于 $1000~\mathrm{mL}$ 水中，此为溶液B。将溶液A与溶液B混合 $(97 + 103)$ 。如果混合液 $\mathrm{pH}$ 不为5.0，则用溶液A或溶液B调整 $\mathrm{pH}$ 至5.0。

# A.3.3 分析步骤

称取试样约 $0.1\mathrm{g}$ ，精确至 $0.0002\mathrm{g}$ ，用柠檬酸-磷酸氢二钠缓冲溶液（ $\mathrm{pH}5.0$ ）溶解，定容至 $1000\mathrm{mL}$ ，摇匀，必要时过滤。取此试样液置于 $1\mathrm{cm}$ 比色皿中，以柠檬酸-磷酸氢二钠缓冲溶液 $(\mathrm{pH}5.0)$ 为空白对照，用分光光度计在 $400\mathrm{nm}\sim 408\mathrm{nm}$ 内的最大吸收波长处测定其吸光度（吸光度应控制在 $0.2\sim 0.8$ 之间，否则应调整试样液浓度，再重新测定吸光度）。

# A.3.4 结果计算

色价 $E_{1\mathrm{cm}}^{1\%}(400\sim 408)\mathrm{nm}$ ，按式(A.1)计算：

$$
E _ {1 \mathrm {cm}} ^ {1 \%} (400 \sim 408) \mathrm {nm} = \frac {A}{c \times 100} \quad \dots \tag{A.1}
$$

式中：

A ——实测试样液的吸光度；

$c$ ——被测试样液的浓度，单位为克每毫升 $(\mathrm{g / mL})$ ；

100——浓度换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

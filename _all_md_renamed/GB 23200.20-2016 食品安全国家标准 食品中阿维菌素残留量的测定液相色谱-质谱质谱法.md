# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T1973—2007《进出口食品中阿维菌素残留量的检测方法 高效液相色谱一质谱/质谱法》。与SN/T1973—2007相比，主要变化如下：

——标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“进出口食品”改为“食品”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T1973—2007。

# 食品安全国家标准

# 食品中阿维菌素残留量的测定 液相色谱一质谱/质谱法

# 1 范围

本标准规定了食品中阿维菌素残留量的液相色谱一质谱/质谱检测方法。

本标准适用于大米、大蒜、菠菜、苹果、板栗、茶叶、牛肉、羊肉、鸡肉、鱼肉、赤芍、食醋和蜂蜜中阿维菌素残留量的检测，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

蜂蜜和食醋样品用水稀释后，以 $\mathrm{C_{18}}$ 固相萃取柱净化；其他样品用乙腈提取，用中性氧化铝固相萃取柱净化，液相色谱一质谱/质谱测定，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈：色谱纯。

4.1.2 无水硫酸钠：使用前 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，在干燥器中冷却至室温，储于密封瓶中备用。

# 4.2 溶液配制

4.2.1 乙酸水溶液 $(0.1\%)$ ：取 $1\mathrm{mL}$ 乙酸，以水定容至 $1000~\mathrm{mL}$

4.2.2 乙腈水溶液 $(1 + 6$ ，体积比)：取 $100~\mathrm{mL}$ 乙腈，加入 $600~\mathrm{mL}$ 水，摇匀备用。

# 4.3 标准品

阿维菌素标准物质（Abamectin， $\mathrm{C_{48}H_{72}O_{14}}$ ，CAS号：71754-41-2）：阿维菌素Bla含量大于 $87\%$ 以下阿维菌素含量均以阿维菌素Bla计。

# 4.4 标准溶液配制

4.4.1 阿维菌素标准储备液：称取适量(精确至 $0.0001\mathrm{g}$ )阿维菌素标准物质，以乙腈溶解配制浓度为 $100~\mu \mathrm{g / mL}$ 的标准储备液，保存于一 $18^{\circ}C$ 冰箱内。

4.4.2 阿维菌素标准中间液：准确移取阿维菌素标准储备液，以乙腈稀释配制成含 $1\mu \mathrm{g / mL}$ 浓度的标准中间液，保存于 $4^{\circ}C$ 冰箱内。

4.4.3 阿维菌素标准工作液：根据需要准确移取适量阿维菌素标准中间液，以乙腈稀释并定容至适当浓度的标准工作液，保存于 $4^{\circ}\mathrm{C}$ 冰箱内。

# 4.5 材料

4.5.1 中性氧化铝固相萃取柱： $3\mathrm{mL} / 1000\mathrm{mg}$

4.5.2 $C_{18}$ 固相萃取柱： $6\mathrm{mL} / 1000\mathrm{mg}$ 。  
4.5.3 有机滤膜： $0.45\mu \mathrm{m}$

# 5 仪器和设备

5.1 液相色谱—质谱/质谱仪，配有大气压化学电离源（APCI源）。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.3 均质器。  
5.4 离心机， $3000\mathrm{r / in}$ 以上。  
5.5 涡旋振荡器。  
5.6 固相萃取装置。  
5.7 旋转蒸发器。  
5.8 氮吹仪。

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按GB2763的规定执行。在取样和制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 苹果、大蒜、菠菜、板栗

苹果、板栗：取样品约 $500\mathrm{g}$ ，用捣碎机捣碎，装入洁净容器作为试样，密封并做好标识；大蒜、菠菜： 取样品约 $500\mathrm{g}$ ，用捣碎机捣碎，装入洁净容器作为试样，密封并做好标识。

# 6.2.2 大米、茶叶、赤芍、食醋

取样品约 $500 \mathrm{~g}$ 用粉碎机粉碎至全部通过 $850 \mu \mathrm{m}$ 筛, 装入洁净容器作为试样, 密封并做好标识; 食醋可装入洁净容器, 密封并做好标识。

# 6.2.3 牛肉、羊肉、鸡肉、鱼肉

取有代表性样品约 $500 \mathrm{~g}$ , 用捣碎机捣碎, 装入洁净容器作为试样, 密封并做好标识。

# 6.2.4 蜂蜜

取有代表性样品约 $500\mathrm{g}$ , 未结晶样品将其用力搅拌均匀, 有结晶析出样品可将样品瓶盖塞紧后, 置于不超过 $60^{\circ}\mathrm{C}$ 的水浴中, 待样品全部溶化后搅匀, 迅速冷却至室温。制备好的样品装入洁净容器内密封并做好标识。

# 6.3 试样保存

苹果、大蒜、菠菜、板栗等试样于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 保存; 牛肉、羊肉、鸡肉、鱼肉等试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存; 大米、茶叶、赤芍、食醋、蜂蜜等试样于室温保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 大蒜、菠菜、苹果、板栗、牛肉、羊肉、鸡肉、鱼肉样品

准确称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样，加入 $5\mathrm{g}$ 无水硫酸钠和 $15\mathrm{mL}$ 乙腈，以 $10000\mathrm{r / min}$ 均质 $2\mathrm{min},3000\mathrm{r / min}$ 下离心 $5\mathrm{min}$ ，上清液经无水硫酸钠过滤并转入浓缩瓶中。用 $10\mathrm{mL}$ 乙腈再提取一次，合并提取液。将提取液于 $40^{\circ}\mathrm{C}$ 水浴下浓缩至 $2\mathrm{mL} \sim 3\mathrm{mL}$ 。

# 7.1.2 大米、茶叶、赤芍样品

准确称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样，用 $2\mathrm{mL}$ 水将试样润湿（茶叶用 $20\mathrm{mL}$ 水润湿），放置 $15\mathrm{min}$ ，加入 $5\mathrm{g}$ 无水硫酸钠和 $15\mathrm{mL}$ 乙腈，以 $10000\mathrm{r / min}$ 均质 $2\mathrm{min},3000\mathrm{r / min}$ 下离心 $5\mathrm{min}$ ，将上清液转入浓缩瓶中。用 $10\mathrm{mL}$ 乙腈再提取一次，合并提取液。将提取液于 $40^{\circ}\mathrm{C}$ 水浴下浓缩至 $2\mathrm{mL} \sim 3\mathrm{mL}$ 。

# 7.1.3 食醋、蜂蜜样品

准确称取 $5\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 均匀试样，加入 $30~\mathrm{mL}$ 水，涡旋混匀。

# 7.2 净化

# 7.2.1大米、大蒜、菠菜、苹果、板栗、茶叶、牛肉、羊肉、鸡肉、鱼肉、赤芍样品

用 $3\mathrm{mL}$ 乙腈对中性氧化铝柱进行预淋洗。将7.1.1和7.1.2中得到的样品提取液转入中性氧化铝柱，用 $5\mathrm{mL}$ 乙腈分两次洗涤浓缩瓶并将洗涤液转入中性氧化铝柱中，调整流速在 $1.5\mathrm{mL} / \mathrm{min}$ 左右，用 $2\mathrm{mL}$ 乙腈淋洗小柱，收集全部流出液。将流出液在 $50^{\circ}\mathrm{C}$ 下吹干，用 $1.00\mathrm{mL}$ 乙腈溶解残渣，滤膜过滤，供液相色谱—质谱/质谱测定。

# 7.2.2 食醋、蜂蜜样品

依次用 $5 \mathrm{~mL}$ 乙腈和 $5 \mathrm{~mL}$ 乙腈水溶液对 $\mathrm{C}_{18}$ 固相萃取柱进行预淋洗。将7.1.3得到的样品稀释液加入 $\mathrm{C}_{18}$ 固相萃取柱，调整流速在 $1.5 \mathrm{~mL} / \mathrm{min}$ 左右，加入 $10 \mathrm{~mL}$ 水淋洗 $\mathrm{C}_{18}$ 固相萃取柱，将固相萃取柱吹干。加入 $5 \mathrm{~mL}$ 乙腈进行洗脱，收集全部洗脱液。将洗脱液在 $50^{\circ} \mathrm{C}$ 下吹干，用 $1.00 \mathrm{~mL}$ 乙腈溶解残渣，滤膜过滤，供液相色谱—质谱/质谱测定。

# 7.3 测定

# 7.3.1 液相色谱一质谱/质谱参考条件

a）色谱柱：色谱柱： $\mathrm{C_{18}}$ 柱， $150\mathrm{mm}\times 2.1\mathrm{mm}$ （内径），粒度 $5\mu \mathrm{m}$   
b) 流动相: 乙腈 + 乙酸水溶液 $(0.1\%) = 70 + 30$   
c) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ ;   
d) 柱温： $40^{\circ}\mathrm{C}$ ；  
e) 进样量: $20 \mu \mathrm{L}$ ;   
f) 离子源: 大气压化学电离源(APCI 源), 负离子监测模式;  
g）质谱仪参数：参见附录A；  
h）监测离子对 $(m / z)$ ：定性离子对(872/565，872/854)，定量离子对(872/565)。

# 7.3.2 色谱测定与确证

根据试样中阿维菌素的含量情况，选择浓度相近的标准工作液进行色谱分析，以峰面积按外标法定量。在上述色谱条件下，阿维菌素的参考保留时间为 $11.3\mathrm{min}$ 。标准溶液的选择离子流图参见图B.1、图B.2。

按照上述条件测定样品和标准工作液，如果检测的质量色谱峰保留时间与标准工作液一致；定性离子对的相对丰度与相当浓度的标准工作液的相对丰度一致，相对丰度偏差不超过表1的规定，则可判断样品中存在相应的被测物。标准溶液的二级质谱图参见图B.3。

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

按数据处理软件处理或式(1)计算样品中阿维菌素残留量。

$$
X = \frac {A \times c \times V}{A _ {s} \times m} \times \frac {1 0 0 0}{1 0 0 0}. \tag {1}
$$

式中：

$X$ ——试样中阿维菌素残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

A ——样液中阿维菌素的峰面积；

V ——样品最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——阿维菌素标准工作液的峰面积；

$c$ ——阿维菌素标准工作液的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$m$ 最终样液代表的试样质量，单位为克 $(\mathrm{g})$

1000——换算系数。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 C 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 D 的要求。

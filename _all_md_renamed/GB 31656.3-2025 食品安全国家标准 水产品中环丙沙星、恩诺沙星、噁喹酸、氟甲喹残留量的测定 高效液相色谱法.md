# 1 范围

本文件规定了水产品中环丙沙星、恩诺沙星、噁喹酸、氟甲喹残留量检测的制样和高效液相色谱测定方法。

本文件适用于鱼类肌肉和皮等可食组织,虾、蟹、贝类的可食组织中环丙沙星、恩诺沙星、噁唑酸、氟甲喹残留量的检测。其他水产品的检测可参照执行。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

试料中残留的环丙沙星、恩诺沙星、噁唑酸、氟甲喹，经酸化乙腈提取，正已烷脱脂， $\mathrm{C_{18}}$ 固相萃取柱净化，液相色谱-荧光检测法测定，外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.2 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ ：色谱纯。

5.1.3 正己烷 $(\mathrm{C_6H_{14}})$ ：色谱纯。

5.1.4 盐酸（HCl）。

5.1.5 磷酸 $\mathrm{(H_3PO_4)}$ 。

5.1.6 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。

5.1.7 氢氧化钠(NaOH)。

5.1.8 二水合磷酸二氢钠 $(\mathrm{NaH_2PO_4}\cdot 2\mathrm{H_2O})$

5.1.9 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。

5.1.10 四丁基溴化铵 $\mathrm{(C_{16}H_{36}BrN)}$

# 5.2 标准品

环丙沙星、恩诺沙星、噁唑酸、氟甲喹：含量 $\geq 98.0\%$ ，详见附录A。

# 5.3 溶液配制

5.3.1 盐酸溶液：取盐酸 $50~\mathrm{mL}$ ，加水稀释至 $100~\mathrm{mL}$ ，混匀。  
5.3.2 磷酸溶液：取磷酸 $1\mathrm{mL}$ ，加水稀释至 $10~\mathrm{mL}$ ，混匀。  
5.3.3 酸化乙腈溶液：取乙腈 $250~\mathrm{mL}$ ，加盐酸溶液 $2\mathrm{mL}$ ，充分混匀。  
5.3.4 乙腈饱和正己烷溶液：取正己烷 $200\mathrm{mL}$ 于 $250~\mathrm{mL}$ 分液漏斗中，加适量乙腈后，剧烈振摇，待分配平衡后，弃去乙腈层，即得。  
5.3.5 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液：取盐酸 $9\mathrm{mL}$ ，用水稀释至 $1000\mathrm{mL}$ 。  
5.3.6 0.1 mol/L 氢氧化钠溶液：取氢氧化钠 $4\mathrm{g}$ ，用水溶解并稀释至 $1000\mathrm{mL}$ 。  
5.3.7 0.05 mol/L 磷酸盐缓冲溶液：取二水合磷酸二氢钠 $1.56 \mathrm{~g}$ ，加 $0.1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液 $79 \mathrm{~mL}$ 溶解，用水稀释至 $200 \mathrm{~mL}$ 。  
5.3.8 0.01 mol/L 四丁基溴化铵溶液 $(\mathrm{pH}3.0)$ : 取四丁基溴化铵 $3.22\mathrm{g}$ , 加水 $900\mathrm{mL}$ 使溶解, 用磷酸溶液调节 $\mathrm{pH}$ 到 3.0, 用水稀释至 $1000\mathrm{mL}$ 。  
5.3.9 洗脱液：取氨水 $25\mathrm{mL}$ ，用甲醇稀释至 $100\mathrm{mL}$ 。

# 5.4 标准溶液制备

5.4.1 环丙沙星、恩诺沙星标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：取环丙沙星、恩诺沙星各约 $10\mathrm{mg}$ ，精密称定，用甲醇溶解并稀释定容至 $100~\mathrm{mL}$ 棕色容量瓶中，摇匀，即得。一 $18^{\circ}C$ 以下避光保存，有效期6个月。  
5.4.2 嗪喹酸、氟甲喹标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：取噁喹酸、氟甲喹各约 $10\mathrm{mg}$ ，精密称定，用 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液 $10~\mathrm{mL}$ 使溶解，加乙腈稀释定容至 $100~\mathrm{mL}$ 棕色容量瓶中，摇匀，即得。一 $18^{\circ}C$ 以下避光保存，有效期6个月。  
5.4.3 混合标准工作液 $(10\mu \mathrm{g} / \mathrm{mL})$ : 精密量取环丙沙星、恩诺沙星、噁喹酸、氟甲喹标准储备液各 $1\mathrm{mL}$ , 于 $10\mathrm{mL}$ 棕色容量瓶中, 用流动相A稀释至刻度, 摇匀, 配制成浓度为 $10\mu \mathrm{g} / \mathrm{mL}$ 混合标准工作液。 $4^{\circ}\mathrm{C}$ 以下避光保存, 有效期1个月。

# 5.5 材料

$\mathrm{C}_{18}$ 固相萃取柱： $500 \, \mathrm{mg} / 3 \, \mathrm{mL}$ ，或相当者。

# 6 仪器和设备

6.1 高效液相色谱仪：配荧光检测器。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
6.3 氮吹仪。  
6.4 超声波振荡器。  
6.5 均质器：15000r/min。  
6.6 离心机：4000 r/min。  
6.7 尼龙微孔滤膜: $0.22 \mu \mathrm{m}$ 。  
6.8 梨形瓶： $100~\mathrm{mL}$   
6.9 分液漏斗： $250~\mathrm{mL}$   
6.10 旋转蒸发器。  
6.11 分液漏斗振摇器。

# 7 试样的制备与保存

# 7.1 试样的制备

按GB/T30891—2014中附录B的要求制样。

a）取均质的供试样品，作为供试试样；  
b）取均质的空白样品，作为空白试样；

c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

- $18^{\circ} \mathrm{C}$ 以下保存，3个月内进行分析检测。

# 8 测定步骤

# 8.1 提取

称取试料 $(5\pm 0.05)\mathrm{g}$ ，加酸化乙腈溶液 $20~\mathrm{mL}$ 、无水硫酸钠 $10\mathrm{g}$ ，高速均质 $1\mathrm{min}\sim 2\mathrm{min}$ ，3000r/min离心 $5\mathrm{min}$ ，取上清液至分液漏斗，用酸化乙腈溶液 $20~\mathrm{mL}$ ，清洗刀头并溶解沉淀，重复提取2次，合并上清液。加乙腈饱和正己烷溶液 $60~\mathrm{mL}$ ，置分液漏斗振摇器(150r/min)上振荡 $20\mathrm{min}$ ，静置，取下层乙腈至梨形瓶， $40^{\circ}C$ 旋转蒸发至干。加磷酸盐缓冲液 $4\mathrm{mL}$ ，超声振荡 $1\mathrm{min}$ ，静置 $30~s$ ，溶液转移至 $15~\mathrm{mL}$ 离心管。再重复提取残余物2次，溶液转移至同一离心管，合并3次残余物溶解液： $3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液，备用。

# 8.2 净化

取 $\mathrm{C_{18}}$ 固相萃取柱，依次用甲醇、水、磷酸盐缓冲液各 $3\mathrm{mL}$ 活化。取上述备用液过柱，流速控制为每秒1滴。水 $3\mathrm{mL}$ 淋洗，抽干，加洗脱液 $5\mathrm{mL}$ ，收集洗脱液， $50^{\circ}\mathrm{C}$ 氮气吹干，加流动相A $1.0\mathrm{mL}$ 使溶解，过 $0.22\mu \mathrm{m}$ 微孔滤膜，高效液相色谱测定。

# 8.3 标准曲线的制备

分别精密量取混合标准工作液适量，用流动相A稀释成浓度分别为 $10\mathrm{ng / mL},50\mathrm{ng / mL},100\mathrm{ng / mL}$ 、 $500\mathrm{ng / mL},1000\mathrm{ng / mL},2000\mathrm{ng / mL}$ 的系列标准溶液，供高效液相色谱分析；现用现配。以峰面积为纵坐标、浓度为横坐标，制作标准曲线。

# 8.4.2 测定法

取试料溶液和相应的标准溶液，作单点或多点校准，以相应药物的保留时间定性，被测试样中环丙沙

星、恩诺沙星、噁喹酸、氟甲喹色谱峰保留时间与相应标准溶液色谱峰的保留时间相比，相差应在 $\pm 0.1\mathrm{min}$ 内。按外标法以峰面积计算。标准溶液及试料溶液中4种喹诺酮类药物响应值均应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液高效液相色谱图见附录B。

# 8.5 空白试验

取空白试样，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times C _ {\mathrm {s}} \times V}{A _ {\mathrm {s}} \times m} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中被测组分残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——试料溶液中相应被测组分峰面积；

$C_{s}$ ——标准工作液测得的相应被测组分溶液浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试料溶液定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准溶液中相应被测组分峰面积；

$m$ ——试料质量的数值，单位为克(g)。

注：计算结果以平行测定结果的算术平均值表示，保留3位有效数字。

# 10 方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法环丙沙星、恩诺沙星的检出限为 $2.5\mu \mathrm{g / kg}$ ，噁喹酸、氟甲喹的检出限为 $5\mu \mathrm{g / kg}$ ；环丙沙星、恩诺沙星的定量限为 $5\mu \mathrm{g / kg}$ ，噁喹酸、氟甲喹的定量限为 $10\mu \mathrm{g / kg}$ 。

# 10.2 正确度

标准添加浓度 $5\mu \mathrm{g} / \mathrm{kg}\sim 10\mu \mathrm{g} / \mathrm{kg}$ ，回收率为 $60\% \sim 120\%$ ；标准添加浓度 $10~\mu \mathrm{g} / \mathrm{kg}\sim 100~\mu \mathrm{g} / \mathrm{kg}$ ，回收率为 $70\% \sim 120\%$ ；标准添加浓度 $100~\mu \mathrm{g} / \mathrm{kg}\sim 500~\mu \mathrm{g} / \mathrm{kg}$ ，回收率为 $80\% \sim 110\%$ 。

# 10.3 精密度

# 10.3.1 重复性精密度

标准添加浓度 $5\mu \mathrm{g} / \mathrm{kg}\sim 10\mu \mathrm{g} / \mathrm{kg}$ ，相对标准偏差 $\leqslant 20\%$ ；标准添加浓度 $10~\mu \mathrm{g} / \mathrm{kg}\sim 100~\mu \mathrm{g} / \mathrm{kg}$ ，相对标准偏差 $\leqslant 15\%$ ；标准添加浓度 $100~\mu \mathrm{g} / \mathrm{kg}\sim 500~\mu \mathrm{g} / \mathrm{kg}$ ，相对标准偏差 $\leqslant 10\%$ 。

# 10.3.2 再现性精密度

标准添加浓度 $5\mu \mathrm{g / kg}\sim 10\mu \mathrm{g / kg}$ ，相对标准偏差 $\leqslant 30\%$ ；标准添加浓度 $10~\mu \mathrm{g / kg}\sim 100~\mu \mathrm{g / kg}$ ，相对标准偏差 $\leqslant 20\%$ ；标准添加浓度 $100~\mu \mathrm{g / kg}\sim 500~\mu \mathrm{g / kg}$ ，相对标准偏差 $\leqslant 15\%$ 。

# 食品中脱氢乙酸的测定

2016-08-31 发布

2017-03-01 实施

# 前言

本标准代替GB/T5009.121—2003《食品中脱氢乙酸的测定》。

本标准与GB/T5009.121—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中脱氢乙酸的测定”；  
——增加液相色谱法；  
液相色谱法采用GB/T23377—2009的方法；  
——增加试样分类；  
——改进气相色谱法中样品的制备和提取方法；  
——修改气相色谱法的色谱条件。

# 食品安全国家标准

# 食品中脱氢乙酸的测定

# 1 范围

本标准规定了果蔬汁、果蔬浆、酱菜、发酵豆制品、黄油、面包、糕点、烘烤食品馅料、复合调味料、预制肉制品及熟肉制品中脱氢乙酸含量的测定方法。

本标准适用于果蔬汁、果蔬浆、酱菜、发酵豆制品、黄油、面包、糕点、烘烤食品馅料、复合调味料、预制肉制品及熟肉制品中脱氢乙酸含量的测定，其他食品可参考执行。

# 第一法 气相色谱法

# 2 原理

固体(半固体)样品，沉降蛋白、经脱脂酸化后，用乙酸乙酯提取；果蔬汁、果蔬浆样品经酸化后，用乙酸乙酯提取；用配氢火焰离子化检测器的气相色谱仪分离测定，以色谱峰的保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。  
3.1.2 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
3.1.3 盐酸（HCl）。  
3.1.4 硫酸锌 $(\mathrm{ZnSO}_4\cdot 7\mathrm{H}_2\mathrm{O})$   
3.1.5 氢氧化钠(NaOH)。

# 3.2 试剂配制

3.2.1 盐酸溶液 $(1 + 1$ ，体积比)：量取 $50~\mathrm{mL}$ 盐酸加入到 $50~\mathrm{mL}$ 水中。  
3.2.2 硫酸锌溶液 $(120\mathrm{g / L})$ ：称取 $12\mathrm{g}$ 硫酸锌，溶于水并稀释至 $100~\mathrm{mL}$ 。  
3.2.3 氢氧化钠溶液 $(20\mathrm{g} / \mathrm{L})$ ：称取 $2\mathrm{g}$ 氢氧化钠，溶于水并稀释至 $100~\mathrm{mL}$ 。

# 3.3 标准品

脱氢乙酸(Dehydroacetic Acid, $\mathrm{C_8H_8O_4}$ , CAS:520-45-6)标准品：纯度 $\geqslant 99.5\%$ 。

# 3.4 标准溶液的制备

3.4.1 脱氢乙酸标准贮备液 $(1.0\mathrm{mg / mL})$ : 准确称取脱氢乙酸标准品 $0.1000\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ) 于 $100\mathrm{mL}$ 容量瓶中, 用乙酸乙酯溶解并定容。 $4^{\circ}\mathrm{C}$ 保存, 有效期为 3 个月。

3.4.2 脱氢乙酸标准工作液：分别精确吸取脱氢乙酸标准贮备液 $0.01\mathrm{mL}, 0.1\mathrm{mL}, 0.5\mathrm{mL}, 1.0\mathrm{mL}$ 、 $2.0\mathrm{mL}$ 于 $10\mathrm{mL}$ 容量瓶中，用乙酸乙酯稀释并定容，配制成浓度为 $1.00\mu \mathrm{g} / \mathrm{mL}, 10.0\mu \mathrm{g} / \mathrm{mL}, 50.0\mu \mathrm{g} / \mathrm{mL}, 100\mu \mathrm{g} / \mathrm{mL}, 200\mu \mathrm{g} / \mathrm{mL}$ 标准工作液。 $4^{\circ}\mathrm{C}$ 保存，有效期为1个月。

# 4 仪器和设备

4.1 气相色谱仪，配氢火焰离子化检测器。   
4.2 天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$ 。  
4.3 离心机：转速 $\geqslant 4000\mathrm{r / min}$   
4.4 超声波清洗器：功率 $35\mathrm{kW}$   
4.5 粉碎机。  
4.6 不锈钢高速均质器。  
4.7 pH计。

# 5 分析步骤

# 5.1 试样制备

5.1.1果蔬汁、果蔬浆：称取样品 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，置于 $50~\mathrm{mL}$ 离心管中，加 $10~\mathrm{mL}$ 水振摇 $1\mathrm{min}$ ，加 $1\mathrm{mL}$ 盐酸溶液(3.2.1)酸化后，准确加入 $5.0~\mathrm{mL}$ 乙酸乙酯，振摇提取 $2\mathrm{min}$ ，静置分层，取上清液供气相色谱测定。  
5.1.2 酱菜、发酵豆制品：样品用不锈钢高速均质器均质。称取样品 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 离心管中，加入约 $15~\mathrm{mL}$ 水、 $2.5~\mathrm{mL}$ 硫酸锌溶液(3.2.2)，用氢氧化钠溶液(3.2.3)调 $\mathrm{pH}$ 至7.5，超声提取 $15\mathrm{min}$ ，转移至 $25~\mathrm{mL}$ 容量瓶中，加水定容。样液移入离心管中， $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 取 $10~\mathrm{mL}$ 上清液，加 $1\mathrm{mL}$ 盐酸溶液(3.2.1)酸化后，准确加入 $5.0~\mathrm{mL}$ 乙酸乙酯，振摇 $2\mathrm{min}$ ，静置分层，取上清液供气相色谱测定。  
5.1.3 面包、糕点、烘烤食品馅料、复合调味料、预制肉制品及熟肉制品：样品用粉碎机粉碎或不锈钢高速均质器均质。称取样品 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 离心管中，加入约 $15~\mathrm{mL}$ 水、 $2.5~\mathrm{mL}$ 硫酸锌溶液(3.2.2)，用氢氧化钠溶液(3.2.3)调 $\mathrm{pH}$ 至7.5，超声提取 $15\mathrm{min}$ ，转移至 $25~\mathrm{mL}$ 容量瓶中，加水定容。样液移入分液漏斗中，加入 $5\mathrm{mL}$ 正己烷，振摇 $1\mathrm{min}$ ，静置分层，取下层水相置于离心管中， $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取 $10~\mathrm{mL}$ 上清液，加 $1\mathrm{mL}$ 盐酸溶液(3.2.1)酸化后，准确加入 $5.0~\mathrm{mL}$ 乙酸乙酯，振摇 $2\mathrm{min}$ ，静置分层，取上清液供气相色谱测定。  
5.1.4 黄油：称取样品 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 离心管中，加入约 $15~\mathrm{mL}$ 水、 $2.5~\mathrm{mL}$ 硫酸锌溶液(3.2.2)，用氢氧化钠溶液(3.2.3)调 $\mathrm{pH}$ 至7.5，超声提取 $15\mathrm{min}$ ，转移至 $25~\mathrm{mL}$ 容量瓶中，加水定容。样液移入分液漏斗中，加入 $5\mathrm{mL}$ 正己烷，振摇 $1\mathrm{min}$ ，静置分层，取下层水相置于离心管中， $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取 $10~\mathrm{mL}$ 上清液，加 $1\mathrm{mL}$ 盐酸溶液(3.2.1)酸化后，准确加入 $5.0~\mathrm{mL}$ 乙酸乙酯，振摇 $2\mathrm{min}$ ，静置分层，取上清液供气相色谱测定。

# 5.2 仪器参考条件

5.2.1 毛细管柱：极性毛细柱（化学键和聚乙二醇固定相， $30\mathrm{m}\times 0.32\mathrm{mm}\times 0.25\mu \mathrm{m}$ ），或相当者。  
5.2.2 柱温升温程序：初温 $150^{\circ}\mathrm{C}$ ，以 $10^{\circ}\mathrm{C / min}$ 速率升至 $210^{\circ}\mathrm{C}, 20^{\circ}\mathrm{C / min}$ 速率升至 $240^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$ 。  
5.2.3 进样口温度： $240^{\circ}C$   
5.2.4 检测器温度： $300^{\circ}C$ 。

5.2.5 载气 $(\mathrm{N}_2)$ 流量： $1.0~\mathrm{mL / min}$   
5.2.6 分流进样，分流比为 $5:1$ ，进样体积 $1.0\mu \mathrm{L}$ 。

# 5.3 标准曲线的制作

将脱氢乙酸标准工作液(3.4.2)分别注入气相色谱仪中，测定相应峰面积，以标准工作液的浓度为横坐标，峰面积为纵坐标，绘制标准曲线。（脱氢乙酸标准色谱图见附录A）。

# 5.4 测定

将测定溶液注入气相色谱仪中，以保留时间定性，同时记录峰面积，根据标准曲线得到测定溶液中的脱氢乙酸浓度。

# 5.5 空白试验

除不加试样外，空白试验应与样品测定平行进行，并采用相同的分析步骤分析。

# 6 分析结果的表述

果蔬汁、果蔬浆试样中脱氢乙酸含量按式(1)计算：

$$
X _ {1} = \frac {\left(c _ {1} - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {1}
$$

其他试样中脱氢乙酸含量按式(2)计算：

$$
X _ {2} = \frac {\left(c _ {1} - c _ {0}\right) \times V _ {1} \times V \times 1 0 0 0}{m \times V _ {2} \times 1 0 0 0 \times 1 0 0 0} \tag {2}
$$

式中：

$X_{1}$ ——试样中脱氢乙酸的含量，单位为克每千克 $(\mathrm{g / kg})$

$c_{1}$ ——试样溶液中脱氢乙酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$c_{0}$ ——空白试样溶液中脱氢乙酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——乙酸乙酯定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——称取试样的质量，单位为克 $(\mathrm{g})$ ；

$X_{2}$ ——试样中脱氢乙酸的含量，单位为克每千克 $(\mathrm{g / kg})$

$V_{1}$ ——试样处理后定容体积，单位为毫升 $(\mathrm{mL})$

$V_{2}$ ——萃取脱氢乙酸所取试样液体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

果蔬汁、果蔬浆取样量 $5\mathrm{g}$ ，确定检出限为 $0.0003\mathrm{g/kg}$ ，定量限为 $0.001\mathrm{g/kg}$ ；其他试样取样量 $5\mathrm{g}$ ，样液定容体积 $25\mathrm{mL}$ ，取 $10\mathrm{mL}$ 样液提取，确定检出限为 $0.001\mathrm{g/kg}$ ，定量限为 $0.003\mathrm{g/kg}$ 。

# 第二法 液相色谱法

# 9 原理

用氢氧化钠溶液提取试样中的脱氢乙酸，经脱脂、去蛋白处理，过膜，用配紫外或二极管阵列检测器的高效液相色谱仪测定，以色谱峰的保留时间定性，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。

10.1.2 乙酸铵 $\mathrm{(C_2H_6O_2N)}$ ：优级纯。

10.1.3 氢氧化钠(NaOH)。

10.1.4 正己烷 $(\mathrm{C_6H_{14}})$ 。

10.1.5 甲酸 $\mathrm{(CH_2O_2)}$ 。

10.1.6 硫酸锌 $(\mathrm{ZnSO_4}\cdot 7\mathrm{H_2O})$ 。

# 10.2 试剂配制

10.2.1 乙酸铵溶液 $(0.02\mathrm{mol} / \mathrm{L})$ ：称取 $1.54\mathrm{g}$ 乙酸铵，溶于水并稀释至 $1\mathrm{L}$ 。

10.2.2 氢氧化钠溶液 $(20\mathrm{g} / \mathrm{L})$ ：称取 $20\mathrm{g}$ 氢氧化钠，溶于水并稀释至 $1\mathrm{L}$ 。

10.2.3 甲酸溶液 $(10\%)$ ：量取 $10~\mathrm{mL}$ 甲酸，加水 $90~\mathrm{mL}$ ，混匀。

10.2.4 硫酸锌溶液 $(120\mathrm{g / L})$ ：称取 $120\mathrm{g}$ 硫酸锌，溶于水并稀释至 $1\mathrm{L}$ 。

10.2.5 甲醇溶液 $(70\%)$ ：量取 $70~\mathrm{mL}$ 甲醇，加水 $30~\mathrm{mL}$ ，混匀。

# 10.3 标准品

脱氢乙酸(Dehydroacetic Acid, $\mathrm{C_8H_8O_4}$ , CAS:520-45-6)标准品：纯度 $\geq 99.5\%$ 。

# 10.4 标准溶液的制备

10.4.1 脱氢乙酸标准贮备液 $(1.0\mathrm{mg / mL})$ ：准确称取脱氢乙酸标准品 $0.1000\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）于 $100~\mathrm{mL}$ 容量瓶中，用 $10~\mathrm{mL}$ 氢氧化钠溶液(10.2.2)溶解，用水定容。 $4^{\circ}C$ 保存，有效期为3个月。  
10.4.2 脱氢乙酸标准工作液：分别吸取脱氢乙酸贮备液 $0.1\mathrm{mL}, 1.0\mathrm{mL}, 5.0\mathrm{mL}, 10\mathrm{mL}, 20\mathrm{mL}$ 于 $100\mathrm{mL}$ 容量瓶中，用水定容。配制成浓度为 $1.00\mu \mathrm{g} / \mathrm{mL}, 10.0\mu \mathrm{g} / \mathrm{mL}, 50.0\mu \mathrm{g} / \mathrm{mL}, 100\mu \mathrm{g} / \mathrm{mL}, 200\mu \mathrm{g} / \mathrm{mL}$ 标准工作液。 $4^{\circ}\mathrm{C}$ 保存，有效期为1个月。

# 11 仪器和设备

11.1 高效液相色谱仪：配有紫外检测器或二极管阵列检测器。  
11.2 分析天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
11.3 粉碎机。

11.4 不锈钢高速均质器。  
11.5 超声波清洗器：功率 $35\mathrm{kW}$   
11.6 涡旋混合器。  
11.7 离心机：转速 $\geqslant 4000\mathrm{r / min}$   
11.8 pH计。  
11.9 $\mathrm{C}_{18}$ 固相萃取柱： $500\mathrm{mg},6\mathrm{mL}$ （使用前用 $5\mathrm{mL}$ 甲醇、 $10~\mathrm{mL}$ 水活化，使柱子保持湿润状态）。

12.2.3 流速： $1.0\mathrm{mL / min}$   
12.2.4 柱温： $30^{\circ}\mathrm{C}$ 。

# 12.3 定性分析

依据保留时间一致性进行定性识别的方法，根据脱氢乙酸标准样品的保留时间，确定样品中脱氢乙酸的色谱峰(参见附录B)。必要时应采用其他方法进一步定性确证。

# 12.4 标准曲线的制作

将脱氢乙酸标准工作液(10.4.2)分别注入液相色谱仪中，测定相应的峰面积，以标准工作液的浓度为横坐标，峰面积为纵坐标，绘制标准曲线。

# 12.5 测定

将测定溶液注入液相色谱仪中，测得相应峰面积，根据标准曲线得到测定溶液中的脱氢乙酸浓度。

# 12.6 空白试验

除不加试样外，空白试验应与样品测定平行进行，并采用相同的分析步骤分析。

# 13 分析结果的表述

试样中脱氢乙酸含量按式(3)计算：

$$
X = \frac {\left(c _ {1} - c _ {0}\right) \times V \times 1 0 0 0 \times f}{m \times 1 0 0 0 \times 1 0 0 0} \quad \dots \tag {3}
$$

式中：

$X$ ——试样中脱氢乙酸的含量，单位为克每千克 $(\mathrm{g} / \mathrm{kg})$ ；

$c_{1}$ ——试样溶液中脱氢乙酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$c_{0}$ ——空白试样溶液中脱氢乙酸的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样溶液总体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——过固相萃取柱换算系数 $(f = 0.5)$ ；

$m$ ——称取试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

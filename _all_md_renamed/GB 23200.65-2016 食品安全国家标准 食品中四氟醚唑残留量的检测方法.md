# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2645—2010《进出口食品中四氟醚唑残留量检测方法》。与SN/T2645—2010相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
——标准名称和范围中“进出口食品”改为“食品”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T2645—2010。

# 食品安全国家标准食品中四氟醚唑残留量的检测方法

# 1 范围

本标准规定了食品中四氟醚唑残留量检测和制样、气相色谱—质谱检测和确证方法。

本标准适用于菠菜、藕、草莓、花生、鸡肉、猪肉、鳕、蜂蜜、板栗、茶叶和酱油中四氟醚唑残留量的测定和确证，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样经乙腈提取，以正己烷液液分配和硅酸镁固相萃取柱净化，用气相色谱一负化学源质谱测定，外标法定量。

# 4 试剂和材料

除另有规定外，所用试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 丙酮 $\mathrm{(C_3H_6O)}$ ：残留级。

4.1.2 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ ：残留级。

4.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：残留级。

4.1.4 氯化钠 $(\mathrm{NaCl})$ ： $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置入干燥器中冷却，备用。

# 4.2 溶液配制

4.2.1 丙酮—正己烷(3+7，体积比)：取 $300~\mathrm{mL}$ 丙酮，加入 $700~\mathrm{mL}$ 正己烷，摇匀备用。

# 4.3 标准品

4.3.1 四氟醚唑标准品（tetraconazole, $\mathrm{C_{13}H_{11}Cl_2F_4N_3O}$ ）：纯度 $\geqslant 96.5\%$

# 4.4 标准溶液配制

4.4.1 四氟醚唑标准溶液的配制：准确称取适量标准品，用少量的丙酮溶解，并以丙酮配制成浓度为 $1.0 \mathrm{mg} / \mathrm{mL}$ 的标准储备液。根据需要再用丙酮一正己烷稀释成适当浓度的标准工作溶液。保存于 $-18^{\circ} \mathrm{C}$ 中。

# 4.5 材料

4.5.1 硅酸镁(cleanertflorisil)固相萃取柱： $3\mathrm{mL} / 500\mathrm{mg}$ ，或相当者。

# 5 仪器与设备

5.1 气相色谱—质谱仪：配有负化学源。

5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.3 固相萃取装置。  
5.4 均质器。  
5.5 旋转蒸发器。  
5.6 氮气浓缩仪。  
5.7 具塞离心管： $50~\mathrm{mL},100~\mathrm{mL}$   
5.8 浓缩瓶： $50~\mathrm{mL},100~\mathrm{mL}$   
5.9 移液管： $1\mathrm{mL},2\mathrm{mL},5\mathrm{mL},10\mathrm{mL}$

# 6 试样制备与保存

# 6.1 取样部位

样品取样部位按GB2763的规定执行。在抽样及制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样制备

# 6.2.1 粮谷

取有代表性样品量 $500\mathrm{g}$ , 用磨碎机全部磨碎并通过 $2.0\mathrm{mm}$ 圆孔筛。混匀, 均分成两份作为试样, 分装入洁净的容器内, 密闭, 标明标记。

# 6.2.2 水果或蔬菜

取有代表性样品 $500\mathrm{g}$ , 切碎后, 依次用食品捣碎机将样品加工成浆状。混匀, 装入洁净的容器内, 密闭, 标明标记。

# 6.2.3 肉及肉制品

取有代表性样品 $500\mathrm{g}$ , 经捣碎机充分捣碎均匀, 装入洁净容器内作为试样。密闭, 标明标记。

# 6.2.4 蜂蜜及蜂制品

取有代表性样品量 $500 \mathrm{~g}$ , 对无结晶的蜂蜜样品将其搅拌均匀; 对有结晶析出的蜂蜜样品, 在密闭情况下, 将样品瓶置于不超过 $60^{\circ} \mathrm{C}$ 的水浴中温热, 振荡, 待样品全部融化后搅匀, 迅速冷却至室温, 在融化时必须注意防止水分挥发。装入洁净的容器, 密闭, 标明标记。

# 6.2.5 茶叶

取有代表性样品 $500\mathrm{g}$ , 用磨碎机全部磨碎并通过 $2.0\mathrm{mm}$ 圆孔筛。混匀, 装入洁净的洁净容器内, 密闭, 标明标记。

# 6.2.6 坚果

取有代表性样品 $500\mathrm{g}$ , 用磨碎机全部磨碎。混匀, 均分成2份作为试样, 分装入洁净的洁净容器内, 密闭, 标明标记。

# 6.2.7 水产品

取有代表性样品 $500\mathrm{g}$ ，从所取样品中取出约 $1\mathrm{kg}$ ，经捣碎机充分捣碎均匀，均分成2份，分别装入洁净容器内作为试样。密封并标明标记。

# 6.3 试样保存

粮谷类试样于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存；水果和蔬菜及其他类试样于 $-18^{\circ}\mathrm{C}$ 以下冷冻保存。

# 7 分析步骤

# 7.1 提取

称取 $10\mathrm{g}$ 试样(精确至 $0.1\mathrm{g}$ )于 $100\mathrm{mL}$ 具塞离心管中，加入 $10\mathrm{mL}$ 水，准确加入 $50\mathrm{mL}$ 饱和乙腈，

用均质器高速匀浆提取 $2\mathrm{min}$ (酱油和蜂蜜仅需剧烈振荡 $10\mathrm{min}$ ), 再加入 $5\mathrm{g}$ 氯化钠, 剧烈震荡 $10\mathrm{min}$ , $3000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。

# 7.2 净化

# 7.2.1 液/液分配净化

取上层提取液 $20 \mathrm{~mL}$ (菠菜、藕、草莓)或 $10 \mathrm{~mL}$ (花生、鸡肉、猪肉、鳕鱼、蜂蜜、板栗、茶叶和酱油)转移至 $50 \mathrm{~mL}$ 具塞离心管中,加入 $10 \mathrm{~mL}$ 饱和正己烷,振摇 $3 \mathrm{~min}$ ,静置分层,弃去上层正己烷相,再用 $10 \mathrm{~mL}$ 正己烷重复操作一次,弃去正己烷相;下层乙腈相收集于 $100 \mathrm{~mL}$ 浓缩瓶中,于 $40^{\circ} \mathrm{C}$ 水浴中浓缩至近 $1 \mathrm{~mL}$ 。

# 7.2.2 固相萃取(SPE)净化

使用前用 $5 \mathrm{~mL}$ 丙酮一正己烷预淋 Florisil 柱。将样液倾入柱中, 用 $12 \mathrm{~mL}$ 丙酮一正己烷进行洗脱, 控制流速小于等于 $2 \mathrm{~mL} / \mathrm{min}$ 。收集全部洗脱液于 $50 \mathrm{~mL}$ 浓缩瓶中, 于 $40^{\circ} \mathrm{C}$ 水浴中浓缩至近干, 氮气吹干。用丙酮一正己烷溶解并定容至 $1.0 \mathrm{~mL}$ , 供气相色谱一质谱仪测定。

# 7.3 测定

# 7.3.1 气相色谱一质谱参考条件

a）色谱柱：HP-5MS石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度：初始温度为 $70^{\circ}\mathrm{C}$ ，以 $30^{\circ}\mathrm{C / min}$ 程序升温至 $200^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ ，再以 $50^{\circ}\mathrm{C / min}$ 程序升温至 $270^{\circ}\mathrm{C}$ ，保持 $4\mathrm{min}$   
c) 进样口温度: $250^{\circ} \mathrm{C}$ ;   
d）色谱—质谱接口温度： $280^{\circ}\mathrm{C}$   
e）载气：氦气，纯度 $\geqslant 99.999\%$ ，恒压模式，柱头压 $1.45\mathrm{MPa}$   
f）进样量： $1\mu \mathrm{L}$   
g）进样方式：无分流进样， $0.65\mathrm{min}$ 后开阀；  
h）电离方式：负化学电离；  
i）离子源温度： $150^{\circ}\mathrm{C}$   
j） 四极杆温度： $150^{\circ}\mathrm{C}$   
k）反应气：甲烷，纯度 $\geq 99.99\%$   
1）测定方式：选择离子监测方式；  
m) 选择监测离子 $(m / z)$ : 定量 117, 定性 217、275、295;  
n）溶剂延迟： $4.0\mathrm{min}$

# 7.3.2 色谱测定与确证

7.3.2.1 根据样液中被测物含量情况，选定浓度相近的标准工作溶液，对标准工作溶液与样液等体积参插进样测定，标准工作溶液和待测样液中四氟醚唑的响应值均应在仪器检测的线性范围内。  
7.3.2.2 如果样液与标准工作溶液的选择离子色谱图中，在相同保留时间有色谱峰出现，并且在扣除背景后的样品质量色谱中，所选离子均出现，所选择离子的丰度比与标准品对应离子的丰度比，其值在允许范围内（允许范围见表1）。在7.3.1条件下四氟醚唑保留时间是 $19.71\mathrm{min}$ ，其监测离子 $(m / z)$ 为117、217、275、295（其丰度比为 $100:38:11:27$ ）对其进行确证；根据定量离子 $m / z117$ 对其进行外标法定量。在7.3.1条件下四氟醚唑标准物的气相色谱—质谱总离子流色谱图和全扫描质谱图见图A.1和图B.1。

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按(1)式计算试样中四氟醚唑残留量。

$$
X = \frac {h \times c \times V}{h _ {s} \times m} \tag {1}
$$

式中：

$X$ ——试样中四氟醚唑残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$h$ ——样液中四氟醚唑的色谱峰高；

$c$ ——标准工作液中四氟醚唑的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$h_s$ ——标准工作液中四氟醚唑的色谱峰高；

$m$ ——最终样液所代表的试样质量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 C 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 D 的要求。  
10 定量限和回收率  
10.1 定量限

本方法四氟醚唑的定量限为 $2\mu \mathrm{g} / \mathrm{kg}$

10.2 回收率

当添加水平为 $0.002\mathrm{mg / kg},0.004\mathrm{mg / kg},0.01\mathrm{mg / kg}$ 时，四氟醚唑的添加回收率参见附录E。

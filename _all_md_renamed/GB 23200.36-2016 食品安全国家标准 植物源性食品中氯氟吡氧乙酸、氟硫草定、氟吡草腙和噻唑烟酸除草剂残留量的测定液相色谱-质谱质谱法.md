# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2214—2008《进出口植物源食品中氟草烟、氟硫草啶、氟吡草腙和噻草啶除草剂残留量的测定液相色谱一质谱/质谱法》。与SN/T2214—2008相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；

——标准名称“进出口植物源食品”改为“植物源性食品”；

——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T2214—2008。

# 食品安全国家标准

# 植物源性食品中氯氟吡氧乙酸、氟硫草定、氟吡草腺和噻唑烟酸除草剂残留量的测定 液相色谱一质谱/质谱法

# 1 范围

本标准规定了植物源性食品中氯氟吡氧乙酸、氟硫草定、氟吡草腙和噻唑烟酸4种吡啶类除草剂的液相色谱—质谱/质谱方法。

本标准适用于大白菜、玉米和橙中氯氟吡氧乙酸、氟硫草定、氟吡草腙和噻唑烟酸4种吡啶类除草剂残留的确证和定量测定，其他食品可参照执行。

# 3 原理

试样中残留的氯氟吡氧乙酸、氟硫草定、氟吡草腙和噻唑烟酸用乙腈提取，经Supelclean $\mathrm{C}_{18}$ 固相萃取柱净化，用液相色谱一质谱/质谱检测和确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
4.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
4.1.3 甲酸（HCOOH）：色谱纯。  
4.1.4 氯化钠 $(\mathrm{NaCl})$ 。

# 4.2 溶液配制

4.2.1 0.1%甲酸的水溶液:准确吸取 $1 \mathrm{~mL}$ 甲酸,转移入 $1000 \mathrm{~mL}$ 容量瓶中,用水定容至刻度,混合均匀。  
4.2.2 $0.1\%$ 甲酸—甲醇 $(50 + 50$ ，体积比）：量取 $0.1\%$ 甲酸的水溶液 $500~\mathrm{mL}$ 至 $1000~\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度，混合均匀。

# 4.3 标准品

4.3.1 吡啶类除草剂标准品：参见附录A。

# 4.4 标准溶液配制

4.4.1 标准储备溶液：准确称取适量的氯氟吡氧乙酸、氟硫草定、噻唑烟酸，分别用甲醇溶解并定容至棕色容量瓶中，浓度相当于 $500\mathrm{mg / L}$ ，储备液可在 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 条件下避光存放6个月。  
4.4.2 混合标准中间溶液：准确吸取适量氯氟吡氧乙酸、氟硫草定、噻唑烟酸标准储备溶液于棕色容量

瓶中, 并加入与其他吡啶类除草剂等质量浓度的氟吡草腙, 用甲醇定容至刻度, 各种吡啶类除草剂的浓度为 $1 \mathrm{mg} / \mathrm{L}$ , 此中间溶液可在 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 条件下避光存放 1 个月。

4.4.3 混合标准工作溶液: 根据需要, 使用前吸取一定量的混合标准中间溶液, 用空白样品基质溶液配制。  
4.5 材料  
4.5.1 固相萃取柱: Supelclean $\mathrm{C}_{18}, 3 \mathrm{~mL} / 500 \mathrm{mg}$ , 或相当者。使用前用 $5 \mathrm{~mL}$ 乙腈处理, 保持柱体湿润。  
4.5.2 微孔滤膜： $0.22\mu \mathrm{m}$ ，有机相。

# 5 仪器和设备

5.1 液相色谱—质谱/质谱仪，配ESI离子源。  
5.2 天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
5.3 粉碎机。  
5.4 聚丙烯离心管： $15\mathrm{mL}$ 和 $50~\mathrm{mL}$ ，具塞。  
5.5 离心机： $5000\mathrm{r / min}$   
5.6 涡旋振荡器。  
5.7 吹氮浓缩仪。  
5.8 固相萃取装置

# 6 试样制备与保存

6.1 取样部位

样品取样部位按 GB 2763 的规定执行。在取样和制样的操作过程中, 应防止样品受到污染或发生残留物含量的变化。  
6.2 试样制备   
6.2.1 大白菜和橙

取代表性样品 $500 \mathrm{~g}$ (不可用水洗), 切碎后用捣碎机将样品加工成浆状, 混匀, 装入洁净容器, 密封并标明标记。

6.2.2 玉米

取代表性样品 $500\mathrm{g}$ ，充分粉碎，混匀，装入洁净容器内，密封并标明标记。

6.3 试样保存

试样于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 冷藏保存。

# 7 测定步骤

7.1 提取

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 样品于 $50\mathrm{mL}$ 的离心管中，加入适量去离子水浸润片刻。加入 $10\mathrm{mL}$ 乙腈，在涡旋振荡器上混合 $3\mathrm{min}$ ，于 $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将上层清液转移至另一支离心管中。分别用 $10\mathrm{mL}, 5\mathrm{mL}$ 乙腈重复以上提取过程，合并提取液。加入适量氯化钠，充分振荡，于 $5000\mathrm{r / min}$ 离心 $3\mathrm{min}$ 。取上层乙腈提取液 $5\mathrm{mL}$ 至 $15\mathrm{mL}$ 离心管中，于 $40^{\circ}\mathrm{C}$ 吹氮浓缩至 $2\mathrm{mL}$ ，充分混匀。

7.2 净化

将样品提取液转入 $\mathrm{C}_{18}$ 固相萃取柱中。用 $4\mathrm{mL}$ 乙腈洗脱,从上样开始收集全部流出液(约 $6\mathrm{mL}$ ), 整个固相萃取净化过程控制流速不超过 1 滴/s。流出液于 $40^{\circ}\mathrm{C}$ 氮气吹至近干,用 $0.1\%$ 甲酸一甲醇溶

液 $(50 + 50$ ，体积比)定容至 $1.0\mathrm{mL}$ ，涡旋混匀后，过 $0.22\mu \mathrm{m}$ 微孔滤膜，供液相色谱一质谱/质谱测定。

# 8 结果计算和表述

按式(1)计算试样中4种吡啶类除草剂的含量。

$$
X = \frac {A \times C _ {s} \times V}{A _ {s} \times m}. \tag {1}
$$

式中：

$X$ ——样品中4种吡啶类除草剂的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——样液中吡啶类除草剂的峰面积；  
$A_{s}$ ——标准工作溶液中吡啶类除草剂的峰面积；  
$C_s$ ——从标准工作曲线所得4种吡啶类除草剂的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$   
$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；  
$m$ ——最终样液代表的试样质量，单位为克 $(\mathrm{g})$ 。  
计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 C 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 D 的要求。

# 10 定量限和回收率

# 10.1 定量限

# 10.1.1 大白菜和橙基质

氯氟吡氧乙酸、氟硫草定、氟吡草腙和噻唑烟酸的定量限均为 $5\mu \mathrm{g / kg}$

# 10.1.2 玉米基质

氯氟吡氧乙酸和氟吡草腙的定量限为 $10\mu \mathrm{g / kg}$ ，氟硫草定和噻唑烟酸的定量限为 $5\mu \mathrm{g / kg}$ 。

# 10.2 回收率

在 $0.005\mathrm{mg / kg}\sim 0.04\mathrm{mg / kg}$ 添加浓度范围内，4种吡啶类除草剂的添加回收率参见附录E。

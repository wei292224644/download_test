# 1 范围

本标准适用于以碳酸钠或碳酸氢钠、可溶性铜盐为原料制得或以阴极铜（电解铜）、液氨、二氧化碳为原料制得的食品营养强化剂碳酸铜。

# 2 化学名称、分子式和相对分子质量

# 2.1 化学名称

碱式碳酸铜

# 2.2 分子式

$$
\mathrm {C u C O} _ {3} \cdot \mathrm {C u} (\mathrm {O H}) _ {2}
$$

# 2.3 相对分子质量

221.11（按2018年国际相对原子质量）

# 检验方法

警告：本标准的检验方法中使用的部分试剂具有毒性或腐蚀性，按国家相关规定操作，使用时需小心谨慎。若溅到皮肤上应立即用水冲洗，严重者应立即就医。在使用挥发性酸时，应在通风橱中进行。

# A.1 一般规定

本标准检验方法中所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 盐酸。  
A.2.1.2 氨水。  
A.2.1.3 亚铁氰化钾。  
A.2.1.4 氢氧化钙。  
A.2.1.5 氨水溶液 $(2 + 3)$ ：量取 $40~\mathrm{mL}$ 氨水，缓慢加入到 $60~\mathrm{mL}$ 水中，混匀。  
A.2.1.6 亚铁氰化钾溶液 $(100\mathrm{g / L})$ : 称取 $10\mathrm{g}$ 亚铁氰化钾, 用水溶解并稀释至 $100\mathrm{mL}$ 。临用时现配。  
A.2.1.7 盐酸溶液 $(1 + 4)$ ：量取 $20~\mathrm{mL}$ 盐酸，缓慢加入到 $80~\mathrm{mL}$ 水中，混匀。  
A.2.1.8 盐酸溶液 $(1 + 2)$ ：量取 $20~\mathrm{mL}$ 盐酸，缓慢加入到 $40~\mathrm{mL}$ 水中，混匀。  
A.2.1.9 氢氧化钙溶液 $(3\mathrm{g} / \mathrm{L})$ ：称取 $0.3\mathrm{g}$ 氢氧化钙，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
A.2.1.10 铁片：光亮未生锈。

# A.2.2 仪器和设备

分析天平：感量 $0.01\mathrm{g}$

# A.2.3 铜的鉴别

A.2.3.1 称取 $1 \mathrm{~g}$ 试样 (精确至 $0.01 \mathrm{~g}$ , 加入 $20 \mathrm{~mL}$ 水, 加入少量盐酸 (A.2.1.1) 酸化试样溶液。将一个光亮未生锈的铁片 (A.2.1.10) 放入酸化后的试样溶液中, 在铁片表面会形成红色的金属铜薄膜。  
A.2.3.2 称取 $1 \mathrm{~g}$ 试样（精确至 $0.01 \mathrm{~g}$ ），加入 $20 \mathrm{~mL}$ 水，加入过量的氨水溶液（A.2.1.5），先生成蓝色的沉淀，然后转变为深蓝色的溶液。  
A.2.3.3 称取 $1 \mathrm{~g}$ 试样(精确至 $0.01 \mathrm{~g}$ ), 加入 $20 \mathrm{~mL}$ 水, 加入少量盐酸(A.2.1.1)酸化试样溶液, 加入亚铁氰化钾溶液(A.2.1.6), 生成红棕色沉淀。此沉淀不溶于盐酸溶液(A.2.1.7)。

# A.2.4 碳酸盐的鉴别

取少量试样，加盐酸溶液(A.2.1.8)后可产生气体，该气体通入氢氧化钙溶液(A.2.1.9)中有白色沉淀产生。

# A.3 铜(Cu)含量的测定

# A.3.1 方法原理

在微酸性条件下，试样溶液中加入过量碘化钾与二价铜定量反应，以淀粉为指示剂，用硫代硫酸钠标准滴定溶液滴定析出的碘。

# A.3.2 试剂和材料

A.3.2.1 硫酸。

A.3.2.2 碳酸钠。

A.3.2.3 乙酸。

A.3.2.4 氟化钠。

A.3.2.5 碘化钾。

A.3.2.6 碳酸钠饱和溶液 $(20^{\circ} \mathrm{C})$ : 在易于溶解的温度下, 称取 $30 \mathrm{~g}$ 碳酸钠, 加入 $100 \mathrm{~mL}$ 热水, 冷却至形成固液两相的饱和溶液。

A.3.2.7 乙酸溶液 $(36\%)$ ：量取 $35.8~\mathrm{mL}$ 乙酸，加水稀释至 $100~\mathrm{mL}$

A.3.2.8 氟化钠饱和溶液 $(20^{\circ}\mathrm{C})$ ：在易于溶解的温度下，称取 $10\mathrm{g}$ 氟化钠，加入 $100\mathrm{mL}$ 热水，冷却至形成固液两相的饱和溶液。

A.3.2.9 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$

A.3.2.10 淀粉指示液 $(10\mathrm{g / L})$ ：称取 $0.5\mathrm{g}$ 可溶性淀粉，加少许水，调成糊状，倒入 $50~\mathrm{mL}$ 沸水中调匀，煮沸。临用时现配。

# A.3.3 仪器和设备

分析天平：感量 $0.0001\mathrm{g}$

# A.3.4 分析步骤

称取约 $0.4\mathrm{g}$ 试样(精确至 $0.0001\mathrm{g}$ )，置于碘量瓶中，加少量水润湿。加入 $0.4\mathrm{mL}$ 硫酸(A.3.2.1)溶解试样，加水至约 $100~\mathrm{mL}$ 。逐滴加入碳酸钠饱和溶液(A.3.2.6)，直至有微量沉淀为止。加入 $4\mathrm{mL}$ 乙酸溶液(A.3.2.7)，加入 $2\mathrm{mL}$ 氟化钠饱和溶液(A.3.2.8)，再加入 $3\mathrm{g}$ 碘化钾(A.3.2.5)。用硫代硫酸钠标准滴定溶液(A.3.2.9)滴定至溶液为淡黄色，加入 $3\mathrm{mL}$ 淀粉指示液(A.3.2.10)，继续滴定至乳白色。同时做空白试验，空白试验应与测试平行进行，并采用相同的试验步骤，取相同量的所有试剂，但不加样品。

# A.3.5 结果计算

铜(Cu)的质量分数 $w_{1}$ 按式(A.1)计算。

$$
w _ {1} = \frac {\left(V _ {1} - V _ {2}\right) \times c \times M}{m \times 1000} \times 100 \% \tag{A.1}
$$

式中：

$V_{1}$ ——滴定试样溶液所消耗的硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴定空白溶液所消耗的硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫代硫酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——铜的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M = 63.546]$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——单位换算系数。

计算结果保留3位有效数字。取平行测定结果的算术平均值为测定结果，2次平行测定结果的绝对差值不大于 $0.3\%$ 。

# A.4 氯化物（以Cl计）的测定

# A.4.1 方法原理

在硝酸介质中，氯离子与银离子生成难溶的氯化银。当氯离子含量较低时，在一定时间内氯化银呈悬浮体，使溶液浑浊，可用于氯化物的目视比浊法测定。

# A.4.2 试剂和材料

A.4.2.1 硝酸。  
A.4.2.2 硝酸银。  
A.4.2.3 氯化钠。  
A.4.2.4 硝酸溶液 $(1 + 1)$ ：量取 $50~\mathrm{mL}$ 硝酸，缓慢加入到 $50~\mathrm{mL}$ 水中，混匀。  
A.4.2.5 硝酸银溶液 $(17\mathrm{g} / \mathrm{L})$ ：称取 $1.7\mathrm{g}$ 硝酸银，用水溶解并稀释至 $100~\mathrm{mL}$ 。贮存于棕色瓶中。  
A.4.2.6 氯化物标准溶液I $(0.1\mathrm{mg / mL})$ ：按照GB/T602规定的方法配制。  
A.4.2.7 氯化物标准溶液 $\mathrm{II}(0.01\mathrm{mg/mL})$ : 吸取氯化物标准溶液 I (A.4.2.6) $10.0\mathrm{mL}$ , 加水稀释至 $100\mathrm{mL}$ 。  
A.4.2.8 不含氯化物的碳酸铜溶液

称取 $1\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )置于 $150~\mathrm{mL}$ 烧杯中，用少量水润湿，加适量硝酸溶液(A.4.2.4)至试样溶解，转移至 $100~\mathrm{mL}$ 容量瓶中，加入 $5\mathrm{mL}$ 硝酸银溶液(A.4.2.5)，用水稀释至刻度，摇匀。放置 $12\mathrm{h}\sim 18\mathrm{h}$ 后，用玻璃砂坩埚抽滤，收集滤液于试剂瓶中。

# A.4.3 仪器和设备

A.4.3.1 分析天平：感量 $0.01\mathrm{g}$ 。  
A.4.3.2 玻璃砂坩埚：滤板孔径为 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$

# A.4.4 分析步骤

称取 $1\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )，置于 $150~\mathrm{mL}$ 烧杯中，用少量水润湿，加适量硝酸溶液(A.4.2.4)至试样溶解，转移至 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。吸取 $10~\mathrm{mL}$ 试验溶液，置于 $50~\mathrm{mL}$ 比色管中，加入 $2\mathrm{mL}$ 硝酸溶液(A.4.2.4)，加水至约 $40~\mathrm{mL}$ ，加入 $1\mathrm{mL}$ 硝酸银溶液(A.4.2.5)，用水稀释至刻度，摇匀，避光放置 $2\mathrm{min}$ 。试样管与标准管进行比浊。

标准管的制备：吸取 $10~\mathrm{mL}$ 不含氯化物的碳酸铜溶液(A.4.2.8)和氯化物标准溶液Ⅱ（A.4.2.7） $5\mathrm{mL}$ ，与试样管同时同样处理。

# A.4.5 结果判定

如果试样管溶液所呈浊度大于标准管溶液，则试样中氯化物（以Cl计）含量 $>0.05\%$ ；反之，氯化物（以Cl计)含量 $\leqslant 0.05\%$ 。

# A.5 硫酸盐（以 $\mathbf{SO}_4$ 计）的测定

# A.5.1 方法原理

在盐酸介质中，硫酸根离子与钡离子生成难溶的硫酸钡，当硫酸根离子含量较低时，在一定时间内硫酸钡呈悬浮体，使溶液浑浊，加入硫酸盐乙醇溶液作为晶种液，可以使粒子大小适当和均匀，可用于硫酸盐的目视比浊法测定。

# A.5.2 试剂和材料

A.5.2.1 盐酸。  
A.5.2.2 95%乙醇。  
A.5.2.3 氯化钡。  
A.5.2.4 硫酸钾。  
A.5.2.5 氯化钡溶液 $(250\mathrm{g / L})$ ：称取 $25\mathrm{g}$ 氯化钡，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
A.5.2.6 乙醇溶液 $(30\%)$ ：量取 $95\%$ 乙醇(A.5.2.2) $31.6~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ 。  
A.5.2.7 硫酸钾乙醇溶液 $(0.2\mathrm{g} / \mathrm{L})$ ：称取 $0.02\mathrm{g}$ 硫酸钾，用乙醇溶液(A.5.2.6)溶解并稀释至 $100~\mathrm{mL}$ 。  
A.5.2.8 硫酸盐标准溶液I $(0.1\mathrm{mg / mL})$ ：按照GB/T602规定的方法配制。  
A.5.2.9 硫酸盐标准溶液 $\mathrm{II}(0.01\mathrm{mg / mL})$ ：吸取硫酸盐标准溶液I（A.5.2.8) $10.0~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ 。  
A.5.2.10 不含硫酸盐的碳酸铜溶液

称取 $1\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），置于 $150~\mathrm{mL}$ 烧杯中，用少量水润湿，加适量盐酸(A.5.2.1)至试样溶解，转移至 $100~\mathrm{mL}$ 容量瓶中，加入 $10~\mathrm{mL}95\%$ 乙醇，在不断振摇下滴加 $5\mathrm{mL}$ 氯化钡溶液(A.5.2.5)，用水稀释至刻度，摇匀。放置 $12\mathrm{h}\sim 18\mathrm{h}$ 后，用玻璃砂坩埚抽滤，收集滤液于试剂瓶中。

# A.5.3 仪器和设备

A.5.3.1 分析天平：感量 $0.01\mathrm{g}$   
A.5.3.2 玻璃砂坩埚：滤板孔径为 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$

# A.5.4 分析步骤

称取 $1\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )，置于 $150~\mathrm{mL}$ 烧杯中，用少量水润湿，加适量盐酸(A.5.2.1)至试样溶解，转移至 $100~\mathrm{mL}$ 容量瓶中，加入 $10~\mathrm{mL}95\%$ 乙醇，用水稀释至刻度，摇匀。将 $0.25~\mathrm{mL}$ 硫酸钾乙醇溶液(A.5.2.7)与 $1\mathrm{mL}$ 氯化钡溶液(A.5.2.5)混合，置于 $25~\mathrm{mL}$ 比色管中（晶种液），放置 $1\mathrm{min}$ 后，加入 $10~\mathrm{mL}$ 试验溶液，加入 $0.5~\mathrm{mL}$ 盐酸(A.5.2.1)，用水稀释至刻度，摇匀，放置 $2\mathrm{min}$ 。试样管与标准管进行比浊。

标准管的制备：吸取 $10~\mathrm{mL}$ 不含硫酸盐的碳酸铜溶液(A.5.2.10)和硫酸盐标准溶液Ⅱ（A.5.2.9） $5\mathrm{mL}$ ，与试样管同时同样处理。

# A.5.5 结果判定

如果试样管溶液所呈浊度大于标准管溶液，则试样中硫酸盐（以 $\mathrm{SO}_4$ 计）含量 $>0.05\%$ ；反之，硫酸盐（以 $\mathrm{SO}_4$ 计）含量 $\leqslant 0.05\%$ 。

# A.6 盐酸不溶物的测定

# A.6.1 方法原理

试样溶于盐酸溶液后，经过滤、洗涤、烘干、称重后确定盐酸不溶物。

# A.6.2 试剂和材料

A.6.2.1 盐酸。

A.6.2.2 盐酸溶液 $(1 + 1)$ ：量取 $200~\mathrm{mL}$ 盐酸，缓慢加入到 $200~\mathrm{mL}$ 水中，混匀。

# A.6.3 仪器和设备

A.6.3.1 分析天平：感量 $0.0001\mathrm{g}$   
A.6.3.2 玻璃砂坩埚：滤板孔径为 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$   
A.6.3.3 电热恒温干燥箱：精度 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 。

# A.6.4 分析步骤

称取 $50\mathrm{g}$ 试样（精确至 $0.0001\mathrm{g}$ ），置于 $500~\mathrm{mL}$ 烧杯中，加入 $100~\mathrm{mL}$ 水，缓慢加入 $200~\mathrm{mL}$ 盐酸溶液(A.6.2.2)，不断搅拌直至可溶部分完全溶解。用已于 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 下干燥至质量恒定的玻璃砂坩埚抽滤，并用水洗至中性。将玻璃砂坩埚和盐酸不溶物在 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 电热恒温干燥箱中干燥至质量恒定。

# A.6.5 结果计算

盐酸不溶物的质量分数 $w_{2}$ 按式(A.2)计算。

$$
w _ {2} = \frac {m _ {1} - m _ {2}}{m} \times 100 \% \tag{A.2}
$$

式中：

$m_{1}$ ——盐酸不溶物及玻璃砂坩埚干燥后的质量，单位为克(g)；

$m_{2}$ ——玻璃砂坩埚的质量，单位为克 $(\mathrm{g})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

计算结果保留3位有效数字。取平行测定结果的算术平均值为测定结果，2次平行测定结果的绝对差值不大于算术平均值的 $15\%$ 。

# A.7 钠、铁、锌、钙、镍、铅、总铬、镉、总砷的测定

# A.7.1 方法原理

试样加硝酸溶解后，用钇标准溶液作内标，在电感耦合等离子体原子发射光谱仪相应的波长处测量其光谱强度，采用内标法计算元素的含量。

# A.7.2 试剂和材料

A.7.2.1 硝酸：优级纯。  
A.7.2.2 氩气（Ar）：纯度大于 $99.9\%$

# A.7.4 分析步骤

# A.7.4.1 试样溶液的制备

称取 $0.5\mathrm{g}\sim 1\mathrm{g}$ 试样(精确至 $0.0001\mathrm{g}$ )，用少量水润湿并分散试样，缓慢加入硝酸(A.7.2.1)至试样完全溶解，然后转移至 $100~\mathrm{mL}$ 容量瓶中，用一级水稀释至刻度，摇匀。

同时做空白试验，空白试验溶液除不加试样外，其他加入试剂的种类和量与试样溶液相同。

# A.7.4.2 工作曲线的绘制

分别吸取 $0\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},4.00\mathrm{mL},8.00\mathrm{mL}$ 混合标准溶液(A.7.2.7)，置于5个 $100~\mathrm{mL}$ 容量瓶中，分别加入 $5\mathrm{mL}$ 硝酸(A.7.2.1)，用一级水稀释至刻度，摇匀。

从待测元素每个标准溶液的光谱强度中减去标准空白溶液的光谱强度，以每个标准溶液中待测元素的质量浓度为横坐标，对应的光谱强度为纵坐标分别绘制各待测元素的标准曲线。

# A.7.5 结果计算

金属元素的质量分数 $w_{3}(\mathrm{mg / kg})$ 按式(A.3)计算。

$$
w _ {3} = \frac {\left(\rho_ {1} - \rho_ {0}\right) \times V \times f}{m \times 1 0 0 0} \times 1 0 0 0 \quad \dots \tag {A.3}
$$

式中：

$\rho_{1}$ ——从工作曲线上查得试验溶液中待测元素的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$\rho_0$ ——从工作曲线上查得空白试验溶液中待测元素的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样消化液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样稀释倍数；

$m$ ——试样的质量，单位为克(g)；

1000 ——单位换算系数。

计算结果保留3位有效数字。取平行测定结果的算术平均值为测定结果，2次平行测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# 1 范围

本文件规定了牛、羊、猪、鸡肌肉和猪肝脏、肾脏、脂肪中类固醇激素类药物（17β-雌二醇、雌三醇、炔雌醇、雌酮)残留量检测的制样和气相色谱质谱测定方法。

本文件适用于牛、羊、猪、鸡肌肉和猪肝脏、肾脏、脂肪中17β-雌二醇、雌三醇、炔雌醇、雌酮残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语与定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的 $17\beta$ 雌二醇、雌三醇、炔雌醇、雌酮，经酶解、提取、净化和衍生，气相色谱-质谱法测定，外标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.3 乙酸乙酯 $\mathrm{(CH_3COOC_2H_5)}$

5.1.4 正己烷 $(\mathrm{C_6H_{14}})$

5.1.5 甲苯 $(\mathrm{C_6H_5CH_3})$ 。

5.1.6 氢氧化钠(NaOH)。

5.1.7 醋酸钠 $(\mathrm{NaC_2H_3O_2}\cdot 3\mathrm{H_2O})$

5.1.8 醋酸铵 $(\mathrm{NH_4C_2H_3O_2})$ 。

5.1.9 冰醋酸 $(\mathrm{C_2H_4O_2})$ 。

5.1.10 二硫赤藓糖醇 $(\mathrm{C_4H_{10}O_2S_2}$ ，DTE）。

5.1.11 三甲基碘硅烷 $\mathrm{(C_3H_9ISi, TMIS)}$ 。

5.1.12 葡萄糖醛酸酶/芳香基硫酸酯酶

5.1.13 N-甲基三甲基硅基三氟乙酰氨(MSTFA)。

# 5.2 标准品

5.2.1 $17\beta$ -雌二醇（ $17\beta$ -Estradiol， $\mathrm{C_{18}H_{24}O_2}$ ，CAS号：50-28-2）：含量 $\geq 98.0\%$   
5.2.2 雌三醇(Estriol， $\mathrm{C_{18}H_{24}O_3}$ ，CAS号：50-27-1)：含量 $\geq 98.0\%$   
5.2.3 炔雌醇(Ethinylestradiol, $\mathrm{C_{20}H_{24}O_2}$ ,CAS号：57-63-6）：含量 $\geq 98.0\%$   
5.2.4 雌酮(Estrone， $\mathrm{C_{18}H_{22}O_2}$ ，CAS号：200-164-5）：含量 $\geq 98.0\%$

# 5.3 溶液配制

5.3.1 氢氧化钠溶液 $(1.0\mathrm{mol} / \mathrm{L})$ ：取氢氧化钠 $40\mathrm{g}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。  
5.3.2醋酸钠缓冲液 $(\mathrm{pH}5.2)$ ：取醋酸钠 $19.0\mathrm{g}$ ，用水溶解并稀释至 $500~\mathrm{mL}$ ，用醋酸调节 $\mathrm{pH}$ 至5.2。  
5.3.3 醋酸铵溶液 $(2.0\mathrm{mol} / \mathrm{L})$ ：取醋酸铵 $46.25\mathrm{g}$ ，用水溶解并稀释至 $300~\mathrm{mL}$ 。  
5.3.4 甲醇水溶液：取 $400~\mathrm{mL}$ 甲醇，加水至 $1000~\mathrm{mL}$   
5.3.5 乙酸乙酯正己烷溶液：取乙酸乙酯 $10\mathrm{mL}$ ，加正己烷使溶解并稀释至 $100~\mathrm{mL}$   
5.3.6 衍生化试剂：称取二硫赤藓糖醇 $0.01\mathrm{g}$ ，用N-甲基三甲基硅基三氟乙酰氨 $5\mathrm{mL}$ 溶解，在液面下加入三甲基碘硅烷 $10~\mu \mathrm{L}$ ，混匀， $2^{\circ}\mathrm{C} \sim 8^{\circ}\mathrm{C}$ 放置过夜，避光防潮密封保存。衍生化试剂应无色，如果发生棕红色等颜色变化，表明试剂失效。

# 5.4 标准溶液的制备

5.4.1 标准储备液 $(100\mu \mathrm{g / mL})$ ：取 $17\beta$ 雌二醇、雌三醇、炔雌醇和雌酮标准品各约 $10\mathrm{mg}$ ，精密称定，分别加甲醇适量使溶解并稀释定容至 $100~\mathrm{mL}$ 容量瓶，配成浓度为 $100\mu \mathrm{g / mL}$ 的标准储备液。 $-18^{\circ}C$ 以下保存，有效期6个月。  
5.4.2 混合标准工作液 $(1\mu \mathrm{g / mL})$ ：分别精密量取 $17\beta$ 雌二醇、雌三醇、炔雌醇和雌酮标准储备液各 $1\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶，用甲醇稀释至刻度，配制成浓度均为 $1\mu \mathrm{g / mL}$ 的混合标准工作液。 $2^{\circ}C\sim 8^{\circ}C$ 保存，有效期1个月。

# 5.5 材料

5.5.1 $\mathrm{C_{18}}$ 固相萃取柱： $500\mathrm{mg / 3mL}$ ，或相当者。  
5.5.2 硅胶固相萃取柱： $500\mathrm{mg / 3mL}$ ，或相当者。

# 6 仪器和设备

6.1 气相色谱-质谱联用仪：配EI源。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 氮吹仪。  
6.4 固相萃取装置。  
6.5 移液器：量程 $200\mu \mathrm{L}\sim 1000\mu \mathrm{L}_{\circ}$   
6.6 移液器：量程 $20\mu \mathrm{L}\sim 100\mu \mathrm{L}_{\circ}$   
6.7 均质机。  
6.8 涡旋混合器。  
6.9 离心机： $10000\mathrm{r / min}$   
6.10 烘箱。  
6.11 pH计。  
6.12 旋转蒸发仪。  
6.13 鸡心瓶： $100~\mathrm{mL}$

# 7 试料的制备与保存

# 7.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存，3个月内进行分析检测。

# 8 测定步骤

# 8.1 提取

取试样 $5\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管，加醋酸钠缓冲液 $10~\mathrm{mL}$ ，均质 $1\mathrm{min}$ ，涡旋振荡 $2\mathrm{min}$ ，加葡萄糖醛酸酶/芳香基硫酸酯酶 $20~\mu \mathrm{L},50^{\circ}C$ 酶解 $2\textrm{h}$ ，加乙酸乙酯 $20~\mathrm{mL}$ ，涡旋振荡 $3\mathrm{min}$ $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液于 $100~\mathrm{mL}$ 鸡心瓶中，残渣中加乙酸乙酯 $20~\mathrm{mL}$ ，重复提取1次，合并上清液， $40^{\circ}C$ 水浴旋转蒸发至近干，用氢氧化钠溶液 $6\mathrm{mL}$ 分3次溶解洗涤鸡心瓶，洗液转入 $50~\mathrm{mL}$ 离心管中，加正己烷 $20~\mathrm{mL}$ ，涡旋振荡 $1\mathrm{min},10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集下层清液，加醋酸铵溶液 $1\mathrm{mL}$ ，用冰醋酸调 $\mathsf{pH}$ 至 $5.0\sim 5.2$ ，备用。

# 8.2 净化

取 $\mathrm{C_{18}}$ 固相萃取柱，依次用甲醇、水各 $5\mathrm{mL}$ 活化。取备用液，过柱，分别用水、甲醇水溶液各 $5\mathrm{mL}$ 淋洗，抽干，加甲醇 $5\mathrm{mL}$ ，洗脱，收集洗脱液， $50^{\circ}\mathrm{C}$ 水浴氮气吹干。加乙酸乙酯正己烷溶液 $5\mathrm{mL}$ 使溶解，备用；取硅胶固相萃取柱，加正己烷 $5\mathrm{mL}$ 活化，取备用液过柱，淋洗，抽干，加乙腈 $5\mathrm{mL}$ ，洗脱，收集洗脱液， $50^{\circ}\mathrm{C}$ 水浴氮气吹干。

# 8.3 衍生

加甲苯、衍生化试剂各 $100\mu \mathrm{L}$ 于上述氮气吹干的玻璃试管中，振荡混合，封口，于 $80^{\circ}C$ 烘箱中衍生 $60\mathrm{min}$ ，冷却，供GC-MS测定。

# 8.4 标准工作曲线制备

精密量取混合标准工作液适量，用甲醇稀释成浓度分别 $10~\mu \mathrm{g / L},50~\mu \mathrm{g / L},100~\mu \mathrm{g / L},200~\mu \mathrm{g / L}$ $500~\mu \mathrm{g / L},1000~\mu \mathrm{g / L}$ 系列标准溶液，各准确量取 $500~\mu \mathrm{L},50^{\circ}C$ 水浴氮气吹干，按步骤7.3进行衍生，冷却后再各加甲苯 $300~\mu \mathrm{L}$ 进行GC-MS测定。以 $17\beta$ -雌二醇、雌三醇、炔雌醇和雌酮各浓度为横坐标，相应的定量离子峰面积为纵坐标，绘制标准曲线或求线性回归方程。

# 8.5 测定

# 8.5.1 色谱参考条件

a）色谱柱：MS石英毛细管色谱柱 $(30\mathrm{m}\times 0.25\mathrm{mm},0.25\mu \mathrm{m})$ ，或相当者；  
b）载气为高纯 $\mathrm{He}$ ，恒流 $1.0\mathrm{mL / min}$   
c）进样口温度 $220^{\circ}\mathrm{C}$   
d) 不分流进样；  
e）进样体积： $1\mu \mathrm{L}$   
f）色谱柱起始温度 $100^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min};20^{\circ}\mathrm{C / min}$ 的升温速率升至 $200^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min};20^{\circ}\mathrm{C / min}$ 的升温速率升至 $260^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ ；再以 $20^{\circ}\mathrm{C / min}$ 的升温速率升至 $280^{\circ}\mathrm{C}$ ，保持 $3\mathrm{min}$ 。

# 8.5.2 质谱参考条件

a）离子源(EI)温度： $230^{\circ}\mathrm{C}$   
b）EM电压：高于调谐电压 $200\mathrm{V}$   
c） 电子能量： $70\mathrm{eV}$   
d) GC/MS传输线温度： $280^{\circ}\mathrm{C}$   
e）四极杆温度： $150~\mathrm{^\circ C}$   
f）选择离子监测(SIM)： $17\beta$ 雌二醇、雌三醇、炔雌醇和雌酮衍生物的监测离子见表1。

# 8.6 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V}{A _ {\mathrm {S}} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中相应的类固醇激素类药物残留量的数值，单位微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ ——标准工作液中相应的类固醇激素类药物含量的数值，单位微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$A$ ——试样中相应的类固醇激素类药物的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中相应的类固醇激素类药物的峰面积；

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1.0\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $1.0\mu \mathrm{g / kg}\sim 10.0\mu \mathrm{g / kg}$ 添加浓度水平的回收率在 $60\% \sim 110\%$

# 10.3 精密度

本方法的批内变异系数 $CV\leqslant 15\%$ ，批间变异系数 $CV\leqslant 20\%$

# 附录A

# 检验方法

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯试剂和GB/T6682中规定的三级水。试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 硝酸：分析纯。  
A.2.1.2 硫化钠：分析纯。  
A.2.1.3 硫酸溶液：取 $5\mathrm{mL}$ 浓硫酸，缓慢加入约 $80~\mathrm{mL}$ 水中，再加水定容至 $100~\mathrm{mL}$   
A.2.1.4 盐酸溶液：取 $50~\mathrm{mL}$ 浓盐酸，加水稀释定容至 $100~\mathrm{mL}$   
A.2.1.5 硫氰酸铵溶液：溶解 $8\mathrm{g}$ 硫氰酸铵于适量水中，用水稀释至 $100~\mathrm{mL}$ 。  
A.2.1.6 吡啶氢氧化钠溶液：溶解 $1.2\mathrm{g}$ 氢氧化钠于 $200~\mathrm{mL}$ 水中，再加入 $100~\mathrm{mL}$ 吡啶，混合。  
A.2.1.7 氨溶液：取氨水 $400~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。

# A.2.2 仪器和设备

A.2.2.1 水浴箱。  
A.2.2.2 分析天平：感量为 $0.1\mathrm{g}$ 和 $0.001\mathrm{g}$

# A.2.3 鉴别方法

# A.2.3.1 呈色试验

A.2.3.1.1 称取 $0.01 \mathrm{~g}$ (精确至 $0.001 \mathrm{~g}$ ) 氯化高铁血红素试样, 加入 $1 \mathrm{~mL}$ 硫酸溶液 (A.2.1.3) 和 $1 \mathrm{~mL}$ 硝酸 (A.2.1.1) 溶解后水浴蒸干。残留物加入 $10 \mathrm{~mL}$ 盐酸溶液 (A.2.1.4) 溶解, 加入数滴硫氰酸铵溶液 (A.2.1.5), 摇匀, 溶液应呈现暗红色。  
A.2.3.1.2 称取 $0.005\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 氯化高铁血红素试样, 加入 $10\mathrm{mL}$ 吡啶氢氧化钠溶液 (A.2.1.6) 溶解, 再加入 $0.1\mathrm{g}$ 硫化钠, 摇匀, 溶液应呈现暗红色。  
A.2.3.1.3 称取 $0.01 \mathrm{~g}$ (精确至 $0.001 \mathrm{~g}$ ) 氯化高铁血红素试样, 加入 $5 \mathrm{~mL}$ 硝酸后摇匀, 加热, 溶液应呈现黄色。冷却后, 用氨溶液 (A.2.1.7) 将 $\mathrm{pH}$ 调整至碱性, 溶液应呈现橙黄色。

# A.2.3.2 红外光谱法

采用溴化钾压片法，按照GB/T6040的规定进行试验，试样的红外光谱应与对照的谱图一致。对照谱图见附录B。

# A.3 氯化高铁血红素 $\left(\mathbf{C}_{34} \mathbf{H}_{32} \mathbf{ClFeN}_{4} \mathbf{O}_{4}\right)$ 含量的测定

# A.3.1 紫外分光光度法

# A.3.1.1 方法提要

氯化高铁血红素用 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液溶解后，采用紫外分光光度法测定，在 $386~\mathrm{nm}\pm 1~\mathrm{nm}$ 处有最大吸收峰，测定吸光度值 $A$ ，根据标准工作液浓度计算试样含量。

# A.3.1.2 试剂与材料

A.3.1.2.1 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：称取 $4.0\mathrm{g}$ 氢氧化钠，加入适量水溶解，用水定容至 $1\mathrm{L}$ 。

A.3.1.2.2 氯化高铁血红素标准物质（Hemin）： $\mathrm{C_{34}H_{32}ClFeN_4O_4}$ ，CAS：16009-13-5，纯度 $\geqslant 97\%$ 。

A.3.1.2.3 氯化高铁血红素标准储备液：准确称取氯化高铁血红素标准物质 $20\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ), 加 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液(A.3.1.2.1)定容至 $100\mathrm{mL}$ , 配制成浓度为 $200.0\mathrm{mg} / \mathrm{L}$ 的标准储备液。该溶液于棕色试剂瓶中 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 避光保存, 有效期 $3\mathrm{d}$ 。

A.3.1.2.4 氯化高铁血红素标准工作液：分别吸取适量氯化高铁血红素标准储备液（A.3.1.2.3），用 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液稀释配制成标准工作液。

# A.3.1.3 仪器和设备

A.3.1.3.1 紫外分光光度计。

A.3.1.3.2 分析天平：感量为 $0.1\mathrm{g}$ 和 $0.0001\mathrm{g}$

# A.3.1.4 分析步骤

称取适量的试样，加入 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶解并定量稀释成每 $1\mathrm{mL}$ 约含 $8\mu \mathrm{g}$ 氯化高铁血红素试样溶液。

取适量试样溶液，采用紫外分光光度法验证试样溶液的最大吸收峰应在 $386\mathrm{nm}\pm 1\mathrm{nm}$ 。确保标准工作曲线及试样样液的吸光度值 $A$ 在 $0.3\sim 0.7$ 。当吸光度值 $A$ 超过此线性范围时，可适当加大试样的稀释倍数。

分别取氯化高铁血红素标准溶液和试样液，以 $0.1\mathrm{mol} / \mathrm{L}$ 的氢氧化钠溶液为空白，在 $386~\mathrm{nm}\pm 1~\mathrm{nm}$ 波长处测定吸光度值 $A$ 。

# A.3.1.5 结果计算

氯化高铁血红素的质量分数 $W_{1}$ 按式(A.1)计算。

$$
W _ {1} = \frac {A \times c \times V}{A _ {\mathrm {s}} \times m \times 1000} \times 100 \% \tag{A.1}
$$

式中：

$A$ ——试样液中氯化高铁血红素的吸光度值；

$c$ ——氯化高铁血红素标准工作溶液的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——最终试样液的定容体积，单位为升 $(\mathrm{L})$ ；

$A_{\mathrm{s}}$ ——氯化高铁血红素标准工作溶液的吸光度值；

$m$ ——最终试样溶液所代表试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

# A.3.2 高效液相色谱法

# A.3.2.1 方法提要

氯化高铁血红素试样用 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液溶解提取后，用高效液相色谱仪测定，保留时间定性，峰面积外标法定量。

# A.3.2.2 试剂与材料

A.3.2.2.1 甲醇：色谱纯。

A.3.2.2.2 乙酸：色谱纯。

A.3.2.2.3 乙酸溶液：吸取 $6\mathrm{mL}$ 乙酸(A.3.2.2.2)，用水定容至 $1\mathrm{L}$ 。

A.3.2.2.4 氯化高铁血红素标准物质（Hemin）：见A.3.1.2.2。

A.3.2.2.5 氯化高铁血红素标准储备液：见A.3.1.2.3。

A.3.2.2.6 氯化高铁血红素标准工作液：临用时用 $0.1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液稀释氯化高铁血红素标准储备液配制相应标准工作液。

A.3.2.2.7 滤膜： $0.45\mu \mathrm{m}$ ，水相。

# A.3.2.3 仪器和设备

A.3.2.3.1 高效液相色谱仪：配有紫外检测器或二极管阵列检测器。

A.3.2.3.2 分析天平：感量为 $0.1\mathrm{g}$ 和 $0.0001\mathrm{g}$

# A.3.2.4 分析步骤

称取 $0.02\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 试样于 $100\mathrm{mL}$ 的容量瓶中，用 $0.1\mathrm{mol/L}$ 氢氧化钠溶液溶解并定容至刻度，摇匀后供高效液相色谱测定。

# A.3.2.5 参考色谱条件

A.3.2.5.1 色谱柱： $\mathrm{C_{18}}$ 柱， $250\mathrm{mm}\times 4.6\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或同等性能的色谱柱。

A.3.2.5.2 流动相：甲醇： $0.6\%$ 乙酸 $= 70:30$ （体积比）。

A.3.2.5.3 柱温： $35^{\circ}C$ 。

A.3.2.5.4 流速： $1\mathrm{mL} / \mathrm{min}$

A.3.2.5.5 检测波长： $399~\mathrm{nm}$

A.3.2.5.6 进样量： $10~\mu \mathrm{L}$

# A.3.2.6 色谱测定

在上述色谱条件下，分别对试样和标准溶液进行测定，氯化高铁血红素的保留时间约为 $5.021\mathrm{min}$ 以保留时间定性，试样的保留时间应与标准溶液的相对保留时间一致，偏差在 $\pm 2.5\%$ 以内。同时记录峰面积，根据标准工作液得到试样氯化高铁血红素的浓度，标准品和试样的色谱图参见附录C。

# A.3.2.7 结果计算

氯化高铁血红素的含量 $W_{2}$ 按式(A.2)计算。

$$
W _ {2} = \frac {A _ {1} \times c \times V}{A _ {\mathrm {s}} \times m \times 1000} \times 100 \% \tag{A.2}
$$

式中：

$A_{1}$ ——样液中氯化高铁血红素的色谱峰面积；

$c$ ——氯化高铁血红素标准储备液的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——最终样液的定容体积，单位为升 $(\mathrm{L})$ ；

$A_{\mathrm{s}}$ ——氯化高铁血红素标准储备液的色谱峰面积；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

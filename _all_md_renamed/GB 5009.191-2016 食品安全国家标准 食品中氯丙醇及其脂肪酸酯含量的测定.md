# 前言

本标准代替GB/T5009.191—2006《食品中氯丙醇含量的测定》、GB/T18782—2002《调味品中3-氯-1,2-丙二醇的测定》、SN/T0548.1—2002《出口酱油中1,3-二氯-2-丙醇和2,3-二氯-1-丙醇的检验方法》。

本标准与GB/T5009.191—2006相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中氯丙醇及其脂肪酸酯含量的测定”；  
——第二法增加 $\mathrm{D}_5 - 2\text{-}\mathrm{MCPD},\mathrm{D}_5 - 2$ ，3-DCP作为内标物质，将净化步骤由手填硅藻土层析柱净化改为硅藻土小柱净化；  
——删除第三法；  
——增加气相色谱-质谱法测定食品中氯丙醇脂肪酸酯含量的检测方法。

# 食品安全国家标准

# 食品中氯丙醇及其脂肪酸酯含量的测定

# 1 范围

本标准规定了食品中氯丙醇和氯丙醇脂肪酸酯的测定方法。

本标准适用于食品中氯丙醇和氯丙醇脂肪酸酯含量的测定。

本标准包括食品中3-氯-1，2-丙二醇含量的测定、食品中氯丙醇多组分含量的测定、食品中氯丙醇脂肪酸酯含量的测定共3个检测方法。

# 第一法 食品中3-氯-1，2-丙二醇含量的测定

# 同位素稀释-气相色谱-质谱法

# 2 原理

本标准采用同位素稀释技术，以 $\mathrm{D}_5 - 3$ -氯-1，2-丙二醇 $(\mathrm{D}_5 - 3$ -MCPD)为内标。试样中加入内标，以硅藻土为吸附剂，进行固相支持液-液萃取，用正己烷-无水乙醚溶液 $(9 + 1)$ 淋洗去除非极性脂质成分，用无水乙醚洗脱3-氯-1，2-丙二醇(3-MCPD)。以七氟丁酰基咪唑衍生，采用气相色谱-质谱仪测定，内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙酸乙酯 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。  
3.1.2 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
3.1.3 无水乙醚 $\mathrm{(C_4H_{10}O)}$ ：使用前需重蒸，重蒸时加少量还原铁粉。  
3.1.4 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.5 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：使用前于 $120^{\circ}C$ 烘烤 $4\mathrm{h}$   
3.1.6 七氟丁酰基咪唑 $\mathrm{(C_7H_3F_7N_2O}$ ,CAS号32477-35-3)：纯度 $\geqslant 98\%$

# 3.2 试剂配制

3.2.1 氯化钠溶液 $(20\%)$ ：称取氯化钠 $20\mathrm{g}$ ，加入 $80~\mathrm{mL}$ 水，搅拌使氯化钠充分溶解。  
3.2.2 正己烷-无水乙醚溶液 $(9 + 1)$ ：将 $450~\mathrm{mL}$ 正己烷与 $50~\mathrm{mL}$ 无水乙醚混合均匀。

# 3.3 标准品

3.3.1 3-氯-1，2-丙二醇(3-MCPD)标准品 $\mathrm{(C_3H_7ClO_2}$ ，CAS号96-24-2）：纯度 $\geqslant 98\%$   
3.3.2 $\mathrm{D}_5 - 3$ -氯-1，2-丙二醇 $(\mathrm{D}_5 - 3$ -MCPD)标准品 $(\mathrm{C}_3\mathrm{D}_5\mathrm{H}_2\mathrm{ClO}_2)$ ：纯度 $\geqslant 98\%$ 。

# 3.4 标准溶液配制

注：标准溶液配制好后均需转移至密闭性非常好的棕色玻璃容器中，于 $-20^{\circ}C$ 贮存。

3.4.1 3-MCPD 和 $\mathrm{D}_5 - 3$ -MCPD 标准储备液（ $1000\mathrm{mg / L}$ ）：分别准确称取 3-MCPD 和 $\mathrm{D}_5 - 3$ -MCPD 各 $10\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ ），用乙酸乙酯溶解，分别转移至 2 个 $10\mathrm{mL}$ 容量瓶中，用乙酸乙酯定容至刻度，混匀。保存期为 2 年。  
3.4.2 3-MCPD标准中间液 $(10\mathrm{mg / L})$ ：准确移取3-MCPD标准储备液 $(1000\mathrm{mg / L})0.1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用正己烷稀释至刻度，混匀。保存期为1年。  
3.4.3 3-MCPD标准工作液 $(1\mathrm{mg / L})$ ：准确移取3-MCPD标准中间液 $(10\mathrm{mg / L})1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用正己烷稀释至刻度，混匀。保存期为6个月。  
3.4.4 $\mathrm{D}_{5}-3$ -MCPD标准工作液(10 mg/L)：准确移取 $\mathrm{D}_{5}-3$ -MCPD标准储备液(1000 mg/L)0.25 mL于25 mL容量瓶中，用正己烷稀释至刻度，混匀。保存期为1年。  
3.4.5 3-MCPD系列标准工作液：分别吸取3-MCPD标准工作液 $(1\mathrm{mg / L})0.01\mathrm{mL},0.05\mathrm{mL},0.1\mathrm{mL}$ $0.2\mathrm{mL},0.4\mathrm{mL},0.8\mathrm{mL},1.6\mathrm{mL}$ 和3-MCPD标准中间液 $(10\mathrm{mg / L})0.32\mathrm{mL}$ 于密闭性很好的适当体积(5mL或 $10~\mathrm{mL}$ )的透明具塞(盖)玻璃管中，分别加入 $\mathrm{D}_5 - 3$ -MCPD标准工作液 $(10\mathrm{mg / L})20~\mu \mathrm{L}$ ，加正己烷至 $2\mathrm{mL}$ ，配制成含3-MCPD质量分别为 $10\mathrm{ng},50\mathrm{ng},100\mathrm{ng},200\mathrm{ng},400\mathrm{ng},800\mathrm{ng},1600\mathrm{ng},$ $3200\mathrm{ng}$ 的系列标准工作液，其中 $\mathrm{D}_5 - 3$ -MCPD的质量均为 $200\mathrm{ng}$ 。临用现配。

# 3.5 材料

3.5.1 硅藻土。  
3.5.2 气密针： $1\mathrm{mL}$   
3.5.3 玻璃层析柱：柱长 $40~\mathrm{cm}$ ，柱内径 $2\mathrm{cm}$

# 4 仪器和设备

4.1 气相色谱-质谱仪。  
4.2 电子天平：感量分别为 $1\mathrm{mg}$ 和 $0.01\mathrm{mg}$   
4.3 超声振荡器。  
4.4 旋转蒸发仪。  
4.5 恒温箱或其他恒温加热器。  
4.6 涡旋振荡器。  
4.7 离心机。

# 5 分析步骤

# 5.1 试样制备

液态样品摇匀；基质均匀的半固态样品和粉状固态样品直接测定；其他样品需匀浆粉碎均匀。制备好的试样于 $0\,^{\circ}\mathrm{C} \sim 5\,^{\circ}\mathrm{C}$ 保存。

# 5.2 试样提取

# 5.2.1 液态试样

称取试样 $4\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 于 $50~\mathrm{mL}$ 烧杯中，加入 $\mathrm{D}_5 - 3$ -MCPD标准工作液 $(10\mathrm{mg / L})20~\mu \mathrm{L}$ ，加

入氯化钠溶液 $(20\%)$ 4g，超声混匀 $5\mathrm{min}$ ，待净化。

# 5.2.2 半固态及固态试样（提取后无明显残渣的半固态及固态试样）

称取试样 $4\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于 $50~\mathrm{mL}$ 烧杯中，加入 $\mathrm{D}_5 - 3$ -MCPD标准工作液 $(10\mathrm{mg / L})20~\mu \mathrm{L}$ ，加入氯化钠溶液 $(20\%)$ $6\mathrm{g}$ ，超声 $10\mathrm{min}$ ，待净化。

# 5.2.3 半固态及固态试样（提取后有明显残渣的半固态及固态试样）

称取试样 $4\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于 $15~\mathrm{mL}$ 离心管中，加入 $\mathrm{D}_5 - 3$ -MCPD标准工作液 $(10\mathrm{mg / L})20\mu \mathrm{L}$ 加入氯化钠溶液 $(20\%)$ $15\mathrm{g}$ ，超声提取 $10\mathrm{min}$ ，以 $5000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，移取上清液，待净化。

# 5.3 试样净化

取硅藻土 $5\mathrm{g}$ ，加入提取液，充分混匀，放置 $10\mathrm{min}$ 。

取 $5\mathrm{g}$ 硅藻土装入层析柱中(层析柱下端填充少量玻璃棉)。将提取液与硅藻土的混合物装入层析柱中，上层加 $1\mathrm{cm}$ 高度的无水硫酸钠。用 $40~\mathrm{mL}$ 正己烷-无水乙醚溶液 $(9 + 1)$ 淋洗，弃去流出液。用 $150~\mathrm{mL}$ 无水乙醚洗脱3-MCPD(流速约为 $8\mathrm{mL / min})$ ，收集流出液，加入 $15\mathrm{g}$ 无水硫酸钠，混匀以吸收水分，放置 $10\mathrm{min}$ 后以漏斗过滤。滤液于 $35^{\circ}C$ 下旋转蒸发至近干(约 $0.5\mathrm{mL}$ )，用 $2\mathrm{mL}$ 正己烷溶解残渣，并转移至密闭性很好的适当体积 $(5\mathrm{mL}$ 或 $10~\mathrm{mL}$ )的透明具塞(盖)玻璃管中，待衍生化。

# 5.4 衍生化

用气密针向正己烷复溶液中加入 $0.04\mathrm{mL}$ 七氟丁酰基咪唑，立即密塞，涡旋混合 $30~\mathrm{s}$ ，于 $70^{\circ}\mathrm{C}$ 保温 $20\mathrm{min}$ 。取出放至室温，加入 $2\mathrm{mL}$ 氯化钠溶液 $(20\%)$ ，涡旋 $1\mathrm{min}$ ，静置，使水相和正己烷相分层，且水相澄清。移出正己烷相，加入约 $0.3\mathrm{g}$ 无水硫酸钠进行干燥，将溶液转移至进样小瓶中，供气相色谱-质谱测定。

同时做3-MCPD系列标准工作液的衍生化。

# 5.5 空白试样溶液制备

称取与试样相同质量的氯化钠溶液 $(20\%)$ 或空白试样，以下步骤按5.2、5.3和5.4与试样同时处理，以考察在试样测定过程中是否存在系统污染。

# 5.6 仪器参考条件

# 5.6.1 气相色谱参考条件

5.6.1.1 色谱柱：含 $5\%$ 苯基亚芳基聚合物或 $5\%$ 苯基-甲基聚硅氧烷的弱极性毛细管气相色谱-质谱柱（柱长 $30~\mathrm{m}$ ，内径 $0.25\mu \mathrm{m}$ ，膜厚 $0.25\mu \mathrm{m}$ ），或性能相当者。  
5.6.1.2 载气：氦气，流速为 $1\mathrm{mL} / \mathrm{min}$   
5.6.1.3 进样口温度： $250^{\circ}C$   
5.6.1.4 进样体积： $1\mu \mathrm{L}$ ，不分流进样，不分流时间为 $0.5\mathrm{min}$ ，溶剂延迟时间为 $5\mathrm{min}$ 。  
5.6.1.5 程序升温： $50^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，以 $2^{\circ}\mathrm{C} / \mathrm{min}$ 升至 $90^{\circ}\mathrm{C}$ ，再以 $40^{\circ}\mathrm{C} / \mathrm{min}$ 升至 $270^{\circ}\mathrm{C}$ ，并保持 $5\mathrm{min}$ 。

# 5.6.2 质谱参考条件

5.6.2.1 离子源：电子轰击源（EI）。  
5.6.2.2 电离能量： $70\mathrm{eV}$   
5.6.2.3 离子源温度： $250^{\circ}C$   
5.6.2.4 传输线温度： $280^{\circ}C$

5.6.2.5 扫描方式：选择离子监测模式（SIM）。  
5.6.2.6 监测离子：3-MCPD衍生物定量离子 $m / z$ 253，定性离子 $m / z$ 275、 $m / z$ 289、 $m / z$ 291； $\mathrm{D}_5 - 3-$ MCPD衍生物定量离子 $m / z$ 257，定性离子 $m / z$ 278、 $m / z$ 294、 $m / z$ 296。

# 5.7 标准曲线的制作

将 $1\mu \mathrm{L}3\text{-MCPD}$ 系列标准工作液的衍生液按浓度由低到高依次注入气相色谱-质谱仪中，测得3-MCPD和 $\mathrm{D}_5 - 3$ -MCPD的衍生物的峰面积，以3-MCPD系列标准工作溶液中的3-MCPD和 $\mathrm{D}_5 - 3$ -MCPD的质量比为横坐标，以3-MCPD和 $\mathrm{D}_5 - 3$ -MCPD的衍生物的峰面积比为纵坐标，绘制标准曲线。

# 5.8 试样溶液的测定

将 $1\mu \mathrm{L}$ 试样溶液注入气相色谱-质谱仪中，测得峰面积，根据标准曲线得到试样溶液中3-MCPD的质量。

# 6 分析结果的表述

试样中3-MCPD的含量按式(1)计算：

$$
X = \frac {A \times f}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中3-MCPD的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

A ——试样溶液中3-MCPD的质量，单位为纳克(ng)；

$f$ ——称取试样后进行分析测定前的稀释倍数；

$m$ ——试样的质量，单位为克(g)；

1000——换算系数。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

食品中3-氯-1，2-丙二醇的检出限为 $0.005\mathrm{mg / kg}$ ，定量限为 $0.01\mathrm{mg / kg}$ 。

# 第二法 食品中氯丙醇多组分含量的测定同位素稀释-气相色谱-质谱法

# 9 原理

本标准采用同位素稀释技术，以 $\mathrm{D}_5 - 3$ -氯-1，2-丙二醇（ $\mathrm{D}_5 - 3$ -MCPD）、 $\mathrm{D}_5 - 2$ -氯-1，3-丙二醇（ $\mathrm{D}_5 - 2-$ MCPD)、 $\mathrm{D}_5 - 1,3$ -二氯-2-丙醇 $(\mathrm{D}_5 - 1,3$ -DCP)和 $\mathrm{D}_5 - 2,3$ -二氯-2-丙醇 $(\mathrm{D}_5 - 2,3$ -DCP)为内标。试样中加入内标，以氯化钠溶液提取，采用硅藻土小柱进行净化，用正己烷淋洗，用乙酸乙酯洗脱氯丙醇，经七氟丁酰基咪唑衍生，以气相色谱-质谱测定，内标法定量。

# 13 分析结果的表述

试样中氯丙醇的含量按式(2)计算：

$$
X = \frac {A \times f}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中各氯丙醇的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

A ——试样溶液中各氯丙醇的质量，单位为纳克(ng)；

$f$ ——称取试样后进行分析测定前的稀释倍数；

$m$ ——试样的质量，单位为克(g)；

1000——换算系数。

计算结果保留三位有效数字。

# 第三法 食品中氯丙醇脂肪酸酯含量的测定气相色谱-质谱法

# 16 原理

试样中加入气代氯丙醇脂肪酸酯，以正己烷提取，以甲醇钠-甲醇溶液进行酯键断裂反应，使氯丙醇脂肪酸酯转化为氯丙醇和相应的脂肪酸甲酯，以冰乙酸终止反应，加入溴化钠溶液，并用正己烷进行液-液萃取，以去除生成的脂肪酸甲酯和其他非极性杂质，然后用硅藻土小柱净化，以乙酸乙酯洗脱氯丙醇，经七氟丁酰基咪唑衍生，以气相色谱-质谱仪检测，内标法定量。

17.4.9 氯丙醇系列标准工作液的制备：分别移取氯丙醇混合标准工作溶液 $(1\mathrm{mg / L})0.01\mathrm{mL}$ 、 $0.05\mathrm{mL},0.1\mathrm{mL},0.2\mathrm{mL},0.4\mathrm{mL},0.8\mathrm{mL},1.6\mathrm{mL}$ 和氯丙醇标准中间液 $(10\mathrm{mg / L})$ 各 $0.32\mathrm{mL}$ 于密闭性很好的适当体积( $5\mathrm{mL}$ 或 $10\mathrm{mL}$ )的透明具塞(盖)玻璃管中，分别加入氘代氯丙醇混合标准工作液 $(10\mathrm{mg / L})20~\mu \mathrm{L}$ ，加正己烷至 $2\mathrm{mL}$ ，混匀，配制成含各氯丙醇质量分别为 $10\mathrm{ng},50\mathrm{ng},100\mathrm{ng},200\mathrm{ng}$ 、 $400\mathrm{ng},800\mathrm{ng},1600\mathrm{ng},3200\mathrm{ng}$ 的系列标准工作液，其中各氘代氯丙醇的质量均为 $200\mathrm{ng}$ 。临用现配。

# 19 分析步骤

# 19.1 试样制备

液态样品摇匀；基质均匀的半固态样品和粉状固态样品直接测定；其他样品需匀浆粉碎均匀。制备好的试样于 $0\,^{\circ}\mathrm{C} \sim 5\,^{\circ}\mathrm{C}$ 保存。

# 19.2 试样提取

# 19.2.1 植物油、动物油等油脂类试样

称取试样 $0.1\mathrm{g}\sim 0.2\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，加入氘代氯丙醇脂肪酸酯混合标准工作液 $(10~\mathrm{mg / L}$ ，以对应的氘代氯丙醇计) $20~\mu \mathrm{L},\mathrm{D}_{5} - 1,3$ -DCP和 $\mathrm{D}_5 - 2,3$ -DCP标准工作液 $(10~\mathrm{mg / L})$ 各 $20~\mu \mathrm{L}$ 。

# 19.2.2 其他试样

称取试样 $2\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ), 加入氘代氯丙醇脂肪酸酯混合标准工作液 $(10\mathrm{mg / L}$ , 以对应的氘代氯丙醇计) $20~\mu \mathrm{L}$ 。加入 $4\mathrm{mL}$ 正己烷, 充分振摇混匀, 超声提取 $20\mathrm{min}$ , 静置分层后, 转移出上层正己烷相。再重复提取2次, 合并正己烷相, 加入 $\mathrm{D}_{5}-1,3-\mathrm{DCP}$ 和 $\mathrm{D}_{5}-2,3-\mathrm{DCP}$ 标准工作液 $(10\mathrm{mg / L})$ 各 $20~\mu \mathrm{L}$ , 以氮气浓缩至约 $1\mathrm{mL}$ 。

注：对于乳粉、咖啡等固体粉末试样，需先加 $2\mathrm{mL}$ 水溶解后再用正己烷提取。对于香肠等动物性食品试样，可采用经乙腈饱和的正己烷作为提取液。

# 19.3 酯键断裂反应

向试样提取液中加入 $0.5\mathrm{mL}$ 甲基叔丁基醚-乙酸乙酯溶液 $(8 + 2)$ 和 $1\mathrm{mL}$ 甲醇钠-甲醇溶液 $(0.5\mathrm{mol} / \mathrm{L})$ ，盖紧盖子，涡旋振荡 $30~\mathrm{s}$ 。室温反应 $4\mathrm{min}$ ，加入 $100\mu \mathrm{L}$ 冰乙酸终止反应。加入 $3\mathrm{mL}$ 溴化钠溶液 $(20\%)$ 和 $3\mathrm{mL}$ 正己烷，涡旋振荡 $30~\mathrm{s}$ ，静置 $1\mathrm{min}$ ，弃去上层正己烷相，再用 $3\mathrm{mL}$ 正己烷萃取一次，弃去上层正己烷相，下层的水相溶液待净化。

注：此步骤中如采用氯化钠溶液(20%)萃取,则经后续步骤测定得到的是氯丙醇脂肪酸酯和缩水甘油酯的总含量。

# 19.4 试样净化

将水相溶液倒入硅藻土小柱中，平衡 $10\mathrm{min}$ 后，用 $15~\mathrm{mL}$ 乙酸乙酯洗脱，收集洗脱液，在洗脱液中加入 $4\mathrm{g}$ 无水硫酸钠，放置 $10\mathrm{min}$ 后过滤，转移滤液至密闭性很好的适当体积 $(5\mathrm{mL}$ 或 $10\mathrm{mL})$ 的透明具塞(盖)玻璃管中，以氮气浓缩至近干（约 $0.5\mathrm{mL}$ ，切忌浓缩至全干），用 $2\mathrm{mL}$ 正己烷溶解残渣，涡旋混匀，待进行衍生化反应。

# 19.5 衍生化

用气密针向净化液中加入 $0.04\mathrm{mL}$ 七氟丁酰基咪唑，立即密塞。涡旋混合后，于 $70^{\circ}\mathrm{C}$ 保温 $20\mathrm{min}$ 取出放至室温，加 $2\mathrm{mL}$ 氯化钠溶液 $(20\%)$ ，涡旋 $1\mathrm{min}$ ，使两相分层，且水相澄清。取出上层有机相，加 $0.3\mathrm{g}$ 无水硫酸钠吸水干燥，将溶液转移至进样小瓶中，供气相色谱-质谱测定。

同时做系列标准工作液的衍生化反应。

# 19.6 空白试样溶液制备

称取与试样相同质量的溴化钠溶液 $(20\%)$ 或空白试样，以下步骤按19.2、19.3、19.4和19.5与试样同时处理，以考察在试样测定过程中是否存在系统污染。

# 19.7 仪器参考条件

# 19.7.1 气相色谱参考条件

19.7.1.1 色谱柱：含 $5\%$ 苯基亚芳基聚合物或 $5\%$ 苯基-甲基聚硅氧烷的弱极性毛细管气相色谱-质谱柱（柱长 $30\mathrm{m}$ ，内径 $0.25\mu \mathrm{m}$ ，膜厚 $0.25\mu \mathrm{m}$ ），或性能相当者。

19.7.1.2 载气：氦气，流速为 $1\mathrm{mL} / \mathrm{min}$

19.7.1.3 进样口温度： $250^{\circ}C$   
19.7.1.4 进样量： $1\mu \mathrm{L}$ ，不分流进样，不分流时间为 $0.5\mathrm{min}$ ，溶剂延迟时间为 $5\mathrm{min}$   
19.7.1.5 程序升温： $50^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，以 $2^{\circ}\mathrm{C} / \mathrm{min}$ 升至 $90^{\circ}\mathrm{C}$ ，再以 $40^{\circ}\mathrm{C} / \mathrm{min}$ 升至 $270^{\circ}\mathrm{C}$ ，并保持 $5\mathrm{min}$ 。

# 19.7.2 质谱参考条件

19.7.2.1 电离源：电子轰击源。  
19.7.2.2 电离能量： $70\mathrm{eV}$ 。  
19.7.2.3 离子源温度： $250^{\circ}C$   
19.7.2.4 传输线温度： $280^{\circ}C$   
19.7.2.5 扫描方式：选择离子监测模式（SIM）。  
19.7.2.6 监测离子：见表1。

# 19.8 标准曲线的制作

将 $1\mu \mathrm{L}$ 系列标准工作液的衍生液按浓度由低到高依次注入气相色谱-质谱仪，测得各氯丙醇和氘代氯丙醇的衍生物的峰面积，以标准工作液中氯丙醇与其对应的氘代氯丙醇的质量比为横坐标，以氯丙醇与其对应的氘代氯丙醇的衍生物的峰面积比为纵坐标，绘制标准曲线。

# 19.9 试样溶液的测定

将 $1\mu \mathrm{L}$ 试样溶液注入气相色谱-质谱仪，测得相应的峰面积，根据标准曲线得到试样溶液中各氯丙醇的质量。

# 20 分析结果的表述

试样中各氯丙醇脂肪酸酯(3-MCPD酯、2-MCPD酯、1,3-DCP酯和2,3-DCP酯)的含量按式(3)计算：

$$
X = \frac {A \times f}{m \times 1 0 0 0} \quad \dots \tag {3}
$$

式中：

X ——试样中3-MCPD酯、2-MCPD酯、1,3-DCP酯和2,3-DCP酯的含量，分别以3-MCPD、2-MCPD、1,3-DCP和2,3-DCP计，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

A ——试样溶液中3-MCPD、2-MCPD、1,3-DCP和2,3-DCP的质量，单位为纳克(ng)；

$f$ —称取试样后进行分析测定前的稀释倍数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果保留三位有效数字。

# 21 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 22 其他

食用油等油脂类样品中3-MCPD酯、2-MCPD酯、1,3-DCP酯和2，3-DCP酯的检出限均为 $0.01\mathrm{mg / kg}$ ，定量限均为 $0.025\mathrm{mg / kg}$ ；其他食品中3-MCPD酯、2-MCPD酯、1,3-DCP酯和2，3-DCP酯的检出限分别为 $0.005\mathrm{mg / kg},0.005\mathrm{mg / kg},0.01\mathrm{mg / kg},0.01\mathrm{mg / kg}$ ，定量限分别为 $0.01\mathrm{mg / kg}$ 产 $0.01\mathrm{mg / kg},0.025\mathrm{mg / kg},0.025\mathrm{mg / kg}$ 。

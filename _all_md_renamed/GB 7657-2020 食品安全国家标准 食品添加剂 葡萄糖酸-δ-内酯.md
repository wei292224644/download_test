# 前言

本标准代替GB7657—2005《食品添加剂 葡萄糖酸-δ-内酯》。

本标准与GB7657—2005相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 葡萄糖酸-δ-内酯”；  
——修改了标准适用范围；  
葡萄糖酸-δ-内酯 $\left(\mathrm{C}_{6} \mathrm{H}_{10} \mathrm{O}_{6}\right)$ 含量指标由 $(99.0 \sim 105.0)\%$ 修改为“ $\geqslant 99.0\%$ ”；  
——增加熔点指标及其测定方法；  
——增加干燥减量指标及其测定方法；  
——删除了重金属(以Pb计)指标要求；  
——项目名称由“砷(As)”修改为“总砷(以As计)”,同时修改了总砷(以As计)的测定方法;  
铅(Pb)指标由“ $\leqslant 0.001\%$ ”修改为“ $\leqslant 2.0~\mathrm{mg / kg}$ ”。

# 检验方法

# A.1 安全提示

本标准试验方法中使用的部分试剂具有毒性或腐蚀性，使用时需小心谨慎并按相关规定操作。若溅到皮肤上应立即用水冲洗，严重者应立即治疗。在使用挥发性酸时，需在通风橱中进行。

# A.2 一般规定

本标准除另有规定外，所用试剂的纯度应为分析纯，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应按GB/T601、GB/T602、GB/T603的规定制备，试验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 显色反应

# A.3.1.1 试剂和材料

氯化铁溶液： $50\mathrm{g / L}$ 。称取 $8.33\mathrm{g}$ 三氯化铁 $(\mathrm{FeCl}_3\cdot 6\mathrm{H}_2\mathrm{O})$ ，溶于盐酸溶液 $(1 + 9)$ 中，用盐酸溶液 $(1 + 9)$ 稀释至 $100~\mathrm{mL}$ 。

# A.3.1.2 分析步骤

称取约 $0.1\mathrm{g}$ 试样，溶于 $5\mathrm{mL}$ 水中，加入1滴氯化铁溶液，溶液应呈深黄色。

# A.3.2 熔点

按GB/T617仪器法进行测定，熔点为 $150^{\circ}\mathrm{C}\sim 154^{\circ}\mathrm{C}$

# A.4 葡萄糖酸-δ-内酯 $\left(\mathrm{C}_{6} \mathrm{H}_{10} \mathrm{O}_{6}\right)$ 含量的测定

# A.4.1 试剂和材料

A.4.1.1 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.1\mathrm{mol / L}$

A.4.1.2 盐酸标准滴定溶液： $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$

A.4.1.3 酚酞指示液： $10\mathrm{g / L}$

# A.4.2 分析步骤

A.4.2.1 称取 $0.6\mathrm{g}$ 干燥减重后的试样，精确至 $0.0002\mathrm{g}$ ，置于锥形瓶中，加 $50~\mathrm{mL}$ 水溶解，加 $50.00~\mathrm{mL}$ 氢氧化钠标准滴定溶液(A.4.1.1)，室温放置 $15\mathrm{min}$ ，加2滴～3滴酚酞指示液，用盐酸标准滴定溶液(A.4.1.2)滴定至溶液变为无色即为终点。  
A.4.2.2 同时不加试样按A.4.2.1操作做试剂空白试验。

# A.4.3 结果计算

葡萄糖酸-δ-内酯 $\mathrm{(C_6H_{10}O_6)}$ 含量(以干基计)的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {\left(V _ {1} - V _ {2}\right) \times c \times M}{m \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——空白试验消耗盐酸标准滴定溶液(A.4.1.2)的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试样消耗盐酸标准滴定溶液(A.4.1.2)的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——盐酸标准滴定溶液(A.4.1.2)的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——葡萄糖酸-δ-内酯的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{C_6H_{10}O_6}) = 178.1]$

$m$ —干燥减重后试样的质量，单位为克 $(\mathrm{g})$

1000——换算系数。

取两次平行测定结果的算术平均值为测定结果，结果表示到小数点后两位。且两次平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 还原性物质（以D-葡萄糖计)的测定

# A.5.1 试剂和材料

# A.5.1.1 费林溶液。

A.5.1.2 葡萄糖 $\left(\mathrm{C}_{6} \mathrm{H}_{12} \mathrm{O}_{6}\right)$ 标准溶液: $1.0 \mathrm{mg} / \mathrm{mL}$ 。称取 $0.100 \mathrm{~g}$ 无水葡萄糖, 精确至 $0.0002 \mathrm{~g}$ , 用水溶解, 定量转移至 $100 \mathrm{~mL}$ 容量瓶中, 稀释至刻度。

# A.5.2 分析步骤

称取 $1\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，用水溶解并稀释至 $25\mathrm{mL}$ ，取 $5\mathrm{mL}$ 试样液，加 $2\mathrm{mL}$ 费林溶液，煮沸 $1\mathrm{min}$ 。取 $1.00\mathrm{mL}$ 葡萄糖 $(\mathrm{C_6H_{12}O_6})$ 标准溶液，稀释至 $5\mathrm{mL}$ ，从“加 $2\mathrm{mL}$ 费林溶液”开始与试样同样处理。试样溶液生成的红色沉淀不得多于标准溶液的红色沉淀。

# A.6 硫酸盐（以 $\mathbf{SO}_4$ 计）的测定

# A.6.1 试剂和材料

# A.6.1.1 盐酸溶液： $1 + 10$

A.6.1.2 氯化钡溶液： $100\mathrm{g / L}$

A.6.1.3 硫酸盐 $\mathrm{(SO_4)}$ 标准溶液： $0.1\mathrm{mg / mL}$

# A.6.2 分析步骤

称取 $0.2\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $15\mathrm{mL}$ 水中，加 $1\mathrm{mL}$ 盐酸溶液（必要时过滤），加 $2\mathrm{mL}$ 氯化钡溶液，稀释至 $50\mathrm{mL}$ ，摇匀，放置 $10\mathrm{min}$ 。取 $0.60\mathrm{mL}$ 硫酸盐 $(\mathrm{SO}_4)$ 标准溶液，加水 $15\mathrm{mL}$ ，从“加 $1\mathrm{mL}$ 盐酸溶液”开始与试样同样处理。试样溶液的浊度不应深于标准溶液的浊度。

# A.7 氯化物（以CI计）的测定

# A.7.1 试剂和材料

A.7.1.1 硝酸溶液：称取 $25\mathrm{g}$ 硝酸置于 $75\mathrm{g}$ 水中。

A.7.1.2 硝酸银溶液： $17\mathrm{g / L}$   
A.7.1.3 氯化物(Cl)标准溶液： $0.1\mathrm{mg / mL}$

# A.7.2 分析步骤

按GB/T9729的规定进行。测定时称取 $0.5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，溶于 $20~\mathrm{mL}$ 水（必要时过滤）制成试液；同时，取 $1.00~\mathrm{mL}$ 氯化物(Cl)标准溶液，加水 $20~\mathrm{mL}$ ，与试样同时同样处理。试样溶液的浊度不应深于标准溶液的浊度。

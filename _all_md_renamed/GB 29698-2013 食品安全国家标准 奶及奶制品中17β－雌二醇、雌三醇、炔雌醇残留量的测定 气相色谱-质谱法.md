# 奶及奶制品中 $17\beta$ 一雌二醇、雌三醇、炔雌醇多残留的测定

# 气相色谱-质谱法

# 1 范围

本标准规定了奶及奶制品中雌激素类药物残留量检测的制样和气相色谱一质谱测定方法。

本标准适用于鲜奶和奶粉样品中 $17\beta$ 一雌二醇、雌三醇、炔雌醇单个或多个药物残留量的检测。

# 3 原理

试料中残留的雌激素，用乙酸乙酯和乙腈混合溶剂提取，固相萃取柱净化，硅烷化试剂衍生，离子模式气相色谱-质谱测定，外标法定量。

# 4 试剂和材料

以下所用试剂，除特殊注明外均为分析纯试剂，水为符合GB/T6682规定的一级水。

4.1 $17\beta$ 一雌二醇、雌三醇、炔雌醇：含量 $\geqslant 98.0\%$

4.2 二硫赤藓糖醇  
4.3 N-甲基三甲基硅基三氟乙酰氨   
4.4 三甲基硅烷  
4.5 乙腈  
4.6 乙酸乙酯  
4.7 甲醇  
4.8 甲苯

4.9 正己烷  
4.10 盐酸  
4.11 氢氧化钠  
4.12 $\mathrm{C}_{18}$ 固相萃取柱：LC- $\mathrm{C}_{18}$ ， $500\mathrm{mg / 3mL}$ ，或相当者。  
4.13 硅胶固相萃取柱：LC-Si， $500\mathrm{mg} / 3\mathrm{mL}$ ，或相当者。  
4.14 $95\%$ 正己烷乙酸乙酯溶液：取正己烷 $95~\mathrm{mL}$ ，用乙酸乙酯溶解并稀释至 $100~\mathrm{mL}$ 。  
4.15 $70\%$ 正己烷乙酸乙酯溶液：取正己烷 $70~\mathrm{mL}$ ，用乙酸乙酯溶解并稀释至 $100~\mathrm{mL}$ 。  
4.16 1mol/L 氢氧化钠溶液：取氢氧化钠 $40\mathrm{g}$ ，用水溶解并稀释至 $1000\mathrm{mL}$ 。  
4.17 5mol/L盐酸溶液：取浓盐酸 $48~\mathrm{mL}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
4.18 衍生化试剂: 取二硫赤藓糖醇 $0.01 \mathrm{~g}$ , 用 N-甲基三甲基硅基三氟乙酰氨 (MSTFA) $5 \mathrm{~mL}$ 溶解,于液面下加三甲基硅烷 $10 \mu \mathrm{L}$ , 混匀, $2 \sim 8^{\circ} \mathrm{C}$ 放置过夜, 避光防潮密封保存。衍生化试剂应无色, 如果发生棕红色等颜色变化, 表明试剂失效。  
4.19 $1 \mathrm{mg} / \mathrm{mL} 17 \beta$ 一雌二醇、雌三醇、炔雌醇标准贮备液：精确称取 $17 \beta$ 一雌二醇、雌三醇、炔雌醇标准品各 $10 \mathrm{mg}$ ，分别于 $10 \mathrm{~mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的标准贮备液。 $-20^{\circ} \mathrm{C}$ 以下保存，有效期6个月。  
4.20 $10 \mathrm{mg} / \mathrm{L} 17 \beta$ 一雌二醇、雌三醇、炔雌醇混合标准工作液：精密量取 $1 \mathrm{mg} / \mathrm{mL} 17 \beta$ 一雌二醇、雌三醇和炔雌醇标准贮备液各 $1.0 \mathrm{~mL}$ ，于 $100 \mathrm{~mL}$ 量瓶中，用甲醇稀释至刻度，配制成浓度为 $10 \mathrm{mg} / \mathrm{L}$ 的混合标准工作液， $2 \sim 8^{\circ} \mathrm{C}$ 保存，有效期1个月。

# 6 试样制备与保存

# 6.1 试料的制备

取适量新鲜或冷藏的空白或供试样品，混合，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

$2\sim 8^{\circ}C$ 保存。

# 7 测定步骤

# 7.1 标准工作曲线制备

精密量取 $10\mathrm{mg / L}$ 混合标工作液适量，用甲醇稀释，配制成浓度为10、50、100、200、500和1000 $\mu \mathrm{g} / \mathrm{L}$ 系列标准工作液，于 $40^{\circ}\mathrm{C}$ 水浴氮气吹干，按衍生化步骤处理，供气相色谱法-质谱测定。以测得峰面积为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.2 提取

# 7.2.1 液态试料

称取试料 $(10 \pm 0.05) \mathrm{~g}$ , 于 $50 \mathrm{~mL}$ 离心管中, 加乙腈 $5 \mathrm{~mL}$ 、乙酸乙酯 $15 \mathrm{~mL}$ , 旋涡振荡 $3 \mathrm{~min}$ , 8000 r/min 离心 $5 \mathrm{~min}$ , 收集上清液于另一 $50 \mathrm{~mL}$ 离心管中, 残渣重复提取一次, 合并两次上清液, 于 $40^{\circ} \mathrm{C}$ 水浴旋转蒸发至近干, 用 $1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液 $6 \mathrm{~mL}$ 分三次溶解, 转至另一 $50 \mathrm{~mL}$ 离心管中, 加正己烷 $20 \mathrm{~mL}$ 旋涡振荡 $1 \mathrm{~min}, 8000 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ , 收集下层提取液, 用 $5 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液调 $\mathrm{pH}$ 至 $5.0 \sim 5.2$ , 备用。

# 7.2.2 固态试料

称取试料（ $10 \pm 0.05$ ）g，于 $50 \mathrm{~mL}$ 离心管中，加乙酸乙酯 $15 \mathrm{~mL}$ ，旋涡振荡 $3 \mathrm{~min}$ ， $8000 \mathrm{r} / \mathrm{min}$ 离心5

min，收集上清液于另一 $50~\mathrm{mL}$ 离心管中，残渣重复提取一次，合并两次上清液， $40^{\circ}\mathrm{C}$ 水浴旋转蒸发至近干，用 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液 $6\mathrm{mL}$ 分三次溶解，转至另一 $50~\mathrm{mL}$ 离心管中，加正己烷 $20~\mathrm{mL}$ 旋涡振荡 $1\mathrm{min}$ ， $8000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，收集下层提取液，用 $5\mathrm{mol} / \mathrm{L}$ 盐酸溶液调 $\mathrm{pH}$ 至 $5.0\sim 5.2$ ，备用。

# 7.3 净化

$\mathrm{C}_{18}$ 柱依次用甲醇 $5 \mathrm{~mL}$ 和水 $5 \mathrm{~mL}$ 活化, 取备用液过柱, 用水 $5 \mathrm{~mL}$ 淋洗, 抽干, 用甲醇 $5 \mathrm{~mL}$ 洗脱, 收集洗脱液, 于 $40^{\circ} \mathrm{C}$ 水浴氮气吹干。用 $95 \%$ 正己烷乙酸乙酯溶液 $5 \mathrm{~mL}$ 溶解残余物, 过经正己烷 $5 \mathrm{~mL}$ 活化后的硅胶柱, 加正己烷 $5 \mathrm{~mL}$ 淋洗, 抽干, 再用 $70 \%$ 正己烷乙酸乙酯溶液 $5 \mathrm{~mL}$ 洗脱, 收集洗脱液, 于 $40^{\circ} \mathrm{C}$ 水浴氮气吹干。

# 7.4 衍生化

残余物加甲苯和衍生化试剂各 $100 \mu \mathrm{L}$ 溶解, 混匀, 封口, 在 $80^{\circ} \mathrm{C}$ 烘箱中衍生 $60 \mathrm{~min}$ , 冷却, 供气相色谱-质谱测定。

# 7.5 测定

# 7.5.1 色谱条件

色谱柱 HP-5 MS 石英毛细管色谱柱（ $30\mathrm{m}\times 0.25\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ），或相当者。

载气为高纯氦气，恒流 $1.0\mathrm{mL / min}$

进样口温度 $220^{\circ}\mathrm{C}$

进样体积： $1\mu \mathrm{L}$ ，不分流；

色谱柱起始温度 $100^{\circ}\mathrm{C}$ (保持 $1\mathrm{min}$ ), 以 $20^{\circ}\mathrm{C} / \mathrm{min}$ 的升温速率升至 $200^{\circ}\mathrm{C}$ (保持 $3\mathrm{min}$ ), 再以 $20^{\circ}\mathrm{C} / \mathrm{min}$ 的升温速率升至 $260^{\circ}\mathrm{C}$ (保持 $5\mathrm{min}$ ), 再以 $20^{\circ}\mathrm{C} / \mathrm{min}$ 的升温速率升至 $280^{\circ}\mathrm{C}$ (保持 $5\mathrm{min}$ );

# 7.5.2 质谱条件

离子源(EI)温度： $200^{\circ}\mathrm{C}$

EM电压：高于调谐电压 $200\mathrm{V}$

电子能量：70eV；

GC/MS 传输线温度： $280^{\circ}\mathrm{C}$ ；

四极杆温度： $160^{\circ}\mathrm{C}$

选择离子监测(SIM): $(\mathrm{m} / \mathrm{z})232$ ，285，326，416（ $17\beta$ 一雌二醇）；311，345，414，504（雌三醇）；

285，300，425，440（炔雌醇）。

# 7.5.3 测定法

# 7.5.3.1 定性测定

通过试样色谱图的保留时间与相应标准品的保留时间、各色谱峰的特征离子与相应浓度标准溶液各色谱峰的特征离子相对照定性。试样与标准品保留时间的相对偏差不大于 $5\%$ ；试样特征离子的相对丰度与浓度相当混合标准溶液的相对丰度一致，相对丰度偏差不超过表1的规定，则可判断试样中存在相应的被测物。

# 7.5.3.2 定量测定

取适量试样溶液和相应的标准溶液，做单点或多点校准，按外标法，以峰面积定量，标准溶液及试样液中 $17\beta$ 一雌二醇、雌三醇和炔雌醇的响应值均应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液、空白试样和空白添加试样的色谱图及质谱图见附录A。

# 7.6 空白试验

除不加试样外，采用完全相同的测定步骤进行平行操作。

# 8结果计算和表述

试料中 $17\beta$ 一雌二醇、雌三醇、炔雌醇的残留量（ $\mu \mathrm{g / kg}$ ）：按下式计算。

$$
X = \frac {A \times C s \times V}{A s \times m}
$$

式中：

$X$ ——供试试料中相应的 $17\beta$ 一雌二醇、雌三醇、炔雌醇残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

Cs——标准溶液中相应的 $17 \beta$ 一雌二醇、雌三醇和炔雌醇浓度, $\mu \mathrm{g} / \mathrm{L}$ ;

A——试样中相应的 $17\beta$ 一雌二醇、雌三醇和炔雌醇的峰面积；

As——标准溶液中相应的 $17 \beta$ 一雌二醇、雌三醇和炔雌醇的峰面积；

V——溶解残余物体积，mL；

$m$ ——供试试料质量，g。

# 9 检测方法灵敏度、准确度和精密度

# 9.1灵敏度

本方法检测限为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1.0~\mu \mathrm{g / kg}$

# 9.2准确度

本方法在 $1\sim 10\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $60\% \sim 120\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

# 水产品中泰乐菌素残留量的测定 高效液相色谱法

# 1 范围

本文件规定了水产品中泰乐菌素残留检测的制样和高效液相色谱测定方法。

本文件适用于鱼、虾、蟹、甲鱼等水产品可食组织中泰乐菌素残留量测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

GB/T30891—2014 水产品抽样规范

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的泰乐菌素，在弱碱性条件下，用乙酸乙酯提取、正已烷脱脂、SCX固相萃取柱净化，高效液相色谱法测定，外标法定量。

# 5 试剂与材料

以下所用的试剂，除特别注明外均为分析纯试剂；水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ ：色谱纯。  
5.1.3 正己烷 $\mathrm{(C_6H_{14})}$ ：色谱纯。  
5.1.4 乙酸乙酯 $\mathrm{(CH_3COOC_2H_3)}$ ：色谱纯。  
5.1.5 磷酸氢二钾 $(\mathrm{K_2HPO_4})$   
5.1.6 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
5.1.7 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$

# 5.2 标准品

酒石酸泰乐菌素(Tylosin tartrate，CAS号：74610-55-2)：含量 $\geq 98\%$

# 5.3 溶液配制

5.3.1 磷酸盐缓冲溶液I（pH8.0）：取磷酸二氢钾 $0.52\mathrm{g}$ 、磷酸氢二钾 $16.73\mathrm{g}$ ，用水溶解并稀释至 $500~\mathrm{mL}$ 。  
5.3.2 磷酸盐缓冲溶液Ⅱ（pH4.0）：用磷酸调节磷酸盐缓冲溶液I的 $\mathrm{pH}$ 至4.0。  
5.3.3 磷酸氢二钾溶液 $(0.1\mathrm{mol} / \mathrm{L},\mathrm{pH}9.0)$ ：取磷酸氢二钾 $1.74\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
5.3.4 磷酸二氢钾溶液 $(0.01\mathrm{mol} / \mathrm{L},\mathrm{pH}2.5)$ ：取磷酸二氢钾 $1.36\mathrm{g}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ ，磷酸调节 $\mathrm{pH}$ 至2.5。

# 5.4 标准溶液制备

泰乐菌素标准储备液 $(100\mu \mathrm{g / mL})$ ：取酒石酸泰乐菌素标准品约 $10\mathrm{mg}$ ，精密称定，加乙腈适量使溶解并定容至 $100~\mathrm{mL}$ 棕色容量瓶中，摇匀，即得。 $2^{\circ}C\sim 8^{\circ}C$ 避光保存，有效期1个月。

# 5.5 材料

SCX固相萃取柱： $500\mathrm{mg / 3mL}$ ，或相当者。

# 6 仪器和设备

6.1 液相色谱仪：配紫外检测器  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 高速组织捣碎机。  
6.4 离心机： $5000\mathrm{r / min}$   
6.5 涡旋混合器。  
6.6 振荡器。  
6.7 旋转蒸发器。  
6.8 固相萃取装置。  
6.9 鸡心瓶： $100~\mathrm{mL}$   
6.10 具塞离心管： $50~\mathrm{mL}$

# 7 试料的制备与保存

# 7.1 试料的制备

按GB/T30891—2014中附录B的要求制样。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存，有效期3个月。

# 8 测定步骤

# 8.1 提取

取试料 $5\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 具塞离心管中，加磷酸盐缓冲溶液 $[5\mathrm{mL}$ ，涡旋混匀 $1\mathrm{min}$ 加乙酸乙酯 $25~\mathrm{mL}$ ，振摇提取 $10\mathrm{min},4500\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液转移至 $100~\mathrm{mL}$ 鸡心瓶中，加乙酸乙酯 $25~\mathrm{mL}$ ，重复提取一次，合并提取液， $45^{\circ}C$ 水浴减压蒸发至近干。

# 8.2 净化

用甲醇 $1\mathrm{mL}$ 溶解残渣，转移至另一 $50~\mathrm{mL}$ 离心管，用磷酸盐缓冲溶液 $\mathrm{II}10\mathrm{mL}$ 和正己烷 $10\mathrm{mL}$ 洗涤鸡心瓶，涡旋混合 $2\mathrm{min}$ ，洗涤液合并于离心管，涡旋混匀， $4500\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去上层正己烷，取下层水溶液，备用。

取SCX固相萃取柱，用甲醇 $5\mathrm{mL}$ 、磷酸盐缓冲溶液 $\mathrm{II}5\mathrm{mL}$ 活化，备用液过柱（控制流速约 $1.5\mathrm{mL / min})$ 。依次用水 $3\mathrm{mL}$ 、甲醇 $5\mathrm{mL}$ 、水 $3\mathrm{mL}$ 、磷酸氢二钾溶液 $3\mathrm{mL}$ 、甲醇 $0.4\mathrm{mL}$ 洗柱，弃去洗出液，抽干。加甲醇 $3.0~\mathrm{mL}$ 洗脱，收集洗脱液，涡旋混匀， $0.45\mu \mathrm{m}$ 微孔滤膜过滤，待上机分析。

# 8.3 标准曲线的制备

精密量取泰乐菌素标准储备液适量，用甲醇稀释制成浓度为 $0.050\mu \mathrm{g / mL},0.10\mu \mathrm{g / mL},0.50\mu \mathrm{g} /$ $\mathrm{mL},1.00\mu \mathrm{g / mL},2.00\mu \mathrm{g / mL},5.00\mu \mathrm{g / mL}$ 的系列标准工作液，供液相色谱仪分析。

# 8.4 测定

# 8.4.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 色谱柱 $(250\mathrm{mm}\times 4.6\mathrm{mm},5\mu \mathrm{m})$ 或性能相当者；  
b）检测波长： $285~\mathrm{nm}$   
c）流动相：A为磷酸二氢钾溶液，B为乙腈，梯度洗脱条件见表1；

# 8.4.2 测定法

取试样溶液和标准溶液各 $50~\mu \mathrm{L}$ ，作单点或多点校准，按外标法以色谱峰面积定量。标准溶液及试样溶液中泰乐菌素响应值应在仪器检测的线性范围之内。根据泰乐菌素标准工作液的保留时间定性，外标法定量。色谱图见附录A。

# 8.4.3 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V}{A _ {\mathrm {S}} \times m} \times 1 0 0 0 \dots \tag {1}
$$

式中：

$X$ ——试样中泰乐菌素残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{S}}$ ——标准溶液中泰乐菌素浓度的数值，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$

A ——试样溶液色谱图中泰乐菌素峰峰面积；

$A_{\mathrm{S}}$ ——标准溶液色谱图中泰乐菌素峰峰面积；

$V$ ——试样溶液定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法的检测限为 $30~\mu \mathrm{g / kg}$ ，定量限为 $50~\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $50~\mu \mathrm{g / kg}\sim 500~\mu \mathrm{g / kg}$ ，添加浓度水平的回收率为 $70\% \sim 110\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

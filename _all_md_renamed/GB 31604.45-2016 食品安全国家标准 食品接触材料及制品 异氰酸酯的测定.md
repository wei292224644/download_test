2016-10-19 发布

2017-04-19 实施

# 前言

本标准代替GB/T23296.22—2009《食品接触材料塑料中异氰酸酯类含量的测定高效液相色谱法》和SN/T2741—2010《进出口食品接触材料高分子材料异氰酸酯含量的测定高效液相色谱法》。

本标准与GB/T23296.22—2009和SN/T2741—2010相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 异氰酸酯的测定”；  
——分析物增加至7个；  
——去掉空白试样的制备；  
——基质标准曲线修改为标准溶液标准曲线；  
——修改了分析结果的表述。

# 食品安全国家标准

# 食品接触材料及制品 异氰酸酯的测定

# 1 范围

本标准规定了食品接触材料用塑料及其制品中甲苯-2,6-二异氰酸酯、二苯基甲烷-4, $4^{\prime}$ -二异氰酸酯、甲苯-2,4-二异氰酸酯、萘-1,5-二异氰酸酯、苯基异氰酸酯、六亚甲基二异氰酸酯、环己基异氰酸酯和以上7种异氰酸酯总量的测定方法。

本标准适用于聚氨酯塑料及其制品中甲苯-2,6-二异氰酸酯、二苯基甲烷-4, $4^{\prime}$ -二异氰酸酯、甲苯-2,4-二异氰酸酯、萘-1,5-二异氰酸酯、苯基异氰酸酯、六亚甲基二异氰酸酯、环己基异氰酸酯和以上7种异氰酸酯总量的检测。

# 2 原理

试样中异氰酸酯经二氯甲烷振荡提取后，与9-N-甲氨甲基蒽进行衍生反应，衍生产物采用 $\mathrm{C_{18}}$ 液相色谱柱分离，荧光检测器测定，外标法定量。

# 3 试剂和材料

除非另有说明，所有试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 二氯甲烷 $\mathrm{(CH_2Cl_2)}$ ：5A分子筛干燥过夜，备用。  
3.1.2 乙醚 $(\mathrm{C}_2\mathrm{H}_6\mathrm{O})$ 。  
3.1.3 磷酸 $\mathrm{(HPO_3)}$ 。  
3.1.4 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。  
3.1.5 $N,N$ -二甲基甲酰胺 $\mathrm{(C_3H_7NO)}$ ：纯度大于 $99\%$   
3.1.6 三乙胺 $(\mathrm{C_6H_{15}N})$ 。  
3.1.7 9-N-甲氨甲基蒽 $\mathrm{(C_{16}H_{15}N)}$ ：纯度大于 $99\%$

# 3.2 试剂配制

3.2.1 衍生试剂 $(260\mathrm{mg / L})$ ：称取 $9 - N$ -甲氨甲基蒽 $0.013\mathrm{g}$ ，用二氯甲烷溶解并转移至 $50~\mathrm{mL}$ 容量瓶中，用二氯甲烷定容，避光保存。现用现配。  
3.2.2 衍生物溶剂：量取 $50~\mathrm{mL}N,N$ -二甲基甲酰胺于 $100~\mathrm{mL}$ 容量瓶中，加入 $40~\mathrm{mL}$ 乙腈，用水定容。  
3.2.3 三乙胺溶液 $(3\%)$ ，体积分数)：在 $1\textrm{L}$ 容量瓶中，加入 $30~\mathrm{mL}$ 三乙胺，用水定容。  
3.2.4 缓冲溶液：在 $1\mathrm{L}$ 容量瓶中，加入 $950~\mathrm{mL}~3\%$ 三乙胺溶液（ $3\%$ ，体积分数），用磷酸调节 $\mathrm{pH}$ 至3，用水定容。

# 3.3 标准品

3.3.1 甲苯-2,6-二异氰酸酯 $\mathrm{(C_9H_6N_2O_2}$ ,CAS号：91-08-7)：纯度大于 $98\%$ 。  
3.3.2 二苯基甲烷-4， $4^{\prime}$ -二异氰酸酯 $\mathrm{(C_{15}H_{10}N_2O_2}$ ，CAS号：101-68-8)：纯度大于 $98\%$ 。  
3.3.3 甲苯-2，4-二异氰酸酯 $\mathrm{(C_9H_6N_2O_2}$ ，CAS号：584-84-9）：纯度大于 $98\%$ 。  
3.3.4萘-1，5-二异氰酸酯 $\mathrm{(C_{12}H_6N_2O_2}$ ，CAS号：3173-72-6）：纯度大于 $98\%$   
3.3.5 苯基异氰酸酯 $\mathrm{(C_7H_5ON}$ ，CAS号：103-71-9)：纯度大于 $98\%$   
3.3.6 环己基异氰酸酯 $\mathrm{(C_7H_{11}ON}$ ，CAS号：3173-53-3)：纯度大于 $98\%$ 。  
3.3.7 六亚甲基二异氰酸酯 $\mathrm{(C_8H_{12}O_2N_2}$ ,CAS号：822-06-0）：纯度大于 $98\%$ 。

# 3.4 标准溶液配制

3.4.1 异氰酸酯储备液 $(1000\mathrm{mg / L})$ ：称取各种异氰酸酯标准物质 $0.01\mathrm{g}$ （精确到 $0.1\mathrm{mg}$ ），二氯甲烷溶解后转移至 $10~\mathrm{mL}$ 容量瓶中，二氯甲烷定容。储备液 $-20^{\circ}C$ 避光干燥保存，有效期为1个月。  
3.4.2 异氰酸酯标准中间液 $(100\mathrm{mg / L})$ ：分别量取 $5\mathrm{mL}$ 二氯甲烷于6个 $10~\mathrm{mL}$ 容量瓶中，分别移入 $1.0~\mathrm{mL}$ 各种异氰酸酯储备液，二氯甲烷定容。标准中间液一 $20^{\circ}C$ 避光干燥保存，有效期为1个月。  
3.4.3 异氰酸酯标准工作液 $(1\mathrm{mg / L})$ ：分别量取 $5\mathrm{mL}$ 二氯甲烷于6个 $10~\mathrm{mL}$ 容量瓶中，分别移入 $100~\mu \mathrm{L}$ 各异氰酸酯标准中间液，二氯甲烷定容，混匀。标准工作液一 $20^{\circ}C$ 避光干燥保存，有效期为2周。  
3.4.4 混合异氰酸酯标准工作液：分别量取 $5\mathrm{mL}$ 二氯甲烷于6个 $10~\mathrm{mL}$ 容量瓶中，依次移入 $0.00~\mathrm{mL}$ 、 $0.01~\mathrm{mL}, 0.05~\mathrm{mL}, 0.10~\mathrm{mL}, 0.25~\mathrm{mL}, 0.50~\mathrm{mL}$ 各异氰酸酯标准中间液，二氯甲烷定容，混匀，得到各异氰酸酯浓度均分别为 $0.0~\mathrm{mg / L}, 0.10~\mathrm{mg / L}, 0.50~\mathrm{mg / L}, 1.0~\mathrm{mg / L}, 2.5~\mathrm{mg / L}, 5.0~\mathrm{mg / L}$ 的混合标准工作液。该混合标准工作液 $-20^{\circ}\mathrm{C}$ 避光干燥保存，有效期为2周。

# 4 仪器和设备

4.1 液相色谱仪：配置荧光检测器。  
4.2 玻璃瓶： $20~\mathrm{mL}$ ，配聚四氟乙烯瓶盖；使用前应干燥。  
4.3 水相滤膜： $0.45\mu \mathrm{m}$   
4.4 pH计。  
4.5 分析天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$

# 5 分析步骤

# 5.1 试样前处理

称取 $1\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 剪碎后的试样(面积不大于 $1\mathrm{cm} \times 1\mathrm{cm}$ ), 放入 $20\mathrm{mL}$ 玻璃瓶中, 加入 $10\mathrm{mL}$ 二氯甲烷和 $1\mathrm{mL}$ 衍生试剂, 密封, 避光, 振荡 $12\mathrm{h}$ , 提取液转移至另一 $20\mathrm{mL}$ 玻璃瓶中, 密封, $-20^{\circ}\mathrm{C}$ 下保存。剩余样品残渣重复提取衍生 $12\mathrm{h}$ , 合并提取液, 氮气浓缩至干, 加入 $10\mathrm{mL}$ 衍生物溶剂, 超声溶解, $0.45\mu \mathrm{m}$ 滤膜过滤, 待上机测定。

# 5.2 测定

b）流动相：A为乙腈；B为缓冲溶液，洗脱梯度见表1。

# 5.2.2 标准曲线的制作

分别量取 $1\mathrm{mL}$ 各浓度混合标准工作液于6个 $20~\mathrm{mL}$ 玻璃瓶中，再分别加入 $10~\mathrm{mL}$ 二氯甲烷和 $1\mathrm{mL}$ 衍生试剂，密封，避光反应 $30\mathrm{min}$ ，氮气浓缩至干，再加入 $10~\mathrm{mL}$ 衍生物溶剂，超声溶解， $0.45\mu \mathrm{m}$ 滤膜过滤，按5.2.1的色谱条件进行测定，以试样中各种异氰酸酯浓度为横坐标，以各异氰酸酯衍生物色谱峰峰面积为纵坐标，绘制标准工作曲线，得到线性方程。异氰酸酯标准溶液衍生物色谱图见附录A。

# 5.2.3 测定

按照5.2.1的色谱条件，对5.1中的待测液进行测定。根据各异氰酸酯衍生物保留时间，确定试样中异氰酸酯的种类。根据各异氰酸酯衍生物色谱峰峰面积，由标准曲线计算出样液中异氰酸酯单体量。

# 6 分析结果的表述

# 6.1 异氰酸酯的计算

异氰酸酯按式(1)计算：

$$
X = \frac {\rho V}{m} \tag {1}
$$

式中：

$X$ ——试样中异氰酸酯的浓度，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——样液中待测异氰酸酯的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

结果保留两位有效数字。

# 6.2 异氰酸酯总量的计算

异氰酸酯总量[以异氰酸根(NCO)计]按式(2)计算：

$$
C _ {t} = \sum f X \quad \dots \tag {2}
$$

式中：

$C_t$ ——试样中异氰酸酯总量的浓度，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$f$ ——异氰酸酯单体转化系数，具体数值见表2。

$X$ ——试样中异氰酸酯的浓度，单位为毫克每千克 $(\mathrm{mg / kg})$ 。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法中甲苯-2,6-二异氰酸酯、二苯基甲烷-4, $4^{\prime}$ -二异氰酸酯、甲苯-2,4-二异氰酸酯、萘-1,5-二异氰酸酯、苯基异氰酸酯、环己基异氰酸酯和六亚甲基二异氰酸酯的检出限为 $\mathrm{0.03~mg / kg}$ ，定量限为 $\mathrm{0.1~mg / kg}$ 。

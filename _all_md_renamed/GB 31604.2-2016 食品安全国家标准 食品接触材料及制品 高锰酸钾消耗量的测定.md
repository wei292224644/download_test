# 食品安全国家标准

# 食品接触材料及制品高锰酸钾消耗量的测定

# 1 范围

本标准规定了食品接触材料及制品中高锰酸钾消耗量的测定方法。

本标准适用于食品接触材料及制品中高锰酸钾消耗量的测定。

# 2 原理

试样浸泡液在酸性条件下，用高锰酸钾标准溶液滴定，根据样品消耗的滴定液的体积计算试样中高锰酸钾消耗量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。

3.1.2 高锰酸钾 $\mathrm{(KMnO_4)}$ 。

# 3.2 试剂配制

3.2.1 硫酸溶液 $(1 + 2)$ ：取硫酸 $100\mathrm{mL}$ 小心加入 $200~\mathrm{mL}$ 水中，混匀。

3.2.2 $0.04\%$ 高锰酸钾溶液：称取 $0.4\mathrm{g}$ 高锰酸钾，加水溶解至 $1000~\mathrm{mL}$

3.2.3 草酸标准溶液 $\left[c\left(\frac{1}{2}\mathrm{H}_2\mathrm{C}_2\mathrm{O}_4\right) = 0.1\mathrm{mol} / \mathrm{L}\right]$ ：按GB/T601配制与标定或商品化产品。

3.2.4 草酸标准滴定溶液 $\left[c\left(\frac{1}{2}\mathrm{H}_2\mathrm{C}_2\mathrm{O}_4\right) = 0.01\mathrm{mol} / \mathrm{L}\right]$ ：吸取 $10.0~\mathrm{mL}$ 草酸标准溶液 $\left[c\left(\frac{1}{2}\mathrm{H}_2\mathrm{C}_2\mathrm{O}_4\right) = 0.1\mathrm{mol} / \mathrm{L}\right]$ ，置于 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度。

# 3.3 标准品

高锰酸钾标准溶液 $\left[c\left(\frac{1}{5} \mathrm{KMnO}_4\right) = 0.1 \mathrm{~mol} / \mathrm{L}\right]$ ：按GB/T601配制与标定或商品化产品。

# 3.4 标准溶液配制

高锰酸钾标准滴定溶液 $\left[c\left(\frac{1}{5} \mathrm{KMnO}_4\right) = 0.01 \mathrm{~mol} / \mathrm{L}\right]$ : 吸取 $10.0 \mathrm{~mL}$ 高锰酸钾标准溶液 $\left[c\left(\frac{1}{5} \mathrm{KMnO}_4\right) = 0.1 \mathrm{~mol} / \mathrm{L}\right]$ , 置于 $100 \mathrm{~mL}$ 容量瓶中, 用水定容至刻度。

# 4 分析步骤

# 4.1 采样方法

按GB5009.156操作。

# 4.2 试样的清洗

按GB5009.156操作。

# 4.3 试样的食品模拟物及模拟条件的选择

按相应的产品标准操作。

# 4.4 试样的迁移试验预处理方法

按GB5009.156操作。

# 4.5 试样的测定

# 4.5.1 锥形瓶的处理

取 $100\mathrm{mL}$ 水，放入 $250~\mathrm{mL}$ 锥形瓶中，加入 $5\mathrm{mL}$ 硫酸 $(1 + 2)$ 、 $0.04\%$ 高锰酸钾溶液 $5\mathrm{mL}$ ，煮沸 $5\mathrm{min}$ ，倒去，用水冲洗备用。

# 4.5.2 滴定

准确吸取 $100~\mathrm{mL}$ 水浸泡液(可根据实际情况调整取样量)于上述处理过的 $250~\mathrm{mL}$ 锥形瓶中，加 $5\mathrm{mL}$ 硫酸 $(1 + 2)$ 及 $10.0~\mathrm{mL}$ 高锰酸钾标准滴定溶液 $(0.01\mathrm{mol} / \mathrm{L})$ ，再加玻璃珠2粒，准确煮沸 $5\mathrm{min}$ 后，趁热加入 $10.0~\mathrm{mL}$ 草酸标准滴定溶液 $(0.01\mathrm{mol} / \mathrm{L})$ ，再以高锰酸钾标准滴定溶液 $(0.01\mathrm{mol} / \mathrm{L})$ 滴定至微红色，并在 $0.5\mathrm{min}$ 内不褪色，记取最后的高锰酸钾标准滴定溶液的滴定量。

另取 $100~\mathrm{mL}$ 水做试剂空白试验。

# 5 分析结果的表述

5.1 试样中高锰酸钾消耗量按式(1)计算：

$$
X _ {1} = \frac {\left(V _ {1} - V _ {2}\right) \times c \times 3 1 . 6 \times V}{V _ {3} \times S} \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——试样中高锰酸钾消耗量，单位为毫克每平方分米 $(\mathrm{mg} / \mathrm{dm}^{2})$ ；

$V_{1}$ ——试样浸泡液滴定时消耗高锰酸钾溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——试剂空白滴定时消耗高锰酸钾溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——高锰酸钾标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

31.6——与 $1.00 \mathrm{~mL}$ 的高锰酸钾标准滴定溶液 $\left[c\left(\frac{1}{5} \mathrm{KMnO}_{4}\right)=1.000 \mathrm{~mol} / \mathrm{L}\right]$ 相当的高锰酸钾的质量, 单位为毫克 (mg);

$V$ ——试样浸泡液总体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{3}$ ——测定用浸泡液体积，单位为毫升 $(\mathrm{mL})$ ；

S 与浸泡液接触的试样面积，单位为平方分米 $(\mathrm{dm}^2)$ 。

5.2 当按实际使用情形计算试样中高锰酸钾消耗量时，按式(2)计算：

$$
X _ {2} = X _ {1} \times \frac {S _ {2}}{V _ {4}} \times 1 0 0 0 \tag {2}
$$

式中：

$X_{2}$ ——按实际使用情形计算试样的高锰酸钾消耗量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$X_{1}$ ——试样中高锰酸钾消耗量，单位为毫克每平方分米 $(\mathrm{mg} / \mathrm{dm}^{2})$ ；

$S_{2}$ ——试样实际包装接触面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$V_{4}$ ——试样实际包装的接触体积或质量，单位为毫升 $(\mathrm{mL})$ 或克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 6 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

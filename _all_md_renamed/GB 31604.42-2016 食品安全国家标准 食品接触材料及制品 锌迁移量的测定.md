# 前言

本标准代替GB/T5009.72—2003《铝制食具容器卫生标准的分析方法》、GB/T5009.64—2003《食品用橡胶垫片(圈)卫生标准的分析方法》、GB/T5009.65—2003《食品用高压锅密封圈卫生标准的分析方法》、GB/T5009.66—2003《橡胶奶嘴卫生标准的分析方法》、GB/T5009.79—2003《食品用橡胶管卫生检验方法》和SN/T2829—2011《食品接触材料金属材料食品模拟物中重金属含量的测定电感耦合等离子体发射光谱法》中锌迁移量的测定。

本标准与GB/T5009.72—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 锌迁移量的测定”；  
——增加了火焰原子吸收光谱法；  
——增加了电感耦合等离子体发射光谱法；  
——增加了电感耦合等离子体质谱法。

# 食品安全国家标准

# 食品接触材料及制品 锌迁移量的测定

# 1 范围

本标准规定了食品接触材料及其制品在食品模拟物中浸泡后锌迁移量测定的火焰原子吸收光谱法、电感耦合等离子体发射光谱法、电感耦合等离子体质谱法和二硫腙比色法。

本标准适用于食品接触材料及其制品中锌迁移量的测定。

# 第一法 火焰原子吸收光谱法

# 2 原理

采用食品模拟物浸泡食品接触材料及制品中预期与食品接触的部分，浸泡液用火焰原子吸收光谱仪进行测定，以 $213.9\mathrm{nm}$ 处吸收强度与其质量浓度的定量关系，测定样品中锌的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为优级纯，水为 $\mathrm{GB / T6682}$ 规定的二级水。

# 3.1 试剂

3.1.1 硝酸 $\mathrm{(HNO_3)}$ 。  
3.1.2 配制食品模拟物所需试剂：依据GB31604.1的规定。

# 3.2 试剂配制

3.2.1 食品模拟物：按照GB5009.156的规定配制。   
3.2.2 硝酸溶液 $(1 + 1)$ ：量取 $500~\mathrm{mL}$ 硝酸，加至 $500~\mathrm{mL}$ 水，混匀。  
3.2.3 硝酸溶液 $(5 + 95)$ ：吸取 $5\mathrm{mL}$ 硝酸，用水稀释至 $100~\mathrm{mL}$

# 3.3 标准品

氧化锌(ZnO，CAS号：1314-13-2)：纯度 $>99.99\%$ ，或经国家认证并授予标准物质证书的一定浓度的锌标准溶液。

# 3.4 标准溶液配制

3.4.1锌标准储备液 $(1000\mathrm{mg / L})$ ：准确称取 $1.2447\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )氧化锌，加少量硝酸溶液(1+1)，加热溶解，移入 $1000~\mathrm{mL}$ 容量瓶，加水至刻度。混匀。  
3.4.2 锌标准中间液 $(10.0\mathrm{mg / L})$ ：吸取锌标准储备液 $1.00~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加硝酸溶液（5+95)至刻度，混匀。

3.4.3锌标准系列溶液：分别吸取锌标准中间液 $0\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},4.00\mathrm{mL},8.00\mathrm{mL},10.0\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加相应食品模拟物至刻度，混匀。此系列锌标准溶液浓度分别为 $0\mathrm{mg / L}$ 产 $0.100\mathrm{mg / L},0.200\mathrm{mg / L},0.400\mathrm{mg / L},0.800\mathrm{mg / L},1.00\mathrm{mg / L}$

注：可根据仪器的灵敏度、线性范围及浸泡液中锌的实际浓度确定标准溶液系列中锌的具体浓度。若所选食品模拟物为中性或碱性，则需添加适量硝酸使该溶液中硝酸浓度约为 $5\%$ （体积分数）。

# 4 仪器和设备

注：所有玻璃器皿均需硝酸溶液 $(1 + 5)$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

4.1 原子吸收光谱仪：配火焰原子化器，锌空心阴极灯。  
4.2 分析天平：感量 $0.1\mathrm{mg}$

# 5 分析步骤

# 5.1 试样的预处理

根据待测样品的预期用途和使用条件，按照GB5009.156和GB31604.1规定的迁移试验方法及试验条件进行迁移试验。浸泡液经充分混匀后，取部分浸泡试液用于分析。若浸泡试液为中性或碱性，则添加适量硝酸使试液中硝酸浓度约为 $5\%$ （体积分数）。同时做试样空白试验。

# 5.2 测定

# 5.2.1 仪器参考条件

参考条件见表A.1。

# 5.2.2 标准曲线制作

按浓度由低到高的顺序将标准系列溶液分别导入火焰原子化器，原子化后测其吸光度值，以浓度为横坐标，吸光度值为纵坐标，制作标准曲线。

# 5.2.3 试样溶液测定

在与测定标准溶液相同的试验条件下，将试样空白和试样浸泡液分别导入火焰原子化器，原子化后测其吸光度值，与标准系列比较定量。

# 6 分析结果的表述

由标准曲线得到试样溶液中锌的浓度，扣除空白值后按GB5009.156进行迁移量计算，得到食品接触材料及制品中锌的迁移量。计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

方法检出限为 $0.02\mathrm{mg / L}$ ，定量限为 $0.06\mathrm{mg / L}$ 。

# 第四法 二硫腙比色法

# 9 原理

试样经浸泡后，在 $\mathrm{pH}4.0\sim 5.5$ 时，锌离子与二硫腙形成紫红色络合物，溶于四氯化碳，加入硫代硫酸钠防止铜、汞、铅、铋、银和镉等离子干扰，与标准系列比较定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的三级水。

# 10.1 试剂

10.1.1 冰乙酸 $\mathrm{(CH_3COOH)}$ ：优级纯。  
10.1.2 硝酸 $\mathrm{(HNO_3)}$ ：优级纯。  
10.1.3 乙酸钠 $\mathrm{(CH_3COONa\cdot 3H_2O)}$   
10.1.4 氨水 $(\mathrm{NH}_3\bullet \mathrm{H}_2\mathrm{O})$ 。  
10.1.5 盐酸（HCl）。  
10.1.6 盐酸羟胺 $\mathrm{(NH_4OCl)}$   
10.1.7 硫代硫酸钠 $(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3)$ 。  
10.1.8 二硫腙 $(\mathrm{C_{13}H_{12}N_4S})$ 。  
10.1.9 甲基橙 $(\mathrm{C_{14}H_{14}N_3NaO_3S})$ 。  
10.1.10 乙醇 $\mathrm{(C_2H_5OH)}$ 。  
10.1.11 配制食品模拟物所需试剂：依据GB31604.1的规定。

# 10.2 试剂配制

10.2.1 食品模拟物：按照GB5009.156的规定配制。   
10.2.2 乙酸溶液 $(2\mathrm{mol} / \mathrm{L})$ ：量取 $10~\mathrm{mL}$ 冰乙酸，用水稀释至 $85~\mathrm{mL}$ ，混匀。  
10.2.3 硝酸溶液 $(1 + 1)$ ：量取 $500~\mathrm{mL}$ 硝酸，加入 $500~\mathrm{mL}$ 水，混匀。  
10.2.4 硝酸溶液 $(5 + 95)$ ：吸取 $5\mathrm{mL}$ 硝酸，用水稀释至 $100~\mathrm{mL}$   
10.2.5 乙酸钠溶液 $(2\mathrm{mol} / \mathrm{L})$ ：称取 $68\mathrm{g}$ 乙酸钠，加水溶解，定容至 $250~\mathrm{mL}$ 。  
10.2.6 氨水溶液 $(1 + 1)$ ：量取 $25~\mathrm{mL}$ 氨水，加水 $25~\mathrm{mL}$ ，混匀。  
10.2.7 盐酸溶液 $(2\mathrm{mol} / \mathrm{L})$ ：量取 $10~\mathrm{mL}$ 盐酸，加水稀释至 $60~\mathrm{mL}$ ，混匀。  
10.2.8 盐酸溶液 $(0.02\mathrm{mol} / \mathrm{L})$ ：吸取 $1\mathrm{mL}$ 盐酸溶液 $(2\mathrm{mol} / \mathrm{L})$ ，加水稀释至 $100~\mathrm{mL}$ ，混匀。

10.2.9 二硫腙-四氯化碳溶液 $(0.1\mathrm{g} / \mathrm{L})$ ：称取 $0.1\mathrm{g}$ 二硫腙，用四氯化碳溶解，定容至 $1000~\mathrm{mL}$ 。  
10.2.10 甲基橙指示剂 $(2\mathrm{g} / \mathrm{L})$ ：称取 $0.2\mathrm{g}$ 甲基橙，用乙醇溶解，定容至 $100~\mathrm{mL}$ 。  
10.2.11 乙酸-乙酸盐缓冲液：量取乙酸溶液 $(2\mathrm{mol} / \mathrm{L})$ 与乙酸钠溶液 $(2\mathrm{mol} / \mathrm{L})$ 等量混合，此溶液 $\mathrm{pH}$ 为4.7左右。用二硫腙-四氯化碳溶液 $(0.1\mathrm{g} / \mathrm{L})$ 提取数次，每次 $10~\mathrm{mL}$ ，除去其中的锌，至四氯化碳层绿色不变为止，弃去四氯化碳层，再用四氯化碳提取乙酸-乙酸盐缓冲液中过剩的二硫腙，至四氯化碳无色，弃去四氯化碳层。  
10.2.12 硫代硫酸钠溶液 $(250\mathrm{g / L})$ ：称取 $25\mathrm{g}$ 硫代硫酸钠，加水 $60~\mathrm{mL}$ ，用乙酸溶液 $(2\mathrm{mol} / \mathrm{L})$ ，调节 $\mathrm{pH}$ 至 $4.0\sim 5.5$ ，加水至 $100~\mathrm{mL}$ ，用二硫腙-四氯化碳溶液 $(0.1\mathrm{g / L})$ 提取数次，每次 $10~\mathrm{mL}$ ，除去其中的锌，至四氯化碳层绿色不变为止，弃去四氯化碳层，再用四氯化碳提取乙酸-乙酸盐缓冲液中过剩的二硫腙，至四氯化碳无色，弃去四氯化碳层。  
10.2.13 二硫腙-四氯化碳使用液：吸取 $1\mathrm{mL}$ 二硫腙-四氯化碳溶液 $(0.1\mathrm{g / L})$ ，加四氯化碳至 $10~\mathrm{mL}$ 混匀。用 $1\mathrm{cm}$ 比色杯，以四氯化碳调节零点，于波长 $520~\mathrm{nm}$ 处测吸光度(A)。用式(1)计算出配制 $100~\mathrm{mL}$ 二硫腙-四氯化碳使用液 $(57\%$ 透光率)所需的二硫腙-四氯化碳溶液 $(0.1\mathrm{g / L})$ 的毫升数(V)。吸取 $V\mathrm{mL}$ 二硫腙-四氯化碳溶液 $(0.1\mathrm{g / L})$ ，用四氯化碳稀释至 $100~\mathrm{mL}$ 。

$$
V = \frac {1 0 \times (2 - \lg 5 7)}{A} = \frac {2 . 4 4}{A} \quad \dots \tag {1}
$$

# 10.3 标准品

氧化锌(ZnO，CAS号：1314-13-2)：纯度 $>99.99\%$ ，或经国家认证并授予标准物质证书的一定浓度的锌标准溶液。

# 10.4 标准溶液配制

10.4.1 锌标准储备液 $(1000\mathrm{mg / L})$ ：准确称取 $1.2447\mathrm{g}$ 氧化锌，加少量硝酸溶液 $(1 + 1)$ ，加热溶解，移入 $1000~\mathrm{mL}$ 容量瓶，加水至刻度。混匀。  
10.4.2 锌标准使用液 $(1.00\mathrm{mg / L})$ ：吸取锌标准储备液 $1.00~\mathrm{mL}$ 于 $1000~\mathrm{mL}$ 容量瓶中，加硝酸溶液 $(5 + 95)$ 至刻度，混匀。

# 11 仪器和设备

注：所有玻璃器皿均需硝酸溶液 $(1 + 5)$ 浸泡过夜，用自来水反复冲洗，最后用水冲洗干净。

11.1 可见分光光度计。  
11.2 分析天平：感量 $0.1\mathrm{mg}$

# 12 分析步骤

# 12.1 试样的预处理

同5.1。

# 12.2 测定

# 12.2.1 仪器测试条件

测定波长： $530~\mathrm{nm}$

# 12.2.2 标准曲线的制作

取分液漏斗6个，分别加入 $0\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},3.00\mathrm{mL},4.00\mathrm{mL},5.00\mathrm{mL}$ 锌标准使用液（相当于 $0\mu \mathrm{g},1.00\mu \mathrm{g},2.00\mu \mathrm{g},3.00\mu \mathrm{g},4.00\mu \mathrm{g},5.00\mu \mathrm{g}$ 锌）。向各分液漏斗中加相应食品模拟物至 $10.0~\mathrm{mL}$ 。再各加入甲基橙指示剂( $2\mathrm{g / L})$ 1滴，用氨水溶液 $(1 + 1)$ 中和溶液由红刚好变黄。各分液漏斗内加 $5\mathrm{mL}$ 乙酸-乙酸盐缓冲液及 $1\mathrm{mL}$ 硫代硫酸钠溶液 $(250~\mathrm{g / L})$ ，混匀后再各加 $10.0~\mathrm{mL}$ 二硫腙四氯化碳使用液，振摇 $2\mathrm{min}$ ，静置分层。用脱脂棉将四氯化碳层滤入 $1\mathrm{cm}$ 比色杯，以四氯化碳调节零点，于波长 $530~\mathrm{nm}$ 处测吸光度，以质量为横坐标，吸光度值为纵坐标，制作标准曲线。

# 12.2.3 试样测定

准确移取 $5.00\mathrm{mL}\sim 10.0\mathrm{mL}$ 试样浸泡液和相同体积的空白液，置于 $125~\mathrm{mL}$ 分液漏斗中，向各分液漏斗中加相应食品模拟物至 $10.0~\mathrm{mL}$ 。再各加入甲基橙指示剂 $(2\mathrm{g / L})$ 1滴，用氨水溶液 $(1 + 1)$ 中和溶液由红刚好变黄。各分液漏斗内加 $5\mathrm{mL}$ 乙酸-乙酸盐缓冲液及 $1\mathrm{mL}$ 硫代硫酸钠溶液 $(250~\mathrm{g / L})$ ，混匀后再各加 $10.0~\mathrm{mL}$ 二硫腙-四氯化碳使用液，振摇 $2\mathrm{min}$ ，静置分层。用脱脂棉将四氯化碳层滤入 $1\mathrm{cm}$ 比色杯，以四氯化碳调节零点，于波长 $530~\mathrm{nm}$ 处测吸光度，最后与标准曲线比较定量。

# 13 分析结果的表述

# 13.1 食品模拟物试液中锌的浓度

食品模拟物试液中锌的浓度按式(2)计算：

$$
X = \frac {m _ {1} - m _ {0}}{V _ {1}} \tag {2}
$$

式中：

$X$ ——食品模拟物试液中锌的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$m_{1}$ ——测定所准确吸取的试样浸泡液中锌的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m_0$ ——测定所准确吸取的空白液中锌的质量，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——所取试样浸泡液体积，单位为毫升 $(\mathrm{mL})$ 。

# 13.2 锌迁移量的计算

由13.1得到的食品模拟物试液中锌的含量，按GB5009.156进行迁移量的计算，得到食品接触材料及制品中锌的迁移量。计算结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 15 其他

当测定用浸泡液的体积为 $5\mathrm{mL}$ 时，方法检出限为 $0.2\mathrm{mg / L}$ ，定量限为 $0.5\mathrm{mg / L}$ 。

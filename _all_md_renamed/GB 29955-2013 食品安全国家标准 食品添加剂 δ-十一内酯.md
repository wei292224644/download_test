# 食品安全国家标准

# 食品添加剂 $\delta$ -十一内酯

# 1 范围

本标准适用于由环戊酮和己醛为原料制得的食品添加剂 $\delta$ 十一内酯。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

6-己基四氢-2-吡喃酮

# 2.2 分子式

$\mathrm{C_{11}H_{20}O_2}$

# 2.3 结构式

![](images/052d4f2d938ad4a5ecb25e30c4f06656f1e99f1ee654abc997deae15c15ef474.jpg)

# 2.4 相对分子质量

184.28（按2007年国际相对原子质量）

# 附录A

# $\delta$ -十一内酯含量的测定

# A. 1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A.1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

试样制备：量取试样 $1\mathrm{mL}$ 溶于 $4\mathrm{mL}$ 丙酮中，摇匀备用。

# A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行，应符合要求。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$   
B.2.2 固定相： $5\%$ 苯基甲基聚硅氧烷。  
B.2.3 膜厚： $0.25\mu \mathrm{m}$   
B.2.4 色谱炉温度：线性程序升温从 $50~\mathrm{^\circ C}\sim 180~\mathrm{^\circ C}$ ，速率 $4\mathrm{^{\circ}C / min}$ ；然后线性程序升温从 $180~\mathrm{^\circ C}\sim$ $250~\mathrm{^\circ C}$ ，速率 $25~\mathrm{^\circ C / min}$ 。  
B.2.5 进样口温度： $250~\mathrm{^\circ C}$ 。  
B.2.6 检测器温度： $250~\mathrm{^\circ C}$ 。  
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 柱前压： $34.45\mathrm{kPa}$   
B.2.10 进样量： $0.2\mu \mathrm{L}$   
B.2.11 分流比： $100:1$

# 1 范围

本标准规定了植物源性食品中9种氨基甲酸酯类农药及其代谢物(参见附录A)残留量的液相色谱-柱后衍生测定方法。

本标准适用于植物源性食品中9种氨基甲酸酯类农药及其代谢物残留量的测定。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样用乙腈提取，提取液经固相萃取或分散固相萃取净化，使用带荧光检测器和柱后衍生系统的高效液相色谱仪检测，外标法定量。

# 4 试剂和材料

除非另有说明，在分析中仅使用分析纯的试剂，水为GB/T6682规定的一级水。

4.1.2 甲醇 $\mathrm{(CH_3OH, CAS}$ 号：67-56-1）：色谱纯。  
4.1.3 二氯甲烷 $\mathrm{(CH_2Cl_2}$ ,CAS号：75-09-2):色谱纯。

4.1.7 2-二甲胺基乙硫醇盐酸盐 $\left(\mathrm{C}_{4} \mathrm{H}_{12} \mathrm{ClNS}, \mathrm{CAS}\right.$ 号：13242-44-9)，或相当者。

# 4.2 溶液配制

4.2.1 甲醇-二氯甲烷溶液(1+99，体积比)：量取 $10\mathrm{mL}$ 甲醇加入 $990~\mathrm{mL}$ 二氯甲烷中，混匀。  
4.2.2 乙腈-甲苯溶液(3+1，体积比)：量取 $100\mathrm{mL}$ 甲苯加入 $300\mathrm{mL}$ 乙腈中，混匀。  
4.2.3 氢氧化钠溶液 $(0.05\mathrm{mol} / \mathrm{L})$ ：称取 $2.0\mathrm{g}$ 氢氧化钠，用水溶解并定容至 $1000~\mathrm{mL}$ ，混匀。

4.2.4 十水四硼酸钠溶液 $(4\mathrm{g} / \mathrm{L})$ ：称取 $4.0\mathrm{g}$ 十水四硼酸钠，用水溶解并定容至 $1000~\mathrm{mL}$ ，混匀。  
4.2.5 OPA 试剂: 称取 $50.0 \mathrm{mg}$ 邻苯二甲醛, 溶于 $5 \mathrm{~mL}$ 甲醇中, 混匀; 再称取 $1.0 \mathrm{~g} 2-$ 二甲胺基乙硫醇盐酸盐, 溶于 $5 \mathrm{~mL}$ 十水四硼酸钠溶液(4.2.4), 混匀; 将上述 2 种溶液倒入 $490 \mathrm{~mL}$ 十水四硼酸钠溶液(4.2.4), 混匀。

# 4.3 标准品

9种氨基甲酸酯类农药及其代谢物标准品参见附录A，纯度 $\geqslant 95\%$

# 4.4 标准溶液配制

4.4.1 标准储备溶液 $(1000\mathrm{mg / L})$ ：准确称取 $10\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )各农药标准品，用甲醇溶解并分别定容到 $10~\mathrm{mL}$ 。标准储备溶液避光一 $18^{\circ}C$ 保存，有效期1年。  
4.4.2 混合标准溶液：准确吸取一定量的单个农药储备溶液于 $10\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度。混合标准溶液，避光 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存，有效期1个月。

# 4.5 材料

4.5.1 固相萃取柱1：氨基填料 $(\mathrm{NH_2})500\mathrm{mg},6\mathrm{mL}$   
4.5.2固相萃取柱2：石墨化炭黑填料(GCB) $500\mathrm{mg}$ ，氨基填料 $(\mathrm{NH}_2)500\mathrm{mg},6\mathrm{mL}$   
4.5.3 乙二胺-N-丙基硅烷硅胶(PSA)： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}_{\circ}$   
4.5.4 十八烷基甲硅烷改性硅胶 $(\mathrm{C}_{18})$ : $40~\mu \mathrm{m}\sim 60\mu \mathrm{m}$ 。  
4.5.5 陶瓷均质子： $2\mathrm{cm}$ （长） $\times 1\mathrm{cm}$ （外径）。  
4.5.6 微孔滤膜(有机相)： $0.22\mu \mathrm{m}\times 25\mathrm{mm}_{\circ}$

# 5 仪器设备

5.1 液相色谱仪：配有柱后衍生反应装置和荧光检测器(FLD)。  
5.2 分析天平：感量 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。  
5.3 高速匀浆机：转速不低于 $15000\mathrm{r / min}$ 。  
5.4 高速离心机：转速不低于 $4200\mathrm{r / min}$ 。  
5.5 组织捣碎机。  
5.6 旋转蒸发仪。  
5.7 氮吹仪：可控温。  
5.8 涡旋振荡器。

# 6 试样的制备

# 6.1 试样制备

蔬菜和水果的取样量按照相关标准的规定执行，食用菌样品随机取样 $1\mathrm{kg}$ 。样品取样部位按照GB2763的规定执行。对于个体较小的样品，取样后全部处理；对于个体较大的基本均匀样品，可在对称轴或对称面上分割或切成小块后处理；对于细长、扁平或组分含量在各部分有差异的样品，可在不同部位切取小片或截成小段后处理；取后的样品将其切碎，充分混匀，用四分法取样或直接放入组织捣碎机中捣碎成匀浆，放入聚乙烯瓶中。

取谷类样品 $500\mathrm{g}$ ，粉碎后使其全部可通过 $425\mu \mathrm{m}$ 的标准网筛，放入聚乙烯瓶或袋中。取油料作物、茶叶、坚果和香辛料各 $500\mathrm{g}$ ，粉碎后充分混匀，放入聚乙烯瓶或袋中。

植物油类搅拌均匀，放入聚乙烯瓶中。

# 6.2 试样储存

试样于 $-18^{\circ}C$ 条件下保存。

# 7 分析步骤

# 7.1 提取和净化

# 7.1.1 蔬菜、水果和食用菌

称取 $20\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）于 $150\mathrm{mL}$ 烧杯中，加入 $40\mathrm{mL}$ 乙腈，用高速匀浆机 $15000\mathrm{r / min}$ 匀浆提取 $2\mathrm{min}$ ，提取液过滤至装有 $5\mathrm{g} \sim 7\mathrm{g}$ 氯化钠的 $100\mathrm{mL}$ 具塞量筒中，盖上塞子，剧烈振荡 $1\mathrm{min}$ ，在室温下静置 $30\mathrm{min}$ 。准确吸取 $10\mathrm{mL}$ 上清液， $80^{\circ}\mathrm{C}$ 水浴中氮吹蒸发近干，加入 $2\mathrm{mL}$ 甲醇溶解残余物，待净化。

将固相萃取柱1(4.5.1)用 $4\mathrm{mL}$ 甲醇-二氯甲烷溶液（4.2.1）预淋洗，当液面到达柱筛板顶部时，立即加入上述待净化溶液，用 $10\mathrm{mL}$ 离心管收集洗脱液，用 $2\mathrm{mL}$ 甲醇-二氯甲烷溶液（4.2.1）涮洗烧杯后过柱，并重复一次，收集的洗脱液于 $50^{\circ}\mathrm{C}$ 水浴中氮吹蒸发近干，准确加入 $2.50\mathrm{mL}$ 甲醇，涡旋混匀，用微孔滤膜（4.5.6）过滤，待测。

# 7.1.2 谷物

称取 $10\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）于 $250\mathrm{mL}$ 具塞锥形瓶中，加入 $20\mathrm{mL}$ 水，混匀后，静置 $30\mathrm{min}$ ，再加入 $50\mathrm{mL}$ 乙腈，用振荡器 $200\mathrm{r / min}$ 振荡提取 $30\mathrm{min}$ ，提取液过滤至装有 $5\mathrm{g} \sim 7\mathrm{g}$ 氯化钠的 $100\mathrm{mL}$ 具塞量筒中，盖上塞子，剧烈振荡 $1\mathrm{min}$ ，在室温下静止 $30\mathrm{min}$ 。准确吸取 $10\mathrm{mL}$ 上清液， $80^{\circ}\mathrm{C}$ 水浴中氮吹蒸发近干，加入 $2\mathrm{mL}$ 甲醇溶解残余物，待净化。

将固相萃取柱1(4.5.1)用 $4\mathrm{mL}$ 甲醇二氯甲烷溶液（4.2.1）预淋洗，当液面到达柱筛板顶部时，立即加入上述待净化溶液，用 $10\mathrm{mL}$ 离心管收集洗脱液，用 $2\mathrm{mL}$ 甲醇二氯甲烷溶液（4.2.1）涮洗烧杯后过柱，并重复一次，收集的洗脱液于 $50^{\circ}\mathrm{C}$ 水浴中氮吹蒸发近千，准确加入 $2.50\mathrm{mL}$ 甲醇，涡旋混匀，用微孔滤膜(4.5.6)过滤，待测。

# 7.1.3 茶叶和香辛料

称取 $5\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）于 $150\mathrm{mL}$ 烧杯中，加入 $20\mathrm{mL}$ 水，混匀后，静置 $30\mathrm{min}$ ，再加入 $50\mathrm{mL}$ 乙腈，用高速匀浆机 $15000\mathrm{r / min}$ 匀浆提取 $2\mathrm{min}$ ，提取液过滤至装有 $5\mathrm{g} \sim 7\mathrm{g}$ 氯化钠的 $100\mathrm{mL}$ 具塞量筒中，盖上塞子，剧烈振荡 $1\mathrm{min}$ ，在室温下静置 $30\mathrm{min}$ 。准确吸取 $10\mathrm{mL}$ 上清液， $80^{\circ}\mathrm{C}$ 水浴中氮吹蒸发近干，加入 $2\mathrm{mL}$ 乙腈-甲苯溶液(4.2.2)溶解残余物，待净化。

将固相萃取柱2(4.5.2)用 $5\mathrm{mL}$ 乙腈-甲苯溶液（4.2.2）预淋洗，当液面到达柱筛板顶部时，立即加入上述待净化溶液，用 $100~\mathrm{mL}$ 旋转蒸发瓶收集洗脱液，用 $2\mathrm{mL}$ 乙腈-甲苯溶液（4.2.2）涮洗烧杯后过柱，并重复一次，再用 $25~\mathrm{mL}$ 乙腈-甲苯溶液（4.2.2）洗脱柱子，收集的洗脱液于 $40^{\circ}\mathrm{C}$ 水浴中旋转蒸发近千，用 $5\mathrm{mL}$ 甲醇冲洗旋转蒸发瓶并转移到 $10~\mathrm{mL}$ 离心管中， $50^{\circ}\mathrm{C}$ 水浴中氮吹蒸发近千，准确加入1.00mL甲醇，涡旋混匀，用微孔滤膜（4.5.6）过滤，待测。

# 7.1.4 油料和坚果

称取 $10\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）于 $150\mathrm{mL}$ 烧杯中，加入 $20\mathrm{mL}$ 水，混匀后，静置 $30\mathrm{min}$ ，再加入 $50\mathrm{mL}$ 乙腈，用高速匀浆机 $15000\mathrm{r / min}$ 匀浆提取 $2\mathrm{min}$ ，提取液过滤至装有 $5\mathrm{g} \sim 7\mathrm{g}$ 氯化钠的 $100\mathrm{mL}$ 具塞量筒中，盖上塞子，剧烈振荡 $1\mathrm{min}$ ，在室温下静置 $30\mathrm{min}$ 。

准确吸取 $8\mathrm{mL}$ 上清液于内含 $1200\mathrm{mg}$ 无水硫酸镁、 $400\mathrm{mg}$ PSA和 $400\mathrm{mgC_{18}}$ 的 $15\mathrm{mL}$ 塑料离心管中，涡旋混匀 $1\mathrm{min}$ ，然后 $4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，吸取 $5\mathrm{mL}$ 上清液于 $10\mathrm{mL}$ 离心管中，在 $50^{\circ}\mathrm{C}$ 水浴中氮吹蒸发近干，准确加入 $2.00\mathrm{mL}$ 甲醇，涡旋混匀，用微孔滤膜(4.5.6)过滤，待测。

# 7.1.5 植物油

称取 $3\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）于 $50\mathrm{mL}$ 塑料离心管中，加入 $5\mathrm{mL}$ 水、 $15\mathrm{mL}$ 乙腈，并加入 $6\mathrm{g}$ 无水硫酸镁、 $1.5\mathrm{g}$ 醋酸钠及1颗陶瓷均质子，剧烈振荡 $1\mathrm{min},4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。

准确吸取 $8\mathrm{mL}$ 上清液于内含 $1200\mathrm{mg}$ 无水硫酸镁、 $400\mathrm{mg}$ PSA和 $400\mathrm{mgC_{18}}$ 的 $15\mathrm{mL}$ 塑料离心

管中, 涡旋混匀 $1 \mathrm{~min}$ , 然后 $4200 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 吸取 $5 \mathrm{~mL}$ 上清液于 $10 \mathrm{~mL}$ 离心管中, 在 $50^{\circ} \mathrm{C}$ 水浴中氮吹蒸发近干, 准确加入 $1.00 \mathrm{~mL}$ 甲醇, 涡旋混匀, 用微孔滤膜(4.5.6)过滤, 待测。

# 7.2.2 标准工作曲线

精确吸取一定量的混合标准溶液，逐级用甲醇稀释成质量浓度为 $0.01\mathrm{mg / L},0.05\mathrm{mg / L},0.1\mathrm{mg / }$ $\mathrm{L},0.5\mathrm{mg / L}$ 和 $1.0\mathrm{mg / L}$ 的标准工作溶液，供液相色谱测定。以农药质量浓度为横坐标、色谱峰的峰面积为纵坐标，绘制标准曲线。

# 7.2.3 定性及定量

# 7.2.3.1 定性

以目标农药的保留时间定性。被测试样中目标农药色谱峰的保留时间与相应标准色谱峰的保留时间相比较，相差应在 $\pm 0.05\mathrm{min}$ 之内。阳性试样需更换 $C_{18}$ 柱进行定性确认。

# 7.2.3.2 定量

外标法定量。

# 7.3 试样溶液的测定

将混合标准工作溶液和试样溶液依次注入液相色谱仪中，保留时间定性，测得目标农药色谱峰面积，根据式(1)，得到各农药组分含量。待测样液中农药的响应值应在仪器检测的定量测定线性范围之内，超过线性范围时，应根据测定浓度进行适当倍数稀释后再进行分析。

# 7.4 平行试验

按 $7.1 \sim 7.3$ 的规定对同一试样进行平行试验测定。

# 7.5 空白试验

除不加试料外，按 $7.1\sim 7.4$ 的规定进行平行操作。

# 8 结果计算

试样中各农药残留量以质量分数 $\omega$ 计，单位以毫克每千克 $(\mathrm{mg / kg})$ 表示，按式(1)计算。

$$
\omega = \frac {V _ {1} \times A \times V _ {3}}{V _ {2} \times A _ {S} \times m} \times \rho \tag {1}
$$

式中：

$\omega$ ——样品中被测组分含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——标准溶液中被测组分质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V_{\mathrm{f}}$ ——提取溶剂总体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——提取液分取体积，单位为毫升 $(\mathrm{mL})$

$V_{3}$ ——待测溶液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A$ ——待测溶液中被测组分峰面积；

$A_{S}$ ——标准溶液中被测组分峰面积；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

计算结果应扣除空白值，计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，保留2位有效数字。含量超过 $1\mathrm{mg / kg}$ 时，保留3位有效数字。

# 9 精密度

在重复性条件下，获得的2次独立测试结果的绝对差值不得超过重复性限 $(r)$ ，参见附录B。

在再现性条件下，获得的2次独立测试结果的绝对差值不得超过再现性限(R)，参见附录B。

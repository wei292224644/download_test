# 检验方法

# A.1 安全提示

本标准试验方法中使用的部分试剂具有毒性或腐蚀性，按相关规定操作，使用时需小心谨慎并按相关规定操作。若溅到皮肤上应立即用水冲洗，严重者应立即治疗。在使用挥发性酸时，要在通风橱中进行。

# A.2 一般规定

本标准除另有规定外，所用试剂纯度应在分析纯以上，所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，应分别按GB/T601、GB/T602、GB/T603的规定制备，实验用水应符合GB/T6682中三级水的规定。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.4 D-泛酸钠含量的测定

# A.4.1 方法提要

采用高效液相色谱法测定， $\mathrm{C_{18}}$ 色谱柱分离，紫外检测器检测，峰面积外标法定量计算样品中D-泛酸钠的含量。

# A.4.2 试剂和材料

A.4.2.1 甲醇。  
A.4.2.2 磷酸二氢钾。  
A.4.2.3 磷酸。  
A.4.2.4 D-泛酸钠标准品(SodiumPantothenate.CAS:867-81-2)，纯度 $\geqslant 99\%$   
A.4.2.5 磷酸二氢钾溶液 $(0.05\mathrm{mol} / \mathrm{L})$ ：称取 $6.8\mathrm{g}$ 磷酸二氢钾(A.4.2.2)，用水溶解并定容至 $1000~\mathrm{mL}$ 。用磷酸(A.4.2.3)调 $\mathrm{pH}$ 至3.0，经 $0.45\mu \mathrm{m}$ 水相滤膜过滤备用。

# A.4.3 仪器和设备

A.4.3.1 天平：感量为 $1\mathrm{mg}$   
A.4.3.2 高效液相色谱仪，带紫外检测器。  
A.4.3.3 pH计：精度为0.01。

# A.4.4 参考色谱条件

色谱参考条件为：

a）色谱柱： $\mathrm{C_{18}(250~mm\times 4.6~mm}$ ，粒径 $5\mu \mathrm{m})$ 或同等性能色谱柱；  
b）流动相：量取 $100\mathrm{mL}$ 甲醇(A.4.2.1)， $900\mathrm{mL}$ 磷酸二氢钾溶液(A.4.2.5)，混匀后经 $0.45\mu \mathrm{m}$ 滤膜过滤备用；  
c) 流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
d) 检测波长: $200 \mathrm{~nm}$ ;   
e) 柱温： $30^{\circ}\mathrm{C}$ ；  
f）进样量： $10~\mu \mathrm{L}$

# A.4.5 分析步骤

# A.4.5.1 标准溶液的制备

称取 $0.05\mathrm{g}$ （精确至 $1\mathrm{mg}$ ）D-泛酸钠标准品，加水溶解并定容至 $100~\mathrm{mL}$ 。该标准溶液每毫升含 $0.5\mathrm{mgD}$ -泛酸钠。

# A.4.5.2 试样溶液的制备

称取 $0.05\mathrm{g}$ （精确至 $1\mathrm{mg}$ ）预先干燥的试样，用水稀释至 $100~\mathrm{mL}$ ，作为试样溶液。

# A.4.5.3 测定

在A.4.4色谱条件下，分别对标准溶液和试样溶液进行色谱分析。将试样溶液的色谱图与标准溶液色谱图相比较，以确定试样溶液色谱图中D-泛酸钠对应的峰(保留时间 $t_{\mathrm{R}}$ 为 $13.2\mathrm{min}$ )。根据标准溶液和试样溶液色谱图中D-泛酸钠的峰面积，计算出D-泛酸钠含量。

# A.4.6 结果计算

D-泛酸钠的含量的质量分数 $\omega$ ，按式(A.1)计算：

$$
w = \frac{m_{\mathrm{s}}}{m_{\mathrm{t}}}\times \frac{A_{\mathrm{t}}}{A_{\mathrm{s}}}\times 100\% \tag{A.1}
$$

式中：

$m_{\mathrm{s}}$ ——标准溶液中D-泛酸钠的质量，单位为毫克 $(\mathrm{mg})$ ；

$m_{\mathrm{t}}$ ——试样质量，单位为毫克 $(\mathrm{mg})$ ；

$A_{\mathrm{t}}$ ——试样溶液色谱图中D-泛酸钠的峰面积值；

$A_{\mathrm{s}}$ ——标准溶液色谱图中D-泛酸钠的峰面积值。

以两次平行测定结果的算术平均值为测定结果。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

# A.5 比旋光度的测定

# A.5.1 仪器和设备

A.5.1.1 天平：感量为 $0.01\mathrm{g}$   
A.5.1.2 比旋光度计。

# A.5.2 分析步骤

称取 $1.25\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )预先干燥的试样，加水溶解并定容至 $25~\mathrm{mL}$ ，按GB/T613规定的方法测定，计算D-泛酸钠的比旋光度。

# A.5.3 结果计算

D-泛酸钠的比旋光度 $\alpha_{\mathrm{m}}(20^{\circ}\mathrm{C},\mathrm{D})$ ，单位以“ $(^{\circ})\cdot \mathrm{dm}^{2}\cdot \mathrm{kg}^{-1}$ ”表示，按式(A.2)计算：

$$
\alpha_ {\mathrm {m}} (2 0 ^ {\circ} \mathrm {C}, D) = \frac {\alpha}{l \times \rho} \quad \dots \tag {A.2}
$$

式中：

$\alpha$ 测得的旋光角，单位为度 $(^{\circ})$

$l$ ——旋光管的长度，单位为分米（dm）；

$\rho$ ——溶液中D-泛酸钠的质量浓度，单位为克每毫升 $(\mathrm{g / mL})$ 。

称取 $2\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样，加水溶解并定容至 $10\mathrm{mL}$ 。按GB/T23769规定的方法测定 $\mathrm{pH}$ 。

# A.8 干燥减量的测定

# A.8.1 试剂和材料

五氧化二磷。

# A.8.2 仪器和设备

真空干燥箱。

# A.8.3 分析步骤

称取约 $2\mathrm{g}$ （精确至 $0.1\mathrm{mg}$ ）试样于已恒重的称量瓶中，置减压干燥箱内，用五氧化二磷做干燥剂，常温减压干燥 $24\mathrm{h}$ 。取出称量瓶，称量。重复以上操作至恒重。

# A.8.4 结果计算

干燥减量质量分数 $\varpi_{1}$ ，按式(A.2)计算：

$$
w _ {1} = \frac {m _ {1} - m _ {2}}{m} \times 100 \% \tag{A.2}
$$

式中：

$m_{1}$ ——干燥前称量瓶和试样质量，单位为克(g)；

$m_{2}$ ——干燥后称量瓶和试样质量，单位为克 $(\mathrm{g})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

以两次平行测定结果的算术平均值为测定结果。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

取 $0.02\mathrm{g}\beta$ -丙氨酸溶解于 $50~\mathrm{mL}$ 水中作对照品溶液。称取 $0.2\mathrm{g}$ 试样，溶于 $5\mathrm{mL}$ 水中作试样液。吸取上述溶液各 $5\mu \mathrm{L}$ ，分别点于同一硅胶G薄层板上，以展开剂(A.11.1.5)展开，晾干，喷茚三酮试液(A.11.1.6)，在 $100^{\circ}\mathrm{C}$ 干燥 $10\mathrm{min}$ ，立即检视，样品溶液如显示与对照品溶液比移值 $(R_{f})$ 相同的斑点，其颜色不得深于对照品，即是 $\beta$ -丙氨酸含量小于 $1\%$ 。

# A. 12 铅(Pb)的测定

称取试样 $10\mathrm{g}$ （精确至 $1\mathrm{mg}$ ）于烧杯中，加水溶解，转移至 $50~\mathrm{mL}$ 容量瓶中，定容，摇匀，作为试样溶液。以下操作按GB5009.12第三法火焰原子吸收光谱法进行。

# A.13 砷（As）的测定

称取试样 $0.5\mathrm{g}$ (精确至 $1\mathrm{mg}$ ) 于烧杯中，加 $10\mathrm{mL}$ 水溶解，转移至 $50\mathrm{mL}$ 容量瓶中，再加入 $2.5\mathrm{mL}$ 盐酸， $5.0\mathrm{mL}$ 硫脲溶液 $(50\mathrm{g/L})$ ，用水定容至刻度，摇匀，作为试样溶液。以下操作同GB5009.76第二法氢化物原子荧光光度法进行。

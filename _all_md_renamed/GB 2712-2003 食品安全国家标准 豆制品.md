# 食品安全国家标准

# 豆制品

2014-12-24 发布

2015-05-24 实施

# 前言

本标准代替了GB2712—2003《发酵性豆制品卫生标准》，部分代替了GB2711—2003《非发酵性豆制品及面筋卫生标准》。

本标准与GB2711—2003及GB2712—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准豆制品”；  
——修改了范围；  
——增加了术语和定义；  
——修改了感官要求；  
——修改了理化指标；  
——修改了微生物限量。

# 食品安全国家标准

# 豆制品

# 1 范围

本标准适用于预包装豆制品。

本标准不适用于大豆蛋白粉。

# 2 术语和定义

# 2.1 豆制品

以大豆或杂豆为主要原料，经加工制成的食品，包括发酵豆制品、非发酵豆制品和大豆蛋白类制品。

# 3 技术要求

# 3.1 原料要求

原料应符合相应的食品标准和有关规定。

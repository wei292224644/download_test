# 前言

本标准代替GB/T5009.157—2003《食品中有机酸的测定》。

本标准与GB/T5009.157—2003《食品中有机酸的测定》相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中有机酸的测定”；

——将适用范围扩充至果冻、固体饮料以及水果罐头等食品；

——增加了乳酸、富马酸和己二酸等被测物。

# 食品安全国家标准

# 食品中有机酸的测定

# 1 范围

本标准规定了食品中酒石酸、乳酸、苹果酸、柠檬酸、丁二酸、富马酸和己二酸的测定方法。

本标准适用于果汁及果汁饮料、碳酸饮料、固体饮料、胶基糖果、饼干、糕点、果冻、水果罐头、生湿面制品和烘焙食品馅料中7种有机酸的测定。

# 2 原理

试样直接用水稀释或用水提取后，经强阴离子交换固相萃取柱净化，经反相色谱柱分离，以保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.3 标准品

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 无水乙醇 $\mathrm{(CH_3CH_2OH)}$ ：色谱纯。  
3.1.3 磷酸 $\mathrm{(H_3PO_4)}$   
3.2.1 磷酸溶液 $(0.1\%)$ ：量取磷酸 $0.1\mathrm{mL}$ ，加水至 $100\mathrm{mL}$ ，混匀。  
3.2.2 磷酸-甲醇溶液 $(2\%)$ ：量取磷酸 $2\mathrm{mL}$ ，加甲醇至 $100~\mathrm{mL}$ ，混匀。  
3.3.1 乳酸标准品 $\left(\mathrm{C}_{3} \mathrm{H}_{6} \mathrm{O}_{3}\right)$ ，纯度 $\geqslant 99\%$   
3.3.2 酒石酸标准品 $\left(\mathrm{C}_{4} \mathrm{H}_{6} \mathrm{O}_{6}\right)$ ，纯度 $\geqslant 99\%$ 。  
3.3.3 苹果酸标准品 $\left(\mathrm{C}_{4} \mathrm{H}_{6} \mathrm{O}_{5}\right)$ ，纯度 $\geq 99 \%$   
3.3.4 柠檬酸标准品 $\left(\mathrm{C}_{6} \mathrm{H}_{8} \mathrm{O}_{7}\right)$ ，纯度 $\geqslant 98\%$ 。  
3.3.5 丁二酸标准品 $\mathrm{(C_4H_6O_4)}$ ，纯度 $\geqslant 99\%$   
3.3.6 富马酸标准品 $\mathrm{(C_4H_4O_4)}$ ，纯度 $\geq 99\%$   
3.3.7 己二酸标准品 $\left(\mathrm{C}_{6} \mathrm{H}_{10} \mathrm{O}_{4}\right)$ ，纯度 $\geq 99\%$ 。

# 3.4 标准溶液配制

3.4.1 酒石酸、苹果酸、乳酸、柠檬酸、丁二酸和富马酸混合标准储备溶液：分别称取酒石酸 $1.25\mathrm{g}$ 、苹果酸 $2.5\mathrm{g}$ 、乳酸 $2.5\mathrm{g}$ 、柠檬酸 $2.5\mathrm{g}$ 、丁二酸 $6.25\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ) 和富马酸 $2.5\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ ）于

$50~\mathrm{mL}$ 小烧杯中，加水溶解，用水转移到 $50~\mathrm{mL}$ 容量瓶中，定容，混匀，于 $4^{\circ}\mathrm{C}$ 保存，其中酒石酸质量浓度为 $25000\mu \mathrm{g / mL}$ 、苹果酸 $50000\mu \mathrm{g / mL}$ 、乳酸 $50000\mu \mathrm{g / mL}$ 、柠檬酸 $50000\mu \mathrm{g / mL}$ 、丁二酸 $125000\mu \mathrm{g / mL}$ 和富马酸 $50\mu \mathrm{g / mL}$ 。

3.4.2 酒石酸、苹果酸、乳酸、柠檬酸、丁二酸、富马酸混合标准曲线工作液：分别吸取混合标准储备溶液(3.4.1) $0.50~\mathrm{mL}, 1.00~\mathrm{mL}, 2.00~\mathrm{mL}, 5.00~\mathrm{mL}, 10.00~\mathrm{mL}$ 于 $25~\mathrm{mL}$ 容量瓶中，用磷酸溶液(3.2.1)定容至刻度，混匀，于 $4^{\circ}\mathrm{C}$ 保存。  
3.4.3 己二酸标准储备溶液 $(500\mu \mathrm{g / mL})$ ：准确称取按其纯度折算为 $100\%$ 质量的己二酸 $12.5\mathrm{mg}$ ，置 $25~\mathrm{mL}$ 容量瓶中，加水到刻度，混匀，于 $4^{\circ}C$ 保存。  
3.4.4 己二酸标准曲线工作液：分别吸取标准储备溶液(3.4.3) $0.50 \mathrm{~mL}, 1.00 \mathrm{~mL}, 2.00 \mathrm{~mL}, 5.00 \mathrm{~mL}, 10.00 \mathrm{~mL}$ 于 $25 \mathrm{~mL}$ 容量瓶中，用磷酸溶液(3.2.1)定容至刻度，混匀，于 $4^{\circ} \mathrm{C}$ 保存。

# 3.5 材料

强阴离子固相萃取柱(SAX)： $1000\mathrm{mg},6\mathrm{mL}$ 。使用前依次用 $5\mathrm{mL}$ 甲醇、 $5\mathrm{mL}$ 水活化。

# 4 仪器和设备

4.1 高效液相色谱仪，带二极管阵列检测器或紫外检测器。  
4.2 天平：感量为 $0.01\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。  
4.3 高速均质器。  
4.4 高速粉碎机。  
4.5 固相萃取装置。  
4.6 水相型微孔滤膜：孔径 $0.45\mu \mathrm{m}$

# 5 分析步骤

警告：实验人员在使用液氮时，应佩戴手套等防护工具，防止意外洒溅，造成冻伤。

# 5.1 试样制备及保存

# 5.1.1 液体样品

将果汁及果汁饮料、果味碳酸饮料等样品摇匀分装，密闭常温或冷藏保存。

# 5.1.2 半固态样品

对果冻、水果罐头等样品取可食部分匀浆后，搅拌均匀，分装，密闭冷藏或冷冻保存。

# 5.1.3 固体样品

饼干、糕点和生湿面制品等低含水量样品，经高速粉碎机粉碎、分装，于室温下避光密闭保存；对于固体饮料等呈均匀状的粉状样品，可直接分装，于室温下避光密闭保存。

# 5.1.4 特殊样品

对于胶基糖果类黏度较大的特殊样品，现将样品用剪刀铰成约 $2\mathrm{mm}\times 2\mathrm{mm}$ 大小的碎块放入陶瓷研钵中，再缓慢倒入液氮，样品迅速冷冻后采用研磨的方式获取均匀的样品，分装后密闭冷冻保存。

# 5.2 试样处理

# 5.2.1 果汁饮料及果汁、果味碳酸饮料

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样(若试样中含二氧化碳应先加热除去)，放入 $25\mathrm{mL}$ 容量瓶中，加水至刻度，经 $0.45\mu \mathrm{m}$ 水相滤膜过滤，注入高效液相色谱仪分析。

# 5.2.2 果冻、水果罐头

称取 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )均匀试样，放入 $50~\mathrm{mL}$ 塑料离心管中，向其中加入 $20~\mathrm{mL}$ 水后在 $15000\mathrm{r / min}$ 的转速下均质提取 $2\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上层提取液至 $50~\mathrm{mL}$ 容量瓶中，残留物再用 $20~\mathrm{mL}$ 水重复提取一次，合并提取液于同一容量瓶中，并用水定容至刻度，经 $0.45\mu \mathrm{m}$ 水相滤膜过滤，注入高效液相色谱仪分析。

# 5.2.3 胶基糖果

称取 $1\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )均匀试样，放入 $50~\mathrm{mL}$ 具塞塑料离心管中，加入 $20~\mathrm{mL}$ 水后在旋混仪上振荡提取 $5\mathrm{min}$ ，在 $4000\mathrm{r / min}$ 下离心 $3\mathrm{min}$ 后，将上清液转移至 $100~\mathrm{mL}$ 容量瓶中，向残渣加入 $20~\mathrm{mL}$ 水重复提取1次，合并提取液于同一容量瓶中，用无水乙醇(3.1.2)定容，摇匀。

准确移取上清液 $10\mathrm{mL}$ 于 $100~\mathrm{mL}$ 鸡心瓶中，向鸡心瓶中加入 $10~\mathrm{mL}$ 无水乙醇(3.1.2)，在 $80^{\circ}\mathrm{C} \pm$ $2^{\circ}\mathrm{C}$ 下旋转浓缩至近干时，再加入 $5\mathrm{mL}$ 无水乙醇(3.1.2)继续浓缩至彻底干燥后，用 $1\mathrm{mL} \times 1\mathrm{mL}$ 水洗涤鸡心瓶2次。将待净化液全部转移至经过预活化的SAX固相萃取柱中，控制流速在 $1\mathrm{mL} / \mathrm{min} \sim$ $2\mathrm{mL} / \mathrm{min}$ ，弃去流出液。用 $5\mathrm{mL}$ 水淋洗净化柱，再用 $5\mathrm{mL}$ 磷酸-甲醇溶液(3.2.2)洗脱，控制流速在 $1\mathrm{mL} / \mathrm{min} \sim 2\mathrm{mL} / \mathrm{min}$ ，收集洗脱液于 $50~\mathrm{mL}$ 鸡心瓶中，洗脱液在 $45^{\circ}\mathrm{C}$ 下旋转蒸发近干后，再加入 $5\mathrm{mL}$ 无水乙醇(3.1.2)继续浓缩至彻底干燥后，用 $1.0\mathrm{mL}$ 磷酸溶液(3.2.1)振荡溶解残渣后过 $0.45\mu \mathrm{m}$ 滤膜后，注入高效液相色谱仪分析。

# 5.2.4 固体饮料

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样，放入 $50~\mathrm{mL}$ 烧杯中，加入 $40~\mathrm{mL}$ 水溶解并转移至 $100~\mathrm{mL}$ 容量瓶中，用无水乙醇(3.1.2)定容至刻度，摇匀，静置 $10\mathrm{min}$ 。

准确移取上清液 $20~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 鸡心瓶中，向鸡心瓶中加入 $10~\mathrm{mL}$ 无水乙醇(3.1.2)，在 $80^{\circ}\mathrm{C} \pm$ $2^{\circ}\mathrm{C}$ 下旋转浓缩至近干时，再加入 $5~\mathrm{mL}$ 无水乙醇(3.1.2)继续浓缩至彻底干燥后，用 $1~\mathrm{mL} \times 1~\mathrm{mL}$ 水洗涤鸡心瓶2次。将待净化液全部转移至经过预活化的SAX固相萃取柱中，控制流速在 $1~\mathrm{mL} / \mathrm{min} \sim$ $2~\mathrm{mL} / \mathrm{min}$ ，弃去流出液。用 $5~\mathrm{mL}$ 水淋洗净化柱，再用 $5~\mathrm{mL}$ 磷酸-甲醇溶液(3.2.2)洗脱，控制流速在 $1~\mathrm{mL} / \mathrm{min} \sim 2~\mathrm{mL} / \mathrm{min}$ ，收集洗脱液于 $50~\mathrm{mL}$ 鸡心瓶中，洗脱液在 $45^{\circ}\mathrm{C}$ 下旋转蒸发近干后，再加入 $5~\mathrm{mL}$ 无水乙醇(3.1.2)继续浓缩至彻底干燥后，用 $1.0~\mathrm{mL}$ 磷酸溶液(3.2.1)振荡溶解残渣后过 $0.45~\mu \mathrm{m}$ 滤膜后，注入高效液相色谱仪分析。

# 5.2.5 面包、饼干、糕点、烘焙食品馅料和生湿面制品

称取 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )均匀试样，放入 $50~\mathrm{mL}$ 塑料离心管中，向其中加入 $20~\mathrm{mL}$ 水后在 $15000\mathrm{r / min}$ 均质提取 $2\mathrm{min}$ ，在 $4000\mathrm{r / min}$ 下离心 $3\mathrm{min}$ 后，将上清液转移至 $100~\mathrm{mL}$ 容量瓶中，向残渣加入 $20~\mathrm{mL}$ 水重复提取1次，合并提取液于同一容量瓶中，用无水乙醇(3.1.2)定容，摇匀。

准确移取上清液 $10\mathrm{mL}$ 于 $100~\mathrm{mL}$ 鸡心瓶中，向鸡心瓶中加入 $10~\mathrm{mL}$ 无水乙醇(3.1.2)，在 $80^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 下旋转浓缩至近干时，再加入 $5\mathrm{mL}$ 无水乙醇(3.1.2)继续浓缩至彻底干燥后，用 $1\mathrm{mL} \times 1\mathrm{mL}$ 水洗涤鸡心瓶2次。将待净化液全部转移至经过预活化的SAX固相萃取柱中，控制流速在 $1\mathrm{mL} / \mathrm{min} \sim$

$2\mathrm{mL} / \mathrm{min}$ ，弃去流出液。用 $5\mathrm{mL}$ 水淋洗净化柱，再用 $5\mathrm{mL}$ 磷酸-甲醇溶液(3.2.2)洗脱，控制流速在 $1\mathrm{mL} / \mathrm{min}\sim 2\mathrm{mL} / \mathrm{min}$ ，收集洗脱液于 $50~\mathrm{mL}$ 鸡心瓶中，洗脱液在 $45^{\circ}C$ 下旋转蒸发近干后，用 $5.0~\mathrm{mL}$ 磷酸溶液(3.2.1)振荡溶解残渣后过 $0.45\mu \mathrm{m}$ 滤膜后，注入高效液相色谱仪分析。

# 5.3 仪器参考条件

# 5.3.1 酒石酸、苹果酸、乳酸、柠檬酸、丁二酸和富马酸的测定

5.3.1.1 色谱柱：CAPECELL PAK MG S5 C<sub>18</sub>柱，4.6 mm×250 mm, 5 μm，或同等性能的色谱柱。  
5.3.1.2 流动相：用 $0.1\%$ 磷酸溶液-甲醇 $= 97.5 + 2.5$ （体积比)比例的流动相等度洗脱 $10\mathrm{min}$ ，然后用较短的时间梯度让甲醇相达到 $100\%$ 并平衡 $5\mathrm{min}$ ，再将流动相调整为 $0.1\%$ 磷酸溶液-甲醇 $= 97.5 + 2.5$ （体积比)的比例，平衡 $5\mathrm{min}$ 。  
5.3.1.3 柱温： $40^{\circ}C$ 。  
5.3.1.4 进样量： $20~\mu \mathrm{L}$   
5.3.1.5 检测波长： $210~\mathrm{nm}$

# 5.3.2 己二酸的测定

5.3.2.1 色谱柱：CAPECELL PAK MG S5 C<sub>18</sub>柱，4.6 mm×250 mm, 5 μm，或同等性能的色谱柱。  
5.3.2.2 流动相： $0.1\%$ 磷酸溶液-甲醇 $= 75 + 25$ （体积比）等度洗脱 $10\mathrm{min}$ 。  
5.3.2.3 柱温： $40^{\circ}C$ 。  
5.3.2.4 进样量： $20~\mu \mathrm{L}$   
5.3.2.5 检测波长： $210~\mathrm{nm}$

# 5.4 标准曲线的制作

将标准系列工作液分别注入高效液相色谱仪中，测定相应的峰高或峰面积。以标准工作液的浓度为横坐标，以色谱峰高或峰面积为纵坐标，绘制标准曲线。（标准图谱见附录A）

# 5.5 试样溶液的测定

将试样溶液注入高效液相色谱仪中，得到峰高或峰面积，根据标准曲线得到待测液中有机酸的浓度。

# 6 分析结果的表述

试样中有机酸的含量按式(1)计算：

$$
X = \frac {C \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中有机酸的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

C 由标准曲线求得试样溶液中某有机酸的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$

$V$ ——样品溶液定容体积，单位为 $(\mathrm{mL})$ ；

$m$ ——最终样液代表的试样质量，单位为克(g)；

1000——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

检出限与定量限均为：

a）果汁、果汁饮料、果冻和水果罐头：酒石酸 $250\mathrm{mg / kg}$ 、苹果酸 $500\mathrm{mg / kg}$ 、乳酸 $250\mathrm{mg / kg}$ 、柠檬酸 $250\mathrm{mg / kg}$ 、丁二酸 $1250\mathrm{mg / kg}$ 、富马酸 $1.25\mathrm{mg / kg}$ 、己二酸 $25\mathrm{mg / kg}$   
b）胶基糖果、面包、糕点、饼干和烘焙食品馅料：酒石酸 $500\mathrm{mg / kg}$ 、苹果酸 $1000\mathrm{mg / kg}$ 、乳酸 $500\mathrm{mg / kg}$ 、柠檬酸 $500\mathrm{mg / kg}$ 、丁二酸 $2500\mathrm{mg / kg}$ 、富马酸 $2.5\mathrm{mg / kg}$ 、己二酸 $50\mathrm{mg / kg}$   
c）固体饮料：酒石酸 $50\mathrm{mg / kg}$ 、苹果酸 $100\mathrm{mg / kg}$ 、乳酸 $50\mathrm{mg / kg}$ 、柠檬酸 $50\mathrm{mg / kg}$ 、丁二酸 $250\mathrm{mg / kg}$ 、富马酸 $0.25\mathrm{mg / kg}$ 、己二酸 $5\mathrm{mg / kg}$ 。

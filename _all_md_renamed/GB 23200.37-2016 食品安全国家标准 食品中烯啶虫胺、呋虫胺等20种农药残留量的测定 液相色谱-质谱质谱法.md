# 前言

本标准代替SN/T2323—2009《进出口食品中吡虫胺、呋虫胺等20种农药残留量检测方法液相色谱一质谱/质谱法》。与SN/T2323—2009相比，主要变化如下：

——标准文本格式修改为食品安全国家标准文本格式；

——标准名称中“进出口食品”改为“食品”；

——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T2323—2009。

# 食品安全国家标准

# 食品中烯啶虫胺、呋虫胺等20种农药残留量的测定液相色谱一质谱/质谱法

# 1 范围

本标准规定了食品中烯啶虫胺、呋虫胺、螺环菌胺、丁苯吗啉、杀螨隆一甲脒、十三吗啉、叶菌唑、杀螨隆一脲、密灭汀、泰妙菌素、多杀霉素A、多杀霉素D、氨基阿维菌素、甲氨基阿维菌素1、甲氨基阿维菌素2、烯丙酰草胺、驱虫磷、烯唑醇、阿维菌素和甲基甲酸胺阿维菌素农药残留量液相色谱一质谱/质谱测定方法和制样方法。

本标准适用于进出口大米、糙米、玉米、大麦和小麦中以上20种农药残留量的检测和确证，其他食品可参照执行。

# 3 原理

试样用乙腈水溶液提取，先用正己烷液液分配，再用石墨碳和N-丙基乙二胺固相萃取柱(PSA)净化，用液相色谱一质谱/质谱仪检测和确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 正己烷 $(\mathrm{C_6H_{14}})$ 。

4.1.2 甲苯 $(\mathrm{C}_7\mathrm{H}_8)$ 。

4.1.3 甲醇 $(\mathrm{CH}_4\mathrm{O})$ 。

4.1.4 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ 。

4.1.5 甲酸 $(\mathrm{CH}_2\mathrm{O}_2)$ 。

4.1.6 氯化钠(NaCl)：分析纯。

4.1.7 七水硫酸镁 $(\mathrm{MgSO_4}\cdot 7\mathrm{H_2O})$ ：分析纯。

# 4.2 溶液配制

乙腈一甲苯溶液 $(3 + 1$ ，体积比)：取 $300~\mathrm{mL}$ 乙腈，加入 $100~\mathrm{mL}$ 甲苯，摇匀备用。

# 4.3 标准品

农药标准品：烯啶虫胺(Nitenpyram，CAS号：150824-47-8)，呋虫胺(Dinotefuran，CAS号：165252-70-0)，螺环菌胺(Spiroxamine，CAS号：118134-30-8)，丁苯吗啉(Fenpropimorph，CAS号：67564-91-4)，杀螨隆一甲脒(Diafenthiuron-methanimidamide，CAS号：136337-81-9)，十三吗啉

(Tridemorph, CAS号: 81412 - 43 - 3), 叶菌唑 (Metconazole, CAS号: 125116 - 23 - 6), 杀螨隆一脉 (Diafenthioron-urea, CAS号: 136337 - 67 - 2), 多杀霉素 A(Spinosyn A, CAS号: 131929 - 60 - 7), 多杀霉素 D(Spinosyn D, CAS号: 131929 - 63 - 0), 烯丙酰草胺(Dichlormid, CAS号: 37764 - 25 - 3), 驱虫磷 (Naphthalophos, CAS号: 1491 - 41 - 4), 烯唑醇(Diniconazole, CAS号: 83657 - 24 - 3)纯度均 $\geqslant 97\%$ ; 密灭汀(Milbemectin, CAS号: 51596 - 10 - 2), 泰妙菌素(Tiamulin-fumarate, CAS号: 55297 - 96 - 6), 氨基阿维菌素(Emamectin-amino), 甲氨基阿维菌素1(Emamectin-methyl amino 1), 甲氨基阿维菌素2 (Emamectin-methyl amino 2), 阿维菌素(Abamectin, CAS号: 71751 - 41 - 2), 甲基甲酸胺阿维菌素 (Emamectin-N-methyl formyl amino)纯度均 $\geqslant 80\%$ 。

# 4.4 标准溶液配制

4.4.1 标准储备溶液：分别准确称取适量的农药标准品，用丙酮配成浓度为 $100\mu \mathrm{g / mL}$ 的标准储备液。标准储备液在 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 条件下储存，每12个月配制一次。  
4.4.2 混合标准储备溶液: 分别吸取适量的密灭汀、泰妙菌素、多杀霉素 A、多杀霉素 D、氨基阿维菌素、甲氨基阿维菌素 1、甲氨基阿维菌素 2、烯丙酰草胺、驱虫磷、烯唑醇、阿维菌素和甲基甲酸胺阿维菌素 12 种农药的标准储备液配成混合标准储备溶液, 为 A 组农药, 混合标准储备溶液的浓度为 $10 \mu \mathrm{g} / \mathrm{mL}$ ; 分别吸取适量的烯啶虫胺、呋虫胺、螺环菌胺、丁苯吗啉、杀螨隆一甲脒、十三吗啉、叶菌唑和杀螨隆一脲的 8 种混合标准储备溶液配成混合标准储备溶液, 为 B 组农药, 混合标准储备溶液的浓度也为 $10 \mu \mathrm{g} / \mathrm{mL}$ 。标准储备溶液在 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 条件下储存, 每 6 个月配制一次。  
4.4.3 混合标准工作溶液: 将 A 组和 B 组农药的混合标准储备溶液分别用乙腈稀释成适当浓度的标准工作液。在 ${0}^{ \circ  }\mathrm{C} \sim  {4}^{ \circ  }\mathrm{C}$ 条件下储存。每 3 个月配制一次。

# 5 仪器与设备

5.1 高效液相色谱-质谱/质谱仪：配有电喷雾离子源(ESI)。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 旋转蒸发仪。  
5.4 振荡器。  
5.5 离心机： $5000\mathrm{r / min}$   
5.6固相萃取柱：石墨碳/N-丙基乙二胺(PSA)， $6\mathrm{mL}$ ， $500~\mathrm{mg} / 500~\mathrm{mg}$ ，或相当者。  
5.7 具塞磨口三角烧瓶： $300~\mathrm{mL}$   
5.8 离心管： $50~\mathrm{mL}$ ，塑料。  
5.9 滤膜： $0.45\mu \mathrm{m}$ ，有机相。

# 6 试样制备与保存

取大米、糙米、玉米、大麦和小麦等有代表性样品约 $500\mathrm{g}$ ，取样部位按GB2763的规定执行，用粉碎机粉碎，混匀，装入洁净的容器内，密闭并标明标记。 $4^{\circ}\mathrm{C}$ 以下保存。

# 7 分析步骤

# 7.1 提取

称取试样 $20\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）于 $300\mathrm{mL}$ 具塞磨口三角烧瓶中，加入 $30\mathrm{mL}$ 水，放置 $30\mathrm{min}$ 。加入 $80\mathrm{mL}$ 乙腈，振荡 $30\mathrm{min}$ ，取上清液 $30\mathrm{mL}$ 置于 $50\mathrm{mL}$ 塑料离心管中，加 $3\mathrm{g}$ 硫酸镁、 $2\mathrm{g}$ 氯化钠和 $10\mathrm{mL}$ 正己烷，振荡 $5\mathrm{min}$ ，以 $3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，准确移取中间层（乙腈层） $10\mathrm{mL}$ ，供净化。

# 7.2 净化

石墨碳/乙二胺基丙基固相萃取柱用 $10\mathrm{mL}$ 乙腈甲苯溶液预淋洗，将7.1中移取的 $10\mathrm{mL}$ 乙腈溶液上样至固相萃取柱(SPE)上，收集流出液。用 $30\mathrm{mL}$ 乙腈甲苯溶液洗脱，收集洗脱液，合并收集溶液，于 $40^{\circ}\mathrm{C}$ 水浴减压浓缩至干，用 $1\mathrm{mL}$ 甲醇溶解残渣，过膜，供液相色谱一质谱/质谱仪测定和确证。

# 7.3.3 色谱测定与确证

# 7.4 空白试验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中各种农药的含量，计算结果需将空白值扣除。

$$
X = \frac {A \times c \times V}{A _ {s} \times m} \tag {1}
$$

式中：

$X$ ——试样中农药的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c$ ——标准工作液中农药的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A$ ——样品溶液中农药的峰面积；

$A_{s}$ ——标准工作溶液中农药的峰面积；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——最终样液所代表的试样量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 食品安全国家标准

# 婴幼儿食品和乳品中脂肪酸的测定

# 1 范围

本标准规定了婴幼儿食品和乳品中脂肪酸的测定方法。

本标准适用于婴幼儿食品和乳品中脂肪酸的测定，第二法不适用于含有被包埋脂肪酸的测定。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

# 第一法 乙酰氯一甲醇甲酯化法

# 3 原理

乙酰氯与甲醇反应得到的盐酸—甲醇使试样中的脂肪和游离脂肪酸甲酯化，用甲苯提取后，经气相色谱仪分离检测，外标法定量。

# 4 试剂和材料

除非另有说明，本方法所用试剂均为分析纯或以上规格，水为GB/T6682规定的一级水。

# 4.1 无水碳酸钠。

# 4.2 甲苯：色谱纯。

# 4.3 乙酰氯。

4.4 乙酰氯甲醇溶液（体积分数为 $10\%$ ）：量取 $40 \mathrm{~mL}$ 甲醇于 $100 \mathrm{~mL}$ 干燥的烧杯中，准确吸取 $5.0 \mathrm{~mL}$ 乙酰氯（4.3）逐滴缓慢加入，不断搅拌，冷却后转移并定容至 $50 \mathrm{~mL}$ 干燥的容量瓶中。临用前配制。

注：乙酰氯为刺激性试剂,配制乙酰氯甲醇溶液时应不断搅拌防止喷溅,注意防护。

4.5 碳酸钠溶液：准确称取 $6 \mathrm{~g}$ 无水碳酸钠（4.1）于 $100 \mathrm{~mL}$ 烧杯中，加水溶解，转移并用水定容至 $100 \mathrm{~mL}$ 容量瓶中。  
4.6 脂肪酸甘油三酯标准品：纯度 $\geq 99\%$ ，脂肪酸种类参见附录A中的表A.1。  
4.7 脂肪酸甘油三酯标准工作液：按试样中各脂肪酸含量及所要分析脂肪酸的种类配制适当浓度的标准工作液，用甲苯定容并分别保存于 $-10^{\circ}\mathrm{C}$ 以下的冰箱中，有效期三个月。

# 5 仪器和设备

5.1 天平：感量为 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$   
5.2 恒温热水浴槽。  
5.3 离心机：转速 $\geqslant 5000$ 转/分钟。   
5.4 气相色谱仪，带FID检测器。  
5.5 冷冻干燥仪。   
5.6 氮吹仪。  
5.7 螺口玻璃管（带有聚四氟乙烯做内垫的螺口盖）： $15\mathrm{mL}$ 。  
5.8 离心管： $50\mathrm{mL}$

# 6 分析步骤

# 6.1 试样处理

6.1.1 试样含水量大于 $5\%$ 时，冷冻干燥至含水量小于 $5\%$   
6.1.2 称取试样 $0.5\mathrm{g}$ （精确到 $0.1\mathrm{mg}$ ）于 $15\mathrm{mL}$ 干燥螺口玻璃管（5.7）中，加入 $5.0\mathrm{mL}$ 甲苯（4.2）。  
6.1.3 称取无水奶油试样 $0.2 \mathrm{~g}$ (精确到 $0.1 \mathrm{mg}$ ) 于 $15 \mathrm{~mL}$ 干燥螺口玻璃管（5.7）中，加入 $5.0 \mathrm{~mL}$ 甲苯（4.2）。

# 6.2 甲酯化提取

# 6.2.1 试样测定液的制备

在试样（6.1.2或6.1.3）中加入 $10\%$ 乙酰氯甲醇溶液 $6.0 \mathrm{~mL}$ （4.4），充氮气后，旋紧螺旋盖，振荡混合后于 $80^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 水浴中放置 $2 \mathrm{~h}$ ，期间每隔 $20 \mathrm{~min}$ 取出振摇一次，水浴后取出冷却至室温。将反应后的样液转移至 $50 \mathrm{~mL}$ 离心管中，分别用 $3.0 \mathrm{~mL}$ 碳酸钠溶液（4.5）清洗玻璃管三次，合并碳酸钠溶液（4.5）于 $50 \mathrm{~mL}$ 离心管（5.8）中，混匀，5000 转/分钟离心约 $5 \mathrm{~min}$ 。取上清液作为试液，气相色谱仪测定。

# 6.2.2 标准测定液的制备

准确吸取脂肪酸甘油三酯标准工作液（4.7） $0.5\mathrm{mL}$ 于 $15\mathrm{mL}$ 螺口玻璃管（5.7）中，加入 $4.5\mathrm{mL}$ 甲苯，其他操作步骤同6.2.1。

# 6.3 色谱参考条件

色谱柱：固定液 $100\%$ 二氰丙基聚硅氧烷， $100 \mathrm{~m} \times 0.25 \mathrm{~mm}$ ， $0.20 \mu \mathrm{m}$ ，或性能相当的色谱柱。

载气：氮气。

载气流速： $1.0\mathrm{mL / min}$

进样口温度： $260^{\circ}C$ 。

分流比：30：1。

检测器温度： $280^{\circ}C$ 。

柱温箱温度：初始温度 $140^{\circ} \mathrm{C}$ ，保持 $5 \mathrm{~min}$ ，以 $4^{\circ} \mathrm{C} / \mathrm{min}$ 升温至 $240^{\circ} \mathrm{C}$ ，保持 $15 \mathrm{~min}$ 。

进样量： $1.0~\mu \mathrm{L}$

# 6.4 试样溶液的测定

分别准确吸取 $1.0 \mu \mathrm{L}$ 脂肪酸标准测定液（6.2.2）及试样测定液（6.2.1）注入色谱仪，平行测定次数不少于两次，以色谱峰峰面积定量。

# 7 分析结果的表述

# 7.1 试样中各脂肪酸的含量的计算

试样中各脂肪酸的含量按公式（1）计算：

$$
X _ {i} = \frac {A _ {\mathrm {s i}} \times m _ {\mathrm {s t d i}} \times F _ {\mathrm {j}}}{A _ {\mathrm {s t d i}} \times m} \times 1 0 0 \dots \tag {1}
$$

式中：

$X_{\mathrm{i}}$ ——试样中各脂肪酸的含量，单位为毫克每百克 $(\mathrm{mg} / 100 \mathrm{~g})$ ；

$A_{\mathrm{si}}$ ——试样测定液中各脂肪酸的峰面积；

$m_{\mathrm{stdi}}$ ——在标准测定液的制备（6.2.2）中吸取的脂肪酸甘油三酯标准工作液中所含有的标准品的质量，单位为毫克 $(\mathrm{mg})$ ；

$F_{\mathrm{j}}$ ——各脂肪酸甘油三酯转化为脂肪酸的换算系数，参见附录A中的表A.1；

$A_{\mathrm{stdi}}$ ——标准测定液中各脂肪酸的峰面积；

$m$ ——试样的称样质量，单位为克（g）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7.2 试样中总脂肪酸的含量计算

试样中总脂肪酸的含量按公式（2）计算：

$$
X _ {\text {T o t a l F A}} = \sum X _ {\mathrm {i}} \dots \tag {2}
$$

式中：

$X_{\text {Total FA}}$ ——试样中总脂肪酸的含量，单位为毫克每百克 $(\mathrm{mg} / 100 \mathrm{~g})$ ；

$X_{\mathrm{i}}$ ——试样中各脂肪酸的含量，单位为毫克每百克（ $\mathrm{mg} / 100 \mathrm{~g}$ ）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7.3 试样中某个脂肪酸占总脂肪酸的百分比（%）的计算

试样中某个脂肪酸占总脂肪酸的百分比（%）Y按公式（3）计算：

$$
Y = \frac {X _ {\mathrm {i}}}{X _ {\text {T o t a l F A}}} \times 1 0 0 \text {或} Y = \frac {A _ {\mathrm {s i}} \times F _ {\mathrm {j}}}{\sum A _ {\mathrm {s i}} \times F _ {\mathrm {j}}} \times 1 0 0 \dots \tag {3}
$$

# 8 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

本方法最低检出限参见附录A中的表A.1。

# 第二法 氨水一乙醇提取法

# 10 原理

乳与乳制品中的脂肪经皂化处理后生成游离脂肪酸，在三氟化硼催化下进行甲酯化反应，经甲酯化后的脂肪酸通过气相色谱柱分离，以氢火焰离子化检测器检测，外标法定量。

# 11 试剂和材料

除非另有说明，本方法所用试剂均为分析纯或以上规格，水为GB/T6682规定的一级水。

11.1 甲醇：色谱纯。  
11.2 乙醚。  
11.3 石油醚：沸程 $30^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$   
11.4 乙醇（体积分数 $\geqslant 95\%$ ）。  
11.5 氨水（体积分数为 $25\%$ ）。  
11.6 正己烷（ $\mathrm{C_6H_{14}}$ ）：色谱纯。  
11.7 高峰氏淀粉酶（Taka-Diastase）： $128\mathrm{U / mg}$ 。  
11.8 三氟化硼甲醇溶液（质量分数为 $14\%$ ）。  
11.9 饱和氯化钠溶液：溶解 $360\mathrm{g}$ 氯化钠于 $1.0\mathrm{L}$ 水中，搅拌溶解，澄清备用。  
11.10 氢氧化钾甲醇溶液（ $0.5 \mathrm{~mol} / \mathrm{L}$ ）：称取 $2.8 \mathrm{~g}$ 氢氧化钾，用甲醇（11.1）溶解，并稀释定容至 $100 \mathrm{~mL}$ ，混匀。  
11.11 焦性没食子酸甲醇溶液（ $10\%$ ）：将 $1.0 \mathrm{~g}$ 焦性没食子酸溶于 $10 \mathrm{~mL}$ 甲醇中配制成 $10\%$ 焦性没食子酸甲醇溶液备用。  
11.12 脂肪酸甲酯标准物质：纯度 $\geqslant 99\%$ ，贮存于 $-10^{\circ} \mathrm{C}$ 以下的冰箱中，脂肪酸种类参见附录A中的表A.1。  
11.13 脂肪酸甲酯标准工作溶液：按试样中各脂肪酸含量及所要分析脂肪酸的种类适当配制其浓度，正己烷定容并贮存于 $-10^{\circ}\mathrm{C}$ 以下的冰箱中，有效期三个月。

# 12 仪器和设备

12.1 天平：感量为 $0.1\mathrm{mg}$ 。  
12.2 抽脂管： $100\mathrm{mL}$ 磨口具塞试管，抽脂管干燥、恒重。  
12.3 旋转蒸发仪。  
12.4 离心机：转速 $\geqslant 5000$ 转/分钟。   
12.5 恒温水浴锅。  
12.6 气相色谱仪，带FID检测器。

# 13 分析步骤

# 13.1 试样制备

预先将需冷藏的试样从冰箱中取出，放至室温。

# 13.1.1 液态试样

称取 $10\mathrm{g}$ （精确到 $0.1\mathrm{mg}$ ）试样于抽脂管中，待测。

# 13.1.2 固态试样

# 13.1.2.1 含淀粉试样

称取试样 $1.0\mathrm{g}$ （精确到 $0.1\mathrm{mg}$ ）至抽脂管中（12.2），加入 $0.1\mathrm{g}$ 高峰氏淀粉酶（11.7），加入 $10\mathrm{mL}45^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 的水，混合均匀后，用氮气排除瓶中空气，盖上瓶塞，置 $45^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 烘箱内 $30\mathrm{min}$ ，取出。

# 13.1.2.2 不含淀粉试样

称取试样 $1.0\mathrm{g}$ （精确到 $0.1\mathrm{mg}$ ）至抽脂管（12.2）中，加入 $65^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 的水 $10\mathrm{mL}$ 溶解试样，振摇，使样品完全分散。

于上述试样（13.1.1和13.1.2）中加入 $2 \mathrm{~mL}$ 氨水（11.5），于 $65^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 水浴锅中放置 $15 \mathrm{~min}$ ，取出轻摇，冷至室温。

# 13.1.3 无水奶油

称取试样 $0.2\mathrm{g}$ （精确到 $0.1\mathrm{mg}$ ）于磨口烧瓶中，按13.3所述进行皂化酯化。

# 13.2 脂肪提取

在制备好的样品中加入 $10\mathrm{mL}$ 乙醇（11.4），混匀。加入 $25\mathrm{mL}$ 乙醚（11.2），加塞振摇 $1\mathrm{min}$ 。加入 $25\mathrm{mL}$ 石油醚（11.3），加塞振摇 $1\mathrm{min}$ ，静置、分层，有机层转入磨口烧瓶中。再加入 $25\mathrm{mL}$ 乙醚（11.2）及 $25\mathrm{mL}$ 石油醚（11.3），加塞振摇 $1\mathrm{min}$ ，静置、分层，有机层转入磨口烧瓶中，再重复操作一次。合并抽提液于磨口烧瓶中，用旋转蒸发仪浓缩至干。

# 13.3 皂化酯化

在浓缩物（13.2）或无水奶油（13.1.3）中加入 $1.0\mathrm{mL}$ 焦性没食子酸甲醇溶液（11.11）。浓缩干燥之后再加入 $10\mathrm{mL}$ 氢氧化钾甲醇溶液（11.10）置于 $80^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 水浴上回流 $5\mathrm{min} \sim 10\mathrm{min}$ 。再加入 $5\mathrm{mL}$ 三氟化硼甲醇溶液（11.8），继续回流 $15\mathrm{min}$ ，冷却至室温，将烧瓶中的液体移入 $50\mathrm{mL}$ 离心管中，分别用 $3\mathrm{mL}$ 饱和氯化钠溶液（11.9）清洗烧瓶三次，合并饱和氯化钠溶液于 $50\mathrm{mL}$ 离心管，加入 $10\mathrm{mL}$ 正己烷（11.6），振摇后，以5000转/分钟离心 $5\mathrm{min}$ ，取上清液作为试液，供气相色谱仪（12.6）测定。

注：三氟化硼甲醇溶液为强腐蚀性试剂，使用时应注意防护。

# 13.4 色谱参考条件

色谱柱：固定液 $100\%$ 二氰丙基聚硅氧烷， $100 \mathrm{~m} \times 0.25 \mathrm{~mm}$ ， $0.20 \mu \mathrm{m}$ ，或性能相当的色谱柱。

载气：氮气。

载气流速： $1.0\mathrm{mL / min}$

进样口温度： $260^{\circ}C$ 。

分流比：30：1。

检测器温度： $280^{\circ}C$ 。

柱温箱温度：初始温度 $140^{\circ} \mathrm{C}$ ，保持 $5 \mathrm{~min}$ ，以 $4^{\circ} \mathrm{C} / \mathrm{min}$ 升温至 $240^{\circ} \mathrm{C}$ ，保持 $15 \mathrm{~min}$ 。

进样量： $1.0~\mu \mathrm{L}$ 。

# 13.5 测定

分别准确吸取 $1.0 \mu \mathrm{L}$ 脂肪酸甲酯标准工作溶液（11.13）及试液（13.3）注入色谱仪，平行测定次数不少于两次，以色谱峰峰面积定量。

# 14 分析结果的表述

# 14.1 试样中各脂肪酸含量计算

试样中各脂肪酸的含量按公式（4）计算：

$$
X _ {\mathrm {i}} = \frac {A _ {\mathrm {s i}} \times C _ {\mathrm {s t d i}} \times V \times F _ {\mathrm {i}}}{A _ {\mathrm {s t d i}} \times m} \times 1 0 0 \dots \tag {4}
$$

式中：

$X_{\mathrm{i}}$ ——试样中各脂肪酸的含量，单位为毫克每百克（ $\mathrm{mg} / 100 \mathrm{~g}$ ）；

$A_{si}$ ——试样溶液中各脂肪酸甲酯的峰面积；

$C_{\mathrm{stdi}}$ ——脂肪酸甲酯标准工作液中各脂肪酸甲酯的浓度，单位为毫克每毫升（ $\mathrm{mg/mL}$ ）；

$V$ ——13.3中加入正己烷的体积：单位为毫升（ $\mathrm{mL}$ ）；

$A_{\mathrm{stdi}}$ ——混合标准工作液中各脂肪酸甲酯的峰面积；

$F_{\mathrm{i}}$ ——各脂肪酸甲酯转化为脂肪酸的换算系数，参见附录A中的表A.1；

$m$ ——试样的称样量，单位为克（g）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 14.2 试样中总脂肪酸的含量计算

试样中总脂肪酸的含量按公式（5）计算：

$$
X _ {\text {T o t a l F A}} = \sum X _ {i} \dots \tag {5}
$$

式中：

$X_{\text {Total FA}}$ ——试样中总脂肪酸的含量，单位为毫克每百克（ $\mathrm{mg} / 100 \mathrm{~g}$ ）；

$X_{\mathrm{i}}$ ——试样中各脂肪酸的含量，单位为毫克每百克 $(\mathrm{mg} / 100 \mathrm{~g})$ ；

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 14.3 试样中某个脂肪酸占总脂肪酸的百分比（%）的计算

试样中某个脂肪酸占总脂肪酸的百分比（%）Y按公式（6）计算：

$$
Y = \frac {X _ {\mathrm {i}}}{X _ {\mathrm {T o t a l F A}}} \times 1 0 0 \quad \text {或} Y = \frac {A _ {\mathrm {s i}} \times F _ {\mathrm {i}}}{\sum A _ {\mathrm {s i}} \times F _ {\mathrm {i}}} \times 1 0 0 \dots \tag {6}
$$

# 15 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

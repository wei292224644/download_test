# 食品安全国家标准

# 食品添加剂 乙酸钠

# 1 范围

本标准适用于由乙酸和氢氧化钠（或碳酸钠）为原料制得的食品添加剂乙酸钠。

# 2 分子式、结构式和相对分子质量

# 2.1 分子式

$\mathrm{C_2H_3NaO_2}$ $n\mathrm{H}_2\mathrm{O}$ （ $n = 3$ 或0）

# 2.2 结构式

CH3COONa nH2O （n=3或0）

# 2.3 相对分子质量

136.08（结晶品）

82.03（无水品）

# 检验方法

# A. 1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682中规定的三级水。

试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603之规定制备。所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A. 2.1.1 硫酸。

A.2.1.2 乙醇。

A.2.1.3 盐酸溶液： $1 + 1$

A. 2.1.4 乙酸氧铀锌溶液:

称取 $10 \mathrm{~g}$ 乙酸双氧铀, 加 $5 \mathrm{~mL}$ 冰乙酸和 $50 \mathrm{~mL}$ 水, 微热使溶解; 称取 $30 \mathrm{~g}$ 乙酸锌, 加 $3 \mathrm{~mL}$ 冰乙酸和 $30 \mathrm{~mL}$ 水, 微热使溶解; 将两溶液混合, 放冷, 过滤, 即得。

A.2.1.5 氯化铁溶液： $90~\mathrm{g / L}$

# A. 2.2 pH试验

称取 $1 \mathrm{~g}$ 样品, 精确至 $0.01 \mathrm{~g}$ , 用适量的水溶解并稀释至 $100 \mathrm{~mL}$ , 测定溶液的 $\mathrm{pH}$ 应为 $8.0 \sim 9.5$ 。

# A.2.3 钠盐试验

铂丝用盐酸溶液湿润后，蘸取样品，在无色火焰中燃烧，火焰应显亮黄色。

称取 $1 \mathrm{~g}$ 样品, 精确至 $0.01 \mathrm{~g}$ , 用适量的水溶解, 加 $1 \mathrm{~mL}$ 盐酸溶液, 用水稀释至 $20 \mathrm{~mL}$ 。取 $1 \mathrm{~mL}$ 该溶液,加 $5 \mathrm{~mL}$ 乙酸氧铀锌溶液, 振摇, 有黄色沉淀产生。

# A.2.4 乙酸盐试验

样品与硫酸和乙醇共热时，产生乙酸乙酯的特殊香气。样品的中性水溶液加氯化铁溶液（ $90\mathrm{g / L}$ ）产生深红色，加无机酸红色消失。

# A. 3 乙酸钠含量的测定

# A.3.1 试剂和材料

A. 3.1.1 冰乙酸。

A.3.1.2 高氯酸标准滴定溶液： $c(\mathrm{HClO}_4) = 0.1\mathrm{mol / L}$

A.3.1.3 结晶紫指示液： $2\mathrm{g} / \mathrm{L}$

# A.3.2 分析步骤

称取 $0.2 \mathrm{~g}$ 干燥后的样品, 精确至 $0.0001 \mathrm{~g}$ , 加 $40 \mathrm{~mL}$ 冰乙酸溶解, 用高氯酸标准滴定溶液滴定。用电位计指示终点。当用指示剂判断终点时, 加几滴结晶紫指示液, 溶液由紫色变为亮蓝绿色即为终点。

在测定的同时，按与测定相同的步骤，对不加样品而使用相同数量的试剂溶液做空白试验。

# A.3.3 结果计算

乙酸钠 $\left(\mathrm{C}_{2} \mathrm{H}_{3} \mathrm{NaO}_{2}\right)$ 的质量分数 $\mathrm{w}$ , 数值以%表示, 按公式 (A.1) 计算:

$$
w = \frac {\left(v _ {1} - v _ {0}\right) \times c \times M}{m \times 1 0 0 0} \times 1 0 0 \tag {A.1}
$$

式中：

$v_{0}$ ——空白试验消耗高氯酸标准滴定溶液的体积，单位为毫升（ $\mathrm{mL}$ ）；

$v_{1}$ ——样品消耗高氯酸标准滴溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——高氯酸标准滴溶液浓度的准确数值，单位为摩尔每升（ $\mathrm{mol} / \mathrm{L}$ ）；

$M$ ——乙酸钠的摩尔质量的数值，单位为克每摩尔（ $\mathrm{g/mol}$ ）（ $M=82.03$ ）；

$m$ ——样品的质量，单位为克（g）；

1000——换算因子。

# A. 4 酸度和碱度的测定

# A.4.1 试剂和材料

A. 4.1.1 氢氧化钠标准溶液: $c(\mathrm{NaOH}) = 0.1 \mathrm{~mol} / \mathrm{L}$ 。

A.4.1.2 盐酸标准溶液： $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$

A.4.1.3 酚酞指示液： $10\mathrm{g / L}$

# A.4.2 分析步骤

称取结晶品样品约 $2.0 \mathrm{~g}$ , 或无水品样品约 $1.2 \mathrm{~g}$ , 精确至 $0.001 \mathrm{~g}$ 。加 $20 \mathrm{~mL}$ 煮沸后冷却的水溶解, 加2滴酚酞指示液, 保持此样品溶液在约 $10^{\circ} \mathrm{C}$ 的温度下进行下述试验。

A. 4.2.1 如果样品溶液无色, 加 $0.10 \mathrm{~mL}$ 氢氧化钠标准溶液时, 应显粉红色。  
A. 4.2.2 如果样品溶液显粉红色,加 $0.10 \mathrm{~mL}$ 盐酸标准溶液时,粉红色应消失。

# A.5 钾试验

# A.5.1 试剂和材料

酒石酸氢钠溶液， $10\mathrm{g / L}$ ：取酒石酸氢钠（ $\mathrm{NaHC_4H_4O\cdot H_2O)}$ 1 g，溶于 $100~\mathrm{mL}$ 水，此溶液用时现配。

# A.5.2 分析步骤

取适量样品，制成 $10\mathrm{g / L}$ 的样品溶液，在澄清的样品溶液中，缓慢滴加几滴酒石酸氢钠溶液，在 $5\mathrm{min}$ 内不产生浑浊即为通过试验。

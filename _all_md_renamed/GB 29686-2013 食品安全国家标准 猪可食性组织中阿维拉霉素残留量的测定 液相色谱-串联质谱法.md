# 液相色谱-串联质谱法

# 1 范围

本标准规定了猪可食性组织中阿维拉霉素残留量检测的制样和液相色谱-串联质谱测定方法。

本标准适用于猪的肌肉、脂肪/皮、肝脏和肾脏中阿维拉霉素残留量的检测。

# 3原理

试料中残留的阿维拉霉素，用丙酮提取，氢氧化钠溶液水解，再用乙酸乙酯提取，氧化铝固相萃取柱净化，以二氯甲氧苯酸为内标物，液相色谱-串联质谱APCI测定，内标法定量。

# 4 试剂和材料

以下所用的试剂，除特别注明外均为分析纯试剂，水为符合GB/T6682规定的一级水。

4.1 阿维拉霉素、阿维拉霉素残留标示物DIA标准品：含量 $\geq 98\%$   
4.2 内标物：二氯甲氧苯酸标准品：含量 $\geq 98\%$   
4.3 甲醇： 色谱纯。  
4.4 正己烷  
4.5 乙睛：色谱纯。  
4.6 乙酸乙酯  
4.7 丙酮   
4.8 甲酸

# 4.9 氢氧化钠

# 4.10 磷酸

4.11 中性氧化铝固相萃取柱： $1000\mathrm{mg / 6mL}$ ，或相当者。  
4.12 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：取氢氧化钠 $4\mathrm{g}$ ，用水溶解并稀释至 $100\mathrm{mL}$ 。  
4.13 洗脱液：取甲酸 $5\mathrm{mL}$ ，用乙腈溶解并稀释至 $100\mathrm{mL}$ 。  
4.14 $1 \mathrm{mg} / \mathrm{mL}$ 阿维拉霉素和阿维拉霉素残留标示物 DIA 标准贮备液：精密称取阿维拉霉素和阿维拉霉素残留标示物 DIA 各 $10 \mathrm{mg}$ , 分别于 $10 \mathrm{~mL}$ 量瓶中, 用甲醇溶解并稀释至刻度, 配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的阿维拉霉素和阿维拉霉素残留标示物 DIA 标准贮备液。 $2 \sim 4^{\circ} \mathrm{C}$ 保存, 有效期 1 个月。  
4.15 $10\mu \mathrm{g / mL}$ 阿维拉霉素和阿维拉霉素残留标示物DIA标准工作液：精密量取 $1\mathrm{mg / mL}$ 阿维拉霉素和阿维拉霉素残留标示物DIA标准贮备液各 $1.0~\mathrm{mL}$ ，分别于 $100\mathrm{mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $10\mu \mathrm{g / mL}$ 的阿维拉霉素和阿维拉霉素残留标示物DIA标准工作液。2-4℃保存，有效期1个月。  
4.16 $1 \mathrm{mg} / \mathrm{mL}$ 内标物二氯甲氧苯酸标准贮备液：精密称取二氯甲氧苯酸 $10 \mathrm{mg}$ ，于 $10 \mathrm{~mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的二氯甲氧苯酸标准贮备液。 $2 \sim 4^{\circ} \mathrm{C}$ 保存，有效期1个月。  
4.17 $2\mu \mathrm{g / mL}$ 内标物二氯甲氧苯酸标准工作液：精密量取 $1\mathrm{mg / mL}$ 二氯甲氧苯酸标准贮备液 $200\mu \mathrm{L}$ ，于 $100~\mathrm{mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配制成浓度为 $2\mu \mathrm{g / mL}$ 的二氯甲氧苯酸标准工作液。 $2 - 4^{\circ}C$ 保存，有效期1个月。

5.1 高效液相色谱-串联质谱仪：配大气压化学电离源（APCI）。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 冷冻高速离心机  
5.5 振荡器  
5.6 旋涡混合器  
5.7 均质机  
5.8 氮吹仪   
5.9 旋转蒸发仪

# 5.10 固相萃取装置

5.11 圆底烧瓶： $50~\mathrm{mL}$   
5.12 离心管： $15\mathrm{mL}$ ， $50~\mathrm{mL}$

# 6 试料的制备与保存

# 6.1 试料的制备

取新鲜或冷冻空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-20°C以下保存。

# 7 测定步骤

# 7.1 提取

称取肌肉或脂肪试料 $(2 \pm 0.05) \mathrm{~g}$ 、肝脏或肾脏试料 $(1 \pm 0.05) \mathrm{~g}$ , 于 $15 \mathrm{~mL}$ 离心管中, 加丙酮 $4 \mathrm{~mL}$ , 涡旋 $5 \mathrm{~min}$ , 于 $6000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ , 取上清液, 于 $50 \mathrm{~mL}$ 圆底烧瓶中。残渣中加丙酮 $4 \mathrm{~mL}$ , 重复提取两次, 合并 3 次上清液, 于 $60^{\circ} \mathrm{C}$ 旋转蒸发至干, 用 $1 \mathrm{~mol} / \mathrm{L}$ 氢氧化钠溶液 $4 \mathrm{~mL}$ 溶解残余物, 转至 $50 \mathrm{~mL}$ 离心管中, 于 $70^{\circ} \mathrm{C}$ 水浴 $2 \mathrm{~h}$ , 取出, 用 $85 \%$ 磷酸调 $\mathrm{pH}$ 至 1.0, 加乙酸乙酯 $4 \mathrm{~mL}$ , 混匀, 于 $6000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ , 取上清液于另一离心管中, 重复两次, 合并 3 次上清液, 加 $2 \mu \mathrm{g} / \mathrm{mL}$ 二氯甲氧苯酸内标 $0.5 \mathrm{~mL}$ (空白试料除外), 混匀备用。

# 7.2 净化

氧化铝萃取柱用乙酸乙酯 $10\mathrm{mL}$ 活化，取备用液过柱，分别用正己烷 $5\mathrm{mL}$ 、乙酸乙酯 $5\mathrm{mL}$ 和甲醇 $5\mathrm{mL}$ 淋洗，用洗脱液洗脱两次，每次 $7\mathrm{mL}$ ，收集洗脱液，于 $50^{\circ}\mathrm{C}$ 旋转蒸发至干，用甲醇 $1.0\mathrm{mL}$ 溶解残余物，涡旋混匀，供液相色谱-串联质谱法测定。

# 7.3 标准曲线的制备

精密量取 $10\mu \mathrm{g / mL}$ 阿维拉霉素残留标示物DIA标准工作液适量，用甲醇稀释，配制成浓度为5、50、100、200、400和 $800~\mu \mathrm{g / L}$ 的系列标准溶液，供液相色谱-串联质谱测定。以特征离子质量色谱峰面积为纵坐标，标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.4 测定

# 7.4.1 液相色谱条件

色谱柱： $\mathrm{C_{18}}$ （ $150 \mathrm{~mm} \times 4.6 \mathrm{~mm}$ ，粒径 $5 \mu \mathrm{m}$ ），或相当者；

流动相：甲醇+水（50+50，v/v）；

流速： $0.6\mathrm{mL / min}$

进样量： $10~\mu \mathrm{g / L}$

柱温： $40^{\circ}\mathrm{C}$ ；

# 7.4.2 质谱条件

离子源：APCI;

扫描方式：负离子扫描；

检测方式：选择反应监测；

蒸发温度： $400^{\circ}C$

离子传输管温度： $275^{\circ}C$

鞘气压力：25arb;

辅助气压力：5arb;

选择反应监测的优化参数见表1。

# 7.4.3 测定法

# 7.4.3.1 定性测定

通过试样色谱图的保留时间与相应标准品的保留时间、各色谱峰的特征离子与相应浓度标准溶液

各色谱峰的特征离子相对照定性。试样与标准品保留时间的相对偏差不大于 $5\%$ ；试样特征离子的相对丰度与浓度相当混合标准溶液的相对丰度一致，相对丰度偏差不超过表2的规定，则可判断试样中存在相应的被测物。

将供试试样和标准溶液的定量离子面积之比作单点校正定量。

# 7.4.3.2 定量测定

取试样溶液和标准溶液，按外标法，以峰面积定量，标准溶液及试样溶液中的阿维拉霉素响应值均应在仪器检测的线性范围内。在上述色谱-质谱条件下，阿维拉霉素标准溶液和空白添加试样溶液中特征离子质量色谱图见附录A。

# 8 结果计算与表述

试料中阿维拉霉素残留标示物（DIA）的残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算

$$
X = \frac {A \times C s \times C i \times A s i \times V}{A s \times C s i \times A i \times m}
$$

X----供试试料中DIA的残留量， $\mu \mathrm{g / kg}$

$\mathrm{C}_{\mathrm{S}}$ -----标准溶液中 DIA 的浓度, $\mathrm{ng/mL}$ ;

A----试样溶液中DIA的色谱峰面积；

$\mathrm{A}_{\mathrm{S}}$ ----标准溶液中DIA的色谱峰面积；

$\mathrm{C_i}$ ---试样溶液中内标物的浓度， $\mathrm{ng / mL}$

$\mathrm{C_{si} - - }$ 标准溶液中内标物的浓度， $\mathrm{ng / mL}$

$\mathrm{A_{si} - - }$ 标准溶液中内标物的色谱峰面积；

$\mathrm{A_j}$ ---试样溶液中内标物的色谱峰面积；

V----样液最终定容体积，mL;

m----供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度、精密度

# 9.1灵敏度

本方法在肌肉和脂肪组织中的检测限为 $10~\mu \mathrm{g / kg}$ ，定量限为 $20~\mu \mathrm{g / kg}$ ；在肝脏和肾脏组织中的检测限为 $20~\mu \mathrm{g / kg}$ ，定量限为 $50~\mu \mathrm{g / kg}$ 。

# 9.2 准确度

本方法在 $20\sim 600~\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $70\% \sim 120\%$

# 9.3 精密度

本方法的批内相对标准偏差 $\leq 20\%$ ，批间相对标准偏差 $\leq 20\%$ 。

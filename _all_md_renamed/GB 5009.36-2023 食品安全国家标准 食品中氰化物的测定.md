# 前言

本标准代替GB5009.36—2016《食品安全国家标准 食品中氰化物的测定》。

本标准与GB5009.36—2016相比，主要变化如下：

——增加了第三法气相色谱-质谱法，第四法离子色谱法，第五法流动注射/连续流动-分光光度法；  
——删除了第三法定性法；  
——修改了标准适用范围、第一法分光光度法试剂配制和前处理步骤、第二法气相色谱法前处理步骤。

# 食品安全国家标准

# 食品中氰化物的测定

# 1 范围

本标准规定了食品中氰化物的测定方法。

第一法分光光度法适用于蒸馏酒及其配制酒、食用酒精、木薯粉、包装饮用水和矿泉水中氰化物的测定。

第二法气相色谱法和第三法气相色谱-质谱法适用于白酒、食用酒精、粮食、包装饮用水和矿泉水中氰化物的测定。

第四法离子色谱法适用于蒸馏酒及其配制酒、食用酒精、包装饮用水、矿泉水和饮料（以杏仁为原料）中氰化物的测定。

第五法流动注射/连续流动-分光光度法适用于蒸馏酒及其配制酒、食用酒精、包装饮用水和矿泉水中氰化物的测定。

# 第一法 分光光度法

# 2 原理

试样在碱性条件下加热除去高沸点有机物或在酸性条件下加热蒸馏，用氢氧化钠溶液溶解或吸收后，在 $\mathrm{pH} = 7.0$ 条件下，用氯胺T将氰化物转变为氯化氰，再与异烟酸-吡唑啉酮作用，生成蓝色染料，与标准系列比较定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 无水乙醇 $(\mathrm{C}_2\mathrm{H}_6\mathrm{O})$ 。  
3.1.2 乙酸锌 $\mathrm{(C_4H_6O_4Zn)}$ 。  
3.1.3 酒石酸 $(\mathrm{C_4H_6O_6})$ 。  
3.1.4 氢氧化钠(NaOH)。  
3.1.5 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$ 。  
3.1.6 磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4)$ 。  
3.1.7 冰乙酸 $\mathrm{(C_2H_4O_2)}$ ： $>99.5\%$   
3.1.8 异烟酸 $(\mathrm{C_6H_5O_2N})$ 。  
3.1.9 吡唑啉酮 $\mathrm{(C_{10}H_{10}N_2O)}$ 。  
3.1.10 氯胺 $\mathrm{T(C_7H_7SO_2NClNa\cdot 3H_2O)}$ 。  
3.1.11 甲基橙 $(\mathrm{C_{14}H_{14}O_3N_3SNa})$ 。  
3.1.12 酚酞 $(\mathrm{C}_{20}\mathrm{H}_{14}\mathrm{O}_4)$ 。

# 3.2 试剂配制

3.2.1 磷酸盐缓冲溶液 $(0.5\mathrm{mol} / \mathrm{L},\mathrm{pH} = 7.0)$ ：称取 $3.40\mathrm{g}$ 无水磷酸二氢钾和 $3.55\mathrm{g}$ 无水磷酸氢二钠，溶于水中，并定容至 $100~\mathrm{mL}$ 。  
3.2.2 氢氧化钠溶液 $(20\mathrm{g} / \mathrm{L})$ ：称取 $2\mathrm{g}$ 氢氧化钠，溶于水中，并定容至 $100~\mathrm{mL}$ 。  
3.2.3 氢氧化钠溶液 $(10\mathrm{g / L})$ ：称取 $1\mathrm{g}$ 氢氧化钠，溶于水中，并定容至 $100~\mathrm{mL}$ 。  
3.2.4 乙酸锌溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 乙酸锌，溶于水中，并定容至 $100~\mathrm{mL}$ 。  
3.2.5 氢氧化钠溶液 $(2\mathrm{g} / \mathrm{L})$ ：量取 $10~\mathrm{mL}$ 氢氧化钠溶液 $(20\mathrm{g} / \mathrm{L})$ ，用水定容至 $100~\mathrm{mL}$ 。  
3.2.6 氢氧化钠溶液 $(1\mathrm{g} / \mathrm{L})$ ：量取 $5\mathrm{mL}$ 氢氧化钠溶液 $(20\mathrm{g} / \mathrm{L})$ ，用水定容至 $100~\mathrm{mL}$ 。  
3.2.7 乙酸溶液 $(1 + 6)$ ：将乙酸和水按 $1:6$ 的体积比混匀。  
3.2.8 酚酞-乙醇指示液 $(10\mathrm{g / L})$ ：称取 $1\mathrm{g}$ 酚酞试剂，用无水乙醇溶解，并定容至 $100~\mathrm{mL}$ 。  
3.2.9 甲基橙指示剂 $(0.5\mathrm{g} / \mathrm{L})$ ：称取 $50~\mathrm{mg}$ 甲基橙，溶于水中，用水定容至 $100~\mathrm{mL}$ 。  
3.2.10 异烟酸-吡唑啉酮溶液：称取 $1.5\mathrm{g}$ 异烟酸溶于 $24~\mathrm{mL}$ 氢氧化钠溶液 $(20~\mathrm{g / L})$ 中，加水至 $100~\mathrm{mL}$ ，另称取 $0.25\mathrm{g}$ 吡唑啉酮，溶于 $20~\mathrm{mL}$ 无水乙醇中，合并上述两种溶液，摇匀，临用现配。  
3.2.11 氯胺T溶液 $(20\mathrm{g / L})$ ：称取 $2\mathrm{g}$ 氯胺T溶于水中，并稀释至 $100~\mathrm{mL}$ ，临用现配。

# 3.3 标准品

氰化物标准溶液 $(50.0\mu \mathrm{g / mL}$ ，以 $\mathrm{CN^{-}}$ 计）：经国家认证并授予标准物质证书的标准品，保质期和储存条件参考标准物质证书。

# 3.4 标准溶液配制

氰离子标准溶液 $(1.0\mu \mathrm{g / mL})$ ：准确吸取 $2.00~\mathrm{mL}$ 氰化物标准溶液 $(50.0\mu \mathrm{g / mL})$ ，置于 $100~\mathrm{mL}$ 容量瓶中，用氢氧化钠溶液 $(2\mathrm{g / L})$ 定容至刻度，临用现配。

注：氰化物为剧毒物质，操作时请在通风橱进行。

# 4 仪器与设备

4.1 可见分光光度计。  
4.2 分析天平：感量为 $0.001\mathrm{g}$   
4.3 恒温水浴锅。  
4.4 电加热板（带温度控制）。  
4.5 $250~\mathrm{mL}$ 和 $500~\mathrm{mL}$ 蒸馏装置。  
4.6 具塞比色管： $10~\mathrm{mL},25~\mathrm{mL}$

# 5 分析步骤

# 5.1 样品前处理

# 5.1.1 木薯粉

5.1.1.1 称取 $10\mathrm{g}$ (精确到 $0.001\mathrm{g}$ ) 试样于 $250\mathrm{mL}$ 蒸馏装置中, 加水约 $150\mathrm{mL}$ , 加塞密封, 在室温下磁力搅拌 $2\mathrm{h}$ , 加入 $20\mathrm{mL}$ 乙酸锌溶液和 $2.0\mathrm{g}$ 酒石酸, 迅速连接好蒸馏装置 (水冷需冷凝器调节温度至 $10^{\circ}\mathrm{C} \sim 15^{\circ}\mathrm{C}$ 进行蒸馏), 将冷凝管下端插入盛有 $10\mathrm{mL}$ 氢氧化钠溶液 $(20\mathrm{g/L})$ 的 $100\mathrm{mL}$ 容量瓶的液面下 (如果室温高于 $25^{\circ}\mathrm{C}$ , 需采取冰袋进行降温, 使蒸馏液温度在 $20^{\circ}\mathrm{C} \sim 25^{\circ}\mathrm{C}$ 之间), 进行蒸馏, 收集

蒸馏液接近 $100 \mathrm{~mL}$ 时, 用水定容至刻度。准确吸取 $1.00 \mathrm{~mL}$ 溶液置于 $10 \mathrm{~mL}$ 比色管中, 加 $2 \mathrm{~g} / \mathrm{L}$ 氢氧化钠溶液至 $5 \mathrm{~mL}$ , 摇匀后, 作为试样溶液。

5.1.1.2 于试样溶液加入2滴酚酞指示液，加入乙酸溶液调至红色褪去，再用 $20\mathrm{g / L}$ 氢氧化钠溶液调至近红色，然后加 $2\mathrm{mL}$ 磷酸盐缓冲溶液，再加入 $0.2\mathrm{mL}$ 氯胺T溶液，摇匀放置 $3\mathrm{min}$ ，加入 $2\mathrm{mL}$ 异烟酸-吡唑啉酮溶液，加水定容至刻度，加塞振荡混合均匀，在 $37^{\circ}C$ 恒温水浴锅中放置 $40\mathrm{min}$ ，待测。

# 5.1.2 蒸馏酒及其配制酒、食用酒精

5.1.2.1 准确吸取 $1.00\mathrm{mL}$ 试样（其中芝麻香型白酒、酱香型白酒、特香型白酒试样吸取 $0.10\mathrm{mL}$ ）于 $25\mathrm{mL}$ 烧杯中，加入 $5\mathrm{mL}$ 氢氧化钠溶液 $(2\mathrm{g} / \mathrm{L})$ ，放置 $10\mathrm{min}$ ，然后放于 $160^{\circ}\mathrm{C} \sim 180^{\circ}\mathrm{C}$ 电加热板上低温加热至溶液剩余约 $1\mathrm{mL}$ ，取下放至室温，用 $2\mathrm{g} / \mathrm{L}$ 氢氧化钠溶液转移至 $10\mathrm{mL}$ 具塞比色管中，最后加 $2\mathrm{g} / \mathrm{L}$ 氢氧化钠溶液至 $5\mathrm{mL}$ 。

5.1.2.2 若酒样浑浊或有色，取 $25.0\mathrm{mL}$ 试样于 $250~\mathrm{mL}$ 蒸馏瓶中，加入 $100~\mathrm{mL}$ 水，加数粒玻璃珠，滴加数滴甲基橙指示剂，将冷凝管下端插入盛有 $10~\mathrm{mL}$ 氢氧化钠溶液 $(20~\mathrm{g / L})$ 比色管的液面下，再加 $1~\mathrm{g} \sim 2~\mathrm{g}$ 酒石酸，溶液由橙黄色变成橙红，迅速连接蒸馏装置进行蒸馏，收集蒸馏液接近 $50~\mathrm{mL}$ ，然后用水定容至 $50~\mathrm{mL}$ ，混合均匀。取 $2.00~\mathrm{mL}$ 馏出液按5.1.2.1操作。

5.1.2.3 同5.1.1.2。

# 5.1.3 包装饮用水、矿泉水

5.1.3.1 量取 $250~\mathrm{mL}$ 水样(氰化物含量超过 $20~\mu \mathrm{g}$ 时，可取适量水样，加水至 $250~\mathrm{mL}$ )置于 $500~\mathrm{mL}$ 蒸馏装置中，加入1滴～2滴甲基橙指示剂，再加入 $5\mathrm{mL}$ 乙酸锌溶液，加入 $1\mathrm{g}\sim 2\mathrm{g}$ 酒石酸，溶液由橙黄色变成了橙红，迅速连接好蒸馏装置，将冷凝管下端插入盛有 $5.0~\mathrm{mL}$ 氢氧化钠溶液 $(20~\mathrm{g / L})$ 的 $50~\mathrm{mL}$ 具塞比色管的液面下。通过调节温度将蒸馏速度控制在 $2\mathrm{mL} / \mathrm{min}\sim 3\mathrm{mL} / \mathrm{min}$ ，收集蒸馏液接近 $50~\mathrm{mL}$ ，然后用水定容至 $50~\mathrm{mL}$ ，混合均匀。取 $10.0~\mathrm{mL}$ 馏出液置于 $25~\mathrm{mL}$ 具塞比色管中。

5.1.3.2 于盛有试样的比色管中加入 $5.0\mathrm{mL}$ 磷酸盐缓冲液，置于 $37^{\circ}\mathrm{C}$ 恒温水浴中，再加入 $0.25\mathrm{mL}$ 氯胺T溶液，加塞混合，放置 $5\mathrm{min}$ ，然后加入 $5.0\mathrm{mL}$ 异烟酸-吡唑酮溶液，加水至 $25\mathrm{mL}$ ，混匀，在 $37^{\circ}\mathrm{C}$ 恒温水浴锅中放置 $40\mathrm{min}$ ，待测。

# 5.2.2 蒸馏酒及其配制酒、食用酒精标准曲线

分别准确吸取 $0\mathrm{mL},0.40\mathrm{mL},0.80\mathrm{mL},1.20\mathrm{mL},1.60\mathrm{mL}$ 和 $2.00\mathrm{mL}$ 氰离子标准液 $(1.0\mu \mathrm{g / mL})$ 于 $50~\mathrm{mL}$ 烧杯中[相当于氰化物 $\mathrm{(CN^{-}}$ 计)质量： $0\mu \mathrm{g},0.40\mu \mathrm{g},0.80\mu \mathrm{g},1.20\mu \mathrm{g},1.60\mu \mathrm{g},2.00\mu \mathrm{g}]$ ，加入 $5\mathrm{mL}$ 氢氧化钠溶液( $2\mathrm{g / L})$ ，放置 $10\mathrm{min}$ ，以下按照5.1.2.1和5.1.2.3与样品同步操作，取制备的标准曲线待测液，用 $1\mathrm{cm}$ 比色皿，以 $2\mathrm{g / L}$ 氢氧化钠溶液为空白溶液调节零点，于波长 $638~\mathrm{nm}$ 处测吸光度，以氰离子质量为横坐标，氰离子的吸光度为纵坐标，绘制标准曲线。

# 6 分析结果的表述

# 6.1 木薯粉结果计算

试样中氰化物（以 $\mathrm{CN^{-}}$ 计)的含量按式(1)计算。

$$
X = \frac {A \times 1 0 0 0 \times V _ {1}}{m \times V _ {2} \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中氰化物含量（以 $\mathrm{CN}^{-}$ 计），单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A$ ——测定试样溶液氰化物质量(以 $\mathrm{CN}^{-}$ 计)，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——试样蒸馏液总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克(g)；

$V_{2}$ ——测定用蒸馏液体积，单位为毫升 $(\mathrm{mL})$ ；

1000 单位换算系数。

注：氰化物含量以HCN计时，结果乘以换算系数1.0385。

计算结果保留3位有效数字。

$$
X = \frac {A \times 1 0 0 0}{V \times 1 0 0 0} \tag {2}
$$

式中：

式中：

2 ——馏出液移取体积，单位为毫升 $(\mathrm{mL})$ ；

1000——单位换算系数。

注：氰化物含量以HCN计时，结果乘以换算系数1.0385。

# 第二法 气相色谱法

# 9 原理

在密闭容器和一定温度下，试样中的氰化物经碱解或提取后，在酸性条件下用氯胺T将其衍生为氯化氰，氯化氰在气相和液相中达到平衡，将气相部分导入气相色谱仪进行分离，电子捕获检测器检测，以外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的二级水。

# 10.1 试剂

10.1.1 氯胺 $\mathrm{T(C_7H_7ClNNaO_2S\cdot 3H_2O)}$ 。  
10.1.2 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ ： $\geq 85\%$   
10.1.3 氢氧化钠(NaOH)。

# 10.2 试剂配制

10.2.1 氯胺T溶液 $(10\mathrm{g / L})$ ：称取 $0.1\mathrm{g}$ 氯胺T，用水溶解定容至 $10~\mathrm{mL}$ ，临用现配。  
10.2.2 磷酸溶液 $(1 + 5)$ ：量取 $10~\mathrm{mL}$ 浓磷酸，加入到 $50~\mathrm{mL}$ 水中，混合均匀。  
10.2.3 $0.1\%$ 氢氧化钠溶液：称取 $1.0\mathrm{g}$ 氢氧化钠，用水溶解定容至 $1\mathrm{L}$ 。  
10.2.4 $0.01\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：称取 $0.4\mathrm{g}$ 氢氧化钠，用水溶解定容至 $1\mathrm{L}$ 。

# 13.1 白酒、食用酒精、包装饮用水和矿泉水中氰化物（以 $\mathbf{CN}^{-}$ 计）结果计算

试样中氰化物（以 $\mathrm{CN^{-}}$ 计)含量按式(5)计算。

$$
X = \frac {\rho - \rho_ {0}}{V} \times 1 0 \tag {5}
$$

式中：

$X$ ——试样中氰化物(以 $\mathrm{CN}^{-}$ 计)的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho$ ——由标准曲线得到的样液中氰化物的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——由标准曲线得到的空白试验中氰化物的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——样品体积，单位为毫升 $(\mathrm{mL})$ ；

10——加酸衍生前顶空瓶中溶液体积，单位为毫升 $(\mathrm{mL})$ 。

注：氰化物含量以HCN计时，结果乘以换算系数1.0385。

计算结果保留3位有效数字。

# 13.2 粮食中氰化物（以 $\mathbf{CN}^{-}$ 计）结果计算

试样中氰化物（以 $\mathrm{CN^{-}}$ 计)含量按式(6)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {6}
$$

式中：

$X$ ——试样中氰化物(以 $\mathrm{CN}^{-}$ 计)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由标准曲线得到的样液中氰化物的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——由标准曲线得到的空白试验中氰化物的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——样品定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品质量，单位为克(g)；

1000——单位换算系数。

注：氰化物含量以HCN计时，结果乘以换算系数1.0385。

计算结果保留3位有效数字。

# 第四法 离子色谱法

# 23 原理

试样进入色谱柱，随着淋洗液的流动，试样中氰化物与色谱柱上的活性交换基团反复发生交换与洗脱，根据氰化物与其他离子在色谱柱上的保留特性不同实现分离，用安培检测器进行测定。以色谱峰的相对保留时间定性，以峰面积或峰高定量。

准确移取 $1.00\mathrm{mL}$ 试样于 $10.0\mathrm{mL}$ 比色管中，用水定容至刻度，混匀，然后经 $0.22\mu \mathrm{m}$ 尼龙微孔滤膜过滤，待测。当样品颜色较深或含糖量较高时，将待测液先通过IC-RP固相萃取柱，整个流速控制在 $4\mathrm{mL / min}$ ，弃去最初的 $3\mathrm{mL}$ 或 $6\mathrm{mL}$ （根据柱子规格选择），再经 $0.22\mu \mathrm{m}$ 尼龙微孔滤膜过滤，待测。

# 26.1.2.3 饮料（以杏仁为原料）

准确移取 $4.00\mathrm{mL}$ 试样于 $10.0\mathrm{mL}$ 比色管中，用乙腈定容至刻度，涡旋混匀 $2\mathrm{min}$ ，转移至离心管中，静置后离心 $10\mathrm{min}$ 。准确移取上清液 $2.5\mathrm{mL}$ ，用水定容至 $10.0\mathrm{mL}$ 。取上述溶液适量，先通过IC-RP固相萃取柱，整个流速控制在 $4\mathrm{mL / min}$ ，弃去最初的 $3\mathrm{mL}$ 或 $6\mathrm{mL}$ （根据柱子规格选择），再经 $0.22\mu \mathrm{m}$ 尼龙微孔滤膜过滤，待测。

式中：

# 30 原理

试样中的氰化物在酸性条件下，通过在线蒸馏将氰化物以氢氰酸的形式蒸馏出，气态氢氰酸被氢氧化钠溶液吸收，在弱酸性条件下，氰离子和氯胺T反应生成氯化氰，再与异烟酸反应，经水解生成戊烯二醛，戊烯二醛与两个巴比妥酸分子缩合生成蓝色染料，与标准系列比较定量。

# 前言

本标准代替GB/T23217—2008《水产品中河豚毒素的测定液相色谱-荧光检测法》、GB/T5009.206—2007《鲜河豚鱼中河豚毒素的测定》和SN/T1569.2—2013《出口河豚鱼中河豚毒素检测方法第2部分：小鼠生物法》。

本标准与GB/T5009.206—2007相比，主要变化如下：

——标准名称修改为“食品安全国家标准 水产品中河豚毒素的测定”；  
——增加了净化步骤；  
——修改了酶联免疫吸附法。

# 食品安全国家标准

# 水产品中河豚毒素的测定

# 1 范围

本标准规定了水产品中河豚毒素的测定方法。

本标准第一法适用于河豚鱼肌肉、肝脏、皮肤和性腺组织中河豚毒素的测定。第二法适用于河豚鱼肌肉、肝脏、皮肤和性腺组织中河豚毒素的测定。第三法适用于河豚鱼、织纹螺、虾、牡蛎、花蛤和鱿鱼中河豚毒素的测定。第四法适用于河豚鱼肌肉、肝脏、皮肤和性腺组织中河豚毒素的测定。

# 第一法 小鼠生物法

# 2 原理

根据河豚毒素易溶于酸性溶液原理，试样制备后经两次乙酸溶液 $(0.5\%)$ 煮沸提取， $20000\mathrm{g}$ 离心收集上清液，将两次上清液合并定容至 $25~\mathrm{mL}$ ，用于小鼠生物试验。根据小鼠注射试样提取液后的死亡时间，查出鼠单位，并按小鼠体重校正鼠单位，计算确定河豚毒素含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，实验用水为GB6682规定的三级水。

# 3.1 试剂

3.1.1 冰乙酸 $\mathrm{(CH_3COOH)}$   
3.1.2 氢氧化钠(NaOH)。

# 3.2 试剂配制

3.2.1 乙酸溶液 $(0.5\%)$ ：将 $5\mathrm{mL}$ 冰乙酸用水稀释至 $1\mathrm{L}$ 。  
3.2.2 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：将 $40\mathrm{g}$ 氢氧化钠用水溶解并定容至 $1000~\mathrm{mL}$ 。

# 3.3 标准品

河豚毒素 $\mathrm{(C_{11}H_{17}N_3O_8}$ ，CAS号：4368-28-9)，纯度 $\geq 98\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 实验动物

小白鼠：体重 $19.0\mathrm{g}\sim 21.0\mathrm{g}$ ，无特定病原体级(SPF)昆明系雄性健康小鼠。

# 4 仪器和设备

4.1 均质器： $\geqslant 12000\mathrm{r / min}$

4.2 电子天平：感量 $0.1\mathrm{g}$   
4.3 离心机：离心力 $\geqslant 30000g$   
4.4 秒表。

# 5 分析步骤

注：为避免毒素的危害，应戴手套进行检验操作。移液器吸头等用过的器材、废弃的提取液等应在氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ 中浸泡 $1\textrm{h}$ 以上，以使毒素分解。

# 5.1 试样制备

# 5.1.1 冷冻河豚鱼样品

将样品装于密封塑料袋内于自来水水流下快速解冻。解冻后用水洗净，用滤纸吸干，分解成肌肉、肝脏、皮肤等部分。分别称量后将各组织剪碎，充分均质。

# 5.1.2 新鲜河豚鱼样品

将样品用水洗净后用滤纸吸干，分解成肌肉、肝脏、皮肤等部分，分别称量后将各组织剪碎，充分均质。

# 5.2 试样提取

# 5.2.1 肌肉组织试样

准确称取试样 $10\mathrm{g}$ (精确至 $0.1\mathrm{g}$ ) 于 $50~\mathrm{mL}$ 离心管中，加入乙酸溶液 $(0.5\%)$ $11~\mathrm{mL}$ ，充分混匀，沸水浴中煮沸 $10\mathrm{min}$ ，不断搅拌以防止组织结块。冷却至室温， $20000\mathrm{g}$ 高速离心 $20\mathrm{min}$ ，移取上清液于 $25~\mathrm{mL}$ 容量瓶中。向沉淀中加入乙酸溶液 $(0.5\%)$ $11~\mathrm{mL}$ ，充分混匀，沸水浴中煮沸 $5\mathrm{min}$ ，不断搅拌以防止组织结块。冷却至室温， $20000\mathrm{g}$ 高速离心 $20\mathrm{min}$ ，移取上清液。合并上清液，混匀，用乙酸溶液 $(0.5\%)$ 定容至 $25~\mathrm{mL}$ 。此提取液 $1\mathrm{mL}$ 相当于 $0.4\mathrm{g}$ 试样。

# 5.2.2 肝脏组织试样

准确称取 $10\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）试样于 $50~\mathrm{mL}$ 离心管中，加入乙酸溶液 $(0.5\%)$ $8\mathrm{mL}$ ，充分混匀，室温下 $20000g$ 高速离心 $20\mathrm{min}$ ，移取上清液于 $25~\mathrm{mL}$ 容量瓶中。用 $8\mathrm{mL}$ 乙酸溶液 $(0.5\%)$ 重复提取1次。合并上清液，混匀，用乙酸溶液 $(0.5\%)$ 定容至 $25~\mathrm{mL}$ 。此提取液 $1\mathrm{mL}$ 相当于 $0.4\mathrm{g}$ 试样。

# 5.2.3 皮肤组织试样

准确称取 $10\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）试样于 $50~\mathrm{mL}$ 离心管中，加入乙酸溶液 $(0.5\%)$ $11~\mathrm{mL}$ ，充分混匀，室温下 $20000g$ 高速离心 $20\mathrm{min}$ ，移取上清液于 $25~\mathrm{mL}$ 容量瓶中。用 $11~\mathrm{mL}$ 乙酸溶液 $(0.5\%)$ 重复提取1次。合并上清液，用乙酸溶液 $(0.5\%)$ 定容至 $25~\mathrm{mL}$ 。此提取液 $1\mathrm{mL}$ 相当于 $0.4\mathrm{g}$ 试样。

注1：试样制备时，冷冻样品需在半解冻状态下进行操作。试样若不能及时检验，应于 $4^{\circ}\mathrm{C}$ 保存，在 $24\mathrm{h}$ 内检验。  
注2：如果河豚鱼样品的脏器或者组织不足 $10\mathrm{g}$ 的情况下，按照以上方法进行提取操作，需适量减少乙酸溶液 $(0.5\%)$ 的加入量。  
注3：提取液中应不含有离心分离液表面上被分离出来的油状物、胶状物或者脂质类。

# 5.3 小鼠试验

选取 $19.0\mathrm{g}\sim 21.0\mathrm{g}$ 的无特定病原体级(SPF)昆明系雄性健康小鼠6只，称量并记录体重。随机分

为实验组和空白对照组两组, 每组 3 只。对每只试验小鼠腹腔注射 $1 \mathrm{~mL}$ 试样提取液或乙酸溶液 $(0.5 \%$ (空白对照液)。若注射过程中注射液溢出, 须将该只小鼠丢弃, 并重新注射一只小鼠。记录注射开始和完毕时间, 仔细观察并记录小鼠停止呼吸时的死亡时间(到小鼠呼出最后一口气止)。

若注射试样提取液后，小鼠死亡时间小于 $7\mathrm{min}$ ，则按附录A中表A.1计算出 $1\mathrm{mL}$ 试样提取液的毒力，以此为基准，配制出使小鼠在 $7\mathrm{min} \sim 13\mathrm{min}$ 死亡所需的稀释液，采用乙酸溶液 $(0.5\%)$ 进行稀释，将稀释液再注射至少3只小鼠以确定试样的毒力。具体示例见附录B。若注射试样溶液后，小鼠的死亡时间大于或等于 $7\mathrm{min}$ ，则直接根据中位数致死时间确定试样的毒力。

小鼠中毒死亡症状：被注射了河豚毒素的小鼠初期安静，随后出现呼吸困难，急促，腹部收缩加快，反应迟钝，继而出现突然疾走，急跑急跳，四处乱窜，东倒西歪，翻转乱跳等挣扎动作，数十秒后，小鼠后腿剧烈抽搐， $2\sim 3$ 次后爬卧不动，腹部呼吸逐渐微弱减慢，最后死亡。以停止呼吸作为判断死亡的标准。

# 6 分析结果的表述

# 6.1 毒力的计算

根据小鼠试验中得到的小鼠致死时间，计算出中位数致死时间，然后按表A.1计算出毒力(若中位数致死时间刚好处于表A.1中给出的两时间中间时，则取两时间中较大的时间值；若介于表A.1中给出的两时间中但不正好处于中间时，则取更接近于中位数致死时间的时间值)。若小鼠体重不正好为 $20.0\mathrm{g}$ ，则按表A.2对鼠单位(MU)进行校正，通过校正后的鼠单位表示出提取液的毒力。

1 MU的定义为使体重为 $20.0\mathrm{g}$ 的无特定病原体级(SPF)昆明系雄性小鼠 $30\mathrm{min}$ 死亡的毒力，相当于 $0.18\mu \mathrm{g}$ 河豚毒素。

试样中河豚毒素的含量按式(1)计算：

$$
X = M \times E \times f \times W \quad \dots \tag {1}
$$

式中：

$X$ ——试样中河豚毒素的含量，单位为鼠单位每克 $(\mathrm{MU} / \mathrm{g})$ ；

$M$ —— $1\mathrm{mL}$ 注射液的鼠单位数，单位为鼠单位（MU）；

$E$ ——提取系数，本方法提取系数为3.11；

$f$ ——试样提取液的稀释倍数；

$W$ ——小鼠体重校正系数。

# 6.2 结果报告

在空白对照组小鼠正常的情况下，河豚鱼各组织中河豚毒素含量进行如下判断和表述：

若小鼠的死亡时间大于 $30\mathrm{min}$ ，结果报告为 $\leq 3.11\mathrm{MU / g}$ 肌肉组织/肝脏/皮肤。

若小鼠的死亡时间小于 $30\mathrm{min}$ ，结果按式(1)计算得到实际结果报告，待测样品中河豚毒素含量为 $\times \times \times \mathrm{MU} / \mathrm{g}$ 肌肉组织/肝脏/皮肤。

若3只小鼠死亡时间不都小于 $30\mathrm{min}$ 或不都大于 $30\mathrm{min}$ ，则需重新注射3只小鼠，以确保3只小鼠死亡时间都小于 $30\mathrm{min}$ 或都大于 $30\mathrm{min}$ ，结果据此判定；若重复注射3只小鼠死亡时间仍出现第一次小鼠死亡情况，则需根据第一次注射3只小鼠的中位数致死时间判定结果。

# 7 其他

本方法的检出限为 $3.11\mathrm{MU / g}$

# 第二法 液相色谱-串联质谱法

# 8 原理

试样经 $1\%$ 乙酸-甲醇溶液提取，免疫亲和柱净化，液相色谱-串联质谱法测定，外标法定量。

# 9 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，实验用水为GB6682规定的一级水。

# 9.1 试剂

9.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
9.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
9.1.3 甲酸(HCOOH)：色谱纯。  
9.1.4 冰乙酸 $\mathrm{(CH_3COOH)}$ ：优级纯。  
9.1.5 乙酸铵 $\mathrm{(CH_3COONH_4)}$ ：色谱纯。  
9.1.6 十二水合磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4\cdot 12\mathrm{H}_2\mathrm{O})$ 。  
9.1.7 二水合磷酸二氢钠 $(\mathrm{NaH_2PO_4}\cdot 2\mathrm{H_2O})$   
9.1.8 氯化钠 $(\mathrm{NaCl})$ 。  
9.1.9 氢氧化钠(NaOH)。

# 9.2 试剂配制

9.2.1 乙酸-甲醇溶液 $(1 + 99)$ ：将冰乙酸和甲醇按1比99的体积比混合均匀。  
9.2.2 乙酸-甲醇溶液 $(2 + 98)$ ：将冰乙酸和甲醇按2比98的体积比混合均匀。  
9.2.3 甲酸溶液 $(1 + 999)$ ：将 $1\mathrm{mL}$ 甲酸加入到 $999~\mathrm{mL}$ 水中，混匀。  
9.2.4 含 $5\mathrm{mmol / L}$ 乙酸铵的 $0.1\%$ 甲酸溶液：称取 $0.19\mathrm{g}$ 乙酸铵，加入 $0.1\%$ 甲酸溶液定容至 $500~\mathrm{mL}$ 。  
9.2.5 甲酸溶液 $(0.1\%)$ -乙腈溶液 $(1 + 1)$ ：甲酸溶液 $(0.1\%)$ 与乙腈等体积混合均匀。  
9.2.6 磷酸盐缓冲溶液(PBS溶液)：称取十二水合磷酸氢二钠 $6.45\mathrm{g}$ 、二水合磷酸二氢钠 $1.09\mathrm{g}$ 、氯化钠 $4.25\mathrm{g}$ ，用水溶解并定容至 $500~\mathrm{mL}$ 。  
9.2.7 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：准确称取 $20.0\mathrm{g}$ 氢氧化钠，用水溶解并定容至 $500~\mathrm{mL}$ 。

# 9.3 标准品

河豚毒素 $\left(\mathrm{C}_{11} \mathrm{H}_{17} \mathrm{~N}_{3} \mathrm{O}_{8}\right)$ , CAS 号: 4368-28-9), 纯度 $\geqslant 98\%$ , 或经国家认证并授予标准物质证书的标准物质。

# 9.4 标准溶液配制

9.4.1 河豚毒素标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ : 准确称取河豚毒素标准品 $10\mathrm{mg}$ (精确至 $0.1\mathrm{mg}$ ), 用少量甲酸溶液 $(0.1\%)$ 溶解, 以甲醇定容至 $100\mathrm{mL}, -18^{\circ}\mathrm{C}$ 以下避光保存。  
9.4.2 河豚毒素标准中间液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：准确量取河豚毒素标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ 适量，以甲酸溶液 $(0.1\%)$ -乙腈溶液 $(1 + 1)$ 稀释并定容，配制成 $1\mu \mathrm{g} / \mathrm{mL}$ 的标准中间液， $-18^{\circ}\mathrm{C}$ 以下避光保存。  
9.4.3 河豚毒素标准系列工作液：准确吸取河豚毒素标准中间液 $(1\mu \mathrm{g / mL})$ 适量，以甲酸溶液 $(0.1\%)$ -

乙腈溶液 $(1 + 1)$ 稀释并定容，得到浓度为 $1\mu \mathrm{g} / \mathrm{L}\sim 1000\mu \mathrm{g} / \mathrm{L}$ 的标准系列工作液，现用现配。

# 9.5 材料

9.5.1 河豚毒素免疫亲和柱：柱规格 $3\mathrm{mL}$ ，最大柱容量 $1000\mathrm{ng}$ ，或等效柱。  
9.5.2 $0.22\mu \mathrm{m}$ 有机相微孔滤膜。

# 10 仪器和设备

10.1 液相色谱-串联质谱仪：配电喷雾离子源。  
10.2 分析天平：感量为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$   
10.3 均质机： $\geqslant 12000\mathrm{r / min}$   
10.4 涡旋振荡器。  
10.5 高速冷冻离心机：转速 $\geq 8000\mathrm{r / min}$   
10.6 固相萃取装置。  
10.7 氮吹仪。

# 11 分析步骤

注：为避免毒素的危害，应戴手套进行操作。移液器吸头等用过的器材、废弃的提取液等应在氢氧化钠溶液(1 mol/L)中浸泡 $1 \mathrm{~h}$ 以上，以使毒素分解。

# 11.1 试样制备

用水清洗鱼体表面的污物，滤纸吸干鱼体表面的水分后用剪刀将鱼体分解成肌肉、肝脏、皮肤和性腺(精巢或卵巢)等部分，各部分组织分别用水洗去血污，滤纸吸干表面的水分后将各组织剪碎，充分均质，装入清洁容器内，并标明标记。

# 11.2 试样提取

称取 $5\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 匀浆试样于 $50~\mathrm{mL}$ 具塞离心管中，加入 $11\mathrm{mL}$ 乙酸-甲醇溶液 $(1 + 99)$ ，涡旋振荡 $2\mathrm{min}$ ， $50^{\circ}\mathrm{C}$ 水浴超声提取 $15\mathrm{min}$ ，以 $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，转移上清液于 $25~\mathrm{mL}$ 容量瓶中。向残渣中加入 $11\mathrm{mL}$ 乙酸-甲醇溶液 $(1 + 99)$ ，重复提取一次，合并上清液，用乙酸-甲醇溶液 $(1 + 99)$ 定容至 $25~\mathrm{mL}$ 。移取约 $10\mathrm{mL}$ 提取液至另一 $50~\mathrm{mL}$ 具塞离心管中， $-20^{\circ}\mathrm{C}$ 冷冻 $30\mathrm{min}$ ，再以 $8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，准确移取 $5\mathrm{mL}$ 上清液，加入 $20~\mathrm{mL}$ PBS 溶液进行稀释，用氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ 调 $\mathsf{pH}$ 在 $7\sim 8$ 范围内，待净化。

# 11.3 试样净化

将免疫亲和柱中封存的PBS溶液以自然流速放出，移入待净化液，以1滴/s的流速过柱，再用 $10~\mathrm{mL}$ 水淋洗，抽干，用 $5\mathrm{mL}$ 乙酸-甲醇溶液 $(2 + 98)$ 洗脱，洗脱液于 $45^{\circ}C$ 氮气吹干，加入 $1.0~\mathrm{mL}$ 甲酸溶液 $(0.1\%)$ -乙腈溶液 $(1 + 1)$ 溶解残渣，超声 $1\mathrm{min}$ ，过 $0.22\mu \mathrm{m}$ 的有机相微孔滤膜后，供液相色谱-串联质谱分析。

b）流速： $0.3\mathrm{mL / min}$   
c） 柱温： $40^{\circ}C$ 。  
d）进样量： $10~\mu \mathrm{L}$

# 第三法 液相色谱-荧光检测法

# 15 原理

采用酸性甲醇溶液提取试样中的河豚毒素，提取液经旋转蒸发至干，用 $2\mathrm{mL}$ 乙酸溶液 $(1\%)$ 溶解残渣，再用磷酸盐缓冲溶液复溶，调 $\mathrm{pH}$ 至 $6.0\sim 7.0$ ，经免疫亲和柱净化，液相色谱-柱后衍生荧光法测

定，以保留时间定性，外标法定量。

# 18 分析步骤

# 18.1 试样制备

用水清洗鱼体表面的污物，滤纸吸干鱼体表面的水分后用剪刀将鱼体分解成肌肉、肝脏、皮肤和性腺(精巢或卵巢)等部分，各部分组织分别用水洗去血污，滤纸吸干表面的水分后将各组织剪碎，充分均质，装入清洁容器内，并标明标记。

# 18.2 试样提取

称取 $2\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ) 试样置于 $50~\mathrm{mL}$ 聚丙烯离心管中，加入 $10~\mathrm{mL}$ 乙酸-甲醇溶液 $(1 + 99)$ ，振荡 $2\mathrm{min}$ ， $60^{\circ}\mathrm{C}$ 水浴超声提取 $30\mathrm{min}$ ， $4500\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。上清液于 $60^{\circ}\mathrm{C}$ 旋转蒸发浓缩至近干，加入 $2\mathrm{mL}$ 乙酸溶液 $(1\%)$ 充分溶解残渣，转移至 $10~\mathrm{mL}$ 聚丙烯离心管中，加入 $8\mathrm{mL}$ PBS 溶液稀释，用氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ 调节 $\mathrm{pH}$ 至 $7\sim 8$ ，待净化。

# 18.3 试样净化

将免疫亲和柱回温至室温，去掉亲和柱堵头，放出柱内保存液后，移入 $10~\mathrm{mL}$ 待净化液。净化液全部流出后，用 $10~\mathrm{mL}$ 水淋洗，适当施压挤干柱内液体。用 $4~\mathrm{mL}$ 乙酸-甲醇溶液 $(2 + 98)$ 洗脱，收集的洗脱液于 $50^{\circ}\mathrm{C}$ 氮气吹干，用乙酸溶液 $(1\%)$ 溶解并定容至 $1~\mathrm{mL}$ ，经 $0.22\mu \mathrm{m}$ 的水相微孔滤膜过滤，滤液供液相色谱-荧光法分析。

# 第四法 酶联免疫吸附法

# 22 原理

试样中的河豚毒素经提取、脱脂后与定量的特异性抗体反应，多余的游离抗体则与酶标板内的包被抗原结合，然后加入酶标二抗与结合到酶标板上的抗体反应，加入底物显色，与标准曲线比较测定河豚毒素含量。

24.1 全波长光栅酶标仪或配有 $450\mathrm{nm}$ 滤光片的酶标仪。  
24.2 电子天平：感量 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$   
24.3 高速离心机： $\geqslant 12000\mathrm{r / min}$   
24.4 组织匀浆器。  
24.5 $37^{\circ}C$ 恒温培养箱。  
24.6 温控磁力搅拌器。

# 25 分析步骤

# 25.1 试样制备

对新鲜或冷冻后解冻的样品，用水清洗鱼体表面的污物，滤纸吸干鱼体表面的水分后用剪刀将鱼体分解成肌肉、肝脏、肠道、皮肤、卵巢(雄性为精囊)等部分，各部分组织分别用水洗去血污，滤纸吸干表面的水分后称重。

# 25.2 试样提取

将待测河豚鱼组织用剪刀剪碎，加入5倍体积的乙酸溶液 $(0.1\%)$ [即 $1\mathrm{g}$ 组织中加入乙酸溶液 $(0.1\%)$ 5mL]，用组织匀浆器磨成糊状。取相当于 $5\mathrm{g}$ 河豚组织的匀浆糊(即 $25~\mathrm{mL}$ )于烧杯中，置温控磁力搅拌器上边加热边搅拌，达到 $100^{\circ}C$ 时持续 $10\mathrm{min}$ 后取下，冷却至室温后， $8000~\mathrm{r / min}$ 离心 $15\mathrm{min}$ ，快速过滤于 $125~\mathrm{mL}$ 分液漏斗中。滤纸残渣用 $20~\mathrm{mL}$ 乙酸溶液 $(0.1\%)$ 分次洗净，合并洗液，置温控磁力搅拌器上边加热边搅拌，达到 $100^{\circ}C$ 时持续 $3\mathrm{min}$ 后取下， $8000~\mathrm{r / min}$ 离心 $15\mathrm{min}$ ，合并上清液。加入等体积的乙醚振摇脱脂，静置分层后，放出水层至另一分液漏斗中并以等体积乙醚再重复脱脂一次，将水层放入 $100~\mathrm{mL}$ 锥形瓶中，于 $35^{\circ}C$ 旋转蒸发去除其中残存的乙醚，将提取液移入 $50~\mathrm{mL}$ 容量瓶中。用氢氧化钠溶液 $(1\mathrm{mol / L})$ 调 $\mathsf{pH}$ 至 $6.5\sim 7.0$ ，并用PBS溶液定容至 $50~\mathrm{mL}$ ，立即用于检测（每毫升提取液相当于 $0.1\mathrm{g}$ 河豚组织试样）。当日不能检测的提取液经减压浓缩去除其中残存的乙醚后不调 $\mathsf{pH}$ ，密封后一 $20^{\circ}C$ 冷冻保存，在检测前调节 $\mathsf{pH}$ 并定容至 $50~\mathrm{mL}$ ，立即检测。

# 25.3 测定

# 25.3.1 包被酶标微孔板

用包被抗原包被酶标板， $120~\mu \mathrm{L / }$ 孔， $4^{\circ}\mathrm{C}$ 静置 $12\mathrm{h}$

# 25.3.2 抗体抗原反应

将抗河豚毒素单克隆抗体工作液分别与不同浓度的河豚毒素标准系列工作液等体积混合于 $2\mathrm{mL}$ 试管内， $4^{\circ}\mathrm{C}$ 静置 $12\mathrm{h}$ 或 $37^{\circ}\mathrm{C}$ 温育 $2\mathrm{h}$ ，备用。此溶液用于制作河豚毒素的标准曲线。

将抗河豚毒素单克隆抗体工作液与试样提取液等体积混合于 $2\mathrm{mL}$ 试管内， $4^{\circ}\mathrm{C}$ 静置 $12\mathrm{h}$ 或 $37^{\circ}\mathrm{C}$ 温育 $2\mathrm{h}$ ，备用。此溶液用于测定试样中河豚毒素的含量。

# 25.3.3 封闭

已包被的酶标板用PBS-T洗液洗涤3次(每次浸泡 $3\mathrm{min}$ )后，加封闭液封闭， $200~\mu \mathrm{L}/$ 孔，置 $37^{\circ}C$ 温育 $2\mathrm{h}$ 。

# 25.3.4 显色及测定

封闭后的酶标板用PBS-T洗液洗涤3次(每次浸泡 $3\mathrm{min}$ )，加入25.3.2中制备好的抗原抗体反应液（在酶标板的适当孔位加抗体稀释液作为阴性对照）， $100~\mu \mathrm{L}/$ 孔， $37^{\circ}C$ 温育 $2\mathrm{h}$ ，酶标板用PBS-T洗液洗涤3次(每次浸泡 $3\mathrm{min}$ )后，加入辣根过氧化物酶标记羊抗鼠抗体工作液， $100~\mu \mathrm{L}/$ 孔， $37^{\circ}C$ 温育 $1\mathrm{h}$ ，酶标板用PBS-T洗液洗涤5次(每次浸泡 $3\mathrm{min}$ )，加新配制的底物溶液， $100~\mu \mathrm{L}/$ 孔， $37^{\circ}C$ 温育 $15\mathrm{min}$ 后，每孔加入 $50~\mu \mathrm{L}$ 硫酸溶液 $(2\mathrm{mol / L})$ 终止显色反应，在波长 $450~\mathrm{nm}$ 处测定吸光度值。

# 25.4 标准曲线的制作

根据25.3.4中获得的河豚毒素标准系列的吸光度值，以河豚毒素标准系列工作液的浓度的对数值（以10为底的对数值)为横坐标，以河豚毒素标准系列工作液的吸光度值与 $0\mu \mathrm{g} / \mathrm{L}$ 的河豚毒素标准工

作液的吸光度值的比值为纵坐标，绘制标准曲线。

# 25.5 空白实验

除不加试样外，均按上述测定步骤进行。

# 26 分析结果的表述

试样中河豚毒素含量按式(4)计算：

$$
X = \frac {c \times V \times f}{m} \tag {4}
$$

式中：

$X$ ——试样中河豚毒素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c$ ——由标准曲线得到的试样待测液中河豚毒素的浓度，单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$V$ ——定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释倍数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

# 27 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 28 其他

本方法的检出限为 $3\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$

# 小鼠生物法计算示例

从 $10.0\mathrm{g}$ 脏器试样提取得到 $25~\mathrm{mL}$ 试样溶液对3只小鼠进行试验，若中位数致死时间是 $5\mathrm{min}$ $15\mathrm{s}$ ，小鼠体重为 $19.5\mathrm{g}$ ，试样提取液的毒力约为 $3.58\mathrm{MU / mL}$ 。按表A.1可以推测，稀释1倍后死亡时间约为 $10\mathrm{min}$ 。将试样溶液稀释1倍进行试验，若得出中位数致死时间为 $10\mathrm{min}$ ，则稀释液的毒力即为 $1.80\mathrm{MU / mL}$ 。因为稀释倍数是2，提取系数为3.11，按表A.2的重量校正系数为0.98，所以毒力计算如下：

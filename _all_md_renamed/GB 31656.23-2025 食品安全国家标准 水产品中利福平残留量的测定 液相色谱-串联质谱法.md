# 1 范围

本文件规定了水产品中利福平残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鱼、虾、蟹、海参等可食部分中利福平残留量的测定，其他水产品的检测可参照执行。

# 4 原理

试样中残留的利福平经乙腈提取，乙腈饱和正己烷除脂净化，液相色谱-串联质谱法测定，内标法定量。

# 5 试剂和材料

以下所用的试剂，除特别注明外均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 正己烷 $\mathrm{[CH_3(CH_2)_2CH_3]}$ ：色谱纯

5.1.2 乙腈 $\mathrm{(CH_3CN)}$ 色谱纯。

5.1.3 甲酸（HCOOH）：色谱纯。

5.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)}$

# 5.2 溶液的配制

5.2.1 乙腈饱和正己烷：量取正己烷 $200\mathrm{mL}$ 于 $250~\mathrm{mL}$ 分液漏斗中，加入适量乙腈后，剧烈振摇，待分配平衡后，弃去下层乙腈层。

5.2.2 $50\%$ 乙腈溶液：移取 $50~\mathrm{mL}$ 乙腈至 $50~\mathrm{mL}$ 水中，混匀。

5.2.3 $0.1\%$ 甲酸溶液：移取 $0.1\mathrm{mL}$ 甲酸，用水定容至 $1000~\mathrm{mL}$ ，混匀。

5.2.4 $0.1\%$ 甲酸乙腈溶液：移取 $0.1\mathrm{mL}$ 甲酸，用乙腈定容至 $1000\mathrm{mL}$ ，混匀。

# 5.3 标准品

5.3.1 利福平（Rifampicin， $\mathrm{C_{43}H_{58}N_4O_{12}}$ ，CAS号：13292-46-1）：含量 $\geqslant 97.0\%$ 。

5.3.2 利福平- $\mathrm{D_8}$ （Rifampicin- $\mathrm{D_8}$ ， $\mathrm{C_{43}H_{50}D_8N_4O_{12}}$ ）：含量 $\geqslant 97.0\%$

# 5.4 标准溶液的制备

5.4.1 标准储备液：取利福平标准品 $10\mathrm{mg}$ ，精密称定，用乙腈适量溶解并定容于 $100~\mathrm{mL}$ 棕色容量瓶，

配制成溶液浓度为 $100\mu \mathrm{g / mL}$ 的标准储备液。一 $20^{\circ}C$ 下避光保存，有效期为1个月。

5.4.2 标准中间液：精密量取利福平标准储备液 $1\mathrm{mL}$ ，用乙腈稀释配制成浓度为 $10~\mu \mathrm{g / mL}$ 的标准中间液。现用现配。  
5.4.3 标准工作液：精密量取利福平标准中间液 $1\mathrm{mL}$ ，用乙腈稀释配制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 的标准工作液。现用现配。  
5.4.4 内标储备液：取利福平- $\mathrm{D_8}$ 标准品 $10\mathrm{mg}$ ，精密称定，用乙腈适量溶解并定容于 $100~\mathrm{mL}$ 棕色容量瓶，配制成溶液浓度为 $100~\mu \mathrm{g / mL}$ 的内标储备液。 $-20^{\circ}C$ 下避光保存，有效期为1个月。  
5.4.5 内标中间液：精密量取利福平- $\mathrm{D}_{8}$ 内标储备液 $1\mathrm{mL}$ ，用乙腈稀释配制成浓度为 $10~\mu \mathrm{g / mL}$ 的内标中间液。现用现配。  
5.4.6 内标工作液：精密量取利福平- $\mathrm{D}_{8}$ 内标中间液 $1\mathrm{mL}$ ，用乙腈稀释配制成浓度为 $1\mu \mathrm{g} / \mathrm{mL}$ 的内标工作液。现用现配。  
5.4.7 内标使用液：精密量取利福平- $\mathrm{D}_{8}$ 内标工作液 $1\mathrm{mL}$ ，用乙腈稀释配制成浓度为 $0.1\mu \mathrm{g} / \mathrm{mL}$ 的内标使用液。现用现配。

# 5.5 材料

5.5.1 陶瓷均质子： $2\mathrm{cm}$ （长） $\times 1$ cm（外径），或相当者。  
5.5.2 尼龙微孔滤膜(有机相)： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 高效液相色谱-串联质谱仪：配电喷雾离子源(ESI)。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$ 。  
6.3 低温高速离心机：转速不低于 $8000\mathrm{r / min}$   
6.4 多管涡旋混合器：转速 $2500\mathrm{r / min}$ 。   
6.5 均质机：转速 $5000\mathrm{r / min}$ 以上。  
6.6 氮吹仪。

# 7 试样的制备与保存

# 7.1 试样的制备

试样的制备参考 GB/T 30891—2014 附录 B。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

-18℃以下保存。

# 8 测定步骤

# 8.1 提取

称取 $(2 \pm 0.05) \mathrm{g}$ 试料, 置于 $50 \mathrm{~mL}$ 塑料离心管中, 加入 $50 \mu \mathrm{L}$ 内标使用液, 加入1粒陶瓷均质子、 $2.0 \mathrm{~g}$ 无水硫酸钠和 $10 \mathrm{~mL}$ 乙腈, 涡旋混合器上以 $2500 \mathrm{r} / \mathrm{min}$ , 涡旋 $1 \mathrm{~min}$ , 超声 $10 \mathrm{~min}$ , 以 $8000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 取上清液转移至另一离心管中, 样品残渣再加入 $5 \mathrm{~mL}$ 乙腈重复提取1次, 合并上清液, 待净化。

# 8.2 净化

往上述乙腈提取液中加入 $10\mathrm{mL}$ 乙腈饱和正己烷， $2500\mathrm{r / min}$ 涡旋混合 $1\mathrm{min},8000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去正己烷层，将下层提取液转移到刻度离心管中， $30^{\circ}\mathrm{C}$ 氮吹至近干，加入 $1\mathrm{mL}50\%$ 乙腈溶液，涡旋混合器上 $2500\mathrm{r / min}$ ，涡旋 $1\mathrm{min},0.22\mu \mathrm{m}$ 滤膜过滤，供液相色谱-串联质谱仪测定。

# 8.3 标准工作曲线的制备

精密量取利福平标准工作液和 $0.1\mu \mathrm{g / mL}$ 内标使用液，用 $50\%$ 乙腈溶液稀释成浓度为 $0.5\mu \mathrm{g / L}$ 、 $1.0\mu \mathrm{g / L},2.0\mu \mathrm{g / L},5.0\mu \mathrm{g / L},10.0\mu \mathrm{g / L},20.0\mu \mathrm{g / L}$ 利福平系列标准工作液。系列标准工作液中利福平- $\mathrm{D_8}$ 的含量均为 $5.0\mu \mathrm{g / L}$ 。现用现配。取系列标准工作液进样，以利福平和利福平- $\mathrm{D_8}$ 峰面积比值为纵坐标、对应利福平浓度为横坐标，绘制标准工作曲线。求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在同样测试条件下，试料溶液中利福平与其内标利福平- $\mathrm{D_8}$ 的保留时间之比与标准溶液中利福平与其内标利福平- $\mathrm{D_8}$ 的保留时间之比偏差在 $1\%$ 以内；且检测到的相对离子丰度应与浓度相当的校正标准溶液相对丰度一致，其允许偏差为 $40\%$ 。

# 8.4.3.2 定量测定

取试料溶液和标准工作溶液，作单点或多点校准，内标法计算。标准溶液及试料溶液中目标化合物的特征离子质量色谱峰峰面积均应在仪器检测的线性范围之内。标准溶液和同位素标准溶液特征离子质量

# 8.5 空白试验

取空白试样，除不加药物外，采用完全相同的操作步骤进行平行操作。

# 9 结果计算与表述

试样中利福平的残留量按标准曲线法或公式(1)计算。

$$
X = \frac {A \times A _ {i s} ^ {\prime} \times C _ {s} \times C _ {i s} \times V}{A _ {i s} \times A _ {s} \times C _ {i s} ^ {\prime} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中利福平残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——试料溶液中被测物质的峰面积；

$A_{is}^{\prime}$ ——标准工作溶液中内标的峰面积；

$C_s$ ——标准工作溶液中被测物质浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$C_{is}$ ——试料溶液中内标浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——溶解残余物所用体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{is}$ ——试料溶液中内标的峰面积；

$A_{s}$ ——标准工作溶液中被测物质的峰面积；

$C_{is}^{\prime}$ ——标准工作溶液中内标浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$m$ ——最终样液代表的试样质量的数值，单位为克(g)。

# 10 检测方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法的检出限为 $0.2\mu \mathrm{g / kg}$ 、定量限为 $0.5\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法利福平在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 5.0\mu \mathrm{g} / \mathrm{kg}$ 添加浓度下的回收率为 $70\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

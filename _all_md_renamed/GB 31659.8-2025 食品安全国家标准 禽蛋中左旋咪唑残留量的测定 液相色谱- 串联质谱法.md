# 液相色谱-串联质谱法

# 1 范围

本文件规定了禽蛋中左旋咪唑残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鸡蛋、鸭蛋、鹅蛋、鸽子蛋、鹌鹑蛋中左旋咪唑残留量的测定，其他禽蛋的检测可参照执行。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 术语和定义

本文件没有需要界定的术语和定义

# 4 原理

在碱性条件下，用乙酸乙酯提取试样中残留的左旋咪唑，提取液经稀盐酸反萃取后用混合阳离子交换固相萃取小柱净化，液相色谱-串联质谱法检测，内标法定量。

# 5 试剂和材料

除另有说明外，所用试剂均为分析纯，水为 $\mathrm{GB} - 16682$ 规定的一级水

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ 色谱纯。  
5.1.2 甲醇 $(\mathrm{CH}_3\mathrm{OH})_2$ 色谱纯  
5.1.3 乙酸乙酯 $\mathrm{(CH_3CH_2OOCCH_3)}$ 色谱纯。  
5.1.4 甲酸铵 $\mathrm{(HCOONH_4)}$ ：色谱纯  
5.1.5 甲酸（HCOOH）：优级纯。  
5.1.6 盐酸 $(\mathrm{HCl})$ ： $36\% \sim 38\%$   
5.1.7 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ ： $25\% \sim 28\%$   
5.1.8 碳酸氢钠 $(\mathrm{NaHCO_3})$   
5.1.9 碳酸钠 $(\mathrm{Na}_2\mathrm{CO}_3)$ 。  
5.1.10 无水硫酸钠 $\mathrm{(Na_2SO_4)}$

# 5.2 溶液配制

5.2.1 $0.01\mathrm{mol} / \mathrm{L}$ 甲酸铵溶液（含 $0.1\%$ 甲酸）：取 $0.63\mathrm{g}$ 甲酸铵和 $1\mathrm{mL}$ 甲酸，用水溶解后稀释至 $1000\mathrm{mL}$ 。  
5.2.2 $0.2\mathrm{mol} / \mathrm{L}$ 盐酸溶液：取 $18~\mathrm{mL}$ 盐酸至适量水中，用水稀释至 $1000~\mathrm{mL}$   
5.2.3 饱和碳酸氢钠溶液：取水 $1000~\mathrm{mL}$ ，加碳酸氢钠至不溶解为止。  
5.2.4 饱和碳酸钠溶液：取水 $200~\mathrm{mL}$ ，加碳酸钠至不溶解为止。  
5.2.5 碳酸盐缓冲液：取 $900~\mathrm{mL}$ 饱和碳酸氢钠溶液、 $100~\mathrm{mL}$ 饱和碳酸钠溶液，混匀。

5.2.6 $5\%$ 氨水甲醇溶液：取 $5 \mathrm{~mL}$ 氨水，用甲醇稀释至 $100 \mathrm{~mL}$ 。  
5.2.7 乙腈-甲酸铵溶液：取 $10\mathrm{mL}$ 乙腈，用 $0.01\mathrm{mol} / \mathrm{L}$ 甲酸铵水溶液(含 $0.1\%$ 甲酸)稀释至 $100\mathrm{mL}$ 。

# 5.3 标准品

5.3.1 标准品：盐酸左旋咪唑(levamisole hydrochloride, $\mathrm{C_{11}H_{12}N_2S}$ ·HCl，CAS号：16595-80-5)或等效有证标准物质，纯度大于 $98\%$ 。  
5.3.2 内标物：盐酸左旋咪唑- $\mathrm{D}_5$ (Levamisole- $\mathrm{d}_5$ Hydrochloride, $\mathrm{C}_{11}\mathrm{H}_7\mathrm{D}_5\mathrm{N}_2\mathrm{S} \cdot \mathrm{HCl}$ , CAS号: 1246819-64-6)，纯度大于 $98\%$ 。

# 5.4 标准溶液的配制

5.4.1 左旋咪唑标准储备液：取盐酸左旋咪唑标准品适量(相当于左旋咪唑约 $10\mathrm{mg}$ )，精密称定，用甲醇溶解并稀释定容于 $10\mathrm{mL}$ 容量瓶，配制成浓度为 $1\mathrm{mg/mL}$ 的左旋咪唑标准储备液， $-18^{\circ}\mathrm{C}$ 下保存，有效期6个月。  
5.4.2 同位素内标储备液：取盐酸左旋咪唑- $\mathrm{D}_5$ 标准品约 $1\mathrm{mg}$ ，精密称定，用甲醇溶解并稀释定容于 $10~\mathrm{mL}$ 容量瓶，配制成浓度为 $100~\mu \mathrm{g / mL}$ 的盐酸左旋咪唑- $\mathrm{D}_5$ 同位素内标储备液， $-18^{\circ}C$ 下保存，有效期6个月。  
5.4.3 左旋咪唑标准中间液：精密移取 $1.0\mathrm{mL}$ 左旋咪唑标准储备溶液，于 $100\mathrm{mL}$ 容量瓶，用甲醇定容，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的左旋咪唑标准中间液， $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 保存，有效期3个月。  
5.4.4 同位素内标中间液：精密移取 $1.0\mathrm{mL}$ 同位素内标储备溶液，于 $10\mathrm{mL}$ 容量瓶，用甲醇定容，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的盐酸左旋咪唑- $\mathbf{D}_5$ 同位素内标中间液， $0^{\circ}C\sim 4^{\circ}C$ 保存，有效期3个月。  
5.4.5 左旋咪唑标准工作液：精密移取 $5.0\mathrm{mL}$ 左旋咪唑标准中间液，于 $100\mathrm{mL}$ 容量瓶，用水稀释，配制成浓度为 $500~\mu \mathrm{g / L}$ 的左旋咪唑标准工作液， $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存，有效期3周。  
5.4.6 同位素内标工作液：精密移取 $1.0\mathrm{mL}$ 同位素内标中间液，于 $100\mathrm{mL}$ 容量瓶，用水稀释，配制成浓度为 $100\mu \mathrm{g} / \mathrm{L}$ 的盐酸左旋咪唑- $\mathbf{D}_5$ 同位素内标工作液， $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 保存，有效期3周。

# 5.5 材料

5.5.1固相萃取(SPE)柱：混合阳离子交换固相萃取柱， $150\mathrm{mg / 6mL}$ ，或相当者。  
5.5.2 滤膜： $0.22\mu \mathrm{m}$ ，有机系。

# 6 仪器和设备

6.1 液相色谱-三重四极杆串联质谱仪，配电喷雾离子源(ESI)。  
6.2 分析天平：感量为 $0.01\mathrm{g}$ 及 $0.00001\mathrm{g}$ 。  
6.3 匀浆机。  
6.4 离心机：转速不低于 $4000\mathrm{r / min}$   
6.5 固相萃取装置：带真空泵。  
6.6 氮吹浓缩仪。  
6.7 涡旋混匀器。  
6.8 振荡器。

# 7 试样制备与保存

# 7.1 试样制备

取适量新鲜或冷藏的空白或供试禽蛋，去壳后混合均匀。

a）取均质的供试样品，作为供试试样；  
b）取均质的空白样品，作为空白试样；  
c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试样 $(5.0 \pm 0.05) \mathrm{~g}$ , 于 $50 \mathrm{~mL}$ 离心管中, 加入同位素内标工作液 $100 \mu \mathrm{L}$ 、碳酸盐缓冲液溶液 $1 \mathrm{~mL}$ , 涡旋混匀。加入乙酸乙酯 $20 \mathrm{~mL}$ 、无水硫酸钠 $5 \mathrm{~g}$ , 涡旋混匀 $1 \mathrm{~min}$ , 振荡 $10 \mathrm{~min}, 4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ 。取上层乙酸乙酯于另一离心管中, 然后加入 $0.2 \mathrm{~mol} / \mathrm{L}$ 盐酸溶液 $10 \mathrm{~mL}$ , 振摇 $5 \mathrm{~min}$ , 以 $4000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 取下层盐酸溶液于另一离心管。乙酸乙酯层中再加入 $0.2 \mathrm{~mol} / \mathrm{L}$ 盐酸 $10 \mathrm{~mL}$ , 重复萃取 1 次, 合并 2 次萃取液, 备用。

# 8.2 净化

依次用甲醇 $5\mathrm{mL}$ 、水 $5\mathrm{mL}$ 、0.2mol/L盐酸 $5\mathrm{mL}$ 对MCX柱进行活化，保持湿润。取备用液过柱，流速控制在 $1\mathrm{mL} / \mathrm{min}\sim 2\mathrm{mL} / \mathrm{min}$ ，依次用水 $5\mathrm{mL}$ 、甲醇 $5\mathrm{mL}$ 淋洗，用 $5\%$ 氨水甲醇溶液 $4\mathrm{mL}$ 洗脱，收集洗脱液，在 $45^{\circ}C$ 水浴中用氮气吹至近干，用乙腈-甲酸铵溶液 $1\mathrm{mL}$ 溶解，过 $0.22\mu \mathrm{m}$ 滤膜后，供液相色谱-三重四极杆串联质谱仪分析。

# 8.3 标准曲线的制备

精密量取 $500\mu \mathrm{g} / \mathrm{L}$ 左旋咪唑标准工作液和 $100\mu \mathrm{g} / \mathrm{L}$ 盐酸左旋咪唑- $\mathrm{D}_5$ 内标工作液适量，用乙腈-甲酸铵溶液稀释，配制成左旋咪唑浓度为 $1.0\mu \mathrm{g} / \mathrm{L},2.3\mu \mathrm{g} / \mathrm{L},5.0\mu \mathrm{g} / \mathrm{L},10.0\mu \mathrm{g} / \mathrm{L},50.0\mu \mathrm{g} / \mathrm{L},100.0\mu \mathrm{g} / \mathrm{L}$ ，盐酸左旋咪唑- $\mathrm{D}_5$ 浓度为 $10\mu \mathrm{g} / \mathrm{L}$ 的系列标准溶液，临用现配，供液相色谱三重四极杆串联质谱仪测定。以左旋咪唑与内标的定量子离子峰面积之比值为纵坐标，左旋咪唑标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.4 测定

# 8.4.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}(50~mm\times 2.1~mm,1.0~\mu m)}$ ，或性能相当者；  
b) 流动相: $0.01 \mathrm{~mol} / \mathrm{L}$ 甲酸铵水溶液 (含 $0.1 \%$ 甲酸) 乙腈 (95: $10, V / V$   
c) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ ;   
d） 柱温： $40^{\circ}C$   
e）进样量： $5\mu \mathrm{I}_{\circ}$

# 8.4.2 质谱参考条件

a）离子源：电喷雾离子源（ESI）；  
b）扫描方式：正离子扫描；  
c）检测方式：多反应监测(MRM)；  
d）雾化气、干燥气、碰撞气均为高纯氮气或其他合适气体，使用前应调节各气体流量以使质谱灵敏度达到检测要求；  
e）干燥气温度： $300^{\circ}C$   
f) 雾化气压力: $275.8 \mathrm{kPa}$ ;   
g）干燥气流量： $10\mathrm{L / min}$   
h）毛细管电压： $4000\mathrm{V}$   
i）定量离子对、定性离子对的碎裂电压(FP)、碰撞能量(CE)见表1。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在相同测试条件下，试样溶液中左旋咪唑与其内标(左旋咪唑- $\mathrm{D}_5$ )的保留时间之比与标准溶液中左旋咪唑与其内标(左旋咪唑- $\mathrm{D}_5$ )的保留时间之比偏差在 $1\%$ 以内；且检测到的相对离子丰度应与浓度相当的校正标准溶液相对离子丰度一致；其允许偏差为 $\pm 40\%$ 。

# 8.4.3.2 定量测定

取试样溶液和相应的标准溶液，做单点或多点校准，按内标法定量。试样溶液及标准溶液中左旋咪唑与左旋咪唑- $\mathrm{D}_5$ 的响应值均应在仪器检测的线性范围内。对于试料中左旋咪唑残留量超过仪器测定线性范围的，在提取时根据药物浓度相应增加内标工作液的添加量，使试样溶液稀释后左旋咪唑的响应在仪器线性范围内，对应盐酸左旋咪唑- $\mathrm{D}_5$ 的浓度与标准曲线制备中的盐酸左旋咪唑- $\mathrm{D}_5$ 浓度一致。在上述色谱-质谱条件下，左旋咪唑和左旋咪唑- $\mathrm{D}_5$ 标准溶液的特征离子质量色谱图见附录A。

# 8.5 空白试验

取空白试样，除不加药物外，采用完全相同的操作步骤进行平行操作。

# 9 结果计算

试样中左旋咪唑的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times A _ {i s} ^ {\prime} \times C _ {s} \times C _ {i s} \times V}{A _ {i s} \times A _ {s} \times C _ {i s} ^ {\prime} \times m} \tag {1}
$$

式中：

$X$ ——试样中左旋咪唑残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——试样溶液中左旋咪唑的峰面积；

$A_{is}^{\prime}$ ——标准溶液中内标的峰面积；

$C_{s}$ ——标准溶液中左旋咪唑浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{i s}$ ——试样溶液中内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{is}$ ——试样溶液中内标的峰面积；

$A_{s}$ 标准溶液中左旋咪唑的峰面积；

$C_{is}^{\prime}$ ——标准溶液中内标浓度的数值，单位为微克每升 $(\mu \mathrm{g / L})$

$m$ 供试试样质量的数值，单位为克 $(\mathbf{g})$ 。

注：计算结果以平行测定结果的算术平均值表示，含量不小于 $1\mu \mathrm{g} / \mathrm{kg}$ 的保留3位有效数字， $1\mu \mathrm{g} / \mathrm{kg}$ 以下保留至小数点后2位。

# 10 检测方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法禽蛋中左旋咪唑的检出限为 $0.1\mu \mathrm{g / kg}$ ，定量限为 $0.5\mu \mathrm{g / kg}$ 。

# 10.2 正确度

在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim 5.0\mu \mathrm{g} / \mathrm{kg}$ 添加水平，禽蛋中左旋咪唑的回收率为 $80\% \sim 110\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

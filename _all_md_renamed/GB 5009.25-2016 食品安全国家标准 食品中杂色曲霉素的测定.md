# 前言

本标准代替GB/T5009.25—2003《植物性食品中杂色曲霉素的测定》和SN/T2483—2010《进出口粮谷中柄曲霉素含量检测方法液相色谱法》。

本标准与GB/T5009.25—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中杂色曲霉素的测定”；  
——增加了液相色谱-串联质谱法；  
——增加了液相色谱法。

# 食品安全国家标准

# 食品中杂色曲霉素的测定

# 1 范围

本标准规定了液相色谱-串联质谱法和高效液相色谱法测定杂色曲霉素的测定方法。

本标准适用于大米、玉米、小麦、黄豆及花生中杂色曲霉素的测定。

# 第一法 液相色谱-串联质谱法

# 2 原理

试样中的杂色曲霉素乙腈-水溶液提取，经涡旋、超声、离心，取上清液经稀释，通过固相萃取柱或免疫亲和柱净化、浓缩、甲醇-水溶液定容、微孔滤膜过滤，液相色谱分离，电喷雾离子源离子化，多反应离子监测检测，同位素内标法定量。

# 3 试剂和材料

除非另有说明，本方法使用的试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.3 氯化钠 $(\mathrm{NaCl})$ 。  
3.1.4 磷酸氢二钠 $\mathrm{(Na_2HPO_4)}$   
3.1.5 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
3.1.6 盐酸。  
3.1.7 氯化钾（KCl）。

# 3.2 试剂配制

3.2.1 乙腈-水溶液 $(80 + 20)$ ：取 $800~\mathrm{mL}$ 乙腈，加 $200~\mathrm{mL}$ 水，混匀。  
3.2.2 乙腈-水溶液 $(40 + 60)$ ：取 $400~\mathrm{mL}$ 乙腈，加 $600~\mathrm{mL}$ 水，混匀。  
3.2.3 甲醇-水溶液 $(40 + 60)$ ：取 $400~\mathrm{mL}$ 甲醇，加 $600~\mathrm{mL}$ 水，混匀。  
3.2.4 甲醇-水溶液 $(70 + 30)$ ：取 $700~\mathrm{mL}$ 甲醇，加 $300~\mathrm{mL}$ 水，混匀。  
3.2.5 磷酸盐缓冲溶液(以下简称PBS)：称取 $8.0\mathrm{g}$ 氯化钠， $1.2\mathrm{g}$ 磷酸氢二钠（或 $2.92\mathrm{g}$ 十二水磷酸氢二钠）， $0.2\mathrm{g}$ 磷酸二氢钾， $0.2\mathrm{g}$ 氯化钾，用 $900\mathrm{mL}$ 水溶解，用盐酸调节 $\mathrm{pH}$ 至7.4，用水定容至 $1000\mathrm{mL}$ 。

# 3.3 标准品

3.3.1 杂色曲霉素标准品 $\left(\mathrm{C}_{18} \mathrm{H}_{12} \mathrm{O}_{6}\right)$ , CAS 号: 10048-13-2): 纯度 $\geqslant 99\%$ , 或经国家认证并授予标准物质证书的标准物质。  
3.3.2 ${}^{13}\mathrm{C}_{18}$ -杂色曲霉素同位素内标： $25\mu \mathrm{g / mL}$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 标准储备溶液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：准确称取杂色曲霉素标准品 $1.00\mathrm{mg}$ （准确至 $0.01\mathrm{mg}$ ），用甲醇溶解并定容至 $10~\mathrm{mL}$ 。溶液转移至试剂瓶中后，密封后 $-20^{\circ}\mathrm{C}$ 下保存，保存期6个月。  
3.4.2 标准工作液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：准确移取 $1.00~\mathrm{mL}$ 的杂色曲霉素标准储备溶液至 $100~\mathrm{mL}$ 容量瓶中，用甲醇定容。 $-20^{\circ}\mathrm{C}$ 下保存，保存期3个月。  
3.4.3 同位素内标工作液 $(1.0\mu \mathrm{g} / \mathrm{mL})$ ：准确移取杂色曲霉素同位素内标 $(25~\mu \mathrm{g / mL})0.40~\mathrm{mL}$ 至 $10~\mathrm{mL}$ 容量瓶中，用甲醇定容。一 $20^{\circ}C$ 下保存，保存期3个月。  
3.4.4 标准系列工作溶液：准确移取标准工作液适量至 $5\mathrm{mL}$ 容量瓶中，加入 $50~\mu \mathrm{L}1.0~\mu \mathrm{g / mL}$ 的同位素内标工作液，用甲醇-水溶液（ $70 + 30$ ）定容至刻度（含杂色曲霉素浓度为 $1\mathrm{ng / mL},2\mathrm{ng / mL},$ $5\mathrm{ng / mL},10\mathrm{ng / mL},20\mathrm{ng / mL},30\mathrm{ng / mL},40\mathrm{ng / mL},50\mathrm{ng / mL}$ 系列标准溶液），临用前配制。

# 3.5 材料

3.5.1 免疫亲和柱：柱容量 $\geq 600\mathrm{ng}$ （柱容量验证方法参见B.2）。  
3.5.2固相萃取柱：N-乙烯吡咯烷酮和二乙烯基苯共聚物填料柱( $200\mathrm{mg} / 6\mathrm{mL}$ )，或相当者。使用前分别用 $5\mathrm{mL}$ 甲醇和 $5\mathrm{mL}$ 水活化。  
3.5.3 微孔滤膜： $0.22\mu \mathrm{m}$

# 4 仪器和设备

4.1 液相色谱-串联质谱仪：配电喷雾离子源。  
4.2 高速粉碎机  
4.3 涡旋混合器。  
4.4 超声波发生器。  
4.5 天平：感量为 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$   
4.6 离心机：转速 $\geqslant 6000\mathrm{r / min}$   
4.7 固相萃取装置（带真空泵）。  
4.8 氮吹仪。  
4.9 试验筛： $1\mathrm{mm}\sim 2\mathrm{mm}$ 孔径。

# 5 分析步骤

# 5.1 试样制备

样品用高速粉碎机将其粉碎后过 $1\mathrm{mm}\sim 2\mathrm{mm}$ 孔径试验筛，混合均匀后取试样 $100\mathrm{g}$ 用于检测。

# 5.2 试样提取

称取 $5\mathrm{g}$ 均质试样(精确至 $0.01\mathrm{g}$ ) 至 $50~\mathrm{mL}$ 离心管中(花生和黄豆样品：称取 $2\mathrm{g}$ 均质试样)，加入

$100\mu \mathrm{L}$ 同位素内标工作液，加入 $20.0~\mathrm{mL}$ 乙腈-水溶液 $(80 + 20)$ ，涡旋混匀后超声 $10\mathrm{min}$ ，在 $6000\mathrm{r / min}$ 下离心 $10\mathrm{min}$ ，取上清液备用。

# 5.3 净化

# 5.3.1 固相萃取柱净化

准确移取 $2.0\mathrm{mL}$ 上述上清液用水稀释至 $8\mathrm{mL}$ 待上样。

将上样液转移至活化好的固相萃取柱中，控制样液以约 $3\mathrm{mL / min}$ 的速度稳定下滴。上样完毕后，依次加入 $5\mathrm{mL}$ 的乙腈-水溶液 $(40 + 60)$ 、 $5\mathrm{mL}$ 的甲醇-水溶液 $(40 + 60)$ 淋洗。待淋洗结束后，用真空泵抽干固相萃取柱，加入 $6\mathrm{mL}$ 乙腈洗脱，控制流速为约 $3\mathrm{mL / min}$ ，用真空泵抽干固相萃取柱，收集洗脱液。在 $60^{\circ}\mathrm{C}$ 下用氮气缓缓吹至干，用甲醇-水溶液 $(70 + 30)$ 定容至 $1.0\mathrm{mL}$ ，涡旋 $30~s$ 溶解残留物， $0.22\mu \mathrm{m}$ 滤膜过滤，收集滤液于进样瓶中以备进样。按同一操作方法做空白试验。

# 5.3.2 免疫亲和柱净化

准确移取 $2\mathrm{mL}$ 上述上清液，加入 $28~\mathrm{mL}$ PBS混匀。

将 $50~\mathrm{mL}$ 一次性注射器筒与亲和柱的顶部相连。将上述样液移至注射器筒中，调节下滴速度，控制样液以约 $3\mathrm{mL / min}$ 的速度稳定下滴。待样液滴完后，往注射器筒内依次加入 $10~\mathrm{mL}$ PBS和 $10~\mathrm{mL}$ 水，以稳定流速淋洗免疫亲和柱。待水滴完后，用真空泵抽干亲和柱。在亲和柱下部放置 $10~\mathrm{mL}$ 刻度试管，取下 $50~\mathrm{mL}$ 的注射器筒，加入 $2\mathrm{mL}$ 乙腈洗脱亲和柱，控制约 $3\mathrm{mL / min}$ 的自然下滴速度，收集全部洗脱液至刻度试管中，用真空泵抽干亲和柱。在 $60^{\circ}C$ 下用氮气缓缓地将洗脱液吹至干，用甲醇-水溶液 $(70 + 30)$ 定容至 $1.0~\mathrm{mL}$ ，涡旋 $30~\mathrm{s}$ 溶解残留物， $0.22\mu \mathrm{m}$ 滤膜过滤，收集滤液于进样瓶中以备进样。按同一操作方法做空白试验。

注：可根据实验室实际情况，选择上述净化方法中的一种净化方法即可。

# 5.4 仪器参考条件

# 5.4.1 色谱参考条件

a）液相色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $100\mathrm{mm}$ ，内径 $2.1\mathrm{mm}$ ，粒径 $1.8\mu \mathrm{m}$ ，或相当色谱柱；  
b）流动相：A相：水，B相：甲醇；

d) 流速: $0.2 \mathrm{~mL} / \mathrm{min}$ ;   
e） 色谱柱柱温： $40^{\circ}C$   
f) 进样量: $10 \mu \mathrm{L}$ 。

# 5.5 标准曲线制作

将标准系列工作溶液按浓度由低到高注入液相色谱一串联质谱仪中，测得相应色谱峰的峰面积，以标准系列工作溶液中杂色曲霉素的浓度为横坐标，杂色曲霉素色谱峰的峰面积与同位素内标色谱峰的峰面积的比值为纵坐标，绘制标准曲线。

# 5.6 试样溶液的测定

将试样溶液注入液相色谱一串联质谱仪中，测得相应色谱峰的峰面积，根据标准曲线得到试样溶液中杂色曲霉素的浓度。如试样溶液中杂色曲霉素的浓度超出线性范围，则需适当减少取样量按5.2、5.3处理试样后重新测定。

# 5.7 定性

试样中目标化合物色谱峰的保留时间与相应标准色谱峰的保留时间相比较，变化范围在 $\pm 2.5\%$ 之内。

待测化合物定性离子色谱峰的信噪比 $\geq 3$ ，定量离子色谱峰的信噪比 $\geq 10$ 。

每种化合物的质谱定性离子必须出现，至少应包括一个母离子和两个子离子，而且同一检测批次，对同一化合物，样品中目标化合物的两个子离子的相对丰度比与浓度相当的标准溶液相比，其允许偏差不超过表1规定的范围。

# 6 分析结果的表述

试样中杂色曲霉素的含量按式(1)计算：

$$
X = \frac {\rho \times V}{m} \times f \quad \dots \tag {1}
$$

式中：

$X$ ——试样中杂色曲霉素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中杂色曲霉素的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——最终定容体积，单位毫升 $(\mathrm{mL})$ ；

$m$ ——试样的称样量，单位克 $(\mathrm{g})$ ；

$f$ ——稀释倍数 $(f = 10)$

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

称取大米、玉米及小麦试样 $5\mathrm{g}$ 时，其检出限为 $0.6~\mu \mathrm{g / kg}$ ，定量限为 $2\mu \mathrm{g / kg}$ ；称取黄豆及花生试样 $2\mathrm{g}$ 时，其检出限为 $1.5~\mu \mathrm{g / kg}$ ，定量限为 $5\mu \mathrm{g / kg}$ 。

# 第二法 液相色谱法

# 9 原理

样品中的杂色曲霉素用乙腈-水溶液提取，经均质、涡旋、超声、离心等处理，取上清液用磷酸盐缓冲液稀释，免疫亲和柱净化、洗脱，氮气吹干浓缩、流动相定容、微孔滤膜过滤，液相色谱分离紫外检测器检测。外标法定量。

# 10 试剂和溶液

除非另有说明，本方法使用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 10.1 试剂和标准样品

10.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
10.1.3 氯化钠 $(\mathrm{NaCl})$ 。  
10.1.4 磷酸氢二钠 $\mathrm{(Na_2HPO_4)}$   
10.1.5 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
10.1.6 盐酸。  
10.1.7 氯化钾(KCl)。

# 10.2 试剂配制

10.2.3 磷酸盐缓冲溶液(以下简称PBS)：称取 $8.0\mathrm{g}$ 氯化钠， $1.2\mathrm{g}$ 磷酸氢二钠（或 $2.92\mathrm{g}$ 十二水磷酸氢二钠）， $0.2\mathrm{g}$ 磷酸二氢钾， $0.2\mathrm{g}$ 氯化钾，用 $900\mathrm{mL}$ 水溶解，用盐酸调节 $\mathrm{pH}$ 至7.4，用水定容至 $1000\mathrm{mL}$ 。

# 10.3 标准品

# 10.4 标准溶液配制

10.4.2 标准工作液 $(1\mu \mathrm{g / mL})$ ：准确移取 $100~\mu \mathrm{L}$ 的杂色曲霉素标准储备溶液至 $10~\mathrm{mL}$ 容量瓶中，用乙腈定容。此溶液浓度为 $1\mu \mathrm{g / mL}$ 。一 $20^{\circ}C$ 下保存，保存期3个月。  
10.4.3 标准系列工作溶液：准确移取标准工作液适量至 $5\mathrm{mL}$ 容量瓶中，用乙腈-水溶液 $(50 + 50)$ 定容至刻度（含杂色曲霉素浓度为 $5\mathrm{ng / mL},10\mathrm{ng / mL},25\mathrm{ng / mL},50\mathrm{ng / mL},75\mathrm{ng / mL},100\mathrm{ng / mL}$ 系列标准溶液），临用前配制。

15 min);   
d) 流速: $0.8 \mathrm{~mL} / \mathrm{min}$ ;   
e）色谱柱柱温： $40^{\circ}\mathrm{C}$   
f) 进样量: $100 \mu \mathrm{L}$ ;   
g）紫外检测器条件：检测波长为 $325~\mathrm{nm}$

# 12.5 标准曲线的制作

将标准系列工作溶液按浓度由低到高注入液相色谱仪进行测定，测得相应色谱峰的峰面积，以标准系列工作溶液中杂色曲霉素的浓度为横坐标，杂色曲霉素色谱峰的峰面积为纵坐标，绘制标准曲线。

# 12.6 试样溶液的测定

将试样溶液注入液相色谱仪进行测定，测得相应色谱峰的峰面积，根据标准曲线得到试样溶液中杂色曲霉素的浓度。

# 13 分析结果的表述

试样中杂色曲霉素的含量按式(2)计算：

$$
X = \frac {\rho \times V}{m} \times f \quad \dots \tag {2}
$$

式中：

$X$ ——试样中杂色曲霉素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——根据试样中杂色曲霉素的色谱峰面积，经计算所得的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——最终定容体积，单位毫升 $(\mathrm{mL})$ ；

$m$ ——试样的称样量，单位克 $(\mathrm{g})$ ；

$f$ ——稀释倍数 $(f = 10)$ 。

计算结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 15 其他

称取大米、玉米、小麦、黄豆及花生 $5\mathrm{g}$ 时，其检出限为 $6\mu \mathrm{g / kg}$ ，定量限为 $20~\mu \mathrm{g / kg}$ 。

# 第三法 薄层色谱法

# 16 原理

试样中的杂色曲霉素经提取、净化、浓缩、薄层展开后，用三氯化铝显色，再经加热产生一种在紫外光下显示黄色荧光的物质，根据其在薄层上显示的荧光最低检出量来测定样品中杂色曲霉素的含量。

# 19 分析步骤

# 19.1 提取

大米、玉米、小麦、黄豆及花生：称取 $20.00\mathrm{g}$ 过 $0.85\mathrm{mm}$ 筛孔的大米、玉米、小麦及黄豆粉碎样品（花生粉碎样品过 $2\mathrm{mm}$ 的筛孔），置于具塞锥形瓶中，加 $80~\mathrm{mL}$ 甲醇-氯化钠溶液 $(90 + 10)$ ，振荡 $30\mathrm{min}$ ，过滤。收集样品液 $40~\mathrm{mL}$ （黄豆、花生样品则取 $20~\mathrm{mL}$ ，加入 $20~\mathrm{mL}$ 提取剂），移入 $250~\mathrm{mL}$ 分液漏斗中，再加入 $25~\mathrm{mL}$ 氯化钠溶液（使甲醇与水之体积比为 $55 + 45$ ）和 $25~\mathrm{mL}$ 石油醚，振摇 $2\mathrm{min}$ ，静置分层。上层石油醚溶液置锥形瓶中，下层溶液仍移入原分液漏斗中，再用 $25~\mathrm{mL}$ 石油醚提取一次。最后将两次上层的石油醚溶液合并，加入 $25~\mathrm{mL}$ 甲醇-氯化钠溶液 $(55 + 45)$ [黄豆、花生样品则加入 $25~\mathrm{mL}$ 甲醇-氯化钠溶液 $(70 + 30)$ ，振摇 $30~\mathrm{s}$ ]，将下层并于原甲醇水层中，重复用甲醇-氯化钠溶液 $(55 + 45)$ 提

取两次[黄豆、花生样品重复用甲醇-氯化钠溶液 $(70 + 30)$ 提取1次]，以提取该层的杂色曲霉素。下层溶液合并后加 $30~\mathrm{mL}$ 三氯甲烷(黄豆、花生样品除加三氯甲烷外，再加 $13~\mathrm{mL}$ 氯化钠溶液，使甲醇与水的体积比为 $55 + 45$ )，振摇 $2\mathrm{min}$ ，静置。待上层混浊液有部分澄清时，即可将下层溶液经放有约 $10\mathrm{g}$ 无水硫酸钠的定量慢速滤纸过滤于蒸发皿中。于分液漏斗中再加 $10~\mathrm{mL}$ 三氯甲烷，重复提取1次，将该下层溶液和用少量三氯甲烷洗滤器的洗液一并放入蒸发皿中。将蒸发皿放置于 $65^{\circ}\mathrm{C}$ 水浴上挥干，然后再在冰浴上放置 $2\mathrm{min} \sim 3\mathrm{min}$ ，加 $1.0~\mathrm{mL}$ 苯将残留物充分混匀，置入小试管中。或将以上蒸发皿中残留物用三氯甲烷移于浓缩管中，于 $65^{\circ}\mathrm{C}$ 用减压吹气法浓缩至干，加入 $1.0~\mathrm{mL}$ 苯，供色谱测定，此 $1\mathrm{mL}$ 大米、玉米及小麦样液各相当于 $10\mathrm{g}$ 样品， $1\mathrm{mL}$ 黄豆与花生样液则各相当于 $5\mathrm{g}$ 样品。

# 19.2 测定

# 19.2.1 单向展开法

# 19.2.1.1 薄层板的制备

称取约 $5\mathrm{g}$ 硅胶 $\mathrm{G}$ ，加相当于硅胶量2倍～3倍左右的水，用力研磨 $1\mathrm{min} \sim 2\mathrm{min}$ 至成糊状后立即倒于涂布器内，推成 $10\mathrm{cm} \times 10\mathrm{cm}$ ，厚度 $0.3\mathrm{mm}$ 的薄层板五块。在空气中干燥约 $15\mathrm{min}$ 后，在 $100^{\circ}\mathrm{C}$ 活化 $2\mathrm{h}$ ，取出，放干燥器中保存。一般可保存 $2\mathrm{d} \sim 3\mathrm{d}$ ，若放置时间较长，可再活化后使用。

# 19.2.1.2 点样

取两块 $10\mathrm{cm}\times 10\mathrm{cm}$ 薄层板，在距板下端各 $0.8\mathrm{cm}\sim 1\mathrm{cm}$ 基线上滴加标准使用液与样液如下：距左边缘 $0.8\mathrm{cm}\sim 1\mathrm{cm}$ 处各滴加 $10~\mu \mathrm{L}$ 标准使用液 $(0.4\mu \mathrm{g / mL})$ ，在距左边缘 $4\mathrm{cm}$ 处各滴加 $80~\mu \mathrm{L}$ 样液（黄豆、花生样品为 $40~\mu \mathrm{L}$ )，然后在第二块板的样液点上加滴 $10~\mu \mathrm{L}$ 标准使用液 $(0.4\mu \mathrm{g / mL})$ ，在滴加样液时可用吹风机冷风边吹边加。

# 19.2.1.3 展开

19.2.1.3.1 横向展开：展开剂是乙醚-正己烷-苯-三氯甲烷-甲酸 $(3 + 9 + 1.5 + 1.5 + 0.6)15.6\mathrm{mL}$ （由于此混合液不成一相，每一展开槽的用量须单独配制）。用前充分摇匀，一并倒入槽内使用。将靠近标准点的一边，放入槽内展至 $9\mathrm{cm}$ 左右取出挥干。

19.2.1.3.1 纵向展开：展开剂为苯-甲醇-冰乙酸 $(90 + 8 + 2$ 或 $92.5 + 6 + 1.5)15\mathrm{mL}$ 。将靠近标准点与样液点的一边放入槽内，展开 $9\mathrm{cm}$ 左右取出挥干。

# 19.2.1.4 显荧光

在薄层板上喷三氯化铝-乙醇溶液 $(200\mathrm{g / L})$ ，置 $80^{\circ}\mathrm{C}$ 加热 $10\mathrm{min}$ ，立即在紫外光灯（波长 $365~\mathrm{nm}$ 下观察结果，待薄层板冷却后再薄薄的喷第2次(不需加热)，可直接观察结果。

# 19.2.1.5 观察与评定结果

在紫外光灯下观察，若第2板的第2点在标准点的相应处出现最低检出量，而在第1板的相同位置上未出现荧光点，则样品中杂色曲霉素含量在 $5\mu \mathrm{g / kg}$ （黄豆、花生样品为 $20~\mu \mathrm{g / kg})$ ；若出现荧光强度与标准点的最低检出量的荧光强度相等，而且此荧光点又同第二板样液的标准点相重叠，则样品中杂色曲霉素含量为 $5\mu \mathrm{g / kg}$ （黄豆、花生样品为 $20~\mu \mathrm{g / kg})$ ；若出现荧光强度比标准点的最低检出量强，则根据其荧光强度估计减少滴加微升数，或将样液稀释后再滴加不同微升数，直至样液点的荧光强度与最低检出量的荧光强度一致为止。在喷三氯化铝第1、2次后分别进行观察评定，两次结果应一致。若结果为阳性，则将薄层板放暗处 $10\mathrm{min}$ ，再观察一次，如样品仍为阳性，进一步作确证试验，即在距薄层板 $(10\mathrm{cm}\times 18.5\mathrm{cm})$ 下端 $3\mathrm{cm}$ 的基线上滴加一个点的 $10~\mu \mathrm{L}$ 标准使用液 $(0.4~\mu \mathrm{g / mL})$ 与3个点的样液，

每点 $16 \mu \mathrm{L}$ 。在样液的一个点上再加滴 $10 \mu \mathrm{L}$ 标准使用液 $(0.4 \mu \mathrm{g} / \mathrm{mL})$ ，另一点上再加滴 $10 \mu \mathrm{L}$ 标准使用液 $(1 \mu \mathrm{g} / \mathrm{mL})$ 。于各点上再加一小滴三氟乙酸，放暗处反应 $10 \mathrm{~min}$ ，热风吹 $5 \mathrm{~min}$ ，使薄层板上的温度不高于 $40^{\circ} \mathrm{C}$ ，用冰乙酸-苯 $(10 + 90)$ 展开 1 次～2 次，直至杂色曲霉素衍生物与杂质分开为止。展开时要避光，以下显荧光步骤同 19.2.1.4。最后将板在紫外光灯下观察，如样液为阳性，应产生与杂色曲霉素标准重叠的衍生物。确证法的最低检出量：大米、玉米、小麦均为 $25 \mu \mathrm{g} / \mathrm{kg}$ （黄豆、花生样品均为 $50 \mu \mathrm{g} / \mathrm{kg}$ )。

# 19.3 分析结果的表述

试样中杂色曲霉素的含量按式(3)计算：

$$
X = 0. 0 0 4 \times \frac {V _ {1} \times D}{V _ {2}} \times \frac {1 0 0 0}{m} \tag {3}
$$

式中：

$X$ ——杂色曲霉素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

0.004 杂色曲霉素的最低检出量，单位为微克 $(\mu \mathrm{g})$

$V_{1}$ ——样液浓缩后体积，单位为毫升 $(\mathrm{mL})$ ；

$D$ ——浓缩样液的总稀释倍数；

$V_{2}$ ——出现最低荧光时滴加样液的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——浓缩样液中所相当的试样质量，单位为克 $(\mathrm{g})$ 。

结果表示到测定值的整数位。

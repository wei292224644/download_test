# 附录A

# 检验方法

# A.1 安全提示

本标准检验方法中使用的部分试剂具有毒性或腐蚀性，操作时须小心谨慎。如溅到皮肤上，应立即用水冲洗，严重者应立即治疗。在使用挥发性酸时，需在通风橱中进行。

# A.2 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。本标准试验中所需标准溶液、杂质标准溶液、制剂和制品，在没有注明其他要求时均按GB/T601、GB/T602、GB/T603的规定制备。本试验所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 盐酸溶液： $1 + 1$   
A.3.1.2 盐酸溶液： $0.1\mathrm{mol} / \mathrm{L}$   
A.3.1.3 氢氧化钠溶液： $1\mathrm{mol} / \mathrm{L}$   
A.3.1.4 溴酚蓝溶液：称取 $100\mathrm{mg}$ 溴酚蓝，溶于 $100~\mathrm{mL}$ 乙醇 $(1 + 2)$ 中，必要时过滤。  
A.3.1.5 硫酸锌溶液： $125\mathrm{g / L}$

# A.3.2 仪器与设备

酸度计：精度 $\mathrm{pH}\pm 0.01$

# A.3.3 分析步骤

# A.3.3.1 铁离子鉴别

称取 $0.50\mathrm{g}$ 试样，置于 $100~\mathrm{mL}$ 烧杯中，加 $5\mathrm{mL}$ 盐酸溶液 $(1 + 1)$ 溶解试样，加过量的 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液，应生成红棕色沉淀。

# A.3.3.2 焦磷酸根离子鉴别

将A.3.3.1试验溶液放置数分钟后过滤，弃去最初的数毫升滤液，在 $5\mathrm{mL}$ 透明滤液中加1滴溴酚蓝溶液后，用 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液滴定至绿色。加 $10~\mathrm{mL}$ 硫酸锌溶液后，用 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液将 $\mathrm{pH}$ 调整至3.8，应生成白色沉淀。

# A.4 铁含量(Fe)的测定

# A.4.1 第一法 重铬酸钾法

# A.4.1.1 方法提要

试样用盐酸溶液溶解，用氯化亚锡还原，再用氯化汞氧化过量的氯化亚锡，以二苯胺磺酸钠为指示剂，用重铬酸钾标准溶液滴定，以此测定试样中铁含量。

# A.4.1.2 试剂和材料

A.4.1.2.1 盐酸溶液： $1 + 2$

A.4.1.2.2 氯化亚锡溶液：称取优级纯氯化亚锡 $(\mathrm{SnCl}_2\cdot 2\mathrm{H}_2\mathrm{O})40\mathrm{g}$ 溶于 $100~\mathrm{mL}$ 浓盐酸中。

A.4.1.2.3 硫酸-磷酸溶液: 将 $75 \mathrm{~mL}$ 浓硫酸缓慢加入至 $300 \mathrm{~mL}$ 水中, 冷却, 加入 $75 \mathrm{~mL}$ 磷酸, 用水定容至 $500 \mathrm{~mL}$ 。

A.4.1.2.4 氯化汞饱和溶液：称取 $64\mathrm{g}$ 氯化汞，溶于 $986~\mathrm{mL}$ 水中。

A.4.1.2.5 二苯胺磺酸钠溶液：称取二苯胺磺酸钠 $300\mathrm{mg}$ ，溶于 $100~\mathrm{mL}$ 水中。

A.4.1.2.6 重铬酸钾标准滴定溶液: $c\left(\frac{1}{6} \mathrm{~K}_{2} \mathrm{Cr}_{2} \mathrm{O}_{7}\right) = 0.1 \mathrm{~mol} / \mathrm{L}$ 。

注意：氯化汞有剧毒！

# A.4.1.3 分析步骤

精确称取试样 $3.5\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），置 $250~\mathrm{mL}$ 烧杯中，加 $75~\mathrm{mL}$ 盐酸溶液 $(1 + 2)$ 溶解试样，加热至沸腾，沸腾约 $5\mathrm{min}$ 。冷却移入 $100~\mathrm{mL}$ 容量瓶，用盐酸溶液 $(1 + 2)$ 定容、混匀，量取该溶液 $25~\mathrm{mL}$ 置 $500~\mathrm{mL}$ 锥形瓶中，加入 $100~\mathrm{mL}$ 盐酸溶液 $(1 + 2)$ ，再煮沸 $5\mathrm{min}$ ，边搅拌边滴加氯化亚锡溶液，直至黄色消失(铁正好还原)。过量加入2滴的氯化亚锡溶液，加 $50~\mathrm{mL}$ 水稀释，冷却至室温。在剧烈搅拌下加入 $15~\mathrm{mL}$ 氯化汞饱和溶液，放置 $5\mathrm{min}$ 后加入 $15~\mathrm{mL}$ 硫酸-磷酸溶液。摇匀，加入 $0.5~\mathrm{mL}$ 二苯胺磺酸钠溶液，用 $0.1\mathrm{mol} / \mathrm{L}$ 重铬酸钾标准滴定溶液滴定至红紫色。

在测定的同时，按与测定相同的步骤，对不加试样而使用相同数量的试剂溶液做空白试验。

# A.4.1.4 结果计算

铁含量(Fe)的质量分数 $\omega_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {c \times \left(V _ {1} - V _ {0}\right) \times M \times 4}{m \times 1000} \times 100 \% \dots \tag{A.1}
$$

式中：

$c$ ——重铬酸钾标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{1}$ ——试样消耗重铬酸钾标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗重铬酸钾标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$M$ ——铁的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{Fe}) = 55.85\mathrm{g / mol}]$

4 —稀释倍数；

$m$ ——试样的质量，单位为克(g)；

1000——换算因子。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.4.2 第二法 碘量法

# A.4.2.1 方法提要

在酸性介质中，试样中的三价铁与碘化钾作用生成游离碘，用硫代硫酸钠标准滴定溶液滴定，以此计算试样中的铁含量。

# A.4.2.2 试剂和材料

A.4.2.2.1 盐酸溶液： $1 + 2$

A.4.2.2.2 碘化钾。

A.4.2.2.3 硫代硫酸钠标准滴定溶液： $c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol / L}$

A.4.2.2.4 淀粉指示液： $5\mathrm{g / L}$

# A.4.2.3 分析步骤

称取约 $3.5\mathrm{g}$ 试样(精确至 $0.0001\mathrm{g}$ )，置 $250~\mathrm{mL}$ 烧杯中，加 $75~\mathrm{mL}$ 盐酸溶液 $(1 + 2)$ 溶解试样，加热至沸腾，沸腾约 $5\mathrm{min}$ 。冷却移入 $100~\mathrm{mL}$ 容量瓶，用盐酸溶液 $(1 + 2)$ 定容、混匀，量取该溶液 $15~\mathrm{mL}$ 于碘量瓶中，加 $3\mathrm{g}$ 碘化钾，于暗处放置 $5\mathrm{min}$ 。用硫代硫酸钠标准滴定溶液滴定，至淡黄色时，加 $0.3~\mathrm{mL}$ 淀粉指示液，继续滴定至蓝色消失为终点。

在测定的同时，按与测定相同的步骤，对不加试样而使用相同数量的试剂溶液做空白试验。

# A.4.2.4 结果计算

铁含量(Fe)的质量分数 $\varpi_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {c \times \left(V _ {1} - V _ {0}\right) \times M}{m \times \frac {15}{100} \times 1000} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$c$ ——硫代硫酸钠标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{1}$ ——试样消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白试验消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$M$ ——铁的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{Fe}) = 55.85\mathrm{g / mol}]$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

15 量取试样溶液的体积，单位为毫升 $(\mathrm{mL})$

100——容量瓶的容量，单位为毫升 $(\mathrm{mL})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 灼烧减量的测定

# A.5.1 仪器和设备

高温炉：可控温度 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 。

# A.5.2 分析步骤

称取约 $1\mathrm{g}$ 试样（精确至 $0.0001\mathrm{g}$ ），置于预先在 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 下灼烧至质量恒定的瓷坩埚中，于 $800^{\circ}\mathrm{C} \pm 25^{\circ}\mathrm{C}$ 高温炉中灼烧 $1\mathrm{h}$ 。放入干燥器中冷却至室温，称量。重复灼烧至前后两次称量相差不超过 $0.5\mathrm{mg}$ 为恒重。

# A.5.3 结果计算

灼烧减量的质量分数 $\varpi_{3}$ ，按式（A.3）计算：

$$
w _ {3} = \frac {m - m _ {1}}{m} \times 100 \% \tag{A.3}
$$

式中：

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——灼烧后试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。两次平行测定结果的绝对差值不大于 $0.2\%$ 。

# A.6 铅(Pb)的测定

# A.6.1 方法提要

试样经处理后，铅离子经三辛基氧化膦-甲基异丁基酮溶液萃取分离，样品溶液导入原子吸收分光光度计中，火焰原子化后，吸收 $283.3\mathrm{nm}$ 共振线，其吸收量与铅含量成正比。

# A.6.2 试剂和材料

A.6.2.1 盐酸溶液： $3 + 1$

A.6.2.2 抗坏血酸溶液： $100\mathrm{mg / mL}$

A.6.2.3 碘化钠溶液： $192.5\mathrm{mg / mL}$

A.6.2.4 抗坏血酸-碘化钠溶液：量取 $100~\mathrm{mL}$ 抗坏血酸溶液和 $100~\mathrm{mL}$ 碘化钠溶液，混合均匀。

A.6.2.5 三辛基氧化膦-甲基异丁基酮溶液: 称取 $5.00 \mathrm{~g}$ 三辛基氧化膦, 用甲基异丁基酮溶解, 定容至 $100 \mathrm{~mL}$ , 配制成 $50 \mathrm{mg} / \mathrm{mL}$ 三辛基氧化膦-甲基异丁基酮溶液。

A.6.2.6 铅标准储备液：准确称取 $1.000 \mathrm{~g}$ 金属铅 $(99.99\%)$ ，分次加少量硝酸，加热溶解，移入 $1000 \mathrm{~mL}$ 容量瓶，加水定容至刻度。混匀，此溶液每毫升含 $1.0 \mathrm{mg}$ 铅。

A.6.2.7 铅工作标准溶液: 移取 $1 \mathrm{~mL}$ 标准储备液至 $1000 \mathrm{~mL}$ 容量瓶, 用水稀释至刻度, 摇匀。移取 $2 \mathrm{~mL}$ 该溶液至 $50 \mathrm{~mL}$ 容量瓶, 加入 $10 \mathrm{~mL}$ 盐酸溶液 $(3 + 1) 、 10 \mathrm{~mL}$ 水、 $20 \mathrm{~mL}$ 抗坏血酸-碘化钠溶液、 $5 \mathrm{~mL}$ 三辛基氧化膦-甲基异丁基酮溶液, 反复振摇。加水使有机溶剂层到瓶颈部, 摇匀, 静置分层。有机溶剂层即是标准工作溶液, 此溶液每毫升含 $0.4 \mu \mathrm{g}$ 铅。

A.6.2.8 空白溶液: 移取 $10 \mathrm{~mL}$ 盐酸溶液 $(3 + 1)$ , 加 $10 \mathrm{~mL}$ 水、 $20 \mathrm{~mL}$ 抗坏血酸-碘化钠溶液、 $5 \mathrm{~mL}$ 三辛基氧化膦-甲基异丁基酮溶液至 $50 \mathrm{~mL}$ 容量瓶, 反复振摇。加水使有机溶剂层到瓶颈部, 摇匀, 静置分层。有机溶剂层即是空白溶液。

# A.6.3 仪器和设备

原子吸收分光光度计：配有铅空心阴极灯。

# A.6.4 分析步骤

# A.6.4.1 试样溶液的制备

称取 $0.50\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ ) 至 $50~\mathrm{mL}$ 容量瓶，加入 $10~\mathrm{mL}$ 盐酸溶液 $(3 + 1)$ 使之溶解，再加入 $10~\mathrm{mL}$ 水、 $20~\mathrm{mL}$ 抗坏血酸-碘化钠溶液、 $5~\mathrm{mL}$ 三辛基氧化膦-甲基异丁基酮溶液，摇匀。加水使有机溶剂层到瓶颈部，摇匀，静置分层。有机溶剂层即是试样溶液。

# A.6.4.2 测定

将试样溶液导入原子吸收分光光度计, 火焰原子化后, 吸收 $283.3 \mathrm{~nm}$ 共振线, 用三辛基氧化膦-甲基异丁基酮溶液将仪器调零, 依次检测空白溶液、标准工作溶液和试样溶液的吸光度。试样溶液的吸光

度不得大于标准溶液的吸光度。

# A.7 砷(As)的测定

称取 $0.5\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ ), 加入 $5\mathrm{mL}$ 浓盐酸溶解, 加入硫脲-抗坏血酸溶液 (称取 $50\mathrm{g}$ 硫脲、 $50\mathrm{g}$ 抗坏血酸, 加水溶解, 定容至 $100\mathrm{mL})1\mathrm{mL}$ , 定容至 $10\mathrm{mL}$ , 作为试样溶液。以下按GB5009.11—2014“第一篇 总砷的测定”第二法中的规定进行。

# A.8 总汞（以 $\mathrm{Hg}$ 计）的测定

称取 $0.5\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )，加入 $5\mathrm{mL}$ 浓盐酸溶解，定容至 $10\mathrm{mL}$ ，作为试样溶液。以下按GB5009.17—2014“第一篇 食品中总汞的测定”第一法中的规定进行测定。

# A.9 氯化物（以Cl计）的测定

# A.9.1 试剂和材料

A.9.1.1 硝酸溶液： $1 + 1$

A.9.1.2 硝酸溶液： $1 + 9$

A.9.1.3 酚酞指示剂： $10\mathrm{g / L}$

A.9.1.4 氢氧化钠溶液： $3\mathrm{mol} / \mathrm{L}$

A.9.1.5 硝酸银溶液： $20\mathrm{g / L}$

A.9.1.6 氯化物标准溶液: $3.55 \mathrm{~g} / \mathrm{L}$ 。使用时, 将此标准溶液稀释 10 倍, 使氯化物 (以 $\mathrm{Cl}$ 计) 质量浓度为 $0.355 \mathrm{~g} / \mathrm{L}$ 。

# A.9.2 分析步骤

称取本品 $1.0\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），置 $100~\mathrm{mL}$ 烧杯中，加入 $15~\mathrm{mL}$ 硝酸溶液 $(1 + 1)$ 在水浴中加热溶解。往此溶液中加入酚酞指示剂数滴和氢氧化钠溶液 $50~\mathrm{mL}$ ，充分混匀后，转移至 $100~\mathrm{mL}$ 容量瓶，稀释至刻度，放置 $10\mathrm{min}$ 后，用干燥滤纸过滤。取滤液 $10~\mathrm{mL}$ ，转移至 $100~\mathrm{mL}$ 容量瓶，稀释至刻度，作为试验溶液。

取试验溶液 $2.0 \mathrm{~mL}$ 于 $50 \mathrm{~mL}$ 比色管内，用硝酸溶液 $(1 + 9)$ 进行中和，加 $1.0 \mathrm{~mL}$ 硝酸银溶液，稀释至刻度，充分混合，暗处放置 $5 \mathrm{~min}$ ，将两比色管放置在黑色背景下，从侧面或上方观察，试验溶液所呈浊度不得大于标准管。

标准管的制备：取 $0.355\mathrm{g / L}$ 氯化物标准溶液 $0.20~\mathrm{mL}$ ，与试验溶液同时同样处理。

# A.10 硫酸盐（以 $\mathbf{SO}_4$ 计）的测定

# A.10.1 试剂和材料

A.10.1.1 浓盐酸。

A.10.1.2 硫酸盐（以 $\mathrm{SO}_4$ 计）标准溶液： $0.48\mathrm{g / L}$

A.10.1.3 氯化钡溶液： $120~\mathrm{g / L}$

# A.10.2 分析步骤

量取A.9.2的滤液 $40~\mathrm{mL}$ 于 $50~\mathrm{mL}$ 比色管内，用浓盐酸进行中和，并过量 $1\mathrm{mL}\sim 2\mathrm{mL}$ ，加 $2\mathrm{mL}$ 氯化钡溶液，充分混合，用水稀释至刻度，放置 $10\mathrm{min}$ ，将两比色管放置在黑色背景下，从侧面或上方观察，试验溶液所呈浊度不得大于标准管。

标准管的制备：取 $0.48\mathrm{g / L}$ 硫酸盐标准溶液 $1.00~\mathrm{mL}$ ，与试验溶液同时同样处理。

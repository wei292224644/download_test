# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T1591—2005《进出口茶叶中9种有机杂环类农药残留量的检验方法》。与SN/T1591—2005，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；

——标准名称中“进出口茶叶”改为“茶叶”；

——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T1591—2005。

# 食品安全国家标准

# 茶叶中9种有机杂环类农药残留量的检测方法

# 1 范围

本标准规定了茶叶中9种有机杂环类农药残留量检验的抽样和制样、测定方法、测定低限及回收率。

本标准适用于茶叶中莠去津、乙烯菌核利、腐霉利、氟菌唑、抑霉唑、噻嗪酮、丙环唑、氯苯嘧啶醇、哒螨灵残留量的检验，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682 分析实验室用水规格和试验方法

# 3 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 3.1 试剂

3.1.1 丙酮 $\mathrm{(CH_3COCH_3)}$ ：重蒸馏。

3.1.2 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：重蒸馏。

3.1.3 氯化钠(NaCl)。

# 3.2 溶液配制

3.2.1 丙酮—正己烷(1+3)溶液：取 $100\mathrm{mL}$ 丙酮，加入 $300\mathrm{mL}$ 正己烷，摇匀备用。  
3.2.2 丙酮一正己烷(2+1)溶液：取 $200~\mathrm{mL}$ 丙酮，加入 $100~\mathrm{mL}$ 正己烷，摇匀备用。

# 3.3 标准品

农药标准品：纯度 $\geq 99\%$

# 3.4 标准溶液配制

标准溶液: 准确称取适量莠去津、乙烯菌核利、腐霉利、氟菌唑、抑霉唑、噻嗪酮、丙环唑、氯苯嘧啶醇、哒螨灵标准品, 用丙酮配制成浓度为 $1.00 \mathrm{mg} / \mathrm{mL}$ 标准储备液。再根据需要用正己烷稀释成相应的标准工作液。

# 3.5 材料

3.5.1 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：经过 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置于干燥器中备用。  
3.5.2 活性炭固相萃取小柱： $250\mathrm{mg}$ ，或相当者。  
3.5.3 中性氧化铝固相萃取小柱： $250\mathrm{mg}$ ，或相当者。

# 4 仪器和设备

4.1 气相色谱仪，配质量选择性检测器。

4.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
4.3 固相萃取装置，带真空泵。  
4.4 多功能微量化样品处理仪，或相当者。  
4.5 离心机： $4000\mathrm{r / min}$   
4.6 涡旋振荡器。  
4.7 离心管： $15\mathrm{mL}$   
4.8 刻度试管： $15~\mathrm{mL}$   
4.9 微量注射器： $10~\mu \mathrm{L}$

# 5 试样制备与保存

# 5.1 试样制备

将所取回的样品磨碎，取样部位按GB2763的规定执行，使通过孔径为 $0.84\mathrm{mm}$ 筛，混匀，均分成2份，装入清洁的容器内，作为试样。密封，并标明标记。在抽样和制样的操作过程中，必须防止样品受到污染或发生残留物含量的变化。

# 5.2 试样保存

将试样于 $-5^{\circ} \mathrm{C}$ 以下避光保存。

# 6 分析步骤

试样中残留的中莠去津、乙烯菌核利、腐霉利、氟菌唑、抑霉唑、噻嗪酮、丙环唑、氯苯嘧啶醇和哒螨灵用丙酮一正己烷提取，采用活性炭小柱和中性氧化铝小柱净化，被测物用丙酮一正己烷洗脱。净化后用气相色谱仪，配有质谱检测器的测定，外标法定量。

# 6.1 提取

准确称取 $1 \mathrm{~g}$ 均匀试样(精确至 $0.001 \mathrm{~g}$ ) 于 $15 \mathrm{~mL}$ 离心管中, 加入 $1 \mathrm{~g}$ 氯化钠, 加入 $2 \mathrm{~mL}$ 蒸馏水, 于涡旋振荡器上混匀 $30 \mathrm{~s}$ , 放置 $30 \mathrm{~min}$ , 再加入 $4 \mathrm{~mL}$ 丙酮和正己烷混合液, 在混合器上混匀 $2 \mathrm{~min}$ 。在 $2500 \mathrm{r} / \mathrm{min}$ 下离心 $1 \mathrm{~min}$ , 吸取上层正己烷提取液于另一 $15 \mathrm{~mL}$ 刻度试管中。再分别加入 $2 \mathrm{~mL}$ 丙酮和正己烷混合液重复提取两次, 合并提取液, 加入 $1 \mathrm{~g}$ 无水硫酸钠干燥。将干燥后的提取液完全转移至另一干净刻度试管中置于微量化样品处理仪上在 $50^{\circ} \mathrm{C}$ 、氮气流吹至约 $1 \mathrm{~mL}$ (溶液 A)。

# 6.2 净化

将活性炭固相萃取小柱和中性氧化铝固相萃取小柱(活性炭固相萃取小柱内填约 $1 \mathrm{~cm}$ 高的无水硫酸钠层)自上而下安装在固相萃取的真空抽滤装置上, 先用 $1 \mathrm{~mL}$ 丙酮 3 次预淋洗小柱, 再用 $1 \mathrm{~mL}$ 正己烷 3 次预淋洗小柱, 保持滴速 $0.5 \mathrm{~mL} / \mathrm{min}$ , 弃去所有淋洗液。

将溶液A依次过活性炭固相萃取小柱和中性氧化铝固相萃取小柱，再用 $6.0\mathrm{mL}$ 丙酮和正己烷混合液淋洗柱子，收集全部洗脱液，置于 $50^{\circ}\mathrm{C}$ 下，氮气流吹至近干。最后用正己烷定容于 $0.5\mathrm{mL}$ ，供GC-MSD分析备用。

# 6.3 测定

# 6.3.1 气相色谱—质谱参考条件

a) 色谱柱: 石英毛细管柱, $5 \%$ 苯基甲基聚硅氧烷固定相。 $30 \mathrm{~m} \times 0.20 \mathrm{~mm}$ (内径), 膜厚 $0.25 \mu \mathrm{m}$ ,或相当者;  
b）色谱柱温度： $70^{\circ}\mathrm{C}$ 保持 $2\mathrm{min}$ ，以 $8^{\circ}\mathrm{C} / \mathrm{min}$ 上升至 $180^{\circ}\mathrm{C}$ ，再以 $3^{\circ}\mathrm{C} / \mathrm{min}$ 上升至 $280^{\circ}\mathrm{C}$ ，保持 $18\mathrm{min}$ ；  
c）进样口温度： $250^{\circ}\mathrm{C}$   
d）色谱一质谱接口温度： $220^{\circ}\mathrm{C}$

e) 载气：氦气（纯度 $>99.995\%$ ），流速 $0.6 \mathrm{~mL} / \mathrm{min}$ ；  
f）进样量： $1\mu \mathrm{L}$   
g）进样方式：无分流进样， $1\mathrm{min}$ 后开阀；  
h）电离方式：EI；  
i）电离能量： $70\mathrm{eV}$   
j）测定方式：选择离子监测方式(SIM)；  
k) 监测离子 $(m / z)$ ：见表1；  
1) 溶剂延迟： $15\mathrm{min}$

# 6.3.2 色谱测定

根据样液中莠去津、乙烯菌核利、腐霉利、氟菌唑、抑霉唑、噻嗪酮、丙环唑、氯苯嘧啶醇和哒螨灵的含量情况，选定峰面积相近的标准工作溶液。标准工作溶液和样液中莠去津、乙烯菌核利、腐霉利、氟菌唑、抑霉唑、噻嗪酮、丙环唑、氯苯嘧啶醇和哒螨灵的响应值均应在仪器检测的线性范围内。对标准工作液的样液等体积参插进样测定。在上述色谱条件下，标准品SIM色谱图参见附录A中图A.1。

# 6.4 空白实验

除不加试样外，均按 $6.1\sim 6.3$ 的规定执行。

# 7 结果计算和表述

用色谱数据处理机或按式(1)计算试样中莠去津、乙烯菌核利、腐霉利、氟菌唑、抑霉唑、噻嗪酮、丙环唑、氯苯嘧啶醇和哒螨灵的含量。

$$
X = \frac {A \times c _ {s} \times V}{A _ {s} \times m} \dots \tag {1}
$$

式中：

$X$ ——试样中莠去津、乙烯菌核利、腐霉利、氟菌唑、抑霉唑、噻嗪酮、丙环唑、氯苯嘧啶醇和哒螨灵的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

A ——样液中莠去津、乙烯菌核利、腐霉利、氟菌唑、抑霉唑、噻嗪酮、丙环唑、氯苯嘧啶醇和哒螨灵的峰面积,单位为平方毫米 $(\mathrm{mm}^{2})$ ；

$c_{s}$ ——标准工作液中莠去津、乙烯菌核利、腐霉利、氟菌唑、抑霉唑、噻嗪酮、丙环唑、氯苯嘧啶醇和哒螨灵的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$A_{s}$ ——标准工作液中莠去津、乙烯菌核利、腐霉利、氟菌唑、抑霉唑、噻嗪酮、丙环唑、氯苯嘧啶醇和哒螨灵的峰面积，单位为平方毫米 $(\mathrm{mm}^2)$ ；

$V$ ——样液最终定容体积，单位为毫克 $(\mathrm{mL})$ ；

$m$ ——最终样液所代表的试样量，单位为克(g)。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 8 精密度

8.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 B 的要求。  
8.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 C 的要求。

# 9 定量限和回收率

# 9.1 定量限

本方法莠去津的定量限是 $0.02\mathrm{mg / kg}$ ，乙烯菌核利定量限是 $0.02\mathrm{mg / kg}$ ，腐霉利定量限是0.02 $\mathrm{mg / kg}$ ，氟菌唑定量限是 $0.38\mathrm{mg / kg}$ ，抑霉唑定量限是 $0.05\mathrm{mg / kg}$ ，噻嗪酮定量限是 $0.01\mathrm{mg / kg}$ ，丙环唑定量限是 $0.05\mathrm{mg / kg}$ ，氯苯嘧啶醇定量限是 $0.02\mathrm{mg / kg}$ ，哒螨灵定量限是 $0.5\mathrm{mg / kg}$ 。

# 9.2 回收率

当不同添加水平时，9种杂环类农药在茶叶中的添加回收率参见附录D。

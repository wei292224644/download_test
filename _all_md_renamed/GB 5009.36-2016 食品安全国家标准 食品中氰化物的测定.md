2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替GB/T5009.36—2003《粮食卫生标准的分析方法》的4.4氰化物、GB/T5009.48—2003《蒸馏酒与配制酒卫生标准的分析方法》的4.7氰化物和GB/T8538—2008《饮用天然矿泉水检验方法》的4.45氰化物。

# 食品安全国家标准

# 食品中氰化物的测定

# 1 范围

本标准规定了食品中氰化物的检测方法。

本标准第一法适用于蒸馏酒及其配制酒、木薯、包装饮用水、矿泉水中氰化物的检测，第二法和第三法适用于蒸馏酒及其配制酒、粮食、木薯、包装饮用水、矿泉水中氰化物的检测。

# 第一法 分光光度法

# 2 原理

木薯粉、包装饮用水和矿泉水中的氰化物在酸性条件下蒸馏出的氰氢酸用氢氧化钠溶液吸收，在 $\mathrm{pH} = 7.0$ 条件下，馏出液用氯胺T将氰化物转变为氯化氰，再与异烟酸-吡唑啉酮作用，生成蓝色染料，与标准系列比较定量。

蒸馏酒及其配制酒在碱性条件下加热除去高沸点有机物，然后在 $\mathrm{pH} = 7.0$ 条件下，用氯胺T将氰化物转变为氯化氰，再与异烟酸-吡唑啉酮作用，生成蓝色染料，与标准系列比较定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 甲基橙 $\mathrm{(C_{14}H_{14}O_3N_3SNa)}$ ：指示剂。  
3.1.2 酚酞 $\mathrm{(C_{20}H_{14}O_4)}$ ：指示剂。  
3.1.3 酒石酸 $(\mathrm{C_4H_6O_6})$ 。  
3.1.4 氢氧化钠(NaOH)。  
3.1.5 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$   
3.1.6 磷酸氢二钠 $\mathrm{(Na_2HPO_4)}$   
3.1.7 乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
3.1.8 异烟酸 $(\mathrm{C_6H_5O_2N})$ 。  
3.1.9 吡唑啉酮 $\mathrm{(C_{10}H_{10}N_2O)}$ 。  
3.1.10 氯胺 $\mathrm{T(C_7H_7SO_2NClNa\cdot 3H_2O)}$ ：保存于干燥器中。  
3.1.11 无水乙醇 $\mathrm{(C_2H_6O)}$ 。  
3.1.12 乙酸锌 $(\mathrm{C_4H_6O_4Zn})$ 。

# 3.2 试剂配制

3.2.1 甲基橙指示剂 $(0.5\mathrm{g} / \mathrm{L})$ ：称取 $50~\mathrm{mg}$ 甲基橙，溶于水中，并稀释至 $100~\mathrm{mL}$ 。

3.2.2 氢氧化钠溶液 $(20\mathrm{g / L})$ ：称取 $2\mathrm{g}$ 氢氧化钠，溶于水中，并稀释至 $100~\mathrm{mL}$ 。  
3.2.3 氢氧化钠溶液 $(10\mathrm{g / L})$ ：称取 $1\mathrm{g}$ 氢氧化钠，溶于水中，并稀释至 $100~\mathrm{mL}$ 。  
3.2.4 乙酸锌溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ 乙酸锌，溶于水中，并稀释至 $100~\mathrm{mL}$ 。  
3.2.5 氢氧化钠溶液 $(2\mathrm{g} / \mathrm{L})$ ：量取 $10~\mathrm{mL}$ 氢氧化钠溶液(3.2.2)，用水稀释至 $100~\mathrm{mL}$ 。  
3.2.6 氢氧化钠溶液 $(1\mathrm{g / L})$ ：量取 $5\mathrm{mL}$ 氢氧化钠溶液(3.2.2)，用水稀释至 $100~\mathrm{mL}$ 。  
3.2.7 乙酸溶液 $(1 + 24)$ ：将乙酸和水按 $1:24$ 的体积比混匀。  
3.2.8 酚酞-乙醇指示液 $(10\mathrm{g / L})$ ：称取 $1\mathrm{g}$ 酚酞试剂，用无水乙醇溶解，并定容至 $100~\mathrm{mL}$ 。  
3.2.9 磷酸盐缓冲溶液 $\left[(0.5 \mathrm{~mol} / \mathrm{L}) \mathrm{pH} 7.0\right]$ ：称取 $34.0 \mathrm{~g}$ 无水磷酸二氢钾和 $35.5 \mathrm{~g}$ 无水磷酸氢二钠，溶于水并稀释至 $1000 \mathrm{~mL}$ 。  
3.2.10 异烟酸-吡唑啉酮溶液：称取 $1.5\mathrm{g}$ 异烟酸溶于 $24~\mathrm{mL}$ 氢氧化钠溶液(3.2.2)中，加水至 $100~\mathrm{mL}$ ，另称取 $0.25\mathrm{g}$ 吡唑啉酮，溶于 $20~\mathrm{mL}$ 无水乙醇中，合并上述两种溶液，摇匀。临用时配制。  
3.2.11 氯胺T溶液 $(10\mathrm{g / L})$ ：称取 $1\mathrm{g}$ 氯胺T溶于水中，并稀释至 $100~\mathrm{mL}$ 。临用时配制。

# 3.3 标准溶液配制

3.3.1水中氰成分分析标准物质 $(50~\mu \mathrm{g / mL})$ ：标准物质编号为GBW(E)080115。  
3.3.2 氰离子标准中间液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：取 $2\mathrm{mL}$ 水中氰成分分析标准物质(3.3.1)，用氢氧化钠溶液(3.2.5)定容至 $100~\mathrm{mL}$ 。

# 4 仪器与设备

4.1 可见分光光度计。  
4.2 分析天平：感量为 $0.001\mathrm{g}$   
4.3 具塞比色管： $10~\mathrm{mL}$   
4.4 恒温水浴锅： $37^{\circ}C\pm 1^{\circ}C$   
4.5 电加热板： $120^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 。  
4.6 $500~\mathrm{mL}$ 水蒸气蒸馏装置。

# 5 分析步骤

# 5.1 木薯粉

5.1.1 称取 $20 \mathrm{~g}$ (精确到 $0.001 \mathrm{~g}$ ) 试样于 $500 \mathrm{~mL}$ 水蒸气蒸馏装置中, 加水约 $200 \mathrm{~mL}$ , 塞严瓶口, 在室温下磁力搅拌 $2 \mathrm{~h}$ 。然后加入 $20 \mathrm{~mL}$ 乙酸锌溶液和 $2.0 \mathrm{~g}$ 酒石酸, 迅速连接好蒸馏装置, 将冷凝管下端插入盛有 $10 \mathrm{~mL} 20 \mathrm{~g} / \mathrm{L}$ 氢氧化钠溶液的 $100 \mathrm{~mL}$ 锥形瓶①的液面下。进行水蒸气蒸馏, 收集蒸馏液接近 $100 \mathrm{~mL}$ 时, 取下锥形瓶①; 同时将冷凝管下端插入盛有 $10 \mathrm{~mL} 20 \mathrm{~g} / \mathrm{L}$ 氢氧化钠溶液的 $100 \mathrm{~mL}$ 锥形瓶②的液面下, 重复蒸馏至收集蒸馏液约 $80 \mathrm{~mL}$ 时, 停止加热, 继续收集蒸馏液近 $100 \mathrm{~mL}$ , 取下锥形瓶②; 取下蒸馏瓶并将其内容物充分搅拌、混匀, 再将冷凝管下端插入盛有 $10 \mathrm{~mL} 20 \mathrm{~g} / \mathrm{L}$ 氢氧化钠溶液的 $100 \mathrm{~mL}$ 锥形瓶③的液面下, 进行水蒸气蒸馏, 至锥形瓶③收集蒸馏液约 $50 \mathrm{~mL}$ , 取下锥形瓶③。将上述锥形瓶①、②和③收集的蒸馏液完全转移至 $250 \mathrm{~mL}(V_{1})$ 容量瓶中, 用水定容至刻度。量取 $10 \mathrm{~mL}$ 溶液 $(V_{2})$ 置于 $25 \mathrm{~mL}$ 比色管中, 作为试样溶液。  
5.1.2 用移液管分别量取 $0.0\mathrm{mL},0.3\mathrm{mL},0.6\mathrm{mL},0.9\mathrm{mL},1.2\mathrm{mL},1.5\mathrm{mL}$ 氰离子标准中间液置于 $25~\mathrm{mL}$ 比色管中，加水至 $10~\mathrm{mL}$   
5.1.3 试样溶液及标准系列溶液中各加 $1 \mathrm{~mL} 10 \mathrm{~g} / \mathrm{L}$ 氢氧化钠溶液和 1 滴酚酞指示液, 用乙酸溶液缓慢调至红色褪去, 然后加 $5 \mathrm{~mL}$ 磷酸盐缓冲溶液, 在 $37^{\circ} \mathrm{C}$ 恒温水浴锅中保温 $10 \mathrm{~min}$ , 再分别加入

$0.25 \mathrm{~mL}$ 氯胺 T 溶液, 加塞振荡混合均匀, 放置 $5 \mathrm{~min}$ 。然后分别加入 $5 \mathrm{~mL}$ 异烟酸-吡唑酮溶液, 加水至 $25 \mathrm{~mL}$ , 混匀。在 $37^{\circ} \mathrm{C}$ 恒温水浴锅中放置 $40 \mathrm{~min}$ , 用 $2 \mathrm{~cm}$ 比色杯, 以零管调节零点, 于波长 $638 \mathrm{~nm}$ 处测吸光度。

# 5.2 蒸馏酒及其配制酒

5.2.1 吸取 $1.0 \mathrm{~mL}$ 试样于 $50 \mathrm{~mL}$ 烧杯中，加入 $5 \mathrm{~mL} 2 \mathrm{~g} / \mathrm{L}$ 氢氧化钠溶液，放置 $10 \mathrm{~min}$ ，然后放于 $120^{\circ} \mathrm{C}$ 电加热板上加热至溶液剩余约 $1 \mathrm{~mL}$ ，取下放至室温，用 $2 \mathrm{~g} / \mathrm{L}$ 氢氧化钠溶液转移至 $10 \mathrm{~mL}$ 具塞比色管中，最后加 $2 \mathrm{~g} / \mathrm{L}$ 氢氧化钠至 $5 \mathrm{~mL}$ 。  
5.2.2 若酒样浑浊或有色，取 $25.0\mathrm{mL}$ 试样于 $250~\mathrm{mL}$ 蒸馏瓶中，加入 $100~\mathrm{mL}$ 水，滴加数滴甲基橙指示剂，将冷凝管下端插入盛有 $10~\mathrm{mL}2\mathrm{g / L}$ 氢氧化钠溶液比色管的液面下，再加 $1\mathrm{g}\sim 2\mathrm{g}$ 酒石酸，迅速连接蒸馏装置进行水蒸气蒸馏，收集蒸馏液约 $50~\mathrm{mL}$ ，然后用水定容至 $50~\mathrm{mL}$ ，混合均匀。取 $2.0~\mathrm{mL}$ 馏出液按5.2.1操作。  
5.2.3 用移液管分别吸取 $0\mathrm{mL},0.4\mathrm{mL},0.8\mathrm{mL},1.2\mathrm{mL},1.6\mathrm{mL},2.0\mathrm{mL}$ 氰离子标准中间液于 $10~\mathrm{mL}$ 具塞比色管中，加 $2\mathrm{g / L}$ 氢氧化钠至 $5\mathrm{mL}$ 。  
5.2.4 于试样及标准管中分别加入2滴酚酞指示剂，然后加入乙酸溶液调至红色褪去，再用 $2\mathrm{g / L}$ 氢氧化钠溶液调至近红色，然后加 $2\mathrm{mL}$ 磷酸盐缓冲溶液（如果室温低于 $20^{\circ}C$ 即放入 $25^{\circ}C\sim 30^{\circ}C$ 水浴中 $10\mathrm{min}$ ，再加入 $0.2\mathrm{mL}$ 氯胺T溶液，摇匀放置 $3\mathrm{min}$ ，加入 $2\mathrm{mL}$ 异烟酸-吡唑啉酮溶液，加水稀释至刻度，加塞振荡混合均匀，在 $37^{\circ}C$ 恒温水浴锅中放置 $40\mathrm{min}$ ，取出用 $1\mathrm{cm}$ 比色杯以空白管调节零点，于波长 $638~\mathrm{nm}$ 处测吸光度。

# 5.3 饮用水、矿泉水

5.3.1量取 $250~\mathrm{mL}$ 水样(氰化物含量超过 $20~\mu \mathrm{g}$ 时，可取适量水样，加水至 $250~\mathrm{mL}$ )置于 $500~\mathrm{mL}$ 水蒸气蒸馏装置中，加入1滴～2滴甲基橙指示剂，再加入 $5\mathrm{mL}$ 乙酸锌溶液，加入 $1\mathrm{g}\sim 2\mathrm{g}$ 酒石酸，溶液由橙黄色变成了橙红，迅速连接好蒸馏装置，将冷凝管下端插入盛有 $10~\mathrm{mL}20~\mathrm{g / L}$ 氢氧化钠溶液的 $50~\mathrm{mL}$ 具塞比色管的液面下。通过调节温度将蒸馏速度控制在 $2\mathrm{mL / min}\sim 3\mathrm{mL / min}$ ，收集蒸馏液约 $50~\mathrm{mL}$ ，然后用水定容至 $50~\mathrm{mL}$ ，混合均匀。取 $10.0~\mathrm{mL}$ 馏出液置于 $25~\mathrm{mL}$ 具塞比色管中。  
5.3.2 另取 $25~\mathrm{mL}$ 具塞比色管，分别加入氰离子标准中间液 $0\mathrm{mL},0.10\mathrm{mL},0.20\mathrm{mL},0.40\mathrm{mL}$ $0.60~\mathrm{mL},0.80~\mathrm{mL},1.00~\mathrm{mL},1.50~\mathrm{mL}$ 和 $2.00~\mathrm{mL}$ ，加 $1\mathrm{g / L}$ 氢氧化钠溶液至 $10.0~\mathrm{mL}$ 。  
5.3.3 于试样和标准管中各加 $5.0 \mathrm{~mL}$ 磷酸盐缓冲液。置于 $37^{\circ} \mathrm{C}$ 恒温水浴中，再加入 $0.25 \mathrm{~mL}$ 氯胺T溶液，加塞混合，放置 $5 \mathrm{~min}$ ，然后加入 $5.0 \mathrm{~mL}$ 异烟酸-吡唑酮溶液，加水至 $25 \mathrm{~mL}$ ，混匀，在 $37^{\circ} \mathrm{C}$ 恒温水浴锅中放置 $40 \mathrm{~min}$ ，用 $3 \mathrm{~cm}$ 比色杯，以纯水做参比，于波长 $638 \mathrm{~nm}$ 处测吸光度。绘制校准曲线，从曲线上查出样品管中氰化物质量。

# 6 分析结果的表述

# 6.1 木薯粉结果计算

试样中氰化物（以 $\mathrm{CN^{-}}$ 计)的含量按式(1)计算：

$$
X = \frac {A \times 1 0 0 0}{m \times V _ {2} / V _ {1} \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中氰化物含量(以 $\mathrm{CN}^{-}$ 计)，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A$ ——测定试样溶液氰化物质量(以 $\mathrm{CN}^{-}$ 计)，单位为微克 $(\mu \mathrm{g})$ ；

1000 ——换算系数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——测定用蒸馏液体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——试样蒸馏液总体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留三位有效数字。

# 6.2 蒸馏酒及其配制酒结果计算

按5.2.1操作时试样中氰化物(以 $\mathrm{CN^{-}}$ 计)的含量按式(2)计算：

$$
X = \frac {m \times 1 0 0 0}{V \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中氰化物含量(以 $\mathrm{CN}^{-}$ 计)，单位为毫克每升 $(\mathrm{mg / L})$ ；

$m$ ——测定用试样中氰化物的质量，单位为微克 $(\mu \mathrm{g})$ ；

1000——换算系数；

$V$ ——试样体积，单位为毫升 $(\mathrm{mL})$ 。

按5.2.2操作时试样中氰化物(以 $\mathrm{CN^{-}}$ 计)的含量按式(3)计算：

$$
X = \frac {m \times 5 0 \times 1 0 0 0}{V \times 2 \times 1 0 0 0} \tag {3}
$$

式中：

$X$ ——试样中氰化物含量(以 $\mathrm{CN}^{-}$ 计)，单位为毫克每升 $(\mathrm{mg / L})$ ；

$m$ ——测定用试样馏出液中氰化物的质量，单位为微克 $(\mu \mathrm{g})$ ；

50,2,1000——换算系数；

$V$ ——试样体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留两位有效数字。

# 6.3 包装饮用水、矿泉水结果计算

试样中氰化物（以 $\mathrm{CN^{-}}$ 计)的含量按式(4)计算：

$$
X = \frac {m \times V _ {1}}{V \times V _ {2}} \tag {4}
$$

式中：

$X$ ——水样中氰化物含量(以 $\mathrm{CN}^{-}$ 计)，单位为毫克每升 $(\mathrm{mg / L})$ ；

$m$ ——从校准曲线上查得样品管中氰化物的质量，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——馏出液总体积，单位为毫升 $(\mathrm{mL})$

$V$ ——水样体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——比色所用馏出液体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留两位有效数字。

# 7 精密度

重复性条件下获得的两次独立测定结果的绝对差值不超过算术平均值的 $10\%$ 。

本方法酒的检出限为 $0.004\mathrm{mg / L}$ ，木薯粉的检出限为 $0.015\mathrm{mg / kg}$ ，水的检出限为 $0.002\mathrm{mg / L}$ ，酒

的定量限为 $0.015\mathrm{mg / L}$ ，木薯粉的定量限为 $0.045\mathrm{mg / kg}$ ，水的定量限为 $0.006\mathrm{mg / L}$ 。

# 第二法 气相色谱法

# 9 原理

在密闭容器和一定温度下，食品中的氰化物在酸性条件下用氯胺T将其衍生为氯化氰，氯化氰在气相和液相中达到平衡，将气相部分导入气相色谱法进行分离，电子捕获检测器检测，以外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 10.1 试剂

10.1.1 氯胺 $\mathrm{T(C_7H_7ClNNaO_2S\cdot 3H_2O)}$ ：保存于干燥器中。  
10.1.2 磷酸 $(\mathrm{H}_3\mathrm{PO}_4) : \geqslant 85\%$ 。  
10.1.3 氢氧化钠(NaOH)。

# 10.2 试剂配制

10.2.1 氯胺T溶液 $(10\mathrm{g} / \mathrm{L})$ ：称取 $0.1\mathrm{g}$ 氯胺T，用水溶解定容至 $10~\mathrm{mL}$ （现用现配）。  
10.2.2 磷酸溶液 $(1 + 5)$ ：量取 $10~\mathrm{mL}$ 浓磷酸，加入到 $50~\mathrm{mL}$ 水中，混合均匀。  
10.2.3 $0.1\%$ 氢氧化钠溶液：称取 $1.0\mathrm{g}$ 氢氧化钠，用水溶解定容至 $1\mathrm{L}$ 。

# 10.3 标准溶液配制

10.3.1 水中氰成分分析标准物质 $(50\mu \mathrm{g} / \mathrm{mL})$ ：标准物质编号为GBW(E)080115。  
10.3.2 氰离子(以 $\mathrm{CN}^{-}$ 计)标准中间溶液：准确移取 $2.00~\mathrm{mL}$ 的水中氰成分分析标准物质(10.3.1)于 $10~\mathrm{mL}$ 的容量瓶，用 $0.1\%$ 氢氧化钠溶液定容，此溶液浓度为 $10~\mathrm{mg / L}$ ，在 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中保存，可使用3个月。  
10.3.3 氰离子(以 $\mathrm{CN}^{-}$ 计)标准工作溶液。移取适量氰离子(以 $\mathrm{CN}^{-}$ 计)标准中间溶液(10.3.2)用水稀释配制成浓度为 $0\mathrm{mg / L}, 0.001\mathrm{mg / L}, 0.002\mathrm{mg / L}, 0.010\mathrm{mg / L}, 0.050\mathrm{mg / L}, 0.100\mathrm{mg / L}$ 的工作溶液。

注：当配制氯胺T溶液浑浊时，需更换新的氯胺T。

# 11 仪器与设备

11.1 气相色谱：配有电子捕获检测器（ECD）。  
11.2 顶空进样器。  
11.3 顶空瓶： $20~\mathrm{mL}$   
11.4 涡旋振荡器。  
11.5 分析天平：感量为 $0.0001\mathrm{g}$ 。  
11.6 离心机：转速 $\geqslant 4000\mathrm{r / min}$   
11.7 超声波清洗器。

# 12 分析步骤

# 12.1 试样制备

取固体试样约 $500\mathrm{g}$ ，用样品粉碎装置将其制成粉末，装入洁净容器，密封，于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 条件下保存。

取液体试样约 $500~\mathrm{mL}$ ，充分混匀，装入洁净容器中，密封，于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 条件下保存。

注：制样操作过程中必须防止样品受到污染。

# 12.2 仪器参考条件

# 12.2.1 顶空分析条件参见附录A

# 12.2.2 气相色谱参考条件

a）色谱柱：WAX毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径） $\times 0.25\mu \mathrm{m}$ （膜厚），或性能相当者；  
b）色谱柱温度： $40^{\circ}\mathrm{C}$ 保持 $5\mathrm{min}$ ，以 $50~^\circ \mathrm{C / min}$ 速率升至 $200^{\circ}C$ 保持 $2\mathrm{min}$   
c）载气：氮气，纯度 $\geq 99.999\%$   
d）进样口温度： $200^{\circ}C$   
e）检测器温度： $260^{\circ}C$   
f) 分流比： $5:1$   
g） 柱流速： $2.0~\mathrm{mL / min}$

# 12.3 标准曲线制作

分别准确移取 $10.0\mathrm{mL}$ 氰离子标准工作溶液(10.3.3)于6个顶空瓶中，加入 $0.2\mathrm{mL}$ 磷酸溶液，涡旋混合，然后加入 $0.2\mathrm{mL}$ 氯胺T溶液，立即加盖密封，涡旋混合，待测。

# 12.4 试样溶液的测定

# 12.4.1 蒸馏酒及其配制酒

准确移取 $0.2\mathrm{mL}$ 试样于顶空瓶中，加入蒸馏水 $9.8\mathrm{mL}$ ，加入 $0.2\mathrm{mL}$ 磷酸溶液，涡旋混合，然后加入 $0.2\mathrm{mL}$ 氯胺T溶液，立即加盖密封，涡旋混合，待测。

# 12.4.2 粮食

准确称取试样 $1\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ，用蒸馏水定容至 $100~\mathrm{mL}$ ，超声提取 $20\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，然后准确移取 $10~\mathrm{mL}$ 提取液于顶空瓶中，加入 $0.2\mathrm{mL}$ 磷酸溶液，涡旋混合，然后加入 $0.2\mathrm{mL}$ 氯胺T溶液，立即加盖密封，涡旋混合，待测。

# 12.4.3 包装饮用水、矿泉水

准确移取 $10\mathrm{mL}$ 试样于顶空瓶中，加入 $0.2\mathrm{mL}$ 磷酸溶液，涡旋混合，然后加入 $0.2\mathrm{mL}$ 氯胺T溶液，立即加盖密封，涡旋混合，待测。

# 12.5 样品空白测定

按照12.4检测步骤到加入 $0.2\mathrm{mL}$ 磷酸溶液，涡旋混合后，通入氮气在 $50^{\circ}\mathrm{C}$ 水浴中吹扫 $15\mathrm{min}$ ，然后加入 $0.2\mathrm{mL}$ 氯胺T溶液，立即加盖密封，涡旋混合，待测，测定结果即为样品空白。

# 12.6 气相色谱检测

标准溶液及样液均按12.2规定的条件进行测定，根据氰化物保留时间定性，测量样品溶液的峰面积(或峰高)响应值，采用外标法定量。样品溶液中氰化物衍生物的响应值应在标准线性范围内，若超出范围，在加磷酸溶液前用水稀释至范围内。在上述色谱条件下，氰化物的保留时间约为 $1.77\mathrm{min}$ 。标准品的色谱图见图A.1。

# 12.7 分析结果的表述

# 12.7.1 蒸馏酒及其配制酒、包装饮用水和矿泉水中氰化物（以 $\mathbf{CN}^{-}$ 计）结果计算

试样中氰化物（以 $\mathrm{CN^{-}}$ 计)含量按式(5)计算：

$$
X = \frac {\rho - \rho_ {0}}{V} \times 1 0 \tag {5}
$$

式中：

$X$ ——试样中氰化物(以 $\mathrm{CN}^{-}$ 计)的含量，单位为毫克每升 $(\mathrm{mg / L})$

$\rho$ ——由标准曲线得到的样液中氰化物的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——由标准曲线得到的样品空白中氰化物的浓度，单位为毫克每升 $(\mathrm{mg / L})$

$V$ ——样品体积，单位为毫升 $(\mathrm{mL})$ ；

10 ——加酸衍生前顶空瓶中溶液体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留三位有效数字。

# 12.7.2 粮食、木薯粉中氰化物（以 $\mathrm{CN}^{-}$ 计）结果计算

试样中氰化物（以 $\mathrm{CN}^{-}$ 计）含量用色谱数据处理机或按式(6)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {6}
$$

式中：

$X$ ——试样中氰化物(以 $\mathrm{CN}^{-}$ 计)的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho$ ——由标准曲线得到的样液中氰化物的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——由标准曲线得到的样品空白中氰化物的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——样品定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——样品质量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

# 第三法 定性法

# 15 原理

氰化物遇酸产生氢氰酸，氢氰酸与苦味酸钠作用，生成红色异氰酸紫酸钠。

# 16 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 16.1 试剂

16.1.1 苦味酸 $\mathrm{(C_6H_3N_3O_7)}$ 。  
16.1.2 酒石酸 $(\mathrm{C_4H_6O_6})$ 。  
16.1.3 碳酸钠 $\mathrm{(Na_2CO_3)}$   
16.1.4 乙醇 $(\mathrm{C}_2\mathrm{H}_6\mathrm{O})$ 。

# 16.2 试剂配制

16.2.1 碳酸钠溶液 $(100\mathrm{g / L})$ ：称取 $10.0\mathrm{g}$ 碳酸钠用水溶解定容至 $100~\mathrm{mL}$ 。  
16.2.2 饱和苦味酸乙醇溶液。

# 16.3 材料

苦味酸试纸：取定性滤纸剪成长 $7\mathrm{cm}$ 、宽 $0.3\mathrm{cm}\sim 0.5\mathrm{cm}$ 的纸条，浸入饱和苦味酸-乙醇溶液中，数分钟后取出，在空气中阴干，储存备用。

# 17 仪器

17.1 取 $100 \mathrm{~mL}$ 锥形瓶，配备一适宜的弹孔橡皮塞，孔内塞以直径 $0.4 \mathrm{~cm} \sim 0.5 \mathrm{~cm}$ ，长 $5 \mathrm{~cm}$ 的玻璃管，管内悬一条苦味酸试纸，临用时，试纸条以碳酸钠溶液湿润。  
17.2 恒温水浴锅：温度可控制至 $40^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ 。

# 18 分析步骤

# 18.1 粮食

称取 $5\mathrm{g}$ 试样，置于 $100~\mathrm{mL}$ 锥形瓶中，加入 $20~\mathrm{mL}$ 水及 $0.5\mathrm{g}$ 酒石酸，迅速塞上悬有苦味酸并以碳酸钠湿润的试纸条的橡皮塞，轻轻摇动使酒石酸溶解，置 $40^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ 水浴中，加热 $30\mathrm{min}$ ，观察试纸颜色变化。如试纸不变色，表示氰化物为负反应或未超过规定；如试纸变色，需再作定量试验。

# 18.2 酒

迅速称取 $20~\mathrm{mL}$ 试样，置于 $100~\mathrm{mL}$ 锥形瓶中，加入 $0.5\mathrm{g}$ 酒石酸，立即塞上悬有苦味酸并以碳酸钠湿润的试纸条的橡皮塞，轻轻摇动使酒石酸溶解，置 $40^{\circ}\mathrm{C} \sim 50^{\circ}\mathrm{C}$ 水浴中，加热 $30\mathrm{min}$ ，观察试纸颜色变

化。如试纸不变色，表示氰化物为负反应或未超过规定；如试纸变色，需再作定量试验。

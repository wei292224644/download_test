# 前言

本标准代替GB14883.2—1994《食品中放射性物质检验 氢-3的测定》。

本标准与GB14883.2—1994相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中放射性物质氢-3的测定”；  
——按照食品安全国家标准的格式对文本进行了调整；  
——补充细化了分析和实验步骤。

# 食品安全国家标准

# 食品中放射性物质氢-3的测定

# 1 范围

本标准适用于各类食品中氢-3（ ${}^{3}\mathrm{H}$ ）的测定。

# 2 原理

鲜样经燃烧-氧化，使游离水和有机物中氢全部转化为水。收集的水纯化后以电解法浓集 ${}^{3}\mathrm{H}$ ，用液体闪烁计数器测量 ${}^{3}\mathrm{H}$ 的放射性。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 过硫酸钾 $\mathrm{(K_2S_2O_8)}$ 。  
3.1.2 过氧化钠 $\mathrm{(Na_2O_2)}$ 。  
3.1.3 磷酸溶液： $85\%$ 浓度，市售。  
3.1.4 2,5-二苯基噁唑 $\left[\mathrm{OC}\left(\mathrm{C}_{6} \mathrm{H}_{5}\right) = \mathrm{NCH} = \mathrm{CC}_{6} \mathrm{H}_{5}\right]$ ：简称PPO，闪烁纯。  
3.1.5 1,4-双-2-(5-苯基噁唑)-苯 $\left[\mathrm{OC}\left(\mathrm{C}_{6} \mathrm{H}_{5}\right)=\mathrm{CHN}=\mathrm{C}\right]_{2} \mathrm{C}_{6} \mathrm{H}_{4})$ ：简称POPOP，闪烁纯。  
3.1.6 萘 $\left(\mathrm{C}_{10} \mathrm{H}_{8}\right)$ 。  
3.1.7 1,4-二氧杂环己烷 $\mathrm{(C_4H_8O_2)}$ 。  
3.1.8 本底水：电导率小于 $2 \times 10^{-6} \Omega \cdot \mathrm{cm}^{-1}$ 。本底水应取自 ${}^{3} \mathrm{H}$ 活度浓度水平尽可能低的水，如（深）地下水。蒸馏取得水样，测定水中 ${}^{3} \mathrm{H}$ 活度浓度 $(\mathrm{Bq} / \mathrm{m}^{3})$ ，并记录测定日期 $(t = 0)$ 。装在硼硅玻璃瓶中在黑暗中恒定温度保存。

# 3.2 试剂配制

闪烁液：取 $6\mathrm{g}$ PPO、 $0.3\mathrm{g}$ POPOP和 $100\mathrm{g}$ 萘，溶解于蒸馏纯化过的1,4-二氧杂环己烷（二氧六环)中并稀释至 $1\mathrm{L}$ 。保存在黑纸包好的棕色瓶中，避光备用。PPO和POPOP均为闪烁纯。也可使用市售已配好的商品闪烁液，但应尽量选用低毒性、高闪点溶解性好的安全高效产品。用户应检验替代的商品闪烁液的性能与最佳使用条件，以保证可接受性。

# 3.3 标准品

标准 ${}^{3}\mathrm{H}$ 水：已知准确 ${}^{3}\mathrm{H}$ 活度，用本底水稀释后备用。

# 4 仪器和设备

4.1 液体闪烁计数器：本底计数率不大于60计数/min， ${}^{3}\mathrm{H}$ 计数效率不小于 $20\%$ ，宜在恒温条件操作。

# 5 分析步骤

# 5.1 采样和预处理

采样和预处理按GB14883.1规定进行。

# 5.2 样品的燃烧与氧化

5.2.1 称取 $1.00\mathrm{kg}$ 洗净、晾干的食品鲜样，装入 ${}^{3}\mathrm{H}$ 氧化燃烧装置的燃烧室内。  
5.2.2 先通氧气，流速控制在 $0.5\mathrm{L / min}\sim 0.7\mathrm{L / min}$ ，赶净装置内空气。  
5.2.3 接通高温炉电源。使氧化室的温度升至 $700^{\circ}\mathrm{C}$ ，再加热燃烧室，当温度升至 $100^{\circ}\mathrm{C}$ 时，就有水分流入接收瓶。保持这个温度，直到水分流出速度变慢时再缓慢升温。  
5.2.4 当温度升到 $200^{\circ}\mathrm{C} \sim 300^{\circ}\mathrm{C}$ 时，升温要尽可能慢，并仔细观察通氧情况。一般燃烧室温度升至 $500^{\circ}\mathrm{C}$ 以上就无馏分流出。  
5.2.5 控制温度在 $600^{\circ}\mathrm{C}$ ，继续燃烧一段时间，使食品样品完全氧化，然后切断电源，停止加热和通气。燃烧室产生的气体经氧化室被氧化，水蒸气通过冷凝管收集于接收瓶。

# 5.3 水样纯化

所收集的水经测量总体积后转入 $500~\mathrm{mL}$ 蒸馏瓶，加入 $20\mathrm{g}\sim 30\mathrm{g}$ 过硫酸钾，氧化回流约 $2\textrm{h}$ ，若溶液仍带色，可再加入 $10\mathrm{g}$ 左右过硫酸钾后回流 $2\textrm{h}$ 。重复氧化回流操作直至完全褪色。将蒸馏瓶接入蒸馏装置蒸馏，所得的水密封在磨口烧瓶内。

# 5.4 电解浓集

5.4.1 电解过程是在图2装置内进行。记录电解前纯化过的水样体积并配成 $1\%$ 过氧化钠溶液作为电

解液。  
5.4.2 电解前镍电极（见图3）应事先浸泡在热稀磷酸溶液中数分钟，取出后用水冲洗烘干，然后装入电解池（见图2）。  
5.4.3 电解时电流密度为 $65 \mathrm{~mA} / \mathrm{cm}^{2}$ ，用自来水冷却，每次电解样品水的同时，可在电解池的对称位置电解两个加有标准 ${ }^{3} \mathrm{H}$ 水与样品等体积的水样，以测定电解过程 ${ }^{3} \mathrm{H}$ 的回收率。  
5.4.4 电解直到电解液体积缩小10倍左右结束，记录电解后体积。  
5.4.5 电解完毕后，直接蒸馏样品三次，把浓集了 ${}^{3}\mathrm{H}$ 的水从电解液中分离出来。  
5.4.6 电解的样品水的浓集系数按式(1)计算：

$$
P = \frac {V _ {\mathrm {i}}}{V _ {\mathrm {e}}} \tag {1}
$$

式中：

$P$ —— 浓集系数；

$V_{i}$ ——电解前样品水的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{\mathrm{e}}$ ——电解后样品水的体积，单位为毫升 $(\mathrm{mL})$ 。

5.4.7 将标准 ${}^{3}\mathrm{H}$ 水按 $5.4.1\sim 5.4.5$ 的步骤处理后按式(2)计算电解过程的 ${}^{3}\mathrm{H}$ 的回收率。

$$
Y = \frac {I _ {\mathrm {e}} V _ {\mathrm {e}}}{I _ {\mathrm {i}} V _ {\mathrm {i}}} \tag {2}
$$

式中：

$Y$ ——电解过程 ${}^{3}\mathrm{H}$ 的回收率；

$I_{\mathrm{e}}$ ——标准 ${}^{3}\mathrm{H}$ 水电解后的放射性活度，单位为贝可（Bq）；

$V_{\mathrm{e}}$ ——标准 ${}^{3}\mathrm{H}$ 水电解后的体积，单位为毫升 $(\mathrm{mL})$

$I_{\mathrm{i}}$ ——标准 ${}^{3}\mathrm{H}$ 水电解前的放射性活度，单位为贝可(Bq)；

$V_{\mathrm{i}}$ ——标准 ${}^{3}\mathrm{H}$ 水电解前的体积，单位为毫升（ $\mathrm{mL}$ ）。

# 5.5 样品制备

# 5.5.1 制备待测样品

准确吸取浓集后水样 $2.00\mathrm{mL}$ 和 $8.00\mathrm{mL}$ 闪烁液混匀，放入 $10\mathrm{mL}$ 样品计数瓶中，旋紧瓶盖，振荡混合均匀保存备用。

# 5.5.2 制备本底样品

将本底水按5.4电解蒸馏后，准确吸取蒸馏后水样 $2.00\mathrm{mL}$ 和 $8.00\mathrm{mL}$ 闪烁液混匀，放入 $10\mathrm{mL}$ 样品计数瓶中，旋紧瓶盖，振荡混合均匀后保存备用。

# 5.5.3 制备标准样品

准确吸取 $^{3}\mathrm{H}$ 标准溶液水样 $2.00~\mathrm{mL}$ 和 $8.00~\mathrm{mL}$ 闪烁液，放入 $10~\mathrm{mL}$ 样品计数瓶中，旋紧瓶盖，振荡混合均匀后保存备用。

# 5.5.4 避光保存

把制备好的样品（包括本底样品、待测样品和标准样品），用酒精湿棉球擦拭清洁瓶体后，同时放入液体闪烁计数器的样品室中，避光 $12\mathrm{h}$ 以上（一般是当天制的样品放入样品室，于第二天测量）。

# 5.5.5 防止交叉污染

在操作每一个可能引起样品间交叉污染的步骤中，要注意避免交叉污染。操作要按先低水平，后高

水平顺序进行。

# 5.6 测量

# 5.6.1 仪器准备

调试仪器达到正常工作状态。选择并确定 ${}^{3}\mathrm{H}$ 测量的能量道宽，使仪器的测量道对所测样品的灵敏度达到最大。

# 5.6.2 测量时长的确定

低水平样品需要长的测量时间，与所需的测量精度有关，一般不少于 $1000\mathrm{min}$ 。计数时间应能保证样品计数的统计涨落能满足测量要求。

# 5.6.3 本底计数率的测定

对制备的本底样品以确定的计数时间进行测量。对于低水平样品的测量，本底样品的计数时间应不少于 $1000\mathrm{min}$ 。

# 5.6.4 仪器效率的测定

对制备的标准样品进行测量，按式(3)计算仪器的计数效率：

$$
\varepsilon = \frac {N _ {\mathrm {s}} - N _ {\mathrm {b}}}{6 0 A _ {\mathrm {s}}} \tag {3}
$$

式中：

$\varepsilon$ ——仪器的计数效率；

$N_{\mathrm{s}}$ ——标准样品计数率，单位为计数每分（cpm）；

$N_{\mathrm{b}}$ ——本底样品计数率，单位为计数每分（cpm）；

$A_{s}$ ——加入到标准样品中 ${}^{3}\mathrm{H}$ 的活度，单位为贝可（Bq），应校正到检测时的活度，样品闪烁液混合物中的淬灭可能导致计数效率的降低。可利用内标准法或外标准道比法进行淬灭校正。

# 5.6.5 测量样品

应按如下顺序进行样品的测量：本底样品一待测样品一标准样品，每个样品的测量时长应一致。

# 5.7 空白试验

在试剂更换或检测不同批次样品时应进行空白试验。

# 6 分析结果的表述

食品中 ${}^{3}\mathrm{H}$ 的放射性活度浓度按式(4)计算：

$$
A = \frac {\left(N _ {\mathrm {t}} - N _ {\mathrm {b}}\right) V}{6 0 V _ {\mathrm {t}} P Y \varepsilon W} \tag {4}
$$

式中：

$A$ ——测量样品中 ${}^{3}\mathrm{H}$ 的放射性活度浓度，单位为贝可每千克 $(\mathrm{Bq/kg})$ ；

$N_{\mathrm{t}}$ ——测量样品的计数率，单位为计数每分（cpm）；

$N_{\mathrm{b}}$ ——本底样品计数率，单位为计数每分（cpm）；

$V$ ——燃烧过程收集的样品水体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{\mathrm{t}}$ ——测量的样品水体积，单位为毫升 $(\mathrm{mL})$ ；

$P$ —— 浓集系数；

Y 电解过程 ${}^{3}\mathrm{H}$ 的回收率；

$\varepsilon$ ——仪器的计数效率；

$W$ ——分析样品的鲜样量，单位为千克 $(\mathrm{kg})$ 。

# 7 其他

# 前言

本标准代替GB/T5413.23-1997《婴幼儿配方食品和乳粉碘的测定》

本标准与GB/T5413.23-1997相比，主要变化如下：

——填充色谱柱改为石英毛细管柱；

——增加了检出限。

本标准的附录A为资料性附录。

本标准所代替标准的历次版本发布情况为：

GB5413-1985、GB/T5413.23-1997。

# 食品安全国家标准婴幼儿食品和乳品中碘的测定

# 1 范围

本标准规定了婴幼儿食品和乳品中碘的测定方法。

本标准适用于婴幼儿食品和乳品中碘的测定。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本标准。

# 3 原理

试样中的碘在硫酸条件下与丁酮反应生成丁酮与碘的衍生物，经气相色谱分离，电子捕获检测器检测，外标法定量。

# 4 试剂和材料

除非另有规定，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

4.1 高峰氏（Taka-Diastase）淀粉酶：酶活力 $\geqslant 1.5\mathrm{U / mg}$   
4.2 碘化钾（KI）或碘酸钾（ $\mathrm{KIO}_3$ ）：优级纯。  
4.3 丁酮 $\mathrm{(C_4H_8O)}$ ：色谱纯。  
4.4 硫酸 $\mathrm{(H_2SO_4)}$ ：优级纯。  
4.5 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ 。  
4.6 无水硫酸钠 $\mathrm{(Na_2SO_4)}$   
4.7 双氧水（3.5%）：吸取 $11.7\mathrm{mL}$ 体积分数为 $30\%$ 的双氧水稀释至 $100\mathrm{mL}$ 。  
4.8 亚铁氰化钾溶液（ $109\mathrm{g / L}$ ）：称取 $109\mathrm{g}$ 亚铁氰化钾，用水定容于 $1000\mathrm{mL}$ 容量瓶中。  
4.9 乙酸锌溶液（ $219\mathrm{g / L}$ ）：称取 $219\mathrm{g}$ 乙酸锌，用水定容于 $1000\mathrm{mL}$ 容量瓶中。  
4.10 碘标准溶液。  
4.10.1 碘标准贮备液 $(1.0\mathrm{mg / mL})$ ：称取 $131\mathrm{mg}$ 碘化钾（4.2）（精确至 $0.1\mathrm{mg}$ ）或 $168.5\mathrm{mg}$ 碘酸钾（4.2）（精确至 $0.1\mathrm{mg}$ ），用水溶解并定容至 $100\mathrm{mL}$ ， $5^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 冷藏保存，一个星期内有效。  
4.10.2 碘标准工作液（ $1.0\mu \mathrm{g} / \mathrm{mL}$ ）：吸取 $10~\mathrm{mL}$ 碘标准贮备液，用水定容至 $100~\mathrm{mL}$ 混匀，再吸取 $1.0~\mathrm{mL}$ ，用水定容至 $100~\mathrm{mL}$ 混匀，临用前配制。

# 5 仪器和设备

5.1 天平：感量为 $0.1\mathrm{mg}$   
5.2 气相色谱仪，带电子捕获检测器。

# 6 分析步骤

# 6.1 试样处理

# 6.1.1 不含淀粉的试样

称取混合均匀的固体试样 $5\mathrm{g}$ ，液体试样 $20\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）于 $150\mathrm{mL}$ 三角瓶中，固体试样用 $25\mathrm{mL}$ 约 $40^{\circ}\mathrm{C}$ 的热水溶解。

# 6.1.2 含淀粉的试样

称取混合均匀的固体试样 $5\mathrm{g}$ ，液体试样 $20\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ），于 $150\mathrm{mL}$ 三角瓶中，加入 $0.2\mathrm{g}$ 高峰氏淀粉酶(4.1)，固体试样用 $25\mathrm{mL}$ 约 $40^{\circ}\mathrm{C}$ 的热水充分溶解，置于 $50^{\circ}\mathrm{C} \sim 60^{\circ}\mathrm{C}$ 恒温箱中酶解 $30\mathrm{min}$ ，取出冷却。

# 6.2 试样测定液的制备

6.2.1 沉淀：将上述处理过的试样溶液转入 $100 \mathrm{~mL}$ 容量瓶中，加入 $5 \mathrm{~mL}$ 亚铁氰化钾溶液（4.8）和 $5 \mathrm{~mL}$ 乙酸锌溶液（4.9）后，用水定容至刻度，充分振摇后静止 $10 \mathrm{~min}$ 。滤纸过滤后吸取滤液 $10 \mathrm{~mL}$ 于 $100 \mathrm{~mL}$ 分液漏斗中，加 $10 \mathrm{~mL}$ 水。  
6.2.2 衍生与提取：向分液漏斗中加入 $0.7\mathrm{mL}$ 硫酸（4.4）， $0.5\mathrm{mL}$ 丁酮（4.3）， $2.0\mathrm{mL}$ 双氧水（4.7），充分混匀，室温下保持 $20\mathrm{min}$ 后加入 $20\mathrm{mL}$ 正己烷（4.5）振荡萃取 $2\mathrm{min}$ 。静止分层后，将水相移入另一分液漏斗中，再进行第二次萃取。合并有机相，用水洗涤两到三次。通过无水硫酸钠过滤脱水后移入 $50\mathrm{mL}$ 容量瓶中用正己烷定容，此为试样测定液。

# 6.3 碘标准测定液的制备

分别吸取 $1.0\mathrm{mL}$ ， $2.0\mathrm{mL}$ ， $4.0\mathrm{mL}$ ， $8.0\mathrm{mL}$ ， $12.0\mathrm{mL}$ 碘标准工作液（4.10.2），相当于 $1.0\mu \mathrm{g}$ 、 $2.0\mu \mathrm{g}$ 、 $4.0\mu \mathrm{g}$ 、 $8.0\mu \mathrm{g}$ 、 $12.0\mu \mathrm{g}$ 的碘，其他分析步骤同6.2。

# 6.4 测定

# 6.4.1 参考色谱条件

色谱柱：填料为 $5\%$ 氰丙基-甲基聚硅氧烷的毛细管柱（柱长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ）或具同等性能的色谱柱。

进样口温度： $260^{\circ}C$ 。

ECD 检测器温度： $300^{\circ}\mathrm{C}$ 。

分流比：1：1。

进样量： $1.0~\mu \mathrm{L}$

# 6.4.2 标准曲线的制作

将碘标准测定液（6.3）分别注入到气相色谱仪中（色谱图参见附录A）得到标准测定液的峰面积（或峰高）。以标准测定液的峰面积（或峰高）为纵坐标，以（6.3）碘标准工作液中碘的质量为横坐标制作标准曲线。

# 6.4.3 试样溶液的测定

将试样测定液（6.2）注入到气相色谱仪中得到峰面积（或峰高），从标准曲线中获得试样中碘的含量 $(\mu \mathrm{g})$ 。

# 7 分析结果的表述

试样中碘含量按式（1）计算：

$$
X = \frac {C _ {s}}{m} \times 1 0 0 \dots \tag {1}
$$

式中：

$X$ ——试样中碘含量，单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$ ；

$C_{s}$ ——从标准曲线中获得试样中碘的含量，单位为微克（ $\mu \mathrm{g}$ ）；

$m$ ——试样的质量，单位为克（g）。

以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留至小数点后一位。

# 8 精密度

在重复条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 9 其他

本标准检出限为 $2.0\mu \mathrm{g} / 100\mathrm{g}$

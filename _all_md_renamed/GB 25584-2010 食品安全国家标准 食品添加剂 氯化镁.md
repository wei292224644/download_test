# 食品添加剂 氯化镁

# 1 范围

本标准适用于以水氯镁石或直接用制盐母液为原料经加工制成的食品添加剂氯化镁。

# 2 规范性引用文件

本标准中引用的文件对于本标准的应用是必不可少的。凡是注日期的引用文件，仅所注日期的版本适用于本标准。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本标准。

# 3 分子式和相对分子质量

# 3.1 分子式

$$
\mathrm {M g C l} _ {2} \cdot n \mathrm {H} _ {2} \mathrm {O} (n = 0, 6)
$$

# 3.2 相对分子质量

95.20 $(n = 0)$ （按2007年国际相对原子质量）

203.3 $(n = 6)$ （按2007年国际相对原子质量）

# 检验方法

# A.1 警示

本标准的检验方法中使用的部分试剂具有腐蚀性，操作者须小心谨慎！如溅到皮肤上应立即用水冲洗，严重者应立即就医。使用有挥发性的有机溶剂的操作应在通风橱中进行。

# A.2 一般规定

本标准的检验方法中所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682—2008中规定的三级水。试验中所需标准滴定溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按HG/T3696.1、HG/T3696.2、HG/T3696.3之规定制备。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 氨水溶液： $2 + 3$   
A.3.1.2 氯化铵溶液： $100\mathrm{g / L}$   
A.3.1.3 碳酸铵溶液： $100\mathrm{g / L}$   
A.3.1.4 硝酸银溶液： $42\mathrm{g / L}$   
A.3.1.5 磷酸氢二钠溶液 $(\mathrm{Na_2HPO_3}\cdot 12\mathrm{H_2O})$ ： $160~\mathrm{g / L}$

# A.3.2 分析步骤

# A.3.2.1 镁离子的鉴别

取约 $0.2\mathrm{g}$ 试样，加 $10~\mathrm{mL}$ 水溶解。加入氯化铵溶液和碳酸铵溶液不产生沉淀，再加入磷酸氢二钠溶液时，应产生白色沉淀。分离沉淀，加氨水溶液，沉淀应不溶解。

# A.3.2.2 氯化物的鉴别

取约 $0.2\mathrm{g}$ 试样，加 $10~\mathrm{mL}$ 水溶解，加硝酸银溶液即产生白色沉淀，此沉淀不溶于硝酸，溶于过量的氨水溶液中。

# A.4 氯化镁的测定

# A.4.1 方法提要

用三乙醇胺掩蔽少量三价铁、三价铝和二价锰等离子，在 $\mathrm{pH}$ 为10时，以铬黑T作指示剂，用乙二胺四乙酸二钠标准滴定溶液滴定钙镁含量，从中减去钙含量，计算出氯化镁含量。

# A.4.2 试剂和材料

A.4.2.1 盐酸溶液： $1 + 1$   
A.4.2.2 三乙醇胺溶液： $1 + 3$   
A.4.2.3 氨-氯化铵缓冲溶液（甲） $(\mathrm{pH}\approx 10)$ 。  
A.4.2.4 硝酸银溶液： $10\mathrm{g / L}$   
A.4.2.5 乙二胺四乙酸二钠标准滴定溶液： $c(\mathrm{EDTA}) = 0.02\mathrm{mol / L}$   
A.4.2.6 铬黑T指示剂。

# A.4.3 分析步骤

# A.4.3.1 试验溶液A的制备

称取约 $10\mathrm{g}$ 试样，精确至 $0.0002\mathrm{g}$ ，置于 $250~\mathrm{mL}$ 烧杯中，用 $20~\mathrm{mL}$ 水溶解。加 $1\mathrm{mL}$ 盐酸溶液，加热至沸，并微沸 $1\mathrm{min} \sim 2\mathrm{min}$ ，冷却后转移至 $250~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀，此溶液为试验溶液A。保留此溶液用于氯化镁含量、钙含量的测定。

# A.4.3.2 测定

移取 $25.00\mathrm{mL}$ 试验溶液A，置于 $250~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。移取 $25.00~\mathrm{mL}$ 上述溶液，置于 $250~\mathrm{mL}$ 锥形瓶中，加入 $50~\mathrm{mL}$ 水、 $5\mathrm{mL}$ 三乙醇胺溶液、 $10~\mathrm{mL}$ 氨-氯化铵缓冲溶液甲和 $0.1\mathrm{g}$ 铬黑T指示剂，用乙二胺四乙酸二钠标准滴定溶液滴定至溶液由紫红色变为纯蓝色。

# A.4.4 结果计算

A.4.4.1 氯化镁含量以六水氯化镁 $(\mathrm{MgCl}_2\cdot 6\mathrm{H}_2\mathrm{O})$ 的质量分数 $\varpi_{1}$ 计，数值以 $\%$ 表示，按公式(A.1)计算：

$$
w _ {1} = \frac {\left[ \left(V _ {1} - V _ {2} / 20\right) / 1000 \right] c M _ {1}}{m \times (25 / 250) \times (25 / 250)} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——滴定试验溶液所消耗的乙二胺四乙酸二钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——A.5滴定钙所消耗的乙二胺四乙酸二钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——乙二胺四乙酸二钠标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——试料的质量的数值，单位为克 $(\mathrm{g})$ ；

$M_{\mathrm{I}}$ ——六水氯化镁 $(\mathrm{MgCl}_2 \cdot 6\mathrm{H}_2\mathrm{O})$ 的摩尔质量的数值，单位为克每摩尔 $(\mathrm{g/mol})(M_1 = 203.3)$ 。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.2\%$ 。

A.4.4.2 氯化镁含量以氯化镁 $(\mathrm{MgCl_2})$ 的质量分数 $\omega_{2}$ 计，数值以 $\%$ 表示，按公式(A.2)计算：

$$
w _ {2} = \frac {\left[ \left(V _ {1} - V _ {2} / 20\right) / 1000 \right] c M _ {2}}{m \times (25 / 250) \times (25 / 250)} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$V_{1}$ ——滴定试验溶液所消耗的乙二胺四乙酸二钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——A.5滴定钙所消耗的乙二胺四乙酸二钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$

$c$ ——乙二胺四乙酸二钠标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——试料的质量的数值，单位为克(g)；

$M_2$ ——氯化镁 $(\mathrm{MgCl}_2)$ 摩尔质量的数值，单位为克每摩尔 $(\mathrm{g / mol})$ $(M_2 = 95.20)$ 。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.1\%$ 。

# A.5 钙的测定

# A.5.1 试剂和材料

A.5.1.1 氢氧化钠溶液： $100\mathrm{g / L}$

A.5.1.2 三乙醇胺溶液： $1 + 3$

A.5.1.3 乙二胺四乙酸二钠标准滴定溶液： $c(\mathrm{EDTA}) = 0.02\mathrm{mol / L}$

A.5.1.4 钙试剂羧酸钠盐指示剂。

# A.5.2 仪器和设备

微量滴定管：分度值为 $0.02\mathrm{mL}$

# A.5.3 分析步骤

移取 $50.00\mathrm{mL}$ 试验溶液A，置于 $250~\mathrm{mL}$ 锥形瓶中，加入 $30~\mathrm{mL}$ 水， $5\mathrm{mL}$ 三乙醇胺溶液，摇动下滴加氢氧化钠溶液，当溶液刚出现沉淀物时，加入 $0.1\mathrm{g}$ 钙试剂羧酸钠盐指示剂，继续滴加氢氧化钠溶液至溶液由蓝色变为酒红色，过量 $0.5\mathrm{mL}$ 。用乙二胺四乙酸二钠标准滴定溶液滴定至溶液由酒红色变为纯蓝色。

# A.5.4 结果计算

钙含量以钙(Ca)的质量分数 $w_{3}$ 计，数值以 $\%$ 表示，按公式(A.3)计算：

$$
w _ {3} = \frac {\left(V _ {2} / 1 0 0 0\right) c M}{m \times 5 0 / 2 5 0} \times 1 0 0 \tag {A.3}
$$

式中：

$V_{2}$ ——滴定试验溶液所消耗的乙二胺四乙酸二钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——乙二胺四乙酸二钠标准滴定溶液浓度的准确数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$m$ ——试料的质量的数值，单位为克(g)；

$M$ ——钙(Ca)摩尔质量的数值，单位为克每摩尔 $(\mathrm{g / mol})(M = 40.01)$ 。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.02\%$ 。

# A.6 硫酸盐的测定

# A.6.1 试剂和材料

A.6.1.1 盐酸溶液： $1 + 1$   
A.6.1.2 硝酸溶液： $1 + 4$   
A.6.1.3 氯化钡溶液： $100\mathrm{g / L}$   
A.6.1.4 硝酸银溶液： $20~\mathrm{g / L}$   
A.6.1.5 甲基红指示液： $2\mathrm{g} / \mathrm{L}$

# A.6.2 仪器和设备

高温炉：能控制温度 $850^{\circ}\mathrm{C} \pm 50^{\circ}\mathrm{C}$ 。

# A.6.3 分析步骤

# A.6.3.1 试验溶液B的制备

称取约 $25\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加适量水溶解，加 $2\mathrm{mL}$ 盐酸溶液，置于电炉上加热至沸，并保持微沸 $1\mathrm{min} \sim 2\mathrm{min}$ 。冷却后转移至 $250\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。用中速定性滤纸干过滤，弃去初滤液 $20\mathrm{mL}$ 。滤液作为试验溶液B。

# A.6.3.2 测定

移取 $25.00\mathrm{mL}$ 试验溶液B，置于 $250~\mathrm{mL}$ 烧杯中，加水至 $50~\mathrm{mL}$ ，加 $1\mathrm{mL}$ 盐酸溶液，在电炉上加热煮沸，在搅拌下滴加 $5\mathrm{mL}$ 氯化钡溶液，滴加时间约为 $1\mathrm{min}$ 。继续搅拌并微沸 $2\mathrm{min} \sim 3\mathrm{min}$ ，然后盖上表面皿，保持微沸 $5\mathrm{min}$ 。取下静置过夜。用慢速定量滤纸过滤，用淀帚擦拭烧杯壁，用温水洗净烧杯和沉淀，洗涤至无氯离子为止（取 $5\mathrm{mL}$ 洗涤液，加 $1\mathrm{mL}$ 硝酸溶液，用硝酸银溶液检验）。

将沉淀连同滤纸转移至已于 $800^{\circ}\mathrm{C} \pm 50^{\circ}\mathrm{C}$ 下灼烧至质量恒定的瓷坩埚中，在电炉上灰化后，置于高温炉中，于 $800^{\circ}\mathrm{C} \pm 50^{\circ}\mathrm{C}$ 灼烧至质量恒定。

# A.6.4 结果计算

硫酸盐含量以硫酸根 $(\mathrm{SO}_4)$ 的质量分数 $\omega_{4}$ 计，数值以 $\%$ 表示，按公式(A.4)计算：

$$
w _ {4} = \frac {\left(m _ {2} - m _ {1}\right) \times 0.4116}{m \times (25 / 250)} \times 100 \% \quad \dots \tag{A.4}
$$

式中：

$m_{2}$ ——坩埚和沉淀的质量的数值，单位为克 $(\mathrm{g})$ ；

$m_{1}$ ——空坩埚的质量的数值，单位为克 $(\mathrm{g})$ ；

0.4116 ——硫酸钡换算成硫酸根的系数；

$m$ ——试料的质量的数值，单位为克 $(\mathrm{g})$ 。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.05\%$ 。

# A.7 水不溶物的测定

# A.7.1 试剂和材料

A.7.1.1 硝酸溶液： $1 + 4$

A.7.1.2 硝酸银溶液： $17\mathrm{g / L}$

# A.7.2 仪器和设备

玻璃砂坩埚：滤板孔径 $5\mu \mathrm{m}\sim 15\mu \mathrm{m}$

# A.7.3 分析步骤

称取约 $10\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $500~\mathrm{mL}$ 烧杯中，加 $150~\mathrm{mL}$ 水，在不断搅拌下加热近沸使样品全部溶解，静置温热 $10\mathrm{min}$ 。用预先于 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 下干燥至质量恒定的玻璃砂坩埚过滤。用热水洗至无氯离子为止（取 $5\mathrm{mL}$ 洗涤液，加 $1\mathrm{mL}$ 硝酸，用硝酸银溶液检查）。在 $105^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 下干燥至质量恒定。

# A.7.4 结果计算

水不溶物以质量分数 $\omega_{5}$ 计，数值以 $\%$ 表示，按公式(A.5)计算：

$$
w _ {5} = \frac {m _ {2} - m _ {1}}{m} \times 100 \% \tag{A.5}
$$

式中：

$m_{1}$ ——玻璃砂坩埚质量的数值，单位为克 $(\mathrm{g})$

$m_{2}$ ——水不溶物和玻璃砂坩埚质量的数值，单位为克 $(\mathrm{g})$ ；

$m$ ——试料的质量的数值，单位为克 $(\mathrm{g})$ 。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $0.02\%$ 。

# A.8 色度的测定

# A.8.1 试剂和材料

500黑曾单位铂-钴标准溶液。

其他同GB/T3143—1982的4.1。

# A.8.2 仪器和设备

比色管：容积 $50~\mathrm{mL}$

# A.8.3 分析步骤

# A.8.3.1 铂-钴标准溶液的制备

移取 $3.00\mathrm{mL}500$ 黑曾单位铂-钴标准溶液，置于比色管中，用水稀释至刻度。

# A.8.3.2 测定

称取 $50.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $100~\mathrm{mL}$ 烧杯中，加约 $40~\mathrm{mL}$ 水溶解。转移至比色管中，用水稀释至刻度，摇匀，放置 $40\mathrm{min}$ 消除气泡。将试样溶液比色管与标准溶液比色管置于白色背景下，沿比色管轴线方向用目测比较，试样溶液所产生的色度不得大于标准溶液所产生的色度。

# A.9 铅的测定

# A.9.1 试剂和材料

A.9.1.1 三氯甲烷。  
A.9.1.2 硝酸。  
A.9.1.3 盐酸溶液： $1 + 4$   
A.9.1.4 氢氧化钠溶液： $100\mathrm{g / L}$   
A.9.1.5 吡咯烷二硫代氨基甲酸铵(APDC)溶液： $20\mathrm{g / L}$

称取 $2.00\mathrm{g}\pm 0.01\mathrm{g}$ 吡咯烷二硫代氨基甲酸铵（APDC)溶于 $100~\mathrm{mL}$ 水中。如有不溶物，使用前过滤。

A.9.1.6 铅标准溶液： $1\mathrm{mL}$ 溶液含铅 $(\mathrm{Pb})0.010\mathrm{mg}$

用移液管移取 $1.00\mathrm{mL}$ 按HG/T3696.2中配制的铅标准溶液，置于 $100~\mathrm{mL}$ 容量瓶中，加水至刻度，摇匀。该溶液现用现配。

A.9.1.7 精密 $\mathrm{pH}$ 试纸： $0.5\sim 5.0$

# A.9.2 仪器和设备

A.9.2.1 分液漏斗： $250~\mathrm{mL}$

A.9.2.2 原子吸收分光光度计：配有铅空心阴极灯。

# A.9.3 测定步骤

# A.9.3.1 铅标准测定溶液的制备

移取 $1.00\mathrm{mL}$ 铅标准溶液，置于 $150~\mathrm{mL}$ 烧杯中，用盐酸溶液调节溶液的 $\mathrm{pH}$ 为 $1.0\sim 1.5$ （用精密 $\mathrm{pH}$ 试纸检验)。将溶液移入分液漏斗中，用水稀释至约 $200~\mathrm{mL}$ 。加入 $2\mathrm{mL}$ 吡咯烷二硫代氨基甲酸铵(APDC)溶液，摇匀。用三氯甲烷萃取两次，每次加入 $20~\mathrm{mL}$ ，将萃取液(即有机相)收集于 $50~\mathrm{mL}$ 烧杯中，在通风橱中用水浴加热蒸发至干。在残余物中加入 $3\mathrm{mL}$ 硝酸，加热近干。加入 $0.5\mathrm{mL}$ 硝酸和 $10~\mathrm{mL}$ 水，加热至剩余液体体积为 $3\mathrm{mL}\sim 5\mathrm{mL}$ ，移入 $10~\mathrm{mL}$ 容量瓶中，用水稀释至刻度。

# A.9.3.2 试验溶液的制备

称取 $10.00\mathrm{g}\pm 0.01\mathrm{g}$ 试样，置于 $150~\mathrm{mL}$ 烧杯中，加入 $30~\mathrm{mL}$ 水，加入 $10~\mathrm{mL}$ 盐酸溶液，盖上表面皿并加热至沸，沸腾 $5\mathrm{min}$ 。冷却，用氢氧化钠溶液调节 $\mathrm{pH}$ 值为 $1.0\sim 1.5$ （用精密 $\mathrm{pH}$ 试纸检验）。然后按A.9.3.1中从“将溶液移入分液漏斗中……”开始操作。

# A.9.3.3 测定

选用空气-乙炔火焰，于 $283.3\mathrm{nm}$ 波长处，用水调零，使用原子吸收分光光度计测定铅标准测定溶液和试验溶液的吸光度。

# A.9.4 结果判定

试验溶液的吸光度不得大于铅标准测定溶液的吸光度。

# A.11.1 试剂和材料

# A.11.2 分析步骤

# A.11.2.1 工作曲线绘制

移取 $0.00\mathrm{mL}, 0.50\mathrm{mL}, 1.00\mathrm{mL}, 1.50\mathrm{mL}, 2.00\mathrm{mL}$ 铵标准溶液，分别置于 $50\mathrm{mL}$ 比色管中，加入 $1\mathrm{mL}$ 酒石酸钾钠溶液，摇匀。加 $2\mathrm{mL}$ 纳氏试剂，摇匀，用水稀释至刻度，摇匀。放置 $15\mathrm{min}$ ，于分光光度计上，在 $420\mathrm{nm}$ 处，使用 $1\mathrm{cm}$ 比色皿，以水调零测量其吸光度。

从每个标准溶液的吸光度中减去试剂空白溶液的吸光度，以铵的质量 $(\mathrm{mg})$ 为横坐标，对应的吸光度为纵坐标，绘制工作曲线。

# A.11.2.2 测定

称取约 $5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $150~\mathrm{mL}$ 烧杯，加水溶解。转移至 $250~\mathrm{mL}$ 容量瓶中，加 $50~\mathrm{mL}$ 氢氧化钠溶液，加水稀释至刻度，摇匀，静置 $2\mathrm{h}$ 。

移取 $20.00\mathrm{mL}$ 上层清液，置于 $50~\mathrm{mL}$ 比色管中，加 $1\mathrm{mL}$ 酒石酸钾钠溶液，摇匀。加 $2\mathrm{mL}$ 纳氏试剂，摇匀，用水稀释至刻度，摇匀。放置 $15\mathrm{min}$ ，于分光光度计上，在 $420~\mathrm{nm}$ 处，使用 $1\mathrm{cm}$ 比色皿，以水调零测量其吸光度。从工作曲线上查出试验溶液中铵的质量。

# A.11.3 结果计算

铵含量以铵 $\mathrm{(NH_4)}$ 的质量分数 $w_{6}$ 计，数值以 $\mathrm{mg / kg}$ 表示，按公式(A.6)计算：

$$
w _ {6} = \frac {m _ {1}}{m \times (2 0 / 2 5 0) \times 1 0 ^ {- 3}} \quad \dots \tag {A.6}
$$

式中：

$m_{1}$ ——从标准曲线上查得铵质量的数值，单位为毫克 $(\mathrm{mg})$ ；

$m$ ——试料的质量的数值，单位为克 $(\mathrm{g})$ 。

取平行测定结果的算术平均值为测定结果，两次平行测定结果的绝对差值不大于 $10\mathrm{mg / kg}$

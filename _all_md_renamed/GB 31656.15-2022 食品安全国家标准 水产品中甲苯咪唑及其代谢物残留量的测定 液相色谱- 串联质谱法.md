本文件按照GB/T1.1—2020《标准化工作导则第1部分：标准化文件的结构和起草规则》的规定起草。

# 液相色谱-串联质谱法

# 1 范围

本文件规定了水产品中甲苯咪唑及其主要代谢物羟基甲苯咪唑和氨基甲苯咪唑残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鱼、虾和鳖可食组织中甲苯咪唑及其主要代谢物羟基甲苯咪唑和氨基甲苯咪唑残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T 6682 分析实验室用水规则和试验方法

GB/T30891—2014 水产品抽样规范

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的甲苯咪唑、羟基甲苯咪唑和氨基甲苯咪唑，用乙酸乙酯提取，正己烷除脂，用液相色谱串联质谱法检测，内标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的—级水。

# 5.1 试剂

5.1.1 甲酸（HCOOH）：色谱纯。  
5.1.2 乙酸乙酯 $\mathrm{(CH_3COOHC_2H_5)}$ ：色谱纯。  
5.1.3 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
5.1.4 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
5.1.5 二甲基亚砜 $\left[\left(\mathrm{CH}_{3}\right)_{2} \mathrm{SO}\right]$ ：色谱纯。

# 5.2 溶液的配制

5.2.1 $50~\mathrm{mmol / L}$ 磷酸氢二钠溶液：取磷酸氢二钠 $8.95\mathrm{g}$ ，用水溶解并稀释至 $500~\mathrm{mL}$ 。  
5.2.2 $0.1\%$ 甲酸溶液：取 $0.5 \mathrm{~mL}$ 甲酸，加水溶解并稀释至 $500 \mathrm{~mL}$ ，混匀。  
5.2.3 甲醇-0.1%甲酸溶液 $(50:50,V / V)$ ：取甲醇、 $0.1\%$ 的甲酸溶液等体积混匀。

# 5.3 标准品

甲苯咪唑（MBZ），含量 $\geqslant 99.0\%$ ；羟基甲苯咪唑（MBZ-OH），含量 $\geqslant 99.0\%$ ；氨基甲苯咪唑（MBZ-NH₂），含量 $\geqslant 99.0\%$ ；氘代甲苯咪唑（MBZ-D₃），含量 $\geqslant 99.5\%$ ；氘代羟基甲苯咪唑（MBZ-OH-D₃），含量 $\geqslant 99.0\%$ 。具体见附录A。

# 5.4 标准溶液的制备

5.4.1 标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：取甲苯咪唑、羟基甲苯咪唑和氨基甲苯咪唑标准品各 $10\mathrm{mg}$ ，精密称定，分别用 $10~\mathrm{mL}$ 二甲基亚砜溶解，加甲醇稀释定容至 $100~\mathrm{mL}$ 容量瓶，摇匀。 $-18^{\circ}C$ 以下保存，有效期3个月。  
5.4.2 混合标准工作液：精密量取甲苯咪唑、羟基甲苯咪唑和氨基甲苯咪唑标准储备液各 $1\mathrm{mL}$ ，用甲醇-0.1%甲酸溶液稀释配制成浓度为 $1000\mathrm{ng/mL}, 100\mathrm{ng/mL}, 10\mathrm{ng/mL}3$ 个浓度混合标准工作液。现用现配。  
5.4.3 内标标准储备液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：取氘代甲苯咪唑和氘代羟基甲苯咪唑标准品各 $10\mathrm{mg}$ ，精密称定，分别用 $10~\mathrm{mL}$ 二甲基亚砜溶解，用甲醇稀释定容至 $100~\mathrm{mL}$ 容量瓶，摇匀。一 $18^{\circ}C$ 以下保存，有效期3个月。  
5.4.4 混合内标工作液：精密量取氘代甲苯咪唑和氘代羟基甲苯咪唑内标标准储备液各 $1\mathrm{mL}$ ，用甲醇 $0.1\%$ 甲酸溶液稀释成含氘代甲苯咪唑和氘代羟基甲苯咪唑浓度分别为 $50\mathrm{ng/mL}$ 和 $10\mathrm{ng/mL}$ 的混合内标工作液。现用现配。

# 5.5 材料

微孔尼龙滤膜： $0.22\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪(LC-MS/MS)：配电喷雾离子源(ESI)。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 离心机： $8000\mathrm{r / min}$   
6.4 涡旋混合器。  
6.5 超声波清洗仪。  
6.6 氮吹仪。  
6.7 具塞聚丙烯离心管： $50~\mathrm{mL}$

# 7 试样的制备与保存

# 7.1 试样的制备

按GB/T30891—2014附录B的要求制样。

a）取均质后的供试样品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

取试料 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 具塞离心管，准确加入混合内标工作液 $100~\mu \mathrm{L}$ ，涡旋振荡 $2\mathrm{min}$ ，加 $50~\mathrm{mmol / L}$ 的磷酸氢二钠溶液 $5\mathrm{mL}$ ，涡旋振荡 $1\mathrm{min}$ ，加乙酸乙酯 $6\mathrm{mL}$ ，涡旋振荡 $1\mathrm{min}$ $8000\mathrm{r / min}$ 离心 $6\mathrm{min}$ ，收集上清液(乙酸乙酯层)，残渣加乙酸乙酯 $3\mathrm{mL}$ 重复提取1次，合并上清液，备用。

# 8.2 净化

取备用液， $40^{\circ}\mathrm{C}$ 氮气吹干，用甲醇- $0.1\%$ 甲酸溶液 $2.0~\mathrm{mL}$ 溶解残留物，加正己烷 $2\mathrm{mL}$ ，涡旋振荡 $1\mathrm{min},8000\mathrm{r / min}$ 离心 $6\mathrm{min}$ ，弃去上层正己烷层，重复去脂1次，取澄清液，过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱仪测定。

# 8.3 标准曲线的制作

精密量取标准工作液和内标工作液适量，用甲醇- $0.1\%$ 甲酸溶液稀释成甲苯咪唑、氨基甲苯咪唑和羟基甲苯咪唑的浓度均为 $0.5 \mathrm{ng} / \mathrm{mL}, 1 \mathrm{ng} / \mathrm{mL}, 2 \mathrm{ng} / \mathrm{mL}, 5 \mathrm{ng} / \mathrm{mL}, 10 \mathrm{ng} / \mathrm{mL}, 20 \mathrm{ng} / \mathrm{mL}, 50 \mathrm{ng} / \mathrm{mL}, 100 \mathrm{ng} / \mathrm{mL}$ 系列标准工作液（内标氘代甲苯咪唑浓度为 $2.5 \mathrm{ng} / \mathrm{mL}$ 、氘代羟基甲苯咪唑为 $0.5 \mathrm{ng} / \mathrm{mL}$ ），供液相色谱-串联质谱测定。分别以各药物浓度与内标物浓度的比值为横坐标 $(x)$ 、各药物的峰面积与同位素内标峰面积的比值为纵坐标 $(y)$ 绘制标准曲线。求回归方程和相关系数。

# 8.4.3 测定法

# 8.4.3.1 定性测定

在同样测试条件下，试料溶液中甲苯咪唑及其代谢物的保留时间与标准工作液中的保留时间之比，偏差在 $\pm 2.5\%$ 以内，且检测到的离子相对丰度，应当与浓度相当的校正标准溶液相对丰度一致，其允许偏差应符合表3的要求。

# 8.4.3.2 定量测定

取试料溶液和标准工作液，作单点或多点校准，甲苯咪唑以氘代甲苯咪唑为内标，羟基甲苯咪唑和氨基甲苯咪唑以氘代羟基甲苯咪唑为内标，以色谱峰面积定量，内标法计算。标准溶液及试料溶液中目标药物的特征离子质量色谱峰峰面积均应在仪器检测的线性范围之内。标准溶液特征离子质量色谱图见附录B。

# 8.5 空白试验

取空白试料，除不加药物外，采用完全相同的测定步骤进行平行操作。

# 9 结果计算和表述

试样中待测物的残留量按标准曲线或公式(1)计算。

$$
X \frac {A _ {i} \times A _ {i s} ^ {\prime} \times C _ {s} \times C _ {i s} \times V}{A _ {i s} \times A _ {s} \times C _ {i s} ^ {\prime} \times m}. \tag {1}
$$

式中：

$X$ ——试样中被测物质残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{\mathrm{s}}$ ——标准工作溶液中被测物质浓度的数值；单位为纳克每毫升 $(\mathrm{ng} / \mathrm{mL})$ ；

$C_{is}$ ——试样溶液中内标浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$C_{is}^{\prime}$ ——标准工作溶液中内标浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A_{i}$ ——试样溶液中被测物质的峰面积；

$A_{is}^{\prime}$ ——标准工作溶液中内标的峰面积；

$A_{is}$ ——试样溶液中内标的峰面积；

$A_{\mathrm{s}}$ ——标准工作溶液中被测物质的峰面积；

$V$ ——试样溶液定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——供试试样质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法灵敏度、准确度、精密度

# 10.1 灵敏度

本方法甲苯咪唑、氨基甲苯咪唑和羟基甲苯咪唑的检测限均为 $0.5\mu \mathrm{g / kg}$ ，定量限为 $1\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法添加浓度为 $1\mu \mathrm{g / kg}\sim 40~\mu \mathrm{g / kg}$ 时，回收率均为 $70\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

# 食品安全国家标准

# 水产品中辛基酚、壬基酚、双酚 A、己烯雌酚、雌酮、17α-乙炔雌二醇、17β-雌二醇、雌三醇残留量的测定 气相色谱-质谱法

# 1 范围

本标准规定了水产品中辛基酚、壬基酚、双酚A、己烯雌酚、雌酮、 $17\alpha$ -乙炔雌二醇、 $17\beta$ -雌二醇、雌三醇残留量检测的制样和气相色谱-质谱测定方法。

本标准适用于鱼、虾、蟹、贝类、海参、鳖等水产品可食组织中辛基酚、壬基酚、双酚A、己烯雌酚、雌酮、17α-乙炔雌二醇、17β-雌二醇、雌三醇残留量的检测。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样中辛基酚、壬基酚、双酚A、己烯雌酚、雌酮、 $17\alpha-$ 乙炔雌二醇、 $17\beta-$ 雌二醇、雌三醇残留经乙酸乙酯提取，凝胶渗透色谱及固相萃取净化，七氟丁酸酐衍生，气相色谱-质谱法测定，外标法定量。

# 4 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 乙酸乙酯（ $\mathrm{CH}_3\mathrm{COOC}_2\mathrm{H}_5$ ）：色谱纯。  
4.1.2 丙酮（ $\mathrm{CH}_3\mathrm{COCH}_3$ ）：色谱纯。  
4.1.3 正己烷（ $\mathrm{C_6H_{14}}$ ）：色谱纯。  
4.1.4 甲醇（ $\mathrm{CH}_3\mathrm{OH}$ ）：色谱纯。  
4.1.5 环己烷（ $\mathrm{C_6H_{12}}$ ）：色谱纯。  
4.1.6 七氟丁酸酐（ $\mathrm{C_8F_{14}O_3}$ ）。  
4.1.7 碳酸钠（ $\mathrm{Na_2CO_3}$ ）。

# 4.2 溶液制备

4.2.1 碳酸钠溶液：称取碳酸钠 $10\mathrm{g}$ ，用水溶解并稀释至 $100\mathrm{mL}$ ，混匀。

4.2.2 $50\%$ 环己烷乙酸乙酯溶液：环己烷与乙酸乙酯等体积混合。  
4.2.3 $50\%$ 甲醇水溶液：甲醇与水等体积混合。

# 4.3 标准品

辛基酚、壬基酚、双酚A、己烯雌酚、雌酮、17α-乙炔雌二醇、17β-雌二醇、雌三醇含量均 $\geqslant 98.0\%$ ，详见附录A。

# 4.4 标准溶液制备

4.4.1 标准贮备液：取辛基酚、壬基酚、双酚 A、己烯雌酚、雌酮、 $17\alpha$ -乙炔雌二醇、 $17\beta$ -雌二醇、雌三醇标准品各约 $10\mathrm{mg}$ ，精密称定，于 $10\mathrm{mL}$ 棕色量瓶，用甲醇溶解并稀释至刻度，配成浓度为 $1\mathrm{mg/mL}$ 的标准储备溶液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期6个月。  
4.4.2 混合标准工作液：分别精密量取标准储备液适量，用甲醇稀释，配成浓度为：辛基酚 $50\mu \mathrm{g} / \mathrm{L}$ ，壬基酚、双酚A $30\mu \mathrm{g} / \mathrm{L}$ ，己烯雌酚 $50\mu \mathrm{g} / \mathrm{L}$ ，雌酮、17α-乙炔雌二醇、17β-雌二醇、雌三醇 $100\mu \mathrm{g} / \mathrm{L}$ 的混合标准工作液。 $2\sim 8^{\circ}\mathrm{C}$ 避光保存，有效期1周。

# 4.5 材料

4.5.1 聚苯乙烯凝胶填料：Bio-Beads S-X3，200～400目。  
4.5.2 HLB固相萃取柱： $60\mathrm{mg / 3mL}$ ，或相当者。  
4.5.3 凝胶净化柱：长 $25 \mathrm{~cm}$ ，内径 $2 \mathrm{~cm}$ ，具活塞玻璃层析柱。将 $50 \%$ 环己烷乙酸乙酯溶液浸泡过夜的聚苯乙烯凝胶填料以湿法装入柱中，柱床高 $20 \mathrm{~cm}$ 。柱床始终保持在 $50 \%$ 环己烷乙酸乙酯溶液中。

# 5 仪器和设备

5.1 气相色谱质谱联用仪：配EI源。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
5.3 均质机。  
5.4 离心机： $4000\mathrm{r / min}$   
5.5 涡旋混合器。  
5.6 氮吹仪。  
5.7 固相萃取装置。  
5.8 聚丙烯离心管： $50~\mathrm{mL}$   
5.9 具塞玻璃离心管： $10\mathrm{mL}$   
5.10 梨形瓶： $100\mathrm{mL}$

# 6 试料的制备与保存

# 6.1 试料的制备

取适量新鲜或解冻的空白或供试组织，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 试料的保存

-18°C以下保存。

# 7 测定步骤

# 7.1 提取

取试料 $5 \mathrm{~g}$ （准确至 $\pm 20 \mathrm{mg}$ ），于 $50 \mathrm{~mL}$ 离心管，加碳酸钠溶液 $3 \mathrm{~mL}$ ，乙酸乙酯 $20 \mathrm{~mL}$ ，漩涡混匀，超声提取 $10 \mathrm{~min}$ ， $4000 \mathrm{r} / \mathrm{min}$ 离心 $10 \mathrm{~min}$ ，取上清液至 $100 \mathrm{~mL}$ 梨形瓶中。残渣用乙酸乙酯 $10 \mathrm{~mL}$ 重复提取一次，合并上清液，于 $40^{\circ} \mathrm{C}$ 旋转蒸发至干，用 $50 \%$ 环己烷乙酸乙酯溶液 $5 \mathrm{~mL}$ 溶解残留物，备用。

# 7.2 净化

# 7.2.1 凝胶净化

将备用液转至凝胶净化柱上，用 $50\%$ 环己烷乙酸乙酯溶液 $110~\mathrm{mL}$ 淋洗，根据凝胶净化洗脱曲线确定收集淋洗液的体积， $40^{\circ}\mathrm{C}$ 旋转蒸干，残渣用甲醇 $1\mathrm{mL}$ 溶解，加水 $9\mathrm{mL}$ 稀释，备用。

凝胶净化柱洗脱曲线的绘制：将 $5\mathrm{mL}$ 混合标准溶液上柱，用 $50\%$ 环己烷乙酸乙酯溶液淋洗，收集淋洗液，每 $10\mathrm{mL}$ 收集一管，于 $40^{\circ}\mathrm{C}$ 水浴中氮吹至干。按7.3的方法衍生，气相色谱-质谱法测定，根据淋洗体积与回收率的关系确定需要收集的淋洗液体积。

# 7.2.2 固相萃取净化

固相萃取柱依次用甲醇 $5 \mathrm{~mL}$ 、水 $5 \mathrm{~mL}$ 活化，取备用液过柱，控制流速不超过 $2 \mathrm{~mL} / \mathrm{min}$ ，用 $50 \%$ 甲醇水溶液 $10 \mathrm{~mL}$ 淋洗，抽干，用甲醇 $10 \mathrm{~mL}$ 洗脱，控制流速不超过 $2 \mathrm{~mL} / \mathrm{min}$ 。收集洗脱液于 $10 \mathrm{~mL}$ 具塞玻璃离心管中，于 $40^{\circ} \mathrm{C}$ 水浴中氮气吹干。

# 7.3 衍生

于上述具塞玻璃离心管中加入七氟丁酸酐 $30 \mu \mathrm{L}$ 、丙酮 $70 \mu \mathrm{L}$ ，盖紧盖，漩涡混合 $30 \mathrm{~s}$ ，于 $30^{\circ} \mathrm{C}$ 恒温箱中衍生 $30 \mathrm{~min}$ ，氮气吹干，精密加入正己烷 $0.5 \mathrm{~mL}$ ，旋涡混合 $10 \mathrm{~s}$ ，溶解残余物，供GC-MS分析。

# 7.4 标准曲线的制备

取混合标准工作溶液 $50 \mu \mathrm{L} 、 100 \mu \mathrm{L} 、 200 \mu \mathrm{L} 、 500 \mu \mathrm{L} 、 1000 \mu \mathrm{L}$ 于 $1.5 \mathrm{~mL}$ 样品反应瓶中， $40^{\circ} \mathrm{C}$ 水浴中氮吹至干，按7.3方法衍生，制成辛基酚、己烯雌酚浓度均为 $5 \mu \mathrm{g} / \mathrm{L} 、 10 \mu \mathrm{g} / \mathrm{L} 、 25$

$\mu \mathrm{g} / \mathrm{L}$ 、 $50~\mu \mathrm{g} / \mathrm{L}$ 、 $100~\mu \mathrm{g} / \mathrm{L}$ 的梯度系列，壬基酚、双酚A浓度均为 $3\mu \mathrm{g} / \mathrm{L}$ 、 $6\mu \mathrm{g} / \mathrm{L}$ 、 $15\mu \mathrm{g} / \mathrm{L}$ 、 $30$ $\mu \mathrm{g} / \mathrm{L}$ 、 $60~\mu \mathrm{g} / \mathrm{L}$ 的梯度系列，雌酮、 $17\alpha-$ 乙炔雌二醇、 $17\beta-$ 雌二醇、雌三醇浓度均为 $10~\mu \mathrm{g} / \mathrm{L}$ 、 $20~\mu \mathrm{g} / \mathrm{L}$ 、 $50~\mu \mathrm{g} / \mathrm{L}$ 、 $100~\mu \mathrm{g} / \mathrm{L}$ 、 $200~\mu \mathrm{g} / \mathrm{L}$ 的梯度系列。分别取 $1\mu \mathrm{L}$ 进样，以定量离子峰面积为纵坐标，浓度为横坐标，绘制标准曲线。

# 7.5 测定

# 7.5.1 色谱参考条件

a) 色谱柱: HP-5ms石英毛细管柱 $(30 \mathrm{~m} \times 0.25 \mathrm{~mm} \times 0.25 \mu \mathrm{m})$ , 或相当者;   
b）载气：高纯氦气，纯度 $\geq 99.999\%$ ，流速 $1.0~\mathrm{mL / min}$   
c）进样方式：无分流进样；  
d）进样量： $1\mu \mathrm{L}$   
e）进样口温度： $250^{\circ}\mathrm{C}$   
f）柱温：初始柱温 $120^{\circ}\mathrm{C}$ ，保持 $2\mathrm{min}$ ，以 $15^{\circ}\mathrm{C / min}$ 升至 $250^{\circ}\mathrm{C}$ ，再以 $5^{\circ}\mathrm{C / min}$ 升至 $300^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$ 。

# 7.5.2 质谱参考条件

a）离子源：EI源；  
b）离子源温度： $230^{\circ}\mathrm{C}$   
c）四极杆温度： $150^{\circ}\mathrm{C}$   
d）接口温度： $280^{\circ}\mathrm{C}$   
e）溶剂延迟：7min；  
f) 选择离子监测 (SIM): 辛基酚、壬基酚、双酚A、己烯雌酚、雌酮、 $17\alpha$ -乙炔雌二醇、 $17\beta$ -雌二醇、雌三醇衍生物的监测离子见表1。

# 8 结果计算和表述

试料中待测药物残留量按式（1）计算：

$$
X = \frac {C _ {s} \times A \times V}{A _ {s} \times m}. \tag {1}
$$

式中：

$X$ —试样中被测组分的残留量, 单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ;

$C_{s}$ ——标准溶液中被测组分的浓度，单位为纳克每毫升（ $\mathrm{ng/mL}$ ）；

$A$ —试样溶液中被测组分的峰面积；

$A_{s}$ 一标准溶液中被测组分的峰面积；

$V$ —试样溶液定容体积, 单位为毫升 $(\mathrm{mL})$ ;

$m$ —试料质量，单位为克（g）。

计算结果需扣除空白值。测定结果用两次平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法的灵敏度、准确度和精密度

# 9.1 灵敏度

本方法的检测限：辛基酚、己烯雌酚分别为 $0.2\mu \mathrm{g / kg}$ ，壬基酚、双酚A分别为 $0.1\mu \mathrm{g / kg}$ 雌酮、 $17\alpha$ -乙炔雌二醇、 $17\beta$ -雌二醇、雌三醇分别为 $0.3\mu \mathrm{g / kg}$ ；定量限：辛基酚、己烯雌酚分别为 $0.5\mu \mathrm{g / kg}$ ，壬基酚、双酚A分别为 $0.3\mu \mathrm{g / kg}$ ；雌酮、 $17\alpha$ -乙炔雌二醇、 $17\beta$ -雌二醇、雌三醇分别为 $1.0\mu \mathrm{g / kg}$ 。

# 9.2 准确度

辛基酚、己烯雌酚在 $0.5 \mu \mathrm{g} / \mathrm{kg} \sim 10 \mu \mathrm{g} / \mathrm{kg}$ 添加浓度范围内，回收率为 $70\% \sim 110\%$ ；壬基酚、双酚A在 $0.3 \mu \mathrm{g} / \mathrm{kg} \sim 6 \mu \mathrm{g} / \mathrm{kg}$ 添加浓度范围内，回收率为 $70\% \sim 110\%$ ；雌酮、 $17\alpha$ -乙炔雌二醇、 $17\beta$ -雌二醇、雌三醇在 $1.0 \mu \mathrm{g} / \mathrm{kg} \sim 20 \mu \mathrm{g} / \mathrm{kg}$ 添加浓度范围内，回收率为 $70\% \sim 110\%$ 。

# 9.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

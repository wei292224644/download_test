# 前言

本标准代替SN/T2678—2010《进出口淡水产品中微囊藻毒素的检测方法 酶联免疫吸附法》。

本标准与SN/T2678—2010相比，主要变化如下：

——标准名称修改为“食品安全国家标准 水产品中微囊藻毒素的测定”；

——增加了液相色谱-串联质谱法。

# 食品安全国家标准

# 水产品中微囊藻毒素的测定

# 1 范围

本标准规定了水产品中微囊藻毒素(环状七肽)的液相色谱-串联质谱和间接竞争酶联免疫吸附的测定方法。

本标准适用于鱼、虾、河蚌等水产品中微囊藻毒素的测定。

# 第一法 液相色谱-串联质谱法

# 2 原理

试样中的微囊藻毒素(MC-LR，MC-RR和MC-YR)经甲醇溶液提取，固相萃取小柱净化，液相色谱-串联质谱测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.2 甲酸（ $\mathrm{CH_2O_2}$ ）：色谱纯。  
3.1.3 甲酸铵 $\mathrm{(CH_5O_2N)}$ ：色谱纯。  
3.1.4 乙腈 $\mathrm{(C_2H_3N)}$ ：色谱纯。  
3.1.5 氮气：纯度 $\geq 99.99\%$

# 3.2 试剂配制

3.2.1 甲醇溶液 $(20\%)$ ：将 $20~\mathrm{mL}$ 甲醇与 $80~\mathrm{mL}$ 水混合均匀。  
3.2.2 甲醇溶液 $(80\%)$ ：将 $80~\mathrm{mL}$ 甲醇与 $20~\mathrm{mL}$ 水混合均匀。  
3.2.3 淋洗溶液： $10~\mathrm{mL}$ 水； $10~\mathrm{mL}$ 甲醇溶液 $(20\%)$ 。  
3.2.4 含 $0.1\%$ 甲酸的甲醇溶液：用甲醇将 $0.1 \mathrm{~mL}$ 甲酸定容至 $100 \mathrm{~mL}$ 。

# 3.3 标准品

微囊藻毒素标准品：微囊藻毒素-LR（MC-LR， $\mathrm{C_{49}H_{74}N_{10}O_{12}}$ ，CAS号101043-37-2）、微囊藻毒素-YR（MC-YR， $\mathrm{C_{52}H_{72}N_{10}O_{13}}$ ，CAS号101064-48-6）、微囊藻毒素-RR（MC-RR， $\mathrm{C_{49}H_{75}N_{13}O_{12}}$ ，CAS号111755-37-4），纯度均 $\geq 95\%$ 。

# 3.4 标准溶液配制

3.4.1 微囊藻毒素标准储备液 $(10\mu \mathrm{g / mL})$ ：分别称取微囊藻毒素标准品MC-LR、MC-YR、MC-RR各

$0.5\mathrm{mg}$ (按实际含量折算,精确至 $0.01\mathrm{mg}$ ), 用 $500\mu \mathrm{L}$ 甲醇溶解,再用水定容至 $50\mathrm{mL}$ 。标准储备液贮存于密闭棕色玻璃瓶中,于 $-20^{\circ}C$ 储存,有效期为3个月。  
3.4.2 标准中间液 $(10\mathrm{ng / mL})$ ：吸取微囊藻毒素标准储备液 $(10~\mu \mathrm{g / mL})10~\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度。标准中间液于一 $20^{\circ}C$ 冷冻避光保存，有效期为3个月。  
3.4.3 基质匹配的混合标准系列工作液：用空白基质溶液将微囊藻毒素标准中间溶液 $(10\mathrm{ng / mL})$ 稀释成含量为 $0.1\mu \mathrm{g} / \mathrm{L}, 0.2\mu \mathrm{g} / \mathrm{L}, 0.5\mu \mathrm{g} / \mathrm{L}, 1.0\mu \mathrm{g} / \mathrm{L}, 2.0\mu \mathrm{g} / \mathrm{L}$ 和 $5.0\mu \mathrm{g} / \mathrm{L}$ 的混合标准系列工作液。现用现配。

# 3.5 材料

3.5.1 固相萃取小柱： $\mathrm{ODSC}_{18}$ 柱， $500~\mathrm{mg / 6~mL}$ ，或性能相当者。  
3.5.2 有机相微孔滤膜： $0.22\mu \mathrm{m}$   
3.5.3 玻璃纤维滤纸：GF/F规格，或性能相当者。

# 4 仪器和设备

4.1 液相色谱-质谱仪：配有电喷雾离子源。  
4.2 分析天平：感量为 $0.001\mathrm{g}$ 和 $0.00001\mathrm{g}$   
4.3 离心机：转速大于 $8000\mathrm{r / min}$   
4.4 均质器。  
4.5 旋转蒸发仪。  
4.6 涡旋振荡器。

# 5 分析步骤

# 5.1 试样制备

取样品的可食部分约 $500\mathrm{g}$ ，充分均质或粉碎，装入洁净容器中，密封，并标明标记。试样置于 $-18^{\circ}\mathrm{C}$ 以下避光保存，一个月内须完成检测。

# 5.2 提取

准确称取 $5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ）试样，置于 $50~\mathrm{mL}$ 离心管中，加入 $10~\mathrm{mL}$ 甲醇溶液 $(80\%)$ ，充分振荡均匀， $8000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，取上清液于新离心管中。残渣重复提取一次，合并两次提取液，提取液定容至 $30~\mathrm{mL}$ ，以玻璃纤维滤纸过滤。取 $6\mathrm{mL}$ 滤液加入 $30~\mathrm{mL}$ 水稀释。待净化。

# 5.3 净化

$\mathrm{C}_{18}$ 固相萃取小柱使用前依次用 $10 \mathrm{~mL}$ 甲醇和 $10 \mathrm{~mL}$ 水活化, 流速控制在 $1$ 滴 $/ \mathrm{s} \sim 2$ 滴 $/\mathrm{s}$ 。将提取液全部过柱后, 用 $10 \mathrm{~mL}$ 甲醇溶液 $(20 \%)$ 淋洗, 以 $10 \mathrm{~mL}$ 含有 $0.1 \%$ 甲酸的甲醇溶液洗脱, 收集洗脱液, 以氮气浓缩至近干。加入 $1 \mathrm{~mL}$ 甲醇溶液 $(20 \%)$ 充分溶解残渣, 用有机相微孔滤膜过滤, 待上机检测。

# 5.4 空白对照试验

分别称取与待测样品基质相同、不含所测微囊藻毒素的空白试样10份于 $50~\mathrm{mL}$ 离心管中。以下操作按5.2和5.3的操作处理。合并10份空白试样溶液，用微孔滤膜过滤。滤液可作为空白基质溶液配制基质匹配的混合标准系列工作液。

b） 柱温： $30^{\circ}C$

# 5.6 标准曲线的制作

分别将 $10~\mu \mathrm{L}$ 混合标准系列工作液按浓度由低到高的顺序注入液相色谱-质谱仪进行测定，得到各微囊藻毒素的峰面积。以各微囊藻毒素的浓度为横坐标，峰面积为纵坐标，绘制标准曲线。微囊藻毒素标准溶液的多反应监测色谱图见图A.1。

# 5.7 试样溶液的测定

将 $10~\mu \mathrm{L}$ 试样溶液注入液相色谱-质谱仪，得到各微囊藻毒素的峰面积，根据标准曲线得到试样溶液中各微囊藻毒素的含量。

# 5.8 定性

试样中微囊藻毒素色谱峰的保留时间与相应标准的色谱峰保留时间的偏差在 $\pm 2.5\%$ 之内。试样中待测物的质谱定性离子至少应包括一个母离子和两个子离子，且试样中目标化合物的定性离子的相对丰度比与浓度相当的标准溶液中该化合物的定性离子的相对丰度比之间的偏差应在表3规定的范围。

# 6 分析结果的表述

试样中微囊藻毒素的含量按式(1)计算：

$$
X _ {1} = \frac {A \times V}{m} \times \frac {1 0 0 0}{1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——试样中微囊藻毒素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

A ——根据标准曲线得到的试样溶液中微囊藻毒素的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——单位换算因子。

结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当取样量为 $5\mathrm{g}$ 时，MC-LR和MC-RR的检出限为 $0.3~\mu \mathrm{g / kg}$ ，定量限为 $1\mu \mathrm{g / kg}$ ；MC-YR的检出限为 $0.17~\mu \mathrm{g / kg}$ ，定量限为 $0.5~\mu \mathrm{g / kg}$ 。

# 第二法 间接竞争酶联免疫吸附法

# 9 原理

试样中的微囊藻毒素经提取及净化处理后与过量的针对微囊藻毒素的特异性抗体反应，多余的游离抗体则与酶标板内的预包被微囊藻毒素人工抗原结合。加入针对微囊藻抗体的酶标二抗和酶对应的底物显色，与微囊藻毒素标准的该反应结果比较，计算试样中微囊藻毒素的含量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 抗微囊藻毒素(MC-LR)的单克隆抗体：杂交瘤技术制备，经亲和层析纯化。

10.1.2 人工抗原：MC-LR-牛血清白蛋白偶联物（MC-LR-BSA）。

10.1.3 牛血清白蛋白(BSA)。

10.1.4 酶标二抗：辣根过氧化物酶(HRP)标记山羊抗小鼠 $\mathrm{IgG}$ 。

10.1.5 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$

10.1.6 十二水合磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4\cdot 12\mathrm{H}_2\mathrm{O})$ 。

10.1.7 氯化钠 $\mathrm{(NaCl)}$ 。

10.1.8 氯化钾(KCl)。

10.1.9 三水合乙酸钠 $\mathrm{(CH_3COONa\cdot 3H_2O)}$

10.1.10 冰乙酸 $\mathrm{(CH_3COOH)}$ 。

10.1.11 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。

10.1.12 明胶。

10.1.13 吐温 $-20(\mathrm{C}_{18}\mathrm{H}_{34}\mathrm{O}_6)$ 。

10.1.14 四甲基联苯胺 $\left(\mathrm{C}_{16} \mathrm{H}_{20} \mathrm{~N}_{2}\right)$ 。

10.1.15 二甲基亚砜(DMSO)。

10.1.16 过氧化氢 $\mathrm{(H_2O_2)}$ 。

# 10.2 试剂配制

10.2.1 乙醇溶液 $(20\%)$ ：将 $20~\mathrm{mL}$ 乙醇与 $80~\mathrm{mL}$ 水混合。  
10.2.2 标准品稀释液：称取 $0.005\mathrm{g}$ 明胶和 $0.1\mathrm{g}$ 叠氮化钠，用水溶解，定容至 $100~\mathrm{mL}$   
10.2.3 磷酸盐缓冲溶液 $(\mathrm{pH}7.4)$ : 分别称取 $0.2 \mathrm{~g}$ 磷酸二氢钾、 $2.9 \mathrm{~g}$ 十二水合磷酸氢二钠、 $8.0 \mathrm{~g}$ 氯化钠、 $0.2 \mathrm{~g}$ 氯化钾, 混合, 用水溶解, 定容至 $1000 \mathrm{~mL}$ 。  
10.2.4 乙酸钠溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：称取 $1.36\mathrm{g}$ 三水合乙酸钠，用水溶解，定容至 $100~\mathrm{mL}$ 。  
10.2.5 乙酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $5.88~\mathrm{mL}$ 冰乙酸，加水定容至 $100~\mathrm{mL}$ 。  
10.2.6 硫酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $55.6~\mathrm{mL}$ 硫酸，沿玻棒缓缓注入约 $200~\mathrm{mL}$ 水中，搅拌，冷却至室温，加水定容至 $1000~\mathrm{mL}$ 。  
10.2.7 包被溶液：称取 $1\mathrm{mg}$ 人工抗原，溶解于 $1000~\mathrm{mL}$ 磷酸盐缓冲溶液 $(\mathrm{pH}7.4)$ 中。  
10.2.8 封闭溶液：称取 $0.5\mathrm{g}$ 明胶，加少量磷酸盐缓冲溶液（pH7.4），加热溶解，冷却后定容至 $1000~\mathrm{mL}$ 。

10.2.9 洗涤溶液(PBS-T)：量取 $0.5\mathrm{mL}$ 吐温-20，用磷酸盐缓冲溶液 $(\mathrm{pH}7.4)$ 定容至 $1000~\mathrm{mL}$ 。  
10.2.10 抗体稀释溶液：称取 $0.5\mathrm{g}$ 明胶，加少量洗涤溶液，加热溶解，冷却后定容至 $1000~\mathrm{mL}$   
10.2.11 二抗溶液：1体积酶标二抗与5000体积抗体稀释溶液混合。  
10.2.12 底物缓冲溶液 $(\mathrm{pH}5.0)$ ：用乙酸溶液 $(1\mathrm{mol} / \mathrm{L})$ 调整乙酸钠溶液的 $\mathrm{pH}$ 为5.0。  
10.2.13 底物贮备溶液：称取 $10\mathrm{mg}$ 四甲基联苯胺，溶于 $1\mathrm{mL}$ 二甲基亚砜中。  
10.2.14 底物溶液：量取 $100\mu \mathrm{L}$ 底物贮备溶液，加 $2\mu \mathrm{L}30\%$ 过氧化氢和 $10~\mathrm{mL}$ 底物缓冲溶液。临用时配制。

# 10.3 标准品

微囊藻毒素-LR（MC-LR， $\mathrm{C_{49}H_{74}N_{10}O_{12}}$ ，CAS号101043-37-2），纯度 $\geq 95\%$ 。

# 10.4 标准溶液配制

微囊藻毒素(MC-LR)标准系列溶液：称取适量微囊藻毒素(MC-LR)，用乙醇溶液 $(20\%)$ 配制成MC-LR浓度为 $0.5\mathrm{mg / mL}$ 的溶液。再用标准稀释液稀释至MC-LR含量为 $10~\mu \mathrm{g / mL}$ 的溶液。再用标准稀释液继续配制MC-LR含量分别为 $0.1\mu \mathrm{g / L},0.2\mu \mathrm{g / L},0.5\mu \mathrm{g / L},1\mu \mathrm{g / L},2\mu \mathrm{g / L}$ 的标准系列溶液。

同5.1。

# 12.3 净化

$\mathrm{C}_{18}$ 固相萃取小柱使用前依次用 $10 \mathrm{~mL}$ 甲醇和 $10 \mathrm{~mL}$ 水活化，流速控制在 $1 \mathrm{~滴} / \mathrm{s} \sim 2$ 滴/s。将提取液全部过柱后，用 $10 \mathrm{~mL}$ 甲醇溶液 $(20\%)$ 淋洗，以 $10 \mathrm{~mL}$ 含有 $0.1\%$ 甲酸的甲醇溶液洗脱，收集洗脱液，以氮气浓缩至近干。用 $500 \mu \mathrm{L}$ 水溶解残渣，待进行间接竞争酶联免疫反应测定。

# 12.4 间接竞争酶联免疫反应

# 12.4.1 包被酶标微孔板

将包被溶液加入酶标微孔板 $(100\mu \mathrm{L} / \text{孔})$ ，于 $4^{\circ}\mathrm{C}$ 放置过夜。

# 12.4.2 封闭酶标微孔板

用洗涤液洗涤放置过夜的酶标微孔板，洗涤3次，每次洗涤 $3\mathrm{min}$ ，加入封闭溶液封闭酶标微孔板（ $200~\mu \mathrm{L}$ /孔），于 $37^{\circ}\mathrm{C}$ 下放置 $2\mathrm{h}$ ，或 $4^{\circ}\mathrm{C}$ 下放置过夜。

# 12.4.3 抗原抗体反应

量取 $500\mu \mathrm{L}$ 抗微囊藻毒素MC-LR的单克隆抗体和 $500\mu \mathrm{L}$ 微囊藻毒素标准系列溶液于 $1.5\mathrm{mL}$ 试管中，混合后用电动振荡器振荡均匀，室温静置 $30\mathrm{min}$ 。这些反应液用于制作微囊藻毒素标准竞争曲线。

量取 $500\mu \mathrm{L}$ 抗微囊藻毒素MC-LR的单克隆抗体和 $500\mu \mathrm{L}$ 净化液于 $1.5\mathrm{mL}$ 试管中，混合后用电动振荡器振荡，室温静置 $30\mathrm{min}$ 。此反应液用于测定试样中微囊藻毒素的含量。

# 12.4.4 竞争反应

用洗涤液洗涤3次封闭过的酶标微孔板(每次洗涤 $3\mathrm{min}$ )，滴加抗体抗原反应溶液（ $100~\mu \mathrm{L}$ /孔）。不同浓度做3次平行试验。 $37^{\circ}C$ 或室温放置 $90\mathrm{min}$ 。在酶标微孔板的适当孔位滴加抗体稀释溶液，作为阴性对照。

# 12.4.5 二抗溶液与抗微囊藻毒素(MC-LR)的单克隆抗体反应

用洗涤液洗涤3次竞争反应后的酶标微孔板(每次洗涤 $3\mathrm{min}$ )，滴加二抗溶液 $(100~\mu \mathrm{L} / \text{孔})$ ， $37^{\circ}\mathrm{C}$ 或室温放置 $30\mathrm{min}$ 。

# 12.4.6 显色及显色后吸光度的确定

用洗涤液洗涤5次经12.4.5反应的酶标微孔板(每次洗涤 $3\mathrm{min}$ )。滴加底物溶液 $(100~\mu \mathrm{L} / \text{孔})$ ， $37^{\circ}\mathrm{C}$ 或室温放置 $15\mathrm{min}\sim 20\mathrm{min}$ ，显色。滴加硫酸 $(1\mathrm{mol} / \mathrm{L})(50~\mu \mathrm{L} / \text{孔})$ ，终止显色反应。

$30\mathrm{min}$ 内，用酶标仪在 $450~\mathrm{nm}$ 处，测定显色后的吸光度值。

# 12.5 测定

# 12.5.1 仪器参考条件

酶标仪： $450~\mathrm{nm}$ 。

# 12.5.2 标准曲线的制作

标准系列溶液的吸光度值与阴性对照吸光度值的比值 $X_{2}$ ，按式(2)分别计算：

$$
X _ {2} = \frac {\mathrm {O D} _ {1}}{\mathrm {O D} _ {2}} \tag {2}
$$

式中：

$X_{2}$ ——吸光度比值；

$\mathrm{OD}_{1}$ ——标准系列溶液的吸光度平均值；

$\mathrm{OD}_2$ ——阴性对照吸光度平均值。

以标准系列溶液的浓度的对数为横坐标，以 $X_{2}$ 为纵坐标，绘制标准曲线。

# 12.5.3 试样溶液测定

计算试样溶液的吸光度值与阴性对照吸光度值的比值, 由标准曲线得到试样溶液中微囊藻毒素的

浓度。

# 13 分析结果的表述

试样中微囊藻毒素的含量 $X_{3}$ ，按式(3)计算：

$$
X _ {3} = \frac {A \times V}{m} \times f \quad \dots \tag {3}
$$

式中：

$X_{3}$ ——试样中微囊藻毒素的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$A$ ——试样溶液中微囊藻毒素的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V$ ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克(g)；

$f$ ——试样溶液的稀释倍数。

计算结果保留三位有效数字。

当酶联免疫法结果呈阳性样品需用第一法确认。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

# 食品接触材料及制品 发泡聚苯乙烯成型品中二氟二氯甲烷的测定

2016-10-19 发布

2017-04-19 实施

# 前言

本标准代替GB/T5009.100—2003《食品包装用发泡聚苯乙烯成型品卫生标准的分析方法》。

本标准与GB/T5009.100—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 发泡聚苯乙烯成型品中二氟二氯甲烷的测定”；  
将样品测定填充色谱柱改成毛细管色谱柱；  
——增加了方法的灵敏度，降低检出限和定量限。

# 食品安全国家标准

# 食品接触材料及制品 发泡聚苯乙烯成型品中二氟二氯甲烷的测定

# 1 范围

本标准规定了发泡聚苯乙烯成型品中二氟二氯甲烷的测定方法。

本标准适用于发泡聚苯乙烯成型品中二氟二氯甲烷的测定。

# 2 原理

根据气体有关定律，将试样放入密封平衡瓶中，用溶剂溶解。在一定温度下，二氟二氯甲烷扩散，达到平衡时，取液上气体注入气相色谱仪中测定。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 $N,N$ -二甲基乙酰胺 $\mathrm{(C_4H_9NO)}$ ：色谱纯。

# 3.2 标准品

二氟二氯甲烷 $(\mathrm{CCl_2F_2})$ （CAS号：75-71-8）：纯度 $>99\%$

# 3.3 标准溶液的制备

3.3.1 二氟二氯甲烷标准储备液 $(100\mu \mathrm{g / mL})$ ：准确称取二氟二氯甲烷标准品 $1\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ 于 $10~\mathrm{mL}$ 棕色容量瓶中，用 $N,N-$ 二甲基乙酰胺溶解定容至刻度，混匀。于 $4^{\circ}C$ 保存，保存期为1年。  
3.3.2 二氟二氯甲烷标准中间液 $(10\mu \mathrm{g / mL})$ ：准确移取二氟二氯甲烷储备溶液 $1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 棕色容量瓶中，用 $N,N$ -二甲基乙酰胺稀释至刻度，混匀。于 $4^{\circ}C$ 保存，保存期为6个月。  
3.3.3 二氟二氯甲烷标准工作液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：准确移取二氟二氯甲烷中间溶液 $1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 棕色容量瓶中，用 $N,N-$ 二甲基乙酰胺稀释至刻度，混匀。于 $4^{\circ}\mathrm{C}$ 保存，保存期为3个月。

# 4 仪器和设备

4.1 气相色谱仪(GC)：附氢火焰离子化检测器(FID)。  
4.2 电子天平：感量为 $1\mathrm{mg}$ 和 $0.01\mathrm{mg}$   
4.3 恒温水浴装置：精度 $1^{\circ}\mathrm{C}$ 。  
4.4 平衡瓶： $20~\mathrm{mL}$   
4.5 气密针： $1.0\mathrm{mL}$

# 5 分析步骤

# 5.1 试样处理

将发泡聚苯乙烯成型品剪成 $3\mathrm{mm}\times 3\mathrm{mm}$ 碎屑，用四分法取样，称取 $0.3\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )放入平衡瓶中。加入 $3\mathrm{mL}N,N$ -二甲基乙酰胺溶解，立即盖塞，轻轻摇匀，放入 $65^{\circ}C$ 恒温水浴中平衡 $15\mathrm{min}$ 取 $1\mathrm{mL}$ 气体供气相色谱仪测定。

# 5.2 仪器参考条件

a）色谱柱： $6\%$ 腈丙苯基 $-94\%$ 二甲基聚硅氧烷毛细管色谱柱，柱长 $30~\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ ，膜厚 $0.18\mu \mathrm{m}$ ，或相当者；  
b）柱温：初始温度 $50~^\circ \mathrm{C}$ ，保持 $5\mathrm{min}$ ，以 $20^{\circ}C / \mathrm{min}$ 升温至 $250^{\circ}C$ ，保持 $2.5\mathrm{min}$   
c）进样口温度： $250^{\circ}C$   
d）检测器温度： $300^{\circ}\mathrm{C}$   
e）载气：氮气，纯度大于或等于 $99.999\%$ ；恒流模式；流速： $1.0~\mathrm{mL / min}$   
f）进样量： $1.0~\mathrm{mL}$ ，分流进样，分流比为 $20:1$   
g）尾吹气：氮气， $60~\mathrm{mL / min}$

# 5.3 标准曲线的制作

在5个平衡瓶中各加入 $3\mathrm{mL}N,N$ -二甲基乙酰，盖塞，用微量注射器取 $0\mu \mathrm{L},3\mu \mathrm{L},15\mu \mathrm{L},30\mu \mathrm{L}$ 产 $150~\mu \mathrm{L},300~\mu \mathrm{L}$ 二氟二氯甲烷标准工作液 $(1\mu \mathrm{g / mL})$ 通过胶塞注入瓶中，轻轻摇匀放入 $65^{\circ}C$ 恒温水浴中平衡 $15\mathrm{min}$ ，即标准曲线浓度为 $0\mathrm{ng / mL},1\mathrm{ng / mL},5\mathrm{ng / mL},10\mathrm{ng / mL},50\mathrm{ng / mL},100\mathrm{ng / mL}_{\circ}$ 分别取液上气 $1\mathrm{mL}$ 注入气相色谱仪中，以二氟二氯甲烷含量为横坐标，峰面积为纵坐标绘制标准曲线。二氟二氯甲烷标准溶液的气相色谱图参见图A.1。

# 5.4 试样溶液的测定

手动或自动将含有 $1\mathrm{mL}$ 试样的液上气注入气相色谱仪中进行测定，得到峰面积，根据标准曲线得到试样溶液中二氟二氯甲烷的含量。同时做空白试验。

# 6 分析结果的表述

试样中二氟二氯甲烷含量按式(1)计算：

$$
X = \frac {\rho \times 1 0 0 0}{m \times 1 0 0 0} \times V \quad \dots \tag {1}
$$

式中：

$X$ ——试样中二氟二氯甲烷的含量，单位为微克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——从标准曲线中得到二氟二氯甲烷标准工作溶液的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$m$ ——样品称样量，单位为克(g)；

$V$ ——样品定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数。

计算结果需扣除空白值，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

按照本标准规定的取样量，本方法对二氟二氯甲烷的检出限为 $0.001\mathrm{mg / kg}$ ，定量限为 $0.01\mathrm{mg / kg}$ 。

# 食品添加剂 甜菊糖苷

2022-06-30 发布

2022-12-30 实施

# 前言

本标准代替GB8270—2014《食品安全国家标准 食品添加剂 甜菊糖苷》。

本标准与GB8270—2014相比，主要变化如下：

——修改了标准范围；  
——修改了分子式、结构式和相对分子质量等相关内容；  
——修改了甜菊糖苷含量(以干基计)的指标要求和检验方法；  
——修改了“灼烧残渣”指标名称为“灰分”；  
——修改了“总碑(以As计)”指标名称为“碑（As）”；  
——增加了铅(Pb)、砷(As)的检验方法；  
——将鉴别试验中 $\mathrm{pH}$ 的要求放入理化指标。

# 食品添加剂 甜菊糖苷

# 1 范围

本标准适用于以甜叶菊(SteviaRebaudianaBertoni)叶为原料，经提取、精制而得的食品添加剂甜菊糖苷。已知糖苷包括甜菊苷、瑞鲍迪苷A、瑞鲍迪苷B、瑞鲍迪苷C、瑞鲍迪苷D、瑞鲍迪苷E、瑞鲍迪苷F、瑞鲍迪苷M、瑞鲍迪苷N、瑞鲍迪苷O、杜克苷A、甜茶苷及甜菊双糖苷。

# A.3.4 分析步骤

# A.3.4.1 混合标准品溶液的制备

# A.3.4.3 试样溶液的制备

称取 $25.0\mathrm{mg}\pm 5.0\mathrm{mg}$ 试样(干基)，精确至 $0.1\mathrm{mg}$ ，置于 $50~\mathrm{mL}$ 的容量瓶中，用乙腈水溶液溶解后稀释至刻度，得到试样溶液。

# A.3.5 测定

# A.3.5.1 标准曲线的绘制

在A.3.3参考色谱条件下，分别对 $200\mathrm{mg / L},400\mathrm{mg / L},800\mathrm{mg / L},1000\mathrm{mg / L}$ 的瑞鲍迪苷A标准溶液和 $100\mathrm{mg / L},200\mathrm{mg / L},400\mathrm{mg / L},500\mathrm{mg / L}$ 的瑞鲍迪苷D标准溶液进行色谱分析，记录各浓度标准溶液对应的峰面积值。以标准溶液色谱图中瑞鲍迪苷A或瑞鲍迪苷D的峰面积值为 $Y$ 轴，以对应溶液浓度 $(\mathrm{mg / L})$ 为 $X$ 轴，绘制标准曲线，分别得到瑞鲍迪苷A标准曲线和瑞鲍迪苷D标准曲线（强制过原点），从标准曲线上分别获得瑞鲍迪苷A线性函数斜率 $(K_{\mathrm{a}})$ 和瑞鲍迪苷D线性函数斜率 $(K_{\mathrm{d}})$ 。

# A.3.5.2 含量测定

在A.3.3参考色谱条件下，分别对混合标准品溶液和试样溶液进行色谱分析。将试样溶液的色谱图与混合标准品溶液的色谱图(见附录B)相比较，以确定试样溶液色谱图中各组分对应的峰。记录试样溶液色谱图中瑞鲍迪苷A、瑞鲍迪苷D、甜菊苷、瑞鲍迪苷B、瑞鲍迪苷C、瑞鲍迪苷E、瑞鲍迪苷F、瑞鲍迪苷M、瑞鲍迪苷N、瑞鲍迪苷O、杜克苷A、甜茶苷和甜菊双糖苷的峰面积。

# A.3.6 结果计算

8种糖苷含量(以干基计)的质量分数 $w_{i}$ 按式(A.1)计算。

$$
w _ {i} = \frac {f _ {i} \times A _ {i}}{K _ {\mathrm {a}} \times c} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

i ——s、a、b、c、f、da、ru、sb, 分别对应甜菊苷、瑞鲍迪苷 A、瑞鲍迪苷 B、瑞鲍迪苷 C、瑞鲍迪苷 F、杜克苷 A、甜茶苷、甜菊双糖苷；

$f_{i}$ —— $i$ 组分与瑞鲍迪苷A的式量比值：1.00（瑞鲍迪苷A）、0.83（甜菊苷）、0.83（瑞鲍迪苷B）、0.98（瑞鲍迪苷C）、0.97（瑞鲍迪苷F）、0.82（杜克苷A）、0.66（甜茶苷）、0.66（甜菊双糖苷）；

$A_{i}$ ——试样溶液色谱图中 $i$ 组分的峰面积值；

$K_{\mathrm{a}}$ ——A.3.5.1中瑞鲍迪苷A线性函数斜率；

$c$ ——试样溶液的浓度，单位为毫克每升 $(\mathrm{mg / L})$ 。

5种糖苷含量(以干基计)的质量分数 $w_{i}$ 按式(A.2)计算。

$$
w _ {i} = \frac {f _ {i} \times A _ {i}}{K _ {\mathrm {d}} \times c} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

$i$ ——d、e、m、n、o，分别对应瑞鲍迪苷D、瑞鲍迪苷E、瑞鲍迪苷M、瑞鲍迪苷N、瑞鲍迪苷O；

$f_{i}$ —— $i$ 组分与瑞鲍迪苷D的式量比值：1.00（瑞鲍迪苷D）、0.86（瑞鲍迪苷E）、1.14（瑞鲍迪苷M）、1.13（瑞鲍迪苷N）、1.27（瑞鲍迪苷O）；

$A_{i}$ ——试样溶液色谱图中 $i$ 组分的峰面积值；

$K_{\mathrm{d}}$ ——A.3.5.1中瑞鲍迪苷D线性函数斜率；

$c$ ——试样溶液的浓度，单位为毫克每升 $(\mathrm{mg / L})$ 。

由式(A.1)和式(A.2)计算得到13种组分的含量 $w_{\mathrm{s}}, w_{\mathrm{a}}, w_{\mathrm{b}}, w_{\mathrm{c}}, w_{\mathrm{f}}, w_{\mathrm{da}}, w_{\mathrm{ru}}, w_{\mathrm{sb}}, w_{\mathrm{d}}, w_{\mathrm{e}}, w_{\mathrm{m}}, w_{\mathrm{n}}$ 和 $w_{\mathrm{o}}$ ，取各组分含量之和即为试样中甜菊糖苷含量。

# A.5.4 参考顶空条件

A.5.4.1 顶空瓶温度： $80^{\circ}C$   
A.5.4.2 顶空瓶平衡时间： $30\mathrm{min}$

# A.5.5 分析步骤

# A.5.5.1 空白溶液的制备

移取 $2\mathrm{mL}$ 水，置于顶空瓶中，迅速压紧瓶盖，备用。

# A.5.5.2 标准溶液的制备

# A.5.5.2.1 甲醇标准溶液的制备

称取 $0.1\mathrm{g}$ 甲醇，精确至 $0.001\mathrm{g}$ ，用水稀释后转移至一个 $1000~\mathrm{mL}$ 的容量瓶中，加水至刻度，摇匀，即得 $100\mathrm{mg / L}$ 的甲醇标准储备溶液。将该溶液配制成一系列浓度为 $2.5\mathrm{mg / L},5\mathrm{mg / L},10\mathrm{mg / L},$ $20\mathrm{mg / L}$ 和 $50\mathrm{mg / L}$ 的甲醇标准溶液。移取上述系列浓度溶液各 $2\mathrm{mL}$ ，分别置于顶空瓶中，迅速压紧瓶盖，备用。

# A.5.5.2.2 乙醇标准溶液的制备

称取 $0.1\mathrm{g}$ 乙醇，精确至 $0.001\mathrm{g}$ ，用水稀释后转移至一个 $100~\mathrm{mL}$ 的容量瓶中，加水至刻度，摇匀，即得 $1000\mathrm{mg / L}$ 的乙醇标准储备溶液。将该溶液配制成一系列浓度为 $50~\mathrm{mg / L},100~\mathrm{mg / L},200~\mathrm{mg / L},$ $500~\mathrm{mg / L}$ 和 $750~\mathrm{mg / L}$ 的乙醇标准溶液。移取上述系列浓度溶液各 $2\mathrm{mL}$ ，分别置于顶空瓶中，迅速压紧瓶盖，备用。

# A.5.5.3 试样溶液的制备

称取 $1.0\mathrm{g}$ 试样，精确至 $0.001\mathrm{g}$ ，用水溶解后转移至一个 $10\mathrm{mL}$ 的容量瓶中，在室温下超声处理约 $3\mathrm{min}$ ，加水稀释至刻度，摇匀。移取该溶液 $2\mathrm{mL}$ 置于顶空瓶中，迅速压紧瓶盖，备用。

# A.5.6 测定

在参考操作条件A.5.3和A.5.4下，分别对空白溶液、标准系列溶液和试样溶液进行测定，记录甲醇或乙醇的峰面积值。以标准系列溶液色谱图中甲醇或乙醇的峰面积值为 $Y$ 轴，以对应溶剂浓度 $(\mathrm{mg} / \mathrm{L})$ 为 $X$ 轴，绘制标准曲线，得到甲醇标准曲线或乙醇标准曲线。根据试样溶液色谱图中甲醇或乙醇的峰面积值，从标准曲线求得试样溶液中甲醇或乙醇的浓度 $(\mathrm{mg} / \mathrm{L})$ 。

# A.5.7 结果计算

试样中甲醇或乙醇的含量 $w$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.3)计算。

$$
w = \frac {c \times V}{m} \quad \dots \tag {A.3}
$$

式中：

$c$ ——从标准曲线求得的试样溶液中甲醇或乙醇的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

# 前言

本标准代替GB25541—2010《食品安全国家标准 食品添加剂 聚葡萄糖》。

本标准与GB25541—2010相比，主要变化如下：

——删去了规范性引用文件；  
——增加了产品分类；  
——增加了液体聚葡萄糖的感官要求及检测方法，删除了气味指标要求及检测方法；  
——增加了液体聚葡萄糖的理化指标及检测方法；  
——修改了理化指标中“灰分”指标，改为“总灰分”；  
——增加了铅的检验方法；  
——增加了砷的指标要求及检验方法；  
——修改了附录A中聚葡萄糖含量的检验方法，增加了高效液相色谱法；  
——修改了附录A中 $\mathrm{pH}$ 的检验方法，细化了试验要求；  
——增加了附录B。

# 检验方法

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682中规定的水。分析中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。本试验所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 浓硫酸。

A.2.1.2 丙酮。

A.2.1.3 苯酚溶液： $50~\mathrm{g / L}$

A.2.1.4 柠檬酸铜碱性试液：称取 $173\mathrm{g}$ 柠檬酸钠 $\mathrm{(C_6H_5Na_3O_7\cdot 2H_2O)}$ 和 $117\mathrm{g}$ 碳酸钠 $\mathrm{(Na_2CO_3\cdot H_2O)}$ ，在加热下溶于约 $700~\mathrm{mL}$ 水中，必要时用滤纸过滤。在另一容器中，称取 $17.3\mathrm{g}$ 硫酸铜 $\mathrm{(CuSO_4\cdot 5H_2O)}$ ，溶于约 $100~\mathrm{mL}$ 水中，然后将此溶液在稳定搅拌下缓慢地加入上述溶液。冷却后，用水稀释至 $1000~\mathrm{mL}$ 混匀。

# A.2.2 分析步骤

A.2.2.1 取1滴 $100\mathrm{g / L}$ 试样液，加入4滴苯酚溶液，迅速加入15滴浓硫酸，应产生深黄至橘黄颜色。

A.2.2.2 取 $1 \mathrm{~mL} 100 \mathrm{~g} / \mathrm{L}$ 试样液，在强烈搅拌下加入 $1 \mathrm{~mL}$ 丙酮，溶液应透明。在该透明溶液中，加入 $2 \mathrm{~mL}$ 丙酮，强烈搅拌，溶液立即呈乳白色浑浊。

A.2.2.3 取 $1\mathrm{mL}20\mathrm{g / L}$ 试样液，加入 $4\mathrm{mL}$ 柠檬酸铜碱性试液，加热至剧烈沸腾 $2\mathrm{min}\sim 4\mathrm{min}$ ，移去热源，让其沉淀、澄清，其上层清液应呈蓝色或蓝绿色。

# A.3 聚葡萄糖的测定

# A.3.1 高效液相色谱法

# A.3.1.1 试剂和材料

A.3.1.1 1,6-脱水-D-葡萄糖标准品(CAS号：498-07-7)：纯度 $\geqslant 98.0\%$

A.3.1.1.2 葡萄糖 $(\alpha -\mathrm{D}$ -葡萄糖)标准品：纯度 $\geq 98.0\%$

A.3.1.1.3 山梨糖醇标准品：纯度 $\geqslant 98.0\%$

A.3.1.1.4 硫酸。

# A.3.1.2 仪器和设备

高效液相色谱仪，配示差折光检测器。

# A.3.1.3 参考色谱条件

A.3.1.3.1 色谱柱：磺酸型苯乙烯-二乙烯苯共聚物树脂柱（苯乙烯为单体，二乙烯苯为交联剂，分离原理为尺寸排阻+离子交换；分离模式为：排阻极限 $>1000$ ，理论塔板数 $n \geqslant 17000$ ，柱长 $300\mathrm{mm}$ ，柱内

径 $8\mathrm{mm}$ ；或其他等效色谱柱。

A.3.1.3.2 流动相：吸取 $0.42\mathrm{mL}$ 硫酸，用水稀释至 $1000~\mathrm{mL}$ ，用 $0.45\mu \mathrm{m}$ 滤膜过滤，用超声波脱气 $15\mathrm{min}$ 。  
A.3.1.3.3 柱温： $60~^\circ \mathrm{C}$   
A.3.1.3.4 流速： $0.5\mathrm{mL / min}$   
A.3.1.3.5 进样量： $20~\mu \mathrm{L}$

# A.3.1.4 分析步骤

# A.3.1.4.1 标准溶液的制备

称取适量1,6-脱水-D-葡萄糖、葡萄糖 $(\alpha -\mathrm{D}$ -葡萄糖)和山梨糖醇标准品用水溶解并配制成 $0.1\mathrm{g / L}$ 产 $0.2\mathrm{g / L},0.4\mathrm{g / L},0.6\mathrm{g / L}$ 系列浓度的标样，经 $0.45\mu \mathrm{m}$ 滤膜过滤后备用。

# A.3.1.4.2 试样液的制备

称取约 $1\mathrm{g}$ 聚葡萄糖试样(精确至 $0.0001\mathrm{g}$ )，用水溶解并定容至 $25\mathrm{mL}$ ，该试样液用 $0.45\mu \mathrm{m}$ 滤膜过滤后备用。

# A.3.1.4.3 测定

在A.3.1.3参考色谱条件下，分别对1,6-脱水-D-葡萄糖、葡萄糖 $(\alpha -D$ -葡萄糖)和山梨糖醇的系列浓度的标样进行测定。以标样峰面积为纵坐标、标准溶液浓度 $(\mathrm{g / L})$ 为横坐标，分别绘制1,6-脱水-D-葡萄糖、葡萄糖 $(\alpha -D$ -葡萄糖)和山梨糖醇的标准曲线。

在A.3.1.3参考色谱条件下，对试样液进行测定，根据标准品的保留时间定性，色谱图见附录B。根据标样峰面积和标样浓度之间的线性关系，分别得到试样液中1,6-脱水-D-葡萄糖、葡萄糖和山梨糖醇的质量浓度 $(\mathrm{g / L})$ 。若试样液中1,6-脱水-D-葡萄糖、葡萄糖和山梨糖醇的质量浓度 $(\mathrm{g / L})$ 不在标准曲线范围内，则应依据样品实际情况调整试样液的浓度至标准曲线浓度范围内。

# A.3.1.4.4 结果计算

试样中聚葡萄糖的含量 $X_{1}$ （以干基、无灰分品计)按式(A.1)计算。

$$
X _ {1} = 1 0 0 - X _ {2} - X _ {3} \quad \dots \tag {A.1}
$$

试样中1，6-脱水-D-葡萄糖的含量 $X_{2}$ （以干基、无灰分品计)按式(A.2)计算。

$$
X _ {2} = \frac {\rho_ {1}}{\rho_ {2}} \times 1 0 0 \quad \dots \tag {A.2}
$$

试样中葡萄糖和山梨糖醇的含量 $X_{3}$ （以干基、无灰分品计)按式(A.3)计算。

$$
X _ {3} = X _ {4} + X _ {5} \quad \dots \tag {A.3}
$$

试样中葡萄糖的含量 $X_{4}$ （以干基、无灰分品计）按式(A.4)计算。

$$
X _ {4} = \frac {\rho_ {3}}{\rho_ {2}} \times 1 0 0 \quad \dots \tag {A.4}
$$

试样中山梨糖醇的含量 $X_{5}$ （以干基、无灰分品计)按式(A.5)计算。

$$
X _ {5} = \frac {\rho_ {4}}{\rho_ {2}} \times 1 0 0 \quad \dots \tag {A.5}
$$

式中：

$\rho_{1}$ ——根据标准曲线求得的1，6-脱水-D-葡萄糖的质量浓度，单位为克每升 $(\mathrm{g / L})$ ；

$\rho_{2}$ ——试样液的质量浓度（根据试样的干燥失重和灰分含量，换算成以干基、无灰分品计的浓

度），单位为克每升 $(\mathrm{g / L})$

$\rho_{3}$ ——根据标准曲线求得的葡萄糖 $(\alpha - \mathrm{D}-$ 葡萄糖)的质量浓度，单位为克每升 $(\mathrm{g} / \mathrm{L})$

$\rho_{4}$ ——根据标准曲线求得的山梨糖醇的质量浓度，单位为克每升 $(\mathrm{g / L})$ 。

计算结果表示到小数点后一位。

# A.3.1.4.5 精密度

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不应超过算术平均值的 $5\%$ 。

# A.3.2 分光光度法

# A.3.2.1 试剂和材料

A.3.2.1.1 葡萄糖 $(\alpha-\mathrm{D}-$ 葡萄糖)标准品：纯度 $\geq 98.0\%$

A.3.2.1.2 苯酚。

A.3.2.1.3 硫酸。

A.3.2.1.4 苯酚溶液： $4\mathrm{g / mL}$ ；准确称取 $80~\mathrm{g}$ 苯酚，加入 $20~\mathrm{mL}$ 水溶解，混匀。

# A.3.2.2 仪器和设备

分光光度计。

# A.3.2.3 分析步骤

# A.3.2.3.1 葡萄糖标准溶液的制备

称取适量的葡萄糖 $(\alpha-\mathrm{D}-$ 葡萄糖)标准品，用水溶解后配制成质量浓度为 $0.2\mathrm{mg / mL}$ 的标准原液。用标准原液配制成 $5\mu \mathrm{g} / \mathrm{mL},10\mu \mathrm{g} / \mathrm{mL},20\mu \mathrm{g} / \mathrm{mL},30\mu \mathrm{g} / \mathrm{mL},40\mu \mathrm{g} / \mathrm{mL},50\mu \mathrm{g} / \mathrm{mL}$ 系列浓度的标准溶液。

# A.3.2.3.2 试样液的制备

称取约 $0.25\mathrm{g}$ 试样（精确至 $0.0001\mathrm{g}$ ），用水溶解并定容至 $250\mathrm{mL}$ ，混匀后，用移液管吸取 $10.0\mathrm{mL}$ ，加水稀释并定容至 $250\mathrm{mL}$ ，此为试样液。

# A.3.2.3.3 葡萄糖标准曲线的绘制及试样液的测定

用移液管吸取 $2.0\mathrm{mL}$ 系列浓度的标准溶液、试样液和蒸馏水（作为空白），分别置于不含丙酮的 $15~\mathrm{mL}$ 螺口小瓶中，分别加入 $0.12\mathrm{mL}$ 苯酚溶液，盖上瓶塞并轻轻混匀。打开瓶塞，迅速加入 $5.0~\mathrm{mL}$ 硫酸。盖紧小瓶，剧烈摇匀。注意：添加硫酸时需戴橡胶手套及其他安全护具。

小瓶在室温下保持 $45\mathrm{min}$ ，然后选择合适的分光光度计测定每一小瓶中溶液在 $490~\mathrm{nm}$ 处的吸光值，测定时用加有蒸馏水的苯酚-硫酸混合液作为空白参比。重复试验3次，得到系列浓度的标准溶液的平均吸光值和试样液的平均吸光值。以系列浓度的标准溶液的平均吸光值为纵坐标。标准溶液浓度 $(\mu \mathrm{g} / \mathrm{mL})$ 为横坐标，绘制标准曲线。

# A.3.2.3.4 结果计算

聚葡萄糖的含量 $X_{1}$ （以干基、无灰分品计）按式(A.6)计算。

$$
X _ {1} = 1. 0 5 \times \frac {1 0 0 \times (A - Y)}{S \times \rho} - P _ {\mathrm {G}} - 1. 1 1 \times P _ {\mathrm {L}} \quad \dots \tag {A.6}
$$

式中：

1.05 ——推导校正因子；

A ——试样液的吸光值；

Y ——标准曲线的 $y$ 轴截距；

S ——吸光值对葡萄糖质量浓度 $(\mu \mathrm{g} / \mathrm{mL})$ 标准曲线的斜率， $S \approx 0.02$

$\rho$ ——试样液的质量浓度（根据试样的干燥失重和灰分含量，换算成以干基、无灰分品计的浓度），单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

1.11 ——1,6-脱水-D-葡萄糖的转换系数；

$P_{\mathrm{G}}, P_{\mathrm{L}}$ ——单体试验中分别测得的葡萄糖和1,6-脱水-D-葡萄糖的含量，%。

计算结果表示到小数点后一位。

# A.3.2.3.5 精密度

在重复条件下获得的两次独立测定结果的绝对差值应不超过其算术平均值的 $10\%$ 。

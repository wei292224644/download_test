# 前言

本标准代替GB/T21911—2008《食品中邻苯二甲酸酯的测定》和SN/T3147—2012《出口食品中邻苯二甲酸酯的测定》。

本标准与GB/T21911—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中邻苯二甲酸酯的测定”；  
——增加了邻苯二甲酸二烯丙酯和邻苯二甲酸二异壬酯两种目标化合物；  
——增加了同位素内标法定量作为第一法；  
——修改了前处理方法；  
——修改了方法的检出限。

# 食品安全国家标准

# 食品中邻苯二甲酸酯的测定

# 1 范围

本标准第一法规定了食品中16种邻苯二甲酸酯类物质含量的气相色谱-质谱联用(GC-MS)的测定方法；第二法规定了食品中18种邻苯二甲酸酯类物质含量的气相色谱-质谱联用(GC-MS)的测定方法。

本标准第一法适用于食品中邻苯二甲酸二甲酯(DMP)、邻苯二甲酸二乙酯(DEP)、邻苯二甲酸二异丁酯(DIBP)、邻苯二甲酸二正丁酯(DBP)、邻苯二甲酸二(2-甲氧基)乙酯(DMEP)、邻苯二甲酸二(4-甲基-2-戊基)酯(BMPP)、邻苯二甲酸二(2-乙氧基)乙酯(DEEP)、邻苯二甲酸二戊酯(DPP)、邻苯二甲酸二己酯(DHXP)、邻苯二甲酸丁基苄基酯(BBP)、邻苯二甲酸二(2-丁氧基)乙酯(DBEP)、邻苯二甲酸二环己酯(DCHP)、邻苯二甲酸二(2-乙基)己酯(DEHP)、邻苯二甲酸二苯酯(DPhP)、邻苯二甲酸二正辛酯(DNOP)、邻苯二甲酸二壬酯(DNP)含量的内标法测定和确证；第二法适用于食品中邻苯二甲酸二甲酯(DMP)、邻苯二甲酸二乙酯(DEP)、邻苯二甲酸二烯丙酯(DAP)、邻苯二甲酸二异丁酯(DIBP)、邻苯二甲酸二正丁酯(DBP)、邻苯二甲酸二(2-甲氧基)乙酯(DMEP)、邻苯二甲酸二(4-甲基-2-戊基)酯(BMPP)、邻苯二甲酸二(2-乙氧基)乙酯(DEEP)、邻苯二甲酸二戊酯（DPP）邻苯二甲酸二己酯(DHXP)、邻苯二甲酸丁基苄基酯(BBP)、邻苯二甲酸二(2-丁氧基)乙酯(DBEP)、邻苯二甲酸二环己酯(DCHP)、邻苯二甲酸二(2-乙基)己酯(DEHP)、邻苯二甲酸二苯酯(DPhP)、邻苯二甲酸二正辛脂(DNOP)、邻苯二甲酸二异壬酯(DINP)、邻苯二甲酸二壬酯(DNP)含量的外标法测定和确证。

# 第一法 气相色谱-质谱法同位素内标法

# 2 原理

在试样中加入气代的邻苯二甲酸酯作为内标，各类食品经提取、净化后经气相色谱-质谱联用仪进行测定。采用特征选择离子监测扫描模式(SIM)，以保留时间和定性离子碎片的丰度比定性，同位素内标法定量。

# 3.3 标准溶液配制

3.3.1 16种邻苯二甲酸酯标准中间溶液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：准确移取邻苯二甲酸酯标准品 $(1000\mu \mathrm{g} / \mathrm{mL})$ $1\mathrm{mL}$ 至 $100~\mathrm{mL}$ 容量瓶中，用正己烷准确定容至刻度。  
3.3.2 16种氘代同位素的邻苯二甲酸酯内标溶液 $(100\mu \mathrm{g} / \mathrm{mL})$ : 准确称取16种氘代同位素的邻苯二甲酸酯内标各 $0.01\mathrm{g}$ (精确到 $0.0001\mathrm{g}$ ) 于 $100\mathrm{mL}$ 容量瓶中, 用正己烷溶解并准确定容至刻度。  
3.3.3 16种氘代同位素的邻苯二甲酸酯内标的标准使用液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：准确移取16种氘代同位素的邻苯二甲酸酯内标 $(100\mu \mathrm{g} / \mathrm{mL})10\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加入正己烷并准确定容至刻度。  
3.3.4 16种邻苯二甲酸酯标准系列工作液：准确吸取16种邻苯二甲酸酯标准中间溶液 $(10\mu \mathrm{g / mL})$ ，用正己烷逐级稀释，配制成浓度为 $0.00~\mu \mathrm{g / mL},0.02~\mu \mathrm{g / mL},0.05~\mu \mathrm{g / mL},0.10~\mu \mathrm{g / mL},0.20~\mu \mathrm{g / mL}$ $0.50~\mu \mathrm{g / mL},1.00~\mu \mathrm{g / mL}$ 的标准系列溶液，同时加入内标使用液（ $10~\mu \mathrm{g / mL}$ )，使内标浓度均为0.125 $\mu \mathrm{g / mL}$ ，临用时配制。

# 5 分析步骤

# 5.1 试样制备

5.1.1 液态样品：取约 $200~\mathrm{mL}$ 样品混匀后放置磨口玻璃瓶内待用。  
5.1.2 半固态和固态样品：分别取约 $200\mathrm{g}$ 样品经粉碎后放置磨口玻璃瓶内待用。

# 5.2 试样处理

# 5.2.1 液态试样

5.2.1.1 液态试样A：液体乳、饮料、酱油、食醋、白酒、蜂蜜等

准确称取试样 $1.0\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）于 $25~\mathrm{mL}$ 具塞磨口离心管中，加入 $125~\mu \mathrm{L}$ 同位素内标使用液，加入 $2\mathrm{mL}\sim 5\mathrm{mL}$ 蒸馏水，涡旋混匀，再准确加入 $10~\mathrm{mL}$ 正己烷，涡旋 $1\mathrm{min}$ ，剧烈振摇 $1\mathrm{min}$ ，超声提取 $30\mathrm{min}$ ， $1000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液，供GC-MS分析。

5.2.1.2 液态试样B：植物油等

液态油脂混匀后准确称取 $0.5\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）于 $10~\mathrm{mL}$ 具塞磨口离心管中，加入 $25~\mu \mathrm{L}$ 同位素内标使用液，依次加入 $100~\mu \mathrm{L}$ 正己烷和 $2\mathrm{mL}$ 乙腈，涡旋 $1\mathrm{min}$ ，超声提取 $20\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液。残渣中加入 $2\mathrm{mL}$ 乙腈，涡旋 $1\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。再加入 $2\mathrm{mL}$ 乙腈重复提取1次，合并3次上清液，待SPE净化。

# 5.2.2 半固态试样

5.2.2.1 半固态试样A：果冻、甜面酱等

准确称取混匀试样 $0.5\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）于 $25~\mathrm{mL}$ 具塞磨口离心管中，加入 $125\mu \mathrm{L}$ 同位素内标使用液，加入 $2\mathrm{mL}\sim 5\mathrm{mL}$ 蒸馏水，涡旋混匀，再准确加入 $10~\mathrm{mL}$ 正己烷，涡旋 $1\mathrm{min}$ ，剧烈振摇 $1\mathrm{min}$ ，超声提取 $30\mathrm{min}$ ， $1000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液，供GC-MS分析。

5.2.2.2 半固态试样B：芝麻酱、含油调味酱等

将样品充分粉碎混匀后准确称取 $0.5\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）于 $10\mathrm{mL}$ 具塞磨口离心管中，加入 $25~\mu \mathrm{L}$ 同位素内标使用液，加入 $1\mathrm{mL}$ 正己烷，涡旋 $2\mathrm{min}$ ，再加入 $5\mathrm{mL}$ 乙腈，涡旋 $1\mathrm{min}$ ，超声提取 $20\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液。加入 $5\mathrm{mL}$ 乙腈重复提取1次，合并上清液。 $40^{\circ}\mathrm{C}$ 氮气吹干，加入 $6\mathrm{mL}$ 乙腈，涡旋混匀，待SPE净化。

# 5.2.3 固态试样

5.2.3.1 固态试样A：乳粉、米粉、鸡精、味精、干酪、糖果、花粉、肉制品、糕点、方便面、果蔬及其制品等

准确称取混匀试样 $0.5\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）于 $25~\mathrm{mL}$ 具塞磨口离心管中，加入 $125\mu \mathrm{L}$ 同位素内标使用液，加入 $2\mathrm{mL}\sim 5\mathrm{mL}$ 蒸馏水，涡旋混匀，再准确加入 $10~\mathrm{mL}$ 正己烷，涡旋 $1\mathrm{min}$ ，剧烈振摇 $1\mathrm{min}$ ，超声提取 $30\mathrm{min}$ ， $1000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液，供GC-MS分析。

5.2.3.2 固态试样B：黄油等

将样品充分粉碎混匀后准确称取 $0.5\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）于 $10~\mathrm{mL}$ 具塞磨口离心管中，加入 $25~\mu \mathrm{L}$ 同位素内标使用液，加入 $1\mathrm{mL}$ 正己烷，涡旋 $2\mathrm{min}$ ，再加入 $5\mathrm{mL}$ 乙腈，涡旋 $1\mathrm{min}$ ，超声提取 $20\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液。加入 $5\mathrm{mL}$ 乙腈重复提取1次，合并上清液。 $40^{\circ}\mathrm{C}$ 氮气吹至近干，加入 $6\mathrm{mL}$ 乙腈，涡旋混匀，待SPE净化。

注：黄油应融化为液态油脂混匀后称取，并在提取过程中保持液态。

# 5.3 SPE净化

依次加入 $5\mathrm{mL}$ 二氯甲烷、 $5\mathrm{mL}$ 乙腈活化，弃去流出液；将待净化液加入SPE小柱，收集流出液；再加入 $5\mathrm{mL}$ 乙腈，收集流出液，合并两次收集的流出液，加入 $1\mathrm{mL}$ 丙酮， $40^{\circ}\mathrm{C}$ 氮吹至近干，正己烷准确定容至 $2\mathrm{mL}$ ，涡旋混匀，供GC-MS分析。

# 5.4 空白试验

除不加试样外，均按5.2、5.3测定步骤进行。

注：整个操作过程中，应避免接触塑料制品。

# 5.7 试样溶液的测定

# 6 分析结果的表述

试样中邻苯二甲酸酯的含量按式(1)计算：

$$
X = \rho \times \frac {V}{m} \times \frac {1 0 0 0}{1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中邻苯二甲酸酯的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——从标准工作曲线上查出的试样溶液中邻苯二甲酸酯的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克(g)；

1000 ——换算系数。

计算结果应扣除空白值。结果大于等于 $1.0\mathrm{mg / kg}$ 时，保留三位有效数字；结果小于 $1.0\mathrm{mg / kg}$ 时，保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法的定量限为：邻苯二甲酸二正丁酯(DBP)定量限为 $0.3\mathrm{mg / kg}$ ，除DBP外其他15种邻苯二甲酸酯定量限均为 $0.5\mathrm{mg / kg}$ 。

# 12.6 标准曲线的制作

将标准系列工作液分别注入气相色谱-质谱联用仪中，测定相应的邻苯二甲酸酯的色谱峰面积，以标准工作液的质量浓度为横坐标，以相应的峰面积为纵坐标，绘制标准曲线。邻苯二甲酸二异壬酯的标准系列工作液单独进样测定。

# 12.7 试样溶液的测定

将试样溶液注入气相色谱-质谱联用仪中，得到相应的邻苯二甲酸酯的峰面积，根据标准曲线得到待测液中邻苯二甲酸酯的浓度。

邻苯二甲酸酯的总离子流色谱图见附录E。

# 13 分析结果的表述

试样中邻苯二甲酸酯的含量按式(2)计算：

$$
X = \rho \times \frac {V}{m} \times \frac {1 0 0 0}{1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中邻苯二甲酸酯的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——从标准工作曲线上查出的试样溶液中邻苯二甲酸酯的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果应扣除空白值。结果大于等于 $1.0\mathrm{mg / kg}$ 时，保留三位有效数字；结果小于 $1.0\mathrm{mg / kg}$ 时，保留两位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

本方法的定量限为：邻苯二甲酸二异壬酯(DINP)的定量限为 $9.0\mathrm{mg / kg}$ ，邻苯二甲酸二正丁酯(DBP)定量限为 $0.3\mathrm{mg / kg}$ ，除DINP和DBP外其他16种目标化合物定量限均为 $0.5\mathrm{mg / kg}$ 。

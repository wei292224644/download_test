# 食品安全国家标准

# 食品中1-甲基咪唑、2-甲基咪唑及4-甲基咪唑的测定

# 1 范围

本标准规定了食品中1-甲基咪唑、2-甲基咪唑和4-甲基咪唑的液相色谱-串联质谱测定方法。

本标准适用于炼乳、冷冻饮品、果酱、豆制品、可可及巧克力制品、糖果、小麦粉制品、饼干、馅料及表面挂浆、肉制品、调味糖浆、调味品(酱油、醋、酱及酱制品、料酒、复合调味料)、饮料、酒类、果冻及膨化食品中1-甲基咪唑、2-甲基咪唑和4-甲基咪唑含量的测定。

# 2 原理

试样加同位素内标后，采用水或酸水提取，经固相萃取柱净化、浓缩，用液相色谱-串联质谱仪测定，同位素内标法定量。

# 3 试剂和材料

除另有说明外，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.3 甲酸（HCOOH）：色谱纯。  
3.1.4 氨水 $(\mathrm{NH}_4\mathrm{OH})$ ：浓度 $25\% \sim 28\%$   
3.1.5 正己烷 $\mathrm{(CH_3(CH_2)_4CH_3)}$ ：色谱纯。  
3.1.6 乙酸铵 $\mathrm{(CH_3COONH_4)}$ ：色谱纯。  
3.1.7 氯化钠 $(\mathrm{NaCl})$ 。

# 3.2 试剂配制

3.2.1 $2\%$ 甲酸水溶液：量取 $20~\mathrm{mL}$ 甲酸，加入到 $980~\mathrm{mL}$ 水中，混匀。  
3.2.2 氨水甲醇溶液：量取 $50~\mathrm{mL}$ 氨水，加入到 $950~\mathrm{mL}$ 甲醇中，混匀。  
3.2.3 乙酸铵溶液 $(5\mathrm{mmol} / \mathrm{L})$ ：称取 $0.39\mathrm{g}$ 乙酸铵，溶于 $1000~\mathrm{mL}$ 水中，混匀。  
3.2.4 乙腈-乙酸铵溶液 $(9 + 1)$ ：将乙腈和乙酸铵溶液( $5\mathrm{mmol / L}$ )按 $9:1$ 的体积比混合均匀。

# 3.3 标准品

3.3.1 1-甲基咪唑标准品 $\mathrm{(C_4H_6N_2}$ ,CAS号：616-47-7)：纯度 $\geqslant 98\%$ ，或为经国家认证并授予标准物质证书的标准物质。  
3.3.2 2-甲基咪唑标准品 $\mathrm{(C_4H_6N_2}$ ,CAS号：693-98-1)：纯度 $\geqslant 98\%$ ，或为经国家认证并授予标准物质证书的标准物质。

3.3.3 4-甲基咪唑标准品 $\mathrm{(C_4H_6N_2}$ ，CAS号：822-36-6）：纯度 $\geqslant 98\%$ ，或为经国家认证并授予标准物质证书的标准物质。  
3.3.4同位素内标1-甲基咪唑- $\mathrm{D}_6(\mathrm{C}_4\mathrm{D}_6\mathrm{N}_2$ ，CAS号：285978-27-0)：纯度 $\geq 98\%$   
3.3.5 同位素内标2-甲基咪唑- $\mathrm{D}_6(\mathrm{C}_4\mathrm{D}_6\mathrm{N}_2$ ，CAS号：1173022-19-9)：纯度 $\geq 98\%$ 。  
3.3.6同位素内标4-甲基咪唑 $\mathrm{D}_6(\mathrm{C}_4\mathrm{D}_6\mathrm{N}_2$ ，CAS号：1219804-79-1)：纯度 $\geqslant 98\%$

# 3.4 标准溶液配制

3.4.1 标准储备溶液 $(1\mathrm{mg / mL})$ ：准确称取1-甲基咪唑标准品 $25\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ )，加水 $1\mathrm{mL}$ 使其溶解，用乙腈转移至 $25~\mathrm{mL}$ 棕色容量瓶中并定容至刻度，摇匀。分别准确称取2-甲基咪唑、4-甲基咪唑标准品 $25\mathrm{mg}$ (精确至 $0.01\mathrm{mg}$ )，用乙腈溶解，转移至 $25~\mathrm{mL}$ 棕色容量瓶中并定容至刻度，摇匀。一 $20^{\circ}C$ 下避光密封保存，有效期6个月。  
3.4.2 混合标准工作溶液 $(2.0\mu \mathrm{g / mL})$ ：分别吸取上述标准储备溶液各 $100~\mu \mathrm{L}$ 于同一 $50~\mathrm{mL}$ 棕色容量瓶中，加乙腈稀释定容至刻度，摇匀，得 $2.0~\mu \mathrm{g / mL}$ 的混合标准溶液。此溶液 $4^{\circ}C$ 以下避光密封保存，有效期3个月。  
3.4.3 同位素内标储备溶液 $(1\mathrm{mg / mL})$ ：分别准确称取1-甲基咪唑- $\mathrm{D}_6$ 、2-甲基咪唑- $\mathrm{D}_6$ 和4-甲基咪唑- $\mathrm{D}_6$ 各 $10\mathrm{mg}$ (精确至 $1\mathrm{mg}$ )，用乙腈溶解，转移至 $10\mathrm{mL}$ 棕色容量瓶中定容至刻度，摇匀。 $-20^{\circ}\mathrm{C}$ 下避光密封保存，有效期6个月。  
3.4.4 同位素内标混合工作溶液 $(1.0\mu \mathrm{g} / \mathrm{mL})$ ：分别吸取上述同位素内标储备溶液各 $100~\mu \mathrm{L}$ 于同一 $100~\mathrm{mL}$ 棕色容量瓶中，用乙腈稀释定容至刻度，摇匀，得 $1.0~\mu \mathrm{g} / \mathrm{mL}$ 的同位素内标混合工作溶液。此溶液 $4^{\circ}C$ 避光密封保存，有效期3个月。  
3.4.5 标准系列溶液：分别吸取混合标准工作溶液 $25\mu \mathrm{L},75\mu \mathrm{L},150\mu \mathrm{L},300\mu \mathrm{L},600\mu \mathrm{L},1000\mu \mathrm{L}$ 至 $5\mathrm{mL}$ 棕色容量瓶中，分别加入 $250~\mu \mathrm{L}$ 的同位素内标混合工作溶液，用乙腈-乙酸铵溶液 $(9 + 1)$ 混合溶液定容至刻度，配制成1-甲基咪唑、2-甲基咪唑和4-甲基咪唑的混合标准系列溶液，质量浓度为 $10\mathrm{ng / mL},30\mathrm{ng / mL},60\mathrm{ng / mL},120\mathrm{ng / mL},240\mathrm{ng / mL},400\mathrm{ng / mL}$ 。在实际样品测定中，可根据样品浓度调整同位素内标混合工作溶液加入量及稀释倍数。

# 3.5 材料

3.5.1 混合型阳离子交换固相萃取小柱：N-乙烯吡咯烷酮-二乙烯基苯共聚物基质-磺酸基， $150\mathrm{mg}$ ， $6\mathrm{mL}$ ，或性能相当者。使用前依次用 $5\mathrm{mL}$ 甲醇活化，再用 $5\mathrm{mL}$ 水淋洗平衡。  
3.5.2 均质子：适用于 $50~\mathrm{mL}$ 离心管。  
3.5.3 有机系滤膜：孔径为 $0.22\mu \mathrm{m}$

注：3.5.1、3.5.2、3.5.3在使用前均应做本底试验，确定无干扰后方可使用。

# 4 仪器和设备

4.1 液相色谱-串联质谱仪：配电喷雾离子源（ESI）。  
4.2 分析天平：感量 $0.001\mathrm{g}$ 和 $0.00001\mathrm{g}$   
4.3 移液器：最大量程 $200\mu \mathrm{L}$ 和 $1000\mu \mathrm{L}$   
4.4 均质器。  
4.5 涡旋混合器。  
4.6 恒温水浴装置。  
4.7 超声波清洗器：功率 $\geq 250\mathrm{W}$   
4.8 高速离心机：转速 $\geqslant 10000\mathrm{r / min}$   
4.9 固相萃取装置。

4.10 氮吹浓缩装置。

# 5 分析步骤

# 5.1 样品前处理

# 5.1.1 试样制备

取适量有代表性的样品，饮料等液态均匀样品直接摇匀；非均匀的样品需匀浆或粉碎均匀，冷冻饮品室温融化后充分搅拌均匀，必要时可 $30~\mathrm{^\circ C}\sim 40~\mathrm{^\circ C}$ 水浴加热搅拌，酱类等可以采用研磨或者均质混匀，巧克力等采用 $50~\mathrm{^\circ C}\sim 60~\mathrm{^\circ C}$ 加热熔融，并趁热充分搅拌均匀。

# 5.1.2 试样提取

# 5.1.2.1 液体试样（如酱油、醋、饮料、酒类等）

称取 $2\mathrm{g}$ 混合均匀的试样（精确至 $0.001\mathrm{g}$ ），置 $20\mathrm{mL}$ 刻度比色管中，加入 $1.0~\mu \mathrm{g / mL}$ 同位素内标混合工作溶液 $100~\mu \mathrm{L}$ ，加 $2\%$ 甲酸水溶液定容至 $20~\mathrm{mL}$ ，摇匀，置离心管中，以 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液待净化。

# 5.1.2.2 半流体试样（如果酱、蚝油、调味糖浆、冷冻饮品等）及胶基糖果、粉圆

称取 $2\mathrm{g}$ 混合均匀的试样（精确至 $0.001\mathrm{g}$ ），置 $50~\mathrm{mL}$ 离心管中，加入 $1.0~\mu \mathrm{g / mL}$ 同位素内标混合工作溶液 $100~\mu \mathrm{L}$ ，混匀，准确加入水 $20~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ， $70^{\circ}\mathrm{C}$ 水浴加热 $10\mathrm{min}$ ，冷却至室温后，超声提取 $20\mathrm{min}$ ，以 $10000\mathrm{r / min}$ 在 $4^{\circ}\mathrm{C}$ 下离心 $5\mathrm{min}$ ，上清液待净化。

# 5.1.2.3 果冻

称取 $2\mathrm{g}$ 混合均匀的试样（精确至 $0.001\mathrm{g}$ ），置 $50~\mathrm{mL}$ 离心管中，加入 $1.0~\mu \mathrm{g / mL}$ 同位素内标混合工作溶液 $100~\mu \mathrm{L}$ ，混匀，加入 $2\mathrm{g}$ 氯化钠，准确加入水 $20~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ， $70^{\circ}\mathrm{C}$ 水浴加热 $10\mathrm{min}$ ，冷却至室温后，超声提取 $20\mathrm{min}$ ，以 $10000\mathrm{r / min}$ 在 $4^{\circ}\mathrm{C}$ 下离心 $5\mathrm{min}$ ，上清液待净化。

# 5.1.2.4 炼乳、可可及巧克力制品

称取 $2\mathrm{g}$ 混合均匀的试样（精确至 $0.001\mathrm{g}$ ），置 $50~\mathrm{mL}$ 离心管中，加入 $1.0~\mu \mathrm{g / mL}$ 同位素内标混合工作溶液 $100~\mu \mathrm{L}$ ，混匀，准确加入 $2\%$ 甲酸水溶液 $20~\mathrm{mL},60^{\circ}C$ 水浴加热 $5\mathrm{min}$ ，加入2颗均质子，涡旋 $5\mathrm{min}$ ，以 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液待净化。

# 5.1.2.5 豆制品、酱及酱制品等其他试样

称取 $2\mathrm{g}$ 混合均匀的试样（精确至 $0.001\mathrm{g}$ ），置 $50~\mathrm{mL}$ 离心管中，加入 $1.0~\mu \mathrm{g / mL}$ 同位素内标混合工作溶液 $100~\mu \mathrm{L}$ ，混匀，准确加入水 $20~\mathrm{mL}$ ，均质 $1\mathrm{min}$ ，以 $10000\mathrm{r / min}$ 在 $4^{\circ}\mathrm{C}$ 下离心 $5\mathrm{min}$ ，上清液待净化。

# 5.1.3 试样净化

# 5.1.3.1 酱油、醋、饮料、酒、炼乳、可可及巧克力制品

吸取 $10\mathrm{mL}$ 上清液，过固相萃取小柱，依次用 $2\%$ 甲酸水溶液 $5\mathrm{mL}$ ，甲醇 $5\mathrm{mL}$ 淋洗，再用氨水甲醇溶液 $10\mathrm{mL}$ 洗脱，收集全部洗脱液， $45^{\circ}\mathrm{C}$ 以下氮吹至近干，准确加入乙腈-乙酸铵溶液 $(9 + 1)1\mathrm{mL}$ ，涡旋 $30\mathrm{s}$ ，过 $0.22\mu \mathrm{m}$ 有机滤膜，滤液供液相色谱-串联质谱仪测定。

# 5.1.3.2 蚝油

吸取 $10\mathrm{mL}$ 上清液，加甲酸 $200~\mu \mathrm{L}$ 混匀酸化，必要时过滤，过固相萃取小柱，依次用 $2\%$ 甲酸水溶

液 $5\mathrm{mL}$ ，乙腈 $5\mathrm{mL}$ 淋洗，抽干， $1.5\mathrm{mL}$ 正己烷淋洗两次，抽干，再用氨水甲醇溶液 $10~\mathrm{mL}$ 洗脱，收集全部洗脱液， $45^{\circ}C$ 以下氮吹至近干，准确加入乙腈-乙酸铵溶液 $(9 + 1)1\mathrm{mL}$ ，涡旋 $30~\mathrm{s}$ ，过 $0.22\mu \mathrm{m}$ 有机系滤膜，滤液供液相色谱-串联质谱仪测定。

# 5.1.3.3 除5.1.3.1、5.1.3.2外的其他基质

吸取 $10\mathrm{mL}$ 上清液，加甲酸 $200~\mu \mathrm{L}$ 混匀酸化，必要时过滤，过固相萃取小柱，依次用 $2\%$ 甲酸水溶液 $5\mathrm{mL}$ ，甲醇 $5\mathrm{mL}$ 淋洗，再用氨水甲醇溶液 $10\mathrm{mL}$ 洗脱，收集全部洗脱液， $45^{\circ}\mathrm{C}$ 以下氮吹至近干，准确加入乙腈-乙酸铵溶液 $(9 + 1)1\mathrm{mL}$ ，涡旋 $30~s$ ，过 $0.22~\mu \mathrm{m}$ 有机系滤膜，滤液供液相色谱-串联质谱仪测定。

# 5.4 定量测定

# 5.4.1 标准曲线的制作

将标准系列溶液由低到高进样检测，以被测组分浓度与对应内标浓度比值为横坐标，被测组分色谱峰面积与对应内标色谱峰面积比值为纵坐标，绘制标准曲线，得到线性回归方程。

标准品色谱图参见附录B图B.1。

# 5.4.2 试样溶液的测定

将试样溶液同标准系列溶液进样测定，计算样品中待测物的含量。

# 5.5 空白试验

不称取试样，按照5.1做空白试验。应确认不含有干扰待测组分的物质。

# 6 分析结果的表述

试样中目标化合物的含量按式(1)计算：

$$
X = \frac {c \times V \times f \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中待测组分的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c$ ——试样溶液中待测组分的质量浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

V ——样品经固相萃取柱净化洗脱后的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——样液的稀释因子；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ ；

1000——换算系数；

计算结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

当取样量为 $2\mathrm{g}$ 时，1-甲基咪唑、2-甲基咪唑和4-甲基咪唑在咖啡饮料、植物饮料中的检出限为 $3\mu \mathrm{g / kg}$ ，定量限为 $10~\mu \mathrm{g / kg}$ ；在其他基质中的检出限为 $10~\mu \mathrm{g / kg}$ ，定量限为 $30~\mu \mathrm{g / kg}$ 。

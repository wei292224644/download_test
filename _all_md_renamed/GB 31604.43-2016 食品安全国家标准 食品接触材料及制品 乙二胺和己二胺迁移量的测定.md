# 食品接触材料及制品 乙二胺和己二胺迁移量的测定

2016-10-19 发布

2017-04-19 实施

# 前言

本标准代替GB/T23296.17—2009《食品接触材料 高分子材料 食品模拟物中乙二胺与己二胺的测定 气相色谱法》。

本标准与GB/T23296.17—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 乙二胺和己二胺迁移量的测定”。

# 食品安全国家标准

# 食品接触材料及制品 乙二胺和

# 己二胺迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中乙二胺和己二胺迁移量的气相色谱的测定方法。

本标准适用于食品接触材料及制品中乙二胺和己二胺迁移量的测定。

# 2 原理

用氯甲酸乙酯作为衍生试剂，将食品模拟物中的乙二胺和己二胺衍生转化为相应的二氨基甲酸乙酯衍生产物，采用气相色谱柱进行分离，氢火焰离子化检测器进行检测。内标法定量，内标物为1,3-丙二胺。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。试验中容器及转移器具应避免使用塑料材质。

# 3.1 试剂

3.1.1 氯甲酸乙酯 $\mathrm{(C_3H_5ClO_2)}$ ：纯度大于 $97\%$   
3.1.2 冰乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
3.1.3 无水乙醇 $\mathrm{(C_2H_6O)}$ 。  
3.1.4 甲苯 $(\mathrm{C}_7\mathrm{H}_8)$ 。  
3.1.5 乙醚 $(\mathrm{C_4H_{10}O})$ 。  
3.1.6 异辛烷 $\mathrm{(C_8H_{18})}$   
3.1.7 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.8 氢氧化钠(NaOH)。  
3.1.9 氨水：浓度为 $25\%$ （质量分数）。  
3.1.10 配制水基、酸性、酒精类、油基食品模拟物所需试剂：符合GB31604.1的规定。

# 3.2 试剂配制

3.2.1 水基、酸性、酒精类、油基食品模拟物：按GB5009.156的规定操作。  
3.2.2 氨水溶液（ $3\%$ ，体积分数）：在 $250~\mathrm{mL}$ 锥形瓶中加入 $200~\mathrm{mL}$ 水和 $30~\mathrm{mL} 25\%$ 氨水溶液，混合均匀。  
3.2.3 氢氧化钠溶液 $(5\mathrm{mol} / \mathrm{L})$ ：称取 $50.0\mathrm{g}$ （精确到 $0.1\mathrm{g}$ )氢氧化钠于 $250~\mathrm{mL}$ 容量瓶中，用水定容。  
3.2.4 乙酸溶液（ $4\%$ ，质量分数）：称取 $40 \mathrm{~g}$ （精确到 $0.1 \mathrm{~g}$ ）冰乙酸于 $1 \mathrm{~L}$ 容量瓶中，用水定容。  
3.2.5 乙醇溶液（ $10\%$ ，体积分数）：量取 $100~\mathrm{mL}$ 无水乙醇于 $1\mathrm{L}$ 容量瓶中，用水定容。

3.2.6 乙醇溶液（ $95\%$ ，体积分数）：量取 $475~\mathrm{mL}$ 无水乙醇于 $500~\mathrm{mL}$ 容量瓶中，用水定容。

# 3.3 标准品

3.3.1 乙二胺 $\mathrm{(C_2H_8N_2}$ ，CAS号：107-15-3)，纯度 $>99\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.2 已二胺 $\left(\mathrm{C}_{6} \mathrm{H}_{16} \mathrm{~N}_{2}\right)$ , CAS 号: 124-09-4), 纯度 $>99\%$ , 或经国家认证并授予标准物质证书的标准物质。  
3.3.3 1,3-丙二胺 $\left(\mathrm{C}_{3} \mathrm{H}_{10} \mathrm{~N}_{2}\right)$ , CAS号: $109-76-2$ , 纯度 $>99 \%$ , 或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 水配制的乙二胺储备液 $(1000\mathrm{mg / L})$ ：准确称取乙二胺 $50~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ）于 $50~\mathrm{mL}$ 容量瓶中，用水定容。  
3.4.2 水配制的己二胺储备液 $(500\mathrm{mg / L})$ ：准确称取己二胺 $25\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ）于 $50~\mathrm{mL}$ 容量瓶中，用水定容。  
3.4.3 水配制的丙二胺储备液 $(500\mathrm{mg / L})$ ：准确称取丙二胺 $25\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ ）于 $50~\mathrm{mL}$ 容量瓶中，用水定容。  
3.4.4 甲苯配制的乙二胺储备液 $(1000\mathrm{mg / L})$ ：准确称取乙二胺 $50~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )于 $50~\mathrm{mL}$ 容量瓶中，用甲苯定容。  
3.4.5 甲苯配制的己二胺储备液 $(500\mathrm{mg / L})$ ：准确称取己二胺 $25~\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )于 $50~\mathrm{mL}$ 容量瓶中，用甲苯定容。  
3.4.6 甲苯配制的丙二胺储备液 $(500\mathrm{mg / L})$ ：准确称取丙二胺 $25\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )于 $50~\mathrm{mL}$ 容量瓶中，用甲苯定容。

配制的储备液可在 $5\% \sim 20\%$ 下避光密封保存，3个月内有效。

# 3.5 器皿和材料

3.5.1 玻璃瓶： $2\mathrm{mL}, 10\mathrm{mL}$ ，瓶盖带有涂覆有聚四氟乙烯的丁基橡胶或者硅橡胶密封垫。  
3.5.2 容量瓶： $50~\mathrm{mL}$   
3.5.3 微量注射器： $10~\mu \mathrm{L}$ 和 $25~\mu \mathrm{L}$   
3.5.4 振荡器。

# 4 仪器和设备

4.1 气相色谱仪：配氢火焰离子化检测器和数据自动分析软件。  
4.2 分析天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$

# 5 分析步骤

# 5.1 标准工作溶液及试样制备

# 5.1.1 标准工作溶液的制备

# 5.1.1.1 水基、酸性食品、酒精类食品模拟物和乙醇溶液(95%，体积分数)标准工作溶液

向7个 $10\mathrm{mL}$ 玻璃瓶中分别加入 $1\mathrm{mL}$ 水和 $10~\mu \mathrm{L}$ 水配制的丙二胺储备液，依次加入 $0\mu \mathrm{L},1\mu \mathrm{L}$

$2\mu \mathrm{L},3\mu \mathrm{L},5\mu \mathrm{L},12\mu \mathrm{L}$ 和 $24~\mu \mathrm{L}$ 水配制的乙二胺储备液和 $0\mu \mathrm{L},1\mu \mathrm{L},2\mu \mathrm{L},4\mu \mathrm{L},6\mu \mathrm{L},8\mu \mathrm{L},10\mu \mathrm{L}$ 水配制的己二胺储备液，加盖密封，混合均匀。得到水中乙二胺浓度分别为 $0.00~\mathrm{mg / L},1.00~\mathrm{mg / L}$ 、 $2.00~\mathrm{mg / L},3.00~\mathrm{mg / L},5.00~\mathrm{mg / L},12.0~\mathrm{mg / L}$ 和 $24.0~\mathrm{mg / L}$ ；己二胺浓度分别为 $0.00~\mathrm{mg / L}$ 、 $0.500~\mathrm{mg / L},1.00~\mathrm{mg / L},2.00~\mathrm{mg / L},3.00~\mathrm{mg / L},4.00~\mathrm{mg / L}$ 和 $5.00~\mathrm{mg / L}$ 。采用同样方式，分别用酸性食品、酒精类食品模拟物和乙醇溶液（ $95\%$ ，体积分数）配制同样浓度系列的乙二胺和己二胺标准工作溶液。

# 5.1.1.2 油基食品模拟物和异辛烷标准工作溶液

分别称取 $1\mathrm{g}$ 油基食品模拟物(精确到 $0.01\mathrm{g}$ ) 于7个 $10\mathrm{mL}$ 玻璃瓶中，每个玻璃瓶中加入 $10~\mu \mathrm{L}$ 甲苯配制的丙二胺储备液，依次加入 $0\mu \mathrm{L},1\mu \mathrm{L},2\mu \mathrm{L},3\mu \mathrm{L},5\mu \mathrm{L},12\mu \mathrm{L}$ 和 $24~\mu \mathrm{L}$ 甲苯配制的乙二胺储备液和 $0\mu \mathrm{L},1\mu \mathrm{L},2\mu \mathrm{L},4\mu \mathrm{L},6\mu \mathrm{L},8\mu \mathrm{L}$ 和 $10~\mu \mathrm{L}$ 甲苯配制的己二胺储备液，加盖密封，混合均匀。得到油基食品模拟物中乙二胺浓度分别为 $0.00~\mathrm{mg / kg}$ 、 $1.00~\mathrm{mg / kg}$ 、 $2.00~\mathrm{mg / kg}$ 、 $3.00~\mathrm{mg / kg}$ 、 $5.00~\mathrm{mg / kg}$ 、 $12.0~\mathrm{mg / L}$ 和 $24.0~\mathrm{mg / kg}$ ；己二胺浓度分别为 $0.00~\mathrm{mg / kg}$ 、 $0.500~\mathrm{mg / kg}$ 、 $1.00~\mathrm{mg / kg}$ 、 $2.00~\mathrm{mg / kg}$ 、 $3.00~\mathrm{mg / kg}$ 、 $4.00~\mathrm{mg / kg}$ 和 $5.00~\mathrm{mg / kg}$ 。采用同样方式，用异辛烷配制同样浓度系列的乙二胺和己二胺标准工作溶液。

# 5.1.2 食品模拟物试液的制备

# 5.1.2.1 总则

食品模拟物试液应按照GB31604.1和GB5009.156的要求从迁移试验中获取，应避光于冰箱中冷藏保存。

当食品接触材料及制品与橄榄油在 $10\mathrm{d},20^{\circ}\mathrm{C}$ 或者 $10\mathrm{d},40^{\circ}\mathrm{C}$ 条件下长期接触时，食品接触材料及制品中迁移出的乙二胺和己二胺会与橄榄油发生反应，导致检测结果偏低，应采用乙醇溶液（ $95\%$ ，体积分数）或异辛烷替代油基食品模拟物进行检测。

# 5.1.2.2 水基、酸性食品、酒精类食品模拟物

准确量取迁移试验中得到的水基、酸性食品、酒精类食品模拟物 $1\mathrm{mL}$ 于 $10~\mathrm{mL}$ 玻璃瓶中，加入 $10~\mu \mathrm{L}$ 水配制的丙二胺储备液，加盖密封，混合均匀。

# 5.1.2.3 油基食品模拟物

准确称取迁移试验中得到的油基食品模拟物 $1.0\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $10~\mathrm{mL}$ 样品瓶中，加入 $10~\mu \mathrm{L}$ 甲苯配制的丙二胺储备液，加盖密封，混合均匀。

# 5.1.2.4 乙醇溶液（ $95\%$ ，体积分数）

准确量取迁移试验中得到的乙醇溶液 $(95\%)$ ，体积分数） $1.0~\mathrm{mL}$ 于 $10~\mathrm{mL}$ 样品瓶中，加入 $10~\mu \mathrm{L}$ 水配制的丙二胺储备液，加盖密封，混合均匀。

# 5.1.2.5 异辛烷

准确量取迁移试验中得到的异辛烷 $1.0\mathrm{mL}$ 于 $10~\mathrm{mL}$ 样品瓶中，加入 $10~\mu \mathrm{L}$ 甲苯配制的丙二胺储备液，加盖密封，混合均匀。

# 5.1.3 空白试液的制备

按照5.1.2的操作处理未与食品接触材料及制品接触的食品模拟物。

# 5.2 食品模拟物试液衍生化处理

# 5.2.1 水基、酸性食品、酒精类食品模拟物和乙醇溶液（ $95\%$ ，体积分数）

向盛有水基、酸性食品、酒精类食品模拟物试液的玻璃瓶中加入 $1\mathrm{mL}$ 氨水溶液（ $3\%$ ，体积分数）、 $3\mathrm{mL}$ 氢氧化钠溶液（ $5\mathrm{mol} / \mathrm{L}$ ）、 $2\mathrm{mL}$ 甲苯和 $200\mu \mathrm{L}$ 氯甲酸乙酯，密封，室温下往复振荡 $30\mathrm{min}$ 。静置分层后，转移 $1\mathrm{mL}$ 上层甲苯清液于 $2\mathrm{mL}$ 玻璃进样瓶中，加入少许无水硫酸钠，离心后取上层清液供气相色谱分析。乙醇溶液（ $95\%$ ，体积分数）按照同样程序处理。

# 5.2.2 油基食品模拟物

向盛有油基食品模拟物试液的玻璃瓶中加入 $5\mathrm{mL}$ 乙醚和 $1\mathrm{mL}$ 乙酸溶液（ $4\%$ ，质量分数），加盖密封，充分摇匀，静置分层。用移液器转移下层水相于 $10\mathrm{mL}$ 玻璃瓶中，再次用 $1\mathrm{mL}$ 乙酸溶液（ $4\%$ ，质量分数）重复提取，合并两次提取液。在提取液中加入 $4\mathrm{mL}$ 乙醚进行清洗，移去醚层，再重复用 $4\mathrm{mL}$ 乙醚清洗，最终油基食品模拟物试液中的乙二胺和己二胺被萃取到水相中，按5.2.1操作步骤进行衍生处理。

# 5.2.3 异辛烷

向盛有异辛烷试液的玻璃瓶中加入 $1\mathrm{mL}$ 乙酸溶液 $(4\%)$ ，质量分数），密封，充分摇匀，静置分层。用移液器转移下层水相于 $10\mathrm{mL}$ 玻璃瓶中，再次用 $1\mathrm{mL}$ 乙酸溶液 $(4\%)$ ，质量分数）重复提取，合并两次提取液。在提取液中加入 $4\mathrm{mL}$ 乙醚进行清洗，移去醚层，再重复用 $4\mathrm{mL}$ 乙醚清洗，最终异辛烷试液中的乙二胺和己二胺被萃取到水相中，按5.2.1操作步骤进行衍生处理。

# 5.3 空白试液衍生化处理

按照5.2的操作处理5.1.3中制备的空白试液。

# 5.4 测定

# 5.4.1 仪器参考条件

仪器参考条件列出如下：

a）色谱柱： $100\%$ 二甲基硅氧烷柱，长 $30~\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ ，膜厚 $5\mu \mathrm{m}$ ，或相当者；  
b）进样口温度： $280^{\circ}C$   
c）检测器温度： $300^{\circ}\mathrm{C}$   
d）炉温： $100^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，以 $25^{\circ}\mathrm{C / min}$ 上升到270，保持 $7\mathrm{min}$   
e) 载气：氮气，流速 $1.8\mathrm{mL} / \mathrm{min}$   
f）进样方式：分流进样，分流比 $10:1$   
g）进样量： $1\mu \mathrm{L}$

# 5.4.2 标准曲线的制作

对5.1.1中制备的标准工作溶液按照5.2进行衍生化处理，在5.4.1所列仪器参数下进行检测，以食品模拟物标准工作溶液中乙二胺或己二胺的浓度为横坐标，以对应的乙二胺衍生物或己二胺衍生物与内标物丙二胺衍生物之间的峰面积比值为纵坐标，绘制标准工作曲线，得到线性方程。标准溶液的色谱图参见附录A。

# 5.4.3 定量测定

对经衍生化处理的食品模拟物试液和空白试液依次进样，分别得到乙二胺衍生物或己二胺衍生物与内标物丙二胺衍生物峰面积对应比值；根据标准曲线得到试样溶液中乙二胺或己二胺的含量，扣除空白值。

# 6 分析结果的表述

由标准曲线得到试样溶液中乙二胺或己二胺的浓度，按照GB5009.156进行迁移量的计算，得到食品接触材料及制品中乙二胺或己二胺的迁移量。计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法对水基、酸性食品、酒精类食品模拟物、乙醇(95%，体积分数)和异辛烷食品模拟物中乙二胺的定量限均为 $1.00\mathrm{mg / L}$ ，己二胺的定量限均为 $0.500\mathrm{mg / L}$ ；油基食品模拟物中乙二胺的定量限为 $1.00\mathrm{mg / kg}$ ，己二胺的定量限为 $0.500\mathrm{mg / kg}$ 。

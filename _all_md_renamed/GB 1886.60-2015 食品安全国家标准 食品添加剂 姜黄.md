# 食品添加剂 姜黄

# 1 范围

本标准适用于以姜科类植物姜黄（Turmeric, Curcuma Longa L.）的根茎为原料，经有机溶剂提取精制而成的食品添加剂姜黄。

# 2 主成分的化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

姜黄素：1，7-双(4-羟基-3-甲氧基苯基)-1，6-二烯-3，5-庚二酮

脱甲氧基姜黄素：1-(4-羟基苯基)-7-(4-羟基-3-甲氧基苯基)-1,6-二烯-3,5-庚二酮

双脱甲氧基姜黄素：1，7-双(4-羟基苯基)-1，6-二烯-3，5-庚二酮

# 2.2 分子式

姜黄素： $\mathrm{C_{21}H_{20}O_6}$

脱甲氧基姜黄素： $\mathrm{C_{20}H_{18}O_5}$

双脱甲氧基姜黄素： $\mathrm{C_{19}H_{16}O_4}$

# 2.3 结构式

![](images/fd03d9a490bca1d364820656b461a5686c1e172f044682086607419d73ba78f6.jpg)

姜黄素： $\mathrm{R}_1 = \mathrm{R}_2 = \mathrm{OCH}_3$

脱甲氧基姜黄素： $\mathrm{R_1 = OCH_3}$ 1 $\mathrm{R}_2 = \mathrm{H}$

双脱甲氧基姜黄素： $\mathrm{R}_1 = \mathrm{R}_2 = \mathrm{H}$

# 2.4 相对分子质量

姜黄素：368.39（按2007年国际相对原子质量）

脱甲氧基姜黄素：338.39（按2007年国际相对原子质量）

双脱甲氧基姜黄素：308.39（按2007年国际相对原子质量）

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 颜色反应

称取质量相当于 $0.1\mathrm{g}$ 、色价为150的试样，溶于 $200~\mathrm{mL}$ 乙醇中，溶液应呈黄色微带绿色荧光。

称取质量相当于 $1\mathrm{g}$ 、色价为150的试样，溶于 $100~\mathrm{mL}$ 乙醇中，加盐酸直至溶液颜色变成浅橙色，此溶液为试样液。在试样液中加入硼酸，溶液应呈红橙色。

# A.2.2 最大吸收波长

取A.3.3色价测定中的试样液，用分光光度计检测，在波长 $420\mathrm{nm}\sim 430\mathrm{nm}$ 之间应有最大吸收峰。

# A.2.3 薄层层析

# A.2.3.1 硅胶薄层板的制法

称取硅胶 $10\mathrm{g}$ ，加水 $20\mathrm{mL}$ ，置于研钵中研磨成稀稠适宜的浆状液，均匀涂布于玻璃板上，涂层厚约 $0.5\mathrm{mm},100^{\circ}\mathrm{C}$ 活化 $30\mathrm{min}$ 。

# A.2.3.2 展开剂

乙醇：3-甲基-1-丁醇：水：氨 $= 4:4:2:1$

# A.2.3.3 试样液的制备

称取质量相当于 $1\mathrm{g}$ 、色价为150的试样，溶于 $100~\mathrm{mL}$ 乙醇中，在约 $3000\mathrm{r / min}$ 条件下离心 $10\mathrm{min}$ 取上清液为试样液。

# A.2.3.4 点样

在硅胶薄层板上，用微量注射器吸取 $5\mu \mathrm{L}$ 试样液点样。

# A.2.3.5 展开与观察

将点好样的薄板放入预先已用展开剂饱和的层析缸中，按上行法展开，待溶剂前沿超过 $10~\mathrm{cm}$ 时，取出，风干。观察应至少有两个黄色斑点， $R_{f}$ 值应在 $0.40\sim 0.85$ 之间。所有斑点在紫外光（约 $366~\mathrm{nm}$ 下应显黄色荧光。

# A.3 色价 $E_{1\mathrm{cm}}^{1\%}(425\pm 5)\mathbf{nm}$ 的测定

# A.3.1 仪器和设备

分光光度计。

# A.3.2 试剂和溶液

$95\%$ 乙醇。

# A.3.3 分析步骤

称取 $0.1\mathrm{g}\sim 0.3\mathrm{g}$ 粉末试样，精确至 $0.0002\mathrm{g}$ ，或称取 $1\mathrm{g}$ 浸膏或液体试样，精确至 $0.0002\mathrm{g}$ ，用 $95\%$ 乙醇溶解，转移至 $100~\mathrm{mL}$ 容量瓶中，加 $95\%$ 乙醇定容至刻度，摇匀，然后吸取 $10~\mathrm{mL}$ ，转移至 $100~\mathrm{mL}$ 容量瓶中，加 $95\%$ 乙醇定容至刻度，摇匀，若溶液浑浊，可用滤纸过滤，得到试样液。取此试样液置于 $1\mathrm{cm}$ 比色皿中，以 $95\%$ 乙醇做空白对照，用分光光度计在 $420~\mathrm{nm}\sim 430~\mathrm{nm}$ 内的最大吸收波长处测定吸光度（吸光度应控制在 $0.2\sim 0.8$ 之间，否则应调整试样液浓度，再重新测定吸光度）。

# A.3.4 结果计算

色价 $E_{1\mathrm{cm}}^{1\%}(425\pm 5)\mathrm{nm}$ ，按式(A.1)计算：

$$
E _ {1 \mathrm {cm}} ^ {1 \%} (425 \pm 5) \mathrm {nm} = \frac {A}{c \times 100} \quad \dots \tag{A.1}
$$

式中：

A ——实测试样液的吸光度；

$c$ ——被测试样液的浓度，单位为克每毫升 $(\mathrm{g / mL})$ ；

100——浓度换算系数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

# A.4 残留溶剂（正己烷、异丙醇和乙酸乙酯）的测定

# A.4.1 仪器和设备

气相色谱仪，带氢火焰离子检测器(FID)和顶空进样器。

# A.4.2 试剂和溶液

A.4.2.1 待测组分标准品：正己烷、异丙醇和乙酸乙酯，色谱纯。  
A.4.2.2 空白样：含非常少溶剂的试样。  
A.4.2.3 3-甲基-2-戊酮：色谱纯，做内标物用。  
A.4.2.4 二甲基甲酰胺(DMF)：色谱纯。

# A.4.3 参考色谱条件

A.4.3.1 色谱柱：石英毛细管柱，长 $30\mathrm{m}$ ，内径 $0.53\mathrm{mm}$ ，涂层为 $100\%$ 二甲基聚硅氧烷，涂层厚度为 $5\mu \mathrm{m}$ 。或同等性能的色谱柱。  
A.4.3.2 载气：高纯氮。

A.4.3.3 载气流速： $2\mathrm{mL} / \mathrm{min}$   
A.4.3.4 柱温： $40^{\circ}\mathrm{C}$ 保持 $5\mathrm{min}$ ，以 $5^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $90^{\circ}\mathrm{C}$ ，保持 $6\mathrm{min}$ 。  
A.4.3.5 进样口温度： $140^{\circ}C$   
A.4.3.6 检测器温度： $300^{\circ}C$   
A.4.3.7 进样：分流进样，分流比 $1:10$ ；进样量 $1.0\mathrm{mL}$ 。

# A.4.4 顶空采样器参考条件

A.4.4.1 试样加热温度： $60^{\circ}C$ 。  
A.4.4.2 试样加热时间： $40\mathrm{min}$   
A.4.4.3 注射器温度： $70^{\circ}C$   
A.4.4.4 传质温度： $80^{\circ}C$

# A.4.5 分析步骤

# A.4.5.1 内标溶液的制备

移取 $50.0\mathrm{mL}$ 二甲基甲酰胺(DMF)到一个 $50~\mathrm{mL}$ 进样瓶中，封盖，称重进样瓶，精确至 $0.0001\mathrm{g}$ 。移取 $15~\mu \mathrm{L}$ 内标物3-甲基-2-戊酮，通过隔片将其注入进样瓶中，混匀，再称重进样瓶，精确至 $0.0001\mathrm{g}$ 。

# A.4.5.2 空白溶液的制备

称取 $0.6\mathrm{g}$ 空白样，精确至 $0.0001\mathrm{g}$ ，置于顶空瓶中。加入 $5.0\mathrm{mL}$ 二甲基甲酰胺(DMF)和 $1.0\mathrm{mL}$ 内标溶液。超声溶解，混匀。

# A.4.5.3 试样溶液的制备

称取 $0.6\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，置于顶空瓶中。加入 $5.0\mathrm{mL}$ 二甲基甲酰胺(DMF)和 $1.0\mathrm{mL}$ 内标溶液。超声溶解，混匀。

# A.4.5.4 标准溶液的制备

溶液A：移取 $50.0\mathrm{mL}$ 二甲基甲酰胺(DMF)到一个 $50~\mathrm{mL}$ 进样瓶中，封盖，称重进样瓶，精确至 $0.0001\mathrm{g}$ 。移取 $50~\mu \mathrm{L}$ 待测组分标准品，通过隔片将其注入进样瓶中，混匀，再称重进样瓶，精确至 $0.0001\mathrm{g}$ 。

称取 $0.6\mathrm{g}$ 空白样，精确至 $0.0001\mathrm{g}$ ，置于顶空瓶中。加入 $4.9\mathrm{mL}$ 二甲基甲酰胺(DMF)和 $1.0\mathrm{mL}$ 内标溶液。通过隔片注入 $0.1\mathrm{mL}$ 溶液A。超声溶解，混匀。

# A.4.5.5 测定

在A.4.3和A.4.4参考操作条件下，分别对试样溶液、空白溶液和标准溶液进行色谱分析。

# A.4.6 结果计算

# A.4.6.1 校准因子 $f_{i}$

校准因子 $f_{i}$ ，按式(A.2)计算：

$$
f _ {i} = \frac {m _ {i}}{m _ {0} \times \left(A _ {\mathrm {f}} - A _ {\mathrm {g}}\right) \times 1 0} \quad \dots \tag {A.2}
$$

式中：

$m_{i}$ ——溶液A中待测组分标准品的质量，单位为毫克 $(\mathrm{mg})$ ；

$m_0$ ——内标溶液中内标物的质量，单位为毫克 $(\mathrm{mg})$ ；

$A_{\mathrm{f}}$ ——标准溶液色谱图中待测组分峰面积与内标物峰面积的比值；

$A_{\mathrm{g}}$ ——空白溶液色谱图中待测组分峰面积与内标物峰面积的比值；

10 ——质量换算系数。

# A.4.6.2 待测组分

待测组分(正己烷、异丙醇和乙酸乙酯)的含量 $\omega_{i}$ , 以毫克每千克 $(\mathrm{mg / kg})$ 计, 按式(A.3)计算:

$$
w _ {i} = \frac {A _ {i} \times m _ {0} \times f _ {i} \times 1 0 0 0}{m _ {2} \times 5 0} \tag {A.3}
$$

式中：

$A_{i}$ ——试样溶液色谱图中待测组分峰面积与内标物峰面积的比值；

$m_{0}$ ——内标溶液中内标物的质量，单位为毫克 $(\mathrm{mg})$ ；

$f_{i}$ ——校准因子；

1000—质量换算系数；

$m_{2}$ ——试样的质量，单位为克(g)；

50 体积换算系数。

由式(A.3)计算得到3种组分的含量 $w_{1}, w_{2}$ 和 $w_{3}$ ，取各组分含量之和即为试样中残留溶剂的含量。

# 食品安全国家标准

# 食品中胭脂虫红的测定

# 1 范围

本标准规定了食品中胭脂虫红的液相色谱测定方法。

本标准适用于食品中胭脂虫红的测定。

# 2 原理

试样中的胭脂虫红，经盐酸溶液提取，固相萃取柱净化后，反相 $\mathrm{C_{18}}$ 液相色谱柱分离，紫外-可见光检测器检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.3 盐酸(HCl)。  
3.1.4 磷酸 $\mathrm{(H_3PO_4)}$ 。

# 3.2 试剂配制

3.2.1 盐酸溶液 $(2.0\mathrm{mol} / \mathrm{L})$ ：量取 $168~\mathrm{mL}$ 盐酸至盛有 $800~\mathrm{mL}$ 水的 $1000~\mathrm{mL}$ 容量瓶中，充分混匀后，冷却至室温，用水定容至刻度。  
3.2.2 磷酸甲醇溶液 $(2\%)$ ：量取 $20~\mathrm{mL}$ 磷酸至盛有 $800~\mathrm{mL}$ 甲醇的 $1000~\mathrm{mL}$ 容量瓶中，充分混匀后，冷却至室温，用甲醇定容至刻度。  
3.2.3 磷酸水溶液 $(0.1\%)$ ：吸取 $1.0~\mathrm{mL}$ 磷酸至盛有 $800~\mathrm{mL}$ 水的 $1000~\mathrm{mL}$ 容量瓶中，充分混匀后，冷却至室温，用水定容至刻度。  
3.2.4 磷酸乙腈溶液 $(0.1\%)$ ：吸取 $1.0~\mathrm{mL}$ 磷酸至盛有 $800~\mathrm{mL}$ 乙腈的 $1000~\mathrm{mL}$ 容量瓶中，充分混匀后，冷却至室温，用乙腈定容至刻度。

# 3.3 标准品

胭脂红酸标准品 $\mathrm{(C_{22}H_{20}O_{13}}$ ，CAS号：1260-17-9)：纯度 $\geqslant 95.5\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 胀脂红酸标准储备液 $(1.00\mathrm{mg / mL})$ ：准确称取 $100.0\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )标准品，用水溶解，并

定容至 $100\mathrm{mL}$ 容量瓶中，摇匀，于 $4^{\circ}\mathrm{C}$ 下避光保存，有效期3个月。

3.4.2 胞脂红酸标准系列工作液：用移液管(精确至 $0.01 \mathrm{~mL}$ )准确吸取标准储备液 $(1.00 \mathrm{mg} / \mathrm{mL})$ $0.5 \mathrm{~mL}$ 于 $100 \mathrm{~mL}$ 容量瓶中；分别准确吸取标准储备液 $(1.00 \mathrm{mg} / \mathrm{mL})$ $0.1 \mathrm{~mL}, 0.2 \mathrm{~mL}, 0.5 \mathrm{~mL}$ 和 $1.0 \mathrm{~mL}$ 于 $10.0 \mathrm{~mL}$ 容量瓶中，用磷酸水溶液 $(0.1\%)$ 定容至刻度，摇匀，配制成质量浓度分别为 $5.00 \mathrm{mg} / \mathrm{L}, 10.0 \mathrm{mg} / \mathrm{L}, 20.0 \mathrm{mg} / \mathrm{L}, 50.0 \mathrm{mg} / \mathrm{L}$ 和 $100 \mathrm{mg} / \mathrm{L}$ 胞脂红酸标准系列工作液。临用现配。

# 3.5 材料

固相萃取柱（ $150\mathrm{mg} / 6\mathrm{mL}$ ，混合型强阴离子交换反相柱，填料为含亲水基团的聚苯乙烯/二乙烯基苯共聚物上键合季铵基团，或相当者）。

# 4 仪器和设备

4.1 高效液相色谱仪：带紫外-可见光检测器。  
4.2 天平：感量分别为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.3 恒温水浴锅。  
4.4 涡旋混合器。  
4.5 超声波发生器。  
4.6 高速离心机。  
4.7 固相萃取装置。  
4.8 氮吹仪。  
4.9 粉碎机。

# 5 分析步骤

# 5.1 试样制备

液态样品摇匀待提取；基质均匀的半固态样品和粉状样品直接待提取；其他样品需匀浆或粉碎均匀待提取。制备好的试样于 $0^{\circ}\mathrm{C} \sim 5^{\circ}\mathrm{C}$ 保存，尽快测定。

# 5.2 试样提取

# 5.2.1 液体、半固体试样（含乳半固体除外）

称取 $2\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 试样于 $50~\mathrm{mL}$ 离心管中，加入盐酸溶液 $(2.0\mathrm{mol} / \mathrm{L})40~\mathrm{mL}$ ，旋紧盖子，摇匀，置于沸水浴中加热 $30\mathrm{min}$ ，取出冷却至室温后，振荡 $5\mathrm{min}$ ，超声 $5\mathrm{min}$ ， $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液转移至 $50.0~\mathrm{mL}$ 容量瓶中，用水定容至刻度，经适量脱脂棉过滤，待净化。

# 5.2.2 固体、含乳半固体试样（调制乳粉除外）

称取 $2\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 试样于 $50~\mathrm{mL}$ 离心管中，加入盐酸溶液 $(2.0\mathrm{mol} / \mathrm{L})40~\mathrm{mL}$ ，旋紧盖子，摇匀，置于沸水浴中加热 $30\mathrm{min}$ ，取出冷却至室温后，振荡 $5\mathrm{min}$ ，超声 $5\mathrm{min}$ ， $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液转移至 $100~\mathrm{mL}$ 容量瓶中，残渣加入盐酸溶液 $(2.0\mathrm{mol} / \mathrm{L})40~\mathrm{mL}$ ，常温条件下，重复提取1次，合并2次提取液，用水定容至刻度，经适量脱脂棉过滤，待净化。

# 5.2.3 调制乳粉

称取 $2\mathrm{g}$ (精确至 $0.001\mathrm{g}$ ) 试样于 $50~\mathrm{mL}$ 离心管中，加入盐酸溶液 $(2.0\mathrm{mol} / \mathrm{L})40~\mathrm{mL}$ ，旋紧盖子，摇

匀，置于沸水浴中加热 $30\mathrm{min}$ ，取出冷却至室温后，振荡 $5\mathrm{min}$ ，超声 $5\mathrm{min}$ ， $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液转移至 $250\mathrm{mL}$ 容量瓶中，残渣加入盐酸溶液 $(2.0\mathrm{mol} / \mathrm{L})40\mathrm{mL}$ ，常温条件下，重复提取2次，合并3次提取液，用水定容至刻度，经适量脱脂棉过滤，待净化。

# 5.3 试样净化

固相萃取柱使用前依次用 $6\mathrm{mL}$ 甲醇、 $6\mathrm{mL}$ 水进行活化，保持柱体湿润，取过滤液 $25.0~\mathrm{mL}$ 通过活化后的固相萃取柱 $(150\mathrm{mg} / 6\mathrm{mL})$ ，依次用 $12\mathrm{mL}$ 水和 $6\mathrm{mL}$ 甲醇淋洗，弃去淋洗液，真空抽至柱体近干，用磷酸甲醇溶液 $(2\%)$ $6\mathrm{mL}$ 进行洗脱，收集洗脱液，于 $40^{\circ}\mathrm{C}$ 水浴下氮吹浓缩至近干，加入 $2.00~\mathrm{mL}$ 磷酸水溶液 $(0.1\%)$ 复溶，涡旋混匀， $12000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液待进样。

# 5.4 空白试验

除不加试样外，按5.2和5.3与试样同时处理。

# 5.5 仪器参考条件

5.5.1 色谱柱： $\mathrm{C_{18}(4.6~mm\times 150~mm,5.0~\mu m)}$ 或性能相当者；  
5.5.2 流动相：A为磷酸水溶液 $(0.1\%)$ ，B为磷酸乙腈溶液 $(0.1\%)$ ，洗脱梯度程序见表1；  
5.5.3 柱温： $40^{\circ}\mathrm{C}$   
5.5.4 流速： $1.2\mathrm{mL} / \mathrm{min}$   
5.5.5 检测波长： $494\mathrm{nm}$   
5.5.6 进样量： $10~\mu \mathrm{L}$

# 5.6 标准曲线的制作

将标准系列工作液分别注入液相色谱仪中，测定相应的峰面积，以标准工作液的浓度为横坐标，以峰面积响应值为纵坐标，绘制标准曲线。胭脂红酸标准溶液的色谱图参见附录A中图A.1。

# 5.7 试样溶液的测定

将试样溶液注入液相色谱仪中，根据保留时间定性，变化范围应在 $\pm 2.5\%$ 之内，峰面积定量，根据标准曲线得到待测液中胭脂红酸的浓度。

# 6 分析结果的表述

试样中胭脂虫红（以胭脂红酸计）含量按公式(1)计算。

$$
X = \frac {c \times V \times 2 . 0 0 \times 1 0 0 0}{m \times 2 5 . 0 \times 1 0 0 0 \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中胭脂虫红(以胭脂红酸计)的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$c$ ——待测试样中胭脂虫红(以胭脂红酸计)的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样提取液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

2.00 ——分取提取液净化后的定容体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数；

$m$ ——试样的取样量，单位为克 $(\mathrm{g})$ ；

25.0 ——分取提取液的体积，单位为毫升 $(\mathrm{mL})$ ；

计算结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

本方法的检出限为：当称样量为 $2\mathrm{g}$ 时，液体及半固体试样（含乳半固体除外）中胭脂虫红（以胭脂红酸计）检出限为 $0.006\mathrm{g / kg}$ ，固体及含乳半固体试样（调制乳粉除外）中胭脂虫红（以胭脂红酸计）检出限为 $0.01\mathrm{g / kg}$ ，调制乳粉中胭脂虫红（以胭脂红酸计）检出限为 $0.03\mathrm{g / kg}$ 。

本方法的定量限为：当称样量为 $2\mathrm{g}$ 时，液体及半固体试样（含乳半固体除外）中胭脂虫红（以胭脂红酸计）定量限为 $0.02\mathrm{g / kg}$ ，固体及含乳半固体试样（调制乳粉除外）中胭脂虫红（以胭脂红酸计）定量限为 $0.04\mathrm{g / kg}$ ，调制乳粉中胭脂虫红（以胭脂红酸计）定量限为 $0.1\mathrm{g / kg}$ 。

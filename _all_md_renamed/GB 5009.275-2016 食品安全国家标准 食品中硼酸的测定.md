2016-12-23 发布

# 前言

本标准代替GB/T21918—2008《食品中硼酸的测定》。

本标准与GB/T21918—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中硼酸的测定”；  
——删除了第二法 电感耦合等离子体原子发射光谱法和电感耦合等离子体质谱法；  
——增加了适用范围；  
——增加了定量限；  
——修改了检验结果表述。

# 食品安全国家标准

# 食品中硼酸的测定

# 1 范围

本标准规定了食品中硼酸的测定方法。

本标准适用于水产品、肉制品（如肉丸）、豆类、面食类、腐竹、粽子、糕点、酱油、凉粉、凉皮等食品中硼酸的测定。

# 2 原理

通过乙基己二醇-三氯甲烷溶液对样品中的硼酸进行快速的富集、萃取，除去共存盐类的影响，利用硫酸与姜黄混合生成的质子化姜黄与硼酸反应生成红色产物。溶液颜色的深浅与样品中硼酸含量成正比，通过比色可以测定样品中硼酸的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 硫酸 $\mathrm{(H_2SO_4)}$ 。  
3.1.2 无水乙醇 $\mathrm{(CH_3CH_2OH)}$   
3.1.3 甲醇 $\mathrm{(CH_3OH)}$   
3.1.4 亚铁氰化钾 $\left[\mathrm{K}_4\mathrm{Fe}(\mathrm{CN})_6\cdot 3\mathrm{H}_2\mathrm{O}\right]$   
3.1.5 乙酸锌 $\left[\mathrm{Zn}(\mathrm{CH}_3\mathrm{COO})_2\cdot 2\mathrm{H}_2\mathrm{O}\right]$   
3.1.6 姜黄色素。  
3.1.7 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.8 2-乙基-1,3-己二醇 $\mathrm{CH_3CH_2CH_2CH(OH)CH(C_2H_5)CH_2OH}]$   
3.1.9 三氯甲烷 $\mathrm{(CHCl_3)}$   
3.1.10 硼酸 $(\mathrm{H}_3\mathrm{BO}_3)$ 。

# 3.2 试剂配制

3.2.1 硫酸溶液 $(1 + 1)$ ：取等体积的硫酸和水，将硫酸沿烧杯壁缓慢注入水中，并用玻璃棒不断搅拌，使热量及时散失。  
3.2.2 亚铁氰化钾溶液：称取 $106.0\mathrm{g}$ 亚铁氰化钾，用水溶解，并稀释至 $1000~\mathrm{mL}$   
3.2.3 乙酸锌溶液：称取 $220.0\mathrm{g}$ 乙酸锌，先加 $30~\mathrm{mL}$ 冰乙酸溶解，用水稀释至 $1000~\mathrm{mL}$   
3.2.4 姜黄-冰乙酸溶液：称取姜黄色素 $0.10\mathrm{g}$ 溶于 $100~\mathrm{mL}$ 冰乙酸中，此溶液保存于塑料容器中。  
3.2.5 2-乙基-1,3-己二醇-三氯甲烷溶液 $(\mathrm{EHD - CHCl}_3$ 溶液)：取2-乙基-1，3-己二醇 $10~\mathrm{mL}$ ，加三氯甲烷稀释至 $100~\mathrm{mL}$ ，此溶液保存于塑料容器中。

# 3.3 标准溶液配制

3.3.1 硼酸标准储备液：准确称取在硫酸干燥器中干燥 $5\mathrm{h}$ 以上的硼酸 $0.5000\mathrm{g}$ ，溶于水并定容至 $1000\mathrm{mL}$ ，保存于塑料容器中。此硼酸标准储备液的浓度为 $500\mu \mathrm{g} / \mathrm{mL}$ 。  
3.3.2 硼酸标准溶液：取硼酸标准储备液 $10.00\mathrm{mL}$ ，加水定容至 $1000\mathrm{mL}$ ，保存于塑料容器中。此硼酸标准溶液的浓度为 $5\mu \mathrm{g} / \mathrm{mL}$ 。所配制溶液于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中可储存3个月。

# 4 仪器和设备

注：试验中使用的容器均为塑料容器。

4.1 电子天平：感量为 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
4.2 分光光度计。  
4.3 高速捣碎机。  
4.4 涡旋振荡器。

# 5 分析步骤

# 5.1 试样制备

固体样品：称取经高速捣碎机捣碎的样品 $2\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 于 $150~\mathrm{mL}$ 塑料烧杯中，加 $40~\mathrm{mL}\sim 60~\mathrm{mL}$ 水混匀，缓慢滴加 $2\mathrm{mL}$ 硫酸，超声 $10\mathrm{min}$ 促进溶解混合。加乙酸锌溶液 $5\mathrm{mL}$ ，亚铁氰化钾溶液 $5\mathrm{mL}$ ，加水定容至 $100~\mathrm{mL}$ ，过滤后作为制备的试样溶液。根据样品含量取 $1.00~\mathrm{mL}\sim$ $3.00~\mathrm{mL}$ 制备的试样溶液于 $25~\mathrm{mL}$ 塑料试管中，加水至 $5\mathrm{mL}$ 。加硫酸溶液 $(1 + 1)1\mathrm{mL}$ ，振荡混匀，加EHD- $\mathrm{CHCl}_3$ 溶液 $5.00~\mathrm{mL}$ ，盖上盖子，涡旋振荡器振摇 $2\mathrm{min}$ ，静置分层，吸取下层的EHD- $\mathrm{CHCl}_3$ 溶液并通过 $\phi 7\mathrm{cm}$ 干燥快速滤纸过滤。过滤液作为样品测试液。

液体样品：称取样品 $2\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 于 $150~\mathrm{mL}$ 塑料烧杯中，加 $40~\mathrm{mL}$ 水混匀，加乙酸锌溶液 $5\mathrm{mL}$ ，亚铁氰化钾溶液 $5\mathrm{mL}$ ，加水定容至 $100~\mathrm{mL}$ ，过滤后作为制备的试样溶液。根据样品含量取 $1.00~\mathrm{mL}\sim 3.00~\mathrm{mL}$ 制备的试样溶液于 $25~\mathrm{mL}$ 塑料试管中，加水至 $5\mathrm{mL}$ ，加硫酸溶液 $(1 + 1)1\mathrm{mL}$ ，振荡混匀，加EHD- $\mathrm{CHCl}_3$ 溶液 $5.00~\mathrm{mL}$ ，盖上盖子，涡旋振荡器振摇 $2\mathrm{min}$ ，静置分层，吸取下层的EHD- $\mathrm{CHCl}_3$ 溶液并通过 $\phi 7\mathrm{cm}$ 干燥快速滤纸过滤。过滤液作为样品测试液。

注：如果萃取过程出现乳化现象，可将萃取液 $3000\mathrm{r / min}$ 离心 $3\mathrm{min}$ 或在测定体系中加入 $1\mathrm{mL}$ 甲醇以避免乳化或沉淀现象，如加入甲醇应保持加水定容总体积为 $5\mathrm{mL}$ 。

# 5.2 标准曲线的制作

准确吸取硼酸标准溶液 $0.00\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},3.00\mathrm{mL},4.00\mathrm{mL},5.00\mathrm{mL}$ 于 $25~\mathrm{mL}$ 塑料试管中，各加水至 $5\mathrm{mL}$ 。加硫酸溶液 $(1 + 1)1\mathrm{mL}$ ，振荡混匀，加 $\mathrm{EHD - CHCl_3}$ 溶液 $5.00~\mathrm{mL}$ ，盖上盖子，涡旋振荡器振摇 $2\mathrm{min}$ ，静置分层，吸取下层的 $\mathrm{EHD - CHCl_3}$ 溶液并通过 $\phi 7\mathrm{cm}$ 干燥快速滤纸过滤。

各取 $1.00\mathrm{mL}$ 过滤液于 $50~\mathrm{mL}$ 塑料试管中，依次加入姜黄-冰乙酸溶液 $1.0~\mathrm{mL}$ ，硫酸 $0.5~\mathrm{mL}$ ，摇匀，静置 $30\mathrm{min}$ ，加无水乙醇 $25~\mathrm{mL}$ ，静置 $10\mathrm{min}$ ，于 $550~\mathrm{nm}$ 处 $1\mathrm{cm}$ 比色皿测定吸光度。以标准系列的硼酸含量 $(\mu \mathrm{g})$ 为横坐标，以吸光度为纵坐标绘制标准曲线。

# 5.3 试样溶液的测定

准确吸取样品测试液 $1.00\mathrm{mL}$ 于 $50~\mathrm{mL}$ 塑料试管中，以下操作同5.2中的显色、比色步骤。测定吸光度值，根据标准曲线得到试样溶液中硼酸的浓度。

# 6 分析结果的表述

试样中硼酸含量按式(1)计算：

$$
X = \frac {C \times 1 0 0 0 \times V _ {1}}{m \times 1 0 0 0 \times V _ {2}} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中硼酸的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$C$ ——由标准曲线得到的试样溶液中的硼酸的浓度，单位为微克 $(\mu \mathrm{g})$ ；

1000——换算系数；

$V_{1}$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——测定所量取的试样溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法的检出限为 $2.50\mathrm{mg / kg}$ ，定量限为 $7.50~\mathrm{mg / kg}$

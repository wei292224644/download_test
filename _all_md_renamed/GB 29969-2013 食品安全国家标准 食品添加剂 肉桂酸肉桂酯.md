# 食品安全国家标准

# 食品添加剂 肉桂酸肉桂酯

# 1 范围

本标准适用于由肉桂酸乙酯和肉桂醇为原料制得的食品添加剂肉桂酸肉桂酯。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

3-苯基-2-丙烯醇3-苯丙烯酸酯

# 2.2 分子式

$$
\mathrm {C} _ {1 8} \mathrm {H} _ {1 6} \mathrm {O} _ {2}
$$

# 2.3 结构式

![](images/93a11eeea40ddc41da173017be6d5ae1e7aac2c8dd1dd67ad875dc553500e17c.jpg)

# 2.4 相对分子质量

264.32（按2007年国际相对原子质量）

# 附录A

# 肉桂酸肉桂酯含量的测定

# A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A. 1.3 检测器：氢火焰离子化检测器。

# A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

试样制备：称取试样 $1\mathrm{g}$ 溶于 $10\mathrm{mL}$ 无水乙醇中，摇匀备用。

# A. 3 重复性及结果表示

按GB/T11538—2006中11.4规定进行，应符合要求。

# B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$   
B.2.2 固定相： $100\%$ 二甲基聚硅氧烷。  
B.2.3 膜厚： $0.45\mu \mathrm{m}$   
B.2.4 色谱炉温度： $260^{\circ}C$ 恒温。  
B.2.5 进样口温度： $280^{\circ}C$ 。  
B.2.6 检测器温度： $300^{\circ}C$ 。  
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 载气流速： $3\mathrm{mL / min}$   
B.2.10 进样量： $0.2\mu \mathrm{L}$   
B.2.11 分流比： $50:1$

2016-10-19 发布

2017-04-19 实施

# 前言

本标准代替GB/T23296.18—2009《食品接触材料 高分子材料 食品模拟物中乙二醇与二甘醇的测定 气相色谱法》。

本标准与GB/T23296.18—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 乙二醇和二甘醇迁移量的测定”；

——增加了食品模拟物的试验种类。

# 食品安全国家标准

# 食品接触材料及制品 乙二醇和二甘醇迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中乙二醇和二甘醇迁移量的气相色谱测定方法。

本标准适用于食品接触材料及制品中乙二醇和二甘醇迁移量的测定。

# 2 原理

对食品容器、包装材料模拟物中乙二醇和二甘醇采用毛细管气相色谱柱进行分离，氢火焰离子化检测器进行检测。其中水基、酸性、酒精类食品模拟物直接进样分析，油基食品模拟物通过水萃取后进样分析，采用内标法定量，内标物为1，4-丁二醇。

# 3 试剂和材料

除非另有说明，所有试剂均为分析纯，水为GB/T6682规定的一级水。试验中容器及转移器具应避免使用塑料材质。

# 3.1 试剂

3.1.1 冰乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
3.1.2 无水乙醇 $(\mathrm{C}_2\mathrm{H}_6\mathrm{O})$ 。  
3.1.3 甲醇 $(\mathrm{CH}_4\mathrm{O})$ 。  
3.1.4 庚烷 $(\mathrm{C_7H_{16}})$ 。  
3.1.5 配制水基、酸性、酒精类、油基食品模拟物所需试剂：符合GB31604.1的规定。

# 3.2 试剂配制

水基、酸性、酒精类、油基食品模拟物：按GB5009.156的规定操作。

# 3.3 标准品

3.3.1 乙二醇 $\mathrm{(C_2H_6O_2}$ ，CAS号：107-21-1）。  
3.3.2 二甘醇 $\mathrm{(C_4H_{10}O_3}$ ，CAS号：111-46-6）。  
3.3.3 1,4-丁二醇 $\mathrm{(C_4H_{10}O_2}$ ,CAS号：110-63-4）。

# 3.4 标准溶液配制

3.4.1 乙二醇 $(7500\mathrm{mg / L})$ 和二甘醇 $(7500\mathrm{mg / L})$ 储备液：分别准确称取乙二醇和二甘醇各 $0.75\mathrm{g}$ 于两个烧杯中（精确至 $0.1\mathrm{mg}$ ），用甲醇溶解后分别转入 $100\mathrm{mL}$ 容量瓶中，用甲醇定容。此溶液于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 密封避光保存1个月。

3.4.2 1,4-丁二醇 $(10000\mathrm{mg / L})$ 储备液：准确称取 $1\mathrm{g}1,4-$ 丁二醇于烧杯中(精确至 $0.1\mathrm{mg}$ )，用甲醇溶解后转入 $100~\mathrm{mL}$ 容量瓶中，用甲醇定容。此溶液于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 密封避光保存1个月。  
3.4.3 1,4-丁二醇 $(800\mathrm{mg / L})$ 溶液：准确吸取 $2\mathrm{mL}$ 的1，4-丁二醇 $(10000\mathrm{mg / L})$ 储备液于 $25~\mathrm{mL}$ 容量瓶中，用甲醇定容。此溶液于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 密封避光保存，该溶液现用现配。  
3.4.4 标准中间溶液：准确吸取 $2\mathrm{mL}$ 的1,4-丁二醇 $(10000\mathrm{mg / L})$ 储备液于6个 $25~\mathrm{mL}$ 容量瓶中，分别加入 $0.0\mathrm{mL},0.5\mathrm{mL},1.0\mathrm{mL},2.5\mathrm{mL},5.0\mathrm{mL},10.0\mathrm{mL}$ 乙二醇和二甘醇储备液，用甲醇定容，得到乙二醇和二甘醇浓度均为 $0.0\mathrm{mg / L},150.0\mathrm{mg / L},300.0\mathrm{mg / L},750.0\mathrm{mg / L},1500\mathrm{mg / L},3000\mathrm{mg / L}$ 的标准中间溶液，1,4-丁二醇浓度为 $800~\mathrm{mg / L}$ 。此溶液于 $0^{\circ}C\sim 4^{\circ}C$ 密封避光保存，该溶液现用现配。

# 4 仪器和设备

4.1 气相色谱仪：配氢火焰离子化检测器和数据自动分析软件。  
4.2 分析天平：感量 $0.0001\mathrm{g}$ 和 $0.01\mathrm{g}$ ，以及 $0.1\mathrm{g}$ 。  
4.3 分液漏斗： $250~\mathrm{mL}$ ，配聚四氟乙烯或玻璃瓶塞，使用前应干燥。  
4.4 容量瓶： $25~\mathrm{mL}$ 和 $50~\mathrm{mL}$ ，以及 $100~\mathrm{mL}$ 。  
4.5 带刻度移液管： $1\mathrm{mL}$ 和 $2\mathrm{mL}$   
4.6 玻璃注射器： $50~\mathrm{mL}$   
4.7 过滤器： $0.22\mu \mathrm{m}$ 有机相滤膜，针式。

# 5 分析步骤

# 5.1 标准工作溶液及试样制备

# 5.1.1 标准工作溶液的制备

# 5.1.1.1 水基、酸性、酒精类食品模拟物标准工作溶液

分别准确吸取各浓度标准中间溶液 $1\mathrm{mL}$ 于6个 $50~\mathrm{mL}$ 容量瓶中，用水定容，得到水中乙二醇和二甘醇浓度均为 $0.0\mathrm{mg / L},3.0\mathrm{mg / L},6.0\mathrm{mg / L},15.0\mathrm{mg / L},30.0\mathrm{mg / L},60.0\mathrm{mg / L}$ 的标准工作溶液。采用同样方式，分别用对应水基、酸性、酒精类食品模拟物配制同样浓度系列的乙二醇和二甘醇标准工作溶液。

# 5.1.1.2 油基食品模拟物标准工作溶液

分别准确称取 $50\mathrm{g}$ （精确至 $0.1\mathrm{g}$ )油基食品模拟物至6个 $250~\mathrm{mL}$ 分液漏斗中，依次加入各浓度标准中间溶液 $1\mathrm{mL}$ ，混合均匀，得到油基食品模拟物中乙二醇和二甘醇浓度分别为 $0.0\mathrm{mg / kg},3.0\mathrm{mg}/$ $\mathrm{kg},6.0\mathrm{mg / kg},15.0\mathrm{mg / kg},30.0\mathrm{mg / kg},60.0\mathrm{mg / kg}$ 。在每个分液漏斗中加入 $50~\mathrm{mL}$ 庚烷，充分混合，加入水振荡萃取 $1\mathrm{min}$ ，静置 $5\mathrm{min}$ ，使两相分层；收集下层水相；再次用 $20~\mathrm{mL}$ 水萃取油基食品模拟物；将两次萃取水相合并，用水定容至 $50~\mathrm{mL}$ ；取 $1\mathrm{mL}$ 定容液通过 $0.2\mu \mathrm{m}$ 滤膜过滤后供测定用。

# 5.1.2 食品模拟物试液的制备

# 5.1.2.1 总则

根据待测样品的预期用途和使用条件，按照GB5009.156和GB31604.1规定的迁移试验方法及试验条件进行操作。迁移试验结束后，将迁移试验所得模拟物充分混匀后，取部分用于分析。

迁移试验获取的模拟物，可在 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱中避光保存一周。

# 5.1.2.2 水基、酸性、酒精类食品模拟物

在 $50~\mathrm{mL}$ 容量瓶中，准确移取 $1\mathrm{mL}$ 的1,4-丁二醇 $(800\mathrm{mg / L})$ 溶液，加入迁移试验中获得的水基、酸性、酒精类食品模拟物定容，混合均匀后取 $1\mathrm{mL}$ 过滤后供测定用。

# 5.1.2.3 油基食品模拟物

准确称取迁移试验中得到的油基食品模拟物 $50\mathrm{g}$ （精确至 $0.1\mathrm{g}$ ）于 $250~\mathrm{mL}$ 分液漏斗中，准确移取 $1\mathrm{mL}$ 的1,4-丁二醇 $(800\mathrm{mg / L})$ 溶液，加入 $50~\mathrm{mL}$ 庚烷，充分混合，加入 $20~\mathrm{mL}$ 水充分振荡萃取 $1\mathrm{min}$ 静置 $5\mathrm{min}$ ，使两相分层，收集下层水相；再次用 $20~\mathrm{mL}$ 水充分萃取油基食品模拟物；将两次萃取水相合并，用水定容至 $50~\mathrm{mL}$ ；混合均匀后取 $1\mathrm{mL}$ 过滤后供测定用。

# 5.1.3 空白试液的制备

按照5.1.2的操作处理未与食品容器、包装材料接触的食品模拟物。

# 5.2 测定

# 5.2.1 仪器参考条件

a）色谱柱：硝基对苯二酸修饰的聚乙二醇毛细管色谱柱，柱长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ ，膜厚 $1\mu \mathrm{m}$ 或同等性能的色谱柱；  
b）柱温： $100^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，以 $10^{\circ}\mathrm{C / min}$ 升温至 $200^{\circ}\mathrm{C}$ ，保持 $8\mathrm{min}$ ；  
c) 载气：氮气，流速 $1\mathrm{mL} / \mathrm{min}$   
d）检测器温度： $220^{\circ}C$   
e）进样模式：不分流进样， $0.5\mathrm{min}$ 后打开分流阀；  
f）进样口温度： $220^{\circ}C$   
g）进样体积： $1\mu \mathrm{L}$

# 5.2.2 标准工作曲线的制作

对5.1.1中制备的标准工作溶液在所列仪器参数下进行检测，以乙二醇或二甘醇质量浓度为横坐标，以乙二醇或二甘醇与内标物1,4-丁二醇色谱峰峰面积对应比值为纵坐标，绘制标准工作曲线，得到线性方程。标准色谱图参见附录A。

# 5.2.3 定量测定

对食品模拟物试液和空白试液依次进样，分别得到乙二醇或二甘醇与内标物1,4-丁二醇色谱峰峰面积对应比值；根据标准曲线得到试样溶液中乙二醇或二甘醇的含量，扣除空白值。

# 6 分析结果的表述

由标准曲线求得的试样溶液中乙二醇和二甘醇的浓度，按GB5009.156进行迁移量的计算，得到食品模拟物及制品中乙二醇和二甘醇的迁移量。结果保留三位有效数字。

# 7 重复性

在重复性条件下获得两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法对水基、酸性、酒精类食品模拟物中乙二醇或二甘醇的定量限为 $3\mathrm{mg / L}$ 。

本方法对油基食品模拟物中乙二醇或二甘醇的定量限为 $3\mathrm{mg / kg}$ 。

# 食品中阿斯巴甜和阿力甜的测定

2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替GB/T22253—2008《食品中阿力甜的测定》、GB/T22254—2008《食品中阿斯巴甜的测定》。

本标准与GB/T22253—2008和GB/T22254—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中阿斯巴甜和阿力甜的测定”；

——增加了适用范围。

# 食品安全国家标准

# 食品中阿斯巴甜和阿力甜的测定

# 1 范围

本标准规定了食品中阿斯巴甜和阿力甜的测定方法。

本标准适用于食品中阿斯巴甜和阿力甜的测定。

# 2 原理

根据阿斯巴甜和阿力甜易溶于水、甲醇和乙醇等极性溶剂而不溶于脂溶性溶剂特点，蔬菜及其制品、水果及其制品、食用菌和藻类、谷物及其制品、焙烤食品、膨化食品和果冻试样用甲醇水溶液在超声波振荡下提取；浓缩果汁、碳酸饮料、固体饮料类、餐桌调味料和除胶基糖果以外的其他糖果试样用水提取；乳制品、含乳饮料类和冷冻饮品试样用乙醇沉淀蛋白后用乙醇水溶液提取；胶基糖果用正己烷溶解胶基并用水提取；脂肪类乳化制品、可可制品、巧克力及巧克力制品、坚果与籽类、水产及其制品、蛋制品用水提取，然后用正己烷除去脂类成分。各提取液在液相色谱 $\mathrm{C_{18}}$ 反相柱上进行分离，在波长 $200\mathrm{nm}$ 处检测，以色谱峰的保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，所用试剂均为分析纯，水为GB/T6682规定的实验室一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 乙醇 $\mathrm{(CH_3CH_2OH)}$ ：优级纯。

# 3.2 标准品

3.2.1 阿力甜标准品 $\mathrm{(C_{14}H_{25}N_3O_4S}$ ，CAS号：80863-62-3）：纯度 $\geq 99\%$   
3.2.2 阿斯巴甜标准品 $\mathrm{(C_{14}H_{18}N_2O_5}$ ,CAS号：22839-47-0）：纯度 $\geq 99\%$ 。

# 3.3 标准溶液配制

3.3.1 阿斯巴甜和阿力甜的标准储备液 $(0.5\mathrm{mg / mL})$ ：各称取 $0.025\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ ）阿斯巴甜和阿力甜，用水溶解并转移至 $50~\mathrm{mL}$ 容量瓶中并定容至刻度，置于 $4^{\circ}C$ 左右的冰箱保存，有效期为 $90~\mathrm{d}$ 。  
3.3.2 阿斯巴甜和阿力甜混合标准工作液系列的制备：将阿斯巴甜和阿力甜标准储备液用水逐级稀释成混合标准系列，阿斯巴甜和阿力甜的浓度均分别为 $100\mu \mathrm{g / mL},50\mu \mathrm{g / mL},25\mu \mathrm{g / mL},10.0\mu \mathrm{g / mL}$ 、 $5.0\mu \mathrm{g / mL}$ 的标准使用溶液系列。置于 $4^{\circ}C$ 左右的冰箱保存，有效期为 $30~\mathrm{d}$ 。

# 4 仪器和设备

4.1 液相色谱仪：配有二极管阵列检测器或紫外检测器。

4.2 超声波震荡器。  
4.3 天平：感量为 $1\mathrm{mg}$ 和 $0.1\mathrm{mg}$   
4.4 离心机：转速 $\geqslant 4000\mathrm{r / min}$

# 5 分析步骤

# 5.1 试样制备及前处理

# 5.1.1 碳酸饮料、浓缩果汁、固体饮料、餐桌调味料和除胶基糖果以外的其他糖果

称取约 $5\mathrm{g}$ (精确到 $0.001\mathrm{g}$ ) 碳酸饮料试样于 $50~\mathrm{mL}$ 烧杯中，在 $50^{\circ}\mathrm{C}$ 水浴上除去二氧化碳，然后将试样全部转入 $25~\mathrm{mL}$ 容量瓶中，备用；称取约 $2\mathrm{g}$ 浓缩果汁试样(精确到 $0.001\mathrm{g}$ ) 于 $25~\mathrm{mL}$ 容量瓶中，备用；称取约 $1\mathrm{g}$ 的固体饮料或餐桌调味料或绞碎的糖果试样(精确到 $0.001\mathrm{g}$ ) 于 $50~\mathrm{mL}$ 烧杯中，加 $10~\mathrm{mL}$ 水后超声波震荡提取 $20\mathrm{min}$ ，将提取液移入 $25~\mathrm{mL}$ 容量瓶中，烧杯中再加入 $10~\mathrm{mL}$ 水超声波震荡提取 $10\mathrm{min}$ ，提取液移入同一 $25~\mathrm{mL}$ 容量瓶，备用。将上述容量瓶的液体用水定容，混匀， $4000\mathrm{r/min}$ 离心 $5\mathrm{min}$ ，上清液经 $0.45\mu \mathrm{m}$ 水系滤膜过滤后用于色谱分析。

# 5.1.2 乳制品、含乳饮料和冷冻饮品

对于含有固态果肉的液态乳制品需要用食品加工机进行匀浆，对于干酪等固态乳制品，需用食品加工机按试样与水的质量比1:4进行匀浆。

分别称取约 $5\mathrm{g}$ 液态乳制品、含乳饮料、冷冻饮品、固态乳制品匀浆试样（精确到 $0.001\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管，加入 $10~\mathrm{mL}$ 乙醇，盖上盖子；对于含乳饮料和冷冻饮品试样，首先轻轻上下颠倒离心管5次（不能振摇），对于乳制品，先将离心管涡旋混匀 $10~\mathrm{s}$ ，然后静置 $1\mathrm{min}$ ， $4000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液滤入 $25~\mathrm{mL}$ 容量瓶，沉淀用 $8\mathrm{mL}$ 乙醇-水 $(2 + 1)$ 洗涤，离心后上清液转移入同一 $25~\mathrm{mL}$ 容量瓶，用乙醇-水 $(2 + 1)$ 定容，经 $0.45\mu \mathrm{m}$ 有机系滤膜过滤后用于色谱分析。

# 5.1.3 果冻

对于可吸果冻和透明果冻，用玻棒搅匀，含有水果果肉的果冻需要用食品加工机进行匀浆。

称取约 $5\mathrm{g}$ (精确到 $0.001\mathrm{g}$ ) 制备均匀的果冻试样于 $50~\mathrm{mL}$ 的比色管中，加入 $25~\mathrm{mL}~80\%$ 的甲醇水溶液，在 $70^{\circ}\mathrm{C}$ 的水浴上加热 $10\mathrm{min}$ ，取出比色管，趁热将提取液转入 $50~\mathrm{mL}$ 容量瓶，再用 $15~\mathrm{mL}~80\%$ 的甲醇水溶液分两次清洗比色管，并每次振摇约 $10~\mathrm{s}$ ，并转入同一个 $50~\mathrm{mL}$ 的容量瓶，冷却至室温，用 $80\%$ 的甲醇水溶液定容到刻度，混匀， $4000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，将上清液经 $0.45\mu \mathrm{m}$ 有机系滤膜过滤后用于色谱分析。

# 5.1.4 蔬菜及其制品、水果及其制品、食用菌和藻类

水果及其制品试样如有果核首先需要去掉果核。

对于较干较硬的试样，用食品加工机按试样与水的质量比为 $1:4$ 进行匀浆，称取约 $5\mathrm{g}$ （精确到 $0.001\mathrm{g})$ 匀浆试样于 $25~\mathrm{mL}$ 的离心管中，加入 $10~\mathrm{mL}~70\%$ 的甲醇水溶液，摇匀，超声 $10\mathrm{min}$ $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液转入 $25~\mathrm{mL}$ 容量瓶，再加 $8\mathrm{mL}~50\%$ 的甲醇水溶液重复操作一次，上清液转入同一个 $25~\mathrm{mL}$ 容量瓶，最后用 $50\%$ 的甲醇水溶液定容，经 $0.45\mu \mathrm{m}$ 有机系滤膜过滤后用于色谱分析。

对于含糖多的、较粘的、较软的试样，用食品加工机按试样与水的质量比为 $1:2$ 进行匀浆，称取约 $3\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )匀浆试样于 $25~\mathrm{mL}$ 的离心管中；对于其他试样，用食品加工机按试样与水的质量比 $1:1$ 进行匀浆，称取约 $2\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )匀浆试样于 $25~\mathrm{mL}$ 的离心管中；然后向离心管加入 $10~\mathrm{mL}$

$60\%$ 的甲醇水溶液，摇匀，超声 $10\mathrm{min}$ ， $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液转入 $25\mathrm{mL}$ 容量瓶，再加 $10\mathrm{mL}$ $50\%$ 的甲醇水溶液重复操作一次，上清液转入同一个 $25\mathrm{mL}$ 容量瓶，最后用 $50\%$ 的甲醇水溶液定容，经 $0.45\mu \mathrm{m}$ 有机系滤膜过滤后用于色谱分析。

# 5.1.5 谷物及其制品、焙烤食品和膨化食品

试样需要用食品加工机进行均匀粉碎，称取 $1\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )粉碎试样于 $50~\mathrm{mL}$ 离心管中，加入 $12\mathrm{mL}50\%$ 甲醇水溶液，涡旋混匀，超声振荡提取 $10\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液转移入 $25~\mathrm{mL}$ 容量瓶中，再加 $10~\mathrm{mL}50\%$ 甲醇水溶液，涡旋混匀，超声振荡提取 $5\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液转入同一 $25~\mathrm{mL}$ 容量瓶中，用蒸馏水定容，经 $0.45\mu \mathrm{m}$ 有机系滤膜过滤后用于色谱分析。

# 5.1.6 胶基糖果、脂肪类乳化制品、可可制品、巧克力及巧克力制品、坚果与籽类、水产及其制品和蛋制品

胶基糖果：用剪刀将胶基糖果剪成细条状，称取约 $3\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )剪细的胶基糖果试样，转入 $100~\mathrm{mL}$ 的分液漏斗中，加入 $25~\mathrm{mL}$ 水剧烈振摇约 $1\mathrm{min}$ ，再加入 $30~\mathrm{mL}$ 正己烷，继续振摇直至口香糖全部溶解(约 $5\mathrm{min}$ )，静置分层约 $5\mathrm{min}$ ，将下层水相放入 $50~\mathrm{mL}$ 容量瓶，然后加入 $10~\mathrm{mL}$ 水到分液漏斗，轻轻振摇约 $10~\mathrm{s}$ ，静置分层约 $1\mathrm{min}$ ，再将下层水相放入同一容量瓶中，再加入 $10~\mathrm{mL}$ 水重复1次操作，最后用水定容至刻度，摇匀后过 $0.45\mu \mathrm{m}$ 水系滤膜后用于色谱分析。

脂肪类乳化制品、可可制品、巧克力及巧克力制品、坚果与籽类、水产及其制品、蛋制品：用食品加工机按试样与水的质量比为 $1:4$ 进行匀浆，称取约 $5\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )匀浆试样于 $25~\mathrm{mL}$ 离心管中，加入 $10~\mathrm{mL}$ 水超声振荡提取 $20\mathrm{min}$ ，静置 $1\mathrm{min},4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液转入 $100~\mathrm{mL}$ 的分液漏斗中，离心管中再加入 $8\mathrm{mL}$ 水超声振荡提取 $10\mathrm{min}$ ，静置和离心后将上清液再次转入分液漏斗中，向分液漏斗加入 $15~\mathrm{mL}$ 正己烷，振摇 $30~\mathrm{s}$ ，静置分层约 $5\mathrm{min}$ ，将下层水相放入 $25~\mathrm{mL}$ 容量瓶，用水定容至刻度，摇匀后过 $0.45\mu \mathrm{m}$ 水系滤膜后用于色谱分析。

# 5.2 仪器参考条件

a）色谱柱： $\mathrm{C_{18}}$ ，柱长 $250~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ □  
b） 柱温： $30^{\circ}C$ 。  
c) 流动相：甲醇-水 $(40 + 60)$ 或乙腈-水 $(20 + 80)$ 。  
d) 流速: $0.8 \mathrm{~mL} / \mathrm{min}$ 。  
e) 进样量: $20 \mu \mathrm{L}$ 。  
f）检测器：二极管阵列检测器或紫外检测器。  
g）检测波长： $200\mathrm{nm}$

# 5.3 标准曲线的制作

将标准系列工作液分别在上述色谱条件下测定相应的峰面积(峰高)，以标准工作液的浓度为横坐标，以峰面积(峰高)为纵坐标，绘制标准曲线。标准色谱图见附录A。

# 5.4 试样溶液的测定

在相同的液相色谱条件下，将试样溶液注入液相色谱仪中，以保留时间定性，以试样峰高或峰面积与标准比较定量。

# 6 分析结果的表述

试样中阿斯巴甜或阿力甜的含量按式(1)计算：

$$
X = \frac {\rho \times V}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中阿斯巴甜或阿力甜的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho$ ——由标准曲线计算出进样液中阿斯巴甜或阿力甜的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样的最后定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

1000——由 $\mu \mathrm{g} / \mathrm{g}$ 换算成 $\mathrm{g / kg}$ 的换算因子。

结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

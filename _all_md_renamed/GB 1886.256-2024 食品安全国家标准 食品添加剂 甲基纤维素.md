# 前言

本标准代替GB1886.256—2016《食品安全国家标准 食品添加剂 甲基纤维素》。

本标准与GB1886.256—2016相比，主要变化如下：

——修改了相对分子质量；

——修改了鉴别试验中的试剂表述；

——修改了黏度的检验方法。

# 食品安全国家标准

# 食品添加剂 甲基纤维素

# 1 范围

本标准适用于以木浆或棉浆等天然纤维素为原料，经碱化和醚化处理后，再经洗涤、干燥后制得的食品添加剂甲基纤维素。

# 2 化学名称、结构式、分子式和相对分子质量

# 2.1 化学名称

纤维素甲基醚

# 2.2 结构式

![](images/ca86832eb87e609aef59de0f86458066901cef6c51a9610cea432b68c32c50d6.jpg)  
$\mathrm{R} = \mathrm{H}$ 或 $\mathrm{CH}_3$

# 2.3 分子式

$$
\left[ \mathrm {C} _ {6} \mathrm {H} _ {7} \mathrm {O} _ {2} (\mathrm {O H}) _ {x} (\mathrm {O C H} _ {3}) _ {y} \right] _ {n}
$$

其中 $x = 1.00\sim 1.55,y = 2.00\sim 1.45,x + y = 3.00(y =$ 取代度）

未取代的结构单元：162.14（按2018年国际相对原子质量）

取代度为2.0的结构单元：190.20（按2018年国际相对原子质量）

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 氢氧化钠。  
A.2.1.2 盐酸。  
A.2.1.3 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $4\mathrm{g}$ 氢氧化钠溶解于 $100~\mathrm{mL}$ 水中。  
A.2.1.4 盐酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：吸取 $8.33~\mathrm{mL}$ 浓盐酸，溶于水并稀释至 $100~\mathrm{mL}$ 。

# A.2.2 试样液制备

在盛有 $100~\mathrm{mL}$ 水的烧杯(液面)中的液面上散布试样 $1\mathrm{g}$ ，静置，当试样绝大部分变成透明的黏性溶液后（约 $5\mathrm{h}$ ），振荡烧杯并用玻璃棒搅拌至试样完全溶解。

# A.2.3 鉴别方法

A.2.3.1 取 $10 \mathrm{~mL}$ 试样液, 加入同体积的氢氧化钠溶液或盐酸溶液时, 此时, 混合液应保持透明、有黏性。  
A.2.3.2 取 $10~\mathrm{mL}$ 试样液，加热，溶液产生雾状和片状沉淀，冷却后又溶解。  
A.2.3.3 取 $10\mathrm{mL}$ 试样液，倾倒在玻璃板上，蒸发除去水分，形成白色薄膜。

# A.3 甲氧基（一 $\mathbf{OCH}_3$ ）含量（以干基计)的测定

# A.3.1 方法原理

利用试样中甲氧基 $\left(-\mathrm{OCH}_{3}\right)$ 与氢碘酸作用生成易挥发的甲基碘，以甲苯为内标，采用气相色谱法测定甲基碘，根据测定生成的甲基碘的量来计算试样中甲氧基（一 $\mathrm{OCH}_3$ )的含量。

警示——在通风橱中进行含有氢碘酸的操作步骤。使用护目镜、耐酸手套和其他合适的安全设备。在处理热玻璃瓶时，由于压力，故应极其小心。如果不慎接触氢碘酸，应用大量清水冲洗，并立即寻求医疗帮助。

# A.3.2 试剂和材料

A.3.2.1 甲苯。  
A.3.2.2 邻二甲苯。  
A.3.2.3 己二酸。  
A.3.2.4 氢碘酸。  
A.3.2.5 甲基碘。  
A.3.2.6 内标溶液：称取 $2.5\mathrm{g}$ 甲苯，精确至 $0.001\mathrm{g}$ ，用邻二甲苯稀释至 $100\mathrm{mL}$ 。

A.3.2.7 标准溶液：称取 $135\mathrm{mg}$ 己二酸，精确至 $0.001\mathrm{g}$ ，置于合适的血清瓶中，加入 $4.0\mathrm{mL}$ 氢碘酸，再加入 $4.0\mathrm{mL}$ 内标溶液，用隔膜盖封住瓶口，称量。通过隔膜，注入 $90~\mu \mathrm{L}$ 甲基碘，再称量，计算加入的甲基碘的质量。混合均匀，静置后分层。

# A.3.3 仪器和设备

气相色谱仪：配备热导检测器，或其他等效的检测器。

# A.3.4 参考色谱条件

A.3.4.1 色谱柱： $1.8\mathrm{m}\times 4\mathrm{mm}$ 毛细管柱，填料为 $125~\mu \mathrm{m}\sim 150~\mu \mathrm{m}$ 含 $10\%$ 甲基硅酮油的色谱纯硅藻土或其他等效色谱柱。

A.3.4.2 柱温： $100^{\circ}\mathrm{C}$ 。  
A.3.4.3 进样口温度： $200^{\circ}C$ 。  
A.3.4.4 检测器温度： $200^{\circ}C$   
A.3.4.5 载气：氦气。  
A.3.4.6 流速： $20~\mathrm{mL / min}$   
A.3.4.7 进样量： $2\mu \mathrm{L}$

# A.3.5 分析步骤

# A.3.5.1 试样溶液的制备

称取 $0.065\mathrm{g}$ 试样，精确至 $0.001\mathrm{g}$ ，置于 $5\mathrm{mL}$ 带压紧隔膜的瓶中，加入与试样等量的己二酸，然后加入 $2\mathrm{mL}$ 内标溶液。小心吸取 $2\mathrm{mL}$ 氢碘酸，加入瓶中，盖紧瓶盖，称量后摇动 $30\mathrm{s}$ ，在 $150^{\circ}\mathrm{C}$ 条件下加热 $20\mathrm{min}$ ，摇动后继续在 $150^{\circ}\mathrm{C}$ 条件下加热 $40\mathrm{min}$ ，冷却 $45\mathrm{min}$ 后称量，若质量损失超过 $10\mathrm{mg}$ ，则弃掉，重新制备试样溶液。

# A.3.5.2 测定

分别吸取标准溶液和试样溶液上层液，按A.3.3参考色谱条件进行色谱分析，记录色谱图(见附录B)，在计算得到等质量甲苯和甲基碘的相对响应因子 $F$ 后，再计算出样品的甲氧基（一 $\mathrm{OCH}_3$ ）含量（以干基计）。

# A.3.5.3 结果计算

A.3.5.3.1 等质量甲苯和甲基碘的相对响应因子 $F$ 按式(A.1)计算：

$$
F = \frac {M _ {1} \times A _ {2}}{M _ {2} \times A _ {1}} \quad \dots \tag {A.1}
$$

式中：

$M_{1}$ ——标准溶液中甲基碘的质量，单位为克 $(\mathrm{g})$ ；

$A_{2}$ ——色谱图中甲苯的峰面积；

$M_{2}$ ——标准溶液中甲苯的质量，单位为克 $(\mathrm{g})$

$A_{1}$ ——色谱图中甲基碘的峰面积。

A.3.5.3.2 甲氧基 $(\mathrm{-OCH_3})$ （以干基计)的含量 $\omega$ 按式(A.2)计算：

$$
w = 2 \times \frac {31}{142} \times F \times \alpha \times \frac {m}{m _ {1} (1 - w)} \times 100 \% \quad \dots \tag{A.2}
$$

式中：

2 ——含量换算系数；

31/142——甲氧基和甲基碘的相对分子质量之比；

$F$ ——实际测得的甲苯和甲基碘的相对响应因子；

$\alpha$ ——试样溶液色谱图中甲基碘和甲苯的峰面积之比；

$m$ ——内标溶液中甲苯的质量，单位为克(g)；

$m_{1}$ ——试样的质量，单位为克(g)；

$\varpi$ ——试样的干燥减量， $\%$ 。

试验结果以平行测定结果的算术平均值为准，保留2位有效数字。在重复性测定条件下获得的两次独立测定结果的绝对差值不超过其算术平均值的 $10\%$ 。

# A.4 黏度的测定

称取适量试样，加入事先用加热至 $80^{\circ}\mathrm{C} \sim 90^{\circ}\mathrm{C}$ 的水制成 $2\mathrm{g} / 100\mathrm{mL}$ 的试样溶液，约 $400\mathrm{r / min}$ 条件下搅拌 $10\mathrm{min} \sim 20\mathrm{min}$ ，置于冰浴中继续搅拌 $40\mathrm{min}$ 后，将试样溶液的温度调整至 $20^{\circ}\mathrm{C} \pm 0.1^{\circ}\mathrm{C}$ ，再选择合适的旋转黏度计(Brookfield旋转黏度计或其他同等性能黏度计)按表A.1进行测定。

若试样在上述条件下不溶，则改为：称取适量试样，加入事先用加热至 $80^{\circ}\mathrm{C} \sim 90^{\circ}\mathrm{C}$ 的水制成 $1\mathrm{g}/$ $100~\mathrm{mL}$ 的试样溶液，在约 $400\mathrm{r / min}$ 条件下搅拌 $10\mathrm{min}\sim 20\mathrm{min}$ ，置于冰浴中冷却，冷却时继续搅拌 $40\mathrm{min}$ ，将试样溶液的温度调整至 $5^{\circ}\mathrm{C}\pm 0.1^{\circ}\mathrm{C}$ ，再选择合适的旋转黏度计(Brookfield旋转黏度计或其他同等性能黏度计)按表A.1进行测定。

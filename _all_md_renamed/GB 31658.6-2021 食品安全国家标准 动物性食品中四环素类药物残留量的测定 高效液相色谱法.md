# 高效液相色谱法

# 1 范围

本文件规定了动物性食品中四环素类药物残留量检测的制样和高效液相色谱测定方法。

本文件适用于猪、牛、羊、鸡的肌肉、肝脏和肾脏，猪、鸡的皮+脂肪，鸡蛋，牛奶，鱼皮+肉，虾肌肉中土霉素、四环素、金霉素、多西环素残留量的检测。

# 2 规范性引用文件

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中残留的四环素类药物，经EDTA·2NaMcllvaine缓冲溶液提取，固相萃取柱净化，高效液相色谱紫外法测定，外标法定量。

# 5 试剂与材料

以下所用的试剂，除特别注明外均为分析纯试剂；水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯  
5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.3 三氟乙酸 $(\mathrm{CF_3CO})(\mathrm{OH})$   
5.1.4 二氯甲烷 $\mathrm{(CH_2Cl_2)}$   
5.1.5 乙二胺四乙酸二钠 $(\mathrm{C_{10}H_{11}N_2Na_2O_8})$   
5.1.6 枸橼酸 $(\mathrm{C_6H_8O_7}\cdot \mathrm{H_2O})$ 。  
5.1.7 磷酸氢二钠 $(\mathrm{NaH_2PO_4}\cdot 12\mathrm{H_2O})$   
5.1.8 草酸 $(\mathrm{H}_2\mathrm{C}_2\mathrm{O}_4\cdot 2\mathrm{H}_2\mathrm{O})$   
5.1.9 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$   
5.1.10 钨酸钠 $(\mathrm{Na_2WO_4})$ 。

# 5.2 标准品

盐酸土霉素含量 $\geqslant 97.0\%$ ，盐酸四环素含量 $\geqslant 97.5\%$ ，盐酸金霉素含量 $\geqslant 93.1\%$ ，盐酸多西环素含量 $\geqslant 98.2\%$ ，见附录A。

# 5.3 溶液配制

5.3.1 柠檬酸溶液：取柠檬酸 $21.01\mathrm{g}$ ，用水溶解并稀释至 $1000\mathrm{mL}$ 。  
5.3.2 磷酸氢二钠溶液：取磷酸氢二钠 $71.63\mathrm{g}$ ，用水溶解并稀释至 $1000\mathrm{mL}$ 。

5.3.3 Mcllvain缓冲溶液 $(\mathrm{pH}4.0)$ ：取构橡酸溶液 $1000~\mathrm{mL}$ 磷酸氢二钠溶液 $625~\mathrm{mL}$ ，混匀，用盐酸或氢氧化钠溶液调 $\mathrm{pH}$ 至 $4.0\pm 0.05$ 。  
5.3.4 EDTA·2Na-McIvaine缓冲溶液：取乙二胺四乙酸二钠 $60.5\mathrm{g}$ ，加McIvaine缓冲溶液 $1625~\mathrm{mL}$ ，溶解，混匀。  
5.3.5 草酸溶液 $(0.01\mathrm{mol} / \mathrm{L})$ ：取草酸 $1.26\mathrm{g}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。  
5.3.6 三氟乙酸溶液：取三氟乙酸 $0.8\mathrm{mL}$ ，用水溶解并稀释至 $1000~\mathrm{mL}$ 。  
5.3.7 硫酸溶液：取硫酸 $1.85\mathrm{mL}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
5.3.8 钨酸钠溶液：取钨酸钠 $7\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$   
5.3.9 草酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：取草酸 $12.6\mathrm{g}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
5.3.10 草酸-乙腈溶液：取草酸溶液 $(1\mathrm{mol} / \mathrm{L})20\mathrm{mL}$ ，用乙腈溶解并稀释至 $100~\mathrm{mL}$ 。

# 5.4 标准溶液制备

5.4.1 标准储备液：取盐酸土霉素、盐酸四环素、盐酸金霉素和盐酸多西环素标准品各约 $10\mathrm{mg}$ ，精密称定，用甲醇溶解并稀释定容至 $10\mathrm{mL}$ 容量瓶，配制成浓度为 $1\mathrm{mg/mL}$ 的土霉素、四环素、金霉素和多西环素标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期1个月。  
5.4.2 混合标准工作液：精密量取土霉素、四环素、金霉素和多西环素标准储备液各 $1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，配制成浓度为 $10~\mu \mathrm{g / mL}$ 的混合标准工作液。 $2^{\circ}C\sim 8^{\circ}C$ 保存。现用现配。

# 5.5 材料

5.5.1HLB固相萃取柱： $500\mathrm{mg / 6mL}$ ，亲脂性二乙烯苯和亲水性N-乙烯吡咯烷酮按特定比例形成的聚合物，或相当者。  
5.5.2 LCX固相萃取柱1)： $500\mathrm{mg / 6mL}$ ，填料为磷酸化聚苯乙烯二乙烯苯高聚物，或相当者。

# 6 仪器和设备

6.1 高效液相色谱仪：配紫外检测器。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 组织匀浆器。  
6.4 涡旋混合器： $3000\mathrm{r / min}$   
6.5 低温离心机：转速可达 $8500\mathrm{r / min}$   
6.6 固相萃取装置。  
6.7 氮吹仪。  
6.8 尼龙微孔滤膜： $0.22\mu \mathrm{m}$   
6.9 离心管： $50~\mathrm{mL}$

# 7 试料的制备与保存

# 7.1 试料的制备

# 7.1.1 组织

取适量新鲜或解冻的空白或供试组织绞碎，并使均质。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

# 7.1.2 牛奶

取适量新鲜或冷藏的空白或供试牛奶，混合均匀。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

# 7.1.3 鸡蛋

取适量新鲜的供试鸡蛋，去壳，并使均质。

a）取均质的供试样品，作为供试试料；  
b）取均质的空白样品，作为空白试料；  
c）取均质的空白样品，添加适宜浓度的标准溶液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存，3个月内进行分析检测。

# 8 测定步骤

# 8.1 提取

# 8.1.1 皮+脂肪

取试料 $5\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，加二氯甲烷 $15~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，振荡 $5\mathrm{min}$ ，加EDTA·2Na-McIvaine缓冲溶液 $15~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，振荡 $5\mathrm{min}$ ， $8500~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液。下层溶液用EDTA·2Na-McIvaine缓冲溶液重复萃取2次，每次 $15~\mathrm{mL}$ ，合并上清液，中性滤纸过滤，备用。

# 8.1.2 肌肉、肝脏、肾脏、牛奶、鸡蛋

称取试料 $5\mathrm{g}$ （准确至 $\pm 0.02\mathrm{g}$ ，加EDTA·2Na-Mcllvaine缓冲溶液 $20~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，振荡 $10\mathrm{min}$ ，加硫酸溶液 $5\mathrm{mL}$ 、钨酸钠溶液 $5\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，8500r/min离心 $5\mathrm{min}$ ，取上清液。残渣用EDTA·2Na-Mcllvaine缓冲溶液 $20~\mathrm{mL},10~\mathrm{mL}$ 各提取2次，合并上清液，中性滤纸过滤，备用。

# 8.2 净化

HLB柱依次用甲醇、水和EDTA·2Na-Mellvaine缓冲溶液各 $5\mathrm{mL}$ 活化。备用液过柱，待全部备用液流出后，依次用水、 $5 \%$ 甲醇溶液各 $10~\mathrm{mL}$ 淋洗，抽干 $30~\mathrm{s}$ ，用甲醇 $6\mathrm{mL}$ 洗脱，收集洗脱液于刻度试管中，加水 $2\mathrm{mL}$ ，混匀，过甲醇 $5\mathrm{mL}$ 、水 $5\mathrm{mL}$ 活化的LCX柱，待全部液体流出后，用水、甲醇各 $5\mathrm{mL}$ 淋洗，抽干 $1\mathrm{min}$ ，草酸-乙腈溶液 $6\mathrm{mL}$ 洗脱，收集洗脱液，于 $40^{\circ}C$ 水浴氮气吹至 $0.5\mathrm{mL}\sim 1.0\mathrm{mL}$ ，再加甲醇 $0.4\mathrm{mL}$ ，用草酸溶液 $(0.01\mathrm{mol / L})$ 稀释至 $2.0\mathrm{mL}$ ，微孔滤膜过滤，高效液相色谱测定（上机溶液应在 $24\mathrm{h}$ 内完成测定）。

# 8.3 标准曲线的制备

精密量取混合标准工作液适量，用草酸溶液 $(0.01\mathrm{mol} / \mathrm{L})$ 稀释成浓度为 $0.05~\mu \mathrm{g / mL}$ 、 $0.1~\mu \mathrm{g / mL}$ 产 $0.2\mu \mathrm{g / mL},0.5\mu \mathrm{g / mL},1\mu \mathrm{g / mL},2\mu \mathrm{g / mL},5\mu \mathrm{g / mL}$ 的系列混合标准液，供高效液相色谱测定。以测得的峰面积为纵坐标、对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.4.2 测定法

取试料溶液和相应的标准溶液，作单点或多点校准，以色谱峰面积定量，按外标法计算。标准溶液及试料溶液中四环素类药物响应值应在仪器检测的线性范围之内。试料中四环素类药物的保留时间与标准工作液相应峰的保留时间相对偏差应在 $\pm 2.5\%$ 以内。在上述色谱条件下，标准溶液的高效液相色谱图分别见附录B。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试料中四环素类药物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times C _ {\mathrm {s}} \times V}{A _ {\mathrm {s}} \times m} \dots \tag {1}
$$

式中：

$X$ ——试料中相应的四环素类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

A ——试料中相应的四环素类药物的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中相应的四环素类药物的峰面积；

$C_{S}$ ——标准溶液中相应的四环素类药物浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

V 最终试料定容体积的数值，单位为毫升 $(\mathrm{mL})$

$m$ 供试试料质量的数值，单位为克（g）。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法在猪、牛、羊、鸡的肌肉，鸡蛋，牛奶，鱼皮十肉、虾肌肉中检测限为 $20~\mu \mathrm{g / kg}$ ，定量限为 $50~\mu \mathrm{g / kg}$ ；在猪、牛、羊、鸡的肝脏、肾脏，猪、鸡的皮+脂肪的检测限为 $50~\mu \mathrm{g / kg}$ ，定量限为 $100~\mu \mathrm{g / kg}$ 。

# 10.2 准确度

本方法在猪、牛、羊、鸡肌肉 $50\mu \mathrm{g / kg}\sim 200\mu \mathrm{g / kg}$ 添加浓度的回收率为 $60\% \sim 120\%$ ，在猪、牛、羊、鸡肝脏 $100~\mu \mathrm{g / kg}\sim 600~\mu \mathrm{g / kg}$ 添加浓度的回收率为 $60\% \sim 120\%$ ，在猪、牛、羊、鸡肾脏组织 $100~\mu \mathrm{g / kg}\sim$ $1200~\mu \mathrm{g / kg}$ 添加浓度的回收率为 $60\% \sim 120\%$ ，猪、鸡皮+脂肪 $100~\mu \mathrm{g / kg}\sim 600~\mu \mathrm{g / kg}$ 添加浓度的回收率为 $60\% \sim 120\%$ ，鱼皮十肉、虾肌肉 $50~\mu \mathrm{g / kg}\sim 200~\mu \mathrm{g / kg}$ 添加浓度的回收率为 $60\% \sim 120\%$ ，在牛奶 $50~\mu \mathrm{g / kg}\sim 200~\mu \mathrm{g / kg}$ 添加浓度的回收率为 $60\% \sim 120\%$ ，在鸡蛋 $50~\mu \mathrm{g / kg}\sim 400~\mu \mathrm{g / kg}$ 添加浓度的回收率为 $60\% \sim 120\%$ 。

# 10.3 精密度

本方法的批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 15\%$ 。

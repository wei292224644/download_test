# 二苯甲酮类物质迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中8种二苯甲酮类物质迁移量的测定方法。

第一法液相色谱法适用于塑料、涂料和涂层、橡胶、使用油墨的食品接触材料及制品中(2-羟基-4-甲氧基苯基)(2-羟苯基)甲酮、2,4-二羟基二苯甲酮、(2-羟基-4-甲氧基苯基)苯基甲酮、4,4'-二羟基二苯甲酮、2-羟基-4-正辛氧基二苯甲酮、2-羟基-4-正己氧基二苯甲酮、二苯甲酮和4,4'-二氟二苯甲酮在水、 $4\%$ (体积分数)乙酸、 $10\%$ (体积分数)乙醇、 $20\%$ (体积分数)乙醇、 $50\%$ (体积分数)乙醇食品模拟物，以及化学替代溶剂 $95\%$ (体积分数)乙醇和异辛烷中迁移量的测定。

第二法液相色谱-串联质谱法适用于塑料、涂料和涂层、橡胶、使用油墨的食品接触材料及制品中(2-羟基-4-甲氧基苯基)(2-羟苯基)甲酮、2,4-二羟基二苯甲酮、(2-羟基-4-甲氧基苯基)苯基甲酮、 $4,4^{\prime}$ -二羟基二苯甲酮、2-羟基-4-正辛氧基二苯甲酮、2-羟基-4-正己氧基二苯甲酮、二苯甲酮和 $4,4^{\prime}$ -二氟二苯甲酮在水、 $4\%$ (体积分数)乙酸、 $10\%$ (体积分数)乙醇、 $20\%$ (体积分数)乙醇、 $50\%$ (体积分数)乙醇、橄榄油食品模拟物，以及化学替代溶剂 $95\%$ (体积分数)乙醇和异辛烷中迁移量的测定。

# 第一法 液相色谱法

# 2 原理

食品接触材料及制品根据GB31604.1及GB5009.156进行迁移试验后，采用液相色谱法进行检测。水、 $4\%$ （体积分数)乙酸、 $10\%$ （体积分数)乙醇、 $20\%$ （体积分数)乙醇、 $50\%$ （体积分数)乙醇食品模拟物直接进样；化学替代溶剂 $95\%$ （体积分数)乙醇经水稀释后过滤进样；化学替代溶剂异辛烷氮吹后用甲醇-水溶液复溶，过滤后进样。峰面积外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.2 无水乙醇 $\mathrm{(C_2H_5OH)}$ 。  
3.1.3 异辛烷 $\mathrm{(C_8H_{18})}$   
3.1.4 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.5 二甲基亚砜 $(\mathrm{C}_2\mathrm{H}_6\mathrm{OS})$ 。

# 3.2 试剂配制

3.2.1 $4\%$ （体积分数)乙酸溶液、 $10\%$ （体积分数)乙醇溶液、 $20\%$ （体积分数)乙醇溶液、 $50\%$ （体积分

数)乙醇溶液的配制按照GB5009.156操作。

3.2.2 $95\%$ （体积分数)乙醇溶液：将无水乙醇和水按 $95:5$ 的体积比混合均匀。  
3.2.3 甲醇-水溶液 $(7 + 3)$ ：将甲醇和水按 $7:3$ 的体积比混合均匀。

# 3.3 标准品

3.3.1（2-羟基-4-甲氧基苯基)(2-羟苯基)甲酮 $\mathrm{(C_{14}H_{12}O_4}$ ,CAS号：131-53-3)：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.2 2,4-二羟基二苯甲酮 $\mathrm{(C_{13}H_{10}O_3}$ ,CAS号：131-56-6)：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.3（2-羟基-4-甲氧基苯基)苯基甲酮 $\mathrm{(C_{14}H_{12}O_3}$ ，CAS号：131-57-7）：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.4 $4,4^{\prime}$ 二羟基二苯甲酮 $\mathrm{(C_{13}H_{10}O_3}$ ，CAS号：611-99-4）：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。  
3.3.5 2-羟基-4-正辛氧基二苯甲酮 $\mathrm{(C_{21}H_{26}O_3}$ ，CAS号：1843-05-6）：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

# 3.4.1 标准储备液（ $1000\mathrm{mg / L}$ ）

准确称取8种二苯甲酮类物质标准品各 $50~\mathrm{mg}$ （精确至 $0.1\mathrm{mg})$ ，分别用甲醇溶解后转移至8个 $50~\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度，混匀。将溶液转移至棕色玻璃容器中，于 $4^{\circ}C$ 下避光密闭保存，保存期为6个月。

# 3.4.2 混合标准中间液

分别吸取 $0.5\mathrm{mL}4,4^{\prime}$ 二羟基二苯甲酮、二苯甲酮和 $4,4^{\prime}$ 二氟二苯甲酮的标准储备液 $(1000\mathrm{mg / L})$ 和 $2.0\mathrm{mL}$ 其他5种二苯甲酮类物质的标准储备液 $(1000\mathrm{mg / L})$ 于 $100~\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度，混匀，得到混合标准中间液，其中 $4,4^{\prime}$ 二羟基二苯甲酮、二苯甲酮和 $4,4^{\prime}$ 二氟二苯甲酮的浓度均为 $5.0\mathrm{mg / L}$ ，其他5种二苯甲酮类物质的浓度均为 $20.0\mathrm{mg / L}$ 。将溶液转移至棕色玻璃容器中，于 $4^{\circ}\mathrm{C}$ 下避光密闭保存，保存期为3个月。

# 3.4.3 混合标准系列工作液

# 3.4.3.1 水、 $4\%$ （体积分数）乙酸、 $10\%$ （体积分数）乙醇、 $20\%$ （体积分数）乙醇、 $50\%$ （体积分数）乙醇混合标准系列工作液

分别吸取 $40~\mu \mathrm{L},100~\mu \mathrm{L},200~\mu \mathrm{L},400~\mu \mathrm{L},600~\mu \mathrm{L}$ 混合标准中间液于5个 $10~\mathrm{mL}$ 容量瓶中，用相应的食品模拟物定容至刻度，混匀，得到混合标准系列工作液，其中 $4,4^{\prime}$ -二羟基二苯甲酮、二苯甲酮和 $4,4^{\prime}$ 二氟二苯甲酮的浓度均为 $0.020\mathrm{mg / L},0.050\mathrm{mg / L},0.10\mathrm{mg / L},0.20\mathrm{mg / L},0.30\mathrm{mg / L}$ ，其他5种二苯甲酮类物质的浓度均为 $0.080\mathrm{mg / L},0.20\mathrm{mg / L},0.40\mathrm{mg / L},0.80\mathrm{mg / L},1.2\mathrm{mg / L}$ 。临用现配。

# 3.4.3.2 化学替代溶剂 $95\%$ （体积分数）乙醇混合标准系列工作液

分别吸取 $50~\mu \mathrm{L},100~\mu \mathrm{L},200~\mu \mathrm{L},400~\mu \mathrm{L},600~\mu \mathrm{L}$ 混合标准中间液于5个 $10~\mathrm{mL}$ 容量瓶中，用 $95\%$ （体积分数）乙醇定容至刻度，混匀，得到混合标准系列工作液，其中 $4,4^{\prime}$ -二羟基二苯甲酮、二苯甲酮和 $4,4^{\prime}$ -二氟二苯甲酮的浓度均为 $0.025\mathrm{mg / L},0.050\mathrm{mg / L},0.10\mathrm{mg / L},0.20\mathrm{mg / L},0.30\mathrm{mg / }$ L，其他5种二苯甲酮类物质的浓度均为 $0.10\mathrm{mg / L},0.20\mathrm{mg / L},0.40\mathrm{mg / L},0.80\mathrm{mg / L},1.2\mathrm{mg / L}_{\circ}$ 分别吸取 $7\mathrm{mL}95\%$ （体积分数)乙醇混合标准系列工作液于5个 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。临用现配。

# 3.4.3.3 化学替代溶剂异辛烷混合标准系列工作液

分别吸取 $50~\mu \mathrm{L},100~\mu \mathrm{L},200~\mu \mathrm{L},400~\mu \mathrm{L},600~\mu \mathrm{L}$ 混合标准中间液于5个 $10~\mathrm{mL}$ 容量瓶中，用异辛烷定容至刻度，混匀，得到混合标准系列工作液，其中 $4,4^{\prime}$ -二羟基二苯甲酮、二苯甲酮和 $4,4^{\prime}$ -二氟二苯甲酮的浓度均为 $0.025\mathrm{mg / L},0.050\mathrm{mg / L},0.10\mathrm{mg / L},0.20\mathrm{mg / L},0.30\mathrm{mg / L}$ ，其他5种二苯甲酮类物质的浓度均为 $0.10\mathrm{mg / L},0.20\mathrm{mg / L},0.40\mathrm{mg / L},0.80\mathrm{mg / L},1.2\mathrm{mg / L}$ 。分别吸取 $2\mathrm{mL}$ 异辛烷混合标准系列工作液于5个 $10~\mathrm{mL}$ 具塞离心管中，分别准确加入 $100~\mu \mathrm{L}$ 二甲基亚砜，混匀，在 $40^{\circ}C$ 水浴条件下氮吹至近干，分别准确加入 $2\mathrm{mL}$ 甲醇-水溶液 $(7 + 3)$ ，涡旋振荡 $30~s$ 溶解残渣，经微孔滤膜过滤，待测定。临用现配。

# 3.5 材料

针式过滤器：孔径 $0.22\mu \mathrm{m}$ ，微孔滤膜材质为聚四氟乙烯(PTFE)或尼龙(Nylon)。

# 4 仪器和设备

4.1 高效液相色谱仪：带紫外检测器或二级管阵列检测器。  
4.2 天平：感量为 $0.1\mathrm{mg}$ 。  
4.3 涡旋混合器。  
4.4 离心机：转速 $\geqslant 5000\mathrm{r / min}$   
4.5 氮气浓缩装置。

# 5 分析步骤

# 5.1 试液制备

# 5.1.1 迁移试验

食品接触材料及制品按照GB31604.1和GB5009.156的要求进行迁移试验，浸泡液恢复至室温后按照5.1.2处理。迁移试验所得浸泡液如不能立即测试，应置于 $4^{\circ}\mathrm{C}$ 冰箱中避光保存不超过 $72\mathrm{h}$ ，恢复至室温后按照5.1.2处理。

# 5.1.2 浸泡液的处理

# 5.1.2.1 水、 $4\%$ （体积分数）乙酸、 $10\%$ （体积分数）乙醇、 $20\%$ （体积分数）乙醇、 $50\%$ （体积分数）乙醇食品模拟物试液

移取适量迁移试验后得到的水、 $4\%$ （体积分数)乙酸、 $10\%$ （体积分数)乙醇、 $20\%$ （体积分数)乙醇、 $50\%$ （体积分数)乙醇浸泡液，经 $5000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液待测定。

# 5.1.2.2 化学替代溶剂 $95\%$ （体积分数）乙醇试液

吸取 $7\mathrm{mL}$ 迁移试验后得到的 $95\%$ （体积分数）乙醇浸泡液于 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀，取约 $1\mathrm{mL}$ 经微孔滤膜过滤，待测定。

# 5.1.2.3 化学替代溶剂异辛烷试液

吸取 $2\mathrm{mL}$ 迁移试验后得到的异辛烷浸泡液于 $10~\mathrm{mL}$ 具塞离心管中，准确加入 $100\mu \mathrm{L}$ 二甲基亚砜，混匀，在 $40^{\circ}\mathrm{C}$ 水浴条件下氮吹至近干，准确加入 $2\mathrm{mL}$ 甲醇-水溶液 $(7 + 3)$ ，涡旋振荡 $30~\mathrm{s}$ 溶解残渣，取约 $1\mathrm{mL}$ 经微孔滤膜过滤，待测定。

# 5.1.3 空白试液制备

# 6 分析结果的表述

# 6.1 非密封制品类食品接触材料及制品二苯甲酮类物质特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

对于除了盖子、密封圈、连接件等密封制品（以下简称密封制品）以外的食品接触材料及制品试样，二苯甲酮类物质特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(1)进行计算。

$$
X _ {1} = \frac {\left(c - c _ {0}\right) \times V}{S} \times \frac {S _ {0}}{V _ {1}} \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——二苯甲酮类物质的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——浸泡液中二苯甲酮类物质的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白浸泡液中二苯甲酮类物质的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$V$ ——迁移试验中试样浸泡液的体积或质量，单位为升(L)或千克(kg)；

$S$ ——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$S_{0}$ ——非密封制品实际使用中与食品接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$V_{1}$ ——非密封制品实际接触固态食品的质量,或实际接触液态食品的体积所对应的食品质量,单位为千克 $(\mathrm{kg})$ ; 各种液态食品按密度为 $1 \mathrm{~kg} / \mathrm{L}$ 将其体积换算为相应的质量。

如试样的实际接触面积/体积(即 $S_0 / V_1$ )未知，则按照每 $6\mathrm{dm}^2$ 的试样面积与 $1\mathrm{kg}$ 的食品接触进行计算。

以重复性条件下获得的两次独立测定结果的算术平均值表示，计算结果至少保留2位有效数字。

# 6.2 密封制品类食品接触材料及制品二苯甲酮类物质特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

当预期用途已知时，密封制品类食品接触材料及制品试样中二苯甲酮类物质特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(2)计算。

$$
X _ {2} = \frac {\left(c - c _ {0}\right) \times V}{S} \times \frac {S _ {0}}{V _ {2}} \tag {2}
$$

式中：

$X_{2}$ ——二苯甲酮类物质的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——浸泡液中二苯甲酮类物质的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白浸泡液中二苯甲酮类物质的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$V$ ——迁移试验中试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

$S$ ——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$S_{0}$ ——密封制品实际使用中与食品接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$V_{2}$ ——密封制品实际使用容器盛装固态食品的质量，或实际接触液态食品的体积所对应的食品质量，单位为千克 $(\mathrm{kg})$ ；各种液态食品按密度 $1\mathrm{kg} / \mathrm{L}$ 将其体积换算为相应的质量。

以重复性条件下获得的两次独立测定结果的算术平均值表示，计算结果至少保留2位有效数字。

# 6.3 密封制品类食品接触材料及制品二苯甲酮类物质特定迁移量的计算（以 $\mathrm{mg}$ /件表示）

当预期用途未知时，密封制品类食品接触材料及制品试样中二苯甲酮类物质特定迁移量以 $\mathrm{mg}/$ 件表示时，按式(3)计算，需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {3} = \frac {\left(c - c _ {0}\right) \times V}{n} \tag {3}
$$

式中：

$X_{3}$ ——二苯甲酮类物质的特定迁移量，单位为毫克每件 $(\mathrm{mg}/$ 件）；

$c$ ——浸泡液中二苯甲酮类物质的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白浸泡液中二苯甲酮类物质的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$

$V$ ——迁移试验中试样浸泡液的体积或质量，单位为升(L)或千克(kg)；

$n$ ——浸泡用密封制品的数量，单位为件。

以重复性条件下获得的两次独立测定结果的算术平均值表示，计算结果至少保留2位有效数字。

# 1 范围

本文件规定了动物性食品中庆大霉素残留检测的制样和液相色谱-串联质谱测定方法。

本文件适用于猪、牛、羊、鸡的肌肉、脂肪、肝脏和肾脏组织以及牛奶和鸡蛋中庆大霉素 $C_1$ 、庆大霉素 $C_{1a}$ 、庆大霉素 $C_2 + C_{2a}$ 残留量测定。

# 3 术语和定义

# 4 原理

试样中残留的庆大霉素用磷酸盐缓冲溶液提取，混合型阳离子交换固相萃取柱净化。液相色谱-串联质谱测定，基质匹配外标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $(\mathrm{CH}_3\mathrm{CN})_2$ 色谱纯

5.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。

5.1.3 甲酸（HCOOH）：色谱纯。

5.1.4 醋酸 $\mathrm{(CH_3COOH)}$ ：色谱纯。

5.1.5 乙酸铵 $\mathrm{(CH_3COONH_4)}$ ：色谱纯

5.1.6 七氟丁酸酐 $(\mathrm{C_8F_{14}O_3})$ ：色谱纯。

5.1.7 三氯乙酸 $(\mathrm{CCl}_3\mathrm{COOH})$

5.1.8 乙二胺四乙酸二钠 $(\mathrm{C_{10}H_{14}N_2Na_2O_8})$

5.1.9 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$

5.1.10 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。

# 5.2 溶液配制

5.2.1 $15\%$ 氨化甲醇溶液：量取氨水 $15~\mathrm{mL}$ ，用甲醇稀释至 $100~\mathrm{mL}$ ，现用现配。  
5.2.2 $0.3\%$ 醋酸溶液：量取醋酸 $0.3\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$   
5.2.3 磷酸盐缓冲液：称取磷酸二氢钾 $1.36\mathrm{g}$ ，用 $980~\mathrm{mL}$ 水溶解，分别加入乙二胺四乙酸二钠 $0.15\mathrm{g}$ 和三氯乙酸 $20\mathrm{g}$ ，溶解混匀并用水稀释至 $1000~\mathrm{mL},4^{\circ}\mathrm{C}$ 保存，有效期1个月。

5.2.4 $2\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液：称取乙酸铵15.416g，用水溶解并稀释至 $100~\mathrm{mI}_{\circ}$   
5.2.5 0.1%甲酸溶液(含 $2\mathrm{mmol / L}$ 乙酸铵)：取 $2\mathrm{mol / L}$ 乙酸铵溶液 $1\mathrm{mL}$ 、甲酸 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$ 。  
5.2.6 $0.1\%$ 甲酸乙腈溶液：取甲酸 $1\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$ 。  
5.2.7 $1\%$ 七氟丁酸酐醋酸溶液：量取 $0.5 \mathrm{~mL}$ 的七氟丁酸酐，用 $0.3\%$ 醋酸溶液稀释至 $50 \mathrm{~mL}$ ，现用现配。

# 5.3 标准品

庆大霉素(Gentamicin, $\mathrm{C}_{60} \mathrm{H}_{123} \mathrm{~N}_{15} \mathrm{O}_{21}$ , CAS号: 1403-66-3): 纯度 $\geqslant 94\%$ 。其中, $\mathrm{C}_{1}$ 组分(CAS号: 25876-10-2)含量为 $31.0\%$ , $\mathrm{C}_{1a}$ 组分(CAS号: 26098-04-4)含量为 $22.2\%$ , $\mathrm{C}_{2}$ 组分(CAS号: 287916-51-2)与 $\mathrm{C}_{2a}$ 组分(CAS号: 59751-72-3)含量为 $46.8\%$ 。各组分含量仅为参考, 也可以采用相应单组分标准品。

# 5.4 标准溶液制备

5.4.1 标准储备液：取庆大霉素标准品约 $10\mathrm{mg}$ ，精确至 $0.1\mathrm{mg}$ ，用乙腈溶解并稀释成浓度为 $1\mathrm{mg/mL}$ 的标准储备液，置于 $-18^{\circ}\mathrm{C}$ 避光保存，有效期6个月。  
5.4.2 标准中间液：准确量取庆大霉素标准储备液适量，用 $1\%$ 七氟丁酸酐醋酸溶液稀释定容，得到庆大霉素浓度为 $10~\mu \mathrm{g / mL}$ ，现用现配。  
5.4.3 系列标准工作液：准确量取庆大霉素标准中间液适量，用 $1\%$ 七氟丁酸酐醋酸溶液稀释定容，得到庆大霉素浓度为 $5\mu \mathrm{g} / \mathrm{L}, 10\mu \mathrm{g} / \mathrm{L}, 50\mu \mathrm{g} / \mathrm{L}, 100\mu \mathrm{g} / \mathrm{L}, 200\mu \mathrm{g} / \mathrm{L}, 500\mu \mathrm{g} / \mathrm{L}$ （其中，庆大霉素 $\mathbf{C}_1$ 组分为 $1.55\mu \mathrm{g} / \mathrm{L}, 3.1\mu \mathrm{g} / \mathrm{L}, 15.5\mu \mathrm{g} / \mathrm{L}, 31.0\mu \mathrm{g} / \mathrm{L}, 62.0\mu \mathrm{g} / \mathrm{L}, 155\mu \mathrm{g} / \mathrm{L};$ 庆大霉素 $\mathbf{C}_{1a}$ 组分为 $1.11\mu \mathrm{g} / \mathrm{L}, 2.22\mu \mathrm{g} / \mathrm{L}, 11.1\mu \mathrm{g} / \mathrm{L}, 22.2\mu \mathrm{g} / \mathrm{L}, 44.4\mu \mathrm{g} / \mathrm{L}, 111\mu \mathrm{g} / \mathrm{L};$ 庆大霉素 $\mathbf{C}_2 + \mathbf{C}_{2a}$ 组分为 $2.34\mu \mathrm{g} / \mathrm{L}, 4.68\mu \mathrm{g} / \mathrm{L}, 23.4\mu \mathrm{g} / \mathrm{L}, 46.8\mu \mathrm{g} / \mathrm{L}, 93.6\mu \mathrm{g} / \mathrm{L}, 234\mu \mathrm{g} / \mathrm{L})$ 的标准工作液，现用现配。

# 5.5 材料

5.5.1 混合模式的阳离子交换固相萃取柱：规格为 $60\mathrm{mg / 3mL}$ ，或相当者。  
5.5.2 微孔水系滤头：聚醚砜(PES)微孔滤膜， $0.22\mu \mathrm{m}$ 。  
5.5.3 塑料进样小瓶。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配电喷雾离子源(ESI)。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$ 。  
6.3 涡旋混合仪。  
6.4 组织匀浆机。  
6.5 超声清洗仪。  
6.6 高速冷冻离心机：转速不低于 $10000\mathrm{r / min}$ 。  
6.7 固相萃取装置。  
6.8 氮吹仪。

# 7 试样的制备与保存

# 7.1 试样的制备

7.1.1 猪、牛、羊、鸡的肌肉、脂肪、肝脏和肾脏组织

取新鲜或冷冻的空白或供试组织，绞碎，并使均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.1.2 牛奶

取新鲜或冷藏的空白或供试牛奶，并使均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.1.3 鸡蛋

取新鲜或冷藏的空白或供试鸡蛋，去壳，混合，并使均质。

a）取均质后的供试样品，作为供试试样；  
b）取均质后的空白样品，作为空白试样；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 提取

称取试样 $(2\pm 0.05)\mathrm{g}$ 于 $50~\mathrm{mL}$ 离心管中，加 $10~\mathrm{mL}$ 磷酸盐缓冲液提取，涡旋混匀 $1\mathrm{min}$ ，超声提取 $5\mathrm{min}$ ，以 $10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。取上清液于另一 $50~\mathrm{mL}$ 离心管中。残渣再加磷酸盐缓冲液 $4\mathrm{mL}$ ，重复提取1次，合并2次提取液并用磷酸盐缓冲液定容至 $15~\mathrm{mL}$ ，涡旋混匀，为备用液，供固相萃取柱净化。

# 8.2 净化

混合型阳离子交换柱依次用甲醇 $5\mathrm{mL}$ 和水 $5\mathrm{mL}$ 活化，取 $7.5\mathrm{mL}$ 的备用液上柱，流干后，依次用水 $5\mathrm{mL}$ 和甲醇 $5\mathrm{mL}$ 淋洗，抽干，用 $15\%$ 氨化甲醇 $5\mathrm{mL}$ 洗脱，收集洗脱液于 $10\mathrm{mL}$ 塑料离心管中，于 $40^{\circ}\mathrm{C}$ 下用氮气吹至近干，加入 $1\mathrm{mL}1\%$ 七氟丁酸酐醋酸溶液复溶，充分涡旋。用 $0.22\mu \mathrm{m}$ 滤膜过滤，滤液装塑料进样小瓶，供液相色谱-串联质谱仪测定。

# 8.3 基质匹配标准曲线的制备

取空白样品，按8.1和8.2处理，获得试样空白基质，分别取庆大霉素标准工作液 $1\mathrm{mL}$ ，溶解吹至近干后的空白试料残余物，制得浓度 $5\mu \mathrm{g} / \mathrm{L},10\mu \mathrm{g} / \mathrm{L},50\mu \mathrm{g} / \mathrm{L},100\mu \mathrm{g} / \mathrm{L},200\mu \mathrm{g} / \mathrm{L},500\mu \mathrm{g} / \mathrm{L}$ 的系列基质匹配标准工作溶液（其中，庆大霉素 $C_7$ 组分为 $1.55\mu \mathrm{g} / \mathrm{L},3.1\mu \mathrm{g} / \mathrm{L},15.5\mu \mathrm{g} / \mathrm{L},34.0\mu \mathrm{g} / \mathrm{L},62.0\mu \mathrm{g} / \mathrm{L},$ $155.0\mu \mathrm{g} / \mathrm{L}$ ；庆大霉素 $C_{1a}$ 组分为 $1.11\mu \mathrm{g} / \mathrm{L},2.22\mu \mathrm{g} / \mathrm{L},11.1\mu \mathrm{g} / \mathrm{L},22.2\mu \mathrm{g} / \mathrm{L},14.4\mu \mathrm{g} / \mathrm{L},111.0\mu \mathrm{g} / \mathrm{L};$ 庆大霉素 $C_2 + C_{2a}$ 组分为 $2.34\mu \mathrm{g} / \mathrm{L},4.68\mu \mathrm{g} / \mathrm{L},2a,4\mu \mathrm{g} / \mathrm{L},46.8\mu \mathrm{g} / \mathrm{L},93.6\mu \mathrm{g} / \mathrm{L},234.0\mu \mathrm{g} / \mathrm{L})$ ，供液相色谱-串联质谱仪测定。以各组分特征离子质量色谱峰面积为纵坐标、标准溶液浓度为横坐标，绘制基质匹配标准曲线。求各组分回归方程和相关系数，当样品中庆大霉素含量超出标准曲线线性范围时，用相应空白基质溶液稀释至标准曲线线性范围内后进行测定。

# 8.4.3 定性测定

通过样品色谱图的保留时间与标准品的保留时间、色谱峰的特征离子与相应浓度标准品色谱峰的特征离子相对照定性。试样溶液与标准溶液被测物保留时间偏差在 $\pm 0.1\mathrm{min}$ 以内；且检测到的相对离子丰度应与浓度相当的校正标准溶液相对离子丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.4.4 定量测定

取空白样品溶液与相应的标准工作液，配制基质标准曲线，做单点或多点校准，按外标法计算试样中庆大霉素的残留量。实测试料采用多点校正的方式定量；对于试料中庆大霉素残留量超过仪器测定线性范围的，将样品用空白基质溶液稀释到线性范围内后进行定量。庆大霉素标准溶液定量离子色谱图见附录A。

# 8.5 空白试验

取空白试样，除不加药物外，采用完全相同的测定步骤进行测定。

# 9 结果计算和表述

按标准曲线或公式(1)计算供试试料中庆大霉素单组分残留量 $X_{\mathrm{C_1}}$ 、 $X_{\mathrm{C_{1a}}} 、 X_{\mathrm{C_2 + C_{2a}}}$ ，再按公式(2)将各组分残留量相加即得到庆大霉素总组分残留量。

$$
X = \frac {C _ {\mathrm {s}} \times A \times V _ {1} \times V _ {3}}{A _ {\mathrm {s}} \times m \times V _ {2}} \dots \tag {1}
$$

$$
X _ {\text {总}} = X _ {\mathrm {C} _ {1}} + X _ {\mathrm {C} _ {1 \mathrm {e}}} + X _ {\mathrm {C} _ {2} - \mathrm {C} _ {2 \mathrm {s}}} \quad \dots \tag {2}
$$

式中：

$X$ ——试料中被测庆大霉素单组分残留量的数值，单位为微克每千克， $(\mu \mathrm{g} / \mathrm{kg})$ ，分别用 $X_{\mathrm{C}_1}, X_{\mathrm{C}_{1\mathrm{a}}}, X_{\mathrm{C}_2 - \mathrm{C}_{2\mathrm{a}}}$ 表示；

$C_{s}$ ——对照溶液中庆大霉素单组分浓度的数值，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$A$ ——试料溶液中庆大霉素单组分的色谱峰面积；

$V_{1}$ 备用液总体积的数值，单位为毫升 $(\mathrm{mL})$

$V_{3}$ ——残余物定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——对照溶液中庆大霉素单组分的色谱峰面积；

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$ ；

$V_{2}$ 过柱上样液体积的数值，单位为毫升 $(\mathrm{mL})$

$X_{\text{总}}$ ——被测试料中庆大霉素总组分残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ 。

注：计算结果以平行测定结果的算术平均值表示，保留3位有效数字。

# 10 检测方法灵敏度、正确度和精密度

# 10.1 灵敏度

本方法在猪、牛、羊、鸡肌肉、脂肪、肝脏和肾脏组织以及鸡蛋和牛奶中的检出限为 $5\mu \mathrm{g} / \mathrm{kg}$ ；定量限为 $10~\mu \mathrm{g / kg}$ 。

# 10.2 正确度

本方法庆大霉素在 $10\mu \mathrm{g} / \mathrm{kg}\sim 10000\mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平下，庆大霉素总组分及各组分的回收率范围为 $60\% \sim 120\%$ 。

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

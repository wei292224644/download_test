# 食品安全国家标准

# 食品添加剂 二氢香芹醇

# 1 范围

本标准适用于食品添加剂二氢香芹醇。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

6-甲基-3-异丙烯基环己醇

# 2.2 分子式

$\mathrm{C_{10}H_{18}O}$

# 2.3 结构式

![](images/dccce72e20b2bd534e42f0f051ef6941c41d5af050ae24b1245c9d0afc1924ed.jpg)

# 2.4 相对分子质量

154.25（按2007年国际相对原子质量）

# 食品中甲醛的测定

2025-03-16 发布

2025-09-16 实施

# 前言

本标准代替GB/T5009.49—2008《发酵酒及其配制酒卫生标准的分析方法》中4.4甲醛的测定方法。

本标准与GB/T5009.49—2008中甲醛的测定方法相比，主要变化如下：

——标准名称更改为《食品安全国家标准 食品中甲醛的测定》；  
——增加了标准的适用范围；  
——保留“分光光度法”作为第一法，更改了方法的适用范围，增加了方法的检出限和定量限，对蒸馏提取过程进行了优化；  
——增加了“液相色谱法”为第二法。

# 食品安全国家标准

# 食品中甲醛的测定

# 1 范围

本标准规定了食品中甲醛的测定方法。

本标准第一法适用于啤酒中甲醛的测定；第二法适用于食品中甲醛的测定。

# 第一法 分光光度法

# 2 原理

样品经水蒸气蒸馏，馏出液中的甲醛与乙酰丙酮反应生成黄色物质，在 $415~\mathrm{nm}$ 处有最大吸收，在一定范围内，其吸光度值与甲醛含量成正比。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 冰乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
3.1.2 乙酰丙酮 $(\mathrm{C}_5\mathrm{H}_8\mathrm{O}_2)$ 。  
3.1.3 乙酸铵 $(\mathrm{C}_2\mathrm{H}_7\mathrm{O}_2\mathrm{N})$ 。  
3.1.4 磷酸 $\mathrm{(H_3PO_4)}$ 。  
3.1.5 二甲基硅油。

# 3.2 试剂配制

3.2.1 乙酰丙酮溶液：称取 $25.0\mathrm{g}$ 乙酸铵溶于 $90~\mathrm{mL}$ 水中，再加入 $0.4\mathrm{mL}$ 乙酰丙酮和 $3.0\mathrm{mL}$ 冰乙酸，用水稀释至 $100~\mathrm{mL}$ ，混匀，移至棕色试剂瓶中。临用现配。  
3.2.2 磷酸溶液 $(200\mathrm{g / L})$ ：称取 $20\mathrm{g}$ 磷酸，用水稀释至 $100~\mathrm{mL}$ ，混匀，室温保存。

# 3.3 标准品

甲醛 $\mathrm{(CH_2O, CAS}$ 号：50-00-0)标准溶液： $1000\mathrm{mg / L}$ ，或其他经国家认证并授予标准物质证书的标准品。

注：甲醛标准溶液也可通过 $36\% \sim 38\%$ 甲醛溶液配制并标定后使用，具体操作见附录A。

# 3.4 标准溶液配制

3.4.1 甲醛标准中间液 $(100\mathrm{mg / L})$ ：准确吸取 $1000~\mathrm{mg / L}$ 甲醛标准溶液 $1.00~\mathrm{mL}$ ，用水稀释并定容至

$10~\mathrm{mL}$ ，配制成质量浓度为 $100~\mathrm{mg / L}$ 甲醛标准中间液， $2^{\circ}C\sim 8^{\circ}C$ 避光保存，有效期3个月。

3.4.2 甲醛标准工作液 $(1.0\mathrm{mg / L})$ ：准确吸取 $100~\mathrm{mg / L}$ 甲醛标准中间液 $1.0~\mathrm{mL}$ ，用水稀释并定容至 $100~\mathrm{mL}$ ，配制成质量浓度为 $1.0~\mathrm{mg / L}$ 甲醛标准工作液。临用现配。  
3.4.3 甲醛系列标准工作液：准确吸取 $1.0\mathrm{mg / L}$ 甲醛标准工作液 $0\mathrm{mL},0.50\mathrm{mL},1.0\mathrm{mL},2.0\mathrm{mL}$ $4.0~\mathrm{mL},8.0~\mathrm{mL}$ 和 $10.0~\mathrm{mL}$ 分别置于 $25~\mathrm{mL}$ 具塞比色管中，加水定容至 $10~\mathrm{mL}$ ，配制成质量浓度分别为 $0\mathrm{mg / L},0.050\mathrm{mg / L},0.10\mathrm{mg / L},0.20\mathrm{mg / L},0.40\mathrm{mg / L},0.80\mathrm{mg / L}$ 和 $1.0\mathrm{mg / L}$ 甲醛标准系列工作液。临用现配。

# 4 仪器和设备

4.1 分光光度计。  
4.2 水蒸气蒸馏装置。  
4.3 水浴锅：可控温度 $100^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ ，精度为 $0.1^{\circ}\mathrm{C}$ 。

# 5 分析步骤

# 5.1 试样制备

啤酒试样应密封保存；测试前将试样混合均匀，并用中速滤纸过滤的方式除去试样中的二氧化碳。

# 5.2 试样分析

# 5.2.1 蒸馏

准确吸取 $25.00\mathrm{mL}$ 试样，置于 $500~\mathrm{mL}$ 蒸馏瓶中，加入 $20~\mathrm{mL}$ 磷酸溶液 $(200\mathrm{g / L})$ 、1mL二甲基硅油、 $50~\mathrm{mL}$ 水，进行水蒸气蒸馏；预先在 $250~\mathrm{mL}$ 接收瓶中加入 $10~\mathrm{mL}$ 水，接收管插入液面下，并置于冰浴中；待收集馏出液约 $195~\mathrm{mL}$ 时，取出；室温下放置 $30\mathrm{min}$ ，转移至 $200~\mathrm{mL}$ 容量瓶中，加水定容至刻度，摇匀备用。

# 5.2.2 显色

准确吸取5.2.1制备的馏出液 $10.0\mathrm{mL}$ ，置于 $25~\mathrm{mL}$ 具塞比色管中。

向上述具塞比色管中加入 $1\mathrm{mL}$ 乙酰丙酮溶液(3.2.1)，盖塞后，混匀。将具塞比色管在沸水浴中放置 $5\mathrm{min}$ ，取出，冷却至室温。

# 5.3 空白试验

除不加试样外，其他步骤均按5.2步骤进行。

# 5.4 标准曲线的制作

按5.2.2的方法，将甲醛系列标准工作液(3.4.3)与试样同时同样处理；冷却至室温后，置于 $1\mathrm{cm}$ 比色皿中，以甲醛标准系列工作液的零点为参比调零，于 $415~\mathrm{nm}$ 测定吸光度值；以标准系列工作液的浓度为横坐标，以其吸光度值为纵坐标，绘制标准曲线。

# 5.5 试样溶液的测定

将5.2和5.3所得溶液置于 $1\mathrm{cm}$ 比色皿中，于 $415~\mathrm{nm}$ 测定吸光度值；若样品测得结果超出标准线

性范围，则用空白试样溶液稀释后再进行测定。

注：所得样液应在 $1\mathrm{h}$ 内进行测定。

# 6 分析结果的表述

试样中甲醛的含量按式(1)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V}{V _ {1} \times 1 0 0 0} \times 1 0 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——试样中甲醛的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho$ ——由标准曲线得到的试样馏出液中甲醛的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——由标准曲线得到的空白馏出液中甲醛的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样馏出液的定容体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——试样的取样量，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数。

计算结果保留2位有效数字。

# 7 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不应超过算术平均值的 $10\%$ 。

# 8 其他

当取样量为 $25.00~\mathrm{mL}$ 时，本方法的检出限为 $0.4\mathrm{mg / L}$ ，定量限为 $1.2\mathrm{mg / L}$ 。

# 第二法 液相色谱法

# 9 原理

在一定温度下，试样中的甲醛与衍生剂2，4-二硝基苯肼(DNPH)反应生成甲醛2，4-二硝基苯腙，净化提取后，经液相色谱分离，二极管阵列检测器或紫外检测器检测，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 乙腈 $(\mathrm{C}_2\mathrm{H}_3\mathrm{N})$ ：色谱纯。  
10.1.2 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。  
10.1.3 磷酸氢二钠 $(\mathrm{Na}_2\mathrm{HPO}_4)$ 。  
10.1.4 2,4-二硝基苯肼 $\left(\mathrm{C}_{6} \mathrm{H}_{6} \mathrm{~N}_{4} \mathrm{O}_{4}\right)$ 。

10.1.5 氯化钠 $(\mathrm{NaCl})$ 。

# 10.2 试剂配制

10.2.1 磷酸缓冲溶液 $(\mathrm{pH} = 2)$ ：量取 $16.6 \mathrm{~mL}$ 磷酸，用水稀释至 $1 \mathrm{~L}$ ，得到溶液 A；再称取 $71.63 \mathrm{~g}$ 磷酸氢二钠，用水溶解并稀释至 $1 \mathrm{~L}$ ，得到溶液 B；分别量取 $725 \mathrm{~mL}$ 溶液 A 和 $275 \mathrm{~mL}$ 溶液 B，混匀备用。  
10.2.2 2,4-二硝基苯肼溶液 $(0.6\mathrm{g} / \mathrm{L})$ ：称取 $0.3\mathrm{g}2,4-$ 二硝基苯肼，用乙腈溶解并稀释至 $500~\mathrm{mL}$ ，室温避光保存。  
10.2.3 衍生液: 取等体积的磷酸缓冲溶液 (10.2.1) 和 2,4-二硝基苯肼溶液 $(0.6 \mathrm{~g} / \mathrm{L})$ , 混匀备用。临用现配。

# 10.3 标准品

同3.3。

# 10.4 标准中间液配制

甲醛标准中间液 $(100\mathrm{mg / L})$ ：准确吸取 $1000\mathrm{mg / L}$ 甲醛标准溶液 $1.00~\mathrm{mL}$ ，用水稀释并定容至 $10~\mathrm{mL}$ ，配制成质量浓度为 $100\mathrm{mg / L}$ 甲醛标准中间液， $2^{\circ}C\sim 8^{\circ}C$ 避光保存，有效期3个月。

# 11 仪器和设备

11.1 高效液相色谱仪：配有二极管阵列检测器或紫外检测器。  
11.2 电子天平：感量为 $1\mathrm{mg}$ 。  
11.3 恒温水浴振荡器：可控温度 $25^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ ，精度为 $0.1^{\circ}\mathrm{C}$ 。  
11.4 涡旋混合器。  
11.5 离心机：转速 $\geqslant 4000\mathrm{r / min}$   
11.6 $0.45\mu \mathrm{m}$ 有机系滤膜。

# 12 分析步骤

# 12.1 试样制备

固体试样，取可食部分，捣碎后制备成均匀试样，装入洁净的容器，密封并做好标识，冷藏保存。液体试样应密封保存，测试前将试样混合均匀；含二氧化碳的试样，用中速滤纸过滤的方式除去二氧化碳。

# 12.2 试样分析

# 12.2.1 衍生

酒类样品，准确吸取试样 $5.00~\mathrm{mL}$ ；其他样品，称取制备好的试样 $2\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 塑料离心管中，准确加入 $20~\mathrm{mL}$ 衍生液（10.2.3)，盖紧管盖，涡旋混匀，置于恒温水浴振荡器中， $25^{\circ}C$ $\pm$ $1^{\circ}C$ 振摇 $1\textrm{h}$ ，使其充分反应。

# 12.2.2 提取

向上述试液中立即加入 $8\mathrm{g}$ 氯化钠，涡旋混匀 $30~\mathrm{s}$ ，以不低于 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上层溶液

于 $20~\mathrm{mL}$ 容量瓶中；在离心管中再加入 $10~\mathrm{mL}$ 乙腈，涡旋混匀 $30~\mathrm{s}$ ，以不低于 $4000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，合并上层溶液，并用乙腈定容至刻度，混匀。经 $0.45\mu \mathrm{m}$ 有机系滤膜过滤，供液相色谱测定。

# 12.3 空白试验

除不加试样外，其他步骤均按12.2步骤进行。

# 12.4 仪器参考条件

12.4.1 色谱柱： $\mathrm{C_{18}}$ 柱（柱长 $250~\mathrm{mm}$ ，内径 $4.6~\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ），或性能相当的色谱柱。  
12.4.2 流动相：乙腈-水（50:50，体积比）。  
12.4.3 柱温： $30^{\circ}C$   
12.4.4 检测波长： $350~\mathrm{nm}$   
12.4.5 流速： $1\mathrm{mL} / \mathrm{min}$   
12.4.6 进样量： $20~\mu \mathrm{L}$

# 12.5 标准曲线的制作

分别准确吸取 $100\mathrm{mg / L}$ 甲醛标准中间液 $0\mathrm{mL},0.050\mathrm{mL},0.10\mathrm{mL},0.20\mathrm{mL},0.40\mathrm{mL},0.60\mathrm{mL}$ 和 $0.80~\mathrm{mL}$ 于 $50~\mathrm{mL}$ 塑料离心管中，按照12.2的衍生和提取步骤，得到质量浓度分别为 $0\mathrm{mg / L}$ 、 $0.25\mathrm{mg / L},0.50\mathrm{mg / L},1.0\mathrm{mg / L},2.0\mathrm{mg / L},3.0\mathrm{mg / L}$ 和 $4.0\mathrm{mg / L}$ 甲醛标准系列工作液。

将标准系列工作液按浓度由低到高依次注入高效液相色谱仪中，测定相应的峰面积，以标准工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。标准溶液的液相色谱图参见附录B中图B.1。

# 12.6 试样溶液的测定

将试样溶液注入液相色谱仪中，得到相应的峰面积，根据标准曲线得到试样溶液中甲醛的浓度。注：衍生后所得样液应在 $24\mathrm{h}$ 内进行测定。

# 13 分析结果的表述

试样中甲醛的含量按式(2)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V}{m \times 1 0 0 0} \times 1 0 0 0 \tag {2}
$$

式中：

$X$ ——试样中甲醛的含量，单位为毫克每千克或毫克每升 $(\mathrm{mg / kg}$ 或 $\mathrm{mg / L})$

$\rho$ ——由标准曲线得到的试样溶液中甲醛的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——由标准曲线得到的空白试验样液中甲醛的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样的取样量，单位为克或毫升 $(\mathrm{g}$ 或 $\mathrm{mL}$ ）；

1000 ——换算系数。

计算结果保留2位有效数字。

# 14 精密度

在重复性条件下获得的2次独立测定结果的绝对差值不应超过算术平均值的 $10\%$ 。

# 15 其他

酒类样品，当取样量为 $5\mathrm{mL}$ 时，本方法的检出限为 $0.3\mathrm{mg / L}$ ，定量限为 $1.0\mathrm{mg / L}$ 。其他样品，当取样量为 $2\mathrm{g}$ 时，本方法的检出限为 $1.5\mathrm{mg / kg}$ ，定量限为 $5.0\mathrm{mg / kg}$ 。

# 附录A

# 甲醛标准溶液的配制及标定

# A.1 试剂

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

A.1.1 甲醛溶液 $(\mathrm{CH}_2\mathrm{O})$ 。

A.1.2 硫酸 $\mathrm{(H_2SO_4)}$ 。  
A.1.3 氢氧化钠(NaOH)。  
A.1.4 可溶性淀粉。  
A.1.5 碘 $(\mathrm{I}_2)$ 。  
A.1.6 硫代硫酸钠 $\mathrm{(Na_2S_2O_3)}$

# A.2 试剂配制

A.2.1 硫酸溶液 $(1\mathrm{mol} / \mathrm{L})$ ：量取 $54~\mathrm{mL}$ 硫酸，缓慢注入适量水中，冷却至室温后用水定容至 $1000~\mathrm{mL}$ 。  
A.2.2 碘溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：按照GB/T601中4.9的方法配制。  
A.2.3 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：按照GB/T601中4.1的方法配制。  
A.2.4 淀粉指示剂 $(5\mathrm{g} / \mathrm{L})$ ：称取 $0.5\mathrm{g}$ 可溶性淀粉，加 $5\mathrm{mL}$ 水使其成糊状，在搅拌下将糊状物缓缓倒入 $90~\mathrm{mL}$ 沸水中，搅匀，煮沸 $2\mathrm{min}$ ，冷却稀释至 $100~\mathrm{mL}$ 。临用现配。  
A.2.5 硫代硫酸钠标准滴定溶液 $(0.100\mathrm{mol} / \mathrm{L})$ ：按照GB/T601中4.6的方法配制和标定。

# A.3 甲醛标准溶液的配制

吸取 $36\% \sim 38\%$ 甲醛溶液 $7.0~\mathrm{mL}$ ，加入 $1\mathrm{mol} / \mathrm{L}$ 硫酸溶液 $0.5\mathrm{mL}$ ，用水定容至 $250~\mathrm{mL}$ ，混匀；吸取上述溶液 $10.0~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加水定容至刻度。

# A.4 甲醛标准溶液的标定

吸取 $10.0\mathrm{mL}$ 甲醛标准溶液(A.3)于 $250~\mathrm{mL}$ 碘量瓶中，加水 $90~\mathrm{mL},0.1~\mathrm{mol / L}$ 碘溶液 $20~\mathrm{mL}$ 和 $1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液 $15~\mathrm{mL}$ ，摇匀，放置 $15\mathrm{min}$ 。再加入 $1\mathrm{mol} / \mathrm{L}$ 硫酸溶液 $20~\mathrm{mL}$ 酸化，用 $0.1000\mathrm{mol} / \mathrm{L}$ 硫代硫酸钠标准溶液滴定至淡黄色，然后加入 $5\mathrm{g} / \mathrm{L}$ 淀粉指示剂 $1\mathrm{mL}$ ，继续滴定至蓝色褪去即为终点。同时做试剂空白试验。

甲醛标准储备液的浓度按公式(A.1)计算。

$$
X = \frac {\left(V _ {0} - V _ {1}\right) \times c \times 1 5 \times 1 0 0 0}{1 0} \tag {A.1}
$$

式中：

$X$ ——甲醛标准储备液的浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V_{0}$ ——滴定空白溶液消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——滴定甲醛溶液消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫代硫酸钠标准滴定溶液的摩尔浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

15 ——0.5 mol 甲醛的质量，单位为克每摩尔 $(\mathrm{g} / \mathrm{mol})$ ；

1000 单位换算系数；

10 ——滴定时吸取甲醛溶液的体积，单位为毫升 $(\mathrm{mL})$ 。

结果保留4位有效数字。

# 食品安全国家标准

# 保健食品中α-亚麻酸、二十碳五烯酸、二十二碳五烯酸和二十二碳六烯酸的测定

# 1 范围

本标准规定了保健食品中 $\alpha$ -亚麻酸、二十碳五烯酸（简称EPA，下同）、二十二碳五烯酸（简称DPA，下同）和二十二碳六烯酸（简称DHA，下同）的气相色谱测定方法。

本标准适用于保健品食品中α-亚麻酸、EPA、DPA、DHA的测定，不适用于以脂肪酸乙酯为有效成分的保健食品中α-亚麻酸、EPA、DPA、DHA的测定。

# 2 原理

试样经酸水解后提取脂肪，其中 $\alpha$ -亚麻酸、二十碳五烯酸（EPA）、二十二碳五烯酸（DPA）、二十二碳六烯酸（DHA）经酯交换生成甲酯后，通过气相色谱分离检测，以保留时间定性，外标法定量。

# 3 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 氢氧化钾（KOH）。  
3.1.2 盐酸（HCl）。  
3.1.3 无水乙醚（ $\mathrm{C_2H_5OC_2H_5}$ ）。  
3.1.4 乙醇（ $\mathrm{CH}_3\mathrm{CH}_2\mathrm{OH}$ ）：体积分数 $\geqslant 95\%$   
3.1.5 石油醚：沸程 $30^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$   
3.1.6 正己烷 $(\mathrm{CH}_3(\mathrm{CH}_2)_4\mathrm{CH}_3)$ ：色谱纯。  
3.1.7 甲醇（ $\mathrm{CH}_3\mathrm{OH}$ ）：色谱纯。  
3.1.8 无水硫酸钠（ $\mathrm{Na_2SO_4}$ ）。

# 3.2 试剂配制

氢氧化钾甲醇溶液（ $0.5\mathrm{mol} / \mathrm{L}$ ）：称取 $2.8\mathrm{g}$ 氢氧化钾，用甲醇溶解并定容至 $100\mathrm{mL}$ ，混匀。

# 3.3 标准品

3.3.1 $\alpha$ -亚麻酸甲酯 $(\mathrm{C_{19}H_{32}O_2})$ ：纯度 $\geq 99.0\%$ 。  
3.3.2 EPA甲酯 $(\mathrm{C_{21}H_{32}O_2})$ ：纯度 $\geqslant 98.5\%$   
3.3.3 DPA甲酯 $(\mathrm{C}_{23}\mathrm{H}_{36}\mathrm{O}_2)$ ：纯度 $\geq 98.0\%$   
3.3.4 DHA甲酯 $(\mathrm{C}_{23}\mathrm{H}_{34}\mathrm{O}_2)$ ：纯度 $\geq 98.5\%$

# 3.4 标准溶液的配制

3.4.1 单个脂肪酸甲酯标准储备液 $(4.0\mathrm{mg / mL})$ ：称取 $100.0\mathrm{mg}\alpha$ -亚麻酸甲酯、EPA 甲酯、DPA 甲酯、DHA 甲酯标准物质于 $25.0\mathrm{mL}$ 容量瓶中，分别用正己烷溶解并定容至刻度，摇匀。此溶液应贮存于 $-18^{\circ}C$ 冰箱中。  
3.4.2 脂肪酸甲酯混合标准中间液 $(1.0\mathrm{mg / mL})$ ：分别吸取脂肪酸甲酯标准储备液 $2.50~\mathrm{mL}$ 于 $10.0~\mathrm{mL}$ 容量瓶中，摇匀，亦为标准曲线最高浓度，临用时配制。  
3.4.3 脂肪酸甲酯标准工作液：分别吸取脂肪酸甲酯中间液 $0.40\mathrm{mL}$ 、 $0.80\mathrm{mL}$ 、 $1.0\mathrm{mL}$ 、 $2.0\mathrm{mL}$ 、 $4.0\mathrm{mL}$ 于 $10.0\mathrm{mL}$ 容量瓶中，用正己烷定容，此浓度即为 $0.040\mathrm{mg/mL}$ 、 $0.080\mathrm{mg/mL}$ 、 $0.10\mathrm{mg/mL}$ 、 $0.20\mathrm{mg/mL}$ 、 $0.40\mathrm{mg/mL}$ 的标准工作液，临用时配制。

# 4 仪器与设备

4.1 气相色谱仪：配有氢火焰离子化检测器（FID）。  
4.2 天平：感量为 $1\mathrm{mg}$ 和 $0.1\mathrm{mg}$   
4.3 旋转蒸发仪。  
4.4 离心机：转速 $\geq 4000\mathrm{r / min}$   
4.5 涡旋混合器。  
4.6 恒温水浴锅。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 试样处理

# 5.1.1.1 固体试样

称取已粉碎混合均匀的待测试样 $0.5 \mathrm{~g} \sim 2 \mathrm{~g}$ （精确到 $0.001 \mathrm{~g}$ ）（含待测组分约 $5 \mathrm{mg} / \mathrm{g} \sim 10 \mathrm{mg} / \mathrm{g}$ ）加入 $50 \mathrm{~mL}$ 比色管中，加 $8 \mathrm{~mL}$ 水，混匀后再加 $10 \mathrm{~mL}$ 盐酸。将比色管放入 $70^{\circ} \mathrm{C} \sim 80^{\circ} \mathrm{C}$ 水浴中，每隔 $5 \mathrm{~min} \sim 10 \mathrm{~min}$ 以涡旋混合器混合一次，至试样水解完全为止，约需 $40 \mathrm{~min} \sim 50 \mathrm{~min}$ 。取出比色管，加入 $10 \mathrm{~mL}$ 乙醇，混合。冷却至室温后将混合物移入 $100 \mathrm{~mL}$ 具塞量筒中，以 $25 \mathrm{~mL}$ 无水乙醚分次洗比色管，一并倒入量筒中。密塞振摇 $1 \mathrm{~min}$ 。加入 $25 \mathrm{~mL}$ 石油醚，密塞振摇 $1 \mathrm{~min}$ ，静置 $30 \mathrm{~min}$ ，分层，将吸出的有机层过无水硫酸钠（约 $5 \mathrm{~g}$ ）滤入浓缩瓶中。再加入 $25 \mathrm{~mL}$ 无水乙醚密塞振摇 $1 \mathrm{~min}$ ， $25 \mathrm{~mL}$ 石油醚，密塞振摇 $1 \mathrm{~min}$ ，静置、分层，将吸出的有机层经过无水硫酸钠（约 $5 \mathrm{~g}$ ）滤入浓缩瓶中，按“再加入 $25 \mathrm{~mL}$ 无水乙醚……，静置、分层、过无水硫酸钠”重复操作一次，将全部提取液用旋转蒸发仪于 $45^{\circ} \mathrm{C}$ 减压浓缩近干。用正己烷少量多次溶解浓缩物，转移至 $25 \mathrm{~mL}$ 容量瓶并定容，摇匀。按5.1.2步骤甲酯化处理。

# 5.1.1.2 油类制品

称取混合均匀的油类制品 $0.2 \mathrm{~g} \sim 1 \mathrm{~g}$ （精确到 $0.001 \mathrm{~g}$ ）（含待测组分约 $10 \mathrm{mg} / \mathrm{g} \sim 20 \mathrm{mg} / \mathrm{g}$ 。）至 $25 \mathrm{~mL}$ 容量瓶中，加入 $5 \mathrm{~mL}$ 正己烷轻摇溶解，并用正己烷定容至刻度，摇匀。按5.1.2步骤甲酯化处理，脂肪酸乙酯型油类制品的物理鉴别参见附录A。

# 5.1.2 甲酯化

吸取待测液（5.1.1.1，5.1.1.2） $2.0\mathrm{mL}$ 至 $10\mathrm{mL}$ 具塞刻度试管中，加入 $2.0\mathrm{mL}$ 氢氧化钾甲醇溶液，立即移至涡旋混合器上振荡混合 $5\mathrm{min}$ ，静置 $5\mathrm{min}$ ，加入 $6\mathrm{mL}$ 蒸馏水，上下振摇 $0.5\mathrm{min}$ ，静置分层后，吸取下层液体，弃去后再反复用少量蒸馏水进行洗涤，并用吸管弃去水层，直至洗至中性（若有

机相有乳化现象，以 $4000\mathrm{r / min}$ 离心 $10\mathrm{min})$ ，吸取正己烷层待上机测试用。

注：如使用塑料离心管或塑料刻度试管进行试样处理须同步进行空白对照试验。

# 5.2 气相色谱参考条件

5.2.1 色谱柱：键合交联聚乙二醇固定相，柱长 $30\mathrm{m}$ ，内径 $0.32\mathrm{mm}$ ，膜厚 $0.5\mu \mathrm{m}$ 或同等性能的色谱柱。  
5.2.2 柱温箱温度：起始温度 $180^{\circ}\mathrm{C}$ ， $10^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $220^{\circ}\mathrm{C}$ ，再以 $8^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $250^{\circ}\mathrm{C}$ ，保持 $13\mathrm{min}$ 。  
5.2.3 进样口温度： $250^{\circ}\mathrm{C}$ ；进样量 $1\mu \mathrm{L}$ ，分流比 $20:1$ 。  
5.2.4 FID检测器温度： $270^{\circ}C$   
5.2.5 载气：高纯氮气，流量 $1.0\mathrm{mL / min}$ ，尾吹 $25~\mathrm{mL / min}$   
5.2.6 氢气： $40~\mathrm{mL / min}$ ；空气 $450~\mathrm{mL / min}$

# 5.3 标准曲线的制作

将 $1 \mu \mathrm{L}$ 的标准系列各浓度溶液（3.4.2、3.4.3），注入气相色谱仪中，测得相应的峰面积或峰高，以标准工作液的浓度为横坐标，以峰面积或峰高为纵坐标，绘制标准曲线（标准溶液气相色谱图见附录 B 中图 B.1）。

# 5.4 试验溶液的测定

将 $1 \mu \mathrm{L}$ 的试样待测液（5.1.2）注入气相色谱仪中，以保留时间定性，测得峰面积或峰高，根据标准曲线得到待测液中各脂肪酸甲酯的组分浓度（样品溶液气相色谱图见附录 B 中图 B.2）。

# 6 分析结果的表述

试样中 $\alpha$ -亚麻酸、EPA、DPA、DHA 含量按式（1）计算：

$$
X _ {i} = \frac {C _ {i} \times V \times F \times 1 0 0}{m \times 1 0 0 0} \dots \tag {1}
$$

式中：

$X_{f}$ ——试样中 $\alpha$ -亚麻酸、EPA、DPA、DHA 的含量，单位为克每百克（g/100g）；

$C_{\mathrm{f}}$ ——由标准曲线查得测定样液中各脂肪酸甲酯的浓度, 单位为毫克每毫升 $(\mathrm{mg} / \mathrm{mL})$ ;

$V$ ——被测定样液的最终定容体积, 单位为毫升 $(\mathrm{mL})$ ;

$m$ ——试样的称样质量，单位为克（g）；

$F$ ——各脂肪酸甲酯转化为脂肪酸的换算系数，其中： $\alpha$ -亚麻酸甲酯转化为 $\alpha$ -亚麻酸的转换系数为 0.9520；EPA 甲酯转化为 EPA 脂肪酸的转换系数为 0.9557；DPA 甲酯转化为 DPA 脂肪酸的转换系数为 0.9592；DHA 甲酯转化为 DHA 脂肪酸的转换系数为 0.9590；

100——单位转换；

1000——单位转换。

计算结果以重复条件下获得的两次独立测定结果的算术平均值表示，保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不超过算术平均值的 $10\%$ 。

# 8 其他

当试样量为 $0.5\mathrm{g}$ ，定容体积为 $25\mathrm{mL}$ ，各脂肪酸的定量限分别为 $\alpha$ -亚麻酸 $0.010\mathrm{g} / 100\mathrm{g}$ ，EPA $0.018\mathrm{g} / 100\mathrm{g}$ ，DPA $0.024\mathrm{g} / 100\mathrm{g}$ ，DHA $0.018\mathrm{g} / 100\mathrm{g}$ 。

# 附录A

# 脂肪酸乙酯型油类制品的物理鉴别

称取 $1\mathrm{g}$ 样品，加入 $1.0\mathrm{mL}$ 无水乙醇，于涡旋混合器（ $2000\mathrm{r / min}$ ）混匀 $30\mathrm{s}$ ，静置，观察油在乙醇中的溶解情况。如果油样不溶于乙醇，出现明显的油和乙醇分层的现象，则判断此产品为脂肪酸甘油酯型油类制品，可采用本标准进行分析；若静置后，看不见油状液滴，溶解完全且溶液清澈透明，可判断此产品为脂肪酸乙酯型油类制品，不能使用本标准进行分析。

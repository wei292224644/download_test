2016-08-31 发布

# 前言

本标准代替GB/T23499—2009《食品中残留过氧化氢的测定方法》。

本标准与GB/T23499—2009相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中过氧化氢残留量的测定”；  
——修改了标准的适用范围；  
——修改了碘量法中的计算公式；  
——修改了钛盐比色法的定量限。

# 食品安全国家标准

# 食品中过氧化氢残留量的测定

# 1 范围

本标准规定了食品中过氧化氢残留量的测定方法。

本标准适用于预包装牛奶、饮料、豆制品、水发产品、鸡爪等食品中过氧化氢残留量的测定。

# 第一法 碘量法

# 2 原理

食品中的强氧化物在稀硫酸中使碘化钾氧化，产生定量的碘，生成的碘以淀粉作指示剂，用硫代硫酸钠标准溶液滴定得到强氧化物总量。加入过氧化氢酶分解去除试样中的过氧化氢，用硫代硫酸钠标准溶液滴定去除过氧化氢后的其他氧化物含量。两次滴定结果之差可计算得到样品中过氧化氢的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 硫代硫酸钠 $\mathrm{(Na_2S_2O_3)}$ 。  
3.1.2 可溶性淀粉 $\left[\left(\mathrm{C}_{6} \mathrm{H}_{10} \mathrm{O}_{5}\right)_{n}\right]$ 。  
3.1.3 碘化钾(KI)。  
3.1.4 硫酸 $\mathrm{(H_2SO_4)}$ 。  
3.1.5 钼酸铵 $\left[(\mathrm{NH}_4)_6\mathrm{Mo}_7\mathrm{O}_{24}\cdot 4\mathrm{H}_2\mathrm{O}\right]$   
3.1.6 过氧化氢酶（单位活力大于 $200000\mathrm{U / mL}$ ）：置于 $-20^{\circ}C$ 保存。  
3.1.7 亚铁氰化钾 $\{\mathrm{K}_4[\mathrm{Fe}(\mathrm{CN})_6]\cdot 3\mathrm{H}_2\mathrm{O}\}$   
3.1.8 乙酸锌 $(\mathrm{C_4H_6O_4Zn}\cdot 2\mathrm{H_2O})$   
3.1.9 乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
3.1.10 活性炭。

# 3.2 试剂配制

3.2.1 淀粉指示剂 $(10\mathrm{g / L})$ ：称取可溶性淀粉 $0.50\mathrm{g}$ ，加少许水，调成糊状，倒入 $50~\mathrm{mL}$ 沸水中调匀，煮沸。临用时现配。  
3.2.2 碘化钾溶液 $(100\mathrm{g / L})$ ：称取 $10.00\mathrm{g}$ 碘化钾，加水溶解，定容至 $100~\mathrm{mL}$ ，贮于棕色瓶中。临用时现配。  
3.3.3 $10\%$ 硫酸溶液（质量分数）：量取 $60~\mathrm{mL}$ 硫酸，缓缓注入约 $700~\mathrm{mL}$ 水中，冷却，稀释至 $1000~\mathrm{mL}$ 。

3.3.4 $3\%$ 钼酸铵溶液：称取 $3.00\mathrm{g}$ 钼酸铵，加 $100~\mathrm{mL}$ 水溶解。  
3.3.5 $0.1\%$ 过氧化氢酶溶液：称 $0.10 \mathrm{~g}$ 过氧化氢酶，用 $100 \mathrm{~mL}$ 蒸馏水分多次将其溶解，冷藏可保存两个月。  
3.3.6 亚铁氰化钾溶液：称取 $106.0\mathrm{g}$ 亚铁氰化钾，用水溶解，并稀释至 $1000~\mathrm{mL}$   
3.3.7 乙酸锌溶液：称取 $220.0\mathrm{g}$ 乙酸锌，加 $30~\mathrm{mL}$ 乙酸，用水溶解，稀释至 $1000~\mathrm{mL}$   
3.3.8 活性炭：将 $100\mathrm{g}$ 活性炭加至 $750~\mathrm{mL}~1~\mathrm{mol / L}$ 盐酸中，回流 $1\mathrm{h}\sim 2\mathrm{h}$ ，过滤，用水洗数次，至滤液中无铁离子 $(\mathrm{Fe}^{3+})$ 为止，然后置于 $110^{\circ}\mathrm{C}$ 烘箱中烘干。

注：检验铁离子方法：利用普鲁士蓝反应。将 $20\mathrm{g / L}$ 亚铁氰化钾与 $1\%$ 盐酸等量混合，将3.3.8洗出滤液滴入，如有铁离子则产生蓝色沉淀。

# 3.3 标准溶液配制

3.3.1 $0.1\mathrm{mol} / \mathrm{L}$ 硫代硫酸钠标准储备液：按GB/T601规定的方法配制和标定。  
3.3.2 $0.0020\mathrm{mol} / \mathrm{L}$ 硫代硫酸钠标准使用液：临用时用标准储备液稀释。

# 4 仪器和设备

4.1 电子天平：感量为 $0.01\mathrm{g}$   
4.2 高速捣碎机。

# 5 分析步骤

# 5.1 试样制备

# 5.1.1 固体样品

称取粉碎均匀的试样可食部分 $10\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ），加适量水溶解，转移入 $100\mathrm{mL}$ 容量瓶中，对蛋白质、脂肪含量较高的样品可加入乙酸锌溶液 $5\mathrm{mL}$ 、亚铁氰化钾溶液 $5\mathrm{mL}$ ，加水定容至刻度 $(V_{1})$ ，摇匀。浸泡 $30\mathrm{min}$ ，用滤纸过滤，滤液作为试样液备用。

# 5.1.2 液体样品

称取 $25\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）试样于 $100~\mathrm{mL}$ 容量瓶中，对蛋白质、脂肪含量较高的样品可加入乙酸锌溶液 $5\mathrm{mL}$ 、亚铁氰化钾溶液 $5\mathrm{mL}$ ，加水定容至刻度 $(V_{1})$ ，摇匀，用滤纸过滤，滤液作为试样液备用。如样品滤液有颜色，加入 $1\mathrm{g}$ 活性炭，振摇 $1\mathrm{min}$ ，用干燥滤纸过滤，弃去初滤液，滤液待用。

# 5.2 测定

分别吸取滤液 $25.0\mathrm{mL}(V_2)$ ，置于A、B两个 $250~\mathrm{mL}$ 碘量瓶中，A瓶中加入 $0.1\%$ 过氧化氢酶溶液 $0.5\mathrm{mL}$ ，加盖混匀，放置 $10\mathrm{min}$ （放置过程中摇动数次）。在A、B两瓶中各加入 $10\%$ 硫酸溶液 $5.0~\mathrm{mL}$ 、碘化钾溶液 $5.0~\mathrm{mL},3\%$ 钼酸铵溶液3滴，混匀，置暗处放置 $10\mathrm{min}$ ，各加水 $50~\mathrm{mL}$ ，分别用硫代硫酸钠标准溶液滴定，待滴至微黄色时，加淀粉指示剂 $0.5\mathrm{mL}$ ，继续滴至蓝色消失，分别记录A、B两瓶消耗硫代硫酸钠标准溶液的毫升数。

# 6 分析结果的表述

试样中过氧化氢的含量，按式(1)计算：

$$
X = \frac {\left(V _ {\mathrm {B}} - V _ {\mathrm {A}}\right) \times c \times V _ {1} \times 1 7 . 0 1}{m \times V _ {2}} \times 1 0 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——样品中过氧化氢的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$V_{\mathrm{B}}$ ——B瓶中消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{\mathrm{A}}$ ——A瓶中消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——硫代硫酸钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{1}$ ——试样处理液总体积，单位为毫升 $(\mathrm{mL})$ ；

17.01——与 $1.00 \mathrm{~mL}$ 硫代硫酸钠标准溶液 $\left[c\left(\mathrm{Na}_{2} \mathrm{~S}_{2} \mathrm{O}_{3}\right)=1.000 \mathrm{~mol} / \mathrm{L}\right]$ 相当的过氧化氢的质量，单位为毫克 $(\mathrm{mg})$ ；

$m$ ——试样质量，单位为克(g)；

$V_{2}$ ——测定用样液体积，单位为毫升 $(\mathrm{mL})$

1000——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

本方法定量限为 $3\mathrm{mg / kg}$

# 第二法 钛盐比色法

# 9 原理

过氧化氢在酸性溶液中，与钛离子生成稳定的橙色络合物。在 $430\mathrm{nm}$ 下，吸光度与样品中过氧化氢含量成正比，用比色法测定样品中过氧化氢的含量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 10.1 试剂

10.1.1 高锰酸钾 $\mathrm{(KMnO_4)}$ 。  
10.1.2 $30\%$ 过氧化氢溶液 $\mathrm{(H_2O_2)}$   
10.1.3 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。  
10.1.4 二氧化钛 $(\mathrm{TiO_2})$ 。  
10.1.5 硫酸铵 $\left[\left(\mathrm{NH}_{4}\right)_{2} \mathrm{SO}_{4}\right]$ 。  
10.1.6 盐酸（HCl）。

# 10.2 试剂配制

10.2.1 钛溶液：称取 $1.00\mathrm{g}$ 二氧化钛 $\left(\mathrm{TiO}_{2}\right)$ 、 $4.00\mathrm{g}$ 硫酸铵于 $250~\mathrm{mL}$ 锥形瓶，加入 $100~\mathrm{mL}$ 浓硫酸，

上面放置一小漏斗，置于可控温电热套中 $150^{\circ}\mathrm{C}$ 保温 $15\mathrm{h} \sim 16\mathrm{h}$ ，冷却后以 $400\mathrm{mL}$ 水稀释，最后用滤纸过滤，清液备用。

10.2.2 $1\mathrm{mol} / \mathrm{L}$ 盐酸溶液：量取 $90~\mathrm{mL}$ 盐酸，加入 $1000~\mathrm{mL}$ 水中。  
10.2.3 硫酸溶液 $(1 + 4)$ ：量取 $10~\mathrm{mL}$ 硫酸，加入 $40~\mathrm{mL}$ 水中。

# 10.3 标准溶液配制

10.3.1 高锰酸钾标准溶液 $\left[c\left(\frac{1}{5} \mathrm{KMnO}_4\right) = 0.100 \mathrm{~mol} / \mathrm{L}\right]$ ：按GB/T601规定的方法配制和标定。  
10.3.2 过氧化氢标准储备液：吸取 $30\%$ 过氧化氢溶液 $1\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，加水至刻度，混匀。

吸取 $20.00\mathrm{mL}$ 过氧化氢标准储备液于 $250~\mathrm{mL}$ 锥形瓶中，加入 $10\%$ 硫酸溶液(3.2.3) $25~\mathrm{mL}$ ，用高锰酸钾标准溶液 $\left[c\left(\frac{1}{5} \mathrm{KMnO}_4\right)=0.100 \mathrm{~mol/L}\right]$ 滴定至微红色。

过氧化氢标准储备液的浓度，按式(2)计算：

$$
X = \frac {1 7 . 0 1 \times c \times V}{2 0 . 0 0} \tag {2}
$$

式中：

$X$ ——过氧化氢标准储备液浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

17.01——每毫升高锰酸钾 $\left[c\left(\frac{1}{5} \mathrm{KMnO}_{4}\right)=0.100 \mathrm{~mol} / \mathrm{L}\right]$ 标准溶液相当于过氧化氢的质量，单位为毫克 $(\mathrm{mg})$ ；

$c$ ——高锰酸钾 $\left[c\left(\frac{1}{5} \mathrm{KMnO}_4\right)=0.100 \mathrm{~mol} / \mathrm{L}\right]$ 标准溶液的浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

V ——滴定所用高锰酸钾标准溶液 $\left[c\left(\frac{1}{5} \mathrm{KMnO}_4\right) = 0.100 \mathrm{~mol} / \mathrm{L}\right]$ 的体积，单位为毫升 $(\mathrm{mL})$ 。

10.3.3 过氧化氢标准使用液：根据10.3.2的标定结果将过氧化氢标准储备液稀释成 $20~\mu \mathrm{g / mL}$ 。

# 11 仪器和设备

11.1 电子天平：感量为 $0.01\mathrm{g}$   
11.2 高速捣碎机。  
11.3 分光光度计：配 $5\mathrm{cm}$ 比色皿。

# 12 分析步骤

# 12.1 试样制备

同5.1。

# 12.2 标准曲线的制作

吸取 $0.00\mathrm{mL}, 0.25\mathrm{mL}, 0.50\mathrm{mL}, 1.00\mathrm{mL}, 2.50\mathrm{mL}, 5.00\mathrm{mL}, 7.50\mathrm{mL}, 10.0\mathrm{mL}$ 过氧化氢标准使用液（相当于 $0\mu \mathrm{g}, 5\mu \mathrm{g}, 10\mu \mathrm{g}, 20\mu \mathrm{g}, 50\mu \mathrm{g}, 100\mu \mathrm{g}, 150\mu \mathrm{g}, 200\mu \mathrm{g}$ 过氧化氢），分别置于 $25\mathrm{mL}$ 带塞比色管中。各加入钛溶液 $5.0\mathrm{mL}$ ，用水定容至 $25\mathrm{mL}$ ，摇匀，放置 $10\mathrm{min}$ 。用 $5\mathrm{cm}$ 比色皿，以空白管调节零点，于波长 $430\mathrm{nm}$ 处测吸光度。以标准系列的过氧化氢浓度 $(\mu \mathrm{g} / \mathrm{mL})$ 对吸光度绘制标准曲线。

# 12.3 试样溶液的测定

吸取 $10.00\mathrm{mL}(V_2)$ 试样液于 $25~\mathrm{mL}$ 带塞比色管中，各加入钛溶液 $5.0~\mathrm{mL}$ ，用水定容至 $25~\mathrm{mL}$ ，摇匀，放置 $10\mathrm{min}$ 。用 $5\mathrm{cm}$ 比色皿，以空白管调节零点，于波长 $430~\mathrm{nm}$ 处测吸光度。同时做试剂空白试验。

如果经过活性炭吸附后试样液仍有颜色干扰，应扣除试样液的本底色，即用 $5.0\mathrm{mL}$ 硫酸溶液 $(1 + 4)$ 代替钛溶液，其他按上述方法操作。

# 13 分析结果的表述

试样中过氧化氢的含量，按式(3)计算：

$$
X = \frac {c \times V _ {1}}{m \times V _ {2}} \quad \dots \tag {3}
$$

式中：

$X$ ——样品中过氧化氢含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样测定液中过氧化氢的质量，单位为微克 $(\mu \mathrm{g})$ ；

$V_{1}$ ——试样处理液总体积，单位为毫升 $(\mathrm{mL})$

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

$V_{2}$ ——测定用样液体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留三位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

本方法检出限为 $0.5\mathrm{mg / kg}$ ，定量限为 $3\mathrm{mg / kg}$

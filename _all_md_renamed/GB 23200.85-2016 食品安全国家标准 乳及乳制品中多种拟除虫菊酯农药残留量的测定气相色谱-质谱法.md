# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2912—2011《出口乳及乳制品中多种拟除虫菊酯农药残留量的检测方法气相色谱一质谱法》。与SN/T2912—2011相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；

——标准名称和范围中“出口乳及乳制品”改为“乳及乳制品”；

——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

# 食品安全国家标准乳及乳制品中多种拟除虫菊酯农药残留量的测定气相色谱一质谱法

# 1 范围

本标准规定了乳及乳制品中2,6-二异丙基萘、七氟菊酯、生物丙烯菊酯、烯虫酯、苄呋菊酯、联苯菊酯、甲氰菊酯、氯氟氰菊酯、氟丙菊酯、氯菊酯、氟氯氰菊酯、氯氰菊酯、氟氰戊菊酯、醚菊酯、氰戊菊酯、氟胺氰菊酯、溴氰菊酯17种多组分农药残留量的气相色谱一质谱检测方法。

本标准适用于液体乳、乳粉、炼乳、乳脂肪、干酪、乳冰激凌和乳清粉中2,6-二异丙基萘、七氟菊酯、生物丙烯菊酯、烯虫酯、苄呋菊酯、联苯菊酯、甲氰菊酯、氯氟氰菊酯、氟丙菊酯、氯菊酯、氟氯氰菊酯、氯氰菊酯、氟氰戊菊酯、醚菊酯、氰戊菊酯、氟胺氰菊酯、溴氰菊酯17种农药残留量的检测和确证，其他食品可参照执行。

# 3 原理

试样采用氯化钠盐析，乙腈匀浆提取，分取乙腈层，分别用 $\mathrm{C_{18}}$ 固相萃取柱和弗罗里硅土固相萃取柱净化，洗脱液浓缩溶解定容后，供气相色谱一质谱仪检测和确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(C_2H_3N}$ ,CAS号：75-05-8）：残留级。  
4.1.2 正己烷 $\mathrm{(C_6H_{14}}$ ，CAS号： $110 - 54 - 3$ ：残留级。  
4.1.3 乙酸乙酯 $\mathrm{(C_4H_8O_2}$ ，CAS号：141-78-6）：残留级。  
4.1.4 氯化钠(NaCl，CAS号：7647-14-5)。

# 4.2 溶液配制

正己烷一乙酸乙酯 $(9 + 2$ ，体积比)：量取 $90~\mathrm{mL}$ 正己烷和 $20~\mathrm{mL}$ 乙酸乙酯，混匀。

# 4.3 标准品

2,6-二异丙基萘等17种农药标准品：纯度 $\geq 98\%$ ，CAS号参见附录A。

# 4.4 标准溶液配制

# 4.4.1 2,6-二异丙基萘等17种农药标准储备溶液

分别准确称取适量的2,6-二异丙基萘等17种农药标准品，用正己烷配制成浓度为 $100\mu \mathrm{g / mL}$ 的标准储备溶液。该溶液于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 保存。

# 4.4.2 2,6-二异丙基萘等17种农药标准工作溶液

根据需要用不含2,6-二异丙基萘等17种农药的空白样品配制成适用浓度的标准工作溶液。该溶液现用现配。

# 4.5 材料

4.5.1 $\mathrm{C_{18}}$ 固相萃取柱： $\mathrm{C_{18}}$ ， $3\mathrm{mL} / 500\mathrm{mg}$   
4.5.2 弗罗里硅土固相萃取柱：Florisil, $3\mathrm{mL} / 500\mathrm{mg}$

# 5 仪器和设备

5.1 气相色谱—质谱仪：配有电子轰击源（EI）。  
5.2 分析天平：感量为 $0.01\mathrm{g}$   
5.3 分析天平：感量为 $0.0001\mathrm{g}$ 。  
5.4 匀浆机：转速不低于 $10000\mathrm{r / min}$   
5.5 离心机：转速不低于 $4000\mathrm{r / min}$   
5.6 氮吹仪。  
5.7 涡流混匀机。

# 6 试样制备与保存

# 6.1 试样制备

取样品约 $500 \mathrm{~g}$ , 取样部位按 GB 2763 的规定执行, 用粉碎机粉碎, 混匀, 装入洁净容器, 密封, 标明标记。在制样的操作过程中, 应防止样品受到污染或发生残留物含量的变化。

# 6.2 试样保存

试样于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 保存。

# 7 分析步骤

# 7.1 提取

准确称取液体乳、冰激凌试样 $2.0\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），加 $0.5\mathrm{g}$ 氯化钠、 $10.0\mathrm{mL}$ 乙腈，于 $10000\mathrm{r}/$ min 匀浆提取 $60\mathrm{s}$ ，再以 $4000\mathrm{r/min}$ 离心 $5\mathrm{min}$ ，准确移取 $5.0\mathrm{mL}$ 乙腈，于 $40^{\circ}\mathrm{C}$ 氮吹至大约 $1\mathrm{mL}$ ，待净化。

准确称取奶酪、奶粉、乳清粉、炼乳、乳脂肪试样 $2.0\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），加 $0.5\mathrm{g}$ 氯化钠、 $5\mathrm{mL}$ 水、 $10.0\mathrm{mL}$ 乙腈，于 $10000\mathrm{r / min}$ 匀浆提取 $60\mathrm{s}$ ，再以 $4000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，准确移取 $5.0\mathrm{mL}$ 乙腈，于 $40^{\circ}\mathrm{C}$ 氮吹至大约 $1\mathrm{mL}$ ，待净化。

# 7.2 净化

# 7.2.1 $\mathbf{C}_{18}$ 固相萃取净化

将7.1所得样品浓缩液倾入预先用 $5\mathrm{mL}$ 乙腈预淋洗的 $\mathrm{C}_{18}$ 固相萃取柱，用 $4\mathrm{mL}$ 乙腈洗脱，收集洗脱液，于 $40^{\circ}\mathrm{C}$ 氮吹至近干，用 $0.5\mathrm{mL}$ 正己烷涡流混合溶解残渣，待用。

# 7.2.2 弗罗里硅土固相萃取净化

将7.2.1所得洗脱液倾入预先用 $5\mathrm{mL}$ 正己烷一乙酸乙酯预淋洗的弗罗里硅土固相萃取柱，用5.0 $\mathrm{mL}$ 正己烷一乙酸乙酯洗脱，收集洗脱液，于 $40^{\circ}\mathrm{C}$ 氮吹至近干，用 $0.5\mathrm{mL}$ 正己烷涡流混合溶解残渣，供气质联用仪测定。

# 7.3 测定

# 7.3.1 气相色谱一质谱参考条件

a）色谱柱：TR-5MS石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或性能相当者；  
b）色谱柱温度： $50^{\circ}\mathrm{C}\xrightarrow{20^{\circ}\mathrm{C / min}}200^{\circ}\mathrm{C}(1\mathrm{min})\xrightarrow{5^{\circ}\mathrm{C / min}}280^{\circ}\mathrm{C}(10\mathrm{min})$   
c）进样口温度： $250^{\circ}\mathrm{C}$   
d） 色谱一质谱接口温度： $280^{\circ}C$   
e）电离方式：EI；  
f）离子源温度： $250^{\circ}\mathrm{C}$   
g) 灯丝电流: $25 \mu \mathrm{A}$ ;   
h）载气：氦气，纯度 $\geq 99.999\%$ ，流速 $1\mathrm{mL / min}$   
i）进样方式：无分流， $0.75\mathrm{min}$ 后打开分流阀；  
j）进样量： $1\mu \mathrm{L}$   
k）测定方式：选择离子监测；  
1) 选择监测离子 $(m / z)$ : 每种农药选择一个定量离子、3个定性离子, 每种农药的保留时间、定量离子、定性离子及定量离子与定性离子丰度比值参见附录B;  
m）溶剂延迟： $8.5\mathrm{min}$

# 7.3.2 色谱测定与确证

根据样液中待测物含量情况，选定浓度相近的标准工作溶液。标准工作溶液和待测样液中2,6-二异丙基萘等17种农药的响应值均应在仪器检测的线性范围内。标准工作溶液与样液等体积参插进样测定。

标准溶液及样液均按7.3.1规定的条件进行测定。如果样液中与标准溶液相同的保留时间有峰出现，则对其进行确证。经确证被测物质色谱峰保留时间与标准物质相一致，并且在扣除背景后的样品谱图中，所选择的离子均出现，同时所选择离子的丰度于标准物质相关离子的相对丰度一致，或相似度在允许偏差之内(见表1)，被确证的样品可判定为阳性检出。

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算试样中2,6-二异丙基萘等17种农药残留量。

$$
X = \frac {A \times C \times V \times 1 0 0 0}{A _ {s} \times M \times 1 0 0 0} \dots \tag {1}
$$

式中：

$X$ ——试样中2,6-二异丙基萘等17种农药残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A$ ——样液中2,6-二异丙基萘等17种农药的峰面积(或峰高)；  
$C$ ——标准工作液中2,6-二异丙基萘等17种农药的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$   
V ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准工作液中2,6-二异丙基萘等17种农药的峰面积(或峰高)；

$M$ ——最终样液所代表的试样质量，单位为克(g)。

1000 ——换算系数。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

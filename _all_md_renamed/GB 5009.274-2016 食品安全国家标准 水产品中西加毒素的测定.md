# 水产品中西加毒素的测定

2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替SN/T3038—2011《出口海产品中西加毒素的检测 小鼠生物法》和SN/T3869—2014《出口水产品中雪卡毒素的测定》。

本标准与SN/T3038—2011和SN/T3869—2014相比，主要变化如下：

——标准名称修改为“食品安全国家标准水产品中西加毒素的测定”；

——保留小鼠生物法为第一法，增加液相色谱-串联质谱法为第二法。

# 食品安全国家标准

# 水产品中西加毒素的测定

# 1 范围

本标准规定了水产品中西加毒素的测定方法。

本标准适用于水产品可食部分中西加毒素的测定。

# 第一法 小鼠生物法

# 2 原理

用丙酮提取水产品中西加毒素，提取物经减压蒸干后，再以 $1\%$ 吐温-60生理盐水为分散介质，制备西加毒素- $1\%$ 吐温-60生理盐水混悬液，将该混悬液注射入小鼠腹腔，观察小鼠存活情况，根据西加毒素致小鼠死亡时间和鼠单位的关系表查出对应的鼠单位，并按小鼠体重校正鼠单位，计算确定西加毒素含量（鼠单位毒力）。

# 3 试剂和材料

除非另有说明，所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 吐温 $60(\mathrm{C}_{64}\mathrm{H}_{126}\mathrm{O}_{26})$ 。  
3.1.2 苦味酸 $\left[\mathrm{HO}-\mathrm{C}_{6} \mathrm{H}_{2}\left(\mathrm{NO}_{2}\right)_{3}\right]$ 。  
3.1.3 丙酮 $\mathrm{(CH_3COCH_3)}$ 。  
3.1.4 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。  
3.1.5 正己烷 $\mathrm{(C_6H_{14})}$ 。  
3.1.6 乙醇 $\mathrm{(CH_3CH_2OH)}$ 。  
3.1.7 乙醚 $(\mathrm{CH}_3\mathrm{OCH}_3)$ 。  
3.1.8 氯仿 $\mathrm{(CHCl_3)}$ 。  
3.1.9 次氯酸钠（NaClO）。  
3.1.10 生理盐水。

# 3.2 试剂配制

3.2.1 次氯酸钠溶液 $(5\%)$ ：称取次氯酸钠 $50~\mathrm{g}$ ，用 $9950~\mathrm{mL}$ 水溶解，搅拌使次氯酸钠充分溶解。  
3.2.2 甲醇溶液 $(90\%)$ ：量取 $100~\mathrm{mL}$ 甲醇，加入 $900~\mathrm{mL}$ 水，混匀。  
3.2.3 乙醇溶液 $(25\%)$ ：量取 $250~\mathrm{mL}$ 甲醇，加入 $750~\mathrm{mL}$ 水，混匀。  
3.2.4 氯仿-甲醇溶液 $(3 + 97)$ ：将氯仿与甲醇按体积比为 $3:97$ 混合均匀。  
3.2.5 $1\%$ 吐温60-生理盐水：用生理盐水将吐温60稀释为吐温60含量为 $1\%$ 的溶液。

# 3.3 标准品

太平洋西加毒素-1（P-CTX-1， $\mathrm{C}_{60}\mathrm{H}_{88}\mathrm{O}_{19}$ ）：纯度 $\geqslant 90\%$ 。

# 3.4 材料

小鼠：昆明系封闭群清洁级(CL)或无特定病原菌级(SPF)，且体重为 $18\mathrm{g}\sim 22\mathrm{g}$ 的健康雄性小鼠。

# 4 仪器和设备

4.1 电子天平：感量为 $0.001\mathrm{g}$ 。  
4.2 均质器。  
4.3 恒温水浴箱。  
4.4 旋转蒸发仪。  
4.5 氮气浓缩仪。

# 5 分析步骤

# 5.1 试样制备

水产品取其可食部分(去骨)，切成小块，制成肉糜。于 $-18^{\circ}C$ 以下保存。

# 5.2 试样提取

取试样 $200\mathrm{g}$ 置于塑料袋内，于 $70^{\circ}\mathrm{C}$ 水浴 $15\mathrm{min}$ 。取出冷却后，加丙酮 $400~\mathrm{mL}$ ，均质 $10\mathrm{min}$ ，用布氏漏斗抽滤并收集滤液。肉渣再重复萃取两次。合并滤液，于 $55^{\circ}\mathrm{C}$ 旋转蒸发浓缩至近干。加入 $100~\mathrm{mL}$ 甲醇溶液 $(90\%)$ 溶解，转移至 $500~\mathrm{mL}$ 分液漏斗中，再加入 $100~\mathrm{mL}$ 正己烷，充分振荡萃取，静置约 $30\mathrm{min}$ 以上，待分层完全，弃去上层正己烷层，收集下层甲醇溶液层。再加入 $100~\mathrm{mL}$ 正己烷重复萃取一次。将下层的甲醇溶液于 $55^{\circ}\mathrm{C}$ 旋转蒸发至干，以 $100~\mathrm{mL}$ 乙醇溶液 $(25\%)$ 溶解，并转移至分液漏斗中，加入 $100~\mathrm{mL}$ 乙醚充分振荡萃取，静置 $10\mathrm{min}$ 以上，待分层完全，收集上层乙醚层。下层溶液再用乙醚萃取两次。合并乙醚萃取液，于 $40^{\circ}\mathrm{C}$ 旋转蒸发至近干。浓缩物用 $5\mathrm{mL}$ 氯仿-甲醇溶液 $(3 + 97)$ 溶解并转移至刻度试管中，于 $45^{\circ}\mathrm{C}$ 用氮气浓缩至干，用 $1\%$ 吐温60-生理盐水定容至 $2\mathrm{mL}$ ，充分振荡，制成均匀西加毒素- $1\%$ 吐温60-生理盐水混悬液。 $0.5\mathrm{mL}$ 该混悬液相当于 $50\mathrm{g}$ 试样。以此混悬液进行小鼠实验。

注：为避免毒素的危害,应戴手套进行操作,用过的器材应在次氯酸钠溶液(5%)中浸泡 $1\mathrm{\;h}$ 以上,以使毒素分解。

# 5.3 小鼠试验

# 5.3.1 实验动物的饲养

小鼠在通风、透光、清洁的室内环境饲养，实验期间正常供给标准啮齿类动物食物和饮水。

# 5.3.2 小鼠试验

将小鼠随机分为试验组、溶剂对照组、空白对照组3组，每组3只小鼠，每只小鼠称重（精确至 $0.5\mathrm{g}$ ），并记录体重。用染色法(如 $3\% \sim 5\%$ 苦味酸溶液或其他染色剂)做好编号。

# 5.3.2.1 试验组

每只试验组小鼠腹腔注射 $0.5\mathrm{mL}$ 试样提取液。若注射时注射液溢出，须将该只小鼠弃去，并重新

注射一只小鼠。

# 5.3.2.2 溶剂对照组

每只溶剂对照组小鼠腹腔注射 $0.5\mathrm{mL}1\%$ 吐温60-生理盐水，以排除吐温60-生理盐水可能产生的影响。

# 5.3.2.3 空白对照组

每只空白对照组小鼠不进行腹腔注射，与实验组和溶剂对照组的小鼠在相同的环境和条件下饲养。

# 5.3.3 小鼠体征观察

记录注射开始和完毕时间，仔细观察并记录小鼠停止呼吸时的死亡时间（到小鼠呼出最后一口气止）。

小鼠中毒死亡症状：活动能力异常（减少至不活动）、步态异常（正常：慢走或快走，异常：兴奋的、不稳的）、腹泻、呼吸困难（急促浅的或喘息）、肤色青紫（主要在阴茎、耳、鼻、口、尾部）、竖毛（轻至显著）、颤抖（阵发性或持续）、麻痹（阵发至后腿麻痹）、多涎（可能浸湿脸部和胸前毛）、直至挣扎跳跃最后死亡。小鼠在注射 $2\mathrm{h}$ 内要仔细观察，连续观察 $400\mathrm{min}$ ，当小鼠接近死亡时更要密切观察，准确记录死亡时间（直至小鼠呼出最后一口气止）。

# 6 分析结果的表述

# 6.1 试样中西加毒素毒力的计算

观察从注射开始至 $400\mathrm{min}$ 小鼠的毒效应体征和存活情况，计算出一组3只小鼠中死亡2只以上的中位数死亡时间。根据中位数死亡时间，在表A.1中查出对应的MU。若小鼠体重小于或大于 $20\mathrm{g}$ ，则根据附录B查出重量校正系数 $C$ 。

试样中西加毒素的毒力按式(1)计算：

$$
X = \mathrm {M U} \times C \times f \quad \dots \tag {1}
$$

式中：

$X$ ——试样中西加毒素的毒力，单位为鼠单位每五十克 $(\mathrm{MU} / 50\mathrm{g})$ ；

MU—0.5mL小鼠腹腔注射液中所含的西加毒素的鼠单位数，单位为鼠单位每五十克 $(\mathrm{MU} / 50~\mathrm{g})$

$C$ ——小鼠体重校正系数；

$f$ ——试样提取液的稀释倍数。

1 MU的定义为使体重为 $20.0\mathrm{g}$ 的无特定病原体级(SPF)昆明系雄性小鼠 $400\mathrm{min}$ 死亡的毒力，相当于 $9.10\mathrm{ng}$ 西加毒素。

# 6.2 结果判断

若实验组所有小鼠死亡时间大于 $400\mathrm{min}$ ，则该报告样品中西加毒素毒力小于 $0.98\mathrm{MU} / 50\mathrm{g}$ 。

若实验组中位数死亡时间 $50\mathrm{min}\sim 400\mathrm{min}$ ，则按式(1)计算，并报告样品中西加毒素毒力为： $\times \times$ $\times \mathrm{MU} / 50\mathrm{g}$ 。

若实验组中位数死亡时间小于 $50\mathrm{min}$ ，则应对试样取液进行稀释，再选取3只小鼠进行实验，直到得到中位数死亡时间为 $50\mathrm{min} \sim 400\mathrm{min}$ 为止，根据式(1)计算，并报告样品中西加毒素毒力为： $\times \times \times$ MU/50g。

# 7 其他

方法的检出限为 $0.98\mathrm{MU} / 50\mathrm{g}$

# 第二法 液相色谱-串联质谱法

# 8 原理

试样中的西加毒素经甲醇提取， $\mathrm{C_{18}}$ 固相萃取柱净化，液相色谱-串联质谱仪测定，外标法定量。

# 9 试剂和材料

除非另有说明，所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 9.1 试剂

9.1.1 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
9.1.2 甲酸 $\left(\mathrm{C}_{2} \mathrm{H}_{4} \mathrm{O}_{2}\right)$ ：色谱纯。  
9.1.3 乙酸铵 $\mathrm{(C_3H_8CO_2N)}$ ：色谱纯。

# 9.2 试剂配制

9.2.1 乙酸铵溶液 $(10\mathrm{mmol / L})$ ：准确称取 $0.077\mathrm{g}$ 乙酸铵于烧杯中，用适量水溶解，转移至 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。  
9.2.2 甲醇-乙酸铵溶液 $(10\mathrm{mmol / L})(95 + 5)$ ：准确量取 $5\mathrm{mL}$ 乙酸铵溶液 $(10\mathrm{mmol / L})$ ，转移至 $100~\mathrm{mL}$ 容量瓶中，用甲醇容至刻度，混匀。  
9.2.3 甲醇溶液（ $50\%$ ）：将甲醇和水等体积混合。

# 9.3 标准品

西加毒素标准品 $\left[\left(\mathrm{P}-\mathrm{CTX}-1, \mathrm{C}_{60} \mathrm{H}_{88} \mathrm{O}_{19}\right), \left(\mathrm{P}-\mathrm{CTX}-2, \mathrm{C}_{60} \mathrm{H}_{88} \mathrm{O}_{18}\right), \left(\mathrm{P}-\mathrm{CTX}-3, \mathrm{C}_{60} \mathrm{H}_{88} \mathrm{O}_{18}\right)\right]$ ：纯度 $\geqslant 90 \%$ 。

# 9.4 标准溶液配制

9.4.1 标准储备液 $(1\mathrm{mg / L})$ ：分别准确称取各适量(精确至 $0.1\mathrm{mg}$ )的西加毒素(P-CTX-1，P-CTX-2，P-CTX-3)标准品，分别用甲醇配制成浓度为 $1\mathrm{mg / L}$ 标准储备液，于一 $18^{\circ}C$ 避光可保存6个月。  
9.4.2 混合标准中间液 $(0.01\mathrm{mg / L})$ ：分别吸取适量西加毒素(P-CTX-1、P-CTX-2、P-CTX-3)标准储备液 $(1\mathrm{mg / L})$ ，用甲醇稀释成西加毒素(P-CTX-1、P-CTX-2、P-CTX-3)浓度均为 $0.01\mathrm{mg / L}$ 的混合标准工作液，于一 $18^{\circ}C$ 避光可保存3个月。  
9.4.3 基质匹配的标准系列工作液：吸取适量的混合标准中间溶液 $(0.01\mathrm{mg / L})$ ，用空白溶液配制成浓度为 $0\mu \mathrm{g / L},0.1\mu \mathrm{g / L},0.25\mu \mathrm{g / L},0.5\mu \mathrm{g / L},1.0\mu \mathrm{g / L},10.0\mu \mathrm{g / L}$ 的基质匹配的标准系列工作液。现用现配。

# 9.5 材料

9.5.1 $\mathrm{C_{18}}$ 固相萃取柱： $200\mathrm{mg / 3mL}$ 。使用前依次用 $6\mathrm{mL}$ 甲醇和 $6\mathrm{mL}$ 水活化，保持柱体湿润。  
9.5.2 微孔滤膜： $0.22\mu \mathrm{m}$ ，有机相型。

# 10 仪器和设备

10.1 液相色谱-串联质谱仪：配有电喷雾离子源(ESI)。  
10.2 分析天平：感量 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$   
10.3 氮气浓缩仪。  
10.4 冻干机。   
10.5 粉碎机。  
10.6 离心机：转速 $\geq 5000\mathrm{r / min}$   
10.7 均质器： $3400\mathrm{r / min}\sim 24000\mathrm{r / min}$   
10.8 涡旋振荡器。  
10.9 超声波发生器。

# 11 分析步骤

# 11.1 试样制备

水产品取其可食部分，切成小块，制成肉糜。于 $-18^{\circ}C$ 储存。

# 11.2 试样提取

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 试样，经冻干后充分碾碎置于 $50~\mathrm{mL}$ 具塞离心管中，加入 $10~\mathrm{mL}$ 甲醇和 $5\mathrm{mL}$ 正己烷，以 $5000\mathrm{r / min}$ 均质提取 $1\mathrm{min}$ ，涡旋混匀后超声提取 $15\mathrm{min}$ ，于 $5000\mathrm{r / min}$ 离心 $3\mathrm{min}$ ，弃去上层正己烷相，将甲醇转移至试管中。残渣再用 $10~\mathrm{mL}$ 甲醇重复提取一次，合并提取液，于 $45^{\circ}\mathrm{C}$ 以氮气浓缩至约 $5\mathrm{mL}$ ，加 $5\mathrm{mL}$ 水，混匀后待净化。

# 11.3 试样净化

将上述待净化液转入活化过的 $\mathrm{C_{18}}$ 固相萃取柱中，以约 $1\mathrm{mL} / \mathrm{min}$ 的流速使净化液全部通过，用 $6\mathrm{mL}$ 甲醇溶液 $(50\%)$ 淋洗并抽干。用 $4\mathrm{mL}$ 甲醇洗脱，收集全部流出液于 $20~\mathrm{mL}$ 离心管中，于 $40^{\circ}\mathrm{C}$ 以氮气浓缩至干。残渣用甲醇-乙酸铵溶液( $10\mathrm{mmol / L})$ 定容至 $1\mathrm{mL}$ ，旋涡混合后过微孔滤膜，供液相色谱-串联质谱测定。

# 11.5 标准曲线的制作

将基质匹配的混合标准系列工作液注入液相色谱-串联质谱仪测定，得到西加毒素(P-CTX-1，P-CTX-2，P-CTX-3)的峰面积。以基质匹配的混合标准工作溶液浓度为横坐标，西加毒素(P-CTX-1，P-CTX-2，P-CTX-3)的峰面积为纵坐标，绘制标准曲线。西加毒素标准的多反应监测色谱图参见附录C。

# 11.6 测定

将空白溶液和试样溶液注入液相色谱-串联质谱仪测定，得到待测物的峰面积。由标准曲线得到试样溶液中西加毒素(P-CTX-1,P-CTX-2,P-CTX-3)的浓度。

# 11.7 空白试验

# 12 分析结果的表述

试样中西加毒素(P-CTX-1，P-CTX-2，P-CTX-3)的含量按式(2)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V}{m \times 1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中西加毒素(P-CTX-1,P-CTX-2,P-CTX-3)的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——从标准工作曲线得到的试样溶液中西加毒素(P-CTX-1,P-CTX-2,P-CTX-3)的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$\rho_0$ ——从标准工作曲线得到的空白溶液中的西加毒素(P-CTX-1，P-CTX-2，P-CTX-3)的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$V$ ——试样溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

1000——换算系数。

结果保留两位有效数字。

# 13 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $30\%$ 。

# 14 其他

本方法西加毒素(P-CTX-1,P-CTX-2,P-CTX-3)的检出限均为 $0.02\mu \mathrm{g / kg}$ ，定量限均为 $0.05\mu \mathrm{g / kg}$ 。

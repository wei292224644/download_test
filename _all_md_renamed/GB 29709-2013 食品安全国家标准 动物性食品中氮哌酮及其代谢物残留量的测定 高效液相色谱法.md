# 动物性食品中氮哌酮及代谢物残留量的测定

# 高效液相色谱法

# 1 范围

本标准规定了动物性食品中氮哌酮及代谢物氮哌醇残留量检测的制样和高效液相色谱测定方法。

本标准适用于猪的肌肉、皮+脂、肝脏和肾脏组织中氮哌酮及代谢物氮哌醇残留量的检测。

# 3原理

试料中残留的氮哌酮及代谢物，用乙腈萃取，正己烷除脂，MCX 柱净化，高效液相色谱测定，外标法定量。

# 4 试剂和材料

以下所用试剂，除特别注明者外均为分析纯；水为符合GB/T6682规定的一级水。

4.1 氮哌酮、氮哌醇标准品：含量 $\geq 98\%$   
4.2 乙腈：色谱纯。  
4.3 正己烷  
4.4 浓氨水  
4.5 甲醇  
4.6 乙酸铵  
4.7 甲酸  
4.8 MCX固相萃取柱： $60\mathrm{mg / 3mL}$ ，或相当者。  
4.9 $2\%$ 的甲酸水溶液：取甲酸 $2\mathrm{mL}$ ，用水溶解并稀释至 $100~\mathrm{mL}$ 。  
4.10 $5\%$ 的氨化甲醇：取氨水 $5 \mathrm{~mL}$ ，用甲醇溶解并稀释至 $100 \mathrm{~mL}$ 。  
4.11 5 mmol/L 乙酸铵溶液：取乙酸铵 $0.385 \mathrm{~g}$ ，用水溶解并稀释至 $1000 \mathrm{~mL}$ ，滤膜过滤。

4.12 $1 \mathrm{mg} / \mathrm{mL}$ 标准贮备液：精密称取氮哌酮和氮哌醇标准品各 $10 \mathrm{mg}$ ，分别于 $10 \mathrm{~mL}$ 量瓶中，用甲醇溶解并稀释至刻度，配成浓度为 $1 \mathrm{mg} / \mathrm{mL}$ 的标准贮备液。 $-20^{\circ} \mathrm{C}$ 以下避光保存，有效期3个月。  
4.13 $1 \mu \mathrm{g} / \mathrm{mL}$ 混合标准工作液: 分别精密量取 $1 \mathrm{mg} / \mathrm{mL}$ 氮哌酮标准贮备液和氮哌醇标准贮备液各 $0.1 \mathrm{~mL}$ , 于 $100 \mathrm{~mL}$ 量瓶中, 用甲醇稀释至刻度, 配制成浓度为 $1 \mu \mathrm{g} / \mathrm{mL}$ 的混合标准工作液。 $-20^{\circ} \mathrm{C}$ 以下避光保存, 有效期 3 个月。

# 5 仪器和设备

5.1 高效液相色谱仪：配紫外检测器。  
5.2 分析天平：感量 $0.00001\mathrm{g}$ 。  
5.3 天平：感量 $0.01\mathrm{g}$ 。  
5.4 冷冻高速离心机  
5.5 氮气吹干装置  
5.6 旋涡混合器  
5.7 固相萃取装置  
5.8 超声波清洗机  
5.9 均质机  
5.10 离心管  
5.11 滤膜： $0.22\mu \mathrm{m}$

# 6 试料的制备与保存

# 6.1 试料的制备

取新鲜或冷冻空白组织、供试组织，去除筋膜，绞碎，并使均质。

——取均质后的供试样品，作为供试试料。  
——取均质后的空白样品，作为空白试料。  
——取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 6.2 样品的保存

-20°C以下保存。

# 7 测定步骤

# 7.1 提取

称取试料（ $5 \pm 0.05$ ）g，于离心管中，加乙腈 $10 \mathrm{~mL}$ ，旋涡混合 $1 \mathrm{~min}$ ，超声 $2 \mathrm{~min}$ ， $5000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ ，取上清液，残渣中加乙腈 $10 \mathrm{~mL}$ 重复提取一次，合并两次上清液。于 $40^{\circ} \mathrm{C}$ 水浴旋转蒸干。

用 $2 \%$ 甲酸水溶液 $5 \mathrm{~mL}$ 溶解残余物, 加正己烷 $4 \mathrm{~mL}$ , 混匀, $5000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 弃正己烷层液, 备用。

# 7.2 净化

MCX 柱依次用甲醇 $3 \mathrm{~mL}$ 、水 $3 \mathrm{~mL}$ 和 $2 \%$ 甲酸水溶液 $3 \mathrm{~mL}$ 活化，取备用液过柱，控制流速小于 $1 \mathrm{~mL} / \mathrm{min}$ ，用 $2 \%$ 甲酸水 $3 \mathrm{~mL}$ 和甲醇 $3 \mathrm{~mL}$ 淋洗，抽干，用 $5 \%$ 氨化甲醇 $4 \mathrm{~mL}$ 洗脱。收集洗脱液，于 $40^{\circ} \mathrm{C}$ 氮气吹干，用流动相 $1.0 \mathrm{~mL}$ 溶解残余物，旋涡混匀，供高效液相色谱法测定。

# 7.3 标准曲线的制备

精密量取 $1\mu \mathrm{g} / \mathrm{mL}$ 混合标准工作液适量，用甲醇稀释，配制成浓度为10、20、50、100、200、500和 $1000\mu \mathrm{g} / \mathrm{L}$ 的系列混合标准溶液。每个浓度重复3次，供高效液相色谱测定，以测得峰面积平均值为纵坐标，对应的标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 7.4 测定

# 7.4.1 色谱条件

色谱柱： $C_{18}$ （ $250\mathrm{mm}\times 4.6\mathrm{mm}$ ，粒径 $5\mu \mathrm{m})$ ，或相当者；

流动相：乙腈 $+5\mathrm{mmol / L}$ 乙酸铵溶液（ $50 + 50$ ，v/v）；

流速： $1\mathrm{mL / min}$

紫外检测波长： $250\mathrm{nm}$

柱温： $40^{\circ}\mathrm{C}$

进样量： $50~\mu \mathrm{L}$

# 7.4.2 测定法

取试样溶液和相应的标准溶液，作单点或多点校准，按外标法，以峰面积计算。标准溶液及试样溶液中氮哌酮和氮哌醇响应值应在仪器检测的线性范围之内。在上述色谱条件下，标准溶液和空白添加试样溶液的高效液相色谱图见附录A。

# 7.5 空白试验

除不加试料外，采用完全相同的步骤进行平行操作。

# 8 结果计算与表述

试料中氮哌酮和氮哌醇残留量（ $\mu \mathrm{g} / \mathrm{kg}$ ）：按下式计算

单点校准： $C = \frac{CsA}{As}$

或标准曲线校准：由 $As = aCs + b$

求得a和b，则 $C = \frac{A - b}{a}$

$$
X = \frac {C V}{m}
$$

式中：

$X$ ——供试试料中相应的氮哌酮及代谢物残留量, $\mu \mathrm{g} / \mathrm{kg}$ ;

A—试样中相应的氮哌酮及代谢物峰的面积；

$A_{S}$ ——标准工作液中相应的氮哌酮及代谢物峰的面积；

$\mathrm{C}_{\mathrm{S}}$ ——标准工作液中相应的氮哌酮及代谢物的浓度, $\mu \mathrm{g} / \mathrm{L}$ ;

C—供试试料溶液中相应的氮哌酮及代谢物的浓度, $\mu \mathrm{g} / \mathrm{L}$ ;

$V$ ——溶解残余物所用流动相体积, $\mathrm{mL}$ ;

$m$ —供试试料质量，g。

注：计算结果需扣除空白值，测定结果用平行测定的算术平均值表示，保留三位有效数字。

# 9 检测方法灵敏度、准确度、精密度

# 9.1 灵敏度

本方法的检测限为 $2\mu \mathrm{g / kg}$ ，定量限为 $5\mu \mathrm{g / kg}$

# 9.2 准确度

本方法在肌肉、皮+脂组织 $5 \sim 60 \mu \mathrm{g} / \mathrm{kg}$ 、在肝脏和肾脏组织 $5 \sim 100 \mu \mathrm{g} / \mathrm{kg}$ 添加浓度水平上的回收率为 $60\% \sim 110\%$ 。

# 9.3 精密度

本方法批内相对标准偏差 $\leq 20\%$ 、批间相对标准偏差 $\leq 20\%$ 。

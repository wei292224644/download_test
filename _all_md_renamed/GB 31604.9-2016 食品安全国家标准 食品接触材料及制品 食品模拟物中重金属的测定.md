# 前言

本标准代替GB/T5009.60—2003《食品包装用聚乙烯、聚苯乙烯、聚丙烯成型品卫生标准的分析方法》、GB/T5009.64—2003《食品用橡胶垫片(圈)卫生标准的分析方法》、GB/T5009.65—2003《食品用高压锅密封圈卫生标准的分析方法》、GB/T5009.66—2003《橡胶奶嘴卫生标准的分析方法》、GB/T5009.67—2003《食品包装用聚氯乙烯成型品卫生标准的分析方法》、GB/T5009.68—2003《食品容器内壁过氯乙烯涂料卫生标准的分析方法》、GB/T5009.70—2003《食品容器内壁聚酰胺环氧树脂涂料卫生标准的分析方法》、GB/T5009.79—2003《食品用橡胶管卫生检验方法》、GB/T5009.98—2003《食品容器及包装材料用不饱和聚酯树脂及其玻璃钢制品卫生标准分析方法》、GB/T5009.99—2003《食品容器及包装材料用聚碳酸酯树脂卫生标准的分析方法》、GB/T5009.100—2003《食品包装用发泡聚苯乙烯成型品卫生标准的分析方法》中的重金属测定方法。

本标准与GB/T5009.60—2003等被代替的标准相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 食品模拟物中重金属的测定”；  
——修改了适用范围的表述；  
——修改了原理的表述；  
——删除了原各标准中的取样方法；  
——删除原各标准中的浸泡条件；  
——修改了分析结果的表述。

# 食品安全国家标准

# 食品接触材料及制品食品模拟物中重金属的测定

# 1 范围

本标准规定了食品模拟物（4%乙酸）中重金属的测定方法。

本标准第一法适用于食品接触材料及制品在食品模拟物(4%乙酸)中重金属的测定。

本标准第二法适用于食品接触橡胶制品在食品模拟物(4%乙酸)中重金属的测定。

# 第一法 直接比色法

# 2 原理

经迁移试验所得的食品模拟物试液中重金属(以铅计)与硫化钠作用，在酸性溶液中形成黄棕色硫化物，与铅标准溶液的呈色相比较。

# 3 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 3.1 试剂

3.1.1 九水合硫化钠 $(\mathrm{Na}_2\mathrm{S}\cdot 9\mathrm{H}_2\mathrm{O})$   
3.1.2 甘油 $\mathrm{(C_3H_8O_3)}$ 。  
3.1.3 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.4 硝酸 $(\mathrm{HNO}_3, 60\%)$ ，质量分数）。

# 3.2 试剂配制

3.2.1 硝酸溶液（ $10\%$ ，质量分数）：移取 $16.7 \mathrm{~mL}$ 硝酸(3.1.4)加水定容至 $100 \mathrm{~mL}$ 。

3.2.2 硫化钠溶液：称取 $15.4\mathrm{g}$ 九水合硫化钠(3.1.1)，加入 $10~\mathrm{mL}$ 水，用玻璃棒充分搅拌直至溶解，然后加入 $30~\mathrm{mL}$ 甘油(3.1.2)，混匀，密闭保存。本溶液常温下可稳定保存一个月。

3.2.3 食品模拟物： $4\%$ 乙酸溶液（体积分数）。量取 $40~\mathrm{mL}$ 冰乙酸(3.1.3)于 $1\mathrm{L}$ 容量瓶中，加水定容至刻度。

# 3.3 标准品

硝酸铅 $\left[\mathrm{Pb}\left(\mathrm{NO}_{3}\right)_{2}, > 98\%\right]$ 或经国家认证并授予标准物质证书的铅标准溶液（每毫升含 $1\mathrm{mg}$ 铅）。

# 3.4 标准溶液配制

3.4.1 铅标准储备液：准确称取 $0.1598\mathrm{g}$ 硝酸铅(3.3，精确至 $0.0001\mathrm{g}$ )，溶于 $10~\mathrm{mL}$ 硝酸溶液

(3.2.1)中，移入 $1000\mathrm{mL}$ 容量瓶内，加水稀释至刻度。 $1\mathrm{mL}$ 此溶液相当于 $100\mu \mathrm{g}$ 铅。

3.4.2 铅标准使用液：吸取 $10.0\mathrm{mL}$ 铅标准储备液(3.4.1)，置于 $100\mathrm{mL}$ 容量瓶中，加水稀释至刻度，摇匀。或移取 $1.0\mathrm{mL}$ 经国家认证并授予标准物质证书的铅标准溶液(3.3)，置于 $100\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。

$1\mathrm{mL}$ 此溶液相当于 $10~\mu \mathrm{g}$ 铅。

# 4 仪器设备

4.1 恒温烘箱。  
4.2 容量瓶： $100~\mathrm{mL},1000~\mathrm{mL}$   
4.3 移液管： $10\mathrm{mL},20\mathrm{mL}$   
4.4 比色管： $50~\mathrm{mL}$   
4.5 量筒： $50~\mathrm{mL}$   
4.6 分析天平：感量 $0.0001\mathrm{g}, 0.1\mathrm{g}$ 。

# 5 分析步骤

# 5.1 试样制备

按照GB5009.156《食品安全国家标准 食品接触材料及制品迁移试验预处理方法通则》及相应产品的国家标准要求，以 $4\%$ 乙酸溶液(3.2.3)作为食品模拟物对样品进行迁移试验，得到食品模拟物试液。在 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱中避光保存。

# 5.2 试样溶液的测定

吸取 $20~\mathrm{mL}$ 经迁移试验后所得食品模拟物试液(5.1)于 $50~\mathrm{mL}$ 比色管中，加水至刻度；另取 $2\mathrm{mL}$ 铅标准使用液(3.4.2)于 $50~\mathrm{mL}$ 比色管中，加 $20~\mathrm{mL}4\%$ 乙酸溶液(3.2.3)，加水至刻度混匀。分别于两个溶液中加2滴硫化钠溶液(3.2.2)，混匀后，放置 $5\mathrm{min}$ ，以白色为背景，从上方或侧面观察并比较模拟物试液及标准溶液的呈色。

# 6 分析结果的表述

当试样呈色深于标准溶液呈色时，食品接触材料及其制品中重金属迁移量（以铅计） $>1\mathrm{mg / L}$ 当试样呈色浅于标准溶液呈色时，食品接触材料及其制品中重金属迁移量(以铅计） $<  1~\mathrm{mg / L}$ 。

# 第二法 掩蔽干扰比色法

# 7 原理

经迁移试验所得的食品模拟物试液中重金属(以铅计)与硫化钠作用，在酸性溶液中形成黄棕色硫化物，与铅标准溶液呈色相比较。

# 8 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

# 8.1 试剂

8.1.1 九水合硫化钠 $(\mathrm{Na}_2\mathrm{S}\cdot 9\mathrm{H}_2\mathrm{O})$   
8.1.2 柠檬酸铵 $\mathrm{(C_6H_{17}N_3O_7)}$ 。  
8.1.3 氰化钾(KCN)。  
8.1.4 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。  
8.1.5 甘油 $\mathrm{(C_3H_8O_3)}$ 。  
8.1.6 冰乙酸 $\mathrm{(CH_3COOH)}$ 。  
8.1.7 硝酸 $(\mathrm{HNO}_3,60\%)$ ，质量分数）。

# 8.2 试剂配制

8.2.1 氰化钾溶液 $(100\mathrm{g / L})$ ：称取 $10\mathrm{g}$ （精确至 $0.1\mathrm{g}$ )氰化钾(8.1.3)，加水溶解，定容至 $100~\mathrm{mL}$ 。  
8.2.2 柠檬酸铵溶液 $(500\mathrm{g / L})$ ：称取 $50~\mathrm{g}$ （精确至 $0.1\mathrm{g}$ )柠檬酸铵(8.1.2)，加水溶解，定容至 $100~\mathrm{mL}$ 。  
8.2.3 硝酸溶液（ $10\%$ ，质量分数）：移取 $16.7 \mathrm{~mL}$ 硝酸(8.1.7)加水定容至 $100 \mathrm{~mL}$ 。  
8.2.4 硫化钠溶液：称取 $15.4\mathrm{g}$ 九水合硫化钠(8.1.1)，加入 $10~\mathrm{mL}$ 水，用玻璃棒充分搅拌直至溶解，然后加入 $30~\mathrm{mL}$ 甘油(8.1.5)，混匀，密闭保存。本溶液常温下可稳定保存一个月。  
8.2.5 食品模拟物： $4\%$ 乙酸溶液（体积分数）。量取 $40~\mathrm{mL}$ 冰乙酸(8.1.6)于 $1\mathrm{L}$ 容量瓶中，加水定容至刻度。

# 8.3 标准品

硝酸铅 $\left[\mathrm{Pb}\left(\mathrm{NO}_{3}\right)_{2}, > 98\%\right]$ 或经国家认证并授予标准证书的铅标准溶液（每毫升含 $1\mathrm{mg}$ 铅）。

# 8.4 标准溶液配制

8.4.1 铅标准储备液：准确称取 $0.1598\mathrm{g}$ 硝酸铅(8.3.1，精确至 $0.0001\mathrm{g}$ )，溶于 $10~\mathrm{mL}$ 硝酸溶液(8.2.3)中，移入 $1000~\mathrm{mL}$ 容量瓶内，加水稀释至刻度。此溶液每毫升相当于 $100\mu \mathrm{g}$ 铅。  
8.4.2 铅标准使用液：吸取 $10.0\mathrm{mL}$ 铅标准储备液(8.4.1)，置于 $100~\mathrm{mL}$ 容量瓶中，加水稀释至刻度，摇匀。或移取 $1.0\mathrm{mL}$ 经国家认证并授予标准证书的铅标准溶液(8.3.1)，置于 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，摇匀。

$1\mathrm{mL}$ 此溶液相当于 $10~\mu \mathrm{g}$ 铅。

# 9 仪器设备

9.1 恒温烘箱。  
9.2 容量瓶： $100~\mathrm{mL},1000~\mathrm{mL}$   
9.3 移液管： $10\mathrm{mL}, 20\mathrm{mL}$ 。  
9.4 比色管： $50~\mathrm{mL}$   
9.5 量筒： $50~\mathrm{mL}$   
9.6 分析天平：感量 $0.0001\mathrm{g}, 0.1\mathrm{g}$ 。

# 10 分析步骤

# 10.1 试样制备

以 $4\%$ 乙酸溶液(8.2.5)作为食品模拟物对样品进行迁移试验，得到食品模拟物试液。在 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$

冰箱中避光保存。

# 10.2 试样溶液的测定

吸取 $20~\mathrm{mL}$ 经迁移试验后所得食品模拟物试液(10.1)于 $50~\mathrm{mL}$ 比色管中；另取 $2\mathrm{mL}$ 铅标准使用液(8.4.2)于 $50~\mathrm{mL}$ 比色管中，加 $20~\mathrm{mL}4\%$ 乙酸溶液(8.2.5)。分别于两个比色管中加入 $1\mathrm{mL}$ 柠檬酸铵溶液(8.2.2)、 $3\mathrm{mL}$ 氨化钾溶液(8.2.1)，加水至刻度混匀，再各加硫化钠溶液(8.2.4)2滴，混匀后，放置 $5\mathrm{min}$ ，以白色为背景，从上方或侧面观察并比较模拟物试液及标准溶液的呈色。

# 11 分析结果的表述

当试样呈色深于标准溶液呈色时，食品接触材料及制品中重金属迁移量(以铅计) $>1\mathrm{mg / L}$ ；当试样呈色浅于标准溶液呈色时，食品接触材料及制品中重金属迁移量(以铅计) $< 1\mathrm{mg / L}$ 。

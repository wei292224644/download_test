# 食品安全国家标准

# 食品添加剂 姜黄素

# 1 范围

本标准适用于以姜科植物姜黄(Curcuma longa L.)的根茎为原料，经有机溶剂提取，再经物理方法精制而得的食品添加剂姜黄素。

# 2 化学名称、分子式、结构式和相对分子质量

# 2.1 化学名称

姜黄素：1，7-双(4-羟基-3-甲氧基苯基)-1，6-二烯-3，5-庚二酮

脱甲氧基姜黄素：1-(4-羟基苯基)-7-(4-羟基-3-甲氧基苯基)-1,6-二烯-3,5-庚二酮

双脱甲氧基姜黄素：1，7-双(4-羟基苯基)-1，6-二烯-3，5-庚二酮

# 2.2 分子式

姜黄素： $\mathrm{C_{21}H_{20}O_6}$

脱甲氧基姜黄素： $\mathrm{C_{20}H_{18}O_5}$

双脱甲氧基姜黄素： $\mathrm{C_{19}H_{16}O_4}$

# 2.3 结构式

![](images/5015fe0965accc28088d872bcb7f59ffea032c9d6998433ce2c376015432ba3b.jpg)

姜黄素： $\mathrm{R}_1 = \mathrm{R}_2 = \mathrm{OCH}_3$

脱甲氧基姜黄素： $\mathrm{R_1 = OCH_3}$ ， $\mathrm{R}_2 = \mathrm{H}$

双脱甲氧基姜黄素： $\mathrm{R}_1 = \mathrm{R}_2 = \mathrm{H}$

# 2.4 相对分子质量

姜黄素：368.39（按2011年国际相对原子质量）

脱甲氧基姜黄素：338.39（按2011年国际相对原子质量）

双脱甲氧基姜黄素：308.39（按2011年国际相对原子质量）

# 附录A

# 检验方法

# A.1 一般规定

本标准所用试剂和水在没有注明其他要求时，均指分析纯试剂和GB/T6682中规定的三级水。试验中所用杂质测定用标准溶液、试剂溶液及样品溶液，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 试剂和材料

A.2.1.1 乙醚。

A.2.1.2 冰乙酸。

A.2.1.3 氢氧化钠溶液： $0.05\mathrm{mol} / \mathrm{L}$

A.2.1.4 盐酸： $1\mathrm{mol} / \mathrm{L}$

A.2.1.5 乙醇： $95\%$

A.2.1.6 硫酸。

A.2.1.7 硅胶薄层板：称取硅胶 $10\mathrm{g}$ ，加水 $20\mathrm{mL}$ ，置于研钵中研磨成稀稠适宜的浆状液，均匀涂布于玻璃板上，涂层厚为 $0.5\mathrm{mm}$ 。 $100^{\circ}\mathrm{C}$ 活化 $30\mathrm{min}$ 。

A.2.1.8 展开剂：三氯甲烷：甲醇 $= 99:1$

# A.2.2 鉴别方法

# A.2.2.1 溶解性

不溶于水和乙醚，易溶于乙醇和冰乙酸。

# A.2.2.2 颜色反应

A.2.2.2.1 称取 $0.1 \mathrm{~g}$ 试样, 溶于 $5 \mathrm{~mL}$ 氢氧化钠溶液中, 此时呈玫瑰红色, 滴加盐酸到酸性后, 溶液即由玫瑰红变为亮黄色。

A.2.2.2.2 称取 $10 \mathrm{mg}$ 试样, 溶于 $5 \mathrm{~mL}$ 乙醇溶液中, 颜色为纯黄色微带绿色荧光, 再加少量硫酸就变为玫瑰红色。

# A.2.2.3 薄层色谱法

A.2.2.3.1 试样溶液的制备：称取试样 $0.1\mathrm{g}$ ，溶于 $10~\mathrm{mL}$ 乙醇溶液中。使用时取上清液。

A.2.2.3.2 点样：在已活化的硅胶板上，用微量注射器吸取 $2\mu \mathrm{L}$ 试样溶液点样。间距为 $1.5\mathrm{cm}$ 。

A.2.2.3.3 展开与观察: 将点好样的薄板放入预先已用展开剂饱和的层析缸中上行法展开 (展开时室温为 $17^{\circ} \mathrm{C} \sim 20^{\circ} \mathrm{C}$ )。待展开剂前沿达到 $15 \mathrm{~cm}$ 时, 取出, 风干。观察应有三个黄色斑点, $R_{\mathrm{f}}$ 值分别为 $0.11 \sim 0.12, 0.23 \sim 0.25, 0.41 \sim 0.43$ 。

# A.3 总姜黄素含量的测定

# A.3.1 方法提要

根据姜黄素乙醇溶液在特定波长下的吸光强度计算总姜黄素含量。

# A.3.2 试剂和材料

$95\%$ 乙醇。

# A.3.3 仪器和设备

分光光度计。

# A.3.4 分析步骤

取约 $0.1\mathrm{g}$ 试样（精确到 $0.0002\mathrm{g}$ ），置于 $100\mathrm{mL}$ 容量瓶，加入 $95\%$ 乙醇，振荡溶解摇匀。用 $95\%$ 乙醇定容混合。移取 $1.0\mathrm{mL}$ 溶液到 $250\mathrm{mL}$ 容量瓶，用 $95\%$ 乙醇定容。用 $1\mathrm{cm}$ 比色皿，测定 $425\mathrm{nm}$ 下的吸光度。

# A.3.5 结果计算

总姜黄素含量的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {A \times 2 5 0 \times 1 0 0}{m \times 1 6 0 7} \quad \dots \tag {A.1}
$$

式中：

$A$ ——试样的吸光度；

250 体积换算系数；

100 ——容量换算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ ；

1607—425 nm处，姜黄素标品乙醇溶液的特定吸收系数。

# A.4 溶剂残留（正己烷、异丙醇和乙酸乙酯）的测定

# A.4.1 原理

将试样放入密封的平衡瓶中，在一定的温度下，使残留溶剂气化达到平衡时，取上层气体注入气相色谱仪中测定，与标准曲线比较定量。

# A.4.2 试剂和材料

注：除另外说明外，所有试剂均为色谱纯。

A.4.2.1 $N,N-$ 二甲基甲酰胺（色谱纯）。

A.4.2.2 标准储备液：称取异丙醇(色谱纯)、正己烷(色谱纯)、乙酸乙酯(色谱纯)各 $0.05\mathrm{g}$ （精确至 $0.0001\mathrm{g})$ ，溶于 $N,N-$ 二甲基甲酰胺中，并用 $N,N-$ 二甲基甲酰胺稀释至 $50~\mathrm{mL}$ 容量瓶中，定容。此溶液 $1\mathrm{mL}$ 相当于 $1.0\mathrm{mg}$ 异丙醇、正己烷、乙酸乙酯。放置于 $4^{\circ}C$ 冰箱保存。

# A.4.3 仪器和设备

A.4.3.1 气相色谱仪：具氢火焰离子化检测器(FID)。

A.4.3.2 分析天平，感量为 $0.1\mathrm{mg}$   
A.4.3.3 容量瓶： $50~\mathrm{mL}$   
A.4.3.4 顶空瓶： $100~\mathrm{mL}$ 小口输液瓶。  
A.4.3.5 封口钳。  
A.4.3.6 气密针： $100~\mu \mathrm{L}$   
A.4.3.7 移液器： $500\mu \mathrm{L},5\mathrm{mL}$   
A.4.3.8 顶空自动进样器（自动顶空方法）。  
A.4.3.9 顶空瓶： $20~\mathrm{mL}$ 顶空样品瓶（自动顶空方法）。

# A.4.4 分析步骤

# A.4.4.1 手动顶空方法

# A.4.4.1.1 标准曲线绘制

取顶空瓶(A.4.3.4)，以微量移液器沿瓶壁注入 $500\mu \mathrm{L}$ 标准储备液，迅速塞上橡胶反口塞，铝盖封口后，在 $80^{\circ}\mathrm{C}\pm 2^{\circ}\mathrm{C}$ 烘箱中，加热 $2\mathrm{h}$ ，取出冷却至室温，用 $100\mu \mathrm{L}$ 微量进样器，按 $20~\mu \mathrm{L},40~\mu \mathrm{L},60~\mu \mathrm{L}$ $80~\mu \mathrm{L}$ 和 $100~\mu \mathrm{L}$ 取顶空气分别注入色谱仪，以其色谱峰高(峰面积)为纵轴，以对应的样品溶剂含量为横轴，制作标准曲线，或进行线性回归得到标准曲线方程。

# A.4.4.1.2 气相色谱参考条件

A.4.4.1.2.1 色谱柱： $6 \%$ 氰丙基/苯基和 $94 \%$ 二甲基聚硅氧烷弹性石英毛细管柱 $60.0 \mathrm{~m} \times 0.32 \mathrm{~mm}$ （内径） $\times 1.8 \mu \mathrm{m}$ ，或相当者。

A.4.4.1.2.2 柱温： $40^{\circ}\mathrm{C}$ 保持 $3\mathrm{min}$ ，以 $2^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升温至 $70^{\circ}\mathrm{C}$ ，再以 $20^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升温至 $180^{\circ}\mathrm{C}$ 保持 $4\mathrm{min}$ ，再以 $30^{\circ}\mathrm{C} / \mathrm{min}$ 的速率升温至 $280^{\circ}\mathrm{C}$ 。

A.4.4.1.2.3 汽化室温度： $200^{\circ}C$ 。  
A.4.4.1.2.4 检测器温度： $300^{\circ}C$ 。  
A.4.4.1.2.5 载气：氮气，纯度 $\geq 99.99\%$   
A.4.4.1.2.6 燃气：氢气，纯度 $\geq 99.99\%$   
A.4.4.1.2.7 载气流速： $2.0~\mathrm{mL / min}$   
A.4.4.1.2.8 氢气流速： $40~\mathrm{mL / min}$   
A.4.4.1.2.9 空气流速： $450~\mathrm{mL / min}$   
A.4.4.1.2.10 尾吹气流量： $20.0\mathrm{mL / min}$   
A.4.4.1.2.11 进样方式：分流进样，分流比 $1:2$   
A.4.4.1.2.12 运行时间： $28.17\mathrm{min}$

# A.4.4.1.3 测定

取顶空瓶(A.4.3.4)，称取 $2\mathrm{g}$ （准确至 $0.0001\mathrm{g}$ )样品，置于瓶底，加 $5\mathrm{mL}N,N-$ 二甲基甲酰胺，立即塞好橡胶反口塞，铝盖封口，然后在 $80~^\circ \mathrm{C}\pm 2~^\circ \mathrm{C}$ 烘箱内加热 $2\textrm{h}$ ，样品瓶经加热气化后，冷却至室温，用微量进样器取 $100~\mu \mathrm{L}$ 顶空气，进行色谱分析，由所得样品色谱峰高(峰面积)，根据标准曲线上查得溶剂含量，计算出样品中各溶剂残留量。

# A.4.4.2 自动顶空方法

# A.4.4.2.1 标准曲线绘制

吸取标准储备液 $0.25\mathrm{mL},0.5\mathrm{mL},1.0\mathrm{mL},2.5\mathrm{mL}$ 和 $5.0~\mathrm{mL}$ 于顶空瓶中分别用 $N,N-$ 二甲基甲

酰胺定容至 $5.0 \mathrm{~mL}$ , 用含聚四氟乙烯涂层垫片的铝盖封口。放入自动顶空仪中, 将平衡好后的样品, 自动导入气相色谱仪中进行分析。以其色谱峰高(峰面积)为纵轴, 以对应的产品溶剂含量为横轴, 制作标准曲线, 或进行线性回归得到标准曲线方程。

# A.4.4.2.2 气相色谱仪参考条件

同A.4.4.1.2。

# A.4.4.2.3 顶空自动进样器参考条件

A.4.4.2.3.1 平衡温度： $80^{\circ}C$   
A.4.4.2.3.2 平衡时间： $40\mathrm{min}$   
A.4.4.2.3.3 定量环温度： $100\%$   
A.4.4.2.3.4 定量环体积： $0.5\mathrm{mL}$   
A.4.4.2.3.5 输线温度： $120^{\circ}C$

# A.4.4.2.4 测定

称取样品 $0.5\mathrm{g}$ (准确至 $0.0001\mathrm{g}$ ) 于顶空瓶(A.4.3.9)中加 $5\mathrm{mL}N,N-$ 二甲基甲酰胺，用含聚四氟乙烯涂层垫片的铝盖封口。放入自动顶空仪中，将平衡好后的样品，自动导入气相色谱仪中进行分析。溶剂残留色谱图见附录B中图B.1。

# A.4.5 结果计算

各溶剂的质量分数 $\omega_{i}$ ，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按式(A.2)计算：

$$
w _ {i} = \frac {m _ {i}}{m} \quad \dots \tag {A.2}
$$

式中：

$m_{i}$ ——从标准曲线中得出的每种溶剂的质量，单位为微克 $(\mu \mathrm{g})$ ；

$m$ ——试样的质量，单位为克（g）。

计算结果保留三位有效数字。在重复条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

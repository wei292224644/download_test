# 食品安全国家标准

# 食品中纳他霉素的测定

2022-06-30 发布

2022-12-30 实施

# 前言

本标准代替GB/T21915—2008《食品中纳他霉素的测定 液相色谱法》。

本标准与GB/T21915—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中纳他霉素的测定”；  
——修改了标准的适用范围；  
——增加了试样预处理方法；  
——增加了油炸肉类、蛋黄酱、沙拉酱和发酵酒等样品处理方法。

# 食品安全国家标准

# 食品中纳他霉素的测定

# 1 范围

本标准规定了食品中纳他霉素的液相色谱测定方法。

本标准适用于干酪和再制干酪及其类似品、酱卤肉制品、肉类(熏、烧、烤)、油炸肉类、西式火腿类、肉灌肠类、发酵肉制品类、蛋黄酱、沙拉酱、糕点、果蔬汁(浆)、果汁饮料、发酵酒中纳他霉素的测定。

# 2 原理

样品经甲醇提取或稀释处理，含油脂或固体样品采用加水后冷冻或离心的方式去除样品中的脂肪或固体颗粒成分，提取液过滤后用反相液相色谱，紫外检测器测定，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.2 冰乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。

# 3.2 标准品

纳他霉素 $\mathrm{(C_{33}H_{47}NO_{13}}$ ，CAS：7681-93-8)：纯度 $\geqslant 98\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.3 标准溶液配制

3.3.1 纳他霉素标准储备液 $(250\mathrm{mg / L})$ ：准确称取 $25\mathrm{mg}$ 纳他霉素标准品（精确至 $0.1\mathrm{mg}$ )，加入适量甲醇及 $1\mathrm{mL}$ 冰乙酸超声溶解，用甲醇定容至 $100~\mathrm{mL}$ ，配成浓度为 $250~\mathrm{mg / L}$ 标准储备液， $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 低温冰箱密封保存，3周内使用。  
3.3.2 纳他霉素标准中间液 $(50\mathrm{mg / L})$ ：准确吸取 $10.0~\mathrm{mL}$ 纳他霉素标准储备液(3.3.1)于 $50~\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度混匀，配成浓度为 $50~\mathrm{mg / L}$ 标准中间工作液，现用现配。  
3.3.3 纳他霉素系列标准工作液：分别吸取 $0.1\mathrm{mL}, 0.2\mathrm{mL}, 0.5\mathrm{mL}, 1.0\mathrm{mL}, 2.0\mathrm{mL}$ 纳他霉素标准中间液(3.3.2)于5个 $10\mathrm{mL}$ 容量瓶中，用甲醇定容，配制成 $0.5\mathrm{mg / L}, 1.0\mathrm{mg / L}, 2.5\mathrm{mg / L}, 5.0\mathrm{mg / L},$ $10\mathrm{mg / L}$ 纳他霉素系列标准工作液，现用现配。

# 4 仪器和设备

4.1 高效液相色谱仪：配紫外检测器。

4.2 电子天平：感量为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。  
4.3 超声波清洗器。  
4.4 离心机（转速 $\geqslant 4000\mathrm{r / min}$ ）。  
4.5 食物粉碎机。  
4.6 涡旋振荡器。  
4.7 有机针头式过滤器： $0.22\mu \mathrm{m}$

# 5 分析步骤

# 5.1 试样预处理

取多个预包装的样品，其中非均匀的液态、半固态样品用组织匀浆机匀浆；固体样品用研磨机充分粉碎并搅拌均匀，均匀的液体直接混匀，取其中的 $200\mathrm{g}$ 装入玻璃容器中，密封后于 $4^{\circ}\mathrm{C}$ 保存。

# 5.2 样品前处理

# 5.2.1 干酪和再制干酪及其类似品、酱卤肉制品、肉类（熏、烧、烤）、油炸肉类、西式火腿类、肉灌肠类、发酵肉制品类、蛋黄酱、沙拉酱

称取 $5.0\mathrm{g}$ 样品(精确至 $0.01\mathrm{g}$ )，置于 $50~\mathrm{mL}$ 离心管中，加入 $15.0~\mathrm{mL}$ 甲醇超声提取 $20\mathrm{min}$ ，加入 $5.0~\mathrm{mL}$ 水混匀，以 $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，于 $-20^{\circ}\mathrm{C}$ 冰箱中冷冻 $1\mathrm{h}$ ，上清液经 $0.22\mu \mathrm{m}$ 有机针头式过滤器过滤，收集滤液约 $2\mathrm{mL}$ 上机测定。

# 5.2.2 糕点样品

称取 $5.0\mathrm{g}$ 样品（精确至 $0.01\mathrm{g}$ ），置于 $50~\mathrm{mL}$ 离心管中，加入 $15.0~\mathrm{mL}$ 甲醇超声提取 $20\mathrm{min}$ ，加入 $5.0~\mathrm{mL}$ 水混匀，以 $4000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，上清液经 $0.22\mu \mathrm{m}$ 有机针式过滤器过滤，收集滤液约 $2\mathrm{mL}$ 上机测定。

# 5.2.3 果蔬汁（浆）、果汁饮料

称取 $5.0\mathrm{g}$ 样品（精确至 $0.01\mathrm{g}$ ），置于 $50~\mathrm{mL}$ 离心管中，加入 $15.0~\mathrm{mL}$ 甲醇超声提取 $20\mathrm{min}$ ，以 $4000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，将上清液定量转移至 $25~\mathrm{mL}$ 容量瓶，用水定容至刻度混匀，经 $0.22\mu \mathrm{m}$ 有机针式过滤器过滤，收集滤液约 $2\mathrm{mL}$ 上机测定。

# 5.2.4 发酵酒

量取适量的酒类样品，经 $0.22\mu \mathrm{m}$ 针式过滤器过滤，收集滤液约 $2\mathrm{mL}$ 上机测定。

# 5.3 仪器参考条件

5.3.1 色谱柱： $\mathrm{C_{18}}$ 色谱柱（ $4.6 \mathrm{~mm} \times 250 \mathrm{~mm}, 5 \mu \mathrm{m}$ ），或性能相当者。  
5.3.2 流动相：A：甲醇，B：乙酸 $+$ 水 $= 5 + 95$ （体积比）。  
5.3.3 流速： $1.0~\mathrm{mL / min}$   
5.3.4 检测波长： $305~\mathrm{nm}$   
5.3.5 柱温： $30^{\circ}C$ 。  
5.3.6 进样量： $10~\mu \mathrm{L}$   
5.3.7 梯度洗脱程序见表1。

# 5.4 标准曲线的制作

将纳他霉素系列标准工作液分别注入液相色谱仪中，测定相应的峰面积，以系列标准工作液中纳他霉素的浓度为横坐标，以纳他霉素的峰面积为纵坐标，绘制标准曲线，纳他霉素标准溶液的色谱图参见附录A中图A.1。

# 5.5 试样溶液的测定

将经过5.2试验方法前处理的待测溶液注入液相色谱仪中，得到待测液中纳他霉素的峰面积，根据标准曲线得到待测液中纳他霉素的浓度。

# 6 分析结果的表述

试样中纳他霉素的含量按式(1)计算，酒类样品中纳他霉素的含量按式(2)计算：

$$
X = \frac {\rho \times V}{m \times 1 0 0 0} \times 1 0 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——试样中纳他霉素的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由标准曲线得到待测溶液中纳他霉素的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

V ——样品中提取溶剂总体积或定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品质量，单位为克 $(\mathrm{g})$ ；

1000——单位换算系数。

计算结果保留两位有效数字。

$$
X = \frac {\rho}{1 0 0 0} \tag {2}
$$

式中：

$X$ ——试样中纳他霉素的含量，单位为克每升 $(\mathrm{g} / \mathrm{L})$ ；

$\rho$ ——由标准曲线得到待测溶液中纳他霉素的含量，单位为毫克每升 $(\mathrm{mg / L})$ ；

1000——单位换算系数。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

试样称取 $5.0\mathrm{g}$ ，按照试验方法操作并定容，发酵酒直接测定，各类食品的检出限和定量限如下：干酪和再制干酪及其类似品、酱卤肉制品、肉类（熏、烧、烤）、油炸肉类、西式火腿类、肉灌肠类、发酵肉制品类、蛋黄酱、沙拉酱、糕点的检出限为 $0.14\mathrm{mg/kg}$ ，定量限为 $0.48\mathrm{mg/kg}$ ；果蔬汁（浆）、果汁饮料检出限为 $0.18\mathrm{mg/kg}$ ，定量限为 $0.60\mathrm{mg/kg}$ ；发酵酒的检出限为 $0.035\mathrm{mg/L}$ ，定量限为 $0.12\mathrm{mg/L}$ 。

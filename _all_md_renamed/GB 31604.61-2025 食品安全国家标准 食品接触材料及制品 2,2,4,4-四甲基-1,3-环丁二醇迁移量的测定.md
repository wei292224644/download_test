# 食品安全国家标准

# 食品接触材料及制品 2,2,4,4-四甲基-1，3-环丁二醇迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品中2，2，4，4-四甲基-1，3-环丁二醇迁移量的测定方法。

本标准适用于塑料和涂层类食品接触材料及制品中2,2,4,4-四甲基-1,3-环丁二醇迁移量的测定。

# 2 原理

食品接触材料及制品采用食品模拟物或替代溶剂进行迁移试验后，水、酸性和含乙醇食品模拟物浸泡液经甲醇稀释后测定；含油脂食品模拟物浸泡液经甲醇萃取后测定；化学替代溶剂（ $95\%$ 乙醇和异辛烷)浸泡液直接上机测定。试液通过气相色谱柱分离，氢火焰离子化检测器测定，保留时间定性，峰面积外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙酸 $(\mathrm{C}_2\mathrm{H}_4\mathrm{O}_2)$ 。  
3.1.2 乙醇 $\mathrm{(C_2H_6O)}$ 。  
3.1.3 异辛烷 $\left(\mathrm{C}_{8} \mathrm{H}_{18}\right)$ ：色谱纯。  
3.1.4 橄榄油：需符合GB5009.156中附录A的要求。  
3.1.5 甲醇 $\mathrm{(CH_4O)}$ ：色谱纯。  
3.1.6 甲基叔丁基醚 $\left(\mathrm{C}_{5} \mathrm{H}_{12} \mathrm{O}\right)$ ：色谱纯。

# 3.2 试剂配制

$4\%$ （体积分数)乙酸溶液、 $10\%$ （体积分数)乙醇溶液、 $20\%$ （体积分数)乙醇溶液、 $50\%$ （体积分数)乙醇溶液和 $95\%$ (体积分数)乙醇溶液：按GB5009.156的规定配制。所用食品模拟物应满足GB31604.1及相关产品标准的规定。

# 3.3 标准品

2,2,4,4-四甲基-1,3-环丁二醇 $\mathrm{(C_8H_{16}O_2}$ ，CAS号：3010-96-6）：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质/标准样品。

# 3.4 标准溶液的配制

# 3.4.1 标准储备液 $(1000\mathrm{mg / L})$

准确称取 $25\mathrm{mg}$ (精确至 $0.1\mathrm{mg})$ 2,2,4,4-四甲基-1,3-环丁二醇标准品，用甲醇溶解后转移至 $25~\mathrm{mL}$ 棕色容量瓶中，用甲醇定容至刻度，充分摇匀后静置，存储于棕色玻璃瓶中。此溶液在 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 冰箱中密封保存，有效期为6个月。

# 3.4.2 标准中间液

3.4.2.1 标准中间液I（ $100\mathrm{mg / L}$ ）：准确移取 $1.00~\mathrm{mL}$ 标准储备液于 $10~\mathrm{mL}$ 棕色容量瓶中，用甲醇准确定容至刻度，充分摇匀后静置，存储于棕色玻璃瓶中。此溶液在 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱中密封保存，有效期为3个月。  
3.4.2.2 标准中间液Ⅱ：分别准确移取 $0.10\mathrm{mL}, 0.20\mathrm{mL}, 0.40\mathrm{mL}, 1.00\mathrm{mL}$ 和 $2.00\mathrm{mL}$ 标准储备液于 $10\mathrm{mL}$ 棕色容量瓶中，用甲基叔丁基醚定容至刻度，充分摇匀后静置，得到质量浓度分别为 $10\mathrm{mg/L}$ 、 $20\mathrm{mg/L}$ 、 $40\mathrm{mg/L}$ 、 $100\mathrm{mg/L}$ 和 $200\mathrm{mg/L}$ 的标准中间液，存储于棕色玻璃瓶中。此溶液在 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 冰箱中密封保存，有效期为3个月。

# 3.4.3 标准工作溶液

3.4.3.1 标准工作溶液A（适用于水基食品模拟物）：分别准确移取 $50~\mu \mathrm{L}, 100~\mu \mathrm{L}, 200~\mu \mathrm{L}, 300~\mu \mathrm{L}$ 和 $500~\mu \mathrm{L}$ 质量浓度为 $100~\mathrm{mg / L}$ 的标准中间液I至5个 $10~\mathrm{mL}$ 容量瓶中，分别准确加入 $2.0~\mathrm{mL}$ 对应的空白食品模拟物(水、 $4 \%$ 乙酸、 $10 \%$ 乙醇、 $20 \%$ 乙醇、 $50 \%$ 乙醇)，再用甲醇稀释定容至刻度，充分摇匀后静置，得到质量浓度分别为 $0.50~\mathrm{mg / L}, 1.0~\mathrm{mg / L}, 2.0~\mathrm{mg / L}, 3.0~\mathrm{mg / L}$ 和 $5.0~\mathrm{mg / L}$ 的标准工作溶液。临用现配。  
3.4.3.2 标准工作溶液B（适用于化学替代溶剂 $95\%$ 乙醇和异辛烷）：分别准确移取 $50~\mu \mathrm{L},100~\mu \mathrm{L}$ $200~\mu \mathrm{L},500~\mu \mathrm{L}$ 和 $1000~\mu \mathrm{L}$ 质量浓度为 $100~\mathrm{mg / L}$ 的标准中间液Ⅱ至5个 $10~\mathrm{mL}$ 容量瓶中，再用相应的化学替代溶剂( $95\%$ 乙醇或异辛烷)定容至刻度，充分摇匀后静置，得到质量浓度分别为 $0.50~\mathrm{mg / L}$ $1.0~\mathrm{mg / L},2.0~\mathrm{mg / L},5.0~\mathrm{mg / L}$ 和 $10~\mathrm{mg / L}$ 的标准工作溶液。临用现配。  
3.4.3.3 标准工作溶液C（适用于含油脂食品模拟物）：分别准确称取 $2.0\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）橄榄油于5个具塞离心管中，分别加入不同质量浓度的标准中间液Ⅱ各 $100~\mu \mathrm{L}$ ，充分摇匀后静置，得到质量分数分别为 $0.50\mathrm{mg / kg},1.0\mathrm{mg / kg},2.0\mathrm{mg / kg},5.0\mathrm{mg / kg}$ 和 $10\mathrm{mg / kg}$ 的标准工作溶液，分别加入 $2.0\mathrm{mL}$ 甲醇溶液萃取，涡旋振荡 $2\mathrm{min}$ 后，静置分层，取 $1\mathrm{mL}$ 上层清液，经 $0.45\mu \mathrm{m}$ 有机相尼龙微孔滤膜过滤后待测。

# 4 仪器和设备

4.1 气相色谱仪：配置氢火焰离子化检测器(FID)。  
4.2 分析天平：感量分别为 $0.1\mathrm{mg}$ 和 $0.001\mathrm{g}$ 。  
4.3 移液器：量程为 $10\mu \mathrm{L}\sim 200\mu \mathrm{L},1.0\mathrm{mL}$ 和 $10.0~\mathrm{mL}$   
4.4 有机相尼龙微孔滤膜：孔径为 $0.45\mu \mathrm{m}$   
4.5 涡旋振荡器。

# 5 分析步骤

# 5.1 试液的制备

# 5.1.1 迁移试验

食品接触材料及制品按照GB31604.1和GB5009.156的要求进行迁移试验。迁移试验所得浸泡液如不能立即测试，应置于 $0\% \sim 4\%$ 冰箱中避光保存不超过 $48\mathrm{h}$ 。若进行下一步试验，应将浸泡液恢复至室温后使用。

# 5.1.2 浸泡液的处理

# 5.1.2.1 水基食品模拟物浸泡液

准确移取迁移试验后的浸泡液 $2.0\mathrm{mL}$ 于 $10~\mathrm{mL}$ 容量瓶中，用甲醇稀释并定容至刻度，充分摇匀，经 $0.45\mu \mathrm{m}$ 有机相尼龙微孔滤膜过滤后待测。

# 5.1.2.2 化学替代溶剂(95%乙醇和异辛烷)浸泡液

取 $1\mathrm{mL}$ 浸泡液，经 $0.45\mu \mathrm{m}$ 有机相尼龙微孔滤膜过滤后待测。

# 5.1.2.3 含油脂食品模拟物浸泡液

准确称取迁移试验中得到的橄榄油浸泡液 $2.0\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）于具塞玻璃试管中，加入 $100~\mu \mathrm{L}$ 甲基叔丁基醚，再加入 $2\mathrm{mL}$ 甲醇萃取，涡旋振荡 $2\mathrm{min}$ 后，静置分层，取 $1\mathrm{mL}$ 上层清液，经 $0.45\mu \mathrm{m}$ 有机相尼龙微孔滤膜过滤后待测。

# 5.1.3 空白试液的制备

按 $5.1.1\sim 5.1.2$ 分别处理未与食品接触材料及制品接触的食品模拟物和化学替代溶剂。

# 5.2 仪器参考条件

5.2.1 色谱柱： $6\%$ 氰丙基/苯基 $94\%$ 二甲基聚硅氧烷毛细管气相色谱柱， $30 \mathrm{~m} \times 0.25 \mathrm{~mm} \times 1.4 \mu \mathrm{m}$ ，或等效色谱柱。  
5.2.2 进样口温度： $250^{\circ}C$ 。  
5.2.3 检测器温度： $300^{\circ}\mathrm{C}$ 。  
5.2.4 升温程序： $90^{\circ}\mathrm{C}$ 保持 $0\mathrm{min}$ ，以 $15^{\circ}\mathrm{C} / \mathrm{min}$ 上升至 $250^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$ 。  
5.2.5 载气：氮气，流速 $1.5\mathrm{mL} / \mathrm{min}$   
5.2.6 进样方式：不分流。  
5.2.7 进样量： $1\mu \mathrm{L}$

# 5.3 标准曲线的制作

按照5.2所列仪器参考条件，对标准工作溶液(3.4.3)从低到高依次进行测定。以标准工作溶液中2,2,4,4-四甲基-1,3-环丁二醇的浓度为横坐标，以对应的峰面积(2,2,4,4-四甲基-1,3-环丁二醇的顺反异构体峰面积之和)为纵坐标，绘制标准曲线。

# 5.4 试样溶液的测定

# 5.4.1 定性测定

按照5.2所列仪器参考条件，分别测定试样溶液和标准工作溶液，如果试样溶液与标准溶液中相对应的色谱峰保留时间偏差在 $\pm 0.5\%$ 范围以内，则可判断样品中存在相应的目标分析物。标准工作溶液的参考色谱图见附录A中图A.1～图A.8。

# 5.4.2 定量测试

按照5.2所列仪器参考条件，分别将试样溶液(5.1.2)和空白溶液(5.1.3)注入气相色谱仪中，得到目标分析物的峰面积，根据标准曲线计算得到待测试样溶液、空白溶液中2,2,4,4-四甲基-1,3-环丁二醇的含量 $c_{\setminus}c_{0}$ 。

空白溶液中2，2，4，4-四甲基-1，3-环丁二醇的含量 $c_{0}$ 不应高于方法检出限。

# 6 分析结果的表述

# 6.1 非密封制品类食品接触材料及制品中2,2,4,4-四甲基-1,3环丁二醇特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

对于非密封制品类食品接触材料及制品，目标分析物2,2,4,4-四甲基-1,3-环丁二醇特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(1)进行计算：

$$
X _ {1} = \frac {\left(c - c _ {0}\right) \times V}{S} \times \frac {S _ {0}}{m _ {1}} \quad \dots \tag {1}
$$

式中：

$X_{1}$ ——目标分析物的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样浸泡液中目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白浸泡液中目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

S——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$S_{0}$ ——样品实际使用时与食品接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$m_{1}$ ——样品实际使用时接触固态食品的质量，或实际接触液态食品的体积所对应的食品质量，单位为千克 $(\mathrm{kg})$ ；各种液态食品按密度为 $1\mathrm{kg} / \mathrm{L}$ 将其体积换算为相应的质量；当实际使用情形下的 $S_{0} / m_{1}$ 未知或无法估算时， $S_{0} / m_{1}$ 按 $6\mathrm{dm}^2 /\mathrm{kg}$ 计，即 $6\mathrm{dm}^2$ 食品接触材料及制品接触 $1\mathrm{kg}$ 食品或食品模拟物。

计算结果保留2位有效数字。

注：目标分析物2,2,4,4-四甲基-1,3-环丁二醇迁移量的测定结果包含顺式异构体与反式异构体之和。

# 6.2 密封制品类食品接触材料及制品中2,2,4,4-四甲基-1,3环丁二醇特定迁移量的计算（以 $\mathrm{mg / kg}$ 表示）

对于密封制品类食品接触材料及制品，当预期用途已知，目标分析物2，2，4，4-四甲基-1，3-环丁二醇特定迁移量以 $\mathrm{mg / kg}$ 表示时，按式(2)进行计算：

$$
X _ {2} = \frac {\left(c - c _ {0}\right) \times V}{S} \times \frac {S _ {0}}{m _ {2}} \tag {2}
$$

式中：

$X_{2}$ ——目标分析物的特定迁移量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c$ ——试样浸泡液中目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白浸泡液中目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克 $(\mathrm{kg})$ ；

$S$ ——迁移试验中试样与浸泡液接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$S_{0}$ ——密封制品实际使用时与食品接触的面积，单位为平方分米 $(\mathrm{dm}^2)$ ；

$m_{2}$ ——密封制品实际使用时与适配容器共同接触固态食品的质量，或接触液态食品的体积所对应的食品质量，单位为千克 $(\mathrm{kg})$ ；各种液态食品按密度为 $1\mathrm{kg} / \mathrm{L}$ 将其体积换算为相应的质量。

计算结果保留2位有效数字。

注：目标分析物2,2,4,4-四甲基-1,3-环丁二醇迁移量的测定结果包含顺式异构体与反式异构体之和。

# 6.3 密封制品类食品接触材料及制品2,2,4,4-四甲基-1,3环丁二醇特定迁移量的计算（以 $\mathbf{mg}$ /件表示）

对于密封制品类食品接触材料及制品，当预期用途未知，目标分析物2,2,4,4-四甲基-1,3-环丁二醇的特定迁移量以 $\mathrm{mg}/$ 件表示时，按式(3)计算，此时需注明采用的迁移试验方法、迁移试验中单个密封制品与食品模拟物接触的面积。

$$
X _ {3} = \frac {\left(c - c _ {0}\right) \times V}{n} \tag {3}
$$

式中：

$X_{3}$ ——目标分析物的特定迁移量，单位为毫克每件 $(\mathrm{mg}/$ 件）；

$c$ ——试样浸泡液中目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白浸泡液中目标分析物的含量，单位为毫克每升 $(\mathrm{mg / L})$ 或毫克每千克 $(\mathrm{mg / kg})$

$V$ ——试样浸泡液的体积或质量，单位为升(L)或千克(kg)；

$n$ ——浸泡时所用密封制品的数量，单位为件。

计算结果保留2位有效数字。

注：目标分析物2,2,4,4-四甲基-1,3-环丁二醇迁移量的测定结果包含顺式异构体与反式异构体之和。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不超过算术平均值的 $15\%$ 。

# 8 其他

本方法对水基食品模拟物（水、4%乙酸、10%乙醇、20%乙醇和50%乙醇)浸泡液中2，2，4，4-四甲基-1，3-环丁二醇迁移量的检出限均为 $1\mathrm{mg / L}$ ，定量限均为 $2.5\mathrm{mg / L}$ ；对化学替代溶剂(95%乙醇、异辛烷)浸泡液中2，2，4，4-四甲基-1，3-环丁二醇迁移量的检出限均为 $0.2\mathrm{mg / L}$ ，定量限均为 $0.50\mathrm{mg / L}$ ；对含油脂食品模拟物浸泡液中2，2，4，4-四甲基-1，3-环丁二醇迁移量的检出限为 $0.2\mathrm{mg / kg}$ ，定量限为 $0.50\mathrm{mg / kg}$ 。2，2，4，4-四甲基-1，3-环丁二醇迁移量的检出限和定量限按第6章进行计算。

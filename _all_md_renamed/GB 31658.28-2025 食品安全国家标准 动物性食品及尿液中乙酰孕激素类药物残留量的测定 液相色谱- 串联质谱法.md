# 1 范围

本文件规定了动物性食品及尿液中乙酰孕激素类药物残留量检测的制样和液相色谱-串联质谱方法。

本文件适用于猪、牛、羊、鸡的肌肉、脂肪、肝、肾脏组织，以及牛奶、羊奶、乳粉（牛、羊）、鸡蛋、猪尿、牛尿中醋酸氟孕酮、醋酸甲地孕酮、醋酸美仑孕酮、醋酸氯地孕酮、醋酸甲羟孕酮、醋酸羟孕酮残留量的测定。

# 4 原理

试样经 $\beta$ -葡萄糖醛苷酶/芳基硫酸酯酶水解后，残留的乙酰孕激素用乙腈提取，增强型除脂固相萃取柱净化。液相色谱-串联质谱测定，基质匹配外标法定量。

# 5 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6582规定的一级水。

# 5.1 试剂

5.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯  
5.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯  
5.1.3 甲酸（HCOOH）：色谱纯  
5.1.4 乙酸铵 $\mathrm{(CH_3COONH_2)}$ ：色谱纯  
5.1.5 萃取盐包：含 $4\mathrm{gNa_2SO_4}$ 、 $1\mathrm{gNaCl}$   
5.1.6 $\beta$ -葡萄糖醛苷酶/芳基硫酸酯酶（ $\beta$ Glucuronidase/aryl sulfatase)， $10000\mathrm{U / mL}$

# 5.2 溶液配制

5.2.1 $60\%$ 乙腈溶液：取乙腈 $600~\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$   
5.2.2 $2\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液：取乙酸铵 $15.4\mathrm{g}$ ，用水溶解并定容至 $100~\mathrm{mL}$ ，用适量乙酸调 $\mathrm{pH}$ 至5.2。  
5.2.3 $0.2\mathrm{mol} / \mathrm{L}$ 乙酸铵缓冲液：取 $2\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液 $10~\mathrm{mL}$ ，用水稀释至 $100~\mathrm{mL}$ ，用适量乙酸调 $\mathrm{pH}$ 至5.2。  
5.2.4 $2\mathrm{mmol / L}$ 乙酸铵溶液（含 $0.1\%$ 甲酸）：取甲酸 $1\mathrm{mL},2\mathrm{mol} / \mathrm{L}$ 乙酸铵溶液 $1\mathrm{mL}$ ，用水稀释至 $1000~\mathrm{mL}$   
5.2.5 $0.1\%$ 甲酸乙腈溶液：取甲酸 $1\mathrm{mL}$ ，用乙腈稀释至 $1000~\mathrm{mL}$

# 5.3 标准品

醋酸氟孕酮(Flurogestone acetate，FGA)、醋酸甲地孕酮(Megestrol acetate，MA)、醋酸美仑孕酮

(Melengestrol acetate, MGA)、醋酸氯地孕酮(Chlormadinone acetate, CMA)、醋酸羟孕酮(Hydroxyprogesterone acetate, HPA)、醋酸甲羟孕酮(Medroxyprogesterone acetate, MPA)含量 $\geqslant 99\%$ 。化合物信息见附录A。

# 5.4 标准溶液制备

# 5.5 材料

5.4.1 标准储备液：取乙酰孕激素标准品各 $10\mathrm{mg}$ ，精确至 $0.1\mathrm{mg}$ ，加甲醇使溶解并定容至 $10\mathrm{mL}$ ，配制成浓度均为 $1\mathrm{mg/mL}$ 的标准储备液。 $-18^{\circ}\mathrm{C}$ 以下保存，有效期3个月。  
5.4.2 中间浓度混合标准溶液：取乙酰孕激素标准储备液各适量，于同一 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，摇匀，配制成浓度为 $50~\mu \mathrm{g / mL}$ 的中间浓度混合标准溶液。醋酸甲羟孕酮单独配制浓度为 $50~\mu \mathrm{g / mL}$ 的中间浓度标准溶液。 $4^{\circ}C\sim 8^{\circ}C$ 保存，有效期1周。  
5.4.3 混合标准工作液：取适量中间浓度混合标准溶液于 $10\mathrm{mL}$ 容量瓶中，用甲醇稀释至刻度，摇匀，配制成浓度为 $1\mu \mathrm{g / mL}$ 的标准工作液。醋酸甲羟孕酮单独配制浓度为 $1\mu \mathrm{g / mL}$ 的标准工作液。现用现配。  
5.5.1 增强型脂肪去除固相萃取柱或性能相当者。  
5.5.2 滤膜：有机系， $0.2\mu \mathrm{m}$

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配有电喷雾离子源(ESI)。  
6.2 天平：感量 $0.01\mathrm{g}$ 。  
6.3 分析天平：感量 $0.00001\mathrm{g}$ 。  
6.4 均质机。  
6.5 涡旋混合器。  
6.6 振荡器。  
6.7 冷冻离心机：10000 r/min 或以上。  
6.8 氮吹仪。  
6.9 超声仪。  
6.10 pH计。

# 7 试样的制备与保存

# 7.1 试样的制备

取适量新鲜或解冻的空白或供试组织，绞碎并使均质。取适量新鲜或解冻的空白或供试牛羊奶样品，均质或混合均匀。奶粉按 $1:5$ 重量比还原成液态奶进行测定。取适量新鲜或冷藏的空白或供试鸡蛋，去壳后混合均匀。尿液样品取样后应在 $-18^{\circ}C$ 以下保存。试验前取适量恢复至室温，混匀备用。

a）取均质的供试样品，作为供试试样；  
b) 取均质的空白样品, 作为空白试样;  
c）取均质的空白样品，添加适宜浓度的标准工作液，作为空白添加试样。

# 7.2 试样的保存

$-18^{\circ} \mathrm{C}$ 以下保存。

# 8 测定步骤

# 8.1 酶解与提取

取组织和鸡蛋样品 $(5 \pm 0.05)$ g, 加入 $0.2 \mathrm{~mol} / \mathrm{L}$ 乙酸铵缓冲溶液 $(\mathrm{pH}=5.2) 3 \mathrm{~mL}$ 。取奶样 $(5 \pm 0.05)$ g, 奶粉样品 $(1 \pm 0.01)$ g(按7.1还原为液态奶), 加入 $2 \mathrm{~mol} / \mathrm{L}$ 乙酸铵溶液 $(\mathrm{pH}=5.2) 0.5 \mathrm{~mL}$ 。取尿液 $5 \mathrm{~mL}$ $(\pm 0.05 \mathrm{~mL})$ , 加入 $2 \mathrm{~mol} / \mathrm{L}$ 乙酸铵缓冲溶液 $(\mathrm{pH}=5.2) 0.5 \mathrm{~mL}$ , 充分混匀。再加入 $\beta$ -葡萄糖

醛苷酶/芳基硫酸酯酶 $100\mu \mathrm{L}$ 混合，于 $55^{\circ}C$ 下避光水浴振荡 $2\mathrm{h}$ 。取出后放置至室温，涡旋混匀 $1\mathrm{min}$ 加入乙腈 $10~\mathrm{mL}$ ，涡旋混匀 $1\mathrm{min}$ ，加入萃取盐包，涡旋混匀 $1\mathrm{min}\sim 2\mathrm{min}$ ，于一 $18^{\circ}C$ 冷冻 $30\mathrm{min},4^{\circ}C$ $9000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。准确量取上层溶液 $4\mathrm{mL}$ 于另一 $10~\mathrm{mL}$ 离心管，加水 $1\mathrm{mL}$ ，涡旋混匀。备用。

# 8.2 净化

取备用液过固相萃取柱，收集全部流出液于离心管中，于 $40^{\circ}\mathrm{C}$ 下氮吹至近干，用 $60\%$ 乙腈水溶液 $1.0~\mathrm{mL}$ 溶解，充分涡旋， $0.2\mu \mathrm{m}$ 滤膜过滤（如遇到样品浑浊，可先 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，然后滤膜过滤）。滤液供液相色谱-串联质谱测定。

# 8.3 基质匹配标准曲线的制备

取空白试样，按8.1和8.2处理样品，获得试样空白基质溶液。分别精密量取适量乙酰孕激素标准工作液，配制成浓度为 $0.5\mu \mathrm{g} / \mathrm{L}, 1\mu \mathrm{g} / \mathrm{L}, 2\mu \mathrm{g} / \mathrm{L}, 5\mu \mathrm{g} / \mathrm{L}, 10\mu \mathrm{g} / \mathrm{L}$ 和 $20\mu \mathrm{g} / \mathrm{L}$ （醋酸甲羟孕酮为 $1\mu \mathrm{g} / \mathrm{L}, 2\mu \mathrm{g} / \mathrm{L}, 4\mu \mathrm{g} / \mathrm{L}, 10\mu \mathrm{g} / \mathrm{L}, 20\mu \mathrm{g} / \mathrm{L}$ 和 $40\mu \mathrm{g} / \mathrm{L}$ 的系列标准溶液各 $1\mathrm{mL}$ ，吹干后用 $1\mathrm{mL}$ 试样空白基质溶液复溶，配制成基质匹配系列标准溶液，供液相色谱-串联质谱测定。以特征离子质量色谱峰面积为纵坐标、对应的基质匹配标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.5 测定法

# 8.5.1 定性测定

通过试样溶液被测物色谱图的保留时间与相应标准溶液的保留时间、各色谱峰的特征离子与相应浓度基质匹配标准溶液各色谱峰的特征离子相对照定性。试样溶液与标准溶液被测物保留时间偏差在 $\pm 0.1\mathrm{min}$ 以内；且检测到的相对离子丰度应与浓度相当的校正标准溶液相对离子丰度一致，其允许偏差为 $\pm 40\%$ 。

# 8.5.2 定量测定

按外标法计算试样中乙酰孕激素的残留量。实测试样采用单点或多点校正的方式定量；对于超过标准曲线最高点浓度的样品，用空白基质溶液将样品稀释到线性范围内后进行定量。乙酰孕激素类6种药物标准溶液定量离子色谱图见附录B。

# 8.6 空白试验

取空白试样，除不加药物外，采用完全相同的测定步骤进行测定。

# 9 结果计算和表述

组织、奶及奶粉、鸡蛋样品和尿液样品中乙酰孕激素类药物的残留量按公式(1)计算。

$$
X = \frac {A \times C _ {\mathrm {s}} \times V _ {1} \times V _ {2}}{A _ {\mathrm {s}} \times m \times V _ {3}} \dots \tag {1}
$$

式中：

$X$ ——组织、奶及奶粉、鸡蛋样品和尿液样品中乙酰孕激素类药物残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ 和微克/升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$A$ ——试样溶液中相应的乙酰孕激素的峰面积；

$C_s$ ——基质标准溶液中相应乙酰孕激素浓度的数值，单位微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$V_{1}$ ——提取步骤中使用提取液乙腈体积的数值，单位毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——提取液氮吹浓缩后的残留液定容体积的数值，单位毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——基质标准溶液中相应的乙酰孕激素的峰面积；

$m$ ——供试试料质量的数值，单位克(g)；

$V_{3}$ ——净化所用提取液体积的数值，单位毫升 $(\mathrm{mL})$ 。

注：计算结果以平行测定结果的算术平均值表示，含量不小于 $1\mu \mathrm{g} / \mathrm{kg}$ 的保留3位有效数字， $1\mu \mathrm{g} / \mathrm{kg}$ 以下保留至小数点后2位。

# 10 检测方法的灵敏度、正确度和精密度

# 10.1 灵敏度

本方法猪、牛、羊、鸡的肌肉、脂肪、肝脏、肾脏组织，牛奶、羊奶、鸡蛋、猪尿、牛尿中醋酸氟孕酮、醋酸甲

地孕酮、醋酸美仑孕酮、醋酸氯地孕酮、醋酸羟孕酮的检出限为 $0.2\mu \mathrm{g / kg}$ （尿液为 $0.2\mu \mathrm{g / L})$ ，定量限为 $0.5\mu \mathrm{g / kg}$ （尿液为 $0.5\mu \mathrm{g / kg})$ ；醋酸甲羟孕酮检出限为 $0.4\mu \mathrm{g / kg}(0.4\mu \mathrm{g / kg})$ ，定量限为 $1.0\mu \mathrm{g / kg}$ （尿液为 $1.0\mu \mathrm{g / kg})$ 。

奶粉中醋酸氟孕酮、醋酸甲地孕酮、醋酸美仑孕酮、醋酸氯地孕酮、醋酸羟孕酮的检出限为 $1\mu \mathrm{g / kg}$ 定量限为 $2.5\mu \mathrm{g / kg}$ 。醋酸甲羟孕酮的检出限为 $2\mu \mathrm{g / kg}$ ，定量限为 $5\mu \mathrm{g / kg}$

# 10.2 正确度

本方法猪、牛、羊、鸡的肌肉、脂肪、肝脏、肾脏组织，牛奶、羊奶、鸡蛋、猪尿、牛尿在 $0.5\mu \mathrm{g} / \mathrm{kg}\sim$ $2.5\mu \mathrm{g} / \mathrm{kg}$ （醋酸甲羟孕酮在 $1.0\mu \mathrm{g} / \mathrm{kg}\sim 5.0\mu \mathrm{g} / \mathrm{kg})$ 添加水平下，奶粉在 $2.5\mu \mathrm{g} / \mathrm{kg}\sim 12.5\mu \mathrm{g} / \mathrm{kg}$ （醋酸甲羟孕酮在 $5.0\mu \mathrm{g} / \mathrm{kg}\sim 25.0\mu \mathrm{g} / \mathrm{kg})$ 添加水平下，乙酰孕激素平均回收率范围为 $60\% \sim 120\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 15\%$ ，批间相对标准偏差 $\leqslant 20\%$ 。

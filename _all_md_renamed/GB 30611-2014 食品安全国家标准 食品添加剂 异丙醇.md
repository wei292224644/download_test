# 附录A

# 检验方法

# A.1 警示

试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和防护措施。

# A.2 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682中规定的三级水。试验方法中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603之规定制备；所用溶液除另有说明外，均为水溶液。

# A.3 鉴别试验

# A.3.1 溶解性

易溶于水、乙醇、乙醚和其他有机溶剂。

# A.3.2 折光率

按GB/T614的规定进行测定。折光率 $n$ （20， $D$ ）应为： $1.377\sim 1.380$

# A.3.3 相对密度

按GB/T 5009.2的规定进行测定。相对密度 $d_{20}^{20}$ 应为 $0.784 \sim 0.788$ 。

# A. 4 酸度的测定

# A.4.1 试剂和材料

A.4.1.1 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.01\mathrm{mol / L}$   
A.4.1.2 酚酞指示液： $10\mathrm{g / L}$

# A.4.2 分析步骤

向 $100 \mathrm{~mL}$ 水中加2滴酚酞指示液，用氢氧化钠标准滴定溶液滴定至粉红色刚出现，并保持 $30 \mathrm{~s}$ 不变，然后加 $50 \mathrm{~mL}$ （约 $39 \mathrm{~g}$ ）试样，混合。再用氢氧化钠标准滴定溶液滴定至恢复粉红色，氢氧化钠标准滴定溶液的耗用量不应大于 $0.7 \mathrm{~mL}$ 。相当于酸度不大于 $0.002 \%$ 。

# 前言

本标准代替GB25594—2010《食品安全国家标准 食品工业用酶制剂》，GB8276—2006《食品添加剂糖化酶制剂》，GB20713—2006《食品添加剂 $\alpha$ -乙酰乳酸脱羧酶制剂》，GB8275—2009《食品添加剂 $\alpha$ -淀粉酶制剂》。

本标准与GB25594—2010相比，主要变化如下：

——增加了酶活力的术语和定义；  
——增加了产品分类、理化要求；  
——在附录中给出了部分酶制剂的酶活力测定方法。

# 食品安全国家标准

# 食品添加剂 食品工业用酶制剂

# 1 范围

本标准适用于GB2760允许使用的食品工业用酶制剂。

# 2 术语和定义

# 2.1 食品工业用酶制剂

由动物或植物的可食或非可食部分直接提取，或由传统或通过基因修饰的微生物（包括但不限于细菌、放线菌、真菌菌种)发酵、提取制得，用于食品加工，具有特殊催化功能的生物制品。

注：商品化的酶制剂产品允许加入易于产品贮存、使用的配料成分。

# 2.2 酶活力

酶在一定条件下催化某一特定反应的能力，即为酶活力，是表达酶制剂产品的一个特征性专属指标。

# 2.3 抗菌活性

抑制或杀灭微生物的能力。

# 3 产品分类

按产品形态分为固体剂型和液体剂型两类。

# 4 技术要求

# 4.1 原料要求

4.1.1 用于生产酶制剂的原料必须符合良好生产规范或相关要求，在正常使用条件下不应对最终食品产生有害健康的残留污染。

4.1.2 来源于动物的酶制剂，其动物组织必须符合肉类检疫要求。

4.1.3 来源于植物的酶制剂，其植物组织不得霉变。

4.1.4 对微生物生产菌种应进行分类学和(或)遗传学的鉴定，并应符合有关规定。菌种的保藏方法和条件应保证发酵批次之间的稳定性和可重复性。

# 4.2 产品要求

# 4.2.1 理化指标

产品酶活力在标示值的 $85\% \sim 115\%$

注：本标准附录A给出的酶活力测定方法供参考。企业可按相应标准中给出的或企业规定的方法测定。

# 附录A

# 酶活力测定方法

# A.1 一般规定

本标准所用试剂和水，在没有注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 $\alpha$ -淀粉酶活力的测定

# A.2.1 $\pmb{\alpha}$ -淀粉酶

能水解淀粉分子链中的 $\alpha -1,4$ 葡萄糖苷键，将淀粉链切断成为短链糊精和少量麦芽糖和葡萄糖，使淀粉黏度迅速下降的酶。

# A.2.2 $\alpha$ -淀粉酶活力

# A.2.2.1 中温 $\alpha$ -淀粉酶活力单位

$1\mathrm{g}$ 固体酶粉(或 $1\mathrm{mL}$ 液体酶)，于 $60^{\circ}\mathrm{C},\mathrm{pH}6.0$ 条件下， $1\textrm{h}$ 液化 $1\mathrm{g}$ 可溶性淀粉，即为1个酶活力单位，以 $\mathrm{U / g(U / mL)}$ 表示。

# A.2.2.2 耐高温 $\alpha$ -淀粉酶活力单位

$1\mathrm{g}$ 固体酶粉(或 $1\mathrm{mL}$ 液体酶)，于 $70^{\circ}\mathrm{C},\mathrm{pH}6.0$ 条件下， $1\mathrm{min}$ 液化 $1\mathrm{mg}$ 可溶性淀粉，即为1个酶活力单位，以 $\mathrm{U / g(U / mL)}$ 表示。

# A.2.3 原理

$\alpha-$ 淀粉酶制剂能将淀粉分子链中的 $\alpha-1,4$ 葡萄糖苷键随机切断成长短不一的短链糊精、少量麦芽糖和葡萄糖，而使淀粉对碘呈蓝紫色的特性反应逐渐消失，呈现棕红色，其颜色消失的速度与酶活性有关，据此可通过反应后的吸光度计算酶活力。

# A.2.4 试剂和材料

A.2.4.1 碘。

A.2.4.2 碘化钾。

A.2.4.3 原碘液：称取 $11.0\mathrm{g}$ 碘和 $22.0\mathrm{g}$ 碘化钾，用少量水使碘完全溶解，定容至 $500~\mathrm{mL}$ ，贮存于棕色瓶中。

A.2.4.4 稀碘液：吸取原碘液 $2.00\mathrm{mL}$ ，加 $20.0\mathrm{g}$ 碘化钾用水溶解并定容至 $500\mathrm{mL}$ ，贮存于棕色瓶中。

A.2.4.5 可溶性淀粉溶液 $(20 \mathrm{~g} / \mathrm{L})$ : 称取 $2.000 \mathrm{~g}$ (精确至 $0.001 \mathrm{~g}$ ) 可溶性淀粉(以绝干计)于烧杯中, 用少量水调成浆状物, 边搅拌边缓缓加入 $70 \mathrm{~mL}$ 沸水中, 然后用水分次冲洗装淀粉的烧杯, 洗液倒入其中, 搅拌加热至完全透明, 冷却定容至 $100 \mathrm{~mL}$ 。溶液现配现用。

注：可溶性淀粉应采用酶制剂分析专用淀粉。

A.2.4.6 磷酸缓冲液 $(\mathrm{pH}6.0)$ ：称取 $45.23\mathrm{g}$ 磷酸氢二钠 $\left(\mathrm{Na}_{2}\mathrm{HPO}_{4} \cdot 12\mathrm{H}_{2}\mathrm{O}\right)$ 和 $8.07\mathrm{g}$ 柠檬酸 $\left(\mathrm{C}_{6}\mathrm{H}_{8}\mathrm{O}_{7} \cdot \mathrm{H}_{2}\mathrm{O}\right)$ ，用水溶解并定容至 $1000\mathrm{mL}$ 。用 $\mathrm{pH}$ 计校正后使用。

A.2.4.7 盐酸溶液 $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$

# A.2.5 仪器和设备

A.2.5.1 分光光度计。  
A.2.5.2 恒温水浴：精度 $\pm 0.1^{\circ}C$   
A.2.5.3 自动移液器。  
A.2.5.4 试管： $25\mathrm{mm}\times 200\mathrm{mm}$   
A.2.5.5 秒表。

# A.2.6 分析步骤

# A.2.6.1 待测酶液的制备

称取试样 $1\mathrm{g}\sim 2\mathrm{g}$ (精确至 $0.0001\mathrm{g}$ ) 或准确吸取 $1.00\mathrm{mL}$ , 用少量磷酸缓冲液充分溶解, 将上清液小心倾入容量瓶中, 若有剩余残渣, 再加少量磷酸缓冲液充分研磨, 最终样品全部移入容量瓶中, 用磷酸缓冲液定容至刻度, 摇匀。用四层纱布过滤, 滤液待用。

注：待测中温 $\alpha$ -淀粉酶酶液酶活力控制酶浓度在 $3.4\mathrm{U / mL}\sim 4.5\mathrm{U / mL}$ 范围内，待测耐高温 $\alpha$ -淀粉酶活力控制酶浓度在 $60~\mathrm{U / mL}\sim 65~\mathrm{U / mL}$ 范围内。

# A.2.6.2 测定

a）吸取 $20.0\mathrm{mL}$ 可溶性淀粉溶液于试管中，加入磷酸缓冲液 $5.00~\mathrm{mL}$ ，摇匀后，置于 $60^{\circ}\mathrm{C} \pm 0.2^{\circ}\mathrm{C}$ （耐高温 $\alpha$ -淀粉酶制剂置于 $70^{\circ}\mathrm{C} \pm 0.2^{\circ}\mathrm{C}$ )恒温水浴中预热 $8\mathrm{min}$ ；  
b）加入 $1.00\mathrm{mL}$ 稀释好的待测酶液，立即计时，摇匀，准确反应 $5\mathrm{min}$   
c) 立即用自动移液器吸取 $1.00 \mathrm{~mL}$ 反应液, 加到预先盛有 $0.5 \mathrm{~mL}$ 盐酸溶液和 $5.00 \mathrm{~mL}$ 稀碘液的试管中, 摇匀, 并以 $0.5 \mathrm{~mL}$ 盐酸溶液和 $5.00 \mathrm{~mL}$ 稀碘液为空白, 于 $660 \mathrm{~nm}$ 波长下, 用 $10 \mathrm{~mm}$ 比色皿迅速测定其吸光度 (A)。根据吸光度查表附录 B, 求得测试酶液的浓度。

# A.2.6.3 结果计算

# A.2.6.3.1 中温 $\alpha$ -淀粉酶制剂的酶活力

中温 $\alpha$ -淀粉酶制剂的酶活力 $X_{1}$ ，单位为 $\mathrm{U / mL}$ 或 $\mathrm{U / g}$ ，按式(A.1)计算：

$$
X _ {1} = c \times n \quad \dots \tag {A.1}
$$

式中：

$c$ ——测试酶样浓度，单位为 $\mathrm{U / mL}$ 或 $\mathrm{U / g}$

$n$ ——样品的稀释倍数。

所得结果表示至整数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

# A.2.6.3.2 耐高温 $\alpha$ -淀粉酶制剂的酶活力

耐高温 $\alpha$ -淀粉酶制剂的酶活力 $X_{2}$ ，以 $\mathrm{U / mL}$ 或 $\mathrm{U / g}$ 计，按式(A.2)计算：

$$
X _ {2} = c \times n \times 1 6. 6 7 \quad \dots \tag {A.2}
$$

式中：

$c$ ——测试酶样的浓度，单位为 $\mathrm{U / mL}$ 或 $\mathrm{U / g}$

$n$ ——样品的稀释倍数；

16.67——根据酶活力定义计算的换算系数。

所得结果表示至整数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的相对误差不得超过 $5\%$ 。

# A.3 葡糖淀粉酶(淀粉葡糖苷酶)活力的测定（黑曲霉及其变异株来源）

# A.3.1 葡糖淀粉酶（淀粉葡糖苷酶）

以淀粉为底物，在一定条件下从淀粉的非还原性末端开始水解 $\alpha -1,4,\alpha -1,6,\alpha -1,3$ 葡萄糖苷键产生葡萄糖的淀粉葡萄糖苷酶。

# A.3.2 葡糖淀粉酶(淀粉葡糖苷酶)活力

$1\mathrm{mL}$ 酶液或 $1\mathrm{g}$ 酶粉在 $40^{\circ}\mathrm{C}, \mathrm{pH}4.6$ 的条件下， $1\mathrm{h}$ 水解可溶性淀粉产生 $1\mathrm{mg}$ 葡萄糖，即为一个酶活力单位，以 $\mathrm{U/mL}$ （或 $\mathrm{U/g}$ ）表示。

# A.3.3 试剂和材料

A.3.3.1 $0.05 \mathrm{~mol} / \mathrm{L}$ 乙酸-乙酸钠缓冲溶液 $(\mathrm{pH}4.6)$ ：称取乙酸钠 $\left(\mathrm{CH}_{3} \mathrm{COONa} \cdot 3 \mathrm{H}_{2} \mathrm{O}\right) 6.7 \mathrm{~g}$ ，吸取冰乙酸2.6 mL，用水溶解并定容至 $1000 \mathrm{~mL}$ 。上述缓冲溶液的 $\mathrm{pH}$ ，应使用酸度计加以校正。

A.3.3.2 $0.05\mathrm{mol} / \mathrm{L}$ 硫代硫酸钠标准滴定溶液。

A.3.3.3 $0.1\mathrm{mol} / \mathrm{L}$ 碘标准溶液。

A.3.3.4 $0.1\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液。

A.3.3.5 $2\mathrm{mol} / \mathrm{L}$ 硫酸溶液：吸取分析纯浓硫酸（相对密度1.84） $5.6\mathrm{mL}$ 缓缓加入适量水中，冷却后，用水定容至 $100~\mathrm{mL}$ ，摇匀。

A.3.3.6 $200\mathrm{g / L}$ 氢氧化钠溶液：称取氢氧化钠 $20\mathrm{g}$ ，用水溶解，并定容至 $100~\mathrm{mL}$ 。

A.3.3.7 $20 \mathrm{~g} / \mathrm{L}$ 可溶性淀粉溶液: 称取可溶性淀粉 $2 \mathrm{~g} \pm 0.001 \mathrm{~g}$ , 然后用少量水调匀, 缓缓倾入已沸腾的水中,煮沸、搅拌直至透明, 冷却, 用水定容至 $100 \mathrm{~mL}$ 。此溶液需当天配制。

注：可溶性淀粉应采用酶制剂分析专用淀粉。

# A.3.4 仪器和设备

A.3.4.1 分析天平：精度 $0.2\mathrm{mg}$ 。  
A.3.4.2 酸度计：精度 $0.01\mathrm{pH}$ 。  
A.3.4.3 分析天平：精度 $0.2\mathrm{mg}$ 。  
A.3.4.4 恒温水浴： $40^{\circ}C$ 土 $0.1^{\circ}C$   
A.3.4.5 连续多档分配器(移液枪)。  
A.3.4.6 磁力搅拌器。

# A.3.5 分析步骤

# A.3.5.1 待测酶液的制备

A.3.5.1.1 液体酶: 使用连续多档分配器准确吸取适量酶样, 移入容量瓶中, 用缓冲溶液稀释至刻度, 充分摇匀, 待测。  
A.3.5.1.2 固体酶: 用 $50 \mathrm{~mL}$ 小烧杯准确称取适量酶样, 精确至 $1 \mathrm{mg}$ , 用少量乙酸-乙酸钠缓冲溶液溶解, 并用玻璃棒仔细捣研, 将上层清液小心倾入适当的容量瓶中, 在沉渣中再加入少量缓冲溶液, 如此反复捣研 3 次 $\sim$

4次，取上清液，最后全部移入容量瓶中，用缓冲溶液定容，磁力搅拌 $30\mathrm{min}$ 以充分混匀，取上清液测定。

注：1.制备待测酶液时，样液浓度应控制在滴定空白和样品时消耗 $0.05\mathrm{mol / L}$ 硫代硫酸钠标准滴定溶液的差值在 $4.5\mathrm{mL}$ $\sim 5.5\mathrm{mL}$ 范围内(酶活力约为 $120~\mathrm{U / mL}\sim 150~\mathrm{U / mL})$ 。  
2. 液体酶根据产品特性也可称取，按克计算。

# A.3.5.2 测定

取A、B两支 $50~\mathrm{mL}$ 比色管，分别加入可溶性淀粉溶液 $25~\mathrm{mL}$ 和乙酸-乙酸钠缓冲溶液 $5~\mathrm{mL}$ ，摇匀。于 $40^{\circ}\mathrm{C} \pm 0.2^{\circ}\mathrm{C}$ 的恒温水浴中预热 $5\mathrm{min} \sim 10\mathrm{min}$ 。在B管中加入待测酶液 $2.0~\mathrm{mL}$ ，立即记时，摇匀。在此温度下准确反应 $30\mathrm{min}$ 后，立即向A、B两管中各加氢氧化钠溶液 $0.2~\mathrm{mL}$ ，摇匀，同时将两管取出，迅速用水冷却，并于A管中补加待测酶液 $2.0~\mathrm{mL}$ （作为空白对照）。

吸取上述A、B两管中的反应液各 $5.0~\mathrm{mL}$ ，分别于两个碘量瓶中，准确加入碘标准溶液 $10.0~\mathrm{mL}$ ，再加氢氧化钠溶液 $15~\mathrm{mL}$ ，边加边摇匀，并于暗处放置 $15\mathrm{min}$ ，取出。用水淋洗瓶盖，加入硫酸溶液 $2\mathrm{mL}$ ，用硫代硫酸钠标准滴定溶液滴定蓝紫色溶液，直至刚好无色为其终点，分别记录空白和样品消耗硫代硫酸钠标准滴定溶液的体积(A、B)。

# A.3.6 结果计算

葡糖淀粉酶制剂的酶活力 $X_{3}$ ，单位为 $\mathrm{U / mL}$ 或 $\mathrm{U / g}$ ，按式(A.3)计算：

$$
X _ {3} = \frac {(A - B) \times c _ {1} \times 9 0 . 0 5 \times 3 2 . 2 \times n \times 2}{5} \times \frac {1}{2} \quad \dots \tag {A.3}
$$

式中：

A ——滴定空白时, 消耗硫代硫酸钠标准滴定溶液的体积, 单位为毫升 $(\mathrm{mL})$ ;

B ——滴定样品时，消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——硫代硫酸钠标准滴定溶液的准确浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

90.05——与 $1.00 \mathrm{~mL}$ 硫代硫酸钠标准溶液相当的葡萄糖的摩尔质量， $\mathrm{g/mol}(M = 90.05)$ ；

32.2 ——反应液的总体积，单位为毫升 $(\mathrm{mL})$

5 ——吸取反应液的体积；

1/2 ——折算成 $1\mathrm{mL}$ 酶液的量；

$n$ ——稀释倍数；

2 ——反应 $30\mathrm{min}$ ，换算成 $1\mathrm{h}$ 的酶活力系数。

以样品测定结果的算术平均值表示，保留3位有效数字。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# A.4 蛋白酶活力的测定

# A.4.1 蛋白酶

能切断蛋白质分子内部的肽键，使蛋白质分子变成小分子多肽和氨基酸的酶。

# A.4.2 蛋白酶活力

蛋白酶活力以蛋白酶活力单位表示，定义为 $1\mathrm{g}$ 或 $1\mathrm{mL}$ 酶，在一定温度和 $\mathrm{pH}$ 条件下， $1\mathrm{min}$ 水解酪蛋白产生 $1\mu \mathrm{g}$ 酪氨酸，即为1个酶活力单位，以 $\mathrm{U} / \mathrm{g}(\mathrm{U} / \mathrm{mL})$ 表示。

# A.4.3 原理

蛋白酶在一定的温度与 $\mathrm{pH}$ 条件下, 水解酪蛋白底物, 产生含有酚基的氨基酸 (如:酪氨酸、色氨酸等), 在

碱性条件下，将福林试剂还原，生成钼蓝与钨蓝，用分光光度计于波长 $680~\mathrm{nm}$ 下测定溶液的吸光度。酶活力与吸光度成比例，由此可以计算产品的酶活力。

# A.4.4 试剂和材料

# A.5 果胶酶活力的测定（黑曲霉来源）

# A.5.1 果胶酶

能水解果胶，生成含有还原性基团产物的酶。

# A.5.2 果胶酶活力

在 $50^{\circ}\mathrm{C},\mathrm{pH}3.5$ 的反应条件下， $1\mathrm{g}$ 固体酶粉(或 $1\mathrm{mL}$ 液体酶)， $1\textrm{h}$ 分解果胶产生 $1\mathrm{mg}$ 半乳糖醛酸，即为1个酶活力单位，以 $\mathrm{U / g}$ 或 $\mathrm{U / mL}$ 表示。

# A.5.3 原理

果胶酶能水解果胶，生成的半乳糖醛酸的还原性糖醛基可用次亚碘酸法定量测定，以此来表示果胶酶的活性。

本方法主要用于检测果胶酶产品中多聚半乳糖醛酸酶和果胶裂解酶的活力，对于果胶酶中的果胶(甲基)酯酶不适用。

# A.6 $\pmb{\alpha}$ -乙酰乳酸脱羧酶酶活力的测定

# A.6.1 $\alpha$ -乙酰乳酸脱羧酶

可以与 $\alpha$ -乙酰乳酸反应脱羧，将 $\alpha$ -乙酰乳酸羧基转化为3-羟基-2-丁酮的酶。

# A.6.2 $\alpha$ -乙酰乳酸脱羧酶活力

在 $30^{\circ}\mathrm{C},\mathrm{pH}6.0$ 的条件下， $1\mathrm{g}$ 或 $1\mathrm{mL}$ 酶样品与底物 $\alpha$ -乙酰乳酸起反应，每分钟生成 $1\mu \mathrm{mol}$ 的3-羟基-2-丁酮(乙偶姻)，即为1个酶活力单位，以 $\mathrm{U / g}$ (或 $\mathrm{U / mL}$ ) 表示。

# A.6.3 分光光度计法

# A.6.3.1 范围

本方法规定了 $\alpha$ -乙酰乳酸脱羧酶酶活力的测定方法。

本方法适用于用分光光度计测定 $\alpha$ -乙酰乳酸脱羧酶制剂中 $\alpha$ -乙酰乳酸脱羧酶的酶活力。本方法不适用于啤酒和其他含醇产品中 $\alpha$ -乙酰乳酸脱羧酶的酶活力的测定。

试样中3-羟基-2-丁酮(乙偶姻)和/或双乙酰的存在会使试验结果偏大。

乙偶姻在贮存时易形成二聚体，从而影响试验结果。但若遵循配制步骤中所述的措施去预防，本方法仍可使用。

# A.6.3.2 原理

$\alpha-$ 乙酰乳酸脱羧酶与底物 $\alpha-$ 乙酰乳酸反应脱羧生成乙偶姻。乙偶姻在碱性条件下与萘酚和肌酸的混合物反应生成红色产物。通过在 $522 \mathrm{~nm}$ 下测定溶液的吸光度，可以从乙偶姻标准曲线上得出反应生成的乙偶姻的量，进而计算出 $\alpha-$ 乙酰乳酸脱羧酶的酶活力。

# A.6.3.3 试剂和材料

# A.6.3.5.4 标准曲线的绘制

以 $522\mathrm{nm}$ 波长下的吸光度为 $Y$ 轴，乙偶姻的浓度 $(\mathrm{mg / L})$ 为 $X$ 轴，绘制标准曲线，计算出标准曲线的斜率 $h$ （或用回归方程计算）。

# A.6.3.5.5 酶样品的反应

将试样溶液和底物先置于 $30^{\circ}\mathrm{C}$ 水浴中预热约 $10\mathrm{min}$ ，然后，按下述方法处理每一个试样溶液：

a) 样品吸光度值 $H_{1}$ : 吸取酶样品溶液 $200.0 \mu \mathrm{L}$ 于 $30^{\circ} \mathrm{C} \pm 0.1^{\circ} \mathrm{C}$ 水浴中的 $10 \mathrm{~mL}$ 试管里, 然后加入预热好的底物 $200.0 \mu \mathrm{L}$ , 用涡旋振荡器充分混匀, 迅速放回到水浴中, 并开始计时。  
b）空白吸光度值 $H_{2}$ ：用缓冲液代替试料溶液，依上述步骤进行操作。  
c) 反应 $20.0 \mathrm{~min}$ 后, 依次向每支试管中加入显色剂 $4.60 \mathrm{~mL}$ , 用涡旋振荡器充分混匀。置于室温, 重新开始计时。  
d) 反应 $40.0 \mathrm{~min}$ 后，使用分光光度计，在 $522 \mathrm{~nm}$ 波长下测定各管溶液的吸光度。

# A.6.3.6 结果计算

$\alpha$ -乙酰乳酸脱羧酶制剂的酶活力 $X_{6}$ ，单位为 $\mathrm{U / mL}$ 或 $\mathrm{U / g}$ ，按式(A.6)计算：

$$
X _ {6} = \frac {\left(H _ {1} - H _ {2}\right) \times 0 . 0 0 1 1 3 5 1 \times F}{m \times h} \quad \dots \tag {A.6}
$$

式中：

$H_{1}$ —样品的吸光度；

$H_{2}$ ——空白的吸光度；

0.001 135 1 ——0.1 g 的乙偶烟所对应的摩尔数；

$F$ —样品溶液反应前的总稀释倍数；

$m$ ——试料的质量，单位为克(g)；

$h$ ——标准曲线的斜率。

# A.6.3.7 结果表示

样品的测定结果用算术平均值表示。

当结果小于 $1\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 时给出1位有效数字；当结果大于等于 $1\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 且小于 $100\mathrm{U} / \mathrm{g}$ (或 $\mathrm{mL}$ ) 时给出2位有效数字；当结果大于等于 $100\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 时给出3位有效数字。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# A.6.4 全自动生化分析仪法

# A.6.4.1 范围

本方法规定了 $\alpha$ -乙酰乳酸脱羧酶活力的测定方法。

本方法适用于用全自动生化分析仪测定 $\alpha$ -乙酰乳酸脱羧酶制剂中 $\alpha$ -乙酰乳酸脱羧酶的酶活力。本方法不适用于啤酒和其他含醇产品中 $\alpha$ -乙酰乳酸脱羧酶酶活力的测定。

本方法的检测限为 $0.6\mathrm{U / g}$ 或 $0.6\mathrm{U / mL}$

试样中3-羟基-2-丁酮(乙偶烟)和/或双乙酰的存在会使试验结果偏大。

乙偶姻在贮存时易形成二聚体，从而影响试验结果。但若遵循配制步骤中所述的措施去预防，本方法仍可使用。

# A.6.4.2 原理

$\alpha-$ 乙酰乳酸脱羧酶与底物 $\alpha-$ 乙酰乳酸反应脱羧生成乙偶姻。乙偶姻在碱性条件下与萘酚和肌酸的混合物反应生成红色产物。通过在 $510 \mathrm{~nm}$ 波长下测定标准溶液的吸光度绘制出标准曲线，对照标准曲线进一步计算出酶样品的活力。

# A.6.4.6 结果计算和表示

# A.6.4.6.1 标准曲线的计算

用参数 Logit-Log 法计算出标准曲线。其中 $Y$ 轴单位为 $\mathrm{OD / min}$ , $X$ 轴单位为标准点的酶活力 $\mathrm{mU / mL}$ 。

# A.6.4.6.2 样品活性的计算

从标准曲线上读出样品最终稀释液的活性力，单位为 $\mathrm{mU / mL}$

$\alpha$ -乙酰乳酸脱羧酶制剂的酶活力 $X_{7}$ ，单位为 $\mathrm{U / mL}$ 或 $\mathrm{U / g}$ ，按式(A.7)计算：

$$
X _ {7} = \frac {A _ {4} \times F \times D}{m \times 1 0 0 0} \tag {A.7}
$$

式中：

$A_{4}$ ——由标准曲线得出的样品最终稀释液的活性力，单位为 $\mathrm{mU / mL}$ ；

$F$ ——溶解样品用的容量瓶的体积，单位为毫升 $(\mathrm{mL})$ ；

$D$ ——稀释倍数；

$m$ ——试样的质量，单位为克(g)；

1000 ——mU 到 U 的单位转换因子。

样品的测定结果用算术平均值表示。当标准对照品稀释液的实验值在 $0.48\mathrm{mU / mL}\sim 0.52\mathrm{mU / mL}$ 时，样品的试验结果有效，可计算平均值。否则，应重新进行试验。

当结果小于 $1\mathrm{U} / \mathrm{g}$ (或 $\mathrm{mL}$ ) 时给出1位有效数字；当结果大于等于 $1\mathrm{U} / \mathrm{g}$ (或 $\mathrm{mL}$ ) 且小于 $100\mathrm{U} / \mathrm{g}$ (或 $\mathrm{mL}$ ) 时给出2位有效数字；当结果大于等于 $100\mathrm{U} / \mathrm{g}$ (或 $\mathrm{mL}$ ) 时给出3位有效数字。当结果小于 $0.6\mathrm{U} / \mathrm{g}$ (或 $\mathrm{mL}$ ) 时，表示为 $<  0.6\mathrm{U} / \mathrm{g}$ (或 $\mathrm{mL}$ )。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $4\%$ 。

# A.7 转葡糖苷酶活力的测定

# A.7.1 转葡糖苷酶

可水解麦芽糖分子及直链麦芽糊精中 $\alpha -1,4-$ 糖苷链，同时将游离出来的葡萄糖残基转移到一个葡萄糖分子或麦芽糖、麦芽三糖分子上，形成 $\alpha -1,6$ 键，生成异麦芽糖、潘糖、异麦芽三糖等含 $\alpha -1,6$ 键的寡糖的酶。

# A.7.2 转葡糖苷酶活力

在 $40^{\circ}\mathrm{C},\mathrm{pH}5.0$ 的条件下， $1\mathrm{mL}$ 酶样品与底物 $\alpha$ -甲基-D-葡萄糖苷起反应， $60\mathrm{min}$ 生成 $1\mu \mathrm{g}$ 的葡

萄糖，即为1个酶活力单位，以 $\mathrm{U / mL}$ 或 $\mathrm{U / g}$ 表示。

# A.7.3 原理

转葡糖苷酶作用于底物 $\alpha$ -甲基-D-葡萄糖苷，生成的葡萄糖与含有葡萄糖氧化酶、过氧化酶的4-氨基安替比林和酚试剂进行显色反应定量测定。

A.7.4.6 $5 \%$ 苯酚溶液: 将苯酚 $\left(\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{OH}\right) 5 \mathrm{~g}$ 于 $60^{\circ} \mathrm{C}$ 溶于 $50 \mathrm{~mL}$ 水, 将溶液冷却至室温, 用水稀释至 $100 \mathrm{~mL}$ 。

A.7.4.7 4-氨基安替比林-苯酚显色剂: 在 Tris-磷酸缓冲液 $40 \mathrm{~mL}$ 中, 加入葡萄糖氧化酶 $(5 \mathrm{mg}$ 葡萄糖氧化酶“Amano”) $550 \mathrm{U}$ 和过氧化物酶 $(0.76 \mathrm{mg}, 165 \mathrm{U} / \mathrm{mg}$ 的过氧化物酶“Amano”) $125 \mathrm{U}$ , 再加入 4-氨基安替比林溶液 $1 \mathrm{~mL}$ 和苯酚溶液 $1.4 \mathrm{~mL}$ , 用 Tris-磷酸缓冲液稀释至 $50 \mathrm{~mL}$ (随配随用)。

A.7.4.8 底物溶液：将 $\alpha$ -甲基葡萄糖 $(\mathrm{C}_7\mathrm{H}_{14}\mathrm{O}_6)2.0\mathrm{g}$ 溶解于水 $50~\mathrm{mL}$ 中，用水稀释至 $100~\mathrm{mL}(5^{\circ}\mathrm{C} \sim 15^{\circ}\mathrm{C}$ 可稳定两周）。

# A.7.5 分析步骤

# A.7.5.1 样品溶液的制备

称取 $1\mathrm{g}$ 酶样，精确至 $\pm 0.0001\mathrm{g}$ 或吸取酶样 $1\mathrm{mL}$ ，精确至 $0.01\mathrm{mL}$ ，加到适当大小的容量瓶中，用经冷却的水稀释至刻度，混匀。

注：配制的样品溶液使其 $0.5\mathrm{mL}$ 的 $A_{60} \sim A_0$ 介于 $0.15 \sim 0.32$ 之间。

# A.7.5.2 测定

吸取底物溶液 $1\mathrm{mL}$ 和 $0.02\mathrm{mol} / \mathrm{L}$ 乙酸-乙酸钠溶液 $1\mathrm{mL}$ 加入 $15\mathrm{mm}\times 150\mathrm{mm}$ 的试管中，在 $40^{\circ}\mathrm{C}\pm 0.5^{\circ}\mathrm{C}$ 的恒温水浴箱中保温 $10\mathrm{min}$ 。加入样品溶液 $0.5\mathrm{mL}$ ，混匀，于 $40^{\circ}\mathrm{C}\pm 0.5^{\circ}\mathrm{C}$ 的恒温水浴箱中准确保温 $60\mathrm{min}$ 后，将试管转移至沸水浴中加热 $5\mathrm{min}$ ，然后用流水快速冷却。冷却后，吸取此溶液 $0.1\mathrm{mL}$ 到试管中，并加入4-氨基安替比林-苯酚显色剂 $3\mathrm{mL}$ ，混匀。将此试管放入 $40^{\circ}\mathrm{C}\pm 0.5^{\circ}\mathrm{C}$ 的恒温水浴箱中，保温 $20\mathrm{min}$ ，测定 $500~\mathrm{nm}$ 处的吸光度 $A_{60}$ 。

空白对照：吸取 $0.02\mathrm{mol} / \mathrm{L}$ 乙酸-乙酸钠溶液 $1\mathrm{mL}$ 和样品溶液 $0.5\mathrm{mL}$ 加入 $15\mathrm{mm}\times 150\mathrm{mm}$ 的试管中，混匀。将试管转移至沸水浴中加热 $5\mathrm{min}$ ，然后用流水快速冷却。冷却后，加入底物溶液 $1\mathrm{mL}$ ，混匀。吸取此溶液 $0.1\mathrm{mL}$ 到空试管中，加入4-氨基安替比林-苯酚显色剂 $3\mathrm{mL}$ ，混匀。将此试管放入 $40^{\circ}\mathrm{C}\pm 0.5^{\circ}\mathrm{C}$ 的恒温水浴箱中，保温 $20\mathrm{min}$ ，测定 $500~\mathrm{nm}$ 处的吸光度 $\mathrm{A}_{\mathrm{s0}}$ 。

精确称取 $105\,^{\circ}\mathrm{C}$ 下干燥 $6\,\mathrm{h}$ 的葡萄糖 $\left(\mathrm{C}_{6} \mathrm{H}_{12} \mathrm{O}_{6}\right) 1000\,\mathrm{g}$ , 溶于 $100\,\mathrm{mL}$ 水中, 分别取 $1\,\mathrm{mL}, 2\,\mathrm{mL}, 3\,\mathrm{mL}, 4\,\mathrm{mL}, 5\,\mathrm{mL}$ , 用水定容至 $100\,\mathrm{mL}$ (每 $1\,\mathrm{mL}$ 该溶液分别含有 $100\,\mu\mathrm{g}, 200\,\mu\mathrm{g}, 300\,\mu\mathrm{g}, 400\,\mu\mathrm{g}$ 和

$500\mu \mathrm{g}$ 的葡萄糖）。

分别吸取以上葡萄糖标准液 $0.1\mathrm{mL}$ 和4-氨基安替比林-苯酚显色剂 $3\mathrm{mL}$ 加入 $15\mathrm{mm}\times 150\mathrm{mm}$ 的试管中，混匀。将这些试管放入 $40^{\circ}\mathrm{C}\pm 0.5^{\circ}\mathrm{C}$ 的恒温水浴箱中，保温 $20\mathrm{min}$ ，测定 $500\mathrm{nm}$ 处的吸光度 $A_{s10}, A_{s20}, A_{s30}, A_{s40}, A_{s50}$ 。

标准液的空白对照：用水来替代葡萄糖标准液，按上述方法测定空白对照液吸光度 $A_{\mathrm{s0}}$ 。

# A.7.6 结果计算

转葡糖苷酶活力 $X_{8}$ ，单位为 $\mathrm{U / mL}$ 或 $\mathrm{U / g}$ ，按式(A.8)计算：

$$
X _ {8} = \left(A _ {6 0} - A _ {\mathrm {s} 0}\right) \times G \times \frac {2 . 5}{0 . 1} \times \frac {n}{0 . 5} \quad \dots \tag {A.8}
$$

其中， $G$ 按式(A.9)计算：

$$
G = \frac {\frac {1 0}{A _ {\mathrm {s} 1 0} - A _ {\mathrm {s} 0}} + \frac {2 0}{A _ {\mathrm {s} 2 0} - A _ {\mathrm {s} 0}} + \frac {3 0}{A _ {\mathrm {s} 3 0} - A _ {\mathrm {s} 0}} + \frac {4 0}{A _ {\mathrm {s} 4 0} - A _ {\mathrm {s} 0}} + \frac {5 0}{A _ {\mathrm {s} 5 0} - A _ {\mathrm {s} 0}}}{5} \quad \dots \dots (\text {A . 9})
$$

式中：

$G$ ——吸光度差为1.000时，由葡萄糖标准曲线求得的葡萄糖量，单位为微克 $(\mu \mathrm{g})$ ；

2.5——反应体系的总容量，单位为毫升 $(\mathrm{mL})$ ；

0.1——反应体系的取样量，单位为毫升 $(\mathrm{mL})$ ；

$n$ ——酶样品的稀释倍数；

0.5——反应体系中酶样品的添加量，单位为毫升 $(\mathrm{mL})$ ；

$A_{\mathrm{sr}}$ 各反应液对应的吸光度值；

$A_{\mathrm{s0}}$ ——空白对照液液的吸光度值。

平行试验相对误差不得超过 $5\%$ 。

# A.7.7 转葡糖苷酶中葡糖淀粉酶活力的测定

# A.7.7.1 原理

酶作用于淀粉溶液生成还原糖。然后加入费林试剂，在加热的情况下，定量生成氧化亚铜沉淀。在加入碘化钾和硫酸后，生成游离碘，此时，立即用硫代硫酸钠溶液滴定。

# A.7.7.2 试剂和材料

A.7.7.2.1 $30\%$ 碘化钾溶液：碘化钾 $150\mathrm{g}$ 溶解于 $350~\mathrm{mL}$ 水，保存在褐色试剂瓶中，避免阳光直射。

A.7.7.2.2 $25\%$ 硫酸溶液：硫酸 $125\mathrm{g}$ 溶于 $375~\mathrm{mL}$ 水。

A.7.7.2.3 $0.05 \mathrm{~mol} / \mathrm{L}$ 硫代硫酸钠溶液: 将定量分析用的 $0.1 \mathrm{~mol} / \mathrm{L}$ 硫代硫酸钠溶液 $500 \mathrm{~mL}$ 用煮沸过的冷却水稀释, 所得溶液的总体积为 $1000 \mathrm{~mL}$ 。配制好后, 应进行标定, 求出浓度校正系数 $f$ 值。

A.7.7.2.4 $1\mathrm{mol} / \mathrm{L}$ 乙酸-乙酸钠缓冲溶液(pH4.5)：将 $1\mathrm{mol} / \mathrm{L}$ 乙酸钠溶液加入到 $1\mathrm{mol} / \mathrm{L}$ 乙酸溶液中，调 $\mathrm{pH}$ 到4.5。

A.7.7.2.5 可溶性淀粉溶液 $\left(\mathrm{pH}4.5\right)$ : 将可溶性淀粉(试剂级)在 $105^{\circ} \mathrm{C}$ 干燥 $4 \mathrm{~h}$ 后称量计算含水量。然后, 根据可溶性淀粉的含水量称取 $0.50 \mathrm{~g}$ (折干) 的可溶性淀粉, 缓慢加入到沸水 $50 \mathrm{~mL}$ 中, 煮沸 $5 \mathrm{~min}$ , 用自来水冷却后, 加入 $1 \mathrm{~mol} / \mathrm{L}$ 乙酸-乙酸钠缓冲溶液 $5 \mathrm{~mL}$ , 用水定容到 $100 \mathrm{~mL}$ 。

A.7.7.2.6 费林试剂：称取硫酸铜 $34.66\mathrm{g}$ 溶解于水中，定容至 $500~\mathrm{mL}$ 。另称取酒石酸钾钠 $173\mathrm{g}$ 和氢

氧化钠 $50\mathrm{g}$ 溶解于水中，定容至 $500~\mathrm{mL}$ 。使用前，精确地取等体积的上述两种溶液，充分混合。

# A.7.7.3 分析步骤

# A.7.7.3.1 样品溶液的制备

用水稀释酶样品，使得到酶液的 $(T_{0} - T_{30})\times f\times 1.62$ 值为 $3\mathrm{mg}\sim 10\mathrm{mg}$ 葡萄糖量。

# A.7.7.3.2 测定

将可溶性淀粉溶液 $10\mathrm{mL}$ 加入到 $100~\mathrm{mL}$ 锥形瓶中，置于 $40^{\circ}\mathrm{C} \pm 0.5^{\circ}\mathrm{C}$ 的恒温水浴箱中，预热 $10\mathrm{min} \sim 15\mathrm{min}$ ，加入样品稀释酶液 $1\mathrm{mL}$ 。准确加热 $30\mathrm{min}$ 后，加入费林试剂 $4\mathrm{mL}$ 使酶失活。将锥形瓶直接在煤气喷灯(或电炉)上加热 $2\mathrm{min}$ 后，立即放在自来水中冷却。随后，加入 $30\%$ 碘化钾溶液 $2\mathrm{mL}$ ，用硫代硫酸钠溶液滴定游离出的碘，以蓝色消失为滴定终点 $T_{30}(\mathrm{mL})$ 。

空白对照试验：以水取代酶液，在另一个锥形瓶中用上述同样的操作步骤测定空白对照值 $T_{0}$ (mL)。临近终点时，加入 $1\%$ 可溶性淀粉溶液[应使用可溶性淀粉(试剂级)另行配制]1滴～2滴，以蓝色消失作为滴定终点。

# A.7.7.4 结果计算

在上述试验条件下，反应 $30\mathrm{min}$ ，反应液中产生相当于 $10\mathrm{mg}$ 葡萄糖的还原糖所需的酶量，定义为1个葡糖淀粉酶活力单位，转葡糖苷酶中葡糖淀粉酶活力 $X_{9}$ ，单位为 $\mathrm{U / mL}$ 或 $\mathrm{U / g}$ ，按式(A.10)计算：

$$
X _ {9} = \left(T _ {0} - T _ {3 0}\right) \times f \times 1. 6 2 \times \frac {1}{1 0} \times n \quad \dots \tag {A.10}
$$

式中：

$T_{0}$ ——空白溶液滴定消耗硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$T_{30}$ ——酶反应液滴定消耗硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——0.05 mol/L 硫代硫酸钠标准溶液浓度的矫正系数；

1.62 ——换算系数；

1/10 ——该分析方法的常数（相当于 $10\mathrm{mg}$ 葡萄糖的还原糖）；

$n$ ——样品的稀释倍数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

# A.8 $\pmb{\alpha}$ -淀粉酶酶活力测定（米曲霉来源）

# A.8.1 $\pmb{\alpha}$ -淀粉酶

可以水解胶凝淀粉、直链淀粉和支链淀粉水溶液内部的 $\alpha -1,4$ 葡萄糖苷键，产生麦芽糖、寡糖和少量葡萄糖的酶。

加入碘化钾和硫酸后，生成游离碘，此时，立即用硫代硫酸钠滴定。

# A.8.3.1 试剂和材料

# A.9 普鲁兰酶活力的测定

# A.9.1.1 普鲁兰酶

能够专一性切开支链淀粉分支点中的 $\alpha -1$ ，6-糖苷键，从而剪下整个侧枝，形成直链淀粉的脱支酶。

# A.9.1.2 普鲁兰酶活力（比色法）

在给定的反应条件下， $1\mathrm{mL}$ 或 $1\mathrm{g}$ 产品每分钟反应产生相当于 $1\mu \mathrm{mol}$ 葡萄糖，即为一个酶活力单位，以 $\mathrm{U / mL}$ 或 $\mathrm{U / g}$ 表示。

# A.9.1.3 原理

普鲁兰酶在一定条件下催化水解普鲁兰多糖生成葡萄糖等还原糖，3,5-二硝基水杨酸与还原糖溶液共热后可生成显棕红色的氨基络合物，在一定范围内颜色深浅与产生的还原糖的量成正比。在 $550~\mathrm{nm}$ 的波长下测定反应溶液的吸光度，测定值与样品中普鲁兰酶的活力成正比。

注：样品中其他可水解普鲁兰多糖的糖酶(如葡糖淀粉酶)的存在会干扰本方法。加入一定量的阿卡波糖可以抑制样品中葡糖淀粉酶的活力，此时方法仍适用。

# A.9.1.4 试剂和材料

A.9.1.4.1 乙酸-乙酸钠缓冲液 $(0.2\mathrm{mol} / \mathrm{L},\mathrm{pH}4.5)$ ：准确称取无水乙酸钠 $4.92\mathrm{g}$ 溶于水中，加冰乙酸 $4\mathrm{mL}$ ，用蒸馏水溶解并定容至 $1000~\mathrm{mL}$ ，配好后用酸度计校正溶液的 $\mathrm{pH}$ 至4.5，用乙酸或乙酸钠进行调节。

注：如样品中含有葡糖淀粉酶时,按 $0.03\%$ 的浓度在缓冲溶液中加入阿卡波糖。

# A.9.1.6.1 葡萄糖标准曲线的绘制

葡萄糖标准曲线的绘制按如下操作：

a) 分别吸取 $0.1\%$ 葡萄糖标准溶液 $0.2 \mathrm{~mL}, 0.4 \mathrm{~mL}, 0.6 \mathrm{~mL}, 0.8 \mathrm{~mL}, 1.0 \mathrm{~mL}, 1.2 \mathrm{~mL}$ 和 $1.4 \mathrm{~mL}$ ,分别加入到7个刻度试管中,用蒸馏水补加至 $2.0 \mathrm{~mL}$ ,配制成每毫升分别含有葡萄糖 $100 \mu \mathrm{g}$ , $200 \mu \mathrm{g}, 300 \mu \mathrm{g}, 400 \mu \mathrm{g}, 500 \mu \mathrm{g}, 600 \mu \mathrm{g}$ 和 $700 \mu \mathrm{g}$ 的标准溶液;  
b）试管中各加入DNS试剂 $3\mathrm{mL}$ ，于沸水中沸腾 $7\mathrm{min}$ (沸腾时开始算起)，取出，迅速冷却到室温，加入蒸馏水 $10\mathrm{mL}$ ，混匀；  
c）用分光光度计在 $550~\mathrm{nm}$ 下测量溶液的吸光度，用空白管溶液调零点，记录吸光度值，以吸光度为纵坐标，以对应的标准葡萄糖浓度为横坐标，绘制标准曲线。

注：以 $0.5\mathrm{mL}$ 蒸馏水代替 $0.5\mathrm{mL}$ 标准葡萄糖液作为空白对照。

# A.9.1.6.2 样品的制备

称取 $1\mathrm{g}$ 酶样，精确至 $\pm 0.0001\mathrm{g}$ 或吸取酶样 $1\mathrm{mL}$ ，精确至 $0.01\mathrm{mL}$ ，用乙酸-乙酸钠缓冲液溶解，磁力搅拌混匀，准确稀释定容（使试样液与空白液的吸光度之差恰好落在 $0.3\sim 0.4$ 范围内），放置 $10\mathrm{min}$ ，待测。如样品中含有葡糖淀粉酶，必须使用含有阿卡波糖的缓冲溶液。

# A.9.1.6.3 测定

取四支 $25~\mathrm{mL}$ 刻度具塞试管(一支空白管，三支样品管)；分别向四支管中，准确加入预热的 $1.00~\mathrm{mL}$ 普鲁兰多糖溶液，将试管置于 $60^{\circ}\mathrm{C} \pm 0.1^{\circ}\mathrm{C}$ 水浴中预热 $10\mathrm{min}$ ；按一定的时间间隔向三支样品管中分别加入 $1.0~\mathrm{mL}$ 稀释好的待测酶液（空白管不加)，如样品中含有葡糖淀粉酶，可在试管中再加入 $40~\mu \mathrm{L}$ 的阿卡波糖溶液，准确计时，反应 $30\mathrm{min}$ 后依次取出；立即向各试管中加入 $3.0~\mathrm{mL}$ DNS试剂，震荡混匀；在空白管中加入 $1.0~\mathrm{mL}$ 待测酶液；将四支管同时放入沸水浴中，加热 $7\mathrm{min}$ 后取出，迅速冷却至室温，加 $10~\mathrm{mL}$ 水，摇匀；以空白管中溶液(对照液)调仪器零点，用分光光度计在波长 $550~\mathrm{nm}$ 下测量溶液的吸光度，取平均值；以吸光度平均值查标准曲线或用线性回归方程求出葡萄糖的含量。

# A.9.1.7 酶活力的计算

普鲁兰酶活力(比色法) $X_{11}$ ，单位为 $\mathrm{U / mL}$ 或 $\mathrm{U / g}$ ，按式(A.12)计算：

$$
X _ {1 1} = \frac {(A - b) \times n}{k \times 3 0 \times 1 8 0} \quad \dots \tag {A.12}
$$

式中：

$A$ ——样品在 $550 \mathrm{~nm}$ 处吸光度的平均值；

$b$ ——标准曲线的截距；

$n$ ——稀释倍数；

$k$ ——标准曲线的斜率；

30 ——酶反应时间，单位为分钟（min）；

180——葡萄糖的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# A.9.1.8 普鲁兰酶活力的测定[染色普鲁兰（红）法)]

# A.9.1.8.1 测定原理

普鲁兰酶在一定条件下催化水解染色红普鲁兰多糖，产生红色降解产物。用乙醇沉淀未降解底物。在 $510~\mathrm{nm}$ 下，用分光光度计测定上清液的吸光度，测定值与样品中普鲁兰酶活力成正比。

注：样品中其他可水解红普鲁兰多糖的糖酶(如葡糖淀粉酶)的存在会干扰本方法。加入一定量的阿卡波糖可以抑制样品中葡糖淀粉酶的活力，此时方法仍适用。

# A.9.1.8.2 试剂和材料

# A.9.1.8.2.1 缓冲液

注：如样品中含有葡糖淀粉酶时,按 $0.03\%$ 的浓度在缓冲溶液中加入阿卡波糖。

b）将试管置于一定温度的水浴中，预热 $5\mathrm{min}$

e）将试管在室温下放置一定时间后离心分离( $4000\mathrm{r / min}$ 时约 $10\mathrm{min}$ )；  
f) 取上清液用分光光度计在 $510 \mathrm{~nm}$ 处测定溶液的吸光度, 用水调零。

$$
X _ {1 2} = \frac {S \times V \times F}{W} \quad \dots \tag {A.13}
$$

式中：

$S$ ——由线性回归计算出的样品稀释液的酶活力；

$V$ ——溶解样品用的容量瓶体积，单位为毫升 $(\mathrm{mL})$ ；

$F$ ——稀释因子；

$W$ ——样品称量。

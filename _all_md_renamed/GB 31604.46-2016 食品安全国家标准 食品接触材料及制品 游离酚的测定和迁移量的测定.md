2016-10-19 发布

2017-04-19 实施

# 前言

本标准代替GB/T5009.69—2008《食品罐头内壁环氧酚醛涂料卫生标准的分析方法》和GB/T5009.99—2003《食品容器及包装材料用聚碳酸酯树脂卫生标准的分析方法》。

本标准与GB/T5009.69—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品接触材料及制品 游离酚的测定和迁移量的测定”；  
——修改了标准的适用范围；  
——增加了方法的检出限；  
——删除了极谱法。

# 食品安全国家标准

# 食品接触材料及制品 游离酚的测定和迁移量的测定

# 1 范围

本标准规定了食品接触材料及制品（水基改性环氧树脂涂料、环氧酚醛涂料、聚碳酸酯）游离酚的测定方法及迁移量的测定方法。

本标准适用于食品接触材料及制品（水基改性环氧树脂涂料、环氧酚醛涂料、聚碳酸酯）游离酚的测定及迁移量的测定。

# 游离酚的测定

# 2 原理

利用溴与酚结合成三溴苯酚，剩余的溴与碘化钾作用，析出定量的碘，最后用硫代硫酸钠滴定析出的碘，根据硫代硫酸钠溶液消耗的量，即可计算出酚的含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 盐酸（HCl）。  
3.1.2 乙醇 $\mathrm{(CH_3OH)}$ 。  
3.1.3 三氯甲烷 $\mathrm{(CHCl_3)}$   
3.1.4 碘化钾(KI)。  
3.1.5 可溶性淀粉 $\left[\left(\mathrm{C}_{6} \mathrm{H}_{10} \mathrm{O}_{5}\right)_{n}\right]$ 。  
3.1.6 溴水 $(\mathrm{Br}_2)$ 。  
3.1.7 无水碳酸钠 $\mathrm{(Na_2CO_3)}$ 。

# 3.2 试剂配制

3.2.1 碘化钾溶液 $(100\mathrm{g / L})$ ：称取 $1.0\mathrm{g}$ 碘化钾，用水溶解并稀释至 $10~\mathrm{mL}$ 。  
3.2.2 淀粉指示液 $(10\mathrm{g / L})$ ：称取 $1.0\mathrm{g}$ 可溶性淀粉，加少量水调至糊状，然后倒入 $100~\mathrm{mL}$ 沸水中，煮沸片刻，现用现配。

# 3.3 标准品

3.3.1 硫代硫酸钠 $\mathrm{(Na_2S_2O_3\cdot 5H_2O)}$ ：优级纯。  
3.3.2 溴酸钾 $\mathrm{(KBrO_3)}$ ：优级纯。

# 3.3.3 溴化钾(KBr)：优级纯。

# 3.4 标准溶液配制

# 3.4.1 硫代硫酸钠标准滴定溶液 $[c(\mathrm{Na}_2\mathrm{S}_2\mathrm{O}_3) = 0.1\mathrm{mol} / \mathrm{L}]$

称取 $26\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )硫代硫酸钠 $\mathrm{(Na_2S_2O_3\cdot 5H_2O)}$ ，加 $0.2\mathrm{g}$ 无水碳酸钠，溶于 $1000~\mathrm{mL}$ 水中，缓缓煮沸 $10\mathrm{min}$ ，冷却，放置两周后过滤，使用前按照按GB/T601标定。

# 3.4.2 溴标准溶液 $\left[c\left(\frac{1}{2}\mathbf{Br}_2\right) = 0.1\mathbf{mol} / \mathbf{L}\right]$

称取 $3.0\mathrm{g}$ 溴酸钾及 $25.0\mathrm{g}$ 溴化钾，溶于 $1000~\mathrm{mL}$ 水中，摇匀，使用前按照按GB/T601标定。

# 4 仪器和设备

天平：感量为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。

# 5 分析步骤

# 5.1 试样蒸馏

称取 $1\mathrm{g}$ （精确至 $0.001\mathrm{g}$ 试样，放入蒸馏瓶内，加入 $20~\mathrm{mL}$ 乙醇溶解（如水溶性树脂，则加入 $20~\mathrm{mL}$ 水溶解），再加入 $50~\mathrm{mL}$ 水，然后用水蒸气加热蒸馏出游离酚，馏出溶液收集于 $500~\mathrm{mL}$ 容量瓶中，控制在 $40\mathrm{min}\sim 50\mathrm{min}$ 内馏出蒸馏液 $300~\mathrm{mL}\sim 400~\mathrm{mL}$ ，最后取少量新蒸出液样，加1滴～2滴溴水，如无白色沉淀，证明酚已蒸完，即可停止蒸馏，蒸馏液用水稀释至刻度，充分摇匀，备用。

# 5.2 滴定测定

吸取 $100~\mathrm{mL}$ 蒸馏液，置于 $500~\mathrm{mL}$ 具塞锥形瓶中，加入 $25~\mathrm{mL}$ 溴标准溶液、 $5~\mathrm{mL}$ 盐酸，在室温下放入暗处 $15~\mathrm{min}$ ，加入 $10~\mathrm{mL}$ 碘化钾溶液，在暗处放置 $10~\mathrm{min}$ ，加 $1~\mathrm{mL}$ 三氯甲烷，用硫代硫酸钠标准滴定溶液滴定至淡黄色，加 $1~\mathrm{mL}$ 淀粉指示液，继续滴定至蓝色消褪为终点。

# 5.3 空白实验

同时取 $20~\mathrm{mL}$ 乙醇加水稀释至 $500~\mathrm{mL}$ ，然后吸取 $100~\mathrm{mL}$ 稀释后的溶液（如水溶性树脂，则吸取 $100~\mathrm{mL}$ 水），按照步骤5.2方法进行测定。

# 6 分析结果的表述

试样中游离酚的含量按式(1)计算：

$$
X = \frac {0 . 0 1 5 6 8 \times \left(V _ {1} - V _ {2}\right) \times c \times 5}{m} \times 1 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——试样中游离酚的含量，单位为克每百克 $(\mathrm{g} / 100\mathrm{g})$ ；

0.01568——与 $1.0~\mathrm{mL}$ 硫代硫酸钠标准溶液 $\left[c\left(\mathrm{Na}_{2}\mathrm{S}_{2}\mathrm{O}_{3}\right)=1.000\mathrm{mol} / \mathrm{L}\right]$ 相当苯酚的质量，单位为克每摩尔 $(\mathrm{g / mol})$ ；

$V_{1}$ ——试剂空白滴定消耗的硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——滴定试样消耗硫代硫酸钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c$ —硫代硫酸钠标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

100 ——换算系数。

结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $5\%$ 。

# 8 其他

本方法的检出限为 $0.02\mathrm{g} / 100\mathrm{g}$ ，定量限为 $0.04\mathrm{g} / 100\mathrm{g}$ 。

# 游离酚迁移量的测定

# 9 原理

在碱性条件 $(\mathrm{pH}9\sim 10.5)$ 下，酚与4-氨基安替吡啉经铁氰化钾氧化，生成红色的安替吡啉，红色的深浅与酚的含量成正比。用有机溶液萃取，以提高灵敏度，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 水基食品模拟物，所用试剂依据GB31604.1的规定。  
10.1.2 磷酸 $(\mathrm{H}_3\mathrm{PO}_4)$ 。  
10.1.3 三氯甲烷 $\mathrm{(CHCl_3)}$ 。  
10.1.4 硫酸铜 $\mathrm{(CuSO_4)}$ 。  
10.1.5 4-氨基安替吡啉 $\mathrm{(C_{11}H_{13}N_3O)}$ 。  
10.1.6 铁氰化钾 $\mathrm{K}_3[\mathrm{Fe}(\mathrm{CN})_6]$   
10.1.7 氯化铵 $(\mathrm{NH_4Cl})$ 。  
10.1.8 氢氧化钠(NaOH)。  
10.1.9 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。

# 10.2 试剂配制

10.2.1 水基食品模拟物，按GB5009.156的规定操作。  
10.2.2 磷酸溶液（ $10\%$ ，体积分数）：量取 $10~\mathrm{mL}$ 磷酸，用蒸馏水定容至 $100~\mathrm{mL}$ 。  
10.2.3 硫酸铜溶液 $(100\mathrm{g / L})$ ：称取 $10.0\mathrm{g}$ 硫酸铜，蒸馏水溶解并定容至 $100~\mathrm{mL}$ 。  
10.2.4 4-氨基安替吡啉溶液 $(20\mathrm{g / L})$ ：称取 $1.0\mathrm{g}4$ -氨基安替吡啉，蒸馏水溶解定容至 $50~\mathrm{mL},4^{\circ}\mathrm{C}$ 保存，一周内使用。  
10.2.5 氯化铵缓冲液 $(\mathrm{pH}9.8)$ ：称取 $20.0\mathrm{g}$ 氯化铵溶解于 $100~\mathrm{mL}$ 氨水中，密封， $4^{\circ}\mathrm{C}$ 保存。  
10.2.6 铁氰化钾溶液 $(80\mathrm{g / L})$ ：称取 $4.0\mathrm{g}$ 铁氰化钾，蒸馏水溶解并定容至 $50~\mathrm{mL}$ 。

10.2.7 氢氧化钠溶液 $(4\mathrm{g} / \mathrm{L})$ ：称取 $0.4\mathrm{g}$ 氢氧化钠，蒸馏水溶解并定容至 $100~\mathrm{mL}$ 。

# 10.3 标准品

苯酚 $\mathrm{(C_6H_5OH}$ ,CAS号：108-95-2)，纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 10.4 标准溶液配制

10.4.1 苯酚标准储备液 $(1000\mathrm{mg / L})$ ：称取苯酚标准品 $0.1\mathrm{g}$ （精确到 $1\mathrm{mg}$ )，用水溶解并转移至 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀，此溶液中苯酚浓度为 $1000\mathrm{mg / L}$ 。  
10.4.2 苯酚标准工作液 $(10\mathrm{mg / L})$ ：吸取 $1.0~\mathrm{mL}$ 苯酚标准储备溶液于 $100~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀，此溶液中苯酚浓度为 $10.0~\mathrm{mg / L}$ 。  
10.4.3 苯酚系列标准工作液：分别吸取 $0\mathrm{mL}, 1.0\mathrm{mL}, 2.0\mathrm{mL}, 5.0\mathrm{mL}, 10.0\mathrm{mL}$ 苯酚标准工作液于5个 $200\mathrm{mL}$ 容量瓶中，用水定容至刻度，标准系列工作液中苯酚的浓度分别为 $0.00\mathrm{mg/L}, 0.05\mathrm{mg/L}, 0.10\mathrm{mg/L}, 0.25\mathrm{mg/L}, 0.50\mathrm{mg/L}$ 。

# 11 仪器和设备

11.1 可见分光光度计。  
11.2 天平：感量为 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。

# 12 分析步骤

# 12.1 试样制备

# 12.1.1 水基食品模拟物

本标准食品模拟试验采用水基食品模拟物，所需试液通过迁移试验获取，可在 $4\mathrm{^{\circ}C}$ 冰箱中避光保存一周。

# 12.1.2 迁移试验

按照GB5009.156及GB31604.1的要求，对样品进行迁移试验，得到食品模拟物试液。如果得到的食品模拟物试液不能马上进行下一步试验，应将食品模拟物试液于 $4^{\circ}\mathrm{C}$ 冰箱中保存。所得食品模拟物试液应冷却或恢复至室温后进行下一步试验。

# 12.1.3 水基食品模拟物的处理

食品模拟物配制过程参照GB5009.156，量取 $200~\mathrm{mL}$ 经过迁移试验的食品模拟物试液于 $500~\mathrm{mL}$ 全磨口蒸馏瓶中，加入 $50~\mathrm{mL}$ 水、 $5\mathrm{mL}$ 硫酸铜溶液，用磷酸溶液调节 $\mathrm{pH}$ 在4以下[亦可用两滴甲基橙指示剂( $1\mathrm{g / L})$ 调至溶液为橙红色]，加入少量玻璃珠进行蒸馏，在 $200~\mathrm{mL}$ 容量瓶中预先放入 $5\mathrm{mL}$ 氢氧化钠溶液，将接收管插入氢氧化钠溶液液面下接收蒸馏液，收集蒸馏液至刻度，混匀。

# 12.2 空白试验

在不含游离酚的食品模拟物中，采用完全相同的分析步骤、试剂和用量，进行平行操作。

# 12.3 标准曲线的制作

分别将5个的苯酚系列标准工作液，全部转移到5个 $250~\mathrm{mL}$ 分液漏斗中，然后分别加入 $1\mathrm{mL}$ 氯

化铵缓冲液、1 mL 4-氨基安替吡啉溶液、1 mL 铁氰化钾溶液，每加入一种试剂，要充分摇匀，放置 $10\mathrm{min}$ ，各加入 $10\mathrm{mL}$ 三氯甲烷，振摇 $2\mathrm{min}$ ，静止分层后将三氯甲烷层经无水硫酸钠过滤于具塞比色管中，用 $2\mathrm{cm}$ 比色皿以零管调零，于波长 $460\mathrm{nm}$ 处测吸光度，绘制标准曲线。

# 12.4 样品测定

将制备的全部样品蒸馏液及试剂空白蒸馏液分别置于 $250~\mathrm{mL}$ 分液漏斗中，以下同12.3依次操作，处理后的样品于波长 $460~\mathrm{nm}$ 处测吸光度。

# 13 分析结果的表述

由标准曲线得到试样溶液中游离酚的浓度(扣除试剂空白值), 按 GB 5009.156 进行迁移量计算, 得到食品接触材料及制品中游离酚的迁移量。计算结果保留两位有效数字。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 15 其他

方法检出限为 $0.010\mathrm{mg / L}$ ，定量限为 $0.033\mathrm{mg / L}$ 。

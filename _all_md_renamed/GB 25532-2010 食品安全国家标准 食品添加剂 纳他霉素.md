# 检验方法

# A.1 一般规定

除非另有说明，在分析中仅使用确认为分析纯的试剂和GB/T6682—2008中规定的水。分析中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在没有注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。本试验所用溶液在未注明用何种溶剂配制时，均指水溶液。

# A.2 鉴别试验

# A.2.1 溶解性

难溶于水，微溶于甲醇。溶于冰乙酸及二甲基甲酰胺。

# A.2.2 颜色反应

取适量样品置于滴试板上，滴加盐酸，有蓝色产生；滴加磷酸，产生绿色，几分钟后变为淡红色。滴加酸的量以能够完全润湿或溶解纳他霉素的样品为度。

# A.2.3 紫外吸收光谱

准确称取 $25\mathrm{mg}$ 样品，转移至 $100\mathrm{mL}$ 容量瓶中，加 $5.0\mathrm{mL}$ 水湿润样品。加 $50\mathrm{mL}$ 冰乙酸-甲醇溶液[冰乙酸：甲醇 $= 1:1000$ (体积比)]，在避光条件下震摇至溶解。再用冰乙酸-甲醇溶液定容至刻度，混匀。移取 $2.0\mathrm{mL}$ 该溶液置于 $100\mathrm{mL}$ 容量瓶中，用冰乙酸-甲醇溶液定容至刻度，混匀。该溶液的紫外吸收光谱与标准样品溶液相比，具有相同的最大和最小吸收波长。

# A.3 纳他霉素的测定

# A.3.1 试剂和材料

a) 纳他霉素标准品。  
b) 乙腈：色谱纯。  
c) 乙酸铵。  
d) 氯化铵。  
e) 四氢呋喃。  
f) 甲醇。  
g) 盐酸溶液: $0.1 \mathrm{~mol} / \mathrm{L}$ 。

# A.3.2 仪器和设备

高效液相色谱仪：配紫外检测器（检测波长为 $303\mathrm{nm}$ ）。

# A.3.3 参考色谱条件

a) 色谱柱: $25 \mathrm{~cm} \times 4.6 \mathrm{~mm}$ (内径) 色谱柱, 内填充十八烷基硅胶; 或其他等效色谱柱。  
b) 流动相: 称取 $3.0 \mathrm{~g}$ 乙酸铵和 $1.0 \mathrm{~g}$ 氯化铵, 溶于 $760 \mathrm{~mL}$ 水中, 混匀。加 $5.0 \mathrm{~mL}$ 四氢呋喃和 $240 \mathrm{~mL}$ 乙腈, 混匀, 用 $0.45 \mu \mathrm{m}$ 的膜过滤器过滤。  
c) 柱温：室温。  
d) 流动相流速: $3 \mathrm{~mL} / \mathrm{min}$ 。  
e) 进样量: $20 \mu \mathrm{L}$

注：系统适用性为重复注入标准溶液三次，所得响应面积的相对平均偏差小于 $1.0\%$ 。

# A.3.4 分析步骤

# A.3.4.1 标准溶液的制备

称取约 $0.02\mathrm{g}$ 纳他霉素标准品，精确至 $0.0001\mathrm{g}$ ，置于 $100\mathrm{mL}$ 容量瓶中，加入 $5.0\mathrm{mL}$ 四氢呋喃，超声波震荡 $10\mathrm{min}$ 。加入 $60\mathrm{mL}$ 甲醇，旋转至溶解。加入 $25\mathrm{mL}$ 水，混匀后冷却至室温。加水定容至刻度，混匀，用 $0.45\mu \mathrm{m}$ 的膜过滤器过滤。

# A.3.4.2 试样液的制备

称取约 $0.02 \mathrm{~g}$ 试样（精确至 $0.0001 \mathrm{~g}$ ），转移至 $100 \mathrm{~mL}$ 容量瓶中，加入 $5.0 \mathrm{~mL}$ 四氢呋喃，超声震荡 $10 \mathrm{~min}$ 。加入 $60 \mathrm{~mL}$ 甲醇，旋转至溶解。加入 $25 \mathrm{~mL}$ 水，混匀后冷却至室温。加水定容至刻度，混匀，用 $0.45 \mu \mathrm{m}$ 的膜过滤器过滤。

# A.3.4.3 测定

在 A.3.3 参考色谱条件下，分别对标准溶液和试样液进行测定，记录主峰面积，根据公式计算出试样中纳他霉素的含量。

# A.3.5 结果计算

纳他霉素的含量 $X_{1}$ 按公式（A.1）计算：

$$
X _ {1} = \left(\frac {M _ {S} \times C _ {S}}{M _ {U} \times \left(1 - X _ {0}\right)}\right) \times \left(\frac {A _ {U}}{A _ {S}}\right) \times 100 \% \dots \tag{A.1}
$$

式中：

$X_{I}$ ——试样中纳他霉素的含量， $\%$ ；

$M_{S}$ ——纳他霉素标准品的质量，单位为克（g）；

$C_{S}$ ——纳他霉素标准品中标示的纳他霉素的含量， $\%$ ；

$M_U$ ——试样的质量，单位为克（g）；

$X_{0}$ ——实测试样的干燥失重， $\%$ ；

$A_{U}$ ——试样液色谱分析得到的主峰面积；

$A_{S}$ ——标准溶液色谱分析得到的主峰面积。

实验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $2\%$ 。

# A.4 比旋光度的测定

A. 4.1 称取适量试样, 以冰乙酸为溶剂, 配制成 $10 \mathrm{mg} / \mathrm{mL}$ 的试样冰乙酸溶液, 用旋光仪进行测定,同时做空白对照。测定温度为 $20^{\circ} \mathrm{C} \pm 0.5^{\circ} \mathrm{C}$ 。

比旋光度 $\alpha_{\mathrm{m}} (20^{\circ} \mathrm{C}, \mathrm{D})$ 数值以 $(^{\circ}) \cdot \mathrm{dm}^{2} \cdot \mathrm{kg}^{-1}$ 表示, 按公式(A.2)计算:

$$
\alpha_ {m} (2 0 ^ {\circ} C, D) = \frac {\alpha}{l \rho_ {\alpha}} \quad \dots \tag {A.2}
$$

式中：

$\alpha$ 测得的旋光角，单位为度 $(^{\circ})$

$l$ ——旋光管的长度,单位为分米(dm);

$\rho_{\alpha}$ ——溶液中有效组分的质量浓度,单位为克每毫升 $(\mathrm{g} / \mathrm{mL})$ 。

A.4.2 其他按GB/T613-2007的规定进行。

# A. 5 pH的测定

称取适量试样，配制成 $10\mathrm{mg / mL}$ 的试样水溶液，用 $\mathsf{pH}$ 计测定 $\mathsf{pH}$ 。

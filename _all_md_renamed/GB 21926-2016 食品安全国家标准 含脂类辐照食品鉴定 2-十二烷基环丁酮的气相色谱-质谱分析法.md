# 前言

本标准代替GB/T21926—2008《辐照含脂食品中2-十二烷基环丁酮测定气相色谱/质谱法》。  
本标准与GB/T21926—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 含脂类辐照食品鉴定 2-十二烷基环丁酮的气相色谱-质谱分析法”；

——改进了硅胶柱层析法。

# 食品安全国家标准

# 含脂类辐照食品鉴定

# 2-十二烷基环丁酮的气相色谱-质谱分析法

# 1 范围

本标准规定了利用2-十二烷基环丁酮鉴定含脂辐照食品的气相色谱-质谱分析法。

本标准适用于脂肪含量大于 $1\%$ 的辐照食品的鉴定。

# 2 原理

采用正己烷将样品中的脂肪和2-十二烷基环丁酮一起萃取出来，经硅胶层析柱净化、浓缩后，用气相色谱-质谱仪测定2-十二烷基环丁酮，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的三级水。

# 3.1 试剂

3.1.1 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ 。  
3.1.2 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $650^{\circ}C$ 灼烧 $4\mathrm{h}$ ，干燥器中冷却，保存备用。  
3.1.3 乙醚 $(\mathrm{C_4H_{10}O})$ 。  
3.1.4 氮气 $(\mathrm{N}_2)$ ：纯度 $\geq 99.999\%$   
3.1.5 氢气（He）：纯度 $\geq 99.999\%$

# 3.2 材料

硅胶： $250\mu \mathrm{m}\sim 500\mu \mathrm{m}$ ，柱层析用。使用前，将硅胶置于 $100^{\circ}C$ 烘箱活化 $12\mathrm{h}$ ，干燥器中冷却。于具塞锥形瓶中加入 $4\%$ （质量比）水去活化，摇动锥形瓶至硅胶分散均匀。将去活化的硅胶于干燥器中平衡 $12\mathrm{h}$ ，一周内可用。

# 3.3 标准品

2-十二烷基环丁酮标准品 $\left(\mathrm{C}_{16} \mathrm{H}_{30} \mathrm{O}\right)$ ：纯度 $\geqslant 99 \%$ 。

# 3.4 标准溶液配制

# 3.4.1 2-十二烷基环丁酮标准储备溶液

准确称取适量2-十二烷基环丁酮标准品(精确至 $0.1\mathrm{mg}$ )，用正己烷溶解，配制成浓度为 $100~\mu \mathrm{g / mL}$ 的标准储备溶液， $-20^{\circ}C$ 贮存，3个月内有效。

# 3.4.2 2-十二烷基环丁酮标准工作溶液

用正己烷稀释标准储备溶液 $(100\mu \mathrm{g / mL})$ ，分别配制成 $0.10~\mu \mathrm{g / mL},0.25~\mu \mathrm{g / mL},0.50~\mu \mathrm{g / mL},1.00$ $\mu \mathrm{g / mL},2.50~\mu \mathrm{g / mL},5.00~\mu \mathrm{g / mL}$ 的各级标准工作溶液，一 $20^{\circ}C$ 贮存，1个月内有效，但出现谱峰异常时应考虑重新配制。

# 4 仪器和设备

4.1 气质联用仪：配备弱极性(如DB-5)或适当性质和规格的毛细管柱，柱长： $30\mathrm{m}$ ，内径： $0.25\mathrm{mm}$ ，膜厚： $0.25\mu \mathrm{m}$ 。  
4.2 索氏提取器。  
4.3 氮吹仪。  
4.4 电动搅拌机。  
4.5 恒温干燥箱。  
4.6 恒温水浴锅。  
4.7 旋转蒸发仪。  
4.8 电子天平：感量 $0.1\mathrm{mg}$ 。  
4.9 玻璃层析柱：柱长： $200\mathrm{mm}\sim 300\mathrm{mm}$ ，内径： $20~\mathrm{mm}$

# 5 分析步骤

# 5.1 试样制备

选取样品中带脂肪的部位，牛肉、猪肉等固体样品切成小块，鸡蛋等液态样品应干燥至半固体状。用电动搅拌机绞碎并混匀，置于密闭玻璃管中或金属箔内。

# 5.2 脂肪提取

称取 $20\mathrm{g}$ 无水硫酸钠和 $20\mathrm{g}$ 样品（可根据样品脂肪含量适当调整）放入纤维材质抽提套管中，混合后用脱脂棉塞住管口。向索氏提取器中加入 $100~\mathrm{mL}$ 正己烷，将抽提套管放入提取器中，再加入 $40~\mathrm{mL}$ 正己烷，以约 $1\mathrm{h}$ 虹吸4次的速度，加热回流萃取 $6\mathrm{h}$ 。

收集全部脂肪提取液，旋转蒸发浓缩至 $2\mathrm{mL}\sim 3\mathrm{mL}(45^{\circ}\mathrm{C}$ 水浴，约 $25\mathrm{kPa})$ 。将浓缩后的脂肪转移至具塞玻璃管中，氮吹浓缩至恒重。

# 5.3 硅胶柱层析

在玻璃层析柱中填充 $30\mathrm{g}$ 去活化的硅胶，硅胶顶部盖上 $1\mathrm{cm}$ 厚的无水硫酸钠，用2倍柱体积的正己烷预淋洗。称取5.2中制备的 $1.0\mathrm{g}$ 脂肪，溶于 $5\mathrm{mL}$ 正己烷，上柱。先用 $150~\mathrm{mL}$ 正己烷淋洗，流速控制在 $2\mathrm{mL} / \mathrm{min}$ ，弃去淋洗液。再用 $480~\mathrm{mL}$ 含有 $1\%$ 乙醚的正己烷溶液洗脱，弃去先流出的 $230~\mathrm{mL}$ 洗脱液，收集剩下的洗脱液。于 $40^{\circ}\mathrm{C}$ 水浴将洗脱液旋转蒸发浓缩至 $2\mathrm{mL}\sim 5\mathrm{mL}$ ，浓缩液转移至 $10~\mathrm{mL}$ 试管中，用 $2\mathrm{mL}\sim 5\mathrm{mL}$ 正己烷洗涤旋转蒸发瓶两次，洗涤液合并至试管中。于 $40^{\circ}\mathrm{C}$ 氮气吹至近干。用正己烷定容至 $0.5\mathrm{mL}$ ，转入进样瓶待测。

# 5.4 仪器参考条件

a）色谱柱升温程序： $120^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，然后以 $15^{\circ}\mathrm{C / min}$ 升温至 $160^{\circ}\mathrm{C}$ ；再以 $0.5^{\circ}\mathrm{C / min}$ 至 $175^{\circ}\mathrm{C}$ ；再以 $30^{\circ}\mathrm{C / min}$ 至 $290^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ ，检测器温度 $250^{\circ}\mathrm{C}$ ；

b）进样口温度： $250~^\circ \mathrm{C}$   
c） 接口温度： $280^{\circ}C$   
d) 离子源：EI源， $70\mathrm{eV}$ ；  
e）测定方式：选择离子检测(SIM)；监测离子 $(m / z):55,98,112$ ，定量离子 $(m / z):98$   
f）载气：氦气，流速 $1.0~\mathrm{mL / min}$   
g）进样量： $1.0\mu \mathrm{L}$   
h）进样方式：不分流进样。

# 6 分析结果的表述

对标准工作溶液与样液等体积参插进样测定，外标法定量。在上述色谱条件下，2-十二烷基环丁酮标准样品的选择离子色谱图见图A.1。2-十二烷基环丁酮的保留时间约为 $17.1\mathrm{min}$ ，质谱图见图A.2。

若样液中检出的2-十二烷基环丁酮色谱峰的保留时间与2-十二烷基环丁酮标准的色谱峰的保留时间一致(相对偏差 $\leqslant 2\mathrm{s}$ )，其质谱图与2-十二烷基环丁酮标准的质谱图基本一致，扣除背景后选择监测离子 $m / z55,98$ 和112均出现，且 $m / z55,112$ 与基峰 $m / z98$ 的相对丰度比与2-十二烷基环丁酮标准的相对丰度比的偏差不超过 $20\%$ ，则可确证样品中检出2-十二烷基环丁酮，表明该样品接受过辐照处理。

# 前言

本标准代替GB1886.40—2015《食品安全国家标准 食品添加剂 L-苹果酸》。

本标准与GB1886.40—2015相比，主要变化如下：

——更改了范围；  
——增加了理化指标水分、丁二酸；  
——更改了理化指标含量、总砷的表述和总砷限量要求，删除了重金属；  
——更改了鉴别、含量、澄清度、富马酸、马来酸、铅、总砷的检验方法；  
——增加了丁二酸的检验方法；  
——更改了附录A、附录B，增加了附录C。

# 附录A

# 检验方法

# A.1 警示

试验方法规定的一些试验过程可能导致危险情况。操作者应采取适当的安全和健康措施。

# A.2 一般规定

本标准所用试剂在未注明其他要求时，均指分析纯试剂。试验中所用标准滴定溶液、杂质测定用标准溶液、制剂及制品，在未注明其他要求时，均按GB/T601、GB/T602和GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指GB/T6682规定的三级水。

# A.3 鉴别试验

# A.3.1 试剂和材料

A.3.1.1 氨水溶液：氨水 $+$ 水 $= 2 + 3$   
A.3.1.2 对氨基苯磺酸溶液： $10\mathrm{g / L}$   
A.3.1.3 亚硝酸钠溶液： $200\mathrm{g / L}$   
A.3.1.4 氢氧化钠溶液： $40\mathrm{g / L}$

# A.3.2 分析步骤

# A.3.2.1 氨盐呈色鉴别

称取 $0.5\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 试管中，加入 $10~\mathrm{mL}$ 水溶解。用氨水溶液中和至中性，加入 $1\mathrm{mL}$ 对氨基苯磺酸溶液，在沸水浴中加热 $5\mathrm{min}$ 。加入 $5\mathrm{mL}$ 亚硝酸钠溶液，置于沸水浴加热 $3\mathrm{min}$ 后，加入 $5\mathrm{mL}$ 氢氧化钠溶液，从上至下观察，应立即显红色。

# A.3.2.2 红外吸收光谱鉴别

按照GB/T6040，采用溴化钾压片法定性分析，测定红外吸收光谱。试样的红外吸收光谱图应与附录B基本一致。

# A.4 含量（以 $\mathbf{C}_4\mathbf{H}_6\mathbf{O}_5$ 计）的测定

# A.4.1 方法提要

以酚酞为指示剂，用氢氧化钠标准滴定溶液滴定试样溶液，根据氢氧化钠标准滴定溶液的消耗量计算试样溶液中的总酸浓度（以 $\mathrm{C_4H_6O_5}$ 计）。

# A.4.2 试剂和材料

A.4.2.1 氢氧化钠标准滴定溶液： $c(\mathrm{NaOH}) = 0.5\mathrm{mol / L}$   
A.4.2.2 酚酞指示液： $10\mathrm{g / L}$

# A.4.3 仪器和设备

分析天平：感量为 $0.0001\mathrm{g}$

# A.4.4 分析步骤

称取约 $1.0\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，加入 $20~\mathrm{mL}$ 无二氧化碳的水溶解，并滴加2滴酚酞指示液，用氢氧化钠标准滴定溶液滴定至微红色，保持 $30~\mathrm{s}$ 不褪色为终点。按与试样相同的测定步骤进行空白试验。

# A.4.5 结果计算

含量(以 $\mathrm{C_4H_6O_5}$ 计)以质量分数 $\varpi_{1}$ 计，数值以百分含量 $(\%)$ 表示，按式(A.1)计算。

$$
w _ {1} = \frac {\left(V _ {1} - V _ {0}\right) \times c \times M}{m \times 1000} \times 100 \% \quad \dots \tag{A.1}
$$

式中：

$V_{1}$ ——试样溶液消耗氢氧化钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{0}$ ——空白溶液消耗氢氧化钠标准滴定溶液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$c$ ——氢氧化钠标准滴定溶液浓度的数值，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$M$ ——1/2L-苹果酸摩尔质量的数值，单位为克每摩尔 $(\mathrm{g / mol})$ （ $M = 67.04)$

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

试验结果以平行测定结果的算术平均值表示，计算结果保留至小数点后两位。

# A.4.6 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.2\%$ 。

# A.5 比旋光度的测定

# A.5.1 仪器和设备

A.5.1.1 旋光仪：备有钠光灯（钠光谱D线 $589.3\mathrm{nm}$ ），精度 $\pm 0.01^{\circ}$ 。

A.5.1.2 分析天平：感量为 $0.01\mathrm{g}$ 。

# A.5.2 分析步骤

称取 $4.25\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，加入 $20~\mathrm{mL}$ 水溶解并定容至 $50~\mathrm{mL}$ ，其余步骤按照GB/T613的规定执行。

# A.5.3 结果计算

比旋光度以 $\alpha_{\mathrm{m}}(20^{\circ}\mathrm{C},\mathrm{D})$ 计，数值以 $(^{\circ})\cdot \mathrm{dm}^{2}\cdot \mathrm{kg}^{-1}$ 表示，按式(A.2)计算。

$$
\alpha_ {\mathrm {m}} = \frac {\alpha}{l \times \rho} \quad \dots \tag {A.2}
$$

式中：

$\alpha$ ——试样溶液在 $20^{\circ}\mathrm{C}$ 测得的旋光度的数值，单位为度 $(^{\circ})$

$l$ ——旋光管长度的数值，单位为分米（dm）；

$\rho$ ——溶液中试样有效组分质量浓度的数值，单位为克每毫升 $(\mathrm{g / mL})$ 。

# A.6 澄清度的测定

# A.6.1 试剂和材料

A.6.1.1 硝酸溶液：硝酸 $+$ 水 $= 1 + 2$

A.6.1.2 硝酸银溶液： $20\mathrm{g / L}$   
A.6.1.3 糊精溶液： $20\mathrm{g / L}$   
A.6.1.4 盐酸标准溶液： $c(\mathrm{HCl}) = 0.1\mathrm{mol / L}$   
A.6.1.5 标准工作溶液(含氯 $0.01\mathrm{mg / mL}$ )：准确移取 $14.1~\mathrm{mL}$ 盐酸标准溶液，加水稀释并定容至 $50~\mathrm{mL}$ 。准确量取该溶液 $10.0~\mathrm{mL}$ ，加水稀释并定容至 $1000~\mathrm{mL}$ 。

# A.6.2 仪器和设备

A.6.2.1 纳氏比色管。  
A.6.2.2 分析天平：感量为 $0.01\mathrm{g}$

# A.6.3 分析步骤

# A.6.3.1 试样溶液的制备

称取约 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于纳氏比色管中，加入 $20~\mathrm{mL}$ 水溶解，摇匀。

# A.6.3.2 标准对照溶液的制备

准确量取 $0.20\mathrm{mL}$ 标准工作溶液，加水至 $20~\mathrm{mL}$ ，加入 $1\mathrm{mL}$ 硝酸溶液、 $1\mathrm{mL}$ 硝酸银溶液和 $0.2\mathrm{mL}$ 糊精溶液，摇匀，避光放置 $15\mathrm{min}$ 。

# A.6.4 结果判定

在无阳光直射的情况下，轴向及侧向观察。试样溶液的浊度不大于标准对照溶液的浊度，即为通过试验。

# A.7 灼烧残渣

称取 $2.5\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，其余步骤按照GB/T9741的规定执行。试验结果以平行测定结果的算术平均值为准，在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.02\%$ 。

# A.8 氯化物的测定

称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，准确移取 $0.4\mathrm{mL}$ 氯化物标准溶液，其余步骤按照GB/T9729的规定执行。试样溶液的浊度不大于标准溶液的浊度，即氯化物含量 $\leqslant 0.004\%$ 。

# A.9 硫酸盐的测定

# A.9.1 试剂和材料

A.9.1.1 盐酸溶液：盐酸 $+$ 水 $= 1 + 4$   
A.9.1.2 氯化钡溶液： $250~\mathrm{g / L}$   
A.9.1.3 硫酸盐标准溶液： $0.1\mathrm{mg / mL}$

# A.9.2 仪器和设备

A.9.2.1 纳氏比色管： $50~\mathrm{mL}$   
A.9.2.2 分析天平：感量为 $0.01\mathrm{g}$

# A.9.3 分析步骤

# A.9.3.1 试样溶液制备

称取 $1.0\mathrm{g}$ 试样，精确至 $0.01\mathrm{g}$ ，置于纳氏比色管中，加水溶解使成约 $40~\mathrm{mL}$ ，摇匀。

# A.9.3.2 对照溶液制备

准确移取 $2.0\mathrm{mL}$ 硫酸盐标准溶液，置于纳氏比色管中，加水溶解使成约 $40~\mathrm{mL}$ ，摇匀。

# A.9.3.3 测定

在试样溶液和对照溶液中分别加入 $0.5\mathrm{mL}$ 盐酸溶液和 $1\mathrm{mL}$ 氯化钡溶液，加水稀释至 $50~\mathrm{mL}$ ，充分摇匀后放置 $10\mathrm{min}$ 。

# A.9.4 结果判定

将试样溶液和对照溶液置于同一黑色背景上，从比色管上方向下观察所产生的浊度。试样溶液的浊度不大于对照溶液的浊度，即硫酸盐含量 $\leqslant 0.02\%$ 。

# A.10 富马酸、马来酸、丁二酸的测定

# A.10.1 原理

试样经处理后，采用氢型阳离子交换色谱柱分离，紫外检测器或二极管阵列检测器检测，外标法定量。

# A.10.2 试剂和材料

A.10.2.1 水：GB/T6682，一级水。

A.10.2.2 硫酸溶液： $c(1 / 2\mathrm{H}_2\mathrm{SO}_4) = 0.01\mathrm{mol} / \mathrm{L}$

A.10.2.3 富马酸标准品 $\mathrm{(C_4H_4O_4}$ ,CAS号：110-17-8)：纯度 $\geqslant 99.0\%$ ，或经国家认证并授予标准物质证书的标准物质。

A.10.2.4 马来酸标准品 $\left(\mathrm{C}_{4} \mathrm{H}_{4} \mathrm{O}_{4}\right)$ , CAS号: 110-16-7): 纯度 $\geqslant 99.0\%$ , 或经国家认证并授予标准物质证书的标准物质。

A.10.2.5 丁二酸标准品 $\mathrm{(C_4H_6O_4}$ ,CAS号：110-15-6)：纯度 $\geqslant 99.0\%$ ，或经国家认证并授予标准物质证书的标准物质。

A.10.2.6 富马酸标准溶液 $(500\mu \mathrm{g} / \mathrm{mL})$ ：准确称取 $50~\mathrm{mg}$ 富马酸标准品，精确至 $0.0001\mathrm{g}$ ，加硫酸溶液溶解并定容至 $100~\mathrm{mL}$ 。置于 $4^{\circ}\mathrm{C}$ 冰箱中保存，有效期6个月。

A.10.2.7 马来酸标准溶液 $(50\mu \mathrm{g} / \mathrm{mL})$ ：准确称取 $5\mathrm{mg}$ 马来酸标准品，精确至 $0.0001\mathrm{g}$ ，加硫酸溶液溶解并定容至 $100~\mathrm{mL}$ 。置于 $4^{\circ}\mathrm{C}$ 冰箱中保存，有效期6个月。

A.10.2.8 丁二酸标准溶液 $(5\mathrm{mg / mL})$ ：准确称取 $500~\mathrm{mg}$ 丁二酸标准品，精确至 $0.0001\mathrm{g}$ ，加硫酸溶液溶解并定容至 $100~\mathrm{mL}$ 。置于 $4^{\circ}\mathrm{C}$ 冰箱中保存，有效期6个月。

A.10.2.9 微孔滤膜： $0.22\mu \mathrm{m}$ ，水相。

# A.10.3 仪器和设备

A.10.3.1 高效液相色谱仪：配备紫外检测器或二极管阵列检测器。

A.10.3.2 分析天平：感量 $0.01\mathrm{g},0.0001\mathrm{g}$

# A.10.4 仪器参考条件

A.10.4.1 色谱柱：以聚苯乙烯-二乙烯基苯树脂为填料的氢型阳离子交换色谱柱， $300\mathrm{mm}\times 7.8\mathrm{mm}$ ，或其他具有同等分析效果的色谱柱。

A.10.4.2 柱温： $45~^\circ C$ 。

A.10.4.3 流动相：硫酸溶液。

A.10.4.4 流速： $0.5\mathrm{mL / min}$

A.10.4.5 进样量： $10~\mu \mathrm{L}$

A.10.4.6 检测波长： $214\mathrm{nm}$

# A.10.5 分析步骤

# A.10.5.1 混合标准溶液的制备

取5个 $100~\mathrm{mL}$ 容量瓶，分别准确吸取富马酸标准溶液、马来酸标准溶液 $1.00~\mathrm{mL},2.00~\mathrm{mL},4.00~\mathrm{mL}$ 产 $5.00~\mathrm{mL},6.00~\mathrm{mL}$ ，丁二酸标准溶液 $0.40~\mathrm{mL},1.00~\mathrm{mL},2.00~\mathrm{mL},3.00~\mathrm{mL},4.00~\mathrm{mL}$ ，置于容量瓶中，用硫酸溶液定容至刻度。配制成富马酸质量浓度分别为 $5.00~\mu \mathrm{g / mL},10.0~\mu \mathrm{g / mL},20.0~\mu \mathrm{g / mL},25.0~\mu \mathrm{g / mL}$ $30.0~\mu \mathrm{g / mL}$ ；马来酸质量浓度分别为 $0.50~\mu \mathrm{g / mL},1.00~\mu \mathrm{g / mL},2.00~\mu \mathrm{g / mL},2.50~\mu \mathrm{g / mL},3.00~\mu \mathrm{g / }$ mL;丁二酸质量浓度分别为 $20.0~\mu \mathrm{g / mL},50.0~\mu \mathrm{g / mL},100~\mu \mathrm{g / mL},150~\mu \mathrm{g / mL},200~\mu \mathrm{g / mL}$ 的混合标准溶液。置于 $4^{\circ}C$ 冰箱中保存，有效期1个月。

# A.10.5.2 试样溶液的制备

称取 $0.5\mathrm{g}$ 试样，精确至 $0.0001\mathrm{g}$ ，用硫酸溶液溶解并定容至 $100\mathrm{mL}$ （可适当调整试样溶液的定容体积，使试样溶液中富马酸、马来酸、丁二酸的质量浓度在绘制的标准曲线范围内），摇匀，经微孔滤膜过滤。

# A.10.6 测定

# A.10.6.1 标准曲线的绘制

在A.10.4条件下，将A.10.5.1中5个不同浓度的混合标准溶液分别注入液相色谱仪中，测得相应的峰面积，以混合标准溶液的质量浓度为横坐标，以峰面积为纵坐标，绘制标准曲线。

# A.10.6.2 试样溶液的测定

在A.10.4条件下，将试样溶液分别注入液相色谱仪中，以保留时间定性，测得富马酸、马来酸、丁二酸的峰面积，根据标准曲线得到试样溶液中的富马酸、马来酸、丁二酸的质量浓度。

# A.10.7 结果计算

富马酸、马来酸、丁二酸的含量以质量分数 $w_{i}$ 计，数值以百分含量 $(\%)$ 表示，按式(A.3)计算。

$$
w _ {i} = \frac {C _ {i} \times V}{m \times 10 ^ {6}} \times 100 \% \tag{A.3}
$$

式中：

$C_i$ ——由标准曲线得到的试样溶液中待测物质质量浓度的数值，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——试样溶液定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

$10^{6}$ ——换算系数。

试验结果以平行测定结果的算术平均值表示，保留两位有效数字。

# A.10.8 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# A.10.9 检出限与定量限

富马酸的检出限为 $0.0040\%$ ，定量限为 $0.010\%$ ；马来酸的检出限为 $0.0020\%$ ，定量限为 $0.0050\%$ ；丁二酸的检出限为 $0.080\%$ ，定量限为 $0.20\%$ 。

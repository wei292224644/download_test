# 前言

本标准代替GB/T5009.24-2003《食品中黄曲霉毒素 $\mathbf{M}_1$ 与 $\mathrm{B}_{1}$ 的测定》。

本标准所代替的历次版本发布情况为：

GB/T5009.24-1985、GB/T5009.24-1996、GB/T5009.24-2003。

# 食品安全国家标准

# 食品中黄曲霉毒素 $\mathrm{M}_{1}$ 和 $\mathrm{B}_{1}$ 的测定

# 1 范围

本标准规定了牛乳及其制品、奶油及新鲜猪组织（肝、肾、血及瘦肉）等食品中黄曲霉毒素 $\mathbf{M}_{1}$ 与 $\mathrm{B_1}$ 的测定方法。

本标准适用于牛乳及其制品、奶油及新鲜猪组织（肝、肾、血及瘦肉）等食品中黄曲霉毒素 $\mathbf{M}_{1}$ 与 $\mathbf{B}_1$ 的测定。

# 2 原理

样品经提取、浓缩、薄层分离后，黄曲霉毒素 $\mathbf{M}_1$ 与 $\mathbf{B}_1$ 在紫外光（波长 $365\mathrm{nm}$ ）下产生蓝紫色荧光，根据其在薄层上显示荧光的最低检出量来测定含量。

# 3 试剂和材料

3.1 甲醇：分析纯。  
3.2 石油醚：分析纯。  
3.3 三氯甲烷：分析纯。  
3.4 无水硫酸钠：分析纯。  
3.5 异丙醇：分析纯。  
3.6 硅胶G：层析用。  
3.7 氯化钠及氯化钠溶液（ $40\mathrm{g / L}$ ）。  
3.8 硫酸（1+3）。  
3.9 玻璃砂：用酸处理后洗净干燥，约相当20目。  
3.10 黄曲霉毒素 $\mathrm{M}_{1}$ 标准溶液：用三氯甲烷配制成每毫升相当于 $10\mu \mathrm{g}$ 的黄曲霉毒素 $\mathrm{M}_{1}$ 标准溶液。以三氯甲烷作空白试剂，黄曲霉毒素 $\mathrm{M}_{1}$ 的紫外最大吸收峰的波长应接近 $357~\mathrm{nm}$ ，摩尔消光系数为 19950。避光，置于 $4^{\circ} \mathrm{C}$ 冰箱中保存。  
3.11 黄曲霉毒素 $\mathrm{M}_{1}$ 与 $\mathrm{B}_{1}$ 混合标准使用液：用三氯甲烷配制成每毫升相当于各含 $0.04 \mu \mathrm{g}$ 黄曲霉毒素 $\mathrm{M}_{1}$ 与 $\mathrm{B}_{1}$ 。避光，置于 $4^{\circ} \mathrm{C}$ 冰箱中保存。

# 4 仪器和设备

4.1 10目圆孔筛。  
4.2 小型粉碎机。

4.3 玻璃板： $5\mathrm{cm}\times 20\mathrm{cm}$   
4.4 展开槽：长 $25\mathrm{cm}$ ，宽 $6\mathrm{cm}$ ，高 $4\mathrm{cm}$ 。  
4.5 紫外光灯： $100\mathrm{W}\sim 125\mathrm{W}$ ，带 $365\mathrm{nm}$ 滤光片。  
4.6 微量注射器。

# 5.2 净化

5.2.1 用石油醚分配净化：将以上收集的提取液移入 $250~\mathrm{mL}$ 分液漏斗中，再按各种食品加入一定体积的氯化钠溶液（ $40~\mathrm{g / L}$ ）（见表1）。再加入 $40~\mathrm{mL}$ 石油醚，振摇 $2\mathrm{min}$ ，待分层后，将下层甲醇-氯化钠水层移于原量筒中，将上层石油醚溶液从分液漏斗上口倒出，弃去。再将量筒中溶液转移于原分液漏斗中。再重复用石油醚提取两次，每次 $30~\mathrm{mL}$ ，最后将量筒中溶液仍移于分液漏斗中。奶油样液总共用石油醚提取两次，每次 $40~\mathrm{mL}$ 。  
5.2.2 用三氯甲烷分配提取：于原量筒中加入 $20 \mathrm{~mL}$ 三氯甲烷，摇匀后，再倒入原分液漏斗中，振摇 $2 \mathrm{~min}$ 。待分层后，将下层三氯甲烷移于原量筒中，再重复用三氯甲烷提取两次，每次 $10 \mathrm{~mL}$ 合并于原量筒中。弃去上层甲醇水溶液。  
5.2.3 用水洗三氯甲烷层与浓缩制备：将合并后的三氯甲烷层倒回原分液漏斗中，加入 $30 \mathrm{~mL}$ 氯化钠溶液（ $40 \mathrm{~g} / \mathrm{L}$ ），振摇 $30 \mathrm{~s}$ ，静置。待上层混浊液有部分澄清时，即可将下层三氯甲烷层收集于原量筒中。加入 $10 \mathrm{~g}$ 无水硫酸钠，振摇放置澄清后，将此液经装有少许无水硫酸钠的定量慢速滤纸过滤于 $100 \mathrm{~mL}$ 蒸发皿中。氯化钠水层用 $10 \mathrm{~mL}$ 三氯甲烷提取一次，并经过滤器一并滤于蒸发皿中。最后将无水硫酸钠也一起倒于滤纸上，用少量三氯甲烷洗量筒与无水硫酸钠，也一并滤于蒸发皿中，于 $65^{\circ} \mathrm{C}$ 水浴上通风挥干，用三氯甲烷将蒸发皿中残留物转移于浓缩管中，蒸发皿中残渣太多，则经滤纸滤入浓缩管中。于 $65^{\circ} \mathrm{C}$ 用减压吹气法将此液浓缩至 $0.4 \mathrm{~mL}$ 以下，再用少量三氯甲烷洗管壁后，浓缩定量至 $0.4 \mathrm{~mL}$ 备用。

# 5.3 测定

# 5.3.1 硅胶G薄层板的制备

薄层板厚度为 $0.3\mathrm{mm}$ ， $105^{\circ}\mathrm{C}$ 活化 $2\mathrm{h}$ ，在干燥器内可保存 $1\mathrm{d} \sim 2\mathrm{d}$ 。

# 5.3.2 点板

取薄层板（ $5\mathrm{cm}\times 20\mathrm{cm}$ ）两块，距板下端 $3\mathrm{cm}$ 的基线上各滴加两点，在距第一与第二板的左边缘 $0.8\mathrm{cm}\sim 1\mathrm{cm}$ 处各滴加 $10~\mu \mathrm{L}$ 黄曲霉毒素 $\mathbf{M}_1$ 与 $\mathbf{B}_1$ 混合标准使用液，在距各板左边缘 $2.8\mathrm{cm}\sim 3\mathrm{cm}$ 处各滴加同样液点（各种食品的滴加体积见表1)，在第二板的第2点上再滴加 $10~\mu \mathrm{L}$ 黄曲霉毒素 $\mathbf{M}_1$ 与 $\mathbf{B}_1$ 混合标准使用液。一般可将薄层板放在盛有干燥硅胶的层析槽内进行滴加，边加边用冷风机冷风吹干。

# 5.3.3 展开

5.3.3.1 横展：在槽内加入 $15 \mathrm{~mL}$ 事先用无水硫酸钠脱水的无水乙醚（每 $500 \mathrm{~mL}$ 无水乙醚中加 $20 \mathrm{~g}$ 无水硫酸钠）。将薄层板靠近标准点的长边置于槽内，展至板端后，取出挥干，再同上继续展开一次。  
5.3.3.2 纵展：将横展两次挥干后的薄层板再用异丙醇-丙酮-苯-正己烷-石油醚（沸程 $60^{\circ}\mathrm{C} \sim 90^{\circ}\mathrm{C}$ ）-三氯甲烷（ $5 + 10 + 10 + 10 + 10 + 55$ ）混合展开剂纵展至前沿距原点距离为 $10\mathrm{cm} \sim 12\mathrm{cm}$ 取出挥干。  
5.3.3.3 横展：将纵展挥干后的板再用乙醚横展1次～2次，展开方法同5.3.3.1。

# 5.3.4 观察与评定结果

5.3.4.1 在紫外光灯下将第一、二板相互比较观察，若第二板的第二点在黄曲霉毒素 $\mathrm{M}_{1}$ 与 $\mathrm{B}_{1}$ 标准点的相应处出现最低检出量（ $\mathrm{M}_{1}$ 与 $\mathrm{B}_{1}$ 的比移值依次为0.25和0.43），而在第一板相同位置上未出现荧光

点，则样品中黄曲霉毒素 $\mathrm{M}_{1}$ 与 $\mathrm{B}_{1}$ 含量在其所定的方法灵敏度以下（见表1）。

5.3.4.2 如果第一板的相同位置上出现黄曲霉毒素 $\mathbf{M}_1$ 与 $\mathbf{B}_1$ 的荧光点，则第二板第二点的样液点是否各与滴加的标准点重叠，如果重叠，再进行以下的定量与确证试验。

# 5.3.5 稀释定量

样液中的黄曲霉毒素 $\mathrm{M}_{1}$ 与 $\mathrm{B}_{1}$ 荧光点的荧光强度与黄曲霉毒素 $\mathrm{M}_{1}$ 与 $\mathrm{B}_{1}$ 的最低检出量（ $0.0004\mu \mathrm{g}$ ）的荧光强度一致，则乳、炼乳、乳粉、干酪与奶油样品中黄曲霉毒素 $\mathrm{M}_{1}$ 与 $\mathrm{B}_{1}$ 的含量依次为 $0.1\mu \mathrm{g} / \mathrm{kg}$ 、 $0.2\mu \mathrm{g} / \mathrm{kg}$ 、 $0.5\mu \mathrm{g} / \mathrm{kg}$ 、 $0.5\mu \mathrm{g} / \mathrm{kg}$ 及 $0.5\mu \mathrm{g} / \mathrm{kg}$ ；新鲜猪组织（肝、肾、血、瘦肉）样品均为 $0.2\mu \mathrm{g} / \mathrm{kg}$ （见表1）。如样液中黄曲霉毒素 $\mathrm{M}_{1}$ 与 $\mathrm{B}_{1}$ 的荧光强度比最低检出量强，则根据其强度逐一进行测定，估计减少滴加微升数或经稀释后再滴加不同微升数，直至样液点的荧光强度与最低检出量点的荧光强度一致为止。

# 5.3.6 确证试验

在做完定性或定量的薄层板上，将要确证的黄曲霉毒素 $\mathbf{M}_1$ 和 $\mathbf{B}_1$ 的点用大头针圈出。喷以硫酸溶液 $(1 + 3)$ ，放置 $5\mathrm{min}$ 后，在紫外光灯下观察，若样液中黄曲霉毒素 $\mathbf{M}_1$ 和 $\mathbf{B}_1$ 点与标准点一样均变为黄色荧光，则进一步确证检出的荧光点是黄曲霉毒素 $\mathbf{M}_1$ 和 $\mathbf{B}_1$ 。

# 6 分析结果的表述

黄曲霉毒素 $\mathbf{M}_1$ 或 $\mathbf{B}_1$ 的含量按式（2）进行计算。

$$
X = 0. 0 0 0 4 \times \frac {V _ {1}}{V _ {2}} \times D \times \frac {1 0 0 0}{m} \dots \tag {2}
$$

式中：

$X$ ——黄曲霉毒素 $\mathrm{M}_{1}$ 或 $\mathrm{B}_{1}$ 含量, 单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ;

$V_{I}$ ——样液浓缩后体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——出现最低荧光样液的滴加体积，单位为毫升 $(\mathrm{mL})$ ；

$D$ ——浓缩样液的总稀释倍数；

$m$ ——浓缩样液中所相当的试样质量, 单位为克(g);

0.0004——黄曲霉毒素 $\mathrm{M}_{1}$ 或 $\mathrm{B}_{1}$ 的最低检出量，单位为微克（ $\mu \mathrm{g}$ ）。

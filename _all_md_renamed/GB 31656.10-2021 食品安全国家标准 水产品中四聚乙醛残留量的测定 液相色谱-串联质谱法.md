# 水产品中四聚乙醛残留量的测定 液相色谱-串联质谱法

# 1 范围

本文件规定了水产品中四聚乙醛残留量检测的制样和液相色谱-串联质谱测定方法。

本文件适用于鱼和虾可食性组织中四聚乙醛残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

GB/T30891—2014 水产品抽样规范

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试料中四聚乙醛经乙腈匀浆提取，提取液经盐析后采用N丙基乙二胺和 $\mathrm{C_{18}}$ 填料净化，液相色谱-串联质谱法检测，内标法定量。

# 5 试剂与材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
5.1.2 甲酸（HCOOH）：色谱纯  
5.1.3 氯化钠 $\mathrm{(NaCl)}$   
5.1.4 无水硫酸镁 $(\mathrm{MgSO_4})$

# 5.2 溶液的配制

5.2.1 $0.1\%$ 甲酸溶液：取甲酸 $1\mathrm{mL}$ ，加水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀。  
5.2.2 $50\%$ 乙腈溶液：移取乙腈 $100~\mathrm{mL}$ ，加水 $100~\mathrm{mL}$ ，摇匀。

# 5.3 标准品

四聚乙醛、四聚乙醛- $\mathrm{D}_{16}$ 标准品，含量均 $\geq 95\%$ ，见附录A。

# 5.4 标准溶液的制备

5.4.1 标准储备液 $(100\mu \mathrm{g / mL})$ ：取四聚乙醛和四聚乙醛- $\mathrm{D}_{16}$ 适量（相当于有效成分 $10\mathrm{mg}$ )，精密称定，分别用乙腈溶解并稀释定容至 $100~\mathrm{mL}$ 容量瓶中，摇匀，即得。 $-18^{\circ}C$ 以下保存，有效期6个月。  
5.4.2 标准中间液 $(5.0\mu \mathrm{g / mL})$ ：精密量取四聚乙醛和四聚乙醛 $\mathrm{D}_{16}$ 标准储备液各 $0.5\mathrm{mL}$ 分别于 $10~\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，摇匀，即得。 $2^{\circ}C\sim 8^{\circ}C$ 避光保存，有效期1个月。  
5.4.3 标准工作液 $(0.25\mu \mathrm{g} / \mathrm{mL})$ ：精密量取四聚乙醛和四聚乙醛- $\mathrm{D}_{16}$ 标准中间液各 $0.5\mathrm{mL}$ 分别于 $10~\mathrm{mL}$ 容量瓶中，用乙腈稀释至刻度，摇匀，即得。现用现配。

# 5.5 材料

5.5.1 N-丙基乙二胺(PSA)填料，40日～60日或相当者。  
5.5.2 $\mathrm{C_{18}}$ 填料，40日～60日或相当者。

# 6 仪器和设备

6.1 液相色谱-串联质谱仪：配自动进样器和电喷雾离子源。  
6.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$   
6.3 均质机。  
6.4 离心管：聚丙烯塑料离心管， $10~\mathrm{mL}$ 和 $50~\mathrm{mL}$   
6.5 离心机： $10000\mathrm{r / min}$ 或以上。  
6.6 涡旋混合器。

# 7 试料的制备与保存

# 7.1 试料的制备

按GB/T30891—2014附录B的要求制样。

a）取均质后的供试料品，作为供试试料；  
b）取均质后的空白样品，作为空白试料；  
c）取均质后的空白样品，添加适宜浓度的标准工作液，作为空白添加试料。

# 7.2 试料的保存

$-18^{\circ} \mathrm{C}$ 以下保存，3个月内进行分析检测。

# 8 测定步骤

# 8.1 提取

取试料 $5\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，于 $50~\mathrm{mL}$ 离心管中，加四聚乙醛- $\mathrm{D}_{16}$ 标准工作液（ $0.25\mu \mathrm{g / mL})$ $100~\mu \mathrm{L}$ ，再依次加入水 $2.0~\mathrm{mL}$ 和乙腈 $10.0~\mathrm{mL}$ ，以 $10000\mathrm{r / min}$ 速度匀质 $1\mathrm{min}$ 后加入氯化钠 $2\mathrm{g}$ ，加盖涡旋混合 $1\mathrm{min}$ 后以 $5000\mathrm{r / min}$ 的速度离心 $2\mathrm{min}$ ，收集上清液，备用。

# 8.2 净化

取备用液 $2.0\mathrm{mL}$ ，转入装有无水硫酸镁 $0.3\mathrm{g},\mathrm{C}_{18}0.1\mathrm{g},\mathrm{PSA}0.1\mathrm{g}$ 的 $5\mathrm{mL}$ 离心管，涡旋混匀 $1\mathrm{min},5000\mathrm{r / min}$ 离心 $2\mathrm{min}$ ，吸取上清液 $0.5\mathrm{mL}$ 用 $0.1\%$ 甲酸溶液稀释至 $1.0\mathrm{mL}$ 混匀，过 $0.22\mu \mathrm{m}$ 滤膜，供液相色谱-串联质谱测定。

# 8.3 标准曲线的制备

精密量取 $0.25\mu \mathrm{g / mL}$ 四聚乙醛标准工作液和 $0.25\mu \mathrm{g / mL}$ 四聚乙醛- $\mathrm{D}_{16}$ 标准工作液适量，用 $50\%$ 乙腈溶液稀释成含四聚乙醛浓度为 $0.05\mu \mathrm{g / L},0.10\mu \mathrm{g / L},0.25\mu \mathrm{g / L},0.50\mu \mathrm{g / L},1.0\mu \mathrm{g / L},2.5\mu \mathrm{g / L}$ 和 $5.0~\mu \mathrm{g} / \mathrm{L}$ 的系列标准溶液（内标浓度均为 $1.25\mu \mathrm{g / L})$ ，供液相色谱-串联质谱测定。以标准溶液浓度为横坐标、四聚乙醛定量离子对峰面积和四聚乙醛- $\mathrm{D}_{16}$ 离子对峰面积比值为纵坐标，绘制标准曲线。求回归方程和相关系数。

# 8.4 测定

# 8.4.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 柱 $(50\mathrm{mm}\times 2.1\mathrm{mm},1.7\mu \mathrm{m})$ 或相当者；  
b）流动相：A为 $0.1\%$ 的甲酸溶液，B为乙腈，梯度洗脱条件见表1；  
c) 流速: $0.3 \mathrm{~mL} / \mathrm{min}$ ;   
d） 柱温： $30^{\circ}C$   
e) 进样量: $10 \mu \mathrm{L}$ 。

# 8.4.3 测定法

取试料溶液和标准溶液，作单点或多点校准，以色谱峰面积定量，内标法计算。标准溶液及试料溶液中目标药物的特征离子质量色谱峰峰面积均应在仪器检测的线性范围之内。试料液中四聚乙醛保留时间与标准工作液中的四聚乙醛保留时间之比，偏差在 $\pm 2.5\%$ 以内，且试料溶液中的相对离子丰度与标准溶液中的相对离子丰度相比，符合表3的要求。标准溶液多反应监测色谱图见附录B.1。

# 8.5 空白试验

取空白试料，除不加标准溶液外，采用相同的测定步骤进行平行操作。

# 9 结果计算和表述

试料中待测物的残留量按标准曲线或公式(1)计算。

$$
X = \frac {A \times A _ {i S} ^ {\prime} \times C _ {S} \times C _ {i S} \times V \times V _ {2} \times 1 0 0 0}{A _ {i S} \times A _ {S} \times C _ {i S} ^ {\prime} \times V _ {1} \times m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试料中四聚乙醛残留量的数值，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C_{iS}$ ——试料溶液中四聚乙醛内标浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

$C_{\mathrm{S}}$ ——标准溶液中四聚乙醛浓度的数值，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$C_{iS}^{\prime}$ ——标准溶液中四聚乙醛内标浓度的数值，单位为微克每升 $(\mu \mathrm{g / L})$

A ——试料溶液中四聚乙醛的峰面积；

$A_{iS}$ ——试料溶液中四聚乙醛内标的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中四聚乙醛的峰面积；

$A_{\mathrm{js}}^{\prime}$ ——标准溶液中四聚乙醛内标的峰面积；

$V$ ——定容体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——用于稀释定容的乙腈提取液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——乙腈提取液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试料质量的数值，单位为克 $(\mathrm{g})$ 。

# 10 检测方法的灵敏度、准确度和精密度

# 10.1 灵敏度

本方法在鱼和虾可食性组织中的检测限为 $0.50~\mu \mathrm{g / kg}$ ，定量限为 $1.0\mu \mathrm{g / kg}$

# 10.2 准确度

本方法在 $1.0\mu \mathrm{g / kg}\sim 2.0\mu \mathrm{g / kg}$ 添加浓度水平上的回收率为 $80\% \sim 120\%$

# 10.3 精密度

本方法的批内相对标准偏差 $\leq 10\%$ ，批间相对标准偏差 $\leq 15\%$ 。

# 食品安全国家标准

# 动物性食品中三氮脒残留量的测定 高效液相色谱法

# 1 范围

本文件规定了动物性食品中三氮脒残留量检测的制样和高效液相色谱检测方法。

本文件适用于牛、羊的肌肉、肝脏、肾脏组织和牛奶、羊奶中三氮肽残留量的检测。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 术语和定义

本文件没有需要界定的术语和定义。

# 4 原理

试样中残留的三氮脒经乙腈水提取（奶中残留的三氮脒经乙酸乙腈溶液提取），弱阳离子交换固相萃取柱净化，高效液相色谱紫外法测定，外标法定量。

# 5 试剂和材料

以下所用的试剂，除特别注明外均为分析纯试剂；水为符合GB/T6682规定的一级水。

# 5.1 试剂

5.1.1 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ ：色谱纯。  
5.1.2 甲酸（HCOOH）：色谱纯  
5.1.3 甲酸铵 $\mathrm{(CH_5NO_2)}$ ：色谱纯。  
5.1.4 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2()$   
5.1.5 冰乙酸 $\mathrm{(CH_3COON)}$

# 5.2 溶液配制

5.2.1 $60\%$ 乙腈溶液：取乙腈 $60~\mathrm{mL}$ ，加水 $40~\mathrm{mL}$ ，混匀。  
5.2.2 $0.5\%$ 乙酸乙腈溶液：取冰乙酸 $2.5 \mathrm{~mL}$ ，用乙腈稀释至 $500 \mathrm{~mL}$ ，混匀。  
5.2.3 $5\%$ 氨水溶液：取氨水 $5\mathrm{mL}$ ，加水 $95~\mathrm{mL}$ ，混匀。  
5.2.4 乙酸甲醇溶液 $(\mathrm{pH}7.0)$ ：取冰乙酸 $6\mathrm{mL}$ ，加甲醇 $94~\mathrm{mL}$ ，混匀，用氨水调节 $\mathrm{pH}$ 至 $7.0\pm 0.05$ 。  
5.2.5 $0.02\mathrm{mol} / \mathrm{L}$ 甲酸铵溶液 $(\mathrm{pH}4.0)$ ：取甲酸铵 $1.26\mathrm{g}$ ，加水 $900~\mathrm{mL}$ 溶解，用甲酸调节 $\mathrm{pH}$ 至 $4.0 \pm 0.05$ ，加水稀释至 $1000~\mathrm{mL}$ 。

# 5.3 标准品

二乙酰胺三氮脒（Diminazeneaceturate， $\mathrm{C_{14}H_{15}N_7\cdot 2C_4H_7NO_3}$ ，CAS号：908-54-3)，含量 $\geqslant 91\%$

# 5.4 标准溶液制备

5.4.1 标准储备液：取二乙酰胺三氮脒标准品适量（相当于三氮脒约 $10\mathrm{mg}$ ），精密称定，加水适量使溶解并稀释定容至 $10\mathrm{mL}$ 棕色容量瓶，配制成浓度为 $1\mathrm{mg/mL}$ 的三氮脒标准储备液。 $4^{\circ}\mathrm{C}$ 避光保存，有效期

1个月。

5.4.2 标准工作液：精密量取 $1\mathrm{mg / mL}$ 标准储备液 $1\mathrm{mL}$ ，于 $100~\mathrm{mL}$ 棕色容量瓶中，用水稀释至刻度，配制成浓度为 $10~\mu \mathrm{g} / \mathrm{mL}$ 的三氮脒标准工作液。 $4^{\circ}\mathrm{C}$ 避光保存，有效期 $7\mathrm{d}$ 。  
5.5 材料  
5.5.1 弱阳离子交换固相萃取柱： $60~\mathrm{mg / 3~mL}$ ，或相当者。  
5.5.2 微孔尼龙滤膜： $0.22\mu \mathrm{m}$

6 仪器和设备

6.1 高效液相色谱仪：配紫外检测器或二极管阵列检测器。  
6.2 分析天平：感量 $0.00001\mathrm{g}$ 和 $0.01\mathrm{g}$   
6.3 pH计。  
6.4 涡旋混合器。  
6.5 涡旋振荡器。  
6.6 高速冷冻离心机：转速 $\geqslant 10000\mathrm{r / min}$   
6.7 固相萃取装置。

7 试样的制备与保存

7.1 试样的制备   
7.1.1 组织

取适量新鲜或解冻的空白或供试组织，绞碎，并均质。

a）取均质的供试样品，作为供试试样；  
b）取均质的空白样品，作为空白试样；  
c）取均质的空白样品，添加适宜浓度的标准溶液，作为空白添加试样。

7.1.2 奶

取适量新鲜或解冻的空白或供试奶样，混合均匀。

a）取混合均匀的供试样品，作为供试试样；  
b）取混合均匀的空白样品，作为空白试样；  
c）取混合均匀的空白样品，添加适宜浓度的标准溶液，作为空白添加试样。

7.2 试样的保存

-18℃以下保存。

8 测定步骤

8.1 提取

8.1.1 组织

取试料 $2\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，置 $50~\mathrm{mL}$ 聚丙烯离心管中，加 $60\%$ 乙腈溶液 $18~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，用冰乙酸调节 $\mathrm{pH}$ 至 $5.5\pm 0.1$ ，振荡 $10\mathrm{min}$ ， $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液，用 $60\%$ 乙腈溶液稀释至 $20.0~\mathrm{mL}$ ，混匀，取 $2.0~\mathrm{mL}$ （肌肉样品取 $10.0~\mathrm{mL}$ )，备用。

8.1.2 奶

取试料 $5\mathrm{g}$ （准确至 $\pm 0.05\mathrm{g}$ ，置 $50~\mathrm{mL}$ 聚丙烯离心管中，准确加入 $0.5\%$ 乙酸乙腈溶液 $10~\mathrm{mL}$ ，涡旋 $1\mathrm{min}$ ，振荡 $10\mathrm{min},10000\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，移取全部上清液，备用。

8.2 净化

取弱阳离子交换固相萃取柱，依次用甲醇 $3\mathrm{mL}$ 、水 $3\mathrm{mL}$ 活化。取备用液过柱，再依次用 $5\%$ 氨水溶液 $3\mathrm{mL}$ 、甲醇 $3\mathrm{mL}$ 淋洗，抽干，加乙酸甲醇溶液 $(\mathrm{pH}7.0)$ $1.0~\mathrm{mL}$ ，洗脱，抽干，收集洗脱液，过微孔尼龙

滤膜，供高效液相色谱仪测定。

# 8.3 标准曲线的制备

精密量取标准工作液适量，用乙酸甲醇溶液 $(\mathrm{pH}7.0)$ 稀释，配制成浓度为 $0.05\mu \mathrm{g / mL},0.1\mu \mathrm{g / mL}$ 产 $0.2\mu \mathrm{g / mL},0.5\mu \mathrm{g / mL},1\mu \mathrm{g / mL},2\mu \mathrm{g / mL},5\mu \mathrm{g / mL}$ 的系列标准溶液，过微孔尼龙滤膜，供高效液相色谱仪测定。以三氮脒色谱峰面积为纵坐标、标准溶液浓度为横坐标，绘制标准曲线。求回归方程和相关系数。

# 8.4 测定

# 8.4.1 液相色谱参考条件

a）色谱柱： $\mathrm{C_{18}}$ 柱 $(250\mathrm{mm}\times 4.6\mathrm{mm}$ ，粒径 $5\mu \mathrm{m})$ ，或相当者；  
b）流动相：A为 $0.02\mathrm{mol} / \mathrm{L}$ 甲酸铵溶液 $(\mathrm{pH}4.0)$ ；B为甲醇；  
c）梯度洗脱：洗脱条件见表1；  
d) 柱温: $30^{\circ} \mathrm{C}$ ;   
e）进样量： $20~\mu \mathrm{L}$   
f）检测波长： $370~\mathrm{nm}$

# 8.4.2 测定法

取试料溶液和相应的标准溶液，作单点或标准曲线校准，按外标法以峰面积计算。标准工作液及试样溶液中三氮胀响应值应在仪器检测的线性范围之内。试料中三氮胀的保留时间与标准溶液中三氮胀的保留时间相对偏差应在 $\pm 2.5\%$ 以内。标准溶液的色谱图见附录A。

# 8.5 空白试验

取空白试料，除不加药物外，采用完全相同的测定步骤进行测定。

# 9 结果计算和表述

试样中待测物含量以质量分数 $X$ 表示，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按公式(1)计算。

$$
X = \frac {C _ {\mathrm {S}} \times A \times V \times V _ {1}}{A _ {\mathrm {S}} \times m \times V _ {2}} \tag {1}
$$

式中：

$X$ ——试样中三氮脒残留量的数值，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A$ ——试样中三氮脒的峰面积；

$A_{\mathrm{S}}$ ——标准溶液中三氮脒的峰面积；

$C_{\mathrm{S}}$ ——标准溶液中三氮脒浓度的数值，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——洗脱液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——提取液定容后体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——过固相萃取柱的备用液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——供试试样质量的数值，单位为克(g)。

# 10 检测方法灵敏度、准确度和精密度

# 10.1 灵敏度

本方法在牛肉和羊肉中的检测限为 $0.05\mathrm{mg / kg}$ ，定量限为 $0.1\mathrm{mg / kg}$ ，在牛肝、牛肾、羊肝和羊肾中的检测限为 $0.2\mathrm{mg / kg}$ ，定量限为 $0.5\mathrm{mg / kg}$ ，在牛奶和羊奶中的检测限为 $0.01\mathrm{mg / kg}$ ，定量限为 $0.02\mathrm{mg / kg}$ 。

# 10.2 准确度

本方法在牛肉和羊肉 $0.1\mathrm{mg / kg}\sim 1\mathrm{mg / kg}$ 、牛肝和羊肝 $0.5\mathrm{mg / kg}\sim 24\mathrm{mg / kg}$ 、牛肾和羊肾 $0.5\mathrm{mg / kg}\sim 12\mathrm{mg / kg}$ 、牛奶和羊奶 $0.02\mathrm{mg / kg}\sim 0.3\mathrm{mg / kg}$ 添加浓度水平上的回收率为 $60\% \sim 110\%$

# 10.3 精密度

本方法批内相对标准偏差 $\leqslant 10\%$ ，批间相对标准偏差 $\leqslant 10\%$ 。

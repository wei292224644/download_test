# 前言

本标准代替GB22255—2014《食品安全国家标准 食品中三氯蔗糖(蔗糖素)的测定》。

本标准与GB22255—2014相比，主要变化如下：

——增加了第二法“高效液相色谱-串联质谱法”；  
——修改了高效液相色谱示差检测器法流动相条件；  
——修改了高效液相色谱法试样前处理条件；  
——修改了高效液相色谱法检出限和定量限。

# 食品安全国家标准

# 食品中三氯蔗糖（蔗糖素）的测定

# 1 范围

本标准规定了食品中三氯蔗糖(蔗糖素)的高效液相色谱和高效液相色谱-串联质谱测定方法。  
本标准适用于食品中三氯蔗糖(蔗糖素)的测定。

# 第一法 高效液相色谱法

# 2 原理

试样中的三氯蔗糖用甲醇水提取，除去蛋白、脂肪，经固相萃取柱净化、蒸干复溶富集后，用高效液相色谱仪，反相 $\mathrm{C}_{18}$ 色谱柱分离，采用示差检测器或蒸发光散射检测器检测，根据色谱峰的保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。  
3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.3 正己烷 $(\mathrm{C_6H_{14}})$ 。  
3.1.4 磷酸氢二钾 $(\mathrm{K}_2\mathrm{HPO}_4\cdot 3\mathrm{H}_2\mathrm{O})$ 。  
3.1.5 乙酸锌 $\left[\mathrm{Zn}\left(\mathrm{CH}_{3} \mathrm{COO}\right)_{2} \cdot 2 \mathrm{H}_{2} \mathrm{O}\right]$ 。  
3.1.6 亚铁氰化钾 $\left[\mathrm{K}_{4} \mathrm{Fe}(\mathrm{CN})_{6} \cdot 3 \mathrm{H}_{2} \mathrm{O}\right]$ 。  
3.1.7 乙酸 $\mathrm{(CH_3COOH)}$ 。  
3.1.8 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.9 中性氧化铝：粒径 $75\mu \mathrm{m}\sim 150\mu \mathrm{m}(100$ 目 $\sim 200$ 目）。

# 3.2 试剂配制

3.2.1 乙酸锌溶液 $(219\mathrm{g / L})$ ：称取 $21.9\mathrm{g}$ 乙酸锌，加 $3\mathrm{mL}$ 乙酸，加水溶解至 $100~\mathrm{mL}$ 。  
3.2.2 亚铁氰化钾溶液 $(106\mathrm{g / L})$ ：称取 $10.6\mathrm{g}$ 亚铁氰化钾，加水溶解至 $100~\mathrm{mL}$ 。  
3.2.3 磷酸氢二钾溶液 $(0.1\%)$ ：称取 $0.1\mathrm{g}$ 磷酸氢二钾，加水溶解至 $100~\mathrm{mL}$ 。  
3.2.4 甲醇-0.1%磷酸氢二钾溶液 $(20 + 80)$ ：将甲醇和 $0.1\%$ 磷酸氢二钾溶液按 $20:80$ 的体积比混合均匀。  
3.2.5 甲醇-水溶液 $(75 + 25)$ ：将甲醇和水按 $75:25$ 的体积比混合均匀。  
3.2.6 水-乙腈溶液 $(89 + 11)$ ：将水和乙腈按 $89:11$ 的体积比混合均匀。

# 3.3 标准品

三氯蔗糖标准品 $\mathrm{(C_{12}H_{19}Cl_3O_8}$ ，CAS号：56038-13-2）：纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准品。

# 3.4 标准溶液配制

3.4.1 三氯蔗糖标准储备溶液 $(10.0\mathrm{mg / mL})$ ：称取三氯蔗糖标准品 $0.25\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )，用水溶解并转移至 $25~\mathrm{mL}$ 容量瓶中，定容至刻度，混匀。储备液置于 $4^{\circ}C$ 冰箱中保存，保存期为6个月。  
3.4.2 三氯蔗糖标准中间溶液 $(1.00\mathrm{mg / mL})$ ：移取 $5.00~\mathrm{mL}$ 三氯蔗糖标准储备溶液 $(10.0\mathrm{mg / mL})$ 于 $50~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。置于 $4^{\circ}C$ 冰箱中保存，保存期为3个月。  
3.4.3 三氯蔗糖标准系列工作溶液：移取适量三氯蔗糖标准中间溶液 $(1.00\mathrm{mg / mL})$ 用水稀释配制成质量浓度分别为 $0.0200\mathrm{mg / mL},0.0500\mathrm{mg / mL},0.100\mathrm{mg / mL},0.200\mathrm{mg / mL},0.400\mathrm{mg / mL},$ $0.800~\mathrm{mg / mL}$ 和 $1.00~\mathrm{mg / mL}$ 的标准系列工作溶液。临用现配。

# 3.5 材料

3.5.1固相萃取柱（柱规格为 $200\mathrm{mg / 6mL},N$ -乙烯基吡咯烷酮和二乙烯基苯亲水亲脂平衡型填料或等效柱）。  
3.5.2 $0.45\mu \mathrm{m}$ 亲水微孔滤膜和 $0.45\mu \mathrm{m}$ 疏水微孔滤膜。

# 4 仪器和设备

4.1 高效液相色谱仪：配示差检测器或蒸发光散射检测器。  
4.2 天平：感量分别为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.3 涡旋混合器。  
4.4 水浴锅。  
4.5 超声波发生器： $50\mathrm{kHz}$   
4.6 离心机：转速 $\geq 3000\mathrm{r / min}$   
4.7 高速离心机：转速 $\geqslant 10000\mathrm{r / min}$   
4.8 固相萃取装置。  
4.9 匀浆机。  
4.10 粉碎机。

# 5 分析步骤

# 5.1 试样制备

液态样品摇匀；基质均匀的半固态样品和粉状样品直接测定；其他样品需匀浆或粉碎均匀。

# 5.2 试样提取

# 5.2.1 酒类试样

# 5.2.1.1 液态酒类试样

称取混匀后酒类试样 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 于蒸发皿中，沸水浴上蒸干，残渣用 $1.00~\mathrm{mL}$ 水溶

解，经 $0.45\mu \mathrm{m}$ 亲水微孔滤膜过滤后，滤液为制备的试样溶液，待测。

# 5.2.1.2 含有固体物质酒类试样

称取混匀后酒类试样 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 于蒸发皿中， $60^{\circ}C$ 水浴上加热 $30\mathrm{min}$ ，用 $10~\mathrm{mL}$ 水分3次冲洗蒸发皿的残渣，冲洗液合并移至 $15~\mathrm{mL}$ 离心管中，涡旋混合器上振荡 $3\mathrm{min}$ ，超声波提取 $20\mathrm{min}$ ，以 $8500\mathrm{r / min}$ 离心 $5\mathrm{min},5\mathrm{mL}$ 水重复提取1次，合并提取液，上清液待净化。

# 5.2.2 醋、酱油、酱及酱制品类试样

5.2.2.1 称取混匀后试样 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )于 $50~\mathrm{mL}$ 离心管，加入 $1.0\mathrm{g}$ 中性氧化铝，加入 $5\mathrm{mL}$ 水，涡旋混合器上振荡 $3\mathrm{min}$ 后，加入 $15\mathrm{mL}$ 甲醇，继续振荡 $30~\mathrm{s}$ ，超声波提取 $20\mathrm{min}$ ，以 $3000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，将上清液移入 $50~\mathrm{mL}$ 离心管中。沉淀物加入 $5.0~\mathrm{mL}$ 甲醇水溶液 $(75 + 25)$ ，玻璃棒搅拌均匀后，涡旋混合器上振荡 $30~\mathrm{s}$ ，以 $3000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，重复提取2次，合并上清液。  
5.2.2.2 将全部上清液转移至分液漏斗中，加入 $30~\mathrm{mL}$ 正己烷，振摇 $2\mathrm{min}$ ，静置分层 $20\mathrm{min}$ 后，下层水相移至蒸发皿，于沸水浴上蒸发，当蒸发皿中液体在 $1\mathrm{mL}$ 左右时用 $9\mathrm{mL}$ 水分3次冲洗蒸发皿，冲洗液合并移至 $15\mathrm{mL}$ 离心管中，超声波处理 $5\mathrm{min}$ ，以 $3000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 待净化。

# 5.2.3 果冻、糖果、蜜饯凉果类试样

称取粉碎混匀后试样 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )于 $50~\mathrm{mL}$ 离心管，加入 $5\mathrm{mL}$ 水，涡旋混合器上振荡 $3\mathrm{min}$ 后，加入 $15~\mathrm{mL}$ 甲醇， $0.50~\mathrm{mL}$ 乙酸锌溶液和 $0.50~\mathrm{mL}$ 亚铁氰化钾溶液，继续振荡 $30~s$ ，经 $60^{\circ}C$ 水浴加热 $15\mathrm{min}$ ，水浴过程中加以振摇分散，以下步骤自5.2.2.1“超声波提取 $20\mathrm{min}$ ，以 $3000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ ”开始，到5.2.2.2“超声波处理 $5\mathrm{min}$ ，以 $3000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ 待净化”为止依次处理。

# 5.2.4 其他试样

称取混匀后试样 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )于 $50~\mathrm{mL}$ 离心管，加入 $5\mathrm{mL}$ 水，涡旋混合器上振荡 $3\mathrm{min}$ 后，加入 $15~\mathrm{mL}$ 甲醇， $0.50~\mathrm{mL}$ 乙酸锌溶液和 $0.50~\mathrm{mL}$ 亚铁氰化钾溶液，以下步骤自5.2.2.1“继续振荡 $30~s$ ，超声波提取 $20\mathrm{min}$ ，以 $3000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 开始，到5.2.2.2“超声波处理 $5\mathrm{min}$ ，以 $3000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 待净化”为止依次处理。

# 5.3 试样净化

# 5.3.1 示差检测器

固相萃取柱使用前依次用 $4\mathrm{mL}$ 甲醇、 $4\mathrm{mL}$ 水活化，保持柱体湿润。

将全部试样提取上清液移入已活化的固相萃取柱，控制液体流速不超过1滴/s，柱上液面为 $2\mathrm{mm}$ 左右时加入 $1\mathrm{mL}$ 水，继续保持液体流速为1滴/s，到柱中液体完全排出后，用 $3\mathrm{mL}$ 甲醇洗脱，收集全部洗脱液，于沸水浴上蒸干，残渣用 $1.00~\mathrm{mL}$ 水溶解（如溶液有浑浊现象可将其移入离心管， $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ )，经 $0.45\mu \mathrm{m}$ 亲水微孔滤膜过滤，滤液为制备的试样溶液，待测。

# 5.3.2 蒸发光散射检测器

固相萃取柱使用前依次用 $4\mathrm{mL}$ 甲醇、 $4\mathrm{mL}$ 水活化，保持柱体湿润。

将全部试样提取上清液移入已活化的固相萃取柱，控制液体流速不超过1滴/s，柱上液面为 $2\mathrm{mm}$ 左右时加入 $1\mathrm{mL}$ 水，继续保持液体流速为1滴/s，到柱中液体完全排出后，用 $3\mathrm{mL}$ 甲醇洗脱，收集全部洗脱液，于沸水浴上蒸干，残渣用 $1.00\mathrm{mL}$ 水-乙腈溶液 $(89 + 11)$ 溶解（如溶液有浑浊现象可将其移入

离心管， $10000\mathrm{r / min}$ 离心 $5\mathrm{min})$ ，经 $0.45\mu \mathrm{m}$ 疏水微孔滤膜过滤，滤液为制备的试样溶液，待测。

注：果冻类样品经提取后的上清液需要 $50~\mathrm{^\circ C}$ 水浴加热后趁热过柱，否则易堵塞萃取柱。

# 5.4 空白试验

不同试样的前处理需要同时做试样空白试验。

# 5.7 试样溶液的测定

将试样溶液和空白试样溶液 $20\mu \mathrm{L}$ 注入液相色谱仪中，得到峰面积，根据标准曲线得到待测液中三氯蔗糖的质量浓度。

注：试样溶液中三氯蔗糖浓度超过标准曲线线性范围时，稀释后测定。

# 6 分析结果的表述

试样中三氯蔗糖的含量按式(1)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \times f \quad \dots \tag {1}
$$

式中：

$X$ ——试样中三氯蔗糖的含量，单位为克每千克 $(\mathrm{g} / \mathrm{kg})$ ；

$\rho$ ——由标准曲线得到的试样溶液中三氯蔗糖的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$\rho_0$ ——由标准曲线得到的试样空白中三氯蔗糖的质量浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V$ ——试样定容的体积，单位为毫升 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——试样质量，单位为克(g)；

$f$ ——稀释因子。

结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

蒸馏酒：取样量为 $2.000\mathrm{g}$ ，定容至 $1.00\mathrm{mL}$ 时，本方法的检出限为 $0.0075\mathrm{g/kg}$ ，定量限为 $0.02\mathrm{g/kg}$ 。其他食品：取样量为 $2.000\mathrm{g}$ ，定容至 $1.00\mathrm{mL}$ 时，本方法的检出限为 $0.01\mathrm{g/kg}$ ，定量限为 $0.03\mathrm{g/kg}$ 。

# 第二法 高效液相色谱-串联质谱法

# 9 原理

试样中的三氯蔗糖用水提取，除去蛋白、脂肪，经固相萃取柱净化，采用高效液相色谱-串联质谱仪检测，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 甲醇 $\mathrm{(CH_3OH)}$ 。  
10.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
10.1.3 正己烷 $(\mathrm{C_6H_{14}})$ 。  
10.1.4 乙酸锌 $\left[\mathrm{Zn}\left(\mathrm{CH}_{3} \mathrm{COO}\right)_{2} \cdot 2 \mathrm{H}_{2} \mathrm{O}\right]$ 。  
10.1.5 亚铁氰化钾 $\left[\mathrm{K}_{4} \mathrm{Fe}(\mathrm{CN})_{6} \cdot 3 \mathrm{H}_{2} \mathrm{O}\right]$ 。

# 10.2 试剂配制

10.2.1 乙酸锌溶液 $(219\mathrm{g / L})$ ：称取 $21.9\mathrm{g}$ 乙酸锌，加 $3\mathrm{mL}$ 乙酸，加水溶解至 $100~\mathrm{mL}$   
10.2.2 亚铁氰化钾溶液 $(106\mathrm{g / L})$ ：称取 $10.6\mathrm{g}$ 亚铁氰化钾，加水溶解至 $100~\mathrm{mL}$ 。

# 10.3 标准品

三氯蔗糖 $\left(\mathrm{C}_{12} \mathrm{H}_{19} \mathrm{Cl}_3 \mathrm{O}_8\right)$ , CAS 号: 56038-13-2): 纯度 $\geqslant 99\%$ , 或经国家认证并授予标准物质证书的标准品。

# 10.4 标准溶液配制

10.4.1 三氯蔗糖标准储备溶液 $(10.0\mathrm{mg / mL})$ ：称取三氯蔗糖标准品 $0.25\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )，用水溶解并转移至 $25~\mathrm{mL}$ 容量瓶中，定容至刻度，混匀。储备液置于 $4^{\circ}C$ 冰箱中保存，保存期为6个月。

10.4.2 三氯蔗糖标准中间溶液 $(10\mathrm{mg / L})$ ：移取 $10.0~\mu \mathrm{L}$ 三氯蔗糖标准储备溶液 $(10.0\mathrm{mg / mL})$ 于 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。置于 $4^{\circ}C$ 冰箱中保存，保存期为3个月。

10.4.3 三氯蔗糖标准系列工作溶液：分别移取三氯蔗糖标准中间溶液 $(10\mathrm{mg / L})0.100\mathrm{mL},0.200\mathrm{mL}$ $0.500~\mathrm{mL},1.00~\mathrm{mL}$ 和 $1.50~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀。配制成三氯蔗糖质量浓度分别为 $0.100\mathrm{mg / L},0.200\mathrm{mg / L},0.500\mathrm{mg / L},1.00\mathrm{mg / L}$ 和 $1.50\mathrm{mg / L}$ 的标准系列工作溶液。临用现配。

# 12 分析步骤

# 12.1 试样制备

液态样品摇匀；基质均匀的半固态样品和粉状样品直接测定；其他样品需匀浆或粉碎均匀。

# 12.2 试样提取

# 12.2.1 酒类试样

# 12.2.1.1 蒸馏酒

称取混匀后蒸馏酒试样 $2\mathrm{g}$ （精确至 $0.001\mathrm{g}$ ）至 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度摇匀，试样经 $0.22\mu \mathrm{m}$ 亲水微孔滤膜过滤，滤液为制备的试样溶液，待测。

# 12.2.1.2 发酵酒和配制酒

12.2.1.2.1 液态试样: 称取混匀后试样 $2 \mathrm{~g}$ (精确至 $0.001 \mathrm{~g}$ ) 于蒸发皿中, $60^{\circ} \mathrm{C}$ 水浴上加热 $30 \mathrm{~min}$ , 用 $10 \mathrm{~mL}$ 水分 3 次冲洗蒸发皿的残渣, 冲洗液全部转移至 $25 \mathrm{~mL}$ 容量瓶中, 用水定容至刻度并摇匀, 待净化。  
12.2.1.2.2 含有固体物质酒类试样: 称取混匀后酒类试样 $2 \mathrm{~g}$ (精确至 $0.001 \mathrm{~g}$ ) 于蒸发皿中, $60^{\circ} \mathrm{C}$ 水浴上加热 $30 \mathrm{~min}$ , 用 $10 \mathrm{~mL}$ 水分 3 次冲洗蒸发皿的残渣, 冲洗液合并移至 $15 \mathrm{~mL}$ 离心管中, 涡旋混合器上振荡 $3 \mathrm{~min}$ , 超声波提取 $20 \mathrm{~min}$ , 以 $8500 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 上清液转移至 $25 \mathrm{~mL}$ 容量瓶中, $10 \mathrm{~mL}$ 水重复提取 1 次, 合并提取液后, 用水定容至刻度并摇匀, 待净化。

# 12.2.2 果冻、糖果、蜜饯凉果类试样

称取 $2\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ ) 于 $50~\mathrm{mL}$ 离心管中，加入 $10~\mathrm{mL}$ 水，涡旋混合器上振荡 $3\mathrm{min}$ ，加入 $1.00~\mathrm{mL}$ 乙酸锌溶液、 $1.00~\mathrm{mL}$ 亚铁氰化钾溶液，经 $60^{\circ}\mathrm{C}$ 水浴加热 $15\mathrm{min}$ ，水浴过程中注意摇散。超声波提取 $20\mathrm{min},8500\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液（必要时可用快速定量滤纸过滤）于 $25~\mathrm{mL}$ 容量瓶中， $10~\mathrm{mL}$ 水重复提取1次，合并提取液后，加水定容至刻度，混匀后转移至 $50~\mathrm{mL}$ 离心管，加入 $15~\mathrm{mL}$ 正己烷，涡旋混合器上振荡 $3\mathrm{min},8500\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去上层正己烷层，下层清液待净化。

# 12.2.3 其他试样

称取 $2\mathrm{g}$ 试样(精确至 $0.001\mathrm{g}$ ) 于 $50~\mathrm{mL}$ 离心管中，加入 $10~\mathrm{mL}$ 水，涡旋混合器上振荡 $3\mathrm{min}$ ，加入 $1.00~\mathrm{mL}$ 乙酸锌溶液和 $1.00~\mathrm{mL}$ 亚铁氰化钾溶液，超声波提取 $20\mathrm{min}$ ， $8500~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取上清液（必要时可用快速定量滤纸过滤）于 $25~\mathrm{mL}$ 容量瓶中， $10~\mathrm{mL}$ 水重复提取1次，合并提取液后，加水定容至刻度，混匀后转移至 $50~\mathrm{mL}$ 离心管，加入 $15~\mathrm{mL}$ 正己烷，涡旋混合器上振荡 $3\mathrm{min}$ ， $8500~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，弃去上层正己烷层，下层清液待净化。

# 12.3 试样净化

固相萃取柱使用前依次用 $4\mathrm{mL}$ 甲醇、 $4\mathrm{mL}$ 水活化，保持柱体湿润。

准确移取 $10.00\mathrm{mL}$ 试样提取液加入已活化的固相萃取柱，控制液体流速不超过1滴/s，待柱中液体完全排出后，用 $3\mathrm{mL}$ 甲醇洗脱，收集甲醇洗脱液于 $10\mathrm{mL}$ 容量瓶中，加水定容，混匀，经 $0.22\mu \mathrm{m}$ 疏水微孔滤膜过滤后，滤液为制备的试样溶液，待测。

注：果冻类样品待净化液需要 $50^{\circ}\mathrm{C}$ 水浴加热后趁热过柱，否则易堵塞萃取柱。

# 12.5.4 定量测定

# 12.5.4.1 工作曲线的制作

# 12.5.4.2 试样溶液的测定

将试样溶液按仪器参考条件进行测定，得到相应的试样溶液的质量色谱峰面积。根据工作曲线得到试样溶液中三氯蔗糖的质量浓度。

# 13 分析结果的表述

蒸馏酒试样中三氯蔗糖含量按式(2)，其他试样按式(3)计算。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V}{m \times 1 0 0 0} \times f \tag {2}
$$

式中：

$X$ ——试样中三氯蔗糖的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho$ ——由工作曲线得出的试样溶液中三氯蔗糖的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——由工作曲线得到的试样空白中三氯蔗糖的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克(g)；

1000——换算系数；

$f$ ——稀释因子。

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V _ {1} \times V _ {3}}{m \times 1 0 0 0 \times V _ {2}} \times f \quad \dots \tag {3}
$$

式中：

$X$ ——试样中三氯蔗糖的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho$ ——由工作曲线得出的试样溶液中三氯蔗糖的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——由工作曲线得到的试样空白中三氯蔗糖的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V_{1}$ ——试样提取液定容体积，单位为毫升 $(\mathrm{mL})$

$V_{3}$ ——试样经净化洗脱后的最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克(g)；

1000——换算系数；

$V_{2}$ ——用于净化分取的试样体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释因子。

结果保留3位有效数字。

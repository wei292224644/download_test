# 前言

本标准代替GB/T4789.14-2003《食品卫生微生物学检验蜡样芽胞杆菌检验》。

本标准与GB/T4789.14-2003相比，主要变化如下：

——修改了标准的中文名称；  
——增加了第二法 蜡样芽胞杆菌 MPN 计数法；  
将选择性分离培养基（MYP）培养条件由 $37\,^{\circ}\mathrm{C}$ 培养 $12\, \mathrm{h} \sim 20\, \mathrm{h}$ 改为 $30\,^{\circ}\mathrm{C} \pm 1\,^{\circ}\mathrm{C}$ 培养 $24\, \mathrm{h} \sim 48\, \mathrm{h}$ ;  
——增加了溶菌酶试验；  
——修改了操作步骤；  
——增加了附录A、附录B。

# 食品安全国家标准

# 食品微生物学检验 蜡样芽胞杆菌检验

# 1 范围

本标准规定了食品中蜡样芽胞杆菌（Bacillus cereus）的检验方法。

本标准第一法适用于蜡样芽胞杆菌含量较高的食品中蜡样芽胞杆菌的计数；第二法适用于蜡样芽胞杆菌含量较低的食品样品中蜡样芽胞杆菌的计数。

# 2 设备和材料

除微生物实验室常规灭菌及培养设备外，其他设备和材料如下：

a) 冰箱： $2^{\circ}\mathrm{C} \sim 5^{\circ}\mathrm{C}$ ；  
b) 恒温培养箱: $30^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 、 $36^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ ;  
c) 均质器；  
d) 电子天平：感量 $0.1\mathrm{g}$ ；  
e) 无菌锥形瓶： $100\mathrm{mL}$ 、 $500\mathrm{mL}$ ；  
f) 无菌吸管： $1\mathrm{mL}$ （具 $0.01\mathrm{mL}$ 刻度）、 $10\mathrm{mL}$ （具 $0.1\mathrm{mL}$ 刻度）或微量移液器及吸头；  
g) 无菌平皿：直径 $90\mathrm{mm}$   
h) 无菌试管: $18 \mathrm{~mm} \times 180 \mathrm{~mm}$ ;  
i) 显微镜：10倍～100倍（油镜）；  
j) L涂布棒。

# 4.2 操作步骤

# 4.2.1 样品处理

冷冻样品应在 $45^{\circ}\mathrm{C}$ 以下不超过 $15\mathrm{min}$ 或在 $2^{\circ}\mathrm{C} \sim 5^{\circ}\mathrm{C}$ 不超过 $18\mathrm{h}$ 解冻，若不能及时检验，应放于 $-10^{\circ}\mathrm{C} \sim -20^{\circ}\mathrm{C}$ 保存；非冷冻而易腐的样品应尽可能及时检验，若不能及时检验，应置于 $2^{\circ}\mathrm{C} \sim 5^{\circ}\mathrm{C}$ 冰箱保存， $24\mathrm{h}$ 内检验。

# 4.2.2 样品制备

称取样品 $25\mathrm{g}$ ，放入盛有 $225\mathrm{mL}$ PBS或生理盐水的无菌均质杯内，用旋转刀片式均质器以8000 $\mathrm{r / min}\sim 10000\mathrm{r / min}$ 均质 $1\mathrm{min}\sim 2\mathrm{min}$ ，或放入盛有 $225\mathrm{mL}$ PBS或生理盐水的无菌均质袋中，用拍击式均质器拍打 $1\mathrm{min}\sim 2\mathrm{min}$ 。若样品为液态，吸取 $25\mathrm{mL}$ 样品至盛有 $225\mathrm{mL}$ PBS或生理盐水的无菌锥形瓶(瓶内可预置适当数量的无菌玻璃珠)中，振荡混匀，作为1:10的样品匀液。

# 4.2.3 样品的稀释

吸取4.2.2中1:10的样品匀液 $1\mathrm{mL}$ 加到装有 $9\mathrm{mL}$ PBS或生理盐水的稀释管中，充分混匀制成1:100的样品匀液。跟据对样品污染状况的估计，按上述操作，依次制成十倍递增系列稀释样品匀液。每递增稀释1次，换用1支 $1\mathrm{mL}$ 无菌吸管或吸头。

# 4.2.4 样品接种

根据对样品污染状况的估计，选择2个～3个适宜稀释度的样品匀液（液体样品可包括原液），以0.3 mL、0.3 mL、0.4 mL接种量分别移入三块MYP琼脂平板，然后用无菌L棒涂布整个平板，注意不要触及平板边缘。使用前，如MYP琼脂平板表面有水珠，可放在25℃～50℃的培养箱里干燥，直到平板表面的水珠消失。

# 4.2.5 分离、培养

# 4.2.5.1 分离

在通常情况下，涂布后，将平板静置 $10\mathrm{min}$ 。如样液不易吸收，可将平板放在培养箱 $30^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $1\mathrm{h}$ ，等样品匀液吸收后翻转平皿，倒置于培养箱， $30^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h} \pm 2\mathrm{h}$ 。如果菌落不典型，可继续培养 $24\mathrm{h} \pm 2\mathrm{h}$ 再观察。在MYP琼脂平板上，典型菌落为微粉红色(表示不发酵甘露醇)，周围有白色至淡粉红色沉淀环（表示产卵磷脂酶）。

# 4.2.5.2 纯培养

从每个平板（符合4.4.1.1要求的平板）中挑取至少5个典型菌落（小于5个全选），分别划线接种于营养琼脂平板做纯培养， $30^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h} \pm 2\mathrm{h}$ ，进行确证实验。在营养琼脂平板上，典型菌落为灰白色，偶有黄绿色，不透明，表面粗糙似毛玻璃状或融蜡状，边缘常呈扩展状，直径为 $4\mathrm{mm} \sim 10\mathrm{mm}$ 。

# 4.3 确定鉴定

# 4.3.1 染色镜检

挑取纯培养的单个菌落，革兰氏染色镜检。蜡样芽胞杆菌为革兰氏阳性芽胞杆菌，大小为（ $1\mu \mathrm{m} \sim 1.3\mu \mathrm{m}$ ） $\times$ （ $3\mu \mathrm{m} \sim 5\mu \mathrm{m}$ ），芽胞呈椭圆形位于菌体中央或偏端，不膨大于菌体，菌体两端较平整，多呈短链或长链状排列。

# 4.3.2 生化鉴定

# 4.3.2.2 动力试验

用接种针挑取培养物穿刺接种于动力培养基中， $30^{\circ}\mathrm{C}$ 培养 $24\mathrm{h}$ 。有动力的蜡样芽胞杆菌应沿穿刺线呈扩散生长，而蕈状芽孢杆菌常呈“绒毛状”生长。也可用悬滴法检查。

# 4.3.2.3 溶血试验

挑取纯培养的单个可疑菌落接种于TSSB琼脂平板上， $30^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h} \pm 2\mathrm{h}$ 。蜡样芽胞杆菌菌落为浅灰色，不透明，似白色毛玻璃状，有草绿色溶血环或完全溶血环。苏云金芽胞杆菌和蕈状芽胞杆菌呈现弱的溶血现象，而多数炭疽芽胞杆菌为不溶血，巨大芽胞杆菌为不溶血。

# 4.3.2.4 根状生长试验

挑取单个可疑菌落按间隔 $2\mathrm{cm}\sim 3\mathrm{cm}$ 左右距离划平行直线于经室温干燥1d～2d的营养琼脂平板上， $30^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h}\sim 48\mathrm{h}$ ，不能超过72h。用蜡样芽胞杆菌和蕈状芽胞杆菌标准株作为对照进行同步试验。蕈状芽胞杆菌呈根状生长的特征。蜡样芽胞杆菌菌株呈粗糙山谷状生长的特征。

# 4.3.2.5 溶菌酶耐性试验

用接种环取纯菌悬液一环，接种于溶菌酶肉汤中， $36^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h}$ 。蜡样芽胞杆菌在本培养基（含 $0.001\%$ 溶菌酶）中能生长。如出现阴性反应，应继续培养 $24\mathrm{h}$ 。巨大芽胞杆菌不生长。

# 4.3.2.6 蛋白质毒素结晶试验

挑取纯培养的单个可疑菌落接种于硫酸锰营养琼脂平板上， $30^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h} \pm 2\mathrm{h}$ ，并于室温放置 $3\mathrm{d} \sim 4\mathrm{d}$ ，挑取培养物少许于载玻片上，滴加蒸馏水混匀并涂成薄膜。经自然干燥，微火固定后，加甲醇作用 $30\mathrm{s}$ 后倾去，再通过火焰干燥，于载玻片上滴满 $0.5\%$ 碱性复红，放火焰上加热（微见蒸气，勿使染液沸腾）持续 $1\mathrm{min} \sim 2\mathrm{min}$ ，移去火焰，再更换染色液再次加温染色 $30\mathrm{s}$ ，倾去染液用洁净自来水彻底清洗、晾干后镜检。观察有无游离芽胞（浅红色）和染成深红色的菱形蛋白结晶体。如发现游离芽胞形成的不丰富，应再将培养物置室温 $2\mathrm{d} \sim 3\mathrm{d}$ 后进行检查。除苏云金芽胞杆菌外，其他芽胞杆菌不产生蛋白结晶体。

# 4.4 结果计算

# 4.4.1 典型菌落计数和确认

4.4.1.1 选择有典型蜡样芽胞杆菌菌落的平板，且同一稀释度3个平板所有菌落数合计在 $20\mathrm{CFU}\sim 200$ CFU之间的平板，计数典型菌落数。如果出现a）~f）现象按4.4.2.1中公式（1）计算，如果出现g）现象则按4.4.2.2中公式（2）计算；

a) 只有一个稀释度的平板菌落数在 $20 \mathrm{CFU} \sim 200 \mathrm{CFU}$ 之间且有典型菌落, 计数该稀释度平板上的典型菌落;  
b) 2 个连续稀释度的平板菌落数均在 $20 \mathrm{CFU} \sim 200 \mathrm{CFU}$ 之间, 但只有一个稀释度的平板有典型菌落, 应计数该稀释度平板上的典型菌落;  
c）所有稀释度的平板菌落数均小于20CFU且有典型菌落，应计数最低稀释度平板上的典型菌落；  
d) 某一稀释度的平板菌落数大于 200 CFU 且有典型菌落, 但下一稀释度平板上没有典型菌落, 应计数该稀释度平板上的典型菌落;  
e）所有稀释度的平板菌落数均大于200CFU且有典型菌落，应计数最高稀释度平板上的典型菌落；  
f）所有稀释度的平板菌落数均不在 $20\mathrm{CFU} \sim 200\mathrm{CFU}$ 之间且有典型菌落，其中一部分小于 $20\mathrm{CFU}$ 或大于 $200\mathrm{CFU}$ 时，应计数最接近 $20\mathrm{CFU}$ 或 $200\mathrm{CFU}$ 的稀释度平板上的典型菌落。  
g)2个连续稀释度的平板菌落数均在 $20\mathrm{CFU}\sim 200\mathrm{CFU}$ 之间且均有典型菌落。

4.4.1.2 从每个平板中至少挑取5个典型菌落（小于5个全选），划线接种于营养琼脂平板做纯培养， $30^{\circ}\mathrm{C} \pm 1^{\circ}\mathrm{C}$ 培养 $24\mathrm{h} \pm 2\mathrm{h}$ 。

# 4.4.2 计算公式

# 4.4.2.1 菌落计算公式（1）

$$
T = \frac {A B}{C d} \dots \tag {1}
$$

式中：

$T$ ——样品中蜡样芽胞杆菌菌落数；

$A$ ——某一稀释度蜡样芽胞杆菌典型菌落的总数；

$B$ ——鉴定结果为蜡样芽胞杆菌的菌落数；

$C$ ——用于蜡样芽胞杆菌鉴定的菌落数；

d——稀释因子。

# 4.4.2.2 菌落计算公式（2）

$$
T = \frac {A _ {1} B _ {1} / C _ {1} + A _ {2} B _ {2} / C _ {2}}{1 . 1 \mathrm {d}} \tag {2}
$$

式中：

$T$ ——样品中蜡样芽胞杆菌菌落数；

$A_{1}$ ——第一稀释度（低稀释倍数）蜡样芽胞杆菌典型菌落的总数；

$A_{2}$ ——第二稀释度（高稀释倍数）蜡样芽胞杆菌典型菌落的总数；

$B_{2}$ ——第二稀释度（高稀释倍数）鉴定结果为蜡样芽胞杆菌的菌落数；

$C_{1}$ ——第一稀释度（低稀释倍数）用于蜡样芽胞杆菌鉴定的菌落数；

$C_2$ ——第二稀释度（高稀释倍数）用于蜡样芽胞杆菌鉴定的菌落数；

1.1——计算系数（如果第二稀释度蜡样芽胞杆菌鉴定结果为0，计算系数采用1）；

# A.16.2 制法

除溴麝香草酚蓝溶液和琼脂外，将A.16.1所述各成分溶解于 $1000.0\mathrm{mL}$ 蒸馏水内，校正 $\mathrm{pH}$ 至6.8，再加琼脂，加热溶化。然后加入溴麝香草酚蓝溶液，混合均匀后分装试管， $121^{\circ}\mathrm{C}$ 高压灭菌 $15\mathrm{min}$ 。制成斜面。

# A. 16.3 试验方法

挑取少量琼脂培养物接种于西蒙氏柠檬酸培养基， $36\,^{\circ}\mathrm{C} \pm 1\,^{\circ}\mathrm{C}$ 培养4d。每天观察结果，阳性者斜面上有菌落生长，培养基从绿色转为蓝色。

# A.17.2 制法

将A.17.1所述成分混合，置流动蒸汽灭菌器内，加热溶解，校正 $\mathrm{pH}$ 至 $7.4\sim 7.6$ ，过滤。分装试管， $121~^\circ C$ 高压灭菌 $10\mathrm{min}$ ，备用。

# A. 17.3 试验方法

挑取可疑菌落接种于明胶培养基， $36\,^{\circ}\mathrm{C} \pm 1\,^{\circ}\mathrm{C}$ 培养 $24\, \mathrm{h} \pm 2\, \mathrm{h}$ ，取出， $2\,^{\circ}\mathrm{C} \sim 8\,^{\circ}\mathrm{C}$ 放置 $30\, \mathrm{min}$ ，取出，观察明胶液化情况。

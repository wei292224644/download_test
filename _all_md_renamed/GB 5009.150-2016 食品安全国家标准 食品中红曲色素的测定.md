# 食品中红曲色素的测定

2016-12-23 发布

2017-06-23 实施

# 前言

本标准代替GB/T5009.150—2003《食品中红曲色素的测定》。

本标准与GB/T5009.150—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中红曲色素的测定”；  
——增加了高效液相色谱法测定红曲色素，可准确定量；  
——扩大了方法的适用范围；  
——改进了样品前处理分析方法；  
——删除了薄层色谱法。

# 食品安全国家标准

# 食品中红曲色素的测定

# 1 范围

本标准规定了食品中红曲红素、红曲素、红曲红胺的测定方法。

本标准适用于风味发酵乳、果酱、腐乳、干杏仁、糖果、方便面制品、糕点、饼干、熟肉制品、酱油、果蔬汁饮料、固体饮料、配制酒、果冻、薯片中3种红曲色素的测定。

# 2 原理

试样用无水乙醇或 $80\%$ 乙醇提取后，经固相萃取柱净化，采用液相色谱检测，以保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.2 无水乙醇 $\mathrm{(CH_3CH_2OH)}$ ：色谱纯。  
3.1.3 冰乙酸 $\mathrm{(CH_3COOH)}$   
3.1.4 乙酸铵 $\mathrm{(CH_3COONH_4)}$

# 3.2 试剂配制

3.2.1 乙酸-乙酸铵溶液 $(\mathrm{pH} = 5)$ ：准确称取 $0.77\mathrm{g}$ 乙酸铵置于 $1\mathrm{L}$ 容量瓶内，加 $900~\mathrm{mL}$ 水溶解，用冰乙酸调节 $\mathrm{pH} = 5.0$ ，加水定容至 $1\mathrm{L}$ ，混匀，经 $0.45\mu \mathrm{m}$ 微孔滤膜过滤后使用。  
3.2.2 甲醇溶液 $(20\%)$ ：量取甲醇 $20~\mathrm{mL}$ ，加水稀释并定容至 $100~\mathrm{mL}$ ，混匀。  
3.2.3 甲醇溶液 $(40\%)$ ：量取甲醇 $40~\mathrm{mL}$ ，加水稀释并定容至 $100~\mathrm{mL}$ ，混匀。  
3.2.4 乙醇溶液 $(80\%)$ ：量取乙醇 $800~\mathrm{mL}$ ，加水稀释并定容至1L，混匀。

# 3.3 标准品

3.3.1 红曲红素标准品 $\mathrm{(C_{23}H_{26}O_5}$ ,CAS号：874807-57-5)，纯度 $\geqslant 99.0\%$   
3.3.2 红曲素标准品 $\mathrm{(C_{21}H_{26}O_5}$ ，CAS号：21516-68-7)，纯度 $\geqslant 97.0\%$   
3.3.3 红曲红胺标准品 $\mathrm{(C_{23}H_{27}NO_4}$ ，CAS号：126631-93-4)，纯度 $\geq 99.0\%$

# 3.4 标准溶液配制

3.4.1 红曲红素、红曲素、红曲红胺混合标准储备溶液：分别准确（精确至 $0.01\mathrm{mg}$ ）称取红曲红素 $0.25\mathrm{g}$ 、红曲素 $2.5\mathrm{mg}$ 、红曲红胺 $5.0\mathrm{mg}$ 于 $50\mathrm{mL}$ 小烧杯中，加甲醇溶解，用甲醇转移到 $50\mathrm{mL}$ 容量瓶

中，定容，混匀，于 $4^{\circ}\mathrm{C}$ 保存，其中红曲红素质量浓度为 $5000\mu \mathrm{g / mL}$ 、红曲素为 $50~\mu \mathrm{g / mL}$ 、红曲红胺为 $100~\mu \mathrm{g / mL}$ 。  
3.4.2 红曲红素、红曲素、红曲红胺混合标准曲线工作液：分别吸取混合标准储备溶液 $0.50\mathrm{mL}$ 、 $1.00\mathrm{mL},2.00\mathrm{mL},5.00\mathrm{mL},10.00\mathrm{mL}$ 于 $25~\mathrm{mL}$ 容量瓶中，用甲醇溶液定容至刻度，混匀，于 $4^{\circ}\mathrm{C}$ 保存。

# 3.5 材料

$N\cdot$ 乙烯吡咯烷酮和二乙烯基苯聚合物： $200~\mathrm{mg},6~\mathrm{mL}$ 。使用前依次用 $5\mathrm{mL}$ 甲醇、 $5\mathrm{mL}$ 水活化。

# 4 仪器和设备

4.1 高效液相色谱仪，配置紫外可见检测器。  
4.2 分析天平：感量为 $0.01\mathrm{mg}$ 和 $0.01\mathrm{g}$ 。  
4.3 离心机：转速 $\geq 4000\mathrm{r / min}$   
4.4 高速均质器。  
4.5 高速粉碎机。  
4.6 固相萃取装置。  
4.7 有机相型微孔滤膜：孔径 $0.45\mu \mathrm{m}$

# 5 分析步骤

# 5.1 试样制备及保存

# 5.1.1 液体样品

将果汁及果汁饮料、果味碳酸饮料等样品摇匀分装，密闭常温或冷藏保存。

# 5.1.2 半固态样品

对果冻、风味发酵乳等样品取可食部分匀浆后，搅拌均匀，分装，密闭冷藏或冷冻保存。

# 5.1.3 固体样品

饼干、糕点、熟肉制品等低含水量样品，经高速粉碎机粉碎、分装，于室温下避光密闭保存；对于固体饮料等呈均匀状的粉状样品，可直接分装，于室温下避光密闭保存。

# 5.2 试样处理

# 5.2.1 液体样品

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样(若试样中含二氧化碳应先加热除去)置于烧杯中，用适量无水乙醇溶解并转移至 $50~\mathrm{mL}$ 容量瓶中，加无水乙醇定容，摇匀。准确移取上述提取液 $20~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 鸡心瓶中，向鸡心瓶中加入 $10~\mathrm{mL}$ 无水乙醇，在 $60^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 下旋转浓缩至近干，用 $2\mathrm{mL}20\%$ 甲醇溶液洗涤鸡心瓶2次。

将上述待提取液全部转移至经过预活化的HLB固相萃取柱中，控制流速在 $1\mathrm{mL} / \mathrm{min}\sim$ $2\mathrm{mL} / \mathrm{min}$ ，弃去流出液。用 $5\mathrm{mL}40\%$ 甲醇溶液淋洗净化柱，弃去流出液，再用 $5\mathrm{mL}$ 甲醇洗脱，控制流速在 $1\mathrm{mL} / \mathrm{min}\sim 2\mathrm{mL} / \mathrm{min}$ ，收集洗脱液于 $25~\mathrm{mL}$ 刻度离心管中，洗脱液在 $45^{\circ}\mathrm{C}$ 下氮吹干后，用 $2\mathrm{mL}$ $40\%$ 甲醇溶液振荡溶解残渣后过 $0.45\mu \mathrm{m}$ 有机滤膜，待测。

# 5.2.2 半固体试样及固体试样

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 均匀试样，放入 $50~\mathrm{mL}$ 塑料离心管中，向其中加入 $20~\mathrm{mL}~80\%$ 乙醇溶液， $15000~\mathrm{r / min}$ 均质提取 $2\mathrm{min},4000\mathrm{r / min}$ 下离心 $3\mathrm{min}$ ，上清液转移至 $50~\mathrm{mL}$ 容量瓶中，向残渣加入 $20~\mathrm{mL}~80\%$ 乙醇溶液重复提取1次，合并提取液于同一容量瓶中，用无水乙醇定容，摇匀。

准确移取上述提取液 $20~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 鸡心瓶中，向鸡心瓶中加入 $10~\mathrm{mL}$ 无水乙醇，在 $60^{\circ}\mathrm{C} \pm$ $2^{\circ}\mathrm{C}$ 下旋转浓缩至近干，用 $2\mathrm{mL}20\%$ 甲醇溶液洗涤鸡心瓶2次。将上述待净化液全部转移至经过预活化的HLB固相萃取柱中，控制流速在 $1\mathrm{mL / min} \sim 2\mathrm{mL / min}$ ，弃去流出液。用 $5\mathrm{mL}40\%$ 甲醇溶液淋洗净化柱，弃去流出液，再用 $5\mathrm{mL}$ 甲醇洗脱，控制流速在 $1\mathrm{mL / min} \sim 2\mathrm{mL / min}$ ，收集洗脱液于 $25~\mathrm{mL}$ 刻度离心管中，洗脱液在 $45^{\circ}\mathrm{C}$ 下氮吹干后，用 $2\mathrm{mL}20\%$ 甲醇溶液振荡溶解残渣后过 $0.45\mu \mathrm{m}$ 有机滤膜，待测。

# 5.3 仪器参考条件

仪器参考条件列出如下：

a）色谱柱：高纯度球形多孔硅胶多点键合 $\mathrm{C_{18}}$ 色谱柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或同等性能的色谱柱；  
b）流动相：A：乙酸-乙酸铵溶液 $(\mathrm{pH} = 5)$ ，B：甲醇，梯度洗脱见表1；  
c） 柱温： $40^{\circ}C$   
d）进样量： $20~\mu \mathrm{L}$   
e）检测波长：红曲红胺为 $264\mathrm{nm}$ ，红曲红素和红曲素为 $390~\mathrm{nm}$ 。

# 5.4 标准曲线的制作

将混合标准系列工作液及混合标准储备液分别注入高效液相色谱仪中，测定相应的峰高或峰面积。以标准工作液的浓度为横坐标，以峰面积为纵坐标，绘制标准曲线(标准图谱见图A.1)。

# 5.5 试样溶液的测定

将试样溶液注入高效液相色谱仪中，得到峰面积，根据标准曲线得到待测液中红曲色素的浓度。如果样品中红曲色素的含量超出标准曲线线性范围，用 $20\%$ 甲醇溶液稀释待测样品溶液，使待测液中红

曲色素浓度在线性范围内。

# 6 分析结果的表述

试样中红曲色素的含量按式(1)计算：

$$
X = \frac {\rho \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中红曲色素的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$\rho$ ——由标准曲线求得试样溶液中某红曲色素的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样品溶液定容体积，单位为 $(\mathrm{mL})$ ；

1000——换算系数；

$m$ ——最终样液代表的试样质量，单位为克 $(\mathrm{g})$ 。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 8 其他

方法检出限(LOD)：当称样量为 $5.0\mathrm{g}$ 时，红曲红素为 $30~\mathrm{mg / kg}$ ，红曲素为 $0.3~\mathrm{mg / kg}$ ，红曲红胺为 $3\mathrm{mg / kg}$ 。

方法定量限(LOQ)：当称样量为 $5.0\mathrm{g}$ 时，红曲红素为 $100\mathrm{mg / kg}$ ，红曲素为 $1\mathrm{mg / kg}$ ，红曲红胺为 $10\mathrm{mg / kg}$ 。

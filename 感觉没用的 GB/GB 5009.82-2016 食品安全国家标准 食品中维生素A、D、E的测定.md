# 前言

本标准代替GB/T5009.82—2003《食品中维生素A和维生素E的测定》、GB5413.9—2010《食品安全国家标准婴幼儿食品和乳品中维生素A、D、E的测定》、GB/T9695.26—2008《肉与肉制品维生素A含量测定》、GB/T9695.30—2008《肉与肉制品维生素E含量测定》、NY/T1598—2008《食用植物油中维生素E组分和含量的测定高效液相色谱法》。

本标准与GB/T5009.82—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中维生素A、D、E的测定”；  
——增加了“食品中维生素E的测定 正相高效液相色谱法”；  
——增加了“食品中维生素D的测定 液相色谱-串联质谱法”；  
——增加了“食品中维生素D的测定 高效液相色谱法”；  
——修改了“食品中维生素A和维生素E的测定 反相高效液相色谱法”；  
——修改了维生素E异构体的反相色谱分离条件，可同时分离测定4种生育酚异构体；  
——删除了苯并芘内标定量法，改用外标法定量；  
——删除了“比色法”测定维生素A。

# 食品安全国家标准

# 食品中维生素A、D、E的测定

# 1 范围

本标准规定了食品中维生素A、维生素E和维生素D的测定方法。

本标准第一法适用于食品中维生素A和维生素E的测定。

本标准第二法适用于食用油、坚果、豆类和辣椒粉等食物中维生素E的测定。

本标准第三法适用于食品中维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 的测定。

本标准第四法适用于配方食品中维生素 $\mathrm{D}_2$ 或维生素 $\mathrm{D}_3$ 的测定。

# 第一法 食品中维生素A和维生素E的测定 反相高效液相色谱法

# 2 原理

试样中的维生素A及维生素E经皂化(含淀粉先用淀粉酶酶解)、提取、净化、浓缩后， $\mathrm{C}_{30}$ 或PFP反相液相色谱柱分离，紫外检测器或荧光检测器检测，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 无水乙醇 $\mathrm{(C_2H_5OH)}$ ：经检查不含醛类物质，检查方法参见A.1。  
3.1.2 抗坏血酸 $(\mathrm{C_6H_8O_6})$ 。  
3.1.3 氢氧化钾(KOH)。  
3.1.4 乙醚 $\left[\left(\mathrm{CH}_{3} \mathrm{CH}_{2}\right)_{2} \mathrm{O}\right]$ ：经检查不含过氧化物，检查方法参见A.2。  
3.1.5 石油醚 $\mathrm{(C_5H_{12}O_2)}$ ：沸程为 $30^{\circ}\mathrm{C}\sim 60^{\circ}\mathrm{C}$   
3.1.6 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
3.1.7 pH试纸 $(\mathrm{pH}$ 范围 $1\sim 14)$ 。  
3.1.8 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.9 淀粉酶：活力单位 $\geqslant 100\mathrm{U / mg}$   
3.1.10 2,6-二叔丁基对甲酚 $\mathrm{(C_{15}H_{24}O)}$ ：简称BHT。

# 3.2 试剂配制

3.2.1 氢氧化钾溶液 $(50\mathrm{g} / 100\mathrm{g})$ ：称取 $50\mathrm{g}$ 氢氧化钾，加入 $50~\mathrm{mL}$ 水溶解，冷却后，储存于聚乙烯瓶中。  
3.2.2 石油醚-乙醚溶液 $(1 + 1)$ ：量取 $200~\mathrm{mL}$ 石油醚，加入 $200~\mathrm{mL}$ 乙醚，混匀。

3.2.3 有机系过滤头（孔径为 $0.22\mu \mathrm{m}$ ）。

# 3.3 标准品

# 3.3.1 维生素A标准品

视黄醇 $\left(\mathrm{C}_{20} \mathrm{H}_{30} \mathrm{O}, \mathrm{CAS}\right.$ 号：68-26-8)：纯度 $\geqslant 95\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.3.2 维生素E标准品

3.3.2.1 $\alpha$ -生育酚 $(\mathrm{C}_{29}\mathrm{H}_{50}\mathrm{O}_2$ ，CAS号：10191-41-0）：纯度 $\geq 95\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.2.2 $\beta$ -生育酚 $(\mathrm{C}_{28}\mathrm{H}_{48}\mathrm{O}_2$ ，CAS号：148-03-8）：纯度 $\geqslant 95\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.2.3 $\gamma$ -生育酚 $(\mathrm{C}_{28}\mathrm{H}_{48}\mathrm{O}_2$ ，CAS号：54-28-4）：纯度 $\geqslant 95\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.2.4 $\delta$ -生育酚 $(\mathrm{C}_{27}\mathrm{H}_{46}\mathrm{O}_2$ ，CAS号：119-13-1)：纯度 $\geqslant 95\%$ ，或经国家认证并授予标准物质证书的标准物质。

# 3.4 标准溶液配制

3.4.1 维生素A标准储备溶液 $(0.500\mathrm{mg / mL})$ ：准确称取 $25.0\mathrm{mg}$ 维生素A标准品，用无水乙醇溶解后，转移入 $50~\mathrm{mL}$ 容量瓶中，定容至刻度，此溶液浓度约为 $0.500\mathrm{mg / mL}$ 。将溶液转移至棕色试剂瓶中，密封后，在一 $20^{\circ}C$ 下避光保存，有效期1个月。临用前将溶液回温至 $20^{\circ}C$ ，并进行浓度校正（校正方法参见附录B）。  
3.4.2 维生素E标准储备溶液 $(1.00\mathrm{mg / mL})$ ：分别准确称取 $\alpha-$ 生育酚、 $\beta-$ 生育酚、 $\gamma-$ 生育酚和 $\delta-$ 生育酚各 $50.0\mathrm{mg}$ ，用无水乙醇溶解后，转移入 $50~\mathrm{mL}$ 容量瓶中，定容至刻度，此溶液浓度约为 $1.00\mathrm{mg / mL}$ 。将溶液转移至棕色试剂瓶中，密封后，在 $-20^{\circ}\mathrm{C}$ 下避光保存，有效期6个月。临用前将溶液回温至 $20^{\circ}\mathrm{C}$ ，并进行浓度校正(校正方法参见附录B)。  
3.4.3 维生素A和维生素E混合标准溶液中间液：准确吸取维生素A标准储备溶液 $1.00\mathrm{mL}$ 和维生素E标准储备溶液各 $5.00\mathrm{mL}$ 于同一 $50~\mathrm{mL}$ 容量瓶中，用甲醇定容至刻度，此溶液中维生素A浓度为 $10.0~\mu \mathrm{g / mL}$ ，维生素E各生育酚浓度为 $100~\mu \mathrm{g / mL}$ 。在一 $20^{\circ}C$ 下避光保存，有效期半个月。  
3.4.4 维生素A和维生素E标准系列工作溶液：分别准确吸取维生素A和维生素E混合标准溶液中间液 $0.20\mathrm{mL},0.50\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},4.00\mathrm{mL},6.00\mathrm{mL}$ 于 $10~\mathrm{mL}$ 棕色容量瓶中，用甲醇定容至刻度，该标准系列中维生素A浓度为 $0.20~\mu \mathrm{g / mL}$ 、 $0.50~\mu \mathrm{g / mL}$ 、 $1.00~\mu \mathrm{g / mL}$ 、 $2.00~\mu \mathrm{g / mL}$ 、 $4.00~\mu \mathrm{g / mL}$ 、 $6.00~\mu \mathrm{g / mL}$ ，维生素E浓度为 $2.00~\mu \mathrm{g / mL}$ 、 $5.00~\mu \mathrm{g / mL}$ 、 $10.0~\mu \mathrm{g / mL}$ 、 $20.0~\mu \mathrm{g / mL}$ 、 $40.0~\mu \mathrm{g / mL}$ 、 $60.0~\mu \mathrm{g / mL}$ 。临用前配制。

# 4 仪器和设备

4.1 分析天平：感量为 $0.01\mathrm{mg}$   
4.2 恒温水浴振荡器。  
4.3 旋转蒸发仪。  
4.4 氮吹仪。  
4.5 紫外分光光度计。  
4.6 分液漏斗萃取净化振荡器。

4.7 高效液相色谱仪：带紫外检测器或二极管阵列检测器或荧光检测器。

# 5 分析步骤

# 5.1 试样制备

将一定数量的样品按要求经过缩分、粉碎均质后，储存于样品瓶中，避光冷藏，尽快测定。

# 5.2 试样处理

警示：使用的所有器皿不得含有氧化性物质；分液漏斗活塞玻璃表面不得涂油；处理过程应避免紫外光照，尽可能避光操作；提取过程应在通风柜中操作。

# 5.2.1 皂化

# 5.2.1.1 不含淀粉样品

称取 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 经均质处理的固体试样或 $50~\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 液体试样于 $150~\mathrm{mL}$ 平底烧瓶中，固体试样需加入约 $20~\mathrm{mL}$ 温水，混匀，再加入 $1.0\mathrm{g}$ 抗坏血酸和 $0.1\mathrm{g}$ BHT，混匀，加入 $30~\mathrm{mL}$ 无水乙醇，加入 $10~\mathrm{mL}\sim 20~\mathrm{mL}$ 氢氧化钾溶液，边加边振摇，混匀后于 $80^{\circ}C$ 恒温水浴震荡皂化 $30\mathrm{min}$ ，皂化后立即用冷水冷却至室温。

注：皂化时间一般为 $30\mathrm{min}$ ，如皂化液冷却后，液面有浮油，需要加入适量氢氧化钾溶液，并适当延长皂化时间。

# 5.2.1.2 含淀粉样品

称取 $2\mathrm{g}\sim 5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )经均质处理的固体试样或 $50~\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )液体样品于 $150~\mathrm{mL}$ 平底烧瓶中，固体试样需用约 $20~\mathrm{mL}$ 温水混匀，加入 $0.5\mathrm{g}\sim 1\mathrm{g}$ 淀粉酶，放入 $60^{\circ}C$ 水浴避光恒温振荡 $30\mathrm{min}$ 后，取出，向酶解液中加入 $1.0\mathrm{g}$ 抗坏血酸和 $0.1\mathrm{g}$ BHT，混匀，加入 $30~\mathrm{mL}$ 无水乙醇， $10~\mathrm{mL}\sim$ $20~\mathrm{mL}$ 氢氧化钾溶液，边加边振摇，混匀后于 $80^{\circ}C$ 恒温水浴振荡皂化 $30\mathrm{min}$ ，皂化后立即用冷水冷却至室温。

# 5.2.2 提取

将皂化液用 $30~\mathrm{mL}$ 水转入 $250~\mathrm{mL}$ 的分液漏斗中，加入 $50~\mathrm{mL}$ 石油醚-乙醚混合液，振荡萃取 $5\mathrm{min}$ ，将下层溶液转移至另一 $250~\mathrm{mL}$ 的分液漏斗中，加入 $50~\mathrm{mL}$ 的混合醚液再次萃取，合并醚层。

注：如只测维生素A与 $\alpha$ -生育酚，可用石油醚作提取剂。

# 5.2.3 洗涤

用约 $100\mathrm{mL}$ 水洗涤醚层，约需重复3次，直至将醚层洗至中性（可用 $\mathrm{pH}$ 试纸检测下层溶液 $\mathrm{pH}$ 值)，去除下层水相。

# 5.2.4 浓缩

将洗涤后的醚层经无水硫酸钠(约 $3\mathrm{g}$ ) 滤入 $250~\mathrm{mL}$ 旋转蒸发瓶或氮气浓缩管中, 用约 $15~\mathrm{mL}$ 石油醚冲洗分液漏斗及无水硫酸钠 2 次, 并入蒸发瓶内, 并将其接在旋转蒸发仪或气体浓缩仪上, 于 $40^{\circ}\mathrm{C}$ 水浴中减压蒸馏或气流浓缩, 待瓶中醚液剩下约 $2\mathrm{mL}$ 时, 取下蒸发瓶, 立即用氮气吹至近干。用甲醇分次将蒸发瓶中残留物溶解并转移至 $10~\mathrm{mL}$ 容量瓶中, 定容至刻度。溶液过 $0.22\mu \mathrm{m}$ 有机系滤膜后供高效液相色谱测定。

# 5.3 色谱参考条件

色谱参考条件列出如下：

a）色谱柱： $\mathrm{C_{30}}$ 柱（柱长 $250~\mathrm{mm}$ ，内径 $4.6  \mathrm{mm}$ ，粒径 $3\mu \mathrm{m})$ ，或相当者；  
b） 柱温： $20^{\circ}C$   
c) 流动相：A: 水；B: 甲醇，洗脱梯度见表1；  
d) 流速: $0.8 \mathrm{~mL} / \mathrm{min}$ ;   
e）紫外检测波长：维生素A为 $325\mathrm{nm}$ ；维生素E为 $294\mathrm{nm}$   
f）进样量： $10~\mu \mathrm{L}$   
g）标准色谱图和样品色谱图见C.1。

注1：如难以将柱温控制在 $20^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ ，可改用 PFP 柱分离异构体，流动相为水和甲醇梯度洗脱。  
注2：如样品中只含 $\alpha$ -生育酚，不需分离 $\beta$ -生育酚和 $\gamma$ -生育酚，可选用 $\mathrm{C_{18}}$ 柱，流动相为甲醇。  
注3：如有荧光检测器，可选用荧光检测器检测，对生育酚的检测有更高的灵敏度和选择性，可按以下检测波长检测：维生素A激发波长 $328~\mathrm{nm}$ ，发射波长 $440~\mathrm{nm}$ ；维生素E激发波长 $294~\mathrm{nm}$ ，发射波长 $328~\mathrm{nm}$ 。

# 5.4 标准曲线的制作

本法采用外标法定量。将维生素A和维生素E标准系列工作溶液分别注入高效液相色谱仪中，测定相应的峰面积，以峰面积为纵坐标，以标准测定液浓度为横坐标绘制标准曲线，计算直线回归方程。

# 5.5 样品测定

试样液经高效液相色谱仪分析，测得峰面积，采用外标法通过上述标准曲线计算其浓度。在测定过程中，建议每测定10个样品用同一份标准溶液或标准物质检查仪器的稳定性。

# 6 分析结果的表述

试样中维生素A或维生素E的含量按式(1)计算：

$$
X = \frac {\rho \times V \times f \times 1 0 0}{m} \tag {1}
$$

式中：

X ——试样中维生素A或维生素E的含量，维生素A单位为微克每百克 $(\mu \mathrm{g} / 100\mathrm{g})$ ，维生素E单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$

$\rho$ ——根据标准曲线计算得到的试样中维生素A或维生素E的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——换算因子（维生素 $\mathrm{A}: f = 1$ ；维生素 $\mathrm{E}: f = 0.001$ ）；

100 ——试样中量以每100克计算的换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

注：如维生素E的测定结果要用 $\alpha$ -生育酚当量(α-TE)表示，可按下式计算：维生素E(mg $\alpha$ -TE/100 g)= $\alpha$ -生育酚(mg/100 g)+ $\beta$ -生育酚(mg/100 g) $\times 0.5+\gamma$ -生育酚(mg/100 g) $\times 0.1+\delta$ -生育酚(mg/100 g) $\times 0.01$ 。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当取样量为 $5\mathrm{g}$ ，定容 $10\mathrm{mL}$ 时，维生素A的紫外检出限为 $10~\mu \mathrm{g} / 100\mathrm{g}$ ，定量限为 $30~\mu \mathrm{g} / 100\mathrm{g}$ ；生育酚的紫外检出限为 $40~\mu \mathrm{g} / 100\mathrm{g}$ ，定量限为 $120~\mu \mathrm{g} / 100\mathrm{g}$ 。

# 第二法 食品中维生素E的测定 正相高效液相色谱法

# 9 原理

试样中的维生素E经有机溶剂提取、浓缩后，用高效液相色谱酰氨基柱或硅胶柱分离，经荧光检测器检测，外标法定量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯。水为GB/T6682规定的一级水。

# 10.1 试剂

10.1.1 无水乙醇 $\mathrm{(C_2H_5OH)}$ ：色谱纯，经检验不含醛类物质，检查方法参见A.1。  
10.1.2 乙醚 $\left[\left(\mathrm{CH}_{3} \mathrm{CH}_{2}\right)_{2} \mathrm{O}\right]$ ：分析纯，经检验不含氧化物，检查方法参见A.2。  
10.1.3 石油醚 $\left(\mathrm{C}_{5} \mathrm{H}_{12} \mathrm{O}_{2}\right)$ ：沸程为 $30^{\circ} \mathrm{C} \sim 60^{\circ} \mathrm{C}$ 。  
10.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ 。  
10.1.5 正己烷 $(n - C_{6}H_{14})$ ：色谱纯。  
10.1.6 异丙醇 $\left[\mathrm{(CH_3)_2CHOH}\right]$ ：色谱纯。  
10.1.7 叔丁基甲基醚 $\left[\mathrm{CH}_{3} \mathrm{OC}\left(\mathrm{CH}_{3}\right)_{3}\right]$ ：色谱纯。  
10.1.8 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
10.1.9 四氢呋喃 $\left(\mathrm{C}_{4} \mathrm{H}_{8} \mathrm{O}\right)$ ：色谱纯。  
10.1.10 1,4-二氧六环 $\mathrm{(C_4H_8O_2)}$ ：色谱纯。  
10.1.11 2,6-二叔丁基对甲酚 $\left(\mathrm{C}_{15} \mathrm{H}_{24} \mathrm{O}\right)$ ：简称BHT。  
10.1.12 有机系过滤头（孔径为 $0.22\mu \mathrm{m}$ ）。

# 10.2 试剂配制

10.2.1 石油醚-乙醚溶液 $(1 + 1)$ ：量取 $200~\mathrm{mL}$ 石油醚，加入 $200~\mathrm{mL}$ 乙醚，混匀，临用前配制。  
10.2.2 流动相：正己烷 $+$ [叔丁基甲基醚-四氢呋喃-甲醇混合液 $(20 + 1 + 0.1)] = 90 + 10$ ，临用前配制。

# 10.3 标准品

10.3.1 $\alpha$ -生育酚 $(\mathrm{C}_{29}\mathrm{H}_{50}\mathrm{O}_2$ ，CAS号：10191-41-0）：纯度 $\geqslant 95\%$ ，或经国家认证并授予标准物质证书的标准物质；  
10.3.2 $\beta$ -生育酚 $(\mathrm{C}_{28}\mathrm{H}_{48}\mathrm{O}_2$ ，CAS号：148-03-8）：纯度 $\geqslant 95\%$ ，或经国家认证并授予标准物质证书的标准物质；

# 12.2 试样处理

警示：使用的所有器皿不得含有氧化性物质；分液漏斗活塞玻璃表面不得涂油；处理过程应避免紫外光照，尽可能避光操作。

# 12.2.1 植物油脂

称取 $0.5\mathrm{g}\sim 2\mathrm{g}$ 油样（准确至 $0.01\mathrm{g})$ 于 $25~\mathrm{mL}$ 的棕色容量瓶中，加入 $0.1\mathrm{g}$ BHT，加入 $10~\mathrm{mL}$ 流动相超声或涡旋振荡溶解后，用流动相定容至刻度，摇匀。过孔径为 $0.22\mu \mathrm{m}$ 有机系滤头于棕色进样瓶中，待进样。

# 12.2.2 奶油、黄油

称取 $2\mathrm{g}\sim 5\mathrm{g}$ 样品(准确至 $0.01\mathrm{g})$ 于 $50~\mathrm{mL}$ 的离心管中，加入 $0.1\mathrm{g}$ BHT， $45^{\circ}C$ 水浴融化，加入 $5\mathrm{g}$ 无水硫酸钠，涡旋 $1\mathrm{min}$ ，混匀，加入 $25~\mathrm{mL}$ 流动相超声或涡旋振荡提取，离心，将上清液转移至浓缩瓶中，再用 $20~\mathrm{mL}$ 流动相重复提取1次，合并上清液至浓缩瓶，在旋转蒸发器或气体浓缩仪上，于 $45^{\circ}C$ 水浴中减压蒸馏或气流浓缩，待瓶中醚剩下约 $2\mathrm{mL}$ 时，取下蒸发瓶，立即用氮气吹干。用流动相将浓缩瓶中残留物溶解并转移至 $10~\mathrm{mL}$ 容量瓶中，定容至刻度，摇匀。溶液过 $0.22\mu \mathrm{m}$ 有机系滤膜后供高效液相色谱测定。

# 12.2.3 坚果、豆类、辣椒粉等干基植物样品

称取 $2\mathrm{g}\sim 5\mathrm{g}$ 样品（准确至 $0.01\mathrm{g}$ ），用索氏提取仪或加速溶剂萃取仪提取其中的植物油脂，将含油脂的提取溶剂转移至 $250~\mathrm{mL}$ 蒸发瓶内，于 $40^{\circ}\mathrm{C}$ 水浴中减压蒸馏或气流浓缩至干，取下蒸发瓶，用 $10~\mathrm{mL}$ 流动相将油脂转移至 $25~\mathrm{mL}$ 容量瓶中，加入 $0.1\mathrm{g}$ BHT，超声或涡旋振荡溶解后，用流动相定容至刻度，摇匀。过孔径为 $0.22\mu \mathrm{m}$ 有机系滤头于棕色进样瓶中，待进样。

# 12.3 色谱参考条件

色谱参考条件列出如下：

a）色谱柱：酰氨基柱(柱长 $150~\mathrm{mm}$ ，内径 $3.0  \mathrm{mm}$ ，粒径 $1.7\mu \mathrm{m})$ 或相当者；  
b） 柱温： $30^{\circ}C$   
c）流动相：正己烷 $+[$ 叔丁基甲基醚-四氢呋喃-甲醇混合液 $(20 + 1 + 0.1)] = 90 + 10$   
d) 流速: $0.8 \mathrm{~mL} / \mathrm{min}$ ;   
e）荧光检测波长：激发波长 $294\mathrm{nm}$ ，发射波长 $328~\mathrm{nm}$   
f）进样量： $10\mu \mathrm{L}$

注：可用Si60硅胶柱(柱长 $250~\mathrm{mm}$ ，内径 $4.6\mathrm{mm}$ ，粒径 $5\mu \mathrm{m})$ 分离4种生育酚异构体，推荐流动相为正己烷与1，4-二氧六环按 $(95 + 5)$ 的比例混合。

# 12.4 标准曲线的制作

本法采用外标法定量。将维生素E标准系列工作溶液从低浓度到高浓度分别注入高效液相色谱仪中，测定相应的峰面积。以峰面积为纵坐标，标准溶液浓度为横坐标绘制标准曲线，计算直线回归方程。

# 12.5 样品测定

试样液经高效液相色谱仪分析，测得峰面积，采用外标法通过上述标准曲线计算其浓度。在测定过程中，建议每测定10个样品用同一份标准溶液或标准物质检查仪器的稳定性。

# 13 分析结果的表述

试样中 $\alpha$ -生育酚、 $\beta$ -生育酚、 $\gamma$ -生育酚或 $\delta$ -生育酚的含量按式(2)计算：

$$
X = \frac {\rho \times V \times f \times 1 0 0}{m} \tag {2}
$$

式中：

$X$ ——试样中 $\alpha$ -生育酚、 $\beta$ -生育酚、 $\gamma$ -生育酚或 $\delta$ -生育酚的含量，单位为毫克每百克 $(\mathrm{mg} / 100\mathrm{g})$ ； $\rho$ ——根据标准曲线计算得到的试样中 $\alpha$ -生育酚、 $\beta$ -生育酚、 $\gamma$ -生育酚或 $\delta$ -生育酚的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——换算因子 $(f = 0.001)$

100 ——试样中量以每百克计算的换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

注：如维生素E的测定结果要用 $\alpha$ -生育酚当量(α-TE)表示，可按下式计算：维生素E(mg $\alpha$ -TE/100 g)= $\alpha$ -生育酚(mg/100 g)+ $\beta$ -生育酚(mg/100 g) $\times 0.5+\gamma$ -生育酚(mg/100 g) $\times 0.1+\delta$ -生育酚(mg/100 g) $\times 0.01$ 。

# 14 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 19.2 试样处理

注：处理过程应避免紫外光照，尽可能避光操作。

# 19.2.1 皂化

# 19.2.1.1 不含淀粉样品

称取 $2\mathrm{g}$ （准确至 $0.01\mathrm{g}$ ）经均质处理的试样于 $50~\mathrm{mL}$ 具塞离心管中，加入 $100\mu \mathrm{L}$ 维生素 $\mathrm{D}_2 - \mathrm{d}_3$ 和维生素 $\mathrm{D}_3 - \mathrm{d}_3$ 混合内标溶液和 $0.4\mathrm{g}$ 抗坏血酸，加入 $6\mathrm{mL}$ 约 $40^{\circ}C$ 温水，涡旋 $1\mathrm{min}$ ，加入 $12\mathrm{mL}$ 乙醇，涡旋 $30~\mathrm{s}$ ，再加入 $6\mathrm{mL}$ 氢氧化钾溶液，涡旋 $30~\mathrm{s}$ 后放入恒温振荡器中， $80^{\circ}C$ 避光恒温水浴振荡 $30\mathrm{min}$ （如样品组织较为紧密，可每隔 $5\mathrm{min}\sim 10\mathrm{min}$ 取出涡旋 $0.5\mathrm{min}$ ），取出放入冷水浴降温。

注：一般皂化时间为 $30\mathrm{min}$ ，如皂化液冷却后，液面有浮油，需要加入适量氢氧化钾溶液，并适当延长皂化时间。

# 19.2.1.2 含淀粉样品

称取 $2\mathrm{g}$ (准确至 $0.01\mathrm{g}$ ) 经均质处理的试样于 $50~\mathrm{mL}$ 具塞离心管中，加入 $100~\mu \mathrm{L}$ 维生素 $\mathrm{D}_2 - \mathrm{d}_3$ 和维生素 $\mathrm{D}_3 - \mathrm{d}_3$ 混合内标溶液和 $0.4\mathrm{g}$ 淀粉酶，加入 $10~\mathrm{mL}$ 约 $40^{\circ}\mathrm{C}$ 温水，放入恒温振荡器中， $60^{\circ}\mathrm{C}$ 避光恒温振荡 $30\mathrm{min}$ 后，取出放入冷水浴降温，向冷却后的酶解液中加入 $0.4\mathrm{g}$ 抗坏血酸、 $12~\mathrm{mL}$ 乙醇，涡旋 $30~\mathrm{s}$ ，再加入 $6\mathrm{mL}$ 氢氧化钾溶液，涡旋 $30~\mathrm{s}$ 后放入恒温振荡器中，同19.2.1.1皂化 $30\mathrm{min}$ 。

# 19.2.2 提取

# 19.2.3 净化

# 19.4 标准曲线的制作

分别将维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 标准系列工作液由低浓度到高浓度依次进样，以维生素 $\mathrm{D}_2$ 、维生素 $\mathrm{D}_3$ 与相应同位素内标的峰面积比值为纵坐标，以维生素 $\mathrm{D}_2$ 、维生素 $\mathrm{D}_3$ 标准系列工作液浓度为横坐标分别绘制维生素 $\mathrm{D}_2$ 、维生素 $\mathrm{D}_3$ 标准曲线。

# 19.5 样品测定

将待测样液依次进样，得到待测物与内标物的峰面积比值，根据标准曲线得到测定液中维生素 $\mathrm{D}_2$ 、维生素 $\mathrm{D}_3$ 的浓度。待测样液中的响应值应在标准曲线线性范围内，超过线性范围则应减少取样量重新按19.2进行处理后再进样分析。

# 20 分析结果的表述

试样中维生素 $\mathrm{D}_2$ 、维生素 $\mathrm{D}_3$ 的含量按式(3)计算：

$$
X = \frac {\rho \times V \times f \times 1 0 0}{m} \tag {3}
$$

式中：

$X$ ——试样中维生素 $\mathrm{D}_2$ (或维生素 $\mathrm{D}_3$ ) 的含量, 单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$

$\rho$ ——根据标准曲线计算得到的试样中维生素 $\mathrm{D}_2$ （或维生素 $\mathrm{D}_3$ )的浓度，单位为微克每毫升 $(\mu \mathrm{g / mL})$

$V$ ——定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——稀释倍数；

100 ——试样中量以每 100 克计算的换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

注：如试样中同时含有维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ ，维生素D的测定结果以维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 含量之和计算。

# 21 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 22 其他

当取样量为 $2\mathrm{g}$ 时，维生素 $\mathrm{D}_2$ 的检出限为 $1\mu \mathrm{g} / 100\mathrm{g}$ ，定量限为 $3\mu \mathrm{g} / 100\mathrm{g}$ ；维生素 $\mathrm{D}_3$ 的检出限为 $0.2\mu \mathrm{g} / 100\mathrm{g}$ ；定量限为 $0.6\mu \mathrm{g} / 100\mathrm{g}$ 。

# 第四法 食品中维生素D的测定 高效液相色谱法

# 23 原理

试样中的维生素 $\mathrm{D}_2$ 或维生素 $\mathrm{D}_3$ 经氢氧化钾乙醇溶液皂化（含淀粉试样先用淀粉酶酶解）、提取、净化、浓缩后，用正相高效液相色谱半制备，反相高效液相色谱 $\mathrm{C}_{18}$ 柱色谱分离，经紫外或二极管阵列检测器检测，内标法(或外标法)定量。如测定维生素 $\mathrm{D}_2$ ，可用维生素 $\mathrm{D}_3$ 作内标；如测定维生素 $\mathrm{D}_3$ ，可用维生素 $\mathrm{D}_2$ 作内标。

# 26 分析步骤

# 26.1 试样制备

将一定数量的样品按要求经过缩分、粉碎、均质后，储存于样品瓶中，避光冷藏，尽快测定。

# 26.2 试样处理

处理过程应避免紫外光照，尽可能避光操作。如样品中只含有维生素 $\mathrm{D}_3$ ，可用维生素 $\mathrm{D}_2$ 做内标；如只含有维生素 $\mathrm{D}_2$ ，可用维生素 $\mathrm{D}_3$ 做内标；否则，用外标法定量，但需要验证回收率能满足检测要求。

# 26.2.1 不含淀粉样品

称取 $5\mathrm{g}\sim 10\mathrm{g}$ (准确至 $0.01\mathrm{g}$ ) 经均质处理的固体试样或 $50\mathrm{g}$ (准确至 $0.01\mathrm{g}$ ) 液体样品于 $150~\mathrm{mL}$ 平底烧瓶中，固体试样需加入 $20~\mathrm{mL}\sim 30~\mathrm{mL}$ 温水，加入 $1.00~\mathrm{mL}$ 内标使用溶液（如测定维生素 $\mathrm{D}_2$ ，用维生素 $\mathrm{D}_3$ 作内标；如测定维生素 $\mathrm{D}_3$ ，用维生素 $\mathrm{D}_2$ 作内标。），再加入 $1.0\mathrm{g}$ 抗坏血酸和 $0.1\mathrm{g}$ BHT，混匀。加入 $30~\mathrm{mL}$ 无水乙醇，加入 $10~\mathrm{mL}\sim 20~\mathrm{mL}$ 氢氧化钾溶液，边加边振摇，混匀后于恒温磁力搅拌器上 $80^{\circ}C$ 回流皂化 $30\mathrm{min}$ ，皂化后立即用冷水冷却至室温。

注：一般皂化时间为 $30\mathrm{min}$ ，如皂化液冷却后，液面有浮油，需要加入适量氢氧化钾乙醇溶液，并适当延长皂化时间。

# 26.2.2 含淀粉样品

称取 $5\mathrm{g}\sim 10\mathrm{g}$ （准确至 $0.01\mathrm{g}$ )经均质处理的固体试样或 $50~\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )液体样品于 $150~\mathrm{mL}$ 平底烧瓶中，固体试样需加入约 $20~\mathrm{mL}$ 温水，加入 $1.00~\mathrm{mL}$ 内标使用溶液（如测定维生素 $\mathrm{D}_2$ ，用维生素 $\mathrm{D}_3$ 作内标；如测定维生素 $\mathrm{D}_3$ ，用维生素 $\mathrm{D}_2$ 作内标)和 $1\mathrm{g}$ 淀粉酶，放入 $60^{\circ}C$ 恒温水浴振荡 $30\mathrm{min}$ ，向酶解液中加入 $1.0\mathrm{g}$ 抗坏血酸和 $0.1\mathrm{g}$ BHT，混匀。加入 $30~\mathrm{mL}$ 无水乙醇， $10~\mathrm{mL}\sim 20~\mathrm{mL}$ 氢氧化钾溶液，边加边振摇，混匀后于恒温磁力搅拌器上 $80^{\circ}C$ 回流皂化 $30\mathrm{min}$ ，皂化后立即用冷水冷却至室温。

# 26.2.3 提取

将皂化液用 $30~\mathrm{mL}$ 水转入 $250~\mathrm{mL}$ 的分液漏斗中，加入 $50~\mathrm{mL}$ 石油醚，振荡萃取 $5\mathrm{min}$ ，将下层溶液转移至另一 $250~\mathrm{mL}$ 的分液漏斗中，加入 $50~\mathrm{mL}$ 的石油醚再次萃取，合并醚层。

# 26.2.4 洗涤

用约 $150~\mathrm{mL}$ 水洗涤醚层，约需重复3次，直至将醚层洗至中性（可用 $\mathrm{pH}$ 试纸检测下层溶液 $\mathrm{pH}$ 值)，去除下层水相。

# 26.2.5 浓缩

将洗涤后的醚层经无水硫酸钠(约 $3\mathrm{g}$ )滤入 $250~\mathrm{mL}$ 旋转蒸发瓶或氮气浓缩管中，用约 $15~\mathrm{mL}$ 石油醚冲洗分液漏斗及无水硫酸钠2次，并入蒸发瓶内，并将其接在旋转蒸发器或气体浓缩仪上，于 $40^{\circ}\mathrm{C}$ 水浴中减压蒸馏或气流浓缩，待瓶中醚剩下约 $2\mathrm{mL}$ 时，取下蒸发瓶，氮吹至干，用正己烷定容至 $2\mathrm{mL}$ $0.22\mu \mathrm{m}$ 有机系滤膜过滤供半制备正相高效液相色谱系统半制备，净化待测液。

# 26.3 测定条件

# 26.3.1 维生素D待测液的净化

# 26.3.1.1 半制备正相高效液相色谱参考条件

半制备正相高效液相色谱参考条件列出如下：

a）色谱柱：硅胶柱，柱长 $250~\mathrm{mm}$ ，内径 $4.6\mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或具同等性能的色谱柱；  
b）流动相：环己烷 $+$ 正己烷 $(1 + 1)$ ，并按体积分数 $0.8\%$ 加入异丙醇；  
c) 流速: $1 \mathrm{~mL} / \mathrm{min}$ ;   
d) 波长: $264 \mathrm{~nm}$ ;  
e) 柱温: $35^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ ;   
f）进样体积： $500\mu \mathrm{L}$

# 26.3.1.2 半制备正相高效液相色谱系统适用性试验

取约 $1.00\mathrm{mL}$ 维生素 $\mathrm{D}_2$ 和 $\mathrm{D}_3$ 标准中间使用液于 $10~\mathrm{mL}$ 具塞试管中，在 $40^{\circ}\mathrm{C} \pm 2^{\circ}\mathrm{C}$ 的氮吹仪上吹干。残渣用 $10~\mathrm{mL}$ 正己烷振荡溶解。取该溶液 $100\mu \mathrm{L}$ 注入液相色谱仪中测定，确定维生素D保留时间。然后将 $500\mu \mathrm{L}$ 待测液注入液相色谱仪中，根据维生素D标准溶液保留时间收集维生素D馏分于试管中。将试管置于 $40^{\circ}\mathrm{C}$ 水浴氮气吹干，取出准确加入 $1.0~\mathrm{mL}$ 甲醇，残渣振荡溶解，即为维生素D测定液。

# 26.3.2 反相液相色谱参考条件

反相液相色谱参考条件列出如下：

a）色谱柱： $\mathrm{C_{18}}$ 柱，柱长 $250~\mathrm{mm}$ ，柱内径 $4.6  \mathrm{mm}$ ，粒径 $5\mu \mathrm{m}$ ，或具同等性能的色谱柱；  
b）流动相：甲醇 $+$ 水 $= 95 + 5$   
c) 流速: $1 \mathrm{~mL} / \mathrm{min}$ ;   
d) 检测波长: $264 \mathrm{~nm}$ ;   
e) 柱温: $35^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ ;   
f）进样量： $100\mu \mathrm{L}$

# 26.4 标准曲线的制作

分别将维生素 $\mathrm{D}_2$ 或维生素 $\mathrm{D}_3$ 标准系列工作液注入反相液相色谱仪中，得到维生素 $\mathrm{D}_2$ 和维生素 $\mathrm{D}_3$ 峰面积。以两者峰面积比为纵坐标，以维生素 $\mathrm{D}_2$ 或维生素 $\mathrm{D}_3$ 标准工作液浓度为横坐标分别绘制维生素 $\mathrm{D}_2$ 或维生素 $\mathrm{D}_3$ 标准曲线。

# 26.5 样品测定

吸取维生素D测定液 $100\mu \mathrm{L}$ 注入反相液相色谱仪中，得到待测物与内标物的峰面积比值，根据标准曲线得到待测液中维生素 $\mathrm{D}_2$ （或维生素 $\mathrm{D}_3$ )的浓度。

# 27 分析结果的表述

试样中维生素 $\mathrm{D}_2$ （或维生素 $\mathrm{D}_3$ ）的含量按式(4)计算：

$$
X = \frac {\rho \times V \times f \times 1 0 0}{m} \tag {4}
$$

式中：

$X$ ——试样中维生素 $\mathrm{D}_2$ (或维生素 $\mathrm{D}_3$ ) 的含量, 单位为微克每百克 $(\mu \mathrm{g} / 100 \mathrm{~g})$

$\rho$ ——根据标准曲线计算得到的试样中维生素 $\mathrm{D}_2$ （或维生素 $\mathrm{D}_3$ )的浓度，单位为微克每毫升 $(\mu \mathrm{g / mL})$

V ——正己烷定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ —待测液稀释过程的稀释倍数；

100 ——试样中量以每 100 克计算的换算系数；

$m$ ——试样的称样量，单位为克 $(\mathrm{g})$ 。

计算结果保留三位有效数字。

# 28 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $15\%$ 。

# 29 其他

当取样量为 $10\mathrm{g}$ 时，维生素 $\mathrm{D}_2$ 或维生素 $\mathrm{D}_3$ 的检出限为 $0.7\mu \mathrm{g} / 100\mathrm{g}$ ，定量限为 $2\mu \mathrm{g} / 100\mathrm{g}$ 。

# A.1 无水乙醇中醛类物质检查方法

# A.1.1 试剂

A.1.1.1 硝酸银。  
A.1.1.2 氢氧化钠。  
A.1.1.3 氨水。

# A.1.2 试剂配制

A.1.2.1 $5\%$ 硝酸银溶液：称取 $5.00\mathrm{g}$ 硝酸银，加入 $100~\mathrm{mL}$ 水溶解，储存于棕色试剂瓶中。  
A.1.2.2 $10\%$ 氢氧化钠溶液：称取 $10.00\mathrm{g}$ 氢氧化钠，加入 $100~\mathrm{mL}$ 水溶解，储存于聚乙烯瓶中。  
A.1.2.3 银氨溶液：加氨水至 $5\%$ 硝酸银中，直至生成的沉淀重新溶解，加入数滴 $10\%$ 氢氧化钠溶液，如发生沉淀，再加入氨水至沉淀溶解。

# A.1.3 操作方法

取 $2\mathrm{mL}$ 银氨溶液于试管中，加入少量乙醇，摇匀，再加入氢氧化钠溶液，加热，放置冷却后，若有银镜反应，则表示乙醇中有醛。

# A.1.4 结果处理

换用色谱纯的无水乙醇或对现有乙醇进行脱醛处理：取 $2\mathrm{g}$ 硝酸银溶于少量水中，取 $4\mathrm{g}$ 氢氧化钠溶于温乙醇中，将两者倾入 $1\mathrm{L}$ 乙醇中，振摇后，放置暗处 $2\mathrm{d}$ ，期间不时振摇，经过滤，置蒸馏瓶中蒸馏，弃去 $150~\mathrm{mL}$ 初馏液。

# A.2 乙醚中过氧化物检查方法

# A.2.1 试剂

A.2.1.1 碘化钾。  
A.2.1.2 淀粉。

# A.2.2 试剂配制

A.2.2.1 $10\%$ 碘化钾溶液：称取 $10.00\mathrm{g}$ 碘化钾，加入 $100~\mathrm{mL}$ 水溶解，储存于棕色试剂瓶中。  
A.2.2.2 $0.5\%$ 淀粉溶液：称取 $0.50\mathrm{g}$ 可溶性淀粉，加入 $100~\mathrm{mL}$ 水溶解，储存于试剂瓶中。

# A.2.3 操作方法

用 $5\mathrm{mL}$ 乙醚加 $1\mathrm{mL}10\%$ 碘化钾溶液，振摇 $1\mathrm{min}$ ，如水层呈黄色或加4滴 $0.5\%$ 淀粉溶液，水层呈蓝色，表明含过氧化物。

# A.2.4 结果处理

换用色谱纯的无水乙醚或对现有试剂进行重蒸，重蒸乙醚时需在蒸馏瓶中放入纯铁丝或纯铁粉，弃去 $10\%$ 初馏液和 $10\%$ 残留液。

# 维生素A、D、E标准溶液浓度校正方法

维生素A、维生素D、维生素E标准溶液配制后，在使用前需要对其浓度进行校正，具体操作如下：

a）取视黄醇标准储备溶液 $50\mu \mathrm{L}$ 于 $10~\mathrm{mL}$ 的棕色容量瓶中，用无水乙醇定容至刻度，混匀，用 $1\mathrm{cm}$ 石英比色杯，以无水乙醇为空白参比，按表B.1的测定波长测定其吸光度；

# 前言

本标准代替GB/T5009.111—2003《谷物及其制品中脱氧雪腐镰刀菌烯醇的测定》、GB/T23503—2009《食品中脱氧雪腐镰刀菌烯醇的测定 免疫亲和层析净化高效液相色谱法》、SN/T1571—2005《进出口粮谷中呕吐毒素检验方法 液相色谱法》。

本标准与GB/T5009.111—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中脱氧雪腐镰刀菌烯醇及其乙酰化衍生物的测定”；  
——增加了方法的适用范围；  
——增加了食品中脱氧雪腐镰刀菌烯醇乙酰化衍生物的测定；  
——增加了同位素稀释液相色谱-串联质谱法；  
——增加了固相萃取柱净化的前处理方式；  
——增加了免疫亲和柱净化-高效液相色谱法；  
——增加了商业化免疫亲和柱评价技术参数要求。

# 食品安全国家标准

# 食品中脱氧雪腐镰刀菌烯醇及其乙酰化衍生物的测定

# 1 范围

本标准规定了食品中脱氧雪腐镰刀菌烯醇及其乙酰化衍生物的测定方法。

第一法为同位素稀释液相色谱-串联质谱法，适用于谷物及其制品、酒类、酱油、醋、酱及酱制品中脱氧雪腐镰刀菌烯醇、3-乙酰脱氧雪腐镰刀菌烯醇和15-乙酰脱氧雪腐镰刀菌烯醇的测定。

第二法为免疫亲和层析净化高效液相色谱法，适用于谷物及其制品、酒类、酱油、醋、酱及酱制品中脱氧雪腐镰刀菌烯醇的测定。

第三法为薄层色谱测定法，第四法为酶联免疫吸附筛查法，适用于谷物及其制品中脱氧雪腐镰刀菌烯醇的测定。

# 第一法 同位素稀释液相色谱-串联质谱法

# 2 原理

试样中的脱氧雪腐镰刀菌烯醇、3-乙酰脱氧雪腐镰刀菌烯醇和15-乙酰脱氧雪腐镰刀菌烯醇用水和乙腈的混合溶液提取，提取上清液经固相萃取柱或免疫亲和柱净化，浓缩、定容和过滤后，超高压液相色谱分离，串联质谱检测，同位素内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

# 3.1 试剂

3.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
3.1.3 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ 。  
3.1.4 氨水 $\left(\mathrm{NH}_{3} \cdot \mathrm{H}_{2} \mathrm{O}\right)$ 。  
3.1.5 甲酸（HCOOH）。  
3.1.6 氮气 $(\mathrm{N}_2)$ ：纯度 $\geq 99.9\%$

# 3.2 试剂配制

3.2.1 乙腈-水溶液 $(84 + 16)$ ：量取 $160~\mathrm{mL}$ 水加入到 $840~\mathrm{mL}$ 乙腈中，混匀。  
3.2.2 乙腈饱和的正己烷溶液：量取 $200~\mathrm{mL}$ 正己烷于 $250~\mathrm{mL}$ 分液漏斗中，加入少量乙腈，剧烈振摇数分钟，静置分层，弃去下层乙腈层即得。

3.2.3 甲醇-水溶液 $(5 + 95)$ ：量取 $5\mathrm{mL}$ 甲醇加入到 $95~\mathrm{mL}$ 水中，混匀。  
3.2.4 $0.01\%$ 氨水溶液：取 $100~\mu \mathrm{L}$ 氨水加入到 $1000~\mathrm{mL}$ 水中，混匀(仅供离子源模式为ESI-时使用)。  
3.2.5 $0.1\%$ 甲酸溶液：取 $1\mathrm{mL}$ 甲酸加入到 $1000~\mathrm{mL}$ 水中，混匀(仅供离子源模式为ESI+时使用）。

# 3.3 标准品

3.3.1 脱氧雪腐镰刀菌烯醇 $\mathrm{(DON,C_{15}H_{20}O_6}$ ,CAS号：51481-10-8）：纯度 $\geq 99\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.2 3-乙酰脱氧雪腐镰刀菌烯醇(3-ADON, $\mathrm{C_{17}H_{22}O_7}$ , CAS号:50722-38-8):纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.3 15-乙酰脱氧雪腐镰刀菌烯醇(15-ADON, $\mathrm{C}_{17}\mathrm{H}_{22}\mathrm{O}_7$ , CAS号:88337-96-6)：纯度 $\geqslant 99\%$ ，或经国家认证并授予标准物质证书的标准物质。  
3.3.4 $^{13}\mathrm{C}_{15}$ -脱氧雪腐镰刀菌烯醇同位素标准溶液 $(^{13}\mathrm{C - DON},^{13}\mathrm{C}_{15}\mathrm{H}_{20}\mathrm{O}_6)$ ： $25\mu \mathrm{g / mL}$ ，纯度 $\geq 99\%$ 。  
3.3.5 ${}^{13}\mathrm{C}_{17}-3-$ 乙酰-脱氧雪腐镰刀菌烯醇同位素标准溶液（ ${}^{13}\mathrm{C}-3-\mathrm{ADON},{}^{13}\mathrm{C}_{17}\mathrm{H}_{22}\mathrm{O}_7$ ）： $25~\mu \mathrm{g / mL}$ ，纯度 $\geqslant 99\%$ 。

# 3.4 标准溶液配制

3.4.1 标准储备溶液 $(100\mu \mathrm{g} / \mathrm{mL})$ : 分别称取 DON、3-ADON 和 15-ADON $1\mathrm{mg}$ (准确至 $0.01\mathrm{mg}$ ), 分别用乙腈溶解并定容至 $10\mathrm{mL}$ 。将溶液转移至试剂瓶中, 在 $-20^{\circ}\mathrm{C}$ 下密封保存, 有效期 1 年。  
3.4.2 混合标准工作溶液 $(10\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取 $100~\mu \mathrm{g / mL}$ DON、3-ADON和15-ADON标准储备液各 $1.0\mathrm{mL}$ 于同一 $10~\mathrm{mL}$ 容量瓶中，加乙腈定容至刻度。在 $-20^{\circ}\mathrm{C}$ 下密封保存，有效期半年。  
3.4.3 混合同位素内标工作液 $(1\mu \mathrm{g} / \mathrm{mL})$ ：准确吸取 $^{13}\mathrm{C}_{15}$ -DON和 $^{13}\mathrm{C}_{17}$ -3-ADON同位素内标 $(25~\mu \mathrm{g / mL})$ 各 $1\mathrm{mL}$ 于同一 $25~\mathrm{mL}$ 容量瓶中，加乙腈定容至刻度。在一 $20^{\circ}C$ 下密封保存，有效期半年。  
3.4.4 标准系列工作溶液：准确移取适量混合标准工作溶液和混合同位素内标工作液，用初始流动相配制成 $10\mathrm{ng / mL},20\mathrm{ng / mL},40\mathrm{ng / mL},80\mathrm{ng / mL},160\mathrm{ng / mL},320\mathrm{ng / mL},640\mathrm{ng / mL}$ 的混合标准系列，其中同位素内标浓度为 $100\mathrm{ng / mL}$ 。标准系列溶液于 $4^{\circ}C$ 保存，有效期7d。

# 4 仪器和设备

4.1 液相色谱-串联质谱仪：带电喷雾离子源。  
4.2 电子天平：感量 $0.01\mathrm{g}$ 和 $0.00001\mathrm{g}$ 。  
4.3 高速粉碎机：转速 $10000\mathrm{r / min}$   
4.4 匀浆机。  
4.5 筛网： $0.5\mathrm{mm}\sim 1\mathrm{mm}$ 孔径。  
4.6 超声波/涡旋振荡器或摇床。  
4.7 氮吹仪。  
4.8 高速离心机：转速不低于 $12000\mathrm{r / min}$   
4.9 移液器：量程 $10\mu \mathrm{L}\sim 100\mu \mathrm{L}$ 和 $100\mu \mathrm{L}\sim 1000\mu \mathrm{L}$   
4.10 固相萃取装置。  
4.11 通用型固相萃取柱：兼具亲水基团(吡咯烷酮基团)和疏水基团(二乙烯基苯)吸附剂填料的固相萃取小柱， $200\mathrm{mg},6\mathrm{mL}$ ，或相当者。  
4.12 DONs专用型固相净化柱，或相当者。  
4.13 脱氧雪腐镰刀菌烯醇免疫亲和柱：柱容量 $\geqslant 1000\mathrm{ng}$ （柱容量和柱回收率验证方法参见A.2）。

# 4.14 水相微孔滤膜： $0.22\mu \mathrm{m}$ 。

# 5 分析步骤

# 5.1 试样制备

5.1.1 谷物及其制品：取至少 $1\mathrm{kg}$ 样品，用高速粉碎机将其粉碎，过筛，使其粒径小于 $0.5\mathrm{mm} \sim 1\mathrm{mm}$ 孔径试验筛，混合均匀后缩分至 $100\mathrm{g}$ ，储存于样品瓶中，密封保存，供检测用。  
5.1.2 酒类：取散装酒至少 $1 \mathrm{~L}$ ，对于袋装、瓶装等包装样品至少取3个包装（同一批次或号），将所有液体试样在一个容器中用均质机混匀后，缩分至 $100 \mathrm{~g}(\mathrm{mL})$ 储存于样品瓶中，密封保存，供检测用。含二氧化碳的酒类样品使用前应先置于 $4^{\circ} \mathrm{C}$ 冰箱冷藏 $30 \mathrm{~min}$ ，过滤或超声脱气后方可使用。  
5.1.3 酱油、醋、酱及酱制品：取至少 $1\mathrm{L}$ 样品，对于袋装、瓶装等包装样品至少取3个包装（同一批次或号），将所有液体样品在一个容器中用匀浆机混匀后，缩分至 $100\mathrm{g(mL)}$ 储存于样品瓶中，密封保存，供检测用。

# 5.2 试样提取

5.2.1 谷物及其制品：称取 $2\mathrm{g}$ （准确至 $0.01\mathrm{g}$ ）试样于 $50~\mathrm{mL}$ 离心管中，加入 $400\mu \mathrm{L}$ 混合同位素内标工作液振荡混合后静置 $30\mathrm{min}$ 。加入 $20.0\mathrm{mL}$ 乙腈-水溶液 $(84 + 16)$ ，置于超声波/涡旋振荡器或摇床中超声或振荡 $20\mathrm{min}$ 。 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液A于干净的容器中备用。  
5.2.2 酒类：称取 $5\mathrm{g}$ （准确至 $0.01\mathrm{g}$ 试样于 $50~\mathrm{mL}$ 离心管中，加入 $200~\mu \mathrm{L}$ 混合同位素内标工作液振荡混合后静置 $30\mathrm{min}$ ，用乙腈定容至 $10~\mathrm{mL}$ ，混匀，置于超声波/涡旋振荡器或摇床中超声或振荡 $20\mathrm{min}$ 。 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液B于干净的容器中备用。  
5.2.3 酱油、醋、酱及酱制品：称取 $2\mathrm{g}$ （准确至 $0.01\mathrm{g}$ ）试样于 $50~\mathrm{mL}$ 离心管中，加入 $400~\mu \mathrm{L}$ 混合同位素内标工作液振荡混合后静置 $30\mathrm{min}$ 。加入 $20.0\mathrm{mL}$ 乙腈-水溶液 $(84 + 16)$ ，置于超声波/涡旋振荡器或摇床中超声或振荡 $20\mathrm{min}$ 。 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液C于干净的容器中备用。

# 5.3 试样净化

注：下述试样的净化方法，可根据实际情况，选择其中一种方法即可。

# 5.3.1 通用型固相萃取柱净化

取 $5\mathrm{mL}$ 上清液A或上清液B或上清液C置于 $50~\mathrm{mL}$ 离心管中，加入 $10\mathrm{mL}$ 乙腈饱和正己烷溶液，涡旋混合 $2\mathrm{min},5000\mathrm{r / min}$ 离心 $2\mathrm{min}$ ，弃去正己烷层后，于 $40^{\circ}C\sim 50^{\circ}C$ 下氮气吹干，加入 $4\mathrm{mL}$ 水充分溶解残渣，待净化。

将固相萃取柱连接到固相萃取装置，先后用 $3\mathrm{mL}$ 甲醇和 $3\mathrm{mL}$ 水活化平衡。将 $4\mathrm{mL}$ 上述水复溶液上柱，控制流速为每秒1滴～2滴。用 $3\mathrm{mL}$ 水、 $1\mathrm{mL}5\%$ 甲醇-水溶液依次淋洗柱子后彻底抽干。用 $4\mathrm{mL}$ 甲醇洗脱，收集全部洗脱液后在 $40^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 下氮气吹干。加入 $1.0\mathrm{mL}$ 初始流动相溶解残留物，涡旋混匀 $10\mathrm{s}$ ，用 $0.22\mu \mathrm{m}$ 微孔滤膜过滤于进样瓶中，待进样。

# 5.3.2 DONs专用型固相净化柱净化

取 $8\mathrm{mL}$ 上清液A或上清液B或上清液C至DONs专用型固相净化柱的玻璃管内，将净化柱的填料管插入玻璃管中并缓慢推动填料管至净化液析出，移取 $5\mathrm{mL}$ 净化液于 $40^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 下氮气吹干。加入 $1.0\mathrm{mL}$ 初始流动相溶解残留物，涡旋混匀 $10~\mathrm{s}$ ，用 $0.22\mu \mathrm{m}$ 微孔滤膜过滤于进样瓶中，待进样。

注：使用不同厂商的 DONs 专用型固相净化柱，在上样、净化等操作方面可能略有不同，可按照说明书要求进行操作。

# 5.3.3 免疫亲和柱净化

事先将低温下保存的免疫亲和柱恢复至室温。

准确移取 $5\mathrm{mL}$ 上清液A或上清液B或上清液C，于 $40^{\circ}\mathrm{C}\sim 50^{\circ}\mathrm{C}$ 下氮气吹干，加入 $2\mathrm{mL}$ 水充分溶解残渣，待免疫亲和柱内原有液体流尽后，将上述样液移至玻璃注射器筒中。将空气压力泵与玻璃注射器相连接，调节下滴速度，控制样液以每秒1滴的流速通过免疫亲和柱，直至空气进入亲和柱中。用 $5\mathrm{mL}$ PBS缓冲盐溶液和 $5\mathrm{mL}$ 水先后淋洗免疫亲和柱，流速约为每秒1滴～2滴，直至空气进入亲和注中，弃去全部流出液，抽干小柱。

准确加入 $2\mathrm{mL}$ 甲醇洗脱亲和柱，控制每秒1滴的下滴速度，收集全部洗脱液至试管中，在 $50~^\circ \mathrm{C}$ 下用氮气缓缓地将洗脱液吹至近干，加入 $1.0\mathrm{mL}$ 初始流动相，涡旋 $30~\mathrm{s}$ 溶解残留物， $0.22\mu \mathrm{m}$ 滤膜过滤，收集滤液于进样瓶中以备进样。

注：使用不同厂商的免疫亲和柱，在样品上样、淋洗和洗脱的操作方面可能略有不同，应该按照说明书要求进行操作。

19.2.1.2 谷物制品（蛋糕、饼干、面包等）：用 $100\mathrm{mL}$ 石油醚分次溶解蒸发皿中的残渣，洗入 $250~\mathrm{mL}$ 分液漏斗中，再用 $30~\mathrm{mL}$ 甲醇-水 $(4 + 1)$ 分次洗涤蒸发皿，转入同一分液漏斗中。振荡分液漏斗 $1.5\mathrm{min}$ ，静置约 $15\mathrm{min}$ 使分层后，将下层甲醇-水提取液过柱净化，不要将两相交界处的白色絮状物放入柱内。

# 19.2.2 柱净化

19.2.2.1 小麦及其制品：在层析柱下端与小管连结处塞约 $0.1\mathrm{g}$ 脱脂棉，尽量塞紧，先装入 $0.5\mathrm{g}$ 中性氧化铝，敲平表面，再加 $0.4\mathrm{g}$ 活性炭，敲紧。将层析柱下端小管插入一橡皮塞，塞在抽滤瓶上，抽滤瓶中放一平底管接受过柱液，将抽滤瓶接上水泵或真空泵，稍稍开启泵，使活性炭压紧，将分液漏斗中的甲醇-水提取液小心地沿管壁加入柱内，控制流速为每15秒18滴 $\sim 20$ 滴(3 mL/min)，甲醇-水提取液过柱快完毕时，加入 $10\mathrm{mL}$ 甲醇-水 $(4 + 1)$ 淋洗柱，抽滤，直至柱内不再有液体流出。过柱速度控制在 $2\mathrm{mL} / \mathrm{min} \sim 3\mathrm{mL} / \mathrm{min}$ ，速度太快净化效果不好，太慢耗时太长。

19.2.2.2 玉米：同20.2.2.1，只是将活性炭的用量改为 $0.3\mathrm{g}$ 。

# 19.3 制备薄层层析用样液

将过柱后的洗脱液倒入 $75~\mathrm{mL}$ 玻璃蒸发皿中，用少量甲醇-水 $(4 + 1)$ 洗涤平底管。将蒸发皿置沸水浴上浓缩至干：

a）小麦：趁热加入 $3\mathrm{mL}$ 乙酸乙酯，加热至沸，在水浴锅上轻轻地反复转动蒸发皿数次，使充分沸腾将残渣中的DON溶出，并将乙酸乙酯挥发至干，再加入 $3\mathrm{mL}$ 乙酸乙酯同样处理一次，将溶剂挥干，最后加 $3\mathrm{mL}$ 乙酸乙酯，加热至沸，放冷至室温后转入浓缩瓶中，再用3份 $1.5\mathrm{mL}$ 乙酸乙酯洗涤蒸发皿，并入浓缩瓶中。  
b）小麦制品和玉米：趁热加入 $3\mathrm{mL}$ 乙酸乙酯，加热至沸，在水浴锅上轻轻地反复转动蒸发皿数次，使充分沸腾，将残渣中DON溶出，放冷至室温后转入浓缩瓶中。加约 $0.5\mathrm{mL}$ 甲醇-丙酮 $(1 + 2)$ 于蒸发皿中，用玻璃搅动溶解残渣，将蒸发皿置水浴锅上挥干溶剂后，加入 $3\mathrm{mL}$ 乙酸乙酯，加热至沸，转动蒸发皿，使充分沸腾，放冷至室温后转入同一浓缩瓶中，再用 $0.5\mathrm{mL}$ 甲醇-丙酮 $(1 + 2)$ 和 $3\mathrm{mL}$ 乙酸乙酯同样处理一次，乙酸乙酯提取液并入浓缩瓶中。

将浓缩瓶置约 $95\%$ 水浴锅上，用蒸汽加热吹氮气浓缩至干，放冷至室温后加入 $0.2\mathrm{mL}$ 三氯甲烷-乙腈 $(4 + 1)$ 溶解残渣留作薄层层析用。

# 19.4 测定

19.4.1 薄层板的制备： $4\mathrm{g}$ 硅胶G加约 $9\mathrm{mL}15\%$ 氯化铝 $(\mathrm{AlCl}_3\cdot 6\mathrm{H}_2\mathrm{O})$ 水溶液，研磨约 $2\mathrm{min}$ 至呈黏稠状，铺成 $5\mathrm{cm}\times 20\mathrm{cm}$ 的薄层板三块，置室温干燥后，于 $105^{\circ}\mathrm{C}$ 活化 $1\mathrm{h}$ ，贮干燥器中备用。  
19.4.2 点样：在每块薄层板上距下端 $2.5\mathrm{cm}$ 的基线上点样。

第一块板：对每一个试样先点第一块薄层板，在距板左边缘 $1.8\mathrm{cm}$ 处点 $25~\mu \mathrm{L}$ 试样液，在距板上端 $1.5\mathrm{cm}$ 处的横线上并与基线试样点相对应的位置上点 $2\mu \mathrm{L}$ DON标准液(50ng)。

第二块板：在第一块板上未显荧光的试样则需在第二块薄层板上距左边缘 $0.8\mathrm{cm}\sim 1\mathrm{cm}$ 处滴加样液点(根据情况估计滴加量，或稀释后定量)，在距板左边缘 $2\mathrm{cm}$ 处和在距右边缘 $1.2\mathrm{cm}$ 处分别滴加两个标准点，DON的量可为 $50~\mathrm{ng},75~\mathrm{ng},100~\mathrm{ng}$ 。再在距板上端 $1.5\mathrm{cm}$ 处的横线上点三个DON标准点（各 $50~\mathrm{ng}$ )，使之与基线上的三个点相对应。

# 19.4.3 展开：

横展剂：乙醚、乙醚-丙酮 $(95 + 5)$ 或无水乙醚。任选其中一种，使试样DON点偏离原点 $0.7\mathrm{cm}\sim$ $1\mathrm{cm}$ ，刚好与杂质荧光分开。

纵展剂：三氯甲烷-丙酮-异丙醇 $(8 + 1 + 1)$

三氯甲烷-丙酮-异丙醇-水 $(7.5 + 1 + 1.5 + 0.1)$

19.4.3.1 横展：在展开槽内倒入 $10 \mathrm{~mL}$ 横展剂。将点好样的薄层板靠样液点的长边斜浸入溶剂，展至板端 $1 \mathrm{~min} \sim 2 \mathrm{~min}$ ，取出通风挥干 $3 \mathrm{~min}$ 。对小麦制品还须再用 $10 \mathrm{~mL}$ 石油醚 $(30^{\circ} \mathrm{C} \sim 60^{\circ} \mathrm{C})$ 横展一次，展至板端过 $1 \mathrm{~min}$ ，取出通风挥干 $5 \mathrm{~min}$ 。  
19.4.3.2 纵展：在展开槽内倒入 $10 \mathrm{~mL}$ 纵展剂。将横展挥干后的薄层板置展开槽内纵展 $15 \mathrm{~cm}$ ，取出通风挥干 $10 \mathrm{~min}$ ，由于DON与杂质分离的效果受空气湿度影响较大，当板面分离效果不太好时即第一块薄层板的试样点附近有杂质荧光干扰时，可按极性大小依次换用以下几种展开方式：

a）三氯甲烷-丙酮-异丙醇 $(8 + 1 + 1)$ 。  
b）三氯甲烷-丙酮-异丙醇 $(8 + 1 + 1)$ 并在展开槽盖内面贴上水饱和的滤纸。  
c) 三氯甲烷-丙酮-异丙醇-水 $(7.5 + 1 + 1.5 + 0.1)$ 。  
d) 三氯甲烷-丙酮-异丙醇-水 $(7.5 + 1 + 1.5 + 0.1)$ 并在展开槽内面贴上水饱和的滤纸。改换纵展方式, 展开第二块薄层板。如展开槽内极性太大, 会使 DON 点变偏。

19.4.4 显荧光：先观察未加热的薄层板可见到显蓝紫荧光的干扰点，这时DON不显荧光，加热薄层板后，杂质点仍显荧光，它在DON荧光点附近，但不干扰DON。然后将此薄层板置 $130^{\circ}\mathrm{C}$ 烘箱中加热 $7\mathrm{min}\sim 10\mathrm{min}$ ，取出放在冷的表面上 $1\mathrm{min}\sim 5\mathrm{min}$ 后于 $365~\mathrm{nm}$ 紫外光灯下观察。

19.4.5 观察与评定：薄层板经横展后，板上样液点的DON点移动 $0.7\mathrm{cm}\sim 1.0\mathrm{cm}$ ，正是根据这一点在纵展后使样品DON点摆脱了杂质荧光的干扰。薄层板上端未经纵展的三个DON标准点可分别作为纵展后样品DON点和两个DON标准点的横向定位点。样品DON点又可与纵展后的DON标准点比较Rf值而定性。这样从横向和纵向两个方面确定样品DON的位置，达到定性的目的。在第一块薄层板上如样品DON点上有很浅的斜的荧光通过，这是过柱时没有掌握好速度，净化不够，也可能是空气湿度的变化，影响分离效果，但两个标准DON点的位置上均无杂质荧光干扰。

如在第一块薄层板上样液未显荧光点，而在第二块薄层板上样液 $25\mu \mathrm{L}$ 加标准 $25~\mathrm{ng}$ 所显荧光强度与标准 $25\mathrm{ng}$ 相等，则样品中DON含量为阴性或为 $50~\mu \mathrm{g / kg}$ 以下。阳性样品概略定量时，虽然薄层板上三个DON点在横展中都稍有移动，但对各点荧光强度无影响，两个标准点均可用于和样品DON点比较荧光强度。

19.4.6 薄层光密度计测定：激发光波长 $340 \mathrm{~nm}$ 、发射光波长 $400 \mathrm{~nm}$ 。在薄层板上标准 DON 荧光点至少在 $100 \mathrm{ng} 、 200 \mathrm{ng} 、 400 \mathrm{ng}$ 时，测得的响应与 DON 的量才呈线性关系。对 DON 含量在 $300 \mu \mathrm{g} / \mathrm{kg}$ 以上的样品才用光密度计测定。当 DON 含量在 $300 \mu \mathrm{g} / \mathrm{kg}$ 时，点样液 $20 \mu \mathrm{L}$ 使测得的 DON 的量落在

$100\mathrm{ng}\sim 200\mathrm{ng}$ 之间。用薄层扫描仪测定时，每块薄层板上滴加两个标准DON点，DON的量为 $100\mathrm{ng},200\mathrm{ng}$ 或 $200\mathrm{ng},400\mathrm{ng}$ 。在激发波长 $340~\mathrm{nm}$ ，发射波长 $400~\mathrm{nm}$ 条件下进行测定，以测得的峰面积值为纵坐标，DON量为横坐标，绘制标准曲线。

# 20 分析结果的表述

试样中脱氧雪腐镰刀菌烯醇的含量按式(3)计算：

$$
X = \frac {C \times V _ {1} \times f \times 1 0 0 0}{V _ {2} \times m \times 1 0 0 0} \tag {3}
$$

式中

$X$ ——试样中DON的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C$ —薄层板上测得样品点上DON的量，单位为纳克 $(\mathrm{ng})$

$V_{1}$ ——加入三氯甲烷-乙腈混合液溶解残渣的体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——样液的总稀释倍数；

1000 ——换算系数；

$V_{2}$ ——滴加样液的体积，单位为毫升 $(\mathrm{mL})$

$m$ 三氯甲烷-乙腈混合液溶解残渣相当样品的质量，单位为克 $(\mathrm{g})$ 。

结果表示到测定值的整数位。

DON含量超过GB2761限量值的试样需用第一法作进一步确证。

每个试样称取两份进行平行测定，以其算术平均值为分析结果。

其分析结果的相对相差应不大于 $60\%$ 。

# 23 原理

试样中的脱氧雪腐镰刀菌烯醇经水提取、均质、涡旋、离心(或过滤)等前处理获取上清液。被酶标记的脱氧雪腐镰刀菌烯醇酶连偶合物，与试样上清液或标准品中的脱氧雪腐镰刀菌烯醇竞争性结合微孔中预包被的特异性抗体。在洗涤后加入相应显色剂显色，经无机酸终止反应，于 $450\mathrm{nm}$ 或 $630\mathrm{nm}$ 波长下检测。试样中的脱氧雪腐镰刀菌烯醇与吸光度在一定浓度范围内呈反比。

# 27.2 结果计算

$$

$$

式中：

$X$ ——食品中脱氧雪腐镰刀菌烯醇的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$V$ ——提取液体积，单位为升 $(\mathrm{L})$ ；

$f$ ——在前处理过程中的稀释倍数；

$m$ ——样品取样量，单位为千克 $(\mathrm{kg})$ 。

本修改单经中华人民共和国国家卫生健康委员会和国家市场监督管理总局于2025年3月16日第2号公告批准，自2025年9月16日起实施。

# （修改事项）

# 一、1 范围

将“第一法为同位素稀释液相色谱-串联质谱法，适用于谷物及其制品、酒类、酱油、醋、酱及酱制品中脱氧雪腐镰刀菌烯醇、3-乙酰脱氧雪腐镰刀菌烯醇和15-乙酰脱氧雪腐镰刀菌烯醇的测定”改为“第一法为同位素稀释液相色谱-串联质谱法，适用于谷物及其制品（含带馅料面米制品）、以小麦粉为原料的焙烤食品（含带馅料焙烤食品）、婴幼儿谷类辅助食品、酒类、酱油、醋、酱及酱制品中脱氧雪腐镰刀菌烯醇、3-乙酰脱氧雪腐镰刀菌烯醇和15-乙酰脱氧雪腐镰刀菌烯醇的测定”。

# 二、3.4.4 标准系列工作溶液

3.4.4 标准系列工作溶液：准确移取适量混合标准工作溶液和混合同位素内标工作液，用初始流动相配制成 $10\mathrm{ng / mL},20\mathrm{ng / mL},40\mathrm{ng / mL},80\mathrm{ng / mL},160\mathrm{ng / mL},320\mathrm{ng / mL},640\mathrm{ng / mL}$ 的混合标准系列，其中同位素内标浓度为 $100\mathrm{ng / mL}$ 。标准系列溶液于 $4^{\circ}C$ 保存，有效期7d。

其中：将“用初始流动相配制成 $10\mathrm{ng / mL},20\mathrm{ng / mL},40\mathrm{ng / mL},80\mathrm{ng / mL},160\mathrm{ng / mL},320\mathrm{ng / mL},$ $640~\mathrm{ng / mL}$ 的混合标准系列"改为“用初始流动相配制成 $10\mathrm{ng / mL},20\mathrm{ng / mL},40\mathrm{ng / mL},80\mathrm{ng / mL},$ $160~\mathrm{ng / mL},320~\mathrm{ng / mL},640~\mathrm{ng / mL},1~280~\mathrm{ng / mL}$ 的混合标准系列”。

# 三、5.1 试样制备

5.1.1 谷物及其制品：取至少 $1\mathrm{kg}$ 样品，用高速粉碎机将其粉碎，过筛，使其粒径小于 $0.5\mathrm{mm} \sim 1\mathrm{mm}$ 孔径试验筛，混合均匀后缩分至 $100\mathrm{g}$ ，储存于样品瓶中，密封保存，供检测用。

其中，将“谷物及其制品”改为“谷物及其制品（含带馅料面米制品）、以小麦粉为原料的焙烤食品（含带馅料焙烤食品）、婴幼儿谷类辅助食品”。

将“取至少 $1\mathrm{kg}$ 样品，用高速粉碎机将其粉碎，过筛，使其粒径小于 $0.5\mathrm{mm} \sim 1\mathrm{mm}$ 孔径试验筛”改为“取至少 $1\mathrm{kg}$ 样品，用高速粉碎机将其粉碎，使其粒径为 $0.5\mathrm{mm} \sim 1\mathrm{mm}$ ”。

# 四、5.2 试样提取

5.2.1 谷物及其制品：称取 $2\mathrm{g}$ （准确至 $0.01\mathrm{g}$ ）试样于 $50~\mathrm{mL}$ 离心管中，加入 $400\mu \mathrm{L}$ 混合同位素内标工作液振荡混合后静置 $30\mathrm{min}$ 。加入 $20.0\mathrm{mL}$ 乙腈-水溶液 $(84 + 16)$ ，置于超声波/涡旋振荡器或摇床中超声或振荡 $20\mathrm{min}$ 。 $10000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液A于干净的容器中备用。

其中，将“谷物及其制品”改为“谷物及其制品（含带馅料面米制品）、以小麦粉为原料的焙烤食品（含带馅料焙烤食品）、婴幼儿谷类辅助食品”。

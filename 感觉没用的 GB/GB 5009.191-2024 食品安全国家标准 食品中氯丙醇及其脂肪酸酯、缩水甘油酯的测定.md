# 前言

本标准代替GB5009.191—2016《食品安全国家标准 食品中氯丙醇及其脂肪酸酯含量的测定》。  
本标准与GB5009.191—2016相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中氯丙醇及其脂肪酸酯、缩水甘油酯的测定”；  
- 氯丙醇含量的测定方法修订为第一篇 食品中氯丙醇含量的测定 同位素稀释-气相色谱-质谱法, 代替原标准第一法 食品中3-氯-1,2-丙二醇含量的测定 同位素稀释-气相色谱-质谱法和第二法 食品中氯丙醇多组分含量的测定 同位素稀释-气相色谱-质谱法;  
在氯丙醇脂肪酸酯含量测定中增加缩水甘油酯的测定，方法修订为第二篇 第一法 同位素稀释-碱水解-气相色谱-串联质谱法、第二法 同位素稀释-酸水解-气相色谱-串联质谱法和第三法 同位素稀释-碱水解-气相色谱-质谱法，代替原第三法 食品中氯丙醇脂肪酸酯含量的测定 气相色谱-质谱法；  
——修改了方法的适用范围和技术参数。

# 食品中氯丙醇及其脂肪酸酯、缩水甘油酯的测定

# 1 范围

本标准第一篇规定了食品中氯丙醇的测定。

本标准第二篇规定了食品中氯丙醇脂肪酸酯和缩水甘油酯的测定。

第一篇适用于含酸水解植物蛋白的调味品、肉及肉制品、婴幼儿配方食品中3-氯-1，2-丙二醇（3-MCPD）、2-氯-1，3-丙二醇(2-MCPD)、1，3-二氯-2-丙醇(1，3-DCP)和2，3-二氯-1-丙醇(2，3-DCP)含量的测定。

第二篇第一法和第二法适用于油脂及其制品、婴幼儿配方食品、乳及乳制品、水产动物及其制品、肉及肉制品、膨化食品、焙烤食品和油炸食品中3-氯-1，2-丙二醇酯(3-MCPDE)、2-氯-1，3-丙二醇酯(2-MCPDE)和缩水甘油酯(GE)含量的测定，第三法适用于油脂及其制品中3-MCPDE、2-MCPDE和GE含量的测定。

# 第一篇 食品中氯丙醇含量的测定 同位素稀释-气相色谱-质谱法

# 2 原理

试样中加入同位素内标，以氯化钠溶液提取，采用硅藻土小柱净化，经正己烷淋洗后，用乙酸乙酯洗脱氯丙醇，经七氟丁酰基咪唑衍生，以气相色谱-质谱仪测定，内标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

3.1.1 乙酸乙酯 $(\mathrm{C_4H_8O_2})$ ：色谱纯。  
3.1.2 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。  
3.1.3 氯化钠 $\mathrm{(NaCl)}$ 。  
3.1.4 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $120^{\circ}C$ 烘烤 $4\mathrm{h}$ 后储存于干燥器中，备用。  
3.1.5 七氟丁酰基咪唑 $\mathrm{(C_7H_3F_7N_2O}$ ,CAS号32477-35-3)：纯度 $\geqslant 97\%$ 。使用后注意密封防潮保存。

# 3.2 试剂配制

氯化钠溶液 $(200\mathrm{g / L})$ ：称取氯化钠 $200\mathrm{g}$ ，溶于水并稀释至 $1\mathrm{L}$ 。

# 3.4 标准溶液配制

# 3.4.1 氯丙醇标准溶液

3.4.1.1 氯丙醇标准储备液 $(1000\mu \mathrm{g} / \mathrm{mL})$ : 称取氯丙醇标准品各 $10.0\mathrm{mg}$ (精确至 $0.01\mathrm{mg}$ ), 分别用乙酸乙酯溶解, 转移至 $10\mathrm{mL}$ 容量瓶中, 用乙酸乙酯定容至刻度, 混匀。转移至密封性良好的棕色玻璃瓶中, 于 $-20^{\circ}\mathrm{C}$ 下保存, 保存期为 12 个月。  
3.4.1.2 氯丙醇混合标准中间液 $(10.0\mu \mathrm{g / mL})$ ：准确移取适量氯丙醇标准储备液 $(1000\mu \mathrm{g / mL})$ ，用正己烷稀释至浓度为 $10.0~\mu \mathrm{g / mL}$ ，摇匀后转移至密封性良好的棕色玻璃瓶中，于一 $20^{\circ}C$ 下保存，保存期为6个月。  
3.4.1.3 氯丙醇混合标准工作液 $(1.00\mu \mathrm{g / mL})$ ：准确移取氯丙醇混合标准中间液 $(10.0\mu \mathrm{g / mL})1.00\mathrm{mL}$ ，加入正己烷稀释至 $10~\mathrm{mL}$ ，混匀。临用现配。

# 3.4.2 氯丙醇内标溶液

3.4.2.1 内标储备液 $(1000\mu \mathrm{g} / \mathrm{mL})$ ：称取氯丙醇同位素内标各 $10.0\mathrm{mg}$ （精确至 $0.01\mathrm{mg}$ )，分别用乙酸乙酯溶解，转移至 $10~\mathrm{mL}$ 容量瓶中，用乙酸乙酯定容至刻度，混匀。转移至密封性良好的棕色玻璃瓶中，于一 $20^{\circ}C$ 下保存，保存期为12个月。  
3.4.2.2 氯丙醇混合内标工作液 $(10.0\mu \mathrm{g / mL})$ ：准确移取适量氯丙醇内标储备液（ $1000~\mu \mathrm{g / mL}$ )，用正己烷稀释至浓度为 $10.0~\mu \mathrm{g / mL}$ ，混匀后转移至密封性良好的棕色玻璃瓶中，于一 $20^{\circ}C$ 以下保存，保存期为6个月。

# 3.4.3 标准系列溶液

分别准确移取氯丙醇混合标准工作溶液 $(1.00\mu \mathrm{g / mL})10.0\mu \mathrm{L},50.0\mu \mathrm{L},100\mu \mathrm{L},200\mu \mathrm{L},400\mu \mathrm{L}$ $800~\mu \mathrm{L},1600~\mu \mathrm{L}$ 于密封性良好的具塞(盖)玻璃管中，分别加入氯丙醇混合内标工作液 $(10.0\mu \mathrm{g / mL})$ $20~\mu \mathrm{L}$ ，再分别移取相应体积的正己烷补充至 $2\mathrm{mL}$ ，混匀，配制成含氯丙醇质量分别为 $10.0\mathrm{ng},50.0\mathrm{ng}$ $100\mathrm{ng},200\mathrm{ng},400\mathrm{ng},800\mathrm{ng},1600\mathrm{ng}$ 的系列标准工作液，其中氯丙醇内标的质量均为 $200~\mathrm{ng}$ 。临用现配。

3.5.1 硅藻土小柱：填料质量为 $5\mathrm{g}$ ，或性能相当者，亦可手工填柱。  
3.5.2 气密针： $1.0\mathrm{mL}$ 或 $0.5\mathrm{mL}$ ，最小刻度不大于 $0.05\mathrm{mL}$ 。

4.1 气相色谱-质谱仪：带电子轰击离子源(EI)。  
4.2 电子天平：感量分别为 $0.01\mathrm{g}$ 和 $0.01\mathrm{mg}$ 。  
4.3 超声波振荡器。  
4.4 氮吹仪。  
4.5 恒温干燥箱。  
4.6 涡旋振荡器。  
4.7 冷冻离心机：最大转速不低于 $5000\mathrm{r / min}$

# 5 分析步骤

# 5.1 试样制备

液态样品称取前摇匀；对于半固态样品和固态样品，取可食部分均质。制备后的试样于 $4^{\circ}\mathrm{C}$ 下冷藏保存。

# 5.2 试样提取

# 5.2.1 液态试样

称取试样 $4\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 于 $15\mathrm{mL}$ 离心管中(对于液态婴幼儿配方食品等低盐试样，称样后加入 $1\mathrm{g}$ 氯化钠)，准确加入内标工作液 $(10.0\mu \mathrm{g} / \mathrm{mL})20\mu \mathrm{L}$ ，超声混匀 $5\mathrm{min}$ ，待净化。

# 5.2.2 婴幼儿配方食品（粉状）

称取试样 $4\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 离心管中，准确加入内标工作液 $(10.0\mu \mathrm{g / mL})20\mu \mathrm{L}$ ，加入 $8\mathrm{mL}200\mathrm{g / L}$ 氯化钠溶液，涡旋，超声 $10\mathrm{min}$ ，于 $4^{\circ}\mathrm{C}$ 下以 $5000\mathrm{r / min}$ 的转速离心 $10\mathrm{min}$ ，取出提取液，再加入 $8\mathrm{mL}200\mathrm{g / L}$ 氯化钠溶液重复提取操作1次，合并提取液，取 $8\mathrm{mL}$ 待净化。

# 5.2.3 其他半固态及固态试样（不包括婴幼儿配方食品）

称取试样 $4\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $15~\mathrm{mL}$ 离心管中，准确加入内标工作液 $(10.0\mu \mathrm{g / mL})20\mu \mathrm{L}$ ，加入 $4\mathrm{mL}200\mathrm{g / L}$ 氯化钠溶液，涡旋，超声 $10\mathrm{min}$ ，以 $5000\mathrm{r / min}$ 的转速离心 $10\mathrm{min}$ ，移取上清液，再加入 $4\mathrm{mL}200\mathrm{g / L}$ 氯化钠溶液重复提取1次，合并上清液，待净化。

# 5.3 试样净化

将待净化液转移至硅藻土小柱中，放置 $10\mathrm{min}$ 。用 $10~\mathrm{mL}$ 正己烷淋洗，弃去淋洗液，然后用 $15~\mathrm{mL}$ 乙酸乙酯洗脱，收集洗脱液。在洗脱液中加入 $5\mathrm{g}$ 无水硫酸钠，不断振摇脱水，放置不少于 $30\mathrm{min}$ 后过滤，转移滤液至密封性良好的具塞(盖)玻璃管中，于 $40^{\circ}\mathrm{C}$ 下氮吹浓缩至约 $0.5\mathrm{mL}$ ，加入 $2.0\mathrm{mL}$ 正己烷，涡旋，待衍生化。

# 5.4 衍生化

于标准系列溶液和试样净化液中用气密针快速加入 $0.05\mathrm{mL}$ 七氟丁酰基咪唑，立即密封，涡旋，放入烘箱中，于 $70^{\circ}\mathrm{C}$ 下保温 $20\mathrm{min}$ 。衍生结束后，取出冷却至室温，加入 $2\mathrm{mL}200\mathrm{g / L}$ 氯化钠溶液，涡旋，静置，使水相和正己烷分层，且水相澄清。取上层正己烷，加入约 $0.3\mathrm{g}$ 无水硫酸钠吸水干燥，将溶

液转移至进样瓶中，供上机测定。

# 5.5 空白试验

称取与试样相同质量的 $200\mathrm{g / L}$ 氯化钠溶液，其余步骤按 $5.2\sim 5.4$ 与试样同时处理。

# 5.6 仪器参考条件

# 5.6.1 气相色谱条件

a）色谱柱：含 $5\%$ 苯基亚芳基聚合物或 $5\%$ 苯基-甲基聚硅氧烷的弱极性毛细管柱（柱长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ ，膜厚 $0.25\mu \mathrm{m}$ ），或性能相当者。  
b）载气：氦气，纯度 $\geqslant 99.999\%$ ，流速为 $1\mathrm{mL / min}$ 。  
c）进样口温度： $250~\mathrm{^\circ C}$ □  
d）进样体积： $1\mu \mathrm{L}$ ，不分流进样。  
e）程序升温： $50~\mathrm{^\circ C}$ 保持 $1\mathrm{min}$ ，以 $2^{\circ}C / \mathrm{min}$ 升温至 $90^{\circ}C$ ，再以 $40^{\circ}C / \mathrm{min}$ 升温至 $270^{\circ}C$ ，并保持 $5\mathrm{min}$ 。

# 5.7.2 定量测定

# 5.7.2.1 标准曲线的制作

以标准系列溶液中氯丙醇和其对应内标的质量比为横坐标，以测定的氯丙醇和其对应内标的峰面积比为纵坐标，绘制标准曲线。

# 5.7.2.2 试样溶液的测定

测得试样溶液中氯丙醇和对应内标的峰面积，并由其比值通过标准曲线查得氯丙醇与对应内标的质量比，从而计算得到待测液中氯丙醇的质量。待测液中氯丙醇和对应内标的峰面积比值应在标准曲线的线性范围内；若超过标准曲线的线性范围，应适当减少取样量再重新测定。

# 6 分析结果的表述

试样中氯丙醇的含量按公式(1)计算：

$$
X = \frac {K \times N _ {i s} \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {1}
$$

式中：

X ——试样中氯丙醇(3-MCPD、2-MCPD、1,3-DCP、2,3-DCP)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$K$ ——由标准曲线查得氯丙醇与对应内标的质量比；

$N_{is}$ ——试样中加入的内标的质量，单位为纳克(ng)；

1000 ——换算系数；

$m$ ——试样称取的质量，单位为克 $(\mathrm{g})$ 。

计算结果应扣除空白值，结果保留3位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果绝对差值不得超过算术平均值的 $20\%$ 。

# 第二篇 食品中氯丙醇脂肪酸酯和缩水甘油酯的测定

# 第一法 同位素稀释-碱水解-气相色谱-串联质谱法

# 9 原理

在提取的脂肪或油脂试样中，加入同位素内标，用氢氧化钠-甲醇水解，使氯丙醇脂肪酸酯、缩水甘

油酯(GE)分别转化为氯丙醇和缩水甘油，加入酸化溴化钠终止水解，水解生成的缩水甘油转化为3-溴-1,2-丙二醇(3-MBPD)，溶液经异辛烷萃取后，水相用苯基硼酸衍生，以气相色谱-串联质谱仪检测，内标法定量。

在碱水解过程中，少部分3-氯-1,2-丙二醇脂肪酸酯(3-MCPDE)的水解产物3-氯-1,2-丙二醇(3-MCPD)可转化为缩水甘油，溴化后生成与缩水甘油酯处理后的同一产物3-MBPD。因此，在计算试样中GE含量时，需要扣除由3-MCPDE水解转化的含量。

# 10 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 10.1 试剂

10.1.1 甲醇 $\mathrm{(CH_3OH)}$   
10.1.2 异辛烷 $\left(\mathrm{C}_{8} \mathrm{H}_{18}\right)$ ：色谱纯。

# 12 分析步骤

# 12.1 试样制备

液态样品称取前应摇匀；对于常温下为固体或半固体的棕榈油或动物油脂及其制品等，取样前于 $60~\mathrm{^\circ C}$ 左右加热熔化，摇匀；取可食部分均质。制备后的试样于 $4^{\circ}C$ 冷藏保存。

# 12.2 试样提取

# 12.2.1 植物油脂、动物油脂

称取试样 $0.1\mathrm{g}$ (精确至 $1\mathrm{mg}$ ) 于 $15\mathrm{mL}$ 离心管中，准确加入内标工作液 $(2.00\mu \mathrm{g} / \mathrm{mL})100\mu \mathrm{L}$ ，混匀，供水解。

注：自加入内标开始的步骤 $(12.2\sim 12.5)$ ，也可采用经验证符合要求的自动化前处理仪器操作。

# 12.2.2 鱼油粉等油脂制品

对于鱼油粉等油脂制品，参照GB5009.6第三法（碱水解法)提取脂肪后，计算脂肪含量，再按照12.2.1步骤操作。

# 12.2.3 婴幼儿配方食品

称取粉状试样 $0.5\mathrm{g}$ (精确至 $1\mathrm{mg}$ ) 或液态试样 $3.0\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 于 $50~\mathrm{mL}$ 离心管中，准确加入内标工作液 $(2.00\mu \mathrm{g} / \mathrm{mL})100\mu \mathrm{L}$ ；加入 $2\mathrm{mL}65^{\circ}\mathrm{C}$ 的热水，涡旋混匀，再加入 $1\mathrm{mL}$ 氨水，涡旋，在 $65^{\circ}\mathrm{C}$ 水浴下振荡 $30\mathrm{min}$ 后，冷却至室温。加入 $3\mathrm{mL}$ 乙醇，涡旋后加入 $5\mathrm{mL}$ 乙醚，振摇 $5\mathrm{min}$ ，然后加入 $5\mathrm{mL}$ 石油醚，继续振摇 $5\mathrm{min}$ 。静置待溶液分层后，转移上层溶液至离心管中；于下层水相中再加入乙醚和石油醚各 $5\mathrm{mL}$ ，重复萃取1次，合并有机相于 $40^{\circ}\mathrm{C}$ 下氮吹至干，供水解。

# 12.2.4 其他食品

称取试样 $3.0\mathrm{g}\sim 5.0\mathrm{g}$ （精确至 $0.01\mathrm{g})$ 于玻璃纤维滤筒中，扎好口，然后置于 $250~\mathrm{mL}$ 三角烧瓶中，

加入 $25 \mathrm{~mL}$ 石油醚后盖塞, 以 $120 \mathrm{r} / \mathrm{min}$ 速度振荡 $2 \mathrm{~h} \sim 3 \mathrm{~h}$ , 将提取液转移, 再次加入 $25 \mathrm{~mL}$ 石油醚提取 $2 \mathrm{~h}$ , 合并两次提取液, 减压浓缩至 $1.0 \mathrm{~mL}$ 。将溶液转移至 $10 \mathrm{~mL}$ 试管中, 用 $6 \mathrm{~mL}$ 石油醚分两次淋洗鸡心瓶内壁, 将淋洗液全部转移至玻璃试管中, 于 $55^{\circ} \mathrm{C}$ 下水浴加热, 之后在 $100^{\circ} \mathrm{C}$ 干燥箱中再加热约 $1 \mathrm{~h}$ , 冷却至室温称重, 计算脂肪含量。准确称取脂肪 $0.1 \mathrm{~g}$ (精确至 $1 \mathrm{mg}$ ) 于 $15 \mathrm{~mL}$ 离心管中, 准确加入内标工作液 $(2.00 \mu \mathrm{g} / \mathrm{mL}) 100 \mu \mathrm{L}$ , 混匀, 供水解。

注：试样中脂肪含量低于 $10\%$ 时，应适当增加称样量，并增加石油醚的用量，使得到的脂肪不少于 $0.3\mathrm{g}$ 。必要时，应根据产品配方特点，采用GB5009.6中第三法(碱水解法)的步骤进行脂肪的提取，再称取脂肪按照12.2.1的步骤操作。

# 12.3 水解

于标准系列溶液、上述试样溶液中加入 $100\mu \mathrm{L}$ 甲苯和 $200\mu \mathrm{L}$ 甲基叔丁基醚，混匀后于 $10^{\circ}\mathrm{C}$ 下冷却 $6\mathrm{min}$ ，加入 $10^{\circ}\mathrm{C}$ 下预先冷却 $2\mathrm{h}$ 的 $0.35\mathrm{mol} / \mathrm{L}$ 氢氧化钠-甲醇溶液 $200\mu \mathrm{L}$ ，立即涡旋，然后于 $10^{\circ}\mathrm{C}$ 下放置 $7\mathrm{min}$ ，进行酯键断裂反应，结束后加入 $600\mu \mathrm{L}$ 酸化溴化钠溶液（ $600\mathrm{g} / \mathrm{L}$ ），涡旋，终止水解反应，溶液待净化。

# 12.4 液-液萃取净化

在上述水解后的溶液中加入 $600\mu \mathrm{L}$ 异辛烷，涡旋，进行液-液萃取，静置分层后弃去异辛烷层，必要时可离心使溶液分层彻底，再次加入 $600\mu \mathrm{L}$ 异辛烷，重复净化步骤，弃去异辛烷层，下层水相供衍生用。

注：若出现凝固现象，可加热溶解，必要时振荡，至试液充分均匀。

# 12.5 衍生化

于净化后的水溶液中加入 $100\mu \mathrm{L}$ 苯基硼酸溶液，涡旋后，加入 $100\mu \mathrm{L}1\%$ 乙二醇溶液和 $1\mathrm{mL}$ 异辛烷，涡旋，静置分层，必要时可离心使分层彻底；取上层溶液经无水硫酸钠脱水后，将溶液转移至进样瓶中，供气相色谱-串联质谱仪检测。

# 12.6 空白对照

除不加试样外，采用与试样操作完全相同的步骤进行，测定结果应低于检出限。

# 12.8.2 定量测定

# 12.8.2.1 标准曲线的制作

测得氯丙醇、3-MBPD及同位素内标的峰面积，以氯丙醇脂肪酸酯的质量（以3-MCPD、2-MCPD计)为横坐标，以氯丙醇与相应内标的峰面积比为纵坐标，绘制氯丙醇脂肪酸酯的标准曲线。同理，以GE的质量(以缩水甘油计)为横坐标，以GE水解溴化后的产物3-MBPD与相应内标的峰面积比为纵坐标，绘制GE的标准曲线。

# 12.8.2.2 试样溶液的测定

测得氯丙醇、3-MBPD和同位素内标的峰面积并计算其比值。根据氯丙醇脂肪酸酯、GE的标准曲线，分别计算得到氯丙醇脂肪酸酯、GE的质量。试液中氯丙醇、3-MBPD与内标的峰面积比值应在标准曲线的线性范围内；若质量超过标准曲线的线性范围，应将试样(提取的脂肪、油脂)用甲苯稀释或调整内标的添加量后重新测定，并使得衍生后试样溶液中内标与标准曲线中内标的质量浓度相同。

# 13 分析结果的表述

# 13.1 试样中氯丙醇脂肪酸酯含量的计算

# 13.1.1 植物油脂、动物油脂、婴幼儿配方食品和脂肪

试样或脂肪中氯丙醇脂肪酸酯的含量按公式(2)计算：

$$
C = \frac {M _ {1}}{m} \times \frac {1 0 0 0}{1 0 0 0} \tag {2}
$$

式中：

C ——试样或脂肪中3-MCPDE、2-MCPDE的含量（以3-MCPD、2-MCPD计），单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$M_{1}$ ——由氯丙醇脂肪酸酯标准曲线计算的3-MCPDE、2-MCPDE的质量，单位为纳克(ng)；

$m$ ——试样或脂肪的称取量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

计算结果应扣除空白值，保留3位有效数字。

# 13.1.2 鱼油粉等油脂制品和其他食品

鱼油粉等油脂制品和其他食品中氯丙醇脂肪酸酯的含量按公式(3)计算：

$$
X = C \times \frac {F}{m} \quad \dots \tag {3}
$$

式中：

$X$ ——试样中3-MCPDE、2-MCPDE的含量(以3-MCPD、2-MCPD计)，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C$ ——按公式(2)计算得到的脂肪中3-MCPDE、2-MCPDE的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$F$ ——试样中提取得到的脂肪质量，单位为克(g)；

$m$ ——试样的称取质量，单位为克 $(\mathrm{g})$ ；

计算结果应保留3位有效数字。

# 13.2 试样中GE含量的计算

# 13.2.1 植物油脂、动物油脂、婴幼儿配方食品和脂肪

# 13.2.1.1 GE总量的计算

GE的总量按公式(4)计算：

$$
G _ {\text {总}} = \frac {M _ {1}}{m} \times \frac {1 0 0 0}{1 0 0 0} \tag {4}
$$

式中：

$G_{\text{总}}$ ——试样或脂肪中GE的总量(以缩水甘油计)，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$M_{1}$ ——由GE标准曲线计算得到GE(以缩水甘油计)的质量，单位为纳克 $(\mathrm{ng})$ ；

$m$ ——试样或脂肪的称取量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果应扣除空白值，保留3位有效数字。

# 13.2.1.2 3-MCPDE转化而成3-MBPD峰面积的计算

在氯丙醇脂肪酸酯标准系列中，以3-MCPDE与 ${}^{13}\mathrm{C}_3 - 3\mathrm{-MCPDE}$ 的峰面积比为横坐标，3-MBPD与 ${}^{13}\mathrm{C}_3 - 3\mathrm{-MBPD}$ 的峰面积比为纵坐标，进行一元线性回归，得到线性方程。按公式(5)计算3-MCPDE转化得到3-MBPD的峰面积 $A_{1}$ 。

$$
A _ {1} = \left(\frac {A _ {2}}{A _ {\mathrm {I S 2}}} \times a + b\right) \times A _ {\mathrm {I S l}} \quad \dots \tag {5}
$$

式中：

$A_{1}$ ——试样中由3-MCPDE转化而成3-MBPD的峰面积；

$A_{2}$ ——试样中3-MCPDE的峰面积；

$A_{\mathrm{IS2}}$ ——试样中内标 ${}^{13}\mathrm{C}_{3}-3$ -MCPDE的峰面积；

$a$ ——回归方程的斜率；

$b$ ——回归方程的截距；

$A_{\mathrm{IS1}}$ ——试样中由内标 ${}^{13}\mathrm{C}_3 - 3$ -MCPDE转化而成 ${}^{13}\mathrm{C}_3 - 3$ -MBPD的峰面积。

# 13.2.1.3 试样或脂肪中GE含量的计算

根据GE的标准曲线，由试样峰面积比值计算得到GE的总量；并由公式(5)中 $A_{1}$ 和 $\mathrm{D}_5 - 3$ -MBPD的峰面积比值计算3-MCPDE转化得到3-MBPD（以缩水甘油计)的质量，即 $M _ { \text{转} }$ 。按公式(6)扣除3-MCPDE转化得到3-MBPD（以缩水甘油计)的量，得到试样中GE的含量。

$$
G _ {1} = G _ {\text {总}} - \frac {M _ {\text {转 化}}}{m} \times \frac {1 0 0 0}{1 0 0 0} \tag {6}
$$

式中：

$G_{1}$ ——试样或脂肪中GE的含量(以缩水甘油计)，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$G_{\text{总}}$ ——GE的总量(以缩水甘油计)，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$M_{\text{转化}}$ ——通过GE的标准曲线计算3-MCPDE转化得到3-MBPD(以缩水甘油计)的质量，单位为纳克 $(\mathrm{ng})$ ；

$m$ ——试样或脂肪的称取量，单位为克 $(\mathrm{g})$ ；

1000——换算系数。

计算结果应扣除空白值，保留3位有效数字。

# 13.2.2 鱼油粉等油脂制品和其他食品

鱼油粉等油脂制品和其他食品中GE的含量按公式(7)计算：

$$
G _ {2} = G _ {1} \times \frac {F}{m} \quad \dots \tag {7}
$$

式中：

$G_{2}$ ——试样中GE的含量(以缩水甘油计)，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$G_{1}$ ——按公式(6)计算得到的脂肪中GE的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$F$ ——试样中提取得到的脂肪质量，单位为克(g)；

$m$ ——试样的称取量，单位为克 $(\mathrm{g})$ ；

计算结果应保留3位有效数字。

# 第二法 同位素稀释-酸水解-气相色谱-串联质谱法

# 16 原理

在油脂试样或提取的脂肪中，加入同位素内标，经氨基固相柱净化后，缩水甘油酯(GE)在酸性条件下与溴化钠反应转化为3-溴-1,2-丙二醇脂肪酸酯(3-MBPDE)。试液经中和后，3-MBPDE及氯丙醇脂肪酸酯经硫酸-甲醇水解后，以固相萃取柱净化后，经七氟丁酰基咪唑衍生，用气相色谱-串联质谱仪检测，内标法定量。

向标准系列工作溶液和上述净化后的试样中加入 $2\mathrm{mL}$ 四氢呋喃，涡旋后，加入 $30~\mu \mathrm{L}~0.70\%$ 酸化溴化钠溶液，涡旋后于 $50^{\circ}\mathrm{C}$ 下溴代反应 $15\mathrm{min}$ 。结束后加入 $3\mathrm{mL}~0.60\%$ 碳酸氢钠溶液中和，涡旋。加入 $3\mathrm{mL}$ 正己烷进行液-液萃取，涡旋，静置待溶液分层后，取上层溶液在 $40^{\circ}\mathrm{C}$ 下氮吹至近干，加入 $1\mathrm{mL}$ 四氢呋喃复溶，备用。

# 19.4 酸水解

于上述复溶溶液中加入 $1.8\mathrm{mL}1.8\%$ 硫酸-甲醇溶液，涡旋后于 $40^{\circ}\mathrm{C}$ 下水解 $16\mathrm{h}$ ，结束后加入 $0.50\mathrm{mL}9.6\%$ 碳酸氢钠溶液中和，涡旋，将溶液在 $40^{\circ}\mathrm{C}$ 下氮吹至约 $1.5\mathrm{mL}$ ，溶液供净化(19.5)。

# 19.5 试液净化

将上述溶液转移至硅藻土小柱上，静置 $10\mathrm{min}$ ，以 $25~\mathrm{mL}$ 正己烷淋洗和 $35~\mathrm{mL}$ 二氯甲烷洗脱，用装有 $10\mathrm{g}$ 脱水剂的三角瓶收集洗脱液，溶液振摇 $1\mathrm{h} \sim 1.5\mathrm{h}$ 使脱水彻底。将溶液过滤至鸡心瓶中，在 $40^{\circ}\mathrm{C}$ 下减压浓缩至 $0.5\mathrm{mL}$ 左右，将溶液转移至密封性良好的 $5\mathrm{mL}$ 玻璃试管中，用 $1\mathrm{mL} \sim 2\mathrm{mL}$ 正己烷分两次淋洗鸡心瓶内壁，将溶液转入试管中，待衍生。

# 19.6 衍生化

于标准系列溶液和试样净化液中用气密针快速加入 $0.05\mathrm{mL}$ 七氟丁酰基咪唑，立即密封，涡旋，放入干燥箱中，于 $70^{\circ}\mathrm{C}$ 下保温 $20\mathrm{min}$ 。衍生结束后，取出冷却至室温，体积不足时补加正己烷至约 $1\mathrm{mL}$ ，加入 $2\mathrm{mL}200\mathrm{g / L}$ 氯化钠溶液，涡旋，使水相和正己烷分层，且水相澄清。取上层正己烷相，加入约 $0.3\mathrm{g}$ 无水硫酸钠吸水干燥，将溶液转移至进样瓶中，供上机测定。

# 第三法 同位素稀释-碱水解-气相色谱-质谱法

# 23 原理

取油脂试样或提取的脂肪两份于A、B管中，加入同位素内标，用氢氧化钠-甲醇水解，反应液经酸化、液-液萃取净化，经乙醚-乙酸乙酯萃取、苯基硼酸衍生后，以气相色谱-质谱仪检测，内标法定量。

A管中缩水甘油酯(GE)碱水解为缩水甘油后，其在酸性条件下与氯化钠作用生成3-氯-1，2-丙二醇(3-MCPD)，这与3-MCPDE水解后的产物相同，由A、B管中3-MCPDE的差值经换算后得到试样中GE的含量。B管用于测定氯丙醇脂肪酸酯的含量。

于上述A、B管中各加入 $600\mu \mathrm{L}$ 正己烷，涡旋，进行液-液萃取，静置分层后弃去正己烷层，再加入 $600~\mu \mathrm{L}$ 正己烷重复净化步骤。若分层不理想，可在 $2000\mathrm{r / min}$ 下离心 $3\mathrm{min}$ ，下层水溶液供衍生用。

注：若溶液出现凝固物现象，可加热消除，必要时振荡，至试液充分均匀。

# 26.5 衍生化

于净化后的水溶液中加入 $0.5\mathrm{mL}$ 乙醚-乙酸乙酯 $(6 + 4)$ ，涡旋，取上层溶液于离心管中，再加入 $0.5\mathrm{mL}$ 乙醚-乙酸乙酯 $(6 + 4)$ 重复萃取步骤，合并萃取液，加入 $30~\mu \mathrm{L}$ 苯基硼酸溶液，涡旋 $30~\mathrm{s}$ 进行衍生，加入 $30~\mu \mathrm{L}1.0\%$ 乙二醇溶液，涡旋，静置 $1\mathrm{min}$ ，加入约 $0.3\mathrm{g}$ 无水硫酸钠吸水干燥，将溶液转移至进样瓶中，供上机测定。

# 26.6 空白试验

除不加试样外，其余步骤与试样一致，同步处理。测定结果应低于检出限。

# 26.7 仪器参考条件

# 26.7.1 气相色谱参考条件

气相色谱参考条件如下：

a) 色谱柱: $5\%$ 苯基-甲基聚硅氧烷的弱极性毛细管气相色谱柱 (柱长 $30 \mathrm{~m}$ , 内径 $0.25 \mathrm{~mm}$ , 膜厚 $0.25 \mu \mathrm{m}$ ), 或性能相当者。  
b）进样体积： $1\mu \mathrm{L}$ ，不分流进样，不分流时间为 $2.0\mathrm{min}$ 。溶剂延迟时间： $5\mathrm{min}$   
c）载气：高纯氦气，纯度 $\geqslant 99.999\%$ ，流速为 $1.5~\mathrm{mL / min}$ 。  
d）柱温程序升温： $70^{\circ}\mathrm{C}$ 保持 $2\mathrm{min}$ ， $20^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $200^{\circ}\mathrm{C}$ ，再以 $40^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $300^{\circ}\mathrm{C}$ ，保持 $4\mathrm{min}$ 。  
e）进样口程序升温条件： $80^{\circ}\mathrm{C}$ 保持 $0\mathrm{min},300^{\circ}\mathrm{C / min}$ 升温至 $165^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$ ，再以 $300^{\circ}\mathrm{C / min}$ 升温至 $320^{\circ}\mathrm{C}$ ，保持 $8\mathrm{min}$ 。

注：若未配置程序升温进样口，进样口温度： $250~\mathrm{^\circ C}$ 。使用柱中反吹时，在最后一个目标物出峰 $1\mathrm{min}$ 之后开始反吹，色谱柱要求见12.7.1注。

# 26.7.2 质谱参考条件

质谱参考条件如下：

a）电离源：电子轰击离子源（EI）。  
b）电离能量： $70\mathrm{eV}$ 。  
c）离子源温度： $250~\mathrm{^\circ C}$ 。  
d）传输线温度： $300~\mathrm{^\circ C}$ 。  
e）扫描模式：选择离子监测(SIM)，监测离子见附录D中表D.1。

# 26.8 测定

在给定分析条件下，将制备好的空白溶液、标准系列溶液和试样溶液注入气相色谱-质谱仪中，进行定性分析和定量测定。

# 26.8.1 定性分析

在相同测定条件下，试液与标准工作液中待测物的保留时间偏差在 $\pm 0.5\%$ 以内，且在扣除背景后

的试样质谱图中，其中所选3个监测离子(对于2-MCPD衍生物，只有两个监测离子)均出现且信噪比 $\geq 3$ 而且其相对丰度与浓度相当的标准溶液中监测离子的相对丰度的偏差应符合表1要求，则可判断试样中存在对应的被测物。标准溶液中氯丙醇及其内标苯基硼酸衍生物的监测离子、色谱图和质谱图分别见附录D中表D.1、图D.1和图D.2。

# 26.8.2 定量测定

# 26.8.2.1 氯丙醇脂肪酸酯标准曲线

测得氯丙醇脂肪酸酯系列标准工作溶液中3-MCPD、2-MCPD和同位素内标的峰面积。以标准工作液中氯丙醇脂肪酸酯的质量为横坐标，标准品与相应内标的峰面积比为纵坐标，绘制3-MCPDE标准曲线和2-MCPDE标准曲线。

# 26.8.2.2 试样溶液的测定

测得3-MCPD、2-MCPD和同位素内标的峰面积比值，根据氯丙醇脂肪酸酯标准曲线计算得到待测溶液中氯丙醇脂肪酸酯的质量，其峰面积比值应在标准曲线的线性范围内；若质量超过标准曲线的线性范围，应将试样(脂肪或油脂)用甲苯稀释后或调整内标的添加量后重新测定 $(26.3\sim 26.5)$ ，使得衍生后试样溶液中内标与标准曲线中内标的质量浓度相同。

# 26.8.2.3 GE定量换算因子的测定

将缩水甘油酯标准系列溶液(24.4.3)经前处理后依次进样，测得其氯化物3-MCPD与 $\mathrm{D}_5 - 3$ -MCPD的峰面积比值，由3-MCPDE标准曲线(26.8.2.1)计算得到GE转化为3-MCPD的质量。以GE的质量为横坐标，3-MCPD的质量为纵坐标，进行一元线性回归，其斜率 $t$ 即为GE定量的换算因子。

# 27 分析结果的表述

# 27.1 试样中氯丙醇脂肪酸酯含量的计算

# 27.1.1 植物油脂、动物油脂和脂肪

此试样中3-MCPDE、2-MCPDE的含量由B管按照公式(10)计算得到。

$$
X = \frac {M}{m} \times \frac {1 0 0 0}{1 0 0 0} \tag {10}
$$

式中：

$X$ ——试样或脂肪中3-MCPDE、2-MCPDE的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$M$ ——由标准曲线计算得到3-MCPDE、2-MCPDE的质量，单位为纳克(ng)；

$m$ ——B管中试样或脂肪的称取量，单位为克(g)；

1000 ——换算系数。

计算结果应扣除空白值，保留3位有效数字。

# 27.1.2 鱼油粉等油脂制品

鱼油粉等油脂制品中氯丙醇脂肪酸酯的含量按公式(11)计算：

$$
X = C \times \frac {F}{m} \quad \dots \tag {11}
$$

式中：

$X$ ——试样中3-MCPDE、2-MCPDE的含量(以3-MCPD、2-MCPD计)，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$C$ ——按公式(11)计算得到的脂肪中3-MCPDE、2-MCPDE的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$F$ ——试样中提取得到的脂肪质量，单位为克(g)；

$m$ ——试样的称取量，单位为克 $(\mathrm{g})$ ；

计算结果保留3位有效数字。

# 27.2 试样中GE含量的计算

# 27.2.1 植物油脂、动物油脂和脂肪

试样或脂肪中GE的含量按照公式(12)计算得到：

$$
G _ {1} = \left(\frac {M _ {\text {总}}}{m _ {\mathrm {A}}} \times \frac {1 0 0 0}{1 0 0 0} - X _ {\mathrm {B}}\right) \times \frac {1}{t} \tag {12}
$$

式中：

$G_{1}$ ——试样或脂肪中GE的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$M_{\text{总}}$ ——由标准曲线计算得到A管试样中3-MCPDE的总量，单位为纳克(ng)；

$m_{\mathrm{A}}$ ——A管中试样或脂肪的称取量，单位为克 $(\mathrm{g})$ ；

1000——换算系数；

$X_{\mathrm{B}}$ ——B管试样或脂肪中3-MCPDE的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$t$ ——GE的换算因子（见26.8.2.3）。

计算结果应扣除空白值，保留3位有效数字。

注：若3-MCPDE与GE含量之比值 $>10$ 时，GE含量可能偏高，应改用第二篇第一法或第二法测定。

# 27.2.2 鱼油粉等油脂制品

鱼油粉等油脂制品中GE的含量按公式(13)计算：

$$
G _ {2} = G _ {1} \times \frac {F}{m} \quad \dots \tag {13}
$$

式中：

$G_{2}$ ——试样中GE的含量(以缩水甘油计)，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$G_{1}$ ——按公式(12)计算得到的脂肪中GE的含量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$

$F$ ——试样中提取得到的脂肪质量，单位为克(g)；

$m$ ——试样的称取量，单位为克 $(\mathrm{g})$ ；

计算结果保留3位有效数字。

# 28 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 29 其他

植物油脂、动物油脂的称取量为 $0.1\mathrm{g}$ 时，3-MCPDE、2-MCPDE和GE的检出限为 $60~\mu \mathrm{g / kg}$ ，定量限为 $200~\mu \mathrm{g / kg}$ 。

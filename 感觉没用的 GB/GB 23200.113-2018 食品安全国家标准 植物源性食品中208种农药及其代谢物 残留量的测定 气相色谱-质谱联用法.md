# 植物源性食品中208种农药及其代谢物残留量的测定气相色谱-质谱联用法

# 1 范围

本标准规定了植物源性食品中208种农药及其代谢物（参见附录A）残留量的气相色谱-质谱联用测定方法。

本标准适用于植物源性食品中208种农药及其代谢物残留量的测定。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的应用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样用乙腈提取，提取液经固相萃取或分散固相萃取净化，植物油试样经凝胶渗透色谱净化，气相色谱-质谱联用仪检测，内标法或外标法定量。

# 4 试剂和材料

除非另有说明，在分析中仅使用分析纯的试剂，水为GB/T6682规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN, CAS}$ 号：75-05-8）。  
4.1.2 乙酸乙酯 $\mathrm{(CH_3COOC_2H_5}$ ，CAS号：141-786）：色谱纯。  
4.1.3 甲苯 $\mathrm{(C_7H_8}$ ，CAS号：108-88-3）：色谱纯。  
4.1.4 环己烷 $\mathrm{(C_6H_{12}}$ ，CAS号：110-82-7)；色谱纯。  
4.1.5 氯化钠(NaCl，CAS号：7647-14-5)。  
4.1.6 醋酸钠 $\mathrm{(CH_3COONa, CAS}$ 号： $6131 - 90 - 4)$   
4.1.7 醋酸 $\mathrm{(CH_3COOH, CAS}$ 号：55896-93-0）。  
4.1.8 硫酸镁 $(\mathrm{MgSO}_4$ ，CAS号：7487-88-9）。  
4.1.9 柠檬酸钠 $\mathrm{(Na_3C_6H_5O_7}$ ，CAS号：6132-04-3）。  
4.1.10 柠檬酸氢二钠 $\mathrm{(C_6H_6Na_2O_7}$ ，CAS号：6132-05-4）。

# 4.2 溶液配制

4.2.1 乙腈-醋酸溶液 $(99 + 1$ ，体积比）：量取 $10\mathrm{mL}$ 醋酸加入 $990~\mathrm{mL}$ 乙腈中，混匀。  
4.2.2 乙腈-甲苯溶液 $(3 + 1$ ，体积比)：量取 $100\mathrm{mL}$ 甲苯加入 $300~\mathrm{mL}$ 乙腈中，混匀。  
4.2.3 GPC流动相：环己烷-乙酸乙酯溶液 $(1 + 1$ ，体积比)：量取 $500~\mathrm{mL}$ 环己烷加入 $500~\mathrm{mL}$ 乙酸乙酯中，混匀。

# 4.3 标准品

- 环氧七氯B内标和208种农药及其代谢物标准品，参见附录A，纯度 $\geq 95\%$ 。

# 4.4 标准溶液配制

4.4.1 标准储备溶液 $(1000\mathrm{mg / L})$ ：准确称取 $10\mathrm{mg}$ （精确至 $0.1\mathrm{mg}$ )各农药标准品，根据标准品的溶解性和测定的需要选丙酮或正己烷等溶剂溶解并定容至 $10~\mathrm{mL}$ ，避光一 $18^{\circ}C$ 保存，有效期1年。  
4.4.2 混合标准溶液（混合标准溶液A和B）：按照农药的性质和保留时间，将208种农药及其代谢物分成A、B两个组。吸取一定量的农药标准储备溶液于 $250~\mathrm{mL}$ 容量瓶中，用乙酸乙酯定容至刻度。混合标准溶液避光 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存，有效期1个月。  
4.4.3 内标溶液：准确称取 $10\mathrm{mg}$ 环氧七氯B（精确至 $0.1\mathrm{mg}$ 用乙酸乙酯溶解后转移至 $10\mathrm{mL}$ 容量瓶中，定容混匀为内标储备液。内标储备溶液用乙酸乙酯稀释至 $5\mathrm{mg / L}$ 为内标溶液。  
4.4.4 基质混合标准工作溶液：空白基质溶液氮气吹干，加入 $20~\mu \mathrm{L}$ 内标溶液，加入 $1\mathrm{mL}$ 相应质量浓度的混合标准溶液复溶，过微孔滤膜(4.5.6)。基质混合标准工作溶液应现用现配。

注：空白基质溶液取样量应与相应的试样处理取样量一致。

# 4.5 材料

4.5.1固相萃取柱：石墨化炭黑-氨基复合柱， $500\mathrm{mg} / 500\mathrm{mg}$ ，容积 $6\mathrm{mL}$   
4.5.2 乙二胺-N-丙基硅烷化硅胶（PSA）： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}_{\circ}$   
4.5.3十八烷基硅烷键合硅胶 $(C_{18})$ ： $40\mu \mathrm{m}\sim 60\mu \mathrm{m}_{\circ}$   
4.5.4 石墨化炭黑（GCB）： $40\mu \mathrm{m}\sim 120\mu \mathrm{m}$   
4.5.5 陶瓷均质子： $2\mathrm{cm}$ （长） $\times 1\mathrm{cm}$ （外径）。  
4.5.6 微孔滤膜（有机相）： $13\mathrm{mm}\times 0.22\mu \mathrm{m}_{\circ}$

# 5 仪器

5.1气相色谱-三重四极杆质谱联用仪：配有电子轰击源（EI）。  
5.2凝胶渗透色谱仪或装置：配有 $25\mathrm{mm}$ （内径） $\times 500\mathrm{mm}$ ，内装Bio-BeadsSX-3填料或相当的净化柱。  
5.3 分析天平：感量 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$   
5.4 高速匀浆机：转速不低于 $15000\mathrm{r / min}$   
5.5 离心机：转速不低于 $4200\mathrm{r / min}$   
5.6 组织捣碎机。  
5.7 旋转蒸发仪。  
5.8 氮吹仪：可控温。  
5.9 涡旋振荡器。

# 6 试样制备

# 6.1 试样制备

蔬菜和水果的取样量按照相关标准的规定执行，食用菌样品随机取样 $1\mathrm{kg}$ 。样品取样部位按照GB2763的规定执行。对于个体较小的样品，取样后全部处理；对于个体较大的基本均匀样品，可在对称轴或对称面上分割或切成小块后处理；对于细长、扁平或组分含量在各部分有差异的样品，可在不同部位切取小片或截成小段后处理；取后的样品将其切碎，充分混匀，用四分法取样或直接放入组织捣碎机中捣碎成匀浆，放入聚乙烯瓶中。

取谷类样品 $500\mathrm{g}$ ，粉碎后使其全部可通过 $425\mu \mathrm{m}$ 的标准网筛，放入聚乙烯瓶或袋中。取油料作

物、茶叶、坚果和香辛料各 $500\mathrm{g}$ ，粉碎后充分混匀，放入聚乙烯瓶或袋中。

植物油类搅拌均匀，放入聚乙烯瓶中。

# 6.2 试样储存

将试样按照测试和备用分别存放。于 $-18^{\circ} \mathrm{C}$ 条件下保存。

# 7 分析步骤

# 7.1 QuEChERS前处理

# 7.1.1 蔬菜、水果和食用菌

称取 $10\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）于 $50\mathrm{mL}$ 塑料离心管中，加入 $10\mathrm{mL}$ 乙腈、 $4\mathrm{g}$ 硫酸镁、 $1\mathrm{g}$ 氯化钠、 $1\mathrm{g}$ 柠檬酸钠、 $0.5\mathrm{g}$ 柠檬酸氢二钠及1颗陶瓷均质子，盖上离心管盖，剧烈振荡 $1\mathrm{min}$ 后 $4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。吸取 $6\mathrm{mL}$ 上清液加到内含 $900\mathrm{mg}$ 硫酸镁及 $150\mathrm{mg}$ PSA的 $15\mathrm{mL}$ 塑料离心管中；对于颜色较深的试样， $15\mathrm{mL}$ 塑料离心管中加入 $885\mathrm{mg}$ 硫酸镁、 $150\mathrm{mg}$ PSA及 $15\mathrm{mg}$ GCB，涡旋混匀 $1\mathrm{min}$ 。 $4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，准确吸取 $2\mathrm{mL}$ 上清液于 $10\mathrm{mL}$ 试管中， $40^{\circ}\mathrm{C}$ 水浴中氮气吹至近干。加入 $20\mu \mathrm{L}$ 的内标溶液，加入 $1\mathrm{mL}$ 乙酸乙酯复溶，过微孔滤膜(4.5.6)，用于测定。

# 7.1.2 谷物、油料和坚果

称取 $5\mathrm{g}$ 试样（精确至 $\mathrm{0.01g}$ ）于 $50~\mathrm{mL}$ 塑料离心管中，加 $10\mathrm{mL}$ 水涡旋混匀，静置 $30\mathrm{min}$ 。加入 $15\mathrm{mL}$ 乙腈-醋酸溶液(4.2.1)、 $6\mathrm{g}$ 无水硫酸镁、 $1.5\mathrm{g}$ 醋酸钠及1颗陶瓷均质子，盖上离心管盖，剧烈振荡 $1\mathrm{min}$ 后 $4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。吸取 $8\mathrm{mL}$ 上清液加到内含 $1200\mathrm{mg}$ 硫酸镁、 $400\mathrm{mg}$ PSA及 $400\mathrm{mgC_{18}}$ 的 $15\mathrm{mL}$ 塑料离心管中，涡旋混匀 $1\mathrm{min}$ 。 $4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，准确吸取 $2\mathrm{mL}$ 上清液于 $10\mathrm{mL}$ 试管中， $40^{\circ}C$ 水浴中氮气吹至近干。加入 $20~\mu \mathrm{L}$ 的内标溶液，加入 $1\mathrm{mL}$ 乙酸乙酯复溶，过微孔滤膜(4.5.6)，用于测定。

# 7.1.3 茶叶和香辛料

称取 $2\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）于 $50\mathrm{mL}$ 塑料离心管中，加 $10\mathrm{mL}$ 水涡旋混匀，静置 $30\mathrm{min}$ 。加入 $15\mathrm{mL}$ 乙腈-醋酸溶液（4.2.1）、 $6\mathrm{g}$ 无水硫酸镁、 $1.5\mathrm{g}$ 醋酸钠及1颗陶瓷均质子，盖上离心管盖，剧烈振荡 $1\mathrm{min}$ 后 $4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。吸取 $8\mathrm{mL}$ 上清液加到内含 $1200\mathrm{mg}$ 硫酸镁、 $400\mathrm{mg}$ PSA、 $400\mathrm{mg}$ $\mathrm{C_{18}}$ 及 $200\mathrm{mg}$ GCB的 $15\mathrm{mL}$ 塑料离心管中，涡旋混匀 $1\mathrm{min}$ 。 $4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，准确吸取 $2\mathrm{mL}$ 上清液于 $10\mathrm{mL}$ 试管中， $40^{\circ}\mathrm{C}$ 水浴中氮气吹至近干。加入 $20\mu \mathrm{L}$ 的内标溶液，加入 $1\mathrm{mL}$ 乙酸乙酯复溶，过微孔滤膜（4.5.6），用于测定。

注：上述处理中净化前的上清液吸取量可根据需要调整,净化材料(无水硫酸镁、PSA、 ${\mathrm{C}}_{18}$ 、GCB)用量按比例增减。

# 7.2 固相萃取前处理

# 7.2.1 提取

# 7.2.1.1 蔬菜、水果和食用菌

称取 $20\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）于 $100\mathrm{mL}$ 塑料离心管中，加入 $40\mathrm{mL}$ 乙腈，用高速匀浆机15000 $\mathrm{r / min}$ 匀浆 $2\mathrm{min}$ ，加入 $5\mathrm{g} \sim 7\mathrm{g}$ 氯化钠剧烈振荡数次， $4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。准确吸取 $10\mathrm{mL}$ 上清液于 $100\mathrm{mL}$ 茄型瓶中。 $40^{\circ}\mathrm{C}$ 水浴旋转蒸发至 $1\mathrm{mL}$ 左右，氮气吹至近干，待净化。

# 7.2.1.2 谷物、油料、坚果、茶叶和香辛料

称取 $5\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）于 $100\mathrm{mL}$ 塑料离心管中，加 $10\mathrm{mL}$ 水涡旋混匀，静置 $30\mathrm{min}$ 。加入 $20\mathrm{mL}$ 乙腈，用高速匀浆机 $15000\mathrm{r / min}$ 匀浆 $2\mathrm{min}$ ，加入 $5\mathrm{g} \sim 7\mathrm{g}$ 氯化钠剧烈振荡数次， $4200\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。准确吸取 $5\mathrm{mL}$ 上清液于 $100\mathrm{mL}$ 茄型瓶中， $40^{\circ}\mathrm{C}$ 水浴旋转蒸发至 $1\mathrm{mL}$ 左右，氮气吹至近干，待净化。

# 7.2.2 净化

1. 用 $5\mathrm{mL}$ 乙腈-甲苯溶液(4.2.2)预洗固相萃取柱(4.5.1)，弃去流出液。下接 $150~\mathrm{mL}$ 鸡心瓶，放入

固定架上。将上述待净化试样用 $3\mathrm{mL}$ 乙腈-甲苯溶液(4.2.2)洗涤至固相萃取柱中，再用 $2\mathrm{mL}$ 乙腈-甲苯溶液(4.2.2)洗涤，并将洗涤液移入柱中，重复2次。在柱上加上 $50~\mathrm{mL}$ 储液器，用 $25~\mathrm{mL}$ 乙腈-甲苯溶液淋洗小柱，收集上述所有流出液于 $150~\mathrm{mL}$ 鸡心瓶中， $40^{\circ}\mathrm{C}$ 水浴中旋转浓缩至近干。加入 $50~\mu \mathrm{L}$ 内标溶液，加入 $2.5\mathrm{mL}$ 乙酸乙酯复溶，过微孔滤膜(4.5.6)，用于测定。

# 7.3 GPC前处理

称取 $1\mathrm{g}$ 食用油试样（精确至 $0.01\mathrm{g}$ ）于 $10\mathrm{mL}$ 样品瓶中，加入GPC流动相(4.2.3)7mL混匀，将试样溶液置于GPC仪上净化，上样体积为 $5\mathrm{mL}$ ，流速为 $5\mathrm{mL} / \mathrm{min}$ ，收集 $1000\mathrm{s}\sim 2700\mathrm{s}$ 时间段的洗脱液。将流出液浓缩至 $5\mathrm{mL}$ ，准确吸取 $4\mathrm{mL}$ 于 $10\mathrm{mL}$ 玻璃离心管中， $40^{\circ}\mathrm{C}$ 水浴中氮气吹至近干。加入 $20~\mu \mathrm{L}$ 的内标溶液，加入 $1\mathrm{mL}$ 乙酸乙酯复溶，过微孔滤膜(4.5.6)，用于测定。

# 7.4 测定

# 7.4.1 仪器参考条件

a）色谱柱： $14\%$ 腈丙基苯基 $-86\%$ 二甲基聚硅氧烷石英毛细管柱； $30\mathrm{m}\times 0.25\mathrm{mm}\times 0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度： $40^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，然后以 $40^{\circ}\mathrm{C} / \mathrm{min}$ 程序升温至 $120^{\circ}\mathrm{C}$ ，再以 $5^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $240^{\circ}\mathrm{C}$ ，再以 $12^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $300^{\circ}\mathrm{C}$ ，保持 $6\mathrm{min}$ ；  
c) 载气：氦气，纯度 $\geq 99.999\%$ ，流速 $1.0~\mathrm{mL / min}$   
d）进样口温度： $280^{\circ}C$   
e）进样量： $1\mu \mathrm{L}$   
f）进样方式：不分流进样；  
g） 电子轰击源： $70\mathrm{eV}$   
h）离子源温度： $280^{\circ}C$   
i）传输线温度： $280^{\circ}\mathrm{C}$   
j）溶剂延迟： $3\mathrm{min}$   
k）多反应监测：每种农药分别选择一对定量离子、一对定性离子。每组所有需要检测离子对按照出峰顺序，分时段分别检测。每种农药的保留时间、定量离子对、定性离子对和碰撞电压，参见附录B。

# 7.4.2 标准工作曲线

精确吸取一定量的混合标准溶液，逐级用乙酸乙酯释成质量浓度为 $0.005\mathrm{mg / L},0.01\mathrm{mg / L},0.05$ $\mathrm{mg / L,0.1mg / L}$ 和 $0.5\mathrm{mg / L}$ 的标准工作溶液。空白基质溶液氮气吹干，加入 $20~\mu \mathrm{L}$ 内标溶液，分别加入 $1\mathrm{mL}$ 上述标准工作溶液复溶，过微孔滤膜(4.5.6)配制成系列基质混合标准工作溶液，供气相色谱-质谱联用仪测定。以农药定量离子峰面积和内标物定量离子峰面积的比值为纵坐标、农药标准溶液质量浓度和内标物质量浓度的比值为横坐标，绘制标准曲线。

# 7.4.3 定性及定量

# 7.4.3.1 保留时间

被测试样中目标农药色谱峰的保留时间与相应标准色谱峰的保留时间相比较，相对误差应在 $\pm 2.5\%$ 之内。

# 7.4.3.2 定量离子、定性离子及子离子丰度比

在相同实验条件下进行样品测定时，如果检出的色谱峰的保留时间与标准样品相一致，并且在扣除背景后的样品质谱图中，目标化合物的质谱定量和定性离子均出现，而且同一检测批次，对同一化合物，样品中目标化合物的定性离子和定量离子的相对丰度比与质量浓度相当的基质标准溶液相比，其允许偏差不超过表1规定的范围，则可判断样品中存在目标农药。

# 7.5 试样溶液的测定

将基质混合标准工作溶液和试样溶液依次注入气相色谱-质谱联用仪中，保留时间和定性离子定性，测得定量离子峰面积，待测样液中农药的响应值应在仪器检测的定量测定线性范围之内，超过线性范围时应根据测定浓度进行适当倍数稀释后再进行分析。

# 8 结果计算

试样中各农药残留量以质量分数 $\omega$ 计，单位为毫克每千克 $(\mathrm{mg / kg})$ 表示，内标法按式(1)计算，外标法按式(2)计算。

$$
\omega = \frac {\rho \times A \times \rho_ {i} \times A _ {s} \times V}{A _ {s} \times \rho_ {s} \times A _ {i} \times m}. \tag {1}
$$

$$
\omega = \frac {\rho \times A \times V}{A _ {s} \times m}. \tag {2}
$$

式中：

$\omega$ ——试样中被测物残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ 基质标准工作溶液中被测物的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$

$A$ ——试样溶液中被测物的色谱峰面积；

$A_{s}$ ——基质标准工作溶液中被测物的色谱峰面积；

$\rho_{i}$ ——试样溶液中内标物的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$

$\rho_{si}$ ——基质标准工作溶液中内标物的质量浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$A_{si}$ ——基质标准工作溶液中内标物的色谱峰面积；

$A_{i}$ ——试样溶液中内标物的色谱峰面积；

$V$ ——试样溶液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样溶液所代表试样的质量，单位为克(g)。

计算结果应扣除空白值，计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，保留2位有效数字。含量超 $1\mathrm{mg / kg}$ 时，保留3位有效数字。

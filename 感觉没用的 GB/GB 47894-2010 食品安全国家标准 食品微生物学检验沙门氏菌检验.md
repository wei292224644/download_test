# 前言

本标准代替GB/T4789.4-2008《食品卫生微生物学检验沙门氏菌检验》。

本标准与GB/T4789.4-2008相比，主要变化如下：

——修改了标准的中英文名称；  
——修改了标准的范围；  
——修改了培养基和试剂；  
——修改了设备和材料；  
——修改了附录A。

本标准的附录A、附录B为规范性附录。

本标准所代替的历次版本发布情况为：

GB4789.4-84、GB4789.4-1994、GB/T4789.4-2003、GB/T4789.4-2008。

# 1 范围

本标准规定了食品中沙门氏菌（Salmonella）的检验方法。

本标准适用于食品中沙门氏菌的检验。

# 5 操作步骤

# 5.1 前增菌

称取 $25\mathrm{g}$ （mL）样品放入盛有 $225~\mathrm{mL}$ BPW的无菌均质杯中，以 $8000\mathrm{r / min}\sim 10000\mathrm{r / min}$ 均质 $1\mathrm{min}\sim 2\mathrm{min}$ ，或置于盛有 $225~\mathrm{mL}$ BPW的无菌均质袋中，用拍击式均质器拍打 $1\mathrm{min}\sim 2\mathrm{min}$ 。若样品为液态，不需要均质，振荡混匀。如需测定 $\mathsf{pH}$ 值，用 $1\mathrm{mol} / \mathrm{mL}$ 无菌NaOH或HCl调 $\mathsf{pH}$ 至 $6.8\pm 0.2$

无菌操作将样品转至 $500 \mathrm{~mL}$ 锥形瓶中，如使用均质袋，可直接进行培养，于 $36^{\circ} \mathrm{C} \pm 1^{\circ} \mathrm{C}$ 培养 $8 \mathrm{~h} \sim$ $18 \mathrm{~h}$ 。

如为冷冻产品,应在 $45^{\circ} \mathrm{C}$ 以下不超过 $15 \mathrm{~min}$ ,或 $2^{\circ} \mathrm{C} \sim 5^{\circ} \mathrm{C}$ 不超过 $18 \mathrm{~h}$ 解冻。

# 5.2 增菌

轻轻摇动培养过的样品混合物，移取 $1\mathrm{mL}$ ，转种于 $10\mathrm{mL}$ TTB内，于 $42^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 培养 $18\mathrm{h}\sim 24$ h。同时，另取 $1\mathrm{mL}$ ，转种于 $10\mathrm{mL}$ SC内，于 $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 培养 $18\mathrm{h}\sim 24\mathrm{h}$ 。

# 5.3 分离

分别用接种环取增菌液1环，划线接种于一个BS琼脂平板和一个XLD琼脂平板（或HE琼脂平板或沙门氏菌属显色培养基平板)。于 $36^{\circ}\mathrm{C}\pm 1^{\circ}\mathrm{C}$ 分别培养 $18\mathrm{h}\sim 24\mathrm{h}$ （XLD琼脂平板、HE琼脂平板、沙门氏菌属显色培养基平板）或 $40\mathrm{h}\sim 48\mathrm{h}$ （BS琼脂平板)，观察各个平板上生长的菌落，各个平板上的菌落特征见表1。

# 5.5 血清学鉴定

# 5.5.1 抗原的准备

一般采用 $1.2\% \sim 1.5\%$ 琼脂培养物作为玻片凝集试验用的抗原。

O血清不凝集时，将菌株接种在琼脂量较高的（如 $2\% \sim 3\%$ ）培养基上再检查；如果是由于Vi抗原的存在而阻止了O凝集反应时，可挑取菌苔于 $1\mathrm{mL}$ 生理盐水中做成浓菌液，于酒精灯火焰

上煮沸后再检查。H抗原发育不良时，将菌株接种在 $0.55\% \sim 0.65\%$ 半固体琼脂平板的中央，俟菌落蔓延生长时，在其边缘部分取菌检查；或将菌株通过装有 $0.3\% \sim 0.4\%$ 半固体琼脂的小玻管1次～2次，自远端取菌培养后再检查。

# 5.5.2 多价菌体抗原（0）鉴定

在玻片上划出2个约 $1\mathrm{cm}\times 2\mathrm{cm}$ 的区域，挑取1环待测菌，各放1/2环于玻片上的每一区域上部，在其中一个区域下部加1滴多价菌体（O）抗血清，在另一区域下部加入1滴生理盐水，作为对照。再用无菌的接种环或针分别将两个区域内的菌落研成乳状液。将玻片倾斜摇动混合 $1\mathrm{min}$ ，并对着黑暗背景进行观察，任何程度的凝集现象皆为阳性反应。

# 5.5.3 多价鞭毛抗原（H）鉴定

同5.5.2。

检出第1相H抗原而未检出第2相H抗原的或检出第2相H抗原而未检出第1相H抗原的,可在琼脂斜面上移种 $1\sim 2$ 代后再检查。如仍只检出一个相的H抗原，要用位相变异的方法检查其另一个相。单相菌不必做位相变异检查。

位相变异试验方法如下：

小玻管法: 将半固体管 (每管约 $1 \mathrm{~mL} \sim 2 \mathrm{~mL}$ ) 在酒精灯上溶化并冷至 $50^{\circ} \mathrm{C}$ , 取已知相的 $\mathrm{H}$ 因子血清 $0.05 \mathrm{~mL} \sim 0.1 \mathrm{~mL}$ , 加入于溶化的半固体内, 混匀后, 用毛细吸管吸取分装于供位相变异试验的小玻管内, 俟凝固后, 用接种针挑取待检菌, 接种于一端。将小玻管平放在平皿内, 并在其旁放一团湿棉花, 以防琼脂中水分蒸发而干缩, 每天检查结果, 待另一相细菌解离后, 可以从另一端挑取细菌进行检查。培养基内血清的浓度应有适当的比例, 过高时细菌不能生长, 过低时同一相细菌的动力不能抑制。一般按原血清 $1:200 \sim 1:800$ 的量加入。

小倒管法：将两端开口的小玻管（下端开口要留一个缺口，不要平齐）放在半固体管内，小玻管的上端应高出于培养基的表面，灭菌后备用。临用时在酒精灯上加热溶化，冷至 $50^{\circ}\mathrm{C}$ ，挑取因子血清1环，加入小套管中的半固体内，略加搅动，使其混匀，俟凝固后，将待检菌株接种于小套管中的半固体表层内，每天检查结果，待另一相细菌解离后，可从套管外的半固体表面取菌检查，或转种 $1\%$ 软琼脂斜面，于 $37^{\circ}\mathrm{C}$ 培养后再做凝集试验。

简易平板法：将 $0.35\% \sim 0.4\%$ 半固体琼脂平板烘干表面水分，挑取因子血清1环，滴在半固体平板表面，放置片刻，待血清吸收到琼脂内，在血清部位的中央点种待检菌株，培养后，在形成蔓延生长的菌苔边缘取菌检查。

# 5.5.4.3 Vi抗原的鉴定

用Vi因子血清检查。已知具有Vi抗原的菌型有：伤寒沙门氏菌，丙型副伤寒沙门氏菌,都柏林沙门氏菌。

# 5.5.4.4 菌型的判定

根据血清学分型鉴定的结果，按照附录B或有关沙门氏菌属抗原表判定菌型。

# A.11.2 制法

除赖氨酸以外的成分加热溶解后, 分装每瓶 $100 \mathrm{~mL}$ , 分别加入赖氨酸。L-赖氨酸按 $0.5 \%$ 加入, DL-赖氨酸按 $1 \%$ 加入。调节 $\mathrm{pH}$ 。对照培养基不加赖氨酸。分装于无菌的小试管内, 每管 $0.5 \mathrm{~mL}$ , 上面滴加一层液体石蜡, $115^{\circ} \mathrm{C}$ 高压灭菌 $10 \mathrm{~min}$ 。

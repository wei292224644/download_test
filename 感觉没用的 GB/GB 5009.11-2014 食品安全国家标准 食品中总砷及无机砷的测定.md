# 前言

本标准代替GB/T5009.11—2003《食品中总碑及无机碑的测定》。

本标准与GB/T5009.11—2003相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中总砷及无机砷的测定”。  
——取消了食品中总碑测定的砷斑法及硼氢化物还原比色法，取消了食品中无机碑测定的原子荧光法和银盐法。  
——增加了食品中总碑测定的电感耦合等离子体质谱法(ICP-MS)；  
——增加了食品中无机砷测定的液相色谱-原子荧光光谱法(LC-AFS)和液相色谱-电感耦合等离子体质谱法(LC-ICP-MS)。

# 食品安全国家标准

# 食品中总碑及无机碑的测定

# 1 范围

本标准第一篇规定了食品中总碑的测定方法。本标准第二篇规定了食品中无机砷含量测定的液相色谱-原子荧光光谱法、液相色谱-电感耦合等离子体质谱法。

本标准第一篇第一法、第二法和第三法适用于各类食品中总碑的测定。第二篇适用于稻米、水产动物、婴幼儿谷类辅助食品、婴幼儿罐装辅助食品中无机砷(包括砷酸盐和亚砷酸盐)含量的测定。

# 第一篇 总碑的测定

# 第一法 电感耦合等离子体质谱法

# 2 原理

样品经酸消解处理为样品溶液，样品溶液经雾化由载气送入ICP炬管中，经过蒸发、解离、原子化和离子化等过程，转化为带电荷的离子，经离子采集系统进入质谱仪，质谱仪根据质荷比进行分离。对于一定的质荷比，质谱的信号强度与进入质谱仪的离子数成正比，即样品浓度与质谱信号强度成正比。通过测量质谱的信号强度对试样溶液中的砷元素进行测定。

# 5 分析步骤

# 5.1 试样预处理

5.1.1 在采样和制备过程中，应注意不使试样污染。  
5.1.2 粮食、豆类等样品去杂物后粉碎均匀，装入洁净聚乙烯瓶中，密封保存备用。  
5.1.3 蔬菜、水果、鱼类、肉类及蛋类等新鲜样品，洗净晾干，取可食部分匀浆，装入洁净聚乙烯瓶中，密封，于 $4^{\circ}\mathrm{C}$ 冰箱冷藏备用。

# 5.2 试样消解

# 5.2.1 微波消解法

蔬菜、水果等含水分高的样品，称取 $2.0\mathrm{g}\sim 4.0\mathrm{g}$ （精确至 $0.001\mathrm{g}$ )样品于消解罐中，加入 $5\mathrm{mL}$ 硝酸，放置 $30\mathrm{min}$ ；粮食、肉类、鱼类等样品，称取 $0.2\mathrm{g}\sim 0.5\mathrm{g}$ (精确至 $0.001\mathrm{g}$ )样品于消解罐中，加入 $5\mathrm{mL}$ 硝酸，放置 $30\mathrm{min}$ ，盖好安全阀，将消解罐放入微波消解系统中，根据不同类型的样品，设置适宜的微波消解程序(见表A.1～表A.3)，按相关步骤进行消解，消解完全后赶酸，将消化液转移至 $25~\mathrm{mL}$ 容量瓶或比色管中，用少量水洗涤内罐3次，合并洗涤液并定容至刻度，混匀。同时作空白试验。

# 5.2.2 高压密闭消解法

称取固体试样 $0.20\mathrm{g}\sim 1.0\mathrm{g}$ （精确至 $0.001\mathrm{g})$ ，湿样 $1.0\mathrm{g}\sim 5.0\mathrm{g}$ （精确至 $0.001\mathrm{g})$ 或取液体试样 $2.00~\mathrm{mL}\sim 5.00~\mathrm{mL}$ 于消解内罐中，加入 $5\mathrm{mL}$ 硝酸浸泡过夜。盖好内盖，旋紧不锈钢外套，放入恒温干燥箱， $140^{\circ}C\sim 160^{\circ}C$ 保持 $3\mathrm{h}\sim 4\mathrm{h}$ ，自然冷却至室温，然后缓慢旋松不锈钢外套，将消解内罐取出，用少量水冲洗内盖，放在控温电热板上于 $120^{\circ}C$ 赶去棕色气体。取出消解内罐，将消化液转移至 $25~\mathrm{mL}$ 容量瓶或比色管中，用少量水洗涤内罐3次，合并洗涤液并定容至刻度，混匀。同时作空白试验。

# 5.3 仪器参考条件

RF功率1550W；载气流速 $1.14\mathrm{L / min}$ ；采样深度 $7\mathrm{mm}$ ；雾化室温度 $2^{\circ}C$ ；Ni采样锥，Ni截取锥。

质谱干扰主要来源于同量异位素、多原子、双电荷离子等，可采用最优化仪器条件、干扰校正方程校正或采用碰撞池、动态反应池技术方法消除干扰。砷的干扰校正方程为： ${}^{75}\mathrm{As} = {}^{75}\mathrm{As} - {}^{77}\mathrm{M}(3.127) + {}^{82}\mathrm{M}(2.733) - {}^{83}\mathrm{M}(2.757)$ ；采用内标校正、稀释样品等方法校正非质谱干扰。砷的 $m / z$ 为75，选 ${}^{72}\mathrm{Ge}$ 为内标元素。

推荐使用碰撞/反应池技术，在没有碰撞/反应池技术的情况下使用干扰方程消除干扰的影响。

# 5.4 标准曲线的制作

吸取适量砷标准使用液 $(1.00\mathrm{mg / L})$ ，用硝酸溶液 $(2 + 98)$ 配制砷浓度分别为 $0.00~\mathrm{ng / mL}$ 、 $1.0~\mathrm{ng / mL},5.0~\mathrm{ng / mL},10~\mathrm{ng / mL},50~\mathrm{ng / mL}$ 和 $100~\mathrm{ng / mL}$ 的标准系列溶液。

当仪器真空度达到要求时，用调谐液调整仪器灵敏度、氧化物、双电荷、分辨率等各项指标，当仪器各项指标达到测定要求，编辑测定方法、选择相关消除干扰方法，引入内标，观测内标灵敏度、脉冲与模拟模式的线性拟合，符合要求后，将标准系列引入仪器。进行相关数据处理，绘制标准曲线、计算回归方程。

# 5.5 试样溶液的测定

相同条件下，将试剂空白、样品溶液分别引入仪器进行测定。根据回归方程计算出样品中砷元素的浓度。

# 6 分析结果的表述

试样中砷含量按式(1)计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——试样中砷的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 或毫克每升 $(\mathrm{mg / L})$ ；

$c$ ——试样消化液中砷的测定浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c_{0}$ ——试样空白消化液中砷的测定浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样消化液总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克或毫升(g或 $\mathrm{mL}$ ）；

1000 ——换算系数。

计算结果保留两位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 8 其他

称样量为 $1\mathrm{g}$ ，定容体积为 $25~\mathrm{mL}$ 时，方法检出限为 $0.003\mathrm{mg / kg}$ ，方法定量限为 $0.010\mathrm{mg / kg}$ 。

# 第二法 氢化物发生原子荧光光谱法

# 9 原理

食品试样经湿法消解或干灰化法处理后，加入硫脲使五价砷预还原为三价砷，再加入硼氢化钠或硼氢化钾使还原生成砷化氢，由氙气载入石英原子化器中分解为原子态砷，在高强度砷空心阴极灯的发射光激发下产生原子荧光，其荧光强度在固定条件下与被测液中的砷浓度成正比，与标准系列比较定量。

# 12.2 试样消解

# 12.2.1 湿法消解

固体试样称取 $1.0\mathrm{g}\sim 2.5\mathrm{g}$ 、液体试样称取 $5.0\mathrm{g}\sim 10.0\mathrm{g}$ （或 $\mathrm{mL}$ )（精确至 $0.001\mathrm{g}$ ，置于 $50~\mathrm{mL}\sim$ $100~\mathrm{mL}$ 锥形瓶中，同时做两份试剂空白。加硝酸 $20~\mathrm{mL}$ ，高氯酸 $4\mathrm{mL}$ ，硫酸 $1.25~\mathrm{mL}$ ，放置过夜。次日置于电热板上加热消解。若消解液处理至 $1\mathrm{mL}$ 左右时仍有未分解物质或色泽变深，取下放冷，补加硝酸 $5\mathrm{mL}\sim 10\mathrm{mL}$ ，再消解至 $2\mathrm{mL}$ 左右，如此反复两三次，注意避免炭化。继续加热至消解完全后，再持续蒸发至高氯酸的白烟散尽，硫酸的白烟开始冒出。冷却，加水 $25~\mathrm{mL}$ ，再蒸发至冒硫酸白烟。冷却，用水将内溶物转入 $25~\mathrm{mL}$ 容量瓶或比色管中，加入硫脲十抗坏血酸溶液 $2\mathrm{mL}$ ，补加水至刻度，混匀，放置 $30\mathrm{min}$ ，待测。按同一操作方法作空白试验。

# 12.2.2 干灰化法

固体试样称取 $1.0\mathrm{g}\sim 2.5\mathrm{g}$ ，液体试样取 $4.00~\mathrm{mL(g)}$ （精确至 $0.001\mathrm{g})$ ，置于 $50~\mathrm{mL}\sim 100~\mathrm{mL}$ 坩埚中，同时做两份试剂空白。加 $150~\mathrm{g / L}$ 硝酸镁 $10~\mathrm{mL}$ 混匀，低热蒸干，将 $1\mathrm{g}$ 氧化镁覆盖在干渣上，于电炉上炭化至无黑烟，移入 $550^{\circ}C$ 马弗炉灰化 $4\mathrm{h}$ 。取出放冷，小心加入盐酸溶液 $(1 + 1)10~\mathrm{mL}$ 以中和氧化镁并溶解灰分，转入 $25~\mathrm{mL}$ 容量瓶或比色管，向容量瓶或比色管中加入硫脲十抗坏血酸溶液 $2\mathrm{mL}$ 另用硫酸溶液 $(1 + 9)$ 分次洗涤坩埚后合并洗涤液至 $25~\mathrm{mL}$ 刻度，混匀，放置 $30\mathrm{min}$ ，待测。按同一操作方法作空白试验。

# 21 分析结果的表述

试样中的砷含量按式(3)进行计算：

$$
X = \frac {\left(A _ {1} - A _ {2}\right) \times V _ {1} \times 1 0 0 0}{m \times V _ {2} \times 1 0 0 0 \times 1 0 0 0} \tag {3}
$$

式中：

$X$ ——试样中砷的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 或毫克每升 $(\mathrm{mg / L})$ ；

$A_{1}$ ——测定用试样消化液中砷的质量，单位为纳克 $(\mathrm{ng})$ ；

$A_{2}$ ——试剂空白液中砷的质量，单位为纳克 $(\mathrm{ng})$ ；

$V_{1}$ ——试样消化液的总体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量(体积)，单位为克(g)或毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——测定用试样消化液的体积，单位为毫升 $(\mathrm{mL})$ 。

计算结果保留两位有效数字。

# 24 原理

食品中无机砷经稀硝酸提取后，以液相色谱进行分离，分离后的目标化合物在酸性环境下与 $\mathrm{KBH_4}$ 反应，生成气态砷化合物，以原子荧光光谱仪进行测定。按保留时间定性，外标法定量。

# 28 分析结果的表述

试样中无机砷的含量按式(4)计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {4}
$$

式中：

$X$ ——样品中无机砷的含量(以As计)，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白溶液中无机砷化合物浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c$ ——测定溶液中无机砷化合物浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样消化液体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

总无机砷含量等于 $\mathrm{As(III)}$ 含量与 $\mathrm{As(V)}$ 含量的加和。

计算结果保留两位有效数字。

# 第二法 液相色谱-电感耦合等离子质谱法（LC-ICP/MS）

# 31 原理

食品中无机砷经稀硝酸提取后，以液相色谱进行分离，分离后的目标化合物经过雾化由载气送入ICP炬焰中，经过蒸发、解离、原子化、电离等过程，大部分转化为带正电荷的正离子，经离子采集系统进入质谱仪，质谱仪根据质荷比进行分离测定。以保留时间定性和质荷比定性，外标法定量。

试样中无机砷的含量按式(5)计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0 \times 1 0 0 0} \tag {5}
$$

式中：

$X$ ——样品中无机砷的含量(以As计)，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$c_{0}$ ——空白溶液中无机砷化合物浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$c$ ——测定溶液中无机砷化合物浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$ ；

$V$ ——试样消化液体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

总无机砷含量等于As（Ⅲ）含量与 $\mathrm{As(V)}$ 含量的加和。

计算结果保留两位有效数字。

# 36 精密度

在重复性条件获得的两次独立测定结果的绝对差值不得超过算术平均值的 $20\%$ 。

# 37 其他

本方法检出限：取样量为 $1\mathrm{g}$ ，定容体积为 $20~\mathrm{mL}$ 时，方法检出限为：稻米 $0.01\mathrm{mg / kg}$ 、水产动物 $0.02\mathrm{mg / kg}$ 、婴幼儿辅助食品 $0.01\mathrm{mg / kg}$ ；方法定量限为：稻米 $0.03\mathrm{mg / kg}$ 、水产动物 $0.06\mathrm{mg / kg}$ 、婴幼儿辅助食品 $0.03\mathrm{mg / kg}$ 。

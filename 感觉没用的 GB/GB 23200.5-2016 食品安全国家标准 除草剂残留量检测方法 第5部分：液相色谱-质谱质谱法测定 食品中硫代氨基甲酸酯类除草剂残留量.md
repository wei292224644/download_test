# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T1737.5—2010《除草剂残留量检验方法第5部分：液相色谱一质谱/质谱法测定进出口食品中硫代氨基甲酸酯类除草剂残留量》。与SN/T1737.5—2010相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；

——标准名称中“进出口食品”改为“食品”；

标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T1737.5—2010。

# 食品安全国家标准除草剂残留量检测方法

# 第5部分：液相色谱一质谱/质谱法测定食品中硫代氨基甲酸酯类除草剂残留量

# 1 范围

本标准规定了食品中9种硫代氨基甲酸酯类农药(参见附录A)残留量的液相色谱一质谱联用测定方法。

本标准适用于大米、大豆、白萝卜、小白菜、椰菜、生姜、茶叶、花生、橙、葡萄、鸡肉、鸡肝和鱼肉中9种硫代氨基甲酸酯类农药残留的定量测定，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

采用乙腈提取试样中残留的硫代氨基甲酸酯除草剂，提取液经HLB和Envi-Carb固相萃取柱净化，液相色谱—质谱联用仪检测和确证，内标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN, CAS}$ 号：75-05-8）：色谱纯。  
4.1.2 正己烷 $\mathrm{(C_6H_{14}}$ ，CAS号： $110 - 54 - 3)$   
4.1.3 丙酮 $\mathrm{(C_3H_6O}$ ，CAS号： $67 - 64 - 1)$   
4.1.4 乙酸 $\mathrm{(C_2H_4O_2}$ ，CAS号：64-19-7）：色谱纯。  
4.1.5 氯化钠(NaCl，CAS号：7647-14-5)。

# 4.2 溶液配制

4.2.1 正己烷一丙酮溶液(7+3，体积比)：准确量取 $70~\mathrm{mL}$ 正己烷和 $30~\mathrm{mL}$ 丙酮，摇匀备用。  
4.2.2 乙腈一水溶液 $(1 + 1$ ，体积比)：准确量取 $50~\mathrm{mL}$ 乙腈和 $50~\mathrm{mL}$ 水，摇匀备用。

# 4.3 标准品

4.3.1 标准物质：参见表A.1，纯度均大于 $99\%$   
4.3.2 内标标准物质： $\mathrm{D}_3$ 一甲萘威 $(100\mu \mathrm{g / mL})$ 。

# 4.4 标准溶液配制

4.4.1 硫代氨基甲酸酯农药标准储备溶液 $(100\mu \mathrm{g} / \mathrm{mL})$ ：准确称取适量的各种硫代氨基甲酸酯标准物质，用乙腈配制成浓度为 $100\mathrm{mg / L}$ 的标准储备溶液，避光一 $18^{\circ}\mathrm{C}$ 保存，保存期为6个月。

4.4.2 硫代氨基甲酸酯农药混合标准中间溶液 $(1.0\mu \mathrm{g} / \mathrm{mL})$ ：吸取每种适量标准储备溶液，用乙腈稀释成 $1.0\mathrm{mg / L}$ 的混合标准工作溶液，避光一 $18^{\circ}\mathrm{C}$ 保存，保存期为1个月。  
4.4.3 内标中间溶液 $(1.0\mu \mathrm{g / mL})$ ：准确吸取适量的内标标准物质，用乙腈配制成浓度为 $1.0\mathrm{mg / L}$ 内标工作溶液，避光一 $18^{\circ}C$ 保存，保存期为1个月。

# 4.5 材料

4.5.1 HLB固相萃取柱或相当者： $6\mathrm{mL} / 200\mathrm{mg}$ 。使用前用 $5\mathrm{mL}$ 乙腈处理，保持柱体湿润。  
4.5.2 Envi-Carb 固相萃取柱或相当者: $3 \mathrm{~mL} / 200 \mathrm{mg}$ 。使用前用 $5 \mathrm{~mL}$ 正己烷一丙酮处理, 保持柱体湿润。  
4.5.3 微孔滤膜： $0.22\mu \mathrm{m}$ ，有机相型。

# 5 仪器和设备

5.1 液相色谱—质谱联用仪：配有电喷雾离子源(ESI)。  
5.2 分析天平：感量 $0.1\mathrm{mg}$ 和 $0.01\mathrm{g}$   
5.3 吹氮浓缩仪。  
5.4 离心机：转速 $>5000 \mathrm{r} / \mathrm{min}$ 。   
5.5 均质器。  
5.6 涡旋振荡器。  
5.7 固相萃取装置。  
5.8 移液器： $10~\mu \mathrm{L} \sim 100~\mu \mathrm{L}$ 和 $100~\mu \mathrm{L} \sim 1000~\mu \mathrm{L}$ 。  
5.9 聚丙烯离心管： $50~\mathrm{mL}$ 或 $15~\mathrm{mL}$ ，具塞。  
5.10 容量瓶： $25~\mathrm{mL}$

# 6 试样制备与保存

# 6.1 试样制备

# 6.1.1 取样部位

样品取样部位按GB2763的规定执行。在取样和制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 6.1.2 大米、大豆、花生和茶叶

取代表性样品约 $500\mathrm{g}$ ，用粉碎机充分粉碎，样品全部过 $425\mu \mathrm{m}$ 的标准网筛。混匀。试样均分为2份，装入洁净容器，密封，并标明标记。

# 6.1.3 白萝卜、小白菜、椰菜、生姜、橙、葡萄、鸡肉、鸡肝和鱼肉

取代表性样品约 $500\mathrm{g}$ ，将其切碎后（不可水洗），用捣碎机将样品加工成浆状，混匀。试样均分为2份，装入洁净容器，密封，并标明标记。

# 6.2 试样保存

大米、大豆、花生和茶叶样品常温下保存。白萝卜、小白菜、椰菜等试样于一 $18^{\circ} \mathrm{C}$ 以下冷冻存放。

# 7 分析步骤

# 7.1 提取

# 7.1.1 大米、大豆、茶叶和花生

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 样品于 $50\mathrm{mL}$ 具塞离心管中，茶叶称取 $2.5\mathrm{g}$ ，加入 $100\mu\mathrm{L}$ 内标中间溶液，加入 $10\mathrm{mL}$ 水，混匀后放置 $1\mathrm{h}$ 。然后加入 $4.0\mathrm{g}$ 氯化钠，再加入 $15\mathrm{mL}$ 乙腈高速均质提取 $3\mathrm{min}$ ，振荡

提取 $15 \mathrm{~min}$ , 于 $5000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 将乙腈层转移至 $25 \mathrm{~mL}$ 容量瓶中。残渣再用 $10 \mathrm{~mL}$ 乙腈重复提取一次, 合并提取液, 并用乙腈定容至 $25 \mathrm{~mL}$ 。

# 7.1.2 白萝卜、小白菜、椰菜、生姜、橙、葡萄、鸡肉、鸡肝和鱼肉

称取 $5\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 样品于 $50\mathrm{mL}$ 具塞离心管中，加入 $100\mu \mathrm{L}$ 内标中间溶液和 $4.0\mathrm{g}$ 氯化钠，以下操作与7.1.1相同。

# 7.2 净化

# 7.2.1 大米、白萝卜、小白菜、椰菜、生姜、橙、葡萄、鸡肉和鱼肉

取 $5.0 \mathrm{~mL}$ 提取液于 $15 \mathrm{~mL}$ 离心管中, $40^{\circ} \mathrm{C}$ 下用 $\mathrm{N}_{2}$ 吹至近干。残渣用 $2.0 \mathrm{~mL}$ 正己烷一丙酮溶液溶解, 将溶解液加入处理过的 Envi-Carb 固相萃取柱中, 以约 $1.5 \mathrm{~mL} / \mathrm{min}$ 的流速使样液全部通过固相萃取柱, 再用 $5 \mathrm{~mL}$ 正己烷一丙酮溶液润洗样品管, 并将润洗液一并加入 Envi-Carb 固相萃取柱, 收集全部流出液于 $15 \mathrm{~mL}$ 离心管中, $40^{\circ} \mathrm{C}$ 用 $\mathrm{N}_{2}$ 吹至近干。残渣用乙腈一水溶液定容至 $1.0 \mathrm{~mL}$ , 涡旋混匀后, 过微孔滤膜, 供液相色谱一质谱联用仪测定。

# 7.2.2 大豆、茶叶、花生和鸡肝

7.2.2.1 取 $5.0 \mathrm{~mL}$ 提取液于 $15 \mathrm{~mL}$ 离心管中, 茶叶提取液取 $10.0 \mathrm{~mL}, 40^{\circ} \mathrm{C}$ 下用 $\mathrm{N}_{2}$ 吹至 $2.0 \mathrm{~mL}$ 左右。将浓缩液转入处理过的 HLB 固相萃取柱中, 以约 $1.5 \mathrm{~mL} / \mathrm{min}$ 的流速使样液全部通过固相萃取柱, 再用 $3 \mathrm{~mL}$ 乙腈淋洗并抽干固相萃取柱, 收集全部流出液于 $15 \mathrm{~mL}$ 离心管中, $40^{\circ} \mathrm{C}$ 下用 $\mathrm{N}_{2}$ 吹至近干。  
7.2.2.2 残渣用 $2.0 \mathrm{~mL}$ 正己烷一丙酮溶液溶解，将溶解液加入处理过的 Envi-Carb 固相萃取柱中，以约 $1.5 \mathrm{~mL} / \mathrm{min}$ 的流速使样液全部通过固相萃取柱，再用 $5 \mathrm{~mL}$ 正己烷一丙酮溶液润洗样品管，并将润洗液一并加入 Envi-Carb 固相萃取柱，收集全部流出液于 $15 \mathrm{~mL}$ 离心管中， $40^{\circ} \mathrm{C}$ 用 $\mathrm{N}_{2}$ 吹至近干。残渣用乙腈一水溶液定容至 $1.0 \mathrm{~mL}$ ，旋涡混匀后，过微孔滤膜，供液相色谱一质谱联用仪测定。

# 7.4 标准工作曲线

吸取适量的混合标准中间溶液 $(1.0\mu \mathrm{g / mL})$ 和内标中间溶液 $(1.0\mu \mathrm{g / mL})$ ，用空白样品提取液配成浓度为 $0\mu \mathrm{g / L},1.0\mu \mathrm{g / L},5.0\mu \mathrm{g / L},10.0\mu \mathrm{g / L},20.0\mu \mathrm{g / L},40.0\mu \mathrm{g / L},100\mu \mathrm{g / L}$ 的基质混合标准工作溶液，内标浓度均为 $20.0\mu \mathrm{g / L}$ 。临用配制，供液相色谱一质谱联用仪测定。以待测物和内标物峰面积的比值为纵坐标，基质混合标准工作溶液浓度为横坐标绘制标准工作曲线。

# 7.5 测定

# 7.5.1 定性测定

每种被测组分选择1个母离子、2个以上子离子，在上述实验条件下，样品中待测物质的保留时间与基质标准溶液的保留时间偏差在 $\pm 2.5\%$ 之内；且样品中各组分定性离子的相对丰度与浓度接近的基质混合标准工作溶液中对应的定性离子的相对丰度进行比较，偏差不超过表2规定的范围，则可判定为样品中存在对应的待测物。

# 7.5.2 定量测定

在仪器最佳工作条件下, 对基质混合标准工作溶液进样, 以待测物和内标物峰面积的比值为纵坐标, 基质混合标准工作溶液浓度为横坐标绘制标准工作曲线, 用标准工作曲线对样品进行定量, 样品溶液中待测物的响应值均应在仪器测定的线性范围内; 如果超出线性范围, 应用空白基质溶液进行适当稀释后进样测定。9 种硫代氨基甲酸酯类除草剂标准物质的多反应监测 (MRM) 色谱图参见图 C. 1。

# 8 结果计算和表述

按式(1)计算试样中硫代氨基甲酸酯类除草剂的残留量。

$$
X = \frac {c \times c _ {i} \times A \times A _ {s} \times V \times 1 0 0 0}{c _ {s i} \times A _ {i} \times A _ {s} \times m \times 1 0 0 0} \tag {1}
$$

式中：

$X$ ——样品中待测组分残留量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

$c$ ——标准工作溶液中药物的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$c_{i}$ ——样液中内标物的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

A ——样液中药物的峰面积；

$A_{s}$ ——标准工作溶液中内标物的峰面积；

V ——样品溶液最终定容体积,单位为毫升 $(\mathrm{mL})$ ；

$c_{si}$ ——标准工作溶液中内标物的浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$ ；

$A_{i}$ ——样液中内标物的峰面积；

$A_{s}$ ——标准工作溶液中药物的峰面积；

$m$ ——最终样液代表的试样质量，单位为克(g)；

1000——换算系数。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附

# 10 定量限和回收率

# 10.1 定量限

本方法对9种硫代氨基甲酸酯类除草剂的定量限均为 $5.0\mu \mathrm{g / kg}$

# 10.2 回收率

在 $5.0\mu \mathrm{g} / \mathrm{kg}\sim 200\mu \mathrm{g} / \mathrm{kg}$ 添加浓度范围内，大米、大豆、白萝卜、小白菜、椰菜、生姜、茶叶、花生、橙、葡萄、鸡肉、鸡肝和鱼肉中9种硫代氨基甲酸酯类除草剂的回收率数据参见附录F。

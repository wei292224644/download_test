# 前言

本标准代替GB1886.174—2016《食品安全国家标准 食品添加剂 食品工业用酶制剂》。

本标准与GB1886.174—2016相比，主要变化如下：

——增加了“酶制剂用辅料”“固定化酶制剂”和“固定化载体”的术语和定义；  
——增加了酶制剂用辅料和固定化载体的技术要求；  
——修改了大肠埃希氏菌的指标要求；  
——增加了酶制剂用辅料名单的附录要求；  
——增加了附录中部分酶活力测定参考方法。

# 食品安全国家标准

# 食品添加剂 食品工业用酶制剂

# 1 范围

本标准适用于《食品安全国家标准 食品添加剂使用标准》(GB2760)和相关公告允许使用的食品工业用酶制剂。

# 2 术语和定义

# 2.1 食品工业用酶制剂

由动物或植物的可食或非可食部分直接提取，或由传统选育或经基因重组技术得到的微生物（包括但不限于细菌、放线菌、真菌菌种)发酵、提取后，或再经进一步纯化、制剂化等工艺制得的(可含有一个或多个活性酶组分)，用于食品工业，具有特殊催化活性的制剂产品。

# 2.2 酶活力

酶在一定条件下催化某一特定反应的活性能力高低的指标。

# 2.3 抗菌活性

抑制或杀灭微生物的特性。

# 2.4 酶制剂用辅料

为酶制剂产品的活性保存、流通贮存、标准化使用等目的而加入的食品原料和食品添加剂。

# 2.5 固定化酶制剂

利用物理和(或)化学的方法将酶固定于载体上，在食品生产过程中以不可溶解状态发挥作用的产品剂型。

# 2.6 固定化载体

固定化酶制剂中起固定酶作用的材料。

# 3 产品分类

按产品形态分为固体剂型（含固定化剂型）和液体剂型。

# 4 技术要求

# 4.1 原辅料要求

4.1.1 用于生产酶制剂的原辅料应符合相关要求，在规定的使用条件下，不应对最终食品产生有害健

康的残留污染。

4.1.2 用于提取酶制剂的动物组织应符合肉类检疫要求。  
4.1.3 用于提取酶制剂的植物组织不应霉变。  
4.1.4 对微生物生产菌种应进行分类学和(或)遗传学的鉴定，并应符合《食品安全国家标准 食品添加剂使用标准》(GB 2760)及相关公告。菌种的保藏方法和条件应保证发酵批次之间的稳定性和可重复性。  
4.1.5 为了酶制剂产品的活性保存、流通贮存和标准化使用，商品化的酶制剂允许加入必要的辅料成分。酶制剂用辅料不应在最终食品中发挥功能作用，一般按生产需要适量使用，在达到预期的目的下尽可能减少使用品种和用量。酶制剂中允许使用的辅料名单见附录A。  
4.1.6 固定化酶制剂所使用的固定化载体应符合《食品安全国家标准 食品接触材料及制品用添加剂使用标准》(GB 9685)、《食品安全国家标准 食品添加剂使用标准》(GB 2760)或其他在食品中允许使用的原料清单的规定。

# 4.2 理化指标

产品酶活力应符合声称。

注：部分酶制剂的酶活力测定参考方法参见附录B。企业可按相应标准中给出的或企业规定的方法测定。

# B.2 $\pmb{\alpha}$ -乙酰乳酸脱羧酶活力的测定

# B.2.1 $\alpha$ -乙酰乳酸脱羧酶

可以与 $\alpha$ -乙酰乳酸反应脱羧，将 $\alpha$ -乙酰乳酸羧基转化为3-羟基-2-丁酮的酶。

# B.2.2 $\alpha$ -乙酰乳酸脱羧酶活力

在 $30^{\circ}\mathrm{C},\mathrm{pH}6.0$ 的条件下， $1\mathrm{g}$ 或 $1\mathrm{mL}$ 酶样品与底物 $\alpha-$ 乙酰乳酸发生反应，每分钟生成 $1\mu\mathrm{mol}$ 的3-羟基-2-丁酮(乙偶姻)，即为1个酶活力单位，以 $\mathrm{U/g}$ (或 $\mathrm{U/mL}$ )表示。

# B.2.3 分光光度计法

# B.2.3.1 范围

本方法规定了 $\alpha$ -乙酰乳酸脱羧酶酶活力的测定方法。

本方法适用于用分光光度计测定 $\alpha$ -乙酰乳酸脱羧酶制剂中 $\alpha$ -乙酰乳酸脱羧酶的酶活力。本方法

不适用于啤酒和其他含醇产品中 $\alpha$ -乙酰乳酸脱羧酶的酶活力的测定。

试样中3-羟基-2-丁酮(乙偶姻)和/或双乙酰的存在会使试验结果偏大。

乙偶姻在贮存时易形成二聚体，从而影响试验结果。但若遵循配制步骤中所述的措施去预防，本方法仍可使用。

# B.2.3.2 原理

$\alpha-$ 乙酰乳酸脱羧酶与底物 $\alpha-$ 乙酰乳酸反应脱羧生成乙偶姻。乙偶姻在碱性条件下与萘酚和肌酸的混合物反应生成红色产物。通过在 $522 \mathrm{~nm}$ 下测定溶液的吸光度，可以从乙偶姻标准曲线上得出反应生成的乙偶姻的量，进而计算出 $\alpha-$ 乙酰乳酸脱羧酶的酶活力。

# B.2.3.5.1 乙偶姻标准曲线的制备

分别吸取配制好的乙偶姻标准溶液 $400\mu \mathrm{L}$ 于6个不同的 $10~\mathrm{mL}$ 试管中。然后按表B.2依次在每个试管中加入显色剂 $4.60~\mathrm{mL}$ ，用涡旋振荡器充分混合均匀。室温下反应 $40.0\mathrm{min}$ 后，使用分光光度计，在 $522~\mathrm{nm}$ 下测定各管溶液的吸光度。

# B.2.3.5.2 标准对照品的制备

建议使用一个有代表性的、稳定的样品作为标准对照品。

在每次分析中，将该标准对照品与样品一同进行检测，以判断试验的重复性。

# B.2.3.5.3 样品溶液的制备

从样品中称取一定量的样品，精确至 $0.0005\mathrm{g}$ ，用溶液稀释。其稀释的倍数要使样品的最终吸光度 $H_{1}$ 落在乙偶姻标准曲线的范围内。

# B.2.3.5.4 标准曲线的绘制

以 $522\mathrm{nm}$ 波长下的吸光度为 $\mathrm{Y}$ 轴，乙偶姻的浓度 $(\mathrm{mg / L})$ 为 $X$ 轴，绘制标准曲线，计算出标准曲线的斜率 $h$ （或用回归方程计算）。

# B.2.3.5.5 测定

将试样溶液和底物先置于 $30^{\circ}\mathrm{C}$ 水浴中预热约 $10\mathrm{min}$ ，吸取酶样品溶液 $200.0\mu \mathrm{L}$ 于 $30^{\circ}\mathrm{C} \pm 0.1^{\circ}\mathrm{C}$ 水浴中的 $10\mathrm{mL}$ 试管里，然后加入底物 $200.0\mu \mathrm{L}$ ，立即计时，并用涡旋振荡器充分混匀，迅速放回水浴中。反应 $20.0\mathrm{min}$ 后，依次向试管中加入显色剂 $4.60\mathrm{mL}$ ，用涡旋振荡器充分混匀。置于室温，重新开始计时。反应 $40.0\mathrm{min}$ 后，使用分光光度计，在 $522\mathrm{nm}$ 波长下测定各管溶液的吸光度。同时用缓冲液代替试样溶液进行空白试验。

# B.2.3.6 结果计算

$\alpha$ -乙酰乳酸脱羧酶制剂的酶活力 $X_{1}$ ，单位为 $\mathrm{U / g}$ （或 $\mathrm{U / mL}$ )，按式(B.1)计算：

$$
X _ {1} = \frac {\left(H _ {1} - H _ {2}\right) \times 0 . 0 0 1 1 3 5 1 \times F _ {1}}{m \times h} \quad \dots \tag {B.1}
$$

式中：

$H_{1}$ —样品的吸光度；

$H_{2}$ ——空白的吸光度；

0.001 135 1—— $0.1\mathrm{g}$ 的乙偶姻所对应的摩尔数；

$F_{1}$ ——样品溶液反应前的总稀释倍数；  
$m$ ——试料的质量，单位为克 $(\mathrm{g})$ ；  
$h$ ——标准曲线的斜率。

样品的测定结果用算术平均值表示。

当结果小于 $1\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 时给出1位有效数字；当结果大于或等于 $1\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 且小于 $100\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 时给出2位有效数字；当结果大于或等于 $100\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 时给出3位有效数字。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $10\%$ 。

# B.2.4.7 结果计算

# B.2.4.7.1 标准曲线的计算

用参数 Logit-Log 法计算出标准曲线。其中 $Y$ 轴单位为 $\mathrm{OD / min}$ , $X$ 轴单位为标准点的酶活力 $\mathrm{mU / mL}$ 。

# B.2.4.7.2 样品活力的计算

从标准曲线上读出样品最终稀释液的活力，单位为 $\mathrm{mU / mL}$

$\alpha$ -乙酰乳酸脱羧酶制剂的酶活力 $X_{2}$ ，单位为 $\mathrm{U / g}$ （或 $\mathrm{U / mL}$ )，按式(B.2)计算：

$$
X _ {2} = \frac {A _ {1} \times F _ {2} \times D}{m \times 1 0 0 0} \quad \dots \tag {B.2}
$$

式中：

$A_{1}$ ——由标准曲线得出的样品最终稀释液的活力，单位为毫酶活力单位每毫升 $(\mathrm{mU} / \mathrm{mL})$ ；

$F_{2}$ ——溶解样品用的容量瓶的体积，单位为毫升 $(\mathrm{mL})$ ；

$D$ —稀释倍数；

$m$ ——样品的质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

样品的测定结果用算术平均值表示。当标准对照品稀释液的试验值在 $0.48\mathrm{mU / mL}\sim 0.52\mathrm{mU / mL}$ 时，样品的试验结果有效，可计算平均值。否则，应重新进行试验。

当结果小于 $1\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 时给出1位有效数字；当结果大于或等于 $1\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 且小于 $100\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 时给出2位有效数字；当结果大于或等于 $100\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 时给出3位有效数字。当结果小于 $0.6\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 时，表示为 $< 0.6\mathrm{U} / \mathrm{g}$ (或 $\mathrm{U} / \mathrm{mL}$ ) 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $4\%$ 。

# B.3 果胶酶活力的测定

# B.3.1 果胶酶

能水解果胶，生成含有还原性基团产物的酶。

# B.3.2 果胶酶活力

在 $50~\mathrm{^\circ C}$ 、 $\mathrm{pH}3.5$ 的反应条件下， $1\mathrm{g}$ 固体酶粉(或 $1\mathrm{mL}$ 液体酶)， $1\textrm{h}$ 分解果胶产生 $1\mathrm{mg}$ 半乳糖醛酸，即为1个酶活力单位，以 $\mathrm{U / g}$ 或 $\mathrm{U / mL}$ 表示。

# B.3.3 原理

果胶酶能水解果胶，生成的半乳糖醛酸的还原性糖醛基可用次亚碘酸法定量测定，以此来表示果胶酶的活力。

本方法主要用于检测果胶酶产品中多聚半乳糖醛酸酶和果胶裂解酶的活力，对于果胶酶中的果胶（甲基)酯酶不适用。

# B.3.7 结果计算

果胶酶制剂的酶活力 $X_{3}$ ，单位为 $\mathrm{U / g}$ （或 $\mathrm{U / mL}$ )，按式(B.3)计算：

$$
X _ {3} = \frac {\left(A _ {2} - B _ {1}\right) \times c _ {1} \times 0 . 5 1 \times 1 9 4 . 1 4 \times n _ {1} \times 1 0}{5 \times 1 \times 0 . 5} \quad \dots \tag {B.3}
$$

式中：

$A_{2}$ ——空白消耗硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$B_{1}$ ——样品消耗硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$c_{1}$ ——硫代硫酸钠标准溶液的摩尔浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

0.51 —— $1\mathrm{mmol}$ 硫代硫酸钠相当于 $0.51\mathrm{mmol}$ 的游离半乳糖醛酸；

194.14 ——半乳糖醛酸的毫摩尔质量，单位为毫克 $(\mathrm{mg})$ ；

$n_1$ ——稀释倍数；

10 ——反应液总体积，单位为毫升 $(\mathrm{mL})$

5 ——滴定时取反应混合物的总体积，单位为毫升 $(\mathrm{mL})$ ；

1 ——反应时加入稀释酶液的体积，单位为毫升 $(\mathrm{mL})$ ；

0.5 ——反应时间，单位为小时(h)。

所得结果表示至整数。

# B.5 转葡糖苷酶活力的测定

# B.5.1 转葡糖苷酶

可水解麦芽糖分子及直链麦芽糊精中 $\alpha -1,4$ -糖苷链，同时将游离出来的葡萄糖残基转移到一个葡萄糖分子或麦芽糖、麦芽三糖分子上，形成 $\alpha -1,6$ 糖苷键，生成异麦芽糖、潘糖、异麦芽三糖等含 $\alpha -1$ 键的寡糖的酶。

# B.5.2 转葡糖苷酶活力

在 $40^{\circ}\mathrm{C},\mathrm{pH}5.0$ 的条件下， $1\mathrm{mL}$ 酶样品与底物 $\alpha$ -甲基-D-葡萄糖苷起反应， $60\mathrm{min}$ 生成 $1\mu \mathrm{g}$ 的葡萄糖，即为1个酶活力单位，以 $\mathrm{U / g}$ （或 $\mathrm{U / mL}$ ）表示。

# B.5.3 原理

转葡糖苷酶作用于底物 $\alpha$ -甲基-D-葡萄糖苷，生成的葡萄糖与含有葡萄糖氧化酶、过氧化酶的4-氨基安替比林和酚试剂进行显色反应定量测定。

# B.5.4 试剂和材料

B.5.4.1 $0.1\mathrm{mol} / \mathrm{L}$ 乙酸溶液：将乙酸 $\mathrm{(CH_3COOH)}6.0\mathrm{g}$ 溶于水，稀释至 $1000~\mathrm{mL}$   
B.5.4.2 $0.1\mathrm{mol} / \mathrm{L}$ 乙酸钠溶液：将乙酸钠 $\mathrm{(CH_3COOHNa)}8.20\mathrm{g}$ 溶于水，稀释至 $1000~\mathrm{mL}$

B.5.4.4 Tris-磷酸盐缓冲液 $(\mathrm{pH}7.2)$ ：将Tris(羟甲基)氨基甲烷 $\left[\mathrm{H}_{2}\mathrm{NC}(\mathrm{CH}_{2}\mathrm{OH})_{3}\right]36.3\mathrm{~g}$ 和二水磷酸二氢钠 $\left(\mathrm{NaH}_{2}\mathrm{PO}_{4} \cdot 2\mathrm{H}_{2}\mathrm{O}\right)50.0\mathrm{~g}$ 溶于 $900\mathrm{~mL}$ 水，用 $2\mathrm{~mol/L}$ 盐酸溶液调节 $\mathrm{pH}$ 至7.2，用水稀释至 $1000\mathrm{~mL}$ 。

B.5.4.6 $5 \%$ 苯酚溶液：将苯酚 $\left(\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{OH}\right) 5 \mathrm{~g}$ 溶于 $50 \mathrm{~mL} 60^{\circ} \mathrm{C}$ 水中，将溶液冷却至室温，用水稀释至 $100 \mathrm{~mL}$ 。

# B.5.5.1 样品溶液的制备

标准液的空白对照：用水来替代葡萄糖标准液，按上述方法测定空白对照液吸光度 $A_{\mathrm{s0}}$ 。

# B.5.6 结果计算

转葡糖苷酶活力 $X_{5}$ ，单位为 $\mathrm{U / g}$ （或 $\mathrm{U / mL}$ )，按式(B.5)计算：

$$
X _ {5} = \left(A _ {6 0} - A _ {0}\right) \times G \times \frac {2 . 5}{0 . 1} \times \frac {n _ {3}}{0 . 5} \quad \dots \tag {B.5}
$$

式中：

$A_{60}$ ——酶样品反应液的吸光度值；

$A_{0}$ ——空白对照液的吸光度值；

$G$ ——吸光度差为1.000时，由葡萄糖标准曲线求得的葡萄糖量，单位为微克 $(\mu \mathrm{g})$ ；

2.5——反应体系的总容量，单位为毫升 $(\mathrm{mL})$ ；

0.1——反应体系的取样量，单位为毫升 $(\mathrm{mL})$ ；

$n_3$ ——酶样品的稀释倍数；

0.5——反应体系中酶样品的添加量，单位为毫升 $(\mathrm{mL})$ 。

其中， $G$ 按式（B.6）计算：

$$
G = \frac {\frac {1 0}{A _ {\mathrm {s} 1 0} - A _ {\mathrm {s} 0}} + \frac {2 0}{A _ {\mathrm {s} 2 0} - A _ {\mathrm {s} 0}} + \frac {3 0}{A _ {\mathrm {s} 3 0} - A _ {\mathrm {s} 0}} + \frac {4 0}{A _ {\mathrm {s} 4 0} - A _ {\mathrm {s} 0}} + \frac {5 0}{A _ {\mathrm {s} 5 0} - A _ {\mathrm {s} 0}}}{5} \dots \dots \dots \dots \dots \dots \tag {B.6}
$$

式中：

$A_{\mathrm{sx}}$ ——各反应液对应的吸光度值；

$A_{\mathrm{s0}}$ ——空白对照液对应的吸光度值。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

# B.5.7 转葡糖苷酶中糖化酶活力的测定

# B.5.7.1 原理

酶作用于淀粉溶液生成还原糖。然后加入费林试剂，在加热的情况下，定量生成氧化亚铜沉淀。在

加入碘化钾和硫酸后，生成游离碘，此时，立即用硫代硫酸钠溶液滴定。

# B.5.7.2 试剂和材料

B.5.7.2.1 $30\%$ 碘化钾溶液：碘化钾 $150\mathrm{g}$ 溶解于 $350~\mathrm{mL}$ 水，保存在褐色试剂瓶中，避免阳光直射。

B.5.7.2.2 $25\%$ 硫酸溶液：硫酸 $125\mathrm{g}$ 溶于 $375~\mathrm{mL}$ 水。

B.5.7.2.3 $0.05 \mathrm{~mol} / \mathrm{L}$ 硫代硫酸钠溶液: 将定量分析用的 $0.1 \mathrm{~mol} / \mathrm{L}$ 硫代硫酸钠溶液 $500 \mathrm{~mL}$ 用煮沸过的冷却水稀释, 所得溶液的总体积为 $1000 \mathrm{~mL}$ 。配制好后, 应进行标定, 求出浓度校正系数 $f$ 值。

B.5.7.2.4 $1 \mathrm{~mol} / \mathrm{L}$ 乙酸-乙酸钠缓冲溶液 $(\mathrm{pH} 4.5)$ : 将 $1 \mathrm{~mol} / \mathrm{L}$ 乙酸钠溶液加入 $1 \mathrm{~mol} / \mathrm{L}$ 乙酸溶液中, 调节 $\mathrm{pH}$ 至 4.5。

B.5.7.2.5 可溶性淀粉溶液 $(\mathrm{pH}4.5)$ ：将可溶性淀粉(试剂级)在 $105^{\circ}\mathrm{C}$ 干燥 $4\mathrm{h}$ 后称量计算含水量。然后，根据可溶性淀粉的含水量称取 $0.50\mathrm{g}$ （折干）的可溶性淀粉，缓慢加入沸水 $50~\mathrm{mL}$ 中，煮沸 $5\mathrm{min}$ ，用自来水冷却后，加入 $1\mathrm{mol / L}$ 乙酸-乙酸钠缓冲溶液 $5.0~\mathrm{mL}$ ，用水定容至 $100~\mathrm{mL}$ 。

B.5.7.2.6 费林试剂如下：

铜溶液：硫酸铜 $34.66\mathrm{g}$ 溶解于水中，定容至 $500~\mathrm{mL}$   
—酒石酸钾钠碱溶液：酒石酸钾钠 $173\mathrm{g}$ 和氢氧化钠 $50\mathrm{g}$ 溶解于水中，定容至 $500~\mathrm{mL}$   
——上述溶液使用前，精确地取等体积的铜溶液和碱液充分混合。

# B.5.7.3 分析步骤

# B.5.7.3.1 样品溶液的制备

用水稀释酶样品，使得到酶液的 $(T_0 - T_{30})\times f\times 1.62$ 值为 $3\mathrm{mg}\sim 10\mathrm{mg}$ 葡萄糖量。

# B.5.7.3.2 测定

将可溶性淀粉溶液 $10\mathrm{mL}$ 加入 $100\mathrm{mL}$ 三角瓶中，置于 $40^{\circ}\mathrm{C} \pm 0.5^{\circ}\mathrm{C}$ 的恒温水浴箱中，预热 $10\mathrm{min} \sim 15\mathrm{min}$ ，加入样品稀释酶液 $1\mathrm{mL}$ 。准确加热 $30\mathrm{min}$ 后，加入费林试剂 $4\mathrm{mL}$ 使酶失活。将三角瓶直接在煤气喷灯(或电炉)上加热 $2\mathrm{min}$ 后，立即放在自来水中冷却。随后，加入 $30\%$ 碘化钾溶液 $2\mathrm{mL}$ ，用硫代硫酸钠溶液滴定游离出的碘，以蓝色消失为滴定终点 $T_{30}(\mathrm{mL})$ 。

空白对照试验：以水取代酶液，在另一个三角瓶中用上述同样的操作步骤测定空白对照值 $T_{0}$ (mL)。临近终点时，加入 $1\%$ 可溶性淀粉溶液[应使用可溶性淀粉(试剂级)另行配制]1滴～2滴，以蓝色消失作为滴定终点。

# B.5.7.4 结果计算

在上述试验条件下，反应 $30\mathrm{min}$ ，反应液中产生相当于 $10\mathrm{mg}$ 葡萄糖的还原糖所需的酶量，定义为1个葡糖淀粉酶活力单位。

转葡糖苷酶中糖化酶活力 $X_{6}$ ，单位为 $\mathrm{U / g}$ （或 $\mathrm{U / mL}$ )，按式(B.7)计算：

$$
X _ {6} = \left(T _ {0} - T _ {3 0}\right) \times f \times 1. 6 2 \times \frac {1}{1 0} \times n \quad \dots \tag {B.7}
$$

式中：

$T_{0}$ ——空白溶液滴定消耗硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$T_{30}$ ——酶反应液滴定消耗硫代硫酸钠标准溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——0.05 mol/L 硫代硫酸钠标准溶液浓度的矫正系数；

1.62 ——换算系数；

1/10——该分析方法的常数（相当于 $10\mathrm{mg}$ 葡萄糖的还原糖）；

$n$ ——样品的稀释倍数。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于算术平均值的 $5\%$ 。

# 前言

本标准代替GB5009.33—2016《食品安全国家标准 食品中亚硝酸盐与硝酸盐的测定》。

本标准与GB5009.33—2016相比，主要变化如下：

——增加了连续流动分析-分光光度法为第三法，将原第三法修改为第四法；  
——第一法修改了样品的前处理方法；  
——第二法修改了样品的前处理方法、标准溶液的配制方法和还原效率的计算方法；  
——修改了方法的检出限和定量限。

# 食品安全国家标准

# 食品中亚硝酸盐与硝酸盐的测定

# 1 范围

本标准规定了食品中亚硝酸盐和硝酸盐的测定方法。

本标准第一法适用于食品(乳蛋白部分水解配方、乳蛋白深度水解配方、氨基酸配方及氨基酸代谢障碍配方的特殊医学用途配方食品除外)中亚硝酸盐和硝酸盐的测定。

本标准第二法和第三法适用于食品中亚硝酸盐和硝酸盐的测定。

本标准第四法适用于新鲜蔬菜、水果中硝酸盐的测定。

# 第一法 离子色谱法

# 2 原理

试样经沉淀蛋白质、除去脂肪后，经提取和净化，以氢氧化钾溶液或氢氧化钠溶液为淋洗液，阴离子交换柱分离，电导检测器或紫外检测器检测。以保留时间定性，外标法定量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为 $\mathrm{GB / T6682}$ 规定的一级水。

# 3.1 试剂

3.1.1 乙酸 $\mathrm{(CH_3COOH)}$ ：色谱纯。  
3.1.2 氢氧化钾(KOH)。  
3.1.3 氢氧化钠 $\mathrm{(NaOH)}$ 。  
3.1.4 $\alpha$ -淀粉酶：酶活力 $\geqslant 1.5 \mathrm{U} / \mathrm{mg}$ 。

# 3.2 试剂配制

3.2.1 乙酸溶液 $(3\%)$ ：量取乙酸 $3\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度，混匀。  
3.2.2 氢氧化钾溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $5.6\mathrm{g}$ 氢氧化钾，加入新煮沸过的冷水溶解，稀释至 $100~\mathrm{mL}$ 混匀。  
3.2.3 氢氧化钠溶液 $(1\mathrm{mol} / \mathrm{L})$ ：称取 $4.0\mathrm{g}$ 氢氧化钠，加入新煮沸过的冷水溶解，稀释至 $100~\mathrm{mL}$ 混匀。

# 3.3 标准品

3.3.1 亚硝酸钠标准品 $(\mathrm{NaNO}_2$ ，CAS号：7632-00-0)：纯度 $\geqslant 99\%$ 。或经国家认证并授予标准物质证书的亚硝酸盐标准品。  
3.3.2 硝酸钠标准品 $\mathrm{(NaNO_3}$ ,CAS号：7631-99-4)：纯度 $\geq 99\%$ 。或经国家认证并授予标准物质证书

的硝酸盐标准品。

# 3.4 标准溶液的制备

3.4.1 亚硝酸钠标准储备液 $(100\mathrm{mg / L})$ ：准确称取 $0.1000\mathrm{g}$ 于 $110^{\circ}\mathrm{C}\sim 120^{\circ}\mathrm{C}$ 干燥至恒重的亚硝酸钠，用水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀，于 $2^{\circ}\mathrm{C}\sim 6^{\circ}\mathrm{C}$ 冰箱保存。有效期为3个月。  
3.4.2 硝酸钠标准储备液 $(100\mathrm{mg / L})$ ：准确称取 $0.1000\mathrm{g}$ 于 $110^{\circ}C\sim 120^{\circ}C$ 干燥至恒重的硝酸钠，用水溶解并稀释至 $1000~\mathrm{mL}$ ，混匀，于 $2^{\circ}C\sim 6^{\circ}C$ 冰箱保存。有效期为3个月。  
3.4.3 亚硝酸钠和硝酸钠混合标准中间液 $(2.00\mathrm{mg / L})$ ：准确移取亚硝酸钠和硝酸钠的标准储备液各 $2.00~\mathrm{mL}$ 于 $100~\mathrm{mL}$ 容量瓶中，用水稀释至刻度。临用现配。  
3.4.4 亚硝酸钠和硝酸钠混合标准使用液：移取亚硝酸钠和硝酸钠混合标准中间液，加水逐级稀释，制成系列标准溶液，质量浓度分别为 $0\mathrm{mg / L},0.0100\mathrm{mg / L},0.0200\mathrm{mg / L},0.0500\mathrm{mg / L},0.100\mathrm{mg / L},$ $0.200\mathrm{mg / L},0.500\mathrm{mg / L},1.00\mathrm{mg / L}$ 和 $2.00\mathrm{mg / L}$ 。临用现配。

注：可根据样品中目标分析物的含量情况，调整标准曲线的范围，至少包含5个浓度水平（不含零点）。

# 3.5 材料

3.5.1 滤膜针头滤器：水性， $0.22\mu \mathrm{m}$   
3.5.2 $\mathrm{C}_{18}$ 净化柱： $1\mathrm{mL}$ ，或等效柱。使用前依次用 $10~\mathrm{mL}$ 甲醇、 $15~\mathrm{mL}$ 水活化。  
3.5.3 Ag净化柱：柱容量2.0离子当量～2.2离子当量，或等效柱。使用前用 $10\mathrm{mL}$ 水活化。  
3.5.4 Na净化柱：柱容量2.0离子当量～2.2离子当量，或等效柱。使用前用 $10\mathrm{mL}$ 水活化。

4.2 食物粉碎机。  
4.3 超声波清洗器。

# 5 分析步骤

# 5.1 试样制备与贮存

# 5.1.1 试样制备

5.1.1.1 液体或半固体样品：取适量样品通过搅拌或振摇使样品充分混匀。  
5.1.1.2 固体粉状样品：取适量样品通过振摇等手段使样品充分混匀。  
5.1.1.3 干酪：取适量样品研磨成均匀的泥浆状。为避免水分损失，研磨过程中避免产生过多的热量。  
5.1.1.4 其他固体样品：取适量样品粉碎至均匀粉末或制成匀浆，备用。

# 5.1.2 试样贮存

5.1.2.1 水果、蔬菜、肉类、水产品等易腐试样应于 $-18^{\circ} \mathrm{C}$ 及以下温度保存。  
5.1.2.2 其他样品按照产品的贮存条件保存。

# 5.2 试样提取

5.2.1 乳及乳制品、婴幼儿配方食品、特殊医学用途婴儿配方食品、特殊医学用途配方食品、燕窝及其

制品：称取液体试样 $10\mathrm{g}$ 、其他试样 $1\mathrm{g} \sim 2.5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），置于 $150\mathrm{mL}$ 具塞锥形瓶中，加水 $80\mathrm{mL}$ ，摇匀，超声 $30\mathrm{min}$ （必要时加热使之溶解或充分溶胀），取出放置至室温，定量转移至 $100\mathrm{mL}$ 容量瓶中，加入 $3\%$ 乙酸溶液 $2\mathrm{mL}$ （可根据沉淀效果适当调整用量），加水稀释至刻度，混匀。转移入 $250\mathrm{mL}$ 具塞碘量瓶，于 $4^{\circ}\mathrm{C}$ 放置 $20\mathrm{min}$ ，取出放置至室温，取部分溶液经滤纸过滤，滤液于 $10000\mathrm{r/min}$ 离心 $15\mathrm{min}$ ，上清液备用。

5.2.2 易糊化的高淀粉含量样品：称取试样 $2.5\mathrm{g}$ （精确至 $0.01\mathrm{g}$ ），置于 $150\mathrm{mL}$ 具塞锥形瓶中，加入 $\alpha-$ 淀粉酶 $0.2\mathrm{g}$ （可根据淀粉含量调整酶用量），加水 $80\mathrm{mL}$ ，摇匀，于 $55^{\circ}\mathrm{C}$ 下酶解 $30\mathrm{min}$ 。从“取出放置至室温”开始同5.2.1操作。  
5.2.3 其他样品：称取试样 $2\mathrm{g}\sim 10\mathrm{g}$ （精确至 $0.01\mathrm{g})$ ，置于 $150~\mathrm{mL}$ 具塞锥形瓶中，加入 $80~\mathrm{mL}$ 水，超声提取 $30\mathrm{min}$ ，每隔 $5\mathrm{min}$ 振摇1次，保持试样完全分散。于 $75^{\circ}C$ 水浴中放置 $5\mathrm{min}$ ，从“取出放置至室温”开始同5.2.1操作。

# 5.3 试样净化

取 $5.2.1\sim 5.2.3$ 备用溶液约 $15~\mathrm{mL}$ ，通过 $0.22\mu \mathrm{m}$ 水性滤膜针头滤器、 $\mathrm{C_{18}}$ 柱，弃去前面 $3\mathrm{mL}$ （如果氯离子大于 $100\mathrm{mg / L}$ 或目标物有干扰时，则需要依次通过针头滤器、 $\mathrm{C_{18}}$ 柱、 $\mathrm{Ag}$ 柱和Na柱，弃去前面 $7\mathrm{mL})$ ，收集后面洗脱液待测。

# 5.4 空白试验

除不加试样外，其他操作与试样的操作一致。

# 5.5 仪器参考条件

5.5.1 色谱柱：具有氢氧化物选择性，可兼容梯度洗脱的二乙烯基苯-乙基苯乙烯共聚物基质，烷醇基季铵盐功能团的高容量阴离子交换柱， $4\mathrm{mm}\times 250\mathrm{mm}$ （带保护柱 $4\mathrm{mm}\times 50\mathrm{mm}$ ），或性能相当的离子色谱柱。

# 5.6 测定

# 5.6.1 标准曲线的制作

将标准系列使用液分别注入离子色谱仪中（标准色谱图见附录A图A.1)，以标准使用液的浓度为横坐标，以峰高或峰面积为纵坐标，绘制标准曲线。

# 5.6.2 试样溶液的测定

将空白和试样溶液注入离子色谱仪中，得到空白和试样溶液的峰高或峰面积，根据标准曲线得到待测液中亚硝酸钠或硝酸钠的质量浓度。

# 6 分析结果的表述

试样中亚硝酸盐(以亚硝酸钠计)或硝酸盐(以硝酸钠计)的含量按公式(1)计算：

$$
X = \frac {\left(\rho - \rho_ {0}\right) \times V \times f \times 1 0 0 0}{m \times 1 0 0 0} \quad \dots \tag {1}
$$

式中：

$X$ ——试样中亚硝酸盐(以亚硝酸钠计)或硝酸盐(以硝酸钠计)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——测定用试样溶液中的亚硝酸盐(以亚硝酸钠计)或硝酸盐(以硝酸钠计)质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——空白液中亚硝酸盐(以亚硝酸钠计)或硝酸盐(以硝酸钠计)质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$f$ ——试样溶液稀释倍数；

1000 ——换算系数；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ 。

结果大于或等于 $1.00\mathrm{mg / kg}$ 时，保留3位有效数字；结果小于 $1.00\mathrm{mg / kg}$ 时，保留2位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 分光光度法

# 9 原理

试样经沉淀蛋白质、除去脂肪后，在弱酸条件下亚硝酸盐与对氨基苯磺酸重氮化后，再与盐酸萘乙二胺偶合形成紫红色化合物，测得亚硝酸盐含量。采用镉柱将硝酸盐还原成亚硝酸盐，测得亚硝酸盐总

量，由此总量减去亚硝酸盐含量，即得试样中硝酸盐含量。

# 13 分析结果的表述

# 13.1 亚硝酸盐含量计算

亚硝酸盐（以亚硝酸钠计）的含量按式(2)进行计算：

$$
X _ {1} = \frac {\rho_ {1} \times 1 0 0 0 \times V \times f _ {1}}{m _ {1} \times \frac {V _ {1}}{V _ {0}} \times 1 0 0 0} \tag {2}
$$

式中：

$X_{1}$ ——试样中亚硝酸盐(以亚硝酸钠计)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho_{1}$ ——测定用样液中亚硝酸盐(以亚硝酸钠计)的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

1000 ——转换系数；

$V$ ——测定用样液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f_{1}$ ——试样溶液稀释倍数；

$m_{1}$ ——试样质量，单位为克 $(\mathrm{g})$   
$V_{1}$ ——测定用样液体积，单位为毫升 $(\mathrm{mL})$ ；  
$V_{0}$ ——试样处理液总体积，单位为毫升 $(\mathrm{mL})$ 。

结果大于或等于 $1.00\mathrm{mg / kg}$ 时，保留3位有效数字；结果小于 $1.00\mathrm{mg / kg}$ 时，保留2位有效数字。

# 13.2 硝酸盐含量的计算

硝酸盐（以硝酸钠计）的含量按式(3)进行计算：

$$
X _ {2} = \left(\frac {\rho_ {2} \times 1 0 0 0 \times V \times f _ {2}}{m _ {2} \times \frac {V _ {3}}{V _ {2}} \times \frac {V _ {5}}{V _ {4}} \times 1 0 0 0} - X _ {1}\right) \times 1. 2 3 2 \quad \dots \tag {3}
$$

式中：

$X_{2}$ ——试样中硝酸盐(以硝酸钠计)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；  
$\rho_{2}$ ——经镉柱还原后测得亚硝酸盐(以亚硝酸钠计)总量的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

1000 ——转换系数；

$V$ ——测定用样液定容体积，单位为毫升 $(\mathrm{mL})$ ；

$f_{2}$ ——试样溶液稀释倍数；  
$m_{2}$ ——试样质量，单位为克 $(\mathrm{g})$ ；  
$V_{3}$ 测亚硝酸盐总量的测定用样液体积，单位为毫升 $(\mathrm{mL})$   
$V_{2}$ ——试样处理液总体积，单位为毫升 $(\mathrm{mL})$ ；  
$V_{5}$ ——经镉柱还原后样液的测定用体积，单位为毫升 $(\mathrm{mL})$ ；  
$V_{4}$ ——经镉柱还原后样液总体积，单位为毫升 $(\mathrm{mL})$ ；  
$X_{1}$ ——由式(2)计算出的试样中亚硝酸盐(以亚硝酸钠计)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

1.232 ——亚硝酸钠换算成硝酸钠的系数。

结果大于或等于 $1.00\mathrm{mg / kg}$ 时，保留3位有效数字；结果小于 $1.00\mathrm{mg / kg}$ 时，保留2位有效数字。

# 第三法 连续流动分析-分光光度法

# 16 原理

试样经沉淀蛋白质、除去脂肪后，在弱酸性条件下，试样溶液经连续流动分析仪，通过膜分离器分离，亚硝酸盐与磺胺重氮化反应后，再与盐酸萘乙二胺偶合形成紫红色化合物，在 $540~\mathrm{nm}$ 处比色测得亚硝酸盐含量；硝酸盐通过镉(如镉柱或镉圈等)还原成亚硝酸盐，进一步测得亚硝酸盐总量，由测得的亚硝酸盐总量减去试样中亚硝酸盐含量，即得试样中硝酸盐含量。

# 20.1 亚硝酸盐含量计算

亚硝酸盐（以亚硝酸钠计）的含量按式(4)进行计算：

$$
X _ {1} = \frac {\left(\rho_ {1} - \rho_ {0}\right) \times V _ {1} \times f _ {1} \times 1 0 0 0}{m _ {1} \times 1 0 0 0} \tag {4}
$$

式中：

$X_{1}$ ——试样中亚硝酸盐(以亚硝酸钠计)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho_{1}$ ——测定用试样溶液中亚硝酸盐(以亚硝酸钠计)的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_0$ ——空白液中亚硝酸盐(以亚硝酸钠计)的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V_{1}$ ——试样溶液体积，单位为毫升 $(\mathrm{mL})$ ；

$f_{1}$ ——试样溶液稀释倍数；

1000 ——换算系数；

$m_{1}$ ——试样质量，单位为克 $(\mathrm{g})$ 。

结果大于或等于 $1.00\mathrm{mg / kg}$ 时，保留3位有效数字；结果小于 $1.00\mathrm{mg / kg}$ 时，保留2位有效数字。

# 20.2 硝酸盐含量的计算

硝酸盐（以硝酸钠计）的含量按式(5)进行计算：

$$
X _ {2} = \frac {\left(\rho_ {3} - \rho_ {2}\right) \times V _ {2} \times f _ {2} \times 1 0 0 0}{m _ {2} \times 1 0 0 0} - X _ {1} \times 1. 2 3 2 \dots \tag {5}
$$

式中：

$X_{2}$ ——试样中硝酸盐(以硝酸钠计)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho_{3}$ ——经镉还原后试样溶液中总硝酸盐(以硝酸钠计)的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$\rho_{2}$ ——经镉还原后试剂空白液中总硝酸盐(以硝酸钠计)的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V_{2}$ ——试样溶液体积，单位为毫升 $(\mathrm{mL})$

$f_{2}$ ——试样溶液稀释倍数；

1000 ——换算系数；

$m_{2}$ ——试样质量，单位为克 $(\mathrm{g})$

$X_{1}$ ——由式(4)计算出的试样中亚硝酸盐(以亚硝酸钠计)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ 。

1.232 ——亚硝酸钠换算成硝酸钠的系数。

结果大于或等于 $1.00\mathrm{mg / kg}$ 时，保留3位有效数字；结果小于 $1.00\mathrm{mg / kg}$ 时，保留2位有效数字。

# 第四法 蔬菜、水果中硝酸盐的测定

# 23 原理

用 $\mathrm{pH}9.6\sim 9.7$ 的氨缓冲液提取样品中硝酸根离子，同时加活性炭去除色素类物质，加沉淀剂去除蛋白质及其他干扰物质，利用硝酸根离子和亚硝酸根离子在紫外区 $219~\mathrm{nm}$ 处具有等吸收波长的特性，测定提取液的吸光度，其测得结果为硝酸盐和亚硝酸盐吸光度的总和。鉴于新鲜蔬菜、水果中亚硝酸盐含量甚微，可忽略不计，测定结果可作为硝酸盐的吸光度，再根据工作曲线得出相应的质量浓度，计算样品中硝酸盐的含量。

# 26 分析步骤

# 26.1 试样制备

选取一定数量有代表性的样品，先用自来水冲洗，再用水清洗干净，晾干表面水分，切碎，充分混匀，于组织捣碎机中匀浆(部分少汁样品可按一定质量比例加入等量水)，在匀浆中加1滴正辛醇消除泡沫。

# 26.2 提取

称取 $10\mathrm{g}$ （精确至 $0.01\mathrm{g}$ )匀浆试样(如制备过程中加水，应按加水量折算)于 $250~\mathrm{mL}$ 锥形瓶中，加水 $100~\mathrm{mL}$ ，加入 $5\mathrm{mL}$ 氨缓冲溶液 $(\mathrm{pH}9.6\sim 9.7)$ ， $2\mathrm{g}$ 粉末状活性炭。振荡（振荡频率为200次/min） $30\mathrm{min}$ 。定量转移至 $250~\mathrm{mL}$ 容量瓶中，加入 $2\mathrm{mL}150~\mathrm{g / L}$ 亚铁氰化钾溶液和 $2\mathrm{mL}300~\mathrm{g / L}$ 硫酸锌溶液，充分混匀，加水定容至刻度，摇匀，放置 $5\mathrm{min}$ ，上清液用定量滤纸过滤，滤液备用。同时做空白试验。

# 26.3 测定

根据试样中硝酸盐含量的高低，吸取上述滤液 $2\mathrm{mL}\sim 10\mathrm{mL}$ 于 $50~\mathrm{mL}$ 容量瓶中，加水定容至刻度，混匀。用 $1\mathrm{cm}$ 石英比色皿，于 $219~\mathrm{nm}$ 处测定吸光度。

# 26.4 标准曲线的制作

将标准曲线工作液用 $1\mathrm{cm}$ 石英比色皿，于 $219~\mathrm{nm}$ 处测定吸光度。以标准溶液质量浓度为横坐标，吸光度为纵坐标绘制工作曲线。

# 27 结果计算

硝酸盐（以硝酸钠计）的含量按式(6)计算：

$$
X = \frac {\rho \times V _ {1} \times V _ {3}}{m \times V _ {2}} \tag {6}
$$

式中：

$X$ ——试样中硝酸盐(以硝酸钠计)的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho$ ——由工作曲线获得的试样溶液中硝酸盐(以硝酸钠计)的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V_{1}$ ——提取液定容体积，单位为毫升 $(\mathrm{mL})$

$V_{3}$ —待测液定容体积，单位为毫升 $(\mathrm{mL})$

$m$ ——试样的质量，单位为克(g)；

$V_{2}$ ——吸取的滤液体积，单位为毫升 $(\mathrm{mL})$

1000——换算系数。

结果大于或等于 $1.00\mathrm{mg / kg}$ 时，保留3位有效数字；结果小于 $1.00\mathrm{mg / kg}$ 时，保留2位有效数字。

# 镉柱还原效率的测定

吸取 $20.0\mathrm{mL}$ 硝酸钠标准使用液(10.4.4)，加入 $5\mathrm{mL}$ 氨缓冲溶液稀释液，混匀后注入贮液漏斗，使流经镉柱还原，用1个 $100~\mathrm{mL}$ 的容量瓶收集洗提液。洗提液的流量应不超过 $6\mathrm{mL / min}$ ，在贮液杯将要排空时，用约 $15~\mathrm{mL}$ 水冲洗杯壁。冲洗水流尽后，再用 $15~\mathrm{mL}$ 水重复冲洗，第2次冲洗水也流尽后，将贮液杯灌满水，并使其以最大流量流过柱子。当容量瓶中的洗提液接近 $100~\mathrm{mL}$ 时，从柱子下取出容量瓶，用水定容至刻度，混匀。取 $10.0~\mathrm{mL}$ 还原后的溶液于 $50~\mathrm{mL}$ 比色管中，以下按12.4“吸取 $0\mathrm{mL},0.200\mathrm{mL},0.400\mathrm{mL},0.600\mathrm{mL},0.800\mathrm{mL},1.00\mathrm{mL}$ ”起操作，根据标准曲线计算测得结果，与加入量一致，还原效率应大于 $95\%$ 为符合要求。

还原效率计算按式(C.1)计算：

$$
X _ {1} = \frac {\rho_ {1} \times 85}{10 \times 69} \times V \times 100 \% \tag{C.1}
$$

式中：

$X_{1}$ ——还原效率， $\%$ ；  
$\rho_{1}$ ——测得亚硝酸盐(以亚硝酸钠计)的质量浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；  
85 —硝酸钠的相对分子质量；  
10 ——测定用溶液相当硝酸钠的含量，单位为微克 $(\mu \mathrm{g})$ ；  
69 —亚硝酸钠的相对分子质量；  
$V$ ——测定用样液定容体积，单位为毫升 $(\mathrm{mL})$ 。

注：如果还原率小于 $95\%$ ，将镉柱中的镉粒倒入锥形瓶中，加入足量的盐酸 $(2\mathrm{mol} / \mathrm{L})$ ，振荡数分钟，再用水反复冲洗。

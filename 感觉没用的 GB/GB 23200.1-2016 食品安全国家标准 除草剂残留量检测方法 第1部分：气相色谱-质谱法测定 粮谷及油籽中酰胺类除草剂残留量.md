# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T1737.1—2006《除草剂残留量检验方法第1部分：气相色谱串联质谱法测定粮谷及油籽中酰胺类除草剂残留量》。与SN/T1737.1—2006相比，主要变化如下：

——标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“进出口食品”改为“食品”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

# 1 范围

本标准规定了粮谷及油籽中酰胺类除草剂残留量检验的抽样、制样和气相色谱一质谱联用测定方法。

本标准适用于大米、大豆中酰胺类除草剂残留量的定量测定，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB2763 食品安全国家标准 食品中农药最大残留限量

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样中除草剂用丙酮和水提取，把提取液中丙酮减压去除后，加入氯化钠溶液，用正己烷反萃取，浓缩正己烷提取液，然后用乙腈提取，弗罗里硅土固相萃取柱净化，样液供气相色谱一质谱测试，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：色谱纯。

4.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。

4.1.3 丙酮 $\mathrm{(CH_3COCH_3)}$ ：色谱纯。

4.1.4 乙醚 $\left[\left(\mathrm{C}_{2} \mathrm{H}_{5}\right)_{2} \mathrm{O}\right]$ ：色谱纯。

4.1.5 无水硫酸钠 $\mathrm{(Na_2SO_4)}$ ：经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，冷却后置于干燥器中备用。

4.1.6 氯化钠(NaCl)：分析纯。

# 4.2 溶液配制

4.2.1 氯化钠溶液 $(10\%)$ ：将 $100\mathrm{g}$ 氯化钠溶于水中，并稀释至 $1000~\mathrm{mL}$ 。

4.2.2 提取剂I：乙腈加入少量正己烷饱和，摇匀。

4.2.3 提取剂Ⅱ：正己烷加入少量乙腈饱和，摇匀。

4.2.4 正己烷-乙醚 $(85 + 15)$ ：取 $85\mathrm{mL}$ 正己烷和 $15\mathrm{mL}$ 乙醚，混匀。

# 4.3 标准品

毒草胺、莠去津、乙草胺、二甲吩草胺、甲草胺、嗪草酮、异丙甲草胺、敌稗、丁草胺、丙草胺、敌草胺标准品（具体信息参见附录A），纯度大于 $98\%$ 。

# 4.4 标准溶液配制

分别准确称取 $(25\pm 0.1)\mathrm{mg}$ 标准品于 $50~\mathrm{mL}$ 容量瓶中，用丙酮溶解并定容，得到浓度为

$500 \mu \mathrm{g} / \mathrm{mL}$ 单标储备液, 此溶液在 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 中可保存 3 个月。根据需要再用丙酮稀释储备液, 配制成适当浓度的混合标准工作液, 此溶液在 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 中可保存 1 个月。

# 4.5 材料

弗罗里硅土固相萃取柱： $3\mathrm{mL} / 125\mathrm{mg}$ ，或相当者。使用前依次用 $5\mathrm{mL}$ 正己烷-乙醚溶液和 $5\mathrm{mL}$ 正己烷预淋洗柱子，流速1滴/s。

# 5 仪器和设备

5.1 气相色谱—质谱仪。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 旋转蒸发仪。  
5.4 固相萃取装置。  
5.5 吹氮浓缩仪。  
5.6 涡旋振荡器。  
5.7 均质机。  
5.8 离心机。  
5.9 茄形瓶： $100~\mathrm{mL},250~\mathrm{mL}$   
5.10 离心管： $15\mathrm{mL},50\mathrm{mL}$   
5.11 微量注射器： $10\mu \mathrm{L}$

# 6 试样制备与保存

# 6.1 试样制备

将样品按四分法缩分至 $1\mathrm{kg}$ , 样品取样部位按GB2763的规定执行。全部磨碎并通过40目筛, 混匀, 均分成2份, 装入洁净的容器内, 密封, 标明标记。

# 6.2 试样保存

将试样于 $-5^{\circ} \mathrm{C}$ 以下避光保存。在抽样和制样过程中, 必须防止样品受到污染或发生残留物含量的变化。

# 7 分析步骤

# 7.1 提取

称取约 $10 \mathrm{~g}$ (精确至 $0.01 \mathrm{~g}$ ) 样品于 $50 \mathrm{~mL}$ 离心管中, 加入 $10 \mathrm{~mL}$ 水和 $20 \mathrm{~mL}$ 丙酮, 均质 $3 \mathrm{~min}$ , 于 $4000 \mathrm{r} / \mathrm{min}$ 离心 $4 \mathrm{~min}$ , 将提取液移入 $250 \mathrm{~mL}$ 茄形瓶中, 离心管中残渣再用 $30 \mathrm{~mL} \times 2$ 丙酮提取, 提取液并入茄形瓶中。在 $38^{\circ} \mathrm{C}$ 下, 减压蒸发去除丙酮, 残液 (约 $10 \mathrm{~mL}$ ) 转移至另一 $50 \mathrm{~mL}$ 离心管中。依次用 $10 \mathrm{~mL} 10 \%$ 氯化钠溶液和 $15 \mathrm{~mL}$ 正己烷洗涤茄形瓶, 洗涤液一并移入离心管中, 振荡 $3 \mathrm{~min}$ , 于 $2500 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ , 收集正己烷相。离心管中的水相中再用 $15 \mathrm{~mL} \times 2$ 正己烷提取, 合并正己烷相。

# 7.2 液一液分配净化

正己烷相中加入适量无水硫酸钠脱水, 将正己烷相完全转移至另一 $250 \mathrm{~mL}$ 茄形瓶中, 于 $50^{\circ} \mathrm{C}$ , 减压蒸发至干。残余物分 2 次, 每次用 $5 \mathrm{~mL}$ 提取液Ⅱ溶解, 一并转移至 $50 \mathrm{~mL}$ 离心管中。分 3 次, 每次加入 $10 \mathrm{~mL}$ 提取液I, 混匀、分层, 乙腈相转移至另一 $50 \mathrm{~mL}$ 离心管中。加入 $10 \mathrm{~mL}$ 提取液Ⅱ, 混匀、分层后弃去正己烷相, 乙腈相转移至 $100 \mathrm{~mL}$ 茄形瓶中, 在 $50^{\circ} \mathrm{C}$ 下, 减压蒸发至干, 残余物用 $5 \mathrm{~mL}$ 正己烷溶解。

# 7.3 固相萃取净化

将上述正己烷溶液移入弗罗里硅土固相萃取柱中，液体过柱流速保持0.5滴/s，用 $15\mathrm{mL}$ 正己烷一

乙醚溶液润洗茄形瓶并转移至柱中进行洗脱，流速为1滴/s，收集全部洗脱液于定量试管中，于 $40^{\circ}\mathrm{C}$ ，在氮气流下吹至近干，加入正己烷溶解残渣并定容至 $1.0\mathrm{mL}$ ，供GC-MS测定。

# 7.4 测定

# 7.4.1 气相色谱参考条件

a）色谱柱：HP-1701MS, $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径)，膜厚 $0.25\mu \mathrm{m}$ ，或相当者。  
b) 载气：氦气（纯度 $>99.999\%$ ），流速： $1.0 \mathrm{~mL} / \mathrm{min}$ 。  
c）色谱柱程序升温条件：初始温度： $70^{\circ}\mathrm{C}$ （保持 $1\mathrm{min}$ ）； $15^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $160^{\circ}\mathrm{C}$ （保持 $1\mathrm{min}$ ）； $2^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $200^{\circ}\mathrm{C}$ （保持 $2\mathrm{min}$ ）； $20^{\circ}\mathrm{C} / \mathrm{min}$ 升温至 $280^{\circ}\mathrm{C}$ （保持 $8\mathrm{min}$ ）。  
d）进样口温度： $270^{\circ}C$ □  
e）进样方式：不分流进样， $1\mathrm{min}$ 后打开分流阀，分流比 $100:1$   
f）进样量： $1\mu \mathrm{L}$

# 7.4.2 质谱参考条件

a）离子源温度： $230^{\circ}\mathrm{C}$ 。  
b）传输线温度： $280^{\circ}\mathrm{C}$ 。  
c) 离子化模式：EI。  
d）扫描范围： $50\mathrm{amu}\sim 400\mathrm{amu}$   
e）电子倍增管电压：自动调谐电压 $200\mathrm{V}$   
f）测试方式：选择离子监测。

# 7.4.3 色谱测定与确证

# 7.4.3.1 定量测定

根据样液中除草剂含量情况，选定峰面积相近的标准工作溶液，标准工作溶液和样液中除草剂的响应值均应在仪器检测的线性范围内。以单点或多点外标法定量，标准工作液和样液应等体积参插进样测定。

# 7.4.3.2 定性测定

a）定性测定的两个依据为：

——被测样品峰与标准样品峰的色谱保留时间相同；

被测样品与标准样品的质谱图相似, 被测样品的监测离子的相对丰度应与标准样品一致,二者之差不大于 $\pm 10\%$ (EI 模式)。各除草剂的监测离子及定量离子参见附录 A。

b）在上述色谱、质谱条件下，毒草胺、莠去津、乙草胺、二甲吩草胺、甲草胺、嗪草酮、异丙甲草胺、敌稗、丁草胺、丙草胺和敌草胺的保留时间分别为 $11.2\mathrm{min}$ 、 $13.2\mathrm{min}$ 、 $13.8\mathrm{min}$ 、 $14.9\mathrm{min}$ 、 $15.3\mathrm{min}$ 、 $15.7\mathrm{min}$ 、 $15.9\mathrm{min}$ 、 $16.9\mathrm{min}$ 、 $19.3\mathrm{min}$ 、 $20.6\mathrm{min}$ 和 $21.9\mathrm{min}$ 。选择离子色谱图参见附录B,质谱图参见图C.1～图C.4。

# 7.5 空白实验

除不加试样外，均按 $7.1\sim 7.4$ 的规定执行。

# 8 结果计算和表述

用色谱工作站或式(1)计算试样中除草剂的含量。

$$
X = \frac {A \times c \times V}{A _ {s} \times m}. \tag {1}
$$

式中：

$X$ ——试样中除草剂含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A$ ——试样中除草剂峰面积；

$A_{s}$ ——标准溶液中除草剂峰面积；

$c$ ——标准溶液浓度，单位为毫克每升 $(\mathrm{mg / L})$ ；

$V$ ——样品溶液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——样品称重，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录D的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 E 的要求。

# 10 定量限和回收率

# 10.1 定量限

毒草胺、莠去津、乙草胺、异丙甲草胺、丙草胺、敌草胺、二甲吩草胺、嗪草酮、敌稗的定量限为0.02 mg/kg；甲草胺、丁草胺的定量限为0.05 mg/kg。

# 10.2 回收率

10.2.1 大米中除草剂添加浓度及其回收率

a）毒草胺添加浓度为 $0.02\mathrm{mg / kg}\sim 2.0\mathrm{mg / kg}$ 范围内，回收率为 $90.3\% \sim 94.5\%$   
b）莠去津添加浓度为 $0.02\mathrm{mg / kg}\sim 2.0\mathrm{mg / kg}$ 范围内，回收率为 $89.8\% \sim 95.5\%$   
c）乙草胺添加浓度为 $0.02\mathrm{mg / kg}\sim 2.0\mathrm{mg / kg}$ 范围内，回收率为 $84.5\% \sim 93.5\%$   
d）二甲吩草胺添加浓度为 $0.02\mathrm{mg / kg}\sim 2.0\mathrm{mg / kg}$ 范围内，回收率为 $84.5\% \sim 93.8\%$   
e) 甲草胺添加浓度为 $0.05 \mathrm{mg} / \mathrm{kg} \sim 2.0 \mathrm{mg} / \mathrm{kg}$ 范围内，回收率为 $83.7\% \sim 93.9\%$   
f) 嗪草酮添加浓度为 $0.02 \mathrm{mg} / \mathrm{kg} \sim 2.0 \mathrm{mg} / \mathrm{kg}$ 范围内, 回收率为 $88.3\% \sim 95.6\%$ ;

10.2.2 大豆中除草剂添加浓度及其回收率

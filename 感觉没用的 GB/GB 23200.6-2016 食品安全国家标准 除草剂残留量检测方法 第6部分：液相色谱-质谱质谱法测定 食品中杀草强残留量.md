# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN1737.6—2010《进出口食品中杀草强残留量的检测方法液相色谱一质谱/质谱法》。与SN1737.6—2010相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
标准名称中“进出口食品”改为“食品”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN 1737.6—2010。

# 1 范围

本标准规定了食品中杀草强残留量的液相色谱—串联质谱(LC-MS-MS)测定方法。

本标准适用于苹果、菠萝、菠菜、胡萝卜、紫苏叶、金银花、姜粉、花椒粉、茶叶、小麦、玉米、花生、肉、鱼及动物肝脏中杀草强残留量的检测和确证；其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

# 3 原理

样品经提取后，用PCX固相萃取柱或ENVI-Carb固相萃取柱净化，液相色谱一质谱/质谱法测定，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为色谱纯，水为符合GB/T6682中规定的一级水。

# 4.1 试剂

4.1.1 冰乙酸 $\mathrm{(C_2H_4O_2)}$ 。  
4.1.2 氨水 $(\mathrm{NH_4OH})$ 。  
4.1.3 二氯甲烷 $\mathrm{(CH_2Cl_2)}$   
4.1.4 丙酮 $(\mathrm{C}_3\mathrm{H}_6\mathrm{O})$ 。

# 4.2 溶液配制

4.2.1 $1\%$ 乙酸溶液：吸取 $10~\mathrm{mL}$ 冰乙酸，用超纯水定容至 $1\mathrm{L}$   
4.2.2 $25\%$ 丙酮一水溶液：量取 $250~\mathrm{mL}$ 丙酮，用超纯水定容至 $1\mathrm{L}$   
4.2.3 $1\%$ 乙酸一丙酮一水溶液：吸取 $10~\mathrm{mL}$ 冰乙酸，用丙酮水溶液定容至 $1\mathrm{L}$   
4.2.4 $5\%$ 氨化甲醇溶液：吸取 $5\mathrm{mL}$ 氨水，用甲醇定容至 $100~\mathrm{mL}$

# 4.3 标准品

杀草强标准物质：Amitrole, $\mathrm{C_2H_4N_4}$ ，分子量：84.08，CAS号：61-82-5，纯度 $\geqslant 99.9\%$ 。

# 4.4 标准溶液配制

4.4.1 杀草强标准储备液：称取适量杀草强标准品(精确至 $0.1\mathrm{mg}$ )于 $50~\mathrm{mL}$ 容量瓶中，用乙腈配制成 $1.0\mathrm{mg / mL}$ 的标准储备液。 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 保存。  
4.4.2 杀草强标准工作液：根据需要用流动相将储备液稀释配制成适用浓度的标准工作液。 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存。

# 4.5 材料

4.5.1PCX固相萃取柱： $3\mathrm{mL} / 60\mathrm{mg}$ ，或相当者(混合型阳离子交换小柱)。  
4.5.2 ENVI-Carb固相萃取柱： $6\mathrm{mL} / 500\mathrm{mg}$ ，或相当者(石墨化非多孔碳)。  
4.5.3 滤膜： $0.2\mu \mathrm{m}$

# 5 仪器和设备

5.1 液相色谱—质谱/质谱联用仪：配备电喷雾离子源(ESI)。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 涡旋振荡器。  
5.4 旋转蒸发仪。  
5.5 高速均质器。  
5.6 离心机。  
5.7 振荡器。  
5.8 粮谷粉碎机。  
5.9 食品捣碎机。  
5.10 花生粉碎机。  
5.11 离心管： $100\mathrm{mL},50\mathrm{mL}$   
5.12 容量瓶： $100\mathrm{ml}$

# 6 试样制备与保存

# 6.1 试样制备

# 6.1.1 取样部位

样品取样部位按GB2763的规定执行。在抽样及制样的操作过程中，应防止样品受到污染或发生残留物含量的变化。

# 6.1.2 小麦、玉米、金银花、茶叶

取有代表性样品约 $500\mathrm{g}$ , 用粉碎机全部粉碎并通过 $2.0\mathrm{mm}$ 圆孔筛。混匀, 装入洁净的容器内, 密闭, 标明标记。

# 6.1.3 水果、蔬菜、鱼、肉、肝脏

取有代表性样品约 $500 \mathrm{~g}$ , 切碎后, 用食品捣碎机将样品加工成浆状。混匀, 装入洁净的容器内, 密闭, 标明标记。

# 6.1.4 姜粉、花椒粉

取有代表性样品约 $100\mathrm{g}$ ，充分混合均匀，过 $2.0\mathrm{mm}$ 圆孔筛，装入洁净容器内，密闭，标明标记。

# 6.1.5 花生

取有代表性样品 $500\mathrm{g}$ ，用磨碎机全部磨碎。混匀，装入洁净的容器内，密闭，标明标记。

# 6.2 试样保存

小麦、玉米、花生、茶叶、姜粉、花椒粉试样于 $0^{\circ}\mathrm{C} \sim 4^{\circ}\mathrm{C}$ 保存；鱼、肉、肝脏及水果和蔬菜类试样于 $-18^{\circ}\mathrm{C}$ 以下冷冻保存。

# 7 分析步骤

# 7.1 提取

# 7.1.1 苹果、菠萝、菠菜、胡萝卜、紫苏叶

称取 $10\mathrm{g}$ 试样(精确至 $0.1\mathrm{g}$ ) 于 $100\mathrm{mL}$ 离心管中, 加入 $20\mathrm{mL}1\%$ 乙酸溶液和 $20\mathrm{mL}$ 二氯甲烷, $20000\mathrm{r / min}$ 均质 $1\mathrm{min}$ , 于 $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ , 上清液转入 $100\mathrm{mL}$ 容量瓶中, 弃去下层二氯甲烷层, 残渣用 $1\%$ 的乙酸溶液 $20\mathrm{mL}$ 和二氯甲烷 $20\mathrm{mL}$ 重复提取一次; 合并上清液, 用 $1\%$ 的乙酸溶液定容至刻度, 移取 $10\mathrm{mL}$ 提取液待净化。

# 7.1.2 金银花、姜粉、花椒粉、茶叶

称取 $1\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ ) 于 $50~\mathrm{mL}$ 离心管中，加 $20~\mathrm{mL}~1\%$ 乙酸水溶液， $20000~\mathrm{r / min}$ 均质 $1\mathrm{min}, 5000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液过滤至 $125~\mathrm{mL}$ 分液漏斗中，残渣再用 $10~\mathrm{mL}~1\%$ 乙酸水溶液重复提取一次，合并上清液，加入 $10~\mathrm{mL}$ 二氯甲烷，振荡 $1\mathrm{min}$ ，静置分层，弃去下层二氯甲烷层。待净化。

# 7.1.3 玉米、花生

称取 $2\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ ) 于 $50~\mathrm{mL}$ 离心管中，加 $20~\mathrm{mL}~1\%$ 乙酸一丙酮水溶液， $20000~\mathrm{r / min}$ 均质 $1\mathrm{min}, 5000~\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，上清液过滤至 $125~\mathrm{mL}$ 分液漏斗中，残渣再用 $10~\mathrm{mL}~1\%$ 乙酸一丙酮水溶液重复提取一次，合并上清液，加入 $10~\mathrm{mL}$ 二氯甲烷，振荡 $1\mathrm{min}$ ，静置分层，弃取下层二氯甲烷层。待净化。

# 7.1.4 小麦、鱼、肉、肝脏

小麦、鱼、肉、称取 $2\mathrm{g}$ 试样，肝脏称取 $1\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ）于 $50\mathrm{mL}$ 离心管中，加 $20\mathrm{mL}$ 丙酮水溶液， $20000\mathrm{r / min}$ 均质 $1\mathrm{min}$ ， $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，提取液过滤至 $125\mathrm{mL}$ 分液漏斗中，残渣用 $10\mathrm{mL}$ 丙酮水溶液重复提取一次，合并上清液，加入 $10\mathrm{mL}$ 二氯甲烷，振荡 $1\mathrm{min}$ ，静置分层，弃去下层二氯甲烷层。提取液中加入 $0.25\mathrm{mL}$ 冰乙酸酸化，待净化。

# 7.2 净化

# 7.2.1 苹果、菠萝、菠菜、胡萝卜、紫苏叶、玉米、花生、小麦、鱼、肉、猪肝

依次用 $3\mathrm{mL}$ 甲醇和 $5\mathrm{mL}$ 水预淋洗PCX小柱，弃去流出液。注入提取液，再分别用 $3\mathrm{mL}$ 水、 $3\mathrm{mL}$ 甲醇淋洗，弃去流出液。用 $2\mathrm{mL}5\%$ 氨化甲醇溶液洗脱，收集洗脱液， $40^{\circ}\mathrm{C}$ 旋转浓缩至近干，残留物用 $1.0\mathrm{mL}$ 流动相溶解，过 $0.2\mu \mathrm{m}$ 滤膜，供液相色谱一质谱/质谱测定。

# 7.2.2 金银花、姜粉、花椒粉、茶叶

7.2.2.1 依次用 $3\mathrm{mL}$ 甲醇、 $3\mathrm{mL}$ 水预淋洗Envi-Carb，弃去流出液，注入提取液，收集流出液。  
7.2.2.2 依次用 $3 \mathrm{~mL}$ 甲醇、 $5 \mathrm{~mL}$ 水预淋洗PCX小柱，弃去流出液。将Envi-Carb净化后的样液转至PCX净化柱，再分别用 $3 \mathrm{~mL}$ 水、 $3 \mathrm{~mL}$ 甲醇淋洗，用 $2 \mathrm{~mL} 5 \%$ 氨化甲醇溶液洗脱，收集洗脱液，于 $40^{\circ} \mathrm{C}$ 旋转浓缩至近干，残留物用 $1.0 \mathrm{~mL}$ 流动相溶解，过 $0.2 \mu \mathrm{m}$ 滤膜，供液相色谱一质谱/质谱测定。

# 7.3.2 色谱测定与确证

根据样液中被测物含量，选定浓度相近的标准工作溶液，对标准工作溶液与样液等体积参插进样测定，标准工作溶液和待测样液中呋虫胺的响应值均应在仪器检测的线性范围内。

# 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用色谱数据处理机或按式(1)计算样品中杀草强残留量。

$$
X = \frac {A \times c \times V}{A _ {s} \times m}. \tag {1}
$$

式中：

$X$ ——试样中杀草强残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

A ——样品溶液中杀草强的峰面积；

$c$ ——杀草强标定工作液的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

V ——样品溶液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——杀草强标准工作溶液的峰面积；

$m$ ——最终样液代表的试样质量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 C 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 D 的要求。

# 10 定量限和回收率

# 10.1 定量限

本方法在苹果、菠萝、菠菜、胡萝卜、紫苏叶、玉米、花生、小麦、姜、鱼、肉、肝脏的定量限为 $0.01\mathrm{mg / kg}$ ，在茶叶、金银花、花椒的定量限为 $0.02\mathrm{mg / kg}$ 。

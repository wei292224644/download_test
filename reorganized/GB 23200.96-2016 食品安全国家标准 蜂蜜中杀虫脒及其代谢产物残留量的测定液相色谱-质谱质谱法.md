# 前言
本标准按照GB/T1.1—2009给出的规则起草。
本标准代替SN/T0213.1—2011《出口蜂蜜中杀虫脒及其代谢产物残留量的测定 第1部分：液相色谱一质谱/质谱法》。与SN/T0213.1—2011相比，主要变化如下：
标准文本格式修改为食品安全国家标准文本格式；
——标准名称“出口蜂蜜”改为“蜂蜜”；
——标准范围中增加“其他食品可参照执行”。
本标准所代替标准的历次版本发布情况为：
—SN/T0213.1—1993、SN/T0213.1—2011。
# 1 范围
本标准规定了蜂蜜中杀虫脒及其代谢物(4-氯邻甲苯胺)的液相色谱一质谱/质谱检测方法。
本标准适用于蜂蜜(洋槐蜜、荆条蜜、蜂巢蜜、杂花蜜、野蜂蜜等)中杀虫脒及其代谢物(4-氯邻甲苯胺)残留量的液相色谱一质谱/质谱测定和确证,其他食品可参照执行。

# 3 原理
试样用氢氧化钠水溶液稀释溶解，经HLB固相萃取柱净化，液相色谱一串联质谱仪测定，外标法定量。
# 4 试剂和材料
除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。
### 4.1 试剂
4.1.1 乙腈 $\mathrm{(CH_3CN)}$ ，高效液相色谱纯。  
4.1.2 甲醇 $\mathrm{(CH_3OH)}$ ，高效液相色谱纯。  
4.1.3 氢氧化钠(NaOH)。  
4.1.4 硫酸 $(\mathrm{H}_2\mathrm{SO}_4)$ 。  
4.1.5 甲酸（HCOOH），纯度 $\geqslant 99\%$
### 4.2 溶液配制
4.2.1 $0.02\mathrm{mol} / \mathrm{L}$ 氢氧化钠溶液：溶解 $800\mathrm{mg}$ 氢氧化钠于 $1\mathrm{L}$ 水中，使用期为1个月。  
4.2.2 $1 \mathrm{~mol} / \mathrm{L}$ 硫酸溶液: 于适量水中缓慢移入 $54.3 \mathrm{~mL}$ 浓硫酸, 边溶解边搅拌, 用水稀释至 $1 \mathrm{~L}$ , 使用期为 3 个月。  
4.2.3 $10\mathrm{mmol / L}$ 硫酸溶液：于适量水中缓慢移入 $10~\mathrm{mL}~1~\mathrm{mol / L}$ 硫酸，边溶解边搅拌，用水稀释至 $1\mathrm{L}$ ，使用期为1个月。  
4.2.4 乙腈水溶液：乙腈：水 $(3 + 7$ ，体积比）。  
4.2.5 $0.1\%$ 甲酸溶液： $1\mathrm{mL}$ 甲酸溶解于水中，并定容至 $1\mathrm{L}$ 。
### 4.3 标准品
杀虫脒标准品（chloridimeform， $\mathrm{C_{10}H_{13}ClN_2}$ ，CAS号：6164-98-3：纯度 $\geq 99\%$ ）和4-氯邻甲苯胺标准品（4-chloro-o-toluidine， $\mathrm{C_7H_8ClN}$ ，CAS号：95-69-2：纯度 $\geq 99\%$ ）。
### 4.4 标准溶液配制
4.4.1 标准储备液的配制：准确称取适量的杀虫脒和4-氯邻甲苯胺标准品，用乙腈配制成浓度为1.0 $\mathrm{mg / mL}$ 的标准储备溶液。该溶液在一 $18^{\circ}C$ 冰箱中保存。有效期为12个月。  
4.4.2 标准中间溶液的配制：用乙腈分别稀释标准储备液至终浓度约为 $1.0\mu \mathrm{g / mL}$ ，低于 $4^{\circ}\mathrm{C}$ 避光冷藏保存。有效期为6个月。  
4.4.3 基质标准工作溶液的配制：根据需要，临用时吸取一定量的标准中间溶液，用基质空白溶液配制成适当浓度的混合标准工作溶液。低于 $4^{\circ}\mathrm{C}$ 避光冷藏保存，现用现配。
### 4.5 材料
4.5.1 HLB固相萃取小柱(亲水亲脂平衡柱): $60 \mathrm{mg}$ (填料: 聚苯乙烯一二乙烯基苯一吡咯烷酮)/3 mL或相当者。使用前, 依次用 $3 \mathrm{~mL}$ 甲醇、 $3 \mathrm{~mL}$ 水活化。  
4.5.2 微孔滤膜： $0.22\mu \mathrm{m}$ ，有机相。
# 5 仪器和设备
5.1 液相色谱—串联质谱仪：配有电喷雾离子源（ESI）。  
5.2 电子天平：感量分别为 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 涡旋振荡器。  
5.4 固相萃取装置。  
5.5 氮吹仪。  
5.6 离心管： $15\mathrm{mI}$   
5.7 玻璃试管： $10~\mathrm{mL}$ ，具刻度。  
5.8 恒温水浴锅。
# 6 试样制备与保存
### 6.1 试样制备
取代表性蜂蜜样品约 $500 \mathrm{~g}$ , 取样部位按 GB 2763 的规定执行。对无结晶的蜂蜜样品, 将其搅拌均匀; 对有结晶析出的蜂蜜样品, 在密闭情况下, 将样品瓶置于不超过 $60^{\circ} \mathrm{C}$ 的水浴中温热、振荡, 待样品全部融化后搅匀, 迅速冷却至室温。在融化时, 必须防止水分挥发。装入洁净容器, 密封, 标明标记。在制样的操作过程中, 应防止样品受到污染或发生残留物含量的变化。
### 6.2 试样保存
试样于常温状态下保存。
# 7 分析步骤
### 7.1 提取
称取试样 $1\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ）于 $15\mathrm{mL}$ 离心管中，加入 $10\mathrm{mL}0.02\mathrm{mol / L}$ 氢氧化钠溶液，在2000r/min下涡旋振荡，混匀。
### 7.2 净化
将上述提取液转移至 HLB 固相萃取柱中, 再加入 $3 \mathrm{~mL}$ 水洗涤液离心管, 过柱。用 $3 \mathrm{~mL}$ 水和 $1 \mathrm{~mL} 10 \mathrm{mmol} / \mathrm{L}$ 硫酸溶液淋洗小柱, 弃去淋出液, 抽干。用 $2 \mathrm{~mL}$ 乙腈洗脱, 收集于 $10 \mathrm{~mL}$ 试管中, 洗脱液在室温下氮吹近干, 准确加入 $1.0 \mathrm{~mL}$ 乙腈: 水 $(3 + 7$ , 体积比), 振荡溶解, 过 $0.22 \mu \mathrm{m}$ 滤膜, 供测定。

### 7.4 空白实验
除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。
# 8 结果计算和表述
用色谱数据处理机或按式(1)计算试样中杀虫脒或4-氯邻甲苯胺残留量。
$$
X _ { i } = \frac { A _ { i } \times C _ { s i } \times V } { A _ { s i } \times m } \tag { 1 }
$$
式中：
$X_{i}$ ——试样中杀虫脒或4-氯邻甲苯胺残留含量，单位为毫克每千克 $(\mathrm{mg / kg})$
$A_{i}$ ——样液中杀虫脒或4-氯邻甲苯胺的峰面积；
$C_{s}$ ——标准工作溶液中杀虫脒或4-氯邻甲苯胺的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$
$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；
$A_{si}$ ——标准工作溶液中杀虫脒或4-氯邻甲苯胺的峰面积；
$m$ ——最终样液代表的试样量，单位为克 $(\mathrm{g})$ 。
计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。
# 9 精密度
9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录C的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值(百分率), 应符合附录 D 的要求。
# 10 定量限和回收率
### 10.1 定量限
本方法的定量限为 $5\mu \mathrm{g} / \mathrm{kg}$
### 10.2 回收率
当添加水平为 $5 \mu \mathrm{g} / \mathrm{kg}, 10 \mu \mathrm{g} / \mathrm{kg}, 20 \mu \mathrm{g} / \mathrm{kg}$ 时, 杀虫脒和 4 -氯邻甲苯胺在不同基质中的添加回收率参见附录 E。

# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T1967—2007《进出口食品中异稻瘟净残留量的检测方法》。与SN/T1967—2007相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；

——标准名称和范围中“进出口食品”改为“食品”。

本标准所代替标准的历次版本发布情况为：

—SN/T1967—2007。

# 食品中异稻瘟净残留量的检测方法

# 1 范围

本标准规定了食品中异稻瘟净的气相色谱法和气相色谱一质谱法。

本标准适用于茶叶、菠菜、苹果、板栗、蜂蜜、食醋、大米、鸡肉、牛肉、鱼肉中异稻瘟净残留量的测定和确证。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB/T6682 分析实验室用水规格和试验方法。

# 3 方法提要

试样中残留的异稻瘟净采用丙酮和正己烷 $(1 + 2$ ，体积比)振荡提取，石墨化炭黑固相萃取柱或中性氧化铝固相萃取柱净化，洗脱液浓缩并定容后，供气相色谱仪和气相色谱一质谱仪测定和确证，外标法定量。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

4.1 正己烷 $\left(\mathrm{C}_{6} \mathrm{H}_{14}\right)$ ：重蒸馏。  
4.2 丙酮 $\mathrm{(CH_3COCH_3)}$ ：重蒸馏。  
4.3 氯化钠(NaCl)。  
4.4 无水硫酸钠 $(\mathrm{Na_2SO_4})$ ：经 $650^{\circ}\mathrm{C}$ 灼烧 $4\mathrm{h}$ ，置干燥器中备用。  
4.5 丙酮一正己烷 $(1 + 2$ ，体积比)溶液。  
4.6 丙酮一正己烷 $(1 + 1$ ，体积比)溶液。  
4.7 异稻瘟净标准物质(iprobenfos, $\mathrm{C_{13}H_{21}O_3PS}$ , CAS号: 26087-47-8): 纯度 $>99\%$ 。  
4.8 标准储备溶液：准确称取适量的异稻瘟净标准物质，用丙酮将配制成 $1000\mu \mathrm{g / mL}$ 标准储备液，再根据检测要求用正己烷稀释成相应的标准工作溶液。标准溶液避光于 $4^{\circ}C$ 保存。  
4.9 石墨化炭黑固相萃取柱： $3\mathrm{mL} / 125\mathrm{mg}$ ，或相当者。  
4.10 中性氧化铝固相萃取柱： $3\mathrm{mL} / 125\mathrm{mg}$ ，或相当者。

# 5 仪器和设备

5.1 气相色谱仪：配火焰光度检测器(FPD)。  
5.2 气相色谱—质谱联用仪，配电子轰击离子源（EI源）。  
5.3 组织捣碎机。  
5.4 粉碎机。  
5.5 涡旋振荡器。  
5.6 固相萃取装置，带真空泵。

5.7 氮吹仪。  
5.8 低速离心机： $3000\mathrm{r / min}$   
5.9 离心管： $15\mathrm{mL}$   
5.10 刻度试管： $15\mathrm{mL}$   
5.11 微量注射器： $10~\mu \mathrm{L}$

# 6 试样制备与保存

## 6.1 试样制备

### 6.1.1 水果或蔬菜

取有代表性样品 $500\mathrm{g}$ , 将其可食用部分切碎后 (不可用水洗涤), 用捣碎机将样品加工成浆状。混匀, 装入洁净的盛样容器内, 密封并标明标记。

### 6.1.2 茶叶及粮谷

取有代表性样品 $500\mathrm{g}$ , 用粉碎机粉碎并通过 $2.0\mathrm{mm}$ 圆孔筛。混匀, 装入洁净的盛样容器内, 密封并标明标记。

### 6.1.3 肉及肉制品

取有代表性样品 $500\mathrm{g}$ , 将其切碎后, 用捣碎机将样品加工成浆状, 混匀, 装入洁净的盛样容器内, 密封并标明标记。

## 6.2 试样保存

茶叶、蜂产品、调味品及粮谷类等试样于 $0^{\circ} \mathrm{C} \sim 4^{\circ} \mathrm{C}$ 保存; 水果蔬菜类和肉及肉制品类等试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。在制样的操作过程中, 应防止样品受到污染或发生残留物含量的变化。

# 7 测定步骤

## 7.1 提取

对于茶叶、板栗、蜂蜜、大米样品，称取 $1\mathrm{g}$ 均匀试样（精确至 $0.001\mathrm{g}$ ），对于菠菜、荞头、苹果、食醋、鸡肉、牛肉、鱼肉样品，称取 $2\mathrm{g}$ 均匀试样（精确至 $0.001\mathrm{g}$ ）。将称取的试样置于 $15\mathrm{mL}$ 离心管中，加入 $1\mathrm{g}$ 氯化钠，加入 $2\mathrm{mL}$ 蒸馏水，于混匀器上混匀 $30\mathrm{s}$ ，放置 $15\mathrm{min}$ 。加入 $3\mathrm{mL}$ 丙酮一正己烷混合液（4.5），在混匀器上混匀 $2\mathrm{min}$ 。 $2500\mathrm{r/min}$ 离心 $1\mathrm{min}$ ，吸取上层正己烷萃取液于另一试管中。再分别加入 $3\mathrm{mL}$ 丙酮一正己烷混合液（4.5）重复提取2次，合并提取液。

## 7.2 净化

7.2.1 茶叶、菠菜、苹果、板栗、蜂蜜样品：将石墨化炭黑固相萃取柱(柱内填约 $1 \mathrm{~cm}$ 高的无水硫酸钠层)安装在固相萃取的真空抽滤装置上，分3次，每次用 $1 \mathrm{~mL}$ 丙酮预淋洗萃取柱，再分3次，每次用 $1 \mathrm{~mL}$ 正己烷预淋洗萃取柱，弃去全部预淋洗液。将正己烷提取液加入到石墨化炭黑固相萃取柱中，待提取液全部流出后，再用 $3 \mathrm{~mL}$ 丙酮一正己烷混合液(4.6)洗脱萃取柱，保持流速 $1.5 \mathrm{~mL} / \mathrm{min}$ ，收集全部流出液，于 $45^{\circ} \mathrm{C}$ 下，氮气流吹至近干。最后用正己烷定容至 $0.5 \mathrm{~mL}$ ，上机备用。  
7.2.2 食醋、大米、鸡肉、牛肉、鱼肉样品：将中性氧化铝固相萃取柱(柱内填约 $1\mathrm{cm}$ 高的无水硫酸钠层)安装在固相萃取的真空抽滤装置上，先用 $3\mathrm{mL}$ 丙酮预淋洗萃取柱，再用 $3\mathrm{mL}$ 正己烷预淋洗萃取柱，弃去全部预淋洗液。将正己烷提取液加入到中性氧化铝固相萃取柱中，待提取液全部流出后，再用 $3\mathrm{mL}$ 丙酮一正己烷混合液(4.6)洗脱萃取柱，保持流速 $1.5\mathrm{mL} / \mathrm{min}$ ，收集全部流出液，于 $45^{\circ}\mathrm{C}$ 下，氮气流吹至近干。最后用正己烷定容至 $0.5\mathrm{mL}$ ，上机备用。

## 7.3 测定

### 7.3.1 气相色谱条件

a）色谱柱：EQUITYTM～1701毛细管柱， $30\mathrm{m}\times 0.32\mathrm{mm}$ （内径），膜厚 $1.0\mu \mathrm{m}$ ，或相当者；  
b）升温程序：初始温度 $100^{\circ}\mathrm{C}$ ，以 $10^{\circ}\mathrm{C / min}$ 升至 $220^{\circ}\mathrm{C}$ ，保持 $10\mathrm{min}$   
c）进样口温度： $250^{\circ}\mathrm{C}$   
d) 检测器温度: $250^{\circ} \mathrm{C}$ ;   
e）载气：氮气，纯度 $\geqslant 99.99\%$ ，流速 $5.0~\mathrm{mL / min}$   
f) 氢气: $75 \mathrm{~mL / min}$ ;   
g) 空气： $100~\mathrm{mL / min}$   
h) 尾吹气: $20 \mathrm{~mL / min}$ ;  
i）进样方式：无分流进样， $1.0\mathrm{min}$ 后开阀；  
j）进样量： $2\mu \mathrm{L}$

### 7.3.2 气相色谱一质谱条件

a）色谱柱：HP-5MS石英毛细管柱， $30\mathrm{m}\times 0.25\mathrm{mm}$ （内径），膜厚 $0.25\mu \mathrm{m}$ ，或相当者；  
b）色谱柱温度：初始温度 $80^{\circ}\mathrm{C}$ ，以 $7^{\circ}\mathrm{C / min}$ 升至 $205^{\circ}\mathrm{C}$ ，再以 $25^{\circ}\mathrm{C / min}$ 升至 $280^{\circ}\mathrm{C}$ ，保持 $5\mathrm{min}$   
c) 进样口温度: $280^{\circ} \mathrm{C}$ ;   
d） 色谱—质谱接口温度： $270^{\circ}C$   
e) 载气：氦气，纯度 $\geq 99.995\%$ ，流速 $0.8\mathrm{mL / min}$   
f） 进样量： $1\mu \mathrm{L}$   
g）进样方式：无分流进样， $1\mathrm{min}$ 后开阀；  
h）电离方式：EI；  
i）电离能量： $70\mathrm{eV}$   
j）检测方式：选择离子监测方式（SIM）；  
k）监测离子 $(m / z):203,204,246,288$ ；定量离子 $(m / z):204$   
1) 溶剂延迟： $10\mathrm{min}$

### 7.3.3 色谱测定与确证

7.3.3.1 根据样液中异稻瘟净含量情况, 选定与样液浓度相近的标准工作溶液。异稻瘟净标准工作溶液和样液中异稻瘟净响应值均应在仪器检测线性范围内。标准工作溶液和样液等体积穿插进样测定,以保留时间定性, 测量峰面积与标准工作液比较进行定量。在 $7.3.1 \sim 7.3.2$ 规定的条件下, 异稻瘟净标准物质的色谱图、总离子流图和质谱图见图 A.1~图 A.3。  
7.3.3.2 在相同实验条件下, 样品中待测物质的质量色谱保留时间与标准工作液相同, 并且在扣除背景后的样品质量色谱中, 所选离子均出现, 经过对比所选择离子的丰度比与标准品对应离子的丰度比, 其值在允许范围内 (允许范围见表 1) 则可判定样品中存在对应的待测物。在 7.3.21 规定的色谱条件下, 异稻瘟净的参考保留时间是 $11.90 \mathrm{~min}$ 。在 7.3.2 规定的色谱条件下, 异稻瘟净的参考保留时间是 $17.70 \mathrm{~min}$ , 其监测离子 $(m / z)$ 丰度比是 $203:204:246:288 = 17:100:15:19$ 。

## 7.4 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

## 7.5 结果计算和表述

用色谱数据处理机或按式(1)计算试样中异稻瘟净的含量。

$$
X = \frac {A \times C _ {s} \times V}{A _ {s} \times m}. \tag {1}
$$

式中：

$X$ ——试样中异稻瘟净的含量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$A$ ——样液中异稻瘟净的峰面积；

$C_s$ ——标准工作液中异稻瘟净的浓度，单位为微克每毫升 $(\mu \mathrm{g} / \mathrm{mL})$ ；

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$A_{s}$ ——标准工作液中异稻瘟净的峰面积；

$m$ ——最终样液所代表的试样量，单位为克 $(\mathrm{g})$ 。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位效数字。

# 8 测定低限、回收率

## 8.1 测定低限

气相色谱—质谱法测定低限为 $0.005\mathrm{mg / kg}$

# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T1920—2007《进出口动物源性食品中敌百虫、敌敌畏、蝇毒磷残留量的检测方法液相色谱一质谱/质谱法》。与SN/T1920—2007相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
——标准名称“进出口动物源性食品”改为“动物源性食品”；  
——标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T1920—2007。

# 食品安全国家标准 动物源性食品中敌百虫、敌敌畏、蝇毒磷残留量的测定液相色谱一质谱/质谱法

# 1 范围

本标准规定了畜、禽分割肉、盐渍肠衣和蜂蜜中敌百虫、敌敌畏、蝇毒磷残留量的制样和液相色谱—质谱/质谱检验方法。

本标准适用于分割肉、盐渍肠衣和蜂蜜中敌百虫、敌敌畏、蝇毒磷残留量的测定，其他食品可参照执行。

# 3 原理

试样中的敌百虫、敌敌畏、蝇毒磷用二氯甲烷或乙酸乙酯提取，提取液经浓缩、脱脂后，用液相色谱一质谱/质谱仪测定，外标峰面积法定量，子离子丰度比定性。

# 4 试剂和材料

除另有规定外，所有试剂均为分析纯，水为符合GB/T6682中规定的一级水。

## 4.1 试剂

4.1.1 乙腈 $\mathrm{(CH_3CN)}$ ：高效液相色谱级。  
4.1.2 甲醇 $\mathrm{(CH_3OH)}$ ：高效液相色谱级。  
4.1.3 乙酸乙酯 $(\mathrm{C_4H_8O_2})$   
4.1.4 二氯甲烷 $\mathrm{(CH_2Cl_2)}$   
4.1.5 环己烷 $(\mathrm{C_6H_{12}})$   
4.1.6 硫酸钠 $\mathrm{(Na_2SO_4)}$ ： $650^{\circ}C$ 灼烧 $4\mathrm{h}$ ，在干燥器内冷却至室温，储于密封瓶中备用。  
4.1.7 乙酸铵 $\mathrm{(CH_3COONH_4)}$ ：优级纯或相当者。

## 4.2 溶液配制

$50\mathrm{mmol / L}$ 乙酸铵溶液：称取 $0.385\mathrm{g}$ 乙酸铵溶于 $1000~\mathrm{mL}$ 水中。

## 4.3 标准品

4.3.1 敌百虫(Trichlorfon, CAS号:52-68-6,分子式: $\mathrm{C_4H_8Cl_3O_4P)}$ 标准品:纯度 $\geq 98.0\%$ 。  
4.3.2 敌敌畏(Dichlorvos，CAS号：62-73-7，分子式： $\mathrm{C_4H_7Cl_2O_4P)}$ 标准品：纯度 $\geqslant 97.6\%$   
4.3.3 蝌毒磷(Coumaphos，CAS号：56-72-4，分子式： $\mathrm{C_{14}H_{16}ClO_5PS)}$ 标准品：纯度 $\geqslant 99.9\%$

## 4.4 标准溶液配制

4.4.1 标准储备液：分别准确称取适量敌百虫、敌敌畏、蝇毒磷标准品，用甲醇配制成浓度为 $100\mu \mathrm{g} / \mathrm{mL}$ 的混合标准储备液。此溶液在 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 中避光保存，可使用3个月。  
4.4.2 空白样品提取液：用不含敌百虫、敌敌畏、蝇毒磷残留的样品，按照7.1制备空白样品提取液。  
4.4.3 标准工作溶液：根据需要用空白样品提取液将标准储备液稀释成 $50\mathrm{ng / mL},100\mathrm{ng / mL},200$ $\mathrm{ng / mL},500\mathrm{ng / mL}$ 的混合标准工作溶液。置于 $0^{\circ}\mathrm{C}\sim 4^{\circ}\mathrm{C}$ 中避光保存，可使用3d。

# 5 仪器和设备

5.1 高效液相色谱—质谱/质谱仪：三重四极杆质谱检测器，配电喷雾离子源(ESI)或相当者。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$ 。  
5.3 超声波清洗器。  
5.4 涡旋振荡器。  
5.5 旋转蒸发仪。  
5.6 离心机： $5000\mathrm{r / min}$   
5.7 均质器。  
5.8 0.45 $\mu \mathrm{m}$ 微孔滤膜。

# 6 试样制备与保存

## 6.1 取样部位

样品取样部位按 GB 2763 的规定执行。在取样和制样的操作过程中, 应防止样品受到污染或发生残留物含量的变化。

## 6.2 试样制备

### 6.2.1 分割肉

取有代表性样品 $500\mathrm{g}$ ，用组织捣碎机捣碎，装入洁净容器作为试样，密封并做好标识。

### 6.2.2 肠衣

取有代表性样品 $100\mathrm{g}$ ，用剪刀剪碎至 $2\mathrm{mm}$ 以下，装入洁净容器作为试样，密封并做好标识。

### 6.2.3 蜂蜜

取有代表性样品 $500\mathrm{g}$ ，搅拌均匀后装入洁净容器内密封并做好标识。

## 6.3 试样保存

分割肉、肠衣、蜂蜜等试样于 $-18^{\circ} \mathrm{C}$ 以下冷冻保存。

# 7 分析步骤

## 7.1 提取与净化

### 7.1.1 分割肉

7.1.1.1 准确称取 $5 \mathrm{~g}$ 均匀试样(精确到 $0.01 \mathrm{~g}$ ) 于 $50 \mathrm{~mL}$ 具塞试管中, 加入 $5 \mathrm{~g}$ 无水硫酸钠混匀, 再加入 $15 \mathrm{~mL}$ 二氯甲烷, 用均质器 $(10000 \mathrm{r} / \mathrm{min})$ 均质 $2 \mathrm{~min}, 4000 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ , 将有机相转移至 $100 \mathrm{~mL}$ 梨形蒸馏瓶中, 残渣再用 $10 \mathrm{~mL}$ 二氯甲烷均质提取 2 次。  
7.1.1.2 离心合并有机相, 于 $40^{\circ} \mathrm{C}$ 旋转蒸发至 $2 \mathrm{~mL}$ , 将样液转移至 $5 \mathrm{~mL}$ 刻度试管中, 并用少量二氯甲烷洗涤梨形蒸馏瓶, 合并洗涤液到刻度试管中, 室温通氮浓缩至干。定量加入 $1.0 \mathrm{~mL}$ 乙腈溶解残渣, 加入 $2 \mathrm{~mL}$ 环己烷涡旋混匀 $2 \mathrm{~min}$ 后, $2500 \mathrm{r} / \mathrm{min}$ 离心 $3 \mathrm{~min}$ , 将下层乙腈相过 $0.45 \mu \mathrm{m}$ 微孔滤膜后, 供液相色谱一质谱/质谱仪测定。

### 7.1.2 肠衣

准确称取 $5\mathrm{g}$ 试样（精确到 $0.01\mathrm{g}$）于 $50\mathrm{mL}$ 具塞离心瓶中，加入 $2\mathrm{g}$ 无水硫酸钠混匀，再加入 $15\mathrm{mL}$ 二氯甲烷，盖上盖混匀。置于超声波清洗器中超声 $30\mathrm{min}$ ，冷却后将有机相过滤转移至 $100\mathrm{mL}$ 梨形蒸馏瓶中，残渣再用 $10\mathrm{mL}$ 用二氯甲烷混匀超声提取2次，离心合并有机相。按照7.1.1.2的规定进行。

### 7.1.3 蜂蜜

称取 $5\mathrm{g}$ 试样(精确到 $0.01\mathrm{g}$ ) 置于 $50\mathrm{mL}$ 具塞离心管中，加 $10\mathrm{mL}$ 水、 $25\mathrm{mL}$ 乙酸乙酯，于涡旋振荡器上混匀 $1\mathrm{min}$ ，以 $3000\mathrm{r / min}$ 离心 $5\mathrm{min}$ 。将上层乙酸乙酯提取液收集于浓缩瓶中，残渣再加入 $20\mathrm{mL}$ 乙酸乙酯。重复上述操作，合并乙酸乙酯提取溶液。在 $50^{\circ}\mathrm{C}$ 以下减压浓缩至约 $3\mathrm{mL}$ 后转移至 $10\mathrm{mL}$ 离心管中，用 $5\mathrm{mL}$ 乙酸乙酯分2次洗涤浓缩瓶，合并洗涤液于离心管中，用氮气吹干。加 $1.0\mathrm{mL}$ 乙腈溶解残渣，并过 $0.45\mu \mathrm{m}$ 微孔滤膜后，供液相色谱一质谱/质谱仪测定。

## 7.2 测定

### 7.2.1 液相色谱一质谱/质谱参考条件

a）色谱柱：柱填料为十八烷基硅烷键合相的色谱柱， $4.6\mathrm{mm}\times 150\mathrm{mm}$ ，膜厚 $5\mu \mathrm{m}$ ，或相当者；  
b）流动相：甲醇： $50\mathrm{mmol / L}$ 乙酸铵，梯度洗脱程序参见附录A；  
c) 流速: $0.50 \mathrm{~mL} / \mathrm{min}$ 或根据仪器条件优化;   
d) 柱温: $40^{\circ} \mathrm{C}$ ;   
e) 进样量: $10 \mu \mathrm{L}$ ;   
f) 离子源: 电喷雾离子源;  
g）扫描方式：正离子；  
h）检测方式：多反应监测(MRM)；  
i）质谱/质谱参考条件参见附录B。

### 7.2.2 色谱测定与确证

按照7.2.1液相色谱—质谱/质谱条件测定样液和标准工作溶液，外标标准曲线法测定样液中的敌百虫、敌敌畏、蝇毒磷残留量。样品中待测物残留量应在标准曲线范围之内，如果残留量超出标准曲线范围，应用空白样品提取液进行适当稀释。在上述色谱条件下，敌百虫、敌敌畏、蝇毒磷的质量色谱峰保留时间约为 $13.2\mathrm{min}, 14.1\mathrm{min}, 15.9\mathrm{min}$ 。标准品的多反应监测色谱图参见图C.1。

在相同的实验条件下，样品与标准工作液中待测物质的质量色谱峰相对保留时间在 $2.5\%$ 以内，并且在扣除背景后的样品质量色谱图中，所选择的离子对均出现，同时与标准品的相对丰度允许偏差不超过表1规定的范围，则可判断样品中存在对应的被测物。

## 7.3 空白实验

除不加试样外，均按 $7.1\sim 7.3$ 的规定执行。

# 8 结果计算和表述

用数据处理软件中的外标法，或绘制标准曲线，按式(1)计算样品中敌百虫、敌敌畏、蝇毒磷残留量。

$$
x = \frac {C \times V}{m \times 1 0 0 0} \dots \tag {1}
$$

式中：

$x$ ——试样中敌百虫、敌敌畏、蝇毒磷的残留量，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

C 由标准曲线而得的样液中敌百虫、敌敌畏、蝇毒磷的浓度，单位为纳克每毫升 $(\mathrm{ng / mL})$

$V$ ——样液最终定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——最终样液所代表的试样质量，单位为克(g)；

1000 ——换算系数。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的2次独立测定结果的绝对差值与其算术平均值的比值(百分率)，应符合附录D的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 E 的要求。

# 10 定量限和回收率

## 10.1 定量限

本方法敌百虫、敌敌畏、蝇毒磷的测定定量限均为 $0.010\mathrm{mg / kg}$

## 10.2 回收率

10.2.1 肠衣中敌百虫添加浓度及其回收率

a) 添加浓度为 $0.010 \mathrm{mg} / \mathrm{kg}$ 时, 回收率在 $73.0\% \sim 94.0\%$ 之间;  
b) 添加浓度为 $0.020 \mathrm{mg} / \mathrm{kg}$ 时, 回收率在 $78.0\% \sim 95.0\%$ 之间;  
c) 添加浓度为 $0.040 \mathrm{mg} / \mathrm{kg}$ 时, 回收率在 $78.0\% \sim 95.0\%$ 之间。

10.2.2 肠衣中敌敌畏添加浓度及其回收率

a) 添加浓度为 $0.010 \mathrm{mg} / \mathrm{kg}$ 时, 回收率在 $76.0\% \sim 98.0\%$ 之间;  
b) 添加浓度为 $0.020 \mathrm{mg} / \mathrm{kg}$ 时, 回收率在 $78.0\% \sim 98.0\%$ 之间;  
c) 添加浓度为 $0.040 \mathrm{mg} / \mathrm{kg}$ 时, 回收率在 $86.0\% \sim 95.0\%$ 之间。

10.2.3 肠衣中蝇毒磷添加浓度及其回收率

10.2.7 蜂蜜中敌百虫添加浓度及其回收率

# 食品安全国家标准 食品中三氯蔗糖（蔗糖素）的测定

2015-01-28 发布

2015-07-28 实施

# 前言

本标准代替GB/T22255—2008《食品中三氯蔗糖(蔗糖素)的测定》。

本标准与GB/T22255—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品中三氯蔗糖(蔗糖素)的测定”；  
——增加了示差检测器的应用；  
——增加固相萃取柱净化、分离的试样制备步骤；  
——增加发酵酒、配制酒、不含蛋白的饮料的试样制备步骤。

# 1 范围

本标准规定了食品中三氯蔗糖（蔗糖素）的测定方法。

本标准适用于食品中三氯蔗糖的测定。

# 2 原理

试样中三氯蔗糖用甲醇水溶液提取，除蛋白、脂肪，经固相萃取柱净化、富集后用高效液相色谱仪、反相 $\mathrm{C_{18}}$ 色谱柱分离，蒸发光散射检测器或示差检测器检测，根据保留时间定性，以峰面积定量。

# 3 试剂和材料

注：除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的一级水。

## 3.1 试剂

3.1.1 甲醇 $(\mathrm{CH}_3\mathrm{OH})$ 。  
3.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
3.1.3 正己烷 $(\mathrm{C_6H_{14}})$ 。  
3.1.4 乙酸锌 $\left[\mathrm{Zn}(\mathrm{CH}_3\mathrm{COO})_2\cdot 2\mathrm{H}_2\mathrm{O}\right]$   
3.1.5 亚铁氰化钾 $\left[\mathrm{K}_4\mathrm{Fe}(\mathrm{CN})_6\cdot 3\mathrm{H}_2\mathrm{O}\right]$   
3.1.6 中性氧化铝（100目～200目）。

## 3.2 试剂的配制

3.2.1 乙酸锌溶液 $(219\mathrm{g / L})$ ：称取 $21.9\mathrm{g}$ 乙酸锌，加 $3\mathrm{mL}$ 乙酸，加水溶解至 $100~\mathrm{mL}$   
3.2.2 亚铁氰化钾溶液 $(106\mathrm{g / L})$ ：称取 $10.6\mathrm{g}$ 亚铁氰化钾，加水溶解至 $100~\mathrm{mL}$ 。  
3.2.3 甲醇水溶液 $(75 + 25)$ ：量取 $75~\mathrm{mL}$ 甲醇，加 $25~\mathrm{mL}$ 水，混匀。  
3.2.4 乙腈水溶液 $(11 + 89)$ ：量取 $11\mathrm{mL}$ 乙腈，加 $89~\mathrm{mL}$ 水，混匀。

## 3.3 标准品

三氯蔗糖标准品 $(\mathrm{C_{12}H_{19}C_{13}O_8})$ ：CAS编号56038-13-2，纯度 $\geq 99\%$ 。

## 3.4 标准溶液的制备

3.4.1 三氯蔗糖标准贮备溶液 $(10.0\mathrm{mg / mL})$ ：称取三氯蔗糖标准品 $0.25\mathrm{g}$ （精确至 $0.0001\mathrm{g}$ )于 $25~\mathrm{mL}$ 容量瓶中，用水定容至刻度，混匀，其中三氯蔗糖浓度为 $10.0\mathrm{mg / mL}$ 。贮备液置于 $4^{\circ}c$ 冰箱中保存，保存期为6个月。  
3.4.2 三氯蔗糖标准中间液 $(1.00\mathrm{mg / mL})$ ：吸取 $5.00~\mathrm{mL}$ 三氯蔗糖标准贮备溶液于 $50~\mathrm{mL}$ 容量瓶中，

用水定容至刻度，混匀，其中三氯蔗糖浓度为 $1.00\mathrm{mg / mL}$ 。置于 $4^{\circ}c$ 冰箱中保存，保存期为3个月。

3.4.3 三氯蔗糖标准工作液：分别吸取 $0.200\mathrm{mL},0.500\mathrm{mL},1.00\mathrm{mL},2.00\mathrm{mL},4.00\mathrm{mL}$ 三氯蔗糖中间液于 $10~\mathrm{mL}$ 容量瓶中，用水定容至刻度。其中三氯蔗糖工作液浓度分别为 $0.0200\mathrm{mg / mL}$ 、 $0.0500\mathrm{mg / mL},0.100\mathrm{mg / mL},0.200\mathrm{mg / mL},0.400\mathrm{mg / mL}$ 。

## 3.5 材料

固相萃取柱（ $200\mathrm{mg}$ ，类型为 $N$ -乙烯基吡咯烷酮和二乙烯基苯亲水亲脂平衡型填料）使用前依次用 $4\mathrm{mL}$ 甲醇、 $4\mathrm{mL}$ 水活化。

# 4 仪器和设备

4.1 高效液相色谱仪：配示差检测器或蒸发光散射检测器。  
4.2 天平：感量为 $0.1\mathrm{mg}$ 和 $1\mathrm{mg}$   
4.3 涡旋混合器。  
4.4 离心机：转速 $\geq 3000\mathrm{r / min}$   
4.5 离心机：转速 $\geqslant 10000\mathrm{r / min}$   
4.6 超声波清洗仪：工作频率 $35\mathrm{kHz}$   
4.7 水浴锅。

# 5 分析步骤

## 5.1 试样制备

### 5.1.1 含蛋白脂肪试样

5.1.1.1 称取粉碎均匀后固体试样 $1\mathrm{g}\sim 2\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )，混匀后液体试样 $1\mathrm{g}\sim 5\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )置于 $50~\mathrm{mL}$ 离心管中，加入 $5\mathrm{mL}$ 水，涡旋混合器上振荡 $3\mathrm{min}$ 后加入 $15~\mathrm{mL}$ 甲醇，继续振荡 $30~s$ ，超声波提取 $20\mathrm{min}$ ，以 $3000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，将上清液移入 $50~\mathrm{mL}$ 离心管中。沉淀物加入 $5.0~\mathrm{mL}$ 甲醇水溶液 $(75 + 25)$ ，玻棒搅拌均匀后，涡旋混合器上振荡 $30~s$ ，以 $3000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ ，重复提取2次，将上清液合并于 $150~\mathrm{mL}$ 分液漏斗中。  
5.1.1.2 在 $150~\mathrm{mL}$ 分液漏斗内，加入 $30~\mathrm{mL}$ 正己烷，振摇 $2\mathrm{min}$ ，静置分层， $20\mathrm{min}$ 后，下层水相移置于 $50~\mathrm{mL}$ 蒸发皿。蒸发皿于沸水浴上蒸发，当蒸发皿中液体在 $1\mathrm{mL}$ 左右时用 $9\mathrm{mL}$ 水分三次冲洗蒸发皿，冲洗液合并移置于 $15~\mathrm{mL}$ 离心管中，超声波处理 $5\mathrm{min}$ ，以 $3000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。  
5.1.1.3 取全部上清液移入已活化的固相萃取柱，控制液体流速不超过每秒1滴，柱上液面为 $2\mathrm{mm}$ 左右时加入 $1\mathrm{mL}$ 水，继续保持液体流速为每秒1滴，到柱中液体完全排出后，用 $3\mathrm{mL}$ 甲醇洗脱，收集甲醇洗脱液。洗脱液置于 $50~\mathrm{mL}$ 蒸发皿内，于沸水浴上蒸干，残渣用 $1.00\mathrm{mL}$ 乙腈水溶液 $(11 + 89)$ 溶解（如溶液有浑浊现象可将其移入离心管， $10000\mathrm{r / min}$ 离心 $5\mathrm{min})$ ，溶液过 $0.45\mu \mathrm{m}$ 滤膜，滤液为制备的试样溶液，备用。

注：果冻类样品经提取后的上清液需 $50^{\circ}\mathrm{C}$ 水浴加热后趁热过柱，否则易堵塞萃取柱。

### 5.1.2 酱及酱制品、醋、酱油

称取混匀后试样 $2\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 离心管中，加入 $1.0\mathrm{g}$ 中性氧化铝，加入 $3\mathrm{mL}$ 水，涡旋混合器上振荡 $3\mathrm{min}$ 后加入 $15~\mathrm{mL}$ 甲醇，以下步骤自5.1.1.1“继续振荡 $30~s$ ，超声波提取 $20\mathrm{min}$ ，以 $3000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 开始，到5.1.1.3“滤液为制备的试样溶液，备用。”为止依次处理。

### 5.1.3 含酒精的试样（发酵酒、配制酒）

称取混匀后试样 $5\mathrm{g}$ （精确到 $0.001\mathrm{g}$ ，置于 $50~\mathrm{mL}$ 蒸发皿中，于沸水浴上蒸干，残渣用 $1.00~\mathrm{mL}$ 乙腈水溶液 $(11 + 89)$ 溶解，溶液过 $0.45\mu \mathrm{m}$ 滤膜，滤液为制备的试样溶液，备用。

### 5.1.4 饮料

称取混匀后试样 $5\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )，置于 $15~\mathrm{mL}$ 离心管中，加入 $5\mathrm{mL}$ 水，涡旋混合器上振荡 $30~s$ 。以 $3000~\mathrm{r / min}$ 离心 $10\mathrm{min}$ 。以下步骤按5.1.1.3处理。

### 5.1.5 风味发酵乳、奶茶

称取混匀后试样 $1\mathrm{g}\sim 5\mathrm{g}$ （精确到 $0.001\mathrm{g}$ )置于 $50~\mathrm{mL}$ 离心管中，加入 $5\mathrm{mL}$ 水，涡旋混合器上振荡 $3\mathrm{min}$ 后加入 $15~\mathrm{mL}$ 甲醇、 $0.50~\mathrm{mL}$ 乙酸锌溶液、 $0.50~\mathrm{mL}$ 亚铁氰化钾溶液，以下步骤自5.1.1.1“继续振荡 $30~s$ ，超声波提取 $20\mathrm{min}$ ，以 $3000\mathrm{r / min}$ 离心 $10\mathrm{min}$ 开始，到5.1.1.3“滤液为制备的试样溶液，备用。”为止依次处理。

不同试样的前处理需要同时做试样空白试验。

## 5.2 仪器参考条件

### 5.2.1 色谱柱

$\mathrm{C_{18}}$ 柱（ $4.6 \mathrm{~mm} \times 150 \mathrm{~mm}, 5 \mu \mathrm{m}$ ）或性能相当者。  
### 5.2.2 流动相

水 $+$ 乙腈 $= 89 + 11$

注：当检测样品基质复杂，强保留物质影响后续检测时，可采取洗脱程序(蒸发光散射检测器适用)，参见附录A。

### 5.2.3 流速

$1.0~\mathrm{mL / min}$   
### 5.2.4 柱温

$35^{\circ}C$ 。

### 5.2.5 示差检测器条件

——检测池温度： $35^{\circ}C$

——灵敏度：16。

### 5.2.6 蒸发光散射检测器条件

按不同品牌蒸发光散射检测器在高水相流动相条件下的要求设置。

如SEDEX75：

雾化压力：3.5Bar；增益：8；蒸发温度： $40^{\circ}\mathrm{C}$ 。或性能相当者。

### 5.2.7 进样量

$20.0\mu \mathrm{L}$

## 5.3 标准曲线的制作

### 5.3.1 示差检测器

取三氯蔗糖标准工作液分别进样 $20.0\mu \mathrm{L}$ ，在上述色谱条件下测定峰面积，然后作峰面积-三氯蔗糖浓度 $(\mathrm{mg / mL})$ 标准曲线，曲线方程依示差检测原理，见式(1)：

$$
y = a x + b \quad \dots \tag {1}
$$

式中：

$y$ ——峰面积；

$a, b$ ——与检测池温度、流动相性质等实验条件有关的常数；

$x$ 三氯蔗糖的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ 。

### 5.3.2 蒸发光散射检测器

取三氯蔗糖标准工作液分别进样 $20.0\mu \mathrm{L}$ ，在给定的色谱条件下测定峰面积，然后作峰面积-三氯蔗糖浓度 $(\mathrm{mg / mL})$ 标准曲线，曲线方程依蒸发光散射检测原理，见式(2)：

$$
y = b x ^ {a} \quad \dots \tag {2}
$$

式中：

$y$ ——峰面积；

$a, b$ ——与蒸发室温度、流动相性质等实验条件有关的常数；

$x$ 三氯蔗糖的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ 。

按仪器数据处理软件的处理方式不同，也可作对数方程，即 $\lg y = b + a\lg x$ 。

## 5.4 试样溶液的测定

取制备的试样溶液和空白试样溶液各 $20.0\mu \mathrm{L}$ 进样，进行高效液相色谱分析。以保留时间定性，以峰面积外标法定量，见附录B。

# 6 分析结果的表述

试样中三氯蔗糖含量按式(3)计算：

$$
X = \frac {\left(c - c _ {0}\right) \times V \times 1 0 0 0}{m \times 1 0 0 0} \tag {3}
$$

式中：

$X$ ——试样中三氯蔗糖的含量，单位为克每千克 $(\mathrm{g / kg})$ ；

$c$ ——由标准曲线查得试样进样液中三氯蔗糖的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$c_{0}$ ——由标准曲线查得空白试样进样液中三氯蔗糖的浓度，单位为毫克每毫升 $(\mathrm{mg / mL})$ ；

$V$ ——试样定容体积，单位为毫升 $(\mathrm{mL})$ ；

$m$ ——试样质量，单位为克 $(\mathrm{g})$ ；

1000 ——换算系数。

结果保留三位有效数字。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 8 其他

当取样量 $2.00\mathrm{g}$ ，定容至 $1.00\mathrm{mL}$ 时，检出限均为 $0.0025\mathrm{g/kg}$ ，定量限均为 $0.0075\mathrm{g/kg}$ 。

# 1 范围

本标准适用于由硬脂酸和乳酸为原料，经酯化、中和得到的食品添加剂硬脂酰乳酸钙。

# 附录A 检验方法

## A.1 一般规定

本标准所用试剂和水在未注明其他要求时，均指分析纯试剂和GB/T6682规定的三级水。试验中所用标准溶液、杂质测定用标准溶液、制剂和制品在未注明其他要求时，均按GB/T601、GB/T602、GB/T603的规定制备。试验中所用溶液在未注明用何种溶剂配制时，均指水溶液。

## A.2 鉴别试验

### A.2.1 钙盐试验

取试样 $1\mathrm{g}$ ，加 $25\mathrm{mL}$ 水和 $5\mathrm{mL}$ 盐酸，加热释放出脂肪酸，使油状层浮出液体表面。水层加草酸铵溶液，即产生白色沉淀。此沉淀不溶于乙酸，但溶解于盐酸。用盐酸湿润后的铂丝蘸取试样，在火焰中呈红色。

### A.2.2 凝固点试验

取试样 $25\mathrm{g}$ ，加热水 $200~\mathrm{mL}$ ，再加 $0.05\mathrm{mol} / \mathrm{L}$ 硫酸溶液 $60~\mathrm{mL}$ ，边加热边搅拌至脂肪酸析出并呈透明状清液。将水层吸出，加沸水充分洗涤，静置分层后，再将水层吸出，洗涤至无硫酸盐（加 $0.1\%$ 甲基橙指示剂不呈红色），收集于小烧杯中，于蒸汽浴上温热至脂肪酸与水层完全分离，并澄清透明。冷却，弃去水层，将脂肪酸熔融后过滤至干燥烧杯中，在 $105^{\circ}\mathrm{C}$ 下干燥 $20\mathrm{min}$ 。该精制脂肪酸的凝固点应不低于 $54^{\circ}\mathrm{C}$ 。凝固点按SN/T0801.17规定的方法测定。

## A.3 总乳酸的测定

### A.3.1 试剂和材料

A.3.1.1 乳酸锂。  
A.3.1.2 石油醚。  
A.3.1.3 硫酸。  
A.3.1.4 硫酸溶液： $1 + 2$   
A.3.1.5 硫酸铜溶液： $20~\mathrm{g / L}$   
A.3.1.6 对苯基酚溶液：称取对苯基酚 $1.5\mathrm{g}$ 溶于 $100\mathrm{mL}0.125\mathrm{mol / L}$ 氢氧化钠溶液中（配制时加热助溶，溶解后为澄清溶液），贮于棕色瓶中， $4^{\circ}\mathrm{C}$ 保存。  
A.3.1.7 氢氧化钾乙醇溶液： $11.2\mathrm{g}$ 氢氧化钾溶于 $250~\mathrm{mL}$ 乙醇中，再用 $25~\mathrm{mL}$ 水稀释。

### A.3.2 仪器和设备

分光光度计。

### A.3.3 分析步骤

#### A.3.3.1 试样储备液的制备

称取约 $0.2\mathrm{g}$ 试样(精确至 $0.0001\mathrm{g}$ )，放入 $125~\mathrm{mL}$ 锥形烧瓶中，加 $0.5\mathrm{mol} / \mathrm{L}$ 氢氧化钾乙醇溶液

$10~\mathrm{mL}$ 和水 $10~\mathrm{mL}$ ，装上空气冷凝器，缓慢回流 $45\mathrm{min}$ ，用水约 $40~\mathrm{mL}$ 洗涤冷凝器和瓶边，在蒸汽浴上加热至无乙醇气味，加硫酸溶液 $6\mathrm{mL}$ ，加热至脂肪酸融化，然后冷却至约 $60^{\circ}\mathrm{C}$ ，加 $25~\mathrm{mL}$ 石油醚，缓慢旋动混合物后立即移入分液漏斗，将水层收集于 $100~\mathrm{mL}$ 容量瓶中。用水洗涤石油醚层两次，每次 $20~\mathrm{mL}$ ，洗液并入容量瓶。用水稀释至 $100~\mathrm{mL}$ 并混合。

#### A.3.3.2 试样溶液的制备

取 $1.0\mathrm{mL}$ 试样储备液，放入 $100~\mathrm{mL}$ 容量瓶中，用水稀释并定容。

#### A.3.3.3 标准储备液的制备

称取 $1.067\mathrm{g}$ 乳酸锂，用水溶解后定容至 $1000~\mathrm{mL}$ ，量取 $10.0~\mathrm{mL}$ ，移入 $100~\mathrm{mL}$ 容量瓶中，再用水稀释并定容。

#### A.3.3.4 标准溶液的制备

分别吸取标准储备液 $1.0\mathrm{mL},2.0\mathrm{mL},4.0\mathrm{mL},6.0\mathrm{mL}$ 和 $8.0~\mathrm{mL}$ ，各放入 $100~\mathrm{mL}$ 容量瓶中，用水定容并混合。这些标准溶液中的乳酸含量分别为每毫升 $1\mu \mathrm{g},2\mu \mathrm{g},4\mu \mathrm{g},6\mu \mathrm{g}$ 和 $8\mu \mathrm{g}$ 。

### A.3.4 测定

分别吸取 $1.0\mathrm{mL}$ 标准溶液系列和试样溶液，分别放入6个试管中。另取 $1.0\mathrm{mL}$ 水放入试管中作为对照。每个试管按如下操作进行：加入1滴硫酸铜溶液，迅速加入 $9.0\mathrm{mL}$ 硫酸，缓慢转动，松开试管塞，在 $90^{\circ}\mathrm{C}$ 水浴中加热 $5\mathrm{min}$ 后立即在冰水浴中冷却 $5\mathrm{min}$ 至 $20^{\circ}\mathrm{C}$ 以下，加3滴对苯基酚溶液，立即摇动，于 $30^{\circ}\mathrm{C}$ 水浴中加热 $30\mathrm{min}$ ，在这段时间内摇动两次以使试剂分散。将试管放入 $90^{\circ}\mathrm{C}$ 水浴中维持 $90\mathrm{s}$ ，取出后立即放入冰水中迅速冷却至室温。盛于 $1\mathrm{cm}$ 吸收池中，用适当的分光光度计在 $570\mathrm{nm}$ 处测定吸光度，绘制标准曲线。对照标准曲线即可求得试样溶液中的总乳酸质量 $(\mu \mathrm{g})$ 。

### A.3.5 结果计算

总乳酸的质量分数 $\varpi_{1}$ ，按式(A.1)计算：

$$
w _ {1} = \frac {m _ {0}}{1 0 0 0 \times m} \times 1 0 0 \% \tag{A.1}
$$

式中：

$m_{0}$ ——试样溶液中的总乳酸质量(查标准曲线得到)，单位为微克 $(\mu \mathrm{g})$ ；

1000——换算系数；

$m$ ——试样的质量，单位为毫克 $(\mathrm{mg})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $0.5\%$ 。

## A.4 酯值（以KOH计）的测定

### A.4.1 试剂和材料

A.4.1.1 硫酸标准滴定溶液： $c\left(\frac{1}{2}\mathrm{H}_2\mathrm{SO}_4\right) = 0.1\mathrm{mol / L}$   
A.4.1.2 氢氧化钾乙醇溶液： $11.2\mathrm{g}$ 氢氧化钾溶于 $250~\mathrm{mL}$ 乙醇中，再用 $25~\mathrm{mL}$ 水稀释。

A.4.1.3 酚酞指示剂： $10\mathrm{g / L}$

### A.4.2 分析步骤

#### A.4.2.1 试样溶液的制备

酸值测定中所保留的中和液。

#### A.4.2.2 测定

取试样溶液，加入 $10\mathrm{mL}$ 氢氧化钾乙醇溶液，加入5滴酚酞指示剂，装上冷凝器，回流 $2\mathrm{h}$ ，冷却后再加5滴酚酞指示剂，用硫酸标准滴定溶液滴定过量的碱。同时用 $10.0\mathrm{mL}$ 氢氧化钾乙醇溶液做空白试验。

### A.4.3 结果计算

酯值（以KOH计） $\varpi_{2}$ ，按式(A.2)计算：

$$
w _ {2} = \frac {c \times \left(V _ {0} - V _ {1}\right) \times M}{1 0 0 0 \times m} \tag {A.2}
$$

式中：

$c$ ——滴定过程中所使用硫酸标准滴定溶液的实际浓度，单位为摩尔每升 $(\mathrm{mol} / \mathrm{L})$ ；

$V_{0}$ ——空白溶液滴定所消耗硫酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{1}$ ——试样溶液滴定所消耗硫酸标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$M$ ——氢氧化钾的摩尔质量，单位为克每摩尔 $(\mathrm{g / mol})[M(\mathrm{KOH}) = 56.1]$ ；

1000——体积换算系数；

$m$ ——试样的质量，单位为克 $(\mathrm{g})$ 。

试验结果以平行测定结果的算术平均值为准。在重复性条件下获得的两次独立测定结果的绝对差值不大于 $2\mathrm{mg / g}$ 。

# GB1886.179—2016《食品安全国家标准 食品添加剂 硬脂酰乳酸钙》第1号修改单

本修改单经中华人民共和国国家卫生健康委员会和国家市场监督管理总局于2022年6月30日第3号公告批准，自批准之日起实施。

## 一、A.3.5结果计算

将式(A.1)中“ $w_{1} = \frac{m_{0}}{1000 \times m} \times 100\%$ ”修改为“ $w_{1} = \frac{m_{0}}{100 \times m} \times 100\%$ ”；式中“1000”修改为“100”，试样的质量 $m$ 单位“毫克 $(\mathrm{mg})$ ”修改为“克 $(\mathrm{g})$ ”。

## 二、A.4.3结果计算

将式(A.2)中“ $w_{2} = \frac{c\times(V_{0} - V_{1})\times M}{1000\times m}$ ”修改为“ $w_{2} = \frac{c\times(V_{0} - V_{1})\times M}{m}$ ”，删除式中的“1000—体积换算系数”。

# 食品安全国家标准 食品添加剂 藻衣草油

2015-09-22 发布

2016-03-22 实施

# 前言

本标准代替GB/T12653—2008《中国薰衣草(精)油》中的食品添加剂部分。

本标准与GB/T12653—2008相比，主要变化如下：

——标准名称修改为“食品安全国家标准 食品添加剂 薰衣草油”。

# 1 范围

本标准适用于用水蒸气蒸馏法从中国薰衣草(Lavandula angustifolia Mill.)的开花部分提取的食品添加剂薰衣草油。

# 附录A 特征组分含量的测定

### A.1 仪器和设备

A.1.1 色谱仪：按GB/T11538—2006中第5章的规定。  
A.1.2 柱：毛细管柱。  
A.1.3 检测器：氢火焰离子化检测器。

### A.2 测定方法

面积归一化法：按GB/T11538—2006中10.4测定含量。

### A.3 重复性及结果表示

按GB/T11538—2006中11.4规定进行。

### B.2 操作条件

B.2.1 柱：毛细管柱，长 $30\mathrm{m}$ ，内径 $0.25\mathrm{mm}$ 。

B.2.2 固定相：聚乙二醇。  
B.2.3 膜厚: $0.25 \mu \mathrm{m}$ 。  
B.2.4 色谱炉温度： $70^{\circ}\mathrm{C}$ 恒温 $16\mathrm{min}$ ，然后线性程序升温从 $70^{\circ}\mathrm{C} \sim 180^{\circ}\mathrm{C}$ ，速率 $2^{\circ}\mathrm{C} / \mathrm{min}$ 。  
B.2.5 进样口温度： $200^{\circ}C$ 。  
B.2.6 检测器温度： $200^{\circ}C$   
B.2.7 检测器：氢火焰离子化检测器。   
B.2.8 载气：氮气。  
B.2.9 载气流速： $1\mathrm{mL / min}$   
B.2.10 进样量：约 $0.3\mu \mathrm{L}$   
B.2.11 分流比： $100:1$

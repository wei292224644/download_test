# 前言

本文件按照GB/T1.1—2020《标准化工作导则》第1部分：标准化文件的结构和起草规则》的规定起草。

本文件系国内首次发布。

# 1 范围

本文件规定了植物源性食品中杀虫单、杀虫双、杀虫环和杀螟丹残留量的气相色谱测定方法。

本文件适用于植物源性食品(韭菜除外)中杀虫单、杀虫双、杀虫环和杀螟丹残留量的测定。

# 2 规范性引用文件

下列文件中的内容通过文中的规范性引用而构成本文件必不可少的条款。其中，注日期的引用文件，仅该日期对应的版本适用于本文件；不注日期的引用文件，其最新版本（包括所有的修改单）适用于本文件。

# 3 原理

试样中杀虫单、杀虫双、杀虫环和杀螟丹用含有半胱氨酸盐酸盐的盐酸溶液提取，在碱性条件下用氯化镍催化衍生转化成沙蚕毒素，用带有电子捕获检测器的气相色谱仪测定，外标法定量。

# 4 试剂和材料

除非另有说明，在分析中仅使用确认为色谱纯的试剂和符合GB/T6682规定的二级水。

## 4.1 试剂

4.1.1 盐酸（HCl，CAS号：7647-01-0）：分析纯。  
4.1.2 六水合氯化镍 $\mathrm{(NiCl_2\cdot 6H_2O}$ CAS号：7791-20-0）：分析纯。  
4.1.3 $L$ -半胱氨酸盐酸盐无水物 $(\mathrm{C_3H_7NO_2S}\cdot \mathrm{HCl},\mathrm{CAS}$ 号：52-89-1)：生物技术级。  
4.1.4 氢氧化钠(NaOH，CAS号：1310-73-2)：分析纯。  
4.1.5 氯化钠(NaCl，CAS号：7647-14-5)：分析纯  
4.1.6 正己烷 $\mathrm{(C_6H_{14}}$ ，CAS号：110-54-3)。  
4.1.7 甲醇 $\mathrm{(CH_3OH}$ ，CAS号：67-56-19  
4.1.8 丙酮 $\mathrm{(CH_3COCH_3}$ ，CAS号：67-64-1）。

## 4.2 溶液配制

4.2.1 盐酸溶液 $(0.1\mathrm{mol} / \mathrm{L})$ ：移取 $4.15~\mathrm{mL}$ 盐酸(4.1.1)用水定容至 $500~\mathrm{mL}$ ，摇匀。  
4.2.2 含有 $1\% L-$ 半胱氨酸盐酸盐的 $0.1\mathrm{mol} / \mathrm{L}$ 盐酸溶液：称取 $2.525\mathrm{g}L-$ 半胱氨酸盐酸盐(4.1.3)，用盐酸溶液(4.2.1)溶解，并定容至 $250~\mathrm{mL}$ ，摇匀。  
4.2.3 氢氧化钠溶液 $(5\mathrm{mol} / \mathrm{L})$ ：称取 $50.0\mathrm{g}$ 氢氧化钠(4.1.4)，用水溶解并定容至 $250~\mathrm{mL}$ ，摇匀。  
4.2.4 氯化镍溶液 $(2\%)$ ：称取 $9.357\mathrm{g}$ 六水合氯化镍(4.1.2）用水溶解并定容至 $250~\mathrm{mL}$ ，摇匀。

## 4.3 标准品

4.3.1杀虫单 $(\mathrm{C}_5\mathrm{H}_{12}\mathrm{NNaO}_6\mathrm{S}_4$ ，CAS号：29547-00-0）。  
4.3.2杀虫双 $\mathrm{(C_5H_{11}NNa_2O_6S_4}$ ，CAS号：52207-48-4）。  
4.3.3 杀虫环草酸盐 $\mathrm{(C_5H_{11}NS_3\cdot C_2H_2O_4}$ ,CAS号：31895-22-4）。  
4.3.4杀螟丹 $\mathrm{(C_7H_{15}N_3O_2S_2}$ ，CAS号：22042-59-7）。

4.3.5 沙蚕毒素草酸盐（ $\mathrm{C_5H_{11}NS_2}\cdot \mathrm{C_2H_2O_4}$ ，CAS号：1631-52-3）。

## 4.4 标准溶液配制

4.4.1 沙蚕毒素标准储备溶液 $(1000\mathrm{mg / L})$ ：称取一定量的沙蚕毒素草酸盐标准品于 $10~\mathrm{mL}$ 烧杯中，用甲醇-水溶液 $(50 + 50$ ，体积比)溶解后转移到 $10~\mathrm{mL}$ 容量瓶中定容，低于一 $18^{\circ}C$ 保存，有效期6个月。

注：沙蚕毒素(149.28 g/mol)的称样量通过与沙蚕毒素草酸盐(239.31 g/mol)摩尔质量之间的折算确定。

4.4.2 沙蚕毒素标准工作溶液（ $100\mathrm{mg / L}$ ）：移取适量的沙蚕毒素标准储备溶液于 $10~\mathrm{mL}$ 容量瓶中，用丙酮(4.1.8)定容，配制成标准工作溶液，低于 $-18^{\circ}C$ 保存，有效期1个月。  
4.4.3杀虫单、杀虫双、杀虫环和杀螟丹标准储备溶液 $(1000\mathrm{mg / L})$ ：分别准确称取适量的杀虫单、杀虫双、草酸氢杀虫环、杀螟丹标准品，用甲醇(4.1.7)溶解后转移到 $10~\mathrm{mL}$ 容量瓶中定容，低于一 $18^{\circ}C$ 保存，有效期6个月。

注：杀虫单(351.39 g/mol)、杀虫双(355.4 g/mol)、杀虫环草酸盐(271.38 g/mol)、杀螟丹(237.34 g/mol)称样量分别通过与沙蚕毒素(149.28 g/mol)摩尔质量之间的折算确定。

4.4.4杀虫单、杀虫双、杀虫环和杀螟丹标准工作溶液 $(100\mathrm{mg / L})$ ：分别移取适量的杀虫单、杀虫双、杀虫环和杀螟丹标准储备溶液于 $10~\mathrm{mL}$ 容量瓶，用甲醇(4.1.7)定容，配制成标准工作溶液，低于一 $18^{\circ}C$ 保存，有效期1个月。

# 5 仪器和设备

5.1 气相色谱仪：配有电子捕获检测器。  
5.2 分析天平：感量 $0.1\mathrm{mg}$ 、感量 $0.001\mathrm{g}$ 和感量 $0.01\mathrm{g}$ 。  
5.3 容量瓶： $10\mathrm{mL},250\mathrm{mL},500\mathrm{mL}$   
5.4 聚丙烯离心管： $50~\mathrm{mL}$   
5.5 振荡仪。  
5.6 离心机：转速 $\geq 3800\mathrm{r / min}$   
5.7 搅拌机。  
5.8 粉碎机。

# 6 试样制备和储存

样品测定部位按照GB2763—2021中附录A的规定执行。食用菌、热带和亚热带水果(皮可食)随机取样 $1\mathrm{kg}$ ，水生蔬菜、茎菜类蔬菜、豆类蔬菜、核果类水果、热带和亚热带水果(皮不可食)随机取样 $2\mathrm{kg}$ ，瓜类蔬菜和水果取4个～6个个体(取样量不少于 $1\mathrm{kg}$ )，其他蔬菜和水果随机取样 $3\mathrm{kg}$ 。对于个体较小的样品，取样后全部处理；对于个体较大的基本均匀样品，可在对称轴或对称面上分割或切成小块后处理；对于细长、扁平或组分含量在各部分有差异的样品，可在不同部位切取小片或截成小段后处理；取后的样品将其切碎，充分混匀，用四分法取一部分或全部用组织捣碎机匀浆后，放入聚乙烯瓶中。

水果和食用菌随机取样 $500\mathrm{g}$ ，粉碎后充分混匀，放入聚乙烯瓶或袋中。

谷类随机取样 $500\mathrm{g}$ ，粉碎后使其全部可通过 $425~\mu \mathrm{m}$ 的标准网筛，放入聚乙烯瓶或袋中。

油料、茶叶、坚果和香辛料(调味料)随机取样 $500\mathrm{g}$ ，粉碎后充分混匀，放入聚乙烯瓶或袋中。

植物油类搅拌均匀，放入聚乙烯瓶中。

试样应于 $-18^{\circ}\mathrm{C}$ 及以下温度条件下保存。

# 7 分析步骤

## 7.1 提取

## 7.1.1 蔬菜、水果和食用菌

称取 $10\mathrm{g}$ 试样（食用菌 $5\mathrm{g}$ ）（精确至 $0.01\mathrm{g}$ ）于 $50\mathrm{mL}$ 具塞聚丙烯离心管(5.4)中，加入 $10\mathrm{mL}L$ -半胱氨酸盐酸盐的盐酸溶液(4.2.2)，振荡 $30\mathrm{min}$ ，待衍生。

淀粉含量高于 $10\%$ 的试样，加入 $10~\mathrm{mL}~L^{-}$ 半胱氨酸盐酸盐的盐酸溶液(4.2.2)，再加入 $2\mathrm{g}$ 氯化钠(4.1.5)和 $5\mathrm{mL}$ 水，振荡 $30\mathrm{min},3800\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，用吸管取全部水相至 $50~\mathrm{mL}$ 具塞聚丙烯离心管(5.4)，待衍生。

## 7.1.2 谷物、油料、坚果和植物油

称取 $5\mathrm{g}$ 试样(精确至 $0.01\mathrm{g}$ )于 $50~\mathrm{mL}$ 具塞聚丙烯离心管(5.4)中，加入 $10~\mathrm{mL}L^{-}$ 半胱氨酸盐酸盐的盐酸溶液(4.2.2)，再加入 $2\mathrm{g}$ 氯化钠(4.1.5)和 $5\mathrm{mL}$ 水，振荡 $30\mathrm{min},3800\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，用吸管取全部水相至 $50~\mathrm{mL}$ 具塞聚丙烯离心管(5.4)，待衍生。

## 7.1.3 茶叶和香辛料(调味料)

称取 $2\mathrm{g}$ 茶叶试样（精确至 $0.01\mathrm{g}$ ）于 $50~\mathrm{mL}$ 具塞聚丙烯离心管(5.4)中，加入 $10~\mathrm{mL}~L^{-}$ 半胱氨酸盐酸盐的盐酸溶液(4.2.2)，振荡 $30\mathrm{min}$ ，待衍生。

称取 $2\mathrm{g}$ 香辛料(调味料)试样(精确至 $0.01\mathrm{g}$ )于 $50~\mathrm{mL}$ 具塞聚丙烯离心管(5.4)中，加入 $10~\mathrm{mL}L$ 半胱氨酸盐酸盐的盐酸溶液(4.2.2)，再加入 $2\mathrm{g}$ 氯化钠(4.1.5)和 $5\mathrm{mL}$ 水，振荡 $30\mathrm{min},3800\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，用吸管取全部水相至 $50~\mathrm{mL}$ 具塞聚丙烯离心管(5.5)，待衍生。

## 7.2 衍生化

上述提取溶液中依次加入 $1\mathrm{mL}$ 氢氧化钠溶液(4.2.3)和 $2\mathrm{mL}$ 氯化镍溶液(4.2.4)，振荡 $30\mathrm{min}$ 。

## 7.3 萃取

蔬菜、水果、油料、坚果和植物油试样中加入 $10~\mathrm{mL}$ 正己烷(4.1.6)，谷物和食用菌试样中加入 $5\mathrm{mL}$ 正己烷(4.1.6)，茶叶和香辛料(调味料)试样中加入 $4\mathrm{mL}$ 正己烷(4.1.6)，试样振荡 $30\mathrm{min},3800\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，取 $1\mathrm{mL}$ 上清液（正己烷相）于进样小瓶中，用于测定。

## 7.4 测定

### 7.4.1 色谱参考条件

a）色谱柱：HP-5(5%苯基甲基聚硅氧烷)毛细管柱 $(0.25\mu \mathrm{m},30\mathrm{m}\times 0.25\mathrm{mm})$ ，或相当者；  
b）色谱柱温度： $70^{\circ}\mathrm{C}$ 保持 $1\mathrm{min}$ ，以 $20^{\circ}\mathrm{C / min}$ 程序升温至 $220^{\circ}\mathrm{C}$ ，保持 $1\mathrm{min}$   
c）载气：氮气（纯度 $>99.999\%$ ），恒流模式；  
d) 载气总流速: $50.0 \mathrm{~mL} / \mathrm{min}$ ;   
e) 柱流速: $1.0 \mathrm{~mL} / \mathrm{min}$ ;   
f） 隔垫吹扫： $3.0~\mathrm{mL / min}$   
g）进样口温度： $250^{\circ}C$   
h) 进样量: $2 \mu \mathrm{L}$ ;   
i）进样方式：不分流进样，开阀时间 $1\mathrm{min}$   
j）检测器：电子捕获检测器（ECD），温度 $280^{\circ}\mathrm{C}$

### 7.4.2 标准工作曲线

用正己烷溶液(4.1.6)将沙蚕毒素标准工作液逐级稀释得到质量浓度分别为 $0.01\mathrm{mg / L},0.1\mathrm{mg / L}$ 、 $0.5\mathrm{mg / L},1\mathrm{mg / L}$ 和 $5\mathrm{mg / L}$ 的系列标准工作溶液，质量浓度由低至高依次进样测定，以峰面积为横坐标，质量浓度为纵坐标绘制标准曲线回归方程。标准溶液色谱图见附录A。

### 7.4.3 测定

在7.4.1色谱参考条件下，沙蚕毒素参考保留时间为 $7.02\mathrm{min}$ ，按保留时间定性。待测样液中沙蚕毒素的响应值应在标准曲线范围内，超过线性范围应稀释后再进样分析，外标法定量。

## 7.5 空白试验

不加试样或仅加空白试样的空白试验应采用与试样测定完全相同的试剂、设备和步骤等进行。

# 8 结果计算

试料中的杀虫单/杀虫双/杀虫环/杀螟丹含量以质量分数 $\omega$ 计，单位为毫克每千克 $(\mathrm{mg / kg})$ ，按公式(1)或公式(2)计算。

$$
\omega = \frac {\rho_ {1} \times A \times V}{A _ {S} \times m} \times \frac {1 0 0 0}{1 0 0 0} \times k \tag {1}
$$

$$
\omega = \frac {\rho_ {2} \times V}{m} \times \frac {1 0 0 0}{1 0 0 0} \times k \tag {2}
$$

式中：

$\omega$ ——试样中杀虫单/杀虫双/杀虫环/杀螟丹残留量的数值，单位为毫克每千克 $(\mathrm{mg / kg})$ ；

$\rho_{1}$ ——标准工作溶液中沙蚕毒素质量浓度的数值，单位为毫克每升 $(\mathrm{mg / L})$

$\rho_{2}$ ——从标准工作曲线中得到的试样溶液中沙蚕毒素质量浓度的数值，单位为毫克每升 $(\mathrm{mg / L})$

$A$ ——试样溶液中沙蚕毒素的色谱图峰面积；

$A_{S}$ ——标准工作溶液中沙蚕毒素的色谱图峰面积；

$m$ ——试样质量的数值，单位为克 $(\mathrm{g})$ ；

$V$ ——试样正己烷提取液体积的数值，单位为毫升 $(\mathrm{mL})$ ；

$k$ ——沙蚕毒素换算成杀虫单/杀虫双/杀虫环/杀螟丹的系数，其中杀虫单为2.35，杀虫双为2.38，杀虫环为1.21，杀螟丹为1.59（若配置标准溶液时已进行折算，则在此步计算过程中 $k = 1$ ；摩尔质量：杀虫单为 $351.39\mathrm{g / mol}$ ，杀虫双为 $355.4\mathrm{g / mol}$ ，杀虫环为 $181.34\mathrm{g / mol}$ ，杀螟丹为 $237.34\mathrm{g / mol}$ ，沙蚕毒素为 $149.28\mathrm{g / mol}$ ）。

计算结果以重复性条件下获得的2次独立测定结果的算术平均值表示，保留2位有效数字，当结果大于 $1\mathrm{mg / kg}$ 时保留3位有效数字。

# 9 精密度

9.1 在重复性条件下，2次独立测定结果的绝对差值不大于重复性限 $(r)$ ，重复性限 $(r)$ 的数据为：

a) 含量为 $0.05 \mathrm{mg} / \mathrm{kg}$ 时, 重复性限 $(r)$ 为 0.012;  
b）含量为 $0.2\mathrm{mg / kg}$ 时，重复性限 $(r)$ 为0.054；  
c）含量为 $0.5\mathrm{mg / kg}$ 时，重复性限 $(r)$ 为0.091；  
d) 含量为 $1\mathrm{mg / kg}$ 时，重复性限 $(r)$ 为0.22；  
e) 含量为 $3\mathrm{mg / kg}$ 时，重复性限 $(r)$ 为0.15；  
f）含量为 $6\mathrm{mg / kg}$ 时，重复性限 $(r^2)$ 为0.34；  
g）含量为 $20\mathrm{mg / kg}$ 时，重复性限 $(r)$ 为1.2；  
（h）含量为 $40\mathrm{mg / kg}$ 时，重复性限 $(r)$ 为3.1。

9.2 在再现性条件下，2次独立测定结果的绝对差值不大于再现性限 $(R)$ ，再现性限 $(R)$ 的数据为：

a）含量为 $0.05\mathrm{mg / kg}$ 时，再现性限 $(R)$ 为0.019；  
b）含量为 $0.2\mathrm{mg / kg}$ 时，再现性限 $(R)$ 为0.064；  
c）含量为 $0.5\mathrm{mg / kg}$ 时，再现性限 $(R)$ 为0.18；  
d）含量为 $1\mathrm{mg / kg}$ 时，再现性限 $(R)$ 为0.37；  
e）含量为 $3\mathrm{mg / kg}$ 时，再现性限 $(R)$ 为0.22；  
f）含量为 $6\mathrm{mg / kg}$ 时，再现性限 $(R)$ 为2.1；  
g）含量为 $20\mathrm{mg / kg}$ 时，再现性限 $(R)$ 为3.9；  
h）含量为 $40\mathrm{mg / kg}$ 时，再现性限 $(R)$ 为7.7。

# 10 定量限

本文件方法的定量限为 $0.05\mathrm{mg / kg}$

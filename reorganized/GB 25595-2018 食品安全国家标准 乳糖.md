# 食品安全国家标准 乳 糖

2018-06-21 发布

2018-12-21 实施

# 前言

本标准代替GB25595—2010《食品安全国家标准 乳糖》。

本标准与GB25595—2010相比，主要变化如下：

——修改了范围；
——修改了术语和定义；
——修改了原料要求；
——修改了理化指标；
——增加了污染物限量。

# 1 范围

本标准适用于食用乳糖。

# 2 术语和定义

## 2.1 乳糖

从牛(羊)乳或乳清中提取出来的碳水化合物，以无水或含一分子结晶水的形式存在，或以这两种混合物的形式存在。

# 3 技术要求

## 3.1 原料要求

原料应符合相应的食品标准和有关规定。

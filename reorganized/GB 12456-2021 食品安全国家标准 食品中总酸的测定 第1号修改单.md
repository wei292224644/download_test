# 前言

本标准代替GB/T12456—2008《食品中总酸的测定》；代替GB/T4928—2008《啤酒分析方法》、GB/T5009.39—2003《酱油卫生标准的分析方法》、GB/T5009.40—2003《酱卫生标准的分析方法》、GB/T5009.41—2003《食醋卫生标准的分析方法》、GB/T10345—2007《白酒分析方法》、GB/T21999—2008《蚝油》中总酸的测定方法。

本标准与GB/T12456—2008相比，主要变化如下：

——标准名称修改为"食品安全国家标准 食品中总酸的测定"；  
——修改了标准的适用范围；  
——增加了"电位滴定法"；  
——修改了"酸碱滴定法"、"pH电位法"的部分技术参数及方法名称。

# 食品安全国家标准 食品中总酸的测定

# 1 范围

本标准规定了果蔬制品、饮料、酒类和调味品中总酸的测定方法。

本标准第一法适用于果蔬制品、饮料（澄清透明类）、白酒、米酒、白葡萄酒、啤酒和白醋中总酸的测定。

本标准第二法适用于果蔬制品、饮料、酒类和调味品中总酸的测定。

本标准第三法适用于果蔬制品、饮料、酒类和调味品中总酸的测定。

# 第一法 酸碱指示剂滴定法

# 2 原理

根据酸碱中和原理，用碱液滴定试液中的酸，以酚酞为指示剂确定滴定终点。按碱液的消耗量计算食品中的总酸含量。

# 3 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

## 3.1 试剂

3.1.1 乙醇 $(\mathrm{C}_2\mathrm{H}_5\mathrm{OH})$ ： $95\%$   
3.1.2 酚酞 $(\mathrm{C}_{20}\mathrm{H}_{14}\mathrm{O}_4)$ 。  
3.1.3 氢氧化钠(NaOH)。

## 3.2 试剂配制

3.2.1 无二氧化碳的水

将水煮沸 $15\mathrm{min}$ 以逐出二氧化碳，冷却，密闭。

3.2.2 酚酞指示液 $(10\mathrm{g / L})$

称取 $1\mathrm{g}$ 酚酞，溶于乙醇 $(95\%)$ ，用乙醇 $(95\%)$ 稀释至 $100~\mathrm{mL}$ 。

3.2.3 氢氧化钠标准滴定溶液 $(0.1\mathrm{mol / L})$

按照GB/T5009.1的要求配制和标定，或购买经国家认证并授予标准物质证书的标准滴定溶液。

3.2.4 氢氧化钠标准滴定溶液 $(0.01\mathrm{mol / L})$ ：用移液管吸取 $100\mathrm{mL}0.1\mathrm{mol / L}$ 氢氧化钠标准滴定溶液至容量瓶，用水稀释到 $1000\mathrm{mL}$ ，现用现配，必要时重新标定。  
3.2.5 氢氧化钠标准滴定溶液 $(0.05\mathrm{mol / L})$ ：用移液管吸取 $50~\mathrm{mL}~0.1~\mathrm{mol / L}$ 氢氧化钠标准滴定溶液至容量瓶，用水稀释到 $100~\mathrm{mL}$ ，现用现配，必要时重新标定。

# 4 仪器和设备

4.1 分析天平：感量 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$   
4.2 碱式滴定管：容量 $10~\mathrm{mL}$ ，最小刻度 $0.05~\mathrm{mL}$   
4.3 碱式滴定管：容量 $25~\mathrm{mL}$ ，最小刻度 $0.1\mathrm{mL}$   
4.4 水浴锅。  
4.5 锥形瓶： $100~\mathrm{mL},150~\mathrm{mL},250~\mathrm{mL}$   
4.6 移液管： $25\mathrm{mL}, 50\mathrm{mL}, 100\mathrm{mL}$ 。  
4.7 均质器。  
4.8 超声波发生器。  
4.9 研钵。  
4.10 组织捣碎机。

# 5 分析步骤

## 5.1 试样的制备

试样放置常温密封保存。

### 5.1.1 液体样品

不含二氧化碳的样品：充分混合均匀，置于密闭玻璃容器内。

含二氧化碳的样品：至少取 $200\mathrm{g}$ 样品（精确到 $0.01\mathrm{g}$ ）于 $500\mathrm{mL}$ 烧杯中，在减压下摇动 $3\mathrm{min} \sim 4\mathrm{min}$ ，以除去液体样品中的二氧化碳。

### 5.1.2 固体样品

取有代表性的样品至少 $200\mathrm{g}$ (精确到 $0.01\mathrm{g}$ ), 置于研钵或组织捣碎机中, 加入与样品等量的无二氧化碳水, 用研钵研碎或用组织捣碎机捣碎, 混匀成浆状后置于密闭玻璃容器内。

### 5.1.3 固液混合样品

按样品的固、液体比例至少取 $200\mathrm{g}$ （精确到 $0.01\mathrm{g}$ ），用研钵研碎或用组织捣碎机捣碎，混匀后置于密闭玻璃容器内。

## 5.2 待测溶液的制备

### 5.2.1 液体样品

称取 $25\mathrm{g}$ (精确至 $0.01\mathrm{g}$ ) 或用移液管吸取 $25.0\mathrm{mL}$ 试样至 $250\mathrm{mL}$ 容量瓶中，用无二氧化碳的水定容至刻度，摇匀。用快速滤纸过滤，收集滤液，用于测定。

### 5.2.2 其他样品

称取 $25\mathrm{g}$ 试样（精确至 $0.01\mathrm{g}$ ），置于 $150\mathrm{mL}$ 带有冷凝管的锥形瓶中，加入约 $50\mathrm{mL}80^{\circ}\mathrm{C}$ 无二氧化碳的水，混合均匀，置于沸水浴中煮沸 $30\mathrm{min}$ （摇动2次～3次，使试样中的有机酸全部溶解于溶液中），取出，冷却至室温，用无二氧化碳的水定容至 $250\mathrm{mL}$ ，用快速滤纸过滤，收集滤液，用于测定。

## 5.3 分析步骤

5.3.1 根据试样总酸的可能含量，使用移液管吸取 $25\mathrm{mL}, 50\mathrm{mL}$ 或者 $100\mathrm{mL}$ 试液，置于 $250\mathrm{mL}$ 三角瓶中，加入 $2\sim 4$ 滴 $(10\mathrm{g / L})$ 酚酞指示液，用 $0.1\mathrm{mol / L}$ 氢氧化钠标准滴定溶液(若为白酒等样品，总酸 $\leqslant 4\mathrm{g / kg}$ ，可用 $0.01\mathrm{mol / L}$ 或 $0.05\mathrm{mol / L}$ 氢氧化钠滴定溶液)滴定至微红色 $30~s$ 不褪色。记录消耗 $0.1\mathrm{mol / L}$ 氢氧化钠标准滴定溶液的体积数值。

### 5.3.2 空白试验

按5.3.1的操作，用同体积无二氧化碳的水代替试液做空白试验，记录消耗氢氧化钠标准滴定溶液的体积数值。

# 6 结果计算

总酸的含量按式(1)计算。

$$
X = \frac {\left[ c \times \left(V _ {1} - V _ {2}\right) \right] \times k \times F}{m} \times 1 0 0 0 \quad \dots \tag {1}
$$

式中：

$X$ ——试样中总酸的含量，单位为克每千克 $(\mathrm{g / kg})$ 或克每升 $(\mathrm{g / L})$ ；

$c$ ——氢氧化钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol / L})$ ；

$V_{1}$ ——滴定试液时消耗氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——空白试验时消耗氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$k$ ——酸的换算系数：苹果酸，0.067；乙酸，0.060，酒石酸，0.075；柠檬酸，0.064；柠檬酸（含一分子结晶水），0.070；乳酸，0.090；盐酸，0.036；硫酸，0.049；磷酸，0.049；

$F$ ——试液的稀释倍数；

$m$ ——试样的质量，单位为克(g)或吸取试样的体积，单位为毫升 $(\mathrm{mL})$ ；

1000 ——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留到小数点后两位。

# 7 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

# 第二法 pH计电位滴定法

# 8 原理

根据酸碱中和原理，用氢氧化钠标准滴定溶液滴定试液中的酸，中和试样溶液至 $\mathrm{pH}$ 为8.2（若酸为磷酸，则终点为 $8.7\sim 8.8)$ 时，确定为滴定终点。按碱液的消耗量计算食品中的总酸含量。

# 9 试剂和材料

除非另有说明，本方法所用试剂均为分析纯，水为GB/T6682规定的二级水。

## 9.1 试剂

9.1.1 氢氧化钠(NaOH)。  
9.1.2 磷酸氢二钾 $\mathrm{(K_2HPO_4)}$ 。  
9.1.3 磷酸二氢钾 $\mathrm{(KH_2PO_4)}$

## 9.2 试剂配制

9.2.1 氢氧化钠标准滴定溶液 $(0.1\mathrm{mol / L})$ ：同3.2。  
9.2.2 pH8.0缓冲溶液：取磷酸氢二钾 $5.59\mathrm{g}$ 和磷酸二氢钾 $0.41\mathrm{g}$ ，用水定容至 $1000~\mathrm{mL}$ 。

# 10 仪器和设备

10.1 分析天平：感量 $0.01\mathrm{g}$ 和 $0.1\mathrm{mg}$ 。  
10.2 pH计：精度 $\pm 0.1(\mathrm{pH})$ 。  
10.3 磁力搅拌器。  
10.4 水浴锅。  
10.5 烧杯： $100~\mathrm{mL},150~\mathrm{mL},250~\mathrm{mL}$   
10.6 移液管： $25\mathrm{mL}, 50\mathrm{mL}, 100\mathrm{mL}$ 。

# 11 分析步骤

## 11.1 试样的制备

按5.1制备。

## 11.2 待测溶液的制备

按5.2制备。

## 11.3 分析步骤

11.3.1 根据试样总酸的可能含量，使用移液管吸取 $25~\mathrm{mL}, 50~\mathrm{mL}$ 或者 $100~\mathrm{mL}$ 试液，置于 $150~\mathrm{mL}$ 烧杯中。将酸度计电源接通，稳定后，根据使用的 $\mathrm{pH}$ 计校正规程或用 $\mathrm{pH}8.0$ 缓冲溶液校正 $\mathrm{pH}$ 计。将盛有试液的烧杯放到磁力搅拌器上，浸入酸度计电极。按下 $\mathrm{pH}$ 读数开关，开动搅拌器，迅速用 $0.1\mathrm{mol / L}$ 氢氧化钠标准滴定溶液(若为白酒等样品，总酸 $\leqslant 4\mathrm{g / kg}$ ，可用 $0.01\mathrm{mol / L}$ 或 $0.05\mathrm{mol / L}$ 氢氧化钠标准滴定溶液)滴定，随时观察溶液 $\mathrm{pH}$ 变化。接近滴定终点时，放慢滴定速度。一次滴加半滴（最多一滴），直至溶液的 $\mathrm{pH}$ 达到终点，记录消耗氢氧化钠标准滴定溶液的体积的数值(各种酸滴定终点的 $\mathrm{pH}$ ：磷酸， $8.7\sim 8.8$ ；其他酸，8.2）。

### 11.3.2 空白试验

按11.3.1的操作，用无二氧化碳的水代替试液做空白试验，记录消耗氢氧化钠标准滴定溶液的体积数值。

# 12 结果计算

同第6章的要求。

# 13 精密度

在重复性条件下获得的两次独立测定结果的绝对差值不得超过算术平均值的 $10\%$ 。

## 二、6 结果计算

在"……结果保留到小数点后两位。"之后增加：

当计算结果以 $\mathrm{mL} / 100\mathrm{mL}$ 表示时，按照式(2)进行计算：

$$
X = \frac {\left[ c \times \left(V _ {1} - V _ {2}\right) \right] \times F}{V} \times 1 0 0 \tag {2}
$$

式中：

$X$ ——试样中总酸的含量，单位为毫升每100毫升 $(\mathrm{mL} / 100\mathrm{mL})$

$c$ ——氢氧化钠标准滴定溶液的浓度，单位为摩尔每升 $(\mathrm{mol / L})$ ；

$V_{1}$ ——滴定试液时消耗氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$V_{2}$ ——空白试验时消耗氢氧化钠标准滴定溶液的体积，单位为毫升 $(\mathrm{mL})$ ；

$F$ ——试液的稀释倍数；

$V$ ——试样的体积，单位为毫升 $(\mathrm{mL})$ ；

100——换算系数。

计算结果以重复性条件下获得的两次独立测定结果的算术平均值表示，结果保留至小数点后两位。

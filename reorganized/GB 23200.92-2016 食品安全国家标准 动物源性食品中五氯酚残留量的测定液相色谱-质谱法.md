# 前言

本标准按照GB/T1.1—2009给出的规则起草。

本标准代替SN/T2445—2010《进出口动物源食品中五氯酚残留量检测方法液相色谱一质谱/质谱法》。与SN/T2445—2010相比，主要变化如下：

标准文本格式修改为食品安全国家标准文本格式；  
——标准名称中“进出口动物源食品”改为“动物源性食品”；  
标准范围中增加“其他食品可参照执行”。

本标准所代替标准的历次版本发布情况为：

—SN/T2445—2010。

# 食品安全国家标准 动物源性食品中五氯酚残留量的测定 液相色谱一质谱法

# 1 范围

本标准规定了动物源性食品中五氯酚残留量的制样、液相色谱一质谱测定方法。

本标准适用于猪肝、猪肾、猪肉、牛奶、鱼肉、虾、蟹等动物源性食品中五氯酚残留的测定，其他食品可参照执行。

# 2 规范性引用文件

下列文件对于本文件的应用是必不可少的。凡是注日期的引用文件，仅注日期的版本适用于本文件。凡是不注日期的引用文件，其最新版本(包括所有的修改单)适用于本文件。

GB/T6682 分析实验室用水规格和试验方法

# 3 原理

试样中的五氯酚残留用碱性乙腈水溶液提取，经MAX固相萃取柱净化，浓缩、定容后，用液相色谱一质谱仪检测和确证，外标法定量。

# 4 试剂和材料

除另有说明外，所用试剂均为分析纯，水为符合GB/T6682中规定的一级水。

## 4.1 试剂

4.1.1 甲醇 $\mathrm{(CH_3OH)}$ ：色谱纯。  
4.1.2 乙腈 $\mathrm{(CH_3CN)}$ ：色谱纯。  
4.1.3 三乙胺 $(\mathrm{C_6H_{15}N})$   
4.1.4 甲酸（HCOOH）：色谱纯。  
4.1.5 乙酸铵 $\mathrm{(CH_3COONH_4)}$   
4.1.6 氨水 $(\mathrm{NH}_3\cdot \mathrm{H}_2\mathrm{O})$ 。

## 4.2 溶液配制

4.2.1 乙腈一水溶液 $(7 + 3)$ ：准确量取 $70~\mathrm{mL}$ 乙腈和 $30~\mathrm{mL}$ 水，混合摇匀。  
4.2.2 $5\%$ 三乙胺乙腈一水溶液：准确量取 $50~\mathrm{mL}$ 三乙胺，转移入 $1000~\mathrm{mL}$ 容量瓶，用乙腈一水溶液（7+3）定容至刻度，混合均匀。  
4.2.3 $5 \%$ 氨水溶液：量取 $200~\mathrm{mL}$ 浓氨水 $(25\%)$ ，转移入 $1000~\mathrm{mL}$ 容量瓶，用水定容至刻度，混合均匀。  
4.2.4 $4\%$ 甲酸甲醇溶液：量取 $10~\mathrm{mL}$ 甲酸，转移入 $250~\mathrm{mL}$ 容量瓶，用甲醇定容至刻度，混合均匀。  
4.2.5 $2\%$ 甲酸甲醇水溶液：取一定量 $4 \%$ 甲酸甲醇溶液与水等体积混合均匀。  
4.2.6 $5 \mathrm{mmol} / \mathrm{L}$ 乙酸铵水溶液：准确称取 $0.3854 \mathrm{~g}$ 乙酸铵，溶于水中，转移到 $1000 \mathrm{~mL}$ 容量瓶中，定容至刻度，混合均匀。

## 4.3 标准品

五氯酚钠标准品(pentachlorophenolsodium salt,CAS号：131-52-2)：纯度 $\geq 94.5\%$ ，或相当者。

## 4.4 标准溶液配制

### 4.4.1 标准储备溶液 $(200\mathrm{mg / L})$

称量经折算相当于 $10 \mathrm{mg}$ (精确到 $0.1 \mathrm{mg}$ ) 五氯酚的标准品, 用甲醇溶解并定容至 $50 \mathrm{~mL}$ 棕色容量瓶中。此储备液可在 $-18^{\circ}\mathrm{C}$ 以下避光存放 12 个月。

### 4.4.2 标准工作溶液

标准工作溶液根据需要，使用前用空白样品基质溶液配制。保存于 $4^{\circ}\mathrm{C}$ 冰箱内。

## 4.5 材料

4.5.1 固相萃取(SPE)柱：Oasis MAX, $3 \mathrm{~mL} / 60 \mathrm{mg}$ , 或相当者。使用前依次用 $5 \mathrm{~mL}$ 甲醇和 $5 \mathrm{~mL}$ 水活化。  
4.5.2 氮气：纯度 $\geq 99.99\%$

# 5 仪器和设备

5.1 液相色谱串联四极杆质谱仪：配电喷雾离子源。  
5.2 分析天平：感量 $0.01\mathrm{g}$ 和 $0.0001\mathrm{g}$   
5.3 涡旋振荡器。  
5.4 离心机， $5000\mathrm{r / min}$   
5.5 固相萃取真空装置。  
5.6 机械真空泵。  
5.7 组织捣碎机。  
5.8 均质器。  
5.9 超声波仪。  
5.10 吹氮浓缩仪。

# 6 试样制备与保存

## 6.1 取样部位

样品取样部位按GB2763的规定执行。

## 6.2 试样制备

制样操作过程中，应防止样品受到污染或发生残留物含量的变化。

### 6.2.1 动物肌肉、肝脏、肾脏、鱼肉、虾和蟹

从所取全部样品中取出有代表性样品约 $500 \mathrm{~g}$ , 用组织捣碎机充分捣碎均匀, 均分成 2 份, 分别装入洁净容器中, 密封, 并标明标记。

### 6.2.2 牛奶样品

从所取全部样品中取出有代表性样品约 $500\mathrm{g}$ ，充分混匀，均分成2份，分别装入洁净容器中，密封，并标明标记。

## 6.3 试样保存

样品于 $-18^{\circ}\mathrm{C}$ 以下冷冻保存。

# 7 分析步骤

## 7.1 提取

### 7.1.1 肌肉、鱼肉、肝脏、肾脏、虾和蟹

称取均质试样 $2 \mathrm{~g}$ (精确到 $0.01 \mathrm{~g}$ ), 置于 $10 \mathrm{~mL}$ 具螺旋盖聚丙烯离心管中, 加入 $6 \mathrm{~mL} 5 \%$ 三乙胺的乙腈一水溶液。肌肉和鱼肉样品均质提取 $2 \mathrm{~min}$ ; 肝脏、肾脏、虾和蟹样品涡旋振荡 $1 \mathrm{~min}$ , 超声提取 $5 \mathrm{~min}$ 。 $3000 \mathrm{r} / \mathrm{min}$ 离心 $5 \mathrm{~min}$ , 收集上清液于一具刻度离心管中。离心后的残渣用约 $6 \mathrm{~mL} 5 \%$ 三乙胺的乙腈一水溶液重复上述提取步骤 1 次, 合并上清液, 混匀。

### 7.1.2 牛奶

称取均质试样 $2\mathrm{g}$ （精确到 $0.01\mathrm{g}$），置于 $10\mathrm{mL}$ 具刻度螺旋盖聚丙烯离心管中，加入 $10\mathrm{mL}5\%$ 三乙胺的乙腈一水溶液，涡旋振荡 $1\mathrm{min}$ ，超声提取 $5\mathrm{min}$ ， $5000\mathrm{r / min}$ 离心 $5\mathrm{min}$ ，收集上清液于一具刻度离心管中。

## 7.2 固相萃取净化

将7.1中所得提取溶液转入经过预处理的MAX固相萃取柱中，以约1滴/s流速使样品溶液全部通过固相萃取柱，弃去流出液。依次用 $5\mathrm{mL}5\%$ 氨水溶液、 $5\mathrm{mL}$ 甲醇、 $5\mathrm{mL}2\%$ 甲酸的甲醇一水溶液淋洗，淋洗液完全通过小柱后，弃去流出液，用真空泵抽干固相萃取柱 $5\mathrm{min}$ 以上。以 $4\mathrm{mL}4\%$ 甲酸甲醇溶液洗脱，洗脱液用干净的 $15\mathrm{mL}$ 具刻度试管收集， $40^{\circ}\mathrm{C}$ 水浴下吹氮浓缩至 $1\mathrm{mL}$ ，用水定容至 $2\mathrm{mL}$ ，混匀。溶液以 $0.22\mu \mathrm{m}$ 有机滤膜过滤，供测定。

## 7.3 仪器参考条件

### 7.3.1 液相色谱参考条件

a）色谱柱：ZORBAX Eclipse XDB-C18, $1.8\mu \mathrm{m},50\mathrm{mm}\times 2.1\mathrm{mm}$ （内径），或相当者；  
b）流动相：甲醇 $+5\mathrm{mmol / L}$ 乙酸铵溶液，梯度洗脱条件见表1；  
c) 流速: $250 \mu \mathrm{L} / \mathrm{min}$ ;   
d) 柱温: $30^{\circ} \mathrm{C}$ ;   
e）进样量： $30~\mu \mathrm{L}$

### 7.4.1 定性测定

### 7.4.2 定量测定

在仪器最佳工作条件下, 对基质混合标准工作溶液进样, 以峰面积为纵坐标, 基质混合标准工作溶液浓度为横坐标绘制标准工作曲线。用标准工作曲线对样品进行定量, 样品溶液中待测物的响应值均应在仪器测定的线性范围内。五氯酚标准物质的多反应监测(MRM)色谱图参见附录B。

## 7.5 空白实验

除不加试样外，均按 $7.1\sim 7.4$ 的规定执行

# 8 结果计算和表述

采用外标法定量，按式(1)计算五氯酚残留量。

$$
X = \frac {c \times V}{m} \times \frac {1 0 0 0}{1 0 0 0}. \tag {1}
$$

式中：

$X$ ——试样中被测组分残留量，单位为微克每千克 $(\mu \mathrm{g} / \mathrm{kg})$ ；

c 从标准工作曲线得到的被测组分溶液浓度，单位为微克每升 $(\mu \mathrm{g} / \mathrm{L})$

V ——样品溶液最终定容体积，单位为毫升 $(\mathrm{mL})$

m 最终样品溶液所代表的试样质量，单位为克(g)；

1000 ——换算系数。

计算结果须扣除空白值，测定结果用平行测定的算术平均值表示，保留2位有效数字。

# 9 精密度

9.1 在重复性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 C 的要求。  
9.2 在再现性条件下获得的 2 次独立测定结果的绝对差值与其算术平均值的比值 (百分率), 应符合附录 D 的要求。

## 10.1 定量限

本方法对五氯酚的定量限为 $1.0\mu \mathrm{g / kg}$
